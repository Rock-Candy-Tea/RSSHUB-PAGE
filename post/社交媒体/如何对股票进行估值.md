<p>       我很喜欢研究股票，而且很早就接触过了“价值投资”。但是每到给所研究的股票进行估值的时候，就不知道该如何下手，看了市面上各种所谓的大V的书，也都没有找到答案。</p><p>       直到最近看到了唐朝写的一本《价值投资实战手册》的书，这本书里面就有详细的“估值”的方法，感觉非常实用，就在这里和大家分享一下我对股票估值的方法。</p><p>       首先我要啰嗦一下，“估值”对股票投资来说，只是一种所谓的“术”，根本说不上“道”的一层。为什么这么说呢？“道”对一个事物来说，就是最本质的东西，那么股票投资最根本的是什么呢？我认为就是股票之后所站立的企业。</p><p>        正是基于这样的认识，在我第1次接触巴菲特的“价值投资”的时候，我就“固执”地认为：你有多了解企业，就是你能在股市能赚多少的钱。这所以说了这么多，就是想告诉大家：<b>估值的基础是基于你对公司的了解程度。</b>而不是有一个“神秘的公式”可以算出企业的价值。要是这样的话，谁都可以在股海里畅游了。</p><p>        对公司的了解程度又可以分解成以下4个问题（唐朝只提了前3个问题），如果能回答以下四个问题，就代表了你对公司有了很深的了解：</p><p><b>    （1）公司的利润是否为真？</b></p><p><b>    （2）公司的利润是否可以持续？</b></p><p><b>    （3）公司是否需要持续投入大量资本来维持利润？</b></p><p><b>    （4）公司可能遇到的风险有哪些？</b></p><p>       以上4个问题就是你对公司进行估值的前提，如果这些问题，你不能很好的回答，那么你进行的估值与看着“K线图”买卖股票没有任何区别，所以你一定要切忌，切记！  </p><p>    说完了前提，那么现在就来说说如何估值吧，这里我就直接引用唐朝书中的内容：</p><blockquote>在All cash is equal 的原则下，对符合三大前提的企业，不区分行业特色、历史波动区间，一律以无风险收益率的倒数作为三年后合理估值的市盈率水平。<br><br><即无风险收益率为3%~4%之间，取值25~30倍PE。无风险收益率4~5%之间，取值20~25倍PE，依此类推>  <br><br>对于部分净利润含现金量低于100%，但其他方面都符合的企业，对净利润予以折扣处理。体现在速算中就是取稍低的市盈率，即1×80%×25=1×20。 <br><br>三年后合理估值的50%就是理想买点；当年市盈率超过50倍，或市值超过三年后合理估值上限的150%，就是卖点，以先到者为准；<br><br>期间波动，呆坐不动。永远好奇的唐朝，公众号：唐书房<a href="https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s%3F__biz%3DMzI5NzA5MDEzNg%3D%3D%26mid%3D2649980534%26idx%3D1%26sn%3Dc91aa28f933af3059c9dffb7bd1a4795%26chksm%3Df4bdb341c3ca3a5764d16d5d848a28b441b51850a3831c45ccea0381eb3a0c792d3c59da968d%26scene%3D126%26sessionid%3D1607514875%26key%3Dfac416e293c6ddd15bb771e796d0a1b288f51c4f5c96bce03d73589f5b9051ceca392bbe4a009a86bcd2fd1deeb44cb3feabd3ed3a9889b98d2422d2e15a8474496f95bf5b276c39502c5621f77851cf0be37b8a7a74bcc90fa7ca4a5da9f9067441dd453ca7d8fa069733349911ff73c881a627d9318096559b24f52c5ee7ee%26ascene%3D1%26uin%3DMjQ5NDgxMTY0MA%253D%253D%26devicetype%3DWindows%2B7%26version%3D6300002f%26lang%3Dzh_CN%26exportkey%3DAXkdV5nKACAnHJkwqZuytgc%253D%26pass_ticket%3D2gwYSnMOosWv1HjUzmu0MLZ5roMuZha8HNs2ln1E6hfm9o67d8n4BDO2OAs6xXbm%26wx_header%3D0" class=" wrap external" target="_blank" rel="nofollow noreferrer">老唐估值法答疑（续）</a></blockquote><p>    我认为以上这段话，就是他的《价值投资实战手册》的核心内容。下面针对上面的内容作一些解释。<br></p><p><b>All cash is equal</b></p><p>    这句话是巴菲特说的，意思大概是：所有赚到的钱都一样的。体现出一种“比较”的思想。举个例子：我种地赚到的钱，和你开公司赚到的钱是一样的。有<b>区别的是：赚得的多和少</b>。<br></p><p><b>无风险收益率</b><br>    无风险收益率可以从银行的<b>保本理财产品的收益率</b>、<b>货币基金的收益率</b>、<b>5年期国债的利率</b>。目前在3%~4%之间。然后取无风险收益率的倒数作为合理的估值PE。<br></p><p class="ztext-empty-paragraph"><br></p><p><b>买入点</b></p><p> 将公司第3年获得的利润按照无风险收益率折现后的50%作为买入点。这是在考虑<b>“安全边际”</b>的情况下得出的结果。<br></p><p>    下面举一个例子，在3%~4%的无风险利率情况下（也即20倍PE~25倍PE），比如一个公司当前的利润为1元，预计这个公司预计每年利润将会增长10%，那么3年后这个公司的利润（无分红）会是：<br></p><p>    第一年利润：1 元</p><p>    第二年利润：1 x (1+10%) = 1.1 元</p><p>    第二年利润：1 x (1+10%) x (1+10%) = 1.21元<br></p><p>    如果我想买入这个公司的股票，他的买入价应该是多少呢？买入价 = 1.21元 x 25 x 50% = 15.125 元，所以在买入这家公司的股价应该低于15元就买入。</p><p>    还是上面这家公司，在它产生1元利润的时候，我已经持有其股票了。按照这两个价格哪一个达到，按照哪一个价低卖出（45元左右）。<br></p><p>1（当前利润） x 50(PE) = 50 元</p><p>1.21（3年后利润） x 25(PE) x 150% = 45.375 元<br></p><p>      在这个例子中，一定要注意上面的25PE和50PE这两个数据，这不是代表着我们预期将来会有25倍和50倍的PE，这只代表我们的合理的心理预期。除非我们对公司利润预估错误，否则就会一直持有。</p><p>    上面就是股票估值的方法，不知道你注意到了没有，上面所有的数据中，只有对公司的利润是一个预计的变量。这说明什么呢？要对一个公司有一个相对准确的估值，就必须要对公司有很深的了解。否则一切都空中楼阁，和自己玩了一场自欺欺人的数字游戏而已。</p><p>    下面我将《价值投资实战手册》中我最喜欢一段话送给大家。股市如此，人生何尝不是这样呢？你认为呢？</p><blockquote>市场会重奖错误的行为，以鼓励他们下一次慷慨赴死。而坚持做且只做正确的事，并愿意为此放弃市场颁发给错误行为的奖金，这就是股市长久盈利的核心秘诀，是投资的核心之道。《价值投资实战手册》（唐朝）</blockquote><p></p>
