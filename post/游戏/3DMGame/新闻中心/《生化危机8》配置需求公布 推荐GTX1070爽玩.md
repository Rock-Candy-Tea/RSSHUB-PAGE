<div>   
<p style="text-indent:2em;">
今天是《生化危机》25周岁生日，卡普空在《<a target="_blank" href="https://www.3dmgame.com/games/residentevil8/">生化危机8</a>》Steam页面上公布了游戏PC配置需求。想运行游戏，最低需要Intel Core 
i5-750，GeForce GTX 1050 Ti显卡和8G内存。想流畅游玩，则需要Intel Core i7 8700、GeForce GTX 
1070和16G内存。
</p>
<p style="text-indent:2em;">
<strong><span style="color:#E56600;font-size:16px;">Steam商店地址：<a href="https://store.steampowered.com/app/1196590/Resident_Evil_Village/" target="_blank"><strong><span style="color:#337FE5;"><u>点击进入</u></span></strong></a></span></strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210322/1616395401_443595.jpg" alt="《生化危机8》配置需求公布 推荐GTX1070爽玩" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
<strong><span style="color:#E56600;font-size:16px;">配置需求</span></strong> 
</p>
<p style="text-indent:2em;">
<strong>最低配置：</strong> 
</p>
<p style="text-indent:2em;">
需要 64 位处理器和操作系统
</p>
<p style="text-indent:2em;">
操作系统：Windows 10(必须为64bit)
</p>
<p style="text-indent:2em;">
处理器：Intel Core i5-7500 / AMD Ryzen 3 1200
</p>
<p style="text-indent:2em;">
内存：8 GB RAM
</p>
<p style="text-indent:2em;">
显卡：英伟达GeForce GTX 1050 Ti with 4GB VRAM / AMD Radeon RX 560 with 4GB VRAM
</p>
<p style="text-indent:2em;">
DirectX 版本: 12
</p>
<p style="text-indent:2em;">
附注事项: 
画面设定为"性能优先"的情况下，能够以1080p/60fps进行游戏。若负荷过大，帧数可能会降低。为了支持光线追踪，显卡性能需达英伟达GeForce RTX 
2060或AMD Radeon RX 6700 XT。 由于游戏仍处于开发阶段，所记述的配置有可能出现变动。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210322/1616395401_827070.jpg" alt="《生化危机8》配置需求公布 推荐GTX1070爽玩" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
<strong>推荐配置：</strong> 
</p>
<p style="text-indent:2em;">
需要 64 位处理器和操作系统
</p>
<p style="text-indent:2em;">
操作系统：Windows 10(必须为64bit)
</p>
<p style="text-indent:2em;">
处理器：Intel Core i7 8700 / AMD Ryzen 5 3600
</p>
<p style="text-indent:2em;">
内存：16 GB RAM
</p>
<p style="text-indent:2em;">
显卡：英伟达GeForce GTX 1070 / AMD Radeon RX 5700
</p>
<p style="text-indent:2em;">
DirectX 版本：12
</p>
<p style="text-indent:2em;">
附注事项: 画面设定为"优先图像"的情况下，能够以1080p/60fps进行游戏。若负荷过大，帧数可能会降低。 
为了支持光线追踪，显卡性能需达英伟达GeForce RTX 2070或AMD Radeon RX 6700 XT。 
由于游戏仍处于开发阶段，所记述的配置有可能出现变动。
</p>
<p style="text-indent:2em;">
<strong>光追配置：</strong> 
</p>
<p style="text-indent:2em;">
显卡：英伟达GeForce GTX 2070(4K/45fps)/英伟达GeForce GTX 3070(4K/60fps)/AMD Radeon RX 
6700 XT(4K/45fps)/AMD Radeon RX 6800(4K/45fps)/AMD Radeon RX 6900 
XT(4K/60fps)
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210322/1616395401_517376.jpg" alt="《生化危机8》配置需求公布 推荐GTX1070爽玩" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
在《生化危机8》中展开令人毛骨悚然的绝命拚搏。伊森一家的平静生活因克里斯·雷德菲尔德而又陷入混乱之中，他再次身陷无法逃离的梦魇中。这款系列第8部主要作品由日益成熟的游戏引擎“RE 
ENGINE”创造，将带来最新<a target="_blank" href="https://www.3dmgame.com/tag/shengcun_1/">生存</a>“动作”<a target="_blank" href="https://www.3dmgame.com/tag/kongbu_1/">恐怖</a>游戏体验。
</p>
<p style="text-indent:2em;">
《生化危机8》将于5月7日发售，登陆PS4、PS5、Xbox One，XSX/S和PC平台，支持中文配音。
</p>
