<h3>
伯利薩</h3>
<div>
 </div>
<div>
　　伯利薩族據傳是古代馬雅人的後裔，擁有獨特的古老文明，不僅精通各式狩獵技巧，在艱苦的環境生存之下，更培養出了堅定不決的意志力與高超的戰鬥能力，擅長使用長弓，其精湛的弓術，為伯利薩族贏得為「英雄」的美名。</div>

<h3>
亞特蘭提斯後裔</h3>
<div>
 </div>
<ul>
<li>
<b>迅捷致命，擅長單點擊殺的神射手「長弓」</b></li>
</ul>
<div>
　　專門進行長距離攻擊的射手，可以無視前排，攻擊場上任何一個目標。雖然血量較低，但因為高敏捷係數加成，長弓有其他職業望塵莫及的迴避能力之外，還擁有「箭雨」、「渾沌之風」等強力範圍技能，給予敵方全體造成毀滅性的傷害。職業特性提供全隊的物理攻擊力加成，在對王作戰時，表現特別出色。</div>

<li>
<b>貫穿敵人，致命撕裂直排的射手「火槍」</b></li>
</ul>
<div>
　　攻擊具有貫穿性，可一次攻擊直排所有單位的「火槍」，最大的特點就是爆發力驚人，擁有高爆擊率和傷害。職業特性是可以提升全體傭兵與自身的爆擊率。施放「爆裂彈」技能的時候，除目標受到直接攻擊外，在砲彈爆裂時，還可對目標周圍十字區塊的敵人皆造成傷害。但高傷害的同時，火槍也擁有一個顯著的缺點，就是生存能力較低，更需要仰賴隊友的保護。</div>

<ul>
<li>
<b>擅長面對多數敵人，範圍轟炸型射手「巨砲」</b></li>
</ul>
<div>
　　「巨炮」的職業特性是提升全體成員的命中率，裝備火炮攻擊範圍極廣，擅長範圍轟炸。因為具備高命中率以及遠程範圍攻擊的特長，在面對密集且大量的敵人時，擁有強大的優勢。不僅如此，巨砲更擁有多種干擾技能：施放「煙霧砲彈」技能時，將散發出煙霧，降低敵人的命中率與攻擊力；以及攻擊後可讓敵人陷入混亂，逆轉戰局的「混亂炮擊」。但也因為巨砲攜帶重裝備的關係，行動力與其他職業相比略低一些，在有限的行動力內完成最有效率的操作，便是致勝的關鍵。</div>

<b>以樂曲削弱對手的輔助職業「樂器」</b></li>

<div>
　　擁有出色的行動力，控場能力優異的遠距離樂手，擁有強大的持續性與減益魔法。攻擊方式是彈奏不同的樂曲，提供隊友強力 buff 或是削弱並控制敵手。職業特性可以增加全隊傭兵的魔法防禦力，不僅如此，樂器與其他遠程型職業相比，有較高的防禦力與更出色的行動力，在戰鬥中更具優勢。</div>

<ul>
<li>
<b>可攻可輔，施展多元魔法的全面型巫師「法杖」</b></li>
</ul>
<div>
　　「法杖」職業是萬能多元的魔法師，有著優異的智力與魔法防禦力，精通各種領域的魔法技能，職業特性可以增加隊伍成員的智力屬性，法杖職業具備恢復生命的法術以及範圍攻擊的手段，在隊伍中擁有舉足輕重的地位。習得「冥想」技能後，可以減少技能的冷卻，使其能夠迅速施放技能來取得優勢，而在戰局緊張之時，亦有恢復生命的技能，對我軍所有成員進行治療，一舉逆轉戰局。</div>

<h3>
亞特蘭提斯人</h3>
<div>
 </div>
<ul>
<li>
<b>高迴避及高敏捷的遠距離範圍連擊職業「長鞭」</b></li>
</ul>
<div>
　　注重爆發力的遠距離攻擊「長鞭」，具有大範圍且靈活的攻擊能力，職業特性可以增加隊伍全體的攻擊力以及連擊率，也因此長鞭有高機率可以在單一回合發動多次攻擊，長鞭能夠針對任何敵方位置的橫排進行橫掃攻擊，因為善於操縱雷電的特性，累積足夠怒氣後可覺醒體內的雷電之力，幻化為「雷神型態」，在場上釋放強大的雷電技能，徹底主宰戰場。</div>

<b>精通各式傷害手段，有著強悍作戰技巧的攻擊型法師「閃電法杖」</b></li>
</ul>
<div>
<div>
　　「閃電法杖」是精通作戰技巧的法系職業，具備操縱雷電的能力，透過呼喚雷電來強化隊伍，並使用雷電法術進行攻擊。職業特性可以增加全隊的魔法技能等級，閃電法杖攻擊時，可對單一敵人及其四周造成閃電傷害。亦能夠施展「雷電祝福」，增加隊伍的攻擊力、連擊率以及技能等級，是特別擅長作戰的魔法型職業。</div>
