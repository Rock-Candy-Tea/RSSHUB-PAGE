<p>《上古卷轴：刀锋》国服已于今日开启全渠道公测。</p>
</div>

                
<div class="topicContent front_content"><p>　　由Bethesda旗下《上古卷轴V：天际》原班人马打造、盛趣游戏代理发行的《上古卷轴：刀锋》国服，已于今日（3月22日）开始全渠道公测，游戏上线苹果商店后获得App Store Today主打推荐。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width: 480px;" src="https://img01.vgtime.com/game/cover/2021/03/22/210322140551813_u54.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>《上古卷轴：刀锋》公测首日获App Store Today主打推荐</figcaption></figure></div><p><span style="letter-spacing: 1.3px;">　　</span><span style="background-color: initial; letter-spacing: 1.3px;">《上古卷轴：刀锋》延续了《上古卷轴》系列的基因及核心特色，曾荣获2019年 Google 年度最具创新力游戏、2019年苹果年度趋势推荐游戏等奖项，多次在海外多个国家的App Store获得精品推荐。</span><br></p><p>　　本作自带城建系统，不用打mod就可以打造属于自己的小镇。随着城镇的重建和升级，玩家也将解锁新的剧情和任务，并能够前往好友的城镇参观和互动。这一设定既满足了当下年轻用户对于模拟经营的玩法的需求，也让老玩家得到了全新的游戏体验。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/22/210322140605177_u54.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>《上古卷轴：刀锋》中的城镇建设系统</figcaption></figure></div><p><span style="letter-spacing: 1.3px;">　　</span><span style="background-color: initial; letter-spacing: 1.3px;">虽然是一款手机游戏，但本作仍拥有着高水准的画面表现效果和音乐音效，游戏对于整体风格和细节设定都极为考究。在音乐音效方面，《上古卷轴：刀锋》拥有特色鲜明的主题音乐和真实感十足的音效，曾获得第十届好莱坞音乐传媒手游类最佳原创配乐奖。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/22/210322140620680_u54.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>《上古卷轴：刀锋》实机游戏画面效果</figcaption></figure></div><p><span style="letter-spacing: 1.3px;">　　</span><span style="background-color: initial; letter-spacing: 1.3px;">为了符合手游用户的操作习惯，游戏结合移动端特点进行了操作上的创新和优化，提供了双手手持、单指操作、单手手持等多种操作模式，玩家可以自由进行横屏竖屏的切换。目前来看，这种可以适应多种游戏场景的操作方式的产品，在目前市面上并不多见，而这样的设计思路，也将《上古卷轴：刀锋》作为硬核RPG的一面展露了出来。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/22/210322140635402_u54.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>多种操作模式进行游戏</figcaption></figure></div><p><span style="letter-spacing: 1.3px;">　　</span><span style="background-color: initial; letter-spacing: 1.3px;">近年来，随着移动游戏市场的蓬勃发展，手机游戏的受众群体不断扩张，不少主机、单机类游戏IP开始尝试向手游平台发展。和手游发展初期时的情况不同，如今的手游用户对于玩法、体验以及画面品质等方面要求更高，尤其是对IP手游而言，玩家在挑选游戏时，除了游戏本身的内容品质，还会考虑世界观、核心内容是否足够还原原作，这对厂商提出了更高的要求。</span><br></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width: 480px;" src="https://img01.vgtime.com/game/cover/2021/03/22/210322140652896_u54.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>《上古卷轴：刀锋》正式上线</figcaption></figure></div><p><span style="letter-spacing: 1.3px;">　　</span><span style="background-color: initial; letter-spacing: 1.3px;">作为Bethesda旗下的金字招牌游戏，《上古卷轴》系列自1994年推出以来一直受到全世界众多玩家的喜爱。凭借极大的自由度、丰富的世界观设定，该系列不仅是欧美游戏销量榜单的常客，在国内也颇具人气。2011年发售的系列第五部作品《上古卷轴V：天际》被不少玩家奉为神作。在百度上搜索“上古卷轴”共有98,700,000个搜索结果，“膝盖中箭”、“睁眼坐马车”等游戏梗也经常被引用于国内各大社交平台以及B站UP主的视频当中，在玩家中拥有相当高的知名度。</span><br></p><p>　　《上古卷轴：刀锋》展现了RPG手游的多样性以及欧美厂商对于IP改编手游的开发思路，背后也体现出拥有大量国外游戏发行经验的盛趣游戏的综合发行实力。本作是盛趣游戏继《辐射：避难所Online》之后引进的第二款Bethesda旗下IP改编手游，既承担着将《上古卷轴》这一知名IP在中国市场的开拓，也是盛趣游戏在硬核RPG手游细分领域的一次全新尝试。</p><p>　　据官方介绍，盛趣游戏对本作进行了较为细腻的本地化工作，每一项功能都经过多次内部测试，并针对国内移动网络环境及玩家习惯进行了优化。为了给游戏的国服版本提前预热，盛趣游戏还在去年10月上线了《上古卷轴：刀锋》与《辐射：避难所Online》的联动活动，促成了两个IP的首次合作，也拉升了玩家的期待值。 随着《上古卷轴：刀锋》国服的正式上线，《辐射：避难所Online》也将开启与《上古卷轴：刀锋》的第二期联动。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/22/210322140709473_u54.png" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>《辐射：避难所Online》开启《上古卷轴：刀锋》第二期联动内容</figcaption></figure></div><p><br></p></div>
