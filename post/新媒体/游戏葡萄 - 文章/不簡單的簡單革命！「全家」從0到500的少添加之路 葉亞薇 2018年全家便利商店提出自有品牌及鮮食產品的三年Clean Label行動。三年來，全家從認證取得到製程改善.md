Label三年計畫，並與國內具公信力的潔淨標示評鑑標準的慈悅國際共同合作，從自家品牌FamilyMart Collection 為先鋒，每一年創下的輝煌戰績，連全家自己都意想不到。</p> <p>「當食品開始重視純粹成分，而通路能做的事可以更多，尤其是號召供應鏈上中下游商品一起動起來，兼具商業利益與健康把關，讓產業與市場、消費者共好。」全家商品本部本部長黃君毅表示，Clean Label 三年計畫的逐年攻略，是一場「不簡單的簡單革命」。</p> <p><img src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" class="lazy" </div> <p>2018年FamilyMart Collection自有品牌商品打頭陣，以國產品項 100% 通過評鑑為目標，商品包含飲料、零食、冷凍調理食品；2019年啟動鮮食的意識改革，將微波食品、飯糰、麵包架上熱銷品逐漸符合。2020年則達成商品通過評鑑數最大化，包括700種原物料符合潔淨標準、500種跨商品與鮮食的認證商品。黃君毅說，全家已能運用700種潔淨原物料來研發新品，消費者一進全家，很容易就能取得「小葉子狀的Clean Label Label認證廠商成長五倍，夥伴高度認同，結結實實為全家的努力打了一劑強心針。</p> <h3>認證廠商5倍成長，啟動供應鏈良性循環</h3> <p>Clean Label的蝴蝶效應，從2018年全家號召供應鏈上中下游成立國內第一個Clean Label 產業聯盟開始，邀集食品產業70家廠商共同參與，包括：佳格食品、聯華食品、光泉、大成長城、南僑集團、可果美、黑松、雀巢、泰山、乖乖等指標大廠，並定期舉辦Clean Label 大型趨勢論壇、廠商說明交流會，提供資訊交流。黃君毅強調，廠商之間口碑交換「你可以，那我也可以」的良性競爭，加上「做Clean Label會上癮」品質差異化的榮譽，越來越多廠商願意加入，使得替代原料用量增加，成本再降低，讓產業生態圈自然良性循環，彼此共榮。</p> <p>當然，要說服別人，自己得更有實力。為了推動Clean Label，全家調整內部生產制度，同步提升員工教育訓練職能。尤其因應Clean Label
