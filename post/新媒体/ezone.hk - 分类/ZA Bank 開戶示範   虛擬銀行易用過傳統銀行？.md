<p>不經不覺，虛擬銀行已登錄香港一年。虛擬銀行類似只設網上銀行的銀行，讓用戶以手機和電腦使用銀行服務，並已參加存款保障計劃，讓客戶存款有最高 50 萬港元保障。部分虛擬銀行所推出的 Debit 卡，更支援銀聯 ATM 提款功能，讓用戶更易從虛擬銀行中取得存款。</p> 
<ul> 
 <li><strong>虛擬銀行有最高 50 萬港元存款保障</strong></li> 
 <li><strong>ZA Bank 開戶需自拍身份證及面容作認證</strong></li> 
 <li><strong>配合轉數快更方便</strong></li> 
</ul> 
<p>虛擬銀行的操作跟一般傳統銀行的 E-Banking 相近，甚至比起網頁版的 E-Banking 介面更易用。以下 ezone.hk 以《ZA Bank》作示範，介紹各項虛銀操作。留意以虛擬銀行 app 開戶時，用戶需要為身份證拍照，亦要以鏡頭自拍面容作認證。</p> 
<p><strong>指紋或人面登入</strong></p> 
<p>首先點擊〈更多〉，進入〈安全設定〉內，啟用指紋或人面辨識驗證功能。只要此 app 取得系統的生物驗證授權，即可使用系統已記錄的指紋或人面辨識資料，快速完成系統登入。</p> 
<p></p><figure itemprop="associatedMedia" itemscope itemtype="http://schema.org/ImageObject"><a class="photoSwipe" data-href="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_22_600.jpg" data-photoindex="2" onclick="showPhotoSwipe($(this));return false;" href="https://ezone.ulifestyle.com.hk/article/undefined"><img class="img-responsive" src="https://ezone.ulifestyle.com.hk/images/loader.gif" data-src="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_22_600.jpg" style="margin-left:auto;margin-right:auto" referrerpolicy="no-referrer"></a><figcaption></figcaption></figure><p></p> 
<p><strong>宜登記轉數快</strong></p> 
<p>轉數快（FPS）對於虛擬銀行來說尤其重要，故此記得進入〈更多〉→〈轉數快和轉賬設定〉→〈我的轉數快〉中，以手機號碼、電郵或 FPS ID 登記轉數快功能。</p> 
<p></p><figure itemprop="associatedMedia" itemscope itemtype="http://schema.org/ImageObject"><a class="photoSwipe" data-href="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_23_600.jpg" data-photoindex="3" onclick="showPhotoSwipe($(this));return false;" href="https://ezone.ulifestyle.com.hk/article/undefined"><img class="img-responsive" src="https://ezone.ulifestyle.com.hk/images/loader.gif" data-src="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_23_600.jpg" style="margin-left:auto;margin-right:auto" referrerpolicy="no-referrer"></a><figcaption></figcaption></figure><p></p> 
<p><strong>設定轉賬限制</strong></p> 
<p>為了提高轉賬安全，建議用戶在〈更多〉→〈轉數快和轉賬設定〉→〈設定限額〉中，設定同名賬戶轉賬限額、第三方賬戶轉賬限額和小額轉賬限額的數值。</p> 
<p></p><figure itemprop="associatedMedia" itemscope itemtype="http://schema.org/ImageObject"><a class="photoSwipe" data-href="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_24_600.jpg" data-photoindex="4" onclick="showPhotoSwipe($(this));return false;" href="https://ezone.ulifestyle.com.hk/article/undefined"><img class="img-responsive" src="https://ezone.ulifestyle.com.hk/images/loader.gif" data-src="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_24_600.jpg" style="margin-left:auto;margin-right:auto" referrerpolicy="no-referrer"></a><figcaption></figcaption></figure><p></p> 
<p><strong>存入資金至虛銀</strong></p> 
<p>從〈賬戶〉點擊黃色按鍵往下拉，其後可選擇以轉數快或賬戶號碼存入金額。轉數快故然是最簡易快捷和免費的存入資金方式，但留意轉數快只支援港幣和人民幣轉賬，無法支援美元轉賬。故此若要存入美元，必須以賬戶號碼轉賬。</p> 
<p></p><figure itemprop="associatedMedia" itemscope itemtype="http://schema.org/ImageObject"><a class="photoSwipe" data-href="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_25_600.jpg" data-photoindex="5" onclick="showPhotoSwipe($(this));return false;" href="https://ezone.ulifestyle.com.hk/article/undefined"><img class="img-responsive" src="https://ezone.ulifestyle.com.hk/images/loader.gif" data-src="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_25_600.jpg" style="margin-left:auto;margin-right:auto" referrerpolicy="no-referrer"></a><figcaption></figcaption></figure><p></p> 
<p>【熱門話題】<a target="_blank" href="https://ezone.ulifestyle.com.hk/article/2883083/%E8%99%9B%E6%93%AC%E9%8A%80%E8%A1%8C%20Mox%20%E5%AD%98%E6%AC%BE%E7%B8%BD%E9%A1%8D%E5%A2%9E%E8%87%B3%2051%20%E5%84%84%E6%B8%AF%E5%85%83">虛擬銀行 Mox 存款總額增至 51 億港元</a></p> 
<p><strong>綁定銀行賬戶</strong></p> 
<p>虛擬銀行內的存款，必須轉賬至傳統銀行戶口，才可經 ATM 或分行提取現金。經常在虛擬銀行與傳統銀行戶口之間轉賬的用戶，可考慮藉〈入錢易存入〉功能，將虛擬銀行跟傳統銀行綁定。</p> 
<p></p><figure itemprop="associatedMedia" itemscope itemtype="http://schema.org/ImageObject"><a class="photoSwipe" data-href="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_26_600.jpg" data-photoindex="6" onclick="showPhotoSwipe($(this));return false;" href="https://ezone.ulifestyle.com.hk/article/undefined"><img class="img-responsive" src="https://ezone.ulifestyle.com.hk/images/loader.gif" data-src="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_26_600.jpg" style="margin-left:auto;margin-right:auto" referrerpolicy="no-referrer"></a><figcaption></figcaption></figure><p></p> 
<p><strong>記錄收款人資料</strong></p> 
<p>從〈賬戶〉點擊黃色按鍵往上拉，即可將虛擬銀行內的存款，以電郵、手機號碼、FPS ID 等轉數快形式，或以銀行賬戶形式號碼轉賬至其他戶口。用戶可在此記錄收款人資料，方便日後轉賬時使用。</p> 
<p></p><figure itemprop="associatedMedia" itemscope itemtype="http://schema.org/ImageObject"><a class="photoSwipe" data-href="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_27_600.jpg" data-photoindex="7" onclick="showPhotoSwipe($(this));return false;" href="https://ezone.ulifestyle.com.hk/article/undefined"><img class="img-responsive" src="https://ezone.ulifestyle.com.hk/images/loader.gif" data-src="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_27_600.jpg" style="margin-left:auto;margin-right:auto" referrerpolicy="no-referrer"></a><figcaption></figcaption></figure><p></p> 
<p><strong>檢視定期息口</strong></p> 
<p>然後點擊〈定存〉，可檢視不同貨幣和存款期的定期存款，所得的存款息率，並選用合適的定期存款服務。</p> 
<p></p><figure itemprop="associatedMedia" itemscope itemtype="http://schema.org/ImageObject"><a class="photoSwipe" data-href="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_28_600.jpg" data-photoindex="8" onclick="showPhotoSwipe($(this));return false;" href="https://ezone.ulifestyle.com.hk/article/undefined"><img class="img-responsive" src="https://ezone.ulifestyle.com.hk/images/loader.gif" data-src="https://resource01-proxy.ulifestyle.com.hk/res/v3/image/content/2910000/2911433/20210318_TONG_28_600.jpg" style="margin-left:auto;margin-right:auto" referrerpolicy="no-referrer"></a><figcaption></figcaption></figure><p></p> 
<p><strong>快速申請貸款</strong></p> 
<p>最後點擊〈貸款〉即可申請貸款，只要輸入月薪、申請金額、還款期等資料，便可看到預計每月還款額和實際年利率，往後還款時便更有預算。</p>
