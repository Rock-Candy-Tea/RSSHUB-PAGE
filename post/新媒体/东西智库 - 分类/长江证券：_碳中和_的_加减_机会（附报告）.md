<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">为实现</span>“碳达峰”、“碳中和”的目标，几乎所有行业均面临“加”和“减”的变化。“加”是指“碳中和”任务下的新增需求，而“减”则来自于供给收缩及结构调整。但不同行业的侧重点和受益程度有所不同，整体可以分成三类：</span></p>
<p style="text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; color: #c0504d; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">一、</span></span></strong><strong><span style="font-family: 宋体; line-height: 150%; color: #c0504d; letter-spacing: 0; font-size: 16px; background: #ffffff;">“加”上有强新增需求的行业，包括电新、新材料、环保等。</span></strong></p>
<p style="margin-left: 48px; text-autospace: ideograph-numeric; line-height: 150%;"><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"> </span></p>
<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">1）电新：</span></strong></p>
<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">①</span></strong><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">行业逻辑：</span></span></strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">“加”：</span><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">经测算，欧洲</span>2030年新能源车销量底线850-900万辆，光伏、风电装机复合增速17%、7%。美国2025年新能源车渗透率达到20-25%。国内“十四五”期间光伏、风电年均装机70-90GW、40-50GW左右。</span></p>
<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">②</span></strong><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">代表标的：</span></span></strong></p>
<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"> </span></p>
<p style="text-align: center;"><img class="rich_pages js_insertlocalimg aligncenter" title="“碳中和”的“加减”机会" src="http://www.dx2025.com/wp-content/uploads/2021/03/7-1616377659.png" alt="“碳中和”的“加减”机会" referrerpolicy="no-referrer"></p>
<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">2）新材料：</span></strong></p>
<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">①</span></strong><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">行业逻辑：</span></span></strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">“加”：</span><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">碳纤维行业，受益于海上风电项目快速发展，需求有望快速增加，国产化龙头逐步崛起将对进口材料进行替代。生物基可降解塑料将替代传统的石化原料制备的塑料产品，降低垃圾焚烧过程二氧化碳排放量，行业未来需求空间广阔。</span></span></p>
<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">②</span></strong><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">代表标的：</span></span></strong></p>

<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">3）环保：</span></strong></p>
<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">①</span></strong><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">行业逻辑：</span></span></strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">“加”：</span><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">其一，提升项目盈利性，通过</span>CCER碳交易获取增量收入，体现在垃圾焚烧和秸秆发电领域。其二，加速行业需求的释放，主要体现在环卫装备电动化趋势加强及短期内汽车尾气治理需求迫切度提升。</span></p>
<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">②</span></strong><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">代表标的：</span></span></strong></p>

<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">1</span></strong><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">）电力：</span></span></strong></p>
<p style="text-indent: 32px; text-autospace: ideograph-numeric; line-height: 150%;"><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">①</span></strong><strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">行业逻辑：</span></span></strong><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;">“加”：</span><span style="font-family: 宋体; line-height: 150%; letter-spacing: 0; font-size: 16px; background: #ffffff;"><span style="font-family: 宋体;">提升对风电光伏等非化石能源消费，带动整个能源和电力结构的转型升级。新能源行业的价值将愈发得到凸显，建议关注新能源发电领域投资机会，其中包括具备先发优势的新能源运营商，以及积极转型新能源的火电标的。</span></span></p>
