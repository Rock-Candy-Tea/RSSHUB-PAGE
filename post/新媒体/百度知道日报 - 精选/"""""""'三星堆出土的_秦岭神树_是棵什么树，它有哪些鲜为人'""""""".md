三星堆出土的“秦岭神树”是棵什么树，它有哪些鲜为人知的谜团？
</h1>

<div id="daily-cont" class="cont">
<p>三星堆文物来“新人”了！</p><p>早在上个世纪时，三星堆就出土了很多文物，其中包括许多人像作品，由于这些人像怪异，造型非常奇特，在国际上引起了很大的轰动，甚至有人认为这些是外星人的后代。</p><p><img src="https://iknow-pic.cdn.bcebos.com/562c11dfa9ec8a1397274480e703918fa0ecc095?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>此次科学家们在三星堆遗址祭祀坑进行考古发掘中，出土了大量文物，比如：黄金面具残片，虽然只有半张面具，但是这张面具非常厚，不需要任何支撑就能够立起，据相关负责人介绍，根据这半张面具推测，完整的面具重量应该超过500克。</p><p><br></p><p>除了黄金面具之外，科学家们又发现了大量的象牙、巨青铜面具以及青铜神树等共500多件重要的文物。由于《盗墓笔记》在网络上的盛行，人们认为三星堆出土的“青铜神树”就是“秦岭神树”的原型，借此机会就好好聊一聊，在三星堆出土的“青铜神树”究竟是棵什么树，围绕着它又有哪些谜团？</p><p><br></p><p style="box-sizing: inherit; ; ; font-size: 22px; position: relative; line-height: 30px; color: rgb(51, 51, 51); font-family: Arial, "Hiragino Sans GB", STHeiti, "Helvetica Neue", Helvetica, "Microsoft Yahei", "WenQuanYi Micro Hei", sans-serif; white-space: normal;">青铜神树</p><p>虽然此次科学家们在三星堆遗址中发掘到了青铜神树，但其实早在1986年8月科学家们就从三星堆遗址中出土了6件树状的文物，其中有一件保存最为完整。从外观上看，这棵青铜神树高395厘米，其中树顶处有残缺，该树是由基座和树干组成。</p><p><img src="https://iknow-pic.cdn.bcebos.com/622762d0f703918f58ec2eaa413d269759eec495?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>从其中一个基座保存较好的青铜神树的造型来看，在神树的基座上有三个像山峰一样的形状，旁边各有一个小人，但小人的双臂有残缺。</p><p><img src="https://iknow-pic.cdn.bcebos.com/58ee3d6d55fbb2fb43174b725f4a20a44623dc95?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>虽然小人的双臂有残缺，但是科学家根据三星堆出土的其他文物猜测，它们应该是双手呈怀抱状，怀抱的东西可能是祭祀用品。不过要说明的是，在较为完整的青铜神树上并没有发现这些小人。</p><p><img src="https://iknow-pic.cdn.bcebos.com/6c224f4a20a446230cf6b6028822720e0cf3d795?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>基座底部有一条自上而下盘旋的龙，龙头朝下，前爪匍匐在树的底座上，看起来非常怪异。</p><p><img src="https://iknow-pic.cdn.bcebos.com/7acb0a46f21fbe096462d0747b600c338744ad95?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>神树一共有三层，每层有三个树枝，每个树枝又一分为二，其中一个树枝上有一个类似于果实一样的基座，基座上站着一只鸟，神树之上一共站立着9只鸟。</p><p><img src="https://iknow-pic.cdn.bcebos.com/6a63f6246b600c339445cda80a4c510fd9f9a195?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p style="box-sizing: inherit; ; ; font-size: 22px; position: relative; line-height: 30px; color: rgb(51, 51, 51); font-family: Arial, "Hiragino Sans GB", STHeiti, "Helvetica Neue", Helvetica, "Microsoft Yahei", "WenQuanYi Micro Hei", sans-serif; white-space: normal;">青铜神树的作用</p><p>9只鸟站在树上的结构，让有些学者认为这是《山海经》所描写的扶桑树：“汤谷上有扶桑，十日所浴，居水中。九日居下枝，一日居上枝。”</p><p>其中9只鸟指的就是九日，在我国文化中，自古就有“太阳是被鸟托起的”传说，比如：金乌传说。也就是说在扶桑树上一共有10个太阳，其中9个太阳在树上休息，另外一个则飞到天空中。而这10个太阳是“一日方至，一日方出，皆载于乌。”也就是说一个太阳回来了，另外一个才出去，始终有9只太阳在神树上，一轮在天上。</p><p>除了扶桑木的说法之外，还有一些学者认为这可能是《山海经》中所描写的建木：“有木，其状如牛，引之有皮，若缨、黄蛇。其叶如罗，其实如欒，其木若蓲，其名曰建木。”原因是因为 在青铜神树上的枝叶、飞禽、走兽以及悬龙、神铃、果实等都符合《山海经》中描写的建木形象。</p><p><img src="https://iknow-pic.cdn.bcebos.com/6159252dd42a2834299a256b4bb5c9ea15cebf95?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>建木的主要作用是沟通天地，连接人与神，传说伏羲、黄帝等上古神话中的任务就是通过这一梯子来往于人间。</p><p><img src="https://iknow-pic.cdn.bcebos.com/2934349b033b5bb5e9eb339e26d3d539b600bc95?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>另外，在青铜神树出土的2号坑中，科学家们发现了这里有大量的鱼和鸟的形象，推测为这应该是古蜀鱼凫时期的文物。</p><p><img src="https://iknow-pic.cdn.bcebos.com/e850352ac65c10387842dda2a2119313b07e8995?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>李白曾经在《蜀道难》中写道：蚕丛及鱼凫，开国何茫然，尔来四万八千岁，不与秦塞通人烟。其中蚕丛是古蜀国开国国王，教会了人类养蚕。鱼凫是第三位国王，鱼凫是一种鱼老鸹，善于捕鱼，鱼凫王教会了人类捕鱼技巧。多说一句，接替鱼凫王位置的是杜宇，号曰望帝，也是唐诗中“望帝君心托杜鹃”中的“望帝”。</p><p><img src="https://iknow-pic.cdn.bcebos.com/5243fbf2b21193139f7fecd075380cd791238d95?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>科学家认为当地的人为了纪念鱼凫，才设计了大量的鱼和鸟的形象，而站在青铜神树上的鸟，就和鱼老鸹很相似。</p><p><img src="https://iknow-pic.cdn.bcebos.com/ac345982b2b7d0a236f517fddbef76094b369a95?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p style="box-sizing: inherit; ; ; font-size: 22px; position: relative; line-height: 30px; color: rgb(51, 51, 51); font-family: Arial, "Hiragino Sans GB", STHeiti, "Helvetica Neue", Helvetica, "Microsoft Yahei", "WenQuanYi Micro Hei", sans-serif; white-space: normal;">神秘消失的三星堆文明</p><p>除了青铜神树之外，三星堆遗址中还出土了大量的文物器皿以及各种青铜器，造型虽然奇特，但是工艺之高并不属于夏商时期。而且在三星堆遗址中还出现了大量的象牙和海贝，然而当地并没有大象和海洋，这意味着这些东西很可能是与外来文明交换而来的。</p><p><img src="https://iknow-pic.cdn.bcebos.com/caef76094b36acaf6edc859c6cd98d1001e99c95?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>既然三星堆文明已经高度发达，那为什么三星堆遗址内的文物中有许多火烧的痕迹，并且一夜之间消失了呢？</p><p>科学家们在距离三星堆遗址50公里远的地方，又发现了一个遗址被命名为“金沙遗址”，从金沙遗址中出土的文物来看，这里的文物和三星堆遗址的文物高度相似，具有明显的传承性。只不过在金沙遗址中出土的祭祀人物形象和三星堆遗址中出土的祭祀人物形象有些许不同，科学家们认为这可能是当地发生了一次内乱，导致权力转移。</p><p><img src="https://iknow-pic.cdn.bcebos.com/3801213fb80e7bec1de610813f2eb9389b506b95?x-bce-process=image/resize,m_lfit,w_450,h_600,limit_1/quality,q_85" referrerpolicy="no-referrer"></p><p>在这场内乱中，大量的青铜器被破坏，还有许多被战火焚烧，这才导致一些青铜器上留下了火烧的痕迹。内乱结束之后，权力中心也移动到了其他地区，导致三星堆文明被遗落，又被岁月埋入到了地底下。</p><p>虽然围绕着三星堆文明还有许多未解之谜，比如：古蜀国的历史，古蜀国文化等，但随着考古学家们的考古和发掘，相信三星堆的秘密会再次呈现在我们的面前。</p><p><br></p>
