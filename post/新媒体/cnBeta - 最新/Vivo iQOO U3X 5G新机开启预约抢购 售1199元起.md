<strong>Vivo 刚刚在电商网站上开启了iQOO U3X 5G 智能机的预约抢购，且初期仅在中国市场销售。不过从外形设计和配置规格上来看，你会发现它和 Vivo Y31s 有许多相似之处。</strong>比如 FHD+ 分辨率 @ 90Hz 高刷新率的 6.58 英寸 LCD 屏，高通骁龙 480 芯片组，前置 8MP @ f/2.0 单摄、后置 13MP 主摄 + 2MP 深度传感器，以及支持 18W 快充的 5000 mAh 大电池。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0322/2e1660db6a2f667.jpg" referrerpolicy="no-referrer"></p><p>作为该公司“游戏智能机”的一员，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://vivo.jd.com/" target="_blank">vivo</a> 并未给 iQOO U3X 5G 预装 Funtouch OS 10.5，而是启用了最新的 Origin OS 。</p><p><img src="https://static.cnbetacdn.com/article/2021/0322/6002cb30d737ea3.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>Origin OS 的特色功能包括 Multi-Turbo 5.0，有助于<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>硬件的长时间稳定发挥，让游戏体验变得更加流畅。</p><p><img src="https://static.cnbetacdn.com/article/2021/0322/831c140d35a4486.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>此外 iQOO U3X 5G 新机具有独特的“内存融合”技术，如果 8GB RAM 不足，还可调用 3GB 系统存储来转储一些数据。</p><p><img src="https://static.cnbetacdn.com/article/2021/0322/9b864bc3b28c905.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>这意味着在特定场景下，你甚至可将它当做一部具有 11GB RAM 的设备来使用。</p><p><img src="https://static.cnbetacdn.com/article/2021/0322/fb3ae8385cc25de.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>言归正传，Vivo 为 iQOO U3X 5G 提供了三档配置选项，分别是 4B RAM + 128GB ROM、6GB RAM + 64GB ROM、以及 8GB + 128GB ROM 。</p><p><img src="https://static.cnbetacdn.com/article/2021/0322/af40e6c826aba67.jpg" referrerpolicy="no-referrer"></p><p>配色方面，iQOO U3X 5G 可选“雅灰”或“幻蓝”，目前三档配置在<a data-link="1" href="https://c.duomai.com/track.php?k=nJ9QWa1VmJxYTPklWYmYDO5IDNy0DZp9VZ0l2cmYiJt92YuQmauc3d3ZkMlYkMlE0MlMHc0RHa9Q" target="_blank">京东</a>和官网上的售价为 <a href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://item.jd.com/10027681737879.html" target="_self">1199 ~ 1499</a> 元。</p>
