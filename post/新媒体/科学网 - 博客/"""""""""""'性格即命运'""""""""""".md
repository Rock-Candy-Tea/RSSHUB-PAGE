<p><span style="font-family: 宋体, SimSun; font-size: 20px;"></span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;"><span style="font-family: 宋体, SimSun; color: #333333; background: white;">《辞海》解释:性格是人对现实的态度和行为方式中较稳定的个性心理特征。它是个性的核心部分，最能表现个体差异。大体包括:</span></span></p><p style="margin-top:0;margin-right:0;margin-bottom:15px;margin-left: 0;text-indent:32px;line-height:24px;background:white;zoom: 1"><span style="font-family: 宋体, SimSun; color: #333333; font-size: 20px;">(1)对现实和自己的态度的特征，如诚实或虚伪、谦逊或骄傲等。</span></p><p style="margin-top:0;margin-right:0;margin-bottom:15px;margin-left: 0;text-indent:32px;line-height:24px;background:white;zoom: 1"><span style="font-family: 宋体, SimSun; color: #333333; font-size: 20px;">(2)意志特征，如勇敢或怯懦、果断或优柔寡断等。</span></p><p style="margin-top:0;margin-right:0;margin-bottom:15px;margin-left: 0;text-indent:32px;line-height:24px;background:white;zoom: 1"><span style="font-family: 宋体, SimSun; color: #333333; font-size: 20px;">(3)情绪特征，如热情或冷漠、开朗或抑郁等。</span></p><p style="margin-top:0;margin-right:0;margin-bottom:15px;margin-left: 0;text-indent:32px;line-height:24px;background:white;zoom: 1"><span style="font-family: 宋体, SimSun; color: #333333; font-size: 20px;">(4)理智特征。如思维敏捷、深刻、逻辑性强或思维迟缓、浅薄、没有逻辑性等。</span></p><p style="zoom: 1"><span style="font-family: 宋体, SimSun; font-size: 20px;"><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">性格的形成取决于基因遗传因素，成长期发育因素以及社会环境的影响因素。性格是人格的表现。体现在开放性，亲和性，勤勉性，情绪性，外向性等方面。</span><span style="font-size: 20px; font-family: 宋体; color: #333333;">性格与兴趣，价值观有着密切的联系。</span></span></p><p style="zoom: 1"><span style="text-indent: 28px; background-color: #FFFFFF; color: #333333; font-size: 20px; font-family: 宋体, SimSun;">古语云:"积行成习，积习成性，积性成命" 。</span><span style="font-size: 20px; font-family: 宋体; color: #333333;"><span style="color: #333333; font-family: arial, sans-serif; font-size: 14px; text-indent: 28px; background-color: #FFFFFF;"></span>性格的定型一般在16-20岁期间。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;">作为老师，面对的是学生。现在网上关于师生关系的话题是剪不断理还乱，站在各自立场，各说各话，似乎都是自己对，对方错。但都忽略了性格因素。其实</span><span style="font-family: 宋体, SimSun; font-size: 20px;">双方的性格对彼此影响很大。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;">我的观点：1.学生都是成年人，性格已经定型，但可以微调。2.教育不是万能的，没有教育是万万不能的。3.相互尊重，因材施教。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;">学生是成年人，有人说“没有教不会的学生，只有不会教的老师”，对此我不敢苟同。此话对中学以前年龄段的学生有效，但对于成年人不一定有效。教育不是万能的。但导师的人品，责任心和能力对大学生，研究生的影响却是深远的。我不敢打包票每一个学生都做到优秀，但尽力就好。成年人的价值观，世界观已经形成，想要改变很难，但不是不可作为。必须尊重学生，平等对待，了解每一个学生。对某个学生的做法不一定适合所有学生</span><span style="font-family: 宋体, SimSun; font-size: 20px;">，要因材施教</span><span style="font-family: 宋体, SimSun; font-size: 20px;">。个性化，差异化培养才是处理师生关系的准则。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;">当师生出现沟通问题时，多为导师强势，学生心理抵抗。导师</span><span style="font-family: 宋体, SimSun; font-size: 20px;">原则性和灵活性的度要把握好。对抗性的性格容易爆发冲突，平衡性的性格使人相处融洽。</span><span style="font-family: 宋体, SimSun; font-size: 20px;">不妨从各自角度反思一下，</span><span style="font-family: 宋体, SimSun; font-size: 20px;">学会心平气和，</span><span style="font-family: 宋体, SimSun; font-size: 20px;">换个方式或许收到奇效。</span></p>                    <br><br>
