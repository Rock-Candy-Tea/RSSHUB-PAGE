<div><strong><em>城农获悉——无人驾驶电动拖拉机初创公司Monarch刚刚完成了2000万美元A轮融资。本轮融资由日本汽车零部件制造商Musashi Seimitsu Industry Co、农业设备制造商凯斯纽荷兰工业和印度拖拉机和机具公司VST Tillers领投，其他投资者包括总部位于旧金山的At One Ventures和纽约专注于可持续发展的投资集团MUUS。</em></strong></div><div><br></div><div><img src="https://cdn.goodq.top/caches/e1c907fda5238d844772c965384e9f54/aHR0cDovL3d3dy5ncm93aW5jaXR5LmNvbS9xZnktY29udGVudC91cGxvYWRzLzIwMjEvMDMvNzYyMTYzZTc5N2NmYzE3MDhiNmRkYTFhMDM4OGEwMmIuanBn.jpg" referrerpolicy="no-referrer"><br></div><div><br></div><div>无人驾驶拖拉机似乎是个具有未来主义的想法，但如果想把它卖给农民，那就请不要叫它机器人。这是Monarch联合创始人兼首席执行官Praveen Penmetsa的心得。“五年前，我们发明了一款外观非常前卫的机器。然而，我们了解到的是，农民们对拖拉机有一种情感，并且喜欢拖拉机现在的样子，所以我们不得不扔掉未来主义的设计。现在，我们的无人驾驶拖拉机还是拖拉机；农民可以使用他们现有的工具，并能看到Monarch拖拉机的价值。”<br><br>Penmetsa将本轮所获投资归因于人们对电动化的兴趣日益浓厚，尤其在汽车领域的势头一直在增强。Monarch拖拉机除了自主驾驶以外，还是一个数据枢纽，<span style="background-color: initial; color: rgb(0, 0, 0);">能实时提醒农田里发生了什么，从而节省了用户去农田勘察的时间。另外还有</span><span style="background-color: initial; color: rgb(0, 0, 0);">“司机可选”功能，允许操作员管理五到七辆拖拉机车队。</span></div><div><br>这笔新资金将用于建造一条微型装配线。Monarch已经从一些最大的葡萄园园主那里收到了预购定金。第一批投放到用户手中的拖拉机称为“试点系列”，将运往加利福尼亚州、俄勒冈州和华盛顿州的农场。在此之后，Monarch将推出一款更接近大型拖拉机的型号。<br><br>目前，Monarch无人驾驶拖拉机的零售价为5万美元，与当今市场上的传统拖拉机相比，价格相当有竞争力，并包含了数据跟踪软件。对于农业科技公司来说，农民们不愿花大价钱购买尖端技术，所以平价一直是一个主要关注点。农民客户不愿意花8万到10万美元购买某种未来主义的拖拉机，所以这意味着“我们的拖拉机必须坚固耐用。我们这里没有两年的投资回报率之说，根据我们看到的情况，它必须在8到18个月内提供投资回报率。”Praveen Penmetsa分享了他的真知灼见。</div><div><br></div><div>要把价格降到负担得起的范围内，就需要专注于农民客户最想要的东西，而不是花哨玩意。为此，Monarch创新了电气架构，允许整辆拖拉机由一台电机供电，而这是实现其具有竞争力价格的另一个因素。为了出售无人驾驶拖拉机，Monarch还选择了试点农场。许多客户需要看到实物的真实表现才能相信，特别是技术创新方面。<br><br>尽管如此，一些农民客户对投资纯电动拖拉机持观望态度，许多人都在等待自己的邻居先充当新科技的“小白鼠”。由于能力有限，Monarch目前以拥有使用大型拖拉机车队的客户优先。出于可持续性或成本效益的原因，这些农场有兴趣将30%到50%的拖拉机车队改成纯电动版。<br><br>农场劳动力问题也在鼓励农民考虑无人驾驶拖拉机方面发挥了作用。拖拉机操作员这种专业工种在农业社区可能供不应求，特别是在农忙旺季；然而，新冠大流行进一步加剧了农场劳动力的挑战。甚至自然不可预知事件，也加速了对无人驾驶拖拉机的需求；如在去年美国发生的野火灾难，许多农民想要进行田间作业，但空气质量实在太差了，<span style="background-color: initial; color: rgb(0, 0, 0);">给农机设备操作员带来了危险。即使在一般情况下，人工仍然会暴露在危险化学品或机械事故危险中。</span></div><div><span style="background-color: initial; color: rgb(0, 0, 0);"><br></span></div><div><span style="background-color: initial; color: rgb(0, 0, 0);">那么现在，如果农民对田间作业感到不舒服，Monarch就<span style="background-color: initial; color: rgb(0, 0, 0);">已经</span>为他们准备好了解决方案。当然无人驾驶拖拉机的大范围普及还要依赖于技术要更加成熟，以及<span style="background-color: initial; color: rgb(0, 0, 0);">无人驾驶农用机械的监督政策与<span style="background-color: initial; color: rgb(0, 0, 0);"><span style="background-color: initial; color: rgb(0, 0, 0);">法规的建立。尤其，在<span style="background-color: initial; color: rgb(0, 0, 0);"><span style="background-color: initial; color: rgb(0, 0, 0);">政策<span style="background-color: initial; color: rgb(0, 0, 0);"><span style="background-color: initial; color: rgb(0, 0, 0);">法规方面，业界还要再等一段时间。</span></span></span></span></span></span></span></span></div>
