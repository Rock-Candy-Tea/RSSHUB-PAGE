<div><strong><em>城农获悉——澳大利亚农业科技初创公司RapidAIM正在开发一种成本效益更高的农作物病虫害监测系统。该公司已经从墨尔本的风险投资公司Tenacious Ventures获得百万级别的融资，但这笔融资金额没有被披露。在此之前，RapidAIM从<strong><em>CSIRO下属的风险投资基金</em></strong>Main Sequence Ventures获得了种子资金。</em></strong><br><br><img src="https://cdn.goodq.top/caches/e1c907fda5238d844772c965384e9f54/aHR0cDovL3d3dy5ncm93aW5jaXR5LmNvbS9xZnktY29udGVudC91cGxvYWRzLzIwMjEvMDMvM2QzODcwOTQ0MThlNDA0ZWFlOWUyYzU1NDJkNDkzZTUuanBn.jpg" referrerpolicy="no-referrer"><br><br>这家总部位于布里斯班的农业科技初创公司是于2018年末创立的。三位联合创始人曾是澳大利亚国家科学机构CSIRO的科学家，研究如何利用无线网络技术监控大面积的农作物害虫，如果蝇，以解决和摆脱对作物虫害的猜测。于是，RapidAIM成立，从CSIRO那里获得了技术许可，并筹集了一轮种子资金。<br><br>RapidAIM团队将他们在CSIRO所做的工作更上一层楼，设计和建造了装有传感器的小型移动捕虫器，让农民可以在土地的任何地方部署。
</div><div><br></div><div>RapidAI开发的这个系统与其他系统的不同之处在于它识别害虫的方式。RapidAIM的传感器不依赖于昂贵且移动性较低的相机和成像技术，而是能够记录与某些昆虫物种相关的独特行为，以便在它们落入捕虫器时识别它们。<br><br>RapidAIM联合创始人兼首席执行官Nancy Schellhorn解释说：“就像指纹可以解锁手机一样，一旦昆虫进入我们的捕虫器，它就会解锁传感器。然后，昆虫的位置会被实时的记录下来，系统使用窄带物联网将其传送到云端。我们的技术一部分是传感器，另一部分是超低功耗通信平台。”<br><br>如今用于监测虫害的大多数为手动捕虫器，都依赖于现成的引诱剂。这意味着可以捕捉到一大堆虫子，但要准确识别哪些是让农民担心的虫害，则是个很棘手的问题。当它们进入RapidAIM的捕虫器时，其传感器和算法可以辨别使哪些昆虫，以及它们是否需要被重视。<br><br>根据RapidAIM的说法，其系统的能效和低成本，让更多的种植者能负担得起，而且比基于摄像头的解决方案更具可扩展性，因为每个捕虫器仅靠两块小型锂离子电池就可以运行长达18个月。它们也很容易使用和维护，只需挂在田地或果园里，一旦发现潜在的虫害问题时可通过智能手机应用程序发出警报。<br><br>到目前为止，RapidAIM已经在澳大利亚东部和新西兰部署了大约500个传感器，为超过450平方公里的农田，170个客户提供服务。虽然目前专注于果蝇，但RapidAIM现在正致力于将其虫害检测平台应用于草地贪夜蛾和苹果蠹蛾。从长远来看，这将使其产品能够同时进入果园和农田市场。</div><div><br></div><div>RapidAIM的解决方案还可以减少化学杀虫剂的使用，帮助农民摆脱依赖“喷洒并祈祷”的农场管理方式。由于基于摄像头的虫害监测成本较高，农民无法以所需的密度部署，也就不能准确、实时地了解他们面临的虫害问题。这意味着有针对性的使用杀虫剂在很大程度上是不可能的，然而<span style="background-color: initial; color: rgb(0, 0, 0);">治理虫害时，时机决定一切。</span></div><div><span style="background-color: initial; color: rgb(0, 0, 0);"><br></span></div><div><span style="background-color: initial; color: rgb(0, 0, 0);">由于RapidAIM可以精确定位害虫，这样种植者就可以知道何时何地发生了什么，从而更明智地、有针对性地使用杀虫剂，或危害小的生物制剂。这样不但提高了效率，还降低成本和损失，对杀虫剂的使用做出改变，也对环境产生积极影响。使用</span><span style="background-color: initial; color: rgb(0, 0, 0);">RapidAIM病虫害监测系统的</span><span style="background-color: initial; color: rgb(0, 0, 0);">一些种植者已经在改变他们的喷洒方法，因为他们可以知道昆虫在一天中最活跃的时间。</span></div><div><br>目前的虫害防治很大程度上依赖于使用化学杀虫剂，往往由于种植者的“以防万一”心理，而进行大规模喷洒。最终的后果是农化制剂被过度使用，导致农民付出了本来可以避免的成本，并对生态造成重大影响。Tenacious Ventures之所以投资RapidAIM，是因为它为一个棘手的农业问题提供了一个相对廉价的解决方案，同时也促进了农业向减少化学品使用的转变。<br><br></div>
