<p>本文翻译自 Carmagazine，翻译文章并不代表我们认同或其观点，仅供参考之用。</p><p style="text-align: center"><img src="https://img.whalegogo.com/article/image/20210215/1613355197952653.jpg" _src="https://img.whalegogo.com/article/image/20210215/1613355197952653.jpg" id="img_32709" width="500"" title="1613355197952653.jpg" referrerpolicy="no-referrer"></p><p>本次为我们驾驶的是 C 级项目首席工程师 Christian Fruh，据我们了解，他已经是奔驰有着丰富经验的经理了，在全面深入负责底盘及辅助系统研发之前，他也负责代表梅赛德斯监控由梅赛德斯与迈凯伦联合打造的 SLR 车型（迈凯伦负责主要的研发与生产），接下来他将全面负责 C 级以及所有的四门轿车、四门敞篷车。</p><p>C  Class 的数据是怎么样？</p><p>我们所试乘的车型是 C300 4MATIC 车型，跟奥迪 45 TFSI 和宝马的 330i 一样，现在已经是 4 缸、2 公升排量统治这一级别的时代了，其输出从 254bhp（奔驰、宝马）- 260bhp（奥迪） 不等。 C300 4Matic 所采用的是奔驰目前 48V 轻混系统，整合了 BSG （启动电机），通过电机的辅助，能给予额外的 20bhp，以减少涡轮迟滞的现象（译者著：现款 C260 同样搭载 48V 电气系统，按实际体验确能有效降低涡轮建压至满压的过渡时间，涡轮迟滞要比 M274 2.0T 低功率版本轻微）。在全油门时 ISG 则能提供额外的 27bhp 动力。</p><p style="text-align: center"><img src="https://img.whalegogo.com/article/image/20210215/1613355209408269.jpg" _src="https://img.whalegogo.com/article/image/20210215/1613355209408269.jpg" id="img_32710" width="500"" title="1613355209408269.jpg" referrerpolicy="no-referrer"></p><p>搭载着两个成年人以及一个胖子的 C300 4MATIC 依然能以 6 秒的时间从静止加速至 100 公里每小时，在时速 60mph 时，发动机的转速仅仅比怠速超出一个手指的宽度，而 NVH 的表现呢？比起刚试驾的 6 缸 S500，这款四缸发动机在接近红转时更轻松从容，也不粗暴。</p><p style="text-align: center"><img src="https://img.whalegogo.com/article/image/20210215/1613355282483532.jpeg" _src="https://img.whalegogo.com/article/image/20210215/1613355282483532.jpeg" id="img_32713" width="500"" title="1613355282483532.jpeg" referrerpolicy="no-referrer"></p><p>惊喜大喜讯的是，这一代的 C 级可以同样选装出现在 S 级的后轮转向，让本身转弯半径不大的 C 在中低速时更灵活。</p><p style="text-align: center"><img src="https://img.whalegogo.com/article/image/20210215/1613355219199058.jpg" _src="https://img.whalegogo.com/article/image/20210215/1613355219199058.jpg" id="img_32711" width="500"/" referrerpolicy="no-referrer"></p><p>在大部分海外媒体所试乘的路段，包括快速的 S 弯、全油门的无限速高速， C 级的乘坐表现都是耀眼的。但在一些 C 级道路（乡村公路），C 级的行驶质感难以让人满意，选用了最宽的冬季轮胎以及头枕并没有装备舒适型头枕，低速的隆隆声，和横向刺激，让刚才印象深刻的运动型悬挂，打了折扣。</p><p>Christian 的解释是：上一代的 C 级提供空气悬挂，而整个生命周期空气悬挂的选装率不到 1%，所以（继续提供空气悬挂选装）不是明智的选择，动态车身控制跟运动型悬挂在 W205 车型是同时出现的，现在成本更高的可变阻尼减震器现在绑定在更加合适的舒适性悬挂上。</p><p style="text-align: center"><img src="https://img.whalegogo.com/article/image/20210215/1613355230984771.jpg" _src="https://img.whalegogo.com/article/image/20210215/1613355230984771.jpg" id="img_32712" width="500"" title="1613355230984771.jpg" referrerpolicy="no-referrer"></p><p>至于在新一代的 AMG C63 ，大家可能也会失望-V8 将不会出现。取而代之的是四缸搭配插电式混合动力，按照鲸跃的理解，发动机将采用 A45 S 上的 M139 2.0T 机器，并有可能进行进一步的强化。海外媒体质疑这样为什么不采用前驱平台， Christian 的解释是：前驱化将会是一个大大的错误。前驱并不能节省多少的重量，反而在牵引力、转向手感、乘坐舒适性以及驾驶体验上将会有倒退。</p><p>C 级的第一印象</p><p>全新的 C 级，在基因遗传上来看，更接近 E ，而不是 S 级。在 C 级上，将全系仅提供四缸车型，这意味着新车在性能以及效率上需要有更突出的表现，以弥补六缸以及八缸发动机的缺席。那些经常考虑到处旅行的，也许更应该考虑二手的 S 级，或者精明地选择斯柯达速派。那些并不是到处旅行的人，新 C 要比临时租来的低配 E 级更合适。</p><p><br></p>
