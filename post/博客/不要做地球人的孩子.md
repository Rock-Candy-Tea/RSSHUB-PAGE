<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">最近写文章很少</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; <div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这件事里值得批评的元素很多</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">还有作文批改方式以及中国教育的问题等。我没有写</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">是因为我学习了胡锡进的复杂中国说</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我觉得这事的确挺复杂</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-hansi-font-family: "Times New Roman";">我需要更大的信息量更多的细节</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这也是我写文章的特点吧</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">不追热点</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我不追热点倒不是觉得追热点有什么问题</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">那些总说追热点是消费当事人是吃人血馒头的人</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我觉得他们是有病的</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我一个时事评论者不写热点难道写你们家老母鸡又下蛋了。还有一个更重要的原因是</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我感觉这个事以及有些评论这个事的人</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">那扑面而来的气息让我窒息</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">就像我不能在炎热的夏天站在黄坑边去批评一堆蛆虫。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">直到当地有关部门以及有些人称小女孩的作文“系抄袭”</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">小女孩本子上的修改符号系学生所画</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">以及走访了班级很多学生和老师证明那个老师没有打骂跳楼自杀的缪可馨。我挺震惊的</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">但回头一想</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这震惊又有点友邦惊诧的感觉</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我还是别装外宾了</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">现如今</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">眼门前</span><span lang="EN-US">.</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">什么烂人什么脏事不会发生呢</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">祖国总体上是伟大复兴的</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">只是细节没来得及处理</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">局部还有些污七八糟。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我倒不是说可馨没抄袭</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我也不是要把她描绘的完美无暇</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我只是觉得</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">你们这个时候说这个千聊</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">证明一个孩子是抄装被发现羞愧跳楼</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">然后你们这些大人就可以洗脱责任就可以心安理得就可以埋怨孩子心理索质太差就可以指责家长教育的不好</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">看一眼这个社会吧</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">看看这些年这些成年人干的这些事说的那些话吧</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">越来越不要脸</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">越来越勇于不要脸</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">而且越来越自我合理化。那些在你们看来是所谓抄越来越自我合理化。那些在你们看来是所谓抄袭的证据</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">不正是你们长期以来亲自参与打造的病态社会在一个孩子身上造成的创伤吗</span><span lang="EN-US">?<o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">约翰·里顿说过这样一句话</span><span lang="EN-US">:</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">”在这个世界上</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">主要的危险是污染和枪炮火光冲天</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">而他们却要挑出说相话的人。”当一个孩子跳楼自杀后</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">他们终于找到了孩子抄装的证据并松了一口气。以前我说过</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这个社会连坏人都在做慈善了，得有多么美好啊，这当然是反讽</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我认为这是一个社会堕落到取得阶段性成果的主要标志。还有一件事就是一个社会开始毫无顾忌地指责受害者了</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">比如说一个女性被骚扰</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">大家开始指责女性着装暴露的问题</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这也是一个社会堕落到取得阶段性成果的标志</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我以为到这就差不多了</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">没想到今天一帮成年人开始指责一个受害的孩子。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这么多年，中国一直有一个很奇怪的现象，弱势群体一旦出现了一点瑕疵就很容易被诟病</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">受害者更是如此</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">而强势群体特别是作恶者</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">有一点点好事即便这个好事明显是为了掩盖他的恶</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">也会被放大和赞美。这个现象学界称之为“完美受害者”</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这个概念是指人们会特别在意受害者身上的瑕症</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">并认为是这些瑕疵导致了受害者受到的伤害</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">进而对受害者失去了同情</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这也给了很多人替加害者洗地的机会</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">甚至引导众人对受害者进行谴责</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这就是”谴责受害者”文化。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">谴责受害者不仅模糊了作恶者的主要责任</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">还加剧了受害者的痛苦</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">之所以出现这种现象</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">是因为公平世界语误”</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">“这种理论认为</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">倾向谴责受害者的人陷入了一种逻辑谬误</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">他们假设世界是完全公平的</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">灾难从不无故降临到人头上</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">只要行为得当就绝不会遭受不幸</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">因此受到伤害一定是因为受害人本身有错</span><span lang="EN-US">.</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">”这至少还是一种有逻辑的分析</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">即便这个逻辑是非人性的</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">但我看到的很多谴责受害者的人并非是这种情况</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">他们的情况很简单</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">就是单纯地不要脸没人性。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">周星驰的电影《西游降魔篇》有这么一段对话</span><span lang="EN-US">, </span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">说我杀人不眨眼</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">你</span><span lang="EN-US">TM</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">问我眼睛干不干</span><span lang="EN-US">!</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">”我说我杀人不眨眼</span><span lang="EN-US">,</span><span style="font-family: 宋体; <span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我们很多成年人经历的痛苦</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman";
