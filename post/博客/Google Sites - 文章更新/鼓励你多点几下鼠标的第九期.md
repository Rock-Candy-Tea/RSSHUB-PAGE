<h1>Design</h1>
<p><a href="https://components.ai/about"><strong>components.ai</strong></a>
数字世界的设计，还有许多方面值得被自动化，这是个帮助设计师通过鼠标点一点做设计的实验项目。摘一段链接中的原文：In design, what can be computed?How might we contribute to our collective design knowledge?If we lived in a world where every possible iteration of a design was already built - what would your workflow be to define what you’d ship? What tools would you use to sift through all of this generated noise?
<a href="http://iconpark.bytedance.com/official"><strong>字节跳动开源的1000+图标</strong></a>
图标表意清晰，大而全，可能是国内最佳。
<a href="https://www.youtube.com/watch?v=ZqlTSCvP-Z0&feature=youtu.be"><strong>How drawing helps you think | Ralph Ammer | TEDxTUM</strong></a>
拥有一定的sketch能力对表达设计想法与沟通都有帮助。这个演讲对我很有启发，可行性也极高。按他说的画了半个月圆圈，尽管暂时还没肉眼可见的进步，但我打算继续练习。也许有同学和我一样不会画画，推荐看看。链接指向Youtube。
<a href="https://spotify.design/article/ask-spotify-design-01"><strong>Spotify Design</strong></a>
分享他们设计团队工作的方方面面。
<a href="http://www.tuzei8.com/rethink-persona/"><strong>重新思考Persona</strong></a>
十分同意，我的想法是：Persona is borken。如果只是机械地使用「设计方法」，有可能做不出解决实际问题的东西。</p>
<h1>Programming</h1>
<p><a href="https://blog.techbridge.cc/2020/07/19/web-monetization-intro/"><strong>Web Monetization简介</strong></a>
一个Web API的提议，看名字就值得期待，对吧？
<a href="https://airbnb.io/visx/"><strong>https://airbnb.io/visx/</strong></a>
Airbnb开源的React可视化原子库，自由组合所需可视化组件。舒服！
<a href="https://coderemixer.com/2020/10/21/safari-is-fast-but-so-what/"><strong>如果 Safari 做不到对，快有何用？</strong></a>
写前端的人应该都被Safari坑过，这篇文章对Safari的某个非标准实现作了一番批判。
<a href="https://grid.layoutit.com/?utm_campaign=Dine"><strong>自动生成为grid布局Html+CSS</strong></a>
鼠标点一点就行，亲测好用，省时省力。
<a href="https://lobe.ai/"><strong>Lobe公测</strong></a>
之前介绍过微软的Lobe，那个鼠标点一点就能训练/使用机器学习模型的工具开始公测了，不再需要申请。
<a href="https://play.tailwindcss.com/"><strong>tailwind PLAY</strong></a>
一直想学tailwind，看到这个，感觉自己快入坑了。玩了会就意识到用它做原型设计再合适不过了，设计效率和对接开发的体验都完胜那些用鼠标点一点的原型工具。
<a href="https://joshwcomeau.com/"><strong>https://joshwcomeau.com/</strong></a>
一个前端工程师的网站，不看一眼会可惜的。
<a href="https://twitter.com/tommyvedvik/status/1317546025385680900?s=21"><strong>真·可视化编程</strong></a>
鼠标点一点，生成对应Vue代码。链接是作者发的推特的，暂未发布，值得关注。
