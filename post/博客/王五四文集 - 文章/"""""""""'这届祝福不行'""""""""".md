<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我的微信公号取了一百三十几个名字</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">就是没有取一个王平安，活该被封号，这次稳了。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">一个人嘴上祝福不难</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">心里祝福不难</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">难的是把对他人的祝福著书立传</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">一个人祝福一个人不难</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">祝福百个人不难</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">难的是祝福全人类全宇宙。他所站立的那个地方</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">正是他的地球</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">他以为他怎么祝福，人类便怎么样，他是什么</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">人民群众便发来什么</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">他是贺电</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">人民群众发来的却是质疑。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">五条人说</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">可以土的掉渣</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">绝不能俗不可耐。当年音乐界土的掉渣的歌曲是蔡国庆老师的《</span><span lang="EN-US">365</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">个祝福》</span><span lang="EN-US"> ,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">但达到俗不可耐水准的只有孙悦老师的《祝你平安》。”一年有三百六十五个日出</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我送你三百六十五个祝福</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">时钟每天转了一千四百四十圈</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我的心每天都藏着一千四百四十多个思念…每一分钟都盼望你平平安安”</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">你明明是蔡国庆老师的粉丝</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">却走上了孙悦老师的不归路。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">贺电副厅</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">你的心情</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">现在好吗</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">你的脸上</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">还有微笑吗</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">人生自古</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">就有许多愁和苦</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">请你多一些开心，少一些烦恼。在这里要说明一下</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我为什么敢提贺电这个名字而不怕被作为敏感词删除呢</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">因为各国领导人经常发来贺电</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">你不能为了一个副厅级贺电</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">伤害了国级贺电吧。求不被跨省</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">作者平安作者平安。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">盘古开天劈地以来</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">把祝福印刷成书的只有《圣经》了</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这是西方的产物</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">现在我们东方终于有了自己的《平安经》。这本印着人民出版社和群众出版社共同出版的书十分不平安</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">可能作者写书时忘了加上出版平安吧。人民出版社发声明称自己从来没出版过这本书</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">群众出版社默默无语</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">看来人民群众还是要分开看待的</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">人民都是无辜的</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">群众里往往藏着坏人。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">很多群众并不能领悟《平安经》的真谛</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">很简单粗暴的把它称之为发财经、升官经</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">有点猪八戒背媳妇</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">全不知滋味的滋味</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">当初唐僧师徒四人西天取经时</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">给他们传经的阿难、迦叶就说过</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">“东土众僧愚迷</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">不识无字之经</span><span lang="EN-US">.</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">””</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">如来也说过</span><span lang="EN-US">, </span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">“白本者</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">乃无字真经，倒也是好的。因你那东土众生</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">愚迷不悟，只可以此传之耳。”这叫什么</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">这叫有什么样的群众就有什么样的经书。作为群众里的知识分子，我认为《平安经》给了我很多启迪</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">它让我读懂了很难读懂的《道德经》</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">男道德它让我读懂了很难读懂的《道德经》</span><span lang="EN-US"> ,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">男道德女道德</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">你道德我道德</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">垃圾道德不分类倒不得。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">五条人说</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我们知识分子不打架。知识分子不打架</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">只拍马屁。知识分子张咏的“拜读《平安经》感言”，里边写到</span><span lang="EN-US">:</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">“从他的新作《平安经》中</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">令人感知到一位学者深邃的灵魂和宽广的情怀。”张老师灵魂深处得打多少次架才能写出这么发人深省的文字。张咏老师还写道“官员阅读此书，领悟初心使命。学者阅读此书，顿悟平安哲理。商贾阅读此书</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">企业平安无虞。民众阅读此书，安享世间太平。”你这么写</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">各级政府看了都开始思考自己存在的意义以及自己和图书馆的区别了，你是想让书取代书记</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">你这不是在捧贺副厅，是在捧杀他。</span><span lang="EN-US"><o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">《平安经》是一本很全面的平安书</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">但其实也有不少漏洞</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">比如说“初生平安、满月平安、白天平安、</span><span lang="EN-US">1</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">岁平安、</span><span lang="EN-US">2</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">岁平安、</span><span lang="EN-US">4</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">岁平安</span> <span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">……那千岁呢万岁呢</span><span lang="EN-US">, </span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">“全世界人民大团结万岁”不需要祝福吗</span><span lang="EN-US">?<o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">“西安火车站北站平安、郑州火车站东站平安、上海虹桥火车站平安…那济南火车站呢</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">丽水高铁站呢</span><span lang="EN-US">?<o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
<span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">“宇宙时空平安、世界大地平安、世界各界平安、人类文明平安、国家地区平安、中华大地平…</span><span lang="EN-US">.</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">国也一样平安吗</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">特朗普也平安吗</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">美国驻成都总领事馆也平安吗</span><span lang="EN-US">?</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">他们怎么能平安呢</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">他们平安了</span><span lang="EN-US">,</span><span style="font-family: 宋体; mso-ascii-font-family: "Times New Roman"; mso-hansi-font-family: "Times New Roman";">我们还能平安吗</span><span lang="EN-US">?<o:p></o:p></span></div>
<div class="MsoNormal" style="text-indent: 21.75pt;">
