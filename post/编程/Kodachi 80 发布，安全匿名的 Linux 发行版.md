                                                                                            <p>Linux Kodachi 8.0 已发布，这是一个基于 Xubuntu 18.04.5 的 Linux 发行版，主打安全和隐私（预装 VPN, Tor 和 DNSCrypt 应用），并实现了匿名和反取证功能。</p> 
<p><strong>主要特性</strong></p> 
<ul> 
 <li>匿名访问互联网</li> 
 <li>所有与互联网的连接都必须先经过 VPN，然后再经过具有 DNS 加密的 Tor 网络</li> 
 <li>除非明确要求，否则不会在正在使用的计算机上留下任何痕迹</li> 
 <li>使用最先进的加密和隐私工具对文件、电子邮件和即时消息进行加密</li> 
</ul> 
<p><img src="https://static.oschina.net/uploads/space/2021/0320/221657_TlOT_2720166.png" referrerpolicy="no-referrer"></p> 
<p>新版本升级了内核，带来了新的主题和仪表盘、以及增加许多有用的调整。</p> 
<ul> 
 <li>Linux Kernel 从 5.9.8 升级到 5.11.7</li> 
 <li>Kodachi 有了新的品牌、新的 logo、壁纸、仪表盘和系统监视器</li> 
 <li>新的仪表盘具有令人难以置信的独特功能（官方表示这是在其他操作系统中永远看不到的，并将此留给用户去探索)</li> 
 <li>支持根据新的 IP 地址自动更改系统时区</li> 
 <li>支持为系统使用虚假的主机名</li> 
 <li>可在 Kodachi 系统监视器中查看 Kodachi 源代码和 shell 脚本</li> 
 <li>可在 Kodachi 系统监视器中查看和监视所有类型的系统日志</li> 
 <li>可使用 Kodachi 生成密码</li> 
 <li>支持一键验证所有 Kodachi 文件源和系统文件</li> 
 <li>支持一键禁用 WiFi、蓝牙、RF 和屏蔽 USB 设备</li> 
 <li>改变对 VPN, DNS TOR, Panic room 的管理</li> 
 <li>为杀毒功能增加了六种开关</li> 
</ul>
