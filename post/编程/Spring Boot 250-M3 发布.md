                                                                                            <p>Spring Boot 2.5.0 第三个里程碑版本<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv2.5.0-M3" target="_blank">已发布</a>，此版本增加了不少新功能，以及修复错误。</p> 
<p>值得关注的如下：</p> 
<ul> 
 <li>OpenMetrics 提供对 Prometheus 的支持</li> 
 <li>支持环境变量前缀</li> 
 <li>支持 Docker Buildpack：Bindings、支持 War</li> 
 <li>增加 DataSource 初始化机制</li> 
 <li>升级文档：外观焕然一新，包括引入“深色主题”</li> 
 <li>升级依赖</li> 
</ul>

<p>如果准备开始使用 Spring Boot 2.5 并尝试新功能，可以在 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fstart.spring.io%2F%23%21type%3Dmaven-project%26language%3Djava%26platformVersion%3D2.5.0.M3%26packaging%3Djar%26jvmVersion%3D11%26groupId%3Dcom.example%26artifactId%3Ddemo%26name%3Ddemo%26description%3DDemo%2520project%2520for%2520Spring%2520Boot%26packageName%3Dcom.example.demo" target="_blank">start.spring.io</a> 上引导一个新项目。</p>
