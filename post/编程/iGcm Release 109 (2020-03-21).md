<h4 style="text-align:start"><span style="color:#222222">Commons changes</span></h4> 
<ul> 
 <li>类支持在jar文件中扫描</li> 
 <li>添加Maven安装脚本</li> 
 <li>通过API支持Elasticsearch</li> 
 <li>支持Hive</li> 
</ul> 
<h4 style="text-align:start"><span style="color:#222222">Netty changes</span></h4> 
<ul> 
 <li>修复了当指定控制器软件包为空时程序无法启动的问题</li> 
 <li>修复控制器中非外部提供的方法的扫描错误</li> 
 <li>未指定修复路径/前缀使路径无法识别</li> 
 <li>修复了使用压缩包时服务无法扫描指定控制器的问题</li> 
 <li>修复视图未传递参数，导致服务无法解析的问题</li> 
</ul> 
<h4 style="text-align:start"><span style="color:#222222">Postgres changes</span></h4> 
<ul> 
 <li>添加<code class="language-plaintext">@PostgresSource</code>注释</li> 
</ul> 
<h4 style="text-align:start"><span style="color:#222222">ClickHouse changes</span></h4> 
<ul> 
 <li>添加<code class="language-plaintext">@ClickHouseSource</code>注释</li> 
</ul> 
<h4 style="text-align:start"><span style="color:#222222">Docs changes</span></h4> 
<ul> 
 <li>添加elasticsearch文档</li> 
 <li>添加hive文档</li> 
</ul>
