                                                                                            <p>Universal Media Server 10.4.0 已经发布。Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，并提供 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>通用 
  <ul> 
   <li>添加了渲染器配置设置 DisableUmsResume，用于支持三星电视等渲染器自身的重启</li> 
   <li>将转码到 H.264 的速度提高了 3 倍</li> 
   <li>修复了 MEncoder 在尝试将 7.1 AAC 降频到 5.1 AC-3 时崩溃的问题</li> 
   <li>修复了渲染器的 SupportedVideoBitDepths 设置</li> 
   <li>优化了禁用恢复功能时的内存使用和速度</li> 
   <li>修复了渲染器分辨率和比特率限制</li> 
   <li>修复了自上一版本以来部分用户的转码问题</li> 
  </ul> </li> 
 <li>渲染器 
  <ul> 
   <li>优化了对索尼蓝光 UBP-X800M2 的 VC1 编解码器的支持</li> 
   <li>优化了对 Kodi 上许多文件类型的支持</li> 
   <li>优化了对 VLC 上 12 位视频的支持</li> 
  </ul> </li> 
 <li>依赖 
  <ul> 
   <li>将 git-commit-id-plugin 从 4.0.3 升级到 4.0.4</li> 
   <li>已回滚 FFmpeg</li> 
  </ul> </li> 
</ul>
