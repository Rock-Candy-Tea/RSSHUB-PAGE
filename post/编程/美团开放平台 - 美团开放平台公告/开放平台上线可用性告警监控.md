<p style="text-align: start;">尊敬的服务商：</p><p style="text-align: start;">      <span style="color: rgb(0, 0, 0)">开放平台将于6月8日上线【服务商可用性监控】，服务商需将开发者账号与微信公众号绑定，如果可用性值低于开放平台设置的标准值，将会在每周一收到相关推送。连锁品牌的告警模板消息后续也会上线，尽请期待。</span></p><p style="text-align: start;"><span style="color: rgb(0, 0, 0)">通过可用性监控，您可以了解自己系统当前稳定性是否能够满足平台标准，</span><strong><span style="color: rgb(0, 0, 0)">当您收到可用性不满足标准值的通知后，请进行内部排查并优化。如果排查中遇到疑问，可联系您对应的开放平台运营同学咨询，</span></strong><span style="color: rgb(0, 0, 0)">从而为广大餐饮商户提供更加稳定的系统服务。</span></p><p style="text-align: start;"><strong><span style="color: rgb(0, 0, 0)">一.可用性业务介绍</span></strong></p><p style="text-align: start;"><strong><span style="color: rgb(0, 0, 0)">服务商可用性=开发者id上周有效的回调接口次数 / 上周全部的回调接口次数。</span></strong></p><ul><li><p><strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">监控周期：</font></span></strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">按周统计服务商developerId的可用性。</font></span></p></li><li><p><strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">可用性标准：</font></span></strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">根据服务商覆盖业务量，餐饮开放平台经过评估后，为服务商的每个developerId所设置可用性标准值。</font></span></p></li></ul><ul><li><p><strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">是否满足可用性标准：</font></span></strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">当上周的服务商可用性＜可用性标准，为【不满足】；当上周的服务商可用性 ≥可用性标准，为【满足】。</font></span></p></li><li><p><strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">全部请求量：</font></span></strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">服务商的developerId，所有回调类接口的调用成功和失败的次数。</font></span></p></li><li><p><strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">不可用请求量：</font></span></strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">服务商的developerId，所有回调类接口的调用失败的次数。<br></font></span><strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">注意：</font></span></strong><span style="color: rgb(0, 0, 0)"><font style="font-size:14px;line-height:22px" data-size="14">针对于您的可用性标准值如有疑问，可联系您对应的开放平台运营同学咨询。</font></span></p></li></ul><p style="text-align:
