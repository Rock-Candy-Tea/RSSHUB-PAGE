                                                                                            <p>Apache OFBiz 17.12.06 现已<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmail-archives.apache.org%2Fmod_mbox%2Fwww-announce%2F202101.mbox%2F%253cCAEvdU_3FOD-Bj%2B%2BodfW%2Bsvkhw43mSPYUHFXACxQYz3xJu3Yd7Q%40mail.gmail.com%253e" target="_blank">发布</a>。官方表示，这是 17.12 系列的第六个版本，也是可能是这个系列的最后一个版本；接下来的更新将会从 18.12 开始。Apache OFBiz 是用于企业流程自动化的开源产品，包括框架组件和业务应用程序。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>将 Tomcat 从 9.0.41 升级到 9.0.43</li> 
 <li>webtools/control/threadList 不再适用于主线</li> 
 <li>将 “财务账户交易” 修改为 “会计交易”</li> 
 <li>修复创建博客或论坛时保存失败的问题</li> 
 <li>修复无法按顺序添加 ASSET_USAGE 类型产品的问题</li> 
 <li>修复 ViewQuoteProfit 以错误的顺序显示字段的问题</li> 
 <li>修复会计交易 pdf 导出中标签重叠的问题</li> 
 <li>修复删除财务帐户时出错的问题</li> 
 <li>在 ObjectType::simpleTypeOrObjectConvert 中绕过 Java GString 到 String 的转换</li> 
 <li>修复运行 MRP 时出错的问题</li> 
 <li>放大 Rainbow Stome 主题的 "more-app" 菜单</li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmail-archives.apache.org%2Fmod_mbox%2Fwww-announce%2F202103.mbox%2F%253CCAEvdU_0wQQJ6RXPqiR%2BVV7VCPX-XtBbFoUtg5R-rE%3Dv%3DhrcKRA%40mail.gmail.com%253E" target="_blank">更新公告</a>。</p>
