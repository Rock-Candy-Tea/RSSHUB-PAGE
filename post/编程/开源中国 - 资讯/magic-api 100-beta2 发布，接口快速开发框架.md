                                                                                            <h3 style="text-align:left">本次更新内容：</h3> 
<ul> 
 <li><strong>新增动态数据源管理(可在页面动态修改数据源)</strong></li> 
 <li><strong>新增路径变量验证</strong></li> 
 <li>修复打成<code>jar</code>包后无法导出接口的BUG</li> 
 <li>修复复选框未回显的问题</li> 
 <li>修复在新增函数时，参数类型不回显的问题</li> 
 <li>修复不能关闭跨域配置的BUG</li> 
 <li>修复打包后无法读取<code>js</code>配置文件的问题</li> 
 <li>修复权限配置不正确的BUG</li> 
 <li>修复配置抛出异常无效的BUG</li> 
 <li>优化<code>JSON</code>构建接口，参数统一封装，减少方法</li> 
 <li>优化读取资源逻辑，增强数据库兼容性</li> 
 <li>优化日志打印，方便排除错误</li> 
 <li>优化模拟测试，增加支持路径变量配置</li> 
</ul> 
<h4 style="text-align:start"><strong>项目介绍</strong></h4> 
<p style="text-align:left"><span style="background-color:#ffffff; color:#40485b">magic-api 是一个基于Java的接口快速开发框架，编写接口将通过magic-api提供的UI界面完成，自动映射为HTTP接口，无需定义Controller、Service、Dao、Mapper、XML、VO等Java对象即可完成常见的HTTP API接口开发</span></p>
