                                                                                            <p>DBeaver 21.0.1 现已发布。DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>连接管理 
  <ul> 
   <li>增加了自动结束闲置交易的功能</li> 
   <li>修复了网络配置文件配置页面</li> 
   <li>修复了连接选择器对话框中的过滤器</li> 
  </ul> </li> 
 <li>数据传输 
  <ul> 
   <li>增加了禁用外键的可能性 (PostgreSQL，Oracle)</li> 
   <li>数值数据类型映射得到改善</li> 
   <li>在错误对话框中添加了“跳过所有错误”按钮</li> 
  </ul> </li> 
 <li>原数据编辑器 
  <ul> 
   <li>列删除现在检查依赖对象</li> 
   <li>“约束/索引创建” 命令现在在 “列编辑” 选项卡中可用</li> 
   <li>修复已删除对象 dup 的问题</li> 
  </ul> </li> 
 <li>SQL 编辑器 
  <ul> 
   <li>改进了脚本选择器弹出面板的 UIX</li> 
   <li>修复了十六进制常量和科学数字的格式</li> 
   <li>改进了列别名的生成</li> 
   <li>修复了多行注释支持</li> 
  </ul> </li> 
 <li>数据编辑器 
  <ul> 
   <li>改进了列式过滤器的 UI</li> 
   <li>为 SQL 脚本生成器添加了额外的配置</li> 
   <li>在空间查看器（GIS）中更好地支持 IE 浏览器</li> 
   <li>虚拟列支持（行添加/删除）已修复</li> 
   <li>单元格的选择已修复</li> 
  </ul> </li> 
</ul>
