<h3 style="margin: 0px; padding: 0px; line-height: 2; text-align: center;"><strong>胡乱裁军，苏联军工的悲催结局</strong></h3><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期精彩看点：</strong></p><p style="margin: 0; padding: 0; line-height: 2;">美苏冷战前后持续四十多年，在这期间，美国和苏联都进行了大刀阔斧的裁军、撤军、削减军费，然而为什么美国越变越强，苏联却走向没落呢？</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">【二战后世界主要国家两次军事战略改革】</strong> </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">二战以后，世界主要国家军事战略经历过多次调整和改革，但规模最大的主要有两次：第一次是50年代前后，第二次是90年代前后。第一次转变用了3年，第二次转变已经持续了大约10年。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/05/31/a3319acefce73187c4a075e8e813d650.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">在第一个转变时期，正值二战结束，军事战略重点要求尽快从战时状态转入和平时期的经济建设方面来，所以继续保持高居不下的军费和数量众多的军备已经没有意义。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">第二次军事战略转变时期国际环境非常复杂，从80年代中期，到1991年冷战结束之后各国最终定下决心进行大规模裁减。其中美国在军费、军备、军队员额等方面进行了史无前例地全面裁减。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">【美苏裁军】</strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">世界各国尤其是美苏在裁军、撤军、削减军费军备主要表现在三个方面。首先，大规模裁军，苏联甚至“瘦身”850万人。其次，大幅度削减军费预算，90年代逐年递减。最后，大量裁减武器装备。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"> <img src="https://pic.qingting.fm/programplaypage/2019/05/31/d47e9b5f24254a7c097bffe137e6d53c.png" style="display: block;
