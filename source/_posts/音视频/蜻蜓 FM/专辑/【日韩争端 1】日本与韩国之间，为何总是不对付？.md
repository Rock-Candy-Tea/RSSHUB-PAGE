padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">美日、美韩都是军事同盟，美国左手拉一个韩国，右手拉一个日本，但是美日韩这三家弄不到一块去，美国都是一手拉一个，都是哥俩好，但是哥仨就是好不了。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">日韩关系问题在二战劳工赔偿这个问题上表现比较充分，主要是韩国现在有很多人的后代要求日本的一些企业给予赔偿，日本的企业以种种理由拒绝赔偿。因为这要赔的话，你都经过几代人了，能赔得起吗？所以就懒得赔偿，双方在这方面产生了一个民间要赔偿，另外一个不给赔偿的情况。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/18/943e8665064d838d3ed6a73ed5ee17d2.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期百科： </strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong>【战犯企业】</strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">所谓“战犯企业”，是指一些日本企业在侵略韩国、中国以及其它亚洲国家的过程中，直接为日本侵略战争服务，帮助、协助、参与了对被侵略国民众的屠杀，掠夺被侵略国的各种经济、文化资源，通过战争强征劳工为其企业无偿劳动，从而事实上犯了战争罪，形同战犯。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/18/74a59e4930ca9fc8c5369528f3327a12.jpeg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">韩国列出的日本“战犯企业”共有136家，著名的三菱、日立、川崎、日产、丰田、东芝等均榜上有名。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong>【日韩请求权协定】 </strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">1965年的《日韩请求权协定》规定，日本政府向韩国政府无偿提供3亿美元（约合人民币20亿元）、有偿提供2亿美元（约合人民币13亿元），以解决两国政府及国民间的所有索赔权问题，韩国签署该协议之后“不能再提出任何主张”，韩国民众不能再向日方索赔。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/18/170a128203dc79a31a0da815c1a216ce.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">日本政府一直声称，该协定已经彻底解决了两国间的劳工问题。但是，2012年以来韩国法院相继做出判决，认定日本殖民统治时期的不法行为产生的损失索赔权，并未因《日韩请求权协定》而消失。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">橘妹在线聊：</strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">小橘子们，我们又见面啦~</p><p style="margin: 0; padding: 0; line-height: 2;">近日，日本和韩国之间因为贸易争端“翻脸”，牵出了两国长期存在的历史遗留问题。小橘子们听完有何感想？欢迎给我们评论留言。 </p>
