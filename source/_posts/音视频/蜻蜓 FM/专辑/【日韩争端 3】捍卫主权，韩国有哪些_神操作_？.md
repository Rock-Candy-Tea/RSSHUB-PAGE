<p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期热点话题：</strong>  </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">近期日韩贸易摩擦持续发酵。日本政府7月1日宣布，从7月4日起对韩国的三种半导体产业原材料加强管制，并将韩国排除在贸易“白色清单”之外。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/25/856539cf4e39bf957b68e38a872208bf.jpeg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">在24日召开的世贸（WTO）总理事会会议上，日韩代表就日本加强半导体材料对韩出口管制（“限贸”）展开激烈辩论。韩方批评日方措施是“不当的经济措施”；日方则反驳，两国重复此前的主张，讨论处于平行线状态。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">事件会如何收场？ </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期局座精彩语录： </strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">1950年朝鲜战争之前，日本是一个战败国，领土只限于四个本岛，没有政治表决权，整个国家是在美国的军管之下，它没有办法对外签协定，整个国家的大拿就是麦克阿瑟，麦克阿瑟是一个驻日美军总司令又是对日本治理的总的司令官，所有的事都他说了算。1950年朝鲜战争爆发了，朝鲜战争爆发了之后，美国它就想做一个什么事情呢，它就想快速的和日本签署一个和约，让日本有独立决策权，有行政权，美国想承认日本是一个独立国家。</p><p style="margin: 0; padding: 0; line-height: 2;"> </p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/25/af20762c6f855e962b2f215e962538e8.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">二战结束以后，美国对日本的态度其实很明确，就是看住日本，防止军国主义复活，这是美国打的一张牌，我记得当时80年代我们跟日本讨论，我们就提出来说美国你在日本驻军太多了，对中国构成威胁，美国当时就说你们中国是希望我们美国在日本撤军吗？我们哑口无言，我们说你想表达什么意思？美国说我现在要从日本撤军的话，日本军国主义复活，将来对中国形成更大的威胁，你们愿意这样吗？其实还真是这么回事。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">美国在日本驻军，起码初始阶段，80年代、90年代，2000年之前还都是防止军国主义复活，但是以后就把军国主义扶植起来，慢慢把它“养肥”了，让它“长胖”。美国怂恿日本，借日本来看住中国。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">有一段时间韩国它的海军打算出第一岛链到太平洋，其实现在它主要还是在东海这块活动，维护它的独岛海域的安全。所以说韩国和日本之间关于独岛/竹岛争端主要是表现在韩国特别善于用兵，有点类似南海的那个包心菜的战略，所以说它把海警，警察，和海空军都作为一道一道的这样的包心菜，一环扣一环，你不能光是海警在前头，是不是，你碰到渔民的时候，警察代表国家执法，行政执法；海警是武装执法，海军维护国家主权，必须结合起来，一线、二线、三线都有一个照应，同时韩国也在独岛上边建立了大量的基础设施，发电，通信，雷达，火力阵地。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期精彩看点：  </strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">近期日本对韩国实施出口管制，两国“翻脸”；而日韩的领土问题也一直是争议不断，两国之间也是为了领土问题采取了各种措施。韩国捍卫主权的哪些手段，让局座啧啧称赞？来听本期开讲，便见分晓。</p><p style="margin: 0; padding: 0; line-height: 2;"> </p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期百科： </strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong>【旧金山对日和平条约】 </strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">旧金山对日和平条约，通称“旧金山和约”，由第二次世界大战的48个战胜国与战败国日本于1951年9月8日在美国旧金山所签订的条约，并于1952年4月28日正式生效。该合约主要是为了解决第二次世界大战后战败国日本的领土及国际地位问题。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/25/be77d0d11fefd621f89bc4d31eb58a2a.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">合约的声明：日本承认朝鲜半岛之独立、放弃台湾、澎湖、千岛群岛、库页岛、南沙群岛、西沙群岛等岛屿的主权。《旧金山和约》是身为战败国的日本确立战后再次崛起和确立国家走向的决定性合约。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">因《旧金山和约》签订时，身为主要战胜国之一的中国被美国、英国、法国等国排除在外，故中国政府自和约签订至今均未承认过《旧金山和约》。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong>【李承晩线】</strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">李承晩线是历史上一条日本和韩国有主权争议水域的分界线，于1952年1月18日设立。当时韩国总统李承晚称为了宣示海洋主权范围及保卫当地的水产物，单方面在日韩两国之间的日本海公海海域上划定这条界线，并禁止外籍渔船闯入。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/25/d806c2995c526914b5d66c858788f2fd.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">韩国方面称这条线为“平和线”，而日本方面，则依照订立界线者而称之为“李承晚线”。为保障韩国渔民的作业，对于违反禁令的渔船（主要是日本籍渔船），韩国方面会采取捉拿、临检等措施，甚至枪击。直到现在，已经有3929名日本渔民被捕，当中导致44人死伤。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">日本方面，强烈要求韩国把界线恢复回麦克阿瑟制定的麦克阿瑟线，起初为韩国所拒绝，直到1961年才取消。但在李承晚线以内的独岛/竹岛仍然被韩国管治。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong>【独岛大捷】</strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">1952年1月18日，韩国发表一个名为“和平线”的发言，宣布对包括独岛在内的周围岛屿和海洋行使主权，日本政府则宣布该岛为日本领土，两国主权争端开启。由于忙于朝鲜战争，韩国政府没有实质行动，但平民洪淳七则自行组织郁陵岛上的回乡官兵和渔民共33人，成立“独岛义勇守备队”，自发坚守独岛3年零8个月，直到韩国政府派警察接管为止。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/25/57ea5d5809835751b83c07ad29b5a3a4.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">期间，洪淳七多次率领守备队员成功抵挡了日本对独岛的攻击。1954年11月21日，日本完全以包围状态攻击独岛，洪淳七进行了顽强的抵抗，最终击退了日本方面的进攻。事后据日本NHK的广播报道，此次造成日本方面死伤16人。这就是韩国方面宣称的1954年11月21日的“独岛大捷”。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">橘妹在线聊： </strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">小橘子们，我们又见面啦~</p><p style="margin: 0; padding: 0; line-height: 2;">近日，日本和韩国之间因为贸易争端“翻脸”，牵出了两国长期存在的历史遗留问题。小橘子们听完有何感想？欢迎给我们评论留言。 </p><p style="margin: 0; padding: 0; line-height: 2;"> </p>
