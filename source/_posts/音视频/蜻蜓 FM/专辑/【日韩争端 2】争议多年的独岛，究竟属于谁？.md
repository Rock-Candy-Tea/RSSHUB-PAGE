padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期局座精彩语录： </strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">独岛它跟钓鱼岛完全是一样的，是美国安排的一个棋子，美国设的一计，韩国说独岛自古以来就是韩国的固有领土，这个话我们很熟悉。据史料记载，朝鲜王朝时期，也就是1392年到1910年的中国明清时期，那个时候独岛还叫于山岛，2012年韩国就把独岛最高的峰更名为于山峰，韩国就说是他们起的名，自古以来就叫于山岛，现在叫于山峰，所以说证明独岛属于韩国，国家能给它命名，就像能给孩子起名字一样，那独岛它就是他们韩国的“孩子”。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"> 根据日韩双方的举证，在历史性传承方面，有一条国际法规定，就是领土获得要有六个要素，首先是先发现，这是一个很明显的要素，就是你谁先发现的，然后先占领的，包括以后填海造地这些都算领土的添附，所以说在先发现这个问题上，韩国明显优于日本。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/22/943e8665064d838d3ed6a73ed5ee17d2.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">一场战争，尤其是大海战，就会结束一个帝国的命运，1588年的英国和西班牙的大海战，从此之后就结束了西班牙日不落帝国这样的统治，英国逐渐就成为世界上最大的一个帝国。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">那个时候信息不像现在，有微博，新媒体等那么快，那个时候信息传播比较慢。韩国突然就发现这个，说日本侵犯了韩国主权，6世纪的时候独岛就是他们韩国的领土，什么时候变成你日本的，你强占了就算你的，那不行啊，你这是用战争方式掠夺的，这不是无主地，是韩国的领土。日韩双方开始就这个主权问题产生争端，双方还曾搁置了一段时间，但是韩国一直都坚持这个独岛就是他们的领土，日本不能够抢占，这个争论一直延伸到现在。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期精彩看点： </strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">近期日本对韩国实施出口管制，两国“翻脸”；而日韩的领土问题也一直是争议不断，那么两国争议多年的独岛究竟属于谁？ 且听本期局座硬核开讲。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期百科： </strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong>【第二次日韩协约】</strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">《第二次日韩协约》又称《乙巳条约》或《韩日协商条约》，是日本帝国与大韩帝国在1905年（按干支纪年为乙巳年）11月17日于韩国首都汉城（今首尔）签订的不平等条约。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/22/62f9c0a08aae9d44bef2c887a6423bb5.png" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">根据协约，日本将大韩帝国纳入自己的保护国，完全剥夺了大韩帝国的财政和外交权。日本根据该条约在韩国设置统监府以控制韩国。《第二次日韩协约》标志着韩国沦为日本的保护国，变成日本事实上的殖民地。该条约遭到大部分韩国人坚决反对，引起了义兵运动等一系列抗争。关于该条约的合法性与有效性也是日本与朝鲜半岛长期以来的争议话题。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong>【美国军管日本】</strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">日本二战战败后，根据《波茨坦条约》，以美国为首的盟军对日本进行军事管治。美国派遣占领军进驻日本，以麦克阿瑟为总司令直接控制日本国内的日常行政与军事事务。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/07/22/df3198f4da6dd77fd294974d75bf28af.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">美国按非军事化及民主化对日本进行彻底改造。主要表现在解除日本兵力，在日本国内推行经济自由化和民主化，进行政治整肃，清算战犯，大力推行美式教育等。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">橘妹在线聊： </strong></p><p style="margin: 0;
