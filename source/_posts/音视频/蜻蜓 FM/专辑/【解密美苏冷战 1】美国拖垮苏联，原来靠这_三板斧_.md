<h3 style="margin: 0px; padding: 0px; line-height: 2; text-align: center;"><strong>美国瓦解苏联 “三板斧”之贸易战</strong></h3><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期热点事件：</strong></p><p style="margin: 0; padding: 0; line-height: 2;">美国将进口自中国的2000亿美元清单商品加征的关税税率由10%提高到25%，此后，中国也宣布了报复性关税措施，这是中美贸易战双方最新一轮的“攻击”。</p><p style="margin: 0; padding: 0; line-height: 2;">而在去年，特朗普政府决定不让其邻国的钢铁和铝产品享受分别为25%和10%的关税赦免。加拿大和墨西哥都作出了报复。</p><p style="margin: 0; padding: 0; line-height: 2;">事实上，贸易战乃是美国整人的老套路，早在八十年代，美国就用“贸易战”来打垮苏联经济。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/05/20/17434ff9dda4f5ceff5d3e727d9e9014.png" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期精彩看点：</strong></p><p style="margin: 0; padding: 0; line-height: 2;">1982年11月，里根总统批准了NSDD（National Security Decisions Directive）-66号指示，声称“通过对苏联经济赖以生存的战略性三位一体的关键性资源进行攻击，进而瓦解苏联经济，这将成为美国的政策。”成为“美苏贸易战”的开始。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong>【里根政府利用“经济战”搞垮苏联】</strong></p><p style="margin: 0; padding: 0; line-height: 2;">里根政府时期美国中央情报局的官员后来写书透露了里根政府利用“经济战”搞垮苏联的做法，我们从这些手法中可以看出，美国政府在对付它的“敌手”时，是不惜操纵、控制市场的。在“捍卫”美国的国家利益时，它决不在乎违反市场的规律。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">———————————————————————————————</p><h3 style="margin: 0px; padding: 0px; line-height: 2; text-align: center;"><br></h3><h3 style="margin: 0px; padding: 0px; line-height: 2; text-align: center;"><strong style="color: rgb(225, 52, 48);">购买本专辑可查看全部内容</strong></h3>
