<h3 style="margin: 0px; padding: 0px; line-height: 2; text-align: center;"><strong>“星球大战”计划，只是里根的骗局？</strong></h3><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期精彩看点：</strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">美苏冷战前后持续四十多年，这期间，美国总统里根通过“星球大战”计划加速了苏联解体，这中间美国究竟用了什么伎俩忽悠了苏联？ </p><p style="margin: 0; padding: 0; line-height: 2;">1987年，美、苏两国首脑签订《美苏消除两国中程和中短程导弹条约》，简称中导条约。条约禁止双方试验、生产和部署射程500至5500公里的陆基巡航导弹和弹道导弹，但对海基和空射的巡航导弹和弹道导弹并没有做出限制。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/06/05/ce6748e20fd7350f5f2d6d7217fc226e.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"> </p><p style="margin: 0; padding: 0; line-height: 2;">2018年10月20日，美国指责俄罗斯4年来多次违反条约规定，准备退出《中导条约》。2019年2月2日起，美方暂停履行《中导条约》相关义务，正式启动为期180天的退约进程。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">同日，俄罗斯总统普京宣布，俄罗斯暂停履行《中导条约》义务，以回应美国。5月30日，普京提交了暂停履行《中导条约》义务的法案。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">【苏联1983年击落韩国波音747客机事件】</strong> </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">1983年，一架从美国阿拉斯加起飞的大韩民航波音747客机，在飞往汉城(首尔)途中偏离航道飞向了堪察加半岛南部以及萨哈林岛，“碰巧”途经了苏联部署在这一地区的战略弹道导弹基地。最终这架客机被奉命起飞的苏—15战斗机击落，机上269名乘客全部罹难。世界舆论一片哗然。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/06/05/5ff60dbf8461b45dd2b67b60b243d2fd.jpg" style="display: block; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">勃列日涅夫在执政期间，废除了赫鲁晓夫制定的限制干部任期制度，导致许多高级干部一直连任下去，造成高层机构长期无法输入新鲜血液，陷入“严重的停滞”，出现了“老人政治”局面。</p><p style="margin: 0; padding: 0; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">自斯大林以后，苏联领导层继任年龄逐渐增大。斯大林成为总书记时是45岁，赫鲁晓夫59岁，勃列日涅夫57岁，安德罗波夫68岁，契尔年科73岁。这些人中间，除去赫鲁晓夫是非正常离任，其余的领导人都是在任上去世。领导人的继任年纪越来越大，带来的另一个问题，就是整个苏联领导层也随之固化。 </p><p style="margin: 0;
