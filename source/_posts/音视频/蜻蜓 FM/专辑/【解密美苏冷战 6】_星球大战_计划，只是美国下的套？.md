<h3 style="margin: 0px; padding: 0px; line-height: 2; text-align: center;"><strong>“星球大战”计划，只是里根的骗局？</strong></h3><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">本期精彩看点：</strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">美苏冷战前后持续四十多年，这期间，美国总统里根通过“星球大战”计划加速了苏联解体，这中间美国究竟用了什么伎俩忽悠了苏联？ </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><strong style="color: rgb(225, 52, 48);">【中导条约】</strong></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">1987年，美、苏两国首脑签订《美苏消除两国中程和中短程导弹条约》，简称中导条约。条约禁止双方试验、生产和部署射程500至5500公里的陆基巡航导弹和弹道导弹，但对海基和空射的巡航导弹和弹道导弹并没有做出限制。</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;"><img src="https://pic.qingting.fm/programplaypage/2019/06/05/ce6748e20fd7350f5f2d6d7217fc226e.jpg" style="display: block; margin: auto; max-width: 100%;" referrerpolicy="no-referrer"></p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">2018年10月20日，美国指责俄罗斯4年来多次违反条约规定，准备退出《中导条约》。2019年2月2日起，美方暂停履行《中导条约》相关义务，正式启动为期180天的退约进程。 </p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0; padding: 0; line-height: 2;">同日，俄罗斯总统普京宣布，俄罗斯暂停履行《中导条约》义务，以回应美国。5月30日，普京提交了暂停履行《中导条约》义务的法案。 </p><p style="margin: 0; padding: 0; line-height: 2;">——————————————————————————</p><p style="margin: 0; padding: 0; line-height: 2;"><br></p><p style="margin: 0px; padding: 0px; line-height: 2; text-align: center;"><strong style="color: rgb(225, 52, 48);">购买本专辑可查看全部内容</strong></p>
