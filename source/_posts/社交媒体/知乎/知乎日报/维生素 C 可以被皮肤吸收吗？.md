<div class="question">
维生素c可以被皮肤吸收吗？

<div class="answer">

<strong>
<img class="avatar" src="https://pic2.zhimg.com/v2-5ec2a04d747dff9845713ffe0964954b_l.jpg?source=8673f162" referrerpolicy="no-referrer">

<a href="https://www.zhihu.com/question/263851944/answer/2111687642" class="originUrl" hidden>查看知乎原文</a>
</strong>

<div class="content">
<p>当然可以，要不然满大街的维 C 精华都是智商税了！</p>
<p>几十年来各种研究都证明了维 C 的透皮吸收效果，还把维 C 的吸收条件研究的仔仔细细。今天我就来详细科普一下关于维 C 吸收的话题。</p>
<ol>
<li>维 C 能被吸收多少</li>
<li>皮肤能留住多少维 C</li>
<li>皮肤每天用多少维 C 合适</li>
<li>维 C 浓度越高越好吗</li>
<li>怎样才能让维 C 快速吸收</li>
</ol>
<p><strong>维 C 能被吸收多少</strong></p>
<p>受到伦理道德的制约，研究人员无法用人皮来做测试。于是只能用动物皮肤来了解维 C 的透皮吸收。目前比较详细的两个研究，一个用小鼠，一个用猪皮。两者的结果差异不大，因此可以作为很好的类比，来了解维 C 在人体皮肤上的透皮吸收数据。</p>
<p>在小鼠皮肤上，使用 1.3%浓度的维 C 溶液。每平方厘米皮肤的维 C 吸收速度达到了每小时 3.43 微克<sup>[1]</sup>。如果一个人的脸比较大（400 平方厘米），那么整张脸每小时能吸收 1.37 毫克维 C。白天八个小时，吸收量就是 11 毫克。注意这里的维 C 浓度只有 1.3%。如果按照市面上普遍的 10%浓度维 C 计算，一张脸八个小时就能吸收 84 毫克维 C，这可真多啊。一般来说，大家使用维 C 精华，一次也就用 0.1-0.2ml。如果浓度 10%，那么就有 10-20 毫克维 C。这点维 C，被脸吸收完全没问题。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-4ca77dd6a25247e08c06cf2223c0edfa_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>维 C 透皮吸收[1]</figcaption></figure>
<p>不过维 C 被吸收，有一部分透过皮肤，进入到体内了。而我们关心的只是停留在皮肤里的那些维 C，毕竟用维 C 是为了让它在皮肤里面抗氧化。否则直接吃维 C 片不香么？</p>
<p><strong>皮肤能留住多少维 C</strong></p>
<p>皮肤外面涂抹维 C，维 C 通过皮肤被身体吸收。因此使用之后，皮肤里面一定有维 C 存在。根据研究，猪皮里面维 C 的饱和浓度是 1000pmol/mg<sup>[2]</sup>。换算下来，每克皮肤里最多能容纳 0.176 毫克维 C。</p>
<p>小鼠皮肤的研究结果也类似，皮肤中维 C 的饱和浓度是 0.056mg/g。差不多是猪皮的三分之一。人类皮肤的数据虽然未知，但是应该也在同样的数量级。我们以脸部皮肤厚度两毫米计算，那么一张脸一共能容纳 8 毫克维 C。好像并不多的样子。毕竟涂抹一次维 C 精华，就能提供 10-20 毫克维 C。</p>
<p><strong>皮肤每天用多少维 C 合适</strong></p>
<p>既然脸上的皮肤最多才能容纳 8 毫克维 C，而一次维 C 精华就能提供 10 毫克以上的维 C，那么有必要每天都用高浓度维 C 吗？答案是不需要。</p>
<p>根据研究，在猪皮上连续涂抹 15%维 C，三天之后，皮肤中的维 C 就达到饱和<sup>[2]</sup>，接下来每天使用，皮肤中的维 C 含量并不会变化。多出来的维 C 要么透过皮肤进入身体里，要么直接停留在皮肤外面浪费掉了。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-dcaf6d745ccc3f1654a8d4223ba09101_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>维 C 在皮肤中饱和[2]</figcaption></figure>
<p>当皮肤中的维 C 饱和之后，即使停用维 C，皮肤中的维 C 含量也能维持在一定水平，五天之内都不会代谢完全。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-a583a692537df51a0cd1376a33f51936_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>停用之后维 C 缓慢下降[2]</figcaption></figure>
<p>所以正确的做法是，连续使用高浓度的维 C 三天，让皮肤中的维 C 浓度达到饱和。随后每天可以减少用量，或者使用低浓度的维 C 即可。如果你手上维 C 精华的浓度是 15%，那么三天之后，把量减少到原来的三分之一，或者直接用 5%的维 C 精华，就可以维持维 C 浓度的饱和。当然，还可以三天打鱼两天晒网，用一天维 C 再停用两天，也不会有问题。</p>
<p><strong>维 C 浓度越高越好吗</strong></p>
<p>维 C 浓度并不是越高越好。浓度太高，一方面是浪费，维 C 在皮肤中很快达到饱和了。用的再多，皮肤就像吸饱了水的海绵。也不会带来额外的作用。没被吸收的维 C 反而可能氧化染色，让皮肤变黄。</p>
<p>另一方面，根据研究，出于某些未知的原因，维 C 在 20%浓度下，透皮吸收的效果最好。浓度提升之后，皮肤中的维 C 含量反而下降<sup>[2]</sup>。</p>
<figure><img class="content-image" src="https://pica.zhimg.com/v2-bfa0cb2778b0beeb4e6ff8f8a7976651_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>维 C 浓度和吸收的关系[2]</figcaption></figure>
<p><strong>怎样才能让皮肤快速吸收</strong></p>
<p>根据维 C 的吸收，研究人员做了个物理模型。最后发现，阻碍维 C 吸收的最大因素是角质层。因为维 C 是亲水的，而角质层尤其是细胞间脂质是亲油的。一般来说，油溶性的东西更容易被吸收。而维 C 本身分子量 176，也不算非常小，所以角质层对维 C 的透皮吸收有一定的阻碍。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-0b2fac1a42cb71ea17115b9192c5aa19_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>维 C 吸收的物理模型[1]</figcaption></figure>
<p>在动物实验中，如果把小鼠的角质层剥离，那么维 C 的吸收会有数量级的提升。从每平方厘米每小时吸收 3.43 微克，提升到 33.2 维克，提升了将近十倍<sup>[1]</sup>。</p>
<p>所以如果你想维 C 快速吸收，那么可以用一些果酸类的产品，让角质层边薄一些，这样维 C 就可以更好的吸收。</p>
<p>这个结果从另一方面也说明，皮肤屏障受损的敏感肌更容易受到维 C 的刺激。因为能被吸收的维 C 更多，更容易让敏感肌产生不适。</p>
