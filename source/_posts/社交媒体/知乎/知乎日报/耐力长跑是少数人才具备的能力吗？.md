<div class="question">
有哪些常识最后演变成了冷知识？

<div class="answer">

<strong>
<img class="avatar" src="https://pica.zhimg.com/v2-a19cbcedc9a8ca57f91f6e1f90346f46_l.jpg?source=8673f162" referrerpolicy="no-referrer">
<span class="author">ASICS亚瑟士，</span><span class="bio">Sound Mind, Sound Body</span>
<a href="https://www.zhihu.com/question/55006387/answer/2322544511" class="originUrl" hidden>查看知乎原文</a>
</strong>

<div class="content">
<p>一个被多数人误认为冷知识的常识——</p>
<p><strong>耐力长跑并非只是少数人才具备的能力，它一直刻在每一个人的基因里。</strong></p>
<p>那些能轻松完成五公里、十公里甚至一个马拉松的人，不见得有异于常人的身体，他们可能只是通过持续（且未必那么漫长）的训练，激发出原本每个人都存在的能力。</p>
<p>只不过多数人都没有意识到自己具有这种能力，展现出来的反而被当成了少数派。</p>
<figure><img class="content-image" src="https://pica.zhimg.com/v2-cdcf4db5796267c4180d049dd05fd74b_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>并且，和很多技巧 / 力量 / 灵活性的运动不同，耐力长跑的能力会伴随生命更长久时间。</p>
<hr>
<p>相比四足行走的动物，人类在绝对速度方面毫无优势可言——安排一条百米赛道，即便最快的短跑选手，也会被猎豹、大象、熊、狗、长颈鹿、马……等远远甩在身后。但如果把终点设得更远些，人类就有机会以比它们更慢的速度追赶、最终追上并完成反超。</p>
<p>这种耐力跑步的能力，一直往前追溯要到几万年前的非洲。原始人类实现直立行走后，慢慢开始使用跑步这一方式进行行动。一来跑步的速度更快，能帮助他们更高效的觅食，二来人类逐渐将跑步用到狩猎活动中，以更长时间的奔跑赢下与猎物的追逐战。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-1e89537d97143dc51c417fbecbdda818_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>从发现的壁画中可以看到，原始人类狩猎的对象往往是比自身力量更大、速度更快的四足动物，但他们最终以不懈的奔跑追逐包围，完成狩猎。图 /wiki</figcaption></figure>
<p>原始人类进化过程中，身体功能和结构上出现的很多变化，也都是为了更好的适应跑步。甚至于在今天，很多进化被认为与直立行走并没有太大关系、而只服务于跑步。</p>
<p>比如哈佛大学人类进化生物学家利伯曼就说过，“人类很多进化与行走无关，它们只是适应性调整的结果。”“当人类具备长时间奔跑的能力，就能一路追赶动物，直到后者疲惫 / 中暑 / 力竭而倒下。这样就获得了丰盛的食物。”</p>
<p>所以这也从另一方面证明了跑步不见得是步行的副产物。</p>
<p>（敲黑板：「走得快了自然会跑起来」的说法并没错，但跑步和走路本质上是两种区别很大的动作）</p>
<p>那原始人类身体究竟发生了哪些变化，从开始直立行走再到可以跑上很久？有一个<strong>耐力跑假说（Endurance running hypothesis），其中最关键的几点——</strong></p>
<ul>
<li><strong><em>低功耗长时间跑步的关键，肌腱和韧带是“储能弹簧”</em></strong></li>
</ul>
<p>人类双腿有非常奇妙的“储能弹簧”，肌腱和韧带。跑动中它们被拉伸，而后在推蹬时产生向前的力。</p>
<p>在相对快速的奔跑中，肌腱和韧带交换势能和动能，以较低的功耗运作，来确保人类能维持该速度长时间跑下去——而在行走中，肌腱和韧带显然并不会发挥这样的效果；</p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-2d62ac5fa7e54bcbb9941f02aef8ed8c_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong><em>降低跑步的冲击力，更大的关节表面积</em></strong></li>
</ul>
<p>跑步中，身体会承受身体 3-4 倍于体重的冲击力。相比同为人科动物的南方古猿，人类的关节表面积（特别是下半身的骨骼关节）明显更大一些，来应对压力。这样每一次落地，身体自身结构就得以更好的去吸收和分解压力。</p>
<p>与关节表面积类似，人类的足底足弓、跟骨结节和跟腱等也都为提供更好的跑步经济性做出了“升级”。比如相比旧石器时代的尼安德特人，现代人显然可以轻松赢得与祖先的赛跑。</p>
<ul>
<li><strong><em>堪称完美的体温调节能力 + 散热效率</em></strong></li>
</ul>
<p>通常这也被认为，经过长期训练的人类能在耐力跑中完胜其它动物的关键原因。</p>
<p>大多数哺乳动物都依靠蒸发汗液来维持体温，每单位表面积下，人类汗腺的排液效率远高于多数动物，加上体毛逐渐褪却，散热能力进一步增强。</p>
<p>这就好比，很多人都知道世界上跑得最快的动物是猎豹，最高时速超过 100 公里 / 小时。然而猎豹全力奔跑速度往往只能维持很短时间（整个捕捉过程不超过 1 分钟），且一天只能进行 6 次这样的奔跑捕捉，否则很可能因为能量消耗过大、体温过高直接倒下——给人类足够的距离，跑赢猎豹并不是多困难的事。</p>
<p><em>延伸阅读：</em></p>
<p><a class="internal" href="https://www.zhihu.com/question/406235763/answer/2068285713">气温升高对跑步的影响大不大？</a><a class="internal" href="https://www.zhihu.com/question/24178961/answer/1631743896">运动大量出汗对身体好吗？</a></p>
<ul>
<li><strong><em>直立让呼吸可根据步态灵活调整频率</em></strong></li>
</ul>
<p>并且人类还进化出符合跑步需要的呼吸方式。四足动物奔跑时，每个运动周期里只能呼吸一次，导致它们的跑步步态和呼吸频率只能协调为一种节奏。但直立的人类胸腔并不会在跑动中被压缩，因而就可以根据步态灵活调整呼吸频率。</p>
<p>口鼻并用的呼吸，也能帮助增加每次更换气体的体积量。</p>
<p><em>延伸阅读：</em></p>
<p><a class="internal" href="https://www.zhihu.com/question/356284986/answer/1101175506">跑步用鼻子还是嘴呼吸?</a></p>
<ul>
<li><strong><em>长跑中，人类会开启稳定且持久的供能系统</em></strong></li>
</ul>
<p>人体三大供能系统，磷酸原（ATP-CP）和糖酵解的供能时间都较短，但靠有氧系统燃烧脂肪（外加糖和蛋白质）的供能方式，虽然输出功率较低，却可以长时间稳定的提供能量。我们日常慢跑、包括更远距离的马拉松，有氧燃脂都是最主要的供能方式。</p>
<p>外加人类进化出了更大的甲状腺和肾上腺，利用碳水化合物和脂肪酸中能量的能力也进一步提高。所以某种程度上，只要强度合理，人类能持续奔跑的时间远远超出想象。</p>
<p>人类这套专为长时间运动打造的有氧系统，较身体其它功能也有更长使用时间——坚持锻炼的话，有氧能力的巅峰可以一直维持到近 60 岁。耐力运动（如铁人三项 / 马拉松 / 超马越野 / 长途骑行）选手也普遍都有更长的运动生涯。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-61bcdf0690c8dc872e0c2123d7573599_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>已经 38 岁、刚刚在休斯顿马拉松刷新全美女子半马纪录的萨哈·霍尔。无数高龄选手的纪录都证明，人类在有氧耐力方面的能量使用，能确保很长时间都处于优秀的跑步状态中。</figcaption></figure>
<p>以上长跑者的基因，一直储存于人类的生命里，只待你开启。</p>
<p>尽管不以速度见长，但在耐力方面，人类可谓一直是陆地动物中的佼佼者，即使在现代，非洲一些部落捕猎也依然依靠长跑追逐包围的方法。当然在当下，我们已经完全无需依靠奔跑去获取能量，相反更多人期望用跑步实现减脂健身的愿望。</p>
<p>不管怎样，不要对长跑心生恐惧，用更为积极的心态去开始跑步，也应该有信心，每个人都可以唤醒自己原本就具备的跑步能力。</p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-13d642ca280c0af65c7118250d412cf1_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>——即便到马拉松这样看似遥不可及的挑战，很多人从开始跑步到站上起跑线，往往并不遥远。</p>
