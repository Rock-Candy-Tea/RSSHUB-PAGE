<div class="content">
<blockquote>铁三车和公路车的区别，相当于 SUV 和双门跑车之间的区别。虽然都是两个车轮子，但它们在设计上由于功能和目标不一样，导致公路车和铁三车相同点仅仅是“都是在公路上骑行”这一共同点。<br>我们将从使用场景 / 骑行环境 / 骑行姿态 / 功能需求 / 运动员需求几个点展开来说。<br></blockquote>
<p><strong><br><strong>使用场景</strong><br></strong></p>
<p>首先在使用场景上，基于目前的竞赛规则，<strong>公路车是可以用在铁三比赛中，而铁三车是不能用在公路车比赛中的</strong>。</p>
<p>基于当初是酒吧里一群喝多了的人们想出来的比赛，铁三比赛自诞生之初就是有“玩”的性质，惟一的要求就是个人独立完成，所以比赛规则主要围绕是<strong>“不能借助外力”</strong>。</p>
<ul>
<li><strong>公路车</strong></li>
</ul>
<p>公路车为了适应比赛的规则和场地特性以及 UCI 规则的条条框框，对于动辄 2000+ 公里、需要翻山越岭，下坡随便 80+km/h 的环法国自行车赛、环意大利自行车赛等世界级比赛来说，公路车不仅仅是为了速度而生，还要兼顾空气动力学、操控性、舒适性，适山地、丘陵、城市等不同的路面和路况，然后<strong>在 UCI 的规则限制下</strong>车架的基本形状、几何角度、重量、管型的长宽比去设计，再根据不同品牌对公路车哲学的不同理解，从而诞生各个品牌类似而又各不相同的公路车。</p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-b096e788ed63120cc306566a0c2efb0d_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>铁三车</strong></li>
</ul>
<p>和公路车的在条条框框下追求全面全能的设计出发点不一样的是，铁人三项赛的赛程都是固定的，比赛距离根据类型分为 40km/90km/180km，大部分赛事都会在比较平坦或者缓和的丘陵地形上骑行，影响比赛结果的因素只有一个：完成比赛的时间。</p>
<p><strong>速度取胜的前提下，铁三车对于空气动力学（低风阻）的设计需求超越了舒适性和操控性</strong>，并且由于赛事不可借助外力的宗旨下，铁三车设计的时候还需要考虑到运动员长距离骑行的能量食物补给存放，水分补充和其便利性，骑行运动时身体肌肉的利用和恢复，另外由于铁三赛事不在 UCI 的管理下，所以<strong>铁三车的设计无需遵循 UCI 的自行车设计规则</strong>，于是诞生了完全不同于公路车的新型竞赛车种。</p>
<figure><img class="content-image" src="https://pica.zhimg.com/v2-d9947a0a4903ce613f4500d75fd5a733_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong><br><br><strong>骑行环境</strong><br><br></strong></p>
<ul>
<li><strong>公路车</strong></li>
</ul>
<p>自行车比赛经常会出现的场面是一大群人像一群大雁一样排在一起骑行，而速度往往在 40km/h 以上。自行车手们之间的距离真的是用“摩肩擦踵”来形容都不为过，稍不留神就导致摔车，而且还会连带周边的车手一起摔车，对此，操控的灵活性决定了自行车运动员不会成为“摔车团灭”的发动机。对于“皮包铁”的自行车运动，摔车轻则损失大量时间（可能就和总冠军失之交臂），重则骨折赛季报销乃至职业生涯报销，所以操控性能对于公路车来说非常重要。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-9711117f28e296bc907749593ccd2184_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>铁三车</strong></li>
</ul>
<p>以 IRONMAN 为例，铁人三项赛的独特在于它的比赛规则是<strong>不允许跟车</strong>（国际铁联的 ITU 比赛因为可以跟车，所以只能使用公路车比赛），所以对自行车的操控性要求大大降低，所以铁三运动员可以使用操控性很差但是比较偏重休息的趴低姿势骑行，所以铁三车的车把也叫<strong>“休息把”</strong>。</p>
<p>除了进出换项区和水站补给，有些比赛的赛道一马平川，甚至连刹车都很少用到，所以运动员可以“趴”在铁三车上完成骑行赛段，对此，铁三车的几何设计就会偏向长轴距，让铁三车更稳定，以弥补操控性不佳的缺陷。</p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-a0d43ffa23cc673bb042c62d77917ce6_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong><br><br><strong>骑行姿态</strong><br><br></strong></p>
<p>公路车和铁三车很重要的区别在于骑行姿态上。</p>
<ul>
<li><strong>公路车</strong></li>
</ul>
<p><strong>公路车的几何设计让骑行者能够更好的操控自行车，在任何时候都可以使用更多的身体去参与</strong>，所以看似只动腿的公路车骑行其实也会动用核心和上肢的力量，让骑行者不管是平路、山路、上上下下的丘陵地形都能操控自如。同时为了适应比赛的节奏，公路车骑行姿态还能动用更多肌肉群，输出更大的功率数，在比赛关键时刻使出致命一击。</p>
<figure><img class="content-image" src="https://pica.zhimg.com/v2-6a63958bae71d787b831b357764a8240_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>铁三车</strong></li>
</ul>
<p>由于<strong>铁人三项运动第一项是游泳</strong>，<strong>对上肢力量消耗比较大，所以进入骑行赛段，运动员采用趴低的姿势手握休息把骑行</strong>，可以让上肢肌肉进行恢复，同时放低上半身的高度还可以降低迎风面积，大大减少空气阻力，骑行运动的主肌肉群也可以和跑步运用的主肌肉群错开，达到节省体能的目的——这<strong>也是为最后一项跑步做体能储备</strong>。</p>
<figure><img class="content-image" src="https://pic4.zhimg.com/v2-4d7a80fff2970284daf90996991f92c4_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong><br><br><strong>功能需求</strong><br><br></strong></p>
<ul>
<li><strong>公路车</strong></li>
</ul>
<p>公路车比赛地形多变，在山地爬坡阶段，自行车要足够轻来减轻运动员的功率输出，达到减少体能的目的；<strong>在最后冲刺阶段，职业运动员往往会输出 1000w 以上的恐怖数据，所以要求车架的刚性要足够好来将运动员的力量转化为速度</strong>。比赛级别的顶级公路车也会贴着 UCI 的最低整车重量 6.8kg 来设计比赛用的公路车。所以在公路车的设计上更最求刚性 / 重量比。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-811f4959688c0ebaa0883873576c19fe_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>铁三车</strong></li>
</ul>
<p>因为<strong>高功率输出会增加肌肉的疲劳，所以铁三运动员则尽量避免输出高功率的输出。</strong>一名高水平的铁三运动员<strong>FTP 为 350w</strong>，那么在比赛中 80%以上的时间也仅仅在有氧区间 220-260w 功率输出，仅仅在超车或者爬坡上才会有 400-500w 的输出范围，所以铁三车无需过分最求刚性重量比。</p>
<p>另一方面因为没有 UCI 的车架管型设计约束，铁三车的设计可以说是天马行空，管型尽量符合流体力学以减少空气阻力，会变得更粗壮；车重无所谓因为铁三比赛一般以平路为主，追求极致空气动力学性能的运动员甚至会用上重量更重但空气阻力更小的后碟轮和前三刀 / 四刀轮来最求边际效益。</p>
<figure><img class="content-image" src="https://pica.zhimg.com/v2-215065ca8abafc6d3f12457994dd9b92_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong><br><strong>运动员需求</strong><br></strong></p>
<ul>
<li><strong>公路车</strong></li>
</ul>
<p>对于职业赛事，公路车赛一般会有后勤车，车上会有技师，可以快速解决爆胎换轮等一些机械故障问题，所以在比赛中还有专门运水的车手负责到集团后方的后勤车里取水，所以公路车上一般会有两个水壶架，仅够运动员 1-2 小时的补给饮用。到了比赛最后阶段，车手都会把水壶扔掉以减少空气阻力和一点重量。</p>
<figure><img class="content-image" src="https://pica.zhimg.com/v2-961259e0f91379ae5998dd0538d73f10_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>铁三车</strong></li>
</ul>
<p>由于赛事规则限制，铁三运动员仅能从赛事官方的水站获取水分和部分能量补给，并且在遭遇机械故障时只能等待官方援助（黄花菜都凉了）。所以大部分运动员都需要自己携带能量胶 / 能量棒，适合自己的运动饮料同时为了满足在休息把上的喝水。</p>
<p>很多品牌都会设计独立的符合空气动力学设计的饮水系统以提供给运动员补给，同时车架上还有设计补给盒仓位，可以让运动员携带能量补给以及部分自行车维修工具等——而这些东西，在 UCI 的规则里，是被禁止的。</p>
<figure><img class="content-image" src="https://pica.zhimg.com/v2-914abbae856c1e15e887e3ce45326fec_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>以上，希望可以帮助解开关于铁三车和公路车的疑惑。</p>
