<p>其实用 5G 和 WiFi 做对比不是很恰当，因为 5G 是「<strong>第五代」移动通信系统</strong>，而 WiFi 其中包含 802.11/a/b/g/n/ac/ad/ax 等等很多「<strong>代」</strong>版本，这样对比有点像在问特斯拉和火车有什么区别。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-4e2703d7224b017a6dd05202a86cc4da_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>WiFi 协议版本及参数 (表格来自 Wiki)</figcaption></figure>
<p>那么，更广泛来讲，从同一维度来做对比，<strong>移动通信系统(xG，x = 1,2,3,4,5)</strong>与<strong>我们现在日常使用</strong>的 <strong>WiFi</strong> 有什么区别？</p>
<blockquote>Note：本回答主要针对的是私人家庭 WiFi。</blockquote>
<p><strong><strong>xG 与 WiFi 的区别</strong></strong></p>
<p>当然，作为用户来讲，我自己最直观的体验是 WiFi 比 xG 要便宜很多，如果我们忽略有线宽带和路由器费用，甚至可以认为用 WiFi 连接互联网是免费的。然而大部分情况下，价格只能反应一部分技术因素。<strong>如果把一个小小的家庭网络，扩展到全国和世界范围就是 xG</strong>。不过这个大范围和小范围之间，还有很大的区别。</p>
<p><strong>为了简单描述这两者的不同，我们需要先从需求说起。</strong></p>
<p><strong><strong>需求区别</strong></strong></p>
<p><strong>竞争性</strong></p>
<p>就 WiFi 和 xG 来说，技术上，他们的区别有点类似<strong>区域自治</strong>和<strong>中央集权</strong>，这种思路导致了大部分 WiFi 节点都是私人(或者公司，或者城市)建设，而 xG 基站是运营商在全国建设。</p>
<p>换句话说，就是在无线信号传输过程中，因为各个私人路由器之间没有交流且共享相同频谱，所以 WiFi 的数据传输是<strong>竞争性的</strong>，而 xG 的数据传输是<strong>非竞争的，有中心化资源调度的</strong>。</p>
<blockquote>这大概就像公路和铁路，开车出门上路，我们不知道下一个路口会不会突然出现一长串闪着红色尾灯的各种汽车堵在前面；而铁路就不会有这种困扰，前方多远有车，最高可以跑多快都有中心调度系统告诉司机。</blockquote>
<p><strong>私密性</strong></p>
<p>同时，一般意义上，WiFi 连接的是私人有线宽带，而 xG 的基站连接的是运营商的骨干网，因此，<strong>WiFi 普遍会有私密性要求</strong>，不能未经许可随意接入。</p>
<blockquote>(不能白看，也不能白... （:</blockquote>
<p><strong>移动性</strong></p>
<p>因为 WiFi 连接的是私人有线宽带，私人宽带接入点固定，同时宽带是有线的，它不会到处乱跑。这意味着 <strong>WiFi 对移动性需求很低，覆盖范围小</strong>，一般只用考虑步行速度对信号传输的影响，不考虑小区切换，而 <strong>xG 的基站存在很高的移动性和小区切换需求(基站覆盖范围称为小区)</strong>，需要考虑比如汽车，火车等高速物体。</p>
<blockquote>除非汽车本身有个 WiFi，不然应该是没有人在汽车上一直连着 WiFi 的吧？目前 WiFi 有 WDS 模式，但是还是不如运营商的小区切换成熟。</blockquote>
<p>这样的<strong>竞争 / 非竞争，私密性和移动性</strong>要求会带来一系列从<strong>功能，技术到覆盖，接入，频谱，速度</strong>等等的不同。</p>
<p><strong><strong>技术区别</strong></strong></p>
<p><strong>1.频谱 / 接入</strong></p>
<p>频谱或许是竞争性最直接的导火索。</p>
<p>WiFi 使用的频谱(2.4GHz/5GHz) 是非授权频谱，就是说这段频谱并未分配 / 拍卖给个人或公司，任何人 / 企业都可以用自己的 WiFi 设备随意接入。xG 使用的频谱是授权频谱，除了获得该频谱的运营商，其他人都无权使用此频谱。</p>
<p>所以，现在大家一打开手机 WiFi，就会立即看到很长很长的无线列表，大部分都是 2.4GHz 路由器。这意味着此频段非常拥堵，可能存在非常多的 noise-like interference 。</p>
<blockquote>这段的意思是，如果其他技术相同，此频段的手机，WiFi 信噪比会比较低，这会导致 WiFi 信号覆盖范围缩小，传输速率较慢。因此，目前 WiFi 协议都在扩展 5GHz，60GHz 等等干扰较低的频段。</blockquote>
<p>有辣么长的列表，而 WiFi 的频段是有限的，一定会产生信道资源竞争。所以，WiFi 最核心的空口协议是 CSMA/CA（Carrier Sense Multiple Access/Collision Avoidance）。它的具体做法是<strong>发送前对信道做检测，如果信道忙碌，那么等一个随机时间再发送。但是检测不是实时的，所以依然有可能存在两个路由一起检测到空闲频谱，同时发送数据，这时候就产生了碰撞问题，会采取重传的方式再次传输。</strong></p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-0cc2b4535460295533735381a6808e64_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>CSMA/CA</figcaption></figure>
<p>而 xG 中，因为接入信道是由基站分配的，而且在分配算法中会考虑干扰因素，所以相比较之下，技术相同的基站覆盖范围会大。同时，<strong>xG 在信号传输之前，已经被基站分配了专属「线路」，所以不需要发送前信道检测，对碰撞重传的要求也很低。</strong></p>
<p>另外一个有关接入的显著不同是，因为运营商需要全地点接入，所以 xG 是不存在密码的，它们采用 SIM 卡中的身份识别，通过收费网关收费。而私人 WiFi 通常是需要密码的。</p>
<p><strong>2. 覆盖，WiFi 组网和移动通信网络</strong></p>
<p>上文说到了 WiFi 的覆盖范围一般都会很低，相比较之下基站会因为发射功率高，频段干扰低，所以覆盖范围大的多。</p>
<p>因为网络速度可能会受太多因素影响，所以这部分不讨论 WiFi 和 xG 的速度问题，实际上上这两者谁比谁快都是有可能的。</p>
<p>但是比如公司大楼，如果想要扩大 WiFi 覆盖范围，以方便服务公司员工。这种情况下，单个无线路由器肯定是不行的，覆盖公司大楼的单个无线路由器一定会超过国家规定的无线电发射功率，所以需要多个路由器联合组网，比如一个无线路由器负责一个房间，同时其他路由器采用同样的名称，并相互协作，这样形成整个大楼的无线网络。</p>
<p>我们都知道，<strong>单节点决策的系统是最高效的系统</strong>。也就是说，如果无线网络中，如果存在多节点协作，最有效率的方式是有一个全网络的控制器，来帮助各个路由器调度和分配时 / 空 / 频谱资源。</p>
<p>在 WiFi 网络(WLAN)里, 就是把家庭路由器中集合一体的<strong>AP(Access Point，接入点)和 AC（Access Controller，控制器)分离</strong>。用 AC 控制全网，并分配资源。</p>
<p>如下图：</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-b0ba0ce77a28c9fe92ac9f94124584f9_720w.png?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>WLAN 设计，图来自 summer 的回答 - 知乎 https://www.zhihu.com/question/59991119/answer/228596693</figcaption></figure>
<p>那么如果再扩大一点呢？</p>
<p>上升到全国范围，单个 AC 显然数据处理速度是不够的，那么每一个区域需要一个类似的 AC，并且各个 AC 之间也需要协同工作，互通有无。这就形成了无线核心网(Core Network)。</p>
<p>而各个 AP 就形成了无线接入网(Radio Access Network)。</p>
<p>运营商的移动通信网络，主要由核心网和接入网两部分组成。</p>
<p>如下图，这样与无线路由器组网(WLAN)是不是就很像了？</p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-ac0a5c2d8e998067b7af049e9eab15ea_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>5G 网络结构</figcaption></figure>
<p>从单路由，到公司级的多路由组网，到全国级的基站覆盖，这大概就是 WiFi 和 xG 之间的区别和联系。</p>
