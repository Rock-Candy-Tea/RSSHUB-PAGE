<p>大龄晚育的好处就是，你身边的姐妹基本上都已经生过孩子了，大把经验可以分享。</p>
<p>产前妈妈最关注的就是待产包和新生儿用品的准备，我生宝宝之前，几个闺蜜都把当时自己使用的待产包分享给我，大家还一起在群里进行讨论，迭代，更新。自己生完孩子，实际应用之后，与居家的十年经验的金牌月嫂进行讨论，又对经验进行了一次更新叠加。</p>
<p>全文有点长。为啥这么啰嗦呢……因为，我整个孕期也在网上到处搜罗待产包的准备，拿到不少列表和表格，但是面对那些物品的清单，一个从没有照顾过小孩子的我发现自己是如此一脸蒙圈，溢乳垫？？隔尿垫？？？<strong>这些都是什么鬼？？？？</strong>很多东西我连名字都没听说过，更不知道是干什么用的，所以……我无法判断自己是否需要。等后来自己亲身经历了这个过程，才发现门道居然这么多！！</p>
<p><strong><strong>所以，我写的这个介绍，尽可能的也介绍一下每个东西的用途，适合人群，和采购的关键点，希望一站式帮助姐妹们解决困惑。</strong></strong></p>
<hr>
<p><strong><strong>Part 1：带到医院</strong></strong><strong>产妇用</strong></p>
<ul>
<li><strong>个人洗漱用品——必备！（不洗脸的人除外）</strong></li>
<ul>
<li>住院的几天时间，按照自己的日常所需，带好自己的洗漱用品。牙膏牙刷洗面奶擦脸油之类的。</li>
<li>强烈推荐一次性洗脸棉，棉柔巾，干净卫生，洗脸擦桌子怎么都好用。目前用过 N 多款，最推荐 ITO 这个，厚实，带颗粒感，洗的干净，耐用，别的有些牌子洗一次就 OVER 了，这一款，一张大概洗三次都可以。孩子出生以后，用得到的场景更多，必囤！</li>
</ul>
</ul>
<ul>
<li><strong>个人证件 + 医疗资料——必备！（谁也不除外）</strong></li>
<ul>
<li>身份证、母子健康证、之前所有的产检材料等</li>
</ul>
<li><strong>弯头吸管——必备！——提前备好</strong></li>
</ul>
<blockquote>对于产妇来说非常实用，因为不论是顺产还是剖腹产，在一段时间内都不能轻松的起床，而躺卧在床上需要喝汤喝水的时候，小小一根吸管就能展现出巨大的作用。</blockquote>
<ul>
<li><strong>吸奶器——背奶上班人群 + 奶牛体质必备，贫奶妈妈或者产后不打算背奶上班的酌情可以省略——提前备好</strong></li>
<ul>
<li>奶牛妈妈建议在预算允许的情况下，买电动双边的。在产后非常虚弱的情况下，又要频繁起夜喂奶或者吸奶，这个时候能省点力气真的是太重要了。强烈建议吸奶器也带到医院去，（我开始以为医院用不到，后来专门让家人回去取了），因为如果是奶牛体质，出奶比较快，有可能前几天就有涨奶乳房痛的情况，这个时候及时排空乳房很重要，大大缓解疼痛，也能避免堵奶。 相信我，哺乳是个大关，​​月子里两个乳房就像两个炸弹，任何让它俩舒服点的东西都值得。</li>
<li>另外这里提到奶牛体质：如何判断奶牛体质呢？据说，这个和遗传有点关系，自己妈妈奶比较多的话，自己奶多的可能性也比较大。但是也不绝对。</li>
</ul>
</ul>
<p>预算充裕可以买美德乐的这个高端款，吸力更大更稳定</p>
<p>一般情况丝韵就够了，我用的就是这个。</p>
<ul>
<li><strong>哺乳背心&哺乳衣或者柔软的家居服——非必需品，旧家居服撩起来也行，有了哺乳衣主要就是更美观方便。——产后再买来得及</strong></li>
<ul>
<li>强烈去医院的时候带上<strong>十分柔软的棉质哺乳衣（或者家居服）</strong>。医院的病号服往往比较硬，如果乳头被宝宝抓破了，穿着硬硬的衣服会感到很疼，有摩擦，这个时候一身十分柔软的哺乳衣也能够大大的缓解乳头的疼痛。</li>
<li>哺乳衣的挑选要点</li>
<ul>
<li>符合产后天气温度</li>
<li>一定要能够方便露出乳房</li>
<li>材质柔软舒适</li>
<li>​上翻盖，交叉式：目前看来这种是最美观大方实用的</li>
</ul>
</ul>
</ul>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-91b6996097647d3e1146ea62d382e2d7_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>上翻交叉哺乳衣，点击图片查看大图</figcaption></figure>
<ul>
<li>哺乳背心</li>
<ul>
<li>大家很多人都推荐美德乐的哺乳文胸，我在月子里，乳房很痛，还扎痒扎痒的，稍微紧一点厚一点的文胸都觉得很难受。试了很多都不舒服，但是哺乳期还不能不穿胸衣，因为要贴防溢乳垫胸衣上，不然漏奶会把衣服都弄湿了很难受。直到最后找到了<strong>贝莱康的哺乳背心</strong>，非常轻薄柔软毫无束缚感，简直是救命恩人。</li>
</ul>
</ul>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-2ce1a8b6085f0ec261b3ae849bbfc680_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>救命恩人哺乳背心</figcaption></figure>
<ul>
<li><strong>棉拖鞋（俗称带脚后跟）——可有可无。别冻着脚跟就行。</strong></li>
<li><strong>一次性内裤——必备！！——提前备好</strong></li>
<ul>
<li>月子里很疲惫，不舒服，会经常出血，还不能碰冷水，这个时候打算让谁来帮自己洗小内内呢？ 小小一点钱买一次性内裤，解决大问题，避免家庭矛盾。超值哦！​</li>
</ul>
<li><strong>溢乳垫 水洗 or 一次性——二者必备其一即可——提前备好</strong></li>
<ul>
<li>奶较多的产妇，哺乳期都离不了溢乳垫。尤其月子里，气虚，漏奶会比较严重，如果不想衣服上湿哒哒的，那就用上吧。</li>
<li>防溢乳垫分为水洗和一次性。</li>
<ul>
<li>长远来看，还是一次性更方便。用过美德乐，贝亲，子初等牌子，最后都败给了朋友送的微商品牌米菲。米菲的防溢乳垫是这几个里面最薄的。</li>
<li>我在月子里，乳房皮肤一度非常脆弱敏感，一次性的溢乳垫网面棉面都似乎对皮肤有点刺激，皮肤扎痒难受，买了一些布面水洗的，好多了，推荐贝莱康品牌。出了月子之后就没有这个问题。这个不是必须，根据个人情况。可以先买好一次性的，月子里试用一下，不行再买水洗的。​</li>
</ul>
</ul>
</ul>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-e82bb3fc7abe40ee9e594f0eee7f5de2_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>一次性的类似这样</figcaption></figure>
<figure><img class="content-image" src="https://pica.zhimg.com/v2-64c0718004c5fff76d4e431a12ab607e_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>可水洗的类似这样</figcaption></figure>
<ul>
<li><strong>乳头霜&乳头敷片——尽量准备（乳头不怕疼的可以不用）——提前准备好</strong></li>
<ul>
<li>产后在医院里的三五天时间就会开奶，初期的哺乳乳头很容易弄破，非常疼痛，这时候乳头霜就非常的管用；及时涂上，能够缓解疼痛，避免继续破。推荐 <strong>lansinoh 品牌</strong></li>
</ul>
</ul>
<ul>
<li>另外强烈强烈推荐一款<strong>睦清乳头舒缓贴</strong>。在乳头疼痛的时候贴上能够大大的缓解。 确实相当管用，最初的几天救了我。有点小贵，但是，缓解疼痛的时候觉得什么都值了。</li>
</ul>
<ul>
<ul>
<ul>
<ul>
<li>当然避免乳头破裂最核心的还是掌握正确的哺乳姿势：<strong><em>要点记住三个字：捏！扁！塞！</em></strong>用手将乳头捏得扁扁的，在孩子嘴张得最大的时候送进去，尽可能让孩子多一些的含住乳头或整个乳晕，不要只嘬乳头一点点记​</li>
</ul>
</ul>
</ul>
<li><strong>护垫 or 卫生巾——必备！！——随时买都行</strong></li>
<ul>
<ul>
<ul>
<li>个人感觉月子卫生巾是个伪概念，我也备了一堆，其实一点也不好用，似乎只是更厚的卫生巾，还没有护翼。 产后的恶露，大致就理解为一次长时间月经来对待就好。产后一周内，出血量比较大，之后就少多了。建议使用​超薄加长护垫，0.1 厚度的那种就好。轻薄一点比较舒服。</li>
</ul>
</ul>
</ul>
<li><strong>好吃的，好玩的——随便你！</strong></li>
<ul>
<ul>
<ul>
<li>进待产室的时候，带上各种能让你觉得舒服的东西吧，好吃的；手机；pad……那段难熬的时光，任何让你多一点点舒适感的东西都值得</li>
</ul>
</ul>
</ul>
</ul>
<hr>
<p><strong>宝宝用</strong></p>
<ul>
<li><strong>衣服——随时买——提前一两件备好！</strong></li>
<ul>
<ul>
<ul>
<li><strong>强烈建议大家不要在产前准备太多的衣服！！！</strong>只需要最小号 52 的，来三身即可。因为孩子真的长的<strong>很快很快很快！个体差异也很大很大很大；</strong>而且天气也在变化，你几乎很难预测它在什么天气的时候能长到多大；我家宝贝就是比较大个子的，两个半月个头 64 了，别人家三个月的衣服，我们已经不能穿了。</li>
<li>一个小技巧是，把你喜欢的衣服放在购物车，配合孩子的成长，到时候再买。​</li>
<li><strong>目前我们认为最方便实用的新生儿衣服要点是：</strong></li>
<ul>
<li>连体！！不要分体！！</li>
<li>不要套头！！​前开扣或者侧边系带！！</li>
<li>不要后面扣子的（孩子躺着会硌）！！</li>
<li>内边无骨设计无骨无骨！！</li>
<li>要连档不要开档！！！（很奇怪对不对？实践情况就是，开裆裤很容易在小朋友动个不停的时候磨蹭到尿不湿，造成漏尿，最好就是连档且档那里有扣子，方便穿脱的：小孩子一天换十来次尿裤，穿脱方便相当重要！）​</li>
</ul>
</ul>
</ul>
</ul>
</ul>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-816b59d497782e2c34dba75ac6bc2cd5_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<ul>
<ul>
<li><strong>帽子——必备！</strong></li>
<ul>
<li>棉质，柔软的，小朋友是很柔弱的，从医院出来，回家路上一定护住脑袋，带个软软的小帽子。不过一般医院都有赠送。</li>
</ul>
<li><strong>肚兜——必备</strong></li>
<ul>
<li>推荐可伸缩肚兜或者系带肚兜，小朋友腿脚凉一点没关系，小肚肚只要不受凉就好</li>
<li>孩子长大一点，可以竖着抱了，伸缩的肚兜总是会被曾到上面去，老妈发挥聪明才智，给加了两个松紧带！棒棒哒！</li>
</ul>
</ul>
</ul>
</ul>
<ul>
<ul>
<ul>
<li><strong>包被——必备——提前准备好</strong></li>
<ul>
<li>根据天气准备带孩子出院时候外面包裹的衣服或者被子。</li>
<li>有些妈妈会准备防惊跳包被或者睡袋，因人而异，我家都是用布包住小朋友，效果也不错，没有包被。</li>
</ul>
<li><strong>袜子&鞋子——必备——提前准备好</strong></li>
<ul>
<li>我们老家那边的人讲究小孩子的脚不要冻着了，尤其是脚心，否则会拉肚子。所以如果不是在大夏天出生的小朋友，还是很必要穿袜子的，选择柔软的棉质袜子，开口要松一些就可以。</li>
</ul>
<li><strong>奶瓶——必备——提前准备好</strong></li>
<ul>
<li>最小号 SS 尺码的奶嘴；奶瓶是需要带到医院的，万一出奶不好，是需要吃一点奶粉的。推荐大众品牌：贝亲。预算多一点可以买可么可多，这个品牌的奶瓶圆圆的，手感很好。不会打碎，防胀气。</li>
</ul>
</ul>
</ul>
</ul>
<ul>
<ul>
<ul>
<li><strong>奶粉——非必备——医院应该有——临时买也可以</strong></li>
<ul>
<li>出奶不好的话可能会用到少量奶粉，一般医院都有。出奶好的妈妈之后就不再需要奶粉了，奶少需要混合喂养的则持续需要。</li>
<li>生孩子之前可能并不知道自己奶的情况怎样，无需屯奶粉，调查好品牌，之后根据情况临时购买即可。</li>
<li>相当不推荐海淘奶粉：近年来海淘奶粉变成了很时髦的事情，好像不买点外国奶粉都不好意思跟人打招呼。不过我是相当不建议海淘代购，食物在飘洋过海的状况下，无人监督，总让人感到不安全。相比之下，我更愿意去购买旗舰店产品。</li>
</ul>
<li><strong>奶瓶清洗剂</strong></li>
<ul>
<li>贝亲或者子初等牌子，一般购买奶瓶会有清洗套装</li>
</ul>
</ul>
</ul>
<li><strong>尿不湿——必备耗材（打算洗尿布的除外）——提前准备好——不要屯太多</strong></li>
<ul>
<ul>
<ul>
<li><strong>小技巧：</strong>产前多收集不同厂家的一些试用品，到时候试一下哪个最合适。勤换尿不湿才是避免红 PP 的重要因素。 另外尿不湿也不要囤太多，NB 的两袋（100 片，两周），S 两袋即可。​ 新生儿一般一天用 10 多片，酌情带到医院​</li>
<li><strong>纸尿裤的选择要点</strong></li>
<ul>
<li><strong>薄厚：</strong>总的来说，纸尿裤当然越薄越好。但带来的负面作用就是容易侧漏。同样的人同样的操作方式，薄的款式的确侧漏的几率更大。不过首先确保轻薄，宝宝不宜红屁股吧。碧芭这款挺薄的。</li>
</ul>
</ul>
</ul>
</ul>
</ul>
<ul>
<ul>
<ul>
<ul>
<ul>
<li><strong>大小</strong>：每个牌子大小略有区别，有些宝妈反应自家宝宝比较胖，似乎大王更适合。</li>
<li><strong>方便买到。</strong></li>
</ul>
</ul>
</ul>
</ul>
<li><strong>湿纸巾——必备耗材——提前备好，但不要囤货太多。</strong></li>
<ul>
<ul>
<ul>
<li>有了小孩子之后，湿纸巾就是一个必备品，消耗量也非常的大，主要作用是给小孩子换尿布的时候擦屁屁用的，要厚实的，大一点的，上面有网格的，才能够方便擦拭。孕期参加母婴展销会，收集了不下二十种湿纸巾的试用装，用完一圈以后，<strong>目前最推荐的是 babycare，足够厚，布一样的，确实不连抽！老妈都非常喜欢。真的是墙裂推荐！！绝对不会后悔！！双 11 必囤！现在宝宝快 2 岁，仍然持续购买</strong></li>
</ul>
</ul>
</ul>
</ul>
<p>（绿盖子比紫盖子的便宜一点，实测稍微薄一点，但是使用体验基本一致，薄的似乎还更软一些，推荐买绿色盖子的）</p>

<ul>
<li><strong>PP 霜——尽量准备——提前备好</strong></li>
<ul>

<p><strong>PART5：居家改造——以下都是非必要选项，只是我觉得比较值得关注的，根据自家情况酌情安排</strong></p>
<ul>
<ul>
<li><strong>灯光调整</strong></li>
<ul>
<li>我们的居室里，灯大多都是悬挂在屋顶，小婴儿只能平躺，所以眼睛也只能看见房顶，如果家里的灯是白色光，很刺眼，那么对婴儿的视力发育很不好，宝宝也会感到很不舒服，所以要在侧面或者角落里配备一些柔和的光源。另外有了孩子之后晚上总会起夜喂奶，这个时候就需要一些随手可以打开，光线柔和，亮度适中的，可以帮助在夜里看清孩子，辅助喂奶的灯光。网上有许多专门的喂奶灯，可以考虑购买。另外有了孩子之后，很可能老人就要在身边帮助照顾孩子，晚上起夜的机会非常多，为了避免老人下床时摔倒，非常建议在床旁安装感应灯，<strong>我家的床边上和走廊里都装上了小米的感应灯，老人特别喜欢。超级推荐！！！最近公婆来家了，也特别喜欢</strong></li>
</ul>
</ul>
</ul>
<ul>
<ul>
<li><strong>洗衣机</strong></li>
<ul>
<li>小孩应该有自己独立的洗衣机。小孩的衣服很容易脏，最好有快洗和煮洗的功能。</li>
</ul>
<li><strong>小厨宝</strong></li>
<ul>
<li>强烈推荐家庭条件允许的情况下装上小厨宝，月子里女生不能动凉水，打开水龙头就能有热水，是非常重要的，另外给孩子擦洗屁屁的时候，经常我也是用湿纸巾再蘸一些热水用，避免湿纸巾太凉。</li>
</ul>
<li><strong>储物空间</strong></li>
<ul>
<li>有了小孩之后，家里物品的数量是几何式增长，所以在生孩子之前，请做好储物空间的规划，小孩子的东西都比较柔软比较小，细碎，小孩子的衣服不建议挂在衣柜里，因为都很小，一般也是棉质地的，叠放最节省空间。所以最实用的就是抽屉，我家族空间是比较充足的，但是在孩子出生之前还是添置了几组亚克力的抽屉柜。相比甲醛的污，同等体积的亚克力抽屉，因为比较薄，所以储物空间会更大，而且也不会有甲醛的污染。，很轻盈，推荐爱丽丝品牌，质量很好。无数人私信问我这个柜子的链接了～～～</li>
</ul>
</ul>
</ul>

<li><strong>双制空调 + 温度计</strong></li>
<ul>
<li>小朋友洗澡抚触玩耍，在小月龄的时候都需要严格控制温度，避免着凉。北方虽然有暖气，但是来暖气之前和停暖气之后很长一段时间，室内还是比较凉的，尤其是如果屋子朝向是向北，屋内是比较阴的。随时需要开空调，再加上一个温度计，把时问调整到最适合孩子的二十四五度左右，再从浴室里抱出来。</li>

<p><strong>Part5：其他重要的准备</strong></p>
<ul>
<ul>
<li><strong>联系好通乳师</strong></li>
<ul>
<li>坐完月子才明白一个道理，分娩只是生孩子的第 1 关，哺乳也有一个大关要过，尤其在月子里，堵奶之类的问题很让人困扰。建议提前联系好通乳师，了解一下有哪些是可以随时上门的。堵奶，产妇不方便出门，但是又会很疼，需要尽快的有通乳师来帮忙。如果家里请了月嫂，请确认月嫂是否具备通乳技能。非常重要。如果堵奶的问题没有解决，引发乳腺炎，孩子和大人都会很遭罪。</li>
</ul>
<li><strong>考察好家旁边的社区医院及环境</strong></li>
<ul>
<li>孩子出生满月后很快就要开始打预防针，这个时候产妇的身体比较虚弱，孩子也非常小，经不起折腾。，强烈建议提前考察好家旁边的设计院在哪里，去了解一下相应的程序，尽量确保到了医院后能够快速的打完针</li>
</ul>
<li><strong>安排好月子期间的居住问题</strong></li>
<ul>
<li>有了小孩家里不是仅仅添了一口人，很有可能也有老人要来家里帮忙照料，提前和家人沟通好。照料者是谁，考虑好如何安排居住的问题。各家对孩子如何睡都有自己的观点，可以选择在小床上，然后小床放在宝爸宝妈的大床边。也有家庭。，孩子是和老人在一个房间的，不论选择什么方式，都应该提前做好安排。</li>
</ul>
