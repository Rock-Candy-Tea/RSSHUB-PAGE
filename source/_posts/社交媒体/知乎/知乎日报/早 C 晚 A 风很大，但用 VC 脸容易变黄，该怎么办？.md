<div class="content">
<p>如果你看到了这篇文章，那么恭喜你，你将成为全中国最先了解“维 C 黄脸”真相的一批人。</p>
<p>关于用了维 C 为什么脸黄，目前中文互联网上的解释都很牵强。于是我去查了一下文献，发现事实远比想象的复杂。除了脸变黄外，手掌，指甲缝还有手指缝都有可能变黄，这种变黄没法用清洁产品洗掉，只能等皮肤慢慢代谢掉。停用之后，基本上一两个星期，皮肤变黄就会消失，而且你会发现肤色比用之前更提亮了一个色号。</p>
<p><strong>目前中文互联网对于维 C 变黄的解释很牵强</strong></p>
<p>关于为什么维 C 上脸之后，有些人会变黄，目前中文互联网的主流说法是维 C 的氧化。因为一些维 C 精华放久了之后，会呈现出比较深的颜色。于是自然而然的认为维 C 上脸之后被氧化，也是染黄皮肤的元凶。但是这过程中，大家常常忽略一个点，那就是维 C 精华的颜色并没有深到可以染黄皮肤的地步。</p>
<p>我手头正好有一瓶接近保质期的维 C 精华，大家可以看到颜色非常深。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-70b0ad00e76c377c0121a95f8250e065_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>把精华滴一点到手上，颜色对比非常明显，然而涂开之后，对皮肤的颜色根本没有影响。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-9fdbc7f5e02da7588c2c8dedf708e775_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>变黄的维 C 涂开之后对肤色没有影响</figcaption></figure>
<p>这是因为维 C 氧化之后，颜色虽然会变深，但是真正有颜色的成分非常少。这些成分又都是水溶性的，根本没法染黄我们的皮肤。</p>
<p><strong>用了维 C 脸黄的元凶竟然是它</strong></p>
<p>为了搞清楚用了维 C 脸黄的原因，我查找了大量文献，终于找到了合理的解释。维 C 染黄角质层的元凶其实是维 C 代谢的产物之一：赤藓酮糖。</p>
<p>看到赤藓酮糖这四个字，你的第一反应一定是：这是什么鬼？</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-f3f7b96e3663a952b5c73bfaf5bf21d2_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>实际上，赤藓酮糖在国外可是个大大有名的成分，它是欧美常用的美黑剂。</p>
<p>和亚洲人不同，欧美人喜欢颜色深的皮肤，这样的人被认为是有钱去海边度假，所以看起来给人感觉比较健康比较美。然而不是所有人都有钱有时间去海边度假，于是国外专门有一类美黑护肤品，而这个赤藓酮糖就是美黑剂的主要成分。它能和角化细胞中的角蛋白结合，形成类黑素，从而让皮肤颜色看起来更深。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-150d244b3047f826b71325dbf0dc3941_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>赤藓酮糖和角化细胞结合，形成类黑素</figcaption></figure>
<p>由赤藓酮糖来的类黑素并不是黑色的，而是红棕色，当用量较少的时候，看起来就是黄不拉几的颜色。实际上，赤藓酮糖的短期使用效果和维 C 染黄脸的效果一毛一样！</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-ed5560f5b0835ada16031f1be564ef7d_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>使用赤藓酮糖（左）后的皮肤颜色明显变黄（来自原料供应商资料）</figcaption></figure>
<p><strong>从维 C 到赤藓酮糖，鬼知道我经历了什么</strong></p>
<p>维 C 的氧化是个非常复杂的过程<sup>[1]</sup>。简单来说，维 C（抗坏血酸）先会变成脱氢抗坏血酸。随后脱氢抗坏血酸进一步变成 2,3- 二酮古洛糖酸。以此为基础，接下来是非常复杂的化学反应，生成的产物也非常多。而赤藓酮糖就是其中的产物之一。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-3a18c39b4293e3ff48746e4a49950b10_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>根据实验，一份维 C 如果氧化降解完全，可以生成至多 0.28 份赤藓酮糖，这个转化率相当高了。一般欧美使用赤藓酮糖作为美黑剂，这个成分的添加量是 5%。所以如果维 C 精华含有 15%的维 C，如果全部氧化完全，相当于 4.2%的赤藓酮糖，和欧美主流的美黑剂差不多了。</p>
<p>好在现在主流的维 C 精华都使用各种方式来稳定维 C，维 C 降解不多。上脸之后，如果做好防晒，停留在角质层中的维 C 被氧化的概率也不太大。产生的赤藓酮糖也就非常少了。</p>
<p><strong>脸黄了怎么办？</strong></p>
<p>很多人用维 C 明明是为了提亮肤色，结果脸变黄了，郁闷的同时还不免担心起来，染黄的脸洗都洗不掉！</p>
<p>其实用了维 C 之后脸变黄大可不必焦虑，直接停用维 C 就是了。停止使用之后，也就缺少了赤藓酮糖的输入，皮肤中也就不会形成类黑素，皮肤就不会继续变黄。</p>
<p>而根据赤藓酮糖的工作原理，它一般只作用于角质层最外层的一些角化细胞。而且我们的皮肤也会正常代谢。随着皮肤的代谢，这些染黄的角化细胞基本上一周就会代谢干净，最长不会超过两周。</p>
<p>如果你又喜欢维 C 的功效，又不想皮肤变黄，那么还有个办法就是搭配使用一些果酸类的产品。日常使用维 C 的时候，手指手掌处的染黄会更明显，因为这些地方的角质层更厚，有更多的角蛋白和赤藓酮糖反应。使用一些温和的果酸类产品剥脱角质层，让角质层薄一些，脸黄就不那么明显了。当然，在使用果酸之前，得确保自己皮肤耐受，而且不是敏感肌。</p>
<p><strong>小结一下，用维 C 脸变黄并不是因为简单的氧化。而是因为维 C 转化成了赤藓酮糖，而这个成分能和角化细胞结合，形成类黑色素，从而染黄角质层。解决的问题很简单，要么停用维 C，脸黄一两周就会消失；要么使用温和的果酸轻微剥脱角质层，减少黄气。</strong></p>
