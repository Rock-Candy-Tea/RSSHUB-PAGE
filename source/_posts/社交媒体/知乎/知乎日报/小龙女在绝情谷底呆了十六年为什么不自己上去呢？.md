<p>从老先生的设计图纸来看，小龙女应该是掉进了连环峡谷。</p>
<p>正如某位匿名高赞答主所说，绝情谷确有两个出口。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-790b7bcf17825e160e480be45cab6ace_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>绝情谷剖面图</figcaption></figure>
<p>如图，A 点，即绝情崖，一切故事与事故的发生地，小龙女和杨过相继从此跳下，没想却发现了陶渊明的世外桃源。</p>
<p>整个绝情谷从外界看，就是 A 和 B 间的距离，像一口井，从 A 投一块石头下去很久能听到水声，说明谷底有水且深不可测。</p>
<p>其实谷底中间还有一块凸出的石头崖子，如图 C，正是这个 C，造就了谷底的别有洞天。</p>
<p>水源从 A 侧的绝情崖而来，然后在 A 和 C 间汇集，水面一直上涨到和 C 石头齐平，所以 A 点投石测得的绝情谷深度其实比较浅，以水面为参考。</p>
<p>水会一直上涨直到漫过 C 石头，在另一边形成瀑布， 然后又汇集形成冲击平原，造就了独特的生态系统养育小龙女十六年。</p>
<p><strong>回到问题，为什么小龙女轻功如此之好还是上不去呢？</strong></p>
<p>如果在 C 和 B 间峡谷，无论如何也上不去，即使是杨过这样的高手。</p>
<p>以地平线为参考，A 侧悬崖高 h1，B 侧高 h2，从建筑图纸看，h2＞＞h1，且 B 侧为倒扣式结构，人最多爬到 G 点就会被重力吸下来，所以 B 侧没有可能。</p>
<p>那 C 点呢，以 C 点凸起的石头为参考先爬上去呢，前文说了，C 点在 B 侧是瀑布，无处下手，当时小龙女清醒过来也很懵逼，明明记得是从崖子上掉下来的，怎么现在成了瀑布呢？</p>
<p>同时从图中，C 瀑布的高度 h3，A 和 C 的高度差 h4，那么 h3＞h4，也就是说，小龙女爬到 C 点要比从 C 爬到 A 更困难。</p>
<p>而小龙龙当时又中了毒功力未完全恢复，就没有反复折腾。</p>
<p><strong>那么他们两个为什么都能到 C 和 B 间呢？</strong></p>
<p>因为他们都向死而生。</p>
<p>C 这块石头有一个特点，那就是在 C 的底部有一个洞，水流也可以从洞里穿过去，这可能是多年被水侵蚀的原因。</p>
<p>这就在 A 和 C 的水池里形成了一个临界线 E，在临界线 E 以上，水流方向是沿着 D 方向，在石头表面流过成为瀑布，而在临界线 E 以下，水流则沿着 F 方向，从 C 的小洞流到另一侧。</p>
<p>很明显，小龙女和杨过都跨越了这个临界线。</p>
<p>杨过是绑着石头跳下去的，他是要把自己淹死，借石头的重力，杨过很快就跨越了临界点，被水流带着穿过了洞口，当他再浮出水面的时候大吃一惊，可能大家还记得古天乐那时候的表情，他自言自语了一下，小龙女受到了类似的作用。</p>
<p>猜想可能是水流的不断侵蚀，C 底部的洞口正在逐渐变大，临界线有上移趋势，这么多年过去了，现在随便一个人都有可能跨越临界线找到小龙女。</p>
<p>再说说小龙女的生活，要生活肯定得有水，这个谷底满足，其次小龙女是靠着蜂蜜活下来的，而蜜蜂采蜜是一定要有花的，谷底有吗？答案是有。</p>
<p>C 和 B 之间的这段冲击平原造就了一个温暖湿润鸟语花香的独特生态圈，从 A 点往下看，由于 C 点的阻碍作用，即使带了望远镜，也看不到 C 和 B 间的平原，以及平原上果着的小龙女。所以外界一直以为绝情谷底就是黑洞洞一片。</p>
<p>但其实呢，小龙女还可以晒太阳，每天正午的一段时间，太阳可以照射到谷底，能量供应植物和花草茂盛生长，而只要开花，蜜蜂总能循着花粉而来，小龙女也可以晒晒太阳补补钙，保持 16 年容颜不老骨头硬朗经得起折腾。</p>
<p>这一定是著名建筑师设计的图纸。</p>
