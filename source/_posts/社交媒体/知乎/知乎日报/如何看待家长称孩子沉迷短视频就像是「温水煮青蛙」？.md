<p>很有意思的话题，我觉得这样的问题未来只会更多，为什么？</p>
<p>因为你们家长还没怎么见识过成长起来的<strong>短视频原住民</strong>。</p>
<p>这则新闻基于中学生，x 音 x 手火起来大概是 2016 年，假设从 2016 年看到现在，16 岁的高中生从 10 岁开始看，13 岁的初中生从 7 岁开始看，也只是个<strong>「半路出家」</strong>的短视频沉迷者。</p>
<p>什么是原住民？是那些从两三岁开始的孩子。我寒假时候去菜场买菜，看到商贩们忙着招呼顾客，自己家的小孩子都拿着爸爸妈妈的手机坐在小凳子上刷短视频，也有稍大一点的玩游戏的，无论是干啥，玩手机时候目不转睛那种专注的神情，让你不敢相信这是活泼好动的三岁小孩。但是手机能让孩子安静、安全，大人也能更方便干活，谁都喜欢这样低价高效的电子保姆啊！</p>
<p>我不禁设想，这些忙于卖菜的爸爸妈妈，如果有闲，会不会制止孩子玩手机呢？不排除有部分家长会重视对孩子的陪伴，会和孩子多互动；但更多的家长有闲的时候也在自己刷自己玩，孩子在旁边看。</p>
<p><strong>我突然意识到一个很残酷的现实：家长在为生活打拼的时候，他们的孩子已经悄悄与同龄人拉开差距——当我抱着砖的时候，就不能抱你；但不抱你，你的未来继续和我一样去抱砖。</strong></p>
<p>我们都说家长要多陪孩子，但真正当了家长，有些是想陪孩子而不得的。我无意苛责这些忙着卖菜甩手机给孩子的父母，有些时候大人也很难也有很多无奈，一切都是知易行难。大人需要的是明白道理，然后尽量往好的方面引导。</p>
<p><strong>短视频有哪些厉害之处呢？</strong><strong>1. 反复给你推送相关的视频，形成信息茧房。</strong><strong>2. 消磨时间，不知不觉之间，让你花了大量的时间。</strong></p>
<p>拿我自己的一段经历来说，我刷到 x 音一个视频：</p>
<blockquote>一对男女撞倒一外卖，狗男女不但不道歉反而百般侮辱，各种夸张的表情动作语言表达瞧不起外卖员，情真意切处，还踹上一脚仁义礼智踢，这时候另一人跑来说了句话，原来外卖员是甲方爸爸。</blockquote>
<p>第一眼就知道结局的固定套路，加上浮夸的表情动作，剧情又假又烂又沙雕！我全程看完居然感到一丝智商上的优越感。</p>
<p>因为看完了，x 音就给我疯狂推送了类似的剧：</p>
<blockquote>女朋友 / 丈母娘 / 远房亲戚 / 小老板 / 拜金女 / 邪恶路人<br>看不起 / 欺负 / 嘲笑 / 羞辱 / 操弄 / 排挤<br>快递小哥 / 外卖小哥 / 清洁工 / 农民工 / 穷小伙 / 貌似宅男的穷亲戚 / 朴实无华的未婚妻<br>后来得知，对方竟是大老板 / 大富豪 / 人生赢家 / 甲方爸爸 / 领导亲戚 / 上司 / 亲爹<br>结果，后悔 / 惊呆 / 哭 / 下跪 / 跪求<br>非常沙雕但似乎有一种魔力让我几乎每条都看完了，并得出以上公式。<br><br><br><br><br></blockquote>
<p>关掉抖音，我想了一下我获得了什么？</p>
<p>快乐？</p>
<p>似乎没有。</p>
<p>知识？</p>
<p>扯淡。</p>
<p>只感到一阵虚无，我更加的无聊。</p>
<p>而且，我明白自己失去了什么。</p>
<p>时间。</p>
<p>我直接把 x 音卸了，至今没怎么用过。</p>
<p><strong>3. 操作简单，立即反馈。</strong></p>
<p>让你停不下来，欲罢不能。就像吃瓜子容易停不下来，因为瓜子一嗑，一颗瓜子仁就蹦出来了，放到嘴里就得到了奖励。相比，剥山核桃就不容易上瘾，不是因为山核桃不好吃，主要是太麻烦了。短视频也一样，没开始看之前，不知道讲什么，带着好奇心开始观看，十几秒钟甚至几秒钟，就看完了，突兀夸张的情节哈哈哈一乐或者赏音悦目的绚丽画面音乐呈现出来，就是一个正反馈。手一滑甚至不用滑动，就是下个吸引人的未知。</p>
<p>正是凭借这些本事，让短视频能塞满人们的碎片时间，让孩子们乐此不疲。</p>
<p><strong>沉迷刷短视频对孩子有什么危害？</strong></p>
<p>我是专注于儿童教育的中学老师，我个人的观点是，短视频这种娱乐方式，孩子越少接触，越晚接触越好，原因很简单：<strong>它能提高幼儿刺激阈值，磨灭了专注度、意志力和主动探究。</strong></p>
<p>说白了就是【孩子，你太年轻，根本把持不住】。</p>
<p>我们想想看，自己小时候一本课外书，能让我们看的如痴如醉，但现在的很多孩子能去玩玩具就绝不会去看书，能玩手机游戏就绝不会去玩玩具，能刷段视频就绝不会玩游戏，毕竟玩游戏有时候还需要动脑子，但刷视频更无须动脑。</p>
<p><strong>手机短视频站在了吸引孩子的注意力和消磨时间的制高点。</strong></p>
<p>很多家长就曾经和我说，我们家的孩子挺聪明的，就是不能吃苦，学习不用功。我往往第一句就是「不只是你的孩子，这是当下孩子的通病，时代的通病。」</p>
<p>其实，吃苦是违背天性的，趋利避害才是天性。孩子难吃苦，是因为孩子更多专注当下。有的大人能吃苦，是因为这些大人能看到未来的「利」，能忍受当下的苦。但有的大人也不能吃苦，是因为为了一个长远目标放弃眼前的舒适，是需要很大能量的。很多大人更是沉迷短视频。</p>
<p>就不说看短视频，有多少人是「长文不看」「不是故事不看」「没图不看」呢？如果读者能耐心看到文章这个位置，你还真是超出一般人的，很难得的了。</p>
<p>一个从小就被各种玩具、游戏、短视频刺激包围的孩子，读书的乐趣就很难体会得到。缺乏专注度和意志力的孩子，自然很难承受住未来学习中的乏味和孤独，也就是吃不了读书的苦。</p>
<p><strong>作为父母，我们可以怎么做？</strong><strong>1. 孩子爱不爱刷，看你。</strong></p>
<p>以身作则，你的孩子在看着你。你给孩子什么环境，孩子就会朝哪个方向发展，尤其是学龄前的孩子，大量的时间在家里，深受家长的影响。</p>
<p>短视频一定程度上丰富了我们的生活，提供了大量灵活就业的岗位，短视频我们躲是躲不过的，它肯定会存在。而一个喜欢刷短视频的父母更容易带出一个喜欢短视频的孩子。</p>
<p><strong>2. 孩子刷什么视频，看你。</strong></p>
<p>郭德纲有句话说：好多人听这个我学坏了，孙砸，你来之前就是那个玩意儿。</p>
<p>越喜欢看什么，短视频就喜欢推给你什么。</p>
<p>而孩子的兴趣点，与父母的引导也是息息相关。在家长的正确养育下，优秀的孩子会选择短视频中的有营养的部分，Ta 会辨认并鄙夷那些垃圾短视频。毕竟当下短视频种类很多，也有科普及教育意义，如果喜欢看这些视频，不能是完全浪费时间。</p>
<p><strong>3. 孩子刷到什么时候，看你。</strong></p>
<p>一个优秀的父母会引导孩子遵守规则，有自制力，而不是，含着奶头一直乐得没完没了。</p>
<p>我最近刚写了一个给孩子立规矩的回答</p>
<p><strong>我们还能怎么做？</strong></p>
<p>可能有人会说，你这站着说话不腰疼，如果孩子就是沉迷短视频，你怎么办？</p>
<p>拿本站来说，ZH 之前就是一直侧重视频功能的，但现在通过扶持想法等一系列手段，似乎在进行一些改变和尝试。我之前一直不太看好 ZH 向视频发展，因为每个平台都有每个平台的调性，深入和专业，可能就是 ZH 的调性，这与一些短视频的格调是相冲的。</p>
<p>同样的道理，每个孩子也会有每个孩子的个性。</p>
<p>如果我真的恰巧有一个特别爱短视频的孩子，可能基于前期教养，也可能这就是他的个性，我仍然坚信堵不如疏，我仍然相信通过父母的努力可以做出一些改变，我可能会投其所好。孩子知道刷短视频很快乐，我就会慢慢引导孩子，让他知道：创作为他人带来快乐的短视频，你会更快乐。</p>
<p>每一个知乎的创作者，都始于刷知乎，但不是每一个刷知乎的会成为创作者。</p>
<p>从一个消费者，成为一名生产者，依然能得到成长，体现他自身的价值。</p>
