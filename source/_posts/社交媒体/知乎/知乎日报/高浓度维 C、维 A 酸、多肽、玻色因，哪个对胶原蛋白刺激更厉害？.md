<div class="question">
高浓度维C，维A酸，多肽，玻色因哪个对胶原蛋白刺激更厉害？

<div class="answer">

<strong>
<img class="avatar" src="https://pic1.zhimg.com/v2-5ec2a04d747dff9845713ffe0964954b_l.jpg?source=8673f162" referrerpolicy="no-referrer">
<span class="author">胖博士，</span><span class="bio">懂你皮肤问题的清华胖师兄</span>
<a href="https://www.zhihu.com/question/420481266/answer/1922621184" class="originUrl" hidden>查看知乎原文</a>
</strong>

<div class="content">
<p>这可真是个难题，目前还没有人把这几个成分放在一起做实验室比较看哪个更厉害。</p>
<p>不过我们可以用柯南的推理能力，从各个单独的测试中推断出，<strong>典型浓度下，维 A 酸>多肽>玻色因～维 C。</strong>虽然有点关公战秦琼的味道，但是数字还是能说明一些问题的。</p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-bcf4ffc59bb2b48cc91d2ad6ddd009bb_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure><p>首先说明一点，在化妆品的功效测试中，人体测试>>3d 皮肤模型>>细胞实验。只有在人体测试中做出了效果，才是真的牛 X。当年欧莱雅测试的不少原料，在 3D 皮肤模型和细胞实验中各种酷炫吊炸天，结果人体测试做不出来结果。</p>
<p>在比较这几个原料的时候，我们把前胶原 -1 作为标尺，那么得出的结论是维 A 酸一骑绝尘。</p>
<p><strong>0.1% 的维 A 酸</strong>，在人体实验中（实验出结果的难度：4-174 人体实验>3D 皮肤模型>细胞实验），对前胶原 -1 的促进高达 119%<sup>[1]</sup>。一般实验中，0.025%的维 A 酸就有非常显著的抗衰老和去皱纹效果。即使是效果稍差的维 A 醇，在低浓度下效果也非常显著。加上极其低廉的价格，维 A 酸就是抗衰老的王者。</p>
<figure><img class="content-image" src="https://pic4.zhimg.com/v2-192128eb4ec118b18e20c9c4d165c381_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>使用 0.1%维 A 酸之后的皮肤前后对比[1]</figcaption></figure><p>多肽的种类很多，这里我们用四肽 -21 来作为代表。在细胞测试中，1ppm 浓度能促进前胶原 -1mRNA 的表达（mRNA 是胶原蛋白合成的信号）提升 100%以上<sup>[2]</sup>，似乎很惊人。然而由于透皮吸收，加上人体是一个复杂的系统，在人体测试中，50ppm 的四肽 -21（对应 2.5%添加量，非常高，一般不现实）对胶原蛋白基因表达的促进作用为 100%左右。在高浓度的人体测试中，多肽表现出了比较好的去皱效果。考虑到多肽的价格和添加量，比维 A 酸还是略逊一筹。</p>
<figure><img class="content-image" src="https://pic4.zhimg.com/v2-0b99dc0b3e1f1a8ab62d7e0622839158_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>4%多肽（对应 80ppm 浓度）下的去皱效果</figcaption></figure><p>在 3D 皮肤模型中，0.016%的羟丙基四氢吡喃三醇（折合大约 0.05%玻色因），对前胶原 -1 的促进是 12%<sup>[3]</sup>，弱于维 A 酸。虽然欧莱雅有玻色因的人体测试数据，但是对于标记物的定量结果缺失。考虑到 3D 皮肤模型到人体测试的折算，低浓度玻色因效果远不如维 A 酸和多肽显著。</p>
<p>在人体测试中，3%玻色因表现出了一定的抗衰老效果。在细胞实验中，玻色因对前胶原 -1 促进的数据缺失，但是 0.008%浓度玻色因对于胶原蛋白 -7 基因的促进作用为 100%以上。<sup>[1]</sup>低浓度人体实验中，玻色因也没有表现出如此显著的去皱纹效果，44 只是在统计学上有一定的体现。（也许欧莱雅高端线 30%的玻色因添加效果能和 A 酸媲美，但是除了欧莱雅之外，没人有便宜的玻色因，市面上的玻色因售价得好几千一公斤，没人加的起那么多）</p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-99082f826edbc1d11e298113dc974535_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>3D 皮肤模型中玻色因对表皮层的刺激[2]</figcaption></figure><p>在细胞实验中，100uM（大约 18ppm）维 C 能促进前胶原 1 mRNA 表达提升 26-120%之间，取平均值为约 70%。</p>
<p>不去考虑逆天的维 A 酸，单单以对 mRNA 的表达来看玻色因，维 C 和多肽对胶原蛋白的促进，实现 100%的浓度分别是，0.026%（260ppm）的玻色因，0.0001%多肽，以及 0.0024%（24ppm）维 C。一般来说，多肽的起效浓度低，在普通配方中，多肽的添加量一般在几十 ppm。所以显而易见，多肽对胶原蛋白的促进效果在这三个成分中最好，玻色因似乎最差。但是考虑到临床数据，玻色因的效果和维 C 差不多。而市面上也没有哪个品牌主打维 C 抗老。</p>
<p>因此根据我的分析，这几个成分对胶原蛋白的刺激（典型浓度的抗衰老效果）</p>
<p>维 A 酸>多肽>玻色因~维 C</p>
<p>当然如果是欧莱雅旗下高端线 30%的玻色因添加，和多肽也许可以一战。</p>
