你见过哪些极美的生物？

<div class="content">
<p>介绍一些蝴蝶飞蛾与美花~</p>
<ul>
<li><strong>绿带燕凤蝶</strong></li>
</ul>
<p>生活在林间沼泽地带，翅膀有部分透明，展翅时非常飘逸，衣袂飘飘，好似踏花而来的林中仙子。</p>
<p><a class="video-box" href="http://link.zhihu.com/?target=https%3A//www.zhihu.com/video/1066482885468463104" target="_blank"> <img class="content-image" src="https://pic3.zhimg.com/80/v2-c6966192f019175f2f8c02d6c6882e26_1440w.jpg" alt referrerpolicy="no-referrer"> <span class="content"> <span class="title"><br></span></span></a></p>
<ul>
<li><strong>蓝闪蝶</strong></li>
</ul>
<p>有种流光溢彩的感觉，闪蝶主要分布于中南美洲。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-e63eb65f05dcc71b28df5f85e776cbcd_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-a114aefbd301b6dc13771cfb6711c904_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>纱钩蛾</strong></li>
</ul>
<p>钩蛾科，如纱般透明，仙气十足~生活在印度，缅甸，台湾。</p>
<figure><img class="content-image" src="https://pica.zhimg.com/v2-9ab96efd34ec0723ced98390c1b42895_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>透翅蝶</strong></li>
</ul>
<p>也称“玻璃翼蝴蝶”，它独特的透明翅膀可以在某种情况情况下进行伪装。</p>
<p>西班牙等地称它为“小镜子”。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-2cbb28bc516dd34caeb7b99ad3d65d47_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong>说完了蝴蝶蛾子，再介绍一些极美的花~蛾蝶更与好花配~</strong></p>
<ul>
<li><strong>冰岛虞美人</strong></li>
</ul>
<p>冰岛虞美人是罂粟科罂粟属的观赏植物，冰岛虞美人有比较多的园艺品种，最好看的是大花的切花用的冰岛虞美人品种名叫 colibri。其种子价格差不多是一美金一粒，也可能是目前世界上表现最好的冰岛虞美人品种。</p>
<p>国外的婚礼用花用此品种非常华丽：丝绒般的质感，超长的花期，茎干粗壮和大花的表现。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-145a29c364064c7ceabac66ee1144898_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-5e11f2796c839a0bcd4475c8356ea1e7_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>莲花升麻</strong></li>
</ul>
<p>莲花升麻是多年生草本植物，原产于日本本州，是日本独有植物，该属仅此一种。</p>
<p>生于林间，喜爱湿润阴凉的气候，淡藕荷色的花色，低垂的花头，纤细而柔弱，宛若低头的娇羞。</p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-d67954ff725c1cddaa1e53dead86eafb_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-322219bdfaeccdbb397f6c3d0a48cc65_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>大丽花‘牛奶咖啡’</strong></li>
</ul>
<p>作为大丽花中极致粉嫩的切花品种,‘牛奶咖啡’不仅花直径非常大，而且常用于婚礼用花，营造浪漫的氛围。深受花友喜爱~</p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-5633af5f6351754ff68875b94957ca55_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-cece2d66a4ec9193f663ac152bd0624d_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>美洲矢车菊</strong></li>
</ul>
<p>原产于美洲，是菊科矢车菊属植物，也是该属中少有的大花型植物，花姿轻盈飘逸，颜色若有若无，纤细而又迷离。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-9317201bd65dc155c8ad4ed6889cdc16_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<ul>
<li><strong>花毛茛</strong></li>
</ul>
<p>有色彩淡雅的花，自然也有绚丽的花~园艺花毛茛真的是美到飞起~国内比较少见这类鲜切花品种。</p>
<p>花毛茛为毛茛科花毛茛属植物，花似牡丹，叶似芹菜~品种非常多，颜色非常丰富~全世界都非常流行的园艺植物。现在这个季节花市应该可以买到一些盆栽品种了~</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-f8b85120feea9cc923f76e30cdafcbdc_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>花毛茛 Pink Picotee</figcaption></figure>
