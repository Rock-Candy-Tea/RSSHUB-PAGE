<div class="question">
为什么《向往的生活》越来越追不下去了？

<div class="answer">

<strong>
<img class="avatar" src="https://pic1.zhimg.com/v2-d43ee5ef574175d2fbd0caf3c414758a_l.jpg?source=8673f162" referrerpolicy="no-referrer">
<span class="author">热風，</span><span class="bio">知之为知之，不知为不知</span>
<a href="https://www.zhihu.com/question/398276926/answer/1319329867" class="originUrl" hidden>查看知乎原文</a>
</strong>

<div class="content">
<p>MC 四人没话找话 / 成语接龙中</p>
<p>「叮零零......」</p>
<p>「彭彭电话来了快去看谁来了，妹妹也跟着去」</p>
<p>「喂，您好，这里是蘑菇屋，请问您想点什么菜」</p>
<p>「我想点红烧龙头」</p>
<p>「啊？」</p>
<p>（彭彭走出房）</p>
<p>「黄老师，那个嘉宾要吃龙头」（疑惑状）</p>
<p>「吃龙头？不给做」（愤怒并瞪大眼）</p>
<p>（一个小时后）</p>
<p>「黄老师，何老师我们来了」（嘉宾欢天喜地状）</p>
<p>「啊，张三，李四，王五，好久不见」（拥抱）</p>
<p>（开始参观蘑菇屋，并赞叹真好）</p>
<p>（硬广中.....）</p>
<p>（开始喝茶）</p>
<p>「黄老师，有我们需要帮忙的吗？</p>
<p>没有吗？那我去把碗洗了吧 / 柴砍了吧 / 桌子擦了吧 / 睡一会儿吧」</p>
<p>（下午傍晚）</p>
<p>「彭彭，你带张三，李四去把玉米收了 / 甘蔗砍了 / 水稻插了 / 鱼捞了，何老师，帮我弄两根葱好吗」</p>
<p>（彭彭和嘉宾穿完装备）</p>
<p>「黄老师，我们出发了」</p>
<p>（黄老师抬头，笑）</p>
<p>（彭彭、张三和李四奋勇干活挥汗如雨中）</p>
<p>「黄老师，我们回来啦」/「累死我们了」</p>
<p>「辛苦辛苦，我给你们准备的姜汁撞奶 / 水果沙拉 / 冰红茶 / 凉茶赶紧喝一下，辛苦辛苦」</p>
<p>（开始做手工 / 静坐 / 围观炉灶 / 洗澡 / 岁月静好没啥事）</p>
<p>（硬广中......）</p>
<p>（天色渐晚）</p>
<p>（所有人围观黄老师与炉灶）</p>
<p>「开饭啦」（何老师欢呼状）</p>
<p>「黄 老 师 辛 苦 啦」（全体欢呼）</p>
<p>「yuuuuohhhhwwwooo」（全体赞美食物）</p>
<p>「这个菜绝了」（何老师长翅膀）</p>
<p>「今天我们欢迎，张三，李四，王五来到蘑菇屋」</p>
<p>「谢 —- 谢 —- 黄 —老 —- 师」</p>
<p>（饭毕）</p>
<p>「唉张三，你会玩黑魔法 / 开还是关 / 数马吗？」 / 「唉，李四，你还记得我们当初拍那个戏的时候王五才 17 岁...」</p>
<p>（玩游戏 / 回忆人生）</p>
<p>「睡觉了，晚安」</p>
<p>（翌日早厨房中）</p>
<p>「唉，张三你怎么起这么早，李四还在睡着呢」</p>
<p>「搞体育的 / 年纪大了 / 睡不着 / 早起锻炼的就是觉少」</p>
<p>（哈哈哈黄老师与嘉宾对视一笑）</p>
<p>（十一点最后一位嘉宾醒来）</p>
<p>「来来来，快吃个早午饭，我刚给你们做的米线 / 炸酱面 / 葱油饼」</p>
<p>「wowwwwww 好好吃」（众人除何老师赞叹中）</p>
<p>「绝了这个米线 / 面 / 饼」（何老师赞叹）</p>
<p>（饭毕）</p>
<p>「黄老师，我们要走了」</p>
<p>「哦哎，彭彭，快去送送张三李四王五」</p>
<p>「好舍不得这里，这里空气好好 / 好放松 / 好温馨 / 像回到小时候 / 好好吃 / 好好玩 / 像家人」（旁白与手写体）</p>
<p>MC 四人没话找话 / 成语接龙中</p>
<p>「叮零零......」</p>
<p>「彭彭，快去接电话」</p>
