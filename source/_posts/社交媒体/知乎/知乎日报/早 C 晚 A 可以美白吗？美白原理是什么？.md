<p>早 C 晚 A 有一定的美白效果，但是高浓度维 C 有可能让脸变黄，早 C 晚 A 不耐受的话，更会有色斑的风险。所以如果你的目标是美白，那么我不建议你早 C 晚 A。当然，如果你正在早 C 晚 A 护肤，也许美白是个意外的收获。</p>
<p><strong>早 C 晚 A 怎么美白？</strong></p>
<p>最近早 C 晚 A 很流行，消费者也跟风去。我们常说的早 C 晚 A 产品，包含了早上使用的高浓度维 C 精华，以及晚上使用一定浓度的 A 醇产品。从原理上来说，维 C 抗氧化，A 醇抗衰老。维 C 能有一定的美白效果。</p>
<p>当紫外线照射到皮肤上，会在皮肤中激发氧自由基。这些自由基一方面氧化皮肤中存在的黑色素前体。另一方面，它会产生信号因子，激活我们的黑色素细胞。于是在黑色素细胞中，酪氨酸在酪氨酸酶和自由基的共同作用下，变成黑色素的前体，随后进一步氧化为黑色素。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-cfb53945aaa2233128ed288646e8cb33_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>所以在黑色素的生产过程中，氧化是不可或缺的条件。而维 C 是非常强的抗氧化剂。它能牺牲自己，成全别人，淬灭体内的自由基，从而减轻皮肤中的氧化压力，减少黑色素的生成。</p>
<p>例如科颜氏淡斑精华（国妆特进字 J20201421），里面的主要活性成分就是维 C 乙基醚以及水杨酸。</p>
<p>乐敦 CC（国妆特进字 J20200460）的主要活性成分则是维 C 以及甘草酸二钾。</p>
<p>所以理论上来说，按照早 C 晚 A 理论来护肤的朋友，也能有一定的美白效果。用更精确的话来说，应该是“早 C 晚 A 有一定的提亮肤色效果”。</p>
<p><strong>早 C 晚 A 美白的局限</strong></p>
<p>从黑色素的生成和代谢过程可以看到，美白的通路有以下几个：</p>
<ol>
<li>抗氧化</li>
<li>抗炎</li>
<li>抑制酪氨酸酶活性</li>
<li>抑制黑色素转移</li>
<li>促进黑色素剥脱</li>
</ol>
<p>虽然早 C 晚 A 有一定的美白效果，但是相对来说，单一的维 C 还不足以覆盖美白的全部通路。因此如果仅有单一的维 C 成分，美白效果很一般，以至于只能叫提亮肤色，而不是美白或者淡斑。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-d89e3477819fa797e052dd89a83580fc_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>事实上，主打维 C 衍生物的科颜氏淡斑精华，里面也复配了促进黑色素剥脱的水杨酸。而乐敦 CC 搭配的则是抗炎的甘草酸二钾。</p>
<p>至于市面上其他卖的更好的美白产品，大多是多种美白通路复配。</p>
<p>例如 OLAY 淡斑小白瓶，通过酰本胺抑制黑色素细胞的活性（原理和抑制酪氨酸酶活性类似），烟酰胺抑制黑色素转移，泛醇和糖海带提取物抗炎，来实现多通路美白的目的。</p>
<p>还有欧莱雅注白瓶，阿魏酸抗氧化，377 抑制酪氨酸酶活性，烟酰胺抑制黑色素转移，辛酰水杨酸促进角质层脱落，也覆盖了绝大部分美白通路。</p>
<p>早 C 晚 A 只能覆盖抗氧化一个通路，因此在美白效果上，不如其他专精的美白淡斑精华。</p>
<p><strong><strong>早 C 晚 A 美白的风险</strong></strong></p>
<p>我们业内人士常说早 C 晚 A 是“猛药护肤”。无论是高浓度的维 C 还是 A 醇，都有一定的刺激性。市面上的维 C 精华，为了保证维 C 的透皮吸收效果，pH 一般都在 3.5 以下，最典型的就是修丽可 CE 精华。而正常皮肤的 pH 在 5-6 之间，因此低 pH 的维 C 会对皮肤造成刺激。</p>
<p>有些人皮肤相对耐受，用起来不会有问题。而有些人皮肤不耐受，会刺激皮肤产生炎症。皮肤炎症释放的炎症因子激活黑色素细胞，于是在炎症部位产生色斑。</p>
<p>A 醇也是如此。根据相关研究，A 醇最典型的副作用除了皮肤刺痛泛红之外，就是色素的不正常沉积。我身旁也有使用 A 醇不当，最后长斑的朋友。</p>
<p>除了不耐受带来的炎症和色斑之外。使用高浓度维 C 美白最大的问题就是皮肤泛黄。因为高浓度的维 C 上脸之后，在光和大气中氧的作用下，很容易被氧化，变成颜色深的二酮古乐糖。它停留在角质层中，有可能让角质层染色变黄。很多人反映使用了维 C 精华之后变成了“黄脸婆”，就是这个道理。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-470f0a7cb3ba00414289d27b7107258d_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>维 C 精华变色</figcaption></figure>
<p><strong>总结起来，早 C 晚 A 有一定的美白效果（更准确的说法是提亮肤色）。但是因为美白通路单一，美白的效果不如常见的美白精华。而且作为猛药，早 C 晚 A 不耐受还会造成色斑等副作用。高浓度的维 C 也有可能染黄角质层。因此不建议用早 C 晚 A 来美白。</strong></p>
