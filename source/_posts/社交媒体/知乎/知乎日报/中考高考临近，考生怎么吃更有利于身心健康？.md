<div class="question">
中考，高考临近，考生怎么吃更有利于身心健康？

<div class="answer">

<strong>
<img class="avatar" src="https://pica.zhimg.com/v2-49d6783dd2eed1a59939c4c3d3ec3ab6_l.jpg?source=8673f162" referrerpolicy="no-referrer">

</strong>

<div class="content">
<p>我高中老师前几天突然问我，家里小朋友马上要中考了，这段时间应该吃点什么好。</p>
<p>然后我就陷入了感慨，当年上高中的时候还是个小不点，竟然一下子马上就要中考了。</p>
<p>参加工作后很长一段时间感觉时间似乎停滞掉了。果真只有看到小朋友长大，才一下子发觉时间过的飞快。</p>
<p>跑题了跑题了，想说中考高考前，<strong>家长对孩子的饮食可能会感到有些许压力。因为在学习上好像也做不了什么，只能在生活上对孩子尽可能的给予帮助。</strong></p>
<p>我老师就说孩子早餐和周末回来都不知道该给他吃什么，每次准备食物都有一种“横冲直撞”的焦虑，逮着什么就做什么，没有一点计划性。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-362e7b8f74d3bca0de36bfcb90422b18_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>其实关于考生的考前饮食，家长不用过度担忧，总体的应对原则是，<strong>平时吃什么现在也吃什么就行了。</strong></p>
<p><strong><strong>我觉得在大考前，有一样东西更重要，那就是心态。</strong></strong></p>
<p>我有一高中同学，前不久以第一作者之名发了篇 nature，他自己非常谦虚，发了个朋友圈就继续默默做实验去了，但我们所有同学都激动万分，到处夸耀“我的科学家同学“事迹（比如我 hh）。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-e224a9f822aa516ab657359799a38778_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>他在上高中时就有一个非常明显的特点，<strong>心态巨好，特别淡定</strong>。</p>
<p>不管是每次模拟考还是最后的高考，他都非常“佛系“，保持自己的学习节奏，<strong>从来不和我们“卷“。</strong></p>
<p>他就是凭借着他良好的心态、踏实的努力、以及认真钻研的精神，一步一个脚印，从刚上高中时的一两百名，到高考顺利考上浙大，然后顺利读研读博，到今天顺利发 nature。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-36280feaccd4c603043eba9f6184db6a_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"><figcaption>来自我们县的报道</figcaption></figure>
<p>说了这么多，想跟各位家长考生表达的是，<strong>在考试前的时候放宽心，保持好心态就可以啦。</strong>顺其自然，一切自有最好的安排。</p>
<p>当然也有一些能帮助保持身心健康的饮食大建议，各位家长可以在不改变以往饮食习惯的基础上加以灵活运用。</p>
<p><strong>1、早餐质量要足够高，才能保证上午三小时充足精力，蛋白质 + 坚果可以延缓午前出现饥饿感。如果早上吃不下太多，可以带一瓶酸奶在上午迟点的时候喝，或者带一包每日坚果当零食吃。</strong></p>
<p>在考前的时间（其实平时也一样），家长要特别重视早餐的质量。我自己在上学时有一点其实做的很不好，就是不爱吃早餐。经常吃两片面包或者吃一碗拌面就过去了，早上时常没有精神，困困的，永远是一副刚睡醒的样子。</p>
<p>然而在我工作后开始关注营养，关注早餐质量开始，我发现良好的早餐能让我一整个上午都保持精力充沛，思维活跃。</p>
<p>就.....感觉有点亏.....</p>
<p>我那时要是好好吃早餐，说不定现在也有发 nature 的同学一半厉害呢（开个玩笑哈哈），</p>
<p>早餐最好是有<strong>一定的碳水 +2 样蛋白质 + 蔬菜（或者水果）+ 坚果</strong>。</p>
<p>给大家举三个例子。</p>
<table>
<tbody>
<tr><th> </th><th>早餐</th></tr>
<tr>
<td>第一天</td>
<td>黑芝麻豆浆 1 杯 + 香菇菜包 2 个 + 煮鸡蛋 1 个</td>
</tr>
<tr>
<td>第二天</td>
<td>抱蛋煎饺 + 牛奶 1 杯 + 一个小苹果</td>
</tr>
<tr>
<td>第三天</td>
<td>香甜八宝粥 + 煎鸡蛋 1 个 + 蓝莓坚果酸奶 1 小杯</td>
</tr>
</tbody>
</table>
<p>黑芝麻豆浆的做法：黑豆 12g、黑米 13g、黑芝麻 5g、核桃 10g、小米 5g、水 500ml，用豆浆机或破壁机的豆浆模式，打完之后加 250ml 牛奶和 20g 冰糖，搅拌两分钟。</p>
<p>这里是 3 杯分量，很好喝，我经常做这个。</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-b3f530e5699d8d55d2b925e1f377eb4d_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>抱蛋煎饺做法：平底锅加一点油和水，加入饺子盖上锅盖中火煎到饺子底部出现脆皮，然后淋入蛋液煎熟（饺子就用家里平时吃的饺子，馅料是荤素搭配的就成，例如韭菜猪肉馅，芹菜猪肉馅）。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-1114889672fce372e88f3f940cccd036_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>八宝粥做法：紫米，糯米，小米，花生，红枣，桂圆肉，葡萄干，莲子，百合，加 8 倍水煮粥，电饭锅预约（但不要加太多糖）。</p>
<p>蓝莓坚果酸奶：酸奶里拌点蓝莓和坚果。</p>
<p><strong>2、午餐要控制精白主食，避免下午昏昏欲睡，影响大脑工作效率。</strong></p>
<p>吃点粗粮的好处我已经讲过很多次了，但这事儿不好办，因为孩子可能平时没有吃粗粮的习惯。<strong>怕孩子抗拒甚至出现肠胃不适。</strong></p>
<p>可以用我之前出过的一个藜麦小米饭食谱，口感上和白米饭非常接近，不会让孩子难以接受，而且也不容易引起胃胀。少量粗粮还改善了白米饭的营养。</p>
<p><a class="internal" href="https://www.zhihu.com/question/321547522/answer/2412571653">长期用杂粮代替米饭会对身体有何影响？</a></p>
<p>在考前比较早的时候就可以让孩子试吃了，但如果发现孩子还是不喜欢，就改回白米饭。</p>
<p>做法：</p>
<p><strong>大米 100g，小米 15g，三色藜麦 15g，玉米 15g，豌豆 15g</strong></p>
<p>大米小米藜麦淘洗好后和平时煮白米饭的程序一样，玉米粒和豌豆粒用水煮一下最后一起混合。</p>
<p>同样给各位家长举三个例子。</p>
<table>
<tbody>
<tr><th> </th><th>午餐</th></tr>
<tr>
<td>第一天</td>
<td>红烧茄子 + 鲫鱼豆腐汤 + 蒜蓉空心菜 + 藜麦小米饭</td>
</tr>
<tr>
<td>第二天</td>
<td>猪肝菠菜汤 + 红烧鸡翅 + 胡萝卜炒莴笋 + 藜麦小米饭</td>
</tr>
<tr>
<td>第三天</td>
<td>清炒包菜 + 银鱼水蒸蛋 + 番茄炖牛腩 + 藜麦小米饭</td>
</tr>
</tbody>
</table>
<p><strong>3、晚餐不要吃的太油腻，以免影响睡眠。</strong></p>
<p>晚餐宜清淡一些，吃太油腻会影响消化和睡眠。</p>
<p>但清淡不是寡淡，也要把饭做的让孩子有胃口吃下去哦。</p>
<p>以前我妈有次不知听谁说考前应该吃清淡点，然后就把饭菜做的很“清淡“，清淡的我都不想吃了。</p>
<p>举三个晚餐的例子。</p>
<table>
<tbody>
<tr><th> </th><th>晚餐</th></tr>
<tr>
<td>第一天</td>
<td>玉米萝卜排骨汤 + 西芹炒肉丝 + 香菇烧油菜 + 红薯饭</td>
</tr>
<tr>
<td>第二天</td>
<td>茄汁虾仁烧豆腐 + 上汤西兰花 + 红豆汤 + 蒸紫薯一小块</td>
</tr>
<tr>
<td>第三天</td>
<td>白切鸡 + 丝瓜蛋汤 + 白灼菜心 + 小米粥</td>
</tr>
</tbody>
</table>
<p>茄汁虾仁烧豆腐做法：2 个熟透的番茄切成小块，下油锅炒出汁，然后加点水，加一个碾碎的咸蛋黄，加点生抽、蚝油和盐。虾仁加点盐、白胡椒粉、少量白糖先腌一下然后下锅煎一煎，老豆腐切成块，加点盐稍微煎一煎，然后和虾仁一起放茄汁里稍微煮一会，拌匀后出锅</p>
<p>上汤西兰花：准备少量肉片，1-2 个水煮蛋（切小丁），3 瓣蒜，一点平菇撕成小条待用，西兰花切块。先下锅炒香肉片和蒜，然后加水煮蛋炒一炒，加点料酒，再加半碗水煮开，用盐和白胡椒粉调味，加平菇和西兰花再煮 2 分钟，出锅撒一点枸杞和葱花（西兰花比较难熟，可以用水先煮熟，没买到西兰花的花可改成娃娃菜菠菜之类的）。</p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-e2ace03d0990e931b6c69893e1c1bb13_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong>还有其他的一些建议：</strong></p>
<p>4、大豆、鸡蛋（黄）富含卵磷脂，卵磷脂是神经递质乙酰胆碱的原料物质，对维持记忆力有好处。</p>
<p>5、奶类、绿叶蔬菜和杂粮，对考生有额外的好处。牛奶、酸奶中含有大量钙，以及牛奶蛋白分解产生的安神肽，能够帮助维持考生情绪稳定，绿叶蔬菜中的镁也有类似的作用。杂粮中的 b 族维生素对大脑高效工作十分有帮助。</p>
<p>6、少量多次喝水，有助于维持身体活动和认知能力</p>
<p>7、不要喝冷饮、饮料、吃烧烤，每天可以适当吃一点儿水果，但是蔬菜吃够了的话水果不爱吃也没事。</p>
<p>差不多就是这些了。</p>
<p>我是章沪川，提前祝各位考生考试顺利，考上理想的学校。</p>
