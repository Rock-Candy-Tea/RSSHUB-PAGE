<div class="content">
<p><strong>（多图预警~）</strong></p>
<p><strong>准确地说，Pokemon 系列游戏销量和风评（包含媒体评分和玩家口碑）从来也都没有什么关系</strong>。初代 Fami 通评分 29，是历代最高销量正作，五代 fami 通满分，是历代销量最低正作。至于各类评分和销量，以及和口碑的对比毫无相关性可言，<strong>宝可梦游戏从来都吃的是 IP，而不是评分</strong>。这也是 Pokemon 系列无论外在争议如何，商业回报一直很稳健的原因所在。销量有时还往往和玩家口碑相反，就很神奇……</p>
<p>虽然，一般玩家基本能分清“叫好”和“叫座”的区别（当然，还有一大堆人根本分不清楚），但是宝可梦的“叫好”和“叫座”有些时候，关系反差过大，令人难以相信……</p>
<hr>
<p><strong>Fami 通历代评分最低：初代《红 / 绿 / 蓝》版，销量最高，3000 万以上</strong></p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-3da442269dea6b2388f2413d7d93fa93_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>谈到 Pokemon，肯定要说的就是初代的“29 分惨案”。日本着名游戏媒体《Fami 通》，在满分 40 分的前提之下，给 Pokemon 打出了 29 分的低分。四位评委分别给出“7、7、7、8”的评分，评语如上图所示，中间一列 2 月 27 日发售的游戏，即为《宝可梦：红 / 绿》（后有《蓝》版）。</p>
<p>可能有部分新玩家，不太了解“《FAMI 通》29 分”是什么概念，<strong>同级别得分中至少 90%是根本没法玩的“垃圾游戏”</strong>……《Fami 通》是一个普遍打分偏高的测评分数体系，这个得分基本就是在评委眼中的“失败作品”……</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-2f8f88833a10e451418f08844ba94530_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong>然而，第一世代的 Pokemon《红》、《蓝》、《绿》（以及资料片《黄（皮卡丘）》）是全世代销量第一名，也是全世代“得分最低”</strong>。即使不计算“致敬动画”的《皮卡丘版》，一世代双版本正作（其实是三版本，只是日版美版不同）也有 3000 万以上销量……</p>
<p>很多时候，纸面意义上的数字都不足以衡量初代 Pokemon 的成功，因为它真的是挽救掌机平台，使得任天堂继续开发 Game Boy 改良款式，度过最低谷，对任天堂、对整个游戏界，可以说都起到了一定影响。一切起源于一个“Fami 通 29 分作品”……</p>
<hr>
<p><strong>Fami 通代评分最高：《宝可梦：黑 / 白》，历代销量最低，仅 1560 万上下</strong></p>
<p>那么 Pokemon 有没有“叫好不叫座”（相对而言）的作品吗？也是有的，还是来自《Fami 通》——堪称宝可梦的“销量毒奶”测评……</p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-1ae883aca01f2f633f5f97df109a913d_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>Pokemon 系列，唯一一个“FAMI 通满分”作品，就是五世代正作——《宝可梦：黑 / 白》。满分拿到 40/40。而平心而论，五世代在 Pokemon 系列中，也真的算“良心作品”，随便拉几个全系列都玩过的玩家，有很大概率，会把五世代看作“最有诚意”的世代，从剧情到资料片（实为续作）都有开创性意义。</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-52bf1ea2d9f9f4941fa295d1b272c12f_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong>然而，《宝可梦：黑白》却尴尬地成为了历代“销量最低正作”（不考虑重制版、资料片）</strong>。就当玩家觉得三世代销量“低谷”（因为断代等原因，三世代销量至今倒数第二，并不像一些模拟器玩家想象得那样具有影响力）不可能重现的时候，五世代来了个“销量垫底”……</p>
<p>当然，这个“垫底”是相对宝可梦而言，1560 万的销量放别家 JRPG（甚至是《最终幻想》等）基本都算是“大卖”的状态了……</p>
<hr>
<p><strong>往往与口碑相悖的销量</strong></p>
<p>实际上，不只是游戏媒体，Pokemon 系列中销量往往和玩家之间口碑、风评也相去甚远，“叫好”的作品未必“叫座”，而“叫座”的作品更未必“叫好”。</p>
<p>目前，主系列游戏大致分为三种：</p>
<ul>
<li>正作（如《红宝石 / 蓝宝石》、《太阳 / 月亮》等）；</li>
<li>重制版（如《宝可梦：火红 / 叶绿》等）；</li>
<li>资料片（如《宝可梦：白金》等）；</li>
</ul>
<p>三种情况，都出现过“叫好不叫座”或者“叫座不叫好”的情况出现，实际商业表现与风评口碑一点联系都没有，随便举几个例子：</p>
<p><strong>正作——《宝可梦：剑 / 盾》目前全世代第三：</strong></p>
<figure><img class="content-image" src="https://pic1.zhimg.com/v2-d78fb07e709ed2ec45f1b5de2ab01795_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>论预热和宣传上的“高开低走”目前没有其他 Pokemon 作品可比《剑 / 盾》。从最初发布时的万众期待到删减图鉴后的“千夫所指”，唯一没有影响的销量本身。目前，《宝可梦：剑 / 盾》销量已经轻松突破 2000 万，远远超越《宝可梦：珍珠 / 钻石》，仅次于初代《红 / 黄 / 蓝》和二世代《金 / 银》（甚至和金银版销量差的都不多了）……</p>
<p>至于质量嘛，反正我觉得和《金 / 银》完全不可同日而语，口碑和风评……（很大原因可能还是排位赛以及没有资料片，转化为 DLC 模式）</p>
<p><strong>资料片——《宝可梦：黑 2/ 白 2》销量不如《宝可梦：究极日月》</strong></p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-f48f35a20b2524fc36a1c6173da213bf_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>如果选“最佳正作”可能争议会不少<strong>，但是如果选“最佳资料片”，那基本玩家内部争议不大，就是五世代的《黑 2/ 白 2》</strong>，名为“资料片”，实为“续作”，故事剧情到人物成长甚至道馆都有变化，相比之前与正作变化不大的资料片，简直时“鹤立鸡群”的存在……</p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-303c1039eb855da6e64d0e527521ec0e_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong>然而，销量上《黑 2/ 白 2》是 850 万左右，而争议巨大的《究极日月》是 900 万以上</strong>。后者成为了“资料片”模式的终结，引来的争议本身并不小，甚至成为 Pokemon 系列口碑滑坡的起点之一，然而销量就是不跟着口碑走的……</p>
<p><strong>重制版——《宝可梦：新金 / 魂银》销量不如《Let's go 皮卡丘 / 伊布》</strong></p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-246fc3fa9e9efc2495b9dae82dd155da_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>别人我就不代表了，如果问我最好的重制版是哪一个，我肯定不假思索地告诉你，<strong>是四世代的重制版《宝可梦：心金 / 魂银》最优秀</strong>，即使别人的答案可能不是这一作，<strong>我也能够肯定，至少不会有人告诉你是《Let's go 皮卡丘 / 伊布》</strong>，这部争议比上面《究极日月》还要大得多，这部作品阉割大量宝可梦游戏核心系统，就是 Pokemon Go 的主机移植版……非要挂着“主系列重制版”的名头，顶着 60 美元定价来卖……</p>
<figure><img class="content-image" src="https://pica.zhimg.com/v2-afa9be0ebcc7548e94b9317028d4a8b5_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong>结果是《宝可梦：心金 / 魂银》销量定格在 1270 万，而《Let's go 皮卡丘 / 伊布》现在已经 1300 万以上</strong>，但是前者的时代已过，销量是固定的，后者因为 switch 还在，销量甚至还有进一步提升的可能性（虽然我也不觉得还能提升多少）……</p>
<p>很多时候，商业上的事情，就没法和“风评口碑”讲什么道理……</p>
<hr>
<p><strong>《珍珠 / 钻石复刻》销量千万不担忧，担忧还能持续多久</strong></p>
<figure><img class="content-image" src="https://pic3.zhimg.com/v2-cd691be57d439ef67327a76e0e468e69_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p><strong>现在《宝可梦：珍珠 / 钻石复刻》已经开放评分，但是已经不重要了，因为预约榜已经“多周霸榜”</strong>，“霸榜”的定义则是——“前三包揽”（包括《宝可梦：阿尔宙斯》）、“前 10 有 7 个”（包括《宝可梦：阿尔宙斯》）、“前 15 有 11 个”（包括《宝可梦：阿尔宙斯》）……</p>
<p><strong>考虑到预约创“最低记录”的《Let's go 皮卡丘 / 伊布》现在都能 1300 万销量，《珍珠钻石复刻》的销量保守悲观估计保住千万级别也是问题不大的……</strong></p>
<figure><img class="content-image" src="https://pic2.zhimg.com/v2-fae7e20d781360edaa942d7911bd65d0_720w.jpg?source=8673f162" alt referrerpolicy="no-referrer"></figure>
<p>问题是这样的情况还能“维持多久”？“骂完记得买”并不是一个“良性发展”的状态，而且无论是 Game Freak 目前的开发能力还是外包其他公司，暂时都不足以支撑起“年货化”的宝可梦，已经回应玩家的预期，已经有消费 IP 的倾向……这并不是积极信号……</p>
