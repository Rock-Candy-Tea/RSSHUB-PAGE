Navis Li的回答<br><br><p>这题我会，除了坐标不在上海而是在北京，基本上条件都和我很符合，我也在 Model 3 和小鹏 P7 之间纠结了很久。</p><p>不卖关子先说结论，我最后选的是小鹏 P7, 四月刚提的四驱鹏翼版，现在也开了两个月了，符合甚至有些超出预期。</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-c1895177ce7e2e774a3003acf5aa7481_1440w.jpg" data-size="normal" data-rawwidth="5887" data-rawheight="4001" data-default-watermark-src="https://pic4.zhimg.com/v2-a9a17eeec29712e3a4b9ec2892fca794_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-c1895177ce7e2e774a3003acf5aa7481_r.jpg" referrerpolicy="no-referrer"><figcaption>上牌之前拍的</figcaption></figure><p>然后说说为啥，简单来我认为两者都是值得买的车，但是 Model 3 的一些短板确实刚好在我接受不了的地方，而 P7 在智能化（特别是中国道路下的智能化），更新频率，人性化，舒适性都是能超过 Model 3 的。</p><h2><b>首先看智能化，这是我最关注的地方：</b></h2><figure data-size="normal"><img src="https://pic1.zhimg.com/v2-18cded00189a4fca78fa5c67bd60133a_1440w.jpg" data-caption data-size="normal" data-rawwidth="865" data-rawheight="649" class="origin_image zh-lightbox-thumb" data-original="https://pic1.zhimg.com/v2-18cded00189a4fca78fa5c67bd60133a_r.jpg" referrerpolicy="no-referrer"></figure><p>在刻板印象里，Tesla 才是智能化领头羊，其实也不能说完全是刻板印象，因为目前 Tesla 在美国的进展还是很让人欣慰的，它们能用摄像头纯视觉识别达到基本可用的效果。</p><p>但是，仅限美国，而且，还不够可靠。</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-94efbf3e93a45bd8cab49b5205860393_1440w.jpg" data-caption data-size="normal" data-rawwidth="865" data-rawheight="905" class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-94efbf3e93a45bd8cab49b5205860393_r.jpg" referrerpolicy="no-referrer"></figure><p>为什么说仅限美国呢？因为 Tesla 最新基于视觉的 FSD 只在美国 Beta 测试，Tesla 的外事务副总裁经常转发 FSD Beta 的内容，看起来的表现确实非常喜人，但是美国的 FSD 跟国内的 FDS 根本就是两个版本，中国区的软件更新大幅滞后于美国区（以年为单位），这一点无论是去哪一个平台搜索，都能看到中国车主的吐槽。</p><p>目前国内选装 FSD 之后只能实现：</p><ul><li>自动泊车</li><li>自动辅助导航驾驶（NOA)</li><li>智能召唤</li></ul><p>也就是下边图里 32000 的那个，如果你不买这个的话，就只有最简单的 ACC 自适应巡航以及一个 LCC 车道保持功能，后边的完全自动驾驶能力（FSD）就是 100% 的期货，现在根本用不了。</p><figure data-size="normal"><img src="https://pic4.zhimg.com/v2-6ab86975a226653b22e6fdd8d78926ae_1440w.jpg" data-caption data-size="normal" data-rawwidth="864" data-rawheight="585" class="origin_image zh-lightbox-thumb" data-original="https://pic4.zhimg.com/v2-6ab86975a226653b22e6fdd8d78926ae_r.jpg" referrerpolicy="no-referrer"></figure><p>而国内 FSD 的选装率只有 2%... 所以你在路上看到的大部分 Tesla 在自动辅助驾驶方面有可能还真没比高级一点的燃油车强多少…</p><p>但是小鹏的自动辅助就要亲民得多，在小鹏 P7 上可以选装的是 XPilot 3.0, 买车之前选装只需要 2 万人民币，比 Tesla 更便宜，它的功能和 Tesla 目前 FSD 已经实现的功能是同级别的：</p><ul><li>可以自动识别车位泊车</li><li>支持自动辅助导航驾驶</li><li>停车场记忆泊车</li><li>全向碰撞预警</li></ul><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-235d2aa70ed18beccd215e2a0f69ad6d_1440w.jpg" data-caption data-size="normal" data-rawwidth="865" data-rawheight="649" class="origin_image zh-lightbox-thumb" data-original="https://pic2.zhimg.com/v2-235d2aa70ed18beccd215e2a0f69ad6d_r.jpg" referrerpolicy="no-referrer"></figure><p>而且这几个功能实测都比 Tesla 目前国内的版本更好用，特别是 NGP 的成功率和舒适性都要远超 NOA, 具体体现在百公里接管次数更少，自动切换限速，可以躲开大货车等等。</p><p><a href="https://www.zhihu.com/zvideo/1361719028122320896" class="internal">这些都是我自己对比试过的，</a>也有非常多的媒体对比评测过，我相信有机会去试驾的话，自己也能感受出来。</p><h2><b>另外就是更新频率：</b></h2><p>Tesla 中国区我们说实话，主要的作用还是销售和制造，在研发上真的没有那么强势，主要的大版本升级都要靠美国，但是美国的自动驾驶团队做出的东西，很显然没可能再国内直接用，这就是一个很尴尬的问题。</p><figure data-size="normal"><img src="https://pic4.zhimg.com/v2-1614b55762e88498e9e9202719fdf62f_1440w.jpg" data-caption data-size="normal" data-rawwidth="880" data-rawheight="721" class="origin_image zh-lightbox-thumb" data-original="https://pic4.zhimg.com/v2-1614b55762e88498e9e9202719fdf62f_r.jpg" referrerpolicy="no-referrer"></figure><p>最简单的，我们可以去搜搜 Tesla 这段时间的 OTA 升级都升级了什么，然后你就会发现，其实基本没有多少涉及自动辅助驾驶的内容，基本都是「车机级别」娱乐系统的升级，今天加一个抖音，明天多一个 QQ 音乐歌词显示，提升一下语音识别准确性这种小改进。</p><p><b>但是小鹏的更新就要大很多也快很多了。</b></p><p>今年春节的时候 NGP 正式推送，三月多就迭代了一个更加完善的版本，最近这几天又推了一个大版本更新 V2.6.0加入了一个大功能更新：停车场记忆泊车，还有很多小的功能点升级。</p><figure data-size="normal"><img src="https://pic4.zhimg.com/v2-8ada6c3fecda8425160807002c6a86a4_1440w.jpg" data-caption data-size="normal" data-rawwidth="865" data-rawheight="392" class="origin_image zh-lightbox-thumb" data-original="https://pic4.zhimg.com/v2-8ada6c3fecda8425160807002c6a86a4_r.jpg" referrerpolicy="no-referrer"></figure><p>昨天试了一下，我现在只需要开车进地库门口，然后它就能自动把车开到我的车位并且停进去了，一路上的减速带，弯道，车辆行人都可以识别出来。</p><p>之后的更新我相信还会越来越多，每隔一段时间就有有新鲜感。</p><h2><b>舒适性，做工细节：</b></h2><p>这点估计都没什么好争议的，Tesla 进口版的做工真的不行，国产之后有了很大的改善，我觉得基本问题也不大了，只是用料确实会没有那么好。</p><p>但是舒适性上 Model 3 真的和小鹏 P7 没得比，最明显的就是噪音：</p><figure data-size="normal"><img src="https://pic1.zhimg.com/v2-5f50f34d98a7af0dfa348c67fca66379_1440w.jpg" data-caption data-size="normal" data-rawwidth="864" data-rawheight="484" class="origin_image zh-lightbox-thumb" data-original="https://pic1.zhimg.com/v2-5f50f34d98a7af0dfa348c67fca66379_r.jpg" referrerpolicy="no-referrer"></figure><p>我自己测试过 P7 在高速上开到 120 公里每小时的速度，噪音还只有 61 分贝，但是 Model 3 就算是最新的换上了双层玻璃的版本，开到 60 公里每小时的时候就已经超过 60 分贝了，如果开到 100 公里就 70 分贝。</p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-f73ecc02dca136a98683343173465dea_1440w.jpg" data-size="normal" data-rawwidth="866" data-rawheight="439" class="origin_image zh-lightbox-thumb" data-original="https://pic2.zhimg.com/v2-f73ecc02dca136a98683343173465dea_r.jpg" referrerpolicy="no-referrer"><figcaption>测试结果来自：超级车间</figcaption></figure><p>这是什么概念呢… 坐飞机的时候够吵了吧？我在坐飞机的时候也实测了一下噪音，其实飞机的噪音都只有 70 分贝左右：</p><figure data-size="normal"><img src="https://pic1.zhimg.com/v2-eee2f06f292cfda30cbbaf1bee30d96d_1440w.jpg" data-caption data-size="normal" data-rawwidth="1119" data-rawheight="1078" data-default-watermark-src="https://pic1.zhimg.com/v2-3f526d299b2852c3a5c1465f08a134ed_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic1.zhimg.com/v2-eee2f06f292cfda30cbbaf1bee30d96d_r.jpg" referrerpolicy="no-referrer"></figure><p>也就是说开着 Model 3 上高速，就可以体验到飞机的噪音，这个东西我是真的接受不了的，确实是太吵了。</p><p>除此之外还有避震，我选的 P7 是四驱版本带 CDC, 可以说是明显比 Model 3 硬到不行的减震要舒服很多，这一点也不太好描述，但是我相信大家去试驾两辆车过个减速带，就肯定明白差距了。</p><h2><b>当然车无完车，P7虽好但也并非完美：</b></h2><p>作为一个也不算便宜的车，P7 默认是没有电动尾门的，所以每次还要手动打开关闭，这一点让我很不爽，不过好在后期官方可以提供加装电尾门，所以也不算什么大事。</p><figure data-size="normal"><img src="https://pic1.zhimg.com/v2-0384cd0d641c91dcf395b074f6d4a25a_1440w.jpg" data-caption data-size="normal" data-rawwidth="6000" data-rawheight="3922" data-default-watermark-src="https://pic3.zhimg.com/v2-997a8d5329c9b567109be03b16c7d47b_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic1.zhimg.com/v2-0384cd0d641c91dcf395b074f6d4a25a_r.jpg" referrerpolicy="no-referrer"></figure><p>另一点就是驾驶性，P7 的外观是非常战斗和运动的，但是说实话它还是一个家用车的调教，主要考虑的是舒适性，而且车重和车的尺寸都要比 Model 3 大一圈，所以如果你追求人车合一的体验，P7 可能不适合你。</p><figure data-size="normal"><img src="https://pic4.zhimg.com/v2-fa6d1ceb57f34dc7146718aacaa44d22_1440w.jpg" data-caption data-size="normal" data-rawwidth="1268" data-rawheight="836" data-default-watermark-src="https://pic3.zhimg.com/v2-3d355acf7824700a0f768b98e213b54a_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic4.zhimg.com/v2-fa6d1ceb57f34dc7146718aacaa44d22_r.jpg" referrerpolicy="no-referrer"></figure><p>反过来 Model 3 就不一样了，我真的推荐性能爱好者买 Model 3 的 Performance 版本，大概需要加 8 万多，然后你就能得到更炫酷的 19 英寸轮毂，更低的悬架，碳纤维尾翼，百公里加速只需要 3.3 秒。</p><p>这时候什么悬架硬，噪音大，OTA 更新慢都不是问题，Model 3P 就是这个价位里真正可以下赛道干人的电动车。</p><h2><b>所以总结一下：</b></h2><ul><li>P7 是一个有着炫酷外观的家用车，它的舒适性明显高过 Model 3 几个档次，在中国大陆的本地化和智能化方面也要强过 Model 3 不少。</li><li>Model 3 则是一个 Tesla 压低价格开始走量的车型，它有这 Tesla 的品牌光环，而且价格也杀到了 25~30 万这个区间，很多人会因为科技属性的刻板印象和一些对外企的偏好选它。</li></ul><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-964abee7754b8ae78820eb42aa99d480_1440w.jpg" data-caption data-size="normal" data-rawwidth="865" data-rawheight="441" class="origin_image zh-lightbox-thumb" data-original="https://pic2.zhimg.com/v2-964abee7754b8ae78820eb42aa99d480_r.jpg" referrerpolicy="no-referrer"></figure><p>所以我真的推荐大家，如果想买一个过日子的，在中国更智能的车，那么可以看看小鹏 P7 的超长续航至尊版本，27 万左右就能拿下，续航还比 25 万的 Model 3 长 200 公里。</p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-3a9bf675dbc9d751fdf0ebcedbfc45bd_1440w.jpg" data-caption data-size="normal" data-rawwidth="864" data-rawheight="411" class="origin_image zh-lightbox-thumb" data-original="https://pic2.zhimg.com/v2-3a9bf675dbc9d751fdf0ebcedbfc45bd_r.jpg" referrerpolicy="no-referrer"></figure><p>如果是追求外国品牌，或者想要人车一的驾驶感受，那么可以买 Model 3 的 Performance 版本，忽略掉舒适性就好。</p>
