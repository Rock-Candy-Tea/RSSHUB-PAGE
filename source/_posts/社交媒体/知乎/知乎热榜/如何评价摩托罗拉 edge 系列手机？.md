佛系指南的回答<br><br><p>从今年的几款motorola手机产品来看，moto手机的产品线逐渐清晰，G系列主打入门级、edge系列主打中高端产品，而razr则是高端产品。今年的联想motorola不再急于求成，用产品力去塑造产品，花时间去沉淀用户基础、口碑，这样的路径在国内，显然是无比正确的：当下国内手机竞争激烈，头部厂商之间高强度竞争，需要巨大的营销资源投入，而刚刚回归国内市场没几年的Motorola手机，并不适合这种盲目的营销竞赛，逐步沉淀用户基础、获得良好的用户口碑才是立足中国市场的长久之道。</p><p>而本次联想Motorola发布会，共发布了两款机型，分别是搭载骁龙870的motorola edge s pro和搭载骁龙778G的motorola edge 轻奢版，前者主打时尚旗舰，后者主打轻薄影像。</p><figure data-size="normal"><img src="https://pic1.zhimg.com/v2-9909909e2f8988def4959c02c504b782_1440w.jpg" data-caption data-size="normal" pro：moto新旗舰，骁龙870+一亿潜望主摄</b></h2><h3><b>1、外观：7.99mm轻薄机身，素皮版质感出色</b></h3><p>motorola edge s pro采用全新外观工艺设计，拥有3种配色、2种材质的后盖，基本上满足了大家对于机身配色的需求，其中素皮材质经常出现在一些高端手机上，而此次motorola edge s pro搭载的素皮颜色为青玉色，配合十字纹素皮材质，使得整机非常有质感。</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-e344320ca00c2a77cba9c670852e47a1_1440w.jpg" data-caption data-size="normal" data-rawwidth="550" data-original="https://pic3.zhimg.com/v2-e344320ca00c2a77cba9c670852e47a1_r.jpg" referrerpolicy="no-referrer"></figure><p>机身方面，motorola edge s pro将机身尺寸控制的非常到位：7.99mm的厚度与189g重量，现在的不少旗舰手机虽然配置出色，但是重量动不动就超过了200+g，用久了手会觉着累，而仅189g重的motorola edge s pro配合7.99mm的厚度，成功的做到了拿着不累、用着舒服的境界。</p><h3><b>2、骁龙870加持，娱乐办公体验出色</b></h3><p>虽然没有用上骁龙888，但我也觉得不遗憾，因为今年888“滚烫的表现”，让这颗旗舰芯片备受争议，退一步选择骁龙870，不仅解决了库存压力，也让价格更加亲民，对于Moto粉来说，是个不错的福音。骁龙870是一颗7nm旗舰级芯片，在今年1月份的时候，由Motorola Edge S全球首发，采用1个高频大核+3个低频大核+4个小核的架构，大核主频高达3.2GHz，GPU则是Adreno 650，这样的组合，无论是日常使用还是娱乐办公，都能轻松的完成。</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-db85f6fa64684d28e1d8bd56b5641ce7_1440w.jpg" data-caption data-size="normal" data-rawwidth="552" data-rawheight="322" data-default-watermark-src="https://pic4.zhimg.com/v2-9ae0119bb1de144d048eb7d4b49179bd_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-db85f6fa64684d28e1d8bd56b5641ce7_r.jpg" referrerpolicy="no-referrer"></figure><p>此外motorola edge s pro的内存&存储升级为LPDDR5+UFS3.1，还支持3GB的虚拟内存，所以，完全不需要担心配置不够用的情况。</p><h3><b>3、144Hz高刷OLED，十亿色屏幕</b></h3><p>motorola edge s pro的Pro，也体现在屏幕上，搭载了一块6.7英寸的OLED屏幕，144Hz刷新率+最高576Hz触控采样率，支持10亿色显示和HDR10+、DCI-P3广色域，无论是观感还是使用体验，都非常出色。这块屏幕也通过了SGS认证，能够一定程度上保护视力。</p><h3><b>4、一亿像素+潜望式50倍变焦，拍照配置出色</b></h3><p>motorola edge s pro的Pro的后置摄像头的方案为1亿主摄+1600万超广角+800万长焦镜头，没有凑数的摄像头，3个镜头都有着自己的价值：1亿主摄是安卓手机这两年的旗舰配置，motorola edge s pro的这颗一亿像素主摄等效焦距26mm，1/1.52英寸超大底，支持像素九合一技术，九合一后单枚像素尺寸为2.1μm，支持0.7μm超清画质直出，800万的长焦镜头为潜望式相投，最高支持5倍光学变焦和50倍的数码变焦，支持OIS光学防抖。1600万超广角镜头使得拍照角度能拓展到121°，也支持2.8cm的微距拍摄。</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-298c59a956c26b8c719de37f16f4903e_1440w.jpg" data-caption data-size="normal" data-rawwidth="552" data-rawheight="311" data-default-watermark-src="https://pic3.zhimg.com/v2-67404c20644934eab4a54962fb0e7b80_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-298c59a956c26b8c719de37f16f4903e_r.jpg" referrerpolicy="no-referrer"></figure><h3><b>5、Ready For 2.0：属于moto的生产力工具</b></h3><p>这两年的手机厂商，都在试图打造掌上生产力，但只有个别厂商做手机也同时做电脑，所以大多数的手机厂商都是心有余力不足，而moto依托于世界第一的笔记本厂商联想的技术，能够做出一套属于自己的跨屏协作方案：Ready For。如今它也迎来了2.0版本，支持有线和无线跨设备协同，实现手机到显示器、电视、投影仪以及笔记本的跨屏显示：用户可通过大屏进行文件处理、邮件收发、图片/视频编辑等工作，且手机与屏幕间操作互不干扰，轻松实现高效多任务处理，也可以将手机变身为网络摄像头、蓝牙遥控器、触控面板等实用控制设备。</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-14847be52d70b21f2068438063bfa481_1440w.jpg" data-caption data-size="normal" data-rawwidth="554" data-rawheight="314" 轻奢版的关键词定义为：不妥协。虽然这款产品相比上面的motorola edge s pro配置弱了一些，价格低了一些，但是却有着自己鲜明的特征：轻薄。</p><figure data-size="normal"><img src="https://pic4.zhimg.com/v2-c677590e58ab156439a13d0fe4123759_1440w.jpg" data-caption data-size="normal" data-rawwidth="544" data-rawheight="272" data-default-watermark-src="https://pic3.zhimg.com/v2-a06d5ecf92d3b59253b96a603beec24f_720w.jpg" class="origin_image zh-lightbox-thumb" referrerpolicy="no-referrer"></figure><p>仅6.99mm厚度+163g重量，是motorola最轻薄5G智能手机。虽然轻薄，价格也相比Pro便宜，但是配置却也不妥协：骁龙778G处理器，6nm工艺的主流级处理器，性能表现相当不错，是这个价位的最热门处理器。</p><p>屏幕方面，完全没有阉割，和Pro一样的144Hz高刷10亿色OLED屏幕，主摄依旧是Pro同款1亿主摄+800W长焦+1600W超广角，只不过没有了潜望式变焦，最高支持3倍光学变焦和30倍数码变焦，综合CPU、拍照、屏幕来看，依旧是一款非常出色的主流级手机。</p><p>如果你对机身重量厚度较为在意，想要一款轻薄的5G手机，那么这款仅163g重量、6.99mm厚度，搭载骁龙778G的motorola edge 轻奢版，也是一个非常好的选择~</p><p></p>
