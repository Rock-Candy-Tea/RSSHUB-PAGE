胖总的回答<br><br><p>可以说：META 抢占了未来的命名制高点。</p><p>毕竟 facebook 的名字最初的由来格局太小了，本质是一个学校花名册而已，代表的也是旧的基于现实社交的 SNS 时代。</p><p>但是 META 就牛逼了。</p><p>我们经常听到「元设计」、「元编程」，还有现在最火的「元宇宙」。而且，meta 作为一个前缀，连接任何一个名词后，对会产生新的意义。</p><p>那么 META 究竟是什么？</p><p>Wikipedia 上对 meta 有如下的解释：</p><blockquote>In epistemology, and often in common use, the prefix meta- is used to mean about (its own category). For example, metadata are data about data (who has produced them, when, what format the data are in and so on). In a database, metadata are also data about data stored in a data dictionary and describe information (data) about database tables such as the table name, table owner, details about columns, – essentially describing the table. Also, metamemory in psychology means an individual's knowledge about whether or not they would remember something if they concentrated on recalling it. The modern sense of "an X about X" has given rise to concepts like "meta-cognition" (cognition about cognition), "meta-emotion" (emotion about emotion), "meta-discussion" (discussion about discussion), "meta-joke" (joke about jokes), and "metaprogramming" (writing programs that manipulate programs).<br><br>In a rule-based system, a metarule is a rule governing the application of other rules.<br><br></blockquote><p>即：</p><blockquote>在认识论中，前缀 meta- 经常被用来表示关乎自身范畴的概念。例如，元数据是关于数据的数据（谁生成了它们、何时生成、数据采用什么格式等等）。在数据库中，元数据也是有关存储在数据字典中的数据的数据，并描述有关数据库表的信息，如表名、所有者和列的详细信息，本质上是 用来描述数据表的。此外，心理学中的元记忆意味着个人对记忆的认知：如果集中精力回忆某件事，他们是否会记住它。现在在「关于 X 的 X」的这一基础上，产生了「元认知」（关于认知的认知）、「元情感」（关于情感的情感）、「元讨论」（关于讨论的讨论）、「元笑话」（关于笑话的笑话）和「元编程」（编写操纵程序的程序）等概念。<br>在基于规则的系统中，元规则是管理其他规则应用的规则。<br></blockquote><p>这就很牛逼了！</p><p>从这一点来说：</p><p>亚马逊其实只是 meta-consumption 的一种实现；</p><p>阿里巴巴只是 meta-business 的一种实现；</p><p>谷歌只是 meta-searching 的实现；</p><p>苹果只是 meta-terminal 的一种实现。</p><p>享受这种万宗归一的感觉不要太爽好吗？</p><p>更何况未来是 meta-verse 的时代，facebook 哦，不！ META 是在说：</p><p><b>我是元宇宙本元，未来世界的性质由我来定义，因为是我描述了一切事物的本质！</b></p><p>所以，认真的说：</p><p>小扎的这步棋下得很好。</p><p>但是有没有破局之法？</p><p>也不是没有，我提供几个公司的改名思路给巨头们：</p><p><b>NONE</b>：正所谓无中生有，道家认为，天下万物生于有，有生于无。你 META 再牛逼，也是「有」的层面，我才是一切的老祖宗。</p><p><b>DAO</b>：道生一，一生二，二生三，三生万物，META 也是道的一种，而且，DAO，除了是「道」的音译，还表示「去中心化组织」，即 Decentralized Autonomous Organization，更符合未来元宇宙的分布式个性化概念。</p><p><b>ZERO / ONE</b>：比上面两个弱了些，但是还可以，意思参考上面两个。</p><p>本人还有无数个起名方案，各大公司感兴趣可以知乎私信我。</p><p>不用谢！</p>
