X底层优化和隐私安全体验。AI智能主动服务、跨设备优化、底层及隐私安全升级是新系统的核心升级，也为折叠屏带来成熟实用的大屏交互体验和更高工作效率。</p><p data-pid="Plu_mo18">问题分成两个部分，一是Magic UI 6.0的升级，二是应用在折叠屏上的亮点。</p><p data-pid="X9yFVW6N"><b>先说Magic UI 6.0的升级。</b></p><h2>一、AI智能主动服务</h2><p data-pid="CjRUdvwO">不同于以往智慧功能需要我们人手主动使用，Magic UI 6.0将Magic Live全面根植于系统之中，提供了YOYO建议的功能，这个在5.0的时候就已经出现，现在更深入。日常使用时时为天气日期透明组件，当有事务需要提醒时变成非透明提醒和天气日期组件二合一，并能滚动显示。</p><p data-pid="O1pHz9ce">同时基于端云协同的AI能力智能感知我们的使用场景和需求，通过手表、手机、耳机等设备为我们提供当时所需的服务或提醒。</p><p data-pid="Z1iCOwSk"><b>1.智慧出行</b></p><p data-pid="pLsG-Qxk">Magic UI 6.0通过及时、精准的主动服务，可以做到想人之所想、快人一步，从而将我们从繁琐的操作和冗余的信息中解放出来，在日常出行时主动提醒、一键直达服务。</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-4f07acfb2841072ac7934c58e7cbfa89_1440w.jpg" data-pid="jgr-9Fsa">Magic UI的智慧识屏功能广受好评，但也需要我们主动唤醒。6.0进一步提供了主动翻译的功能。</p><p data-pid="RW0sGoN3">当我们处理外文信息时，只需复制所需的文字，YOYO建议便会主动识别、一键翻译成系统设置的语言，不需要再点开翻译软件，更加简洁高效和直观。</p><p data-pid="qajl1jae"><b>3.智慧文件夹</b></p><p data-pid="NMg85YRD">我们手机中一般都会有几十上百款应用，需要好多页才能放下，Magic UI 6.0本身提供了大文件夹功能，可以不用点开文件夹即可点击里面的APP，让四个APP的位置放了8个可直接点击的APP。</p><figure data-size="normal"><img src="https://pic1.zhimg.com/v2-14032d93c76704ecee81688b10915be0_1440w.jpg" data-rawwidth="1096" data-rawheight="448" data-size="normal" data-caption class="origin_image zh-lightbox-thumb" V操作，同时也可使用电脑的键盘进行信息输入，而这时控制的是Magic V内的应用和界面，因此可以实现电脑和Magic V的无缝操作，一体化体验更好，不需要再拿起手机单独操作。</li></ul><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-b2c511bedd95d7be7cb83b4025a579b5_1440w.jpg" data-rawwidth="982" data-rawheight="375" data-size="normal" data-caption class="origin_image zh-lightbox-thumb" Turbo移植到新的芯片平台带来三大Turbo X，提供更好的信号、游戏、日常使用体验。</p><figure data-size="normal"><img src="https://pic4.zhimg.com/v2-24990d361fbc8cc9256398ee82718685_720w.jpg" data-rawwidth="354" data-rawheight="248" data-size="normal" data-caption class="content_image" referrerpolicy="no-referrer"></figure><p data-pid="asSJkVM0">说一下其中的LINK Turbo X，一个是四网协同，主卡、副卡、2.4GWIFI网络、5GWIFI网络，根据信号情况的不同选择不同的网络；二是三网并发，主卡、2.4GWIFI网络、5GWIFI网络可实现并发下载，从而实现超越单一网络下的下载速度。荣耀在信号方面的努力和能力，的确一直有目共睹。</p><h2><b>三、安全隐私升级</b></h2><p data-pid="NfbYZyC3">在5.0的时候Magic UI已经提供了自动模糊位置、应用最小权限授权管理、维修模式等诸多保护隐私安全的措施，6.0继续加码和升级。</p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-ab58e9ac9cc8f61e27122a4092c4e083_1440w.jpg" data-rawwidth="1037" data-rawheight="378" data-size="normal" data-caption class="origin_image zh-lightbox-thumb" data-original="https://pic2.zhimg.com/v2-ab58e9ac9cc8f61e27122a4092c4e083_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="l6u5CxCF"><b>1.剪切板去隐私</b></p><p data-pid="2LLbAPCs">目前手机基本不关机或重启，存储在剪切板里面的信息就会一直保留。当我们使用应用或网站时，有可能对其开放独取剪切板的权限，如果被一些网站或应用识别，那剪切板里面的敏感信息就容易泄露，长久积累有可能拼合出用户的隐私信息链条。6.0支持自动清除剪切板数据，避免信息泄露的潜在风险。</p><p data-pid="Q5aNu5rB"><b>2.丢失模式</b></p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-f12a81916a2d781512e83abb75cef92d_1440w.jpg" data-rawwidth="808" data-rawheight="415" data-size="normal" data-caption class="origin_image zh-lightbox-thumb" data-original="https://pic2.zhimg.com/v2-f12a81916a2d781512e83abb75cef92d_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="KJ-i6WE9">这可以看作时维修模式的一种升级。当用户发现手机丢失时，可远程锁定SIM卡使得SIM卡换手机无法使用，同时拒绝强制关机，并会隐藏来电和信息显示，以此防止手机脱离控制或假冒用户身份行为。</p><p data-pid="cv91A_rz"><b>再说在折叠屏上的亮点</b></p><h2>一、硬件安全存储芯片+软件防护系统</h2><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-3b251198ef45b985f67b5580ef4a4437_1440w.jpg" data-rawwidth="720" data-rawheight="350" data-size="normal" data-caption class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-3b251198ef45b985f67b5580ef4a4437_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="ciDuLmPM">荣耀Magic V在内部配置了一颗独立存储的芯片，用于加密存储开机密码、人脸和指纹等信息，与主芯片隔离。芯片与手机绑定，防刷机或恶意读取。</p><figure data-size="normal"><img src="https://pic4.zhimg.com/v2-971faf530803abace6bdb79518f09db3_1440w.jpg" data-rawwidth="1152" data-rawheight="581" data-size="normal" data-caption class="origin_image zh-lightbox-thumb" data-original="https://pic4.zhimg.com/v2-971faf530803abace6bdb79518f09db3_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="8jnz7yYf">且以此芯片为基础构建了荣耀的TEE安全防护系统，当应用需要调用指纹等信息时只能调用TEE接口获取成功与否的结果，而不会直接获取信息。这些信息全部存储在独立存储芯片里面，不会上云，用端侧安全保证用户隐私。包括上面提到的各种ai功能，也都是利用ai模型在本地运行，不会将用户敏感数据上传云端。</p><p data-pid="dXgMFbSa">对于很多在乎个人隐私的用户或商务人士，荣耀Magic V提供的是极致的安全。</p><h2>二、沉浸大屏体验</h2><p data-pid="w78fRLpl"><b>1.大屏显示优化</b></p><p data-pid="_piGos8R">折叠屏最大的意义是多提供了一块小平板是的屏幕，不能简单将手机的显示逻辑用在展开后的大屏，而要针对性优化。</p><p data-pid="mS_Gw1Yk">荣耀Magic V将大屏内容分成三种：</p><ul><li data-pid="fqL0QJg1">内容型：如阅读类型APP，直接大幅增加内容数量</li><li data-pid="chO9lbJU">列表-详情：如系统设置/文件管理菜单，通过分栏实现父子层级展示</li><li data-pid="C0oqYtjB">简短独立任务：一些不适合大屏展示的内容，居中靠下展示，减小页面跳转</li></ul><p data-pid="r5btCfGJ"><b>2.大屏操作优化</b></p><p data-pid="NSMj2YfX">直板机宽度较窄，适合单手操作，功能键一般位于中下区域；折叠屏大屏这样不易操作，将功能键尽力放在大屏易操作的两侧，而非中间。</p><p data-pid="NIDgoSdo"><b>3.内外屏一致性交互</b></p><p data-pid="8_qMK7rV">当用户从外屏切换到内屏，或内屏切换到外屏时，Magic V做了很多交互优化，保证用户体验的一致性。</p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-2985db330d248d60291cd677cd57e12c_1440w.jpg" data-rawwidth="1126" data-rawheight="432" data-size="normal" data-caption class="origin_image zh-lightbox-thumb" data-original="https://pic2.zhimg.com/v2-2985db330d248d60291cd677cd57e12c_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="b9-c0MrW"><b>4.分屏导航键，一键分屏</b></p><p data-pid="Eza_KBad">折叠屏系统体验中，应用内分屏和多应用分屏一直是最核心的优化重心，也是改善用户体验的关键。应用内分屏Magic UI有大家熟知的平行视界，且是自动触发，用户的使用率较高。</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-5aceb55c7e1e0628737573ba1010d7c6_1440w.jpg" data-rawwidth="1018" data-rawheight="471" data-size="normal" data-caption class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-5aceb55c7e1e0628737573ba1010d7c6_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="APfTjz08">但多应用分屏，多为手势操作主动触发，但很多不了解数码的用户并不能开启，部分开启甚至不到10%，从而造成体验上缺憾。为此，荣耀在屏幕中上方会主动显示一个分屏导航键，在用户视野内提供统一的交互入口，方便用户一键使用。同时导航键的存在会自然而然引起用户的注意，进行点击尝试等，从而起到提醒的作用。</p><p data-pid="ng72iae6"><b>5.特定优化</b></p><p data-pid="a4c9n4lf">荣耀与视频、金融、创作工具等软件合作，共同推出适合大屏体验的体验优化。</p><p data-pid="Mq1V0Pcc">和咪咕视频合作提供多路视频支持，在观看体育比赛的时候实现同时看多长比赛，或看同一场比赛的不同角度，从而获得更好的体验。</p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-7cd485b65af9ab16d033e176985a5b0a_1440w.jpg" data-rawwidth="950" data-rawheight="434" data-size="normal" data-caption class="origin_image data-pid="6-2DmtmN">和同花顺合作提供更好的金融体验，一屏看到股票列表和K线详情，无需频繁切换股票列表、K线和操作等信息模块间，一键查看股票走势，信息一览无余、尽在手中。</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-83daba023b9268197e5abe809459d735_1440w.jpg" data-rawwidth="1152" data-rawheight="529" data-size="normal" data-caption class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-83daba023b9268197e5abe809459d735_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="B2OggtKV">和Ovicut应用合作提供更好的创作体验，实现多轨道音视频剪辑、编辑素材。其实也不止这个应用，折叠屏剪视频的确对于随时随地想制做轻量视频的朋友，素材展示更丰富，功能展示更多、更方便。</p><p data-pid="D5xJgEuR"><b>6.跨设备体验</b></p><p data-pid="diPg7IoV">    这个在上面已经提到了，就不在展开。在跨设备交互方面，荣耀做的一直都很出色，让Magic V能够像小平板一样与笔记本交互确实也是一个非常棒的内屏体验升级，真正发挥大屏的价值。</p><h2>总结</h2><p data-pid="btgtb0Yy">通过以上这些东西，我们可以看到，Magic UI 6.0真对折叠屏的体验有了全方位的优化，努力帮折叠屏从科技尝鲜往日常实用转变，为用户提供细腻体贴又独一无二的折叠屏体验。Magic UI 6.0也是未来荣耀所有手机的新系统，升级自然不止于此。</p><p data-pid="byroopmr">总的来说，正如开头所言：Magic UI 6.0首次发布Magic Live引擎，并以此为基础整合软硬件能力推陈出新，为我们提供智慧文件夹、智慧出行等主动服务能力，多屏协同、荣耀分享、YourPhone等跨设备体验，三大Turbo X底层优化和隐私安全体验。AI智能主动服务、跨设备优化、底层及隐私安全升级是新系统的核心升级，也为折叠屏带来成熟实用的大屏交互体验和更高工作效率。</p><p></p>
