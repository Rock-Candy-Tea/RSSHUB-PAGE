木头不木的回答<br><br><p data-pid="lW1DSnkt">暂时还不清粗你为啥想学 Python，但无外乎两个原因。第一自己感兴趣，想业余时间研究一下；第二是想以此来进入互联网行业工作。</p><p data-pid="GA4Vc2qt">但不管怎样，都希望你在开始自学之前想清楚自己的目的，只有目的清楚了，行动才有方向。</p><p data-pid="RfQp4fzL">至于你说的可不可以自学 Python，答案是完全可以。而且很多人都靠自学 Python 入职互联网，找到了不错的工作。</p><p data-pid="XDcRQYEX">接下来我们来看下如何自学 Python。</p><h3>被动学习 vs 主动学习 </h3><p data-pid="_WvM4uH-">在正式的开始学习之前，有必要了解一下<b>主动学习和被动学习</b>。</p><p data-pid="5S8iX9Af">关于主动和别动学习，还有一个非常著名的「学习金字塔」理论。</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-6a0324e6fb33e1dd4f332f349c281f38_1440w.jpg" data-caption data-size="normal" data-rawwidth="710" data-rawheight="395" data-default-watermark-src="https://pic1.zhimg.com/v2-e5bc9cc23ff5099d3cb89cdee5018afc_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-6a0324e6fb33e1dd4f332f349c281f38_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="WxevicuD">被动学习是指：听讲、阅读、视听、演示。这种被动的学习只能让你做到内容留存率的 5%、10%、20% 和 30% 的知识。</p><p data-pid="Pbws6A5U">而主动学习则是通过讨论、实践、教授给他人。这种方式会将原来被动学习的内容留存率从 5% 提升到 50%、75% 和 90%。</p><p data-pid="uKWhd-fS">而大部分入门 Python 的自学者都是在被动学习。不信，你看下你的学习方式是不是这样子的。</p><p data-pid="fRlJ5CDN">找一份教材或者教程，从头看到尾，中间跟着敲一下代码，运行一下 demo 看看是否正常，上心一点的可能会记一丢丢笔记，但也就局限于此了。</p><p data-pid="IDFMnzkU">这不就是典型的<b>被动学习</b>么。</p><p data-pid="Jyxja_Uo">要想高效的学习 Python，事实上任何一种知识点的学习都是如此，就要摒弃这种被动学习方式，化被动为主动效率才会更高。</p><p data-pid="MLXXlVT9">当你摒弃被动学习拥抱主动学习之后，你的学习效率自然会提升不少。</p><p data-pid="YhTJ2IuB">每当学完一个知识点之后，可以在公众号输出自己的学习笔记和心得，也可以和交流群里的小伙伴一起讨论如何更好的学习，甚至可以帮一些小伙伴解决他们学习过程中遇到的问题。</p><h3>学习路径</h3><p data-pid="60qjdJ9V">学习 Python 围绕基础知识、进阶知识、项目实战三块来跟进即可。</p><h3>基础知识</h3><p data-pid="R3Y2EytB">Python 的基础知识包括基本的数据结构、条件、循环、文件处理、函数、常用模块、错误和异常等模块。这部分的知识比较琐碎，小伙伴们可以选择合适自己的视频或者教程等资料。一个星期基本就能掌握个七七八八了。</p><h3>进阶知识</h3><p data-pid="5T6lFzjq">进阶知识则包含面向对象编程、迭代器、生成器、装饰器、操作符重载、上下文管理器、并发编程、垃圾回收等。</p><p data-pid="ex4qtbfK">这部分知识相对基础知识来讲，稍微难理解一些，但放在整个大的编程环境来看的话其实还是比较简单的。</p><h3>项目实战</h3><p data-pid="3cLSfxwn">如果将一座高楼大厦比做一个项目的话，那基础知识和进阶知识则相当于组成这大楼的一块块砖头。</p><p data-pid="a-kNIPQd">但如果只有砖头的话离盖高楼还差距十万八千里。这座楼的结构是怎么样子的，每块砖的位置要放在哪里，和另外一块砖如何衔接，哪里需要钢筋等都要提前想好。只有这样最后建成的大楼才会稳。</p><p data-pid="WRsuZWM_">所以学完基础和进阶知识之后一定要找一个项目来练手，只有在项目中才能不断的加强你对各个知识点的掌握程度，也只有这样你的进步速度才是最快的。</p><h2>珍藏资源推荐</h2><p
