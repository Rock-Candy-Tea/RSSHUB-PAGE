Summer Clover的回答<br><br><p data-pid="51E1ECQl">春兰秋菊，各擅胜场。</p><p><br></p><p data-pid="XLqfiPV-">但是细一想，这个问题说道说道也是比较有意思的。</p><p data-pid="YefSfGhy">按Google Scholar Metrics的统计，ICLR、NeurIPS、ICML三大会议在AI领域的Top Publications前三名遥遥领先其他会议和期刊。（CV和NLP被单独分类。）</p><p><br></p><p data-pid="A0kFO-LU">第一个差别是接受量。</p><p data-pid="2ICVDkVz">三大会差别很大的是，NeurIPS的接受量比ICLR和ICML加起来还高。所以NeurIPS是以大得多的接受量取得了和ICML、ICLR类似的影响力。</p><p><br></p><p data-pid="NAxRxmBt">第二个差别是范围。</p><p data-pid="ZZMnmhga">很多上个年代（2012年前）就开始做core ML的组其实是不怎么投、也不适合投ICLR的。单篇论文来说影响力，深度学习三巨头创办的ICLR影响力最大、最关注深度学习，但没有覆盖到很多做ML但不做DL的组。</p><p data-pid="ssMbUJdl">ICML现在的属性可能是最贴近core ML社区的。</p><p data-pid="-rUlYixf">而NeurIPS接受量大得多的一部分原因也是更多ML应用的文章会投到NeurIPS来。有更多ML应用的文章愿意投到NeurIPS对NeurIPS也并不是坏事。</p><p><br></p><p data-pid="ztwT2-l6">这三个顶会最大的风险可能是像AAAI一样接受方向变得过于繁杂，以至于没有合适的审稿机制选出好的投稿、影响部分组的投稿热情。在这方面，NeurIPS因为其规模和范围的原因，审稿压力可能是最大的。</p><p><br></p><p data-pid="1ykHsJ_z">深度学习的大火对整个ML社区的构成还是带来了很大的影响。</p><p data-pid="BPKLM4Nx">一个趋势是深度学习投稿量剧增，以至于ICLR这样以深度学习为对象的顶会应运而生而且获得巨大成功；</p><p data-pid="WjsH6KJm">还有一个趋势是ML的技术非常迅速地在CV、NLP等领域扩张了影响力，以至于绝大部分这些领域有影响力的工作都是ML/DL的成功应用。也有部分ML新应用（/新故事）的论文投到了ML顶会里来，其中很多发表在NeurIPS上。</p><p><br></p><p data-pid="y3rxW0cF">以前整个社区的人员构成大概是：Theory < Core ML ～ ML Application。</p><p data-pid="TDMgz3Is">现在整个社区的人员构成大概是：Theory < Core ML < ML Application。</p><p><br></p><p data-pid="bujTSRDj">最后Theory在ICML可能占比更多一点，core ML（包括DL）还是ICML的主体；ICLR偏爱Theory/core ML里和DL相关的部分，也有很多DL应用；NeurIPS则是规模、范围都更大的会议，数学味在变淡，Application的工作在NeurIPS里最多。</p><p><br></p><p data-pid="X8U4KC8n">我博士所在组是典型传统意义上的core ML组，算是少部分能在深度学习时代继续维持比较高的影响力的lab。除了老板给力以外，很大一个原因是我们组的主流研究不需要care模型是深度模型还是传统机器学习模型。都是一样，就是representation不同的model而已。</p><p data-pid="WJh_XpEQ">(很多上个年代做core ML的组在2012年后影响力迅速地衰退了，比如一些魔改SVM的组。)</p><p data-pid="05Z4u65h"><br></p><p data-pid="iqRjIySl">随着社区规模的变大，我们组近年的publication在ICML出版物里占比还算可以；在NeurIPS出版物里占比明显降低了（NeurIPS今年已经两千五的接受量了！我刚学ML的时候，简直不敢想。）；ICLR以前一直投得少，近两年稍微多一些。</p><p><br></p><p data-pid="Q_6miYke">个人而言，我这样做DL Theory的研究者投稿时其实还是遇到哪个就投哪个。但最后结果是publication里有一半是ICML（总数并不多），说明三大会还是有自己的偏好的。博士老板常年担任ICML和NeurIPS审稿的SAC，也担任过NeurIPS General Chair；但老板并不在ICLR担任SAC(组里一位资深research scientist学长会担任AC)。这也说明同样的学者对三个会议里的影响力和偏好也会有一定差别。</p><p><br></p><p><br></p><p data-pid="qa30ddxu"><br></p><p data-pid="EQ9fMH6w">不过这些差别都不影响这三者是ML当之无愧的三大顶会。</p><p data-pid="1BHjSV7n">社区构成的差别也会随时间变化。</p><p data-pid="8VHvHuob">最好的机器学习工作几乎都会投到这里——这是顶会影响力的根本来源。</p><p></p>
