韦易笑的回答<br><br><p data-pid="ehkerwx2">有些插件看着好玩，装上看一下你就不会再用了，还有些插件所有人都推过，一堆人推来推去还是那么几个，越看越无聊，因此再说这些也没什么意思，还不如直接看下载排行榜去。</p><p data-pid="1hEoDI0q">我选插件就一个标准：提升工作效率！分享几个我自己用的能让生活变得更容易的插件：</p><p data-pid="YOhgDyod">1）Project Manager：</p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-66ffd54f3752b77b1f824790438b2f7a_1440w.jpg" data-caption data-size="normal" data-rawwidth="1120" data-rawheight="800" data-default-watermark-src="https://pic2.zhimg.com/v2-3defe1e969a8b3bcf06d618216b0dcba_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic2.zhimg.com/v2-66ffd54f3752b77b1f824790438b2f7a_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="L2LJxwd9">用 vscode 经常在项目里切换来切换去，Open Directory 效率太低，命令行 <code>code . </code> 用着也不方便，这个 Project Manager 是解决这个痛点存在的：</p><ul><li data-pid="HDRPgek4">左上是项目收藏面板，点击 “软盘”图标可以收藏当前项目，下次直接点击就切换。</li><li data-pid="_AfuygKV">左下是项目扫描，设定几个目录，会自动扫描这几个目录下面所有 svn/git 项目，双击切换。</li></ul><p data-pid="N5TdmjmR">有了这两个面板，项目切换随心所欲，想切就切。</p><p data-pid="bmyLGXY1">2）Notes</p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-8ef0d3376c7a6757a9848e1b284d9f51_1440w.jpg" data-caption data-size="normal" data-rawwidth="1018" data-rawheight="653" data-default-watermark-src="https://pic1.zhimg.com/v2-08dbb018aa097899b63f3ed1898bd2a2_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic2.zhimg.com/v2-8ef0d3376c7a6757a9848e1b284d9f51_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="bhvLnVFh">笔记插件，安装后，设置一个存储笔记的地方，然后该路径下的 markdown 文件会专门显示在 Notes 插件的面板里，集中管理。当然有更专业的笔记软件，这个插件的优势是就在 vscode 的侧面板里，非常容易访问到，不需要多开一个软件，不影响当前 vscode 正在打开的项目，适合随手记录一些比较零碎的东西。</p><p data-pid="SbkCkwJD">尤其适合记录一些代码相关的东西，然后设置保存在 onedrive 目录里，多台电脑云同步。搭配其他 markdown 插件，可以方便的实现笔记内链（按 CTRL+鼠标左键跳转），图床之类的功能。</p><p data-pid="5PslcHzU">3）vscode-mindmap</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-18b91ccbe47569416dda99b101d2885e_1440w.jpg" data-caption data-size="normal" data-rawwidth="1018" data-rawheight="653" data-default-watermark-src="https://pic1.zhimg.com/v2-8402035765eff8a2d6c6d9144541c824_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-18b91ccbe47569416dda99b101d2885e_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="QXZICO4e">脑图支持，项目里新建一个 <code>.km</code> 文件，然后用它打开，直接编辑脑图，适合在没有装 xmind 的情况下，画一些脑图，同事支持多布局，多种风格；还有一个叫做 nano mindmap 的代替品：</p><figure data-size="normal"><img src="https://pic1.zhimg.com/v2-e1649687505c1b231d571f5473a6bc52_1440w.jpg" data-caption data-size="normal" data-rawwidth="1018" data-rawheight="653" data-default-watermark-src="https://pic1.zhimg.com/v2-b4e776b30f1894c2fc356452896f4482_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic1.zhimg.com/v2-e1649687505c1b231d571f5473a6bc52_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="dYrqmnTW">可以直接打开编辑 <code>.xmind</code> 的文件，不过快捷键设置上我更喜欢上面那个。</p><p data-pid="TDe3gWvX">4） <a href="http://link.zhihu.com/?target=http%3A//draw.io" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">http://</span><span class="visible">draw.io</span><span class="invisible"></span></a></p><figure data-size="normal"><img src="https://pic4.zhimg.com/v2-d1c619cf68204b33514f99a2990c2dfa_1440w.jpg" data-caption data-size="normal" data-rawwidth="3262" data-rawheight="1568" data-qrcode-action="none" data-default-watermark-src="https://pic1.zhimg.com/v2-4221704d2f37189c60d7a2af261ff32e_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic4.zhimg.com/v2-d1c619cf68204b33514f99a2990c2dfa_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="U_MOA-ol">很多人推荐过的矢量图工具，我是重度使用的，如果说上面两个记笔记和脑图的插件我也还是同时会用其他笔记和脑图的软件，那么矢量图方面，我只用 <a href="http://link.zhihu.com/?target=http%3A//draw.io" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">http://</span><span class="visible">draw.io</span><span class="invisible"></span></a>，功能强，专业度高。</p><p data-pid="CMmYa5IU">5）Dictionary Completion</p><figure data-size="normal"><img src="https://pic4.zhimg.com/v2-3e00fcee7aaab871131a75d14cb4a247_1440w.jpg" data-caption data-size="normal" data-rawwidth="802" data-rawheight="306" data-default-watermark-src="https://pic1.zhimg.com/v2-82c4d167983d76d8f1be96fdb42cad55_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic4.zhimg.com/v2-3e00fcee7aaab871131a75d14cb4a247_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="wMI62mlC">字典补全，可以设置一个常用英文单词词典，然后辅助你输入单词的，写英文文档很有帮助，有些长单词知道怎么读，但是写出来经常写错的，有这个就比较有帮助了，支持模糊匹配，拿不准时随意输入点你记得的部分，它也能给你推荐最接近的。</p><p data-pid="57fopHxb">6）HexEditor</p><figure data-size="normal"><img src="https://pic3.zhimg.com/v2-4c46a4a57e6803b2705496e2991ed70d_1440w.jpg" data-caption data-size="normal" data-rawwidth="1204" data-rawheight="653" data-default-watermark-src="https://pic2.zhimg.com/v2-16900fc551513fc4f64f5a324fb7168b_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-4c46a4a57e6803b2705496e2991ed70d_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="bgJyjaBl">二进制编辑器，安装后在左边文件树那里选择一个文件，鼠标右键 “Open With" 然后选择 HexEditor data-size="normal" data-rawwidth="1208" data-rawheight="658" data-default-watermark-src="https://pic3.zhimg.com/v2-13076a0322042d5f29b88121e7c6abfd_720w.jpg" class="origin_image zh-lightbox-thumb" data-original="https://pic3.zhimg.com/v2-9d2aade321246d687a897532c89dc627_r.jpg" referrerpolicy="no-referrer"></figure><p data-pid="RCNXPweB">经常在文档几个不同位置跳来跳去的，这个书签插件很实用，右键菜单里直接设置/取消书签，快捷键在不同的书签位置跳转，左边还有当前书签列表，双击立马跳转。</p><p data-pid="_6CaNoSZ">8）GitHub Repositories</p><figure data-size="normal"><img Repository 然后帖任意一个 github 项目的 url 就能打开了：</p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-acdf978daf752143409a537920c167f4_1440w.jpg" data-caption data-size="normal" 下来太费时间，用这个插件就可以快速打开远程仓库，按需请求文件内容，读起代码来飞快。</p><p data-pid="_W22egkE">9）MetaJump</p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-dba161c966852ce3129a047921c5f273_1440w.jpg" data-caption data-size="normal" data-rawwidth="1097" data-rawheight="653" data-default-watermark-src="https://pic2.zhimg.com/v2-f051463e6818911704749a163eac3bbd_720w.jpg" class="origin_image easymotion，如果你经常使用键盘，那么按 ALT+/ 然后再随便敲一个字母，比如上图中敲了个小写字母 "n" ，然后屏幕内所有出现 "n" 的地方都被蓝色高亮起来，并且上面覆盖了一个快捷字母，再按一下相应字母，就可以直接跳到该处，比如再按一下 "a" 就能跳到 13 行的 "notebook...." 的 "n" 字母上。</p><p data-pid="ejgivRqi">总共按三次键就可以移动到屏幕内任意地方，用习惯了光标移动行云流水。 </p><p data-pid="4Vk6w8wo">10）Blockman</p><figure data-size="normal"><img
