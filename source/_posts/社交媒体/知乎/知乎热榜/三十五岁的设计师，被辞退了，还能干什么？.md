苏建宽的回答<br><br><p data-pid="-Xay-zSQ">设计师有很强的行业属性，工作内容有着很大的差异，像机械设计师、工业设计师、建筑设计师、平面设计师、品牌设计师、产品设计师、UI 设计师、用户体验设计师等等，包括 Kevin 老师都可以叫形象设计师，不知道你属于哪一种？</p><p data-pid="CmSGzKFa">看到“三十五岁被辞退”这个标签，一下就让我联想到了互联网行业，从2019年起，互联网行业最劝退的就是 UI 设计师。</p><figure data-size="normal"><img src="https://pic2.zhimg.com/v2-5a0f2b5bca41ed2ca623f1b485dff0d0_1440w.jpg?source=b1748391" data-caption data-size="normal" data-rawwidth="482" data-rawheight="500" data-default-watermark-src="https://pic4.zhimg.com/v2-6b76fb140b236ef80286d33cff194555_720w.jpg?source=b1748391" class="origin_image zh-lightbox-thumb" data-original="https://pic2.zhimg.com/v2-5a0f2b5bca41ed2ca623f1b485dff0d0_r.jpg?source=b1748391" referrerpolicy="no-referrer"></figure><p data-pid="tbVSZx8T">我在这个行业工作了十七年，既做过设计师，也招过设计师，对这个岗位还算比较了解，为什么UI 设计师这么卷，以及很多人被辞退，说说我的看法。</p><h2><b>一、市场需求在萎缩</b></h2><p data-pid="Jt8vO8Ls">移动互联网行业经过十几年的野蛮发展，已经有很多相关专业、非相关专业的同学纷纷进入并站稳了脚跟。现在行业已经进入了平稳发展期，没有以前那么大的用人需求了。尤其这两年大环境不太好，很多企业都在缩编，设计师这种不能直接创造价值的，是首轮被优化的对象。</p><p data-pid="_SB-X2He">如果有哪家公司大量招聘新人设计师，那就要小心了，第一种是打着招聘旗号做培训的，第二种是要优化老员工控制成本的。</p><h2><b>二、就业门槛低，竞争压力大</b></h2><p data-pid="URzyrE_e">和研发等技术类职位相比，UI 设计师是互联网行业的一个低门槛职位，每年不仅有大量的设计类专业的学生想成为 UI 设计师，还有很多非设计背景的同学和已经在泛设计领域（平面、建筑、园艺、品牌、视觉等等）工作的同学想转行成为 UI 设计师，这其中既有互联网行业高薪的诱惑，也有众多培训机构的推波助澜。相对与其他岗位，UI 设计入门很容易，一般经过几个月的系统学习就能满足工作的基本要求。 </p><p data-pid="IE3ixQjB">但是 UI 设计想精通又很难（不要以为自己能临摹几个界面、画几个图标、再耗几年的资历就算资深设计师了）。如果没有天赋，想要真正上道儿，没有长期的打磨、积累是很难突破的，更何况设计风格几年就会变一次，即便已经入了行，中途放弃学习提升，也迟早会被后浪拍到沙滩上。</p><h2><b>三、对于大多数人来说，UI 设计师只适合入门，要学会提升</b></h2><p data-pid="pZwRTR1E">认识一些工作 10 年以上，工作核心内容仍然是 UI 设计的朋友，我承认人家做的确实好，我再怎么努力也达不到那个水平。UI 设计师这个职位可以帮助很多人进入互联网行业，但是要在公司内获得成长，必须做更贴近公司核心业务的工作。</p><p data-pid="NivZSwSR">我有几个朋友在某大厂做到设计岗的 P7、P8，很多人可能觉得他们的工作就是视觉设计、管理相关的，实际上他们的工作主要就是公司的具体业务，就是挂着设计师头衔的产品经理、项目经理。 包括我自己，2005 年以设计师（那时头衔普遍是美工）的身份参加工作，这 15 年来从设计师、创业者、项目经理、产品经理等身份不断转变。 </p><p data-pid="lxuRBO13">互联网行业日新月异，需要终身学习，所以我们必须不断拓展自己的能力，要么在<b>专业能力上精益求精</b>，在水平上高出别人一截，要么就<b>拓展自己的能力宽度</b>，学习交互设计、产品经理相关的技能成为全链路设计师。</p><p data-pid="D2Wx2qfb"><b>那么，设计师如何构建自己的核心竞争力？</b> 在长期的发展路径上，我建议要么选择向纵深发展，要么选择横向发展。</p><p data-pid="KAmdYXdB"><b>1.提高专业能力（纵向）</b></p><p data-pid="Z8QxpBvb">纵向发展就是在UI 层面做精，设计入门容易，精通困难，要提高自己的水平，如果没有天赋，就需要较长时间的修行，比别人付出更多的汗水，才能做出更好的作品。而且，这条路会很窄，因为UI设计在整个大的产品体系下对产品产生的价值有限，很快就能到顶了；</p><p data-pid="Ckrulxph"><b>2.扩展技能宽度（横向）</b></p><p data-pid="BMeBWLwq">横向发展就是多去学习交互、产品，慢慢接触到公司核心业务，做到全栈的能力，为以后发展打下基础。在很多中小公司，设计师就是要承担很多职责，这就是你的竞争力。所以现在很多设计师开始学习用户调研、产品设计方面的知识，成为全链路设计师，这样才有求职竞争力。</p><p data-pid="RFMiOnYj">下面这是我们的一位朋友，大众点评的 UED 总监的一个 PPT 里的一张图，讲到了设计师的成长路径，UI 设计师在进入职场之后，在公司框架和时间精力允许的情况下可以多接触用户研究、交互设计以及产品经理相关的知识，做的事情尽量贴近公司业务。<b>从设计走向产品、再走向产品总监或者其他管理层的例子很多，设计师的未来不一定只是设计师。</b></p><figure data-size="normal"><img src="https://pic1.zhimg.com/v2-17a60767254394354cbed85bea6afc03_1440w.jpg?source=b1748391" data-caption data-size="normal" data-rawwidth="2080" data-rawheight="1240" data-original="https://pic1.zhimg.com/v2-17a60767254394354cbed85bea6afc03_r.jpg?source=b1748391" referrerpolicy="no-referrer"></figure><p data-pid="CaMl7Y4t">每个人的思考方式、职业规划都存在差异，是继续做UI设计师，还是往其他方向发展，或者是做一个自由设计师，要靠题主自己根据自身情况及未来发展方向做好抉择。</p><p data-pid="QAiK2aIB">如果是做 UI 设计师的话，需要正视自身的能力问题，尽快弥补，也需要关注行业的细分领域，重新寻找就业机会。</p><p
