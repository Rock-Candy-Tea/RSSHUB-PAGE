src="https://pic1.zhimg.com/v2-9fe592fc77f928467234630286baeb1a_1440w.jpg?source=b1748391" data-caption data-size="normal" data-rawwidth="587" data-rawheight="786" data-pid="vffK5Abp">在证券公司做程序员，不仅要注重专业技术，也要重视业务逻辑，需要了解很多金融类相关的专业知识。</p><h3><b>二、各大银行</b></h3><blockquote data-pid="8eMdzUj7"><b>总结一下就是3点：稳定+薪资还行+没有营销压力。</b><br><b>整体上大银行的待遇＞中小银行</b><br><b>地方性银行总行的待遇＞国有行省级分行</b><br><br></blockquote><p data-pid="5iTB9Y2G">银行是个比较大的话题：3家政策行（稳定+高薪），6大国有行，12家全国股份行。</p><p data-pid="0rshxmq7">每个省的地方性城市银行，地方性农商行，未改制的省联社，还有一批地级市小银行。</p><h3><b>1）银行分类</b></h3><p data-pid="J5YbaUE9"><b>首先银行主要分为如下5类</b>：</p><p data-pid="LT8POSGq"><b>3家</b>政策性银行（国家开发银行、进出口银行、农业发展银行），总行+各省的省级分行</p><p data-pid="BfX0rQbC"><b>6家</b>中央管辖的国有行（中农工建交+邮储），结构为：总行+各省的省级分行</p><p data-pid="seRuLkaA"><b>12家</b>全国性股份制商业银行(中信、华夏、招商、深发、光大、民生、浦发、渤海、广发、兴业、恒丰、浙商），结构为：总行+各省的省级分行</p><p data-pid="5LaDBWWa"><b>已完成股份制改制</b>的各省<b>地方性</b>城市银行+农村商业银行。</p><blockquote data-pid="QGS-DCUw"><b>144家城市商业银行</b>（南京银行、杭州银行、宁波银行、重庆银行、中原银行、北京银行、上海银行，四川银行等等），<br><b>212家农村商业银行</b>（北京农商银行、重庆农村商业银行、深圳农村商业银行、广州农村商业银行、上海农商银行等等），<b>经营范围</b>基本只在本省。<br></blockquote><p data-pid="Qp50Uool"><b>25家未改制</b>的<b>省级农信联社</b>（贵州省农村信用社、河北省农村信用社联合社，辽宁省农村信用社联合社等），<b>经营范围</b>只在本省。</p><h3><b>2）根据层级分类</b></h3><p data-pid="ZzwAB0Us"><b>银行科技信息岗分为以下几类</b>：</p><p data-pid="sKs9fumP">整体上，总行下属<b>科技信息部</b>>总行下属<b>研发中心</b>>=<b>省级分行</b>>=<b>省会城市支行</b>>=<b>科技子公司</b>>其他地级市支行。</p><p data-pid="giOIw2a5">还有些信用卡中心，数据中心我不了解就不说了。</p><p data-pid="GptDIYif">1、<b>总行下属的科技信息部</b>。很少写代码，主要是做技术规划，做管理（管理外包）和运维，所以招人比研发中心少。</p><blockquote data-pid="RtLdPVJg"><b>招聘要求：</b>一般211硕士起步，学历要求最高。<br><b>稳定与待遇</b>：稳定，拿总行平均绩效，待遇不差，看银行盈利水平，每个银行差异较大，入职税前15-28万之间，后续30万+。基本不加班，朝九晚五。<br></blockquote><p data-pid="dORStcwk">2、<b>总行部门下属的软件研发中心</b>。<b>专职写代码</b>，招人比较多的：工行研发中心（西安成都珠海广州杭州北京），中行研发中心（西安合肥），农业银行研发中心（天津北京成都）。</p><blockquote data-pid="FgbqHSZf"><b>招聘要求</b>：一般211硕士起步，211&985本科也可以，本科进去性价比高，学校好，基本上去了就要。<br><b>稳定与待遇</b>：稳定，但是待遇不太高，转正后大概税前15万-25万，根据职称涨薪资。要加班，加班多少看部门。<br></blockquote><p data-pid="r5Ymmd4X">3、<b>全国性银行省分行的科技岗</b>。全国性银行在每个省都有省分行，<b>多数要写代码</b>。</p><blockquote data-pid="6xdxckO9"><b>招聘要求：</b>硕士起步，或者211本科起步，每个省可能不一样。<br><b>稳定与待遇：</b>稳定，薪资大概拿省分行平均工资，<b>待遇看当地的经济情况（效益）</b>，一般12-20万。广州和深圳，转正后就20多万，干三年，职级上去了，一年总包应该也是30万左右吧。加班看具体情况。<br></blockquote><p data-pid="62_wdYQI">4、<b>省会城市支行</b>的科技信息岗位。省分行下属管理各个地级市支行，地级市里当然<b>省会城市</b>待遇最好。<b>一般不写代码</b>。</p><blockquote data-pid="nxVkjoKr">招聘要求：本科起步，省会城市竞争最大。<br><b>稳定与待遇</b>：稳定，薪资大概12万-20万。待遇看当地的<b>盈利</b>情况。<b>省会支行待遇可能高于省分行</b>。比如<b>杭州支行</b>待遇可能高于<b>浙江省分行</b>待遇。<br></blockquote><p data-pid="IdJ6GQyX">5、<b>总行下属的科技子公司</b>。银行内部的外包公司。比如招银网络科技，建信金融科技，中银金科，浙商银行的子公司“易企银”，兴业数金，专职写代码。</p><blockquote data-pid="LALyvEfo"><b>招聘要求</b>：本科起步。<br><b>稳定与待遇</b>：稳定性肯定是下降了，但是待遇上去了，转正后一般税前20-28万。加班会比较多。优点：<b>薪酬较高（20-30万），招聘量大</b>。缺点：<b>稳定性差，无行员编制，加班多，技术一般</b>。<br></blockquote><p data-pid="nw0f4LVD">6、<b>非省会地级市分支行的科技岗</b>。非省会地级市&区县的科技岗，<b>不写代码</b>，干运维提需求这些。</p><blockquote data-pid="k_BC9cC3"><b>招聘要求</b>：本科。<br><b>稳定与待遇：</b>稳定，待遇比较一般了，税前10万左右吧，看当地分行的效益拿平均绩效，一般在当地还是中上水平。<br></blockquote><p data-pid="fJm8CxYD"><b>求职推荐：</b>3家政策性银行总部（国开行、农发行、进出口银行），最稳定。</p><p data-pid="EszrSt4P">农业银行，邮储银行（不是子公司同时待遇又高），渤海银行（天津有额外<b>20%</b> 补充公积金）</p><p data-pid="82Len9_j">杭州银行，宁波银行，四川银行。</p><p data-pid="mK_92GDm">深圳农商行，广州农商行，重庆农商行。</p><p data-pid="DXAJtHNx">工行广东省分行，广州支行、深圳支行。</p><p data-pid="ABM_CLUR"><b>不同银行、不同地区、不同部门差异很大的，具体问题具体分析，多上牛客，多看offershow，还有脉脉。想尽办法问到内部情况。</b></p><h3><b>三、烟草+国家电网+中石油+铁路集团</b></h3><p
