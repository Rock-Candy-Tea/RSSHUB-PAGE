凶悍的大灰狼的回答<br><br><p data-pid="0ZU5Xi3S">估计可以让你干到陪着公司一起倒闭为止。</p><p data-pid="CRnP6Rry">.</p><p data-pid="jeD79FPb">C++肯定不能缓解隔壁JAVA程序员们的中年危机。</p><p data-pid="5hn8Qhbm">.</p><p data-pid="0SeoEX0-">职业寿命跟就业市场的人力需求和从业者数量有关系。</p><p data-pid="-9zIhC_A">JAVA开发者有中年危机，是因为JAVA开发门槛低，前期涌入了太多赚快钱的人，JAVA的人是特别过剩的，被年龄劝退，就在所难免。</p><p data-pid="WnknDZrw">C++开发者职业寿命长，是因为它难用，门槛高，劝退了很多的人。它是属于过去已经卷过了，现在就不再卷的行业。</p><p data-pid="lpfi3_WW">你老板敢劝退C++开发人员，他负责的产品就可能直接躺平给老板看，老板也很难找到适合接手他工作的人。</p><p data-pid="OLZhuFA2">……</p><p data-pid="k8zlVVj8">我一位开发同事，离职某公司20年了，最近被返聘回去。</p><p data-pid="0a7vzzbz">我问他，能适应吗？</p><p data-pid="CfLOMPCC">他说，没关系，代码还基本上是他20年前离开的样子</p><p data-pid="yWCYTYOM">另一个离职的开发同事，转行做了渠道商，然后他跟我吐槽，二十年前的那些BUG还在他代销的产品里，他都替原公司着急，恨不得回去帮原公司改BUG。</p>
