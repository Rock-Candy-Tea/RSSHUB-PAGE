离群鸟独飞的回答<br><br><p data-pid="nMS1u9Zi">以前，我问vivo手机上的小V，“零除以零等于多少”，它会说：</p><p data-pid="6cpO1ubk">“假如你有0块饼干，要分给0个朋友，每个人能分到几块？你看，这个问题没有任何意义吧？饼干爱好者会难过，因为没有饼干吃，而你也会难过，因为你一个朋友也没有。“</p><figure data-size="normal"><img src="https://pic1.zhimg.com/v2-d0591609e9e4ccd0638870f814c1fbed_720w.jpg?source=b1748391" data-rawwidth="294" data-rawheight="296" data-size="normal" data-caption class="content_image" referrerpolicy="no-referrer"></figure><p data-pid="KIzrRncO">抛开这个伤人的回答不论，除以零确实是个困扰很多人的问题。十五除以三等于五，九除以三等于三，一除以零是多少？小学数学就会告诉你，答案是不能除。<b>但是为什么？零也是个数字，它到底哪里特殊了？</b></p><h3>小学篇 </h3><p data-pid="hnxbHj-9">小学算术里，这个问题很简单。那时我们把除法定义成“把一个东西分成几份”，分成一二三四五六七份都很容易想象，但是你要怎么把10个饼干分给0个人呢？想象不出来嘛！所以不能除。</p><p data-pid="R-NwPtgj">敏锐的同学可能会想到，要是0个饼干分给0个人的话，本来无一物，好像就没关系了。但既然无物也无人，每个人分得多少都是可能的呀，根本无法给出一个单一确定的数值。</p><p data-pid="L7Px1nF5">这结论没错，但这都是凭直觉而得到的东西。你想象不出来，不一定意味着它没有。<b>远古时代的数学是建立在直觉上的</b>，买菜是够用了，<b>但要进一步发展，就必须要有定义和证明</b>——所以，我们上了中学。</p><h3>中学篇</h3><p data-pid="nG0ST4nE">现在我们开始接触最最基本的代数学——也就是<b>解方程</b>。我们发现，除法和乘法互为逆运算，所以问   1 / 0 = ?</p><p data-pid="NcpMBXFg">就等于是解方程   0 * x = 1</p><p data-pid="OY7uq8m1">好了，按照定义，0乘以任何数都是0，不可能等于1，所以满足x的数字不存在，所以不能除。</p><p data-pid="vZn4BJKP">同样，如果问  0 / 0 = ?</p><p data-pid="v-T5o9bh">就等于是解方程  0 * x = 0</p><p data-pid="VfyDUZNC">同理，任何数字都可以满足x，所以也不能除——无法确定一个单一的答案。</p><h3>高中篇</h3><p data-pid="ysoT3AEu">等到接触了基本的形式逻辑，我们又会发现另一种证明方式：<b>反证法</b>。</p><p data-pid="mHeHBX1K">一堆真的表述，不能推出一个假的表述，所以如果我们用“能够正常地除以零”加上别的一堆真表述，最后推出假的来，那只能说明“除以零”这件事情不成立了。</p><p data-pid="RnVGKfYc">所以，已知  0 * 1 = 0</p><p data-pid="ds3mTuE0">  0 * 2 = 0</p><p data-pid="gIVtlKCk">推出 0 * 1 = 0 * 2</p><p data-pid="T0Rx1MX2">两边同时除以零，得到 ( 0 / 0 ) * 1 = ( 0 / 0 ) * 2</p><p data-pid="lgiFq6Fa">化简得到 1 = 2。</p><p data-pid="i_X4d3fi">这显然是错的啦。</p><p data-pid="4fjSpGMn">那么，问题解决了吧！其实还没有。想想另一个问题：-1的平方根是多少？</p><p data-pid="rCEPeFWD">你可能会说，-1不能开平方根，因为所有数的平方都是非负的。但是这说的是实数，我要是增加一个定义呢？定义i^2=-1，这就创造出了<b>虚数</b>，也就是<b>虚数的基本定义</b>，于是-1也能开平方根了。</p><p data-pid="nRP0Ila_">那么，<b>为何不能定义一个“新”的数，让 1 / 0 也等于它，并为这个数设立一套运算法则呢？</b>这就得去大学里回答了。</p><h3>大一篇</h3><p data-pid="hSe2A8th">刚学微积分课程就会立刻接触到∞这个符号。咦，这不就是“无限”嘛。我们都学了<b>极限</b>的概念了，那么我令b趋向于0，然后把a/b的极限定义为无穷，不行吗？</p><p data-pid="t5DRYkIf">这就立刻遇到一个问题，它的左极限和右极限不一样啊。b是从负的那头靠近0，还是正的那头？这一个是越来越负，一个是越来越正，碰不到一起去。这样的极限是没法定义的。</p><p data-pid="Qw26eDtw">因此，微积分课程里会反复说，<b>虽然用到了∞这个符号，</b>但是这只是代表一个趋势，绝对不是一个真正的数，<b>不可参与运算</b>。</p><h3>大二篇</h3><p data-pid="sm6WTo2z">我还没有上大二，没有学习到新的方法证明，读者老爷们不好意思啦</p><figure data-size="normal"><img
