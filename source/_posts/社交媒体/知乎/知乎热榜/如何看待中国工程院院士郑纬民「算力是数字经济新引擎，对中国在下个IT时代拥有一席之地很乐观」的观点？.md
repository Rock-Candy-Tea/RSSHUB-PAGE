温戈的回答<br><br><p data-pid="4vuTL1Xn">郑院士的观点可以说比较客观，且在互联网时代已经被部分验证。将来的计算会从客户端向云端转移，CIPU则是顺应此趋势下的产物。</p><p data-pid="R4WPErJ8">AI技术的不断发展，以及新的芯片应用场景不断涌现，各种PU（Process Unit）层出不穷。除了常见的传统CPU/GPU，DPU（Data Processing Unit）、NPU（Neural-Network Processing Unit）等芯片进入了人们的视野，26个英文字母也面临着即将用尽的情况。阿里云的CIPU（Cloud Infrastructure Processing Units），即用四个字母来做缩写，是一款创新型的云基础设施处理器。</p><p data-pid="Hgr3nsig">仔细的来看一下CIPU的架构，会发现它是为飞天云操作系统打造的专用处理器，是传统CPU的扩展加强版，除了担负起CPU的各种任务外，还有可以管理数据中心的硬件，如GPU、异构服务器等，对存储资源云化加速、控制整个服务器网络。CIPU的出现是用户需求和分布式技术发展共同催生的结果。</p><figure data-size="normal"><img src="https://pica.zhimg.com/v2-74799a5c8e1676aaa643ce42cf00cfbc_1440w.jpg?source=b1748391" data-caption data-size="normal" data-rawwidth="865" data-rawheight="467" data-default-watermark-src="https://pica.zhimg.com/v2-c2717d1294197087f6e823677cb93815_720w.jpg?source=b1748391" class="origin_image zh-lightbox-thumb" data-original="https://pica.zhimg.com/v2-74799a5c8e1676aaa643ce42cf00cfbc_r.jpg?source=b1748391" referrerpolicy="no-referrer"></figure><p data-pid="265yvU6G">在5G、自动驾驶、人工智能、元宇宙等新技术的推动下，数据经济的蓬勃发展。就我个人而言，也会租用云端服务器做芯片设计，好处是方便快捷、成本低，可以加速产品落地。在未来存储和计算转向云端是趋势。如果云服务器只在云端硬件做优化，还是远远不够的，所以阿里又自主研发了高性能低延迟的网络技术。HPCC（High Precision Congestion Control） 是在高性能的云网络环境下，对现有的拥塞控制的一种替代方案，在计算机网络里，传统的拥塞控制算法主要通过在端上调节流量，以维持网络最佳平衡状态，HPCC 的核心理念是利用精确链路负载信息直接计算合适的发送速率，这样用户的感受是即高速，又稳定。</p><p data-pid="Qrdp9jyQ">纵观科技巨头发展史，会发现仅有硬件或者只有软件，是无法发挥整个系统最佳性能的。比如英特尔和微软组成的wintel联盟，苹果的A系列移动处理器+IOS操作系统，都是软硬件结合，让性能发挥极致的典范。在飞天操作系统、洛神云网络，盘古存储系统等软件设施已经完备的情况下，急需一颗能协调各方的芯片，来发挥整个阿里云计算系统的性能，那么CIPU则是一款突破现有芯片体系的产品，软硬件协同设计是搭建良好云生态必不可少的一块拼图。</p><p data-pid="MCeGQTvq">在信息化时代，各个公司甚至国家越来越注重算力，算力即是国力、即是竞争力。在航空航天、工业控制、基础设施、个人应用等领域，对算力的需求越来越高。市面上的DPU、IPU、FPU都是为了解决算力出现的产品，但是仅限于某一个的领域，和CPU、GPU的通用性相比较差。而CIPU则是融合了DPU、IPU的设计理念和优点，数据吞吐更快，<a href="https://www.zhihu.com/search?q=%E8%99%9A%E6%8B%9F%E5%8C%96&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2528008768%7D" class="internal">虚拟化</a>云化能力更强。CIPU不是CPU/GPU的替代品，而是在云计算场景下更具竞争力的芯片设计方案，算力最大化的同时降低成本，节省功耗，让用户可以享受到算力的普惠。为什么这么说，看一下CIPU的几个特性就知道了。</p><p data-pid="4FtATPcA">一是在网络方面，RDMA分布式高性能网络的应用，让用户体验高性能，低延迟。在传统的CPU中，大家对DMA（Direct Memory Access）可能更熟悉，RDMA则是remote DMA，即远程的<a href="http://link.zhihu.com/?target=https%3A//baike.baidu.com/item/DMA/2385376" class=" wrap external" target="_blank" rel="nofollow noreferrer">DMA</a>技术，为了解决网络传输中服务器端数据处理的延迟而产生的。RDMA让计算机可以直接存取其他计算机的内存，而不需要经过处理器的处理。RDMA将数据从一个系统快速移动到远程系统的内存中，而不对操作系统造成任何影响，达到高<a href="https://www.zhihu.com/search?q=%E5%90%9E%E5%90%90%E7%8E%87&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2528008768%7D" class="internal">吞吐率</a>、低延迟、占用CPU资源少的效果，RMDA的原理如下图所示。</p><figure data-size="normal"><img src="https://pic1.zhimg.com/v2-337540ce897e759c323ed63926330482_1440w.jpg?source=b1748391" data-caption data-size="normal" data-rawwidth="673" data-rawheight="337" data-default-watermark-src="https://pic4.zhimg.com/v2-3d230abdcdc6f3a029159fb227a89a5b_720w.jpg?source=b1748391" class="origin_image zh-lightbox-thumb" data-original="https://pic1.zhimg.com/v2-337540ce897e759c323ed63926330482_r.jpg?source=b1748391" referrerpolicy="no-referrer"></figure><figure data-size="normal"><img data-original="https://pic3.zhimg.com/v2-bb2ff8123ffac699a195b24ba3f24c23_r.jpg?source=b1748391" referrerpolicy="no-referrer"></figure><p data-pid="Aft48d3o">二是在对存算分离架构的块存储接入进行硬件加速，提供超高性能的云盘。存算分离是一种架构思想，存储计算分离架构可以让大数据集群充分利用资源，可弹性拓展，更灵活，更符合云计算的特性。CIPU可以通过全硬件虚拟化和转发加速，存储时延最低可至30us（PLX），IOPS高达300万，存储带宽可达200 Gbps。这一速度超过了业绩所有的同类产品。PLX云盘和其他云盘性能参数如下图所示。</p><figure data-size="normal"><img src="https://pica.zhimg.com/v2-2fc3c176f11ca2c2fc00272a5840a145_1440w.jpg?source=b1748391" data-caption data-size="normal" data-original="https://pica.zhimg.com/v2-2fc3c176f11ca2c2fc00272a5840a145_r.jpg?source=b1748391" referrerpolicy="no-referrer"></figure><p data-pid="cLR6uw-n">三是CIPU对其他云基础设施的协调管理。CIPU相比CPU，更加的强调管理的功能，除了云化资源、网络资源、存储资源外，CIPU还可以智能调度CPU服务器、GPU服务器，异构服务器等，根据需要来快速介入不同的服务器，充分发挥算力，进而提高大数据、AI、深度学习等场景下的效率。</p><p data-pid="87s2Yt40">在美国对中国发起科技战以来，芯片等核心技术越来越被科技公司所重视，核心科技自主不仅能保证公司长远发展的稳定，更能强化其在业界的竞争力。在过去几十年，芯片一直为国外所主导，现如今，这个主导趋势在向国内转移。向阿里等具备资金、技术和创新能力的公司，理应走在科技的前沿。创新是科技公司长久发展的根本，期待有更多的公司能研发出创新型的软硬件解决方案，为科技发展赋能，为社会提供更优质的服务资源。</p>
