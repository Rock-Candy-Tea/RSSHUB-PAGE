报告表示，随着 IDFA 在苹果更新政策后广告定位能力大幅下降，利用头部 IP 进行宣传这一备选方案的重要性也在逐渐增加。针对这一情况，Sensor Tower 对 2020 年美国手游市场中的 IP 联动作品的表现做了详细分析。<br>
<br>
<strong><font color="#de5650">美国手游 IP 市场规模达数十亿美元</font></strong><br>
<br>
Sensor Tower 发现，在 2020 年美国市场最畅销的 10 大手游中，有三款都与知名 IP 展开了合作。这其中包括 Niantic 的「精灵宝可梦 GO（Pokémon GO）」、腾讯旗下的「绝地求生（PUBG Mobile）」，以及 Activision 与腾讯合作开发的「使命召唤：移动版（Call of Duty: Mobile）」。<br>
<br>
除此之外，来自 Scopely 的「漫威英雄战队（Marvel Strike Force）」以及 Bandai Namco 的「七龙珠 Z：爆裂大战（Dragon Ball Z: Dokkan Battle）」也登上了美国 App Store 和 Google Play 手游畅销榜的前 20 名。总体而言，这五款 IP 联动游戏在 2020 年累积营收 14 亿美元，其中「精灵宝可梦 GO」（Pokémon GO）获得了 4.8 亿美元。<br>
<br>

<strong><font color="#de5650">IP 联动手游市场份额占比达 23%</font></strong><br>
<br>
据统计，在 2020 年的美国手游玩家总支出中，有 23% 来自基于 IP 的游戏产品；IP 联动手游在下载总量中的占比相对更少，为 17%。不过值得注意的是，IP 联动手游的市场份额占比已经远远超过了它们的实际数量占比。虽然只有 9% 的游戏与 IP 联动，但它们却占据了近 1/4 的玩家支出。<br>
<br>

除了市场份额占比较高之外，IP 授权手游的增速也与非授权游戏相当，甚至超越了非授权游戏。如果将美国市场中 Top 10 IP 授权游戏与非 IP 授权游戏进行对比，那么这两类游戏在 2020 年有着相似的营收增幅（33.8%）。不过美国市场中最畅销的 10 款 IP 授权游戏在下载量上增长 31%，而十大非 IP 授权游戏的下载量则下降了 3.9%。<br>
<br>
<strong><font color="#de5650">头部 IP 优势明显</font></strong><br>
<br>
在 2020 年的美国手游市场中，主机游戏 IP 占全部 IP 联动手游营收的 1/3，这是其他 IP 类型的两倍多。在这类游戏中，「精灵宝可梦 GO」（Pokémon GO）占了去年 IP 联动手游玩家总支出的 12%，「绝地求生」（PUBG Mobile）以 7% 的份额紧随其后，「使命召唤:移动版」（Call of Duty: Mobile）则 6% 位居第三位。<br>
<br>
在其他 IP 类型中，漫画以 13% 的市场份额排名第二，电视节目以 12% 的市场份额排名第三。其中漫画也是增长最快的 IP 类型，同比增长 54 个百分点。<br>
<br>

美国手游市场中的漫画 IP 中主要以日本漫画为主，例如「七龙珠 Z（Dragon Ball Z）」和「游戏王（Yu-Gi-Oh!）」等，这些漫画深受美国用户欢迎。与此同时，漫威（Marvel）则占据了漫画 IP 中的绝大多数玩家支出，「漫威英雄战队（Marvel Strike Force）」和「漫威超级争霸战」（Marvel Contest of Champions）是该类型中的领头羊。<br>
<br>
<strong><font color="#de5650">硬核 IP 增速最快</font></strong><br>
<br>
最后，Sensor Tower 分析了 2020 年美国各类手游营收增幅与联动 IP 类型的关联性。数据显示，射击、RPG 和动作等硬核游戏在 IP 联动手游中占比最高，这几类游戏的平均增幅也最大。<br>
<br>

此外，虽然益智类是美国市场中份额最高的游戏品类之一，但该类游戏的营收中只有 5% 来自 IP 授权产品。与此同时，体育类手游中 IP 联动游戏则占所有玩家支出的 1/3 以上。<br>
