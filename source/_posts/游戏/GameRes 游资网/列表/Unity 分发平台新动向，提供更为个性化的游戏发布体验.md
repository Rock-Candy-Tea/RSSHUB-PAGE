作为 Unity 的移动游戏分发解决方案，Unity 分发平台使开发者不需要为每个应用市场创建不同版本，只需提供一个版本和一个元数据配置文件，即可通过 Unity 分发平台控制台轻松地将游戏提交到每个合作伙伴商店。<br>
<br>
Unity 分发平台的规模正在持续扩大、体验正逐步个性化。下文将介绍 Unity 分发平台的新功能、新合作伙伴，以及其它最新动向。<br>
<br>
<div align="center">
<img id="aimg_999942" aid="999942" zoomfile="https://di.gameres.com/attachment/forum/202108/11/130341c1lyuk4v7gl3jz7v.jpg" data-original="https://di.gameres.com/attachment/forum/202108/11/130341c1lyuk4v7gl3jz7v.jpg" width="600" inpost="1" src="https://di.gameres.com/attachment/forum/202108/11/130341c1lyuk4v7gl3jz7v.jpg" referrerpolicy="no-referrer">
</div><br>
<strong><font color="#de5650">通过 My Game Plan 服务提交游戏</font></strong><br>
<br>
Unity 分发平台将提供更为个性化、有针对性的游戏发布体验：My Game Plan。My Game Plan 服务将整合至 Unity 分发平台控制台，开发者可自行决定是否使用。如果使用功能，需提供以下信息：<br>
<br>
<ul><li>希望在哪些应用商店发售游戏</li><li>游戏采用哪种变现方式（广告、IAP、本体付费、或多种方式组合）</li><li>游戏将支持哪些语言</li><li>游戏的发售地区<br>
</li></ul>接下来，这个功能将根据收集到的信息提出建议，帮助用户为游戏提交制定一个"游戏计划"，实时指导开发者根据每个商店的具体要求来修改元数据及游戏包，以此来尽可能减小游戏被 Unity 分发平台及商店拒收的可能，让开发者能在较短的时间内完成游戏发售。<br>
<br>
My Game Plan 已正式推出，用户可在 Unity 分发平台控制台的 My Gams 功能栏或 Unity 分发平台的操作面板页面访问这一功能。<br>
<br>

</font></font></div><div align="center"><font size="2"><font color="#808080">最新合作伙伴Uptodown</font></font></div><br>
<strong><font color="#de5650">将帮助开发者创建商店页面</font></strong><br>
<br>
Unity 分发平台最新的商店合作伙伴 Uptodown 不仅会提供便于浏览的商店页面和 80% 的收入分成，还将为开发者制作定制的图像资源与文本内容。这样做的原因是为了增加游戏的曝光度、优化搜索引擎，让游戏能被更多人看见，从而提高下载和安装量。<br>
<br>
在超过 1.3 亿的 MAU 和全球市场覆盖率的加持下，Uptodown 的这个优势可为移动游戏制造更多的宣传、体验及售出机会。<br>
<br>

<strong><font color="#de5650">其他Unity分发平台资讯</font></strong><br>
<br>
Unity 分发平台的控制台将与所有其他 Unity 服务和产品一起迁移到 Unity Dashboard，而且 Unity 分发平台的定位将更加简单，即将服务本身和信息资源关联起来，帮助开发者利用 Unity 分发平台更为高效地分发游戏。<br>
<br>
Unity 分发平台目前已登陆 Unity Dashboard，并将在今年夏天完成迁移。<br>
<br>
Unity 分发平台同样能满足高级用户的技术需求，如果想要实现游戏包自动化部署，用户可使用 Unity 分发平台团队编写的 API 直接将游戏二进制文件上传至控制台。<br>
<strong><font color="#de5650"><br>
Unity分发平台的未来计划</font></strong><br>
<br>
如今，全球有上千个游戏工作室正在使用 Unity 分发平台在各个应用商店上发售移动端游戏。此外，有超过 1500 款游戏正逐渐接触到全球各地的新受众，且这个数量还在稳固增长。<br>
<br>
包括 Azur Games 和 Animoca Brands 等发行商，以及市面上一些闻名遐迩的 3A 移动端游戏开发商都在使用 Unity 分发平台。这正是因为 Unity 分发平台能帮助安卓开发者在应用商店上发售游戏时节省资源、提高效率，从而提高游戏的市场占有率，为开发者开拓一条更为稳固的成功之路。<br>
