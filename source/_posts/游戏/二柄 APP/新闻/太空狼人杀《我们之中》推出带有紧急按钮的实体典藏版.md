【太空狼人杀《我们之中》推出带有紧急按钮的实体典藏版】《我们之中》开发商Innersloth宣布将与Limited Run Games合作，在PC平台上推出实体典藏版。Limited Run Games经常将数字游戏转换为实体游戏，并提供大量额外的内容。玩家们可以在太平洋时间5月11日上午7点从Limited Run Games商店开始预购《我们之中》实体典藏版，售价为79.99美元。实体典藏版里最有趣的东西是盒子本身，它的形状就像是游戏中的紧急按钮桌。盒体是蓝面白边，最上面写着“紧急情况”，中间是一个装在塑料盒子里的闪亮的红色大按钮，正等着你按下。更棒的是，紧急按钮实际上是有用的，而且还设置了各种各样的《我们之中》官方声音。在其中包含了这款游戏的两个副本，很遗憾它们都是数字版本，套餐中没有实体副本。它们其中一份是Steam的，而另一份则是Epic的。至于典藏版的剩余内容，有许多玩家所期待的内容和一些其他有趣的内容。其中有一个内含游戏环境音乐的CD盒；以及一个冒名顶替者主题的铁盒，里面有一张18 × 24英寸的双面海报、一张名为“概念艺术和和访谈”的光盘。<br><img src="https://erbingeditor.diershoubing.com/202105061306114143html_img.jpg" referrerpolicy="no-referrer"><br><img src="https://erbingeditor.diershoubing.com/202105061306141241html_img.jpg" referrerpolicy="no-referrer"><br><img src="https://erbingeditor.diershoubing.com/202105061306181690html_img.jpg" referrerpolicy="no-referrer"><br><img src="https://erbingeditor.diershoubing.com/202105061306189413html_img.jpg" referrerpolicy="no-referrer"><br><img src="https://erbingeditor.diershoubing.com/202105061306195018html_img.jpg" referrerpolicy="no-referrer">
