【《洛奇英雄传》制作人新作《Project HP》公布正式名称】由Nexon开发，《洛奇英雄传》制作人李恩锡主导的新作《Project HP》今日公开了正式名称“Warhaven”。本作是一款免费游戏，主打多人冷兵器对战，支持中文，现已上线Steam商城页面，具体发售暂未公布。具体配置如下：

最低配置: 
操作系统: Windows 10 64 bit
处理器: Intel Core i7-4790 3.6GHz / AMD Ryzen 5 2600 3.4GHz
内存: 16 GB RAM
显卡: NVIDIA GTX 950 / Radeon R9 270X
DirectX 版本: 11

推荐配置: 
操作系统: Windows 10 64 bit
处理器: Intel Core i7-10700 2.90GHz / AMD Ryzen 5 3700X 3.6GHz
内存: 16 GB RAM
显卡: NVIDIA RTX 2060 / AMD RX 5600XT
