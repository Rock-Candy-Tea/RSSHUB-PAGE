【《COD先锋》PC版详细配置要求公布】《COD先锋》官方目前公布了游戏PC版详细配置要求，要求如下：
操作系统
最低：Windows 10 64位(最新更新)
推荐/电竞/终极4K：Windows10 64位(最新更新) 或 Windows 11 64位(最新更新)

CPU
最低：Intel Core i3-4340 或 AMD FX-6300
推荐：Intel Core i5-2500K 或 AMD Ryzen 5 1600X
竞技：Intel Core i7-8700K 或 AMD Ryzen 7 1800X
终极4K：Intel Core i9-9900K 或 AMD Ryzen 9 3900X

内存
最低：8 GB
推荐：12 GB
电竞/终极4K：16 GB

硬盘空间
最低：发布时为36 GB(仅多人模式和僵尸模式)
推荐/电竞/终极4K：发布时为61 GB

Hi-Rez缓存(Hi-Rez缓存是可选的硬盘空间，可用于传输高分辨率图像，该选项可以在游戏设置中关闭)
最低/推荐/电竞：32 GB
终极4K：64 GB

显卡
最低要求：NVIDIA GeForce GTX 960 或 AMD Radeon RX 470
推荐：NVIDIA GeForce GTX 1060 或 AMD Radeon RX 580
电竞：NVDIA GeForce RTX 2070/RTX 3060 Ti 或 AMD Radeon RX 5700XT
终极4K：NVIDIA GeForce RTX 3080 或 AMD Radeon RX 6800 XT

显存
最低：2 GB
推荐：4 GB
电竞：8GB
终极4K：10GB

NVIDIA/AMD推荐驱动程序
NVDIA：472.12
