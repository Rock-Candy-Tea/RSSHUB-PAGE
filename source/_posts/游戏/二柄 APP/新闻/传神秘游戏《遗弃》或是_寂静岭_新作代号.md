近期有一个关于“寂静岭”很玄学的新传闻，也不知道是真是假，这个传闻称之前索尼发布会上亮相的恐怖游戏“遗弃（Abandoned）”，虽然这个游戏的开发商名为BLUE BOX Game Studios，但却很可能是小岛工作室、Konami、索尼合作的“寂静岭”新作（虽然之前Konami曾辟谣过类似传闻）。比如外媒playstationlifestyle就挖出了一下细节（虽然听起来比较玄学）：
1.BLUE BOX Game Studios这家工作室很神秘，过去几乎没有开发过什么游戏，却能突然登上索尼发布会
2.BLUE BOX Game Studios官推曾发过一条推文，指出“Abandoned”并不是游戏的实际名称，并暗示游戏名是“S”开头，并以“L”结尾。
3.BLUE BOX Game Studios的LOGO和PlayStation Studios的LOGO字体相同。
4.该工作室目前唯一出现在大众视野的开发者名为Hasan Kahraman，似乎也找不到关于他的任何从业经历。有趣的是，Hasan Kahraman、Hideo Kojima名字首字母相同，甚至如果将“Hideo”从日语翻译成土耳其语，就会变成“Kahraman”。
5.之前夏日游戏节上公布的《死亡搁浅 导演剪辑版》预告中，弩哥一直在玩耍盒子的梗，现在看来可能其实另有含义。
6.当年小岛就在《合金装备5：幻痛》上有玩过类似的花招，最初是“白鲸工作室”公布了一款名为“幻痛”的游戏，最后小岛才证实这就是《合金装备5：幻痛》。

当然，对于以上传闻也有玩家和媒体认为或许真的就是巧合，只是大家实在太想玩到“寂静岭”新作，所以才脑补过多了。<br><img src="https://erbingeditor.diershoubing.com/6/2021/06/19/1817302889.jpg" referrerpolicy="no-referrer"><br><img src="https://erbingeditor.diershoubing.com/6/2021/06/19/1817375336.jpg" referrerpolicy="no-referrer"><br><img src="https://erbingeditor.diershoubing.com/6/2021/06/19/1833201589.jpg" referrerpolicy="no-referrer"><br><img src="https://erbingeditor.diershoubing.com/6/2021/06/19/1833204449.jpg" referrerpolicy="no-referrer">
