<p>大家小时候或许也有在书本报刊上玩到过下面这样的纸上迷宫：只要拿上一支笔，从图上的起点出发，描出一条通向出口的线就算通关。<br></p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/8059/90311f900c7176183c8b526772304055.png_mw680water" width="1053" height="791" class alt="1991年海天出版社出版的《迷宫世界》，照片来自微博@Toysdrugs" title="1991年海天出版社出版的《迷宫世界》，照片来自微博@Toysdrugs" src="https://alioss.yystv.cn/doc/8059/90311f900c7176183c8b526772304055.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">1991年海天出版社出版的《迷宫世界》，照片来自微博@Toysdrugs</span></p><p>规则虽然简单，但设计优秀的迷宫却能在小小的一页纸上就搭建起一个庞大复杂的小世界，小玩家们往往画着画着一节课的时间就过去了。</p><p>当年的大家也舍不得一个迷宫只玩一遍就作废，所以往往是用铅笔轻轻地描线，通关了再用橡皮把笔迹都小心地擦去，这样一本迷宫书能爱不释手地玩上好几通。</p><p>而随着时代变迁，当年以黑白印刷为主、线条简单直白的纸上迷宫也早已日新月异地进化，现如今的迷宫书不仅会有主题和故事，迷宫本身往往也会和画面场景相融，既是游戏书，也是优美的插画绘本。</p><p>比如你第一眼看到《迷宫大侦探皮埃尔》系列的时候，多半并不会发现这是一本迷宫书。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/8059/1872401aec3c315b282ab155d1a620cd.png_mw680water" width="350" height="350" class alt="《迷宫大侦探皮埃尔》系列已被翻译成28种语言畅销30多个国家" title="《迷宫大侦探皮埃尔》系列已被翻译成28种语言畅销30多个国家" src="https://alioss.yystv.cn/doc/8059/1872401aec3c315b282ab155d1a620cd.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">《迷宫大侦探皮埃尔》系列已被翻译成28种语言畅销30多个国家</span></p><p>《迷宫大侦探皮埃尔》诞生于日本的IC4DESIGN，这家艺术工作室以绘本风格的插画作品而著称，纽约时报、奔驰、尼康等企业都曾采用IC4DESIGN的作品用于宣传。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/8059/581e06227f38361c932ae458d283215c.jpeg_mw680water" width="1069" height="1118" class alt="松下公司就曾把IC4DESIGN的插画作为官网首页的背景图" title="松下公司就曾把IC4DESIGN的插画作为官网首页的背景图" src="http://www.yystv.cn/p/undefined" referrerpolicy="no-referrer"><span class="picinfo">松下公司就曾把IC4DESIGN的插画作为官网首页的背景图</span></p><p>IC4DESIGN的插画通常有着极其丰富的细节、鲜明艳丽的色彩以及富有层次感的构图，既在整体上带给观众十足的视觉冲击力，又让人忍不住放大图片观摩把玩图上的每一个细节，可以说就算不是设计成迷宫，也已经具备了 “可玩性”。</p><p>而当《迷宫大侦探皮埃尔》在2014年推出的时候，立刻就成为了同类书籍中的佼佼者。</p><p>迷宫书通常是面向儿童的益智读物，但在《迷宫大侦探皮埃尔》的书评里，你总能看见说自己玩得比孩子更沉迷的父母们。</p><p>阅读者将在书籍中扮演大侦探皮埃尔，穿过一个又一个迷宫场景，追捕嫌犯。这些迷宫并不是横平竖直的简单线条组成，而是一个个由丰富细节构筑起的场景。在岔路尽头挡住你去路的往往不是一堵墙，而是看热闹的人群，或是恐龙探出的尾巴。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/8059/ed2cb6c5116ec7773a25f45b0a7bf709.png_mw680water" width="1080" height="650" class alt="乍看起来眼花缭乱的画面，放大后则是一个个生动有趣的细节" title="乍看起来眼花缭乱的画面，放大后则是一个个生动有趣的细节" src="https://alioss.yystv.cn/doc/8059/ed2cb6c5116ec7773a25f45b0a7bf709.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">乍看起来眼花缭乱的画面，放大后则是一个个生动有趣的细节</span></p><p>除了画面靓丽、细节丰富，《迷宫大侦探皮埃尔》的玩法内容也比一般迷宫书要更加丰富。</p><p>除了传统的“从起点连线到终点”以外，这里的每一个迷宫还设置了一些小任务，比如找到星星或是其他收藏品，又或者是解决图上遇到的小谜题，这么一来玩家即便是走进岔路也时常会有惊喜。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/8059/3f3ac2d1adcb0612acf04170f234318a.png_mw680water" width="600" height="404" class alt="每个迷宫的下方会给出支线任务和小目标" title="每个迷宫的下方会给出支线任务和小目标" src="https://alioss.yystv.cn/doc/8059/3f3ac2d1adcb0612acf04170f234318a.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">每个迷宫的下方会给出支线任务和小目标</span></p><p>单个迷宫里遍布了互相关联的线索，而大迷宫之间也有连贯性，你常常能在新一页迷宫上找到之前见过的人物，他们可能是一直在寻找宝物的探险家，也可能是一对出来约会的情侣……玩家通关一个又一个迷宫，这些人物的故事也随之发展。</p><p>这么一来，玩家的体验也不再只是以上帝视角解开一个迷宫，更像是经历了一场身临其境的冒险。明明都是静态的画页，读完却仿佛看了一场动画电影。</p><p>那如果把《迷宫大侦探皮埃尔》制作成电子游戏，让场景里的人物都动起来，那是不是会变得更加有趣好玩呢？——一名法国的游戏开发者就在陪女儿阅读这套丛书时产生了这样的想法，通过公司联系到了IC4DESIGN进行合作，又耗费三年时间实现了这个想法，制作出了电子游戏《迷宫大侦探》。</p><p>游戏忠实地还原了绘本插画的美术风格，玩家们不再需要放大镜就可以看清迷宫的每个细节。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/8059/89cb99f395acf40b98f04c4528c39706.png_mw680water" width="1080" height="608" class alt="游戏的主菜单界面就已经充满了IC4DESIGN的风格" title="游戏的主菜单界面就已经充满了IC4DESIGN的风格" src="https://alioss.yystv.cn/doc/8059/89cb99f395acf40b98f04c4528c39706.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">游戏的主菜单界面就已经充满了IC4DESIGN的风格</span></p><p>绘本里那些栩栩如生的角色和场景如今真正地“跃然纸上”，建筑和物件之间的景深层次也得以展现，代入感和冒险感都更上一个台阶。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/8059/7b2bdb738f4c7305edf31716722a2d87.gif_mw680" width="480" height="270" class alt="玩家操作的主角代替了在绘本上画线" title="玩家操作的主角代替了在绘本上画线" src="https://alioss.yystv.cn/doc/8059/7b2bdb738f4c7305edf31716722a2d87.gif_mw680" referrerpolicy="no-referrer"><span class="picinfo">玩家操作的主角代替了在绘本上画线</span></p><p>数字化也实现了角色之间的互动和对话，让这个天马行空的想象世界变得更加丰富多彩。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/8059/9932e988d1d496ecdc48352b2cbf5cf7.png_mw680water" width="1080" height="836" class alt="每走两三步就能遇到可以互动的场景，也能发现许多有趣的秘密" title="每走两三步就能遇到可以互动的场景，也能发现许多有趣的秘密" src="https://alioss.yystv.cn/doc/8059/9932e988d1d496ecdc48352b2cbf5cf7.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">每走两三步就能遇到可以互动的场景，也能发现许多有趣的秘密</span></p><p>设计师们也借着这个机会添加了更多的细节和彩蛋，在游戏里处处能遇到意向不到的致敬，你会在楼顶上碰上正准备信仰之跃的刺客，也能在商店瞧见膝盖中了一箭的士兵，还能给路边的汽车绑上气球送走。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/8059/d87fb650394f2eee383f95f087e63480.png_mw680water" width="585" height="387" class alt="如果你玩过《合金装备5 幻痛》，那对这一幕一定不会陌生" title="如果你玩过《合金装备5 幻痛》，那对这一幕一定不会陌生" src="https://alioss.yystv.cn/doc/8059/d87fb650394f2eee383f95f087e63480.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">如果你玩过《合金装备5 幻痛》，那对这一幕一定不会陌生</span></p><p>玩家探索的角落越多，地图上也会呈现更多的指示线索，这样的动态难度降低了卡关的挫败感。而悦耳舒适的配乐配音，也让阅读这个故事的体验变得更加轻松愉快。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/8059/c96204227930a7c8ad4d6eb17ca7f2a5.png_mw680water" width="674" height="874" class alt="《迷宫大侦探》里的小游戏和收集品比起绘本也更加丰富" title="《迷宫大侦探》里的小游戏和收集品比起绘本也更加丰富" src="https://alioss.yystv.cn/doc/8059/c96204227930a7c8ad4d6eb17ca7f2a5.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">《迷宫大侦探》里的小游戏和收集品比起绘本也更加丰富</span></p><p>不论是想独自回味童年，还是要与人一起探索攻略迷宫，只要你童年时也曾有过进入书中世界亲身探险的梦想，那么《迷宫大侦探》或许真的可以实现这样的体验。</p>        <br>
