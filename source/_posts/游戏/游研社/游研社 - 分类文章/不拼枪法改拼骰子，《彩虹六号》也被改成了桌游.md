<div>   
<blockquote class="doc-blockquote"><p><b>
    这回打不中人，是天亡我，非马枪之罪也。</b></p></blockquote><p>
    育碧的《彩虹六号：围攻》是一款高度强调战术策略的射击游戏，而桌游同样考验玩家的智慧。把两种游戏的体验合二为一，便是桌游开发商Mythic Games取得官方授权后，开发的衍生桌游《六号：围攻》（6：Siege）。</p><p>
    《六号：围攻》自6月23日起在Kickstarter网站上发起众筹，15分钟内就达到了10万美元的众筹目标，本文截稿时，《六号：围攻》已经筹到了超过83万美元，预计在明年6月正式发售。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/4de8eb166adcf74a54b14be81c73fb38.png_mw680water" width="1049" height="505" src="https://alioss.yystv.cn/doc/8073/4de8eb166adcf74a54b14be81c73fb38.png_mw680water" referrerpolicy="no-referrer"></p><p>
    仅是看到宣传图中琳琅满目有模有样的干员模型、零件与算子，不难看出，《六号：围攻》的游戏体验绝对不是打打牌那么简单，反倒更像是在下一盘棋，严格来说是一盘战术级兵棋。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/042e1801dc43f75130125421237159e1.png_mw680water" width="969" height="545" src="https://alioss.yystv.cn/doc/8073/042e1801dc43f75130125421237159e1.png_mw680water" referrerpolicy="no-referrer"></p><p>
    《六号：围攻》主打1V1对战，对战双方分别操纵由五名不同干员组成的进攻方与防守方，完成各自的任务目标。任务形式与原作相同，分为控制区域、拆除炸弹、拯救人质三种，而消灭对方所有干员，永远都是普适的胜利条件。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/82ea07de2df3188844065350731509f8.gif_mw680" width="680" height="370" src="https://alioss.yystv.cn/doc/8073/82ea07de2df3188844065350731509f8.gif_mw680" referrerpolicy="no-referrer"></p><p>
    作为桌游，一局《六号：围攻》消耗的时间可能够我们在原作中打上四五局游戏，规模却仅相当于原作中的一回合。游戏流程划分为部署阶段与行动阶段，行动阶段有五个回合，每一回合又分为进攻方与防守方轮流操纵干员的两个行动轮。</p><p>
    制作团队额外准备了配套的手机APP，效仿国际象棋，限制部署与行动阶段各回合的时间。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/428f3222c6e5e0a80ead7800fd06e240.png_mw680water" width="835" height="494" src="https://alioss.yystv.cn/doc/8073/428f3222c6e5e0a80ead7800fd06e240.png_mw680water" referrerpolicy="no-referrer"></p><p>
    尽管时间与操作次数受限，但一如原作中的表现，干员能做的事情非常多。</p><p>
    彩虹小队的干员来自世界各地的特种部队，拥有独立的特殊技能、能力属性与武器装备。《六号：围攻》对此进行了忠实还原，根据原作中的表现，给干员设计符合游戏机制的技能，指定拆除墙面与障碍物的能力强弱，划分从近到远三段射程下不同的武器伤害。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/0a958dac9bf39c0dd87c6e066ba2981f.gif_mw680" width="680" height="370" class alt="被破坏后可出入的墙面  用特定部件表示" title="被破坏后可出入的墙面  用特定部件表示" src="https://alioss.yystv.cn/doc/8073/0a958dac9bf39c0dd87c6e066ba2981f.gif_mw680" referrerpolicy="no-referrer"><span class="picinfo">被破坏后可出入的墙面  用特定部件表示</span></p><p>
    干员的武器伤害标记在干员卡牌的右上角。开火射击时，射程按照敌我双方距离的格子数进行判断，而实际造成的伤害由掷骰子决定。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/cc800d3b30177a969e13d183ea810423.gif_mw680" width="320" height="174" src="https://alioss.yystv.cn/doc/8073/cc800d3b30177a969e13d183ea810423.gif_mw680" referrerpolicy="no-referrer"></p><p>
    伤害以黄色、橙色、红色这三种伤害骰子作为单位，骰子每一面可能有0-3个伤害标记，打不打得中完全看脸。一般来说，骰子数量越多，射击命中率越高；骰子颜色越深，造成的潜在伤害越严重。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/28e1d462b0436d275bdc9c2b21b79505.png_mw680water" width="695" height="367" class alt="Glaz使用狙击步枪  中远距离伤害可观，近战效率低下" title="Glaz使用狙击步枪  中远距离伤害可观，近战效率低下" src="https://alioss.yystv.cn/doc/8073/28e1d462b0436d275bdc9c2b21b79505.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">Glaz使用狙击步枪  中远距离伤害可观，近战效率低下</span></p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/25318b3e072fa88ab4c4b9395bb8ab2d.png_mw680water" width="676" height="374" class alt="持有盾牌的Blitz以手枪作为主武器  命中率很高，威力不足" title="持有盾牌的Blitz以手枪作为主武器  命中率很高，威力不足" src="https://alioss.yystv.cn/doc/8073/25318b3e072fa88ab4c4b9395bb8ab2d.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">持有盾牌的Blitz以手枪作为主武器  命中率很高，威力不足</span></p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/7368484b3c398b291a05d7086eefd93d.png_mw680water" width="699" height="377" class alt="携带大口径机枪的Tachanka  全距离都有输出，命中率堪忧" title="携带大口径机枪的Tachanka  全距离都有输出，命中率堪忧" src="https://alioss.yystv.cn/doc/8073/7368484b3c398b291a05d7086eefd93d.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">携带大口径机枪的Tachanka  全距离都有输出，命中率堪忧</span></p><p>
    伤害标记会累积到干员卡牌上，一旦某个干员拥有的伤害标记，大于等于干员卡牌标明的护甲值，就算是被消灭了。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/d78092979ed0f57e227159fa56f1eced.gif_mw680" width="320" height="174" src="https://alioss.yystv.cn/doc/8073/d78092979ed0f57e227159fa56f1eced.gif_mw680" referrerpolicy="no-referrer"></p><p>
    另外，有掩体或者进攻方干员的防暴盾存在时，被击中的干员将获得一定的伤害减免。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/e09fb1b95776bb86cdb4049b61beaff7.png_mw680water" width="699" height="373" class alt="Montagne牺牲射击能力架好盾牌  能为自己与相邻格的队友减免3点伤害" title="Montagne牺牲射击能力架好盾牌  能为自己与相邻格的队友减免3点伤害" src="https://alioss.yystv.cn/doc/8073/e09fb1b95776bb86cdb4049b61beaff7.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">Montagne牺牲射击能力架好盾牌  能为自己与相邻格的队友减免3点伤害</span></p><p>
    干员不仅能在自己的行动轮里主动开火，还可以像XCOM之类的战棋游戏那样，使用“监视”（Overwatch）策略进行架枪。假如敌方干员行动时进入己方的监视范围，己方干员可立刻进行一次射击。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/03dae48fe0ec0db83b4c57845ff4d4fd.png_mw680water" width="406" height="439" src="https://alioss.yystv.cn/doc/8073/03dae48fe0ec0db83b4c57845ff4d4fd.png_mw680water" referrerpolicy="no-referrer"></p><p>
    《六号：桌游》中的很多操作都涉及到“视线”这一核心机制。墙面会阻挡视线，而开火射击一类的行动，首先就强制要求敌人在“视线”之内；计算伤害时，也要考虑视线中的障碍物带来的伤害减免。</p><p>
    出于公平起见，制作组在每盒桌游里放了一把视线尺。如果对视线的判断存在异议，玩家可以提出质疑，使用视线尺重新测定。质疑成功的话，玩家可以选择增加自己30秒的行动轮倒计时，或者减少对手的30秒倒计时；如果质疑失败，这个权利就要交给对手了。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/0178edfe16a56f9d5cff7031653b425e.jpeg_mw680water" width="807" height="420" src="http://www.yystv.cn/p/undefined" referrerpolicy="no-referrer"></p><p>
    桌游媒介相较电子游戏能力有限，不可避免地存在没法还原原作的地方，制作组也做了一定的取舍。</p><p>
    比如说，原作中双方信息的不透明性，在桌游中无法体现，因为所有干员的位置都在桌子上清晰可见。</p><p>
    为此，制作组引入了一定的暗棋机制。进攻方开局便处于暴露状态，而防守方的每一位干员都拥有两个隐藏干员标记，一个真实标记代表干员的实际位置，另一个是虚假的诱饵，只有防守方玩家知道其中的虚实。</p><p>
    防守方可以把一位干员的移动力自由分配给两个标记。当隐藏干员标记暴露在进攻方视线内，或者进行射击、使用装备等必须暴露身份的行动时，诱饵标记将移出游戏，真实标记才会现形。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/152fb47c3dcf924c0a501b81168bc03e.gif_mw680" width="320" height="181" class alt="移除隐藏干员标记  将干员模型放置到真实的位置上" title="移除隐藏干员标记  将干员模型放置到真实的位置上" src="https://alioss.yystv.cn/doc/8073/152fb47c3dcf924c0a501b81168bc03e.gif_mw680" referrerpolicy="no-referrer"><span class="picinfo">移除隐藏干员标记  将干员模型放置到真实的位置上</span></p><p>
    桌游一般在二维平面进行，想要做出原作里从不同楼层垂直进攻的效果，可谓难上加难。制作组不得已进行了简化，在主楼层外设置了两个独立的上层区域，进攻方防守方各一个。</p><p>
    干员可以从指定的箭头区域上楼，对主楼层的一整个房间进行“监视”架枪的操作；如果同处于上层的对手已经在这个房间架好了枪，双方就要展开交火了。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/aeff2d6ff94909985268d6425aee3bde.png_mw680water" width="800" height="705" class alt="蓝色箭头只有进攻方能够使用  对应原作中进攻方挂绳上楼的设定" title="蓝色箭头只有进攻方能够使用  对应原作中进攻方挂绳上楼的设定" src="https://alioss.yystv.cn/doc/8073/aeff2d6ff94909985268d6425aee3bde.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">蓝色箭头只有进攻方能够使用  对应原作中进攻方挂绳上楼的设定</span></p><p>
    另外，在拯救人质的模式里，人质不阻挡视线，不算作掩体，且任何手段都伤害不到人质，“俄式反恐”彻底没戏，未免有些遗憾。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/7d898c191ce5eab1d6dd7ab419f19d51.png_mw680water" width="416" height="139" class alt="出自官方规则书" title="出自官方规则书" src="https://alioss.yystv.cn/doc/8073/7d898c191ce5eab1d6dd7ab419f19d51.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">出自官方规则书</span></p><p>
    《六号：围攻》没有完整还原原作，而是在用另一种形式重现攻坚体验。它的计算量比较繁复，硬核程度堪比原作，价格则是超越原作——</p><p>
    同时收录所有干员及地图扩展包的众筹档位，需要199美元（约合人民币1285元），以今年Steam夏促的价格计算，足以在国区购买5份终极版本的原作，美区则是3份。</p><p>
</p><p class="picbox">
    <img data-original="https://alioss.yystv.cn/doc/8073/9f1e82c0360b5c6a0a3bc8222bb05dab.png_mw680water" width="889" height="405" src="https://alioss.yystv.cn/doc/8073/9f1e82c0360b5c6a0a3bc8222bb05dab.png_mw680water" referrerpolicy="no-referrer"></p><p>
    然而这样的价格对于桌游来说算是中规中矩，从众筹结果便能看出来，能够接受这一价格，愿意在物质与精神上予以支持的玩家起码有上万人。如果《六号：围攻》找到合适的中文代理，在国内的价格应该会便宜一些，哪怕找不到一起玩的朋友，买来收藏似乎也不错。</p>        <br>
