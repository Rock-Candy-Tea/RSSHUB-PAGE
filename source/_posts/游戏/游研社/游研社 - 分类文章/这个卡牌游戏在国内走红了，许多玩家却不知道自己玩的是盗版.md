<blockquote class="doc-blockquote"><p>当原作者还在担心自己“用一个劣质游戏糟蹋了绝妙的想法”时，模仿者却早已出手。</p></blockquote><p>《堆叠大陆》（Stacklands）是今年4月9日登录Steam平台的一款轻量级卡牌游戏，它的画风可爱、机制也颇有意思，玩起来有点《蜘蛛纸牌》和《密教模拟器》的影子。</p><p>游戏的制作组Sokpop虽然仅有四人，但向来以快速、精品、独具巧思的小游戏产出而闻名，《堆叠大陆》就是很明显的例子之一：</p><p>游戏里的一切都是以卡片的形式所呈现，玩家需要把握好食物与材料的产出，一步步壮大自己的村庄。而所谓“堆叠”，意思就是把作为生产力的村民卡放在灌木卡上产生浆果、岩石产出石头、树木产出木材。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/e6ba78ace35575361d495ad16324eb95.gif_mw680" class alt="人类文明的开始只是一个卡包" title="人类文明的开始只是一个卡包" width="500" height="280" src="https://alioss.yystv.cn/doc/9516/e6ba78ace35575361d495ad16324eb95.gif_mw680" referrerpolicy="no-referrer"><span class="picinfo">人类文明的开始只是一个卡包</span></p><p>虽然体量小，但游戏的完成质量可以说相当精湛，非常适合在一个闲暇的下午用来好好放松心情，Steam上的玩家们也给出了好评如潮的评价。</p><p>只是Sokpop的“小毛病”同样也体现在《堆叠大陆》上，开发者们相当热衷于不断开发新的小游戏，已经上架的游戏基本不会再更新多的内容与DLC。在玩家们的催促下，《堆叠大陆》才在7月初更新了一个海岛的拓展地图。</p><p>而这短短的三个月之间，《堆叠大陆》的盗版已经起码更新完了五个章节，大陆、海洋、外太空、仙侠、魔幻，比正版还勤快。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/23b3ade93affcb023a5f4ea9f361c922.png_mw680water" width="610" height="210" src="https://alioss.yystv.cn/doc/9516/23b3ade93affcb023a5f4ea9f361c922.png_mw680water" referrerpolicy="no-referrer"></p><p>而这导致了一种尴尬的局面，大部分玩过《堆叠大陆》的玩家压根不知道自己玩的是盗版。</p><h3><span><span class="r-triangle">1</span></span></h3><p powered-by="xiumi.us"><br><span>《堆叠大陆》的开发流程很简单，主要设计师Aran在一条视频中提到，游戏完全诞生于他某天想制作一款卡片版乡村模拟器的想法，他要把所有东西都化简为卡片，再以此模拟出世界的发展路线。</span></p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/bf49818d9da364acdc2a32de6bf43116.png_mw680water" class alt="设计简图" title="设计简图" width="1080" height="606" src="https://alioss.yystv.cn/doc/9516/bf49818d9da364acdc2a32de6bf43116.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">设计简图</span></p><p>于是他立即做出了一个简单的游戏原型，这个小demo得到了同事们的一致认可，也得到了一点宝贵的意见：添加一个金钱系统，供玩家出售多余卡牌和购买村庄进阶时需要的资源包。</p><p>玩家需要借由初始卡包里的资源发展，不断出售多余物资，获得的金钱用来购买高级卡包“开包”，以此解锁新的建筑与材料。解锁新卡包时的成就感还是蛮足的，卡包内隐藏的野兽与敌对生物也给游戏增添了不少乐趣。</p><p>例如20金币包中有一张说明写着“它看上去很凶残”的野狼卡，被玩家开出来之后，它就会开始攻击附近的村民；然而只要拿另一张没什么用处的骨头卡叠放到狼上，它就会变成一只可以参加大生产的狗。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/467427f1ef29b6f012b2622306fba0dc.png_mw680water" class alt="只是看上去很凶残" title="只是看上去很凶残" width="659" height="468" src="https://alioss.yystv.cn/doc/9516/467427f1ef29b6f012b2622306fba0dc.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">只是看上去很凶残</span></p><p>这些有点意思的小细节在游戏中还挺多的，足以满足玩家对“堆叠”效果的期待。因此接下来Aran花了一段时间完善了卡牌系统，找一名美工朋友制作好美术就把游戏上架了。</p><p>《堆叠大陆》的整个开发过程仅用了几个月，开发方便也挺好玩，对于Sokpop来说这就是相当成功的一款作品了——只是这些优点，对于盗版厂商来说同样也是一种便利。</p><p>在游戏发行后的两个月，国内短视频平台上就出现了成堆的“堆叠大陆”小游戏。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/34598db79f0e0d5b603ebcf837430f93.jpeg_mw680water" width="1080" height="331" src="http://www.yystv.cn/p/undefined" referrerpolicy="no-referrer"></p><p>以Sokpop的人力来说，这肯定不是官方自己的账号与程序，游戏内的界面与卡片也只是看起来相似。比较有可能的原因是其它厂商解包了原始文件，然后使用相同的素材把游戏重新写了一遍上架而已。</p><p>但就是这么一款制作粗劣的小程序，以它为主的视频在平台上却斩获了上百万的播放与点赞，随着后期更多内容的更新，千万的播放合集也不在少数。</p><p>游戏本身好玩是其一，其二是不知道有意为之还是程序员能力不足，这个盗版的《堆叠大陆》有着一堆原版都没有的奇怪Bug。</p><p>在正常的游戏流程中，养育村民是一件很麻烦的事，要增加人口，就必须先建造房子，把两张村民卡拖进房子里，经过一段时间的等候后才会生成婴儿，而婴儿又得等待至少一回合的时间才能正式成为具有生产力的成年人。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/64f167d32d42e592270a375697e729e9.gif_mw680" class alt="正常情况" title="正常情况" width="600" height="351" src="https://alioss.yystv.cn/doc/9516/64f167d32d42e592270a375697e729e9.gif_mw680" referrerpolicy="no-referrer"><span class="picinfo">正常情况</span></p><p>这期间婴儿也会消耗食物，一旦食物供给不足，村民们就会因饥饿而死亡，死后的村民则会变成无法出售、劳作、生育，还会占用一格卡片库存的尸体卡，除了合成墓地之外再没有别的用处。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/c5aada8a4f0ac40c60900082604e0b9d.jpeg_mw680water" class alt="一般来说都是玩家们竭力避免的负面卡片" title="一般来说都是玩家们竭力避免的负面卡片" width="279" height="347" src="http://www.yystv.cn/p/undefined" referrerpolicy="no-referrer"><span class="picinfo">一般来说都是玩家们竭力避免的负面卡片</span></p><p>但是在盗版的《堆叠大陆》中，尸体似乎继承了生者的意志，虽然没法再参与生产，却依旧可以战斗，这让它们成了游戏内唯一无敌的存在——一个没有血条的亡灵战士。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/aa3076c29599662dc0ac37a6951ca1d8.gif_mw680" class alt="赶尸人看了都直呼内行" title="赶尸人看了都直呼内行" width="451" height="207" src="https://alioss.yystv.cn/doc/9516/aa3076c29599662dc0ac37a6951ca1d8.gif_mw680" referrerpolicy="no-referrer"><span class="picinfo">赶尸人看了都直呼内行</span></p><p>这一特性让尸体的价值大幅上升，再加上另一个更加离谱的Bug，盗版的《堆叠大陆》以此衍生出了一套专门养尸的“尸体流”。</p><p>整套操作是这样的：游戏前期要尽快建好一座房子，然后不断购买卡包以开出第二名村民。</p><p>在得到第二名村民后让两人进屋生娃，接着刻意减少食物供给，或者利用刷新的怪物送死，达到两名村民死亡，婴儿还活着的局面——在正版游戏中，成年人全部成为尸体时游戏就结束了，但是在盗版中游戏依然会继续。</p><p>这里就是整个流派的重点，尸体合成的墓地卡是可以作为五金币的价格出售的，理论上只要有大量的尸体，就足以完成一套比常规生产线路效率还高的“尸体-金币”生产线。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/a91999cf73db5419edc7a8d9e1b1535b.gif_mw680" class alt="“分裂生殖”" title="“分裂生殖”" width="799" height="435" src="https://alioss.yystv.cn/doc/9516/a91999cf73db5419edc7a8d9e1b1535b.gif_mw680" referrerpolicy="no-referrer"><span class="picinfo">“分裂生殖”</span></p><p>盗版《堆叠大陆》账号的运营者显然也是这么想的，在他们发布的第一条视频中，就以此为卖点，向大家隆重介绍了这个游戏的独特玩法。只是视频里的游戏似乎还在开发初期阶段，他们利用的是村民生孩子会一口气生11个的Bug（现已修复），再把所有婴儿饿死达到积累原始资本的目的，也就是说——所有尸体都是由婴儿产生的。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/2776257de5f283750003bb75a3438e7c.jpeg_mw680water" class alt="这不人道，我不认可" title="这不人道，我不认可" width="914" height="698" src="http://www.yystv.cn/p/undefined" referrerpolicy="no-referrer"><span class="picinfo">这不人道，我不认可</span></p><h3><span><span class="r-triangle">2</span></span></h3><p powered-by="xiumi.us"><br><span>荒诞、离奇，再加上原版的新奇玩法，盗版《堆叠大陆》很快催生出了一大批制作游戏视频的主播，而视频的风行又带来了大量的玩家。虽然有人在评论区指出了盗版的问题，但依然有不明就里的部分玩家会在遇到正版游戏时反而产生疑惑。</span></p><p>B站UP主洛黑城是最早制作正版《堆叠大陆》视频攻略的人之一，他在4月19日就制作了相关的攻略视频，但令他疑惑不解的事就此发生了——许多观众在评论里询问“为什么我的尸体不能打怪了？”</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/70375482bdd13cbbaa8be218dc905ab4.jpeg_mw680water" class alt="“我和UP玩的不是同一个游戏”" title="“我和UP玩的不是同一个游戏”" width="1023" height="674" src="http://www.yystv.cn/p/undefined" referrerpolicy="no-referrer"><span class="picinfo">“我和UP玩的不是同一个游戏”</span></p><p>火爆的现象让盗版厂商嗅到了可乘之机，于是他们接二连三地推陈出新。在盗版《堆叠大陆》中，第二章的名字叫“飞向宇宙”，他们决定让玩家与感染性极强的丧尸病毒作战，要在紧迫的时间中制造出火箭飞往太空。</p><p>第三章“永生遐想”，他们添加了排行榜和复制人设定，村民每过一天就会克隆一次，在越来越多需要喂养的村民中，玩家要尽可能保持食物的充足，在更短的天数内创造更多的人口。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/e4dfe89e771fb80d201b63271ad0dbd2.jpeg_mw680water" width="1021" height="471" src="http://www.yystv.cn/p/undefined" referrerpolicy="no-referrer"></p><p>第四章“星辰大海”取消了游戏原先的“工作桌”设计，取而代之的是一块广阔的海洋地图，玩家要扮演一个海上的求生者，依靠漂流而来的物资活下去。</p><p>第五章“无尽黑暗”则延续着第四章的设定，玩家扮演求生者，在白天尽快建设，夜晚防御攻击欲望更高的敌人。</p><p>制作了这么多玩法之后，事到如今人们也很难称其为“盗版”，或许称之为“模仿者”更为合适。虽然内容粗糙，但大部分题材确实是原版本身并没有制作出来的，尤其是还有开发者主动把游戏进行了本土化，制作了由此衍生的另一款游戏《堆叠修仙》。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/d92465576436b20feb80248345872030.jpeg_mw680water" width="1068" height="493" src="http://www.yystv.cn/p/undefined" referrerpolicy="no-referrer"></p><p>游戏的框架与《堆叠大陆》大体相似，内容却完全改头换面了一番，简单来说，这是个把网络仙侠小说都改进去了的游戏。</p><p>玩家要炼丹渡劫，一级一级升到大罗金仙，所有资源都被替换成了仙侠小说里常见的天材地宝。在《堆叠修仙》的第二章甚至依次添加了剧情要素，玩家开局就会被一个神秘人推下悬崖，接下来就是常见的练级套路。</p><p>游戏还有颇为另类的云游模式，虽然实际只是一个偷菜的“仙家农场”。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/ede233a53fc43e5f28ffd93ae5428113.jpeg_mw680water" class alt="怎么你们仙人也偷菜啊？" title="怎么你们仙人也偷菜啊？" width="1080" height="499" src="http://www.yystv.cn/p/undefined" referrerpolicy="no-referrer"><span class="picinfo">怎么你们仙人也偷菜啊？</span></p><p>光从内容上很难评价这究竟是好事还是坏事，毕竟众多的模仿本身证明了《堆叠大陆》的可塑性。从法律层面上来讲，单纯的游戏玩法也并没有版权的说法，就像《Flappy Bird》和《别踩白块》流行时的状况一样，借此推动一类游戏的风靡并不是不可行。</p><p>它确实风靡了一阵子，不仅是短视频平台的小游戏，iOS、谷歌，各大应用商店都出现了模仿者的身影，在TapTap与4399上你也依然可以看见许多“堆叠”而成的游戏，甚至部分模仿者的评价和下载数都还不错。</p><p>但糟糕的地方在于，这些《堆叠大陆》的“模仿者”通常来说都很难提供正常的游戏体验，无处不在的广告就足以证明一切。</p><p>玩家如果在卡包中开出了不想要的物资，看广告就可以刷新，提前解锁建筑、复活村民都可以通过万能的广告大法解决。我在试玩Tap Tap上的《堆叠修仙记》时，刚打开游戏弹出的就是广告页面，有时甚至变本加厉地增加广告次数。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/a310b20d7dc295cd40b85fe0ebed3df2.png_mw680water" class alt="我全都要=看两次广告" title="我全都要=看两次广告" width="1080" height="506" src="https://alioss.yystv.cn/doc/9516/a310b20d7dc295cd40b85fe0ebed3df2.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">我全都要=看两次广告</span></p><p>为了尽可能强迫玩家们看广告，小程序版的《堆叠大陆》还特意提高了游戏难度。在正版游戏中怪物的刷新机制为“12+6”——第一次刷怪在12回合，之后每6回合刷新一次，但小程序版的《堆叠大陆》将怪物的刷新时间、频率大幅提升了，而这一切，只是为了多争取一点广告的播放时间而已。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/50597736a997612917c668640318ee8e.jpeg_mw680water" class alt="一位民兵对阵两熊三巨鼠的地狱绘图" title="一位民兵对阵两熊三巨鼠的地狱绘图" width="626" height="469" src="http://www.yystv.cn/p/undefined" referrerpolicy="no-referrer"><span class="picinfo">一位民兵对阵两熊三巨鼠的地狱绘图</span></p><h3><span><span class="r-triangle">3</span></span></h3><p powered-by="xiumi.us"><br><span>众多模仿者的出现自然被原开发者Aran注意到了，玩家也在Discord频道中询问是不是能有什么办法让“赝品”们下架。</span></p><p>Aran只是以一种相当无奈的语气回复：“我们也知道这些问题，但是真的很难办到，哈哈。”他也解释说，App Store并没有一个可以反馈信息的途径，唯一的办法就是给苹果客服发邮件投诉，只是苹果客服在收到邮件后会把这封邮件转发给……赝品的开发商。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/dde92284df01f228717c2d0ec710173a.png_mw680water" class alt="伤敌一千自损八百" title="伤敌一千自损八百" width="1110" height="220" src="https://alioss.yystv.cn/doc/9516/dde92284df01f228717c2d0ec710173a.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">伤敌一千自损八百</span></p><p>与这些模仿者相比，Sokpop本身就显得良心无比。在游戏上线Steam后，Aran开始担心自己仓促上架的游戏无法得到认可，因为游戏内还有一堆平衡性工作要做，于是他们很少见地加紧对游戏进行了一系列补丁更新。</p><p>在添加了和平模式与难度选项后，他们又发现《堆叠大陆》卖得实在太好了，还有许多玩家留言需要更多新内容。</p><p>于是在7月11日更新的海岛一章中，类似“野狼-狗”的有趣设计多了起来，一些玩家呼声很高的便捷操作也都被添加进了游戏。拖动卡片时时间暂停、卡片网格自动对齐、新建筑“温室”自动种植农作物，无论哪一个都是奔着提升玩家游戏体验的目标去的。</p><p class="picbox"><img data-original="https://alioss.yystv.cn/doc/9516/33a60bdbac3772d412418ef4e7583ec1.gif_mw680" class alt="强迫症患者狂喜" title="强迫症患者狂喜" width="799" height="481"
