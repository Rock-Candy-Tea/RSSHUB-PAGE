<blockquote class="doc-blockquote">    <p>好文捞一手。原文发布于2020年2月，近日《风色幻想》系列宣布将于8月9日打包登陆Steam，新作《风色幻想SP~封神之刻~ Remake》也将于年内发售，趁此机会带大家回顾系列中不错的入门级作品《风色幻想SP~封神之刻》。</p></blockquote><p align="left">    <span style="font-size:13px;">        <strong>            <span style="font-size:13px;color:#000000;"></span></strong></span>千禧年初的国产游戏市场是“黄金时代”。《天之痕》《秦殇》《剑侠情缘2》、《新绝代双骄2》《幽城幻剑录》等不少人心中的经典作品竞相问世。那个年代玩家是幸福的，他们的可选择面很广，但整体市场的大趋势是以“武侠游戏”为主，因此江湖儿女情长成就一段段佳话。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/27197df0d3713b0ae471f303e8360016.png_mw680water" title="再经典的味道，吃多了也腻 " alt="再经典的味道，吃多了也腻 " class width="602" height="452" align src="https://alioss.yystv.cn/doc/6204/27197df0d3713b0ae471f303e8360016.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">再经典的味道，吃多了也腻 </span></p><p>    不少厂商也在“主流武侠”之外抢占市场。这其中就不得不提弘煜科技的《风色幻想》系列。尽管《风色幻想》初代流程颇短，而且以今天的眼光来看也存在很多难以忍受的不足，但是它还是打出了名气，受到玩家欢迎。</p><p>    而《风色幻想 魔导圣战》的攻略集结尾附送的插图，也暗示着续作的存在。</p><p>    <img data-original="https://alioss.yystv.cn/doc/6204/e5c7b4f35fd2553965df35b598f86254.png" title="那时候还叫《风色幻想2nd》——宣传和成品有点“不一致”也是系列的“传统艺能”了 " alt="那时候还叫《风色幻想2nd》——宣传和成品有点“不一致”也是系列的“传统艺能”了 " class data-original="https://alioss.yystv.cn/doc/6204/c4eb0b392b00ffe46b39e6302771357c.png_mw680water" title="《风色幻想SP~封神之刻》 " alt="《风色幻想SP~封神之刻》 " class width="395" height="589" align src="https://alioss.yystv.cn/doc/6204/c4eb0b392b00ffe46b39e6302771357c.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">《风色幻想SP~封神之刻》 </span></p><h3 style="text-align:center;">    <strong>        <span class="r-triangle">            可能是跨界跨得最广的国产游戏        <br><br></span>    </strong></h3><p>    对于很多想要了解《风色幻想》系列的新玩家来说，《风色幻想SP~封神之刻》是一部不错的入门级作品。因为相比于其他作品，它是独立成章的一代，除了故事风格类似，基本上跟其他作品没啥关联。</p><p>    这是一款特别的作品，充满着各种意想不到的“神展开”。</p><p>    我们从女主角依莉丝的身上就可见一斑。</p><p>    一般来说，很多游戏开局时，游戏主角人物都处于一种比较弱小的情况。即使是天赋异禀，也要慢慢在身边人的帮助下成长，变强。但是《风色幻想SP》走了一条不同寻常的路，上来就是内斗，依莉丝的老师连同一群魔兵就这么围着她打了起来。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/06ac90ea44ae7860c8c6d028c9c8728a.png_mw680water" title="左右上下都为男 " alt="左右上下都为男 " class width="748" height="558" align src="https://alioss.yystv.cn/doc/6204/06ac90ea44ae7860c8c6d028c9c8728a.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">左右上下都为男 </span></p><p>    不过，依莉丝好歹是新生魔王，身负全族重任，带领大家走向更好的未来，打败这些小喽啰自然不在话下，最后也没动手杀了自己的老师。</p><p>    而更为神奇的是，依莉丝实际上是游戏中的最终boss反派“西鲁菲”的一个表现，她的真实身份是曾经的裘卡。而她的好朋友“裘卡”也是未来的“ 西鲁菲”的一滴眼泪变成，目的是回到过去杀死曾经的依莉丝。只是由于彼此之间的相处，最终她成为了依莉丝的好姐妹。</p><p>    换句话说，光是女主角一人便以过去、现在、未来三重身份出现在同一个世界里面。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/122e839aaac18db2df0982e1530f2946.png_mw680water" width="802" height="545" src="https://alioss.yystv.cn/doc/6204/122e839aaac18db2df0982e1530f2946.png_mw680water" referrerpolicy="no-referrer"></p><p>    <strong>        你以为这个是穿越剧？    </strong></p><p>    <strong>        不是的。    </strong></p><p>    依莉丝的原形是“西鲁菲”。她是原始之海的神，所有的生命都从那里衍生出来，可以说她是整个游戏世界的造物主，这非常类似《新世纪福音战士》中的莉莉丝。巧合的是，依莉丝的母亲名字就是“光之圣女莉莉丝”。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/7d2ebdc5841487190df9cfbe85353194.png_mw680water" title="万物交融的原始之海，有“人类补完计划”那味了 " alt="万物交融的原始之海，有“人类补完计划”那味了 " class width="743" height="555" align src="https://alioss.yystv.cn/doc/6204/7d2ebdc5841487190df9cfbe85353194.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">万物交融的原始之海，有“人类补完计划”那味了 </span></p><p>    而游戏后面反复提及的“同化”概念，即所有人回归原始之海，无分彼此，意识共享，也很像大家熟知的“人类补完计划”（橙汁！），而各种关于“容器”、“怪兽冲击”、等剧情，也让人不得不想起“使徒来袭”。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/3a8e3127ee5ac92da0a905bb1079da30.png_mw680water" title="一句有点“糟糕”的台词 " alt="一句有点“糟糕”的台词 " class width="748" height="557" align src="https://alioss.yystv.cn/doc/6204/3a8e3127ee5ac92da0a905bb1079da30.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">一句有点“糟糕”的台词 </span></p><p>    <strong>        你以为这个是致敬《EVA》的游戏吗？    </strong></p><p>    <strong>        也不尽然。    </strong></p><p>    不少玩家初次看见依莉丝和裘卡居然是被男主角修召唤到人类世界时，心里一般都默认这是个奇幻游戏了，已经想好怎么用魔法来大杀四方，也想着等等敌人会有什么酷炫招式。然后，他们都会看见敌人的小兵，荷枪实弹走了进来，一顿机枪加激光枪扫射。</p><p>    “魔法少女们，时代变了！”</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/75527affb2c5ee5400ebe950f7e34c22.png_mw680water" title="这玩意比魔法好用多了 " alt="这玩意比魔法好用多了 " class width="775" height="362" align src="https://alioss.yystv.cn/doc/6204/75527affb2c5ee5400ebe950f7e34c22.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">这玩意比魔法好用多了 </span></p><p>    在接下来跟随着修去反抗他昔日的好兄弟——新月帝国皇帝菲里德的过程中，还会看见很多匪夷所思的高科技产物，比如说堪比“星际战舰”的芙蕾雅号，在上面战斗时不仅要把兵打掉，还要干掉船舱边的炮塔。连传说中的“西鲁菲”也是被“科技”塞进了容器里面。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/a7c131998034fbdb1d3cf69a367cb469.png_mw680water" title="新月帝国的“芙蕾 雅”号 " alt="新月帝国的“芙蕾 雅”号 " class width="775" height="579" align src="https://alioss.yystv.cn/doc/6204/a7c131998034fbdb1d3cf69a367cb469.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">新月帝国的“芙蕾 雅”号 </span> <span style="line-height:1.8;color:#B2B2B2;font-size:13px;"></span></p><p>    我们甚至可以在里面看见关于“人造人”的故事。玩家队伍中有一位名叫“麒麟”的神秘女子，她的来历非常奇怪，一手“裁之刃”玩得是出神入化。她特别喜欢搞爆破，看见天界之门以后，就沉迷在用炸药炸掉它。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/64691c58063a7510a3829d0e41c8f31d.png_mw680water" title="麒麟讲述木偶的故事 " alt="麒麟讲述木偶的故事 " class width="750" height="552" align src="https://alioss.yystv.cn/doc/6204/64691c58063a7510a3829d0e41c8f31d.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">麒麟讲述木偶的故事 </span></p><p>    但实际上，她是由她最敬重的“爷爷”迪耶利用技术所造出来的躯体，用以承载自己的灵魂。本该是没有灵魂、没有思考的“机器人”，却自己动了起来，甚至一度想要逃离，却又一度被迪耶破坏掉。但过后迪耶还是将她视若己出，洗去了她的记忆，养育起来，即使她有了自主意识，即使她有可能失控。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/7acbb5b5ffa2e9db2baf65f791a72f82.png_mw680water" title="迪耶的“缸中之脑” " alt="迪耶的“缸中之脑” " class width="746" height="555" align src="https://alioss.yystv.cn/doc/6204/7acbb5b5ffa2e9db2baf65f791a72f82.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">迪耶的“缸中之脑” </span></p><p>    其实游戏中的大部分高科技都来自于迪耶，他进入天界之门后找到了许多技术，用以改造光之一族居住的人类世界。而这个“门后的世界”，跟现实很像。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/bc7eae6dda5f681edd3c198da1b64936.png_mw680water" title="天界之门后的背景 " alt="天界之门后的背景 " class width="746" height="561" align src="https://alioss.yystv.cn/doc/6204/bc7eae6dda5f681edd3c198da1b64936.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">天界之门后的背景 </span></p><p>    <strong>        所以最后这是一个科幻故事吗？    </strong></p><p>    <strong>        也不好下定论。    </strong></p><p>    很多玩家第一次游玩《风色幻想SP》时，都会被里面眼花缭乱的设定震撼到，奇幻、科幻、宿命、魔法、技术等诸多元素混合在一起，一层又一层带出的世界观，令人充满遐想，而编剧恶魔猫在当年写作这个故事时还在读书。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/f7d7e7cf60b17b7f7e6bae6434d05f39.png_mw680water" title="这段对话也很“引人遐想” " alt="这段对话也很“引人遐想” " class width="743" height="550" align src="https://alioss.yystv.cn/doc/6204/f7d7e7cf60b17b7f7e6bae6434d05f39.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">这段对话也很“引人遐想” </span></p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/790c684fc6007d3bf5e6bd6f7adc69b0.png_mw680water" title="好兄弟，你这朋友我交定了 " alt="好兄弟，你这朋友我交定了 " class width="742" height="553" align src="https://alioss.yystv.cn/doc/6204/790c684fc6007d3bf5e6bd6f7adc69b0.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">好兄弟，你这朋友我交定了 </span></p><p>    多种元素的结合让游戏充满着想象空间，鼓励着玩家思考着的游戏的更多可能性。</p><p>    而“可能性”的触发需要“尝试”。</p><h3 style="text-align:center;">    <strong>        <span class="r-triangle">            强调“尝试”的游戏        <br><br></span>    </strong></h3><p>    《风色幻想SP》沿用了一代的“魔兽系统”——依莉丝可以通过技能抓取战斗中的怪物，然后在城镇的魔兽训练房中对其进行训练，调整和养殖，还可以将两个魔兽进行合成，变成更强的魔兽。你可以通过“尝试”不同魔兽的搭配，发展壮大你的队伍。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/5215ee3ab35d25323d570a6dc5f56afc.png_mw680water" title="魔兽妙妙屋 " alt="魔兽妙妙屋 " class width="642" height="482" align src="https://alioss.yystv.cn/doc/6204/5215ee3ab35d25323d570a6dc5f56afc.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">魔兽妙妙屋 </span></p><p>    这里也诞生了一种特别有意思的玩法。由于在《风色幻想SP》中，怪物等级是随着主角等级的提高而提高，而每个角色的经验值在战场上都是单独计算——使用技能或者击杀敌人才有经验。所以，完全可以让主角站在一边观战，而让魔兽们下场打架。这样就既不会遭遇强力怪物，又能够打败低等级敌人轻松赢得战斗。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/ba29dec44ca0010ae40b381c65c870d0.png_mw680water" title="你也可以让魔兽们去恰饭（米亚也可以是魔兽） " alt="你也可以让魔兽们去恰饭（米亚也可以是魔兽） " class width="642" height="482" align src="https://alioss.yystv.cn/doc/6204/ba29dec44ca0010ae40b381c65c870d0.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">你也可以让魔兽们去恰饭（米亚也可以是魔兽） </span></p><p>    游戏中的大部分情报和支线任务触发，都来源于一个很简单的操作：去广场听八卦。听多了，就自然开启不同的任务，所以每到一个新地方，第一时间就是到广场听人聊天。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/9af596d8ce9e54b2598fd550194a7bdf.png_mw680water" title="可爱的猫猫支线开启！ " alt="可爱的猫猫支线开启！ " class width="744" height="533" align src="https://alioss.yystv.cn/doc/6204/9af596d8ce9e54b2598fd550194a7bdf.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">可爱的猫猫支线开启！ </span> <span style="line-height:1.8;color:#B2B2B2;font-size:13px;"></span></p><p>    机缘巧合之下， 你还会到达游戏中的“圣地”——FY城。这里不仅售卖很多宝贵道具，如让你属性猛涨的“《风色幻想SP》攻略册”，还能听见制作人员们的吐槽。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/d25f44b8d3e856932bc3b9eb9672de21.png_mw680water" width="640" height="480" src="https://alioss.yystv.cn/doc/6204/d25f44b8d3e856932bc3b9eb9672de21.png_mw680water" referrerpolicy="no-referrer"></p><p>    PS：恶魔猫说出这段话是有隐情的。在《风色幻想SP》的开发过程中，他已经到达法定兵役年龄，需要前往兵营服役。</p><p>    “尝试”甚至能够帮你触达更好的游戏结局。《风色幻想SP》共有四个结局，很多玩家初次通关都进入了“圣贤王”结局，击败西鲁菲后，修成为了新月帝国的皇帝，麒麟陪伴在他身边，但是他终其一生都没有再见到依莉丝。</p><p>    但是若是你进入了另外一条故事线，在你最后跟西鲁特决战的时候，稍微走动一下，走到依莉丝的尸体上面，结局就会发生变化。根据道具“回忆之书”的有无，进入新的不同结局，一个更为美好的未来展开了。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/c0013975d30f69e2935f822696532b98.png_mw680water" title="笔者比较喜欢这个结局 " alt="笔者比较喜欢这个结局 " class width="748" height="560" align src="https://alioss.yystv.cn/doc/6204/c0013975d30f69e2935f822696532b98.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">笔者比较喜欢这个结局 </span> <span style="line-height:1.8;color:#B2B2B2;font-size:13px;"></span></p><p>    而游戏还有一个进入回归“原始之海”结局，在同化后的决战中，若是被西鲁特直接打败，画面背景转到现代社会背景。如果你当年买的是台版正版，会在说明书里面发现一首名为《恋爱》的歌。这首歌就用在这里，它的前奏很长，一些玩家看见结局字幕就按了ESC，结果都没听出这是首人声曲。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/cce2bf64f9cb34c966cc22454b3c300d.png_mw680water" title="这个结局有点梦中梦的感觉 " alt="这个结局有点梦中梦的感觉 " class width="746" height="550" align src="https://alioss.yystv.cn/doc/6204/cce2bf64f9cb34c966cc22454b3c300d.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">这个结局有点梦中梦的感觉 </span></p><h3 style="text-align:center;">    <strong>        <span class="r-triangle">            故事之后还有继续        <br><br></span>    </strong></h3><p>    打通结局并不仅是游戏的结束。当制作人员名单字幕播完以后，你会发现系统居然让你存档，然后等你再度回到游戏标题界面时，你会发现游戏名称居然变了。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/4c7f22b68706548f75ed9edd4d9af8cc.png_mw680water" title="《风色幻想SP 终焉之塔》 " alt="《风色幻想SP 终焉之塔》 " class width="749" height="498" align src="https://alioss.yystv.cn/doc/6204/4c7f22b68706548f75ed9edd4d9af8cc.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">《风色幻想SP 终焉之塔》 </span></p><p>    重新进入游戏以后，大地图的海拉尔村西方会开放一个新场景“终焉之塔”。要进去是需要条件的，先去村庄多讲几次话“八卦”一下，进入竞技场提高自己的称号，得到最高称号后，才能进入塔内，里面会有整整一百关的战斗，甚至可以看见《风色幻想》初代人物和工作室成员登场。</p><p>    这个爬塔的传统也在系列后续作品中得到了保留。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/c9f2460d179e3786bc5a9de3f300137f.png_mw680water" title="通关终焉之塔后可以看见“三巨头”的留言，请支持正版 " alt="通关终焉之塔后可以看见“三巨头”的留言，请支持正版 " class width="750" height="554" align src="https://alioss.yystv.cn/doc/6204/c9f2460d179e3786bc5a9de3f300137f.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">通关终焉之塔后可以看见“三巨头”的留言，请支持正版 </span></p><p>    真可谓是“买一个玩一年”。</p><p>    不仅如此，《风色幻想SP》中还附赠了一个《风色幻想》初代的特别篇《风色幻想Forever~龙的新娘》。据说这部分剧情由成员Miharu负责，因为那时候的恶魔猫服兵役去了，因此剧情上多少跟初代有所出入。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/025d12c2e2ef09eaa8b9cea31ceef22d.png_mw680water" width="747" height="552" src="https://alioss.yystv.cn/doc/6204/025d12c2e2ef09eaa8b9cea31ceef22d.png_mw680water" referrerpolicy="no-referrer"></p><p>    给游戏加上这么多附赠内容，可以看出制作组成员对于游戏的厚望。</p><p>    《风色幻想SP》市后广受好评，一年下来拿奖无数，是《风色幻想》系列真正意义上的“封神之刻”。</p><p>    据弘煜曾经的官网介绍，《风色幻想SP～封神之刻》获得：</p><ul class="list-paddingleft-2">    <li>        <p>            台北电脑工会举办的2001年度GAMESTAR游戏选拔最佳角色战略首奖。        </p>    </li>    <li>        <p>            《游戏世界》2000年8月号、6月号、5月号销售排行榜第一名。        </p>    </li>    <li>        <p>            《软体世界》2000年4月号非常销售排行榜第一名。        </p>    </li>    <li>        <p>            《次世代游戏情报》2000年1月号期待新作排行榜第一名。        </p>    </li></ul><p>    后来还曾经推出韩文版，远销海外。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/ff33316a803b52e70ead9b4e25aeb269.png_mw680water" width="758" height="553" src="https://alioss.yystv.cn/doc/6204/ff33316a803b52e70ead9b4e25aeb269.png_mw680water" referrerpolicy="no-referrer"></p><p>    然而，之后的道路就没这么顺利了。随着《风色幻想SP》的大获成功，“三巨头”开始了自己的宏大策划，既然前作依靠毫无拘束的风格赢得巨大成功，接下来的《风色幻想2》自然是进行了大刀阔斧的改革，乃至于整个游戏都变成了回合制RPG游戏。</p><p>    《风色幻想2:aLIVE(Wind Fantasy 2: aLIVE》并没有获得预想中的成功，一开始反而受到不少玩家的恶评。原因不在本文铺开叙述了，总的来说还是变化太大了，但作为整个“菲利斯多”篇来说，是不可缺少的一环。</p><p>    但从那时开始，《风色幻想SP》进入漫长的冷却期。诚然它是个独立作品，跟成篇的《风色幻想1-4》、《风色幻想5-XX》没得比，但基本上没有单机作品跟它联动，也着实落寞了点。直到多年后在2010年公布消息的《风色幻想online2》中，才使用到了它作为原作背景。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/9791f5203e4c09a85ba6242ae9b2f0b0.png_mw680water" title="《风色幻想online2》 " alt="《风色幻想online2》 " class width="502" height="702" align src="https://alioss.yystv.cn/doc/6204/9791f5203e4c09a85ba6242ae9b2f0b0.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">《风色幻想online2》 </span></p><p>    2018年，弘煜科技曾经表示，Bilibili正与它洽谈将《风色幻想 SP》动画化的可能性，角色和剧情内容都将包含在授权范围内。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/c887856d60a6b2f15273b8c67c6fc0bb.png_mw680water" width="640" height="360" src="https://alioss.yystv.cn/doc/6204/c887856d60a6b2f15273b8c67c6fc0bb.png_mw680water" referrerpolicy="no-referrer"></p><p>    2019年，在ChinaJoy 2019上，弘煜科技表示将推出一款策略战棋手游，将完成呈现原作并彻底强化。游戏现今尚未公布推出时间。</p><p>    大概，他们还有很多项目要忙吧。</p><h3 style="text-align:center;">    <strong>        <span class="r-triangle">            独一无二的作品 </span></strong></h3><p><br>尽管《风色幻想SP》隶属于整个《风色幻想》系列中，但是它对于我来说是独一无二的作品。</p><p class="picbox" align="left">    <img data-original="https://alioss.yystv.cn/doc/6204/b72c3140aed61bd8e4497b83cc5bc8ce.png_mw680water" title="时间和空间的尽头" alt="时间和空间的尽头" class width="775" height="531" align src="https://alioss.yystv.cn/doc/6204/b72c3140aed61bd8e4497b83cc5bc8ce.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">时间和空间的尽头</span> 以今天的眼光来看，它的画面已经落后，战斗也比较冗长，尤其是雷翁大桥上面那几场战斗，魔甲兵们疯狂开“威力增幅”与“动力全开”简直能让人抓狂，但是每次重玩，我都会有新的体验。</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/d09f72d71916b62908595426f233eb37.png_mw680water" title="可怕的米亚姐姐 " alt="可怕的米亚姐姐 " class width="741" height="548" align src="https://alioss.yystv.cn/doc/6204/d09f72d71916b62908595426f233eb37.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">可怕的米亚姐姐 </span></p><p>    小时候的我会有意识地分辨谁是好人，谁是坏人。依莉丝是好人，因为她帮我打跑坏人；菲力德是坏人，因为他来打我，还把世界带入战火中；西鲁菲把全部人炖成一锅汤（回归原始之海），更是坏蛋中的坏蛋，修这个人是不是真的自私？</p><p class="picbox">    <img data-original="https://alioss.yystv.cn/doc/6204/bd74cf3f9bdf1818d869114b576339e1.png_mw680water" title="修的本愿 " alt="修的本愿 " class width="746" height="556" align src="https://alioss.yystv.cn/doc/6204/bd74cf3f9bdf1818d869114b576339e1.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">修的本愿 </span></p><p>    慢慢地，心态开始变化了，我开始站在反派角度想问题，会去思考他们的做法对不对。世间如此混乱，最后归入一体，世界大同，也不是一种解决办法吗？甚至会动手写一些东西，探索一些可能。</p><p>    很多个夜晚，我就想着这些问题入睡。</p><p>    如今，已经过去十几年了。 <br></p><p align="right"><span style="font-size: 80%;">文/留守农村大半个月的江如红</span></p>        <br>
