<p>在2018年Ben Brode（后文将简称为BB）卸任《炉石传说》游戏总监的职位选择独立门户，那时的炉石玩家社区哀嚎声一片。不仅仅是因为Ben Brode是一个幽默风趣擅长rap的社区活宝，更重要的是因为他是《炉石传说》这款游戏的重要创始人之一，是他将这股卡牌游戏的热潮席卷了全球。</p><p>依稀记得BB在谈到为什么离开时曾说道：“team5这个创意团队从当初两三个人发展到如今的规模，已经不再需要我来担任设计卡牌的工作，虽然炉石早已羽翼丰满，但我还想再去尝试游戏的设计。女巫森林是我最后参与设计的游戏版本。”</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/b76f2659f15658b9f4715733bfc2fcaa.png_mw680water" data- data-id="269855rz" alt width="905" height="300" src="https://alioss.yystv.cn/news/b76f2659f15658b9f4715733bfc2fcaa.png_mw680water" referrerpolicy="no-referrer"></div><p>与BB一同离开的还有好几位当时《炉石传说》的核心工作人员，其中包括《炉石传说》的执行制作人兼暴雪资深副总裁Hamilton Chu，他们一同在BB的车库创立了新的工作室：Second Dinner。并于不久后拿到了网易的投资，和漫威的IP授权。</p><p>届时的玩家社群只知道BB在制作一个漫威IP的游戏，但不知道究竟是什么类型。以至于后来的各种漫威IP卡牌游戏都曾被误传为BB的作品：像是漫威对决（手游），漫威群英传（桌游）。</p><p><strong>就在北京时间的5月20号，这款神秘的游戏终于向全世界公布了，它就是《Marvel Snap》。</strong></p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/afe52d68985b5c86fde1dce9c42e773d.png_mw680water" data- data-id="269855bc" alt width="1080" height="608" src="https://alioss.yystv.cn/news/afe52d68985b5c86fde1dce9c42e773d.png_mw680water" referrerpolicy="no-referrer"></div><p>不出意外，它还是一款卡牌游戏。</p><p>意外的是，这机制怎么越看越像一款桌游：《大杀四方》……</p><p>更巧的是，《大杀四方》在去年刚好出了一个漫威主题的扩展。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/f1624a164d6e531c49355b6445f6d4b7.png_mw680water" data- data-id="269855su" alt width="800" height="800" src="https://alioss.yystv.cn/news/f1624a164d6e531c49355b6445f6d4b7.png_mw680water" referrerpolicy="no-referrer"></div><p><strong><span>到底有多像？</span></strong></p><p>先来说说《大杀四方》这款游戏吧，自2012年问世以来，就一直保持着持续稳健的扩展更新，时至今日已经有近百个派系被创作出来用于混编构筑游戏。</p><p>玩家的卡组构筑方式十分简单，只需要挑两套派系不同的卡组，将它们混合洗牌即可开始游戏。游戏过程中，双方玩家需要通过打出手中的角色牌和战术牌形成最大的收益效果，来争夺中央基地牌的分数。最先拿到15分的玩家获胜。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/bc6a3e16f68891bef3d6c96e1a64e331.jpg_mw680water" data- data-id="269855nh" alt="一局大杀四方的桌面战况" class title="一局大杀四方的桌面战况" width="1080" Snap》</strong>，我只能说不愧是曾经的炉石之父，在降低卡牌游戏门槛这一点上的能力依旧相当强劲。</p><p>玩家卡组由12张不同的角色牌组成，游戏过程中会逐步揭示3个地点以及对应的效果，玩家打出手中的牌到对应的场景即可增加自己针对此地的占领值，没有先后手，玩家回合是同步的。游戏总共只进行6个回合，结束时同一场景谁的点数大谁占领这个场景，三个场景占领至少两个就是最终的赢家。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/ba8195df8f32f58bf1bbc60cef84cafa.jpg_mw680water" data- data-id="269855ss" alt="这是一局结束我占领了三个地点" class title="这是一局结束我占领了三个地点" width="1080" data-original="https://alioss.yystv.cn/news/a73f456b4800e2f56e1ae523e850de0e.png_mw680water" data- data-id="269855nf" alt="BB在推特上回复关于《大杀四方》（Smash Up）的相关问题" class title="BB在推特上回复关于《大杀四方》（Smash Up）的相关问题</span></div><p>如果用成熟的电子卡牌游戏举例子，《Marvel Snap》更多的会让人想到昆特牌，Artifact，国产的2047等比点数的游戏。</p><p>但我在看完官方实机演示和相关解说视频，并且拿到测试资格自行体验之后，我更加确信了自己的想法。</p><p><strong>《大杀四方》区域控制玩法才是这款游戏的最核心策略点。</strong></p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/154ad0ad03a98a9d2ab45953e7f840a0.png_mw680water" data- data-id="269855cs" alt="两张美国队长效果设计的惊人一致" class title="两张美国队长效果设计的惊人一致" Snap》的游戏初期将会有50多种漫威场景，以及150多种角色卡牌。玩家之间的信息攻防和心理博弈全部都要围绕二者进行。</p><p>显然角色与场景之间的互动才是游戏最核心的玩法，这与《大杀四方》可以说是完全一致。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/62fe35440f45c6bb4763c3341e894d44.png_mw680water" data- data-id="269855hr" alt width="1080" height="532" src="https://alioss.yystv.cn/news/531b556b69aa386ad847386fa672466e.jpg_mw680water" referrerpolicy="no-referrer"></div><p>当我看到又一款桌面游戏机制因为好的改动和适配搬上了电子游戏平台，心中不免慨叹：桌游的魅力其实远不止如此。</p><p>在《Marvel Snap》之前还有许许多多的卡牌游戏搬上过电子游戏平台，这种情况早已屡见不鲜。</p><p><strong><span><span class="r-triangle">优秀的桌游机制总会发光</span></span></strong></p><p>DBG（Deck Building Game）作为桌游的一大游戏机制，一直以来都是玩家之间津津乐道的玩法设计。（最近的SDJ大奖黑标提名《沙丘》就加入了DBG玩法。）</p><p>前炉石设计师Peter Whalen的成名之作dream quest 就是一个融合了“rouguelike”与“DBG”玩法的游戏。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/ce929680b8c9bda6e7653add66377e3f.png_mw680water" data- data-id="269855cx" alt="《dream quest》游戏界面" class title="《dream quest》游戏界面" width="1080" height="807" quest》游戏界面</span></div><p>虽然游戏界面和美术毫无亮点，甚至很多人都不知道这个游戏。但这个玩法创意的确影响了后来的知名独立卡牌游戏——杀戮尖塔。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/0a2c416341e32a05db1a6e4b0e72beb7.png_mw680water" data- data-id="269855qi" alt width="918" height="516" referrerpolicy="no-referrer"></div><p>杀戮尖塔的爆红无需多言，不论知名游戏公司还是手游小厂，都开始在自己旗下的游戏中增加“rouguelike”模式和"DBG"设计。“爬塔”这个词突然摇身一变，成为了这类玩法模式的标签。</p><p>《冒险公社》是由一群从国内互联网大厂离职寻求梦想的游戏人制作的，我到现在都还记得他们当初在营地上发布游戏Demo，分享游戏制作的帖子，他们喜欢桌游，坚定选择，忠心梦想。</p><p>于是，《冒险公社》就这样一步一个脚印被创作出来了，目前在steam上有着高达90%的好评率。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/4dc3e88ef23b3fc4543b56b88ab507d5.png_mw680water" data- data-id="269855cl" alt width="900" height="337" src="https://alioss.yystv.cn/news/4dc3e88ef23b3fc4543b56b88ab507d5.png_mw680water" referrerpolicy="no-referrer"></div><p>《冒险公社》的基础规则与设定与经典桌游《印加宝藏》相似——游戏的制作组也直言不讳的说明游戏的灵感来自于这款经典桌游——探索小岛，寻找宝藏，最终获得金币数最多的玩家获胜。</p><p>玩家之间的勾心斗角总是能带来无限欢乐。</p><p><strong>这些脱胎于经典桌游的规则保证了游戏有一个坚实的土壤，而在这之上《杀戮尖塔》与《冒险公社》</strong></p><p><strong>都结出了属于自己的累累硕果。</strong></p><p><strong><span><span class="r-triangle">完整的搬运也是一种选择</span></span></strong></p><p>受到玩家认可的桌游还有很多。它们在桌面领域打响了名声，作者以及发行方自然也希望能将这些优秀的作品扩散到更广阔的人群。</p><p><strong>平台适配</strong></p><p>于是出现了这样一些桌游，它们在被完整搬运到电子平台上的同时也做了相应的平台适配，于是收获了不错的效果。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/40326237f68ecbd83efcbede54554ea1.png_mw680water" data- data-id="269856ek" alt="BGG排名第一《幽港迷城》" class title="BGG排名第一《幽港迷城》" width="639" data-original="https://alioss.yystv.cn/news/06acfd0c50fe89a30e9d8f0a9710e422.png_mw680water" data- data-id="269856wi" alt="《展翅翱翔》" class title="《展翅翱翔》" width="757" height="700" class="picinfo">《展翅翱翔》</span></div><p>在完整搬运了桌游规则的基础上，还增加了鸟类动画和声效，从而深入强化了玩家对于鸟类观察者的代入。游戏优秀的单人模式也在电子平台上得到了很好的体现。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/a2a4a8a434bb05ac93ea29e381c02111.png_mw680water" data- data-id="269856dr" alt="《茂林源记》" class title="《茂林源记》" width="652" height="625" src="https://alioss.yystv.cn/news/a2a4a8a434bb05ac93ea29e381c02111.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">《茂林源记》</span></div><p>茂林源记在原有地图的基础上进行了全新的UI设计，更加适配电脑玩家的需求。音乐，画风和各种动画不免让人想到《动物森友会》，算是桌游改编的优秀案例。</p><p><strong>只求模拟</strong></p><p>TTS（Tabletop Simulator）是一款功能强大的桌游模拟器。这里的模拟是指切实的现实模拟，手拿，投掷，洗切等手部操作都是需要鼠标来完成。也就是说这里不存在背后的系统结算逻辑，一切都是玩家自行操作。</p><p>玩家想要体验一款桌游，只需要拿到相应的贴图文件然后进行一个初始布置即可开始游戏。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/8e5b0a36f82d991627217e47ff19a303.png_mw680water" data- data-id="269856td" alt="TTS展示界面" class title="TTS展示界面" width="938" height="526" src="https://alioss.yystv.cn/news/8e5b0a36f82d991627217e47ff19a303.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">TTS展示界面</span></div><p>BGA（BoardGameArena）是一个为特定桌游制作了背后结算系统的网页平台，玩家可以在这里体验到那些机制经典，结算简单的桌游。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/bed812254e9454c28a4f1a1628119dd4.png_mw680water" data- data-id="269856ly" alt="这是我的BGA游戏收藏，除此之外还有很多" class title="这是我的BGA游戏收藏，除此之外还有很多" width="1080" height="603" src="https://alioss.yystv.cn/news/bed812254e9454c28a4f1a1628119dd4.png_mw680water" referrerpolicy="no-referrer"><span class="picinfo">这是我的BGA游戏收藏，除此之外还有很多</span></div><p>不论是TTS还是BGA，设计者的初衷都是尽可能简单方便地将大家喜爱的桌游搬运到网络环境中。没有对细节的追求，都是纯粹的极致体验模拟，更加实用主义。</p><p><strong>反面教材</strong></p><p>至于失败的案例，我想大家一定知道常年霸榜steam差评率第一的，正是一款桌面游戏的电子版，甚至还有通过给差评率第二游戏刷好评来确保第一地位不可动摇的操作。在这里就不说名字了。</p><p><strong><span class="r-triangle">桌游适配电子平台，应该做些什么？</span></strong></p><p>一旦决定要将桌游搬上电子平台，而且不是制作简单的系统逻辑。那么考虑到的事情就相当多了。这里我询问了有相关移植适配经验的同事来分享了一些经验。</p><p><strong>系统逻辑</strong></p><p>逻辑结算可以说是最简单的部分，因为桌游的规则说明书已经事无巨细的讲解非常清楚了。</p><p><strong>屏幕尺寸</strong></p><p>屏幕的尺寸适配应该是最难的挑战，因为桌游的所有内容都需要直观体现在一个小屏幕上。很多区域之间的结算效果都需要增加交互来体现。整体的排布往往需要一遍又一遍地修改。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/b81248335ccd1b445fd421b8fde1e771.jpg_mw680water" data- data-id="269856qd" alt width="1080" height="608" src="https://alioss.yystv.cn/news/b81248335ccd1b445fd421b8fde1e771.jpg_mw680water" referrerpolicy="no-referrer"></div><p><strong>匹配逻辑</strong></p><p>开房间，单人模式，机器人模式。每种模式之间都会有些微的差别。是必须要做，且很麻烦去做的事情。</p><div class="picbox"><img data-original="https://alioss.yystv.cn/news/3a5a8d650eb69fde446abc0f4ef7926e.png_mw680water" data- data-id="269856hl" alt width="452" height="476"
