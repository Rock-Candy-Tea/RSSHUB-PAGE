<p style="text-indent:2em;">
微软的Game Pass服务共有三个版本：Xbox Game Pass（主机游戏），Xbox Game Pass PC版（电脑游戏）以及Xbox Game Pass Ultimate，现在在2021 TGA上，微软正式宣布Xbox Game Pass PC版将更名为PC Game Pass。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211210/1639120761_764043.jpg" alt="微软宣布PC版Xbox Game Pass更名为PC Game Pass" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
除了名字发生改变之外，PC版XGP的内容不会发生改变。按照玩家的习惯，PC Game Pass可以简称为PGP。
</p>
