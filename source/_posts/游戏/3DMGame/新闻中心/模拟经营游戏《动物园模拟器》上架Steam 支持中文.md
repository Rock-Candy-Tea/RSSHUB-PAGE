<div>   
<p style="text-indent:2em;">
模拟经营游戏《<a target="_blank" href="https://www.3dmgame.com/games/zooicide/">动物园模拟器</a>（Zoo 
Simulator）》<a href="https://store.steampowered.com/app/2129220/Zoo_Simulator/" target="_blank">现已上架Steam</a>，自带简体中文。在动物园模拟器游戏里，玩家的主要工作就是运营、管理好你自己的动物园。不断对其进行环境提升，同时要照顾到所有动物的需求，确保他们能享受平和的生活！你想看到大批游客来游玩吗?那就悉心打造一座你自己的动物园吧!
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220906/1662423180_107285.jpg" alt="模拟经营游戏《动物园模拟器》上架Steam 支持中文" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
<strong>翻修你的动物园！</strong> 
</p>
<p style="text-indent:2em;">
在动物园模拟器游戏里，你的首要任务之一将是把动物园恢复如初。你必须要投入大量精力，为你未来要养的动物们清理不易注意到的小巷和被破坏的地方。做这些工作，你需要用到许多工具，诸如电焊机和高压水枪等。请记住 
- 辛勤的工作后随之而来的将是一群快乐的游客！
</p>
<p style="text-indent:2em;">
<strong>不同种类的动物</strong> 
</p>
<p style="text-indent:2em;">
动物园里有动物生活才能叫动物园，是吧?在这款游戏里，你有权决定哪些动物可以生活在这片设施里。要知道，在动物园里生活着的动物越有意思，越会有更多的游客想去看它们，也就意味着你的生意会有更多的收入。坐下来看看电脑，从世界各地最奇特、最令人惊奇的动物中选择一些吧!
</p>
<p style="text-indent:2em;">
<strong>为动物们奔走</strong> 
</p>
<p style="text-indent:2em;">
你的动物园里会出现来自世界最遥远角落的动物，它们需要合适的生活条件 - 
因此你必须为其提供好的巢穴、藏身处以及它们喜欢的植物。游戏中会出现多种不同的元素，你可以进行自由移动，定位及旋转，从而为动物们打造出独特的生活区域。
</p>
<p style="text-indent:2em;">
<strong>时刻关心它们</strong> 
</p>
<p style="text-indent:2em;">
在动物园模拟器游戏里，你还需要时刻注意动物们的健康状况，它们会以和平、健康及行动来回报你。按时给它们喂食、关心它们，并特别注意保证它们的安全 - 
动物们喜欢咬他们的鼻子，只要有机会就会咬上一口! 正因为如此，动物才更有可能向游客展示自己，赢得他们的同情。
</p>
<p style="text-indent:2em;">
<strong>不断扩充你的动物园</strong> 
</p>
<p style="text-indent:2em;">
经营业务的同时也要不断地发展壮大。那样将会有更多的游客来访，因为会更多的景点让他们在游玩的时间里感到更加愉悦。注意设置一定数量的公共厕所及卫生设施，摆满长椅的休息区和他们能够进餐的地方！还有，别忘了不定时地引进一些有意思的新的动物品种。这是非常重要的环节，游客们不但对动物园流连忘返，而且还会多邀请他们的朋友一起来。
</p>
<h3 style="text-indent:2em;">
系统需求
</h3>
<p style="text-indent:2em;">
<strong>最低配置:</strong> 
</p>
<p style="text-indent:2em;">
操作系统: Windows 7 64 Bit / Windows 8 64 Bit / Windows 10 64 Bit
</p>
<p style="text-indent:2em;">
处理器: Intel Core i3 3.0 GHz
</p>
<p style="text-indent:2em;">
内存: 8 GB RAM
</p>
<p style="text-indent:2em;">
显卡: Nvidia GeForce GTX 1050
</p>
<p style="text-indent:2em;">
<strong>推荐配置:</strong> 
</p>
<p style="text-indent:2em;">
操作系统: Windows 7 64 Bit / Windows 8 64 Bit / Windows 10 64 Bit
</p>
<p style="text-indent:2em;">
处理器: Intel Core i5 3.4 GHz
</p>
<p style="text-indent:2em;">
内存: 16 GB RAM
</p>
<p style="text-indent:2em;">
显卡: NVidia GeForce GTX 1060
</p>
