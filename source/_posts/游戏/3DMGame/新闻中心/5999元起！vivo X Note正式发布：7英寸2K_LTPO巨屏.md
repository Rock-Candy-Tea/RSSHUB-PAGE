<div>   
<p style="text-indent:2em;">
今晚19:30，vivo召开一场重磅发布会，推出了全新的X系列成员，其中包括在如今这个时代依然能称之为大屏旗舰的vivo X Note。
</p>
<p style="text-indent:2em;">
作为一款大屏旗舰，vivo X Note搭载了7英寸OLED屏，在如今主流手机都控制在6.6英寸左右的时代，只有折叠屏的尺寸能与之媲美了，不过机身却非常轻薄，整机重量216g（素皮），厚度为8.75mm，vivo专门优化了重心，不会出现坠手感。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220411/1649692645_465129.jpg" alt="5999元起！vivo X Note正式发布：7英寸2K/LTPO巨屏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
大屏手机的屏幕素质当然不能拖后腿，vivo X Note这块屏幕几乎是业内天花板级别，采用三星E5材质打造的双曲面屏，拥有21:10黄金宽比例，分辨率达到3080×1440，达成16项Display Mate表现记录，并拿到了整体A+级别的认证。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220411/1649692661_599791.png" alt="5999元起！vivo X Note正式发布：7英寸2K/LTPO巨屏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
同时，vivo X Note还支持LTPO自适应刷新3.0，从系统端输出对应的刷新频率，显示刷新率跟随系统刷新率，实现全通路刷新率自适应，功耗降低效果更佳，兼顾高刷和续航。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220411/1649692677_180036.png" alt="5999元起！vivo X Note正式发布：7英寸2K/LTPO巨屏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
最重要的是，vivo X Note配备了3D大面积超声指纹，指纹识别区域是单点光电指纹的11.1倍，识别区域高达578mm²，基本可以盲解，而且录入过程只需要按压一次，手指沾水也依然能快速识别。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220411/1649692697_772779.png" alt="5999元起！vivo X Note正式发布：7英寸2K/LTPO巨屏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
核心性能上，vivo X Note搭载目前安卓最强芯片之一骁龙8 Gen1，采用先进的4nm MEP制程工艺，集成Adreno730 GPU和骁龙X65 5G基带等，搭配上双超频版UFS3.1/增强版LPDDR5，可将12G运存扩展成等效16G使用的内存融合2.0，让整体性能及其强劲。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220411/1649692711_370291.png" alt="5999元起！vivo X Note正式发布：7英寸2K/LTPO巨屏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
此外，新机还同时配备了自研V1芯片，独立于GPU之外，内建超画质AI系统，在影像、显示、游戏等多方面实现专业级的视觉增强，同时降低GPU负载，大幅降低功耗，达到更好的影像效果，并提供游戏插帧体验。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220411/1649692725_365060.png" alt="5999元起！vivo X Note正式发布：7英寸2K/LTPO巨屏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
续航方面，vivo X Note内置了块5000mAh（典型值）超大电池，官方称充满一次足够连续玩大型游戏7小时，用耳机听音乐52小时，看流媒体视频12.8个小时，完全可以摆脱电量焦虑。
</p>
<p style="text-indent:2em;">
同时还支持80W有线闪充、50W无线闪充，以及10W无线反向充电，有线状态12分钟即可充满一半，充满5000mAh，只需要不到35分钟。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220411/1649692740_773781.png" alt="5999元起！vivo X Note正式发布：7英寸2K/LTPO巨屏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
vivo X Note还标配了1.5m数据线和Type-C口充电器，不仅让日常使用更轻松，还能够满足多个电子设备的充电需求，差旅场景也更从容。
</p>
<p style="text-indent:2em;">
影像方面，vivo X Note搭载了蔡司四摄模组，规格分别为5000万像素OIS超大底主摄、4800万像素114°超广角主摄、1200万专业人像镜头，以及一颗800万像素60倍OIS潜望式超远摄，全系镜头标配蔡司T*镀膜认证。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220411/1649692753_367481.png" alt="5999元起！vivo X Note正式发布：7英寸2K/LTPO巨屏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
其中主摄为三星GN1旗舰级传感器，拥有1/1.31"大底，潜望式长焦镜头支持5倍光学变焦，这两颗镜头均配备了OIS光学防抖，能有效提升成片率以及画面效果，配合EIS混合防抖算法，还能在多场景发挥出色的防抖性能。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220411/1649692767_975840.png" alt="5999元起！vivo X Note正式发布：7英寸2K/LTPO巨屏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
除了硬件之外，vivo与蔡司合作，模拟蔡司首款数码相机ZX1的成像风格，共同研发了“蔡司自然色彩”，更加还原人眼所见的色彩风格。
</p>
<p style="text-indent:2em;">
还支持蔡司超级夜景实现拍摄纯净夜景的效果，减少杂光、炫光、“有害”光，还原纯净夜色，同时还能基于V1芯片的强大算力，在原有黑光夜视的基 础上进一步提升，实现预览状态的提亮，在拍照前即可看到如成 片的亮度效果(特定亮度环境)，不再摸着黑凭感觉取景。
</p>
<p style="text-indent:2em;">
其他方面，vivo X Note还拥有IP68防尘防水、红外遥控、独立HiFi芯片、X轴线性马达等配置，高端旗舰标配几乎一应俱全。
</p>
<p style="text-indent:2em;">
另外还有一大亮点就是实体静音按键，这是vivo体系内首度搭载静音键，按钮符合人体工学设计， 通过双霍尔传感器+磁铁形式纵向拨动，只需向下拨动开启静音，即可关闭提示音，而媒体音和闹钟仍保留。
</p>
<p style="text-indent:2em;">
vivo X Note拥有晴山蓝、大地灰、璨夜黑三款配色可选，黑色为玻璃后壳版本，提供8+256GB、12+256GB、12+512GB三种存储组合可选，售价分别为5999元、6499元、6999元，将于4月15日线上线下同步开售。
</p>          
</div>
