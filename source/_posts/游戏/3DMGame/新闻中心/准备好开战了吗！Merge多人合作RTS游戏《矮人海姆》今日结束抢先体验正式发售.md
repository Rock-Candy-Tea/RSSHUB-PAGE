<p style="text-align:center;text-indent:2em;">
V1.0版在Steam上推出，限时首周7折特惠
</p>
<p style="text-indent:2em;">
发行商Merge Games和挪威开发商Pineleaf 
Studio今日骄傲宣布，他们的多人合作RTS游戏《<a target="_blank" href="https://www.3dmgame.com/games/dwarfheim/">矮人海姆</a>》(DwarfHeim)将于今天结束抢先体验，并于今天在Steam上发布V1.0正式版。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210927/1632734607_280992.png" alt="准备好开战了吗！Merge多人合作RTS游戏《矮人海姆》今日结束抢先体验正式发售" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
为了庆祝发布，想要购买《矮人海姆》的玩家可以在Steam上享受首周7折特惠，折扣持续到10月3日。游戏发售定价80人民币，支持简体中文。
</p>
<p style="text-align:center;">
<iframe src="//player.bilibili.com/player.html?aid=890742286&bvid=BV1nP4y1a7KD&cid=415381854&page=1" width="100%" height="458" frameborder="no" allowfullscreen="true">
</iframe>
</p>
<p style="text-indent:2em;">
《矮人海姆》是新一代的即时战略游戏，具有令人兴奋的实时对战合作游戏模式，吸引世界各地的多人游戏爱好者。玩家的角色对你的部族的成功至关重要，选择正确的角色类型与否直接影响正常战斗的运筹帷幄。
</p>
<p style="text-indent:2em;">
对于喜欢单人游戏的玩家来说，《矮人海姆》也支持玩家和AI对战，享受单人游戏的乐趣。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210927/1632734607_712591.png" alt="准备好开战了吗！Merge多人合作RTS游戏《矮人海姆》今日结束抢先体验正式发售" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
除了游戏的更新和修复，《矮人海姆》的V1.0版本将包括大量的新功能，以增强充满行动力的RTS游戏体验，包括。
</p>
<p style="text-indent:2em;">
n 新的泰恩角色可供选择，每个人都有一套新的独特能力
</p>
<p style="text-indent:2em;">
n 比赛结束后，各方仍可在大厅内进行操作。
</p>
<p style="text-indent:2em;">
n 生存系统优化，新增一个具有新敌人的激烈血月事件
</p>
<p style="text-indent:2em;">
n 新的生存界面
</p>
<p style="text-indent:2em;">
n 新的防卫升级系统
</p>
<p style="text-indent:2em;">
n 更新了寻路工具
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210927/1632734607_642443.png" alt="准备好开战了吗！Merge多人合作RTS游戏《矮人海姆》今日结束抢先体验正式发售" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
游戏特色：
</p>
<p style="text-indent:2em;">
n 征服模式：《矮人海姆》的主要多人合作模式。选择一个等级，与其他两个玩家一起，共同撕开敌人的防御，摧毁他们的市政厅。
</p>
<p style="text-indent:2em;">
n 
冲突模式：该模式可用于单人游戏和在线合作游戏。在这个具有挑战性的模式中测试你的策略，让你的矮人在一系列的地图中与人工智能控制的对手部族对抗，每张地图都比上一张更具挑战性。
</p>
<p style="text-indent:2em;">
n 
生存模式：一个艰苦的、快节奏的挑战模式。所有等级、建筑、单位和能力都会立即解锁。在生存模式中，挑战自己并试图在巨魔的冲击下活下来。生存模式将在单人游戏和在线合作游戏中提供。
</p>
<p style="text-indent:2em;">
n 
<a target="_blank" href="https://www.3dmgame.com/tag/shahe_1/">沙盒</a>模式：一个更轻松的游戏模式，适合那些喜欢慢慢来的玩家。所有单位和建筑都会立即解锁。在这个创造性的游戏模式中，建立你自己的经验，并创造你的完美定居点。沙盒模式也可用于单人游戏和在线合作游戏。
</p>
