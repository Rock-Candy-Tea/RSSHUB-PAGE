<p style="text-indent:2em;">
近日，制作《变异泥巴》的独立游戏开发商Atooi宣布将在Switch平台推出休闲在线派对新作，游戏命名为《Run Box Run》，计划将于今年推出。
</p>
<p style="text-indent:2em;">
据Atooi创始人Jools Watsham表示，有关《Run Box Run》的更多信息将在不久之后公布，喜欢的玩家可以关注后续。
</p>
