<h3>
游戏剧情：
</h3>
<p style="text-indent:2em;">
在不久的将来，木卫六——泰坦上的一次剧烈流星碰撞将大量稀有物质散落到冰冷的月球表面。一家名为“TECC”的矿业公司派遣大量人员建立了一个殖民地来收获资源。尽管取得了不小的发展，但当殖民地陷入沉寂时，任务戛然而止。几年后，第二个小组被派去收拾残局并调查发生了什么。
</p>
<p style="text-indent:2em;">
他们发现之前的挖矿队已经没有了踪迹，让新来的人相信每个人都已经死了……知道他们找到了你。
</p>

<h3>
游戏玩法：
</h3>
<p style="text-indent:2em;">
玩家将要与同样在受苦的惠更斯营地打交道，帮助居民解决威胁他们生存的问题。每一刻都很重要，世界不会等你——营地的居民有自己的需求，什么都不做就无法永远生存。游戏不会等待您的决定——小心，否则您将成为自己最大的敌人！
</p>

<h3>
游戏特色：
</h3>
<div>
<ul>
<li>
生存
</li>
<ul>
<li>
· 通过满足日常需求（例如饥饿和口渴）来度过每一天。
</li>
<li>
· 治疗受到的伤害，例如骨折和失血。
</li>
<li>
· 收集装备，或从环境中收集材料。
</li>
<li>
· 在探索废弃矿山的同时管理氧气和能量。
</li>
<li>
· 与时间本身的流逝斗争。
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210617/1623918695_118461.gif" alt="地底生存回合制游戏《地下设施2》Steam商店页面上线" referrerpolicy="no-referrer"> 
</p>
</li>
</ul>
<li>
战斗
</li>
<ul>
<li>
· 直观而快速的回合制战斗。
</li>
<li>
· 用近战和远程武器对抗来自泰坦的恐怖怪物。
</li>
<li>
· 敌人会进化，随着时间的推移获得更强的力量。
</li>
<li>
· 利用纳米机器的力量来使用强大的能力。
</li>
<li>
· 保护您的反应堆免受入侵者的侵害。
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210617/1623918706_512921.gif" alt="地底生存回合制游戏《地下设施2》Steam商店页面上线" referrerpolicy="no-referrer"> 
</p>
</li>
</ul>
<li>
制作
</li>
<ul>
<li>
· 将设备分解为材料和工艺组件。
</li>
<li>
· 生产盔甲、枪支、近战武器、手榴弹和升级。
</li>
<li>
· 制造性能增强剂和医疗设备。
</li>
<li>
· 研究项目以设计更好的版本，或自己创造高级设计图。
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210617/1623918714_369911.gif" alt="地底生存回合制游戏《地下设施2》Steam商店页面上线" referrerpolicy="no-referrer"> 
</p>
</li>
</ul>
<li>
探索
</li>
<ul>
<li>
· 探索惠更斯营的采矿前哨。
</li>
<li>
· 与居民交谈并与各种营地商人进行交易。
</li>
<li>
· 发现并帮助营地居民解决问题。
</li>
<li>
· 找出所发生事件的证据，并存活足够长的时间逃离泰坦。
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210617/1623918724_760394.gif" alt="地底生存回合制游戏《地下设施2》Steam商店页面上线" referrerpolicy="no-referrer"> 
</p>
</li>
</ul>
<li>
管理
</li>
<ul>
<li>
· 世界会随着玩家的行为和选择做出改变。
</li>
<li>
· 帮助管理营地日益减少的资源，并找到生产更多资源的方法。
</li>
<li>
· 将任务分配给镇上的居民，或帮助他们完成自己的任务。
</li>
<li>
· 收获材料并改善城镇的基础设施。
</li>
<li>
· 建造和升级工艺站以制造先进的设备。
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210617/1623918734_667898.gif" alt="地底生存回合制游戏《地下设施2》Steam商店页面上线" referrerpolicy="no-referrer"> 
</p>
</li>
</ul>
</ul>
</div>
<p style="text-indent:2em;">
游戏目前暂未公布发布日期，开发者表示游戏还处于Alpha早期开发阶段，功能和设计可能会发生改变；目前该作暂不支持简体中文。
</p>
