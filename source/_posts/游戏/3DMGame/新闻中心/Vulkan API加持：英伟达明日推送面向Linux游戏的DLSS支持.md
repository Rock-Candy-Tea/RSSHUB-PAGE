<div>   
<p style="text-indent:2em;">
本月初，英伟达宣布了要将 GeForce RTX 系列显卡的“深度学习超级采样”（DLSS）技术引入 Linux / Steam Play 
平台，而现在正是时候。即使当前尚无任何原生支持 DLSS 的 Linux 游戏，但该项目主要是给基于 Steam Play（Proton 
项目）的游戏提供尝鲜，后者旨在让 Windows 游戏在 Linux 上顺畅运行。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210622/1624366031_613940.jpg" alt="Vulkan API加持：英伟达明日推送面向Linux游戏的DLSS支持" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
在即将推出的最新版 Linux 驱动程序中，英伟达将通过 Vulkan API 来实现对 Linux 游戏的 DLSS 支持。
</p>
<p style="text-indent:2em;">
英伟达指出，对采用 Vulkan API 的 DLSS 游戏支持，将于本周二正式上线。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210622/1624366064_500423.jpg" alt="Vulkan API加持：英伟达明日推送面向Linux游戏的DLSS支持" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
届时 Linux 平台玩家也将能够充分利用 GeForce RTX GPU 专用的 AI Tensor Core 
核心来加速《毁灭战士：永恒》、《无人深空》、以及《德军总部：新血脉》等游戏的帧率。
</p>
<p style="text-indent:2em;">
不过要让基于 Direct3D 的 DLSS 游戏在 Steam Play 下调用 DXVK / VKD3D-Proton 
功能，所需部分仍要等到今秋才能准备就绪。
</p>
<p style="text-indent:2em;">
最后，除了对 Linux 平台的 DLSS 测试版驱动支持，470 系列驱动还有望改善对 Wayland 的支持。
</p>          
</div>
