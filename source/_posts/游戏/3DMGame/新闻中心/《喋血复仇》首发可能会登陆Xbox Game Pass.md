<p style="text-indent:2em;">
Turtle Rock开发的《<a target="_blank" href="https://www.3dmgame.com/games/b4b/">喋血复仇</a>》首发当天可能会登陆主机和PC端的Xbox Game Pass。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210612/1623503389_513915.jpg" alt="《喋血复仇》首发可能会登陆Xbox Game Pass" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
《喋血复仇》最近上线了一个视频，视频描述中写道“《喋血复仇》在发售时将登陆Xbox Game Pass（Xbox One以及Windows 10 
PC）”。后来据说这段视频被隐藏为“私人”。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210612/1623503398_504998.jpg" alt="《喋血复仇》首发可能会登陆Xbox Game Pass" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
华纳兄弟计划的《喋血复仇》展示将于6月14日早晨5点开始，去年夏季，Xbox总裁Phil Spencer说他们专注于将更多的第三方游戏登陆Xbox 
Game Pass，今年早些时候微软签到了SE的射击游戏《Outriders》，后者首发登陆了Xbox Game Pass。
</p>
