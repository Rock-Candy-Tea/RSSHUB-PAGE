<div>   
<p style="text-indent:2em;">
因为《罪恶装备》、《龙珠斗士Z》和《苍翼默示录》几部热门作品，Arc System Works 已成为市场上最好的格斗游戏开发商之一，但该工作室正在寻找另一个具有可识别
IP 的主流热门游戏。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220810/1660096908_937247.jpg" alt="《罪恶装备》开发商Arc System Works希望积极寻求新IP合作项目" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
在 EVO
2022 上，Arc
System Works 首席执行官
Minoru Kidooka 解释了该工作室如何通过联系
IP 权利人来寻求新的机会，而不是被动地等待机会落入其手中。
</p>
<p style="text-indent:2em;">
“我们需要通过
IP 扩展格斗游戏社区，”Kidooka
对 IGN 说：“未来，如果我们有这样的机会，我们会积极推动与新的 IP 所有者合作。”
</p>
<p style="text-indent:2em;">
ASW 对合作并不陌生，因为它开发了屡获殊荣的《龙珠斗士Z》、3DS《海贼王》游戏、2012 年的《女神异闻录4：午夜竞技场》和 2020 年的《碧蓝幻想Versus》。
Kidooka 补充说，该工作室的原创作品《罪恶装备》和《苍翼默示录》不会被遗忘，因为开发商的目标是开发在美国计划和开发的
IP，特别是《苍翼默示录》具有超越格斗游戏类型的潜力。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220810/1660097258_566142.jpg" alt="《罪恶装备》开发商Arc System Works希望积极寻求新IP合作项目" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
ASW 最新的作品同样是与热门 IP 合作的格斗游戏《地下城与勇士：决斗》。该作虽然有一些平衡问题，但是很好的兼并了休闲玩家和高玩的体验，为许多玩家带来了乐趣，在 Metacritic 上获得了 80 分的媒体评价，以及 6.6 分的用户评价。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220810/1660097289_427211.jpg" alt="《罪恶装备》开发商Arc System Works希望积极寻求新IP合作项目" referrerpolicy="no-referrer">
</p>
