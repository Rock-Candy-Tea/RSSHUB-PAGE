<div>   
<p style="text-indent:2em;">
在今日（6月1日）召开的2021台北国际电脑展（Computex 2021）上，AMD正式发布了Radeon RX 6000M系列笔记本电脑显卡，基于高度可扩展的RDNA 2架构。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210601/1622519086_435815.png" alt="AMD RX 6000M笔记本显卡发布：6800M大幅领先3080" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
AMD官方表示，RNDA 2架构可以在获得与前代RDNA架构相同性能的同时，功耗水平降低50-60W。其中，高端产品AMD Radeon RX 6800M，配备了2300MHz的游戏时钟频率、高达12GB的GDDR6显存，适用于要求苛刻的游戏。
</p>
<p style="text-indent:2em;">
AMD Radeon RX 6800M显卡将首发搭载于ROG Strix G15笔记本，运行游戏《生化危机8：村庄》时，在1440p分辨率的最高画质下可实现高达120帧的高性能表现。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210601/1622519099_482664.png" alt="AMD RX 6000M笔记本显卡发布：6800M大幅领先3080" referrerpolicy="no-referrer"> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210601/1622519104_196269.png" alt="AMD RX 6000M笔记本显卡发布：6800M大幅领先3080" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
AMD Radeon RX 6000M系列移动GPU中包括了Navi 22、Navi 23和Navi 24几款GPU，旗舰SKU为Radeon RX 6800M。由AMD官方晒出的PPT可知，Radeon RX 6800M在多款游戏测试中都大幅领先英伟达RTX 3080 8GB显卡，比如《生化危机8：村庄》、《尘埃 5》、《使命召唤17：黑色行动冷战》、《无主之地3》和《刺客信条：英灵殿》。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210601/1622519240_249053.jpg" alt="AMD RX 6000M笔记本显卡发布：6800M大幅领先3080" referrerpolicy="no-referrer"> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210601/1622519240_578557.jpg" alt="AMD RX 6000M笔记本显卡发布：6800M大幅领先3080" referrerpolicy="no-referrer"> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210601/1622519240_943117.jpg" alt="AMD RX 6000M笔记本显卡发布：6800M大幅领先3080" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
除了领先的移动游戏体验，AMD 还宣称RX 6800M的能源效率较英伟达RTX 3080移动显卡提升了40%。
</p>
<p style="text-indent:2em;">
Radeon RX 6700M中端移动显卡SKU，它也使用Navi 22 GPU，但流处理器数量减少到了2304个。主频依然是2.3GHz，辅以160 bit+10GB GDDR6显存+80MB Infinity Cache无限缓存，TGP功耗为135W，适合1440p/1080p游戏体验。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210601/1622519311_442530.png" alt="AMD RX 6000M笔记本显卡发布：6800M大幅领先3080" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
面向主流移动GPU市场的AMD Radeon RX 6600M，以及面向移动工作站客户的Radeon Pro W6600M 。两款SKU均使用了Navi 23 RDNA 2核心，预计拥有28组CU/1792个流处理器，辅以128 bit 8GB GDDR6显存+32MB Infinity Cache无限缓存，TGP功耗为100W。
</p>          
</div>
