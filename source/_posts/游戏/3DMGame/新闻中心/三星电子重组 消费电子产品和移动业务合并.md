<div>   
<p style="text-indent:2em;">
三星电子宣布对其业务进行内部重组，领导层发生了多次变化。该公司将有两位新的 CEO 负责监督两个部门：设备解决方案部门，即半导体业务，以及一个新的 SET 部门，包括移动、电视和消费电子产品。这是自 2017 年以来三星结构的最大变化，当时该公司任命了三个独立部门的三位领导人。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211207/1638866737_780159.jpg" alt="三星电子重组 消费电子产品和移动业务合并" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
视觉显示业务负责人 Jong-hee (JH) Han 现在也是负责 SET 部门的联席首席执行官。三星称赞他带领公司连续 15 年在全球电视销售中位居榜首，并表示他“有望加强 SET 部门不同业务之间的协同作用，并帮助推动新业务和技术的发展。”
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211207/1638866744_646094.jpg" alt="三星电子重组 消费电子产品和移动业务合并" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
另一位新任 CEO 是前三星电机 CEO Kyehyun Kyung。他过去还曾在三星的闪存产品团队和 DRAM 设计团队工作，现在将负责领导三星至关重要的高利润半导体部门。
</p>
<p style="text-indent:2em;">
三星还任命了 SET 和 DS 部门的北美办事处负责人，以及新的首席财务官 Hark Kyu Park。管理层重组有效地将这家庞大的科技巨头的组织结构简化为分别针对消费品和组件的两项业务。
</p>

</p>
