<div>   
<p style="text-indent:2em;">
微软宣布他们将扩展Xbox Game Pass 
Ultimate所提供的内容，这包括游戏Demo以及通过云串流游玩用户已经购买的游戏，这包括用户已经拥有的部分游戏或是在Xbox Game 
Pass游戏库之外购买的游戏。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220609/1654780156_968161.jpg" alt="Xbox Game Pass Ultimate功能拓展：将支持Demo和串流购买的游戏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
Xbox游戏体验兼平台副总裁Ashley McKissick解释说：“玩家告诉我们的其中一个事情是他们想串流他们所拥有的游戏，甚至这些游戏并不在Game 
Pass游戏库中。所以今年晚些时候，如果你是Game Pass 
Ultimate会员，你将能够从你购买的云游戏和你已经拥有的游戏中串流，只要它们在目录中是支持云的。”
</p>
<p style="text-indent:2em;">
微软还公布了Project Moorcroft项目，Xbox Game 
Pass会员可以游玩即将发售游戏的Demo。该项目将在明年推出，一开始会专注于为全世界的独立开发者提供更多的机会为他们的游戏打造兴奋点。
</p>
<p style="text-indent:2em;">
参与的开发者将能看到他们的Demo表现如何并获得补偿。
</p>
<p style="text-indent:2em;">
微软Sarah 
Bond解释说：“这就像你以前去E3，去PAX，你会参观一些你最喜欢的开发者，他们会给你提供一个游戏的关卡，让你坐在那里游玩。工作室的人就在你旁边......你可以体验游戏，他们可以让他们正在打造的东西和即将到来的东西产生兴奋。
</p>
<p style="text-indent:2em;">
但这些机会越来越小，越来越难以复制。
</p>
<p style="text-indent:2em;">
所以我们说，"为什么我们不把Game Pass变成一个展示厅?我们为什么不让开发者把他们游戏的一部分发布到Game 
Pass中呢，为即将到来的游戏预热，并在他们调整和准备发售他们的游戏时获得真正有价值的反馈?"
</p>
<p style="text-indent:2em;">
最后，微软表示他们将在世界各地新增“数千个”定制Xbox Series X机架，使其云游戏容量增加125%。
</p>          
</div>
