<div>   
<p style="text-indent:2em;">
微软宣布，台服Xbox Game Pass及Xbox Live Gold服务预计从2月起开始下调。整体调降幅度约1/3，例如主力的Xbox Game Pass Ultimate价格从原本的1个月新台币459元（约合人民币106元）降低至新台币299元（约合人民币69元），Xbox Game Pass（PC/主机）从原本的1个月新台币320元（约合人民币74元）降低至新台币199元（约合人民币46元）。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220118/1642483395_294981.jpg" alt="Xbox台服XGP、XGPU等订阅服务降价 1个月约46元起" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
官方表示，Xbox会定期评估Xbox Game Pass 及Xbox Live Gold在各个市场当地的价格，以为各地区的会员及玩家带来最好的服务。这些价格调整会根据每个市场的情况而订定，也是微软去年底开始在一些地区进行价格调整的延续。其中Xbox Game Pass Ultimate包含Xbox Live Gold会员的权益、最先进的多人游戏网路、以及上百款适用于主机与PC的高品质游戏，更包含EA Play会员。
</p>
