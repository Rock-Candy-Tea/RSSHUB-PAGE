<div>   
<p style="text-indent:2em;">
微软对 Windows 11 中的
HDR 押注很大，并于本周发布了适用于 Windows 11 操作系统的《Windows HDR Calibration》校准应用程序。微软将很快通过
Microsoft Store 渠道发布该软件，感兴趣的朋友，届时可以体验下三种不同的测试模式。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220219/1645215954_186891.jpg" alt="微软发布适用于Windows 11的HDR校准应用程序" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
微软解释称，尽管某些型号的显示器已经过出厂校准，但新工具还是有助于确定一切是否正常。
</p>
<p style="text-indent:2em;">
“首先，Windows HDR Calibration 会尝试确定用户可见的最低一档黑暗细节水平。其次，软件会测量最高亮度下的细节可见性。然后应用程序会运行测试，以确定用户显示器的亮度。”
</p>
<p style="text-indent:2em;">
微软表示，他们听到了用户的反馈和需求，知道不少人都对这方面的体验感到沮丧，于是要求将
Xbox HDR 游戏校准应用程序引入 Windows 平台。
</p>
<p style="text-indent:2em;">
在不远的将来，你将能够通过即将推出的 Windows HDR Calibration 应用程序，来提高 HDR 显示器的色彩准确和一致性。
</p>
<p style="text-indent:2em;">
如果你你已在处于 Windows Insider Dev，或许已经注意到【系统 → 显示 → HDR 设置】中新增的【HDR
显示校准】选项。
</p>
<p style="text-indent:2em;">
不过由于该应用程序仍处于早期访问阶段，大部分用户暂时还无法获得同样的体验。
</p>          
</div>
