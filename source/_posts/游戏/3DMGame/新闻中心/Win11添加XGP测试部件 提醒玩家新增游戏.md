<p style="text-indent:2em;">
微软的首个新的
Windows 11 小部件应该可以更容易地找出哪些游戏将加入 Xbox Game Pass PC 免费游戏库。该小部件目前仅适用于
Windows Insider 测试人员。该部件旨在为 XGP 的游戏库提供一个“窗口”，让用户可以查看最近添加的内容和即将离开游戏库的游戏。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220804/1659594414_396342.png" alt="Win11添加XGP测试部件 提醒玩家新增游戏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
微软目前在
Windows 11 中的小部件选择相当有限，但
Game Pass 小部件表明微软正在尝试其他应用程序，不像天气、日历和新闻那么基础。Windows
11 最终应该支持第三方小部件，但添加
Game Pass 可能是暂时改变小部件板的好方法，可能有助于推广订阅服务。
</p>
<p style="text-indent:2em;">
除了让您看到哪些游戏即将上线和上线之外，新的
Game Pass 小部件还显示某些类别的游戏，并提供了一个通往
Xbox 应用程序的门户。微软表示，它也在努力为该小部件添加其他功能，包括个性化推荐以及让您登录
Xbox 帐户并启动最近玩过的游戏的方式。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220804/1659594378_562086.png" alt="Win11添加XGP测试部件 提醒玩家新增游戏" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
如果您是
Windows 11 测试人员，则可以通过在
PC 上打开小部件面板，然后单击用户资料旁边的加号图标来添加
Game Pass 小部件。点击游戏通行证旁边的“加号”图标，小部件应该会出现在您的面板上。
</p>
