<div>   
<p style="text-indent:2em;">
1月21日下午5点，专注于独立游戏的中国发行商Gamera Game为玩家们带来了新一期的线上发布会。在这场名为“Gamera Game Now虎年特别节目”的发布会上，包括《戴森球计划》《三伏》《水银疗养院》等在内的数十款游戏依次登场，下面就让我们来看看都有哪些值得关注的情报。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753020_627749.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
<strong>戴森球计划</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753005_358851.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
放在发布会最开头的是迎来上线一周年的《戴森球计划》，在感谢玩家支持的同时，官方以有趣的形式总结了上线一年以来游戏中的各项大数据。在昨天晚上，《戴森球计划》刚刚更新了大版本“进化的伊卡洛斯”，目前游戏正在8折促销中，感兴趣的玩家不妨打开游戏，感受一下《戴森球计划》这一年以来的变化。
</p>
<p style="text-indent:2em;">
<strong>维斯塔利亚传说外传</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753046_801037.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
《维斯塔利亚传说外传：希尔瓦比西之圣剑》是经典日式战棋游戏《维斯塔利亚传说》的续作。由叱咤日本游戏行业数十年的老将亲自操刀，叙述前作尚未完结的故事。男主角塞特将会和老搭档和新伙伴们并肩作战，揭开隐藏在维斯塔利亚大陆上的更多秘密。本作的简体中文版正在制作之中，预计在2022年内上线。
</p>
<p style="text-indent:2em;">
<strong>罪狱：soul</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753062_332929.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
《罪狱：soul》是由GHF GAME开发的克苏鲁风格惊悚冒险游戏。男主角秦选死后堕入地狱，与那些未被黑暗所吞噬的灵魂开启了艰难的求生之路。游戏采用独特的黑白手绘风格结合克苏鲁元素，描绘出一个充斥着绝望的至暗世界。本作预计将于2022年5月正式发售。
</p>
<p style="text-indent:2em;">
<strong>侠之信条</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753078_434982.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
《侠之信条》是维克多游戏开发的武侠角色扮演游戏。以架空的明朝初年为背景，武林领袖朱元璋在带领武林推翻元朝后，转而大肆屠戮武林人士。玩家将扮演一个初入江湖的小角色，从三大阵营的数十种职业中选择适合自己的方向进行发展，最终探寻到属于自己的“侠之信条”。本作将于1月26日正式发售。
</p>
<p style="text-indent:2em;">
<strong>天使之王：梅塔特隆的飞升</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753087_993055.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
由crim开发的3D动作游戏《天使之王：梅塔特隆的飞升
高清复刻版》曾经凭借一句“大丈夫，萌大奶”走红网络。游戏在正式更新官方中文版之后，让广大中国玩家也能感受到这款经典动作游戏的魅力所在。
</p>
<p style="text-indent:2em;">
<strong>6栋301房</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753095_574254.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
《6栋301房》是由帧间工作室开发的一款休闲解谜游戏，游戏通过独特的视角展现出一名老年阿尔茨海默病患者的日常生活。玩家将设身处地体验阿尔茨海默病患者所面临的困境，感受他们眼中的世界。《6栋301房》也是Gamera Game此前公布的“叶绿素计划”中的作品之一，预计将于2022年内发售。
</p>
<p style="text-indent:2em;">
<strong>最后的木头</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753109_293971.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
由国内双人团队Just Us GameLab开发的海洋生存模拟游戏《最后的木头》，将在1月27日结束EA更新到正式版——废岛探险。在新版本中，玩家将可以体验到登陆岛屿、建造设施、开采矿物、饲养动物、高科技武器、全新敌人等等诸多内容，可以在其中感受到更为紧张刺激的生存体验。
</p>
<p style="text-indent:2em;">
<strong>水银疗养院</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753116_459499.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
由成都乙钛游戏开发的像素风解谜游戏《水银疗养院》自去年公布以来便以其精美的画面风格和引人入胜的设定吸引了不少玩家的目光，本次发布会上的视频以主角之一威利的视角对游戏中的部分登场角色进行了介绍。目前游戏正在摩点上进行众筹，众筹内容包括游戏的激活码、角色纪念币、手账本等，感兴趣的玩家千万别错过了。
</p>
<p style="text-indent:2em;">
<strong>开普勒斯</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753123_845280.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
《开普勒斯》是由国内工作室TARO开发的生存沙盒游戏，最多支持4人联机。游戏给予玩家极高的自由度，通过随机生成的地图和地下城，让每局游戏都能带来不一样的体验。玩家需要采集各种素材，建造设施，生产生存的必需品，在探索和战斗中获取道具和装备，进而挑战更强大的敌人乃至BOSS。本作目前正处于抢先体验阶段，正式版有望在未来几个月内推出。
</p>
<p style="text-indent:2em;">
<strong>烟火</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753130_138297.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
由拾英工作室开发的惊悚冒险游戏《烟火》在去年初带给了无数玩家感动，也成为了近年来口碑最好的国产游戏之一。临近发售一周年之际，《烟火》的开发者和发行商也通过视频回顾了一年以来发生的点点滴滴，藉此表达对玩家的感谢之情，最后公布了《烟火》即将开启盲盒众筹的消息，并展示了人气角色陈青穗陈老师的盲盒造型。
</p>
<p style="text-indent:2em;">
<strong>三伏</strong> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220121/1642753141_684549.jpg" alt="Gamera Game Now虎年特别节目 多款游戏情报首度公开" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
整场发布会最后亮相的同样是来自拾英工作室的作品——《三伏》。这款备受期待的新作此次为我们带来了一则“精神污染”效果极强的广告片，广告介绍了一款名为“神童壹号营养液”的产品，阴间的色调和造型诡异的三眼兔子直接把效果拉满。这则广告和之前《三伏》放出过的“三眼神童”海报遥相呼应，至于这背后究竟是个怎样的故事，恐怕只要到游戏中去寻找答案了。
</p>
<p style="text-indent:2em;">
以上便是“Gamera Game Now虎年特别节目”的所有情报，如果你对其中部分作品感兴趣的话，不妨现在就将其加入愿望单。我们也期待Gamera Game在虎年为玩家们带来更多有趣好玩的独立游戏。
</p>
