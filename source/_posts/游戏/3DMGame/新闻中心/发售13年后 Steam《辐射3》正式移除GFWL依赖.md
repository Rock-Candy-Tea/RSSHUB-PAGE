<div>   
<p style="text-indent:2em;">
《<a target="_blank" href="https://www.3dmgame.com/games/fallout3/">辐射3</a>》在PC平台发布近13年后的今天，该游戏进行了更新，将Games For Windows Live从游戏中移除。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211013/1634117326_717516.jpg" alt="发售13年后 Steam《辐射3》正式移除GFWL依赖" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
B社昨天（10月12日）通过Steam宣布，他们修补了《辐射3》，使其不再依赖Games For Windows Live。除了提到“游戏不再需要Games For Windows Live，现在将会发布”以外，B社还指出《辐射3》也将停止安装Games For Windows Live依赖项，已安装的用户B社建议卸载游戏重新安装一次。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211013/1634117373_668197.jpg" alt="发售13年后 Steam《辐射3》正式移除GFWL依赖" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
《辐射3》在Steam上的难度是出了名的大。如果你试图在没有Games For Windows Live的情况下打开游戏通常会导致《辐射3》无法运行，这意味着此前的玩家需要修改游戏名称或下载Games For Windows Live文件才能游玩游戏。
</p>
<p style="text-indent:2em;">
几年前当《辐射3》在GOG上发行时，其Games For Windows Live的要求和其他一些问题都被修复了。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211013/1634117386_191181.jpg" alt="发售13年后 Steam《辐射3》正式移除GFWL依赖" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
微软的平台（《黑暗之魂》、《GTA4》等游戏的强制性平台）多年来一直不受PC游戏玩家欢迎。除了要求Steam玩家创建一个额外的帐号来玩在线游戏以外，这项服务还经常给那些网络连接不稳定的玩家带来很多问题。
</p>
