<div>   
<p style="text-indent:2em;">
国产消消乐游戏《Mirror 2: Project X》1月21日上架Steam， 提前开启抢先体验。在开启抢先体验之后，《Mirror 2: Project X》在2小时内就登上了Steam“全球热销商品”排行榜第一名，为了回馈玩家，官方将游戏的剧情模式从6章升级到8章。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220123/1642902772_811073.jpg" alt="《Mirror 2: Project X》Steam全球热销商品第1 剧情将升级" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
官方更新公告：
</p>
<p style="text-indent:2em;">
<strong><span style="color:#E53333;">感谢大家的热情支持，让《Mirror 2: Project X》仅在2小时内就登上了Steam“全球热销商品”排行榜第一名！为表答谢，我们决定将游戏本体的故事模式章节从6章追加至8章！</span></strong> 
</p>
<p style="text-indent:2em;">
本作为一次性买断制，不需要分章节购买，您现在购买的EA版游戏未来将持续更新至包含8个章节的完整版。未来定价也将保持不变（中国大陆区域为11元）。同时，免费追加服装DLC【Mirror 2: Project X - Costumes】
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220123/1642902858_168304.jpg" alt="《Mirror 2: Project X》Steam全球热销商品第1 剧情将升级" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
目前，《Mirror 2: Project X》在Steam上收获了12091篇评价，97%好评如潮，现在游戏促销价6折，国区售价6.6元。
</p>
<p style="text-indent:2em;">
《Mirror 2：Project X》是一款基于UE4引擎打造的卡通渲风格剧情向游戏。在游戏中，你是联邦研究保护局FBRP的研究员，为保护人类应对来源不明的各种超自然现象及物体的研究工作。你将走进一个位面重叠的平行宇宙世界，人类面临的不可知事物何其之多，你所做出的每个选择都将影响游戏的结局。
</p>

</p>
