<div>   
<p style="text-indent:2em;">
微软已经开始向 Insider Dev 渠道推送 Windows 11 
新的内置应用，更多内置应用的新版更新也在路上。这次已经更新的包括截图工具、计算器、邮件和日历，界面上适配新的 Windows 11 视觉风格。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210815/1629035538_370741.jpg" alt="新版Windows 11内置应用开始测试" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
Windows 11 新版截图工具集合了旧版截图和 Snap & Sketch 图片编辑功能，系统全局快捷键 WIN + SHIFT + S 
可触发截图，并且整个界面已经为深色主题适配。
</p>
<p style="text-indent:2em;">
Windows 11 计算器由 C# 重写，仍然是有标准计算、科学计算、绘图模式、单位转换等功能，界面也为深色主题适配。
</p>
<p style="text-indent:2em;">
邮件和日历应用也再次优化设计，更加贴近 Windows 11 设计风格。
</p>
<p style="text-indent:2em;">
在新的预览版 Windows 11 Build 22000.132 中，微软也推出了 Chat（Microsoft Teams）聊天集成。
</p>
