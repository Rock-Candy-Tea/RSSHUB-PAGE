<div>   
<p style="text-indent:2em;">
首席 3D 环境艺术家 Andrew Averkin 通过 NVIDIA Omniverse 和 Autodesk 3ds Max 分享童年记忆。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050373_845703.jpg" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
编辑注：本文是我们每周发布的 NVIDIA Studio 系列内容的一部分，旨在分享特邀艺术家作品及创作故事、提供创意启发和技巧，并展示 NVIDIA Studio 技术如何加速创意工作流。
</p>
<p style="text-indent:2em;">
在本周的 NVIDIA Studio创意加速中，我们将推出 4 月版 NVIDIA Studio 驱动以及热门 3D 应用（包括 UE 5、Cinema4D 和 Chaos Vantage）的优化。该驱动还支持 Blender 和 Redshift 新推出的 NVIDIA Omniverse 连接器。
</p>
<p style="text-indent:2em;">
如果数字世界的达芬奇们希望升级他们的数码画布，可以详细了解新发布的联想 ThinkPad P1 NVIDIA Studio 设计本，也可以选用现已推出的 Asus ProArt Studiobook 16、MSI Creator Pro Z16 和 Z17。
</p>
<p style="text-indent:2em;">
这些更新是 NVIDIA Studio 的创作优势的一部分：显著加速 3D 创意工作流，这对于本周的 In the NVIDIA Studio 精选创作者，即 NVIDIA 首席 3D 环境艺术家 Andrew Averkin 来说至关重要。
</p>
<p style="text-indent:2em;">
<strong>4 月版 Studio 驱动带来强大性能</strong>
</p>
<p style="text-indent:2em;">
从备受期待的 UE 5 发布开始，4 月版 NVIDIA Studio 驱动支持许多新的创意应用更新。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050373_155854.png" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
火力全开的 UE 5 Lumen 全局光照。
</p>
<p style="text-indent:2em;">
NVIDIA RTX GPU 支持 Lumen（UE5 新推出的全动态化全局光照系统），适用于软件和硬件光线追踪。配合 Nanite，开发者能够创造出全动态化全局光照下包含大量几何细节的游戏和应用。上月的 GTC 上，NVIDIA 推出了更新版 Omniverse 连接器，其功能包括从 UE 5 导出 Nanite 网格的源几何信息。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050373_658378.png" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
City Sample 是可免费下载的示例项目，展示了“The Matrix Awakens: An Unreal Engine 5 Experience”中城市场景的构建方式。
</p>
<p style="text-indent:2em;">
通过 Epic Games 制作的插件，UE 5 为大量 NVIDIA 技术提供支持。借助视图中经 RTX 加速的光线追踪和 NVIDIA DLSS，新想法的迭代和改进变得更为简单快速。对于成品，这些技术会为精美的光线追踪图形提供渲染支持，同时利用 AI 提高帧率。
</p>
<p style="text-indent:2em;">
借助 UE5 中的 NVIDIA Reflex，在 RTX GPU 上运行的即刻面市的 PC 游戏即可体验超出想象的低延迟，无需进行下载。
</p>
<p style="text-indent:2em;">
NVIDIA 实时降噪器提供实时性能，提高了艺术创作的效率。借助 RTX 全局照明，可以消除离线光照贴图和光探针烘焙的漫长等待时间。借助 RTX GPU 强大的处理能力、官方 UE 插件系列和新一代 UE5，您可以灵感飞扬，尽情创作。
</p>
<p style="text-indent:2em;">
Maxon 的 Cinema 4D 版本 S26 包含新的布料和绳索动态模拟（由 NVIDIA RTX GPU 加速），可让艺术家以更高的真实感、更快的速度为数字化对象建模。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050373_821630.png" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
其他功能包括 OpenColorIO 适配，这是一种新的摄像头和增强建模工具。
</p>
<p style="text-indent:2em;">
Chaos Vantage 借助 NVIDIA RTX GPU 专有的实时光线追踪技术，增加了法线贴图支持、将纹理模型转换为粘土模型以突出光照的新功能，以及阴影的环境光遮蔽。
</p>
<p style="text-indent:2em;">
NVIDIA Omniverse 连接器更新为实时工作流提供了新功能和选择。Blender 添加了新的 blendshape I/O 支持，以确保细节丰富的多面对象能够正确实现自动渲染和动画。此外，还有新的 USD 扩展地图，可实现大规模的电影级可视化。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050373_561115.png" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
使用 NVIDIA RTX A6000 GPU 在 Redshift 中渲染只需几分钟；仅使用 CPU 时则需要一个多小时。
</p>
<p style="text-indent:2em;">
Blender 和 Redshift 增加了基于 Hydra 的渲染。艺术家现在可以在所有 Omniverse 应用的视图中使用他们喜欢的渲染器。
</p>
<p style="text-indent:2em;">
<strong>新版 Studio 驱动，全新 Studio 设计本</strong>
</p>
<p style="text-indent:2em;">
4 月，NVIDIA Studio 系列还将推出新的联想移动工作站，另外还有三款产品上市。
</p>
<p style="text-indent:2em;">
联想 ThinkPad P1 采用轻薄设计、16 英寸面板，可在 GeForce RTX 和 NVIDIA RTX 专业显卡的助力下实现出色性能，并且最高可配新的 NVIDIA RTX A5500 笔记本电脑 GPU。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050373_451108.png" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
Dolby Vision、HDR400 和 165Hz 显示器助力联想 ThinkPad P1 成为适合创作者的出色设备。
</p>
<p style="text-indent:2em;">
来自其他合作伙伴的 Studio 设计本包括近期发布的 Asus ProArt Studiobook 16、MSI Creator Pro Z16 和 Z17，所有这些产品现在均可购买。
</p>
<p style="text-indent:2em;">
<strong>与 Andrew Averkin 一起追溯回忆</strong>
</p>
<p style="text-indent:2em;">
Andrew Averkin 是乌克兰的 NVIDIA 首席 3D 环境艺术家。他擅长创建逼真的 3D 场景，专注于旨在唤起温暖怀旧情绪的真实感。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050374_523780.jpg" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
Andrew Averkin 的作品“Boyhood”。
</p>
<p style="text-indent:2em;">
Averkin 以共情为主导，旨在创造 “展现创作激情和艺术想象的温暖的艺术作品”。
</p>
<p style="text-indent:2em;">
他使用 NVIDIA Omniverse Create 应用和 Autodesk 3ds Max（由 NVIDIA RTX A6000 GPU 加速）创作了 When We Were Kids。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050374_879291.jpg" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
多个光源可提供能增加真实感的深度和阴影。
</p>
<p style="text-indent:2em;">
在此处，Averkin 重现了数码时代之前玩玩具的场景，从而激发观众充分的想象力。
</p>
<p style="text-indent:2em;">
Averkin 最初在 Autodesk 3ds Max 中为 When We Were Kids 进行建模。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050379_745147.jpg" title="1651043838848169.jpg" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
场景中的特写镜头展现了对细节的精准把握。
</p>
<p style="text-indent:2em;">
Autodesk 3ds Max 中的 RTX GPU 加速视图和 RTX 加速 AI 降噪使得文件较大时也能实现流畅交互。
</p>
<p style="text-indent:2em;">
“我记得我们还是孩子时……这就是这个故事在脑
</p>
<p style="text-indent:2em;">
海中最初呈现的模样。我使用了三个 RTX A6000 
</p>
<p style="text-indent:2em;">
GPU 和 UE5 降噪器，将一张图像的渲染时间缩短
</p>
<p style="text-indent:2em;">
到几秒钟，助力我完全专注于艺术。”
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050379_102012.jpg" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
借助 Omniverse Create，用户可以实时搭建物理属性准确的复杂模拟和 3D 场景。
</p>
<p style="text-indent:2em;">
然后，Averkin 将 When We Were Kids 导入 Omniverse Create，对其 3D 场景进行实时照明、模拟和渲染。
</p>
<p style="text-indent:2em;">
Averkin 表示，“大部分素材取自 Epic ，我的主要目标是处理光照场景、构图和氛围。”
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050379_126034.jpg" title="1651043838183043.jpg" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
Averkin 专注于怀旧和逼真效果，帮助观看者感受昔日的生活状态。
</p>
<p style="text-indent:2em;">
在 Omniverse Create 中，Averkin 为艺术作品使用专用的照明工具、自动更新原始的 Autodesk 3ds Max 文件（无需进行混乱且耗时的文件转换和上传），最后使用 RTX GPU 快速渲染最终文件。
</p>
<p style="text-indent:2em;">
此前，Averkin 曾在 Axis Animation、Blur Studio 等公司工作。要查看他的作品集和喜欢的项目，请访问 ArtStation。
</p>
<p style="text-indent:2em;">
深入了解 NVIDIA Studio 创意加速
</p>
<p style="text-indent:2em;">
我们为创作者提供大量资源，帮助他们详细了解本周精选艺术家使用的应用，以及 RTX 和 GeForce RTX GPU 如何加速创意工作流。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/xiaz/20220427/1651050380_151529.jpg" referrerpolicy="no-referrer">
</p>

了解“The Storyteller”的幕后花絮，该项目在 Omniverse 中创建，展示了令人惊叹、逼真的复古风格的作家房间。
</p>
