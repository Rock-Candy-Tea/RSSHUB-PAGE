<div>   
<p style="text-indent:2em;">
根据IGN韩国爆料，SE还有超过六款游戏目前正计划登陆Xbox Game Pass，登陆时间正在计划中，而且这几乎是业界公开的秘密。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210327/1616848552_441317.jpg" alt="传SE有超过6款游戏将加入到Xbox Game Pass" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
最近《最终幻想12》和《八方旅人》加入了Xbox Game 
Pass，而SE发行的科幻射击游戏《Outriders》也已确认在首发当天加入XGP（主机/云端）。
</p>
<p style="text-indent:2em;">
此前，已有消息称育碧Plus也会加入到Xbox Game Pass，方式类似于早前的EA Play加入到Xbox Game Pass终极版中。
</p>
