<p style="text-indent:2em;">
外国非盈利人工智能研发组织 OpenAI 旗下编程产品 Codex 已经正式开放封闭 Beta 测试。这款工具旨在将自然语言（英语、中文等）翻译为编程语言。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210922/1632295281_349773.png" alt="OpenAI最新技术Codex 不会代码也能编游戏" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
OpenAI Codex beta 测试目前只能通过在线等待列表参与。它是一个简单的网络工具，具有三个窗口：一个用于输入命令，一个显示由这些命令生成的代码，一个显示代码的作用。理论上，您可以将 Codex 用于十多种编码语言的各种任务，但最酷的用途是使用少量自然语言指令编写简单的 Javascript 游戏。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210922/1632295333_728547.png" alt="OpenAI最新技术Codex 不会代码也能编游戏" referrerpolicy="no-referrer"> 
</p>

</p>
<p style="text-indent:2em;">
该工具可以轻松处理简单的命令，像是“创建一个黑色圆形”；它也可以正确解释有前后关系的命令，例如“将黑色圆形变小”；它甚至可以可以理解多个步骤的指令，比如“如果矩形和圆形重叠，让圆形往反方向移动”。
</p>
<p style="text-indent:2em;">
OpenAI 还用一个飞船小游戏对 Codex 的 Javascript 功能进行了官方演示，在演示过程中没有进行任何人工干预。仅仅把分钟的时间内，演示人员通过 Codex 成功编写了许多游戏机制，包括得分、教程、目标以及飞船加速功能等。
</p>

<p style="text-indent:2em;">
目前看来，Codex 距离实际使用还有很长的路要走。但是硅谷的投资者对“低代码”或“无代码”工具越来越感兴趣，这些工具旨在使编程更快、更简单，让那些不精通任何编程语言的人也能使用。
</p>
