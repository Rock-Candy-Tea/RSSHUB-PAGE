<div>   
<p style="text-indent:2em;">
微软于今日宣布将在将首次在印度尼西亚、马来西亚、菲律宾、泰国和越南推出PC Game Pass服务，目前在者五个东南亚的国家已经上线了预览版，当地的玩家可以开始注册PC Game Pass预览版账号，预览版包含100多种高品质游戏的库，其中包括 Xbox Game Studios旗下首发的新游戏、B社的经典游戏，以及每月EA Play会员低价资格等，而且还将不断地添加新游戏。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220329/1648537237_283996.png" alt="微软将在东南亚推出PC Game Pass服务 预览版已上线" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
只需几分钟的注册就可加入PC Game Pass预览版的体验当中，该服务将支持当地支付系统，玩家在预览期间试用仅需花费1马来西亚林吉特（约为人民币1.5元）。由于这是在该地区引入PC Game Pass的第一步，参与者可以测试并分享他们的反馈，以便官方可以改进整个服务。微软表示将在今年晚些时候在印度尼西亚、马来西亚、菲律宾、泰国和越南地区正式推出PC Game Pass服务，届时会分享相关的具体信息。
</p>
<p style="text-indent:2em;">
Xbox亚洲业务负责人Jeremy Hinton表示，公司的使命很简单，就是将游戏的乐趣和游戏社区带给地球上的每个人。将PC Game Pass带到东南亚的新国家是团队实现这一目标的重要组成部分，团队正努力接触到这些数十亿的玩家。
</p>
