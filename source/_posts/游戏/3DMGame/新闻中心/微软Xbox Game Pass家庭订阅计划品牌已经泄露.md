<div>   
<p style="text-indent:2em;">
本月早些时候，微软开始测试其受欢迎的Xbox Game Pass服务的家庭计划。新订阅计划允许多达五个成员通过他们的个人微软账户充分享受Xbox Game Pass Ultimate服务。对于有多个使用Xbox作为主要平台的游戏玩家的家庭来说，这可能是一个非常诱人的订阅服务。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220828/1661641203_392012.jpg" alt="微软Xbox Game Pass家庭订阅计划品牌已经泄露" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
现在，该服务的官方品牌标识已泄露。意大利网站Aggiornamenti Lumia日前放出了其标识图片，其中写道“好友与家人”。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220828/1661641219_246502.jpg" alt="微软Xbox Game Pass家庭订阅计划品牌已经泄露" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
有趣的是，微软并没有将其作为一个家庭计划来推广，至少在品牌上是这样，也许不仅仅是家庭，微软希望鼓励玩家们也可在好友之间分享。
</p>
<p style="text-indent:2em;">
关于费用，微软还没有正式宣布。不过在对爱尔兰和哥伦比亚地区Xbox Insiders的测试中，官方提醒体验者，他们的Xbox Game Pass Ultimate订阅将被转换为新的订阅层级。一个月的Xbox
Game Pass Ultimate会员费为15美元，但它会被转换为18个月的新好友和家庭层级，大致相当于相当于后者的25美元/月。爱尔兰的用户能够找到21.99欧元/月的参考价格。
</p>
