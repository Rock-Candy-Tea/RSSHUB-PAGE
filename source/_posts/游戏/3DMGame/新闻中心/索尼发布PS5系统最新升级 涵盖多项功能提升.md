<div>   
<p style="text-indent:2em;">
4月27日，索尼发布了PS5主机系统固件升级。这次升级后，PS5系统版本来到了21.01-03.00.00，提升多项系统性能，升级文件约900MB，下面一起来看看官方的更新日志：
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210204/1612392887_864145.jpeg" alt="索尼发布PS5系统最新升级 涵盖多项功能提升" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
版本：21.01-03.00.00
</p>
<ul style="text-indent:2em;">
<li>
<p>
现在，您可以在 USB 扩展存储容量中存储 PS5 游戏。
</p>
<ul style="text-indent:2em;">
<li>
<p>
要从主机存储器中将 PS5 游戏移动到 USB 扩展存储容量，请转到主页屏幕或游戏库，打开您要移动的 PS5 游戏的选项菜单，然后选择移动到 USB 扩展存储容量。
</p>
</li>
<li>
<p>
要玩 USB 扩展存储容量中的 PS5 游戏，请转到游戏库，并从选项菜单中将游戏复制到主机存储器。复制游戏比重新下载游戏快得多。如果 USB 扩展存储容量上的游戏有追加内容或已更新，这也比从光盘上重新安装更快。
</p>
</li>
</ul>
</li>
<li>
<p>
您现在可以使用 PS4 与玩家尽享“分享游玩”。在 PS4 上使用分享游玩邀请您的朋友观看您玩游戏，甚至是将控制器移交给他们。
</p>
</li>
<li>
<p>
我们向辅助功能中添加了缩放功能。使用缩放功能放大屏幕上的内容。转到设定  > 协助工具  > 显示  > 缩放 以启用此功能。
</p>
</li>
<li>
<p>
我们在控制中心的声音下添加了禁用游戏聊天语音 。如果您打开此设定，别人将无法听到您的声音，您也无法听到游戏聊天中其他玩家的声音。此设定不适用于派对聊天语音。
</p>
</li>
<li>
<p>
在 Game Base 中，我们更新了以下内容： 
</p>
<ul style="text-indent:2em;">
<li>
<p>
您可以更轻松地访问最近的派对和查看在线的朋友。使用 R1 和 L1 键快速地在各选项卡之间移动。
</p>
</li>
<li>
<p>
您可以为所在的每个派对打开或关闭通知。
</p>
</li>
<li>
<p>
您现在可以在派对语音聊天和某些游戏的游戏聊天中调整其他玩家的声音音量。从控制中心的语音聊天功能卡中选择要调整其音量的玩家。 
</p>
</li>
</ul>
</li>
<li>
<p>
在简介中，我们更新了以下内容：
</p>
<ul style="text-indent:2em;">
<li>
<p>
您现在可以比以往更轻松地分享您最喜爱的游戏时刻。在“共享”选项卡下，选择转到媒体库 ，然后选择您最喜欢的媒体进行分享。
</p>
</li>
<li>
<p>
我们在“共享”选项卡下将视频和广播分开，以便浏览。
</p>
</li>
</ul>
</li>
<li>
<p>
您现在可以通过进入设定  > 保存数据和游戏/应用程序设定  > 管理关注的游戏来管理所有关注的游戏。这可让您不必导航到单个新闻帖子或游戏中心就可以取消关注游戏。 
</p>
</li>
<li>
<p>
对于支持 HDMI 设备关联功能的显示器，现在可以分别启用或禁用一触式播放和关闭关联。转到设定  > 系统  > HDMI  以更改这些设定。
</p>
<ul style="text-indent:2em;">
<li>
<p>
在启用一触式播放后，打开 PS5 主机也会打开已连接的显示器。 
</p>
</li>
<li>
<p>
在启用关闭电源链接后，关闭连接的显示器将使 PS5 主机处于待命模式中。
</p>
</li>
</ul>
</li>
</ul>
<ul style="text-indent:2em;">
<li>
<p>
现在，您可以在视听者限制设定中为尚未加入或已登录 PlayStation Network 的 PS5 主机用户和访客设定国家或地区。
</p>
<ul>
<li>
<p>
您选择的国家或地区将决定对 PS5 游戏和应用程序应用何种等级系统。要更改用户的国家或地区，请转到设定  > 家庭和视听者限制，选择用户，然后选择游戏和应用程序的年龄级别下的 PS5 。
</p>
</li>
<li>
<p>
您还可以在 PS5 主机上为新用户和访客预先设定国家或地区。为此，请转到设定  > 家庭和视听者限制  > PS5 主机限制 > 新用户视听者限制  > PS5。
</p>
</li>
</ul>
</li>
<li>
<p>
现在，您可以在游戏库中隐藏游戏。如果您隐藏游戏，则该游戏不会显示在您的收藏选项卡下。
</p>
<ul>
<li>
<p>
要隐藏游戏，请转到您的收藏 选项卡，打开游戏选项菜单，然后选择隐藏。
</p>
</li>
<li>
<p>
要查看隐藏的游戏，请转到您的收藏 选项卡，打开排列及筛选 菜单，然后选择隐藏的项目。
</p>
</li>
<li>
<p>
要取消隐藏游戏，请打开隐藏的项目 筛选器，打开您要取消隐藏的游戏的选项菜单，然后选择取消隐藏。
</p>
</li>
<li>
<p>
您在 PS4 主机上隐藏的游戏也会在 PS5 主机上隐藏。
</p>
</li>
</ul>
</li>
</ul>
<p style="text-indent:2em;">
其他更新的功能
</p>
<ul style="text-indent:2em;">
<li>
<p>
我们添加了以下功能以便进行保存数据的管理：
</p>
<ul>
<li>
<p>
在您开始或恢复 PS5 游戏时，您的主机会检查云存储器中较新的保存数据。如果您在云存储器中有更近的保存数据，则可以选择同步数据。
</p>
</li>
<li>
<p>
现在，直接从主页屏幕管理保存数据更轻松。当您打开游戏的选项菜单时，现在可以看到上传/下载保存数据 和检查保存数据的同步状态。
</p>
</li>
</ul>
</li>
<li>
<p>
现在，我们支持更多的表情符号。 
</p>
</li>
<li>
<p>
在奖杯中，我们更新了以下内容：
</p>
<ul>
<li>
<p>
现在，您可以访问在赢得奖杯时拍摄的画面截图和视频。选择已获得的奖杯，以查看相关媒体。
</p>
</li>
<li>
<p>
现在，您可以选择要保存奖杯画面截图和视频的奖杯等级。要更改这些设定，请转到设定  > 截取画面和播放  > 奖杯。
</p>
</li>
<li>
<p>
现在，当您选择隐藏的奖杯时，您可以按下方形键，快速显示隐藏的信息。
</p>
</li>
<li>
<p>
现在，您可以通过在选项菜单中选择转到奖杯 直接从奖杯功能卡转到游戏的奖杯列表。
</p>
</li>
<li>
<p>
在为奖杯选择排列顺序后，您的奖杯会在所有游戏中保持该排列顺序。
</p>
</li>
<li>
<p>
现在，您可以在奖杯 屏幕上按下方形键查看奖杯统计。
</p>
</li>
<li>
<p>
现在，当您按珍贵度或获得日期排序奖杯时，奖杯功能卡将显示珍贵度或获得日期信息。
</p>
</li>
</ul>
</li>
<li>
<p>
如果您安装此升级，则会关闭协助工具设定中的启用聊天转录 。如果您要使用聊天转录功能，请再将此设定打开。
</p>
</li>
<li>
<p>
在功能卡的多任务特征中，我们升级了以下内容：
</p>
<ul>
<li>
<p>
现在，当您将注意力集中在功能卡上的多任务图标时，您将看到有关可用的多任务模式的提示：画中画、侧边固定或两者兼有。
</p>
</li>
<li>
<p>
我们添加了一个按键指南，向您展示如何与当前处于多任务模式的功能卡交互。
</p>
</li>
</ul>
</li>
<li>
<p>
在控制中心，即使正在充电，您也可以看到控制器的剩余电量。
</p>
</li>
<li>
<p>
在游戏或定期服务中心里，您将看到当前可用的促销优惠，包括 PlayStation Plus 和 EA Play 用户的优惠。要查看未显示在中心的选项，请按下 ...（更多选项） 键。
</p>
</li>
<li>
<p>
要玩您通过在隐私设定中选择对其他玩家隐藏您的游戏 隐藏的游戏时，其他玩家不会再在您的在线状态旁边看到有关您当前正在玩的游戏的信息。
</p>
</li>
<li>
<p>
某些可以显示 1080p/120 Hz 的 PC 监控器现在支持 120 Hz 的游戏显示。如果在 120 Hz 下查看游戏时遇到问题，请转到设定  > 屏幕和视频  > 视频输出，然后关闭启用 120 Hz 输出。
</p>
</li>
<li>
<p>
我们增加了一个选项，在使用不支持 HDR 的游戏或应用程序时，可以将视频输出切换到非 HDR。要更改此设定，请转到设定  > 屏幕和视频  > 视频输出  > HDR。
</p>
</li>
</ul>
<ul style="text-indent:2em;">
<li>
<p>
我们对系统软件进行了升级，以提高稳定性和性能。
</p>
</li>
</ul>
