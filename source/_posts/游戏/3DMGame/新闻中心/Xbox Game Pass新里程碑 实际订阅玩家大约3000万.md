<div>   
<p style="text-indent:2em;">
微软一直在努力推广他们的Xbox Game 
Pass服务，今年1月，微软官方公布的订阅数据为1800万人，但从这之后，XGP实际的订阅人数肯定会更高。那么现在有多少呢？看起来，XGP的订阅人数已经达到了一个非常不错的里程碑。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210930/1632961166_335949.jpg" alt="Xbox Game Pass新里程碑 实际订阅玩家大约3000万" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
在最近的雅虎金融“fireside chat”直播节目中，Xbox老大Phil Spencer和2K Games的老版Strauss 
Zelnick进行了互动。后者说漏嘴，称Xbox Game 
Pass现在有大约3000万订阅玩家。但狡猾的Spencer并没有上钩，只是说他们今年1月公布的数据是1800万，同时也没有否定Zelnick说的数据。
</p>
<p style="text-indent:2em;">
微软最近还下调了<a href="https://www.3dmgame.com/news/202109/3824715.html" target="_blank">Xbox Game Pass在中国香港，智利和以色列三地的价格</a>，随着价格的调整，相信会有越来越多的玩家加入Xbox Game Pass。当然决定Xbox Game Pass数量增长的关键还是在于每月新增的游戏阵容。最近SE大作《漫威复仇者联盟》<a href="https://www.3dmgame.com/news/202109/3824700.html" target="_blank">官宣加入Xbox Game Pass</a>。
</p>
