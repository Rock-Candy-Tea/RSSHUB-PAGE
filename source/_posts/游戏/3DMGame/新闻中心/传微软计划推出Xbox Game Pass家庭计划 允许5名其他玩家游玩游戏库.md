<p style="text-indent:2em;">
据微软系媒体Windows Central报道，Xbox Game Pass将迎来一个玩家等待许久的“家庭计划（Family Plan）”。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220401/1648768628_909742.jpg" alt="传微软计划推出Xbox Game Pass家庭计划 允许5名其他玩家游玩游戏库" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
尽管微软已经为Xbox Live金会员提供了多年的家庭计划，但Xbox Game 
Pass一直是单一账号订阅。不过即将到来的家庭计划将改变这一切。家族计划将允许XGP的订阅者在多个主机和家庭中共享该服务。
</p>
<p style="text-indent:2em;">
报道指出一个更高级别的Xbox Game 
Pass将允许最多其他5名玩家游玩XGP的完整游戏库，前提是这些玩家和主账号持有人是居住在同一个国家。这一新计划有望使用微软现有的家庭账号系统，该系统被用于Office 
365等服务。
</p>
<p style="text-indent:2em;">
目前尚不清楚Xbox Game Pass和PC Game Pass是否有不同的计划，另外我们也不清楚价格。当然家庭计划肯定要比个人订阅更贵。
</p>
<p style="text-indent:2em;">
目前XGP月费是9.99美元，玩家可以在首发当天游玩Xbox第一方游戏以及合作的第三方游戏。Xbox Game Pass Ultimate整合了Xbox 
Live金会员，月费14.99美元，给了玩家游玩多人游戏的权限以及享受每月的Games With Gold免费游戏。
</p>
