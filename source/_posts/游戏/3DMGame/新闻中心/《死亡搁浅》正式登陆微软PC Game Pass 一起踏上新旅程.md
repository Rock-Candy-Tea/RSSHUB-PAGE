<p style="text-indent:2em;">
全球发行商505
Games与小岛工作室联合宣布，《<a target="_blank" href="https://www.3dmgame.com/games/deathstranding/">死亡搁浅</a>》即将于8月23日正式登陆微软的PC 版Game Pass。小岛秀夫先生监制的《死亡搁浅》广受好评，是一部为玩家带来颠覆性玩法的史诗级作品，本次正式登陆微软游戏生态系统，以供订阅了PC版Xbox Game Pass的玩家体验。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220819/1660901777_734010.png" alt="《死亡搁浅》正式登陆微软PC Game Pass 一起踏上新旅程" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
《死亡搁浅》由著名演员诺曼·瑞杜斯（Norman Reedus）饰演山姆·波特·布里吉斯（Sam "Porter" Bridges），玩家将穿越一系列被称为 "死亡搁浅" 的超自然现象所破坏的美国荒芜区，向西行进以连接那些在人类几近灭绝后的幸存者。《死亡搁浅》通过充满感情的记叙形式，讲述了一个坚持不懈冒险的故事，参与微软PC版 Game pass的订阅的玩家可直接进行游玩。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220819/1660901787_676960.png" alt="《死亡搁浅》正式登陆微软PC Game Pass 一起踏上新旅程" referrerpolicy="no-referrer"> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220819/1660901792_906856.png" alt="《死亡搁浅》正式登陆微软PC Game Pass 一起踏上新旅程" referrerpolicy="no-referrer"> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220819/1660901798_108220.png" alt="《死亡搁浅》正式登陆微软PC Game Pass 一起踏上新旅程" referrerpolicy="no-referrer"> 
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220819/1660901803_986016.png" alt="《死亡搁浅》正式登陆微软PC Game Pass 一起踏上新旅程" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
玩家可通过独特的Social
Strand System（社会链系统）与世界各地的人联结，贡献来之不易的资源，改善周遭的环境以及其他玩家的游戏世界。富有战略性地放置高速公路、空中滑索、安全屋和其他建筑，使得各位玩家更易于穿越广袤大地，并获得赞赏。在《死亡搁浅》中玩家可以互相为对方的建筑进行点赞，令各位派送员更加团结。
</p>
