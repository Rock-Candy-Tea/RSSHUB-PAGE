<div>   
<p style="text-indent:2em;">
这个名为PrintNightmare的漏洞在本月月初被人发现，安全研究人员意外发布PoC代码后，该漏洞才于上周被披露，黑客可以利用该漏洞以管理员权限访问并执行系统中的任意代码，目前微软公司已确认该漏洞为严重危险等级。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210707/1625651476_987705.png" alt="微软发布紧急Windows更新 修复PrintNightmare漏洞 " referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
微软官方目前已开始推出Windows崩溃修复程序来解决PrintSpooler服务中出现的严重错误。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210707/1625651488_754882.jpg" alt="微软发布紧急Windows更新 修复PrintNightmare漏洞 " referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
由于Windows系统默认运行打印功能，微软被迫为所有操作系统分别发布各自的修复补丁。到目前为止，已经发布的修复补丁涵盖的系统有Windows Server 2019、Windows Server 2012 R2、Windows Server 2008、Windows 8.1、Windows RT8.1以及大多数版本的Windows10。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210707/1625651499_263041.jpg" alt="微软发布紧急Windows更新 修复PrintNightmare漏洞 " referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
Windows Server 2012、Windows Server 2016 和 Windows 10 版本 1607 的安全更新即将推出。
</p>
