<div>   
<p style="text-indent:2em;">
《<a target="_blank" href="https://www.3dmgame.com/games/minecraft/">我的世界</a>》今天（11 月 3 日）终于正式登陆了 PC 平台 XGP，同时游戏原始 Java 版本和 Win 10/11 基岩版都将整合进统一的《我的世界》启动器。因为 XGP 订阅服务的运作模式，这也意味着玩家只需要支付一份的价格就可以购买两个不同版本的游戏。而同时这也将是《我的世界》未来的运作模式。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211103/1635907459_908537.jpg" alt="《我的世界》登陆XGP 终于推出各版本PC整合版" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
Mojang 项目经理 Gama Aguilar-Gamez 表示这样做就可以消除 PC 玩家的选择困难：“你想要玩《我的世界》？太棒了，现在买一份《我的世界》就可以同时获得两个版本！”
</p>
<p style="text-indent:2em;">
从今天起，通过 PC 版 XGP 下载《我的世界》的玩家将通过整合版的启动器进行游戏，该启动器可以同时运行 Java 和基岩版。Gama 表示：“这是让该启动器从过去的 Java 启动器转变为《我的世界》启动器的第一步。我们希望做到当玩家表示想要玩《我的世界》并考虑在 PC 上游玩时，不会感到困惑或难以确定他们需要哪个版本的游戏。”
</p>
<p style="text-indent:2em;">
同时 Gama 还说道：“我们不鼓励玩家购买两个版本。如果您真的同时想要玩两个版本，请尝试 PC 版的 XGP。”
</p>
<p style="text-indent:2em;">
现在微软商店上已经有了全新免费下载的《我的世界》启动器。Mojang 表示任何人都可以免费下载，但是玩家必须要拥有想要游玩的版本，或者成为 XGP 订阅者才能游玩对应版本的游戏。
</p>
