<div>   
<p style="text-indent:2em;">
今天Xbox Game Pass官方推特发表推文，附图中囊括了280个目前已加入XGP的游戏角色名。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210622/1624330313_961213.png" alt="XGP官推暗示《审判之眼》即将加入Xbox Game Pass" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
但是紧随其后，官推又发出新图，在增加了黑曜石工作室评论中要求的《Grounded》角色“BURG.L”同时，划去了一个角色的名字。这位叫作“Takayuki Yagami”的角色正是《审判之眼》的男主角——八神隆之。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210622/1624330323_637015.png" alt="XGP官推暗示《审判之眼》即将加入Xbox Game Pass" referrerpolicy="no-referrer">
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210622/1624330607_668135.jpg" alt="XGP官推暗示《审判之眼》即将加入Xbox Game Pass" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
大量的角色名使得大多数没注意看的玩家刚开始还没有意识到该角色也位列其中，官推特意划去反而引起了玩家们激烈的讨论，认为这是官方在暗示《审判之眼》即将加入XGP。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210622/1624330342_637152.png" alt="XGP官推暗示《审判之眼》即将加入Xbox Game Pass" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
也有玩家大呼亏了，作为《如龙》系列粉丝，他才购买《审判之眼》、《如龙7》没多久，《如龙7》已经加入XGP了。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210622/1624330353_678240.png" alt="XGP官推暗示《审判之眼》即将加入Xbox Game Pass" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
也有玩家说：“不要担心，我们还有另外280个理由去玩Xbox游戏。”
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210622/1624331116_598252.png" alt="XGP官推暗示《审判之眼》即将加入Xbox Game Pass" referrerpolicy="no-referrer">
</p>
