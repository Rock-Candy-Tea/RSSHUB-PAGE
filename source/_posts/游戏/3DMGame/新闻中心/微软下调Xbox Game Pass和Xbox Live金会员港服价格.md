<div>   
<p style="text-indent:2em;">
微软将下调Xbox Game Pass终极版，Xbox Game Pass（主机/PC）以及Xbox 
Live金会员在中国香港，智利和以色列三地的价格。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210929/1632879202_918534.jpg" alt="微软下调Xbox Game Pass和Xbox Live金会员港服价格" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
微软表示他们会定期评估当地的服务价格，以最好地服务每个国家的客户，价格调整是根据当地市场情况进行的。
</p>
<p style="text-indent:2em;">
此外，微软还给外媒VGC提供了这三个地区当前和新的价格对比，将于9月29日（今天）通知给玩家。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210929/1632879233_863068.jpg" alt="微软下调Xbox Game Pass和Xbox Live金会员港服价格" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
图表显示，Xbox 
Live金会员月费价格从79港币下调至59，半年费用从309下调至229，年费则是从459港币下调至349港币（接近之前的半年费用）。而Xbox Game 
Pass PC端费用更加便宜，月费从39降至29。
</p>
<p style="text-indent:2em;">
早在今年6月，微软就暗示过Xbox Game Pass价格下调的可能。当时Xbox服务和平台负责人Liz Hamren表示：
</p>
<p style="text-indent:2em;">
“我们需要创新，将我们的游戏和服务带给世界各地更多的人。我们正在研究如何为Xbox Game 
Pass引入新的订阅报价。重点是，我们正在创造性地思考如何将游戏的乐趣和社区传递给地球上的所有人，跨越设备、地理位置和财务现实”。
</p>
<p style="text-indent:2em;">
Xobx Games Pass 
2017年6月上线，已经成为微软游戏业务的核心，截止到2021年1月，吸引了超过1800万订阅用户。根据外媒VentureBeat，XGP服务现在将近2000万订阅用户，但距离盈利还很遥远。
</p>
