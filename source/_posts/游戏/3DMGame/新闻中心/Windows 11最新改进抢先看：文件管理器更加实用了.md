<div>   
<p style="text-indent:2em;">
Windows 11在最新的Build 22563版本中偷偷上线不少新功能。其中，文件管理器功能变得更加实用。
</p>
<p style="text-indent:2em;">
在Windows 
11最新预览版中，在文件管理器中增加了使用右键菜单将某个文件固定的功能。而在此前，文件管理器只支持固定文件夹。一旦你固定住一个文件，你就能在快速访问中的“固定住的文件”部分看到它，如下面的截图所示：
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220226/1645864334_346524.png" alt="Windows 11最新改进抢先看：文件管理器更加实用了" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
只是有点遗憾的是，固定的文件无法出现在左侧的快速访问侧边栏中，这有点令人沮丧。
</p>
<p style="text-indent:2em;">
Windows 11 Build 22557 
版本中开始引入的一项新功能，是能够看到文件夹中项目的预览。当你将文件夹视图设置为除列表或详细信息之外的任何内容时，就会出现这种情况。上面可以看到一个带有超大图标的例子。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220226/1645864340_752730.png" alt="Windows 11最新改进抢先看：文件管理器更加实用了" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
文件夹预览是以前所有版本的 Windows 中都有的一个方便功能，它是许多人每天都在使用的东西，所以它被 Windows 11 
删除的事实激怒了许多用户。从 Windows 11 Build 22557 开始，现在可以在资源管理器中预览文件夹的文件，而无需打开该文件夹。
</p>
<p style="text-indent:2em;">
在昨天发布的 Windows 11 Build 22563 中，文件管理器还引入了其他一些后台改进。首先是快速访问搜索现在包括来自 
OneDrive、下载和任何索引位置的内容。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220226/1645864347_587755.jpg" alt="Windows 11最新改进抢先看：文件管理器更加实用了" referrerpolicy="no-referrer">
</p>
