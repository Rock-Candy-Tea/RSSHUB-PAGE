<div>   
<p style="text-indent:2em;">
《船舶墓地模拟器（Ship Graveyard 
Simulator）》<a href="https://store.steampowered.com/app/1286880/Ship_Graveyard_Simulator/" target="_blank">现已在Steam发售</a>，国区定价50元，目前发售特惠，售价45元（降价10%）。游戏自带简体中文，获得玩家“多半好评”，好评率70%。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211111/1636637356_423410.jpg" alt="《船舶墓地模拟器》Steam现已发售 获“多半好评”" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
“欢迎来到世界上最大的船舶墓地。放眼望去，绵延数十公里，数以百计的废弃船只布满海岸线。这是数千万吨的钢材。选择最有价值的废船，然后展开你的冒险。成为拆船工人的一员，体验全世界最危险的工作之一。切割、拆除、焊接！一点一点的拆除整艘船，然后出售拆除的部位来升级你的拆船设备。”
</p>
<h3 style="text-indent:2em;">
关于这款游戏：
</h3>
<p style="text-indent:2em;">
把握机会探索隐藏在这些巨大船体内的神秘空间，探索船上的每一层甲板寻找高价值物品。
</p>
<p style="text-indent:2em;">
拆船工人的工作是世界上最危险的工作之一，作业人员必须使用简陋的工具进执行拆除工作，而且还有可能接触有毒物质。
</p>
<p style="text-indent:2em;">
出售你在船上拆除的物品或是用来以物易物。
</p>
<p style="text-indent:2em;">
注意自身安全，严重事故随时可能发生，甚至会有死亡的可能性。
</p>
<h3 style="text-indent:2em;">
系统需求
</h3>
<p style="text-indent:2em;">
<strong>最低配置:</strong> 
</p>
<p style="text-indent:2em;">
需要 64 位处理器和操作系统
</p>
<p style="text-indent:2em;">
操作系统: Windows 7 64 Bit / Windows 8 64 Bit / Windows 10 64 Bit
</p>
<p style="text-indent:2em;">
处理器: Intel Core i3 3.0 GHz
</p>
<p style="text-indent:2em;">
内存: 8 GB RAM
</p>
<p style="text-indent:2em;">
显卡: NVidia GeForce GTX 780
</p>
<p style="text-indent:2em;">
DirectX 版本: 11
</p>
<p style="text-indent:2em;">
存储空间: 需要 20 GB 可用空间
</p>
<p style="text-indent:2em;">
<strong>推荐配置:</strong> 
</p>
<p style="text-indent:2em;">
需要 64 位处理器和操作系统
</p>
<p style="text-indent:2em;">
操作系统: Windows 7 64 Bit / Windows 8 64 Bit / Windows 10 64 Bit
</p>
<p style="text-indent:2em;">
处理器: Intel Core i5 3.4 GHz
</p>
<p style="text-indent:2em;">
内存: 12 GB RAM
</p>
<p style="text-indent:2em;">
显卡: NVidia GeForce GTX 970
</p>
<p style="text-indent:2em;">
DirectX 版本: 11
</p>
<p style="text-indent:2em;">
存储空间: 需要 20 GB 可用空间
</p>
