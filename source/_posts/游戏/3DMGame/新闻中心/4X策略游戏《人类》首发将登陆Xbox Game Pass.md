<div>   
<p style="text-indent:2em;">
Amplitude和世嘉打造的4X策略游戏《<a target="_blank" href="https://www.3dmgame.com/games/humankind/">人类</a>》首发当天将登陆PC和Xbox Game Pass。
</p>
<p style="text-indent:2em;">
《人类》是Amplitude 
Studios倾力打造的宏伟巨作，它是一款<a target="_blank" href="https://www.3dmgame.com/tag/lishi_1/">历史</a>战略游戏，你能够在其中重新书写人类的整个故事——它汇集了文化、历史和价值观，能够让你打造一个专属于自己的独特文明。你会如何缔造《人类》的未来?
</p>
<p style="text-align:center;text-indent:2em;">
<img src="https://img.3dmgame.com/uploads/images/news/20210809/1628515192_777214.jpg" alt="4X策略游戏《人类》首发将登陆Xbox Game Pass" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
<strong>最低配置:</strong>
</p>
<p style="text-indent:2em;">
需要 64 位处理器和操作系统
</p>
<p style="text-indent:2em;">
操作系统: Windows 7, 64-bit
</p>
<p style="text-indent:2em;">
处理器: Intel i5 4th generation / AMD FX-8300
</p>
<p style="text-indent:2em;">
内存: 8 GB RAM
</p>
<p style="text-indent:2em;">
显卡: NVIDIA GTX 770 / AMD R9 290
</p>
<p style="text-indent:2em;">
DirectX 版本: 11
</p>
<p style="text-indent:2em;">
存储空间: 需要 25 GB 可用空间
</p>
<p style="text-indent:2em;">
<strong>推荐配置:</strong>
</p>
<p style="text-indent:2em;">
需要 64 位处理器和操作系统
</p>
<p style="text-indent:2em;">
操作系统: Windows 7, 64-bit
</p>
<p style="text-indent:2em;">
处理器: Intel i5 6th generation (or better) / AMD Ryzen 5 1600 (or better)
</p>
<p style="text-indent:2em;">
内存: 8 GB RAM
</p>
<p style="text-indent:2em;">
显卡: NVIDIA GTX 1060 (or better) / AMD RX 5500-XT (or better)
</p>
<p style="text-indent:2em;">
DirectX 版本: 11
</p>
<p style="text-indent:2em;">
存储空间: 需要 25 GB 可用空间
</p>
