<div>   
<p style="text-indent:2em;">
Feral Interactive 是一家专门将游戏移植到 MacOS 和其他平台的工作室。昨天（7月27日），在宣布《<a target="_blank" href="https://www.3dmgame.com/games/twstroy/">全面战争传奇：特洛伊</a>》的 Mythos DLC 之后，官方在推特上宣布，他们将在 Windows 发布后将会把游戏本体和DLC移植至MacOS版本。然而，与此同时，它也表示不会恢复先前宣布的 Linux 移植的工作，因为V社已经有效地扼杀了它的市场。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210728/1627454829_664481.png" alt="《全面战争传奇：特洛伊》原计划Linux移植版被取消" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
“Linux版移植工作在Epic独占时被暂时搁置，并且我们在Steam版本发售后也不会继续进行移植工作，”工作室解释说：“我们将继续评估将游戏一直到Linux的可能性，但考虑到V社推出了Porton，基于（系统）原生游戏的需求量普遍减少。”
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210728/1627454837_234387.png" alt="《全面战争传奇：特洛伊》原计划Linux移植版被取消" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
由 Valve 开发并于 2018 年发布的 Proton 是一个Linux兼容层，使基于 Windows 的游戏能够在 Linux 上运行。它运行良好：它是基于 Wine 兼容软件的一个分支，具有使其处理游戏的能力比 Wine 本身更强的功能。它是 Steam Play 的核心部分，也是即将推出的 Steam Deck 掌机的核心，使其可以开箱即用，掌机将采用基于 Linux 的 SteamOS，而Proton将支持基于 Windows 的游戏。
</p>
<p style="text-indent:2em;">
虽然有Linux用户和游戏玩家对这一决定感到失望，但是官方也只是表示：“展望未来，我们将评估将新游戏引入 Linux 的可行性，但我们将继续在必要时支持现有产品。”
</p>

</p>
