<p style="text-indent:2em;">
开发商ConcernedApe宣布《<a target="_blank" href="https://www.3dmgame.com/games/stardewvalley/">星露谷物语</a>》将于今年秋季登陆Xbox Game Pass（PC/主机）。目前该作已经登陆了PC 
Steam，PSV，iOS，安卓，PS4，Xbox One和Switch。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210811/1628638153_178857.jpg" alt="《星露谷物语》今秋登陆PC和主机端Xbox Game Pass" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
《星露谷物语》2016年2月27日登陆了Steam，支持简体中文，获得玩家“好评如潮”评价。
</p>
<p style="text-indent:2em;">
<strong>此外：</strong>
</p>
<p style="text-indent:2em;">
《邪恶天才2》将在今年第四季度登陆Xbox Game Pass
</p>
<p style="text-indent:2em;">
《废墟图书馆》现已登陆了Xbox Game Pass
</p>
<p style="text-indent:2em;">
《荒神2》9月17日登陆Xbox Game Pass
</p>
<p style="text-indent:2em;">
《开拓者：正义之怒》2022年3月1日登陆Xbox Game Pass
</p>
<p style="text-indent:2em;">
第一人称摄影游戏《Pupperazzi》也会登陆Xbox和Xbox Game Pass
</p>
