<div>   
<p style="text-indent:2em;">
Redmi Note 10T去年7月在印度上市发售，售价约合人民币1200元（4GB+64GB）。
</p>
<p style="text-indent:2em;">
日前，小米在日本市场推出Redmi Note 
10T新机，4GB+64GB售价34800日元（约合人民币1765元），虽然与在印度市场推出的机型同名，但配置却有差异。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220417/1650179236_462093.png" alt="Redmi Note 10T海外发布：防护等级堪比小米11 Ultra" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
据了解，在日本发布的Redmi Note 10T搭载高通骁龙480处理器，而在印度市场推出的则搭载联发科天玑700处理器。
</p>
<p style="text-indent:2em;">
值得一提的是，Redmi Note 10T日本版还支持IP68防尘防水，与小米11 Pro和“安卓之光”小米11 Ultra同款配置。
</p>
<p style="text-indent:2em;">
据悉，所谓“6”是电气设备最高防尘等级，可以完全防止外物及粉尘进入机身;“8”是电气设备最高的防水等级，可以在一定压力下长时间浸水，而不受到损坏。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220417/1650179243_145877.png" alt="Redmi Note 10T海外发布：防护等级堪比小米11 Ultra" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
核心配置上，Redmi Note 10T日本版搭载骁龙480处理器，配备一块6.5 
英寸居中打孔LCD屏，支持90Hz刷新率，分辨率为2400*1080，后置5000万像素主摄和200万像素景深镜头。
</p>
<p style="text-indent:2em;">
骁龙480采用8nm工艺，集成八核Kryo 460 CPU，包括两个2.0GHz A76、六个1.8GHz A55，集成Adreno 619 
GPU，整合骁龙X510 5G基带。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220417/1650179252_675504.png" alt="Redmi Note 10T海外发布：防护等级堪比小米11 Ultra" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
续航上，新机内置5000mAh电池，支持18W充电。
</p>
<p style="text-indent:2em;">
其他方面，Redmi Note 10T日本版预装基于Android 11的MIUI 13操作系统，后续可升级Android 12。
</p>
<p align="center">

</p>
