<div>   
<p style="text-indent:2em;">
部分《<a target="_blank" href="https://www.3dmgame.com/games/minecraft/">我的世界</a>》玩家似乎对 Java 版全新的官方启动器不太满意，因为它引发了许多问题。尽管官方推出该启动器原本旨在让 PC 玩家更容易访问游戏的 Windows 和 Java 版本，但是许多人报告称，新启动器会导致游戏无法启动、加载时间过长或根本无法加载游戏等问题。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211130/1638244821_891142.jpg" alt="《我的世界》新启动器 部分玩家运行Java版遇到问题" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
例如，部分《我的世界》玩家报告了以下错误消息：“启动游戏失败！无法复制文件。错误详细信息：系统无法将文件移动到其他磁盘驱动器。”而一些人则遇到了“《我的世界》启动器在您的账户中不可用。错误代码：0x803F8001。”的错误代码，又或是“<a target="_blank" href="https://www.3dmgame.com/tag/xiangsu_1/">像素</a>格式未加速”。最后一个错误信息是由那些因启动器导致老显卡驱动故障的玩家报告的。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211130/1638244827_511668.jpg" alt="《我的世界》新启动器 部分玩家运行Java版遇到问题" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
新的启动器还结合了所有的《我的世界》系列游戏，包括《我的世界：地牢》以及《我的世界基岩版》。但是，除了运行游戏遇到问题外，根据论坛 Reddit，玩家还称遇到下载时间长达两小时或是根本没有下载的问题。除此之外，游戏的 Mod 不兼容、帧率低下，甚至导致电脑蓝屏等问题频出。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211130/1638244860_803434.jpg" alt="《我的世界》新启动器 部分玩家运行Java版遇到问题" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
油管 Up 主 DCMX Gaming 上传的视频提供了新启动器下载 Mod 的教程，解决方案包括使用旧启动器、重启电脑、检查玩家的防火墙和杀毒软件是否阻止了启动器正常运行、重新下载安装启动器，或是直接从微软商店下载启动器等。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20211130/1638244895_881309.jpg" alt="《我的世界》新启动器 部分玩家运行Java版遇到问题" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
《我的世界》开发商 Mojang 正通过客服中心以及官方推特给玩家提供帮助。官方要求玩家提交屏幕截图以帮助确定问题并迅速解决。
</p>          
</div>
