<div>   
<p style="text-indent:2em;">
世嘉股份有限公司宣布，现已开启PC（Steam）平台<a target="_blank" href="https://www.3dmgame.com/tag/rhythm_1/">节奏</a>动作游戏《初音未来 Project DIVAMEGA39’s+》的限时折扣促销活动，折扣幅度最高可达30%，截止到2022年8月5日（周五）。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220802/1659428182_356232.jpg" alt="《初音未来 Project DIVA MEGA39’s+》低至七折 限时折扣促销截止到8月5日" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
<strong><span style="font-size:16px;color:#009900;">游戏Steam商店页面：</span></strong><a href="https://store.steampowered.com/app/1761390/" target="_blank"><strong><span style="font-size:16px;">点击这里</span></strong></a>
</p>
<p style="text-indent:2em;">
《初音未来
Project DIVA MEGA39’s+》是一款简单爽快，“玩起来像唱歌一般”的节奏游戏，游戏中模型（服装）丰富的精美PV广受好评。本作也是“Project DIVA”系列首次登陆PC平台的作品。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220802/1659428201_364459.jpg" alt="《初音未来 Project DIVA MEGA39’s+》低至七折 限时折扣促销截止到8月5日" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
本作的节奏游戏部分收录了超过170首乐曲。除了2020年在Nintendo Switch™平台发售的《初音未来
Project DIVA MEGA39’s》中的收录曲目外，还能够游玩追加下载内容（DLC）中的乐曲。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220802/1659428209_408912.jpg" alt="《初音未来 Project DIVA MEGA39’s+》低至七折 限时折扣促销截止到8月5日" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
此外，若购买数字豪华版“VIP
Edition”或DLC“Extra Song Pack”，则可以游玩的曲目将超过240首。
</p>
<p style="text-indent:2em;">
不仅如此，本作还推出了PC版的独有功能，3DPV的视觉风格可以在“动画画风”和“现实画风”之间进行选择。“PV照片模式”与“T恤编辑功能”等系列经典内容也会在本作中出现。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20220802/1659428217_911555.jpg" alt="《初音未来 Project DIVA MEGA39’s+》低至七折 限时折扣促销截止到8月5日" referrerpolicy="no-referrer">
</p>
<p style="text-indent:2em;">
促销从今天8月2日（周二）起至8月5日（周五）为止，普通版降价25%，数字豪华版“VIP Edition”降价30%。机会难得，千万不要错过！
</p>
