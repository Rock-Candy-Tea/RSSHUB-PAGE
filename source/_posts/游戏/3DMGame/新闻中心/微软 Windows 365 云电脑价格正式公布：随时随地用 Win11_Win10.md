<div>   
<p style="text-indent:2em;">
在Inspire 2021合作伙伴大会上，微软终于揭开了他们的Windows 365云电脑产品的面纱。
</p>
<p align="center">
<img src="https://img.3dmgame.com/uploads/images/news/20210803/1627990197_176001.jpg" alt="微软 Windows 365 云电脑价格正式公布：随时随地用 Win11/Win10" referrerpolicy="no-referrer"> 
</p>
<p style="text-indent:2em;">
Windows 365 是建立在 Azure 虚拟桌面服务基础上的一项新的云服务，将允许企业从云端在任何设备上体验 Windows 10 或 
Windows 11 系统。
</p>
<p style="text-indent:2em;">
Windows 365 服务今天开始普遍提供，微软官方定价也随之公布。
</p>
<p style="text-indent:2em;">
<strong>Windows 365 有两款变体:</strong> 
</p>
<p style="text-indent:2em;">
Windows 365 商业版：上限为 300 个用户;每个用户 20 美元（约 129.2 元人民币），用于购买 1 个虚拟核心、2GB 内存和 
64GB 存储空间以及 Windows Hybrid Benefit（不含的话是 24 美元）;每月费用 158 美元，8 个虚拟核心，32GB 内存和 
512GB 存储，以及 Windows Hybrid Benefit（不含的话是 162 美元）。
</p>
<p style="text-indent:2em;">
Windows 365 企业版：300 个用户以上;每个用户 20 美元，用于 1 个虚拟核心，2GB 内存和 64GB 存储空间;每月费用 158 
美元，8 个虚拟核心、32GB 内存和 512GB 存储空间。
</p>
<p style="text-indent:2em;">
Windows Hybrid Benefit 允许公司将其现有许可证应用于云服务。
</p>
<p style="text-indent:2em;">
Windows 365 服务支持所有应用程序，包括视频编辑等高性能应用程序。
</p>
<p style="text-indent:2em;">
每个用户的定价是相似的，在增长到包括更多 CPU、内存、存储和（对于商业套餐）带宽的套餐中略有不同。如果你已经在使用 Windows 10 
专业版许可证，那么最便宜的设置（1 个 vCPU、2GB 内存、64GB 存储和 12GB 带宽），每人每月 20 美元，没有许可证则为 24 美元。
</p>
<p style="text-indent:2em;">
一款拥有 4 个 vCPU、16GB 内存和 128GB 存储空间的“高级”套餐，每人每月价格为 66 美元，有 4 
美元的折扣，微软的常见问题表明，如果用户遇到需要更多支持的任务，他们可以升级。
</p>
<p style="text-indent:2em;">
最强大的套餐是每人每月158美元（有折扣），用于流转一台拥有8个vCPU、32GB 内存和 512GB 存储的电脑。
</p>
<p style="text-indent:2em;">
这当然比购买一台同等的电脑要便宜，而且如果它不能开机、损坏或被盗，也不需要任何形式的维护。对于远程工作者来说，这可能是得到一台新的工作笔记本电脑或只是收到一个Windows365.com 的登录账号之间的区别。
</p>
<p style="text-indent:2em;">
<strong>要使用 Windows 365，用户需要满足以下要求：</strong> 
</p>
<p style="text-indent:2em;">
使用 Windows Pro 终端：Windows 10 E3 + EMS E3 或 Microsoft 365 F3/E3/E5/BP
</p>
<p style="text-indent:2em;">
使用非 Windows Pro 终端：Windows VDA E3 + EMS E3 或 Microsoft 365 F3/E3/F5/BP
</p>
<p style="text-indent:2em;">
Azure 订阅
</p>
<p style="text-indent:2em;">
有了Windows 365，企业可以根据自己的需求选择云电脑的大小，并且可预测每用户每月定价。由于Windows将在云中运行，用户可以即时启动到他们的个人云电脑，并在不同的设备上流转他们的业务应用程序、数据和设置。你只需要一个网络浏览器来访问Windows 365云电脑。因此，你可以从任何设备上享受Windows 365，包括你的 Mac、iPad、Linux PC 和 Android 设备。
</p>
