<p>微软的Game Pass 一共有三个版本，有仅限在Xbox主机上使用的Xbox Game Pass（简称XGP），有在电脑上使用的Xbox Game Pass for PC，当然也有最具人气的Xbox Game Pass Ultimate（简称XGPU），不仅能在Xbox和PC上使用，还包含了Xbox金会员与EA Play。</p><p><img class="lazy" data-height="528" src="https://i1.max-c.com/imgx/2021/12/10/51cf5b2f57b944658abfc06a28a368b91639128123.jpg" data-width="940" referrerpolicy="no-referrer"></p><p>在2021 TGA上，微软宣布将Xbox Game Pass PC版改名为PC Game Pass。微软表示，仅仅是名字和图标发生了变化，提供的服务内容并不会更改，不会影响玩家的游玩。</p><p><img class="lazy" data-height="1281" src="https://i1.max-c.com/imgx/2021/12/10/f368edf1c2d744b3b5ca6a8254d3ae411639128124.jpg" data-width="1080" referrerpolicy="no-referrer"></p><p>XGPU性价比真的很高，在电商平台上，不到120元的价格就能买到三年的XGPU。2021 TGA年度游戏《双人成行》也在XPG库中，以及《极限竞速：地平线5》和《瘟疫传说：安魂曲》等大作都能直接游玩。</p><p><img class="lazy" data-height="607" src="https://i1.max-c.com/imgx/2021/12/10/5e7dba77488e49fd93da3dc0fd8a932b1639128125.jpg" data-width="900" referrerpolicy="no-referrer"></p><blockquote><p>来源：游侠网</p></blockquote>
