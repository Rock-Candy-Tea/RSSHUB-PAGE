<p>每日分享。</p><p>1.微软商店（Microsoft Store），在9月29日宣布了一项新政策：从亚马逊和Epic Games Store开始，将在Microsoft Store上提供第三方商店的应用程序。</p><p>微软在更新博客中称：“本着这种精神，我们宣布对我们的 Microsoft Store on Windows 策略进行另一项重大更新，这将允许在 Windows 上的 Microsoft Store 中发现第三方店面应用程序。就像任何其他应用程序一样，第三方店面应用程序将有一个产品详细信息页面，可以通过搜索或浏览找到该页面。以便用户可以像 Windows 上的 Microsoft Store 中的任何其他应用程序一样轻松地找到并安装它。”<br></p><p>“ 今天，我们分享 Amazon 和 Epic Games 将在未来几个月内将他们的店面应用程序带到 Microsoft Store，我们期待将来也欢迎其他商店。”<br></p><p>这并不是在微软商店里直接浏览Epic Store，而是可以通过微软商店下载和安装Epic商店应用程序。此外微软还允许在商店中上架第三方替代浏览器，最早的两个是 Opera 和 Yandex。<br></p><p>Epic Games 首席执行官蒂姆 · 斯威尼 ( Tim Sweeney
