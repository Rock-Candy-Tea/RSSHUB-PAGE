<p><img class="lazy" data-height="320" src="https://imgheybox.max-c.com/bbs/2022/04/21/5a3601490e15980ff7c83b0397380add/thumb.jpeg" data-width="690" referrerpolicy="no-referrer"></p><p><img class="lazy" data-height="2400" src="https://imgheybox.max-c.com/bbs/2022/04/21/53e56ed25ece9cd3e7466540a01cc942/thumb.jpeg" data-width="1080" referrerpolicy="no-referrer"></p><p>雾切池陪跑无工大剑已经尘埃落定，作为一个上次满定轨没能氨出雾切的刻厨，我在痛定思痛过后想和各位聊聊无工大剑这把武器。</p><h2>老规矩，看武器先看属性，90级无工大剑属性如图所示：608白值，位于五星第三档，副属性49.6％攻击，与常驻老牌大剑狼末一样。武器特效：获得20％护盾强效（仅对装备该武器的角色生效），攻击命中时获得4％攻击，持续8s，每0.3s可触发一次，最高可叠加5层，即20％攻击，护盾庇护下攻击力加成翻倍，即共40％攻击。</h2><p>个人评价：上限较高，但绑定护盾角色，限制配队，且与常驻老牌武器狼末属性高度相似。总体强度一般。</p><p>接下来是碎碎念时间：这次陪跑无工大剑确实伤透了不少人的心，主要是原本爆料陪跑是狼末，但公布出来是无工，如果是狼末，总体中规中矩，可捞，但变成了无工瞬间就成为了毒池。</p><p>究其原因，还是无工大剑自身强度不行加上常驻的狼末不少人都有。无工大剑目前的处境十分尴尬，本身作为攻击力加成高的武器打不过大剑主 c 们的专武，加上常驻还有一把持有率不错的同为攻击力大剑的狼末，这使得无工大剑作为一把限定武器非常尴尬，并且狼末本身被动常驻20％攻击力，虽然第二被动触发较难，但可以作为主 c 的过渡武器，在主 c 有更好的武器之后可以直接给副 c 甚至辅助们使用，但无工大剑强制要求站场同时绑定护盾导致这把武器给绑定护盾的大剑主 c 使用才能发挥全部实力，加上目前大环境大剑主 c 不吃香以及护盾角色的价值下降（深渊 DPS 要求过高导致不得不将护盾角色替换成副 c 或增伤辅助），种种原因都让无工大剑被玩家们睡弃。老米完全可以放个狼末甚至冬极天空弓之类的中规中矩或者稍微强点的武器陪跑，直接口碑流水双赢，但它偏偏要塞个无工大剑这种废铁来恶心玩家，更别说四星武器也是非常恶心，我只能相信这就是像某些 up 说的那样是来试探玩家底线的。</p><h2>而且现在居然还能看到有人踩无工大剑捧不灭月华的魔怔人出现，甜甜圈在我看来是比无工大剑还要烂的武器，无工大剑固然 fen ，但甜甜圈更是重量级。具体的不解释了，懂得都懂。</h2><p><a href="heybox://open_subject" target="_blank">#原神2.6版本创作者征集活动#</a> <br></p>
