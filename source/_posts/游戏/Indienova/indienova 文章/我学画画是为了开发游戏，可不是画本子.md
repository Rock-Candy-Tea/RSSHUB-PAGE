<p>大家好，我是桑二哥，我是一名即将上路的独立游戏开发者。微博@桑二哥，B 站@桑二哥</p><p>最近试着画了个角色，感觉有一点点进步，有些小开心，在这里跟大家分享一下喜悦，先谢谢大家的捧场。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/05/u-1616821480047h93.png" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/05/u-1616821480058U9t.png" referrerpolicy="no-referrer"></p><p><br></p><p>我小时候的梦想是当一名画家，从小我就特别喜欢画画，在我不到 9 年的求学生涯中，我一直是班里画的数一数二的吧，板报橱窗也一直都是我负责画插画，我上小学的时候没有专门的美术老师，记忆中也没学习什么关于画画的技巧，好像我天生就会画画一样。</p><p>我上小学是在九几年，上课用的投影仪也很原始，就是在一张透明塑料一样的卡片上用钢笔写上内容，再用投影仪照射出来，有一次学校要来领导检查，老师让我画一幅画投影用，我还很清晰的记得我画了个将军，盔甲什么的都记得，我很重视这次机会，自然是拿出十二分力气，画了足足一节课，在老师办公室里画的，没上课，可以名正言顺的不上课感觉还是挺爽的。</p><p>老师看了赞不绝口，我自己也很惊讶原来我用点力就能比以前画的更好。</p><p>五年级的时候画黑板报，那是 2001 年，应该画什么才比较符合 2001 这个数字呢？我灵光一闪，在整个黑板上画了一条龙，盘成 2001 的形状，来上课的老师看到这条龙也无不称赞，我自己也很满意能有这样的机会露一手，毕竟从小性格懦弱的我存在感不强，唯一的长处就是会画画。</p><p>初中，我最喜欢学校举办画画比赛，有些女同学不会画就会来找我给他们画，其中不乏别班的女生，她们会给我买很多零食，我给她们画些美少女战士什么的。</p><p>2005 年辍学，我就成了一名灯具安装工，上班忙得很，也没时间画画，07 年我去建筑工程学校参加为期三个月的预算员培训班，那时候年轻不知道学习，上班也听不进去，就偷偷画漫画，考试的时候我可傻了眼，正愁要不要作弊呢，抬头看了下四周，好家伙，别人都在抄，那我也抄吧，总算顺利混了个资格证书，托舅舅的关系进了建筑公司实习。</p><p>这可算是坐上办公室了，开心的不得了，咱也算白领了哈~</p><p>空闲里我还学着用 windows 画板画个画，印象最深的是这张特南克斯。</p><p><br></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/05/u-1616821480463tBD.jpg" referrerpolicy="no-referrer"></p><p>很快一年的实习期结束，不用想也知道结果。后来我成了一名焊工，天天在车间里跟不锈钢板打交道。</p><p>2011 年，通过打工攒了点钱，自己配了台电脑，自学了平面设计，进入平面设计行业，然后自学 3DMAX 室内效果图，进入装修行业，又自学了游戏编程，终于进入游戏行业。其实我从 2013 年就开始学习编程，断断续续的也没学出个啥东西，相当痛苦，最后放弃编程转学 3DMAX，离开装修行业后不甘心又捡起编程，这才入门，这 5 年间我竟然一张画也没画过。</p><p>2016 年来到北京，我从农村到县城到市里再到北京，一步一步终于走出来了，总算是进入了游戏行业，从一个焊工转变成游戏开发人员——程序员，离画画是越来越远了，后来工作不太顺利，项目黄了，我也就回老家了，打算独立开发游戏，买了显示器，从 steam 上买了 Aseprite，人不被逼到份上是不知道自己有多少潜力的，我从来没画过像素画，没画过序列帧，我用了三四天的时间画完了几套动作，借鉴了很多游戏，比如待机动作借鉴的拳皇草薙京，有些动作借鉴了恶魔城月下夜想曲里的阿鲁卡多，有些动作借鉴的甄子丹。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/05/u-1616821480719Z6d.png" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/05/u-1616821480733jYw.png" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/05/u-161682148077556D.png" referrerpolicy="no-referrer"></p><p>我还做了个 DEMO，把画的角色在 Unity 里运行起来，不过后来没坚持下去，不得不继续北上打工，找了个关卡策划的工作，本以为轻轻松松的，谁知道这家公司挺鸡贼，什么都想让我干，关卡做完了画 UI，反正我简历写着精通 PS，UI 画完了做程序，反正我简历写着做过程序员，但工资却只字未提，依然是入职时的 7k。</p><p>我虽然做过平面设计，但没有画过 UI，我甚至都不习惯用画板，一开始我用鼠标画 UI，画的挺烂，但还是合格了，可以用，画了几套之后我就有点放飞自我，画了一套别人都看着很丑的 UI，我自己倒是觉得有点意思，有个策划说我画的像十几年前的 UI，还给我找了个他觉得不错的游戏给我看别人是怎么画的，我看了一眼说我没看出来这 UI 好在哪里，主 UI 问我：不服？我笑了笑说：这真不是服不服的事，我是真没觉得这 UI 画的好。</p><p>胳膊拧不过大腿，还是得重新画，我还真有点较劲的感觉，开始拿出真正的实力，也强迫我自己用画板，我知道用鼠标是很难画好的，为了这口气我也得适应画板，结果真超出主 UI 的预料了，本来这套是给另一个项目组画的，主 UI 跟我说这套画的有意思，就抢过去给他项目用了，我有点傲气的说：上一套我没认真画。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/05/u-1616821480947LP5.png" referrerpolicy="no-referrer"></p><p>就这一套 UI 的历练，我就直接上了一个层次，从来没想过我能用画板画出这种效果。虽然画 UI 不是我本职工作，但也学到了不少东西，刚好把我的短板给补一下。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/05/u-1616821481007FuN.png" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/05/u-1616821481028OdI.png" referrerpolicy="no-referrer"></p><p>之前我无法独立开发游戏就是因为我画画是短板，我画不出我想要的那种画，现在我觉得我离目标更近了，我又画了一张银河战士主角萨姆斯，画了有一周吧，就挺难受的，比画像素小人麻烦多了。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/05/u-1616821481086R79.png" referrerpolicy="no-referrer"></p><p>前几天有了些灵感，是从塞尔达传说得到的灵感，来自林克的一身衣服，好像叫海利亚服，戴着帽子，刚好把脸挡住，这就省去了很多工作量，画个帽子可以不用画头发不用画脸，戴个围巾可以不用画脖子，不会画肌肉我就把胳膊腿都画成棍子，就这样我画了个线稿，迭代了好几个版本，上完色之后竟然感觉还有点意思，配色是参考的火影忍者里仙人模式的鸣人，红黄黑色调，我之前画的像素小人也是参考的鸣人。总之，感觉越来越好了，我一定能完成自己的游戏。</p>
