<p>距离上次更新日志已有半年多，当时参加完杉果展会后，想着下篇更新日志的题目是“2020 算是停滞不前吗”，结果打错成 2021，没想到一语成谶，大半年多过去了，老子的进度推进得如蜗牛般缓慢……心情复杂，心态多变，还是照例先讲游戏相关，再吐苦水。</p><h4>2020 最开心的事情</h4><p>2020 年过得很快，是我短暂的人生至今为止感觉过得最快的一年，好像还没怎么抓住就流走了，有时我恍惚间还觉得是 20 或 19，不敢相信这是 21。偶然间看到些年度总结，嗯……没什么实感，有个 up 主提了个问题“20 年最开心的事情是什么”，当时愣了好久，硬是没想出来，毕业一波三折，工作不甚理想，情感没啥进展，疫情遗憾不少，到底有什么值得开心的事情吗？</p><p>后来还是想到了一些，但感觉还不够，就像哈利在练防御咒（名字记不清了）时苦苦思索的状态。啊，终于回忆起来，是上次参加展会后，玩家对作品的一些评价让我备受鼓舞，这个让我很开心，很满足。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477551881fup.png" width="746" referrerpolicy="no-referrer"></p><p>这是当时杉果线上展会的总体评价，哈哈哈传世之作蛮搞笑的，其实很多我怀疑是完成任务随意打分的（他们真是慷慨啊），就一两句与游戏无关或游戏摘抄下来的话，但还是非常感谢他们花时间体验这款“充满艺术感的简笔画作品”，批评最多的是画风和引导，鼓励的话就是说这个游戏有想法（这评价在我看来已经很高了）和玩起来挺放松的。当时我作品的封面在众多优秀作品里可谓独树一帜（暂时找不到图 orz），一眼就能找到这个简笔画，所以才有超过两位数的人来体验吧！</p><p>当然啦，什么投资天使或奇迹并没有敲门，我甚至过了很久才去看玩家评价（当时太忙了），之后在奶牛关上有两个很有意思的评价——</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477552574fIA.png" width="727" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-114775526001LE.png" width="722" referrerpolicy="no-referrer"></p><p>说真的，当时见这两个评价我都愣了，心里既窃喜又不安，好比孩子被拐卖的父母多年后与娃儿相聚，狂喜之余又觉得不真实：我家儿子长这样吗？怎么跟我印象中不太一样啊。那时我过了挺久才回复，不是不想，是真的不知该怎么回复好，放假我才有时间摸一下游戏，然后发现自己对作品十分陌生，好像它不是出自我之手一般。嗯……后来找到一点感觉才敢回复，发现我自己创作真的挺依赖感觉的。扯个题外话，我知道很多人不喜欢这样，甚至对这种“即兴”是带批判态度的，我曾经给某些人看过我的剧本开头（即 demo 剧情），他一眼就看穿情节都是即兴写的（我至今没想明白他怎么看穿的），接着推荐我去读剧本写作的书籍（他不知道我都看过好几遍了，哈）。我不是不知道那些三段式，但我还是选择相信灵感，觉得这样对故事本身来说会更自然一点吧，不过这个东西是有点见仁见智，从工业角度上讲当然有套路是最好的（这里说一下陈星汉的风之旅人，我个人觉得故事结构可以更开放自由一点）。</p><p>我把上面两个评价单拎出来，不仅因为他们评得认真、观点相似，更主要是他们都提到“剧情里的中学生口吻很真实”，我个人倒是挺惊讶的，因为我从未这么觉得。我只是觉得大多数 gal 的日文腔都太重了（不管男主女主），我就尽量避免，但我明白这个离“中学生口吻真实”还差很远，那是很难做到的（不是加多点脏话就行），所以当我看到这样的评价出现还不止一位时，真的是非常惊讶，说受宠若惊也不为过哈哈哈。希望剩下的写作中也能继续做到这一点吧（假如有做到的话）。</p><h4>UI 更新</h4><p>评价中有不少提到 UI 和按钮的不完善，我就在这方面弄了一下——</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477552805ybr.png" width="594" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477552815Nom.png" width="605" referrerpolicy="no-referrer"></p><p>首先换的就是标题页面，之前想法是加强沉浸感，就用校门做了个标题页面的背景，后来发现粗糙的来，也没有很好传达那个意图，光标移上去也没有反馈，可谓说糟糕透顶。嗯……一时又想不到用什么做背景好，突然就想不如拿封面做一个吧，试了后发现还蛮奇妙，挺有 feel 的（哈哈哈等过了一段时间，我就又觉得像坨 shit）。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477553123RVS.png" width="720" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477553147JKx.png" width="714" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-114775531610QL.png" width="717" referrerpolicy="no-referrer"></p><p>接着就到选课页面，明显之前风格太硬了，图标画得也让人云里雾里，实验成分太多，只有我自己看得懂。当时玩的心态多一点，什么都想着试一试，实在不够专业，还弄了个“鼠标移至课程按钮，会随机播放相关图片”，现在看来没个卵用，还干扰了玩家。而从未玩过 iphone 手机的我久仰其非线性动画大名，看了不少相关介绍，发现苹果在 UI 设计上确实下了不少功夫，值得广大游戏开发者学习（感觉我们在游戏 UI 方面还有很大进步空间），但精力有限，过多打磨性价比不高，我就简单照葫芦画瓢，感觉是有进步了，起码看上去像点回事了（看看到时能不能弄个比苹果更酷的嘿嘿）。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477553375Aiq.png" width="709" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477553398QHQ.png" width="711" referrerpolicy="no-referrer"></p><p>之前的学校地图也是十分糟糕，本想弄个好看点的地图，但后来发现不一定要弄地图（虽然一般模拟养成玩法的，都是展现地图再点击地点），怎么说呢，因为“地图”这个东西是有点出戏的，你想想平常我们出门去某个地方，难道会看地图吗？地图这个概念，潜意识里象征俯瞰，俯视，探险等东西，与日常生活隔得比较远。所以我就将地点弄成卡片，希望能有沉浸感，更生动，但感觉卡片也不一定是最好的选择，嗯……只好先这样吧……</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477553511iw3.png" width="715" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477553534pYY.png" width="717" referrerpolicy="no-referrer"></p><p>城市地图也是同理，地图难免会抽象一点，而我想让城市页面有一种亲切感，就像那些手绘插画那样给人触手可及的感觉，不过个人画功着实菜鸡，只好慢慢来，慢慢画了……</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477553606K9i.png" width="727" referrerpolicy="no-referrer"></p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477553622sKV.png" width="727" referrerpolicy="no-referrer"></p><p>为了弥补背景不够，我就画了张校门（我终于会画点透视，开窍了……），从简笔画上升为菜鸟画，但大部分还是实景照片，连后期都懒得后期，反正最后都还是要画，只是现在先填充（有个坏消息就是，公有版权的泼辣图库关闭了，我还要逐步替换之前用上的照片……）。只能恨自己当初为什么没学美术，不然哪用为此头疼。</p><p>对话框我也换了一版，之前那个着实太抽象了，抽象不是问题，问题是形象不够鲜明，不能让人一眼就知道那图标代表什么，所以我就将悬浮状态弄成中文字，希望这样能帮助玩家了解每个按钮的意思，建立图标与实际功能的联系。原本我想着将按钮弄成书包，笔啊这些现实事物，符合主角的身份，现在觉得抽象一点也无所谓，减少玩家的干扰信息或信息接收量，或许更能加强玩家的沉浸感。总的来看，基本上 ui 都齐全了，悬浮和音乐反馈也有了，还多了不少 BGM，就是整个游戏更有质感了。感觉 UI 这东西啊，多思考多尝试，总会有更好的答案滴。所以这一版本虽然没怎么更新剧情，但也算有不少进度吧（小小的自我安慰一下）！</p><p><br></p><p>----个人生活苦水分割线----</p><h4>支教生活</h4><p>支教，不知大家怎么看待这个东西，我现在来了，懂了，无非四字——钱少事多，当然啦，可能其他工作也是如此，又或者只是我运气差了点，去的学校不够好，现在离一年任期还剩 3 个月，中间心态真是变化剧烈，从对现实的不忿不满到如今的钝化无感，不知这是好是坏。也不知道怎么说起，感觉现实真的犹如沙漠中一堵无边无际，高不见顶的墙矗立在你的面前，你想过去，但怎么都找不到出口，你甚至连墙是由什么做的都不知道。你看到很多朋友都在拼命往上爬[刘 1] ，或者就躺在地上接受爬不过去的事实，而我还是傻傻地仰起头看着那堵墙，抱着一种夹杂愤怒，不解，困惑，荒谬，震惊，不安，恐惧等种种反应的莫名情绪，看着它。那种感觉，难以用言语表达，为什么会有那堵墙在哪里呢，哪怕我读过一点书，可能知道墙从哪里来或者它建造机制是什么，但还是接受不了这样的事实——为什么会这样呢。当时没有毕业的时候，我就已远远看见那堵墙了，但真的走到面前去面对的时候，才发现它真的很……很……很什么，还是说不出来，想不到那个词，不知怎么形容，“无限”暂时是比较恰当的，种种情绪才因此而生。罗翔说，人总要接受事与愿违。他说得对，但我情感上接受不了，总想一刀刺过去，划开那丑陋不堪的面孔；朋友说，你改变不了，就只能接受。他说得可能更对，现实的，太现实的，而我自觉或不自觉地，主动或被迫地开始接受现实，接受事实的现实或现实的事实。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2021/04/u-11477554149PMT.png" width="731" referrerpolicy="no-referrer"></p><p>所以我真没觉得教育或支教有多大意义，我基本不向学生说“读书有什么意义”或者“能帮助你走出大山，过上好生活”之类的东西，因为我自己都不太信这套，跟朋友聊天都在聊 house，car，money 那些，又怎么敢跟学生传授我自己都不认可的价值观呢。大环境不改变，怎么教都白扯。教育是因，但主要是果，不平等的因永远种不出平等的果，所以高考只是打着平等的遮羞布。我也知道这么想不太政治正确，也很消极，但……扯个题外话，拿些互联网巨头举例，一方面它拿奶头乐软件（不知大家觉得，游戏是否身处其中呢）让农村孩子沉迷，另一方面它又研发教育软件，服务有能力消费它的人群，而这两方面工作，它都拿出无与伦比的高薪酬吸引全国最顶尖的人才去完成。那最后，我们的孩子怎么办，那些身处经济不发达甚至落后地区的孩子怎么办，怎么跨越所谓的“信息壁垒”？我理解“人往高处走”，但孩子是无辜的。我甚至有个悲观的猜想，按如今在线教育和人工智能的发展趋势，教育不平等只会越拉越大，到时衡水模式将彻底消失，因为你再怎么刷题也比不上全国学历最高的人才研发的人工智能老师，到那时……好吧，到时再说吧，但愿我是错的。</p><h4>创作迷茫</h4><p>本来，去个支教，想着一是自由一点，不用那么应试，后来才发现都一样，也还是要围着成绩转啊转；二是感觉应该挺有空，可以多点时间搞创作，结果忙得要吐，屁事一堆，又不适应，备课教学上我都尽力而为，学生可以不学，但老师不能不教啊，人家课件很多都网上下载，而我还在苦苦寻思怎么教出新意，自己怎么才能搞明白，再怎么讲给学生。唉，其实备课累点都没啥，最主要是村里人素质有限（不是我偏见，只是现在不是 cat 时期或以前），公社或基层组织的退位，家长苦于生计无力教育，互联网的不好风气，你老师想说扭转趋势，把那个素质弄上来，真没那么容易，学校承担了太多不该承担的职责了，真的是远超负荷，特别是本来农村小学资源投入就有限的情况下。因此，老教师们教完就走或者单纯靠凶管学生，这在经济学上是最优解，我能理解。但你说这样的东西叫教育，恕我不敢苟同。</p><p>总之，就是没时间没精力搞创作了（有点像电池，没电了只能休息充电，没有多余电量搞别的了），本来我还坚持时不时弄一下，后来发现这样不仅没劲，还效率奇低，就干脆不弄了。寒假才找到感觉，推了一下进度。现在开学了……发现自己真的创作感觉所剩无几，每次重新找回来都觉得煎熬难受，当然生计所迫没办法，人总要吃饭。之前想着教完这一年就全职干，起码要写完剧本，可现在心有点软，不舍得离开学生，另一方面积蓄也不够多。但可能更重要的原因是我不太想依赖感觉，假如感觉或灵感是没那么重要的话，那么什么时候推进都没有太大关系（只要世界不毁灭），还有就是我害怕，我害怕真到全职创作的那一天，自己会做得不那么好，效率不如人意，到那时就没有工作或学业这些借口了。所以那些利用业余时间搞得一手好作品的人，真的是好厉害，真想知道他们是怎么做到的，我也想那样，但感觉真的有点难……</p><p>因此就有点尴尬，有点迷茫，可能最要命是没有激情，激情下降到最低点了（也许创作难免，只是或早或迟）。我有时会幻想“天降投资”，明天有人砸我一大堆钱就好了。说来也搞笑，曾经对商业爱理不理的我现在却经常在看，尤其看着游戏业界新闻时总觉得有点滑稽，我离游戏界远得不能再远了，看再多好像也没用啊。我也关心起那些独立游戏比赛，想着为了作品，拥抱下商业好像也没什么，也算是学习的一部分吧。某种程度上，也能督促一下我在有限时间内更新多点内容，这也算是成长吧。这方面不知道怎么说，我自己也知道如今单人做游戏的情况不多了，何况我能力还那么有限，想脱颖而出或受人青睐是很难得的，但还是想试一试，听听反馈也不错，吃不了什么亏。毕竟，有了投资，游戏品质通常能更上一层，哪个作者不想呢。</p><h4>总结</h4><p>其实还有很多牢骚话，但说多了也没意思（很少人会读到这里吧），大概就是这样，希望……也不知道希望什么好了，不久前看到朋友圈师姐结婚了——不得不感叹，老子青春结束了，老子也好想结婚啊 <span class="spoiler-span">CTM</span>！虽然连个对象都没有，但来个幸福的结婚照，流点幸福的泪水，好像也不赖，对吧。</p><p><br></p><p>Ps: 啊对，我用了新素材又弄了个新 pv，有兴趣的可以来捧个场 <a href="https://www.bilibili.com/video/bv1bV411e7Fs" rel="noopener
