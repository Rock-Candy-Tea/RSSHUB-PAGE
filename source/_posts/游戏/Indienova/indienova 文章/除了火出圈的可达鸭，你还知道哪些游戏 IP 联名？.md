<p><strong>文：Along</strong></p><p>最近，一个离谱到不能再离谱的微博话题吸引了我的注意力：任天堂抢注可达鸭商标。我去逛了一圈，原来是肯德基 X《宝可梦》联名推出的可达鸭音乐盒火爆全网，有没有一种可能性，咱就是说，可达鸭本来就是任天堂的？</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475234wrW.gif" referrerpolicy="no-referrer"></p><p>随着可达鸭热度持续发酵，这位老宝可梦俨然成了新网红，最近甚至出现了一“鸭”难求的现象，不仅各地肯德基缺货严重，二手交易平台和各路黄牛也没有放过商机，随便搜索就能看到远超原 69 元儿童套餐的售价。尽管肯德基方面表示坚决不予支持加价行为，但显然粉丝的热情并未因此消退，不过这里还是劝各位一句理性消费，给黄牛党送钱实在不值得。</p><p>借此机会，不如今天我们就来聊聊那些游戏 IP 联名产品，看看它们究竟收割玩家热情的“智商税”，还是错过遗憾终生的收藏好物。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475241mHK.jpg" style="width: 47.55%;" referrerpolicy="no-referrer"></p><h4>懂不懂世界第一游戏 IP《宝可梦》的含金量啊？</h4><p>根据统计机构 Statista 公布的数据，自任天堂 1996 年推出《宝可梦》系列游戏以来，全世界范围内人们在各种《宝可梦》相关业务上的消费总额约为 1000 亿美元，这个数字使《宝可梦》超越了 Hello Kitty 和米老鼠，成为世界上最赚钱的 IP。</p><p>统计数据显示，这 1000 亿美元 IP 衍生品、周边产品是赚钱的重要基础之一，老任的想法显而易见，靠独占游戏牢牢地维持着粉丝群体的基本盘，然后用 IP 自身的价值生产其他产品。最后靠着东半球最强法务部门的能力，保证正版产品在市场上的占比。这就是《宝可梦》这个 IP 吸金能力强劲最基础的保障。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475252Onw.jpg" referrerpolicy="no-referrer"></p><p>从具体的《宝可梦》IP 联名产品来看，老任倒确实也算不上“割韭菜”。上文提到的肯德基 X 《宝可梦》的联名，你只要吃一份 69 元的儿童套餐，就可以获得心仪的实体周边。前些天公布的三星 X《宝可梦》联名的 Galaxy Buds 2 耳机，售价也只有大概 700 元人民币，不仅与原版售价相差不大，我们还能获得精灵球造型的耳机盒。虽然硬件性能别无二致，但能把精灵球当做饰品挂在包包上还是很令人心动的。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475258Q31.jpg" referrerpolicy="no-referrer"></p><p>此前雷蛇 X《宝可梦》联名同样是在原价基础上稍稍加价，即可获得带有皮卡丘图案的无线鼠标、伊布主题的键盘，虽然与精灵球耳机盒相比没有那么独特，但无线鼠标的充电底座中加入皮卡丘标志性的闪电尾巴，辨识度这块属实拿捏的死死的。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475263soV.png" referrerpolicy="no-referrer"></p><p>类似的联名产品还有很多很多，超过 1000 亿美元的消费总额已经印证了这个 IP 的成功，诸多良心的联名产品也牵动无数粉丝的热情。不过还是希望老任和 GAME FREAK 继续保持联名产品性价比的同时，也不要忽略游戏本身的质量，毕竟这才是维持粉丝基本盘的必要条件。没错，说的就是即将于 2022 年冬季发售《宝可梦 朱/紫》。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475269uYt.jpg" referrerpolicy="no-referrer"></p><h4>SE 旗下与时尚界结下不解之缘的《最终幻想》</h4><p>《最终幻想》系列从诞生至今已经走过了 35 年的时间，作为 2017 年吉尼斯世界纪录认证系列的“数量最多的角色扮演游戏系列”，说是 SE 旗下最重要的 IP 应该没什么争议。它的 IP 联名产品同样是多如牛毛，其中仅是时尚界合作名单就能列出一大堆。</p><p>最新的优衣库 X《最终幻想》联名 UT 已经出现在大街小巷，单件售价 99 元维持了一贯定价并不算贵，但我个人认为这次联名也没有跳脱出 UT 只是简单印花缺乏设计感的“旧疾”。当然说归说买归买，谁能拒绝可爱的拉拉肥呢？更何况购入两件以上还可获得特典扑克一套。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475278LmA.png" referrerpolicy="no-referrer"></p><p>除了相对平价的优衣库联名，《最终幻想》也曾和诸多奢侈品品牌有过合作。2012 年 SE 就曾为普拉达（Prada）设计了春夏男装系列。当年 4 月 12 日作为系列 25 周年庆典的一部分，CGI 拍摄（计算机生成动画）的《最终幻想》联名出现在了 Arena Homme+ 杂志上。这期杂志包括一份 12 页的专题，其中《最终幻想 13-2》的诺艾尔·克莱斯、萨兹‧卡兹罗伊、斯诺·维里耶、霍普·埃斯特海姆等角色逐一亮相。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-39419534752858qs.jpg" referrerpolicy="no-referrer"></p><p>此外 ，2016 年 SE 又找了国际知名品牌路易威登合作推出联名产品。《最终幻想 13》中的角色“雷霆”埃克莱尔·法隆（Lightning）为路易威登的春夏系列服装和手袋担任模特。“雷霆”出现在各种杂志广告中，从头到脚都穿着时尚服装，表演着她在游戏中的标志性动作，而且她还成为了社交媒体上路易威登的新代言人。可惜不出所料的这两个联名系列如今早已被炒成了天价，而且呈现出有价无市的态势，在各大交易平台看到的频率都非常低。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475291oyq.png" referrerpolicy="no-referrer"></p><p>如果你要问这些大牌联名值不值或者性价比如何，我肯定是回答不值。当然我们也不能贬低那些购买者，毕竟喜欢一个东西并且它在自己的消费能力之内，那就是千金难买爷高兴的事儿。SE 曾表示与时尚品牌的合作体现了公司在保持传统的同时寻求创新和革命性理念，以提供高质量产品的共同价值。或许作为大多数的玩家群体，我们的关注点还是放在《最终幻想 16》或者买件 UT 更为实在。不过如果你是富哥们儿，那，那买也不是不可以啦~</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475297f9P.jpg" referrerpolicy="no-referrer"></p><h4>卡普空的《街头霸王》系列总是不缺奇奇怪怪的联名</h4><p>卡普空作为老牌大厂联名的服装品牌同样非常多，不过上文聊完衣服相同的品类就先放一放。《街头霸王》不管是不是格斗游戏玩家一定都不陌生，这个 IP 的联动产品可谓是五花八门，音乐合成器、手表、滑板、充电宝、手机壳应有尽有。不过这次我想介绍的是一台游戏机，准确的说是街机。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475305kgL.png" referrerpolicy="no-referrer"></p><p>今年 4 月份陈冠希主理的潮流品牌 CLOT 宣布，同《街头霸王》游戏 IP 推出联名限量版复古家用街机，其中内置 12 款卡普空经典游戏，包括《街头霸王 II：冠军版》、《超级街头霸王 II Turbo》、《黑暗行者》等等。人民币 5580 元的售价谈不上便宜，但哪个游戏玩家不曾幻想过在家里摆一台街机，打造一个属于自己的游戏室呢？可惜限量版属于手慢无的稀缺货，这里依然不推荐大家加价购买，毕竟市面上可以选择的街机其实还蛮多的。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475312mcN.jpg" referrerpolicy="no-referrer"></p><p>无论现在你玩《街头霸王》摇杆党、手柄党还是键盘党，当初在街机厅接触这个 IP 的经历一定是童年最美好的回忆，而《街头霸王》那些奇奇怪怪的联名，或许就是这个 IP 在今天茶余饭后送上的乐子。2020 年《街头霸王》曾和摩托车品牌 SYM（三阳）合作过联名产品，推出了 DRG 158 和 FNX 125 的新款车型。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475318I4S.jpg" referrerpolicy="no-referrer"></p><p>看到这儿哪位要问了，这联名怎么看着就只换了个颜色啊......对于这个问题我想说你是对的，而且联名圈有太多太多这样的先例，我就不一一点名了。没诚意或者说割粉丝韭菜的联名，究竟能不能起到破圈和营销的目的，我并非专业人士不过多评价。但作为玩家看着那些犹如智商税一般的产品，哪怕不买仍会感觉自己的热情被无端消耗掉。所以对于联名产品，各位朋友还是要保持理智切莫冲动消费。</p><h4>结语</h4><p>总的来说，我并不讨厌 IP 联名，如果你能将玩家喜欢的游戏 IP 放在他们需要的其他商品上，并且整体设计得十分出色，我想哪怕稍微贵一些很多玩家也是愿意购入不会反感的。但如果只是劣质的贴图换色消费情怀，那这所谓的联名也就成了一场彻彻底底的商业营销，最终影响游戏 IP 本身的价值。</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475327kYX.jpg" referrerpolicy="no-referrer"></p><p>好啦，那今天就聊到这里。一家之言难免片面且狭隘，不知道你对这些游戏 IP 联名产品持什么态度呢？期待在评论区看到大家的分享和交流。如果觉得我们做的不错，可以点赞关注。当然，也欢迎提出意见和建议，让我们下篇文章再见。啵啵~！！！</p><p><img class="fr-fin fr-dib" src="https://hive.indienova.com/farm/blog/2022/05/u-3941953475334Wvb.jpg"
