<h4>CiGA Game Jam</h4><p>华语圈最大的线下 Game Jam 活动——CiGA Game Jam 开始报名啦！</p><p>无论你是在游戏公司的打工人，还是游戏开发相关专业的高校学生，亦或是自学想要开发游戏新人，只要你对编程、互动设计、叙事探索、美术设计、音效等领域有一技之长和自己的想法，都欢迎加入到 CiGA Game Jam 的活动中来！</p><p><img class="fr-dib" src="https://hive.indienova.com/farm/article/picture/2021/06/pic-16246197130479714u6.jpg" data-id="22761" data-result="success" referrerpolicy="no-referrer"></p><p>这不是一场比赛，而是一群热爱游戏开发的小伙伴们的聚会！用 48 小时来共同挑战游戏极限开发，来结识更多的 Game Jammer！</p><h4>什么是 CiGA Game Jam？</h4><p>Game Jam 是一种特定的活动形式，在 Game Jam 中开发者要求在规定的时间内（通常为 48 小时），根据现场给出的主题，完成一款包含核心玩法的游戏 Demo。Game Jam 活动不仅要求开发者们在有限时间内迸发出创意，同时还是一次与时间的比拼。</p><p>CiGA Game Jam 是华语游戏圈最大的线下 Game Jam 活动，2020 CiGA Game Jam 共有 2000 多名开发者报名，顺利开发完成了超过 330 款游戏 Demo。</p><h4>12 大线下站点报名开始！</h4><p>由于疫情的原因，2020 CiGA Game Jam 只能通过线上的形式进行。尽管线上的形式让组队没有了物理地域的限制，却也少了几分仪式感。更多的开发者还是希望能够在线下组队、交流、互相试玩进行探讨，结识更多的新朋友。</p><p>为了满足广大开发者对于线下站点的需求，我们对线下站点进行了公开招募。最终，2021 CiGA Game Jam 共有 11 大线下站点共开发者报名。</p><p><img class="fr-dib" src="https://hive.indienova.com/farm/article/picture/2021/06/pic-162461980972198110h.jpg" data-id="22762" data-result="success" referrerpolicy="no-referrer"></p><p>在此感谢各地合作伙伴对 2021 CiGA Game Jam 的大力支持，其中有大学生游戏开发联盟、凉屋游戏、帕斯亚科技&帕斯亚学院、椰岛游戏、铃空游戏、武汉独立游戏联盟、云南独立游戏联盟、之幻网络、小白工作室、叽咪叽咪、游戏茶馆、墨舞文化这样的老朋友，也有幻步游戏、海彼游戏、心光流美、cocos 引擎、AF 国际艺术教育这样的新朋友。（感谢排名不分先后）</p><p><img class="fr-dib" src="https://hive.indienova.com/farm/article/picture/2021/06/pic-16246198259539827le.jpg" data-id="22763" data-result="success" referrerpolicy="no-referrer"></p><p>同时也感谢游戏便当给本次活动报名提供技术支持。</p><p>在大的疫情防控背景下，各大线下站点都将只开放有限的参与名额，报名由各地组织方进行审核。整体活动将以线上+线下的方式进行，在线上我们将开设面向所有人的线上站点，不方便到场和未能加到线下的开发者都可通过线上站点来参与本次 Game Jam, 所有环节在线上均有完整体验。</p><h4>2021 CiGA Game Jam 活动安排</h4><p>报名时间：<strong><span style="color: rgb(216, 55, 98);">即日起至 7 月 7 日</span></strong></p><p><br></p><p>活动流程：</p><p><strong>1、活动官网报名</strong></p><p>报名链接：<a href="https://www.ciga.me/cgj" rel="noopener noreferrer" target="_blank">https://www.ciga.me/cgj</a><br>或 <a href="https://www.youxibd.com/v2/gamejam/ggj2021" rel="noopener noreferrer" target="_blank">https://www.youxibd.com/v2/gamejam/ggj2021</a></p><p>点击页面右上角立即报名并登记个人信息，线上站点不设置观众和媒体身份报名，注意按需求填写信息，报名成功可按提示加群，注意有身份验证。</p><p><br></p><p><strong>2、线下站点报名</strong></p><p>线上报名成功后，有线下站点的信息可查看，注意查看线下站点的地址、开放时间、设施、开发说明等，一般线下站点参与通过后是可以到线下站点进行实地开发，自带设备，以及在最后开发完成后到现场进行演示（关于证件在线下站点领取还在评估中），请注意线下站点有人数上限，以站点组织者审核为主，原则上只接受同城或者附近距离的开发者；</p><p>根据实际疫情的情况和管控，线下站点会及时灵活做出调整，请计划参加线下站点的开发者给予理解。</p><p><strong>请注意：以上注册将确保你可以全程参与到这次活动中，包括本地活动页面的信息收录和最后的作品上传，感谢理解</strong></p><p><br></p><p>线下站点列表：</p><p><img （线下展示）</p><p>今年的现场说明、分享视频、主题发布等，在线上的站点都会有全部的内容进行，各存在的线下站点也会进行同样的主线信息发布，我们会做一场专门的主播来做相关信息分享和说明以及本次神秘主题的发布，具体直播的流程、时间和链接将在活动开始前发布。</p><p><img class="fr-dib"
