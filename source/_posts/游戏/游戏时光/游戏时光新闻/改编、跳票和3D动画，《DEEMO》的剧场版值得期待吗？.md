<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　昨天我在网上冲浪的时候知道了《DEEMO》的剧场版动画放出了新的 PV 和正式的电影标题，不过我忙于补觉和写其他东西就没有太关注。今天一看站内还没有相关的报道就找时间补了篇<a href="https://www.vgtime.com/topic/1118719.jhtml">新闻</a>。</span><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172332797_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　<a href="https://www.vgtime.com/topic/1070562.jhtml">《DEEMO》确定要推出剧场版动画是2019年底的事情</a>，当时制作组放出了一段 PV，说是 2020 年内播出。我一直以为是这部电影没选择在国内上映，网上暂时还没有的资源，结果是人家根本还没上映，甚至新 PV 连个大致完成的时间都没有说，直接在最后打出“Coming Soon”。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172356529_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　正好有朋友看了预告，觉得这动画跳票这么长时间是不是不太稳啊。再加上这片子不是传统的手绘动画，而是选择了 3D 动画，没有游戏的那种味道了，就私下问我《DEMMO》这个剧场版到底值不值得期待。正好我对这部作品也很感兴趣，就单独写篇文来说一下。</p><p>　　为了照顾到不怎么玩音游的朋友，我先简单介绍一下《DEEMO》。《DEEMO》是雷亚在2013年为移动平台制作的一款下落式音游，我个人是比较喜欢雷亚和他们家音游的，毕竟现在还在出高质量单机音游的游戏厂商也不多，基本上没得选。因为游戏人气很高，《DEEMO》后续还登陆了主机和掌机平台。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172408982_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　在游戏里，一个小女孩从天而降，掉进了一个神秘的地底世界。如果要送小女孩回家，就需要玩家演奏音乐，让大厅里的树不断生长，最后达到足够的高度，让小女孩回家。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/21032917244646_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　动画方面我还是那个老观点，做得好的 3D 动画并不会比 2D 动画表现得差。很多朋友现在一见到 3D 动画就会下意识产生排斥感，这是有很多原因的，特别是一些制作粗糙的 3D 动画确实在不好的方面打响了名声。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172458684_u93176.gif" class referrerpolicy="no-referrer"><br></p><p>　　再加上现在 3D 用得好的作品相较来说不算多，这就让观众产生了“3D 动画大概率在画面上表现会比传统手绘动画要差”的印象。</p><p>　　不过《DEEMO》的画面并不复杂，而且大家如果要去电影院看这部电影，肯定不是为了画面去的，而是奔着《DEEMO》的故事和音乐，所以综合下来其实选择用 3D 制作的影响不是很大。</p><p>　　而且真要钻牛角尖的话，2019 年推出的《迪默 重生》已经选择了全 3D 画面，你完全可以当做这部动画是在《迪默 重生》的基础上制作的。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172513292_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　本次负责动画制作的是老牌动画公司 Production I.G，近几年他们比较有名的作品有《强风吹拂》和《舞动青春》，实力是肯定没得说的。同时这两部动画很棒，这里也推荐各位去看一看。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172526654_u93176.gif" class referrerpolicy="no-referrer"><br></p><p>　　Production I.G 的实力是绝对不差的，剩下的就看有没有尽全力制作和雷亚经费给得够不够了。</p><p>　　比起画面，我反倒是担心的是《DEEMO》的剧情能不能撑得起一部剧场版动画的时长。《DEEMO》的剧情基本上都藏在各种小细节里，而且核心剧情相当简单，短短几句话就能概括完毕，制作组如何安排剧情就成了难题。</p><p>　　雷亚是尝试过扩充《DEEMO》故事线的，他们出过《DEEMO》的小说，我没有看过所以这部分不好评价。但据看过的朋友反映，这小说写得不太行。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172550753_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　虽然大家都说《DEEMO》的故事很感动，但游戏原作本身更多的是展现一个意境，让玩家去感受那种氛围。如果换一种载体，就很难把这种氛围完美还原出来。</p><p>　　我建议大家最好放低对动画的预期，最差的情况是大家去电影院看了时长一个多小时的大型 MV。</p><p>　　值得一提的是虽然官网只写着本作的主题曲由梶浦由記创作，但根据 Fami 通的报道，除了主题曲外她还将负责剧中伴奏的制作，高质量音乐在《DEMMO》剧场版动画中也不会缺席。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172600905_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　说完了动画和剧情，我们来聊聊动画的延期。这次动画很明显是受到了疫情的影响，从 2019 年底的预告视频里已经可以看出主要角色的 3D 模型已经制作完毕，如果按照原计划要在 2020 年上映的话，说明很多前期工作在当时已经完成了。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/21032917261051_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　按理来说，这些工作都做完了，哪怕是疫情拖慢了现场的工作，制作压力应该会小很多才是。但非常奇怪的是动画在跳票了这么久之后，最新的预告并没有给出一个确定的上映时间。</p><p>　　再结合<a href="https://deemomovie.jp/hans/index.html">动画官网</a>的时间线，有很大概率是整个动画企划在疫情最严重的的时候推倒重来了。动画的时间线从 2020 年 11 月开始，一直更新到昨天的 AnimeJapan，如果不告诉你动画的第一段 PV 是在 2019 年底公布，你可能会以为《DEEMO》的动画是在去年年底才开始启动。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172620423_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　从 2020 年 11 月建立官方推特账号，新推出《DEEMO》的漫画；2021 年 2 月 22 日年宣布参加 AnimeJapan；2021 年 2 月 26 日宣布漫画的连载日期；2021 年 3 月 27 日漫画正式开始连载。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172628779_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　到昨天 3 月 28 日 AnimeJapan 活动开始，并在当天公布了动画的正式日文标题和最新的 PV。感觉一切都在顺利推进中，除了那个令人不安的“Coming Soon”。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172638996_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　如果是正常因为疫情影响而延期，那我估计《DEEMO》的剧场版大概在今年夏季就能上映。但现在整个企划看起来就像是去年年底才重回正轨，这就增添了很多不确定性。</p><p>　　总结一下，如果你没有玩过《DEEMO》，但听很多人的安利想要从这个剧场版动画开始入坑，那我的建议是适当降低预期。《DEEMO》的故事温柔且悲伤，但不一定能够通过动画完美表现出来。如果不是那种完全碰不了音游的玩家，我推荐你还是老老实实从游戏本体开始入手吧。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/210329172649439_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　至于《DEEMO》的老玩家，反倒会比较容易包容制作上的各种缺陷。毕竟我们已经体验过了《DEEMO》最好的部分，想看动画无非是当年那份感动的延续，能看到角色动起来就已经很满足了。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/29/21032917270647_u93176.jpg" class referrerpolicy="no-referrer"><br></p></div>
