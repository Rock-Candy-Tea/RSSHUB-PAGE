<div class="topicContent front_content"><p>　　Atlus 此前宣布了“P4”衍生格斗游戏《女神异闻录4 无敌究极后桥背摔》重制版将于 2022 年 3 月 17 日登陆 PS4/Switch/Steam 平台的消息，游戏将支持简体中文，官方现公布了本作收录要素 。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227125936668_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p>　　《女神异闻录4 无敌究极后桥背摔》是《女神异闻录4 午夜竞技场》（P4U）的续作，于 2013 年登陆了 PS3/Xbox 360 和街机平台， 收录了包含《女神异闻录3》“特别课外活动部”长大后姿态的成员们共计 37 名角色，重制版以街机 Ver.2.5 版本为基础 ，收录几乎所有付费DLC，和只能通过杂志特典获得的角色新增颜色以及新增 BGM，并提升了画面分辨率，“GOLDEN ARENA模式”和“挑战模式”也是街机最新版，让游戏变得更加耐玩。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130335258_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130335803_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>

 剧情模式

</h4><p>　　《女神异闻录4 无敌究极后桥背摔》将收录三个故事篇章，分别为“P-1 GRAND PRIX”“P-1 CLIMAX”“ADACHI”，即使动作苦手玩家也无须担心，切换为“自动模式”就能让系统自动操控角色进行对战，从而安心体验剧情。 </p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/21122713032347_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130323242_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>

 EpisodeⅠ：P-1 GRAND PRIX 

</h3><p>　　讲述 2012 年发售的首部人格面具格斗游戏《女神异闻 录4 午夜竞技场》的故事。 在解决《P4G》事件 2 个月后，深夜电视再次出现。 在深夜电视中突然开始了以八十神高中为舞台的格斗秀！ 与主办者“小熊”十分相像的自称学生会长的神秘美少女以及《P3》中的暗影行者也出现在这场格斗秀 中。 “自称特别搜查队”在注意到异常事态后，再次向深夜电视的谜团发起挑战！

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130058481_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>

 EpisodeⅡ：P-1 CLIMAX

</h3><p>　　本章内容是《女神异闻录4 无敌究极后桥背摔》中新增的故事掌机，即“P-1 GRAND PRIX”的后续故事。 突如其来的红雾与暗影军团出现在八十稻羽市。率领着暗影军团，并自称是幕后黑手的少年“皆月翔”阻 拦在主角一行人面前。 本篇章的故事以现实中的八十稻羽市为舞台，并从 《P4G》和《P3》不同角色的视角进行叙述。 “幕后黑手”的真实身份究竟是…… 

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130107581_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>

 Episode Ⅲ：ADACHI 

</h3><p>　　本章实际上是《女神异闻录4 无敌究极后桥背摔》的一个付费剧情 DLC，《P4G》中的重要角色足立透将加入到游戏中，在被主角一行人打败后，本应被关押在监狱中的他为 何再次现身？ 在该剧情线中玩家能够从足立透的视角深入了解主角们无法得知的背后真相。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130114106_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>

黄金竞技场模式

</h4><p>　　“黄金竞技场”模式是一个融合了 RPG 玩法的模式，玩家可以让角色积攒经验值升级，为他们分配属性点数，按自己的喜好培养角色，从 200 多种技能中进行组合搭配，最终击破地下城的最终层。 游戏中玩家可以选择提供支援的辅助角色，发展与辅助角色的“社群”关系，让辅助角色在战斗提供独特的帮助。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130142883_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/21122713014249_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>其它模式

</h4><ul><li>街机模式：以通关为目标与 CPU 控制的角色进行对战的模式，能够体验街机版的故事。 </li><li>高分模式：以获得高分为目标，逐个击败 CPU 控制角色的模式。对战分数会显示在在线分数排行榜上。 </li><li>联网模式：与朋友进行对战，或与实力相近的玩家进行对战的模式。可以在分数排行榜上比拼对战成绩。 </li><li>Versus 模式：在本地进行对战的模式。不仅能够与朋友进行对战，还能与 CPU 进行对战。 </li><li>鉴赏模式：鉴赏游戏中的插画、BGM、声音等内容的模式。推进游戏便能解锁其中内容，以 100％ 完成率为目标努力吧！ </li><li>教学模式：讲解游戏基本操作与系统的练习模式。即使是初次游玩也能安心上手。 </li><li>训练模式：能够自由练习指令技与连招的模式。可以通过详细设定模拟出各种情况并进行针对性练习。 </li><li>挑战模式：可挑战针对不同角色设置的“任务”的挑战模式，可以在挑战过程中自然地学会各种连招。 </li></ul><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130234102_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130239629_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130243253_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/27/211227130246206_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>
