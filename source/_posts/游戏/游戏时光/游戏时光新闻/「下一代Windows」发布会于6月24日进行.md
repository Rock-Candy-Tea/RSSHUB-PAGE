<div class="topicContent front_content"><p>　　微软宣布，将于北京时间 6 月 24 日 23:00 举行公布「下一代 Windows」的线上活动。Windows 桌面版的 UI 将全面革新。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/03/210603103204593_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　Windows 的 UI 改造已经进行了一段时间了，代号为“太阳谷”（Sun Valley），该更新预计将有一个新的开始菜单和任务栏布局、图标、音效、应用程序设计和动画。</p><p>　　知名媒体 Windows Central 推测，微软可能会把这个大更新作为一个契机，为 Windows 取一个新名字。可能叫做“Windows 11”，或仅仅是“Windows”。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/03/210603103253329_u77470.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>  Windows Central 制作的 logo  </figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　六年来，Windows 10 一直是 Windows 的最新版本，而且基本保持不变。这次更新会不会改名，或有更多改名之外的变化，还有待确认。</span><br></p><p>　　近来，微软取消了 Windows 10X，该版本曾是基于 Windows Core OS 构建的现代操作系统。取消后，微软宣布将把 Windows 10X 的优点带到完整版的 Windows 中，这可能会是 6 月 24 日发布内容的一部分。</p><p style="text-align: right; ">来源：<a href="https://www.windowscentral.com/microsoft-will-unveil-next-generation-windows-june-24">Windows Central</a></p></div>
