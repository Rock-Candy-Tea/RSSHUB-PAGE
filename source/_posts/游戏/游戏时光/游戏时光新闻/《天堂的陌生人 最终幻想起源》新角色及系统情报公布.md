<div class="topicContent front_content"><p>　　Square Enix 旗下交由 Team Ninja 开发的动作冒险游戏《天堂的陌生人 最终幻想起源》将于 2022 年 3 月 18 日发售，官方今日公布了最新情报，其中包括三名角色的人物简介和游戏系统、场景等情报。  <br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115131600_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>

新角色

</h4><h3>

苏菲亚（CV：浅野麻由美）

</h3><p>　　一名女战士，和杰克一行人相同，对于打倒混沌有着强烈信念，因此成为了这场旅途的伙伴。不知为何，她熟悉世间的各种事象，甚至还用这些知识拯救了队伍。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115137304_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>

阿斯托斯（CV：上田燿司）

</h3><p>　　居住在西边王城的黑暗精灵之王，知晓水晶的下落。虽然看起来秀丽端庄，但其实有些病态，并且他似乎认识杰克一行人。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115146334_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>

提亚马特（CV：？？？）

</h3><p>　　风之混沌，他会利用修长的脖子和尾巴直接攻击，或用风属性的“袭卷狂风”和“暴风”进行攻击。从多张嘴里呈放射状喷射而出的突袭“巨量爆发”威力巨大。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115151189_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>

过场事件

</h4><p>　　在西边王城等待著一行人的正是黑暗精灵之王阿斯托斯，他向杰克一行人展示了一张近未来风格的地图，似乎正在提出某些建议。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115248155_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p>　　一行人在阿斯托斯的指点下到达目的地，等待他们的竟是风之混沌。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115254122_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p>　　苏菲亚似乎和杰克一行人一样持有黑水晶......？

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115258800_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>

战斗系统

</h4><p>　　战斗时角色能在两种职业中随时切换，且可根据职业的组合制定各种战术。角色的职业有两种，一是“剑士”“魔术士”之类的基础职业，二是通过锻炼基础职业而解锁的进阶职业。如果继续培养进阶职业，也许还能解锁更高阶的职业。玩家的职业能力可通过成长系统“职业树”来解锁，职业点数则可通过击败敌人获得。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115403319_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>

基础职业“大剑士”

</h3><p>　　杰克的初期职业，可装备一把擅长防御和蓄力攻击的大剑。固有动作“回旋连斩”是挥舞大剑，横扫大范围的旋转攻击。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115416912_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>基础职业“魔术士”

</h3><p>　　该职业的武器为短杖，可使用基本的属性攻击，且可以根据咏唱时间来强化魔法，比如“火焰”⇒“中火焰”⇒“大火焰”等。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115420864_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3><span style="letter-spacing: 1.3px;">基础职业“格斗士”</span></h3><p>　　专门使用勾爪和拳套等格斗武器进行近距离作战的职业，可以通过固有动作“爆裂拳”连续打击敌人以提高拳气。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115425414_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>

场景

</h4><h3>

西边王城

</h3><p>　　在这座古城里，因争斗而留下的伤痕历历在目，这里似乎与某位善于谋划且暴虐至极的君主渊源颇深。由于巨大的弓箭会射向杰克一行人，因此前行时必须注意躲避。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115556764_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>水晶蜃楼

</h3><p>　　此建筑物由水晶般的材料建造而成，装潢神秘且奇特，与名为“幻象塔”的建筑有着神似之处。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115603369_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>

浮游城

</h3><p>　　正如其名，這是一座高耸入云的城堡。在城堡里可以看到许多先进文明的痕迹，比如机械样式的机关等，据说最深处还摆放着“风之水晶”。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/15/211215115608435_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>
