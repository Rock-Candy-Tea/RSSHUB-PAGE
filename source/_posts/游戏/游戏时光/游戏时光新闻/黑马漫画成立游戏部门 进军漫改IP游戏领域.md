<div class="topicContent front_content"><p>　　近年来，随着漫改游戏的大热，全球许多老牌漫画公司也纷纷涉足游戏领域以开拓新疆土。近日，北美知名漫画公司 Dark Horse Comics （黑马漫画）也搭上了游戏制作的顺风车，成立了一个名为 Dark Horse games 的游戏部门。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/02/210602152059330_u7414.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　据其</span><a href="https://games.darkhorse.com/#contact" style="background-color: initial; letter-spacing: 1.3px;">官网页面</a><span style="background-color: initial; letter-spacing: 1.3px;">介绍，该部门的工作主要围绕游戏及数字业务展开，致力于通过与游戏开发商合作，将公司旗下的漫画 IP 转变为横跨主机、PC、手机以及云端的电子游戏项目。不仅如此，该部门还将打算设立内部工作室，将一些知名度不太高的 IP 打造成含游戏在内的多媒体项目。</span><br></p><p>　　该部门负责人 Johnny B. Lee 在接受媒体采访时表示：“我们有着诸如《地狱男爵》这样的经典作品，（因此）在游戏制作以及开展业务合作方面还是大受欢迎的。如此一来，我们将会与合作伙伴共同评估故事 IP 与角色宇宙是否能够完美契合其内部的游戏开发。”</p><p>　　“从我多年来与不少 3A 游戏开发者打交道的经验来看，游戏开发人员更倾向于创造游戏的核心玩法，再与 IP 进行巧妙融合，而不是生硬地将 IP 套进游戏之中。这是我们尤其注重的一点。”

Lee 补充道。</p><p>　　另据

Lee 介绍，黑马游戏目前正在与独立及 3A 游戏开发商进行深入洽谈，如果进展顺利，玩家们将可在明年第一季度之前开始听到基于黑马旗下 IP 的新作消息。</p><p style="text-align: right;">来源：<a href="https://www.gamespot.com/articles/dark-horse-comics-starts-a-gaming-division/1100-6492241/?ftag=CAD-01-10abi2f">Gamespot</a></p></div>
