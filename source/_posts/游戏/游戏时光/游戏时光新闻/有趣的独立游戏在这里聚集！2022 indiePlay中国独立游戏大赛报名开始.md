<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　作为中国最高品质的独立游戏大赛，indiePlay诞生于2015年，今年已经来到了第8届。在过去的8年时间里，indiePlay见证了中国独立游戏的成长，每一年的活动都在持续的成长和拓展。</span><br></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/22050616234180_u292926.png" title class referrerpolicy="no-referrer"><br></span></p><p>　　大赛得到了来自indienova、IGDA、UGADP以及众多行业媒体的支持。同时今年游戏动力VGA也作为独立游戏深度宣发合作平台加入到活动中。</p><p style="text-align: center;"><img style="width: 350px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162408359_u292926.png" class referrerpolicy="no-referrer"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162413423_u292926.jpg" title referrerpolicy="no-referrer"><br></p><p>　　游戏动力成立于2018年，是目前国内最具影响力的游戏媒体之一，秉承“让更多人体会到游戏的乐趣”的理念，游戏动力以用户需求制作内容，主要涵盖PC游戏、主机游戏、手游、动漫、电影等文化领域。内容形式有资讯、评测、时评、攻略、视频实况、VLOG等。同时游戏动力还有一个活跃度极高的用户社区（网站&APP）。</p><p style="text-align: center;"><img style="width: 350px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162427505_u292926.png" class referrerpolicy="no-referrer"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162432195_u292926.png" title referrerpolicy="no-referrer"><br></p><p>　　我们希望能够通过indiePlay，发现更多优秀的国产游戏，也让海外的优秀独立游戏作品有更好的对于中国市场的展示渠道，基于CiGA多年来在独立游戏领域的耕耘和积累，依托indiePlay这个倡导纯粹的展示交流平台，将优秀的国产游戏推荐给国内甚至全球热爱游戏的玩家，同时介绍更多有趣和优质的海外游戏给到国内的玩家。随着今年游戏动力作为特别平台合作伙伴加入进来，也将在传播的资源和内容体现方面给到开发者更多的支持。</p><p style="text-align: center;"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162449705_u292926.png" title class referrerpolicy="no-referrer"><br></p><p>　　我们相信独立游戏是充满创意的、探索的，题材是多元的、丰富的。在每一年的报名中都能收到许多充满个性、特色鲜明的作品，从中能够明显的感受到开发者对于游戏的体验追求、玩法上的探索或是明确的自我表达。我们也一直认为，独立游戏是在当前游戏产业中不可或缺的先锋力量，对于新的游戏体现形式和表达不断进行着尝试，让更多有生命力的创意者可以用游戏的形式进行创作，打开一扇又一扇新体验的大门，使得整体的游戏行业不断的焕发着活力，而又透露出更独特的魅力。</p><p style="text-align: center;"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162502518_u292926.jpg" title class referrerpolicy="no-referrer"><br></p><p>　　最近几年，学生开发者的报名数量不断提升，在他们的作品中能感受到更年轻和奔放的活力。CiGA中国独立游戏联盟通过与UGDAP大学生游戏联盟的合作，让更多学生开发者加入到独立游戏的开发中来，通过活动给到优秀学生开发者鼓励与正向的引导，同时也能通过indiePlay这个平台让学生开发者走出校园有更多接触实际游戏行业，以及和同好交流的机会。</p><p style="text-align: center;"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/22050616251547_u292926.png" title class referrerpolicy="no-referrer"><br></p><p>　　同时，随着indiePlay在海外的知名度不断提升，这两年海外作品的报名数也不断提升，虽然目前海外的独立游戏作品还仅能参与“最佳海外”的奖项，但丝毫不影响更多工作室报名参加的热情。</p><p style="text-align: center;"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/2205061625286_u292926.png" title class referrerpolicy="no-referrer"><br></p><p>　　2022 indiePlay中国独立游戏大赛报名期为2个多月，在7月报名截止后是2个月左右的评审期，年中CiGA Game Jam活动中的优秀作品也将进入大赛的评审，通过评审后公布各个奖项的入围名单。最终的入围作品将在年底WePlay文化展的现场进行展示与试玩。每一年WePlay现场的indiePlay入围区都在扩张，也以更好的形式呈现给现场的观众，届时有全国最热爱游戏的玩家群体，以及来自全国乃至海外的独立开发者群体，对于独立游戏开发者来说，将是展示自己作品并与玩家和行业人士交流绝好的机会。</p><p style="text-align: center;"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162619980_u292926.png" title class referrerpolicy="no-referrer"><br></p><p>　　去年由于疫情原因，很遗憾 WePlay 5周年未能以一个完整的形态和大家见面，不过去年我们依旧保留了indiePlay线下颁奖环节，并请来坂口博信、神谷英树、稻叶敦志、小高和刚等重量级嘉宾进行开奖。</p><p style="text-align: center;"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162628308_u292926.png" title class referrerpolicy="no-referrer"><br></p><p>　　依托于CiGA多年和海外的合作积累，每一年indiePlay的颁奖嘉宾都是大牌云集，能够从这些海外知名的游戏开发者手中接过奖杯，对于独立开发者来说是一种鼓励，更是一种传承。</p><p style="text-align: center;"> <img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162635136_u292926.jpg" class referrerpolicy="no-referrer"></p><p>　　在2021年indiePlay大赛，共有来自2364名开发者的311款游戏报名参加活动，今年期待有更多的游戏加入到indiePlay的大家庭中来！</p><hr class="vg_hr" contenteditable="false"><h3>报名方式：</h3><p>　　访问<a 1.3px;">（更多详细流程请参照页面的提示和说明）</span></p><p>　　温馨提示，如果您的游戏在某些方面有特别的设计，如音乐、故事等，建议在填写介绍资料的时候对这些点重点标明，将有助于评委对参赛游戏有更进一步的了解。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162716610_u292926.png" class referrerpolicy="no-referrer"><br></p><p>　　2022年CiGA中国独立游戏联盟的各项活动已经全面启动，与各类海外合作活动的报名全面展开中！欢迎关注CiGA公众号获取更多信息。</p><p style="text-align: center;"><img style="width: 350px;" src="https://img01.vgtime.com/game/cover/2022/05/06/220506162725611_u292926.png" class
