<div class="topicContent front_content"><p>　　二次元 roguelike 策略手游《少女前线:云图计划》今日全面开启公测，玩家将扮演该计划的负责人“教授”，参与到这场决定失联人形们生死存亡的冒险中。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144207945_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　作为少女前线 IP 衍生新作，《少女前线：云图计划》在剧情上致力于补齐少前的世界观。如果你是前作的指挥官，那么你将会通过本作更加了解与你并肩作战的人形们曾经的故事。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144405110_u93176.jpg" referrerpolicy="no-referrer"><br></p><p>　　玩家们需要在这片荒芜的云图世界搜寻幸存的“流亡者”人形，组建起自己的战术小队。在云端的末世背景中建立起属于自己的“绿洲”家园。这片“绿洲”将由你亲手打造成一座繁荣立体的高科技城市，并会随着现实时间同步，呈现出黎明、白日、黄昏、深夜等景象。<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/21092314423153_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　音乐制作团队「Vanguard Sound」创作的游戏原声，将伴着教授的思绪交响于光怪陆离的云图世界。如果在游戏里听不过瘾，游戏 OST 已经随公测同步上线 <a href="https://music.apple.com/cn/album/1584075220">AppleMusic</a> 了。<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144111713_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　游戏结合了模拟养成、即时策略、Roguelike 等元素，从队伍组建到战前准备，再到即时战斗部分，都有着自由搭配、多线选择、丰富随机性的游戏体验，即使用同一阵容攻略同一关卡也会有不同的感受。</p><p>　　在角色培养方面，每个角色都有着相同高度的成长天花板，即使你抽到初始一星的人形，也可以通过“心智扩容”系统培养升级，最终成为与初始三星人形同一高度的五星角色。</p><p>　　每个人形都有着自己独特的技能特性，培养成型后，所有人形都能发挥出自己的特长，在战场拥有一席之地。这样就保证了每个人形都不会仅通过数值差异进行区别。<span style="background-color: initial; letter-spacing: 1.3px;">玩家可以将这些拥有不同技能与特性的人形自由搭配，产生出无多种战术可能性。</span></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144502995_u93176.jpg" class referrerpolicy="no-referrer"><br></span></p><p>　　“函数卡”是《少女前线：云图计划》roguelike 玩法的核心体现，每当战斗结束，都会获得随机函数卡。函数等同于关卡内有效的 buff，在攻略关卡过程中，选择并进阶适当的函数卡，或收集函数卡，形成函数连锁反应，都能大幅增强小队的战力。</p><p>　　进入战场后，玩家可以自行决定前进路线，在对战的冲突区、获取函数的交易区、进行补给的恢复区，以及情况不定的异常区、诊断区中进行选择，根据战况来搭建一套完整的 build 通关。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144523928_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　你遭遇的敌人也可能具有强力的协议帮助。协议是对战中的强力 buff，根据类型对敌我双方都有效果。你可以在攻略关卡过程中，通过制定相应的行进路线，对其协议进行削弱；也可以不考虑敌人的增强，只专注于增强自身。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144600916_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: left;"><span style="background-color: initial; letter-spacing: 1.3px;">　　战斗前的排兵布阵，能够让你的队伍分布更加合理，提高队员存活几率，增加对敌击杀效率。</span><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/21092314462049_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　战斗地图上的特殊格子还具有独特的地形效果，能同时影响敌方与我方，产生强化或弱化效果。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144629997_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　战斗过程中，根据场上局势及时使用战术技能，也能达到出其不意的战术效果。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144645991_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　角色奥义技能能够带来强大的战术收益，甚至一举扭转整个战局。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144707655_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　除此之外，场外援助能让替补的队员与场上的队员一同战斗。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144724796_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　3D 化的宿舍系统则继承了少前宿舍的风格与 DIY 要素，玩家还可以通过虚拟摇杆控制自己在宿舍中行走，亲身体会云图世界的魅力。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/23/210923144305208_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　各位教授，你们准备好在云图世界里大展身手了吗？<br></p></div>
