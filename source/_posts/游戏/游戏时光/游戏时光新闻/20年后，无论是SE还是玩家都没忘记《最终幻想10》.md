<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　几天前，在《Fami通》的《最终幻想10》20 周年开发者访谈中，<a href="https://www.vgtime.com/topic/1129041.jhtml">野村哲也和鸟山求称《最终幻想10-3》已经有了剧本大纲，未来开发新作的可能性并不为零</a>。SE 能记得《最终幻想10》系列当然让人开心，但提前公布的消息让今天这个《最终幻想10》发售 20 周年纪念日显得有些冷清，好在这并不会影响玩家们早就准备好的庆祝活动。</span></p><p>　　没有官方组织，玩家间也没有互打招呼，一场网络庆生会就这样开始了。</p><h4><span style="background-color: initial; letter-spacing: 1.3px;">画笔动起来</span></h4><p>　　看来有不少玩家在 20 年后成为了一个大触，推特上《最终幻想10》20 周年的贺图简直不要太多。网友<a href="https://twitter.com/Emy_san_arts/status/1416866468445265920">@Emy_san_arts</a>画了提达与尤娜相拥的场景，并配文：我在你怀里，《最终幻想10》周年快乐。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/19/210719183035714_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　这次为《最终幻想10》画贺图的大触们一般都会选择画提达与尤娜这对情侣，这就显得<a href="https://twitter.com/ayukichi_illust/status/1416934827496939521">@ayukichi_illust</a>的画比较特别，因为她在所有的《最终幻想》男主角里最喜欢的就是提达，所以只画了一个人。画师在进行创作时有时候比起随大流，会更喜欢忠于自己的欲望。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/19/210719183101347_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　也有一些比较有自己风格，表现欲望强烈的画师，像是这位名叫<a href="https://twitter.com/KatsudonRiku/status/1416904896385802242">KatsudonRiku</a>的网友，我隐约从提达的脸上看出了《刃牙》的味道。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/19/210719183131863_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　如果实在是不习惯上面的风格，也可以看看网友<a href="https://twitter.com/kitaya2017/status/1416779921406320642">@kitaya2017</a>的贺图，不仅把主要人物都画出来了，背后的召唤兽也很有魄力：</p><blockquote><p>　　无论玩多少次《最终幻想10》都会很开心，无论玩多少次《最终幻想10》都会伤心到哭泣，这是我最喜欢的作品。我很感激我们的相遇。</p></blockquote><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/19/210719183151892_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　网友<a href="https://twitter.com/okaybeex/status/1416939422537289730">@okaybeex</a>的画给人一种小清新的感觉，她在推文中写道：</p><blockquote><p>　　现在是日本时间的 7 月 19 日，也就是说今天是《最终幻想10》20 周年纪念日！这款游戏对还是个小屁孩的我来说意义重大，直到今天我成为大人后也是如此。过去 20 年间它一直是我生命中重要的一部分，生日快乐《最终幻想10》！！！我最喜欢的游戏！</p></blockquote><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/19/210719183215739_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/19/210719183228732_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　<a href="https://twitter.com/decafdraws/status/1416873157093257216">@decafdraws</a>的这张图就很厉害了，一股浮世绘的味道扑面而来：</p><blockquote><p>　　很难相信自那之后已经过去了 20 年，作为我接触的第一款《最终幻想》，《最终幻想10》对我来说一直意义非凡。这是我在 2012 年绘制的作品，灵感来自葛饰北斋的艺术作品。</p></blockquote><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/19/210719183253621_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　除了正经画图的，这次画贺图的朋友里也有创意优先的。这张图来自 LINE 表情包作者<a href="https://twitter.com/MOFloveMEAT/status/1416955253354688512">@MOFloveMEAT</a>，虽然画面上的并不是提达与尤娜本人而是 cosplay，但这张画实在是可爱得我不想错过。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/19/210719183326785_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　接下来这幅画看上去有点像小孩子的简笔画，但这是在庆祝《最终幻想10》20 周年，而不是画师大比拼，网友<a href="https://twitter.com/rimimi60771857/status/1416920568423534593">@rimimi60771857</a>的这幅画简单但又不失真诚：</p><blockquote><p>　　《最终幻想10》生日快乐，我以前打游戏都是靠着朋友和弟弟的帮助过关，作为游戏白痴的我第一款个人通关的游戏就是《最终幻想10》，好久不见了小尤娜(♡ >ω< ♡)</p></blockquote><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/19/210719183351618_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　最后这位朋友应该不属于画师，<a href="https://twitter.com/Ria_neearts/status/1416787085873360901">@Ria_neearts</a>使用的是 3D 模型，但两人在向日葵花田里展露笑颜的场景得到了很多网友的认可。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/19/210719183420821_u93176.jpg" class referrerpolicy="no-referrer"><br></p><h3 style="text-align:
