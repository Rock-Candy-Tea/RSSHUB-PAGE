<div class="topicContent front_content"><h3>内容更新：</h3><p>　　目前有不少网友分享了自己的看法，比较靠谱的有像本站用户苍白的正义分析的那样。在去年视频网站 Crunchyroll <a href="https://www.vgtime.com/topic/1131633.jhtml">为自家会员赠送 3 个月 PC 版 XGP 会员</a>活动期间，注册多个小号获取一定数量的激活码。</p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　再使用技术手段在一个账号上叠加使用多个会员激活码，最后购买升级服务转换成 Xbox Game Pass Ultimate 会员。</span><span style="letter-spacing: 1.3px;">当然，微软官方并未透露本次的处理细节，以上分析仅供参考，请勿当作本次事件的真实情况。</span><br></p><p style="text-align: center;"><img src="https://img01.vgtime.com/game/cover/2022/06/17/22061715285569_u93176.jpg" class style="width: 600px;" referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　此外，有的网友已经向第三方店铺方面进行了询问，有的店铺建议顾客联系微软客服，有的店铺称仍在讨论解决方案。如果有订阅服务出问题的玩家，建议尽快向店家反映情况。</span><br></p><p style="text-align: center;"><img src="https://img01.vgtime.com/game/cover/2022/06/17/220617152947852_u93176.jpg" class style="width: 600px;" referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/17/2206171538188_u93176.jpg" referrerpolicy="no-referrer"><br></p><p>　　目前微软方面还未放出更多消息，如事件有新的进展，我们会继续跟进报道。<br></p><hr class="vg_hr" contenteditable="false"><h3>原文：</h3><p>　　今天有不少玩家发帖表示，自己在凌晨收到了微软发来的邮件，称玩家所使用的 Xbox Game Pass 服务是通过非法手段获得的，违反了微软的服务协议，已将订阅服务取消。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/17/220617102704280_u93176.jpg" referrerpolicy="no-referrer"><br></p><p>　　在退款邮件中显示，用户为服务所支付的 39 阿根廷比索（约人民币 2 元）已通过原支付方式返还。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/17/22061710271198_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　好在微软除了退还已支付的费用外，目前还没有对这些账号进行其他处理。</p><p>　　基本上这次遇到问题的玩家，并非直接通过微软官方渠道订阅 Xbox Game Pass 服务，而是在第三方平台上低价（多为人民币 100 元左右）开通的服务。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/17/220617102726935_u93176.png" class referrerpolicy="no-referrer"><br></p></div>
