<div class="abstract">
<p>世嘉试图复制《堡垒之夜》的成功。</p>
</div>

                
<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　据彭博社记者望月崇报道，世嘉飒美公司正在开发高预算级别的 Dreamcast 主机经典《疯狂出租车》《涂鸦小子》重启作品，并试图复制 Epic Games 旗下《堡垒之夜》的成功。</span><br></p><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/04/19/220419164822179_u32815.jpg" class style="width: 700px;" referrerpolicy="no-referrer"><figcaption contenteditable="true" class style="display: block;">美版《涂鸦小子》《疯狂出租车》</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　这两部作品将作为世嘉此前公开的“超级游戏”计划的一部分，用来实现世嘉寻求长期收入与构建在线社群的愿景。《堡垒之夜》对此类游戏的运营有着深刻影响：免费游玩、跨多平台、主持赛事，在常规战斗以外，还包含载具、建造、社交在内等多种额外活动，用以激励玩家进行内购。</span><br></p><p>　　

据知情人消息，《疯狂出租车》新作已经开发超过一年，东京世嘉团队预期在两到三年内发售该作。在去年的世嘉飒美年度财务报告中，《疯狂出租车》和《涂鸦小子》曾一同出现在世嘉旗下 IP 资产行列，并且可能会获得旧作高清化、重制或新作重启的机会。另据消息源的说法，目前两款作品都在早期开发阶段，也有被取消的可能性。</p><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/04/19/220419164727890_u32815.jpg" class style="width: 700px;" referrerpolicy="no-referrer"><figcaption contenteditable="true" class style="display: block;">排在过往 IP 前两位的即是《疯狂出租车》《涂鸦小子》</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　在《疯狂出租车》原作中，玩家负责扮演一位速度高于一切的出租车司机，而在《涂鸦小子》中，玩家则是街头轮滑帮派的成员，一边涂鸦一边完成各种挑战。两部初代作品都于 2000 年登陆世嘉 Dreamcast 平台，系列旧作虽从未获得大范围的商业成功，但成功吸引了一批死忠粉丝。</span><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　截至发稿时，世嘉方面尚未对此报道发表评论。</span></p><p style="text-align: right;">来源：<a href="https://www.bloomberg.com/news/articles/2022-04-19/sega-reboots-crazy-taxi-jet-set-radio-to-chase-fortnite-riches">彭博社</a></p></div>
