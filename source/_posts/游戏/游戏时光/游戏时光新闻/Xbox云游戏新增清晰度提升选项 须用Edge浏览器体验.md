<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　微软官方宣布，「Xbox Cloud Gaming」云游戏服务新增“清晰度提升”选项，玩家可享受更优质的云游戏。需要注意的是，该选项只能在微软自家的 Edge 浏览器开启。</span><br></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width: 900px;" src="https://img01.vgtime.com/game/cover/2021/11/30/211130182220744_u202875.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>官方对比图</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　现在玩家可通过 Edge Canary 测试版浏览器进行体验，微软表示明年会将该功能推广至所有 Edge 用户。</span><br></p><p>　　以下是官方给出的体验步骤：</p><p>　　1. 下载<a href="https://www.microsoftedgeinsider.com/en-us/download/canary">微软 Edge Canary 测试版浏览器；</a><br></p><p>　　2. 为确保顺利进入 Edge Canary，用户可访问 <a href="edge://settings/help">Edge 设置界面</a>并确认使用 96.0.1033.0 或更高版本；</p><p>　　3. 前往 <a href="https://www.xbox.com/en-US/play">Xbox 云游戏服务网站，</a>登入账户并开始游戏；</p><p>　　4. 打开「更多操作」（…）菜单；</p><p>　　5. 选择“启动清晰度提升”选项。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/30/21113018274069_u202875.jpg" class referrerpolicy="no-referrer"><br></p><p>　　另外微软还明确了一些<a href="https://www.reddit.com/r/xboxinsiders/wiki/xgspreview/audiovideohelp#wiki_what_is_clarity_boost_and_how_do_i_enable_it.3F">使用细节</a>。如遇黑屏，请关闭窗口并重新打开或更新 Xbox 应用。</p><p style="text-align: right;">来源：<a href="https://news.xbox.com/en-us/2021/11/29/clarity-boost-with-xbox-cloud-gaming-on-edge-browser/">Xbox</a></p></div>
