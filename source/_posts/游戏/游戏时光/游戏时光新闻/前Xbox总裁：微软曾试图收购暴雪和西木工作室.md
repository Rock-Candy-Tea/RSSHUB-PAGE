<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　近日，前 Xbox 总裁艾德·弗莱斯（Ed Fries）做客第 105 期 XboxEra 播客，分享了他任职期间（1986-2004 年）微软为收购暴雪娱乐、西木工作室与任天堂所做过的努力。</span><br></p><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/04/19/220419151745795_u32815.jpg" class style="width: 600px;" referrerpolicy="no-referrer"><figcaption contenteditable="true" class style="display: block;">前Xbox总裁艾德·弗莱斯</figcaption></figure></div><h4><b style="letter-spacing: 1.3px; background-color: initial;">西木工作室</b></h4><p>　　弗莱斯表示，他是一名《命令与征服》忠实玩家，同时也玩过《沙丘2》《银翼杀手》等西木工作室<span style="letter-spacing: 1.3px;">（Westwood Studios）</span><span style="background-color: initial; letter-spacing: 1.3px;">的经典作品，是不折不扣的即时战略游戏粉丝。</span></p><p>　　当时弗莱斯筹备起草了一份面向西木工作室的收购合同，并对合同里微软开出的条件充满信心。由于弗莱斯恰逢新婚，于是他在嘱咐团队千万不要搞砸这个合同之后，踏上了前往澳洲的十日蜜月旅行。</p><p>　　蜜月归来后的故事，想必不用再做赘述：EA 以更高的价格介入收购，成功将西木工作室纳入麾下，并最终于 2003 年将工作室关闭。</p><p>　　直至今天弗莱斯也认为，如果当初西木跟了微软，也许会比在 EA 发展得更好一些。</p><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/04/19/220419150509798_u32815.jpg" class style="width: 500px;" referrerpolicy="no-referrer"><figcaption contenteditable="true" class style="display: block;">西木工作室《命令与征服》</figcaption></figure></div><h4><b style="background-color: initial; letter-spacing: 1.3px;">暴雪娱乐</b></h4><p>　　另一件让弗莱斯意难平的是针对暴雪娱乐<span style="letter-spacing: 1.3px;">（Blizzard Entertainment）</span><span style="background-color: initial; letter-spacing: 1.3px;">的收购。《魔兽争霸》是彼时如雷贯耳的知名即时战略游戏系列，而同类型另一代表作品《帝国时代》也是后来微软在 PC 游戏起家的根基之一。可以说，90 年代的微软是一家有即时战略游戏情节的公司，有收购暴雪的意愿也是情理之中。</span></p><p>　　作为暴雪粉丝的弗莱斯，自然也不想错过收购暴雪的机会。1996 年暴雪的母公司 Davidson & Associates 决定出售暴雪时，微软立刻就出手了。然而事与愿违，包含此次收购在内，微软先后被两家竞价更高的公司拼了下去。</p><p>　　第一家（1996 年）是此次成功收购了暴雪与雪乐山的美国 CUC 集团，彼时的 CUC 主营包含租赁营地在内的旅游业，输给一家“野营公司”让弗莱斯感到难以置信。</p><p>　　在 CUC 陷入财务造假丑闻后，暴雪随雪乐山几经转手再次被出售，而第二家（1997 年）收下暴雪的公司更是让弗莱斯绝望，竟然是一家来自法国的自来水处理巨头。熟悉这家公司的玩家想必已经猜到了，这家公司就是有着“娱乐帝国”野心的维旺迪（Vivendi）。</p><p>　　是的，收购暴雪心切的微软，先后输给了一家旅游公司和一家水务公司，与暴雪失之交臂。<span style="background-color: initial; letter-spacing: 1.3px;">不过艾德·弗莱斯在分享完这段故事的最后还是笑了笑，说：</span></p><blockquote class>　　我想，微软终于还是成功买到了暴雪。</blockquote><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/04/19/220419151204767_u32815.jpg" class style="width: 500px;" referrerpolicy="no-referrer"><figcaption contenteditable="true" class style="display: block;">暴雪娱乐《魔兽争霸2：黑潮》</figcaption></figure></div><h4><b style="background-color: initial; letter-spacing: 1.3px;">任天堂</b></h4><p>　　在聊收购故事的最后，弗莱斯分享了更多当年微软寻求与任天堂合作的细节。这一则故事最早由前微软娱乐和设备部门总裁罗比·巴赫（Robbie Bach）分享，也即知名的“任天堂把微软笑出了大楼”的故事。</p><p>　　他说到，当时微软的办公园区和美国任天堂紧挨着，于是他们就决定去问问看，以至于任天堂一时没搞清楚微软是来做什么的。<span style="background-color: initial; letter-spacing: 1.3px;">弗莱斯记得很清楚，任天堂方问了他一个问题：</span></p><blockquote>　　PlayStation 和 任天堂64 谁的性能表现更强？</blockquote><p>　　弗莱斯当时的回答是「任天堂64」主机，但任天堂自己并不认可这个答案。这让任天堂觉得，当时的微软恐怕还没搞明白他们要做什么，他们能为任天堂带来什么，为什么要找上任天堂，于是合作一事就没了下文。</p><p><i>　　*编注：就纸面数据而言，「任天堂64」的 CPU 频率其实更高。但在实机表现上，两者各有千秋。</i></p><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/04/19/220419153139787_u32815.jpg" class style="width: 600px;" referrerpolicy="no-referrer"><figcaption contenteditable="true" class style="display: block;">PlayStation 和 任天堂 64 主机</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　弗莱斯还提到，实际上微软当时只是寻求合作，至少在他任期内，微软并没有真正要收购任天堂的计划，而合作方负责硬件，微软负责软件是 Windows 操作系统如日中天时微软的典型思维，这在初代 Xbox 开发团队的竞争中也有所体现（在 </span><a href="https://www.vgtime.com/topic/1142870.jhtml" style="background-color: initial; letter-spacing: 1.3px;">Xbox 20 周年纪录片</a><span style="background-color: initial; letter-spacing: 1.3px;">中，Windows CE 团队相比 DirectX 团队要更受青睐）。</span><br></p><p>　　此外，弗莱斯还否定了内部对 EA 的收购意见，因为他觉得 EA 的游戏微软与其收购他宁愿自己做。比尔·盖茨也曾与索尼 SCE 集团的久多良木健有过对话，但是双方无法就合作一事达成一致，后来微软自行开发了 Xbox 进入主机市场。</p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　在长达近 2 小时的本期播客中，艾德·弗莱斯还聊到不少他在职时 Xbox 内部的趣闻、他对 Xbox Game Pass 的看法，以及当年他离职微软差点让菲尔·斯宾塞也想辞职的故事等等，感兴趣的朋友可以点击</span><a href="https://xboxera.com/2022/04/17/episode-105-its-pronounced-freeze/" style="background-color: initial; letter-spacing: 1.3px;">这里</a><span style="background-color: initial; letter-spacing: 1.3px;">收听。</span><br></p><p style="text-align: right;">来源：<a href="https://xboxera.com/2022/04/17/episode-105-its-pronounced-freeze/">XboxEra</a></p></div>
