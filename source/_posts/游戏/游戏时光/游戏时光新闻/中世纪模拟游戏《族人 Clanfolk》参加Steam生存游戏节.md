<div class="topicContent front_content"><p>《族人 Clanfolk》是一款中世纪殖民地模拟游戏，玩家将来到苏格兰高地，与大自然本身对抗。你可以把定居点发展成热闹的旅店，贸易中心，或是一个繁荣而自给自足的家园。目前游戏正在参加 Steam 生存游戏节，游戏总评为特别好评，官方自带简繁中文。<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/08/220808180046292_u93176.png" class referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">在游戏里，你可以创建你的血统或从现有的氏族中选择，掌管一个在严酷的苏格兰高地求生的多代家庭。你的家庭成员会结婚、生子、变老，并最终离开人世。但他们的贡献和努力成果将会留在世间，为子孙后代留下一笔遗产。</span><br></p><p>这期间你需要统筹安排，确保每个人都吃好睡好；适当进行娱乐，但不要影响家族为凛冬而做的准备。在晨间准备、社交和休息之间穿插一天的家务劳动：播种、翻土、钓鱼、打猎、手工制作和烹饪。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/08/22080818012152_u93176.png" class referrerpolicy="no-referrer"><br></p><p>每个家族成员都拥有自己独特的特质、愿望、梦想和技能，他们自身的一技之长决定了他们更适合做什么方面的工作。他们会关心自己，关心别人，随着时间的推移而逐渐成长。教他们打猎、烹饪、清洁、建造或耕作，确保在长辈们安息时，所有的核心生存技能都能传承到下一代。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/08/220808180154962_u93176.png" class referrerpolicy="no-referrer"><br></p><p>游戏里玩家还需要注意季节的交替，<span style="background-color: initial; letter-spacing: 1.3px;">每个季节都有独特的挑战和机遇，无论是采摘浆果、诱捕鳗鱼、狩猎兔子，还是寻找蘑菇地，充分利用中世纪环境中的机会。</span></p><p>土壤条件的变化和一系列其他因素影响着你作物的生长。翻耕土壤、利用刀耕火种等技术、培训优秀农民以确保你的家族食无忧。</p><p>环境变化还带来了持续的挑战，保护你的牲畜、器具和族人的安全，免受大雨和零度以下低温的影响。在室外留下的水会变成冰，必须用火来融化；在雨中浸泡的食物虽然不容易着火，但很快就会变质。没有农舍的农场动物会因健康状况不佳而死亡。在寒冬，除非你的族人穿着合适的毛皮衣物，否则当他们离开温暖的火堆旁时就有可能会被冻僵。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/08/22080818020788_u93176.png" class referrerpolicy="no-referrer"><br></p><p>除了季节变化，玩家还需要<span style="background-color: initial; letter-spacing: 1.3px;">为一些不可避免的灾难做好准备。健康状况不佳、恶劣的天气和熊熊燃烧的大火都会使繁荣的家园化为一片火海或者墓地。</span></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/08/220808180228634_u93176.png" class referrerpolicy="no-referrer"><br></span></p><p>注意，你的家族并不是高原上唯一的氏族，商人、旅行者和四处找活干的的邻居们都会来拜访你，给你结交新的朋友、赚取钱财和扩大家园的机会。</p><p>贸易站会吸引其他氏族的成员来售卖有用的物品，或者购买你出售的商品。你在其他氏族的声誉决定了他们的商人带来的货物类型，由于每个氏族都有自己的特产，结交合适的朋友可以帮你更好地生存下去。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/08/220808180305930_u93176.png" class referrerpolicy="no-referrer"><br></p><p>当旅行者经过你的家园时，他们会寻求住宿，如果你愿意的话，可以为他们提供住所。特别满意的客人可能会延长他们的逗留时间或支付高额的费用。建立一个热闹的旅店可以吸引更多人。</p><p>来自其他氏族的工人们会不时地来看你的工作告示板，寻求以劳动换取金钱的机会。一些额外的技术工人对收获作物、加强你的狩猎队伍或为你的新旅店打工有很大的帮助。</p><hr class="vg_hr" contenteditable="false"><h3>福利时间</h3><p>我们将在评论区里，抽两人送出《族人 Clanfolk》的 Steam 兑换码。</p></div>
