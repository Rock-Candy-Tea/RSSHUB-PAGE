<div class="topicContent front_content"><p>　　根据 ZeniMax 的招聘信息，《毁灭战士》开发商 id Software 正在开发一款未公布的 FPS 游戏。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/04/17/220417141814464_u77470.jpg" title referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　最新的一批招聘岗位都明确指出了所属的项目是 id Software 的未公布的游戏项目。比如</span><span style="letter-spacing: 1.3px;"><a href="https://jobs.zenimax.com/requisitions/view/2542">高级玩法程序员</a>、</span><span style="background-color: initial; letter-spacing: 1.3px;"><a href="https://jobs.zenimax.com/requisitions/view/2543">高级UI程序员</a>。</span><br></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/04/17/220417142920576_u77470.jpg" class title referrerpolicy="no-referrer"><br></span></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　</span><span style="background-color: initial; letter-spacing: 1.3px;">其中，</span><span style="background-color: initial; letter-spacing: 1.3px;">高级玩法程序员</span><span style="background-color: initial; letter-spacing: 1.3px;">岗位</span><span style="background-color: initial; letter-spacing: 1.3px;">要求“在游戏行业工作 5 年以上或在第一人称射击游戏方面有明显的工作经验”。此外，还最好拥有“任何版本的 </span><span style="background-color: initial; letter-spacing: 1.3px;">id Tech </span><span style="background-color: initial; letter-spacing: 1.3px;">引擎经验”。</span><span style="background-color: initial; letter-spacing: 1.3px;">id Tech </span><span style="background-color: initial; letter-spacing: 1.3px;">是 id Software 自有的游戏引擎，被用于其旗下大部分游戏。</span></p><p>　　id Software 曾开发过许多知名 FPS 作品，诸如《毁灭战士》和《雷神之锤》。这款未公布的 FPS 新作有可能是某个知名系列的续作。外媒根据使用 <span style="background-color: initial; letter-spacing: 1.3px;">id Tech </span><span style="background-color: initial; letter-spacing: 1.3px;">引擎这一点推断，应该不大可能是《狂怒》系列续作，因为《狂怒2》使用了共同开发商 Avalanche Studios 的 APEX 引擎，下一作转为使用 </span><span style="background-color: initial; letter-spacing: 1.3px;">id Tech </span><span style="background-color: initial; letter-spacing: 1.3px;">的可能性略低。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2022/04/17/220417143037799_u77470.jpg" class title referrerpolicy="no-referrer"><figcaption contenteditable="true" class>雷神之锤</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　岗位职责中提到了“合作实现新的游戏功能：玩家和角色的行为、战斗和能力机制”，因此游戏中可能会包含增加角色能力的机制。这在《雷神之锤》中是存在，考虑到更早之前的传闻，外媒推测它有可能是《雷神之锤》重启作。不过，为《毁灭战士》系列引入这个机制也是有可能的。</span><br></p><p>　　当然，这甚至可能是 id Software 的一个全新 IP，因为招聘信息中的细节非常模糊，所以没法进一步推测。</p><p style="text-align: right;">来源：<a href="https://exputer.com/news/industry/doom-developer-unannounced-fps-game/">Exputer</a></p></div>
