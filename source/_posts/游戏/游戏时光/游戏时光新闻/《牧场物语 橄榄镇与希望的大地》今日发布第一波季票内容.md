<div class="topicContent front_content"><p>　　《牧场物语 橄榄镇与希望的大地》在今天发布了季票的第一波追加内容，包含主角和恋爱候补们的新服装“布偶装套装”以及迷你剧本“橄榄镇 神秘档案”。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/25/210325171815358_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　给主角新增的“布偶装套装”是两种以布偶为主题的服装。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/25/210325171825752_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　不仅主角自己可以换装，还可以一次性给 10 名恋爱候补都换上布偶装。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/25/21032517183863_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/25/210325171846389_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/25/210325171851354_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　在追加的迷你剧本“橄榄镇 神秘档案”里，玩家将与麦奇和辛蒂一起揭开橄榄镇怪异事件的真相。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/25/21032517190819_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/25/210325171915304_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　这是系列首次加入的推理剧情，将提供与本篇不同的游戏体验。迷你剧本还附带新的服装。</span><br></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/03/25/210325171925299_u93176.jpg" class referrerpolicy="no-referrer"><br></span></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　更多</span><span style="background-color: initial; letter-spacing: 1.3px;">追加内容将在未来依次发布，其中还</span><span style="letter-spacing: 1.3px;">包含了以过去的 3 部作品为题材，追加恋爱候补和居民的“恋爱候补包”。</span></p></div>
