<p>不叫Windows 10了？</p>

<div class="topicContent front_content"><p>　　微软对今年的 Windows 计划保持着令人难以置信的高度保密。我们已经知道的是，今年秋天 Windows 将有一次重大的用户体验改进，代号为“太阳谷“（Sun Valley），可能会在 6 月为此举行一次专门的发布活动。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/26/210526104206986_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　在微软 Build 2021 大会的主题演讲中，CEO 萨提亚·纳德拉稍稍谈到了这一重大更新。虽然没有透露什么细节，不过他将该更新称为「次世代Windows」(the next generation of Windows)，和微软首席产品官 Panos Panay 选择的用词一致。听起来该更新很可能不会被称为 Windows 10。</p><p>　　纳德拉在他的主题演讲中说：</p><blockquote><p>　　很快我们将分享这个过去十年中 Windows 最重要的更新之一，为开发者和创作者释放更大的经济机遇。在过去的几个月里，我一直在内部测试，我对下一代 Windows 感到无比兴奋。我们对你们的承诺是这样的。我们今天将为每一个 Windows 开发者创造更多的机会，并欢迎每一个正在寻找最具创新性的、新的、开放的平台来构建和发布应用程序并实现盈利的创作者。我们将很快分享更多内容。</p></blockquote><p>　　微软 Build 是一个面向开发者的大会，不过「次世代Windows」对 PC 玩家来说也将是一个大变化，已经泄露出来的一些细节表明它是对 Windows 用户界面的一次重大更新。</p><p>　　今年 7 月是 Windows 10 的六周年纪念，似乎会是一个公布重要内容的好时机。</p><p style="text-align: right;">来源：<a href="https://www.pcgamer.com/microsoft-ceo-says-the-next-generation-of-windows-is-coming-soon/">PC Gamer</a></p></div>
