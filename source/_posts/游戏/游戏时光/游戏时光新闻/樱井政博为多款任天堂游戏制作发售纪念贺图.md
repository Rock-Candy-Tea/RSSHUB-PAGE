<div class="topicContent front_content"><p>　　11 月 21 日是很多任天堂游戏问世的日子，樱井政博发推表示今年他想要正经介绍一下这些游戏，并且推文的配图都是使用《任天堂明星大乱斗SP》制作的。以下游戏介绍选自樱井政博的推文。</p><ul><li><b>《迈克泰森的击拳热斗》（1987年）</b></li></ul><p>　　这是一款与体格相差悬殊的拳击选手进行拳击比赛的游戏，原本只是作为奖品送出，在迈克泰森加入后才向大众发售。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121316193_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《超级马力欧世界》（1990年）</b></li></ul><p></p><p>　　这是耀西的首秀，与其说是一个可选角色，倒不如说耀西是作为载具登场的。所以，在紧要关头被玩家抛弃也是没办法的事。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121324362_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《零式赛车》（1990年）</b></li></ul><p></p><p>　　比同时代的其他赛车游戏更强调速度感，当时飞隼队长还只出现在游戏说明书上。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121331220_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《塞尔达传说 众神的三角力量》（1991年）</b></li></ul><p></p><p>　　在 SFC 发售一年后推出，这时候「塞尔达」系列的基础差不多已经完成，堪称名作。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121337700_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《刺猬索尼克 2》（1992年）</b></li></ul><p></p><p>　　继承了前作的速度感，也是塔尔斯首次登场的作品，能够将画面分为上下两部分游玩。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121345934_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《超级森喜刚2》（1995年）</b></li></ul><p></p><p>　　迪迪和迪茜要去营救被绑架的森喜刚，虽然他俩有着相似的体型和名字，但迪茜似乎更有优势。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121352632_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《迪迪刚赛车》（1997年）</b></li></ul><p></p><p>　　和《马力欧赛车》差不多的游戏，但你能坐飞机。这还是班卓熊首次登场的作品。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121359925_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《塞尔达传说 时之笛》（1998年）</b></li></ul><p></p><p>　　这是「塞尔达」系列首个完全 3D 的作品，不朽的名作。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/21112212140839_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《宝可梦 金/银》（1999年）</b></li></ul><p></p><p>　　第二世代「宝可梦」，皮丘在本作中首次登场。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121414162_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《罪与罚～地球的继承者～》（2000年）</b></li></ul><p></p><p>　　一款经典的射击游戏。在 N64 上推荐使用十字键进行移动，用摇杆进行瞄准。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121428808_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《任天堂明星大乱斗 DX》（2001年）</b></li></ul><p></p><p>　　制作的时候非常辛苦，不过做游戏不都是这样吗。需要玩家进行高密度的操作，既是游戏的优点，也是需要反思的地方。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121435229_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p></p><ul><li><b>《宝可梦 红宝石/蓝宝石》（2002年）</b></li></ul><p></p><p>　　GBA 上的第一款「宝可梦」。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/22/211122121445605_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　现在樱井政博已经开始更新 11 月 22 日发售的游戏贺图，感兴趣的朋友可以前去围观。</p><p style="text-align: right;">来源：<a href="https://twitter.com/Sora_Sakurai/status/1462239370874867713">Twitter</a></p></div>
