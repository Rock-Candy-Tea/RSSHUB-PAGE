<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　每次宝可梦发布会后推特上都会有大批画师为宝可梦绘制同人图，今天我们收集了一些大触的作品，一起来欣赏一下他们的创意和脑洞与宝可梦碰撞后产生的作品。</span><br></p><p>　　网友 <a href="https://twitter.com/onigin_pixelart/status/1428069540286517248">onigin_pixelart</a> 可能很怀念以前宝可梦游戏里的像素画风，他用同样的风格重绘了新登场的 4 只宝可梦。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820132317185_u93176.png" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820132323634_u93176.png" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820132328558_u93176.png" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820132334956_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　<a href="https://twitter.com/henryvargasart/status/1428018555056889860">henryvargasart</a> 使用光影的技巧非常娴熟，如果不是翻了他首页的其他作品，我都分不出这几张图是绘画作品还是 3D 模型。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820132450406_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820132459923_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/21082013251794_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　在 henryvargasart 那里落单的勇士雄鹰当然不是没人喜欢，网友 <a href="https://twitter.com/KamabokoBun/status/1428011670387843075">KamabokoBun</a> 就表示已经等不及在游戏里拥有一只勇士雄鹰了。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/21082013255638_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　要说这次新公布的宝可梦里人气最高的就是卡蒂狗了，不过也有的朋友觉得这个设计不好看。有的网友认为原本的卡蒂狗要好看得多，看这俊俏的脸庞，坚毅的眼神，自信且迷人的身姿，这才是玩家的好伙伴。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820132633361_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　再看新的卡蒂狗，浓密的毛发甚至遮住了眼睛，给人感觉像是没睡醒的懒散样。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820132651361_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　所以网友 <a href="https://twitter.com/yourfavisbald2/status/1427999411179859975">yourfavisbald2</a> 就尝试把卡蒂狗身上的毛发都去掉了，包括那几搓挡住眼睛的白毛。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820132733509_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　就我在推特上观察的情况来看，其实多数网友喜欢这个形态卡蒂狗，特别是有很多看完发布会后为了卡蒂狗爆肝的画师，相关 tag 下就数卡蒂狗的同人图最多了。</p><p>　　网友 <a href="https://twitter.com/YukimiKitune/status/1428288555336404993">YukimiKitune</a> 表示他很喜欢这次卡蒂狗的感觉。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820132955667_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　画师 <a href="https://twitter.com/SharkySnapz/status/1428230157370863616">SharkySnapz</a> 画福瑞比较多，所以他画的卡蒂狗看起来多了几分人类的感觉。</span><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820133202821_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　除了新的宝可梦，《宝可梦传说 阿尔宙斯》还对战斗系统进行了一些调整，野生宝可梦会目露凶光主动袭击玩家，相应片段里登场的是女主角，这就让网友开始进行一些有趣的想象。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820133235347_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　本作女主角的形象一公布就有网友有种莫名的即视感，四角头巾，齐肩的黑色长发，还要在野外和宝可梦（动物）打交道，这不就是《黄金神威》里的阿希莉帕嘛？</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/21082013324969_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　特别是宝可梦里还有海豹球这只宝可梦，跟她搭配在一起可以还原《黄金神威》里阿希莉帕棒打海豹的名场景。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820133307471_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820133315703_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　再加上《黄金神威》里的阿希莉帕强大且可靠，已经有不少看过这些同人作品的网友后接受「宝可梦新作的主角很强大」这个二设。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820133335997_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　所以网友 <a href="https://twitter.com/NonCoral_Art/status/1428166205274263555">NonCoral_Art</a> 就认为主角怎么可能会被野生宝可梦追着跑呢，区区一只圈圈熊根本不带怕的好嘛。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820133411781_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　至于这只追着主角跑的圈圈熊，自然也逃不过<a href="https://twitter.com/TerribleTavi/status/1427995753654493186">网友</a>的恶搞：</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/20/210820133446149_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　除了长相凶恶的圈圈熊，<a href="https://twitter.com/DigitalsunLite/status/1428038851168190473">有人</a>认为在宝可梦世界里只看外表不顾等级就贸然出手容易吃瘪。</p><p style="text-align: center;"><img style="width:600px;" 1.3px;">　　也有对本次公开的新机制和主角二设不感兴趣的朋友，他们在看过预告片后觉得女主角很可爱，单纯就是因为喜欢女主角而动起手里的画笔。</span><br></p><p style="text-align: center;"><img style="width:600px;"
