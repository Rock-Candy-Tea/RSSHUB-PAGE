<div class="topicContent front_content"><p>据报道，多名网友表示自己的《最后生还者 第一部》限定收藏版「火萤版」在运输中被损坏。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/09/04/22090414213766_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>这个「火萤版」包含游戏本体、限量铁盒、重新印刷且有新封面的《最后生还者 美国梦》漫画1-4 期和游戏初期解锁内容。售价为 99.99 美元，仅在索尼的 PlayStation Direct 商店销售。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/09/04/220904140618125_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>根据<a href="https://www.reddit.com/r/thelastofus/comments/x4b71t/comment/imw9ojk/?utm_source=share&utm_medium=web2x&context=3">网友晒出的图片</a>，「火萤版」的运输包装为简单的纸袋。显然很难保护里面的纸盒尖角。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/09/04/220904141256120_u77470.jpg" class referrerpolicy="no-referrer"><img style="width: 350px;" src="https://img01.vgtime.com/game/cover/2022/09/04/220904142156403_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>加之包装非常草率，<a href="https://twitter.com/RillIris/status/1565811070270201857">有的网友表示</a>，当纸袋贴不到另一面时，包装者就直接贴在了游戏盒子上，在盒子上留下了许多难以处理的残留物。  <br></p><p style="text-align: center;"><img style="width: 480px;" src="https://img01.vgtime.com/game/cover/2022/09/04/220904142240383_u77470.jpeg" class referrerpolicy="no-referrer"><img style="width: 350px;" src="https://img01.vgtime.com/game/cover/2022/09/04/220904142224578_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>并且，「火萤版」还是 PlayStation Direct 商店独占，意味着没有办法像过去那样在本地商店买到品相完美的版本。即使会有加价，也比这样因为邮寄而损坏好。</p><p>到目前为止，PlayStation Direct 商店并未为受影响的玩家提供替换或退款，只提供了一张该商店的八折券以表歉意。顽皮狗和索尼都没有对这个问题发表声明。</p><p style="text-align: right;">来源：<a href="https://www.gamesradar.com/the-last-of-us-part-1-damaged-firefly-editions-will-not-be-replaced-or-refunded-by-sony/">GamesRadar+</a></p></div>
