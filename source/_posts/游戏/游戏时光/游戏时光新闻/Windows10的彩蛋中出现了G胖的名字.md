<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　近期，著名的 Windows 忠实粉丝 Lucas Brooks 发现了一个隐藏在 Windows 1.0 中的彩蛋，展示了该系统的研发人员名单，而其中包含了 Gabe Newell，也就是我们熟知的“G胖”。</span><br></p><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/03/27/220327111000393_u315131.png" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>Windows 1.0中的彩蛋</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　Windows 1.0 自 1985 年 11 月正式发布以来，已经将近 37 年，但其中依旧有着许多我们尚未知晓的秘密。你可能会奇怪为何名单会出现 G 胖的名字，因为 G 胖当年就在微软工作，直到 1996 年他才与 M</span><span style="letter-spacing: 1.3px;">ike Harrington 离开微软，一起创建了 Valve。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/03/27/220327112438554_u93176.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>年轻的G胖</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　此次 Lucas Brook 是在操作系统附带的一个笑脸位图文件数据的末尾发现了这段额外的加密数据。据他所说，以当时的工具根本无法提取位图文件中的数据，就算有人提取出来了，也不可能发现这些额外的加密数据。</span><br></p><p>　　虽然 Lucas Brook 目前通过逆向工程发现了这个职员表，但他还没有弄清楚如何在不进行黑客攻击的情况下正常访问这个彩蛋。他相信在 Windows 1.0中能够通过一系列按键组合解锁该隐藏名单，因为在其他更高版本的 Windows 系统中就有着类似的机制。</p><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/03/27/220327110804253_u315131.png" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>Windows 2.0中的彩蛋</figcaption></figure></div><p style="text-align: right;"><span style="background-color: initial; letter-spacing: 1.3px;">来源：</span><a href="https://kotaku.com/windows-1-0-easter-egg-secret-37-years-later-hidden-cre-1848704671" style="background-color: initial; letter-spacing: 1.3px;">Kotaku</a><br></p></div>
