<div class="topicContent front_content"><p>近日，亚克系统（Arc System）社长木戸岡稔在 IGN 的采访中称，未来他们将更积极地寻求与版权方的合作，为这些 IP 制作格斗游戏。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/08/220808123312675_u93176.jpeg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>木戸岡稔</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">在采访中木戸岡稔表示：</span><br></p><blockquote><p>我们需要通过 IP 合作，来扩大格斗游戏的用户社群。未来如果有机会，会积极地推动与版权方的合作。</p></blockquote><p>亚克系统制作了很多有名的格斗游戏，但木戸岡稔觉得对于普通玩家来说，他们的游戏还是有些小众，特别是在欧美地区。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/08/220808123155649_u93176.jpeg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">为了改变现状，公司决定多在 IP 合作上下功夫。过去是版权方找上门寻求合作，现在他们打算在合作方面不要那么被动。最后，木戸岡稔希望亚克系统能创造出引起欧美玩家共鸣的游戏。</span><br></p><p style="text-align: right;">来源：<a href="https://www.ign.com/articles/evo-2022-arc-system-works-wants-to-be-less-passive-with-ip-collaborations">IGN</a></p></div>
