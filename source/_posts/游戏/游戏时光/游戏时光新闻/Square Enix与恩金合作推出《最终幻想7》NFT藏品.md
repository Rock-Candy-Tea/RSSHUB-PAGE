<div class="topicContent front_content"><p>　　Square Enix 宣布与 NFT（非同质化货币）基础服务提供商 ENJIN（恩金）合作，推出《最终幻想7》的 NFT。</p><p>　　和常见的 NFT 销售模式不同，本次《最终幻想7》25 周年纪念 NFT 捆绑在<a href="https://store.na.square-enix-games.com/en_US/product/764037/final-fantasy-vii-bring-arts-action-figure-cloud-strife-digital-plus-edition-action-figure">克劳德可动人物模型</a>的豪华版中。玩家可以选择以 130 美元仅购买人物模型，也可以 150 美元购买捆绑了两个 NFT 兑换码的豪华版。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/07/21/220721105432306_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　在今年年末，该系列还将推出实体卡片，附有数字代码可兑换 NFT。</p><p>　　本次的合作方，ENJIN，是一家注册在新加坡的游戏社区和 NFT 服务提供商，他们拥有自己的基于波卡链的区块链项目 Efinity，本次《最终幻想7》NFT 也将在该链上发布。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/07/21/22072111032514_u77470.jpeg" class referrerpolicy="no-referrer"><br></p><p>　　外媒 <a href="https://www.theverge.com/2022/7/20/23271832/square-enix-nft-final-fantasy-vii-enjin-blockchain">The Verge 对此评论称</a>，Square Enix 选择基于《最终幻想7》来推出 NFT 简直是莫大的讽刺，NFT 是一种消耗大量电能的技术，可能会加速生态失衡，而《最终幻想7》的主题就关于一个大公司为了能源而吸走星球的生命。如果现实也有雪崩组织，他们必将站出来反对 NFT。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/07/21/220721110159920_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　为缓解反对声音，ENJIN 表示他们提供的是“对环境友好”的解决方案，并致力于在 2030 年前“实现碳中和的 NFT”。</p><p>　　特别需要注意的是，虽然这个可动模型现在就可以预订，但要到 2023 年 11 月才会出货，如果到时 ENJIN 不再提供相关 NFT 服务，那么就无法查看虚拟认证证书和虚拟人物模型，意味着这些 NFT 将全无价值。Square Enix 在商品页面的“购买前提示”中也指出了这一点。</p><p>　　Square Enix 社长松田洋祐曾于年初表示：<a href="https://www.vgtime.com/topic/1144266.jhtml">基于 NFT 的生态会是未来游戏的趋势之一</a>。</p><p style="text-align: right;">来源：<a href="https://www.eurogamer.net/square-enix-to-release-final-fantasy-nfts">Eurogmaer</a> / <a href="https://kotaku.com/final-fantasy-vii-square-enix-nft-blockchain-scam-1849241965">Kotaku</a></p></div>
