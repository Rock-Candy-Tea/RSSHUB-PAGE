<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　想必大家这几天已经在各种地方看过这两张图了，<a href="https://weibo.com/6189056148/Kl1sDAYta?refer_flag=1001030103_&type=comment#_rnd1624858712044">网传</a>这是四川某处的比特币矿场，现在这个矿场已经断电，无法继续挖掘比特币。</span><br></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628133624494_u93176.jpg" class referrerpolicy="no-referrer"><br></span></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628133631386_u93176.jpg" class referrerpolicy="no-referrer"><br></span></p><p>　　虽然国内的主流媒体没有进行报道，但是从国外一些大型比特币专门站发布的<a href="https://twitter.com/BitcoinMagazine/status/1407307411879247875">新闻</a>，我们还是能侧面得知国内的矿场最近有一些动作。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628134039767_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　这可能是大众第一次了解虚拟货币矿场的规模究竟有多离谱，成千上万张显卡同时工作，机房后面是成排的工业风扇。整个矿场耗电量巨大，普通的线路根本无法满足用电需求。</p><p>　　那边矿老板正愁着矿场今后该怎么办，这边却有无数的网友在想着同一件事：没了矿场，他们能买到便宜的显卡了嘛？</p><h4>把矿场全部驱逐出去</h4><p>　　前几年比特币作为个人投资项目火热的时候，大家对比特币如何带来收益多少有些了解。基本上矿工们当天的收益就是生产的比特币×币价-电费，大家每天用自家显卡赚点买菜钱。直到资本进场，为了追求利益最大化搞出了矿场。</p><p>　　按照比特币等虚拟货币的设计，用户通过每个计算机节点以其算力来抢夺记账权，从而得到比特币奖励。挖矿的硬件设备叫“矿机”，购买矿机的个体挖矿者叫做“矿工”，托管矿机并提供电力和网络的地方就称为“矿场”。</p><p>　　有的人以为一间矿场，投入最大的是购买矿机，毕竟中高端显卡的市场价那简直没眼看，一个专业矿场动辄几千上万张显卡，前期投入太大了。但这与后续的电费开销比起来根本算不了什么。</p><p>　　大型矿场的收益=生产的比特币×币价-矿机成本-电费-维护费-人工成本-矿场折旧费，其中占比最大的是电费。虽然矿机昂贵，但对大型矿场而言，矿机并不是最重要的开支，挖矿的最大成本是电费。</p><p>　　根据<a href="http://www.sfccn.com/2021/3-13/zNMDE0MDZfMTYzMTczNQ.html">相关报道</a>，随着越来越多的人加入比特币市场，挖矿的商业模式在这几年已经发生了巨大的变化。仅以电费一项来说，由于新疆冬季的电费电费便宜，同一台矿机，在新疆矿场挖价值 10 元的比特币要交 5 元电费，在家挖矿挖 10 元的比特币要就交 10 元电费。</p><p>　　所以找到便宜且稳定的电力资源，就成了经营矿场的必修课。因此电力资源充足而电费便宜的地区就成为了矿场的聚集地，例如冬季火电丰富的新疆、内蒙古，以及夏季水电丰富的云南、四川、贵州。</p><p>　　就以四川为例，我们来看一下四川省的河流水系图，四川的水资源非常丰富，这给四川省带来了充足的水电。这些水电资源丰富地区，他们的丰水期和枯水期电力供应差别巨大，丰水期一度面临着数百亿度电“投产即遭弃”的困境。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628134246493_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　这丰富且廉价的电力资源，不正是矿工们梦寐以求的天堂嘛？</span><span style="background-color: initial; letter-spacing: 1.3px;">丰富的水电资源让云贵川地区在夏季成为了挖矿的首选地，但这一切都在这几天成为了过往云烟。</span></p><p>　　尽管虚拟货币在全球保持着极高的关注度，但它的合法性在国内一直没有定论。直到5月底，金融委第五十一次会议中明确提出“打击比特币挖矿和交易行为，坚决防范个体风险向社会领域传递”，这才<a href="http://www.sfccn.com/2021/6-8/zNMDE0MDZfMTY0ODIzNA.html">明确了打击比特币挖矿的信号</a>。</p><p>　　不仅如此，<a href="https://jingji.cctv.com/2021/06/22/ARTIqjdh9bN36R9JWevWzPIX210622.shtml">中国人民银行还约谈了相关的支付机构</a>，之后支付宝、中国建设银行、中国工商银行和中国邮政储蓄银行等均发布声明，表示将持续开展对虚拟货币交易的打击治理行动，禁止使用银行服务用于虚拟货币交易。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/21062813441923_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　这一连串动作不仅打击了国内的矿老板，也影响了整个币圈。那几天<a href="http://www.xinhuanet.com/world/2021-06/23/c_1127588562.htm">比特币价崩</a>，还连带拉低了英伟达和 AMD 的股价。可以看到 21 号当天，两家显卡巨头的股价都有不同程度的下跌。虽然跌幅不大，但也说明华尔街精英们挺在意国内的动作。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628134439611_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　当然，这对普通玩家来讲并不重要，资本市场的事情我们不 care，最重要的是显卡的价格终于迎来了一波大跳水。</span></p><h4>平价显卡不是梦</h4><p>　　尽管显卡供应不足的根本原因是<a href="https://www.vgtime.com/topic/1127058.jhtml">芯片产能不足</a>，但趁着币圈震荡，在矿老板考虑对策的时候，显卡的价格已经来到了一个相对较低的水平。</p><p>　　目前某海鲜市场上一张 RTX3080 显卡从近 2 万跌到了 1 万左右，虽然距官方指导价 5499 仍有一段不小的距离，但对于一些急需显卡的朋友来说现在的价格已经比之前低了不少，并且显卡的价格还有进一步下探的趋势。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628141349651_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　可以说这次行动对显卡市场的影响是实打实的，网上已经出现了很多所谓“女生自用”“老婆不让买只好二手出售”的实惠显卡。</p><p>　　同时还有一些不满黄牛抬高显卡市场价的网友，在某海鲜市场发布了下面这样的虚假商品，想进一步搅动二手显卡市场。这些虚假的商品可以扰乱买家在 App 上浏览的内容，拉低显卡在平台上显示的均价，意图显卡能更快降价。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628134536227_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　当然，还有的朋友想等显卡价格正常后蹲到售价更低的矿卡。国内的矿场开不下去了，肯定会有大把矿卡流入市场，到时候矿卡还不是随便挑？反正显卡用个三四年就换了，只要这期间不出毛病，那必然是血赚。</p><p>　　事情也许并没有你们想象中的那么美好，大多数矿卡可能还是难逃没日没夜挖矿的命运，并不会流入市场给玩家捡漏。至于这其中的原因，咱们还得了解一下整个虚拟货币市场的动向。</p><h4>低价的矿卡可能没你想象中的多</h4><p>　　我们都知道一个以太坊矿场少说也有几百上千张显卡，很多朋友认为断电之后这些矿机就没用了，矿老板只能含泪在海鲜市场上出售手里的显卡，回点本钱。但实际上这些显卡大概会流向三个方向。</p><p>　　一个是将集中的大型矿场转为小型的家庭式矿场。也就是说矿老板们不会再搞那么大动作，建专业的大型厂房，而是转入地下，以家庭为单位进行挖矿。</p><p>　　虽然有一定的风险，但依然能享受相对低价的电力资源。不过这种方式比较适合规模不大的矿场，不然矿老板手里那么多显卡都要分散出去，管理成本太高了。</p><p>　　于是就有了第二条出路，将整个矿场送往海外。当初选择新疆、内蒙古和云贵川地区作为矿场基地，就是因为那里的电费便宜，现在挖矿在国内不好推进了，那就到国外挖吧。虽然成本变高不会像以往那么赚钱，但多少也是赚啊。</p><p>　　<a href="https://news.cctv.com/2021/05/25/ARTIZ5AVOVN8tfKWofsziZWe210525.shtml">新华网记者在矿场进行采访时</a>，就有矿工表示他们准备漂洋过海，将“矿场”搬到俄罗斯、芬兰、加拿大这些电力比较充足的国家。</p><p>　　同时网上有人爆料，已经有一些矿老板想到国外发展，将矿机打包寄出。虽然需要支付高额的国际运费和大笔的关税，但至少生意保下来了，这一波不亏。<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628140658478_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　这里特别提一下美国的迈阿密，他们的市长听说国内打击虚拟货币后一阵狂喜，<a href="https://www.livebitcoinnews.com/miami-mayor-francis-suarez-opens-his-city-to-chinese-btc-miners/">想邀请中国的矿老板到他们那边发展</a>。</p><p>　　人家敢这么说当然也有相关的优势。迈阿密是世界上最早建立数据中心的城市之一，挖矿所需的东西和数据中心其实差不多，这就意味着迈阿密相关的服务与配套设施都很齐全。</p><p>　　而且迈阿密有土耳其角核电站坐镇，他们用的电力能源是廉价的核电，所以电费会比较便宜。同时迈阿密市长还称他们正在与佛罗里达电器照明公司合作，可以给矿工们提供一个相对合理的用电价格。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628134624625_u93176.jpg" class referrerpolicy="no-referrer"></p><p style="text-align: left;"><span style="background-color: initial; letter-spacing: 1.3px;">　　只要虚拟货币不垮，未来应该会有很多矿老板选择带着矿场出走海外，在异国他乡继续干他们的老本行。</span><br></p><p>　　最后才是流入二手市场，我们玩家能接触到的矿卡。这部分显卡大都来自一些出坑的散户，或是大型矿场在转移时淘汰下来的显卡，质量有好有坏，非常看运气。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628135149242_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　而且矿卡的数量可能没有大家想象中的那么多，有的朋友以为国内矿老板手里的显卡都将在近期流向市场。到时候几十万张矿卡挤到市场上，显卡的价格一定会压得很低。<span style="background-color: initial; letter-spacing: 1.3px;">但我们这一轮分析下来你也知道了，最终流入市场的矿卡或许并不多，想要低价捡漏的朋友，最好不要抱太大希望。</span></p><hr class="vg_hr" contenteditable="false"><p>　　到这里我们来总结一下，一直以来持币观望想要购入显卡的朋友，近期可以出手了。想低价购入矿卡的朋友，可能需要打探一下国内矿场的走向，看看这些矿老板是就此收手还是在海外继续挖矿，这会很大影响矿卡的价格。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/28/210628135202122_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　最后祝各位都能用合适的价格入手心仪的显卡。</p></div>
