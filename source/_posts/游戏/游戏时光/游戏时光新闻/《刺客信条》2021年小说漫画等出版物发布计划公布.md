<div class="topicContent front_content"><p>　　伴随《刺客信条》系列故事走入越来越多国家的历史，《刺客信条》系列也在全球各地有了越来越多粉丝和二次创作作品，目前育碧正与多家出版合作伙伴合作，通过推出官方小说、漫画、网络漫画、有声读物等媒介进一步扩展开《刺客信条》宇宙，官方今日公布了 2021 年在全球推出的出版物计划。  <br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422115956147_u162815.jpg" class referrerpolicy="no-referrer"><br></p><hr class="vg_hr" contenteditable="false"><h4>小说  </h4><h3>《刺客信条：碎片》青年三部曲（原创系列）  </h3><p>　　《碎片》是与法国出版社Editis共同合作推出的全新青年成人读物系列。这部动作冒险的历史小说将以身陷刺客与圣殿骑士百年纷争的青少年为特色，并探讨手足关系的本质。该系列的每本书都会带来新的主人公并将读者带往不同的时间段和地点。第一卷设定于19世纪的日本，第二卷则是13世纪的苏格兰，而第三卷设定于17世纪的法国。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422115940108_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>《刺客信条：传国玉玺》系列（原创系列）

</h3><p>　　从周朝（公元前4世纪）到明朝（17世纪），该系列将通过10部探索中国历史的小说来拓展《刺客信条》系列的世界观。帝王、朝臣、传奇武术家、诗人、哲学家以及中国历史上许多的重要人物将在这场世俗的斗争中发挥自己的作用。由六位中国知名作家创作的这些小说作品中将有新的刺客和敌人加入到《刺客信条》的世界中，并将很快与中国的粉丝和读者们见面。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422115924462_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3><span style="letter-spacing: 1.3px;">《刺客信条：大明风云》（编年史系列）</span></h3><p>　　由中国畅销作家燕垒生所著的《刺客信条：大明风云》是三部曲动作冒险的开篇，讲述了16世纪中国的年轻刺客，同时也是《刺客信条编年史：中国》游戏中备受粉丝们喜爱的主人公——少芸的故事。改小说将由Aconyte Books于2021年6月1日在美国及加拿大地区发行。 </p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422115911150_u162815.jpg" class referrerpolicy="no-referrer"> <br></p><h4>漫画（图像小说）</h4><h3>《刺客信条：王朝》漫画（原创系列）  </h3><p>　　育碧与新漫画合作首次以网络漫画形式在中国地区推出的 《刺客信条：王朝》目前累计阅读量已超过1.5亿次，并在腾讯漫画“武侠”分类排名第三。该小说将很快以平装本在其他地区发行，读者将跟随试图从内战中拯救唐朝的刺客李萼一同冒险。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422115859945_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>《刺客信条：英灵殿》短篇漫画（编年史系列）

</h3><p>　　粉丝们很快就能在这部设定于《刺客信条：英灵殿》世界观下的漫画中，探索在“狼吻者”艾沃尔拓荒之前，一对勇敢的维京兄弟的史诗故事。  </p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422115742460_u162815.png" class referrerpolicy="no-referrer"><br></p><p>  </p><h3>《刺客信条：英灵殿》图像小说（法语版）（编年史系列）</h3><p>　　合作方 Glénat Editions 将于2021年秋季在法国地区发行这一原创图像小说。读者们将跟随一位充满抱负的刺客和一位基督教僧侣，从雷文斯索普到苏格兰的神秘高山展开一段探索之旅。</p><h3>网络漫画（原创系列）  </h3><p>　　这部韩国的网络漫画将作为《刺客信条：黑旗》的续集，进一步扩展爱德华∙肯威的故事。目前该漫画正由合作伙伴 Redice 制作中。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422115705835_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>

有声读物

 

</h4><h3>《刺客信条：大明风云》广播剧（编年史）  </h3><p>　　由中国领先的播客平台喜马拉雅制作和推出的由《刺客信条：大明风云》小说改编的广播剧由中国知名艺人、演员柳岩为主人公少芸献声。通过丰富的音效以及动人的声音演绎，这部广播剧将是听众完美沉浸于16世纪明朝中国的完美版本。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422115654796_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p>　　由于作品的种类越来越多元化，为了方便粉丝们更好地挑选适合自己的读物，官方对《刺客信条》已经推出以及即将到来的跨媒体作品进行分类，即“经典系列”“编年史系列”“原创系列”三个大类，“经典系列”基于游戏中的故事直接改编；“编年史系列”则是以备受喜爱的标志性刺客为特色的全新冒险故事；“原创系列”则是以全新主人公和时期编写的全新故事。
