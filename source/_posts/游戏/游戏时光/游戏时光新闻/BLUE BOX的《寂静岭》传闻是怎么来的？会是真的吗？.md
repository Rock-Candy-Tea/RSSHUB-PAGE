<div class="topicContent front_content"><p>　　还记得几年前小岛秀夫第一次公布《潜龙谍影5：幻痛》的时候吗？他编造了一个名为 Moby Dick Studios 的虚构工作室，并在几个月内扮演着一个名为 Joakim Mogren 的虚构开发者，宣布了一个名为《幻痛》的游戏，之后才公布就是《潜龙谍影5》。</p><p>　　最近的 BLUE BOX 游戏工作室和其作品“Abandoned”会不会是小岛秀夫的故技重施？毕竟小岛秀夫在宣传上创意百出，比如被取消的《寂静岭》的可玩预告《P.T.》。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617135446865_u77470.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　<a href="https://www.playstationlifestyle.net/2021/06/16/blue-box-game-studios-abandoned-kojima-silent-hill/">外媒 PlayStationLifeStyle 对此做了大量的研究</a>，虽然明显的线索比《幻痛》当时要少，但足以合理怀疑这一切并不单纯。</span><br></p><h4>BLUE BOX公布“Abandoned”</h4><p>　　2021 年 4 月 7 日，<a href="https://blog.playstation.com/2021/04/07/abandoned-a-cinematic-survival-sim-hits-ps5-later-this-year/">PlayStation 博客上出现了一篇文章</a>，公布了一款即将推出的 PS5 独占第一人称生存恐怖游戏，游戏名为“被遗弃”（Abandoned）。文章作者是 BLUE BOX 游戏工作室的游戏总监 Hasan Kahraman。看起来就像一篇平平无奇的预告新游戏的文章，没有引起什么注意。事实上还有不少的负面评论，指出的问题从低劣的配音工作到粗糙的帧率不一而足。</p><p>　　几个月后，BLUE BOX 游戏工作室透露，将发布一个 PS5 应用程序，让玩家在 PS5 上实时运行“被遗弃”这款游戏的预告片。这就有点独特了，甚至 PlayStation 工作室第一方的游戏都没有这么做过，所以这个不知名的游戏工作室会成为第一个，这引来了一些好奇的人。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/2106171333144_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　该应用程序最初宣布于 6 月 20 日发布，但随后又称将于 6 月 22 日发布。曾经有过什么游戏是以一个游戏来作为预告的？噢！这简直就像是某种《P.T.》重演。</p><h4>《寂静岭》传闻</h4><p>　　本周，BLUE BOX 游戏工作室发布了一条推文，表示“被遗弃”（Abandoned）并不是游戏的实际名称，游戏名称其实是以“S”开头，以“L”结尾的。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617133456590_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　此时，关于 BLUE BOX 是小岛的另一个宣传诡计以及“被遗弃”其实就是寂静岭的理论已经在流传，所以此言一出立刻引爆。该工作室迅速在推特上道歉，并称它无意让人们认为这是《寂静岭》。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617133511588_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　他们有时提到《被遗弃》（Abandoned）时会加引号，也确认过“Abandoned”只是开发代号。被遗弃这个词很可能暗示的是《P.T.》遭受的对待。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617133746176_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　他们还声称 4 月的预告只是一个开发公告，而“游戏本身从未被展示过”，那么 PS 博客上那些截图和视频会是什么游戏的呢？</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617133759427_u77470.jpg" class referrerpolicy="no-referrer"><br></p><h4>蓝色盒子意指PS工作室？</h4><p>　　看看 BLUE BOX 游戏工作室的标志，它和 PlayStation 工作室的标志惊人相似，外形轮廓和字体都是一样的。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617134521446_u77470.png" class referrerpolicy="no-referrer"><br></p><p>　　并且，PS 工作室的标志，在第一方游戏开始前，会有一个发出蓝色光芒的动画，简直就是一个“蓝色盒子”（BLUE BOX）。还有别忘了 PS4 和 PS5 实体游戏都是装在蓝色盒子里的。</p><h4>Hasan Kahraman何许人也</h4><p>　　BLUE BOX 游戏工作室似乎只有一名员工，即 Hasan Kahraman，巧合的是，缩写是HK，与小岛秀夫（Hideo Kojima）的名字缩写相同。尽管他们自称已经是成立多年的支援型工作室，但找不到任何关于他们过去所协助游戏的记录，也找不到其他开发者称他们正在制作“被遗弃”。这位唯一已知的开发人员 Hasan Kahraman <a href="https://www.linkedin.com/in/hasan-kahraman-346a7696/?originalSubdomain=nl">在 LinkedIn 上的资料</a>很简陋，除了声称他从 2014 年 4 月开始在该公司工作外，没有其他信息。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617134051357_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　接下来这个就有点特别。<a href="https://apps.apple.com/cn/developer/hasan-kahraman/id1519024838">Hasan Kahraman 有一个 Apple App Store 页面</a>，里面有一些低质量游戏，数量刚好足以让最后一个程序必须点“查看全部”才能看到。这唯一一个不是游戏的应用叫《Go Fit XL》，简介中有一处写道：“Personal Training ( PT ) Programs”。为什么要特地用括号写出缩写呢？这难道只是一个巧合？</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617131834276_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617131845575_u77470.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617131854508_u77470.jpg" referrerpolicy="no-referrer"><br></p><h4>BLUE BOX的社交频道</h4><p>　　<a href="https://www.blueboxgamestudios.com/">BLUE BOX 游戏工作室的网站</a>没有什么可看的，除了联系方式什么也没有。一个正常存在的游戏公司应该有的信息都没有，比如说招聘信息。</p><p>　　<a href="https://www.youtube.com/c/Blueboxgamestudios/featured">BLUE BOX 游戏工作室的 YouTube 账户</a>建立于 2015 年 1 月 20 日，没有任何视频，个人资料背景图是一张山丘的图片，山的低洼处有一些云雾。山加上雾…让人不禁想到寂静岭。 </p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617134255406_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　<a href="https://twitter.com/BBGameStudios">BLUE BOX 的推特账户</a>是 @BBGameStudios，用的是 BB 这个缩写。想到了什么？《死亡搁浅》的 BB。《潜龙谍影》系列的 Big Boss。<br></p><p>　　<a href="https://store.steampowered.com/curator/33594383-BLUE-BOX-Game-Studios/">BLUE BOX 的 Steam 主页</a>列出的游戏只有一个：抢先体验游戏《The Haunting: Blood Water Curse》。类似经典恐怖游戏《零》系列。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/21061713485341_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　列在 BLUE BOX 游戏工作室名下的，还有一款名为<a href="https://download.cnet.com/THE-WHISPERER-A-PARANORMAL-INVESTIGATION-GAME/3000-2097_4-78300932.html">《The Whisperer》的安卓游戏</a>，现在已经找不到了。它的图标是一个黑色的手印，与《死亡搁浅》中的图标非常相似。以及一个名为<a href="https://download.cnet.com/Tales-of-the-Six-Swords-Early-Access/3000-7536_4-78178455.html">《Tales of the Six Swords》的安卓游戏</a>，现在也找不到了，这两个游戏的下载量都只有 5 次。<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/21061713492979_u77470.jpg" class referrerpolicy="no-referrer"><br></p><h4>其他暗示</h4><p>　　据称，BLUE BOX 游戏工作室位于荷兰，成立于 2015 年，正好是《P.T.》发布一年后，小岛离开科乐美的时候。</p><p>　　同时，夏季游戏节上公布的<a href="https://www.vgtime.com/topic/1125581.jhtml">《死亡搁浅 导演剪辑版》预告</a>中，山姆·布桥斯的背上有一个荷兰国旗。正是 BLUE BOX 游戏工作室的所在地。他还钻进了一个箱子，某种程度上，有可能也是对 BLUE BOX 的暗示。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/17/210617135626934_u77470.png" class referrerpolicy="no-referrer"><br></p><h4>最后</h4><p>　　<a href="https://www.gamerevolution.com/news/684994-blue-box-game-studios-silent-hill-kojima-tweets">外媒 GameRevolution 提出了这样的观点</a>：BLUE BOX 否认与小岛和寂静岭有关系是巧妙的误导，因为如果 BLUE BOX 游戏工作室是一个实际上不存在的假工作室，那么他们不可能与任何东西有关系，所以他们并没有说谎。</p><p>　　目前，《寂静岭》IP 仍为科乐美所有，也没有任何关于 IP 易手的消息。然而，他们可能会将 IP 授权给外部工作室开发，这也不会多令人惊讶。</p><p>　　总之，BLUE BOX 游戏工作室的一切让人感觉并不简单，蕴含暗示。但这仅仅是误解和巧合吗？等到 6 月 22 日“被遗弃”预告的 PS5 应用程序发布，要么揭开秘密，要么继续出现新的暗示，这一切不远了。</p></div>
