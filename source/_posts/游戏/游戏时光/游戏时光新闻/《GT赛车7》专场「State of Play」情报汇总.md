</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/02/03/220203065445563_u524.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><b><br></b></p><p><b>　　模拟：</b></p><p>　　《GT赛车7》包含有对时间变化和天气的模拟，具体表现有昼夜、云层变化、晴雨状况、星空夜景等等。这些变化也会对驾驶造成影响。玩家可以通过仪表盘判断接下来的天气情况。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/02/03/220203072511642_u524.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　本作的 PS5 版支持自适应扳机和触觉反馈。按照官方的说法，玩家在驾驶时可以感受到轮胎锁定的振动，以及在不同材质路面上驾驶时不同的感受。如果过弯太快，会有前轮试图修正方向盘的震动反馈。</span><br></p><p>　　PS5 版也支持 3D 音效功能。</p><p><br></p><p><b>　　咖啡厅：</b></p><p>　　将为玩家展示「赛车文化」。玩家在其中可以查阅车辆菜单，了解收集车辆的渠道，以及车辆/车厂的历史故事和文化。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/02/03/220203072752831_u524.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/02/03/220203073913922_u524.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　在「咖啡厅」完成车辆菜单可以解锁新的车辆。此外玩家也可以前往「品牌中心」购买更多车辆，在「品牌中心」可以购买大约 300 种 2001 年以后的车辆。</span><br></p><p><br></p><p><b>　　调校：</b></p><p>　　游戏提供大约 60 种不同类型的汽车改装零配件。通过合适的调校可以大幅提高车辆性能。</p><p>　　此外也有丰富的外观改装部件，包含有 650 种以上的空气动力学零件、130 种轮胎、1200 种以上配色等等。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/02/03/220203073057874_u524.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/02/03/220203073311701_u524.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/02/03/220203073540907_u524.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p>　　改装完后还可以分享给其他玩家，供其他玩家下载。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/02/03/220203075641878_u524.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><b>　　回放模式：</b></p><p>　　本作除了传统的固定镜头的比赛回放模式外，还包含「音乐回放」模式。在「音乐回放」模式中，游戏会动态地生成镜头，配合背景音乐来回看比赛。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/02/03/220203080225962_u524.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><br></p><p><b>　　摄影模式：</b></p><p>　　游戏包含有来自全世界各地，2500 个以上的拍摄景点，并在其中拍摄 HDR 照片，照片具有极高的编辑自由度。此外玩家也可以在“赛事照相模式”中对自己参与过的比赛进行拍摄及编辑。
