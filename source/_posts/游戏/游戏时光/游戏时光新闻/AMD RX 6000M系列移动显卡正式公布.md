<div class="topicContent front_content"><p>　　在今天的 Computex 2021 展上，AMD 官方发布了AMD Radeon RX 6000M 系列移动显卡，画质增强超级分辨率技术（FSR）、AMD Advantage 游戏笔记本等最新情报。  <br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/01/210601153259201_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p>　　AMD Radeon RX 6000M 系列旨在为游戏笔记本带来一流的性能、难以置信的视觉保真度和身临其境的游戏体验，它建立在突破性的 AMD RDNA 2 游戏架构上，与 AMD RDNA 架构相比提供了高达 1.5 倍的性能提升，或在相同的性能水平上降低高达 43% 的功耗，同时还为下一代笔记本引入了 AMD Infinity Cache 和 DirectX Raytracing（DXR）等先进技术。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/01/210601153706438_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/01/210601153711360_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/01/210601153716190_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/01/210601153722520_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/01/210601153729848_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p>　　该系列显卡中最为旗舰也最快的显卡要数 AMD Radeon RX 6800M，可为玩家提供桌面级的性能并随时随地为超高帧速 1440P 游戏提供动力。6800M 的 GPU 能带来不同凡响的 1440P/120 FPS 性能，并充分融合了光线追踪、计算和传统特效，可实现逼真的视觉效果。6700M 的 GPU 也能提供 1440P/100 FPS 的游戏性能，新层次的视觉效果和效率，专为新一代高级游戏本和创作本而设计。6600M 的 GPU 则为轻薄笔记本打造了高刷新率的 1080P/100 FPS 的游戏性能。

<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/01/210601153323543_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p class>　　Radeon RX 6000M 系列带来的众多革命性的功能：  <br></p><ul><li><b>AMD Infinity Cache：</b>在 GPU 芯片上集成了高达 96MB 的末级数据缓存，减少了延迟和功耗，从而提供比传统架构设计更高的游戏性能。</li><li><b>AMD</b><b><b> </b>Smart Access Memory</b>：通过为 AMD 锐龙处理器提供对整个高速 AMD Radeon GDDR6 显存的访问，为特定 AMD 驱动的笔记本释放更高的性能。</li><li><b>AMD SmartShift Technology</b>：在 AMD 锐龙移动处理器和 AMD Radeon 显卡之间动态转换笔记本电脑的功率，从而进一步提升游戏性能。</li><li><b>AMD Radeon Chill</b><b>：</b>节能功能，可根据游戏中的动作调节帧率，从而帮助节省电池消耗。</li><li><b>AMD FidelityFX</b>：超过 45 款游戏支持，AMD FidelityFX 是一个开源的视觉增强效果工具包，供游戏开发者通过 AMD GPUOpen 使用。它是一个包括光栅化照明、阴影和反射效果，以及新的 AMD FidelityFX Super Resolution（FSR）技术的强大组合，可以以更小的性能开销将其集成到最新的游戏中。</li></ul><p><br></p><p>　　AMD Advantage 游戏笔记本电脑是 AMD 与其全球 PC 合作伙伴们共同探索得出的全新框架，将 AMD Radeon RX 6000M 系列移动显卡、AMD Radeon 软件和 AMD 锐龙 5000 系列移动处理器与 AMD 独有的智能技术、AMD FreeSync Premium 认证的显示器、高速 NVME 存储、最佳散热设计以及其它先进的系统设计特性相结合，使得该笔记本可为当今视觉效果要求苛刻的游戏提供超过 100 FPS 的流畅游戏体验、不插电全天候视频播放（超过 10 小时）、144Hz+ 高刷新率、逼真清晰的屏幕显示效果（超过 300 尼特）以及可定制化的温度调节以确保长时间游戏过程中的凉爽体验。  <br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/01/210601153352207_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p>　　2021 年将有多款 AMD Advantage 笔记本上市，如 RO<span style="background-color: initial; letter-spacing: 1.3px;">G Strix G15/17 AMD Advantage 版游戏本，配备AMD Radeon RX 6800M 显卡、AMD 锐龙 9 5900HX 移动处理器以及 AMD Smart 技术，该游戏本预计将于 6 月初在百思买（Best Buy）和其他大型零售商上市；OMEN 16 笔记本，配备 AMD Radeon RX 6600M 显卡、AMD 锐龙 9 5900HX 移动处理器以及 AMD Smart 技术，预计很快也将在京东上架；此外包括联想和微星在内的 AMD Advantage 笔记本预计也将于今年晚些时候上市。 </span></p><p>　　此外 AMD 宣布锐龙 R7-5700G 和 R5-5600G 两款 APU 的售价分别为 359 美元（约合 2286 元）和 259美元（约合 1649 元），预计将于 8 月 5 日开卖。  <span style="background-color: initial; letter-spacing: 1.3px;"><br></span></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/01/210601153359321_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: right;">来源：<a href="https://www.amd.com/zh-hans/press-releases/2021-06-01-amd-computex-2021-rdna-2-amd-advantage-0">AMD官网</a></p></div>
