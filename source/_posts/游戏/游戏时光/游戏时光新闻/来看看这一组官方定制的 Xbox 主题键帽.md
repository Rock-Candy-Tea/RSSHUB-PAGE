<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　近日有网友发现，Xbox 官方在 4 月初给多位西班牙语主播寄了一盒特别的礼物。官方既宣传了旗下 PC Game Pass 订阅服务，也因礼物内容收获了一波好评。</span><br></p><p>　　礼物盒上写着「Game Pass 也有 PC 游戏」，打开后可以在里面找到 1 年份的 PC Game Pass 兑换码，以及 8 个定制的 Xbox 主题键帽，这些键帽造型也都相当精致。<br></p><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/04/04/220404100740605_u32815.jpg" class style="width: 700px;" referrerpolicy="no-referrer"><figcaption contenteditable="true" class style="display: block;">这一盒来自人称「西语数毛社」的El Analista de Bits</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/04/04/220404101015397_u32815.jpg" class style="width: 700px;" referrerpolicy="no-referrer"><figcaption contenteditable="true" class style="display: block;">这一盒来自知名雷蛇合作主播 Amanda Ethieen</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　8 个键帽从左到右分别对应《帝国时代4》《喋血复仇》《光环 无限》《极限竞速 地平线5》《哈迪斯》《上古卷轴5 天际》《微软模拟飞行》《我的世界》，而这 8 款游戏目前也都在 PC Game Pass 阵容中。</span><span style="background-color: initial; letter-spacing: 1.3px;">据收到礼物的主播表示，这些定制键帽工艺细腻，手感也很好，部分键帽还有为 RGB 键盘准备的透光设计。</span><br></p><p>　　遗憾的是，这一组定制键帽是 Xbox 官方拿来宣传用的，数量稀少而且无法在市场上买到。<span style="background-color: initial;
