<div class="topicContent front_content"><p>　　2022年6月23日，索尼（中国）有限公司举办线上发布会，公布了电视新品的国行阵容：BRAVIA XR系列新品——画谛®系列8K Mini LED电视Z9K、4K Mini LED电视X95EK、搭载新型OLED面板（QD-OLED）的画谛®系列A95K、4K OLED电视A80EK/A80K，及已于3月上市的4K全阵列式背光液晶电视X90K/X91K。同时，BRAVIA系列新品——4K液晶电视X85K、X80K，也在本次发布会中亮相。</p><p>　　BRAVIA XR新品全系搭载进一步升级的XR认知芯片，结合XR动态背光系统大师版、XR特丽魅彩MAX技术、多声道屏幕声场技术等多项索尼音画控制技术，精准控制不同的显示面板，呈现索尼有史以来最高水准的音画效果。</p><p>　　同时，一经发布便揽获“CES创新大奖”的BRAVIA摄像头及其功能，为今年的电视产品提供了多样的交互方式和使用场景，其中，AI环境感应调节Pro为电视机前的用户打造理想视听位。此外本次国行阵容更包含了专为中国市场研发的智慧功能（如AI语音操控、智能互联；通过用户熟悉的“微信”实现用电视大屏视频通话等功能），带来贴合本地用户需求的智能娱乐使用体验，使今年的新品成为索尼有史以来最好用、易用、有趣的客厅娱乐中心。</p><p>　　作为最好的显示技术领导品牌之一，索尼继续践行“为用户提供更好的视听体验”这个电视业务五十多年发展历程中不变的目标和承诺；此外，进一步结合自身在影视、音乐、游戏等领域中的技术积累和产业链优势，基于对用户需求的精准洞察，以领先的技术为核心，为用户的家庭娱乐生活带去索尼的独特价值。今年，以XR认知芯片驱动的索尼电视系列新品，与索尼家庭影音产品（如搭载索尼360空间声场映射技术的HT-A9、HT-A7000）和PlayStation®5，为用户构建出如临现场般的沉浸式客厅娱乐体验。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/23/220623202849572_u54.jpg" class referrerpolicy="no-referrer"> </p><h2>XR认知芯片，电视“大脑”2022再进化</h2><p>　　回顾电视业务的发展历程，索尼凭借持续的技术创新，不断打破市场的既定印象和技术壁垒。早在20余年前行业里普遍以显示屏判定画质的固有认知下，索尼先发布局电视芯片的研发，并以首款电视芯片的成功、打破了行业认知，确立了芯片是决定音画体验的根本，带领电视产业向新的方向继续前进。</p><p>　　索尼芯片的实力，尤其是对电视音画表现算法的精准控制、栩栩如生的内容还原，离不开索尼在广播电视及专业影像业务中深厚的技术积累和产业链优势。拥有完整“拍、编、播”产业链以及全链路先进软硬件技术的索尼，具备内容制作、再现领域丰富的领先经验和海量影像、声音“大数据”积累。随着持续不断的技术迭代和自我突破，索尼实现了无论采用何种屏幕与技术，电视芯片都是打造索尼出众体验的核心。</p><p>　　2021年索尼再次领先行业推出具有认知智能的XR认知芯片，能够模拟人类认知事物的视听方式处理优化内容，实时交叉分析关联对象、优化处理图像元素。尤其是在人类视觉聚焦区域，呈现栩栩如生的画面，打造出令人印象深刻的逼真场景。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/23/220623202859297_u54.png" class referrerpolicy="no-referrer"><br><span style="background-color: initial; letter-spacing: 1.3px;">XR认知芯片</span></p><p>　　今年，这颗电视的最强大脑——XR认知芯片，在算法方面得到进一步增强，以更专业的方式增强画面的色彩、景深表现，并为多项音画控制技术提供更精准的智能识别与把控，全方位提升电视的音画表现。</p><h2>音画技术精控多屏，视听体验再次升级</h2><p>　　在XR认知芯片的驱动下，结合多项索尼音画控制技术，索尼电视精准控制LED、OLED以及今年新引入的Mini LED、QD-OLED，屏幕的优秀性能得到充分地调动和发挥，音画表现再次升级，实力还原创作者意图，打造出如临现场般的沉浸体验。</p><p>　　画谛®系列8K Mini LED电视Z9K，在XR认知芯片与XR动态背光系统大师版的驱动下，Mini LED背光得以实现更精准的控光；XR特丽魅彩Pro技术，在不损失亮度的前提下呈现优异的明暗对比、细腻的光线效果与绚丽的色彩表现。X-抗反射屏与广锐视角，减少屏幕反射并扩大观看角度，让用户与家人共享看电视的乐趣。在聆听方面，多声道屏幕声场的魔框声场扬声器及前置低音炮，营造出沉浸式3D环绕声场。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/23/220623202914602_u54.jpg" class referrerpolicy="no-referrer"><br><span style="background-color: initial; letter-spacing: 1.3px;">索尼画谛®系列8K Mini LED电视Z9K</span></p><p>　　画谛®系列A95K，采用了QD-OLED面板（新型OLED），在XR认知芯片的算法及新研发的XR特丽魅彩MAX优化下，展示出透亮饱满的色彩，以逼真的色彩还原现实世界。XR OLED对比度增强Pro则为A95K带来更高的对比度，令画面微小细节清晰可见。A95K搭载银幕声场旗舰版，配合更大更强劲的声场驱动器，精准匹配声音与画面，呈现“音画合一”的影院级视听盛宴。同时，A95K在设计上采用新颖的底座支架设计，前置可紧贴墙壁，后置则以全面屏展现无视觉干扰的沉浸画面。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/23/220623202925610_u54.jpg" class referrerpolicy="no-referrer"><br><span style="background-color: initial; letter-spacing: 1.3px;">索尼画谛®系列QD-OLED电视A95K</span></p><p>　　另一款使用Mini LED背光的4K电视X95EK，在XR认知芯片及XR特丽魅彩Pro、XR对比度增强旗舰版等画质控制技术的优化下，有效增强黑色层次、亮度和原生对比度，并拓宽色域，呈现更接近真实、更具立体感的画面。带有双低音炮的多声道屏幕声场，通过精准的声音带给用户更强的临场感。一体化无界全面屏设计，使用户无视觉干扰地专注于电视引人入胜的剧情之中。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/23/220623202940687_u54.jpg" class referrerpolicy="no-referrer"><br><span style="background-color: initial; letter-spacing: 1.3px;">索尼4K Mini LED电视X95EK</span></p><p>　　OLED电视A80EK/A80K同样具备出众的音画表现，XR认知芯片、XR OLED对比度增强Pro、XR特丽魅彩Pro、银幕声场旗舰版等索尼音画控制技术，加之极简的外观设计，让用户享受优质视听。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/23/220623202952817_u54.png" class referrerpolicy="no-referrer"><br><span style="background-color: initial; letter-spacing: 1.3px;">索尼OLED电视A80K</span></p><p>　　索尼对4K液晶电视产品线也进行了升级，出色的音画表现，为热爱观影、游戏的用户提供了丰富的选择，满足不同消费者的个性化需求。</p><p>　　4K液晶电视X90K/X91K，自3月初上市以来，便成为游戏玩家喜爱的PlayStation®5官配搭档。不仅原生具备游戏模式、支持4K 120FPS、VRR和ALLM，还搭载专为PlayStation®5打造的两项功能——HDR自动映射和内容自适应图像模式。在XR认知芯片、全阵列式背光、XR特丽魅彩PRO、XR对比度增强专业版、多声道屏幕声场等技术的控制和优化下，用户得以沉浸于逼真的游戏世界。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/23/220623203002242_u54.jpg" class referrerpolicy="no-referrer"><br><span style="background-color: initial; letter-spacing: 1.3px;">索尼4K液晶电视X90K</span></p><p>　　此外，尺寸丰富的4K液晶电视X85K及X80K，满足用户对音画品质与价格的双重需求，成为入手感受索尼电视音画实力的开始。</p><h2>BRAVIA摄像头，理想音画智慧呈现</h2><p>　　索尼电视通过芯片技术和音画控制技术的突破和升级，持续刷新消费者的视听体验；同时以不断精进的电视智能，提升对音画表现的智慧控制和优化。基于所处环境，比如不同的家居环境、甚至是一天当中随时间不断变换的光线和明暗场景，索尼电视通过智慧调节、始终为用户呈现理想的音画效果。2022年，依托新搭载的BRAVIA摄像头，音画表现的智慧处理再次得到突破，使“基于用户位置做实时、定制的优化呈现”成为可能。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/23/220623203018135_u54.png" class referrerpolicy="no-referrer"> </p><p>　　电视产品增加摄像头自然会带来交互的拓展性和可能性（如视频通话、AI健身、体感游戏等），这也是目前行业里对摄像头的常规功能定义。索尼对摄像头产品功能的定义中，除了实现以上常规交互外，首当其冲的还是要服务电视最本质的核心——音画体验。这便是2022 BRAVIA摄像头实现的诸多智慧功能中，具有索尼特色和价值的AI环境感应调节Pro。</p><p>　　该功能通过摄像头识别用户所处位置，自动调节优化，呈现出对用户来说理想的画面和声音。当用户与电视的距离产生变化、或是用户随意坐在沙发非正中央的位置时，电视可根据距离和观看角度，自动调整亮度与人声、左右声道的延迟与均衡，让声音与画面和谐一体。这样一来，只要用户坐在电视前，处处都是能享受索尼最佳音画的视听位，影院级体验轻松可得。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/23/220623203026700_u54.png" class referrerpolicy="no-referrer"> </p><h2>索尼构建客厅娱乐中心，易用好用本地化</h2><p>　　“用创意和科技的力量感动世界”是索尼不变的企业宗旨，其中DTC（直接面向消费者）的电子业务中，聆听用户的需求和反馈是非常重要的工作。索尼持续关注中国消费者的声音，聚焦用户在娱乐场景下的多元应用需求，打造易用好用的视听娱乐功能；融入本地化IoT生态，同时利用索尼自有生态优势，联合家庭音频产品、游戏主机等客厅娱乐设备，为创作者与消费者构建交流的桥梁。</p><p>　　2021年年底索尼已将关于中国的产品企划业务，从东京总部转移至中国，深入了解中国消费者的需求与偏好，从产品企划的初始阶段就贴合中国用户的使用习惯，同时积极携手广大的中国本地业务伙伴，一同为用户打造令其满意的好产品。索尼始终践行“植根中国、长远发展”的长久承诺，依托多样化的业务和人才优势、黑科技与娱乐内容组合、及本土化运营的先进管理实践，与利益相关方一起携手助力中国经济和社会的繁荣发展。</p><p>　　本次发布的电视新品均搭载用户所熟知的新版安卓TV系统，为用户带来安全性更强、流畅度更高、扩展性更强的交互体验。索尼打造“以内容为导向”的UI界面，开机后无需进入APP即可在首页快速寻找喜爱的内容；同时，将APP内容置于Tab栏，用户可以根据个人需求定制化，让操作体验更符合自己的喜好；此外，索尼电视支持京东小家与腾讯小微两大智能家居平台，极大地拓展了在AIoT智能家居领域的适配性和实用性。</p><p>　　新加入的BRAVIA摄像头，除了提升电视理想音画的智慧呈现外，还带来更多趣味功能。基于中国用户的使用习惯，索尼推出的视频通话功能，通过微信小程序即可实现，不限手机品牌、无需额外安装其他专属应用，只要有微信即可使用，操作简单便捷；还有AI健身、魔法相机等多元娱乐场景待消费者探索和体验；手势控制，只需记住几个简单的手势就可以控制电视，解放双手、轻松享受。</p><p>　　此外，BRAVIA摄像头配有实体物理开关、可一键关闭镜头，在带去有索尼价值娱乐体验的同时，也保护用户的数据安全和隐私，尊重用户的使用习惯。</p><p>　　除BRAVIA摄像头外，索尼还为国内用户打造了两款本地化设计的新遥控器：高端遥控器、以及一款材质略有不同的标准遥控器。高端遥控器将与OLED和Mini LED电视搭配组合，其它型号配备标准遥控器。高端款遥控器，采用了金属拉丝的外观设计，按键包覆了特殊涂层，易于清洁和使用；搭载自动背光功能外，还有寻找功能，当用户寻找遥控器时可让遥控器发出蜂鸣声，辅助用户迅速定位。此外，索尼也提供了远场语音功能，只需要说出唤醒语“小微小微”，就可通过语音操控电视。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/23/220623203036272_u54.jpg" class referrerpolicy="no-referrer"> </p><p>　　在内容方面，索尼积极合作New TV极光等国内头部视频平台，以海量、新鲜的电影、电视剧、综艺、网剧等影视资源构成内容矩阵。同时，索尼与New TV极光携手打造了“极光校准模式”，进一步提升流媒体内容的视觉效果，让用户充分体验索尼电视呈现的高水准音画。此外，同样归属索尼旗下、拥有蜘蛛侠等经典IP的索尼影业，不断拓展业务版图，借助专业影像及影视后期的技术积累，推出并持续扩充IMAX Enhanced等更高规格的精彩影视内容，在索尼电视的呈现下，为用户带去影院级观影体验。</p><h2>索尼电视+家庭音频+PlayStation®，客厅趣“玩”好搭档</h2><p>　　本次BRAVIA XR系列电视全系支持中置声场同步模式，适配搭载了索尼360空间声场映射技术（360SSM）的旗舰级家庭音频产品HT-A9、HT-A7000，在充分展现电视“音画合一”体验的基础上，进一步提升声音的定位表现，呈现原汁原味的精彩聆听体验。观影、游戏、欣赏音乐、健身、亲子等场景，用户均可感受具有索尼高附加值的优质体验。</p><p>　　同时，作为PlayStation®游戏王国的缔造者，索尼整合在游戏开发及硬件技术领域的优势，为电视产品打造专属“Perfect for PS5”游戏功能，带来PlayStation®与索尼电视生态联动的增益价值，打造更具沉浸感的游戏享受。</p><p>　　索尼电视享誉业界的音画视听体验，贴近用户、易用好用的智能功能，结合索尼丰富的娱乐生态，以此为基础，索尼为用户构建了智能时代的娱乐生活空间。未来，索尼也将贯彻匠心精神，以具有魅力的视听呈现，为追求更高生活品质的用户传递感动，带去具有索尼价值的产品和体验。</p></div>
