<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　日前，《潜行者2》开发商 GSC Game World 宣布将在这款未发售新作中加入 NFT（非同质化货币），结果在玩家之间引起巨大争议。目前官方已经在社交媒体回应了争议，宣布撤回在游戏中加入 NFT 的决定。</span><br></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2021/12/17/2112171745471_u292926.jpg" title class referrerpolicy="no-referrer"><br></span></p><p>　　GSC Game World 在早些时候宣布，会跟 NFT 平台 DMarket 合作在《潜行者2》中应用区块链技术，销售 NFT 产品。除了销售物品或建筑的玩家冠名权，以及独占的手套、纹身、皮肤、徽章、收集卡外，还有一项特别权益：只要玩家在 2022 年 1 月举行的拍卖会上竞拍到一个 NFT，即可获得前往开发商工作室进行全身扫描，建立自己的数字化身并成为游戏中一名 NPC 的机会。</p><p>　　该消息在公布后引起玩家们的极大不满。面对质疑，官方在社交媒体发布长文进行解释并请求玩家们理解。</p><p style="text-align: center;"><img style="width: 480px;" src="https://img01.vgtime.com/game/cover/2021/12/17/211217174303941_u292926.jpg" title class referrerpolicy="no-referrer"><br></p><p>　　官方表示 NFT 产品完全不是强制性的，拥有与否都不会影响游戏性和故事方面的体验；所有 NFT 销售收入都将用于本作的开发工作当中，并会控制 NFT 对游戏产生最低限度的负面影响。</p><p>　　不过发布后没多久，官方就删除了这条推文，然后重新发布了一条<a href="https://twitter.com/stalker_thegame/status/1471620399997886472">新推文</a>。新推文写道：</p><blockquote><p>　　我们听到了大家的诉求。根据我们收到的反馈，我们决定取消《潜行者2》中与 NFT 相关的内容。</p></blockquote><p>　　现在越来越多游戏厂商开始考虑在自家作品中加入 NFT 内容，比如早前育碧就<a href="https://www.vgtime.com/topic/1142355.jhtml">宣布进军 NFT 领域</a>，而 <a href="https://www.vgtime.com/topic/1139604.jhtml">EA 的 CEO 也表示 NFT 和「边玩边赚」类游戏是「我们行业的未来」</a>。但目前玩家们普遍比较抗拒厂商们的这类操作，像是<a href="https://www.vgtime.com/topic/1142434.jhtml">育碧之前的 NFT 项目宣传片就被玩家们踩到不得不隐藏视频</a>。</p><p style="text-align: right; ">来源：<a href="https://www.eurogamer.net/articles/2021-12-15-s-t-a-l-k-e-r-2-to-include-metaverse-of-nfts">Eurogamer</a> <a href="https://www.theverge.com/2021/12/15/22834567/stalker-2-nfts-metahuman-dmarket">The Verge</a></p></div>
