<div class="topicContent front_content"><p>　　Mojang 工作室宣布此前透露过的“Java+基岩”《我的世界》PC 平台捆绑包将于 6 月 7 日正式开售，这意味着玩家今后仅需一份游戏的价格可同时入手两个版本的游戏，已购买过游戏的玩家也可免费获得另一版本游戏。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/05/220605172740361_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p>　　为不知道各种版本区别的新玩家科普一下，这里的《我的世界 Java版》指的是 Windows7/8/10/11、Mac、Linux 平台的原生版本“我的世界”，而《我的世界 基岩版》是微软买下 Mojang 工作室后专门开发的主机、手机、PC 多个版本经过最终整合诞生的版本。众所周知《我的世界 Java版》开放的模组和材质资源让其一直焕发着社区活力，而《我的世界 基岩版》则有着自己独有的手柄支持、市场、性能优化、跨平台联机等特色。</p><p>　　不过这里次的“Java+基岩”PC 平台捆绑包限定了 PC 基岩版（又称 Win10 版），两个游戏打包以一款游戏的价格进行发售，已经购买《我的世界 Java 版》的玩家如果已经将 Mojang 账号迁移至微软账号，则账号下会自动多出一份“Win10 基岩版”游戏；同样的如果玩家此前在微软 PC 游戏商店购买过本作，那么你的微软账号可以直接在 Java 启动器中登陆游戏。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/06/05/220605172952101_u162815.jpg" class referrerpolicy="no-referrer"><br></p></div>
