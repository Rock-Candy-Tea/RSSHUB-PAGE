<div class="topicContent front_content"><p>　　Epic 游戏商店现已支持人民币支付购买游戏，国区默认使用人民币付款。</p><p class>　　这一更新后，商店内部分游戏的国区价格有所变化，比如《往日不再》价格从此前的 42.99  美元（折合人民币 276 元）下降到现在的 179 元（比 Steam 商店便宜了 100 元）。</p><p class style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/12/210512101701673_u524.jpg" class referrerpolicy="no-referrer"><br></p><p class>　　但《底特律 成为人类》的价格则从此前的 18.09 美元（折合人民币 116 元）变成了现在的 229 元。</p><p class style="text-align: center;"><img style="width: auto;" src="https://img01.vgtime.com/game/cover/2021/05/12/210512101720238_u524.jpg" class referrerpolicy="no-referrer"><br></p><p class>　　Epic 官博表示目前系统还在测试中，大家可以去<a href="https://weibo.com/7287111107/Kf5G6rKSy?from=page_1006067287111107_profile&wvr=6&mod=weibotime">微博</a>留言反馈问题。 </p><p class style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/12/210512102246363_u524.jpg" class referrerpolicy="no-referrer"><br></p><p class><br></p></div>
