<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　又是一年愚人节，大家想必已经等不及要看各大厂商使出浑身解数的整活了。为了与正经新闻有所区分，我们特意收集了这些愚人节玩笑供大家乐呵乐呵。</span><br></p><p>　　为庆祝距《进击的巨人》完结还有 8 天，漫画杂志《别册少年Magazine》官方在推特上举行了“选一个幸运观众送等身大 1:1 超大型巨人手办”活动。</p><p style="text-align: center;"><img style="width: 600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175024264_u93176.jpg" class title referrerpolicy="no-referrer"><br></p><p>　　需要说明的是，按照《进击的巨人》中的设定，别名为“破坏之神”的超大型巨人高达 60 米，我们比较熟悉的光之巨人迪迦奥特曼正常情况下也不过 53 米。真要做出来这可能会是世界上最大，造价最昂贵的等身大手办了。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175036728_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　手游这边的整活就花样很多了。知名的射爆手游《碧蓝航线》在推特上宣布将制作前日谭 OVA 动画《碧蓝航线 びそくぜんしんっ！ びそくのルーツ》。 </p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175048925_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　海报上的四位人物毫无疑问是刚刚完结的 TV 动画《碧蓝航线 微速前行！》里的四位主人公。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175058376_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　OVA 海报的画风我原本以为是来自著名的漫画《北斗神拳》。有一个同人画家很喜欢用北斗神拳的画风来绘制其他作品里登场的角色，以为这个整活是从那得到了灵感。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175132741_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　不过有评论区的朋友说应该是来自上世纪的《装甲骑兵》，对比之后发现画风和标题样式都一模一样。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401211404923_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　至于《Fate/Grand Order》与《Thunderbolt Fantasy 东离剑游纪3》的愚人节整活就比较绕，不清楚的朋友可能奇怪这俩是怎么搞到一起的，我来简单介绍一下。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175142707_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　毫无疑问，这次整活在背后穿针引线的就是“爱的战士”虚渊玄，型月宇宙中讲述第4次圣杯战争的小说《Fate/Zero》就是由他所写的。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175202772_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　虚渊玄特别喜欢霹雳布袋戏，还专门跑去找人家合作，最后如愿成为霹雳布袋戏《东离剑游记》的编剧。怎么想这次型月的整活都绕不开他。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175215274_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　《守望先锋》的愚人节整活倒是比较普通，只是给人物加上了滑稽的大眼睛。但相对来说也算是人畜无害，玩家可以直接参与并乐在其中。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175231206_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175237976_u93176.gif" class referrerpolicy="no-referrer"><br></p><p>　　拳头游戏的愚人节整活看起来也比较正经，他们宣布将推出一款恋爱模拟游戏《Valorant：特务终极恋曲》，预计 2021 年内推出。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175311775_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　在游戏里玩家将为了成为 Valorant 的特务而接受训练，在考验身心的严格训练之余，玩家还能与每一位 Valorant 的特务建立特殊的关系。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175319159_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　也有厂商整活太像真的，骗到不少人，甚至是专业的媒体。今天 343 工作室就在推特上宣布因为你懂的原因，《光环：无限》将延期至 2022 年夏季。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175332356_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　或许是考虑到《光环 无限》的实际开发情况，外媒 Kotaku 误将其当做真实的报道，并发了一篇新闻稿。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/21040117534643_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　在确认了这只是愚人节整活后，Kotaku 也在推特上进行了道歉。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175355850_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　整活大队里也有提前整活的，<a href="https://www.vgtime.com/topic/1118909.jhtml">比如《Muse Dash》前几天的愚人节特惠活动</a>，就让人有些摸不着头脑。一般来说愚人节整活都是集中在 4 月 1 日凌晨或者上午发，提前两天整活的我也是第一次见。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175410201_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　这份官方公告我们也在后台讨论了很久，最后才确定这是一个提前的愚人节整活，而不是真的干不下去要跑路了。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175426700_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　雷蛇也不会放过今年的整活大赛，它在今天宣布将推出一款名为「Razer Rapunzel Chroma Hair Dye」的染发剂，这将是世界上第一款 Chroma RGB 幻彩美发产品。</p><p>　　该产品提供了全光谱可定制的 Chroma RGB 幻彩光效，让你的每一根发丝都 R！G！B！起来，轻松成为大街上回头率最高的玩家。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175437295_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　值得一提的是 2016 年雷蛇在愚人节推出过雷蛇烤面包机的玩笑，说要推出一款“Razer Toaster”烤面包机。这款面包机拥有雷蛇一贯的黑色机身和荧光绿配色的三头蛇 Logo，可以在每一片吐司印上雷蛇的 Logo。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175454472_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　就在大家都以为是整活的时候，雷蛇居然真的在 2019 年推出了这款烤面包机。所以，这个染发剂是真是假，我们几年后再看看吧，万一真的有呢。</p><p>　　除了上面这些，有太像假的真新闻，以至于不得不在报道时特意提醒这并不是一则愚人节玩笑。比如因为《猫娘乐园》而闻名的开发商 Neko Works 就宣布<a href="https://www.vgtime.com/topic/1119054.jhtml">去年愚人节整活的《猫郎乐园》是真的</a>，并非愚人节玩笑。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175519377_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　鉴于这条消息实在是太像愚人节玩笑，而且公布的时间点非常微妙，我们也只能在报道时再三强调这并不是尝试的愚人节整活，而是真正的新作消息。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175529385_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　除了厂商，也有玩家参与到了今天的整活活动中。推特上名为 <a href="https://twitter.com/MH_WT_Houseki">@MH_WT_Houseki</a> 的用户就在今天“爆料”了怪物猎人新作《Monster Hunter：Dive》的存在。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175601224_u93176.jpg" referrerpolicy="no-referrer"><br></p><p>　　这款游戏是一款完全得开放世界游戏，将于 3021 年登录 PS4、PS5 平台，支持 VR、支持全球 13596 种语言以及最多 16 人的在线对战。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175553110_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　或许是名字太有诱导性，宣传图也像模像样，目前他已经修改了自己推特账户的名字，并声明这只是一个愚人节玩笑，请各位不要当真。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175614457_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　当然，愚人节玩笑也就图一乐，除了欢乐什么也不能带给我们。但我们在愚人节这天追寻的可不就是“快乐至上，理性靠边”嘛？就让愚人节整活来得更猛烈些吧。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175624583_u93176.jpg" class referrerpolicy="no-referrer"><br></p></div>
