<div class="topicContent front_content"><p>　　索尼算是正式<a href="https://www.vgtime.com/topic/1118849.jhtml">宣判了几个线上商店的死亡日期</a>，PS3 和 PSP 商店是 2021 年的 7 月 2 日，稍晚一点的 PSV 商店则是 8 月 27 日。关服消息一落地，海内外各大沉寂已久的论坛和板块又突然活跃起来。</p><p>　　坦白说阴阳怪气调侃的人有不少，比如下面这样的：</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width: 480px;" src="https://img01.vgtime.com/game/cover/2021/04/01/21040117174767_u229514.jpg" class="vg_insert_img_onfocus" referrerpolicy="no-referrer"><figcaption contenteditable="true" class>不敢相信我们会永远失去这样的隗宝</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width: 480px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401171834436_u229514.jpg" class="vg_insert_img_onfocus" referrerpolicy="no-referrer"><figcaption contenteditable="true" class>当索尼员工关闭 PS3 商店的时候，他们看到了什么？</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width: 480px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401173233766_u229514.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>是不是消失得太快了？</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　当然更多人还是表达了惋惜和不舍，以及试图赶在某些游戏彻底消失前做一些补救。</span><span style="background-color: initial; letter-spacing: 1.3px;">虽然还有三、四个月的「死缓」时间，但官方明显在逐步撤出老商店的服务器资源，导致玩家们用起来特别难受。</span></p><p>　　VGtime 俱乐部和贴吧就有不少人就提到了几个相同的问题。</p><p>　　总结下来大概包括：下载游戏显示 404 报错，开加速器和改 DNS 都没用；通过搜索「PS3」等关键字访问老商店的渠道被阻断；以及一大批游戏都没有购买选项，得用一些邪门歪道才行。<span style="background-color: initial; letter-spacing: 1.3px;">甚至有人决定凌晨爬起来试一试玄学，看能不能拿下一套数字版的《战神》合集。</span></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401172348907_u229514.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401172357763_u229514.jpg" class referrerpolicy="no-referrer"><br></p><p>　　事实上也不要以为本土作战的老外能好到哪去，Reddit 上某个大哥才刚刚买了一台蓝色 PSV，突然发现和「变砖」也没什么区别，因为根本打不开 PS 商店和浏览器。</p><p>　　还有人下载东西时遇到错误代码「NP-9948-0」，想从 PS3 传数据又弹出「E-80558309」。屡受折磨之下，研究出用信念解决问题的方法 —— 在睡眠模式下关闭 wifi，让机器躺上一个小时再试。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401172548169_u229514.jpg" class referrerpolicy="no-referrer"><br></p><p>　　可想而知，相当一批玩家陷入了「得赶快买游戏」的恐慌之中。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401172715860_u229514.jpg" class referrerpolicy="no-referrer"><br></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401174315890_u229514.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>赶在七月前再买一波 PS3 游戏，最后的机会了！</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　Reddit 上还有个投票的主题是「你决定在商店关闭前投入多少钱」，七成的人愿意为此买单。</span><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401172613194_u229514.jpg" class referrerpolicy="no-referrer"><br></p><p>　　特别像是一些从未出过实体版的独占游戏得到了青睐，有人根据 PSV、PSP、PS3 的库整理了推荐购买列表，油管也出现了长达三个半小时的「PSV 百款最佳游戏展示」。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401175001426_u229514.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　至于前文中因为 PS 商店拉胯没法购买、下载游戏的朋友同样想出一些办法。</span><br></p><p>　　由于官方明确说在商店关服后仍然可以「输入代码来兑换游戏和 PS+ 会员资格」，Humble Bundle   上对应 PS 内容的激活码似乎可作保存用途。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401172820568_u229514.jpg" class referrerpolicy="no-referrer"><br></p><p>　　可以预见的是，近期有一批实体版的 PSV、PSP、PS3 游戏可能会涨价。</p><p>　　比如亚马逊上 PSV 版的《传颂之物 二人的白皇》打折 25 美元，很快就被抢光了。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401172847391_u229514.jpg" class referrerpolicy="no-referrer"><br></p><p>　　当然捡漏的也有，10 欧元的 PS3 版《杀戮地带》，以及学校书店 2 美元一张的 PS3 光盘屡见不鲜。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width: 480px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401174356625_u229514.jpg" class="vg_insert_img_onfocus" referrerpolicy="no-referrer"><figcaption contenteditable="true" class>1.99 美元一张从学校书店淘来的</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　其实相似的局面，在几年前 Wii 数字商店关闭前就闹过一次，有人在 PSNPROFILES 论坛怒喷数字发行政策，引发了「数字党 vs. 实体党」的小规模战争。</span><br></p><p>　　当时我记得一名玩家通过 PS 商店为数字版辩护，指出索尼的服务器结构与任天堂截然不同，因为分离 PS3 服务器并把它关掉的成本要比继续运行更高，没想到两年后就被打脸……</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401173112411_u229514.jpg" class referrerpolicy="no-referrer"><br></p><p>　　不过任天堂确实狠一点，2019 年时甚至关闭了下载已购买游戏的渠道。</p><p>　　在抢救游戏方面，收藏家 Shawn Sackenheim 正在做一些努力，从 90 年代中期开始就一直在撰写被删除游戏的故事，他把相关的日期、数据和链接统计起来，最后创建了网站 Delisted Games。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401172952494_u229514.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　当某些游戏从商店消失的时候，Delisted Games 会整理一部分以合法渠道能够玩到的方法，目前已经收录了 1279 款游戏。</span><br></p><p>　　即使到了数字时代，也没有什么东西是永恒的，只能庆幸还有这样的团队存在了。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/01/210401174222395_u229514.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>那些已废弃的数字商店</figcaption></figure></div><p><br></p></div>
