<div class="topicContent front_content"><p>　　大家与主机、PC游戏经历了漫长岁月，想必对它们的过去略知一二。移动端游戏虽然与二者相比仍显年轻，但同样历史悠久。它随着2012—2014年的手游产业爆发开始在游戏行业占据一席之地，进入了广大玩家们的生活。如今的它们经历了长时间的进化，已经衍生出多种种类、玩法，成为了大量玩家活用碎片化时间的娱乐手段。其中也涌现出了不少在玩法、画面、音乐等方面素质过人的优秀作品。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/23/211223174542486_u54.png" class referrerpolicy="no-referrer"><br></p><p>　　为了挖掘来自于各个平台、在这一年间有着闪光之处的游戏作品，也是为了向这些作品的创作者传达来自业界和玩家们的肯定之声，今年UCG将与OPPO游戏中心合作，联合举办首届“OGA（OPPO Game Awards）年度游戏大赏”。</p><h2>关于OGA的奖项</h2><p>　　如今的电子游戏类型迥异，风格多彩，蕴含着丰富表现形式。因此本届“OGA年度游戏大赏”设立了年度最佳游戏、年度最佳剧情、年度最佳音乐、年度最佳创意、年度最佳美术、年度最受期待游戏奖这6个针对移动端游戏的奖项，还有针对主机端的UCG特别推荐奖，共计7个奖项，来让大家能最直观有效的看到今年有哪些游戏作品理应得到应有的关注和肯定。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/23/211223174607633_u54.png" class referrerpolicy="no-referrer"><br></p><h2><span style="background-color: initial; letter-spacing: 1.3px;">谁来为OGA提名</span></h2><p>　　俗话说“一千个人眼中有一千个哈姆雷特”。在提名哪些游戏参与本次大赏上，是绝对不能听取一家之言的。所以本次“OGA年度游戏大赏”特别邀请了业界知名的游戏开发者、游戏媒体作为嘉宾阵容，为本次参与评选“OGA年度游戏大赏”各个奖项的游戏作品进行前期的提名推荐。其中就包括了《暗黑破坏神：不朽》的游戏总监-Wyatt Cheng、游戏葡萄主笔-赵伦等多为业界“老炮”。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/23/211223174622333_u54.png" class referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　除了上述具体到个人的业界“老炮”，还有不少知名的游戏媒体，包括：机核网、游研社、游戏葡萄、JUMP、游民星空、游戏茶馆，当然也少不了我们UCG啦！</span><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/23/211223174632201_u54.png" class referrerpolicy="no-referrer"><br></p><h2>他们如此提名</h2><p>　　各位嘉宾会综合自己对游戏游玩后的体验和玩家的评价等因素，给到除“UCG特别推荐奖项”外的其余年度奖项各4款提名游戏。而后将针对各个奖项下所提名游戏所出现的频次进行汇总筛选，来最终确认年度最佳游戏、年度最佳剧情、年度最佳音乐、年度最佳创意、年度最佳美术奖项下各4款游戏提名、年度最受期待奖项下的6款游戏提名。UCG还会为“UCG特别推荐奖项”给出四款提名游戏</p><p>　　说到这里，大家应该对 “OGA年度游戏大赏”有了一个基本的认识，同时也对业界老炮们推荐了哪些游戏，这些游戏又入围了什么奖项感到好奇与期待吧？</p><p>　　比如游戏葡萄以“用「氤氲丹青」呈现了一种更鲜活的国风美学”的评价提名《忘川风华录》入围最佳美术奖。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/23/211223174643761_u54.jpg" class referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　JUMP提名推荐入围最佳剧情的《坎公骑冠剑》，给出的推荐语便是：在快速游戏的时代，让人回归初心的“塞尔达式”复古RPG精致之作。</span><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/23/211223174659361_u54.jpg" class referrerpolicy="no-referrer"><br></p><p>　　大家想要完整查看嘉宾推荐的奖项入围名单，那就千万不要错过接下来开启的玩家最终投票活动。</p><h2>好的游戏需要你的肯定</h2><p>　　入围名单只是开始阶段，而这些奖项到底花落谁家，还需要大家的投票决定。“2021 OGA年度游戏大赏”玩家投票通道会在2021年12月23日到2022年1月5日的14天中，开启。</p><h3 style="text-align: center;"><a href="https://activity-cdo.heytapimage.com/cdo-activity/staticActivity/PbjY0P/htmls/PbjY0P.html?actId=47089&at=1&ta=0&maxage=0&c=0&preload=1&channelName=ucg#/home">投票入口</a></h3><p>　　每天大家都能在活动页面上，在每一个奖项所提名的游戏中选择一款投下自己宝贵的一票。在每天参与完成投票后，还会获得一次抽奖机会，有机会获得5G手机、手柄、键盘、鼠标以及OGA与UCG杂志联名定制的礼品。</p><p>　　在2022年1月6日投票结束后，“OGA（OPPO Game Awards）年度游戏大赏”会就最终投票结果和各个奖项花落谁家进行展示和颁奖。希望大家都能动动手指多多参与投票活动。大家的参与，不光是对这些游戏作品的肯定和鼓励，也是自己游戏态度的展示。</p><h2>最后</h2><p>　　本届OGA年度游戏大赏的发声或许并不是特别洪亮，但我们还是希望为玩家和业界做出的这次新的尝试，可以为“发现好游戏”和“玩好游戏”奉献一份力量！。</p></div>
