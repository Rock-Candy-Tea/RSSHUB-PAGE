<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　国内独立游戏发行商 Gamera Game 于今日宣布，将启动一项面向大学生和微小独立游戏开发团队（3 人及 3 人以下独立游戏开发团队）的帮扶计划——叶绿素计划。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/10/211210161839556_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　作为发行过《戴森球计划》《了不起的修仙模拟器》《港诡实录》《烟火》等几十款独立游戏的国内发行商，Gamera Game 一直致力于将好玩有趣的游戏带给更多的玩家。伴随着叶绿素计划的启动，Gamera Game 将在发行的基础上给予大学生和微小独立游戏开发团队更多的扶持和帮助。所有加入“叶绿素计划”的项目，在销量达到 10000 份之前，Gamera Game 将不参与任何形式的收入分成。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/10/21121016192067_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　除 Gamera Game 自身之外，不少游戏行业的从业者也将为“叶绿素计划”助力，包括媒体、主播、社区平台以及翻译、配音团队等等。相信他们的加入将会为“叶绿素计划”的游戏开发者们带来更有力的帮助。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/10/211210161928980_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/10/211210161940862_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/10/211210161948905_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　“叶绿素计划”首批包含 9 款国人开发的游戏，分别是：</span><br></p><ul><li>烟火</li><li>三伏</li><li>寻宝浪客·念</li><li>最末行程</li><li>开普勒斯</li><li>瓶子先生与他的梦想</li><li>6栋301房</li><li>乌合之众</li><li>窗户之外</li></ul><p><span style="background-color: initial; letter-spacing: 1.3px;">　　其中《6栋301房》《乌合之众》和《窗户之外》这三款游戏是此前并未公开过的全新作品。</span></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　《6栋301房》是一款由中国传媒大学学生开发的独立游戏，围绕着一名老年阿尔茨海默症患者的日常生活，以解谜的形式，用细腻的笔触让玩家身临其境地感受阿尔茨海默症患者在生活中面临的困扰。而《乌合之众》和《窗户之外》则是首次公开了游戏 logo，关于这两款游戏的更多信息将在之后陆续公开。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/10/211210162004364_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/12/10/211210162011701_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　Gamera Game 的负责人表示，未来将会有更多的国产独立游戏加入到叶绿素计划当中，也希望能帮助更多的独立游戏开发者，将他们的创意和作品呈现在玩家面前。</span></p></div>
