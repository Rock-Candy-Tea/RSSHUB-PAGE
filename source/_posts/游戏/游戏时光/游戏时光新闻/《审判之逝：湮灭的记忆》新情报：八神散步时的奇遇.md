<div class="topicContent front_content"><p>　　世嘉旗下“审判之眼”新作《审判之逝：湮灭的记忆》即将于 2021 年 9 月 24 日登陆 PS5/PS4/Xbox Series X|S/Xbox One，官方放出了本作的新情报，介绍了八神在散步时能体验到的各种内容。</p><hr class="vg_hr" contenteditable="false"><h4><span style="background-color: initial; letter-spacing: 1.3px;">让“散步”更加充实的各种内容</span></h4><p><span style="background-color: initial; letter-spacing: 1.3px;">　　除了在神室町与伊势佐木异人町解决案件与谜团的“支线案件”以外，八神还能在街上找到各种物品、与猫咪互动等，采取多种行动。</span></p><h3><span style="background-color: initial; letter-spacing: 1.3px;">破解街上松鼠涂鸦的谜团，并获得特别道具的“搜寻涂鸦”</span></h3><p><span style="background-color: initial; letter-spacing: 1.3px;">　　在神室町和伊势佐木异人町的许多地方不知为何有松鼠的涂鸦，使用能够通过涂鸦中松鼠的姿势联想到的侦探小工具，便能够找到特别道具。</span></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　有些涂鸦在十分隐蔽的地方，因此移动的过程中要留心仔细观察周围。找到全部的松鼠涂鸦，并解开为何各个地方都有涂鸦的谜题。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153017577_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>发现了竖起耳朵在听着什么的松鼠涂鸦。</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153022556_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>使用侦探小道具“助听器”便能发现发出声音的东西就在附近。</figcaption></figure></div><h3>与猫咪沟通，并和它们处好关系</h3><p>　　在街上时有机会发现“猫咪聚集地”。给猫咪起名字，喂东西，摸摸它们以示喜爱吧。与猫咪的关系变好后，会收到来自猫咪的礼物，并且猫咪还会偶尔出现在事务所中。<br></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153139280_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153145796_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>好好记住猫咪的名字是与猫咪进行交流并处好关系的关键。</figcaption></figure></div><h3>返还“掉落物”</h3><p>　　在街上散步时，偶尔会看到闪闪发光的“掉落物”。通常失主就在附近，捡起掉落物并与失主搭话吧。物归原主后，
失主会表示感谢并赠予礼物。</p><p>　　这些礼物包括平常难以获得的稀有道具，主动拾取掉落物来帮助正在烦恼的人们吧。<br></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153245398_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>在前方发现了掉落物以及不停张望的失主。</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153251934_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><h3>回复体力之外，还能获得特别效果的“用餐”</h3><p>　　在饮食店用餐时，选择菜单中带有图示的料理就餐，八神便能够获得各种特别效果，包括一定时间内敌人更容易掉
落道具、战斗中获得的 Sp 增加等，积极地用餐来获得各种效果加成吧。当然，进餐还能回复体力。<br></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153413248_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>不要错过菜单中左上角带有标记的菜品，积极点单吧。</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153432618_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>图中此次进餐获得了更容易找到稀有素材的效果。</figcaption></figure></div><h4>帮助八神进行各种活动的便捷手机功能</h4><p>　　不仅能够习得技能，本作的手机还搭载了拥有各种功能的 App。</p><p>　　“拍照功能”以及可以回顾案件经过的“CASE FILE”自不必说，本作还加入了能够寻找纠纷发生地的“热门发文搜寻器”、呼叫
出租车的“TAXI”等功能。</p><p>　　在此介绍其中最让人耳目一新的“校园日志”、“TownGo”以及
“调查小工具”App。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153658294_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><h3><span style="background-color: initial; letter-spacing: 1.3px;">帮助掌握青春群像剧进度的“校园日志”</span></h3><p><span style="background-color: initial; letter-spacing: 1.3px;">　　本作的一大内容就是“青春群像剧”。除了八神担任外聘教师的悬疑小说研究社外，八神将潜入以高中社团为主的 9
个团体。“校园日志”便是确认这些团体当前状况的 App。可以通过 App 确认各个团体当前的剧情进度、进入到下一
阶段的条件，以及各个团体的人际关系图等等。</span></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　此外，还能确认当前八神作为外聘老师的指导力。有时指导力未达到指定数值时便无法推进故事，根据自己的需求及时确认数值吧。指导力可以通过参与团体活动来增加。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153708978_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>可以清楚了解各个团体当前进度以及进入下一阶段所需条件的便利App。</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/2109161537168_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>随着故事发展，各个团体的人际关系也会发生变化，对掌握各个团体当前的状况也很有帮助。</figcaption></figure></div><h3>为街上的活动和战斗增添更多乐趣的任务 App“TownGo”</h3><p>　　前作中的 App 名为“KAMGO”，不过本作中故事舞台变为神室町与伊势佐木异人町两个城市，因此更名为“TownGo”。</p><p>　　本作的 App 中设置了更加丰富多彩的任务，能够帮助玩家更深入地游玩。任务根据难度分为金/银/铜三个等级，难
度越高完成后获得的 Sp 越多。<br></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153838762_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>本作特色的“青春群像剧”也有对应的任务。</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153856426_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><h3>“侦探小工具”可以从手机启动！</h3><p>　　本作中有数个“侦探小工具”可用于协助八神的调查，可以通过智能手机来使用。在街上时，还可以使用侦探小工具来呼叫侦探狗。此外，本作同样可以在室外放出无人机来进行调查。操纵无人机飞往八神无法到达的高处并进行调查吧。<br></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153929675_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916153942635_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><h4>活用游戏技巧的高难度任务模式“审判特训”</h4><p>　　在“审判特训”中，可以游玩附带各种条件的战斗、体能、匿踪、追逐等任务。“审判特训”会根据游戏正篇的进度，解锁新的任务。</p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　初次完成各个任务后会获得能在正篇中使用的道具，还有只能通过“审判特训”获取的道具。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916154126304_u8855.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/09/16/210916154140100_u8855.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>正篇中出现的敌人会在强化后出现在部分战斗任务中，可以在这里挑战有别于正篇的多种战斗任务。</figcaption></figure></div><p><br></p></div>
