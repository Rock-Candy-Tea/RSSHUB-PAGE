<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　微软宣布，直通存储 API（DirectStorage API）现已脱离开发者预览，正式在 Windows 平台上面向全体开发者推出。</span><br></p><p>　　此前，想要参加直通存储技术开发者预览的公司与个人，需要报名并与微软签署 NDA 保密协议。而此次该技术的正式推出，意味着开发者们不再需要与微软直接联系，下载更新用于 DirectX 12 游戏的调试工具 <a href="https://devblogs.microsoft.com/pix/pix-support-for-directstorage/">PIX for Windows</a> 即可体验该 API 带来的好处。</p><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/03/15/220315110513638_u32815.jpg" class="vg_insert_img_onfocus" style="width: 600px;" title referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice" style="display: none;"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　对于玩家来说，尽管拥有支持 NVMe 协议的固态硬盘就能很大程度上体验到该技术带来的提升，但微软仍然推荐各位玩家使用针对存储做了优化的 Windows 11 系统。</span><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　此外，AMD 将携手 Luminous Productions，在 2022 年 GDC 开发者大会上展示《Forspoken》对直通存储 API 的应用，并带来实机技术演示。</span><br></p><div class="vg_insert_img"><figure contenteditable="false"><img src="https://img01.vgtime.com/web/topic/2022/03/15/220315110524732_u32815.jpg" class style="width: 600px;" title referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice" style="display: none;"></figcaption></figure></div><p><br></p><p style="text-align: right; ">来源：<a href="https://devblogs.microsoft.com/directx/directstorage-api-available-on-pc/">微软</a></p></div>
