<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　</span><span style="background-color: initial; letter-spacing: 1.3px;">　　不知道有没有朋友发现，CDPR 在修复游戏各种 Bug 的同时，宣传部门也没有闲着，放出了游戏里很多有趣的数据，比如 23.5% 的玩家升满了街头声望，老维总共收到 1470 亿欧元的还款，约 400 万个 V 获得了强尼·银手的保时捷 911 Turbo 等。</span></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/15/210715183801624_u93176.jpg" class referrerpolicy="no-referrer"><br></span></p><p>　　原本大家觉得这些数据还有点意思，但在使用同样宣传手段的时候，《赛博朋克2077》并没有质的变化，CDPR 也没有给游戏加新内容，给人感觉这些宣传十分僵硬。时间一久，就有玩家表示不想再看这些毫无新意的数据，开始用自己的方式表达对 CDPR 宣传活动的不满。</p><p>　　在 Reddit 上有网友自行发起了名为“<a href="https://www.reddit.com/search/?q=Cyberpunk%20By%20Numbers">Cyberpunk By Numbers</a>”的话题，不是讨论官方已经放出的数据，而是用魔法来打败魔法，用数据的形式来吐槽游戏里那些不合理的地方。</p><p>　　最初是一个名为 pablo397 的网友在《赛博朋克2077》的游戏版发了一张图：“<a href="https://www.reddit.com/r/LowSodiumCyberpunk/comments/oisyoh/cyberpunk_by_the_numbers_v_purchased_0_apartments/">玩家总共购买了 0 栋公寓</a>”，本意是想取笑 CDPR 公布的这些数据没什么意义，希望这个的梗图能引起 CDPR 的注意，让他们尽快推出更新。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/15/210715182436950_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　或许是说到了玩家的心坎里，这几天“Cyberpunk By Numbers”这个话题在 Reddit 上有着相当高的热度，目前该话题下点赞量最高的帖子是“<a href="https://www.reddit.com/r/cyberpunkgame/comments/ojfm3r/cyberpunk_by_numbers_v_has_modified_its/">赛博朋克统计数据：V 总共修改了 0 次外观</a>”，共有 2.24 万网友点赞。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/15/210715180225814_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　这个“0 次”就很有灵性，你看夜之城的科技那么发达，玩家也有钱到义体医生那里消费，不管是从游戏的前期宣传还是从世界观设定来看，修改角色外观应该是《赛博朋克2077》里必不可少的功能。但游戏并没有为玩家提供修改外观的功能，所以这项数据绝对是正确的。</p><p>　　这种新颖的吐槽方式得到了很多人的回应，比如网友 presidentdinosaur115 就在回复里说：“这明明是一个设定在赛博朋克世界的游戏，但我除了身上的衣服和自己的手臂什么都没法自定义，我甚至没有办法更换我的发型。”</p><p>　　在“Cyberpunk By Numbers”话题下点赞量第 2 高的帖子是：“玩家共花费了 0 欧元在车辆的改装上”。虽然汽车在《赛博朋克2077》里扮演着重要的角色，但是你并不能给你的爱车喷上其他颜色的喷漆，也无法对车辆的性能进行改装，甚至不能买个车库，在车库里看看你都买了哪些好车。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/15/210715180237706_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　在做梗图吐槽游戏里缺少的功能时，网友们也不忘吐槽游戏里那些做得不怎么好的地方：“夜之城的警察在玩家背后传送了 730 万次。”《赛博朋克2077》里警察出动的机制非常神秘，警察不是接到报警后才开着警车赶往现场，而是直接刷新在玩家周围，出警神速。尽管后续补丁对警察的机制做出了一些调整，但效果显然没有让所有玩家满意。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/15/210715180250759_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　这个 730 万次当然不是 CDPR 官方统计的数字，帖子里还有不少人认为 730 万有点低了，楼主的格局不够大。</p><p>　　除了这些花时间做图的网友，也有人在评论区写下自己的创意，比如：有 0 个路人注意到在街上裸奔的 V、解决了夜之城所有赛博精神病案件的玩家数为 0 、V 有 1234567 次穿着衣服洗澡……</p><p>　　自从发现了这种乐趣后，越来越多的人开始加入梗图的创作队伍或是在帖子下面发表自己的意见，创意也不再局限于游戏内的某项功能，甚至延伸到了游戏本身。</p><p>　　像是这个“在 2020 年圣诞节到今年 6 月期间，共有 0 名玩家在 PlaystationStore 购买了《赛博朋克2077》。”说的就是去年游戏从 Playstation 商店下架，<a href="https://www.vgtime.com/topic/1126363.jhtml">直到最近才重返商店</a>一事。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/15/210715180307846_u93176.png" class referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　现在“Cyberpunk By Numbers”这个话题下，网友已经不只是在单纯的吐槽 CDPR 在网上公布的数据，吐槽游戏里的各项不合理功能，还包含了玩家对游戏的理解，以及讨论游戏未来更新时可能会加入的功能。尽管对游戏的吐槽占了回复的绝大部分，并且吐槽的内容来来回回就是那几点，网友们还是乐此不疲地参加相关的讨论。</span><br></p><p>　　网友 Kiwiteepee 就认为这个话题简直有股魔性：“我真的很喜欢这些帖子，尽管它们评论区的内容基本上都一样。像是游戏缺少的功能、影响体验的 Bug、该骂的是不是开发人员而是管理层、美术设计优秀、警察和路人 AI 太蠢了、不能更改角色外观……哪怕 100 条回复都是相同的内容，我也能看上一整天。”</p><p>　　我觉得 papi1368 的评论可以很好地回答这个问题：“这就像一群热爱这款游戏的玩家在期待游戏会变得更好。”</p><p>　　现在还在讨论《赛博朋克2077》的网友，基本上都是这款游戏的核心玩家，大家聚在一起用这种奇怪的方式分享自己在夜之城的所见所闻所感，甚至是对游戏未来的期待。</p><p>　　我们不知道这项活动会不会传到 CDPR 耳边，对游戏的更新产生一些影响，尽快推出一些让大家都满意的更新。但在《赛博朋克2077》变成大家认可的品质前，还会有更多的“Cyberpunk By Numbers”梗图出现在网络上。</p></div>
