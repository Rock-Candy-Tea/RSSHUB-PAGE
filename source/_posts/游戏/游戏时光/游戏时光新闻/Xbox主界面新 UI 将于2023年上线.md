<div class="topicContent front_content"><p>微软现已开启 Xbox 主界面新 UI 的测试，这一界面预计在 2023 年正式上线，面向所有玩家开放。</p><p>以下是官方放出的预览图，可以看到新出现了「Jump In」一栏排列有最近游玩的游戏，首个游戏的图标并不像现在 UI 那样更大一些。</p><p>此外诸如设置、商店、搜索和「我的游戏与应用」这类常用应用也以新的样式在主界面呈现，让玩家更便捷地使用。</p><p>当玩家向下滚动屏幕时，还会看到基于玩家游戏偏好给出的精选类别和推荐。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/09/09/220909110036407_u524.png" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/09/09/220909110126895_u524.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p style="text-align: right;">来源：<a href="https://news.xbox.com/en-us/2022/09/08/xbox-insiders-help-shape-the-new-xbox-home-experience/">Xbox</a></p></div>
