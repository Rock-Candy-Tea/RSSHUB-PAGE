<div class="topicContent front_content"><p>　　世嘉旗下“审判之眼”新作《审判之逝：湮灭的记忆》即将于 2021 年 9 月 24 日登陆 PS5/PS4/Xbox Series X|S/Xbox One，官方今日公布了本作“青春群像剧”中的 “拳击馆”与“飙车族”两大玩法。  <br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/29/210729181018668_u162815.jpg" class referrerpolicy="no-referrer"><br></p><p>　　此前官方曾介绍过八神合理利用了外部人员也可以指导学校社团的制度，主动提出担任诚棱高中社团的顾问老师，在悬疑小说研究社、热舞社、机器人社这几个社团中活跃，享受“青春群像剧”的同时挖掘案件的关键线索，本次介绍的就是另外两个学生团体“拳击馆”和“飙车族”，玩家同样可以在这里挑战对应的迷你游戏。</p><h4>拳击馆

</h4><p>　　八神从“悬研社”社长天泽镜子那里得知，有学生在教授的地下网站接受替人寻仇的委托，为此八神潜入了位于伊势佐木异人町的拳击馆“轰木 拳馆”，对这里练拳的诚棱高中二年级学生“押切拳也”进行了调查。 押切拳也有着无与伦比的拳击天赋，但另一方面，最近在本地发生的突然袭击他人并进行单挑的“试拳打人”事件 的犯人也被怀疑是他。据说拳也会对能够与自己抗衡的强者敞开心扉。 为了追寻事件真相，八神不得不也开始在拳馆练拳，但在这里等待他的是诸多性格乖张的拳馆学员以及拳也不为人知的过去。 并且还与某个意料之外的人物重逢。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/29/210729181035313_u162815.jpg" class="vg_insert_img_onfocus" referrerpolicy="no-referrer"><figcaption contenteditable="true" class>  轰木拳馆的主人轰木豪，指导风格非常粗暴  </figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/29/210729181053562_u162815.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>  拳馆内还聚集着许多有前科和原黑道的好斗之人  </figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/29/210729181102703_u162815.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>   “试拳打人”事件的嫌疑人押切拳   </figcaption></figure></div><h4>迷你游戏“拳击”

</h4><p>　　在拳击馆进行的练习赛与通常的战斗在操作上有相似之处，但需要将头部和身体区分开来进行上下段攻击、防御，对头部进行攻击可以给对方造成较大伤害，而对身体进行攻击则可以削减对方精力。精力以圆形量表显示，过少时将无法连续攻击，若耗尽则会失去活动能力。此外体力量表耗尽则会直接倒地，1 回合让对手倒地3次，或对手在击倒状态下无法起身便可获胜。根据对手的特点，可以选择逐渐消耗其精力的战术， 也可以一口气削减其体力。出拳的种类有伤害低速度快的刺拳、高伤害的直拳、靠近才能击中但威力巨大可以破坏敌人防御的勾拳等，可以使用诸多拳击独有的招式。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/29/21072918112083_u162815.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>引诱对方攻击自己的身体、用踮步躲避攻击、趁机用直拳狠狠击打其面部</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/29/210729181134365_u162815.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>  发动必杀的 EX 动作瞬间让敌人倒地   </figcaption></figure></div><p><span style="font-size: 30px; font-weight: bold; letter-spacing: 1.3px;">飙车族     </span><br></p><p>　　诚棱高中三年级学生周防游马，是一名成绩优秀、品行端正的优等生，在校内还是被诸多学生畏惧，严格取缔违反校规行为的“风纪委员长”。某一天，他为了说服诚棱高中的学生离开异人町最大的飙车族团体“天堂造物”而直接 与“天堂造物”进行谈判，但不知为何却在一夜之间成为了飙车族的总代表。面对突如其来的变化，天泽认为这背后有“某人的意图”，便委托八神潜入飙车族团体进行调查。潜入调查的目的，是找出周防突然发生变化的原因，并让沉迷于危险的“死亡竞速”的少年少女们悔过自新。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/29/21072918121485_u162815.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>  天堂造物的规则是“服从更快、更有胆识的人  </figcaption></figure></div><p><span style="font-size: 30px; font-weight: bold; letter-spacing: 1.3px;">迷你游戏“飙车”     </span><br></p><p>　　八神需要驾驶自己的机车与飙车族竞速，活用“冲刺”、“冲撞”、“防御”以率先抵达终点就是该迷你游戏的核心：冲刺可以提供爆发般的加速，但需要通过在转弯时漂移等方式积攒冲刺量表；冲撞可在高速行驶时横向移动机车撞击敌人；防御可以抵挡敌人攻击，但一直处于防御状态会降低车速。 打倒一定数量的机车骑士后，与头领的死亡竞速便会开始，死亡竞速开始时会在一圈后的位置设置终点，甩开首领，抢先抵达终点便可获胜。在竞速中，机车骑士以及首领会进行烟火、音波、踢踹等攻击，如果多次遭到攻击，伤害会不断累积，导致冲刺效果下降。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/29/210729181237752_u162815.jpg" class="vg_insert_img_onfocus" referrerpolicy="no-referrer"><figcaption contenteditable="true" class>  “暴力摩托”  </figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/29/210729181249834_u162815.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>  一开始有 4 种性能、外观不同的机车，随游戏进行还可以改装或解锁更多类型  </figcaption></figure></div><p><br></p></div>
