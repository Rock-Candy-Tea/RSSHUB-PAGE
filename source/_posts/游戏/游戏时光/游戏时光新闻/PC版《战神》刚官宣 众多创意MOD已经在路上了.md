<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　想必大家已经知道，<a href="https://www.vgtime.com/topic/1138401.jhtml">《战神》即将在明年 1 月 15 号登录 PC 平台</a>。「Only On PlayStation」游戏该不该上 PC 的争论这次当然有，不过这一回有更多网友在讨论，到底能给 PC 版的《战神》制作什么有趣的 MOD。</span><br></p><p>　　谈到这次的 MOD 热潮，必然绕不开《战神》总监 cory barlog，他在 PlayStation 官推下留言：“什么时候能见到托马斯小火车的 MOD？”</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192004112_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　很多游戏在登陆 PC 平台后，总有人会制作托马斯小火车的角色替换 MOD，比如《怪物猎人 世界》还有《生化危机2 重制版》，大家都见怪不怪了，《战神》肯定也逃不过这个命运。但玩家怎么也没有想到，游戏总监居然会亲自下场玩梗。</p><p>　　那还能怎么办，当然是一起加入这个 MOD 狂欢啊。这不，很快就<a href="https://twitter.com/Ambientpotato/status/1450854369918955520">有人</a>设计出了奎爷造型的托马斯火车，你还别说这完成度挺高的。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192043453_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　神通广大的网友当然不会止步于「托马斯」这一个元素，不久就<a href="https://twitter.com/MOHAMMEDFI10/status/1450868796567269379">有人提出了新点子</a>，不如我们把奎爷换成芙蕾雅吧，孩子这么小，不能没有妈啊。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192114906_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　网友 <a href="https://twitter.com/TheeMockingjqy/status/1450858055395012612">TheeMockingjqy</a> 则是想到了一个特别的点子，奎爷不再是个“年迈的老父亲”，一切又回到最初的起点。而我们的新战神则褪去童稚，拥有了属于硬汉的发型。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192150658_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　还有人认为这次北欧之旅，本质上就是「战神与他年迈的老父亲」，那我们为什么控制的是老父亲而不是战神呢？</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192203273_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　那些把父子俩替换成其他作品角色的 MOD，再跟上面的脑洞对比就显得有些普通了，感觉缺少了点惊艳感。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192229267_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192236787_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　网友 <a href="https://twitter.com/GamingFilosofia/status/1450861104113012739">GamingFilosofia</a> 的 MOD 思路也是替换角色，但我还是要把他的想法单独列出来，主要是他贴出的画太好看了。他觉得只需要给那些充满创意的 MOD 社区两天时间……不是一周或两周，只要 48 小时，你就会得到奎爷的美女替换 MOD，像是 2B 啊什么的。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192341156_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　我也给游戏装过美女 MOD，但每次一到剧情部分，醇厚男声和 MOD 的强烈反差总让我无法接受。也许是和我想到了一起，我发现 <a href="https://bbs.nga.cn/read.php?tid=29070437&rand=980">NGA</a> 上有朋友把目标转移到了女武神身上。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192501857_u93176.png" class referrerpolicy="no-referrer"><br></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192517687_u93176.gif" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>那你很勇哦</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　想来也是，当年玩《战神》可没少被女武神痛扁，要是能给他们换套衣服，估计心态也会好点。</span><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192546387_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　除了这些奇奇怪怪的需求，网友 Veget4_Evolve 想要的 MOD 显然更注重实用性：“emm，既然《战神》即将登陆 PC 平台，我希望能有表现得和 2016 年 E3 预告片一样棒的画质增强 MOD，当年的预告片看起来甚至比即将推出的《战神 诸神黄昏》画面还要好。”</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192605387_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　除了文章里提到这些，PC 版《战神》正式推出后肯定还会出现有更多创意 MOD，也许就和网友 <a href="https://twitter.com/TheeMockingjqy/status/1450944852267397121">TheeMockingjqy</a> 认为的那样，PC 版《战神》和 MOD 就是绝配，有谁会拒绝这些创意 MOD 呢？</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/10/21/211021192816716_u93176.png" class referrerpolicy="no-referrer"><br></p></div>
