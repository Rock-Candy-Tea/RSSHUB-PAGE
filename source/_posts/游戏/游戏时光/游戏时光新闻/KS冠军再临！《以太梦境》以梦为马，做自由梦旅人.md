<p>由游卡桌游带来的《以太梦境》中文版已于5月24日正式上线新物集。</p>

<div class="topicContent front_content"><p>　　2018 年，借由《污痕圣杯：阿瓦隆的陨落》在 KS 上取得的巨大成功，出版社 Awaken Realms 一举登上了美式桌游大厂的金字塔尖，他们于 2019 年 7 月开始在 KS 众筹的新游戏《Etherfields》（中译《以太梦境》）也因此收获了全球海量玩家的关注。</p><div src="https://player.bilibili.com/player.html?aid=715707672&bvid=BV1HQ4y1R7S6&cid=341869521&page=1" height="616.5"></iframe><figcaption contenteditable="true" class="vg_insert_video_notice"></figcaption></figure><div class="vg_insert_video_cover bili_cover bili_cover_new" contenteditable="false" style="height:616.5px"><b class="bili_play"></b><span class="vg_insert_video_del" contenteditable="false"></span></div></div><p><a href="https://www.bilibili.com/video/BV1HQ4y1R7S6">B站观看地址</a></p><p>　　当然，盛名从来不意味着优秀，在挑剔的桌游玩家眼中，虚妄的名声比起真实的游戏性来说更像累赘。好在获得了 32582 人支持，共筹得超 397 万英镑的《以太梦境》用它强大的实力向大家证明，它被寄予的厚望并非仅是造梦者的黄粱一梦。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/21052417354160_u54.jpg" class referrerpolicy="no-referrer"> </p><p>　　这款游戏的剧情在风格上和 AR 社之前的作品截然不同，虽然仍是一个剧情驱动的大型美式合作游戏，但绝非是普通的冒险故事。游戏中，玩家将作为入梦者展开寻找记忆碎片的历程——你只是一个飘荡的灵魂，遗忘了过去，身处一个浩瀚而超现实的梦境世界之中。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173556448_u54.png" class referrerpolicy="no-referrer"> </p><p>　　但残存的一丝理智告诉你，你出现在此绝非偶然，于是你开始探索这离奇、杂乱的梦境世界，试图在看似无序的一个个梦中了解自己是谁，以及到底发生了什么。它就像一场充满了谜题的密室逃脱，需要运用逻辑与直觉，在各种图片中寻找线索，在若干碎片化的信息之间找到隐藏的连结。</p><p>　　《以太梦境》异乎寻常的背景设定让它注定很难被轻松理解。登陆KS时的众筹成绩虽然亮眼，但很多玩家其实是冲着它绮丽无比的美工模型以及 AR 社这块金字招牌去的，对于它实际的游玩体验很难给出准确的预测。而当 KS 玩家陆续收到游戏真正开始测评之后，才发现 AR 社不并是在故弄玄虚，它的游戏体验确实惊艳且超乎想象。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173607425_u54.png" class referrerpolicy="no-referrer"> </p><h2>梦幻重现，飘忽不定而又近在咫尺的艺术表现  </h2><p>　　初入《以太梦境》世界，你可能会对它的设定一头雾水，根本不知道发生了什么，以及等待你的将会是什么。突然的醒来、消失的记忆、诡异的梦境世界，这三者的组合让前方的冒险完全是一张白纸。它将悬疑气氛烘托得很到位，而这也注定了它几乎没有什么实质性的内容可以展示——在荒诞的梦里寻找真实本就离谱。让玩家一见倾心的实际上还是它巧夺天工的美术。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173624279_u54.jpg" class referrerpolicy="no-referrer"> </p><p>　　仅游戏的封面美术就可以算得上是近几年最优秀的作品之一。它通过大量使用对比度强烈的无彩色营造出了极具层次感的光影效果。画面乍一看虽相对简单，但却能通过对光的把握，在平面中构建出纵深，抓人眼球。没有任何文字作为注解，而这如梦境般朦胧的画面却好像在诉说着什么，吸引人去探索背后的故事。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173638112_u54.png" class referrerpolicy="no-referrer"> </p><p>　　比封面更吸引人的当然是游戏的模型。AR 社作为模型涂装工作室起家的团队，经过《污痕圣杯:阿瓦隆的陨落》《复仇女神号》等作品的千锤百炼，《以太梦境》作为年度头牌游戏自然也倾注了他们全部的技巧和心血。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/21052417364737_u54.jpg" class referrerpolicy="no-referrer"> </p><p>　　而它特殊的背景设定也让它在模型设计方面几乎没有束缚。每一个梦境都是一个如露水般转瞬即逝的世界，一切绮丽的幻想都可以合理存在，任由创造力和想象力自由播种。从玩家所扮演的角色、遇到的游戏人物、造成阻碍的 BOSS，到游戏的 token 以及所见的场景，《以太梦境》的模型都已为你量身打造、方便使用、易于辨别，最精彩的还是它对各个尺度下细节的把控。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173701463_u54.jpg" class referrerpolicy="no-referrer"> </p><p>　　《以太梦境》以其特有的、与其他桌游完全不同的方式，将你扯入到这个庞大神秘的梦境世界中，天马行空的美术与模型表现又会让你觉得这些飘忽不定的梦仿佛触手可及。但模型只是锦上添花的存在，沉浸、享受梦境，才是它真正的魅力。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173710844_u54.jpg" class referrerpolicy="no-referrer"> </p><h2>拨开云雾，光怪陆离而又曲径通幽的奇妙剧情</h2><p>　　尽管史诗般的剧情并不是《以太梦境》的主要卖点，但游戏剧情仍是它至关重要的一环。实际的游戏过程是在一个梦境中穿梭，就和你每天会做的梦一样，每一个梦都有着无需逻辑且完全不同的背景，而各个梦境中展开的故事也并不会互相桎梏。它更像是一个个单元剧，有限的内容难以做到恢弘壮丽，但精巧的构思却能让人印象深刻。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173716301_u54.png" class referrerpolicy="no-referrer"> </p><p>　　而这些梦境也并非全无关系。游戏的叙事像梦一样迷离破碎，但是其中一些暗含隐喻的文字互相勾连，会向玩家透露一些碎片化的信息。朦胧的信息指向似是而非的答案，玩家可以通过自己的直觉、逻辑、推理能力、以及情商去拼凑出完整的答案，这种用自己的力量拨开云雾的感觉实际上也很吸引人。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173726402_u54.png" class referrerpolicy="no-referrer"> </p><p>　　玩家在游戏中的一大目标就是理解这个梦境世界到底发生了什么，某种意义上这才是游戏的主线，这种看似超脱于游戏之外的探索，实际却仍被笼罩于游戏中的构思十分巧妙。</p><p>　　虽然你做的梦可能是荒诞无逻辑的，但这并不意味着你在梦中的所作所为毫无意义。游戏中，你每一次与梦境中的一切互动都会留下印记并导致惊奇的结果，从而改变这神奇的梦境世界的节奏。记住，正常世界的规则和逻辑在《以太梦境》中并不适用，在这里，哪怕是最狂野的异想天开都可以成为可能。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173733670_u54.jpg" class referrerpolicy="no-referrer"> </p><h2>密室逃脱，严丝合缝而又玄机暗藏的解密体验</h2><p>　　梦是不需要逻辑的，个中深意只有玩了《以太梦境》后才能真正理解。开放性的故事背景解放了设计上的束缚，设计师的无边创造力让每一个梦境都可以有完全不同的背景设定，连画风都会出现变化。</p><p>　　玩家面临的场景千差万别，解决问题的手段也丰富多样，比如你可以在一个黑暗的梦境中掏出一个手电筒。游戏体验变化多端，因此每一场梦境都令人充满期待。</p><p>　　另一方面，《以太梦境》的基础构架也相当稳固，它能够允许不同梦境的体验在同一个框架下展现出完全不同的面貌，还能够允许在探索梦境的过程中不断追加新的要素。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173744330_u54.jpg" class referrerpolicy="no-referrer"> </p><p>　　当然，梦里的一切也并非毫无章法可循，《以太梦境》的梦境其实类似于一般美式游戏中的一个章节，有自己独立的地图和故事，玩家必须要完成这个章节的主要目标才能获胜。但与通常的冒险类美式游戏相比，这款游戏的具体表现形式略有不同。</p><p>　　在梦境中，与敌人战斗可以是选项之一，但绝对不是主要目标，玩家真正要做的是根据当前的需求，解决眼前的难题。与这个梦境中的道具和居民互动，最终完成任务。游戏过程有点像是密室逃脱游戏，探索、调查以及随之而来的脑力风暴，它们的占比要比普通的战斗大很多。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173754351_u54.png" class referrerpolicy="no-referrer"> </p><p>　　解谜部分的游戏体验常被拿来与《诡镇奇谈卡牌版》做对比，因为它们都是故事驱动的卡牌游戏，并且也有着相近的游戏体量。但实际来说，玩家在《以太梦境》中的行动要灵活而自由得多。而且《以太梦境》的基础版体量已接近于《诡镇奇谭卡牌版》三个季度的内容，其性价比绝对超值。</p><h2>定制角色，自由发散而又切合个性的脑力风暴</h2><p>　　作为一款美式游戏，怎么能少了专属于玩家的角色呢？在《以太梦境》中，每个玩家一开始都要选择一名冒险的梦游者，比如赌徒、专家、大力士、自由人等。</p><p>　　这看起来与一般游戏无异，但是别忘了在这款游戏的剧情设定中，这些角色几乎是忘记一切的。当你挑选好自己的角色后，要和他一起去挖掘其背后被遗忘的故事，因此代入感可比一开始就有完整设定的角色要高出不少。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173806317_u54.jpg" class referrerpolicy="no-referrer"> </p><p>　　具体用于体现玩家角色个性的内容是他们的牌库。比如赌徒这个人物的牌库中有很多的运气要素，他有各种投骰子和重掷骰子的方法；大力士则有着强壮的肌肉，能够使用蛮力对抗各种敌人。</p><p>　　角色的成长方式是经典的 DBG（牌库构筑），随着玩家对梦境探索的深入，使用的卡牌也会逐步迭代。卡牌可以提供用于执行基础行动的资源，比如用于攻击的红色资源，用于移动的黄色资源以及用于交涉的绿色资源，它们也可以作为行动卡打出以待未来启动。</p><p>　　DBG 的经典构筑思路是精简牌库以增加强力牌的打出频率，但在这款游戏里却并不能通用。因为每当玩家耗尽牌库时都会受到一次惩罚（其中之一是受到一点伤害，一次梦境会在受到7点伤害后失败），因此玩家在构筑牌库时需要更加注意构成的平衡，策略性很是可观。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173816968_u54.jpg" class referrerpolicy="no-referrer"> </p><p>　　在一个 DBG 游戏中使用三种不同的资源其实是比较激进的，这会大大增加抽卡的运气成分，增加玩家能够执行的行动中的不确定性。但好在设计师对此也做出了平衡——玩家在资源不足却仍想做出某种行动时，可以选择进行一次冒险，投掷骰子并有一定概率获得相应数量的资源。通过运气的方式解决了由运气带来的资源不平衡，虽然依然靠运气却将主动权交到了玩家手中，称得上是巧妙的设计。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173826641_u54.jpg" class referrerpolicy="no-referrer"> </p><p>　　不过需要注意的是，会在游戏里成长的并不仅仅只有玩家。在玩家构筑自己卡组的同时，敌人和环境卡组也在同步改变，每一个你击败过的对手随时可能变本加厉地卷土重来。游戏中有一个会被反复使用的沉睡卡组，其中的怪物玩家需要与它反复战斗。而在玩家做出一些决定后，会有新的怪物被添加到这个卡组中，然后一次又一次地出现在玩家面前。</p><p>　　过去从未真正过去，这如同一个吞噬了所有其他梦的梦，一切都是吊诡又合理的。毕竟这是梦境所赋予你的自由，下至从现实逃离到你的“梦境”中，上至操控时间与重置现实都无需太讶异，由此也为玩家带来极佳的浸入感。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524173835869_u54.jpg" class referrerpolicy="no-referrer"> </p><h2>小结</h2><p>　　《以太梦境》是一首任由玩家释放天赋，挥洒灵感的诗歌，一个个构筑精妙的梦境散发出的光华比史诗级的冒险更引人瞩目。但此前由于其如梦似幻的单词和巨大的英文文本阅读量，国内玩家很难自由探寻迷梦中的真相。</p><p>　　但如今这已不再是难题，由游卡桌游带来的《以太梦境》中文版已于 5 月 24 日正式上线新物集，以太之境，以梦为马，你愿入梦做时空中的自由梦旅人吗？</p><p>　　扫描小程序二维码获取更多信息：</p><p style="text-align: center;"><img style="width: 350px;" src="https://img01.vgtime.com/game/cover/2021/05/24/210524174833354_u54.jpeg"
