<div class="topicContent front_content"><p>      游戏行业业内人士 <a href="https://twitter.com/serkantoto/status/1516140701028823040?s=21&t=asSH1UVASUX6Kuk21NO84g">Dr. Serkan Toto</a>（Kantan Games 首席执行官兼分析师）发推透露，他听说了《艾尔登法环》开发商 From Software 可能会成为索尼收购对象的候选者的传闻。</p><p>　　实际上，该传闻最早在 4 月初就传播过，甚至社交媒体上还有人宣称收购已经完成，就等官宣了。不过当时正值愚人节，大家普遍认为是开玩笑。现在 Toto 作为业内人士煞有介事地拿出来说，也许可以认为他也觉得该传闻有一定真实的可能性。<br></p><p style="text-align: center;"><img style="width: 480px;" src="https://img01.vgtime.com/game/cover/2022/04/19/220419104102842_u244746.jpg" class referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　Toto 在后续推文中简单介绍了 From Software 的历史沿革。游戏并非 </span><span style="background-color: initial; letter-spacing: 1.3px;">From Software 最初的业务，该公司最初主要开发商用短剑，比如猪养殖饲料投放相关的管理软件。后来由于 90 年代初日本泡沫经济崩坏，公司才转型开发游戏。</span><br></p><p><span style="letter-spacing: 1.3px;">　　Toto 对 From Software 于 2014 年被角川集团以相当于今天 2250 万美元的价格收购 80% 股份感到十分痛心，他认为 </span><span style="letter-spacing: 1.3px;">From Software 的价值远高于此。而经过后续一系列操作，目前 </span><span style="letter-spacing: 1.3px;">From Software 早已是角川集团旗下子公司了。</span></p><p><span style="background-color: initial; letter-spacing: 1.3px;">      这不是近期唯一的索尼收购传闻。4 月 15 日小岛秀夫在</span><a href="https://twitter.com/Kojima_Hideo/status/1514822206353973249" style="background-color: initial; letter-spacing: 1.3px;">推特</a><span style="background-color: initial; letter-spacing: 1.3px;">上分享了一张 PS 工作室宣传图，引起了玩家的众多猜测，认为在暗示小岛工作室会被索尼收购，但小岛秀夫随后发文澄清到“这样做看起来虽然很有误导性，但小岛工作室一直是并将继续是一家独立游戏开发工作室”。</span><br></p><p>　　知名记者 Jeff Grubb 近日在一次<a href="https://twitter.com/TWTHEREDDRAGON/status/1514727587209220098?ref_src=twsrc%5Etfw%7Ctwcamp%5Etweetembed%7Ctwterm%5E1514727587209220098%7Ctwgr%5Ehb_1_10%7Ctwcon%5Es1_&ref_url=https%3A%2F%2Fgamingintel.com%2Fplaystation-acquisition-bigger-kojima-productions%2F">播客直播节目</a>中也提到，他听说到的索尼收购对象，会是规模比小岛工作室更大的厂商。<br></p><p style="text-align: center;"><img style="width: 480px;" src="https://img01.vgtime.com/game/cover/2022/04/19/22041910414918_u244746.jpg" class referrerpolicy="no-referrer"><br></p><p>      据悉目前 From Software 对收购持开放态度。继今年年初<a href="https://www.vgtime.com/topic/1146538.jhtml">索尼以 36 亿美元收购 Bungie</a> 后，最近有关索尼收购的消息频出，或许我们很快就能看到相关进展，无论它是不是 From Software。</p><p style="text-align: right; ">消息来源：<a href="https://www.dualshockers.com/playstation-to-acquire-from-software-claims-popular-gaming-insider/">DualShockers</a></p></div>
