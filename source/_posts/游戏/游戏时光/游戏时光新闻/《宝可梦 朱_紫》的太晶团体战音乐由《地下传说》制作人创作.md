<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">《地下传说》制作人 Toby Fox 今日</span><a href="https://twitter.com/tobyfox/status/1557503052487479297?s=21&t=NaOl0bdwZ2-uFrzIR1eijg" style="background-color: initial; letter-spacing: 1.3px;">发推</a><span style="background-color: initial; letter-spacing: 1.3px;">表示，《宝可梦 朱/紫》中「太晶团体战」的背景音乐是由他创作的。而且在先前的宝可梦发布会中，Fox 所创作的部分音乐就已经亮相。</span></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/11/220811130106266_u93176.png" class referrerpolicy="no-referrer"><br></span></p><p>Fox 还分享了这首战斗音乐背后的创作故事。当游戏的总监给 Fox 布置任务时，先展示了一张路卡利欧与其头顶上的一朵巨型花朵插图，并告诉这位作曲家这张图就是本作中「新的游戏机制」（其实就是后来公布的太晶化）。<span style="letter-spacing: 1.3px;">一切都从这幅插图开始，Fox 需要用自己对它的理解来创作对应的曲目。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/11/220811125841763_u93176.png" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>Fox分享的插画</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/11/220811130010652_u93176.jpeg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>太晶团体战</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">其实 Toby Fox 已经与 Game Freak 在多个游戏项目上有过合作。Fox 曾经为 Game Freak 的 RPG 游戏《小镇英雄》创作了音乐，还为《宝可梦 剑/盾》中的对战塔谱写了主题曲。</span><span style="background-color: initial; letter-spacing: 1.3px;">早在今年六月，Toby Fox 就在推特上</span><a href="https://www.vgtime.com/topic/1170369.jhtml" style="background-color: initial; letter-spacing: 1.3px;">确认</a><span style="background-color: initial; letter-spacing: 1.3px;">，自己将为这部《宝可梦》新作创作音乐。</span><br></p><p style="text-align: center;"><span style="letter-spacing: 1.3px;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/08/11/220811125811704_u93176.jpeg" class referrerpolicy="no-referrer"><br></span></p><p style="text-align: right;">来源：<a href="https://www.nintendolife.com/news/2022/08/toby-fox-composed-pokemon-scarlet-and-violets-terastal-raid-battle-music">Nintendo Life</a></p></div>
