<div class="topicContent front_content"><p>　　提起日本，很多人可能会首先想到其发达的成人内容产业，还有影视剧里经常出现的情侣酒店。<span style="background-color: initial; letter-spacing: 1.3px;">对很多国人来说，日本的情侣酒店可能和国内的酒店小卡片是一个性质的东西，让人想到一些羞羞的事情……但我们对日本的情侣酒店可能还多了一份好奇。</span></p><p>　　很难找出准确的话来描述这种感觉，一方面我们的理性告诉我们这并非适合与人大大咧咧讨论的话题，但另一方面我们又对这个用华丽外表包装的神秘场所产生了向往。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/02/210402112803384_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　可能会有人说这当然是日本，却并非日本的全部。</p><p>　　所以我们今天要讲述的，只是关于日本情侣酒店的一些片面故事和细节。有些可能是你早已在营销号中看烂的内容，有些是出乎意料的，日本当下情侣酒店的现实。</p><h4>情侣酒店打街机，这是什么Play？</h4><p>　　网友 <a href="https://twitter.com/kinoko_mobdoon/media">@kinoko_mobdoon</a> 是网络上的一个小透明，他乐于在社交网络上分享自己的生活，推特的内容也非常贴近普通人的日常生活。</p><p>　　有时是一张上学路上随手拍的电车照，有时是手游的截图，有时是路边美食的照片，甚至连去 711 便利店买零食都会想办法发到推特上。</p><p>　　但他推特里内容里除了这些日常琐事，占比最多的毫无疑问是《太鼓达人》。哪怕他是个重度的推特使用者，与《太鼓达人》有关的推文数量也是多得离谱。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/02/210402112848410_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　他既不是什么流量明星，也不是什么所谓的 KOL，只是一个正常使用推特的普通人。哪怕在发布了那条爆火的推文后，关注他的人也不过 855 人。</p><p>　　@kinoko_mobdoon 在 3 月 30 日发布了一条推文：</p><blockquote><p>这是宫城的一家情侣旅馆<br>我非常想住在这里，哈哈哈</p></blockquote><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/02/210402112902495_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　配图是一张旅馆的照片，画面右边的《太鼓达人》街机格外引人注目。没想到就是这么一条随手吐槽的推文，为他赢来了近两万人转发和十万个点赞。</p><p>　　大家显然对这么一间能够随意游玩《太鼓达人》的情侣旅馆套房感到好奇，在推文下问了很多问题。有问《太鼓达人》框体是什么版本的，有问入住价格的，甚至有宫城县当地的居民来问酒店具体的名字，想要亲自前去住上一晚的。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/02/210402112920587_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　这么高的热度也引来了这家情侣酒店官方的注意，可惜他们带来的并不是好消息：</p><blockquote><p>感谢您光临我们的酒店，<br>图里的是我们公司在仙台市的酒店。<br>目前《太鼓达人》的街机现在仍处于故障中，我们正在抓紧维修，非常抱歉。</p></blockquote><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/02/210402112932575_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　@kinoko_mobdoon 似乎多少还是有些羞于讨论情侣酒店，并没有给酒店官方任何回复，只是回答了几个《太鼓达人》玩家的问题，然后感叹自己随手发的一张图居然能引来这么多讨论，甚至拿到了 10 万个赞。</p><h4>落后于时代的游戏设备</h4><p><span style="background-color: initial; letter-spacing: 1.3px;">　　这家情侣酒店的 305 房就是这次大热的自带《太鼓达人》街机的房间，可惜目前街机正在维修中，想要去实地看看的朋友需要再等些时日。</span><br></p><p style="text-align: center;"><img src="https://img01.vgtime.com/game/cover/2021/04/02/210402162915716_u93176.jpg" class style="width: 600px;" referrerpolicy="no-referrer"><br></p><p>　　酒店的 605 房也是个大家伙，直接在房间里放了台跳舞机。</p><p style="text-align: center;"><img src="https://img01.vgtime.com/game/cover/2021/04/02/210402162929272_u93176.jpg" class style="width: 600px;" referrerpolicy="no-referrer"><br></p><p>　　剩下的零星也有像是 201 房和 301 房备有 Switch 啊，502 房和 602 房放置的 VR 设备啊什么的，倒是非常适合玩家过来住一住。</p><p style="text-align: center;"><img src="https://img01.vgtime.com/game/cover/2021/04/02/210402162940680_u93176.jpg" class style="width: 600px;" referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img src="https://img01.vgtime.com/game/cover/2021/04/02/210402162948426_u93176.jpg" class style="width: 600px;" referrerpolicy="no-referrer"></p><p>　　尽管情侣酒店里有游戏机并非独例，在日本的情侣酒店专门检索网站 <a href="https://happyhotel.jp/index.jsp">HAPPY HOTEL</a> 里，专门有“电视游戏”的选项。如果只勾选这个选项的话，系统会提示你符合条件的酒店已经超过 200 家，为了让用户更好地挑选酒店，需要进一步缩小范围。但像上面这样游戏主题如此突出的房间确实很少见。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/06/210406150003925_u93176.png" class referrerpolicy="no-referrer"></p><p>　　单看数量的话似乎游戏设备早已扎根于情侣酒店中，成为了情侣酒店里非常常见的东西。但如果再细细查看每一家酒店的介绍就会发现这只是繁荣的表象，实际上游戏与情侣酒店的磨合比我们想象中要差劲许多。</p><p>　　还是拿上面那家酒店的太鼓达人街机来举例，那台街机已经是很多年前的版本了，与其说是专门购入的设备，更像是从倒闭街机厅里低价收来的便宜货。</p><p>　　在另一家房间里设置有街机的<a href="https://kobe-eldia-luxury.com/room/royal.html">情侣酒店</a>，情况也大致相同，内设的街机并非最新款。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/06/210406152114496_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/06/210406152121594_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　除了街机，主机也是同样的情况。在玩家们为抢购新主机而苦恼的 2021 年，现在仍有酒店在为顾客提供 PS3 主机，<a href="https://www.vgtime.com/topic/1118849.jhtml">要知道今年 PS3 的商店就要关闭了</a>。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/06/210406153118464_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　甚至这种房间已经备好游戏设备的还是少数，更多情侣酒店的情况是：你如果想使用酒店内的游戏设备，那需要和前台提前预约。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/06/210406153604347_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　可能很多玩家看到这里多少会有些失望，这与大家认知中的主流游戏文化似乎偏离得太远了。<span style="background-color: initial; letter-spacing: 1.3px;">不过要知道，几年前的情侣酒店并不会专门把游戏当做自己的卖点，与其说这是可有可无的噱头，倒不如说是“情侣酒店 x 游戏”这个新领域的萌芽。</span></p><h4>美容按摩团体聚餐，情侣酒店正在求变</h4><p>　　不只是我们，和上文的 @kinoko_mobdoon 一样，很多日本人也对情侣酒店抱着固有的印象，甚至会避免在日常生活中直接谈论情侣酒店。</p><p>　　但情侣酒店并非只有能旋转的圆床和大得离谱的浴室，它早已不是人们心中那个只有羞羞事情的场所。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/02/210402113001790_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　在日本轻小说《里世界郊游》的第 5 卷中，有一个篇章专门讲述主角一行 5 人在新宿歌舞町开情侣酒店女子会的经历。</p><p>　　当时我还觉得作者也太扯了吧，哪有开女子会去情侣酒店开的，真当大家不知道“Love Hotel”是什么意思嘛？结果今天一查才知道其实是我孤陋寡闻了。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/02/210402162758681_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　情侣酒店仍然是与之相关服务业的重要组成部分，但它同时也在尝试接纳更多的客源，提供更多的服务。</p><p>　　很多的情侣酒店已经调整了经营政策，以往的客源大都是想要进一步加深感情的情侣，客层比较单一。现在甚至已经出现了允许团体、家庭甚至是单人入住的情侣酒店，并且除了住宿外有的酒店也开始发展配套的餐饮服务。</p><p>　　也有的酒店提供的是自助餐，具体提供的服务类型会根据不同的酒店而产生细微区别。但总的来说很多情侣酒店已经开始提供特殊服务套餐，例如生日聚会，女士团体美容度假，单身派对套餐，甚至包括温泉和桑拿在内的情侣水疗。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/02/210402162856935_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　与其说是情侣酒店，更像是我们印象中的度假村。</p><p>　　像是上面提到的<a href="https://sendai-luna.com/">带《太鼓达人》街机的情侣酒店</a>，因为地址比较偏僻，提供的服务并没有那么全面。但在看了酒店的所有房间后，也能发现和我们印象中很不同的地方。</p><p>　　情侣酒店有这么大的变化，或许我们应该感谢的，是那个直到 2021 年都没有开成的 2020 年东京奥运会，是它让情侣酒店向更多人敞开了大门。</p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　奥运会能够为举办国带来极其庞大的客源，日本早就预见到他们的住宿资源有限，无法安置随着 2020 年东京奥运会而来的所有旅行者。</span><a href="https://japantoday.com/category/national/govt-to-convert-love-hotels-into-regular-lodging-facilities-ahead-of-2020-olympics" style="background-color: initial; letter-spacing: 1.3px;">于是他们在 2016 年时决定将一部分情侣酒店改建为常规的住宿设施</a><span style="background-color: initial; letter-spacing: 1.3px;">。</span><br></p><p>　　不过这并非针对所有情侣酒店，而是面向那些表现不好的情侣酒店。也就是说如果你的情侣酒店经营不善，就要被拿去改建为正常的酒店，这是很多情侣酒店老板不愿意看到的。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/06/210406161036137_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　情侣酒店其实入住率挺高的，平时能到 40%。而且因为功能相对单一，并不需要像正常酒店那样在人力方面有那么高的支出，两三个前台和几个打扫卫生的保洁员就足够了。</p><p>　　如果改为普通酒店就意味着更大的经营成本，酒店拥有者可能赚得更少，甚至还存在转型失败的风险。生存还是死亡，这是处在时代变革中的情侣酒店需要思考的。</p><p>　　这就让这些年情侣酒店开始变着花样吸引更多顾客，希望通过多元的设施来照顾到更多人群，游戏和料理就是近几年新加入的服务。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/06/210406170127176_u93176.jfif" class referrerpolicy="no-referrer"><br></p><p>　　所以今后有人跟你说去情侣酒店通宵打《太鼓达人》的时候不要太惊讶，因为在情侣酒店里你真的可以打个痛快……如果还剩时间的话。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/06/210406173236499_u93176.jpg" referrerpolicy="no-referrer"><br></p></div>
