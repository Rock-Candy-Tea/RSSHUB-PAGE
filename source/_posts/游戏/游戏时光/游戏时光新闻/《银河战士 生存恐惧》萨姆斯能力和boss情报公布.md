<div class="topicContent front_content"><p>　　任天堂旗下经典 2D 平台动作系列“银河战士”新作《银河战士 生存恐惧》即将于 2021 年 10 月 8 日发售，官方昨日公布了本作<a href="https://www.vgtime.com/topic/1133487.jhtml">第二段预告视频</a>，且本作官网更新了更多关于萨姆斯的能力与登场 boss 的情报。

<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828094339526_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>

剧情

</h4><p>　　据 8 月 20 日的情报显示，萨姆斯为了探寻寄生生命体 X 的存在前往了 ZDR 星球后，但很快就遭遇变故并丢失了强力装甲和大部分能力，官方在新的预告中给出的答案是萨姆斯在刚抵达 ZDR 时遇到了一名幸存的鸟人族战士，但它看起来却并非善类，做出攻击行动的萨姐反被完全击溃，这才有了前一段预告中萨姆斯倒地并且身上装甲发生改变的一幕。  <br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/27/210827231642501_u162815.jpg" class referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/27/210827231648678_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/27/210827231654679_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/27/210827231659739_u162815.jpg" referrerpolicy="no-referrer"><br></p><p style="text-align: left;">　　本次的预告仍留下了很多问题，原本性情温和鸟人族还有其它的幸存者吗？预告中拿着长矛的“鸟人族战士”是机械吗？鸟人族战士在打败萨姆斯之后为什么留了萨姆斯一命？这些问题就只能亲自去游戏中体验了。  <br></p><h4 style="text-align: left;">能力  </h4><h3>恒离子能力“Flash Shift”  </h3><p>　　萨姆斯最新入手的恒离子能力，允许玩家短距离闪避或冲刺，无论是空中还是地上都能使用，特别是萨姆斯能够连续闪避三次，使得赶路或者战斗更加便捷且弦酷。  <br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/21082809492236_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828094928415_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>恒离子能力“扫描”</h3><p>　　此前作品中登场过的能力，允许萨姆斯消耗一部分恒离子能量释放音波，将一定范围内的可破坏砖块显示出来。

<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828094939526_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828094943588_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>电光索</h3><p>　　上次预告中就已出现电光索，本次情报更是体现了萨姆斯的“蜘蛛吸附”和“电光索”两个能力的搭配，玩家可直接对墙上的磁力表面进行抓取，让飞檐走壁变得更加轻松。

</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828095018184_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>冰冻导弹</h3><p>　　此前作品中登场过的冰冻导弹，使得玩家无须用冰冻光束冷冻敌人后再用导弹轰击，这也验证了本作的导弹将有多个可选升级的导弹配件。

<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828095047192_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>追踪导弹</h3><p>　　曾在《银河战士 Prime》系列中登场过的能力，作为导弹的升级能力，玩家可以锁定 5 个对象发射追踪导弹，但需要注意的是每一个对象要消费萨姆斯三发弹药，这对玩家的弹药库存是一个很大挑战。

<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828095054120_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828095100408_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>十字炸弹</h3><p>　　对 4 个方向进行轰炸的炸弹升级能力，用来前往难以到达的房间。

<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828095124893_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>加速冲刺</h3><p>　　系列粉丝非常熟悉的能力，可以对沿路敌人造成伤害并摧毁障碍物。官网还称本作的加速将更加流畅，萨姆斯在冲刺途中可以攻击、滑铲、跳跃，甚至蹬墙跳改变方向。

<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828095153934_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828095157181_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>加速突进</h3><p>　　又被称为“凤凰之舞”的里技巧，萨姆斯在加速冲刺状态下下蹲即可蓄力，跳起后能朝一个方向笔直突进，这个能力在大部分作品中完全忽略也可通关，但熟练运用就能获得许多隐藏奖励。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/21082809520335_u162815.jpg" referrerpolicy="no-referrer"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828095209112_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h3>团身旋击</h3><p>　　萨姆斯最标志性的能力，也是游戏 LOGO 的图标来源，一般获得该能力时也象征着萨姆斯几乎得到了最强能力，游戏也进入终盘，玩家只需在空中跳跃就能用装甲周身的能量消灭挡路的敌人。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/08/28/210828095303652_u162815.jpg" class referrerpolicy="no-referrer"><br></p><h4>BOSS</h4><p>　　从视频中还能看到萨姆斯的老对手“克雷德”将登场本作，依旧身形巨大的它本作被拘束住了，但对萨姆斯来说仍少不了一场恶战。此外视频里还透露了鸟人战士、水下的巨大乌贼形、军用机器人之类丰富多样的 boss 挡在萨姆斯面前。</p><p style="text-align:
