<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　任天堂在今早的直面会上整了个大新闻，不仅为我们带来了《塞尔达传说 旷野之息2》的最新预告，还确定游戏将在 2022 年发售。</span><br></p><p>　　我们没办法直接跳到 2022 年，但《野炊2》的新预告片透露的内容可不少，就这么 1 分 38 秒的视频，有网友从中看到了亿点点小细节。</p><hr class="vg_hr" contenteditable="false"><p>　　首先从视频可以看出，林克的右手被红黑色物质污染，右手可能发生了一些变化。网友 <a href="https://twitter.com/AbelMunizJr/status/1404842192280031235">AbelMunizjr</a> 觉得这一幕有点既视感，难道《野炊2》有受到宫崎骏名作《幽灵公主》的影响嘛？</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153600935_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153607152_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153612149_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　林克在《野炊2》中展现了一系列新能力与道具，并且没有使用席卡石板，有人猜测林克可能在这一作中遗失了席卡石板。但网友 <a href="https://twitter.com/TAHK0/status/1404873160223182849">TAHK0</a> 的脑洞显然要更大一些，他认为这是任天堂重新审视《塞尔达传说 旷野之息》已有设定的结果。</p><p>　　视频里几乎没有给林克正脸，或许黑红色物质对林克的影响比我们想象中的要大，不仅影响到了手臂，甚至影响到了林克的脸。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153641802_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　如果是这样，那网友 <a href="https://twitter.com/LiKovacs/status/1404868204594229253">LiKovacs</a> 可能就白画了个这么俊俏的林克。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/21061615365462_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　任天堂上次为我们带来的《野炊2》的预告，那听起来极其怪异且不协调的音乐给我们留下了深刻的印象。当时就有人提出将视频倒放，音乐会听起来正常一些。有人将这次的预告视频倒放之后，有一个重大发现。</p><p>　　推友 <a href="https://twitter.com/TAHK0/status/1404947544665296896">TAHK0</a> 表示在它深入研究后，发现将新预告倒放，我们听到的是《塞尔达传说 旷野之息》里的音乐《100年前の出来事》。这难道是预示着在《野炊2》里我们能回到过去或是体验 100 年前林克的种种经历？</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153707671_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　网友 <a href="https://twitter.com/BanditNintendo/status/1404903789967155210">BanditNintendo</a> 认真分析了视频里林克在高处望远的片段，再仔细对比《塞尔达传说 旷野之息》的地形，推测出了视频中的大概位置。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153730413_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153738229_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　再仔细对比一下视频与游戏就发现，《塞尔达传说 旷野之息》的一些石柱并没有在视频中看到。也许我们这时候看到的，就是古代的海拉鲁大陆。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153819437_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153825245_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　<a href="https://www.reddit.com/r/zelda/comments/o0t5uc/botw2_comparison_shots_between_the_e3_2021/">Reddit</a> 网友将《野炊2》预告里出现的场景与《塞尔达传说 旷野之息》游戏中的场景进行了对比，两作的景观不仅有比较明显的变化，一些细节也有些许不同。从这三张对比图中，我们可以一窥续作里发生了什么变化。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616162632943_u93176.png" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616162639524_u93176.png" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616162647298_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　Reddit 网友 karonoz 表示他并不希望《野炊2》和《塞尔达传说 旷野之息》的地图那么相似，如果相似度太高，他可能不是很乐意在续作里去探索海拉鲁大陆，因为他早就在海拉鲁大陆上花了几百小时进行探索。</p><p>　　当然，不是所有人都对视频有那么细致的观察，网友 <a href="https://twitter.com/ActualAero/status/1404876134173581315">ActualAero</a> 只看到我们的林克换发型了，而且发型神似漫威宇宙里的索尔。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153905190_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　新衣服的出现让不少画师创作欲望喷涌而出，网友 <a href="https://twitter.com/olyvee_/status/1404886461292351495">olyvee_</a> 画了一张林克的新形象，这幅画和任天堂的视频很搭，强调新服装，并且没有给林克正脸，进行一定的留白处理。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153920357_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　如果网友们关于《野炊2》额猜测是真的，那网友 <a href="https://twitter.com/GifZelda/status/1404889952446468097">GifZelda</a> 就找到了很有意思的东西。他发现了《塞尔达传说 旷野之息》里有一处对话：我有时能在梦里见到它。你认为以前的人是住在天上的嘛？我觉得这是真的。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153934232_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　如果《野炊2》真的发生在古代的海拉鲁，那我们也许真的能在续作见到在天上飞的人。</p><p>　　虽然我们可能已经见到了，那个人就是林克……</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153944664_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p>　　河南的朋友可能看到林克在天上飞的镜头就坐不住了，这是任天堂去河南取景了嘛？<br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616161408956_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616161239724_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　网友 <a href="https://twitter.com/cannolicat31/status/1404852369242873862">cannolicat31</a> 表示《野炊2》与《塞尔达传说 天空之剑》很像，似乎是任天堂有意为之。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616153958944_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616154006420_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616154011620_u93176.jpg" class referrerpolicy="no-referrer"><br></p><p><span style="background-color: initial; letter-spacing: 1.3px;">　　网友 </span><a href="https://twitter.com/twilisita/status/1404841435501891588" style="background-color: initial; letter-spacing: 1.3px;">twilisita</a><span style="background-color: initial; letter-spacing: 1.3px;"> 表示他看了新预告后已经充分理解了任天堂为什么要先发售《塞尔达传说 天空之剑HD》，是想让我们回想起在空中飞翔的畅快感。</span><br></p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/06/16/210616154031242_u93176.png" class referrerpolicy="no-referrer"><br></p><p>　　林克在预告里出尽了风头，但塞尔达公主就没这么幸运了。网友 <a href="https://twitter.com/Stealth40k/status/1404882304875499524">Stealth40k</a> 表示：《野炊2》的预告片让他有点失望，塞尔达公主似乎又从游戏中消失了。就算玩家不能在续作里控制塞尔达公主，希望她至少能成为林克的旅行伙伴。</p><p style="text-align: center;"><img referrerpolicy="no-referrer"><br></p><p>　　虽然有《野炊2》这样的核弹，但大家也没忘了之前舅舅党的爆料。本次任天堂直面会确实公布了新硬件，但并不是大家期待已久的 NS Pro，而是《塞尔达传说》主题的 Game&Watch。</p><p>　　网友 <a href="https://twitter.com/YOGB_Mate/status/1404839956044992516">YOGB_Mate</a> 开玩笑称：这个《塞尔达传说》主题的 Game&Watch 就是老任钦定的《野炊2》游玩设备。</p><p style="text-align: center;"><img style="width:600px;" 秒都待不下去了。</p></div>
