<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">2017 年，由 Arkane Studios 开发的《掠食》正式发售。但在此之前，Human Head Studios 于 2006 年推出了一款射击游戏，名称也叫《掠食》（PREY）。</span><br></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/09/01/220901114239818_u202875.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>《掠食》（2017）</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/09/01/220901114352861_u202875.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>《掠食》（2006）</figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">据维基百科记载，最初 </span><span style="background-color: initial; letter-spacing: 1.3px;">Human Head Studios 计划开发续集《掠食2》，后来其知识产权由 3D Realms 转移至 Bethesda，《掠食2》陷入难产并在 2014 年被取消开发。而在 2016 年 E3 展上，</span><span style="letter-spacing: 1.3px;">Bethesda</span><span style="background-color: initial; letter-spacing: 1.3px;"> 宣称 </span><span style="background-color: initial; letter-spacing: 1.3px;">Arkane 正在开发系列重启作，也就是 2017 年的《掠食》。</span></p><p>尽管两款游戏的名字都叫《掠食》，但这两部作品之间并不存在实质性联系。对于“重名”一事，Arkane 创始人拉斐尔·科兰托尼奥（Raphaël Colantonio）近期给出了一些解释。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/09/01/220901114532912_u202875.jpg" class referrerpolicy="no-referrer"><br><i>拉斐尔·科兰托尼奥</i></p><p>根据科兰托尼奥的陈述，2017 年的《掠食》之所以叫这个名字，是由发行商 Bethesda 决定的。作为游戏的开发商，Arkane 那时还没有足够的话语权。</p><p>在参加 AIAS 的播客节目时，

科兰托尼奥明确表示自己不喜欢《掠食》这个名字，他也讨厌说谎，但当时的经历让他感觉很糟糕。</p><blockquote><p>我想向他们（2006 年《掠食》的开发人员）道歉很多很多次，我真的没有机会，因为我真的不认识那些人。我们从来没想过‘窃取他们的 IP’，再把这个 IP 变成我们的。这很恶心，这不是我们想做的。</p></blockquote><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/09/01/220901114641383_u202875.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">科兰托尼奥还认为，2017 年的《掠食》没有达成商业目标也与它的名称有关，因为这个名字“适得其反”。熟悉 2006 年《掠食》的玩家再接触 2017 年的《掠食》，很可能感觉自己被游戏名误导了。至于不喜欢 2006 年《掠食》的玩家，他们自然不会对 2017 年的《掠食》感兴趣。</span><br></p><p>

科兰托尼奥说道，虽然他自己很感谢 Bethesda 的支持和信任，但对于“《掠食》重名”一事，他一直感到很沮丧。在 2017 年《掠食》发售后不久，科兰托尼奥就离开了 Arkane，并且成立了新团队 WolfEye Studios。2022 年 3 月，WolfEye 推出了自己的新作《诡野西部》。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2022/09/01/220901114915995_u202875.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>  《诡野西部》  </figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">报道的最后，

科兰托尼奥提到他在游戏行业内的新目标是“真正保持独立”，以及创造一些小规模游戏。</span><br></p><p style="text-align: right;">来源：<a href="https://kotaku.com/prey-2017-2006-bethesda-imsim-name-arkane-colantonio-1849481519">Kotaku</a>/<a href="https://www.gamespot.com/articles/prey-director-says-bethesda-forced-arkane-to-use-that-name-calls-decision-a-gross-mistake/1100-6507032/?ftag=CAD-01-10abi2f">GameSpot</a></p></div>
