<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　<a href="https://www.vgtime.com/topic/1120089.jhtml">此前我们曾报道过，随着内部 CMOS 电池的老化与网络关停，PS4 将无法进行离线游戏。</a>此外 PS5 的功能也会因内部电池问题受到影响。不过现在，索尼已经解决了该隐患。</span><br></p><p style="text-align: center;"><span style="background-color: initial; letter-spacing: 1.3px;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/03/211103115948711_u202875.jpg" class referrerpolicy="no-referrer"><br></span></p><p>　　今年 9 月，索尼为 PS4 用户推送了 9.0 版本系统更新。此次更新后，PS4 玩家可以在

CMOS

电池损坏的情况下正常打开游戏。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/03/211103120010530_u202875.jpg" class referrerpolicy="no-referrer"><br></p><p>　　另一方面，国外播主 Hikikomori Media 近日对自己的 PS5 进行了测试。他摘除了 PS5 内部的

CMOS 电池并将主机完全断网，以此模拟最糟糕的情况。</p><p class>　　此次测试的结果是，PS5 和 PS4 的实体版与数字版游戏均可在 PS5 上正常运行，单人游戏功能不受影响，奖杯也可以在离线状态下获得。唯一的例外是 PS+ 订阅游戏，它们会因为网络验证问题而无法打开。</p><p class style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/03/211103120034284_u202875.jpg" class referrerpolicy="no-referrer"><br></p><p class>　　PS 主机的 CMOS 电池主要负责内部时钟的供电。一般情况下，当玩家启动游戏时，系统会比对主机内部与服务器的时间，查验玩家的游戏状态。当

CMOS

电池电力耗尽或损坏，系统就会要求玩家手动输入时间，并尝试在线同步。</p><p class style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/11/03/21110312021458_u202875.jpg" class referrerpolicy="no-referrer"><br></p><p class><span style="background-color: initial; letter-spacing: 1.3px;"> 　根据先前测试，如果在上述前提下，PSN 服务器又发生问题，主机就会提示出错，玩家将无法进入游戏。如今这个问题不复存在，即便相关网络服务关停，PS 玩家们也能照常体验单机内容。</span></p><p class style="text-align: right;">来源：<a href="https://www.youtube.com/watch?v=fsB0NosDlnA&t=534s">Hikikomori Media</a>/<a href="https://www.videogameschronicle.com/news/ps5-reportedly-no-longer-loses-functionality-when-its-cmos-battery-dies/">VGC</a></p></div>
