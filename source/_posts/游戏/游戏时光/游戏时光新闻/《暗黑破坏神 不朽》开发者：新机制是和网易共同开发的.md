<div class="topicContent front_content"><p>　　《暗黑破坏神 不朽》于 4 月 22 日在澳洲开始封闭 Alpha 测试，这次测试至少会持续一个月。该作游戏总监 Wyatt Cheng 和原则游戏设计师 Scott Shicoff 接受了媒体联访，以下为要点整理。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422145033771_u77470.jpg" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><h4>新的阵营玩法</h4><p>　　本作中，玩家需要选择加入三个阵营的一个：不朽者、影盟或冒险者。不朽者的目标是让庇护之地变为更安全的地方；影盟的目标是推翻现任不朽者。而两个目标都不喜欢的玩家，可以加入冒险者。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422145225844_u77470.jpg" class referrerpolicy="no-referrer"><br></p><p>　　对于这个玩法，Scott Shicoff 分享了设计逻辑。三个阵营是为了“不管玩家要加入哪一边，希望都能让玩家觉得可以做出贡献”。不朽者和影盟有各自不同的任务跟游戏内容，比如影盟，有“像是主题任务这样的契约，也有很多任务的大型任务，这些影盟玩家可以接的任务都会是为了要推翻或是想办法打倒现任不朽者来设计。”</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/21042214534055_u77470.png" class referrerpolicy="no-referrer"><br></p><p>　　而被问到这些新机制是怎么设计出来的，Shicoff 回答道：</p><blockquote><p>　　大家知道，《暗黑破坏神 不朽》是我们和网易共同开发的作品，两边都有提供点子，都是双方共同合作制作以及决定。当我们提出斗争循环的概念时，双方都丢出很多意见想法，例如影盟可以偷宝物、不朽者能来阻止等，各自抛出许多想法来讨论。与其说是单一灵感来源，不如说是双方脑力激荡的成果。另外，我们在 MMO 制作上有丰富经验，网易则是有丰富手机游戏制作经验，结合后就是玩家们看到的《暗黑破坏神 不朽》。</p></blockquote><h4>「肝」才是提升强度的主要方式</h4><p>　　对于一款免费游玩的手游，对付费的引导是很多人关心的问题。对于课金换取强度，《暗黑破坏神 不朽》是怎么处理的？Shicoff 对此强调：“在《暗黑破坏神 不朽》里如果想要变强就是要有更高的参与度，也就是参加更多任务和活动。”简单来说，就是“去打怪、获得宝物和收集传奇装备或是道具，透过道具的升等加上符文来强化装备”。</p><p style="text-align: center;"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422145300129_u77470.png" class referrerpolicy="no-referrer"><br></p><p>　　不过，Shicoff 随后也补充说到：</p><blockquote><p>　　如果玩家选择去购买战场通行证，确实会带来有一些好处。但实际上跟玩家在游戏中杀怪打宝的好处相比来说，是比较不明显的。</p></blockquote><p>　　对于担心需要参加特定机制才能变强的隐忧，Shicoff 也进行了澄清。表示：“只有透过这个机制（比如「奇恩的试炼」副本）才能获得更好的装备、如果不能参加我就会比别人弱之类的，这并不是《暗黑破坏神》的精神，也不是我们想做的。”</p><h4>没有支持手柄和移植其他平台的计划</h4><p>　　游戏总监 Wyatt Cheng 表示，对于支持手柄，这从“技术上来说有相当的困难度以及挑战”，所以没有确定能在发售时支持手柄，但如果情况有更新会尽快发布。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422145611945_u77470.png" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　另外，Wyatt Cheng 还表示本作是一款专为移动平台设计的独一无二的游戏，“在手机平台上的操作接口还有游戏性操控、社交方面”都是开发组积极为手机做了加强的部分。比如，因为玩家很难在行动装置同时打字跟玩游戏，所以开发者讨论了内建语音功能。又比如，因为手机的触屏操控方式，所以增加了「集气」机制，有按住不放，瞄准后射出去这样的技能。他认为这在 PC 平台不太对，但对于手机就是很自然的攻击方式。</span></p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422145632314_u77470.png" referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><h4>是一款生生不息的游戏</h4><p>　　对于游戏剧情有没有结尾，设计师 Scott Shicoff 回避了这个问题，只表示：“《暗黑破坏神 不朽》是一款生生不息的游戏。”</p><p>　　当然，对于游戏终局的玩法，开发团队还是分享了不少，比如地狱骨匣跟斗争循环。游戏总监 Wyatt Cheng 表示，《暗黑破坏神 3》玩家封顶之后能做的事情就是刷大小秘境。而在《暗黑破坏神 不朽》，他们希望提供更多元的玩法跟活动。技术测试的反馈也证明玩家确实喜欢这样。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422145654571_u77470.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>流放仪式</figcaption></figure></div><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/210422145705954_u77470.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>副本 Boss</figcaption></figure></div><h4>武僧是最多玩家玩到封顶的职业</h4><p>　　Wyatt Cheng 还分享了一些技术测试的数据。在之前的测试中，最多玩家创建的新角色的职业是秘术师。但是如果以每个职业玩家玩到封顶的比例来看，比例最高的反而是武僧。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/04/22/21042214545023_u77470.png" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class>  秘术师  </figcaption></figure></div><p><span style="background-color: initial; letter-spacing: 1.3px;">　　本次测试还会增加一个职业——圣教军，带有该职业标志性的技能「战马奔腾」和「祝福之盾」，Wyatt 希望玩家会喜爱新推出的圣教军。</span><br></p><div
