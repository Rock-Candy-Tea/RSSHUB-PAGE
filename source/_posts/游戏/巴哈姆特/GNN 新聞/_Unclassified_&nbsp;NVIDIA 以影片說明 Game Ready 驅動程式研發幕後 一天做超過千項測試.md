<div>
<div>
<div>
　　NVIDIA 今日推出新影片，說明旗下 NVIDIA GeForce Game Ready 驅動程式為了協助強化遊戲體驗，研發團隊所投注的心力，同時宣布此次也配合新作《沙丘：香料戰爭 (Dune: Spice Wars)》推出了新款 Game Ready 驅動程式。</div>

<div>
 </div>
<div>
　　NVIDIA 研發團隊在影片中表示，NVIDIA 一天之內要將已推出和即將推出的遊戲，對 Game Ready 驅動程式進行超過 1,000 項測試。光是 2021 年，就進行了超過 180 萬小時的測試，相當於在一年內投入 214 年的時間來提升 Game Ready 驅動程式的品質。</div>
<div>
 </div>
<div>
<div class="video-container">

</div>
<div>
 </div>
<div>
　　NVIDIA 表示，NVIDIA 與開發人員共同合作並進行微調，在數千種桌上型電腦和筆記型電腦硬體設定進行廣泛測試，希望獲得最佳效能和可靠性，藉由 GeForce Game Ready 驅動程式為遊戲玩家帶來最佳的遊戲體驗。該計畫與遊戲開發人員合作，在遊戲正式上市前定期交換遊戲版本和驅動程式。NVIDIA 與遊戲開發商共同尋找對遊戲進行最佳化調整的方式及解決方案，並相對應地反覆調整，確保遊戲和 Game Ready 驅動程式在發布時能提供最佳的品質及效能。</div>

<div>
　　NVIDIA 指出，有鑑於顯示驅動程式是整體遊戲體驗不可或缺的一環，NVIDIA 不會發布測試版的 Game Ready 驅動程式。就像正式上市的完整版本遊戲，會比測試版的遊戲有著更高的標準一樣，NVIDIA 覺得只有提供與正式發布版本一樣的高品質驅動程式才能做到這一點。每個 Game Ready 驅動程式皆取得微軟 WHQL 認證。</div>
<div>
 </div>
<div>
　　至於影片提供針對《沙丘：香料戰爭 (Dune: Spice Wars)》的新款 Game Ready 驅動程式，NVIDIA 也一併推出 Reflex Stats，讓玩家能在射擊遊戲中，即時顯示延遲測量結果。</div>
