<div><br></div><div>桌遊（桌上遊戲）在近十餘年來，是世界風行的活動。不分年齡層都可以玩，是近來時尚的娛樂選擇！現在，我們臺灣原創的桌遊也可以「線上玩」！</div><div><img src="https://truth.bahamut.com.tw/s01/202207/4d5e9891e249486883fbff4a5f8eb1cb.JPG" class="gallery-image" border="0" referrerpolicy="no-referrer"></div><div>Board Game Arena（桌遊競技場）自2010年成立，已是現今全世界最大的線上桌遊網站。在疫情期間，人與人實際見面的困難度上升，因此，線上桌遊網站的用戶激增。根據網站資料統計，目前有超過800萬名會員，他們來自200多個國家。Board Game Arena提供約500款不同的桌上遊戲，每天都有超過萬人在線上捉隊廝殺、比拼鬥智。而網站也提供了42種語言（包含繁體中文），讓你能突破語言的藩籬，輕鬆與外國玩家一同遊戲！</div><div><img src="https://truth.bahamut.com.tw/s01/202207/c9b57b2e886ab79900dca540640e84f2.JPG" class="gallery-image" border="0" referrerpolicy="no-referrer"></div><div>這樣一個世界知名的線上桌遊平台，卻鮮少看見臺灣遊戲的身影，許多資深的桌遊玩家引頸期盼多年，終於看見臺灣國產原創桌遊，閃亮登場！</div><div><br></div><div>臺灣桌遊出版社「玩聚設計」在今年7月，於Board Game Arena正式的推出線上版的《蒐靈祭》，並且全面開放免費遊玩！</div><div><img src="https://truth.bahamut.com.tw/s01/202207/c47df3cce339bbeb6c4b55f2301c9753.JPG" class="gallery-image" border="0" referrerpolicy="no-referrer"></div><div>《蒐靈祭》來頭不小，遊戲設計師曾設計多款好玩遊戲，有口皆碑；美術設計更是當年紅透半邊天的「Deathigner 死神訓練班」！遊戲（實體版）甫出版，便在當年的台灣新銳桌遊博覽會拿下最佳美術第一名、最佳遊戲設計第二名的佳績！</div><div><br></div><div>「玩聚設計」匯集臺灣的設計師、編輯、美術、程式開發員，共同打造了這款線上版的《蒐靈祭》，所有參與製作的成員都是來自臺灣！終於能夠向Board Game Arena上800萬名用戶驕傲的宣布，這就是臺灣原創的遊戲！歡迎你們來體驗！</div><div><img src="https://truth.bahamut.com.tw/s01/202207/5c4e84273dd2546fce74bc76d761c559.JPG" class="gallery-image" border="0" referrerpolicy="no-referrer"></div><div>從7月15日晚上於Board Game Arena上遊戲推出至7月20日 ，才短短幾天就已經累積了1012人次的玩家，平均每天遊玩人次203人。可以說不分日夜，每天、每小時都有人在玩線上版的《蒐靈祭》！</div><div>「衝破疫情脫困境，呼朋引伴線上遊」。桌遊，還是面對面實際與人互動，一起歡笑，一起遊玩最好玩！但在疫情之下，線上桌遊也是個安全、便利，並且明智的好選擇！</div><div><br></div><div>立即上Board Game Arena遊玩吧：<a href="https://ref.gamer.com.tw/redir.php?url=https%3A%2F%2Fboardgamearena.com%2Fgamepanel%3Fgame%3Dsoulaween" target="_blank">https://boardgamearena.com/gamepanel?game=soulaween</a><br><br>惠請媒體先進刊登報導。</div>
