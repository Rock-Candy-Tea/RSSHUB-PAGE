<div>
<div>
<div>
　　《蔚藍 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Celeste" target="_blank">Celeste</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E9%AD%94%E5%A1%94%E5%A4%A7%E4%BA%82%E9%AC%A5+%E5%B4%9B%E8%B5%B7" target="_blank">魔塔大亂鬥 崛起</a>》的獨立研發團隊Extremely OK Games 今日公開了新作《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E5%9C%B0%E7%90%83%E4%B9%8B%E5%88%83" target="_blank">地球之刃</a>（暫譯，Earthblade）》，為像素風格 2D 探索動作遊戲。</div>
<div>
 </div>
<div>
<ul class="bh-grids-img">
<li class="bh-grids-img-box" style="width: 99.87%;">

</ul>
</div>
<div>
　</div>
<div>
　　Extremely OK Games 由加拿大獨立研發者 Maddy Thorson 領軍，並以 Matt Makes Games 名義推出了《蔚藍 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Celeste" target="_blank">Celeste</a>》、《魔塔大亂鬥》遊戲，但後來決定新開 Extremely OK Games 團隊，主要是想要表達 Maddy Thorson 並非遊戲創意的唯一來源，希望讓整個團隊的努力受到大家的認可，加上原本分散各地的成員剛好此時搬到位於溫哥華的新辦公室，因而決定以 Extremely OK Games 名義來開發新作。</div>
<div>
 </div>
<div>
　　Extremely OK Games 公開首款遊戲就是《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E5%9C%B0%E7%90%83%E4%B9%8B%E5%88%83" target="_blank">地球之刃</a>》，研發團隊在官方 Twitter 展示遊戲美術宣傳圖與遊戲音樂。研發團隊表示，他們還不想透露太多關於《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E5%9C%B0%E7%90%83%E4%B9%8B%E5%88%83" target="_blank">地球之刃</a>》遊戲內容與其故事的資訊，他們只是想讓大家知道遊戲的名聲，這樣大家就不用再稱呼它為《EXOK4》了。</div>

<div>
 </div>
<div>
　　玩家可能還不會這麼快看到遊戲上市，因為研發團隊表示，現在大聲說出《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E5%9C%B0%E7%90%83%E4%B9%8B%E5%88%83" target="_blank">地球之刃</a>》在《蔚藍 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Celeste" target="_blank">Celeste</a>》上市後五年內推出（註：2018 年 1 月問世），聽起來很樂觀，但事實上他們不知道到底要花多久時間才能推出。</div>
