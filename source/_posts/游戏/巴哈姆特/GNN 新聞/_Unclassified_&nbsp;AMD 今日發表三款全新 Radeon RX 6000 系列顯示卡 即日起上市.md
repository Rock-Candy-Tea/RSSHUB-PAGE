<div>
<div>
<div>
　　AMD 今日公開 AMD Radeon RX 6000 系列顯示卡的最新成員 AMD Radeon RX 6950 XT、Radeon RX 6750 XT 以及 Radeon RX 6650 XT 顯示卡。</div>
<div>
 </div>
<div>
　　AMD 表示，結合 2.1GHz 遊戲時脈以及 16GB 的高速 GDDR6 記憶體，AMD Radeon RX 6950 XT 顯示卡可在 4K 解析度並開啟最高畫質設定下，為 3A 遊戲大作與電競遊戲帶來卓越效能與視覺效果。AMD Radeon RX 6750 XT 顯示卡在 1440p 解析度並開啟最高畫質設定下可提供高效能遊戲體驗，而 AMD Radeon RX 6650 XT 顯示卡則在開啟最高畫質設定模式下，在眾多最新遊戲中呈現流暢的高更新率 1080p 遊戲體驗。</div>
<div>
 </div>
<div>
<ul class="bh-grids-img">
<li class="bh-grids-img-box" style="width: 99.87%;">

</ul>
</div>
<div>
 </div>
<div>
　　官方表示，全新推出的 AMD Radeon 顯示卡皆採用 AMD RDNA 2 遊戲架構，具備製程最佳化以及增強的韌體與軟體，同時提供高頻寬、低延遲的 AMD Infinity Cache技術與 18Gbps 的高速 GDDR6 記憶體。此外，全新 AMD Radeon 顯示卡支援 Microsoft Windows 11、Microsoft DirectX 12 Ultimate、AMD FidelityFX Super Resolution (FSR) 1.0、即將推出的 AMD FSR 2.0 以及 AMD Radeon Super Resolution 升級技術，同時也支援其他先進功能。</div>
<div>
 </div>
<div>
　　AMD 全球副總裁暨繪圖事業群總經理 Scott Herkelman 表示，全球目前有 30 億遊戲玩家，而且數量仍在持續增加，其中約半數是 PC 使用者。隨著玩家持續增加，這些新推出的 Radeon 顯示卡將提供超越以往的遊戲體驗與效能。</div>
<div>
 </div>
<h3>
高效能、高視覺逼真度以及更好的遊戲體驗</h3>
<div>
 </div>
<div>
　　AMD 指出，全新 AMD Radeon RX 6950 XT、Radeon RX 6750 XT 以及 Radeon RX 6650 XT 顯示卡主要特點與功能包括：</div>
<div>
 </div>
<ul>
<li>
AMD Software: Adrenalin Edition-為基於 AMD 顯示卡與處理器的系統充分釋放遊戲體驗的潛能。最近釋出的新版驅動軟體可為眾多特定 DX11 遊戲提供高達 10% 的效能提升。此外，基於驅動軟體的 AMD Radeon Super Resolution 1.1 現增加對搭載 Radeon 顯示核心的特定 AMD Ryzen 6000 系列處理器的支援。</li>
</ul>
<ul>
<li>
AMD FidelityFX 技術－AMD FidelityFX 技術是一個可為遊戲開發者提供視覺提升效果的開源工具套件，可在 AMD GPUOpen 上獲取。AMD FidelityFX 技術包括 AMD FidelityFX Super Resolution (FSR) 1.0 空間放大技術，目前已支援超過 80 款遊戲，且數量正持續增加。此外，AMD FidelityFX 技術也包括即將推出的 AMD FSR 2.0，此為先進的時間放大（temporal upscaling）解決方案，可為支援的遊戲提升畫面更新率，並為眾多顯示卡提供接近或優於原生的影像品質。</li>
</ul>
<ul>
<li>
AMD Smart Access Memory (SAM) 技術－AMD Radeon RX 6000 系列顯示卡搭配特定 AMD Ryzen 桌上型處理器與 AMD 500 系列主機板時，透過讓 AMD Ryzen 處理器存取系統中整個高速 GDDR6 繪圖記憶體，從而釋放更強大的遊戲效能。在搭載 AMD Radeon RX 6950 XT 顯示卡與 AMD Ryzen 7 5800X3D 處理器的系統上，SAM 能提供高達 14% 的效能提升。</li>
</ul>
<ul>
<li>
AMD Infinity Cache 技術－在 GPU 晶片（die）上整合最後一層資料快取，旨在降低延遲與功耗，實現比傳統架構更強的遊戲效能。</li>
</ul>

<h3>
AMD FidelityFX Super Resolution 2.0 遊戲支援</h3>
<div>
 </div>
<div>
　　AMD 表示，AMD FidelityFX Super Resolution 2.0 是 AMD 廣被採用的開源、跨平台升級技術，FSR 2.0 能根據先前的畫面資料在任何解析度下呈現接近或優於原生的影像品質，並提升支援遊戲的畫面更新率。FSR 2.0 無需專門的機器學習硬體即可廣泛支援顯示卡產品和平台，包括 AMD 與其他特定競爭對手的解決方案。首批支援 AMD FSR 2.0 的遊戲包括由 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Arka" target="_blank">Arka</a>ne Studios 與 Bethesda 聯合製作發行的《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%AD%BB%E4%BA%A1%E5%BE%AA%E7%92%B0" target="_blank">死亡循環</a>》，預計將於 2022 年 5 月 12 日透過更新提供支援。未來幾個月也將有更多遊戲加入支援 FSR 2.0 陣容，其中包括《失落迷城》、《Delysium》、《星戰前夜》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%A8%A1%E6%93%AC%E8%BE%B2%E5%A0%B4" target="_blank">模擬農場</a> 22》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E9%AD%94%E5%92%92%E4%B9%8B%E5%9C%B0" target="_blank">魔咒之地</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E5%BE%AE%E5%B0%8F%E4%B8%96%E7%95%8C%E7%94%9F%E5%AD%98%E6%88%B0" target="_blank">微小世界生存戰</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E5%BE%AE%E8%BB%9F%E6%A8%A1%E6%93%AC%E9%A3%9B%E8%A1%8C" target="_blank">微軟模擬飛行</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E9%80%86%E6%B0%B4%E5%AF%92" target="_blank">逆水寒</a>》、《Perfect World Remake》、《Swordsman Remake》以及《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%9C%AA%E7%9F%A5+9+%E8%99%9F%EF%BC%9A%E8%A6%BA%E9%86%92" target="_blank">未知 9 號：覺醒</a>》。</div>
<div>
 </div>
<h3>
AMD Radeon Raise the Game 遊戲大禮包</h3>
<div>
 </div>
<div>
　　AMD 宣布，即將推出新版 AMD Radeon Raise the Game 遊戲大禮包，玩家購買指定 AMD Radeon RX 6000 系列產品，即可免費兌換遊戲。即日起，玩家透過參與活動的電子零售通路購買指定 Radeon 產品，即可獲贈一組代碼，於本季稍後兌換促銷遊戲，活動詳情將於屆時發布。新版遊戲大禮包將包括《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E9%BB%91%E8%A1%97%E8%81%96%E5%BE%92" target="_blank">黑街聖徒</a>》與《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E7%8B%99%E6%93%8A%E7%B2%BE%E8%8B%B1+5" target="_blank">狙擊精英 5</a>》，另有多款遊戲將於促銷活動登場時宣布（獲贈的遊戲大禮包依購買產品不同，遊戲會有所差異）。</div>
<div>
 </div>
<h3>

<div>
 </div>
<div>
