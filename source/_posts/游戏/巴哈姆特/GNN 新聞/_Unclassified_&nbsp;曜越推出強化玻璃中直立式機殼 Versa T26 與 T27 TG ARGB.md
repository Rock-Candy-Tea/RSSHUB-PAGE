<div>
　　曜越今日推出 Versa T26 與 T27 TG ARGB 強化玻璃中直立式機殼，最新的 Versa 機殼可支援至 E-ATX 主機板，前方內建三顆 120mm 5V ARGB 主板連動風扇能支援華碩 AURA SYNC、技嘉 RGB Fusion、微星 Mystic Light Sync 及華擎 Poly<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=chrome" target="_blank">chrome</a> 軟體。使用者可透過機殼的 RGB 燈光控制鍵切換 27 種燈效模式。</div>
<div>
 </div>
<div>
　　官方指出，Versa T26 與 T27 TG ARGB 整體機身設計相同，兩款機殼左側採用 4mm 厚度強化玻璃，但在前面板則分別有不同的造型設計。Versa T26 前方四塊方形設計的面板由網孔和強化玻璃組成，不僅提升前方進氣幫助散熱，同時達到絕佳的透視效果。Versa T27 前方大面積的 T 字網孔面板不僅大幅度提升進氣效果，也展現代表曜越的品牌標誌。</div>

<div>
　　Versa T26 與 T27 TG ARGB 強化玻璃中直立式機殼擁有擴充性，可支援 3.5 吋擴充裝置或 2.5 吋的 SSD，同時可容納長 175mm 塔式 CPU 散熱器、顯示卡長度 380mm (無水箱幫浦)、電源供應器長度 200mm 的安裝空間 (無硬碟支架)。7+2 的 PCI-E 插槽讓使用者可選擇直立式擺放顯卡，而電源保護殼的設計讓線材收納整理更加便利。</div>

<h3>
曜越 Versa T26 與 T27 TG ARGB 強化玻璃中直立式機殼 產品特色</h3>
<div>
 </div>
<ul>
<li>
<b>Versa T26 強化玻璃與網孔面板</b></li>
</ul>
<div>
　　網孔和強化玻璃組合而成的特殊面板造型為 Versa T26 TG ARGB 的招牌特色。透視的效果兼顧良好散熱設計，可謂一舉兩得。</div>
<div>
 </div>
<ul>
<li>
<b>Versa T27 網孔面板強化風流</b></li>
</ul>
<div>
　　Versa T27 TG ARGB 前方為大面積網孔面板，可大幅度提升整體散熱效果。</div>
<div>
 </div>
<ul>
<li>
<b>內建三顆 ARGB 主板連動風扇</b></li>
</ul>
<div>
　　內建前方三顆 120mm 5V ARGB 主板連動風扇，可透過 I/O 上的 RGB 燈光控制鍵或是華碩、技嘉、微星、華擎主板燈光控制軟體操控 RGB 燈光效果。實體 RGB 燈光控制鍵可變換 27 種模式，RGB 燈光控制和主板連動操控之間切換也只需簡單地長按 RGB 燈光控制鍵三秒即可。</div>
<div>
 </div>
<ul>
<li>
<b>電源保護殼設計</b></li>
</ul>
<div>
　　電源保護殼設計，提供玩家線材收納理線區域，保持外觀簡潔不雜亂也提供更良好的通風的系統環境。</div>
<div>
 </div>
<ul>
<li>
<b>優異內部擴充和通風設計</b></li>
</ul>
<div>
　　Versa T26 和 T27 的設計除了賦予它價格上的高度市場競爭力外，也同時擁有足夠的內部擴充性。可容納長 175mm 塔式 CPU 散熱器、顯示卡長度 380mm (無水箱幫浦)、電源供應器長度 200mm 的安裝空間 (無硬碟支架)。散熱支援部分，除了內建前方三顆 120mm 5V ARGB 主板連動風扇，另外可選擇安裝三顆 140mm 風扇在前方。上方最大可支援安裝三顆 120mm 或兩顆 140mm 風扇。水冷排安裝上，前方和上方最大可支援至 360mm 或 280mm，為遊戲玩家提供理想的散熱系統解決方案。</div>
<div>
 </div>
<ul>
<li>
<b>簡約 I/O 介面設計</b></li>
</ul>
<div>
　　便捷優雅設計的 I/O port 包含兩組 USB 3.0 I/O 埠和 RGB 燈光控制鍵。</div>
<div>
 </div>
<ul>
<li>
<b>支援主板燈光控制軟體</b></li>
</ul>
<div>
　　支援華碩 AURA SYNC、技嘉 RGB Fusion、微星 Mystic Light Sync 及華擎 Poly<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=chrome" target="_blank">chrome</a> 軟體，玩家可透過上述之主機板應用軟體程式控制 LED 燈光模式，無需安裝額外燈光軟體或控制器。更多訊息請查詢華碩、技嘉及微<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%98%9F%E5%AE%98" target="_blank">星官</a>方網站。</div>
