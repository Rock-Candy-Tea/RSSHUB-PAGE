<div>
<div>
<div>
　　Razer 今（18）日宣布推出最新 Razer Leviathan 系列音箱「Leviathan V2 X」，外型小巧，提供強勁音效和寬廣聲場，採用 USB Type C 和藍牙兩種連接方式，讓玩家在娛樂時能獲得更佳的體驗。</div>
<div>
 </div>
<div>

</ul>
</div>
<div>
 </div>
<div>
　　Razer 表示，Leviathan V2 X 是一款精巧型電腦長形音箱，配備兩個全音域驅動單體和兩個被動輻射器，提供優異的清晰音質與共鳴深度，讓玩家在進行任何娛樂活動時都能獲得沉浸式的音效體驗。音箱長度 40 公分，可放置在市面上大多數螢幕下方，不佔桌面空間。</div>
<div>
 </div>
<div>
　　以 USB Type C 線材即可替 Leviathan V2 X 提供電力並傳輸訊號。經官方測試，Leviathan V2 X 距離 1 公尺能輸出 90dB 的音量 <span style="color:#b22222;">*</span> ；而 Levithan V2 X 也承襲系列設計，底部搭載遊戲裝置燈光生態系統 Razer Chroma RGB ，並擁有 14 個可獨立自訂的發光區域，可打造更多元化的個性燈效模式，並實現與遊戲內互聯回應。</div>
<div>
 </div>
<div>
<span style="font-size:13px;"><span style="color:#b22222;">* 依據輸出電源的功率而有所不同。</span></span></div>
<div>
 </div>

<div>
 </div>
<div>
　　此外，Leviathan V2 X 同時支援藍牙 5.0，玩家可以透過 Razer Audio App 或 Razer Synapse3 在手機或是電腦間切換，以及進行許多自訂設置，包括音量調節、等化器預設、燈效自訂等。</div>
