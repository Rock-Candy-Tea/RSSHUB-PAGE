<div>
　　微軟今日於台北國際電腦展 COMPUTEX 2022 發表以「新 PC 世代及 Windows 11 在混合世界下的未來樣貌」為題的專題演講，講述 PC 在疫後混合工作與生活新常態中扮演比過去更加重要的角色。微軟強調，為滿足混合世界的需求所設計的 Windows 11 增加許多新功能，同時也更著重在包容性設計，搭配 Windows 365 雲端服務及 Pluton 硬體安全性，希望讓 Windows 11 能夠適用於每個人、各種環境及各種裝置。</div>

<h3>
微軟透露全球有超過 14 億的 Windows 用戶，Windows 11 升級速度是 Windows 10 的兩倍</h3>
<div>
 </div>
<div>
　　微軟執行副總裁兼首席產品長 Panos Panay 表示：「注重品質的 Windows 11 獲得使用者及企業用戶空前的支持，不僅使用者升級到 Windows 11 的速度是 Windows 10 同期的兩倍、企業採用 Windows 11 的速度比 Windows 10 更快，Windows 11 產品滿意度也是 Windows 有史以來最高的。」全球有超過 14 億的 Windows 用戶，這個數字還在繼續成長。</div>
<div>
 </div>

<div>
 </div>
<div>
　　因應混合成為疫後新常態，微軟裝置合作夥伴業務全球副總裁 Nicole Dezen 對混合提出工作、學習及遊戲等不同情境的定義。Dezen 表示，混合不僅是彈性的工作方式和地點，它使人們能夠以不同的方式思考會議、辦公空間與協作方式。這徹底改變了學習方法，讓學生保持參與，找到新的教學方式。在遊戲上，混合已經產生了一種新的遊戲方式和跨多個平台的互動方式，混合意味著彈性、客製化和適應能力。</div>
<div>
 </div>
<div>
　　Windows 365 推出市場已屆一年，微軟執行副總裁兼首席產品長 Panos Panay分享，Windows 365 基本上將 Windows 帶到雲端，讓 Windows 可在任何裝置上使用，稱為 Cloud PC。透過瀏覽器即可使用 Windows 365 強大功能，所有應用程式、設定和數據都安全地儲存在微軟雲端中，並串流傳輸到任何裝置。讓使用者不管是使用任何裝置，只要登入 Windows 365，Cloud PC 就在使用者上次離開的地方。</div>
<div>
 </div>
<div>
　　台灣微軟總經理孫基康此次也受邀擔任 COMPUTEX 2022 全球記者會與談嘉賓，與外貿協會董事長黃志芳、台北市電腦公會理事長彭双浪、廣達集團創辦人暨總裁林百里、台達研究院院長闕志克、Arm 台灣總裁曾志光共同討論「全球科技產業的數位韌性」。他指出，企業能否因應這波浪潮，建構科技強度、擁抱雲端，並善用數據提供的機會，將是後疫情時代成功關鍵。隨著 5G、AIoT 應用逐漸普及，以及更多企業加速數位轉型，數位化佈局將逐步改變企業營運與決策，進而強化數位韌性，化挑戰為轉機。</div>
