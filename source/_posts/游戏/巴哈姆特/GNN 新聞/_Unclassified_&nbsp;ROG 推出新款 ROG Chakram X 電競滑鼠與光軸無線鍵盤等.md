<div>
<div>
<div>
　　ROG 玩家共和國今（7）日推出四款電競配備，包括：「ROG Cetra True Wireless 真無線藍牙耳機」、「ROG Chakram X 無線電競滑鼠」、「ROG Strix Scope RX TKL Wireless Deluxe 光軸無線鍵盤」與「ROG Strix Scope NX Wireless Deluxe 機械式無線鍵盤」。</div>
<div>
 </div>
<h3>
<b>ROG Cetra True Wireless 真無線藍牙耳機</b></h3>
<div>
 </div>
<div>
　　官方指出，ROG Cetra True Wireless 真無線藍牙耳機內建經特別調校的 10mm ASUS Essence 驅動單體，具備遊戲模式與混合式主動降噪技術 (ANC)，不僅能提供與視訊同步的低延遲音訊，還可偵測、過濾來自耳機內、外部噪音；搭配以手指輕觸即可於遊戲中快速進行調整的觸控選項，將為玩家帶來更多勝利優勢；此外，ROG Cetra True Wireless 具備 27 小時電池續航及快速充電技術，耳機盒還可透過相容的無線充電版進行充電，加上 IPX4 防水功能，將提升耳機保護力與耐用性。</div>
<div>
 </div>
<div>
<b>ROG Cetra True Wireless，建議售價：NT$3,490</b></div>
<div>
 </div>

 </div>
<h3>
<b>ROG Chakram X 無線電競滑鼠</b></h3>
<div>
 </div>
<div>
　　ROG Chakram X 為華碩首款搭載「ROG AimPoint」光學感測器的電競滑鼠，結合 36,000dpi 的感測器具備 8,000Hz 輪詢率，可提供玩家精準度，同時支援有線 USB、2.4 GHz RF、藍牙三種連線模式，可配對至多三台裝置；搭配全新搖桿設計，玩家得以滑鼠輔助鍵盤指令，選擇類比或數位模式，符合個人遊戲需求；升級後的第二代獨家可更換微動插槽，能快速替換 3-pin 機械式微動開關與 5-pin 光學微動開關，提供不同點擊阻力，延長滑鼠使用壽命，加上 DPI On-The-Scroll 功能、免螺絲磁吸式按鍵 / 上蓋與可客製化銘版，則可讓玩家即時調整滑鼠靈敏度，方便玩家自由更換客製配件。此外，ROG Chakram X 還可透過有線或 Qi 無線充電，享受長達 144 小時的遊戲時間。</div>
<div>
 </div>
<div>
<b>ROG Chakram X 建議售價：NT$4,990</b></div>

<h3>
<b>ROG Strix Scope RX TKL Wireless Deluxe 光軸無線鍵盤 / ROG Strix Scope NX Wireless Deluxe 機械式無線鍵盤</b></h3>
<div>
 </div>
<div>
　　ROG 表示，本次新推出的 ROG Strix Scope RX TKL Wireless Deluxe 光軸無線鍵盤，以及 ROG Strix Scope NX Wireless Deluxe 機械式無線鍵盤，擁有穩定敏捷的觸發力、回彈力與高精準度外，亦採用 PBT 二色成型鍵帽，以提供良好觸感與耐用性，同時還具備 2.4 GHz RF、藍牙、有線 USB 三種連線模式，其中藍牙模式至多可配對三台裝置，方便玩家隨時隨地連線使用；搭配符合人體工學的磁吸式人造皮革手托設計，就算是馬拉松式的持久戰，也能有舒適的遊戲體驗；此外，兩款無線鍵盤均內建獨家 Aura Sync 同步燈效，使用者可依個人喜好直覺設定。</div>
<div>
 </div>
<div>
<b>ROG Strix Scope RX TKL Wireless Deluxe 光軸無線鍵盤，建議售價：NT$5,490</b></div>
