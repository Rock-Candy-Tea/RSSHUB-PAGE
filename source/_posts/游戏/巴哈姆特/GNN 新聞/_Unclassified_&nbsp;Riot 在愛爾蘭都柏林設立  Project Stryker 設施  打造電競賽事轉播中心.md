<div>
　　Riot Games 宣布，AWS Project Stryker 計畫中三座遠端轉播中心（Remote Broadcast Center，RBC）的首座設施正式啟用。這個計畫是創新的全球轉播製作方法，希望改變全世界的電競賽事轉播。 </div>

 </div>

<div>
　　Riot Games 表示，經過三年開發，Project Stryker 的都柏林 RBC 將作為地區及國際電競賽事的中央轉播中心，轉播內容會橫跨 Riot 三大電競品牌：《英雄聯盟》電競（LoL 電競）、《特戰英豪》電競以及《激鬥峽谷》電競。世界各地的電競賽事轉播可以傳送到都柏林 RBC，並在那裡將內容用多種語言製作、發佈並播送給全球數百萬的電競粉絲。</div>
<div>
 </div>
<div>
　　「對於 Riot Games 和電競的未來，今天可真是令人振奮的一天，」Riot Games 電競部門總裁 John Needham 說道。「Project Stryker 在這三年來取得了長足的進步，從一個未來展望的抽象概念，搖身一變成為最先進的創意空間，進一步拓展了Riot在都柏林的足跡，並向大家展現這座城市跟娛樂和創新的緊密聯繫。這裡就是設立 Project Stryker 歐洲總部的絕佳地點。」 </div>
<div>
 </div>
<div>
　　官方表示，會選擇都柏林作為 Project Stryker RBC 三個全球地點中的第一個，是因為這座城市的基礎軟硬體設施品質、在歐洲市場的地位、人力資源，以及在全球電競體系的戰略位置等條件，使其成為 Project Stryker 歐洲總部的理想地點。 </div>
<div>
 </div>
<div>
　　Riot 指出，在都柏林地區進行詳盡調查後，他們最終在斯沃司城中選擇了前身為 Wright Venue 的場地。這裡曾被視為都柏林最具代表性的娛樂場所之一，未來也將保有其娛樂事業的根基，作為 Project Stryker Dublin 轉播製作、工程、活動、音效和平面設計專業人士的工作空間，每年將創造超過 6300 小時的產出。   </div>
<div>
 </div>
<div>
　　官方表示，Project Stryker 是 Riot 在都柏林留下的第二道足跡，而他們現有的市中心辦公室已有 165 位員工，其專業橫跨商務營運、開發、工程、在地化和玩家客服。市中心辦公室的規模目前僅次於洛杉磯全球總部。 </div>

<div>
　　Riot 指出，他們除了原有的 LoL 電競之外，再加入《特戰英豪》冠軍巡迴賽和《激鬥峽谷》電競，成為 Riot 的三大國際電競賽事，分別拓展了線上 MOBA、第一人稱射擊遊戲（first-person shooter，FPS）和手機 MOBA 的電競領域。在這三款遊戲的電競賽事中，地區性和全球性的電競比賽和轉播製作都呈現了巨幅增長，促使 Riot 去創造符合成本效益且可在全球施行的製作解決方案，且日後要能持續擴大規模，這也反映了 Riot 未來在電競和科技方面的投資。 </div>
<div>
 </div>
<div>
　　官方表示，Project Stryker 三座 RBC 都巧妙地設在時區相差八小時的地點，創造「日不落」的轉播模式，可以全年無休地支援電競轉播製作。在電競賽事期間，現場的小型發送裝置會透過 Riot Direct 將即時影像傳送回 RBC。Riot Direct 是全球私人網路供應商，全世界所有玩家發出的每個 Riot 遊戲封包都需要透過它來傳送。接著，影像會轉送到控制室，製作成各種語言的節目。 </div>
<div>
 </div>
<div>
　　除了內容製作以外，每個 RBC 還會作為全球賽事硬體的集中倉儲和運輸地點，確保在 Riot 全球電競賽事中，職業選手能夠使用品質絕佳的產品。儲存內容是 Project Stryker的 另一個功能。每個地點還會設有數據中心和媒體內容資料庫，用來確實保存檔案以供未來查閱。 而在 Project Stryker 中，Cisco、Amazon 網路服務都是 Riot 合作夥伴。</div>
<div>
 </div>
<div>
　　官方表示，Project Stryker RBC 專用空間，例如多功能工作室與 AR/XR 攝影棚，為 Riot 遊戲和娛樂工作室提供發揮創意與潛能的製作機會，而媒體內容資料庫則確保電競內容都有留下存檔，以供日後查閱。 </div>

<div>
　　Project Stryker Dublin 在本月初進行了它的首場國際賽事：《激鬥峽谷》 Icons 世界大賽，目前則正在支援於哥本哈根舉行的《特戰英豪》第二階段大師賽。下一座 RBC 目前正在大西雅圖地區開發中，預計完工時間為 2023 年初。而第三座、也是最後一座 Project Stryker RBC 將設置於亞太地區，預定將於 2024 年 Q1 開始全面營運。 </div>
