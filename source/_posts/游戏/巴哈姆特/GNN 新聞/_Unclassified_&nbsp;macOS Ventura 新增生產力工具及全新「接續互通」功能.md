　　Apple 今天預覽 macOS Ventura，世上最先進桌上型電腦作業系統的最新版本，將 Mac 使用體驗提升至全新層次。「幕前調度」為 Mac 使用者提供在 app 及視窗中流暢切換的嶄新形式，讓使用者得以保持專注地完成眼前任務。「接續互通相機」利用 iPhone 作為 Mac 的網路攝影機，實現以往無法達成的任務。透過 FaceTime 即將推出的「接力」功能，使用者可將在 iPhone 或 iPad 上的通話流暢地轉移到 Mac 上。 「郵件」以及「訊息」皆迎來絕佳的全新功能，讓這兩款 app 比以往更好用。Mac 上最快的瀏覽器 Safari 以通行密鑰功能，引領使用者進入無密碼的未來。此外，強大且廣受喜愛的 Apple 晶片，結合 Metal 3 中的新開發者工具，讓 Mac 的遊戲體驗達到歷來最佳。</div>

　　Apple 軟體工程資深副總裁 Craig Federighi 表示：「macOS Ventura 推出多項強大功能及創新技術，為 Mac 提供超乎以往的體驗。新功能『幕前調度』讓在 app 和視窗之間切換變得空前簡單且迅速，同時也更容易專注於眼前的任務；『接續互通相機』將 Desk View、攝影棚燈光等全新視訊會議功能帶到所有 Mac 機型。再加上『訊息』的實用新功能；『郵件』的最先進搜尋技術，以及 Spotlight 的設計更新，Ventura 帶來多項創新體驗，讓顧客使用 Mac 的方式變得更加豐富。」</div>

<h3>
跨 app 與視窗工作的全新方式</h3>
<div>
 </div>
<div>
　　「幕前調度」可以自動整理開啟的 app 和視窗，讓使用者專心於任務之餘，仍可一覽無遺。使用者正在使用的視窗會占據畫面中央的主要位置；而其他開啟的視窗則會顯示在左側，便於快速且簡易地切換任務。使用者在執行特定任務或專案時，可以將視窗群組起來，顯示不同工作所需的 app。「幕前調度」還可搭配 macOS 其他視窗工具一起使用，包含「指揮中心」以及「分割顯示」，使用者可輕鬆單擊返回桌面。</div>
<div>
 </div>
<h3>
Apple 裝置透過「接續互通」合作無間</h3>
<div>
 </div>
<div>
　　有了「接續互通相機」，Mac 顧客現在可以使用自己的 iPhone 作為網路攝影機，解鎖以往網路攝影機所沒有的全新能力。藉由「接續互通」之力，Mac 可以自動辨識並使用附近的 iPhone 相機，無須喚醒或選擇；iPhone 則能與 Mac 無線連結，創造更高的靈活度。「接續互通相機」為所有 Mac 電腦提供多項創新功能，包含「人物居中」、「人像」模式，以及在背景較暗時，巧妙提亮使用者臉孔的全新攝影棚燈光效果。此外，「接續互通相機」透過 iPhone 上的超廣角相機促成了 Desk View 功能，可同時呈現使用者的臉部以及桌面的俯瞰視角，特別適合製作 DIY 影片、在 FaceTime 通話裡呈現素描創作等場合。</div>

<h3>
macOS 重要 app 與功能的強力更新</h3>

<div>
　　Safari 提供 Mac 最快、最節能的瀏覽體驗之外，也擁有開創性的隱私權功能。在 macOS Ventura 中，Safari 推出強大的全新方式讓使用者共同瀏覽：透過共享「標籤頁群組」，使用者不僅能將 Safari 裡的喜好頁面分享給親友和同事，也能即時查看其他人正在瀏覽的分頁。使用者也可以在共同的「起始頁面」上建立書籤列表，甚至直接在 Safari 上進行「訊息」對話或 FaceTime 通話。非常適合共同規劃旅行或進行專案研究。</div>

 </div>
<div>
　　有了通行密鑰，在 Safari 中瀏覽網頁變得更加安全。這種新一代憑證更安全且容易使用，專為取代密碼而設計。通行密鑰是一種獨特的數位鑰匙，僅保留在裝置上，而永遠不會被儲存到網路伺服器上，因此駭客無法洩露通行密鑰，或欺騙使用者共享。通行密鑰讓安全登入變得更簡單，可以用 Touch ID 或 Face ID 取得生物辨識，也能在 iCloud 鑰匙圈透過端對端加密，同步到 iPhone、Mac 和 Apple TV 上。這些密鑰也將能跨 app 和網路運作，讓使用者透過 iPhone 便可登入非 Apple 裝置上的網頁或 app。</div>

<h3>
引人入勝的遊戲體驗</h3>
<div>
 </div>
<div>
　　Apple 晶片的強大效能讓每一台新款 Mac 都能輕鬆運行 AAA 級遊戲，包含即將上架的作品如 EA 的 《極速房車賽 Legends》（GRID Legends）以及 Capcom 的《惡靈古堡 8：村莊》（Resident Evil Village）。iPad 同樣由強大的 Apple 晶片驅動，因此遊戲開發者能將 AAA 級遊戲體驗帶給更多的使用者，例如由 Hello Games 所推出的《無人深空》（No Man's Sky）也將在今年稍晚登陸 Mac 以及 iPad。</div>
<div>
 </div>
<div>
　　Metal 3 作為軟體的最新版本，強化了跨 Apple 各平台的遊戲體驗，並帶來將 Mac 遊戲體驗提升到新高度的全新功能，在未來幾年釋放 Apple 晶片的完整潛力。MetalFX Upscaling 讓開發者能夠使用計算密集度較低的影格，然後應用高品質的空間放大和頁框交錯反鋸齒技術，快速算繪複雜場景。這項效能加速將帶給玩家更真切的感受，以及令人驚豔的圖像。Fast Resource Loading API 也將為遊戲開發者帶來助益；該 API 提供從儲存空間到 GPU 之間更直接的路徑，以將等待時間降到最低，如此一來，遊戲能更輕鬆存取高品質材質和幾何圖形，打造出更廣闊的世界，從而帶來逼真、身歷其境的遊戲體驗。</div>
<div>
 </div>
<h3>
macOS Ventura 帶來的更多絕佳體驗</h3>
<div>
 </div>
<ul>
<li>
「原況文字」運用裝置端智慧，能跨系統辨識圖片中的文字。現在，此項功能也支援暫停的影片影格，以及日文與韓文。使用者現在也可以將主題從影像中提取出來，拖放到其他 app 裡。而「圖像查詢」亦擴大了辨識功能，可認出動物、鳥類、昆蟲、雕像，以及更多地標</li>
</ul>
<ul>
<li>
iPhone 的「天氣」和「時鐘」 app，以及它們廣受使用者熟知且喜愛的所有功能，現在也經過最佳化適用於 Mac 了。</li>
</ul>
<ul>
<li>
全新輔助使用工具，包含適用於所有語音內容的「即時字幕」；在通話中使用 Type to Speak，以及能協助「旁白」使用者進行校閱的 Text Checker 等實用功能。</li>
</ul>
<ul>
<li>
「系統偏好設定」重新命名為「系統設定」，經過重新設計後，更清爽精簡且易於操作，iPhone 和 iPad 的使用者將備感熟悉。</li>
</ul>
<ul>
<li>
在多種新工具的幫助下，macOS 的安全性更加鞏固，能為 Mac 阻擋更多攻擊。其中 Rapid Security Response 不僅能在一般的更新間維持運行，且無須重啟就能讓安全性保持在最新狀態。</li>
</ul>
