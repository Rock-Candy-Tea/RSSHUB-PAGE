<div>
　　電腦硬體品牌 MSI 今日宣布，與《新世紀福音戰士》系列聞名的 EVANGELION e: PROJECT(EVA e:PROJECT) 聯手合作，打造 Evangelion Unit-01 (又稱 Eva-01TEST TYPE) 主題電腦。</div>
<div>
 </div>
<div>
<ul class="bh-grids-img">
<li class="bh-grids-img-box" style="width: 99.74%;">

</ul>
</div>
<div>
 </div>
<div>
　　MSI 表示，「EVANGELION e: PROJECT」將福音戰士的世界觀和精神融入MSI 電競品牌產品中，首次展開合作關係，EVANGELION e:Project 聯名系列包含 MSI 主機板、一體式水冷散熱器、電源供應器和機殼，設計靈感源自福音戰士中的「EVA-01 TEST TYPE」初號機，採用 Evangelion Unit-01（又名稱為 Eva-01 測試型）獨家配色聯名設計。詳細內容可看此篇巴哈姆特 GNN 搶先開箱報導：<a href="https://gnn.gamer.com.tw/detail.php?sn=231470" target="_blank">MSI 微星 x EVANGELION e:Project 聯名主機殼、主機板、水冷、電源供應器開箱介紹</a>。</div>
<div>
 </div>
<h3>
MSI MAG B660 TOMAHAWK EVA e-PROJECT 主機板</h3>
<div>
 </div>
<div>
　　MSI 表示，限量版 MAG B660 TOMAHAWK EVA e:PROJECT主機板強調具備完善供電系統與擴充支援，是 EVA-01 TEST TYPE 初號機聯名款重要基石。標誌著經典「NERV」、EVANGELION 標誌散熱片。再加上 MSI 的 Mystic Light 獨特的 EVA 燈光效果，為聯名桌機增添風采。</div>

<h3>
MSI MAG CORELIQUID C240 EVA e-PROJECT 一體式水冷</h3>
<div>
 </div>
<div>
　　MSI 指出，MAG CORELIQUID C240 EVA e:PROJECT 一體式 (AIO) 水冷散熱器是本系列的另一個核心元件，NERV 標誌結合 ARGB 燈效和優異解熱能力，希望藉此展現 EVA-01 TEST TYPE 初號機出色戰鬥力。</div>

<h3>
MSI MAG A650BN EVA e-PROJECT 電源供應器</h3>
<div>
 </div>
<div>
　　官方表示，MAG A650BN EVA e:PROJECT 電源供應器 (PSU) 可為 EVA-01 TEST TYPE 初號機聯名款，提供高穩定和出色電源效率，並結合經典配色外殼和風扇格柵。</div>

<h3>
MPG GUNGNIR 110R EVA e-PROJECT 機殼</h3>
<div>
 </div>
<div>
　　官方指出，MPG GUNGNIR 110R EVA e-PROJECT 聯名款機殼隱含驚喜。紫和綠配色設計配合，鋼化玻璃側板採用令人熟悉的 Evangelion (福音戰士) ，而機殼的另一側面板展示了 EVA-01 TEST TYPE 的精神圖騰，藉此完整呈現 EVA-01 TEST TYPE 初號機的靈魂。</div>
