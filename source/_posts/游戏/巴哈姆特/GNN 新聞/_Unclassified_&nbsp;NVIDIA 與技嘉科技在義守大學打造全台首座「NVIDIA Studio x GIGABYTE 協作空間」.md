<div>
　　NVIDIA 今（7）日宣布，與技嘉科技（GIGABYTE）攜手義守大學數位多媒體設計學系（以下簡稱數媒系）共同打造全台首座「NVIDIA Studio x GIGABYTE 協作空間」，引進 NVIDIA Studio 平台與通過 NVIDIA Studio 認證的 GIGABYTE AERO 16 高效能創作者筆記型電腦，積極發展 3D 創新應用，為義守大學數媒系提供最佳的應用工具並開拓先進的創作環境，共同培育台灣新媒體設計與數位內容開發產業所需人才。</div>

<h3>
提供學生取得 NVIDIA 創作工具和應用程式的管道</h3>
<div>
 </div>
<div>
　　官方指出，NVIDIA Studio 平台專為各類創作者與專業人士提供支援，透過加速的創作者應用程式及人工智慧（AI）工具，讓創作者可以更輕鬆、更快速的方式從事創作，並大幅穩定創作流程。NVIDIA Studio 認證的筆記型電腦由 NVIDIA RTX GPU 提供支援，是專為要求嚴苛的創作者而設計及測試。</div>
<div>
 </div>
<div>
　　NVIDIA 另推出多款專為創造者打造的 RTX 加速軟體，包含用於先進 3D 設計與協作的 NVIDIA Omniverse 平台、用於直播的 NVIDIA Broadcast 應用程式，以及運用 AI 描繪風景的 NVIDIA Canvas。</div>

<div>
　　NVIDIA 台灣區業務協理邾亦為表示：「我們很榮幸此次能與技嘉科技和義守大學合作，透過實體協作空間的設立，將 NVIDIA Studio 平台帶進校園，持續為創作者帶來節省時間的工具，並讓學生們可以運用符合業界標準的平台進行創作，發揮無限的創意可能。」</div>
<div>
 </div>
<h3>
用於教學和課程內容的 NVIDIA Studio</h3>
<div>
 </div>
<div>
　　官方指出，導入 NVIDIA Studio 平台後，義守大學數媒系將實際應用於 3D 動畫設計、3D 遊戲設計、元宇宙科技藝術概論等實務課程。其中 3D 動畫設計課程將結合 Omniverse Machinima 軟體應用程式，讓學生能運用 AI 工具與 RTX 渲染技術，創作電影般的擬真人物動畫與動畫場景。NVIDIA Omniverse 及 Machinima 皆為 NVIDIA 提供給創作者免費下載的工具。</div>
<div>
 </div>
<h3>
NVIDIA Studio x GIGABYTE 協作空間</h3>
<div>
 </div>
<div>
　　此外，「NVIDIA Studio x GIGABYTE 協作空間」將整合義守大學的「元宇宙科技藝術設計中心」，建立先進的新媒體設計流程並強化產學合作關係。透過協作空間的虛擬實境智慧攝影棚與 NVIDIA Studio 平台，學生可以運用符合業界標準的配備與軟體，了解端到端的 3D 製作流程。官方指出，學生將學習從前端企劃與設計、動畫製作、虛擬遊戲場景設計，到後期製作與視覺特效，打造獨一無二的元宇宙。</div>

<div>
　　義守大學數位多媒體設計學系孫志彬主任表示：「我們很高興能與 NVIDIA 和技嘉科技攜手，共同規劃建置台灣首座 NVIDIA Studio 協作空間，強化本系的數位內容設計能量，成為台灣大專院校的先行者。目前課程著重於 3D 協同設計、虛實整合設計、3D 動畫和遊戲美術開發，預期將大幅受益於 NVIDIA Studio 平台的軟硬體資源，進一步鏈結高雄亞灣 5G XR 產業資源與就業需求，厚植學生的學習力與就業力。」</div>
<div>
 </div>
<h3>
GIGABYTE AERO 16 是內容創作者的絕佳工具</h3>
<div>
 </div>
<div>
　　官方指出，本次結合技嘉旗下專為藝術家與內容創意者所打造的創作者系列筆記型電腦 AERO 16。AERO 16 系列採用 NVIDIA GeForce RTX 30 系列筆記型電腦 GPU，並通過 NVIDIA Studio 認證，對於常見的影音編輯軟體、3D 建模工具等皆帶來顯著的優化，同時能夠大幅縮短在支援的軟體工具上進行多媒體預覽、多媒體轉檔、3D 影像處理、平面視覺處理，甚至是特效觀賞的時間，為數媒系師生帶來更加順暢的創作旅程。</div>

<h3>
學術合作與交流</h3>
<div>
 </div>
<div>
　　官方表示，未來 NVIDIA 將與義守大學共同舉辦一系列的講座活動，邀請產業界專業人士分享數位創作領域的實際應用案例，讓師生能更加了解產業常見的實務做法和見解。</div>
