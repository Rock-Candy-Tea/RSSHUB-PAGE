<div>
<div>
<div>
　　華碩今（11）日推出多款全新 AMD RX6000 顯示卡，包括：「ASUS TUF Gaming Radeon RX 6950 XT」、「ROG Strix Radeon RX 6750 XT」、「ASUS Dual Radeon RX 6750 / 6650 XT」，以及「ASUS Dual Radeon RX 6400」與「Phoenix Radeon RX 6400」，規格從入門至高階、藉此吸引玩家的目光。</div>
<div>
 </div>
<h3>
世界級效能與耐用性─ASUS TUF Gaming Radeon RX 6950 XT</h3>
<div>
 </div>
<div>
　　官方指出，ASUS TUF Gaming Radeon RX 6950 XT，搭載 2.9-slot 散熱器，可透過 MaxContact 技術提供強大散熱，搭配三組最新軸向式風扇與 0dB 技術，能使顯示卡在輕度負載或閒置時保持靜音。承襲 TUF Gaming 系列設計，護蓋由金屬製成，底部除有強化背板保護，具備可加速散熱的通風口與不鏽鋼 I/O 擋板；再加上內建雙 BIOS 開關，可自行選擇效能或安靜模式；機殼上的 TUF Gaming 標誌可透過 Aura Sync 技術與相容電腦設備的 RGB 燈效同步，展現玩家獨有風格。</div>

<h3>
格調與實力的完美化身─ROG Strix Radeon RX 6750 XT </h3>
<div>
 </div>
<div>
　　華碩表示，ROG Strix Radeon RX 6750 XT 同樣採用 2.9-slot 散熱器、內建 MaxContact 技術、三組升級版的 0dB 軸向式風扇，以及雙 BIOS 開關；搭配 FanConnect II 接頭，可同步連動 GPU 溫度與機殼風扇，達冷卻效果；ROG Strix Radeon RX 6750 XT，不僅擁有 RGB 燈效綴飾的護蓋，並相容所有 Aura Sync 組件；背板部分，除提升外觀與穩定性，右側通風口還可幫助散熱，維持系統運作；不鏽鋼 I/O 擋板與強化金屬框架則能為 Super Alloy Power II 頂級組件提供卓越耐久性，讓顯示卡更加完善，是電腦組裝中不可或缺的關鍵要角。</div>

<h3>
小巧尺寸兼具優異性能─ASUS Dual Radeon RX 6750 XT</h3>
<div>
 </div>
<div>
　　官方指出，可彈性安裝於小型電腦的 ASUS Dual Radeon RX 6750 XT，其中兩組採用雙滾珠軸承的軸向式風扇，可將氣流直接導向發熱源，並在 GPU 溫度達 55°C 以下或執行低負載任務時停止運轉，提供安靜作業空間；顯示卡護蓋頂部除有精緻燈條點綴，強化背板亦提供簡約外觀與保護功能，搭配獨家全自動製程技術與 144 小時測試過程，藉此大幅提升產品品質。</div>
<div>
 </div>
<h3>
匯聚玩家所需必備效能─ASUS Dual Radeon RX 6650 XT</h3>
<div>
 </div>
<div>
　　華碩表示，ASUS Dual Radeon RX 6650 XT 內建雙滾珠軸向式風扇，可提供傑出散熱，在低負載時維持靜音；全自動製程技術，能確保耐用性和使用壽命。保護背板和不鏽鋼 I/O 擋板，能維持顯示卡穩定運作發揮最佳效能。</div>

<h3>
為 1080p 遊戲而生─ASUS Dual & Phoenix Radeon RX 6400</h3>
<div>
 </div>
<div>
　　官方表示，ASUS Dual 和 Phoenix Radeon RX 6400 顯示卡，皆經過最佳化，可為主流 1080p 遊戲裝備提供全新選擇。其中 ASUS Dual Radeon RX 6400 採用兩個具有阻隔環的軸向式風扇，可將氣流直接導向散熱器；0dB 技術能在顯示卡達到 52°C 以下時停止風扇運轉，打造寧靜使用空間；而鋁製背板除可使顯示卡更加堅固，為系統帶來吸睛的拋光效果，搭配細膩燈效，亦讓護蓋上的幾何線條增添風采。</div>

　　官方表示，儘管配備精簡單風扇，且無鋁製背板及燈光效果，但 Phoenix Radeon RX 6400 採用與 ASUS Dual Radeon RX 6400 類似設計；透過獨家全自動製程，以及嚴格的 144 小時測試程序製造，確保開箱後就能立即運作。</div>
