<div>
　　HTC 今（12）日宣布推出「VIVE Focus 3 眼球追蹤套件」及「VIVE Focus 3 表情偵測套件」，擴展 VR 一體機生態系統。</div>
<div>
 </div>
<div>
<div>
　　HTC 表示，VIVE Focus 3 全新追蹤套件增強了使用者在虛擬環境中的協作和人際互動，其適用於虛擬活動、技能培訓、社交聊天、VIVE Sync 遠程會議、動畫及動作捕捉等互動應用中，並提供企業一個分析及取得用戶行為反應，以及更易於了解體驗效能和互動表現的新方式。 </div>

<h3>
VIVE Focus 3 表情偵測套件</h3>
<div>
 </div>
<div>
　　「VIVE Focus 3 表情偵測套件」可精準捕捉臉部表情及嘴部動作，透過內建單追蹤攝像鏡頭捕捉嘴唇、頜骨、牙齒、舌頭、下巴和臉頰的 38 種混合動作，展現真實臉部動作與表情，讓使用者可以感受真實的人際交流，提升彼此間互動的真實感。 </div>
<div>
 </div>
<div>
　　HTC 表示，該套件具備 60Hz 追蹤率和優化的臉部追蹤運行時間，以近乎零延遲方式，將嘴唇與聲音同步，提供高傳真的交流；而「VIVE Focus 3 表情偵測套件」搭配 USB Type-C 連接埠可簡易快速連接至 VIVE Focus 3 頭戴式裝置，無須附加任何組件或轉接配備。 </div>
<div>
 </div>
<div>
　　「VIVE Focus 3 表情偵測套件」適用於教育、醫療保健、人力資源和創意產業的人際互動應用中，可提升不同領域的培訓效能，包含演講指導、客戶管理服務、工作技能培訓等。使用者能追蹤及為改善個人情緒作準備，協助在現實生活中呈現更好的培訓效果。 </div>

<h3>
VIVE Focus 3 眼球追蹤套件</h3>
<div>
 </div>
<div>
　　「VIVE Focus 3 眼球追蹤套件」為虛擬化身提供逼真的眼球運動和眨眼效果，幫助在虛擬會議、聊天群組和 VIVE Sync 遠程會議互動中，展示真實的非口語式表現力。此外，有了該套件的輔助，可使應用程式中的注視控制成為可能。而使用者還可運用眼球追蹤技術優勢，深入取得更多數據，提升 VR 沉浸感及眼球動態數據分析。</div>
<div>
 </div>
<div>
　　「VIVE Focus 3 眼球追蹤套件」配有支援紅外線施照器的雙攝影鏡頭，能捕捉目光來源及方向、瞳孔大小、位置和睜眼幅度等數據。搭配磁吸式模組可輕鬆連結頭戴式顯示器。此外，該套件也支援瞳距調整功能可與 VIVE Focus 3 配合，幫助用戶找到觀看最佳點。透過眼球追蹤及動態注視點渲染，還可動態調節 GPU 工作負荷，減少外圍視覺中的 GPU 負荷，針對視線範圍內內容進行優化。</div>
<div>
 </div>
<div>
　　HTC 透露，「VIVE Focus 3 眼球追蹤套件」及「VIVE Focus 3 表情偵測套件」皆與 Unity、Unreal Game Engine 和 Native 相容。內容開發人員可藉由 VIVE Wave SDK 獲得參數、配置注視點渲染等功能，以及 OpenXR 開放授權標準亦即將推出支援一體機的服務，讓第三方合作夥伴開發行動 XR 內容及優化設備性能。 </div>

<h3>
專業一體機 VR 生態系統 </h3>
<div>
 </div>
<div>
　　HTC 表示，VIVE Focus 3 擁有專業級 VR 硬體和軟體生態系統，HTC VIVE 在台推出三款相容配件「VIVE Focus 3 眼球追蹤套件」、「VIVE Focus 3 表情偵測套件」和「VIVE Focus 3 手腕追蹤器」，為企業在打造沉浸 VR 體驗內容及裝置功能層面，提供了良好靈活性。</div>
