　　英特爾今日宣布，旗下第 11 代 Intel Core S 系列桌上型電腦處理器（代號 Rocket Lake-S）日正式在台上市，強調最高擁有 8 核心 16 執行緒，採用 Cypress Cove 微架構，相較前一世代提升 19％ IPC 效能，還有全新 Intel Adaptive Boost Technology（ABT）提升多核心/多執行緒表現，藉此吸引電競玩家的目光。</div>

　　英特爾表示，第 11 代 Intel Core S 系列桌上型電腦處理器由 Core i9-11900K 旗艦級產品領軍，享有最高達 5.3 GHz、8 核心、16 執行緒、Intel Smart Cache 容量亦達 16MB。可超頻的第 11 代 Intel Core 桌上型處理器，支援 DDR4-3200 記憶體規格，協助平台推動流暢的遊戲過程。</div>

<div>
　　英特爾指出，以新款 Cypress Cove 微架構為基礎所打造的第 11 代 Intel Core S 系列桌上型處理器，帶來更好的硬體與軟體轉換效率以提升原始遊戲性能。新款微架構對比上一代，為最高時脈核心帶來最高 19％ 的每時鐘週期指令數量（IPC）提升幅度，搭載 Intel Xe 架構的 Intel UHD 繪圖晶片更擁有豐富的媒體與智慧型繪圖顯示能力。此世代的改進包括：</div>
<div>
 </div>
<ul>
<li>
對比上一代 IPC 效能提升幅度最高達 19％</li>
<li>
搭載 Intel Xe 架構的 Intel UHD繪圖晶片，最高能夠提升 50％ 繪圖晶片效能</li>
<li>
Intel Deep Learning Boost 與 Vector Neural Network 指令支援加速人工智慧（AI）推論 －大幅度改善深度學習工作負載效能</li>
<li>
經強化後的超頻工具與功能，提升超頻彈性以及效能調校與使用體驗</li>
</ul>
