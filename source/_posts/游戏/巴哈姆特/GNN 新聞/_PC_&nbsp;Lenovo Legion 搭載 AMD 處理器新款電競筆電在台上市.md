<h3>
Lenovo 基於玩家回饋 推出全新升級 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 系列 衝刺電競筆電銷量</h3>
<div>
 </div>
<div>
　　受到宅經濟的影響，全球電競市場需求強勁。最新 IDC 數據報告指出，2020 年全球電競 PC 裝置的總出貨量達到 4,960 萬台；台灣市場也較前年成長 9%。Lenovo 旗下子品牌 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 作為 2019 年以來亞太地區成長最快的電競品牌，今年再度推出為專業玩家打造的全新 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 系列筆電。 Lenovo 台灣總經理林祺斌表示：「Lenovo 電競筆電銷售表現亮眼，我們相信這是因為 Lenovo 重視消費者的使用體驗，並根據玩家回饋作為進步原動力的成果。」</div>
<div>
 </div>
<div>
　　Lenovo 針對全球 7,800 位消費者進行問卷調查，深度了解玩家需求後推出新的電競筆電系列，包含搭載 AMD Ryzen 5000H 系列處理器的旗艦機款 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 7、規格頂尖的 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 5 Pro，和主流玩家首選的 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 5，另有耳機配件 Lenovo <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> H600 Wireless Gaming Headset，期待提供台灣電競玩家更完整的產品組合和升級的遊戲體驗，再造亮眼佳績。</div>

<h3>
效能、螢幕、鍵盤、音效 全方位滿足專業玩家所重視的規格</h3>
<div>
 </div>
<div>
　　Lenovo <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 7、<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 5 Pro 以及 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 5 具備以下特色：</div>
<div>
 </div>
<ul>
<li>
<b>狂野的效能配置：</b></li>
</ul>
<div>
　　全新系列搭載效能頂尖的 AMD Ryzen-H 系列處理器，可以提升 70% 的遊戲電競效能以及增加 50% 的繪圖運算效能，提供遊戲玩家和創作者所需的動力；<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 獨有的 Coldfront 3.0 散熱管理系統整合了多重散熱機制，採用渦輪增壓雙風扇以及四通道散熱裝置，增加冷卻效率；<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 7 另配有 2.0mm+ 均熱板加大散熱面積，達到高效散熱。電池續航時間也是消費者關注的重點，<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 筆電最高採用 80Wh 大容量電池，電池續航時間最高延長到 8 小時以上，還可透過 FN + Q 鍵切換筆電效能，依據不同場景選擇極致效能、智慧（AI）或省電模式，達到節能的目的。此系列更支援 30 分鐘即可充滿 60% 快充，以及 Type-C 100w 充電，非常適合需要長時間移動的玩家。</div>
<div>
 </div>
<ul>
<li>
<b><a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 新系列筆電大幅升級螢幕體驗：</b></li>
</ul>
<div>
　　根據 Lenovo 最新的電競玩家觀察報告顯示，電競電腦螢幕的刷新率、顏色精準度以及解析度等為消費者關注的重點，而 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 7 和 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 5 Pro 不僅配備 16 吋的 QHD（2560 x 1600 像素）IPS 螢幕，更將 16:10 的黃金比例螢幕放置在近 15 吋機身上，與傳統的 15.6 吋顯示器相比，提供了額外 11％ 的螢幕空間，提升視覺品質並擴展視野。此外，有了 4 面窄邊框、90% 螢幕占比、100％ sRGB 色域、Dolby Vision 、VESA DisplayHDR 400 ，和 NVIDIA G-SYNC 的支援，使用者可以隨時沉浸在臨場感十足的世界。Lenovo <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> AI Engine 更可透過機器學習自動檢測和優化系統性能，在執行 3A 遊戲大作時，有效提升最高 15% 的每秒數幀率（FPS），確保流暢的遊戲畫面。</div>
<div>
 </div>
<ul>
<li>
<b>可自訂色彩的 TrueStrike 鍵盤：</b></li>
</ul>
<div>
　　<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> TrueStrike 鍵盤配備創新的軟著陸鍵程，可以提供絕佳的按鍵回饋。配備獨立全尺寸方向鍵，並支援全鍵盤防鬼鍵，保證按下多個按鍵不衝突，提供電競玩家最順手的攻擊。 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 5 Pro 和 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 5 兩款機種支援 4 區 RGB 鍵盤；<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 7 更支援 Corsair iCUE 單鍵 RGB 鍵盤，每顆按鍵皆可獨立設定，可自訂多達 1600 萬色的顏色呈現。</div>
<div>
 </div>
<ul>
<li>
<b>栩栩如生的音效技術 Nahimic Audio：</b></li>
</ul>
<div>
　　遊戲玩家追求的身歷其境的效果需要有絕佳的聽覺體驗加持，Nahimic Audio 的沉浸式 3D 音訊，大幅強化了驚艷的音效及細節。Sound Tracker 追蹤技術掌握聲音出處，以便輕鬆辨識敵人方位。搭配 BT Link 技術，可以將手邊所有的藍牙喇叭連接到 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 筆電，打造環繞音場享受 3D 音效。</div>
<div>
 </div>
<div>
<b>Lenovo <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> H600 Wireless Gaming Headset 的特色</b></div>
<ul>
<li>
<b>無線耳機提供絕佳的音效：</b>新的 Lenovo <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> H600 Wireless Gaming Headset 提供競賽級的音效，2.4GHz 無線接收器的接收距離高達 12 公尺，降低延遲時間至低於 35ms。</li>
</ul>

<h3>
電競筆電不只有遊戲功能 高效性能也符合專業工作需求</h3>
<div>
 </div>
<div>
　　<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 電競筆電強調高效能、高畫質、高音質、散熱佳等特色，吸引開發者與影音創作人士將電競筆電用於工作，即使是重度耗能的運算或是需要多工處理的工作也能無縫進行。搭載 AMD Ryzen 5000 系列處理器與 NVIDIARTX 3000 系列顯示卡的強勁效能， <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 7 和 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 5 Pro 在接近 15 吋的機身上安置了 16 吋螢幕，窄邊框設計極大化可視範圍，適合設計師等專業工作者一次查看更多資訊，縮短時間完成專案。</div>

 </div>
<h3>
定價與上市資訊</h3>
<div>

<div>
　　搭載最新 AMD Ryzen 處理器的 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 筆電正式在台上市，旗艦 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 7 建議售價 NT49,990 元起、<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 5 Pro 建議售價 NT39,990 元起、<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 5 建議售價 NT31,990 元起。即日起於 Lenovo 官網可依據玩家自身喜好與需求，打造專屬 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 筆電，在 5/31 前完成下單並於活動頁進行線上登錄，前 30 名完成任務的消費者即可獲得 <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Legion" target="_blank">Legion</a> 水瓶 + 帽子一組（活動細則及詳情請見官網活動頁）；同時也可在崑碁電腦享限量早鳥首賣。</div>
