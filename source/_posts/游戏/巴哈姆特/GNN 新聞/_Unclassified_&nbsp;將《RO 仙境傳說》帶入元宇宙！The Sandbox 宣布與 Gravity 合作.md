<div>
　　The Sandbox 近日宣布，與 Gravity 建立合作關係，預計將《RO <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA" target="_blank">仙境傳說</a>》IP 帶入元宇宙。The Sandbox 表示，透過合作關係，《RO <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA" target="_blank">仙境傳說</a>》將在 The Sandbox 元宇宙中建立土地，還有各種根據《RO <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA" target="_blank">仙境傳說</a>》 IP 打造的內容和創作 NFT。此外，用戶可以參與未來以《RO <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA" target="_blank">仙境傳說</a>》宇宙和人物為主題的遊戲大賽。</div>

　　《RO <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA" target="_blank">仙境傳說</a>》是 Gravity 旗下的主力 IP，自 2002 年推出 PC 線上遊戲《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=RO+%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA+Online" target="_blank">RO 仙境傳說 Online</a>》以來，該 IP 在 PC、行動裝置上都取得了成功。爾後也推出像是《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=RO+%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA%EF%BC%9A%E5%AE%88%E8%AD%B7%E6%B0%B8%E6%81%86%E7%9A%84%E6%84%9B" target="_blank">RO 仙境傳說：守護永恆的愛</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=RO+%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA+Online%EF%BC%9A%E8%B5%B7%E6%BA%90" target="_blank">RO 仙境傳說 Online：起源</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=RO+%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA%EF%BC%9A%E6%96%B0%E4%B8%96%E4%BB%A3%E7%9A%84%E8%AA%95%E7%94%9F" target="_blank">RO 仙境傳說：新世代的誕生</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA%EF%BC%9A%E5%A5%B3%E6%AD%A6%E7%A5%9E%E4%B9%8B%E8%B5%B7%E7%BE%A9" target="_blank">仙境傳說：女武神之起義</a>》和《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=RO+%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA%EF%BC%9A%E6%84%9B%E5%A6%82%E5%88%9D%E8%A6%8B" target="_blank">RO 仙境傳說：愛如初見</a>》等一系列作品，足見玩家們對於此 IP 的喜愛程度。</div>

　　The Sandbox 韓國公司 CEO Cindy Lee 表示：「我們很高興與《RO <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA" target="_blank">仙境傳說</a>》合作，《RO <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA" target="_blank">仙境傳說</a>》長期以來受到全世界玩家的喜愛。透過這次合作，《RO <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA" target="_blank">仙境傳說</a>》給玩家帶來的興奮和快樂可以藉由元數據內容以一種新的方式進行調整。」</div>

<div>
　　Gravity 業務主管 Harry Choi 說道，「這次合作意義重大，世界各地的用戶可以在 The Sandbox 元宇宙中造訪《RO <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA" target="_blank">仙境傳說</a>》，我們正計劃用《RO <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E4%BB%99%E5%A2%83%E5%82%B3%E8%AA%AA" target="_blank">仙境傳說</a>》IP 為用戶提供多樣化的內容，以便在元宇宙中玩耍。敬請期待。」</div>

<h3>
甚麼是 The Sandbox？</h3>
<div>
 </div>
<div>
　　The Sandbox 是一個去中心化的虛擬世界，與 Adidas、Snoop Dogg、華納音樂集團、The Walking Dead、Deadmau5、雅達利、模擬樂園、愛心小熊、藍精靈等主要 IP 和品牌合作，推動了最近虛擬地產需求的增長。基於現有的 The Sandbox IP（在全球流動裝置上的安裝量超過 4,000 萬次），The Sandbox 元宇宙為玩家和創作者提供了一個去中心化且直觀的平台，讓他們能夠創造沈浸式 3D 世界和遊戲體驗，並安全地儲存、交易自己的作品，並從而盈利。</div>
