<div>
<div>
　　微軟遊戲部門執行長菲爾‧史賓瑟（Phil Spencer）今日（9/1）在微軟官方部落格發表一篇名為「<a href="https://blogs.microsoft.com/on-the-issues/2022/09/01/gaming-everyone-everywhere/" target="_blank">不論何人不論何處的遊戲：我們對收購 Activision Blizzard 的看法</a>」的文章，透露微軟計畫讓來自 Activision Blizzard 的暢銷遊戲《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E9%AC%A5%E9%99%A3%E7%89%B9%E6%94%BB" target="_blank">鬥陣特攻</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%9A%97%E9%BB%91%E7%A0%B4%E5%A3%9E%E7%A5%9E" target="_blank">暗黑破壞神</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%B1%BA%E5%8B%9D%E6%99%82%E5%88%BB" target="_blank">決勝時刻</a>》加入 Game Pass 遊戲訂閱服務的陣容。</div>
<div>
 </div>

<div>
 </div>
<div>
　　菲爾‧史賓瑟在文章中說明了微軟收購 Activision Blizzard 的用意。他表示微軟希望能擴大玩家遊玩遊戲的管道，特別是成長最快的行動平台。為了擴大玩家遊玩遊戲的選擇，讓遊戲更容易被玩家獲得，微軟採用兩種方式來達成，其一是創建 Game Pass 提供玩家訂閱的選擇，另一是透過雲端技術讓遊戲登上行動平台。</div>
<div>
 </div>

</ul>
</div>
<div>
 </div>
<div>
　　對此，微軟計畫在完成收購 Activision Blizzard 的程序後，讓 Activision Blizzard 旗下備受玩家喜愛的遊戲陣容，包含《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E9%AC%A5%E9%99%A3%E7%89%B9%E6%94%BB" target="_blank">鬥陣特攻</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%9A%97%E9%BB%91%E7%A0%B4%E5%A3%9E%E7%A5%9E" target="_blank">暗黑破壞神</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%B1%BA%E5%8B%9D%E6%99%82%E5%88%BB" target="_blank">決勝時刻</a>》加入 Game Pass 提供的遊戲中，並繼續發展這些遊戲的社群。於此同時，微軟亦將秉持著讓《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%B1%BA%E5%8B%9D%E6%99%82%E5%88%BB" target="_blank">決勝時刻</a>》繼續在其他平台推出的原則，一如先前《<a class="acglink" href="http://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%88%91%E7%9A%84%E4%B8%96%E7%95%8C" target="_blank">我的世界</a>》的模式，繼續在包含 PlayStation 在內的平台上推出相同版本的《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E6%B1%BA%E5%8B%9D%E6%99%82%E5%88%BB" target="_blank">決勝時刻</a>》系列作，讓玩家有所選擇。</div>
<div>
 </div>
<div>
　　菲爾‧史賓瑟在文末強調，微軟將繼續本著透明開放的精神與審查這次收購案的監管機關接觸，尊重與歡迎其所提出的棘手問題。微軟堅信徹底的審查將表明微軟與 Activision Blizzard 的結合將有利於產業與玩家。</div>
