<div>
<div>
<div>
　　主機板、顯示卡廠商華擎科技今日發表基於 AMD Radeon RX 6950 XT、AMD Radeon RX 6750 XT 和 AMD Radeon RX 6650 XT GPU 的全新 OC Formula、Phantom Gaming 和 Challenger 系列產品。</div>
<div>
 </div>
<div>
　　華擎科技表示，新顯示卡利用突破性的 AMD RDNA 2 遊戲架構，包括流程最佳化與軟體和韌體增強，並提供高頻寬、低延遲的 AMD Infinity Cache 快取技術和 18Gbps GDDR6 顯示記憶體。它們還支援 Microsoft Windows 11 和 Microsoft DirectX 12 Ultimate、AMD FidelityFX 超級解析度（FSR）、即將推出的 AMD FSR 2.0i 和 AMD Radeon 超級解析度（RSR）升頻技術。</div>

 </div>

<div>
 </div>
<div>
　　官方表示，華擎 Radeon RX 6000 系列顯示卡支援各種獨家功能，如條紋環形扇、貼合熱導管、金屬中框、金屬背板和 Polychrome SYNC 技術，提供散熱效率、堅固結構、ARGB LED 燈效和時尚的外觀設計。</div>
<div>
 </div>
<h3>
華擎 Radeon RX 6950 XT OC Formula 16GB 顯示卡</h3>
<div>
 </div>
<div>
　　官方指出，全新旗艦華擎 Radeon RX 6950 XT OC Formula 16GB 顯示卡達到 2495MHz 加速頻率 。華擎全新設計的條紋環形扇以其條紋扇葉和環形結構進一步強化氣流。此外，還保留了原本 OC Formula 的特點，包括 21 相供電架構，為 GPU 提供穩定的電力，全金屬外殼設計搭配三風扇散熱系統，提供出色的散熱效率。內建 ARGB LED 也能與支援 Polychrome SYNC 的華擎主機板同步燈光效果，讓用戶客製化燈光效果。總體而言，華擎 Radeon RX 6950 XT OC Formula 16GB 顯示卡相比原設計引入了更多新元素，進一步提升遊戲玩家的 4K 遊戲體驗。</div>

<h3>
華擎 Radeon RX 6750 XT Phantom Gaming D 12GB OC 顯示卡</h3>
<div>
 </div>
<div>
　　官方表示，高階華擎 Radeon RX 6750 XT Phantom Gaming D 12GB OC 顯示卡提供高達 2623MHz 的加速頻率，配備高速 18Gbps 12GB GDDR6 顯示記憶體。Phantom Gaming 3X 三風扇散熱系統搭配定制的條紋軸流風扇、導流鰭片和超貼合熱導管，提高散熱效率。時尚的金屬背板加強了結構並能防止電路板彎曲，ARGB 風扇和 ARGB LED 面板搭配 Polychrome SYNC 軟體工具，允許用戶客製化燈光效果。</div>

<h3>
華擎 Radeon RX 6750 XT Challenger Pro 12GB OC 顯示卡</h3>
<div>
 </div>
<div>
　　針對主流用戶的華擎 Radeon RX 6750 XT Challenger Pro 12GB OC 顯示卡具備 2618MHz 的加速頻率，並配備超高速 18Gbps 12GB GDDR6 顯示記憶體。官方表示，三風扇散熱設計搭配獨特的條紋軸流風扇提供出色的散熱效果，時尚的金屬背板能加強結構並防止板彎曲，超合金顯卡料件為長時間運作提供穩定性。</div>

<h3>
華擎 Radeon RX 6650 XT Phantom Gaming D 8GB OC 顯示卡</h3>
<div>
 </div>
<div>
　　華擎表示，Radeon RX 6650 XT Phantom Gaming D 8GB OC 顯示卡提供 2694MHz 的加速頻率，配備高速 17.5Gbps 8GB GDDR6 顯示記憶體。配備 Phantom Gaming 3X 三風扇散熱系統，結合條紋軸流風扇、導流鰭片和超貼合熱導管大幅提升散熱效率。此外，金屬背板可以加強結構並防止板彎曲。ARGB 風扇和 ARGB LED 燈板，結合 Polychrome SYNC 軟體，讓用戶可以完全客製化和控制自己的燈光效果。</div>

ASRock Radeon RX 6650 XT Challenger D 8GB OC 顯示卡</h3>
