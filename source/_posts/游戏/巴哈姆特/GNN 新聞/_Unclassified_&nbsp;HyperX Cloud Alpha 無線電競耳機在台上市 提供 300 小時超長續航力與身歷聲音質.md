<div>
　　HP 旗下知名電競周邊產品領導品牌 HyperX 宣布，在今年初的 CES 2022 發表、擁有 300 小時超長電池續航力的 HyperX Cloud Alpha 無線電競耳機，已於今日（4/19）正式在台上市。官方表示，Cloud Alpha 無線電競耳機維持 HyperX 品牌一貫的配戴舒適度，透過 DTS Headphone:X 提供身臨其境的音效體驗，進階版的雙音腔驅動單體可帶來遊戲中更準確且令人驚豔的清晰音質。</div>

<div>
 </div>
<div>
　　HyperX 表示，Cloud Alpha 無線電競耳機旨在為玩家提供頂級的遊戲音效體驗，採用進階版雙音腔技術，可將中、高音與重低音分開調音，提供令人驚豔的清晰音質。結合 DTS Headphone:X 空間音效，帶來精確的聲音定位和身歷其境的 3D 環繞音效，使玩家於遊戲世界中遠離塵囂，或專注於電競比賽之中。耳機採用直徑 50mm 驅動單體，更輕盈的設計之下仍同時保有原始有線耳機版本的絕佳音質和表現。</div>
<div>
 </div>
<div>
　　Cloud Alpha 無線電競耳機耳罩上的控制鍵可快速調整音量、麥克風靜音及啟用麥克風監聽功能。可拆式降噪麥克風具備 Discord 和 TeamSpeak 認證，透過消除背景噪音有效強化耳機麥克風收音品質，提供更清晰的音質。附帶的 LED 靜音指示燈，更可輕鬆辨別麥克風狀態。</div>

<div>
　　HyperX 表示：「HyperX 致力於設計和研發符合玩家需求且具絕佳品質的產品。Cloud Alpha 無線電競耳機擁有出色的性能、舒適度和產品品質，單次充電可以使用高達 300 小時領先業界的續航時間，非常適合需要優越的音質，同時又渴望長時間自在暢玩、不願被線材束縛的玩家。」</div>
<div>
 </div>
<div>
　　HyperX Cloud Alpha 無線電競耳機採用 2.4GHz 無線傳輸技術，無線範圍可達 20 公尺，在不犧牲音質的前提下，為玩家帶來自由無束縛的遊戲體驗。透過操作簡單的 HyperX NGENUITY 軟體可輕鬆確認剩餘電量、調整等化器 (EQ) 及其他自訂功能。此外，這款耳機採用耐用的鋁合金結構，搭配經典記憶泡棉及透氣人造皮革耳墊，為玩家帶來 HyperX 專屬的佩戴舒適體驗。</div>
