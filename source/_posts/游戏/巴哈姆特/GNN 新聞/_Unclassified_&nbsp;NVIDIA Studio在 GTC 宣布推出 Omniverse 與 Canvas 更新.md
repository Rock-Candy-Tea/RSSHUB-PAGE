<div>
<div>
<div>
　　NVIDIA 今日整理在今年 GTC 大會中的重點消息，希望未來幫助遊戲開發人員獲得創意的提升。</div>
<h3>
NVIDIA 重點消息回顧</h3>
<div>
 </div>
<div>
<b>專為創作者及遊戲開發人員提供的 Omniverse 更新</b></div>
<ul>
<li>
NVIDIA Omniverse 平台針對 Omniverse Create、Machinima 和 Showroom 等應用程式進行大規模更新，以及即將推出的 View。<br>
 </li>
</ul>
<div>
<div class="video-container">
<iframe allowfullscreen="allowfullscreen" frameborder="0" height="auto" class="lazyload" data-sizes="auto" src="https://www.youtube.com/embed/xWJtFXYlAPc?wmode=transparent" width="100%"></iframe></div>
</div>
<div>
 </div>
<ul>
<li>
NVIDIA Omniverse 的 Unreal Engine Connector 和 Adobe Substance 3D Material Extension 測試版現已推出，並擴大 Omniverse 商業生態系，而 Adobe Substance 3D Painter Connector 不久後也將推出測試版。</li>
</ul>
<ul>
<li>
藝術家現在可以在所有 Omniverse 應用程式的視窗中使用 Pixar HD Storm、Blender Cycles、Chaos V-Ray、Maxon <a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=Redshift" target="_blank">Redshift</a> 和 OTOY Octane 渲染器。</li>
<li>
專為開發人員打造的 Omniverse 提供一個創造虛擬世界使用的協作環境。</li>
</ul>
<div>
 </div>
<div>
<b>NVIDIA Canvas 更新</b></div>
<ul>
<li>
NVIDIA Canvas 對風格樣式進行了更新，為藝術家提供 10 種不同的預設濾鏡，可修改畫作的外觀與氛圍，並增強藝術家的創造力。</li>
</ul>
<div>
 </div>
<div>
<div class="video-container">
<iframe allowfullscreen="allowfullscreen" frameborder="0" height="auto" class="lazyload" data-sizes="auto" src="https://www.youtube.com/embed/wKztRskmsig?wmode=transparent" width="100%"></iframe></div>
</div>
<div>
 </div>
<div>
<b>新款 NVIDIA RTX 筆記型電腦 GPU</b></div>
<ul>
<li>
出門在外的專業人士可以選擇新款筆記型電腦 GPU，16GB 的記憶體容量可以處理要求作業：RTX A5500、RTX A4500、RTX A3000 12GB、RTX A2000 8GB 及 NVIDIA RTX A1000。</li>
</ul>

 </div>

<b>新款 Studio 筆記型電腦</b></div>
<ul>
<li>
三款全新 Studio 筆記型電腦即日起開始販售。ASUS ProArt Stud<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=iob" target="_blank">iob</a>ook 16 和 MSI 的 Creator Z16P 與 Z17 推出不同款的 GPU 供創作者選擇，以提升和擴展創意的可能性。</li>
</ul>

<div>
 </div>
<div>
<b>三月份 Studio 驅動程式現已推出</b></div>
<ul>
<li>
即日起開放下載的三月份 Studio 驅動程式，最佳化一系列最新的創意應用程式更新，包括新的 Omniverse 和 Canvas 升級。</li>
<li>
支援 Blender 3.1 更新項目，該更新增加 USD 預覽表面材質匯出功能，將更容易地在包括 Omniverse 等支援 USD 的應用程式間移動素材。</li>
</ul>
<div>
 </div>
<div>
　　NVIDIA 亦發起 #MadeInMachinima 競賽，邀請藝術家挑戰將經典的遊戲角色融入到 Omniverse Machinima 中的電影短片，將有機會贏得 NVIDIA Studio 筆記型電腦。3 月 29 日至 6 月 27 日開放上傳作品。</div>
