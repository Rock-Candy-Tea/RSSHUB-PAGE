<div>
<div>
　　CD PROJEKT RED 今日宣布，《<a class="acglink" href="http://acg.gamer.com.tw/search.php?encode=utf8&kw=%E5%B7%AB%E5%B8%AB" target="_blank">巫師</a>（The Witcher）》系列遊戲的下一款作品目前正在使用 Unreal Engine 5 進行開發，將為系列作的新傳奇以及與 Epic Games 的技術合作拉開帷幕。</div>
<div>
 </div>
<div>
　　今天的公告也是 CD PROJEKT RED 自推出《巫師》系列前一款 RPG《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E5%B7%AB%E5%B8%AB+3%EF%BC%9A%E7%8B%82%E7%8D%B5" target="_blank">巫師 3：狂獵</a>》以來，第一次正式確認了《巫師》系列最新作的存在。《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E5%B7%AB%E5%B8%AB+3%EF%BC%9A%E7%8B%82%E7%8D%B5" target="_blank">巫師 3：狂獵</a>》共獲得了 250 座年度遊戲獎項，後來又推出《石之心》和《血與酒》等資料片延續了作品週期。</div>
<div>
 </div>
<div>

</ul>
</div>
<div>
 </div>
<div>
　　新作的預告圖片中央放置了一個圖騰，伴隨著「新的傳奇即將開始」這句話。除了初步確認《巫師》系列的新作存在之外，並沒有提供其他進一步的細節，例如對應平台、開發時程表或上市日期等。</div>
<div>
 </div>
<div>
　　CD PROJEKT RED 指出，作為與 Epic Games 多年戰略合作的一部分，開發引擎將轉向 Unreal Engine 5。自 2011 年推出《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E5%B7%AB%E5%B8%AB+2%EF%BC%9A%E7%8E%8B%E5%9C%8B%E5%88%BA%E5%AE%A2" target="_blank">巫師 2：王國刺客</a>》以來，CD PROJEKT RED 一直使用自家的 REDengine 技術來開發遊戲。而這次與 Epic 的結盟關係不僅包括授權，還包括 Unreal Engine 5 的技術開發、以及 Unreal Engine 未來版本的支援等。CD PROJEKT RED 將與 Epic 公司的開發者合作，主要目的在於為開放世界體驗量身打造引擎 —— 從《巫師》系列的下一款遊戲開始。</div>
<div>
 </div>
<div>

</ul>
</div>
<div>
 </div>
<div>
　　CD PROJEKT RED 的技術長（CTO）Paweł Zawodny 談到 Unreal Engine 5 在這款遊戲以及未來遊戲中的應用時表示：「我們內部 RED 2.0 轉型的核心內容之一是更加注重技術，而我們與 Epic Games 的合作也是基於這一原則。從一開始，我們就沒有考慮傳統的授權方式；我們和 Epic 都認為這是一個長期的、令人滿意的技術合作關係。對 CD PROJEKT RED 來說，在最早的階段就決定下一款遊戲的技術方向是非常重要的，因為在過去我們花費了大量的資源和精力，在後續的每一款遊戲推出時不斷發展和調整 REDengine。這次的合作是如此令人興奮，因為這將提升開發的可預測性和效率，同時使我們能夠獲得最先進的遊戲開發工具。我已經迫不及待地想看到我們使用 Unreal Engine 5 打造的傑作了！」</div>
<div>
 </div>
<div>
　　Epic Games 的創辦人兼執行長 Tim Sweeney 談到了這次合作時也提到：「Epic 一直在建構 Unreal Engine 5 ，使團隊能夠以前所未有的規模和擬真度創造動態的開放世界。我們對有機會與 CD PROJEKT RED 合作、共同推進互動故事和遊戲的極限深感榮幸，這一努力將使開發者社群在未來幾年內受益。」</div>
<div>
 </div>
<div>
　　在宣布上述消息的同時，CD PROJEKT RED 也同步確認，作為《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E9%9B%BB%E9%A6%AD%E5%8F%9B%E5%AE%A2+2077" target="_blank">電馭叛客 2077</a>》基礎的 REDengine 技術仍然會被用於開發即將推出的《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E9%9B%BB%E9%A6%AD%E5%8F%9B%E5%AE%A2+2077" target="_blank">電馭叛客 2077</a>》DLC。</div>
