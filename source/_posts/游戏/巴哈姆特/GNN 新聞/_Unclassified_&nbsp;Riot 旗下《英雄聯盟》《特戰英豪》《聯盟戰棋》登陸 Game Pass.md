<div>
<div>
　　微軟於今（13）日展開的「Xbox & Bethesda Games Showcase」直播發表會中，宣布 Riot 旗下多款遊戲未來將登陸 Game Pass 的消息。</div>
<div>
 </div>
<div>
<ul class="bh-grids-img">
<li class="bh-grids-img-box" style="width: 99.87%;">

</ul>
</div>
<div>
 </div>
<div>
　　這次在直播發表會公開的消息中，宣布了 Riot Games 旗下多款遊戲將於今年冬季登陸 Game Pass 的消息，目前在首波公開的消息中《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E8%8B%B1%E9%9B%84%E8%81%AF%E7%9B%9F" target="_blank">英雄聯盟</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E8%8B%B1%E9%9B%84%E8%81%AF%E7%9B%9F%EF%BC%9A%E6%BF%80%E9%AC%A5%E5%B3%BD%E8%B0%B7" target="_blank">英雄聯盟：激鬥峽谷</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E8%8B%B1%E9%9B%84%E8%81%AF%E7%9B%9F%EF%BC%9A%E8%81%AF%E7%9B%9F%E6%88%B0%E6%A3%8B" target="_blank">英雄聯盟：聯盟戰棋</a>》、《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E7%89%B9%E6%88%B0%E8%8B%B1%E8%B1%AA" target="_blank">特戰英豪</a>》，以及《<a class="acglink" href="https://acg.gamer.com.tw/search.php?encode=utf8&kw=%E7%AC%A6%E6%96%87%E5%A4%A7%E5%9C%B0%E5%82%B3%E8%AA%AA" target="_blank">符文大地傳說</a>》都將與 Game Pass 結合，而配合微軟的 Game Pass，以上遊戲包含英雄、牌組、探員在內的種種要素都將直接解鎖。</div>
<div>
 </div>
<div>
　　未來官方將進一步公開 Riot Games 旗下遊戲登陸 Game Pass 的詳細資訊，對於這次兩邊合作有興趣的人，可關注後續相關報導。</div>
