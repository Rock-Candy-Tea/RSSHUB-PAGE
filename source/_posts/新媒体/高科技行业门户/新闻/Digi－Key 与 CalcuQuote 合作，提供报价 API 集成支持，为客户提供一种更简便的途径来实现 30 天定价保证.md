<p style="text-indent: 2em; text-align: left;">美国 ， 明尼苏达 ， 锡夫里弗福尔斯市 － 2021 年 12 月 08 日</p><p style="text-indent: 2em; text-align: left;">全球供应品类极丰富、发货极快速的现货<span class="hrefStyle"><a href="https://ee.ofweek.com/tag-%E7%94%B5%E5%AD%90%E5%85%83%E5%99%A8%E4%BB%B6.HTM" target="_blank" title="电子元器件">电子元器件</a></span>分销商 Digi－Key Electronics 与 CalcuQuote 达成合作，实现了 Digi－Key 报价 API 与该公司技术的集成，为客户提供了一个更简便的入口，以很少的开发投入来连接 Digi－Key 领先的 API。</p><p style="text-align:center"><img src="https://cors.zfour.workers.dev/?http://images.ofweek.com/Upload/News/2022-01/07/Sunny/1641538217503023950.png" title="1641538217503023950.png" alt="image.png" referrerpolicy="no-referrer"></p><p style="text-indent: 2em; text-align: center;">Digi－Key 已将 CalcuQuote 技术集成到其面向客户的报价 API 上</p><p style="text-indent: 2em; text-align: left;">与标准“价格和可供货数量”功能选择相比，Digi－Key 的报价 API 可实现 30 天的定价保证，并通过更高效的报价和按需获取的实时数据提供更顺畅的订购服务。通过与 CalcuQuote 的这种合作关系，Digi－Key 客户能够确保所报价格的安全，且无需进行大量投资软件开发即可利用上该技术。</p><p style="text-indent: 2em; text-align: left;">Digi－Key 数字技术办公室 － B2B 经理 Nathan Pray 表示：“我们很高兴看到我们的报价 API 能够与 CalcuQuote 技术集成在一起，为客户提供一个经济高效、快速的与 Digi－Key API 解决方案连接的选择。这种集成能力可以让全球采购专业人士轻松体验更顺畅、更高效的订购服务，同时确保了 30 天的定价安全。”</p><p style="text-indent: 2em; text-align: left;">CalcuQuote 产品副总裁 David Sharp 指出：“通过这项开创性的升级，客户和 Digi－Key 代表可以访问相同的信息，确保价格在 30 天内有效。即便市场动荡，我们的客户仍可对产品和定价信心满满，而 CalcuQuote 第一个让这一切成为可能。”</p><p style="text-indent: 2em; text-align: left;">如需了解有关 Digi－Key API 解决方案的更多信息，请访问 API 网站。您可以在该网站上申请免费电子书，详细了解 Digi－Key API 工具所能带来的好处。</p><p style="text-indent: 2em; text-align: left;"><strong>关于 CalcuQuote</strong></p><p style="text-indent: 2em; text-align: left;">CalcuQuote 是一家为 EMS 行业提供报价和供应链软件的厂商。公司着眼于未来，通过优化运营和实施可持续的数字解决方案来改善报价和供应链流程的速度、准确性和效率。CalcuQuote 创立于 2014 年，目前为 24 个国家的 200 多家 EMS 公司提供服务。</p><p style="text-indent: 2em; text-align: left;"><strong>关于 Digi－Key Electronics</strong></p><p style="text-indent: 2em; text-align: left;">Digi－Key Electronics 是一家全球性的电子<span class="hrefStyle"><a href="https://ee.ofweek.com/tag-%E5%85%83%E5%99%A8%E4%BB%B6.HTM" target="_blank" title="元器件">元器件</a></span>综合服务授权分销商，总部设在美国明尼苏达州锡夫里弗福尔斯市，用户可通过其线上 Marketplace 查询大量的相邻产品和技术。我司经销着来自 2200 多家优质品牌制造商的 1140 多万种产品。此外，Digi－Key 的线上 Marketplace 为与技术创新相关的所有物品（物联网、工业自动化、测试、测量等）提供与众不同的购物体验。</p>
