margin-bottom: 10px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); white-space: normal; text-indent: 2em; text-align: left;">在此背景下，以AI、计算机视觉、物联网等为代表的数字化创新技术正在成为零售商的核心竞争力之一，有助于帮助零售商降低运营成本、提升运营效率，为用户带来独特的购物体验。越来越多的零售商正在期望，将AI技术作为提升业务的关键要素，以优化洞察消费者偏好、优化门店运营等工作，进而实现收益的跃升。</p><center><p style="margin-top: 0px; margin-bottom: 10px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); white-space: normal; text-indent: 2em; text-align: left;"><strong>汉朔携手英特尔、微软推出零售AI方案</strong></p><p style="margin-top: 10px; margin-bottom: 10px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); white-space: normal; text-indent: 2em; text-align: left;">为应对上述挑战，汉朔携手英特尔、微软推出针对零售行业的AI方案，主要应用在智慧货架管理和自助收银防损两个方面。通过AI相机和机器人等不同设备，在门店可进行价签检测、商品识别、缺货核检等功能，整体方案对陈列和缺货的检测精度高于95％；同时，方案中的边缘设备可完善解决敏感数据保护的问题，相关数据还能够传输至后端系统进行处理，支持进一步的数据处理以及业务决策。</p><p style="margin-top: 10px; margin-bottom: 10px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); white-space: normal; text-indent: 2em; text-align: left;">据了解，汉朔AI方案搭载英特尔酷睿处理器的边缘服务器／英特尔视频AI计算盒，可充分满足AI模型推理等负载在性能方面的严苛要求。该处理器同时集成了英特尔锐炬Xe显卡，即使面对复杂的AI计算工作，也能够流畅高效完成，解决了部分零售商渴望引入AI方案，但却对强大算力支持和高昂部署成本望而却步的顾虑。</p><p style="margin-top: 10px; margin-bottom: 10px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); white-space: normal; text-indent: 2em; text-align: left;">英特尔中国区物联网及渠道数据中心事业部总经理郭威表示，“每一次零售业的变革都离不开技术力量的驱动，AI、虚拟化、物联网、边缘计算等数字化技术实现了由业务创新驱动的消费者个性化体验升级和零售业务重塑，使零售业发生了很大的变革。英特尔将助力构建健康、活跃的全球智慧零售生态，解决智慧零售发展中至关重要的计算，存储，网络，算法、数据等问题，驱动零售领域更多的创新，解锁未来全新机遇。”</p><p style="margin-top: 10px;
