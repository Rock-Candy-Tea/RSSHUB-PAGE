<p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">“只要技术到位，无线不仅能做到媲美总线，甚至更胜一筹。”技术出身的罗勇凯，一直坚信这点。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">过去，在世界500强的首席软件工程师经历，培育出他内心的产品梦，“旧时王谢堂前燕，飞入寻常百姓家”，小燕的名称正源于此，用极致的技术打造理想的产品，向更多人传递美好生活体验。</p><p style="text-align: center"><img src="https://cors.zfour.workers.dev/?http://images.ofweek.com/Upload/News/2021-10/19/yuanchen/1634611489871035210.png" title="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" alt="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">从小燕科技成立开始，短短几年间便在行业扎稳根基，布局完善生态的智能系统，不得不说也正来自于公司整体技术方面的优势。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">2021年10月13日，小燕科技新品发布会在温州全屋智能旗舰店盛大举办。现场，小燕科技创始人罗勇凯与媒体分享和交流了他对于行业和技术的看法，以下为采访全文。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;"><strong style="margin: 0px; padding: 0px;">媒体：小燕科技的品牌slogan叫“改变一点点 就改变了很多”，想表达的主要是什么？</strong></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">罗：很多人觉得，真正的<span class="hrefStyle"><a href="https://smarthome.ofweek.com/" target="_blank" title="智能家居">智能家居</a></span>离广大群众还很遥远，需要部署完善的智能系统，消耗巨大的成本，是有钱人才玩得起的游戏。我认为不是的，从2016年开始，陆陆续续推出的智能墙壁开关、智能窗帘、智能门锁产品，其实对普通用户而言，都是触手可及的，从一支产品切入，既能满足眼前最实际迫切的需求，同时也会逐步教育用户，让他们切身感受到智能在家居空间中的便利，而且他们会发现，门槛并没有想象得那么高，全屋智能行业从技术到产品到服务已经相对成熟。我们slogan想要表达的是，作为产品和系统开发者，我们先完成一大步，这样用户只需要改变一点点，从这“一点点”入手，真正体验过后，用户就会感受到智能的美好。</p><p style="text-align: center"><img src="https://cors.zfour.workers.dev/?http://images.ofweek.com/Upload/News/2021-10/19/yuanchen/1634611496363085795.png" title="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" alt="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;"><strong style="margin: 0px; padding: 0px;">媒体：精卫智能调光驱动是一款特别的调光产品，主要体现在它的调光深度和精确兼容上，调光深度可以看出小燕强大的技术优势，精确兼容意味着它具有广阔的市场前景，这方面有什么值得分享的？</strong></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">罗：调光深度是我们一直在做的事，百万级只是目前的水平，难保以后不会继续提升。精确兼容的特性其实是希望精卫不止可以让传统LED灯具实现智能调光，更可以充分发挥LED灯具的潜能，调出“绝世好光”。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">跟传统的硬件产品研发方式不同，小燕是以做纯软件的思维方式做硬件。我们首先实现了一个硬件模拟器，然后在模拟器上实现了具有百万级以上深度的调光算法，最后再在真实硬件上做软硬件联调，这样不但大大提高了研发速度，并且我们最终将电流精确控制在50纳安级别，从而达到百万深度的调光效果。在额定电流的匹配上我们也做了特殊设计，最终可以实现毫安级的电流匹配，消除了传统拨码开关存在50毫安左右的额定电流误差，这样可以更精准匹配市面上各种各样的LED灯具。</p><p style="text-align: center"><img src="https://cors.zfour.workers.dev/?http://images.ofweek.com/Upload/News/2021-10/19/yuanchen/1634611508220002650.png" title="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" alt="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">之所以把调光深度做得这么高，其实是为了最大限度模拟自然光。自然光可以说是一种调光深度无限大的光，经过数百万年的进化，人类早已适应了自然光，而自然光的渐变是非常平滑、细腻、柔和的，没有灯光瞬间切换的突然感。目前市面上很多调光设备，虽然号称无极调光，但是光线开启和熄灭的那两个阶段依然存在那种突然感，灯光一下子就亮，一下子就熄，说到底还是体验没到位。小燕的做法是通过精准匹配电流，找到灯光将亮未亮、趋近于0的那一点，实现柔和缓慢地渐变，这看起来是极其细微的东西，却像我们的slogan说的那样，“改变一点点，就改变了很多”，如果要让用户充分感觉舒适，调光深度一定要去突破。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;"><strong style="margin: 0px; padding: 0px;">媒体：除了新推出的精卫调光驱动带来的百万级深度调光，小燕产品在调光技术上还有其它优势吗？</strong></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">罗：同样还是软件算法带来的优势，或许看一盏灯的调光效果还不是那么明显，但是看一整个场景就会很明显。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">如果打开我们的小燕在家App界面，开启超级同步选项，便可以实现很多灯光的同步亮灭。这项技术的难点在于，不止是很多灯一起开、一起关，连灯光渐变的节奏都能保持高度一致。就像国庆大阅兵的方阵，想要做到踢正步整齐划一，只能靠反复练习，而小燕的超级同步技术，也是需要在自有软件算法中狠下功夫、反复优化才得以实现的。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">目前，我们旗下的智能灯光产品都可以支持超级同步。</p><p style="text-align: center"><img src="https://cors.zfour.workers.dev/?http://images.ofweek.com/Upload/News/2021-10/19/yuanchen/1634611515264001277.png" title="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" alt="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;"><strong style="margin: 0px; padding: 0px;">媒体：这次发布会的地点，温州瑞安小燕旗舰店，听说从筹备到开业花了整整半年时间，在疫情仍然存在的情况下，开这家店的意义是什么？</strong></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">罗：在开关窗帘时代，场景的重要性还没有凸显，因为它们都是相对单一的功能。等到了<span class="hrefStyle"><a href="https://smarthome.ofweek.com/CAT-91102-smartlighting.html" target="_blank" title="智能照明">智能照明</a></span>时代，光是一个布满整个场景的氛围式的存在，场景的展示作用就变得相当重要。光好不好，直接感受才知道。而光的美学感受和照明功能都与实地场景不可分割，我们不能单独把照明设计从家居设计中拎出来，所以一个与光完美适配的体验场景就很重要。客户看到我们的灯可以做成这样的效果，更容易被打动，比如这家店，还没开业已经提前销售了逾千支产品。</p><p style="text-align: center"><img src="https://cors.zfour.workers.dev/?http://images.ofweek.com/Upload/News/2021-10/19/yuanchen/1634611528605038524.png" title="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" alt="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: center;">小燕温州旗舰店－影音室场景图</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;"><strong style="margin: 0px; padding: 0px;">媒体：现在是全屋智能行业高速发展的时代，无主灯的流行让智能调光成为刚需，而无主照明使用的灯具动辄几十乃至上百盏，对无线智能系统的稳定性和传输效率提出了很高要求。这方面小燕会做哪些事吗？或者说已经在做布局？</strong></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">罗：你提到了目前很关键的一个需求——无线系统的稳定性。稳定是相对的，当接入设备不多、覆盖面积不广时，大家都可以做到比较稳定。但是当项目面积扩大、项目需求复杂化后，就能看出实力的差异。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">过于，小燕在单网关传输效率上反复优化算法，可以实现自动计算最佳路径，选择最空闲通畅的路线快速传输，那时候一个网关接入100＋设备不在话下。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">但是随着接入设备数量的增加，我们也发现，上升空间越来越小，设备传输效率没那么理想了。一般来说这时候就要选择放弃，标注最大接入设备数量就好，而那种接入几百甚至上千设备的全屋智能方案，就只能交给总线去完成。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">但是“认输”可不是我们的风格，就像百万级深度调光一样，这也是前人没有想过、做过的事，我们认为有必要做，所以就去做了。同理，设备接入数量的限制有必要去突破，传输效率和稳定性也必须继续去做优化，所以不管再难的技术，也要去攻克。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">本地多网关融合技术是我们捧出的答卷。百万级深度调光、超级同步、本地多网关融合，这是我们目前的三大技术优势。用户平时在App界面上只需要看到一个网关就行了，但实际上它可以是多个不同网关的融合，接入到不同网关里的设备之间的传输效率不亚于同一个网关里的设备，甚至可以说比之前的速度更快。</p><p style="text-align: center"><img src="https://cors.zfour.workers.dev/?http://images.ofweek.com/Upload/News/2021-10/19/yuanchen/1634611538273059524.png" title="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" alt="专访小燕科技CEO罗勇凯：三大技术优势，赋能行业差异化竞争" referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">这样接入设备的数量大幅度提升，以前只能接入100＋设备的，现在可以接入1000～2000个，覆盖面积可达5000平，我们的经销商在面对大宅别墅时也能够稳定布局无线智能系统，做大项目更自信，更能体现出专业度。</p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); text-indent: 2em; text-align: left;">个人认为，做<span class="hrefStyle"><a href="https://smarthome.ofweek.com/tag-%E6%99%BA%E8%83%BD%E5%AE%B6%E5%B1%85%E6%8E%A7%E5%88%B6%E7%B3%BB%E7%BB%9F.HTM" target="_blank" title="智能家居系统">智能家居系统</a></span>，稳定的系统是基础，向上的扩容是必须，这一定是未来发展的趋势。</p>
