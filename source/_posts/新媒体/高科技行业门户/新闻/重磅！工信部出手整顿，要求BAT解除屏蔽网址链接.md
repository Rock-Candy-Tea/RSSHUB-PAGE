white-space: normal; background-color: rgb(255, 255, 255); text-indent: 2em; text-align: left;">据媒体报道，9月9日下午，工信部信管局有关业务部门召开了“屏蔽网址链接问题行政指导会”。会上，工信部提出有关即时通信软件的三点合规标准，要求限期内各平台必须按标准解除屏蔽，否则将依法采取处置措施。</p><p style="text-align: center;"><img src="https://cors.zfour.workers.dev/?http://images.ofweek.com/Upload/News/2021-9/13/liteng/20210913113943_1631504383470655538.jpg" width="600" referrerpolicy="no-referrer"></p><p style="text-align: center;"><span style="font-size: 14px;">图源pixabay</span></p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); white-space: normal; background-color: rgb(255, 255, 255); text-indent: 2em; text-align: left;">三条合规标准分别是：</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); white-space: normal; background-color: rgb(255, 255, 255); text-indent: 2em; text-align: left;">（1）具有外部网址链接访问功能的即时通讯软件，对于用户分享的同种类型产品或服务的网址链接，展示和访问形式应保持一致；</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); white-space: normal; background-color: rgb(255, 255, 255); text-indent: 2em; text-align: left;">（2）具有外部网址链接访问功能的即时通讯软件，用户在即时通讯中发送和接收合法网址链接，点击链接后，在应用内以页面的形式直接打开；</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); white-space: normal; background-color: rgb(255, 255, 255); text-indent: 2em; text-align: left;">（3）不能对特定的产品或服务网址链接附加额外的操作步骤，不能要求用户手动复制链接后转至系统浏览器打开。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); white-space: normal; background-color: rgb(255, 255, 255); text-indent: 2em; text-align: left;">对于以上三条标准，若企业在9月17日前仍未整改完成，工信部将直接启动处置措施，不再约谈。处置措施包括：向社会公众曝光问题企业，通过媒体渠道点名批评；将企业纳入重点监管范畴，重点关注并开展现场检查，对发现的其他问题从严从重处罚；若企业拒不整改，将依法依规下架其APP并启动相关行政处罚，并将其企业纳入信息通信行业信用管理体系内，根据规定限制其开展互联网相关业务。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); white-space: normal; background-color: rgb(255, 255, 255); text-indent: 2em; text-align: left;">当天参会的企业包括<span class="hrefStyle"><a href="https://iot.ofweek.com/tag-%E9%98%BF%E9%87%8C%E5%B7%B4%E5%B7%B4.HTM" target="_blank" title="阿里巴巴">阿里巴巴</a></span>、腾讯、字节跳动、百度、华为、小米、陌陌、360、网易等。与会企业都明确表态，严格按标准要求落实。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); white-space: normal; background-color: rgb(255, 255, 255); text-indent: 2em; text-align: left;">业内人士称，工信部提出的相关合规标准，有助于维护用户合法权益，营造开放包容、公平竞争的生态环境，夯实互联网行业长远发展基础。</p>
