<p><strong>Intel官方泄露13代酷睿700系主板</strong></p>
<p>Intel将在今年下半年发布Raptor Lake 13代酷睿处理器，虽然还是LGA1700封装接口，但同时会有新的700系列主板芯片组。意外的是，Intel在一份官方文件中泄露了700系列芯片组的规格，而且至今没有撤掉。</p>
<p>这份文件其实是关于600系列芯片组规格说明的，并没有直接列出700系列的名字，但部分参数却有两种数值，明显是把下一代提前列上了。整体不出所料，700系列的变化非常小，主要是PCIe总线、USB接口的调整。</p>
<p>Z790 PCIe 4.0总线从12条增加到20条，PCIe 3.0则从16条减半到8条，可以支持更多、更高速的SSD和扩展外设。另外，Z790 USB 3.2 Gen2x2 20Gbps从4个增加到5个，也是全系列唯一调整USB接口的。H770 PCIe 4.0、PCIe 3.0分别为16条、8条，对比现在H670分别增加4条、减少4条。B760 PCIe 4.0 6条增至10条，PCIe 3.0 8条减半到4条。至于入门级的H710，和B610没有任何不同，当然也有可能不会出现H710，而是继续使用H610。其他，暂时看不到不同。</p>
<p>兼容性方面，700系列能上12代酷睿肯定是毫无技术问题的，但是600系列能否升级13代酷睿，就看Intel是否会“按照惯例”，在供电等方面设置障碍了。</p>

<p><strong>AMD锐龙7 5800X3D上市时间泄露</strong></p>
<p>今年初AMD在CES展会上发布了一款特殊的锐龙5000处理器，那就是锐龙7 5800X3D，后缀3D代表着3D V-Cache，多了额外的64MB缓存，总计96MB的缓存让它的游戏性能提升多达40%。不过40%的性能提升是个别游戏，平均下来游戏性能提升15%，毕竟频率也从锐龙7 5800X的3.8-4.7GHz降低到了3.4GHz~4.5GHz，但综合下来也足以让它冲击12代酷睿i9-12900K，夺回最佳游戏处理器的称号。</p>
<p>对于锐龙7 5800X3D处理器，最大的悬念就是上市时间和价格了，等了这么久，现在终于有一点明确的消息——锐龙7 5800X3D已经出货，可能会在3月底上市。爆料者Greymon55是知名的小道消息之王，很多爆料都很靠谱，而且这次他说消息来自封装厂，上市时间的消息应该还是比较准的。</p>
<p>3月底上市也符合AMD之前的表态，说是春季上市，反正再拖下去这颗处理器就没多大意义了，毕竟5nm Zen4传说提前到5月~6月份发布，9月份之前就要上市。</p>
<p>当然，对锐龙7 5800X3D处理器处理器来说还有一个关键问题，那就是售价，它的游戏性能对标i9-12900K，后者售价在国内依然要4999元，而锐龙7 5800X现在要2199元，锐龙7 5800X3D处理器应该会在3000元左右，对游戏玩家来说应该还是挺有吸引力的。</p>

<p><strong>DP 2.0将引入新认证 拒绝标准混乱</strong></p>
<p>前不久，HDMI官方令人费解的取消了HDMI2.0标准，将其划分为HDMI2.1标准的子集。此举引起了外界的广泛质疑，这无疑会造成HDMI2.1标准混乱的情况。为了避免出现和HDMI2.1一样的争议，视频电子标准协会 (VESA) 表示，为了确保客户能够分辨线缆或设备是否支持该标准，将采用新的认证计划。</p>
<p>据介绍，DisplayPort 2.0支持DisplayPort UHBR（超高比特率）数据传输，为了分清规格，VESA推出了新的DP40和DP80 UHBR线缆。其中，带有DP40标记的线缆将支持 UHBR10 标准，总带宽40Gbps；带有DP80标记的线缆将支持UHBR20 标准，总带宽80Gbps。</p>
<p>VESA表示，支持DP40和DP80标准的全尺寸DisplayPort和Mini DisplayPort的线缆已经投入生产，首批显示器和其他设备已经通过认证测试，预计很快将和消费者见面。</p>

<p><strong>惠普34英寸5K带鱼屏一体机开售</strong></p>
<p>昨天，惠普推出了新款ENVY 34带鱼屏一体机，现已在惠普官方天猫店上架，原价29499元，首发价28999元。</p>
<p>惠普ENVY 34带鱼屏一体机配备了一块34英寸21:9带鱼屏，5120×2160分辨率，98% DCI-P3色域，屏占比高达92.7%。配置方面，该一体机搭载了11代酷睿桌面处理器i9-11900，8核16线程，2.5GHz基本频率，至高可睿频至5.2GHz。它还配备RTX 3080显卡，8GB显存，另外配备32GB DDR4高频内存和2TB SSD。这款一体机配备了1600万像素磁吸摄像头，可以随意吸附在屏幕四周任何位置。底座支持无线充电，QI协议，最高15W</p>
<p>接口方面，提供了两个雷电4、HDMI、有线网口、读卡器以及USB-C和USB-A接口。它还搭载惠小微智能语音助手，可实现中文语音命令识别，通过语音交互提升工作效率，智能记录常用程序，应用启动方便快捷。轻唤小微，也能快速获取音乐、天气、新闻资讯等内容和服务。</p>

<p><strong>联想发布入门级翻转本Flex 5i</strong></p>
<p>日前，联想发布了新款翻转本Flex 5i Chromebook，配备14英寸全高清触控屏，360度铰链，可以在平板电脑、帐篷或站立三种模式下使用。</p>
<p>性能方面，联想Flex 5i Chromebook最高搭载英特尔酷睿i5-1235U Alder Lake处理器、8GB LPDDR4X内存和 512GB SSD。此外，它还提供Pentium 8505和Core i3-1215U处理器选项，低至4GB内存，以及64GB或128GB的eMMC存储。</p>
<p>不管什么处理器，该翻转本均配备1080P屏幕，支持WiFi 6E和蓝牙5.0，立体声2W扬声器、背光键盘、指纹读取器。提供两个Type-C接口还有USB接口、HDMI接口、3.5mm耳机接口以及Micro SD接口。</p>
<p>需要注意的是，它运行的并不是Window系统，而是Chrome OS。虽然Chromebook长期以来一直以具有入门级规格的廉价笔记本电脑而著称，但Chrome OS 本身作为操作系统已经变得更加强大。Flex 5i Chromebook预计会在6月上市，起售价在500美元（约合人民币3155元）左右。</p>
<p>联想还有更便宜的Chromebook Flex 3i将于5月上市，售价400美元（约合人民币2525元）以上，但该型号配备15.6英寸显示屏和英特尔Jasper Lake处理器。</p>

<p><strong>Redmi Note 11E上架</strong></p>
<p>3月2日消息，从小米官方商城获悉，日前，Redmi Note 11系列新成员——Redmi Note 11E上架小米商城，提供4GB+128GB、6GB+128GB两款配置，售价分别为1199元、1299元，将于3月18日上午10:00开售。</p>
<p>与Redmi Note 11相比，Redmi Note 11E最大变化之一就是将处理器换成了联发科天玑700（Redmi Note 11为天玑810）。据悉，天玑700采用7nm工艺制程，八核CPU架构，包括两个大核A76、六个小核A55，最高频率分别为2.2GHz、2.0GHz，GPU为Mali-G57，采用LPDDR4x内存+UFS 2.2机身存储。</p>
<p>屏幕方面，Redmi Note 11E配备一块6.58英寸FHD+水滴屏，支持90Hz刷新，240Hz触控采样率，对比度为1500:1。核心配置上，新机内置5000mAh大电池，支持18W快充，后置5000万像素超清主摄+200万人像景深镜头，支持超级夜景、电影模式、HDR等，前置为500万像素美颜镜头。</p>
<p>其他方面，Redmi Note 11E拥有神秘黑境、微醺薄荷、冰晶银河三款配色，预装MIUI 13，支持3.5mm耳机孔、红外遥控、Type-C接口。</p>

<p><strong>腾讯“微信客服”商标被驳回</strong></p>
<p>3月1日消息，企查查APP显示，“微信客服”的四个商标都被驳回复审。而在2021 年 7 月，腾讯科技（深圳）有限公司申请“微信客服”商标，国际分类为设计研究、社会法律、通讯服务、广告销售。</p>
<p>看到“商标”被驳回，网友们调侃道：微信“没有”客服的原因找到了。据悉，去年7月份，微信“微信客服”功能内测，不过这并非是普通消费者所想的那个客服，其定位和公众号、视频号一样，拥有独立的业务属性。</p>
<p>据官方介绍，用户可以在微信内、外各个场景中接入微信客服，用户可以发起咨询，企业可通过API接口回复消息，做好客户服务，可知微信客服功能目前是面向企业级用户推出的服务功能。</p>
<p>至于微信用户，在遇到问题时联系微信客服，则可以通过微信人工客服电话联系，但是不少网友反馈，微信客服电话，人工接通率不高，大部分时间是在和机器人无效对话。</p>
