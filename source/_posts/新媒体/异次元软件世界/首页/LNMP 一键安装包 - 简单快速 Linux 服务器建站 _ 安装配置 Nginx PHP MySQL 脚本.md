target="_blank" rel="noopener">Vultr</a>、<a href="https://www.iplaysoft.com/go/bandwagon" target="_blank" rel="noopener">搬瓦工</a>、<a href="https://www.iplaysoft.com/go/linode" target="_blank" rel="noopener">腾讯云</a>、<a href="https://www.iplaysoft.com/go/aliyun" target="_blank" rel="noopener">阿里云</a>、<a href="https://www.iplaysoft.com/go/ucloud" target="_blank" rel="noopener">Ucloud</a> 等众多 <a href="https://www.iplaysoft.com/go/vps" target="_blank" rel="noopener">VPS 服务器</a>上测试通过。</p><h4>1、使用 SSH 客户端工具连接 VPS 服务器。</h4><p>比较常用的 SSH 客户端工具有 <a href="https://www.iplaysoft.com/p/putty" target="_blank" rel="noopener">Putty</a> (Windows) 或 <a href="https://iterm2.com/?utm_source=iplaysoft.com&hmsr=iplaysoft.com" target="_blank" rel="noopener">iTerms2</a> (Mac) 等。</p><h4>2、安装 <code>wget</code> 和 <code>screen</code> 命令</h4><p>由于脚本中使用到这两个命令，而部分系统并未默认安装它们，所以需要先安装一下。</p><p><strong>CentOS / RHEL 等系统执行：</strong></p><link rel="stylesheet" type="text/css" href="https://cdn.iplaysoft.com/common/lib/highlight/styles/atom-one-dark.css"> <pre class="language-bash"><code>sudo yum install -y screen wget</code></pre><p><strong>Ubuntu / Debian / RaspberryOS 等系统执行：</strong></p><pre class="language-bash"><code>sudo apt-get -y install screen wget</code></pre><p><strong>然后执行 screen 命令，防止半途掉线：</strong></p><pre class="language-bash"><code># 使用 screen 命令的目的是可以在掉线后恢复安装过程
screen -S lnmp</code></pre><h4>3、一键安装 LNMP 稳定版：</h4><pre class="language-bash"><code>wget http://soft.vpser.net/lnmp/lnmp1.8.tar.gz -cO lnmp1.8.tar.gz && tar zxf lnmp1.8.tar.gz && cd lnmp1.8 && ./install.sh lnmp</code></pre><p>运行上面的 LNMP 安装命令后，会出现菜单让你选择安装 MySQL 或 MariaDB 的版本，也可以选择不安装数据库。安装数据库或 PHP 7 版本，都需要主机有 1G 或以上的内存。</p><p><img class="lazyload aligncenter size-medium" src="https://img.iplaysoft.com/wp-content/uploads/2021/lnmp/mysql_select.png" alt="安装 MYSQL 数据库" width="382" height="186" referrerpolicy="no-referrer"></p><p>选择数据库及版本之后就要设置数据库的 root 密码了，如下图所示，这里最好用 <a href="https://www.iplaysoft.com/1password.html" target="_blank" rel="noopener">1Password</a> 或记事本给记录下来以后备用。接着脚本会询问你是否需要启用 MySQL InnoDB 引擎，此项默认为开启，一般也是建议开启，直接回车或输入 <code>y</code> 即可</p><p><img class="lazyload aligncenter size-medium" src="https://img.iplaysoft.com/wp-content/uploads/2021/lnmp/php_select.png" alt="安装 PHP" width="342" height="158" referrerpolicy="no-referrer"></p><p>然后就可以选择安装 PHP 的版本了。如果你打算运行的程序比较新更新比较勤，那么都建议安装 PHP 7+ 的版本，性能会好很多。但如果你要跑一些版本老旧的 PHP 代码，为了兼容性，你可以选择安装 PHP 5.6。</p><p><img class="lazyload aligncenter size-medium" src="https://img.iplaysoft.com/wp-content/uploads/2021/lnmp/php_memory.png" alt="PHP 内存" width="376" height="88" referrerpolicy="no-referrer"></p><p>继续会提示问你是否要安装内存优化 Jemalloc 或 TCmalloc，默认不安装。没特殊需要，直接回车跳过即可。</p><p>之后脚本就会提示 "Press any key to install... or Press Ctrl+c to cancel" ，这时按回车键确认即可开始安装过程。由于<span style="color: #ff6600;">安装过程涉及到网络速度和机器性能影响，几分钟到几十分钟都有可能</span>，耐心等待安装完成即可。</p><p>如果遇到自己的电脑网络掉线，可以重新连接 SSH，再执行 <code>screen -r lnmp</code> 可以查看到你当前的 lnmp 安装进程，可以执行：<code>screen -D -r lnmp</code> 进行恢复。</p><p><img class="lazyload aligncenter size-medium" src="https://img.iplaysoft.com/wp-content/uploads/2021/lnmp/install_success.png" alt="安装 LNMP" width="536" height="603" referrerpolicy="no-referrer"></p><p>安装成功后，可以看到结果上有类似这样的字眼：</p><blockquote><p>Nginx: OK<br>
MySQL: OK<br>
PHP: OK<br>
PHP-FPM: OK</p></blockquote><p>接下来你的主机就可以开始使用了！你可以使用 <code>lnmp vhost add</code> 命令来添加“<a href="https://www.iplaysoft.com/tag/%E7%BD%91%E7%AB%99" target="_blank" rel="noopener">网站</a>”，然后根据后面提供的目录路径，将你的网页程序代码上传到服务器即可，<a href="https://lnmp.org/faq/lnmp-vhost-add-howto.html?utm_source=iplaysoft.com&hmsr=iplaysoft.com" target="_blank" rel="noopener">具体可参考添加虚拟主机/网站方法</a>。最后，修改你的<a href="https://www.iplaysoft.com/go/domain" target="_blank" rel="noopener">域名</a>解析指向即可。</p><h3>LNMP 命令一览：</h3><ul><li>LNMP 状态管理：lnmp &#123;start|stop|reload|restart|kill|status&#125;</li><li>LNMP 各个程序的状态管理：lnmp &#123;nginx|mysql|mariadb|php-fpm|pureftpd&#125; &#123;start|stop|reload|restart|kill|status&#125;</li><li>虚拟主机 (网站) 管理：lnmp vhost &#123;add|list|del&#125;</li><li>数据库管理：lnmp database &#123;add|list|edit|del&#125;</li><li>FTP 用户管理：lnmp ftp &#123;add|list|edit|del|show&#125;</li><li>给已存在的虚拟主机(网站)添加 <a href="https://www.iplaysoft.com/go/ssl" target="_blank" rel="noopener">SSL 证书</a>：lnmp ssl add</li><li>通过 DNS API 方式生成证书并创建虚拟主机：lnmp dns &#123;cx|dp|ali|...&#125;</li><li>只通过 DNS API 方式生成 SSL 证书：lnmp onlyssl &#123;cx|dp|ali|...&#125;</li></ul><h3>LNMP 一键安装包的默认安装目录路径</h3><ul><li>Nginx 目录: /usr/local/nginx/</li><li>MySQL 目录 : /usr/local/mysql/</li><li>MySQL 数据库所在目录：/usr/local/mysql/var/</li><li>MariaDB 目录 : /usr/local/mariadb/</li><li>MariaDB数据库所在目录：/usr/local/mariadb/var/</li><li>PHP目录 : /usr/local/php/</li><li>多 PHP 版本目录 : /usr/local/php5.5/ 其他版本前面5.5的版本号换成其他即可</li><li>PHPMyAdmin 目录 : /home/wwwroot/default/phpmyadmin/ 强烈建议手工将此目录改名为不易猜到的名字，比如 wo-de-php-guan-li。phpmyadmin 可自己从官网下载新版替换，以防老版本爆出漏洞。</li><li>默认网站根目录 : /home/wwwroot/default/</li><li>Nginx 日志目录：/home/wwwlogs/</li><li>/root/vhost.sh 添加的虚拟主机配置文件所在目录：/usr/local/nginx/conf/vhost/</li><li>PureFtpd 目录：/usr/local/pureftpd/</li><li>PureFtpd web 管理目录： /home/wwwroot/default/ftp/</li><li>Proftpd 目录：/usr/local/proftpd/</li><li>Redis 目录：/usr/local/redis/</li></ul><h3>LNMP 配置文件位置路径</h3><ul><li>Nginx 主配置 (默认虚拟主机) 文件：/usr/local/nginx/conf/nginx.conf</li><li>添加的虚拟主机配置文件：/usr/local/nginx/conf/vhost/域名.conf</li><li>MySQL 配置文件：/etc/my.cnf</li><li>PHP 配置文件：/usr/local/php/etc/php.ini</li><li>php-fpm 配置文件：/usr/local/php/etc/php-fpm.conf</li><li>PureFtpd 配置文件：/usr/local/pureftpd/etc/pure-ftpd.conf</li><li>PureFtpd MySQL 配置文件：/usr/local/pureftpd/pureftpd-mysql.conf</li><li>Proftpd 配置文件：/usr/local/proftpd/etc/proftpd.conf</li><li>Proftpd 用户配置文件：/usr/local/proftpd/etc/vhost/用户名.conf</li><li>Redis 配置文件：/usr/local/redis/etc/redis.conf</li></ul><h3>写在后面：</h3><p><a
