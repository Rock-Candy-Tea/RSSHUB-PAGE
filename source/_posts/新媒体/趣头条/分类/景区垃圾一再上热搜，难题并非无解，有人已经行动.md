letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">暑期出游热潮，新疆独库公路垃圾遍地的话题引起全网热议，文旅部门发文倡议，呼吁人们文明游览，自觉维护生态环境，同时，以“无痕山林”为代表的绿色户外活动理念一再被提及。</span></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">记者联系了中国绿发会自然教育专业委员会顾问专家、国内第一家专业的无痕环境教育机构——</span><a href="http://www.chinalnt.com/index.asp" style="margin: 0px; padding: 0px; color: rgb(0, 165, 235); -webkit-tap-highlight-color: rgba(0, 0, 0, 0); letter-spacing: 0.5px; white-space: normal; background-color: rgb(255, 255, 255);">“无痕中国”</a><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">联合创始人大麦，聊了聊有关公路、景区垃圾的困扰和解决方法以及无痕环境理念在中国的本土实践。</span></p><p><strong style="margin: 0px; padding: 0px; color: rgb(51, 51, 51); letter-spacing: 0.5px; white-space: normal; background-color: rgb(255, 255, 255);">景区和公路垃圾: 是生态保育的基础题，却也是全球难题</strong></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">景区和公路垃圾是老生常谈的问题，每到旅游旺季，游客随手乱扔垃圾等不文明现象屡见不鲜。“这是个全球难题，尤其是以旅游作为支柱产业、旅游资源发达的国家和地区都一直备受困扰”，大麦说。</span></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">以闻名于世的威尼斯为例，根据当地政府数据统计，旅游业“贡献”了28%至40%的垃圾，其中包括大量塑料水瓶。今年夏天，当局倡议人们使用能重复使用的水瓶，倡导游客利用散落在水城广场和小巷中的饮水喷泉，还推出了一款应用程序，配显示所有最近喷泉的地图。</span></p><p style="text-align: center;"><img alt="今年夏天，威尼斯当局倡议人们使用重复使用的水瓶 Forbes India 图" src="http://static.1sapp.com/quadmin/images/2022/07/28/1658998530543794773.jpg" data-size="800,600" size="800,600" class referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 6px 0px 20px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); color: rgb(128, 128, 128); line-height: 20px; letter-spacing: 0.5px; white-space: normal; background-color: rgb(255, 255, 255); text-align: left;">今年夏天，威尼斯当局倡议人们使用重复使用的水瓶 Forbes India 图</p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">全球其他国家如何应对这些难题可以作为“他山之石”，大麦介绍了多年来他收集和研究的情况: 首先是服务站和加油站的配置设计，美国公路上的自助加油站配有清洁和处理垃圾的设备，一个加油点位配一套，让驾乘族来处理垃圾。</span></p><p style="text-align: center;"><img alt="美国esso加油站, 灰色的是垃圾桶, 方便加油时的垃圾丢弃 大麦 图" src="http://static.1sapp.com/quadmin/images/2022/07/28/1658998530569066149.jpg" data-size="1280,853" size="1280,853" class referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 6px 0px 20px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); color: rgb(128, 128, 128); line-height: 20px; letter-spacing: 0.5px; white-space: normal; background-color: rgb(255, 255, 255); text-align: left;">美国esso加油站, 灰色的是垃圾桶, 方便加油时的垃圾丢弃 大麦 图</p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">其次是对不文明行为的处理，美国马里兰州、路易斯安那州和马萨诸塞州专门立法，来解决公共高速公路、沿海地区和休闲区乱扔垃圾问题。在马萨诸塞州，如果车主和乘客被发现从车窗扔垃圾或垃圾，驾驶者的驾驶执照可能会被暂扣7天。由于麻省的地理位置和气候，一旦干旱，当地灌木和干草容易起火，因此人们从车里掸一下烟也被视为“乱扔垃圾”。苏格兰于2014年提出了垃圾计划，任何丢弃垃圾的人都可能被处以80英镑罚款，如果不支付，可能会被起诉并面临最高2,500英镑的罚款。</span></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">大麦分享自己去韩国汉拿山徒步时的观察，他记录下了城板岳入口的窗口处注明，其中明确表示“禁止丢弃垃圾……发现以上行为将按照《韩国自然公园法》处以罚款”，而免费向游客提供的《汉拿山国立公园》手册有韩文、英文和日文三个版本，其中也提示“请把垃圾随身带回，以保护动植物和自然环境”。</span></p><p style="text-align: center;"><img alt="韩国汉拿山标示牌 大麦 图" src="http://static.1sapp.com/quadmin/images/2022/07/28/1658998530567550656.jpg" data-size="438,600" size="438,600" class referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 6px 0px 20px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); color: rgb(128, 128, 128); line-height: 20px; letter-spacing: 0.5px; white-space: normal; background-color: rgb(255, 255, 255); text-align: left;">韩国汉拿山标示牌 大麦 图</p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">“这是全球各地都面临的难题，哪怕它只是一道看似非常基础的题目”，大麦感慨，“中国生物多样性保护与绿色发展基金会、四川省绿色江河环境保护促进会等多个机构也对我国的景区垃圾做过相应的调研，2018年，生态环境部、中央文明办等五部门联合发布《公民生态环境行为规范（试行）》（简称“公民十条”），引领公民践行生态环境责任……这些举措都慢慢地在渗透或影响人们”。</span></p><p style="text-align: center;"><img alt="《公民生态环境行为规范（试行）》（简称“公民十条”） 生态环境部宣传视频截图" src="http://static.1sapp.com/quadmin/images/2022/07/28/1658998530571340384.png" data-size="1056,592" size="1056,592" class referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 6px 0px 20px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); color: rgb(128, 128, 128); line-height: 20px; letter-spacing: 0.5px; white-space: normal; background-color: rgb(255, 255, 255); text-align: left;">《公民生态环境行为规范（试行）》（简称“公民十条”） 生态环境部宣传视频截图</p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">多年的研究和实践经历，大麦对“我们能做什么？”有了自己的思考。他认为，“首先，在现有的政策法规的基础上，根据国情实际和可操作性，我们期待地方性的立法，一方面基于各地环境问题实际情况，尤其是旅游资源发达、生态旅游为重要业态的地区，例如云南、四川、内蒙古等省市自治区，通过法律对不文明现象进行制约。其次，在执行层面，在旅游景点、自然保护区的管理和维护上，期待更多部门、企业的联动。例如垃圾桶的数量、摆放点位密度、垃圾分类，终端处理以及设施运输，进行更科学、更精准地设计和管理。再次，对加油站和服务区的功能升级，应对节假日高峰客流带来的垃圾问题，妥善、高效处理这些问题。还有，生态教育和传播需要长期、不懈地发力。”</span></p><p><strong style="margin: 0px; padding: 0px; color: rgb(51, 51, 51); letter-spacing: 0.5px; white-space: normal; background-color: rgb(255, 255, 255);">“无痕”的中国本土实践</strong></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">“无痕”（“Leave No Trace”，简称“LNT”），意为“人过不留痕”，是一个国际性的生态伦理和环境行动，主要目的是规范公众的生态环境行为，倡导公众降低在旅行和户外中对环境的影响，中文语境中普遍翻译为“无痕”“无痕山林”“无痕山野”。</span></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">创立于2012年12月的 “无痕中国”是中国大陆首个专业的无痕环境教育机构，通过系列无痕环境课堂来推动公民对环境、水资源、大气、垃圾、生物多样性问题的关注和解决，推动政府可持续发展政策的制定和实施。</span></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">大麦指出，无痕运动从启蒙至今已有近40年历史，但若只是照搬海外做法并不妥当，因此，他和伙伴们开始了长期的无痕环境的中国本土化探索。</span></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">“国际上‘无痕’伦理中有一条‘妥善处理野外用火’，但在中国绝大多数景区和保护区，都严禁用火，因此在撰写《LNT无痕环境伦理》中，结合我们国家实际国情做了细化和梳理”。</span></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">“无痕中国”总结，“照顾好自己、照顾好同伴、照顾好环境”是实现LNT无痕七个原则的有效路径。在具体实施过程中，“无痕中国”遵循“原则可视化”的逻辑，用清晰的标志、图像解释抽象的概念。</span></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">团队还依照“体验式、游戏化和在地化”方法，设计了关于《公民十条》、垃圾分类、河流和赏鸟等多门类课程，让参与者积极、欢乐地沉浸在无痕环境教育活动中。</span></p><p style="text-align: center;"><img alt="2020年，在深圳培训无痕LNT环境讲师 无痕中国 图" src="http://static.1sapp.com/quadmin/images/2022/07/28/1658998530559087232.jpg" data-size="1280,960" size="1280,960" class referrerpolicy="no-referrer"></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 6px 0px 20px; -webkit-tap-highlight-color: rgba(0, 0, 0, 0); color: rgb(128, 128, 128); line-height: 20px; letter-spacing: 0.5px; white-space: normal; background-color: rgb(255, 255, 255); text-align: left;">2020年，在深圳培训无痕LNT环境讲师 无痕中国 图</p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);"> 过去7年里，团队与不同地区的不同政府部门、社会机构和社会力量合作，推动了无痕环境的建设和发展，国家林草局的《自然教育指导师手册》，也使用了“无痕中国”环境教育案例。</span></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">大麦感言，其实公众对生态保育的关注和热情很高，多年的实践让他们有了一套规模化、可复制的活动和课程系统。</span></p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">今年六月，无痕团队联合济南文旅部门推出“无痕露营”活动，发布了《无痕露营倡议书》，提醒游客行前做好准备、减少使用洗涤剂和一次性用品，不伤害一草一木，不留下一片垃圾，不要打扰到他人等，通过倡议带动更多游客践行无痕环境建设。</span></p><p style="text-align: center;"><img alt="无痕露营原则 无痕中国 图" src="http://static.1sapp.com/quadmin/images/2022/07/28/1658998530576244592.jpg" data-size="1899,2721" size="1899,2721" class referrerpolicy="no-referrer"></p><p style="margin-top: 0px; text-align: left;">无痕露营原则 无痕中国 图</p><p><span style="color: rgb(51, 51, 51); letter-spacing: 0.5px; background-color: rgb(255, 255, 255);">大麦表示，近两年，新能源企业、户外活动机构陆续与“无痕中国”探讨了多个无痕环境项目，随着疫情带来的户外活动热潮，他相信会有越来越多的企业、社会组织助力，共同推动中国生态环境教育和发展。</span></p>
