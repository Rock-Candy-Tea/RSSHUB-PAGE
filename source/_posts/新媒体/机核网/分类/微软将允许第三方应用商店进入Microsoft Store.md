</figure><div class="story_block story_block-text " data-block="true" data-editor="f9kq9" data-offset-key="5jjv7-0-0"><div data-offset-key="5jjv7-0-0" class="public-DraftStyleDefault-block public-DraftStyleDefault-ltr"><span data-offset-key="5jjv7-0-0"><span data-text="true"> “就像任何其他应用程序一样，第三方商店的应用程序将有一个产品详细信息页面——可以通过搜索或浏览找到——这样用户就可以像 Microsoft Store 中的任何其他应用程序一样轻松地找到并安装它在 Windows 上，” Microsoft Store 总经理 Giorgio Sardo 解释说。“今天，我们公布 Amazon 和 Epic Games Store 将在未来几个月内将他们的商店应用程序带到 Microsoft Store 上，我们未来也欢迎其他商店。” </span></span></div></div><div class="story_block story_block-text " data-block="true" data-editor="f9kq9" data-offset-key="kcl8-0-0"><div data-offset-key="kcl8-0-0" class="public-DraftStyleDefault-block public-DraftStyleDefault-ltr"><span data-offset-key="kcl8-0-0"><span data-text="true">虽然第三方游戏平台的应用程序本身登陆了 Microsoft Store，但这并不意味着用户可以在 Microsoft Store 中发现和安装来自微软竞争对手商店的单独应用程序和游戏。作为 Windows 11 更新中 Microsoft Store 的鼓励政策，开发者的如果使用微软之外的替代支付平台，开发者将会获得全部收益，但这政策不包含游戏。不过 Microsoft Store 中游戏的收入分成从今年八月起将由30%降低到12%。</span></span></div></div>
