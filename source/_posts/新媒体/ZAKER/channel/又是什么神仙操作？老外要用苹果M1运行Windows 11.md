<p>【CNMO 新闻】用苹果电脑运行 Windows 并不是什么新鲜事，相对 Windows 电脑，苹果电脑的优势在于软件与生态，但同样，封闭的软件生态也会成为一些人眼中的弊端，所以有些用户会选择将自己的苹果电脑刷成 Windows 系统。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202206/629d768c8e9f093d5e5fdf77_1024.jpg" data-height="338" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202206/629d768c8e9f093d5e5fdf77_1024.jpg" referrerpolicy="no-referrer"></div></div>苹果 M1 芯片<p></p><p>近日，Arminder Singh 在推特上宣布，他正在进行一个新项目，目标是在搭载 M1 芯片的苹果电脑上运行 Windows 11。由于 M1 芯片不同于其他处理器，Windows 没有针对其 CPU 进行优化，所以 Arminder 仍需要一段时间才能完成这一目标。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202206/629d768c8e9f093d5e5fdf78_1024.jpg" data-height="337" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202206/629d768c8e9f093d5e5fdf78_1024.jpg" referrerpolicy="no-referrer"></div></div>Windows 11<p></p><p>当然，这个项目的实际意义不大，因为很少会有用户在购买带有 M1 处理器的苹果电脑后，用它去运行 Windows 系统。</p><p>目前还没有正式的方法在带有苹果 M1 芯片的系统上启动 Windows 11，微软去年回避了用户提出的这项建议，并声明在 M1 芯片上运行 Windows 11 是不受支持的。但同时，由于这一操作没有被官方禁止，Arminder 在理论上可以通过不断调试来达成他的目标，不过目前尚不清楚他什么时候能成功。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
