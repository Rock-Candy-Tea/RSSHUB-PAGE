<p>IT 之家 1 月 26 日消息，据微软博客发布，Windows 10 KB5009596 已经发布，它实际上带有一个新功能，可以帮助你迁移到 Windows 11。此外，Windows 10 可选更新还带有一系列的一般性改进和修复，包括修复了所有蓝牙设备的问题。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202201/61f0beb08e9f091a870c1901_1024.jpg" data-height="364" data-width="696" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202201/61f0beb08e9f091a870c1901_1024.jpg" referrerpolicy="no-referrer"></div></div>KB5009596 是可选的更新，它比作为 Windows Insider 预览版要安全得多。如果你的 Windows 10 安装有问题，而且你不想等到 2 月的强制性 " 星期二补丁 " 的推出，你可以选择安装该可选更新。<p></p><p>最新的可选更新的主要亮点是支持名为 " 同步你的设置 " 的新功能。这个备份功能可以让你在微软账户中维护已安装的应用程序列表，这样你就可以在同一设备或其他设备上迁移到 Windows 11 后轻松恢复这些应用程序。</p><p>这项功能显然是微软为过渡用户向 Windows 11 迁移所做努力的一部分，它将在未来几周向所有人推出。此外，一个错误已经被修复，Get-TPM PowerShell（检查 Windows 11 的 TPM 信息的命令）会出现错误信息 "0x80090011"。</p><p><strong>Windows 10 KB5009596 ( Build 19044.1503 ) 完整更新内容</strong></p><p>微软正在更新《新闻和兴趣》任务栏小组件，其新功能将允许用户直接管理他们的 Microsoft Edge 配置文件。像往常一样，你可以从浏览器的《新闻和兴趣》标签管理 Edge 配置文件。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202201/61f0beb08e9f091a870c1902_1024.jpg" data-height="572" data-width="1029" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202201/61f0beb08e9f091a870c1902_1024.jpg" referrerpolicy="no-referrer"></div></div>在错误修复方面，Windows 10 Build 19044.1503 已经修复了 UI 自动化功能可能破坏 Outlook 桌面客户端的问题。同样，微软已经解决了一个问题，即当你启用快速启动时，Windows Subsystem for Linux 2（WSL2）可能会停止正常工作。<p></p><p>修复了一个错误，即一些应用程序可能在使用笔触觉 API 时崩溃。另一个错误是，设备报告自己不符合条件访问的要求。</p><p>同样，微软也修复了一个可能破坏打印的问题，如果你在 Windows 10 版本 2004 或更高版本上使用 USB，打印机将打印错误的输出。另一个故障已被修复，在 Microsoft Edge 中播放某些环绕声音频时发生，特别是在浏览 Netflix 等平台时。</p><p>IT 之家获悉，Windows 10 更新还解决了一个影响远程桌面屏幕的问题，特别是当你的软件使用高级视频编码（AVC）时。</p><p>更新中包括的其他错误修复和改进：</p><p>Internet Explorer 已经更新了新的横幅，将提醒用户其预先宣布的退役时间表。</p><p>微软修复了一个导致 vpnike.dll 和 rasmans.dll 死锁的性能问题。</p><p>微软已经修复了一个关键的错误，这个错误会使操作系统崩溃，并发出模糊的错误信息 IRQL_NOT_LESS_OR_EQUAL。</p><div id="recommend_bottom"></div><div
