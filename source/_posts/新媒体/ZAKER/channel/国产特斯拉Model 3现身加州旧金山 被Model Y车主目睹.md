<p>【TechWeb】12 月 10 日消息，据国外媒体报道，当地时间周四，一辆由特斯拉上海超级工厂所生产的 Model 3，出现在了加州旧金山的道路上，引发广泛关注。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202112/61b2df5a8e9f096af504a0e6_1024.jpg" data-height="258" data-width="500" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202112/61b2df5a8e9f096af504a0e6_1024.jpg" referrerpolicy="no-referrer"></div></div>这一辆现身旧金山的国产特斯拉 Model 3，是在路上行驶时，被一位特斯拉爱好者发现的，当时他正驾驶 Model Y 行驶在旧金山的公路上，注意到了前面一辆车尾有中文特斯拉字样的 Model 3，他拍下了照片，随后发给了外媒。<p></p><p>从外媒的报道来看，起初后车 Model Y 的车主，并不确定这一辆 Model 3 是否来自上海超级工厂，毕竟车尾的中文特斯拉标志可以通过第三方购买。</p><p>外媒在报道中表示，出现在旧金山的这辆带有中文特斯拉字样的 Model 3，与美国生产的 Model 3 明显不同，制造商的车牌也表明这是一辆来自上海超级工厂的 Model 3，车牌上也有与美国产 Model 3 不同的信息。</p><p>上海超级工厂所生产的 Model 3，虽然已出口到了欧洲等市场，但特斯拉在北美市场所销售的 Model 3，是产自加州弗里蒙特工厂，上海超级工厂所生产的 Model 3 现身旧金山，还出现在了道路上，也就难免引发外界关注。但目前还不清楚上海超级工厂所生产的 Model 3，为何会出现在加州，也不清楚驾驶这辆 Model 3 的，是特斯拉方面的人员还是特斯拉的消费者。</p><p>外媒在报道中表示，上海超级工厂生产的 Model 3 现身加州，有多种原因，可能是用于测试，也可能监管方面的考虑。</p><p>外媒在报道中也提到，特斯拉在 10 月份曾宣布，他们准备将使用磷酸铁锂电池的 Model 3，推向美国市场及全球市场，特斯拉上海超级工厂生产的 Model 3，采用的就是磷酸铁锂电池。</p><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
