<p>IT 之家 6 月 27 日消息，据 MacRumors 报道，在苹果推出配备 M2 芯片的全新 13 英寸 MacBook Pro 之后，人们发现售价 1,299 美元的 256GB 存储空间基本款的 SSD 读 / 写速度明显低于上一代同等型号。</p><p>Max Tech 和 Created Tech 等 YouTube 频道使用 Blackmagic 的 Disk Speed Test 应用测试了 256GB 版本，发现 SSD 的读取和写入速度均在 1450 MB/s 左右，比配备 M1 芯片和 256GB 存储空间的 13 英寸 MacBook Pro 的读取速度慢 50% 左右，写入速度慢 30% 左右。</p><p>Max Tech 分享的测试数据如下：</p><p>13 英寸 MacBook Pro ( M1/256GB ) 读取速度：2,900</p><p>13 英寸 MacBook Pro ( M2/256GB ) 读取速度：1,446</p><p>13 英寸 MacBook Pro ( M1/256GB ) 写入速度：2,215</p><p>13 英寸 MacBook Pro ( M2/256GB ) 写入速度：1,463</p><p>IT 之家了解到，Yuryev 拆解了新款 M2 13 英寸 MacBook Pro，发现 256GB 版本只配备了一个 NAND 闪存芯片，而之前的机型有两个 NAND 芯片，每个为 128GB。这种差异可能解释了为什么新款的 SSD 速度较慢。</p><p>不过，如果用户再多花点钱，选择 512GB 版本的 M2 13 英寸 MacBook Pro，就会发现 SSD 速度提高了，苹果尚没有回应这一差异的原因。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
