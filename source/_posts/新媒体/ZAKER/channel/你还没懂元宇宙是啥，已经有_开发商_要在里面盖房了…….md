应是一个平行于现实世界的虚拟世界，拥有完整运行的社会和经济系统，现实中的人能以 " 数字化身 " 形式进入虚拟时空，并获得尽可能接近真实的体验。用户可在 " 元宇宙 " 中进行娱乐、社交、消费、内容创作等。</p><p>在他们的预想里，现实世界里城市的所有元素，都能在元宇宙找到映射。电影院、体育馆、游乐园、咖啡厅、舞台 …… 犹如建造城市一般，这些都需要大量的创作者进行设计，才能让元宇宙不断更新迭代，或许未来也会产生如今无法想象的工作模式。</p><p><strong>元宇宙社区 </strong></p><p><strong>用人气值排名</strong></p><p>万物皆可元宇宙，当社区成为一种元宇宙世界，我们的生活会发生什么样的改变？</p><p><strong>" 元宇宙一部分可以理解为，现实世界发生的事、身份 ID 映射到虚拟世界当中去 "。</strong>数字资产专家、京彩未来家总裁张其亮先生对 36 氪未来可栖作者表示，</p><p>张其亮与作者分享，社区空间，可能更容易缔造一个关于元宇宙的商业故事：</p><p>在他的设想中，元宇宙里每个居住在这个特定社区的人拥有身份 ID，每个 ID 之间的社交关系就是你的社区邻居。社区系统录入每个 ID 的资料，包括该 ID 拥有的车辆、宠物和衣食住行相关生活习惯的大数据，但不对外公开。</p><p>对外公开的是每个人在社区元宇宙的人气值，相当于一个分数排名，ID 的日常行为作为分数录入到大数据当中，<strong>比如：没有做垃圾分类、在家里制造噪音会减分，但帮助社区元宇宙维护更好的环境、帮助其他 ID 会加分。</strong></p><p>当某个有宠物的 ID 显示去其他城市出差的状态，恰好他又是一个有高人气值的 ID，其他邻居 ID 看到了这种状态会更愿意帮助照看他的宠物、和其他可能发生的状况，自己也可以获得加分。<strong>每个 ID 之间可以互不相识，但能做到互帮互助。</strong></p><p><strong>通过社区元宇宙的世界规则设定，社区或许也可以实现收支平衡。</strong>在正常上缴物业费的基础上，ID 如果作出元宇宙空间的减分行为，也可以选择交罚款的方式避免减分。</p><p>当社区空间成为一个现实空间之外的地方，并且囊括并实现了 " 社区世界 " 里的关系、沉浸感、多元化、经济系统、文明的话，某种程度上，" 元宇宙的社区空间 " 或许可以通过一种超现实世界空间的约束，打造出一个更和谐的邻里空间。</p><p></p><div class="img_box" id="id_imagebox_5" </strong></p><p><strong>已经有了</strong></p><p>当元宇宙概念与建筑联系到一起，<strong>分为实体建筑空间和虚拟建筑空间。</strong>当元宇宙应用到实体建筑本身，目前更像是一种建筑设计参数的应用。</p><p>扎哈 · 哈迪德建筑事务所（Zaha Hadid Architects）合伙人帕特里克 · 舒马赫（Patrik Schumacher）曾在 2008 年提出了 " 参数化主义 " 的概念。舒马赫致力于打通城市物理空间与虚拟世界的通道，他也坚信，技术能使建筑更好地承担起社会建构的职能。</p><p>根据三联生活周刊报道，参数化主义虽然尚未成为最主流的建筑风格，但它未来很有潜力，因为它可以让所有的工程技术朝着正确、合理的方向发展。<strong>它可以通过拓扑优化工具和计算，让复杂的结构工程非常地有序。</strong>比如北京大兴国际机场，为了使空间更加开阔，在结构设计上使屋面与承重结构一体化，通过参数化计算出 12800 块玻璃、12300 个球形节点和超过 60000 根连杆，经过层层模拟，最终得以将这座施工难度极大的建筑完成。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " " 在皮不在骨 "。</strong></p><p>对于将元宇宙应用到实体建筑上。业内人士对 36 氪作者表示，<strong>" 总有人想做更现实的事情，但好像是一种混淆，元宇宙的本质是将大幅减少我们的活动空间 "。</strong></p><p>扎哈 · 哈迪德建筑事务所也正在研发真正的虚拟元宇宙建筑。舒马赫对三联生活周刊表示，在新冠疫情的冲击之下，我们的工作环境和工具都 " 线上化 " 了，所有人都在网上，而元宇宙作为虚拟空间的一个新事物，它需要大量的作品来吸引人。</p><p>熟知现实世界建构的建筑师，在元宇宙里从事虚拟空间开发，将会是未来很重要的研究和设计工作浪潮。" 因此，我们也在努力研究，如何设计一个与真实世界相同的三维虚拟世界。<strong>未来人们不再需要从不同的接口进入网络，而是可以在同一个三维虚拟空间中完成所有的事情。"</strong></p><p>建筑师们想做绝对化可以实现线上办公的空间，也就是通过设计研发，<strong>未来我们可以单独的坐在一个空间，但通过虚拟三维世界的入口，直接到达公司，和同事、合作方进行交流，</strong>很多功能都被植入到元宇宙空间中去。</p><p>对于建筑师，一定程度上是摆脱束缚后，在虚拟空间作出更有想象力的建筑设计。国建管理员会议主席 Nova 在公开活动中表示，虚拟世界没有现实世界中建筑行业的条条框框，可以让建筑走向更纯粹的艺术。</p><p><strong>也就是当元宇宙的概念被广泛应用时，走在前列的建筑设计师不仅会绘制 3D 图纸，还会写代码。而奉行传统概念的建筑师，或许会遇到元宇宙建筑世界的水土不服，这背后更像是一种知识领域的碰撞与结合。</strong></p><p>跨现实艺术家、虚拟空间建筑师胡胡近期在公开场合表示，<strong>基于元宇宙世界的建筑龙头企业，其实已经出现，但 " 虚拟世界的开发商 " 是否能长期存在，取决于是否被我们日常生活所需要。</strong></p><p>本文来自微信公众号 "
