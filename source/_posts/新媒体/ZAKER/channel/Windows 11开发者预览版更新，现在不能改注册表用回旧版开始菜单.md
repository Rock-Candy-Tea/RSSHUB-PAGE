<p>如果大家有关注微软最新推出的 Windows 11 操作系统的话，应该都知道它对比起 Windows 10 在界面上有不少的改变，包括任务栏以及开始菜单都有大改。但是在预览版的 Windows 11 里面，用户仍然可以通过修改注册表来实现把 Windows 10 里的开始菜单也给弄出来 ,。不过现在这个方法就失效了，因为微软在 Windows 11 的最新版更新中去掉了这个 " 功能 "。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202107/60ec0e438e9f091d1b54677e_1024.jpg" data-height="672" data-width="1200" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60ec0e438e9f091d1b54677e_1024.jpg" referrerpolicy="no-referrer"></div></div>可能有的用户对于 Windows 11 在界面上的大改也不是那么适应，开始列的摆放以及菜单的样子与 Windows 10 都一样，一来开始按键现在是摆在了任务栏中间，二来开始菜单也改头换面，并且去掉了那些动态磁贴。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202107/60ec0e438e9f091d1b54677f_1024.jpg" data-height="869" data-width="837" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/60ec0e438e9f091d1b54677f_1024.jpg" referrerpolicy="no-referrer"></div></div>改完注册表之后的开始菜单<p></p><p>根据 Bleeping Computer，对于这部分的用户其实之前是有个方法让他们在 Windows 11 预览版里把 Windows 10 的开始菜单调出来的，具体来说就是通过修改注册表。用户只要在 "HKEY_CURRENT_USERSoftwareMicrosoftWindowsCurrentVersionExplorerAdvanced" 里创建一个 "Start_ShowClassicMode" 的项目，并且把数值设为 1，之后重启 Windows Explorer 程序就可以看到 Windows 10 的开始菜单了。</p><p>而上周四微软为 Dev 预览版 的 Windows 11 推出了 Build 22000.65 更新，这个更新除了为 Windows 11 带来更多功能方面的完善外，例如右键菜单刷新选项外，同时也去掉了上述改注册表的操作。外媒 Bleeping Computer 也测试过，发现改注册表来使用 Windows 10 开始列表的操作是不再可行了，不过现在不知道之后微软会不会再次启用这项修改。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
