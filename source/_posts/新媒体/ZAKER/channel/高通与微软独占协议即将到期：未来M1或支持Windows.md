<p>微软已经发布了 Windows 11 系统，而苹果也发布了 M1 Max 以及 M1 Pro 芯片，按照以往来说，苹果 Mac 产品会支持 Windows 系统，不过到目前为止苹果官方还没有说 M1 系列处理器能够支持 Windows，过去有人称苹果搭载 M1 系列处理器的终端产品不支持 Windows 的最大原因是 M1 不兼容，不过现在有消息称，微软与高通签订了独占协议，称只有 X86 处理器以及高通的处理器才能支持 Windows 系统，因此苹果 M1 处理器才不支持。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202111/619c48c08e9f09448057e03e_1024.jpg" data-height="422" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/619c48c08e9f09448057e03e_1024.jpg" referrerpolicy="no-referrer"></div></div>来自著名安卓开发者论坛 XDA 的消息称，微软已经和高通在之前签订了一份独占协议，根据该协议，微软将会和高通一同研究 Windows 系统在高通处理器上顺利运行，而高通 ARM 芯片也是唯一支持 Windows 原生运行的 ARM 芯片。当然消息还称对于微软来说，这项独占协议是有期限，而现在该期限即将来临，因此未来微软也将授权给其他厂商以支持 Windows。<p></p><p>显然该项规定未来除了利好苹果之外，也将利好联发科等芯片厂商，毕竟这一次的天玑 9000 处理器好评如潮，未来不排除联发科将会针对 Windows 系统推出相对应的 ARM 芯片，此外苹果 M1 系列处理器就不说了，对于其他品牌来说在性能上处于碾压般的优势。不过高通拿着独占协议似乎也没有推出什么为 Windows 打造的旗舰芯片，未免有点太过于浪费该协议了。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
