<p>IT 之家 7 月 3 日消息 外媒 9to5 Mac 报道，iPhone 流行的 "OldOS"TestFlight 应用程序背后开发者 Zane Kleinberg 又带来了新的工具，允许你通过网络浏览器使用经典的 macOS 小组件。Kleinberg 解释了经典的 macOS 小组件是如何在 Tiger 中首次推出的，它们完全是由 HTML、CSS 和 JavaScript 构成。这使他能够恢复苹果设计的几个老的小组件，包括计算器、世界时钟、单位转换器、贴纸和平铺拼图。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202107/60e04f7f8e9f093e0a15a03e_1024.jpg" data-height="1066" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60e04f7f8e9f093e0a15a03e_1024.jpg" referrerpolicy="no-referrer"></div></div>Kleinberg 的新网站最好通过 Mac 或 iPad 上的浏览器使用，但它也可以在 iPhone 上使用。在 iPhone 上使用该网站时，小组件将垂直堆叠，而不是水平堆叠。<p></p><p>所有的小组件都像你所期望的那样工作。事实上，它们与苹果公司过去在 macOS 上提供的小组件完全相同。它们基于网页的架构意味着它们中的一些至今仍可使用，而 Zane 的网站只是为了再次浮现它们。你仍然可以翻转它们来改变设置，如贴纸的颜色或世界时钟的城市。遗憾的是，该网站只限于 Zane 设置的几个小组件。希望有一天用户能够在网站上添加自己的小组件，以尝试旧的第三方 Dashboard 小组件。</p><p>IT 之家获悉，对于那些怀念 Dashboard 的用户来说，这是一次怀旧之旅。苹果在 2019 年发布 Catalina 时将 Dashboard 从 macOS 中移除。Big Sur 为小组件引入了新的设计，并将它们放在一个网格视图中，从屏幕的右侧滑出。目前的 macOS 小组件迭代并不像 Dashboard 那样具有互动性，所以如果你一直在怀念小组件大行其道的旧时代，那么你应该看看 Zane 的最新创作。</p><p>Zane 最近还更新了 OldOS，增加了设置自定义墙纸的功能。最新版本还包括当你给 iPhone 充电时的旧电池锁屏设计。下次更新将包括全新的应用程序，Zane 给出了电视图标的暗示。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202107/60e04f7f8e9f093e0a15a03f_1024.jpg" data-height="360" data-width="612" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/60e04f7f8e9f093e0a15a03f_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
