19043.1266。另一方面，版本 20H2（2020 年 10 月更新）将获得 Windows 10 19042.1266。同样，这个累积预览更新将在仍然运行 Windows 10 版本 2004 的设备上显示为 Build 19041.1202。<p></p><p>IT 之家提醒，对 Windows 10 版本 2004 版的支持将在今年 12 月结束，微软将很快停止发布可选更新。</p><p>下面是 Windows 10 KB5005611 ( Build 19043.1266 ) 更新内容：</p><p>修复了新闻和兴趣的问题。</p><p>修复了 Outlook 应用程序和插件的问题。</p><p>修复了强迫新闻和兴趣小组件在任务栏上的问题。</p><p>修复了系统崩溃问题。</p><p>修复了使用数字助理录音时声音失真问题。</p><p>通过今天的累积更新，微软终于修复了一个问题，即在浏览同步到账户的电子邮件或日历时，Outlook 等应用程序会崩溃。此外，还修复了一个错误，即 Outlook 插件可能阻止用户在其他 Windows 应用程序中输入回复的问题。</p><p>微软也在研究新闻和兴趣的问题。安装此更新后，当你使用某些屏幕分辨率或缩放选项时，新闻和兴趣不应该再出现模糊。</p><p>此外，微软还修复了一个错误，即当你启用夏令时（DST）变化时，系统时钟可能显示不正确的时间。</p><p>微软还修复了一个错误，即当你在正常使用后重新启动系统时，会导致你的设备停止工作。另一个 Bug 已被修复，当任务栏没有固定在屏幕底部（默认位置）时，任务栏应用程序可能无法工作。</p><p>此外，Windows 10 Build 19043.1266 对使用 Microsoft Teams 等应用程序共享受保护内容时的一个错误进行了修复。更新后，受保护的内容，如数字版权管理（DRM）媒体，将对所有人显示为空白。以前，DRM 内容对一些用户来说是透明的。</p><p>该补丁还附带了更多错误修复：</p><p>微软修复了系统重启后，Windows 无法记住 NumLock 的状态的问题。</p><p>微软修复了导致 lsass.exe 内存泄漏的问题。</p><p>微软修复了 BSOD 蓝屏死机的错误。</p><p>微软表示，目前没有发现更新中的关键问题，但如果遇到新的错误，你可以随时使用命令提示符或设置卸载 Windows 10 更新。</p><p>对于 Windows 11，微软已经发布了一个单独的更新，包括质量改进和错误修复。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
