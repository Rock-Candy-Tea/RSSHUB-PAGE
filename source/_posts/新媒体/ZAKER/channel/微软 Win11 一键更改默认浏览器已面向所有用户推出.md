<p>IT 之家 4 月 13 日消息，今年 3 月底，微软为 Windows 11 发布了 Build 22000.593 ( KB5011563 ) 可选更新，该版本让用户可以更容易地切换 Windows 11 中的默认浏览器。</p><p>而在今日发布的 Build 22000.613（KB5012592）更新中，这一功能已经面向所有用户推出。该更新是一个强制性更新，而不仅仅是一个可选更新。</p><p>Windows 11 刚刚发布时，引入了一种切换默认浏览器的繁琐方式，要求用户手动更改每个文件扩展名（如 HTML 和 PDF）的关联。这一举措受到了许多浏览器公司和用户的批评。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202204/625673f38e9f09040f0725b9_1024.jpg" data-height="625" data-width="1104" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/625673f38e9f09040f0725b9_1024.jpg" referrerpolicy="no-referrer"></div></div>上个月，微软推出了可选的 Windows 11 版本，允许用户一键更改某些文件扩展名（例如 HTML）的默认浏览器。IT 之家了解到，该方法仍然不会切换所有扩展的默认设置，Edge 仍然是打开 PDF、SHTML、SVG、FTP 和其他 Web 相关文件的默认应用，除非用户一一手动更改所有文件关联。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202204/625673f38e9f09040f0725ba_1024.jpg" data-height="599" data-width="820" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202204/625673f38e9f09040f0725ba_1024.jpg" referrerpolicy="no-referrer"></div></div>Mozilla 和 Vivaldi 等竞争对手此前表示，对微软的改进表示赞赏，但其仍做的不够好。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
