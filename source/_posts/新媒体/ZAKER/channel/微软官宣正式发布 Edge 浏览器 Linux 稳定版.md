<p>IT 之家 11 月 3 日消息，几天前，在微软 Linux 资源库中出现了首个 Edge 浏览器稳定版。尽管第一个 Edge Canary 浏览器 Linux 版是在 2020 年 10 月发布的，但一年来没有任何浏览器的稳定版本推出。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202111/618236b68e9f09090b0a073b_1024.jpg" data-height="496" data-width="809" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/618236b68e9f09090b0a073b_1024.jpg" referrerpolicy="no-referrer"></div></div>今天，微软正式宣布 Edge 浏览器 Linux 稳定版的到来，并表示 :<p></p><p>" 一个共同的要求是你需要 Microsoft Edge 跨越你环境中的操作系统的广度。去年 10 月，我们在预览版频道（Dev 和 Beta 频道版）中为 Linux 提供了 Microsoft Edge。现在，该浏览器已通过稳定版频道普遍适用于 Linux。这一里程碑式的进展正式完善了 Microsoft Edge 通过稳定频道提供给所有主要平台。Windows、Mac、iOS、Android，现在还有 Linux。要在 Linux 上使用 Microsoft Edge，用户可以从我们的网站上下载它，或者使用命令行从 Linux 软件包管理器中检索它。"</p><p>IT 之家获悉，为了与 Linux 粉丝庆祝这一时刻，微软 Edge 冲浪游戏（edge://surf）现在在最新的 Edge Dev 频道构建版本 97 中采用了 Tux 企鹅元素，这是 Linux 系统的吉祥物。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202111/618236b68e9f09090b0a073c_1024.jpg" data-height="453" data-width="789" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/618236b68e9f09090b0a073c_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
