杨雪梅</p><p>编辑 / 韩大鹏</p><p><strong>划重点：</strong></p><p>1、过暗的银幕亮度、笨重的 3D 眼镜、抖动的镜头、以及高于 2D 影片的票价，让 3D 处境尴尬，甚至成为了鸡肋。</p><p>2、其实影片 3D 版本并非作者创作，更多是由于资本方、影院为迎合市场喜好的 " 宣发商业行为 "。没有 3D 效果的 3D 观影，只会徒增不适感。</p><p>3、从观众抵制伪 3D 来看，观影人群的素质提升，对内容要求也越来越高。对电影人制作人提出了更高要求，别把观众再当 " 韭菜 " 来割。</p><p>" 戴着 3D 眼镜看完《沙丘》，头有点头晕。"</p><p>" 票价 3D 要比 2D 贵，但是体验真不值得。"</p><p>"《沙丘》2D、3D 版都看了，差别真的很大。如果不是像《阿凡达》那样制作就是 3D 版本的，坚决不要看假 3D。"</p><p>" 没有刷过 IMAX 2D，就相当于没有真正地刷过《沙丘》。"</p><p>............</p><p>近日，科幻电影《沙丘》上映后，围绕要看 3D 还是 2D 版本，网友吵了起来，甚至出现了一波抵制 3D 版本的热潮。这也让不少已经看了 3D 版本的观众直呼后悔。</p><p>这与早间年，观众对于特效技术、3D 大片的追捧大为不同，从《阿凡达》上映时 3D 版电影一票难求，到如今的 3D 自由，整体而言，中国观众对于 3D 大片的审美已经发生了变化，兴奋感淡去，转而对故事、色彩、体验等细节有了更高的要求。</p><p>实际上，观众反感的并不是 3D 电影，而是原本以 2D 电影语言拍摄，后期转制为 3D 的 " 伪 3D"。也就是说，《沙丘》的拍摄手法和展现度无疑是 2D 最佳，转制 3D 版并非作者创作，更属于资本方、影院为迎合市场喜好的 " 宣发商业行为 "，这让观众的观影体验大打折扣。</p><p><strong>探访：3D 遭抵制，影院 2D 涨价？</strong></p><p>从网友争议来看，大部分人认为，戴上 3D 眼镜，除了字幕外不会有任何 3D 感，还失去了 2D 原有的画面和镜头感，分辨率从 4K 降低到 2K，电影的亮度也大幅降低，而 3D 版本下，《沙丘》的沙漠风光场景更显得昏暗——整体观影体验很差。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" referrerpolicy="no-referrer"></div></div><strong>《沙丘》两个版本的视觉效果对比（图片来自网络）</strong><p></p><p>随后，《007：无暇赴死》上映后也出现了类似的呼声。两部影片上映后，不少粉丝和观众直接向影院呼吁，要求调整排片。</p><p>据悉，影院会接到影片 2D 版、3D 版、IMAX 2D 版的数字硬盘，然后影院根据市场情况进行排片。据观察，《沙丘》刚上映的时候，影院普遍以 3D 版本排片为主，后期影迷对于 2D 排片的呼声越来越高，部分影院开始增加了 2D 的排片。</p><p>从海淀区一家 CGV 影城的排片来看，近期热映的电影中，《007：无暇赴死》的 2D、3D、IMAX3D 版本均有排片。从价格来看，该片最贵的是 IMAX3D、IMAX2D 版本，售价均为 98 元；其次是 2D 版本，票价 85 元；最便宜的是 3D 版本，售价 73 元、75 元。</p><p>该影院《沙丘》则只有 2D 版本，IMAX2D 售价 98 元。</p><p>对于只排 2D 版本的原因，这家影院工作人员表示，前期上映的时候，观众反映画面偏暗，观影体验不是很好，所以都改为 2D 的了，后续也不会再排 3D 版本了。</p><p>至于价格问题，上述工作人员表示，是根据影厅来定价的，一般来说，IMAX 激光厅是最贵的，其次是 SCREENX 厅，普通厅相对便宜。不过，这家影院将 2D 版本都排在 SCREENX 厅播放，价格也是较普通厅贵。</p><p>此外，根据观察，海淀区的几家华联影城，两部影片只有 3D 版本。影城工作人员表示，只拿到 3D 版本。</p><p>而另外一家金逸影城，《沙丘》和《007: 无暇赴死》两部影片则只有 2D 版本，没有 3D 版本。影城工作人员表示，影院是两种版本都有的，但是考虑到近期疫情下，3D 眼镜消毒问题，所以只排了 2D 的场。</p><p>从票价来看，目前两部影片 2D 版本的价格已经和 3D 版本相差不大，甚至高于 3D 版票价。虽然不同影院对于 2D 和 3D 版本的排片有不同的解释，但不少消费者却认为，供需决定价格，是因为 2D 版本需求变高，影院进而进行了价格和排片的调整。</p><p>此前，3D 版本票价普遍比 2D 版本要贵，且上座率更高。随着《沙丘》和《007：无暇赴死》两部影片的上映，2D、3D 之争愈演愈烈，在一定程度上已经影响了市场规则。</p><p><strong>观察：3D 两个极端，从追捧到抵制</strong></p><p>这种境况与 11 年前 3D 电影在国内市场备受追捧的境况截然相反。</p><p>2009 年 12 月，科幻电影《阿凡达》以 2D、3D 和 IMAX-3D 三种制式在北美上映，一个月后在中国首映。电影上映后，出现了 IMAX 3D 版及 3D 版本一票难求的情况，甚至出现少数影院趁势涨价、影院经理因播放故障给观众下跪等事件。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202111/618497dfb15ec06c9769c9da_1024.jpg" data-height="551" data-width="979" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/618497dfb15ec06c9769c9da_1024.jpg" referrerpolicy="no-referrer"></div></div>资料显示，这主要由于在 2010 年之前，国内只有 4500 多块电影银幕，3D 银幕更是只有 500 块。<p></p><p>更有统计数据显示，中国电影市场 2005 年到 2009 年的银幕增长率都不到 20%。而《阿凡达》的火爆，也直接助推了国内 3D 影厅的建设和 3D 观影需求的爆发。</p><p>电影市场的银幕增长率从 2010 年开始暴涨到 30% 以上，2011 年涨到了 48.4%，其中超过一半的增长都是支持 3D 银幕。截至 2013 年底，全国新增银幕 5077 块，银幕总数达 18195 块。</p><p>如今 11 年过去，国内 3D 电影市场，已经从当年一票难求到 3D 自由了，但是今天的观众，却不再对 3D 兴奋和趋之若鹜了。</p><p>相反，在 3D 市场的火爆之下，国内上映的 3D 版电影，出现了粗制滥造的问题，体验还不如 2D，强行打着 3D 噱头营销，导致观众较为反感。此外，3D 效果也不再成为稀罕，随着技术的发展，通过手机和电脑也能实现 3D 观影体验。</p><p>《科学辟谣》中专家提到，3D 电影的亮度不及 2D，主要是因为商业影院的 3D 眼镜左右镜片是两块特殊的镜片，在光学学科专业名称为 " 偏振片 "，它能起到滤光的作用，这就会使电影的亮度大打折扣。</p><p>此外，一些影厅为了节约成本采用单放映机，这种 3D 的画面亮度大概只有 2D 的 1/4-1/5。但一些高质量的 3D 影厅采用双放映机系统，配合上高亮度的激光光源放映机和高增益、高反射率的金属幕布后，3D 电影的画质和亮度得到了大大提升。所以一些带有 " 双机 "、" 激光厅 " 等字眼的厅要比普通厅观影体验更好，当然价格也更高。</p><p>过暗的银幕亮度、笨重的 3D 眼镜、抖动的镜头、以及高于 2D 影片的票价，让 3D 在很大的程度上，并没有提升观影质量，反而成为了鸡肋。</p><p>当然，一些电影还是适合 3D 观看的，比如有大量特效的科幻片，其立体的画面和真实震撼的效果更适合 3D 呈现；但是一般的剧情片和动作片，2D 与 3D 的效果差异并不大，反而 2D 电影画质更加细腻，票价相对低，更能提升观影体验。</p><p>从市场环境来看，一些伪 3D 电影，正在被市场摒弃。</p><p><strong>延展：伪 3D 积怨已久，难以为继？</strong></p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" data-original="http://zkres1.myzaker.com/202111/618497dfb15ec06c9769c9db_1024.jpg" data-height="651" data-width="938" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/618497dfb15ec06c9769c9db_1024.jpg" referrerpolicy="no-referrer"></div></div>" 该抵制的不是 3D 电影，而是后期转制的伪 3D"，在关于 2D、3D 版本的投票中，有 40% 的网友认同此说法。而有 37% 的网友认为，3D 版本的观看体验远不如 2D，只有 7% 的网友认为，3D 版本更有大片感、更炫酷。<p></p><p>实际上，中国观众对于转制 3D 积怨已久。此前《银翼杀手 2049》等电影上映的时候，就有不少抵制伪 3D 的声音。</p><p>和《007：无暇赴死》类似，《谍影重重 5》、《速度与激情》7、8、外传、9 等影片，也被环球影业出于 " 利益权衡 "，转制了 " 伪 3D 版本 "。</p><p>在这里，3D 已经成了商业化的一种手段，一方面为满足更多观众的需求，另一方面，同样一部影片，3D 版普遍比 2D 版票价高，这样影院的分成也会更高。</p><p>此前，《沙丘》导演丹尼斯 · 维伦纽瓦 9 月接受美版《时尚芭莎》采访时提到，"3D 是我和电影之间的过滤器 "，他还建议观众能够以 IMAX 2D 来体验该片，而不是 3D 这个 " 小玩意 "（gadget）。</p><p>这让不少观众意识到，影片 3D 版本并非作者创作，而是属于资本方、影院为迎合市场喜好的 " 宣发商业行为 "。没有 3D 效果的 3D 观影，只会徒增观影的不适感。</p><p>中国青年剧作家导演向凯表示，从观众抵制伪 3D 来看，观影人群的素质提升，对内容要求也越来越高。电影人制作人也应该不断加强提升专业水平和素质修养。别把观众再当 " 韭菜 " 来割。</p><p>如今，随着观众意识形态的变化，伪 3D 的生意，可能也做不下去了。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
