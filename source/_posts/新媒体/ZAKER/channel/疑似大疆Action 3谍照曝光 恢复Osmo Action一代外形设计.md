<p>前段时间有外媒传出消息，大疆可能会在今年 9 月发布全新的 Action 3 运动相机，最近网上也出现了一张可能是大疆 Action 3 运动相机的外观谍照，从谍照中进一步确认了 Action 3 沿用了 Osmo Action 一代的外观设计，DJI Action 2 的磁吸快拆模块化方案被舍弃了。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202209/6316bb968e9f091dae4a8335_1024.jpg" data-height="469" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/6316bb968e9f091dae4a8335_1024.jpg" referrerpolicy="no-referrer"></div></div>同时，从谍照中可以看到，大疆 Action 3 与 Osmo Action 一代一样配有方形前屏设计，镜头也都是圆形的造型，但还是有些不同，Osmo Action 一代的正面右下角配有散热孔位，但大疆 Action 3 却没有，显然机身散热方案上做了不小的调整。当然大疆之所以放弃 DJI Action 2 的磁吸快拆设计，主要原因应该也是散热方面的问题，此前已有不少用户抱怨 DJI Action 2 的机身过热问题，但磁吸快拆的方案显然对机身散热没有特别好的应对办法，大疆也必然会在 Action 3 的机身散热上进行优化。<p></p><p>不过目前还没有 Action 3 的配置信息，但规格上应该还是会比 Osmo Action 一代提升不少，有可能会继续沿用 DJI Action 2 的 1/1.7 英寸图像传感器。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
