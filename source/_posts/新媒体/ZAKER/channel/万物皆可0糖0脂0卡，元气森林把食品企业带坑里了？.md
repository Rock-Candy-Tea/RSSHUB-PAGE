传统的月饼。<p></p><p>近日，不少网友反映称，吃了知乎官方送的月饼后出现腹泻，对此知乎方面回应称，初衷是选购更健康的 " 低糖月饼 "，忽略了代替蔗糖的 " 麦芽糖醇 " 可能导致一部分人不耐受。</p><p>上述月饼是知乎为用户、KOL 等赠送的定制月饼，ZAKER 新闻注意到，该款月饼采用了麦芽糖醇、赤藓糖醇两款甜味剂代替传统月饼中的白砂糖、海藻糖等。</p><p>问题就出在糖醇上。知乎用户、中国澳洲注册营养师 " 初夏之菡 " 推算得知，知乎这款每块 75g 的月饼，糖醇总量在 15g 甚至更多。麦芽糖醇属于较低耐受度的糖醇，摄入过量易引起腹泻。</p><p>无糖、低糖月饼并不是什么新鲜东西，最早是厂家为糖尿病患者等特殊群体设计的产品。但在目前消费者追求 " 减糖 " 的背景下，万物皆可 "0 糖 0 脂 0 卡 "，盲目拼凑概念将部分食品饮料产品 " 带入误区 "。</p><p></p><div class="zaker_div"><div class="zk_h2_lrborder zk_lrbgcolor_red"><div class="zk_lrbordercon"><strong>无糖食品的兴起 </strong></div></div><div class="edi_oper"></div></div><p></p><p>人几乎天生就是嗜糖的。</p><p>在《盐糖脂：食品巨头是如何操纵我们的》一书中，作者迈克尔 · 莫斯提到，人类口腔的所有部位甚至包括上颚都嗜甜如命。口腔中每 10000 个味蕾就有一个特殊的甜味感受器，这些感受器与大脑中的 " 快乐地带 " ( pleasure zone ) 紧密相连。</p><p>食品巨头依赖低价的盐、糖、脂肪，将它们进行无数种组合对食品注入让人无法抗拒的力量。过度摄入盐糖脂对人类健康造成了威胁。</p><p>而随着现代社会对健康和形体的重视，嗜糖又是人性使然，食品加工企业利用人对 " 糖 " 又爱又恨的心理推出了 " 无糖 " 饮料，将蔗糖、果糖等易被人体吸收的原料换成了赤藓糖醇、麦芽糖醇等代糖。</p><p>在中国，" 无糖饮料 " 市场的引爆源于元气森林无糖气泡水的问世。</p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202109/613b2447b15ec0272f074a88_1024.jpg" data-height="1133" data-width="1700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202109/613b2447b15ec0272f074a88_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><p>在元气森林之前，不少企业都推出了无糖饮料，如统一旗下的茶里王、农夫山泉旗下的东方树叶，但由于口味不佳导致消费者不买账，茶里王退出了内地市场，农夫山泉的东方树叶甚至在 2016 年被列为 " 史上最难喝的饮料 "TOP5。</p><p>也正是在 2016 年，元气森林推出了无糖茶饮 " 燃茶 "，这款产品添加了赤藓糖醇作为甜味剂，其在 2019 年时的销售额占元气森林销售额的 30%。</p><p>由于人体内缺乏代谢赤藓糖醇的酶系，赤藓糖醇进入人体后不参与糖的代谢，几乎不会产生热量和引起血糖的变化，但又能满足人嗜糖的需求。</p><p>引爆无糖饮料市场的，是元气森林 2018 年推出的无糖气泡水，这款产品仍然使用了赤藓糖醇等代糖作为甜味剂，保持了接近蔗糖的口味。" 元气森林 " 让消费者知道，无糖饮料一样能 " 好喝 "。</p><p>一炮而红的元气森林气泡水带火了无糖饮料市场，在其推出的 2018 年，中国无糖饮料市场规模增长近 60%。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202109/613b2447b15ec0272f074a89_1024.jpg" data-height="664" data-width="842" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/613b2447b15ec0272f074a89_1024.jpg" referrerpolicy="no-referrer"></div></div>在看到元气森林的成功和无糖饮料市场的巨大发展空间后，不少品牌 " 跟风 " 涌入这个市场。<p></p><p>伊利、健力宝、农夫山泉、雀巢优活、喜茶、怡宝等品牌都推出了多款无糖饮料。此外，统一茶里王重回内地市场。</p><p>网易数读曾统计一份无糖饮料图鉴，目前甚至脉动、王老吉、椰树椰汁等产品都能找到无糖版本。</p><p><b></b></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><b><img class="lazy opacity_0 " id="img_3" data-original="http://zkres2.myzaker.com/202109/613b2447b15ec0272f074a8a_1024.jpg" data-height="1887" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/613b2447b15ec0272f074a8a_1024.jpg" referrerpolicy="no-referrer"></b></div></div><p></p><p>实际上，" 低糖 " 也是政策所倡导的选择。2017 年，国务院办公厅发布《国民营养计划 ( 2017-2030 年 ) 》，提出要推进 " 减盐、减油、减糖 " 的健康生活方式。</p><p>过度摄入糖的害处无需赘述，人的口舌之欲注定人类不能抗拒糖的诱惑，而对健康、形体美的追求越来越受到重视，无糖、低糖饮料与高盐糖脂食物一样，成为了食品企业的下一个利用点，甚至有的企业在产品上 " 强行拼凑 " 概念。</p><p></p><div class="zaker_div"><div class="zk_h2_lrborder zk_lrbgcolor_red"><div class="zk_lrbordercon"><strong>概念拼凑与悖论</strong></div></div><div class="edi_oper"></div></div><p></p><p>回到知乎的月饼，实际上，早在多年前就有 " 因食用稻香村无糖月饼导致腹泻 " 的相关报道，当时商家回应称，该款产品是为糖尿病患者等特殊人群设计的。</p><p>但所谓 " 无糖 "、" 低糖 " 月饼，对<b>追求形体的普通消费者</b>来说，似乎是一种悖论。</p><p>ZAKER 新闻注意到，尽管这款月饼中的糖换成了代糖，但其余原料如淀粉、油仍然是巨大的碳水和脂肪来源。</p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" referrerpolicy="no-referrer"></div></div><p></p><p>泛科普平台果壳指出，控制体重的关键是控制总能量的收支平衡，只有能量摄入小于支出，才能够实现减肥。</p><p>也就是说，低糖 / 无糖月饼不一定能够减少消费者的热量摄入，更何况月饼还是节庆期间食用的低频消费品，于普通消费者而言，" 低糖 / 无糖 " 更像是商家强行拼凑上去的概念。</p><p>实际上，这种概念拼凑的情况并不少见，如所谓 "0 糖 0 脂 " 低度酒。</p><p>ZAKER 新闻注意到，目前市场上不少低度酒都打着 "0 糖 " 的旗号，如淡真、RIO 强爽、JOJO 等。</p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" data-original="http://zkres1.myzaker.com/202109/613b2447b15ec0272f074a8c_1024.jpg" data-height="800" data-width="800" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202109/613b2447b15ec0272f074a8c_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><p>"0 糖 0 脂 0 卡 " 本就是消费者追求健康的同时过过嘴瘾的需求，在酒饮产品中打出这些概念，本身就是一个悖论，反而忽略了消费者摄入酒精以获取愉悦感的本质需求。</p><p>《柳叶刀》2018 年 8 月 23 日发表的一篇关于酒精的研究称，"Our results show that the safest level of drinking is none"，即：饮酒并没有所谓的 " 安全摄入值 "，只要摄入酒精，就对健康有害处。</p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_6" 或涉嫌虚假宣传。</p><p>酒是用粮食或水果酿造的，其原料本身就具有糖分热量，即使是用代糖替代了添加的蔗糖等，也不可能做到所谓 "0 糖 "。</p><p>据深燃财经报道，泡泡米酒联合创始人刘发禄在一场线下沙龙中也表示，" 酒本来就是粮食精，不可能没有能量，0 糖 0 卡是虚假宣传。"</p><p>值得一提的是，万物皆可 "0 糖 0 脂 0 卡 "、企业盲目跟风的本质是 " 创新能力微弱 "。</p><p>在元气森林气泡水大火后，不少品牌盲目跟风拼凑 "0 糖 0 脂 0 卡 " 的概念，无糖饮料市场出现同质化的问题，但一款产品的生命周期是有限的。根据 21 世纪产业研究院发布的《2021 水饮创新趋势报告》，口味性饮料产品一般 2～3 年走完生命周期。</p><p>此外，不少网红品牌都采用了代工的模式，元气森林早期也由健力宝代工，后期才筹建自有工厂。</p><p>在产品生命周期短、同质化严重的情况下，企业只有不断创新产品、完善自身供应链，才能够持续打造爆款，在 "0 糖 0 脂 0 卡 " 带来的红利消失后得以生存。</p><p></p><div class="zaker_div"><div
