<p>IT 之家 5 月 3 日消息，据 onMSFT 报道，你可能已经知道，微软将于今年 6 月 15 日停止支持 Internet Explorer。一些用户已经过渡到 Microsoft Edge 或第三方浏览器，现在更容易在 Windows 11 中将其设置为默认浏览器。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202205/6270aef58e9f09395c6e682b_1024.jpg" data-height="768" data-width="1024" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202205/6270aef58e9f09395c6e682b_1024.jpg" referrerpolicy="no-referrer"></div></div>微软再次站出来恳求用户在他们停止对 IE 的支持之前就进行过渡。并设置自己的停用日期，而不是等待实际停止日期。<p></p><p>通过博客文章，微软提供了一个详细的账户，用户可以通过该账户来实现相同的目的：</p><p>确保设置 IE 模式：微软鼓励组织设置和测试 IE 模式，并使用标识为依赖于 IE 的站点。这将有助于确定在 IE 退役后可能面临的任何兼容性问题。</p><p>将你的 IE 停用日期设置为组织模式：一旦你作为组织确信准备好更广泛地使用 IE 模式，你就可以设置一个日期来禁用 IE 策略。</p><p>让用户导入他们的数据：如果你已设置好所有内容，那么是时候通知用户更改了。微软提供了有助于过渡的资源，例如 IE 退役后最终用户采用的资源。</p><p>微软已经继续更新并推荐 Microsoft Edge，这将有助于为用户提供最佳体验。微软已经更新了浏览器的 IE 模式以促进这种预期的变化。" 要轻松导入数据，用户可以将 edge://settings/ importData 复制并粘贴到 Microsoft Edge 地址栏中，然后从 " 从 ... 导入 " 下的选项中选择 "Microsoft Internet Explorer"。这些步骤也包含在 IE 中退役最终用户采用资源电子邮件系列。"</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
