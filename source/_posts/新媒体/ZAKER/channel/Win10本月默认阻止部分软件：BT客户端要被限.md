<p>每次重装完新系统，大家可能会注意到 Win10 系统会提示某些软件不受欢迎，以前关掉就好了，然而从 8 月份起微软会默认开启这一功能，这会导致包括 BT 客户端在内的一些软件被禁用。</p><p><strong>01</strong></p><p><strong>Win10 系统默认拦截捆绑程序</strong></p><p>我们知道微软在 Win10 中捆绑了 Windows Defender 这个杀毒软件，这让 Win10 的安全有了更好的保证。</p><p>而在 Win10 2004 版本中，微软为 Windows Defender 增加了一个新的可选功能，可以拦截不需要的应用程序（Potentially Unwanted Apps，即 PUA）。</p><p>PUA 有时候可能不被认为是恶意软件，不过它们也带来了其他很多困扰。当你在互联网下载某个软件，可能就会捆绑着广告软件等 PUA。</p><p>根据微软的说法，PUA 包括广告软件、Torrent（仅限企业电脑）、挖矿程序、捆绑软件、营销软件等等，这让你的系统安装了额外的不需要的软件，非常烦人。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202108/610bb37a8e9f090a58507d6b_1024.jpg" data-height="337" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202108/610bb37a8e9f090a58507d6b_1024.jpg" referrerpolicy="no-referrer"></div></div>去年，微软在 Win10 2004 中增加了一项新功能，可以让你在使用 Windows Defender 时开启 SmartScreen 机制，这能够阻止潜在的不需要的应用程序安装。<p></p><p>在之前，SmartScreen 只出现在 Edge 浏览器上，下载软件的时候 Edge 浏览器通过 SmartScreen 扫描是否存在 PUA。</p><p>而 Windows Defender 支持 SmartScreen 后，对于 PUA 的筛选是系统全局的，这可以在整个系统中拦截你不需要的捆绑软件。</p><p>而现在，SmartScreen 的存在被进一步强化了。</p><p>在之前，它作为一个可选的集成功能存在于系统中，而微软最近宣布，在 Win10 2020 年 5 月更新之后的系统，会默认启用 SmartScreen 功能。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202108/610bb37a8e9f090a58507d6c_1024.jpg" data-height="259" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202108/610bb37a8e9f090a58507d6c_1024.jpg" referrerpolicy="no-referrer"></div></div>微软的支持文件显示，从 2021 年 8 月初开始，将会默认在 Windows Defender 中打开 SmartScreen 功能。<p></p><p>不过如果你按时不想开启 SmartScreen，例如想要避免误杀，也可以手动在系统中设置关闭，具体步骤如下。</p><p><strong>· 打开 " 设置 "。</strong></p><p><strong>· 选择隐私和安全性。</strong></p><p><strong>· 打开 Windows 安全中心。</strong></p><p><strong>· 在左侧选项栏中找到应用和浏览器控制选项卡。</strong></p><p><strong>· 找到 " 基于声誉的保护 " 设置选项。</strong></p><p><strong>· 启用或禁用 " 阻止可能不需要的应用 " 选项。</strong></p><p>" 阻止可能不需要的应用 " 功能有两个选项，分别是阻止本地的应用程序和阻止下载。</p><p>第一个选项启用，Windows 会检测设备上已经存在的本地文件、应用程序，而第二个选项则只在 Edge 浏览器中生效，可以阻止下载中的数据。</p><p>根据支持文件，Windows 的这个新的安全策略将会在 8 月初生效，这意味着新的机制可能已经开始分批推送了。</p><p>此外，微软还改善了系统的其他安全机制，例如为基于 Chromium 内核的 Edge 浏览器作了安全方面的改进，包括集成 Windows Hello 和 HTTPS-only 模式等等。</p><p><strong>02</strong></p><p><strong>BT 客户端要被限</strong></p><p>不受欢迎的应用程序 ( PUA ) 跟木马 / 病毒不同，通常没有明确的恶意，但是它可能安装其他不受欢迎的软件，更改数字设备的行为，或者执行用户未批准的活动或意料之外的活动。从 8 月份开始，微软表示禁用 PUA 软件就成为 Win10 系统的默认设置，用户运行某些软件或者下载的时候会被阻止，不需要该功能的则要自己去安全设置中取消勾选。</p><p></p><div class="img_box" id="id_imagebox_2" 应用的范围实在太广，很容易就被误伤。对老司机来说，BT 种子很常用，然而一些 BT 客户端就有可能被这个设置禁止，下片都会麻烦点。<p></p><p>小伙伴们，你们是想要系统的安全防御还是 BT 下载呢？</p><p>（编辑：张毅）</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
