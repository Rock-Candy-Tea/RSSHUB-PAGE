<p>日前，佳能公开了一项新专利，它旨在利用 USB Type C 接口作为风道，用外置散热器吸走相机内部热量，以解决相机过热问题。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202204/6256327d8e9f095a68117e6d_1024.jpg" data-height="835" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202204/6256327d8e9f095a68117e6d_1024.jpg" referrerpolicy="no-referrer"></div></div>在 2020 年佳能 EOS R5 上市之初，就爆出了过热问题，大大限制了相机 4K120P、8K30P 视频拍摄时长。其后，出现了多个第三方解决方案，希望结果过热问题，但是直到佳能推出了 EOS R5 改款—— EOS R5C，才用主动散热 + 去掉机身防抖的方式解决过热问题。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202204/6256327d8e9f095a68117e6e_1024.jpg" data-height="960" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202204/6256327d8e9f095a68117e6e_1024.jpg" referrerpolicy="no-referrer"></div></div>曝光的相机散热器专利显示，它带有一个 USB Type C 接口，但是该接口并非用作供电或是传输数据之用，而是作为风道之用，外置的散热器会利用风扇从 USB Type C 接口把处理器、CMOS 传感器产生的热量抽走，降低机内温度，又几乎不影响机身体积尺寸。不得不说，这是一个很有创意的想法，只是 USB Type C 接口是否太小了，实际操作时必须要非常高的风速才能带走热量，也许换成 HDMI Type A（大 HDMI 接口）是一个更好方法。此外，外置散热器会影响侧翻屏幕工作以及干扰其它接口使用，需要好好设计机身。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
