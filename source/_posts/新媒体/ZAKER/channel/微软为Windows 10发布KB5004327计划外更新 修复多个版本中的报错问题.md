<p><strong>微软为 Windows 10 发布了一个计划外的更新，用于解决操作系统 20H1、20H2 和 21H1 版本中的 0x80073D26 和 0x8007139F 错误。</strong>KB5004327 更新是针对那些在安装或打开 Xbox Games Pass 游戏时遇到问题的系统。任何受该问题影响的人在错误发生时都会被重定向到微软商店的游戏服务页面。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202106/60c604058e9f097ed2225b9e_1024.jpg" data-height="547" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202106/60c604058e9f097ed2225b9e_1024.jpg" referrerpolicy="no-referrer"></div></div>这个更新取代了微软之前发布的一个相当复杂的问题解决方案，最初的解决方法涉及使用 PowerShell 脚本来修复游戏服务，许多用户觉得这样做太困难。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202106/60c604058e9f097ed2225b9f_1024.jpg" data-height="382" data-width="534" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202106/60c604058e9f097ed2225b9f_1024.jpg" referrerpolicy="no-referrer"></div></div>该公司在 Twitter 上宣布了该更新的可用性，微软在 Windows 消息中心账号中表示：<p></p><p>计划外更新以解决阻止安装或打开 Xbox Game Pass 游戏的问题：</p><p>微软已经发现一个问题，影响到少数使用 Windows 10 20H1、Windows 10 20H2 和 Windows 10 21H1 版本的设备，并已加快解决，现在可以作为计划外更新使用。</p><p>这个更新解决了一个问题，即试图在你的 Windows 10 设备上安装或打开 Xbox Game Pass 游戏时可能会被重定向到游戏服务的微软商店页面。有关该问题的更多信息，请参阅 KB5004327。问题表现在当你在 Windows 10 设备上安装或启动游戏服务时，出现错误 0x80073D26 或 0x8007139F。</p><p>如果你没有遇到这个问题，就不需要安装这个更新。</p><p>正如微软在其推文中解释的那样，该更新可通过 Windows Update 和微软更新目录获得。如果你想通过 Windows Update 下载它，需要手动检查可选更新。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
