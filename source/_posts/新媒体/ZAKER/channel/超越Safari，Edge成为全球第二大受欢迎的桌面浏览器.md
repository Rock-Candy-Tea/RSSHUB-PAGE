<p>整理 | 章雨铭 责编 | 屠敏</p><p>曾几何时，编程语言界的 Java、C、C++，浏览器界的 Chrome、Safari、IE，被称之为各自领域中的 " 铁三角 "。殊不知，技术更迭，趋势已在悄然间发生了变化，此前的铁三角悄然间演变为如今的 Java、C 与 Python，以及 Chrome、Safari、Edge。更甚的是，在浏览器领域，据 StatCounte 的最新数据显示，谷歌的 Chrome 浏览器占比 66.64%，依然遥遥领先，<strong>但 Microsoft Edge 以 10.07% 的市场占比首次超过 Safari（9.61%），成为全球第二大受欢迎的桌面浏览器。</strong></p><p></p><div class="img_box" id="id_imagebox_0" onclick><div 增长份额，这可能与 Edge 目前的稳定更新以及与 Windows 的深度绑定有关。反观 Safari，作为苹果产品的默认浏览器，拥有很大的用户基数，最近 Mac 的销量依然在上升，但是市场占比却在下降，这说明在 Mac 上使用其他浏览器的人数增多。</p><p><strong>移动端表现平平</strong></p><p><strong>不过，纵观移动浏览器市场，Edge 可谓是 " 名落孙山 "，排名并未进入前六。</strong>虽然 Edge 在移动端的表现确实不算很出色，但主要的原因还是因为谷歌和苹果占据的安卓和 iOS 两大操作系统，所以其浏览器自然在移动端处于领先地位。在最新的数据中，Chrome 和 Safari 市占比分别为 63.57%、24.82%，留给其他浏览器的份额已经所剩无几。</p><p></p><div class="img_box" Chromium 的主要 Web 浏览器只有火狐和 Safari。2005 年，微软放弃了曾经风靡全球的 IE 浏览器，发布了一款全新的浏览器 Edge，并将其作为 Windows 10 的默认浏览器绑定推广。但是到了 2020 年，这款浏览器也惨遭抛弃，<strong>被与其同名的浏览器取而代之。</strong>值得注意的是，此 Edge 非彼 Edge，其二者除了名字相同，其余部分跟原来的 Edge 以及 IE 没有任何关系，<strong>新版 Edge 是基于谷歌主导的开源 Chromium 内核开发而成</strong>，核心功能方面和 Chrome 没什么两样。</p><p><strong>超越 Chrome 指日可待 ?</strong></p><p>有的网友非常看好 Edge 的前景，认为其有可能超越 Chrome。当然也有很多网友反驳说 Edge 是基于
