<p>近日，微软面向 Insider 渠道发布了 Windows 11 Build 22526，这是一个主要用户修复错误的版本，例如做了一些内部优化，以及改善了 AirPods 的体验等等。</p><p>这次系统更新的具体名称是 "Windows 11 Insider Preview 22526（rs_prerelease）"，用户打开 Windows 设置后哦，加入 Insider 计划的 Dev 通道，即可在几分钟内检查到该更新并下载安装。</p><p>据了解，Build 22526 并不和任意特定版本的 Win11 系统相关联，这意味着这个版本不一定会出现在正式版中，它只是作为一个分支供内部测试。</p><p>不过，它带来的改进会在下个大版本更新中向所有人开放，Win11 的下个大版本更新应该会在夏天确定功能，这个名为 " 太阳谷 2" 的更新会在今年下半年正式到来。</p><p>需要注意的是，Windows 11 Build 22526 是来自开发渠道的版本，它可能并不稳定，不推荐作日常使用。下面是 Windows 11 Build 22526 的一些更新内容介绍。</p><p>首先，微软在该版本中正试验一项新功能，当按下 "Alt+Tab" 键后，切换多任务的界面将会以窗口的形式呈现，而不是占满全屏。</p><p>其次，微软引入了对 AirPods 宽频语音的特性支持，这一变化将有利于 AirPods、AirPods Pro 或 AirPods Max。在功能预览更新后，在某些情况下，语音通话的音频质量将大大改善。</p><p>此外，Win11 的资源管理器也有所改进，文件位置索引更快，这样用户就能在文件资源管理器中快速找到重要文件。</p><p>对于企业用户，新版的 Win11 也有所改进，Windows 11 企业版上默认启用凭证卫士。如果是企业连接的 E3 和 E5 许可的 PC，这项特性会激活。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202201/61d9562fb15ec01ad717c76d_1024.jpg" data-height="337" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202201/61d9562fb15ec01ad717c76d_1024.jpg" referrerpolicy="no-referrer"></div></div>下面是 Windows 11 Build 22526 中的其他一些改进。<p></p><p>修复了在文件资源管理器中搜索，导致 explorer.exe 崩溃的问题。</p><p>修复了搜索弹窗错误的问题。</p><p>改进了搜索结果中出现的应用图标的分辨率。</p><p>修复了 Spotlight 集合的问题。</p><p>修复了小工具面板分辨率可能出错的问题。</p><p>修复了小工具面板可能会暂时出现空白的问题。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202201/61d9562fb15ec01ad717c76e_1024.jpg" data-height="449" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202201/61d9562fb15ec01ad717c76e_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
