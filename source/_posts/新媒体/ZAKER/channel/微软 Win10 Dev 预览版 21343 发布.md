安全的图标，讲述人图标，以及最近的记事本图标。现在，我们正在采取下一步措施，更新整个文件资源管理器中使用的许多系统图标。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " 之家获悉，请注意：升级到此版本后，任何固定在快速访问上的自定义文件夹将消失。你可以在更新到此版本之前将其记录下来，或者暂停更新，直到我们进行修复。<p></p><p>为了在产品显示文件中实现更大的一致性，微软做出了几项改变，例如文件夹图标的方向和默认的文件类型图标。值得注意的是，桌面、文档、下载和图片等顶层用户文件夹采用了新的设计，应该可以更容易地将它们一目了然地区分开来。另外，回收站图标也更新了。</p><p></p><div 中的图标有很多，而且随着时间的推移，更多的图标将继续更新。<p></p><p><strong>对 Windows Sandbox 和微软 Defender Application Guard（MDAG）的改进</strong></p><p>Windows Sandbox 和 Microsoft Defender Application Guard 都利用了 Windows 10 内置的特殊容器技术。此前，容器运行时与主机相匹配。从 Build 21343 开始，引入了针对容器场景设计和优化的新运行时。它是轻量级的，并允许 Windows Sandbox 和 Microsoft Defender Application Guard 更快的启动。</p><p>微软表示，预计这一变化不会影响 Windows Sandbox 内部的应用程序兼容性。然而，这是一个重大的变化，用户可能会遇到一些行为上的差异。</p><p>此外，Windows Sandbox 现在包括新的基于 Chromium 的 Microsoft Edge 浏览器（自 Build 21313 起）。</p><p><strong>更改和改进</strong></p><p>微软正在将 " 开始 " 中的 "Windows 管理工具 " 文件夹更名为 "Windows 工具 "。微软正在努力更好地组织 Windows 10 中所有的管理和系统工具。</p><p>【新闻和兴趣】更新推出：继上一次更新语言和市场后，本周微软还将把体验引入中国地区。微软将继续向 Windows Insiders 推出新闻和兴趣，所以目前还不是每个人都可以在开发频道中使用。</p><p>微软现在向开发频道中所有使用简体中文 IME 的 Windows Insiders 推出新的 IME 候选窗口设计。</p><p>微软正在更新触摸键盘中的 " 获取帮助 " 链接，现在显示 " 了解更多 "。</p><p>微软更新了文件资源管理器在重命名文件操作，现在支持使用 CTRL 左 / 右箭头在文件名中的单词之间移动光标，以及 CTRL Delete 和 CTRL Backspace 一次删除单词，就像 Windows 中的其他地方一样。</p><p>微软对 Windows 中与网络相关的表层进行了一些更新，使显示的符号使用最近在开发频道中添加的更新系统图标。</p><p>根据反馈，如果共享经验页面确定了你的账户连接问题，现在会直接将通知发送到行动中心，而不是需要驳回重复通知。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
