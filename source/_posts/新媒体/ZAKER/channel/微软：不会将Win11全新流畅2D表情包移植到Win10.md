<p>IT 之家 11 月 27 日消息，据 Windows Latest 报道，微软终于开始在生产渠道面向 Windows 11 用户推出基于 Fluent Design 的新表情符号。新的表情包体验是基于微软流畅设计，它与 Windows 11 的现代外观和感觉相匹配。这个新的 Fluent 表情包还支持 Unicode 表情符号 13.1 版或更高版本。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202111/61a1f86a8e9f092d9d7c6358_1024.jpg" data-height="392" data-width="696" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/61a1f86a8e9f092d9d7c6358_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家获悉，新的表情包体验与 Windows 11 KB5007262（Build 22000.348）一起推出，这是新的可选累积更新，未经用户许可，它不会下载或安装。<p></p><p>KB5007262 是可选的累积性更新，但微软表示，很快表情符号体验将作为 Windows 11 强制更新上线。换句话说，你可以期待这些新的表情符号在 2021 年 12 月的 " 星期二补丁 " 中发布。</p><p>不过，看起来微软并没有计划将这些现代表情符号带到 Windows 10。全新表情符号仍将是新 Windows 11 操作系统的专属，而 Windows 10 用户将不会得到任何新的表情符号，至少目前是这样。然而，这可能会在未来发生变化。</p><p>与以前 Windows Phone 时代的表情符号不同，全新的表情符号看起来很现代，色彩丰富，更有趣。然而，新的表情符号看起来与微软之前预告的 3D 刷新版有很大不同。Windows 11 的新表情符号实际上是 2D 的，而不是 3D 的。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
