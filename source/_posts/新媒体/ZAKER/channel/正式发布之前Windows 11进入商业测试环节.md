<p>在官宣 Windows 11 将于 10 月 5 日正式发布后，<strong>今天微软已经邀请企业用户测试和验证新系统。</strong>这是 Windows 11 发展中的一个重要里程碑，它意味着企业 / 商业客户正在测试新的操作系统，以便在他们的组织中进行部署。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202109/6131783a8e9f097986116cbf_1024.jpg" data-height="394" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/6131783a8e9f097986116cbf_1024.jpg" referrerpolicy="no-referrer"></div></div>参加 Windows Insider Program for Business 的组织应该在 Windows Update 中看到一个新的通知。除了 Windows Insider 计划，Windows 11 现在还可以通过 Windows Server Update Services（WSUS）和 Azure Marketplace 获得。<p></p><p>Windows 11 是完全可选的，用户可以选择继续使用他们当前的 Windows 10 版本，如 21H1 或 20H2 功能更新。或者他们也可以选择升级到 Windows 10 21H2 版本（2021 年 10 月更新），并在未来几个月继续接受质量改进。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202109/6131783a8e9f097986116cc0_1024.jpg" data-height="374" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/6131783a8e9f097986116cc0_1024.jpg" referrerpolicy="no-referrer"></div></div>如果你的硬件符合升级条件但是不想要升级，那么可以选择 " 暂时停留在 Windows 10" 选项。当你选择这个选项时，将提供 Windows 10 21H2 版本来代替。如果你的设备不符合硬件要求，上述通知将不会出现。<p></p><p>与 Windows 11 不同，Windows 10 的 21H2 版本是一个小版本，有一般的改进和一些针对企业的新安全功能。对于 Windows 10 用户来说，并没有一套大规模的变化计划。Windows 11 测试的下一步也是最后一步将是面向普通消费者的 Windows Insider 计划的发行预览版渠道。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
