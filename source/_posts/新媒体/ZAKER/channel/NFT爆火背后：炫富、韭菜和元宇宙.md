的正确打开方式。<p></p><p><strong>文</strong>｜ 邱晓芬</p><p><strong>编辑｜</strong>苏建勋</p><p><strong>封面来源 | </strong>IC photo</p><p>一幅电子图片，最多能卖多少钱？</p><p>答案是 6930 万美金。数字艺术家 Beeple 不久前就在佳士得上拍卖出了这样的一张图片：这幅名为《每一天：最初的 5000 天》（everyday:the first 5000 days）的图片，由 Beeple 在过去十三年里每天的画作拼贴而成。</p><p>制作成 NFT 后，这幅图片从 100 美元起拍，最后以 70 万倍的价格成交。很难想象，在搭上 NFT 快车之前，Beeple 只是一个默默无闻笔耕不辍的画师，作品最高只能卖出几百美元，而如今，《每一天》这幅画的售价基本是在世艺术家作品拍卖价的第三名。</p><p>这位 40 岁的美国画师一夜间财富自由的背后，NFT（Non-Fungible Token，非同质化代币）其实不是一个新鲜的名词，但要理解往往耗费一些想象力。</p><p>一张百元大钞的价值只是一百元，但是如果上面多了科比的签名，那价值就会超越 100，并且难以估量。如果说原先的比特币等等（FT）就像这张百元大钞一样代表着一种货币形式的话，那 NFT 就代表着一种独一无二、无法被拆分的资产。</p><p>佳士得的这次拍卖只是在艺术圈里震出了水花，名人效应的加持还在加速 NFT 出圈。</p><p>一边是以太坊狂热爱好者们践行信仰。不久前，推特创始人杰克 · 多西把自己的第一条推特作为 NFT 拍卖，最后成交价 290 万美元；" 币圈教主 " 马斯克和女朋友，甚至在短短 20 分钟内便以近 600 万美元的价格，拍卖出了 NFT 形式的歌曲。</p><p>另一边，篮球明星库里、影视明星徐静蕾等等名人也患上了收集 NFT 头像的兴趣，纷纷在 ins 换上了高价抢购的头像——库里最新的头像，一只由马赛克色块拼接成的猴子样式，就耗费了 20 万美金。</p><p>NFT 的余波还在持续，互联网大厂、时尚品牌们在接连闯入，越来越多的品牌和 IP 在探索代币化，支付宝、腾讯、LV.....</p><p>当局中人疯狂迷恋 NFT，并为之支付天价费用，大家着迷的到底是什么？</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " 万美金，买一个身份认证</strong><p></p><p>不是所有人都能理解花几十万美金买一副 NFT 头像的乐趣——虽然背靠区块链技术的 NFT 作品，一切但是可溯源的，但虚拟世界的数字拥有权，并没有办法和现实世界的所有权划上等号。</p><p>举个例子，推特创始人的第一条微博，虽然属于那位愿意花重金买下的人，但其他人依旧可以对这条推特进行转发 / 评论 / 点赞；</p><p>库里重金买下的那个马赛克色块 INS 头像，也可以被网络上所有人复制粘贴，甚至是使用。似乎买下了这些作品的人，<strong>只能在以太坊的环境下才能证明自己才是这幅作品真正的拥有者。</strong></p><p>但在 NFT 狂热爱好者看来，这种割裂并不要紧。一名币圈人士向 36 氪分析，币圈已经有了分级，早期入行的，已有少数人赚到了特别多的钱，对于这些人来说，<strong>NFT 头像是一种身份的象征，" 用来装点门面，显得有钱又有趣。"</strong></p><p>这种炫耀之所以成立，是因为 NFT 虽然已经打开了认知，并不是所有人能上牌桌的大众游戏。</p><p>目前最大的 NFT 平台是 "OpenSea"，按照规则，买家需要给平台支付交易额的 2.5% 作为手续费。<strong>每一次交易，都是一次 ETH 的燃烧</strong>。对于一些手头并不算十分宽裕的币圈人来说，尤其是在经历了 5 月份的闪崩之后，NFT 平台的这种交易规则，相当于给很多人设置了无形的准入门槛。</p><p>" 一般要有钱有闲的人才玩得起 "，另一位币圈人士直言。</p><p>不管是换上 NFT 头像，还是入手数字艺术作品，这种 " 高贵 " 的身份标签如何维系，很大程度依赖于<strong>小圈子里面的集体共识</strong>。前述人士告诉 36 氪，一般默认的规则是，如果有人用上了未属于自己购买的 NFT 头像，大家会认为是一件很羞耻的事。</p><p>这看起来虽然有种自欺欺人的荒诞感，如果从收藏品的角度来看，也不是无法理解的。现实生活中收藏古董 / 字画等等的行为，映射到虚拟世界中也不稀奇。收集 NFT 的币圈人，<strong>很大程度还有投资增值的考虑。</strong></p><p>早期入手 NFT 的玩家早就制造出了一个个财富故事，且不费吹灰之力。</p><p>2017 年，以太坊就推出过一个名叫 "CryptoPunks" 的像素头像项目，只要是拥有以太币的人都可以免费领取一个头像，上限是一万，每人限领一个，用户领完后就可以放到二级市场交易。</p><p></p><div 头像未来可能会涨，大家如果不介意立马变现，有看到喜欢的，就先买再说 "，前述币圈人士表示。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" 的故事里。</p><p>有消息指出，竞拍到《每一天》这一作品的买家，不是收藏家，而是一位资深币圈玩家。</p><p>这位深谙市场运作技巧的买家，早在拍卖前的几个月，已经快速发布了名为 "B20" 的代币，打着 " 让艺术品收藏更民主 " 的口号，这个项目和艺术品收藏有了强关联，还在此前斥资百万买入了 Beeple 的多幅作品。</p><p>在其中标后，这 1000 万枚代币的价格，从 0.36 美元一路飙升到了 28 美元的最高位。而这位幕后玩家，通过这种<strong>关联交易的方式，在这场竞拍当中不仅赚到了噱头，而且扣除拍卖支付的费用，反而还倒赚了一大笔钱。</strong></p><p>有意思的是，在这场拍卖当中，同样擅长炒作的孙宇晨也参与其中，还是出价排名第二位的买家。</p><p>在明星效应、令人咋舌的财富故事刺激下，越来越多的人被吸引上牌桌。各路数据指出，<strong>NFT 卖家和买家的数量在持续扩大。</strong></p><p>而只有当更多的人加入，这场击鼓传花的金钱游戏才能得以成立。在任何一场零和博弈当中，一旦有人赚钱，就意味着有人亏钱。但由于没有人能准确猜中谁才是幸运儿，这才是这场击鼓传花游戏的致命诱惑。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div NonFungible<p></p><p>有人还在观望，有人已经迫不及待离场。</p><p>既得利益者 Beeple 本人，一度直言，"NFT 市场的泡沫太大 "，所以在拍卖结束后，他快速将交易所得的数字货币，兑换成了实实在在的钱，落袋为安。</p><p>一位亏钱的网友在推特上直截了当—— "<strong>NFT 连基本的二八法则都不遵守，</strong>1% 赚钱，99% 都是赔钱的 "。</p><p>不过，热浪还在延续。数据显示，"Rarible"、"Opensea" 这些主要 NFT 交易平台的销售额，在今年第二季度，增长了 50 到 100 倍。其中，Opensea 的累计销售额已经超过 1 亿美元。</p><p>陡然向上的增长曲线也在撩动着资本涌入。一个切面足以证明 NFT 这把火有多猛烈：就连这些中间交易平台也在疯狂圈钱融资。今年起，各大 NFT 平台飞速完成千万美元级别以上的 A 轮融资：</p><p>3 月份，OpenSea、SuperRare 先后宣布完成 2300 万美元、900 万美元 A 轮融资；</p><p>随后，NFT 平台 Bitski、Rarible 也快速跟进 1900 万美元、1420 万美元的 A 轮融资；......</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_6" data-original="http://zkres1.myzaker.com/202105/60b18627b62b827d466da83c_1024.jpg" 的正确打开方式</strong><p></p><p>拨开 NFT 外壳的那些荒诞的炒作、暴富的故事，其实 NFT 的意义也是有目共睹的。</p><p>数字艺术是 NFT 落地最快的应用之一。借助 NFT 平台，艺术家可以通过将作品货币化，从而直接链接受众，寻找到更丰富的变现方式。此外，数字艺术可复制性很强，而在 NFT 的方式下，" 真伪 " 的概念被粉碎，因为所有的产品都是独一无二的，因此也减少了版权问题的困扰。</p><p>游戏玩家可能是最能适应 NFT 这种模式的群体。由越南游戏开发商 Sky Mavi 开发的链游项目 Axie Infinity，每日的收入到达 111.9 万美元——很少有人知道的是，王者荣耀的日均收入只是 84 万美元，<strong>这款 NFT 游戏默默成为全球最赚钱的游戏。</strong></p><p>不止是眼前的这些。随着 " 元宇宙 " 概念重新火起来，<strong>NFT 应用还在逐渐走向更庞大的体系。</strong></p><p>作为 web2.0 的延续，元宇宙构建的是一个与映射现实的虚拟世界，隐藏着下一代社交、在线办公、电商的形态。人工智能、5G 和 VR 这些老生常谈的技术只是基建，<strong>要真正成熟运转起来，还需要真正产生商业经济活动。</strong></p><p>在区块链的价值体系中，NFT 作为非同质化代币，可以代表所有的独一无二的资产，收藏品、游戏装备、数字艺术品等等。这可能是同质化代币（FT）所无法实现的。</p><p>基于 NFT，用户在元宇宙的世界里做交易，构建属于自己的所有权世界，<strong>元宇宙才得以加速沉淀应用场景，构造社区，养成交易规模。</strong></p><p>NFT 的最初，是源自于 1993 年工程师哈尔 · 芬尼分享的一个 " 加密交易卡 " 的概念。他当初的想法是，这些由抽象字符构成的独特艺术，会被密码学狂热爱好者爱上，还会成为好友之间的谈资，勾起人们的收藏欲，进而衍生出更多有意思的新玩法。</p><p>现在看起来，这些想法都被应验了，只是来的有点慢。</p><p>他本人，在提出概念后的 16 年后，2009 年，才真正从中本聪处拿到世界上第一笔比特币转账；直到 2017 年，" 加密猫 " 的出现才引发了第一次的 NFT 热潮；2021 年，NFT 才出有出圈的趋势，被广泛讨论。</p><p>可能如今依旧有人无法理解：NFT 为何现在才火？NFT 收藏品流动性不强，持有这些资产的意义何在？NFT 可以被复制，为什么还要花高价买？</p><p><strong>或许放到元宇宙这个开放世界里面，所有就说得通了。</strong></p><p>《头号玩家》电影曾经对未来世界做出了美好预言。在绿洲（Oasis）这个世界里，每个人都有自由的角色，在这里，不同次元的影视游戏经典角色汇聚，丰富的数字内容和物品也可以随意流通。在这个浮华的虚拟世界里，一个在现实世界中的 loser，也可以成为超级英雄。</p><p><strong>" 每个梦想的背后，都藏着一个精力旺盛且勇于冒险的人。除了惊喜，他们什么都不期待 "。</strong></p><p>哈尔 · 芬尼所说的，可能是现下 NFT 狂热的最好注解。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
