<p>IT 之家 10 月 12 日消息 Windows 11 现在正在分阶段推出，可以正式安装在受支持的硬件上。根据微软的一份声明，Windows 11 为 Windows 10 设备提供免费升级，但是不支持的设备可能没有资格获得未来的累积更新。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202110/6164df7c8e9f0953d75325bf_1024.jpg" data-height="392" data-width="696" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202110/6164df7c8e9f0953d75325bf_1024.jpg" referrerpolicy="no-referrer"></div></div>尽管如此，很明显，大多数 Windows 10 设备是符合运行 Windows 11 的性能要求的，这是因为 Windows 11 本质上是基于 Windows 10 打造的，基本上都是关于设计的改进。然而，微软正在强制执行严格的硬件要求，以提供绝对的最佳安全体验。<p></p><p>在一个新的视频中，微软强调了 Windows 11 所要求的安全功能的重要性。据微软称，Windows 11 在设计上非常注重安全问题。Windows 11 需要可信的平台模块（TPM 2.0）、安全启动、基于虚拟化的安全（VBS）等，这些安全功能通常包含在较新的设备中，也提高了升级的门槛。</p><p>微软进一步解释说，他们希望通过 Windows 11 为个人电脑安全设定一个标准，旧设备不被支持是因为它们缺乏所有这些安全功能。</p><p>值得注意的是，TPM 或安全启动在 Windows 10 上也是有的，但它们现在被强制执行，以确保 Windows 11 用户的安全体验。TPM 是物理或虚拟芯片，用于保护加密密钥、用户凭证和其他敏感数据。</p><p>微软说：" 对于 Windows 11，我们要求新安装的系统默认使用 TPM 2.0，过去五年中制造的大多数计算机均支持。"</p><p>" 如果一台机器被入侵，我们可以帮助你把损失降到最低。Windows 11 将阻止这些类型的攻击，因为我们正在使用安全启动和可信启动，它同时使用所需的 UEFI 和 TPM 硬件。" 微软公司补充说：" 安全启动和可信启动能阻止试图修改你的早期启动文件的 rootkits 或 bootkits。"</p><p>在制定严格的要求之前，微软对旧硬件进行了测试，以确定与 TPM 等安全功能的兼容性。基于这些，微软得出结论：在新的或支持的设备上运行 Windows 11 是很重要的。</p><p>IT 之家了解到，微软还对旧硬件进行了一系列测试，以确定用旧处理器运行 Windows 11 时的性能问题。微软的分析显示，旧设备会遇到性能问题，用户可能会经常遇到应用程序崩溃。不支持的个人电脑可能会比符合要求的电脑多出现 52% 的蓝屏，在较新的设备上，使用现代驱动更新管理和 OEM 厂商发布的现代驱动，蓝屏错误已经减少，而这些在旧设备上是没有的。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
