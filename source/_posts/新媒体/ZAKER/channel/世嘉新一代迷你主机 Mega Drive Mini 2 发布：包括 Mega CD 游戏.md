<p>IT 之家 6 月 3 日消息，世嘉现公布了新一代迷你主机 —— Mega Drive Mini 。这款新主机基于 Mega Drive 2 打造而来，将于 10 月 27 日发售，日本定价 9980 日元（约 512.97 元人民币），不过世嘉暂未宣布是否会在其他地区发售。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202206/6299f9b58e9f090fab781d3e_1024.jpg" data-height="488" data-width="1024" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202206/6299f9b58e9f090fab781d3e_1024.jpg" referrerpolicy="no-referrer"></div></div>据悉，这款新主机将收录 50 款 Mega Drive 和 Mega CD 游戏，这其中有街机移植作品，也有 " 神秘新作 "。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202206/6299f9b58e9f090fab781d3f_1024.jpg" data-height="414" data-width="736" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202206/6299f9b58e9f090fab781d3f_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家了解到，第一款 Mega Drive Mini 于 2019 年推出，预装了 42 款游戏，包括《刺猬索尼克》、《愤怒的街道 II》和《变色龙》，并捆绑了 2 个用于多人游戏的有线控制器。<p></p><p>迄今为止已确认的 Mega Drive Mini 2 游戏包括：</p><p>Silpheed ( Mega CD ) </p><p>Shining Force CD ( Mega CD ) </p><p>Sonic CD ( Mega CD ) </p><p>Mansion of Hidden Souls ( Mega CD ) </p><p>Popful Mail ( Mega CD ) </p><p>Virtua Racing ( Mega Drive ) </p><p>Bonanza Bros ( Mega Drive ) </p><p>Shining in the Darkness ( Mega Drive ) </p><p>Thunder Force IV ( Mega Drive ) </p><p>Magical Taruruto ( Mega Drive ) </p><div id="recommend_bottom"></div><div id="article_bottom"></div>
