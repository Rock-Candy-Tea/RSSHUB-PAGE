<p>Windows 12 开发版本截图曝光</p><p>网络上日前曝光了微软新一代 Windows 系统的截图，版本号是 Windows 12 Build 12.0.30000，从标记来看仅限于微软内部测试。</p><p>但 Windows 12 还很遥远，预计最早也要到 2026 年才能开始测试，也就是 Windows 11 发布 5 年之后才有望升级，现在还早。</p><p>然而这个截图的真实性遭到了很多人的质疑，因为实在有点敷衍，所谓 Windows 12 系统的背景及控制面板的设置还是 Windows 7 系统的，PS 都不够用心。</p><p>但是话说回来，在微软自己打破承诺，推出了 Windows 11 系统之后，Windows 12 系统的存在几乎没法质疑了，反正微软这次没说 Windows 11 是最后一代 Windows 系统，后续显应该恢复之前的 5 年升级一代模式。</p><p>微软再次提及 Windows 11 系统配置要求</p><p>Windows 12 即使存在，也是几年之后的事情了，这几年，Windows 11 才是绝对的主流。</p><p>Windows 11 目前可用于测试，Windows Insider 计划中的用户可以下载早期预览版本。Windows 11 的稳定版将在今年晚些时候推出，用户将能够免费升级，但这只有在你有合适的硬件的情况下才会如此。正如你可能知道的那样，Windows 11 带有新的最低系统要求，那些拥有不兼容硬件的人将只能使用 Windows 10。</p><p>新的要求对于使用旧笔记本和 PC 的人来说是个坏消息，微软再次澄清，对于使用不支持的硬件的用户来说，所有的 Windows 11 升级路径都将被阻止。这以消息是在关于 Windows 11 升级路径和商业 / 企业部署工具的新问答中解释的。</p><p>" 我们知道有些人没有资格使用 Windows 11，这很糟糕，但需要记住的是，我们这样做的原因是为了让设备更有生产力，并 [ 确保他们拥有 ] 比以往任何时候都更安全的状态，以便他们能够在这个新的工作环境中保持保护，" 微软说。</p><p>微软将能够利用遥测数据确定设备的兼容性，而 Windows Update 则负责告诉用户是否可以安装 Windows 11。</p><p>" 我们不希望所有的管理员或所有的消费者不得不不断地考虑 [ 关于兼容性 ] ，" 微软方面表示。" 我们所做的实际上是把一些强制执行的东西带进了计算机，所以与其说你不得不考虑这个问题，不如说你的设备实际上能够在被提供 Windows 11 和浪费时间去下载安装之前自行确定 "。</p><p>这意味着很可能 Windows 11 兼容性错误提示将直接出现在企业的 Windows Update 或 Windows Update 管理工具中。与 Windows 10 更新保障的性质不同，用户不能通过修改组策略绕过这些限制。</p><p>" 组策略将无法使你绕过 Windows 11 的硬件强制执行。我们将阻止你将设备升级到不支持的状态，因为我们真的想确保你的设备保持支持和安全，" 该公司澄清说。</p><p>目前，只有英特尔第八代、Zen 2 CPU 或更新的芯片与 Windows 11 兼容，但第七代 CPU 也有可能得到支持。</p><p>最大的争议来自于 Windows 11 还需要 TPM 2.0，这是一种安全功能，现在已经大规模内置于新发布的笔记本电脑和台式机中。值得庆幸的是，暂时测试版系统的用户能够通过修改注册表绕过 TPM 2.0 的要求，但是安装新操作系统的最低 CPU 要求是强制性的。</p><p>看到这里，似乎，升级 PC 或者购买新电脑还是很有必要的感觉。</p><p><strong>01</strong></p><p><strong>Windows 12 开发版本截图曝光</strong></p><p></p><div class="img_box" id="id_imagebox_0" onclick><div
