<p>IT 之家 8 月 19 日消息 今天微软推送了 Windows 10 Build 19044.1200（21H2）发布预览版，还宣布向正在使用 Windows 10 21H1 版本的发行预览用户推送了 Build 19043.1200（KB5005101）。这个构建版本包括新的错误修复。</p><p></p><div class="img_box" id="id_imagebox_0" data-width="1200" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/611dc6138e9f096c196f300f_1024.jpg" referrerpolicy="no-referrer"></div></div>下面是 Windows 10 Build 19043.1200 更新内容<p></p><p>修复和改进</p><p>修复了使用户无法跟踪分布式组件对象模型（DCOM）的激活失败的问题。</p><p>修复了可能导致 Windows 远程管理（WinRM）服务在高负载时停止工作的线程问题。</p><p>修复了导致 Windows Management Instrumentation（WMI）提供者主机进程停止工作的问题。这是因为在使用期望状态配置（DSC）时出现了未处理的访问违规。</p><p>修复了导致存储在不同卷上的分布式文件系统（DFS）路径之间的文件迁移失败的问题。当你使用 Move-Item 命令的 PowerShell 脚本实施迁移时，会出现这个问题。</p><p>修复了在发生低内存条件后，你无法向 WMI 存储库写入的问题。</p><p>修复了在高动态范围（HDR）显示器上重置标准动态范围（SDR）内容亮度的问题。这发生在你重新启动系统或远程重新连接到系统之后。</p><p>修复了可能导致外部显示器在休眠后显示黑屏的问题。当外部显示器使用某种硬件接口连接到扩展坞时，这个问题可能发生。</p><p>修复了当你在 VBScript 中使用嵌套类时发生的内存泄漏的问题。</p><p>修复了在开箱体验（OOBE）过程中，你无法在用户名框中输入任何文字。这个问题在你使用中文输入法编辑器（IME）时发生。</p><p>修复了导致使用匀场的应用程序停止工作的问题。这个问题发生在没有安装 edgegdi.dll 的设备上。错误信息是：" 由于没有找到 edgegdi.dll，代码执行无法进行 "。</p><p>修复了可能使你无法最小化使用非主题窗口的应用程序。</p><p>修复了可能导致你的设备在触摸输入手势期间停止工作的问题。如果你在做手势的过程中让更多的手指接触到触摸板或屏幕，就会出现这个问题。</p><p>修复了一个调整图像大小的问题，该问题可能会产生闪烁和残余的线条伪影。</p><p>修复了在 Office 365 应用程序中复制和粘贴文本框的问题。IME 会阻止你在文本框中插入文本。</p><p>修复了一个阻止 USB 音频耳机在支持 USB 音频卸载的笔记本电脑上工作的问题。如果你在笔记本电脑上安装了第三方音频驱动程序，就会出现这个问题。</p><p>修复了当在代码完整性策略中指定包族名称规则时，该问题使代码完整性规则无法正确工作。这个问题的发生是因为对大小写敏感的名称的处理不正确。</p><p>修复了使 ShellHWDetection 服务无法在特权访问工作站（PAW）设备上启动，并使你无法管理 BitLocker 驱动器加密的问题。</p><p>修复了 Windows Defender Exploit Protection 中的一个问题，使一些 Microsoft Office 应用程序无法在具有某些处理器的机器上工作。</p><p>修复了导致 IME 工具条即使在关闭远程应用程序时也会出现的问题。</p><p>修复了当你配置策略 " 在系统重启时删除超过指定天数的用户配置文件 " 时，可能会出现该问题。如果用户登录的时间超过了策略中指定的时间，设备可能会在启动时意外地删除配置文件。</p><p>修复了微软 OneDrive 同步设置 " 始终保留在此设备上 " 的问题。在你安装 Windows 更新后，该设置会意外地重置为 " 仅已知文件夹 "。</p><p>修复了当用户取消日语重新转换时，会提供错误的古文假名结果。</p><p>修复了一个罕见的情况，它使蓝牙耳机无法使用高级音频分配规范（A2DP）连接音乐播放，导致耳机只能用于语音通话。</p><p>增加了 " 目标产品版本 " 策略。有了这个，管理员可以指定他们希望设备迁移到或保持在哪个 Windows 产品上（例如，Windows 10 或 Windows 11）。</p><p>增加了本地安全授权（LSA）查询缓存中的默认条目数，以改善高查询量情况下的查询性能。</p><p>修复了在原地升级期间可能会创建重复的内置本地账户的问题，如管理员或访客账户。如果你之前重命名了这些账户，这个问题就会发生。因此，本地用户和组 MMC 快照（msc）在升级后出现空白，没有账户。此更新从受影响机器上的本地安全账户管理器（SAM）数据库中删除重复的账户。如果系统检测到并删除了重复的账户，它会在系统事件日志中记录一个目录服务 -SAM 事件，ID 为 16986。</p><p>修复了 srv2！Smb2CheckAndInvalidateCCFFile 中的一个停止错误 0x1E。</p><p>修复了可能导致传输验证失败的问题，错误是 "HRESULT E_FAIL 已从对 COM 组件的调用中返回 "。当你使用 Windows Server 2008、Windows Server 2008 R2 或 Windows Server 2012 作为源时，这个问题会发生。</p><p>修复了可能导致重复数据删除过滤器检测到 reparse 点中的损坏后系统停止工作的问题。这个问题的发生是由于以前的更新中引入的重复数据删除驱动程序的变化。</p><p>修复了使用带有备份选项（B）的 Robocopy 命令来修复数据丢失的问题。当源位置包含分层的 Azure 文件同步文件或分层的云文件时，会出现这个问题。</p><p>停止从过时的存储健康功能中运行针对 OneSettings APIs 的查询。</p><p>启用了 1400 多个新的移动设备管理（MDM）策略。通过它们，你可以配置组策略也支持的策略。这些新的 MDM 策略包括管理模板（ADMX）策略，如 App Compat、事件转发、服务和任务调度器。从 2021 年 9 月开始，你可以使用微软端点管理器（MEM）设置目录来配置这些新的 MDM 策略。</p><p>已知问题</p><p>在你下载一个可选的更新后，Windows Update 设置页面可能会挂起。如果你遇到这种情况，请关闭并重新打开 Windows Update 设置页面。</p><p>IT 之家获悉，要检查并查看你是否在运行 Windows 10 21H1 版本，只需在任务栏上的 Windows 搜索框中输入 "winver"。它应该显示为 "21H1 版 "。</p><div id="recommend_bottom"></div><div
