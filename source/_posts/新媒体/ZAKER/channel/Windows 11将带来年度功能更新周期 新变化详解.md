<p>当 Windows 10 在 2015 年首次推出时，由于微软曾经夸下海口说这是 " 最后一版 Windows"，因此我们很长时间以来不知道下一个版本的 Windows 会被称为什么。6 月 24 日，微软正式宣布 Windows 11 为下一代 Windows，新的操作系统将在今年晚些时候开始向部分用户推出。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202107/60e176968e9f0950666cc8b2_1024.jpg" data-height="467" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/60e176968e9f0950666cc8b2_1024.jpg" referrerpolicy="no-referrer"></div></div>在 Windows 10 推出后，微软一直按照春季 / 秋季的时间表进行一年两次功能更新，操作系统的开发也与 Azure 云服务的演变进度保持一致。Windows 作为一种服务经过多年的发展，现在以 Windows 11 的形态将在今年晚些时候取代 Windows 10。<p></p><p>随着 Windows 11 的推出，微软将转向年度功能更新的节奏，这意味着一年只发布一个功能更新，改变了目前一年两次的时间表（Windows 10 为半年一次，上半年打磨质量，下半年增加新功能）。据微软官员称，第一个大更新将在明年下半年发布。</p><p>微软支持新版本 Windows 11 的时长为 24 个月（消费者版），但对于企业版和教育版将有 36 个月的支持。与 Windows 10 一样，Windows 11 也将继续接收每月的累积更新，也就是所谓的 " 补丁星期二 " 或 "B" 版本。</p><p>对于那些感兴趣的热心用户，微软也将发布 Windows 11 的可选累积更新，这些可选的更新将首先提供给 Windows Insider 计划的发布预览频道的测试者。</p><p>Windows 更新将继续像现在的 Windows 10 那样交付，但微软说这些累积更新将 " 缩小 40%"。微软说，它致力于 " 对 Windows 11 的投资 "，用户可以期待更快的更新过程，可靠和高效的体验。除了这些质量改进，微软还在试验一项新功能，它将告诉用户设备重新启动和安装更新预计需要多长时间。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202107/60e176968e9f0950666cc8b3_1024.jpg" data-height="158" data-width="292" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60e176968e9f0950666cc8b3_1024.jpg" referrerpolicy="no-referrer"></div></div>这项新功能将适用于所有的 Windows 更新；.NET 框架、周二补丁和可选更新。显然，对于不需要重启系统的更新，它将不会显示出来。<p></p><p>微软对 Windows Update 的估计时间将出现在开始菜单和设置界面中，安装时间将基于处理器、内存和存储等硬件。</p><p>在目前的预览版中，Windows 错误地报告了五分钟左右的估计时间，导致它的参考意义并不大，但这个功能目前正在完善中，它最终会变得更好。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
