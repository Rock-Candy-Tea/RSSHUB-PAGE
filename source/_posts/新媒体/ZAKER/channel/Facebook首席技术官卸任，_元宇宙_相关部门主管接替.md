<p>品玩 9 月 23 日讯，近日，Facebook 公司首席技术官迈克 · 施洛普夫 ( Mike Schroepfer ) 周三宣布，他将卸任 CTO 一职。施洛普夫已在 Facebook 工作 13 年，他将在明年某个时候过渡为 Facebook 的首位资深院士 ( Senior Fellow ) 。</p><p>2022 年，Facebook 现任硬件部门主管安德鲁 · 博斯沃思 ( Andrew Bosworth ) 将接替施洛普夫成为 CTO。</p><p>博斯沃思在 Facebook 担任增强现实 ( AR ) 和虚拟现实 ( VR ) 技术副总裁，负责创建了 Facebook AR/VR 部门，并在 2020 年更名为 Facebook 现实实验室 ( FRL ) 。作为 CTO，他会继续领导 FRL。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
