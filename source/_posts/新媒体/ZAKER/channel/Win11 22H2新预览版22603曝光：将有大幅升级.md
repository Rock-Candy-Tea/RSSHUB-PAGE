<p>两周前，微软面向 Dev 通道的 Insider 会员推送了 Windows 11 最新预览版 Build 22598，同时还开放了 ISO 镜像下载。</p><p>不过，本周并没有新版迭代，微软程序员说是没找到质量可靠的候选版。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202204/626666f68e9f095473776fe7_1024.jpg" data-height="375" data-width="616" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/626666f68e9f095473776fe7_1024.jpg" referrerpolicy="no-referrer"></div></div>没想到细心的网友注意到，Build 22603 已经在 Github 的 Windows Terminal 项目中出现，这预计将是一次大幅升级。<p></p><p>前一版 22598 中，微软为用户带来了 Spotlight 测试 4K 桌面壁纸、Media Player 播放器资料库体验 / 性能升级等。</p><p>另外值得一提的是，在前不久在混合办公大会上，微软承诺将有一批新特性在 Win11 上线，包括多标签功能的资源管理器、我的电脑无缝集成 OneDrive、Windows 云电脑做成独立 APP 并支持离线运行、开始菜单支持文件夹、优化的手机互联等。</p><p>外界猜测，它们最快会在 4 月底登陆，只是 22H2 正式版还要一段时间，微软社区版主说是 11 月。</p><p>编辑｜熊乐</p><p><strong>· END<strong>·</strong></strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
