<p>IT 之家 4 月 26 日消息，微软今日面向 Win10 21H2/21H1/20H2 发布 KB5011831 补丁，版本号升级到 19042.1682、19043.1682 和 19044.1682。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div Internet Explorer 停止工作的问题。</p><p>更新在某些用户登录或注销时显示黑屏的问题。</p><p>更新一个问题，该问题可能导致 Microsoft OneDrive 文件在重命名并按 Enter 键后失去焦点。</p><p>更新一个问题，该问题导致未单击、点击或鼠标点击新闻和兴趣面板时显示该面板。</p><p>更新阻止用户更改登录到 Windows 设备时已过期的密码的问题。</p><p>IT 之家了解到，Win10 KB5011831 补丁是一项可选更新，用户需要在 Windows 更新的 " 可选更新 " 中手动查找并下载。</p><p>以下为完整更新日志：</p><p>新增功能，添加了为 Windows 的安全启动组件提供服务的改进。</p><p>对于已加入混合 Azure AD Autopilot 方案，将 Azure Active Directory ( Azure AD ) 注册的超时时间从 60 分钟更改为 90 分钟。这也解决了在超时时导致异常的争用条件。</p><p>解决了导致远程桌面会话关闭或重新连接在等待辅助功能快捷方式处理程序 ( sethc.exe ) 时停止响应的问题。</p><p>解决了某些销售点终端在重启期间遇到最多 40 分钟的 OS 启动延迟的问题。</p><p>解决了错误从注册表返回执行策略设置的问题。</p><p>解决了使用输入方法编辑器 ( IME ) 复制和粘贴文本时导致 Internet Explorer 停止工作的问题。</p><p>解决了某些用户登录或注销时显示黑屏的问题。</p><p>解决了导致 Kerberos 身份验证失败且错误为 "0xc0030009 ( RPC_NT_NULL_REF_POINTER ) " 的问题。当启用远程凭据防护时，客户端计算机尝试使用远程桌面协议 ( RDP ) 连接到另一台计算机时发生这种情况。</p><p>解决了阻止您更改登录到 Windows 设备时已过期的密码的问题。</p><p>解决了在应用不需要重启的 Windows Defender 应用程序控制 ( WDAC ) 策略时可能导致 Windows 停止工作的问题。</p><p>解决了可能无法将组策略的安全部分复制到计算机的问题。</p><p>解决了导致密钥分发中心 ( KDC ) 代码在域控制器关闭期间错误地返回错误消息 "KDC_ERR_TGT_REVOKED" 的问题。</p><p>解决了可能阻止图形处理单元 ( GPU ) 远程桌面上的负载均衡无法按预期工作的问题。</p><p>解决了阻止在 Microsoft Foundation 类 ( MFC ) 对话框中实例化版本 11 及更高版本的 Microsoft RDP 客户端控制的问题。</p><p>解决了重命名文件并按 Enter 键后可能导致 Microsoft OneDrive 文件失去焦点的问题。</p><p>解决了未单击、点击或鼠标显示新闻和兴趣面板的问题。</p><p>解决了使用韩语 IME 时无法将 Shift KeyUp 事件传递给应用程序的问题。</p><p>解决了使用 Netdom.exe 或 Active Directory 域和信任管理单元列出或修改名称后缀路由时可能发生的问题。这些过程可能会失败。错误消息是 " 系统资源不足，无法完成请求的服务。在主域控制器模拟器上安装 2022 年 1 月安全更新后，会出现此问题 ( PDCe ) 。</p><p>解决了启用字体缓解策略时显示错误的 IME 模式指示器的问题。</p><p>解决了导致主域控制器 ( 根域的 PDC ) 在系统日志中生成警告和错误事件的问题。当 PDC 错误地尝试扫描仅传出信任时，会出现此问题。</p><p>解决了影响服务器消息块 ( SMB ) 多通道连接并可能会生成 13A 或 C2 错误的问题。</p><p>解决了将网络驱动器映射到服务器消息块版本 1 ( SMBv1 ) 共享时出现的问题。重启 OS 后，无法访问该网络驱动器。</p><p>解决了导致分布式网络名称资源上的虚拟计算机对象 ( VCO ) 密码设置失败的问题。</p><p>解决了当 Client-Side Caching ( CSC ) 清理方法无法删除创建的资源时损害池的问题。</p><p>解决了可能导致服务器锁定的问题，因为非分页池会增长并占用所有内存。重启后，当您尝试修复损坏时，会再次出现同一问题。</p><p>减少高输入 / 输出操作中每秒高输入 / 输出操作的资源争用开销 ( IOPS ) 在单个文件上具有多个线程的方案。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
