<p>这次 iPhone 14 Pro 系列的发布没有让人眼前一亮，特别是药丸挖孔设计让不少人无法接受，<strong>但是苹果为了美化开孔而设计的的灵动岛功能则获得大批用户的肯定。</strong>据了解 iPhone 14 Pro 系列的灵动岛可以显示约 30 种信息，比如来电提醒、AirPods 连接、Face ID 认证、Apple Pay、AirDrop、AirPlay、Apple Watch 解锁 iPhone、充电和低电量提醒、铃声 / 静音模式切换、飞行模式等等。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202209/6328189a8e9f0962411a41ea_1024.jpg" data-height="1000" data-width="463" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/6328189a8e9f0962411a41ea_1024.jpg" referrerpolicy="no-referrer"></div></div>但是，作为 " 国民 APP" 的微信竟然还没有适配灵动岛功能，<strong>有网友发现 iPhone 14 Pro 系列在运行微信时，消息弹窗只能显示在灵动岛下方</strong>，并没有发布会展示的那样丝滑。为什么微信还没有适配灵动岛功能呢？一位博主解开了其中的原因。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202209/6328189a8e9f0962411a41eb_1024.jpg" data-height="631" data-width="639" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/6328189a8e9f0962411a41eb_1024.jpg" referrerpolicy="no-referrer"></div></div>海外博主 Javi 近日曝光了灵动岛的开发代码，他本来是想安装完 Xcode 14.1 Beta 版（iOS 开发程序）了解下 iPhone 14 Pro 的灵动岛 API 接口情况，<strong>但发现苹果尚未开放 API 接口，这表明第三方开发人员很难自行适配灵动岛这个系统级的交互功能。</strong><p></p><p>另外，第三方软件想要适配灵动岛需要向苹果官方申请开放 API 接口，这也是目前很多应用还没有适配灵动岛的原因。</p><p>不过，随着 iPhone 14 系列的全面上市，应用适配问题应该很快会被解决，只是时间问题。对于苹果灵动岛感兴趣，但又不想当先行者的朋友可以等一等。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202209/6328189a8e9f0962411a41ec_1024.jpg" data-height="501" data-width="658" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/6328189a8e9f0962411a41ec_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
