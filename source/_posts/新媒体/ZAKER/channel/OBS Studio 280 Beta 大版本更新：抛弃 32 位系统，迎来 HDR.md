<p>IT 之家 8 月 3 日消息，经典的免费开源流媒体软件 OBS Studio 迎来 28.0 Beta 1 大版本更新，带来了多项新功能，UI 框架升级到 Qt6。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202208/62ea269a8e9f097c0e1e3eee_1024.jpg" data-height="461" data-width="820" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202208/62ea269a8e9f097c0e1e3eee_1024.jpg" referrerpolicy="no-referrer"></div></div>更新说明：<p></p><p>支持 HDR 和 10-bit 颜色（Windows、macOS / Linux）</p><p>原生支持苹果 M1 / M2 芯片（macOS）：注意，第三方插件也需要适配 M1 / M2 芯片才能使用</p><p>将 UI 框架升级到 Qt6（所有平台），意味着 OBS Studio 不再支持以下操作系统，许多依赖于 Qt5 组件的第三方插件也将无法工作：</p><p>Windows 7 和 8</p><p>macOS 10.13 和 10.14</p><p>Ubuntu 18.04</p><p>所有 32 位操作系统</p><p>原生集成 obs-websocket 5.0 插件</p><p>更新 AMD 编码器 ( Windows ) </p><p>在 macOS 12.5+ 上支持 ScreenCaptureKit，从而显著提高捕获性能 ( macOS ) </p><p>苹果 VT 编码器迎来重大改进 ( macOS ) </p><p>支持特定应用的音频捕获 ( Windows ) </p><p>集成英伟达背景去除工具 ( Windows ) </p><p>按时间或文件大小自动分割录音（所有平台）</p><p>新的默认主题 Yami（所有平台）</p><p>IT 之家了解到，OBS Studio 28.0 Beta 1 已在 GitHub 发布，还有更多修复内容可以在日志中查看。该版本可以通过 GitHub 下载，也能在 Steam 平台选择 Beta 版进行下载。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202208/62ea269a8e9f097c0e1e3eef_1024.jpg" data-height="512" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202208/62ea269a8e9f097c0e1e3eef_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
