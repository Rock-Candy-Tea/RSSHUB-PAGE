<p>Windows 11 系统虽然在用户界面上进行了大幅的调整，但依然遗留了不少旧页面，有些甚至可以追溯到 Windows 98 时代。</p><p>而在即将到来、代号为 "Sun Valley 2" 的 Windows 11 下个功能更新中，微软将对这些旧页面进行升级，从而给用户提供更统一的使用体验。</p><p>在 Windows 11 Build 22572 最新预览版中，微软正测试升级后的现代化打印功能，该功能可以通过记事本等本地应用程序启动。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202203/622c0b318e9f0905f87c65e1_1024.jpg" data-height="408" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/622c0b318e9f0905f87c65e1_1024.jpg" referrerpolicy="no-referrer"></div></div>当你通过 NotePad 或 WordPad 打印文档时，你会看到一个基于 WinUI 和 Fluent 设计原则的现代界面，与 Windows 11 的其他 UI 保持一致。<p></p><p>正如你在上面的现代 " 打印 " 对话框截图中所看到的，微软已经更新了 " 打印机 " 等按钮，以使用 WinUI 2.6。然而，这次设计更新删除了一些功能，如 " 打印机偏好 "、" 位置 " 和其他高级选项，这些功能在传统版本中是可用的。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202203/622c0b318e9f0905f87c65e2_1024.jpg" data-height="337" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/622c0b318e9f0905f87c65e2_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
