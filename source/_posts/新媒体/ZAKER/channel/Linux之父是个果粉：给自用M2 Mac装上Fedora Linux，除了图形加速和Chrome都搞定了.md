<p>要说苹果 Macbook 最忠实的大神用户，<strong>Linus Torvalds</strong>当属其一。</p><p>当支持 M2 芯片的几个 Linux 项目还在测试时，Linus 就已经想办法给自己的<strong>M2 Macbook Air</strong>移植了 Fedora Linux。</p><p>Linux Kernel 5.19 正式版，就是他在自己的 M2 Macbook Air 上发布的。</p><p>今年的 Linux Plumbers 大会（LPC，Linux 内核开发者峰会）上，Linus 在接受 ZDNet 采访时，谈到了在 M2 Mac 上使用 Linux 的感受、Rust 引入 Linux 的进度、以及一些工作上的事情：</p><p>我真不是一个工作狂。</p><p>但同时他也透露，自己 "<strong>并不需要游戏</strong>"。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" Linux</b></p><p>Linux 有好几个发行版（Fedora、Ubuntu、Asahi 等），Linus 通常用的是 Fedora。</p><p>因此，他在 M2 Macbook Air 出来后，立刻就入手了一台，并成功向这台电脑上移植了 Fedora Linux。</p><p>无论是编译内核、还是给内核打补丁，都可以正常工作，而且 Macbook 的 16GB 内存 +256GB 硬盘配置也还不错。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" "Linus 版 "Fedora Linux 系统也有一些限制。</p><p>一方面，它<strong>不能处理 3D 图形</strong>（不支持 M2 FPU 图形加速）， 无法使用屏幕调光等功能；另一方面，<strong>Chrome</strong>也还不能在 Fedora Linux 上运行。</p><p>对于不支持 M2 FPU 的问题，Linus 的回应是 " 反正我也不需要游戏 "。（手动狗头）</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" Chrome 同步。<p></p><p>不过整体来说问题不大，毕竟 Chromium 可以正常运行。</p><p><b>" 我不是一个工作狂 "</b></p><p>虽然给 Linux 内核做了不少贡献，但提及工作量，Linus 表示自己并不是一个 " 工作狂 "：</p><p>我之所以现在还在做它，是因为我随时可以抽身。但一段时间后我又会觉得无聊，所以回来继续工作。</p><p>事实上，相比参加这次的 LPC 2022，Linus 坦言自己更想潜水。就在会议召开前，他已经在博奈尔（Bonaire，西印度群岛中荷属安的列斯群岛的岛屿）潜了 6 天水。</p><p>对于 " 真正的工作狂 " 这一称号，Linus 颁给了 Greg Kroah-Hartman。Greg 同样是 Linux 核心开发者之一，目前在 Linux 基金会工作，是 Linux 内核稳定分支（-stable）的维护者：</p><p>虽然很多任务可以自动化完成，但总归是无穷无尽的。他却每周都能完成。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div Rust？</b><p></p><p>虽然早在几个月前，Linus 就透露过会在 Linux 内核中引入 Rust。</p><p>BUT，在 8 月发布的 Linux 6.0 第一个候选版本中，Rust for Linux 的补丁却并没有被合并……</p><p>被问及合并时间时，Linus 表示 " 不敢保证 "，甚至在 10 月份的 6.1 版本也不一定出炉。</p><p></p><div
