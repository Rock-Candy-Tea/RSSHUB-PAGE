<p>IT 之家 9 月 2 日消息 外媒 Windows Latest 报道，微软已经宣布了 Windows 11 正式版将在 10 月 5 日发布，并且微软也开始警告测试者即将对 Windows Insider 计划做出的改变。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202109/613090a98e9f096f2005265b_1024.jpg" data-height="960" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/613090a98e9f096f2005265b_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家获悉，微软坚持要求 Windows 11 新操作系统运行所需的最低硬件要求。Windows 11 可以安装在英特尔第八代酷睿处理器（Coffee Lake）、AMD Ryzen 2000 及更新的处理器上，另外还有部分英特尔第七代处理器。遗憾的是，运行英特尔 Kaby Lake、AMD Zen 1 芯片或更早的设备上不支持官方升级到 Windows 11 操作系统。<p></p><p>如果你未能满足上述要求，就不能通过 Windows Update 升级到 Windows 11。如果你已经加入 Windows Insider 计划 Dev 开发频道，并且已经开始在不支持的电脑上测试 Windows 11 系统，你应该会在 Windows Update 中看到一条警告信息，解释了当前的情况。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202109/613090a98e9f096f2005265c_1024.jpg" data-height="656" data-width="1030" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/613090a98e9f096f2005265c_1024.jpg" referrerpolicy="no-referrer"></div></div>该信息指出，不受支持设备的用户将不能接收 PC 上的构建版本，需要重新安装 Windows 10。这是因为微软已经更新了对 Windows 11 预览版的要求，所以没有配备支持的处理器或模块的设备不再被允许获得新的构建版本。<p></p><p>对于那些仍想参与 Windows Insider 计划的用户，他们可以加入 Release Preview 发布预览频道，可以测试 Windows 10 版本 21H2（2021 年 10 月更新）。</p><p>该消息表明，对于使用不支持硬件的 Windows 11 测试者来说，这是一条末路，他们应该回退到 Windows 10。然而，微软现在进行了一些澄清，用户将继续获得 " 累积更新补丁 "，直到 Windows 11 正式发布 / 普遍可用，此前的提醒通知是关于即将从 Nickel（镍）分支开发的构建版本。</p><p>" 获得例外的 PC 将不再收到来自 Dev 开发频道的任何新的未来 Insider Preview 构建版本，但仍将获得 Windows 11 的累积性更新，直到达到普遍可用。正如我所说的，这里有一个 * 新构建 * 的区别，" 微软 Brandon LeBlanc 在 Twitter 上指出。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202109/613090a98e9f096f2005265d_1024.jpg" data-height="413" data-width="752" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/613090a98e9f096f2005265d_1024.jpg" referrerpolicy="no-referrer"></div></div>虽然微软目前不打算阻止不支持的 Windows 11 设备的累积更新，但这种情况可能很快就会改变，用户可能被禁止通过 Windows Update 获得必要的每月更新。<p></p><p>这是因为这些设备处于 " 不受支持 " 状态，即没有资格获得微软官方的支持。</p><p>从理论上讲，通过对 Windows 注册表进行修改，可以绕过这些限制。如果你对这些 " 漏洞 " 感到不舒服，你需要回滚到 Windows 10，以获得定期的累积和小功能更新。</p><p>微软称，Windows 10 至少会技术支持到 2025 年。</p><p>Windows 11 的推出计划</p><p>本周早些时候，微软正式宣布了 Windows 11 的推出日期，确认生产渠道中的特定设备（非 Insider）将从 10 月 5 日开始接收新操作系统升级更新。</p><p>Windows 11 已经处于测试的最后阶段，预计 Insider 计划的参与者将很快收到最终版本。微软将继续发布累积性更新，以修复错误 Bug 并提高操作系统的总体性能。</p><p>Windows 11 推送工作将持续到 2022 年，预计将在 2022 年夏天完成。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
