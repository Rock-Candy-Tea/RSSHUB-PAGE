<p>IT 之家 1 月 26 日消息，Linux 之父 Linus Torvalds 昨日似乎情绪有点失控，向 GitHub 提交了一个 README 文件，狠狠吐槽了 Linux 系统。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202109/61373118b15ec044c2267bfa_1024.jpg" data-height="428" data-width="760" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202109/61373118b15ec044c2267bfa_1024.jpg" referrerpolicy="no-referrer"></div></div>Linus 表示：" 大家好，我是 Linus Torvalds，红极一时的 Linux 的作者。你可以查看 repo 的 url 和文件等内容，可以证明百分之百是我。我删除了 Linux，因为我讨厌它，现在我认为它很烂。你应该去用这个很棒的操作系统，它叫做 Windows XP，我刚刚发现它很棒。"<p></p><p>难道 Linus 要删库跑路吗？从目前来看并不是，他只是向大家发了个牢骚。截至 IT 之家发文，Linux 存储库还好好的。</p><p>其实这不是 Linus 第一次吐槽 Linux 系统，他在早些年就看到了 Linux 桌面版的混乱，各大桌面 Linux 发行版的应用程序兼容问题很严重，它们随意修改核心，向后兼容也很差，应用分发体验也不如 Windows 系统。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202111/61a6491b8e9f09571f3b4720_1024.jpg" data-height="669" data-width="901" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/61a6491b8e9f09571f3b4720_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
