<p>IT 之家 5 月 11 日消息，今日美国专利商标局正式授权苹果一项专利，该专利显示通过在屏下使用光纤，可以以更高的速率和精度采集指纹数据。</p><p>苹果专利指出，可能需要捕获靠近设备的对象或用户的二维 ( 2D ) 或三维 ( 3D ) 图像。在某些情况下，2D 或 3D 图像可以是指纹、面部或视野 ( FoV ) 中场景的图像。</p><p>专利还涵盖了针对使用光纤或光纤束的光学传感的系统、设备、方法和装置，特别是针对近距离显示的光学传感。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202205/627b5c4c8e9f090a7020abac_1024.jpg" data-height="1565" data-width="1200" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/627b5c4c8e9f090a7020abac_1024.jpg" referrerpolicy="no-referrer"></div></div>苹果公司指出，电子设备通常包括用于提供照明、感知物体的接近程度、捕捉图像等的光电元件。与许多将光电元件置于显示屏下方并通过显示屏发射或接收电磁辐射（这可能导致 95-99% 的光传输损失）的解决方案相比，苹果公司专利申请中描述的技术使用光纤或光纤束，将位于屏幕下方、部分下方或邻近的光电元件之间的电磁辐射，沿着显示屏的边缘（例如，到显示屏和手机边框之间的间隙）传送到光学透射元件或表面的区域。<p></p><p>光学透射部件或表面的区域可以与屏幕相邻或靠近屏幕，电磁辐射可以以类似的方式被感知 / 检测。使用一根或多根光纤，在显示器附近发射和检测的电磁辐射，强度可能比通过显示器发射或检测的电磁辐射衰减得少。</p><p>在各种描述的实施例中，光纤可以具有非线性路径；可以启用具有不同发送 / 接收基线的传感器；可与光发射器、光检测器或光收发器结合使用；可以复用、拆分或组合；可以与光路路由控制器或其他可以控制光纤的元件耦合等等。</p><p>IT 之家了解到，此前包括郭明錤在内的众多分析师表示苹果会使用屏下指纹，但是在历年来的 iPhone 身上并没有采用该技术。作为一个专利，苹果是否会对其进行商业化还是未知数，此外由于目前组件短缺，为了防止因缺少组件导致无法出货，苹果应该近期不会推出该技术。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
