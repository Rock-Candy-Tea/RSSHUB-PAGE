<p>对于很多用户来说，想要恢复误操作删除的软件都是一件有些麻烦的事情，即便是在微软商店的软件，想要重新下载也需要在库中进行翻找。</p><p>但目前，微软正在开发一项名为 " 恢复应用程序 " 的新功能，只需点击一下就能够恢复已被删除的应用程序。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202205/6294225b8e9f0912fb6caa79_1024.jpg" data-height="321" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202205/6294225b8e9f0912fb6caa79_1024.jpg" referrerpolicy="no-referrer"></div></div>该功能将具有全部还原和选择应用程序两个选项，点击全部还原会一次性恢复所有被卸载的应用程序，而点击选择应用程序则能够选取需要的程序下载。<p></p><p>不过，由于该功能是基于微软商店实现的，现阶段仅支持用户用同一账号在商店中下载过的应用，如果是通过其他途径安装的应用，将无法用此方式恢复。</p><p>除了这项有一定实用性的功能外，微软还围绕商店，为 Win11 测试多项新功能。</p><p>其中 Windows 搜索与微软商店变得更加紧密，当用户搜索一项未被安装的应用程序时，系统将展示商店的搜索结果，方便用户安装应用。</p><p>同时，微软也放开了对商店的限制，允许及开发者将 Win32 应用程序提交至商店，并允许使用 GitHub Action 或 Rest API 自动提交应用程序。</p><p>这些功能和优化的加入，将使得 Win11 的微软商店内容更加丰富，使用也更为便捷，配合新的一键恢复功能，将为用户带来更好的使用体验。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202205/6294225b8e9f0912fb6caa7a_1024.jpg" data-height="337" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202205/6294225b8e9f0912fb6caa7a_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
