<p>IT 之家 4 月 5 日消息，华为公开了一种芯片堆叠封装及终端设备专利，解决因采用硅通孔技术而导致的成本高的问题。</p><p>国家专利局专利信息显示，华为技术有限公司于 4 月 5 日公开了一项芯片相关专利，公开号 CN114287057A。专利摘要显示，这是一种芯片堆叠封装及终端设备，涉及半导体技术领域，其能够在保证供电需求的同时，解决因采用硅通孔技术而导致的成本高的问题。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202204/624b9c3a8e9f09168074a36e_1024.jpg" data-height="395" data-width="947" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/624b9c3a8e9f09168074a36e_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家了解到，专利文件显示，该芯片堆叠封装 ( 01 ) 包括：<p></p><p>设置于第一走线结构 ( 10 ) 和第二走线结构 ( 20 ) 之间的第一芯片 ( 101 ) 和第二芯片 ( 102 ) ；</p><p>所述第一芯片 ( 101 ) 的有源面 ( S1 ) 面向所述第二芯片 ( 102 ) 的有源面 ( S2 ) ；</p><p>第一芯片 ( 101 ) 的有源面 ( S1 ) 包括第一交叠区域 ( A1 ) 和第一非交叠区域 ( C1 ) ，第二芯片 ( 102 ) 的有源面 ( S2 ) 包括第二交叠区域 ( A2 ) 和第二非交叠区域 ( C2 ) ；</p><p>第一交叠区域 ( A1 ) 与第二交叠区域 ( A2 ) 交叠，第一交叠区域 ( A1 ) 和第二交叠区域 ( A2 ) 连接；</p><p>第一非交叠区域 ( C1 ) 与第二走线结构 ( 20 ) 连接；</p><p>第二非交叠区域 ( C2 ) 与第一走线结构 ( 10 ) 连接。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202204/624b9c3a8e9f09168074a36f_1024.jpg" data-height="317" data-width="552" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202204/624b9c3a8e9f09168074a36f_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
