<p> [ PConline 杂谈 ] 微软已经发布了下一代 Windows 系统—— Windows 11，Win11 的正式版将会在下半年面向用户免费推送。多年来，Windows 一直是最适合游戏的 PC 系统，此前的 Win7、Win8 乃至 Win10 都加入了许多促进 PC 游戏进一步变革的特性，例如新的 DirectX 接口、新的 WDDM 模型等等，想要发挥新硬件新游戏的特性，最新的 Windows 系统是必不可少的。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" 又将成为新一代的系统级游戏神器。那么问题来了，Win11 都加入了哪些和游戏息息相关的新特性？游戏玩家有必要升级 Win11 吗？今天就一起来简单聊聊吧。<p></p><p><strong>Win11 才能发挥新 CPU 的威力？</strong></p><p>如果你关注 DIY，那么应该会知道今年将会是桌面 CPU 迎来重大变革的一年。在今年，英特尔将会推出第 12 代酷睿处理器 Alder Lake，该处理器的独特之处，在于使用了类似移动处理器的大小核设计——小核心为日常使用环境提供高能效，而大核心则保证重负荷环境下的性能上限。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202108/610de0ec8e9f093cb4790bd8_1024.jpg" data-height="394" data-width="700" 代酷睿也会使用大小核设计，这需要操作系统优化调度，才能发挥应有性能<p></p><p>这种设计似乎兼顾了能效和性能，但要真正完美发挥芯片的特性，还需要操作系统调度配合。操作系统必须足够聪明，才能准确判断什么时候使用小核心，什么时候调用大核心。很遗憾，由于桌面处理器现在才开始使用大小核设计，因此当前的 Windows 系统都没有对其优化，想要发挥出 Alder Lake 这样的大小核架构的威力，就需要升级到 Windows 11 了。</p><p>实际上，已经有媒体测试对比过 Win11 和 Win10 在大小核 CPU 平台下的性能差异。媒体使用了三星 Galaxy Book S 设备，它基于 Lakefield Core i5-L16G7 平台，而 Lakefield 就是典型的大小核设计的 CPU，可以将其看作是 Alder Lake 的前哨。下面是具体的测试结果。</p><p>首先是 GeekBench 5 测试，Win11 相比 Win10 21H1，在多线程上有 5.8% 的优势，在单线程上则有 2% 的优势。Win11 的性能对比 Win10 21H2 更好，但提升幅度并不算令人振奋，期待最终版能有更好的表现。</p><p></p><div class="img_box" id="id_imagebox_2" 的优势。这个测试结果是通过三种不同的测试得出的，而 Core i5-L16G7 这颗处理器的确在 Win11 中表现出了更强的性能。<p></p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 对驱动依赖较大，而新系统和新硬件都尚未有完善的驱动匹配。<p></p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" 的性能，这会给游戏带来更高的帧数、更低的延迟和更少的卡顿——对大小核架构优化不足，是有可能让异架构核心切换延迟明显增加的，而 Win11 显然对大小核优化更好。<p></p><p>除了 Alder Lake，桌面 PC 在未来还会迎来更多大小核异架构 CPU，例如 big.LITTLE 设计的 ARM 处理器等等。无论如何，Win11 的确能更好地发挥出新型 CPU 的威力，据悉微软在将来还会通过 Windows Update，来进一步优化大小核混合架构的 CPU，游戏玩家想要发挥硬件的极限性能，可别错过 Win11 了。</p><p><strong>Win11 才能发挥出新显卡的威力？</strong></p><p>我们知道每一代 Windows 都会更新 DirectX 接口，并更新 WDDM 显卡驱动模型，配合新 GPU 带来更多更先进的图形特性，Win11 也不例外。</p><p>首先，Win11 会引入新的 DirectX 特性，例如 DirectStorage。DirectStorage 可以让 GPU 直接访问 SSD，减少了图形数据中转环节，在降低延迟、减少读图时间的同时，也增加了显存的存储空间，为现在游戏设计愈发流行的开发世界大地图提供了更有利的运行环境。目前 NV 和 AMD 最新一代显卡都已经支持 DirectStorage 特性，例如 N 卡中的 RTX IO，A 卡的 SAM 就是这类技术。而厚道的是，Win10 也可以通过系统更新来支持 DirectStorage，获得类似的体验。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy 和内存<p></p><p>其次，Win11 会引入 WDDM 3.0 显示驱动模型，这会改变一些 GPU 的调用机制。例如，它允许 Linux 子系统直接管理更多显存，显著提升 Linux 子系统的图形性能。虽然这和游戏玩家关系不大，不过 Win11 对 GPU 调度的改进不仅限于此。</p><p></p><div class="img_box" id="id_imagebox_7" GPU。<p></p><p>对于拥有多个 GPU 的设备，这个机制可以让用户更明确地选择某个 GPU 运行游戏，不再会出现集显玩游戏的惨状，更完美得发挥出硬件应有的性能。</p><p><strong>Win11 才能让显示器潜力兑现？</strong></p><p>现在越来越多显示器支持 HDR，和传统的 SDR 相比，HDR 能够更精确地显示画面亮度，并提高亮度上限。但不是有了 HDR 显示器，就可以享用到 HDR 的，目前仍有大量游戏基于 SDR 开发，并不支持 HDR。此情此景下，Win11 就很有必要了。</p><p></p><div class="img_box" id="id_imagebox_9" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " 引入了自动 HDR，这是一个已经出现在 Xbox Series X/S 中的特性。就如名字所描述的那样，自动 HDR 可以让本身只支持 SDR 的 DirectX 11 和 DirectX 12 游戏，全自动地通过 HDR 来提升的色彩和亮度。只要玩家拥有合适的显示器，老游戏也会通过 HDR 得到明显的视觉提升。</p><p>如果你已经拥有了 HDR 显示器，而当前玩的游戏不支持 HDR，那么 Win11 就很值得一试了。</p><p><strong>Win11 游戏资源更加丰富？</strong></p><p>近年来，微软一直在整合家用游戏机和 PC 游戏资源，在 Win10 中即有推出 Xbox 的相关应用，并在微软商店中上架了大量和 Xbox 平台共享的 XGP 游戏。在 Win11 中，这种游戏资源共通的思路得到了进一步强化，Win11 直接集成了 Xbox 应用程序，提供更加直观的游戏管理方式的同时，也提供了优秀游戏、与其他玩家交流和切磋的社区平台。</p><p></p><div class="img_box" id="id_imagebox_10" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_10" 的资源共享也得到了进化，目前微软旗下的 Xbox 游戏工作室与 Bethesda 都已在主机和 PC 上同步发售新游戏，并在发售首日加入 XGP（Xbox Game Pass 服务）。据介绍，这种跨平台同步发售策略未来还将适用于更多工作室与合作厂商，让玩家自由选择主机或 PC 并保持体验一致。</p><p>简单来说，使用 Win11，可以更加轻易地获取到 Xbox 平台上的游戏资源。虽然现在 Win10 也可以通过安装 Xbox 应用来获得类似体验，不过预装相关功能的 Win11，无疑还是在这方面更加上心。</p><p><strong>总结</strong></p><p>近年来，其他桌面操作系统也在游戏方面开始发力，例如苹果推出了 Matel API，并开始运营 Arcade 游戏订阅服务，试图让移动平台和桌面平台的游戏资源互通。不过短期来看，Windows 在游戏领域的地位依然无人能撼动，如果你用 PC 玩游戏，Windows 仍是首选。</p><p>每一代 Windows 都在游戏方面下了不少苦心，这让 Windows 多年来都是最为适合游戏的桌面系统。在 Win11 中，我们也可以看到微软对新型硬件的适配兼容，对于先进图形技术的应用和推广。在 Win11 时代，PC 上又会诞生怎样的游戏大作，带来怎样的全新体验？我们共同期待吧。</p><div
