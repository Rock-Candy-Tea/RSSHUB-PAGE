<p>IT 之家 6 月 11 日消息 苹果已经开始测试一种新的密钥验证功能。值得一提的是，这也是微软和谷歌等企业所倡导的新特性，他们都在尽力推广去密码化的身份验证功能。</p><p>据称，这是一种新的身份验证技术，在保留密码的易用性同时保证更高的安全性。作为 iCloud （钥匙串）Keychains 的一部分，该技术的测试版本将在今年晚些时候与 iPhone、iPad 和 Mac 新系统一同到来。</p><p>据外媒 CNET，苹果工程师加勒特戴维森 ( Garrett Davidson ) 表示，苹果正在开发一种新的密码认证功能，用户可以使用基于 Face ID 和 Touch ID 的账户认证来代替密码。</p><p>" 因为只需轻轻一按即可登录，因此它比当今几乎所有常见的身份验证形式都更容易、更快、更安全。"</p><p>据介绍，全新 iOS 15 和 macOS Monterey 中的 "iCloud 密钥 " 会在 iCloudKeychain 中存储一个名为 " 密钥 " ( passkey ) 的新的 WebAuthn 证书，苹果将藉此帮助用户来完成创建和登录账户、一键登录等操作。</p><p>当您使用这个功能创建一个帐户时，你完全不需要去手动输入密码。你可以通过 Touch ID 或 Face ID 登录并验证该账户。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202106/60c2a73d8e9f094eda1c357f_1024.jpg" data-height="683" data-width="1092" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202106/60c2a73d8e9f094eda1c357f_1024.jpg" referrerpolicy="no-referrer"></div></div>在其 iCloud Keychain 演示中，苹果先让用户使用 iPhone 和 Face ID ( 左边和中间 ) 建立安全账户。之后，当你登录某 App 或网站时，系统会确认你的用户名和应用程序的名称，并使用 Face ID ( 右 ) 验证你的身份。<p></p><p>IT 之家了解到，谷歌和微软同样也在力推这一概念。例如谷歌已于 2018 年在 Chrome 和 2019 年在 Android 中内置了 WebAuthn 支持。</p><p>谷歌核心系统高级副总裁 Jen Fitzpatrick 在 5 月的谷歌 I/O 开发者大会上表示：" 当今最常见的安全漏洞仍然是密码 "" 我们的最终使命是创造一个不需要密码的未来 "。</p><p>此外，该安全验证技术目前仅适用于苹果设备，不过苹果也正在与 FIDO 和万维网联盟的合作伙伴进行磋商，试图将该解决方案带到非 Apple 设备上。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
