<p>以前三星的三大业务部门是消费电子业务 ( CE ) 、IT& 移动通讯业务 ( IM ) 、设备解决方案业务 ( DS ) 。而三星在 12 月 7 日公布了新的领导层任命，手机和消费电子业务合并为 SET 部门，并更换联席 CEO。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202112/61b04323b15ec0097d5fd026_1024.jpg" data-height="486" data-width="740" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202112/61b04323b15ec0097d5fd026_1024.jpg" referrerpolicy="no-referrer"></div></div>图源 Shutterstock<p></p><p>韩钟熙（Jong-Hee Han）被提升为副董事长和新合并的 SET 部门的 CEO，同时继续领导显示业务。韩钟熙时年 59 岁，电视研发领域出身，是让三星电视的核心人物。</p><p>前三星电机 CEO 庆桂显 ( Kyung Kye-hyun ) 被任命为 DS 部门 CEO。庆桂显时年 58 岁，半导体设计专家出身，曾任三星电子闪存产品和技术团队负责人以及 DRAM 设计团队的成员。</p><p>其余人事变动包括：</p><p>前执行副总裁兼 DS LSI 销售和营销主管朴勇仁（Yongin Park），被任命为总裁兼 DS LSI 总裁；</p><p>前总裁兼 DS 部门企业管理办公室负责人 Hark Kyu Park，被任命为总裁兼首席财务官；</p><p>前副董事长兼 DS 部门 CEO 金基南（Kinam Kim），被任命为三星先进技术研究院主席 ( （三星研发中心）</p><p>前三星电子北美办事处副总裁的崔京植（Choi Kyung-sik），被任命为 SET 事业部总裁兼北美办事处负责人</p><p>前总裁兼 DS LSI 业务负责人 Inyup Kang，被任命为 DS 部门总裁兼北美办事处负责人</p><p>新的业务部门安排，反映了时代的变迁。最近 20 年，三星最赚钱的业务从平板电视（2000 年之后）变成手机（2010 年后），再变成现在的半导体。</p><p>2016 年是个关键节点（Galaxy Note7 那年），三星移动业务和 DS 业务此消彼长，DS 业务开始占据主导，毕竟半导体和显示面板业务都在 DS 部门。在 2017 年 Q1，三星 76.67% 的利润都源自 DS 业务，而且状况一直维持到 2021 年。三星 2021 年 3 季度财报显示，手机和消费电子业务总共只贡献了四分之一的营业利润，而另外四分之三是来自 DS 业务。害，真的是时代变了。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
