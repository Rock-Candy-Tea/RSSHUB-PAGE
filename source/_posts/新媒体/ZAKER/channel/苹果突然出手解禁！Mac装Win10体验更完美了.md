<p>在 Mac 上安装 Windows 10，多数离不开苹果出品的 Boot Camp 工具（苹果官方名为启动转换助理）。</p><p>细心的用户发现，在最近更新的 v6.1.15 版本中，Boot Camp 添加了对 Windows 系统精准触摸板驱动的支持，换言之，Mac 用户安装 Win10 后无需寻找三方驱动也能原生使用触摸板手势等高级功能了。</p><p>苹果提到，<strong>所谓的手势包括轻触替代左击，右下角轻触代表右击，滑动手势、缩放手势以及三指、四指组合等。</strong></p><p>不过，在随后的支持文档中，苹果指出，只有配备了 T2 安全芯片的 Mac 设备才能启用 Boot Camp 的精准触摸板功能，换言之，机型包括：</p><p>iMac ( Retina 5K, 27 英寸 , 2020 ) </p><p>iMac Pro</p><p>Mac Pro ( 2019 ) </p><p>Mac Pro ( 架式 , 2019 ) </p><p>Mac mini ( 2018 ) </p><p>MacBook Air ( Retina, 13 英寸 2020 ) </p><p>MacBook Air ( Retina, 13 英寸 , 2019 ) </p><p>MacBook Air ( Retina, 13 英寸 , 2018 ) </p><p>MacBook Pro ( 13 英寸 , 2020, 两个雷电 3 接口 ) </p><p>MacBook Pro ( 13 英寸 , 2020, 四个雷电 3 接口 ) </p><p>MacBook Pro ( 16 英寸 , 2019 ) </p><p>MacBook Pro ( 13 英寸 , 2019, 两个雷电 3 接口 ) </p><p>MacBook Pro ( 15 英寸 , 2019 ) </p><p>MacBook Pro ( 13 英寸 , 2019, 四个雷电 3 接口 ) </p><p>MacBook Pro ( 15 英寸 , 2018 ) </p><p>MacBook Pro ( 13 英寸 , 2018, 四个雷电 3 接口 ) </p><p>另外，必须说明的是，M1 处理器的 MacBook 暂时无法使用 Boot Camp 安装 Win10。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202106/60c435ee8e9f090ed57423c7_1024.jpg" data-height="300" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202106/60c435ee8e9f090ed57423c7_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
