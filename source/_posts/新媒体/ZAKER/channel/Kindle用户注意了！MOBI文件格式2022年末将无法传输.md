<p>对于不少 Kindle 用户来说，通过个人文档服务将电子书传到设备上都是一项 " 常规操作 "，而在这个过程中，最常用，也是 Kindle 支持最好的，莫过于是 MOBI 格式。</p><p>但今天，亚马逊修改了客户帮助，告知用户，从 2022 年末起，将无法再通过个人文档服务将任何 MOBI 格式的文件（包括 .AZW 和 .MOBI）发送至设备。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202204/626cc5138e9f09153126a981_1024.jpg" data-height="239" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/626cc5138e9f09153126a981_1024.jpg" referrerpolicy="no-referrer"></div></div>官方表示，这是因为 MOBI 是较旧的文件格式，不适用于最新的 Kindle 功能。<p></p><p>但实际上，作为亚马逊一手带起的文件格式，MOBI 格式是 Kindle 适配效果最好的电子书格式，也因此成为了不少用户的首选。</p><p>在 MOBI 格式无法传输后，个人文档服务依旧支持下列文件格式：</p><p>Microsoft Word（.DOC、.DOCX）</p><p>HTML（.HTML、.HTM）</p><p>RTF ( .RTF ) </p><p>文本 ( .TXT ) </p><p>JPEG（.JPEG、.JPG）</p><p>GIF ( .GIF ) </p><p>PNG ( .PNG ) </p><p>BMP ( .BMP ) </p><p>PDF ( .PDF ) </p><p>EPUB ( .EPUB ) </p><p>不过，上述格式在 Kindle 上的阅读体验均无法达到 MOBI 的表现。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202204/626cc5138e9f09153126a982_1024.jpg" data-height="400" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/626cc5138e9f09153126a982_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
