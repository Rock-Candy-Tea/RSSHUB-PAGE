<p>IT 之家 4 月 26 日消息，马斯克今日以 440 亿美元（约 2873.2 亿元人民币）买下推特 Twitter，成为了 IT 行业的重磅消息。在本月早些时候，马斯克还承诺如果收购推特，将开源其算法。</p><p>IT 之家小伙伴 @Ryen_D 发现，推特今日在 GitHub 创建了一个名为 "the-algorithm（算法）" 的空白仓库，仅推出六小时就引来了五千多个 Star。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202204/626766ea8e9f097a407356f8_1024.jpg" data-height="536" data-width="1137" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/626766ea8e9f097a407356f8_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家了解到，马斯克此前表示，他相信 " 给言论自由提供一个包容性舞台是极为重要的 "。他认为 Twitter 算法应该变成开源算法，应该将后台代码放在 Github 平台。<p></p><p>看来马斯克确实说到做到，收购消息刚公布就跟进开源，目前尚不清楚算法文件何时将上线。</p><p>马斯克在新闻稿中的一份声明中称：" 言论自由是民主运作的基石，而推特是一个数字城镇广场，人们在这里讨论对人类未来至关重要的问题，我还希望通过使用新功能增强产品、使算法开源以增加信任、击败垃圾邮件机器人以及对所有人进行身份验证，让推特比以往任何时候都更好。"</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
