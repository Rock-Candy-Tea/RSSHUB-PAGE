<p>本周，Windows 11 推送了几个新版本，其中 Beta 通道收获 KB5015890，对应 Build 22621.450 或者 Build 22622.450，前者屏蔽了新功能。</p><p>微软表示，新系统增强了 Defender 防御勒索恶意软件的能力，还修复了一个长期恼人的 BUG，即由于 sihost.exe 进程导致 CPU 占用率高的问题。</p><p>sihost.exe 是 Windows 系统核心进程之一，负责开始菜单、任务栏和其它关键图形元素，比如透明度等。此前有相当一部分用户发现，即便是纯净安装，也无法解决因为 CPU 占用率高而导致的卡顿。</p><p>其实，在 Windows 22H2 的正式通道，微软也在测试 KB5016700 补丁，修复的也是同一个问题。正式版预计 9 月或者 10 月开始小规模推送，年底前扩大范围。</p><p>说到新版，Dev 通道的 Insider 会员则收获 Build 25174，日志中进提到一个新功能，就是加入了 PC Game Pass 小部件，可以轮播最新加入服务的新游戏。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div"><img class="lazy opacity_0 zaker_gif_cache" id="img_0" data-original="http://zkres2.myzaker.com/202208/62eb307f8e9f09108317ac0c_1024.jpg" data-gif-url="http://zkres2.myzaker.com/202208/62eb307f8e9f09108317ac0c_raw.gif" data-height="400" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202208/62eb307f8e9f09108317ac0c_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
