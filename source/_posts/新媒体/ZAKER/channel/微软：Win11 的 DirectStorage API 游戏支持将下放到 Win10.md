<p>IT 之家 7 月 20 日消息 微软在上个月正式发布了 Win11 操作系统，为游戏玩家带来了重磅技术 DirectStorage API。这个新的 API 改变了游戏从 NVMe SSD 驱动器读取数据的方式，从而实现更快的读取速度和更短的加载时间。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202106/60d7f4fc8e9f097b3d01dd71_1024.jpg" data-height="461" data-width="820" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202106/60d7f4fc8e9f097b3d01dd71_1024.jpg" referrerpolicy="no-referrer"></div></div>近日，微软 DirectX 博客官方人员宣布，适用于 Win11 的 DirectStorage SDK 已经推出，同时官方确认，DirectStorage API 将适用于 Win10 1909 版本及以上。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202107/60f629238e9f0912e9590b6b_1024.jpg" data-height="472" data-width="1210" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60f629238e9f0912e9590b6b_1024.jpg" referrerpolicy="no-referrer"></div></div>不过按照官方解释，由于 Win11 在设计之初就考虑到了 DirectStorage API ，因此将采用全新升级的存储堆栈；而 Win10 的 DirectStorage API 会在原来的基础上优化传统存储堆栈，但并未指出两者的性能差距有多少。<p></p><p>IT 之家了解到，该技术只在 NVMe SSD 硬盘上可用，其余硬盘将以原有速度加载游戏，所以最好把游戏放在 NVMe SSD 中。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
