<p>IT 之家 7 月 28 日消息 外媒 9to5 Mac 报道，早期关于 iPhone 14 的传闻表明，苹果计划明年取消刘海凹槽，而屏幕下的 Touch ID 或 Face ID 是实现这一目标的两种可能方式。今天苹果被授予的一项专利描述了如何将这两种形式的生物识别认证嵌入到显示屏中。</p><p>甚至有可能将前置摄像头嵌入到屏幕下方，不过这可能是以后的发展 ......</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202107/610148f5b15ec0140006e540_1024.jpg" data-height="700" data-width="1400" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/610148f5b15ec0140006e540_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家获悉，苹果专利解释说，目前的 iPhone 设计将成像传感器放在显示屏旁边，而不是显示屏的一部分 -- 尽管凹槽设计在某种程度上看起来像是屏幕的一部分。<p></p><p>" 通常情况下，成像传感器被安置在保护罩下面，与显示屏堆叠在一起。因此，传统的电子设备同时包含显示堆栈和成像传感器，通常需要大面积的保护罩，该保护罩延伸到显示堆栈的外围，以保留空间来容纳成像传感器。这种传统结构不合理地增加了环绕显示屏的边框区域的明显尺寸，同时也不合理地增加了电子设备外壳的尺寸和体积。"</p><p>该专利的主要重点是将图像传感器嵌入显示器，以读取指纹 -- 即嵌入式 Touch ID。</p><p>" 为了简化描述，后面的许多实施例都提到了一种结构，其中电子设备实现了穿透式成像，以捕捉用户触摸电子设备显示屏的特定区域（" 成像区域 "，在成像孔上方）的指纹的一个或多个图像。"</p><p>然而，苹果公司指出，同样的方法可以用于其他形式的生物识别，包括 Face ID。</p><p>" 如上所述，电子设备可以为任何合适的成像、传感、数据聚合或光捕捉目的实现穿透式成像，包括但不限于 [ ... ] 人脸识别。"</p><p>将 Face ID 和 Touch ID 同时嵌入到显示屏中，可以让所有用户都满意，也意味着 Face ID 也可以嵌入到 Mac 显示屏中，而不需要有更宽的边框或凹槽。彭博社记者最近表示，Face ID 将在几年内出现在 Mac 设备上。</p><p>从长远来看，前置摄像头也有可能被嵌入到显示屏中。然而，早期的尝试大大降低了质量，包括雾霾、低对比度和不准确的白平衡。然而，这些问题不是屏下 Face ID 的障碍，所以几乎可以肯定我们会先看到这一步。</p><p>当然，现在是 iPhone 14 的早期阶段，我们照例提醒，苹果为各种技术申请专利，但从未明确表明会将其推向市场。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
