<p>微软即将发布的 Windows 11 在用户界面进行了许多积极的改变，但默认应用程序的体验是一种退步：微软即将发布的 Windows 11 将使切换默认浏览器变得更加困难，Mozilla、Opera 和 Vivaldi 等浏览器竞争对手对此表示担忧。</p><p>在 Windows 11 中，微软已经改变了设置默认应用程序的方式。与 Windows 10 一样，当你安装一个新的浏览器并首次打开一个网络链接时，会有一个提示出现。不过，这也是唯一可以轻松切换浏览器的机会。除非你勾选 " 始终使用此应用程序 "，否则默认值将永远不会被改变。忘记切换 " 始终使用此应用程序 " 选项，或者简单地选择你想要的浏览器，然后再也看不到这个提示，是非常常见的。</p><p></p><div class="img_box" id="id_imagebox_0" 中的默认应用提示，您只会看到一次<p></p><p>如果你忘记了在第一次启动时设置默认浏览器，与 Windows 10 相比，现在在 Windows 11 中切换默认浏览器的体验非常混乱。Chrome 和许多其他竞争对手的浏览器经常会提示用户将其设置为默认，并将 Windows 用户扔到设置的默认应用程序部分来启用。</p><p>微软已经改变了 Windows 11 中默认应用程序的分配方式，这意味着你现在必须按文件或链接类型设置默认，而不是单一的开关。就 Chrome 而言，这意味着需要同时改变 HTM、HTML、PDF、SHTML、SVG、WEBP、XHT、XHTML、FTP、HTTP 和 HTTPS 的默认文件类型。</p><p>Windows 10 允许你快速、轻松地切换默认的电子邮件、地图、音乐、照片、视频和网络浏览器应用程序，但 Windows 11 不是，这是一个不必要的漫长过程。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 上的趋势越来越担心，"Firefox 高级副总裁 Selena Deckelmann 在一份声明中说。" 自 Windows 10 以来，用户不得不采取额外和不必要的步骤来设置和保留他们的默认浏览器设置。这些障碍是混乱的，似乎旨在破坏用户对非微软浏览器的选择。"</p><p>并非只有 Firefox 有这样的担忧。</p><p>Vivaldi 发言人说：" 微软有这样做的历史，而且似乎他们正逐渐变得更糟。随着每一个新版本的 Windows 的到来， [ 改变默认值 ] 越来越难。他们明白，他们能让人们使用他们的浏览器的唯一方法就是锁定他们。"</p><p>微软 Edge 的另一个竞争对手 Opera，也对微软对 Windows 11 默认应用程序的改变感到担忧：" 当一个平台供应商为了提高自己产品的地位而模糊了一个普通的用例，这是非常不幸的，"Opera 的浏览器主管 Krystian Kolondra 在一份声明中说。" 我们想鼓励所有的平台供应商尊重用户的选择，允许在他们的平台上进行竞争。剥夺用户的选择权是一种倒退 "。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div Windows 11 中浏览器的唯一问题。微软在 Windows 10 的搜索体验中一直忽视默认浏览器的选择，该公司推出的任务栏小工具也忽视了默认浏览器，迫使用户进入 Edge。</p><p>Windows 11 延续了这一趋势，搜索仍然强迫用户进入 Edge，现在又推出了一个新的专用小部件区域，也忽略了默认的浏览器设置。" 看来，Windows 11 小工具会忽略用户的默认浏览器选择，而是打开微软 Edge，"Brave 发言人在一份声明中说。"Brave 把用户放在第一位，我们谴责 Windows 11 的这种做法，因为默认浏览器的选择对个人和他们的隐私有很多影响。用户应该有选择的自由 "。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div
