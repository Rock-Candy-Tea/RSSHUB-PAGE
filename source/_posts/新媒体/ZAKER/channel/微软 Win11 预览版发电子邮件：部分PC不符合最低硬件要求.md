<p>IT 之家 9 月 3 日消息 微软一直在通过 Windows Update 页面中的横幅提醒通知那些 PC 电脑不符合微软最低硬件要求的 Windows 11 Insiders 测试用户，他们需要降级到 Windows 10 系统。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202109/6131d84fb15ec00c374e88de_1024.jpg" data-height="692" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/6131d84fb15ec00c374e88de_1024.jpg" referrerpolicy="no-referrer"></div></div>现在，微软更进一步，开始向不符合微软最低硬件要求的用户发送电子邮件说明，表示，<p></p><p><strong>" 你的电脑不符合预览 Windows 11 的最低系统要求。</strong></p><p>嗨，Windows Insider。</p><p>Dev 开发频道中的一些电脑没有切换频道的选项，因为他们被赋予了安装 Windows 11 预览版的例外情况，正如我们在 6 月 24 日的博文中所述。这些电脑不符合预览 Windows 11 的最低硬件要求，今后将不会从 Dev 频道获得新的 Windows 11 Insider Preview 构建版本。</p><p>根据我们的数据，你有一台电脑被给予了这种例外。这台电脑必须使用我们提供的媒体（ISO）重新安装回 Windows 10，然后可以加入发布预览频道来预览 Windows 10 更新。</p><p>请查看你的 Windows 更新设置中的信息以了解更多信息。</p><p>感谢你参与 Windows Insider 计划 !</p><p>Windows Insider 团队 "</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202109/6131d84fb15ec00c374e88df_1024.jpg" data-height="1781" data-width="720" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/6131d84fb15ec00c374e88df_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家获悉，微软此前表示，他们打算继续向受影响的用户发布 Windows 11 的累积更新补丁，直到 10 月 5 日 Windows 11 全面上市，但之后用户需要降级到 Windows 10 系统。微软没有解释如果用户忽略这一提示会发生什么，只是表明这些用户不会再收到后续的 Windows 11 Dev 大版本更新。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
