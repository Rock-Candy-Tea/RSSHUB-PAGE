<p>一直以来，Chrome 浏览器都是世界上公认的最受欢迎的浏览器，而火狐 Firefox 作为和 Chrome、Edge（IE）相提并论的三大浏览器之一，也积累了相当高的人气，尤其在 Linux 中。不过火狐这几年的日子似乎并不安生，除了市占率持续降低，越来越多的网站开始停止支持火狐浏览器。</p><p>此前，日本 NHK 宣布其视频平台 NHK+ 于 5 月 23 日之后正式停止对火狐浏览器的支持，建议用户使用 Chrome、Edge 或 Safari 等其他浏览器的最新版本。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" 也将火狐排除在了支持范围外，使用火狐打开网站会弹出 " 您的浏览器不受支持 " 的提示。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 商务管理。<p></p><p>但实际更新后，所有最新版本的 Firefox 均会显示上述提示信息，包括 Firefox Stable、Firefox ESR 和 Firefox Nightly。</p><p>目前，火狐用户可以通过更改浏览器的用户代理（UA）方式来跳过苹果商务网站对该 Web 浏览器的限制，从而正常浏览网站。据悉，用户代理通常用作标识符，它可能会显示浏览器的名称和版本以及其他信息。</p><p>当用户将 Firefox 用户代理改为 Chrome 时，苹果商务网站就会加载登录提示，提示可以正常登录该网站。</p><p>作为曾经最强的浏览器，火狐走到如今地步令人唏嘘。</p><p><strong>● </strong><strong>火狐的诞生</strong></p><p>当年，在微软 IE 称霸浏览器市场之前，1994 年诞生的网景（Netscape）才是浏览器界的 " 一哥 "，巅峰时期市场占有率近 90%，并且那时候网景浏览器还是苹果的默认浏览器。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202207/62d035e48e9f09700653ce1f_1024.jpg" data-height="433" data-width="515" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202207/62d035e48e9f09700653ce1f_1024.jpg" referrerpolicy="no-referrer"></div></div>而作为竞争对手，在 IE 之前微软并没有自己的默认浏览器，其重心仍在操作系统上。<p></p><p>相比同期浏览器，网景浏览器的优势诸多，比如更具实用性和稳定性，当然最重要的是，它推出了免费试用版，即个人免费，商用收费。凭借这样的商业模式，网景浏览器很快就劈开了广大的下沉市场。</p><p>1995 年，比尔 · 盖茨发布了一封内部信，指出浏览器在未来将会是整个互联网的入口，以后大家无论上网做什么，第一件事就是先打开浏览器。而这份计划上，微软表态首要做的事，是打败网景。</p><p>微软先是试图收购网景，不过以失败告终。于是微软决定自己做，比如砸钱 20 亿美元来购买、兼并和开发。</p><p>不过早先几个版本的 IE 在技术上始终落后于网景，然而微软放出了一个大招：将 IE 捆绑在 Windows 95 操作系统上。要知道，当时全球 86% 的电脑都安装了 Windows 操作系统，而 IE 则成为每台电脑的默认浏览器。这无异于是将浏览器价格直接拉到了零，也使得 IE 很快得以普及。</p><p>随着微软不断改进 IE，网景逐步失去优势，被迫不断降价。</p><p>1998 年前后，IE 的市场份额开始超越网景，成为市面上使用最多的网页浏览器。后者在这一年年底则被美国网络公司美国在线（AOL）收购。</p><p>此后，IE 一路高歌猛进，一度在 2002 年和 2003 年达到峰值，占据约 95% 的市场。相比之下，网景浏览器的市占率只剩 13％。</p><p>但网景的故事并未就此结束。</p><p>网景在被收购之前便开放了浏览器软件的源代码，成立了 Mozilla 基金会，也就是火狐浏览器（Mozilla Firefox）里的那个 Mozilla。</p><p>另一边，由于微软不断的进行 " 反向创新 "，导致其越来越臃肿，运行速度越来越慢，再加上安全问题，更多的用户开始转向第三方浏览器。</p><p>乘着东风，2004 年，Mozilla 推出了火狐浏览器。凭借其速度，安全性以及开源标准，它赢得了用户和开发者的青睐。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" data-original="http://zkres1.myzaker.com/202207/62d035e48e9f09700653ce20_1024.jpg" data-height="384" data-width="557" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202207/62d035e48e9f09700653ce20_1024.jpg" referrerpolicy="no-referrer"></div></div>在 2008 年，火狐浏览器突破 1 亿下载量；到 2009 年，火狐的市场份额已经达到了 32%，是当时公认的最受欢迎的浏览器；直至 2014 年，火狐占据着全球浏览器市场份额的 35%-40%。<p></p><p>但从那以后，随着更强大的竞争对手 Chrome 浏览器的崛起，火狐面临的情况就变得并不乐观。</p><p><strong>● 从辉煌走向落寞</strong></p><p>今年 6 月，陪伴无数网友的 IE 浏览器正式退役。而承载了很多人青春记忆的火狐浏览器是否落得相同命运呢？</p><p>火狐浏览器在巅峰时期曾与 IE 浏览器分庭抗礼，两者占据了桌面浏览器 90% 的份额。只是，这款曾经最好用的浏览器的高光时刻并未保留太久。</p><p>今年 2 月中旬，火狐浏览器的市场份额跌到 4%；在 3 月浏览器市场占比率的数据中，火狐市场占比则仅有 3.4%。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" 1.96 亿用户，也就是说，仅用不到三年的时间，用户便流失了 4800 万，平均每年流失 1600 万人。<p></p><p>火狐浏览器还面临更麻烦的事——营收已下滑到了 5 亿美元，其中 80% 还是靠谷歌合作。</p><p>据了解，谷歌每年都会向 Mozilla 支付数亿美元的版税，以便谷歌将自己的搜索引擎设置为 Firefox 浏览器的默认版本。</p><p>今年 2 月 19 日，火狐浏览器官方发布一篇声明，火狐浏览器更新新版本之后，部分用户的默认搜索引擎将会发生改变。这不得不令人怀疑，火狐与谷歌的续签协议没有谈拢。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy referrerpolicy="no-referrer"></div></div>要知道，火狐浏览器的主要营收来源还是通过默认浏览器产生的搜索引擎分成，而今用户流失、默认搜索引擎发生改变，毫无疑问会影响到火狐浏览器的营收。<p></p><p>另外，火狐浏览器一直没能抓住手机用户，占有率不断下滑，移动平台占有率不到 0.5%。</p><p><strong>● </strong><strong>火狐用户为何流失如此之快</strong></p><p>一方面是由于市场竞争的压力，包括微软在内的科技公司都开始转向 Chromium 内核，用户可选择的浏览器只会越来越多，再加上 Chrome 浏览器、Edge 浏览器以及 Safari 浏览器分别是安卓设备、Windows 系统以及苹果设备上的默认浏览器，实质上，这就已经为它们争取到了大量的用户。</p><p>更何况这些科技巨头在吸引用户时所采取的措施相当激进。</p><p>此前，针对那些使用 Chromium Edge 访问 Chrome 网络商店下载扩展的用户，Chrome 网络商店都发出了警告，提醒他们使用 Chrome 浏览器进行访问，以免出现安全问题，微软方面同样也采取了类似措施。</p><p>另一方面则要归结到火狐自身性能上。</p><p>近年来，火狐一直致力于改进隐私体验，但却没有更多实质性的性能改进。有业内人士曾直言：Firefox 浏览器的隐私证书与它的任何商业对手相比都毫不逊色。但不幸的是，这些隐私功能默认情况下是不启用的，只能用户在需要的时候手动开启。</p><p>甚至火狐浏览器的某些重大改版被批评是破坏用户体验，这让大多数追求性能体验的用户开始抛弃火狐转向其他浏览器。</p><p>同时火狐浏览器频繁且大幅度的修改浏览器界面、设置位置，也加剧了用户的流失。</p><p>用户一旦流失就很难再找回，就连前 Firefox 工作人员也表示，不要对火狐浏览器的复兴抱有太多希望。</p><p>但正如 Firefox 的高级副总裁 Selena Deckelmann 所言，火狐浏览器并不需要向 Chrome 或 Safari 那样大，就能取得成功，火狐真正想要成为的是一个可行的选择，让每个人都有不同的选择，从而使互联网变得更好。</p><p>这不仅是 Deckelmann 对火狐浏览器未来的期许，也是很多网友的想法，市场需要这种差异化的存在，若想阻止互联网巨头对市场的垄断，火狐浏览器的存在必不可少。</p><p>但目前来看，情况似乎并不乐观。随着停止支持火狐浏览器的网站增加，这个继承了网景衣钵的 " 精神续作 " 似乎前景渺茫。你认为火狐会顺利渡过难关吗？</p><p>编辑：崔崔</p><p><strong></strong></p><div
