系列<p></p><p>MI300 内部可以大致分为三层结构，<strong>底层是庞大的中介层 ( Interposer ) ，面积约 2750 平方毫米</strong>，MLID 直言这是他见过的最大的。</p><p><strong>中介层之上，是一系列 6nm 工艺的 Base Die ( 基础芯片 ) ，也可以叫做区块 ( Tile ) </strong>，集成负责输入输出的 IO Die、其他各种 IP 模块、可能的缓存，每个区块面积约 320-360 平方毫米。</p><p><strong>每个 6nm 区块之上，是两个 5nm 工艺的 Compute Die ( 计算芯片 ) </strong>，单个面积约 110 平方毫米，内部就是各种计算核心和相关模块，但据说可以定制选择不同的模块，满足不同计算需求。</p><p>同时，<strong>每个计算芯片对应一颗 HBM3 高带宽内存</strong>，容量暂时不详。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3"
