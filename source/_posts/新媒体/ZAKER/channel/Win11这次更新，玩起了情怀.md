<p>每次微软更新都是一片吐槽声，估计各位看官和微软自己都习惯了。但是这次不一样，微软玩起了情怀，意外地引起了大家的共鸣。</p><p>今日，微软面向 Windows 11 Dev 频道用户发布了对相机 App、媒体播放器的更新，带来了多项新功能。媒体播放器更新到了 11.2206.30.0 版本，该版本添加了对 CD 翻录的支持，支持的格式包括 AAC、WMA、FLAC 和 ALAC。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202207/62d036288e9f09710b257c4a_1024.jpg" data-height="630" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202207/62d036288e9f09710b257c4a_1024.jpg" referrerpolicy="no-referrer"></div></div>Windows 11 的电影和电视 App 已原生支持 ARM64，可以获得更好的播放性能。不过，由于微软打算用媒体播放器 App 来替代电影和电视 App，所以会将视频文件类型关联默认从电影和电视 App 更换到 媒体播放器 App。<p></p><p>实际上，这个功能在 Win7 系统里的 Windows Media Player 里面已经有了，那时候没有这么多支持格式而已。不过，对于 CD 翻录这个功能，你首先得有一个刻录机 / 光驱，但是现在很多人都不会配这个设备了。尤其是笔记本，为了做得越来越轻薄，早就放弃了光驱的设计。当然，还是有人对 CD 保持了原有的热情，还会在 DIY 主机的时候专门置办一套设备。有网友表示，目前 CD 机已经成古董了，某平台二手货需要好几千元，所以 CD 翻录这功能还是很有必要的。这对于喜欢听 CD 的用户来说，真的算是一次有情怀的更新。</p><p>另外，在这次更新中相机 App 更新到了 2022.2206.2.0 版本，带来了全新 Windows 11 外观。此外，该应用除了现有对相机、视频、文档和白板捕获模式的支持外，新增支持了二维码和条形码扫描功能。</p><p>还有，在近期推送的 Build 25158 版本中，微软为任务栏加入了两种新的搜索设计，这两种新设计将 Win11 搜索功能原本的 " 放大镜 " 图标改为了类似 Win10 的搜索框，只是在长度上有所不同。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202207/62d036288e9f09710b257c4b_1024.jpg" data-height="134" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202207/62d036288e9f09710b257c4b_1024.jpg" referrerpolicy="no-referrer"></div></div>除了搜索框外，此次更新还为小部件界面做出了改动。更新后，当出现动态通知消息（如新闻）时，小部件图标会显示 " 红点 " 通知标记，提醒用户打开阅读。<p></p><p>在打开小部件窗口后，动态消息会置顶，以确保用户能够第一时间看到。</p><p>你觉得这次 Win11 更新细节有用吗？</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
