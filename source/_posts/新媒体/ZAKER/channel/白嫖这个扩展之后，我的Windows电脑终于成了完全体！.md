<p> 有印象的差友应该还记得前几天托尼写了一篇有关 HEVC 的文章（ 传送门 ），里边有提到很多浏览器其实都不支持 GPU 硬解 HEVC 视频。</p><p>这就导致我们在看 HEVC 编码格式视频的时候，只能通过 CPU 来实现软解码，<strong>电脑很容易就会处在一种高负载的运行状态，从而造成电脑的卡顿：</strong></p><p></p><div class="img_box" 其实在那篇文章里我少提了一点，那就是 Windows 电脑自带的 " 电影和电视 " 播放器，<strong>它实际上也不支持解码 HEVC 格式的视频。</strong><p></p><p><strong></strong></p><p>如果你把 iPhone 或者安卓手机拍的 HEVC 格式视频导入到电脑，然后选择用自带的播放器打开，它就会给你弹出一个该项目编码格式不支持的提醒：</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/6258ccceb15ec0249d5e5300_1024.jpg" referrerpolicy="no-referrer"></div></div>包括照片也是一样。<p></p><p>要是你在手机上设置了用 HEIF 格式拍摄照片，那么把这些照片原汁原味的导入到电脑上之后，大概率也无法直接用 Windows 自带的 " 照片 " 查看。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202204/6258ccceb15ec0249d5e5301_1024.jpg" data-height="559" data-width="1078" HEVC 解码，HEIF 图片的浏览也会受到影响。<p></p><p>解决的办法也不是没有。</p><p>我们只需下一个支持 HEVC 解码的播放器，或者在微软商店花 7 块钱安装 " HEVC 视频扩展 "，就可以播放 HEVC 视频或者查看 HEIF 照片了：</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" data-original="http://zkres1.myzaker.com/202204/6258ccceb15ec0249d5e5302_1024.jpg" data-height="710" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/6258ccceb15ec0249d5e5302_1024.jpg" referrerpolicy="no-referrer"></div></div> 今天托尼写这篇文章的目的嘛，<strong>就是教差友们怎么白嫖到这个 " HEVC 视频扩展 "，并且让你电脑的浏览器支持 GPU 硬解 HEVC 格式视频。</strong><p></p><p>先来说说怎么白嫖。</p><p>其实绝大部分 Windows 电脑，在出厂的时候制造商就已经预付了 HEVC 解码器授权费，所以按理说只要是符合条件的设备，理论上都能免费下载。</p><p>不过这个免费下载入口你直接在微软商店搜是搜不到的，为啥搜不到具体原因我也不是很清楚，反正目前我们只能通过其它方式来找到这个下载入口。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" data-original="http://zkres1.myzaker.com/202204/6258ccceb15ec0249d5e5303_1024.jpg" data-height="722" data-width="1080" 应用，然后在设置里点击 " 安装 HEVC 视频解码扩展 "，电脑就会自动跳出这个下载入口：</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" 解码了，<strong>就算你把哔哩哔哩客户端删除，它依旧会保留在你电脑上。</strong><p></p><p>至于让浏览器支持 GPU 硬解 HEVC 视频。</p><p>我们首先需要在微软 Edge 浏览器下载一个名为 " User-Agent Switcher and Manager " 的插件：</p><p></p><div class="img_box"
