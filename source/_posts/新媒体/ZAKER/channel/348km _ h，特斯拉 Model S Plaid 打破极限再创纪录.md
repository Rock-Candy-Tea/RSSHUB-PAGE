<p>IT 之家 7 月 1 日消息，特斯拉在推出 Model S Plaid 时表示这辆车可以达到 200 英里 / 小时（321km / h）的极速状态，但由于软件锁的存在，用户买到的车都达不到这个速度，后来特斯也也顺势推出了 " 赛道模式 "，最高速度为 281km / h。</p><p>现有这么一群人成功破解了特斯拉软件限制，再加上升级了更大的刹车套件和更高性能的轮胎（米其林 Pilot Super Sport），在一次测试中成功用这款车跑出了 216 英里 / 小时（348 公里 / 小时）的速度，从而与保时捷 918 Spyder 和法拉利 Enzo 媲美。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202207/62be62528e9f091e284a69f1_1024.jpg" data-height="696" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202207/62be62528e9f091e284a69f1_1024.jpg" referrerpolicy="no-referrer"></div></div>这是迄今为止特斯拉 Model S Plaid 或任何特斯拉汽车达到的最快速度。特斯拉此前并未表示为何不给用户提供此前承诺的 200 英里 / 小时速度，可能是刹车，也可能是电机，不过特斯拉后来还提供 20000 美元升级碳陶瓷套件，可能会在一定程度上缓解这个问题。<p></p><p>Ingenext 是一家总部位于加拿大魁北克的公司，以销售特斯拉汽车解锁模块而闻名，该公司已成功绕过 Model S Plaid 的软件限制，从而完全解锁极限速度。</p><p>实际上，Ingenext 团队之前曾在其他地方尝试过让车辆达到 200 英里 / 小时的速度，但他们确实没有足够的空间来安全跑到这个速度。</p><p>后来他们发现，Trois-Rivières 机场有一条约 3 公里长的跑道，非常适合进行此类测试。因此，今天早上，Ingenext 的 CEO Guillaume André 亲自驾驶他的 Model S Plaid 来尝试打破魁北克三河机场的最高速度记录。</p><p>IT 之家了解到，这并不是 Ingenext 团队第一次解锁 Model S Plaid 极限速度，他们之前已经达成 295km / h、305.7km / h、304.8km / h 和 303.0km / h 的成就解锁。跑完之后，甚至从雪佛兰后备箱里掏出了快充器来补能 ……</p><p>安德烈开着特斯拉 Model S Plaid 行驶了大约 2 公里才达到上述最高速度，后面随即开始进行减速（跑到 3km），而且仅仅是减速的缓存距离就超过 6000 英尺，结果证明即使这款车使用更高性能的刹车套件，他也需要几乎数千米来进行减速。也就是说，虽然这次尝试以成功告终，但他们在尝试这一操作时也伴随了极高的风险。</p><p>从他们 YouTube 来看，这辆时速 216 英里的特斯拉 Model S Plaid 声音听起来就好像是一架喷气式飞机从你面前起飞，但它实际上只是一辆普通的量产版家用 4 门轿车。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
