<p>4 月 28 日消息，微博管理员宣布，为进一步保障用户权益、提升用户体验，微博将于 4 月 28 日对用户 "IP 属地 " 功能进行升级。<strong>即日起平台全量开放评论展示发评 IP 属地小尾巴功能，上线 " 个人主页一级页面展示 IP 属地 " 功能。</strong></p><p></p><div class="img_box" id="id_imagebox_0" data-width="784" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/626a464e6227685936000013_1024.jpg" referrerpolicy="no-referrer"></div></div><strong>IP 属地功能为强制开启，用户无法主动关闭。</strong>微博会根据你近期发博、发评论和投票的 IP 属地来判定你所属的地区，国内用户将显示到省份 / 地区，国外用户显示到国家。微博表示，平台只展示用户对应 IP 属地位置，根据运营商 IP 信息进行解析，不会获取其他信息，所以不需过于担心个人隐私泄露问题。<p></p><p>从官方展示的示意图中看到，个人主页上会直接显示你的 IP 属地，点开每个人的微博主页，就能看到他现在位于哪里，是国内还是国外。另外，<strong>微博评论内容下方的小尾巴，不再显示手机型号，全部改成 IP 属地，以后数码爱好者没办法看到你是什么品牌手机用户，数码圈可能会少了些热闹。</strong></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div 功能，用户在发送内容或发表评论时，都会展示你的 IP 属地。很多人觉得这项功能会泄露用户的位置信息，但说实话 IP 属地只显示到省份，并不是精确到那个小区哪个房间，谈不上泄露个人位置信息。<p></p><p><strong>这项功能主要是为了避免有些人故意冒充事件当事人，然后进行恶意造谣、蹭热度等不良行为，尽可能保证网络内容的真实、透明。</strong>现在网络太发达，网上键盘侠和不安分子越来越多，一些人自认为平台无法追踪定位到它们，就在网上造谣、蹭热度或传播不正当内容，破坏网络的安全，也引发民众的恐慌。</p><p></p><div
