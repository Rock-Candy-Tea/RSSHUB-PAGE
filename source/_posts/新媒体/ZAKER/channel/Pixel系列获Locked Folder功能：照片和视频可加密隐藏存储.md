<p><strong>援引外媒 9to5Google 报道，在最新的 Pixel 更新中获得了 Locked Folder 功能，允许用户在主相册之外对特定的图片和视频进行加密隐藏。</strong>在 Locked Folder 页面中，Google 表示该功能目前仅适用于 Pixel 系列设备。不过在 I/O 大会上，Google 承诺将会在年内会向其他 Android 设备开放。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202106/60c032c38e9f091d2d19d358_1024.jpg" data-height="466" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202106/60c032c38e9f091d2d19d358_1024.jpg" referrerpolicy="no-referrer"></div></div>在 Pixel 3 及更高型号中，用户可以通过 Library > Utilities > Locked Folder 来启用 Locked Folder 功能。一旦设置好了，你就可以在这里看到你的秘密东西。你能够将现有的照片移动到锁定的文件夹，这将从你的常规库和图片可能出现的任何现有的备忘录中删除它们。值得注意的是，根据 Google 的说法，你不能备份锁定文件夹中的任何照片或视频，所以在隐私和便利之间有一定的权衡。如果你不再需要保持隐私，可以将锁定文件夹中的任何东西移出。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202106/60c032c38e9f091d2d19d359_1024.jpg" data-height="735" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202106/60c032c38e9f091d2d19d359_1024.jpg" referrerpolicy="no-referrer"></div></div>Google 在支持页面中还表示，用户可以在相机应用中进行设置，将拍摄好的图像直接保存到 Locked Folder 文件夹中，使它们永远不会进入你的正常资料库。这可能是有用的，例如，你在一个抗议活动中，想确保你拍的任何照片不会被其他可能拿到你手机的人看到。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
