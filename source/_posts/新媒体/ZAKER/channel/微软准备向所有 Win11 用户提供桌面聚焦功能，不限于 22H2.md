<p>IT 之家 5 月 23 日消息，微软在去年 6 月推出了全新的 Windows 11 系统，微软当时宣布每年只会发布一个大版本更新。因此总会有一些人担心他们需要等待 12 个月才能获得新的功能。</p><p>那么，我们是否要等到 Windows 11 22H2 版才能得到一些新功能呢？不一定，WindowsLatest 表示微软将提前发布 Windows 11 22H2 的新功能：桌面聚焦（Spotlight）功能。</p><p>目前来看，Windows 11 比 Windows 10 更加模块化，因此微软可以在主要功能更新之外为现有功能（如 Windows 设置）提供一些细节更新，例如 Windows 功能体验包与累积更新都可以代理少量的新特性支持。</p><p>微软已经在功能更新之外发布了 WSA（适用于 Android 的 Windows 子系统），预计将会随着秋天的 22H2 大版本一同带来。</p><p>IT 之家了解到，微软上周更新了 Win11 的 22000.706（KB5014019）发布预览版，上线了为 22H2 版本设计的桌面聚焦功能，例如每天自动帮你换壁纸。</p><p>简单来说，Spotlight 是一个很有趣的功能，微软自 Windows 8 发布以来在锁屏界面上加入了这一功能，可以利用 Bing 的图库自动更改锁屏背景，而且你还可以查阅相关图片的资料。</p><p>因为这一功能大受用户喜爱，有人想方设法地提取出了 Windows 锁屏聚焦中保存的 Bing 图片作为桌面壁纸，甚至还有开发者专门开发了自动保存锁屏聚焦背景图的小插件。</p><p>微软似乎听取了用户长久以来的反馈，并正在通过 Windows 11 Build 22000.706 向桌面添加 Spotlight 支持，用户完全不需要再等到 Win11 22H2 上线就可以用上这一功能。</p><p>当然，目前此更新仅限于发布预览版，但不出意外的话将会在几天后开始向稳定频道中的所有用户推出，敬请期待。</p><p>此前微软表示，Spotlight 是 Windows 11 22H2 所独有的功能，但微软似乎已经在通过累积更新逐步向 22H2 更新之外添加或扩展现有功能。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202205/628ae44c8e9f09023e0df31f_1024.jpg" data-height="937" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202205/628ae44c8e9f09023e0df31f_1024.jpg" referrerpolicy="no-referrer"></div></div>如上图所示，您可以通过 " 设置 > 个性化 > 背景 > 个性化您的背景 " 来打开该功能。你只需要选择 "Windows 聚焦 " 即可，之后微软将使用 Bing 壁纸自动帮你更换桌面背景。<p></p><p>微软指出，此功能的工作方式与目前已经用在锁屏界面上的 "Windows 聚焦 " 工作方式相同。您可以通过相关图标来了解有关当前背景图的更多信息。你只需要在桌面右上角点击相机图标，就可以弹出相关的 Bing 页面，从而看到每个背景图片对应的信息。</p><p>默认情况下，微软还将在桌面上为新设备自动启用 Windows Spotlight（或全新安装 Windows 11）。当然，不喜欢的话您可以随时从 " 设置 " 中禁用该功能。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
