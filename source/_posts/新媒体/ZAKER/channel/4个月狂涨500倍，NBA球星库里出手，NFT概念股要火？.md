猿猴 " 不到两小时便销售一空。</p><p>据 opensea 最新数据，该项目销售额已突破 4 亿美元，交易人数达 5400 人，最低价达 16 万美元。这意味着，仅仅 4 个月时间，该产品的价格暴涨超 500 倍。</p><p>NFT 概念股近期持续火热。8 月 28 日，NFT 交易平台 OpenSea 交易额达 2.35 亿美元，创下历史新高。其中 Crypto Punks 交易额排名第一，交易额超 4.45 万枚 ETH ( 约 1.43 亿美元 ) 。</p><p>国盛证券宋嘉吉研报指出，NFT 在艺术创作领域的应用需要在用户体验、合规性、资产性上取得平衡，稳健合规发展。今年以来，NFT 人气爆棚，NFT 艺术品售价高达数百万美元。NFT 技术在艺术创作领域有着巨大潜力，NFT 的发展为 IP 文化产业带来了新的商业模式和利润空间，为内容平台带了新的工具，文化创作经济将迎来外部推动。</p><p>中信证券 8 月 30 日发布的研报指出，未来 NFT 市场规模尚有较大发展空间，同时未来资产种类会更加丰富，更多艺术品、卡牌、NFT 游戏、以及其他新的应用或将面世。技术层面预计也有较大提升空间，包括算力平台、云计算等。此外，未来数字人民币的发展以及法律制度的完善有望共同驱动 NFT 行业持续发展。建议关注目前已有布局的互联网大厂如腾讯控股 ( 00700.HK ) 、阿里巴巴 -SW ( 09988.HK ) 、TME ( TME.N ) 、网易 ( NTES.O ) 、Roblox ( RBLX.N ) 等。同时也建议关注有潜力开展数字 IP 运营和衍生的平台如阅文集团 ( 00772.HK ) 、哗哩哗哩 ( BILI.O ) 、快手 ( 01024.HK ) 、芒果超媒 ( 300413 ) 等；探索潮流艺术价值的如泡泡玛特 ( 09992.HK ) 等；有潜力进行游戏相关 NFT 探索的如心动公司 ( 02400.HK ) 等。</p><p>经财联社梳理，A 股概念股方面，弘业股份参与江苏文交所 25% 股份，江苏文交所为文化物权、股权、版权等各类文化产权提供交易流转平台，采用 " 互联网 + 金融 + 文化 " 模式。</p><p>安妮股份旗下 " 版权家 " 版权服务平台已成为 DCI 版权保护体系首批示范应用平台；公司致力研究并推进区块链技术在版权领域的应用，参与编写《信息技术区块链和分布式账本技术参考架构》国家标准等相关文件。</p><p>博瑞传播已与中国数字图书馆有限责任公司通过西南联合产权交易所以公开摘牌方式参与成都文交所增资扩股项目。</p><p>东方电子子公司海颐软件是一家提供行业信息化解决方案、行业数据管理和运营服务的高新技术企业，公司参与过区块链艺术品交易等应用方面的研发。</p><div
