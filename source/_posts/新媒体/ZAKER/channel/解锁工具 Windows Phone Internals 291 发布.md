<p>IT 之家 8 月 12 日消息 Windows Phone 社区目前保持活力的一大因素在于解锁扩展并安装上 Windows 10/Windows 11ARM 系统，而这一切需要感谢 Rene Lergner（又名 Heathcliff74），这是因为其制作发布了 Windows 10 Mobile 解锁方案，通过 Windows Phone Internals 工具 " 破坏 " 了微软引导程序保护，实现完整版的 Windows 10/Windows 11 系统安装和运行。</p><p>而 Windows Phone Internals 工具在 2018 年进行了开源，众多开发者可以贡献自己的力量。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/6114df748e9f0979f610b4da_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家获悉，近期，解锁工具 Windows Phone Internals 发布了 2.9 和 2.9.1 版本，由 @gus33000 发布，2.9.1 版本除了使用新发布的 .NET 5 SDK 构建，修复了一些安全漏洞之外，与之前的 2.9 版本相比没有任何变化。<p></p><p>Windows Phone Internal 2.9 版本修复了各种问题并提供了大量的增强功能！</p><p>增加了对 Build 15254.603 版本的支持。</p><p>增加了备份设备配置分区的功能。</p><p>增加了从大容量存储中关机的功能。</p><p>增加了为所有规格 A 设备从大容量存储中重启的功能。</p><p>增加了对从旧版 Flash 应用（如早期固件中的应用）收集安全信息的支持。</p><p>当检测到处于启动器模式的手机时，在空视图中增加了一个新的指标。</p><p>增加了重新解锁已经解锁的 Spec B 设备的功能，在你以某种方式破坏设备上的安全启动解锁或设备被半解锁的情况下非常有用。</p><p>增加了高通紧急充电模式的记录功能。</p><p>WPinternals 现在是一个 .NET 5 应用程序。</p><p>为固件文件切换到一个新的 blob 存储服务器。</p><p>合并了 WinUSB 和 DiscUtils 的上游变化。</p><p>一般项目维护和代码清理。</p><p>更新解锁 Spec B 设备的逻辑，使其更加可靠。</p><p>提高了下载页面的可靠性。</p><p>为 Spec A 设备实施了新的解锁程序，使 UEFI SecureBoot 解锁，使安装其他操作系统更容易。</p><p>实现了对 Spec B 设备的 MMOS 支持（切换到标签模式选项）。</p><p>在 Flash 模式中实现了电池状态（可从信息视图中访问）。</p><p>在 Flash 模式下实现了保险丝配置信息（可从信息视图中访问）。</p><p>为规格 A 和一些规格 B 的设备实现了从海量存储中重启。</p><p>实现了从 Flash 模式下的关机（可从开关模式菜单中访问）。</p><p>为 Spec B 的设备实施了新的 Flash hack，提供更快和更可靠的 Flash 会话。</p><p>修复了破损的 Spec A 解锁程序。</p><p>修复了一些手机上的标签模式和终端命令的通信问题。</p><p>修复了影响 sahara 协议可靠性的问题。</p><p>修复了在解锁过程中影响 ACL 变化失败的问题。</p><p>修复了中断启动程序可能失败的问题。</p><p>修复了标签模式下的 x50 手机在断开连接后不能被检测到的问题。</p><p>改进了错误信息。</p><p>修复了影响正确识别 MainOS 驱动器字母的问题。</p><p>看起来更快。</p><p>IT 之家了解到，由于微软提供了 ARM 设备兼容的 Windows 10 版本，一群独立开发人员在过去几年里一直在维护一个非官方的 Lumia Windows on ARM 项目，这就是 Lumia WOA。现在 Lumia 950 XL 手机已经被成功刷入 Windows 11 ARM 系统。</p><div
