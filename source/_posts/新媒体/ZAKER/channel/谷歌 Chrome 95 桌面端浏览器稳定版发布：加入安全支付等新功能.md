EyeDropper API 改善了网站类应用和 PWA 应用的使用体验，同时 Chrome 现阶段还将部分 PWA 应用设置成某些 URL 类型文件的潜在默认打开应用，使 PWA 应用的使用体验更偏向于本地应用。<p></p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div
