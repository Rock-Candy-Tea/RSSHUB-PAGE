<p>据 CFM 闪存市场数据显示，自 2022 年二季度至今，渠道市场 SATA 及 PCIe SSD 平均跌幅分别达 38.8% 和 29.7%，个别容量如 120GB 和 480GB SATA SSD 跌幅均达 40% 之多。严重的倒挂行情令市场疲于杀价，尤其在以价换量见成效后，近日跌幅明显有所收窄，尤其是 1TB PCIe SSD 开始止跌，目前市场有逐渐回稳的迹象。那么，临近开学季，是时候选购 PCIe 4.0 SSD 了吗？‍</p><p>追求极致速度，必选 PCIe 4.0 SSD</p><p>对于 PCIe 传输协议，可能很多人只关注到 PCIe 4.0 SSD 和 PCIe 3.0 SSD。实际上，PCI SIG 已经官宣了 PCIe 7.0 参数。去年 Intel 第 12 代酷睿平台上已经首发了 PCIe 5.0，今年 AMD 锐龙 7000 平台也加入了对 PCIe 5.0 的支持，今年 1 月份，PCI-SIG 组织宣布 PCIe 6.0 规范标准 v1.0 版本正式发布，它被认为是 PCIe 问世近 20 年以来变化最大的一次。不到半年，PCI SIG 又宣布开发 PCIe 7.0，并前瞻了核心参数。其速度在 PCIe 6.0 的基础上再次实现带宽翻倍，达到 128GT/s，x16 通道双向可以达到 512GB/s。在消费级 PCIe 5.0 SSD 还未面世前，PCIe 4.0 SSD 就是你能选择的最快的固态硬盘了。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202208/630250068e9f0975ec655ee0_1024.jpg" data-height="347" data-width="609" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202208/630250068e9f0975ec655ee0_1024.jpg" referrerpolicy="no-referrer"></div></div>看到这里，想必大家对于 PCIe 4.0 SSD 与 PCIe 3.0 SSD 最大的区别已经明白了吧。要想追求速度，那必然选择 PCIe 4.0 SSD。可能对于有些用户而言，PCIe 3.0 SSD 已经能够满足日常工作生活需求，但是 PCIe 4.0 SSD 的速度优势在应对游戏、设计、直播、剪辑时更能提供高效稳定的传输。同时，PCIe 4.0 SSD 也根据不同用户需求提供不同价位、不同性能的选择。<p></p><p>不纠结预算的用户，可以选择带有独立缓存（DRAM）的 PCIe 4.0 SSD。带有 DRAM 芯片的固态硬盘比没有 DRAM 的固态硬盘拥有更好的性能。对于内置独立缓存的 SSD 来说，独立缓存可以提高输入 / 输出（I/O）性能和耐用性。临时保存已从闪存读取的数据、要写入闪存的数据或地址映射表的 DRAM 要比 NAND 闪存快得多。你的电脑不需要在固态硬盘上寻找相关数据，而是可以直接进入高速 DRAM 找到读写地址表。因此，你的电脑不需要等待很长时间，SSD 就可以检索或写入到它需要的数据，在各项应用中都能感受到更快的传输体验。</p><p>当然，带有独立缓存的旗舰级 PCIe 4.0 SSD 速度也不会让大家失望。目前高性能 PCIe 4.0 SSD 读取速度已经超过 7300MB/s，写入速度超过 6500MB/s，I/O 性能也达到了读写 1000K IOPS。这样的传输性能可以有效地提升多数设计类软件的使用体验和执行效率，尤其是在 3D 渲染输出、视频工程文件输出的效率问题上，也可以大大节约时间。同时，越快的 4K 写入速度可以大大缩短素材加载时间以及大量零散文件存储的时间。</p><p></p><div class="img_box" id="id_imagebox_1" DRAM-less 固态硬盘。为了让用户在使用中能够体验到快速稳定的传输，厂商利用了 NVMe 1.2 以后才有的新技术，也就是 HMB ( Host Memory Buffer ) ，即主机内存缓冲来降低无缓存带来的速度影响。<p></p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202208/630250068e9f0975ec655ee2_1024.jpg" data-height="281" data-width="554" 缓存）的部分高速存储空间来提升 SSD 的 I/O 性能，并且由于它的设计不是为了替换 SSD 内置的独立缓存，而是作为补充，因此它不会从主机中消耗大量的内存，只需几十 MB 便能够满足用户需求，可以利用你的 CPU 的 DRAM 多达 64MB 来跟踪映射，而不是在固态硬盘层面使用 DRAM。由于 NVMe 接口在主机和 SSD 主控之间提供了非常快的传输速度，因此 SSD 主控可以访问主机内存而不会造成性能损失。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" data-original="http://zkres1.myzaker.com/202208/630250068e9f0975ec655ee3_1024.jpg" data-height="400" data-width="609" 5150M/s，顺序写入速度超过 4800MB/s，也远远的超过了 PCIe 3.0 SSD 的速度极限。所以，从单纯追求产品性能而言，选择 PCIe 4.0 SSD 优势多多。<p></p><p>另外，在支持平台的问题上，AMD 和 Intel 在 2021 年可是下了狠功夫的。Intel 在 2021 年接连发布了第 11 代酷睿和第 12 代酷睿两代产品，主板也跟着推出了 Z590 和 Z690 两代芯片组。AM4 平台从 2020 年到现在依然在服役，Zen3 架构的锐龙 5000 系列搭配 B550/X570 芯片组在实际应用中甚至可以让 PCIe 4.0 SSD 获得更快的顺序读写速度。众多主板品牌匹配支持 PCIe 4.0 的处理器推出了多款产品，价格不一，也是为 PCIe 4.0 SSD 的普及做好了充足的准备。</p><p></p><div 内存）+PCIe 4.0 SSD 或者是 AMD 锐龙 5000 系列 +DDR4 内存 +PCIe 4.0 SSD。开学季来临，加上近期闪存普遍跌价，搭配电商平台以及品牌给出的优惠，可以用更惊喜的价格体验到 PCIe 4.0 SSD 更快的传输速度。<p></p><p><strong>不想换平台，PCIe 3.0 SSD 也能再战几年</strong></p><p>看到 PCIe 4.0 SSD 的强劲性能，是不是已经心动了？而这一切性能的展现，都需要在 PCIe 4.0 平台上才能实现，尤其是高端旗舰级产品更需要有性能更强的 CPU 来支持。你要是不想换平台，选择一款 PCIe 3.0 SSD 也能再战好几年。</p><p>对于很多用户来说，主流配置更多的是要求性价比高，还要能保证用四年不落伍。实际上，这个要求并不难满足的，选择 PCIe 3.0 SSD 也能玩游戏，日常学习使用完全够用。就算以后你在预算充足的时候，升级了全新平台，现在选购的 PCIe 3.0 SSD 也是可以继续使用的，毕竟 M.2 接口是向下兼容的。如果日常使用电脑的机会并不是很多，入门级 PCIe 3.0 SSD 的速度都能满足你日常制作传输一些文档、图表，以及影音娱乐需求。</p><p></p><div class="img_box" id="id_imagebox_5" 是有很多选择的。如果你在使用电脑过程中，对于传输速度并没有太高的要求，建议你将原本打算升级 PCIe 4.0 SSD 平台的预算用于选择大容量硬盘。每个人的数据量都是在不断增长的，并且增长速度不同。与其纠结那点传输速度（要是主板不支持，PCIe 4.0 SSD 速度也不会很快），不如选择更大容量的 PCIe 3.0 SSD，这才是更迫切对的需求。<p></p><p><strong>不管你选 PCIe 4.0 还是 PCIe 3.0 SSD，国产存储也有惊喜</strong></p><p>近年来，国产存储加大研发投入，在主控、闪存方面都获得了巨大的突破和进展。经历了实体清单、疫情造成的芯片荒，国产存储在这样的多重影响之下逐步获得了消费者的认可，这为以后消费级市场国产存储的进一步价格优势奠定了基础。如今国产存储正在以越来越快的研发速度追赶一线品牌，为未来市场上出现性价比更高的产品做好了准备，这对于消费者来说是非常高兴的事情。</p><p>在固态硬盘产品线中，国科微、英韧科技、联芸主控已经应用到多款国产产品中，而长江存储的闪存发展成绩也是可喜可贺的，长江存储最近已向一些客户交付了其自主研发的 192 层 3D NAND 闪存的样品，预计将在今年年底前正式推出相应产品。相信今年，国产存储品牌还将会给消费者交出更具有竞争力的新产品。</p><p>编辑｜熊乐</p><p><strong><strong></strong></strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
