<p>IT 之家 10 月 21 日消息，微软今天宣布在 Windows 上为预览用户提供 "Android Subsystem for Windows" 的第一个预览版。美国地区的小伙伴可通过在运行 Intel、AMD 和 Qualcomm 平台且符合条件的设备上加入 Beta Channel 体验新功能。</p><p>用户可以从 Microsoft Store 发现和安装 Android 应用。</p><p>通过 Snap Layouts 功能，用户可以轻松并排运行 Android 应用程序，并将其固定到 " 开始 " 菜单或任务栏，并通过鼠标、触摸或手写笔笔等输入设备进行操控。</p><p>Android 应用程序还集成到 Alt + Tab 和任务视图中，以帮助您在对您最重要的应用程序之间快速来回切换。</p><p>您可以在操作中心查看来自 Android 应用程序的通知，还可以在 Windows 应用程序和 Android 应用程序之间共享剪贴板。</p><p>许多 Windows 辅助功能也同样适用于 Android 应用，微软正与亚马逊合作以提供更多改进。</p><p>IT 之家了解到，要运行这些移动应用程序，您需要 Windows 11 版本 21H2 Build 22000 .xxx 或更高版本。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202110/61709ba88e9f09093a2e3561_1024.jpg" data-height="810" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202110/61709ba88e9f09093a2e3561_1024.jpg" referrerpolicy="no-referrer"></div></div>随后，微软发布了与适用于 Android 的 Windows 子系统相关的开发人员文档。<p></p><p>适用于 Android 的 Windows 子系统使 Windows 11 设备能够运行 Amazon Appstore 中提供的 Android 应用程序。</p><p>当用户首次从 Microsoft Store 安装 Amazon Appstore 或从 Microsoft Store 安装 Android 或 Amazon 应用程序时，Microsoft Store 将在后台自动安装 Windows Subsystem for Android（运行 Android 11）。Amazon Appstore 将提供精选的 Android 应用程序目录。</p><p>微软还确认，在 Windows 11 上运行的 Android 应用程序可以自由调整大小。这些应用程序在调整大小时应该具有响应性，并且可以使用 Windows 操作 / 手势进行捕捉。</p><p>有兴趣提交适用于 Windows 11 的 Android 应用程序的开发人员可以查看以下指南。</p><p>设置您的开发环境，包括 先决条件、 安装 Amazon Appstore 和使用 设置应用程序。</p><p>处理 Windows 设备的输入兼容性注意事项，例如： 键盘输入、 鼠标输入以及 窗口管理和调整大小。</p><p>在 Windows 11 设备上测试和调试您的应用。</p><p>将应用程序提交到 Amazon Appstore：您的应用程序必须在 Amazon Appstore 中可用才能在 Windows 11 设备上运行。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
