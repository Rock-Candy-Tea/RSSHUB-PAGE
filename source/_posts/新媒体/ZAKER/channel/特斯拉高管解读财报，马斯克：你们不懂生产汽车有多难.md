<p>4 月 27 日消息，据外媒报道，当地时间周一，电动汽车制造商特斯拉公布了 2021 年第一季度财报，显示其营收近 104 亿美元，每股收益 0.93 美元，业绩超过华尔街预期，并连续七个季度实现盈利。</p><p>财报发布后，特斯拉投资者关系主管马丁 · 维埃查（Martin Viecha）、首席执行官埃隆 · 马斯克 ( Elon Musk ) 、首席财务官扎卡里 · 柯克霍恩 ( Zachary Kirkhorn ) 以及其他高管出席了财报电话会议，解读财报并回答了投资者与分析师的提问。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" referrerpolicy="no-referrer"></div></div><strong>马斯克：特斯拉再次刷新了季度纪录</strong><p></p><p>马斯克首先介绍了特斯拉第一季度总体业绩情况，称特斯拉在该季度再次刷新了以往纪录。他说，特斯拉已经看到人们对电动汽车的看法发生了变化，需求比以往任何时候都更加强劲。</p><p>马斯克表示：" 电动汽车需求是我们见过的最好的，这也是今年第一季度业绩与往年相比如此之好的原因。过去，第一季度业绩往往比其他季度相比稍显疲软。"</p><p>马斯克还提到了 Model 3 在豪华电动轿车市场取得的胜利，它甚至击败了宝马 3 系等老牌车型。至于 Model Y，这款车有可能成为世界上最畅销的汽车。马斯克估计，这种情况将在 2022 年的某个时候成为现实。</p><p>至于特斯拉的全自动驾驶（FSD）软件，测试始终在取得进展，尽管马斯克承认这是最难解决的技术问题之一。马斯克还强调了特斯拉只关注视觉的做法，重申了他之前在推特上关于 " 雷达最终将失败 " 的声明。</p><p>马斯克称，解决 FSD 的问题需要有海量数据支持。在这种情况下，特斯拉已经占据优势，因为其有超过 100 万辆汽车上路，这应该有助于该公司处理小概率情况。马斯克说：" 尽管相当棘手，但我们有信心完善 FSD。"</p><p>马斯克在回应目前困扰汽车业的芯片短缺问题时表示：" 在第一季度，我们经历了特斯拉面临过的最艰难供应链挑战。"</p><p>至于 Model S 和 Model X，马斯克指出，挑战比预期的要多。他列举了这些车辆的诸多改进，比如采用了新的内饰、电池组、电机，推出了许多新功能。马斯克还补充说，特斯拉只是对已经生产的汽车进行了改进，5 月份可能会增加产量。</p><p>马斯克表示，Model X 应该在 2021 年第三季度增加产量。他说：" 我们的目标是每周生产 2000 辆 Model S 和 Model X，我相信这两款旗舰车型的需求会很大。"</p><p>马斯克还称，新款 Model S 和 Model X 的生产成本实际上更低，德国柏林和美国得克萨斯州工厂明年可能会进行批量生产。最后，他感谢特斯拉团队的努力。</p><p>柯克霍恩解释了 Model S 和 Model X 的延误，并承认延误对公司的财务来说是个不利影响因素。他还表示，由于持续的全球供应短缺，特斯拉正在经历诸多挑战，不过该公司正在与合作伙伴合作解决这些挑战。</p><p>柯克霍恩透露，特斯拉出售其持有的约 10% 比特币，获利 2.72 亿美元。</p><p><strong>散户投资者提出五大质疑</strong></p><p>散户投资者开始提问，<strong>首先是关于自动驾驶计算机 Dojo 项目的问题</strong>。马斯克指出，目前人们认为特斯拉是一家汽车和能源公司，但从长远来看，人们应该把特斯拉视为机器人公司。他说：" 我认为我们正在建立世界上最强大的硬件和软件团队。" 看看特斯拉的技术发展情况，人们会发现，特斯拉已经到了需要比市场提供更强大东西的水平。</p><p>为此，特斯拉创建超级计算机来帮助训练其神经网络具有重要意义。马斯克在描述 Dojo 项目背后的想法时说：" 想象下这样的系统：它拥有非常好的眼睛，可以同时看到所有方向，永不知疲倦，有冗余性，反应时间超人，这样的系统将实现高度的安全水平。" 我们拥有 100 多万辆上路的汽车，这会提供庞大的数据。明年，这个数字将增长到 200 万辆。</p><p>马斯克表示，特斯拉实际上在自动注释方面变得越来越擅长，这几乎是神经网络训练的 " 终极目标 "，也是 Dojo 的设计目标。他说：" 我们认为 Dojo 在硬件和能源使用方面的成本效益，可能比我们所知的第二好解决方案高出一个数量级。可能其他人也会使用它，我们愿意与你们共享。"</p><p><strong>第二个问题与太阳能屋顶有关，</strong>投资者们想知道这款产品的价格上涨以及增产情况。马斯克透露，太阳能屋顶的需求非常强劲，但他承认，特斯拉在评估安装太阳能屋顶瓷砖的难度时犯了错误。他说：" 你不可能有一个适用于所有人的系统。"</p><p>马斯克重申了特斯拉将 Powerwall 及其太阳能产品捆绑在一起的决定，并补充说，去年生产的电池具有更高的峰值容量。有了捆绑包，马斯克表示，安装的难度会下降很多，安装人员甚至不需要碰房子的断路器。根据马斯克的说法，这对控制安装规模很重要。</p><p>马斯克继续讨论 Powerwall 的潜力，称这种家用电池提供的虚拟发电厂能力令人印象深刻。考虑到世界正在迈向 " 电动汽车成为常态 " 的时代，这一点尤其值得注意。有鉴于此，必须有一种方法来生产更多的电力，以满足全电动化未来所产生的需求。</p><p>考虑到特斯拉的使命，这一转变将对该公司有利。马斯克说：" 对于特斯拉和电力公司来说，这代表着繁荣的未来。否则，我们将看到更多今年早些时候得克萨斯州寒流带来的电力供应危机，电力公司将无法为客户提供服务。"</p><p><strong>第三个问题，柯克霍恩被问及有关加密货币的问题。</strong>他重新提供了有关特斯拉比特币投资和最终出售的信息，以及该公司支持比特币支付的决定。根据柯克霍恩的说法，特斯拉始终在寻找储存现金的方式。考虑到传统投资提供的回报比特币要少得多，比特币被认为是实现这个目标的更好途径。</p><p>柯克霍恩说：" 比特币是保存特斯拉现金的好地方，而且能够从中获得更多回报。" 考虑到特斯拉在几个月内变现了超过 2 亿美元的收益，这个决定看起来绝对是值得的。他表示：" 我们始终对比特币市场的流动性感到满意，并坚信比特币的长期价值。"</p><p><strong>第四个问题，有关特斯拉汽车事故问题。</strong>马斯克表示，媒体对特斯拉汽车在得克萨斯州撞树起火事件的报道 " 极具欺骗性 "。据该公司汽车工程副总裁拉尔斯 · 莫拉维 ( Lars Moravy ) 表示，特斯拉正在与得克萨斯州政府就致死车祸展开合作。</p><p>特斯拉已经与相关机构进行了研究，以了解车祸发生的更多细节。根据调查结果，汽车的方向盘似乎变形了，撞车时有人坐在驾驶座上。车里所有的安全带都解开了。</p><p><strong>第五个问题与车辆生产有关。</strong>马斯克表示，人们仍然不理解生产的困难。他说，大规模生产是极其困难的，特斯拉是百年来第一家实现汽车量产且未破产的公司。马斯克还说：" 特斯拉经历了几次生存危机才能做到这一点。"</p><p>马斯克分享了特斯拉多年来经历的诸多挑战，比如因为地毯和 USB 端口等琐碎问题而被迫停产等。他称：" 解决这些问题简直令人疯狂。"</p><p><strong>分析师更关注新电池、FSD 等细节</strong></p><p>New Street Research 分析师皮埃尔 · 费拉古（Pierre Ferragu）询问了有关 4680 电池容量以及目前所取得的进展。马斯克透露，特斯拉有个小型试点工厂，用于生产 4680 电池，年产能为 10GWh。</p><p>他说：" 我们的电池还没有达到足够可靠、可以装进汽车的地步，但我们认为它已经接近这个门槛儿。我们已经订购了在柏林和奥斯汀工厂生产电池的设备，我们对待这件事的态度非常严肃，并坚信明年能实现 4680 电池量产。"</p><p>马斯克还补充说，看起来特斯拉距离 4680 电池的量产还有大约 12 个月或不超过 18 个月的时间。与此同时，特斯拉也在寻求与现有供应商合作生产 4680 电池。4680 电池革命并不是特斯拉独自推动的，宁德时代、LG 和松下等都有参与。最终，特斯拉有望将供应商的产量提高一倍以上。</p><p>费拉古还询问特斯拉能源业务的后续情况。马斯克表示，特斯拉在能源和汽车领域拥有相当高的利润率，但应该指出的是，该公司的电动汽车业务比能源领域的探索更成熟。然而，Powerwall 业务已经成型，而且利润率相当不错。马斯克说：" 我们有明确的路线图来降低 Megapack 的成本。"</p><p>沃尔夫研究公司分析师询问有关特斯拉推出 FSD 软件的情况，例如订阅模式及其影响。柯克霍恩表示，特斯拉正在努力推出 FSD 订阅服务，尽管这项服务的某些方面仍需更多努力。他说：" 我们希望很快就能推出，并看看人们对此有什么反应。"</p><p>瑞士信贷分析师丹 · 利维 ( Dan Levy ) 询问特斯拉及其正在进行的成本削减举措。特斯拉说，制造汽车是个复杂的过程，但如果看看该公司生产某些车型 ( 如 Model Y ) 取得的进步，就会发现该领域正在进行许多创新，这应该会改善主营业务成本（COGS）。</p><p>总体而言，马斯克及其团队在今年电话会议上的发言似乎显得有点儿谨慎，预测像 4680 电池这样的项目时间表时更为保守。特斯拉也没有提供 2021 年汽车交付量的具体预测。但考虑到目前的芯片和零部件供应短缺挑战，这可能不完全是个糟糕的策略。</p><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
