股市场一个新的风口：NFT。</p><p>自从比特币、区块链兴起以来，新奇概念层出不穷，数字世界似乎正在加速变化。元宇宙已经让人费解，NFT 浪潮又席卷而至。</p><p>据市场调查机构 Chainalysis 报告，2021 年 NFT 市场规模已经至少达到 269 亿美元，约合人民币 1700 亿元。小朋友的致富之路，就是通过发行 NFT 数字艺术品，赚到大笔以太币。</p><p>新奇事物总伴随争议。徐静蕾、林俊杰等明星都持有过大火的 NFT 头像，而看好 NFT 的支持者认为，艺术收藏品将仅仅是 NFT 应用的一小部分，其前景要远为广阔。</p><p><strong>但也有区块链从业者向市界指出，关于 NFT 是否有价值，每个人的看法都很片面。有的项目是在认真布局，有的则属于「滥竽充数」。</strong></p><p>可以肯定的是，NFT 浪潮正吸引越来越多关注。12 月 21 日，英国电信公司沃达丰成功将世界上第一条短信以 NFT 形式拍卖，成交价 10.7 万欧元，约合人民币 76.9 万元。</p><p>NFT 到底是什么？其价值与风险何在？本文尝试给出解答。</p><p></p><div Days》（以下简称「《5000 天》」）的艺术品。</p><p>这是一幅巨大的拼贴作品，由数字艺术家 Beeple 过往 5000 多天的作品汇聚而成。不过其特殊之处倒不在内容，而在形式。与大众熟知的实物艺术品不同，《5000 天》属于纯数字化的 NFT 作品。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div 万美元成交，创造了新的纪录。NFT，这个大众仍旧陌生的事物，成了名副其实的造富神器。</p><p>NFT，全称为 Non-Fungible Token，一般翻译为<strong>非同质化通证，或者非同质化代币。它是区块链技术的一种应用。</strong></p><p>第一个 NFT 项目，是诞生于 2017 年 6 月的 CryptoPunks。它由纽约一所实验室的夫妇发起，生成一系列像素风的朋克角色头像。头像总数为一万个，包括 3840 个女性，6039 个男性，以及一些猿人、外星人等。</p><p>最开始免费发放，感兴趣的人不多。但之后逐渐受到追捧不断升值，<strong>其中一枚头像，在 2021 年 3 月卖出了 4200 枚以太币的高价。以当前行情计算，约合人民币 1 亿元。</strong></p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" 诞生四个月后，一款名为 CryptoKitties（加密猫） 的游戏，也大大推动了 NFT 市场的火爆。</p><p>眼下，艺术收藏品和游戏是 NFT 应用最受关注的领域。尤其是 NFT 数字艺术品，受到不少名人关注。</p><p>马斯克很早就表现出了对 NFT 的兴趣，他曾表示，<strong>NFT 是美元的另一种形态</strong>。马斯克还在今年 3 月份宣布，将「把一首关于 NFT 的歌作为 NFT 来出售」。而在体育节，NBA 球星库里花费大约 18 万美元购买 NFT 头像。</p><p>中国娱乐圈同样有人参与，徐静蕾就曾在社交媒体上更换过 NFT 头像，胡彦斌将自己珍藏的歌曲《和尚》未公开 demo 铸成 2001 份的 20 周年纪念黑胶 NFT，作为七夕节礼物送给粉丝。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" data-original="http://zkres2.myzaker.com/202112/61c6afdbb15ec01a3801d82c_1024.jpg" 市场正迅速膨胀。据相关行业报告，NFT 在 2021 年的交易规模将至少达到 269 亿美元。</p><p>那么，NFT 到底有何价值？又为何能够流行？</p><p><strong>02</strong></p><p><strong>一个和元宇宙有关的故事</strong></p><p>探讨 NFT 的价值，需要从它的特性出发。</p><p>作为非同质化代币，NFT 不同于比特币等同质化代币，具有不可拆分和唯一性。凭借这种特性，NFT 成为锚定各类资产的数字凭证。</p><p>简单点说就是通过 NFT 可以给各种东西打上个「戳」，有了这个「戳」全世界都知道这东西是你的，并且不可被擦除，且永久记录。</p><p>具体方法，是将该资产的相关权利内容和历史交易等信息，记录在智能合约里，并在对应的区块链上，给该资产生成一个无法篡改的独特编码。</p><p><strong>NFT 相当于一种所有权凭证。除了前文提到的艺术品，游戏皮肤、装备、虚拟地产等，都可以通过 NFT 来锚定。</strong></p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_6" data-original="http://zkres2.myzaker.com/202112/61c6afdbb15ec01a3801d82d_1024.jpg" data-height="810" data-width="1080" 可以大大提高数字内容的可交易性。</p><p>网络中一张普通的照片，谁都可以复制、传播，原作与复制品并无不同。但是通过 NFT 锚定的照片，具有唯一性。用户将真正且永久地拥有这份数字内容的所有权，从而有助于版权保护。</p><p>同时对艺术创作来说，NFT 还有一个极具吸引力的特性，<strong>就是当其 NFT 作品被不断转售时，艺术家依然可以收取一定比例的版税。</strong></p><p>这些特性无疑将激发艺术创作的活力。</p><p>此外，随着数字技术发展，数字作品的创作门槛不断降低，也推动了 NFT 的发展。不经过长期学习没办法创作油画，但借助软件，普通人也可以创作 NFT 作品</p><p>但 NFT 的潜能不止于此。致力于打造高性能区块链公链平台的 Conflux 向市界表示，如果把 NFT 限定于艺术，就把它说小了。</p><p><strong>从发展前景来看，NFT 还与备受科技大厂追捧的元宇宙概念相关。</strong></p><p>其品牌总监刘畅指出，元宇宙很大的问题在于经济系统的建立，NFT 能够给元宇宙中的资产提供流动性，有助于解决这个问题。</p><p>实际上，业内对 NFT 的看好，很大程度在于它解决了确权问题。除了数字艺术品，NFT 甚至可以和实体资产结合，为用户提供资产证明。</p><p>比如通过 NFT 记录在区块链上的房产信息，能够进行信息溯源，保证数字内容上链后的真实性和所有权归属。</p><p>从技术上看，<strong>上链信息的安全有充分保障。因为篡改信息需要掌控超过 50% 区块链算力，几乎不可能实现。</strong></p><p>这就大大拓展了 NFT 的想象空间。眼下，NFT 已经引起国内企业关注。阿里巴巴推出了 NFT 交易平台蚂蚁链粉丝粒，腾讯推出了 NFT 交易平台幻核 App，网易文创旗下的三三工作室联合 Nervina Labs，发行了 NFT 项目「小羊三三纪念金币」。</p><p>对新兴概念向来敏感的 A 股市场，当然也没错过这股潮流。近期受追捧的一些公司，便兼具元宇宙和 NFT 两种概念。</p><p>美盛文化近一个月股价上涨 68%。2021 年 9 月，美盛文化公司参股公司元気星空宣布与全资子公司同道大叔建立战略合作伙伴关系，打造星座 NFT。同时元気星空签约艺术家，完成米巧铭「熊与机器人」系列主题等 NFT 作品的发布。</p><p>汤姆猫近一月股价上涨 15.7%。其在 9 月底答复投资者提问时表示，将针对「会说话的汤姆猫家族」IP，开展 NFT 数字收藏品授权业务。</p><p>视觉中国近一个月股价上涨 23.4%。<strong>在半年报中，视觉中国表示，将根据视觉数字艺术资产属性与特点，利用 NFT 技术对社区进行升级，对创作者持续赋能，增强资产变现能力。</strong></p><p>不过有区块链业内人士并不看好上述企业布局。他指出，真正的 NFT 产品应该是部署在区块链公链之上，而国内公司的 NFT 产品往往并非如此。</p><p>比如，蚂蚁粒粉丝链基于阿里巴巴的蚂蚁链进行确权和交易，幻核则基于腾讯的致信链。</p><p>另外有区块链从业者指出，在版权保护方面，核心问题是确权，但 NFT 存在漏洞。<strong>「上链的资产只能证明你是第一次上链，但不能证明因为你是第一次上链，所以版权是你的」。</strong></p><p>也就是说，NFT 应用前景的根基并不稳固。二级市场对 NFT 概念的追捧，难以摆脱炒作嫌疑。</p><p><strong>03</strong></p><p><strong>财富冒险游戏</strong></p><p>价值不会被凭空制造。NFT 的造富故事伴随乱象。前述区块链从业者向市界描述了 NFT 近半年状况。</p><p>据该人士介绍，<strong>在 6、7 月的时候，NFT 圈以炒头像为主，之后到 11 月，以炒游戏为主。</strong>区块链游戏发行方会让用户去购买游戏里的装备或道具。而这些装备、道具也属于 NFT，可以到相应的交易平台。</p><p>一些道具和装备，确实可以炒到很高的价格。但问题在于，这到底是击鼓传花的炒作，还是说真正有用户认可了游戏？区块链游戏的头部产品 Axie Infinity，普及了边玩边赚（play to earn）的模式。</p><p>据天风证券总结，这一游戏经济系统由 SLP/AXS 代币，以及战斗、繁殖三个核心模块组成：</p><p>新玩家向老玩家购买游戏里面的宠物 Axie，凑够三只 Axie 组成队伍，通过战斗赢取 SLP 代币；游戏本身通过消耗 SLP 代币和支付 Gas 费用（一种交易手续费）繁殖新的 Axie；战斗和繁殖获得的 SLP 和 Axie 再放进市场出售，来获取收入。</p><p></p><div class="img_box" id="id_imagebox_7" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_7" 用于维持游戏运转的成本以及活动奖励。<p></p><p>如果说乐趣是引导人行为的根本动力，那么利益就是更深层的动力。<strong>Axie Infinity 这种边玩边赚的特性，让一些人看到了赚钱的希望，从而促成了游戏本身的流行。</strong></p><p>先是菲律宾，后来是印度、印度尼西亚、巴西，Axie Infinity 越来越受欢迎。根据 Dappradar 数据，Axie Infinity 在 2021 第三季度产生了超过 7.76 亿美元的交易额，截至 10 月 16 日，Axie Infinity 历史交易总额达到 25.7 亿美元，排在 NFT 收藏项目的第一名。</p><p>Axie Infinity 仿佛打开了新世界的大门，其他游戏公司开始闻风而动。前述区块链相关从业者向市界表示，今年，尤其一些传统游戏公司，扎堆去找投资人，「有些投资人可能一天得收到三四十份 BP（商业计划书）」。</p><p><strong>概念火了，相应的项目方案就多了，里面有专心做事的，也不乏一些「滥竽充数」的。</strong></p><p>对于 NFT 的发展，即便在区块链这个圈子里，大家的意见尚未统一。</p><p><strong>无论是比特币还是 NFT，其价值很大程度上依赖于人们的共识。</strong>认可它们的人越多，概念本身就稳固，市场规模就会随之扩大。但与加密货币引起的狂热相比，NFT 还不够火。</p><p>关于元宇宙，刘畅认可罗永浩曾引述的观点：<strong>当我们生活在虚拟世界的时间远远超过在现实世界的时间，元宇宙就会来临。他相信，未来五到十年，这个时刻就会来临。因此，刘畅相信 NFT 前景广阔。</strong></p><p>不过前述区块链从业者对 NFT 的价值存疑，在他看来，可以把 NFT 当做一个投资或者投机品。「就像炒股，有些公司的基本面不行，但就是有人在炒。」</p><p>总的来说，有的投资者确实看好 NFT 前景，有的则是比较纯粹的投机谋利。关于 NFT 价值的探讨，每个人的看法都片面，也不会有标准答案。该人士投资了多种 NFT 产品，有的价值已经翻倍，有的则亏损过半。</p><p>回到关于 NFT 产品的造富神话上，12 岁小男孩可以凭借 NFT 在两个月赚 34 万美元，诚然激动人心。然而完整的故事是，这位小男孩从 5 岁起就接触编程，在推出「怪异鲸鱼」之前，他还创作了一款 NFT 艺术收藏品，销量并不好。</p><p><strong>艺术界对 NFT 也存在分歧。前佳士得拍卖师曾劝告，购买 NFT 毫无意义。</strong></p><p>至于 NBA 球星花 18 万美元买一个 NFT 头像的故事，看起来出手豪阔。不过要知道，库里年收入 4000
