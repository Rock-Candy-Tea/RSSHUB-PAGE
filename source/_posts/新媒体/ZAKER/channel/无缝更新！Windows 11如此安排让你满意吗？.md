<p>10 月 5 日，Windows 11 就会正式亮相了，关于其更新，你会担心受到影响吗？</p><p>微软详细介绍 " 无缝更新 " 概念</p><p>当微软在 6 月宣布 Windows 11 时，该公司承诺将提供比此前 Windows 10 累积更新体积小 40% 的更新包。现在，微软的技术人员终于解释了 Windows 11 将如何提供 " 无缝更新 "，这些更新将在后台进行。预计 Windows 11 将遵循 Windows 10 的更新节奏，因此可以期待每个月都有新的累积性更新且每年秋季都有功能更新。</p><p>除了更少的功能更新外，Windows 更新质量的提高也来自于对 Windows 中的更新引擎进行的优化。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202109/615588d38e9f0940a329e536_1024.jpg" data-height="311" data-width="553" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/615588d38e9f0940a329e536_1024.jpg" referrerpolicy="no-referrer"></div></div>图片来源：微软官方<p></p><p>例如，当你检查更新并有一个新的补丁可用时，Windows 现在会将新的软件包与计算机上已有的内容进行比对，只从服务器上下载必要的部分，这将使更新的大小减少约 40%。</p><p>微软官员进一步解释说，同样的机制也将适用于 Windows Update for Business 或微软 Endpoint Manager 等服务。企业可以使用配置管理器管理驱动程序和关键安全更新。另一方面，消费者可以依靠 Windows 11 的可选更新功能。</p><p>微软已经承诺，Windows 11 的更新过程将减少 " 对用户的影响 "。换句话说，像活动时间这样的功能现在将被默认打开，微软将使用新版本的 AI/ML 机器学习技术来确定用户没有在积极使用 PC 的时间以保证尽量少打扰用户，Windows 更新将在这个时候被自动下载并在完成后提示用户安装。</p><p>值得注意的是，Windows 11 的更新过程却不会比 Windows 10 大幅加快，所以用户仍然需要重新启动机器，然后盯着那个屏幕几分钟到几十分钟不等。</p><p>微软目前计划在 10 月 5 日推出 Windows 11。</p><p>微软正在完成最后准备</p><p>Windows 11 进入了发布预览频道，如果你安装了兼容的驱动程序，现在就可以在适合的设备上进行测试了。不过需要注意，Windows 11 距离正式版还剩下一点调整，因此在 Windows 11 Build 22000.194 中可能会有一些小问题。</p><p>微软正在计划在未来几周改善 Windows 11 21H2 版本的整体体验，这是微软推动服务的正常节奏。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202109/615588d38e9f0940a329e537_1024.jpg" data-height="293" data-width="553" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202109/615588d38e9f0940a329e537_1024.jpg" referrerpolicy="no-referrer"></div></div>图片来源：微软官方<p></p><p>目前，Windows 11 正在通过微软的计划提供给预览版的测试中。就如截图所示，用户访问 " 设置 ">" 更新和安全 ">"Windows Update"，检查更新，并在注册发布预览通道后，手动选择 Windows 11 的 " 下载和安装 " 选项，即可升级到 Windows 11 的预览版。</p><p>而如果选择 " 暂时停留在 Windows 10"，那么就会得到 Windows 10 21H2 的推送。</p><p>如果你的设备硬件兼容没有问题，那么就可以很简单地获得 Windows 11 的升级，这个过程是非常直接的。Windows 11 的升级包大小可能在 3GB 左右，算是比较大，这是因为它比过去几年发布的 Windows 10 功能更新提供了很多变化。</p><p>Windows 11 将会带来开始菜单的新外观，新的通知中心，新的微软商店，新的收件箱应用程序等等。</p><p>重新设计的开始菜单没有动态磁贴，任务栏的功能也有所不同，这是 Windows 11 的最大变化之一。而微软目前还没有完成对开始菜单的改动，微软目前正在开发一个名为 " 登录选项 " 的新模块，这会出现在电源选项的位置。</p><p>不过，即使你的设备兼容 Windows 11，也并不一定就能在 10 月 5 日收到 Windows 11 正式版的推送，届时更新将会只提供给部分设备，如果推送没什么问题，那么在几周后才会面向更多兼容设备推送 Windows 11 正式版。如果你等不及，也可以使用 Media Creation Tool 等工具来手动升级。</p><p>CPU 等硬件不兼容仍可安装 Win11</p><p>过去的 3 个月中争议最多的就是 Windows 11 的硬件兼容了，微软这次设定的门槛很奇特，会导致很多人无法升级 Windows 11。在硬件兼容问题上，包括 4GB 内存、64GB 存储、UEFI 安全启动等在内的要求没啥问题，主要是 TPM 2.0 及 CPU 要求，其中 CPU 需要八代酷睿及锐龙 2000 系列及之后，前不久微软放宽了部分七代酷睿支持 Windows 11，但主要局限于高端型号，大部分七代酷睿 CPU 依然不兼容。</p><p>对于达不到硬件兼容的设备，微软表示可以通过官方的 Meadia Tool 工具自行安装，但是微软表示这种安装方法不能保证稳定，崩溃次数比兼容设备高了 52%。</p><p>除了崩溃更多，现在非兼容设备上安装 Windows 11 还会有另外的代价，那就是 Windows 更新不能用了，微软不会给这类设备提供更新维护。</p><p>以上两个政策还不管用的话，微软现在还推出了另外的政策，那就是在不兼容设备上安装 Windows 11 系统，会弹出提示称这台 PC 没有达到安装要求，微软不建议执行系统升级之类的操作，设备也不在厂商的保修范围内，您的设备就不受支持或者无权获得更新。</p><p></p><div class="img_box"
