<p>IT 之家 8 月 16 日消息 微软上周推送了新版本的 Windows 11 预览版，版本号为 Build 22000.132，带来多种新版小工具 UI 更新。</p><p>除了 Windows 计算器、邮件、日历等应用程序，微软已经开始在 Windows Insider ( Dev Channel ) 中向测试人员推出全新的截图工具。</p><p>新版截图工具结合了 Windows 经典的截图工具以及 " 截图和草图 " 应用的优点，它现在只有一个简单的名字 " 截图工具 "。IT 之家了解到，这款应用程序已经升级了 Windows 11 全新的 WinUI 控件，可带来更时尚的外观和体验。</p><p>外媒 WindowsLatest 为我们展示了这款 Windows 自带的 " 剪切工具 " ( Snipping tool ) ，演示了微软究竟做了哪些努力，例如它可以区域截屏、为截图添加标记，并保存截图等。</p><p>在为 Windows 11 升级全新 WinUI 之后，微软准备将其带到更多内置工具中。</p><p>截图工具</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202108/6119b2a1b15ec01f443782a4_1024.jpg" data-height="435" data-width="798" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202108/6119b2a1b15ec01f443782a4_1024.jpg" referrerpolicy="no-referrer"></div></div>首先，我们使用键盘快捷键或 Windows 搜索打开新版截图工具。<p></p><p>单击 " 截图模式 " 并下拉菜单，选择你所需的形状。此时屏幕变暗，你此时即可拖动方框进行截图。松开鼠标之后，你刚刚截的图便会出现在截图工具中。之后，你还可以使用笔或其他工具添加符号以及印记。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202108/6119b2a1b15ec01f443782a5_1024.jpg" data-height="776" data-width="1288" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/6119b2a1b15ec01f443782a5_1024.jpg" referrerpolicy="no-referrer"></div></div>这个新的截图工具充分结合了此前两款经典应用的优点，用户可以在其中使用所有功能。因此外媒认为，Windows 11 带来了比 Windows 10 更好的截图体验。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
