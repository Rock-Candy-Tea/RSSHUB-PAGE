<p><strong>Steam Deck 可以安装 Windows 系统，但之前存在缺乏驱动问题。现在 V 社已经大部分解决了这个问题，推出了 Steam Deck Windows 驱动程序。</strong>目前有 GPU、Wifi 和蓝牙驱动，但还缺乏声音驱动。需要注意的是，Steam Deck 目前只能运行 Windows 10 系统，不支持双启动。这意味着如果你安装 Windows，那么它就是你能使用的唯一系统，除非你卸载它。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202203/622ab47c8e9f095a1a767aa6_1024.jpg" data-height="348" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202203/622ab47c8e9f095a1a767aa6_1024.jpg" referrerpolicy="no-referrer"></div></div>V 社还在想办法支持 Windows 11 和双启动，不过这可能要在将来某个时候才能实现。<p></p><p>声音驱动还不支持，但 USB-C 或蓝牙可以提供音频。如果安装了 Windows，扬声器和 3.5 毫米端口将无法传输声音。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202203/622ab47c8e9f095a1a767aa7_1024.jpg" data-height="343" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/622ab47c8e9f095a1a767aa7_1024.jpg" referrerpolicy="no-referrer"></div></div>如果你想安装 Windows，用户可以选择更不可逆的 SSD 安装或是安全第一的 SD 卡 /U 盘安装。不管哪一种，基本的方法是在 U 盘上下载 Windows 驱动程序。<p></p><p>首先关机，然后进入启动菜单，然后，按住降低音量键，再次按下电源键。在启动菜单中，你将选择 U 盘 ( 或 microSD 卡 ) 来安装 Windows。遵循那里的指示。</p><p>需要指出的是，由于 Steam Deck 的 Windows 驱动还非常新，处于起步阶段，所以外媒 GameSpot 还是建议玩家谨慎一些，现在不要安装 Windows，除非将来支持双启动。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202203/622ab47c8e9f095a1a767aa8_1024.jpg" data-height="353" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/622ab47c8e9f095a1a767aa8_1024.jpg" referrerpolicy="no-referrer"></div></div><strong>Steam Deck Windows 驱动程序现已推出</strong><p></p><p>大家好！</p><p>我们是否提过 Steam Deck 是一台 PC ?
