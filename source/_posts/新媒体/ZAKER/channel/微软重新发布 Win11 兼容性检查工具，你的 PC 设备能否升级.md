<p>IT 之家 8 月 28 日消息 外媒 Windows Latest 报道，想把你的设备升级到 Windows 11，但不确定 Windows 11 的兼容性？微软终于重新发布了 Windows 11 兼容性检查工具，现在可以很容易地弄清最新版本的操作系统是否与你的设备兼容。</p><p></p><div class="img_box" id="id_imagebox_0" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202108/612980cc8e9f0930eb43491f_1024.jpg" referrerpolicy="no-referrer"></div></div>Windows 11 于 2021 年 6 月宣布，其新的设计功能将使其成为对一些用户有吸引力的升级。微软后来宣布了对其下一个操作系统的严格要求，很多用户不能从他们现有的 PC 免费升级到 Windows 11-- 因为它不符合必要的硬件要求。<p></p><p>英特尔第八代核心 CPU、Ryzen 2000 系列及以上的 PC 正式支持 Windows 11。根据 Windows Insiders 遥测数据，微软现在已经增加了对部分英特尔第七代芯片的支持。根据更新的列表，Windows 11 现在将在以下英特尔第七代芯片上运行。</p><p>英特尔酷睿 X 系列芯片。</p><p>英特尔 Xeon W 系列处理器。</p><p>英特尔酷睿 i7-7820HQ（曾用于 Surface Studio 2 等设备）。</p><p>换句话说，微软正在放弃对旧的和广泛使用的英特尔第七代酷睿处理器的支持。英特尔第六代和其他处理器也不被支持。事实上，微软自己的 Surface 阵容中也有很多设备不符合 Windows 11 升级资格。</p><p>微软在硬件要求上立场坚定，看起来旧的处理器不会被正式支持。</p><p>检查 Windows 11 的兼容性</p><p>为了确保你的系统能够运行 Windows 11，微软开发了一款名为 "PC Health Check" 的工具。使用这个工具，用户可以检查他们的硬件和软件是否能与最新版本的 Windows 一起工作。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy 健康检查工具，该工具此前在用户抱怨该应用程序不能正常工作后被移除。<p></p><p>IT 之家获悉，PC 健康检查工具作为微软 Windows Insider 预览版计划的一部分可供下载，现在可以安装它来检查你的硬件和软件是否能与 Windows 11 一起工作。如果你的设备配备了较新的处理器、TPM 2.0 和 UEFI 安全启动，则正式支持 Windows 11。</p><p>如何使用 PC 健康检查工具检查 Windows 11 的兼容性</p><p>要知道你的设备是否支持 Windows 11，请采取这些步骤。</p><p>进入设置 > 系统 > 关于，并验证你的架构。</p><p>打开微软的下载页面，选择正确版本的 PC Health Check。如果你的设备支持 64 位 PCU 架构，选择第一个版本。</p><p>双击 WindowsPCHealthCheckSetup.msi 来安装该工具。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2"
