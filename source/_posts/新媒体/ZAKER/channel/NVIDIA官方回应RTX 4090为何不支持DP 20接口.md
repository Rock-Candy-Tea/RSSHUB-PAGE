<p>日前，NVIDIA 发布了 RTX 4090 和 RTX4080 显卡，7199 到 12999 元的售价引发了网友的广泛吐槽。NVIDIA 对此的解释是圆代工价格上涨，显卡涨价也是身不由己。而另一个引发广大网友热议的焦点就是，售价近万元的显卡却依然采用的是 DP 1.4a，而没有使用最新的 DP 2.0 接口。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202209/632cb5af8e9f0909ab1fb09b_1024.jpg" data-height="177" data-width="593" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/632cb5af8e9f0909ab1fb09b_1024.jpg" referrerpolicy="no-referrer"></div></div>对此，NVIDIA 官方也给出了回应称，目前 DP 1.4 接口已经能支持 8K 60Hz 输出，消费级显示器支持 DP 2.0 接口，还有一段路要走。这也说明，NVIDIA 官方认为现在 DP 1.4 完全可以满足日常使用，毕竟支持 DP 2.0 的显示器并不多，成为主流趋势还为时尚早。<p></p><p>DP 2.0 标准于 2019 年推出，其理论带宽已经达到 80Gbps，同时还采用了全新的 128/132b 编码机制，将有效率提升至 97%，实际可有带宽达到了 77.4 Gbps，相当于当前 DP 1.4 的三倍左右。不过，虽然 DP 2.0 标准是免费的，但是其商业化支持并不算好，所以显卡厂商对支持 DP 2.0 并不积极。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
