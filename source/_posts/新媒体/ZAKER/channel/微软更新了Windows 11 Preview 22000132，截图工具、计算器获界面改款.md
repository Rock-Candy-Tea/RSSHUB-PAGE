<p>微软自前不久公布了 Windows 11 以来，测试版系统的更新就很积极，上个月底是 100 尾号，现在最新已经出到 132 了，而这一次 Windows 11 Insider Preview Build 22000.132 更新，除了那些修复 bug 和改进稳定性的常规变化外，最主要是放在系统自带基础 app 的改款上。</p><p>为此微软也发布了博文来介绍有那啥 Windows app 获得更新，包括有截图工具、计算器、邮件以及日程表，其实主要还是 app 的 UI 风格上换用 Windows 11 全新风格，毕竟这些 app 都属于小应用，功能本就是那样子的。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202108/61164fd38e9f0923fe782030_1024.jpg" data-height="960" data-width="1200" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/61164fd38e9f0923fe782030_1024.jpg" referrerpolicy="no-referrer"></div></div>当中截图工具（Snipping Tool）算是比较大改变的一个，它现在整合了原来的截图工具以及 Snip & Sketch（这个在 Windows 10 上就是没有中文名的）两个，前者大家也不陌生，而后面一个是截了图之后，还可以在上面手写画图加标注，所以其实功能上是很大重复的，两者整合才是合理，现在截图工具仍保持了徽标键 +Shift+S 的快捷键组合，可以区域、全屏或者窗口这些常规截图操作，然后还可以编辑，并支持 Dark mode 系统主题等等。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202108/61164fd38e9f0923fe782031_1024.jpg" data-height="931" data-width="1200" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/61164fd38e9f0923fe782031_1024.jpg" referrerpolicy="no-referrer"></div></div>至于计算器、邮件和日程表就主要是界面更新了，也就是有了圆角边框和毛玻璃效果，不过微软表示这计算器是用 C# 语言编写的，并且把代码开放到 GitHub 上供大家参考，而值得一提的是，其实计算器除了最简单的标准模式，还有科学、图形、编程、日期计算，以及货币、容量、长度单位转换等等，这可能是不少人不知道的（Windows 10 上早就有了）。<p></p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202108/61164fd38e9f0923fe782032_1024.jpg" data-height="642" data-width="1200" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/61164fd38e9f0923fe782032_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
