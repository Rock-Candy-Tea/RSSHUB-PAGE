<p>Windows 11 总体上有很多改进，包括新功能、改进的开始菜单、对 Android 应用的支持等等，但是在他们的设备上安装第一个预览版的用户还遇到了其他一些更微妙的变化。其中之一似乎与臭名昭著的 BSOD 有关，与以前的 Windows 版本相比，它显然带有一种新的颜色。</p><p><strong>01</strong></p><p><strong>Win 11 中的蓝屏死机界面大变样</strong></p><p>正如人们过简单地阅读它的名字可以很容易地发现，蓝色死亡屏幕应该是蓝色的，但是在 Windows 11 中，报错屏幕带有黑色背景。</p><p>虽然很难说这是不是一个临时的变化，还是微软想让这个屏幕即使在 Windows 11 的完整版本中也是黑色的，但即使是永久的转换，BSOD 的名称仍然有效。不过，希望黑屏死机不会出现得太频繁，因为 Windows 11 总体上应该更加稳定和可靠。</p><p>不过，作为测试计划的一部分而发布的构建版本带有绿色背景，因此从技术上讲，它们也被称为绿屏死机，因为微软希望将这些错误与 Windows 10 稳定版本中发生的错误更好地区分开来。这非常合理，所以目前 Windows 11 的黑色版本可能是基于同样的方法，尽管显然没有理由微软不会在预览程序中坚持使用绿色。</p><p>该公司没有提供任何关于黑屏死机的信息，所以看看这个黑屏死机是否坚持在操作系统的稳定版本中，或者蓝色的报错屏幕是否回归，将会很有趣。</p><p><strong>02</strong></p><p><strong>文件夹图标不再显示内部文件预览</strong></p><p>说实话，Windows 10 已经很少遇到蓝屏死机了，在目前曝光的 Windows 11 大改中，" 文件夹图标不再显示内部文件预览 " 反倒更让人关注一些。</p><p>在 Windows 11 系统中微软对文件夹图标进行了重大改进，当这些视觉上的变化受到不少用户的肯定和欢迎同时，有个忽略的细节就是新图标不再显示文件夹内容的预览。网友 OmNomDeBonBon 在注意到这个细节之后分享在 Reddit 上，这一变化意味着你无法再轻易看到一个文件夹内有哪些文件。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202107/60dedf0c8e9f0963ee56dbfa_1024.jpg" data-height="752" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60dedf0c8e9f0963ee56dbfa_1024.jpg" referrerpolicy="no-referrer"></div></div>这一变化可能解释了 Windows 11 中的一些速度改进，目前还不清楚微软是否计划在正式版中重新引入这一功能。还有一个问题是，这种预览在水平文件夹下会是什么样子，一些 Windows 11 用户建议 Windows XP 风格的缩略图将是完美的解决方案。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202107/60dedf0c8e9f0963ee56dbfb_1024.jpg" data-height="411" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/60dedf0c8e9f0963ee56dbfb_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
