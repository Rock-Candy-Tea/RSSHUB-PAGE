<p>IT 之家 11 月 15 日消息，微软近日在美区微软官方线上商城上架了一款基于高通骁龙平台的 Windows on ARM PC 开发套件，型号为 ECS LIVA Mini Box QC710 Desktop，售价 219 美元（约合人民币 1397 元）。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202111/6191e5d8b15ec03b4566e72d_1024.jpg" data-height="394" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/6191e5d8b15ec03b4566e72d_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家从美区微软官方线上商城了解到，该套件的整体外观为黑色，尺寸为 35.05 mm x 119.13mm x 116.58 mm，重量 0.23kg，在设备的顶部中心位置有一个大大的内凹开关按钮。该套件采用高通骁龙 7c 计算单元（Compute）SC7180，微软表示该款芯片可为平台提供快速的相应能力，该芯片发热量低，无需风扇即可高效运行。其他硬件方面该套件搭配 4GB 内存以及 64GB 存储空间，随机附带 64 位 Windows 10 家庭版操作系统。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202111/6191e5d8b15ec03b4566e72e_1024.jpg" data-height="394" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/6191e5d8b15ec03b4566e72e_1024.jpg" referrerpolicy="no-referrer"></div></div>接口配置方面，ECS LIVA Mini Box QC710 Desktop 开发套件拥有一个 USB Type – C 接口（支持 PD 充电）、一个支持 USB 3.2 Gen 1 的 USB Type – A 接口、一个支持 USB 2.0 的 USB Type – A 接口、一个 MicroSD 卡插槽、一个 HDMI 视频输出接口（规格暂时未知）以及一个以太网接口。无线连接方面该开发套件支持 Wi-Fi 5 且支持蓝牙（规格暂时未知）。<p></p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202111/6191e5d8b15ec03b4566e72f_1024.jpg" data-height="394" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/6191e5d8b15ec03b4566e72f_1024.jpg" referrerpolicy="no-referrer"></div></div>ECS LIVA Mini Box QC710 Desktop 开发套件目前可在美区微软官方线上商城购买，但现阶段仅面向开发人员出售。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
