<p>没人知道<strong>外星文明</strong>的数量到底是多还是少。像 "<strong>大过滤器</strong>" 之类的概念限制了外星文明的存在。但一些科学家相信，宇宙中高等文明的数量与它们的平均寿命成正比。文明平均寿命跨度可以从一个世纪到几百万年。那些已经延续了上百万年的文明一定已经发展成为了高度发达的文明。因此，离我们最近的外星文明或许可以近到探测到电磁信号或者被我们的望远镜捕获。传统意义上科学家认为地外文明离我们的距离大约在<strong>200 光年</strong>。</p><p>甚至达尔文也指出，并非所有的个体一开始都具有相同的特征。因此，影响生存的力量对个体和物种的影响并不相等。总有一些变异被证明更适应生存，而且比其他变异更容易生存。在探索星际生命及其形式的过程中，我们 ( 人类 ) 发展出了许多理论，以解开生命隐藏的奥秘。其中一个理论是<strong>黑暗森林</strong>，它恐怖到会让人起鸡皮疙瘩。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " referrerpolicy="no-referrer"></div></div>在介绍黑暗森林理论之前，让我们先了解几个重要的关键概念。这些概念与黑暗森林理论同样重要，也是这一假说的构成背景。<p></p><p>这些概念包括：</p><p><b>德雷克方程</b></p><p>下面的八个术语导致了在搜寻外星文明方面的一场革命。它们是：</p><p>N = R* · fP · Ne · fl · fi · fc · L</p><p>N 是可以与人类沟通的文明的数量；</p><p>R* 是恒星形成的平均速率；</p><p>fp 是拥有行星的恒星在全部恒星中所占的比例；</p><p>Ne 是每个拥有行星的恒星系中宜居行星的平均数；</p><p>fl 是在宜居星球中发展出生命的比例；</p><p>fi 是有生命的星球中演化出智慧生命的比例；</p><p>L 是能够对外交流的所有文明的平均演化时间。</p><p>法兰克 · 德雷克（Frank Drake）等科学家发展出了一套可以估算外星生命出现概率的公式。这个公式（德雷克方程）用于估算银河系中活跃而有对外交流能力的地外文明的数量。德雷克方程总结了影响我们探测到外星文明所发出无线电信号的可能性的各种因素。</p><p>但目前科学家尚未发现可以明确证明外星人存在的无线电信号。（尽管我们已经探测到了几种疑似的无线电波，但这些证据并不充足。）</p><p><b>费米悖论</b></p><p><b> ——在寻找外星人方面，一个永恒的热点话题</b></p><p>这个悖论说的是，银河系中有几十亿颗恒星——其中一定有一些恒星拥有类地行星——这些恒星中一定有一些远比我们的太阳更古老——所以几百万年前它们就可能孕育出了智慧生命——一些智慧生命或许已经在探索进行星际旅行——所以外星人很久之前就到访过地球——但人们并没有发现外星人到访的确切证据。</p><p>无数的潜在宜居行星与毫无进展的生命搜寻构成了费米悖论的两个方面，并且得到了经验数据的支持。第一个论断，也就是存在相当数量的宜居行星，由费米提出，并且由于系外行星频繁出现，这个论断得到了理论支持。根据现有的模型，银河系中存在数十亿颗宜居星球。对于费米悖论的第二部分，也就是人类尚未发现外星生命存在的证据，科学家们正在进行研究。研究既包括了对任何形式生命的普遍搜寻，也包括对<strong>智慧生命</strong>的偏重。1960 年以来，科学家进行了许多类似的研究。</p><p>你可能会认为，<strong>费米悖论看起来只是一个推理过程</strong>。是的，它确实是一个推理过程，一个很复杂的推理过程。</p><p><b>方法</b></p><p>在 1950 年到 1970 年期间，科学家们认为生命是在类似孤岛的环境中演化的，所以物理上的交流被认为是不可能的。交流的唯一方法是通过<strong>电磁信号</strong>。因此，SETI、OZMA、CETI（译注：均为早期外星探索项目）等许多项目上马，向深空中发射信号。</p><p>但到目前为止，还没有一种被称为外星或神秘信号的电磁波被精确地证实过。所谓的 "<strong>大沉默</strong>" ——也就是缺乏地外文明存在的证据 ( 来自大卫 · 布林的文章《大沉默——关于地外智慧生命的争议》 ) ——改变了人类搜寻外星智慧文明的方式。</p><p><strong>摩尔斯信息</strong><strong>。</strong>这包括用摩尔斯电码向金星发送无线电信号，科学家们当时仍然认为金星可能适宜居住。另一种方法是借助无线电信号：<strong>主动 SETI</strong> ( Active Search for Extra-Terrestrial Intelligence，主动搜寻地外智慧生命 ) 项目尝试向有智慧的地外生命发送信息。主动 SETI 信息通常以无线电信号的形式发送。</p><p>类似的还有<strong>METI</strong> ( Messaging to extraterrestrial Intelligence，向外星智慧体发送信息 ) 。SETI 研究的是寻找来自外星人的信息。METI 负责创造信息并向外星人传送。</p><p>METI 的一条信息说 :" 你并不孤单 !"</p><p><b>我们预期中外星生命的类型：</b></p><p>科学家尚未对生命做出精确的定义。此外，也没有一种单一的方法可以来定义生命的存在与否。当谈到生命时，我们想到的是我们地球上出现的生命，但外星生命可能不同。例如，我们认为生命具有以下特性 : 碳基，有膜，消耗能量，需要液态水，进行细胞分裂、生长、繁殖和代谢活动，等等。</p><p><strong>原核生物</strong>：是一种单细胞生物，比如细菌。</p><p>科学家霍伊尔（Hoyle）和他的团队提出了一个与单细胞生命有关的理论。他在望远镜接收的信号中发现了冻干大肠杆菌的 3~9 μm 红外特征——这意味着太空中可能充满了适应深空生活的单细胞生物。</p><p><strong>智能生物</strong>：比如我们，或海豚、黑猩猩、类人猿，或可能是人类——类似于我们自己的物种。</p><p><strong>智慧生物</strong>：智慧生命意味着以智慧行事的存在，例如，罗马人将女神密涅瓦视为智慧存在。</p><p><strong>未知的 / 不能说的 / 快的和更先进的</strong>：我们甚至无法预测这些外星生物。我们不知道它们可能有什么特征。</p><p>康奈尔大学的卡尔 · 萨根（Carl Sagan）进行了大量的实验室研究，他试图模拟行星、卫星和彗星的表面和大气化学。这些发现表明，在对太阳系的多种观测下，常见的能源 ( 如紫外线辐射和放电 ) 可以促进复杂有机分子的大量形成，其中包括生命体中的化学成分。这些发现最初受到了一些质疑。目前已知，这些复杂有机物存在于巨大的星云内部以及太阳系外许多行星的表面。宇宙中似乎有很多构成生命的基元。萨根认为生命无处不在。</p><p><b>益处</b></p><p>科学家们正在寻找因为找到外星生命的<strong>益处</strong>。——不只是人类，其他物种也一样。——不只是其他地球物种，而是整个宇宙。我们首先要服务<strong>地球生命</strong> ( 包括所有物种 ) ！我们的要求是 :</p><p><strong>技术</strong>——如果外星人到达地球，那将说明它们具有技术优势。我们绝不会错过乘坐他们的飞碟飞出银河系的机会。</p><p><strong>联系</strong>——我们不想独自生存。多年来，我们囿于宇宙的一角而对其他地方一无所知。我们想要走出这个深渊，我们希望整个宇宙和平。</p><p><strong>我们的生存</strong>——我们的资源有限。随着时间的推移，我们的人口将会增长。仅有的一个地球不足以满足我们的需求。在我们的外星兄弟姐妹的帮助下，我们可以在极端条件下生存 ( 当世界末日来临 ) 。这个生存点是黑暗森林理论的核心思想。</p><p><strong>探索太空</strong>——也许去其他星系度假——谁不想呢 ? 我想。你想。每个人都想。</p><p>一旦与外星人接触，我们就可能通过他们的先进技术拓宽我们的视野，了解宇宙的未解之谜。这些都是与地外文明交流的好处。现在来谈谈 " 在沟通之前 " 必须牢记的事情。</p><p><b>黑暗森林理论</b></p><p>这个词来自于中国科幻作家刘慈欣的系列小说《<strong>三体</strong>》。黑暗森林假设外星人故意保持沉默，这解释了我们为什么没有听到外星人的消息。该理论的要点是 :</p><p>" 宇宙就是一座黑暗森林，每个文明都是带枪的猎人，像幽灵般潜行于林间，轻轻拨开挡路的树，竭力不让脚步发出一点儿声音，连呼吸都小心翼翼 … 他必须小心、因为林中到处都有与他一样潜行的猎人。如果他发现了别的生命不管是不是猎人、不管是天使还是魔鬼，不管是娇嫩的婴儿还是步履蹒跚的老人，也不管是天仙般的少女还是天神般的男孩，能做的只有一件事：开枪消灭之？在这片森林中，他人就是地狱，就是永恒的威胁，任何暴露自己存在的生命都将很快被消灭。这就是宇宙文明的图景，这就是对费米悖论的解释。"</p><p><strong>——刘慈欣，《黑暗森林》</strong></p><p>这一概念有两个基础：</p><p>生存是每个文明的第一需要</p><p>文明不断增长和扩张</p><p>就连理论物理学家史蒂芬 · 霍金（Stephen Hawking）也说人类必须保持沉默，避免吸引外星人的注意。因为外星文明可能会被地球吸引，它们拥有比光速更快的交通手段，来寻找我们宝贵的资源。</p><p><b>黑暗森林理论的重要性</b></p><p>黑暗森林理论为思考提供了一个新的维度。我们不知道高度先进的外星文明是敌是友。这些文明可能把我们视为伙伴、猎人乃至猎物。生存所需的资源十分紧缺，每一个这样的文明都迫切需要获得资源来扩张和生存。现在，我们知道太空中的沉默是有原因的——也许高度发达的文明害怕被发现——这可能是他们至今没有发出任何信号的原因。</p><p>我们的文明和外星文明之间缺乏理解，这可能会引起怀疑。我们无法知道外星物种的真正意图，它们也无法知道我们的意图。因此，在这种情况下，考虑攻击其他文明将被认为是明智的选择。通过这种方式，黑暗森林的假设可能会看到现实之光。而且，最明智的决定就是保持沉默，不暴露自己。</p><p><strong>达尔文的进化论和黑暗森林有某种相似性。达尔文的自然选择进化论基于三个重要因素 : 变异、繁殖和遗传。变异与生存密切相关。生物体物理特征的变化往往有利于一个物种的生存斗争，这些变化被保存下来并传递下去，因为拥有这些变异的个体往往能生存下来。黑暗森林理论 ( Dark Forest Theory ) 的情况也类似。黑暗森林假说让我们想起了生存本能，它同时指出适者生存不仅适用于我们的星球，而且适用于整个宇宙，在外太空交流时必须给予适当的考虑 ……</strong></p><div
