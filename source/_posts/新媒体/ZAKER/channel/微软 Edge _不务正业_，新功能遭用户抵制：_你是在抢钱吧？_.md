功能</strong></p><p>本月中旬，<strong>微软宣布了一项正在测试的新功能 BNPL（Buy now, pay later），即支持用户在 Edge 上购物时分期付款，且 " 通常是免息的 "</strong>。</p><p>据介绍，在美国一般只有 Target、Walmart 等特定电子商务网站才会提供 BNPL 功能，但现在微软通过与金融服务公司 Zip 的合作，实现了在 Edge 浏览器提供 BNPL 支付选项，适用范围为 35 美元 - 1000 美元内的订单交易，可在 6 周内分 4 期付款。</p><p>为了便于操作，微软将这一支付选项的申请流程设置得较为简便：</p><p>在结账页面，可在输入卡号时找到 BNPL 选项：</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/61a5caa68e9f0971572b6863_1024.jpg" referrerpolicy="no-referrer"></div></div>用户可通过把 Microsoft 帐户与 Zip 帐户关联，省去从 Zip 端登录这一步，直接在 Edge 中选择 BNPL。<p></p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" 频道推送的 Microsoft Edge 测试版中可用，默认会推送给所有使用 Microsoft Edge 96 版的用户。<p></p><p><strong>网友：" 真没必要 "</strong></p><p>在介绍完 BNPL 这一功能后，微软还补充道 " 如果您在使用此功能时遇到任何问题，欢迎及时反馈 "。如它所愿，用户们的反馈非常激烈，只不过普遍都不是使用感受，而是针对 BNPL 这个功能是否有存在的必要。</p><p>@Cameron_Bush：" 这听起来像是一个糟糕的想法，它只会被媒体视为无耻的抢钱行为。<strong>希望你们好好考虑一下这件事，为了这个功能换来众多负面报道不值得啊</strong>。"</p><p>@BioTurboNick：" 我专门注册了一个帐户来回应：<strong>拜托，不要为了这些收入让 Edge 浏览器进一步变得臃肿好吗？</strong>这样很像在重复 90/00 年代你们给 IE 浏览器加的那些默认安装的糟糕功能。这种情况如果继续下去，那我将不得不放弃 Edge 了。"</p><p>@Matt Gajownik："<strong>我们使用浏览器不是为了这个功能好吗？</strong>"</p><p>@amsprich：" 我喜欢新的 Edge，自从它问世以来我一直在使用它。但<strong>说真的，不要增加这个功能，根本没有人需要它！</strong>"</p><p>@MWehrstedt：" 这个想法太可怕了，<strong>我最多只能接受它作为可选的扩展程序而不是浏览器必须附带的功能</strong>。而且，这项功能只在美国可用啊。"</p><p><strong>不务正业，抢购物网站的活？</strong></p><p>从这些网友的留言可以看出，他们对 Edge 浏览器揽了购物网站的功能这一点很不能理解，或者可以说，<strong>自从 Edge 开始不顾用户反馈，不断增加一些 " 不务正业 " 的功能时，用户的不满就已经逐渐滋生了。</strong></p><p>在用户的呼声中，最令他们不满的就是 Edge 在购物方面投入过多：降价提醒、推荐可靠商品、自动查找优惠券、价格比较工具等一系列本该是购物网站而非浏览器应该提供的功能。</p><p>诚然，这些功能对部分习惯通过 Edge 购物的人来说十分友好，但对那些只希望拥有一款快速浏览器的用户而言实在是个负担：这些功能大多作为 Edge 浏览器自带功能，在后台进行无休止的数据跟踪，消耗很多资源，导致浏览器甚至设备本身的运行速度都变得很慢。当然，这些功能也可以禁用，但需要用户手动设置，即出一个新功能就禁一个，颇为繁琐。</p><p>如果说 Edge 能很好兼顾它作为浏览器的职责，" 捯饬 " 这些新功能倒也还能理解，但问题在于，有许多网友反映 Edge 并没有想着提升 " 本职工作 "：</p><p>GPU 进程已经开始消耗大量内存；</p><p>数学求解器也是个没用的功能，不是每个用户都还是小孩；</p><p>打印预览也非常慢，可以努力改进这一性能；</p><p>在 PDF 上的搜索速度也不容乐观；</p><p>…</p><p>正因如此，此次微软宣布 BNPL 功能才引起了用户的强烈反对：<strong>不想着解决当前遗留问题，成天捣鼓这些大多数用户视为 " 鸡肋 "、拖垮浏览器性能的无用性能</strong>。</p><p>最后，引用一位网友的发言："Edge 不是一款普通的浏览器，<strong>它还是全球桌面操作系统市场份额第一的 Windows 默认自带浏览器。这对 Edge 而言，不仅是一种好处，更是要确保对所有用户负责</strong>。"</p><p>对于此事，你有什么看法？你常用的浏览器也有这种问题吗？</p><div
