<p>据外媒报道，压力相关的精神疾病 -- 包括重度抑郁症和创伤后应激障碍 ( PTSD ) 的生物学机制目前尚不清楚。<strong>现在，一项新研究详细阐述了参与控制身体应激反应的蛋白质之间的相互作用，另外它还指出了这种反应出错时的潜在治疗靶点。</strong></p><p>这项研究是由麦克莱恩医院的研究人员领导的国际团队展开，相关研究已发表在《Cell Reports》上。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202107/60ed384e8e9f097da02ae5ba_1024.jpg" data-height="467" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/60ed384e8e9f097da02ae5ba_1024.jpg" referrerpolicy="no-referrer"></div></div>研究强调：<p></p><p><strong>新研究揭示了关键蛋白质如何相互作用来调节身体对压力的反应；</strong></p><p><strong>针对这些蛋白质可能有助于治疗或预防与压力相关的精神疾病。</strong></p><p>这项研究的首席作者 Jakob Hartmann 博士指出：" 身体的应激反应失调会损害大脑并促进情绪和焦虑症的易感性。" 据悉 ，Hartmann 是麦克莱恩恐惧实验室神经生物学的助理神经科学家，另外还是哈佛医学院精神病学讲师。</p><p>Hartmann 称：" 海马体是调节压力反应的关键大脑区域。当我们注意到三种重要的应激调节蛋白在海马体定位上的有趣差异时我们产生了进行这项研究的想法。"</p><p>研究人员在非人体组织和死后脑组织中进行的实验揭示了这些蛋白质 -- 糖皮质激素受体 ( GR ) 、矿物皮质激素受体 ( MR ) 和 FK506 结合蛋白 51 ( fkbp5 ) -- 是如何相互作用的。</p><p>具体来说，在正常条件下，是 MR 而非 GR 控制 FKBP5 的产生。在压力情况下，FKBP5 会降低 GR 对结合应激激素的敏感性。FKBP5 似乎可以通过调节海马体中 MR:GR 平衡来微调应激反应。</p><p>这项研究的高级作者 Kerry J. Ressler 医学博士会出：" 我们的研究结果表明，GR、MR 和 FKBP5 的治疗靶向可能在控制中枢和外周应激调节方面是互补的。"Ressler 是麦克莱恩医院的首席科学官、麦克莱恩抑郁症和焦虑症部门的主任、哈佛医学院的精神病学教授。</p><p>" 此外，我们的数据进一步强调了磁共振信号在跟压力相关的精神疾病中的重要作用，但很大程度上还未被重视。这项研究的发现将为未来的研究开辟新的方向，"Ressler 补充道。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
