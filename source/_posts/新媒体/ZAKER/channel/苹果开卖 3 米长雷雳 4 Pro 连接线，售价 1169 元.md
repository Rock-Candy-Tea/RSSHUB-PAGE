<p>IT 之家 5 月 4 日消息，苹果最近开卖了 3 米长的雷雳 4 Pro 连接线，可用于将 Mac 连接到苹果的 Studio Display 显示器和其他兼容的设备，售价为 1169 元。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202205/627295e98e9f096ce90fee2f_1024.jpg" data-height="773" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/627295e98e9f096ce90fee2f_1024.jpg" referrerpolicy="no-referrer"></div></div>据官方介绍，这条 3 米的连接线采用<strong>黑色编织设计，卷起时不易缠结</strong>，并支持雷雳 3、雷雳 4 和 USB 4 数据传输 ( 速率最高可达 40Gb / s ) ，USB 3.1 第二代数据传输 ( 速率最高可达 10Gb / s ) ，DisplayPort 视频输出 ( HBR3 ) 以及最高可达 100 瓦的充电能力。<p></p><p>用户可以使用这条连接线将配备雷雳 3 或雷雳 4 ( USB-C ) 端口的 Mac 连接至配备雷雳 ( USB-C ) 和 USB 端口的显示器和设备，如 Studio Display、Pro Display XDR、基座和硬盘。</p><p>IT 之家了解到，苹果此前开卖了 <strong>1.8 米长的雷雳 4 Pro 连接线</strong>，售价为 949 元。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
