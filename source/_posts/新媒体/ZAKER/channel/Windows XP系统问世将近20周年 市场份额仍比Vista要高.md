<p>微软的 Windows 系统有过很多经典，对当前一代的网民来说，Windows XP 系统可能是最难忘的，国内很多人触网都在 Windows XP 平台，明天它就要满 20 岁了。<strong>微软在 2001 年 2 月 5 日公布了 Windows XP 系统，当年 8 月 24 日发布了 Windows XP 系统的 RTM 正式版，它取代了 1999 年的 Windows 2000 系统，</strong>不过正式上市是当年的 10 月 25 日。</p><p>相比之前的系统，Windows XP 在那个年代有很多创新，其中最主要的就是 UI 界面，同时还改进了性能，启动速度更快，也更稳定，当然更经典的还有 IE6 浏览器，这个更让人记忆深刻了。</p><p>对国内用户来说，很多人也恰好是 Windows XP 时代开始接触电脑、上网，国内最出名的当属各种 Ghost 魔改版 XP 系统，包括番茄花园、雨林木风等等，后来还惹上了微软的官司，这种魔改系统才逐渐消失，现在只有爱好者制作的非盈利版 Windows 系统了。</p><p><strong>Windows XP 系统在 2002 年到 2008 年之间升级了三次 SP3 服务包，最终在 2009 年结束了主流支持，</strong>不过扩展支持支持到了 2014 年，在微软的 Win 系统中相当长寿了。</p><p>时至今日，Windows XP 早就不是主流了，但它在全球依然不算灭绝，<strong>统计网站 Statcounter 的数据显示它依然有 0.59% 的份额，比 Vista 的 0.26% 依然高出一倍多。</strong></p><p>不过话说回来，这个网站是国外的，国内很多 Windows XP 用户并不知道更不会访问这个网站，所以 Windows XP 的实际份额更高，国内一些老机器可能还在坚守 Windows XP。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202108/6123551d8e9f0935357a40cf_1024.jpg" data-height="525" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202108/6123551d8e9f0935357a40cf_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
