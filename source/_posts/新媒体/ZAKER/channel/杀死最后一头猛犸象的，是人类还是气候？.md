<p>17,000 多年前，一头雄性猛犸象来到了生命的尽头，在阿拉斯加停下了自己的脚步，在 28 年的时光里，它曾和象群一同迁徙，也曾离开象群独自旅行 ......</p><p>最终，它倒下了。在它的象牙上，时光静止在了 28 岁，它死亡的那一天。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div </p><p><strong>象牙上，是猛犸象的一生</strong></p><p>说起象牙，大家想到了什么？象牙制品？偷猎？</p><p>其实，比起被打磨加工成工艺品，象牙对于科学家而言，或许有着更加重要的意义。对于象而言，若非象牙折断等意外情况，<strong>象牙几乎会伴随着象的一生</strong><strong>，从出生到死亡</strong>，就像树的年轮一样，日复一日，年复一年，<strong>作为最忠诚的记录者，记录了 " 象的一生 "</strong>，即便是比现代象块头大上许多的猛犸象也不例外。</p><p>就像之前为大家介绍过的鲸的耳屎一样（戳这里查看原文），<strong>科学家会通过象牙来了解猛犸象活了多久，并追踪猛犸象的活动轨迹，甚至找到一些和猛犸象灭绝有关的珍贵线索</strong>。</p><p>今年 8 月，一项发表在《科学》上的研究就从一根 1.7 米长的象牙入手，为我们演绎了一部有关 "<strong>冰河时期一只猛犸象的生死之旅</strong>" 的连续剧。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202111/6192f5a08e9f09361e1916e4_1024.jpg" data-height="400" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/6192f5a08e9f09361e1916e4_1024.jpg" referrerpolicy="no-referrer"></div></div>猛犸象牙的横截面，生长线被染成了蓝色<p></p><p> ( 图片来源：Nature ) </p><p>具体而言，研究人员先是将象牙从中劈开，暴露出其中的生长层，再通过激光在猛犸象的象牙上对 340,000 个点进行取样并分析其化学成分。而后，他们将每个样点的同位素分布与阿拉斯加，以及加拿大西北部的地质图进行了比较，并从发现猛犸象遗骸的位置进行回溯，<strong>通过算法绘制出了这头猛犸象最有可能走过的路线</strong>。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202111/6192f5a08e9f09361e1916e5_1024.jpg" data-height="531" data-width="593" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/6192f5a08e9f09361e1916e5_1024.jpg" referrerpolicy="no-referrer"></div></div>猛犸象的迁移路线，虚线表示可能的轨迹，白色的象形为遗骸发现处 ( 图片来源：参考文献 1 ) <p></p><p>根据这幅路线图，我们可以简单窥见这头猛犸象的一生：</p><p>年轻时，这头公象的足迹遍布育空河（Yukon River）流域和广阔的阿拉斯加内陆，在这段时间里，它有规律地在不同的领地区块 ( 图中黄色和橙色区域 ) 之间进行着重复的长途旅行，这和现代象群的迁徙行为十分类似，<strong>预示着年轻的猛犸象或许是随着象群进行迁徙的</strong>；</p><p>在大约 16 岁的时候，通过象牙中多样化的同位素模式，研究人员推断<strong>它可能离开了象群，开始自由迁移</strong> ( 图中橙色和红色的区域 ) 。在这段时间里，这头公象的活动轨迹开始变得不太规律，并维持了 10 年之久；</p><p><strong>在生命最后的时光里，这头公象的活动范围骤然变小</strong>，缩小到了北极圈内阿拉斯加北部海岸附近的一个区域 ( 图中的灰色区域 ) ，并最终在这里结束了自己不算太长的一生。</p><p>值得一提的是，象牙底部记录了一种独特的同位素模式 --<strong>δ 13 C 减少，而 δ 15 N 大幅增加，这被认为是哺乳动物饥饿的明显标志，说明饥饿可能是导致这头公象死亡的最终原因</strong>。</p><p>1.7 米，28 年，34 万次取样，科学家们通过象牙里的同位素为这头公象谱写了独一无二的传记，我们固然无法回到过去，但不断发展的技术让我们得以回溯 17000 多年前一头猛犸象的一生。</p><p>不过比起一头猛犸象的故事，人们更关心的或许是这个物种为何走向了灭绝。</p><p><strong>人类活动？气候剧变？</strong></p><p><strong>什么给了猛犸象致命一击</strong></p><p>一直以来，人们都通过<strong>化石</strong>或者<strong>掩埋在冻土里的</strong><strong>遗骸</strong>来对猛犸象进行研究，但由于化石或者遗骸的稀缺，我们很难准确地断言猛犸象何时消失，以及什么才是导致猛犸象灭绝的确切原因。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" data-original="http://zkres2.myzaker.com/202111/6192f5a08e9f09361e1916e6_1024.jpg" data-height="400" data-width="537" 年在俄罗斯发掘的一头冰冻猛犸象<p></p><p>（图片来源：inverse.com）</p><p>科学家们就猛犸象灭绝的原因争论了 100 多年。</p><p><strong>一些研究认为人类活动是猛犸象灭绝的罪魁祸首</strong>。具体而言，科学家们指出，猛犸象是一种适应能力极强的生物，在它们游荡在地球的数百万年间，足迹遍布除了澳大利亚和南美洲以外的所有大陆。在这期间，气候发生了多次动荡，猛犸象们经历了数个冰河时期，族群依旧繁盛。<strong>但随着人类活动范围的扩张，猛犸象和人类密切接触后，一切都变得不同了，" 巨兽们 " 在很短的时间里消失殆尽</strong>。</p><p>古人类在猎杀猛犸象</p><p>（图片来源：Wall street journal）</p><p>今年 2 月，一篇发布在预印本平台上的研究指出，<strong>人类活动加速了猛犸象的灭绝</strong>。研究人员分析了在有人类活动和没有人类活动情况下各个不同区域 ( 欧洲、亚洲，以及白令海峡 ) 的猛犸象丰度，并进一步模拟了这些区域有 / 无人类活动时猛犸象灭绝的时间差异，结果让人大吃一惊，<strong>在某些区域，人类活动甚至使得猛犸象的灭绝进程加快了 4000 年</strong>。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy 2）<p></p><p>但另外一些研究则认为，导致猛犸象灭绝的并不是人类，<strong>给予猛犸象致命一击的是气候的剧变</strong>，这里画个重点，<strong>气候的剧变不仅代表了气候的变化，更是强调了气候变化的快速</strong>。</p><p>为了在缺少化石和遗骸的情况下准确地对猛犸象灭绝的原因进行推断，科学家们另辟蹊径，通过<strong>环境中遗留的动植物的 DNA</strong> ( eDNA ) 来分析猛犸象在气候变化下的种群动态。</p><p>这听起来就不是一个很简单的工作，事实也正是如此，研究团队为了得出具有全球性意义的结论，经历了 20 年艰苦卓绝的采样工作，在 74 处发现猛犸象遗骸的地点采集了土壤样品以用于后续分析。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" data-original="http://zkres1.myzaker.com/202111/6192f5a08e9f09361e1916e9_1024.jpg" data-height="968" 3）</p><p>通过对 eDNA 的分析，研究人员绘制了猛犸象在不同时期的种群分布图。通过分布图，<strong>我们可以清晰地看出猛犸象分布范围的减小，并且其遗传多样性也随着分布范围的缩小而降低</strong>。</p><p>研究指出，随着末次盛冰期 ( ~12000 年前 ) 的结束，冰川开始融化，猛犸象逐渐变得没有容身之所，但气候变化带来的影响还不仅限于此。随着气候变暖，湿润温暖的环境使得栖息地中的植被类群开始发生变化，<strong>猛犸象习惯的草原变成了森林和湿地，这一剧变使得猛犸象面临食物短缺的窘境，最终族群走向了灭亡</strong>。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div "，突然在某天离开了家门，就再也没有回来。</p><p><strong>死亡并不是安宁 -- 被惊扰的猛犸象</strong></p><p>在前文中，我们聊到猛犸象遗骸的稀缺限制了相关的研究。大家都知道，有需求往往就会滋生新的供需关系，事实上也确实如此。</p><p>近些年来，一股新的 " 淘金热 " 席卷了俄罗斯北部靠近北极的区域，各方对猛犸象遗骸的需求在俄罗斯催生出了一个全新的职业 --"<strong>猛犸象猎人</strong>"，猎人们在河流里，在冻土中寻找那些可以让他们一夜暴富的 " 白金 "-- 猛犸象的象牙。</p><p></p><div class="img_box" id="id_imagebox_7" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0
