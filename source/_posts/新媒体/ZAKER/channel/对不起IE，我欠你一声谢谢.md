浏览器都是我们第一个接触到的上网软件，以至于小雷每次看到那个蓝色的图标都会有种亲切感。</p><p>不过，<strong>IE 浏览器受限于前期的代码底层设计问题，在运行速度、稳定性，功能多样性等方面都有着很大缺陷，</strong>对比后来推出的 Firefox、Chrome、猎豹、遨游等各种浏览器可以说是毫无优势。</p><p></p><div class="img_box" 浏览器的唯一目的就是下载一个新的浏览器。然后就只有在打开某些第三方浏览器不兼容的网站时，小雷才会想起 IE 浏览器的存在，不知道有多少人是和小雷一样的？<p></p><p>对于大多数人来说，用过 Chrome 等浏览器后，都会对简陋的 IE 浏览器嗤之以鼻，认为它一点都不好用。但是，大家真的了解 IE 浏览器吗？<strong>许多人估计都曾经对此有过不解，为什么微软对 IE 浏览器如此不上心？</strong>以至于被 Chrome 完全碾压。</p><p>实际上，IE 浏览器在最后的版本更新中，还是加入了许多新功能，而且在速度、稳定性等方面都有明显改进。比如 IE 浏览器的最后一个版本—— IE 11，其实就已经加入了诸如扩展程序等功能，在运行速度等方面甚至能够与 Chrome 抗衡。</p><p></p><div class="img_box" 推出得实在是太晚，这个与 Windows 8 系统一起推出的浏览器面对已经习惯了 Chrome、Firefox 等浏览器的网友们，几乎没有掀起一点点的波澜，就被人们习惯性的扫入系统角落地垃圾堆中。<p></p><p>在知乎上曾经有人用 IE 11 和 Chrome 等浏览器做过测试，发现在大多数情况下，IE 11 都能够提供比 Chrome 更流畅的使用体验。但是网友的习惯和误解已经是根深蒂固的，对于 IE 浏览器卡顿、崩溃的印象更是几乎成为常识，以至于没有人愿意去了解一下 IE 11。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy 7、8 还是后来的 IE 9、10，确实都远不如 Chrome。当然，后续微软也知道 IE 的名声是救不回来了，所以很干脆的停止了 IE 11 地更新，转而推出 Edge，那就是另一个故事了。<p></p><p><strong>最早的 Windows 浏览器并不是 IE</strong></p><p>很多人可能觉得 IE 浏览器是 Windows 上第一个诞生的浏览器，实际上并不是，第一个想到制作浏览器方便普通用户上网的其实是 Netscape，又称网景公司，80 后的网友估计对这家公司还有一些印象。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy 浏览器定义了许多关于浏览器的标准和技术，其中一些甚至沿用至今，比如经常用于追踪用户使用习惯的 cookies、开发用的 JavaScript 语言等，<strong>直到今天我们所使用的浏览器中都依然有着 Netscape 浏览器的影子。</strong><p></p><p>最初微软对于浏览器并不在意，甚至乐见其成，因为不管怎么说都是应用在 Windows 系统的一个软件，能够为 Windows 系统带来更多的用户。但是，Netscape 在强势崛起后开始研发一个让用户可以通过浏览器页面进行操作的网络应用系统，有点类似于如今的各种网页在线程序。</p><p>Netscape 的举动引起了微软的担忧，彼时的 Windows 系统的市场地位还没有彻底巩固，所以微软一直在致力于掐灭所有可能威胁到自己的对手。为了降低用户对 Netscape 的依赖，微软开始与另一家公司合作，并于 1995 年推出了 IE 浏览器，随后两者就展开了一场持续长达 3 年的战争。</p><p></p><div class="img_box" id="id_imagebox_7" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_7" 系统了如指掌的情况下依然差点不敌 Netscape，直到使出杀手锏捆绑式免费使用，才最终击败了 Netscape 成为浏览器市场的霸主。<p></p><p>在曾经最大的竞争对手 Netscape 退出浏览器市场后，IE 浏览器的市场占有率在 2005 年一度达到顶峰，占据了 90% 以上的市场，也就是说<strong>整个互联网有 90% 以上的人通过 IE 浏览器进入网络世界，当时的 IE 浏览器算得上是无可置疑的霸主。</strong></p><p>不过，Netscape 虽然被收购，但是在收购完成前 Netscape 的创始人做出了一个决定，彻底公开 Netscape 浏览器的源代码，并且出资成立了一个非营利性组织 Mozilla。这个决定直接让全球浏览器市场进入了一个新的战场，以 Mozilla 为基础孵化出了著名的 Firefox 等浏览器算是继承了 Netscape 的遗志，继续着与 IE 的战斗。</p><p></p><div class="img_box" id="id_imagebox_8" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_8" 80%，数年后更是完成了前辈的遗志夺取了市场占有率第一的宝座。不过，Firefox 很快也遇到了自己的一生之敌—— Chrome，后来的故事大家估计都知道了，Chrome 现在的市场占有率达到 60% 以上，几乎没有能够与之抗衡的存在。<p></p><p></p><div class="img_box" id="id_imagebox_9" onclick><div 内核，成为了 Chrome 的 " 亲兄弟 "。<p></p><p><strong>IE 浏览器要没了，会对我们造成影响吗？</strong></p><p>看到 IE 浏览器即将被删除的消息后，不少网友除了在感叹 " 爷青结 " 外，也发出了灵魂提问：考公、考试、银行等网站咋办？想必大家都记得，每当用 Chrome 登录这些网站时总会遇到当前浏览器不兼容的提示，必须用 IE 处理器才能正常使用，如果 IE 浏览器没了，怎么办？</p><p></p><div class="img_box" id="id_imagebox_10" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_10" 浏览器中已经内置了 IE 浏览器的兼容模式，用户只需要打开该功能就可以正常浏览仅兼容 IE 浏览器的网站。</strong><p></p><p>而且，停用的仅仅是个人用户版本的 IE 浏览器，面向企业、政府的 IE 浏览器将仍然得到微软的支持。但是，微软关闭 IE 浏览器的决心肯定是不会变的，即使继续提供支持，也不会长久，也算是在倒逼企业和政府对自己的网站进行升级，对于大家来说反而是一件好事。</p><p><strong>结语</strong></p><p>对于许多网友来说，IE 浏览器所代表的过去，即使有着令人头疼的各种问题，但是当我们第一次通过 IE 浏览器看到广阔的世界时，那种激动、开心的心情也是如此地让人印象深刻。</p><p><strong>别了，IE。</strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
