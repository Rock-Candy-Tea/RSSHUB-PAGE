<p>对矩声而言，Element 系列无疑是这老牌劲旅门下犹如里程碑的产品系列。几年前 Element X 初代发布时，凭借全新的 ID 设计语言和完善易用的专属软件让人耳目一新，音质方面也延续了矩声一直在海内外市场的良好口碑。直至近期换代机型 Element X2 正式发布前，Element X 初代都一直是三耳君常给烧友们推荐的一体机型号，可见初代产品的产品力在几年的销售周期里面并未见落后。然而，正因珠玉在前，至于三耳君也不禁思疑产品形式没有太大突破的 Element X2 是否还能带给玩家足够的惊喜。</p><p><strong>功能依旧全面，且更强大又适应潮流</strong></p><p></p><div class="img_box" id="id_imagebox_0" 遥控的用户，或者设备初打开未联网的状态，大多数基本操作均可通过这个交互区在几步以内完成，上手几乎无难度。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 组并联组成，官方表示有着比起前作具有更高的驱动电流和输出功率，同时保证了更低的失真与噪声。<p></p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" 的高品质音频信号，并能在输出范围内均保持超低的噪音和失真，可灵活对应有源音响和独立后级功放的使用。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5"
