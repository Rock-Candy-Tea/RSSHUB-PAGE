<p>AMD 将于今年下半年推出 Radeon RX 7000 系显卡，在规格上有很大的提升，此外在功耗上也有比较大的提升，有消息称 AMD 的 Radeon RX 7900 作为旗舰显卡，将会采用 MCM 架构，从而让流处理器的数量再次提升。目前国外著名的爆料人士公布了 Radeon RX 7000 系显卡的发布时间，与大家想的有所不同的是，这一次 AMD 将会直接为大家带来旗舰级的显卡也就是 Radeon RX 7900 显卡，至于次旗舰的 Radeon RX 7800 显卡，将会在 2023 年才跟大家见面。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202204/626b5151b15ec0053526d4fd_1024.jpg" data-height="531" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/626b5151b15ec0053526d4fd_1024.jpg" referrerpolicy="no-referrer"></div></div>以往 AMD 发布新一代显卡，总是会有多个型号，比如说 700、800 以及 900 一同发布，不过这一次 AMD 似乎要率先发布旗舰显卡 RX 7900，其他的都要缓缓。预计 AMD 将会在今年的第四季度推出 Radeon RX 7900 显卡，而 Radeon RX 7800 显卡则要到 2023 年的第一季度，至于之后的 Radeon RX 7700 显卡则还没有消息什么时候发布。<p></p><p>看起来有一种可能，那就是 AMD 将会在 COMPUTEX 上发布全新的 Radoen RX 7000 系列显卡，包括架构，实际游戏测试成绩以及规格，而明年的 CES 2023 可能发布 Radeon RX 7700 显卡，估计在 2023 年的第二季度出货。其中一个原因就是 Radeon RX 7000 系显卡性能提升比较给力，先给 Radeon RX 6000 系显卡清一波库存，现在有消息称 RX 7700 显卡的性能比 RX 6900 XT 还给力，这让 RX 6000 系显卡实在是有点尴尬。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
