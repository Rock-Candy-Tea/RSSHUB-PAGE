<p>IT 之家 8 月 15 日消息 创意迸发，项目开花，微软介绍称，Windows 11 的默认背景名为 " 绽放 "，意思是在桌面上 " 开花 "，代表了 Windows 用户的多样性。</p><p>微软在最新一期的 Windows Insider Program 视频中向用户介绍了关于 Win11 的一些背景故事，例如默认壁纸和 logo。</p><p>微软设计团队解释道，新的样式可与 Windows 11 新材质相得益彰。而 Windows 11 的 logo 设计要简单一些。</p><p>一位设计团队成员先是问道 :" 我们如何才能让 Windows 和微软走得更近 ?" 他表示 :" 我们想到了微软的标志，所以我们把它变成了蓝色。" 他解释称，蓝色往往是人们联想到 Windows 时最先感受到的颜色。</p><p>IT 之家了解到，虽然 Win11 目前变化不大，但将带来大面积的图形样式升级。这说明广大用户多年来一直在反馈的 Windows 外观问题终于得到了微软重视。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202107/6100b2598e9f097e157fa764_1024.jpg" data-height="512" data-width="820" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/6100b2598e9f097e157fa764_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
