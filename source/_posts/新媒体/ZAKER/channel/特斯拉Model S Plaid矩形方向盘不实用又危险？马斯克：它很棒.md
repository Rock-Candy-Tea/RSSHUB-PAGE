<p>特斯拉最新发布的 Model S Plaid 上采用了全新的轭式方向盘，这表明特斯拉在未来车型上押注于这种矩形方向盘，让人不免担忧未来的特斯拉赛博卡车（Cybertruck）也可能采用类似的方向盘。</p><p>问题在于，这种方向盘在操控上仍存在一些问题。但特斯拉首席执行官埃隆 · 马斯克 ( Elon Musk ) 表示，能够解决问题的渐进式转向系统还需要几年时间。</p><p>当特斯拉在最新款的 Model S 上推出轭式方向盘的时，引发了不少争议。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202106/60cbff4a8e9f092c805c5a85_1024.jpg" data-height="326" data-width="660" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202106/60cbff4a8e9f092c805c5a85_1024.jpg" referrerpolicy="no-referrer"></div></div>图示：Model S Plaid 上所采用的矩形轭式方向盘<p></p><p>特斯拉一直在开发线控转向系统。业内一度认为，如果线控转向系统不成熟，特斯拉就不会冒险将备受争议的轭式方向盘推向市场。</p><p>然而，当特斯拉上周开始交付全新的 Model S Plaid 时，车内所搭载的方向盘已经换成了矩形。事实上，当车辆在较低速度下行驶时，轭式方向盘很不好用。</p><p>有人在推文中抱怨，" 作为一个在结冰路面上开了一辈子车的人，我清楚在车轮打滑的情况下，如何快速获得对车辆的控制才是最重要的问题。"</p><p>但在某些驾驶场景中，轭式方向盘不仅不实用，甚至还有潜在的危险。</p><p>例如，如果车辆在结冰的路面上打滑，让驾驶员失去控制，就很难通过快速打方向盘来恢复对汽车的控制。</p><p>但马斯克在评论这款新方向盘时表示，" 它很棒 "，并明确至少几年内都不会出现渐进式转向系统：" 我用轭式方向盘已经开了有一段时间，在我看来这很棒。渐进式转向需要复杂的传动装置或没有直接机械连接的线控系统驱动。我们的目标是在几年后实现。"</p><p>这样看来特斯拉的新转向系统在几年内都不会上市。在此之前，用户只能期待特斯拉对转向系统做出更多的增量改进。</p><p>此前也有搭载轮式方向盘的 Model S Plaid 原型车。但马斯克并没有回答关于特斯拉是否会将轮式方向盘作为选配装置的问题。</p><p>轭式方向盘不免让一些喜爱特斯拉的人感到担忧，因为这不仅仅是关于全新 Model S 的问题。</p><p>如果马斯克真的特别喜欢轭式方向盘，加之特斯拉也在赛博卡车和跑车等车型的原型车上搭载了轭式方向盘，那么这款方向盘很可能就是特斯拉汽车方向盘的未来。</p><p>但目前轭式方向盘在 Model S Plaid 上的操控问题也会出现在赛博卡车和特斯拉电动跑车上。</p><p>可以预见，赛博卡车在推拽或者泊车时，这种问题会更严重。</p><p>虽然轭式方向盘在汽车高速行驶的确更好，但低速行驶时暴露出的缺点依旧不可忽略。无论如何，特斯拉决定在没有渐进式转向解决方案的情况下直接采用轭式方向盘让人深感意外。（辰辰）</p><p> 本文来源：网易科技报道 责任编辑： 杨胜忠 _NB24315</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
