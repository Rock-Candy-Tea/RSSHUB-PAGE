<p>品玩 7 月 7 日讯，微软今天推出了一个紧急 Windows 修复补丁，以修复存在于 Windows Print Spooler 服务中的一个关键缺陷。该漏洞被称之为 "PrintNightmare"，在安全研究人员无意中公布了概念验证（PoC）的利用代码后，于上周被曝光。</p><p>微软已经发布了带外安全更新以解决该漏洞，并将其评为关键漏洞，因为攻击者可以在受影响机器上以系统级权限远程执行代码。由于 Windows Print Spooler 服务在 Windows 上默认运行，微软不得不为 Windows Server 2019、Windows Server 2012 R2、Windows Server 2008、Windows 8.1、Windows RT 8.1 以及 Windows 10 的各种支持版本发布补丁。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
