<p>今天（11 月 17 日），微软宣布全新的 Media Player 播放器将向处在 Dev 渠道的 Win 11 用户推出，此次更新后的 Media Player 将会全面取代现有的 Groove 音乐。</p><p>Media Player 是微软在 2009 年推出的多媒体播放器，用户通过 Media Player 能够浏览包括音乐和视频在内的多内体内容，是微软步入新世纪后的元老级应用之一。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" Player<p></p><p>不过虽然该软件有着足够老的资历，但实际上它的上一次更新已经是能够追溯至 Win 7 时代的 Media Player 12 了，在这之后虽然 Windows 系统仍会预装该应用，但是已经几乎完全边缘化。</p><p>而在此次更新中，微软对 Media Player 的 UI 进行了堪称 " 脱胎换骨 " 一般的改动。</p><p></p><div 相同的设计语言，在 UI 元素上以简约化，现代化和质感化为主要设计思路，使得新 Media Player 的观感更加符合当下的用户审美，但同时在一些细节上又能够看出对老 Media Player 的致敬。</p><p>而在功能上，微软表示将会把新 Media Player 打造成一个功能齐全的音乐库，并且将自动迁移用户在原本 Groove 音乐中的全部内容。</p><p>此外，Media Player 原本的视频播放能力也获得了保留，用户可以使用新 Media Player 浏览、管理和观看本地视频，新 Media Player 将自动显示用户电脑上的所有音乐和视频内容。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div Player 与旧版的 Media Player 并不冲突，这两者依旧可以共存，但 Win 10 的 Groove 音乐将会被新 Media Player 取代，成为历史。</p><p>不过需要额外注意的是，作为一款刚刚在 Dev 渠道推送的应用，新 Media Player 有着诸如无法从网络波返回，无法应用主题偏好的 UI 元素等多种 Bug。</p><p></p><div
