的日子：以淘宝和抖音链接为例，微信仅仅开放了个人会话链接，而且仅能打开浏览器页面，不能直接跳转；这样的用户体验很差，聊胜于无。</p><p>答案很简单：<strong>因为 9 月 17 日从来就不是 " 平台互联互通 " 的大限。</strong>在 9 月 13 日的官方新闻发布会上，工信部从来没有给出准确的 " 互联互通时间表 "，只是强调要 " 分阶段、分步骤 " 解除屏蔽链接。照此精神，腾讯做出的表态也是 " 分阶段、分步骤 "。所谓 9 月 17 日是最后日期的说法，仅仅是坊间传闻，并无约束力。</p><p>如果我们仔细阅读工信部的官方文件和讲话，就会得知：工信部现在要求的，确实只是 " 开放链接跳转 " 而已。所谓链接跳转，就是能够直接点开链接，并在微信内置的浏览器当中以 HTML 页面的形式展现，仅此而已。如果想在打开淘宝或抖音链接之后跳转 APP，或者以微信身份自动登录，那就需要开放 API，而工信部从未做出这样的要求。当然，今后会不会做出这样的要求，不得而知。</p><p>简而言之，当一个链接被转发到微信时，它可能得到四种待遇之一：</p><p><strong>打开小程序</strong>，小程序拥有最丰富的 API、最强大的功能，其打开记录会保存在 " 最近使用的小程序 " 当中。例如，你在《王者荣耀》里勇夺 MVP，把战绩炫耀给微信好友，他打开的就是一个小程序。</p><p><strong>打开一个开放 API 的 H5 页面，</strong>这个页面往往具备比较丰富的功能，例如使用微信账户注册、点赞、加购物车、唤起 APP 等。京东、拼多多、快手、B 站等绝大部分与腾讯关系较好的第三方，其链接均享受这种待遇。</p><p><strong>打开一个不开放 API 的普通页面</strong>，该页面的功能非常单薄，用户往往只能浏览；要唤起 APP 的话，还要复制链接再跳转到浏览器或 App Store。9 月 17 日以后，淘宝和抖音的链接就享受这种待遇。</p><p><strong>直接被屏蔽，无法打开，</strong>或被提示 " 该链接已停用 "。凡是带有诱导式分享，例如引诱用户下载 APP、转发参与抽奖等活动的链接，均有可能被屏蔽，其中不仅仅包括淘宝和抖音。</p><p>显然，如果淘宝和抖音的链接只能享受上面第三种待遇的话，那真是聊胜于无。尤其对于淘宝而言，这种状况非常不适合促成交易，还不如当初的 " 淘口令 "；对抖音会稍好一点，但把用户沉淀到抖音 APP 难如登天。这两家绝不会就此罢休，会努力追求更好的待遇。腾讯这边肯定也在紧张地评估，做出各种应对方案。</p><p>对于任何第三方平台而言，微信都是一个既诱人又可畏的流量来源。所谓诱人，是因为它的用户最多、用户黏性最高、承载了高频的强社交，从而特别适合裂变式的营销；所谓可畏，是因为它可能反过来吞噬外界的流量，让第三方平台被架空，乃至丧失存在的必要性。</p><p>我们需要确认一个事实：任何商家、任何内容方，均有在微信经营 " 私域流量 " 的动力。众所周知，只有一个用户沉淀到了微信群，他才是你的 " 核心用户 "。微信社群运营是一门博大精深的学问，所以企业微信总能受到销售人员的欢迎。对于经验丰富的品牌或创作者而言，无论淘宝还是拼多多、抖音还是快手、B 站还是小红书，自己积累的流量一般会汇总到微信——所以说，微信是一切流量的尽头。</p><p>因此，<strong>第三方平台不得不陷入 " 既当又立 " 的困境</strong>：它们一边大声疾呼，要求微信开放链接、开放 API，以便微信用户沉淀到自身平台；一边又禁止自己的内容方 / 品牌方留微信号、留公众号，阻止任何用户回流到微信。在大部分直播平台，主播只要在直播间说 " 微信 " 二字，就可能被封；在大部分电商平台，商家给你发来的微信 ID 几乎肯定会被屏蔽，导致买家和卖家都学会了谐音打哑谜。</p><p>在一个绝对公平的环境下，所有的链接都会无限制地开放，那么微信的流量会被不停地导出到第三方，第三方的流量又会被不停地导回微信，由此形成一种类似自然界水循环的 " 互联网流量大循环 "。然而，这种循环，不但会被第三方平台讨厌，也会被微信讨厌，因为它太混乱、太不可控了！所有平台都会被搞得一塌糊涂，用户只会沉淀到内容方 / 产品方上面，对平台毫无忠诚度可言。相信主管部门也不会喜欢这种情况，因为它会大大提升监管执法的难度。</p><p>既然 " 无限制开放 " 对大家来说都是不可接受的，那结局肯定还是 " 有限制开放 "。准确的说，微信开放外链之后的发展路径，无非是以下三种：</p><p>微信对一切第三方大幅开放 API，允许内部登录、唤起 APP，乃至读取微信社交关系链；第三方可以对微信予取予求，却不必让出任何利益。在这种情况下，微信将沦为第三方的流量池，微信用户的体验也会大幅下降。</p><p>微信对第三方选择性开放 API，实施苛刻限制。例如，它可以要求淘宝、抖音的链接仅能以小程序方式打开，而且不得唤起 APP，从而阻断用户流出自身平台。在这种情况下，微信凌驾于一切第三方之上，把它们变成了自身生态系统的一部分。</p><p>微信与第三方各退一步，达成妥协。例如，第三方页面可以唤起 APP，但要有节制；微信用户可以在第三方链接内登录，但不得被读取社交关系链；通过微信分享所产生的电商交易，必须使用微信支付，等等。这样，第三方获得了流量，微信也获得了好处。</p><p>事实上，只有第三条路径是可行的，因为腾讯不会接受第一条路径，而其他平台不会接受第二条路径。问题在于，第三条路径完全建立在妥协之上，每个平台都有自己的底线和核心诉求，如何保证能达成妥协呢？</p><p>以外界最关注的淘宝和抖音为例，它们在微信平台的核心诉求大不相同——在现阶段，淘宝更重视通过社交电商激活淘特、增加 GMV，而抖音更重视通过社交传播促进用户增长。假设它们都愿意与微信展开平等的谈判，它们能接受的条件可能很不一样，谈判进程可能也很不一样。</p><p>无论如何，有一件事情是肯定的：<strong>腾讯的竞争对手不可能在微信平台享受与腾讯的战略合作伙伴相似的待遇。</strong>要知道，京东可是拥有至关重要的微信一级入口，拼多多、美团、贝壳拥有微信支付二级入口，快手内容则可以直接被分享到微信看一看信息流。而且，就算对这些合作伙伴，微信有一些底线也是不容让步的。例如，任何商品链接均不能直接转发到朋友圈，哪怕是京东、拼多多或美团。既然如此，要在朋友圈看到淘宝或抖音商品链接的带 API 分享，大概是不可能的事情。</p><p>综上所述，如果淘宝和抖音链接在微信内部只能以不带 API 的普通页面形式打开，意义就很有限了。即便是这种非常初级的打开，目前也仅限于微信个人会话，而不适用于微信群。接下来一段时间，有几个问题特别重要，将决定平台互联互通的走向：</p><p>工信部等主管部门是否会对微信平台的 API 提出规定或指导性意见？</p><p>第三方平台禁止产品方 / 内容方公布微信号、拉微信群的行为，是否也在平台互联互通的整顿范围之内？</p><p>归根结底，我们能不能看到腾讯、阿里、字节跳动之间，就互联互通问题进行一系列开诚布公的平等谈判？</p><p>我相信，无论答案是什么，都不会很快揭晓。这不是平台互联互通的结束，而只是一个开始罢了。</p><div id="recommend_bottom"></div><div
