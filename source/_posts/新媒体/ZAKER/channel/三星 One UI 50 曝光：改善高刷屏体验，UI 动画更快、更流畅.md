<p>IT 之家 6 月 10 日消息，One UI 5.0 是三星定制 Android 系统的下一个主要版本。该公司将在今年晚些时候与 Android 13 一起发布 One UI 5.0。</p><p>据 SamMobile 获得的信息，One UI 5.0 的主要改进目标之一，就是 " 加速用户界面 "。爆料称，三星正在努力提高 One UI 5.0 的 UI 导航速度，还将优化动画，使其更流畅、更快。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202204/624f706eb15ec020b36ad302_1024.jpg" data-height="400" data-width="778" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/624f706eb15ec020b36ad302_1024.jpg" referrerpolicy="no-referrer"></div></div>这种在动画过渡上的小改动将对用户体验产生重大影响，尤其配合三星高端设备上搭载的高刷屏，我们可以期待一下三星的成果。<p></p><p>IT 之家此前报道，SamMobile 获得的消息称，三星准备从 7 月份开始公开测试 Android 13 和 One UI 5.0，会给 Beta 测试人员和公司充足的时间来完善新固件，以便在 2022 年底或 2023 年初正式发布。</p><p>一旦准备就绪，数十种 Galaxy 智能手机和平板电脑就有资格获得即将推出的 Android 13 和 One UI 5.0 公测更新。像往常一样，更新、更高端的设备将优先更新，因此首批搭载 Android 13 的 Galaxy 手机可能是 Galaxy S22 和 / 或 Galaxy Z Fold 4 和 Flip 4 等。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
