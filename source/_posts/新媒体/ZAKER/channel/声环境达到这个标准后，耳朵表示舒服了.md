<p>对于城市里的空气污染、光污染，我们往往很敏感，但你有没有注意到，在照顾眼睛、鼻子的同时，我们常常忽略耳朵的感受？</p><p>若是一直处于嘈杂的道路周边或人声鼎沸的环境中，时间长了我们就会感觉身心疲惫；若是居住环境也有噪音干扰的话，多数人会陷入焦虑且情绪不稳定的状态中。</p><p></p><div class="img_box" id="id_imagebox_0" 图库<p></p><p>那么，在城市的日常生活环境中，多大的声音会让我们感觉舒适？人类耳朵能听到的最小声音强度是多少，能接受的最大声音强度是多少？用手遮住耳朵声音强度会衰减掉多少呢？</p><p>带着这些问题，我们一起来聊聊听觉与声环境。</p><p><strong>Part.1</strong></p><p><strong>面对视力表心慌慌，那试试听阈测试</strong></p><p>首先我们从听觉感知说起，人耳听觉能感受到的声波频率范围是 20Hz~20000Hz，频率低于 20Hz 的声波为次声，频率高于 20000Hz 的声波为超声。</p><p>一些动物能听到次声和超声，<strong>比如大象能够发出并听到次声，而蝙蝠、海豚能够发出并听到超声。</strong></p><p>声音强度很大时，次声和超声可通过非听觉途径作用于躯体。</p><p>由于人体器官较低的固有频率在次声范围内，当次声作用于人体会产生共振，引起生理反应和病理变化，<strong>高强度的次声</strong>可损伤人体内脏，所以军事上有研制次声武器。</p><p><strong>高强度的超声</strong>会损害人体组织，主要是因为超声的热效应和空化效应等，这些效应的物理学基础是粒子的机械振动。诊断超声所用的剂量远低于治疗超声，所以不会对人体产生损害。</p><p>人耳听觉能感知到的最小声强（即声波平均能流密度的大小，用于描述声音大小）称为<strong>听阈</strong>。不过，不要误以为听阈是一个固定值，主观判断听到和听不到之间没有一个很明显的界限，在声音强度很微弱的时候，被试者其实是时而能听到，时而听不到。</p><p></p><div Level，HL）作为听阈。将听力正常青年人所能听到的平均最小声压级定为听力零级，即 0dB HL。</strong></p><p>1997 年，世界卫生组织（WHO）根据 500Hz、1000Hz、2000Hz 和 4000Hz 气导平均阈值，将听力损失分级标准定为 4 个等级，具体分级见表 1。</p><p></p><div class="img_box" id="id_imagebox_2" HL。</p><p><strong>轻度听力损失者在安静环境中交谈没有问题，但是在噪声环境中可能听不清谈话声</strong>。并且新的分级增加了单侧听力损失的标准，具体分级见表 2。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img 平均听阈）<p></p><p>（图片来源：参考文献 4）</p><p><strong>Part.2</strong></p><p><strong>警惕环境里的噪音</strong></p><p>从听力分级标准中可以看出，正常听力的人能听到小于 20dB HL 声强的声音，但是当声强达到 120dB HL 时，我们将会浑身不适，难以忍受。</p><p>我们日常所处的声环境中，不同的声源所产生的声音强度有所不同：从最小的耳语到高声交谈，声强大约在 20~80dB HL 之间；在工厂的生产车间里，噪声级经常高于 85dB ( A ) 以上；割草机发出的声音强度能达到 100dB HL；赛车、飞机和鞭炮的声音强度能达到 120dB HL 以上。</p><p><strong>在使用声级计进行环境声强级测试时一般使用 A 计权，单位用 dB ( A ) 表示，此处单位标为 dB HL 的环境声强是将 dB ( A ) 转换成了听力级。</strong></p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div 以上的声环境中，就可能导致听阈出现暂时性阈值的改变，所能听到的最小声强将增大</strong>。若是不能及时转移到安静的环境中，听力则可能出现永久性听阈改变，即出现<strong>噪声性听力损失</strong>。</p><p>噪声环境中，<strong>声音强度越大或所处时间越长，引起的暂时性阈移越大</strong>。</p><p>因此，一些行业必须要制定相关的噪声标准，比如建筑施工行业 2016 年岗位职业卫生岗位操作规程，就对噪声接触制定了卫生限值，噪声卫生限值最高不应超过 115dB ( A ) 。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " 岗位职业卫生岗位操作规程）</p><p><strong>Part.3</strong></p><p><strong>保护听力，从捂耳朵做起</strong></p><p>为了预防噪声对听力的损害，经常接触噪声的人员应注意佩戴护耳器，比如耳塞、耳罩、防噪声头盔等。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_6" data-original="http://zkres2.myzaker.com/202111/619777868e9f093e4712c4a1_1024.jpg" data-height="579" data-width="868" 种防噪耳塞的声衰减值进行了测试。</p><p>结果显示，在 125Hz~8000Hz 范围内，防噪耳塞对声音的衰减值随着频率的提高而增大；在 125Hz、250Hz、500Hz 频率处的声衰减值在 20dB 以下；在 1kHz、2kHz、3kHz、4kHz、6kHz、8kHz 频率处的声衰减值在 20dB~50dB 之间。</p><p>可见，防噪耳塞对声音的衰减效果比较有限。<strong>不过日常生活中如果能使得到达耳内的声强降低到 85dB 以下，佩戴护耳器就能起到防护的作用。</strong></p><p>平时我们最常用的抵挡噪音的方法，莫过于用手捂耳朵了。那么，手掌能够衰减掉多少声音呢？</p><p>有研究表明，<strong>在 125Hz~8000Hz 范围内的噪音环境中，当被试者向内按压耳屏致外耳道完全封闭时，声衰减范围在 15~45dB 之间。</strong></p><p>可见即使我们完全堵住外耳道，也不能将外界的声音完全衰减掉，所以我们平时遮住耳朵通常也可以听到声音，只不过音量小了点罢了。但是当我们突然暴露在高声强环境下的时候，用手堵住耳朵确实能起到保护听力的作用。</p><p><strong>Part.4</strong></p><p><strong></strong></p><p><strong>小点声，拒绝声环境污染</strong></p><p>如今，我们越来越认识到声环境对于我们身心健康的重要性，那不同的声环境声音强度要控制在多少才合适呢？</p><p><strong>我国对不同种类的声环境功能区制定了声环境质量标准，不同声功能区的环境噪声限值见表 4。</strong></p><p><strong></strong></p><p>声功能区类别昼间限值 /dB ( A ) 夜间限值 /dB ( A ) 0 类：康复疗养区等特别需要安静的区域。50401 类：以居民住宅、医疗卫生、文化教育、科研设计、行政办公为主要功能，需要保持安静的区域。55452 类：以商业金融、集市贸易为主要功能，或者居住、商业、工业混杂，需要维护住宅安静的区域。60503 类：以工业生产、仓储物流为主要功能，需要防止工业噪声对周围环境产生严重影响的区域。65554a 类：高速公路、一级公路、二级公路、城市快速路、城市主干路、城市次干路、城市轨道交通（地面段）、内河航道两侧区域。70554b 类：铁路干线两侧区域。7060</p><p>表 4 ，夜间噪声限值在 45dB ( A ) 。</p><p><strong>实际上，现实中的不同生活场景都充斥着背景噪声。</strong></p><p>有研究显示，在安静环境中背景噪声平均在 40dB ( A ) 左右，在有背景音乐、户外、广播或电视环境中，背景噪声平均在 50dB ( A ) 左右，至于商店、火车站候车厅及站台、厨房、汽车内部、多人交谈环境中，背景噪声可以达到 60dB ( A ) 以上。</p><p>由此可见，在很多环境中背景噪声可以达到 60dB ( A ) 以上，声音质量达不到舒适的要求。</p><p>为了让大家拥有适宜身心健康的各种声环境，城市建设者做了不少努力。比如在现代建筑设计方面注重采用有隔声和吸声功能的材料，在道路两旁设置绿化带和隔离带等。</p><p>声环境的维护需要各行各业工作者的共同努力。对于我们个人来说，不在公共场合大声喧哗，开车时遵守禁鸣规定等，对于声环境的维护都非常重要，让我们一起来共建更加舒适的声环境吧！</p><p><strong></strong></p><div
