<p>NVIDIA GTX 1650 重新放货？这文章是不是穿越了？没有，更不是愚人节新闻，这是 NVIDIA 官方的决定。</p><p><strong>01</strong></p><p><strong>NVIDIA GTX 1650 意外回归</strong></p><p>过去一年显卡市场的混乱，可以说是历史上从未有过的。新冠疫情的冲击，数字货币挖矿的扰乱，再加上厂商本身的策略变化，让显卡行业乱成了一锅粥，各种缺货买不到。RTX 30 系列至今仍停留在中高端市场，因此在中低端领域，上一代的 GTX 16 系列仍有很大的生存空间，只是供货同样不理想。</p><p>比如说入门级的 GTX 1650，2021 年开年至今各家厂商都供货严重不足，主要原因是 NVIDIA 把芯片基本都分给了笔记本市场，DIY 这边分配极少，导致极度短缺。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202104/60659183b15ec011505cb234_1024.jpg" data-height="300" data-width="640" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202104/60659183b15ec011505cb234_1024.jpg" referrerpolicy="no-referrer"></div></div>现在，NVIDIA 终于要调整了。<p></p><p>据最新消息，NVIDIA 预计会在 4 月中下旬向 AIC 厂商补充供应大量的 GTX 1650 芯片，大概在 5 月左右，各家厂商的出货量会开始逐步增加。</p><p>这次调整主要针对中国大陆市场，只希望不要也被矿工们给截胡了。</p><p>GTX 1650 是近两年最低端的 NVIDIA 显卡，基于 12nm 工艺、图灵架构的 TU117 核心，896 个流处理器，搭配 128-bit 4GB GDDR5/GDDR6 显存，整卡功耗 75W。</p><p>它还有个升级版 GTX 1650 Super，1280 个流处理器，标配 4GB GDDR6 显存，整卡功耗 100W。</p><p>别看如此低端，在去年底 RTX 20 系列纷纷停产断供、RTX 30 系列还没跟上的时候，GTX 1650 一度都成了销售主力，线上线下需求猛增，结果导致严重缺货、价格飙升。</p><p>目前在电商平台，GTX 1650 在售型号和货源不多，价格 1600-2000 元左右，GTX 1650 Super 在售的还不少，但依然普遍缺货，价格 1500 元～2500 元左右。</p><p><strong>02</strong></p><p><strong>GeForce 显卡的 GPU 虚拟机直通功能</strong></p><p>说实话，花钱购买一块有年代感的显卡，心理多少不是个滋味儿，预期纠结于买卡，不如思考下如何把手里的卡潜能榨干。</p><p>英伟达 GeForce 显卡上的 GPU 直通技术已经使用了一段时间，它允许虚拟机从主机上访问 GPU。然而，过去 NVIDIA 并不支持这项技术，但这种情况已经发生了改变。NVIDIA 现在完全支持 GeForce 卡的 GPU 直通。</p><p>英伟达 GeForce GPU Passthrough 让 Linux 用户终于可以玩 Windows 游戏了。所有基于 Kepler 架构或更新的 GeForce 显卡都可以利用 GPU 直通来运行 Windows 虚拟机。如果您需要在笔记本电脑上运行虚拟机，GPU 必须基于 Maxwell 架构或更新的架构。</p><p>这确实也包括 Titan GPU。新增的 GPU 直通支持功能可以让开发者使用虚拟机在不同的操作系统中只使用一台机器测试游戏和软件。对于基于 Linux 系统的游戏玩家来说，新增的功能将允许 Linux 用户使用虚拟机玩基于 Windows 的游戏。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div 直通测试版支持。英伟达表示，启用虚拟化后，Linux 主机上的 GeForce 客户现在可以在虚拟的 Windows 操作系统上启用 GeForce GPU 直通功能。有一些 GeForce 的使用案例中，该功能是有益的，例如，想要运行 Linux 主机并能够启动 Windows 虚拟机玩游戏的 GeForce 客户，游戏开发者希望在一台机器上同时测试 Windows 和 Linux 的代码。<p></p><p>GeForce GPU passthrough 支持 1 个虚拟机。GeForce 不支持 SR-IOV。如果你想让多个虚拟机直接访问一个 GPU，或者希望 GPU 能够为多个虚拟机分配虚拟功能，你需要使用 NVIDIA Tesla、Quadro 或 RTX 企业级 GPU。GeForce R465 或更高版本的驱动程序支持 GeForce 虚拟化（beta 版），Windows 10 R465 驱动程序支持的所有 GeForce/TITAN GPU 均可启用该功能。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" GPU。对于那些需要在一个 GPU 上运行多个虚拟机的用户来说，GeForce 显卡并不能满足他们的需求。为了在一个 GPU 上运行多个虚拟机或能够为多个虚拟机分配虚拟功能，需要使用 Tesla 或 Quadro 显卡。目前 GeForce 显卡的 GPU 直通功能在 R465 或更高版本的驱动下，还处于测试版支持中。<p></p><p>（编辑：张毅）</p><div
