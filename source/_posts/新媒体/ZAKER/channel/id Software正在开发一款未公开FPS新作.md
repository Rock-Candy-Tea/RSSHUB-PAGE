<p>根据 ZeniMax 的招聘信息，《毁灭战士》开发商 id Software 正在开发一款未公布的 FPS 游戏。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202204/625be4eb8e9f09587a70f28f_1024.jpg" data-height="540" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/625be4eb8e9f09587a70f28f_1024.jpg" referrerpolicy="no-referrer"></div></div>最新的一批招聘岗位都明确指出了所属的项目是 id Software 的未公布的游戏项目。比如高级玩法程序员、高级 UI 程序员。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202204/625be4eb8e9f09587a70f290_1024.jpg" data-height="667" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/625be4eb8e9f09587a70f290_1024.jpg" referrerpolicy="no-referrer"></div></div>其中，高级玩法程序员岗位要求 " 在游戏行业工作 5 年以上或在第一人称射击游戏方面有明显的工作经验 "。此外，还最好拥有 " 任何版本的 id Tech 引擎经验 "。id Tech 是 id Software 自有的游戏引擎，被用于其旗下大部分游戏。<p></p><p>id Software 曾开发过许多知名 FPS 作品，诸如《毁灭战士》和《雷神之锤》。这款未公布的 FPS 新作有可能是某个知名系列的续作。外媒根据使用 id Tech 引擎这一点推断，应该不大可能是《狂怒》系列续作，因为《狂怒 2》使用了共同开发商 Avalanche Studios 的 APEX 引擎，下一作转为使用 id Tech 的可能性略低。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202204/625be4eb8e9f09587a70f291_1024.jpg" data-height="674" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202204/625be4eb8e9f09587a70f291_1024.jpg" referrerpolicy="no-referrer"></div></div>岗位职责中提到了 " 合作实现新的游戏功能：玩家和角色的行为、战斗和能力机制 "，因此游戏中可能会包含增加角色能力的机制。这在《雷神之锤》中是存在，考虑到更早之前的传闻，外媒推测它有可能是《雷神之锤》重启作。不过，为《毁灭战士》系列引入这个机制也是有可能的。<p></p><p>当然，这甚至可能是 id Software 的一个全新 IP，因为招聘信息中的细节非常模糊，所以没法进一步推测。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
