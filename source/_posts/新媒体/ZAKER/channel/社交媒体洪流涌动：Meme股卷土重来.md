股卷土重来</strong></p><p>两个星期飙升四倍多，三天时间暴跌七成。大海肆虐般的 Meme 股现象正在美国股市卷土重来，而这一次股价坐上过山车的则是美国家居用品连锁商场 Bed Bath&Beyond ( 以下简称 BBBY ) 。</p><p>在半个月时间里，BBBY 从 4.5 美元的最低点急剧飙升到 28 美元的高峰，飙升了四倍半；然后又在仅仅三个交易日内自由落体坠到了 8.5 美元的价位。这种剧烈的波动性正是 Meme 股的最典型特征。</p><p>带动 BBBY 股价飙升的主要推力是来自散户投资者的持续买盘，尤其是来自 Reddit 论坛上的 wallstreetbets ( 以下简称为 WSB ) 小组，这里有上千万对股市投资感兴趣的用户。BBBY 是 WSB 小组这个月最热门的炒股话题。</p><p>据统计，过去三个星期，WSB 上有 12.3 万次提到了 BBBY，是 7 月份的 18.5 倍。8 月初，一位用户名 TheDude0007 的用户在 WSB 宣布自己购入了 4.5 万美元的 BBBY 股票，吸引了大批散户跟进买入，触发了这只股票的疯狂上涨。</p><p>同期成为 Meme 股的还有美国连锁影院 AMC，只是没有 BBBY 这么走势疯狂。从 7 月底开始，这支股票在两个多星期内上涨了 70%，远远超过同期标准普尔指数仅仅 6% 的涨幅。而自 2020 年疫情爆发之后，虽然 AMC 业务因为疫情遭受重创，股价却飙升了 150%。</p><p>不过，真正的妖股则是香港上市的尚乘数科 ( AMTD Digital ) 。这家名不见经传的企业创办仅有三年，雇员只有几十人，上市一个月却股价飙升 200 多倍，甚至市值一度接近 4000 亿美元，压倒了诸多互联网巨头企业。但狂野上涨之后，则是极速暴跌。半个月后，这家妖股的市值已经缩水到了 310 亿美元。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202208/630826358e9f093b273e6306_1024.jpg" data-height="810" data-width="1080" <p></p><p><strong></strong><strong>带头大哥套现走人</strong></p><p>而导致 BBBY 股价大跌的重要因素则是一位重要投资者的套现走人。就在 BBBY 股价上周达到最高点时，突然传出了亿万富翁投资者、宠物用户网站 Chewy 创始人瑞恩 · 科恩 ( Ryan Cohen ) 突然抛光了所有 BBBY 持股和看涨期权的消息。" 带头大哥 " 套现离场引发了小散们的恐慌，BBBY 当天就掉头向下暴跌 40%，随后持续回落到 10 美元的价位。</p><p>被称为 "Meme King" 的科恩此前买入 700 万股 BBBY 的平均价格是 15.32 美元，而上周卖出的最高价格则是 29.22 美元。据 CNBC 估计，这只股票在 8 月前两个星期的疯狂飙升，给科恩带来了 5900 万美元的回报。这还没有算入他从看涨期权中获得的收益。</p><p>科恩完成此次套现只用了 5 个月时间。今年 3 月，科恩宣布自己收购了 BBBY 9.8% 的股份，公开施压要求 BBBY 董事会进行改革，引发了该股股价的短暂回升。但随后 BBBY 又因为财报低迷而重新陷入了低谷。直到 8 月份股价大幅飙升之后，科恩才迎来了套现离场的机会。</p><p>并非偶然的是，科恩也是去年 GameStop 股价飙升背后的重要推动者。他同样是以投入介入，出任 GameStop 的董事长，给 WSB 散户投资者带来了可投资的利好信号。去年 1 月，GameStop 股价如火箭般飙升 17 倍，从 17 美元一路狂飙到 500 美元，不仅导致空头惨重，更引发了震惊美国的 Robinhood" 拔网线 " ( 暂停交易 ) 事件，甚至吸引了美国政府的介入调查。</p><p><strong>什么是 Meme 股？</strong></p><p>Meme 这个英文单词最早来自于英国生物学家理查德 · 道金斯 ( Richard Dawkins ) 在 1976 年的著作《自私的基因》 ( The Selfish Gene ) ，源自希腊语的模仿一词 "mim ē ma"。在互联网兴起之后，Meme 就被指代那些在网络上引发疯狂转发复制的热门图片、视频等内容，也被泛指那些迅速引发模仿和扩散的群体性行为。</p><p>理解了 Meme 的涵义，也就明白了 Meme 股的意思。当一支股票在社交网络上成为热议话题，吸引散户投资者疯狂买入，带动股价在短时间内急剧飙升，那么这就造就了又一支 Meme 股。从这个角度来说，把 Meme 股翻译成 " 网红股 " 或许比较适合。</p><p>Meme 股的现象背后，离不开社交网络的推动，是 " 带头大哥 " 的振臂一呼，带动了万千散户入市。尽管单个散户的买盘可能只有几百几千美元，但数以万计的散户力量汇聚之后，其强大的拉盘冲击力甚至可以冲垮在华尔街只手遮天的那些对冲基金巨头们。</p><p>暴增的交易量，非合理的飙升，恐慌性的暴跌，或许是对 Meme 股走势最好的描述。在社交媒体的炒作与煽动下，持续涌入的散户投资者以洪荒之力将这些股票推高到完全无理性的高位。然而，一旦 " 带头大哥 " 离场套现，散户们就可能出现恐慌，引发 Meme 股急剧暴跌。</p><p><strong>明日黄花陷入困境</strong></p><p>从 GameStop 开始，到 AMC 到 BBBY 到黑莓；从去年开始，美国股市先后有数支股票享受了 Meme 股的待遇，承受了散户投资者的疯狂涌入，经历了股价迅猛飙升又急剧回落的坐山车待遇，成为了美国股市乃至社会的聚焦热点。</p><p>回顾这些 Meme 股公司，大多基本面缺乏吸引力，甚至业务面临衰退，经营前景暗淡。在正常情况下，这些股票都不是理性投资者的选择目标，也顺理成章成为了做空投资者的打压对象。但是这些本来没有太大投资价值的股票，却成为了被散户们热捧的对象。</p><p>时过境迁，因为电子商务、流媒体和移动互联网大潮的兴起和新冠疫情的冲击，这些企业已经不复当年的风采，业绩已经明显下滑，股价更是陷入低迷。但另一方面，对那些已经成年的散户投资者来说，这些品牌都曾经是他们青春记忆的一部分。</p><p>GameStop 是美国视频游戏产品线下零售商，AMC 是美国最大的连锁影院，BBBY 是美国知名家居产品连锁商场，黑莓则曾经是智能手机里的顶尖专业品牌。就在 BBBY 股价飙升之前，这家公司的财报令人失望。第一财季净销售额下滑 25%，净亏损 3.6 亿美元，经营现金流是负 4 亿美元。现金头寸从去年同期的 11 亿美元急剧缩水到 1.1 亿美元。</p><p>尽管这些 " 明日黄花 " 已经陷入财务困境，但依然还在努力寻找新的业务模式，寻求新的融资。去年 GameStop 股价飙升的背后，则是关于他们计划拓展线上游戏租赁业务，计划成为 " 游戏领域 Netflix" 的传言。对于这些股价已经跌到谷底的传统企业来说，这种新业务的传言无疑是对股价的强心针。</p><p><strong>Meme 股多重背景</strong></p><p>这波 Meme 股的散户洪流最早可以追溯到 2020 年的美国疫情期间。在新冠疫情爆发之后，美国各地先后陷入停摆，大量民众开始在家歇业或者远程工作，有足够的自由空间上网。随后美国政府开始普遍发放疫情补助款，普通民众也可以拿到 2000 美元的补助金，不少人因此将补助金投入了股市。</p><p>与此同时，在美联储的无限放水宽松政策支撑下，美国股市不仅收复了疫情爆发之初的大幅下滑失地，更开始了一段延续到今年年初的牛市。一边是经济因为疫情陷入衰退，失业率飙升超过两位数，一边是股市在联储支撑下红红火火，所有人都想入市挣钱。美国经济和股市在 2020 年出现了冰火两重天的奇特景象。</p><p>用亿万富翁库班在 2020 年中的话来说，" 在这个牛市里，每个人都在挣钱，都觉得自己是投资天才，甚至很多根本不懂投资的人也入市了。" 他认为美国股市已经和经济基本面脱节，只是因为美联储的货币政策才出现泡沫。</p><p>Meme 股就是在这样的多重背景综合作用下出现的：疫情停摆导致大量民众在家上网，他们从政府拿到补助金可用于投资，美联储推动股市节节攀升，吸引了大量的散户投资者跃跃欲试。而这些经验不足的散户投资者则倾向于在网上寻找投资建议和买股推荐。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202208/630826358e9f093b273e6307_1024.jpg" data-height="613" data-width="1080" <p></p><p><strong>两大网络平台造就</strong></p><p>在 Meme 股现象的背后还有两个重要的互联网平台：社交媒体平台 Reddit 和散户投资平台 Robinhood。Reddit 上的 wallstreetbets ( 以下简称为 WSB ) 小组创建于 2012 年，这个投资社区吸引了 1250 万名用户，每一天都有数以万计的散户投资者在这里讨论个股走势和投资建议。几乎所有的 Meme 股都是开始于一些投资大 V 在这里的 " 一呼百应 "。当然，YouTube 和 Discord 等其他社交平台也在 Meme 股现象中扮演了重要角色。</p><p>而 2015 年起推出零佣金服务的 Robinhood 则为散户提供了入市的最好渠道。与传统券商不同，Robinhood 的营收并不来自于投资佣金，而是来自于他们与做市商的交易提成。Robinhood 的 " 低门槛、零佣金、灵活自主操作 " 模式更吸引了刚刚开始学习投资的年轻用户。这家平台的口号就是 " 让所有人都能投资 " ( Investing for everyone ) 。</p><p>Robinhood 的去年数据显示，该平台的主力用户群多是年轻个人用户，平均年龄只有 28 岁，而且半数都是首次进行投资，缺乏投资知识。因此，Robinhoo 成为了余钱不多的散户入市投资最佳选择，也和 WSB 上的散户投资者存在着高度重合。</p><p>2021 年第一季度的数据显示，Robinhood 的客户平均资产只有 4552 美元，远远低于老虎证券的 5.69 万美元，更只有盈透证券客户平均资产 24.95 万美元的一个零头。财报数据显示，2020 年 Robinhood 用户新投入资金数为 180 亿美元，远远高于 2019 年的 67 亿美元。这也与美国政府大量发放新冠补助金，刺激年轻人入市投资有着直接关系。</p><p><strong>毫不犹豫碾压散户</strong></p><p>Vanda Research 的统计数据显示，这个月美国散户投资者平均每天买入 13.5 亿美元的股票和交易所上市基金 ( ETF ) ，已经是今年 1 月以来的最高水平，但和去年年初的时候依然有明显差距。随着美联储开始进入加息周期，市场担心美国经济陷入衰退，美国股市也在今年 1 月开始进入下行通道。</p><p>实际上，散户们投资更多的还是苹果、特斯拉这些巨头企业。根据 Vanda Research 的统计，特斯拉才是美国散户投资者买入最多的股票，但散户们的力量并不足以带动这家市值 3000 亿美元的电动车巨头股票剧烈波动。相比之下，BBBY 的市值只有 8 亿美元，散户们的蜂拥而上很容易引发大幅震荡。</p><p>基金管理公司先锋集团在研究报告中认为，Meme 股是完全不同于传统股票投资的存在。传统股票投资的推动因素是公司业务基本面，而 Meme 股的影响因素则是社交媒体的话题热度。只要能引发效仿热度，业绩低迷的股票也能一飞冲天成为 Meme 股，但这样的涨势是脱离基本面的，因此能否延续高位完全在于该股的网络热度。</p><p>先锋集团建议散户投资者要有清晰的投资目标，建立多样化投资组合，控制自己的买入成本，并有长期的投资策略。简单的说，就是建议年轻网民不要盲目跟风，更不要把鸡蛋放到同一个篮子里。因为洪水来得快，也退得猛，</p><p>雅虎财经的特约分析师索西 ( Brian Sozzi ) 也呼吁跟风 Meme 股的散户投资者们保持冷静。他写道，自己理解现在的投资者理念和传统长期投资者有着明显差别，也理解年轻人可能对 BBBY 或是 GameStop 这样的公司抱有情怀，更明白散户投资者对科恩这样的活跃投资者有着很高的预期。但索西警告称，散户投资者应该对科恩这样的带头大哥保持警惕，毕竟科恩的目的就是通过低买高卖赚钱，只要能够达到目的，他会毫不犹豫地碾压散户投资者们。</p><div
