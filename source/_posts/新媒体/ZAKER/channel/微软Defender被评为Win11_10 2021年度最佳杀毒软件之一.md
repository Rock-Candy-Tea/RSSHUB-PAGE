<p>IT 之家 11 月 27 日消息，据 Windows Latest 报道，AV-TEST 发布了最新报告，预装在 Windows 10 和 Windows 11 中的 Windows Defender 是 2021 年最好的反病毒软件之一。AV-TEST 是一家独立的 IT 安全机构，进行各种测试，根据性能、安全和其他因素对反病毒软件进行排名。</p><p></p><div 月的最佳防病毒程序评估报告。值得指出的是，该结果是针对 Windows 10 家庭用户的，该安全机构测试了来自不同供应商的 21 个安全程序，包括诺顿、卡巴斯基和微软。<p></p><p>根据报告，AV-TEST 对 Windows Defender、Avira、McAfee、Avast、AVG 和其他几个反病毒应用程序进行了各种安全方面的测试。例如，该组织针对零日恶意软件、网络攻击、受感染的电子邮件等测试了这些防病毒解决方案。</p><p>" 我们一直使用所有产品的最新公开版本进行测试。他们被允许在任何时候自我更新，并查询他们的云端服务，" 安全报告中写道。</p><p>AV-TEST 有严格的测试程序，Windows 10 上的 Windows Defender 在所有保护、性能和可用性方面都获得了满分。McAfee 和 Avast 等厂商的其他安全解决方案也获得了满分，但 Windows Defender 是唯一的免费防病毒解决方案。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img AV-TEST 结果截图中可以看到，Windows Defender 获得了 18 分，该组织宣布微软 Defender 是 Windows 10 的顶级安全解决方案之一，估计也是 Windows 11 的顶级安全解决方案。</p><p>用于 Windows 11 的新 Defender 应用程序</p><p>IT 之家获悉，作为改善跨平台安全解决方案的努力的一部分，微软显然正在开发新的安全中心。该应用程序被称为 "Microsoft Defender Preview"，它将允许用户监控他们的 Windows 11 PC、Mac 和移动平台（Android 和 iOS）。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div
