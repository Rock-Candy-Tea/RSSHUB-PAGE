<p>由于 Windows 和 Android 系统之间的鸿沟，玩家想在电脑上玩手游，一般只能通过模拟器实现，而且不少热门游戏都对平台进行了限制，比如《和平精英》等，手游玩家和模拟器玩家会进入不同的区 / 服，体验并不好。</p><p>那么，能不能在 Windows 电脑上越过模拟器，直接玩 Android 游戏呢？谷歌的答案是：能，明年上线！</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202112/61b32a978e9f0930c1774dc6_1024.jpg" data-height="315" data-width="448" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202112/61b32a978e9f0930c1774dc6_1024.jpg" referrerpolicy="no-referrer"></div></div>今天，在 " 游戏界奥斯卡 "TGA 颁奖典礼上，谷歌预告了一项新技术：将允许玩家在手机、平板电脑或 Chromebook 上玩游戏后，在桌面 PC 上进行无缝游戏。<p></p><p>谷歌发言人 Alex Garcia-Kummert 表示，公司正在独立开发儿歌应用，并且承诺将在明年正式发布。虽然没有公布明确时间，但可以肯定的是，游戏是直接在 Windows 电脑上本地运行，而非来自云端，也就是说，对于网络的要求可能不会那么严格，游戏体验也不会动不动就 "460"（因网络延迟而造成的卡顿）。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202112/61b32a978e9f0930c1774dc7_1024.jpg" data-height="461" data-width="820" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202112/61b32a978e9f0930c1774dc7_1024.jpg" referrerpolicy="no-referrer"></div></div>其实早在 Windows 发布的时候，就已经宣布在 Windows 11 系统底层建立了一套安卓子系统，可以直接运行部分安卓应用，但当时用于演示的大多是日常应用而非游戏。其原理和我们常见的模拟器比较类似，都是直接在 Windows 系统中开启一个窗口运行游戏，不用进行系统切换。<p></p><p>而且这个功能是由微软和亚马逊合作推出的项目，仅能使用亚马逊 AppStore 所提供的应用程序，所以应用和游戏的丰富程度肯定会大打折扣。</p><p>而这次谷歌自行下场开发的 Windows 应用程序，主流 APP 自然都能找到。按照谷歌的说法，可以进行 " 无缝游戏 "，也就是说数据应该是和移动平台互通的，不会存在单独的区 / 服，如果真是这样，对于用户来说还是十分友好的。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
