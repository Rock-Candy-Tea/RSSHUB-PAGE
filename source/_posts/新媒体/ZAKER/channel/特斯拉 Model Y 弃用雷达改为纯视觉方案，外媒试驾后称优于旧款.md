<p>海外频道 DAErik 发布了一期视频，谈到了新款 Model Y 与旧款车型的区别。</p><p>新款与旧款间最大的差别恐怕就是不再装有毫米波雷达了。有些人反对移除毫米波雷达主要是害怕自车辆的安全性下降，毕竟能力的削弱可能会导致一些至关重要的主动安全性功能失效，比如自动紧急制动、车道偏离预警 / 避让等。</p><p>DAErik 表示，新款 Model Y 的纯视觉自动驾驶方案与旧款有雷达方案相比，至少在用户日常体验层面没有明显区别，他甚至表示新款车型的自动驾驶表现依然很好，车辆行驶起来比甚至比旧款更灵敏、果断、精确。</p><p>至于不少人都关心的主动安全性功能，由于没遇到有可能会触发上述功能工作的危险情况，他无法断言纯视觉方案的表现如何。</p><p>不过，新款在一些细节上比旧款要更加让人喜欢，比如雨刷器上的玻璃水喷嘴、前排车窗双层玻璃、防眩目外后视镜等。他认为，抛开自动驾驶系统表现不谈，新款 Model Y 也是比旧款更加优秀的车。</p><p>一、纯视觉方案引争议 车主却说表现更好</p><p>不久前，特斯拉官网上发布了关于智能驾驶系统 " 纯视觉转型 "（Transitioning to Tesla Vision）的公告。公告表示，最早于 2021 年 5 月交付到北美市场的 Model 3 和 Model Y 将不再搭载任何形式的雷达系统，此前 Autopilot、FSD 和其他所有主动安全性功能将全部由纯视觉感知系统和神经元网络处理系统来执行。</p><p>相比于有雷达的旧款车，无雷达的新款车只是会有少数功能暂时无法激活或者在设定上有所调整，大多数功能都得以保留。而且，特斯拉承诺会在几个星期之内通过 OTA 的方式重新为车辆添加暂时缺失的功能。</p><p></p><div class="img_box" id="id_imagebox_0" referrerpolicy="no-referrer"></div></div>▲旧款与新款车型主动安全性功能对比<p></p><p>此举引发了广泛争议，不管是消费者还是测试机构，都怀疑新款车型的主动安全性功能有所缺失。</p><p>毕竟毫米波雷达擅长的是精确探测物体与车辆间的距离，纯视觉方案中摄像头恐怕并不能很好地探测距离，而测距功能的削弱可能会导致不少主动安全性功能失效，比如自动紧急制动（AEB）、车道偏离预警 / 避让等。</p><p>DAErick 体验过新款 Model Y 之后表示，大家的担心恐怕是没有必要的。他和一些朋友都曾经长时间驾驶、使用旧款 Model Y，对这款车的 Autopilot 系统及其功能有深入的了解和深刻的印象。</p><p>试乘试驾过新款 Model Y 之后，他们一致认为至少在 Autopilot 系统的表现上，纯视觉方案的表现相比有雷达方案而言毫不逊色，甚至更好。</p><p>他们发现，新车的 Autopilot 操纵车辆的方式比旧款更果断、精确，更像人类中的 " 老司机 "。</p><p>比如在面对一个大弯道时，系统会抓住恰到好处的时机，柔和地进行减速。随后稍微打一点方向盘调整车头，慢慢加大转向角度把车身掰进弯道。通过弯心后，系统会柔和地完成方向回正，并以同样顺滑的方式完成再加速。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div"><img 可以很柔和地通过弯道<p></p><p>整个过程中，车辆始终处于道路相对中央的位置，没有出现轮胎马上要压线了系统才反应过来、然后突兀地改变行车状态的情况。</p><p>这样的表现往往是人类司机中经验丰富之人才有的。DAErick 和朋友们都觉得，或许搭载了毫米波雷达的旧款车让人们从直觉上认为它更安全，但它上面 Autopilot 的最佳表现也不过如此了。</p><p>二、主动安全性尚无权威背书 打消疑虑还需测试结果</p><p>值得指出的是，现在的纯视觉方案要求车主必须先激活远光灯自动开启功能，然后才能使用 Autopilot。DAErick 认为这无可厚非，毕竟失去了毫米波雷达辅助感知环境，摄像头在没有光照不足的情况下无法有效地进行观测。不过，远光灯的频繁开启可能会给对向车辆的驾乘者造成视线干扰、引起不适，这实在是没办法的事情。</p><p>另外，纯视觉的 Autopilot 只能在车速不超过 75mph（约合 121km/h）的情况下才能正常工作。特斯拉曾经承诺未来会通过 OTA 的方式给新款车进行功能性补全，但并未透露最终会让 Autopilot 在什么样的速度范围下正常工作。</p><p>格外值得强调的是，DAErick 上传的视频中，试驾片段都是在光照充足、交通畅通场景下获取的。也就是说，人们不能简单地认为纯视觉方案在光线不足、交通拥堵的情况下能有同样优秀的表现。</p><p>还有一点既值得欣慰又令人遗憾，那就是 DAErick 提供的试驾片段中，没有出现外界车辆紧急变线 / 刹车或者行人鬼探头的情况。</p><p>人们最怕的就是这种危险的情况，自动紧急制动、车道偏离预警 / 避让这样的主动安全性功能也是针对这种危险的情况而诞生的。纯视觉方案暂时饱受质疑，很大程度上也是因为不少人不相信移除毫米波雷达后系统还能从容应付上述情况、避免事故发生。</p><p>偏偏 DAErick 和朋友们自始至终没遇到可能会触发自动紧急制动或车道偏离预警 / 避让功能的情况，所以人们还不能仅仅根据他们的言论就判定纯视觉方案的主动安全性好或不好。当然了，没遇到上述危险情况是人的福气，人是否平安永远比车是否安全更重要。</p><p>另外，DAErick 并没有透露试驾车是否拥有 FSD 功能，所以他们的言论仅适用于对 Autopilot 的描述，不能沿用到 FSD 上。</p><p>有传闻称美国国家道路交通安全管理局（NHTSA）和高速公路安全保险协会（IIHS）可能会在最近对搭载了纯视觉自动驾驶方案的特斯拉车型进行测试，以对其整车安全性重新进行评估。届时，测试结果将是粉碎一切流言的最佳武器。</p><p>三、细节改进带来更好体验 新款就是比旧款好</p><p>除了智能驾驶系统之外，DAErick 还对新款 Model Y 上的一些细节改动进行了介绍。他认为，这些细节上的变化可能直接被某些人忽略了，但正是这些细节让新款车提供的用户体验胜过旧款车。</p><p>最令他满意的改进是玻璃水喷嘴。在他的旧款 Model Y 上，只有前机盖后面有一个玻璃水喷嘴，喷射面非常小，每次使用雨刷器的时候，他都会发现前挡风玻璃上有些地方怎么也刷不干净。</p><p>新款车型上，机盖后方的喷嘴被增加到了两个，喷射面覆盖更广。更让他惊喜的是，新款车雨刷器上居然自带喷嘴。每当雨刷器摆动起来，喷嘴就有机会把玻璃水滋到前挡风玻璃上的几乎每一个角落。</p><p>这样的话，雨刷器简单摆动几个来回，整片前挡风玻璃都变得非常干净，给人的感官体验非常好，实在是赏心悦目。</p><p></p><div 指出，自己的旧款 Model Y 前门玻璃是单层的，而新款车型则与新款 Model 3 一样，都采用了双层玻璃。双层玻璃既能降低人在车内听到的噪音的音量，又能让冷气或热量更不容易流失，还能提升玻璃整体强度，可谓是一举三得。</p><p>还有一点是防眩目外后视镜。其实 DAErick 也没想到新款车有这个配置，因为他的旧款车没有，官方给出的新款车配置明细也只是提到了后视镜支持电动折叠且拥有自加热除雪功能。</p><p>但是，试驾车确实拥有防眩目后视镜，虽然 DAErick 也不知道是通过电子原理还是贴膜实现的，但他明显感觉到镜片里后车的灯光亮度要比身后传来的灯光亮度更暗。</p><p>其他的改动还包括亮起来时亮度更高且颜色更醒目的尾灯、样式更新颖且有带锁储物格和 USB 插口的中控扶手台等，总地来说都是让 DAErick 喜欢的改动。</p><p>不过，DAErick 明确指出，如果想要购买新款 Model Y 的话千万别买 7 座版，因为最后排的乘坐体验减值令人发指。身高 5 英尺 10 英寸（约合 1.78 米）的他坐在最后排的时候，就算是窝着脖子，头也会直接顶到后挡风玻璃上。</p><p>他的一位女性朋友也表示 Model Y 最后排空间太小，身高 5 英尺 1 英寸（约合 1.55 米）的她头倒是离后挡风玻璃还有几厘米距离，但横向空间、纵向的腿部空间还是太小了，这样的位置只适合非常年幼的孩子座。</p><p>二者都表示，如果你真的对内部空间和座位数有需求，最好去买 Model X。</p><p>结语：车辆本身升级也值得关注</p><p>马斯克宣布智能驾驶系统开启纯视觉化转型后，放弃了毫米波雷达的新款 Model 3 和 Model Y 成为了被怀疑、甚至口诛笔伐的对象。有人指出，放弃毫米波雷达对降低系统成本的作用微乎其微，反而大概率会让自动紧急制动、车道偏离预警 / 避让这样至关重要的主动安全性功能失效。</p><p>此次 DAErick 参与的新款 Model Y 体验活动中并没有出现能触发上述功能的危险情况，所以他并不能对纯视觉方案的主动安全性做出更多的评价。</p><p>不过，作为一个旧款 Model Y 车主，他指出了一些可能会被不少人直接忽略、但却能切实提升用户体验的细节，比如雨刷器上的玻璃水喷嘴、防眩目外后视镜等。</p><p>纯视觉方案到底靠不靠谱只有权威机构出具的测试报告说了算，但新款 Model Y 的用户体验是大家都能说出来的，也是跟人们日常用车关系更密切的，毕竟不是所有人都会时时刻刻陷入危险情况中。</p><p>至少，DAErick 认为新款 Model Y 的用户体验比旧款的更好。且不论智能驾驶系统表现如何，他从细节上感受到了来自厂家的诚意。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
