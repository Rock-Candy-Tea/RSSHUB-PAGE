<p>IT 之家 8 月 26 日消息 外媒 MacRumors 报道，据爆料人士 Jon Prosser 称，苹果目前正在测试新的 Face ID 硬件，可以让用户在戴着面具口罩或雾状眼镜时解锁他们的 iPhone。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202108/6126d3da8e9f097ea3504d1e_1024.jpg" data-height="506" data-width="900" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202108/6126d3da8e9f097ea3504d1e_1024.jpg" referrerpolicy="no-referrer"></div></div>▲ 升级后的 Face ID 原型渲染图<p></p><p>Prosser 解释说，苹果正在使用一款紧贴 iPhone 12 的外壳测试更先进的 Face ID 硬件。该保护套允许 iPhone 绕过其内置的 Face ID 系统，而使用该保护套的系统。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202108/6126d3da8e9f097ea3504d1f_1024.jpg" data-height="829" data-width="757" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/6126d3da8e9f097ea3504d1f_1024.jpg" referrerpolicy="no-referrer"></div></div>Prosser 认为，Face ID 原型阵列与今年早些时候 iPhone 13 系列的 CAD 文件中描述的布局完全一致。这是因为外壳中的 Face ID 原型硬件比 iPhone 12 的硬件窄得多，这与 iPhone 13 机型上预计的更细的凹槽相一致。<p></p><p>由于正在进行大规模测试以收集尽可能多的数据，显然需要外部 Face ID 硬件。据称，这使得苹果可以在不分发未发布的 iPhone 的情况下，给大量员工提供新硬件。</p><p>参与的员工被要求在一系列条件下佩戴面罩和眼镜的组合来测试新硬件。员工只是被要求进行测试，但没有被告知案例中的 Face ID 传感器阵列有什么具体变化。</p><p>IT 之家获悉，据报道，新硬件允许 iPhone 在戴着面具、不借助 Apple Watch 和戴着有雾的眼镜时用 Face ID 解锁。</p><p>虽然 Prosser 强调了该手机壳的 Face ID 硬件与即将推出的 iPhone 13 上预期的更小的凹槽有惊人的相似之处，但目前还不清楚这种改进的 Face ID 是否会随着 iPhone 13 或未来的 iPhone 机型而出现。</p><p>这些测试显然是最近进行的，因此 Prosser 表示，如果改进后的 Face ID 不能立即与 iPhone 13 系列一起使用，它可能会在以后的软件更新中推出。</p><p>Prosser 选择不分享所谓的原型机的图片，因为在上面发现了识别标记，而是分享了一些基于图片的渲染图。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
