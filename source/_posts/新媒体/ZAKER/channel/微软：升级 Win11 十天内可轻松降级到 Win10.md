<p>IT 之家 7 月 8 日消息 Windows 11 已经到来，正式版将在今年晚些时候或明年年初开始向消费者免费推出。如果你遇到了硬件、应用程序的问题，或者就是不喜欢 Windows 11，微软称都可以轻松回滚到 Windows 10。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202107/60e6555f8e9f094a093f3dd0_1024.jpg" data-height="392" data-width="696" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/60e6555f8e9f094a093f3dd0_1024.jpg" referrerpolicy="no-referrer"></div></div>与 Windows 10 以前的功能更新相比，Windows 11 有着很大的视觉升级，包括新的开始菜单、通知中心、系统托盘、重新设计的任务栏和圆角。根据联想和微软的说法，如果你想恢复到 Windows 10，你将有 10 天的时间进行原状态（in-place）降级。<p></p><p>如果用户在过去 10 天内升级到 Windows 11，降级过程是非常直接的。如果超过了 10 天，那么这个过程就稍微复杂一些，因为需要清洁安装之前的操作系统（即 Windows 10）来卸载 Windows 11。</p><p>" 一旦你升级到 Windows 11，你有 10 天时间利用回滚功能移回 Windows 10，同时保留之前的文件和数据。如果在 10 天之后，你需要备份你的数据，并进行‘清洁安装’以移回 Windows 10，" 联想的支持文件中写道，该文件是根据微软的意见编写的。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202107/60e6555f8e9f094a093f3dd1_1024.jpg" data-height="704" data-width="968" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/60e6555f8e9f094a093f3dd1_1024.jpg" referrerpolicy="no-referrer"></div></div>如果你升级到 Windows 11 后不到 10 天，可以很容易地降级到 Windows 10。该功能是通过 Windows 内置的恢复和备份镜像实现的，用户可以通过设置 > 更新和安全 > 恢复来进行降级。<p></p><p>然而，如果已经超过了 10 天，那么将无法进入恢复选项，但仍然可以恢复旧的操作系统。正如微软和联想所指出的，可以通过执行清洁安装来恢复到 Windows 10。</p><p>IT 之家了解到，要进行 Windows 10 的清洁安装，可以使用微软的媒体创建工具，这个工具将允许你重新安装 Windows 10 并制作一个可启动的 U 盘。然而，这个过程将删除你的应用程序和设置，这意味着你需要手动备份文件和应用程序。</p><p>如果你不喜欢媒体创建工具，也可以手动下载 Windows 10 ISO，并手动创建自己的 USB 驱动器。你也可以在另一个分区上重新安装 Windows 10，并在需要时重新激活许可证。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
