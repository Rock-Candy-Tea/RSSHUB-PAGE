<p>IT 之家 7 月 16 日消息 今天微软发布了 Windows 10 21H1 Build 19043.1147 ( KB5004296 ) 更新，面向 Release Preview 发布预览频道推送。</p><p>在这篇关于 Windows 10 21H1 博客文章中，微软还透露，他们将很快向 Beta 频道的 Windows Insiders 用户推送 Windows 11 预览版版本。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202107/60f170b48e9f091fce635744_1024.jpg" data-height="960" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/60f170b48e9f091fce635744_1024.jpg" referrerpolicy="no-referrer"></div></div><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202107/60f170b48e9f091fce635745_1024.jpg" data-height="120" data-width="1062" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60f170b48e9f091fce635745_1024.jpg" referrerpolicy="no-referrer"></div></div>微软表示，<p></p><p>" 注意：21H1 的 Beta 通道中的 Windows Insiders 将不会收到这个更新。我们将很快向这些内测人员推送 Windows 11 Insider Preview 构建版本。"</p><p>微软指出，Beta 频道是早期采用者的理想选择。Beta 频道中的内测人员和 IT 专业人士可以检查即将到来的 Windows 功能，同时仍然获得由微软验证的相对可靠的更新。</p><p>IT 之家获悉，据传闻，微软 Windows 11 将在今年 10 月推向硬件 PC 设备，微软将需要努力工作（并需要大量的协助），以验证该操作系统在这么短的时间内准备好 RTM 正式版，首个 Windows 11 Beta 预览版预计将在 7 月底之前发布。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
