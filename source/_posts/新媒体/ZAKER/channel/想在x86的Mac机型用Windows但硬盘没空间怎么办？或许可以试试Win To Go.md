<p>现在 Mac 用户群并不小，其中大部分都是使用 Macbook 系列产品的用户，包括 Macbook、MacBook Air、以及 MacBook Pro，作为生产力工具，进行内容创作。虽然苹果早在去年就已宣布，今后两年内将逐步转向自研芯片，但是在未来一段时间内，仍然有大量搭载英特尔 x86 处理器的 Macbook 系列产品在使用。</p><p></p><div Windows，近期小编就遇到类似的情况。<p></p><p>相比搭载自研芯片的产品，x86 处理器在这方面有着一定的便利性。在过去，大多会使用 BootCamp（仅限于 x86 处理器）在 Mac 上来安装 Windows 系统，不过随着技术的成熟，以及苹果进一步转向自研芯片，越来越多的人使用虚拟机的方式。除了可能在使用过程中会遇到一些莫名其妙的问题，这两种方式都有着一定的局限性。</p><p>BootCamp 需要额外分配 Windows 系统需要使用的空间，与 macOS 并不通用，对于普遍存储空间相对捉襟见肘的 Mac 产品来说，似乎有点浪费。虚拟机需要一定的电脑操作知识，但部分用户缺乏这样的基础。除此以外，还有另外一种方式，既不会占用 Mac 上原有的宝贵空间，也可以像平常使用 Windows 系统那样操作，那就是 Windows To Go。</p><p>简单来说，Windows To Go 就是将 Windows 系统安装到一个 USB 驱动器中（比如 U 盘或移动固态硬盘）。这个安装了 Windows 系统的驱动器，可以在任何一台没有安装 Windows 系统的设备中运行 Windows 系统。平常的 Mac 就属于这种情况。（进一步了解可参阅微软官方的 Windows To Go 功能概述）</p><p>相比使用 BootCamp 或虚拟机，Windows To Go 的优势有：</p><p>不会占用 Mac 上原有的硬盘空间，不需要用的时候也无须接入，而且必要时在其他设备也能使用。</p><p>macOS 与 Windows 系统相互独立，互不影响。</p><p>可以提供足够好的体验，若采用移动固态硬盘，无论性能还是价格都很合适。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div 或以上。一般情况下，可以在移动固态硬盘、U 盘和移动机械硬盘之间选择。小编选择的是致钛木星 10 512GB 移动固态硬盘，一方面价格适中，另外一方面其性能和容量都足够满足需求。<p></p><p>在现阶段，512GB 的容量已经可以满足使用的存储需要。作为一款固态硬盘，致钛木星 10 相比相比 U 盘和移动机械硬盘，读写速度有着绝对的优势，综合性能表现上要好很多。不但可以充分利用了 USB 3.0 接口的带宽，而且实际使用体验也会有很大的不同。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" 存储设备。</p><p>备用的 USB 存储设备，用于启动 BootCamp 的操作。</p><p>需要有线鼠标。</p><p>按照微软官方的做法，Windows 10 系统需要是企业版或者教育版，因为这两个版本才有 Windows To Go 的工具。不过对于大部分的用户，使用的都并非这两个版本的 Windows 系统，那么可以选择通过第三方工具完成（具体步骤可网上搜索）， 在 Windows 8/8.1/10 系统的 PC 上都可以完成了，操作并不难。</p><p>如果是配备了 Apple T2 安全芯片的 Mac 产品，需要开机后按 " +R" 键进入 "macOS 实用工具 "，在 " 实用工具 " 里选择 " 启动安全性实用工具 "，改成 " 允许从外部介质或可移动介质启动 "。</p><p>开机后，按 "Option"，选择 "EFI Boot"，然后就像正常的 PC 一样操作设置进入 Windows 系统的界面。在这个过程中，个别 Mac 机型可能会遇到键盘或触摸板不能正常使用的情况，可以选择连接有线鼠标，或者选择辅助工具里的 " 屏幕键盘 "。</p><p>完成相关操作，进入 Windows 系统后，将存有 "WindowsSupport" 文件夹的 USB 存储设备接入。将该文件夹复制粘贴到装有 Windows 系统的移动固态硬盘里，选择里面的 "BootCamp" 文件夹，点击 "Setup"，接着就像正常安装应用程序一样，将对应 Mac 机型的相关 Windows 驱动程序都会安装好。结束后，重启再重新进入 Windows 系统，所有功能就可以正常使用了。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div 驱动器是目前最平衡且实际的解决方案，足够的速度、足够的存储容量、以及合理的价格。致钛木星 10 有着不少优点，满足了使用的需要，包括：<p></p><p>体积小，重量轻，方便携带。</p><p>速度和容量满足平常使用需要。</p><p>坚固的材质，减小设备损坏的机率，保证数据安全。</p><p>由于苹果已转向自研的 Arm 架构芯片，逐渐淘汰搭载英特尔 x86 处理器的机型，加上微软出于安全等多方面因素考虑，已经停止继续开发 Windows To Go，这种方式可能会慢慢不再适用。不过在未来的几年时间里，使用搭载英特尔 x86 处理器的 Mac 用户，有需要时仍然可以在上面使用 Windows 10 系统，进行简单的操作，而且不必浪费原机型内宝贵的存储空间，是一种比较方便、快捷、合适的选择。</p><div
