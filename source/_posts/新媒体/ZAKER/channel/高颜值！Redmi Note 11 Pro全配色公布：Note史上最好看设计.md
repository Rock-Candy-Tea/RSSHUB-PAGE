<p>10 月 28 日消息，今晚 19:00，Redmi Note 11 系列新品发布会如约而至，在介绍了小米新零售、Note 系列销量以及新策略等情况后，今晚的主角—— Redmi Note 11 系列正式登场。</p><p>ID 设计上，Redmi Note 11 Pro 号称 Note 有史以来最好的设计，拥有潮流小立边 + 全机身统一 R 角，有板有眼。</p><p>另外，在 AG 磨砂玻璃的加持下，整机触感温润不留指纹，质感绝佳，8.34mm 的轻薄机身，更是进一步提升手机握持感。</p><p>配色方面，Redmi Note 11 Pro 拥有三款旗舰配色，包括迷雾森林、时光静紫、神秘黑境。</p><p>值得一提的是，这一次，Note 11 Pro 首次挑战闪光少女风，为女孩们献上 2021 冬日特调—— " 浅梦星河 " 配色。</p><p>官方表示，璀璨微钻设计，像她们身上的无数闪光点，希望每个女孩都能心怀热爱，绽放最耀眼的光。</p><p>此外，Redmi Note 11 Pro 还拥有 Note 有史以来最好的三星 120Hz 高刷屏：6.67 英寸 2.96mm 针孔屏，以及 1.75mm 超窄边框。</p><p></p><div class="img_box" id="id_imagebox_0"
