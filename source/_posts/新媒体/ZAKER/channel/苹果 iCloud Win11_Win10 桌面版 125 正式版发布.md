<p>IT 之家 8 月 17 日消息 外媒 MacRumors 报道，苹果公司今天发布了新版本的 iCloud for Windows 应用，支持 Windows 11、Windows 10 系统等，iCloud 12.5 版本的更新为 Windows 用户增加了新的 iCloud 钥匙链密码管理器应用。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202108/611b01e08e9f09047460ab04_1024.jpg" data-height="700" data-width="864" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/611b01e08e9f09047460ab04_1024.jpg" referrerpolicy="no-referrer"></div></div>有了新的密码管理选项，运行 Windows 的用户可以访问 iCloud 钥匙链密码，并可以添加、编辑、复制和粘贴、删除，以及查询用户名或密码。苹果在今年 1 月份发布了 Windows 版 iCloud 的更新版本，暗示了新的密码应用，但苹果直到现在才实施。<p></p><p>微软 Edge 浏览器和谷歌 Chrome 浏览器的新 iCloud 密码扩展与密码功能协同工作，让 iCloud 密码在 Windows 设备上自动填写，就像在 Safari 浏览器中一样。为网站创建的用户名和密码会自动添加到密码应用程序中。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202108/611b01e08e9f09047460ab05_1024.jpg" data-height="701" data-width="872" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/611b01e08e9f09047460ab05_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家获悉，Windows 版 iCloud 钥匙链密码存储在一个加密的数据库中，并使用加密通道传输到浏览器扩展，密码不以明文存储。<p></p><p>iCloud for Windows 密码是 iCloud for Windows 12.5 的一部分，Windows 用户可以从微软商店下载它。适用于 Windows 的 iCloud 应用程序旨在让 Windows 用户从 Windows 设备访问他们的 iCloud 文件、照片、电子邮件等。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
