授权，也就是预装到电脑里一起卖，这种相较零售版便宜许多，并且系统价格包含到了电脑售价中，消费者不容易感知到。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" Windows 7 的更新，持续不断地推出新的盗版系统，一直到 2015 年 Windows 10 发布时，盗版依然是主流。而在今天，随着品牌机（OEM 授权）的普及和消费者版权意识的加强，正版 Windows 才逐渐回到舞台中央。<p></p><p><b>「掏空」Windows，教微软做用户体验</b></p><p>番茄花园们的「成功」离不开盗版，但又不完全依赖盗版，除了让用户免费使用外，他们还通过各种技术手段帮微软把 Windows 变得更加好用，甚至搞出了许多创新性的功能和体验，受到了广大用户欢迎。</p><p>2000 年代的头十年，中国市场的电脑普及程度和电脑配置普遍比较落后，尤其是在 Windows XP 发布初期，大部分电脑并不能流畅运行，而番茄花园通过一些神优化，不仅让 XP 变得更流畅，而且还更好看。在番茄花园的自我介绍中，他们毫不避讳地写着「为什么选择番茄花园」，这其中包括：能够通过正版验证、集成系统最新补丁、可选漂亮主题、可选必要优化等。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div 小时左右，首先得经过漫长等待，当你终于看到「欢迎使用」界面后，接下来迎接你的却是一片空白的桌面，而且右下角莫名其妙的放着「回收站」，对广大菜鸟用户来说，连「我的电脑」和「宽带连接」都找不出，更别说后续装软件、装驱动了。更有意思的是，2006 年微软发布新系统 Windows Vista 之后，由于处在软硬件新老交替期间，Vista 暴露出了很多兼容性问题，后来有些「正版用户受害者」实在难以忍受，只好去电脑城重新安装盗版 XP 系统。<p></p><p></p><div class="img_box" id="id_imagebox_8" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_8" 元的售价，明显更容易被接受。当时消费者的版权意识普遍薄弱，许多人也不知道自己在用盗版，再加上盗版系统确实有着比原版更好的体验和高效的装机方式，于是久而久之，盗版成为了主流，甚至后来有很多人都不知道，Windows 系统也是需要花钱买的，因为大部分人认为，他们是为装系统花的钱，而不是操作系统本身。<p></p><p></p><div class="img_box" XP 系统停留在了 1.21 版本，如今打开这款绝版系统的安装程序后，还会自动播放《北京欢迎你》的背景音乐。<strong>（在极客之选公众号回复「XP」即可获取番茄花园 1.21 绝版系统）</strong></p><p>一年后的 2009 年 8 月 20 日，苏州一法院对「番茄花园」涉嫌侵犯著作权一案作出判决，主犯洪磊被判三年半，并处罚金 100 万元。此外，在本案中全面策划并操控「番茄花园」商业运作的孙显忠亦被判处有期徒刑三年半，并处罚金 100 万元；被告单位成都共软网络科技有限公司被判处没收违法所得 292 万余元，并处罚金 877 万余元。</p><p></p><div class="img_box" id="id_imagebox_10" onclick><div class="content_img_div 小时内删除。如果您觉得满意，请购买正版！」，这种「掩耳盗铃」的声明显然是灰色产业的惯用话术，不过番茄花园的确只提供修改版系统，而且是免费下载，至于市面上的光盘均为第三方制作和销售，番茄花园并没有因此获利。</p><p></p><div class="img_box" id="id_imagebox_11" onclick><div class="content_img_div 系统的最初定价并不合理，没有考虑到中国市场的特殊现状，到了 2008 年时，Windows XP 已经不是最新的系统，家庭版的价格已经从发布时的 1498 元降到了 960 元，而在番茄花园事件后，微软再次下调至 399 元。</p><p></p><div class="img_box" id="id_imagebox_12" onclick><div class="content_img_div Windows Windows 操作系统和 Office 办公软件属于最高级别的垄断，在此之前，微软也在很多国家遭到了类似的反垄断调查。然而在中国微软回应道：由于微软的系统和软件大部分以非正版方式存在，正版的市场份额很小，因此构成垄断的前提不存在。也就是说，和盗版划清界限，让微软在中国化解了反垄断危机。</p><p></p><div class="img_box" id="id_imagebox_13" 年代来到中国后，微软面对的是一个盗版横行、版权意识薄弱的市场，所以干脆灵活变通，睁一只眼闭一只眼，用盗版去换市场，在中国这就是利益最大化，而且即便是盗版，微软也可以将部分主权掌握在自己手中。所以微软从「纵容」盗版发展再到后来打击盗版，这一系列操作从法理上讲没有任何问题，至于合不合乎情理，就要看站在什么立场了。<p></p><p><b>天下没有免费的午餐</b></p><p>像对待番茄花园一样，微软早在
