<p>IT 之家 5 月 22 日消息，" 环境变量 " 是 Windows 系统中简化命令行的一项重要工具，然而，该工具一直都不怎么方便，需要用户打开重重设置，然后在一个很复古的窗口中进行设置。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202205/6289f6c18e9f09726747868e_1024.jpg" data-height="303" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/6289f6c18e9f09726747868e_1024.jpg" referrerpolicy="no-referrer"></div></div>在 PowerToys 的 GitHub 页面，微软已经承认环境变量页面的 " 可读性 " 非常差，初学者很难理解路径或指定文件夹等内容。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202205/6289f6c18e9f09726747868f_1024.jpg" data-height="677" data-width="1136" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202205/6289f6c18e9f09726747868f_1024.jpg" referrerpolicy="no-referrer"></div></div>于是，微软打算在 PowerToys 中添加一个名为 " 环境管理器 " 的工具，让管理环境变量变得更容易。<p></p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div"><img class="lazy opacity_0 zaker_gif_cache" id="img_2" data-original="http://zkres1.myzaker.com/202205/6289f6c18e9f097267478690_1024.jpg" data-gif-url="http://zkres1.myzaker.com/202205/6289f6c18e9f097267478690_raw.gif" data-height="716" data-width="779" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/6289f6c18e9f097267478690_1024.jpg" referrerpolicy="no-referrer"></div></div>从图中可以看到，该工具允许用户像管理列表一样插入环境变量，并帮助用户定义运行可执行文件的路径。用户可在 " 用户变量 " 和 " 系统变量 " 下直接点击 " 添加变量 " 按钮，看起来是不是很方便。<p></p><p>IT 之家了解到，该工具的环境变量编辑规则与原版相同，如果需要添加多个值，则需要用分号来隔开。</p><p>微软名为 PowerToys 的免费系统实用工具套件从 Windows XP 时代复活，并于 2019 年 5 月引入 Windows 10 系统。可以用于高级用户调整和简化 Windows 操作，以提高效率。通过与 Windows 10/Windows 11 shell 搭配使用，同时让用户可以自定义各个工作流。</p><p>目前该工具仍处在开发中，我们可以期待一下在 PowerToys 之后的版本中看到它。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
