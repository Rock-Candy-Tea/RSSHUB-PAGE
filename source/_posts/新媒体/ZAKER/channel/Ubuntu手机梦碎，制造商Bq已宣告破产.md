<p>提及智能手机，如今大家的第一反应显然是搭载 iOS 或 Android 系统的机型，而这自然是当下智能手机市场由这两大阵容瓜分的现状所塑造出来的结果。事实上 iOS 与 Android 崛起的这段时间，同时也几乎是其他移动操作系统溃败的十余年。日前，据海外媒体 omgubuntu 的报道显示，首家 Ubuntu 手机制造商 Bq 已破产。</p><p></p><div 推出了首款 Ubuntu 手机 Aquaris E4.5 Ubuntu Edition，所搭载的是 Ubuntu Touch 系统。彼时，在这款 Ubuntu 手机问世后，Ubuntu 团队甚至曾预告了国内手机厂商魅族将发布 Ubuntu Phone 的消息。<p></p><p>然而在经历了上市初期一机难求的情况后，Bq 的 Aquaris E4.5 Ubuntu Edition 很快就陷入了困境。之后，魅族的 Ubuntu Phone 也只有海外版的 MX4 与 PRO 5 两款机型，Bq 本身的 Ubuntu 手机项目更是在 2017 年被 Canonical 放弃。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202109/613232d18e9f096094145352_1024.jpg" data-height="320" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/613232d18e9f096094145352_1024.jpg" referrerpolicy="no-referrer"></div></div>接下来在 2018 年，当越南最大的私营企业 Vingroup 试图进入手机行业时，其收购了 Bq 的 51% 股份，与高通达成了专利许可协议，并与全球领先的计算摄影技术开拓者和解决方案供应商虹软达成合作。在这一通操作之下，找到了 " 金主 " 的 Bq 也开始负责生产 Vsmart 品牌手机。<p></p><p>然而遗憾的是，在经历了 3 年的尝试后，Vingroup 认为自己不会在这一领域取得成功，选择退出移动设备的生产，因此 Bq 也随之被放弃，甚至 Bq 的官网已经已不再提供服务，也使得 Ubuntu 手机成为了过去式。</p><p>作为 Linux 操作系统最为成功的发行版，Ubuntu 如今在全球拥有过亿的用户量，以及一个相当成熟的开发者社区，并且背后的 Canonical 公司还有一个富豪老板马克 · 沙特尔沃思（Mark Shuttleworth）。既有钱、又有技术，还有种子用户的情况下，Ubuntu 在移动端即便不说是 " 梦幻开局 "，但显然也绝非是困难。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" 也更有前瞻性。在开发团队最初的计划表中，Ubuntu Touch 和 Android 采用了相同的内核，但在界面、底层逻辑、软件商店上则都使用的是 Ubuntu 的技术和美术风格，同时还支持 ARM 和 X86 两大处理器架构，通过数据连接、同步和共享实现融合。而这一理念，实际上就是微软 Windows 10/11 的同款。<p></p><p>事实上在许多业内人士看来，Ubuntu 手机之所以会失败，当初 Aquaris E4.5 Ubuntu Edition 在宣传上过于夸大可能是其中一个非常重要的原因。硬件配置上，Aquarius E4.5 采用了一块 4.5 英寸的 960×540 分辨率屏幕，搭载 CPU 主频 1.3GHz 的联发科 MT6582 主控，以及 1GB RAM+ 和 8GB ROM 存储组合，配备前置 500 万像素摄像头以及 1300 万像素后摄，但其售价却是 169.9 欧元，这样的配置配合并不算低的价格所造成的结果，就是性价比较低。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" 手机的受众群体，也就是以及追求新鲜感的极客用户来说或许还不是什么大事。其更大的问题在于，研发团队在开发 Ubuntu Touch 时候过于注重融合，却忽视了基础体验。彼时，在社区内用户反馈地很多的问题如今看起来甚至都很令人哭笑不得，例如电池指示器数据和实际电量不匹配、无法在国内漫游、定位服务不准确、蓝牙只支持音频设备、UI 挡住了拨号键、移动数据不稳定导致不能上网等等，甚至于一些漏洞在修复后，经过数次 OTA 升级居然还会重新出现。<p></p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " Aquaris E4.5 Ubuntu Edition 的出师不利，也直接影响到了后续一众 Ubuntu Touch 设备。因为 Canonical 管理层曾表示，Ubuntu Touch 项目只有从移动市场拿下大约 1% 的份额才能持续生存，也就是每年要销售大约 1100 万台 Ubuntu 手机。<p></p><p>而除了销量不足以支撑外，Ubuntu Touch 还遇上了 Windows Phone 同样的困境，也就是愿意为 Ubuntu Touch 开发应用的开发者数量不足，以至于消费者没有太多的应用能够使用。不过当时 Windows Phone 没能成功，是因为彼时 Android 和 iOS 已经成势，而 Ubuntu Touch 则是 Canonical 本身的体制问题。据悉，Canonical 可以称得上是远程办公的先行者，其雇员分布在全球各地，但在不同时区工作也导致遇到问题的开发者，经常会出现找不到具体负责人的情况。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 "
