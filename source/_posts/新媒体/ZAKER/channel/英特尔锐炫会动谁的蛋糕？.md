maomaobear<p></p></strong><p><strong>图 /IC Photo</strong></p><p>2021 年 8 月份，英特尔架构日活动中，英特尔正式对外公布了锐炫 ( Intel Arc ) 系列独立显卡，基于 Xe HPG 架构，由台积电 N6 制程工艺代工生产，支持 DX12 Ultimate、硬件光线追踪、XeSS AI 超采样等先进技术。将于 2022 年的 CES 展会后正式发布。</p><p>根据英特尔的官方路线图显示，锐炫 ( Intel Arc ) 系列独立显卡的开发代号为 Alchemist，后续还有 Battlemage、Celestial、Druid 等多款新品。</p><p>根据爆料，满血版 DG2 ( Alchemist ) 大约与 NVIDIA RTX 3070 Ti 性能相当。</p><p>不过，按照 nVIDIA 的计划，到了 2022 年三季度，nVIDIA 的下一代显卡就会发布。英特尔面对是不是 30 系列，而是 40 系列。Alchemist 满血也只能与 3070 抗衡，性能是落后。</p><p>那么，靠这个性能，英特尔锐炫能有前途吗？它会动 nVIDIA 和 AMD 的蛋糕吗 ?</p><p><strong>英特尔的第一次尝试</strong></p><p>其实，英特尔进入显卡市场非常非常早。电脑从字符界面进入图形界面以后，就有显卡需求。从字符界面到 3D 界面进化的时间也非常快。几年时间，在显卡市场就出现了百家争鸣的局面。nVIDIA、ATI、Voodoo、S3、PowerVR……，都在这个时代竞争。</p><p>英特尔当时没有图形技术，所以它购买了 Real 3D 的技术，这家公司当时被军火厂商洛克希德马丁控股，3D 技术是给军用模拟现实的。这就是英特尔的第一代独立显卡 i740。</p><p>当年的显卡号称 6 个月淘汰一代，与现在 2 年一代的节奏差距巨大。英特尔根本跟不上节奏，独立显卡只做了一代就不做了。</p><p>这个 i740 后来被集成到主板的 810 芯片组，变成 i752，英特尔也就在独立显卡市场销声匿迹了。</p><p>因为，除了游戏之外，大多数电脑没有 3D 性能的需求，所以英特尔把这个市场放给 nVIDIA 和 AMD，但是把芯片组市场牢牢把握。用集成显卡事实占据了大量市场份额。</p><p><strong>英特尔的第二次尝试</strong></p><p>这个局面持续了几年，一直到 nVIDIA 在 3D 显卡上发展出通用计算。</p><p>当时，在 PS3 上，IBM 和索尼合作，搞了异构处理器。把 CPU 和简单计算的算力单元分开，以编程难度为代价提升性能。</p><p>这个思路到了 GPU 上，nVIDIA 直接让 GPU 去做通用计算，进入高性能计算市场。</p><p>这就动了英特尔的奶酪，英特尔随机第二次进军显卡。这次是以高性能计算为目标，兼顾 3D 显卡应用。</p><p>英特尔的 Larrabee 堆了奔腾处理器核心，原本是做显卡用了的，后来发现性能没啥优势，就做加速计算卡了，所谓众核架构。</p><p>英特尔在这上面推出过几代产品，中国的超级计算机天河也用过英特尔众核架构的产品。</p><p>但是，这个时候 nVIDIA 推出的 CUDA 已经非常流行，nVIDIA 可以一块芯片多用，做 GPU，做通用计算都行。成本好控制，英特尔则是专用的芯片，而且高性能计算市场极小，成本很高。</p><p>这样，英特尔无法与 nVIDIA 竞争，所以很快就放弃了，这是英特尔的第二次尝试。</p><p><strong>人工智能的威胁</strong></p><p>本来高性能计算市场是一个小众市场，英特尔竞争不过也就罢了。但是最近几年人工智能崛起了。人工智能也是一个需要大量算力的需求，人工智能的崛起，带来了 nVIDIA 的春天，这个市场比高性能计算更大，而英特尔的高性能处理器，只能做控制节点用。</p><p>一个高性能计算单元，两个英特尔的 CPU，8 块 nVIDIA 的计算卡，英特尔被边缘化了。而且英特尔这个控制器，换 AMD 也行，nVIDIA 用 ARM 处理器也行。</p><p>这样一来，英特尔必须做显卡了。只有做显卡，才能平摊芯片设计和制造成本，在高性能计算上与 nVIDAI 竞争。</p><p>所以，英特尔开始第三次做显卡，这就是锐炫。英特尔希望，自己的显卡有量，能拉低成本，让自己的计算卡与 nVIDIA 竞争。不被边缘化。</p><p>是人工智能的崛起，让英特尔第三次尝试做显卡。</p><p><strong>锐炫能成功吗 ?</strong></p><p>从目前的数据看，在性能上，英特尔还不如 nVIDA，可能会落后一代。</p><p>但是英特尔的优势在于能集成。</p><p>英特尔只需要选出甜点的位置。一个是 1060，1070 这个位置，目标是 1080p 下面，能要 xbox one x，ps4pro 上跨平台的游戏。这个性能作为中端产品去集成。</p><p>还有一个是 3060，2080 这个位置，对应的是 PS5 上的跨平台游戏，能上 4k 分辨率。这个集成到高端产品。</p><p>产品可以游戏本为主，加强散热与功耗。做成台式机替代的游戏本，也可以做成 NUC，外接显示器或者大屏幕电视。但是英特尔不给用户插显卡的机会，用户买了 nVIDIA 和 AMD 显卡没有地方插。</p><p>作为后来者，英特尔一定要开放，用户算 AI，挖矿，算双精度浮点都行。整机可以卖便宜点，单独的 CPU 可以卖贵点，让游戏用户尽可能选英特尔的一体化方案。</p><p>这样普及化，英特尔的 GPU 成本可以降低。英特尔又有攒芯片的技术，可以把几个小核心攒成大的，互联性能损失不太大。利用成本优势，造独立大显卡，给用户挖矿，算 AI，科学计算跑超算。这样同等算力，英特尔会比 NVIDIA 便宜。</p><p>GPU 性能不断进步，但是 PC 游戏盗版严重，主流游戏厂商要靠游戏机市场活着，不能让游戏机跑不动新游戏。</p><p>所以与游戏机性能类似的显卡，可以玩很长时间的跨平台游戏。英特尔只要把握住节点，用集成和廉价去竞争，是很有机会，在 GPU 市场战胜 nVIDIA 和 AMD 的。</p><p>而 GPU 市场规模上去，计算卡也就便宜了，英特尔也会撼动 nVIDIA 的通用计算的市场。</p><div
