<p>电脑用户应该都用过 IE 浏览器，不过这个始于 Win95 时代的浏览器在 27 年后还是要被放弃了，微软现在的重点放在了 Edge 浏览器上，不仅技术更新，而且性能更好，CPU 及内存占用大幅下降。</p><p>因为 Edge 浏览器采用的是 Chromium 内核，所以和 Chrome 浏览器关系密切，所以两者相互借鉴也是正常的。之前 Chrome 就根据 Edge 的 UI 设计，完成了对下载功能的优化更新。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div"><img class="lazy opacity_0 zaker_gif_cache" id="img_0" data-original="http://zkres2.myzaker.com/202209/632995738e9f090afb7eef79_1024.jpg" data-gif-url="http://zkres2.myzaker.com/202209/632995738e9f090afb7eef79_raw.gif" data-height="493" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202209/632995738e9f090afb7eef79_1024.jpg" referrerpolicy="no-referrer"></div></div>近日，谷歌在 Chrome 的开发版中，加入了与 Edge 类似的侧边栏搜索功能，该功能允许突出显示网页中的任何文本，以便在网络上查找有关它的更多信息，且无需打开新标签。通过这一功能搜索到的结果将会显示在网页右侧的窗口中。<p></p><p>在实际操作上，CHrome 的侧边栏搜索功能与 Edge 的如出一辙，甚至连操作逻辑都几乎一样，只需用户选中文本，然后在右键菜单中选中 " 在 Google 上搜索 " 即可。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202209/632995738e9f090afb7eef7a_1024.jpg" data-height="347" data-width="621" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202209/632995738e9f090afb7eef7a_1024.jpg" referrerpolicy="no-referrer"></div></div>不过，该功能并未默认开启，想要使用需要先在 Chrome Canary 中输入 "Chrome://flags"，找到 Search web in side panel 项，启动后重启。<p></p><p>编辑：熊乐</p><p>· END<strong>·</strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
