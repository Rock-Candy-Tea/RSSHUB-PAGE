<p>长久以来，由于自身的无形和非排他性，数字艺术作品始终是被轻视的——直到 NFT 的出现。我们所习惯的是，所有人都可以截屏、右键保存或复制数字艺术到自己的设备上，然后得到与原作一模一样的作品。</p><p>但现在，以区块链、智能合约和加密货币为底层技术的 NFT（Non-fungible token，非同质化代币） 可以实现对数字艺术作品的确权与保护。</p><p>价值通常是人类对事物的赋予，而非存在于事物本身。挂在博物馆墙上或线下展览中有形的艺术作品，由于其明显的存在性、独特性和原创性的证明，具有很高的价值。即使《蒙娜丽莎》有过很多复制品和伪造品，也并不会影响悬挂在卢浮宫的达芬奇原作的价值。</p><p></p><div class="img_box" id="id_imagebox_0" 的独占性、不可替代性、不可伪造性是对艺术作品的 「唯一真实 」数字副本的认证。因此，有了 NFT 作为 「信任标志」，参与 NFT 购买交易的各方可以对作品的真伪放心，即使在与身份匿名的在线用户交易时也是如此——背后的区块链和智能合约技术确保了安全交易和资产的可追溯性。</p><p>而其实 NFT 的用途远不止能让我们在互联网上拥有一张 JPEG，它能对数字艺术作品进行确权，理论上也可以和实体产业相结合，对其他事物进行确权。</p><p>万物皆可 NFT？</p><p>想象这样一个世界：你平生最珍贵的物质财产——车、服饰、艺术藏品，将真正永远地属于你。当未来的考古学家发掘出这些「历史文物」，其中的数据会被破译，他们能追溯到你的所有权，再追溯到你生命所在的那段时间。</p><p>这就是区块链对 NFT 等数字资产的承诺，NFT 不只代表着「数字藏品」，还允许一切数字资产的代币化和商业化，其形式包括捕捉的瞬间、游戏、收藏品、交易卡、音乐、备忘录、电子门票、钱包和域名，甚至是「元宇宙」上的财产。比方说，在一些很 in 的活动上，NFT 可以充当电子门票并捆绑一些线下服务。</p><p>今年的 Coachella 音乐节就是一个例子：所有持票参与的人都可以获得一个免费的 In Bloom NFT，它是一颗虚拟花种子，种子会在每周音乐节开始之前开放，随机开出六种花之中一种，分别代表 VIP 休息室、快速通道、摩天轮门票、周边礼品等现实权益，持有人可在音乐节当天兑换其相应的线下升级权益。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div 也是一门生意。最近，一家成立不久的 NFT 公司 Americana Technologies 宣布筹集了 690 万美元的种子资金，由 Reddit 的创始人 Alexis Ohanian 的风险投资公司 776 牵头，知名 NFT 交易平台 OpenSea 也参与其中。</p><p>这家公司由一位名叫 Jake Frey 的设计师于 2021 年初创立，致力于让品牌和创作者把实物商品（服饰、汽车和艺术收藏品）等转化为 NFT。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" 官网首页｜Americana.io<p></p><p>它的核心产品是一种通用芯片，可以将芯片嵌入到任何物品中，再将其链接到区块链（目前仅支持以太坊区块链），从而附加一个数字分类账，提供真实性证明、所有权证明和整个所有权历史。</p><p>「Americana 是 Web3 的下一步，」Reddit 的创始人、Americana 的投资人 Ohanian 说，「物理世界中的事物的来源是纷乱而庞杂的，Americana 创立后，有史以来第一次，大大小小的品牌都可以将他们的认证产品上链，甚至在该产品每次转售时都可以获得版税。」</p><p>智能资产：NFT 对实物资产的优势</p><p>实物资产是否有必要转化为 NFT？比较有说服力的观点是，这为交易实物资产提供了许多优势，可以总结概括为以下几点：</p><p><strong>证明出处</strong></p><p>通过解决资产来源的不确定性，NFT 实际上可能会提高资产的价值。</p><p><strong>安全性和不变性</strong></p><p>由于 NFT 使用 DLT（Distributed ledger technology 分散式账本技术），交易记录存储在不可变的分布式分类账中，提供资产所有权和转移的防篡改记录。NFT 持有者仍需要确保他们的私钥足够安全。但是，与单个集中式资产注册表相比，DLT 和密码学的使用提供了额外的安全保护。</p><p><strong>权力下放和去中介化</strong></p><p>传统的收藏品买卖方式往往高度集中。拍卖行可以对奢侈品的拍卖收取溢价，因为它们的作用是验证物品的出处，并将潜在的买家和卖家聚集在一起进行集中拍卖。相比之下，NFT 可以提供一种去中心化的方式来证明出处，同时还支持跨去中心化网络的交易。</p><p><strong>智能合约集成</strong></p><p>NFT 可以嵌入智能合约中，以提供额外的实用程序和自动化交易。例如，一旦买方提供了必要的 KYC（Know your customer 了解你的客户） 文件，智能合约可以提供实物资产 NFT 的转移，这有助于降低交易风险和对中介的需求。</p><p><strong>二级交易货币化</strong></p><p>还可以对智能合约进行编程，以便在二次销售时支付特许权使用费。例如，可以对代表稀有雕塑所有权的 NFT 进行编程，以允许原始雕塑家在每次出售雕塑时获得一定版税。</p><p><strong>创造性的可能性</strong></p><p>除了代表所有权之外，NFT 还可以让持有者以其他方式与物品进行交互，例如在代币本身中嵌入额外的数字资产或充当独家内容或活动的门票。因此，对于创作者来说，NFT 可以作为分发更多内容的工具。</p><p><strong>可访问性和包容性的提高</strong></p><p>NFT 可以开辟新的所有权和资产参与的途径，否则个体收集者可能无法获得这些资产。这可能是通过让藏书者拥有一页稀有的第一版，或者实现跨大陆和社会鸿沟的资产流动性交易。</p><p>增长和灰色地带</p><p>数字市场行业最近的一个趋势是「生态系统」兴起。意思是一个生态系统可以将多个数字平台集成到一个服务中，提供其内部开发的相互连接的数字工具和平台。</p><p>例如，加密货币交易所 Binance （币安）已经将其业务扩展到 NFT 市场、区块链技术、借记卡、游戏、教育和其他数字服务。Binance 还发行了 Binance Coin，作为一种功能性代币（Utility Token），在 Binance 生态系统中使用它时有所优惠，比如支付的交易费用较少。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" data-original="http://zkres1.myzaker.com/202206/6298bcfd8e9f093f0a14445f_1024.jpg" data-height="607" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202206/6298bcfd8e9f093f0a14445f_1024.jpg" referrerpolicy="no-referrer"></div></div>Binance 生态系统｜Binance<p></p><p>然而，NFT 的新颖性和颠覆性也在各个行业中构成了极大的不确定性。虽然物权法管辖土地和房屋等有形财产，而知识产权法管辖艺术创作等无形财产，但 NFT 的法律地位仍然存在争议。</p><p>收藏家可以「拥有」一件实物艺术品，如画作或雕塑。然而，物理对象的这种所有权并不自动产生艺术品上的任何知识产权。对于收藏家拥有艺术品的任何知识产权（比如以商业方式复制、改编或公开展示作品的权利），艺术家必须通过合同（比如通过许可协议）来授予收藏家这种权利。</p><p>相反，数字艺术则是完全无形的性质，本质上是一种数据形式，而 NFT 代表区块链上的特定数据。针对这个特点，也有有人认为：数据作为自由流动的信息，不能受版权保护或被盗用。这就好比一种方式、食谱、技术或一条信息无法获得版权保护一样。NFT 及其嵌入的数字艺术作品由于其无形性质而根本不能「购买」，并且所涉及的交易纯粹是 IP 许可性质，其中 NFT 的目的类似于收据。因此，NFT 的「购买」位于所有权转让和有限许可授予之间的灰色地带。</p><p>此外，如果没有适当的平衡来保障公平竞争，自由贸易可能弊大于利。加密货币、区块链和 NFT 技术都以去中心化金融（DeFi）为中心，因此这种技术的监管对于防止滥用和保护消费者权益至关重要。于是人们对现有法律机制是否充分的问题感到担忧。</p><p>智能合约达成的交易是否有效且可执行？加密货币可以作为法定货币接受吗？NFT 是否作为证券进行监管？区块链上的交易记录可以作为可靠的证据吗？如果一件艺术品被盗并在 NFT 市场上市会怎样？</p><p>在当前的大数据和数字平台时代，NFT 确实为企业带来了革命性的发展潜力。但在 NFT 交易中转移了哪些合法权利？NFT 持有者可以利用他们购买的 NFT 是否有限制？传统企业如何将 NFT 作为其业务的一部分？NFT 如何助力企业数字化转型？</p><p>这些问题的答案将会在 NFT 未来更长远的发展中逐渐浮出水面。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
