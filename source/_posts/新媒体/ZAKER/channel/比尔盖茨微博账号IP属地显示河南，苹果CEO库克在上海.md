<p>IT 之家 5 月 2 日消息，近期微博、抖音、知乎、微信公众号等平台纷纷宣布开启或已经开始 IP 属地展示功能。并且账号 IP 属地以运营商提供信息为准，用户暂时无法主动开启或关闭相关展示。</p><p>公告指出，近期，国内国际热点事件频发，引发网友高度关注。通过用户投诉，其发现存在个别用户冒充热点事件当事人，编造、传播不实信息的情况，在网络空间造成了不良影响。为维护网络传播秩序，进一步打击仿冒搬运、造谣传谣等行为，平台将显示用户发布内容时的 IP 属地。</p><p>不过 IP 属地上线后，一些博主发现平日里关注的一些海外人物其实都在国内。比如微软联合创始人比尔盖茨微博账号 IP 属地显示为河南，特斯拉 CEO 马斯克账号 IP 属地为北京，苹果 CEO 蒂姆库克 IP 属地在上海。很显然，平日里发布的微博可能并非是本人操作，而是由其他工作人员代理发布。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202205/626f2eec8e9f096fa728d72c_1024.jpg" data-height="1334" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/626f2eec8e9f096fa728d72c_1024.jpg" referrerpolicy="no-referrer"></div></div><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202205/626f2eec8e9f096fa728d72d_1024.jpg" data-height="1334" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/626f2eec8e9f096fa728d72d_1024.jpg" referrerpolicy="no-referrer"></div></div><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202205/626f2eec8e9f096fa728d72e_1024.jpg" data-height="1334" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/626f2eec8e9f096fa728d72e_1024.jpg" referrerpolicy="no-referrer"></div></div>近期，@微博管理员 发布 "IP 属地功能升级公告 "，宣布全量开放评论展示发评 IP 属地小尾巴功能，同时上线 " 个人主页一级页面展示 IP 属地 " 功能。官方表示，为减少冒充热点事件当事人、恶意造谣、蹭流量等不良行为，确保传播内容的真实、透明，站方于今年 3 月上线展示用户 "IP 属地 " 功能。该功能旨在更透明地向广大用户呈现真实情况，无法由用户主动开启或关闭。<p></p><p>《微博 IP 属地功能大升级：全量强制开放评论展示小尾巴，上线个人主页一级页面展示》</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
