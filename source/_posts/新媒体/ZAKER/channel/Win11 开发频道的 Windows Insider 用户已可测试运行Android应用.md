<p>IT 之家 11 月 5 日消息，微软官方 Windows Insider 账号于当地时间 11 月 5 日在 Twitter 上宣布，现阶段 Windows 开发频道的 Windows Insider 用户也可在 Windows 11 系统上测试运行 Android 应用。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202111/6184a5f28e9f090a94343781_1024.jpg" data-height="811" data-width="648" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/6184a5f28e9f090a94343781_1024.jpg" referrerpolicy="no-referrer"></div></div>微软此前只对 Windows Beta 版频道的美国 Windows Insider 用户推送了 Windows SubSystem for Android（WSA）子模块，现在终于也对 Windows 开发频道的 Windows Insider 用户开放了这一新特性。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202111/6184a5f28e9f090a94343782_1024.jpg" data-height="610" data-width="909" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/6184a5f28e9f090a94343782_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家了解到，Windows 开发频道的 Windows Insider 用户现已可在微软应用商店中查找并安装自己喜爱的 Android 应用，并通过新特性 Snap Layouts 运行 Android 应用。用户也可以将安装的 Android 应用固定在开始菜单或任务栏中，可以使用 Windows 传统交互模式和 Alt + Tab 等多任务切换键对已安装的 Android 应用进行管理和操作，并使用 Windows 原生的剪贴板功能在 Android 应用和 Windows 11 系统中进行复制粘贴，用户也可以在系统通知中查看来自 Android 应用的通知。<p></p><p>微软方面之前也表示现阶段正在与亚马逊积极合作，为广大用户提供更多可以直接运行在 Windows 11 上的 Android 应用，并会对用户的反馈的问题加以改进。</p><p>《微软 Win11 安卓子系统已支持运行 IT 之家、支付宝等 APK 应用（附手动侧载安装教程攻略）》</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
