<p>靴子落地。</p><p>在影驰官网，名为 RTX 3080 黑将 [ FG ] 、RTX 3080 黑将 OC [ RG ] 、RTX 3070 黑将 [ FG ] 的显卡新鲜上架。</p><p>后缀的 [ FG ] 是何意思？</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202105/609759a0b15ec064102e2277_1024.jpg" data-height="400" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202105/609759a0b15ec064102e2277_1024.jpg" referrerpolicy="no-referrer"></div></div>产品规格页面显示，正如传言，<strong>它们换装了 LHR（Lower Hash Rate）核心，也就是从 GPU 层面对挖矿做了限制，其中 RTX 3080 是 GA102-202 ( LHR ) ，RTX 3070 是 GA104-302 ( LHR ) 。</strong><p></p><p>影驰介绍，RTX 3070 显卡具有限制挖矿性能的特性 ( 较低的哈希率 ) ，以太坊算力约 25MH/s，RTX 3080 显卡具有限制挖矿性能的特性 ( 较低的哈希率 ) ，以太坊算力约 43MH/s。</p><p>当然，RTX 3080/3070 只是个开始，预计 RTX 3060/3060 Ti 等也将陆续完成替换，5 月底发布的 RTX 3080 Ti/3070 Ti 也自不用说。</p><p>不过，就目前的消息来看，RTX 3090 似乎会是个例外。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202105/609759a0b15ec064102e2278_1024.jpg" data-height="512" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202105/609759a0b15ec064102e2278_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
