<p>ZAKER 科技 6 月 8 日消息，当地时间周二，欧盟达成一项协议，规定包括手机和平板在内的电子产品都必须采用标准化的充电接口。 欧盟内部市场和消费者委员会（IMCO）宣布，在该地区销售的所有智能手机、平板电脑、电子阅读器、数码相机等电子产品必须使用 USB-C 接口（Type-C 接口）。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202206/62a000cc8e9f09735959bf32_1024.jpg" data-height="450" data-width="800" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202206/62a000cc8e9f09735959bf32_1024.jpg" referrerpolicy="no-referrer"></div></div>规定预计在 2024 年秋天实施，距今至少还有两年缓冲时间，各大厂商有着非常充分的时间可以调整，仔细看一下协议可以发现，欧盟要求所有厂商统一 " 充电器端 Type - C 接口 "，而不是 " 手机端 Type - C 接口 "，实际上苹果自 iPhone12 系列附带的数据线已经符合标准，目前在售的大多数充电器也符合欧盟的规定。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202206/62a001218e9f0944e74377f9_1024.jpg" data-height="674" data-width="1257" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202206/62a001218e9f0944e74377f9_1024.jpg" referrerpolicy="no-referrer"></div></div>不过也有业内人士认为，欧盟的协议未来可能存在补充条款，要求各大厂商同样支持手机端 Type - C 接口，未来数据线必须 C 转 C 模式。虽然苹果暂时逃脱了限制，未来还存在诸多不确定性。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
