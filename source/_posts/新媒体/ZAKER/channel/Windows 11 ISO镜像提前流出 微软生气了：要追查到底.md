<p>让微软没有想到的是，6 月 24 日才会公布的内容，新 Windows 居然提前这么久才公布，而且相关镜像已经公开被用户下载，这让他们很生气。</p><p><strong>据外媒报道称，针对一些公共论坛上泄露所谓的 Windows 11 ISO 镜像系统行为，微软已经非常生气了，并表示要追查到底。</strong></p><p>从微软目前的态度来看，6 月 24 日发布的新 Windows 系统，应该就是目前已经泄露的 Windows 11 了，至于为什么会突然被泄露，目前还不知情。</p><p>另外，微软已经否认 Windows 11 ISO 镜像是其故意提前泄露的。</p><p>之前有消息称，Windows 10 系统用户会免费升级 Windows 11，而 Windows 7、8.1 用户呢？</p><p><strong>如果你也担心这个问题，那么大可不必，从外媒 XDA 最新的报道来看，Windows 11 已支持了 Windows 7 和 Windows8.1 的产品密匙，Win8 的用户需升级到 Win8.1 之后支持。</strong></p><p>这个举措也表明，Windows 11 应该会跟之前的策略一样，让上述系统用户免费升级。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202106/60ca913f8e9f090a70068cbd_1024.jpg" data-height="400" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202106/60ca913f8e9f090a70068cbd_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
