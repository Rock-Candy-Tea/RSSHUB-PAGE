<p>IT 之家 8 月 23 日消息，8 月 18 日，微软面向 Dev 频道用户推送了 Windows 11 Build 25182 版本。根据 PhantomOcean3 的最新发现，该版本有一个隐藏选项，可以加快系统任务栏的加载速度。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202208/6304f3098e9f094ac74d4ece_1024.jpg" data-height="729" data-width="733" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202208/6304f3098e9f094ac74d4ece_1024.jpg" referrerpolicy="no-referrer"></div></div>从下面的演示可以看到，使用 Vive Tool 开启 id:39751186 的选项后，任务栏的加载动画明显变短了。该功能仍在测试阶段，所以 IT 之家不建议大家在主力机上尝试使用。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202208/6304f3098e9f094ac74d4ecf_1024.jpg" data-height="360" data-width="640" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202208/6304f3098e9f094ac74d4ecf_1024.jpg" referrerpolicy="no-referrer"></div></div>开启方法：<p></p><p>从其 GitHub 存储库下载 Vive Tool</p><p>提取 Vive Tool 文件到某个目录</p><p>以管理员身份启动 Windows 终端</p><p>使用 CD 命令导航到 Vive Tool 的文件夹。例如，CD C:Vive。</p><p>输入 vivetool /enable/id:39751186 并按回车。</p><p>重新启动 Windows 资源管理器</p><p>如果要恢复更改，可以使用命令 vivetool /disable/id:39751186。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
