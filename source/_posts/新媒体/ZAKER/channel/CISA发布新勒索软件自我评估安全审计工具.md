<p>据外媒报道，<strong>美国网络安全与基础设施安全局 ( CISA ) 发布了勒索软件就绪评估 ( RRA ) ，这是其网络安全评估工具 ( CSET ) 的新模块。</strong>RRA 是一种安全审计自我评估工具，适用于那些希望更好地了解自己在防范和恢复针对其信息技术 ( IT ) 、操作技术 ( OT ) 或工业控制系统 ( ICS ) 资产的勒索软件攻击方面做得如何的组织。</p><p>CSET 模块则是量身定制的 RRA，用于评估不同级别的勒索软件威胁准备，无论组织的网络安全成熟度都能为其提供帮助。</p><p>CISA 在该工具的维基页面上写道："RRA 还提供了一个清晰的改进路径，它包含了一个由基础、中级和高级类别组成的不断演进的问题。"</p><p>CISA 表示，RRA 可用来抵御日益增长的威胁，因为它在以下这几方面都有效：</p><p>帮助组织根据公认的标准和最佳实践建议病以系统性的、有纪律的和可重复的方式评估其有关勒索软件的网络安全状况；</p><p>指导资产所有者和运营商通过一个系统过程来评估他们的运营技术 ( OT ) 和信息技术 ( IT ) 网络安全实践以应对勒索软件的威胁；</p><p>提供带有图表和表格的分析仪表板，这些图表和表格以摘要和详细的形式显示评估结果。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202107/60de7f7b8e9f0930e22c5c1b_1024.jpg" data-height="1092" data-width="2538" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60de7f7b8e9f0930e22c5c1b_1024.jpg" referrerpolicy="no-referrer"></div></div><strong>如何使用 RRA 安全审计工具</strong><p></p><p>要使用自我评估工具必须要先安装 CSET，然后：</p><p>登录或启动 CSET 应用；</p><p>开始新的评估；</p><p>在 Assessment Configuration 屏幕中选择 Maturity Model；</p><p>从 Maturity Model 屏幕中选择 Ransomware Readiness Assessment；</p><p>现在可以完成 RRA 评估了。查看教程以获得额外的指令或查看 Help 菜单中的 RRA 指南。</p><p>CISA 此前曾发布过 Aviary，这是一个用于审查 Microsoft Azure Active Directory ( AD ) 、Office 365 ( O365 ) 和 Microsoft 365 ( M365 ) 环境下遭到网络攻击后活动的工具。Aviary 的工作原理是分析使用 Sparrow 生成的数据输出。Sparrow 是一个基于 PowerShell 的工具，用于检测 Azure 和 Office 365 中的潜在漏洞应用和账号。</p><p>此外，CISA 还发布了 CHIRP（全称 CISA Hunt and Incident Response Program），这是一个基于 Python 的取证工具，可以检测 SolarWinds 黑客在 Windows 系统上的活动迹象。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
