<p>在 " 加量降价 " 的刺激之下，刚刚上市不久的 iPhone 13 系列也让一众消费者高呼 " 真香 "，在开售后不仅线上渠道被 " 挤爆 "，线下的 Apple Store 更是排起了长队。就在大家陆续收到 iPhone 13 系列机型后，一年一度针对新款 iPhone 的拆解、分析，也已陆续上线。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202110/615f180e8e9f093fa1077b0f_1024.jpg" Repair Guru 在拆解 iPhone 13 时发现，Face ID 信息很可能与屏幕绑定在了一起，并且如果将其更换为第三方屏幕，则将收到 Face ID 将停止工作的 "Face ID Is Not Available" 通知。为了进一步验证此时，这位 YouTuber 还将两台同型号 iPhone 13 的屏幕拆下来进行了对换，然而即便是换上原装屏幕，手机也依然会提示无法验证屏幕是否为正品，此时再进入设置清除 Face ID 信息重新录入后，便收到了 Face ID 已失效的通知。<p></p><p>而在他拔掉距离传感器的接口、只连接屏幕排线，并再次互换屏幕后发现，原装屏幕依然可以正常录入人脸信息并使用 Face ID 解锁，但非原装屏幕的 Face ID 则会停止工作。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202110/615f180e8e9f093fa1077b10_1024.jpg" referrerpolicy="no-referrer"></div></div>对于这一情况，iFixit 方面认为，苹果在 iPhone 13 系列上将 Face ID 与屏幕串行在一起，直接将元器件信息写入到了硬件层，让 Face ID 和屏幕形成了强关联性。这位 YouTuber 在视频中则猜测，苹果可能将 Face ID 的部分信息储存在了屏幕芯片中，这代表着在更换屏幕时可能要对原信息进行提取并转移，并且这一步势必需要专业的工具及软件才能完成。<p></p><p>但事实上，这并非苹果首次采取这一措施。此前在 iPhone 12 系列中，iFixit 发现如果消费者在非官方渠道更换了后摄模组，将会导致其无法使用超广角摄像头进行拍摄，并且将无法使用人像模式。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" 系列上，苹果方面则加入了一个验证机制，以便于用户判断自己使用的设备是否为原装屏幕，并建议用户不要自行或通过非官方授权渠道更换屏幕。甚至在更早的指纹解锁时代，更换屏幕导致 Touch ID 失灵现象也并不鲜见。<p></p><p>为什么苹果要做这样的决策呢？许多朋友的第一反应，可能会是苹果想要从中牟利。毕竟根据官方公布的维修价格显示，更换 iPhone 13 Pro Max 的屏幕需要 2559 元，定位最低的 iPhone 13 mini 也需要 1699 元，这样的价格也被外界认为是其赚取超额利润的源泉之一。但根据苹果方面的说法，" 自 2009 年以来，每年维修服务的成本都超过了其所产生的收入 "。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div referrerpolicy="no-referrer"></div></div>但售后不挣钱，甚至是一个需要花钱来支撑的部分，其实早已是消费电子行业的常态。作为一家以利润为核心追求的商业公司，售后维修如果真的无利可图，甚至让其背上包袱，那么正确的抉择一定是将售后服务外包，而事实上这也是苹果近年来一直在做的事情。<p></p><p>2019 年秋季，苹果启动了独立维修商计划，向正规的第三方维修商提供与授权服务提供商 ( ASP ) 相同的正品零部件、维修工具、诊断手册，以及必要的培训。而使用着官方零部件、遵循苹果标准诊断程序的独立维修，也就与生产 MFi 配件的制造厂商一样，都是苹果体系中的 " 雇佣兵 "，既让消费者有了更多的选择，但实质上苹果也依然把控着这一生态。</p><p></p><div class="img_box" 配件往往比没有通过这一认证的配件更贵一样。并且考虑到目前第三方维修商的技术水平水涨船高，按照正常的市场竞争逻辑，苹果独立维修商的市场竞争力不太够的情况下，势必会被价格优势打倒。<p></p><p>可这样一来，许多消费者还是会选择第三方维修，并导致苹果希望杜绝第三方维修的打算落空。因此为了给授权服务提供商以及独立维修商提供足够的客源，苹果将 Face ID 与屏幕进行绑定，且需要到官方渠道进行维修的做法，就是效仿当初用 " 可能不支持此配件 " 的提示，来力挺通过 MFi 认证的配件。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " 与原装屏幕绑定在一起后，就让想要换第三方屏幕的消费者要面临更换后 Face ID 变成摆设的代价，这自然就会让大多数消费者三思而后行。毕竟 iPhone 动辄 5000 元以上的售价，对于许多人来说并不算便宜。<p></p><p>还有观点认为，在给独立维修商一个稳定的客流外，推销自家 Apple Care+ 可能也是其中的原因之一。正常情况下，消费者选择第三方屏幕的原因，往往是因为官方的保外维修价格太高，而作为对比，数百元的第三方屏幕显然就便宜得多了。但在这种情况下，有了 Apple Care+ 作为托底的选择，用户就能以一个更为合理的价格获得原装配件，自然也就不会去选择第三方维修了。</p><p></p><div class="img_box" id="id_imagebox_6" iPhone 13 系列上将 Face ID 与原装屏幕牢牢地进行捆绑，也被认为是其 " 反维修权 " 战略的一环，毕竟只有千方百计的给第三方维修 " 添堵 "，才能继续维持现状。而对于用户来说，购买 Apple Care+ 来获得更加低廉的持有成本，其实远比寄希望于品质无法保证的第三方维修要好得多。毕竟苹果官方修不好是可以直接换新机的，可第三方维修一旦给修坏，接下来或许就是无尽的麻烦了。</p><p>【本文图片来自网络】</p><p><strong></strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
