<p>不知道有没有 Win 11 用户吐槽过任务栏的天气小部件，反正小编是看别人吐槽过，说它不仅难看，显得任务栏杂乱无章，位置也不准确，人明明在这头，天气预报的位置却在那头。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202208/62fa1e1fb15ec048975c1ef6_1024.jpg" data-height="324" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202208/62fa1e1fb15ec048975c1ef6_1024.jpg" referrerpolicy="no-referrer"></div></div>网友 @Florianb 甚至在推特发文，直言天气小部件几乎没用，因为它是用电脑所在的 IP 来估计所在位置，而天气应用程序是用设备的实际 GPS 来给出天气信息。<p></p><p>此前，微软高级产品经理 @Brandon LeBlanc 还亲自回应过 Win 11 天气读数存在 bug 的问题，并表示正在研究中。同时还补充说，当系统从睡眠状态恢复时，天气读数可能会出现短暂的暂停或冻结现象。</p><p>看上去这个天气小部件很不靠谱嘛 !</p><p>不过，值得一提的是，天气小部件最近有了一些升级。</p><p>近日，@Florianb 发文，最近有什么新改进吗？现在的天气小部件显示的位置正确了。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202208/62fa1e1fb15ec048975c1ef7_1024.jpg" data-height="576" data-width="539" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202208/62fa1e1fb15ec048975c1ef7_1024.jpg" referrerpolicy="no-referrer"></div></div>这一疑问得到了 Windows 工程团队的 Jen Gentleman 的肯定回复，他表示，微软最近一直在做改进小部件位置检测的工作，这也是小位置图标有时候会说小部件正在使用您的位置的原因。<p></p><p>也就是说，现在使用天气小部件，有时通知区域中会显示位置图标，而这正是因为 Windows Widgets 正在使用电脑的精确位置，从而提供更准确的天气预报。</p><p>有网友得知消息后去证实，发现现在的位置确实是准确的。</p><p>当然了，位置准确本来就是天气预报应该具备的条件。但由于 GPS 定位需要 GPS 模块硬件，而大部分电脑是没有内置 GPS 定位模块的，能知道位置是联网后根据本机 IP 地址和周围 IP 地址比对后形成的大致位置坐标。</p><p>一般也只有专业的户外用的电脑才会标配有 GPS 模板，而且，相比手持 GPS 设备（比如车载 GPS），把定位软件塞进电脑里的工程量要大得多。因此，电脑的天气位置确定也就没有手机那么准确了。</p><p>一码归一码，还是有很多用户表示天气小部件非常烦人的，因为电脑并不会 24 小时开机，也不会时刻随身携带，大家会更习惯看手机上的天气 APP 一些。这样一来，天气小部件似乎就并没什么用处了。</p><p>不仅如此，除了显得任务栏丑和位置不准确之外，Win 11 天气小部件还有莫名其妙弹出来等缺点。是谁上网课准备开麦回答问题，但弹出来的却只有天气预报？是谁每次关机的时候却被提醒要先打开天气预报？是亲身经历过的用户们。</p><p></p><div class="img_box" id="id_imagebox_2" 部分下，将 " 小组件 " 关闭即可。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" referrerpolicy="no-referrer"></div></div>就是这么简单！<p></p><p><strong>卸载它</strong></p><p>天气小部件的图标虽然很好隐藏，但俗话说，" 斩草要除根。" 隐藏并不能从根本上解决问题，例如资源占用。只要它还在后台持续运营，那么将会一直占用资源，对于平时完全不用天气小部件的用户，可以选择把它卸载掉。</p><p>要卸载天气小部件，就要卸载 Win11 小组件，这就需要用到 CMD。对此，大家可以使用管理员权限打开 Windows Terminal 终端的 " 命令提示符 "，或者使用管理员权限运行 CMD，输入以下命令：</p><p>wingetuninstall MicrosoftWindows.Client.WebExperience_cw5n1h2txyewy</p><p>按下回车键，Win11 小组件就会被卸载掉了。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div 小组件，则可以执行以下命令：</p><p>winget install 9MSSGKG348SP</p><p>总而言之，天气小部件的实用性还是不高，而这也正是用户们不喜欢它的原因。由此可见，无论是什么功能，都应当对用户有用，且好用，否则也只是自讨没趣、白费功夫而已。</p><p>你会去关闭天气小部件吗？</p><div id="recommend_bottom"></div><div
