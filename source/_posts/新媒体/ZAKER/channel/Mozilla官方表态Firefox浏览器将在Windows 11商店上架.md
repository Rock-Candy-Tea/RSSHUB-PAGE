<p>Windows 11 的首次亮相也标志着彻底重构的微软商店的相应推出，其中最大的变化之一是微软如何处理对想要发布其应用程序的开发者的限制。更具体地说，微软不再阻止第三方商店在微软商店中上市，所以像 Epic Games Store 这样的 " 店中店 " 也可以下载了，这和苹果引发争议甚至官司的封闭政策完全不同。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202110/615dad3c8e9f0974905e959c_1024.jpg" data-height="406" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202110/615dad3c8e9f0974905e959c_1024.jpg" referrerpolicy="no-referrer"></div></div>同时，这家软件公司已经改变了关于浏览器引擎的政策，允许它们使用自己的引擎在微软商店发布。这意味着浏览器现在可以简单地在微软商店中提供下载，而不需要对驱动它们的技术进行任何改变。<p></p><p>而 Mozilla 是一直支持这种做法的组织之一，它对这一变化表示赞赏，并解释说 Firefox 也将在今年年底前在微软商店中提供。</p><p>微软已经放松了对其 Windows 商店的限制，这些限制实际上禁止了第三方浏览器进入商店。该公司说：" 我们多年来一直主张在 Windows 操作系统上为用户提供更多的选择和控制权。我们欢迎这一消息，即他们的商店现在对公司和应用程序更加开放，包括像 Firefox 这样的独立浏览器。我们相信，一个更健康的互联网是人们有机会从多样化的浏览器和浏览器引擎中进行选择。Firefox 浏览器将于今年晚些时候在 Windows 商店推出 "。</p><p>所有其他大型浏览器也有可能在微软商店上线，虽然 Google 还没有对这样的计划发表看法，但 Google Chrome 浏览器也有望出现在 Windows 11 用户面前 Vivaldi 和其他基于 Chromium 的浏览器也可能在微软商店发布。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
