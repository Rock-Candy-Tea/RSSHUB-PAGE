<p>IT 之家 8 月 30 日消息 外媒 Windows Latest 报道，微软 Windows 11 新系统将在今年晚些时候推出，微软已经确认，只有符合条件的设备才能通过 Windows Update 获得新的操作系统更新。如果你的 PC 设备与 Windows 11 不兼容，微软表示，你可以使用 Media Creation Tool 等通过 ISO 镜像安装运行新的操作系统，但官方不建议使用这一过程。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202108/612c3b008e9f0953044ea00e_1024.jpg" data-height="1104" data-width="1213" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202108/612c3b008e9f0953044ea00e_1024.jpg" referrerpolicy="no-referrer"></div></div>微软已经提供了哪些设备可升级获得 Windows 11 更新的具体信息。如果你的设备配备了英特尔第 8 代处理器、Ryzen 2000 和更新的设备，就被认为是支持的。更新的列表还包括对部分英特尔第 7 代处理器的支持，如酷睿 X 系列、至强 W 系列和酷睿 i7-7820HQ 芯片。<p></p><p>不幸的是，至少现在不会有更多的芯片加入到这个名单中。微软还确认，部分 Surface 产品，如 Surface Laptop 1 代不会升级到 Windows 11。微软官方对 Surface 产品也不会有任何特殊待遇，因为只有较新的机型才会被允许安装新的 Windows 11。</p><p>在问答环节，微软之前确认他们可以识别有硬件或软件兼容性问题的设备。</p><p>现在，你需要使用 PC 健康检查工具来检查与 Windows 11 的兼容性问题。然而，在 Release Preview 发布预览频道中，微软还增加了一个通知，将揭示更多情况。</p><p></p><div 做好准备。<p></p><p>" 好消息 —— 你的电脑符合 Windows 11 的最低系统要求。在我们为你准备好的过程中，何时提供的具体时间可能有所不同。注意：一些 Windows 10 的功能在 Windows 11 中是不可用的。某些应用程序和功能可能有额外的需求。" —— Windows Update。</p><p>它还链接到相关的支持文章，其中包括关于 Windows 11 要求和删除的功能的更多信息。</p><p>这是一个有用的步骤，特别是如果用户不熟悉 Windows 11 的配置要求。</p><p>微软目前正在向 Windows Insider 计划的发行预览频道的测试者推出上述兼容性信息。</p><p>当然微软很可能也在不符合要求的设备上也会出现类似的兼容性信息。微软试图使用遥测数据来识别兼容性问题，这个过程无法被欺骗。</p><p>IT 之家获悉，除了这些改进之外，微软还发布了 "PC 健康检查 " 的新版 Windows 11 兼容性检查工具，以更清楚地说明不兼容的设备。例如，该应用程序现在会提到，Secure Boot 安全启动被禁用或设备缺乏对 TPM 2.0 的支持。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
