<p>关注多项式方程的<strong>解之间的关系</strong>，而不是精确解本身，埃瓦里斯特 · 伽罗瓦（Évariste Galois）由此改变了现代数学的进程。</p><p></p><div class="zaker_div"><div id="zkvideo16285710161002" class="zk_upvideo" data-zkinfo="%7B%22status%22%3A%22finished%22%2C%22imgUrl%22%3A%22http%3A%2F%2Fzkres.myzaker.com%2Fimg_upload%2Feditor%2F2021%2F08%2F10%2F1628571014956_W1920H1080S.jpeg%22%2C%22videoUrl%22%3A%22http%3A%2F%2Fzkres.myzaker.com%2Fimg_upload%2Feditor%2Farticle_video%2F2021%2F08%2F10%2F16285710145923.mp4%22%2C%22name%22%3A%2216285710145923.mp4%22%2C%22width%22%3A1920%2C%22height%22%3A1080%7D"></div></div><p></p><p>埃瓦里斯特 · 伽罗瓦在 20 岁时的一次决斗中受了致命伤，而在此之前，他发现了多项式方程的隐藏结构。通过研究它们<strong>解之间的关系</strong>—— 而不是解本身 —— 他创造了新的概念，这些概念后来成为许多数学分支的重要组成部分。</p><p>没有人知道伽罗瓦为什么在 1832 年 5 月 30 日清晨来到巴黎的决斗场，但据说在决斗的前一天晚上，他熬夜完成了最后的手稿。在那里，他写道 :</p><p><strong>...... 寻找这些计算的根源 ! 这就是群的操作。由它们的复杂性而不是它们的外观来分类 !</strong></p><p><strong>我相信，这就是未来数学家的使命。这是我从事这项工作的方向 ......</strong></p><p>伽罗瓦的宣言是从数学困境中产生的。在 16 世纪，数学家们研究了 x2−2 和 x4−10x2+ 22 等多项式。他们试图找到某个<strong>简单的公式</strong>来计算这些多项式的根 —— 使方程等于 0 的 x 值 —— 但只能在<strong>最高次幂不大于 4</strong>的情况下找到它们。</p><p>除此之外，伽罗瓦自己证明了并不存在这样的公式。因此，他设计了一种研究根的新方法：<strong>他意识到可以研究它们之间的代数关系，而不用精确地计算它们</strong>。专注于它们的复杂性，而不是它们的外观。</p><p>在思想上，他的观点类似于考虑一个形状的不同对称性。这些是重新定位形状的方法，目的是在变换后使得图形看起来仍然相同（例如旋转一个正方形 180 度）。多项式根之间的对称性在于这样一种操作，根之间相互交换使它们保持相同的代数关系。</p><p>正如某些形状比其他形状对称性更高 ( 例如一个圆有无限多的对称轴；一个正方形则只有 8 个轴 ) ，你可以<strong>更自由地排列</strong>某些多项式方程的根。</p><p>斯坦福大学的 Brian Conrad 表示，“有些重新排列根的方法可能与代数规则不相容。从这个意义上说，某些根可能无法完全互换。”</p><p>在保持代数一致性的同时，根之间的可交换程度是一个微妙的性质，它告诉数学家们很多用于识别多项式的特征。而这些特征仅仅通过观察是看不到的。我们很最容易找到这样的例子。让我们来看两个多项式，每个都有三个根（因为每个的最高指数是 3）：</p><p>f ( x ) = x3 − 7x +5</p><p>g ( x ) = x3 − 7x +7</p><p>表面上看，它们几乎是一样的。但实际上，多项式其中一个的根可以比另一个的根以更多的方式重新排列。</p><p>我们先关注 f ( x ) ，假设这个多项式有三个根：a b c。我们可以用代数方法把它们组合起来得到一个新的值，方法是取每两个根的乘积，然后把它们相加。而对于所有三次多项式（即以 3 为的最高次幂，且立方项系数为 1 的多项式）而言，<strong>上述方法所得到的特殊的代数表达式总是等于线性项的系数</strong>。在我们的例子中，这一表达式等于 - 7。</p><p>我们得到这个代数方程：</p><p>ab + ac + bc = −7</p><p>现在让我们重新排列根，保持 c 不变，将 a 和 b 交换，我们得到 :</p><p>ba + bc + ac = −7</p><p>将根以这种方式重新排列保持了它们之间的代数关系：这个方程仍然成立。因为乘法和加法是交换的，这意味着交换顺序（比如移动了根），但不会改变答案。事实上，在这个例子中，共计六种重新排列根的方法 ( 包括它们不改变的方法 ) 都保留了这种关系 :</p><p>a, b, c: ab + ac + bc = −7</p><p>b, a, c: ba + bc + ac = −7</p><p>c, b, a: cb + ca + ba = −7</p><p>a, c, b: ac + ab + cb = −7</p><p>b, c, a: bc + ba + ca = −7</p><p>c, a, b: ca + cb + ab = −7</p><p>现在我们来看第二个多项式，g ( x ) = x3−7x + 7。如果我们把根记为 r、s 和 t，那么和 f ( x ) 的类似方程也成立 :</p><p>rs + rt + st = −7</p><p>这对于任何三次多项式都是成立的，第一项是 x3，线性项是 - 7x。同样，共计六种可能的排列均保持式子等于 - 7。但奇怪的是，对于 g ( x ) 而言，不是所有的组合都被认为是多项式的对称性。</p><p>这是因为它的根之间的代数关系更为复杂：它的根还满足另外一种特殊的代数关系。这种特殊关系式为： ( r−t ) ( r−s ) ( t−s ) = 7（这里我们假定 r<s, s<t）。下面这六个可能的组合式中这样三个能同时满足这两个代数关系：rs + rt +st="-7" 以及 ( r&minus;t ) r&minus;s t&minus;s></s,></p><p>r, s, t: ( r − t ) ( r − s ) ( t − s ) = 7</p><p><strong>s, r, t: ( s − t ) ( s − r ) ( t − r ) = −7</strong></p><p>t, s, r: ( t − r ) ( t − s ) ( r − s ) = −7</p><p>r, t, s: ( r − s ) ( r − t ) ( s − t ) = −7</p><p>s, t, r: ( s − r ) ( s − t ) ( r − t ) = 7</p><p>t, r, s: ( t − s ) ( t − r ) ( s − r ) = 7</p><p>加粗的三个组合式保留了所有根之间的代数关系。实际上不限于这两个例子。因此，<strong>这三个重排组合被认为是多项式的对称性</strong>。</p><p>这两个多项式具有不同程度的复杂性，这一点乍一看并不明显，但当你采用伽罗瓦发明的视角时，它就变得显而易见了。</p><p></p><div class="img_box" id="id_imagebox_0" referrerpolicy="no-referrer"></div></div>伽罗瓦发明的现在称为伽罗瓦群的数学客体比他短暂的生命更长久，并改变了数学的未来<p></p><p>伽罗瓦将他的思维方式运用到了后来被称为“伽罗瓦群”的新的数学对象中。这些对象标定了给定多项式的根之间的代数关系的复杂性。在这些关系中，根的重排方案可以一个接一个地被应用，<strong>但最终可以回到开始的地方</strong>—— 就像你可以对一个正方形进行对称变换，然后发现重复操作能刚好回到你开始的位置。</p><p>这个想法反映了数学中群的一般概念：<strong>群是一个包含对称操作的集合</strong>，无论这些对称操作适用于一个多项式的平方根还是根。伽罗瓦群是群概念的第一个例子，伽罗瓦的思想发展成为今天一个强大而普遍的研究领域 —— 群论。</p><p>伽罗瓦群为研究多项式方程提供了一个强有力的视角。如果你知道一个多项式的伽罗瓦群，那么它的根的行为就可以通过使用许多群论的工具来理解。通过这种方法获得的见解比通过对多项式本身进行代数运算得到的见解要有启发性。</p><p>宾夕法尼亚大学的 David Harbater 表示：“你得到了关于伽罗瓦群的一条信息，那么它延展开来，告诉你更多的信息。”</p><p>例如，伽罗瓦群能够马上告诉你一个多项式是否可以彻底解出来。它允许你比较不同多项式的潜在结构。伽罗瓦群同样可以用来研究代数和数论中的各种数学对象，从而为其他方法无法得到的问题提供解决方案。</p><p>Conrad 说：“将一个关于多项式的问题转化为一个关于群的问题，这种方法为许多其他的数学运算和技术打开了大门。这些运算和技术在多项式的原始语言中是不容易描述的。”</p><p>这种推广性使得伽罗瓦群在上个世纪的许多最著名的数学课题中发挥了核心作用，在 Gerd Faltings 1983 年对<strong>莫德尔猜想</strong>的证明和 Andrew Wiles 1994 年对<strong>费马最后定理</strong>的证明中都有体现。</p><p>伽罗瓦群也是当今数学中正在进行的一些最令人兴奋的工作的核心。比如伽罗瓦群是一项名为 Langlands 的项目的关键。这一项目计划将一个关于多项式的问题变成了一个关于伽罗瓦群和另一类特殊群之间关系的问题，而这将更复杂，也更有启发性。</p><p>虽然伽罗瓦的生命短暂，但他这项最伟大的成就将在未来的几个世纪里继续推进数学的发展 —— 尽管我们很难准确地预测这一过程是如何进行的。</p><p>正如威斯康星大学麦迪逊分校的 Jose Rodriguez 所说：“<strong>伽罗瓦群总是出现在令人惊讶的地方</strong>。”</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
