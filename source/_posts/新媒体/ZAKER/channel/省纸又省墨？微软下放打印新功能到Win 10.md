<p>ZAKER 科技 8 月 5 日消息，微软正在将 Windows 11 的新打印功能引入 Windows 10，官方此前表示，将在未来几个月 / 几年内向 Windows 10 提供 " 一组范围内的功能 "，新的打印功能就是这一承诺的最新证明。</p><p>据 Windows Latest 报道，Windows 10 的打印功能将引入 PIN 码保护，将 PIN 码添加到打印选项后，只有当用户在打印机上输入相同的 PIN 码，否则打印机不会打印该作业。这是避免重复连接 / 错误打印输出的新方法之一。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202208/62ecd3e7b15ec0476c08fff2_1024.jpg" data-height="679" data-width="852" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202208/62ecd3e7b15ec0476c08fff2_1024.jpg" referrerpolicy="no-referrer"></div></div>据微软称，PIN 码可以减少纸张和碳粉的浪费，同时，该功能还为用户提供了更高的隐私和安全性，尤其是在拥有多台打印机的环境中。<p></p><p>这些 Windows 11 的新打印功能正在下放到 Windows 10 Build 19044.1806 ( KB5014666 ) 中，用户需要加入 Release Preview 频道。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
