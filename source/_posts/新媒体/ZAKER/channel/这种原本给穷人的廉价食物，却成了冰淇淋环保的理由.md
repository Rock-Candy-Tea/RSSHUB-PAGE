<p>去年，一个关于冰淇淋的新闻闹得沸沸扬扬—梦龙的冰淇淋中外用料不同，中国用植脂，国外用乳脂。梦龙称这样是为了环保和健康。</p><p>很明显，这其实是为了降低成本，如果你看过它的发明史，会更加确信这一点。</p><p>19 世纪，法国皇帝拿破仑三世为了一个问题发了愁，<strong>直接给军队和下层人发黄油做奖励太划不来了，得想个法子降低成本才行。</strong></p><p>当时，一个法国化学家站了出来，叫 伊波利特 · 米格 - 穆列斯 Hippolyte Mège-Mouriès，他提出了一个解决方案，用牛油和脱脂牛奶混合加工即可。</p><p></p><div class="img_box" ……</strong></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 1945 年，最初始的那个配方，牛油加脱脂牛奶的组合几乎从市场上完全消失。<p></p><p>1950 年后，牛油供应短缺加上氢化技术进步，美国市场上所有的人造黄油都改为植物油 + 氢化脂肪，也就是植脂，反式脂肪酸是其中的主要成分。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div referrerpolicy="no-referrer"></div></div><strong>由于氢化脂肪是一个敏感的名词，所以多数食品生产商都会在包装上，以另一些名字代替氢化脂肪，以图掩人耳目，以下是部份名词，都可能是氢化脂肪</strong>：<p></p><p>植物人造黄油 ( Vegetable Margarine ) </p><p>植物脂肪 ( Vegetable Fat ) </p><p>植物起酥油 ( Vegetable Shortening ) </p><p>硬化脂肪 ( Hardened Fat ) </p><p>植物脂肪和植物油 ( Vegetable Fats and Oils ) </p><p>氢化植物油 ( Vanaspati ) </p><p>菜油 ( Vegetable Ghee ) </p><p>部份氢化油 ( Partially Hydrogenated Oil ) </p><p>为了统一概念，下面统称植脂。</p><p>这种人造的黄油和天然的黄油外表不太一样，简单来说就是不够黄，人造的是白色的，看起来很没有食欲，因此生产厂家给人造黄油加了色素，看起来天然一些。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" data-original="http://zkres2.myzaker.com/202203/622a93918e9f0925ad42fa22_1024.jpg" data-height="320" data-width="543" referrerpolicy="no-referrer"></div></div>不过光是靠便宜来吸引人买还不够，<strong>生产厂家还给植脂附送了另一个概念：健康。</strong><p></p><p>所谓健康就是低脂，植脂通常比动物油脂肪含量更低。</p><p>在 20 世纪 40 年代，冠状动脉疾病是美国人死亡的主要原因。在当时，科学家都认为食物中胆固醇和饱和脂肪过高和心血管疾病关系很大（科学在不断进步就是这个道理，没有什么是一成不变的，很多东西都有局限性）。</p><p>顺理成章的，降低食物中的饱和脂肪成了健康指南。低脂的植脂做的人造黄油就是推荐的食物材料之一。</p><p>基于当时的研究水平，植脂也得到了国际医学协会指导方针和非功利机构的认可。市场很快响应号召，餐厅都用上了含有反式脂肪酸的人造黄油，不再用饱和脂肪过多的动物油（比如猪油）。</p><p></p><div " 胆固醇）和甘油三酯的含量，同时降低高密度脂蛋白（" 好 " 胆固醇）的含量，增加炎症因子，创造一个完美的致动脉硬化环境。<p></p><p>而且，含有反式脂肪酸的植脂会增加患心脏病、中风、2 型糖尿病的风险，并与几种癌症有关。</p><p></p><div class="img_box" id="id_imagebox_7" onclick><div 反式脂肪酸且低成本的植脂走向商业化还得有很长的一条路。想要健康，还是少吃点加工食品吧。</p><p><strong></strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
