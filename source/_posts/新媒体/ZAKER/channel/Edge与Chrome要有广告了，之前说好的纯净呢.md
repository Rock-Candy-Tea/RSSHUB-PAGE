<p>在今年的 6 月 16 日，诞生于 1995 年、有着 27 年历史的 Internet Explorer 浏览器（下文简称为 IE）正式被微软放弃，从此 IE 作为 Windows 操作系统一部分的故事也告一段落。与此同时，另一老牌浏览器 Firefox 的日子也不太好过，包括苹果商务网站、日本 NHK 旗下视频平台 NHK+ 等，一大批网站纷纷宣布停止对 Firefox 浏览器的支持。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" 系列的宣传，点击则会跳转至微软官方商城。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 新标签页内的各选项中都找不到广告相关内容，有网友在翻看了这一页面的源代码后发现，微软实现广告的做法是通过加入一个对象名为 HotSoptContainer ( 热点容器 ) ，以及子对象 hotSpotRenderRegion ( 热点渲染区域 ) ，也就是在 Windows 10 锁屏界面上小提示一样的设计。<p></p><p>无独有偶，今年 6 月中旬，谷歌的 Chrome 浏览器上也出现了原生广告的痕迹。当时，是 YouTube 正在进行美妆购物节，谷歌在 Chrome 新标签页里就直接加上这个活动的广告，点击后就会跳转至活动页面，就与双十一、618 等国内电商购物节时出现在各大网站、APP 上的广告如出一辙。但在关于 YouTube 美妆购物节广告出现前，Chrome 的这个位置则被用来放置关于疫情等重要信息的提示。</p><p></p><div 自问世以来能够迅速风靡全球，除了更高的性能与丰富的扩展功能外，就要数简洁、纯净的特质最受用户青睐了。特别是与一众国内市场的浏览器产品相比，Chrome 乃至 Edge 的特色就是无广告，这也是许多老用户向他人推荐两种的重要理由。如果有广告，Chrome 也会在初次安装时，询问用户是否将 YouTube 等自家应用固定在新标签页上。<p></p><p>那么问题就来了，为什么谷歌和微软要在浏览器上加如广告呢？其实答案很简单，就是 " 地主家也没有余粮了 "。当下的情况，是微软这两年疯狂 " 买买买 "，诸如 75 亿美元收购游戏大厂 ZeniMax Media、数百亿美元的动视暴雪，再加上云服务及其他业务林林总总的收购，近两年其为了收购花费了近千亿美元。</p><p></p><div Stadia。以至于谷歌公司高层在近期表示，作为一家公司，谷歌的生产率与现有员工数量并不匹配，未来不排除裁员的可能性。没错，如今不仅仅是国内互联网厂商面临 " 过冬 " 的问题，美国科技巨头同样也在为开源节流烦心。<p></p><p>并且不同于微软的 Azure、Windows、Office，以及谷歌旗下的 Android、搜索引擎和数字广告等营利部门，微软 Edge 与谷歌 Chrome 其实都是纯粹的成本部门。作为互联网上一门 " 古老 " 的生意，早期浏览器的定位为软件，当时通过销售拷贝为生的浏览器反而是为数不多能挣钱的业务，网景公司就依靠浏览器成为了当时与雅虎齐名的互联网企业。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" data-original="http://zkres2.myzaker.com/202208/62e7c38e8e9f09234d6b6e44_1024.jpg" data-height="323" data-width="600" 则进行了商业模式的迭代。如今，国内浏览器产品的典型做法，就是右下角的弹窗广告、网站导航页、出错页面广告等，而海外市场的 Firefox 则选择了将内置搜索引擎卖出一笔好价钱。比如，Firefox 就与谷歌在 2020 年达成协议，将浏览器内置的搜索引擎设置为谷歌，为此后者付出了 3 年 13.5 亿美元的代价。<p></p><p>然而进入移动互联网时代后，浏览器作为流量入口的地位却在不断下降。对于微软、谷歌，乃至腾讯等巨头而言，浏览器最大的作用就是流量入口。PC 互联网时代，浏览器可谓是用户接触互联网的最核心渠道，有了浏览器就有了 web 生态。并且随着宽带和移动通信技术的进步，操作系统对本地数据存储依赖性越来越低，更多的计算和数据都放在了 web 端。</p><p></p><div 的重要原因。当然了，事实告诉我们，谷歌、微软预想的 web 生态并没有如期而至。如今谷歌和微软手误浏览器这个流量入口，却好似空守宝山而不得入。在当下这个即便巨头都开始追求效率、追求产出的情况下，Edge 和 Chrome 选择将流量变现自然也不令人感到意外。<p></p><p>只是有一个问题，对于 Chrome、Edge 的用户来说，最担心的情况莫过于这仅仅只是一个开始，为自家的业务打广告或许尚能接受，但未来要是全面效仿国内的浏览器产品，可该如何是好？</p><p>【本文图片来自网络】</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
