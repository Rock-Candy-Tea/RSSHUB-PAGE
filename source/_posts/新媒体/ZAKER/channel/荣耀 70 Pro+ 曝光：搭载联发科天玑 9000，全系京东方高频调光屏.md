<p>品玩 4 月 18 日讯，爆料者 @Yash Raj Chaudhary 放出了最新的荣耀 70 系列规格。从之前的爆料来看，这一系列的机型将于下个月发布，但目前还没有出现比较靠谱的渲染图。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202204/625d13758e9f09525d5898c0_1024.jpg" data-height="160" data-width="689" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/625d13758e9f09525d5898c0_1024.jpg" referrerpolicy="no-referrer"></div></div> 图源：微博截图<p></p><p>据称，荣耀 70 全系列将选取京东方屏幕，而且都支持高频 PWM 调光，标准版将搭载高通骁龙 7，而 Pro 和 Pro + 则是天玑 8100 和天玑 9000，而且还配备了旗舰级的双主摄 + 长焦设计。</p><p>荣耀在去年 6 月发布了荣耀 50 系列，又在 12 月发布了荣耀 60 系列，如果不出意外的话就会在这几个月推出新一代的荣耀 70 系列。</p><p>荣耀 70</p><p>该机预计将搭载高通骁龙 7Gen1 芯片，采用京东方类钻排列的屏幕 ( 2400*1080 ) ，支持高频调光 + 120Hz 刷新率 + 10bit 色，后置 1 亿像素 + 800 万像素超广角 + 200 万像素微距三摄，支持 66W 快充配备 4800mAh 电池，还有 Z 轴线性马达、立体声双扬、NFC 等</p><p>荣耀 70 Pro</p><p>该机预计将搭载联发科天玑 8100，同样的京东方类钻屏 ( 2600*1200 ) ，高频调光 + LTPO 1-120Hz 自适应刷新率 + 10bit ，后置 5000 万像素主摄 ( IMX766+OIS ) +5000 万像素超广角 + 800 万长焦三摄，电池快充相同，X 轴线性马达、立体声双扬、NFC、红外遥控。</p><p>荣耀 70 Pro+</p><p>该机预计将搭载联发科天玑 9000，京东方类钻屏 ( 2800*1300 ) ，高频调光 + LTPO 1-120Hz 自适应刷新率 + 10bit，后置 5000 万主摄 ( IMX766+OIS ) +5000 万像素超广角 + 1200 万长焦三摄，100W 快充 + 4600mAh 电池，X 轴线性马达、立体声双扬、NFC、红外遥控。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202204/625d13758e9f09525d5898c1_1024.jpg" data-height="794" data-width="590" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/625d13758e9f09525d5898c1_1024.jpg" referrerpolicy="no-referrer"></div></div> 图源：截图<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
