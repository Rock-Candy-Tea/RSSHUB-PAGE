<p>IT 之家 5 月 24 日消息 据外媒 Windows Latest 报道，根据其发现的一系列招聘信息，微软正准备为 Windows 10 的设计做出一些 " 根本性 " 改变。</p><p>微软已经对大型设计改革进行了一段时间的研究。今年早些时候，微软在一则招聘公告中悄然确认了将有 " 令人振奋的 " 更新。目前尚无关于 " 太阳谷（Sun Valley）" 的详细信息，但外媒了解到，新的视觉外观将专注于圆角，类似于 Windows 10X 的功能和过渡。</p><p>同时，在本月初发布的新职位列表中，微软证实，它希望在未来几年大幅改变 Windows 平台的用户体验。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202105/60934d7ab15ec0035c04bfdb_1024.jpg" data-height="365" data-width="696" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202105/60934d7ab15ec0035c04bfdb_1024.jpg" referrerpolicy="no-referrer"></div></div>▲图自 Windows Latest<p></p><p>该职位由微软互动体验平台（IXP）团队发布。后者负责 Windows 和 Windows 应用的 UX（User Experience，用户体验）。该团队还负责开始菜单、Office、Microsoft Edge 和原生 Windows 应用的核心、渲染和输入平台。</p><p>" 我们还拥有很多 Win32 API，以及所有 Windows 特性和应用的基础架构。这个团队为 Windows 带来了触控，为 Windows 笔记本带来了精准触控板，为二合一设备带来了 Windows Ink。"</p><p>IT 之家了解到，微软在一份招聘广告中指出：" 我们正开始一段多年的‘旅程’，通过提供最佳的交互体验，展示最先进的硬件，来革新 Windows UX 平台。" 该招聘广告截至 5 月 24 日仍在其网站上。</p><p>此外，在另一份不再提供的招聘信息中，负责窗口体验的微软团队也证实，他们正在 " 重新构想这个平台，目标是在 Windows 上实现最佳交互体验 "。</p><p>根据 Windows Latest 的消息，微软也在为 Windows 开发一种新的手势体验。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
