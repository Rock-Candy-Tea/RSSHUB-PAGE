<p>整理 | 章雨铭 责编 | 屠敏</p><p>1995 年 8 月 16 日，Windows 第一个默认浏览器—— Internet Explorer（IE）首次亮相，由于和 Windows 的捆绑安装，IE 浏览器迅速占领市场。在其最鼎盛时期，IE 可谓是称霸市场，占有率达 90% 以上，是现在的 Chrome 都所不及的。</p><p>而现在，IE 正在淡出舞台。2021 年微软官宣称，2022 年 6 月 15 日 IE 11 浏览器将正式 " 退休 "，取而代之的是更好的 Microsoft Edge。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " referrerpolicy="no-referrer"></div></div>（图源微软）<p></p><p>时光飞逝，转眼这一天就来了。</p><p><strong>昔日霸主，如今没落</strong></p><p>曾经 IE 浏览器可谓是浏览器界的霸主，作为 90 年代中期 Windows 的标配，再加上当时设计的界面简洁美观、网页交互性强等特点，几乎是 " 秒杀 " 其他浏览器。</p><p>不过后来，由于微软管理的松懈、互联网的飞速发展，IE 浏览器逐渐被 " 赶超 "，最终落队。而 2015 年替代品 Microsoft Edge 的出现，也意味着留给 IE 浏览器的时间不多了。</p><p>你最近使用过 IE 浏览器吗？</p><p>相信已经几乎没有了。根据 StatCounter 数据显示，2022 年 5 月，IE 浏览器的市场占比率已经只剩下 0.64% 了，当初的辉煌早已不在了。可能在大多数人的记忆里，登录一些必须要使用 IE 浏览器才能启动的老旧的网页和服务，比如一些政府网站和银行网站，才会使用 IE 浏览器。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202206/62a9303f8e9f09092b731e09_1024.jpg" data-height="792" data-width="1238" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202206/62a9303f8e9f09092b731e09_1024.jpg" referrerpolicy="no-referrer"></div></div>（图源 StatCounter）<p></p><p>而 Microsoft Edge 是一个很好的替代品，它拥有更棒的兼容性，不仅可以免除 IE 浏览器开启老旧网站、服务的必要性（Edge 内置了 IE 模式，用户可以直接从 Edge 上访问那些基于 Internet Explorer 的旧版网站和应用程序），而且能够给用户提供更顺畅的上网体验，减少用户使用不同浏览器的麻烦，以提供生产力。</p><p>IE 浏览器的 " 退场 " 似乎成为了必然。</p><p></p><div " 对用户有何影响</strong></p><p>在 IE 浏览器正式退休之后的几个月时间里，IE 桌面应用程序将会先被重定向到微软 Edge 浏览器（用户启动 IE 11 时会自动跳转到 Edge），最终微软将通过后续的 Windows 更新永久禁用 IE。需要注意的是，用户不用完全卸载 IE，因为 Edge 浏览器的 IE 模式需要依靠 IE 11 来运作。</p><p>另外，微软宣布大多数 Windows 版本中的 IE 浏览器都会被停用，比如：Windows 10 客户端 SKU（版本 20H2 及更高版本）和 Windows 10 IoT（版本 20H2 及更高版本）的 IE 11。在 Windows 11 中，默认浏览器已经变为 Edge 了，所以 IE 浏览器也不可用了。</p><p>但是并非所有的 IE 浏览器都会被停用，不受停用影响的包括：Edge 中的 IE 模式，以及在以下系统版本中的 IE 11 桌面版：</p><p>Windows 8.1</p><p>Windows 7 ESU（带有扩展安全更新）</p><p>Windows Serve LTSC（所有版本）</p><p>Windows Server 2022</p><p>Windows 10 客户端 LTSC（所有版本）</p><p>Windows 10 IoT LTSC（所有版本）</p><p>而且由于 Windows 系统标配的 Edge 浏览器中已经内置了 IE 浏览器的兼容模式，用户只需要打开该功能就可以正常浏览仅兼容 IE 浏览器的网站，所以一些考公、考试、银行等网站也能用 Edge 打开。</p><p>微软对 Edge 浏览器中的 IE 模式的支持将至少到 2029 年，这个支持也需要符合当前和未来的 Windows 系统的生命周期，如果对某个版本的 Windows 的支持在 2029 年之前结束，那么对该版本 IE 模式的支持也将结束。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 "
