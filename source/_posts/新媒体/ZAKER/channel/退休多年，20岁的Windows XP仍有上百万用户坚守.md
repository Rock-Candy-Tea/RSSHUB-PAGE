<p>整理 | 祝涛 </p><p>2001 年 10 月 25 日，微软生产的操作系统 Windows XP 正式登陆零售商店，曾备受用户的喜爱。2009 年微软结束了对 Windows XP 的主流支持， 2014 年，微软结束了对 Windows XP 的外延支持，该系统正式退出舞台。作为微软历史上最成功的操作系统之一，Windows XP 陪伴无数人度过了漫长的岁月。如今，这个伴随着一代人成长的系统迎来了 20 岁的生日。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202110/6178f6f18e9f091ca3286abd_1024.jpg" referrerpolicy="no-referrer"></div></div>高低起伏的绿山，漂浮着层积云和卷云的蓝天——相信这张图片你一定见过吧。这就是 Windows XP 的默认桌面壁纸，名叫 "Bliss（无比幸福）"，深深印在一代人的记忆里。<p></p><p>Windows XP 是微软公司推出的供个人电脑使用的操作系统，包括商用及家用的台式机、笔记本电脑、和平板电脑等，其名字 "XP" 的意思是英文中的 " 体验 "（experience）。Windows XP 是继 Windows 2000 及 Windows Me 之后的下一代 Windows 操作系统，也是微软首个面向消费者且使用 Windows NT 架构的操作系统。</p><p><strong>发展历程</strong></p><p>2001 年 8 月 24 日，微软副总裁吉姆 · 奥尔钦发布了 Windows XP 的 RTM 版本。</p><p>2001 年 10 月 25 日，微软正式发布 Windows XP 零售版。</p><p>2002 年 9 月，微软推出 Windows XP SP1 补丁包。</p><p>2004 年 9 月，微软推出 Windows XP SP2 补丁包。</p><p>2005 年，微软还发布了 64 位的 Windows XP 客户端和 Windows Server 2003 服务器 2 个系列，支持 Intel 和 AMD 的 64 位桌面处理器。</p><p>2008 年 4 月，微软推出 Windows XP SP3 补丁包，并宣布这是为 Windows XP 最后一次升级。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202110/6178f6f18e9f091ca3286abe_1024.jpg" referrerpolicy="no-referrer"></div></div>2008 年 6 月 30 日，微软停止 Windows XP 销售。<p></p><p>2009 年 4 月 14 日，微软停止 Windows XP 的主流技术支持，包括新的 IE、DirectX、MSN 等微软服务不能在 XP 上使用。</p><p>2014 年 3 月 11 日，微软中国宣布将与奇虎 360 公司合作，为中国 XP 用户提供过渡期间的安全防护服务，以及 Windows 8 升级方案。</p><p>2014 年 4 月 8 日，微软停止对 Windows XP 的所有版本的支持与服务，包括补丁、升级和漏洞修复。</p><p>2017 年 5 月 13 日，由于 "Wannacry" 勒索蠕虫事件影响巨大，微软发布公告，再次对已停止支持的 Windows XP 发布特别补丁。</p><p><strong>现状</strong></p><p>2011 年 9 月底前，Windows XP 是世界上使用人数最多的操作系统，市场占有率达 42%；在 2007 年 1 月，Windows XP 的市场占有率曾达历史高峰（后被 windows 10 超越），超过 76%。根据 Netmarketshare 公司对全球网络用户的统计数据显示，2012 年 8 月份，统治操作系统市场长达 11 年之久的 Windows XP 最终被 Windows 7 超越。</p><p>发行 20 年后，Statcounter 的最新数据显示，现有 Windows 电脑用户中仍有 0.59%（大概 780 万）的用户在使用老式的 Windows XP，值得注意的是早在 2009 年和 2014 年微软就分别结束了对 WinXP 的主流支持和外延支持。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div 的黑桃王、跳棋、双陆棋、黑白棋、红心大战等游戏已关闭服务器无法连线与其他玩家游玩，且系统无法支持更新优化，微软已正式淘汰 Windows XP 产品。如今许多用户都会在新的操作系统例如 Windows 7、Windows 8、Windows 8.1、Windows 10 桌面上安装虚拟 Windows XP 运行许多怀旧游戏软件。<p></p><p>还记得文章开头提到的经典壁纸吗？许多人曾尝试在同一地方还原它，但当年那片景象已然改变，农场主的篱笆、葡萄树占据了大片面积，景色远没有 20 年前那么纯净了。就像微软这 20 年一直改造 Windows 一样，Windows XP 也终究回不去巅峰期了。</p><p>20 年后的今天，Windows XP 仍然赢得了许多人的心，可惜的是，它终将只能留在我们的记忆中，而不是我们的电脑里。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
