<p>6 月 15 日消息，微软于本周三永久关闭 IE 浏览器，这意味着 IE 时代的落幕，也给那些用 IE 内核开发内部系统的企业带来不少麻烦，其中受影响最大的是很多日本企业。</p><p>存续 20 多年的 IE 浏览器退出历史舞台，会给使用基于 IE 内核系统的企业机构带来一系列问题。日本可能是受微软这一举措影响最大的国家。因为今年 3 月份的一项调查发现，日本有 49% 的公司仍在使用 IE 浏览器以及基于 IE 内核的内部应用。其中最常用的是内部管理、数据交换和会计系统等应用。自从微软在一年前宣布将永久关闭 IE 浏览器以来，所有这些应用程序都应该更新或转版，但有报道称许多举措都拖延了。</p><p>日本各地企业现在必须迅速采取行动，才能确保企业内部基于 IE 内核的应用程序能够正常运行。</p><p>IE 浏览器曾经是全球浏览器领域的霸主，事实上也是网络标准的制定者，但 IE6 之后因功能繁多和性能不佳而逐步被用户所抛弃。市场研究公司 Statcounter 的数据显示，谷歌 Chrome 浏览器和 Mozilla 的火狐浏览器等不断抢占更多市场份额，今年 5 月份 IE 浏览器的全球市场份额仅为 0.64%。</p><p>微软 IE 浏览器关闭后，相关功能由 Edge 浏览器接棒。这是一款基于 Chromium 内核的浏览器，与 Chrome 兼容，并支持大部分相同功能。微软宣布，Edge 浏览器会在今后一段时间内继续支持 IE 模式。（辰辰）</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
