<p>我们知道，Windows 9 从来没有在公众面前出现过，微软在 Windows 8.1 之后直接发布了 Windows 10，而现在 Windows 的系统版本更是来到了 Windows 11。然而，有人却发现，在最新一代 Windows 系统也就是 Windows 11 中，其实竟然暗藏着 Win9 的相关信息。由此看来，Windows 9 曾经在微软内部立项，但不知为何它没有成为正式产品。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202203/62280bafb15ec026664fe214_1024.jpg" data-height="756" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/62280bafb15ec026664fe214_1024.jpg" referrerpolicy="no-referrer"></div></div>据了解，有用户在 Twitter 上发布了一张截图，称 Windows 10 和 Windows 11 中都可以找到这个带有 Windows 9 相关的注释字串。Twitter 用户 Xeno 展示彩蛋的时候，称从 Windows 10 build 9860 到最新的 Windows Insider Dev Build ( 22567 ) ，这个 Windows 9 的字符串都可以观察得到。除了 Windows 9 外，Windows 8.1 的代号 Windows Blue 也有出现。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202203/62280bafb15ec026664fe215_1024.jpg" data-height="320" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/62280bafb15ec026664fe215_1024.jpg" referrerpolicy="no-referrer"></div></div>我们知道，尽管 Windows 9 并未出现，但它的蛛丝马迹存在也并非是一件很不可思议的事情。当微软决定不同方向的战略的时候，合并 Windows 项目是很常见的做法。因此，Windows 9 并不是什么特别值得注意的东西，Windows 9 也并不会是微软暗藏的大招，它可以作为微软成功发布的操作系统中的一个参考。但发掘 Windows 9 的蛛丝马迹仍然是有趣的琐事，就像你突然发现地球上曾经存在某个国家一样。<p></p><p>据了解，微软似乎还有 Windows 12 的规划。Windows 9 成为往事，而微软曾经信誓旦旦称 Windows 10 是最后一个 Windows 系统。但现在我们都知道，不但迎来了 Windows 11，甚至有媒体称，连 Windows 12 都似乎有眉目了。未来的 Windows 会是怎样的呢？它们会不会还继续暗藏 Windows 9 的历史踪迹？我们拭目以待吧。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
