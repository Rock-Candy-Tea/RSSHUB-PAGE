<p>11 月 16 日消息，当地时间周一，美国国务院宣称，俄罗斯用陆基导弹摧毁了其一颗在轨卫星，造成数千块碎片散布到地球轨道上。美国已经确认了 1500 多块可追踪的碎片，以及数千块无法追踪的较小碎片。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202111/619348ad8e9f091e6b06023f_1024.jpg" data-height="439" data-width="660" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/619348ad8e9f091e6b06023f_1024.jpg" referrerpolicy="no-referrer"></div></div>同时，俄罗斯航天局 Roscosmos 表示，由于国际空间站每隔 90 分钟 ( 绕地球一周的时间 ) 就会接近大团的太空碎片，驻留在上面的七名宇航员今天早上不得不选择就地避难。美国宇航局（NASA）尚未确认通过国际空间站的太空碎片是否来自俄罗斯的反卫星测试 ( ASAT ) ，该机构也没有立即回置评请求。<p></p><p>然而，美国国务院表示，这些太空碎片对国际空间站构成危险。该机构发言人内德 · 普莱斯 ( Ned Price ) 称：" 这次试验将大大增加国际空间站上的宇航员以及其他人类航天活动的风险。俄罗斯这种危险和不负责任的行为危及我们在太空中的长期可持续性。"</p><p>负责跟踪环绕地球轨道的太空物体和碎片的美国太空司令部发表声明称，该部门今天早上 " 意识到了太空中发生了一起碎片产生事件 "。</p><p>美国太空司令部说：" 我们正在积极确定碎片区域的特征，并将继续确保所有太空国家获得必要的信息，以便在受到影响时操纵卫星机动。我们还在与包括国务院和 NASA 在内的跨部门机构进行合作，并将在不久的将来提供最新情况。"</p><p>在过去的几年里，美国太空司令部始终在跟踪来自俄罗斯的反卫星试验。2020 年，该部门报告了俄罗斯反卫星技术 ( Nudol ) 的两次测试。然而，这两次试验似乎都没有真正摧毁太空中的任何目标。</p><p>私人太空跟踪公司 LeoLabs 在推特上证实，他们观察到多个物体出现在名为 Kosmos 1408 的俄罗斯旧卫星周围，据传这颗卫星是反卫星测试的目标。太空跟踪专家乔纳森 · 麦克道尔 ( Jonathan McDowell ) 表示，如果情况属实，那么这次试验产生的碎片云很可能就是今天早上威胁国际空间站的碎片云。</p><p>在今天早上国际空间站最初几次通过碎片云的过程中，部分宇航员躲入了 SpaceX 的载人龙飞船中，该飞船目前正与国际空间站对接，它上周刚刚将四名新宇航员送到空间站。不过，宇航员们最终打开了飞船的舱门，进入了国际空间站。目前有七人居住在国际空间站，其中两人是俄罗斯宇航员。</p><p>俄罗斯航天局 Roscosmos 随后证实：" 目前，国际空间站的工作人员正在按照飞行计划进行例行操作。今天迫使宇航员按照标准程序进入飞船的碎片已经离开了国际空间站轨道，空间站处于安全区域。"</p><p>NASA、Roscosmos 和他们的合作伙伴定期监测空间站周围的 " 安全围栏 "。空间站的轨道经常需要机动，以躲避进入该区域的太空碎片。</p><p>CBS 太空记者威廉 · 哈伍德 ( William Harwood ) 在推特上写道：" 细节还不清楚，但国际空间站的 7 名机组人员今天早些时候躲进了载人龙飞船，因为预计他们会接近 ( 或穿过 ) 卫星解体造成的‘碎片云’。"</p><p>这是空间站的标准程序，宇航员在遇到任何紧急情况时都要撤入飞船中，以防需要紧急撤离。（小小）</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
