上的任何形式的音频内容自动生成字幕；</p><p>②语音访问。允许用户使用语音来控制 PC 以及创作文本；</p><p>③ Narrator 自然声音。它可为用户带来更接近自然的语音、更悦耳的音频，以帮助阅读或浏览网页。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div 中的系统范围的实时字幕<p></p><p>在线安全、数据隐私也是用户最关心的问题，因此微软为 Microsoft Defender 带来了一些新功能。</p><p>微软表示，适用于个人的 Microsoft Defender 扩展了 Windows 的内置安全性，并将设备保护扩展到用户的 Android 和 iOS 手机和 Mac。同时在美国更新了一系列 Microsoft 365 安全功能。微软号称 Windows 11 是迄今为止最安全的 Windows 版本。</p><p>生产力方面，通过对 Snap 布局的增强、新 Focus 功能以及性能和续航优化，新的 Windows 11 2022 更新将帮助用户提高工作效率。</p><p>此外，22H2 优化了多任务界面的对齐布局，用户可以通过更好的 " 触摸 " 导航在 Microsoft Edge 中快速搜寻多个浏览器选项卡，使 Snap 布局更加通用。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" 的使用越来越多地用于通信、创造力和游戏，因此在更新中改善了 Windows Studio Effects 视频和音频通话功能，带来了新的开发工具、额外的游戏功能以及 Microsoft Store 更新。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div 效果<p></p><p>在游戏性能上，新版实现了性能优化，改善了游戏延迟，并解锁了窗口游戏的自动 HDR 和可变刷新率等功能。同时，微软通过 Xbox 应用程序直接内置到 Windows 11 中的 Game Pass，玩家可以访问数百种高质量的 PC 游戏。</p><p>此外微软兑现了其关于在 Windows 11 上支持 Android 应用的承诺，为满足特定功能硬件要求的 Windows 中的智能应用控制，可以阻止可能具有潜在恶意的不受信任和未签名的应用程序<p></p><p>微软还透露，将采取措施改进 Windows 更新体验。Windows 更新现已具有碳环保意识，当设备插入、打开、连接到 Internet 并且区域碳强度数据可用时，Windows 更新将在一天中的特定时间安排安装，这样可以减少碳排放。微软还对睡眠和屏幕关闭的默认电源设置进行了一些更改，以帮助减少电脑空闲时的碳排放。</p><p>除了上述已有的功能，微软还宣布将在 10 月份推出一系列额外的体验：</p><p>◆文件资源管理器增强。包括将 " 选项卡 " 引入文件资源管理器，你可以固定重要文件，以便快速和方便地访问 ，还包括基于 OneDrive 功能和 Microsoft 365 帐户的个性化建议。</p><p>◆照片应用优化。将引入一个新的照片管理功能，带来一个相册界面，简化浏览、查找、管理和收藏等方面的功能，而且你还可以用 OneDrive 快速备份照片，并提供 " 回忆 " 功能。</p><p>◆复制建议。当复制电话号码、日期时会获取操作方面的建议，例如用 Teams 或 Skype 打电话或在日历中添加新事件等。</p><p>◆任务栏溢出。微软将在任务栏中增加一个溢出入口，允许你快速查看所有溢出的程序。</p><p>◆快速分享到更多设备。你可以快速发现并分享到更多的设备，包括使用附近共享的桌面。</p><p>目前，Windows 11 22H2 版本更新 ISO 镜像已经在官网上线，有需求的用户可以前往下载。</p><p>编辑：崔崔</p><div
