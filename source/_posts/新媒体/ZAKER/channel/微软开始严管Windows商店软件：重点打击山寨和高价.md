<p>虽说微软 Windows 系统早就提供了官方商店，不知道多少人和小狮子一样从来都没用过，主要是觉得里面的软件良莠不齐，想找到好用的应用犹如大海捞针。</p><p>之所以出现这种情况，很大一部分原因在于微软的 Windows 生态是非常开放的，然而这也导致了软件质量良莠不齐，特别是 Windows 商店，之前微软审核太松，让一些山寨甚至流氓软件上架，现在微软终于下定决心管理了，将禁止山寨的软件上架，而且定价过高的也不行。</p><p>由于之前的审核不严，有一些开发者会利用微软的机制将一些免费软件或者开源软件包装成自己开发的软件，上架 Windows 商店，而且部分软件定价离谱，让不少用户上当受骗，有些人可能记得之前的 switchy sharp 插件就被人动过手脚一事。</p><p>针对这些问题，微软日前更新了 Windows 商店的新政策，最新的 10.8.7 版中明确了几点要求，微软做了新规定。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202206/62b03cc5b15ec00789095fdc_1024.jpg" data-height="414" data-width="616" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202206/62b03cc5b15ec00789095fdc_1024.jpg" referrerpolicy="no-referrer"></div></div>在您确定产品或应用内购买的定价的情况下，您的数字产品或服务的所有定价（包括销售或折扣）必须：<p></p><p>遵守所有适用的法律、法规和监管指南，包括但不限于联邦贸易委员会打击欺骗性定价指南。</p><p>不要试图从开源或其他通常免费提供的软件中获利，也不要试图相对于您的产品提供的特性和功能定价过高。</p><p>从官方的要求来看，微软打击的两个重点问题就是山寨及过高的定价，这会让一些靠山寨别人产品的开发者断了财路，不过对用户来说可以避免再被忽悠上当。</p><p>编辑：熊乐</p><p><strong>· END<strong>·</strong></strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
