Windows 11 增添了不少新功能，也有不少功能被阉割 ……<p></p><p>这已经不是微软的第一次了，如果要给被微软抛弃、宣布「死亡」的产品都立一块墓碑，那么恐怕一个墓园都未必能放得下，而当中不乏「好物」，却免不了被微软「砍刀部』下手的命运，本文想简单为大家盘点几款 —— 另外需要说明的是，那些因新版本的推出而退出历史舞台的产品不算盘点范围内。</p><p><strong>|</strong><strong> MSN ( Windows Live Messager ) </strong></p><p>对于 80、90 年代的朋友来说，MSN 这个词应该跟 QQ 一样那么熟悉，它是微软开发的一款即时通讯软件，后来被改名成为 Windows Live Messager；在当时的印象里，MSN 似乎是要比 QQ 更高级的通讯工具，甚至被不少人当作能用来跟老外进行网络交流的利器。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202109/6149c2308e9f0963fd5fc894_1024.jpg" referrerpolicy="no-referrer"></div></div>到了 2010 年后，MSN 跟国内外其它的社交平台，例如 Twitter、Facebook、新浪博客等进行了整合，但不久后，MSN 渐渐淡出了我们的视野 … …<p></p><p>其实 MSN 并没有消失，如今只是变成了 Microsoft Service Network，成为微软的一个纯粹的门户网站，知道的人也没多少。</p><p><strong>| </strong><strong> Windows Vista </strong></p><p>对微软来说，Windows Vista 是进入 21 世纪最里程碑式又具有强烈悲情色彩的产品之一了。从系统本身来看，Vista 大刀阔斧地更改了整个系统的界面，甚至解决了 XP 系统上的各种遗留问题。</p><p>虽说，那些被新版本取代的旧版产品不算本文讨论的主题，但 Vista 实在比较特殊 ……</p><p></p><div class="img_box" 并不差，甚至「过于优秀」，大量的毛玻璃界面、动画等，无时无刻在耗着电脑的各项资源，而当时市面上绝大多数的电脑硬件都远低于系统的要求，抛开实际市场来开发，最终被市场抛弃，只怪微软在 Vista 上走得太激进，也算是生不逢时了吧。<p></p><p>所幸，微软感受到了市场的反馈，在后面的 Windows 7 系统中，优化了各项资源，降低了对硬件性能的要求，而后的装机量可见，Windows 7 成为继 XP 之后，微软最成功的一套操作系统。</p><p><strong>| </strong><strong> Windows Media Center</strong></p><p>Windows Media Cente（简称 WMC）是从 Vista 起配有的一个功能（软件），可以当作是一个媒体娱乐中心，界面跟 PlayStation 有点像，有着非常炫酷的动画和较为集中的媒体集合体验，如果用户的硬件能支持，甚至可以将有线电视、音频、视频输入到 WMC 播放。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " 开始，这项功能就被阉割掉了。<p></p><p><strong>|</strong><strong> Windows Mobile / Windows Phone / Windows 10 Mobile</strong></p><p>微软的手机梦或者说是移动梦，相信大家都很熟悉了，从当初的 Windows Mobile 到 Windows Phone，再到最后的 Windows 10 Mobile，微软这一路走来着实坎坷；若要细聊，也不是一时半会能讲得清的，这里就不花笔墨了，不如留出三秒钟为这几款移动系统默哀悼个念。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " </strong><strong>动态磁贴</strong><p></p><p>动态磁贴是伴随着 Windows Phone 7 系统发布，而正式进入公众视野的，动态磁贴一来打破了传统图标式桌面死板单调的样式，二来打破了一个个 app 形成的孤岛，将 app 内的信息依据用户的喜好整合到桌面上，不仅是移动端的 Windows Phone，后面桌面端 Windows 8、8.1、10 在开始菜单都用了这种动态磁贴的图标样式。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" 生态支撑不起动态磁贴的发展，倒不如取消；于是在 Windows 11，微软终于抛弃了活泼的动态磁贴，回归了平庸无趣的图标样式，个人觉得挺可惜的。<p></p><p><strong>| </strong><strong>Project Astroia</strong></p><p>严格意义上来说，Project Astroia 并不是一个软件，而是一个架构、一种服务，在 Windows 10（Mobile）发布时推出的，原旨在允许开发者在 Project Astroia 官网上提交软件的 Android 版本 APK 安装包，由它来检测兼容性，并对关键的 API 进行逐一对应，转译、移植到 Windows 10 Mobile 平台，开发者只需要改动很少的代码就能变相地完成 Windows 10 Mobile 版本 app 的适配，降低 Windows 10 Mobile 开发者的门槛。</p><p>这一架构公布后，似乎让份额不到 1% 的 Windows 10 Mobile 看到了希望，只可惜第一个转译的 app 还没正式推出市场，就郁郁而终了 —— 或许是因为微软不想亲眼看到 Windows 10 生态变味，觉得还可以再为自己的生态赌一把吧。</p><p>不过好消息是，正式版 Windows 11 将会跟 Amazon Appstore 合作，直接兼容 Android app，至于用了什么原理暂时还未知晓，但换了角度看，还没来得及看看这世界的 Project Astroia 也算是得到安慰了。</p><p>众所周知，微软其实不只是一家软件公司，还是硬件公司，历史上被微软抛弃的硬件「好物」也并不少。</p><p><strong>| </strong><strong>Zune</strong></p><p>对于大众来说，Zune 这个名字似乎有点陌生，其实它是微软于 2006 年推出的一款播放器（当然后来 Windows Phone 7 系统的管理软件也用了这个名字），是为了与当时如日中天的苹果 iPod 竞争的产品，而它创新的操作界面，成为了后来 Windows Phone 的设计母版。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div 从此成为一个存在于历史的词语。<p></p><p><strong>| </strong><strong>Kinect</strong></p><p>Kinect 其实是 Xbox 的一个 3D 体感配件，它能对前方人进行即时动态捕捉、识别人脸、声音采集等，是在 2009 年的 E3 展会上亮相的，跟当时 Xbox 360 匹配使用；而在 2013 年，微软发布了新一代的 Kinect。</p><p></p><div class="img_box" id="id_imagebox_7" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_7" 捆绑销售，但实际市场的反响并没有太好，精确度不够是关键，游戏开发商也不太愿意针对 Kinect 开发游戏，最后微软在 Xbox One S 机型中，去掉了 Kincet 的接口，就这几乎等于给 Kincet 判了死刑。<p></p><p><strong>|</strong><strong> Lumia</strong></p><p>Lumia 的故事，不得不跟 Windows Phone 系统结合在一起看，也让不少 Nokia 的爱好者流下了时代的泪水。</p><p>在 Nokia 爱好者的心中，促成微软对其设备与服务部门收购的 Nokia 前任总裁史蒂芬 · 埃洛普，就像是罪人般的存在，让 Nokia 放弃了当时正在路上的 Meego 系统，全面与微软合作采用 Windows Phone 系统，推动微软收购 Nokia 相关部门后就回到了微软，担任设备部门执行副总裁。</p><p></p><div class="img_box" id="id_imagebox_8" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_8" Logo，却变成了扶不起的阿斗，价格虚高、产品力低下，颓势已经到了无法扭转的地步，最终把 Lumia 品牌砸在了手上，并亲手将 Lumia、Windows Phone（Windows 10 Mobile），还有微软的移动 / 手机梦一同送进坟墓，从此江湖上再无 Lumia。<p></p><p>好了，以上为大家从软件和硬件层面盘点了那些被微软「砍刀部门」相中，并抛弃了的「好物」，也希望能唤醒你心中的记忆，当然，被抛弃的「好物」远不止上面那些，如果你有不同的意见，或者认为有更值得分享的，也欢迎在下方留言与大家互动哦。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
