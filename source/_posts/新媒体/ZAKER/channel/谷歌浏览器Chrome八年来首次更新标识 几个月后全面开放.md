<p>2 月 7 日消息，据外媒报道，谷歌 Chrome 终于在八年后首次更新了其标识，以便更好地 " 与谷歌更现代化的品牌表达方式保持一致 "。这款浏览器的设计师埃尔文 · 胡 ( Elvin Hu ) 在推特上发布了重新设计的标识，并向人们展示了微妙的变化。</p><p>新的 Chrome 标识消除了外环中红色、黄色和绿色之间的阴影，实质上，此前的这种设计让三种颜色从屏幕上 " 抬起 "，显得更为突出。而最新设计让三种颜色变得更为平坦，同时还 " 优化了比例，使颜色更加鲜艳 "。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202202/62007d868e9f0972a70ae98b_1024.jpg" data-height="444" data-width="538" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202202/62007d868e9f0972a70ae98b_1024.jpg" referrerpolicy="no-referrer"></div></div>在使用 " 单色 " 的同时，谷歌确实在主图标上添加了 " 微妙的渐变 "，以缓解绿色和红色之间存在的 " 令人不快的颜色振动 "。此外，中间的蓝色圆圈也大得多。<p></p><p>在所有系统上，Chrome 标识看起来也不尽相同。在 ChromeOS 上，新的 Chrome 标识看起来将更有活力、色彩更鲜艳，并与桌面上的其他图标相辅相成。</p><p>MacOS 上，Chrome 的标识仍然会有些阴影，让它更显突出。Windows 10 和 Windows 11，Chrome 标识将有更多的渐变，以适应其他 Windows 图标的外观。</p><p>埃尔文表示，如果用户使用 Chrome Canary（Chrome 的开发者版），他们将看到新的标识。但在接下来的几个月里，它将对其他所有人开放。</p><p>对于 Chrome 标识的测试版和开发者版，用户可以看到很多新的图标，最大的变化是 iOS 上的测试版应用程序的蓝图风格图标。埃尔文称，设计团队曾试图添加一条白线作为颜色之间的边界。然而，这种设计让标识看起来太小，可能会使其在其他谷歌应用程序中更难识别。</p><p>自 2008 年以来，Chrome 标识变得更具活力和简约。最初的标识闪闪发光，上面有个三维标志，最终变成了 2D 风格。</p><p>除了对标识进行重新设计，设计团队还对 Chrome 的 " 关闭所有标签模式对话框 " 功能进行了测试。根据 XDA 开发人员的说法，尽管 " 关闭所有标签 " 菜单选项已经被植入该软件很长时间时间了了，但仍有些用户会不小心点击该选项。</p><p>为了解决这个问题，设计团队在 Chrome Canary 100 中启用了 chrome://flags 页面。这意味着当用户点击 " 关闭所有标签 " 菜单时，屏幕上会显示一个新的确认信息。</p><p>对话框一出现，用户就可以选择 Cancel ( 取消 ) ，即不关闭打开的选项卡，或选择 "Close All Tabs" ( 关闭所有选项卡 ) 确认操作。</p><p>如果要完成此过程，系统将关闭所有选项卡，浏览器也将关闭。如果你重新打开浏览器，会出现一个新的标签，通常会显示谷歌搜索或新标签页。</p><p>谷歌正在试验其确认框，而其他谷歌应用程序 ( 如 Gmail ) 中的 Clear 和 Close 操作只会显示一条带有撤销按钮的消息。如果你想尝试这一功能，可以下载 Chrome Canary，因为它还没有推广到其他渠道。谷歌尚未声明是否计划在未来推出这一功能。</p><p>2021 年，谷歌鼓励 Chrome 用户更新安卓设备上的软件，以实现关闭选项卡功能。同年 10 月，Chrome 用户还被要求更新软件以修复两个漏洞。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
