梁宇鹏</p><p>在小米、荣耀、OPPO 纷纷宣布 " 进军高端 "" 对标苹果 " 这几年，蓝厂 vivo 似乎一直没有跟上友商们的节奏，不仅砍掉了高端品牌 NEX 系列，在售价万元的折叠屏市场也一直缺少动作。</p><p>近日，vivo 发布了旗下两款高端新机，vivo X Note 和 vivo X Fold，都是启用的新品牌。前者起售价达到了 5999 元，而后者则是 8999 元。</p><p>在此之前，vivo 只有 X 70 系列这一款高端新机在售，俗话说 " 独木难支 "，如今两款新机的发布也算是补全了产品线，跟上了小米、荣耀、OPPO 的节奏。</p><p>如今，这四家手机厂商基本形成了相同高端产品线——即两款直板手机，一款折叠屏手机。</p><p>显然，vivo 是来得最晚的那个。</p><p></p><div class="img_box" 有两种说法，往好的说前人趟过的坑，碰过的雷，都可以主动规避；往坏的说，前人已经把市场占领得差不多了，留给后来者的机会不多了。</p><p>在格外同质化的手机市场，vivo 的处境两者兼有。</p><p>小米历来都是 " 堆料 " 高手，过去两年，凭借小米 10、小米 11 系列，在高端市场初步站稳了脚跟。OPPO 和荣耀虽然目前市场份额不高，但产品力都不差，例如 OPPO Find X5 Pro 的自研 NPU、荣耀 Magic 4 Pro 支持高频调光等。</p><p>留给 vivo 发挥的空间，确实不多。</p><p>于是，我们看到了一款极为少见的，拥有 7 英寸大屏的 vivo X Note ——试图用大屏的差异化去竞争。</p><p>首先一点，不管产品力如何，vivo X Note 至少在屏幕大小上已经和其他高端机有了明显区分。毕竟，在名为 "Note" 手机里，最出名的大屏手机三星 Galaxy Note 系列也基本没有达到过这个尺寸。</p><p>不过，"7 英寸 " 的实际表现可能不如这个数字看上去的那么 " 震惊 "。</p><p>如今手机屏幕普遍在 6 英寸以上，三星 Galaxy S22 Ultra 作为 " 引领大屏手机风潮 " 的 Note 系列的继承者，也达到了 6.8 英寸。iPhone 13 Pro Max 也有 6.7 英寸。</p><p>vivo X Note 的 7 英寸虽然比它们更大，但并未超出太多。因此，对真正喜欢大屏手机的用户来说，vivo X Note 的大屏优势并不明显。</p><p>而对普通用户而言，vivo X Note 的 7 英寸又大大增加手机宽度，在使用体验上更加不便。</p><p>而在这块大屏之外，除了一个面积更大的屏下指纹解锁，vivo X Note 在其他配置上与目前小米、OPPO（包括一加）、荣耀并没有太大区别。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" "，不如说是手机市场内卷到极致后，vivo 为数不多的选择：vivo 需要的不是大屏，而是高端市场的差异化。<p></p><p>7 英寸大屏手机过去通常都出现在中低端手机上，因此，超大屏手机也常常有着 " 傻大粗 " 的标签。</p><p>高端市场只有华为 2018 年发布的 Mate 20X 一款，这款手机市场反应一般，很多人甚至从未听说过，在华为的高端产品序列里至今也未迎来后续更新款。</p><p>因此，如何扭转大屏手机带来的品牌劣势，也是 vivo X Note 需要克服的问题。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div zk_lrbgcolor_red"><div class="zk_lrbordercon"><strong>不夭折就是胜利</strong></div></div><div class="edi_oper"></div></div><p></p><p>从 2020 年开始，各家厂商都纷纷暗自较劲，剑指高端市场。荣耀独立后的 Magic 系列对标 iPhone；OPPO 开始自研芯片并让一加回归，合力冲击高端市场；小米则直接喊出 " 对标苹果 "。</p><p>但 vivo 却是 " 荣米 OV" 四家中 " 最低调 " 的。</p><p>vivo 在高端化路上虽起步较早，但却后续乏力，缺少品牌稳定性。在相继砍掉 Xplay 系列和 NEX 系列之后，vivo 再也没有拿出过令人眼前一亮的旗舰产品。</p><p>尽管 vivo 近年来在影像系统上下足功夫，左手微云台，右手蔡司联名，但始终是 " 雷声大雨点小 "。</p><p>根据 Counterpoint 的统计数据，2021 年 vivo 在全球高端市场（＞400 美元）的份额仅为 3%，华为、小米、OPPO 均高于 vivo。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div 在高端市场并没有站住脚跟，反而在电视广告、综艺节目、节日晚会等强势营销中，让自身被长久打上了 " 重营销、轻技术 " 的标签。<p></p><p>对比 " 荣米 OV" 的高端旗舰系列，小米是数字系列、MIX 系列和折叠屏系列；OPPO 是 Find X 系列、一加数字系列和折叠屏系列；荣耀则是数字系列、Magic 系列和折叠屏系列。</p><p>vivo 这次两机连发，显然只是跟上了同行们的脚步。</p><p>此前，第一手机界研究院院长孙燕飚曾对 ZAKER 科技表示，国产手机高端化需要大量的时间和资金投入，高端布局要用 5 年、10 年去规划坚持，才有可能成功。</p><p>因此，对现阶段的 vivo 来说，这两款新机能不能在高端市场 " 一炮而红 " 或许不是最重要的，更关键的是，不要让它们步入 Xplay 系列和 NEX 系列的夭折命运。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5"
