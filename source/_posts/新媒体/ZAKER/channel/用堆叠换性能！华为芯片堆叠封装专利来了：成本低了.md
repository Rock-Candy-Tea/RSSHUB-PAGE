<p>今日，从国家知识产权局官网获悉，华为公开了一种芯片堆叠封装及终端设备专利，申请公布号为 CN114287057A，<strong>可解决因采用硅通孔技术而导致的成本高的问题。</strong></p><p>专利摘要显示，该专利涉及半导体技术领域，其能够在保证供电需求的同时，解决因采用硅通孔技术而导致的成本高的问题。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202204/624b9f318e9f0915c706d779_1024.jpg" data-height="329" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/624b9f318e9f0915c706d779_1024.jpg" referrerpolicy="no-referrer"></div></div>具体来看，<strong>该芯片堆叠封装 ( 01 ) 包括：</strong><p></p><p>设置于第一走线结构 ( 10 ) 和第二走线结构 ( 20 ) 之间的第一芯片 ( 101 ) 和第二芯片 ( 102 ) ；</p><p>所述第一芯片 ( 101 ) 的有源面 ( S1 ) 面向所述第二芯片 ( 102 ) 的有源面 ( S2 ) ；第一芯片 ( 101 ) 的有源面 ( S1 ) 包括第一交叠区域 ( A1 ) 和第一非交叠区域 ( C1 ) ，第二芯片 ( 102 ) 的有源面 ( S2 ) 包括第二交叠区域 ( A2 ) 和第二非交叠区域 ( C2 ) ；</p><p>第一交叠区域 ( A1 ) 与第二交叠区域 ( A2 ) 交叠，第一交叠区域 ( A1 ) 和第二交叠区域 ( A2 ) 连接；</p><p>第一非交叠区域 ( C1 ) 与第二走线结构 ( 20 ) 连接；</p><p>第二非交叠区域 ( C2 ) 与第一走线结构 ( 10 ) 连接。</p><p>在前不久的华为 2021 年年报发布会上，华为轮值董事长郭平表示，未来华为可能会采用多核结构的芯片设计方案，以提升芯片性能。同时，<strong>采用面积换性能，用堆叠换性能，使得不那么先进的工艺也能持续让华为在未来的产品里面，能够具有竞争力。</strong></p><p>值得一提的是，这是华为首次公开确认芯片堆叠技术。也就是说，可以通过增大面积，堆叠的方式来换取更高的性能，实现低工艺制程追赶高性能芯片的竞争力。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202204/624b9f318e9f0915c706d77a_1024.jpg" data-height="250" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202204/624b9f318e9f0915c706d77a_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
