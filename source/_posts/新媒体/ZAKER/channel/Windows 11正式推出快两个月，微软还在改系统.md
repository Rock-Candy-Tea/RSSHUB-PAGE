<p>11 月 30 日消息，今年 6 月份，微软发布全新操作系统 Windows 11，并于 10 月份正式推送。但两个月时间过去，Windows 11 操作系统仍然是一项未完成的工作。目前的 Windows 11 系统较两个月前正式推送时增加了不少功能，修复了存在的部分 Bug，但仍然缺少微软所承诺的一些元素。</p><p>最近一段时间，微软技术团队部分专注于修复操作系统早期问题，部分人则在慢慢添加微软最初宣布的一些新功能。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/61a59786b15ec0501f5411ed_1024.jpg" referrerpolicy="no-referrer"></div></div><b>已经正式发布的功能</b><p></p><p>自 10 月份以来，微软在 Windows 11 操作系统添加、调整或修复的功能主要有：</p><p><b>针对锐龙处理器的性能优化：</b>微软和 AMD 都发布软件补丁，以解决 Windows 11 系统与新一代 AMD 锐龙处理器性能的兼容问题。独立测试证实，补丁使得锐龙处理器在 Windows 11 系统下的运行性能与 Windows 10 一致，但仍有改进空间。</p><p><b>部分 Bug 修复：</b>微软通过几次更新解决了包括任务栏渲染、虚拟打印错误以及系统崩溃等诸多问题。像 Windows 这样大而复杂的操作系统软件总是有很多问题，公开发布也给微软修复漏洞提供了大量用户数据。</p><p><b>外观更新：</b>最近 Windows 的更新又将蓝屏死机界面换回了蓝色，在系统中添加了更加活泼的表情符号。但这些小物件的调整只是说明，Windows 11 是一架尚未成型就起飞的飞机。</p><p><b>目前正在测试功能</b></p><p>微软有些应用和功能目前可以通过 Windows Insider Program 的 Beta 或 Dev 渠道进行测试，但它们还没有正式向公众发布。</p><p><b>支持 Android 应用程序：</b>微软在全新操作系统中引入了可运行 Android 应用的 Windows 子系统，允许 Windows 11 用户通过亚马逊应用商店或微软商店安装 相应 Android 应用程序。但其对用户硬件的要求更高，还要启动 BIOS/UEFI 虚拟化功能。目前在 Windows 11 上运行 Android 应用这一功能的预览版仅面向 Beta 频道的 Windows Insider 用户，只支持一小部分 Android 应用。</p><p><b>全新媒体播放器：</b>微软在 Windows 11 还加入了一个全新的媒体播放器，替代 Groove 音乐应用程序，后者是微软在 Windows 10 和 11 中的默认音乐播放器。这款应用被称为 " 媒体播放器 "，明显是向经典的 "Windows 媒体播放器 " 致敬。目前这款应用只对那些在 Dev 频道中注册的 Windows Insiders 用户开放。</p><p><b>适用于 Linux 的 Windows 子系统（WSL）： </b>和 Windows 10 一样，Windows 11 也有适用于 Linux 的 Windows 子系统。但与 Windows 10 中的版本相比，其增加了更多新特性。目前 Windows 11 用户可以在微软商店中下载预览版 WSL。该公司表示，最终微软商店中的预览版 WSL 将成为正式版。</p><p><b>麦克风状态和静音功能：</b>微软曾在今年 6 月份时展示过这项功能，但最终没有在 10 月份的最初 Windows 11 版本中上线。当用户在通话时，麦克风图标会出现在任务栏中。用户可以看到哪个应用程序正在使用麦克风，还可以通过点击图标打开静音或取消静音。这一功能目前仅支持 Dev 版 Windows 11 操作系统中的 Microsoft Teams 应用，但微软表示，其他应用程序也可以添加对该功能的支持。</p><p><b>Your Phone 应用界面调整：</b>Windows 11 操作系统中的 Your Phone 应用用户界面经过调整，通知功能变化最大。但这款应用的功能升级并不多，缺少对 iPhone 的某种支持，也无法在非三星、非微软的 Android 手机上打开应用程序或镜像手机屏幕。</p><p><b>尚未实现或应该实现的功能</b></p><p><b>深色模式：</b>微软在今年 8 月份承诺会对 Windows 11 进行部分重新设计，但仍然没有暗模式。目前也不清楚深色模式何时会成为现实。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 为统一微软的设计语言及其内置应用做了大量工作，但像记事本、写字板、设备管理器或注册表编辑器等内置管理工具都还停留在旧的设计层面上。微软要想完全致力于改变设计语言，就不能只是调整一些表面元素。</p><p><b>更灵活的任务栏：</b>微软应该让用户可以选择把任务栏放在屏幕左侧或右侧，也应该让用户通过拖拽文件到任务栏应用程序来打开文件。（辰辰）</p><div
