<p>Windows 11 正式推出也有半个月时间了，可能还是有很多人都没有升级，其中一个原因应该就是新系统除了界面上的大改以后，也没有什么特别的地方，而作为微软这一代视窗系统最重要的功能，支持 Android app 并没有随正式版到来，直到最近的更新后，微软才在测试版 Windows 11 上提供给大家。</p><p>在微软官方介绍中提到，Windows 11 的这个 Andorid app 支持功能，来自于微软特别开发的 Windows Subsystem for Android（简称 WSA），也就是个虚拟机系统，所以要求电脑开启 Hyper-V 这类虚拟化功能，微软表示只要是符合 Windows 11 运行要求的电脑，都可以完好支持这个 Andorid app 的功能。</p><p>虽然微软表示现阶段只有 beta 通道的 Windows 11 Build 22000.xxx 系列测试系统，才会收到 Andorid app 支持的第一个预览版推送，但对于正式版 Windows 11 的用户，已经有网友分享了 WSA 的文件包（通过 powershell 代码安装），感兴趣的玩家可以去搜索国内论坛来安装尝试下。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202110/61714a8c8e9f093376784dff_1024.jpg" data-height="720" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202110/61714a8c8e9f093376784dff_1024.jpg" referrerpolicy="no-referrer"></div></div>而由于目前这个测试仅提供给美国地区的 Beta 通道，所以其它地区的用户，就需要更改系统地区为美国并重启后，才能正常访问 Amazon App Store 来下载安装 Android 应用，不过也有网友反映是可以通过 adb 方式来安装 Apk 的，但需要注意的是，这只是个测试版功能，运行性能和 app 兼容性还有很多问题，大家暂时还不要用于正常的办公和游戏用途，<p></p><p>目前在 WSA 上运行的 Android 系统版本为来自 AOSP 的 Android 11，所以是没有 GMS 套件的，也就需要通过第三方应用商店来安装 app，预计在我们国内的 Windows 11 上，应该会换用本地化的应用商店吧，比如腾讯、百度什么的。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
