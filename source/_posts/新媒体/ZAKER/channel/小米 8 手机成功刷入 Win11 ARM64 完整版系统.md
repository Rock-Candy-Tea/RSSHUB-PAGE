<p>IT 之家 6 月 29 日消息 今天凌晨，微软面向 Dev 开发频道发布了第一个 Windows 11 Insider Preview 版本，即 Build 22000.51! 微软表示，随着我们在未来几个月内完成产品的定型，我们将与你一起验证这一体验。在这个早期预览中，你将可以尝试上周展示的许多新功能，但不是全部。</p><p>微软 Windows 11 Insider Preview 版本 Build 22000.51 镜像也已流出，其中包括 Windows 11 ARM 预览版本，有开发者已经在 Lumia 950 XL 刷入了 Windows 11 ARM 预览版本，现在有 IT 之家网友表示，搭载骁龙 845 处理器的小米 8 手机也已成功刷入 Windows 11 ARM64 完整系统，不过该机型上的 USB、触屏还不能正常工作，无法控制。</p><p>下面是小米 8 手机进入 Windows 11 ARM64 系统后的桌面，" 开始 " 菜单等在任务栏居中显示。>><a href="http://iphone.myzaker.com/zaker/link.php?pk=60db50798e9f090b977ef362&b=aHR0cHM6Ly93d3cuYmlsaWJpbGkuY29tL3ZpZGVvL2J2MU00NHkxcTdrYw==&bcode=3c2cab2b&target=_new" target="_blank">视频传送门</a></p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202106/60db4ee48e9f090aeb15f568_1024.jpg" data-height="1066" data-width="800" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202106/60db4ee48e9f090aeb15f568_1024.jpg" referrerpolicy="no-referrer"></div></div><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202106/60db4ee48e9f090aeb15f569_1024.jpg" data-height="1066" data-width="800" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202106/60db4ee48e9f090aeb15f569_1024.jpg" referrerpolicy="no-referrer"></div></div>项目地址 : https://github.com/edk2-porting/edk2-sdm845<p></p><p>微软官方此前早已经对 Windows 10 Mobile 停止支持，不过在 WoA 项目开发者的努力下，仍然会有开发者去为旧机型刷入新系统。</p><p>由于微软提供了 ARM 设备兼容的 Windows 10 版本，一群独立开发人员在过去几年里一直在维护一个非官方的 Lumia Windows on ARM 项目，这就是 Lumia WOA。现在微软发布了 Windows 11 ARM 预览版本，又有新的系统可以尝试了。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
