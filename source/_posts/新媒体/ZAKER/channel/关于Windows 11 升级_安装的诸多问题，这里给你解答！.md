<p>Windows 11 正式版已经于 10 月 5 日全面上市，我们昨天也推送了电脑报首款 Windows11 系统笔记本宏碁蜂鸟 · 未来的测评（点击这里查看机型测评和 Windows 11 简单体验）。不少同学看了之后，对 Windows 11 全新的 UI 界面和丰富的新功能很感兴趣，但同时又存在着一些疑惑。比如哪些电脑可以升级 Windows 11？什么时候可以升级？如果 PC 不符合硬件规格，可以继续使用 Windows 10 吗？牛叔今天就来解答这些问题。</p><p><strong>Q：我现在想买一台电脑，以后可以安装 Windows 11 吗？</strong></p><p>A：目前出售的大多数 Windows 10 电脑都可以升级到 Windows 11。要升级到 Windows 11 的电脑，它必须满足最低硬件规格和某些功能所需的特定硬件，微软官方的升级推出计划仍在敲定中，计划于 2021 年底开始并持续到 2022 年，具体时间将因设备而异。</p><p>而根据微软官方，升级 Windows 11 的最低硬件要求如下所示：</p><p>· 处理器：1GHz 或更快的支持 64 位的处理器（双核或多核）或系统单芯片 ( SoC ) </p><p>· 内存：4GB RAM</p><p>· 存储：64GB 或更大的存储设备</p><p>· 系统固件：支持 UEFI 安全启动</p><p>· TPM：受信任的平台模块 ( TPM ) 2.0 版本</p><p>· 显卡：支持 DirectX 12 或更高版本，支持 WDDM 2.0 驱动程序</p><p>· 显示器：对角线长大于 9 英寸的高清 ( 720p ) 显示屏，每个颜色通道为 8 位</p><p>如果你看不太明白上面的规格，也没关系，牛叔用一句通俗易懂的话解释：以近几年（比如四年以内）主流价位的笔记本为例，几乎都能升级最新的 Windows 11 系统，新系统对硬件要求不高。</p><p>另外，大家也可以下载安装 " 电脑健康状况检查 " 应用（微软官方推出的工具）来检测自己的电脑是否符合升级要求。这款应用使用非常简单，在微软官网下载之后即可安装，打开之后就会出现下面的页面，点击 " 立即检查 " 就能查询结果，整个过程耗时大概也就是 1 分钟的时间。这同样适用于检查老旧电脑能否升级。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div 还能继续使用吗？</strong><p></p><p>A：当然可以。根据微软的官方回答，微软承诺将对 Windows 10 的支持延续到 2025 年 10 月 14 日。</p><p><strong>Q：什么时候可以升级 Windows 11？</strong></p><p>A：升级推送计划目前仍在完善当中，对于大多数目前在用的设备，微软计划于 2022 年初推出升级。当然，满足升级到 Windows 11 条件的 Windows 10 电脑不一定同时收到升级推送，要查看你的电脑是否符合升级条件，可以下载并运行电脑健康状况检查应用来看看。</p><p><strong>Q：怎样知道升级已在我的 Windows 10 电脑上可用？</strong></p><p>A：在 Windows 11 升级推送期间，微软将进行一些后台测试和对特定电脑进行验证。Windows 更新将向用户指出其电脑是否符合升级条件以及何时可以升级。同时，用户可以进入 " 设置 "/"Windows 更新 " 中进行查看。比如牛叔用的这台 i7 9750H+RTX 2060 组合的笔记本，打开该界面之后，就已经看到支持升级到 Windows 11 的提示。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div 要付费吗？升级持续多长时间？</strong><p></p><p>A：免费升级。只要符合升级条件，即可免费升级。对于符合要求的系统，免费升级没有具体的结束日期。但是，微软保留最终终止支持免费升级的权利，结束日期将不会早于升级广泛推出之后一年。</p><p>另外，升级后的免费升级的版本与用户 Windows 10 设备上的版本相同。如果用户使用的是 Windows 10 家庭版，则将升级至 Windows 11 家庭版。</p><p><strong>Q：升级和更新有何区别？</strong></p><p>A：更新可以在 Windows 的版本中包括缺陷修复和安全修复以及新功能，更新在一年中会不断推出。而升级是改变版本。例如，从 Windows 10 升级到 Windows 11，或从 Windows 家庭版升级到 Windows 专业版。</p><p>同时，下载并安装 Windows 11 很可能比正常的 Windows 10 功能更新时间长。在下载 Windows 11 的同时用户可以继续使用电脑，然后可以选择在不使用电脑的时候进行安装。</p><p><strong>Q：升级 Windows 11 之后 , 还能回到 Windows 10 吗？</strong></p><p>A：可以。在安装了 Windows 11 升级之后，有 10 天时间用户可以回退到 Windows 10，同时保留用户升级时随之迁移的文件和数据。10 天之后，要恢复到 Windows 10，用户需要备份数据，然后执行 " 干净安装 "。</p><p><strong>Q：如果我的电脑运行的是 Windows 10 S 模式，可以升级到 Windows 11 吗？</strong></p><p>A：如果你的电脑满足最低硬件要求，那么 Windows 10 家庭版 S 模式可以升级到 Windows 11 家庭版 S 模式。如果你的 Windows 10 电脑运行的是专业版 S 模式，需要退出 S 模式才能升级到 Windows 11 专业版。Windows 11 专业版不支持 S 模式。</p><p><strong>Q：Windows 11 是否比 Windows 10 占据更多电脑空间？</strong></p><p>A：不是。Windows 11 和 Windows 10 需要的硬盘空间大致相同。但是在升级过程中需要额外的空间。Windows 将在升级完成 10 天之后清理所占用的这些额外硬盘空间。</p><p>最后，关于升级系统，还有一些细节需要注意：</p><p>Windows 11 升级前的注意事项</p><p>• 备份重要的用户文件（所有磁盘数据）</p><p>• 先去电脑品牌官网确认是否有对应的新版驱动，是否通过兼容性检测</p><p>• 暂时卸载所有第三方的防毒软件</p><p>• 创建 2 个以上的管理员账户（防止当前用户配置文件损坏无法进入）</p><p>• 尽量不要使用超级管理员账户登录系统来进行升级</p><p>• 备份系统盘的 BitLocker 的密码，或者先解锁再升级</p><p>• 确定系统磁盘有足够的空间</p><p>（编辑：阿勇）</p><div id="recommend_bottom"></div><div
