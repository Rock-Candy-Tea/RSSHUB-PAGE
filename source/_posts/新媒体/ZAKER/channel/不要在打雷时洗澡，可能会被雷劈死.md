<p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202209/63121fd88e9f09101e62a414_1024.jpg" data-height="721" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202209/63121fd88e9f09101e62a414_1024.jpg" referrerpolicy="no-referrer"></div></div>图片来源：pixabay<p></p><p>撰文｜James Rawlings</p><p>翻译｜曾欣欣</p><p>审校｜王昱</p><p>全球每年约有 2.4 万人死于闪电，另有 24 万人受伤。英国气象局近日已经多次发布 " 黄色雷暴警告 "，强调可能会频繁出现的雷电天气。虽然被闪电击中的几率很低，但知道如何在雷雨期间保护自身安全是很重要的。</p><p>对于雷雨天气，大多数人都熟悉的基本安全防范措施，比如避免站在树下或靠近窗户，不要使用有线电话（手机除外）。但你知道，在雷雨天也应该避免淋浴、洗澡或洗碗这些跟水有关的活动吗 ?</p><p>要理解其中的原因，你首先需要了解一下雷暴和闪电产生的机制。</p><p>雷暴的形成有两个基本因素：湿气和上升的暖空气——这与夏季密切相关。高温和潮湿会产生大量的潮湿空气，它们上升到大气中，会形成雷暴。</p><p>具体来说，这是因为云中含有数百万的水滴和冰晶，这些水滴的相互作用带来了闪电。上升的水汽与下落的冰晶相撞摩擦，使冰晶带上负电荷，而水滴则带上正电荷。在一场雷暴中，云层就像巨大的范德格拉夫起电机，将正负电荷分开，在云内形成巨型的电荷分离。</p><p>当雷云在地球上空移动时，它们在地面上产生了相反的电荷，这就是吸引闪电击向地面的原因。雷暴会通过在正负区域之间放电来平衡它的电荷，因为这种放电的路径通常是阻力最小的，所以导电能力强的东西 ( 比如金属 ) 更容易在暴风雨中被击中。</p><p>对雷雨天气最有用的建议是：打雷时，进屋去。然而，这并不意味着你可以完全远离雷暴。在雷雨天，有一些室内活动的风险几乎与待在室外一样大。</p><p>阻力最小的路径除非你坐在室外 ( 浴缸里 ) 泡澡或在雨中淋浴，否则你被闪电击中的可能性非常小。如果闪电击中你的房子，电流会沿着对地面阻力最小的路径传播。像金属线或水管里的水这样的东西为电流提供了一个方便的传导路径，使其能够到达地面。</p><p>淋浴提供了两种东西 ( 水和金属 ) ，是电流传导的理想路径。它可能会让你的淋浴变得不那么安全。美国疾病控制与预防中心强烈建议人们在雷雨期间避免所有与水有关的活动——甚至包括洗碗，从而降低你被雷暴袭击的风险。</p><p>在雷暴期间，还有其他风险需要注意。其中一个可能看起来不太明显的是靠在混凝土墙上。虽然混凝土本身的导电性不强，但如果它被金属梁 ( 称为 " 钢筋 " ) 加固，就可以为闪电提供传导路径。与此同时，还要避免使用任何插在电源插座上的东西 ( 电脑、电视、洗衣机、洗碗机 ) ，因为所有这些都可能为雷击提供路径。</p><p>根据一般的经验法则，如果你能听到远处的雷声，那么说明你离风暴足够近了，即使没有下雨，闪电就能到达你身边。雷击可以发生在离母风暴 16 公里远的地方。通常情况下，在听到最后一声惊雷后的半小时后再去淋浴是比较安全的。但雷雨通常喜欢在最后留下一个大的，你一定不想成为最后一部分的 " 烟花 " 吧 !</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
