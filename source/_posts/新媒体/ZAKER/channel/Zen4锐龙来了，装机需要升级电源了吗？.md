<p>8 月 29 日，AMD 正式发布了基于 Zen4 架构的 Ryzen7000 系处理器 ，而在这个月 27 日，我们就能买到它了。关注这款换代产品的朋友不妨关注钛师父的首发评测。而相信准备了充足预算的 DIY 发烧友不会错过装机的机会。那么问题来了，Zen4 装机要升级电源吗？</p><p></p><div class="img_box" id="id_imagebox_0" 吧<p></p><p>对于不超频的玩家，那么理解 AMD 这一代处理器的功耗要求的话，只需要复习下 TDP 和 PPT 功耗的概念。</p><p>TDP 是我们很早以来就习惯的一种功耗计算概念，是 Thermal Design Power 热设计功耗的缩写。一般是指 CPU 在运行时，可产生的最大热量，单位是瓦特，主要用于和处理器匹配散热器的依据。不过 Intel 有自己的定义，即 " 热设计功耗 ( TDP ) 以瓦特为单位，表示所有活动内核在 Intel 定义的高复杂性工作负载下，以基本频率运行时消耗的平均功率。"</p><p>无论如何，它定义的其实是一种发热功耗，但实际上 CPU 的发热并不是功耗的全部，因为还有很大一部分的功耗实际上是用于内部的晶体管电路的运算。这些运算行为导致的元件发热，以及 CPU 本身的供电传输和传输损耗中散发的功耗发热是 TDP，但运算耗能本身并不全在 TDP 之内。</p><p>因此，TDP 作为散热器的选购指导是没有错的，实际上现在基本上知名品牌的散热器，都会依据 TDP 测试出自己的散热支持 TDP 功率，作为玩家选购的参考。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202209/63186da48e9f0908ed611d74_1024.jpg" referrerpolicy="no-referrer"></div></div><div class="img_box" id="id_imagebox_2" onclick><div src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/63186da48e9f0908ed611d75_1024.jpg" referrerpolicy="no-referrer"></div></div>例如玩家群体中名气很大的大霜塔 V5 就标识了最大支持 TDP 为 260W<p></p><p>但是既然 TDP 并不是一个电耗单位，那么电源选择当然不能依据它了。特别是自从 Intel 发明 " 睿频 "、PL1、PL2 功率后，CPU 的实际功率已经可以大大超过 TDP 功率。而 AMD 在 Zen 架构以后，引入了 Precision Boost 精准频率提升技术（PBO），以及 XFR 自适应动态扩频技术，同样也让 CPU 不仅在人工超频状态大大超过 TDP，甚至在自动超频（例如利用 AMD Ryzen Master）时功耗也可以超过 TDP。</p><p>因此 AMD 引入了 PPT 功率，也就是封装跟踪功率（"PPT"，Package Power Tracking）。PPT 功耗的阈值就是允许在 CPU 插座的供电通路上允许的插座功耗。具有高线程数 " 重负载 " 线程的应用可能会遇到 PPT 限制，可以通过提高 PPT 限制来缓解提高和挖掘 Zen 架构 CPU 的性能。</p><p>而对于 Zen2 和 Zen3 架构的锐龙 CPU 来说，在额定为 105W TDP 处理器的主板上，Socket AM4 的 PPT 至少 142W。</p><p>在额定为 65W TDP 处理器的主板上，Socket AM4 的 PPT 是至少 88W。</p><p>而 PPT 功率也只能说是接近 CPU 的实际耗电。为什么？因为输送到 CPU 插座的电力，在经过主板 CPU 供电单元的 VRM 电路组时候，又产生了一层损耗。根据 CPU 实际负载情况，在 PPT 功率上加 10% 到 30%，才是真正的 CPU 部分的实际功耗！</p><p>如果超频：你还需要知道 EDC 和 TDC</p><p>而如果是超频，那么除了 PPT 以外，你还需要知道 Zen 架构 CPU 的 EDC 和 TDC 的概念。</p><p>无论你是在 AMD Ryzen Master 超频软件上软超，还是在 BIOS 中打开 PBO 限制 " 硬 " 超，都会看到除了 PPT 功耗外的 EDC 和 TDC 限制。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/63186da48e9f0908ed611d77_1024.jpg" referrerpolicy="no-referrer"></div></div>热设计电流（"TDC"）。在热限制的情况下，特定主板的 VRM 单元可以提供的最大电流（安培）。在 Zen2 和 Zen3 上，额定为 105W TDP 处理器的主板上，TDC 默认值是至少 95A。在额定为 65W TDP 处理器的主板上，TDC 默认值是至少 60A。<p></p><p>电气设计电流（"EDC"）。特定主板的 VRM 在<strong>极限峰值</strong>条件下可在短时间内提供的最大电流（安培）。同样对于 Zen2 和 Zen3 架构，额定为 105W TDP 处理器的主板，默认值为 140A。</p><p>在额定为 65W TDP 处理器的主板上，默认值是 90A。</p><p>而在 Zen2 和 Zen3 架构下，目前 AMD 为软超频用户的设置的默认限制是：PPT 功耗 500W，TDC=200A，EDC=220A。</p><p>但是注意，并不是说你把这三个值推到最高，你就能获得最高的超频性能。不同的 Zen 架构（至少目前的 Zen3）处理器，其 CCD 等对于 TDC 和 EDC 的敏感性不同，温度墙也不同，盲目推高其中一个只能使得另一个或两个参数以及 CPU 温度达到阈值，反而性能下降。精细灵活调整三者关系才是获得最佳稳定性能的超频策略。</p><p>Ryzen7000：170W？不，是 230W！</p><p>接下来就是新发布的 Ryzen7000 系的功耗要求了。AMD 最初表示，Socket AM5 将有 170W 的 PPT 限制。然而，AMD 后来澄清说，它分享的原始数字是错误的。相反，现在看到更正后的处理器 TDP 为 170W，如 Ryzen 7000，这比目前 Ryzen 5000 处理器的 105W TDP 限制有了很大的提高。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" data-original="http://zkres1.myzaker.com/202209/63186da48e9f0908ed611d78_1024.jpg" data-height="621" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/63186da48e9f0908ed611d78_1024.jpg" referrerpolicy="no-referrer"></div></div>此外，AM5 PPT 实际上是 230W。这比上一代 Ryzen 5000 的 142W 限制有显著增加。<p></p><p>总的来说，这一增长代表了比 AMD 目前的 Zen3 桌面旗舰产品增加了 65W TDP 和 88W PPT。</p><p>同时 AMD 也表示， TDP*1.35 是 Zen 架构时代处理器 TDP 与 PPT 的标准计算方法，Zen4 架构也不例外（170*1.35=229.5）。因此，如果以后有新的 TDP 的 Ryzen CPU 面市，大家也可以据此来计算功耗。</p><p>因此带入到非超频用户或默认预设文件软超频用户的需求，则实际 CPU 稳定供电需求为 230W（PPT）*1.1 或 1.3，也就是主板 VRM 供电需求上限为 300W。加上内存和板载声卡显卡、M.2 固态硬盘等的耗电，不包括显卡在内，400W 的电源是合适的。</p><p>而本世代非旗舰光追显卡 TDP 普遍在 300W 左右。<strong>因此对于 Zen4 架构独显平台，700W 及以上的电源将成为主力旗舰 U 的配机刚需。</strong></p><p>不过低功耗电源难道从此退出市场？也不尽然。从一张泄露的 AM5 散热器设计要求表我们可以推算出，AMD 后续将在 Zen4 架构中推出 45W、65W、95W、105W、120W 的 TDP 的产品，而这些入门和中端产品，仍然可以使用目前广大的 400~600W 中低功耗电源配机。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_6"
