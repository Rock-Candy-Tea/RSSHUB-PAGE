<p>在昨晚的三星新品发布会上，Galaxy S22 系列手机正式与我们见面，这些新手机开箱即运行基于 Android 12 的 One UI 4.1 系统，除此之外新系统还加入了最新的虚拟内存技术。</p><p>三星的虚拟内存技术名为 RAM Plus，三星在 One UI 4.1 中对 RAM Plus 技术进行了不小的调整，虚拟内存容量从之前固定的 4GB 调整了多项可选，包括 2GB、4GB、6GB 以及 8GB。</p><p>虚拟内存技术并不少见，国内大部分手机品牌均已支持此技术，该技术的工作原理是使用手机存储空间来扩展内存的容量，已达到提高手机多程序同时运行能力的目的。</p><p>据悉，虽然 RAM Plus 是 One UI 4.1 的附带功能，但并不是每一款可以升级 One UI 4.1 的手机都会支持，具体还要看三星的更新策略。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202202/620482b28e9f0947664136b8_1024.jpg" data-height="926" data-width="424" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202202/620482b28e9f0947664136b8_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
