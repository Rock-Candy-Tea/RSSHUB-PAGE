<p>距离 Windows 11 操作系统正式推出已经不到两周时间了（也就是很快国庆假期了），微软最近也重新上线他们提供大家检测自己电脑是否符合升级到 Windows 11 的要求，不过这次新工具似乎和上次一样 " 无理 "，一些旧电脑仍可能会被告知无法升级到 Windows 11。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202109/614b00338e9f0955b20b6193_1024.jpg" data-height="675" data-width="1200" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202109/614b00338e9f0955b20b6193_1024.jpg" referrerpolicy="no-referrer"></div></div>在微软给出的最低系统要求中，他们只要求 CPU 是 "1 GHz 或更快的支持 64 位的处理器（双核或多核）"，像是 Core i7-7700K、6700K 这些当年的高端型号无论是规格，还是实际性能都是完全符合，所以也是让用户感到很困惑的，微软此前有表示他们后续可能会完善硬件的支持，上个月也确实新增了对 Skylake-X 架构 CPU 的支持，但在目前阶段，微软对老硬件配置还是不够友好的。<p></p><p>不过微软也证实了，如果用户选择自行通过 ISO 文件来安装 Windows 11 也是可行的，只是后续的 Windows Update 也需要自己手动升级，因为微软所说的 " 不支持 " 实际上是指他们不保证用户能完善体验 Windows 11，而是不代表电脑真的完全无法安装和运行使用，但实际情况如何，还是要留到 10 月 5 日正式版推出后才能知晓了。</p><p>当然如果你的电脑实在太旧了，确实是需升级换代的话，可以选择微软与 OEM 厂商合作推出的 Windows 11 Ready 机型，而如果是装机的话，那买最新一代 Intel 或 AMD CPU，以及确保主板支持 TPM 2.0 模块就好，另外微软自己也会在 9 月 22 日（美国时间）带来新一代 Surface 系列电脑，也是可以关注一下的。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
