<p><strong>近日，<strong>微软表</strong>示将会在后续更新中对 Windows 11 的右键菜单进行优化调整。</strong>这可以说是几乎所有 Windows 11 用户最期待的更新内容之一。在 Windows 11 中，微软对 Windows 一脉相承的右键菜单进行了大量的重新设计，新的右键菜单看起来更简洁，也更美观，<strong>但这种简洁和美观却是建立在对用户体验的破坏性影响上的。</strong></p><p><strong>在 Windows 11 的右键菜单中，大量的功能被收纳进了二级菜单中</strong>，这导致很多用户都反馈称 Windows 11 的右键菜单使用困难。</p><p>快科技获悉，在 Windows 11 的后续版本更新中，微软有计划改动右键菜单的二级菜单，并启用全新的二级菜单设计。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202111/6191cd0f8e9f09070c61e360_1024.jpg" data-height="419" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/6191cd0f8e9f09070c61e360_1024.jpg" referrerpolicy="no-referrer"></div></div>Windows 11 新右键菜单设计示意图<p></p><p>微软表示新的二级菜单将会提升呼出的效率，从而解决用户对于右键菜单操作缓慢且繁琐的反馈。</p><p>此外，新的二级菜单将采用和 Windows 11 其他 UI 统一的设计风格，这使得新的二级菜单变得更加干净整洁，<strong>但问题是当前测试版本的新二级菜单会无视用户自定义的强调色，而是坚持显示蓝色。</strong></p><p>此次微软对于右键菜单的改动将会先对 Dev 频道的用户进行推送更新，在经过一段时间的测试之后再对其他用户进行推送。</p><p>虽然微软官方并未透露这个过程会耗费多久的时间，<strong>但是有人认为此次更新将会在 2022 年 10 月的 Windows 11 22H2 版本更新中推送。</strong></p><p>除了对右键菜单的优化外，<strong>微软还计划对 Windows 11 中频繁出现的由于任务栏和文件资源管理器导致的崩溃进行修复</strong>，这个修复补丁将会在未来两周的更新中与用户见面。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
