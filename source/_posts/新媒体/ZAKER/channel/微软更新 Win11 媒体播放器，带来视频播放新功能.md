<p>IT 之家 5 月 7 日消息，微软在 2021 年 11 月宣告了适用于 Windows 11 的新版 Media Player，并于 2022 年初向公众发布。</p><p>微软之前为 Windows Media Player 带来了一些测试版更新，其中也包含新功能。该更新现已面向 Windows 11 所有用户提供，感兴趣的小伙伴可以在 Microsoft Store 上下载。</p><p>微软之前宣布用 Windows Media Player 取代了 Groove Music，而随着该公司停止使用旧版 Windows Media Player 后，就一直有大量用户抱怨内置服务不好用。</p><p>这个新版本的媒体播放器是 Groove Music 的重制版，但微软希望让它成为经典的 Windows Media Player 继任者。</p><p>随着今天更新的发布，Windows 11 全新的现代媒体播放器也得到了重大改进。IT 之家知悉，目前新版 Media Player 版本号是 11.2203.30.0，从目前来看有两个显著变化：</p><p>歌单：</p><p>如图所示，你现在可以体验新的音乐库。</p><p></p><div class="img_box" id="id_imagebox_0"
