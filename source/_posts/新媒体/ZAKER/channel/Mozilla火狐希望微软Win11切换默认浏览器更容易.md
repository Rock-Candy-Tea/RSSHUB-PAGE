<p>IT 之家 4 月 5 日消息，据 onMSFT 报道，尽管微软在 Windows 11 中对其默认浏览器设置开发做出了改变，但 Mozilla 等一些公司认为，" 还可以做得更多 "。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202204/624c043c8e9f09144a7bc68e_1024.jpg" data-height="598" data-width="793" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202204/624c043c8e9f09144a7bc68e_1024.jpg" referrerpolicy="no-referrer"></div></div>微软通过 Windows 11 补丁更新让默认浏览器设置体验更接近于旧版 Windows 选项精简过程，但据 Mozilla 称，这仍然是一个相当复杂的过程。<p></p><p>" 微软改善用户的默认浏览器设置体验的变化是朝着正确方向迈出的一步。但是在尊重 Windows 上默认浏览器选择方面还可以做得更多。人们应该有能力简单轻松地设置默认状态，所有操作系统都应该提供官方开发对默认状态的支持。在实践中，我们还希望看到在减少将一个新的浏览器设置为默认状态所需的步骤方面取得进展，以及开放和提供应用程序设置默认状态所需的 API，供其他微软应用程序使用。"</p><p>并非只有 Mozilla 认为，微软正在用不必要的障碍来搅浑水，以获得人们在 Windows 7、8 和 10 中更习惯的默认浏览器体验。</p><p>在过去一年左右的时间里，微软已经从用户和第三方浏览器开发者那里获得了关于其对默认设置的奇怪做法的意见反馈。</p><p>与微软过去的做法相反，该公司似乎鼓励其 Windows 团队将默认浏览器设置过程分解为广泛和具体的选择，涉及到用户了解什么是 FTP、.svg、.xml、IRC 和许多其他协议，而不是他们发现微软 Edge 仍然在随机的场合中出现。</p><p>虽然微软可能将其 Windows 11 默认浏览器的变化作为细化套餐来推销给那些受委托为特定协议分配特定应用程序的 IT 管理员，但普通用户通常希望他们的默认选择设置仍然是简单可选的。</p><p>与大多数软件的发展一样，Windows 11 仍然是一个不断发展的平台，也许微软将继续精简默认浏览器体验，与第三方合作伙伴和用户达成某种平衡。</p><p><strong>相关阅读：</strong></p><p>《微软 Win11 正式版终于支持一键更改默认浏览器》</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
