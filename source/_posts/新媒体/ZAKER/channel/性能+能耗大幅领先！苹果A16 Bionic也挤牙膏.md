<p>2022 年的苹果发布会比往年来得更早，除了 iPhone 14、iPhone 14 Max 继续使用 A15 Bionic 外，iPhone 14 Pro 和 iPhone 14 Pro Max 升级到 A16 Bionic，但关于芯片部分少得可怜，也从侧面反映了 A16 Bionic" 乏善可陈 "。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202209/63297cf58e9f093017546a1c_1024.jpg" data-height="365" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/63297cf58e9f093017546a1c_1024.jpg" referrerpolicy="no-referrer"></div></div>在发布会上，苹果有意避开 A16 Bionic 的性能提升，转而关于能效 ( power efficiency ) 、显示 ( display ) 和摄像头 ( camera ) 。苹果 A16 Bionic 采用 4nm 工艺，虽然苹果明确是 N4 ( 第一代 4nm ) 还是 N4P ( 4nm 升级版 ) ，但普遍认为苹果使用 " 满血版 " 的 N4P。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202209/63297cf58e9f093017546a1d_1024.jpg" data-height="599" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/63297cf58e9f093017546a1d_1024.jpg" referrerpolicy="no-referrer"></div></div>在硬件规格上，苹果 A16 Bionic 拥有 160 亿个晶体管，比 A15 Bionic 的 150 亿有提升。CPU 部分依然是 2 大核 +4 小核的设计 ;5 核心的 GPU，内存带宽提升 50% 的 ;17 TOPS 标称性能的 16 核 Neural Engine AI 单元 ; 用以支持或显示屏幕 ( LTPO ) 最低 1Hz 刷新率、Always-On 特性、Dynamic Island 动画抗锯齿、屏幕更高的峰值亮度等功能的新显示引擎 ( Display Engine ) ; 新的 ISP 不仅能够支持 4800 万像素摄像头，联合 CPU、GPU、Neural Engine 和 ISP 异构核心，可为每张照片带来最高 4 万亿次的操作。<p></p><p>与 A15 Bionic 的 150 亿个晶体管相比，A16 Bionic 仅 10 亿个晶体管的提升，显然无法满足全方位提升 SoC 性能的需求。这一点在发布会上也得到证实，苹果将更多的精力放在能效、显示和摄像头上。</p><p>不过得益于采用台积电 4nm 工艺，CPU 部分的频率有所提升，理论的跑分成绩也有提高。根据目前网上的信息可以看到，A16 Bionic 的单核跑分在 1870 左右，作为对比的 A15 Bionic 的粉在 1720 左右，提升幅度约 8.7%，与频率 ( 3.23GHz～3.46GHz ) 提升幅度相差无几。</p><p>虽然性能提升幅度有限，但苹果 A16 Bionic 在能效方面的提升巨大，是苹果真正的优势。苹果表示 A16 Bionic 的 2 个高性能核心 ( 大核心 ) 相比于 A15 更快，同时降低 20% 的功耗，为 CPU 带来更稳定的持续性能输出，并让手机在高负载下更省电 ; 在能效核心 ( 小核心 ) 方面，相同性能下是比拥有竞品更好的能效，功耗只有竞品的 1/3。</p><p>GPU 方面，苹果表示内存带宽增加 50%，并表示 " 对于图形性能要求高的游戏很好 "。换句话说，苹果很有可能为 iPhone 14 Pro 升级 LPDDR5 运行内存，进一步释放 GPU 性能潜力。但考虑到仅一句带过，GPU 方面的改进和提升，应该与 CPU 相同，纯粹通过提升频率拉高性能，并通过更大的内存带宽释放性能。</p><p>总的来说，与上一代 A15 Bionic 相比，苹果确实在 A16 Bionic 上 " 挤牙膏 "，但依然大幅度领先安卓竞品。至于一贯强大的能效方面，苹果的领先幅度只会更大。至于 GPU 部分，性能提升与 CPU 相似，通过拉高频率和带宽提升性能，并借助 4nm 工艺提升能效比。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202209/63297cf58e9f093017546a1e_1024.jpg" data-height="408" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/63297cf58e9f093017546a1e_1024.jpg" referrerpolicy="no-referrer"></div></div>iPhone 14 Pro 芯片的第二个核心是显示上，iPhone 14 Pro 带来灵动岛、ProMation 和锁定屏幕常亮不熄屏等功能。不过整体上更偏向于用户体验，芯片上几乎没有可讲的内容，更多的还是通过芯片控制屏幕，以实现更好的能效比。<p></p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" data-original="http://zkres2.myzaker.com/202209/63297cf58e9f093017546a1f_1024.jpg" data-height="389" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202209/63297cf58e9f093017546a1f_1024.jpg" referrerpolicy="no-referrer"></div></div>苹果 A16 Bionic 的第三个着力点是摄像头，iPhone 14 Pro/Max 主摄换用 4800 万像素的 CMOS，ISP 引擎的数据计算量翻两番 ( 4 倍 ) ，ISP 需要做包括 Deep Fusion 之类多帧堆栈之类的操作，数据量级和 4800 万就不一样，ISP 的性能提升自然必不可少。<p></p><p>iPhone 14 Pro 系列支持以 Cinematic 模式录制 4K 24fps 视频，杜比视界 HDR 最高 4K 60fps 视频拍摄以及新的 Action 运动模式 ( 防抖 ) ，都需要算力的支持。</p><p>编辑点评：从整体规格来看，苹果 A16 Bionic 有 " 挤牙膏 " 的嫌疑，CPU、GPU 部分的性能提升有限，但依然大幅领先同时期竞品。同时，台积电 4nm 给 SoC 带来更高的核心频率和更低的功耗，进一步提升 SoC 的能耗比。至于 ISP、NPU 方面的提升，更多的事托 4800 万像素 CMOS 带来的计算量提升，但亮点依然是能效。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
