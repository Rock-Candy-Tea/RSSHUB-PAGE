<p>大家在日常生活中可能都听过这样一个说法：" 公蚊子不吸血，吸血的都是母蚊子 "，事实也确实如此，在自然界中，公蚊子一般偏爱甜甜的东西，比如糖水、花蜜和果汁等等，而母蚊子则总是趁我们不备，一口一个大包，让人不胜其烦。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div 图片来源：eyeonnature.files.wordpress.com ) </p><p>可是，大家知道为什么公蚊子不吸血吗？</p><p><strong>公蚊子：不是不想吸，而是吸不了</strong></p><p>俗话说的好，" 工欲善其事，必先利其器 "，这话用在蚊子身上可以说再合适不过了。</p><p>具体而言，成年蚊子的身体可以分为头胸腹三个部分，其中蚊子头部的口器 ( Proboscis ) 对蚊子食谱的影响不言而喻。一般而言，雌蚊的口器又尖又长，可以刺穿人或者动物的皮肤然后吸食血液，相比之下，雄蚊的口器则较为短小，无法刺穿皮肤，故而雄蚊也不以血液为食，而是转而通过吸食花蜜，果汁来维生。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202205/629422d18e9f0913522bae1d_1024.jpg" data-height="425" data-width="586" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202205/629422d18e9f0913522bae1d_1024.jpg" referrerpolicy="no-referrer"></div></div>雄蚊 ( 左 ) 和雌蚊 ( 右 ) 在口器上的区别<p></p><p> ( 图片来源：vivadifferences.com ) </p><p>如果我们把雄蚊子比作上门行窃的小偷的话，缺了作案工具，自然是没办法完成犯罪过程的。但问题就来了，犯罪未遂可不代表没有贼心，同样地，公蚊子是因为没办法吸上血所以不吸血。<strong>那么当血液唾手可得时，公蚊子会吸吗？</strong></p><p>对于致倦库蚊 ( Culex quinquefasciatus ) 的雄蚊而言，答案或许是肯定的。</p><p><strong>致倦库蚊</strong>隶属于蚊科库蚊属，库蚊和伊蚊、按蚊三属被称为蚊界的三大巨头，其中致倦库蚊作为中国南方最常见的 " 家蚊 " 之一，会在每年的 3~11 月份到各家各户游荡，尤其喜爱在黄昏、夜里出来觅食，大家就算不认得，也大概率被致倦库蚊叮咬过。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202205/629422d18e9f0913522bae1e_1024.jpg" ( 图片来源：entnemdept.ufl.edu ) </p><p>2016 年，一项发表在《生理学前沿》的论文发现，<strong>致倦库蚊的雄蚊在有机会吸血时，竟然也是毫不拒绝的</strong>。</p><p>具体而言，科学家们先是给雄蚊们提供了一个带血的棉球以了解它们会不会闻风而来，结果显而易见，雄蚊们拒绝不了这份诱惑；进一步，研究者们给雄蚊们增加了一点点的难度，给带血的棉球包上一层封口膜 ( Parafilm ) ，结果表明，雄蚊们依旧乐此不疲地围着棉球吸血。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" data-original="http://zkres1.myzaker.com/202205/629422d18e9f0913522bae1f_1024.jpg" data-height="299" data-width="756" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/629422d18e9f0913522bae1f_1024.jpg" referrerpolicy="no-referrer"></div></div>雄蚊们围着带血的棉球 ( 左 ) 和被封口膜包裹的带血棉球 ( 右 ) 吸血<p></p><p> ( 图片来源：参考文献 ) </p><p>为了搞清楚雄蚊是更喜欢血液，还是更喜欢糖水，科学家们设计了一个特殊的装置对雄蚊的觅食偏好进行研究，在这个装置中，沾了糖水和血液的棉球被分别放在笼子两侧，雄蚊可以自由选择要去哪儿吸，当它们落在棉球上开始吸食液体时，研究人员会统计不同棉球上蚊子的数量。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" data-original="http://zkres1.myzaker.com/202205/629422d18e9f0913522bae20_1024.jpg" data-height="650" data-width="1063" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/629422d18e9f0913522bae20_1024.jpg" referrerpolicy="no-referrer"></div></div>通过特殊装置探究雄蚊对血液和糖水的偏好<p></p><p> ( 图片来源：参考文献 ) </p><p>结果同样十分有趣，在 2 分钟内，<strong>落在糖水棉球上吸食的雄蚊数量 ( 3.5 ± 0.17 ) 和落在血液棉球上的雄蚊数量 ( 2.87 ± 0.61 ) 之间并没有显著差异，这意味着血液和糖水对雄蚊的吸引力基本上是一样的</strong>。</p><p>对此，科学家们推测，雄蚊可能拥有和雌蚊类似的味觉受体，这或许解释了为什么雄蚊也会对血液趋之若鹜。但和雌蚊不同的是，雄蚊在吸血后的下场和雌蚊相比可以说是天壤之别。咱用一句话概括就是 " 贼心有了，工具也有了，但贼没了 "。</p><p><strong>血液是雌蚊的营养补充，</strong></p><p><strong>却是雄蚊的穿肠毒药</strong></p><p>大家想必也猜到了那些吸了血的雄蚊的下场，虽然不是当场暴毙吧，但也着实命不久矣。还是古话说的好：" 彼之蜜糖，我之砒霜 "，只是未曾想，这话居然可以用来形容血液对于雄蚊和雌蚊的不同影响。</p><p>还是之前那项研究发现，<strong>当用血液去喂养雄蚊时，它们的寿命会急剧缩短，通常不到 4 天</strong>，而采用 10% 蔗糖溶液喂养的雄蚊却可以生存超过一个月。这里肯定有人要说，刚刚从蛹变为成虫的雄蚊自身只储备了很少的养分，如果无法从糖水中获取养分，那 " 饿死 " 也挺正常啊。</p><p>为此，科学家们进一步实验排除了雄蚊可能营养不良的可能。</p><p>这次他们选择用水、血液、糖水 ( 10% ) 、糖水 ( 10% ) - 血液 ( 10-40% ) 混合物来喂养雄蚊，结果同样让人感到惊讶，用血液和水喂养的雄蚊在 5 天内纷纷死去，用蔗糖喂养的雄蚊则依旧撑过了一个月，而用血液 - 蔗糖混合物喂养的雄蚊平均只活了 7 天，且随着混合物中血液浓度的提升，雄蚊的生存时间也出现了降低，当血液浓度为 40% 时，雄蚊只能存活 3.1 天</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" data-original="http://zkres1.myzaker.com/202205/629422d18e9f0913522bae21_1024.jpg" data-height="398" data-width="708" </p><p>上述结果表明，<strong>血液对于雄蚊存在毒性</strong>，这也在意料之中，毕竟脊椎动物血液中的血红细胞分解后会释放出有毒的血红素 ( 毒性在于它可以产生活性氧 ) 。</p><p>看到这，大家估计会好奇一个问题，都是蚊子，为啥雌蚊可以吸血而不死，雄蚊就暴毙呢？</p><p><strong>1. 为什么雌蚊可以吸血而不死？</strong></p><p>对于这个问题，科学家们确实已经进行了一定程度的探索。</p><p>在过去的研究中，科学家们认为<strong>以脊椎动物血液为食的节肢动物们已经进化出了在血餐后应对血红素毒性的保护性适应，即中肠防御机制</strong>。</p><p>有趣的是，在各种食血节肢动物中，中肠防御机制的具体生理结构多种多样，但作用却殊途同归，它们都是<strong>以特殊的方式来将血红素聚集为血红素结晶</strong> ( hemozoin ) 。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_6" data-original="http://zkres1.myzaker.com/202205/629422d18e9f0913522bae22_1024.jpg" data-height="231" data-width="276" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/629422d18e9f0913522bae22_1024.jpg" referrerpolicy="no-referrer"></div></div>从疟原虫中分离出的血红素结晶<p></p><p> ( 图片来源：wikiwand.com ) </p><p>1992 年的一项研究表明，对于雌蚊而言，在吸血后中肠形成的围食膜 ( Peritrophic matrix，PM ) 结构对形成血红素晶体十分重要。PM 是一种袋状结构，由几丁质、蛋白质和糖蛋白组成。<strong>雌蚊的 PM 具有三个主要功能，包括防止或减少病原体入侵、调节血餐的消化，以及保护上皮细胞免受机械和化学损伤。</strong></p><p>但在 2008 年，科学家们通过破坏埃及伊蚊体内 PM 的形成来探索缺乏 PM 会不会影响雌性埃及伊蚊的生存，结果出乎意料，<strong>PM 缺失不会影响雌蚊的存活率，但 PM 确实会影响雌蚊对于血液的消化</strong>，故而，雌蚊体内应该还存在其他的去除血红素毒性的办法。</p><p><strong>2. 为什么雄蚊吸血会暴毙？</strong></p><p>关于这个问题就更难回答了，毕竟，平时雄蚊也不吸血，怎么研究这个问题都是个问题。</p><p>不过给雄蚊喂血的那帮研究人员认为，这或许是因为雄蚊体内没有用于消化血液的酶。毕竟，在 2001 年的另一项研究中，科学家们就已经证实在致倦库蚊和埃及伊蚊中，雌蚊体内腺苷脱氨酶 ( ADA ) 的水平是雄蚊体内 ADA 水平的 30 倍以上，或许雄蚊体内的其他一些和血液消化相关的酶的水平也存在这样的规律呢？</p><p><strong>如何防蚊？科学家们有话说</strong></p><p>比起雄蚊是否吸血，大家更关心的可能是如何让蚊子不吸自己的血，对于防蚊叮咬，科学家们也是有着很大的脑洞。</p><p>在以前的文章中，我们曾探讨过在房间里头放置一定浓度的糖水来减少蚊虫对自己的叮咬（戳这里看防蚊方法），不过，科学家们的探索欲可不仅限于此。</p><p>在去年的菠萝科学奖颁奖典礼上，来自马来西亚砂拉越大学多样性和环境保护研究所、马来西亚开曼群岛大开曼岛蚊子研究与控制部门的哈马迪 · 丁博士和他的团队探索了一种新型的蚊子防治手段——用格莱美奖音乐让蚊子失去食欲。</p><p></p><div class="img_box" id="id_imagebox_7" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_7"
