<p>还记得当年对 64 位 Windows 系统的厌烦吗？各种兼容问题让人们迟迟不愿意升级，可转眼，32 位 Windows 系统已经被游戏玩家各种嫌弃了 ……</p><p><strong>32 位 Windows 系统被玩家嫌弃</strong></p><p>32 位 Windows 系统已经被玩家嫌弃，因为新的数据表明几乎所有人都在使用 64 位系统。</p><p>根据最新一期的 Steam 软硬件报告显示，只有极少数用户仍在运行 32 位的 Windows 系统，而 64 位的 Windows 系统显然占据主导地位。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202111/6183de39b15ec047b343f0ad_1024.jpg" data-height="850" data-width="553" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/6183de39b15ec047b343f0ad_1024.jpg" referrerpolicy="no-referrer"></div></div>图片来源：Steam 软硬件报告<p></p><p>更准确的是，64 位 Windows 10 系统是 Steam 玩家的首选，因为这个操作系统版本以超过 89% 的份额连续一个月稳居领先位置。古老的 Windows 7 64 位是亚军，占 3.89%，而最近发布的 Windows 11 64 位现在是第三名，占 1.82%。</p><p>榜单中第一个 32 位 Windows 版本是 Windows 7，份额为 0.22%，其次是其 Windows 10 ，份额为 0.09%。</p><p>这份报告中的另一个好消息涉及 Windows 11，因为新的操作系统在短短一个月内已经达到了 1.82% 的份额。而最有可能的是，这些数字实际上要高得多，因为 Windows 11 的 Windows Insider 构建仍被认定为 Windows 10。</p><p><strong>榜单上消失的 Windows XP</strong></p><p>对于 Steam 玩家而言，Windows XP 早已不是考虑对象，可每次在 Windows 系统拉通排名的时候，少了 Windows XP 总觉得不是滋味儿。</p><p>微软今年正式开始推出了 Windows 11，自信满满地告别 Windows 10，准备迈向全新阶段。然而，在全世界仍有相当数量的用户仍选择死守 Windows XP，有个国家的市场占有率甚至超过了 50%。</p><p>截至 2021 年 9 月份，在西亚国家亚美尼亚，仍有 53.5% 的用户在使用 Windows XP 系统，排名第二的是 32.8% 的 Windows 10，再来则仅有 11% 的 Windows 7。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202111/6183de39b15ec047b343f0ae_1024.jpg" data-height="291" data-width="553" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/6183de39b15ec047b343f0ae_1024.jpg" referrerpolicy="no-referrer"></div></div>图片来源：statcounter 报告<p></p><p>一年前，Windows XP 市场占有率掉落到了 44.17%，一度要被 Windows 10 超过，进入 2021 年却逐渐攀升回到 60%，在今年四月份又跌至 32%，随后又拉升到 50% 以上。</p><p>Windows XP 发布至今已经整整 20 年，堪称有史以来最长寿的计算机操作系统，微软早已经将其淘汰。</p><p>当微软于 2009 年 4 月结束对 XP 的主流支持时，它仍在全世界 75% 的 Windows 计算机上运行。在 2014 年结束对 XP 的扩展安全支持时，仍有大约 19% 的人仍在使用 XP。小编知道我们的读者很多也是 Windows XP 的忠实粉丝，这里很好奇大家还会在 Windows XP 上玩游戏吗？</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
