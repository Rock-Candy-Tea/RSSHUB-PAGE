<p>" 移动互联网流量红利枯竭 "，是近几年经常被讨论的话题。其大意是如今很难有一款产品能轻松利用移动网络的特性，低成本实现用户指数级增长。</p><p>这个观点可能已经被业务陷入困境的互联网公司们证明是对的，但它却也在局限着人们对移动化的想象——流量可不是 " 移动互联 " 的全部。</p><p><b>相反，其实在整个互联网世界更为底层的硬件和基础设施层面，移动互联的威力才刚刚彰显。而这比流量的去向重要多了。</b></p><p>只要看看 ARM 在今天的得势，就知道我在说什么。这是一个最有代表性的从移动场景诞生的技术 " 流派 "，今天已经正式开始 " 反噬 "PC 领域。</p><p>蠢蠢欲动</p><p>在 PC 领域，除了独树一帜的苹果 Mac，很少有竞争者能够冲击到已稳固数十年的 X86 加 Windows 系统。苹果从去年开始逐渐将 Mac 端的处理器芯片转为 ARM 架构，这使得 ARM 在个人电脑端的芯片份额突飞猛进，到今年第三季度 ARM 已占据 8% 的 PC 端芯片份额，而在苹果 M1 上市之前，这个数字仅为 2%。</p><p>虽然这归功于苹果的号召力，但 ARM 正在侵蚀传统 PC 领域（X86 加 Windows 系统）的市场却是不争的事实。要知道 ARM 起势于以智能手机为代表的移动互联时代，苹果、三星、高通、华为的移动设备端芯片大多采用的都是 ARM 架构，ARM 可谓一统移动端半导体 IP 市场。</p><p>ARM 在移动端初获成功后，也一直想开拓其他市场，而 PC 时代的霸主英特尔对移动市场同样有想法。但不管是英特尔早年的 StrongARM、XScale、Atom，还是 ARM 与微软、谷歌在轻薄本上的尝试，双方虽在彼此地盘各有试探，却都不算成功。因此，人们通常认为以英特尔为代表的 X86 和 ARM 仅在各自领域更有优势。</p><p>但如今，事情起了变化。</p><p>苹果只是最容易被关注的那个。其它移动端芯片厂商也开始对 ARM+PC 这块蛋糕蠢蠢欲动，背后一个重要原因在于，高通与微软之间用于 Windows 的排他性协议或许即将到期，这意味着在手机芯片领域颇具实力的三星、联发科、华为等厂商具备了挑战 X86+Windows 的可能。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202112/61b6aa6f8e9f0979a85eb1ee_1024.jpg" data-height="343" data-width="550" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202112/61b6aa6f8e9f0979a85eb1ee_1024.jpg" referrerpolicy="no-referrer"></div></div>高通与微软的排他性协议何时到期将成为变局的关键。在今年 9 月份的时候，微软曾表示苹果 M1 芯片的电脑不能搭载 Windows 11 系统，只有高通旗下的 ARM 架构芯片才能对 Windows 系统提供原生支持。这虽然不是一个十分普遍的需求，但无疑也给不少用户浇了一盆冷水。苹果基于 ARM 架构的 M1 芯片电脑虽然性能强劲，但在兼容性上仍需优化，不少 X86 加 Windos 用户不使用 Mac 的原因或是不习惯 Mac 的系统，或是 Windows 生态下一些必不可少的商业软件无法在 Mac 上使用。<p></p><p>这对于 Mac 似乎问题不大，但这种排他性展露无遗，而它对其他想开拓传统 PC 处理器芯片市场的厂商来说，则是决定性的。即便 "AMD Yes" 也只能是在过去 X86 的遗产上来抢夺 PC 处理器芯片市场，既不依靠 X86 架构，也不使用 Windows 系统，来打造又一个如苹果一样受市场欢迎的品牌无疑是太难太难。</p><p>它们需要等待一个裂缝。因此，当国外媒体 XDA-Developers 在近日透漏，高通与微软在 Windows on ARM 上的合作排他协议或即将到期时，联发科在发布 4nm 的天玑 9000 系列处理器之前，就已迫不及待的表示，将进军 Windows on ARM 市场，甚至三星及其 Exynos 处理器的 PC 也将在市面上出现。</p><p>当 Windows 生态和芯片架构的底层优化上没有了使用差异后，PC 芯片市场上面临的就是真枪实剑的性能、功耗、发热等综合性能的硬战。X86 架构和 ARM 架构孰优孰劣、谁将一统市场行业内也是争议不断，这其中既涉及底层 RISC 指令集和 CISC 指令集方面的技术差异，也涉及服务器、PC 和智能移动端的不同使用场景，需要设计者在芯片性能、功效、能耗和应用场景上做出选择。</p><p>技术在芯片市场重要，但已不是唯一衡量标准。随着技术的进一步发展，不管是 ARM 还是 X86，都在逐渐补齐各自的短板，不再仅采取 RISC 和 CISC 中某一种指令集，而是变成 RISC 和 CISC 你中有我、我中有你，可以说标着 RISC、CISC 的 CPU 们，很难仅通过架构来决定其优劣。</p><p>算力也青睐 " 流量 "</p><p>最直接的争夺即将展开，而竞争的天平似乎早已开始倾斜。</p><p>在这场竞争中，旧体系依赖着过往的强势以及合纵连横形成的技术优势，在范式转移中显得似乎没那么牢靠，反而，ARM 背后聚集的一众移动市场玩家，在移动互联网普及之中却积攒了关键的资源。</p><p>比如在芯片产能分配上的话语权。</p><p>未来真正决定处理器们算力的关键变成了是否采用最新工艺和各芯片设计中的微架构。芯片的先进工艺很好理解，虽然芯片性能并不是简单的芯片制程越小性能越强，但在功耗、发热、面积、价格等多因素综合考虑来看，可以简单理解为 10nm 制程的芯片就是很难超过 5nm。</p><p>芯片设计中的微架构同样重要，例如即便是同样是采用 7nm 制程、ARM 架构、台积电代工的苹果 A13 芯片和高通骁龙 865 芯片，它们的处理器单核、多核跑分性能均有差异，其原因就在于不同公司设计的芯片微架构也有高下之分。</p><p>因此，简单来看，决定各厂商芯片算力强弱的关键在于是否抢占到了三星、台积电有限的先进产能和自身芯片设计实力，因为三星和台积电是唯一有能力生产 5nm 先进制程的芯片厂商。</p><p><b>现实是，移动市场中更有优势的公司在芯片代工产业链中声量更大</b>，苹果和高通分别包揽了台积电和三星的优先产能（当然包括三星自家产品），英特尔也意识到了这个问题，提出了 IDM2.0 计划，将不再死磕自家工厂，委托台积电制造先进制程，有媒体爆料英特尔的 14 代酷睿或将采用台积电的 3nm 制程。</p><p>三星、台积电先进制程的芯片优先供给谁非常重要，这甚至可以直接决定委托客户的产品销量。例如，近期小米宣布其新款机型将将首发骁龙 8 Gen 1，OPPO、vivo、联想、摩托摩拉都在抢高通芯片的首发权，原因在于数码产品买新不买旧，在客户预算充足的情况下自然不会去买上一代的骁龙 888 机型，那么谁抢占了新款处理器的首发权，谁就率先抢占到了市场先机。</p><p>这个道理同样适用于高通、苹果、联发科等芯片设计厂商，缺芯的大环境下，抢占三星、台积电的优先供货权变得异常重要。过去，这种对先进制程优先产能的抢夺仅发生在移动芯片市场，自有工厂的英特尔和强大的 X86 加 Windows 生态使得领域内玩家较少，PC 硬件市场的争夺并没有那么严重，但未来随着移动端芯片厂商的加入，PC 端的大乱斗或许也将到来。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 的原因与台积电的产能正专注于为苹果定制芯片有关，但不管是高通还是苹果，其所需代工厂提供的最新工艺都需要不断调试升级，不像车用芯片一般有一套成熟的方案，因此这也需要着大量资金。<p></p><p>在移动端有足够实力让最强芯片代工厂尝试新进制程的公司屈指可数——苹果、高通、三星、联发科和以往的华为，PC 端则是 AMD、英伟达和终于让台积电代工的英特尔。若排除 GPU 领域的英伟达，从市值来看移动端阵营的整体实力确实远超 PC 端，而在移动化浪潮之前这是不可想象的，过去即使是苹果也仅仅是强盛时期英特尔的一个零头，代工之王台积电更只是俯仰在英特尔的鼻息之下。</p><p>移动化浪潮创造了巨大的市场，也造就了移动端这些为人所熟知的公司，这些公司所赚得的钱仅需一部分继续投入到移动端硬件和生态的成长上便能形成惊人的复利，苹果的 M1 系列芯片是最好的证明之一，在以往，ARM 架构的芯片用在 PC 领域是一件难以想象的事情。</p><p>这样的话语权更迭，也对整个产业链带来深刻影响。变化正在蔓延开来。</p><p>比如服务器市场。X86 处理器过往一直占据着这里的主导地位，在前几年，一直都是英特尔和 AMD 两家公司瓜分着服务器市场，其中英特尔市场份额超过 90%，AMD 则吃掉剩下的部分，ARM 虽然眼馋于服务器市场但一直难有进展。</p><p>但对于没有密切关注这个领域变化的人来说，令人诧异的是，近年来各互联网巨头已纷纷开始自研 ARM 架构的服务器芯片，包括国外的亚马逊、谷歌甚至英特尔的老朋友微软，国内的腾讯、阿里巴巴、华为等等都积极参与其中。</p><p>如此重视一个市场份额看似依然很低的技术，背后原因仍归功于移动互联浪潮。ARM 服务器和移动 APP 端云同构，在移动端的应用具有天然的优势，兼容性好，指令执行效率高（不需要翻译转化），随着短视频时代的到来，数据中心所需吞吐的移动端流量变大，ARM 服务器的优势或许会进一步彰显。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
