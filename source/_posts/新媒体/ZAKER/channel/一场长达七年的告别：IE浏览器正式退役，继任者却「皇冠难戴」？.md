韩大鹏</p><p><strong>划重点：</strong></p><p>微软公司于 6 月 15 日永久关闭 IE 浏览器。近些年来，这款拥有 28 年历史的浏览器，屡次入选 " 糟糕科技产品 " 排行。业内人士普遍认为，微软放弃它是明智之选，" 没有必要再恋战 "。</p><p>今天，IE 浏览器正式退出历史舞台。从巅峰时刻占据全球 95% 以上浏览器市场份额，到如今被战略性放弃，IE 浏览器用 28 年走完了这段里程碑式的旅程。</p><p>28 年间，计算机行业经历了从 PC 时代到移动互联网时代的全新变革。期间，Google Chrome、FireFox 等浏览器后浪辈出，蚕食并最终抢走了本来属于 IE 的浏览器市场。与此同时，Facebook、TikTok 等新型网络应用的出现，也不断侵蚀着浏览器曾经作为唯一网络入口的特殊地位。</p><p>展望下一个十年，随着国内移动互联网高速增长的神话逐渐褪色，新一代的 Web3.0 开始暗潮涌动。IE 浏览器退役后，企业和消费者们准备好了吗？而未来的浏览器故事，又该如何继续？</p><p><strong>Edge 接过 IE" 大旗 "</strong></p><p>现实是残酷的，两组数据足以验证：</p><p>在风头最盛的时候，IE 系列浏览器一度达到 95% 的市场占有率。然而截至今年 4 月，IE 浏览器在全球台式电脑浏览器市场中的份额，已不到 2%；</p><p>2006 年，IE 6 浏览器被评为了 " 有史以来第八糟糕科技产品 "。而在 2011 年的 " 科技史上 50 种最糟糕科技产品 " 名单中，IE6 也位列第 11 名。</p><p>于是，早在 2015 年 3 月，微软确认将放弃 IE 品牌，转而在 Windows10 上用 Microsoft Edge 取代 Internet Explorer。至今，IE 浏览器退出历史舞台这一出戏，已持续上演了 7 年有余。</p><p>7 年间，微软分别在 2016 年 1 月、2020 年 8 月以及 2021 年 3 月，先后宣布停止对 Internet Explorer8/9/10 三个版本的技术支持，停止微软 365 应用程序对 IE 11 以及 legacy Edge 浏览器的支持。</p><p>而最近一次是今年 5 月 16 日，微软正式官宣了 IE 浏览器将于 6 月 16 日退役。" 我们已经看到，随着新的 Web 标准和创新的发展，开发浏览器的时候需要全盘考虑，而其在技术上已经超越了 Internet Explorer。" 微软方面表示。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 浏览器停用以来，微软便一直在积极地与企业和个人消费者沟通，渐进地关闭一些 IE 浏览器功能和服务，同时提醒外界将业务切换到 Edge 或其他浏览器上，以确保他们能顺利地过渡所需资源。<p></p><p>"Microsoft Edge 为人们带来了比 Internet Explorer 更快速，更安全，更现代的 Web 体验。" 微软方面表示，在性能方面，与 IE 相比，Microsoft Edge 将渲染时间缩短了 85%，CPU 利用率减少了 33%，内存利用率减少了 32%。</p><p>在国内，由于中国互联网行业发展初期 IE 浏览器比较普及，这导致了很多网站开发者做的页面是针对 IE 特殊标准设计的。虽然目前 IE 浏览器在国内的市场份额已经比较低，但在 IE 退出后，仍有一些银行、高校等机构的网页或者小众网站，会受到一定程度的影响。</p><p>"IT 管理员需要尽快了解企业中每一位用户使用的浏览器情况，如果用户依然在使用传统的 IE 浏览器，则要为其尽快安排升级计划。"360 浏览器技术专家绕宇（化名）对新浪科技建议道，" 在 Microsoft 365 管理中心门户中，IT 管理员可以打开‘使用情况’报告，查看企业中有多少用户还在使用 IE 浏览器，并通过邮件等方式提醒用户升级浏览器。"</p><p></p><div 浏览器兼容模式，开展相关业务。<p></p><p>据微软方面对新浪科技介绍，目前 Edge 已经支持 InternetExplorer（IE）模式，这是目前唯一一款内置支持基于 IE 的 Web 应用和站点的浏览器，对 ActiveX 控件等功能具有向下兼容性支持。企业机构可以为其用户设置 IE 模式，个人用户也可以根据需要访问 IE 模式。</p><p><strong>IE 是如何失败的？</strong></p><p>浏览器的故事始于 1994 年的网景浏览器，但其中统治时间最长、最为著名的还数 IE 系列浏览器。凭借着与 Windows 操作系统捆绑销售，同时作为 Windows 默认浏览器等先天优势，在风头最盛的时候，IE 系列浏览器一度达到 95% 的市场占有率。</p><p>但在后来的发展中，IE 浏览器却开始在激烈的浏览器市场争夺竞争中处于下风，并开始被开发者和用户们抛弃。据网络分析公司 StatCounter 数据，截至今年 4 月，IE 浏览器在全球台式电脑浏览器市场中的份额，已不到 2%。</p><p>"IE 浏览器所采用的 Trident 内核，并不完全基于 W3C 标准开发。同时还是一个闭源系统，这导致开发者无法直接修改或封装该内核，而 IE 浏览器本身的更新迭代速度又跟不上，久而久之，就被同行赶超。"2009 年曾脱产创业，亲自参与过移动浏览器研发设计的唐荣兴对新浪科技表示。</p><p>据唐荣兴介绍，继 IE 浏览器，由 Mozilla 开发的 FireFox 将浏览器代码进行了开源，进一步降低了浏览器的创新门槛。此外，谷歌推出的 Chrome 浏览器也通过良好的组件化建设，支持更轻更快地迭代开发。苹果等公司也纷纷基于各自优势，推出了自己的浏览器。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" 浏览器期间只迭代更新了 4 次。" 激烈的市场竞争之下，产品迭代速度缓慢的 IE 浏览器开始根本赶不上整个行业的速度。于是 " 速度慢、反应迟缓、功能单一 " 等用户抱怨声音，也随之而来。<p></p><p>2006 年，IE 6 浏览器被评为了 " 有史以来第八糟糕科技产品 "。而在 2011 年的 " 科技史上 50 种最糟糕科技产品 " 名单中，IE6 也位列第 11 名。</p><p><strong>下一代浏览器将有何改变？</strong></p><p>" 在过去几年间，Firefox、Chrome 等浏览器一直都在面向未来进行开发与产品迭代，然而 IE 浏览器却很少如此。" 在唐荣兴看来，IE 浏览器属于微软的防御型产品，并非一级产品，如今被微软放弃也不失为一个明智之选，" 没有必要再恋战了 "。</p><p>IE 浏览器的退役，是一个时代的终结。但历史的车轮下，浏览器的江湖故事并未因此而变得风平浪静。</p><p>"Firefox 已经支持 WebAR 了，在未来虚实结合的世界，通过 AR、GUI 等进行内容浏览的方式将会发生，这可能会催生新的浏览器。" 此外，" 基于区块链等技术实现的 Web3.0 时代，底层架构的变化，也可能催生新的浏览器。"</p><p>360 浏览器绕宇补充道，" 未来，随着越来越多的办公应用逐渐实现 SaaS 化，跨平台的 SaaS 市场，也将成为未来浏览器市场新的增长点。"</p><p>不断更迭变迁的互联网底层技术，层出不穷的竞争对手 …… 下一代浏览器的形态究竟是怎样的？IE 浏览器的继任者能否顺利突围 ? 答案值得期待。</p><div id="recommend_bottom"></div><div
