<p>昨天微软正式对外开放了旗下云电脑 Windows 365 的订阅服务。这个用户只要在订阅后就可以直接在浏览器里运行云端 Windows 10 或者 Windows 11 的服务可以说是大受欢迎。不过也是因为太受欢迎，导致试用版的服务器都用完了，因此微软只能暂停 Windows 365 的试用服务。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202108/610aad438e9f097a72533a85_1024.jpg" data-height="608" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/610aad438e9f097a72533a85_1024.jpg" referrerpolicy="no-referrer"></div></div>如果有人不知道 Windows 365 是什么的话，它是微软为企业或者商业用户所推出的一个云端电脑订阅服务，总共分为基本、标准以及高端 3 个级别，订阅的价格则是从 31 美元至 66 美元一个用户不等，而云电脑的配置则是有 2 至 4 个 vCPU、4 至 16GB 内存以及最大 128GB 的存储空间。当然，如果用户是有其他需求的话也可以自定义配置，这方面的价格则是从 20 美元至最高 158 美元不等。<p></p><p>而为了吸引更多潜在用户，微软同时也推出 Windows 365 的免费试用，配置分别为 2 至 4 个 vCPU、4 至 16GB 内存以及 128GB 存储空间，有兴趣的用户可以试用两个月。</p><p>不过，微软很显然是低估了免费试用版的需求，因为供免费试用的服务器很快就被占完了，当新用户尝试申请免费试用时，只会看到这样的一个错误页面。微软 Windows 365 项目管理总监 Scott Manchester 之后在推特上确认，微软暂时停止了 Windows 365 的试用服务，同时也正在部署更多的新服务器。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202108/610aad438e9f097a72533a86_1024.jpg" data-height="509" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202108/610aad438e9f097a72533a86_1024.jpg" referrerpolicy="no-referrer"></div></div>如果现在用户想试用 Windows 365 的话，可以在这个页面登记，一旦有免费试用再次恢复的话就可以收到通知。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
