<p>IT 之家 10 月 22 日消息，根据 IT 之家网友 @顺理成章 投稿，微软今日为 Win11 推送了 10 月 KB5006746 可选累积更新，进行了大量修复。升级后，系统版本变成 Win11 Build 22000.282 正式版。本次更新内容与此前向预览体验通道推送的更新类似，最重要的是针对 AMD 锐龙系列处理器修复了 L3 缓存性能下降的问题。</p><p></p><div 崩溃的问题。</p><p>修复了用户使用最新版日文 IME 输入法为文件夹重命名时，出现的故障。</p><p>修复了用户设置锁屏界面背景为幻灯片显示时，偶尔出现的纯黑背景问题。</p><p>修复了蓝牙鼠标键盘偶尔响应时间慢的问题。</p><p>提升了设备在重启后，对进入系统时间估计的准确性。</p><p>修复了可能阻止 Xbox Game Bar 录屏的故障。</p><p>修复了一些应用在 Win11 系统性能下降的 Bug。</p><p>修复了某些情况下，阻止 " 讲述人 " 和其它屏幕阅读器在打开开始菜单之后发出通知的问题。</p><p>解决了阻止搜索窗口出现在第二台显示器上的问题。</p><p>解决了阻止用户使用 Shift 键同时打开多个相同应用，并且在任务栏图标显示错误的问题。</p><p>更新了任务栏聊天图标的视觉设计和动画效果。</p><p>为少数用户修复了一个问题，该问题会阻止开始菜单正常工作，并且任务栏为旧样式。</p><p>本次更新的改善和修复：</p><p>解决了系统启动早期可能会卡住的一个问题。</p><p>解决了某些电脑在注册表中使用非 ASCII 字符时，可能返回 0x38 错误的问题。</p><p>解决了少数用户在从 Win10 升级至 Win11 后，开始菜单无法正常工作，并且任务栏为旧样式的问题。</p><p>修复了 AMD 锐龙处理器在 Win11 中，L3 缓存性能下降的问题。此前该问题可能导致 L3 缓存延迟增加三倍，游戏性能下降明显。</p><p>解决了可能导致设备停止响应的终端处理问题。</p><p>修复了导致 PowerShell 无限创建子目录的问题。该问题出现在用户使用 PowerShell"Move-Item" 命令来将一个目录移动到其子目录时发生，这一 Bug 会导致系统停止响应。</p><p>解决了可能导致服务管理器应用在用户卸载 Hyper-V 后消失的问题。</p><p>解决了 Windows 远程管理服务在高负载下停止工作的问题。</p><p>解决了导致 WMI 体用程序主机进程停止工作的问题。这一 Bug 是由于在使用 Desired State Configuration ( DSC ) 时，未处理的访问冲突发生。</p><p>解决了在更新组策略时，强行关闭设备会导致设备停止响应的问题。</p><p>解决了导致存储在不同分区上的分布式文件系统（DFS）路径间文件迁移失败的问题。当用户使用 PowerShell"Move-Item" 命令时，这一问题会发生。</p><p>修复了在内存不足时，阻止用户写入 WIM 存储库的问题。</p><p>修复了 PowerShell 快捷方式文件中，硬编码字体（hardcoded font）会扭曲日文、中文、韩文字体的问题。</p><p>修复了在不计入毫秒的事件中，时间格式错误的问题。</p><p>解决了错误呈现某些增强元文件格式（EMF）文件的问题。如果用户使用第三方应用 ExtCreatePen ( ) 或 ExtCreateFontIndirect ( ) 方式构建 EMF 文件，会出现这种故障。</p><p>为管理员提供 Edge 浏览器 IE 模式下，将 HTML 对话框缩放重置为默认值的选项。</p><p>解决了企业模式站点列表从 IE11 浏览器重定向到 Edge 的问题。在某些情况下，重定向会在 Edge 的多个选项卡打开同一个站点。</p><p>解决了输入法编辑器（IME）中输入某些字符时，导致网页浏览器停止工作的问题。</p><p>修复了 JScript9.dll 中，PropertyGet 的问题。</p><p>修复了当用户在 VBScript 中使用嵌套类时发生内存泄露的问题。</p><p>修复了电脑设置为快速启动时，重启后无法保持 Number Lock 的问题。</p><p>解决了移动某些程序窗口的问题。如果文件资源管理器窗口在屏幕上可见，移动这些应用程序窗口可能会异常缓慢。</p><p>......</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
