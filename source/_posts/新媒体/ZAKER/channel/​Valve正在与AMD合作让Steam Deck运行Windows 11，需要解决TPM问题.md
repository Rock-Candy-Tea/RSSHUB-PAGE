<p>Valve 前段时间推出的 Steam Deck 是新一代的掌上娱乐系统，预计首批产品会在今年年底出货，它搭载一个 7 英寸 1280*800 的 IPS 屏幕，16GB LPDDR5 内存，存储空间有 64GB、256GB、512GB 三个版本，使用了 AMD Van Gogh APU，是 Zen 2 架构 CPU 内核加 RDNA2 的核显，虽然自带的是用 Arch Linux 是 SteamOS，但由于用的是 x86 的处理器，所以用 Windows 10 是不成问题的，至于微软即将推出的 Windows 11，Valve 表示正在与 AMD 合作以提供支持。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202108/611262db8e9f096db46d0b77_1024.jpg" data-height="333" data-width="768" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202108/611262db8e9f096db46d0b77_1024.jpg" referrerpolicy="no-referrer"></div></div>PCGamer 就 Steam Deck 是否支持 Windows 11 询问了 Valve，得到的答复是他们正在研究 TPM，但目前还没有任何迹象表明运行 Windows 11 会出现任何问题。<p></p><p>微软的 Windows 11 需要平台支持 SecureBoot/TPM 2.0 才能运行，目前 AMD 所有 Zen 2 架构的桌面平台都可支持，而 Van Gogh APU 用的就是 Zen 2 内核，所以并不存在任何问题，但 TPM 还需要 BIOS 的支持，这就是目前 Valve 和 AMD 目前需要解决的问题。</p><p>如果 Steam Deck 支持 TPM 的话，它就可以运行新一代的 Windows 11 系统，这赋予它更多的功能，除了原本就能支持 Steam 平台的游戏外，还可以运行其他的 Windows 游戏，还可以摇身一变成为一台掌上办公设备，Windows 11 系统自带的安卓模拟器也使其可以直接运行安卓 app，泛用性大幅提升。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
