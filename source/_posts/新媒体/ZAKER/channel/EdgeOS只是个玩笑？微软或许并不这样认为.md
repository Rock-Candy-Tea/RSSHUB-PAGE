<p>此前在 6 月 16 日，微软的 IE 浏览器（Internet Explorer，下文简称为 IE）正式退出历史舞台，这款诞生于 1995 年的浏览器也终于走完了它毁誉参半的 " 一生 "。而在 IE 退役后，接棒的则是 Edge。在拥抱 Chromeium、并摆脱了 "Chrome 下载器 " 的的宿命后，Edge 也正式走上了台前，如今其已拥有 10.11% 的桌面浏览器市场份额、也成为了全球第二大浏览器，更是企业市场的首选。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" "EdgeOS"Beta 版本截图，并称 " 泄漏的微软重塑 Windows 11 计划，将会切换到全新基于网页的技术驱动框架 "。<p></p><p>当然了，这张像模像样的截图最终被抓到了马脚，只是一个编辑版本的 Winver 窗口，顶栏显示的还是 " 关于 Windows"、而不是 " 关于 EdgeOS"，甚至 Build 版本号也没有正确排列。</p><p></p><div "EdgeOS"，其中有 " 作为 EdgeOS 团队的一员，我们在操作系统中提供核心网络，为微软提供动力 " 这样的表述。当然，这也只是从侧面证明了微软内部确实存在一个 "EdgeOS 团队 "，但真正意义上的 EdgeOS 是否会问世，还要打上一个问号。<p></p><p></p><div class="img_box" id="id_imagebox_2" onclick><div src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202207/62d6bc288e9f094ca23f53bf_1024.jpg" referrerpolicy="no-referrer"></div></div>不过外界相信 EdgeOS 的存在，自然是基于事实基础的。过去的几个月里，微软方面一直在向 Edge 中添加大量额外、且非必要的功能，到目前为止，其已经为 Edge 加入了一个完整的侧边栏，其中包含一个计算器、单位转换器，以及照片应用程序，还将 Windows 11 照片应用中的编辑功能也搬到了 Edge 上，允许用户在保存网页图片前对其进行简单的编辑，甚至还有指向整个 Office 办公套件和 Outlook 日历的链接。<p></p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" data-original="http://zkres1.myzaker.com/202207/62d6bc288e9f094ca23f53c0_1024.jpg" 添加了专门用于游戏的新主页，其中包括新闻、指南、直播和锦标赛的精彩片段等，并与 Xbox 进行了集成，如果用户使用 Xbox Live 帐户登录，就将看到与用户最近玩过游戏相关的内容。此外，如果用户拥有 Xbox Game Pass Ultimate 订阅，甚至还能够通过云端串联的方式，在拥有 Clarity Boost 技术的 Edge 浏览器上直接玩游戏。<p></p><p>如此种种，也使得微软能够自夸 Edge 是 " 最适合游戏玩家的浏览器 "。</p><p>那么微软有能力打造 EdgeOS 吗？答案显然是肯定的。作为全球最为知名操作系统 Windows 的开发公司，微软在操作系统开发方面的实力显然毋庸置疑。准确来说，做一个类似于谷歌 ChromeOS 的系统其实并不难，甚至难度远小于 Android、iOS，乃至鸿蒙。而 ChromeOS 则是基于 Gentoo 深度定制的 Linux，轻量化的设计让其主要工作是通过 WebApp 实现。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" data-original="http://zkres1.myzaker.com/202207/62d6bc288e9f094ca23f53c1_1024.jpg" data-height="375" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202207/62d6bc288e9f094ca23f53c1_1024.jpg" referrerpolicy="no-referrer"></div></div>所谓 WebApp，本质上就是运用各种浏览器技术实现在线 / 离线运行的网页。ChromeOS 的开发理念就在于，随着现代化浏览器的普及，各种扩展程序的出现极大程度上丰富了 Web 生态，既然浏览器借助日益丰富的 WebApp 就能完成用户要解决的任务，那为什么不从操作系统的层面将电脑变成一个浏览器呢？<p></p><p>事实上，微软不光有能力打造一个类似于 ChromeOS 的 EdgeOS，他们对此可能同样也有着强烈的意愿。自去年秋季推出 Windows 11 后，微软就一直通过包括但不限于停止对 Windows 8 支持在内的各种方式，来催促用户升级到 Windows 11，但结果是大量用户并不买账。</p><p>据资产管理软件供应商 Lansweepe 在今年 4 月发布的一份报告显示，Windows 11 的普及遇到了较大阻碍，截止 2022 年 4 月其安装率仅仅只有 1.44%。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" data-original="http://zkres2.myzaker.com/202207/62d6bc288e9f094ca23f53c2_1024.jpg" data-height="399" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202207/62d6bc288e9f094ca23f53c2_1024.jpg" referrerpolicy="no-referrer"></div></div>用户不愿意升级 Windows 11 的核心原因，或许就是门槛太高。必须要主板支持 TPM2.0 模块才能正常升级，而且不能完美支持 intel 酷睿 8 代之前的 CPU，这就意味着绝大多数 2015 年前的电脑将无法获得升级。作为一个相对耐用的消费电子产品，有相当多用户在智能手机时代更换电脑的频率已经被极大拉长了，这也造成了无法升级 Windows 11 的老电脑可能有着极为可观的存量。<p></p><p>由于 " 安迪 · 比尔定律 " 的存在，让硬件所提高的任何计算能力总是会被新一代的软件消耗殆尽，老机型在面对新时代软件的时候往往就会力不从心。然而这一部分老机型同样也是不可忽视的市场，这不，谷歌日前就正式发布了 Chrome OS Flex。而这则是一款专门面向老款 PC 和 Mac 的精简版操作系统。</p><p></p><div class="img_box" id="id_imagebox_6" 的基本盘，如果大量的老机型转向 Chrome OS Flex，此消彼长之下，Windows 的影响力自然就会被削弱。所以面对 Chrome OS Flex 的进攻，微软拿出类似定位的 EdgeOS 来进行防守，显然也在情理之中。<p></p><p>【本文图片来自网络】</p><div id="recommend_bottom"></div><div
