<p>IT 之家 10 月 26 日消息，根据 Alumia 的一条推文显示，微软似乎正在为 Windows 11 系统准备全新的 Windows Defender Preview 应用程序。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202110/6177b79d8e9f092a9b1d102b_1024.jpg" data-height="451" data-width="680" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202110/6177b79d8e9f092a9b1d102b_1024.jpg" referrerpolicy="no-referrer"></div></div>该应用程序代号为 GibraltarApp，似乎是使用 WPF 和 XAML 重建的。它将取代当前 Windows 11 中的内置 Defender 应用程序。<p></p><p>据称，它将为用户提供 " 简单、无缝和个性化的保护 "，并有望在不久的将来向 Windows 内测人士推出。</p><p>IT 之家获悉，微软 Windows 11 于今年 10 月正式发布，其整体界面的重大变化证明了操作系统版本号的跳跃。Windows 11 结合了 Fluent Design 和 WinUI 的优点，为满足最低要求的设备提供了现代和触摸友好的体验。</p><p>随着 Windows 11 的推出，"Fluent Design 流畅设计 " 变得更好，微软还推出了一种名为 "Mica 云母 " 的新设计材料，将允许应用程序和桌面元素与主题壁纸相匹配。然而，这些变化主要影响系统元素，这意味着应用程序开发人员需要更新他们的应用程序，以获得 Windows 11 的最佳效果。</p><p>此前，微软 Windows 11 一批原生应用正在获得视觉 UI 更新，包括闹钟和时钟、画图、截图工具、记事本、照片、录音机、计算器等等。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
