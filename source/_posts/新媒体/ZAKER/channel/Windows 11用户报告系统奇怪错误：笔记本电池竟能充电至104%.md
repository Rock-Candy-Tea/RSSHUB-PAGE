<p>每个人都喜欢从他们的笔记本电脑中挤出尽可能多的寿命，而每一个新 Windows 迭代都对延长电池寿命做出了巨大的承诺。但 Windows 11 中的一个奇怪错误意味着你可能会得到比你所希望的更多的东西。<strong>在 Reddit 上，一位用户分享了他们的笔记本电脑电池充电到 100% 以上的截图。</strong></p><p>据悉，一位名为 Zondax 的 Reddit 用户惊讶地发现，他们的 Windows 11 笔记本电脑已经充电到 103%。在发布了一张截图以证明 " 过度充电 " 后，他们后来更新了帖子，说电池已经达到 104%。</p><p>很显然，微软还没能够用 Windows 11 打破物理定律，该公司还没能让电池的电量超过其容量。</p><p>不可否认的是，这是一个奇怪的错误，并且微软还没有就这个问题谈论太多。然而，在回应这个故事时，这家公司的工程团队的一名成员告诉媒体：" 我们已经对这个漏洞展示调查，并将尽快将其修复。"</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202111/618740238e9f0960c64d72a2_1024.jpg" data-height="598" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/618740238e9f0960c64d72a2_1024.jpg" referrerpolicy="no-referrer"></div></div>据悉，Zondax 正在（或至少曾经）运行的是 Windows 11 build 22483.1000，所以它有可能是跟预览版相关的一些怪异现象。不过到目前为止，这似乎是唯一一个关于 Windows 11 的报告，所以这可能只是一个孤立事件。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
