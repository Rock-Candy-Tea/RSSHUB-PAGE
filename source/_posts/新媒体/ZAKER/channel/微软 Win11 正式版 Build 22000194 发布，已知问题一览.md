<p>IT 之家 10 月 5 日消息 Windows 11 正式版已经发布，部分用户已经可以开始升级。过去三个月，微软一直在与 Insider 用户一起测试 Windows 11。</p><p>今日，微软宣布了 Win11 正式版 Build 22000.194 已知的三个问题（未来可能会出现更多），我们一起来看看：</p><p><b>Oracle VirtualBox 兼容性问题</b></p><p>微软表示 VirtualBox 和 Windows 11 之间存在兼容性问题。升级到 Windows 11 后，如果已安装 Hyper-V，则 VirtualBox 计算机将无法访问。</p><p>微软已经应用了一个更新保护，来阻止使用 VirtualBox 的设备进行更新。微软正在调查这些报告，很快就会发布一个修复程序。</p><p>用户也可以选择手动删除 VirtualBox 或禁用 Hyper-V，可能需要 48 小时之后才能获得升级。</p><p><b>Intel Killer 网卡问题</b></p><p>微软意识到一些 Intel Killer 网卡存在 Windows 11 问题。在这些设备上安装 Windows 11 可能会导致删除 UDP，并为基于 UDP 的协议造成性能问题的情况。</p><p>例如，在线播放视频时，用户可能会遇到网站加载时间较慢的问题。</p><p>IT 之家了解到，该问题将在 10 月 12 日进行的下一次 10 月安全更新中修复。</p><p><b>C c C c 浏览器兼容性问题</b></p><p>Windows 11 更新会造成越南浏览器 C c C c 浏览器崩溃，使用该浏览器的用户请不要更新。</p><p>目前微软官方正计划进行一次大规模的 Windows 11 累积更新，以解决操作系统中存在的问题。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
