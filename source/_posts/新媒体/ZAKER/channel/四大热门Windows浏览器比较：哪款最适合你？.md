<p>如果单纯从用户规模来看，Chrome 无疑是最受欢迎的浏览器，其次是 Microsoft Edge 和 Mozilla Firefox。但每个人的使用需求都不尽相同，有些人偏好流畅的上网体验，有些人偏好优秀的个性定制，有些人偏好丰富的功能集成，那么在 Windows 平台上哪款浏览器最适合你呢？</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202206/62a6a945b15ec0158579e974_1024.jpg" Chrome、Edge、Firefox 和 Vivaldi 四款热门浏览器进行了横向比较，在对基础性能比较之外，并罗列了各自浏览器的优缺点，方便你根据使用需求在 Windows 平台上找到最适合你的浏览器。<p></p><p><strong>跑分</strong></p><p>为了测量的公平性，NeoWin 分别在 AMD 和 Intel 两个平台上对上述四款浏览器进行了测试，请记住，根据硬件的强大程度，系统的性能可能会略有不同。</p><p>普通消费者可能并不关心他们的浏览器在 Jetstream 基准测试或类似测试中获得多少分。尽管如此，原始性能值仍然是浏览器如何处理要求苛刻的任务的一个很好的指标。随着网站变得越来越复杂和 Web 应用程序越来越流行，快速的 JavaScript 性能是获得令人满意的体验的关键。</p><p><strong>● Jetstream 2</strong></p><p>Jetstream 2 是一个复杂的基准测试，用于衡量 Web 应用程序的性能。启动速度快、执行代码速度快、运行流畅的浏览器会获得更高的分数。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202206/62a6a945b15ec0158579e975_1024.jpg" data-height="456" data-width="760" 的浏览器都获得了或多或少相同的分数。然而，Firefox 的表现要差得多，并且在日常浏览中可能无法跟上 Chromium 的步伐。<p></p><p><strong>● Motionmark</strong></p><p>Motionmark 是另一个显示 Chrome 的 Blink 引擎优于其他引擎的基准测试。该基准测试浏览器渲染越来越流行的复杂图形和效果的能力。</p><p></p><div 以明显较低的分数位居第四。在 Intel 机器上，所有 Chromium 浏览器都收到了类似的结果。<p></p><p><strong>● Web 应用响应能力</strong></p><p>分数越高，您在运行网站和 Web 应用程序时的体验就越好。在 AMD 上，Firefox 击败了 Edge 和 Vivaldi，并且在基于 Intel 的 PC 上进行测试时处于同等水平。 Chrome 在两个系统上都以压倒性的优势领先于竞争对手。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" 内存占用</strong><p></p><p>虽然测试可能不太科学，但更加真实。NeoWin 启动了 10 个网页（YouTube 视频、商店、大型文档页面和具有复杂动画的网站）来查看每个浏览器消耗了多少内存。同样，结果有点令人惊讶。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" data-original="http://zkres1.myzaker.com/202206/62a6a945b15ec0158579e978_1024.jpg" data-height="615" data-width="1024" 350MB，比 Firefox 少 10MB。 Vivaldi 以最佳的 RAM 效率让我们感到惊讶：它只需要 960MB 的 RAM 即可显示所有十个选项卡。<p></p><p>值得一提的是，Microsoft Edge 具有冻结非活动标签以减少 RAM 消耗的功能。 Edge 中的休眠选项卡可以为每个选项卡节省大约 30-40 MB 的 RAM，但我们的测试显示，在将九个选项卡置于休眠状态后，并没有显着节省。使用消耗大量内存的 "heavy" 选项卡时，您可以获得更好的结果。</p><p><strong>Chrome 的性能最佳，是否就意味着它最好？不完全是。</strong></p><p>尽管 Chrome 始终如一的更好的原始性能、卓越的兼容性和一流的跨平台功能不容忽视，但 Google 的浏览器在其他几个方面都落后于竞争对手。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" 浏览器的一个重要原因是，大多数开发人员将精力集中在优化 Chromium 上，有时会忽略替代方案。而且由于 Firefox 和 Safari 与 Chrome 相比，用户群相对较低，因此事情陷入了恶性循环。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div 使浏览器立即启动。 Edge 还可以说是一流的跨平台密码管理器。其他值得注意的功能包括性能优化工具、购物助手、收藏品、睡眠标签等等。</p><p>另一件值得指出的事情是，Edge 可以说是 Windows 上最好看的 UI。当然，您不会因为外观而选择浏览器，但 Chrome 开始觉得开发人员只是不关心视觉效果。</p><p></p><div class="img_box"
