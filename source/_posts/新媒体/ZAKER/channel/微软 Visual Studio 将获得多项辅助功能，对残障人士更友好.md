<p>IT 之家 7 月 2 日消息 微软近日在官方博客发布了一篇《让我们一起使 Visual Studio 更易用》的文章，宣布将与残障人士积极沟通，为 Visual Studio 带来更多辅助功能。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202107/60de8ed98e9f092c17399db0_1024.jpg" data-height="499" data-width="1049" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60de8ed98e9f092c17399db0_1024.jpg" referrerpolicy="no-referrer"></div></div>官方表示，不会优先考虑特定的某一类残障，希望所有使用 Visual Studio 的残障人士都能够与他们沟通，说出自己对产品的改进需求。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202107/60de8ed98e9f092c17399db1_1024.jpg" data-height="350" data-width="560" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60de8ed98e9f092c17399db1_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家了解到，微软已经列出了多项新的辅助功能：<p></p><p>Whitespace Rendering（空白字符渲染）</p><p>该功能将允许视障人群自定义由空格填充的字符空格数，从而在使用特大字体编程时，可以将缩进改为单个字符宽度，获得更好的体验。</p><p>Option Collecting（精简选项）</p><p>Visual Studio 有数百个选项，允许用户配置和自定义体验以更好地满足他们的需求。然而，过多的选项杂烩在一起，有的是能更改的，有的是不能更改的，这对残障人士来说并不友好。</p><p>所以，微软打算精简 Visual Studio 的选项，只显示用户可用的几个。该功能还将扩展到隐私或安全等其他领域。</p><p>Focus Mode（专注模式）</p><p>微软在 Win10 中推出了 " 专注模式 "，允许用户过滤掉一些通知，防止受到打扰。微软打算在 Visual Studio 里也采用这个功能，用户可以自由选择哪些通知是可以被显示的，以防止在写代码时受到干扰。</p><p>Audio Cues（音频提示）</p><p>微软将把音频提示连接到 Visual Studio 中的事件，以语音的形式播报事件，以帮助视障人群更好地写代码。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
