<p>IT 之家 10 月 21 日消息，苹果周一发布了 14 英寸和 16 英寸新款 MacBook Pro，首次搭载了苹果自研的 M1 Pro 和 M1 Max SoC，现在有网友发现了 M1 Max 的 GFXBench 5 跑分。</p><p>M1 Max 拥有一个 32 核 GPU，每个单元有 16 个执行单元。每个执行单元有 8 个 ALU，使 M1 Max 的 ALU 总数达到 4,096 个。据苹果公司称，该 GPU 的性能高达 10.4 TFLOPs。理论上，M1 Max 的 GPU 性能应该跟 GeForce RTX 3060 移动版相当，后者提供 10.91 TFLOPs 的性能。</p><p>在发布会上，苹果声称 M1 Max 的 GPU（60W）可与英伟达的 GeForce RTX 3080 Mobile（160W）媲美。据悉，苹果 M1 Max 以较低的功率提供同等的性能。该公司还提供了与 GeForce RTX 3080 的 100W 版本的比较，其中 M1 Max 显示功耗降低了 40%。就目前而言，我们应该对苹果公司的数字持谨慎态度，因为该公司没有提供任何说明，只是使用了 " 选定的行业标准基准 "。</p><p>值得一提的是，GFXBench 源于智能手机的基准测试，因此，测试并不适合现代显卡。与 3DMark 不同，建议谨慎对待 GFXBench 的结果。此外，M1 Max 是在 Mac OS X 上使用 Metal API 进行测试的，而 GeForce RTX 3080 Mobile 和 Radeon RX 6800M 是在 Windows 和 OpenGL API 上进行的，所以对比的是离屏成绩。</p><p></p><div class="img_box" id="id_imagebox_0"
