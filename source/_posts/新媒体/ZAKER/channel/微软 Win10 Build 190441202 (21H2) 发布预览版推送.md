<p>IT 之家 9 月 2 日消息 微软近期面向 Release Preview 发布预览频道推送了 Windows 10 21H2 Build 19044.1202 ( KB5005101 ) 。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202109/6130cb528e9f09675415bf8d_1024.jpg" data-height="808" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202109/6130cb528e9f09675415bf8d_1024.jpg" referrerpolicy="no-referrer"></div></div>该更新将自动提供给已经在发布预览频道中使用 Windows 10 版本 21H2 的 Windows Insiders。对于尚未使用 Windows 10 版本 21H2 的内部人员 -- 该更新是通过在 Windows Update 中的 " 探索者 " 体验提供的，并且只提供给那些因为他们的电脑不符合 Windows 11 的硬件要求而从 Bete 频道转移到 Release Preview 频道的内部人员。这意味着这些内部人士将需要进入设置 > 更新和安全 > Windows Update，并选择下载和安装 Windows 10 版本 21H2。<p></p><p>IT 之家获悉，该更新除了作为 Windows 10 Build 19044.1200 的一部分列出的所有改进外，还包括以下变化。</p><p>修复了导致 Windows Update 设置页面在用户下载一个可选的更新后停止响应的问题。</p><p>要检查并查看你是否在运行 Windows 10 版本 21H2，只需在任务栏上的 Windows 搜索框中输入 "winver"。它应该显示为 "21H2 版本 "。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
