，简单查了一下英特尔的 9 代、 10 代、 11 代 CPU 后发现，性能有点落伍的老 CPU 在价格上倒是一点都不落伍。。。<p></p><p>这背后，难道有一场英特尔割韭菜的阴谋吗？难不成老黄的黑手也已经伸进了 CPU 领域？</p><p>这个问题啊，归根结底，还是英特尔换主板换得太勤了。</p><p>首先，英特尔 CPU 并不是随便找块主板安上去就能用的，它们支持的主板都不太一样，通常来说，某一代 CPU 只会支持两代主板左右。</p><p>也就是说，当年给 8 代酷睿用的主板，完全不兼容现在的 12 代酷睿，想换新处理器，就得先掏钱买块新主板。</p><p>这就导致了 CPU 和主板价格变成了此消彼长的关系，有些老 CPU 价格高，是因为它支持的主板种类较多并且便宜，而有些老 CPU 价格低，也是因为支持主板少且有点小贵。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" Z390 到 H310 ，统统可以把握住。<p></p><p>即使是高贵的华硕败家之眼 Z390 主板，二手价格也只要七八百左右，而那些 B 和 H 开头的主板更加白菜价，三四百就能买到还不错的了。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div ，可以说是那些不习惯 Windows 10 小伙伴的最佳选择了。<p></p><p>而比较冷门的 CPU 价格就远不如它了，像 i7-6700 虽然只差了两年时间，但价格就缩水了快一半，只要七八百就能买到它了。</p><p>试想一下，倘若你在 2017 年组了一台电脑，当时为了省钱，只用了 i5 的 CPU ，随着日子流逝，你发现这匹老马跑游戏啥的有点力不从心了。</p><p>大部分人想的肯定是换一匹马，而代表马鞍的主板就卸下来继续给新马用，不用再多花钱了。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy 一起换，另一种就是换一颗旧款高性能 CPU 。<p></p><p>但前者不仅多花了钱，还需要重新装一遍系统，再加上走线理线的功夫，几乎等于重新装一次机，要多麻烦就有多麻烦。</p><p>后者就很简单了，拆下散热器，开盖换下 CPU ，再涂一遍硅脂就完事了。</p><p>而且像 8 代酷睿 i7 的性能，拿到今天也完全够用，处理工作兼玩个网游也绰绰有余，相信很多小伙伴在动手能力不够强的情况下，会更倾向于后一种，省时省钱省力。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 "
