<p>IT 之家 8 月 18 日消息，微软近期透露了关于一首歌让 Windows XP 笔记本电脑崩溃的疯狂故事。涉及到的歌曲是美国流行女流行歌手珍妮特杰克逊的《Rhythm Nation》。</p><p>而造成电脑崩溃的原因是，这首歌包含 5400 RPM 硬盘驱动器 ( HDD ) 型号的自然共振频率之一，即使在今天，这在笔记本电脑中也很常见。事实上，即使是在附近没有播放该歌曲的笔记本电脑也会崩溃。然而，微软没有透露具体频率是多少，而且很难确定 HDD 振动频率在 0 到 5000 Hz 之间是哪个，尽管在 12.5Hz、87.5Hz、1100Hz、1450Hz、1700Hz、1850Hz 处出现某些峰值。</p><p>IT 之家获悉，为了解决这个问题，设备制造商必须为特定频率添加一个过滤器以防止电脑崩溃，该过滤器在音频播放期间检测并删除了违规频率。</p><p>如果你已经使用 Windows 10 和 Windows 11 等新的电脑设备，收听这首歌曲时不会造成 PC 崩溃，不过不能排除某些运行 Windows XP 的 PC 电脑还会出现崩溃。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202208/62fdaae78e9f0904a4058576_1024.jpg" data-height="720" data-width="720" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202208/62fdaae78e9f0904a4058576_1024.jpg" referrerpolicy="no-referrer"></div></div>当然，关于自然共振频率的任何故事都离不开 1940 年塔科马海峡大桥的倒塌（轻度至中度的风就可以导致大桥来回摇摆，中心的摆动可达每 4 到 5 秒几英尺。最终造成蛇形共振倒塌）。<p></p><p>此外，拉里奥斯特曼 ( Larry Osterman ) 在一款特定游戏中也有类似的经历，该游戏导致原型 PC 崩溃。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202208/62fdaae78e9f0904a4058577_1024.jpg" data-height="1055" data-width="1094" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202208/62fdaae78e9f0904a4058577_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
