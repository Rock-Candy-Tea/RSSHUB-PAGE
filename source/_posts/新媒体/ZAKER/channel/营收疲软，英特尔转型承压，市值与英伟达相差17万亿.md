@视觉中国<p></p><p>文丨杨剑勇</p><p>纳斯达克指数不断创出历史新高，其中，以苹果、微软、亚马逊、谷歌、Facebook 等为代表的科技巨头表现抢眼，市值屡创新高。截至 7 月 23 日，上述五大巨头市值分别为 2.48 万亿美元、2.18 万亿美元、1.84 万亿美元、1.82 万亿美元、1.05 万亿美元，五大科技巨头累计市值高达 9.37 万亿美元，换算人民币超 60 万亿元。</p><p><strong>值得一提的是，迈进市值 2 万亿美元俱乐部后的微软，继续创出历史新高，能看出这家巨头渡过了中年危机。</strong>不仅在资本市场表现强劲，营收与利润同样也稳健增长，背后得益于纳德拉带领下，积极迈向云、AI 转型，核心产品和战略都发生了巨变，尤其智能云服务已成长为微软最大业务板块，收获全球智能化所带来的红利。</p><p>各行各业利用微软智能云进行创新，由 Microsoft Azure 公有云平台、Microsoft 365 生产力云平台、Microsoft Dynamics 365 商业智能应用平台、Power Platform 低代码开发平台组成的微软智能云矩阵，以及微软开发工具和服务等赋能每一个组织数字化转型。据了解，有超过 95% 的世界《财富》500 强企业采用了微软智能云服务。</p><p><strong>英特尔则没有那么幸运。遥想当年，凭借 Wintel 组合，英特尔与微软站在科技顶端。</strong>如今微软再次焕发生机，摇身一变成为全球最具成长性的科技企业。尽管英特尔多年来积极推动转型，" 以数据为中心 " 方向是核心战略，但遗憾的是，转型战略最重要的板块营收依旧疲软，且芯片市场腹背受敌，面临 AMD、英伟达强大竞争。</p><p>时间变迁，潮起潮落，企业要想屹立不倒，须顺应时代变革，自我变革是驱动企业持续领先的根本动力之一。虽然英特尔依旧活跃在全球科技界，对全球科技发展举足轻重，但面对强大的竞争，营收疲软、制造工艺落后竞争对手以及数据中心板块竞争激烈，复苏前景未卜。</p><p><strong>从营收来看，营收与利润双双下滑。</strong>" 以数据为中心 " 作为英特尔转型核心板块，数据中心事业群营收在 2021 年第二季度下降 9%，营收为 65 亿美元。不过，该板块下营收规模较少的物联网与自动驾驶 Mobileye 营收则表现强劲。该季度，物联网事业群营收为 9.84 亿美元，同比增长 47%；Mobileye 营收为 3.27 亿美元，同比增长 124%。但以数据为中心的核心板块中，数据中心营收下降，使得整体营收疲软。该季度总营收 196.31 亿美元，上年同期为 197.28 亿美元；净利润 50.61 亿美元，上年同期为 51.05 亿美元。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" referrerpolicy="no-referrer"></div></div><strong>从资本市场来看，相比其他大型科技企业市值不断创新高不同的是，英特尔在资本市场低迷形成鲜明对比。</strong>整体来看，营收疲软，尤其转型方向数据中心板块下降，以及芯片工艺、制造等落后于竞争者等多重因素，财报发布后在资本市场遭遇重挫，股价下降 5.29%，总市值为 2150 亿美元。相比英伟达市值，英特尔黯然失色。当前，英伟达市值 4874 亿美元，这个规模相当于 2.2 个英特尔，市值足足高出 2724 亿美元，换算人民币超 1.76 万亿元。<p></p><p><strong>从市场竞争看来，英特尔面临更强大竞争者进入 CPU 领地。</strong>AMD 作为全球第二大 CPU 芯片供应商，也在不断入侵数据中心领域。据官方介绍，包括亚马逊、微软、甲骨文、戴尔、IBM 云、VMware 及其它厂商宣布推出基于新型 AMD EPYC 7Fx2 处理器家族的产品。面对 AMD 大举进入数据中心的同时，英伟达也基于 Arm 构建的新处理器。</p><p>英伟达推出三款基于 Arm IP 构建的新处理器，预示着开始入侵英特尔核心 CPU 领地。例如 Grace 是基于 Arm 架构，具有革命性数据中心 CPU。主要面向大型数据密集型 HPC 和 AI 应用，将推动 AI、云和高性能计算提升至新高度。基于 Grace 的系统与英伟达 GPU 紧密结合，性能将比目前最先进的 NVIDIA DGX 系统（在 x86 CPU 上运行）高出 10 倍。</p><p>当然，在 CPU 领域，英特尔依旧有强大的话语权。但以 AMD、Arm、英伟达等芯片厂商不断向英特尔发起挑战，带来巨大竞争压力。尤其具有高壁垒的中央处理器将会受到英伟达冲击，而英伟达在 GPU 芯片市场一尘绝骑，全球科技巨头发展人工智能，均依赖于其 GPU 方案来训练人工智能系统。到如今再次延伸至 CPU 领地，未来更具想象空间。</p><p><strong>额外要指出的是，不仅 AMD、英伟达等芯片厂商直接竞争，包括科技巨头们也在积极推动自研芯片，以此减少对英特尔的依赖。</strong>苹果则是最具代表性的厂商之一，采用 ARM 设计电脑芯片。库克在今年春季发布会上表示：搭载苹果自研芯片 M1 的 Mac 电脑销量已经超过了搭载英特尔处理器的 Mac 电脑。且新款 iPad Pro 也用上了这款芯片，速度刷新 ipad 记录。此前苹果官方介绍显示：M1 拥有苹果公司迄今最快的中央处理器核心。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" 市场造出比苹果 M1 更好的芯片。不仅苹果，还有华为积极发展 ARM 服务器芯片、亚马逊 AWS 云服务器也在采用 ARM 设计芯片。整体来看，个人计算电脑在苹果带动下，对 Arm 对产业链影响深远，其他品牌可能会加大 ARM 架构上发力，无形中会给英特尔带来竞争压力。<p></p><p>最后，高性能 CPU 英特尔依旧是不可忽视的重要力量。但处在转型关键时期，这家拥有超 50 年历史的半导体厂商面临众多挑战，面对飞速发展的信息科技，相比其他科技巨头而言，显得力不从心。新 CEO 能否重塑昔日辉煌也值得期待。在物联网、人工智能带来的智能化浪潮下，能为半导体厂商带来巨大机遇。<strong></strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
