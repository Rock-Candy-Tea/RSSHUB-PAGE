<p>今年早些时候，微博、抖音、知乎、快手等知名互联网社区纷纷开始上线账号 IP 属地展示功能，在用户 ID 后显示账号 IP 属地的大致位置。今天，B 站官方账号哔哩哔哩社区小管家发布《关于在个人主页、信息发布等位置展示账号 IP 属地的公告》，宣布 B 站也将上线该功能。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202207/62d4f4c68e9f0928ba462676_1024.jpg" data-height="553" data-width="697" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202207/62d4f4c68e9f0928ba462676_1024.jpg" referrerpolicy="no-referrer"></div></div>根据公告内容，<strong>为了保障社区真实有序的讨论氛围，持续防治冒充热点事件当事人蹭流量等不良行为，社区将逐步开放 " 展示账号 IP 属地 " 功能，相关功能今日起逐步开放测试，预计在一周后正式上线。</strong><p></p><p>功能上线后，用户能够在个人主页、信息发布页等位置查看 IP 属地，境内展示到省（自治区、直辖市），境外展示到国家（地区），属地信息已网络运营商提供信息为准，用户无法关闭。</p><p>值得注意的是，IP 属地并不等于 IP 地址，<strong>公开的 IP 属地仅能获取用户的大致位置信息，无法直接精准定位，用户并不需要担心隐私会因此泄露。</strong></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202207/62d4f4c68e9f0928ba462677_1024.jpg" data-height="467" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202207/62d4f4c68e9f0928ba462677_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
