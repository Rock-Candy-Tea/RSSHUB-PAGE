<p>面向 Beta 和 Dev 频道的 Windows Insider 项目成员，今天微软推出了 Windows 11 Build 22616 预览版更新。该预览版在引入常规修复和性能优化之外，还为 Xbox Game Bar 引入了新的控制栏功能。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div 游戏手柄配对或连接到注册在 Dev 或 Beta 频道的 Windows 11 电脑上时，控制器栏将打开。你会看到最近玩过的三个游戏，以及 Steam 或 EA Origin 的游戏启动器。启动器也可以通过 Xbox 按钮手动调用。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div explorer.exe 的性能和可靠性可能随着时间的推移而降低的地方进行调整，最终导致错误检查。小工具、设置、窗口化、任务管理器和 Windows 沙盒的崩溃也得到了修复。<p></p><p>Windows 11 Build 22616 修复内容如下</p><p>● 修复了一个问题该问题导致一些应用程序在使用日文 IME 时按下半宽 / 全宽键时崩溃。</p><p>● 修复了一个问题即如果你试图从屏幕侧面用手势打开小工具板，你会看到小工具板打开，然后立即关闭。</p><p>● 修复了一个问题即用户无法使用刷卡功能来可靠地调用 Widgets。</p><p>● 修复了一个问题当投射到某些无线设备时，可能导致快速设置崩溃。</p><p>● 修复了一个问题，即使用三指屏幕触摸手势进行最小化可能导致整个系统的动画停止工作。</p><p>● 修复了一个影响最近飞行中 DWM 可靠性的问题。</p><p>● 修复了一个问题该问题导致一些应用程序（如设置）在升级或重启后重新打开时出现空白。</p><p>● 修复了一个问题，当从命令栏点击效率模式时，导致进程列表在应用程序和后台组之间波动。</p><p>● 修复了一个潜在的问题，在某些情况下可能导致 Windows 沙盒中的一些文本显示为黑框。</p><p>● 修复了一个问题导致 Windows Update 通知显示它们是从 "Windows.SystemToast.WindowsUpdate.MoNotification" 而不是 "Windows Update" 发送的。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img
