<p>10 月 28 日晚，红米召开新品发布会，正式带来了 Redmi Note 11 系列，分为 Note 11、Note 11 Pro 以及 Note 11 Pro+。</p><p>现在 Note 11 和 Note 11 Pro+ 两款手机已经来到我们评测室，下面先为大家带来 Note 11 Pro+ 图赏，其双 11 售价仅 1899 元起。</p><p>该机采用双面玻璃机身，梦幻配色搭配硬朗直边设计，可以说是最具设计感的一代 Redmi Note，配色提供浅梦星河、神秘黑境、迷雾森林、时光静紫四种配色，我们拿到的是迷雾森林版本。</p><p>Redmi Note 11 Pro 最大卖点之一就是 120W 快充了，官方称之为 " 神仙秒充 "，不但是 Redmi 首款支持 120W 快充的手机，在同价位机型中也是 " 神仙 " 级别的存在。</p><p>Redmi Note 11 Pro+ 采用了行业内先进的高效率电荷泵，配合 MTW 多极耳双串电芯有效降低充电阻抗，降低温升。同时内置等效 4500mAh（2250mAh x 2）大电池，1 分钟充电 10%，5 分钟充电 45%，仅 15 分钟充至 100%。</p><p>新机配备 6.67 英寸三星 AMOLED 屏幕，2400x1080 分辨率，120Hz 高刷新率，360Hz 触控采样率。它还支持 DCI-P3 广色域显示，支持息屏显示，360° 前后双光线传感器，自动亮度调节更顺滑自然。</p><p>性能方面，Redmi Note 11 Pro+ 搭载联发科天玑 920 芯片，采用台积电 6nm 工艺制程，2x 最新一代 Cortex-A78 + 6 x A55 处理器，搭配 4 核心 Mail-G68 图形处理器，安兔兔跑分 50 万，支持《王者荣耀》90 帧模式。</p><p>Redmi Note 11 Pro+ 支持 SA/NSA 双模 5G，支持双 5G 待机，支持 5G 双载波聚合，支持 MediaTek 5G UltraSave 智能功耗管理技术。</p><p>影像方面，Redmi Note 11 Pro+ 搭载后置三摄方面，分别是 1.08 亿像素主摄（1/1.52 英寸超大底，九合一 2.1μm 大像素，F/1.89）、800 万超广角相机（120° 超大视角）、200 万微距相机（1.75μm 大像素）。</p><p>其他配置方面，对称式立体声扬声器、杜比全景声、多功能 NFC、X 轴横向线性马达、3.5mm 耳机接口、红外遥控、IP53 生活防泼溅等一个不缺。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div
