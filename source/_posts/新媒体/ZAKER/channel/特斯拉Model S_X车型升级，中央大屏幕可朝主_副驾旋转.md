<p>IT 之家 5 月 4 日消息，特斯拉对 Model S 和 Model X 车型进行了更新，为中控屏幕配备了新的电动旋转功能，提供更好的观看体验。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202205/6271e4158e9f097f8f596ef7_1024.jpg" data-height="671" data-width="1425" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202205/6271e4158e9f097f8f596ef7_1024.jpg" referrerpolicy="no-referrer"></div></div><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202205/6271e4158e9f097f8f596ef8_1024.jpg" data-height="479" data-width="607" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/6271e4158e9f097f8f596ef8_1024.jpg" referrerpolicy="no-referrer"></div></div>在过去的几年里，特斯拉在车内娱乐方面投入了大量的精力，从流媒体服务的特斯拉剧院到视频游戏。该汽车制造商甚至还在围绕娱乐做一些硬件设计，特斯拉的两款旗舰车型之前更新了新的大尺寸水平中央显示屏，更适合观看视频，还有一个由游戏电脑驱动的新后座显示屏。<p></p><p>现在，特斯拉通过增加旋转功能，使 Model S 和 Model X 的中央显示屏更加适合娱乐。视频显示，Model S 和 Model X 旋转功能并不是像比亚迪那种水平或垂直旋转，而是向主驾或副驾水平倾斜，使驾驶员或前排乘客获得更好的观看体验。</p><p>值得一提的是，视频中的自动开启前备箱功能是车主自己加装的，并非特斯拉官方功能。</p><p>目前还不清楚特斯拉何时对 Model S 和 Model X 进行了改动，但发布视频的车主提到该车是在 4 月的最后一周生产的，因此猜测自那时起生产的 Model S 和 Model X 车辆都配备了这个新版本的中控显示屏。</p><p>IT 之家了解到，特斯拉目前不支持在车辆未停放时在中央显示屏上播放视频，但随着特斯拉实现更高水平的自动驾驶，这种情况有望在未来得到改变。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
