三件套，这家老牌科技公司似乎就再没什么拿的出手的产品。</p><p>然而在资本市场，却是另一番光景。</p><p>截至 2021 年底，微软过去六年的股价一路狂飙，市值翻了 4 倍多，最高突破 2.5 万亿美元，超越苹果登顶全球市值第一 [ 1 ] [ 2 ] 。上个月刚公开第三季财报，微软就因为强劲的业绩表现，收获华尔街投行的一片看好，被分析师评价为美股受困于通胀之下的 " 最佳防守选择 " [ 3 ] 。</p><p>如今的微软，到底靠什么挣钱？又如何做到几十年屹立不倒呢？</p><p><strong>———</strong></p><p><strong>微软倒闭？都是你的错觉</strong></p><p>中文互联网世界里，流传着微软创始人比尔 · 盖茨的一句名言，" 微软离破产只有 18 个月 "，可能让很多人以为这家公司曾经陷入倒闭危机，但这句话原本只是为了鼓励创新 [ 4 ] 。</p><p>另外一种可能是，大众对于科技公司的感知，往往体现在其推出的消费级产品和工具身上。</p><p>比如上个世纪大名鼎鼎的 Windows95，微软仅凭这一版操作系统就奠定了人们心中 "PC 霸主 " 的地位 [ 5 ] ，再加上 Office、IE 浏览器、MSN 等风靡一时的数字工具，也加深了印象。</p><p></p><div class="img_box" Windows 跟 Office / 图虫创意<p></p><p>了解微软的人，可能还会将 " 倒闭 " 言论的起源，归结于它经历过的一段低谷时期。</p><p>进入 21 世纪，当互联网浪潮从 PC 市场转向移动端，搜索引擎、智能手机、电子商务、社交媒体等风口 " 吹 " 起了一波新兴的科技巨头。互联网新贵们占领媒体头条，引起大众热议，成为华尔街的新宠，身家也水涨船高，一个接一个挤入千亿美元俱乐部。</p><p>到了 2012，有媒体看来，接近 40 岁的微软也跟同时代的科技公司一样，因为没跟上潮流，错过移动互联网的风口，遭遇了 " 中年危机 " [ 6 ] 。</p><p>2012 年第四财季，微软迎来 1986 年上市以后的首次亏损，市值也在 2500 亿美元上下徘徊 [ 7 ] [ 8 ] 。</p><p>然而这段时期并没有持续太久。事实上，当萨提亚 · 纳德拉作为第三任总裁在 2014 年接手微软之后，这家老牌科技巨头就一步步获得了 " 重生 " [ 9 ] 。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div 。</p><p>时至今日，微软的印钞能力依然恐怖。</p><p>单是 2022 财年第三季度，微软就录得 494 亿美元营收，超过腾讯 2021 半个财年的收入，净利润 167 亿美元，比 " 宇宙行 " 工商银行（2022 年第一季度）还多出 200 多亿人民币 [ 12 ] [ 13 ] [ 14 ] 。</p><p>这么大体量的公司，近五年的净利润年复合增长率更是接近 24% [ 15 ] 。作为参考，比微软晚一年成立的苹果，在这段时间的净利润年复合增长率约为 14% [ 16 ] 。</p><p>放在哪个行业都相当厉害的增速，当然不是靠着吃老本就能轻松取得。否则，微软怎么会放任你电脑里的盗版 Office 软件一直 " 存活 " 到今天。</p><p>过去 20 年，一直都在扩张战略版图的微软身上，多了很多标签，电脑公司、手机公司、游戏公司、云服务公司、社交媒体公司，还买了一堆你可能很熟悉的公司，包括领英、Skype、Github，诺基亚手机业务，甚至 " 差点 " 收购了 Tik Tok 美国业务。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div Office，微软已经进行了多次跨界转型，试图在新的时代重新定义自己作为科技巨头，而非软件公司的身份。</p><p>不变的是，这家公司其实一直都活的好好的。</p><p><strong>———</strong></p><p><strong>巨头的印钞机，</strong></p><p><strong>不再只有 Windows 和 Office</strong></p><p>即便存在感不如从前，诞生于 1975 年的微软依然横跨 PC、移动互联网两个时代，20 多年坚守在全球市值最高的十家公司名单之内 [ 8 ] [ 17 ] ，可以说跟倒闭毫不沾边。</p><p>这背后主要的 " 功臣 "，首先就是 Windows 这项业务。</p><p>真正开启一个 " 视窗化 "PC 时代的是 Windows95。开始菜单、任务栏、文件管理器、拨号上网、即插即用，还有《扫雷》、《空当接龙》等趣味游戏，功能强大的 Windows95，一经上市就引发抢购热潮，第一年卖出了足足 4000 万套 [ 18 ] ，当年全球 PC 销量也才不过 6000 万台 [ 19 ] 。</p><p></p><div class="img_box" id="id_imagebox_4" Flicker@John Pasden<p></p><p>消费市场反应火热，微软的股价应声上涨，1995 年夏天其市值达到 500 亿美元，第一次进入标普全球市值最高的十家公司行列，也是其中唯一的科技公司 [ 17 ] 。</p><p>除了创造财富，Windows 的重要意义还在于，它让微软得以构筑起一条很宽的经济护城河。</p><p>Windows 是一个操作系统，也是一个开发平台。在不断更新版本的同时，微软也基于这个平台推出了很多应用软件，比如 Office 三件套、IE 浏览器、Exchange 邮件等等。当微软选择将一些应用软件捆绑在 Windows 平台上，软件行业的单个竞品往往很难招架 [ 20 ] 。</p><p>比如上个世纪末，IE 浏览器就曾击败市占率一度接近 90% 的网景浏览器 [ 21 ] [ 22 ] 。这一切都基于 Windows 的 " 垄断 " 地位。2000 年 Windows 操作系统的市场份额已经达到了 97% [ 23 ] 。</p><p>随着时代变迁，IE 浏览器已被微软淘汰 [ 21 ] ，而 Windows 和 Office 这对 " 黄金搭档 " 受益于网络效应（使用人数越多，越难被替换），依旧牢牢霸占 PC 行业第一 [ 24 ] [ 25 ] 。</p><p></p><div 的贡献固然重要，但是要说真正扛起微软 2 万亿市值，推动其高速增长的，其实是一项你可能不太了解，但已经离不开的服务——云计算。</p><p>在线协作办公、远程会议、地图导航、数据存储，跟朋友微信聊天，或者随便去谷歌搜索一个问题的答案 ...... 这么说吧，如今你在互联网上的一切活动，很有可能都是基于云计算。</p><p>微软提供的云计算服务，直接面向消费者的，包括可以让你共享文件、在线编辑的 Office 365，基于实时聊天的团队协作工具 Teams，还有去年刚推出的云游戏服务 xCloud 等等。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_6" Flicker@Marco Verch<p></p><p>不过要说赚钱，面向企业级用户的商用云服务，才是微软的重中之重。</p><p>2021 财年，微软 1680 亿美元的总营收中有 691 亿美元，也就是 41% 来自于商业云。公有云平台 Azure 是驱动这一增长的关键部门，可以说是微软最新的 " 摇钱树 " [ 26 ] 。</p><p>Azure 具体做什么呢？简单来说，Azure 提供三种服务，SaaS ( 软件即服务 ) 、PaaS ( 平台即服务 ) 、IaaS ( 基础设施即服务 ) ，区别在于用户获得的云计算服务的级别。</p><p>举个例子，如果你要建一个网站，SaaS 就类似于一站式服务，从服务器到开发的应用程序，甚至设计模板，都由云服务公司提供，你只需要在用户界面管理即可。</p><p>而 Paas、Iaas 留给用户的自主权依次上升，Paas 可以让用户在云平台部署应用程序，Iaas 则进一步可以安装操作系统、或安装应用程序等各种软件。</p><p>Azure 的客户遍布各行各业，《财富》杂志评选的 500 强企业中有 95% 都使用过它的服务 [ 27 ] ，比如苹果，曾经有段时间 iCloud 的部分用户数据就跑在 Azure 上（后来转到了谷歌） [ 28 ] 。</p><p>除此之外，微软的第四大业务——游戏，近年的业绩表现也很亮眼。</p><p></p><div class="img_box" id="id_imagebox_7" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_7" 。</p><p>错失了移动互联网时代的微软，这次一出手就是 687 亿美元 [ 30 ] ，相当果断。微软也有这个底气，手握 1300 亿美元现金储备，和多元的业务收入，绝对算得上 " 家大也大 " [ 26 ] 。</p><p><strong>———</strong></p><p><strong>" 中年转型 "，微软做对了什么</strong></p><p>在业内人士看来，除了为游戏业务的增长加码，微软收购暴雪很有可能是在下一盘大棋，布局 Windows、云计算之后的第三个增长点。</p><p>这盘棋，就是元宇宙。</p><p>微软近年的一系列动作都跟元宇宙的产业链密切相关，推出混合现实头戴显示设备 Holens，收购语音识别技术公司 Nuance，发布虚拟协作办公平台 "Mesh for Microsoft Teams" [ 31 ] 。而游戏作为元宇宙的入口，或者说最有可能率先应用的娱乐场景，也是目前科技巨头们的必争之地。</p><p></p><div class="img_box" id="id_imagebox_8" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 Professional Photographer<p></p><p>在押注未来之前，为了走出 " 中年危机 "，以一个全新的形象迎接机遇和挑战，微软踏上了转型的道路。</p><p>直观来看，微软做的最主要的一件事就是调整企业战略，发力云计算服务。</p><p>近年来，微软云计算的拳头产品，主要就是前面提到过的 Azure。截至 2022 年 Q2，微软 Azure 收入规模已经超过全球云服务老大亚马逊 AWS 营收的 60%，5 年前这一比例仅为 30% [ 32 ] 。</p><p>得益于 Azure 的增长，从 2017 年底以来，微软的云服务市场份额增长了近 9 个百分点，2021 年第四季度达到 21%，跟亚马逊的差距正在逐渐缩小 [ 33 ] 。</p><p>做了几十年操作系统跟应用软件的微软，具备提供云服务的天然优势，尤其是 SaaS 业务。它让用户通过互联网连接和使用基于云的应用程序，常见有电子邮件、日历和办公工具，比如 Office 365 [ 34 ] 。</p><p>在 SaaS 领域，微软如今已跃居全球第一。调查数据显示，2021 年第二季度，微软以 19% 的市占率力压 Salesforce（11%）、甲骨文（9%）等对手，亚马逊 Saas 云市场份额则未能跻身前 20 [ 32 ] 。</p><p></p><div
