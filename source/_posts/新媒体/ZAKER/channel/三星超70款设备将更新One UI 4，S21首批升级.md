<p>继此前三星方面在海外市场针对 Galaxy S21 系列启动了基于 Android 12 的 One UI 4.0 Beta 版测试后，这一新款系统也引发了外界的诸多关注。日前有消息源透露，官方预计 One UI 4.0 将会面向超过 70 款机型提供，而按照惯例 Galaxy S21 系列大概率将会首批迎来升级。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202109/6134c03a8e9f094eba1aaa85_1024.jpg" data-height="400" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/6134c03a8e9f094eba1aaa85_1024.jpg" referrerpolicy="no-referrer"></div></div>根据目前已经曝光的相关信息显示，此次 One UI 4.0 同样将会带来一些重大的 UI 方面更改，其中包括更新的图标和颜色，以及 Android 12 所提供的 Material You 元素。此外，据称其还将为骁龙 888 和 Exynos 2100 带来进一步的优化，并改进了 Samsung Knox 和 Samsung Notes App。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202109/6134c03a8e9f094eba1aaa86_1024.jpg" data-height="600" data-width="518" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202109/6134c03a8e9f094eba1aaa86_1024.jpg" referrerpolicy="no-referrer"></div></div>随着 One UI 4.0 的 Beta 版测试即将在海外市场率先启动，也意味着其大概率在不久后就将在国内开启，而 Galaxy S21 系列三款机型则将作为首批机型尝鲜。但至于 One UI 4.0 的具体适配时间表，则还有待后期三星方面更进一步消息的确认，因此有兴趣的朋友不妨继续保持关注。<p></p><p>【本文图片来自网络】</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
