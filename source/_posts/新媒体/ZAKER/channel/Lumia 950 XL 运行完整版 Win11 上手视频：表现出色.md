<p>IT 之家 6 月 30 日消息 微软上周正式发布了全新的 Windows 11 操作系统，并且昨天向 Dev 通道推送了首个测试版 Build 22000.51 预览版。目前，关于采用英特尔和 AMD 处理器的现有 PC 是否支持运行 Windows 11 的最终说法仍悬而未决，但基于 ARM 的设备就相对明确了，似乎只有高通骁龙芯片得到官方支持，但这并没有阻止一些开发者让 Windows 11 在其他 ARM 设备上运行，其中就包括旧的 Lumia 950 XL。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202106/60dc16198e9f0953cd34cce9_1024.jpg" data-height="720" data-width="1280" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202106/60dc16198e9f0953cd34cce9_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家昨天曾报道，开发者 @Gustave Monce 成功将最新的 Windows 11 预览版安装到了 Lumia950 XL 手机上，并成功运行，现在这位开发者给出了 Lumia 950 XL 运行完整版 Win11 的快速上手视频。<p></p><p>Lumia 950 XL 运行完整版的 Windows 11，这对许多 Windows 爱好者来说可能并不意外。Lumia 950 XL 在 2015 年推出，开箱运行 Windows 10 Mobile。由于一些开发者的努力，该机也能称运行完整版的 Windows 10 on ARM。</p><p>视频显示，抛开性能方面，Windows 11 在这款六年前的手机上几乎毫无问题地运行，令人印象深刻。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202106/60dc16198e9f0953cd34ccea_1024.jpg" data-height="672" data-width="741" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202106/60dc16198e9f0953cd34ccea_1024.jpg" referrerpolicy="no-referrer"></div></div>值得一提的是，Reddit 用户 theSPEEDCAT 尝试了在 Raspberry Pi 4 上运行 Windows 11，Raspberry Pi 4 是基于 ARM 的设备，搭载博通处理器，已经可以运行各种形式的 Windows 10，包括物联网版，甚至非官方的完整版本。有人指出，该系统运行 Windows 11 的速度明显比 Windows 10 快。<p></p><p>IT 之家了解到，这些实验使用的镜像是基于早期泄露的 Windows 11 版本，可能与最终版本的内容相差甚远。Windows 11 对 ARM64 设备会有什么要求也是未知数，微软不太可能允许操作系统如此轻易地安装在不支持的硬件上。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
