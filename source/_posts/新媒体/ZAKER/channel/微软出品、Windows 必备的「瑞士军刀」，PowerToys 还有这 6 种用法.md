<p>PowerToys 是 Microsoft 推出的系统工具合集，距离少数派上次介绍已经过去近两年时间。虽然目前仍在预览（Preview）阶段，PowerToys 在过去这段时间里依然保持着相对活跃的更新频率，功能也随着一次次更新日渐丰满。</p><p>在最新的 v0.61.0 版本中，PowerToys 已经内置了 12 种不同的系统工具。不少此前在 Windows 上需要借助第三方工具才能实现的功能，通过 PowerToys 的集成变得更加便捷。</p><p>如果你仍未体验过这款工具，本文将以最新版本的体验为例，为你详细介绍 PowerToys 中值得一试的亮点功能。</p><p><strong>▍</strong><strong>如何安装</strong></p><p>PowerToys 开发组提供了三种安装途径 —— GitHub Release、Microsoft Store，以及 WinGet / Chocolatey / Scoop 包管理器。</p><p>安装完成后 PowerToys 就会自动启动，你也可以在开始菜单中找到 PowerToys ( Preview ) 并手动打开。</p><p><strong>▍</strong><strong>多窗口分屏</strong></p><p>微软在 Windows 11 中带来了全新的贴靠布局（Snap Layouts）功能。贴靠布局默认提供 6 种分屏布局，除了像以往那样使用 <strong>Win + 方向键</strong> 或拖动窗口进入分屏，我们还可以拖动窗口到屏幕顶部到贴靠布局的悬浮窗，或将鼠标指针移动到窗口的最大化按钮上悬浮来进行触发。</p><p></p><div class="img_box" id="id_imagebox_0" FancyZones 就要灵活且好用得多。</p><p>FancyZones 的优势在于拖动程序窗口时不会干扰其它程序，按需取用，通过快捷键可以切换需要使用的布局，在 PowerToys 设置中，支持创建自定义窗口布局，对窗口行为进行预设。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div 的设计语言，比较和谐。</p><p><strong>▍</strong><strong>鼠标实用工具</strong></p><p>如果你是大尺寸屏幕或「带鱼屏」用户，除了 FancyZones，PowerToys
