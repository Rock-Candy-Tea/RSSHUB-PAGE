可变光圈显示效果<p></p><p>欧菲光表示，所谓<strong>摄像头光圈指的是让光线进入到传感器内开口孔径的大小</strong>。在快门不变的情况下，光圈越大，进光量越多，画面比较亮；光圈越小，画面比较暗。</p><p>据介绍，欧菲光可变光圈摄像头模组，使用大尺寸感光元器件，搭配 F1.4 的大光圈镜头，同时在 Lens 外表面增加可变光圈马达，<strong>通过可变光圈马达带动 6 片叶片的开合</strong>，使模组实现了光圈从 F1.4 ( 4.0mm ) → F2.0 ( 2.9mm ) → F2.8 ( 2.05mm ) → F4.0 ( 1.44mm ) 的多档调节，<strong>并且各光圈间可以在 20ms 内快速实现切换</strong>。此外，该模组在 F1.4 → F4.0 光圈全行程闭环控制，可通过软件优化实现单颗模组任意光圈的切换。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" 之家了解到，欧菲光指出，通过导入可变光圈技术，欧菲光可变光圈摄像头模组可以自动切换光圈大小，控制光线进入量，<strong>让手机仅使用单颗摄像头模组，无需其他镜头的辅助</strong>，即可在复杂的光线条件下，精准匹配亮光与暗光的场景，并实现在大光圈的背景虚化和小光圈的景深锐利间轻松切换。</p><p>在光线强烈的地方，可以通过缩小光圈的方式带来更清晰的画面；而在低光下或夜间拍摄，增大光圈就可以获得更多的曝光，以降低画面中的噪点。即在不同的亮暗场景下使用不同大小的光圈能够拍出观感更好的照片。此外，还可根据被拍摄物的不同，在大小光圈之间轻松切换。大光圈能够让背景模糊化，更加突显人物，同时也会造成浅景深，在拍摄风光照时，小光圈可能是更好的选择。</p><p></p><div
