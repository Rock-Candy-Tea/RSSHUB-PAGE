class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202110/617a088e8e9f09097e21e4db_1024.jpg" data-height="418" data-width="625" referrerpolicy="no-referrer"></div></div>被拔下的智齿<p></p><p>（图片来源：downtownwhitbydentistry.ca）</p><p>在回答我们为什么要拔智齿这个问题之前，先让我们了解一下什么是智齿。</p><p><strong>还不是为了腾出地方多长点脑子</strong></p><p>智齿又称<strong>" 第三磨牙 "</strong>，通常在 17 岁以后才会萌出，是咱们这一生中最后生长出来的牙齿。此时主人的心智已然成熟，故而人们将其与智慧联系在一起，就有了智齿这个 " 美名 "。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202110/617a088e8e9f09097e21e4dc_1024.jpg" data-height="556" data-width="360" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202110/617a088e8e9f09097e21e4dc_1024.jpg" referrerpolicy="no-referrer"></div></div>第三磨牙，即 " 智齿 " 的位置<p></p><p>（图片来源：Henry Gray）</p><p>但对于一些人而言，智齿似乎是一种生物学上的不幸，这些 " 额外 " 的磨牙似乎不太适应于他 / 她们的口腔。<strong>由于智齿萌出较晚，萌出时口腔内已然没有它们的容身之处，故而经常处在阻生的状态之下</strong>，这会引发许多口腔疾病，以至于每年都有大量的智齿被医生从口腔中取出。</p><p>不过，大家可能不知道的是，比起我们的祖先和近亲，咱们的智齿已经缩小了不是一点半点。<strong>在现代人类中，智齿通常非常小，甚至没有发育；而在许多古人类中，智齿则相对较大，其咀嚼面甚至是现代人类的两倍。</strong></p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202110/617a088e8e9f09097e21e4dd_1024.jpg" data-height="333" data-width="585" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202110/617a088e8e9f09097e21e4dd_1024.jpg" referrerpolicy="no-referrer"></div></div>不同人种间右下颌牙列的比较，最右边的为现代人<p></p><p>（图片来源：论文截图）</p><p>具体而言，<strong>在 700 万年的进化历程中，由于脑容量的增加，人类的颌骨变得越来越短，这样的变化让本来还有一席之地的智齿逐渐变得无处安放</strong>。但颌骨短归短，该长的牙还是会长，为了让颌骨能放下所有牙齿，人类的牙齿变得越来越小，其中呈现出最明显变小趋势的就是位于口腔后部的第三磨牙 ( m3 ) ，也就是智齿。</p><p>科研人员对许多古人类的化石进行了测量和分析，发现确切属于人族的三个属中，真人属 ( Homo ) 的几个人种 ( 如智人、直立人 ) ，其智齿的咀嚼面积几乎都小于南方古猿 ( Australopithecus ) 以及地猿 ( Ardipithecus ) 。即便是在真人属中，200 万年前出现的直立人 ( Homo erectus ) 的智齿咀嚼面积也有现代人的 1.5 倍。</p><p>不过大家可能也发现了，尽管智齿的咀嚼面积在减小，但在前文的牙列化石中，智齿长得还挺正常，远不像现在这样，呈现出恼人的阻生状态。</p><p>那么问题就来了，如果咱们的智齿都像咱们的祖先一样，长得乖乖的，那不仅不用拔除，还可提供额外的咀嚼功能，这岂不是白挣了四颗植牙的钱吗（歪，诺贝尔吗，经济学奖什么时候颁给我）？</p><p>事实上，现代人中，确实有部分人的智齿表现得十分正常，智齿的冠方最高缘与咬合平面平齐，<strong>倘若不是病理性的，就不需要拔除</strong>。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" data-original="http://zkres2.myzaker.com/202110/617a088e8e9f09097e21e4de_1024.jpg" data-height="214" data-width="866" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202110/617a088e8e9f09097e21e4de_1024.jpg" referrerpolicy="no-referrer"></div></div>高位智齿、中位智齿以及低位智齿<p></p><p>（图片来源：论文截图）</p><p>故而，当我们在问起 " 为什么要拔智齿 " 时，其实是在询问这样一个问题：<strong>为什么有些人的智齿会呈现病理性的阻生状态？</strong></p><p>问题的答案，可不仅仅是因为咱们的颌骨变短。</p><p><strong>人类吃 " 软饭 "，智齿来背锅</strong></p><p>众所周知，颌骨的进化其实是一个十分漫长的过程，当比较当代人和几个世纪前的人们的下颌的大小时，研究人员并未发现显著的差异，那问题究竟出在哪了呢？</p><p>咱们把格局打开，简单分析一波：<strong>颌骨如果没有显著变化，那在空间有限的情况下，差异是不是就出在牙齿们对空间的利用率上呢？</strong></p><p>答案或许是肯定的。</p><p>在研究单个牙齿时，科学家们发现<strong>500 年前人们的牙齿的磨损率要高得多</strong>，粗糙的饮食使得牙齿的邻间磨损增加，进而减少了牙齿的整体长度和宽度，而换牙又是一个渐进的过程，这就让那时的人们在同样有限的空间里，塞下了 32 颗牙齿。而相比 500 年前的人们，柔软的饮食让咱们牙齿间的缝隙增大 ( 怪不得吃啥啥塞牙 ) ，到最后，留给智齿的空间就很少了，这也是<strong>当代人牙冠被其他组织覆盖的比例要高很多 ( 75% ) 的原因之一</strong>。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" data-original="http://zkres1.myzaker.com/202110/617a088e8e9f09097e21e4df_1024.jpg" data-height="354" data-width="631" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202110/617a088e8e9f09097e21e4df_1024.jpg" referrerpolicy="no-referrer"></div></div>发现于 1600 年前墨西哥遗址的头骨，可以看出有第三磨牙（图片来源：ancient-origins.net）<p></p><p>大家可能也发现了，<strong>问题的本质在于咱们现在食用的食物越来越软</strong>。</p><p>在人类进化的过程中，我们牙齿的变化似乎与饮食以及食物制备技术的改变保持一致：</p><p><strong>·</strong>直立人时代，人们已经掌握了用石器去敲击和烹饪食物，这使得咀嚼更加容易，巨大的磨牙对于直立人们开始变得不那么重要；</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" data-original="http://zkres2.myzaker.com/202110/617a088e8e9f09097e21e4e0_1024.jpg" data-height="452" data-width="866" Science）</p><p><strong>·</strong>大约 12,000 年前，人类开始尝试耕作和驯养野生动物。这种生活方式的改变使得人类可以生产小麦粉、牛奶，进而制成更加柔软的食物，人们的牙齿也因此越来越小；</p><p><strong>·</strong>如今，工业化生产让我们的食物变得更加精细和柔软，随之而来的就是越来越多的人存在智齿的病理性阻生问题，只能去寻求牙医的帮助。</p><p>有的同学可能要问，牙齿变小不是节约空间吗，这样一来，就算颌骨变短，智齿也能有容身之所啊 ~</p><p>这话放在工业化时代到来之前确实没毛病，毕竟当时的 " 柔软 " 的食物，嗯，可能放在现在，也不是那么好嚼。工业化时代的到来，彻底打破了牙齿大小和颌骨长度之间的微妙平衡。从进化的角度来看，几个世纪不过是昙花一现——几乎没有足够的时间让我们的牙齿和颌骨适应柔软的加工食品所带来的变化。</p><p>事实上，<strong>柔软的食物不仅让现代人牙齿之间的缝隙变大，缺乏咀嚼锻炼还会进一步影响人们下颌骨的发育，此消彼长之下，最后萌出的智齿自然而然就没有了容身之处</strong>，成为了让人烦恼的问题。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_6" </p><p>当捋清了思路之后，不难发现，<strong>现代人中频发的病理性智齿阻生，或许是一个发育问题，而不是一个进化问题</strong>，这就让我们可以通过拔牙以外的方式去解决智齿的困扰。</p><p><strong>远离智齿困扰，换个吃法试试</strong></p><p>科学家早就注意到，现代人中许多与牙列相关的问题，比如智齿阻生、牙齿拥挤，以及咬合不正，<strong>其根源或许不在于遗传差异，而是植根于生活方式的不同</strong>。过去的一项研究表明，营狩猎采集生活的人群具有相对长而窄的下颌，而以农耕为生的人群具有相对短而宽的下颌骨，这样较短的下颌骨容易带来更多的牙齿问题。</p><p></p><div referrerpolicy="no-referrer"></div></div>不同生活方式人群的下颌骨的形态特征存在差异<p></p><p>（图片来源：论文截图）</p><p><strong>显然，在短短几代人的时间里，进化的力量还不足以让我们的下巴显著缩小</strong>。同样，也没有任何证据表明选择压力会让下颌较长的人群产生更多的后代，从而使得这种特征永久化。</p><p><strong>恰恰相反，不同的生活方式会在短时间内让人群发生显著的生理变化</strong>。例如，自二战以来，日本男性在儿童时期从大量食用大米转向食用更多的乳制品和蛋白质，从而使得日本男性成年后的平均身高增加了大约 5 英寸。</p><p>这样显著的变化同样表现在下颌的形态上，以印度的爷孙三代为例，爷爷出生在印度的乡村，年轻时带着孩子来到英国，孙子则出生于工业化社会，不过是祖孙三代的时间，我们就可以发现孙子比起爷爷，已经是明显的 " 天包地 "。</p><p></p><div class="img_box" id="id_imagebox_8" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_8" data-original="http://zkres2.myzaker.com/202110/617a088e8e9f09097e21e4e3_1024.jpg" data-height="291" data-width="543" 进展》的新研究中，科学家发现人类的下颌牙弓直到 22 岁才不再继续伸长。</p><p>所以 ...</p><p>美好的生活，从锻炼下巴开始？</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
