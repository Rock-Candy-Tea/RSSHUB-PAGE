<p>IT 之家 9 月 24 日消息 今日，微软向发布预览通道的用户推送了 Windows 11 首个发布预览版 Build 22000.194。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202109/614d0ef98e9f0908414a6b8f_1024.jpg" data-height="394" data-width="554" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/614d0ef98e9f0908414a6b8f_1024.jpg" referrerpolicy="no-referrer"></div></div>应用更新：<p></p><p>今日推出的版本包括以下应用程序更新：截图工具、计算器和时钟。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202109/614d0ef98e9f0908414a6b90_1024.jpg" data-height="569" data-width="491" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202109/614d0ef98e9f0908414a6b90_1024.jpg" referrerpolicy="no-referrer"></div></div>修复：<p></p><p>解决了一个问题，如果您启用然后禁用对比主题，它会导致标题栏中出现伪影，在某些情况下使最小化 / 最大化 / 关闭按钮难以看到和使用。</p><p>修复了某些连接设备可能导致无法使用蓝牙的崩溃问题。</p><p>缓解了导致字幕未按预期出现在某些应用程序中的问题，尤其是日语字幕。</p><p>修复了导致某些 PC 在现代待机期间进行错误检查的问题。</p><p>缓解了在 " 设置 " 中的搜索框中输入某些第三方输入法时可能导致候选窗口呈现在屏幕上的其他位置（未附加到搜索框）和 / 或插入到搜索框中的字符不显示的问题。</p><p>我们修复了导致 PowerShell 创建无限数量的子目录的问题。当您使用 PowerShell Move-Item 命令将目录移动到其子目录之一时，会出现此问题。结果，音量填满，系统停止响应。</p><p>此版本包括一项更改，使虚拟机 ( VM ) 上的 Windows 11 系统要求的实施与物理 PC 相同。之前创建的运行 Insider Preview 版本的 VM 可能不会更新到最新的预览版本。在 Hyper-V 中，需要将 VM 创建为第 2 代 VM。只要满足硬件要求，在 VMware 和 Oracle 等供应商的其他虚拟化产品中的 VM 中运行 Windows 11 将继续工作。有关 Windows 11 系统要求的更多详细信息 - 请参阅此处的博客文章。</p><p>已知的问题：</p><p><strong>一般的：</strong></p><p>我们正在调查 Beta 频道中来自 Insiders 的报告，其中在升级到 Windows 11 后，他们看不到新的任务栏并且 " 开始 " 菜单不起作用。如果您受到影响，要解决此问题，请尝试转到 Windows 更新 > 更新历史记录，卸载 Windows 的最新累积更新，然后通过检查更新重新安装。</p><p>我们正在修复导致某些 Surface Pro X 出现 WHEA_UNCORRECTABLE_ERROR 错误检查的问题。</p><p><strong>开始：</strong></p><p>在某些情况下，使用从开始或任务栏搜索时可能无法输入文本。如果遇到此问题，请按键盘上的 WIN + R 启动 " 运行 " 对话框，然后将其关闭。</p><p>右键单击 " 开始 " 按钮 ( WIN + X ) 时缺少系统和 Windows 终端。</p><p><strong>任务栏：</strong></p><p>切换输入法时，任务栏有时会闪烁。</p><p><strong>搜索：</strong></p><p>单击任务栏上的搜索图标后，搜索面板可能无法打开。如果发生这种情况，请重新启动 "Windows 资源管理器 " 进程，然后再次打开搜索面板。</p><p>搜索面板可能显示为黑色，并且在搜索框下方不显示任何内容。</p><p><strong>小工具：</strong></p><p>小部件板可能显示为空。要解决此问题，您可以注销然后重新登录。</p><p>小部件可能会在外部显示器上以错误的尺寸显示。如果遇到这种情况，您可以先在实际 PC 显示器上通过触摸或 WIN + W 快捷方式启动小部件，然后在辅助显示器上启动。</p><p><strong>微软商店：</strong></p><p>我们将继续努力提高商店中的搜索相关性。</p><p><strong>Windows 沙盒：</strong></p><p>在 Windows Sandbox 中，单击任务栏上的切换器图标后，语言输入切换器不会启动。作为一种解决方法，用户可以通过以下任何硬件键盘快捷键切换他们的输入语言：Alt + Shift、Ctrl + Shift 或 Win + Space（第三个选项仅在 Sandbox 全屏时可用）。</p><p>在 Windows Sandbox 中，单击任务栏中的 IME 图标后，IME 上下文菜单不会启动。作为变通方法，用户可以使用以下任一方法访问 IME 上下文菜单的功能：</p><p>通过设置 > 时间和语言 > 语言和区域 >（例如，日语）三点 > 语言选项 >（例如，Microsoft IME）三点 > 键盘选项访问 IME 设置。</p><p>或者，您还可以启用 IME 工具栏（一个替代 UI）来快速调用特定的 IME 功能。从上面继续，导航到键盘选项 > 外观 > 使用 IME 工具栏。</p><p>使用与每种 IME 支持的语言相关联的一组独特的硬件键盘快捷键。（请参阅： 日语 IME 快捷方式、 繁体中文 IME 快捷方式）。</p><p>IT 之家了解到，Windows 11 正式版将于 2021 年 10 月 5 日推出。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
