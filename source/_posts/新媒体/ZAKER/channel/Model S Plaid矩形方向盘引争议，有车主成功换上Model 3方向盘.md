<p>IT 之家 11 月 4 日消息，据 Electrek 报道，一位特斯拉 Model S Plaid 车主成功将矩形方向盘换成了普通的 Model 3 方向盘，包括方向盘柄等所有部分。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202111/61835ce28e9f09331d33dde5_1024.jpg" data-height="572" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/61835ce28e9f09331d33dde5_1024.jpg" referrerpolicy="no-referrer"></div></div>报道称，特斯拉 CEO 埃隆马斯克（Elon Musk）证实，特斯拉不打算提供普通方向盘的选项。这一点的确认，为 Model S Plaid 和其他所有使用新方向盘的特斯拉汽车改装创建了一个新市场。<p></p><p>Model S Plaid 早期车主 Ryan Huber 最近完成了对方向盘的改造，甚至成功让 Model 3 的方向盘与 Model S 软件兼容。</p><p>IT 之家了解到，Huber 表示，他不打算把自己的改造作为套件出售，但他正在 Github 上分享他改造的过程，以帮助其他想要尝试这样做的人。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202111/61835ce28e9f09331d33dde6_1024.jpg" data-height="1422" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/61835ce28e9f09331d33dde6_1024.jpg" referrerpolicy="no-referrer"></div></div>需要注意的是，这并非一件容易的事。以此次改造为例，Huber 必须编写自定义固件才能使 Model 3 方向盘在 Model S Plaid 软件上工作。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
