<p>Windows 11 正式版已经推送，新版系统需要 TPM 2.0 才能运行。</p><p>今天，微软帮助和支持官方微博解释了微软 Windows 11 操作系统为什么需要 TPM 2.0。</p><p>微软表示，<strong>我们之所以需要 TPM 2.0 才能运行 Windows 11，这是因为它是安全相关功能的重要构建基块。</strong></p><p><strong>微软认为 TPM 2.0 可以提供基于硬件的隔离和加密，有利于结合 Windows 11 系统的安全特性以应对威胁，保护个人信息。</strong></p><p>据悉，TPM 英文全称 Trusted Platform Module ( 可信平台模块 ) ，是一项安全密码处理器的国际标准。TPM 就是利用经过安全验证的加密密钥为设备带来更强的安全性，它也是许多安全应用的核心，Windows 11 升级要求的 TPM 2.0 就是最新的版本。</p><p>TPM 被用来进行密钥计算和加密存储，Windows 必须借助这个芯片才能实现 Windows 设备加密、高级 BitLocker 加密等安全特性，帮助 Windows 更加严格地保护用户数据。</p><p>由于 TPM 使用的是完全的专用电路，整个计算和存储的过程都可以不经过内存，并不会在硬盘中留下痕迹，所以密钥生成、核验及存储安全性非常之高，这有点类似于苹果产品中专门用来存放加密数据的 T 系列芯片。</p><p><strong>检查方法：</strong></p><p>按 Windows+R 打开 " 运行 "，或直接在 Windows 10 搜索栏中输入 tpm.msc 查看结果。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202110/616ae68e8e9f093ab321f186_1024.jpg" data-height="569" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202110/616ae68e8e9f093ab321f186_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
