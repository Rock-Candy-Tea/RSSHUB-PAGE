<p>9 月 10 日消息，就在传统汽车制造商福特挖走苹果汽车项目负责人道格 · 菲尔德 ( Doug Field ) 两天之后，苹果决定让苹果手表项目高管、前 Adobe 首席技术官凯文 · 林奇 ( Kevin Lynch ) 负责苹果汽车项目。</p><p>这也是苹果汽车项目 " 泰坦计划 " ( Project Titan ) 最近一次调整负责人。随着负责人的更替，苹果汽车项目重点已经发生变化。据周四早些时候报道称，苹果可能不再计划与其他汽车制造商合作推出自动驾驶电动汽车。报道称苹果现在计划单干。</p><p>林奇于今年 7 月份开始参与苹果汽车项目，当时他协助开发苹果汽车软件。2013 年，林奇辞去工具开发商 Adobe 首席技术官一职加入苹果。在 Adobe 工作时，林奇曾就苹果将 Flash 排除在 iOS 系统之外的决定作出官方回应。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202109/613ad1618e9f0960b7295670_1024.jpg" data-height="440" data-width="660" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202109/613ad1618e9f0960b7295670_1024.jpg" referrerpolicy="no-referrer"></div></div>负责苹果汽车项目后，林奇仍向苹果首席运营官杰夫 · 威廉姆斯 ( Jeff Williams ) 汇报工作，而不是向去年被公司要求监督苹果汽车项目的人工智能负责人约翰 · 詹安德里亚 ( John Giannandrea ) 。<p></p><p>苹果汽车前任负责人菲尔德从 2018 年开始负责这一项目，直到本周被福特挖走。在此之前，他曾在特斯拉工作数年，帮助特斯拉推出 Model 3 等车型。</p><p>据报道，苹果于 2014 年上线电动汽车项目，一波三折后将项目重点转向开发自动驾驶技术。但最近，苹果似乎又开始与主要汽车制造商展开了对话，还考虑过收购电动汽车初创公司 Canoo。不过，苹果汽车项目现在到底将走向何方，现在还未有定论。（辰辰）</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
