<p>"What is dead may never die." 与《权力的游戏》中的白行者一样，IE 浏览器也加入了亡灵的行列。微软在 Windows 11 中驯服了 IE 兽，防止任何人召唤它并运行其古老的代码，但现在一位 Twitter 用户打破了枷锁，让 IE 在 Windows 11 上自由运行。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202208/62ec8d888e9f0953510544a5_1024.jpg" data-height="600" data-width="550" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202208/62ec8d888e9f0953510544a5_1024.jpg" referrerpolicy="no-referrer"></div></div>IE 浏览器本来就不应该在 Windows 11 上运行。微软在 Windows 11 中禁用了 IE，使其成为 20 多年来第一个没有 IE 的 Windows 版本。事实上，如果你试图在 Windows 11 中启动 IE 浏览器，操作系统会强迫你进入 Microsoft Edge。<p></p><p>但是 Twitter 用户 @XenoPanther 发现了一个让 IE 在 Windows 11 上运行的方法。虽然这个 Windows 里存在 26 年的明星在 6 月正式被淘汰，但由于操作系统的古老（和被遗忘）的部分让它到 2022 年仍然可以被使用。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202208/62ec8d888e9f0953510544a6_1024.jpg" data-height="508" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202208/62ec8d888e9f0953510544a6_1024.jpg" referrerpolicy="no-referrer"></div></div><strong>方法是：</strong><p></p><p>开始菜单中搜索 Internet 选项，启动控制面板小程序，选择程序标签，点击 " 管理加载项 "，然后点击 " 了解有关工具栏和扩展的详细信息 "。出于某种原因，IE 浏览器会绕过迫使你进入 Edge 的命令而启动。浏览器就像你所期望的那样工作，面对所有的网络不兼容和安全噩梦又会以熟悉的配方重新回归。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
