<p>关于 Radeon RX 6900XTX 的传闻也不是一天两天的事了，实际上之前就有厂商推出采用 Navi 21 XTXH GPU 的显卡，那些显卡比标准版的 RX 6900XT 拥有更高的频率，但名字依然是叫 Radeon RX 6900 XT，那么 RX 6900XTX 又会怎么样呢？</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202108/612e01af8e9f09743708b619_1024.jpg" data-height="900" data-width="1200" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202108/612e01af8e9f09743708b619_1024.jpg" referrerpolicy="no-referrer"></div></div>推特用户 @CyberCatPunk 晒出了一张幻灯片，上面出现了 RX 6900 XTX 版本的字样，它拥有 24.93TFLOP 的浮点计算能力，而目前 RX 6900XT 的浮点计算能力是 23.04 TFLOP，它的 GPU 加速频率是 2250MHz，这样算下来的话 RX 6900 XTX 的加速频率能到 2435MHz，频率提升幅度有 8%，那么性能估计也会涨 8%。<p></p><p>除了 GPU 频率变动之外，搭配的显存频率也更高了，RX 6900 XTX 将使用 18Gbps 的 GDDR6 显存，当然容量还是会维持在 16GB，与现在使用 16Gbps 的 RX 6900 XT 相比，频率提高了 12.5%，显存带宽会增加到 576GB/s，随着 GPU 与显存频率的提高，功耗也肯定会上去的，现在 RX 6900 XT 的 TGP 是 300W，预计 RX 6900 XTX 会涨到 350W。</p><p>没啥意外的话 RX 6900 XTX 应该就是 Navi 21 XTXH GPU 再加高频 GDDR6 显存的变体，目前 RX 6900 XT 用的 GPU 已经把 Navi 21 所有 CU 都开出来了，所以核心规模不会有变化，这种频率提升的变动对于零售市场来说意义并不大，而且现在已经有不少水冷版的 RX 6900 XT 在售，所以这个 RX 6900 XTX 很有可能是仅面向 OEM 市场的。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
