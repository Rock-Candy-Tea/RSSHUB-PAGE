NFT，这些 NFT 运行在以太坊和 Polygon 上。很快，它还会增加对 Solana 和 Flow NFT 的支持。而在此之前（2022 年 5 月），Meta 旗下 Instagram 已经开始向部分用户开放使用 NFT。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202207/62befa6a8e9f09624f5a3ed2_1024.jpg" data-height="498" data-width="589" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202207/62befa6a8e9f09624f5a3ed2_1024.jpg" referrerpolicy="no-referrer"></div></div>NFT，又称非同质化代币，运行在区块链上（比如以太坊、Polygon、Solana 等），每个 NFT 都具有独特性、不可篡改性。NFT 实质上是一段数据，它可以实现数据归个人所有，而非目前的情形，所有的数据基本都掌控在互联网平台手中。<p></p><p>NFT 的形态多种多样，未来你在 facebook 上发的一个帖子、在 Instagram 上发的一张图片，你上传的一段音乐都可能是 NFT。而和过去不一样的是，你在网上发的每个 NFT，都天生自带价格，它只属于你，这一创新从底层推翻了目前互联网的商业模式。互联网创业者不再追求垄断数据、扩大规模，而是追求数据归个人所有，平台服务人们更好交易和流通数据。</p><p>Libra（Facebook 发起的一项试图推翻各国货币主权的加密货币项目）的失败，让扎克伯格认识到激进的策略是不现实的，它会引起传统势力的强硬反扑。但是小扎的野心，并没有消失，他选择了更加缓和的方法。</p><p>NFT，元宇宙的第一步</p><p>如何踏出元宇宙的第一步？</p><p>有的人选择了 AR、VR，有的人选择了物联网，也有的人选择了 Web3（基于区块链的下一代互联网）。而作为全球最大互联网公司的掌门人，扎克伯格不做选择，他都要。</p><p>一位在 Meta 工作的博士汪淼向虎嗅表示：" 目前做 NFT 的团队，应该在 Reality labs。" 汪淼是 Meta 的一员，虽然博士期间研究的领域是人工智能和机器学习，但五六年前便关注 Crypto 和 Web3，直至如今仍保持着强烈的兴趣。</p><p>他不是个例，在美国，不仅硅谷的人才大量涌入 Crypto 和 Web3，华尔街精英和 SEC 官员也将 Web3 作为自己的下一站。</p><p>NFT 则是 Web3 中最性感的应用赛道之一。Meta 产品经理 Navdeep Singh 在周三的 Twitter 中分享了 NFT 在 Facebook 上的样子。用户将在他们的 Facebook 个人资料上拥有一个 "NFT 收藏 " 的地方，他们可以在其中展示他们的 NFT。NFT，代表了你对某些数据的拥有权。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202207/62befa6a8e9f09624f5a3ed3_1024.jpg" data-height="383" data-width="680" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202207/62befa6a8e9f09624f5a3ed3_1024.jpg" referrerpolicy="no-referrer"></div></div>用户能够将他们的加密货币钱包连接到他们的 Facebook 个人资料。他们还可以将他们的 NFT 转换为 Facebook 帖子，帖子和以前一样，人们依然对其做出点赞、评论和分享等反应。<p></p><p><strong>从目前的应用来看，NFT 实际上并没有让 Facebook 变得更好玩，那么为什么 Meta 还要在自己的两大旗舰产品中都加入 NFT 这一元素？</strong></p><p>这或许和 NFT 可以为互联网带来诸多新的特质有关。</p><p>在现实世界中，人们创造一副画，如果这幅画具有非常大的欣赏价值、收藏价值或者艺术价值，它将被收藏者赋予非常高的价格，比如梵高的画。</p><p>然而，在数字时代越累越多的艺术家，包括但不限于画家、作家、音乐家、设计师，他们更倾向于在电脑上进行创作。他们中有许多作品极具价值，而且未来越来越多伟大的作品，将以数字的形式展现。<strong>当今世界，人们在虚拟世界或者元宇宙的时间比在现实世界更多，人们特别是互联网一代的人们有创作、收藏和交易 " 数字化名作 " 的强烈需求。</strong>而 NFT 则为这个需求提供了最好的技术形式，并且将打破互联网上所有东西都免费的状态，激励优秀的创作者创作出优秀的作品。</p><p>而 facebook 进军 NFT，实际上是提前感受到了 " 互联网原住民们 " 对在数字世界创作、收藏、交易和分享的需求，并且进行了布局。</p><p>元宇宙，代表的是一种人类数字化时代发展的必然趋势。微信、抖音、淘宝 ...... 实际上都是一个个 " 简陋版元宇宙 "，代表的是人们可以在网上聊天、欣赏和购物。如果说元宇宙 1.0 十分简陋，只能实现部分沉浸式，那么元宇宙 2.0 则更进一步，让人们更沉浸式的身入其中。</p><p><strong>人们可以将之前很多不可以在互联网（元宇宙 1.0）做的事情，通过技术的进步，可以在元宇宙 2.0 做了。NFT，就是这样的一个技术。</strong></p><p>当代的梵高，或许是一个在电脑上进行创作的年轻人，他创作出的东西极具艺术和人文上的颠覆性，而 NFT 可以将他的作品确权，让他的作品独一无二。</p><p>新时代的周杰伦，是一个在网络上发布音乐作品的年轻人。它的音乐作品，依然会被人们免费传唱，只不过它发布的第一个音乐 NFT，将随着他的音乐地位而让其价值水涨船高。互联网原住民们，会对元宇宙有一种沉浸式的强烈需求。而 NFT 可以实现。</p><p>元宇宙，是现实世界的 " 映射 "。在现实世界，人们有经济系统、有创作和收藏、每一件物品都独一无二各有价格。那么，在元宇宙中，亦是如此。数字货币（FT）对应的是独立的经济系统，非同质化代币（NFT）则对应的是元宇宙中的每一件作品。</p><p>腾讯的 Q 币不能代替数字货币，也没有一个大厂都独立做出独一无二、有共识的 NFT。这是区块链的核心价值所在，大厂不行、中心化机构不行、国家也不行，它可以。</p><p>FT（数字货币）和 NFT（非同质化代币）的融合成为了 Web3。Web3 是沉浸式元宇宙不可或缺的基石。</p><p>Meta 的对手们</p><p>目前，绝大多数美国互联网大厂都意识到了 NFT 和 Web3 的重要性。</p><p>虎嗅在《美国互联网大厂 " 扎堆 "Web3》一文中写到：Meta、谷歌、亚马逊、推特、eBay、奈飞、Paypal、Square 等至少 8 家美国互联网大厂付出实际行动探索 Crypto 或者 Web3。</p><p>尤其是同为社交巨头的推特，它更是领先各大互联网厂商，率先在自己的产品中 " 植入 "NFT。而奈飞等视频巨头，也在其多个优质影视作品中尝试 NFT 的使用，比如《怪奇物语》和《爱、死亡和机器人》。</p><p>许多粉丝对 NFT 嗤之以鼻，将 NFT 技术视为骗局，然而互联网大厂明白自己要做什么。一位投资人曾向虎嗅表示：" 一个现象级的产品火遍全球非常快，到那时再想分一杯羹，是异想天开。" 而不论是美国的互联网巨头、中国的互联网巨头，乃至俄罗斯和日本的互联网巨头，特别是美中日俄四国的社交巨头都已经在布局 NFT，原因也正是在此。</p><p>对于 Meta 而言，他的对手不仅有 Twitter 和 TikTok 等传统互联网巨头，更大的威胁在于 Web3 公司。</p><p>据虎嗅观察，国内有十多家 Web3 社交公司，在美国甚至更多，他们是 Web3 世界的 " 原住民 "，对 NFT 等事物有着更深的认知和探索。他们致力于做出和 Meta 完全不一样的去中心化的社交图谱，这或许能实现用户真正拥有自己的 " 社交关系 "，无论到哪一个平台，用户都能把自己的社交关系进行自主迁移。</p><p>虎嗅发现，在国内，蚂蚁的 NFT 平台鲸探、腾讯的 NFT 平台幻核亦在探索基于 NFT 关系网的社交，只不过目前整个 NFT 的使用场景还过于单一，社交功能未能上线。无聊猿，不仅是一个 NFT，更是一个紧密的社交生态。而随着越来越多的无聊猿出现，关系图谱的极大丰富，基于 NFT 的社交产品或将成为可能。</p><p>届时，Meta 会依旧成为下一个时代的 " 社交霸主 " 吗？</p><div id="recommend_bottom"></div><div
