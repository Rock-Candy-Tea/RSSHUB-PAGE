<p>微软意外地泄露了其下一个操作系统的名称 -- 它可能不是 Windows 11。6 月 24 日，微软将举办一个大型活动，届时公司将公布其 Windows 操作系统的未来。尽管我们知道操作系统的外观和功能将发生变化，但我们不知道的是这个新改进的 Windows 版本将被称为什么。</p><p><strong>01</strong></p><p><strong>Windows 或将学习 macOS 的代号命名法</strong></p><p>在此之前已经有很多猜想，包括 Windows 365、WindowsOS 或简单的 "Windows"，但到目前为止，大多数线索都指向它是 Windows 11。</p><p>从某种程度上说，Windows 11 这个名字很有意义，它帮助微软摆脱了围绕着 Windows 10 的一些负面情绪，同时也对外宣布这是一个巨大的变化的事实。然而，微软过去曾说过，Windows 10 将是 Windows 的 " 最后 " 版本。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202106/60c1de7bb15ec017997380b1_1024.jpg" data-height="185" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202106/60c1de7bb15ec017997380b1_1024.jpg" referrerpolicy="no-referrer"></div></div>虽然该公司完全有权利改变主意，推出 Windows 11，但另一条线索已经浮出水面，表明微软可能计划在其软件的未来版本中采用 macOS 风格的命名惯例。<p></p><p>苹果公司给其操作系统的新更新打上了地理标签，最新的是 macOS Monterey，最近在微软的一些元数据中发现的一个补充信息（后来被删除）表明，之前 Windows 更新的代号 - 太阳谷很可能最终成为产品的实际名称。</p><p>昨天发布的微软支持页面提到了两个不同的 Windows 版本 --Windows 10 和 Windows Sun Valley，你可以在下面的截图中看到这一点。</p><p>此后不久，对 "Windows Sun Valley" 的提及很快被删除，这表明将其列入页面是一个错误。现在，可能是微软无意泄露这个名字，这很可能就是它被删除的原因，当然也可能是这只是新版本的内部名称而已。</p><p>我们不会有太长的时间来确定这些，因为现在离 Windows 发布活动只有两个多星期了。</p><p><strong>02</strong></p><p><strong>微软正开发 Android 模拟器</strong></p><p>除系统外，新证据表明微软正开发适用于 Windows 11 的 Android 模拟器的消息也让不少人关注。</p><p>去年 11 月，有消息称 Windows 10 21H2 功能更新即将支持 Android 应用程序。今天又有了新的证据，微软首席软件设计工程师 Hideyuki Nagase 做了一些准备工作，能够让 Android 模拟器在 Windows Subsystem for Linux 下良好运行。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202106/60c1de7bb15ec017997380b2_1024.jpg" data-height="532" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202106/60c1de7bb15ec017997380b2_1024.jpg" referrerpolicy="no-referrer"></div></div>在 WSLG 预发布版本的条目中写道：" 我们早些时候听说，在 Project Latte 项目下，微软将允许开发者将 Android 应用打包成 MSIX 文件，并通过微软商店发布。然后，这些应用程序将在 Android 模拟器中运行 "。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
