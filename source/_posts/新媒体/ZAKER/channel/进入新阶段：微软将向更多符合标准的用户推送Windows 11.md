<p>有消息称，微软的 Windows 11 更新计划在近日进入了新阶段，将正式向更多符合硬件标准的电脑推送 Windows 11，但这仍需一定时间。</p><p>微软在一段时间的用户测试之后，已经开始进行 Windows 11 的大规模推广工程，只要符合最低硬件标准的电脑在理论上都能够接收此次推送的 Windows 11 系统。</p><p>根据微软的说法，微软将依靠 AI 技术来判断电脑是否符合更新 Windows 11 的最低标准，并将向符合标准的电脑推送新的系统更新。但依旧有用户反映称自己的电脑虽然满足了升级 Windows 11 的标准，却没有获得更新推送。</p><p>据悉，在遇到这种情况后，用户可以选择在设置中自行检查系统更新来获取更新推送，而如果仍无法获得推送可以等待微软后续推送，或是在设置中切换测试版系统。</p><p>Windows 11 作为微软为 Windows 10 打造的 " 接棒人 "，在 UI 设计上进行了相当大规模的改动，在 Windows 11 中微软舍弃了 Windows 10 标志性的磁贴设计，转而将 Windows 10 后期部分采用的亚克力设计作为了主要的设计语言，这使得 Windows 11 在视觉质感以及部分操作上相较 Windows 10 有了不小的变化。</p><p>不过需要注意的是，虽然微软已经开始了对 Windows 11 的广泛推广，但是此前公开的标准依旧存在，对于硬件性能不符合标准的电脑来说想要升级 Windows 11 仍然不是那么方便。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202110/617ce6868e9f09759f1de9bd_1024.jpg" data-height="399" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202110/617ce6868e9f09759f1de9bd_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
