<p>AMD Radeon RX 550 是 AMD 在 2017 年发布的一款入门级显卡，其搭载了 Polaris 12 GPU，采用 14nm FinFET 工艺制造，拥有 512 个流处理器，配备 2GB 或 4GB 的 GDDR5 显存，显存位宽为 128 位，TDP 为 50W。</p><p>据 Hermitage Akihabara 报道，近期在日本地区多家零售商重新发售了 Radeon RX 550，出自 AMD 的合作伙伴玄人志向，型号为 "RD-RX550-E4GB / LP"，为 4GB 版本，配备了 DVI-D+HDMI 输出端口。其长度为 167mm、高度为 68mm、厚度为 16mm，属于半高单槽规格，这点倒是很适合小型主机使用。暂时还能确认到底是 AMD 重新大量提供 Radeon RX 550，还是玄人志向突然找出大批存货。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202201/61ee786d8e9f091284088f30_1024.jpg" data-height="750" data-width="1000" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202201/61ee786d8e9f091284088f30_1024.jpg" referrerpolicy="no-referrer"></div></div>Radeon RX 550 推出的时候，性能要比当时普遍的核显强三到四倍，不过随着时间的推移，其性能甚至被核显赶上。根据目前的基准测试来看，Radeon RX 550 的性能也就比自家的 Vega 11 核显好一些，如果和其他独立显卡比较，要比英特尔的 Iris Xe Max（DG1）独立显卡和英伟达的 GeForce GT 1030 强一些。<p></p><p>如果再看看价格，可能会感到不可思议。玄人志向这款 4GB 版的 Radeon RX 550 价格为 17600 日元，约合人民币 981 元。要知道当年 AMD Radeon RX 550 的官方建议零售价为 599 元，即便是品牌不同、散热设计有区别、而且是 4GB 版本，也到不了近千元的级别。相信在 2017 年 Radeon RX 550 刚发售的时候不会有人会相信，近五年以后，仍会有超过 50% 的涨幅。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
