<p>IT 之家 9 月 21 日消息，初代 Switch 上市至今已将近六个年头，这款搭载 NVIDIA Tegra X1 定制芯片的掌机已经无法满足日益俱增的游戏机能和用户体验的需求，任天堂显然已经在酝酿新一代的掌机平台。</p><p>任天堂方面此前表示，目前 Switch 表现很好，新机型会在适当时机公布，不过目前 NS 刚度过它生命周期的一半。这表明任天堂确实在开发新机，但恐怕不会在近期上市。</p><p>爆料人 @kopite7kimi 此前透露，Switch Pro（命名未定）搭载的芯片代号 Black Knight，是英伟达专门定制的 Tegra Orin T239 芯片，基于 Ampere GPU 架构（带有 2048 个 CUDA 核心）+ ARM Cortex-A78AE CPU 核心。</p><p>现有最新泄露情报表明新 NS 确实可能会采用英伟达 Tegra 239 芯片。据 Reddit 网友 u / followmeinblue 所述，这是一款具有八核 CPU 的 SoC，他的消息得到了一名英伟达员工的证实，还有任天堂开发者在 Linux 内核邮件列表中发表了相关评论。</p><p>截至此次泄漏，我们现在知道有关下一代 Nintendo Switch 控制台的以下详细信息：</p><p>英伟达 T239 SoC</p><p>8 核 CPU - 可能是 ARM Cortex A78C、A78</p><p>基于安培的 GPU，可能包含一些 Lovelace 功能</p><p>第二代 Nintendo Switch 图形 API 部分包含 DLSS 2.2 特性和光线追踪支持</p><p>IT 之家了解到，英伟达 Tegra X1 发布于 2015 年 3 月，制程为台积电 20nm，TDP 15 瓦。CPU 为 4 个 1.9GHz 的 Cortex-A57 核心和 4 个 1.3GHz 的 Cortex-A53 核心。GPU 基于英伟达自家 PC 显卡的 Maxwell 架构，256 核心，频率 1GHz。</p><p>除了任天堂 Switch，搭载 Tegra X1 的设备还有 Nvidia Shield ( 基于安卓 TV 的多媒体游戏机 ) 和谷歌 Pixel C 安卓平板。不过，任天堂 Switch 内置的 Tegra X1 是阉割版，4 颗 A53 被禁用，频率为 1.02 GHz，低于完全版 Tegra X1 的 1.9GHz。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202209/632a83308e9f090d3008b623_1024.jpg" data-height="353" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202209/632a83308e9f090d3008b623_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
