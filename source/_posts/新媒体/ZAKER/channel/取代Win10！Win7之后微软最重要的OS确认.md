的猜测，下面，就让我们一同看看那些传闻找那个的 Windows 11 吧！<p></p><p><strong>03</strong></p><p><strong>将自带处理器调度方面的改进</strong></p><p>实际上，微软将于 6 月 24 日公布下一版 Windows 的名称，有报道称，下一代桌面操作系统将被称为 "Windows 11"。该公司在内部将其下一代操作系统称为代号 Sun Valley 和 Cobalt，但在 6 月 24 日正式发布时，它将被简单地称为 Windows 11。对于那些已经运行 Windows 10 的人来说，这将是一个免费的可选升级，它将在今年秋天向主流受众发布。</p><p>目前我们对未宣布的 Windows 11 或 Windows" 太阳谷 " 配备了一个系统调度器，主要用来确定应该获得下一个处理器时间片段的线程，控制多任务，提高平台的整体性能。调度器被设计为使用调度优先级来确定下一个应该运行的线程，但目前它没有针对混合 CPU 进行适当的优化。<p></p><p>据报道，即将推出的 Windows 版本将具有 " 大规模调度更新 "，以支持英特尔即将推出的 CPU。</p><p>这是来自 YouTube 播主 Moore's Law is Dead 的说法，他还声称，英特尔期待已久的第 12 代阵容 "Alder Lake 芯片 " 将在 10 月与微软的下一代 Windows 一起推出。</p><p>据报道，英特尔的 Alder Lake 芯片将引入全新的主板插槽，它将同时使用普通的 CPU 内核和高效 / 低功耗的内核，为笔记本电脑提供更好的性能和更长的电池寿命。一些报道称，Alder Lake CPU 将比 Rocket Lake 提供重大的性能升级。</p><p>微软显然正在与英特尔合作，为 "Alder Lake-S" 桌面处理器进行性能优化。据报道，Windows 将获得改进后的调度器，以支持这种混合处理器（为不同任务设计的 CPU 内核）。</p><p><strong>04</strong></p><p><strong>一套崭新的 UI</strong></p><p>在之前已经爆料的 " 太阳谷 " 中，微软宣称要给 Windows 一套崭新的 UI，试图解决现版 Win10 中一个棘手的问题 —— 多风格 UI 混搭。其实有关微软的审美，业界早有定论，概括起来就是两句话 " 生不逢时，老的不愿换！"。在下一版 Windows 的猜测中，我们也很难绕过一个已经被微软判过死刑的系统 —— Win10X。</p><p></p><div class="img_box" 恰恰是先有了功能，再想办法塞到系统里。尽管我们不知道下一版 Windows 是否会对这一问题做一点点修改，但可以确认的是，微软将会在虚拟桌面与多显示器间做一个均衡。</p><p></p><div class="img_box" id="id_imagebox_9" onclick><div class="content_img_div perview_img_div"><img 等形成了一个巨大的生态系统。此外华为的鸿蒙尽管暂时还不够强大，但也开始整合手机、电视、平板、汽车、桌面等诸多场景。从这个角度上看，微软重视平板模式，或许正是它庞大计划的开始而已，毕竟 " 统一大生态 " 才是未来能和其他对手抗衡的最大胜算。</p><p>相对于命途坎坷的 Win10X，Win11 前途显然更明朗一些，如果微软真的将使用 "Windows 11" 这个全新的系统代号，那么这代表着
