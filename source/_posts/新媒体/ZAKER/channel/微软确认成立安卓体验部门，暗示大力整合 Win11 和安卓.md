<p>IT 之家 4 月 17 日消息，在一份新的工作清单中，微软已确认将其 Android 工作整合到一个名为 "Android Microsoft 平台和体验 " 的部门中。此举可能允许 Windows 11 和 Android 之间的进一步集成。</p><p>据 Windows Latest 报道，本周早些时候，微软为其新部门 "Android Microsoft 平台和体验 " 发布了几份工作清单，旨在将 App / 服务 Phone Link 或 Your Phone、SwiftKey、Microsoft Launcher 以及包括 "Surface Duo 体验 " 在内的其他服务归于一个部门。</p><p>工作清单上指出："Android Microsoft 平台和体验团队正在构建未来。我们是世界一流的平台、中间件、应用程序和服务组织，可在各种外形尺寸上为生活带来端到端的体验 —— 通过在他们使用的设备上与 Windows、M365 和 Azure 的强大连接为客户提供支持。"</p><p>目前尚不清楚微软究竟计划如何在提升 Android 体验，但预计未来几个月会对 Phone Link 和 Microsoft Launcher 等应用程序进行重大改进。</p><p>IT 之家了解到，此前据 Windows Central 的一份报告，微软已经进行了重组，以便更好地为 Android 开发软件。</p><p>根据微软首席产品官 ( CPO ) Panos Panay（我们熟悉的 PP 哥）在一份备忘录中宣布的重组计划，Surface Duo OS、SwiftKey、Phone Link、Microsoft Launcher 和其他 Android 项目将成为新的 Android 微软平台和体验 ( AMPX） 部门。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
