内存已经有了，真正上架开卖的却没有几家，金泰克算是动作迅速的一家，已经在电商平台开始预约销售 DDR5 内存条—金泰克速虎 T4 系列，自带散热马甲，机甲科幻外观很炫酷。规格方面为 DDR5-4800，单条容量 16GB，电压 1.1V，时序 40。目前售价 16GB 容量为 999 元，不少网友表示这个价格在预料之中。可见，DDR5 内存真的离我们近了。<p></p><p><strong>DDR5 内存与 DDR4 到底有何不同？</strong></p><p>实际上早在 2016 年，美光就曾发布关于 DDR5 内存的相关信息。从美光当时公布的文件来看，DDR5 内存将从 8GB 容量起步，最高可达单条 32GB，I/O 带宽能达到 3.2-6.4Gbps，同时电压 1.1V，内存带宽将为 DDR4 内存的两倍。此外，美光当时还在芯片论坛上表示 DDR5 内存将从 3200Mhz 起步，主流内存频率可达 6400Mhz。从已经公布的产品情况来看，当时美光的预测偏差不大。目前已经亮相的 DDR5 内存大多是 4800MHz，但厂商也并未表示不会出 3200MHz 系列。同时，据公开数据显示，在主板开启 XMP 超频的话，内存频率可达 8400MHz。</p><p>美光 2016 年发布关于 DDR5 内存的消息</p><p>在官方介绍中，针对 DDR5 和 DDR4 内存的一些硬性指标进行了对比。与上一代产品 DDR4 相比，DDR5 提供 2 个独立子通道，速度 ( 带宽 ) 快约 2 倍，同时额定电压为 1.1v，低于 DDR4 ( 1.2v ) ，耗电降低约 20%。</p><p></p><div class="img_box" id="id_imagebox_4" 内存简单对比<p></p><p>事实上，DDR5 模块与 DDR4 模块之间的一个关键区别就是前者存在子通道。一个标准的 DDR5 模块有两个独立的子通道，每个子通道最多有两个物理软件包级别。每个 DRAM 包都可以在主 / 次拓扑中配置，以实现增加密度的额外逻辑级别。正因为如此，内存的并发性在 DDR4 的基础上提升了一倍，信号能够更完整高效地传递。DDR5 DIMM 新架构提高了并发性，并使系统中可用的内存通道增加了一倍，从而实现内存性能的提升。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" 内核运算能力上，DDR5 内存增加了内置纠错码（ECC），全面实现数据纠错能力，进一步提高数据完整性，同时也缓解了系统错误校正负担并充分利用 DRAM 读写的高效机制。同时，有的 DDR5 品牌在研发时还加入了 SAME-BANK Refresh（同步刷新）功能。此命令允许刷新每个 BG 中的一个 Bank，使所有其他 Banks 保持打开状态以继续正常操作。</p><p></p><div Bandwidth Memory ) 产品群在模具 ( Die ) 状态下，省略包装组装过程，直接与 SoC ( System On Chip ) 连接，不会产生包装副产品 ( 废弃物 ) ，耗电小，属于环保半导体产品。HBM2E 通过设计创新保障耗电竞争力，通过改良以 TSV ( Through Silicon Via ) 技术为代表的工艺改善发热情况，实现了高性能、低耗电、低发热的特性。也就是说，相比 DDR4 内存来讲，不仅性能上有了更大的提升，在低碳环保等方面也做了进一步的突破。</p><p><strong>第 12 代酷睿处理器的黄金搭档，DDR5 普及之路还很长</strong></p><p>未来的应用场景不断革新和进步，对存储技术提出了更严苛的要求，DDR5 就是在这样的需求之下产生的。如今第 12 代酷睿处理器对于 DDR5 内存的支持已经板上钉钉，网上爆料 Z690 芯片组和 DDR5 内存将作为这款新 CPU 的黄金搭档出现。而据公开资料显示，DDR5 内存的强劲性能也将更加适合专业应用领域，比如服务器、云计算、数据中心、高性能计算机等应用领域，此次内存迭代也为行业客户提供了更出色的内存解决方案。</p><p>但是，对于普通消费者而言，DDR5 的普及之路还很长。首先，想要用上 DDR5 内存，你就需要更换 CPU、主板和内存，这将是一笔不小的费用。其中，从上市的预约价格中看到 16GB×2 DDR5 内存套装报价 999 元，比同容量 DDR4 内存高出近 1 倍。其次，相信不少用户关注到的 DDR5 内存性能介绍中都发现了，DDR5 内存普遍延迟较高。虽然厂商已经在不断地进行优化提升，前文中提到的 Trident Z5 系列就突破极限将时序拉紧至 CL36-36-36-76 的超低延迟，从数值上看依然不低。如果你忍受不了这样的延迟反应速度，会被 DDR5 内存逼疯的。另外，对于大多数消费者来说，DDR4 内存的性能已然能够满足大量用户游戏、娱乐影音、专业设计等等场景使用。同时，目前的芯片组平台也并没有达到需要全部更换的程度。所以，大部分人仍然会继续使用 DDR4 内存，同时也会关注 DDR5 内存的发展。</p><p><strong><strong></strong></strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
