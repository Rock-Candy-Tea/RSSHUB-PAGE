<p>IT 之家 9 月 3 日消息 微软名为 PowerToys 的免费系统实用工具套件从 Windows XP 时代复活，并于 2019 年 5 月引入 Windows 10 系统。可以用于高级用户调整和简化 Windows 操作，以提高效率。IT 之家了解到，通过与 Windows 10/Windows 11 shell 搭配使用，同时让用户可以自定义各个工作流。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" 版本，其中有大量的错误修复，包括针对醒目的高 CPU 使用率。今天，微软推出了 PowerToys 0.45.0 版本，重点是优化现有功能，提高稳定性，并进行可访问性改进。此外，它还包含安装程序的更新和新的设置菜单 UI，通过利用新的造型元素提供了全新的体验。风格设计是通过 Fluent UX 完成的，使其与 Windows 11 的 UI 更加一致。<p></p><p>下面是更新内容：</p><p>改变内容</p><p>删除了 ModuleImageLink。</p><p>用 combobox 替换了主题 radiobuttons 设置。</p><p>将项目中的 C# 组件合并为一个测试框架。</p><p>StyleCop 和警告的修复。</p><p>删除了未使用的命名空间别名。</p><p>将 .NET JSON 标准化为 System.Text.Json。</p><p>修正了各种 .xaml 破损的链接和图标。</p><p>更新了 " 编辑布局 " 和 " 创建自定义布局 " 控件的名称属性，使其不包括私有的 Unicode 字符。</p><p>将 bitmask 变量从 size_t 改为 uint64_t，这将使 PowerToys 能够支持每个布局超过 40 个区域。</p><p>调整了遥测数据，以更好地分辨 PowerToys Run 在启动时的启动情况。</p><p>插件 " 直接激活短语 " 设置改名为 " 直接激活字符串 "。</p><p>更改时更新了环境变量。</p><p>一般改进</p><p>设置和 OOBE 窗口更新为流畅设计的用户体验 ! 我们希望你喜欢应用程序菜单具有新的现代视觉，因为我们的产品与即将到来的 Windows 11 界面保持一致。</p><p>在设置中增加了显示版本历史的按钮。</p><p>签署了 PowerToysSetupCustomActions.dll。</p><p>改进了更新可用和更新就绪信息的视觉。</p><p>改进了自动更新的体验。</p><p>将 OOBE 主题颜色与设置主题颜色对齐。</p><p>将 " 以管理员身份重启 " 按钮的标签调整为 " 以管理员身份重启 PowerToys"，以避免含义上的歧义。</p><p>在设置侧边栏增加了彩色图标。</p><p>修正了 OOBE 中 Microsoft Docs 和 PowerToys 发行说明链接无法通过键盘导航的可访问性问题。</p><p>修正了设置标题的对齐方式。</p><p>修正了在浅色模式下，下一个更新文本是可见的。</p><p>更新了 " 了解更多 " 的文字，使之更具描述性。</p><p>更新了更新上的 " 阅读更多 " 文字，使之更有描述性。</p><p>添加了系统托盘中的文档链接。</p><p>修正了安装 PowerToys 时文件使用中的问题引起的错误。</p><p>修正了当 PowerToys 以管理员身份运行时，从 " 开始 " 菜单打开设置的问题。</p><p>Awake</p><p>在翻译错误模板中增加了 Awake 选项。</p><p>调整了不活跃设置的描述，以改进对实用程序被禁用和不活跃的区分。</p><p>颜色选择器</p><p>修正了改变 RGB 值时不更新颜色的 HEX 值的问题。</p><p>修正了屏幕阅读器在激活 " 复制到剪贴板 " 时不通知的无障碍问题。</p><p>修正了用户不能用鼠标悬停信息图标内容的无障碍问题。</p><p>修正了颜色选择器格式顺序无法通过键盘访问的问题。</p><p>修正了屏幕阅读器通知 " 编辑颜色格式 " 按钮的名称不正确，以及根本不通知 " 切换开关 " 按钮的无障碍问题。</p><p>FancyZones</p><p>调整了 " 保存和应用 " 编辑器按钮，以适应本地化的文本大小。</p><p>修正了在高对比度模式下 " 创建新布局 " 按钮的可见性。</p><p>修正了与编辑器用户界面有关的缩放奇怪问题。</p><p>修正了双击 " 编辑布局 " 按钮时编辑器崩溃的问题。</p><p>修正了编辑器在显示区域后立即崩溃的问题。</p><p>修正了通过键盘导航编辑器选项时，在未选择的 Canvas 选项上按回车键会启动 Grid 编辑器的问题。</p><p>修正了 FancyZones 不能恢复控制台应用程序的问题。</p><p>修正了 Canvas 编辑器和网格编辑器的窗口高度。</p><p>修正了因 KERNELBASE.dll 导致的崩溃问题。</p><p>修正了 FancyZone 图标在更高的 DPI 设置下更平滑的问题。</p><p>修正了在区域布局之间转换时的崩溃。</p><p>修正了 FancyZones 在改变布局时不能调整窗口大小的问题。</p><p>调整了布局设置，以便在取消对某一特定布局的更改后重置快捷键。</p><p>文件资源管理器附加组件</p><p>修正了即使没有选择 " 启用标记 "，标记文件仍会被预览的问题。</p><p>图片调整器</p><p>增加了警告：如果文件使用的编码不兼容，带有动画的 GIF 文件可能无法正确调整大小。</p><p>键盘管理器</p><p>改进了 KBM "System.Windows.Controls.Ribbon" 时未找到文件的异常。</p><p>修正了空引用异常的崩溃。</p><p>修复了注册表插件加载崩溃的问题。</p><p>修正了为实用程序设置注册表键值时的未授权访问异常。</p><p>改进了插件属性折叠时警告的可见性。</p><p>增加了插件管理器的搜索。</p><p>修正了 VSCode
