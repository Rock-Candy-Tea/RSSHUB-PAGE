<p>知名性能测试软件 3DMark 前几天推出了一个新的 CPU 专用测试工具 CPU Profile，与已经有的 3DMark 的 DirectX 基准测试中包含的 CPU 测试不一样，3DMark CPU Profile 为 CPU 基准测试引入了一种新方法，可以显示 CPU 性能如何随使用的内核和线程数量而变化。</p><p></p><div class="img_box" 3DMark CPU Profile 基准测试跑了一下，包括 AMD 最新一代 Ryzen 5000 系列的 Ryzen 9 5950X 到 Ryzen 5 5600X，以及 Intel 最新十一代酷睿的 i9-11900K 到 Core i5-11600K，其次还加入了 AMD 上一代的热门处理器 Ryzen 9 3900X、Ryzen 7 3700X、Ryzen 5 3600 和 Intel 上一代的 Core i9-10900K、Core i7-10700K、Core i5-10600K 以及 Core i5-10400F，一共 14 款处理器。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60e43438b15ec05a7f1324c5_1024.jpg" referrerpolicy="no-referrer"></div></div>测试平台如上，内存是 2 条 16GB 的 XPG 龙耀 D50 DDR4-3600 内存组成的 32GB 双通道配置，并且在 BIOS 中打开 XMP 让内存运行在 3600MHz 的频率，其他均采用默认设置。显卡驱动使用的 GeForce Driver 466.77，操作系统则为 Windows 10 64 位 Build 20H2。<p></p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202107/60e43438b15ec05a7f1324c6_1024.jpg" data-height="3870" data-width="765" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/60e43438b15ec05a7f1324c6_1024.jpg" referrerpolicy="no-referrer"></div></div>就结果解读的话，3DMark CPU Profile 的最大线程成绩，代表了 CPU 在使用所有可用的线程时的全部性能潜能。对应的实际使用情况是在游戏之外的极其繁重的多线程工作负载中，例如电影质量的渲染、模拟和科学分析这些应用场景。就结果来看，具有 32 线程的 Ryzen 9 5950X 无疑是这 14 款处理器中表现最好的。<p></p><p>优先采用纯物理核心满足负载</p><p>3DMark CPU Profile 的 16 线程成绩，代表了需要密集计算的任务下处理器的能力，例如数码内容创建和 3D 渲染都得益于多线程，并且 3DMark 表示目前 16 线程的分数与估算实际游戏性能的相关性较小。从结果来看，16 线程成绩仍然是 Ryzen 9 5950X 最高，想必原因是 Ryzen 9 5950X 本身具有 16 个物理核心，可以仅凭 16 个物理核心满足 16 个线程的负载而并不需要用到超线程，因为一个物理核心在超线程技术下得到的两个逻辑核心的计算性能是远远比不上两个物理核心的。</p><p>游戏性能可以参考，但并不完全代表游戏性能</p><p>3DMark CPU Profile 的 8 线程成绩是这其中最应该受到广大游戏玩家的重视，3DMark 表示现代的 DirectX 12 游戏更好地利用了 4 内核以上的多线程性能，CPU 的游戏性能通常与 8 线程的分数更接近且相关联。并且其还表示这个分数还与 3DMark Time Spy 的 CPU 分数高度关联。</p><p>通过测试结果来看，8 线程成绩一定程度上反映了 CPU 实际的游戏表现，比如得分最高的确实是目前地表最强的游戏处理器 Core i9-11900K。但是 AMD 这边却距离实际的游戏表现有些出入，比如只有单 CCD 的锐龙 7 5800X 不存在双 CCD 的锐龙 9 5900X 和 Ryzen 9 5950X 那样的跨 CCD 通信问题，所以内核延迟会更低，实际的游戏性能表现来说锐龙 7 5800X 是比锐龙 9 5900X 和 Ryzen 9 5950X 整体要好一些的。</p><p>当然，3DMark CPU Profile 提到 "CPU 的游戏性能通常与 8 线程的分数更接近且相关联 " 也并不意味着其 8 线程成绩完全就是代表游戏性能的，不然这里就叫 " 游戏性能 " 而不用叫 "8 线程性能了 "，所以这里也是考虑了在 8 线程的渲染、模拟等计算任务下的性能表现，只是说想了解和比较处理器的游戏性能的话，可以参考 8 线程成绩而已。</p><p>3DMark CPU Profile 的 4 线程成绩和 2 线程成绩则一定程度上反映了运行旧游戏的表现，基于 DirectX 9 开发的旧游戏的瓶颈通常是在现代游戏电脑的 CPU 方面而非 GPU 方面，现代的 GPU 对于这些游戏来说其渲染能力通常是过剩的，帧数的瓶颈在于 CPU 一定数量核心的整体性能，而这个数量通常在 2-4 之间。现在流行的电竞游戏，如《刀塔 2》、《英雄联盟》和《反恐精英：全球攻势》的帧率通常与 2 线程或 4 线程的分数相关。当然，与 8 线程成绩一样，4 线程成绩和 2 线程成绩也并不是完全代表游戏表现，不再赘述。</p><p>最后，1 线程分数是处理器表现的基础测试，也就是之前总是会测的单核性能了。不过 3DMark 也表示在游戏和现实中的用户场景中，多线程分数更符合实际性能的参考，意思就是说建议用户选购 CPU 的话还是根据自己的使用场景去关注前面接近的某项多线程的成绩。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div 界面<p></p><p>总的来说，由于各种软件的不一样，对于 CPU 的调度能力也不一样，并且对于现在 CPU 来说，为了在功耗、散热能力与性能发挥之间取得最好的使用体验，现在的多核心处理器一般会采取这样的频率策略——当仅启用少数几个核心 / 线程的时候，这少数几个核心则可以达到尽量高的频率，而软件调用了更多核心 / 线程的时候，则这些核心的频率则相对会有些降低。</p><p>所以尽管用户可能去了解处理器的最大频率、基础频率甚至其他方面，但是对于实际比较处理器在不同应用场景下的体验差别还是很难一概而论。而 3DMark CPU Profile 通过不只是单纯测试全核或者单线程性能来尽量全面和准确的表达处理器在不同场景下的性能表现还是相当方面用户去全面了解一款处理器的，或者在多款处理器之间进行比较也都是很方便的。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
