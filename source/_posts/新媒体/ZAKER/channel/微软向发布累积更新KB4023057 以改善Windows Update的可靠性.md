<p><strong>微软不时地推送累积更新 KB4023057，以帮助提高 Windows Update 的可靠性。到目前为止，这一直是针对 Windows 10 的，但现在微软已经首次为 Windows 11 发布了该更新。</strong>该软件是一个清理补丁，微软之前推送的补丁是为了在不久的将来推出一个更大的更新，它做了一些工作，使更大的服务更新更顺利地推出。</p><p><strong>该补丁执行了以下操作：</strong></p><p>该更新可能会要求你的设备保持更长时间的非睡眠状态，以实现更新的安装。</p><p>如果检测到问题，该更新可能会尝试重置网络设置，并将清理可能阻止更新成功安装的注册表键。</p><p>此更新可能会修复禁用或损坏的 Windows 操作系统组件，这些组件决定了更新对您的 Windows 10 版本的适用性。</p><p>此更新可能会压缩您的用户配置文件目录中的文件，以帮助释放足够的磁盘空间来安装重要更新。</p><p>此更新可能重置 Windows Update 数据库，以修复可能阻止更新成功安装的问题。因此，你可能会看到你的 Windows 更新历史被清除了。</p><p>目前还不清楚 KB4023057 是为什么更大的更新做准备，但微软的遥测可能已经检测到他们的新操作系统的每月更新安装问题。</p><p>KB4023057 是一个强制性的更新，目前没有已知的问题，正在逐步推广到所有支持的 Windows 11 用户。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202111/6187f9ac8e9f0972854e196c_1024.jpg" data-height="393" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202111/6187f9ac8e9f0972854e196c_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
