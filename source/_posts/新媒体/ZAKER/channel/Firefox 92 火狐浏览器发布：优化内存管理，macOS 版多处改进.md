<p>IT 之家 9 月 8 日消息 火狐浏览器近日发布了 Firefox 92 版本，本次更新主要带来了内存管理优化，并且为 macOS 版本带来了功能、外观上的小更新。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202109/613857d28e9f097c0b44adf6_1024.jpg" data-height="421" data-width="775" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202109/613857d28e9f097c0b44adf6_1024.jpg" referrerpolicy="no-referrer"></div></div>新增：<p></p><p>更安全的连接：Firefox 现在可以使用 HTTPS RR 作为 Alt-Svc headers 自动升级到 HTTPS</p><p>多个系统的视频播放已支持全范围的色彩水平</p><p>Mac 用户现在可以从 Firefox 文件菜单中访问 macOS 共享选项</p><p>在 macOS 上启用了对包含 ICC v4 配置文件的图像的支持</p><p>修复：</p><p>如果 Thunderbird 在 Firefox 之后被安装或更新，那么 Firefox 的屏幕阅读器和其他无障碍工具的性能不再严重下降</p><p>macOS VoiceOver 现在可以正确报告使用 aria-expanded 属性标记为 "expanded" 的按钮和链接</p><p>一个标签中的 open alert 不再导致使用同一进程的其他标签出现性能问题</p><p>各种安全问题的修复</p><p>改进：</p><p>macOS 上的书签工具栏菜单现在遵循 Firefox 的视觉风格</p><p>证书错误页面已被重新设计，以获得更好的用户体验</p><p>重构 Firefox 的 JavaScript 内存管理，以提高性能和使用更少的内存</p><p>IT 之家此前报道，Firefox 92 将再次尝试默认支持 AVIF 图像格式，不过官方公告显示，对 AVIF 的支持将推迟到 Firefox 93 版本。</p><p>目前谷歌 Chrome、微软 Win10、网飞已经支持了 AVIF 格式，而 Firefox 火狐浏览器曾在今年初尝试支持了 AVIF，但由于出现了 Bug，使得 Firefox 87 中没有正式上线。</p><p><strong>桌面版下载地址：</strong>点此下载</p><p><strong>桌面国际版下载地址：</strong>点此下载</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
