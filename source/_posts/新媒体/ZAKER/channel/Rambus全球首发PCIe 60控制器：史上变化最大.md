<p>日前，PCI-SIG 组织正式发布了 PCIe 6.0 标准规范，也是该技术诞生以来变化最大的一次，不仅带宽继续提升，底层脚骨和功能特性也发生了翻天覆地的变化。</p><p>今天，Rambus 全球首个发布了完全符合 PCIe 6.0 的控制器，支持全部新特性，主要面向高性能计算、数据中心、人工智能与机器学习、汽车、物联网、国防、航空等高精尖领域。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202201/61f261bcb15ec001dd2c07fe_1024.jpg" data-height="762" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202201/61f261bcb15ec001dd2c07fe_1024.jpg" referrerpolicy="no-referrer"></div></div>该控制器支持 PCIe 6.0 64GT/s 传输数据率，x1 通道即可带来 8GB/s 的单向物理带宽 ( 相当于 PCIe 4.0 x4 ) ，x16 则高达 256GB/s，双向就是 512GB/s。<p></p><p>当然，全新的 PAM4 脉冲调制也是必须有的，编码状态翻番到 4 个，最高可承载 30GHz 频率，而且加入了 FEC ( 前向纠错 ) ，可纠正信号错误、保证数据完整性。</p><p>其他新特性：</p><p>－ 支持固定尺寸 FLIT ( 流量控制单元 ) ，确保高带宽效率</p><p>－ 内部数据通道基于最大链接速度和带宽自动调节 ( 256/512/1024 比特 ) ，优化吞吐</p><p>－ 支持端点、启动端口、双模、切换端口配置</p><p>－ 集成 IDE ( 完整数据加密 ) ，优化性能，监视和保护链接抵御物理攻击</p><p>－ 向下兼容 PCIe 5.0/4.0/3.1/3.0</p><p>Rambus 没有透露何时出货 PCIe 6.0 控制器。</p><p><strong>· END<strong>·</strong></strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
