<p>Windows 7 不再得到微软的支持，因为这个 2009 年的操作系统已经在 2020 年 1 月达到生命的终点。<strong>但当然，这并不一定意味着每个人都已经放弃了 Windows 7，因为许多个人和组织仍然在运行它，主要是因为应用程序的兼容性原因。</strong></p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202107/60e4706b8e9f0975a04c9cea_1024.jpg" data-height="385" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60e4706b8e9f0975a04c9cea_1024.jpg" referrerpolicy="no-referrer"></div></div>当然，这些计算机中的大多数已经把额外的安全措施落实到位，所以在理论上，即使他们正在运行 Windows 7，他们的网络也不会暴露在攻击之下。而愿意支付支持费用的组织仍然通过 ESU 计划获得安全更新，该计划允许他们获得额外支持，只要支付按设备台数计算的费用。<p></p><p>现在，Windows 11 正在开发中，微软希望一些组织将他们的 Windows 7 设备也迁移到新的操作系统，尽管对于老式计算机来说，首先要支持它显然是一个挑战。</p><p>然而，如果某台 Windows 7 电脑确实符合 Windows 11 的系统要求，它将不被允许直接迁移到新的操作系统，而只能通过清洁安装。</p><p>这是因为 Windows 7 电脑没有提供直接升级到 Windows 11 的方法，在这种情况下，获得新操作系统的唯一方法就是从头开始。</p><p>微软对此表示：" 现在可以购买的大多数设备将可以升级到 Windows 11。你将可以选择升级、清洁安装来升级到 Windows 11。对于符合硬件要求的 Windows 7 设备，将需要清洁安装以使用 Windows 11。"</p><p>全新安装的过程显然需要额外的工作，特别是备份数据，所以建议 IT 管理员提前计划整个网络的迁移工作。</p><p>第三方统计显示，Windows 7 仍然是第二大最受欢迎的操作系统，尽管它从 2020 年 1 月起不再获得支持。但随着 Windows 11 的首次亮相，微软表示，Windows 7 用户没有理由不升级到这个操作系统，至少在应用兼容性方面是如此。</p><p>" 微软致力于确保你的应用程序能在我们的软件的最新版本上运行，Windows 11 的构建也考虑到了兼容性。我们的承诺是，在 Windows 7/8.1/10 上运行的应用程序将在 Windows 11 上运行。如果你有 App Assure 服务权限，微软将免费帮助你修复它们，" 该公司解释说。"App Assure 是一项服务，帮助客户解决他们在部署我们最新版本的软件时可能遇到的应用问题。对于拥有 150 个客户机的合格企业客户，App Assure 不需要额外费用。有关资格要求的更多信息，请浏览此页面。访问 App Assure 页面并登录，提交您对专用 App Assure 经理的援助请求。如果您在提交协助请求时遇到任何问题，请联系 ACHELP@microsoft.com，我们的团队成员将协助您 "。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
