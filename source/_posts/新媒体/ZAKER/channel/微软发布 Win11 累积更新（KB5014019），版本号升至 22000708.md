<p>IT 之家 5 月 25 日消息，微软刚发布了一个 Windows 11 累积更新，它将目前 Win11 正式版（21H2）版本号提升到 22000.708，有需要的用户可检查一下 Windows 可选更新并下载。</p><p>从更新内容来看，这与上周四的 Windows Insiders 更新相同，包括 Direct3D 的问题修复，这一 Bug 曾一度导致某些系统上的应用崩溃，甚至蓝屏死机。</p><p>此外，微软还带来了新的 Windows Spotlight 功能，该功能原先被认为是为 Windows 11 22H2 而特地开发的新特性，但微软已经将其向现有 21H2 版本推送。所有用户在安装此更新后即可体验到这一功能。</p><p>桌面背景的 Windows 聚焦类似于此前锁屏界面的 Windows 聚焦，可自动为你的桌面带来新的 Bing 壁纸。如果你不是很喜欢这项功能的话，也可以通过 " 设置 " 关闭，详情可参考 IT 之家此前报道。</p><p>以下是本次更新的其他亮点：</p><p>新功能：我们为儿童账号完善了家庭安全认证系统，尤其是当你请求延长亮屏（游玩）的时间时。</p><p>新功能：Windows 聚焦功能上线，为你的桌面带来了每日常新的背景图。有了这个功能，系统就会自动帮你将新的图片设置为桌面背景。</p><p>我们修复了导致 Input ( TextInputHost.exe ) 应用程序停止工作的问题。</p><p>我们修复了 searchindexer.exe 中影响在 Microsoft Visio 中搜索形状的问题。</p><p>当用户登录到 Azure Active Directory ( AAD ) 时，我们通过断开互联网来阻止他们绕过强制注册。</p><p>我们修复了一个问题，可能运行一个 AnyCPU 应用程序作为一个 32 位进程。</p><p>我们修复了一个阻止 Azure 期望状态配置 ( DSC ) 场景的问题，该场景具有多个部分配置，无法按预期工作。</p><p>我们修复了一个影响 Win32_User 或 Win32_Group WMI 类的远程过程调用 ( RPC ) 的问题。运行 RPC 的域成员联系主域控制器 ( PDC ) 。当多个 rpc 同时发生在多个域成员上时，这可能会使 PDC 不堪重负。</p><p>我们修复了在添加具有单向信任的可信用户、组或计算机时发生的问题。出现 " 所选对象与目标源类型不匹配 " 的错误消息。</p><p>我们修复了在性能监视器工具的性能报告中无法显示应用程序计数器部分的问题。</p><p>我们修复了在改变显示模式后无法保持显示亮度的问题。</p><p>我们修复了一个问题，这个问题可能会影响一些使用 d3d9.dll 的特定显卡的应用程序，并可能导致这些应用程序意外关闭。</p><p>我们修复了一个影响 IE 模式窗口框架的问题。</p><p>我们修复了影响组策略模板的问题。</p><p>我们修复了阻止 Internet 快捷方式更新的问题。</p><p>我们修复了导致某些用户在登录和退出 Windows 时看到黑屏的问题。</p><p>我们修复了导致输入法编辑器 ( IME ) 丢弃字符的问题，如果您在 IME 转换以前的文本时输入字符。</p><p>我们修复了影响桌面复制 API 的问题，该问题会影响显示方向并导致屏幕上出现黑色图像。</p><p>我们修复了当低完整性级别 ( LowIL ) 应用程序打印到空端口时导致打印失败的问题。</p><p>我们修复了在您使用静默加密选项时阻止 BitLocker 加密的问题。</p><p>我们修复了在打开 Windows Defender 应用程序控制 ( WDAC ) 时运行脚本时导致误报的问题。这可能会导致 AppLocker 事件 8029、8028 或 8037 在不应该出现在日志中时出现。</p><p>我们修复了应用多个 WDAC 策略时出现的问题。当策略允许脚本运行时，这样做可能会阻止脚本运行。</p><p>我们修复了影响可信平台模块 ( TPM ) 驱动程序的问题，该问题可能会增加系统的启动时间。</p><p>我们修复了在您结束会话时可能导致远程桌面客户端应用程序停止工作的问题。</p><p>我们修复了影响 Microsoft Defender Application Guard ( MDAG ) 、Microsoft Office 和 Microsoft Edge 鼠标光标的行为和形状方向的问题。当您打开虚拟图形处理单元 ( GPU ) 时会出现此问题。</p><p>我们修复了当您将鼠标悬停在任务栏上的小部件图标上时导致小部件出现在错误显示器上的问题。</p><p>当您单击或点击图标并且任务栏在左侧对齐时，我们为小部件图标添加了动画。</p><p>我们修复了一个问题，该问题会影响居中对齐的任务栏上默认小部件图标的呈现。</p><p>我们修复了当显示器的每英寸点数 ( dpi ) 缩放比例大于 100% 时导致搜索结果中的应用程序图标模糊的问题。</p><p>我们修复了由于缓存管理器中的写入缓冲区计算错误而导致文件复制速度变慢的问题。</p><p>我们修复了在使用 Microsoft OneDrive 时用户注销时可能导致系统停止响应的问题。</p><p>我们修复了一个已知问题，如果您使用 " 控制面板 " 中的 " 备份和还原 " ( Windows 7 ) 应用程序创建恢复光盘（CD 或 DVD），可能无法启动它们。安装 2022 年 1 月 11 日或之后发布的 Windows 更新后会出现此问题。</p><p>您可以在此处下载适用于 x64 或 ARM64 PC 的独立安装程序，或者通过设置 > 更新和安全 > Windows 更新 >> 可选更新找到下载和安装更新的部分。</p><p>您可以在此处查看官方发行说明。</p><p></p><div class="img_box"
