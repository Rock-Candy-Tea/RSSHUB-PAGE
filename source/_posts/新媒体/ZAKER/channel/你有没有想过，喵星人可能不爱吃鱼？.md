~</p><p><strong>起源于非洲的家猫，</strong></p><p><strong>因为能捕鼠而成为人们的好朋友</strong></p><p>想要知道家猫是不是天生爱吃鱼，一个行之有效的办法就是去看看家猫祖先的食谱里是不是有很多鱼。</p><p>还记得在此前关于家猫驯化的科普中，我们曾谈到，在一项发表于知名期刊《科学》上的研究中，科学家们对非洲南部、阿塞拜疆、哈萨克斯坦、蒙古和中东的 979 只野猫和家猫的 DNA 样本进行了测序和分析。</p><p>研究结果表明，野猫可以分为 5 个主要的谱系 ( lineage ) ，分别是欧洲野猫 ( F.s. silvestris ) 、荒漠猫 ( F.s. bieti ) 、亚洲野猫 ( F.s. ornata ) 、南非野猫 ( F.s. cafra ) 以及一个囊括了家猫并分布于中东的野猫和非洲野猫的谱系，<strong>研究人员将这个谱系称为非洲野猫 ( F.s. lybica ) </strong>。</p><p>滑动查看猫星人美照和遗传谱系 ↓</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div Nature）</p><p><strong>这也就意味着，家猫的祖先是非洲野猫这个亚种，只要了解非洲野猫爱吃啥，也就可以推断家猫是不是生来就是 " 鱼控 " 啦。</strong></p><p>根据相关的文献记录，在非洲西部，非洲野猫捕食<strong>老鼠、沙鼠</strong>、野兔、中小型鸟类和蜥蜴；在非洲南部，它们还会攻击羚羊、小鹿和家畜，甚至是孩童；在卡拉哈迪大羚羊国家公园 ( Kgalagadi Transfrontier park，非洲第一座跨国界保护公园 ) ，非洲野猫主要捕食<strong>鼠类</strong>，其次捕食鸟类、小型爬行动物和无脊椎动物。</p><p>从上述的食谱中，我们可以发现，<strong>非洲野猫最钟爱的还是鼠类</strong>，食谱里甚至都看不到鱼的踪影。</p><p>不仅如此，在前文发表在《科学》上的研究中还指出，在非洲野猫这个谱系中，从以色列、阿联酋和沙特阿拉伯的偏远沙漠采集的非洲野猫 DNA 样本和家猫的 DNA 进行对比，在遗传上几乎无法区分，这意味着中东地区或许才是家猫们的 " 快乐老家 "。</p><p>一般而言，科学家们认为家猫可能存在两次驯化的浪潮，而第一次驯化发生在 7,500 年前的新月沃地，那么新月沃地在哪呢？</p><p><strong>它就在——中东！</strong></p><p></p><div class="img_box" id="id_imagebox_4" " 神兽 " 都吃些啥。</p><p>中国关于猫的记载最早见于《诗经》，在诗经中曾提到：" 有熊有罴，有猫有虎 "；而后到了春秋时期，《礼记》中则指出：" 古之君子使之必报之，迎猫为其食田鼠也 "。这同样也说明了猫一开始被人们驯养是为了让其捕捉老鼠。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" data-original="http://zkres2.myzaker.com/202202/620474eb8e9f09768915de41_1024.jpg" data-height="1502" data-width="1080" 世纪左右，中国开始有相猫的著作问世，比如北宋会稽陆佃的《坤雅》中就有这样的记载：" 猫有黄、黑、白、驳数色 …，目如金眼，及上颚多棱者为良 "。</p><p>自唐宋以来的一千多年时光里，古人爱猫的情感在史料和文献中体现的淋漓尽致，南宋洪迈创作的文言志怪集《夷坚志》中就有记载：畜一猫甚大，<strong>极爱之</strong>，常置于旁。猫娇呼，<strong>则取鱼肉和饭以饲</strong>。</p><p>上面这段话透露出了两个重要的信息，<strong>" 极爱之 " 表明了作者对于猫的喜爱，" 取鱼肉和饭以饲 " 则表明当时喂养猫用的是鱼和饭</strong>。不仅如此，南宋诗人胡仲弓在其诗作《睡猫》中写道：" 瓶吕斗粟鼠窃尽，床上狸奴睡不知。无奈家人犹爱护，<strong>买鱼和饭养如儿</strong>"，再次表明当时的人们大都用鱼和饭喂猫。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div <p></p><p><strong>至于为什么是鱼和饭作为猫的猫粮呢？大概率是因为当时鱼比较便宜，大家都能吃得上鱼。</strong></p><p>史料中对宋人生活的记载部分，大都提及一般民众每日的收入就在 100 文左右。举个例子，在《嘉定赤城志》里，临海县令彭仲刚就指出过：" 农工商贩之家，朝得百金，暮必尽用，博奕饮酒，以快一时，一有不继，立见饥冻 "。</p><p>而这样的购买力在宋朝想天天吃上肉几乎是不可能的，要知道在宋朝，猪肉价格并不低，苏东坡找 " 顾屠 " ( 京兆尹顾子敦 ) 开玩笑 " 买肉 " 时掏出三十钱 ( 三十文 ) ，招呼着说 " 且片批四两来 "，这样换算一下，一斤肉差不多得 120 文左右，也就是一天的工资。类似的，《夷坚志 · 丁志》第十七卷也记载了宋高宗在位时江南地区的羊肉价格，每斤涨到 900 文。</p><p><strong>比起猪肉、羊肉，鱼肉的价格就亲民多了</strong>，陆游在《入蜀记》中写道：" 鱼贱如土，一百钱可饱二十口，又皆巨鱼，欲觅小鱼饲猫，不可得 "。当然，鱼肉价格也有贵的时候，比如北宋开封的鱼价就差不多要 100 文，不过这可能与冬天鱼得从外地运输进来有关。</p><p><strong>故而，猫爱吃鱼可能是因为当时的铲屎官们多数都只能喂它们吃鱼有关。</strong>此外，人们还发现，猫会食用鱼的内脏，这样一来，用鱼喂猫就更加顺理成章了，一方面可以帮助人们避免浪费食物，另一方面也可以解决猫主子的吃饭问题。这样一来，久而久之，大家也就认为猫爱吃鱼是猫的天性了 ~</p><p><strong>既然猫并非天生爱鱼，</strong></p><p><strong>还要给猫喂鱼吗？</strong></p><p>事实上，相信现在大多数铲屎官已经不会给自家的喵星人煮鱼吃了，猫粮、罐头，以及冻干成为了铲屎官们新的选择，在上述这些猫粮中，已经添加了猫必须的养分，特别是牛磺酸。</p><p>笔者到某宝随便查询了一个牌子的猫粮，其营养成分中就包含了牛磺酸，牛磺酸对猫来说十分重要，可以帮助它保证眼部和肠胃的健康，但猫无法在体内合成这种物质，只能从外部摄取，所以就得通过吃肉、吃鱼来进行补充。</p><p>当然，如果铲屎官非要给猫喂鱼也不是不行，不过必须注意寄生虫感染的风险，同时还要注意别让猫星人被鱼刺卡着了。</p><p>那么问题来了，亲爱的朋友们，你们平时会喂猫吃鱼吗？</p><div
