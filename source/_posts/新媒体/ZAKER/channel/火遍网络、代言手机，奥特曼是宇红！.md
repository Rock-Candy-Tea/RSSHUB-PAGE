营销？</strong></p><p><strong>1、聚焦 90 后受众，通过联名情怀营销</strong></p><p>不同于如今互联网发展带来的信息圈层极度分化，在以电视为媒介的 20 余年前，奥特曼承载着传统媒体在国内几乎是人尽皆知，开启了一代人的童年偶像启蒙。如今，已经很难在创作出一个能拥有如此庞大粉丝群体的 IP 形象，尤其是对于营销 " 油盐不进 " 的大多数 90 后男性受众而言。</p><p>早在迪迦代言手机之前，奥特曼就推出过各种联名款，均主打情怀营销。2019 年，Undone 与奥特曼合作推出联名款手表，全球限量 50 只；2019 年 2 月，广州开张一家 " 奥特曼 " 奶茶店，开业首日卖出 1000 多杯茶饮，排队能排 40 多分钟；2020 年，优衣库联名奥特曼上架亲子装，目前淘宝店铺的 T 恤月销量高达 6000+。</p><p><strong>2、赋予 IP 新内涵，找到品牌契合点，引爆社交话题</strong></p><p>不同于传统营销环境，立足于互联网环境下，任何产品都应该具备互联网产品属性：流量、粉丝和社交属性。品牌需要具备运营产品粉丝的意识和能力，产品是具备社交属性的，可以被粉丝安利给其他消费者，自身应该也具备融入到社交语境当中的特质。</p><p>而 IP 联名最显著的一个特点就是，品牌借用 IP 资产，给自己的品牌带来创新元素、既有粉丝群体、增加品牌社交营销的切入点。从品牌打造 IP 化产品需求的角度出发，借助 IP 有助于品牌打造更真实、有趣的消费场景；塑造更刺激、兴奋的消费体验；实现更广泛、自主的活动规模化覆盖；建立更丰富、立体的品牌联想，从而赋予品牌更丰富的附加价值。</p><p><strong>3、不只是 IP 形象，更是偶像，适合创意内容输出</strong></p><p>新闻中，当躺在病床上的孩子被问道：是谁给你的勇气啊？他毫不犹豫地回答：奥特曼。这种人文关怀是奥特曼额外的特殊意义，超越了他本身的 IP 商业价值。</p><p>除此之外，最经典，影响最深的还有杰克奥特曼的五大誓言，每个奥迷都会倒背如流。质朴简单，贴近生活，符合青少年特征，都让「奥特曼」的很多故事特别适合进行创意内容的输出。</p><p><strong>奥特曼五大誓言</strong></p><p>1、不能饿着肚子去上课（不要做无准备之战）</p><p>2、好天气要出来晒被子（要善于利用机会）</p><p>3、过马路要当心来往车辆（防范于未然）</p><p>4、不要依靠别人（学会独立自强）</p><p>5、光着脚在地上玩（遵循自然规律）</p><p>在逐渐浮躁，并且格外关注销量和效果的时代，我们看到仍有一些品牌，坚持创意，坚持做优异广告。与商品不同，优异广告把消费这个词彻底解构了，它提供免费消费。它们来自商品，却不再需要商品的支撑，更不再仅仅服务商品，反而让品牌和用户产生更深层次的共鸣。</p><p><strong></strong></p><div
