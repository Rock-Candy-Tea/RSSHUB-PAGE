浏览器不支持新的网页开发模范，要想让网页看起来更好看，只能让程序员为其妥协，写更多的兼容代码。<p></p><p><strong>再看看微软曾经推出的 Windows Phone 手机系统，为难程序员，似乎是一脉相承的传统。</strong></p><p>终于在 2011 年，科技史上最糟糕的 50 种科技产品的评选中，IE 浏览器获得了 11 名的 " 佳绩 "，2006 年的时候，还排在了第 8 名。</p><p>按照黑马这么说，这样一款糟糕的浏览器，怎么会存活这么久，市场份额还一度占到了 90% 以上。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" IE 浏览器，是有很深厚的情怀所在，第一次打开互联网，也是通过点击这小小的蓝色 "IE" 按钮进入。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" IE，并不是现在这样，市场份额飞速上涨，绝对不仅仅是捆绑了 Windows 系统。</strong></p><p>在 90 年代，IE 浏览器是最早有自己的网页设计标准的，其开发的 CSS 模板，直接统一了网页界面的标准，还能显示图像。</p><p>在前段时间，微软还联合了苹果、谷歌和 Mozilla 这几大浏览器制造商，宣布要统一网页适配的问题，让每个网页在不同的浏览器中显示的图片和文字比例是一致的。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" 浏览器能够占到这么高的市场份额，本身也是有一定实力的，否则也会像今天那样被用户和时代所抛弃。</p><p>当时也不只是 IE 浏览器，还有一个不输 IE 浏览器的网景浏览器，在和 IE 的竞争中逐渐落败。</p><p><strong>那么 IE 浏览器是从什么时候开始落寞的？</strong></p><p>竞争从来都是很纯粹的，当你烂的像一坨 " 屎 " 的时候，而对手比你好还比你用心，没落也在所难免。</p><p>这个更加用心的浏览器是 Chrome 浏览器，更快、更安全以及更加开放。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " 浏览器留了一个大杀招：<strong>开源</strong><strong>。</strong><p></p><p>直接把源代码开源，就犹如 Android 一样，基于 Chromium 核心的浏览器蜂拥而至，时至今日，我们用的浏览器基本上也都是基于 Chromium 核心开发的。</p><p>像 360 浏览器、UC 浏览器、QQ 浏览器等常用浏览器，还包括微软自家的 Edge 浏览器，也在 浏览器也注定会慢慢消失在大众视野中。<p></p><p>至于这 " 慢慢 " 需要多久的时间？这让黑马想起 Windows XP 系统，在 2014 年正式停止更新维护后，很长一段时间，Windows XP 的市场份额还一直高居不下。</p><p>因为总有一部分人，拒接接受新的技术，止步于旧的技术，就像曾经的 IE 浏览器那样固步自封。</p><p>IE 浏览器，或许不会彻底退出历史的舞台，即便它已经不安全，但国内的一些网站也还是非 IE 不可。</p><p><strong>一些内网系统的验收标准，也该像浏览器那样与时俱进了。</strong></p><p>撰文：LZH</p><p>编辑：LZH</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
