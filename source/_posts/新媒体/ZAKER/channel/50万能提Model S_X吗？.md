<p>前几天，我和同事在工位上正在高谈阔论，进行思维漫游。</p><p>已经拥有进口 Model 3 的老板，听到我们正在探讨特斯拉的「下一款车」国产后会有多便宜的时候，给我们开玩笑地说，啥时候能买到 50 万的 Model S 啊？</p><p>这 50 万的 Model S/X 那肯定得国产才有希望啊，要这么说，我可就来劲了，今天我就本着求真务实的态度，来看看我们能不能买到 50 万的国产 Model S/X。</p><p><strong>1</strong></p><p><strong>供需</strong></p><p>国产的原动力，那就是需求。假定市场上，供不应求，且这个缺口足够大，利润足够高，那么国产的最大阻碍就将消解。</p><p>我们其实能从以往的销量中，试着预判一下 Model S/X 对应的市场容量和未来的销量趋势。</p><p></p><div 未来的销量完全是沿着之前的经验在做准备，但看起来确实有些保守了。</p><p>虽然规划了 10 万辆 / 年的产能，但特斯拉显然是不可能在今年实现这个产能目标了。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img Model S/X 产出。第二季度过后，Model S 的产线改造以及提产准备任务基本结束，生产了 2340 辆 Model S/X，并交付了 1895 辆。<p></p><p>在 7 月 28 日，特斯拉推迟了 Model S 的交付日期，从长续航版本的交付日期从之前的 9-10 月延迟至明年的 2-3 月，Plaid 版本从 8 月交付推迟到了明年 1-2 月。甚至本来预计明年 1-2 月交付的 Model X 也推迟到 2-3 月。国内新订单，则统一显示交付时间为明年的第一季度。</p><p>甚至在 7 月，弗里蒙特工厂的 Model S 产线还经历了一周左右，无官方解释的停工，猜测大概率是因为「缺芯」问题。假如因为「缺芯」问题而制约了 Model S/X 的产量，那么在解决芯片问题前，特斯拉也不会也没有能力扩大 Model S/X 的生产。</p><p></p><div 的国产，那么有哪些难点需要解决。</p><p><strong>2</strong></p><p><strong>国产难点</strong></p><p><strong>Giga Shanghai</strong></p><p>国产的需求肯定是要落地在特斯拉上海工厂的，那么上海工厂可以新建 Model S/X 的产线吗？</p><p></p><div class="img_box" 产线进行改造，以适配 Model 3 的生产。事实上，弗里蒙特工厂的 Model S/X 产线，只有停产升级的历史。</p><p>上海工厂的 Model 3/Y 产线肩负全球需求重任</p><p>特斯拉上海工厂最初规划产能为 50 万辆 / 年，并且属于 Model 3/Y 产线。弗里蒙特工厂主要供应北美市场，而上海工厂不仅是向国内提供交付，还肩负着向欧洲、亚太等地区交付的重任。以特斯拉 7 月的销量为例，总共售出 32968 辆，而国内销量为 8621 辆，出口占 3/4 为 24347 辆。</p><p>特斯拉面对需求旺盛的欧洲市场，正在进一步提升上海工厂 Model 3/Y 的产能，自然是不会转向 Model S/X 的生产。</p><p>既然改建现有的 Model 3/Y 产线不太现实，那新建一个呢？</p><p></p><div class="img_box" 50 万辆年产能，根据特斯拉 2021Q1 财报显示，现有产能为 45 万辆，主要为中国国内市场以及欧洲市场提供 Model Y/3 车型。并且，目前在上海特斯拉的四个项目：第一期超级工厂第一阶段项目、零部件生产工艺提升项目、超级工厂厂房项目以及新车型先期启动项目，目前还未竣工的项目仅剩下零部件生产工艺提升项目。可以看到，特斯拉没有预留其他产线的建设规划。<p></p><p>在今年 3 月，上海工厂对面地块进行了竞标拍卖，而特斯拉并没有出手拿下。假如将这个地块扩建为工厂，大约可以增加 20 万辆至 30 万辆的年产能，或者可以用来新建 Model S/X 的产线。</p><p>但特斯拉并没有这么做，这说明特斯拉目前并没有扩产计划，当然有人猜测是由于中美经贸摩擦问题。</p><p><strong>优先级</strong></p><p>相比于 Model S/X 国产这个事情，其实马斯克有更多更急迫更重要的事情要去推进。我清楚地看见，在马斯克的小本本上，关于硬件制造的「to do list」已经得满满当当。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy 原定于今年在得州的奥斯汀工厂投产，目前也放弃了今年末交付的承诺，显示的交付时间是明年。在此前间，马斯克会盯紧 Cybertruck 的生产进度。而这个得州奥斯汀工厂，目前仍在建设中，虽然有些区域已经开始调试了。而工厂目前的另一个重点，是在 Model Y 产线上的一体压铸的前后车身工艺测试。<p></p><p>还比如近期传得火热的紧凑型轿车「Model 2」。</p><p>去年年初国产 Model 3 交付时，马斯克表示将会在中国建立车辆设计团队，生产一款专门为中国人设计的车型。在去年 9 月份的电池日上，马斯克还表示将会生产一款售价 2.5 万美元（约合 16 万人民币）的更廉价车型。很多人将这两则信息认定是在描述同一款车型，而目前最新消息，这款车型最快将在年底亮相，并在明年进行量产。得，又是一个明年。</p><p>除此之外，甚至我们可以预见，4680「可乐罐」电池在明年中实现量产后，非常珍贵的产量应该会让马斯克出现精分：究竟分给国产 Model S/X 还是美产 Model S/X 呢？或者是 Cybertruck？还是 Model 3/Y 先上？</p><p><strong>3</strong></p><p><strong>50 万到底能不能买 Model S/X</strong></p><p>和同事的聊天中，有人设想，用国产前后的宝马 5 系列、奔驰 E 级、奥迪 A6 来做参考，以求透视 Model S/X 国产后售价和销量表现，但这其实意义不大。</p><p>首先，这几台车国产的时间是在均在 2000 年初，那时中国的汽车市场的大门正在打开，这些最早的合资车企，吃到了市场快速增长的红利，顶峰可达每月 2 万左右的销量。而 2008 年后，中国车市开始进入存量时代，再叠加现在全球疫情影响，目前这几台车月销在 1 万已经很欣慰了。现在除了存量、疫情这些背景，人们的消费观念也在发生转变，这其中差异太大太多。</p><p></p><div class="img_box" 万呢？</p><p>假如我们沿着成本、利润率为导向的思路走，会发现国产化率决定了这些指标，而这，让我又走进了一个死胡同。</p><p></p><div class="img_box" id="id_imagebox_7" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " Model S/X 国产前期，选用国内的供应商，能提供什么成本、何种性能的零件，特斯拉还需要根据这些东西来定价。这就导致，我们从成本上，难以预判 Model S/X 的价格。</p><p></p><div class="img_box" id="id_imagebox_8" onclick><div class="content_img_div perview_img_div"><img 有太多理由可以在现阶段被否决了。对于这种可做可不做的事情，马斯克一向不急于推进，因为有太多更重要、更确信、更紧急的事情需要解决，而国产 Model S/X 这种选题，怕是也很难出现在近期的特斯拉内部的议题中了。</p><p>所以我们的结论是，特斯拉中短期，至少 2-3 年内，不会将 Model S/X 产线引入国内，也就是不会国产。</p><p>但是我又灵光一现，假如，Model S/X 真的国产了，且售价在 50-80 万元左右，或者随着成本有了更低的下限，那国内新势力会承受怎样的暴击呢？</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
