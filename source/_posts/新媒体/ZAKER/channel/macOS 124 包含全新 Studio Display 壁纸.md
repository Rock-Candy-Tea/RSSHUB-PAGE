<p><strong>今天早些时候，苹果发布了 macOS Monterey 12.4 RC 版，除了新功能外，系统还增加了一张壁纸。这张壁纸是 Studio Display 发布会上，用于展示全新 Studio Display 显示器的壁纸。</strong>在苹果官网 Studio Display 介绍页面也可以看到这张壁纸。</p><p>壁纸将 STUDIO 这个单词的每个字母进行重新设计，有蓝色、紫色和橙色。与 macOS 其他壁纸不同，Studio Display 壁纸只有一种模式，没有深色模式支持。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202205/627e093f8e9f0943ab7ebb97_1024.jpg" data-height="548" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202205/627e093f8e9f0943ab7ebb97_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
