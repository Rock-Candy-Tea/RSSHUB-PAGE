<p>你还没升级 Windows 11 吗，是硬件不支持，还是对系统没兴趣？</p><p>在微软最新的财报电话会议上，公司首席执行官纳德拉（Satya Nadella）盛赞 Windows 11 系统，表示其月度活跃用户已刷新历史记录。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202207/62e2793bb15ec06dc331246c_1024.jpg" data-height="349" data-width="612" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202207/62e2793bb15ec06dc331246c_1024.jpg" referrerpolicy="no-referrer"></div></div>纳德拉表示尽管目前 PC 市场出现回落，但总体销量仍高于疫情前水平。自然在全球对新电脑的强劲需求中，也给微软带来了巨大的利益，人们依赖 PC 的独特的生产技能、实现丰富的交互体验以及保持网络在线。<p></p><p>纳德拉也赞扬了 Microsoft Edge 浏览器。这款基于 Chromium 的浏览器不仅在 Windows 平台上架，而且已经登陆 macOS、Linux、iOS 和 Android 平台。</p><p>同表扬的还有 Azure Virtual Desktop 和 Windows 365，尤其是将本季度的成绩和去年同期进行比较。纳德拉表示，微软正在转变 Windows 的体验方式，并通过 Azure Virtual Desktop 和 Windows 365 进行管理。Azure Virtual Desktop 月活跃用户同比增长了 60%。</p><p>编辑｜熊乐</p><p><strong>· END<strong>·</strong></strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
