<p>对于那些还在坚守 Windows 8.1 的用户来说，微软已经发出警告了，快点换新设备吧。</p><p>据外媒报道称，微软已经发出警告，让那些还在坚守 Windows 8.1 的用户，赶快进行升级设备，换 Windows 10 或者 Windows 11 都可以。</p><p>2013 年 10 月 17 日晚上 7 点，Windows 8.1 在全球范围内上架，新系统的开始屏幕变得更具个性化，背景也变得更加漂亮，当用户从 Windows Store 安装新应用的时候，这些应用将被收入 All Apps（所有应用）中。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202206/62b57daf8e9f0912243b877a_1024.jpg" data-height="347" data-width="614" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202206/62b57daf8e9f0912243b877a_1024.jpg" referrerpolicy="no-referrer"></div></div>另外，用户使用下滑手势可进入所有应用列表，并且支持使用筛选器来找到特定的应用程序组，用户可通过名字、安装日期、常用应用以及分类来搜索应用。<p></p><p>对于这次的警告，微软表示接下来将会对相应的用户弹窗提醒，而他们并不提供 Windows 8.1 免费升级到 Windows 10/Windows 11 的通道，唯一的办法就是换设备。</p><p>按照微软的说法，那些坚持使用 Windows 8.1 的用户应该考虑购买一台支持 Windows 10 或 Windows 11 的设备，之所以没办法升级，是因为 Windows 8.1 系统是 10 年设计的，跟现在的系统架构已经完全不匹配了。</p><p>编辑：熊乐</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
