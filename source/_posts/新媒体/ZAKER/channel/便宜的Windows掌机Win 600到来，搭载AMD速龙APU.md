<p>对于有买过开源掌机的玩家，应该都会认识 Anbernic 这个品牌，他们此前推出过基于 DingooOS 和 Linux 系统的模拟游戏掌机，现在也趁着热度推出了自家首款 Windows 掌机 Win 600，据称会定位在千元价位，大大降低了 Windows 掌机的入手门槛。</p><p>由于此前是做过掌机的，所以 Anbernic 在体形和握持手感的设计上还是有些经验的，Win 600 在保持与其它 Windows 掌机相同的 7 英寸屏幕，机身尺寸却要小很多，与一台任天堂 Switch 接近，而且重量也控制在 500g 以下，这对于握在手上玩还是比较适合的，至于按键布局就采用下置双摇杆的形式，但可能出于定位，它就没有配备线性扳机按键了。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202206/62bc26fc8e9f092a276d3537_1024.jpg" data-height="608" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202206/62bc26fc8e9f092a276d3537_1024.jpg" referrerpolicy="no-referrer"></div></div>图片来自 Retro Game Corp<p></p><p>当然 Win 600 能做相对小巧，也与它内置硬件不太强有很大关系，它搭载了一颗 AMD 速龙银牌 3050e 的移动 CPU，这是一颗双核四线程的 APU，最高频率只有 2.8GHz，已经调整到 15W TDP，用于上上网、处理文字工作还是没问题的，而 GPU 部分为 Vega 3，也就是只有 3 个 CU，最大频率为 1000MHz。</p><p>只是 Anbernic 给它搭配了一条单通道 DDR4-2400 的内存，不过从 ETA Prime 的拆解来看，Win 600 居然采用了 SO-DIMM 的内存插槽，可以自行更换，在换了上更高频率的 DDR4-3600 内存后，GPU 的跑分成绩有大幅提升，不过这颗 GPU 也就跑跑一些老的 3D 游戏，以及用于模拟器使用。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202206/62bc26fc8e9f092a276d3538_1024.jpg" data-height="608" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202206/62bc26fc8e9f092a276d3538_1024.jpg" referrerpolicy="no-referrer"></div></div>图片来自 ETA Prime<p></p><p>其它硬件就还包括 256GB M.2 SSD，同样是可以自行更换的，而内置的电池只有 39Whr，基本是要插着电玩游戏，机身带有一个 USB Type-A，以及一个全功能 USB-C，支持外接显示输出等等。</p><p>目前 Anbernic 还没有公布正式售价，但有消息表示起步配置（AMD 3020e）的价格会在 300 美元左右，这相比目前 Windows 掌机普遍要卖到 5-6000 元还是要便宜很多，预计这部 Win 600 会在 7 月份上市开卖。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
