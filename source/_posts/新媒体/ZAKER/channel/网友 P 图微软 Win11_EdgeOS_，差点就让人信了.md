<p>IT 之家 7 月 12 日消息，Twitter 用户 FireCube 经常会泄露 Windows 即将推出的功能或发布一些有趣的内容，虽然很多时候是真的，但今天这张 "EdgeOS"Beta 截图欺骗了许多人。</p><p>FireCube 发布了一张基于 Windows 11 的 "EdgeOS"Beta 的假想截图。这是故意伪造的，微软实际上并没有打造一款谷歌 Chrome OS 的竞争对手系统。</p><p>在 Twitter 帖子中，FireCube 为其添加了一些描述，称 " 泄漏 微软计划在切换到新的基于 Web 技术驱动的 shell 后重新塑造 Windows 11 品牌 "，并附有以下图片：</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202207/62cd26d08e9f09238f0ac860_1024.jpg" data-height="1920" data-width="922" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202207/62cd26d08e9f09238f0ac860_1024.jpg" referrerpolicy="no-referrer"></div></div>屏幕截图是编辑过的 winver 窗口，但顶栏显示的还是 " 关于 Windows"，而不是 " 关于 EdgeOS"，而且 Build 版本号甚至没有正确排列或间隔。足以证明这是假冒的。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202207/62cd26d08e9f09238f0ac861_1024.jpg" data-height="517" data-width="531" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202207/62cd26d08e9f09238f0ac861_1024.jpg" referrerpolicy="no-referrer"></div></div>▲ 微软 Windows 10 版本的关于窗口<p></p><p>虽然这只是为了开个玩笑，但微软最近在 Edge 浏览器功能集相关的行动方面确实让 "EdgeOS" 变得非常可信。在过去的几个月里，微软一直在向 Edge 浏览器添加大量额外的、甚至一些不必要的功能，使其本身成为一款 mini OS。</p><p>到目前为止，微软 Edge 浏览器已添加了完整的侧边栏，其中包含计算器、单位转换器以及照片应用，甚至还有指向整个 Offcie 套件和 Outlook 日历的链接。而且 Edge 现在也是一款游戏浏览器，微软称 Edge 是 " 最适合游戏玩家的浏览器 "。微软以后也可能会添加更多此类功能。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
