<p>IT 之家 4 月 16 日消息，据 Neowin 报道，网络安全公司 Trustwave 的安全团队 SpiderLabs 已警告 Windows 用户注意一项名为 Vidar 的新恶意软件，该软件将自己伪装成 Microsoft 支持或帮助文件。因此，毫无戒心的用户很容易成为受害者，而 Vidar 是一种恶意软件，可以窃取被利用受害者的信息。</p><p>微软编译的 HTML 帮助 ( CHM ) 文件虽然现在有点不常见，但用于提供各种帮助文档等。这种恶意的 Vidar CHM 恶意软件通过电子邮件以 ISO 的容器形式分发。该 ISO 伪装成 "request.doc" 文件。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202204/625a63d6b15ec062bd60fbfc_1024.jpg" data-height="245" data-width="624" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202204/625a63d6b15ec062bd60fbfc_1024.jpg" referrerpolicy="no-referrer"></div></div>在 request.doc ISO 文件中，有几个恶意文件，包括名为 "pss10r.chm" 的 Microsoft Compiled HTML Help ( CHM ) 和名为 "app.exe" 的可执行文件。一旦用户被诱骗提取这些文件，用户系统就会受到威胁。前者 "pss10r.chm"，实际上通常是一个合法文件，但随附的 exe 文件却是 Vidar。<p></p><p>下面是合法的 "pss10r.chm" 与在 Vidar 中使用的恶意软件的比较图像：</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202204/625a63d6b15ec062bd60fbfd_1024.jpg" data-height="397" data-width="624" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202204/625a63d6b15ec062bd60fbfd_1024.jpg" referrerpolicy="no-referrer"></div></div>恶意 CHM 的目的是运行另一个文件，即包含 Vidar 恶意软件的 app.exe，以成功传递有效负载。<p></p><p>如上所述，Vidar 是一种窃取型恶意软件，可以从浏览器等其他地方窃取信息和数据。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
