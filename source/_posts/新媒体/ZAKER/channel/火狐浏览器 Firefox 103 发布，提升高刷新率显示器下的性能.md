<p>IT 之家 7 月 27 日消息，Mozilla 近日发布了 Firefox 103 火狐浏览器，带来了多项新功能，包括改进了刷新率为 120 Hz 及以上显示器的性能，并突出显示了 PDF 表单上的必填字段，画中画字幕也得到了扩展。</p><p>更新说明如下：</p><p>通过切换到 modern lock API，在 CPU 高负载期间提高了 macOS 下的响应能力。</p><p>PDF 表单中的必填字段将突出显示。</p><p>提高了高刷新率显示器 ( 120Hz+ ) 的性能。</p><p>现在可以直接从画中画窗口更改字幕字体大小。此外，画中画字幕现在可在 Funimation、Dailymotion、Tubi、Hotstar 和 SonyLIV 获得。</p><p>可以使用 Tab、Shift+Tab 和箭头键访问选项卡工具栏中的按钮。</p><p>Windows 下的 " 使文本变大 " 可访问性设置现在影响所有 UI 和内容页面，而不仅适用于系统字体大小。</p><p>支持在 Windows 10 和 Windows 11 上安装期间固定到 Windows 任务栏。</p><p>在从表单控件复制文本时，会保留不间断空格，从而防止自动换行。</p><p>通过 Linux 上的 DMA-Buf 修复了英伟达二进制驱动程序上的 WebGL 性能问题。</p><p>修复了 Firefox 启动可能因处理 Web 内容本地存储而显著减慢的问题，该问题对拥有盘片硬盘驱动器和大量本地存储的用户影响最大。</p><p>删除了允许在证书中使用 SHA-1 签名的配置选项：证书中的 SHA-1 签名早已确定不够安全，现在改为不受支持。</p><p>多项安全修复。</p><p>Firefox 103 版本在 Windows 和 macOS 中会自动更新，IT 之家小伙伴也可以在右上角菜单中的帮助 > 关于 Firefox 中查找更新。</p><p>Firefox 103 安卓版更新内容如下：</p><p>从 Firefox for Android 的搜索栏中访问标签和书签。点击搜索栏中的搜索引擎图标，打开一个下拉菜单。选择书签或标签页。</p><p>当切换到一个不同的搜索项目或引擎时。</p><p>搜索中的任何现有文本将被保留。</p><p>自动完成的结果将被更新。</p><p>搜索结果将更新。</p><p>图标或搜索修改器图标将被更新。</p><p>修复了小的页面加载性能退步。</p><p>修复了证书错误安全问题 CVE-2022-34469。</p><p>修复了 URL 安全问题。</p><p>修复了 Twitter 视频倒着播放的问题。</p><p>修复了打字时网页内容被涂成黑色。</p><p>修复了图形崩溃问题。</p><p>修复了部分网站在增强型跟踪保护的严格模式下不工作的问题。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
