<p>Intel 的 12 代酷睿此前我们的关注点都在 CPU 本身，毕竟首发的都是带 K 的处理器，基本上都是配独显使用的，核显基本就是拿来亮机或者拿去做辅助的硬件加速功能，Intel 在宣传的时候基本都没提及到关于核显的部分， 在发布前的媒体会议上也就简单说过是 Alder Lake-S 所整合的核显与上一代的 Rocket Lake-S 一样是基于 Xe-LP 架构的。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" 提升至 7680*4320 @60Hz，细节改动还是有的。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 核显频率最高的是 Core i9-12900K，但那它来测试核显就太没实际意义了，所以我们拿了主流级的 Core i5-12600K 来测， 会分别测试它搭配 DDR5 和 DDR4 内存时的性能，两种内存各测试两套频率，分别是 DDR5-5600、DDR5-4800、DDR4-3600、DDR4-2666，DDR5 内存的时序是 CL40，而 DDR4 内存的时序是 C16。<p></p><p>而 UHD 730 则是使用 Core i5-12400 来测试，它就只测用 DDR4-3600 时的性能了。用来对比的包括上一代的 Core i5-11600K 以及对手的 Ryzen 5 5600G，两者都是搭 DDR4-3600 内存来跑的。</p><p></p><div class="img_box" id="id_imagebox_6" DDR5-4800 内存时，核显的性能差距还是比较明显的，但 DDR5-5600 内存带来的性能提升就少得可以忽略。在同样搭配 DDR4-3600 内存时，UHD 770 的性能较 UHD 750 大概有 5% 的性能提升，增幅不大，UHD 730 性能则比 UHD 750 低 15% 左右，当然了论核显性能还是 AMD 强，Ryzen 5 5600G 的核显性能比 UHD 770 强不是一点半点。<p></p><p></p><div class="img_box" id="id_imagebox_7" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_7" 分辨率下进行的，具体画质设置请看上表，各游戏都有所不同，其实这没什么好比的，和上面的 3DMark 测试差不多，UHD 770 在搭配 DDR5-4800 时较 DDR4-3600 有较明显的提升，但 DDR5-5600 带来的增幅很小。而在同样使用 DDR4-3600 时，Core i5-12600K 核显表现都要优于 Core i5-11600K 的核显，这里自然会有 CPU 性能差异带来的影响，但实际上看游戏的帧数的话，你会发现两者其实没啥本质上差别，基本上网络游戏都可以较流畅的运行，开中或者开高画质都可以，但 3A 大作的话，就只能在低画质下玩一些老游戏了， 至于那个 UHD 730，就真的只能来玩下《英雄联盟》和《CS:GO》了，其他的都不怎么行。<p></p><p>当然和 Ryzen 5 5600G 一比的话，Intel 的核显性能真不行。</p><p>由于 Alder Lake 的核显把视频编码器数量翻倍，为此我们也来测测它视频导出的效率，采用 DDR4-3600 内存，对比对象换成了 CPU 性能相近的 Core i7-11700K。</p><p></p><div class="img_box" id="id_imagebox_8" onclick><div class="content_img_div perview_img_div"><img 1080p 的 H.264 格式。从导出时间来看用核显做硬件加速时 Core i5-12600K+UHD770 的组合是要比 Core i7-11700K+UHD750 快了将近一分钟的，但这不太清楚是视频编码器的影响还是 CPU 本身的影响，但比较尴尬的是，使用 Core i5-12600K 纯软件导出是速度比用核显加速快多了，不过用核显加速的话 CPU 占用了并不高，你可以把 Pr 扔一边去做别的东西，纯软件导出速度是块，但话 CPU 占用是满的。<p></p><p>这 Alder Lake 带的 UHD 770 核显较上代确实有一定的性能提升，不过这性能提升都是通过提升频率带来的，如果配 DDR5 内存的话由于内存带宽增加，性能提升幅度会更大，但现在有钱买 DDR5 内存的人肯定不用烦独显的问题。至于那个 UHD 730，性能确实不怎么样，但玩下《英雄联盟》和《CS:GO》还是可以的。</p><p>视频导出效率确实比上代有所提升，毕竟编码器数量从 1 个增加到 2 个，不过核显的性能还是有限，和纯 CPU 导出性能差距太大，说到底还是需要一张独显才是最高效的。</p><p>这 12 代的核显性能也就够玩下需求不高的网络游戏，或者一些比较老的 3A 大作，基本上这核显还是给你亮机用的，没有什么质的改变，真想要高性能核显的话，还是去找 AMD 吧。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
