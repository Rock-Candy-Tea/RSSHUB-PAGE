<p>Windows 11 发布后，民间高手们也探索出一些玩法，比如在一加 6T/ 小米 8 等骁龙 845 手机上刷入、在树莓派上运行等。虽然乍一听，感觉骁龙 845 手机跑 Windows 11 不过是炫技玩玩，难堪大用，但折腾一番后发现，事情被低估了。</p><p><strong>01</strong></p><p><strong>骁龙 845 手机成功刷入 Windows 11</strong></p><p>最早做到一加 6T 运行 Windows 11 的是油管 UP 主 edi194，他发现设备可以跑起一些大型游戏，甚至还制作表格列举了 50 多款的测试结果。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202107/60eac0958e9f090870672329_1024.jpg" data-height="407" data-width="700" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202107/60eac0958e9f090870672329_1024.jpg" referrerpolicy="no-referrer"></div></div>可玩性较高的有《我的世界：基岩版》（1080P 分辨、60 帧）、《求生之路 2》（1080P 40 帧）、《极品飞车 9：最高通缉》（1080P、40 帧）、《极品飞车 17：最高通缉》（1080P、50FPS）、《传送门 2》（1080P、60 帧）、《GTA：圣安地列斯》等。<p></p><p>B 站 Up 主极客湾也做了比较详细的测试，发现《古墓丽影 9》《CS:Go》体验都尚可，甚至可以连接罗技 G25 外设玩《欧洲卡车模拟 2》等。</p><p><strong>02</strong></p><p><strong>那些刷入 Windows 11 的手机</strong></p><p>给手刷入 Windows 11 系统，早期是 Lumia 950 XL 成功移植 Windows 11，关键是移植成功了还支持通话、发短信功能，一下点燃了人们的玩机兴趣。</p><p>而来自中国武汉的大神 Xilin Wu 也成功将 Windows 11 系统移植到运行高通骁龙 845 的小米 8、一加 6T 手机上。不过该操作系统的大部分内容是无法使用，例如开发者称目前并不支持通话。</p><p>为了在 Android 手机上运行 Windows 11 系统，开发者已经创建了自己的工具 / 驱动程序，并为骁龙 845 启用了自定义的 UEFI 环境。这是因为高通公司的 xbl ( UEFI 固件 ) 和 abl ( 加载 Linux 内核所需 ) 在零售设备上签名，它们不能被修改。因此，不可能使用库存引导程序启动 Windows。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202107/60eac0958e9f09087067232a_1024.jpg" data-height="536" data-width="782" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202107/60eac0958e9f09087067232a_1024.jpg" referrerpolicy="no-referrer"></div></div>需要注意的是，微软并没有计划将 Windows 11 引入到移动端。而正是这些充满热情、技术娴熟的达人，通过创建定制工具和驱动程序才成功在手机上运行这款桌面系统。<p></p><p>目前，Windows 11 只能在一加 6T、小米 8 和一些骁龙原型机上运行。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
