<p>中国人有多爱喝酒？在中国，凡是有宴席的地方，必定有酒。据有关数据统计，中国人一年喝掉的酒换算起来约等于一个西湖。足以证明，喝酒对于中国人来说，是一项重要的社交活动。</p><p>在我国，啤酒是酒精行业的重要成员。它于二十世纪初传入中国，属外来酒种。啤酒以大米、小麦、水、酒花和酵母等为主要原料，经酵母发酵作用可酿制而成干啤酒、全麦芽啤酒、头道麦汁啤酒、黑啤酒等。经过包装后通过商超、便利店、餐饮店等渠道送至消费者手中。</p><p>数据显示，2021 年 1-4 月，中国规模以上啤酒企业累计产量 1140.9 万千升，同比增长 33.6%，相比 2019 年同期基本持平，消费恢复明显。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " referrerpolicy="no-referrer"></div></div><strong>啤酒还是精酿的好</strong><p></p><p>在消费升级的带动下，人们的味蕾变得更加挑剔，大众早已不满足于传统的工业水啤的口感，啤酒消费也正向高端化、品质化迈进，其中，精酿啤酒大行其道。</p><p>精酿啤酒是指用更传统的工艺、更好的原材料酿造而成的高浓度的优质啤酒。相比于寡淡的工业啤酒，精酿啤酒的味道往往更加丰富独特，在酿造过程中最大限度的保留了原料自身的风味，可以品尝到浓浓的麦芽香气。精酿啤酒的主要材料往往都比较简单，麦芽、酒花、酵母和水。在原料产地的选择上，精酿乐于使用当地种植生产的最好最新鲜的原料。</p><p>除此之外，水果、蜂蜜、鲜花皆可入味其中，造就了个性多元化、个性十足的精酿啤酒，很受年轻人们的欢迎。</p><p></p><div 图源自视觉中国<p></p><p>不同于过去人们一味追求进口的精酿啤酒，如今国产精酿也在强势崛起，进口精酿一杯至少要六、七十元，国产精酿只需 30 到 50 元，由于性价比更高、物流时间更短，国产啤酒往往能呈现出更新鲜、更优良的品质，逐渐成为国内消费者们的首选。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202105/60a8dbc68e9f0966500c3eb9_1024.jpg" referrerpolicy="no-referrer"></div></div><strong>巨头纷纷布局高端啤酒</strong><p></p><p>目前，啤酒市场整体格局稳定，集中度高。前五名市占率 ( CR5 ) 约为 92% 左右。2020 年销量市占率从高到低依次为华润 ( 31.9% ) 、青岛 ( 22.9% ) 、百威英博 ( 19.5% ) 、燕京 ( 10.3% ) 和嘉士伯 ( 7.4% ) 。</p><p>未来三年，啤酒市场规模将超过 5000 亿元，高端市场占有比例超 40%，即 2000 亿的市场。</p><p>今年以来高端产品销量明显提升，低端产品逐渐淘汰，与过去追求销量不同。国产啤酒巨头加码高端市场动作频繁。</p><p>青岛啤酒于 2020 年 7 月相继推出了 " 百年之旅、琥珀拉格、奥古特、鸿运当头、经典 1903、桶装原浆啤酒、皮尔森、青岛啤酒 IPA" 等高端产品。</p><p>在青岛啤酒 2020 年的年报中显示，2020 年全年实现销量 782 万千升，其中，百年之旅、琥珀拉格、奥古特、鸿运当头等高端产品共实现销量 179.2 万千升。测算得出，青岛啤酒的高端占比达 22.9%，高端产品组合在提升业绩中的战略地位可见一斑。</p><p>与此同时，市占第一的华润啤酒，在牵手喜力后，于 2020 年取得了次高档及以上啤酒销量增长 11.1%，销量达 146 万千升的业绩。华润啤酒也打造了 "4+4" 的产品矩阵，即雪花旗下的勇闯天涯 SuperX、马尔斯绿、匠心营造、脸谱四大高端品牌，加喜力旗下的喜力星银、虎牌、红爵、苏尔四大高端品牌，纵深推进高端产品组合。</p><p>类似的，嘉士伯与重庆啤酒完成资产重组后，后者直接从一家区域型啤酒企业成长为全国性啤酒企业，拥有 " 本地强势品牌 + 国际高端品牌 " 的品牌组合，即嘉士伯、乐堡、1664 等国际品牌，还有网红与市场红的双料王——乌苏及重庆、山城等本地强势品牌。重庆啤酒甚至提出，要通过一系列市场活动，持续推进产品高端化，推动销售模式变革。</p><p>燕京亦调整并坚持 1+3 品牌发展战略，即以 " 燕京 " 为主品牌，同时培育漓泉、惠泉、雪鹿 3 个区域优势品牌。2021 年第一季度，燕京高端产品 U8 单品销量同比增长 560%。</p><p>更甚者，华南知名啤酒品牌——珠江啤酒也实施了 3+N" 产品战略，即 "1 ( 雪堡 ) +1 ( 纯生 ) +1 ( 珠江 ) "+"N ( 特色 ) "。2020 年，珠江啤酒完成啤酒销量 119.94 万吨，同比下降 4.65%; 营业收入 42.49 亿元，同比增长 0.13%。</p><p>珠江啤酒在 2020 年年报中指出，公司高端精酿实现销量同比增长 34%，其中，纯生啤酒销量占比 41%，罐类产品销量占比 31%。97 纯生高端核心产品，完成销量 7.72 万吨，同比增长 176.77%。</p><p>可见，中国精酿啤酒市场有相当大的发展潜力，同时随着国内消费不断升级，啤酒高端化正成趋势。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div referrerpolicy="no-referrer"></div></div><strong>国产啤酒高端化任重道远</strong><p></p><p>放眼全球，中国是全世界最大的啤酒市场。有业内分析认为，预计 2023 年中国啤酒消费量将达 511.5 亿升，市场规模将达 1029 亿美元。在这样的体量下，需要国产啤酒有更优质的产品输出，高端化是其思路之一。</p><p></p><div 图源自视觉中国<p></p><p>不可否认的是，目前国产啤酒高端化还存在较大困难。主要是与当前的 " 商务消费场景 " 不搭。高端啤酒的价格严重低于白酒甚至葡萄酒，很难融入商务消费场景。茅台啤酒早已沉寂，茅台葡萄酒愈战愈勇，这也从另一个角度佐证高端酒的基因暂时不属于啤酒品类。</p><p>但从发展前景来看，啤酒产业一直是一个大产业，大消费市场一直在升级，市场供给者一直在变化。随着人们生活水平的提高，人们已经不再满足基本的吃饱，转向更高、更美、更精细、更多样化等消费需求，类似于更健康、更休闲、更美丽等消费升级出现。高端啤酒是啤酒行业分化出的一个细分垂直行业。啤酒业务扩展赛道非常广阔，随着市场竞争的加剧，行业会分化出更多的新的细分行业。业内专家认为，只要没有不可抗因素的阻断，新的细分行业又会产生新的行业发展红利。</p><p>当前，国产啤酒如何走、怎样形成高端化转型，是今后很长一段时间内企业发展所需要研究的课题。</p><p>财经早餐综合自央视财经、中国产经新闻、Donews
