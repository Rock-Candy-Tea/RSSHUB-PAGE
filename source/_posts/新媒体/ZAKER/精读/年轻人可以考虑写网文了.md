年度财报，对这些质疑做出了有力的回击。<p></p><p>财报显示，阅文平台的作家、作品数量在 2020 年都获得了增长。截至 2020 年底，阅文已积累了超过 900 万位作家，新加入作家超 90 万，作品总数达到 1,390 万部，全年平台新增字数约 460 亿。</p><p>不仅如此，包括天下霸唱、酒徒、沐轶、徐公子胜治在内的多位大神级作家，在阅文新合同发布后，都宣布了回归阅文。阅文全年总收入 85.3 亿元，其中下半年收入达到 52.7 亿元，环比上半年增长 61.5%，网文市场的天花板一次次被推高，其商业价值也变得越来越多元。</p><p>阅文白金作家希行在接受采访时表示，阅文在作家福利更新后不仅在分成上有所提高，影视版权开发也带来了原作的曝光度，" 我的小说卖得很顺利，影视版权开发与我的作品曝光度同时都有了增加，双向提升了知名度。"</p><p><strong>大 IP 的转化，让网文与影视化的互动模式逐渐成熟。</strong>无论网文平台还是作者，都有望通过将网文作品改编成有声、影视剧、动漫、游戏等获得新收入，这种商业化前景的不断扩大，对网文作者是明显的行业利好。</p><p>这种利好在不断升级。6 月 3 日，阅文在上海举办了 2021 阅文年度发布会，这是 " 新文创周 " 的首场活动，也是阅文新管理层履新以来，在 " 新文创 " 战略指导下，经历一整年调整、实践、升级后的首次对外发布会。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" referrerpolicy="no-referrer"></div></div>程武<p></p><p>市场风云变幻，阅文也在不断求新求变。在当天的发布会上，阅文集团首席执行官、腾讯集团副总裁兼腾讯影业首席执行官程武，提出了战略升级 " 大阅文 " ——基于腾讯新文创生态，以网络文学为基石，以 IP 开发为驱动力，开放性地与全行业合作伙伴共建 IP 生态业务矩阵。程武表示，<strong>对于 " 大阅文 " 来说，未来应不再盲目、线性地进行 IP 改编，而关注如何把人才、资源、不同内容业态有机结合起来，推动 IP 生态链进入实践环节。</strong></p><p>在 " 大阅文 " 战略下，IP 的创作和开发思路发生了改变，更注重提升 IP" 整体生命周期价值 "，并进一步激发 IP 潜在价值，从而拓宽阅文整体发展空间。会上阅文发布的全新愿景：" 为创作者打造最有价值的 IP 生态链，成为全球顶尖的文化产业集团。" 这也是阅文第一次将包括漫画家、编剧、导演等 IP 全产业链上的所有创作者集纳到战略愿景中。</p><p>这意味着，阅文开始驶向新阶段。</p><p>近几年来，网文逐渐占据 IP 改编的 C 位，经由网文 IP 改编的影视化作品在起步阶段就更容易获得外界关注和热度。</p><p>《新华 · 文化产业 IP 指数报告（2021）》评选出 TOP50 的 IP 中，网文原生 IP 占比最高，达 40%，比 2017-2018 大幅提升。《网络文学 IP 影视剧改编潜力评估报告》显示，2018 年、2019 年共 309 部热播影视剧中，由网络文学 IP 改编的作品有 65 部，占比约 21%；而在热播度最高的 100 部影视剧中，这一占比高至 42%。</p><p><strong>IP 一体化运营产业链一旦打通，可以创造出极大价值。</strong>发布会上提到，2019 年好莱坞最赚钱的 10 部电影中，迪士尼占了 7 部。迪士尼通过动画、影视、衍生品、文旅等多种 IP 生态体系，使不同内容彼此影响、放大，形成更强的势能。IP 生态链成为了作品成功率提升的重要保障，即使个别作品口碑不佳，也不会动摇整个 IP 产业链的根基。</p><p></p><div 一位网文行业资深观察者表示，在他看来如何让网文作品的商业价值得到更大的开发，如何让作者的粉丝运营能够基于一个 IP，在诸多形态中实现粉丝经济的维护 …… 这无疑对网文平台的综合运营能力提出了更高的要求。<p></p><p>而阅文在新管理层上任后，越来越多地接入腾讯的资源，推动腾讯已有的业务的布局与阅文联动，并随着越来越多的合作，这种联动也变得驾轻就熟。</p><p></p><div "，在《庆余年》之后，更加深入、更加系统的切入影视领域。《赘婿》制片人刘闻洋在接受媒体采访时提到，从腾讯影业评估 IP 开发立项，到阅文提供 IP 版权并输出内容理解力，再到新丽组建编剧团队改编创作、集合制片团队拍摄，三家发挥了重要的协同作用，使得资源得到了最大化利用。在这种协同作用下，《赘婿》从项目开机到上线，只花了 8 个月，而行业的一般耗时是一到两年。<p></p><p>在业内人士看来，从《庆余年》到《赘婿》，再到《斗罗大陆》《锦心似玉》这些内容其实都有一个共同点：<strong>优质文学 IP 通过影视化精良制作呈现，再借助影视 " 放大器 " 作用覆盖更广泛用户，进而推动 IP" 破圈 "。</strong></p><p>发布会上，阅文宣布了包括有声、出版、动漫、影视、游戏、IP 商业化和线下消费等三级开发体系和未来规划，提出要在 IP 生命周期之前接入为 IP 制定最适合的长线规划。</p><p>" 我们希望更多的用户能够在包括文字、动漫、电影、电视、游戏等多样化的娱乐媒体形态中，不断享受阅文 IP 带来的沉浸感和愉悦感。同时随着时间的推演，也不断会有新的优质 IP 产生，形成 IP 生态循环。我们也将以 IP 为抓手，不断推进产业生态合作，拓宽我们的边界。我们相信，IP 在各个细分领域都有着巨大的增量空间和发展潜力。" 这是程武对阅文集团未来的展望。</p><p><strong>IP 生态链的完善，商业模式的迭代，带来的改善已有端倪</strong>——作家们获得回报的渠道不再局限于文字形式，同时也使内容创作者与下游的对接更紧密，视野和资源也更广阔。</p><p>但与此同时，阅文并没有停止对创作者的加大投入，在这次发布会上也明确提出未来阅文的网络文学发展，要看 " 是否给作家带来更长期的收入 "，不再单一地以 DAU、利润率等数据来衡量发展水平，可以说，是种明确的态度表达。</p><p>而过去一年里，经历了管理层变革、合同风波后，阅文转型的决心肉眼可见，作家生态建设在作家眼中有一定成效。</p><p>希行对此感受十分明显。她在接受采访时提到，小到码字工具 " 作家助手 "，大到选择免费还是付费、分成机制，以及推动作品曝光量的各种活动，都有了明显的提升。</p><p>在阅文合同风波结束后，也有不少之前离开的大神作家回归了阅文，徐公子胜治就是在当时决定回到阅文的。在回归前，他写了一篇万字长文《文学网站与作者》，细数了网络文学二十年来的合同变迁。</p><p></p><div class="img_box" id="id_imagebox_4" 从现在来看，网文行业的历史问题在我们这个时间点有了转向，原先有一些不公平、不合理的条款被删除了，也给了作者更多的选择权，这是个非常好的趋势。" 在徐公子看，当时是作者开始注重并争取自己的权益，而平台也意识到这种业态趋势不能再这么发展下去，应该更尊重作者的权益，建立一个更良好的合作关系。<p></p><p>在作家 " 会说话的肘子 "（以下简称 " 肘子 "）眼中，最大的变化在于，作者参与 IP 改编的话语权更大了。肘子提到，头部作者的 IP 改编开发作者可以深层次参与到改编中，以前这种情况很少，<strong>" 新管理团队主动促使原著作者的参与度，原著作者的话语权也变得更大一些了。"</strong></p><p>根据肘子了解，虽然以前外界一直喊 IP 时代到来，但由于售价昂贵，作者的版权并不容易出售，但在最近一年来，他身边的中部甚至只写短篇的作者都收到了版权费、版权金，中腰部作者的 IP 改编率有了明显提升。" 新管理团队进来以后，把整个腾讯影业、腾讯动漫等泛娱乐产业链自然整合起来了，阅文财报上也有体现，IP 版权类的收入增长很明显。" 查阅阅文 2020 年度的财报可以看到，阅文一年对外授权的 IP 超 200 个。</p><p>阅文随财报发布的对投资人的信中写道，" 在过去，每个 IP 授权协议都随着一串销售数字结束；而今天，每个 IP 授权协议都是一个 IP 长期价值产生的开始，阅文集团会与行业内的合作伙伴展开充分合作，为实现 IP 的长期价值增长共同努力，打造 IP 行业生态，并从 IP 的长期价值创造中获益。"</p><p>程武在发布会演讲中提到，<strong>中国目前 IP 产业工业化的能力仍然处在初级阶段，在产业成熟度、IP 运营能力、IP 人才储备上，都需要长期探索。</strong></p><p>他不止一次地在演讲中强调人才的重要性。</p><p>过去一年里，除了改善作家生态外，阅文还将目光放在了整个产业链上的创作者，将目标锁定在为整个行业培养年轻的后备军。</p><p>" 中国影视行业的蓬勃发展，归根到底，需要有新生力量源源不断的注入，我们跟好莱坞最大的差距不在于资本，而在于好莱坞有整个成熟的工业体系，有很多具有专业能力的人支撑起来。所以我们深信，让这些新生力量去探索互联网 + 电影的更多可能性，这对中国未来影视产业的创新和发展有重要意义。" 程武在接受采访时如是说道。</p><p>去年 10 月，腾讯影业发起，新丽传媒、阅文影视发起了 " 青年导演培养计划 "，并公布了第一批入选者。自带互联网基因，盘活影视资源、打通产业链上下游的平台型生产机构，开始为科班出身、手握国内外奖项的青年导演搭戏台。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0
