<p><strong>文 / 羊城晚报全媒体记者 赵亮晨</strong></p><p>年龄危机在当今引起了众多关注，有些互联网大厂的程序员经常自嘲 "35 岁就没人要了 "，没想到连 C 罗都遇到了同样的烦恼。</p><p>最近这位 37 岁的巨星想离开曼联寻找新东家，却遭遇了包括拜仁慕尼黑、马德里竞技和尤文图斯等豪门的明确拒绝。而且这些闭门羹还格外烫嘴，比如拜仁就明确说 "C 罗不符合我们俱乐部的哲学 "，马竞球迷更是拉起横幅抵制称："C 罗别来！"</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202207/62e601298e9f0972f83a09ca_1024.jpg" data-height="489" data-width="693" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202207/62e601298e9f0972f83a09ca_1024.jpg" referrerpolicy="no-referrer"></div></div>马竞球迷不欢迎 C 罗<p></p><p>围绕 C 罗的争议和质疑之声也越来越多，这位以自律的职业精神著称的球员，似乎突然之间变成了 " 全民公敌 "。</p><p>这是怎么一回事？</p><p>C 罗和曼联的合同还剩一年，由于曼联下赛季没有欧冠席位，因此 C 罗希望改换门庭，转投一家能打欧冠比赛的俱乐部。除了在更高竞技舞台保持状态的目的之外，他还希望能够创造连续 20 个赛季征战欧冠的纪录。</p><p>合同仅剩一年，球员有意向找新东家，本来算是 " 常规操作 "。但是，具体到 C 罗这一次，似乎就值得推敲。</p><p>C 罗的名气实在太大，他的转会意愿成了地球人都知道的秘密，无论这样的张扬是有意为之还是无可奈何，都会给现东家曼联造成很大的被动，C 罗公开的态度可能让其他球队在谈判时占据有利地位，掌握更大的议价主动权。</p><p>更关键的问题在于，被 C 罗置于尴尬境地的是对其职业生涯意义非凡的曼联。一年前，当 C 罗回归老特拉福德球场时，他和曼联的重逢被传为佳话，他的个人品牌形象也从中获得了不小的提升和收益，于情于理，似乎都不该对曼联如此决绝。</p><p>曼联上赛季虽然没能拿到欧冠席位，但在战术上也算是待 C 罗不薄，众星拱月般地让他保持了不错的个人数据。</p><p>C 罗惹怒不少球迷的另一个问题在于，他不但要走，而且不愿意牺牲任何个人利益，不愿降薪。在目前总体不太景气的欧洲足坛，有钱的豪门不愿意负担一位老将的高昂薪水，更愿意将资金用在年轻球员身上，普通球队则更难以容纳 C 罗这样一条 " 鲸鱼 "。</p><p>而 C 罗的转会悬而未决的时间越久，曼联新赛季的备战受到的影响就越大。</p><p>现代足球是基于商业逻辑运转的，球员和俱乐部的分分合合本来无可厚非，能够终老于和自己最有感情的俱乐部已经是一种奢侈。就连和 C 罗并称 " 绝代双骄 " 的梅西，也已经离开了巴塞罗那。</p><p>巨星和 " 母队 " 也不乏出现矛盾、不欢而散的案例。但是，像 C 罗这样仅仅因为球队打不上欧冠，就旗帜鲜明大张旗鼓要求转会的，还是比较罕见的。</p><p>足球并不只有合同的约束，而是有着各种情感的联结羁绊，这是它区别于纯粹的商业活动的根本原因。所以，无论是球员还是俱乐部，如果一味地在商言商或者 " 目标导向 "，势必就会在类似的争议中成为球迷的众矢之的。</p><p>当然，这不只是 C 罗的责任，他的金牌经纪人门德斯的强势操作也是一大原因——早已习惯在转会市场上呼风唤雨的门德斯，甚至提出曼联先和 C 罗续约、然后再将 C 罗租借出去的傲慢要求。</p><p>球员和母队的感情，经纪人是不可能感同身受的，这种情感区隔有时候可以帮助球员利益最大化，有时候则会在寻求利益最大化的过程中 " 翻车 "。</p><p>作为个体，C 罗仍然是堪称楷模的敬业球员。但是，在这次的转会争议之后，曾经视他为球队历史旗帜人物之一的曼联球迷，也许会有不少人改变这种想法。不仅如此，关于足球的情感是共通的，其他俱乐部的球迷也可以很容易代入曼联球迷的处境，与之共情，C 罗成为 " 全民公敌 " 也就不足为奇了。 ( 更多新闻资讯，请关注羊城派 pai.ycwb.com ) </p><p><strong>来源 | 羊城晚报 ·
