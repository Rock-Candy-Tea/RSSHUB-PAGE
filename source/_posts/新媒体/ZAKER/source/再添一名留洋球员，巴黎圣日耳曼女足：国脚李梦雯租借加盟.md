<p>北京时间 9 月 2 日，巴黎圣日耳曼女足官方宣布，中国女足国脚李梦雯租借加盟球队。李梦雯现年 27 岁，司职左后卫，作为轮换球员参加了今年女足亚洲杯和东亚杯，此前她在江苏无锡女足效力。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202209/6312fdfa8e9f09387e704bcc_1024.jpg" data-height="1080" data-width="1620" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202209/6312fdfa8e9f09387e704bcc_1024.jpg" referrerpolicy="no-referrer"></div></div>此次李梦雯租借加盟的法甲豪门巴黎圣日尔曼俱乐部，国内球迷并不陌生，此前女足国脚王霜就在这家俱乐部效力，这也是大巴黎时隔两年半再次引入中国女足球员。 据悉，李梦雯将为大巴黎效力直至 2023 年 6 月。<p></p><p>李梦雯来自苏州吴江，小时候是练田径的，主攻 400 米，2006 年小学 4 年级时改练足球，最初是踢前锋的，后来进了国少队改打右后卫。2017 年 10 月，她第一次进入女足国家队，贾秀全时期曾是主力右后卫，目前在水庆霞国家队是主力替补，除了右后卫，她还可以打左后卫和中场，是一个多面手。</p><p>李梦雯一直都想留洋，去年奥运会后，这种意愿更加强烈。原本，她想去意大利，也有经纪人帮她联系了，后来，她和江苏女足前主帅杰斯林聊天，询问他的意见，结果，他问李梦雯，想不想去大巴黎，" 当时还是很惊讶的，后来想想，觉得有机会一定要尝试一下。" 接受采访时，李梦雯说。</p><p>李梦雯表示：" 留洋的想法一直都有，从开始进入职业生涯就梦想留洋。这次真的可以留洋了，是圆了自己的一个梦。" 她还透露，自己曾两次拒绝开出高薪的武汉队，" 留洋是我的梦想，但凡有机会，我还是会去留洋，希望自己开阔视野。"</p><p>据统计，目前中国女足有多名球员在外留洋，包括沈梦露、沈梦雨 ( 凯尔特人 / 苏格兰 ) ，涂琳俪 ( 冰岛甲 ) 、赵瑜洁 ( 丹麦 / 克厄 ) 、王霜 ( 路易斯维尔竞赛 / 美国 ) 、唐佳丽 ( 马德里 CFF/ 西班牙 ) 、杨淑慧 ( 莫斯科火车头 / 俄罗斯 ) 、张琳艳 ( 草蜢女足 / 瑞士超 ) 。</p><p>潇湘晨报记者周诗浩</p><p>更多精彩内容，请在应用市场下载 " 晨视频 " 客户端。欢迎提供新闻线索，一经采纳即付报酬。爆料微信关注：xxcbwx，24 小时报料热线 0731-85571188。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
