个。</p><p>赵：这种感觉似曾相识呀，C 罗在尤文图斯也出现过这种情况。C 罗 2018 年夏天加盟尤文，尤文在 2017-2018 赛季打进 86 个意甲进球，但在 2018-2019 赛季只打进 70 个意甲进球。</p><p>徐：C 罗在尤文时，受影响最大的是迪巴拉，阿根廷人的意甲进球数由 2017-2018 赛季的 22 个暴跌至 2018-2019 赛季的 5 个。所以说到底就是 C 罗只是挤占了球队的存量，没有贡献增量。</p><p>硬走，到底能去哪里？</p><p>刘：C 罗如果真的可以转会离开曼联，你们有没有帮他选好一个适合去的球队呢？</p><p>徐：现在问题在于，C 罗转会前景不是很被看好，因为没有太多下家可以接受他，他就像一头鲸鱼，确实太大了。</p><p>赵：让我们先看巴黎圣日耳曼，那里毕竟已经有三大超级巨星了，即便是以大巴黎这个资金的雄厚程度，他们应该也是吃不下 C 罗的，因为他的薪水实在太高了。如果大巴黎吃下 C 罗的合同，就完全没办法向欧足联的财政公平政策交代了。</p><p>刘：C 罗 50 万欧元的周薪确实太高了，媒体硬把他推荐给拜仁慕尼黑。但是拜仁高层说得很不客气，直言他与球队的文化不是很契合。那么切尔西有没有可能买下 C 罗呢？</p><p>徐：切尔西刚换了老板，新老板肯定希望带来新官上任三把火的效果，因此有引进巨星来提振士气和提高球队影响力的需要。切尔西更换老板以后，球队阵容变化非常大。既然切尔西的人员流动这么大，如果 C 罗可以加盟，全队正好可以重新磨合。</p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div
