<p><a href="http://iphone.myzaker.com/zaker/link.php?pk=6287be9db15ec003263153a1&b=aHR0cHM6Ly93d3cuY2hpbmFuZXdzLmNvbS5jbi8=&bcode=244cb3a3&target=_new" target="_blank">中新网</a>上海 5 月 19 日电 ( 记者 郑莹莹 ) 根据上海市民政局 19 日在其微信公众号上公布的数据，据初步统计，截至 2022 年 5 月 18 日，上海市疫情防控社会捐赠收入已达 11.72 亿元 ( 含上海市教育发展基金会接受联合国儿童基金会捐赠的 361.10 万美元，折合人民币 2386.20 万元物资 ) ，捐赠支出已达 11.09 亿元。</p><p>上海市民政局作为沪公益慈善事业的行政管理部门，负责对上海全市慈善组织开展社会捐赠及使用的情况进行监督。</p><p>具体而言：</p><p>一、上海市慈善基金会</p><p> ( 1 ) 捐赠收入累计 3.06 亿元 ( 含承诺捐赠 581.66 万元 ) ，其中：资金收入 1.66 亿元，物资折款 1.40 亿元；</p><p> ( 2 ) 捐赠支出累计 2.86 亿元，其中：资金支出 1.49 亿元，物资支出 1.37 亿元。</p><p>二、上海市红十字会</p><p> ( 1 ) 捐赠收入累计 1.61 亿元，其中：资金收入 0.44 亿元，物资折款 1.17 亿元；</p><p> ( 2 ) 捐赠支出累计 1.46 亿元，其中：资金支出 0.30 亿元，物资支出 1.16 亿元；</p><p>三、上海联劝公益基金会、上海市职工帮困基金会、上海市青少年发展基金会等 3 家开展公开募捐活动的基金会</p><p> ( 1 ) 捐赠收入累计 6308.64 万元，其中：资金收入 2936.65 万元，物资折款 3371.99 万元；</p><p> ( 2 ) 捐赠支出累计 4733.71 万元，其中：资金支出 1374.64 万元，物资支出 3359.07 万元；</p><p>四、其他 205 家基金会</p><p> ( 1 ) 捐赠收入累计 6.41 亿元，其中：资金收入 2.60 亿元，物资折款 3.81 亿元。</p><p>捐赠收入主要包括：华侨基金会 1.16 亿元，复星基金会 0.33 亿元，老年基金会 0.27 亿元，35 家高校教育基金会 2.44 亿元，其他 167 家基金会 2.21 亿元。</p><p> ( 2 ) 捐赠支出累计 6.30 亿元，其中：资金支出 2.22 亿元，物资支出 4.08 亿元。</p><p>捐赠支出主要包括：华侨基金会 1.14 亿元，复星基金会 0.34 亿元，老年基金会 0.27 亿元，35 家高校教育基金会 2.09 亿元，其他 167 家基金会 2.46 亿元。</p><p>注：捐赠支出大于收入情况，主要是有些基金会从历年结余的资金和物资中安排。 ( 完 ) </p><div id="recommend_bottom"></div><div id="article_bottom"></div>
