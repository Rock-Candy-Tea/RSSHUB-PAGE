<p>编者按：</p><p>每一个努力生活的中国人，都是最美的奋斗者。也正是因为亿万奋斗者，才有了今日之中国。十年，致敬每一个奋斗的你。让我们一起，踔厉奋发新时代，笃行不怠向未来。</p><p>【解说】整理衣装、签出车单、核对信息，32 岁的动车司机袁彬如往常一样，从西安北车站司机休息室出发，开始值乘西安至郑州高铁。十年间，陕西高铁里程迈上了 1000 公里大关，袁彬也从一名电力机车司机转岗为动车组司机。</p><p>【同期】西安机务段动车组司机 袁彬</p><p>我的父亲也是一名火车司机，从事过蒸汽机车、内燃机车和电力机车 ( 驾驶工作 ) ，从小对我的影响很大，在我心中从小就有一个火车头的梦想，最终我也通过自己的努力，然后合格地成为一名电力司机，最后转岗为动车组司机。</p><p>【解说】袁彬和他的父亲袁建昌见证了陕西铁路的发展与变迁——动车组司机队伍的扩大、机车技术的升级、动车开行比例的跃升。</p><p>今年 59 岁的袁建昌是西安机务段的一名退休火车司机，在司机这个岗位上干了 36 年。在袁建昌的家中，他翻出有些泛白的蓝色火车司机制服穿上，与身着白色制服的儿子坐在一起，讲着自己的职业生涯。</p><p>【同期】西安机务段退休司机 袁建昌</p><p>我们上班的时候，当时开蒸汽机车，平均时速下来就是三四十公里。随着这个时代的发展，到我退休的时候，我们的机车已经达到了 160 公里 ( 每小时 ) 。这几年发展非常快，动车发 300 多公里 ( 每小时 ) ，但是我们那个时候年龄已经到了，不适应于这种速度，所以我们没有能够开上动车，很遗憾。</p><p>【解说】袁建昌开过西宝 ( 西安至宝鸡 ) 、郑西 ( 郑州至西安 ) 等铁路线。据他介绍，20 世纪 90 年代，从西安开往郑州的火车需要十几个小时，如今只需要约两个小时。中国铁路的飞速发展，让他对曾经的驾驶岗位有些不舍。每次与儿子交谈，他总会传授一些工作中的小技巧和应该注意的安全问题。</p><p>【同期】西安机务段动车组司机 袁彬</p><p>父亲经常教导我，虽然现在开上了时速 300 公里的动车组，但是我们根本的任务还是要把每一趟列车，每一位旅客安全地送到目的地，这才是我们最终的目标。所以在作业中必须要严格要求自己，以最高标准完成每一趟值乘任务。</p><p>【解说】十年来，在袁建昌、袁彬等铁路工作者的努力下，陕西高铁大 " 米 " 字形高铁网连接全国。乘坐在飞驰的列车上，游客一日之内便可跨黄河、钻秦岭、穿沙漠、越高原，尽览辽阔国土上的多样风情。</p><p>记者 党田野 张远 西安报道</p><p>责任编辑：【刘湃】</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
