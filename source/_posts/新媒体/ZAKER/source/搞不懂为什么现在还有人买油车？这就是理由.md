<p>现阶段，买油车还是电车，依旧是很多人的烦恼。</p><p>"<strong>完全想不明白，现在大家为什么还买油车，除了能闻点汽油味，别的还有什么好？听个响？</strong>"</p><p>不久前，蔚来创始人李斌在 NIO Day 上发出了灵魂拷问。</p><p>除此之外，李斌还含蓄地表示油车就如同机械表一样，更多的是情怀，而如今还在吐槽充电焦虑的也基本上是汽油车用户在操着电动车用户的心。</p><p>虽然李斌一番言论主要是为自家刚发布的新车 ET5 撑场子，但如此诋毁油车的做法，还是遭到了众多网友的炮轰，有网友一针见血地指出：" 现在开油车起码没有里程焦虑症、充电桩焦虑症、空调焦虑症、暖风焦虑症、冬天趴窝焦虑症 ……"</p><p>尽管电车的发展是大势所趋，2021 年开始已经有了明显的替代迹象，渗透率的持续攀升就是最好的证明，但在这样的高光之下，自燃事件频发、充电时间太长、电动续航腰斩等各种各样的问题，依旧困扰着电动车。</p><p><strong>德勤日前的一项全球调查发现，在汽车制造商向电气化未来迈进的同时，对电动汽车的局限性持谨慎态度的消费者继续被内燃动力汽车所吸引。</strong></p><p>仅在美国，就有 69% 的消费者表示，他们希望下一辆汽车配备内燃机（ICE）动力系统。尽管全球对可持续性的兴趣日益浓厚，但超过一半的美国消费者 ( 53% ) 不愿意为电动车替代燃油车多支付超过 500 美元。</p><p>美国消费者希望电动汽车能够达到 805 公里的续航里程</p><p>德勤的《2022 年全球汽车消费者研究》是基于去年 9 月至 10 月间，来自 25 个国家的 2.6 万多名消费者的回应。</p><p>当被问及 ICE 和 EV 动力系统时，消费者给出了以下回答：</p><p>在美国、德国、日本、韩国、印度和东南亚的消费者中，对气候变化和减排的担忧一直是采用电动汽车的两大动力之一。</p><p>韩国 ( 23% ) 、中国 ( 17% ) 和德国 ( 15% ) 消费者对纯电动汽车 ( BEV ) 的偏好最高，而日本消费者对混合动力和插电式混合动力汽车 ( HEV/PHEV ) 的偏好最高 ( 48% ) ，其次是韩国 ( 35% ) 。</p><p>大多数电动汽车意向者希望在家中为其车辆充电，特别是在日本（76%）、印度（76%）、美国（75%）和德国（70%）。韩国 ( 38% ) 和东南亚 ( 29% ) 对公共充电的需求最高。</p><p>在那些计划在家给汽车充电的人中，三分之二（66%）的美国人倾向依赖传统电网。与此同时，印度、中国和东南亚的消费者计划同时使用常规电网和可再生能源。</p><p>续驶里程是德国 ( 24% ) 、中国 ( 22% ) 和美国 ( 20% ) 消费者对电动汽车最关心的问题。公共充电基础设施的缺乏是亚洲最关心的问题（东南亚为 28%，韩国为 26%，印度为 23%，日本为 19%）。</p><p>美国消费者希望电动车续航里程能够达到 500 英里（805 公里）以上，而在中国、日本和印度，行驶 250 英里（403 公里）左右是可以接受的。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
