<p>1 月 9 日消息，据外媒报道，特斯拉电动汽车不仅能帮助车主节省汽油开支，甚至还能用于开采加密货币赚钱。最近多位特斯拉车主爆料称，他们利用特斯拉汽车 " 挖矿 "：，每月最多可赚 800 美元（约合 5100 元人民币）收益。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202201/61da39078e9f09454800e8b3_1024.jpg" data-height="372" data-width="660" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202201/61da39078e9f09454800e8b3_1024.jpg" referrerpolicy="no-referrer"></div></div>特斯拉车主西拉杰 · 拉瓦尔利用 Model3 开采加密货币<p></p><p>对于特斯拉车主西拉杰 · 拉瓦尔 ( Siraj Raval ) 来说，他几乎尝试了所有利用 2018 年款特斯拉 Model 3 挖掘加密货币的方法。拉瓦尔在苹果 Mac mini M1 上运行免费的比特币挖掘软件，并通过将逆变器插入位于特斯拉汽车中央控制台的 12 伏电源插座来为电脑供电。他还将互连的 GPU（图形处理单元）连接到特斯拉行李箱上，利用汽车的内部电池来运行这些机器。</p><p>但拉瓦尔最喜欢的开采以太坊这样加密货币的方法是使用 JavaScript 侵入特斯拉本地 CPU。虽然这可能会使他的汽车保修期失效，但拉瓦尔认为这是值得的。他解释说，在以太坊和 Polygon 的价格于 2021 年达到顶峰时，他每个月都能利用这两种产品净赚 800 美元。</p><p>比特币矿商亚历杭德罗 · 德拉托雷 ( Alejandro De La Torre ) 表示，利用特斯拉汽车采矿就像连接到任何其他电源一样。他解释称：" 主要因素是电价。如果通过电动汽车做这件事更便宜，那就这样吧！"</p><p><b>如何用特斯拉汽车挖币</b></p><p>早在 2018 年，美国威斯康星州史上首位电动汽车经销商克里斯 · 阿拉西 ( Chris Alless ) 决定对他的特斯拉汽车进行改造。</p><p>与拉瓦尔类似，阿拉西已经尝试了几种不同的方法，将他的特斯拉 Model S 改造成加密货币采矿设备。在行业中，加密货币挖掘是个能源密集型的过程，世界各地的机器将它们的计算能力贡献给整个网络，以创造新的加密货币并验证现有的交易。他们通过运行专门的软件来处理复杂的数学方程来做到这一点。要参与进来，基本上你只需要电脑和持续的电源供应。</p><p>阿拉西在电源逆变器的帮助下，将 Bitmain Antminer S9 ( 专门用于开采世界上最受欢迎加密货币的采矿设备 ) 直接插入他的汽车电池，尝试开采比特币。逆变器将特斯拉电池的电压调整到与 Antminer 兼容的水平。阿拉西还成功地使用了车辆的内部固件来开采替代币。</p><p>在谈到这个过程时，阿拉西称 " 这没什么大不了的 "。他使用车里的内置电脑和屏幕导航到一个网页，这个网页是他专门为加密货币 Monero 而设置的。他解释称：" 我可以在浏览器中运行挖掘程序。"</p><p>在拉瓦尔尝试过的所有技术中，最有利可图的技术包括侵入特斯拉内部计算机，以及将 GPU 直接插入汽车的电机中。拉瓦尔使用 JavaScript 创建 Web 应用程序，可以从他的特斯拉汽车中检索数据，在某些情况下，还可以修改数据。</p><p>他说：" 这实际上就是带轮子的电脑，侵入这辆电脑车是如此简单。" 他形容这个过程实质上是劫持了汽车的内部固件，以允许额外的电力使用。随后，他将五个 GPU 连接到特斯拉电池上，然后在运行两种不同的散列算法之间切换：一种是挖掘以太坊，另一种是生成 Polygon。</p><p>专业级矿商证实，从理论上讲，这种方式是行得通的。比特币开采服务提供商 Compass 的首席执行官兼创始人惠特 · 吉布斯 ( Whit Gibbs ) 解释说：" 所有的机制都在那里。如果你有电源，有足够空间，有能力增加冷却功能，那么电池肯定有足够的电量来启动 ASIC 并运行它。"</p><p><b>汽车充电费仍是关键</b></p><p>利用特斯拉汽车挖掘加密货币是否被证明有利可图，与车主何时购买汽车有很大关系。例如，阿拉西在 2017 年 1 月之前购买了他的汽车，这意味着他被赋予了一项特权：在他的汽车生命周期内，他可以免费且无限制地使用特斯拉充电站网络。</p><p>阿拉西估计，在 2018 年，在 60 小时的时间窗口内，他将赚取价值 10 美元的比特币，所有这些比特币都是盈利的，因为他不必支付电费。但他表示，尽管利润丰厚，但现在几乎不再值得。他说。" 目前，尽管比特币的价格大幅上涨，但开采难度也在上升。在同样的时间里，使用完全相同的设备，我可能会看到价值 1 美元或 2 美元的比特币。"</p><p>Monero 的开采也同样徒劳无功。阿拉西解释称：" 这种方法行得通吗？的确可行。但它开采了任何有价值的东西，以便能够以任何方式、形状或形式盈利吗？并非如此。"</p><p>不过，拉瓦尔对利用特斯拉汽车盈利的潜力更为乐观。他说，尽管他必须为汽车充电付费，但电池本身是 " 无与伦比的 "，所以他可以得到很多好处。</p><p>拉瓦尔的车每次充电可以行驶 515 公里，充电大约需要 10 到 15 美元。如果他每天开几个小时的车，每一周半就需要充电一次，这样他每月的账单就在 30 美元到 60 美元之间。拉瓦尔估计，他每天用特斯拉电池挖矿大约 20 个小时。尽管以太坊和 Polygon 等替代币的价格容易波动，但拉瓦尔已做了其他准备，以确保盈利。</p><p>阿拉西承认，他不再为采矿而烦恼。他说：" 开采难度太高了，我可以在麦当劳工作赚更多的钱。" 阿拉西确实提供了一个替代方案，即利用雇主的电力开采加密货币。他说：" 如果你为一家公司工作，最好利用好雇主的福利，而不是开车去采矿。"</p><p>但对拉瓦尔来说，利用特斯拉汽车挖掘加密货币不仅仅是一个噱头。相反，这是他计划的一大部分，即让他的特斯拉成为 " 全自动驾驶出租车 "，在不驾驶的任何时候都能赚取加密货币。</p><p>尽管特斯拉首席执行官埃隆 · 马斯克 ( Elon Musk ) 吹嘘汽车成为自动驾驶出租车的潜力，但今天的特斯拉仍远未能够实现自动驾驶。尽管如此，拉瓦尔相信这样的未来。他说：" 特斯拉汽车将利用来自运输服务和加密货币挖掘的收益来支付养护费用，如维修、电费和升级，并将这些费用投资于新兴加密货币社区网络的多元化投资组合。"（小小）</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
