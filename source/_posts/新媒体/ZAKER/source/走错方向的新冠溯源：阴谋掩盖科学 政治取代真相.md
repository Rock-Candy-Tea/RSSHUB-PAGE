政治取代真相</p><p>作者 张焕迪</p><p>新冠病毒需要溯源。在今年世界卫生组织专家组赴中国联合开展为期 28 天的实地研究后，已经得出了重要结论：新冠病毒 " 非常可能 " 的来源是动物或自然界；" 极不可能 " 源于实验室泄漏。</p><p>政治病毒也要溯源。美国背离科学和事实，对溯源问题大搞政治化和污名化，甚至大搞胁迫，不择手段干扰国际溯源研究合作，已经对国际合作抗疫造成严重影响。在相关问题上，已有大量国际知名专家学者发声，反对将新冠溯源政治化，强调 " 科学引路 " 的重要性。</p><p><strong>专家斥 " 实验室泄漏论 " 毫无根据</strong></p><p>美国生态健康联盟主席、英国动物学家彼得 · 达扎克 ( Peter Daszak ) 今年亲赴武汉，参与世卫组织开展的溯源研究工作，他在多个场合公开斥责 " 实验室泄漏论 " 毫无根据。</p><p>达扎克在接受美国有线电视新闻网采访时透露：" 我们与武汉的实验室已合作了 15 年，知道他们所做的一切，从未见到他们的科研人员撒谎， ( 武汉实验室 ) 更没发生过任何‘异常状况’。" 他着重强调：" 新冠暴发之前，武汉的实验室中不存在这种病毒。"</p><p>荷兰病毒学家、世卫组织专家马里恩 · 科普曼斯 ( Marion Koopmans ) 接受英国广播公司采访时质疑道：" 美国持续炒作‘实验室泄漏论’已有近一年时间，如果确实有真凭实据，美国有责任分享出来。" 她还补充说：" 我与同事讨论过实验室泄漏的可能性，答案就白纸黑字地写在世卫组织的报告上。"</p><p>美国《连线》杂志科学新闻记者亚当 · 罗杰斯 ( Adam Rogers ) 一针见血地指出，所谓 " 实验室泄漏论 "，不过是部分人在利用科学本身的不确定性牟取政治利益。</p><p><strong>科学界广泛认同新冠来源于自然</strong></p><p>俄罗斯科学院通讯院士，新西伯利亚国立大学生物纳米技术、微生物学和病毒学实验室主任，生物科学博士谢尔盖 · 涅捷索夫接受俄罗斯卫星通讯社专访时重申，迄今为止，大多数专家和学者都认为新冠来源于自然。</p><p>涅捷索夫直言，尽管目前新冠的来源有诸多假设，还有阴谋论者臆造不同版本，但目前 " 大多数专家和学者，而不是那些键盘病毒学家和集市上的长舌妇，都认为这 ( 新冠 ) 是自然来源。" 他强调，现在最关键的是仔细分析疫病本身。</p><p>俄罗斯叶卡捷琳堡病毒感染科学研究院院长亚历山大 · 谢苗诺夫 ( Alexander Semyonov ) 同样表示：" 新冠病毒来自大自然，认为病毒是人造的看法不合理也不专业。"</p><p>俄罗斯病毒学家、世卫组织专家利沃夫强调，大多数专业人士毫不怀疑新冠病毒起源于自然界，而所谓 " 人为制造 " 一说大多出自政客之口，就如俄罗斯总统普京所说：" 政客们都摇身一变成了病毒学家。"</p><p>利沃夫还透露，包括他自己在内的许多世卫组织专家都面临来自美国的压力，" 这些美国人不想让病毒学家说出关于疫情的实情。"</p><p><strong>科学问题成了谁的政治工具？</strong></p><p>不难看出，如今科学界对新冠的真实来源已有基本结论，炒作 " 实验室泄漏论 " 等论调不过是部分国家借题发挥，试图将科学问题政治化的伎俩罢了。</p><p>美国罗伯特莫里斯大学传播与组织领导学院教授安东尼 · 莫雷蒂 ( Anthony Moretti ) 曾表示，科学界的调查已经非常有力地证明了 " 实验室泄漏论 " 的不可能性，继续推广这种缺乏实质内容的理论，纯粹是为了满足美西方国家削弱中国全球信誉的需要而已。</p><p>亚历山大 · 谢苗诺夫也认同此看法。他说：" 非常遗憾的是，对新冠的研究涉及太多政治因素。而在当前的中美关系中，美国认为指责中国更有利。"</p><p>新冠溯源现在是，以后也是科学问题。美西方国家选择性无视摆在面前的真凭实据，五次三番炒作 " 实验室泄漏论 "，多次重提对中国再次溯源调查，无非是想借此对中国施压。对他们来说，在新冠溯源这一问题上，真正重要的不是科学事实，而是政治利益至上的立场 " 站位 "。</p><p>但愿美西方国家有朝一日能领悟爱因斯坦曾说过的这句话：精神错乱的人才会一遍又一遍地做着同样的事情，却期望得到不同的结果。 ( 完 ) </p><p>中国军团里藏了多少 " 吃货 "？这届奥运会给我看饿了……</p><p>机场→演出→游船串联病例，防疫警报全面拉响！</p><p>警惕疫情 " 破防 "：这次疫情因何而起？暴露出哪些问题？</p><p>蜥蜴土拨鼠桌旁伴餐
