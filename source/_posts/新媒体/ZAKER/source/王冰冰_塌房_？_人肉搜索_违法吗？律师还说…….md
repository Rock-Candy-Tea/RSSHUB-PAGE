吉林分会场采访记者。这几年，其凭借清新外表、活力形象圈粉无数，成为观众及网友追捧的对象。</p><p>截至发稿，王冰冰个人微博现有粉丝为 519.4 万人。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy 425 分 ) 。就此，也有不少网友留言称，四级仅一次没过无法说明什么，" 毕竟还有第二次的机会 "。</p><p>目前，央视方面和王冰冰还未对此作出回应。</p><p><strong>" 人肉搜索 " 涉嫌违法</strong></p><p>这些年，公众人物隐私遭曝光话题屡见不鲜，网友讨论度高。那，问题来了，" 人肉 " 他人信息是否触犯法律？民众应如何维护自己的合法权益？12 日，中新网记者就此采访多位律师，为网友释疑解惑。</p><p>北京市中闻律师事务所权益合伙人贾虹律师介绍称，" 人肉搜索 " 以非法使用他人个人信息、违法散布他人隐私信息为主要表现形式。" 它 ( ‘人肉搜索’ ) 涉及违法，可能会侵犯他人的隐私权、名誉权，姓名权、肖像权等，也可能会涉及侮辱诽谤和非法获取个人信息等问题。"</p><p>她指出，民法典规定，自然人享有隐私权。任何组织或者个人不得以刺探、侵扰、泄露、公开等方式侵害他人的隐私权。隐私是自然人的私人生活安宁和不愿为他人知晓的私密空间、私密活动、私密信息。</p><p>此外，自然人的个人信息受法律保护。个人信息是以电子或者其他方式记录的能够单独或者与其他信息结合识别特定自然人的各种信息，包括自然人的姓名、出生日期、身份证件号码、生物识别信息、住址、电话号码、电子邮箱、健康信息、行踪信息等。</p><p>" 个人信息中的私密信息，适用有关隐私权的规定；没有规定的，适用有关个人信息保护的规定。" 她指出。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 写恐吓信或者以其他方法威胁他人人身安全的； ( 二 ) 公然侮辱他人或者捏造事实诽谤他人的； ( 三 ) 捏造事实诬告陷害他人，企图使他人受到刑事追究或者受到治安管理处罚的； ( 四 ) 对证人及其近亲属进行威胁、侮辱、殴打或者打击报复的； ( 五 ) 多次发送淫秽、侮辱、恐吓或者其他信息，干扰他人正常生活的； ( 六 ) 偷窥、偷拍、窃听、散布他人隐私的。</p><p>" 公众人物的隐私权是需要有一定范围的让渡和限制的，享受公众关注以及公众人物得到的利益同时，也需要放弃部分个人隐私权的行使，但是 ( 评论的 ) 让渡和限制都需要在一定合理范围内。" 贾虹表示，可有关王冰冰之前的是否结婚、离异的信息，结合不知真假的婚礼照片、恋爱博客、不当评价的信息就不属于应容忍的正常评价了。</p><p>" 再有，目前的有关（疑似）其大学英语四级信息，显然是只提供了那次没过的，有些断章取义。" 她表示。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" referrerpolicy="no-referrer"></div></div>图片来源：王冰冰微博<p></p><p><strong>该如何维权？支招！</strong></p><p>北京德亮律师事务所万方亮律师提到，此前，国家互联网信息办公室发布《网络信息内容生态治理规定》。根据规定，网络信息内容服务使用者和生产者、平台不得开展网络暴力、人肉搜索、深度伪造、流量造假、操纵账号等违法活动。</p><p>此外，《中华人民共和国民法典》第一百一十一条规定：" 自然人的个人信息受法律保护。任何组织或者个人需要获取他人个人信息的，应当依法取得并确保信息安全，不得非法收集、使用、加工、传输他人个人信息，不得非法买卖、提供或者公开他人个人信息。"</p><p>遭遇 " 人肉 "，个人隐私信息被恶意泄露、传播，应如何维权？贾虹律师表示，首先，要及时做好取证工作，保留侵权页面、侵权者用户信息的页面，为后续维权做好准备。其次，要向信息发布平台发送符合要求的删除通知。比如列明权利人、权利凭证、侵权信息以及删除要求等。</p><p>" 最后，如果事态仍未得到控制，可以采取民事诉讼、刑事自诉 ( 侮辱诽谤 ) 和报警等处理方式。" 贾虹说。</p><p>万方亮也提醒：民法典颁布后，从法典的层面规定严禁所谓的人肉搜索，保护当事人的隐私权和名誉权，说明了法律加强了对个人权利的保护。如果有人利用所谓的 " 人肉搜索 " 侵犯他人的合法权益，可以依法维权。 ( 完 ) </p><div id="recommend_bottom"></div><div id="article_bottom"></div>
