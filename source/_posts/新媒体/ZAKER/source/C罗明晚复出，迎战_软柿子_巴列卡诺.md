src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202207/62e4b42db15ec04dd27427fe_1024.jpg" referrerpolicy="no-referrer"></div></div>葡萄牙巨星 C 罗<p></p><p>曼联今天和明天将要进行两场热身赛，今天 19 时 45 分在挪威奥斯陆迎战老对手马德里竞技；明天 23 时在老特拉福德球场迎战西甲球队巴列卡诺，那不仅是新主帅滕哈格在曼联主场的执教首秀，也很可能是 C 罗备战新赛季的首秀。</p><p>马竞原本与 C 罗充满爱恨情仇，但因为他久疏战阵，在迎战这个强硬对手时复出显然不是一个好的选择，因此他倾向于在主场捏 " 软柿子 " 巴列卡诺时复出。</p><p>马竞是 C 罗进球最多的球队之一，无论在皇家马德里还是在尤文图斯，他都带领球队在欧冠赛场淘汰或战胜过马竞。</p><p>2013-2014 赛季是马竞最接近欧冠冠军的一次，在那届欧冠决赛，马竞 1 比 0 领先皇马到了伤停补时读秒阶段，被拉莫斯攻入绝平进球，皇马经过加时赛以 4 比 1 大胜。</p><p>2018-2019 赛季欧冠 1/8 决赛，马竞首回合以 2 比 0 战胜尤文，C 罗在次回合比赛中上演帽子戏法，带领尤文以总比分 3 比 2 完成大逆转。</p><p>可是在上赛季欧冠 1/8 决赛比赛中，马竞以总比分 2 比 1 淘汰曼联，C 罗表现平平。近期出现 C 罗加盟马竞的传闻，在马竞上一场大胜努曼西亚的热身赛中，马竞球迷在看台上打出 " 不欢迎 CR7（C 罗）" 的标语。</p><div class="img_box" id="id_imagebox_1" 罗缺席了曼联在亚洲和澳大利亚的四场热身赛，如果在对马竞的热身赛复出，无疑充满话题性。但是在曼联公布的比赛名单中，没有 C 罗的名字，缺席这场比赛但又没有伤病的球员，可望在明天主场对巴列卡诺时参赛，其中包括 C 罗。</p><p>巴列卡诺上赛季在西甲排名第 12 位，是皇马的同城德比对手。2015 年 12 月 20 日，皇马曾在西甲主场以 10 比 2 狂胜巴列卡诺。</p><p>在那场比赛中，巴列卡诺一度以 2 比 1 反超比分，但随后被罚下两名球员，皇马把比赛变成进球表演。那是 "BBC 组合 "C 罗、贝尔、本泽马的全盛时期，他们合力攻入 9 球。</p><p>C 罗通过主罚点球和鱼跃冲顶攻入两球，还为贝尔送上一次助攻；贝尔上演大四喜，本泽马上演帽子戏法。</p><p>就算 C 罗明晚在对巴列卡诺时复出，但不代表他新赛季会留在曼联。C 罗近日带着经纪人门德斯与曼联管理层以及主帅滕哈格有过会谈，重申了自己要转会离队的想法，但曼联也重申了希望 C 罗履行与球队最后一年合同的想法。</p><p>根据最新消息，C 罗可能转会加盟老东家葡萄牙体育，那是一支可以参加下赛季欧冠的球队，也是 C 罗的母队。如果 C 罗最终成行，就会上演明星前锋倦鸟思归的帽子戏法——武磊决定回归老东家上海海港，苏亚雷斯决定回归老东家乌拉圭民族队。</p><p><strong>题图 | 视觉中国</strong></p><p><strong>责编 | 葛万里</strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
