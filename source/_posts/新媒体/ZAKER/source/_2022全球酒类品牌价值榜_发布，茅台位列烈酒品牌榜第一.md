Portfolios ) 榜单上，比利时的百威英博 ( AB InBev ) 升至首位，中国的茅台（Moutai）位列第二。前十名还有：中国的五粮液 ( Wuliangye ) 、荷兰的喜力 ( Heineken ) 、英国的帝亚吉欧 ( Diageo ) 、日本的朝日 ( Asahi Group ) 、法国的保乐力加 ( Pernod Ricard ) 、美国的摩森康胜 ( Molson Coors ) 、美国的百加得 ( Bacardi ) 和丹麦的嘉士伯集团 ( Carlsberg Group ) 。<p></p><p>Brand Finance 早前在英国伦敦发布的《Brand Finance2022 年度全球最具价值烈酒品牌 50 强》榜单及报告认为，世界上最具价值的烈酒品牌是茅台，这是一种由高粱发酵而成的白酒。</p><p>据知，这是茅台自 2016 年开始，连续第 7 次登顶全球最具价值烈酒品牌 50 强。</p><p>报告认为，茅台是世界上最具价值的烈酒品牌，价值 429 亿美元。该品牌在 2022 年度全球最具价值烈酒品牌 50 强排行榜上保持了第一名的位置，继续以其标志性的饮品引领行业并主导了中国市场。</p><p>报告指出，茅台善于利用其 i 茅台数字化营销平台向中国客户提供产品。该应用程序能够让用户通过全新的方式进行线上支付。</p><p>报告还提到了茅台冰淇淋，认为茅台与中国蒙牛乳业合作推出的含有酒精的冰淇淋，开拓了年轻人的市场。</p><p>"50 大烈酒品牌 " ( Spirits 50 ) 总价值 1426 亿美元，中国 10 个品牌上榜，价值 976 亿美元，占比 68.4%。茅台、五粮液、泸州老窖、洋河包揽前四名。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 在一份声明中指出，经济的重启极大地推动了酒类行业的增长，烈酒品牌同比增长了 6%。<p></p><p>今年 1 月 26 日，Brand Finance 发布的《2022 年全球品牌价值 500 强》中，茅台排名第 32 位，获得 AAA 的品牌评级，在全球酒行业排名第一，并与苹果、亚马逊、微软、华为、微信等知名品牌一道，跻身全球品牌价值前五十位的顶级阵营。</p><p>Brand Finance 是目前全球唯一品牌价值评估国际标准 ISO10668、品牌评价标准 ISO20671 的核心制定者，也是全球唯一的独立第三方品牌价值评估与咨询机构。该机构通过评估品牌强度指数、行业特许费率、企业预计营业收入等数据确定品牌价值，其专业性和独立性受到国际广泛认可，是评判全球品牌价值的重要依据。</p><p>Brand Finance 使用 " 授权节省法 " ( royalty relief ) 计算品牌价值，即测算未来授权使用这样一个品牌所应该支付的费用。</p><p>就在这个 7 月，第十九届 " 世界品牌大会 " 发布了 2022 年《中国 500 最具价值品牌》分析报告。茅台以 3660.28 亿元 ( 人民币 ) 的品牌价值位列第 15 位，在 84 个食品饮料品牌中排名第一。</p><p></p><div class="img_box" id="id_imagebox_2"
