<p>中新经纬 5 月 27 日电 ( 常涛 马静 魏薇 ) 教辅教材，一直被认为是出版行业的一座金山。</p><p>近日，成立于 1950 年的人民教育出版社 ( 下称人教社 ) 因教材插图 " 丑 "，引发大量关注，频上热搜。在此背后，人教社有怎样的生意版图？</p><p><strong>产品江山</strong></p><p>官网显示，人教社的产品覆盖教材、教辅、图书、期刊、电子音像、数字产品等六大板块。其中教材和教辅两类基本覆盖了一个学子从幼儿园到大学毕业的每一阶段需要，从必修教材到选修课本，从同步练习到课外素养阅读。中职和幼师亦包含在其中。图书业务则主要涉及教育用书、考试用书、工具书、大众读物等。期刊方面则主办了小学语文在内的 7 类刊物。</p><p><strong>记忆中随附在课本背后的光盘也是人教社的电子音像产品之一。</strong>官网显示，各个电子音像产品价格不一，差异不小。<strong>最便宜的是幼儿园教师参考书磁带，标价 6.5 元，小学数学二年级上册的教学录像光盘标价则为 96 元</strong>。</p><p>值得关注的是，人教社也加入了近年来各教育和硬件巨头扎堆进入的学生平板电脑 ( Pad ) 业务。</p><p>官网显示，<strong>人教 Pad 是人教社为中小学师生设计的智能教学终端，预装正版人教数字教材，分为两种配置，依据质保期限不同，定价不同，价格区间在 3680 元— 4380 元之间</strong>。人教社在 2016 年曾披露，第二代人教 Pad 已服务珠海市 50 余所学校。</p><p>与人教 Pad 紧密相关的人教数字教材目前则已在广东、广西、上海、天津、河南、宁夏等 20 个省市自治区实现规模化应用和实验探索，覆盖师生 3000 余万。</p><p>中新经纬记者 27 日以个人消费者身份致电人教社官方客服得知，目前人教 Pad 不向个人消费者出售，均为学校统一采购，采购价以官网披露的价格为准，不存在 " 量大从优 " 之类的优惠。</p><p>人教社数字产品板块中的平台服务近年也为人教社带来不少创收。Wind 显示，2021 年至今，人教社共中标内蒙古自治区教育厅及甘肃省教育厅的 3 个教师培训计划，累计成交金额为 523.6 万元，其中仅内蒙古自治区教育厅民族学校教师培训项目成交金额就为 398.8 万元。据人教社官网披露，目前其教师网络培训和服务平台用户目前已覆盖全国，拥有注册用户 230 余万人。</p><p><strong>投资版图</strong></p><p>官网显示，人教社成立于 1950 年 12 月 1 日，是教育部所属的一家大型专业出版社，致力基础教育教材和其他各级各类教材及教育图书的研究、编写、出版和发行。目前，人教社已成为国内综合实力最为雄厚的单体出版社。2010 年 12 月和 2011 年 3 月，中国教育出版传媒集团有限公司和中国教育出版传媒股份有限公司相继成立，人教社成为其核心成员单位。</p><p>天眼查 APP 显示，人教社对外共投资了 13 家公司，但其中 7 家公司已注销。</p><p>目前，在人教社的投资版图中，仅剩下 6 家存续公司，涵盖了出版物印刷发行、电子音像出版物、数字出版等，6 家公司分别为人教数字出版有限公司 ( 投资比例 60% ) 、人教教材中心有限责任公司 ( 投资比例 100% ) 、人民教育电子音像出版社有限公司 ( 投资比例 100% ) 、人教文化交流中心有限公司 ( 投资比例 100% ) 、陕西西北人教玉成文化传媒有限公司 ( 投资比例 20% ) 和北京文硕实业股份有限公司 ( 投资比例未披露 ) 。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div 家公司投资金额看，<strong>前四位分别为人民教育电子音像出版社 ( 投资数额 5.09 亿元 ) ，人教数字出版有限公司 ( 投资数额为 1.35 亿元 ) 和人教教材中心有限责任公司 ( 投资数额为 1.15 亿元 ) 和人教文化交流中心有限公司 ( 投资数额 5000 万元 ) 。</strong></p><p>按照公司成立时间梳理，成立时间最早的是人教教材中心有限责任公司，成立于 1998 年 10 月，中国教育新闻网曾报道，人教教材中心有限责任公司承担了人教社除中小学教材以外的纸介质图书的发行及储运任务。</p><p>1999 年，人教社先后投资成立两家公司，分别为人教文化交流中心有限公司和人民教育电子音像出版社有限公司。前者成立于 1999 年 7 月，其经营范围包括书刊的批发；计算机软件开发；计算机硬件设备、网络工程设计、系统集成、网站建设、网页设计、电脑设计；纸张、印刷材料加工、销售等。</p><p>后者是人教社投资超 5 亿元，也是其投资金额最高的公司。官网显示，主要从事幼儿教育、基础教育、职业和成人教育等方面的电子音像出版物及其数字化作品的编辑、制作、出版发行业务。</p><p>在布局了电子音像出版物和纸质图书发行之后，2012 年 5 月 23 日，人教数字出版有限公司正式成立。官网显示，该公司是人民教育出版社为加快推进数字化转型升级而成立的专职、专业、专注的教育数字出版公司，注册资本达 2.25 亿元。</p><p>除了上述 4 家投资金额较高的公司外，人教社还投资了 2 家公司。一家是出现在人教社官网的陕西西北人教玉成文化传媒有限公司，人教社对其投资了 400 万元。据中华读书报 2012 年的一篇报道，彼时人教社和陕西人民社决定联手重组陕西西北人教图书发行有限公司。据透露，新公司人教玉成将围绕人教版教材和教学资源产品，在教材的推广模式上实现从单一代理推广向合作推广方式的转变，在教学资源产品的推广上实现本土化。</p><p>最后一家为北京文硕实业股份有限公司，天眼查 APP 显示，该公司成立于 1994 年 6 月，注册资本为 5000 万元，经营范围为人教社出版的中小学教材、图书的供型和租型印刷等。但人教社对其持股比例未披露，该公司也并未出现在人教社官网和公开媒体报道中。</p><p><strong>发行渠道</strong></p><p>据人教社第 32 次全国中小学教材工作会议上披露，<strong>2020 年人教社和各地印刷发行单位全年共发行中小学教材 38.87 亿册，其中，国家统编三科教材 9.25 亿册。</strong>其中，新华书店系统是人教社教材的主要发行渠道。</p><p>据一位省级新华书店的内部人士向中新经纬介绍，人教社的教材是通过当地出版单位租型后，再由新华书店系统发行的。</p><p>据了解，租型是指出版单位从其他出版单位租入型版自己印制、发行出版物，并按出版定价的一定比例向出租单位支付专有出版权再许可权使用费。</p><p>" 就好比省级出版单位把人教社的 ( 教材版权 ) 租过来，每个省都有类似这样的单位。一般来说人教社的教材在一个省只有一个租型单位，也只有新华书店能发行，全省的中小学生只有通过这个渠道购买。" 该内部人士说。</p><p>江苏凤凰传媒 ( 江苏凤凰新华书店为其子公司 ) 在 2019 年一份报告中提到，当年公司全面恢复了与人教社中断了十年之久的教材租型、教辅代理协议，成为了人教社在江苏市场唯一的代理单位。</p><p>中新经纬记者注意到，人教社与新华书店渊源颇深。</p><p>据人教社官网 2011 年一篇报道介绍，1951 年 3 月 26 日，人教社与新华书店总店联合召开了 " 第一届全国教科书出版会议 " 和 " 课本发行会议 "，确定了课本由人教社编辑出版、各地印造，并由新华书店统一发行，" 信用预定，预印预发，先远后近，先山区后平川 "。从此经过人教社与各地新华书店的密切协作达成 " 课前到书人手一册 " 的目标，历经 60 年不变。</p><p>据人教社官网 2015 年刊发的《讲述新中国教材出版背后的故事》一文提到，人教社教材在送到学生手里，要经过了提前征订、按需印刷，人教供型、地方租印、新华书店配送等流程。 ( 中新经纬 APP ) </p><div id="recommend_bottom"></div><div id="article_bottom"></div>
