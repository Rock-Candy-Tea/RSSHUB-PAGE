郎朗摄<p></p><p><strong>抓住退耕还林这个机会！</strong></p><p>在乌蒙腹地深处，夏无酷暑，冬无严寒的六盘水，因独特的气候特征而被命名为 " 中国凉都 "。</p><p>山峦起伏、耕地零碎，" 高、少、碎、瘦 " 可谓是其农业生产条件的真实写照，一度使六盘水农业成为偏于一隅的 " 冷门 "。但是，当地适合的纬度、奇妙的温度、适宜的湿度、特定的高度，却为其他作物提供了机会。</p><p>凉都有三宝：茶叶，刺梨，猕猴桃。其中，刺梨被称为 " 维 C 之王 "，因其喜人的经济价值和生态效益，成为兼顾经济发展和环境保护的典型案例。</p><p>在六盘水市水城区野钟乡，刺梨 " 千斤园 " 示范基地中，一个个青涩的刺梨正挂在枝头，摸上去有些扎手，要等到 8-9 月，这些青果才会变成金黄色的成熟刺梨。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" 贵阳日报供图<p></p><p>野钟乡野钟村村民唐召良正在地里修剪刺梨多余的枝叶。这位有着 43 年从医经验的乡村医生，在退休之后，放下听诊器，拿起了修枝剪。" 以前种玉米，2015 年开始种刺梨，现在我有 15 亩地，种刺梨的收入是种玉米的 3 倍！"</p><p>恐怕谁也不曾想到，刺梨——这种曾被视为到处可见的野果子，也有翻身为自己正名的一天。而刺梨抓住的这个机会，叫做 " 退耕还林 供图<p></p><p><strong>逆袭：野果变身凉都宝</strong></p><p>自被设立为 " 退耕还林 " 试点示范县之后，水城区先后进行了两轮退耕还林行动。</p><p>原本，当地人普遍种植玉米，投入精力多，收益却很小，而且由于需要不断翻土地，加之雨水的重刷，使得当地水土流失，石漠化严重。所以，" 退耕还林 " 的政策一出来，玉米便首先退出了耕作作物的行列。</p><p>不种玉米了，村民们靠什么吃饭？一场 " 水城好作物 " 的评选拉开了帷幕。</p><p>只不过谁也没想到，最后是没有种植背景、随处可见、口感微酸的 " 野果子 " 刺梨拔得头筹。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div 评审 " 们认为，水城区野钟乡的地理条件适合刺梨生长，种植刺梨也不涉及翻地，且无需投入太多精力，它的生长也不怕有其他绿植争夺养分；同时，刺梨的维 C 含量极高，适合做成果酒、果酱、罐头、饮料等多种产品，经济价值极高。</p><p>" 从 2015 年开始引入刺梨种植以后，我们已经成功带动了野钟村几十户贫困户脱贫，每户的年收入都能达到万元以上。" 野钟乡林业站站长唐忠广介绍。</p><p>目前，水城区全区刺梨品种主要以贵农 5 号为主，种植面积 40 万亩，涉及 25 个乡 ( 镇、街道 ) ，96 个村 ( 居 ) ，覆盖农户 3.7 万户 16.2 万人 ( 其中，建档立卡户 1.1 万户 4.3 万人 ) 。2020 年采收、加工鲜果 5105 吨，压榨原浆 3216 吨，实现产值 1.6 亿元。</p><p>更令人惊喜的是，种植刺梨之后，该地区的土壤质量大幅提高，土壤及植被的厚度也大大增加，这有效避免了以往此地水土流失的情况。</p><p>既能保持水土、绿化环境，又能创造经济价值，乘着 " 退耕还林 " 的春风，刺梨从 " 野孩子 " 变成了农户的 " 掌中宝 "。 ( 完 ) </p><p>责任编辑：姜雨薇</p><div
