<p>iPhone 上那道大刘海，一直是众人吐槽的重点对象。吐槽了将近四年的刘海屏，有的人还在继续吐槽，有的人已经渐渐习惯。但要说拿掉刘海，让 iPhone 屏幕再次完整，绝对是广大果粉的共同心愿。</p> 
<p>纵使安卓厂商纷纷拥抱了屏幕指纹识别，玩起了屏下摄像头，苹果依然坚持采用刘海屏与 Face ID 这对组合。无刘海的真全面屏 iPhone，难道只存在于想象之中？</p> 
<p>这可未必。</p> 
<h2 label="一级标题" style><strong>屏下 Touch ID 终到来</strong></h2> 
<p>本周日，彭博社记者马克·古尔曼（Mark Gurman）在其最新的《Power On》专栏中指出，苹果曾进行测试，在 iPhone 上应用屏幕指纹 Touch ID 技术。</p> 
<p>屏幕指纹 Touch ID 的终极目标是什么？当然是实现无刘海真全面屏了。iPhone 的大刘海之所以拿不掉，是因为刘海里面藏着大量的元器件。如果 iPhone 从 Face ID 转向屏幕指纹 Touch ID，就可以省下一大片空间，为打孔屏甚至屏下摄像头真全面屏铺路。</p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20210823/v2_ee66e1f949844098aede6d2b08f55aaa_img_000" referrerpolicy="no-referrer"></p> 
<p>无独有偶，爆料者 MauriQHD 今年初就表示，苹果正在测试一款无刘海、四面边框更宽的 iPhone 13 原型机。种种蛛丝马迹暗示着，苹果拿掉刘海屏的计划，已在实施。</p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20210823/v2_37400da186694e6fa2bb0f7e7f34147f_img_000" referrerpolicy="no-referrer"></p> 
<p>稍有遗憾的是，古尔曼和MauriQHD 都倾向于，iPhone 13 不会采用这一设计，今年的 iPhone 还是熟悉的刘海屏。</p> 
<p>天风国际分析师郭明錤早先曾预测，2022 年的 iPhone 14 系列中，至少高端型号会弃用刘海屏，转向打孔屏设计；<strong>如果顺利的话，屏下指纹、无打孔真全面屏 iPhone 预计会在 2023 年到来。</strong></p> 
<h2 label="一级标题" style><strong>未来属于屏下 Face ID ？</strong></h2> 
<p>屏下摄像头配屏幕指纹 Touch ID，听起来是最合理、最接近现实的真全面屏设计。只是由 Touch ID 到 Face ID 然后又回到 Touch ID，供应链恐怕会受到不小的影响，开发者和用户也免不了要受一番折腾。</p> 
<p>对此，古尔曼表示，<strong>他坚信苹果已经为高端 iPhone 押宝 Face ID，其长期目标是实现屏下 Face ID。</strong></p> 
<p>为此他提出了两种可能：</p> 
<ul style="list-style-type: disc;" class=" list-paddingleft-2"> 
 <li><p>高端型号的 iPhone 采用屏下 Face ID、真全面屏，低端型号继续采用刘海全面屏 Face ID；</p></li> 
 <li><p>高端型号的 iPhone 采用屏下 Face ID、真全面屏，低端型号采用屏幕指纹 Touch ID。</p></li> 
</ul> 
<p>两种假设言而有理，开发者和用户可以尽可能少地做出改变，只不过要让屏下 Face ID 成为现实，苹果和诸多供应链厂商得打磨好一阵子了。</p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20210823/v2_d41a87a642074dc98c41fede65fa9c28_img_000" referrerpolicy="no-referrer"></p> 
<p>追本溯源，屏幕指纹与屏下摄像头本是同根生，光学屏幕指纹从原理上说也是一种屏下摄像头技术，只不过藏身屏幕下方的传感器负责捕捉指纹图像，成像质量没有自拍要求那么苛刻。</p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20210823/v2_627350f9cd7c413584478facbc417ccf_img_000" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>（光学屏幕指纹原理图）</p> 
<p>为了保证屏下摄像头的效果，首先要确保摄像头区域的透光，今年安卓厂商这边普遍采用了缩小单个像素点的方式，再利用缩小后像素之间的间隙来透光。</p> 
<p>如此成像效果差强人意，已经足以应对日常使用，摄像头在非工作状态下也能很好地隐形。只不过，当前的方案并不能达成 100% 的透光，往往需要通过软件和算法的优化改善成像，同时该方案应用于高分辨率屏幕的效果还有改进空间。</p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20210823/v2_4f68d55e5b744a4a9820b689867d5fbd_img_000" referrerpolicy="no-referrer"></p> 
<p>众所周知，苹果不喜欢将就。如果苹果为了屏下摄像头，舍弃 iPhone 招牌的高分辨率视网膜屏幕，免不了招来非议。以苹果的风格，只有屏下摄像头方案足够成熟，能完美搭配高分辨率屏幕并装下 Face ID 组件之时，才会把无刘海真全面屏配 Face ID 的理想 iPhone 带到现实。</p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20210823/v2_5189ecd796eb4972b18c4ca32aff010c_img_000" referrerpolicy="no-referrer"></p> 
<p>问题来了，你会准备钱包等这款真全面屏 iPhone 吗？屏下 Face ID 和屏下 Touch ID，哪个才是你的理想之选呢？</p>
