<p>36氪出海获悉，跨境电商数字 API 平台 Dowsure 日前宣布已完成近2000万美元的 B 轮融资。本轮融资由一家未公开的全球顶级战略投资者领投，创业工场（VenturesLab）、前海一方，以及现有投资方 Plug & Play China 参投。据悉，本轮新筹集的资金将主要用于拓展国际市场以服务更多海外本土卖家、加深 eBay、Shopee 等多平台的布局、以及研发人才的招募。</p> 
<p>近年来跨境电商发展迅猛，大批跨境电商卖家和品牌通过入驻亚⻢逊等电商平台，或自建独立站的方式将产品卖至全世界。这其中就包含了一众想要在海外市场“<a class="project-link" data-id="510938" data-name="掘金" data-logo="https://img.36krcdn.com/20200929/v2_970f289b354d49818f84dace6d796aa8_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/510938" target="_blank">掘金</a>”的中国卖家和品牌。据外媒统计，在跨境电商备受关注的2021年，亚⻢逊平台上的卖家超过600万，其中超过一半来自中国。</p> 
<p>对于跨境电商卖家，充足的资金至关重要。然而，对于中小型卖家来说，“缺钱”往往是他们真正的痛点。许多中小企业卖家都没有足够的固定资产可以用于抵押，因此银行很难依托于抵押等方式为跨境电商卖家们提供资金支持。而解决这些卖家资金需求一直是全球尤其是中国金融机构和银行的目标。</p> 
<p>成立于2016年的 Dowsure 作为专注于跨境电商行业的科技公司，通过研发数字 API 平台，搭建覆盖全球跨境电商的基础网络，并运用数字资产价值评估模型，向各方进行科技赋能。客户包括银行、保险公司、卡组织、风投机构，以及产业合作伙伴如广告、供应链和物流服务商等。据 Dowsure 战略负责人 Jacky 介绍，2021年 Dowsure 已经与多方达成合作，包括互联网巨头如 Amazon、eBay、Shopee、Facebook 等，大型银行如<a class="project-link" data-id="25115" data-name="中国银行" data-logo="https://img.36krcdn.com/20210806/v2_cf871746dbb84106ac4c5ceddcd16580_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/25115" target="_blank">中国银行</a>、<a class="project-link" data-id="25376" data-name="浦发银行" data-logo="https://img.36krcdn.com/20210806/v2_623a567df055498c9c50af7f009b0f32_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/25376" target="_blank">浦发银行</a>、<a class="project-link" data-id="25107" data-name="广发银行" data-logo="https://img.36krcdn.com/20210806/v2_1653709abdf74771bed5872010418d86_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/25107" target="_blank">广发银行</a>、<a class="project-link" data-id="3968507" data-name="汇丰" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3968507" target="_blank">汇丰</a>银行等，保险公司如人保财险、平安财险、大地财险等。</p> 
<p>谈及业务模式，Jacky 介绍称，Dowsure 通过其跨境电商云平台提供 API 及 SaaS 服务。在金融领域，帮助金融机构开发跨境电商金融产品，助力金融机构建立服务跨境电商中小卖家的能力。例如，亚马逊卖家可以通过卖家贷款推荐计划，向银行提交申请，由银行对卖家进行独立风控后，向卖家提供资金。卖家可凭借其数字资产价值，通过 Dowsure 科技赋能，获得银行信贷服务。同时，Dowsure 也向产业合作伙伴提供服务，帮助广告、供应链<a class="project-link" data-id="651649" data-name="及物" data-logo="https://img.36krcdn.com/20210814/v2_23dea533503d490da34e619048601806_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/651649" target="_blank">及物</a>流等服务商提高其业务质量，建立更强的客户获取及服务能力。</p> 
<p>关于 Dowsure 的盈利模式，据悉，Dowsure 的主要收入来源于技术服务费，通过 API 的调用次数来收取费用。另外也提供 SaaS 服务，如全套跨境电商云服务平台。</p> 
<p>谈及未来的发展，Jacky 表示，除了进一步拓展海外市场，与更多合作伙伴建立 API 对接外，Dowsure 也将进一步拓展更多数字资产应用场景，未来将不仅服务于跨境电商平台卖家，还将为更多数字经济的中小企业参与者提供服务。</p>
