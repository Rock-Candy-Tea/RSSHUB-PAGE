<p>这个在 Windows 10 中经常被归纳到「你不知道的 N 个技巧」，如今却在 Windows 11 里「C 位出道」的功能，到底要怎么用呢？</p>

<h2 label="一级标题" style>来，认识下虚拟桌面！</h2> 
<p>虽然本文是通过 Windows 11 演示的，但功能与 Windows 10 并无太大差异，我们也不建议现在升级到 Windows 11。所以，即使你正在使用 Windows 10 系统，也可以放心使用本文中提到的技巧和快捷键，来提高工作、学习效率。</p> 
<p>在 Windows 上，虚拟桌面被集成在了任务视图里，只需点击图标进入多任务视图界面，你就会看到「新建桌面」的选项，点击它，一个全新的桌面变成功建立了。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_d1d21004880a4e85940c34122e7608ac_img_000" data-img-size-val="1000,667" referrerpolicy="no-referrer"></p> 
<p>进入新桌面，你会发现刚刚打开的所有软件都「消失」了，其实它们只是被留在了之前的桌面里。你可以在新的桌面开启其他软件，甚至更换壁纸，这些操作都不会影响第一个桌面，当然，通过点击任务视图按钮，你不仅能在多个桌面间切换，还能把软件在不同的桌面间移动。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_dd238a725dbd49a280660d35478349c8_img_000" data-img-size-val="640,361" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲<span style="letter-spacing: 0px;">图片来自：Windows 11 宣传视频</span></p> 
<p>当然，鼠标点击是最基础的操作，真正的电脑「大神」是不屑于使用鼠标的。我们可以使用「Win+Ctrl+D」新建虚拟桌面，使用「Win+Ctrl+左右方向键」在多个虚拟桌面间切换，按下「Win+Ctrl+F4」关闭当前桌面，不用担心被关闭桌面上的软件，它们会和你<a class="project-link" data-id="81906" data-name="一起" data-logo="https://img.36krcdn.com/20210709/v2_647b9860d6f7437caf1be2501d37698a_img_000" data-refer-type="1" href="https://www.36dianping.com/space/4772100123" target="_blank">一起</a>回到左侧的桌面里。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_ce39771282cd4df49343538c5df54819_img_000" data-img-size-val="640,360" referrerpolicy="no-referrer"></p> 
<p>当然，虚拟桌面的操作不止有这几个快捷键，你也可以按下「Win+Tab」，再通过「Tab+左右方向键」来选择桌面，但这样操作的效率并没有提高多少，只能算是多一种选择罢了。</p> 
<p>如果你使用笔记本电脑，在触控板上三<a class="project-link" data-id="70487" data-name="指上" data-logo="https://img.36krcdn.com/20200729/v2_5f032bc1fd5a410ca804e019bc071e7e_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/70487" target="_blank">指上</a>滑就能进入任务视图，四指左右滑动则是切换不同桌面。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_eab5402927c74af7876cbbb72d227b71_img_000" data-img-size-val="640,1137" referrerpolicy="no-referrer"></p> 
<p>至此，你就学会了 Windows 虚拟桌面的所有操作，但提到虚拟桌面，就不得不提 macOS，那么，Windows 的虚拟桌面，在体验上可以与 macOS 一较高下么？</p> 
<h2 label="一级标题" style>Windows VS macOS，谁的「桌面」更好用</h2> 
<p>Windows 和 macOS，来自全球两大科技公司的桌面操作系统，经常被放在一起比较。今天，我们就来对比一下两者的桌面到底是谁更好用。</p> 
<p>首先，在同屏多任务这一操作上，macOS 相比 Windows 差的比较多。Windows 可以通过拖动窗口到屏幕的角落、侧边实现全屏、双分屏、四分屏等操作，也可以使用快捷键（Win+方向键）调整软件的位置，而 macOS 能做到的，只有双击软件顶部全屏而已。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_82d1ad833f1b4e84b75276ae12683e6c_img_000" data-img-size-val="1000,742" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲<span style="letter-spacing: 0px;">Windows 11 还加入了快速分屏的功能</span></p> 
<p>不过，macOS 用户也不用太羡慕，虽然系统原生缺失，但依然可以使用 Magnet 等软件来实现与 Windows 类似的分屏功能。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_53d9cf620504462b97363b352284795b_img_000" data-img-size-val="640,360" referrerpolicy="no-referrer"></p> 
<p>在同屏多任务上比不过 Windows 的 macOS，到虚拟桌面上则打了一个漂亮的翻身仗。</p> 
<p>将鼠标指针放在 macOS 软件<a class="project-link" data-id="518404" data-name="左上角" data-logo="https://img.36krcdn.com/20201106/v2_8f4b0574248840509172e7542ff37978_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/518404" target="_blank">左上角</a>的绿色圆点上，你会看到「进入全屏幕」的选项，点击它，软件会开启一个「虚拟桌面」，隐藏顶部菜单栏和程序坞，实现沉浸式的全屏幕体验。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_7bffdeff33a04a5dbd96c48d953e1298_img_000" data-img-size-val="640,360" referrerpolicy="no-referrer"></p> 
<p>而在 Windows 上，我们是无法实现没有系统界面、没有软件顶部菜单的零干扰全屏体验的。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_14989356ee384fbc914c22f9bdec2df7_img_000" data-img-size-val="1000,610" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲<span style="letter-spacing: 0px;">只能通过 Word「专注模式」等软件功能实现</span></p> 
<p>不过，可能是为了兼容不同配置的电脑，可能是因为 Windows 11 还在测试阶段（但 macOS Monterey 也在测试阶段），在虚拟桌面切换时总偶有卡顿，就算没有卡顿，在动画上也没有 macOS 流畅。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_21c48df0c1514dd6beaece4fb92b5e3c_img_000" data-img-size-val="1079,203" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲<span style="letter-spacing: 0px;">Windows 11 虚拟桌面添加与切换</span></p> 
<p><img src="https://img.36krcdn.com/20210712/v2_af1060f07ec0450a991c1a07f5eeef8d_img_000" data-img-size-val="1076,171" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲<span style="letter-spacing: 0px;">macOS Monterey 虚拟桌面添加与切换</span></p> 
<p>总体来说，Windows 虽然少了全屏单软件的沉浸模式，但强大的单屏多任务的功能，依然让它拥有了超越 macOS 的体验。不过，macOS 的不足可以靠第三方软件弥补，而 Windows 不太流畅的系统动画，让它在体验上显得不那么高级。</p> 
<h2 label="一级标题" style>「摸鱼+工作」？这不是正确用法</h2> 
<p>说到虚拟桌面，许多小伙伴都会露出「你懂得」的表情：老板没来时玩游戏，老板到了身后快速切换到工作桌面，虚拟桌面可是摸鱼利器！的确，虚拟桌面可以实现上述操作，但这样未免也太大材小用。在搞清楚为什么要用虚拟桌面之前，我们得先清楚不用虚拟桌面会有哪些不方便的地方。</p> 
<p>在 Windows 中，我们可以使用「Alt+Tab」在多个软件之间切换，但窗口太多时，切换的快捷键就显得不那么快捷了。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_e2e72f7eca6842e29734ef643d200be4_img_000" data-img-size-val="1000,667" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲<span style="letter-spacing: 0px;">窗口太多，就没有效率可言了</span></p> 
<p>为了解决这一问题，许多人会选择增加一块显示屏，而虚拟桌面所起到的作用，和为电脑增加外接显示屏是一样的。你可以把软件在多个「显示屏」之间拖动，也可以把不同种类、不同用途的软件分布到不同的「显示屏」上，让一台电脑拥有多个办公学习环境，从而更高效的工作。</p> 
<p>与外接显示屏不同的，就是要把看向另一个屏幕的头部动作，改成按下「Win+Ctrl+方向键」的手上动作（在 macOS 上就是 Ctrl+方向键啦）。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_6d723266508848fa930274c803707a2f_img_000" data-img-size-val="1000,381" referrerpolicy="no-referrer"></p> 
<p>所以，虚拟桌面的意义，在于更高效的完成一项任务，而不是三心二意的同时进行多个任务。</p> 
<p>完成一项任务时，我们一般会有三方面的需求：创作输出、资料参考、沟通交流。对应需求，我们也可以对应设置三个虚拟桌面。</p> 
<ul class=" list-paddingleft-2"> 
 <li><p>创作桌面：Word、PhotoShop、Visual Studio 等创作类软件</p></li> 
 <li><p>资料桌面：浏览器、待办、PDF 阅读器等阅读日程类软件</p></li> 
 <li><p>交流桌面：微信、邮件、QQ、钉钉等聊天沟通类软件</p></li> 
</ul> 
<p>这样，你就可以在完成一项工作时，让查看文档、制图写作、沟通交流这三个需求互不干扰的同时，高效协同。即不会因为闪动的聊天软件影响创作，也不会因为同一屏幕放入太多软件而导致频繁切换窗口。</p> 
<p><img src="https://img.36krcdn.com/20210712/v2_39c9b8c67f01402dbb69d84ac71387af_img_000" data-img-size-val="1000,667" referrerpolicy="no-referrer"></p> 
<p>当然了，工作并不是生活的全部，你也可以一个屏幕全屏玩游戏，另外一个屏幕搜索攻略；一个屏幕看直播，另一个屏幕和同好交流。当一个桌面只有与当前工作相关的窗口、看不到无关内容时，你获得的不只是整洁的桌面，还有专注的状态。总之，更高效的完成一项任务，才是虚拟桌面的正确打开方式。</p>
