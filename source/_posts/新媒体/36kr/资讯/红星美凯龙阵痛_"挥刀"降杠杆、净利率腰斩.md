<p style="text-align: start;white-space: normal;text-size-adjust: auto;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);vertical-align: inherit;margin-left: 15px;margin-right: 15px;line-height: normal;margin-bottom: 5px;"><strong><span style="font-size: 12px;color: #A5A5A5;"></span></strong></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">“美好的生活需要诗和远方，更需要沙发和床”，这是<a class="project-link" data-id="1679686033462017" data-name="红星" data-logo="https://img.36krcdn.com/20220401/v2_da5146bde2c04429bd5b525d0dbc201a_img_000" data-refer-type="1" href="https://36kr.com/project/1679686033462017" target="_blank">红星</a><a class="project-link" data-id="1679696732672774" data-name="美凯龙" data-logo="https://img.36krcdn.com/20220401/v2_da7e807c2c45474fad83e73166749395_img_000" data-refer-type="1" href="https://36kr.com/project/1679696732672774" target="_blank">美凯龙</a>（601828.SH、01528.HK）的董事长车建新曾对美好生活的解读，红星美凯龙也在沙发和床的家居领域占据头筹。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">据沙利文数据显示，2021年红星美凯龙零售额占家居连锁商场的份额继续提升至17.5%，持续位列第一，几乎是第二至第五名的总和。位列第一的红星美凯龙在2021年也交出了一份看似“亮眼”的“成绩单”。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">3月30日盘后，红星美凯龙发布年报，据年报显示，2021年其实现营业收入155.13亿元，同比增长8.97%，实现净利润21.89亿元，同比增长6.05%。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">但是投资者对这份年报似乎并不买单，年报发布次日，红星美凯龙股价微跌0.73%，截至4月20日，红星美凯龙股价报收7.22元/股，对应市值314.41亿元，对比2021年高点13.80元/股，接近腰斩。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><img class="rich_pages wxw-img js_insertlocalimg" data-cropselx1="0" data-cropselx2="548" data-cropsely1="0" data-cropsely2="256" data-ratio="0.4675925925925926" data-s="300,640" data-type="jpeg" data-w="1080" style="width: 548px;height: 256px;" src="https://img.36krcdn.com/20220420/v2_fa51d2bd5da748238f06f6270816233e_img_000" referrerpolicy="no-referrer"></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;">在「不二研究」看来，红星美凯龙尽管2021年年报数据看似不错，但是对比疫情前仍承压严重，盈利能力下滑严重；</span><span style="font-size: 15px;font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;">曾经的“重资产、重杠杠”模式失灵，债务问题凸显，尽管近年采取“轻资产、重运营、降杠杠”战略，取得一定成效，但短期内掣肘仍存，意图布局高端家电，增加新的增长极，但也前途未卜。</span><span style="font-size: 15px;font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">世界上唯一不变的是变化，变革期的红星美凯龙阵痛还在持续。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;margin-bottom: 5px;text-align: center;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.12719298245614036" data-type="png" data-w="1140" data-width="80%" style="vertical-align: inherit;width: 591.188px;" width="80%" src="https://img.36krcdn.com/20220420/v2_493ada7a93f140cfa67106a099c9dcff_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;text-align: center;"><span style="font-size: 18px;color: #99CCFF;"><strong>业绩受困，盈利能力向下</strong></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">1986年，红星美凯龙的创始人车建新，以从亲戚手中借的600元创办了一家家具作坊，于2000年开设第一家红星美凯龙家具卖场，随后红星美凯龙便开始加速扩张，分别于2015年、2018年登陆港股、A股，成为国内首家家居零售行业A+H股两地上市公司。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">但是，疫情后的红星美凯龙日子并不好过，据其财报显示2019-2021年红星美凯龙分别取得营收164.69亿元，142.36亿元，155.13亿元，同比增长15.66%，-13.56%，8.97%。同期，分别取得净利润46.86亿元，20.64亿元，21.89亿元，同比</span><span style="caret-color: #000000;color: #000000;font-family: -webkit-standard;text-align: start;text-size-adjust: auto;font-size: 15px;">增长-0.41%，-55.95%，6.05%。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;margin-bottom: 5px;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.6851063829787234" data-type="jpeg" data-w="940" style="box-sizing: border-box;vertical-align: inherit;width: 739px;" src="https://img.36krcdn.com/20220420/v2_0b65b560692e4ad4a1867d92c2a0a161_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">仅看2021年的数据，红星美凯龙取得了营收和净利润的双增长，但是这样的增长却是建立在2020年的低基数下，若将2021年数据与2019年比，则会看到不一样的情况。据计算，红星美凯龙2021年营收同比2019年全年下降5.80%，净利润则同比下降53.29%。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">如此看来，红星美凯龙的业绩仍然处于困境，并且净利润的下降幅度远高于营收的下降幅度。显然，红星美凯龙的盈利能力出现了一定问题。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">据红星美凯龙财报显示，2019-2021年其毛利率分别为65.17%，61.51%，61.67%；同期，净利率为28.45%，14.50%，14.11%。从盈利能力看，其毛利率并未出现大幅波动，净利率于2020年近乎腰斩。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;margin-bottom: 5px;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.6851063829787234" data-type="jpeg" data-w="940" style="box-sizing: border-box;vertical-align: inherit;width: 739px;" src="https://img.36krcdn.com/20220420/v2_c5fa7f2e98b944ebb325ea5ba1c91e46_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">「不二研究」认为，在红星美凯龙盈利能力下降的背后，是其“重资产，高杠杆”模式的拖累，红星美凯龙的债务问题凸显，为此，红星美凯龙不得不选择转型。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;margin-bottom: 5px;text-align: center;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.12719298245614036" data-type="png" data-w="1140" data-width="80%" style="vertical-align: inherit;width: 591.188px;" width="80%" src="https://img.36krcdn.com/20220420/v2_568286e0f81f47e9bc0c047364622bd9_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;text-align: center;"><span style="color: #99CCFF;"><strong><span style="font-size: 18px;">轻资产降杠杆，成效几何？ </span></strong></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">红星美凯龙自2010年以来便尊奉联动拿地策略，由于家居商城的招商引资作用受政府欢迎，很适合在商城附近拿下商住用地。只要手里握着地，就不愁借融不到钱。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">红星美凯龙也通过此方式顺理成章的大力发展其“重资产、高杠杆”的商业模式。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">但是，水能载舟，亦能覆舟，伴随着近年房地产市场的逐步稳定，外加“三道红线”的限制，高杠杆的拿地模式被终结，红星美凯龙盈利能力下降和债务问题开始凸显。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">据其年报显示，截至2021年末，红星美凯龙投资性房地产价值高达955.75亿元，在全部A股公司中排名第五，前四者分别为中国建筑、中国平安、新城控股、<a class="project-link" data-id="1678455493653506" data-name="招商蛇口" data-logo="https://img.36krcdn.com/20220331/v2_415213b969614ed29508d8bc43c61f2c_img_000" data-refer-type="1" href="https://36kr.com/project/1678455493653506" target="_blank">招商蛇口</a>。红星美凯龙并未将此科目加入固定资产，因此每期末都会对其作出评估，以此判定利润增减。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">据红星美凯龙财报显示，2019-2021年，投资性房地产公允价值变动收益分别为16.01亿元，4.97亿元，3.74亿元，分别占其归母净利润比例为35.74%，28.71%，13.36%。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;margin-bottom: 5px;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.6851063829787234" data-type="jpeg" data-w="940" style="box-sizing: border-box;vertical-align: inherit;width: 739px;" src="https://img.36krcdn.com/20220420/v2_a208152f38d24210949d256b2832e102_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">投资性房地产公允价值所产生的利润大幅下降，也是红星美凯龙利润下滑的重要原因之一。另外，「不二研究」认为，若将此项科目还原至现实，其逻辑却经不起推敲，近两年商业地产低迷，唱空文章随处可见。3月7日，有投资者于上证e互动表</span><span style="font-size: 15px;">示曾花80万买的红星美凯龙商铺目前40万都无法出手，对其房产公允价值变动额提出质疑。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><img class="rich_pages wxw-img js_insertlocalimg" data-cropselx1="0" data-cropselx2="548" data-cropsely1="0" data-cropsely2="376" data-ratio="0.6851063829787234" data-s="300,640" data-type="jpeg" data-w="940" style="width: 549px;height: 376px;" src="https://img.36krcdn.com/20220420/v2_fc535486f4cc466c8dc6c71df4214fce_img_000" referrerpolicy="no-referrer"></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">另外，红星美凯龙的负债率也居于高位，据其财报显示，2019-2021年，红星美凯龙的资产负债率分别为59.95%，61.16%，57.44%。2021年较2020年降低3.72个百分点，负债率在高位上有所好转。红星美凯龙为了优化自身而提出的“轻资产、重运营、降杠杠”战略中的“降杠杠”取得了成效。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;margin-bottom: 5px;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.6851063829787234" data-type="jpeg" data-w="940" style="box-sizing: border-box;vertical-align: inherit;width: 739px;" src="https://img.36krcdn.com/20220420/v2_4dc777eae4e6405bae6b488ece625b1c_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">但是“轻资产、重运营”似乎收效甚微，据红星美凯龙年报显示，其商场业务可分为自营商场和委管商场两部分，自营商场是“重资产”的代表，委管商场是“轻资产”的代表。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">截至2021年末，红星美凯龙经营了95家自营商场，278家委管商场。其中，自营商场收入达80.95亿元，同比增长21.10%，<span style="font-size: 15px;letter-spacing: normal;">占总收入比例从2020年的46.94%</span><span style="caret-color: #000000;color: #000000;letter-spacing: 0px;text-align: start;text-size-adjust: auto;font-size: 15px;">增至202</span></span><span style="font-size: 15px;letter-spacing: normal;">1年<span style="caret-color: #000000;color: #000000;letter-spacing: 0px;text-align: start;text-size-adjust: auto;font-size: 15px;">年</span><span style="caret-color: #000000;color: #000000;text-align: start;text-size-adjust: auto;font-size: 15px;letter-spacing: -0.5px;">的52</span>.18%，出</span><span style="font-size: 15px;"><span style="caret-color: #000000;color: #000000;text-align: start;text-size-adjust: auto;font-size: 15px;">租</span>出租率为94.1%。而委管商场方面却表现较差，总收入为32.56亿元，同比下降9.06%，占总收入比例从2021年的25.15%下降至20.99%，出租率则由2020年的90.7%微涨至91.4%。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;margin-bottom: 5px;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.6851063829787234" data-type="jpeg" data-w="940" style="box-sizing: border-box;vertical-align: inherit;width: 739px;" src="https://img.36krcdn.com/20220420/v2_d8e8960c64f345619e1bcc71c028a563_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">「不二研究」认为，若从此层面看，红星美凯龙“轻资产、重运营、降杠杠”战略仅降杠杠一项取得了成效，红星美凯龙目前的核心仍是其自营商场，其地产商的基因短期内难以拔除，红星美凯龙还需探索第二增长曲线。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;text-align: center;margin-bottom: 5px;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.12719298245614036" data-type="png" data-w="1140" data-width="80%" style="vertical-align: inherit;width: 591.188px;" width="80%" src="https://img.36krcdn.com/20220420/v2_c94daa8e20194a8d9261a749b1e59196_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;text-align: center;"><span style="color: #99CCFF;"><strong><span style="font-size: 18px;">布局家电，如何"虎口夺食"？</span></strong></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">为了探索第二增长曲线，红星美凯龙强势杀入家电市场。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">2020年下半年，红星美凯龙在其上海总部的真北路商场三楼开除了一万方经营面积的电器区，正式发力高端电器。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">截至2021年年末，红星美凯龙已开业104家智能电器生活馆，预计在2023年将智能电器生活馆覆盖全国商场，并且红星美凯龙智能电器生活馆已经与超200家电器品牌合作，其中有30多家头部电器品牌与红星美凯龙签订了深度战略合作协议。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;text-align: center;margin-bottom: 5px;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.678" data-type="png" data-w="1000" style="vertical-align: inherit;width: 739px;" src="https://img.36krcdn.com/20220420/v2_3d24fa9687a74c77ab16ee6e77107d6d_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;text-align: center;"><span style="font-size: 12px;color: #FFFFFF;background-color: #000000;">图源:pinterest</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">家电行业真的可以成为红星美凯龙的第二增长曲线吗？</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">「不二研究」认为，红星美凯龙在全国商场、储备商业用地布局等，为其布局家电带来独特优势，但是家电行业的诸多挑战不容忽视。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">硬币的另一面，据《2021年中国家电市场报告》显示，2021年我国家电市场规模8811亿元，<a class="project-link" data-id="1678272160723973" data-name="京东" data-logo="https://img.36krcdn.com/20220331/v2_c05e240eba464dd28d00b16923addd7a_img_000" data-refer-type="1" href="https://36kr.com/project/1678272160723973" target="_blank">京东</a>、苏宁、天猫、国美四家共占据了68.6%的市场份额，红星美凯龙想要在家电市场上分一杯羹，必然会触碰到他们的“蛋糕”，巨头们不会这么容易让其抢占份额，面临的竞争挑战较大。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;margin-bottom: 5px;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.6851063829787234" data-type="jpeg" data-w="940" style="vertical-align: inherit;width: 739px;" src="https://img.36krcdn.com/20220420/v2_22a503110e0c42ee82b9cd9d539f97c9_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">与此同时，红星美凯龙的“智能电器生活馆”模式，似乎依然是重资产模式，对于目前饱受重资产模式掣肘的红星美凯龙而言，“智能电器生活馆”这种模式能不能一直走下去尚未可知。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">在「不二研究」看来，红星美凯龙布局家电市场探索第二增长曲线是一次好的尝试，但是步子迈得太大，容易“误伤”自己，若家电布局能成功在存量市场抢到“蛋糕”，对红星美凯龙而言，新的增长极将会显现，但若是不成功，红星美凯龙或将再次不堪重负。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-left: 15px;margin-right: 15px;margin-bottom: 5px;text-align: center;"><span style="font-size: 15px;"><img class="rich_pages wxw-img" data-ratio="0.12719298245614036" data-type="png" data-w="1140" data-width="80%" style="vertical-align: inherit;width: 591.188px;" width="80%" src="https://img.36krcdn.com/20220420/v2_0a6a99e592c94b4aa964edd99d4accf1_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;text-align: center;"><span style="color: #99CCFF;"><strong><span style="font-size: 18px;">家居界地产大王：变革阵痛何时休？</span></strong></span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">世界上唯一不变的就是变化，企业想要顺应时代发展，变革在所难免。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">红星美凯龙昔日奉行的“重资产、高杠杆”模式已不适用于目前，其“轻资产、重运营、降杠杠”战略目前来看步子是迈对了，但是需要时间去消化变革的阵痛，而家电布局看似能为红星美凯龙的变革阵痛带来缓冲，但若是业绩不达预期，高速的扩张或将反噬自身。</span></p> 
<p style="caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);text-align: start;white-space: normal;text-size-adjust: auto;vertical-align: inherit;line-height: 1.75em;margin-bottom: 15px;margin-left: 15px;margin-right: 15px;"><span style="font-size: 15px;">红星美凯龙，作为家居零售行业当之无愧的龙头，其优势不言而喻，变革的前路虽布满荆棘，在疫情反复、经济环境多变的当下，阵痛何时能度过尚未可知。</span></p>
