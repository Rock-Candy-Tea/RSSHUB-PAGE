<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">一位小朋友在纠缠妈妈许久后，终于坐上了VR太空椅，体验了一把“上天”的快乐。<strong>这一幕并非是科幻电影里的场景，而是部分游客</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>抢先体验“元宇宙”</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;"><strong>未来场景的一处缩影。</strong></span> 
</section> 
<p style="visibility: visible;box-sizing: border-box;padding: 0 8px;"><img class="rich_pages wxw-img" data-ratio="0.6453703703703704" data-w="1080" style="border-radius: 5px;vertical-align: middle;display: block;margin: 0 auto;width:100%;" src="https://img.36krcdn.com/20220408/v2_d149dd90427b4bfe9b84c805ac0da0b9_img_000" referrerpolicy="no-referrer"></p> 
<p style="margin: 0 8px 25px;white-space: normal;visibility: visible;line-height: 1.75em;"><span style="visibility: visible;letter-spacing: 0.5px;font-size: 12px;color: #888888;">▲VR太空舱</span></p> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">周末或者节假日时，我们常常能看见不少VR游戏区前排起了长队，年轻人和小朋友们都想体验一回VR的乐趣。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">但是，对于一些家长和年长者来说，他们认为这类VR游戏的价格稍贵，并且体验不佳。“我有点想吐。”一位家长体验几分钟后晕晕乎乎从VR太空舱走下来。</span> 
</section> 
<p style="visibility: visible;box-sizing: border-box;padding: 0 8px;"><img class="rich_pages wxw-img" data-ratio="0.8540540540540541" data-w="740" style="border-radius: 5px;vertical-align: middle;display: block;margin: 0 auto;width:100%;" src="https://img.36krcdn.com/20220408/v2_7fd59f02fd8b461886b1e8dbbedcaa3e_img_000" referrerpolicy="no-referrer"></p> 
<p style="margin: 0 8px 25px;white-space: normal;visibility: visible;line-height: 1.75em;"><span style="visibility: visible;letter-spacing: 0.5px;font-size: 12px;color: #888888;">▲小朋友正在体验VR游戏</span></p> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">像这些太空椅、摩托车、战车等“大家伙”，常常出现在游乐场、大型商场、网吧等场地，是普通人<strong>最容易接触到</strong>的VR设备，也是线下VR体验馆中<strong>最常见</strong>的VR游乐设备。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">与80后、90后钟爱的电玩城相比，线下VR体验馆或许正在成为00后、10后的游戏娱乐中心“主力”。</span> 
</section> 
<p style="visibility: visible;box-sizing: border-box;padding: 0 8px;"><img class="rich_pages wxw-img" data-ratio="0.75" data-w="1080" style="border-radius: 5px;vertical-align: middle;display: block;margin: 0 auto;width:100%;" src="https://img.36krcdn.com/20220408/v2_74fc6aaeeb1a4119995c605393bf81e6_img_000" referrerpolicy="no-referrer"></p> 
<p style="margin: 0 8px 25px;white-space: normal;visibility: visible;line-height: 1.75em;"><span style="visibility: visible;letter-spacing: 0.5px;font-size: 12px;color: #888888;">▲VR游戏区域</span></p> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">线下VR体验馆起源于2015年的VR行业热潮，曾一度被认为是VR产业<strong>最快实现商业化变现的途径</strong>。据市场分析机构Greenlight Insights发布的线下VR市场调查报告，全球范围内线下VR体验馆的数量将在2023年增长到超24500个。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>线下VR体验馆有哪些类型？玩起来的感受如何？背后又有哪些厂商？火了这么多年，线下的VR体验到底是什么样？</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">在亲自体验一番线下VR体验馆，并和相关负责人深入交流后，我们发现线下VR体验馆的厂商们正在快速扩张自己的“商业版图”。</span> 
</section>

</section> 
<section style="margin-right: 8px;margin-left: 8px;white-space: normal;max-width: 100%;text-align: center;box-sizing: border-box !important;overflow-wrap: break-word !important;"> 
 <strong style="max-width: 100%;color: rgb(0, 0, 0);letter-spacing: 0.5px;box-sizing: border-box !important;overflow-wrap: break-word !important;">再迎爆发期，线下VR体验馆度过“寒冬”</strong> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">线下VR体验究竟在玩些什么？</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">在探访了多家VR体验店后，<strong>我们发现根据场地规模线下VR体验馆可以分为三种类型：</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>中岛</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;"><strong>、</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>大空间综合体验店</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;"><strong>以及</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>大型VR娱乐主题乐园</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="color: #2A5CAA;"><strong><span style="font-size: 15px;letter-spacing: 0.5px;">中岛</span></strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">往往只有两三台设备，像是蛋椅或者是过山车等小型VR游戏设备。中岛的VR游戏设备一次体验时长往往在</span> 
 <strong><span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;">10分钟到15分钟</span></strong> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">，体验价格在</span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>40元到60元</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">不等。</span> 
</section> 
<p style="visibility: visible;box-sizing: border-box;padding: 0 8px;"><img class="rich_pages wxw-img" data-ratio="0.75" data-w="1080" style="border-radius: 5px;vertical-align: middle;display: block;margin: 0 auto;width:100%;" src="https://img.36krcdn.com/20220408/v2_c9c5ded6d9d4409887502596c4fc6fd3_img_000" referrerpolicy="no-referrer"></p> 
<p style="margin: 0 8px 25px;white-space: normal;visibility: visible;line-height: 1.75em;"><span style="visibility: visible;letter-spacing: 0.5px;font-size: 12px;color: #888888;">▲中岛类型的VR游戏区域</span></p> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">而</span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>大型空间综合体验店</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">的价格会略高一些，往往在</span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>150元到200元</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">左右，时长也在</span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>10分钟到30分钟</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">不等。但这类线下VR体验馆不仅包括VR体验场地，往往还提供一些桌游场地。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">最后一类</span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>大型VR娱乐主题乐园</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">往往是指以各种VR设备搭成的VR主题乐园。行业人士透露，目前国内还没有完全落地的大型VR娱乐主题乐园，南京、长沙等多地均在推进相关项目。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">国外方面，美国线下VR娱乐公司The Void曾经在2015年打造一个虚拟现实娱乐中心。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">The Void的虚拟娱乐中心场景会依照虚拟世界的地形搭建起来，并且加入喷气、洒水等多种特效，让人们通过虚拟现实场景和实际主题乐园相结合形式，感受虚拟世界的乐趣。</span> 
</section> 
<p style="visibility: visible;box-sizing: border-box;padding: 0 8px;"><img class="rich_pages wxw-img" data-ratio="0.6638888888888889" data-w="1080" style="border-radius: 5px;vertical-align: middle;display: block;margin: 0 auto;width:100%;" src="https://img.36krcdn.com/20220408/v2_006dc003c0b04d44b1e5bada0674da61_img_000" referrerpolicy="no-referrer"></p> 
<p style="margin: 0 8px 25px;white-space: normal;visibility: visible;line-height: 1.75em;"><span style="visibility: visible;letter-spacing: 0.5px;font-size: 12px;color: #888888;">▲The Void</span></p> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">不过，由于疫情的影响，The Void在2020年停止运营，各大场馆均闭馆。近期The Void更新了自己的官网，并宣布他们将带着更好的VR技术回归。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">事实上，不仅是The Void，不少线下VR体验馆同样随着市场的变化，努力度过行业“寒冬期”。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">从现有材料来看，线下VR体验馆多数起源</span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>2015年</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">左右国内爆发的VR行业热潮。在那时，曾经有业内人士预测，<strong>线下VR体验馆会像网吧、游戏厅一样成为未来娱乐产业主体形态</strong>。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">但实际情况是，当年的线下VR体验馆并没有像当时多数人所设想的那样，迅速渗透至各大商场，取代网吧、游戏厅等场景，反而是处在</span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>不温不火</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">的状态。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">中国VR体验店联盟的《中国VR体验店现状白皮书》显示，至2016年11月，<strong>全国VR体验店超过3000家，但当年盈利的却不到30%。</strong></span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">就连</span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>Meta</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">也曾在2017年关闭了200家Oculus Rift线下体验店，差不多砍掉了在全美开设的500家体验店的40%。</span> 
</section> 
<p style="visibility: visible;box-sizing: border-box;padding: 0 8px;"><img class="rich_pages wxw-img" data-ratio="0.5842592592592593" data-w="1080" style="border-radius: 5px;vertical-align: middle;display: block;margin: 0 auto;width:100%;" src="https://img.36krcdn.com/20220408/v2_c883406023464fdf8f0f4323943d3b0a_img_000" referrerpolicy="no-referrer"></p> 
<p style="margin: 0 8px 25px;white-space: normal;visibility: visible;line-height: 1.75em;"><span style="visibility: visible;letter-spacing: 0.5px;font-size: 12px;color: #888888;">▲Oculus Rift线下体验店</span></p> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;">

</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">大朋VR是上海乐相科技有限公司旗下的VR品牌，该公司成立于2015年4月，是一家软硬件一体化的VR技术公司。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">国际调研机构Counterpoint数据显示，大朋VR 2020年Q4海外出货量排名全球Top5。就在2021年11月，大朋VR拿到了最新一轮的千万美元融资。现在，大朋VR已经将自己的商业触角延伸至四十多个国家和地区。其中，新加坡、马来西亚、日本等多个国家，大朋VR的业务占比高达30%。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">大朋的产品线在B端和C端均有分布，其中P系列主要是面对B端、E系列面对C端消费者，主打功能包括影院、游戏等消费级场景。</span> 
</section> 
<p style="visibility: visible;box-sizing: border-box;padding: 0 8px;"><img class="rich_pages wxw-img" data-ratio="0.5601851851851852" data-w="1080" style="border-radius: 5px;vertical-align: middle;display: block;margin: 0 auto;width:100%;" src="https://img.36krcdn.com/20220408/v2_7f0a5664dea147398c32882c1207a16d_img_000" referrerpolicy="no-referrer"></p> 
<p style="margin: 0 8px 25px;white-space: normal;visibility: visible;line-height: 1.75em;"><span style="visibility: visible;letter-spacing: 0.5px;font-size: 12px;color: #888888;">▲大朋E3头显</span></p> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">大朋VR CEO陈朝阳曾在接受媒体采访时说道：“近几年来，大朋内部一直是C端和B端双打的局面。因为市场成熟度原因，其中向B端偏的更多一些，这不仅来源于产业里虚拟人、数字人发展的启发，更多是关于大朋想要做一些差异化和有价值的事情。”</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;"><strong>他提到与C端的产品拼参数不同，面向B端的产品需求在提供同样性能的情况下，尽可能地为客户提供</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;color: #2A5CAA;"><strong>较高的性价比</strong></span> 
 <span style="font-size: 15px;letter-spacing: 0.5px;"><strong>。</strong></span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">现有资料表明，早在2017年，乐客VR的E3蛋椅系统就与大朋E3头显适配，并且联合出售，两个大朋E3一体机VR头显和E3蛋椅系统的售价为8800元。除了乐客VR，国内VR娱乐厂商幻影星空等同样也会在部分设备上采用大朋E3一体机VR头显。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;"><strong>3、STEPVR：逐步自研硬件产品，构建完整生态体验</strong></span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">与前面两家线下VR体验馆在VR设备方面与硬件厂商合作不同，<strong>STEPVR已经逐步开始研发自己的VR头显</strong>，根据自己对VR体验的看法，量身打造自己所需的VR头显。</span> 
</section> 
<p style="visibility: visible;box-sizing: border-box;padding: 0 8px;"><img class="rich_pages wxw-img" data-ratio="0.75" data-w="1080" style="border-radius: 5px;vertical-align: middle;display: block;margin: 0 auto;width:100%;" src="https://img.36krcdn.com/20220408/v2_f6d6b1bd857f4eb0a67c5143423ff7ef_img_000" referrerpolicy="no-referrer"></p> 
<p style="margin: 0 8px 25px;white-space: normal;visibility: visible;line-height: 1.75em;"><span style="visibility: visible;letter-spacing: 0.5px;font-size: 12px;color: #888888;">▲未来战场二代头显</span></p> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">与其他线下VR体验馆不同，STEPVR主要从事线下VR大空间综合体验馆，VR游戏体验区一般能达到千余平方米以上。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">STEPVR是北京国承万通信息科技的品牌，拥有VR线下大空间产品品牌“未来战场”。据相关负责人介绍，目前STEPVR还能支持异地大空间VR游戏对战，不同地区的人们可以同时体验线下VR娱乐的乐趣。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">“目前的热门游戏都具有联机、技能对打和PVP对战等要素，我们为消费者提供的游戏同样也是这种类型。”<strong>STEPVR的创始人兼总裁郭成说</strong>。<strong>他认为在内容生态方面需要优先给消费者提供最好玩，最热门的游戏类型。</strong></span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">在硬件方面，STEPVR已经完成近4代产品迭代，目前多数场馆使用的为第二代产品，也就是STEPVR与惠普公司联手开发的VR头显，和STEPVR自己研发的“背包”。随着自研化的节奏加快，STEPVR第三代自研VR头显和轻量化背心已经进入调试阶段，第四代产品也正在紧密研发的过程中。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">从这些厂商的动作来看，不少VR硬件公司都在布局B端和C端两条赛道，并且正在和线下VR内容公司联手打通VR内容生态。</span> 
</section> 
<section style="margin-right: 8px;margin-left: 8px;white-space: normal;max-width: 100%;text-align: center;box-sizing: border-box !important;overflow-wrap: break-word !important;"> 
 <span style="max-width: 100%;font-size: 40px;box-sizing: border-box !important;overflow-wrap: break-word !important;"><strong style="max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;"><em style="max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;">04</em></strong></span> 
 <span style="max-width: 100%;font-size: 40px;color: #00b0ec;box-sizing: border-box !important;overflow-wrap: break-word !important;"><strong style="max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;"><em style="max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;">.</em></strong></span> 
</section> 
<section style="margin-right: 8px;margin-left: 8px;white-space: normal;max-width: 100%;text-align: center;box-sizing: border-box !important;overflow-wrap: break-word !important;"> 
 <strong style="max-width: 100%;color: rgb(0, 0, 0);letter-spacing: 0.5px;box-sizing: border-box !important;overflow-wrap: break-word !important;">结语：线下VR体验馆“遍地开花”</strong> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">在经过走访和调查多家线下VR体验馆后，我们发现线下VR体验馆正在迅速扩张。不仅各大厂商想出各种花样吸引客户加盟，同时不少硬件厂商也在为客户提供更好的性价比服务。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">但线下VR体验馆也存在一定的不足之处，较高的价格、单一的游戏体验以及内容生态壁垒等不足之处也都直观地呈现在消费者的面前。</span> 
</section> 
<section style="margin: 15px 8px 25px;white-space: normal;line-height: 1.75em;text-align: justify;"> 
 <span style="font-size: 15px;letter-spacing: 0.5px;">线下VR体验馆正在成为越来越多的消费者进入VR世界最直接、快捷的方式。随着近期VR技术的突破，线下VR体验馆或迎来一场新的爆发阶段。</span> 
</section>
