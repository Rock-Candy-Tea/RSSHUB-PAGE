<p>编辑丨三明治</p> 
<p>苹果耳机上新了。</p> 
<p>在苹果宣布Apple Music将支持无损音乐和杜比全景声的空间音频功能后，似乎所有人都嗅到了苹果音频新品即将到来的信号。然而请果粉们不要激动，我们今天要说的不是你们期待已久的AirPods 3，也不是鲜有传闻的AirPods Pro 2。因为苹果最新推出的真无线耳机不是AirPods，而是Beats。</p> 
<p><img src="https://img.36krcdn.com/20210616/v2_2934188f98784d8f91f4f03784c89a37_img_000" data-img-size-val="1080,646" referrerpolicy="no-referrer"></p> 
<p>昨天傍晚，苹果旗下的音频公司Beats正式发布Beats Studio Buds真无线降噪耳机，这是Beats继Powerbeats Pro后推出的第二款真无线耳机。跟Powerbeats Pro不同，这款产品采用更加时尚、轻盈而舒适的设计，不仅支持IPX4级抗汗抗水功能，还首次加入主动降噪功能和通透模式。</p> 
<p>自打AirPods发售以来，数码圈就流传着这么一句话：AirPods是搭配iPhone的最佳无线耳机，没有之一。如今，Beats Studio Buds的出现给予果粉更多的选择。那么，消费者应该趁现在购买降价的AirPods 2，还是购买Beats Studio Buds？今天小雷就来和大家聊一聊。</p> 
<h2 label="一级标题" style>这不是AirPods</h2> 
<p>从外观设计上看，Beats Studio Buds的外观设计和此前的Beats系列耳机相去甚远。这款耳机的充电仓是圆润的鹅卵石形状，和目前市面上流行的苹果AirPods Pro、<a class="project-link" data-id="25167" data-name="华为" data-logo="https://img.36krcdn.com/20200729/v2_7c7826d711824e758a8e1511c9d7eecc_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/25167" target="_blank">华为</a>FreeBuds Pro、OPPO Enco X、<a class="project-link" data-id="121888" data-name="小米" data-logo="https://img.36krcdn.com/20200729/v2_7ad3d765f1d844018b8f4e75c4c8050d_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/121888" target="_blank">小米</a> FlipBuds Pro等产品的充电仓都很类似，设计更加偏向主流审美。</p> 
<p>耳机主体部分，Beats Studio Buds既没有采用Powerbeats Pro这类运动耳机标志性的挂耳结构，也没有采用类似AirPods系列的耳机柄设计。正如名称中的“Buds”那样，该机采用类似三星Galaxy Buds、Redmi AirDots 3那样的豆式耳机设计，体积也相当的小巧。</p> 
<p><img src="https://img.36krcdn.com/20210616/v2_536b5d006e52412285300898989f1330_img_000" data-img-size-val="1080,612" referrerpolicy="no-referrer"></p> 
<p>那么Beats Studio Buds有哪些特点呢？首先，Beats Studio Buds采用入耳式设计。要知道，AirPods 2采用的半入耳式设计虽然佩戴舒适，但是降噪效果很差，在嘈杂的环境中必须开大声音才能听清。作为对比，根据外媒实测，Beats Studio Buds拥有着不错的佩戴体验，而且被动降噪效果更强，这算是不错的提升。</p> 
<p>重点来了，Beats Studio Buds支持主动降噪功能，能有效阻隔风噪等各种外界噪声，还可以通过长按机身“b”键切换至通透模式，这使其成为目前苹果旗下最便宜的真无线降噪耳机。遗憾的是，根据外媒实测，Beats Studio Buds的主动降噪效果聊有胜无，而且通透模式也缺乏AirPods Pro那样的自然感觉，但比起半入耳式的AirPod 2还是有提升的。</p> 
<p><img src="https://img.36krcdn.com/20210616/v2_fde78f861e294948bf04368800d143e7_img_000" data-img-size-val="682,346" referrerpolicy="no-referrer"></p> 
<p>值得一提的是，尽管各家媒体都在鼓吹Beats Studio Buds支持空间音频技术，然而这款产品支持的空间音频和AirPods Pro、AirPods Max支持的空间音频并不是一个东西。Beats Studio Buds支持杜比全景声的空间音频功能，可以自动播放可用音轨中与杜比全景声中混合的空间音频，这是Apple Music五月更新的功能，基本所有AirPods和Beats耳机都支持。</p> 
<p>另一方面，目前在AirPods Pro和AirPods Max上实现的空间音频技术则是基于头部追踪的，其特点是会随着用户头部的运动产生大量数据，反馈四面八方的音频信息，这点对于没有陀螺仪的Beats Studio Buds是做不到的。</p> 
<p><img src="https://img.36krcdn.com/20210616/v2_fbe0c09bf3cb4fb18fecd2a5d69f24de_img_000" data-img-size-val="742,332" referrerpolicy="no-referrer"></p> 
<p><img src="https://img.36krcdn.com/20210616/v2_5fbdcb18236c49d1aab35108d0166049_img_000" data-img-size-val="910,29" referrerpolicy="no-referrer"></p> 
<p>当然，Beats不是AirPods，这款产品和AirPods 2有着不少差异。目前看来，Beats Studio Buds最大的缺点是没有配备苹果W1/H1芯片，这和以往的AirPods/Beats产品形成极大差距。尽管这款产品依然支持Siri语音控制、支持开盖快速配对苹果设备，但是不支持佩戴监测、不支持iCloud同步和多设备自动切换，在功能上有所阉割。</p> 
<p>此外，尽管Beats Studio Buds采用USB Type-C充电，在通用性上面提升不少，但是这款产品并不支持无线充电，基本可以看做是为了成本妥协做出的牺牲。</p> 
<p>总结一下，尽管这款产品佩戴体感不错，音质表现尚可，支持主动降噪，采用C口充电，但是它阉割了不少功能，对iPhone的适配不仅不如AirPods，甚至不如前代产品Powerbeats Pro。因此对果粉而言，还是花多点钱选择适配更好的AirPods Pro比较好。</p> 
<h2 label="一级标题" style>“在安卓上表现一样好”</h2> 
<p>有的人可能会问了，既然没有优势，那么苹果为何还要推出这款产品？这就值得我们好好分析一下了。根据Counterpoint提供的2021年Q1 TWS市场调查报告，2021第一季度全球TWS耳机销量同比增长44%，其中苹果虽然继续占据主导地位，但由于市场竞争的不断加剧，所占份额有所下滑。</p> 
<p><img src="https://img.36krcdn.com/20210616/v2_5cd5c47100444f09b1839f5e208021a6_img_000" data-img-size-val="950,123" referrerpolicy="no-referrer"></p> 
<p>根据Counterpoint 2021年全球TWS市场预测，全球TWS市场预计到2021年将同比增长33%，达到3.1亿台。预计苹果将在2021年继续占据主导地位，但全年份额稳步下滑，跌至TWS市场的三分之一以下。小米、三星和华为等品牌正在逐步拉近与苹果产品的差距，这些厂商预计将在2021年展开激烈竞争。</p> 
<p><img src="https://img.36krcdn.com/20210616/v2_e61374e6eaac49db86cf7c4375af068e_img_000" data-img-size-val="1024,598" referrerpolicy="no-referrer"></p> 
<p>如今，苹果正面对着三大问题。第一，产品选项不多，只有AirPods 2、AirPods Pro以及AirPods Max三款产品可选。第二，产品价格太贵，就连最基础的AirPods 2首发价也达到了159美元（附充电盒需要199美元），根据市场分析，更多的消费者在疫情期间倾向选择中档或入门级的TWS耳机。第三，产品的针对性太强，AirPods有很多“仅限苹果”的便利功能，使它基本无缘安卓市场。</p> 
<p>苹果需要推出一款兼容性更好的平价耳机，这样既可以弥补苹果TWS耳机产品线在中低端价位上的缺失，也能让用户在售价上有更多的选择，还能剑指安卓市场。遗憾的是，根据调查，果粉并不愿意看到廉价AirPods产品的出现。既然没法对AirPods下手，那苹果只好选择另外<a class="project-link" data-id="3969340" data-name="一条" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3969340" target="_blank">一条</a>道路。</p> 
<p><img src="https://img.36krcdn.com/20210616/v2_67a909a5fbe142e3b03ecb94a9acc26d_img_000" data-img-size-val="640,397" referrerpolicy="no-referrer"></p> 
<p>Beats Studio Buds就是为了解决这些问题诞生的。这款产品的出现，为苹果用户提供了一个新的廉价选择，如果不介意一些功能缺失的话，它甚至可以成为AirPods的廉价替代品。</p> 
<p>在容纳安卓用户方面，Beats Studio Buds做得比苹果公司任何产品都更加完美。安卓用户只需下载Beats App，即可享受和苹果设备几乎一致的开盖弹窗、快速配对、切换聆听模式、检查电池电量和接收固件更新，唯一缺失的就是免提语音助手功能。</p> 
<p><img src="https://img.36krcdn.com/20210616/v2_6e2e6e5651d24026aca7867a08c3f5d6_img_000" data-img-size-val="640,514" referrerpolicy="no-referrer"></p> 
<p>对安卓用户来说，iCloud同步和多设备自动切换功能是用不着的，没有苹果芯片这点缺陷根本无伤大雅。支持USB Type-C充电，使得Beats Studio Buds对安卓用户而言格外友好，只需随手一插即可充能，就连没有无线充电似乎也没什么大不了的。</p> 
<p>价格方面，Beats Studio Buds的起步价为149美元/1099元，这已经算是苹果TWS耳机产品线的最低售价。不难看出，苹果希望在利用AirPods产品线维持高端市场的同时，利用Beats产品线的品牌价值和相对较好的兼容性进攻中低端和安卓市场的野心。</p> 
<p><img src="https://img.36krcdn.com/20210616/v2_e8a6821206c7411b9de0492632f75857_img_000" data-img-size-val="883,563" referrerpolicy="no-referrer"></p> 
<p>问题来了，1099元的售价虽然不贵，但也算不上便宜。举个例子，小米的TWS旗舰耳机FlipBuds Pro售价799元，OPPO的TWS旗舰耳机Enco X售价999元，华为的TWS旗舰耳机FreeBuds Pro售价1099元起。对比这些竞品，Beats Studio Buds价格没有优势，又不支持佩戴检测、无线充电等实用功能。除了Beats的品牌效应以外，产品力确实有些不足。</p> 
<p>总而言之，小雷并不建议大家首发入手这款产品。从Beats所有产品的价格走势来看，这款产品应该在暑假就会降价，在某些电商平台甚至可能对半砍。如果最终日常售价能够来到600元左右的话，对于耳机选购预算低的读者来说，Beats Studio Buds应该还算是个可以的选择。当然，追求音质和最佳体验的用户，还是老老实实购买自家品牌的旗舰耳机吧。</p>
