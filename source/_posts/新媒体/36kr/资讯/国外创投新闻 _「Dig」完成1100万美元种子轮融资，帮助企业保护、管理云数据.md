<p>据 <a target="_blank" rel="noopener noreferrer nofollow" href="https://techcrunch.com/2022/05/19/dig-data-secured-public-cloud/">Techcrunch</a> 报道，初创公司「Dig」日前完成了1100万美元的种子轮融资，本轮资金由 Team8 领投，CrowdStrike、Cyber​​Ark 和 Merlin Ventures 参投。「Dig」计划运用新资金将其员工人数从30人增加到50人。</p> 
<p>「Dig」是一家总部位于特拉维夫的云数据安全初创公司，旨在帮助企业保护存储在公共云环境中的数据。根据「Dig」的说法，企业数据通常是一些网络犯罪分子的最终目标，这已不是什么秘密，但很多组织对存储在公共云环境中的数据（如亚马逊、谷歌和微软运行的环境）没有可见性或控制权。「Dig」因此开发了数据检测和响应 (DDR) 解决方案，声称可以帮助企业实时发现、保护和管理他们的云数据。</p> 
<p class="image-wrapper"><img data-img-size-val="1889,835" src="https://img.36krcdn.com/20220520/v2_8b808062c95b45a1b3adc0adabadef6c_img_png" referrerpolicy="no-referrer"></p> 
<p class="img-desc">图片来自官网</p> 
<p>“公司不知道他们在云中保存了哪些数据，不知道数据在哪里，甚至也不知道如何保护数据。他们拥有保护端点、网络、API 的工具，但却没有保护公共云中数据的工具，” 「Dig」的联合创始人兼首席执行官 Dan Benjamin 表示。在去年10月创立「Dig」之前，Benjamin 在 Microsoft 领导多云安全，并在 Google Cloud for Startups 指导 CTO 。</p> 
<p>「Dig」声称，与现<a class="project-link" data-id="1713118059047428" data-name="有解" data-logo="https://img.36krcdn.com/20200424/v2_a55b83a38993466a99e2251c6aa4cd0f_img_png" data-refer-type="1" href="https://36kr.com/project/1713118059047428" target="_blank">有解</a>决方案不同，它会立即分析和响应对云数据的威胁，触发可疑或异常活动警报，阻止攻击、数据泄露和员工数据滥用。该解决方案为公共云中的所有数据资产的使用方式提供背景信息，并跟踪每个数据源是否支持 SOC2 和 HIPAA 等合规性要求。</p> 
<p>“就在前一周，我们与一家大型金融上市美国公司整合，五分钟后，我们就收到了警报。我们发现，他们将所有财务报告复制到了不属于他们的外部 AWS 账户，” Benjamin 说，“我们经常看到这样的事情，因为没有人真正了解这些数据的正确使用方式。”</p> 
<p>Benjamin 与资深企业家 Ido Azran 和 Gad Akuka 一起创立了这家初创公司，三位联合创始人名字的第一个字母拼写为「Dig」。「Dig」目前与 Microsoft Azure 和 AWS 合作，即将支持谷歌云平台，他们的最终目标是超越公共云，提供一种解决方案来保护组织内任何位置的数据。</p> 
<p>“对于一般的企业而言，数据位于五个主要位置：端点、电子邮件、内部部署、SaaS 和公共云，” Benjamin 说，“虽然我们只涵盖公共云，但我相信，最终客户最需要的就是一个能够保护任何位置数据的单一平台。”</p>
