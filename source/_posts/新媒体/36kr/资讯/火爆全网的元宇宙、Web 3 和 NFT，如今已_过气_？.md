<div>   
<p>提及 2021 年的技术热词，“元宇宙”、“Web 3”和“NFT”绝对排得上号：Facebook 直接将公司更名为 Meta、著名硅谷顶尖风投公司 Andreessen Horowitz（A16Z）加注 Web 3、受人追捧的天价 NFT 艺术品…</p> 
<p>这一桩桩、一件件，令无数身处技术圈的从业者们不得不开始关注这些突然爆火的词语。</p> 
<p>然而，就在我们好不容易将这些新兴词汇消化理解后，却发现不论是“元宇宙（Metaverse）”、“Web 3”还是“NFT”，似乎已不再那么地频繁出现在各大技术网站——它们难道已经“过气”了？</p> 
<p>带着这样的疑惑，近日一位 Medium 编辑 Stephen Moore 发声：元宇宙和 NFT 的繁荣盛世即将破灭。</p> 
<h2><strong>元宇宙热度不再？</strong></h2> 
<p>2021 年突然爆火的元宇宙，因 Facebook 的改名为其热度又“添了把柴”，过去一年在科技圈掀起了一股巨大的“元宇宙热潮”：</p> 
<p>“元宇宙第一股”的开放式游戏创建平台 Roblox 上市，首日股价大涨 54%，市值突破 400 亿美元；微软 CEO Satya Nadella 曾表示，正在建立一个企业元宇宙；英伟达 8 月主动爆料，其 CEO 黄仁勋在 4 月的视频发布会中有 14 秒的片段，是由实时仿真和协作平台 Omniverse 人工合成的，而英伟达希望该平台可以成为构建元宇宙的基础设施…</p> 
<p>在这之中，将公司名都改为元宇宙（Metaverse）的 Meta 更可谓全力冲击。Meta CEO Mark Zuckerberg 在更名时曾放言：“这（元宇宙）将是继移动互联网之后，互联网发展下个篇章的重要组成部分。”</p> 
<p>但根据上个月 Meta 所公布的 2021 年第四季度财报显示，Meta 旗下负责公司元宇宙业务的部门 Reality Labs 在 2021 年的净亏损约为 102 亿美元——对此，Stephen Moore 将 Meta 所执着的元宇宙成为“反乌托邦的梦想”，并预言 Meta 还将为此“烧掉”更多的钱。</p> 
<p>除此之外，如果查看 Chrome 浏览器中“元宇宙”的搜索趋势，可以发现其热度巅峰即为 Facebook 更名，此后整体呈跌落趋势：</p> 
<p class="image-wrapper"><img data-img-size-val="700,238" src="https://img.36krcdn.com/20220401/v2_f6fd529e18024579af62fa38451ca7df_img_000" referrerpolicy="no-referrer"></p> 
<h2><strong>NFT 价值大跌</strong></h2> 
<p>元宇宙热度下，NFT 也随之兴起。2021 年 3 月，数字艺术家 Beeple 的一幅画改变了 NFT 艺术品的发展：Beeple 将 2007 年 5 月以来每天创作的 5000 张数字图片按时序排列拼凑为一幅巨型拼贴 NFT 作品，并将其取名为《Everydays：The First 5000 Days》，最终历时 15 天的拍卖，以近 7000 万美元的“天价”成交落锤。</p> 
<p>Beeple 的事迹令许多人坚信，人们是愿意为纯粹的数字产品支付巨款的，也由此引发了一场 NFT 艺术品的“狂热”：一幅班克西的艺术画作被人为烧毁，此后该画作的 NFT 版本以 38 万美元的高价售出（该价格相当于原画价格的 4 倍）；万维网源代码以 NFT 形式拍卖，成交价高达 540 万美元；NBA 球星库里花 18 万美元购入 NFT 猴子头像…</p> 
<p>准确来说，NFT 并非新概念，它早在 2017 年因区块链养猫游戏 CryptoKitties 就曾小小“出圈”过一把，去年由于 Beeple 和元宇宙爆火再度掀起了新一轮热潮（注：许多人认为元宇宙将成为使用这些 NFT 的数字“游乐场”）。</p> 
<p>但事实证明，随着元宇宙热度下降，NFT 的价值也不复从前：</p> 
<p>据 Stephen Moore 了解，自今年 1 月攀至巅峰后，如今 NFT 月度买家首次跌破 80 万人，NFT 的平均价格也从 1 月份的 6800 美元跌至 2000 美元；二次销售量也从 1 月份的每天 38000 件下降到 7900 件；甚至整个 NFT 市场都受到了下跌的影响：Coin Market Cap 数据显示，NFT 市场的价值已从 230 亿美元跌至约 100 亿美元，缩水超一半以上。</p> 
<p>甚至同样查看 NFT 在 Chrome 浏览器中的搜索趋势，也可发现近来其热度的下降：</p> 
<p class="image-wrapper"><img data-img-size-val="700,250" src="https://img.36krcdn.com/20220401/v2_c1deefe721d246ab8b2385395664d9cd_img_000" referrerpolicy="no-referrer"></p> 
<h2><strong>“去中心化”的 Web 3 遭质疑</strong></h2> 
<p>除了元宇宙和 NFT 热度减退，有关 Web 3 的争议也越来越大。</p> 
<p>Web 3 也是去年突然火起来的一个概念——截止目前，它仍仅限于概念阶段。总体而言，Web 3 指的是基于边缘计算、人工智能和区块链等技术的去中心化系统和技术，旨在取代目前我们所处于的 Web 2 时代。</p> 
<p>在这之中，“去中心化”是许多人支持 Web 3 的一大理由：用户可自行组合不同的服务，定制自己的 Web 和搜索引擎，还可以访问并控制自己的资产而无需任何提供商的许可。但目前问题在于，Web 3 真的能做到这一点吗？</p> 
<p>Whisper Systems 联合创始人兼 CTO Moxie Marlinspike 对此进行了一项实验：他像每个初尝 NFT 的人一样做了一个图片，并将其 mint 成 NFT 放在 OpenSea（一个 NFT 市场）上卖。在发现 NFT 本身对该图片没有任何核查后，他将提供图片的 Web 服务器设置成根据不同 IP 呈现不同的图像来测试 OpenSea 的反应。</p> 
<p>随后没几天，OpenSea 作为一个中心化平台下架了他的 NFT，这点 Moxie 可以理解，他惊讶的是他自己 metamask 小狐狸钱包里的 NFT 也消失了——说消失就消失，这岂不是违背了 Web 3 所呼吁的“去中心化”？Moxie 对此担忧道：“Web3 社区所期待的结果似乎与我们目前已看到的不同。”</p> 
<p>国内 @木遥 曾就 Moxie 的这篇文章总结为：“这整体上体现了一个不可避免的从去中心化滑向中心化的趋势。”</p> 
<blockquote> 
 <p>区块链确实是不可篡改的，但没有人真的直接在底层区块链上工作（太麻烦了），大家都会自然依赖生态系统里各种现成的工具，而这些工具为了效率上的竞争，自然而然就会趋于中心化。</p> 
</blockquote> 
<p>以太坊创始人 V 神（Vitalik Buterin）也对这篇文章回应道：“这是现状没错，但这不是 Web 3 应有的样子。”V 神指出，真正的 Web 3 世界应该有一个连续的过渡光谱，在最易用的中心化平台和最难用的自己搞一个服务器之间有大量的过渡态以适应不同的应用场景，只是如今这中间的过渡态是缺失的。不过 V 神坚信：“未来这一缺失一定会被建立起来。”</p> 
<h2><strong>无法有效落地，再酷炫的概念也“白搭”</strong></h2> 
<p>以上热词“过气”的背后，免不了受到了外界因素的影响，例如当前国际形势下，大多媒体和投资者自然会选择更紧要的事项去宣传。但 Stephen Moore 认为，这些概念热度不再的最大原因在于：看客们很难找到这些概念落地后的实在价值与意义。</p> 
<p>支持元宇宙、Web 3 和 NFT 的公司和创作者很多，但他们目前仍处于摸索阶段，也尚未替这些概念找到真正有价值的用例。雪上加霜的是，随着 NFT 骗局不断发生，诸如“Web 3 是一场去中心化的骗局”、“元宇宙就是换了个说法的 AR/VR”等声音越来越大，公众逐渐开始怀疑：这些热词或许只是资本的炒作。</p> 
<p>自然，这一切负面现象都可以用一句“现在还处于早期阶段”来解释。但换个角度思考，是否应在早期阶段就合理规避隐患问题、并确定其发展方向？</p> 
<p>不过，尽管目前元宇宙、Web 3 和 NFT 不如巅峰期火热，但相关技术仍在持续发展，或许有朝一日它们会确立自己的定位、真正成为其支持者们努力拥护的新“去中心化”互联网的基石。不过至少目前，Stephen Moore 认为这一切只不过是“资本主义下炒作的无意义、昂贵、资源消耗的结果”。</p> 
<p>那么，你如何看待这些技术热词，你又是否坚信未来这些概念能有效落地？</p>
