<p>网易公告称，Cloud Village 于 2016 年 2 月 2 日在开曼群岛注册成立，并自 2016 年 10 月起独立运营本公司的音乐流媒体平台网易云音乐。网易云音乐2020 年在线音乐服务的月活跃用户超过 180 百万，而 Cloud Village 亦通过其平台运营直播和广告等其他业务。该平台在所提供的音乐方面带来差异化和优质的用户体验，并专注于发现和推广新兴音乐人。于本公告日期，本公司持有 Cloud Village 已发行股本总额的约 62.46%。</p> 
<p><img src="https://img.36krcdn.com/20210526/v2_bfc345fa5d5641d5a7f656d0abaac2cd_img_000" data-img-size-val="630,485" referrerpolicy="no-referrer"></p> 
<p>以下为公告正文内容：</p> 
<p label="正文" style><strong>拟议分拆</strong></p> 
<p>现建议，拟议分拆将以 Cloud Village 股份全球发售的方式进行。拟议分拆完成后，本公司将持有 Cloud Village 不少于 50%的投票权，因此，Cloud Village 将仍为本公司的子公司。</p> 
<p>进行拟议分拆的理由及裨益 基于以下理由，董事会认为拟议分拆将对本公司及 Cloud Village 有利，其中包括：(i) Cloud Village 集团的业务预期将经历相对快速的业务扩张，并将吸引专注于寻求音乐流媒体业务方面高增长机会的投资者群体，这不同于网易集团业务相对 更为多元化的业务模式；</p> 
<p>(ii)鉴于在香港联交所上市将有以下优势：</p> 
<p>(a)提升 Cloud Village 集团在其用户和其他业务伙伴间的形象，以及其招募人才的能力；</p> 
<p>(b)使 Cloud Village 集团将来于需要时可单独 且直接、独立地进入股票及债务资本市场，以及进一步提升其获取银行信贷融 资的能力；</p> 
<p>(c)使 Cloud Village 集团管理层的职责和责任更直接与其经营和财务 表现挂钩；</p> 
<p>(d)为评级机构和金融机构提供 Cloud Village 集团的清晰信用状况 ，以便对音乐流媒体业务的信用进行分析和提供贷款，故预期 Cloud Village 集 团的价值将因建议分拆而提升，而本公司作为 Cloud Village 的控股股东将会因 此获益；</p> 
<p>(iii) 拟议分拆将使本公司专注于网易集团的业务，从而使投资者能更好地评估本公 司的价值。</p> 
<p>此外，本公司和 Cloud Village 均认为，建议分拆能够更好地反映 Cloud Village 集团基于自身优点的价值，并提高其运营和财务透明度，而投资 者藉此将能够单独评估 Cloud Village 集团的表现和潜力，并与网易集团区别开来。</p>
