<p><a class="project-link" data-id="3967413" data-name="微软" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3967413" target="_blank">微软</a>Win11安卓 子系统在官宣后经过多次曝光，现在终于要来了。 </p> 
<p>据IT之家网友 @扶摇直上九万里 投稿，Win11 安卓子系统将面向 Beta 通道的 <strong>美国用户</strong> 开启测试。</p> 
<p class="image-wrapper"><img data-img-size-val="1080,242" src="https://img.36krcdn.com/20211022/v2_4f91c82d6e2a429596c4294f21cd55ee_img_000" referrerpolicy="no-referrer"></p> 
<p>微软表示，在 Windows 11 的安卓 App 将拥有熟悉、轻松、集成的体验： </p> 
<blockquote> 
 <p>用户可以将安卓 App 固定到开始菜单或任务栏，并通过鼠标、触摸或笔输入与它们交互； </p> 
 <p>安卓 App 还可以集成到 Alt + Tab 和任务视图中，帮助用户在 App 之间快速切换； </p> 
 <p>用户可以在操作中心中查看来自安卓 App 的通知，或在 Windows 应用程序和安卓 App 之间共享剪贴板； </p> 
 <p>微软还添加了无障碍体验，许多 Windows 辅助功能设置都适用于安卓 App。 </p> 
</blockquote>

<p>微软与亚马逊合作，为 Windows 内部人员首先推出了 50 款安卓 App，以便在一系列硬件上进行测试和验证。微软将在未来几个月通过 Windows Insider 程序发布更多的安卓 App。 </p> 
<p>这 50 款 App 中包含： </p> 
<blockquote> 
 <p>手机游戏：《王国纪元》《琼的旅程》《 Coin Master》等； </p> 
 <p>阅读：Kindle App、漫画 App 等； </p> 
 <p>儿童内容：Khan Academy Kids、Lego Duplo World 等。 </p> 
</blockquote> 
<p>IT之家了解到，Win11 的安卓子系统包括 Linux 内核和基于Android<a class="project-link" data-id="4262185" data-name="开源项目" data-logo="https://img.36krcdn.com/20210603/v2_43fe3145b6494227bd8db07dcdc0147b_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/4262185" target="_blank">开源项目</a>（AOSP）版本 11 的 Android 操作系统。它作为 Amazon Appstore 安装的一部分通过 Microsoft Store 分发。 </p> 
<p>该子系统在 Hyper-V 虚拟机中运行，就像 Linux 子系统一样，可以将 AOSP 环境中 App 的运行时和 API 映射到 Windows 图形层、内存缓冲区、输入模式、物理和虚拟设备以及传感器，可以在<a class="project-link" data-id="3968654" data-name="英特尔" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3968654" target="_blank">英特尔</a>、AMD、高通的 CPU 上运行。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,608" src="https://img.36krcdn.com/20211022/v2_6870f8fe73ec470685c562b9ab31fd58_img_000" referrerpolicy="no-referrer"></p> 
<p>微软为大家提供了一些更详细的说明以帮助用户了解更多有关于在 Windows 上运行 安卓App 的方法。 </p> 
<p>需要注意的是，适用于 Android 的 Windows 子系统目前只能通过 Windows 预览体验计划中的预览获得。您的设备还必须满足特定的Windows 11要求。 </p> 
<p>IT之家了解到，要运行这些移动应用程序，您需要 Windows 11 版本 21H2 Build 22000 .xxx 或更高版本，并 <strong>将地区设置为美国</strong> 。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,608" src="https://img.36krcdn.com/20211022/v2_fae458cbf8d1435da7579a9dd84f9290_img_000" referrerpolicy="no-referrer"></p> 
<p>要查看您的 PC 所需的最低要求，首先需要满足 Windows 11规格 和系统要求，然后还需要满足 Windows 11 部分功能的特定要求。 </p> 
<p>有两种方式可以安装 Android 子系统： </p> 
<blockquote> 
 <p>打开 Microsoft Store 应用程序并搜索 Amazon Appstore。选择 获取 并安装它。在安装过程中，适用于 Android 的 Windows 子系统将安装在您的 PC 上。 </p> 
 <p>在 Microsoft Store 上搜索移动应用。Amazon Appstore 和适用于 Android 的 Windows 子系统将在安装移动应用程序之前安装。 </p> 
</blockquote> 
<p>安装完成后，Windows Subsystem for Android Settings 应用程序和 Amazon Appstore 将出现在“开始”菜单和您的应用程序列表中。 </p> 
<p>从亚马逊应用商店下载移动应用程序需要亚马逊账户。准备就绪后，打开 Amazon Appstore 并使用您的 Amazon 帐户登录。 </p> 
<p>安装移动 App 后，您可以直接从“开始”菜单和应用程序列表中打开运行。 </p> 
<p>该子系统使用独立于 Windows 的文件系统 —— 这些文件无法传输到 Windows。此外，无法在子系统和任何移动应用程序中访问 Windows 中可用的文件。如果需要，以下是查看子系统文件的方法： </p> 
<blockquote> 
 <p>打开 适用于 Android 设置 应用程序的 Windows 子系统。 </p> 
 <p>选择 文件。 </p> 
 <p>您在子系统中的文件将打开，您可以浏览保存在子系统中的任何图像、视频、音频、文档和下载。 </p> 
</blockquote> 
<p>需要注意的是，在 PC 上浏览移动应用程序是一种完全不同的体验。以下是一些有关日常使用的提示： </p> 
<blockquote> 
 <p>使用鼠标上的滚轮或触控板手势滚动浏览应用程序。您还可以使用鼠标或触控板在应用程序中单击并向上、向下、向左和向右拖动。 </p> 
 <p>选择移动应用程序标题栏中的后退按钮可返回上一页。 </p> 
 <p>一些游戏支持通过方向键走动，有些可能还支持鼠标来移动。 </p> 
</blockquote>
