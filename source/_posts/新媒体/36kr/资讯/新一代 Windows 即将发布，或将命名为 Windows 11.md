<p>编者按：本文来自<a class="project-link" data-id="3968527" data-name="微信" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3968527" target="_blank">微信</a>公众号<a target="_blank" rel="noopener noreferrer" href="https://mp.weixin.qq.com/s/iqYnk7BfNaP1W4gXyei-xA">“APPSO”（ID:appsolution）</a>，作者：李华，36氪经授权发布。</p> 
<p>在上个月举办的 Build 2021 中，<a class="project-link" data-id="3967413" data-name="微软" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3967413" target="_blank">微软</a>并未提及下一代 Windows 将会在何时推出。微软首席执行官萨蒂亚·纳德拉（Satya Nadella）在演讲中透露：最近几个月他一直在测试「下一代 Windows」。</p> 
<p>「很快，我们将分享过去十年来 Windows 最重要的更新之一。」</p> 
<p class="image-wrapper"><img data-img-size-val="1024,768" src="https://img.36krcdn.com/20210607/v2_6e238ef2746649a3921f98f9659c9df5_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">▲微软 CEO 萨蒂亚·纳德拉，图片来自：OnMSFT</p> 
<p>6 月 2 日，微软宣布将于本月 24 日发布新一代 Windows，这家软件巨头已经开始向媒体发送邀请。但这个邀请似乎别有深意。</p> 
<p>发布会邀请中最为显眼的就是一扇白色的窗户，这有可能会是 Windows 11 的 Logo。光从窗户外照射进来，在地上形成了一个<a class="project-link" data-id="110228" data-name="中间带" data-logo="https://img.36krcdn.com/20200729/v2_1caa4a595c4a42d6a419ad8e82767ee7_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/110228" target="_blank">中间带</a>有竖条的光影，看起来非常像阿拉伯数字「11」。</p> 
<p>微软随后推出了该图像的动画版本，证实了「微软故意去掉窗户横杠的阴影」这一说法。</p> 
<p class="image-wrapper"><img data-img-size-val="1080,720" src="https://img.36krcdn.com/20210607/v2_38915cc149704d0db583348485ef54e8_img_000" referrerpolicy="no-referrer"></p> 
<p>发布会的时间貌似也暗藏玄机。发布会将会在 24 日的美国东部时间 11 点发布，这并不是 Windows 及 Surface 发布会常见的开始时间。</p> 
<p>在发布邀请后，微软高管优素福·马赫迪（Yusuf Mehdi）表示他「自 Windows 95 以来就没有对新版本的 Windows 如此兴奋。」</p> 
<p class="image-wrapper"><img data-img-size-val="1080,1081" src="https://img.36krcdn.com/20210607/v2_be7f61019b0746968acc4fd75001158e_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">▲Windows 95，图片来自：Yusuf Mehdi</p> 
<p>假如说微软真的准备推倒 Windows 10，转向 Windows 11，最直观的变化应该就是 UI 了。</p> 
<p>微软上个月宣布，其最初为双屏设备打造的操作系统 Windows 10X 将不再发售。有消息称目前微软正在将 Windows 10X 上的视觉元素带到 Windows 中来。其中包括新的开始菜单、系统图标，以及文件管理器的改进等，微软还计划对内置的 Windows 应用程序进行圆角化的更新。</p> 
<p>我们终于可以和 Windows 95 系列图标说再见了吗？</p> 
<p class="image-wrapper"><img data-img-size-val="885,436" src="https://img.36krcdn.com/20210607/v2_6e5373d05e484781a748edf44179e0b0_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">▲Windows 95 系列图标，图片来自：The Verge</p> 
<p>除了视觉上的更新，应用商店的改进似乎也是新一代 Windows 的重点项目。</p> 
<p>据报道，微软正在彻底改革其 Windows 应用商店，甚至允许开发者提交任何 Windows 应用程序，其中包括 Chrome 以及 Firefox 等浏览器。</p> 
<p>过去，Windows 应用商店要求开发人员将他们的 win32 应用程序打包为 MSIX，并使用微软自家的更新机制和商业平台。而在新版应用商店中，微软将允许开发人员向商店提交标准 EXE 或 MSI 包，并且可以通过开发人员自己的内容交付网络 (CDN) 管理更新。</p> 
<p class="image-wrapper"><img data-img-size-val="1080,720" src="https://img.36krcdn.com/20210607/v2_761fca5beb8649389db4b8e16156cfaf_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">▲图片来自：The Verge</p> 
<p>另外， 微软似乎已经准备好去解决那些「疑难杂症」了。</p> 
<p>其中包括在多台显示器上重新排列应用图标的问题、即将推出的 Xbox Auto HDR 功能，以及对于蓝牙音频的改进等。</p>
