<p>一直以来，对数码圈有所关注的朋友都有这样一个共识：办公选 Windows 或者 Mac 都行，但是打游戏绝对不能选 Mac。</p> 
<p>作为游戏爱好者的果子自然也是这么认为的，Mac 平台不仅 3A 大作极少，而且运行《CS:GO》这样的老 FPS 射击游戏，帧率也远不如同价位的 Windows 游戏本。</p> 
<p class="image-wrapper"><img data-img-size-val="952,617" src="https://img.36krcdn.com/20220608/v2_18faaa79578945828fa880fdf16f5c4b_img_000" referrerpolicy="no-referrer"></p> 
<p>所以如今果子虽然上班会使用 Mac 办公，但是家里却一直在使用 Windows 电脑，因为它不仅能办公，而且还能打游戏，属实是有点超值。</p> 
<p>但是令果子万万没想到的是，这一认知或许会因为苹果今早为 macOS Ventura 推出的 Metal 3 API 而被彻底改变……</p> 
<h2><strong>Metal 3 大幅提升 Mac 游戏性能</strong></h2> 
<p>2014年，苹果首次发布了名为 Metal 图形 API 接口。这是一款面向系统底层、兼顾图形和计算功能的 API，相当于是苹果为 iOS 平台做的类似微软 DX 一样的 API。</p> 
<p>当时苹果在发布会上演示了采用该 API 接口开发的 MOBA 手游《虚荣》，画面表现力极其出色，几乎与同期电脑 MOBA 端游的画质不相上下，这无疑是给了当时的手游开发者极大的惊喜。</p> 
<p class="image-wrapper"><img data-img-size-val="678,452" src="https://img.36krcdn.com/20220608/v2_a51275b45e6a408db7cfc0c426c511cf_img_000" referrerpolicy="no-referrer"></p> 
<p>在今天凌晨的 WWDC 大会上，苹果再次给了所有游戏开发者一个巨大的惊喜——为 macOS Ventura 带来 Metal 3 API 接口。</p> 
<p class="image-wrapper"><img data-img-size-val="848,486" src="https://img.36krcdn.com/20220608/v2_abb283a09ba945a49d8a9f281b4dae6d_img_000" referrerpolicy="no-referrer"></p> 
<p>Metal 3 API 可以理解成一个为 Mac 游戏开发者准备的开发工具包，其中含有许多能够明显提升 Mac 平台游戏体验的 API 接口和功能。</p> 
<p>例如 MetalFX Upscaling 技术能够让游戏大幅加快复杂场景的渲染速度，在提供更高分辨率的同时，也能够为用户提供更加流畅的高帧率游戏体验，帮助游戏开发者打造更具沉浸感的游戏视觉效果。</p> 
<p class="image-wrapper"><img data-img-size-val="937,476" src="https://img.36krcdn.com/20220608/v2_dac0850fe83a47c7bbad6e771aa93ab2_img_000" referrerpolicy="no-referrer"></p> 
<p>快速资源加载 API 可以加快从 GPU 处理器到内存、SSD 的读取速度，让游戏能够更快地找到需要渲染的资源，直接点说就是读图更快、更流畅，加载游戏资源的速度更快。</p> 
<p>卡普空高管就在大会上表示，采用了 Metal 3 API 开发的《生化危机：村庄》能够在最新的 MacBook Air 上以 1080P 分辨率完<a class="project-link" data-id="1679700972000003" data-name="美运" data-logo="https://img.36krcdn.com/20220401/v2_61db78f21d7043d7bfbfaf03a5488542_img_000" data-refer-type="1" href="https://36kr.com/project/1679700972000003" target="_blank">美运</a>行，在 Mac Studio 上甚至能够以 4K 分辨率运行。</p> 
<p>联想到此前 Mac 运行 3A 游戏大作时的“丑态”，不得不说，Metal 3 对 Mac 平台游戏性能的提升完全可以算是革命性的。</p> 
<h2><strong>解决 Mac 平台游戏缺失问题</strong></h2> 
<p>苹果 Mac 平台缺少游戏的原因，不仅仅在于 Mac 产品的 GPU 性能差，还在于苹果一直没能给开发者提供高效的 GPU 调用接口。</p> 
<p>这就好比苹果开了一家饭店，雇一个厨师炒菜，并且采购的蔬菜、肉都非常不错，但是只给厨师一把水果刀切菜，即便厨师怎么努力，最终做菜的效率（游戏运行效率）都非常低。</p> 
<p class="image-wrapper"><img data-img-size-val="1080,612" src="https://img.36krcdn.com/20220608/v2_ea81eea19c5b4235a367ebdb6006ccde_img_000" referrerpolicy="no-referrer"></p> 
<p>因此相当多的游戏引擎开发商和游戏厂商就纷纷选择放弃适配 Mac 平台，例如虚幻引擎、《守望先锋》等等一系列引擎和游戏，毕竟巧厨难为水果刀之炊嘛~</p> 
<p>而解决水果刀（推出 Metal 3 API）的问题后，显然就相当于给了厨师一把屠龙刀，让他能够高效切菜、尽情施展厨艺，把每一道菜都做得又快又好吃。</p> 
<p>所以我们可以看到本次 WWDC 大会上，被拉来站台的卡普空高管就宣布将为 Mac 平台带来原生适配的《生化危机：村庄》，之后还将会带来《无人深空》等游戏。</p> 
<p class="image-wrapper"><img data-img-size-val="955,564" src="https://img.36krcdn.com/20220608/v2_50cb8d162ba14416b47e8b1e3f149e8d_img_000" referrerpolicy="no-referrer"></p> 
<p>因此果子也在此预测一波：随着 Mac 平台高性能图形 API 接口的推出，未来适配 Mac 的游戏引擎和游戏大作将会越来越多，毕竟 M 系列芯片的强大 GPU 性能已经让 Mac 在游戏市场中有了站起来的能力啦~</p> 
<h2><strong>为未来发布 VR 眼镜铺路</strong></h2> 
<p>苹果本次为 macOS 带来 Metal 3 等一些图形 API 接口，仅从表面来看是为了改变 Mac 平台缺乏游戏的问题，但是果子却认为苹果的用意远远不止于此，这其实苹果在为未来发布 VR 眼镜铺路。</p> 
<p class="image-wrapper"><img data-img-size-val="974,580" src="https://img.36krcdn.com/20220608/v2_2b0012226f7c4005ac9d6f2c6fb93b1c_img_000" referrerpolicy="no-referrer"></p> 
<p>如同此前知名苹果产品爆料人郭先生所说，苹果 VR 眼镜的目标是为了在十年内取代 iPhone 的地位，初代产品主打游戏等娱乐需求。</p> 
<p>那么让我们回看当下的 VR 市场，目前 VR 游戏主要有两种形式。</p> 
<p>一体机 VR：这种模式不需要其他额外运算设备，仅需要 VR 设备本体即可运行游戏，由 VR 设备内的处理器运算，日常使用十分方便。</p> 
<p>但是也因为一体机 VR 性能普遍较差（目前最强的一体机 VR 芯片也就只是等同于骁龙 865 的 XR2 芯片），所以游戏画质、流畅度还有较大进步空间。</p> 
<p class="image-wrapper"><img data-img-size-val="1070,566" src="https://img.36krcdn.com/20220608/v2_9b41829c1f2b4f49bc5e59ec39032079_img_000" referrerpolicy="no-referrer"></p> 
<p>PC VR：相比画质较差的一体机 VR 游戏，需要高性能 PC 运行游戏，然后通过有线线缆、无线方式串流到 VR 设备中的 PC VR 游戏，有着更高的流畅度和更出色的画面表现。</p> 
<p>根据此前关于苹果 VR 眼镜的相关爆料、渲染图来看，苹果 VR 眼镜一定会是一款走一体机 VR 路线的产品。</p> 
<p>然而因为一体机 VR 游戏对设备机能、散热要求较高，由外接设备运行的 VR 游戏可玩性更强，所以苹果 VR 眼镜也必然和 Quest 2、Pico Neo 3 一样支持串流 PC VR 游戏。</p> 
<p class="image-wrapper"><img data-img-size-val="1046,577" src="https://img.36krcdn.com/20220608/v2_ed8d7e75baef463d9629dce3e2a078cf_img_000" referrerpolicy="no-referrer"></p> 
<p>不过苹果显然不会白白放过建立下一个生态壁垒（自有 VR 游戏生态）的机会，因此苹果目前最重要的工作便是将 Mac 的游戏生态建立起来，为后续搭建自家的 Mac VR 游戏生态而铺路。</p> 
<p>当下发布为 macOS Ventura 带来暌违 5 年的 Metal 3 API 就是最好的证据之一，只要能够借此推动更多的游戏引擎开发商、游戏开发商加入到 Mac 生态当中，那么苹果后期实现在游戏领域的翻身未尝不可能。</p> 
<p class="image-wrapper"><img data-img-size-val="967,510" src="https://img.36krcdn.com/20220608/v2_195ff838907444a881159ba908d0a465_img_000" referrerpolicy="no-referrer"></p> 
<p>所以总的来说，Metal 3 API 的发布其实不仅仅为是解决 macOS 游戏短缺而来，或许短期内可能会大幅改善 macOS 平台游戏少的问题，但是苹果醉翁之意不在酒，最终的目的是为了传闻中 VR 眼镜而铺路。</p> 
<p>坦率地讲，果子本人对苹果 VR 眼镜还是非常期待的，尽管传言称它的定价高达 3000 美元，但是如果能够如爆料所说拥有那么奢华的配置，吃半年的土买它也未尝不可。</p> 
<p class="image-wrapper"><img data-img-size-val="1021,576" src="https://img.36krcdn.com/20220608/v2_5391764bda084afdba5492bad8b27466_img_000" referrerpolicy="no-referrer"></p> 
<p>那么假如最终苹果 VR 眼镜定价 3000 美元（约 21000 元人民币）的话，你会选择购买吗？</p>
