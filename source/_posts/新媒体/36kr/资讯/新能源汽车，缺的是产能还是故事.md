  <p><strong style="font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;">各地政府给钱、给土地、给政策，为造车新势力续命，扩大产能，讲更大的故事。</strong></p>

<p><img src="https://img.36krcdn.com/20220413/v2_f7bd0f9f34e74040b5dff1a7386b91f7_img_jpg" data-img-size-val="2974,1472" referrerpolicy="no-referrer"></p> 
<p><span style="color: rgba(0, 0, 0, 0.63); font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif; letter-spacing: 2px;">  因为疫情，新能源汽车交付雪上加霜。</span></p> 
<p style="white-space: normal;margin-left: 8px;margin-right: 8px;line-height: 1.75em;"><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">上周末，因为吉林、上海、江苏的供应链陆续停产，蔚来宣布整车生产暂停。<a class="project-link" data-id="1679800745218825" data-name="长城汽车" data-logo="https://img.36krcdn.com/20220401/v2_4fb498dee67545ac8f18421b69398ca2_img_000" data-refer-type="1" href="https://36kr.com/project/1679800745218825" target="_blank">长城汽车</a>在11号也对外表示，因为供应链影响，公司产能受限。</span></p> 
<p style="white-space: normal;margin-left: 8px;margin-right: 8px;line-height: 1.75em;"><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;"><a class="project-link" data-id="1678523609854980" data-name="比亚迪" data-logo="https://img.36krcdn.com/20220331/v2_b97e2db3334d4ec4800582be94d562f5_img_000" data-refer-type="1" href="https://36kr.com/project/1678523609854980" target="_blank">比亚迪</a>在十天前宣布停产燃油车，All in新能源车。业内人士分析认为，这是比亚迪为交付积压的40万新能源车订单做出的取舍。</span></p> 
<p style="white-space: normal;margin-left: 8px;margin-right: 8px;line-height: 1.75em;"><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">新能源车企都很难。</span></p> 
<p style="white-space: normal;margin-left: 8px;margin-right: 8px;line-height: 1.75em;"><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">交付困难，甚至因疫情停产的同时，新能源车企也在疯狂投资建厂。</span></p> 
<p style="white-space: normal;margin-left: 8px;margin-right: 8px;line-height: 1.75em;"><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">重庆、武汉、北京亦庄、顺义、合肥、金华、上饶，新能源车企纷纷落子。<a class="project-link" data-id="1679639145206537" data-name="理想汽车" data-logo="https://img.36krcdn.com/20220401/v2_bc702cfb97f14e8ca5cf3917de5e781a_img_000" data-refer-type="1" href="https://36kr.com/project/1679639145206537" target="_blank">理想汽车</a>北京第二工厂刚刚确定，又传来第三工厂落户重庆的消息。<a class="project-link" data-id="1679736330195717" data-name="蔚来汽车" data-logo="https://img.36krcdn.com/20220401/v2_184520fc753d46c685c836a2d60cd5bc_img_000" data-refer-type="1" href="https://36kr.com/project/1679736330195717" target="_blank">蔚来汽车</a>在合肥的第二基地也将正式投产，<a class="project-link" data-id="1678405418005513" data-name="小鹏汽车" data-logo="https://img.36krcdn.com/20220329/v2_dbdeed1720f94457b0d3bf5086cb295b_img_png" data-refer-type="1" href="https://36kr.com/project/1678405418005513" target="_blank">小鹏汽车</a>在广州的工厂2022年底投产，武汉工厂也已敲定，2023年投产。</span></p> 
<p style="white-space: normal;margin-left: 8px;margin-right: 8px;line-height: 1.75em;"><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">新能源车企的战斗，从市场延伸到工厂。</span></p> 
<p style="white-space: normal;margin-left: 8px;margin-right: 8px;line-height: 1.75em;"><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">这一轮建厂大战，背后有什么逻辑和秘密？</span></p> 
<p style="white-space: normal;margin-left: 8px;margin-right: 8px;line-height: 1.75em;"><strong style="font-size: 20px; font-family: "Segoe UI", "Lucida Grande", Helvetica, Arial, "Microsoft YaHei", FreeSans, Arimo, "Droid Sans", "wenquanyi micro hei", "Hiragino Sans GB", "Hiragino Sans GB W3", FontAwesome, sans-serif; color: rgb(30, 86, 160); letter-spacing: 2px; text-align: left;">绑定地方，有政策有资金</strong></p> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <strong><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">新能源车企去地方建厂，排在第一位考量的是地方政府的资金支持。</span></strong> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">“新能源车企建厂，需要扩大产能是一方面，但是更重要的，是希望获得地方政府一些资金等方面的支持，让自己活下来，这个意义更大。”汽车行业业内人士李强表示，新能源车企建厂，虽然也需要花钱，但都是小钱，当地政府给的资金的扶持，包括后面税收的政策，完全可以覆盖建厂的成本。这是行业里公开的秘密。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">新能源汽车不盈利，是目前最大的问题。李强说，这些企业都缺钱，都在找钱，“你找政府投，政府为什么要投？你搬到他那里去，他就给你投。政府给了钱，车企能活下去，资方一看，某地政府有实力，也会进行跟进。”</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">他举了一个例子，某个这两年很火的国产汽车品牌，在江苏某地级市建了一个基地。该地级市最早是和吉利谈的，当时的条件是给30亿，吉利没同意，最后那个地方政府和某国产品牌谈定了。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <strong><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">除了资金的支持，土地、贷款、政府采购等政策条件，是新能源车企另一项重大选择理由。</span></strong> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">地方政府对于建厂车企，会在土地、贷款、税收上也会给一些优惠，因为新能源车企盈利能力弱，所以这些实实在在的政策，他们非常看重。对于车企来说，哪里的政府给的政策好，支持力度大，他们就去哪里。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">李强告诉陆玖财经：“地方政府会给车企一些在当地采购的支持。比如在本地更换出租车或者招投标的时候，都会为该企业量身定做设置一些条件，来进行支持。”</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">他进一步透露，爱驰在江西上饶建了一个生产基地，整个上饶政府甚至江西政府都会给爱驰汽车创造一些在当地销售的条件。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">这是各地政府通用的策略。比亚迪也是一样，深圳政府也会给很多支持，比如深圳的出租车，大部分都是比亚迪的，有油车也有电动车。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">不过据业内人士透露，这些扩产的投资，各地政府除了做非常严格的尽调之外，和车企签署的条款也有详细规定，一是专款专用，另外，政府也对车企有兜底的保障。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">那么，地方政府为什么要想尽办法吸引这些车企呢？</span> 
</section> 
<p style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">李强认为，这还是要追溯到2020年前后，国家的碳达峰和碳中和政策。国家大力支持，地方政府当然大力跟进。各地政府都在抢夺新能源之都的名号，<strong>能拉到这些车企，不仅仅是政绩，因为汽车产业链长，在未来的产业协同、经济拉动上，都会有长效的价值。</strong></span></p> 
<p style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"><strong style="caret-color: rgb(0, 0, 0); color: rgb(30, 86, 160); font-size: 20px; letter-spacing: 2px;">扩产能，讲更大的故事</strong></p> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <strong><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">除了拿到政府投资和好的优惠政策之外，扩产能对于融资也有巨大的拉动作用。</span></strong> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">安徽芜湖汽车行业人士苏晓冬向陆玖财经透露，一些新能源车企要上市，也需要讲更好的故事，所以也会扩厂建设。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">“比如零跑汽车，今年3月份已经向港交所提了上市申请。原来的设计产能相对来说比较保守，他们跟金华市政府谈了一块地，面积是按照25万产能去打造的。现在需要上市募资，大家可以根据25万的产能，算出来它未来的营业额。”</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">苏晓冬说，为了做高估值，零跑干两件事，<strong>第一件事就是拿土地建厂，把产能提上去。</strong>零跑第二工厂落户杭州大江东，根据公开报道，产能规划大概20万，这样，零跑就变成近50万的产能。除此之外，杭州国资向零跑投了30亿。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">“第二件事就是做新项目。近50万的产能，需要新项目支撑，要推出新的车型来支撑，你发现零跑汽车今年一下子上了4款车型，这4款车会分别在2022、2024年才陆陆续续批产，后年还会批产两款车。”</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">产能近50万，6款车型，这样的话，零跑的未来预期就很大了。相比现在的估值，投资人就会觉得它未来可期，对研发投入、融资上市很有帮助。“零跑现在是10多万的销售量，未来会有50万的产能，看起来是5倍的增长，这个资本故事就很好看了。”</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">某Q打头的国产新能源车企旗下有两款中低端车，其中一款月销过万，这两款车都是在安徽某地生产，去年开始，该车企就在跟安徽另一个地级市谈建厂的问题，今年已经开始盖厂房了，两款车中的另一款车型，未来就在该地生产。</span> 
</section> 
<p style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">“当地政府提供土地、建厂等支持，生产一台车，会补贴近3000元，除此之外，还有税收上的一些支持。”这家车企计划明年上市，苏晓冬认为，两个车型分开生产的目的，一个重要的原因就是为了上市，另外就是为了拿补贴。</span></p> 
<p label="正文" style><strong style="caret-color: rgb(0, 0, 0); color: rgb(30, 86, 160); font-size: 20px; letter-spacing: 2px;"> 这一波狂奔会不会一地鸡毛？</strong></p> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">目前来看，新能源车市场面临交付难的困境，那么，这一波新能源车企的扩产能冲动，能否大大缓解当前供不应求的市场现状？</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">其实不然。李强认为，<strong>新能源车企这一轮交付慢，不是因为产能，更大的原因还是因为上游供应链的问题。</strong></span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">“这两天行业内有一个大新闻，就是中国最大的线束厂安波福停产了，这种上游零件厂商，或者因为疫情影响，或者因为封城政策影响，影响了生产，加上芯片短缺，直接导致了汽车交付问题。”李强说，“实际上，<strong>除了特斯拉，还没听说过哪一个工厂因为产能问题影响交付的。</strong>蔚小理去年大体销量也就十万台左右，任何一个基地，都可以完成这个产能。”</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">一款燃油车需要300-500颗芯片，一款新能源汽车，要用到2000多颗芯片，陆玖财经了解到，最近市场很缺eps系统（电动助力转向）需要的ecu芯片，导致包括理想在内的很多车企交不出车来。燃油车的交付也面临着同样的问题。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">苏晓冬认为，很多新能源车的质量还有很多需要提高的地方。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">“我们原来的传统车企，都想的是怎么把质量做好，材料用得好，把性能等这些硬的东西提高上去，结果新能源车企认为，汽车就是一个壳子加四个轮子，他们着力在智能化、网联化上。很多新能源车企的代工厂，找的都是原来一些低端车的工厂，硬的东西做得还是不行。”</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">去年年底，某巨头推出合作的第一款智能汽车，苏晓冬说，他正好那段时间去重庆，看了生产该汽车的工厂，“巨头的技术是挺好的，但你要想，它的代工厂是哪里，以前是生产面包车的，<strong>工厂的硬件比较差，供应链也很一般，原来的工艺人员都是做面包车的，</strong>这辆车我还试驾了，感觉一般。起码比奇瑞、吉利差。”</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">不过，新能源车企新势力引领了一波风向，苏晓冬说，现在很多传统车企也开始在智能化网联化和内饰上下功夫了。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">新能源汽车产能过剩，已经屡屡被行业提及。根据乘用车市场信息联席会年初的数据，2021年我国新能源乘用车的专用产能已有569.5万辆，产能利用率约为58.4%。潍柴动力董事长谭旭光在公司2021年业绩发布会上，就曾提到过“新能源无序资本膨胀”的问题。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">3月底，在第八届中国电动汽车百人会论坛上，国家发改委相关领导表示：新能源汽车现有基地达到合理规模前，不再新增产能布点。政府层面开始为新能源车企扩建踩刹车。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">兼并重组、盘活闲置资产提高产能，是比较务实的做法，在今年的两会上，全国人大代表、原长城汽车总裁王凤英将此做法作为两会提案之一。收购众泰临沂生产基地、接管猎豹在荆门的生产基地等，长城汽车也是这么做的。</span> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">“地方政府普遍希望，把原有的经营不善的汽车厂进行技改重建，他们也会追加资金支持技改。”上述业内人士说。</span>

 <strong><span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">那么，如今的建厂热潮，会不会只是新能源车企的蒙眼狂奔，未来会不会出现产能过剩的情况？</span></strong> 
</section> 
<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">李强告诉陆玖财经，其实在2020年，国家层面曾就双资质新能源汽车，发过十几张牌照，希望通过这种方式，能够带动燃油车企做新能源。目前看来，那些拿到资质的企业，活下来的寥寥无几。</span>

<section style="white-space: normal;line-height: 1.75em;margin-left: 8px;margin-right: 8px;"> 
 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">在2020年，江苏、江西两个省份都曾在新能源车企的引入上栽过跟头，成为反面教材。拜腾汽车自2017年成立以来，融资大概84亿元，却未能造出一台量产车。赛麟、博郡等新能源车企都被央视点名批评。</span>

 <span style="color: rgba(0, 0, 0, 0.63);font-family: -apple-system-font, "system-ui", "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;background-color: #FFFFFF;letter-spacing: 2px;">这轮新能源车企扩张大潮，未来会不会也是一地鸡毛？</span>
