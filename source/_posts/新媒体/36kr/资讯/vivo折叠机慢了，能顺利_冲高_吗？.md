<p>在中国已经热闹起来折叠屏市场上，今年4月才入场的vivo属于“迟到者”。 </p> 
<p>2018年10月，<a class="project-link" data-id="1678510974219273" data-name="柔宇科技" data-logo="https://img.36krcdn.com/20220331/v2_a48f7502a6bb47179b9fff90421575af_img_000" data-refer-type="1" href="https://36kr.com/project/1678510974219273" target="_blank">柔宇科技</a>发布了国内首款折叠屏FlexPai，同年，在NEX系列新机发布会前，vivo执行副总裁胡柏山向媒体表示，自己并不看好折叠手机短期前景。</p> 
<p>vivo也的确没有着急推出折叠机。查看其产品副总裁黄韬近期发布的微博可知，vivo在过去四年时间，也就是<strong>从2018年到2022年，一直在研发和布局折叠屏相关的业务和技术。迟迟未推出产品，是因为在几十版方案中也未找到更接近“完美”的选项。</strong></p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220413/v2_b876f088d86d43b897c0b7dab440d9b0_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">vivo产品副总裁黄韬，图源发布会直播间 </p> 
<p>终于在2022年4月11日，作为vivo首款折叠机vivo X Fold姗姗来迟，售价仅8999元起。至此，除了苹果以外，中国主流手机厂商全部完成“折叠”梦。</p> 
<p>那么<strong>这款打磨良久的vivo X Fold，究竟有没有大招？能不能打动市场消费者？</strong></p> 
<p><strong>多位使用折叠屏手机的机主均肯定了vivo X Fold的铰链技术，但并没有打算购买该款新机。</strong></p> 
<p>其中一位使用华为MateX和三星Galaxy Z Fold3 5G的机主对连线Insight说：“X Fold虽然定位人群为高端商务人士，但对我来说，我只认可三星和华为这两个品牌的折叠手机，vivo的品牌力还有很大的提升空间。”</p> 
<p>一些手机经销商，也未把vivo X Fold纳入其销售范围内。一位经销商对连线Insight坦言：<strong>“折叠屏在五线城市的市场规模不算大，而且除了华为，其他国产手机厂商的折叠机价格跳水很快。我目前暂时不售卖vivo X Fold。”</strong></p> 
<p>但不论用户和经销商对vivo X Fold的评价如何，vivo也到了必须推出折叠屏手机的时候了。</p> 
<p>仅2022年前后一个月内，OPPO和荣耀相继分别推出各家首款折叠屏手机OPPO Find N和Magic V。并且市场研究机构CINNO Research在年初发布报告，称2022年全球市场折叠屏智能手机销量，将同比增长107%。<strong>不少业内人士认为，今年折叠屏手机有计划迎来新转折点。</strong></p> 
<p>更重要的一点是，<strong>vivo急需一款定位高端且高价的折叠屏手机，弥补自己在高端手机领域的影响力。</strong></p> 
<p>值得注意的是，vivo推出首款折叠机后，三星就包揽了小米OV的屏幕供应，成为隐藏背后的最大赢家。</p> 
<p>众所周知，柔性AMOLED屏是折叠屏手机中最关键的部件之一，全球折叠机应用面板几乎都以三星为主，此次vivo与三星合作也不难理解。</p> 
<p>随着国内头部手机厂商齐聚折叠屏市场，今年折叠机或将迎来新一轮较量。“迟到”的vivo能否真正做到“厚积薄发”，尚且需一段时间才能有答案。</p> 
<h2><strong>vivo的折叠故事新吗？ </strong></h2> 
<p><strong>起码在vivo看来，vivo X Fold这款产品足以打动自己。</strong></p> 
<p>2018年国内出现第一款折叠屏手机时，胡柏山就曾坦诚表示，vivo已经在着手折叠屏手机的相关研发工作，并且持续几年布局该业务，希望能找到更接近“完美”的产品方案。</p> 
<p>直到2022年，vivo才认为“时机”终于成熟，<strong>在品牌副总裁贾净东看来，至少vivo X Fold做到打动了vivo自己。</strong></p> 
<p>就在3月28日vivo官宣vivo X Fold在4月11日发布的当天，贾净东也在微博发布了一篇文章《谈谈vivo的“大先生”》，文章中的主角“<strong>大先生”就是vivo X Fold。</strong></p> 
<p>贾净东解释“大先生”的“大”有两层含义。<strong>第一层含义的“大”是折叠后展开的屏幕大得专业、大得惊艳；另一层含义的“大”是做大事情的目标客户。</strong>也就是vivo之前产品系列未能覆盖的高端商务人群。</p> 
<p>而且，与同样想冲击高端市场的小米、荣耀和OPPO相比，vivo X Fold的确大而全。</p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220413/v2_ce4707b8e598425da3fefc079a0e9520_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">发布会主题“集大成者”，图源发布会直播间 </p> 
<p><strong>vivo X Fold的主推宣传点是屏幕铰链方案。</strong>铰链的功能类似于门的合页，是折叠屏手机的关键硬件之一，它直接影响手机折痕程度是否明显。</p> 
<p><strong>据发布会透露，vivo X Fold经过多代方案迭代，最终采用浮翼式铰链，托起凹陷的屏幕，基本可以做到无折痕。</strong></p> 
<p>这一铰链解决方案，在当下国产折叠机市场中的确比较“能打”。小米mix fold采用的铰链工艺是精密铰链，荣耀Magic V采用水滴铰链；OPPO Find N采用拟锥式铰链。</p> 
<p>但申万宏源的报告同时也指出，vivo X Fold铰链因添加了航天材料，成本预计超1200元，属于“荣米OV”中成本最高的一家。</p> 
<p>除了铰链，<strong>在屏幕材质方面，vivo与OPPO Find N选择相同，采用了较先进的UTG（超薄玻璃）工艺，能使屏幕折叠时比较耐磨。</strong></p> 
<p>目前，折叠机的屏幕材质可分为CPI和UTG两种。小米Mix fold、荣耀Magic V的屏幕材质就采用了传统的透明聚酰亚胺（CPI）。CPI好处是成本低，但“塑料感”较重且容易有划痕。</p> 
<p>在屏幕体验方面，根据发布会信息，<strong>vivo X Fold属于行业第一款带屏幕指纹的折叠屏手机，</strong>相比之下，其他不少厂商的折叠机指纹设计为侧边栏指纹。</p> 
<p>在国产手机们纷纷冲击高端的背景下，尽可能提高屏幕刷新率，早就成为国产手机厂商们极力表现的方向。vivo X Fold则将“内卷”再次推向新高度，其内外屏幕均采用120HZ双高刷新率。</p> 
<p><strong>除了屏幕折痕优化、铰链耐久性和指纹技术具备超前性，vivo X Fold在芯片处理器、影像、续航等更多配置方面，属于市场主流水平。</strong></p> 
<p>冲击高端，芯片是硬通货之一。<strong>vivo X Fold号称搭载骁龙8 Gen1 SPU定制旗舰芯片，比标准骁龙8多了一个“SPU”</strong>（安全处理单元），可以提供基于底层硬件的安全解决方案。<strong>但消费者对于这一差异点并不买账。</strong></p> 
<p>不少网友得知搭载骁龙8 Gen1芯片时，在发布会直播间评论“就冲这一芯片，我就决定等下一代了，一个优秀的芯片是手机长久使用的基础”“搭载骁龙8，其实可以选择pass这款新机了”。<strong>众所周知，骁龙8 Gen1芯片夸张的功耗和散热难题，一直被众多消费者吐槽。</strong></p> 
<p><strong>出于堆料的需求，vivo X Fold屏幕体型大且重。</strong>外屏达到6.53英寸，是许多直板手机的主屏尺寸。展开后的内屏尺寸是8.03英寸，宽度达到144.87mm，重量约311g，<strong>属于“荣米OV”内屏尺寸、面积最大的一家。</strong></p> 
<p>可以参考的一组数据是，苹果平板iPad mini第一代到第五代都是7.9英寸的屏幕，近三代的iPad mini重量均在300克以下。</p> 
<p><strong>但身材“发福”的vivo X Fold并没有配上更大容量电池，</strong>手机重量同样超过300克的小米MIX Fold，电池容量5020mAh，vivo X Fold比前者少了420mAh。</p> 
<p>为了赢得商务人士的青睐，vivo X Fold还加入vivo系列之前没有的物理静音键，这也是iPhone独有的特色。iPhone一直以来坚持保留的静音键，是专为会议场合而生的一个小设计。</p> 
<p>或许是怕引发搬运iPhone设计的质疑，<strong>vivo X Fold的静音键略有些改动——将上下推动的条型，改成了左右推拉的圆型设计。</strong></p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220413/v2_bbfb7e7314a14f6891ef6223e3b4ff91_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">vivo X Fold的静音键，图源vivo官方微博 </p> 
<p><strong>影像方面，vivo X Fold带来了用户期待已久的潜望式长焦镜头。</strong>很多普通手机用户第一次拍月亮，就是通过手机的潜望式长焦镜头实现的。</p> 
<p>潜望式长焦镜头曾是安卓手机厂商角逐高端市场的一条重要赛道，几乎是各家安卓旗舰的“标配”、产品发布会上最浓墨重彩地介绍环节之一。但因物料成本高、占用较多手机内部空间和机身重量等问题，成为了诸多手机产品经理们规划新机时的“非必要”选项。</p> 
<p><strong>但vivo X Fold的潜望式长焦镜头是2020年小米10青春版同款潜望式长焦镜头，最高50倍变焦，仅有800万像素。</strong>目前市场上，有些手机潜望式长焦镜头像素已经可达6400万像素。</p> 
<p>需要了解一点，潜望式长焦的焦距要比主摄长很多，这意味着前者对防抖性能的要求更高。<strong>但在vivo X Fold上，vivo旗舰机标配的微云台防抖居然没有出现。</strong></p> 
<p>因此，<strong>vivo X Fold影像的亮点是蔡司T*镀膜标准和超长焦镜头，但整体表现在旗舰机里或属于一般水平。</strong></p> 
<p>折叠屏已经成为发展高端市场的必备操作，通过“秀肌肉”展现技术和产品实力，进而拉拢用户。vivo作为最后入场者，在铰链等屏幕技术方面的确有明显创新，但这并不意味着vivo X Fold就能成为受消费者欢迎的产品。</p> 
<h2><strong>折叠机能撑起vivo高端路吗？ </strong></h2> 
<p><strong>折叠屏，对vivo实现高端梦很重要。</strong></p> 
<p>“荣耀OV”等国产手机品牌都从中低端起家，之后才转向高端市场，远落后华为、苹果一大截。</p> 
<p>为了加强自身砝码，国产手机厂商去年一年使劲浑身解数。<strong>小米2021年祭出尘封许久的MIX系列，并接着推出首款折叠屏手机持续冲击高端；OPPO声势浩大的推出Find X3系列。</strong>反观vivo，它在2021年几乎没有什么高举高打的大动作。</p> 
<p>其实vivo在高端市场的探索也未停止，包括vivo X Fold。只不过，<strong>vivo一直属于“落后者”中的“慢跑者”。</strong></p> 
<p>国产手机很早之前就出现用旗舰机型进入高端市场的做法。比如OPPO十年前生产第一代Find的时候，就开始了高端之路；2012年和2015年，华为分别推出了主打高端市场的P系列和Mate系列；小米在2016年10月推出MIX系列；荣耀紧随其后，两个月后火速推出对标华为 Mate、P 系列的 Magic。</p> 
<p><strong>vivo则有些“慢”。在2018年之前，它没有真正主打高端市场的产品线，</strong>当时的产品规划是X Play为旗舰系列；X作为中端系列，是主力出货机型；Y则是低端入门系列。并且从2016年开始，X Play再无新机发布。</p> 
<p><strong>vivo在高端路线的布局转折点出现在2018年6月，NEX旗舰系列姗姗来迟，成为vivo进入高端市场的敲门砖。</strong>为了体现该系列的“高端”调性，NEX定下了最低接近4000元的价格。</p> 
<p>彼时胡柏山对外宣布，NEX系列上市后半年，市场销量超200万，成为4000元以上价位（高端手机价格带）中卖的最好的产品系列，带来80亿收入。</p> 
<p>此后，NEX每隔几个月便发布一款新品，相继推出NEX双屏版、NEX 3 5G、NEX 3s等产品。但后几款产品表现越来越差，比如2020年推出的NEX 3s开售不久价格便跳水。即便瀑布屏在当时很有新意，但消费者并不买单。</p> 
<p><strong>NEX 3s也成为NEX系列最后一次在国内手机市场上亮相的机型。</strong></p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220413/v2_b79e9964d91e4f5a9e59405cefd3a01b_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">NEX双屏版，图源NEX智慧旗舰 </p> 
<p><strong>市场和消费者当年也用实际行动给出了自己的答案。</strong></p> 
<p>从IDC发布的2020年上半年中国600美元（约合人民币3789元）以上高端手机市场排名报告来看，华为与苹果两家占了88.1%，小米、OPPO和三星所占份额分别为4%、2.6%、2.5%。<strong>至于vivo，则被列进了“Other”之中，侧面反映了NEX系列销量惨淡。</strong></p> 
<p>之后的两年时间，即便荣耀、小米、OPPO在高端系列百花齐放，vivo也只是时不时传出NEX新机发布延期的消息，将主要精力放在深耕中高端X系列。</p> 
<p>直到2022年3月，vivo被曝出NEX及事业部已经被撤销，对此vivo品牌副总裁贾净东回应称，未来将由X系列担任旗舰化重任，<strong>“对NEX的任何期待，都可以在X系列新品上得到答案”。至此，vivo高端产品线正式由NEX变为X。</strong></p> 
<p>显然，vivo布局高端市场，走出了一条完全不同于友商的路线——<strong>通过提升X系列定位，最终让其跻身旗舰行列。S系列继承了原先X系列的定位。</strong></p> 
<p>升级后的X系列急需一款有足够产品差异的新机，既可以在X系列中加入一条全新产品线，又不会影响S系列的产品布局。已经准备4年的折叠屏手机，是最合适也是唯一的选择，也是为数不多能让消费者期待的产品。 </p> 
<p>对于vivo而言，X Fold是一个全新品类，它可以更好地弥补自己高端手机领域的影响力。毕竟荣耀、小米和OPPO都通过这一方法树立了自己高端形象，并将产品成功“特色化”。</p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220413/v2_0760707175264aa598e27ae01b423ee0_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">vivo X Fold，图源vivo官方微博 </p> 
<p>不过，一位折叠机用户向连线Insight坦言：“使用折叠屏手机的主要群体就是高端商务人士，这类小众群体消费力高，但对产品要求也很苛刻，比较钟爱三星、华为等旗下折叠机产品成熟的高端品牌。”</p> 
<p>对错过入局黄金点的vivo来说，最后一个入场，压力不小。加之缺乏足够亮眼的自研技术，背负vivo高端梦的X Fold，未来之路显然并不太明朗。</p> 
<h2><strong>折叠机爆发背后，三星是最大赢家？ </strong></h2> 
<p>今年以来，有关折叠机的利好消息不断传出，不少业内人士认为，2022年或可能是折叠屏手机的转折点。IDC数据则显示，可折叠设备在2021年占领高端安卓手机1/3的市场。</p> 
<p><strong>在“折叠屏”概念产品爆发的前夜，不仅意味着国产手机厂商提高高端市场份额的机会有望增加，对折叠屏核心零部件的产业链更是一大利多。</strong></p> 
<p>作为折叠屏的“门面”，国金证券指出，<strong>折叠屏手机推出之后，最先受益的供应商当属OLED面板厂商。</strong>这意味着，既做厂商又做供应链的折叠机老玩家三星，或将成为最大赢家。</p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220413/v2_10c7983d334942c7bff8856e9143cd9b_img_000" referrerpolicy="no-referrer"></p> 
<p>首先来看目前布局柔性AMOLED的国内企业，主要有<a class="project-link" data-id="1678272160723973" data-name="京东" data-logo="https://img.36krcdn.com/20220331/v2_c05e240eba464dd28d00b16923addd7a_img_000" data-refer-type="1" href="https://36kr.com/project/1678272160723973" target="_blank">京东</a>方、<a class="project-link" data-id="1679691066757894" data-name="维信诺" data-logo="https://img.36krcdn.com/20220401/v2_d2ec7d6bda294a36b63449cd064d2739_img_000" data-refer-type="1" href="https://36kr.com/project/1679691066757894" target="_blank">维信诺</a>、深天马、TCL科技旗下华星光电和柔宇科技。</p> 
<p>其中，京东方是华为Mate Xs、Mate X折叠屏手机的独家供应商，并为OPPO Find N提供外屏；TCL华星光电为小米MIX FOL提供屏幕。</p> 
<p>可以看出，<strong>目前国内显示屏厂商已经能满足折叠机市场的部分需求。</strong>群智咨询副总经理兼首席分析师陈军接受北京商报采访时分析：“京东方、TCL华星、天马、维信诺等面板厂柔性OLED基本上已量产，国内整个OLED技术水平达到了品质比较高的水平。”</p> 
<p><strong>但从折叠机整体市场来看，三星屏幕仍是当下手机厂商的主流选择。</strong>在全球范围内，三星自研的柔性OLED面板无论是研发技术，还是市占率，都处于领先水平。</p> 
<p>2020年，市场调研机构群智咨询曾发布了2019年全球AMOLED智能手机面板出货量排名，其中三星位列全球第一。</p> 
<p>OPPO Find N的内屏以及vivo X Fold内外屏，均由三星提供。并且不少科技博主爆料，小米第二款折叠机的屏幕，或也将由三星提供。</p> 
<p>并且，<strong>连线Insight梳理“荣米OV”折叠机发布时间后发现，四家手机厂商均在三星放开折叠屏幕面板业务，开始对外销售折叠屏面板后，才陆续推出折叠机。这之前，三星显示仅向三星电子出售折叠屏元件。</strong></p> 
<p>国内这场折叠机产品狂欢的背后，存在着一个所有国产手机厂商都无法回避的问题：<strong>部分国产手机厂商是借用三星折叠屏，才有了折叠屏手机市场的入场券。</strong></p> 
<p>这也意味着，在折叠屏领域，国内显示屏厂商并未与三星的屏幕技术拉开距离。</p> 
<p>折叠屏手机已经成为缓解国内手机厂商增长焦虑的“救命”良方，即便重度依赖三星，它们还是要以抢占市场先机为主，不能停下脚步。</p> 
<p>不论未来各大国产折叠机的市场份额孰高孰低，它们背后的折叠屏供应商三星已经“躺赢”。但华为留下的启示“用自研敲开高端的大门”需刻在国产手机厂商的DNA里，提高自主研发能力，是重中之重。</p>
