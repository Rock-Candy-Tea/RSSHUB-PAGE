<p>近日，Linux基金会宣布启动Agstack基金会。这是一个非盈利性质的基金会，主要针对全球农业生态系的开源数位基础设施专案。</p> 
<p>未来，将会创建、维护和强化用于农业数据和应用程序的开放数字基础设施，旨在促进全球农业领域相关者之间的合作，包括私营企业、政府和学术界，以此来提高全球农业效率。</p> 
<p class="image-wrapper"><img data-img-size-val="1080,418" src="https://img.36krcdn.com/20210508/v2_4eed138de9384f0da1f683d70abaddbe_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">Agstack基金会官网：https://agstack.org/</p> 
<h2>当开源走进农业，会带来什么冲击？</h2> 
<p>随着新型技术的发展，数字化技术逐渐深入到农业产业各个环节。人工智能与自动化工具、联网设备等，帮助农业领域优化了作物生长，规避了劳动力短缺等障碍。</p> 
<p>而随着“数字鸿沟”的不断缩小，开源技术也渗透到了乡间田野，带来<a class="project-link" data-id="4260438" data-name="了数" data-logo="https://img.36krcdn.com/20210422/v2_8e636ec7be434dd5bf7deebc8bed2b62_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/4260438" target="_blank">了数</a>据和工具的额外好处，降低了数据和工具的使用门槛，帮助企业避免受困于由少数公司运营的专有软件。</p> 
<p>AgStack基金会的创始成员来自各个领域。有来自科技公司的，也有许多来自农业企业的成员。合作伙伴和核心成员包括Agralogics、Call for Code、Digital Green、Farm基金会、farmOS、HPE、IBM和NIAB等，另外还有来自加州大学农业与自然资源学院、普渡大学农业资讯实验室等学术单位。</p> 
<p>相较于其他早已收到互联网及新技术冲击的领域，全球农业数字化仍属于初步阶段。由于效率低下，回报周期长，其面临的种种问题都等待着转型：</p> 
<ul> 
 <li>资源限制（劳动力、水）</li> 
 <li>经济依赖（供应方）</li> 
 <li>人口增长（需求方）</li> 
 <li>气候变化（外因）</li> 
 <li>消费者对食品/健康的态度（需求方面）</li> 
</ul> 
<p>因此，全球农业领域也是一个价值极高的生态系统，农业数字化有着巨大的前景，有望以实现差异化的规模和速度，推进包容性农业转型。但由于历史遗留问题，这种转变的实现进度缓慢。</p> 
<p>Linux基金会之所以成立AgStack，是认为全球生产的粮食中有33%被浪费了，但世界上还有9%的人还处在挨饿或营养不良的生活状态中。导致这种结果的主要原因是，传统农业系统效率低下，在日益增长和复杂的农业工业链中举步维艰。</p> 
<p>AgStack执行董事Sumer Johal表示，如今全球农业生态系统迫切需要数字化，因为缺乏可重复使用的工具和资料，损失了许多生产力和创新力。AgStack将使用开源软件，推动创新农业的效率和规模。为了进一步降低农业数字化的门槛，用全球可访问、可开源、可重复使用和专业化的数字基础设施，来降低成本和减少磨合，从而加速这种转变，实现效率提升。</p> 
<h2>不产生数据和软件，专注构建和管理平台</h2> 
<p>如今，AgStack建立了公开的存储库，可以用于创建和发布模型，提供免费的公共资料及可互通的框架、可供跨专案使用和特定目的的扩种套件和工具箱。</p> 
<p>同时，这些成果都是构建在现有的技术与基础之上的，包括AgGateway、UN-FAO和CAFA等农业基础，、Sentinel、NOAA和Soilgrids等公共资料。在开源项目上，包含的也是开发者们比较熟悉的Hyperledger、Kubernetes、Open Horizon等等。</p> 
<p class="image-wrapper"><img data-img-size-val="1080,496" src="https://img.36krcdn.com/20210508/v2_9859b67f70a4443a865e20193864e9dc_img_000" referrerpolicy="no-referrer"></p> 
<p>并且，AgStack基金会不会参与构建软件应用程序，而是专注于成员和用户构建、管理和运行应用程序所需的软件基础架构。</p> 
<p>AgStack基金会的技术项目将提供一套丰富的软件、数据工具和框架，使得农业生态系统的内容创建者和内容消费者能轻松、通用地访问。包括以下服务：</p> 
<ul> 
 <li>开放和免费提供可发布和可重复使用的软件代码，如数百万个与农业相关的害虫、农作物、天气、动物等模型；</li> 
 <li>通过程序访问更多的免费公共和专有数据源；</li> 
 <li>开放和免费访问软件应用框架，以创建、贡献或使用这些模型和数据（适用于Web和移动设备）；</li> 
 <li>为特定项目提供词典及实用程序的扩展程序和工具箱</li> 
</ul> 
<p class="image-wrapper"><img data-img-size-val="1080,394" src="https://img.36krcdn.com/20210508/v2_e0a0d8ec61e04e1792bab76065eee7d6_img_000" referrerpolicy="no-referrer"></p> 
<p>“我们很高兴为AgStack的构建和发展提供论坛，” Linux基金会项目总经理兼高级副总裁Mike Dolan说。“通过使用开源软件来标准化农业数字基础设施，AgStack可以降低成本，加速集成并实现创新。看到像农业这样的行业使用开源原则进行创新，真是太神奇了。”</p> 
<h2>那些有意思的开源项目</h2> 
<p>如果说开源是农业数字化进程中的必经之路，听起来还比较合理，但如果你觉得开源只能做这些事情，那可能就“So Young”了。</p> 
<p>虽然开源项目和商业项目有着一定程度的冲突，但也无法阻止技术人的热情和脑洞。</p> 
<p>比如SpaceX。马斯克造火箭这事儿已经不是什么稀奇事，但在去年5月31日SpaceX载人“龙飞船”将NASA宇航员鲍勃·贝肯和道格·霍利成功送入太空后，SpaceX粉丝组织r/SpaceX随之也在Github上开源了SpaceX火箭相关的数据——SpaceX-API，果然是敢想敢做。</p> 
<p>SpaceX-API 是一个用于火箭、核心舱、太空舱、发射台和发射数据的开源 REST API。REST API包含三大部分，clients、app(应用)和原始数据：</p> 
<p class="image-wrapper"><img data-img-size-val="865,1079" src="https://img.36krcdn.com/20210508/v2_6ef50ae53e26414bac3fd17aa092d14f_img_000" referrerpolicy="no-referrer"></p> 
<p>他们整理的API，基本包含了SPaceX成立迄今的所有火箭数据。API包含SpaceX的龙飞船、Starship、发射历史数据、核心舱、起落架、载荷信息等等，应有尽有。</p> 
<p class="image-wrapper"><img data-img-size-val="641,329" src="https://img.36krcdn.com/20210508/v2_c53a376eab7c4d13bbf4dd5672da25a9_img_000" referrerpolicy="no-referrer"></p> 
<p>再比如<a class="project-link" data-id="3969182" data-name="英伟达" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3969182" target="_blank">英伟达</a>将人脸生成器StyleGAN2开源后，登上了Github趋势榜第一。StyleGAN2重点修复artifacts问题，并进一步提高了生成图像的质量。一步步地生成人工的图像，从非常低的分辨率开始，一直到高分辨率（1024×1024），通过分别地修改网络中每个级别的输入，控制在该级别中所表示的视觉特征，从粗糙的特征（姿势、面部形状）到精细的细节（头发颜色），而不会影响其它的级别。有种“徒手捏脸”的快感。</p> 
<p>StyleGAN2一经开源，点燃了一众开发者的热情，开始纷纷“创造”出大量真假难辨的人脸。</p> 
<p class="image-wrapper"><img data-img-size-val="800,314" src="https://img.36krcdn.com/20210508/v2_ab06627ed4e94b4890a9d6d9d8320fe9_img_000" referrerpolicy="no-referrer"></p> 
<p>还有一些令人意想不到的开源项目，比如用GitHub生成书籍：GitbookIO/gitbook · GitHub</p> 
<p>根据个人 Github 信息生成个人简历 resume/http://resume.github.com · GitHub 等等……</p> 
<p>这么看来，开源走进农业也并不突兀。随着各领域加速数字化进程，以及产业互联网的推进，未来，人们将更深刻地感受到农业及其他传统领域数字化转型带来的高效便捷。</p>
