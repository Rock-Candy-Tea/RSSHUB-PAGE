<h2>01</h2> 
<p>最近这段时间，吴昕参加《妈妈你真好看》这档综艺节目，因在节目中动不动就哭泣崩溃，引发了不少网友的反感乃至质疑。</p> 
<p>在一期节目中，吴昕为妈妈搭配的造型以及为这个造型准备的PPT都不是很出彩，于是吴昕妈妈被留在了待定席，结果吴昕的反应让人始料不及。</p> 
<p>她觉得是自己能力不行，连累妈妈没能顺利晋级，于是情绪瞬间失控大哭起来，甚至冲出了演播厅，这让评委刘雯非常震惊，连忙跑去安稳吴昕，没想到吴昕的情绪更加失控，这让刘雯感到愧疚不已，一边和吴昕道歉，一边也痛哭起来。</p> 
<p class="image-wrapper"><img data-img-size-val="639,263" src="https://img.36krcdn.com/20210520/v2_5019d8596220490983b443feafe05070_img_000" referrerpolicy="no-referrer"></p> 
<p>后来经过了解得知，原来，吴昕给妈妈挑选衣服时就没有挑到好的，她觉得自己对不住妈妈；这回因为自己的原因连累妈妈没能成功晋级，这让吴昕更加自责难过——情绪一下子就爆发了。</p> 
<p>娱乐八卦掌柜一针见血地指出——</p> 
<ul> 
 <li>“作为一个有着15年主持人经验的吴昕，拥有这样的表现无疑是不成熟，也是不应该的。她本该是控场的那个人，结果却因为过度自责情绪失控，反倒成为了节目最不稳定的因素，这实在不是一个资深节目主持人该有的状态。”</li> 
</ul> 
<p>有网友吐槽说，这个吴昕太矫情了，上一次节目就要哭一次，实在让人反感；很多人也表示，吴昕在节目中情绪失控痛哭，实在无法引发大家的共情，甚至会有些“卖惨”的嫌疑；有人建议她多提高业务能力，还有人嘲讽说她掌握了“<a class="project-link" data-id="431525" data-name="财富密码" data-logo="https://img.36krcdn.com/20201106/v2_f3cc37f661d74c5e8336981865107e17_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/431525" target="_blank">财富密码</a>”......</p> 
<p class="image-wrapper"><img data-img-size-val="545,257" src="https://img.36krcdn.com/20210520/v2_e308f9e731a54e7f90b7fdc2ff584cdb_img_000" referrerpolicy="no-referrer"></p> 
<p>关于这个事件本身，<a class="project-link" data-id="427808" data-name="我在这里" data-logo="https://img.36krcdn.com/20200729/v2_b26ab8b8b8d3439dae0e9b0b5146918b_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/427808" target="_blank">我在这里</a>不想过多评论；但结合我的咨询经验、综合吴昕在各种节目中的表现来看，我非常认同娱乐八卦掌柜的观点，那就是——</p> 
<ul> 
 <li>“吴昕这样的表现，是常见过度自卑的表现......这样的哭泣，其实只是廉价的泪水，毫无价值。”</li> 
</ul> 
<p>那么，自卑心理到底是如何产生的？对我们的职业发展有着怎样深远的影响呢？</p> 
<h2>02</h2> 
<p>想要弄清楚这个问题，就需从人们脑海中一个非常顽固的思维模式说起，这个思维模式叫做“自责”。</p> 
<p>某医院的精神科主任曾经指出：“容易产生自责情绪的人往往伴随了轻度的抑郁症，容易一点带面把灾难放大，因为我不好，所以我抑郁了，因为我抑郁了所以我不好，走进了思想的怪圈。”</p> 
<p>两年多前，有个叫婷婷（化名）的来访者通过网上的文章顺藤摸瓜地找到了我们，不辞劳苦从外地赶来，和我进行了一次线下咨询。</p> 
<ul> 
 <li>婷婷，英语专业出身，本科学历。</li> 
</ul> 
<p>婷婷的第一份工作是一家幼儿英语培训机构的助教老师，半年不到，她就被机构以“无法胜任工作”为由劝退了。</p> 
<p>后来，她又做过行政文员、人事助理、电话销售等各种工作，但奇怪的是，每段工作经历时间都不长，最短两个月，最长只有半年......前来咨询的时候，婷婷在一家公司做文案。</p> 
<p>她很珍惜这次工作机会，按照她的话说，她不指望通过一次咨询就能搞清楚所有的困惑，但起码通过梳理，能知道自己过往经历失败的原因，才不至于以后再犯同样的错误。</p> 
<p>“能具体说说，你在最近的工作中遇到的让你困惑的一件事吗？”我问。</p> 
<p>婷婷想了想，娓娓道来。</p> 
<p>那还是她刚入职公司文案岗一个多月的时候，一天，部门里有个老员工陈姐（化名）正好路过婷婷的工位，看她不太忙的样子，便问：“你能帮我做个事吗？”</p> 
<ul> 
 <li>原来，陈姐这边有个客户把之前的文案给退回来了，按照陈姐的说法，这个客户比较麻烦，没有十次八次修改很难搞定，但自己这边正在赶一个比较重要的文案，所以她就想让婷婷帮个忙，稍微把这个文案调整修改下，再由她发给客户审阅即可。</li> 
</ul> 
<p>听到这样的请求，婷婷内心多少有些发怵；看着婷婷欲言又止的模样，陈姐说：“没事，不要紧张，你随便搞搞就行。”</p> 
<p>婷婷硬着头皮应承了下来。</p> 
<h2>03</h2> 
<p>下班后，婷婷回到出租屋里，在网上大量搜索文案知识，希望能把自己接手的这第一单文案做漂亮些；同时也生怕自己如果做得不好，会遭客户及同事嘲笑。</p> 
<p>结果第二天一早，陈姐就找到婷婷：“你怎么回事？我昨天不是和你说了吗？让你在昨晚下班前改好发我，你怎么没有按时发我？一大早客户就打电话说我......到底是怎么回事？”陈姐非常愤怒。</p> 
<ul> 
 <li>婷婷更是一头雾水，她确实记得陈姐交代她下班前给到的，但是婷婷总觉得不够好，加上婷婷认为，如果下班前她没上交，陈姐也应该催她的......</li> 
</ul> 
<p>她以为陈姐没有催她，是默许她可以加班把文案改<a class="project-link" data-id="190747" data-name="好推" data-logo="https://img.36krcdn.com/20201111/v2_1eaabe3d02f14e26ac070d25b53b05d9_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/190747" target="_blank">好推</a>迟交的......谁想到陈姐手头事情太多，一下子把这件事情给忘记了，直到今天早上被客户一顿埋怨，这才发现婷婷没有及时上交，便朝她发了一通火。</p> 
<p class="image-wrapper"><img data-img-size-val="1080,720" src="https://img.36krcdn.com/20210520/v2_a6fe36d7160c4574af67192ed393ee23_img_000" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>“你当时没有和对方确认文案提交的时间吗？或者说，你看她没有催你，是否想过和对方确认下提交的时间呢？”我问。</li> 
 <li>“晓璃老师，实不相瞒，我当时大脑中闪过这样的念头，但我很担心陈姐数落我，说我连这么简单的事情还要问她......”婷婷叹了口气。</li> 
 <li>“但你可曾想过，不和对方确认，自己擅作主张推迟时间，难道对方就不会数落你了吗？”我问。</li> 
</ul> 
<p>婷婷的眼睛里闪现出了一丝泪花，她似乎有话要说，但不知为何，又变得沉默了。</p> 
<p>看着婷婷沉默的样子，我感慨万千。</p> 
<h2>04</h2> 
<p>我深知，婷婷正是陷入到了前文提及的“自责”的思维模式里。</p> 
<ul> 
 <li>在这种思维模式下，当事人很容易把错误归因于自身太弱，仿佛自己无法改变，于是一旦工作中出现了问题，就会有意无意地选择逃避，不敢正面问题，更谈不上积极解决了。</li> 
</ul> 
<p>而这种“自责”思维模式的产生，离不开个人的原生家庭环境。</p> 
<p>通常说来，如果一个人在童年时代总是被“过度干涉”或者“过度保护”，TA会很容易形成“自责”的思维模式。</p> 
<p>发展心理学告诉我们，3-6岁这个阶段是容易产生内疚的年龄，在这个阶段，如果个体没有安全感，长大后容易对过错内疚；6-12岁这个阶段是容易产生自卑的年龄，在这个阶段，如果个体没有安全感，长大后就很容易不自信。</p> 
<p>如果我没有猜错，婷婷小时候应该就是那个被“过度保护”的孩子。</p> 
<p>你可曾有过这样的记忆？</p> 
<ul> 
 <li>很小的时候，你对一样东西充满好奇，伸出小手想要去触碰它的时候，突然，妈妈厉声予以阻止，对你说这样东西是多么可怕与危险，让你不要去碰；</li> 
 <li>上幼儿园的时候，你听从老师的“自己的事情自己干”的教诲，拿出自己的脏手帕清洗，结果你的妈妈过来，夺去了你手里的手帕，告诉你这些活儿不用你干；</li> 
 <li>上学的时候，有次考试你的成绩不理想，你胆战心惊地拿出试卷给父母看，你的父母对你劈头盖脸一顿否定，甚至还说出对你很失望之类的话......</li> 
</ul> 
<p>不知从什么时候开始，你感觉自己的内心越来越没有力量了，你认定自己是脆弱无助的。</p> 
<p>与此同时，你还摸索出了一个“生存策略”，那就是——</p> 
<ul> 
 <li>在强势的父母面前，与其被他们指责，不如自我指责，甚至用嚎啕大哭彰显自己的后悔与懊恼，然后你发现，大人们似乎妥协了，他们就没有之前那样指责你了。</li> 
</ul> 
<p>没错，一个人的思维模式乃至行为模式之所以根深蒂固影响深远，就是因为它们的形成非一朝一夕，它是你在适应环境的过程中强化出来的一套“生存逻辑”。</p> 
<p>但这种模式最可怕的地方在于，一旦你长大了，需要独自面对工作、生活乃至人际关系等问题的时候，你的柔弱、眼泪乃至自责就是在对外释放一种“我不行，我不能承担责任”这样的信号，试问谁还敢用你呢？......如此一来，你的职业发展不陷入危机才怪！</p> 
<h2>05</h2> 
<p>通过系统的分析和梳理，婷婷终于明白了自己的问题，咨询收尾的时候，她的眼睛里露出了久违的光亮，仿佛多年的阴霾一扫而光。</p> 
<p>“晓璃老师，谢谢你，要不是你的分析，我始终不明白自己的问题出在哪里。”婷婷说：“其实就像你分析的那样，我不能逃避，后期需要和主管同事积极沟通，积极行动，才能从根本上改变自己的困境。”</p> 
<p>我告诉婷婷，这种改变可能非一朝一夕就能完成，中间可能会有反复：可以多阅读些心理学的书籍、多参加些活动、多和人交流......等有了成就体验之后，再回过头来看今天的一切，对你而言就不再是困扰了；如果你认为，单靠个人力量很难改变，建议寻求专业帮助。</p> 
<p>行文至此，我想说——</p> 
<ul> 
 <li>其实，我们每个人是可以坚强的、迅猛的、勇敢的......就算从来没有人告诉过你，那么从今天开始，请自己告诉自己：一切都不是不可改变的。</li> 
</ul> 
<p>我们需要觉察和警惕的，恰恰是多年来形成的固定思维模式，只有我们予以察觉并予以调整，才能走出思想的禁锢......如此，你的职业发展才能迎来新的生机与转机。</p> 
<p>以上。</p>
