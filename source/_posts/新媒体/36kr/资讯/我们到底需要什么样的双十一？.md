<div>   
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">又是一年一度双十一。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">每当此时，总要发些感慨多说几句。对这个节日也曾有过不同的看法，如最初我更倾向于认为这是一个以促内需为主要要素的节日，零售业在此时<a class="project-link" data-id="95377" data-name="得到" data-logo="https://img.36krcdn.com/20210807/v2_966db147ab4646ef82349f069ce61219_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/95377" target="_blank">得到</a>极大释放，拉动内需，其后，<a class="project-link" data-id="31884" data-name="我的" data-logo="https://img.36krcdn.com/20210909/v2_1d49199c81664bf6b53a23b91e8adf0d_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/31884" target="_blank">我的</a>思想又逐渐转向零售与制造部门之间的关系，认为以牺牲终端市场价格（动辄五折）为代价的规模增长，可能会牺牲品牌商利润（相当部分品牌商是被裹挟参与的），这并不是一个可持续的购物节。</span></p> 
<p style="text-align: center;"><img class="rich_pages wxw-img js_insertlocalimg" data-backh="356" data-backw="578" data-fileid="508134640" data-ratio="0.615506329113924" data-s="300,640" data-type="png" data-w="632" style="width: 100%;height: auto;" src="https://img.36krcdn.com/20211023/v2_a394b011f93b4d13baa4dba76c4f2c08_img_000" referrerpolicy="no-referrer"></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">在我反复摇摆的看法中，双十一已经进入到第13个年头，受疫情影响宏观经济仍然有较大的不确定性，我们此时再看双十一。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">在2021年初，我曾对零售业抱以信心，主要逻辑为：CPI与PPI的剪刀差效应越发明显，从品牌商抑或是整个制造业角度考量，若要减轻成本端不断上涨的压力，最有效的办法就是提高终端销售规模，一方面以规模效应摊薄成本，另一方面也就是更重要的，若需求端旺盛企业是可以通过涨价的形式来根本上缓解生产压力。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">但上述情况并没如愿发生，在Q3宏观数据公布之后，两组数据的剪刀差仍有放大的势头。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><img data-backh="184" data-backw="415" data-fileid="508134637" data-ratio="0.4421875" data-type="png" data-w="640" height="183" style="width: 100%;height: auto;" width="415" src="https://img.36krcdn.com/20211023/v2_0801d3aa06384bc8989e3ccbfd748d75_img_000" referrerpolicy="no-referrer"></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">从年初PPI同比回正之后，就有专业人士预测1-2个季度以后，制造端的成本就会传导至销售端，会迎来可怕的通胀，甚至有人呼吁要对货币政策提前下手。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">但价格传导机制并未如上述所推演那般平滑传导至终端，相反在需求端仍然相对萎靡时<strong>，价格是很难上扬，CPI的表现既是货币现象（福利德曼说一切通胀都是货币现象），但另一方面，在经济行为中CPI又是需求旺盛程度的投射，抛开需求只强调货币问题并不能解释当下剪刀差的问题。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">CPI与社消的双双回落都反映了当下我们需要注意的尚不到通胀的时候，而是进一步刺激需求，刺激了需求，也就产生了规模，<strong>无论是微观企业抑或是宏观经济都需要需求的提振。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><img data-backh="218" data-backw="362" data-fileid="508134639" data-ratio="0.6021220159151194" data-type="png" data-w="754" height="218" style="width: 100%;height: auto;" width="362" src="https://img.36krcdn.com/20211023/v2_7fba22edac704ee8a5c71337c70d7626_img_000" referrerpolicy="no-referrer"></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">在上图中我们不难发现当下社消仍是处于低谷的，经济学界对提振社消可能有更多复杂的手段，诸如在货币政策或者生活保障方面做足文章，提高低收入阶层的消费边际效应，带动增长。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">这是长期的目标我们是极为赞同的，但短期内要迅<a class="project-link" data-id="578319" data-name="速见" data-logo="https://img.36krcdn.com/20210814/v2_9bb1ca04cf334b75971ce3db907f9fbd_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/578319" target="_blank">速见</a>效，我们还是希望从线上入手，<strong>从目前来看，能够快速激活消费热情的除了线上，并未有其他更有效的方法。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">我们以618效果为例。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">作为年中大促，618声势与规模与双十一越发接近，直接表现在6月电商交易规模占当季度比例的大幅攀升。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><img data-backh="218" data-backw="362" data-fileid="508134635" data-ratio="0.6021220159151194" data-type="png" data-w="754" height="218" style="width: 100%;height: auto;" width="362" src="https://img.36krcdn.com/20211023/v2_0c8084df5fc044a7b29bf92190d3fd7a_img_000" referrerpolicy="no-referrer"></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">2020-2021两年中6月实物电商规模占当季度电商规模比重迅速上扬，联系到彼时线上零售总规模仍处于下行压力放大的时间窗口，<strong>618作为大促很大程度上稀释了外界环境恶化对电商和零售业的冲击。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">换句话说，如果无618，Q2的电商数据可能要下调。此前有观点认为大促更多是蓄水效应，乃是将需求积蓄在一天内，造成<a class="project-link" data-id="4260438" data-name="了数" data-logo="https://img.36krcdn.com/20210422/v2_8e636ec7be434dd5bf7deebc8bed2b62_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/4260438" target="_blank">了数</a>字繁荣。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">这有一定道理，但实际上却忽略了两大变量：1.蓄水观点更多是零和效应，但在大促中，我们是看到当期线上零售是净增的；2.在集中大促中，流量膨胀，平台或商家的获客成本降低，用户的规模效应又进一步降低成本，这乃是正向循环。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">因此，数字繁荣貌似<a class="project-link" data-id="29872" data-name="有道" data-logo="https://img.36krcdn.com/20201021/v2_7a729cd6be444a02ac5ecdf2c0ccbb30_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/29872" target="_blank">有道</a>理，但其实并不客观。</span></p> 
<p style="text-align: left;font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">如果说618是Q2社消和线上零售的功臣，那么双十一对于零售带动能力要远大于618，见下图</span></p> 
<p style="text-align: left;font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-family: 宋体;font-size: 16px;"><img data-backh="242" data-backw="415" data-fileid="508134633" data-ratio="0.5842956120092379" data-type="png" data-w="866" height="242" style="width: 100%;height: auto;" width="415" src="https://img.36krcdn.com/20211023/v2_07dbea069ae645768d1c0d271c9af260_img_000" referrerpolicy="no-referrer"></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">无论从规模抑或是占当月电商规模比重，双十一都要超越618，<strong>如果在扩内需中618对于上半年经济起到了积极的效应，那么双十一在接下来的价值会更大。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">这也是我们对双十一仍然持以乐观态度的重要原因，也希望今年双十一可以继续创造纪录。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">对双十一表扬之后，我们要对其开始 “批判”了。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">在相当长时间内，双十一一直是天猫的独角戏，当一个平台在一定周期内具有类似垄断地位时，就自然有了捆绑商家的能力和动力，自有双十<a class="project-link" data-id="81906" data-name="一起" data-logo="https://img.36krcdn.com/20210709/v2_647b9860d6f7437caf1be2501d37698a_img_000" data-refer-type="1" href="https://www.36dianping.com/space/4772100123" target="_blank">一起</a>，平台与商家龃龉的消息就未曾间断，诸如强行规定价格，提高流量成本，规则变动等等，平台为讨好用户，扩大C端流量规模，而商家的主要市场份额又在天猫，矛盾自然凸显。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">如前文所言，双十一对于今时经济贡献一在需求规模刺激，二在有效以需求端为刺激，使实体企业可以在此获得收益，以往以牺牲商家盈利的方式是值得反思的。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">于是，我又整理在近十余年的CPI走势中，标注了11月的CPI情况，见下图</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><img data-backh="164" data-backw="415" data-fileid="508134636" data-ratio="0.39593301435406697" data-type="png" data-w="836" height="164" style="width: 100%;height: auto;" width="415" src="https://img.36krcdn.com/20211023/v2_096855a5e3524be9b89212ce7a4f203d_img_000" referrerpolicy="no-referrer"></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">图中我将11月单独标注，如果用经济学最基本原理分析，天猫引领的双十一可能会打破短期的供给均衡，尤其在近几年非天猫体系零售也纷纷参与双十一，当天也成了全国零售业的狂欢，需求旺盛CPI理论上是要上扬的，或者会在其后保持CPI的增长。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">在上图中，双十一启动之前的2008年上下，11月CPI走势相对凌乱和不规则，其后CPI整体是在上扬，其后今年我们发现当期CPI下降的概率更大，我们将此归因为：<strong>愈演愈烈的低价策略下的双十一是以抑制终端价格为代价的。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">在宏观经济向好，通胀预期强烈的背景下，此手段可减轻宏观经济的通胀压力，这也是我国电商业的一大贡献，但如前文所言，<strong>当下我们不仅是要提高需求，更应该要在市场中通过提振需求能使实体经济分享超额利润，而非以往由单一平台把持。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">我个人希望今年平台们都不要再强调低价，给供应商留有余地，甚至用户流量起来，一些紧俏优质商家获得价格溢价机会也并非不可，<strong>在过去，电商将低价视为天然使命，以让商家让利获得用户流量，再将流量销售于商家，今年这种循环希望可以得到根本性的改观。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">接下来我们再谈谈垂直细分品类的机会。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">我个人一直在关注服饰行业，这是一个最先进行电商化的品类，也是阿里的护城河品类之一，但在近年该品类一方面进入了下行通道，而另一方面，阿里在此品类的控制能力也有明显的下滑，如白牌商家向<a class="project-link" data-id="81304" data-name="拼多多" data-logo="https://img.36krcdn.com/20210807/v2_3ea17e428efb4e328cf195247203db18_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/81304" target="_blank">拼多多</a>迁移，而白牌与品牌商品其实是共享同一生态的，白牌权重的降低，如森林缺少灌木，阿里的用户和流量基础有被侵蚀的风险。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">但我在今年的观察中，一直比较倾向于认为该品类在产业端可能已经度过调整期，见下图。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><img data-backh="218" data-backw="362" data-fileid="508134638" data-ratio="0.6021220159151194" data-type="png" data-w="754" height="218" style="width: 100%;height: auto;" width="362" src="https://img.36krcdn.com/20211023/v2_755f845441a94733a459474052cb60eb_img_000" referrerpolicy="no-referrer"></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">从2015年至今，“穿”这一大品类的线上增长多是在下行通道中，与此同时，“吃”则相对稳定，“用”亦是在下降。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">但进入2021年之后，“穿”就有了逆转的苗头，似乎有触底的可能。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">我们继续看下图。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><img data-backh="218" data-backw="362" data-fileid="508134634" data-ratio="0.6021220159151194" data-type="png" data-w="754" height="218" style="width: 100%;height: auto;" width="362" src="https://img.36krcdn.com/20211023/v2_38d7fa31ebd9467db1f6a6fce4a3f560_img_000" referrerpolicy="no-referrer"></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">两张如放置在一起对比，很容易发现，2015年确实是服饰行业的一大转折点，概念后，固定投资增长一路下滑，可以说是零增长，我们也知道，一个行业如果要保持产能的稳定输出，是要每年进行必要的固定资产投入的，我们只能将零增长解释为行业在稳步去产能。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">联系到彼时宏观层面也进行了比较大手笔去产能工作，也强化了我们上述判断，但进入2019年之后，固定投资有了好转的迹象，若无疫情这一突发事件影响，很可能2020年之后该品类会进入一个上升周期。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">根据工信部数据，2021年1－8月，服装行业规模以上企业12520家，累计营业收入9056亿元，同比增长9.6%；利润总额396亿元，同比增长9.5%；利润率4.4%，与上年同期基本持平。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">相较于其他品类，我们觉得服装行业在一个需求不振的周期内，还是相对比较稳的，同期家电行业利润增长就严重落后于销量增长，利润是被稀释的。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">我们之所以用服饰为例子，只是为证明：<strong>任何一个商业模式做到万亿元以上规模都要不同程度上受到周期性影响，服饰在过去的6年时间里之于阿里也恰是如此。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">但周期性的“妙”就在于他是有周期的，有低谷就会有转机，有转机就蕴含希望，多年调整之后，相当部分行业其实已经完成了去产能化，要进入一个新的周期。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">其主要表现为：</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">其一，国货概念兴起；</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">在传统经济分析中，去产能主要的手段是价格调控，也就是通过价格的调整使行业出清，而留存下来的企业又会缓慢获得产品定价能力，进而获得溢价，这个过程比较长，但概括起来就是这样简单。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">在新国货的概念中，除了IP之外，我其实看到了冠以“新国货”名义后价格的上扬，<strong>也就是说，品牌商在行业完成出清之后，已经在考虑溢价的问题</strong>，而在出清完成之前，全行业都在价格甩卖，“新国货”这种溢价是很难行得通的。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">所以我倾向于认为新国货会强化我对部分领域出清工作的判断。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">其二，零售渠道分散。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">关于双十一预热，可能舆论更为关注李佳琪和薇娅在直播动辄数十亿的销量，我们且不论销量规模真实性（可能有退换货以及未完成支付等问题），单说令人咋舌的数据，我都感觉有些担心，如果一到两个直播可以霸占百亿级别的销量，那么，为巩固其地位，就有动机倒逼供应商以低价，寡头集中化的效应难道要在直播产业中再演么？</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">雕爷在双十一之前有过一篇文章，其大意为，双十一已无必要，毕竟主播们都掌握了供应链的秘密，可以每天都是双十一。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">这当然掀起了舆论很大的声响，从观点上我是比较认同，李佳琪和薇娅可以决定一个品类的行业定价权，这并非夸张，只要他们愿意，他们确实也有能力做到天天双十一的价格。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">但在此观点背后，我是略有惶恐，<strong>在一个追求“渠道去中心化”的时期，过分集中的弊端其实愈发凸显，我也不希望李佳琪和薇娅真的拥有行业定价能力。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">好在虽然数字惊人，但电商平台之间的渠道已经极为分散，天猫率先启动的双十一已非独角戏，也非与<a class="project-link" data-id="27570" data-name="京东" data-logo="https://img.36krcdn.com/20210806/v2_40d01763039949f194e3818a1b4fa75d_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/27570" target="_blank">京东</a>的二人转，而是真正零售业的大事，抖音，<a class="project-link" data-id="30859" data-name="快手" data-logo="https://img.36krcdn.com/20210806/v2_6d9c4ce1088b45e3b264cace0dc60702_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/30859" target="_blank">快手</a>，拼多多平台越多参与，用户越热情，渠道越分散，活动其实越成功，商家越有可能在平台的胶着竞争中获得利益。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;"><strong>当渠道分散化之后，商家与平台谈判能力上升，与去产能结合，可能会是一个正向积极的意义。</strong></span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">最后，作为价值分析的，我还是简单概括我对行业公司的一些看法。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">监管对行业收紧至今，行业头部企业（如美团，京东，阿里等等）等等，经营增长虽然下降，但并未萎缩，基本面并未发生逆转，且这些企业手中现金丰沛，通过回购等行为，稳定了资本市场的预期。</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">禁止资本无序扩张之后，这些企业在财务层面可能要思考：究竟如何消耗手中越来越多的现金，以往是自我发起新的业务，动辄百亿补贴拿下一个新场景。此后，消费互联网的机会收缩，现金的消耗可能就有两个去处：</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">1.创投化，2B和企业级应用的会比较多，在一级和二级市场可能会使得这些行业的头部企业进行重新的定价和估值，我们此后也会逐渐分享看法；</span></p> 
<p style="font-size: medium;font-family: DengXian;caret-color: rgb(0, 0, 0);color: rgb(0, 0, 0);white-space: normal;"><span style="font-size: 16px;">2.回购常态化，以往互联网公司很少发股息，认为给投资者分钱不如自我扩张，溢价增值能力更强，但扩张放缓后，何不分钱给投资者，对于科技股可能有更好的预期。</span></p>
