<p>尽管 Windows 11 的问世伴随着一系列质疑，但今年年初微软官方曾声称：“用户升级 Windows 11 的速度是当年升级 Windows 10 的两倍。”当时许多人感慨：这就是无法逃避的“真香”定理吧！</p> 
<p>可事实证明，Windows 11 似乎不如微软所说的那般受欢迎：前脚 AdDuplex 刚宣布目前 Windows 11 增长停滞，近来 Steam 硬件和软件调查数据显示，游戏玩家似乎也不那么待见 Windows 11。</p> 
<p class="image-wrapper"><img data-img-size-val="986,580" src="https://img.36krcdn.com/20220419/v2_d48b9755216641f98d4f24e972601780_img_000" referrerpolicy="no-referrer"></p> 
<h2>1 Windows 10 依旧是主导</h2> 
<p>一直以来，Steam 每个月都会进行一次匿名调查，由用户自愿发送数据，分析得出近期 Steam 用户所使用的主流计算机软件和硬件。</p> 
<p>由于 Windows 11 推出至今已有 6 个多月，外媒 arsTECHNICA 将近 7 个月（发布前 1 个月+发布后 6 个月）来 Steam 平台的调查数据，与 2015 年 Windows 10 推出时的 7 个月数据进行对比，以此得出游戏玩家在接受 Windows 10 和 Windows 11 速度方面的差异：</p> 
<ul> 
 <li>2015 年推出后，7 个月内已有超过 30% 的 Steam 用户将设备系统升级为 Windows 10；</li> 
 <li>自去年 10 月发布至今，只有 17.44% 的 Steam 用户选择采用 Windows 11。</li> 
</ul> 
<p>从以上数据可以看出，Steam 用户迁移到 Windows 11 的速度约是当初迁移至 Windows 10 的一半：Windows 10 用 7 个月的时间抢夺了几乎三分之一的 Steam 设备市场，2021 年 9 月在超过 90% 的 Steam 设备上运行，甚至直到今日都以超过 77% 的占比占据着主导地位；而 Windows 11 在同样的时间内只赢得了 17.44% Steam 用户的“青睐”。</p> 
<p class="image-wrapper"><img data-img-size-val="930,253" src="https://img.36krcdn.com/20220419/v2_c0bc2bb22a7b41519f602c48c31577b7_img_000" referrerpolicy="no-referrer"></p> 
<p>实际上，就算不从 Steam、AdDuplex 等第三方数据推测，通过对比微软当年对 Windows 10 的营销手段也能发现，Windows 11 的发展形势不太乐观——微软不再以数据说话了。</p> 
<p>当年，对 Windows 10 数据感到满意的微软，时不时就会提一句有多少台设备搭载了 Windows 10，例如“Windows 10 设备达到 5 亿台，一年增长三分之二”、“Windows 10 现在在 6 亿台设备上运行”等等，用这些漂亮的数据展示 Windows 10 被 Windows 7/Windows 8 用户所接受的速度之快。</p> 
<p>但从近几个月来看，微软没有在 Windows 11 上采取相同策略，起码不再用 Windows 11 设备的具体数量作为噱头了，由此导致各方媒体只能依靠第三方数据来判断人们接受 Windows 11 的速度有多快。</p> 
<h2>2 Windows 11 的尴尬处境 & 硬件“高门槛”</h2> 
<p>既然如此，那为什么 Windows 11 无法复制当年 Windows 10 的成绩？具体分析下来，主要有两个原因。</p> 
<p>首先，相比 Windows 10，Windows 11 推出的时机较为尴尬。</p> 
<p>当初 Windows 10 的问世可谓“得天独厚”：用户界面变更过大的 Windows 8 不受欢迎，Windows 7 又缺少一些 Windows 8 引入的新功能和改进，此时推出的 Windows 10 就是集大成者。</p> 
<p>从 Windows 10 推出后的 Windows 版本市场份额变化，可以看出 Windows 10 的出现是多么“正好”：Windows 8 的用户大量流失，使用率从 35% 左右下降到 19%，一小部分 Windows 7 用户也都在升级至 Windows 10。</p> 
<p>如果说 Windows 10 的出现，取代了一个较新却不受欢迎的操作系统（Windows 8）以及一个广受欢迎却老化的操作系统（Windows 7），那去年推出的 Windows 11 是为了取代什么，一个很少有人真正嫌弃、甚至表现良好的现代操作系统 Windows 10？</p> 
<p>其次，或许也是最重要的原因：Windows 11 的硬件“高门槛”。这也是 Windows 11 自官宣以来被吐槽最多的一个要求。</p> 
<p>根据微软官方提供的信息，如果用户想将 PC 系统升级为 Windows 11，需满足以下硬件要求：</p> 
<p class="image-wrapper"><img data-img-size-val="1080,530" src="https://img.36krcdn.com/20220419/v2_deda4aaf128141f1858618acec7f9feb_img_000" referrerpolicy="no-referrer"></p> 
<p>但其中，光 TPM 2.0 这一点，就已经将许多老旧设备拒在 Windows 11 的门外。外媒 arsTECHNICA 也因此猜测，Windows 11 在 Steam 设备系统中占比低的原因，很有可能是因为许多 PC 无法安装 Windows 11。</p> 
<p>尽管 Windows 11 过高的硬件要求遭到许多网友抵制，但直到如今，微软也毫不退让，以“确保设备安全性”为由，依旧坚持着这一高标准。</p> 
<h2>3 网友：“Windows 11 不算真正的升级”</h2> 
<p>对于 Steam 玩家也不欢迎 Windows 11 的现象，许多网友表示非常理解，因为 Windows 11 实质上没有发生很大的改变。</p> 
<p>@powermacguy：“就我个人而言，我没有理由升级到 Windows 11，它没有什么令人信服的优点。我会升级到 Windows 10，只是因为 Windows 7 正在失去对更新处理器和显卡的支持。”</p> 
<p>@Dark Jaguar：“还有一个简单的事实是，与 Windows 10 相比，Windows 11 根本就不算真正的升级。所以，我很疑惑为什么微软不坚持 Windows 10 是最后一个操作系统的承诺。”</p> 
<p>@Max A：“Windows 11 实际上并没有做任何新的改变，所以我觉得没必要升级。如果只是为了‘漂亮的圆角’，那真是愚蠢的改变。”</p> 
<p>除此之外，许多用户也再次对 Windows 11 的硬件要求发出抨击。</p> 
<p>@whiteknave：“最简单的解释：‘这台 PC 目前不满足运行 Windows 11 的最低系统要求’。可能在我对我的 PC 硬件进行全面升级时，我会‘升级’到 Windows 11。但在那之前，我将继续运行 Windows 10。”</p> 
<p>@TinCoyote：“离谱的是，我家的设备都不符合升级要求。一台搭载 Core i7 的游戏本，一个具备 AMD Ryzen 7 的工作电脑，两台 PC 都满足了 TPM 的要求，结果‘处理器’不符合要求！我无语了，微软意思是说 Core i7 还不够新是嘛？疯了吧！”</p> 
<p>还有部分网友吐槽，Windows 11 发布至今仍是“半成品”。</p> 
<p>@dcdp：“还有一个问题在于，我感觉 Windows 11 至今都只完成了部分。目前 Windows 11 正在进行更改的数量和范围，给我的感觉就像是微软以半成品的状态推出了 Windows 11，并计划随着时间的推移逐渐添加功能。可能这个想法适用于创建新应用的初创公司，但它显然不适合用在对计算机实际工作至关重要的操作系统上啊！”</p> 
<p>那么，你目前是否升级至 Windows 11？感受如何？如果还未升级是出于什么原因呢？</p>
