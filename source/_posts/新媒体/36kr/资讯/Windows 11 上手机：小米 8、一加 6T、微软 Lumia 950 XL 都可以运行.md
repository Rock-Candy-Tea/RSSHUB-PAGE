<p>众所周知，<a class="project-link" data-id="3967413" data-name="微软" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3967413" target="_blank">微软</a>曾努力奔赴移动端并推出了 Windows Phone，那你有想过近期官宣的 Windows 11 运行在手机上的样子吗？</p> 
<p>国外一位工程专业的学生 Gustave Monce 对此感到好奇，恰逢微软推出了面向 Insider 的 Windows 11 预览版，他便尝试在 Lumia 950 XL（Lumia 950 XL 是 5 年前微软首批配备 Windows 10 移动版的设备之一，也是最后一款 Windows Mobile 手机）上运行 Windows 11，结果真的成功了！</p> 
<p class="image-wrapper"><img data-img-size-val="587,320" src="https://img.36krcdn.com/20210705/v2_13bbb9fe89a249d5a9dce4a397c918d3_img_000" referrerpolicy="no-referrer"></p> 
<h2>01 Windows 11 运行在 Lumia 950 XL 上</h2> 
<p>Gustave Monce 能在 Lumia 950 XL 上运行 Windows 11 并非一蹴而就：“这一切都始于 5 年前，当时 Windows 10 移动版已快要过时。”</p> 
<p>事实证明，Gustave Monce 在接受 THE Verge 采访时提到的这个观点是正确的。随着 Android 和 iOS 设备的激增，Windows 10 移动版的市场份额早已微乎其微。</p> 
<p>彼时，Gustave Monce 携手同样想在 Lumia5.7 英寸屏幕上运行桌面操作系统的 Bingxing Wang 开启了 Lumia WOA —— 一个为 Lumia 950 和 Lumia 950 XL 带来 Windows 桌面操作系统的项目。</p> 
<p>“随着时间的推移，这个项目的讨论组逐渐形成，或者说，对在 Lumia 950 上完整运行 Windows 10 感兴趣的人们组成了一个社区。”Gustave Monce 还指出：“我们现在基本上是一个大约 15 人的团队，并且已经取得了很大进展。”</p> 
<p>据 Gustave Monce 介绍，为了使 Windows 11 能在 Windows Phone 上运行，其团队已移植和逆向工程驱动程序，甚至还编写了自己的驱动程序，最终总算是成功了。</p> 
<p>虽然 Windows 11 能够运行在 Lumia 950 XL 上，但由于其本身并非移动操作系统，因此并不能完整运行，在手机上呈现出来的效果也自然不比 Android 和 iOS：在双击屏幕、启动文件浏览器、点击开始菜单和完成其他任务时会出现延迟，屏幕旋转也很麻烦。</p> 
<p>除此之外，Gustave Monce 也承认 Windows 11 系统下的 Lumia 950 XL 相机无法正常工作、电池寿命受到影响、通话/短信也会偶尔中断。如果要作为移动操作系统，目前的 Windows 11 自然不合格。</p> 
<p>不过尽管如此，他的这番研究也令我们了解到 Windows 11 大刀阔斧的 UI 在智能手机上的样貌以及 Snap Layouts 等功能在小屏设备上是如何实现的。</p> 
<p class="image-wrapper"><img data-img-size-val="243,319" src="https://img.36krcdn.com/20210705/v2_20d01c74d24f4d66be6f2f373244cf51_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">（Windows 11 系统下的 Lumia 950 XL）</p> 
<p>对于这番成果，Gustave Monce 谦虚表示：“我不会说我们做得很快，因为总体看来我们是花了四年时间才达到目前的状态。”Gustave Monce 解释道：“基于 Windows 10 的所有工作以及我们之前为支持 Windows 10X 所做的工作，在 Lumia 950 XL 上运行 Windows 11 是<a class="project-link" data-id="3969340" data-name="一条" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3969340" target="_blank">一条</a>自然而然的前进道路，并且真的不需要太多时间就可以开始。”</p> 
<h2>02 小米 8、一加 6T等 Android 手机也可以运行 Windows 11</h2> 
<p>有了 Lumia 950 XL 这个例子，相信不少人会开始思考：那 Windows 11 系统是否也能移植到 Android 手机？国内一位来自华中师范大学第一附属中学的大神 Xilin Wu 告诉我们：可以。</p> 
<p>作为 Renegade 项目（一个尝试将 edk2 移植到不同手机的项目）背后的首席开发人员，Xilin Wu 表示 Windows 11 ARM64 可以移植到搭载高通骁龙 845 芯片的手机上。</p> 
<p>由于高通 UEFI 的 XBL（UEFI 固件，负责芯片驱动及充电等核心应用功能）和 ABL（加载 Linux 内核所需）不能修改，因此 Xilin Wu 的团队创建了自己的工具/驱动程序，为骁龙 845 启用定制的 UEFI 环境。</p> 
<p>得益于 Renegade 团队的努力，目前已有部分开发者将 Windows 11 系统移植到运行高通骁龙845 的<a class="project-link" data-id="121888" data-name="小米" data-logo="https://img.36krcdn.com/20200729/v2_7ad3d765f1d844018b8f4e75c4c8050d_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/121888" target="_blank">小米</a> 8、一加 6T 和部分骁龙原型机上：</p> 
<p class="image-wrapper"><img data-img-size-val="637,719" src="https://img.36krcdn.com/20210705/v2_41ef977d751c4948ad835fe1ee3b7518_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">（Windows 11 系统下的一加 6T）</p> 
<p class="image-wrapper"><img data-img-size-val="1062,933" src="https://img.36krcdn.com/20210705/v2_f22f976307da4a3e864c0065845fb908_img_000" referrerpolicy="no-referrer"></p> 
<p class="img-desc">（Windows 11 系统下的小米 8）</p> 
<p>但不像 Lumia 950 XL 的 LTE 调制解调器可以被 Windows 11 识别以实现呼叫功能，开发者明确指出目前移植到 Android 手机中的 Windows 11 不支持呼叫，并且系统的大部分功能也无法在手机中正常使用。</p> 
<p>除此之外，国外还有研究团队已实现在 Rasberry Pi 4 上启动并运行Windows 11。虽然同样存在一定限制，但这个处于早期阶段的<a target="_blank" rel="noopener noreferrer" href="http://mp.weixin.qq.com/s?__biz=MjM5MjAwODM4MA==&mid=2650851671&idx=2&sn=a84702b3d8fb152a6b130b7b5a41e73a&chksm=bd58b8848a2f3192b7c9593e73fdc1779e8325889f34523d8fabde23ff0c2f3cbdc6214a34c4&scene=21#wechat_redirect">Windows 11 预览版</a>本身就有许多问题，因此未来还有提升的余地。</p> 
<p>不过我们也无需对此抱有太高期待，因为微软推出的 Windows 11 并未真正针对移动设备进行优化，把 Windows 11 移植到手机上只是部分开发者们的实验性操作或兴趣使然，想要将其当做一款移动操作系统还是远远不够的。</p>
