<p>编辑：王与桐</p> 
<p>36氪独家获悉，获客搜索引擎「Fork AI」获数百万美元天使轮融资，本轮投资来自险峰长青，资金主要用于加大产品研发投入和团队建设。</p> 
<p><strong>「Fork AI」的客户群体主要是云计算厂商</strong>。虽然云计算行业发展迅速，从无到有只用了十几年，但国内云计算企业的获客方式从未发生结构性改变：主要的获客方式还是通过SEM渠道投放，比如搜索引擎<a class="project-link" data-id="28215" data-name="百度" data-logo="https://img.36krcdn.com/20210325/v2_fa02010c4a8b46da9f4e1d3b1fd59f22_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/28215" target="_blank">百度</a>、360、搜狗等。<a class="project-link" data-id="99343" data-name="字节跳动" data-logo="https://img.36krcdn.com/20210322/v2_7113c787db844092b25d03ca617cf4ef_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/99343" target="_blank">字节跳动</a>的崛起让DSP投放出现了更多渠道，除了<a class="project-link" data-id="24961" data-name="腾讯" data-logo="https://img.36krcdn.com/20201201/v2_016524a9a477434cb3584e1558f3257a_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/24961" target="_blank">腾讯</a>系的腾讯广告、公众号、朋友圈等，字节系的抖音、今日头条、西瓜等也可以作为选择。<strong>搜索引擎对云计算企业的主要价值来自于规模化线索的输入，但基于广告匹配的规模化线索导入无法实现对线索的价值判断</strong>。投放获取的线索存在<a class="project-link" data-id="601928" data-name="企业购" data-logo="https://img.36krcdn.com/20200729/v2_45f6b4e5095f4201946c94f949462c86_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/601928" target="_blank">企业购</a>买需求不明确的状况，需要通过呼叫中心去拨打电话进一步确认企业采购意向，这一行为成本高，成交率低是行业现状。</p> 
<p>Fork AI CEO何李石曾在<a class="project-link" data-id="6608" data-name="七牛云" data-logo="https://img.36krcdn.com/20210714/v2_3b6e8e98b2be4a7687bfcc8a46ca2153_img_000" data-refer-type="1" href="https://www.36dianping.com/space/4720601013" target="_blank">七牛云</a>任职多年，对于云厂商的获客痛点有很多体会，他决定针对性的去解决这些问题，于是何李石于2019年创立了杭州晓富科技技术有限公司，并于2021年推出Fork AI。Fork AI是晓富科技为云厂商提供服务的基础，这是一款覆盖了主流应用商店数据的搜索引擎，通过检测应用商店中APP使用SDK 及 API 的动态，实现对该应用背后的企业和开发者们云服务需求的挖掘。</p> 
<p class="image-wrapper"><img data-img-size-val="830,478" src="https://img.36krcdn.com/20210719/v2_c3a2de0fc0a04492a1daffacef4fc998_img_jpeg" referrerpolicy="no-referrer"></p> 
<p class="img-desc">来源：Fork AI </p> 
<p>有<a class="project-link" data-id="4260438" data-name="了数" data-logo="https://img.36krcdn.com/20210422/v2_8e636ec7be434dd5bf7deebc8bed2b62_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/4260438" target="_blank">了数</a>据作为基础，Fork Stack 产品于2021年7月上线。这是一款面向销售、市场群体的 SaaS 产品，为国内云厂商提供获客服务。Fork Stack 产品设计逻辑来自于创始团队七年间在云计算公司帮助销售市场分析移动应用技术栈的经验。</p> 
<p>举例来说，如果一个应用的数据量出现大服务增加，那么他们必然需要尽快扩大对云服务的采购。同样，通过对这款应用更多动态的检测与分析，也能让销售人员和客户成功人员提前为客户准备好更加完善的解决方案。Fork Stack 产品就是创始团队将该经验工具化、产品化，并应用于云计算行业的直接体现。</p> 
<p class="image-wrapper"><img data-img-size-val="830,582" src="https://img.36krcdn.com/20210719/v2_5a668dfbba5e4a5b8b17f565dd0b21cd_img_jpeg" referrerpolicy="no-referrer"></p> 
<p class="img-desc">来源：Fork AI </p> 
<p>作为一款SaaS工具<strong>，Fork Stack的收费模式是企业或个人定阅付费，可按月或按年支付。</strong>订阅用户将获得数据导出（包括移动应用导出、SDK 检测导出、服务商检测导出）和应用检测（包括服务商检测、SDK 检测）两种服务套餐，企业可以根据需求针对数据和SDK的检测、导出服务进行定制。目前产品覆盖了国内外主流应用商店及商店上的移动应用、移动开发者产品和他们的云服务需求信息；覆盖数千个公有云SDK 及 API 功能分类信息。通过公开数据检测为云厂商获取目标客户状态信息。并最终为云计算企业降低无效投放，为销售、市场、客户成功人员提升工作效率。<strong>个人帐户有免费版、基础版和高级版三种，基础版1400元/月、高级版12400元/月</strong>，若需要更大数量的导出服务，将需要额外支付增值费用。除个人用户外，晓富科技目前已为国内3家头部云服务企业提供了定制服务。</p> 
<p>Fork AI所在的市场想象空间很大。从需求端看，根据Frost&Sullivan的数据，从2015年到2019年，中国云服务的市场规模复合年增长率为37.7％。这意味着Fork AI等厂商有着充足的客户来源，接下来就看各家所切的功能是否能满足客户需求。</p> 
<p>一般来说，CRM作为企业管理销售和客户的工具，是接近企业获客需求的一方，然而在国内也一直难免被诟病只服务销售管理并不服务销售本身。Salesforce被大量企业广泛使用的原因之一，就是积极的完善了协助企业管<a class="project-link" data-id="66258" data-name="理客" data-logo="https://img.36krcdn.com/20200729/v2_520954e0892e41c28095e8f8976aa8eb_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/66258" target="_blank">理客</a>户信息、补充客户信息、协助销售分析客户需求信息的生态。如Salesforce生态中的data.com 和clearbit都是为企业销售人员提供数据增强服务的应用工具。综合来讲就是CRM除管理客户外还可以给企业销售填充基础信息，而clearbit作为数据增强服务工具，则能更进一步辅助销售补充客户特定的信息，如企业联系人的邮箱但 Salesforce也只占 CRM 市场中的 20%。</p> 
<p>疫情后国内针对SaaS领域的投资也非常活跃，根据数据显示，仅2020上半年，我国SaaS行业共发生融资事件280起，总金额达354.13亿元。至于国内CRM和细分领域数据增强工具的生态何时形成，形成过程中又会触发什么样的连锁反应，这也是Fork AI等厂商在融资道路上的机会。</p> 
<p>之所以能有以上市场认知，源于晓富科技的团队。其成员主要来自于国内外优秀云计算公司及优秀开源社区，包括 Ruby 解释器及高性能 I/O 库 event 贡献者、网络安全专家等，平均行业经验10年以上，在企业服务、编程语言及编译器、网络安全、Web 产品开发等领域拥有丰富的技术积累。</p> 
<p>创始人兼CEO何李石2012 年加入七牛云，负责过七牛云客户服务团队从 0 到 1 的创建全过程；联合创始人牛小满曾负责某上市公司云计算业务销售拓展，2014 年进入七牛云。</p>
