<p>自 iOS 15 正式发布以来，iOS 15.4 可能是最受大家关注的一个版本，也有可能是最有用的一次更新，因为在这个新版本中 Apple 终于带来了呼声极高的 Face ID 支持戴口罩解锁的功能，这对于仍处于疫情防控中的广大民众来说无疑是一个好消息。另外，在发布会上演示的通用控制功能也首次露面，让 Apple 全家桶用户的生态闭环体验更上一层楼。 </p> 
<p>接下来，就让我们<a class="project-link" data-id="81906" data-name="一起" data-logo="https://img.36krcdn.com/20210709/v2_647b9860d6f7437caf1be2501d37698a_img_000" data-refer-type="1" href="https://www.36dianping.com/space/4772100123" target="_blank">一起</a>来看看 iOS 15.4 的新特性吧。 </p> 
<p><strong>▍Face ID 支持戴口罩解锁</strong></p> 
<p>自新冠疫情爆发以来，iPhone 的 Face ID 功能就饱受诟病，因为它无法在用户戴口罩的情况下解锁手机，在展示健康码、行程码、付款二维码等情况下特别不方便。虽然在 Apple Watch 上推出了一个联动解锁 iPhone 的功能，但是很多 iPhone 用户并不同时拥有 Apple Watch，所以这个功能并没有从根本上解决问题。终于在 iOS 15.4 版本中，Apple <a class="project-link" data-id="187722" data-name="上线了" data-logo="https://img.36krcdn.com/20220120/v2_4ea0942a63674b4d8b32a55f9a5167b9_img_000" data-refer-type="1" href="https://www.36dianping.com/space/4210300001?mp=zzquote" target="_blank">上线了</a>支持戴口罩解锁的 Face ID 功能，可谓是「普<a class="project-link" data-id="52891" data-name="天同" data-logo="https://img.36krcdn.com/20210807/v2_fdef78ec2e174845a242033c2b253c27_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/52891" target="_blank">天同</a>庆」了。 </p> 
<p><strong>值得注意的是，此功能仅支持 iPhone 12 或更新的 iPhone</strong>  。 </p> 
<p>在更新到新版本后，系统会显示一个关于 Face ID 支持戴口罩解锁的欢迎界面，在这个界面中用户可以选择是否需要 Face ID 支持在戴口罩的情况下解锁手机，如果选择需要，系统就会重新扫描脸部特征。需要注意的是，在重新扫描脸部的时候并不需要佩戴口罩，如果第一次戴着眼镜录入了脸部信息，系统会在第二次录入时要求你摘下眼镜完成录入。扫描完成后，就可以在戴着口罩的情况下正常使用 Face ID 来解锁手机了。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_cbbe6b88bac541e2aefe03daf142fab8_img_000" referrerpolicy="no-referrer"></p> 
<p>更新到 iOS 15.4 开发者测试版后，我已经体验了一段时间戴着口罩解锁手机，每一次 Face ID 的反应速度都非常快，就像没有戴着口罩一样。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_931f1ebc06544c55ad11952810526cef_img_000" referrerpolicy="no-referrer"></p> 
<p>也许有人会感到好奇，通过 Apple Watch 解锁和戴着口罩使用 Face ID 有什么不同？答案是戴着口罩使用 Face ID 和正常使用 Face ID 的权限是一致的，而通过 Apple Watch <strong>只能解锁 iPhone</strong>，并不能在移动支付、登录验证等场景下生效。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_f7dbb0cd7685422189f9ca9f9629f900_img_000" referrerpolicy="no-referrer"></p> 
<p>此外，可能有部分用户想了解戴口罩解锁 Face ID 的功能是否支持 iPad，答案是不行，戴着口罩解锁 Face ID 的功能仅在升级到 iOS 15.4 的相关 iPhone 机型上可用。 </p> 
<p><strong>▍通用控制首次登场</strong></p> 
<p>通用控制功能可以让 Mac 和 iPad 共用一套键鼠，两台设备需要登录同一个 iCloud 账号，并且 Apple ID 需要开启两步验证。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_988f971bc64b4c0f9bba44b729cb7b26_img_000" referrerpolicy="no-referrer"></p> 
<p>在将 Mac 升级到 macOS 12.3 以及将 iPad 升级到 iPadOS 15.4 之后，系统会默认启用通用控制功能，如果你发现没有生效，可以到 Mac 的「系统偏好设置 - 显示器 - 高级」或者 iPad 的「设置 - 通用 - 隔空播放与接力」下查看并开启相关的选项。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_415ff81b27814d70a5a2ed27120f8dcf_img_000" referrerpolicy="no-referrer"></p> 
<p>当需要启用通用控制功能时，只需在 Mac 的「系统偏好设置 - 显示器」界面的左下方点击「添加显示器」，然后选中需要控制的设备即可。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_7ab30aec73c74298970119c385bba74c_img_000" referrerpolicy="no-referrer"></p> 
<p>通用控制开启后，我还可以拖动 iPad 的窗口位置或者调整菜单栏放置的窗口，就像一块外接显示器一样。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_edd4610c08b54aa68a78c0e3c204ba35_img_000" referrerpolicy="no-referrer"></p> 
<p>除了共享键鼠，通用控制功能还可以让我直接在 Mac 和 iPad 之间移动文件，我尝试了 JPEG、PDF、CSV 等格式的文件，都可以无缝地在两个设备之间拖动，这个过程非常流畅。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_0acbca202c8f43bba9a4b12b5d5e3958_img_000" referrerpolicy="no-referrer"></p> 
<p>与随航功能不同，通用控制并不能把 iPad 当作 Mac 的扩展或镜像屏幕，如果你还想使用随航，那么在「系统偏好设置 - 显示器 - 显示器设置」中也可以把 iPad 设置为「用作扩展或镜像显示器」。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_68904429b290461da7314f7543a740c0_img_000" referrerpolicy="no-referrer"></p> 
<p>通用控制功能只能在 2016 年或者更新的 MacBook Pro 和 MacBook、2018 年或者更新的 MacBook Air、2017 年或者更新的 iMac、2015 年的 5K iMac、iMac Pro、2018 年或者更新的 Mac mini、2019 年的 Mac Pro 等设备上使用，而 iPad 只能是 iPad Pro、第三代或者更新的 iPad Air、第六代或者更新的 iPad 和第五代或者更新的 iPad mini。  </p> 
<p><strong>▍其它值得关注的更新</strong></p> 
<p>iOS 15.4 和 iPadOS 15.4 新增了超过 30 个 emoji，包括融化的脸、偷看的脸、巨魔、<a class="project-link" data-id="339951" data-name="鸟巢" data-logo="https://img.36krcdn.com/20210810/v2_420710679e674b5dbe2074a8abff0de1_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/339951" target="_blank">鸟巢</a>、滑梯等。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_37765b3e7ab346e1b6f8891462029b32_img_000" referrerpolicy="no-referrer"></p> 
<p>图片来源：Emojipedia </p> 
<p>iCloud Keychain 钥匙串功能可以为登录信息添加笔记，除了网址、用户名、密码和二步验证码之外又多了一种信息类型。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_8edbe839221243cca6e651420f13316a_img_000" referrerpolicy="no-referrer"></p> 
<p>在已经适配 SharePlay 的应用中，可以通过共享菜单直接发起同播共享，不需要再通过 FaceTime 通话来发起。 </p> 
<p class="image-wrapper"><img data-img-size-val src="https://img.36krcdn.com/20220315/v2_40819117f2434effbdd9418f80f1f74f_img_000" referrerpolicy="no-referrer"></p> 
<p>iPad 的控制中心可以添加调节外接键盘背光亮度的控件，比如 Magic Keyboard 的背光亮度就可以通过这个控件来直接控制。 </p> 
<p>有开发者在 Twitter 上表示，在 iPhone 13 Pro 和 iPhone 13 Pro Max 机型上，即使开发者没有做过适配，更新 iOS 15.4 之后第三方应用也可以自适应 ProMotion。 </p>
