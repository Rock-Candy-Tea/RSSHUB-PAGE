<p>近年来，医疗健康产业创新不断涌现，并发展成为最活跃的行业之一，36氪见证和陪伴着这一领域创新企业的成长，并为行业带来专业的声音。</p> 
<p>为此，36氪推出了“医疗健康行业周报”这一栏目，接下来将定期发布，为读者提供资本、政策、财务、新产品等多角度的行业资讯。</p> 
<p>4月10日医疗健康行业周报请查收。行业周报收录了本周医疗健康领域「36氪创投频道」报道的融资新闻，以及其他值得关注的国内外行业新闻动态。</p> 
<h3><strong>投资融资</strong></h3> 
<ul> 
 <li><strong>亚瑟医药完成数亿元A轮融资</strong></li> 
</ul> 
<p>浙江亚瑟医药（Arthur Group）近日宣布完成数亿元人民币A轮融资，本轮融资由毅达资本领投，长兴金控、嘉兴科技城基金、杭州睿赢资产、湖州新骋等私募基金跟投。所得资金将用于建设亚瑟医药符合中国美国欧盟标准的注射剂生产线和以高端制剂上下游产品线为核心的管线搭建。同时，亚瑟也将致力推动国内首个采用连续化制剂工艺生产品种的FDA申报。亚瑟医药成立于2020年3月，目前已搭建了2个超2000平米的研发中心，并建立了4条原料药产线，超百人的研发团队，以及数十条以高端药物为核心的产品研发管线。</p> 
<ul> 
 <li><strong>华芯医疗完成数亿元A+轮战略融资</strong></li> 
</ul> 
<p>专注于AI药物及递送系统开发的剂泰医药（METiS Pharmaceuticals）近日宣布连续完成两轮融资，融资金额共计1.5亿美元。两轮融资由人保资本、国寿股权领投，红杉中国、五源资本、招银国际、光速中国、Monolith、峰瑞资本等新老股东跟投。METiS早期由<a class="project-link" data-id="1678491007366149" data-name="晶泰科技" data-logo="https://img.36krcdn.com/20220331/v2_6c61b7851d024ad6abe4d9257ceefc86_img_000" data-refer-type="1" href="https://36kr.com/project/1678491007366149" target="_blank">晶泰科技</a>孵化，致力于推动AI药物及递送系统平台开发。目前，METiS已申请了多项专利技术，并布局多条小分子和核酸研发管线，其中进展较快的管线预期于2023年开展关键性临床。</p> 
<ul> 
 <li><strong>锐明新药完成亿元新一轮融资</strong></li> 
</ul> 
<p>治疗性小分子眼科新药研发平台的<a class="project-link" data-id="1679798981923588" data-name="锐明新药" data-logo="https://img.36krcdn.com/20220401/v2_04ce2b8cdf0641aea41165288def0d5a_img_000" data-refer-type="1" href="https://36kr.com/project/1679798981923588" target="_blank">锐明新药</a>近日宣布完成由华润正大生命科学基金领投，<a class="project-link" data-id="1687050413098247" data-name="金鼎资本" data-logo="https://img.36krcdn.com/20220406/v2_b723950abd814bb390459aaa6349bba0_img_000" data-refer-type="1" href="https://36kr.com/project/1687050413098247" target="_blank">金鼎资本</a>跟投的亿元级融资。本次募集的资金将主要用于公司多款产品的IND申报与临床探索和新项目的引进。公司现有的管线产品都是自主研发的1.1类新药，在知识产权上布局有多个眼科疾病领域专利的保护；同时也将从海外引进先进的眼科项目。</p> 
<ul> 
 <li><strong>泽安生物医药完成1700万美元Pre-A轮融资</strong></li> 
</ul> 
<p>泽安生物医药（LTZ Therapeutics）近日宣布完成1700万美元Pre-A轮融资。本轮融资由险峰旗云领投，启明创投、<a class="project-link" data-id="1678535651193860" data-name="泰格医药" data-logo="https://img.36krcdn.com/20220331/v2_dc717b073fc041af8cbb48304e2be2cf_img_000" data-refer-type="1" href="https://36kr.com/project/1678535651193860" target="_blank">泰格医药</a>参与投资。所募资金将支持公司进一步开发技术平台、构建产品管线、扩充运营团队。LTZ是一家聚焦免疫疗法的生物技术公司，拥有减少免疫抑制、重组先天免疫、调节后天免疫的三大技术平台。</p> 
<ul> 
 <li><strong>奥昇医疗完成数千万元A轮融资</strong></li> 
</ul> 
<p>超声无创治疗医疗器械创新企业奥昇医疗近日完成数千万A轮融资。本轮融资由旸昀资本、涌铧投资联合领投，沂景资本、万沣资本跟投，Pre-A轮领投方幂方资本追加跟投。融资资金将主要用于推进公司前列腺增生无创消融的临床研究及空化超声在其他适应症的拓展探索。奥昇医疗2019年成立于深圳，是一家聚焦于医用超声诊断、治疗手术机器人开发的企业。</p> 
<ul> 
 <li><strong>大睿生物完成3300万美元A轮融资</strong></li> 
</ul> 
<p>大睿生物医药（上海）有限公司近日宣布完成3300万美元A轮融资，以推进自有递送平台的建立以及药物管线研发。本轮融资获得包括礼来亚洲基金、招银国际、Platanus、丰川投资和澜亭资本在内的投资机构支持。大睿生物成立于2021年，借助于递送平台进行对核酸药物的模块化研发，主要针对代谢、中枢神经、眼科、肿瘤等疾病领域。</p> 
<h3><strong>行业动态</strong></h3> 
<ul> 
 <li><strong>服务药企线上营销，「莲藕健康」连续2年实现净盈利</strong></li> 
</ul> 
<p>近年来，在带量采购、两票制、医保控费等一系列医改政策的影响，过往的“以药养医“、“带金销售”现象正在发生剧烈的改观，医药企业特别是跨国药企开始在院外市场寻求新的市场增长。 再者，传统的医药营销模式的弊病正在显现，互联网医院、医疗AI、数字疗法等技术和模式创新，激发药企向数字化营销转型，以互联网为依托的数字化营销正在成为许多大药企的重点业务，医药数字化营销的大潮已来。</p> 
<p>基于行业know-how积累，王燕雄在2018年创立了医药数字化营销企业——莲藕健康。王燕雄早年曾在地方卫生局和药监局工作，在2010年时创立了医药电商企业康爱多，后康爱多被上市公司太安堂3.5亿元收购，王燕雄随后离开再度创业。</p>

<p>“去年融资热起来以后，国内冒出来100多家合成生物学公司，<a target="_blank" rel="noopener noreferrer" href="https://36kr.com/project/1679777087951621">这里</a>面肯定有很多已经存在的公司，像以前做工业酶、原料药的，或者做玻尿酸的，现在加上了合成生物的标签。可能到了期望膨胀阶段吧，”一位VC投资人讲道。他2019年投资的合成生物学企业估值已经涨了20多倍，那时候火的品类还是大麻素CBD。</p>
