<div>   
<p>在 Steam Deck 首发体验视频中，尽管试玩游戏数量有限、不涉及 UI 界面，各路媒体、科技博主仍然给出了几乎一致的好评，知名科技博主 LinusTechTips 更是给出了颠覆性的评价： </p> 
<blockquote> 
 <p><strong>20 年来最具创新的游戏 PC。 </strong></p> 
</blockquote> 
<p>以游戏《Control》为例，LinusTechTips 的<a class="project-link" data-id="1657012" data-name="测试数据" data-logo="https://img.36krcdn.com/20220120/v2_0c2f1ce09db94b4f8ea8cc598fb2bb31_img_000" data-refer-type="1" href="https://p.36kr.com/space/2710010340?mp=zzquote" target="_blank">测试数据</a>显示，Steam Deck 的运行帧率高于目前的主流 Windows 掌机。另一位博主 Phawx 更是在视频展示了长达 10 分钟的多款游戏运行帧率对比，结果也是 Steam Deck 帧率领先。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,501" src="https://img.36krcdn.com/20220214/v2_dc619a4a22a44e2983820223e5b65ce1_img_000" referrerpolicy="no-referrer"></p> 
<p>掌机的另外一个体验要素——续航，Steam Deck 的实机表现也不错，Phawx 的实际测试显示《鬼泣5》的实际续航可以达到 2 小时，而 LinusTechTips 则表示其平均续航在 3.3~ 8 小时内，不同的游戏设置会显著影响续航。 </p> 
<p>硬件性能和续航表现都不错，Steam Deck 真的要成了？ </p> 
<p class="image-wrapper"><img data-img-size-val="1080,608" src="https://img.36krcdn.com/20220214/v2_22cf1d5f76394c4d9a748187a35c7294_img_000" referrerpolicy="no-referrer"></p> 
<p>现在下断言还太早，软件系统才是 Steam Deck 最大的限制，基于 Linux 系统的 SteamOS 3.0 是这台游戏机最强的矛，也是它最脆弱的地方——原生支持 Linux 系统的游戏太少。 </p> 
<h2 label="一级标题" style><strong>系统才是 Steam Deck 最大的「限制」 </strong></h2> 
<p>7 年前，同样是 Steam 游戏机，同样基于 Linux 系统，同样是运行主机游戏。 </p> 
<p>实际表现却和今天大不一样，到停产时合计销量也没有超过 50 万台，这对于一家游戏大厂而言，几乎是一个无法接受的结果。 </p> 
<p>销量低迷的原因有很多，其中有两点最为重要，首先是 Steam 初次推出游戏机时，更多的是以一位平台运营者的角色出现，除了自己制造游戏机，其他厂商也可以通过授权制造。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,605" src="https://img.36krcdn.com/20220214/v2_b63c946e87964d328accaea5e2880ae2_img_000" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲ <span style="letter-spacing: 0px;">最初的 Steam 游戏机. 图片来自：wiki </span></p> 
<p>这导致最终到玩家手中的产品规格混乱，更重要的是没有多少玩家愿意用 Linux 系统，可以直接玩的游戏实在太少。 </p> 
<p>Steam Deck 更像是吸收初代产品失败教训下的产物，瞄准便携掌机市场，硬件、系统、游戏商店等都由 Steam 母公司 Valve 控制。 </p> 
<p>至于 Linux 系统缺乏游戏厂商支持的问题，则通过另一项重要技术 Proton 解决。 </p> 
<p class="image-wrapper"><img data-img-size-val="913,431" src="https://img.36krcdn.com/20220214/v2_a704b69e085a488f95e8d7f05dc84043_img_000" referrerpolicy="no-referrer"></p> 
<p>初代 Steam 游戏机失败后，Valve 并没有放弃 SteamOS， 反而持续投入更新，并于 2018 年推出了 Proton 兼容技术，旨在将 Windows 游戏以兼容的方式在 Linux 系统中运行。 </p> 
<p>Proton 大大扩张了 SteamOS 可运行的游戏阵容，Valve 还以它建立了一个技术和游戏社区，开发者和游戏爱好者们得以参与其中，以 Linux 系统体验Windows 游戏。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,578" src="https://img.36krcdn.com/20220214/v2_1f49e6f7c6dc4ea486236cb6aac0ac4b_img_000" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲ <span style="letter-spacing: 0px;">Panton 社区主页 </span></p> 
<p>162207。 </p> 
<p>这是 Proton 社区目前已经测试的游戏数，但并不意味着它们都能完<a class="project-link" data-id="527871" data-name="美运" data-logo="https://img.36krcdn.com/20210813/v2_ae8bd4b232ef4a589280a47c3c2e189d_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/527871" target="_blank">美运</a>行，社区设定了一个评价体系，分别是白金、金、银、铜、无法运行五个档位。 </p> 
<blockquote> 
 <p><strong>白金：完美运行金：调整后完美运行银：有一部分小问题，但可玩铜：可玩，但经常会遇到游戏崩溃等问题。 </strong></p> 
</blockquote> 
<p>比如《<a class="project-link" data-id="331494" data-name="赛博" data-logo="https://img.36krcdn.com/20210810/v2_ad0b14836e86460880d64547a1f1125a_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/331494" target="_blank">赛博</a>朋克 2077》、《GTA5》、《辐射 4》都取得了金等级评分，调整后可运行。 </p> 
<p class="image-wrapper"><img data-img-size-val="756,727" src="https://img.36krcdn.com/20220214/v2_373f5320a70242d89fa49ee4946868d4_img_000" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲ <span style="letter-spacing: 0px;">部分获得金及以上评级的游戏 </span></p> 
<p>Valve 开放的态度，持续推动 Proton 技术和社区发展，一定程度上扭转了 Linux 系统缺乏游戏的主流印象。 </p> 
<p>另一方面，这也意味着 Steam Deck 支持的游戏阵容和 Windows 端有一段距离，你在 Steam 买了的游戏，到了 Steam Deck 不一定能玩。 </p> 
<p>更重要的是根据 Proton 数据库显示，热门游戏《命运 2》、《绝地求生 》、《Apex Legends》的评级都是不可玩。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,608" src="https://img.36krcdn.com/20220214/v2_0fafd9027a8c4a579826923fa242d8a4_img_000" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲ <span style="letter-spacing: 0px;">《命运 2》 </span></p> 
<p>除了部分单机游戏，Steam Deck 在多人游戏上遇到的问题恐怕更大。 </p> 
<p>一般而言，像《堡垒之夜》等多人竞技游戏，为了保证玩家体验，下载安装时会自动匹配安装一种名为 Easy Anti-Cheat 的反作弊技术，问题恰恰出现在这里。 </p> 
<p>Steam Deck 与 Easy Anti-Cheat 之间存在一些适配问题，游戏开发者仍然要针对机器做特别适配，《战锤：末世鼠疫 2》的开发者就表示适配难度比想象高，工作量不小。 </p> 
<p>《堡垒之夜》的 CEO 蒂姆・斯威尼则表示不会为 Steam Deck 更新适配，不支持反作弊技术可能会影响现有玩家体验，而 Linux 系统游戏本身也是一个小众市场。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,608" src="https://img.36krcdn.com/20220214/v2_e4d6cb25090e45d3b0ec384eafb78a9a_img_000" referrerpolicy="no-referrer"></p> 
<p>多人联机游戏、部分热门游戏的缺位，往往意味着购入 Steam Deck 的玩家减少，市场一旦太小，愿意适配的游戏开发者就更少了，最终陷入一个不断负面化的循环。 </p> 
<p>可以预见的是，推动 Proton 技术发展，兼容反作弊技术，持续测试游戏和扩大游戏阵容将会是 Steam Deck 的常态。 </p> 
<p>游戏，才是一款游戏机的核心。 </p> 
<h2 label="一级标题" style><strong>PC 游戏掌机，是冒险也是机会 </strong></h2> 
<p>吸取了初代 Steam 游戏机失败教训的 Valve，自然清楚游戏阵容的重要性，Steam Deck 正式发货前，还时不时有媒体报道 Valve 在更新游戏名单。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,608" src="https://img.36krcdn.com/20220214/v2_de1cb702e8b84e21a24c93d2317efa51_img_000" referrerpolicy="no-referrer"></p> 
<p>Proton 游戏运行数据库中有不少游戏是玩家、第三方开发者测试的，其中部分游戏仍然需要简单调整才能完美运行，这也会导致官方版本、调整版本等各种版本出现，普通玩家根本无法分清楚。 </p> 
<p>这时候就要靠 Valve 了，通过完善的检测确认最适合的版本，达成「完美 30 帧运行所有游戏」的目标。 </p> 
<p>Valve 在 Steam Deck 官网贴出了兼容分级，一共分为通过验证、可玩、不支持、未知四个档位，这些都会在 Steam Deck 游戏商店中以显眼的标识提示玩家。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,439" src="https://img.36krcdn.com/20220214/v2_e8e1b0af50fa4330954eb4b3a2d2ba8b_img_000" referrerpolicy="no-referrer"></p> 
<p>最佳的通过验证选项对游戏有一系列测试要求，主要包括输入、显示、无缝体验、系统支持四个维度。 </p> 
<p>检测标准要求游戏所有交互方式应该和 Steam Deck 匹配，包括按钮、触控、唤起虚拟键盘，游戏全程都应该正确流畅地触发交互。不然即便游戏能以默认分辨率（1280x800 或 1280x720）<a class="project-link" data-id="637483" data-name="完美显示" data-logo="https://img.36krcdn.com/20210814/v2_8068dc4ac514472686f0c6cbf133b45a_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/637483" target="_blank">完美显示</a>，体验也不好。 </p> 
<p>Windows 以兼容性闻名，这也导致了游戏交互方式复杂，键鼠或是手柄，不同的游戏有不同的选择，比如 FPS 射击游戏玩家往往更喜欢键鼠操作，更稳定舒服。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,608" src="https://img.36krcdn.com/20220214/v2_4dcbcef3603346c3a502352404d2d148_img_000" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲ <span style="letter-spacing: 0px;">图片来自：PCGamesN </span></p> 
<p>Steam Deck 要把 Windows 游戏塞进一款掌机中，考验的不仅仅是硬件、系统支持，后续的交互测试也很重要。 </p> 
<p>兼容性方面则要求不能有兼容性警告，使用 Proton 技术兼容的游戏应该支持相应的组件，包括反作弊组件。 </p> 
<p>从测试到云服务、串流服务、社区支持，再加上还未露出全貌的创意工坊 Mod 支持，可以看出 Valve 在尽力解决一切可能出现的问题，真正让玩家随时随地、开机即玩。 </p> 
<p>甚至于在自家服务之外，Valve 也保持了了一如既往的开放性。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,545" src="https://img.36krcdn.com/20220214/v2_5def610113214d44b3c1ac002998cc4f_img_000" referrerpolicy="no-referrer"></p> 
<p label="图片描述" classname="img-desc" class="img-desc" style>▲ <span style="letter-spacing: 0px;">Steam Deck 的价格. </span></p> 
<p>Steam Deck 三档价格主要区别是内存，引发了玩家的内存忧虑，第三方内存价格虽然低，但恐怕不能保证性能体验，只能被动去买价格高出近 2000 元的 512GB 版本。 </p> 
<p>实际上 Valve 早就回应了这一问题，eMMC 内存和 SD 卡并没有那么慢，实际测试游戏加载时间仅慢 12% 和 18%。 </p> 
<p class="image-wrapper"><img data-img-size-val="1080,522" src="https://img.36krcdn.com/20220214/v2_f843ac49c6a3470599cd1f4c5d795bc8_img_000" referrerpolicy="no-referrer"></p> 
<p>拆卸维修也是一样，尽管 Valve 不建议大家自己修理更换部件，但还是公布了完整的视频，教大家更换摇杆等部件。 </p> 
<p>就算是觉得 Linux 兼容性一般，游戏阵容中不包含自己想玩的游戏，Valve 也承诺玩家可以为 Steam Deck 刷入 Windows 系统，就可以体验所有 Windows 游戏了。 </p> 
<p>或许，Steam Deck 采用 Linux 系统，除了保持简洁、专为游戏设计，还有降低成本的原因，毕竟 Windows 授权也是一笔费用。 </p> 
<p>这一切都在告诉玩家，Valve 在想尽办法降低游玩成本，提升玩家体验。 </p> 
<p>掌机 PC 是一个全新的的领域，这既是机会也是冒险，Valve 要参与硬件、系统兼容、测试等全流程，都做好了才能让更多游戏加入，让玩家真正满意。 </p> 
<p>到那时，Steam Deck 才能算是成功。 </p>
