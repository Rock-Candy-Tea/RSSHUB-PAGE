<p>在不久前的微软财报电话会议上，该公司CEO纳德拉曾透露，目前已有超过1000万用户通过Xbox Cloud Gaming来玩云游戏。很显然，这些用户中很大一部分是因为Xbox Cloud Gaming是微软为Xbox Game Pass Ultimate（下文简称为XGPU）高级订阅服务所提供的福利，本着尝鲜的态度来体验。</p> 
<p class="image-wrapper"><img data-img-size-val="600,337" src="https://img.36krcdn.com/20220507/v2_68489aae9a0d49bfb1c6af45aa717413_img_000" referrerpolicy="no-referrer"></p> 
<p>不过或许在不久的将来，微软的Xbox Cloud Gaming或将迎来一大批真正的玩家。</p> 
<p>日前微软与Epic Games方面共同宣布，任何拥有运行Android、iOS或iPadOS设备的用户，都将可以使用Xbox Cloud Gaming来免费体验《堡垒之夜》，并且不需要额外购买XGPU。</p> 
<p>微软方面在官方社区Xbox Wire上是这样说的，“不需要安装或会员资格，你需要做的就是在浏览器上访问一个特定网址，然后用微软账户登录，与朋友聚会或在《堡垒之夜》中获得下一个大逃杀胜利”。也就是说只需要有一个微软账号，就可以在Xbox Cloud Gaming平台体验《堡垒之夜》了。</p> 
<p>别看只是玩家能在Xbox Cloud Gaming上免费玩到一款游戏这个小小的变化，但其背后很有可能会是移动互联网生态最中坚的应用商店经济被彻底颠覆的征兆。</p> 
<p class="image-wrapper"><img data-img-size-val="600,338" src="https://img.36krcdn.com/20220507/v2_3da35f7a52674c73838fd93d944e95aa_img_000" referrerpolicy="no-referrer"></p> 
<p>事实上，微软与Epic Games此次合作并非凭空而至，早在去年Epic Games和苹果官司的庭审上，微软方面就不仅仅是声援，更选择了为Epic Games出庭作证。而具体到《堡垒之夜》登陆Xbox Cloud Gaming，则与微软在今年GDC游戏开发者大会上宣布成立单独的云游戏部门有关，并且该部门的工作是“将与游戏工作室合作开发云原生游戏，并将它们引入Xbox生态系统”。</p> 
<p>作为微软的亲密合作伙伴，Epic Games用《堡垒之夜》这样一个拥有数以亿计用户，且风靡全球的爆款游戏来“捧个人场”显然就是理所当然的事情了。</p> 
<p>作为鼎鼎有名的游戏大厂，微软自己其实就有一大批脍炙人口的游戏，但《堡垒之夜》却有一个其他游戏罕见的优势，那就是这是一款跨平台游戏。尽管跨平台游戏也并不少见，但支持跨平台联机的游戏无疑就不那么多了，而跨平台联机天生就契合云游戏的特质，毕竟后者是基于屏幕、而不是基于平台的。更妙的是，在如今支持跨平台联机的游戏中，《堡垒之夜》还是玩家数量最多的，这也就意味着会有大量的玩家会被引到Xbox的云游戏体系中，并有望加速微软云游戏业务的成熟。</p> 
<p class="image-wrapper"><img data-img-size-val="600,336" src="https://img.36krcdn.com/20220507/v2_05380c939f2047b39a3123c517aad675_img_000" referrerpolicy="no-referrer"></p> 
<p>而至于另外一方的Epic Games，在这一合作中几乎可以说是获利巨大，并可能是最大的赢家。Epic Games的收获用一句话来说，就是《堡垒之夜》终于曲线登陆iOS端了。</p> 
<p>此前在2020年夏季，Epic Games更新了旗下热门游戏《堡垒之夜》，但其中设置了向Epic Games直接付款的选项。但这一试图绕过App Store应用内购买（IAP）机制的操作，很快就招致了苹果方面的打击，当天下午《堡垒之夜》就被从App Store下架，此后苹果方面甚至还吊销了Epic Games的开发者帐户与证书。</p> 
<p>由此，Epic Games与苹果关于App Store垄断的大戏正式拉开帷幕。尝试绕过应用内购买机制的代价，是《堡垒之夜》被苹果一直下架至今，而这则代表这款游戏自此就无法扩大在iOS端的用户数量，甚至于用户群体还可能出现萎缩。彼时有分析机构给出的数据，是Epic Games因为被苹果方面下架，所导致的单月损失或高达2500万美元以上。</p> 
<p class="image-wrapper"><img data-img-size-val="600,338" src="https://img.36krcdn.com/20220507/v2_d8baf9f570b64dfea9f1f911940b1d5b_img_000" referrerpolicy="no-referrer"></p> 
<p>考虑到《堡垒之夜》本身其实已经巅峰不再，从2020年8月到2022年5月的这18个月的时间里里，Epic Games因为失去了iOS端《堡垒之夜》导致的收入损失或将达到4.5亿美元之巨。与体量近似的《王者荣耀》来计算，也显然也是一笔不小的数字，这对于试图发力数字游戏分发的Epic games来说，自然也是难以忍受的事情。</p> 
<p class="image-wrapper"><img data-img-size-val="600,594" src="https://img.36krcdn.com/20220507/v2_ed568ba4438a4b75aa18a0651218c09a_img_000" referrerpolicy="no-referrer"></p> 
<p>借助Xbox Cloud Gaming平台，《堡垒之夜》终于将能够与iOS端玩家重逢了。更何况微软并不是苹果，收取抽成的可能性微乎其微，这就等于说Epic Games此次真的能够实现当初绕过IAP的构想。</p> 
<p>合着与苹果打了两年的官司后，Epic Games等于是拿着18个月《堡垒之夜》IOS端的收入，掀起了一阵针对App Store体系的舆论浪潮，并让苹果实现IAP体系的关键支撑——反转向条款(anti-steering provisions)被从法律层面禁止。甚至于使得苹果不得不在部分国家和地区，拟开放第三方支付渠道。</p> 
<p>站在这样的角度来看，微软此次与Epic Games的合作，苹果几乎堪称是一败涂地。尽管微软与Epic Games均未提及付费相关的信息，但重新走苹果的IAP支付仅仅只存在理论上的可能性了。</p> 
<p>对于苹果来说，兜兜转转一圈下来，原本下架《堡垒之夜》是其警告开发者绕开IAP体系只有死路一条的的杀鸡骇猴，但现在这只“鸡”居然通过云游戏活了过来，可不就是亏大了。苹果方面或许没有想到，去年6月同意Xbox Cloud Gaming登陆iOS的决定，会在一年后迎来如此戏剧性的一幕。</p> 
<p class="image-wrapper"><img data-img-size-val="600,304" src="https://img.36krcdn.com/20220507/v2_3c40952e1af54e468679be97df7fa43e_img_000" referrerpolicy="no-referrer"></p> 
<p>并且这一通操作的后果同样也相当直白，那就是苹果方面对于App Store的控制力将会下降，甚至可能会出现无法控制开发者行为的现象。在高度封闭且要听苹果指挥棒起舞的iOS生态内，出现公然对抗苹果、却只付出相对微小代价的“刺头”，接下来或许就是“秦失其鹿，天下共逐之”的剧本了。</p> 
<p>所以接下来，苹果方面大概率是要站出来，将此次微软与Epic Games的合作搅黄，不然的话，其他游戏厂商如果也有样学样，不就乱套了。</p> 
<p>但问题是，苹果要如何反制呢？云游戏从本质上来说，与Netflix等视频流媒体是一样的，并且像2020年冬季那样直接禁止云游戏APP上架，在如今反垄断的市场背景下俨然已经不太可能。</p> 
<p>不过相信苹果乃至谷歌，都不会对微软与Epic Games的这番操作无动于衷，因为从根本上来说，这是微软方面在从应用商店手中争夺游戏分发的权力。众所周知，游戏类应用虽然仅仅只是应用商店中的一小部分，但是无论App Store还是Google Play，抑或国内的各种安卓第三方渠道，游戏应用所贡献的收入都占据了绝对的主力。</p> 
<p class="image-wrapper"><img data-img-size-val="600,599" src="https://img.36krcdn.com/20220507/v2_3c6dee5272f0491f9b1fe94f5b79203e_img_000" referrerpolicy="no-referrer"></p> 
<p>根据Sensor Tower公布的数据显示，去年苹果方面从App Store中获得的收入中有69％来自游戏。从某种意义上来说，正是得益于游戏类应用的贡献，各应用商店才有了不断运营并更新迭代的能力。现在《堡垒之夜》登陆Xbox Cloud Gaming，则给其他游戏厂商带来了一个不一样的选择。</p> 
<p>一旦游戏开发者都去与云游戏平台合作，那么应用商店的收入自然也就会下降了。反过来说，如果云游戏攻陷不了移动端，甚至在移动端都不能立足，自然也就不用再奢谈未来了。</p> 
<p class="image-wrapper"><img data-img-size-val="600,338" src="https://img.36krcdn.com/20220507/v2_5eb5c45257ed4512ad6901d480c5baea_img_000" referrerpolicy="no-referrer"></p> 
<p>因此接下来，苹果、谷歌、微软，乃至Epic Games极有可能将会联袂奉上可能决定云游戏行业未来的大戏。</p>
