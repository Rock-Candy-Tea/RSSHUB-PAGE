<p>又一年过去了，你的 2021 年过得怎么样？步入新的一年，<a class="project-link" data-id="41920" data-name="少数派" data-logo="https://img.36krcdn.com/20210807/v2_7aec430dc7e3475285906c8178551f5d_img_000" data-refer-type="2" href="https://36kr.com/projectDetails/41920" target="_blank">少数派</a>依然想与你对刚刚过去的 2021 年来一次回顾。今年依旧是特殊的一年，疫情的阴霾仍然没有散去，效率工作、个人健康、安全隐私等话题，仍然是许多人关心和关注的议题。 </p> 
<p>从工具到服务、再到系统，少数派在过去的一年中崭露头角的多款 App 中，选出了 11 位「年度之选」。不管 2021 它们是否在你的主屏上留下了足迹，都希望新的一年，这些 App 能够帮你高效工作、品质生活。 </p> 
<h2><strong>🏆 年度之选：跨平台服务</strong></h2> 
<p><strong>▍Obsidian</strong></p> 
<p>卡片笔记与双向链接的「出圈」，让笔记软件领域再开启新的篇章，而 Obsidian 无疑是一众同类软件中的佼佼者。尽管只有两位核心开发者，Obsidian 凭借其开放的架构设计、活跃的社区氛围，以及高频的功能更新与体验优化，逐渐成为了<a class="project-link" data-id="3969555" data-name="大众" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3969555" target="_blank">大众</a>尝试卡片笔记与双向链接的最佳选择。 </p>

<p><strong>▍VS Code</strong></p> 
<p>🔗 https://code.visualstudio.com </p> 
<p>发布于 6 年前的 VS Code 能成为 2021 年的年度之选，正表明用户对它的喜爱不减。VS Code 在 2021 年持续进化，不断拓展自己的应用场景。 </p> 
<p>尽管在实际开发工作流中的作用还有待验证，但 VS Code for the Web 可以让用户开始畅想属于未来的文本编辑器；不断优化与 Jupyter 结合使用体验（交互窗口，支持调试 Jupyter notebook 代码），让数据科学工作者更加得心应手；对于写作者最重要的 Markdown 方面，也实现了对 KaTeX 的支持，优化了图片预览体验。如此频繁且务实的更新节奏，让我们更加期待 VS Code 的 2022 年。 </p>

<h2><strong>🏆 年度之选：iOS / iPadOS 应用</strong></h2> 
<p><strong>▍NOMO RAW</strong></p> 
<p>与 NOMO CAM 主打胶片模拟效果不同，来自同一开发商出品的 NOMO RAW 是一款专业相机应用。16bit 图像处理流程、支持 Apple ProRAW、全面的手动调节能力、简单易用的交互界面，不管是摄影新手还是专业用户，都能轻松上手 NOMO RAW。与同类应用不同的是，NOMO RAW 还提供了多个色彩模拟方案，能够让你一键模拟如同徕卡、索尼等多款相机的拍摄效果，找到自己喜欢的视觉风格。 </p>

<p><strong>▍隐私洞见</strong></p> 
<p>隐私一直是各大厂商和许多用户关注的议题，越来越多的操作系统和应用也都开始在隐私功能上发力，今年还推出了 《中华人民共和国个人信息保护法》 这一重要法案。如果要说过去一年隐私话题热度最高的 App，《隐私洞见》当仁不让。 </p> 
<p>利用 iOS 15 最新推出的 App 隐私报告功能，《隐私洞见》将原本难懂的 JSON 数据，转化成了人人都能明白的可视化图表，吃上了「第一只螃蟹」，让不少 App 的流氓行径无所遁形，也启发了更多相关 App 问世。不管是社会影响力还是关注度，《隐私洞见》成为年度 App 都当之无愧。 </p>

<h2><strong>🏆 年度之选：macOS 应用</strong></h2> 
<p><strong>▍Parallels Desktop</strong></p> 
<p>如果打算在 Mac 运行 Windows 软件，Parallels Desktop 是一个绕不过的方法。Parallels Desktop 是 macOS 下的虚拟机工具，可以虚拟包括 Windows 在内的绝大部分系统，让我们可以用上其他系统以及专有软件。 </p> 
<p>从 2020 年开始 Apple 就进入了两年的芯片过渡期，而 Parallels Desktop 也于第一时间通过官方的 API 适配了最新的芯片——让我们得以在 M1 上继续「愉快」地使用其他平台的专有应用。 </p>

<p><strong>▍Infuse</strong></p> 
<p>当我们讨论 Apple 平台的视频播放器时，Infuse 一定是 Big Players 中的一员。Infuse 不仅支持了 Apple 全家桶中能够播放视频的全部设备，还几乎支持所有常用格式的视频文件。同时，Infuse 还是一款剧集管理工具，能够自动下载元数据，还能保持影视资源库的整洁有序。 </p>

<p><strong>▍PowerToys</strong></p> 
<p>🔗 https://github.com/microsoft/PowerToys/releases/ </p> 
<p>又有谁能比 Microsoft 更懂 Windows 呢？从 2020 年<a class="project-link" data-id="3967413" data-name="微软" data-logo="https://img.36krcdn.com/20200916/v2_811751a081924fa9af8741ce120bd7bf_img_png" data-refer-type="2" href="https://36kr.com/projectDetails/3967413" target="_blank">微软</a>开始恢复更新 PowerToy 工具以后，PowerToy 一直处于更新迭代的状态，不断给 Windows 加入实用的新功能。PowerToys 今年依然加入了不少新特性，比如：为 PowerToy Run 引入外部插件、鼠标实用工具以及一键关闭麦克风摄像头等功能。 </p> 
<p>当然 PowerToy 也考虑到了不同用户的不同的技术背景，无论是专业用户还是普通消费者都可以感受到 Windows 下自动化带来的乐趣。 </p>

<p><strong>▍WSA</strong></p> 
<p>今年 Microsoft 为我们带来了全新的操作系统 Windows 11，但对于不少人而言这个新系统好像只是「换皮」。但正式上线的 Windows Subsystem for Android（以下简称 WSA）却带来了 Windows 生态下的又一次变革——用 Android App 丰富 Windows 的生态。 </p>
