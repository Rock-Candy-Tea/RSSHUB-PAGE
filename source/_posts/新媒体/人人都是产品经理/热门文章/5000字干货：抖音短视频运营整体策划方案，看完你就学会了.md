<blockquote><p>编辑导读：抖音作为人们最常用的短视频软件之一，凭借着优质的内容获得了用户的喜爱，不少企业也开始在抖音上运营自己的账号。但是，很多人并不知道具体应该如何运营。本文作者从自身工作经历出发，总结了一些经验，与你分享。</p></blockquote>
<p><img data-action="zoom" class="size-full wp-image-4264134 aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2020/11/AIsHcL4OSkI9JmUbcsaM.jpg" alt width="900" height="420" referrerpolicy="no-referrer"></p>
<p>抖音短视频很火，不管是个人、公司都想通过短视频的流量来变现。但是抖音已经步入成熟期，过去随便发个视频就能火的时代已经过去了，门槛越来越高。没有好的创意、内容、运营策略来长期维护是很难做起来的。</p>
<p>最近，很多朋友跟我说，他们公司老板要做抖音短视频，申请了账号，但是一直做不来，发了视频没人看。</p>
<p>现在，老板让我把抖音账号运营起来，我应该怎么办？不知道从哪入手？完成没有思路。</p>
<p>基于这样的需求下，我给提供一个抖音短视频运营的整体方案，结合实际操作的案例，尝试把抖音短视频运营的流程讲明白，但在真正的实操过程中，可能还会遇到很多细节上的问题，欢迎私信沟通，在此希望可以给大家提供一些思路，作为参考。</p>
<h2 id="toc-1">一、抖音平台用户分析</h2>
<h3>1. 为什么要做抖音平台的用户分析？</h3>
<p>因为要了解喜欢刷抖音的那一批人里面，会有多少是你们的目标受众？他们都喜欢看什么类型的内容？如何通过内容去吸引他们的关注，粘住他，让他成为你们忠实粉丝，最后成功变现。</p>
<h3>2. 需要分析哪些数据？</h3>
<ul>
<li>基本的用户属性，年龄、性别、消费习惯等。</li>
<li>地域分布，省份、城市。</li>
<li>用户活跃时间段。</li>
<li>用户兴趣内容偏好。</li>
</ul>
<h2 id="toc-2">二、竞品分析：对手在抖音平台的情况</h2>
<h3>1. 竞品的抖音账号内容展现形式分析</h3>
<p>因为同⾏就是我们的前辈，可以减少大量的试错成本，如果他们的内容能够做起来，并且做得还不错，那我们可以直接换个形式复制过来。</p>
<p>如果做得很差，那他们做过的形式，我们就不要再尝试了。不管做什么，记住，要时刻死盯同行。</p>
<p>上次有个朋友是做教育行业的，我们平时可以看到很多教育行业在抖音上的内容输出形式，大概可以分为以下三种：</p>
<ul>
<li>知识讲解（无人物出境）+配音。</li>
<li>知识讲解（人物出境）+配音。</li>
<li>剧情模式，通过剧情演绎内容。</li>
</ul>
<p>如果你也是做教育行业的抖音账号，那么内容输出的形式基本也是这三种，然后结合自己公司的资源、团队、成本选定。</p>
<h3>2. 抖音账号分析，可以通过哪几个纬度？</h3>
<ul>
<li>固有数据：作品数量、粉丝量（粉丝属性）、获赞</li>
<li>更新频率：作品发布的规律</li>
<li>作品内容：人设、剧本、拍摄、演员、画面、声音、封面、标题等</li>
<li>作品数据：播放、点赞、评论、分享</li>
</ul>
<h2 id="toc-3">三、公司现有的抖音账号诊断分析</h2>
<p>如果公司现在有抖音账号，但是没有运营起来，可以按照以上分析竞品的纬度来分析，找到原因，然后改进。</p>
<ul>
<li>账号：标签、粉丝属性（性别、年龄）、粉丝活跃</li>
<li>视频内容：内容关键词、标签、爆款视频情况、更新频率</li>
<li>人设、内容形式、DOU投放</li>
<li>如果有过违规、被限流，建议重新申请账号</li>
<li>违规、限流、封号等问题，请参考【抖音社区自律公约】：7类违规词，8大违规行为</li>
</ul>
<p>PS：如果公司目前没有抖音账号，可以从注册开始，详情请往下看。</p>
<h2 id="toc-4">四、抖音短视频运营策略【重点部分】</h2>
<h3>1. 账号注册，注册蓝V号，还是个人号？</h3>
<p>企业号——蓝V表示及认证信息，官方认可；昵称锁定保护，企业号之间昵称不允许重名，先到先得，昵称全匹配搜索时置顶显示，蓝V认证费用600元/年（认证不通过不能退钱）。</p>
<p>部分功能介绍如下：</p>
<p>个人号——无上述功能权限，但免费。</p>
<p>建议：人力、物力、财力允许的情况下，可以做矩阵号，蓝V+个人。</p>
<h3>2. 账号搭建</h3>
<ul>
<li>顶部背景图设计</li>
<li>头像设计</li>
<li>昵称设置</li>
<li>个签签名</li>
<li>其他资料设置等</li>
<li>如果是蓝V账号，可以添加官网连接、联系电话，直接与用户建立联系。</li>
</ul>
<h3>3. 人设打造</h3>
<p><strong>1）什么是人设？</strong></p>
<p>人物设定简称人设，在电影、喜剧、游戏等场景中，会有各类人物设定。明星等公众人物刻意塑造的形象。</p>
<p>比如，好男人/国民老公/模范丈夫：文章、陈赫、薛之谦、吴秀波，演员受影视剧角色塑造的影响大。</p>
<p>小鲜肉：鹿晗、吴亦凡等。</p>
<p><strong>2）抖音为什么需要做人设？</strong></p>
<p>抖音需要人设的理由有千万条，每个人都有自己见解，这里不做阐述。但说来说去，就一句话，每个人设代表着kol的定位，比如：</p>
<ul>
<li>代古拉k的人设就是毁舞专家。</li>
<li>办公室小野的人设就是办公室的小奇葩（饮水机煮火锅，奇葩）。</li>
</ul>
<p><strong>3）企业在选择人设的时候，就需要仔细衡量产品/品牌与Kol的人设标签契合度是否匹配</strong></p>
<p>PS：好的人设可以迅速圈粉，吸引受众的道具，同时可以提高转化率。</p>
<p><strong>4）账号定位</strong></p>
<p>账号定位做过公众号运营的就很容易理解，简单的来说就是：专注于一个垂直领域进行内容输出。</p>
<p>如果账号是定位于美食的，就坚持只发美食相关的视频；定位于舞蹈的，就坚持发舞蹈；定位于正能量语录的，就坚持发正能量；定位是穿搭的，就坚持发穿搭……内容越专业，越垂直，吸引到的粉丝，也就越精准，转化率相对来说就很高。（前期最好要做原创内容）</p>
<p>除了垂直领域的内容，要时刻关注热点题材、热门视频，容易获得流量。</p>
<p>PS：垂直定位，一个账号只做一个领域的内容，如果公司的业务较多，建议做矩阵号。</p>
<p><strong>5）账号运营，提升播放量</strong></p>
<p>抖音最大的魅力在于，抖音的流量分配是去中心化的，我们每个人刷到的抖音内容都是不一样的，能让人一直刷一直爽。</p>
<p>同理，你拍的任何一个视频，无论质量好还是质量坏，发布了之后一定会有播放量，从几十到上千都有可能。这个就是经常提到的：流量池。抖音给每一个作品都提供了一个流量池，视频传播的效果好与坏，就取决于你的作品在这个流量池里的表现。</p>
<p>抖音评价你的作品在流量池中的表现，会参照4 个标准： 完播率、点赞量、评论量、转发量。</p>
<p>账号运营实操的过程中会涉及到很多的细节，但是在写方案的时候不需要把所有的细节都体现出来，主要体现以下3点即可：</p>
<p>除了视频内容本身以外的运营细节：</p>
<ul>
<li>写好标题：吸引用户浏览的重要因素，别人一看你的标题新鲜有趣，才会继续看看你发的视频内容，同时也能引导用户留言和互动；</li>
<li>选好封面：最好统一、吸睛；</li>
<li>BGM：根据剧情切换。可以是欢乐、幽默、煽情及热门音乐；</li>
</ul>
<p>免费的推广方式：</p>
<ul>
<li>动用内部员工：用自己的个人小号，看完视频之后进行点赞、评论、分享。</li>
<li>其它新媒体渠道：公众号、社群等。</li>
</ul>
<p>付费的推广方式：</p>
<ul>
<li>Dou+是抖音官方推出唯一通过购买增加流量方式，为品牌找寻合适的用户，节省大量精力和时间；（100元，最低5000个播放量）</li>
<li>根据账号的运营情况，做好推广预算；</li>
<li>每个视频DOU+预算控制在300元；小技巧：1个小时投100，总共投三轮，当数据下滑，再投100；</li>
</ul>
<p>以下是我实操的投放数据：</p>
<p>推广结束之后还能看到你花费的钱究竟获得了多少展示量、播放量、互动量（关注、转发、分享数据）。</p>
<p>用户管理：</p>
<p>抖音蓝V账号，在企业营销方面提供更多赋能，在PC端登录，更方便操作。可以平台内做到精细化的用户运营，有些功能类似微信公众号。</p>
<p>私信自动回复：</p>
<p>私信自动回复是当用户第一次向企业号发送私信时，企业号设置自动回复的内容。设置一段好的自动回复，给潜在用户留下深刻印象，让首次对话即使在非工作时间的用户也有良好体验，甚至可以引导用户进行下一步行为。（设置跳转连接、菜单栏）</p>
<p>关键词自动回复：</p>
<p>关键词自动回复则是根据用户发送内容进行关键字匹配回复，相应关键字可以触发相应回复，批量、迅速地解决用户群中存在的常见问题、新品上市、线下店铺位置信息、合作等。</p>
<p>自定义菜单：</p>
<p>企业号还可以设置自定义菜单，自定义菜单最多支持两级，菜单支持的响应动作包括跳转链接和触发自动回复，方便快捷维持用户黏性。</p>
<ul>
<li>私信回复，可以添加微信</li>
<li>400电话，拨打电话按键，显示有意向</li>
<li>对用户进行细化分层</li>
<li>官网连接，通过表单收集到的客户信息</li>
<li>后台对意向客户的信息进行管理，让销售进一步跟进转化</li>
<li>评论回复，找到潜在用户</li>
</ul>
<p>抖音个人账号功能没有蓝V全面（毕竟免费），在用户管理上关注私信和评论即可。</p>
<p><strong>6）数据管理</strong></p>
<p>首先，做运营，不管做什么平台，数据分析都是非常重要的工作。通过数据分析可以判断内容的质量，还可以指导我们的运营策略。</p>
<p>上面提到抖音判断视频作品的4 个标准： 完播率、点赞量、评论量、转发量。至少这4个指标是我们时刻需要关注的。</p>
<p>数据分析可以通过抖音号自带的后台统计，也可以通过第三方平台统计。每个运营人对数据分析都有着自己的方法，能找出核心问题就OK。</p>
<p>下面提供一下我平时做的数据分析表作为参考，主要分为：基础数据、互动数据、互动率数据、DOU推广数据。</p>
<ul>
<li>抖音号自带后台（基本满足）</li>
</ul>
<ul>
<li>抖音第三方数据分析平台：飞瓜数据、抖大大、卡思数据、新抖等。功能强大，可以监测多平台、多账号、多维度等。（数据不一定精准、仅供参考）</li>
</ul>

<p><strong>7）变现方式</strong></p>
<p>变现是做账号的最终目的，但是如果过早的去追求变现，是难的。前期要花时间精力是做好内容，抖音平台内容是本质，建议粉丝量在5万以上，再变现（产品植入、直播、导流）。</p>
<p>变现的前提条件：有一定的粉丝量，粉丝粘性高。</p>
<ol>
<li>电商变现，通过视频+橱窗的方式结合，需要有电商店铺（淘宝、天猫、京东）、抖音小店。</li>
<li>直播变现，带货+打赏。</li>
<li>私域流量，潜在用户导流到微信，圈养粉丝，进一步成交。</li>
<li>接广告变现。</li>
</ol>
<h2 id="toc-5">五、抖音短视频运营规划</h2>
<p>这一部分内容，具体的规划需要结合公司的实际情况来制定，一般分为以下4个阶段进行。</p>
<p><strong>第一阶段，品牌知晓期。</strong></p>
<p>通过人设，输出大量与品牌调性一致的内容对话受众。万变不离其宗，抖音运营第一步，让大家知道自己的品牌。</p>
<p><strong>第二阶段，受众感知期。</strong></p>
<p>通过爆款的内容视频、话题、活动让受众更直观了解自己的品牌/服务理念，并有意愿进一步了解产品和体验产品。</p>
<p><strong>第三阶段，销售扩张期。</strong></p>
<p>通过大约（3-6个月）的发酵，我们已经利用有效内容沟通到了一批目标受众，我们将集中通过KOL效应聚拢人群，通过商品橱窗、视频加入产品连接、直播的方式等带动产品销售。</p>
<p><strong>第四阶段，品牌升级期。</strong></p>
<p>通过深度内容及多元形式，向受众展示自己更深度的品牌理念，打造铁杆粉。</p>
<h2 id="toc-6">六、团队配置、人员分工</h2>
<p>标配版短视频团队架构：</p>
<ol>
<li>编导：定视频的方向、写脚本、统筹整个的拍摄计划。</li>
<li>摄影：设计分镜头，布光以及把控拍摄整个过程。</li>
<li>剪辑：视频拍完之后，根据剧本要求，进行剪辑、包装。一般抖音视频时长2-3分钟，或者1分钟左右。</li>
<li>运营：收到剪辑完之后视频，设计视频头图、标题、简介、推荐位等，分发到各个平台，统计分析视频的数据，对各个平台的视频进行管理、用户互动等。</li>
</ol>
<p>如果有预算的情况下，一定要配置专业的演员，预算不够的情况下，在企业员工里挑选：颜值高，外形清新亮丽，镜头感强，有强烈的表现力。</p>
