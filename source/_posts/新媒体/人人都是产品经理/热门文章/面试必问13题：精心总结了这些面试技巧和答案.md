<p>你好，我是运营面试官Aaron，有8年互联网运营经验，担任过互联网公司首席运营官COO，现担任某公司联合创始人。我是站在面试官角度，只告诉你拿来即用的面试经验，助你斩获offer，轻松找工作！</p>
<p>不管求职哪个岗位都必问的13道面试问题（含面试回答）：</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/XphcaIstfaUM770BTrav.png" alt width="699" height="470" referrerpolicy="no-referrer"></p>
<p><img data-action="zoom" class="aligncenter wp-image-4381347" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/wSYQnNpN1EbTUJFkI2W4.jpg" alt width="999" height="902" referrerpolicy="no-referrer"></p>
<h2 id="toc-1">一、请你进行自我介绍3分钟</h2>
<h3>1. 面试官角度</h3>
<p>考查你的逻辑条理性和口头表达能力。</p>
<h3>2. 面试技巧</h3>
<ul>
<li>自我介绍是你见到面试官后，获得的第一次向面试官展示自我亮点的最重要机会。好的自我介绍是可以让面试官对你产生兴趣，甚至让面试官当即确定你就是他们要找的候选人。所以，请勿抵触自我介绍；</li>
<li>时长3分钟左右；</li>
<li>表达需要口语化，不要过于官方，更不能直接把简历背诵出来；</li>
<li>工作成果一定要用“数据”来表达，数据！数据！数据！（重要事情说三遍）；</li>
<li>自我介绍并不是固定不变，每次去面试前都需要根据“岗位招聘要求JD”来适当修改一下；</li>
<li>自我介绍是简历上【自我评价】+【工作经历】的精炼版。</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/0OOHQZJRwHU47A2Qo6z9.png" alt width="903" height="346" referrerpolicy="no-referrer"></p>
<h2 id="toc-2">二、你的离职原因是什么？</h2>
<h3>1. 面试官角度</h3>
<p>通过离职原因来考察你的个人状态。没有面试官喜欢负能量的人，所以谈到离职原因，你千万不要一味地抱怨。</p>
<h3>2. 面试注意事项</h3>
<ul>
<li>切记不要说领导、同事、公司的坏话！</li>
<li>如果没有特殊原因，建议不要频繁跳槽（除非你真的是超级牛逼的人），没有哪个面试官喜欢稳定性低的人。</li>
</ul>
<h3>3. 面试技巧</h3>
<p>一定要说成“客观原因”，即使是个人主观原因，也要往“客观原因”来回答。</p>
<h3>4. 不推荐的回答</h3>
<ul>
<li>工资太低</li>
<li>经常加班</li>
<li>工作压力大</li>
<li>与同事或上司相处不合</li>
<li>公司内部勾心斗角</li>
</ul>
<p>提醒：以上原因，即使你到了新公司，也许还会存在这些情况。如果真的是为这些原因而离职，建议还是慎重考虑清楚再决定是否离职。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/vzDxbahcyOFkZJkFa0E9.png" alt width="901" height="348" referrerpolicy="no-referrer"></p>
<h2 id="toc-3">三、你上家公司的薪资是多少？</h2>
<h3>1. 面试官角度</h3>
<p>根据你在上家公司的薪资来作为你offer薪资的参考。</p>
<h3>2. 面试技巧</h3>
<p>这个只能如实回答，不能作假，因为有些公司需要你提供“薪资证明”或“工资流水”。也不需要把薪资拆解出来，有些面试官会根据你上家公司的底薪来定offer薪资。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/TtGFHgSIj9CrngCwuOqf.png" alt width="903" height="169" referrerpolicy="no-referrer"></p>
<h2 id="toc-4">四、你现在找工作主要关注哪些方面？</h2>
<h3>1. 面试官角度</h3>
<p>了解公司是否符合你的找工作标准。</p>
<h3>2. 面试技巧</h3>
<p>实话实话，大胆放心地说出自己的找工作标准，不忘初心（不然等入职后才发现掉坑了，后悔莫及）。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/siQTBbLRnwV660YoaLS7.png" alt width="904" height="161" referrerpolicy="no-referrer"></p>
<h2 id="toc-5">五、对于这个岗位，你认为你有哪些优势？</h2>
<h3>1. 面试官角度</h3>
<ul>
<li>你的自我认知能力；</li>
<li>给你个机会来说服面试官为什么要录用你而不是录用别人。</li>
</ul>
<h3>2. 面试技巧</h3>
<ul>
<li>给出明确、正面的回答，不宜太过谦虚，也不宜含含糊糊；</li>
<li>一定结合JD来回答，同时列举一个简单事例来佐证自己的卖点，目的是告诉面试官“我就是你要招的人”（eg：JD要求是“注重细节”，则要往“注重细节”方面作答）；</li>
<li>一般回答“3个优势”即可。</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/7Y1Em6AMqcVCZxT1nbWY.png" alt width="901" height="215" referrerpolicy="no-referrer"></p>
<h2 id="toc-6">六、说下你的缺点</h2>
<h3>1. 面试官角度</h3>
<p>你的缺点是否会影响到本职工作。</p>
<h3>2. 面试技巧</h3>
<ul>
<li>不能回答对本职工作影响较大的缺点，即不要回答JD上要求必备的特征；同时举例子来阐述说明这个缺点影响了自己的工作，最后谈谈如何改正这个缺点（eg：JD要求是“注重细节”，你却说“经常粗心”）；</li>
<li>避免避重就轻，回答非职业缺点或者算不上缺点的缺点，要回答真实的缺点。（非职业缺点：比如挑食、不擅长陪女友逛街、做饭经常做糊；不能算缺点的缺点：比如熬夜会困、待人接物太客气）；</li>
<li>一般回答“1个不足点”即可，缺点说太多会降低面试官的好感度，人总是会记住别人不好的一面，这就是人性。</li>
</ul>
<h3>3. 不能回答的致命缺点</h3>
<ul>
<li>有拖延症（让面试官认为你执行力很差）；</li>
<li>喜欢独处，独来独往（让面试官认为你无法融入团队）；</li>
<li>粗心（让面试官认为你粗心大意，做事总是出错）。</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/w38j0G3WVEUVnXfkJ9fk.png" alt width="903" height="274" referrerpolicy="no-referrer"></p>
<h2 id="toc-7">七、你平时是怎么提升自我？</h2>
<h3>1. 面试官角度</h3>
<ul>
<li>你是否通过不断地学习来提升工作能力</li>
<li>你的知识获取方式</li>
</ul>
<h3>2. 面试技巧</h3>
<p>提升自我可从【输入性学习+输出性分享】这两方面作答：</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/U2syHtb4Lbd6WkG0kB8I.png" alt width="899" height="176" referrerpolicy="no-referrer"></p>
<h2 id="toc-8">八、你平时有什么兴趣爱好？</h2>
<h3>1. 面试官角度</h3>
<ul>
<li>想了解工作以外的你是怎么样的？</li>
<li>通过了解你的兴趣爱好来判断你对工作的钻研程度；</li>
<li>如果一个应聘者有投入时间并长期坚持的爱好，那很容易得到面试官的青睐。因为他知道自己追求的目标是什么而且愿意为目标付出行动和坚持。这种劲头用在工作上，也很容易脱颖而出。相反，如果仅仅只是兴趣而缺乏行动和钻研精神，对自己长期的爱好都可以这么马虎，工作上不可能相信他会太负责或者有专业精神。</li>
</ul>
<h3>2. 面试技巧</h3>
<ul>
<li>爱好是指你自己花了很多时间练习，强于一般人的这种可以叫特长的爱好，而并非是常见的“看电影、刷抖音”等兴趣爱好（除非你面试的是游戏公司或抖音运营岗）；</li>
<li>一般回答1个即可，让面试官觉得你很专注，重点是强调突出你兴趣爱好的成果。</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/UN7nTxCnNEfKxplu9iez.png" alt width="907" height="240" referrerpolicy="no-referrer"></p>
<h2 id="toc-9">九、你怎么看待加班？</h2>
<h3>1. 面试官角度</h3>
<ul>
<li>面试官是希望听到“认同加班”的回答；</li>
<li>我本人是不提倡加班的，但我依然在面试的时候会问求职者，了解ta对加班的看法，是否一刀切地排斥加班。</li>
</ul>
<h3>2. 面试技巧</h3>
<p>你不必为了迎合公司而宣称自己“365天24小时做牛做马在所不惜”。</p>
<h3>3. 面试注意事项</h3>
<ul>
<li>不要回答“只要钱给够了，我可以一天到晚不休息而工作”等玩笑话；</li>
<li>不要去问面试官是否有加班费，会让面试官认为你眼界太小太在乎钱了，建议在后面进行薪资谈判再去问HR。</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/MsSHivjy2R8oV8YPirAB.png" alt width="902" height="256" referrerpolicy="no-referrer"></p>
<h2 id="toc-10">十、你可以说一下未来的职业规划吗？</h2>
<h3>1. 面试官角度</h3>
<ul>
<li>你的自我规划是否清晰；</li>
<li>你是否有上进心、目标感和自我驱动力，从而判断你是否有潜力；</li>
<li>你的个人发展是否与公司发展相符，从而判断你未来的稳定性。</li>
</ul>
<h3>2. 面试技巧</h3>
<ul>
<li>实话实话，一般说最近3年的职业规划即可；</li>
<li>职业规划一般遵循以下两条基本原则；</li>
<li>①选择有前途的行业（关键），好行业才能给你充足的发展空间；</li>
<li>②保持专注，专注才能积累，积累才能取得成就；</li>
<li>回答重点不是你的职业目标，而是如何实现你的职业目标；</li>
<li>提前了解应聘公司的背景、现状与未来，回答时适当与公司发展相贴合，让面试官觉得，你的职业规划与公司发展相吻合，来证明你的稳定性。</li>
</ul>
<h3>3. 面试注意事项</h3>
<p>你千万不能说“学习本领后自己去创业”，如果面试官继续追问你是否会创业，你也不要把话说得太死、太直接。你可以回答：对创业可以持观望的态度，看大环境和机会，并说明自己短时间内没有创业的打算。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/run4wpZ6vN1hTYCvz6oS.png" alt width="905" height="284" referrerpolicy="no-referrer"></p>
<h2 id="toc-11">十一、你期望薪资是多少？</h2>
<h3>1. 面试官角度</h3>
<ul>
<li>你对于薪资的态度是怎样的？</li>
<li>你是如何正确地评估自己的价值？</li>
<li>你怎样看待自身能力水平和薪资的差距？</li>
<li>你的期望薪资是否和公司的标准一致？</li>
</ul>
<p>注意：面试官并不是在这时候跟你谈判薪资。</p>
<h3>2. 面试技巧</h3>
<p>这道题很关键，是决定你最终offer薪资的重要因素之一。如果超过JD薪资范围，则基本上没戏；如果低于JD薪资范围，则被面试官认为专业能力不足。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/5rap5ekjjG5VTTh8efUO.png" alt width="900" height="288" referrerpolicy="no-referrer"></p>
<h2 id="toc-12">十二、你大概什么时候可以到岗？</h2>
<h3>面试官角度</h3>
<ul>
<li>了解你的最快上班时间</li>
<li>判断你是否有其他面试安排</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/02/xbZWqjFYoT2hKwrnnaYX.png" alt width="898" height="154" referrerpolicy="no-referrer"></p>
<h2 id="toc-13">十三、你有什么问题要问我吗？</h2>
<h3>1. 面试官角度</h3>
<p>通过你提问的问题来侧面反映你的格局。</p>
<h3>2. 面试技巧</h3>
<ul>
<li>问一些与工作内容或岗位相关的问题，既让面试官认为你很想加入公司，又可以为你反选offer提供更多信息作为参考；</li>
<li>一般最多问2个即可。</li>
</ul>
<h3>3. 面试注意事项</h3>
<ul>
<li>不能说没有问题想问，会让面试官认为你不是很想加入公司；</li>
<li>如果是用人部门面试，切记不要问薪资、福利、上班时间等人事行政问题，一来让面试官认为你格局小，唯利是图，只在乎钱；二来薪资福利等问题是HR范畴。</li>
</ul>
