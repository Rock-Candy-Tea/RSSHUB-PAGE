<blockquote><p>抖音作为流量的聚集地，其广告位的规则如何？笔者依据抖音品牌的广告矩阵，对其进行了介绍和分析。</p></blockquote>
<p><img data-action="zoom" class="aligncenter size-full wp-image-2901393" src="https://image.woshipm.com/wp-files/2019/09/MBnQr2R7vXtNocedilZT.jpg" alt width="800" height="450" referrerpolicy="no-referrer"></p>
<p>抖音品牌广告产品矩阵：抖音开屏广告、信息流广告、贴纸广告、音乐库广告、达人广告。</p>
<h3>1. 抖音开屏+信息流广告</h3>
<p><img data-action="zoom" class="aligncenter" src="https://image.woshipm.com/wp-files/2019/09/hqutMZlubmSUL9y2urJ0.png" alt width="654" height="332" referrerpolicy="no-referrer"></p>
<p>产品/品牌宣传标配：开屏宣传无干扰，信息流内容强覆盖。</p>
<h3>2. 抖音信息流+达人广告</h3>
<ol>
<li>方式一：信息流投放客户广告素材，外链跳转H5落地页，达人视频辅助宣传。</li>
<li>方式二：使用抖音信息流为达人合作视频引流，以原生内容形式软广植入广告，减少用户对广告的抵触加强内容传播深度。</li>
</ol>
<p><img data-action="zoom" class="aligncenter" src="https://image.woshipm.com/wp-files/2019/09/oaQ539o2UuTTHiQ6p22x.png" alt width="657" height="300" referrerpolicy="no-referrer"></p>
<p><img data-action="zoom" class="aligncenter" src="https://image.woshipm.com/wp-files/2019/09/NwyLzQc6pUt5if9PLmQG.png" alt width="657" height="301" referrerpolicy="no-referrer"></p>
<h3>3. 抖音开屏+信息流+达人广告</h3>
<ol>
<li>方式一：开屏＆信息流投放客户广告素材，外链跳转H5落地页，达人视频辅助宣传。</li>
<li>方式二：开屏落地页跳转视频详情页，并额外投放信息流广告，加大传播影响力。</li>
</ol>
<p><img data-action="zoom" class="aligncenter" src="https://image.woshipm.com/wp-files/2019/09/CIhe4Zs7lFwwD390sTMO.png" alt width="647" height="633" referrerpolicy="no-referrer"></p>
<h2 id="toc-1">一、抖音开屏广告-产品介绍</h2>
<h3>1. 展现规则</h3>
<ul>
<li>开机第一入口，视觉冲击强，无干扰，强势锁定新生代消费主力</li>
<li>每日每个用户可看到的开屏广告的次数不限。注意：新安装用户，7日内不出开屏广告。</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.woshipm.com/wp-files/2019/09/cnJHS6JzqoV6mbCZ1aF0.png" alt width="224" height="433" referrerpolicy="no-referrer"></p>
<h3>2. 产品说明</h3>
<ul>
<li>定向功能：CPT不支持定向，CPM支持年龄、性别、地域、平台定向。</li>
<li>展现规则：静态3秒、动态4秒、视频5秒，支持纯展示及落地页跳转（支持设置直达链接，跳转抖音APP内部页面）。</li>
<li>禁投行业：在广审行业限制基础上增加以下限制：成人用品类；视频类产品；社区社交类产品；拍摄工具类产品，争议行业投放前建议咨询广审团队。</li>
</ul>
<h3>3. 投放效果</h3>
<ul>
<li>静态开屏点击率均值：5%-5.5%</li>
<li>动态开屏点击率均值：6%-8%</li>
<li>视频开屏点击率均值：10%-15%</li>
</ul>
<p>广告效果受素材内容影响，数据仅供参考。</p>
<h3>4. 售卖方式</h3>
<p>Q2抖音开屏刊例：CPM售卖支持年龄、性别、地域、平台定向，按刊例溢价规则加收。</p>
<ul>
<li>年龄、性别不加收；</li>
<li>地域：核心城市+30%，重点城市+10%；</li>
<li>平台：IOS端+30%。</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.woshipm.com/wp-files/2019/09/dlRmyQVJxd7r0FZbQqwV.png" alt width="658" height="178" referrerpolicy="no-referrer"></p>
<p style="text-align: center;">《抖音》2018年Q2移动端常规硬广资源全国版刊例价</p>
<h2 id="toc-2">二、抖音原生信息流-产品介绍</h2>
<p>品牌可推广5-60s短视频，在抖音推荐流中以原生广告样式进行传播展示，无缝融入抖音推荐流中，帮助广告主在抖音实现营销推广的目的。</p>
<h3>1. 产品卖点</h3>
<ul>
<li>霸屏展现，无干扰：以全屏形式展现在抖音推荐流内，广告内容传递无干扰，带来移动端竖屏全新视觉体验。</li>
<li>账号关联，强聚粉：原生信息流与品牌抖音账号打通，支持品牌做粉丝持续累积，突破品牌营销发展瓶颈。</li>
<li>支持分享，促传播：用户可将广告内容分享与抖音站内好友或其他社交平台，已与当前热门社交软件打通，助力品牌全平台传播。</li>
</ul>
<h3>2. 效果参考</h3>
<ul>
<li>点击率均值：0.15%-0.4%（点击率=点击量/展现量）</li>
<li>互动率均值：3%+（互动率=有过广告互动行为的 UV/广告曝光 UV）</li>
<li>互动行为包括：点赞、评论、转发、关注、进入主页、查看相关音乐、点击话题。</li>
</ul>
<h2 id="toc-3">三、抖音单页信息流—产品介绍</h2>
<p><img data-action="zoom" class="aligncenter" src="https://image.woshipm.com/wp-files/2019/09/YTNZV6Jb5O5zqnMDC36c.png" alt width="676" height="341" referrerpolicy="no-referrer"></p>
<h3>1. 产品逻辑</h3>
<ul>
<li>推荐流4-1：抖音推荐流第一刷第四个视频。</li>
<li>不限位置保量GD：用户浏览抖音推荐流时，每隔5分钟可刷到一次广告，品牌广告会出现在用户推荐流前六十个视频里。</li>
<li>广告频控：每条广告每天对每个用户只展示1次。</li>
<li>新用户保护策略：新安装用户，7日内不出广告。</li>
<li>支持广告评论关闭：若勾选此项，用户无法在此条广告视频上发表评论，投放过程中支持修改。</li>
<li>数据披露：展现量、点击量、视频播放量、互动数据（点赞、评论、分享、新增关注数、主页访问量）</li>
<li>第三方监测：支持第三方曝光，点击，视频播放量监测。合作第三方：AdMaster、秒针、国双、尼尔森</li>
<li>支持应用下载：点击跳转按钮安卓静默下载、IOS跳转APP Store。</li>
</ul>
<h3>2. 售卖方式</h3>
<ul>
<li>购买方式：GD售卖，不限位置保量GD刊例价240元/CPM，抖音推荐流4-1刊例价320元/cpm，仅支持提前预定未来28天 的库存，不支持超期限预定。</li>
<li>定向方式：支持年龄、性别、地域、平台定向，按刊例溢价规则加收，年龄、性别不加收；核心城市+30%，重点城市+10%；IOS端+30%。</li>
<li>素材要求：同一物料可连续投放，最多投放天数不得超过5天（注：相同素材若连续投放，客户抖音账号下会出现多个重复视频，如1条视频连续投放5天，该投放抖音账号下会出现5条相同视频；同时，这5个视频素材相互独立，评论、点赞、转发等数据会各自积累不会叠加）。</li>
<li>禁投行业：在广审行业限制基础上增加以下限制：成人用品类；视频类产品；社区社交类产品；拍摄工具类产品，抖音新增二审流程，争议行业投放前建议咨询广审团队。</li>
</ul>
<h2 id="toc-4">四、抖音贴纸广告—产品介绍</h2>
<p>品牌定制的抖音贴纸，用户在拍摄时，可在贴纸栏下载使用。</p>
<p>贴纸类型：2D脸部挂件贴纸、2D前景贴纸。</p>
<h3>1. 产品卖点</h3>
<ul>
<li>使用场景原生，用户主动使用，接受度高，互动时间长；</li>
<li>激发用户主动传播，分享传播后，触发二次使用；</li>
<li>广告形式生动，提升品牌在抖音用户中好感度；</li>
<li>直接展示产品效果，激发用户潜在购买行为。</li>
</ul>
<h3>2. 售卖规则</h3>
<ul>
<li>售价：刊例160万/7天</li>
<li>位置：3天抖音贴纸栏第一行第四位，4天位置随机，连续购买，需要设计多款贴纸，以保证每周更换1次。</li>
<li>该产品属于非标产品，不保量，不支持第三方监测；</li>
<li>抖音对贴纸是否具备上线标准具有最终决定权。</li>
</ul>
<h2 id="toc-5">五、抖音商业音乐入库广告—产品介绍</h2>
<p>将客户自有版权商业音乐录入抖音音乐库，入库后，用户可在音乐库中搜索该音乐，并用于抖音平台视频上传/拍摄。</p>
<h3>1. 入库权益</h3>
<ul>
<li>歌曲名称＆歌曲内容可含品牌词；</li>
<li>音乐库歌曲名搜索置顶：入库后精准搜索歌名，入库音乐搜索结果置顶；搜索结果置顶有效期3个月，3个月后乐库可搜，位置随机。</li>
<li>挑战赛相关音乐推荐绑定：音乐入库后可同客户自发挑战赛主页做关联绑定；关联后，点击该挑战赛主页下方参与按钮跳转音乐页后，入库音乐推荐置顶，长期有效，一首入库歌曲限绑定一个挑战赛主页。</li>
</ul>
<h3>2. 售卖方式</h3>
<p>售卖价格：</p>
<p><img data-action="zoom" class="aligncenter" src="https://image.woshipm.com/wp-files/2019/09/DRtW3C2HcWCBNGtPK0ek.png" alt width="636" height="121" referrerpolicy="no-referrer"></p>
<h3>3. 售卖规则</h3>
<ul>
<li>抖音对音乐是否具备入库标准具有最终决定权，音乐审核通过后才可下单；</li>
<li>商业音乐官方入库后，音乐库精准搜索入库歌名，搜索结果置顶；</li>
<li>同一入库音乐，仅限绑定一个挑战赛主页推荐位；</li>
<li>该产品属于非标产品，不保量，不支持第三方监测；</li>
<li>非标产品下单前需先核价并过审素材。</li>
</ul>
