<div>   
<blockquote><p>产品如何去提升转换率，试试用LIFT模型去分析一下~</p></blockquote>
<p><img data-action="zoom" class="aligncenter size-full wp-image-2393366" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2019/05/gIV0GBtyA0Gv90S6SQzm.jpg" alt width="800" height="450" referrerpolicy="no-referrer"></p>
<p><strong>LIFT模型原理：</strong></p>
<p>基于数据认识的基础，在营销信息中找出问题，设计有关解决这些问题的假设，提高产品效益，得出改进问题的结果，提升转换率。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2019/05/bJnhNGe26qLprZBm3XyP.png" alt width="750" height="300" referrerpolicy="no-referrer"></p>
<p><strong>影响 LIFT模型六个因素：</strong></p>
<ul>
<li><strong>载体：</strong>价值主张</li>
<li><strong>推动因素：</strong>相关性 、清晰度、紧迫性</li>
<li><strong>阻碍因素：</strong>焦虑性、注意力分散</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2019/05/7vOkX7lZWh9DidGoXBwD.png" alt width="322" height="370" referrerpolicy="no-referrer"></p>
<h2 id="toc-1">方法1：强化价值主张</h2>
<h3>什么是价值主张？</h3>
<ul>
<li>是产品转换率潜力的载体，是客户购买的理由；</li>
<li>是产品和服务最重要的属性，是从客户的角度来看产品的匹配点和差异点。</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2019/05/ioWZ8TyZdhfYlZLfrFrD.png" alt width="822" height="341" referrerpolicy="no-referrer"></p>
<p>感知的好处大于采取行动的成本，你的潜在客户是有动力的！</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2019/05/KvdC5q3MSKtDVQEydq47.png" alt width="511" height="337" referrerpolicy="no-referrer"></p>
<h3>2. 如何优化价值主张</h3>
<p><strong>（1）优化”认知“——理解是购买的前提</strong></p>
<p>当你在传达你产品性能特点和好处时，不要忘记过滤率这些影响认知的因素：</p>
<ul>
<li>颜色理解；</li>
<li>文字引申含义；</li>
<li>文化差异；</li>
<li>期望页面的内容；</li>
<li>看到与自己无关的信息时，内心的感受和情绪；</li>
<li>当时所处的环境；</li>
<li>以前公司的过往；</li>
<li>类似产品的体验和服务；</li>
<li>质量、金钱和价值等方面的理念。</li>
</ul>
<p><strong>（2）优化”有形的性能 和 无形的情感“——为购买找个理由</strong></p>
<p>能够满足客户表达的需求，证明他们感性的决定到购买的决定是正确的。所以你需要奖励来增强你有形的性能，以及一些手段增强潜在客户对你无形好处的感觉。</p>
<p>有形的性能及好处（证明感性的决定是正确的）：</p>
<ul>
<li>产品或服务的性能；</li>
<li>奖励和提供物。</li>
</ul>
<p>无形的情感及好处（证明购买的决定是正确的）：</p>
<p>信誉度：</p>
<ul>
<li>专业评审、媒体评论、奖项、思想领导力、案例研究</li>
<li>专业素质（广告文字来自前后一致和无差别的错）</li>
</ul>
<p>社会认同感：</p>
<ul>
<li>推荐证言、客户公司商标、社交媒体、客户评分和评论；</li>
<li>客户人数（社会价值证明）、名人代言。</li>
</ul>
<p><strong>（3）优化”成本“——客户价位心理</strong></p>
<p>潜在的客户最容易算出你产品或服务的直接成本，除了公司的标准定价策略和方法外，还要考虑强大的价位心理学，如：</p>
<ul>
<li>数字声音心理特征（在英语在1和2印象不够深刻，所以从尾数字1或2改成3、6、7、8）；</li>
<li>锚定效应（先展示价格次序，给予用户心理准备。任意较大数字放在最前）等。除了这些，还要考虑“关联成本”和“速递成本”。</li>
</ul>
<h3>3. 小结</h3>
<ul>
<li>影响价值主张的四个因素：认知、有形的性能、无形的情感、用户心理价位。</li>
<li>优化的目的：降低用户心理认知障碍，提升用户的行动力！</li>
</ul>
<h2 id="toc-2">方法2：放大相关性</h2>
<h3>1. 什么是相关性</h3>
<p>你所创造的有利的刺激信息留下“气味路径”决定了你能否抓住和保持潜在用户从兴趣到购买的注意力。</p>
<h3>2. 如何优化相关性</h3>
<p>为用户标识他们所有需要的信息和产品，并且内容专注与价值主张最相关的声明上。加强页面和潜在客户的期望及需求之间的匹配度和一致度。</p>
<p><strong>（2）优化营销漏斗相关性</strong></p>
<p>保持页面和潜在客户认为他们看到落地页的相关性，你潜在客户漏斗所接触的点有：</p>
<ul>
<li>对原先的品牌的解决方案的认知度；</li>
<li>广告、邮件、社交媒体或其他推荐来源做的推荐行为；</li>
<li>你的落地页面；</li>
<li>对公司、产品、服务的探索；</li>
<li>邮件、电话、在线表格、实际交易、渠道伙伴或者其他回复渠道完成的转化的交易；</li>
<li>销售人员（产生销售线索的人）；</li>
<li>售后体验。</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2019/05/moSbNHgcqVb69yIG2366.png" alt width="797" height="414" referrerpolicy="no-referrer"></p>
<p><strong>（2）优化来源媒体相关性</strong></p>
<p>当你的广告或者网站的访客从某个地方来的时候，要考虑访客来源的媒介。转化下一步依然保持相关路径，强调独特的优势，而不是过分使用通用的益处（如：保证满意，省钱省力）。所以需要考虑媒体来源和创建落地页面来源的相关性。</p>
<p><strong>（3）优化目标受众相关性</strong></p>
<p><strong>先做的事情——细分目标受众群里</strong></p>
<ul>
<li>数据库细分：了解平均客单价，测试他们承受范围内的组合优惠；</li>
<li>广告来源细分：区分访客到网上来是付费搜索关键词还是广告，从而判断他们兴趣点和处在购买周期阶段，给予适合优惠；</li>
<li>媒体来源细分：社交媒体访客更加娱乐导向或者早期的用户召唤行为，搜索落地页更加指导性的，邮件落地页是非常专注的；</li>
<li>落地页面细分：提供适合经过市场细分的内容和优惠链接。</li>
</ul>
<p>受众细分优势：你了解的越多，信息相关性就越好。。</p>
<p>受众细分劣势：市场细分增加维护成本，降低相关性，增加决策的复杂度。</p>
<p><strong>需要确定的事情——是否要市场细分</strong></p>
<p>市场细分领域是否要有独特的需求以及独特的市场消息，因为市场细分增加了你的消息细分的时间和复杂度。如果确定并值得市场细分，那么你需要考虑下面事情。</p>
<p><strong>考虑的事情——维护成本、标记风险和决策复杂度</strong></p>
<ul>
<li>维护成本：如果你有三个产品，需要维护三个不同页面，每个都市场细分为三个客户群体，那么需要维护九个独特的页面。你可以测试下每个市场细分是否能够提升转换率，增加维护成本是否有价值。</li>
<li>标记风险：给受众群体不小心标错了标签，会降低相关性，所以需要测试那些标签是否适合用在受众身上的。尽可能做受众来源标记目标定向。</li>
<li>决策复制度：每增加一个市场细分的决定，就有可能使用户困惑，也可能会考虑自己是否被迫选择一条与其他人不同的路径，从而打开另一个路径来看是否有更好的优惠条件。</li>
</ul>
<p><strong>在做的事情——用户召唤行为划分</strong></p>
<ul>
<li>立即购买型客户:落地页和标题应该与广告相匹配；</li>
<li>需要更多信息决策型客户：提供有帮助的信息，把产品定位成解决客户需求一个的方案；</li>
<li>寻找报价，对比价格型客户。</li>
</ul>
<p><strong>最后在做的事情——腔调相关性</strong></p>
<p><strong>（4）优化导航相关性</strong></p>
<p>由于每个用户习惯查找产品的方式不同，用户常用查找产品方式有：类目浏览查找、搜索查找、在文字链接查找产品名字或者描述、图片和标签查找、特价和优惠查找等方式。所以在细分目标受众之后，分析网站数据，测试用的假设，在以多种不同的组合页面拿来做测试（当然也可混合版本，包含特价和优惠，在加上按领域浏览的链接），进行测试，得出结果。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2019/05/quHasTJpuqRsnOaetE9f.png" alt width="902" height="382" referrerpolicy="no-referrer"></p>
<p><strong>（5）优化竞争相关性</strong></p>
<p>竞争对手的价值主张和市场消息会直接影响你的转换率，所以将自己创意与竞争对手相比较。</p>
<h3><strong>3. 小结</strong></h3>
<ul>
<li>影响相关性的五个因素：营销漏斗相关性、来源媒体相关性、目标受众相关性、导航相关性、竞争相关性。</li>
<li>优化的目的：保持潜在用户从兴趣到购买的注意力。</li>
</ul>
<h2 id="toc-3">方法3：提升清晰度</h2>
<h3>1. 什么是清晰度？</h3>
<p>高清晰的沟通是可以理解的、连贯的和准确的，可以清楚并快速传达你价值主张和用户召唤行为，让受众花在理解上时间最少化。</p>
<h3>2. 如何优化清晰度</h3>
<p>加强页面组织和内容组织关联性和视觉流向，提升内容的吸引力。</p>
<p><strong>（1）优化信息层级</strong></p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2019/05/8MDYMwk3FI8cwyzEtSIP.png" alt width="702" height="311" referrerpolicy="no-referrer"></p>
<p><strong>（2）优化视觉流向</strong></p>
<p><strong>视觉流向</strong></p>
<ul>
<li>位置：图片、图案、文字排放位置，因为它改变了视觉的流向和内容的重点。</li>
<li>元素：视觉元素应用同样会引导和抑制视觉流向（如：形状、线条和图案）。</li>
</ul>
<p><strong>图片：</strong></p>
<p>应该传递和支持你的价值主张，勿要用装饰图片。统一尺寸和对齐避免分散访客注意力。</p>
<p><strong>色彩：</strong></p>
<ul>
<li>对比：强调有选择重要的元素，而弱化次要内容。</li>
<li>含义：不同背景文化有不同色彩的含义。</li>
<li>一致：如：网站的链接蓝色。</li>
</ul>
<p><strong>（3）优化文案</strong></p>
<p>给客户提供一个可以“解决问题方法的产品”，文案的基调和价值主张是基于客户的需求。</p>
<p><strong>（4）优化CTA</strong></p>
<p>CTA按键可见度要高，它比设计和色彩更加重要。需要考虑，按键的大小、颜色、位置。</p>
<h3><strong>3. 小结</strong></h3>
<p>影响清晰度的四个因素：信息层级清晰化、设计清晰化、文案清晰化、用户召唤行为清晰化</p>
<p>优化的目的：视觉流向、文案龚写和CTA是保证页面清晰度的重要方向。</p>
<h2 id="toc-4">方法4：降低焦虑感</h2>
<h3>1. 什么是焦虑感</h3>
<p>感知环境中的危险，然后判断是否能够安全的前进。</p>
<h3>2. 如何优化焦虑感</h3>
<p>将焦虑感来源最小化，把焦虑感转化成优势来刺激用户行为。</p>
<p><strong>（1）优化隐私焦虑感</strong></p>
<p>在优化隐私焦虑感需要注意以下原则：</p>
<ul>
<li>帮助文字提醒（仅展示该列的字段，而不是把所有的都显示出来）</li>
<li>多选择少填写</li>
<li>给填写一个理由</li>
<li>默认选择</li>
<li>任务中勿要有其他不相关链接</li>
<li>保持行动按键在一屏内</li>
<li>少一些指令式</li>
<li>填写任务步骤可多页面展示</li>
<li>安全提醒</li>
</ul>
<p><strong>（2）优化可用性焦虑感——让用户学习你的网站有多容易</strong></p>
<p><strong>（3）优化完成性焦虑感——需要给潜在的客户信息作为满足他们的需求担保</strong></p>
<p><strong>（4）优化努力度焦虑感——将复杂的需求看起来更容易完成</strong></p>
<p><strong>（5）把焦虑转化成你优势</strong></p>
<p>人们在避免损失的驱动力要比追逐利益点驱动力强大很多。</p>
<p>让人觉着这件东西对他来说变成更有价值，也就是“禀赋效应”</p>
<h3>3. 总结</h3>
<p>降低焦虑感的四个方向：隐私焦虑感、可用性焦虑感、完成性焦虑感、努力度焦虑感。</p>
<p>优化的目的：降低用户焦虑感，刺激用户行为。</p>
<h2 id="toc-5">方法5：降低注意力分散</h2>
<h3>1. 什么是注意力分散</h3>
<p>用户的注意力 从主要的价值主张信息和CTA引开的元素。</p>
<h3>2. 如何优化注意力分散</h3>
<p>提升用户第一印象阶段和消息阶段：</p>
<ul>
<li>第一印象阶段：访客快速判断页面是否吸引人，文案是否适量，界面是否友好。</li>
<li>消息阶段：用户寻找来理解页面组织方式，以及最重要的信息放置方式。</li>
</ul>
<p><strong>（1）优化第一印象注意力分散</strong></p>
<p>首屏页面排列和内容是最重要的，页面的这部分要为页面的第一印象负责。</p>
<p>所以应当把自己放到潜在客户的角度来看问题，检查背景纹理、不对称设计和复杂图形处理是否得当。</p>
<p><strong>（2）优化消息注意力分散</strong></p>
<p>当面临太多选择的时候，人们因为无法选择而放弃。第一印象后，消息的质量和交互尤为重要。</p>
<p>所以这个时候需要你检查信息是否过多、是否有不相关的内容或其他图片引起注意力分散问题。</p>
<h3>3. 小结</h3>
<ul>
<li>影响注意力分散的二个因素：第一印象注意力分散、消息注意力分散；</li>
<li>优化的目的：把最重要的一件事牢牢印在潜在客户的脑海里。</li>
</ul>
<h2 id="toc-6">方法6：制造紧迫性</h2>
<h3>1. 什么是紧迫性</h3>
<p>紧迫性是和决定周期的长短相关的，紧迫性取决潜在客户下决定要有多块？</p>
<h3>2. 如何优化紧迫性</h3>
<ul>
<li>内部的驱动力+外部的影响=紧迫性；</li>
<li>要给CTA，立刻执行的感觉和原因。</li>
</ul>
<p><strong>（1）优化内在紧迫性</strong></p>
<p>基于你潜在客户的已有需求和欲望前提下，你要给予感染力的外表和购买的一个理由。</p>
<p style="text-align: center;"><strong>制造内在紧迫感=感染力外表+理性的论据支持</strong></p>
<p>感染力的外表，如：买滋滋声，不卖牛排，或者多人排队购买。</p>
<p>理性化：购买是感性的，给购买一个理由是理性的。运动器械，不要放弃青春活力。</p>
<p><strong>（2）优化外在紧迫性</strong></p>
<p>向潜在的客户展示优惠，营造出产品或者服务功能好处之外的需求和欲望，如：限时优惠、突出购买等。</p>
<h3>3. 小结</h3>
<ul>
<li>影响紧迫性的二个因素：内在紧迫性、外在紧迫性。</li>
<li>优化的目的：利用害怕失去的感觉，减少用户决定周期的长度。</li>
</ul>
