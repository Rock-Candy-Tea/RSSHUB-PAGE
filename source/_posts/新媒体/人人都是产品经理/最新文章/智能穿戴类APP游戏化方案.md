<blockquote><p>编辑导语：如今，不少人都热衷于智能穿戴设备，因此也出现了越来越多的智能穿戴手表、手环、眼镜等。穿戴式智能设备时代的来临意味着人的智能化延伸，通过这些设备，人可以更好的感知外部与自身的信息，能够在辅助下更为高效率的处理信息。本文作者分享了她对智能穿戴类APP游戏化的方案，希望可以对你有所启发。</p></blockquote>
<p><img data-action="zoom" class="aligncenter size-full wp-image-4434078" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/vqwszfcNnegu9MO38Ql5.jpg" alt width="900" height="420" referrerpolicy="no-referrer"></p>
<h2 id="toc-1">一、 明确产品目标和用户需求</h2>
<p>对于智能手表的APP来说，用户是因为买了手表才来使用APP的，用户在购买手表前，要给用户一个理由，为什么买你的手表？从手表本身来看，它能为用户提供什么价值？</p>
<p>那么宣传上就要从以下四大价值点入手，从而推导用户的动机，进而制订策略放大用户动机，引导购买。</p>
<ol>
<li>使腕部看起来更漂亮：作为装饰品，是款美观的手表，有科技感，追求个性化，还可以换自己喜欢的表盘；</li>
<li>便捷的生活工具：省时省力，将节省下来的时间和力气花在更值得做的事情上，提高效率，也就是在节省自我资源；</li>
<li>健康分析：对自己健康的关注，不想失去健康和生命，希望自己可以越变越好，是对自己负责的行为，也是对关心自己的人负责；</li>
<li>运动分析：对自己的运动能力和超越自我的关注。</li>
</ol>
<p>以上四大价值点，也是日后游戏化的四个主要分支。如果用户购买了手表，也就承认了他的需求，“通过使用手表及对应的APP，让腕部更美观，生活更便捷、身体更健康、运动能力更强。”</p>
<p>由于用户已经付了费，他对于购买的手表会有一种所有权和拥有感，我们要帮助用户实现他的目标，否则他们就体会不到手表的价值，会觉得被骗了钱，不再对品牌和产品忠实，不会复购甚至阻止他人购买。</p>
<p>所以，从商业角度来看，产品的终极目标就是：提高用户的复购率、自传播率，一切业务行为都应该以这个为目的。</p>
<p>复购率和自传播率的直接数据可以从调研中得出，是定性的（复购还可以从账号绑定设备量得出，但排除不了用户用不同账号绑定的情况）。此外，还有一些定量的数据可以从侧面反映这两个指标。</p>
<p>愿意复购和自传播的用户都是被产品满足了其期待价值的用户，所以我们可以也看这四条价值对应业务的数据变化情况，下面明确下业务指标。</p>
<h2 id="toc-2">二、定义业务指标</h2>
<ul>
<li>调研得出的复购率（定性）</li>
<li>调研得出的自传播率（定性）</li>
<li>各项手表功能的使用频次（定量）</li>
<li>各项手表对应的APP功能的使用频次（定量）</li>
</ul>
<h2 id="toc-3">三、定义用户类型</h2>
<ul>
<li>喜欢科技感和美观、喜欢个性化的人；</li>
<li>对便捷生活和智能生活有着期待的人；</li>
<li>重视自己的身体健康，希望摆脱亚健康状态的人；</li>
<li>热爱运动，希望不断提升自己运动能力的人。</li>
</ul>
<h2 id="toc-4">四、定义期望行为</h2>
<p>用户一旦完成了期望行为，即会来到胜利状态。</p>
<h3>1. 发现阶段</h3>
<p>核心：让用户快速找到使用产品的原因</p>
<ul>
<li>通过朋友推荐或广告宣传购买了手表</li>
<li>下载APP、注册登录</li>
<li>成功绑定设备并将设备升级到最新版本</li>
</ul>
<h3>2. 入门阶段</h3>
<p>核心：让用户开始学习游戏玩法和工具</p>
<ul>
<li>在手机上更换一次表盘（变美分支）</li>
<li>在手表上更换一次表盘（变美分支）</li>
<li>在手表上体验主推的生活小工具（便捷分支）</li>
<li>在手机上设置主推的生活小工具（便捷分支）</li>
<li>将手表的健康监测功能逐一体验一遍（健康分支）</li>
<li>在APP上查看各项健康监测结果（健康分支）</li>
<li>将手表的运动监测功能任意体验一项（运动分支）</li>
<li>在APP上查看运动记录（运动分支）</li>
</ul>
<h3>3. 塑造阶段</h3>
<p>核心：促进用户重复进行日常行为以完成他们的目标</p>
<ul>
<li>每周至少更换一次表盘，在手表或手机上换都可以（变美分支）；</li>
<li>将手表上非主推的小工具也体验一遍（便捷分支）；</li>
<li>每天至少使用一款生活小工具（便捷分支）；</li>
<li>健康监测功能常开，每天查看健康监测数据（健康分支）；</li>
<li>每天至少进行一项运动，并在APP上查看运动数据（运动分支）。</li>
</ul>
<h3>4. 终局阶段</h3>
<p>核心：加固加深产品体验，留住忠实用户</p>
<ul>
<li>使用在线表盘编辑工具，自己制作表盘（变美分支）；</li>
<li>每天使用多款手表工具，包括闹钟、日程、天气、支付、乘车等（便捷分支）；</li>
<li>制订健康管理计划，用手表监测并提醒自己（健康分支）；</li>
<li>制订运动计划，用手表监测并提醒自己（运动分支）；</li>
<li>向他人分享健康和运动心得；</li>
<li>大众代言人，指出产品的优缺点，分享给更多的人看到。</li>
</ul>
<h2 id="toc-5">五、定义反馈机制</h2>
<p>反馈机制明确了用户胜利状态的进度，让用户感受奖励的不确定性，每个反馈机制都应该带来新的触发器，进而继续触发期望行为。</p>
<h3>1. 哪些位置可以植入反馈机制和触发器？</h3>
<ul>
<li>手表上</li>
<li>APP首页</li>
<li>APP站内信</li>
<li>APP内其他资源推广位</li>
<li> APP内全局提示bar</li>
<li>APP健康和运动数据详情页</li>
</ul>
<h3>2. 可以应用哪些反馈机制？</h3>
<ul>
<li>进度条</li>
<li>任务板</li>
<li>任务完成提示</li>
<li>随机神秘宝箱</li>
<li>勋章墙</li>
<li>收藏集</li>
<li>称号表</li>
<li>荣耀海报相册</li>
<li>健康项目达标等级图：睡眠、压力、静息心率、血氧、体重、身体成分等</li>
<li>运动技能树</li>
<li>用户活跃度：按腕部变美、便捷生活、关注健康、加强运动四个分支来展现。</li>
<li>用户成长等级阶梯：按个性化等级、便捷生活等级、健康等级、运动等级划分。</li>
<li>用户排行榜</li>
<li>用户互动反馈</li>
<li>卡通人物形象</li>
</ul>
<h2 id="toc-6">六、定义奖励和激励</h2>
<ul>
<li>角色成长的经验值</li>
<li>成就象征</li>
<li>活跃等级</li>
<li>用户成长等级</li>
<li>点亮的运动技能点</li>
<li>达标的健康项</li>
<li>各项排名</li>
<li>他人的认可和鼓励</li>
<li>完成健康计划和运动计划的成就感</li>
<li>卡通人物形象及其道具。</li>
<li>收藏集的卡片碎片</li>
<li>手表配件优惠券</li>
<li>手表优惠券</li>
<li>运动产品优惠券</li>
<li>手表配件</li>
<li>手表</li>
<li>运动产品</li>
<li>漂亮的腕部展现</li>
<li>便捷的生活方式</li>
<li>健康的身体</li>
<li>强大的运动能力</li>
</ul>
<h2 id="toc-7">七、安排游戏玩法</h2>
<p>从八大核心动机角度去安排：</p>
<h3>1. 史诗意义与使命感</h3>
<p>通过广告、首页提示和站内信通知，灌输用户彰显独一无二的自我的重要性、简便的生活能为我们带来什么、健康的身体能为自己和世界带来什么，强大的运动能力如何让个人成为少数体能精英，唤醒用户的使命感。</p>
<h3>2. 进步与成就感</h3>
<p>设置任务板，每当用户完成任务后，给用户加相应的经验值和活跃度。随着经验值增加，用户对应分支的等级也会升高。活跃度会加速经验值的增长，活跃度越高经验值增长的越快。</p>
<p>将健康项目设置为人物护甲，每达标一个健康项目，就点亮对应的护甲，显示连续达标了多久，累计达标了多久。并且将护甲分级，健康状态越高，护甲等级越高（可以在后期拓展护甲的玩法）。</p>
<p>设置技能树（天赋图），将用户做过的运动项视为学会的技能将其点亮。根据用户的运动纪录，显示每项运动技能的等级，当达到一定等级时，赋予用户称号，并且将称号分级。</p>
<p>当用户完成特定的或系列的任务后，授予用户勋章，设置勋章墙进行展览。当健康计划和运动计划有进度时，即时提示用户。计划完成时生成荣耀海报，将所有的荣耀海报组成相册，增加用户成就感。</p>
<h3>3. 创意授权与反馈</h3>
<ul>
<li>自定义表盘</li>
<li>用表盘工具制作表盘</li>
<li>制订自己的健康计划并执行</li>
<li>制订自己的运动计划并执行</li>
</ul>
<h3>4. 所有权与拥有感</h3>
<p>在用户完成任务后，随机给用户神秘宝箱，点击可随机获得一张卡片的碎片，同一张卡片的碎片集齐后可解锁卡片，卡片是一张配图加简介，配图可下载后当做表盘。配图可以是著名建筑、景点、动物等，需要不断收藏，构成完整的收藏集。</p>
<p>给用户推送消息提醒用户查看自己的生活周报、月报、年报。如果用户超过3天没有数据更新，推送消息提醒用户查看个人数据。</p>
<p>给用户设置卡通人物形象，并让用户自己搭配服饰肤色等等。这个形象会随着个性化、便捷生活、健康、运动的等级以及活跃度的变化而变化，搭配人物的道具也会发生变化。</p>
<h3>5. 社交影响及关联性</h3>
<ul>
<li>展示用户不同分支的等级的排行榜；</li>
<li>健康分项的讨论圈；</li>
<li>展示用户运动能力等级的排行榜；</li>
<li>运动分项的讨论圈；</li>
<li>在圈子中设立团队任务：需要合作完成的健康达标任务，或合作完成的运动纪录；</li>
<li>将用户的个性化等级、健康项目等级、便捷生活等级、活跃度、技能树、运动技能等级、成就、称号、勋章、荣耀相册、收藏集展示在个人主页；</li>
<li>将精英用户和飙升用户展示出来；</li>
<li>用户可以将荣耀海报、勋章、称号、收藏、成就等分享出去；</li>
<li>用户可以对他人进行表态互动，用户会收到通知。</li>
</ul>
<h3>6. 稀缺性与渴望</h3>
<ul>
<li>限制用户每天可以完成的任务数量，对应的分支等级越高，可以完成的任务数越多；</li>
<li>显示待解锁的特殊任务，一定等级的用户才可以做；</li>
<li>设置每周末运动经验加倍、任务奖励几率加倍活动；</li>
<li>在健康项目没有打标前，隐藏对应的健康等级；在运动技能点没有点亮时，隐藏技能等级，技能专项记录等；在某一分支的收藏没有任何东西时，隐藏其分支；显示待解锁的高级成就；隐藏高级的人物形象设置等。</li>
</ul>
<h3>7. 未知性与好奇心</h3>
<ul>
<li>锁定中的模块；</li>
<li>随机掉落的任务奖励；</li>
<li>可拓展的收藏分支；</li>
<li>只展示自己排名附近的用户；</li>
<li>在运动中取得特殊的成就时，会获得之前未知的特殊称号。</li>
</ul>
<h3>8. 亏损与逃避心</h3>
<ul>
<li>用图表展示出用户的历史数据记录成就荣耀收藏等，提醒用户已经投入了多少；</li>
<li>健康情况降低时，护甲会减少；</li>
<li>提醒用户如果不好好对待自己的身体会发生什么；</li>
<li>用卡通人物形象直观反映出健康和运动情况的变化，提醒用户注意；</li>
<li>活跃度是会降的，如果任务失败，相应的活跃度就会逐渐降低；</li>
<li>在计划中显示用户可执行计划的剩余天数；</li>
<li>在奖励加倍时间段内，显示倒计时。</li>
</ul>
