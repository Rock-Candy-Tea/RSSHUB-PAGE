<h2 id="toc-1">一、第一章：报告概述</h2>
<h3>1. 产品定位与平台概述</h3>
<p>产品定位：兼具社交和财富管理功能的美港A股线上交易平台</p>
<p>平台概述：富途通过自主研发的一站式数字化金融服务平台富途牛牛，提供面向个人投资者的市场数据、资讯、社交等服务，旗下富途证券等美港股券商提供面向个人投资者的美股、港股、A股通的股票交易和清算、融资融券、财富管理等服务，并以用户为中心构建完善的金融科技生态系统，连接用户、投资者、企业、证券分析师、媒体和意见领袖。</p>
<h3>2. 市场现状</h3>
<p>近年来受益于证券市场总体规模增长、移动端技术进步、以及新生代投资者对线上渠道的偏好，全球主要证券市场线上交易额快速增长。中国作为个人投资者人数最多的国家在境外投资市场的渗透率不断增加。</p>
<p>近几年经济发展和股市提振，越来越多的散户加入炒股理财的群体，而随着部分基金经理向管理基金加入港股仓位获得规模和收益双丰收，吸引内地股民将目光投向港美股市场。</p>
<h3>3. 目标用户</h3>
<p>1）用户分层</p>
<p>根据投资能力和投资意愿，将富途新股产品的用户分为有专业水准的深度用户和投资小白两大类，后续将根据两类用户的用户画像和需求、痛点进行产品、运营和营销等方面的差异化分析，便于输出有针对性的提升建议和执行方案。</p>
<p>2）用户特质—场景—需求</p>
<p>基于两类用户群体，确定典型的用户特质，结合场景和需求进行概括，在此基础上展开后续分析。同时将在产品、运营、营销层面的分析中挑选部分需求，结合竞品分析，给出具体解决方案。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/04/8hKh7tSLWZ88c36iq7hj.png" alt width="802" height="430" referrerpolicy="no-referrer"></p>
<h2 id="toc-2">二、第二章：产品层面分析</h2>
<h3>1. 投资小白</h3>
<p><strong>1）了解港股打新相关机制和参与方式</strong></p>
<p>该需求主要是打新初期了解阶段，投资小白在平台内了解打新产生的，痛点是高效便捷了解打新的基本机制和参与方式。接触港股打新的第一步就是在平台内搜索并了解相关机制和参与方式，存在的问题包括但不限于：</p>
<ul>
<li>什么是打新？</li>
<li>如何参与打新？</li>
<li>新股认购需要准备多少本金，额外费用和收益率有没有预期数字？</li>
<li>打新风险大不大？</li>
<li>认购成功后的操作步骤？何时可以卖出？</li>
<li>富途平台可以为打新提供哪些差异化服务和优势？</li>
</ul>
<p>针对以上问题，目前富途采用了在【交易—新股中心】中投放滚动banner图的方式进行科普和解答疑问，但图文形式只能带给用户初步了解，真正促使用户参与打新活动的是有打新经验、拥有相关丰富知识的人对新用户的鼓动和反馈。</p>
<p>三个提升建议：</p>
<ul>
<li>社区内开设打新专区或专栏，供用户发表参与打新活动的感受和经验，同时提供渠道给老用户向新用户传授打新经验。</li>
<li>在新股中心中设置【牛牛圈-课堂】的引流通道，帮助新手用户通过课程快速了解打新。</li>
</ul>
<p>用户初次参与打新时，希望可以观看解读视频或以图文形式详细了解打新知识，【牛牛圈-课堂】功能模块的目标用户恰好为小白以及专业知识一般的股民，以提供课程的方式吸引这些新手参与打新活动，同时提供较好的用户体验保证打新活动的参与人数和关注度持续增长。</p>
<p>同时，课堂板块拥有打新秘籍专区，但位置较深，对投资小白来说难以触及，在新股中心插入打新秘籍专区的icon或者快捷进入按钮，一方面帮助用户更清晰易懂地了解打新的秘诀，另一方面也提高【课堂】功能模块的日活量。</p>
<ul>
<li>以图文和视频结合的方式帮助用户快捷高效了解打新知识</li>
</ul>
<p>在【交易-新股认购】3分钟玩转港股打新的H5界面中插入视频解读，辅以文字描述和图片解读，由于用户进入新股认购界面易触达banner图，因此进入H5页面观看视频的可能性较高，且观看视频帮助提高用户活跃时间。</p>
<p><strong>2）完成开户入金认购</strong></p>
<p>参与打新的新手对于招股书等呈现细节数据的文件材料没有很高的敏感度，因此从获取保荐人和基本行业信息等浅层信息的角度，给用户积极的心理暗示，促使认购行为的产生。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/04/ouafrX2fhqSRdfEEBIXy.jpg" alt width="458" height="317" referrerpolicy="no-referrer"></p>
<p style="text-align: center;">老虎证券新股申购界面</p>
<p>根据对老虎证券的新股申购界面的竞品分析，在【交易—新股认购】模块的新股信息列表中增加保荐人和行业、公司介绍等信息，有助于提振用户对投资的信心。</p>
<p><strong>3）独立浏览投资专业信息</strong></p>
<p>用户获取新股信息有两种方式：一种是独立了解，一种是社交互动。其中独立了解新股资讯的方式包括搜索公司相关信息和最新动态、平台内浏览相关文章和交流帖等。</p>
<p>使用频率最高的是浏览【资讯—新股】内的文章，获取对新股的了解。但由于发布方和文章主题的差别，导致内容与内容的关联性较弱，用户看完一个帖子对目标新股产生兴趣，无法快速找到针对同一个新股的分析报告，目前文章底部的热点推荐无法解决此问题。</p>
<p>建议采用以公司名和股票名为标签的内容整合机制，用户浏览文章时，平台方会根据标签信息推送相关的文章，提高用户持续浏览文章和专栏内容的时效性。</p>
<p><strong>4）基于打新的社交互动</strong></p>
<p>注：将用户可能发生的与其他用户的互动（评论、私聊）和发帖分享统统归纳为社交互动的需求。</p>
<p>用户在参与打新活动的前、中、后阶段，都有和平台内其他用户产生社交互动的需求，但需求的痛点略有区别。</p>
<ul>
<li>打新前，主要是获取对打新的专业建议的阶段，用户需要了解准确权威的打新攻略，为自己的操作提供准则和确定执行方案；</li>
<li>打新过程中，用户需要针对盈亏情况确定是否需要加仓/减仓/清仓，这是的社交活动主要是观望其他用户的操作和分析，对后续自己的行为进行指导；</li>
<li>打新后，用户产生话题讨论或分享经验的行为，目的是分享自己对打新活动的理解、心情和经验教训，基于打新的社交讨论是对整个过程的回顾和复盘。</li>
</ul>
<p>因此，对于富途牛牛新股产品的社交层面的用户需求，也是围绕过程的三个阶段展开。</p>
<ul>
<li>打新前——官方开设港股投资打新训练营</li>
</ul>
<p>以课程（直播课/录播课）+社交（聊天室）的形式向打新新手科普相关知识，基于打新话题提升用户粘性和活跃度，同时带动聊天群的互动量和活跃人数。这一需求涉及社群运营，将放在第三章中给出具体的解决方案。</p>
<ul>
<li>打新中——引入专业投资人给出操作意见</li>
</ul>
<p>在讨论区开辟置顶或精选评论模块，引入专业的投资人作为意见领袖，跟进新股的各项数据分析，给用户提供参考性增减仓意见。</p>
<ul>
<li>打新中——股票首页增设投票控件</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/04/dsQOUZe16IbXp8c6v9pE.png" alt width="703" height="80" referrerpolicy="no-referrer"></p>
<p>同花顺的股票走势图下方有弹幕评论和投票控件，增加互动量。富途的股票主页面则以交易数据为主，专业性更强，符合主要用户群体的期望，但对打新的新用户不够直观和易懂。</p>
<p>在股票的首页可增设投票小控件，“看涨”、“看跌”选项可供选择，在用户关注报价波动趋势同时了解其他用户对新股的趋势分析，帮助用户提前做好趋势估计和操作计划。</p>
<ul>
<li>打新后——评论区分享趋势图</li>
</ul>
<p>由于相当一部分用户会在新股上市前一天暗盘或上市当天抛售股票，因此需要引导这部分用户完成分享反馈行为。提供强提醒引导用户进入评论区进行心情分享或收益公布，例如分享趋势图和收益率/收益具体金额，满足自身的荣誉感。</p>
<h3>2. 资深股民</h3>
<p><strong>1）注重深广度的社交</strong></p>
<p>资深股民坚持使用富途牛牛的原因除了良好的用户体验外，还包括输出专业意见和广泛社交带来的精神激励和身份认同。</p>
<p>富途的牛牛圈和资讯的内容形态均以深度长文为主，注重文章的质量和内容的深度。短评或图文分享的曝光度有限，且集中在自己的关注动态中，因此社交对于资深股民吸纳粉丝和关注的长尾效应受到局限。为了能够让这部分人群能够更到获赞、涨粉、提高知名度和影响力的精神激励，需要提高曝光次数和范围。</p>
<p>【牛牛圈—关注】的信息流中，目前穿插了“可能感兴趣的人”，但位置较深，且和其他内容区分度不高，不够对用户造成直观的视觉提醒。</p>
<p>可将感兴趣的人置顶，或用不同颜色的色块予以区别，提高打新话题活跃用户在这个功能模块内出现的几率，提高对新股认购感兴趣的用户对这部分内容的关注度，很容易产生关注、浏览。</p>
<p>对于打新相关的话题，可以使用banner图或轮播图的方式在牛牛圈内进行选择性投放。资深股民进入发布自己的见解，由于内容的优质性较获得别人的点赞、关注或订阅；同时在【牛牛圈—推荐—正在讨论】中对该类话题进行投放，可帮助这些资深股民实现用户增长。</p>
<h2 id="toc-3">三、第三章：运营层面分析</h2>
<h3>1. 用户激励方案设计</h3>
<p><strong>1）现有用户激励方案</strong></p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/04/HpsmZvfaXbOBOeegPbPK.png" alt width="602" height="610" referrerpolicy="no-referrer"></p>
<p style="text-align: center;">现有用户激励方案</p>
<p><strong>2）用户激励方案的修改</strong></p>
<ul>
<li>物质奖励</li>
</ul>
<p>对投资欠缺经验的小白们，他们对于新股的自身认知和分析能力相对较弱，也较容易受他人影响盲目追随，这类用户倾向于跟随KOL进行股票投资。</p>
<p>平台方也可以顺应这个行为，鼓励新用户和对打新有意愿的投资者多浏览多关注新股认购KOL，在【新手成长任务】和【每日任务】中增设浏览并参与打新话题讨论的任务，用户完成任务后同样给予积分奖励。</p>
<ul>
<li>精神激励</li>
</ul>
<p>现有的勋章体系对于创作、输出自己观点的激励没有太多的关注，为了鼓励用户以点赞、评论、聊天室内发言、撰写动态专栏笔记的形式参与社交互动，平台可以设计推出一系列创作者的勋章激励，这些用户的内容也可以获得不同程度的曝光奖励，实现涨粉涨关注度。</p>
<p><strong>3）活动运营——打新创作者计划</strong></p>
<p>对于打新这种全年都可进行的行为，可以举办打新创作者计划，鼓励新老打新参与者输出自己的心得体会，输出的优质内容可获得物质奖励，如一手腾讯股票或理财卡，还可以获得一定程度的精神奖励，如奖励优质内容输出者的头像边有“优质创作者”的logo标识或勋章，以后参与社区讨论时更容易引起别人关注。</p>
<p>优质内容可在社区内沉淀，供以后的用户参与打新时参考。类似的活动还可以推广到更大范围，如财富管理、期货交易等，进一步形成一个具有海量优质内容的素材池，对于今后专栏撰写和文章的策划也能提供资源和素材。</p>
<h3>2. 社群运营——打新社交需求的运营提升方案</h3>
<p>目前富途牛牛的聊天室功能尚未充分利用，聊天室内的内容以分享的文章专栏为主，用户间的互动少，粘性低。</p>
<p>在第二章中提及打新社交互动可由官方开设“港股投资打新训练营”，需要引入工作人员做好社群运营和促活，保障群内生态的健康和防止低劣信息的产生扩散造成不良影响。</p>
<ul>
<li>开始打新课程前进行用户问卷调查和小范围的用户访谈，了解用户在正式了解打新知识前对打新的顾虑和疑问，做好答复，同时跟进疑问的后续解决。</li>
<li>课程过程中了解用户的学习情况，开设打卡活动，对于按时完成学习的用户给予奖励，起到督促和鼓励的作用，</li>
<li>课程后了解用户的满意度和课程改进建议，做好用户体验闭环，为后续课程的改进做好反馈</li>
<li>活动运营——完成打新课程后可根据市场行情遴选打新收益汇报高的新股，在群内分享官方分析和打新建议，鼓励散户认购新股。针对认购成功的用户，可采用一对一答疑或邀请有经验的股民进群分享后续操作。</li>
<li>为保持聊天室持续的活跃度，运营人员可策划每日话题或转发高质量文章到群内引导大家参与讨论。</li>
</ul>
<h2 id="toc-4">四、第四章：营销层面分析</h2>
<h3>1. 确定营销策略</h3>
<p>将打新风险作为营销的突破口，向用户阐明打新的预期风险和规避措施，在站内站外宣传推广，提高平台名望和用户口碑。</p>
<p>现设计基于风险普及宣传的营销策略。</p>
<h3>2. 平台内营销策略</h3>
<p><strong>1）提示打新风险</strong></p>
<p>在新股认购前，提示用户注意新股破发风险，以及认购费、佣金、交易费、融资利息等费用。帮助用户明晰风险后，再予以课程、打新训练营等活动的投放，提醒用户提高甄别新股的能力，学会选学会挑，一方面增加用户的信任，另一方面帮助用户规避一部分风险。</p>
<p><strong>2）科普出金成本</strong></p>
<p>大部分平台在宣传新股认购时注重入金操作，对于出金的操作和提醒很少，科普出金的费用组成和比例虽然可能导致认购数量的减少，但会增加用户对平台的认购度，有望增加用户粘性。这部分内容可以借助聊天室或弱提醒功能进行定期的推送。</p>
<h3>3. 平台外营销策略</h3>
<p><strong>1）驳斥打新谣言</strong></p>
<p>很多大V通过微信公众号和百家号等渠道推荐港股打新，强调包赚不赔，并在文末附上港股开户和9.9元打新培训的软文链接，这是利用信息不对称严重损害用户权益的行为，同时也让不少用户对港股打新望而生畏。</p>
