<h2 id="toc-1">一、私域营销体系的定义</h2>
<p>构建私域为网络营销体系提供了全新的管理视角。</p>
<p>私域是公司为用户创建一个以服务、互动和营销为导向的网络区域。</p>
<h2 id="toc-2">二、构建私域营销体系的原因</h2>
<ol>
<li>能够近距离并可持续性的接触到客户。</li>
<li>降低网络营销活动的获客成本。</li>
<li>增强客户和品牌之间的情感联系。</li>
</ol>
<h2 id="toc-3">三、私域营销体系对于品牌的价值</h2>
<p>如何定义品牌、IP、私域之间的关系：</p>
<ul>
<li>品牌：品牌的目标是让产品与消费者之间形成烙印关系。</li>
<li>IP：将品牌转化为多种意识形态元素塑造的形象。</li>
<li>私域：网络营销过程中品牌和IP建立的专属区域。</li>
</ul>
<h2 id="toc-4">四、构建私域营销体系的准则</h2>
<h3>1. 客户关系管理（CRM）</h3>
<p>从管理理论的角度来看，私域管理将赋予CRM全新的运营模式。</p>
<p>CRM的目标是吸引新客户，留住老客户，将现有客户转变为忠实客户。</p>
<p>私域和CRM的目标一样，但CRM更倾向于流程、效率、数据等内部管理，私域则倾向于营销、内容、社交等外部管理。</p>
<p><strong>私域是连接CRM和SCRM的重要组成部分。</strong></p>
<h3>2. 品牌区域领袖（BRL）</h3>
<p><strong>1）品牌推广</strong></p>
<p>如果将品牌营销看成一场战役，推广的重点就是攻占电视、广播、路牌等重要区域。然而，随着移动互联网的快速发展，我们已经进入了信息碎片化时代。原来几个重要的区域被分成了若干个区域，每个区域都有非常快的变化周期和不同的规律。因此，企业需要根据不同地区制定相应的品牌战略。</p>
<p><strong>2）品牌IP化</strong></p>
<p>品牌做的越大灵活性就越差，品牌IP化就起到了关键作用。如果把品牌想象成一支军队，那么IP就是特种部队，或者是游击队，化整为零，统一安排，统一标识，根据不同地区的不同情况，自主制定战略，拓展版图，完成任务。有些公司特别注重品牌推广，从而忽视了IP的建设，以至于投入巨大，但收效甚微。</p>
<p><strong>3）IP</strong><strong>与私域的关系</strong></p>
<p>IP是一种可以变化成为多种意识形态的元素，公司创造它，重塑它，并让它成为一个区域的领袖，目标是对该区域的用户群体产生足够的影响力。</p>
<p><strong>4）品牌区域领袖</strong></p>
<p>公司在打造IP时一定不要被品牌决策局限住，要勇于重塑和突破，让它在各个区域展现出更多的形态。但是，这种形态上的变化使其不再像IP。</p>
<p>作者提出一个新的叫法：<strong>品牌区域领袖 Brand Regional Leader</strong>。再搞个缩写：<strong>BRL</strong>。</p>
<p><strong>5）私域为什么需要BRL</strong></p>
<p>当用户来到私域体验时，发现这里只有品牌和IP。对用户而言，再巧妙的设计方案也是一种营销行为。但是，如果用户在私域中找到一个人、一个朋友、一个智者、一个偶像、一个领袖，情况就大不相同了。</p>
<p><strong>6）品牌、IP和BRL之间的关系</strong></p>
<p>品牌是公司的根基，IP是品牌的树干，BRL是品牌的枝和叶。公司不仅需要打造一棵“大树”，还需要在各区域展示出去，将品牌枝繁叶茂、生机勃勃的样子尽情释放。</p>
<p><strong>7）BRL的四种类型</strong></p>
<p>① 领域专家</p>
<p>苹果的乔布斯、小米的雷军、格力的董明珠。他们不仅代表了公司和品牌，也将自己描绘成这个领域的专家。BRL也可以是公司的技术人才，并且在该领域取得了突出的成绩，专家往往赢得尊重，被赋予权威。如果你的公司没有这种人，可以找这方面有影响力的专家，达成战略合作。</p>
<p>② 品牌代言人</p>
<p>完全服务于公司的独立形象，例如小完子和小美子，她们就是完美日记的代言人。因为代言人是真实存在的，所以公司在培养代言人时，品牌战略要倾向于代言人的特点，代言人要与品牌充分融合。</p>
<ul>
<li>本文所讲的品牌代言人不是为品牌代言的公众性人物，而是完全服务私域的IP形象。</li>
<li>为品牌代言人打造 IP 形象。</li>
<li>品牌代言人不仅要代表品牌形象，还要参与到产品、价格、服务等工作中。</li>
<li>公司应该按照培养高管的标准来培养品牌代言人。</li>
<li>公司选择代言人时要考虑到目标用户群体的需求。</li>
<li>品牌代言人是BRL最重要的一环。</li>
</ul>
<p>品牌代言人设计要点：</p>
<ol>
<li>基于人物真实形象，添加并修饰人物故事。</li>
<li>品牌代言人需要具备网络中高频活动的属性。</li>
<li>如果是快消品行业，除了打造品牌代言人以外，还可以招募数个兼职类品牌代言人，利用其自身特点打造真实形象，介于KOC和UGC之间。这一类品牌代言人时效性较短，需要多种IP形象支撑。</li>
<li>如果是toB公司，品牌代言人不需要高频活跃，但专业性要突出。</li>
</ol>
<p>③ 关键意见领袖（KOL）</p>
<p>拥有更多、更准确的产品信息，且为相关群体所接受或信任，并对该群体的购买行为有较大影响力的人。薇娅和李佳琦是KOL的典型代表。</p>
<ul>
<li>公司与KOL合作的时候，不仅需要让商品得到推广和销售，更需要让品牌代言人得到推广。</li>
<li>品牌代言人与KOL的互动不宜过多，但每次互动都要把动静搞大点。</li>
</ul>
<p>关于KOL，公司可以设计两套方案：</p>
<ol>
<li>付费与KOL合作，让品牌代言人参与其中，达到销售目标和私域引流的双重目的。</li>
<li>公司打造KOL，虽然需要付出很多时间成本，但带给私域的好处要明显大于付费合作的KOL。</li>
</ol>
<p>④ 关键意见消费者（KOC）</p>
<p>KOC本身就是一个消费者，分享的内容多为亲身体验。他们更贴近消费者，更注重与粉丝的互动。</p>
<ul>
<li>与KOC互动是品牌代言人的主要任务之一。</li>
<li>种子用户与KOC属于同一群人。</li>
<li>公司不仅要培养全职KOC，还要从种子用户中挑选KOC进行合作。</li>
</ul>
<p><strong>8）BRL的建设有两个主要方向</strong></p>
<ol>
<li>在公司内部打造BRL时，领域专家和品牌代言人是最适合打造BRL的两种类型。</li>
<li>公司与外部KOL和KOC进行合作，合作所产生的用户过渡给品牌代言人接管。</li>
</ol>
<p><strong>对于营销预算充足的公司，四种类型的BRL都可以打造，同时在外部寻找适合公司的四种类型BRL进行合作，搭建流量矩阵。</strong></p>
<h3>3. 内容营销</h3>
<p><strong>1）</strong><strong>IP故事</strong></p>
<p>找到了合适的人选做BRL之后，你需要为BRL写一个故事。乔布斯、雷军、董明珠、微娅、罗永浩、小完子等等，每个人都有属于自己的故事。故事可以是真实的，也可以是虚构的。用户并不在乎你的故事是真是假，只在乎你的故事是否有趣。</p>
<p><strong>2）</strong><strong>内容种类</strong></p>
<p>剧情、段子、励志、知识、案例、研究、经验、观点、新闻、搞笑、八卦、情感、鸡汤、爆料、排行等等，也可以根据各平台的内容做分类。选择种类后，结合BRL特点，就可以开始创作了。</p>
<p><strong>3）</strong><strong>品牌生产内容（BGC）</strong></p>
<p>BGC有三个工作重点：</p>
<ol>
<li>BGC延续品牌现有的内容，将BRL融入到BGC中。</li>
<li>BGC围绕BRL持续的制造新内容。</li>
<li>BGC做内容框架，与UGC合作生产内容。</li>
</ol>
<p>BGC需要经常与PGC、OGC、UGC合作输出内容，在内容更新上形成互补，可以更好的适应网络热度更新迭代快速的特点。</p>
<p>很多PGC公司都有KOL，很多OGC本身就是KOC。</p>
<p>最难的部分是UGC，BGC需要设计足够巧妙的策划案才能与UGC的合作达到理想的效果。</p>
<p>可以关注一下小米花费200万改LOGO的事件是如何引起UGC讨论的，非常有趣。</p>
<h3>4. 圈层</h3>
<p><strong>1）</strong><strong>私域为什么要圈层化？</strong></p>
<ul>
<li>用户有需求，每个用户都会寻找属于自己的圈子。</li>
<li>当用户聚集在私域，并且形成了圈层，信息传递和互动会更加精准。</li>
<li>为营销与内容输出提供战略方向。</li>
</ul>
<p><strong>2）</strong><strong>圈层原则</strong></p>
<p>① 认同原则</p>
<ul>
<li>共同的兴趣与爱好。</li>
<li>共同的态度与价值观。</li>
<li>共同的消费习惯。</li>
<li>共同的生活方式。</li>
</ul>
<p>② 领袖</p>
<ul>
<li>一个拥有聚集能力的领袖。</li>
<li>一个可以被追随的领袖。</li>
</ul>
<p>③ 终极目标</p>
<ul>
<li>不同用户的兴趣与爱好。</li>
<li>不同用户的态度与价值观。</li>
<li>不同用户的消费习惯。</li>
<li>不同用户的生活方式。</li>
</ul>
<p>我们可以做到将拥有共同的兴趣、爱好、态度、价值观、消费习惯、生活方式的用户聚集在一起。然而，随着圈子不断扩大，不同的用户必然会出现。这个时候，圈层营销会发生变化，我们又要想办法破圈，但破圈又违背了认同原则。所以，在构建圈层的初期，我们要制定终极目标。终极目标应主张以BRL为中心，逐步向外拓展。</p>
<p><strong>3）</strong><strong>构建圈层营销的步骤</strong></p>
<ul>
<li>参照圈层原则，准确规划圈子。</li>
<li>确定BRL的中心地位。</li>
<li>设计圈层：分析什么才是用户需求的商品。并且要以内容为主，销售为辅。</li>
<li>对圈层进行深入的研究，持续升级与扩建。</li>
</ul>
<p><strong>4）</strong><strong>圈层案例</strong></p>
<ul>
<li>完美日记：美妆-大牌平替</li>
<li>泡泡玛特：潮玩</li>
<li>花西子：美妆-国风</li>
<li>江小白：都市文艺青年</li>
<li>超级文和友：美食与体验</li>
<li>孩子王：母婴、教育与家庭</li>
</ul>
<h3>5. 用户情感链接</h3>
<p><strong>1）为什么要建立用户情感链接</strong></p>
<p>私域不能依靠卖产品长期运营，公司要考虑用户更深层的感受。</p>
<p><strong>2）用户的深层需求</strong></p>
<ul>
<li>用户不买产品的时候要做什么？</li>
<li>公司除了卖产品还能做什么？</li>
<li>如何保持社区活跃？</li>
<li>如何与用户保持长久稳定的关系？</li>
</ul>
<p><strong>3）品牌的深层服务</strong></p>
<ul>
<li>孩子王除了销售产品，还担任着老师、顾问、专家、甚至是催乳师等角色。</li>
<li>文和友除了美食以外，将文化和体验感提升到了新的境界，用户不仅味蕾得到了满足，感官也得到了满足。</li>
<li>江小白将情感文案和白酒融合在一起，迎合了酒桌文化“我们喝的不是酒，是感情”。</li>
<li>做好用户情感链接是培养用户对品牌忠诚的基础。</li>
</ul>
<p><strong>4）构建情感链接的步骤</strong></p>
<ol>
<li>获取用户的基本信息。</li>
<li>挖掘用户情感需求。</li>
<li>激发用户的情感共鸣。</li>
</ol>
<h3>6. 社区氛围</h3>
<p><strong>1）什么是社区氛围</strong></p>
<p>指在社区中逐渐形成的，具有一定特征，能够被用户认可的氛围和环境。</p>
<p><strong>2）营造社区氛围的步骤</strong></p>
<p>① 成立氛围小组</p>
<ul>
<li>围绕BRL配备独立的策划、设计、文案等人员。</li>
<li>气氛小组成员点子要多、性格活跃、不拘一格。</li>
<li>气氛小组要独立运营。</li>
</ul>
<p>② 挑选种子用户</p>
<ul>
<li>种子用户至少要认同品牌、IP、BRL其中的一项。</li>
<li>种子用户不仅自愿进入社区，还愿意拉人进来。</li>
<li>种子用户表达、讨论、互动的意愿非常强烈。</li>
</ul>
<p>③ 创造社区文化</p>
<p>给用户个身份，制定文化、名称、代号、标识、等级、勋章、荣誉等等。</p>
<p>④ 社区日常运营</p>
<ul>
<li>准备话题、活动、竞赛、红包等多样性主题。</li>
<li>安排群托，每日在社区内调动气氛。</li>
</ul>
<p><strong>3）关键问题</strong></p>
<ol>
<li>内容平台的社区氛围是BGC和UGC之间通过合作关系与博弈关系所激发的结果，两者属于对等的位置。</li>
<li>私域的社区偏向于组织关系，属于领导者与追随者的关系，所以营造氛围要以用户体验为导向。</li>
<li>营造社区氛围的核心目的是希望私域变成一个有温度的购物中心，而不是变成一个仓储式大卖场。</li>
</ol>
<h2 id="toc-5">五、构建私域营销体系的步骤</h2>
<h3>1. 确立私域构建计划</h3>
<p>公司在考虑部署私域营销体系之前，首先确定这一体系具体的营销目标。</p>
<ol>
<li>升级CRM体系。</li>
<li>增加商品销售额。</li>
<li>升级互联网营销体系。</li>
</ol>
<h3>2. 围绕BRL建立团队</h3>
<p>管理者先要确定BRL人选，并围绕BRL建立一支具有互联网思维的团队。BRL人选至关重要，尤其是品牌代言人，必须具有培养价值，并在未来的规划中应属于公司管理层的一份子。</p>
<h3>3. 确立私域思维</h3>
<p>私域思维由用户思维和产品思维两部分组成。以用户和产品命题、搜索相关数据进行有针对性的、全方位的设计与分解，捕获其中的问题和解决方案、最后再进行一次综合整理，得出的结论即是私域思维。</p>
<h3>4. 明确私域用户定位</h3>
<p>围绕五个主要问题进行研究：</p>
<ol>
<li>用户是谁？</li>
<li>用户想要什么？</li>
<li>用户想看什么？</li>
<li>如何满足用户的需求？</li>
<li>用户有什么习惯？</li>
</ol>
<h3>5. 明确互联网营销策略</h3>
<p>以用户为中心，洞察与分析用户的心理和行为。</p>
<p>定位具有互联网营销思维的品牌与产品。</p>
<p>让BRL活跃于各大流量平台。</p>
<h3>6. 选择公域平台</h3>
<p>确保所选择的公域平台有足够的流量可以导向私域。</p>
<p>了解其平台的规则和玩法，如何与私域相匹配。</p>
<h3>7. 开发与部署</h3>
<p>私域营销体系的开发，需要公司各部门共同努力。为了使私域得以迅速实现，公司应先将公司文化、品牌、IP等核心的思维与策略网络化，之后再不断的添加新内容。</p>
<h2 id="toc-6">六、构建私域营销的两种维度</h2>
<ol>
<li>围绕微信生态圈构建，侧重点在产品销售。</li>
<li>围绕BRL构建，侧重点在于打造领袖形象。</li>
</ol>
<p>具体的侧重点要分析品牌的用户特征适合哪一种。无论哪一种，BRL都占主导地位。</p>
<p>两种维度的区别：</p>
<ol>
<li>如果是快消品品牌或者产品种类很多，可围绕微信生态圈做主战场。</li>
<li>非快消品或单一产品则适合围绕BRL做合理、细分、跨区域、跨平台的布局。</li>
<li>如果公司的产品种类不多，但执意要完成微信生态圈私域构建的情况，为了预防营销内容过于同质化，公司则需要根据用户的购买特征引入更多的周边产品。</li>
</ol>
<h2 id="toc-7">七、私域营销体系中的名词与定义</h2>
<h3>1. 品牌区域领袖（BRL）</h3>
<p>构建私域的主角一定是BRL，不是品牌，也不是IP。品牌代言人是第一选择，其次是领域专家。</p>
<h3>2. 品牌代言人</h3>
<p>私域的品牌代言人与传统的品牌代言人有着显著的区别。传统公司通常会邀请具有一定传播力的公众性人物，比如明星，专家等等。在私域内，公司需要打造一名品牌代言人，她不仅要通过各种媒介传播信息，还要参与公关、促销、客服等所有的商业关联活动。可以说，以 IP 打造的代言人是私域构建最重要的环节，缺了这个环节就不能叫私域。</p>
<h3>3. IP主</h3>
<p>IP主实际是品牌代言人的一种分身，她的虚拟身份是私域的主人。</p>
<h3>4. IP 角色</h3>
<p>每个品牌的IP都不一样，这个无法复制，因为它具有多种形态意识。比较常见的有虚拟IP形象和领导人形象。</p>
<ul>
<li>虚拟 IP：泡泡玛特、三只松鼠</li>
<li>拟人化 IP：小完子、江小白</li>
<li>领导人 IP：雷军、董明珠</li>
<li>场景 IP：超级文和友</li>
<li>文化 IP：花西子</li>
</ul>
<h3>5. 用户思维</h3>
<p>分析用户的情感、心理、物质需求，为用户编故事，植入产品。从参与设计到征求意见，再到邀请用户体验，每一个环节都可以和用户一起完成。尽量在情感上打动用户，让用户实现升华。</p>
<h2 id="toc-8">八、私域营销体系的载体</h2>
<h3>1. 微信个人号</h3>
<ul>
<li>BRL之品牌代言人，也称之为IP主。IP主是用户的直接管理者，包括推广、营销、客服、互动等内容。微信群、朋友圈是IP主和用户的主要沟通渠道。</li>
<li>BRL可以设定一个角色，由不同的客服来操作，形成多个账号一个人设的结构，例如：完美日记的小完子。</li>
<li>BRL也可以设定多种角色，所有的BRL会统一标识，但不会统一人设，例如孩子王的宝妈顾问。</li>
</ul>
<h3>2. 微信公众号</h3>
<p>公众号在用户管理方面的作用：</p>
<ol>
<li>接收来自各个平台的用户，引导用户与BRL对接并进入专属的圈层与社区，实现用户的正向流动。</li>
<li>引领私域内用户去公域参加各项活动，实现用户的逆向流动。</li>
<li>品牌的内容、广告、活动、福利、通告等信息发布。</li>
</ol>
<h3>3. 微信小商店</h3>
<p>设立私域专属商场，与京东、淘宝、官网等商场形成价格策略上的互补性和灵活性。</p>
<h3>4. 微信小程序</h3>
<p>为用户建立兴趣社区，在销售以外的环节强化品牌与用户的情感链接。</p>
<h3>5. 小红书</h3>
<p>微信生态圈适用于品牌代言人（IP主），小红书适合打造KOL和KOC，两者可形成互补关系。</p>
<h3>6. 官网</h3>
<p>宣传公司的形象、文化、产品、服务的种类、价格以及联系方式等信息。</p>
<p>微信生态圈在私域营销体系属于狭义范畴，官网则属于广义范畴。</p>
<h3>7. 内容平台</h3>
<p>抖音、快手、B站、知乎、微博等平台是混合域概念，也属于私域营销体系的一部分。该部分用户与微信生态圈的逆向流动用户共同活跃于各大网络平台。</p>
<h3>8. O2O营销</h3>
<p>内容平台推广、私域营销与线下体验共同组成O2O营销体系。</p>
<p>理论上、领域专家、品牌代言人、关键意见领袖、关键意见消费者可以适用于所有私域营销体系的载体。但是从运营策略的角度看，领域专家适合各大网络平台，品牌代言人适合于微信生态圈，KOL和KOC更适合混合域的平台。</p>
<h2 id="toc-9">九、私域营销体系的关键问题</h2>
<h3>1. 组织结构</h3>
<p>公司的品牌部门和私域部门一定要分开运营，千万不要让私域运营成为品牌部门的下属。私域的重心在于打造品牌区域领袖，在一个点上发力。而品牌战略要全面布局，两个部门的运营策略有很大区别，无论谁领导谁都会出现重心偏差。</p>
<h3>2. 各区域流量区别</h3>
<ul>
<li>公域流量：具有曝光量，但不具备客户管理与运营权限。</li>
<li>混合域流量：受平台制约，对客户的管理与运营权限较小。</li>
<li>私域流量：对客户的管理与运营权限较大。</li>
</ul>
<h3>3. 从宏观理解私域</h3>
<p>无论是公域还是私域，只有好的产品才有好的流量。</p>
<p>私域运营能力在未来，会成为各家公司最核心的能力之一。</p>
<h2 id="toc-10">十、私域营销体系的战略布局</h2>
<h3>1. 不合理的战略布局</h3>
<ul>
<li>公司渴望消费用户。当建立一个私域时，大多数公司的目标是将公域流量引入私域流量，找到留住用户的方法，销售、复购，周而复始。如果你的私域营销体系是这种，可能会出现消费用户逐渐离开的现象，公司再想办法从公域导来流量，随着用户的不断流失，运营成本也会不断增加。</li>
<li>公司想尽办法捆住用户。各种活动、规则、附加条件的不合理设计，就是强迫用户留下来，用户无法互通互达，无法自由穿梭。最终，用户会被一个问题所困扰，选择你还是选择你的竞争对手？</li>
</ul>
<h3>2. 合理的战略布局</h3>
<ul>
<li>私域不是封闭的，是对外开放的。</li>
<li>私域是根据地，这里是汇聚用户的地方，不是捆住用户的地方。</li>
<li>私域不是营销的终点，而是营销的发起点，向外扩张才是核心问题。</li>
</ul>
