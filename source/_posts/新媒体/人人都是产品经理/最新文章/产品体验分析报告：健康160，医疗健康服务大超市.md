<p>看病难是我们国家从古至今的难题，随着科技互联网时代的发展，看病难已经逐渐易化。随着移动互联网的飞速发展，我们通过网络解决问题的效率越来越高。</p>
<p>看病也一样，各大健康类的网站、app、小程序、公众号的实现，人们再担心挂不到号，找不到医生，了解不到医院的最新消息等，这些问题都已不再是问题。</p>

<h2 id="toc-1">二、需求整理</h2>
<h3>1. 用户需求分析</h3>
<p>网上看病，大家都希望挂号更加公平和简单、问诊看病可以不用等。健康160以用户需求为出发点，解决了用户预约挂号、买药、各大健康类信息指导、医院评价、医生资质查询、线上问诊等多个核心需求。</p>
<p><strong>1）需求</strong></p>
<ul>
<li>各大医院入驻，就近看病更简单。</li>
<li>发现不舒服时，希望及时了解相关知识和别人的治疗对话，快速了解自己的病情。</li>
<li>线上一对一咨询专业的医生，快速了解自己的病情。</li>
<li>随时查阅医院的评分、等级、距离、医生的评价、等级，让大家放心看病。</li>
<li>洗牙、美容、矫正牙齿等等活动的宣传。</li>
<li>各大方面的专业知识。</li>
<li>特殊情况快速挂号</li>
</ul>
<p><strong>2）目标用户</strong></p>
<ul>
<li>有一定知识储备但是不专业的人。</li>
<li>快速挂号想看病的病人。</li>
<li>想寻找性价比最高、评价最好的医生看病。</li>
<li>注重日常护理比如洗牙、妇科、育儿、心理咨询等非实际疾病。</li>
<li>就诊后参与评价，为平台的建设添砖加瓦。</li>
</ul>

<h3>3. 产品功能结构图</h3>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/4GQL1fQ0muKgqnDoIQzH.jpg" alt width="650" height="1312" referrerpolicy="no-referrer"></p>
<h2 id="toc-1">三、主要功能体验</h2>
<h3>1.登录注册</h3>
<ul>
<li>登录方式：手机号+密码、手机号+验证码、Apple账号、移动通信、微信、QQ、微博</li>
<li>找回密码：手机号+验证码+新密码</li>
<li>注册：手机号+验证码</li>
<li>输入错误：弹出框提示</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/zevqLqxsmtMHk8wKktHX.png" alt width="804" height="342" referrerpolicy="no-referrer"></p>
<p>爽点：</p>
<ul>
<li>页面简洁，支持Apple账号登录，对苹果用户友好。</li>
<li>切换密码登录和验证码登录时，只是当前页面输入框的改变，不跳转新页面。</li>
<li>点击忘记密码、立即注册后，数字键盘立即弹出，方便操作。</li>
</ul>
<p>缺点：</p>
<ul>
<li>每个页面的顶部没有标明名称，比如登录页面、注册页面，容易给用户造成视觉阻碍。</li>
<li>弹出框后， 触碰其他区域会与隐藏弹出框的触碰区域冲突，比如点击眼睛显示/隐藏密码，容易被识别为隐藏弹出框区域。</li>
</ul>
<h3>2. 挂号</h3>
<p><strong>1）选择医院</strong></p>
<p>首页点击【挂号】，进入到挂号页面。页面主要分类项目、医院，医院列表会显示名称、图片、区域、医院等级、总评分、距离、特点标签。</p>
<ul>
<li>当未登录时：医院的排名按照综合排名。</li>
<li>当登陆成功后：排名按照最近去过的医院+综合评分的医院。</li>
</ul>
<p>用户可以点击需要看病的项目按钮，进入该项目的综合排名的医院页面。比如“皮肤科”，具体项目页面中，医院的排行则直接按照了综合评分进行排行。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/QMyrLNzJtuzWGuEP9V74.png" alt width="702" height="479" referrerpolicy="no-referrer"></p>
<p><strong>2）选择科室</strong></p>
<p>点击某个医院，进入该医院的挂号页面。医院挂号页面中，包含了该医院的热门科室、其他科室、每个科室的标签（如“市重点”）、在该医院提前放号的天数中都没有号的时候，不显示“挂”字。</p>
<p>说明：不论是从具体的项目按钮（比如皮肤科）点击进来，然后再点击医院；还是在挂号页面中直接点击的医院。进入医院的挂号页面相同，都是默认的挂号页面+排列中的第一项目（比如“热门科室“）。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/TZb6Tdm0fZsVdALppjdJ.png" alt width="704" height="489" referrerpolicy="no-referrer"></p>
<p><strong>3）选择医生</strong></p>
<p>点击项目（科室，比如“骨关节肿瘤科”），进入科室页面。页面包含排期、医生信息列表、医生评分、预约量、医生专业标签、是否有号标签等。</p>
<ul>
<li>科室挂号页面中，排期默认是全部；</li>
<li>排期中的日期显示内容：星期几+月/日+是否有号；</li>
<li>切换日期时，医生列表页面刷新。</li>
</ul>
<p>医生列表按照“咨询+有号”-“有号”-“咨询+加号”-“无”进行排列。</p>
<ul>
<li>当提前开放预约的日期段里有该医生的号时，显示：“有号”；</li>
<li>当该医生开通了咨询服务时，显示“咨询”；</li>
<li>当医生开通了加号服务时，有约满的情况下，显示“加号”；</li>
<li>当该医生在日期段内没有排期时，排在列表末尾，无标签显示。</li>
</ul>
<p><strong>4）选择日期</strong></p>
<p>点击医生，进入到医生的挂号页面。日期默认选中为科室页面切换的日期，且手机显示排在最左边首列。往左滑可以查看该日期之前的日期挂号，右滑查看该日期之后的挂号。</p>
<ul>
<li>未约满时，显示预约按钮；</li>
<li>无排期时，显示无；</li>
<li>约满时，显示约满。</li>
</ul>
<p>根据医院的工作时间，一般12:00-14:00为固定休息时间。所以健康160直接先分为了上午、下午进行选择，然后再选择具体的时间段，当上午的任一时间段有号时，显示“预约”按钮。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/EBbAly8NuOOXfZx7ZURb.png" alt width="699" height="405" referrerpolicy="no-referrer"></p>
<p><strong>5）选择时间段</strong></p>
<p>点击“预约”按钮，弹出时间段选择窗口。窗口中展示：具体日期、医院名称+科室、挂号费用、默认选中的时间段、剩余可挂号数。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/2G1Vjqvnpcnx55v126v0.png" alt width="500" height="540" referrerpolicy="no-referrer"></p>
<p><strong>6）挂号完成</strong></p>
<p>选择看病的时间段，点击【确认】，进入预约挂号提交页面。在提交页面，可以提前录入自己的病情说明。这里还多了个弹窗：获取健康码。疫情发生后，去医院都是纸质档填写信息，微信扫码录入显示绿码，体温正常方可通行。现在可以直接在挂号中录入，节省了患者很多时间。</p>
<p>获取健康码为可选功能，可以直接跳过，或者填写信息录入健康码。最终回到预约挂号确认页面，点击【完成】，挂号预约成功。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/BDhOrQvbe96bTtSoJJgV.png" alt width="803" height="346" referrerpolicy="no-referrer"></p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/kfXJzGCpMcyb1seUJoJA.png" alt width="307" height="793" referrerpolicy="no-referrer"></p>
<p>总结：医院的项目需要重点考虑到的是成年人和老年人，很多人不会操作app，所以重点是一个功能怎么简单易操作，关键信息在最显眼的地方并且多步操作都有显示。</p>
<p>爽点：</p>
<ul>
<li>医院与我的距离显示，不需要用户在通过其他软件查。</li>
<li>日期和时间段的选择清晰明了，趋向于傻瓜式操作。</li>
<li>医生挂号页面的刷新按钮，会让用户有一种公平的安全感。用户不需要疑惑是否要重新进入页面才能刷新。</li>
<li>申请加号（需要写明原因、最多选择2个时间段）。这对突发情况的患者非常友好，医生也能帮助到那些确实着急的患者。</li>
<li>获取健康码。去过医院的都知道，需要填写纸质档、扫码获取健康码、量体温，非常麻烦。</li>
<li>挂号看病时间还未到的时候，挂号信息会一直显示在主页。就像网上买菜后，卖家离你还有多远一样。这种感觉怎么说呢，如果说买菜是等待惊喜到来，看病是等待天使到来。</li>
<li>取消订单有次数限制</li>
</ul>
<p>缺点：</p>
<ul>
<li>挂号页面中，点击某个项目进入项目（科室）页面后，点击一个医院，进入医院的挂号页面。挂号页面并没有给用户自动选中或者定位到用户选择的这个项目（科室）。</li>
<li>科室选择页面中，诊前按钮不够明显。</li>
<li>获取健康码时间过早，不能保证就诊前患者仍然是这个状态。可以做到线上取号这一步的后面。</li>
<li>写病情，过于鸡肋。一般医生不会去看（是否是给客户使用的后台提示不明显）。用户也会担心隐私泄露。</li>
<li>预约挂号确认页面，信息过少。可以增加挂号医院、科室、日期、时间段的信息确认。</li>
</ul>
<h3>3. 在线问诊</h3>
<p>健康160中，在线问诊并没有专门的按钮显示。</p>
<p>但是在每个页面都可以找到与在线问诊相关的模块。比如底导航：问医生、首页：找名医、医院页面：咨询、医生主页：咨询、搜索结果页面：诊前咨询。通过底导航的问医生来分析在线问诊的功能。</p>
<p>在问医生首页中，首先是通过咨询的类型分为了三个模块：图文咨询、私人医生、免费咨询。附加了一些小的快捷按钮：极速咨询、我要开药、名医电话、报告解读、视频咨询（发现健康160的产品老喜欢把操作重复做，比如咨询类型选择后进入医院选择，医生选择后又进入咨询选择，挂号也是）。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/04/wU2rrSzE3g6a6ikl5lVz.png" alt width="704" height="707" referrerpolicy="no-referrer"></p>
<p><strong>1）免费咨询</strong></p>
<p>问医生首页中，点击【免费咨询】，选择咨询人-输入咨询的内容-添加图片（选填）-提交咨询-等待医生回复。等待回复页面点击返回，回到我的咨询页面。</p>
<p>免费咨询，所有医生都可以回复。患者最多可回复10条消息。得到回复效率高，但不能保证咨询的准确度，且问题将被公开。</p>
<p>以前有问题，大家都习惯去百度问。现在的人有问题，大家都知道去对应的专业平台查找或询问，在健康问医方面，健康160提供了患者和医生沟通的平台。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/04/GrxU7GRZjmsHdZDCB3z0.png" alt width="806" height="430" referrerpolicy="no-referrer"></p>
<p><strong>2）极速咨询</strong></p>
<p>问医生首页中，点击【极速咨询】，选择咨询人-选择咨询费用-输入咨询的内容-添加图片（选填）-提交咨询-等待医生回复。等待回复页面点击返回，回到我的咨询页面。</p>
<p><strong>3）问医生</strong></p>
<p>通过寻找针对性的医生，再来选择对应的咨询服务，还可以通过问医生付费快速加号。医生利用自己业余时间处理特殊情况能获得一定的报酬，是对职业和自己的尊重，患者也能在第一时间解决问题。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/04/AJG3osb5EZ4uVDgbBA5x.png" alt width="805" height="434" referrerpolicy="no-referrer"></p>
<p><strong>4）优点</strong></p>
<ul>
<li>问题的文字内容公开化，方便大家搜索、对比、更加了解自己的情况；</li>
<li>咨询类型多，针对不同需求的人群；</li>
<li>提问过程中，可以直接导入之前的就诊历史。</li>
</ul>
<p><strong>5）缺点</strong></p>
<ul>
<li>页面杂乱，类似的流程和按钮过多。比如 主页的图文咨询、私人医生、电话咨询、视频咨询可否直接省略。变为3大模块：免费咨询、极速咨询、找医生。</li>
<li>有弹出广告的页面多，有个小小的疑问：贵公司产品是不是很喜欢阿里的产品，比如支付宝、淘宝、盒马…..</li>
</ul>
<h3>4. 个人中心</h3>
<p>相比其他的主页，个人中心页面简洁，包含的功能非常全，每个功能按钮分工明确。用户的挂号就诊情况、咨询、加号、评价、收藏等。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/04/7b5LxEcq3sm1DjfyZaWu.png" alt width="801" height="343" referrerpolicy="no-referrer"></p>
<h2 id="toc-4">四、总结</h2>
<p>个人认为健康160是一个功能比较全面，使用流畅的app（资讯除外，资讯、文章、搜索问答详情全都是外链的H5）。目前广东入驻的医院很多，对于本地生活的人来说看病时很方便的，别的城市也有属于自己的医疗类app，市场的扩展需要时间。</p>
<p>用户来健康160更多的是寻找自己需求的解决方法，比如挂号、问诊。平常并没有来此app的欲望，但是从主页底部的资讯和视频、个人主业可以看出，公司是想做社交的。</p>
<p>app上并没有社交专门的模块、比如像小红书、keep一样，把资讯和视频做成瀑布流。用于在当前屏幕里可以看到最多的消息、同时选择自己喜欢的内容。</p>
<p>作为一名产品经理，我们时常要从用户的角度考虑一个功能为什么要这么做？还有更好的方法吗？与竞品对比优势在哪里？后续的扩展等等，但是在这个app里我好像没有感受到有特色的东西。</p>
<p>哎，可能级别不够吧，作为一名初级产品经理要学的还有太多太多。</p>
