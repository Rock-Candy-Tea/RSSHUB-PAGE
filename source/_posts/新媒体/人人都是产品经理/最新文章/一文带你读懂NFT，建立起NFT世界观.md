<h2 id="toc-1">一、什么是 NFT ？</h2>
<p>NFT 的全称是Non-Fungible Token。</p>
<p>Non：我们都知道就是不的意思。</p>
<p>Fungible：是指同质化代币，比如说人民币，你和我都有我们可以交换。</p>
<p>Token：翻成凭证会比较好解释，我们手里的纸币就是一种凭证，只是由不同的国家发行。比如英镑是英国的，人民币是中国的。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/OEVzmpNlGIvrsnNOD0sa.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p>中文全称为”非同质化代币”不可互换的意思。是用区块链技术给你的资产发放的唯一令牌，可以理解为虚拟世界里资产的所有权证书，它的的特点就在于“唯一”。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/zXpRbwjpbRV4dOWyb5kJ.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p>在应用中体现为两点：</p>
<ol>
<li><strong>鉴别真伪： </strong>NFT 现在主要应用在艺术加密领域，因为以前的数字艺术品很容易被复制，难以分辨真伪。现在有了 NFT 技术，可以通过区块链上唯一的证书来确定归属人是谁。</li>
<li><strong>更多收入：</strong>以往艺术品是一锤子买卖，现在通过 NFT 技术加持每一次转卖，作者都能够得到版税。因为每一次转卖的价格经手人都会被记录在公证证书上，所有人都可以看到。所以现在世界上各大艺术拍卖行也在推进 NFT 加密艺术品交易的完善。</li>
</ol>

<h2 id="toc-3">三、国内外的区别</h2>
<p>大家看到这先别激动，像这样赚钱发财的也只有在国外了，在国内的 NFT 环境中还是有许多的差异。因为在国内没有 NFT ，只有不可交易的数字藏品。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/uCP5Nb14FeT1WHvyzjho.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p>我把国内外差异通过表格进行了梳理，帮助大家直观的理解。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/OQeeoKreHE5eaYD0JdHx.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p><strong>最大的不同可以体现在这几个方面：</strong></p>
<ul>
<li><strong>发行：</strong>国内上的是私有链相对私有化，相对不透明不稳定。</li>
<li><strong>交易：</strong>国内只能通过使用人民币购买，不能在二级市场交易。</li>
</ul>
<p>所以在国内不能击鼓传花也没有任何资本属性，更不能成为资产。在国内想要从 NFT 里获利会比较困难。当然即便如此，仍然⽆法阻挡⼀些炒作。咱们来看看国内的 NFT 市场的情况。</p>

<h2 id="toc-5">五、如何通过 NFT 获益</h2>
<p>那么普通人该如何在 NFT 里获益呢？我们可以选择这两个方向：</p>
<ol>
<li><strong>当收藏者：</strong>一定要是自己熟悉的行业，才有鉴别艺术品的能力，比如说你是NBA的球迷，可以收藏NBA的球星卡，选择一个你看好的新秀然后收藏下来坐等起飞。</li>
<li><strong>当创作者：</strong>设计师可以踊跃地参与进来，制作音频或者图像，通过以下平台进行加密上传，然后坐等有缘人。</li>
</ol>

<p>目前这两个是 NFT 比较大的平台，通过简单的上传将图片摇身一变为 NFT 作品。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/gmp2dZMJfc0HcsDaihs3.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p>不过除了把作品变为 NFT 进行售卖外，还可以利用 NFT 来进行各种品牌营销，在国内已经有大量的品牌参与其中，比如江小白以 NFT 作为营销手段，推出江小白回蓝战士。为品牌方带来3倍的流量，创造了超出日常16倍的单日销售额。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/aG4sdjN3lqsrYZgJpjCC.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<h2 id="toc-6">六、NFT 未来趋势</h2>
<p>看到这里相信各位看官已经对 NFT 有了基本的世界观，在本文的最后我将 NFT 未来的一些趋势进行了梳理，希望大家对 NFT 能看得见的现在，望得见的未来。</p>
<p>在这之前，我们先了解一下 NFT 价值的三大支撑。</p>
<ol>
<li>故事性</li>
<li>收藏性</li>
<li>可用性</li>
</ol>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/YEvlXdTIjbcW8PxoEraG.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p>一件艺术品 NFT 可能具有独特的特征或名称，从而使其变得稀缺，但只要持有者无法使用它，它仍将是一件收藏品，所以 NFT 还需要故事和收藏/流通的价值。</p>
<ul>
<li><strong>王者荣耀：</strong>就为每个角色拍摄了短片，赋予人物故事。</li>
<li><strong>虎年邮票：</strong>由官方的背书，转售方便，具有很高的收藏价值。</li>
<li><strong>游戏皮肤：</strong>人物外观好看，同时可以展示给别人看，并且有属性上的加成。</li>
</ul>
<h3>1. 游戏 X NFT</h3>
<p>在传统游戏中里面的道具并不完全属于玩家。深究起来道具其实是被锁定在游戏中的。首先道具的属性、使用、数量和价格的决定权被发行商牢牢抓住。其次游戏中断或关闭，游戏内的道具也都会消失，最终拥有游戏及其内容知识产权的是开发者和发行商。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/eGCwGJk6mqf5mvzBIKPi.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p><strong>当游戏遇上 NFT 会发生什么？</strong></p>
<p>比如 NFT 知名游戏Axie Infinity，它的玩法是玩家可以购买类似神奇宝贝的怪物进行对战。</p>
<p>厂商将游戏内的所有权和管理权下放，所以任何决策和财产的积累会由玩家决定而不是开发者，它运用独特的商业系统打造了一个较为合理的虚拟国度。在越南和菲律宾，许多玩家真的辞掉了自己的工作，每天在AxieInfinity游戏中投入几个小时就能过上更好的生活。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/m30veGNGz0rA6lEJU7P4.gif" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p>甚至有一部专门的纪录片《PLAY-TO-EARN | NFT Gaming in the Philippines》，介绍“边玩边赚”游戏到底是如何改善菲律宾偏远地区人民生活的。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/Iab09xWHki1zz1y6Dzat.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<h3>2. 音乐 X NFT</h3>
<p>音乐 NFT 的发展十分迅速，不管是艺人工作室还是音乐平台都在纷纷入局。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/jzPh1tdNRyw1fM4OwTec.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p>音乐 NFT 看起来像是在网易、itunes上购买歌曲的替代品。但其实只是购买了听音乐权。只是一个许可罢了。相比之下 NFT 音乐可以让你拥有真正的所有权。从投资的角度看，如果购买的歌手成功了，他们投资的音乐 NFT 也会随之升值。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/FazIn5IEJEgsOq2lAwun.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<h3>3. 电影 X NFT</h3>
<p>在艺术和音乐行业进行了大规模革命之后，未来改变最大的可能是“电影业”。因为 NFT 可以在电影的发行、制作、销售去解决问题。</p>
<ul>
<li><strong>发行：</strong>每进行一次销售，都能拿到版税。</li>
<li><strong>制作：</strong>如《女性世界》将由 NFT 作品角色打造，将众多角色编写成一部电影，而在电影里拥有角色的发言权。</li>
<li><strong>销售：</strong>如《黑客帝国》发布了一些系列的IP进行宣传销售。</li>
</ul>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/fCz2nVou7Wh1J3g9Uo75.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p>在未来 NFT 作品和电影互相转化或是一个趋势。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/rcerbeozGSz6YIVSGWZB.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<h3>4. 时尚 X NFT</h3>
<p>时尚 NFT 是一种代币化的可穿戴物品，例如衣服、配饰和鞋子，可以在虚拟世界里展示使用，并且不用担心仿冒品的出现。例如Gucci Virtual 25可通过AR的形式在现实世界中实现穿搭。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/jLloSINBMFrOl6ZC6u8w.gif" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p>还有各大奢侈品牌都在涉水 NFT 。比如BURBERRY和DG Family都参与了进来</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/C829SdZxivyJulbE867l.png" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<p>Gucci就在去年举办了元宇宙时装周有40,000的人参与了进来，之后 NFT 也在时尚行业迅速流行开来。</p>
<p><img data-action="zoom" class="rich_pages wxw-img aligncenter" title="设计师的财富密码？一文带你读懂NFT。" src="https://image.yunyingpai.com/wp/2022/07/s01lU1Ms897OK2bKWk8h.gif" alt="设计师的财富密码？一文带你读懂NFT。" width="596" height="336" referrerpolicy="no-referrer"></p>
<h2 id="toc-7">七、结语</h2>
<p>在未来会有更多虚拟场景+ NFT 的组合，各圈层还有格局未打开，在虚拟经济的带领下，会有更多变化发生！当然机会和风险肯定是并存的，在NFT 世界坑也很多，比如正在流行的碎片化 NFT ，通过将艺术品分为多份降低其售价，金融玩法非常重。</p>
<p>在这圈子里流量是唯一的衡量标准。比如：</p>
<ol>
<li>朋克9998有可能是左右手倒腾</li>
<li>买beeple作品的人，是为了炒作，因为其本身是加密货币的发行者。</li>
</ol>
<p>最后希望能让你建立起 NFT 世界观，因为篇幅的限制本期很多没有细说，比如 NFT 作品特点等，后期咱再来填坑吧。</p>
