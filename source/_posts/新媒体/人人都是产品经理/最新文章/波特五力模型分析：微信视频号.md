<h2 id="toc-1">一、波特五力模型介绍</h2>
<h3>1. 定义</h3>
<p>波特五力模型是迈克尔·波特（Michael Porter）于20世纪70年代初提出。用于分析行业竞争态势的工具，可以有效的分析企业的竞争环境，对企业战略制定产生全球性的深远影响。</p>
<p>具体应用场景如下：</p>
<ol>
<li>新市场评估：企业进入一个新市场应该采用何种竞争战略，即市场开发；</li>
<li>竞争策略制定：企业为了提高收入或者利润，希望能够采用新的竞争战略去打败竞争对手，即提升盈利。</li>
<li>行业进行分析：波特五力模型不是针对某个产品进行分析，具体对产品分析就是研究策略。</li>
</ol>
<p>波特五力模型表示行业中存在着决定竞争规模和程度的五种力量，这五种力量综合起来影响着产业的吸引力以及现企业的竞争战略决策。</p>
<p>五种力量分别为：</p>
<ul>
<li>（上游）供应商的讨价还价能力；</li>
<li>（下游）购买者的讨价还价能力；</li>
<li>潜在进入者威胁；</li>
<li>替代品的威胁；</li>
<li>同行业竞争者竞争程度。</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/lVBnieE1ua5NqJst9NCU.png" alt width="483" height="278" referrerpolicy="no-referrer"></p>
<h3>2. （上游）供应商的讨价还价能力</h3>
<p>定义：卖方/供方（suppliers）是向企业提供生产所需的产品或服务的议价能力，所需商品包括燃料、原材料装备、劳动力以及财务资源；那些能够增强卖方议价能力的因素，相反也减少了买方的议价能力。</p>
<p>一般来说，满足如下条件的上游供应商会具有比较强大的讨价还价力量：</p>
<ul>
<li>供应商集中度高：在少数几家供应商主导市场的行业中，供应商议价能力比买方更高，垄断企业讨价还价能力很强。</li>
<li>转换成本高、依赖程度高：买方从一家供应商转移至另一家付出的代价或者遭受的损失很大 或者 很难找到相应的竞争替代品。</li>
<li>技术门槛、技术专利：供应商在某一个领域具备技术的优越性，并且占据领域较多的核心技术专利；</li>
<li>供应商的竞争威胁：供方能够方便地实行一体化（产业链纵向延伸的能力），而买方是中间交易人时，供应商能够切断其业务，因此其议价能力也更高。</li>
</ul>
<h3>3. 购买者的讨价还价能力</h3>
<p>定义：购买者（buyers）是指企业的直接客户，而不一定是最终客户购买商品的议价能力。如果购买者有足够的议价能力，他们就能开出更低的价格，或坚持要求产品或服务做出昂贵的改进。</p>
<p>一般来说，满足如下条件的买方会具有比较强大的讨价还价力量：</p>
<ul>
<li>购买者集中度高：少数大客户占全部销售额较高的行业中，买方议价能力更强。买方数量少，并且采购量大；</li>
<li>转换成本低：家如果能轻易地从一家供应商转移至另一家供应商，那么，他们就拥有很强的谈判能力。</li>
<li>购买者的竞争威胁：如果购买者有能力给自己提供原材料，或者可能获得这种能力，则它的议价能力就高。与供应商进行谈判时，购买者如果从事卖方的业务则会增加对卖方的威胁，这叫做后向一体化，即整合上游的供应商。</li>
</ul>
<h3>4. 潜在进入者威胁</h3>
<p>定义：潜在的进入者是还没涉猎到该行业，他已经在朝这个方向发展或者可能进入该行业索带来的威胁；进入某一行业的难易程度会影响竞争的程度。</p>
<p>进入者威胁越大，对行业中的现有的竞争者越不利；一个有吸引力的行业，一般具有较高的进入壁垒， 以降低新竞争者的威胁。</p>
<p>—般来讲，进入壁垒有如下几种：</p>
<ul>
<li>生产规模：一旦现有从业者具备大规模生产能力，潜在进入者要跟上他们的代价就很高。如研发成本极高的医药行业。</li>
<li>经验：现有从业者已经积累了丰富的生产经验和管理运营经验，随着经验的提高，成本会降低；</li>
<li>人才：现有从业者积累了人才资源，潜在进入者需要更多的成本搭建起团队；</li>
<li>获得供应或者分销渠道：许多行业的生产商都控制着供应和（或）分销渠道，可以理解为代理权。</li>
<li>阻力：现有企业会为了阻止潜在进入者而采取极大的阻力，可以采用价格战或营销闪电战的形式。</li>
<li>合法性或者政府行为：法律对于潜在进入者的限制表现在各个方面，从专利保护 (如医药业），到市场监管（如退休金出售），再到直接的政府行为（如关税）；</li>
<li>差异化：差异化意味着比竞争对手提供更具有感知价值的产品或服务，差异化也是竞争优势的重要来源。</li>
</ul>
<h3>5. 替代者的威胁</h3>
<p>定义：替代品的威胁是分析自己的产品或者服务，找到是否有本行业之外的替代品，评估替代品造成的威胁；</p>
<p>替代品（subsititutes）是通过不同的流程提供与行业内产品和服务价值近似的产品和服务；例如，在汽车行业中，铝就是钢铁的替代品；火车就是汽车的替代品。</p>
<p>关于替代品，需要牢记以下两个重点：</p>
<ul>
<li>性价比对于替代品威胁特别关键：替代品即使价格更高，只要能提供消费者看重的性能优势。</li>
<li>行业外部效应是替代品概念的核心：很多时候替代品来自于既有从业者所在行业之外，因此不能将其混同于行业内竞争者的威胁；替代品概念的价值在于使管理者关注行业之外，并考虑更远期的威胁和限制。</li>
</ul>
<h3>6. 同行业竞争者的威胁</h3>
<p>竞争对手（与替代者不同）是指面向相同的消费者群体提供类似产品和服务的企业。例如 QQ、微信，陌陌等社交产品就是竞争对手，而抖音短视频则是替代品。</p>
<p>竞争者的威胁程度取决于一下几点：</p>
<ul>
<li>竞争者规模相当：当竞争者规模相当时，就会产生激烈的竞争行为，任何一家都想通过压低价格等方式超过竞争对手</li>
<li>竞争者市场占有率：竞争者市场占有率决定竞争者的威胁程度，占有率越高，需要更多的成本去抢占市场；</li>
<li>差异化：在商品市场中，如果产品或服务几乎没有什么差异，竞争就会加剧， 因为消费者能够轻易地在不同竞争企业之间进行转换，唯一的竞争因素就是价格。</li>
<li>用户心智模型：竞争者产品或品牌在消费者中形成心智模型之后，想去改变用户的使用习惯会更难，威胁也越大；</li>
</ul>
<h2 id="toc-2">二、微信视频号分析</h2>
<h3>1. 行业格局</h3>
<p>据艾瑞咨询《2020年中国资讯短视频市场洞察白皮书》，广告主求稳、平台方和内容方求大、MCN和营销商求生；从现阶段来看，短视频营销产业链及合作模式还在不断探索和变化之中，因此对产业各方来说，仍然是机会与威胁并存。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/HtM44bPmeFFQg6uktu04.png" alt width="1341" height="759" referrerpolicy="no-referrer"></p>
<p>短视频行业已步入稳定发展的成熟期，由原来的蓝海市场进入到红海市场，竞争格局保持相对稳定，各头部互联网公司都对短视频进行布局，短视频平台多，竞争激烈。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/yCG0NrhUnJIdtsjORLPU.png" alt width="683" height="284" referrerpolicy="no-referrer"></p>
<h3>2. 波特五力模型分析</h3>
<p><strong>1）供应商的议价能力 </strong></p>
<p>短视频行业产业链中，上游供应商核心是内容生产端，短视频创作者有两种分层方式：</p>
<ul>
<li>头部内容创作者：由优质的KOL组成，明星以及 MCN机构孵化；</li>
<li>中长尾内容创作者：多由草根或素人组成；</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/NQCtIsbcJxxgvrugndJM.png" alt width="1544" height="582" referrerpolicy="no-referrer"></p>
<p><strong>2）购买者的议价能力</strong></p>
<p>微信视频号的购买者主要为，根据艾瑞咨询《2020年中国资讯短视频市场洞察白皮书》，得出以下的短视频用户调研信息数据：</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/MosduiadQPqsDFinw41u.png" alt width="676" height="253" referrerpolicy="no-referrer"></p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/BoomjYYRYNYwDMUIsJXb.png" alt width="1615" height="441" referrerpolicy="no-referrer"></p>
<ul>
<li>根据上述的分析，用户群体集中在青少年以及25-35岁的人群，点赞和收藏资讯短视频还是最为常用的功能，在调研中，近7成的用户会使用微信生态分享给好友和朋友圈；</li>
<li>用户关注内容领域top3为时事新闻占到（70.4%）、生活健康（60.8%）、影视娱乐（55.6%） 用户使用短视频的主要因素是 内容获取快，并且提供资讯内容专业，权威；</li>
<li>据艾瑞咨询报告，抖音，快手等娱乐社交属性较强的短视频用户量最大，占到百分之36%，而微博、微信等社交平台占到10.7%，用户可选择的平台比较多</li>
</ul>
<p>对应策略：</p>
<ul>
<li>根据常用的用户群体，引入更多专业优质的内容资源；</li>
<li>内容创造者上可以针对 top5领域，设置对应的奖励任务吸引更多内容创造者创作；</li>
</ul>
<p>结论：购买者的议价能力-高。</p>
<p><strong>3）替代品议价能力</strong></p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/zlT6EeyOeTySvoBZHyot.png" alt width="1863" height="510" referrerpolicy="no-referrer"></p>
<p>短视频使用场景分析：</p>
<ul>
<li>使用产品场景：观看资讯短视频无需用户保持长时间、持续的注意力，符合当前碎片化的阅读习惯，且视频和文字结合的方式有利于用户快速吸收资讯信息；因此，在除了工作、学习、健身以外的大部分空闲时间用户均习惯于观看资讯短视频；</li>
<li>使用时间：短视频的时间分布在全天各个时间段和场景中，在三个时间段较为集中∶早上6-8点，中午12-2点以及傍晚 8-10点。</li>
</ul>
<p>短视频查看因素：</p>
<ul>
<li>用户观看资讯短视频的目的性较强，最主要因素是为了跟踪某个热门话题相关资讯内容。</li>
<li>其他主要是短视频获取信息效率更高，用户注特定专业拍客拍摄的资讯短视频内容以及进行回复评论，社交互动。</li>
</ul>
<p>针对用户的使用场景，对于短视频来说，替代品可以分为以下几种：</p>
<ul>
<li>长视频：电视剧，电影以及综艺节目；</li>
<li>游戏音乐；</li>
<li>社交及资讯应用；</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/qiKPtXjHPO5WwinDYAQ0.png" alt width="697" height="272" referrerpolicy="no-referrer"></p>
<p><strong>4）潜在进入者威胁</strong></p>
<p>潜在进入者有以下：</p>
<ul>
<li>中大型的互联网公司（威胁程度：中）：美团，携程微博，知乎，拼多多等，有流量有资源，可借助自己的优势进入短视频行业；</li>
<li>创业团队或公司（威胁程度：低）：想进入短视频行业进行创业；</li>
<li>线下制造或零售企业（威胁程度低）：华为，中兴等公司；</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/Uubk5AAngmUCssPgeVZt.png" alt width="1541" height="609" referrerpolicy="no-referrer"></p>
<p><strong>5）同行竞争者威胁</strong></p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/coS8MDGrDMIvEU4qAUB1.png" alt width="1314" height="370" referrerpolicy="no-referrer"></p>
<p>据Mob研究院，抖音、快手双雄争霸，两者用户高达60%。西瓜视频、微视等腰部平台仍在追赶，竞争激烈，一众尾部平台面临着生存考验。</p>

<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/03/zlcap1fhWWVESDKD7ET4.png" alt width="1951" height="873" referrerpolicy="no-referrer"></p>
<h3>4. 总结</h3>
<p>经过对市场的调研分析，以及微信视频号竞争态势和优劣势分析，得出腾讯视频号在短视频行业的几个结论点：</p>
<ol>
<li>红海市场，前期仍可期：短视频行业的特点为成熟期，规范化，高增长，白热化，大众化，未来发展趋势依然可期，市场仍可抢占份额；</li>
<li>差异化策略：通过波特无力分析，供应商议价能力-中等，购买者议价能力-高，替代者威胁度-中，潜在进入者威胁度-低，同行竞争威胁度-高，所处于的行业竞争激烈，微信视频号的战略策略要建立起自己的差异化以及目标用户群体，利用好优势流量，优化内容和算法技术，保证优质内容创造者能在视频号中获得对应的报酬，享受前期红利。</li>
<li>先内容后引流：优势在于流量，劣势在于内容，没有内容会重蹈覆辙，流量进来，内容不好，流量流失的问题。让优质内容创造者能在视频号中获得对应的报酬；</li>
</ol>
<p>本次基于个人对短视频行业的了解做了简要分析，如有不当之处，请各位指出。</p>
