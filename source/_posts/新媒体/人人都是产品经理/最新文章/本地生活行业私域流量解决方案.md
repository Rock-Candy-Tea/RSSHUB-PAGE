<blockquote><p>编辑导语：关于私域，每个人都有自己的见解。本文主要探讨本地生活行业的私域运作方法，公域流量的褪去，迎来了私域增长。但私域流量也终将会有遇到增长乏力的时候，面对这种情况，该如何解决？一同来与作者一起探讨这方面的解决方案吧。</p></blockquote>
<p><img data-action="zoom" class="size-full wp-image-5183999 aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/10/PLC0DiSOKDakdkx6rpcu.jpg" alt width="900" height="420" referrerpolicy="no-referrer"></p>
<p>为了节省大家的时间，先对本方案的适用对象进行说明。</p>
<ul>
<li>本方案适用对象为本地生活类目商家；</li>
<li>本方案不同于美团、联联周边游等销售渠道类解决方案；</li>
<li>本方案旨在帮助本地服务类商家建立一套长期可持续发展的私域电商解决方案，着眼于眼前利益者，欲一招制敌者、想以此快速回笼资金者，均不在本服务行列；</li>
<li>自身业务做不好，产品满意度低，店内环境脏乱差、服务跟不上，复购率低的商家不在本服务行列。</li>
</ul>
<p>关于私域，每个人都有自己的理解，为了方便探讨，结合数赞这几年的私域运作经验，我们先来对私域达成一个共识。</p>
<p>谈到私域，不得不提到公域，中国近20年互联网的发展，主要是以平台经济模型为主，人们在平台流量红利的背景下，熟悉的是在平台规则上的流量玩法，当时，流量不分公域、私域、统称流量。</p>
<p>伴随流量红利的消失，公域平台受制于先天模式的影响，排名、竞价、流量分发这些都是平台必然的盈利模式，在这种情况下，商家的困境就出现了，流量获取成本蹭蹭上涨，企业要维持自身的增长，要么获得新用户，要么发挥老用户的价值，大家不约而同的选择了老用户这条线，这个时候私域流量就出现了。</p>
<p>用三条最简单的判断标准来衡量什么是私域流量。</p>
<ol>
<li>可随时触达</li>
<li>可多次触达</li>
<li>有基本共识</li>
</ol>
<p>以此判断，微信公众号、服务号、微博、抖音、小红书等平台上的粉丝都不算做是私域流量，或者称他们为广义上的私域流量，真正符合私域流量的是个人微信号和微信群。</p>
<h2 id="toc-1">一、为什么要做私域？</h2>
<p>两年前我们在做客户的时候，讨论最多的是这个问题，现在我们讨论最多的是如何做好私域的问题。但是为了方案的完整性，我们还是要最这个问题做简要介绍。</p>
<ol>
<li>经营转向：从渠道、产品、服务转向用户。</li>
<li>掌握主动权：用户获取成本越来越贵，平台成本越来越高，自己的用户却没有用好。</li>
<li>疫情刺激</li>
</ol>
<h2 id="toc-2">二、如何做好私域？</h2>
<p>为了让大家更快速、更高效的理解，我会结合真实案例进行说明。</p>
<p>案例介绍：</p>
<p>沐青汤泉亳州店，于2019年10月15日开始试营业，一个新品牌刚到一个新城市，效果自然不会太好。10月底，数赞与沐青达成合作。从11月1日开始到30日，亳州沐青汤泉生活达人社区成为了一个拥有接近30000用户（23000微信群用于+7000公众号用户）的大社区。</p>
<p>同时，从各群选拨了50位群管，维护50个官方社群，通过50位群管裂变出了1200多位推荐达人（团长），活动一经推出，沐青官方社群以及1250位推荐达人的朋友圈第一时间会爆发出来。从200张门票不到1分钟抢光到500张票不到2分钟抢完，再到4000张门票4个小时抢光等，这套体系正在发挥越来越大的威力。</p>
<p>一个月从0到1，从用户拉新，到留存促活，再到转化，这个案例很有代表性。我们知道这套体系的前提，必须是商家对主体业务的精准把握，主营产品一定要够硬，这个是本质，离开了它，所有的努力都会白费。沐青汤泉恰巧这点做的非常好。</p>
<p>下面我从三个方面来讲：拉新、治理，运营。</p>
<p>私域运营第一步拉新，主要解决以下三个问题。</p>
<h3>1. 用户从哪里来？</h3>
<ul>
<li>公域：美团、抖音、小红书、地方优选平台、大V等</li>
<li>门店：员工、进店流量、公众号、门店周边商家等</li>
<li>裂变：通过1、2步导入种子用户，设置福利快速裂变</li>
</ul>
<h3>2. 用户往哪里去？</h3>
<ul>
<li>企微群/普通群</li>
<li>企微个人号/普通微信号</li>
</ul>
<h3>3. 用户为什么要进入你的私域？</h3>
<ul>
<li>社群价值</li>
<li>裂变福利</li>
</ul>
<p>我们以微信群拉新举例，先确定自己的社群主题，主题要结合自己的品牌，同时要提倡一种生活态度，这样不落俗套，用户觉得跟一般微信群不一样。</p>
<p><strong>（1）社群价值与裂变福利</strong></p>
<p>举例：</p>
<p>社群名称：亳州沐青汤泉生活达人社区</p>
<p>社群公告：欢迎来到亳州沐青生活达人社区！热爱生活，传递价值。</p>
<p>在这里，你可以享受以下权利：</p>
<ul>
<li>第一时间获取优惠活动信息</li>
<li>会员专属商品和定期商家福利</li>
<li>代理优质活动，赚高额佣金</li>
<li>城市生活达人养成记（美食、美景、美文、美人等）</li>
</ul>
<p>每周五为广告日，合法广告都可以打，其余时间不允许！ 请珍惜入群机会！</p>
<p>其次确定进群福利，在这个问题上，商家应该思考一下。现在进群的用户将成为你可以重复使用的资源，你不用每次花费大量广告费去重复获取的用户。所以要确定好进群福利，制定只有群内用户才可以享受的政策，不在群里的坚决不能享受，并且不断强化，大家才会觉得进群是有价值的。</p>
<p>举例：</p>
<p>沐青汤泉群内福利：</p>
<ul>
<li>门票到店价58元，美团价格49元，群内用户38元。</li>
<li>群内用户到店即送精品美味爆米花一份。</li>
</ul>
<p>设置裂变福利政策，根据我们对沐青汤泉业务的分析，发现它对用户年龄、收入、工作、性别等要求没有那么严格，所以对应商家的用户画像范围就很大，简单来说就是，群内用户的数量跟销售量是成正相关的，所以我们就急需扩大用户数量，设置一个好的裂变政策，会起到事半功倍的效果。</p>
<p>举例：</p>
<p>沐青汤泉群内福利：</p>
<ul>
<li>邀请好友入群送门票，邀请5位好友入群送1张票，邀请10位好友入群送2张票，以此类推。</li>
<li>门票到店价58元，美团价格49元，群内用户38元。</li>
<li>群内用户到店即送精品美味爆米花一份。</li>
</ul>
<p>在拉新环节，各商户因自身情况不同，需制定不同政策。对用户要求不严格的可以采用沐青这种粗犷式发展思路，具有明显行业个性的商家，需要采用精细化的运营思路。</p>
<p>关于治理，传统社群治理要么采用一人专职管理所有群或者每位员工管理一个群的方式进行日常运营，一人管理多群的问题在于精力有限，只能采用粗放的经营模式，就像机器人一样。员工管理虽然可以从专业的角度解决社群用户提出的问题，但是会影响员工的本职工作。</p>
<p><strong>现在，我们换一种方式来治理。从群里选拨KOC（关键意见消费者）来服务群，只要分三步：招募、赋能，淘汰。</strong></p>
<p>举例：</p>
<p>截止目前，亳州沐清汤泉达人社群人数已经超过20000位，未来还会越来越多！</p>
<p>为了给大家提供一个更好的群内环境，现在我们准备在群里征集群管理（群管）协助负责群内管理。</p>
<p>群管主要职责：</p>
<ul>
<li>欢迎每一个进群的成员</li>
<li>移除不符合群规的成员</li>
<li>回答成员提出的问题</li>
<li>维护群内健康良好环境</li>
</ul>
<p>同时我们会为群管提供相应的权利和服务。</p>
<p>这个时候，你需要进行简单选拨，选拨三要素。</p>
<ol>
<li>职业，宝妈为主。</li>
<li>经验，做过分享达人</li>
<li>时间，闲暇时间多</li>
</ol>
<p>这个选拨不用太严格，因为后期还会有一系列的淘汰机制。</p>
<p>在报名的时候，不需要告诉群管的福利，因为那样会有很多人是冲着福利来的，往往是做不好的。但在选拨的时候就要告诉大家管理员的福利了。</p>
<p>举例：</p>
<p>沐青汤泉群管理福利：</p>
<ul>
<li>洗澡免门票</li>
<li>管辖群内票务销售赚佣金</li>
<li>发展销售团队赚更多佣金</li>
</ul>
<p>对于群管的更新淘汰主要从两个方面，第一是在群里起到的管理作用，是否能有效履行管理员的职责，这个考核者是商家。但是商家在对管理员进行考核的时候，一定要明白，不能为了更换管理员而更换管理员，要提前讲清楚规则，在什么样的情况下可以更换管理员，如果随意更换管理员，将会对其他管理员造成影响，起到不好的作用。</p>
<p>举例：</p>
<p>沐青管理员行为规范：</p>
<ul>
<li>两天内不在群里发言</li>
<li>多次对群内用户提出的问题不管不问</li>
<li>在群里发同行业（竞品）活动信息</li>
</ul>
<p>出现以上行为，商家可以解除管理员权限，请大家了解！</p>
<p>除此之外，我们把管理员行为转化为积分，在管理员群内，按照积分进行淘汰，这个属于规则淘汰，对所有管理员一视同仁，大家不会有意见。</p>
<p>举例：</p>
<p>班级名称：亳州沐青汤泉生活达人管理员</p>
<p>班级目标：每人日赚200元</p>
<p>班级制度：逢3末位淘汰制（每月3、13、23、30号）</p>
<p>管理单位：积分榜第1-5名组成管理员委员会</p>
<p>群内事宜，全权由管委会处理，组长为第一执行人。</p>
<p>根据规则，管理员群将会出现一种你追我赶的情形，而不是通过人为的管理。</p>
<p>现在私域流量有了一定规模，治理团队搭建完成，我们就可以进入运营环节了，我们主要围绕私域的价值点进行运营。</p>
<p><strong>（2）围绕社群价值点运营</strong></p>
<p>① 以私域促公域</p>
<p>拉新环节我们主要借助了公域的流量，现在我们就要用私域的力量反哺公域了。假如我们的点评评分是3.2分，我们可以借助社群的力量，快速让评分达到5分。</p>
<p>在社群内发布任务，用户发布九宫格+120字好评可以赠送两张门票。同样抖音、小红书等都可以采用这种方式，虽然目前私域是企业最快的增长渠道，但是占的整体份额还较低，公域仍然是企业的重要阵地。</p>
<p>② 倾听用户的声音，解决用户的问题</p>
<p>私域拉近了用户和商家之间的距离，用户对产品服务最敏感，能关注到商家关注不到的细微地方，不要担心负面的信息，解决负面信息的最好方式就是勇敢面对，用户喜欢这样的参与感。商家可以根据用户的反馈，快速调整优化自己的策略。</p>
<p>③ 提高私域销售占比，掌握定价权</p>
<p>丰富私域商城产品线，满足用户的多样需求；不定期举办新颖的营销活动，以好玩儿为主；不断强化私域的价值，培养用户在私域购买的习惯；长此以往，我们就可以避免对价格、对平台的依赖，不打价格战。</p>
<p>④ 扩大企业边界</p>
<p>自媒体时代，人人都是广告主。在企业里，保安可以做销售，会计、行政、保洁、厨师等都可以做销售。在企业外，用户也可以作为企业的销售员，除了销售，还可以做群管、做文案、做设计、做营销、做策划等。</p>
<p>举例：</p>
<p>沐青汤泉常规成人票38元/张，佣金8块，两周销售量为7000张</p>
<p>儿童票19.9元/张，佣金5块，两周销售3000张</p>
<p>如何让员工、让用户成为我们的推广员？我们要为每个活动设置一个营销佣金，这个很重要，因为你需要靠成百上千的团长帮你把活动推广出去，这里设置的营销佣金相当于是广告费了。</p>
<p><strong>以上，对经营三部曲“拉新”“治理”“运营”的大体分析就结束了。看完以后，大家会觉得并没有太多新意，这种感觉是真实的。每一个环节都不是我们自行创造的的，都是市场上有过的行为，我们只是从众多行为中抽丝剥茧，找出重点，进行重新组合，演变出一套完整的体系。</strong></p>
<p>另外，商家在执行这套方案的过程中，不会与商家目前从事的行为产生冲突，无论是在线下推广、线上推广、增加销售渠道等多种实践上。</p>
<p>只是建议商家不论采用哪种方式，都要把落脚点放在私域流量的建设上，凡是和商家产生过链接的用户，都要尽量邀请到社群中，不要堕入一种一边获取新用户，一边流失老用户的糟糕循环里。</p>
<p>同时采用社交电商的方式，让社群用户可以享受到社群用户的福利，让管理员享受管理员的福利，让帮你传播活动信息的用户享受参与分成的权利，让参与到每个环节的用户都能得到与之相匹配的权利。</p>
<p><strong>综上，目的就是建立一个以商家为主体的可持续增长的用户运营体系。此为数赞私域流量解决方案1.0版，希望商家基于这个版本结合自己的实际情况打造自己的2.0版。谢谢！</strong></p>
