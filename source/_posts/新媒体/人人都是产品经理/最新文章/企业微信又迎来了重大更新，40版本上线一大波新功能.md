<p><strong>2022年1月10日，企业微信又迎来了重大更新~</strong></p>
<p><strong>全新4.0版本来啦！</strong></p>
<p>企业微信4.0版本更新啦，这也是企业微信2022年第一个版本，此次更新后，企业微信“连接”能力达到新高度，连接微信、线上线下，服务连接升级！此外，在内外部协作、上下游协作方面均有超大更新！</p>
<p>下面粮仓来给大家详细介绍，此次企业微信4.0版本的新功能。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/01/Rv0uMq8JJbzlQL4xqBqR.png" alt width="1413" height="670" referrerpolicy="no-referrer"></p>
<h2 id="toc-1">01连接微信升级，线上、线下全面连接客户</h2>
<h3>1）微信客服能力升级，支持全场景接入</h3>
<p>在本次更新中，微信客服能力再次升级，<strong>提供给企业的一种全新的、连接客户的能力</strong>，在【微信外】也可接入微信客服了！此外，也可在已有的客服系统中接入微信客服。</p>
<p>登录企业微信电脑端，点击应用管理-微信客服，即可进行全场景的接入。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/01/HqlkOY1peeFDdEiwi7MY.png" alt width="1364" height="773" referrerpolicy="no-referrer"></p>
<p>① 企业可在微信内的视频号、公众号、小程序、微信搜索、微信支付凭证，以及微信外的App、网页等<strong>多个入口，接入微信客服</strong>。</p>
<p>② 客户无需加好友即可在微信里和客服沟通，和微信聊天有一致的体验。客服回复后，客户将在微信里收到新消息提醒。</p>
<p>③ 可邀请客户添加成员的企业微信或加入客户群，升级为专属服务。</p>
<p>粮仓近期也即将上线微信客服功能，可根据 企业实际业务场景，自定义客服接待方式。可针对不同场景生成渠道链接，根据客户咨询场景自动打上渠道标签。等等一系列的微信客服拓展功能。扫码文末二维码获取内测权限吧。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/01/g9emEZP8n1eSHayaIBv7.png" alt width="1708" height="1820" referrerpolicy="no-referrer"></p>
<h3>2）客户联系升级，朋友圈运营中台开启企微朋友圈运营新篇章</h3>
<p>本次更新后，企业微信接口更加开放，支持通过接口发布内容到客户朋友圈，这就意味着，<strong>企业的【朋友圈运营中台】</strong>，有了更大空间。</p>
<p>以粮仓·企微管家中的朋友圈任务功能为例，<strong>管理员编辑好朋友圈内容，给企业全员推送</strong>。企微成员仅限确认，即可将朋友圈高效的发表出去。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/01/lAevuqkyWYSxkosAJsLg.png" alt width="1280" height="720" referrerpolicy="no-referrer"></p>
<h2 id="toc-2">02 多款效率工具升级，内外协作更高效</h2>
<p>本次年度大更新后，企业微信电脑端、手机端一级菜单入口发生大改动。在电脑端，<strong>邮件、文档、会议三大功能模块成为一级菜单</strong>，再次<strong>强化了企业微信内部协作能力</strong>。</p>
<p>会议延续了腾讯会议高清、稳定、流畅的开会体验，并集成屏幕共享、电子白板、主持会控、会议录制及纪要等功能。特别值得注意的是，会议录制及纪要功能。</p>
<p>主持人可开启会议录制，录制完成后自动生成会议纪要</p>
<p>会议纪要支持搜索关键词，快速定位会议内容</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/01/SJR7fpTFwDoZ6T5D2rgK.png" alt width="500" height="356" referrerpolicy="no-referrer"></p>
<p><strong>全新【文档】功能</strong>，便于企业内部的文档协作、工作分派。可直接在文档中艾特企业成员，可进行更加便捷的文档权限管理、协作。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/01/0hr964PrCGO9lBvAM6oT.png" alt width="980" height="570" referrerpolicy="no-referrer"></p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/01/NcrSSghLi9bcnQYcljP6.png" alt width="1081" height="812" referrerpolicy="no-referrer"></p>
<p>此外，<strong>会议、邮件功能，也均有更多升级</strong>。其中，会议功能，支持开启会议录制，录制完成后自动生成会议纪要，方便回看会议和分享。邮件功能， 支持对收发件人快捷发起群聊，讨论邮件内容。</p>
<h2 id="toc-3">03 连接上下游，与经销商和供应商高效协作</h2>
<p>此次功能更新后，<strong>企业微信的【上下游功能】进一步升级</strong>。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/01/mJhiHcvm1CSFY37aiDqI.png" alt width="1678" height="1802" referrerpolicy="no-referrer"></p>
<p><strong>1）上下游管理</strong>支持迁移群、二维码、导入名单等多种方式邀请企业加入，高效连接上下游。</p>
<p><strong>2）上下游沟通</strong>使用上下游通讯录直接联系，跨企业找人跟找同事一样方便。</p>
<p>发消息可知对方已读未读，更有<strong>超大文件、会话存档、2000人大群</strong>等能力助力沟通。</p>
<p>可以群发产品动态等消息给上下游联系人，批量发送，即刻触达。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/01/rIIyK2Nh5GUls5xirZpc.png" alt width="1359" height="816" referrerpolicy="no-referrer"></p>
<p><strong>3）上下游协作</strong>会议、文档、日程、微盘等协作工具全面支持上下游，内外协作更高效。</p>
<p>除了企业微信的更新外，近期粮仓·企微管家也更新了不少功能。【智能表单】【自动打标签】【个人名片】等等一些列的功能，助力大家使用好企微，做好私域流量运营。</p>
<p>扫码下方二维码，联系粮仓客户经理，即可免费领取文中提到的【朋友圈任务】【微信客服】【智能表单】【自动打标签】【个人名片】等私域营销小工具。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/01/cJgrcjs2mi7GDNVK5yE9.png" alt width="820" height="580" referrerpolicy="no-referrer"></p>
<h2 id="toc-4">最后的最后</h2>
<p>此次企业微信更新的全新4.0版本，<strong>在连接微信、内外协作、上下游沟通3大方面，均有“重量级”更新</strong>。企业可以更加便捷地使用企业微信，开展外部客户服务与内部协作管理。</p>
<p>下面一起来看看此次企微官方公布的更新明细吧：</p>
<blockquote><p><strong>一、连接微信，线上、线下全面连接客户</strong></p>
<p>1.微信客服</p>
<ul>
<li>微信客服是提供给企业的一种全新的、连接客户的能力。</li>
<li>企业可在微信内的视频号、公众号、小程序、微信搜索、微信支付凭证，以及微信外的App、网页等多个入口，接入微信客服。</li>
<li>客户无需加好友即可在微信里和客服沟通，和微信聊天有一致的体验。客服回复后，客户将在微信里收到新消息提醒。</li>
<li>可邀请客户添加成员的企业微信或加入客户群，升级为专属服务。</li>
</ul>
<p>2.视频号</p>
<ul>
<li>企业可以绑定视频号，并在视频号主页和直播间展示“添加企业微信”。客户点击即可添加成员的企业微信，获得服务。</li>
<li>可在成员的对外信息页上展示视频号，让更多客户看到企业的视频号。</li>
<li>支持转发视频号动态和直播给客户和客户群，或转发到客户朋友圈，传播企业的品牌和活动。</li>
</ul>
<p><strong>二、多款效率工具升级，内外协作更高效</strong></p>
<p>1.文档 · 腾讯文档技术提供</p>
<ul>
<li>延续了腾讯文档稳定、流畅的编辑体验，超大文件秒开，协作内容实时同步。</li>
<li>可快捷添加同事为文档成员，和同事基于文档内容进行协作；还可以在文档中@同事，分派工作；与我相关的文档，通知更加及时。</li>
<li>可添加微信客户、上下游合作伙伴和其他联系人为文档成员，微信用户在微信上即可接收通知并参与，内外协作更高效。</li>
<li>支持丰富的插入能力，可在文档内插入个人名片和群聊名片、聊天记录、日程、应用页面等，协作体验更顺畅。</li>
<li>表格具备完备的计算能力，支持插入多种函数，计算速度全面优化，还支持幻灯片、收集表等格式，可满足汇报、收集等场景。</li>
<li>聊天中支持自动识别文档链接，还可在工作签名、群公告中添加文档，可用于常见问题、项目信息同步等场景。</li>
<li>可设置文档成员的浏览和编辑权限，文档转发至聊天时，还可进行快捷赋权。管理端支持使用数据分析、日志查看、二级回收站等安全管控能力。</li>
</ul>
<p>2.会议 · 腾讯会议技术提供</p>
<ul>
<li>延续了腾讯会议高清、稳定、流畅的开会体验，音视频性能卓越，异地办公协作更高效。</li>
<li>可在侧边栏、工作台和聊天中快速发起会议，也可通过日程预约会议，在会前收到日程提醒直接入会。</li>
<li>可邀请同事、微信客户、上下游合作伙伴和其他联系人一起开会，微信用户支持使用腾讯会议和小程序参会。</li>
<li>会中可开启虚拟背景和视频美颜，可发起屏幕共享、白板演示、互动批注、分组讨论等更多互动。</li>
<li>支持开启会议录制，录制完成后自动生成会议纪要，方便回看会议和分享。</li>
<li>支持会议密码、设置参会范围、屏幕水印、等候室等多种权限设置，管理端可查看企业会议使用数据和成员发起记录等。</li>
</ul>
<p>3. 日程</p>
<ul>
<li>新建日程时可以一键预约会议了。</li>
<li>新建日程时可以查看同事的日程闲忙，快速安排会议时间了。在个人信息页和群聊信息页也支持查看同事日程。</li>
<li>企业可以创建公共日历了，统一安排企业日程，如放假安排、培训讲座等，成员可自主订阅。</li>
<li>新建日程预约会议时，可快捷创建会议纪要了，方便成员在开会时记录会议重要信息。</li>
</ul>
<p>4. 邮件</p>
<ul>
<li>提供了更正式的工作邮件地址，可以方便地收发同事邮件，也可以做到全球邮件不丢信、秒送达。</li>
<li>支持对收发件人快捷发起群聊，讨论邮件内容。</li>
<li>智能分类未读邮件、审批邮件、日程邮件等不同类型邮件；从邮件中发起的日程邀约，还可在日程应用中统一管理。</li>
<li>继承了微信上亿用户的信息安全防护经验。管理端提供业务邮箱、收发信管理、登录管理、操作日志等功能，让信息管理更安全。</li>
</ul>
<p><strong>三、连接上下游，与经销商和供应商高效协作</strong></p>
<p>1. 上下游管理</p>
<ul>
<li>组建上下游通讯录，按企业汇总和管理经销商、供应商等合作伙伴的联系人。</li>
<li>支持分组整理企业，按对接人设置查看权限，统一管理上下游。</li>
<li>支持迁移群、二维码、导入名单等多种方式邀请企业加入，高效连接上下游。</li>
</ul>
<p>2. 上下游沟通</p>
<ul>
<li>使用上下游通讯录直接联系，跨企业找人跟找同事一样方便。</li>
<li>发消息可知对方已读未读，更有超大文件、会话存档、2000人大群等能力助力沟通。</li>
<li>可以群发产品动态等消息给上下游联系人，批量发送，即刻触达。</li>
</ul>
<p>3. 上下游协作</p>
<ul>
<li>会议、文档、日程、微盘等协作工具全面支持上下游，内外协作更高效。</li>
<li>可以共享应用给上下游联系人，轻松实现跨企业业务协同。</li>
</ul>
<p><strong>四、开放能力</strong></p>
<p>1. 微信客服</p>
<ul>
<li>支持通过接口管理微信客服，包括分配客服会话和收发消息、管理客服帐号、管理接待人员等。</li>
<li>也可在已有的客服系统中接入微信客服。</li>
</ul>
<p>2. 客户联系</p>
<ul>
<li>支持通过接口发表内容到客户的朋友圈了。</li>
</ul>
<p>3. 上下游</p>
<ul>
<li>共享自建、第三方应用给上下游中的企业后，可以通过接口获取上下游企业信息，打通应用使用。</li>
<li>可通过接口创建上下游会话，更便捷地业务沟通。</li>
</ul>
<p>4. 基础能力和OA应用</p>
<ul>
<li>应用消息支持图文、按钮、投票、多项选择等多种卡片样式，并可在用户点击消息中的按钮后，通过接口对卡片内容进行更新。</li>
<li>支持企业通过接口创建和获取成员的直属上级，并可授权上级信息给第三方，满足更多应用场景。</li>
<li>企业可通过接口获取成员的打卡数据并为成员进行排班，也可将打卡接口权限授权给服务商。</li>
</ul>
<p>5. 代开发模式</p>
<ul>
<li>开放自建应用代开发模式，企业可通过扫码授权服务商代开发自建应用，更加安全高效便捷。</li>
</ul>
</blockquote>
