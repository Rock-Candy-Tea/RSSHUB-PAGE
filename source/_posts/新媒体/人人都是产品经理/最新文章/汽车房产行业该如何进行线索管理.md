<blockquote><p>编辑导语：汽车房产行业的营销业务比一般的快消品要复杂一些，尤其在线索管理上。作者从基础概念+线索构成+管理要点三个方面总结了相关的技巧，我们一起来看看吧。</p></blockquote>
<p><img data-action="zoom" class="size-full wp-image-4839449 aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/07/mL4QtqqzMgfOovNEB9pU.jpg" alt width="900" height="420" referrerpolicy="no-referrer"></p>
<p>汽车、房产行业的营/销业务要比一般快销品复杂些，尤其是特有的线索管理。这里主要讲2个行业上线索管理的相同和差异点。为方便大家的理解，先统一业务概念。</p>
<h2 id="toc-1">一、基础概念</h2>
<p><strong>客户：</strong>传统意义上，客户指用金钱或某种有价值的物品来换取接受财产、服务、产品或某种创意的自然人或组织，即：发生过交易的称为客户。</p>
<p>互联网时代，客户的范围更宽，也衍生出与客户相关、不同业务场景下的特定分类（以汽车行业为例）。</p>
<p><img data-action="zoom" class="rich_pages aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/07/hO7Nur1Fy94vi08ngV8C.png" alt width="813" height="348" referrerpolicy="no-referrer"></p>
<p>粉丝到会员，或者客户等级，都是指基于客户价值或客户运营的不同维度。车和房因为是主业产品，且从客户角度、使用潜客和保客做了特殊对待。</p>
<p>但其实，当我们想卖什么产品时，粉丝和会员都是这个产品的潜客，成交、拥有了就是保客，只是车和房特殊些。都可以采用漏斗方式管理，聚焦时间精力，让营销人员全力以赴。</p>
<ul>
<li>有些是通过内容、营销自动化方式来引导转化，比如快消品；</li>
<li>有些是依赖专业服务人员来服务、转化，比如说房、车的销售，车辆保养回场的招揽，都可以用漏斗来监控。</li>
</ul>
<p>自动化的方式主要依赖转化率，而专业服务的不仅有转化率、还要考虑服务满意度。</p>
<p>以上是针对C端消费者的，实际上客户可以是最终消费者，也可以是代理人、分销商或供应链内的某个人或组织。</p>
<p><strong>线索与潜客的区别</strong>汽车行业在定义客购车意向时，在过去有两个常见的行业称呼：线索和潜客，代表意向的两个不同阶段。</p>
<p>没有到店的是线索，到店的是潜客（建档、标定意向度），区别在于客户有没有到店。</p>
<p>从业务实质看，线索是指客户的购买意向、描述的是“事”，潜客是指客户的一种属性定义，描述的是“人”，不能混淆。业务和产品上建议：</p>
<ul>
<li>购车意向，用线索、意向、商机等词语均可准确表达。</li>
<li>验证客户的意向等级也不再依靠客户到店才能识别。</li>
<li>不同阶段可以使用不同的线索状态和级别来定义，没有必要给两个定义。</li>
<li>继续沿用“潜客”，容易与客户的概念混淆，尤其在一个客户购买多台车时。</li>
<li>业务定义最好与系统一致，如此大家的理解和系统表达不会出现异议，数字化转型尤其重要。</li>
</ul>
<p>线索首先要有客户、进入客户管理对象或客户池的。有客户后，才是管理购买意向。所以，在平台设计时，首先使用客户OneID策略创建客户，再创建线索。</p>
<p><strong>渠道的新定义</strong>以往业务中，渠道是指线下分销渠道。进入互联网运营后、渠道的范围更宽，包括了线下和线上，也不仅仅是分销，也可以是传播。</p>
<p>我们不知道哪天哪个渠道会成为分销主力或传播主力。所以，对渠道的设计就需要多些考虑，比如哪些是自有渠道、哪些负责获客、哪些负责转化、哪些负责留存，必须考虑好合理（短而有效）的转化路径，营销预/决算的统计口径等等。渠道、场景、触点、内容/产品相辅相成，才能让渠道发挥更大价值。</p>
<p>关于渠道价值的评估，请查看《<a href="http://www.woshipm.com/it/3760497.html">车企的渠道价值评估</a>》<strong>推广计划。</strong></p>
<p>在2017年前，大部分车企在思考渠道融合和梳理如何评价渠道价值，还没有完全考虑清楚渠道、活动、内容的关系。</p>
<p>一方面是因为新渠道的不断加入和丰富，如何科学的分钱、达到更多获客是主要目标；另一方面，内容投放由广告代理公司操作，车企仅关注CPT/CPM/CPC品牌广告指标，还未考虑CPS/CPA等效果广告指标（真的关注是把指标写在合同里，不是合同标的物、不触发结算和合同条款就是虚的）。</p>
<p>2019年前后，广告投放策略有了明显的改变，品牌广告与效果广告预算从以前的7:3转为4:6。</p>
<p>追溯效果广告的贡献，就必须回溯到整个投放。投放遵循的基础规则：以合适的价格，在合适的时间、将合适的内容通过合适的渠道推送给合适的人群，一个5个“合适”的组合就是一个推广计划。</p>
<p>为了追求推广计划更高的贡献，广告优化师就得不断的调整、直至找到5个“合适”的最佳组合，就会出现某车的广告在某垂媒同一个点位同一天的上午和下午的内容不同（非AI推荐导致的差异）。</p>
<p>我们说明这么多概念，目的是为了推导出线索的构成。<strong>线索的构成</strong>线索需要回答的基础问题：谁、什么时候、从哪里来、因为什么来、想买什么、找谁买。</p>
<p>如此，我们需要这样的基础结构：时间+客户ID+商品ID+渠道ID+活动ID+推广计划ID+分销员ID（客户/用户ID）+销售组织ID（对照下这样的数据结构，如有不同，我们聊聊）。</p>
<ul>
<li>【什么时候】时间：获取时的时间，可以由不同渠道、通过接口实时同步的时间，或销售顾问手工创建的时间。</li>
<li>【谁】客户ID：按照OneID归一策略创建的客户ID。不能直接采用手机号这样简单的方式来处理。</li>
<li>【想买什么】商品ID：商品或服务是为了表达客户留资时的产品倾向或意愿。关于房产、汽车行业的商品体系搭建，未来分享。</li>
<li>【从哪里来】渠道ID：客户留资时的渠道。</li>
<li>【因为什么来】活动ID：促使客户留资的活动。</li>
<li>【因为什么来】推广计划ID：投放时的推广计划。推广计划中的渠道与线索中的渠道有可能一致，也有可能不一致。一个是执行渠道，一个是传播渠道。举例：店头做活动，在某渠道上传播，店头就是执行渠道，其他都是传播渠道。</li>
<li>【因为什么来】分销员ID：可以分享时的客户ID，也可以是渠道之间的推荐而使用用户ID。</li>
<li>【找谁买】销售组织ID：可以具体到经销商或楼盘项目。</li>
</ul>
<p>线索基本信息的构成，是为了更清晰的评价渠道/活动/内容的贡献，以及广告优化师的能力。从线索本身和管理角度，我们把线索分为4种：原始线索、唯一线索、渠道线索、客户线索。</p>
<ul>
<li>原始线索：符合线索创建规则的第一手信息为原始线索，经线索去重、合并处理后产生唯一线索。或多次分发给不同经销商或渠道，生成唯一线索u或渠道线索。</li>
<li>渠道线索：由原始线索分发或渠道方自建的线索，因试驾、结算、交车业务由渠道方报备到经销商生成唯一线索。</li>
<li>唯一线索：经去重合并处理后的线索信息。作为经销商业务管理对象，进行线索清洗、派发、分配、跟进、分析等。有唯一性判断逻辑。</li>
<li>客户线索：从客户维度，按照一个客户一个需求来进行分析和预测（绝大部分情况下，1个客户在一个固定期间内只会买1台车或1套房），需要从客户角度来归集需求量，以支撑高层的营销、销售、整车供应的决策。</li>
</ul>
<p><img data-action="zoom" class="rich_pages aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/07/6aCcYue4z2HoG0IvbpbE.png" alt width="701" height="350" referrerpolicy="no-referrer"></p>
<ul>
<li>原始线索中，客户ID+渠道ID是必须有的。</li>
<li>唯一线索/渠道线索中，需要执行归一策略：客户ID+销售组织ID是必须有的。房产比较特殊，有销售（业务）组织、项目组织、渠道组织，线索中是项目组织或渠道组织。</li>
</ul>
<h2 id="toc-2"><strong>二、线索管理的核心要点</strong></h2>
<p>在目标→执行→差距→优化中，优化的是客户/产品/渠道/活动/内容运营策略和执行。线索管理是整车销售的管理主线，也是房产行业销售管理的业务主线。</p>
<p><img data-action="zoom" class="rich_pages aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/07/uvKPdgtmo7GT4JViLOF9.png" alt width="808" height="407" referrerpolicy="no-referrer"></p>
<p>汽车的线索管理：主机厂牵头，经销商为主，渠道为辅。未来，车生活一体的服务商（经销商集团）将成为主要力量。</p>
<p><img data-action="zoom" class="rich_pages aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/07/MVhYPgjB6pFyDp9pWQU6.png" alt width="835" height="414" referrerpolicy="no-referrer"></p>
<p>房产销售的线索管理：（现阶段）代理公司牵头，渠道为主（相对业务较重），开发商分发补充。下一阶段，代理商为主、开发商和渠道为辅。未来，售房与物业一体的服务商（包括房产商）将成为主要力量。</p>
<p><img data-action="zoom" class="rich_pages aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/07/yoURC9TOx9FY1Bz0Vp5V.png" alt width="772" height="416" referrerpolicy="no-referrer"></p>
<p>房和车有非常明显的地域属性，制造和品牌运营的厂家是不会一个个地方铺的，社会化分工也不会让主机厂一条线全做的。未来分享城市运营时聊聊。</p>
<p><strong>（1）运营主体</strong></p>
<ul>
<li>汽车</li>
<li>房产</li>
<li>主机厂</li>
<li>经销商集团</li>
<li>开发商</li>
<li>代理商</li>
</ul>
<p><strong>（2）线索归一与分发</strong></p>
<ul>
<li>原始线索/唯一线索层做归一（去重/合并）。唯一线索分发给经销商。不分发原始线索给渠道（可操作业务量不大）。</li>
<li>原始线索不归一，唯一线索/渠道线索层做归一（去重/合并），应对不同场景。原始线索分发给不同经销商或渠道。</li>
<li>原始线索不归一，项目线索归一（去重/合并）。原始线索分发给项目线索。</li>
<li>原始线索不归一，唯一线索/渠道线索层做归一（去重/合并），应对不同场景。原始线索分发给不同项目或渠道。</li>
</ul>
<p><strong>（3）城市运营</strong></p>
<ul>
<li>全国布局、但未细耕城市。</li>
<li>以城市为核心展开5个维度的运营，特别重视活动运营。</li>
<li>全国布局、但未细耕城市。</li>
<li>以城市为核心展开5个维度的运营，特别重视渠道运营。</li>
</ul>
<p><strong>（4）商品体系</strong></p>
<ul>
<li>整车/配件/服务/普通商品/虚拟商品5大类车系→SPU，车系→SKU，VIN码为序列号。</li>
<li>房/服务/普通商品/虚拟商品4大类楼盘→SPU，户型→SKU，房间号为序列号。</li>
</ul>
<p><strong>（5）CDP策略</strong></p>
<ul>
<li>一个品牌车企一套CDP，集团建数据集市、提取各CDP数据并反哺</li>
<li>一套CDP即可</li>
<li>一套CDP即可</li>
<li>一套CDP即可</li>
</ul>
<p><strong>（6）活动运营</strong></p>
<ul>
<li>侧重大型品牌活动。</li>
<li>跟随主机厂品牌或主题活动，多样性，更贴近消费者。</li>
<li>侧重品牌活动。</li>
<li>主导活动运营，贴近消费者。</li>
</ul>
<p>主机厂不分发原始线索，是因为大部分线索是集采&补贴制的，主机厂不能将经销商也花了钱的线索分发给其他经销商。完全归属主机厂的线索量较小，操作上意义不大。</p>
<p><strong>线索全过程管理规则与指标</strong>汽车和房产在从线索获取到转化整个过程，业务环节、规则与关注点十分相似：定目标、盯过程、拿结果，在营销、销售各个环节提效、加速转化。</p>
<p><img data-action="zoom" class="rich_pages aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/07/i9CabS523TSfqA3DX8OC.png" alt width="736" height="334" referrerpolicy="no-referrer"></p>
<p>以房产为例：</p>
<p><img data-action="zoom" class="rich_pages aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/07/3DvrlRJwPZdmrV9UFCQi.png" alt width="756" height="392" referrerpolicy="no-referrer"></p>
<ul>
<li>营销关注线索量和值，销售关注转化。</li>
<li>渠道贡献分析、AI清洗与分发、跟进时效，都是房产和汽车重点关注的。</li>
</ul>
<p>线索评级是转化过程的核心，可以帮助顾问准确判断客户。一般分4个部分评价：</p>
<ul>
<li>明确客户的购买需求；主要使用人、购买动机、购买用途、车型配置与偏好or 户型与周边配置、预算等等。</li>
<li>推荐产品是否满足客户需求：外观、配置、性能、服务等是否符合客户期望。</li>
<li>解除购买障碍：限购、牌照、付款能力等。</li>
<li>客户沟通频次：线上线下互动频次等说明客户意愿和和谁买的可能性。</li>
</ul>
<p>整个业务管理聚焦在客户价值运营（转化为目标）、客户服务体验（服务为核心）、业务流程效率（运营/服务能力为核心）上。</p>
<ul>
<li>客户旅程中的场景触点覆盖，缩短转化路径、实现客户价值变现。线索管理具体在整车销售业务相关的活动、电商玩法、试驾、下订、成交、交车等业务上等。</li>
<li>优质的服务体验输出，提升客户满意度。线索管理具体在活动满意度、试驾满意度、展厅服务满意度等。</li>
<li>在营/销协同运营、精准营销、顾问跟进转化、业务决策能力、工具与AI应用覆盖等领域实现创新应用与突破，可有效提升业务流程效率。线索管理具体在AI清洗、渠道价值评估、顾问转化能力评估、线索评级、营销动态协同体系与决策BI等相关平台的构建和组织的辅导成长上。</li>
</ul>
