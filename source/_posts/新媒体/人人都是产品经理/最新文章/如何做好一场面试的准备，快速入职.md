<blockquote><p>本文针对于产品经理的岗位，从面试官的视角分析应该如何正确地进行面试。作者分享了五个方面的心得体会，教你应对面试官的种种问题，让你快速入职。</p></blockquote>
<p><img data-action="zoom" class="size-full wp-image-780442 aligncenter" src="https://image.yunyingpai.com/wp/2022/03/WMTf4ZvY1jmQDq3gGbBk.png" alt referrerpolicy="no-referrer"></p>
<p>该怎么才能做好一场面试的前期准备？</p>
<p>总的来说我们可以把面试官的诉求归纳为：找到一位与<strong>工作需求相匹配+产品基础+行业背景+产品感觉+问题解决能力+有可塑性的+优秀</strong>的产品经理，以下是我个人针对于面试做的一些准备分析拆解。</p>
<h2 id="F0NqC">一、公司</h2>
<p>通过对一家公司进行产品角度以及业务角度拆解，了解公司在市场的位置，加深你对于公司的一个认知，以下以我个人针对于朴朴做的公司分析举例。</p>
<h3 id="CPoGy">1. 产品角度</h3>
<p>范围：福州、厦门、广州、深圳、佛山、武汉和成都七座城市。</p>
<p>竞品：叮咚买菜；永辉超市；每日优鲜；盒马；美团买菜。</p>
<p><strong>（1）优势</strong></p>
<p>快速：自建的配送团队，半小时达；</p>
<p>配送距离：在配送范围也区别于每日优鲜3km的配送距离，仅覆盖仓周边1.5km；</p>
<p>菜价低：各类供应商支持政策下，朴朴拿到的商品报价，往往比市面价低30%-35%，正常规格的蔬菜价格能低至1元以内，这远比菜市场和商超实惠。</p>
<p><strong>（2）劣势</strong></p>
<p><strong>规模不经济</strong>：朴朴超市综合毛利率在16%左右，日均4000单是一家门店的盈亏平衡点。但截至目前，仅有厦门、福州两座城市越过了这一盈亏线。</p>
<p><strong>（3）难点</strong></p>
<p><strong>供应链管理</strong>：由于生鲜商品具有易耗性，且利润率低，所以需要去控制供应链的成本。</p>
<p><strong>商品选择：</strong>有限的货品，主打“便捷性”和“即时性”，凸出生鲜的“标准化+品质化”，在前置仓模式下，如何确定标准化商品的种类与数量？此外，类似于7/11便利店的品牌自制饭团，生鲜电商的品质化商品又可能是什么？</p>
<p><strong>数字化能力：</strong>对本地社区居民的需求预测，降库存和无所谓的商品损耗。</p>
<p><strong>产品本质</strong>：一个更快速送达的电商，网点以7-11便利店的形式开。</p>
<h3 id="zgORJ">2. 业务角度</h3>
<p>2020年中国生鲜电商行业商业模式主要包括传统生鲜电商（天猫生鲜、京东生鲜）、平台“到家”模式（美团、饿了么）、“到店＋到家”模式（盒马鲜生、7fresh）、前置仓模式（每日优鲜、朴朴超市、叮咚买菜）、到柜自提模式、社区团购模式。</p>
<p><strong>（1）市场</strong></p>
<p>2019 年生鲜电商行业市场交易规模达 2796.2 亿元，较上一年增长 36.7%。2020 年受疫情影响，消费者对于生鲜到家的需求急速增长，生鲜电商市场交易规模将会有显著的提升。预计到 2023 年，生鲜电商市场交易规模将超过 8000 亿元——由此可见，生鲜电商行业未来依然有巨大的增长潜力。</p>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/03/CcMbycj5p56HCmkfg6Uf.png" alt width="380" height="188" referrerpolicy="no-referrer"></p>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/03/nyp7voqVlrV3JC3tb9Bn.png" alt width="380" height="188" referrerpolicy="no-referrer"></p>
<p><strong>（2）风险</strong></p>
<p>前置仓模式下如何做到盈亏平衡？</p>
<p>21年社区团购模式的增幅524%＞前置仓的49.1%。</p>
<p>巨头的入场，美团、菜鸟、苏宁小店……</p>
<p>2021Q2数据显示，叮咚买菜日均单量65万+，每笔订单均价60元左右。每日优鲜日均26万+订单，每笔订单均价为96.1元。预计叮咚买菜今年营收能达到200亿，但从市场规模上来看，前置仓模式的增长空间其实是有限的。</p>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/03/IOR1Pfh5mqAoHQtSwjwN.png" alt width="347" height="255" referrerpolicy="no-referrer"></p>
<h2 id="EAFLp">二、职位要求</h2>
<p>该岗位看重什么？将核心标签提炼出来，例如：三年经验、游戏产品策划、数据分析、提案能力、有过游戏产品经验。在面试中突出以上5个标签，强化自己项目经验遇到的问题及解决方案，不打无准备之仗，每次面试要有必要的准备。</p>
<ul>
<li>负责过商品中台</li>
<li>三年经验</li>
<li>对商品领域有了解</li>
<li>平台化产品架构能力</li>
<li>全局业务理解</li>
</ul>
<h3 id="NRLZf">1. 岗位职责</h3>
<p>这部分数据来自于岗位招聘的JD，认真看。</p>
<p>1. 负责商品中台相关产品，如成本中心、价格中心、库存中心等的规划、设计，为平台商品运营的体验及效率负责；</p>
<p>2. 负责平台商品的全生命周期管理体系搭建；</p>
<p>3. 主动挖掘业务方的真实需求，输出需求分析、产品原型、需求说明、功能描述等文档；</p>
<p>4. 实现产品开发管控、上线发布、用户反馈、跟进解决的完整闭环。</p>
<h3 id="tNwSE">2. 任职要求</h3>
<p>这部分数据来自于岗位招聘的JD，认真看。</p>
<p>1. 本科及以上学历，有3年以上自营电商平台或B端类的产品经验优先；</p>
<p>2. 对商品领域有较深的理解，具备平台化产品架构设计能力，能承担整体商品相关的产品规划，为业务持续发展而建设高可用性的系统；</p>
<p>3. 有全局的业务理解（供应链、营销），持续规划和体系化优化的意识和能力，目标驱动，联动运营；</p>
<p>4. 具备较强的沟通能力，具有高度责任感和较强的团队合作意识。</p>
<h3 id="jm4c3">3. 薪资待遇</h3>
<p>举例：</p>
<p>薪资：30-45K 15薪。</p>
<p>base地：杭州/福州。</p>
<p>公积金：公积金12%。</p>
<p>绩效：季度绩效奖金+浮动年终奖+高回报期权。</p>
<h2 id="v93PX">三、我是谁</h2>
<p>在这个环节你需要填充一些自我介绍，以及和该公司比较相关的一至两个项目，通过项目来告知面试官你具备的能力，介绍的部分主要是自己的工作经验，学历，过往公司等等。重点是别说废话, 着重业绩、工作、经验产出。尽量突出自己在业务上的贡献，以及个人在某个领域上的钻研，对于销售、市场等业务，人脉资源也是你的资本。比如说 :</p>
<ul>
<li>业务类 : 熟悉计算广告, B2B, 保险, 金融, VR , 流媒体, 在线直播等业务模型。</li>
<li>技术类 : 丰富的数仓建模 和 数据挖掘经验, 从 0 到 1 构建企业级数据仓库。</li>
<li>运营类 : 站内 MAU 突破8千万，并且将 M站 用户留存从 15% 提升到23%。</li>
<li>市场类 : 将公司获客成本从 99元/人降低到54元/人, 并且开拓 20 个新渠道。</li>
<li>销售类 : 2020 年第三季度超额完成 KPI, 为项目组带来 200W 利润。</li>
</ul>
<h2 id="SbB6Z">四、为什么是我</h2>
<p>我一般将该部分拆分为，产品基础能力+岗位匹配度+其他价值（包含可塑性的部分）。</p>
<h3 id="HXpuB">1. 专业能力</h3>
<p>你的产品经理的基本功是什么，达到什么样的程度？</p>
<ul>
<li>0~3年，细讲</li>
<li>3~5年，带过</li>
<li>5~10年，省略</li>
</ul>
<h3 id="d1YIC">2. 岗位匹配度</h3>
<p>这个部分就是比较重要的部分了，在现在越来越细分的赛道中，招聘的时候最重要的就是岗位的匹配度了，将自己的条件与你前面提炼出来的五个点互相结合寻找匹配点。</p>
<h3 id="BWwZt">3. 我能给公司带来什么</h3>
<p>从部门管理以及产品能力上进行一些拓展，包含自身的一些资源渠道优势等等。</p>
<h2 id="iB5uv">五、面试记录</h2>
<p>在这边做下一些面试相关的记录内容，方便后续复盘。</p>
<p>如面试时需要注重考虑如何增加“眼缘”值：</p>
<p>视频面试：面试的时候要找个明亮、整洁的地方，打理好自己的仪容仪表。</p>
<p>现场面试：打理仪容仪表+刷牙+清新口气。</p>
<h3 id="vFiBf">1. 一面</h3>
<p>HR面，30min，一般只是聊些基本情况。</p>
<h3 id="aGrPd">2. 二面</h3>
<p>部门面，聊业务和项目，随文附上最近经常被问到一些问题：</p>
<h3 id="RN4KI">3. 你职业生涯中遇到最大的挑战是什么，怎么解决的？在这过程中你学到了什么？</h3>
<p>首先很明确的一点就是挑战是无处不在的，不可能没有挑战，如果你回答没有挑战的话，那么很明显你是在摸鱼，同时这个问题也是面试官了解你能力模型的一个好问题，透过这个问题，他能判断出你的设计能力，复盘能力以及协作能力。</p>
<h3 id="WpXUd">4. 你觉得你/工作期间做的最好的项目是哪个? 为什么? 具体介绍一下?</h3>
<p>首先要介绍项目背景、自己在项目中承担的角色，以及为项目带来的价值、优化点。面试官通常会根据该部分的内容向你提问，你需要确保几点 ：</p>
<ul>
<li>你写的内容都是你做过且能说的清楚的</li>
<li>总结一些在项目上遇到的问题和优化点</li>
<li>细化自己在每个项目上的贡献和成长</li>
</ul>
<h3 id="MfRRl">5. 你用了我们的产品么？对我们的产品有啥建议？</h3>
<p>这个其实就是所谓的求职心了，你想入职一家公司，去人家公司面试前，连别人公司做啥的都不知道，这不用说在面试过程中会错漏百出了，在面试官的心里更会留下不好的影响。</p>
<h3 id="JVJIn">6. 为什么从 xxx公司离职?</h3>
<p>问这个问题本质上是公司和求职者进行双向匹配，公司通过该问题了解你的雷点是什么，然后判断自己公司是否有这个雷点，作为求职者就大大方方的说自己的想法就好，不用去抱怨公司，也不必支支吾吾，因为遮掩下入职的公司也会触及到你的雷点，导致你再次离职。</p>
<h3 id="v5CIL">7. 你后续的职业规划是什么样的？</h3>
<p>职业规划包括 3 个要素：职业定位、目标设定、通道设计。</p>
<p>职业定位，就是你对产品岗位的定位，因为产品经理岗位划分有很多种；根据行业分，有金融产品经理，供应链产品经理，电商产品经理，社交产品经理等等；根据职能分，有商业产品经理，策略产品经理，内容产品经理等等；根据业务流分，有中台产品经理，后台产品经理，移动端产品经理等等。</p>
<p>目标设定，可以用产品岗职级来看，产品经理的职业发展同大部分岗位一样，也需要一路升级打怪，职业发展之路可分为：产品助理、产品经理、高级产品经理、产品专家（专业方向）、产品总监（管理方向）、事业部负责人、产品副总裁 / 联合创始人 / CEO。</p>
<p>通道设计，可以从时间段来规划，比如刚来公司一个月怎么样，一年怎么样，三年后希望怎么样，等等。</p>
