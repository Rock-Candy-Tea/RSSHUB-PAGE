<h2 id="toc-1">一、什么是模块化思维</h2>
<p><strong>模块化思维是职场进阶的思维模式。</strong></p>
<p>不少职场人将工作或项目看成是一锅浆糊，不分主次；</p>
<p>亦或者，将企业的流程视作圣旨，日复一日重复执行；</p>
<p>企业的运营确实少不了一些重复执行工作，这些基础而又枯燥的工作也成为了管理上的心病。</p>
<p>模块化思维可大致分为员工及企业两方面。</p>
<p>员工执行角度：</p>
<ul>
<li>模块化思维可以把复杂繁琐的工作分解，提前完善细化后的模块材料，降低工作难度；</li>
<li>模块化思维能帮助员工把握工作的主次，合理分配工作的时间和精力，提高工作效率；</li>
</ul>
<p>企业管理方面：</p>
<ul>
<li>项目模块化帮助企业理清业务流程，完善业务执行时可能出现的问题，减少信息不匹配带来的业务及管理风险；</li>
<li>项目模块化利于企业梳理自身资源，优化业务线价值，提高合作的成功率、产品的附加价值；</li>
<li>项目模块化便于企业管理员工，提升员工执行效率，有助于提高组织内各部门相互协作的配合度；</li>
</ul>
<p>橘先前与某家乙方跨境电商服务平台的VP沟通时，她也对部门工作及项目模块化运作表示出期待。</p>
<p>沟通时，就以下几个问题展开：</p>
<ul>
<li>如何更好地提高人效？</li>
<li>如何提高甲方满意度？</li>
<li>如何整合资源提高项目单价？</li>
</ul>
<h2 id="toc-2">二、项目工作模块化（思路）</h2>
<p>把企业经营比作一台电脑，各个职能部门即是各个产品模块，好比总经办/董事会是CPU，市场运营及品牌部门是显卡，客服部门是显示屏，公关部门是水冷系统。</p>
<p>跨部门协作的各位员工，也像每个产品模块的零件。</p>
<p>既然作为零件，就或多或少会承担对应的责任。</p>
<p>也无关要紧，工作中总有机会遇上繁琐的重复任务。</p>
<p>回归正题，此时应如何处的？</p>
<p><strong>1）要素规整法</strong></p>
<p>重复且分散无关联的工作内容，很难有统一的流程将它们串联。</p>
<p>把相似类型的工作内容合并同类项，并提前做好模板内容的预设。在工作发生之前提前做好准备，最好做到一键复制黏贴的效果。</p>
<p>例如：客服工作；行政事务等等。</p>
<p>每日反复回复用户大量相似问题的答疑及产品销售引导；</p>
<p>行政六大模块执行暂且不提，报表、报销及文档归档等都够提一壶；</p>
<p><strong>员工侧模块化思维实操</strong></p>
<p><strong>1）客服工作</strong></p>
<p>客服的核心工作是做好用户反馈，避免客诉的产生。</p>
<p>而日常使用的客服文档便是规整法的运用之一，</p>
<p>优秀的客服文档即可一键发送，快速培训运营人员。</p>
<p>优点：</p>
<p>①客服文档统一并完善了客服的执行话术，让人数众多的客服人员完成统一、专业的话术对外回复。</p>
<p>②客服文档将可能产生的用户咨询问题及客诉问题提前准备，用于快速回复用户疑问，减少工作重复话术打字，且可用于快速培养专业客服。</p>
<p><strong>2）行政事务</strong></p>
<p>行政事务的繁琐毋庸置疑，</p>
<p>各位看官想想自己贴发票报销的时候，</p>
<p>或者安排文秘贴发票报销的时候（doge）。</p>
<p>一起看看行政真实案例：</p>
<p>（以某家制造业公司为例）</p>
<ul>
<li>文书管理：行文管理、档案管理；</li>
<li>资产管理：行政采购、非生产性固定资产管理；</li>
<li>安全管理：保安管理、生产安全、消防管理；</li>
<li>后勤管理：食堂管理、宿舍管理、清卫管理、车辆管理；</li>
<li>公关管理：行政法律事物、对外联络；</li>
</ul>
<p>正如上述案例，如果这家企业的行政各模块事务仅仅是刚刚的二级模块化，橘敢保证执行人员仍会是“忙里忙外、满身大汗”。</p>
<p>使用要素规整法，将执行模块进行最小活动单位细分，进行同类项规整。</p>
<p>在同一时间内或同一地点内，处理或提前准备同类型的工作。</p>
<p>可能有人不太明白，为什么地点位置，也可以加入同类项规整内？</p>
<p>例如：后勤管理要求的检查地点安排在宿舍食堂区域；</p>
<p>而文书管理、安全管理安排在行政主楼；</p>
<p>如果作为一名行政人员，今天去宿舍食堂查查，中途又跑到行政主楼测测。路程所花费的时间，即是无效的。</p>
<p>不做无用功，把时间花在点子上；</p>
<p>别自我感动，思考是否能更优解？</p>
<p><strong>2）流程化思维</strong></p>
<p>项目流程化，作为出入于各大公司的老鸟的我们，绝不陌生。</p>
<p>标准作业流程SOP/OA工单流程是企业常用的一种作业方法</p>
<p>企业的标准体系由技术、管理、工作三方面标准体系构成。</p>
<p>其目的在于：</p>
<ul>
<li>实现各模块管理规范化，实现企业把控各模块运作；</li>
<li>便于员工培训，短期培养出合格的对应模块的员工，提升员工可替代性，增加企业话语权；</li>
<li>便于追踪生产、管理及项目执行的责任环节，及时追责调整；</li>
<li>梳理并树立良好的企业形象，取得客户信赖和满意度；</li>
</ul>
<p>例：产品开发流程</p>
<p>idea产生—需求分析—项目评估—PRD撰写—需求评审—开发测试—发布上线—运营监控—项目效果后评估</p>
<p>通过上述实操案例，让我们将话题回归至与VP沟通的话题里。</p>
<p>此时的流程化思维，指的是在具体制定某项运营工作，完整记录用户参与的全部流程，并且针对流程中的各个关键节点进行详细设计的思维方式。</p>
<p>在思考项目运营流程时，应从整体出发，到局部细化。</p>
<p>通过参与者和流程顺序的相互关联性，将一切相关的活动步骤和内容进行梳理，然后不断优化的思考方式。也可以把这样的流程化思维作为一种客户化思维，根据客户的需求及行为，从客户的满意度角度出发设计流程对接。</p>
<p>* 如何构建流程化思维？</p>
<p>企业所处的行业、企业管理的方式、企业的组织架构及企业各部门沟通链路的不同等等诸多因素导致流程构建的不同。</p>
<p>但是，流程化思维的核心是不变的，以客户为导向，将客户的需求及行为融入到员工的管理体系、考核制度和流程。</p>
<p>这应该成为上下一致的共识，企业的生存和发展需要客户及订单的维系，无论甲方还是乙方，作为商业主体进行经营，必然有另一方的合作。如果企业抛开客户谈管理，只不是一厢情愿的自我感动。（毕竟不是谁都是刘特佐这种空手套大神）</p>
<p>承接流程化思维构建的核心，实际思考时可以参照以下的逻辑进行排序：</p>
<ul>
<li>时间顺序：步骤、流程类；</li>
<li>结构顺序：空间、地理位置类；</li>
<li>程度顺序：事情的轻重缓急、重要性……</li>
</ul>
<h2 id="toc-3">三、业务产品模块化 —— 案例思考</h2>
<p>本文“第二大点”描述了1、2点话题的回答及思考，本节聊聊第3点话题的内容。</p>
<p>* 如何提高项目单价？</p>
<p>在说这个问题前，橘想先举例两个案例：</p>
<ul>
<li>好莱坞电影剪辑宣发流程；</li>
<li>中央电视台广告业务对外合作；</li>
</ul>
<p><strong>1）好莱坞电影剪辑制片流程</strong></p>
<p>导演剪辑版：最能代表导演意图，完全按照导演的思路来剪辑的影片；</p>
<p>一部电影拍摄完成</p>
<p>↓</p>
<p>后期制作粗剪环节</p>
<p>（把镜头按照剧本排列）</p>
<p>↓</p>
<p>剪辑师精剪</p>
<p>（剪辑版本完成）</p>
<p>↓</p>
<p>导演精剪：导演剪辑版的作品产出</p>
<p>（根据导演的喜好和对镜头的选择进行调整）</p>
<p>↓</p>
<p>制片人入场剪辑</p>
<p>（代表投资方、大片场、发行方）</p>
<p>↓</p>
<p>上映宣发版本电影</p>
<p>如果把电影看作一件商品，这些人就是这个商品的出资方和销售方，所以他们当然就是商品的实际控制人。也就有权利对影片进行修改。</p>
<p>虽然影片是导演的作品，但是更是他们的商品。</p>
<p>“咱们先别谈艺术，谈艺术伤钱！”</p>
<p>所以，这些人的修改通常不是从艺术角度考虑的，</p>
<p>而是从用户或市场的方面出发的。</p>
<p>例如：降低观影门槛、增加流行元素、偏向大多数受众心理、是否要去掉一些观众可能会反感的或者限制级的镜头等等因素。</p>
<p>最重要的是电影要缩短。</p>
<p>导演拍的内容通常不想去掉太多，毕竟都是他的心血。但是片方会考虑到排片的因素，内容能缩短就能多排片，增加了排片次数，就能提升观影人数，才能多赚钱！</p>
<p>这也是为什么上映版本通常不是导演剪辑版的原因。</p>
<p>但导演剪辑版并非没有它存在的用途，</p>
<p>它变成了一种营销方式。比如很多人好奇电影都上映了，为什么还有DVD和蓝光碟这东西？如果这DVD的内容还是上映版本，我相信看过电影的人基本不会购买。</p>
<p>可是，DVD内容里包含了导演剪辑版，有观众没有看过的镜头，所以观众想看得掏腰包了。</p>
<p>正如近期，《正义联盟导演剪辑版》风靡网络，4个多小时的观影时间，仍然抵不住人们冲向院线的脚步。这部上映日期在2017年的《正义联盟》电影，又一次重获公众视野的焦点。各大漫威系UP主，如B站努力的Lorre，纷纷奔走相告。</p>
<p><strong>* 绝大多数的时候，商品不是由设计者定的，而是有金主爸爸们。</strong></p>
<p><strong>2）央视央广广告业务对外合作</strong></p>
<p>企业投放央视广告，不仅是为了品牌曝光，也是冲着CCTV的金字招牌为品牌背书，提升品牌价值。</p>
<p>那么作为甲方的品牌方的诉求是什么？</p>
<p>下面橘将展示两个央视央广的内部案例。</p>
<p>↓</p>
<p>从图可知，甲方的需求分为基础需求和增值服务；</p>
<p>甲方投放CCTV广告的核心目的：</p>
<ul>
<li>品牌/产品曝光；</li>
<li>获取背书，提升品牌力；</li>
</ul>
<p>CCTV广告的投放通常是节目投放和门户站宣传，例如综艺节目、大型活动节目（春节等）、新闻节目等栏目投放。投放类型大多为切片广告、广告插播、公关文稿等。</p>
<p>但是，上述投放类型其实不能完全涵盖甲方的核心目的，即甲方有深入的需求层次，也可以理解为乙方可以更深入优化产品。</p>
<p>TVC电视广告，不过是央视最核心的展示平台。</p>
<p>联系核心渠道的衍生价值，CCTV的优势即如下：</p>
<ul>
<li>具备优秀的制作能力，可以制作企业所需的并带有CCTV台标的广告宣传片、企业荣誉牌匾；</li>
<li>为企业提供一站式服务，满足企业重要仪式需求，例如为企业租赁高星级酒店办理签约仪式、拍摄并制作荣誉牌匾；</li>
<li>办理广告播出证明，提供符合规定的证明及二维码；</li>
<li>多官方平台联动推广，央视网、央视影音、央视频等CCTV渠道宣传，覆盖多年龄段用户；</li>
</ul>
<p>乙方在面对甲方前，会整合自身的资源，设计对应的产品及服务；</p>
<p>甲方在沟通乙方时，根据自身市场需求，选择乙方的服务并评估；</p>

<h2 id="toc-4">四、业务模块化-实战思路（品牌跨境服务）</h2>
<p>橘并未过于深入研究过跨境电商板块，本期选择这个主题还是因为一个承诺。所以，这个模块还是多以自身对于市场及品牌的理解来阐述。</p>
<h3>1. 信息密度=品牌高度 —— 广告需求</h3>
<p>越是知名的品牌，网络上检索到的资料信息越多；</p>
<p>相反不知名的品牌，可以检索到的资料信息越少；</p>
<p>橘在之前写过的《投放策略笔记：小红书投放思维》里，曾详细描述过小红书投放策略。</p>
<p>其中的案例，雅诗兰黛在小红书投放策略中显示每月投放了大量KOL测评，保持内容在小红书APP内的信息密度。并且，通过SEM信息流投放，增加品牌信息密度。</p>
<p>小红书的用户只要在小红书搜索【雅诗兰黛】或对应关键词，便能跳出成百上千条优质内容信息。这些内容引导消费者建立对品牌的认知、理解，从而使品牌建立真正的品牌力，而非纸上谈兵。</p>
<p>回归主题，对应品牌跨境服务项目。</p>
<p>考虑品牌方对于出海品牌高度的需求，转化为信息密度需求。整合企业资源，量化产品价格对应的效果供甲方选择。或可提供代运营服务，类似于名鞋库为各大品牌做的代运营业务。</p>
<p>* 提升企业资质，整合外网资源，提供对应渠道及价格，提升品牌方在外网渠道的信息密度；</p>
<ul>
<li>跨境电商平台:速卖通、Wish、亚马逊、eBay、阿里巴巴国际站、Lazada、Shopee等；</li>
<li>社交平台：Facebook、Google、Instagram、Tiktok、YouTube、Twitter等；</li>
</ul>
<h3>2. 海外供销渠道整合 —— 经销渠道</h3>
<p>一些品牌方面对海外复杂的政治环境、贸易环境、人文环境，往往束手无策，导致其对海外业务望而却步。</p>
<p>为什么选择跨境服务的乙方协助？</p>
<p>正是基于品牌方最迫切的期待</p>
<p>—— 把货物顺利地销往海外，在海外打出品牌知名度，占领海外市场，并且顺利地把钱款入账</p>
<p>乙方此刻可以整合“历史成绩单”，将优质的资源挑选出来并进行深度合作，合并为甲方期待的业务。</p>
<p>例如：</p>
<ul>
<li>海量海外买家资源；</li>
<li>优质经销商资源；</li>
<li>多个国家的清关能力；</li>
<li>币种结算能力；</li>
</ul>
<h3>3. 提供海外物流仓储——海外仓</h3>
<p>出口海外的物流不像国内快递，不仅涉及报关报税等方面，运输时长及仓储都是麻烦事。自跨境电商步入数字贸易时代后，物流的时效、成本直接影响了品牌方的产品。</p>
<p>要如何理解品牌出口？</p>
<p>即为中国企业与目的国卖家的商品之间必有“一战”。</p>
<p>如果产品发货慢得像蜗牛，相同价格和品质下，还有什么竞争力呢？</p>
<p>那么，品牌方的需求即呼之欲出 —— 海外仓。</p>
<p>在跨境电商下半场的竞争，应体现在服务及时效上，当前大多数有实力的跨境服务商在全球多点布局海外仓业务。</p>
<p>以打造品牌方商品的出海通路为目的，协同品牌方提升发货效率，实现高效配送至经销商或客户手中。</p>
<p>橘对海外仓的理解并不深入，它其实拥有很详细、繁琐的业务内容。</p>
<p>以物流行业涉及的领域为例，基础托运服务外，海外仓还对应着各类增值服务。</p>
<p>大体可以分为以下三类型：</p>
<ul>
<li>出库增值：①出库增加/更改产品或单品标签；②出库更换/新增外包装箱及装箱；③截单时间后订单提前出库；④自提单紧急出库服务费…</li>
<li>库内增值：① 库内增加/更改产品或单品标签；②库内指定单品更改产品或单品标签；③库内单品多面拍照…</li>
<li>入库增值：①更换单品条码（海外验货仓）；②付费预约送仓；③上架前拍照辨识…</li>
</ul>
<p>上述三点，仅仅是需求对应产品的模块化思维的一些思考，当然不能完全概述品牌出海的需求和跨境服务乙方的所有业务；</p>
<p>与本文第三章节的案例思考类似，这章节的核心思维即需考虑到甲方在其产品、品牌、市场入驻某个海外市场时所必须/急迫/期待的需求。</p>
<p>正如上述所言，橘简单从以下几点总结下品牌出海下，品牌方的需求及跨境服务乙方可提供的业务模块：</p>
<ul>
<li>广告需求：KOL合作服务、多平台流量业务服务、KOC铺量合作服务；—— 满足企业提升品牌力的需求</li>
<li>代运营需求：国内外品牌认知，提供代运营服务及人才支持服务；—— 满足企业尝试性运营的需求，对应降本增效</li>
<li>货款回款需求：妥投收款服务、金融贷款服务、货币结算服务；—— 满足企业回款或兑换货币入账的需求（不完全）</li>
<li>经销渠道建设需求：商品清关通关服务、优质经销商伙伴资源；—— 满足企业快速筛选并评估经销渠道价值</li>
<li>市场调研需求：区域市场竞品调研服务、市场经营分析服务、海外市场舆情服务；—— 满足企业市场调研的需求</li>
<li>技术需求需求：海外版数据中台软件服务、数字贸易一站式平台服务；—— 满足企业对海外市场情况精细化及快捷运营的需求</li>
<li>物流仓储需求：海外仓服务、物流服务商对接服务；—— 满足企业产品出口海外地区的需求</li>
</ul>
