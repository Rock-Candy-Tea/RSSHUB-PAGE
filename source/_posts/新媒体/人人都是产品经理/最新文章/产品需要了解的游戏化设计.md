<div>   
<blockquote><p>编辑导读：为什么很多人沉迷于游戏？因为它的反馈非常及时，无论是正向的还是反向的，能一直吸引用户的注意力。如果将这些游戏化设计应用到普通的产品中，应该怎么做呢？</p></blockquote>
<p><img data-action="zoom" class="size-full wp-image-5502226 aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/06/BUmtdssyLCMevOtB4zIx.jpg" alt width="900" height="420" referrerpolicy="no-referrer"></p>
<p>现在很多的电商像淘宝的叠猫猫、盖楼、超级喵运会、芭芭农场；拼多多的多多果园；支付宝的蚂蚁森林；饿了吗美团等的免费领水果的小游戏等，似乎都在通过游戏作为载体来提高自身业务的活跃。游戏分为营销化游戏和运营化游戏，营销化游戏更多像以上那些以H5（内嵌），小程序为载体。</p>
<p>从设计的角度来说，营销化的游戏更像是产品而非一件作品，是将重复任务包装成竞技游戏；而以工作室、制作人制度，往往能给项目组一定创作空间的运营化游戏，更倾向于将此类的游戏设计称之为作品。</p>
<p><strong>产品与作品的区别就</strong><strong>在于此，产品设计是依照于一套既定规则下的产物。</strong></p>
<p>其实在游戏业务中的产品设计，就好似一架桥梁，我始终觉得产品和游戏之间的关系是密不可分的，许多产品的设计思维能够在游戏中内化体现，反之也是一样。</p>
<p>本文仅仅是从产品设计的角度去分析、思考并提供一种思路的游戏化设计，希望能够对平时的产品设计中挖掘出独特的想法，而外在的其他游戏因素如未来政策、市场前景等噪音暂不在分析范围之内。</p>
<h2 id="toc-1">一、简析游戏化内因</h2>
<h3>1.1 游戏行为的动机</h3>
<ul>
<li>喜欢有趣的事物</li>
<li>喜欢创造</li>
<li>喜欢挑战</li>
<li>喜欢被人认可</li>
<li>喜欢收集</li>
<li>喜欢惊喜</li>
<li>不喜欢失去</li>
<li>想要不容易获得的东西</li>
</ul>
<h3>1.2 游戏对人产生的情绪变化</h3>
<ul>
<li>玩家可以在重复的游戏中不断学会失败是成长的机会，而不是无助、压力和恐惧</li>
<li>由好奇的心理、挫败的体验和获得的满足产生的乐观和自信</li>
<li>游戏能够锻炼人不断承受失败和心理压力的坚韧素质</li>
<li>游戏的一个特质是人往往会透过重复的失败而最终达到目的，能够极大的降低玩家的试错焦虑并</li>
<li>获得积极的成就感</li>
</ul>
<h3>1.3 游戏设计的重要元素</h3>
<ul>
<li>奖励</li>
<li>目标</li>
<li>规则</li>
<li>挑战</li>
</ul>
<h3>1.4 心理学应用</h3>
<ul>
<li>游戏化设计：掌控感、成就感、心流、损失厌恶，等等。</li>
<li>徽章系统：荣誉感、目标驱动、攀比心理、炫耀、资历证明，情感沉淀，等等。</li>
<li>邀请机制：实用性分享、激励性推广。</li>
</ul>
<h2 id="toc-2">二、游戏化设计分析</h2>
<h3>2.1 充值篇</h3>
<p><strong>① 游戏中的货币设计</strong></p>
<ul>
<li>金币：提供实时、可量化反馈  （成就感↑）</li>
<li>钻石：主要收入来源 （成就感↑、差异性↑）</li>
<li>体力或各种代币：限制游戏次数；玩法多样性（社交性↑、差异性↑）</li>
</ul>
<p><strong>② 充值比例</strong></p>
<p>常见的货币充值比例为1:10或1:100，可以提升用户消费心理满足感</p>
<p><strong>③ 充值活动</strong></p>
<p>首充：诱导性，刺激消费</p>
<p><strong>a.完成充值后打通了交易流程，极小成本获得高收益</strong></p>
<p><strong>b.修正用户从不花钱——花钱的心理暗示</strong></p>
<ul>
<li>每日充值/充值签到：刺激用户持续充值，形成充值习惯</li>
<li>累计充值，充值全服奖励：带动未消费或小额消费玩家，暗示玩游戏需要消费形成从众心理</li>
<li>结合VIP/通行证刺激充值，返利模式，高额投资回报率等，体现承诺一致心理</li>
<li>100%转盘抽奖</li>
<li>充值倒计时：损失厌恶心理，因害怕失去而做出冲动性决策</li>
</ul>
<h3>2.2 消费篇</h3>
<p><strong>① 差异性消费</strong></p>
<p><strong>a.道具：不同的购买力，达成购买力平衡</strong></p>
<ul>
<li>金币购买</li>
<li>钱币购买</li>
</ul>
<p>设计原因：让付费玩家得到心理满足，让非付费玩家形成逆袭错觉</p>
<p><strong>b.皮肤：为角色形象付费，独特性得到心理满足</strong></p>
<p>设计原因：社交中的形象管理</p>
<p><strong>c.购买特权：</strong></p>
<ul>
<li>VIP等级刺激持续充值</li>
<li>不同等级权利</li>
<li>特殊标识</li>
</ul>
<p>设计原因：让付费玩家体验优越感</p>
<p><strong>② 为确定性消费</strong></p>
<p><strong>a.抽奖——让人消费上瘾的抽奖系统（博彩心理）</strong></p>
<ul>
<li>高二消费购买确定性</li>
<li>多次消费购买确定性</li>
<li>去除无用道具的消费</li>
</ul>
<p>设计原因：<strong>人们追求确定性，却会因为不确定性上瘾</strong></p>
<p><strong>b.进阶的确定性</strong></p>
<p>游戏中武器/角色进阶的确定性可通过加成功率/幸运率道具提升确定性</p>
<p><strong>c.属性的确定性——为属性的差异性付费</strong></p>
<p>设计思路：设计多个属性，每个属性有不同随机值，每次刷新随机改变等</p>
<p><strong>③为时间消费</strong></p>
<p><strong>a.购买时间/体力</strong></p>
<p>设计原因：简化成长路径/节省时间，更好的体验</p>
<p><strong>b.为加速付费</strong></p>
<ul>
<li>快速开箱</li>
<li>离线挂机高额收益</li>
</ul>
<h3>2.3 活跃篇</h3>
<p><strong>① 常见KPI</strong></p>
<p>PV（Page View）、UV（Unique Visitor）、DAU（Daily Active User）、WAU（Weekly Active User）、MAU（Monthly Active User）、平均页面停留时间、次日留存率、3日留存率、7日留存率、30日留存率等</p>
<p><strong>② 提升平均停留时间</strong></p>
<p><strong>a.每日任务</strong></p>
<ul>
<li>任务清单</li>
<li>宝箱或者额外奖励</li>
</ul>
<p>设计原因：Get Thing Done以及随机奖励的诱惑性</p>
<p><strong>b.提高活跃度——短期/一周、中期/一月、长期/季度</strong></p>
<p><strong>短期：</strong></p>
<p>限时活动：</p>
<p>设计原因：保证保底在线时间，拉动次日留存</p>
<p>倒计时奖励：</p>
<p>设计原因：损失厌恶，不确定性奖励诱惑</p>
<p>定时抽奖：</p>
<p>设计原因：损失厌恶，随机奖励的诱惑性</p>
<p>签到礼包：</p>
<p>设计原因：损失厌恶，保证上线次数</p>
<p>全民福利：</p>
<p>设计原因：诱发从众效应，刺激消费</p>
<p><strong>中期：</strong></p>
<p>超值月卡：</p>
<p>设计原因：承诺一致，行为修正认知</p>
<p>成长基金：</p>
<p>设计原因：承诺一致，存量绑架，损失厌恶</p>
<p>升级解锁礼包等：</p>
<p>设计原因：帮助玩家确立目标，树立动态目标</p>
<p><strong>长期：</strong></p>
<p>终身权益：</p>
<p>设计原因：承诺一致，制作存量，损失厌恶</p>
<p>动态目标：</p>
<p>设计原因：对抗成就感，奖励诱惑</p>
<p><strong>③新手引导</strong></p>
<p><strong>任务系统+有效引导：主线+挑战任务</strong></p>
<p>设计原因：get things done</p>
<p><strong>快速升级渠道引导</strong></p>
<p>设计原因：对下一次升级产生期待，数据的量化提升</p>
<p><strong>超额奖励发放</strong></p>
<p>设计原因：损失厌恶，养成大量消费习惯</p>

<h2 id="toc-3">三、后记</h2>
<p>我上一篇文章中曾提到，产品就是对现实世界的建模，而对于游戏，同样适用，甚至可以说游戏化的设计在我眼里更是来源于生活并且高于生活的。</p>
<p>游戏更像是人类需求中自我实现的层级，可以在虚拟世界得到我们在真实世界无法触及的感受，<strong>我们在游戏中认真的生存甚至达成精彩的表现，其实是我们的大脑在寻找下一注的多巴胺——人类最原始的动力来源</strong>。</p>
<p>所以在产品设计中，我们能在游戏化的设计中借鉴到哪些呢？可以一起探讨一下。</p>
