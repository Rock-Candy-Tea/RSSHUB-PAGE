<p><img data-action="zoom" class="size-full wp-image-782789 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/XCXQDwhPacAit9xRdfTh.png" alt referrerpolicy="no-referrer"></p>
<p>即使你没有听过川沙妲己，那你也一定听说过玲娜贝儿，之所以称她为川沙妲己，是因为迪士尼在上海浦东新区川沙镇，而玲娜贝尔是一只可爱又迷人的粉色小狐狸。</p>
<p><img data-action="zoom" class="wp-image-453686 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/Rsq0UlFO4MW9xmMDQxoy.gif" width="500" height="281" referrerpolicy="no-referrer"></p>
<p>粉嫩的毛发、滴溜圆的眼睛、宽阔柔软的耳朵，加上一条标志性的毛茸茸大尾巴，玲娜贝儿的预告片出来的第一天，就凭借人畜无害的外表吸粉无数。</p>
<p>作为迪士尼家族的新成员，玲娜贝儿一出道就直接包圆了娱乐圈的热搜，热度完全不亚于当红女明星，横扫 B 站、抖音短视频平台，玲娜贝儿的玩偶被黄牛哄抢，价格抬升 6 倍之高达到上千元，再一次拯救了迪士尼的营业额。</p>
<p><img data-action="zoom" class="wp-image-453687 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/JJYAZF6feDmXaOv6q5zz.jpg" width="500" height="282" referrerpolicy="no-referrer"></p>
<p>为什么人们会这样热衷于一个虚拟的 IP 人物形象？作为 IP 人物形象玲娜贝儿是怎么设计出来的？</p>
<p>接下来带着这样的问题我们一起来看下 IP 人物形象的设计都有哪些门道。</p>
<p>在讨论 IP 形象之前，我们先来看下 IP 是什么。</p>
<p>IP 是 Intellectual Property 的简写，翻译过来是知识产权的意思，在互联网语境下已经延伸出多种含义，可以说一千个人就有一千种关于 IP 的说法。</p>
<p>广义上的“IP”依托于内容存在于各个领域，包括概念 IP、小说 IP、形象 IP 和电影 IP 等任意一种文化创意形态。</p>
<p><img data-action="zoom" class="wp-image-453689 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/sEONP00ZxXVrO0Yx27no.jpg" width="550" height="309" referrerpolicy="no-referrer"></p>
<p>在有内容支撑并可被持续创作的情况下，一个价值观、一个人格都可以形成一个 IP。IP 的终极目标是追求价值和文化的认同，可以跨形态、跨时代、跨行业。</p>
<p>IP 本身其实是特别庞大且难被完全概括的一个存在，至今也还存在争议（尤其在法律层面）。但再庞大的概念，也总能被分类。大概收集了下资料，目前 IP 可以大致地分为形象化、文旅符号化、故事、世界观这四个类型。</p>
<p><img data-action="zoom" class="wp-image-453690 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/laEatAP1M9zoJs9q5vMd.jpg" width="600" height="337" referrerpolicy="no-referrer"></p>
<p>第一个类型形象化也就是我们所说的 IP 形象。</p>
<p>形象是将产品拟人化，并加入一些故事线，且能快速的让消费者记住的特点。IP 形象作为更为具象化的产物，能够帮助企业共建良好的品牌形象，扩展企业的盈利空间。当 IP 形象能够做到足够惹人喜爱时，它自身就会有超越品牌的个性魅力，可以反过来为品牌吸引新用户，这是一种营销玩法的升级。</p>
<p>比如比较经典和著名的 IP 就有小黄人、MM 巧克力豆小人、米其林、哆啦 A 梦，以及漫威、迪士尼家族等。而品牌中常见的就有京东 JOY 小狗、腾讯企鹅、美团袋鼠等等。</p>
<p><img data-action="zoom" class="wp-image-453691 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/qgQndoBlQaF8c624j9wN.jpg" width="549" height="308" referrerpolicy="no-referrer"></p>
<p>那对企业和品牌来说，打造 IP 需要从 IP 形象的设计开始。打造的结果，就是塑造一个大众喜爱的，且拥有长期生命力和商业价值的 IP 形象。</p>
<p>对 IP 形象有一定了解的人都会知道熊本熊，他作为熊本县的吉祥物，以其高度拟人化的形象加之熊本县营业部兼幸福部长的头衔，制造出各种有趣的故事，带动当地人民与之互动。</p>
<p><img data-action="zoom" class="wp-image-453692 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/1dJKJ9TwJDdmbgWVySEs.jpg" width="550" height="309" referrerpolicy="no-referrer"></p>
<p>一个吉祥物熊本熊的创意设计 IP，短短两年便为该地带来 12 亿美元的经济效益以及相当于 9000 万美元的广告和宣传效应，该县也从一个籍籍无名的农业城市，一举成名成为日本乃至世界著名的旅游大县。</p>
<p><img data-action="zoom" class="size-full wp-image-453694 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/AnOkmWyGwsPqg9Mo7Rcp.gif" width="435" height="249" referrerpolicy="no-referrer"></p>
<p>文旅符号就例如是，我们中国的长城、故宫，美国的自由女神，巴黎埃菲尔铁塔等，它们都可以作为 IP 文化符号来运营，从而产生商业价值。</p>
<p><img data-action="zoom" class="wp-image-453695 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/b1jac7eFwrHgBlPHnuaG.jpg" width="549" height="308" referrerpolicy="no-referrer"></p>
<p>就拿故宫来说，作为中国明清宫廷文化符号的故宫紫禁城，故宫是一个拥有 600 年历史的文化符号。</p>
<p>故宫拥有众多的皇宫建筑群、文物、古迹，也是中国传统文化最典型的象征，这也是如今故宫能成为文化领域超级 IP 的重要基础，故宫文创也随之“出圈”。故宫文创深度挖掘丰富的明清皇家文化元素，将有五千年历史故宫的建筑、故宫的文物以及背后的故事，融合现代人喜欢的时尚表达理念，最终打造出深受消费者喜爱的故宫元素文创产品。</p>
<p><img data-action="zoom" class="wp-image-453696 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/mWLYCAsvYXWYzYXsGf1X.jpg" width="550" height="309" referrerpolicy="no-referrer"></p>
<p>故事类 IP 就包括哈利波特、指环王、黑客帝国等内容作品，当然还有樱桃小丸子、蜡笔小新、奥特曼等等知名 IP。</p>
<p><img data-action="zoom" class="wp-image-453697 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/uiWeUhdQcU7NhmRzfm5Y.jpg" width="549" height="308" referrerpolicy="no-referrer"></p>
<p>故事 IP 往往也能够跨界、形成综合产业、并且在人们的内心形成一种“超级文化符号”，而迪士尼本质就是一个童话故事乐园，为了让消费者买单，其就一定要进行各种故事的创作。任何一个 IP 人物的诞生，都需要故事在其背后呈现。</p>
<p>所以，如果仅仅凭空去创作一个 IP 人物，却没有其故事，那么是无效的。任何 IP 的创作都需要一个故事在其中附加。当然，这个故事中，就必然会包含人物的性格特点、爱好等等相关联的因素。</p>
<p>世界观 IP 就例如像星球大战，权利的游戏，流浪地球，以及玩具总动员、疯狂动物城、寻梦环游记等等。</p>
<p><img data-action="zoom" class="wp-image-453698 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/CfslEbGXSHjwXtsY2OY5.jpg" width="550" height="309" referrerpolicy="no-referrer"></p>
<p>世界观 IP 也有人设和故事，但内容会更关注世界观本身，IP 内容的世界观构建占比较高。比如流浪地球就是从一个高概念的、极具脑洞的科幻设定出发：“如果太阳急速衰老膨胀，人类能否带着地球一起逃亡，去外星系寻找新的家园？”由此创建出一个独特的未来世界，一切事情、人物、场景皆因此而生。</p>
<p>那么在了解完 IP 的概念以及分类后，一个完整的 IP 形象设计的诞生需要经过怎样的流程呢？</p>
<p>IP 形象的创作流程不是线性的，而是环形，是一个不断探索和进化的过程。主要为策划、设计、应用、进化 4 大步骤。</p>
<p><img data-action="zoom" class="wp-image-453699 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/zdcaP7p2QQiLQ817RNRb.jpg" width="550" height="309" referrerpolicy="no-referrer"></p>
<p>第一部分 IP 策划包含 IP 背景调研、性格特征、世界观、个人故事。</p>
<p>那首先通过大量用户调研、公司、市场多个角度的信息收集，就可以确定出 IP 形象所传递的价值和受众，以及 IP 所要传达的核心理念同时也会综合考虑品牌调性和营销策略，决定 IP 的形象定位以及人设。</p>
<p><img data-action="zoom" class="wp-image-453700 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/gpus6wJjSKmPmMVtIVBx.jpg" width="550" height="309" referrerpolicy="no-referrer"></p>
<p>性格特征是将 IP 人性化的重要方法，人性化能够使 IP 具有独立人格和魅力，能够形成 IP 形象的名字、身份、性格、标签和口头禅等等。</p>
<p>例如大家可能熟悉的三只松鼠 IP，三只松鼠建立了一个萌宠的品牌形象和亲切的主人文化。他们是“松鼠小贱、松鼠小美、松鼠小酷”三个可爱形象，他们有不同的星座、爱好以及个性等特征，这样的品牌设定具有趣味性，能够快速吸引目标用户关注。</p>
<p>此外，三只松鼠以主人和宠物之间的关系，替代传统商家和用户之间关系，客服以松鼠口吻与用户交流：亲切称呼用户为主人。这样的沟通方式提升了用户的消费体验，打造超预期服务，增强用户粘性，品牌由此具备了立体形象，被赋予人格属性。</p>
<p><img data-action="zoom" class="wp-image-453701 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/99GmaXNcYU5iBFSM3PtJ.jpg" width="551" height="311" referrerpolicy="no-referrer"></p>
<p>世界观的设定则依托于一个文化母体，比如《哪吒之魔童降世》同时有封神和哪吒的经典神话，以及现代二次元新潮人类文化。</p>
<p><img data-action="zoom" class="wp-image-453702 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/MTled4nLOJTkxKETGTFH.jpeg" width="550" height="310" referrerpolicy="no-referrer"></p>
<p>又比如在漫威宇宙里，同时有神话、新科技、外星、当代传奇等多种文化母体存在，雷神是北欧神话的，钢铁侠和绿巨人是新科技的，银河护卫队是外星的，而美国队长则来源于二战，还有奇异博士有藏传佛教，等等。</p>
<p><img data-action="zoom" class="wp-image-453703 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/fdy8qQl8hMBqlITDcPIG.jpeg" width="550" height="310" referrerpolicy="no-referrer"></p>
<p>有了人性化和世界观，就能够产生故事，通过故事则能够体现出 TA 何种价值取向、人生态度和处世方式。</p>
<p>我们都知道，人类这个群体实际上是需要故事才能生存下来的，人类社会其实是一个想象的共同体。那么任何故事的创造就自然会吸引人们去阅读，不论这个故事是真是假，故事本身就是意义。那么这个时候，一个好的故事塑造就可以打动人，唤醒人的心智。</p>
<p>我们来看下玲娜贝儿的人设是什么样的。</p>
<p>在迪士尼讲述的故事里，达菲在森林里迷路后，是玲娜贝儿通过达菲身上的痕迹为线索，最终帮达菲找到了回家的路，在这个设定里，玲娜贝儿是个小小冒险家，她热爱大自然，日常拿着放大镜在森林里东看西看，又有着大大咧咧的个性，非常喜欢交朋友。</p>
<p><img data-action="zoom" class="wp-image-453704 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/jiu6YpMGqUO76DJWDYLU.gif" width="550" height="309" referrerpolicy="no-referrer"></p>
<p>第二部分则是开始进行 IP 形象设计，但设计不仅是简单的方向探索与草图绘制，在进行 IP 设计时，需要充分考虑形象的外观，适当的甚至艺术化的夸张抽象，让形象的性格特点立住人设。</p>
<p><img data-action="zoom" class="wp-image-453705 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/YQoiFaZiBdvGPe6rlInU.jpg" width="550" height="309" referrerpolicy="no-referrer"></p>
<p>首先，我们根据策划时确定的形象关键词做一些研究和构思来确定风格方向，然后通过协调的身体比例决定详细的造型人物设计。</p>
<p>所有正常的形象都由一个基础的头身比例得来，这是在画人体中最常用的「头身比」法则。一般来说，我们平时常见的成年人是 7-9 头身，青少年是 6-7 头身，孩童是 3-5 头身，Q 版一般是 2-3 头身，头身比越大，越成熟，越写实，反之亦然。</p>
<p><img data-action="zoom" class="wp-image-453706 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/BKRYfHcngHkdt4M9Os9U.jpg" width="550" height="725" referrerpolicy="no-referrer"></p>
<p>日本形象创作的总结图，基本的 2 头身，展现不同的身体画法，分别对应不同的特点。第一种是偏写实的风格，第二种是略微夸张的表现，第三种是在第二种的基础上更加夸张。</p>
<p>在身体比例的设计上大概有这样一个规律：没有脖子更可爱憨厚、重心低比重心高更低龄、手脚简化比手脚复杂更低龄。</p>
<p><img data-action="zoom" class="wp-image-453707 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/0sqtYoBfajXWhbY7LOxf.jpg" width="551" height="307" referrerpolicy="no-referrer"></p>
<p>同样的身体配上不同的面部，给人的感受也不一样，性格表现也会不一样。例如熊本熊的脸就是从一千多个面部组合中选出来的。</p>
<p><img data-action="zoom" class="wp-image-453708 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/oriKF6Cg9igLr8ZhfNLR.jpg" width="551" height="358" referrerpolicy="no-referrer"></p>
<p>同样的面部配上不同的身体，给人的感受也会不一样。不过身体比例的审美一直在变化，尤其是近两年盲盒大爆发。随着审美的变化，我们不再追求骨骼上的可爱，如今的很多潮玩重心都靠上，大长腿。可爱还是依旧的可爱。</p>
<p><img data-action="zoom" class="wp-image-453709 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/rdRAQYZe1KwfqGHq482f.jpg" width="551" height="162" referrerpolicy="no-referrer"></p>
<p>按照重心原则，手脚协调，脸部和身体的个性配搭来做形象，似乎不会有太大风险，但打破常规设计，制造一些有趣的特点，也往往会利于形象的传播，更容易被人们所接受。</p>
<p><img data-action="zoom" class="wp-image-453710 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/wrizgkQLL3Jbt4UmJK7e.jpg" width="549" height="260" referrerpolicy="no-referrer"></p>
<p>由日本漫画家田中克己推出的缘子小姐，一个看似正常的白领美少女，做出匪夷所思的动作，灵感就来自上班族的白领小姐。试想当你伏案工作的很长时间后，忽然抬头发现可爱的缘子小姐的话，想必非常的治愈吧。</p>
<p><img data-action="zoom" class="wp-image-453711 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/jboZVJO7lqswujLY4m0i.jpg" width="550" height="294" referrerpolicy="no-referrer"></p>
<p>再比如盲盒届的初代萌神，molly，吸引大家不停买买买的是不停变化的任务装饰和小部件，非常精致。</p>
<p><img data-action="zoom" class="wp-image-453712 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/cazgCvA1QGRtOs94I6tQ.jpg" width="550" height="273" referrerpolicy="no-referrer"></p>
<p>再比如 KAWS 的联名千千万，别说服饰变化，就连整个造型都会改变，唯一不变的就是它标志性的 xx 眼睛。</p>
<p><img data-action="zoom" class="wp-image-453713 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/tGyCEQaRHnAjTVH2EJQG.jpg" width="551" height="244" referrerpolicy="no-referrer"></p>
<p>当下，IP 形象已经成为品牌视觉资产的重要组成部分，因此需要根据品牌特性将 IP 进行符号化设计，抽象出 IP 形象的外形特征，来应用于不同物料的延展和品牌的视觉符号。</p>
<p>例如我们看到一大两小三个圆组成的图像就很容易联想到米奇，这就是米奇非常核心的识别造型。</p>
<p><img data-action="zoom" class="wp-image-453714 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/mVvwh84QYaH6PUp06aHZ.jpg" width="550" height="312" referrerpolicy="no-referrer"></p>
<p>便于记忆的同时也能很方便的运用在各个场景中。而一个成功的角色塑造，外形上需要有让人记忆深刻的视觉锤，反复地以一种简单却概括、符号化的标志敲打人们的视觉记忆点，这种强符号化的标志通常人们眼睛移开 10 秒，再回想一下也能记住一个大概。</p>
<p>颜色对 IP 形象的重要性也不言而喻，传统的 IP 形象色彩搭配通常不会超过三种。这是因为如果在 IP 形象的颜色过于复杂，是不便于用户记忆的。</p>
<p><img data-action="zoom" class="wp-image-453715 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/1qrbozmfOigLkm6u0p8N.jpg" width="549" height="265" referrerpolicy="no-referrer"></p>
<p>随着 IP 的设计推进，IP 形象延展以及运营的设计规范也是 IP 形象的重要设计工作，IP 素材延展则是根据故事和世界观为形象设计匹配的动作和场景等等。最终目的就是丰富形象，使角色形象能够立体起来。如果说外观是 IP 角色吸引用户的第一个要素，那么真正能让用户为之动容的，则是角色背后妙趣横生的背景故事。</p>
<p>例如 LINE FRIENDS IP 讲述了布朗熊和可妮兔的爱情故事。</p>
<p><img data-action="zoom" class="wp-image-453716 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/Kj4HuqOGtt7mF9aNqZDC.jpg" width="552" height="311" referrerpolicy="no-referrer"></p>
<p>IP 形象也不再局限于插画或者动画的表达，IP 形象也将应用在品牌传播、商业合作等各个方面，因此在物料设计上也可以进一步推进 IP 和 VI 二合一视觉体系。</p>
<p>如果大家对国内外一些 IP 运营公司有了解的话，应该会知道他们在日常运营中会涉及到一个重要的设计工作，那就是 IP 图库开发以及授权规范。</p>
<p>其中以迪士尼角色图库为代表，针对米奇这一形象就开发了数不胜数的图库。</p>
<p><img data-action="zoom" class="wp-image-453717 aligncenter" src="https://image.yunyingpai.com/wp/2022/04/gf9hTonoK3qEb1FZmh8f.jpg" width="552" height="291" referrerpolicy="no-referrer"></p>
<p>IP 图库同样是为了规范、保证品牌方在使用 IP 时能够产出统一标准的视觉，使得 IP 不会因为二次创作模糊掉。</p>
<p>以上提到的 IP 设计流程，只是成功 IP 形象的基本，IP 形象想要深入人心还需要核心的精神元素。</p>
<p>当我们回望历史上一些知名 IP 时，我们会发现他们的成功都是顺应时代发展的，例如日本动漫经典之作 EVA，将美少女元素和机器人科幻元素连结在一起固然是新颖的视觉元素，但其内核还是深深依附于时代背景的思考，诞生于日本的泡沫经济阶段，年轻人对社会充满失望。于是 EVA 把世界观植入未来，表达一种鼓励和憧憬，希望人们在即将到来的新世纪，可以战胜自己心中的恐惧。</p>
