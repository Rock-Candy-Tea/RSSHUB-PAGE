<h2 id="toc-1">一、产品功能概述</h2>
<p>微信阅读app主要实现了用户找书、读书、社交这三方面功能。</p>
<ol>
<li>找书：查看书架，用户自行搜索、朋友在读和系统推荐4个途径；</li>
<li>读书：可设置阅读页面进行个性化阅读风格；阅读页面还原纸质书籍质感，强调精致排版；阅读时可划线、写想法和分享，主打精心打磨的阅读体验；</li>
<li>社交：通过朋友在读书籍/收藏书单推荐、分享/评论/点赞/组队、个人主页设置、阅读时长比拼等途径实现。</li>
</ol>
<h2 id="toc-2">二、产品结构概览</h2>
<p>APP包含阅读、书架、发现、我，共4个页面。其中，阅读页主要实现找书和参加活动功能，书架页主要实现看书功能，发现页主要实现社交和参加活动功能，我页主要实现信息查看（个人信息、购买信息及阅读信息）和设置。</p>

<h2 id="toc-3">三、结构分解与分析</h2>
<h3>3.1 阅读</h3>
<p>阅读作为app的首页，主要提供找书籍的功能，分为搜索模块、阅读历史模块、卡片信息流模块、福利场入口4个模块。其中搜索模块、阅读历史模块、卡片信息流模块依次排列在页面，福利场入口通过在阅读页下拉进入。</p>
<p>各模块功能如下：</p>
<p><strong>（1）搜索模块：包含搜索栏、书城入口和分类栏</strong></p>
<ul>
<li>点击搜索栏->弹出键盘，并显示热门搜索和猜你喜欢</li>
<li>点击书城->进入书城页</li>
<li>左右滑动分类栏->展示分类栏</li>
<li>点击分类栏某分类->进入书城某分类页</li>
</ul>
<p><strong>（2）阅读历史模块：包含阅读时间模块和近期阅读模块</strong></p>
<ul>
<li>点击阅读时间->进入阅读时长页</li>
<li>点击近期阅读->进入书籍阅读页，定位上次阅读位置</li>
</ul>
<p><strong>（3）卡片信息流模块：该模块包含若干卡片信息流，通过下滑操作，以卡片信息流为载体，分为活动运营、阅读推荐、福利场3类主题</strong></p>
<p><img data-action="zoom" class="aligncenter" src="http://image.woshipm.com/wp-files/2021/12/z6yHZx10RCFfumkhZwLX.png" alt width="450" height="385" referrerpolicy="no-referrer"></p>
<p>通过3类共15种卡片，微信读书使书籍信息和活动福利更精准地触及目标用户，达到书籍推荐，提高用户活跃度的目的。</p>
<p><strong>（4）福利场模块：该模块通过在阅读页进行下拉操作进入，以卡片信息流的形式展示活动详情</strong></p>
<p>卡片包含活动名称和活动入口，共包含每日一答、组队抽终身无限卡、邀请微信朋友、周四集赞得12天联名卡、TOP 200/飙升分享得1天无限卡、翻一翻、（每周一）无限会员日、读书小队8个活动。活动机制为用户通过参加游戏、邀请、组队、分享、点赞，获取无限卡、书币或免费书，以福利驱动最终达到拉新促活的目的。</p>
<ul>
<li>点击每日一答->进入微信小程序界面</li>
<li>点击组队抽终身无限卡->进入活动介绍页面，点击邀请好友加入队伍->进入微信转发页</li>
<li>点击邀请微信朋友->进入微信转发页</li>
<li>点击周四集赞得12天联名卡->进入活动介绍页面，点击分享到微信群/朋友圈->分别进入微信转发页和朋友圈页面</li>
<li>点击TOP 200/飙升分享得1天无限卡->进入活动介绍页面，点击分享->进入微信转发页</li>
<li>点击翻一翻->进入活动介绍页面，点击任一卡片->弹出弹窗，获得赠书</li>
<li>点击（每周一）无限会员日->进入活动介绍页面，点击购买/抽奖->进入购买会员页面</li>
<li>点击读书小队->进入微信小程序界面，界面包含活动介绍和活动入口，点击活动入口->进入微信转发页</li>
</ul>
<p>页面详情结构如下：</p>
<p><img data-action="zoom" class="aligncenter" src="http://image.woshipm.com/wp-files/2021/12/8EzLTcqJsLeBrtzgG0k8.png" alt width="599" height="1254" referrerpolicy="no-referrer"></p>
<ul>
<li>页面在顶部提供书籍搜索入口，有输入关键字、书城内查找、按类查找3种方式，该部分在页面占比较小；</li>
<li>中上部提供本周阅读时间和近期阅读的书籍入口，主要起到让用户迅速掌握自己阅读状况，提供书籍阅读快捷入口的作用，该部分的页面占比也较小；</li>
<li>页面中心为一张随机推荐卡片，该卡片占据页面最优位置，最大占比并选用最大字体，使用户注意力集中于此。</li>
</ul>
<p>通过页面不同模块位置设计及占比分配，可推测微信读书app<strong>在用户读书前找书的环节更倾向于通过系统推荐、朋友在读、活动推荐的方式给用户展示书籍</strong>，帮助寻找书籍，而不是让用户自行查找。</p>

<p><strong>3.1.1 书城</strong></p>
<p>书城页主要实现搜索书籍、分类查找书籍、推荐书籍的功能，可划分为搜索模块、分类模块、推荐模块3个模块。</p>
<p>各模块功能如下：</p>
<p><strong>（1）搜索模块：包含摇一摇和顶部搜索框</strong></p>
<ul>
<li>点击搜索栏->弹出键盘，并显示热门搜索和猜你喜欢</li>
<li>点击摇一摇->终端有震动音效，并弹出一张推荐卡片，卡片展示书籍封面、作者、简介，和“立即阅读”入口</li>
</ul>
<p><strong>（2）分类模块:划分为8大分类（男生小说、女生小说、听书、纸书、XX月新书、分类榜、书单、免费书）</strong></p>
<p>点击某分类入口->进入该分类详情页</p>
<p><strong>（3）推荐模块：该模块用2个banner进行区间划分，隔离出3个推荐书籍/书单的信息流梯队</strong></p>
<p><img data-action="zoom" class="aligncenter" src="http://image.woshipm.com/wp-files/2021/12/RzX0WIQJxJCv7h0azIpA.png" alt width="449" height="221" referrerpolicy="no-referrer"></p>
<ul>
<li>点击“换一批”->该子模块中内容更新</li>
<li>点击“查看全部”->进入该子模块</li>
<li>子模块通过右滑操作，展示下一屏</li>
</ul>

<p>书城页面以banner进行模块区隔，划分出搜索模块、分类模块、推荐模块3个模块。整体界面呈现总-分的逻辑，搜索栏和分类栏占据顶部位置，占据较小页面；推荐模块由各种主题的展示板组成，占据页面中心位置，通过下滑操作进行浏览页面。</p>

<h3>3.2 书架</h3>
<p>书架页面主要提供<strong>搜索书籍、登录电脑端/墨水屏端、导入书籍、查看并管理书架/书单、阅读书籍</strong>5大功能，可划分为顶部功能模块、书架管理模块、书单管理模块。</p>
<p>各模块功能如下：</p>
<p><strong>（1）顶部功能模块：包含搜索框、＋号键和分区选择</strong></p>
<p>其中搜索书籍结果优先展示书架内搜索结果，再展示书城内搜索结果；＋号键包含登录电脑/墨水屏版，从电脑/iCloud/微信导入文件，书城入口的功能；分区选择包含二级选择，通过一级分类选择进入书架或书单，通过二级分类进入更详细的书籍分区</p>
<ul>
<li>点击搜索框->弹出弹出键盘，并显示热门搜索和猜你喜欢</li>
<li>点击＋号键->弹出弹窗，显示功能入口</li>
<li style="text-align: left;">点击电脑/墨水屏版->弹出扫码页面</li>
<li style="text-align: left;">点击从电脑导入->弹出扫码页面</li>
<li style="text-align: left;">点击iCloud->弹出最近浏览文件界面</li>
<li style="text-align: left;">点击从微信导入->进入微信转发界面，点击某对话框/人员->进入文件记录页</li>
</ul>
<p><strong>（2）书架管理模块：包含书架分类，选择书籍进行操作，查看个人主页的功能</strong></p>
<p>其中，书架分类包含默认、更新、进度、购买、分类5个分类方式。选择书籍后，可进行私密阅读，下载到本地，添加到书单，置顶，书籍分组，移出书架6种操作</p>
<ul>
<li>点击书架/书单，进行书架管理模块和书单管理模块转换</li>
<li>点击默认/更新/进度/购买/分类,或左右滑动，书架按照不同分类方式展示书籍</li>
<li>点击选择->可对书籍进行批量选择，执行操作</li>
<li>点击查看我的个人主页->进入我的个人主页</li>
</ul>
<p><strong>（3）书单管理模块：包含书单分类、新建书单、查看个人主页的功能</strong></p>
<p>其中，书单分类包含我的书单和收藏的书单两大类。</p>
<ul>
<li>点击新建书单，可以对书单进行命名，描述，添加书籍</li>
<li>点击书架/书单，进行书架管理模块和书单管理模块转换</li>
<li>点击我的书单/收藏的书单，或左右滑动，展示书单</li>
<li>点击新建->进入新建书单页</li>
<li>点击查看我的个人主页->进入我的个人主页</li>
</ul>

<p><strong>3.2.1 阅读页</strong></p>
<p>阅读页的书籍包含<strong>扉页、其他信息页（版权信息、寄语…)、正文、读完页</strong>。主要实现用户了解书籍，阅读书籍，并对书籍打分的功能。</p>
<p><strong>（1）扉页</strong></p>
<p>主要介绍书籍名称、作者、推荐值、阅读人数、书籍字数等信息。</p>
<p><strong>（2）其他信息页</strong></p>
<p>针对出版书籍，提供了版权信息、寄语、导语等页面。</p>
<p><strong>（3）正文</strong></p>
<p>提供划线/查看热门划线句、写想法、分享/分享书签、页面显示设置4个主要功能，其他附加功能包括查看章节、查询、进入圈子、听书等功能。</p>
<ul>
<li>划线/查看热门划线：针对微信图书中书籍大量为出版书籍，具有高质量的特点，匹配了其他阅读app不具有的划线功能和查看热门划线句的功能，并且该功能在我页完成了笔记搜索和批量复制功能，完成了较好的逻辑闭环；</li>
<li>写想法：该功能起到批注的作用；</li>
<li>分享/分享书签：针对微信读书是基于社交的阅读app，增加了分享到微信朋友圈和分享书签（图片）的形式。分享到朋友圈仅需一步操作，分享操作简单；书签以黑底金字，样式简单大方，增加用户分享欲望。该功能较好的完成了社交功能；</li>
<li>页面显示设置：页面显示设置主要囊括调节阅读进度、是否开启自动阅读、阅读背景、页面亮度、字体大小、边行距、字体等。其中，调节阅读进度、自动阅读速度、页面亮度时，采取拖拽进度条（而不是定制分级）的形式，使读者找到更匹配的“度”，打造个性化阅读。</li>
</ul>
<p><strong>（4）读完页</strong></p>
<p>提供阅读记录信息和他人书评。阅读信息包含读完日期、阅读本书时长、和此书是用户阅读完的第X本书，且可以图片形式分享阅读成就。</p>

<ul>
<li>阅读页的页面设置做到了还原纸质书籍的质感，包括书籍版权页、彩色插图、文字排版等，提高用户阅读体验。</li>
<li>阅读页的功能做到了功能完善、重点突出、页面清晰。</li>

<h3>3.3 发现</h3>
<p>发现页的主要功能是社交和参加活动，分为社交模块和活动模块。</p>
<p>各模块功能如下：</p>
<p><strong>（1）社交模块：通过查看朋友的想法和小圈子，分别对标熟人社交和陌生人广场式社交</strong></p>
<p>a）朋友的想法：主要实现发布想法、浏览朋友想法、和朋友互动(转发、评论、点赞)的功能。页面包含右上角发想法键和浏览区;</p>
<p>b）小圈子：主要实现搜索圈子、发布想法、浏览想法、互动的功能。</p>
<p>“小圈子”模块内交互:</p>
<ul>
<li>点击小圈子->进入该模块内</li>
<li>点击发布想法->进入发布想法页面</li>
<li>点击他人想法->进入想法详情页</li>
<li>点击圈子宇宙->进入圈子集合页</li>
</ul>
<p><strong>（2）活动模块：包含免费领书，福利场和微信听书</strong></p>
<ul>
<li>点击免费领书->进入页面与在阅读页的卡片信息流中的”免费读“相同；</li>
<li>点击福利场->进入页面与阅读页下拉的“福利场“模块相同；</li>
<li>点击微信听书卡片->获取免费听书节目->进入页面与阅读页的卡片信息流中的”微信听书馆“相同；</li>
</ul>
<p>可见该活动模块为选取阅读页中个别主推的卡片信息流，单独设置入口。</p>

<p>发现页面主要有顶部导航栏和页面的卡片入口组成，导航栏包含摇一摇功能，卡片入口则包含互动类入口，如朋友的想法、小圈子、活动。</p>

<p><strong>3.3.1 朋友的想法</strong></p>
<p>页面包含右上角发想法键和浏览区，主要功能为<strong>写想法和浏览朋友想法并互动</strong>。该模块属于熟人社交，通过熟人效应，增加用户了解一本新书的意愿，提高用户粘性。</p>
<p><strong>（1）写想法</strong></p>
<ul>
<li>写想法的场景：包含阅读时的划线/划线+写想法，阅读后评分/评分+书评，在朋友的想法页面写想法，在圈子页写写想法同步到朋友的想法；</li>
<li>写想法的功能：正文撰写、添加标题、表情、引用书籍、文字格式整理；</li>
<li>想法的权限：公开、仅关注的人可见、私密；</li>
</ul>
<p><strong>（2）页面交互</strong></p>
<ul>
<li>点击写想法->进入写想法页面</li>
<li>点击他人想法->进入想法详情页</li>
<li>点击他人引用书籍->进入书籍阅读页</li>
</ul>

<p><strong>3.3.2 圈子</strong></p>
<p>圈子是针对陌生人社交的新功能，形式类似于社区。圈子的设立主题较广泛，不局限于书籍，每个圈子由圈主和成员组成，圈子里的想法包含用户昵称，正文，图片，话题4个元素。</p>
<p>圈子模块包含顶部导航区和浏览区，主要实现了发现圈子，加入圈子，发表想法，浏览想法并互动的功能。</p>
<p><strong>（1）导航区：导航区包含三个功能区，右上角的查看圈子键、写想法键和下方的我的圈子入口栏</strong></p>
<ul>
<li>查看圈子：该功能键提供查看全部圈子、我加入的圈子、我发的帖子；</li>
<li>写想法键：该圈子想法可添加话题，选择圈子；</li>
<li>我的圈子入口：提供我的圈子的快速入口，按照圈内帖子更新时间最近排列。</li>
</ul>
<p>该模块通过左右滑动进行展示用户加入的所有圈子</p>
<p><strong>（2）浏览区：浏览加入的圈子的想法和热门话题，可对他人想法进行转发、回复和点赞。浏览区的推荐排序的权重高评论>高赞>最近</strong></p>
<ul>
<li>写想法的场景：进入圈子、新建想法；</li>
<li>写想法的功能：添加话题、添加圈子、同步到朋友的想法；</li>
</ul>
<p>交互：</p>
<ul>
<li>点击发想法人的头像->进入他人个人主页</li>
<li>点击想法->进入想法详情页</li>
<li>点击分享->可分享至微信、朋友圈</li>
<li>点击话题->进入此话题的想法页（不限圈子主题）</li>
<li>点击换一换->更换3个热门话题</li>
</ul>
<p>页面详情结构如图：</p>
<p><img data-action="zoom" class="aligncenter" src="http://image.woshipm.com/wp-files/2021/12/Ab0WkJf3dPkb7iySJ9b5.png" alt width="450" height="436" referrerpolicy="no-referrer"></p>
<p>页面导航栏起到找圈子，进圈子的功能。浏览区起到看想法，找合适想法的功能。</p>
<p>目前，圈子功能上线时间较短，在功能逻辑和用户体验上有较大进步空间，比如：</p>
<ul>
<li>发现圈子的途径里只有发现推荐的圈子，用户无法进行搜索。</li>
<li>用户发表圈子想法的场景只有在进入圈子后新建，该场景需要用户发布意愿较高；可以打通多个场景，比如在阅读时进行划线分享/书签分享，降低用户发帖思考成本和发帖时间。</li>
<li>目前用户写想法时，全部依靠自身能力完成想法内容，可适当通过文字识别和图片识别等方式推荐用户契合的圈子和话题。</li>
</ul>

<h3>3.4 我</h3>
<p>我 的核心功能是查看或更改个人主页和设置，查看购买情况、互动情况和阅读成果。主要包括<strong>顶部导航栏、个人主页模块、购买模块、朋友互动模块、阅读成果模块</strong>5个模块。</p>
<p><strong>（1）顶部导航栏</strong></p>
<p>主要包括左侧的消息和右侧的设置。消息可以查看收到的转发、评论、赞及私信。设置可以设置阅读页面、权限和登录状态。</p>
<ul>
<li>阅读页面的设置：包括锁屏、横屏、隐藏他人想法、缩进、翻页方式、深色模式，旨在提升用户阅读体验；</li>
<li>权限管理：包含隐私、青少年模式、通知、缓存4方面的管理。</li>
</ul>
<p><strong>（2）个人主页模块</strong></p>
<p>主要包含头像、签名、自我介绍和个人主页入口。个人主页则提供详细的个人信息，包含个人信息、互动信息、阅读信息及评论信息。</p>
<p><strong>（3）购买模块</strong></p>
<p>主要提供账户和无限卡的信息管理和购买入口，及购物车和订单状况的查看。</p>
<p><strong>（4）朋友互动模块</strong></p>
<p>主要展示读书排行榜和微信好友关注情况。</p>
<p><strong>（5）阅读成果模块</strong></p>
<p>主要包括勋章、笔记、阅读记录和订阅3方面总结。</p>
<ul>
<li>勋章：颁发指标包括阅读天数/时长、阅读/读完书籍数、获赞量及想法发布量；</li>
<li>笔记：主要功能在于<strong>快速查看并导出阅读时的划线句</strong>，功能实现逻辑为自行搜索关键词和按照书籍-章节分类，可实现批量复制；</li>
<li>阅读记录和订阅：通过细致分类达到快速定位书籍的目的，如书籍按照阅读和订阅划分，阅读记录通过浏览历史、赞过、在读和读完来划分，订阅划分为订阅的书籍和订阅的作者。</li>
</ul>
<p>页面详情结构如图：</p>
<p><img data-action="zoom" class="aligncenter" src="http://image.woshipm.com/wp-files/2021/12/KNXK9Ny4dQhB0mpnin7a.png" alt width="600" height="1265" referrerpolicy="no-referrer"></p>
<p>页面的信息以卡片的形式进行呈现，通过将相似主题（购买/互动/阅读）的信息卡片进行整合，并合理规划不同分区，让用户在视觉上形成页面区隔。该方法使得页面展示的信息和入口繁多，但页面整体简洁大方，用户使用起来较迅速。</p>

<p><strong>3.4.1 个人主页</strong></p>
<p>个人主页是用户通过<strong>权限设置</strong>，将个人信息、阅读信息、评论观点3部分信息<strong>选择性展示</strong>给微信其他用户查看，也是用户<strong>回顾阅读成果</strong>的板块，满足用户的社交需求和满足感需求。</p>
<p>个人主页由顶部导航栏、个人信息模块、阅读信息模块和评论信息模块组成。</p>
<p><strong>（1）顶部导航栏：分享功能</strong></p>
<ul>
<li>分享形式：4类</li>
</ul>
<p>a) 与微信端打通，包括分享到微信对话，微信朋友圈和同步微信状态；</p>
<p>b) 保存到相册；</p>
<p>c) 微信读书内分享给书友；</p>
<p>d) 分享到其他app。</p>
<ul>
<li>分享内容：3种</li>
</ul>
<p>a) 个人信息，包括头像，昵称，自我介绍</p>
<p>b) 阅读成果，包括读完本数，获赞数，累计阅读时长，输出想法数。并且将阅读成果的抽象数据转化成生活里的具体单位，形象且充满趣味性，激励用户阅读及分享的欲望。</p>
<p>c) 最新阅读书籍，包括最近动态，体现用户互动的即时性，使得用户更想分享以获得同时在读的书友。</p>
<p><strong>（2）个人信息模块</strong></p>
<p>包含头像，昵称，签名，自我介绍，阅读时长，获赞数，关注数，勋章。</p>
<p><strong>（3）阅读信息模块</strong></p>
<p>包含书架和书单。与书架页中的书架和书单进行区隔的是，此处的内容仅展示公开书籍。该设计便于用户进行主页打造和隐私保护。</p>
<p><strong>（4）评论信息模块</strong></p>
<p>包含点评与想法。</p>

<h2 id="toc-4">四、总结</h2>
<p>通过对微信读书4个主页面进行拆解，从用户找书、读书、社交3个主要场景进行分析。</p>
<ul>
<li>找书：基于社交（好友在读）、历史数据、活动运营，以卡片信息流的形式推荐书籍/书单，较好的为用户提供合适的书籍；</li>
<li>读书：提供精致的书籍排版，通过拖拽进度条精确定制阅读设置。提供划线和批量复制划线句的功能，较好地复原了纸质书的正式感和阅读习惯，且操作流程简单；</li>
<li>社交：创造了查看朋友在读书籍，写想法，发圈子，比拼时长，参与活动，将动态分享到微信状态等社交场景，增加了用户的互动性和用户粘性。</li>
</ul>
<p>总的来说，微信读书提供了完备的推荐图书功能、阅读风格定制功能和社交功能，用户路径流畅，功能逻辑清晰，能提供高质量的阅读体验。</p>
