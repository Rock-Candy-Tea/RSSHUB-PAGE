<blockquote><p>编辑导语：开发新的硬件产品时，有时会外包给现成的产品设计公司，在这个过程中，如何选择外包公司至关重要。本文作者从外包五要素和外包地图两大方面，分析选择外包公司时应该注意的点，感兴趣的小伙伴们一起来看一下吧。</p></blockquote>
<p><img data-action="zoom" class="aligncenter size-full wp-image-5390337" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/04/d2aq7kF27XRCL4xTEZkm.jpg" alt width="900" height="420" referrerpolicy="no-referrer"></p>
<p>如果你正在或准备开发新的硬件产品，尤其是在你不具备技术意识和能力的情况下，你可能会认为使用现成的产品设计公司是最好的选择。</p>
<p>但是，在外包产品开发时，有五个因素需要考虑。</p>
<h2 id="toc-1">一、外包五要素</h2>
<h3>1. 保持对产品的控制，并不断学习</h3>
<p>当你将产品移交给设计公司时，你将会丢失很多控制权。</p>
<p>你想出了这个创意，开始热情地追求自己的理想，并且已经准备好要将其付诸实践。</p>
<p>你需要指导项目向前推进，并需要始终保持循环跟进，这并不意味着你必须自己做所有事情，但是你需要保持对每个开发步骤的控制。</p>
<p>如果你在这个关键的时刻，从开发流程中抽身而出，你将无法实现你的预期。</p>
<p>也就是说，如果你将所有内容移交给设计公司，你将很难理解整个开发过程，也不会建立未来所需的技能。</p>
<h3>2. 实施制衡，以减少风险</h3>
<p>一些设计公司拥有许多检查和纠正问题的工程师，他们可以执行产品开发的每个步骤，但是，此类服务级别的价格也会很高。</p>
<p>因此，经验丰富的自由职业者也是很多初创企业的首选，但这也意味着，你可能需要依靠一个人或几个不同领域的人来进行产品开发，并且没有人来检查他们的工作。</p>
<p>无论工程师的技能高低如何，当设计的复杂性增加时，出错的风险也会增加，此时你就必须使用审核机制来降低风险，你可以通过雇用多个同一领域的自由工程师来解决此问题。</p>
<p>如果医生告诉你需要进行大手术，你可能会寻求其他意见，同样，对产品的设计发表第二意见会降低开发过程中的风险。</p>
<h3>3. 聘请专业工程师</h3>
<p>当你想将新的硬件产品推向市场时，通常需要许多专业的工程师。</p>
<p>对于电子硬件产品，可能需要雇用以下类型的人员：</p>
<ul>
<li><strong>电路设计工程师：</strong>设计电路原理图</li>
<li><strong>Layout 程师：</strong>根据原理图设计印刷电路板（PCB）</li>
<li><strong>固件/测试工程师：</strong>对微控制器进行编程并测试电子设备</li>
<li><strong>移动应用程序开发人员：</strong>开发自定义移动应用程序</li>
<li><strong>工业/ 3D设计工程师：</strong>开发外壳或配件</li>
<li><strong>机械工程师：</strong>如果产品需要运动部件，则需要机械工程师</li>
</ul>
<h3>4. 不要把所有的鸡蛋都放在一个篮子里</h3>
<p>当你将项目交由一家设计公司或一名自由工程师负责，这也会以其他方式增加你的风险。</p>
<p>例如：</p>
<ul>
<li>很多自由职业者会在业余时间做项目，因此，如果他们在一段时间内要忙于其他事情，你的项目大概率会延期，同时，他们可能会随身携带项目中所有有价值的细节。</li>
<li>当设计公司控制项目的各个方面时，对你来说将是一个重大的风险。</li>
</ul>
<p>当你雇用多名工程师时，你的风险相对会更小一些。</p>
<h3>5. 管理产品开发以节省成本</h3>
<p>聘请单个工程师或公司进行产品开发也会增加成本，工程师的费率根据其专业而有所不同。</p>
<p>例如，电路设计人员可能比固件编程人员或 PCB 布局设计人员收取更多的费用。</p>
<p>当你雇用一名工程师或一家设计公司时，他们可能会向你收取最高的费用。</p>
<p>要实现成功的产品开发过程，你必须超越自己的舒适区并不断学习，不要忽视这个宝贵的机会来发展新技能，并挑战自己的恐惧。</p>
<h2 id="toc-2">二、外包地图</h2>
<p>如果你已经决定将某些产品开发或业务流程外包出去，那么如何决定外包哪些流程以及使用哪些合作伙伴呢？</p>
<p>外包地图可以帮助你为团队选择合适的资源，该工具根据战略和技术两个维度来评估你的选择，通过确定满足需求的最佳技能矩阵来优化组织任何级别的决策。</p>
<h3>1. 外包地图</h3>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/04/KprS0U1DNkx3jsa881az.png" alt width="726" height="635" referrerpolicy="no-referrer"></p>
<p>工具优点：</p>
<ul>
<li>提供选择合作伙伴的清单</li>
<li>通过确定给定项目规模的最佳外包方法来降低风险</li>
<li>提供根据需求选择外包合作伙伴的客观标准</li>
</ul>
<p>该工具解决了哪些业务问题：</p>
<ul>
<li>这个工具提供了一个路线图，通过充分利用内外部资源快速为你的项目配备人员</li>
<li>该地图消除了过程中的一些猜测，并通过提供有意义的建议将风险降至最低。最后，它有助于为管理层提供理由，在必要时加快审批速度</li>
</ul>
<h3>2. 应用外包地图</h3>
<p>1~3 级别分别表示：</p>
<ul>
<li>1 = 无或低</li>
<li>2 = 有时或中等</li>
<li>3 = 是或高</li>
</ul>
<p>对于下面的每个问题，请根据1~3的级别分配一个数值。</p>
<p>战略重要性：</p>
<ul>
<li>这个过程是战略的关键组成部分吗？</li>
<li>后续流程会再次使用它吗？</li>
<li>它是产品差异化的重要组成部分吗？</li>
</ul>
<p>技术难度：</p>
<ul>
<li>它在技术上是否复杂，是否使用尖端组件？</li>
<li>它是否依赖于内部专利或深厚的公司专有技术？</li>
<li>您想捕获 IP 以备将来使用吗？</li>
</ul>
<p>对战略和技术维度的答案求平均，以产生从1~3的两个数字，通过使用平均值作为 X 和 Y 的坐标绘制圆心，评级结果将显示在外包地图上。</p>
<p>你应该将此过程单独应用于你准备外包的不同类型的工作，例如，外包软件开发将生成与外包薪资管理不同的地图。</p>
<p>例如：一个团队希望为新产品外包一段代码，经过讨论，他们决定该计划具有低战略重要性（战略轴上的 1）和中等技术复杂性（技术轴上的 2）。</p>
<p>该工具使他们相信，通过使用自由职业者或众包服务，他们会找到最佳的外包合作伙伴。</p>
<p>你可以通过考虑以下问题来增强外包地图，这些问题可以帮助你评估地图建议的类别内的特定供应商。</p>
<p>同样，对每个答案使用 1~3 的级别，但不要平均答案。</p>
<p>相反，使用它们来优化外包地图中的选择，以便你可以在具有竞争力的选项之间进行选择。</p>
<ul>
<li>需要非常大的努力？</li>
<li>需要长期努力？</li>
<li>开发成本重要吗？</li>
<li>需要说英语吗？</li>
<li>受到内部项目管理薄弱的挑战？</li>
<li>上市时间短很重要吗？</li>
</ul>
<h3>3. 你还应该知道什么</h3>
<p>虽然不可能将这个复杂的决定简化为一套黑白规则，但外包地图将帮助你快速做出决定。</p>
<p>同时，你还应该考虑内部文化、对知识产权的敏感性、成本和先前的经验等因素。</p>
<p>如果你不熟悉外包或远程开发，就会存在很多问题会影响你考虑的资源质量，包括沟通问题和资源周转问题，外包地图将有助于将这些风险降至最低。</p>
<p>如果你在小型项目上使用低成本资源，请同时雇用两个人来并行完成一项小型的、有时限的任务，然后选择提供更高质量输出的那个方案。</p>
