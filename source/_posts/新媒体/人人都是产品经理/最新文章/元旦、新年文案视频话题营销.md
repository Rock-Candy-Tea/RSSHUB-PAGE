<h2 id="toc-1">一、崭新/ 第一</h2>
<h3>关键词：新年、期待、开始</h3>
<p>话题文案可以是这样的：<strong>《愿你有一个新的start，新的smart》</strong>，创意动图上，车轮驶过大大的“t”变成了“m”巧妙的将新年开始与品牌连接在一起。还有：</p>
<ul>
<li><strong>《新愿已定，keep going》</strong></li>
<li><strong>《这是最好的开场白，微笑和掌声正盛装赶来》</strong></li>
</ul>
<p>买了新台历了吧，打开看看，<strong>《你好，2022日历的第一页》。</strong>励志鼓励的文案当然少不了，尤其是这么难得2021，<strong>《多少不为人知的奋力前行和惊心动魄，成就此刻泰然自若，愿你新年笃定自信，舞步之间自成独一无二的精彩》。</strong></p>
<p>给每一个前行的人<strong>《有低谷就有高峰，新年继续拾光逐梦》</strong>。在有些人看来这些矫作了，还是来点直接的，<strong>《2022，新的一年，继续搞钱》。</strong></p>

<p><strong>品牌体验营销活动：</strong>这个方向的体验活动方面可以这样来构思：<strong>《你的元旦福气蛋》线上H5</strong>，消费者可以在此<strong>h5中以圆形为基础，画出自己心里想要的物品</strong>（可鬼畜可灵魂画手可美好温暖），表达元旦从0开始的美好寓意，最后<strong>根据用户所画内容生成带有美好寓意或者幽默诙谐式的裂变海报。</strong></p>

<p>还有，<strong>《新年新样》的活动，</strong>用户参与话题打卡，发布自己新年新样原创短视频，比如<strong>新年做的第一顿饭、第一次健身等</strong>，可参与抽奖活动获得礼品。</p>
<p>也可以对“元旦”的寓意和字面意思会结合自身产品进行结合，制造话题捆绑营销：</p>
<ul>
<li>“新年元气蛋”</li>
<li>“圆满锦囊”“时光胶囊”</li>
<li>“一岁一礼”“新样神仙水”</li>
<li>“福蛋驾到”</li>
<li>“新年的第一xxxx（第一杯奶茶/第一次旅行）”</li>
</ul>
<h2 id="toc-2">二、疫情/健康</h2>
<h3>关键词：温暖、守护</h3>
<p>年年我们都有问候，今年的问候和祝福会显得格外珍贵，面对疫情，<strong>我们需要更多鼓励和信心，这是全世界的”情绪大地“，</strong>借助这个表达，能引起最大共鸣，但不可过于商业和浅薄。看话题文案：</p>
<ul>
<li>《新年新愿望，世界无冠状》</li>
<li>《告别疫鸣惊人，但求一帆风顺》</li>
<li>《去年疫留的缺憾，是为新年美好的铺垫》</li>
<li>《新年没有战疫，只有温暖心意》</li>
</ul>
<p>一些新年的小期许和小美好：</p>
<ul>
<li>《千帆过尽，向新而行》</li>
<li>《新的一年，三餐四季，存同伐疫》</li>
<li>《那里，好吗？这里，安好！》</li>
</ul>
<p>可以把”这里“和”那里“换成你的城市和你关心的朋友的城市，还有最重磅的：<strong>《</strong><strong>新的一年，送你件抗疫战服，领子叫信念，扣子叫挂念，袖子叫携手，拉链叫必胜》</strong>。</p>
<p>视频的创意文案思路可以是这样，<strong>《新年calling》/《新年recall》</strong>主题的视频，将商场拟人化，给予一个人设，视频内容对比有疫情和没有疫情时候大家的新年都是什么样子的，传达疫情快点结束的美好祈愿。</p>
<p><strong><strong><strong><strong>体验营销活动看</strong></strong></strong></strong><strong>看京东手机。</strong>2021新年俗以<strong>《瑞兽迎新年，有机有好运》</strong>为主题创新营销玩法，活动关注疫情之下的2020，和追求美好品质生活、<strong>提高免疫力、自然健康长寿结合神兽神话典故。</strong></p>
<p>跨界合作年货节推出<strong>新春瑞兽限量装及瑞兽手办套装</strong>，国潮风格IP打造“春节优选年货”。</p>

<h2 id="toc-3">三、祝愿/好运</h2>
<h3>关键词：压岁钱、期待、感谢</h3>
<p>虽然2022可能会更难，但还是用心为你许下每一个文字的心愿，<strong>《2022，愿你以梦为马，不负韶华》</strong>，还要<strong>《感谢有你一路相伴，新年给你全新宠爱》</strong>，<strong>《用期待一份礼物的眼神，憧憬下一段美好的旅程》。</strong></p>
<p>2022大家各种物质的愿望，海阔天空、天涯海角世界旅行的祝愿大都回归了家庭和身边人，<strong>《初心依旧，从不将就》，《陪你跨越分秒，也愿一起共度余生》，《新</strong><strong>年如意，好料如一》。</strong></p>
<p>哈哈，既然暂时没有了远方，那就和我一起来厨房。<strong>《新年就要举糕糕》（糕点类）。</strong></p>

<h2 id="toc-4">四、大一岁/变老</h2>
<h3>关键词：时间、平凡、去年</h3>
<p><strong>话题文案来了：</strong></p>
<ul>
<li>《恭喜大家，离退休的日子又近了一年》好丧啊</li>
<li>《长大后发现，只有新年快，少了乐》，这个也是，没有快乐的成年世界啊</li>
<li>《解不开生活的难题，就先翻过这一夜》，等明年继续解哈</li>
<li>《岁月醉人，先干为敬》这是打不过先跑的节奏，是喝酒类</li>
<li>《或许人生就是这样，又见一个轮回》</li>
</ul>
<p>这个真的很棒，但愿对你有启发哦，<strong>好的沟通都是源自细致的洞察，再加上表达的技巧和方法，</strong>4A广告公司的擅长哦，<strong>《Aimer爱慕：新年，长大不如长进》，《你关心你是不是又长一岁，我只关心你吃得好不好（食品新零售）。</strong></p>

<p><strong>电商类：《打工人新年送礼指南》，</strong>以幽默诙谐的表现手法，以一个男性打工人为例，在新年时刻，如何<strong>给女朋友、母亲、父亲等送上最满意的礼物，而不是翻车现场。</strong></p>
<p>其实也可以表达过年回家送礼的翻车现场，娱乐加主题。</p>

<h3>关键词：美好、追逐</h3>
<p><strong>依然是励志大方向的，但更多往前看的话题文案：</strong></p>
<ul>
<li>《一路拾捡光芒，奔赴新的方向》</li>
<li>《没有一夜暴富，只有人生抱负，新年新起点！》今年这么难，一夜暴富就算了，一年能富也都行</li>
<li>忘掉不开心和困难《翻过这一年，记住这一刻》</li>
<li>《2022的一年四季，陪你一起甜蜜给力》</li>
</ul>
<p>如果说2019爱你依旧，2020爱你爱你，<strong>2021 依然爱你，那么，2022就是爱了还爱。</strong></p>
<ul>
<li>《新的一年，新的相见，大步跨年，一往无前》</li>
<li>《不管新年旧年，只想看见你的笑脸》，商家活脱脱拟人化的社交营销范式</li>
<li>《不管新年旧年，你的容颜不会跨年》永远18岁哦（美妆/服饰）</li>
<li>《旧年翻篇，一切如新，新年就要图个红》（美妆/服饰）</li>
</ul>

<h2 id="toc-6">六、续航/ 进步</h2>
<h3>关键词：正能量、梦想、力量</h3>
<p>过往的一年，电能源车，充电学习，手机充电，新的一年就在眼前，你的电量是否够续航2022起航，主题文案：</p>
<ul>
<li>《过往皆序章，今日正起航》，面对新的一年，茫然但也要很勇敢，因为，我们无路可退</li>
<li>《世界很大，有时也怕；新年有梦，自带盔甲》</li>
<li>《为了这一天，我们准备了一年，新年新起航》</li>
<li>《2021闯关成功，2022更上一层》，好不容易过了2021，2022也许会更Easy</li>
<li>《家里一应俱全，家人好好跨年》，如果不幸被隔离了，这句话好有底气</li>
<li>《没有事与愿违，只有给你的满心安排》（商场），但愿2022不再有太多的意外</li>
</ul>

<h2 id="toc-7">七、回望/过去</h2>
<h3>关键词：盘点、纪念、再见、怀念</h3>
<p><strong>虽然2021有太多不美好，但是还会有很多人依依不舍。话题文案：</strong></p>
<ul>
<li>《喝尽旧的遗憾，斟满新的希望》</li>
<li>但愿《所有不如意都是误差，未来终会开花》</li>
<li>《新年新对象，年年不重样<strong>》</strong></li>
</ul>
<p>这个可不是针对女朋友的，是给换房换车的广告营销。</p>
<p>还是羡慕这种人的心态<strong>《我纠结是吃到明年，还是到明年再吃》</strong>，<strong>《二零二疫，后会无期》</strong>但愿疫情真的能再见。很多人巴不得尽快告别过去一年，<strong>《2021，不满意退换2022》</strong>。</p>
<p>看看大牌的路虎极光：<strong>《别不敢孤芳自赏，别不敢哗众取宠，别不敢喜新厌旧。别了，2017》，</strong>这是个广告片，空了找来看看，小编的我就刚才认真看完了，深深打动，老板，我要……lizhi（励志）好好干。我们部门小伙伴给大家一个高仿版的，<strong>《再不会,畏手畏脚，再不会,轻言放弃，再不会,礼貌谦让，再见，2021》</strong>另湖，勇敢另下去哈哈。</p>

<h2 id="toc-8">八、回家/陪伴</h2>

<p>这个方向的创意素材和文案真的很多很多，<strong>熟悉的歌曲，诗词，名人名言，民间故事、俗语等，</strong>取之不尽。仔细<strong>应用《海量光速的创意方法》将产品卖点解构和这些素材进行组合创意。</strong></p>

<p>总要有句结束语：虽然人生在世会有种种不如意，但你仍可以在幸福与不幸中作选择。——王小波。</p>

<p>Nothing is worth more than this day．没有什么胜过今天的宝贵。——Goethe歌德</p>
<p>The object of a New Year is not that we should have a new year. It is that we should have a new soul and a new nose; new feet, a new backbone, new ears, and new eyes.</p>
