<h2 id="toc-1">一、行业背景</h2>
<p>中国经济水平提高，高度规范的城市化和人口代际更替的影响下，大量中国人处于亚健康的心理状况，对此引发的睡眠问题日益增多，产生新的趋势。中国大健康产品市场庞大，其中睡眠市场规模也不容小觑。</p>
<ul>
<li>根据前瞻产业研究院发布的《中国大健康产业战略规划和企业战略咨询报告》，2020年光是中国的大健康产业规模，就突破了10万亿元。</li>
<li>根据 《中国城镇居民心理健康白皮书》显示，2018年有73.6%的中国人处于心理亚健康状态，其中16-39岁人群的心理状态最差，受到职场，人际关系，个人焦虑等困扰。</li>
<li>根据《2019-2020线上睡眠消费报告》和《2020喜临门中国睡眠指数报告 》来看，2018年-2019年间失眠人群占比从24.9%提升到36.1%</li>
<li>《京东大数据研究院：2019-2020线上睡眠消费报告》中提及，截止2018年，中国的整体睡眠市场容量达到1万亿元。</li>
</ul>
<h3>1. 新趋势</h3>
<p>疫情期间政府着重关注居民心理健康问题。</p>
<p>2020年1月起，政府出台各项有关心理抑郁的预防和居民心理疏导的政策，有15条与心理健康相关政策，8条与新冠疫情相关。</p>
<p>睡眠问题群体趋向年轻化，其中有三类典型睡眠人群，分别是“996上班族”“年轻妈妈”“00后”。</p>
<p>中国睡眠研究会调查显示，我国有超过3亿人有睡眠障碍。根据《2019-2020线上睡眠消费报告》和《2020喜临门中国睡眠指数报告 》来看，2018年-2019年间失眠人群占比从24.9%提升到36.1%，其中90后的占比从16.4%提升到36.7%，2019年只有不足30%的人感觉自己睡眠后精神饱满。2020年90后和00后的睡眠时长达不到全国平均睡眠时长6小时55分，情绪问题和生活压力是影响他们睡眠的头等困扰。其中着重提及了三类群体，996上班族，年轻妈妈，00后三位典型群体</p>
<h3>2. 业务模式</h3>
<p>心理健康行业以心理问题严重性及是否达到医学诊断标准的心理障碍，分为针对泛心理问题，以咨询辅助为主的“心理健康服务”和 针对精神病例，以专业医药治理的“心理医疗服务”。</p>
<p>目前中国的互联网产品集中在心理健康服务市场，针对睡眠问题，助眠的主流产品有以下3个</p>
<ul>
<li>蜗牛睡眠，用户量和市场规模最大，在apple store累计下载量约为1千7百万，安卓设备累计总下载量1.34亿。核心功能是提供助眠声音，记录睡眠梦话，形成睡眠报告，分析睡眠质量。</li>
<li>小睡眠 : apple store累计下载量约为5百万，安卓设备累计总下载量9千万。核心功能是以用户自建白噪音组合，分享白噪音社区，有明星睡眠故事，电台故事和冥想功能，为用户助眠。</li>
<li>潮汐，apple store累计下载量约为5百万，安卓设备累计总下载量6千5百万亿。核心功能是为用户搭建专注，睡眠，小憩，呼吸场景，提供自然主题的白噪音，冥想指导课程，睡眠故事等内容。</li>
</ul>
<p>以下是2020.5.26-0221.5.26期间，潮汐，蜗牛睡眠和小睡眠在apple store不同榜单的平均每日排名&未上版次数，数据来源七麦数据 和 蝉大师。</p>

<h2 id="toc-2">二、产品介绍</h2>
<h3>1. 产品定位</h3>
<p>潮汐是一款注重自然美好体验，用“自然白噪音”和“冥想指导”帮助用户更好地专注，睡眠，冥想，放松的身心健康的工具型产品。把用户生活依据工作和休息划分为四个场景，极简设计，用户可以选择潮汐的合适场景去交替地适配生活，打造自然美好的场景体验。</p>
<p>潮汐为用户打造一个保持身心平和，活在当下和感悟自然的泛生活场景，帮助用户成为更平和的自己。</p>
<h3>2. 用户&需求</h3>
<p>用户使用场景</p>
<ul>
<li>睡眠不好、深受失眠困扰的都市人群</li>
<li>常常分心、无法专心致志的拖延症患者</li>
<li>身处嘈杂环境、经常被打断的创意人</li>
<li>长期焦虑、精力疲惫的高压人群</li>
<li>追求平和身心的冥想练习者</li>
<li>希望成为更好自己的你</li>
</ul>
<p><strong>1） 用户画像</strong></p>
<p>以“潮汐APP”为关键词，微博搜索收集近90天的30个账户内容，分别为20条个人账户，10个特殊账户（特殊身份，大v），分析用户情况</p>
<p>个人账户群体特点：</p>
<p>主要是30岁以下的大学生和职场白领，个别是年轻妈妈和高级白领。</p>
<p>女性为主，一二线城市城市偏多</p>
<p>喜欢分享有质感的生活，有情调和文艺范，学生群体喜欢分享追星动态</p>
<p>分享内容有两类：</p>
<ul>
<li>安利白噪音对专注和助眠很有效，部分人会文字描述想象的联想画面。</li>
<li>分享“潮汐日帖”的推送，附带日帖页面截图，用“美”来形容。</li>
</ul>
<p>推广的特殊身份和大v账户特点：</p>
<ul>
<li>考研，留学，知识分享类博主，符合一二线城市群体的喜好倾向。</li>
<li>推广内容的侧重点</li>
<li>宣称产品功能“白噪音”+“专注/助眠”</li>
<li>形容词为“好用/好看”，突出产品的工具属性和美好的品牌调性</li>
</ul>
<p><strong>2）用户需求</strong></p>
<p>工作和休息是一天中花最多时间的两件事，工作顺利和足够休息是幸福感的基础。但是受制于外部干扰和内在思绪困扰，工作和休息出问题，工作做不好，休息睡眠不好，影响人幸福感的下降。</p>
<h3>3. 场景</h3>
<p>主要场景有“工作场景”,“休息场景”和“情绪困扰场景”，用户受到的干扰有“外在干扰”和“内在干扰”。</p>
<ul>
<li>工作：指的是用户在不同情景中有不同的目标任务。如成年人白天工作是完成岗位的工作任务，在晚上消遣时可以阅读则阅读是其当下的工作；学生白天任务是完成读书学习，在晚上收拾家务时则收拾家务是其当下的工作。</li>
<li>休息：主要指的是睡觉，也包括其它恢复精神体能、放松身体/脑力的活动。</li>
</ul>
<p><strong>1）工作场景的特点</strong></p>
<ul>
<li>时间不可控，多数任务由复杂的小任务组成，复杂性决定了时间不可控，尤其在多人协同更是如此，只能预估大致时间，时间划分维度较粗，以天、上下午为单位。</li>
<li>多数情况处于公共场所，需要顾及其他人。</li>
<li>工作场景主要受到外在干扰为主，人在工作时注意力会集中在当下，内在干扰小</li>
</ul>
<p>嘈杂的外界噪音的影响，人通过5感接收外界信息，大多数信息依靠视觉、触觉和听觉。相较于眼睛和肢体动作，耳朵是唯一的被动听觉器官，无法控制。</p>
<p>电子设备的干扰，被手机中的讯通和娱乐软件影响。</p>
<p><strong>2）休息场景的特点</strong></p>
<p>休息最重要&常用的方式是睡觉，睡觉是一个明确可控目标。同时工作日的睡觉有严格的时效性限制。因为现代工作日制度，工作日的休息场景有晚上睡觉和中午午休，起床对时效性的把握需要精确到分。</p>
<ul>
<li>晚上睡眠取决于隔天几点起床上班</li>
<li>午休取决于剩余公司制度的休息时间</li>
</ul>
<p>多数情况是在家的私人空间。</p>
<p>休息场景的干扰</p>
<ul>
<li>嘈杂的外界噪音的影响</li>
<li>内心思绪烦乱干扰</li>
</ul>
<p><strong>3）情绪困扰场景的特点</strong></p>
<ul>
<li>有明显生理反应，如流泪，呼吸急促，心跳加快，头脑充血急躁等，对当下的判断和把控力下降，无法专注当下。</li>
<li>多数由用户思绪烦乱引起。</li>
</ul>

<p>以下逐一分析潮汐的四个核心功能</p>
<p><strong>1）专注场景（专注计时器）</strong></p>
<p>市面上流行的计时打卡类产品有两种形态：</p>
<ul>
<li>第一类是“ToDo清单为核心”的时间管理，围绕着任务清单的前后流程，提供可定制的专业清单，对任务执行时间计时，提供多样化的时间数据图表，主要满足的三个需求：标记防止遗漏，查看任务完成情况、罗列工作任务，排列时间优先级、记录任务和时间数据等，如 番茄ToDo 。</li>
<li>第二类是“用户执行任务的计时环节”为核心的时间管理，弱化清单任务管理，强调以多样的方式帮助用户专注完成当下的任务，如 <a href="https://apps.apple.com/cn/app/id866450515">把专注时间可视化和游戏化为植树活动—— Forest</a><a href="https://apps.apple.com/cn/app/id866450515">* *</a>。</li>
</ul>
<p>我认为第二种相比第一类更好，理由如下：</p>
<ol>
<li>以人为核心，用户的门槛更低。打卡计时工具产品的最核心需求是“通过产品来帮助自己完成任务，提高效率，达成目标，成为更好的人”。第一类产品的核心用户需要满足“强执行力”和”和“耐心”2个要求，前者保证用户能完成任务，后者保证用户肯花时间做计划清单，但人懒惰且趋于安稳，满足以上需求的群体是少数的，在群体协作，有多人监督驱动的场景会很合适。从用户的整个生命周期来看，培养和教育用户改变习惯很难，更多的用户安装这类产品是受到短期自我激励热情的触发，雄心壮志的热情遮盖自身缺乏执行力、缺少耐心和外界干扰等常见问题，等用户进入规划和执行的环节，就会容易受到上述的各种原因影响，最后往往没有完成设定的任务，遭受失败打击（参考以下微博搜索的用户反馈截图）。相比之下，第二种产品以人为核心**，**围绕着用户专注执行的场景，通过隔绝噪音，游戏奖励驱动等多种形式来驱动用户完成目标，让用户关注当下，优先来完成任务，鼓励用户做出行动，因为哪怕是完成一件小事，这最小的一步也是真正让用户踏上通往“更好的人”的正确道路。（举个比喻，第一种产品类似让用户在前期规划好路线，设计好每一步和沿路的风光。但用户往往因为没有走到目标地点而望着目标差距沮丧。第二种鼓励用户在道路上行走，让用户不过多关注未来的目标，而是享受沿路行走的幸福体验，关注自己走了多远，而非差多远抵达目标。）</li>
<li>流程效率更高，第一种产品的主要流程从建立计划开始，多样的定制功能虽然满足了用户追求功能越多越好的性价比心理，但是提升了用户调教磨合产品的门槛，也相当于增设了更多的流程，存在用户把时间分心在美化界面和调试计划的可能。对比之下，第二种产品去掉计划设定的各种弹窗或页面跳转环节，突出开始执行的入口，任务用可选的简单属性标签做区分，避免用户前期投入的过多的操作成本。如forest、潮汐，把最重要执行入口放在首页。</li>
</ol>
<p>潮汐属于第二种，通过多样化的自然场景声音来帮助用户营造合适的专注空间，且提供足够的功能。</p>
<ol>
<li>多样的自然白噪音帮助用户排除外界干扰，搭建沉浸空间，专注当下完成目标。</li>
<li>提供了3类不同的计时场景模式，按照用户情况自行适配，“自定义时长模式”，“工作+休息的番茄钟”和“无限模式”。</li>
<li>设计隔绝机制深度专注，限制软件影响，让用户更好地投入当下的任务。</li>
<li>提供打卡记录和标签，满足用户达成目标的成就感和满足感。</li>
</ol>
<p><strong>2）睡眠/小憩场景</strong></p>
<p>潮汐把休息划分为睡眠和小憩。因为情景时长和影响程度不同，睡眠可以理解为以睡眠目的一种特殊专注场景，能使用专注场景类似的解决方案</p>
<p>相比专业型助眠指导的产品，如蜗牛睡眠，小睡眠等，搭建睡眠社区，输出细致睡眠报告，围绕睡眠做全方面流程功能服务。</p>
<p>潮汐的睡眠场景只有3类功能，工具属性更强：</p>
<ol>
<li>提供多样的助眠声音内容，分别是“睡眠故事”“睡眠冥想”和“白噪音”来帮助用户入睡。</li>
<li>提供闹钟，多样的唤醒功能。有常规的闹钟提醒用户起床，摇晃机制帮助用户更好地拜托赖床状态，用户体验更好。</li>
<li>提供睡眠记录，让用户能够看到自己的睡眠信息，入睡时间，起床时间，总时长，睡眠质量评分。</li>
</ol>
<p><strong>3）呼吸场景</strong></p>
<p>潮汐为情绪困扰和自主冥想需求设计，提供两种呼吸节奏模式，分别是平复情绪，缓解压力的“平衡呼吸” 和 帮助助眠的“4-7-8呼吸”</p>
<p>两种模式差别在于呼吸时间的节奏和屏气环节，4-7-8的呼吸节奏更长，增加了屏气环节，能帮用户进入睡眠状态。用户按照自己的需求在1-15分钟内设置时间和呼吸节奏，提供白噪音。换气节点没有采用人声提示，而是区别背景音的特殊音效，能让用户更投入呼吸，去注意细微声音的变化，一定程度减少用户分心的情况。</p>
<p>在功能设计有两个好处：</p>
<ol>
<li>适用场景更全面，用户体验更好，利用动画、声音、振动的方式能够让引导用户睁眼，闭眼的情况都能通过多种感官来把控呼吸节奏。</li>
<li>培养用户呼吸习惯。呼吸是冥想最核心的事，平稳呼吸能抚平情绪，回归理性。大多数人对冥想一知半解，呼吸场景帮助用户入门，掌握呼吸节奏，减低后续冥想练习的学习难度。</li>
</ol>
<p><strong>4）潮汐日帖</strong></p>
<p>潮汐每天更新一个日帖，日帖内容以文字搭配一张漂亮的图片。内容文字是具有禅意的诗歌句子。</p>
<p>从KANO模型来看，属于魅力属性强的亮点功能：</p>
<ul>
<li>对用户对话感，强化用户对潮汐产品调性“人与宇宙，自然同归”的心智认知，潮汐的用户偏有生活格调，喜好文字，潮汐通过诗意内容达到和用户的对话感，切合用户的喜好。如潮汐的图文强调自然感悟和精神意境，调动想象联想能力——悟，弱化实体感知，强化产品本身自然空净，气定平和的产品调性，一定程度上弱化工具属性，强化交流属性。</li>
<li>增加用户传播分享渠道，潮汐注重使用体验，利用精美图文的价值感，提升产品传播率，类似旅游景点的免费纪念品。</li>
<li>提供产品运营推送渠道，提升产品打开率，有意义的文字，让产品具备对话感，也符合用户群体文艺范的特点，增加有效的接触用户的渠道。</li>
</ul>
<h3>4. 核心资源</h3>
<p>潮汐的核心是音频，分为“场景声音的白噪音 ”和“ 冥想练习/睡眠故事”。由六个基本元素组成是最小组成单位，来源是产品内部上传，定期更新，内容以音频类型和标签作为分类依据。</p>
<p>潮汐最小单位是单个音频，由6个信息构成。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/06/gnIscbtgoNYntLbyzi0I.png" alt width="666" height="287" referrerpolicy="no-referrer"></p>
<p><strong>1）场景声音的白噪音 如何构建场景</strong></p>
<p>潮汐采用“虚实结合”的方式调动用户想象力构建场景，因为直白展现的东西是不美的，通过具象化+联想性的信息要素，用半露半掩的方式引人遐想。</p>
<p>人对场景的感知来自视觉，听觉，触觉对环境的反馈，提供的信息越丰富，用户对场景的感知越强。而主要信息来源是视觉感官，结合大脑有想象力的特点，脑补其他信息要素（听觉，触觉等），这也是我们在网络上看到壮丽的自然风光会心神荡漾的原因，视觉画面传达到大脑后，大脑运用想象力强化身临其境的代入感。刘慈欣短篇小说中《带上她的眼睛》，一方的人借助智能眼球看到另一方眼睛的画面，直接感受对方的空间。</p>
<p><strong>2）冥想练习/睡眠故事 和 场景声音 对比分析</strong></p>
<p>两者虽然都是音频，主题互补，但是冥想练习/睡眠故事的商业价值比白噪音更高。</p>
<ul>
<li>冥想练习/睡眠故事有实用属性和知识价值属性，相比白噪音只有美这一个单一属性卖点，且当前消费者在认知上更多地把失眠的困扰当做“病”，冥想练习的内容专业性强，有利于更大的议价空间。</li>
<li>因为内容专业性，冥想练习相比白噪音的不替代性更强，而白噪音专注和放松身心的功能，听歌产品音频内容产品等能替代。</li>
</ul>
<p>在内容主题来看，两种声音内容互补。白噪音以人的环境为核心，按照自然、生命、宇宙主题去延伸拓展内容。而冥想指导围绕着人的行为本身，去覆盖更多的行为场景。</p>
<p><img data-action="zoom" class="aligncenter" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2021/06/1CsWCZ9DBy9mHhAOefR0.png" alt width="2876" height="2170" referrerpolicy="no-referrer"></p>
<h2 id="toc-3">三、品牌设计&体验</h2>
<p>潮汐作为工具产品属性，在交互设计和视觉风格在注重效率的同时，营造人与自然和谐之美的产品品牌属性。前者让潮汐注重细节体验，让用户更高效，后者营造了差异化认知，让用户更舒服。</p>
<h3>1. 交互体验</h3>
<p>交互的完美体验体现在很多小细节方面，只介绍前2点，着重分析潮汐在新用户登录的过程中的设计引导。</p>
<ul>
<li>首页左右滑动切换播放场景声音</li>
<li>音频信息页面的底部按钮文案</li>
<li>冥想练习/睡眠故事的音频进度条藏在背景页的上下滑</li>
<li>用户可以不注册登录的情况，直接体验六个已下载的场景声音，无差别体验四种场景，查看日帖，查看场景的数据情况等大部分核心功能</li>
<li>后台的小浮窗功能</li>
<li>…..</li>
</ul>
<p><strong>1）交互细节一：首页左右滑动切换播放场景声音</strong></p>
<p>首页默认是用户最后播放的场景声音，用户选择场景后如果不更换场景声音，则默认按照首页的场景声音来播放。</p>
<ul>
<li>从用户操作上看流程更快捷，用户受到场景触发而使用产品，用户打开产品后需要做两件事“选择对应场景”和“选择声音”。因为不同场景的差别只有时间条件不同，且白噪音音频具有适配性，不同的场景可以共用同个音频，所以潮汐前置了声音库的选择，首页左右滑动也方便用户更换场景声音。</li>
<li>满足更多用户需求，扩展可玩场景，潮汐的核心在于声音，首页展示声音的功能设计，满足用户在没有场景目的，单纯地听一听声音，类似听歌的情景需求。</li>
</ul>
<p><strong>2）交互细节二：音频信息页面的底部按钮文案</strong></p>
<p>潮汐为音频内容设计了两套会员体系，获取音频的按钮是第一个付费入口，在音频的信息页面，潮汐依据声音属性、音频文件状态，用户属性在底部按钮设计了不同的文字状态做出区分。</p>
<ul>
<li>统一用户认知，便于用户理解不同音频的所有权。用下载，会员独享等文字的变化，用户可以最快理解出声音的使用权是永久还是暂时会员特权，且按钮下方还会文字提示会员到期日期。</li>
</ul>

<h2 id="toc-4">四、运营&商业化</h2>
<h3>1. 产品运营</h3>
<p>内容运营：</p>
<ul>
<li>推送“日帖/文章”</li>
<li>上新“白噪音、冥想课程和睡眠故事”</li>
<li>更新“白噪音”</li>
</ul>
<p>数据指标方面：</p>
<p>我认为潮汐有两个核心运营指标，分别针对“场景声音的白噪音 ”和“ 冥想练习/睡眠故事”</p>
<p>因为，我的理解是“场景声音的白噪音”是用于培养用户进阶“冥想练习”习惯，两者属于不同阶段的用户。</p>
<ul>
<li>最终目标上看，“场景声音的白噪音 ”和“ 冥想练习/睡眠故事”目标一致，都是帮助用户活在当下，更好生活、工作和休息。</li>
<li>在声音形式上差别大，“场景声音类似”但是当前中国社会，冥想的接受&传播度很低，需要时间去培养&教育用户，白噪音是一个简单&合适的载体</li>
<li>潮汐本身对音频内容划分“场景声音的白噪音”和“冥想练习/睡眠故事”两个会员体系。后者的价格更高。</li>
</ul>
<p><strong>1）“场景声音的白噪音”的核心指标是“平均用户每天创建场景的次数”</strong></p>
<ul>
<li>以标签，时间和场景声音来作为维度组合，代表用户对声音场景适应性的认可度和依赖。</li>
<li>在商业化方面，单个用户更多的场景空间会带来更多的场景声音曝光，也增加用户付费概率</li>
</ul>
<p><strong>2）“冥想练习/睡眠故事”的核心指标是“用户累计使用天数”</strong></p>
<ul>
<li>冥想的效果在短期是不明显，需要长久潜移默化使用体验，用户市场接受度不够还需要时间去培养用户行为。睡眠场景是培养用户冥想行为的突破点，因为其相较于冥想场景，效果更显而易见。</li>
<li>商业化方面也有利于更多的获利
<ul>
<li>更长的使用天数意味着更多的会员周期。</li>
<li>冥想的商业价值比白噪音更高。两者虽然都是音频内容，但是冥想练习有实用属性和知识价值属性，相比白噪音只有美这一个单一属性卖点，有更多的获利空间。</li>
</ul>
</li>
</ul>

<h2 id="toc-5">五、版本迭代演变</h2>
<p>潮汐最开始的定位是“白噪音番茄钟”，后续扩展“ 专注工作、学习、减压、冥想与白噪音的番茄钟计时器”，再到如今“睡眠、专注与冥想的白噪音番茄钟”。</p>
<h3>1. 潮汐的优势</h3>
<p>第一点是以自然之美，天人平和的产品设计营造的差异化定位，相较竞品能覆盖更多的用户场景。</p>
<ul>
<li>相比游戏化专注计时器“forest”，潮汐兼具睡眠场景。</li>
<li>相比有趣&好玩定位的“小睡眠”，潮汐的定位在全年龄层有更好的适配性。</li>
<li>相比专业睡眠定位的“蜗牛睡眠”，潮汐兼具生活场景。</li>
</ul>
<p>第二点是根据产品定位延伸，通过“美的声音，视觉和文字内容”和“极简的交互设计”带来的独特体验。</p>
<h3>2. 潮汐的劣势</h3>
<p>和垂直领域的产品对比，打卡计时功能 和 助眠功能的专业性差，损失部分对某项需求强依赖的用户群体</p>
<p>冥想练习内容较为全面，但是入口深。目前“睡眠场景-助眠冥想”是潮汐从白噪音到冥想内容扩展的切入点，潮汐为了培养用户“冥想适用任何场景的认知”，在助眠场景为主题页面中，提供18个不同的助眠冥想练习内容。</p>
<p>综上全篇，对于未来，我预估潮汐会集中拓展更多的冥想内容，逐步把专注场景和冥想场景打通。</p>
