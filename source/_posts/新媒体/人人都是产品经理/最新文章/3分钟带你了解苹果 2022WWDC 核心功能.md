<blockquote><p>编辑导语：6月7日凌晨，苹果在线上举办了WWDC2022发布会，发布会主要内容包含：iOS 16、watchOS 9、iPadOS 16、macOS Ventura，以及M2芯片加持的全新设计的Macbook Air和Macbook Pro，这篇文章详细为我们介绍了这些产品更新的核心功能，一起来看看吧。</p></blockquote>
<p><img data-action="zoom" class="size-full wp-image-819879 aligncenter" src="https://image.yunyingpai.com/wp/2022/06/RSxIz8G93rV3CkR95N3R.jpg" alt referrerpolicy="no-referrer"></p>
<h2 id="toc-1">一、新品一：iOS 16</h2>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/AvDeIF3no4rnEmYRzbo1.png" alt width="556" height="312" referrerpolicy="no-referrer"></p>
<h3>1. 更新1：锁定屏幕</h3>
<ul>
<li>保留iPhone风格，但更个性化、更美观</li>
<li>提供锁屏界面自定义编辑，例如调整锁定屏幕上的多种元素，包括字体、颜色、小组件等</li>
<li>提供多种锁屏图库可供选择</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/Gkq6NNbo17QRenxgcFJj.png" alt width="555" height="417" referrerpolicy="no-referrer"></p>
<h3>2. 更新2：消息通知</h3>
<ul>
<li>收到通知时，消息在锁屏底部滚动显示，也可以隐藏通知</li>
<li>Live Activities：通过锁定屏幕实时查看通知，包括Uber、比赛、音乐播放等</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/6sRv3BGfymBFAiN0GGon.png" alt width="648" height="180" referrerpolicy="no-referrer"></p>
<h3>3. 更新3：专注模式</h3>
<ul>
<li class>专注模式与锁定屏幕匹配：不同专注模式可以匹配不同的锁定屏幕。或通过切换不同的锁定屏幕从而切换专注模式</li>
<li>专注模式可应用于APP层级：利用专注过滤条件Focusfilters，对浏览器网页、邮件、日程等进行筛选过滤</li>
</ul>
<h3>4. 更新4：Message信息</h3>
<ul>
<li>新增三项新功能：编辑刚刚发送的信息；撤回信息；标记未读</li>
<li>Share With You：将同播共享功能SharePlay拓展到Message APP，可以使用信息共享视频并同步观看</li>
</ul>
<h3>5. 更新5：Dictation听写</h3>
<ul>
<li>使用听写功能的同时，键盘保持启用，从而可以在语音和触控输入间自由切换</li>
<li>听写可以自动在文字里加入标点符号，和听写表情符号</li>
<li>推出有Swift编码语言编写的App Intents API，无需设置就可以运行快捷指令，开发者可以在更多APP中调用Siri</li>
</ul>
<h3>6. 更新6：Live Text 实况文本</h3>
<ul>
<li>实况文本功能拓展到视频</li>
<li>新增基于实况文本的快速操作，例如换算汇率，翻译外文</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/q2EfOlaz2nXV0xsUjD7m.png" alt width="686" height="387" referrerpolicy="no-referrer"></p>
<h3>7. 更新7：Visual Look Up</h3>
<ul>
<li>通过图片识别，可以提供图片内丰富的资讯</li>
<li>通过图片识别，可将识别主体从背景中提取出来拖放到其他APP中</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/JR1AbsFmgIM8jINDzTIk.png" alt width="688" height="391" referrerpolicy="no-referrer"></p>
<h3>8. 更新8：Wallet 钱包</h3>
<ul>
<li>共享钥匙：通过邮件、信息或者WhatsApp等与亲友共享钥匙。与IETF协作，让共享钥匙成为一个行业标准让大家免费采用</li>
<li>Tap to Pay：目前在美国各地有数百万商家可以直接通过iPhone进行无接触支付</li>
<li>Apple Pay Later：将一笔Apple Pay支付款项分成四等额，分摊在六周内还款，没有利息和费用</li>
<li>该功能在所有接受Apple Pay的地方均可使用包括APP或者线上</li>
<li>分期付款可以通过钱包App管理，帮用户维持在预算范围内</li>
<li>Apple Pay Order Tracking：使商家将详细的收据和追踪信息，直接发送至钱包App，便于用户获取订单的最新信息</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/CqjmtcZdiggwwpdkmFnH.png" alt width="827" height="157" referrerpolicy="no-referrer"></p>
<h3>9. 更新9：Map 地图</h3>
<ul>
<li>今年晚些时候将会把新地图引入11个国家和地区</li>
<li>Map kit升级：优化地图细节，包括土地覆盖、道路等；优化驾驶导航、公交信息等；新增四处看看功能</li>
<li>新增Multistop routing多途经点路线规划功能：可提前规划15个途经点，并可在Mac进行编辑</li>
<li>公交出行：可直接查看车费、将交通卡添加至钱包、交通卡余额不足提醒及缴费</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/vPrawcS56aqIeJEx9oKG.png" alt width="685" height="388" referrerpolicy="no-referrer"></p>
<h3>10. 更新10：Sport 运动</h3>
<p>Apple News新增my sports功能：用户可以在新闻列表关注各种联赛、运动员信息。即将在美国、英国、加拿大和澳大利亚推出</p>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/UyH66iPbuNsVp43t4Fs2.png" alt width="687" height="388" referrerpolicy="no-referrer"></p>
<h3>11. 更新11：Family Sharing 家庭共享</h3>
<ul>
<li>通过家庭共享，将订阅服务共享给最多五位家庭成员</li>
<li>简化创建儿童账户流程</li>
<li>推出iCloud共享图库：最多可以与五个人共享，用户可以筛选放入共享图库中的图片</li>
<li>用户可以在拍照时进行设置：拍照的同时直接将照片发送到共享图库</li>
<li>用户可以设置自动激活共享功能，在共享图库的成员附近拍照时会自动启动</li>
<li>为你推荐标签页会智能提供共享建议</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/oepi072qM2mTyroXBmPL.png" alt width="687" height="386" referrerpolicy="no-referrer"></p>
<h3>12. 更新12：Safety Check 安全检查功能</h3>
<ul>
<li>快速查看和重置之前授予他人的访问权限</li>
<li>阻止设备通过查找网络将位置共享给他人，并重置所有App的系统隐私权限</li>
<li>可退出所有其它设备上的iCloud，并将信息App和FaceTime通话权限限制在手中的设备</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/8apAlqEKHrP2ro2xWdJD.png" alt width="687" height="389" referrerpolicy="no-referrer"></p>
<h3>13. 更新13：Home 家庭</h3>
<ul>
<li>建立新的智能家居连接标准matter，以HomeKit作为新标准的基础，该标准使得智能家居配件可以跨平台协同操作</li>
<li>目前有130多种产品正在生产计划中</li>
<li>家庭App重新设定浏览、组织和查看配件的方式：所有房间和收藏都整合入首页</li>
<li>增加温控、灯光、安全等类别，并在屏幕顶部添加每个类别情况的概览</li>
<li>重新设计模块，通过形状和颜色轻易识别不同的配件</li>
<li>小组件可添加至锁屏，方便用户查看</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/iZPkvPpcFeUG8DTOOAVf.png" alt width="827" height="154" referrerpolicy="no-referrer"></p>
<h3>14. 更新14：CarPlay</h3>
<ul>
<li>新一代CarPlay可以在各个驾驶屏幕上提供内容</li>
<li>与车辆硬件系统深度整合，无需退出CarPlay车载，就可直接操控汽车收音机或空调等设备</li>
<li>iPhone支持的小组件可匹配呈现在屏幕上，以及仪表盘中心</li>
<li>新一代CarPlay车载与整个仪表盘互通，与车辆的实时系统连接，呈现车速、呈现车速、油量等各种驾驶信息</li>
<li>提供多种风格的仪表盘选项，用户可自定义仪表盘的风格、颜色、背景和布局</li>
<li>明年晚些时候发布的汽车将陆续搭载</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/DtKXIwByMJINYozbcGTZ.png" alt width="750" height="455" referrerpolicy="no-referrer"></p>
<h2 id="toc-2">二、新品二：watchOS 9</h2>
<h3>1. 更新1：添加4个新的表盘：天文、月相、欢乐时光、大都会表盘</h3>
<p style="font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; font-size: 16px; font-style: normal; font-weight: 400;"><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/CeHCP1HM789twQ9UFm6S.png" alt width="685" height="387" referrerpolicy="no-referrer"></p>
<h3>2. 更新2：全新的Siri UI和横幅通知</h3>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/wV95E7i2aRJGI0jWYlHS.png" alt width="686" height="387" referrerpolicy="no-referrer"></p>
<h3>3. 更新3：Fitness 体能训练</h3>
<ul>
<li>添加三项新的跑步测量指标：垂直振幅、步长、触地时间</li>
<li>优化体能训练视图，每个视图有更多指标易于用户查看</li>
<li>新增心率区间，便于用户快速查看，了解训练强度</li>
<li>推出新的自定义训练计划，并添加新的提示，这些提示将在运动中通过触觉和声音反馈对用户进行指引</li>
<li>iOS 16的iPhone用户可以使用体能App的部分功能</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/Mzwe4sKUs7wtJr6tqOp8.png" alt width="797" height="227" referrerpolicy="no-referrer"></p>
<h3>4. 更新4：Sleep 睡眠</h3>
<p>Sleep Stage：Apple Watch通过加速感应器和心率传感器的信号，检测用户的所处的睡眠阶段，用户通过App可以了解每个睡眠阶段的情况</p>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/GGDsxa643tcoQQ5BoN20.png" alt width="803" height="224" referrerpolicy="no-referrer"></p>
<h3>5. 更新5：心脏健康</h3>
<p>通过房颤历史追踪房颤时长，辨别一天或一周内房颤最频繁的时间，是首个长期提供房颤信息的功能</p>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/Zi91SScvOOv65JspoQiW.png" alt width="687" height="387" referrerpolicy="no-referrer"></p>
<h3>6. 更新6：Medications 用药</h3>
<ul>
<li>帮助用户记录用药，提醒用户用药</li>
<li>建立用药清单，掌握用药时间</li>
<li>提示用户药物的相互作用</li>
<li>健康共享：共享亲人的健康数据</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/K9gQHVqn9V9xKuNTtf3W.png" alt width="686" height="388" referrerpolicy="no-referrer"></p>
<h2 id="toc-3">三、新品三：M2芯片</h2>
<ul>
<li>第二代5nm制程技术；封装超过200亿个晶体管，比M1多25%</li>
<li>内存控制器：提供100GB/s的统一内存带宽，比M1多50%；统一内存为24GB</li>
<li>CPU：8核CPU</li>
<li>相比M1，相同功耗下，性能提升18%；</li>
<li>相比10核PC笔记本电脑芯片，相同功耗下，性能为10核芯片的1.9x，PC芯片峰值性能下，功耗仅为10核芯片的1/4；</li>
<li>相比12核PC笔记本电脑，M2能提供12核芯片接近90%的峰值性能，功耗仅为1/4</li>
<li>GPU：10核GPU，比M1多两个</li>
<li>相比M1，相同功耗下，性能提升25%；最大功耗水平下，性能提升最高可达35%</li>
<li>相比10核PC笔记本电脑芯片，相同功耗下，性能为10核芯片的2.3x，PC芯片峰值性能下，功耗仅为10核芯片的1/5；</li>
<li>搭载全新一代的安全隔区和神经网络引擎：神经网络引擎每秒可以处理15.8万亿次操作，比M1快40%</li>
<li>搭载新一代媒体处理引擎：支持8K H.264和HEVC视频的更高带宽视频解码器，能播放多条4K和8K视频</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/GQyY4g1TA9utewAeKdvI.png" alt width="837" height="669" referrerpolicy="no-referrer"></p>

<li>视频编辑：较上一代提升38%</li>
<li>电池：视频播放时间最长可达18小时，包括一个带有两个USB-C端口的紧凑性电源适配器；全新支持现有的67W适配器快速充电，可在30分钟内充电50%</li>

<li>上市时间：下个月</li>
</ul>

<h2 id="toc-6">六、新品六：macOS Ventura</h2>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/7baZspjIhUwU8mql9PpS.png" alt width="686" height="378" referrerpolicy="no-referrer"></p>
<h3>1. 更新1：Stage Manager 台前调度</h3>
<p>自动将所有窗口排列于一边，将正在使用的APP放于正中，并能重叠放置窗口，适用于多任务处理</p>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/tUh68piEIIeluc2z87lU.png" alt width="685" height="389" referrerpolicy="no-referrer"></p>
<h3>2. 更新2：Spotlight</h3>
<ul>
<li>快速打开APP，查找，运行快捷方式等</li>
<li>该功能目前也加入到iPhone主屏幕底部</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/lRW2xwiJFJt61Dsw6V5L.png" alt width="688" height="386" referrerpolicy="no-referrer"></p>
<h3>3. 更新3：邮件</h3>
<ul>
<li>更新功能：撤销发送；定时发送；跟进建议；稍后提醒</li>
<li>优化搜索功能：可以看到最近分享的文件、链接等，自动识别错字</li>
</ul>
<h3>4. 更新4：Safari</h3>
<ul>
<li>新增共享标签页功能，可以实时查看共享好友正在浏览的网站，并可以通过内置的协作功能进行协作</li>
<li>Passkey 通行密钥：使用FaceID等生物识别信息，而非普通符号和数字密码。无法被盗用，不会泄露；</li>
<li>可以用iCloud钥匙串在Apple设备上安全同步</li>
<li>与FIDO alliance、微软、谷歌合作保证通行密钥能在不同平台间无缝使用</li>
<li>非Apple设备也可以用iPhone登录网站或者App</li>
</ul>
<p><img data-action="zoom" class="aligncenter" src="https://image.yunyingpai.com/wp/2022/06/mOJpCxVykHiEufcUInF1.png" alt width="858" height="158" referrerpolicy="no-referrer"></p>
<h3>5. 更新5：Metal 3</h3>
<ul>
<li>推出MetalFX Upscaling采样，让开发人员可以更快速的渲染丰富而视觉复杂的图形</li>
<li>推出Fast resource loading API，通过提供从存储到统一内容系统的更直接路径，来最大限度的减少等待时间</li>
</ul>
<h3>6. 更新6：互通功能</h3>
<ul>
