微博的评论区聚集各种不同的声音，小红书的评论区是另一个种草区，抖音的评论区都是“勇人”，每个产品的评论区都各具特色，也是产品生态中重要的一环。对一个产品来说，评论区为何如此重要？想打造一个好的评论区，应该怎么做？一起来文中看看吧。</p></blockquote>

<p>如果没有评论区，我们熟悉的那些产品，会变成什么样子？</p>
<ul>
<li>若微博的评论区挂掉了，只能看到寥寥几句博文和配图。我们无从得知事情背后有什么内幕，也看不到任何相关人士的直接声明，更没有人和我们一起八卦、给热搜脑部细节，刷微博的乐趣，顿时就少了大半。</li>
<li>若淘宝的评论区关闭了，我们无从得知其他人购买商品后的评价，看不到真实拍摄的买家秀，无法确定一件商品是否和描述的一般。我们又怎么下定决心，从千里之外网购一件商品？</li>
<li>若网易云的评论区没有了，我们虽然还能欣赏美妙的音乐，却在音乐之外，无法在找到那一句直击心灵的句子，无法找到那歌曲旋律伴随下发生的故事，感觉也缺少点什么。</li>
<li>若贴吧的评论功能消失了……贴吧直接就失去了存在的意义。</li>
</ul>
<p>几乎所有产品，在达到一定体量后，都有让用户能围绕某个话题进行互动的区域。</p>
<p>虽然风格各异，但又精彩纷呈，成为了产品重要的组成部分。给用户提供着更多的乐趣，也持续吸引着用户，把更多的精力、情感投入到与产品的休戚与共中。</p>
<p>对一个产品来说，评论区为何如此重要？</p>
<p>想打造一个好的评论区，又该如何施为？</p>
<p>且让我们走入评论区中。</p>
<h2 id="toc-1">一、用户需要评论区</h2>
<p>需求，是一切功能的出发点。</p>
<p>正因为用户有写评论、看评论的需求，评论区才能诞生、发展、火热。</p>
<p>但写评论、看评论的需求，并不简单，其中交织着多种人性与利益。</p>
<h3>1. 人人都爱写评论</h3>
<p><strong>1</strong><strong>）评论表达情绪——人最在乎的是自己</strong></p>
<p>我们对外界的认知，是基于自身感受的映射。</p>
<ul>
<li>说起汶川地震，很多家不在四川的朋友，未必记得当时有多少人遇难，却能够清晰记得，在感受到大地震颤的时候，自己正在做什么、自己有什么反应。</li>
<li>说起某部电影、某部小说，里面一些情节可能已经记不太清，但自己当时追剧时哭的稀里哗啦的事情，却还有着深刻的印象。</li>
</ul>
<p>我们接触内容时，无论是看视频，还是看热搜，也包括读书的时候，我们最为关注往往不是内容本身，而是在过程中，自身的感受。</p>
<p>人最在乎的，永远是自己。</p>
<p>而在浏览网络时，出现情绪波动，无论喜怒哀乐，我们都的情绪也都需要抒发和表达的空间——评论区。</p>
<ul>
<li>看到震惊认知的新闻事件，通过评论发泄自己的情绪。</li>
<li>某个镜头、某个内容戳中了自己的笑点，哪怕简单的发个“哈哈哈哈哈哈”，也是对自己情绪的表达。</li>
<li>在淘宝上好评，抒发买到好货的开心；又在货品不称心时差评，未必是多么想警示他人，更多的是想表达自己的不满。</li>
</ul>
<p><strong>2</strong><strong>）互动寻求认同——我们需要群体的接纳</strong></p>
<p>人是社会性动物，我们很难完全独立、一个人生存，不与其他任何人产生瓜葛。</p>
<p>远古时代，若没有部落的保护，独自一人无法对抗野兽、饥荒。</p>
<p>即使现代，若不与任何人接触，也无法顺利的工作、生活。就算宅家点外卖，也总要有农民为你产粮，有饭店接下你的订单，有厨子给你烹调，有外卖小哥送到你的家门前。</p>
<p>生在群体中，活在社会里，我们都极为在意群体的看法。</p>
<p>一个被群体所认可的人，就能够顺风顺水。一个被群体所排斥的人，无论在公司、在家庭、在朋友圈子中，都生活不易。</p>
<p>所以，我们都极为在意群体的看法。并且，会主动找机会，不断验证“自己的观点是被群体所接纳的”这件事，从而获得安全感、力量感。</p>
<p>而评论，就是我们验证接纳，寻找认同与共鸣的阵地。</p>
<p>面对一个外部事件，我们可以与其他人，自由发表看法、表述观点，而不产生什么严重责任。</p>
<p>在古时候，士大夫在酒桌上评论家国大事，村里人在地头间评论家长里短。</p>
<p>现在的评论区，无非是把这种古代就有的行为，挪移到了线上、扩大了参与范围。</p>
<p>但人们发表评论的这部分动机，却未必有太多变化：</p>
<ul>
<li>看到很多人给自己的评论点赞，会感到很开心，感受到自己所找寻的认同。</li>
<li>有观点相似的人回复，会展开热烈讨论，也是找到了共鸣的快乐。</li>
</ul>
<p><strong>3</strong><strong>）树立社交形象——我们在乎群体中的地位</strong></p>
<p>同样基于人类的社会性，只只被群体接纳，对我们来说还不够，我们还希望自己在群体中拥有较高的地位，享受别人的尊重，享有更多的特权。</p>
<p>所以，无论在家中，还是在工作环境中，我们都会努力维护自己的形象、地位、权威。</p>
<p>不过，人有时是分不清现实与虚幻的。</p>
<p>虽然，网络上的网友，可能不会对我们的实际生活，产生任何影响。但那种与身边人互动的感觉，却依然存在。</p>
<p>所以，当我们遇到树立形象的机会，或者遇到可能损伤地位的遭遇时，很多人会把实际生活中的情绪，带入到网络中，更积极的展开形象工作：</p>
<p><strong>A</strong><strong>、厮杀杠精</strong></p>
<p>假如在公司里，总有人找你抬杠，你却不敢回应，时长日久，其他人也会觉得你好欺负，会有更多人来针对你。</p>
<p>远古时期，猴群中的新旧交替，也是从年轻猴子挑衅老猴王开始的。如果猴王不能打败新猴，就没人再承认他的地位。</p>
<p>而在网络当中，如果有人与你抬杠、讽刺挖苦你，甚至是无来由的谩骂，我们的这种情绪，也会被激发出来，并与之一杠到底——我们的潜意识中认为，我们是在为自己的地位、为自己的生存而战。</p>
<p>不过这有时就是一种错觉，因为在那些我们不常去的网络社区，甚至是在匿名的状态，遇到这些情况时，我们的形象（也包括虚拟形象）并不会有任何实际折损。但我们依然会被激发出斗性，与杠精、喷子斗的起劲。</p>
<p><strong>B</strong><strong>、形象升维</strong></p>
<p>如果在身边的环境中，有人分享了错误的信息，而我们用专业的知识进行指正，就能塑造我们博学的形象。如果有什么消息，我们说出了背后的内幕，就能塑造我们消息灵通的形象。</p>
<p>远古时候也是一样，能说出哪里有果树，能知道哪里有危险野兽的人，最会受人尊敬。</p>
<p>而在网络上的评论区里，我们会因为同样的出发点，做出相似的举动：围绕某个话题，详细分析知识点、分析背后的各种故事，这才有了无数评论区的大神。甚至仅仅是用我们创作的一句话调侃，让无数人发笑，也是一种成功。</p>
<p>而这种行为，正在逐渐演变为一种轻创作模式：</p>
<ul>
<li>相比写文章、做视频，它的行为成本更低。</li>
<li>但它也能给人带来形象提升的感觉。即使这种感觉，有时候是假的，也不妨碍人们从中感受到形象提升般的快乐。</li>
<li>在相应的文章、视频、消息下评论，观看对象都是精准目标群体，更容易引起共鸣。</li>
</ul>
<p>因而，越来越多的人，投入到评论区的轻创作当中，并从中体会快乐。</p>
<p>同样也有很多知名品牌，找到了这个机会，在评论区中轻创作，创造有趣、有内涵的评论，收获网友的认可。</p>
<p>相比普通网友，多半是感受形象提升的感觉，而没太多实际受益的状况，这些品牌通过评论区所带来的受益，却是真真实实，能够产生价值的：</p>
<ul>
<li>基于品牌本身的知名度。普通网友评论一通，观看者看个开心、点个赞，却根本记不住是谁评论的。下次再见到他的评论，也未必会认得。但很多品牌本身就被人所熟知，它在评论区中轻创作的表现，也能被知道它的人，和它本身的高度绑定在一起，而不会白白流失。</li>
<li>提高品牌亲和度。这种评论的行为，打破了过往品牌与用户间的距离，增加了更多接触途径，通过拟人化的方式，让网友能近距离对品牌产生更深刻的印象。</li>
</ul>
<h3>2. 人人都爱看评论</h3>
<p>喜欢写评论的人多，喜欢看评论的人同样不少。</p>
<p>正是这两者的相互促进，随时转换，才形成了氛围热烈的评论区。</p>
<p><strong>1</strong><strong>）还是寻找认同</strong></p>
<p>同样是基于认同，不只写评论可以找到认同，看评论，同样可以得到认同感。</p>
<p>人们可以从与自己一样的观点中，找到力量、得到快乐：</p>
<ul>
<li>在评论区中，看到其他人的评论，和自己正在想的一样，就会觉得心情很舒畅。</li>
<li>人们观看视频时，会从弹幕中寻找有没有和自己观点一样的弹幕存在。</li>
<li>淘宝上选购商品，看到买过的人说好，从而下定自己的购买决心，确定自己的购买行为。</li>
<li>互联网嘴替，也正是出于这一动机。从其他人的发言中，找到了相似的观点，甚至是对自己观点的更好表述，感受到共鸣。并且这一观点也得到了大量点赞，就会产生一种自己的观点也被广泛认可的感觉。</li>
</ul>
<p><strong>2</strong><strong>）寻找延申信息</strong></p>
<p>每个人的知识区不同，思维方式也不同。</p>
<p>我们还可以从其他人的评论中，获得关于某个话题的延申知识，找到一些额外乐趣。</p>
<ul>
<li>影视剧评论区里的二次创作，能让我们获得更多观影感受的延续。</li>
<li>微博评论区里的网友爆料，能让我们知道事情的更多内幕。</li>
<li>B站评论区的玩梗，能让我们找到更多的笑点。</li>
<li>淘宝评论区里的买家秀，能让我们看到商品更多的真实情况。</li>
<li>豆瓣网友的影评，能让我们发掘更多的观点。</li>
</ul>
<p>而且，人们看评论的热情，会与写评论的热情，相互循环促进：</p>
<ul>
<li>看评论的人，会在观看中，随时转化为创作者，写下评论。</li>
<li>写评论的人，会根据看评论者的态度，创造出人们愿意看、愿意点赞、愿意二次评论的评论。从而变成更为优质的内容轻创作者。</li>
</ul>
<p>看评论与写评论，持续相互促进，才有了各大评论区热火朝天的人气。</p>
<h2 id="toc-2">二、产品推动评论区</h2>
<p>写评论、看评论，看似是用户与用户之间的事情。</p>
<p>但产品如果就这么躺平，任评论区自由生长，评论区也未必能够兴盛。</p>
<p>我们依然可以做很多幕后工作，引领评论区创造更大价值。</p>
<h3><strong>1.</strong> <strong>定性</strong></h3>
<p><strong>1</strong><strong>）决定优先展现什么</strong></p>
<p>评论区的位置有限，人们观看评论的时间、精力也有限。</p>
<p>要把有限的空间利用起来，就需要对最优先展现的内容，进行取舍。</p>
<p>而这些展现方式，各有优缺点：</p>
<ul>
<li><strong>优先展现最新评论</strong>——有时效性，能够促进用户间的互动，减少评论区延迟属性导致用户无法有效互动的问题。同时，也便于跟踪某些热点话题的最新动态。</li>
<li><strong>优先展现最早评论</strong>——符合逻辑顺寻。如果用户大量阅读评论，将会有很好的阅读体验。</li>
<li><strong>优先展现最热评论</strong>——优质内容吸引。根据用户的点赞数进行反馈，把人们最关注、最认可的评论放在最前列，能够让其他看评论的人，更容易第一时间受到吸引。</li>
<li><strong>优先展现最多回复</strong>——争议话题吸引。最多人回复的内容，往往具有极大争议。能吸引更多的看评论者吃瓜，也能吸引更多人参与到评论当中。</li>
<li><strong>复合展现</strong>——展现逻辑分割，一部分优先展现热度较高的评论，吸引看评论的人。另一部分按照阅读逻辑展现，能够让愿意了解详情的用户，也能找到事情的脉络、有顺序的阅读。例如虎扑社区，评论区就进行了这样的划分。</li>
</ul>
<p><strong>2</strong><strong>）决定展现形式</strong></p>
<p>不同的展现形式，同样适合不同的使用情况。</p>
<ul>
<li><strong>平铺式</strong>——不分级，不引用，简单、直接、清爽。适用于讨论量较少的环境。例如微信朋友圈、微信公众号。</li>
<li><strong>盖楼式</strong>——引用前期发言，清晰展现逻辑链。用户阅读时逻辑线索清晰，能够看到每个讨（Si）论（bi）的前因后果。但缺点是较占空间，会大量展现重复内容。</li>
<li><strong>讨论组式</strong>——划分一级回复、二级回复，重点话题可以重点讨论，便于互动。例如：贴吧、微博。</li>
</ul>
<p>根据自己平台的特性，选择不同的展现逻辑、展现形式，提升用户的阅读体验，才能让更多的人愿意看、愿意写、看的过瘾、写的来劲。</p>
<h3>2. 激励</h3>
<p><strong>1</strong><strong>）从无到有的激励</strong></p>
<p>如果在产品发展初期阶段，评论区空空，产品需要付出更多的努力，对评论区的行为进行引导。</p>
<ul>
<li><strong>利益引导</strong>——发言给积分，积分换升级、积分换奖励。通过一些外部的奖励，促进用户的评论行为。</li>
<li><strong>佯作热闹</strong>——让自己的运营人员先进行评论，把评论区搞热。若一个评论区完全空白，用户也会觉得没有人气，找不到自己想要的共鸣、形象，懒得评论。所以需要自己的员工先装好评论人，而且是用心创作评论。当用户看到有人评论，而且从中找到有趣、有共鸣的评论时，才会迈出自己从0到1的评论步伐。</li>
</ul>
<p><strong>2</strong><strong>）从有到多的激励</strong></p>
<p>当产品已经有了一定的评论群体，评论区的工作重点就需要转移。</p>
<ul>
<li><strong>停止外部激励</strong>——物质激励只能作为短时间的促进，长期使用，会阻碍用户评论过程中获得内部激励，那些找共鸣、树形象、发感慨的感觉。</li>
<li><strong>优质评论激励</strong>——激励需要从人们写评论的出发点入手，把优质的评论置顶、推荐，从而让更多人看到，让写评论者收获更多点赞，他们才能更加满足，并继续、加强自己的评论行为。</li>
</ul>
<h3>3. 塑造评论文化</h3>
<p>当评论区积累了一定的活跃用户后，就该考虑评论区文化的塑造问题。</p>
<p><strong>1</strong><strong>）引导</strong></p>
<p>根据自身产品特性，鼓励某种类型的评论，并让人们都看到这种鼓励。</p>
<p>会有更多的人，为了获得鼓励，而试图找寻平台鼓励的原则，从而去创作平台想要的评论类型。</p>
<p>比如网易云音乐，把评论作为广告打出。不仅让感到新鲜、有趣的新人下载了网易云，也让更多的评论者，学习使用类似风格进行评论，营造了更好的评论区文化氛围。</p>
<p><strong>2</strong><strong>）约束</strong></p>
<p>一些不良的评论，不仅会损害观看者的观看体验，更会损害评论创作者的创作热情。一个作者认真创作，却被喷子无理由怒骂，很有可能一气之下不再发言。 所以，这些不良评论，就要主动控制。</p>
<p>但具体什么是不良评论，需要根据自己的产品特性来定义。</p>
<ul>
<li>在抖音、微博的评论区中，与人抬杠，在争吵中让话题升级，反而会引来更多人观看。所以一定限度内的怼人，符合这些平台的特性，反而应该保留。</li>
<li>在一些知识分享平台，争吵、引战的杠精，会严重影响创作者的创作热情。一个杠精，可能逼走10个轻度创作者。这种情况下，就必须进行处理，把杠精和杠精行为，赶出评论区。</li>
</ul>
<h2 id="toc-3">三、尾声</h2>
<p>写评论的人有故事，回评论的人有怒气。</p>
<p>看评论的人有心事，赞评论的人有快意。</p>
<p>只有管理评论区、引导评论区的运营人，假装自己有故事、有怒气、有心事、有快意，默默的写下一条条言不由衷的评论。</p>
<p>纵然编出的评论，吸引了无数用户的再评论，又能怎样？</p>
<p>“我评论了你的评论，却唯独没有评论你”。</p>
