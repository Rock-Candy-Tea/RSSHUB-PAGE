<div>   
<p>俄罗斯对乌克兰的战争开始六个月后，关于战斗的发展及其结束情景的不确定性正在增加。</p>
<p>为了阐明美国对地面力量平衡性质的理解、华盛顿对当前事态发展的立场以及未来几周和几个月的情景，半岛电视台采访了美国最著名的军事学院西点军校国际关系教授罗伯特·皮尔森。</p>
<p>这位学者的研究重点是俄罗斯和前苏联加盟共和国的外交和国内政策，他的新书《21世纪的俄罗斯伟大战略》即将出版。</p>
<p>皮尔森拥有耶鲁大学博士学位和斯坦福大学硕士学位，并且是现代战争研究所的非常驻研究员。</p>
<p>以下是半岛电视台采访的对话文本：</p>
<figure id="attachment_311465" aria-describedby="caption-attachment-311465" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-311465" src="https://chinese.aljazeera.net/wp-content/uploads/2022/08/1-275.png?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-311465" class="wp-caption-text">皮尔森：冲突陷入僵局，双方都无法在当地取得显著成果 (社交网站)</figcaption></figure>
<ul>
<li>
<h3>从战争开始到我们现在所看到的，战斗进程是如何发展的？</h3>
</li>
</ul>
<p>最初，俄罗斯认为它可以在几天内迅速摧毁乌克兰军队，迅速占领基辅，并用民选政府取代其傀儡政府，虽然俄罗斯控制了乌克兰南部和顿巴斯地区的大片领土，但它在改变乌克兰政权的主要政治目标上很早就失败了。</p>
<p>面对这样的挫折和顽强的乌克兰防守，俄罗斯不得不打一场缓慢、艰苦且代价高昂的战争，以扩大其占领。今天，冲突已接近胶着，任何一方都无法在当地取得巨大的领土收益，而且在可预见的未来也不太可能改变。</p>
<p>随着俄罗斯试图巩固对战争初期占领的被占领土的控制，前线将发生激烈的战斗，但最近乌克兰对俄罗斯前线后方的军事打击，使这变得更加困难。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-311469" src="https://chinese.aljazeera.net/wp-content/uploads/2022/08/image-1-20.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul>
<li>
<h3>许多专家认为，乌克兰军队无法与俄罗斯抗衡，但它确实做到了。那是怎么发生的？</h3>
</li>
</ul>
<p>最初的俄罗斯入侵似乎没有得到很好的计划和执行，士兵装备不良，设备维护不善，俄罗斯也未能部署一支庞大而足以入侵和占领欧洲第二大国的部队，这部分是由于对必要兵力的错误估计，也是由于莫斯科决定避免总动员和大规模征兵。</p>
<p>相比之下，包括专业人士、志愿者和专业战士在内的乌克兰军队则表现好于预期，这部分归功于西方自 2014 年以来对乌克兰军队的训练和军备的投资，以及美国和欧洲国家提供的高效重型武器，但最重要的是乌克兰人的坚持和他们捍卫民主和祖国的决心，这是俄罗斯士兵所缺乏的。</p>
<ul>
<li>
<h3>与乌克兰相比，俄罗斯在人力、自然资源和军火库方面拥有巨大优势，经过6个月的战斗，情况仍然如此吗？</h3>
</li>
</ul>
<p>总之，俄罗斯拥有更大的物质基础和人口基础，使其能够继续对乌克兰的战争，但是他们使用了大部分先进和精确的武器，这使得他们现在求助于旧的炸弹、导弹和大炮库存，其能力不如被摧毁的先进武器。</p>
<p>随着俄罗斯继续消耗其弹药和军事装备，西方对军事工业相关技术的制裁将使俄罗斯难以更新其武器库，俄罗斯还担心，如果它要求进行全面动员和大规模招募，可能会出现内部政治不稳定，因此，克里姆林宫似乎也在对其军事人员进行克制。</p>
<p>另一方面，乌克兰显然要依赖西方的重型武器来继续战斗，只要这些武器继续流动，乌克兰就可以继续战斗，如果西方停止支持，他们就会处于危险的境地。</p>
<p>我相信，如果这是他们保护自己免受俄罗斯入侵者的最后一个想法，乌克兰人将用棍棒和石头战斗，但在物质、经济和军事资源方面，他们目前远不如俄罗斯自给自足。</p>
<figure id="attachment_311471" aria-describedby="caption-attachment-311471" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-311471" src="https://chinese.aljazeera.net/wp-content/uploads/2022/08/1-276.png?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-311471" class="wp-caption-text">俄罗斯轰炸乌克兰后的情景 (阿纳多卢通讯社)</figcaption></figure>
<ul>
<li>
<h3>您预计这场战争会持续数月或数年吗？</h3>
</li>
</ul>
<p>我想，这场战争将持续数年，我对此毫无疑问。</p>
<ul>
<li>
<h3>俄罗斯对乌克兰发动战争的主要战略后果是什么？</h3>
</li>
</ul>
<p>自布尔什维克革命以来，俄罗斯已将自己与西方隔离到了前所未有的程度。与此同时，俄罗斯更加接近中国，这加速了一种已经存在多年的越来越靠近中国的趋势，然而，俄罗斯冲突和乌克兰战略缺陷，让中国与俄罗斯的伙伴关系受到影响。</p>
<p>尽管两国将继续在不结盟情况下建立伙伴关系，但北京似乎更具主导性，换言之，这场战争清楚地向世界表明，未来几十年将是中美两极战略竞争的国际体系，俄罗斯处于次要地位。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-311475" src="https://chinese.aljazeera.net/wp-content/uploads/2022/08/image-2-10.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul>
<li>
<h3>决定向乌克兰开战是否是俄罗斯总统普京的误判，为什么？</h3>
</li>
</ul>
<p>普京“入侵”的决定是一个不同寻常的误判，因为它未能实现——而且已经破坏了——他执政20多年来试图实现的几乎所有重大战略目标。</p>
<p>俄罗斯在周边国家乃至全球的影响力处于后苏联时代最低状态，而这场战争的高昂代价和随之而来的制裁，严重损害了俄罗斯未来的经济发展，虽然战争并没有让北约变得更加强大和团结，但寻求加入北约的国家数量也在增加，与此同时，俄罗斯一无所获，却付出了巨大的代价，看不到尽头，这确实是一个大错误。</p>
<figure id="attachment_311477" aria-describedby="caption-attachment-311477" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-311477" src="https://chinese.aljazeera.net/wp-content/uploads/2022/08/1-277.png?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-311477" class="wp-caption-text">一名乌克兰士兵检查海马斯火箭发射器，这是美国支持乌克兰的武器之一 (盖帝图像)</figcaption></figure>
<ul>
<li>
<h3>欧美是否会继续支持乌克兰，为什么？</h3>
</li>
</ul>
<p>目前，美国两党强烈支持乌克兰，但在共和党的孤立主义派别中也有一些声音，质疑我们是否应该承担保卫乌克兰的代价，特别是因为他们认为欧洲必须承担更多的负担。</p>
<p>虽然这是少数人意见，但如果共和党在即将到来的秋季中期选举中获得参众两院多数席位，这些孤立主义者可能会在国会获得更大的影响力，这可能会减缓或停止美国对乌克兰的援助。</p>
<p>今年冬天将是对欧洲的真正考验，由于俄罗斯天然气和石油中断和减少，欧洲正在努力应对电力短缺问题，这种经济痛苦可能足以迫使欧洲各国政府取消一些制裁和其他限制，以免在下次选举中被弹劾。</p>
<p>至于西方国家为什么继续支持乌克兰，援引很简单。二战后建立了存在于西方的国际体系，并在共产主义崩溃后得到了加强，符合我们的国家利益，俄罗斯对乌克兰领土的入侵、占领和吞并，威胁到这一体系及其提供的稳定。</p>
<p>简而言之，在这次“公然侵略”之后，我们所处的世界比俄罗斯可以侵犯其邻国的主权而不会产生任何后果的世界要好，因此，当我们保卫乌克兰时，我们就是在保卫国际秩序和独立国家的主权。</p>
<ul>
<li>
<h3>您预计这场战争在接下来的几个月和几年会发生什么情况？</h3>
</li>
</ul>
<p>尽管未来总是难以预测，但我认为，最有可能出现的情况是长期而激烈的僵局，尽管双方都会发动代价高昂的攻击，但交换的领土很少，沿接触线的战斗将愈演愈烈，而乌克兰特种部队将在俄罗斯占领的地区发动越来越大胆和令人震惊的袭击，并可能进一步深入俄罗斯领土。</p>
<p>如果发生这种情况，俄罗斯可能会对主要人口中心发动越来越多的毁灭性惩罚性打击，以此作为回应，乌克兰中部和西部城市——更不用说基辅——也可能受到更多的袭击，以报复乌克兰的成功袭击。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-311482" src="https://chinese.aljazeera.net/wp-content/uploads/2022/08/image-3-4.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul>
<li>
<h3>这场战争如何以及何时结束？</h3>
</li>
</ul>
<p>对我而言，没有尽头。所以，无法说明这场战争将如何收场，这取决于俄罗斯和乌克兰交战的时期，以及目前许多不可预测的变数。但我认为，目前双方都对外交解决途径不感兴趣，所以，战争将肆虐并继续下去。</p>
<ul>
<li>
<h3>对于那些试图了解当前和未来战争性质的人来说，最初的教训是什么？</h3>
</li>
</ul>
<p>尽管我们经常谈论战争的“新形式”， 一个非常古老的原则仍然存在，那就是战争的输赢在于“后勤”，如果没有其他的话，这场战争让我们想起了这个事实。</p>
