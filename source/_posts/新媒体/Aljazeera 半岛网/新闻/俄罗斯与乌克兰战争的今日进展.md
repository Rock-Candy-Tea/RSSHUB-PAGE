<h3>以下是俄罗斯与乌克兰战争今日的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>普京向芬兰表示：放弃中立将是一个错误</strong></p>
<p>克里姆林宫称，俄罗斯总统普京告诉芬兰总统绍利·尼尼斯托，赫尔辛基选择放弃中立地位并加入北约将是一个错误。</p>
<p>普京表示，芬兰并不存在安全威胁，其外交政策立场的潜在变化可能会对双边关系产生负面影响。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>七国集团警告全球粮食危机</strong></p>
<p>七国集团警告称，乌克兰战争加剧了全球粮食和能源危机，并威胁到了贫穷国家。</p>
<p>德国外交部长贝尔伯克主持了七国集团的高级外交官会议，并表示这场战争已经成为了一场“全球危机”。</p>
<p>贝尔伯克表示，除非找到释放乌克兰粮食的办法，否则在未来几个月内将有多达5000万的人口面临饥饿，尤其是在非洲和中东地区。需要指出的是，乌克兰粮食在全球供应中占据了相当大的份额。</p>
<p>该组织在一份声明中表示，“俄罗斯的侵略战争造成了近现代历史上最严重的食品和能源危机之一，现在已经威胁到了全球最脆弱的人群。”</p>

<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>芬兰称其总统与普京就加入北约的问题举行了会谈</strong></p>
<p>芬兰总统办公室表示，芬兰总统尼尼斯托与俄罗斯总统普京就芬兰申请加入北约一事进行了交谈，这项申请预计将于本周末正式宣布。</p>
<p>尼尼斯托在其办公室发布的一份声明中指出，“此次谈话是直接的，并且没有加剧负面情绪。避免紧张关系是重要的问题。”</p>
<p>这项声明补充称，“通话是由芬兰发起的。”</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>七国集团：中方不应该支持俄罗斯</strong></p>
<p>七国集团呼吁中方不要帮助俄罗斯，包括削弱国际制裁或为莫斯科在乌克兰的行动辩护。</p>
<p>该集团认为，中方应该支持乌克兰的主权和独立，而不是“协助俄罗斯发起的军事行动”。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>半岛电视台记者：哈尔科夫部分恢复“正常”</strong></p>
<p>半岛电视台记者自乌克兰东部城市哈尔科夫附近报道称，俄罗斯军队现在正从这片饱受战争蹂躏的地区撤出，该市有一种恢复“正常”的感觉。</p>
<p>记者指出，“州长表示，人们开始返回，但是他担心俄罗斯人可能在乌克兰人夺回的一些地区埋下了地雷。”</p>
<p>“但是这座城市正在从一定程度上部分恢复正常。”</p>

<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>七国集团“绝不承认”由俄罗斯通过武力改变的边界</strong></p>
<p>七国集团(G7)表示，绝不承认俄罗斯在对乌克兰发动的战争中试图改变的边界，并承诺对基辅提供持久的支持。</p>
<p>七国集团外交部长在德国北部举行了为期3天的会谈，并在会后发表声明称，“我们绝不承认俄罗斯试图通过军事行动而改变的边界，我们将在支持包括克里米亚在内的乌克兰的全部领土完整和国家主权的情况下，保持我们之间的接触。”</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>立陶宛总统瑙塞达：乌克兰战争让欧洲看清了普京的意图</strong></p>
<p>北约在立陶宛的驻军已经增加至大约3000名士兵。但是维尔纽斯仍加入了其他波罗的海国家的行列，并要求增派部队。</p>
<p>那么，俄罗斯对其他国家的攻击真的迫在眉睫吗?怎样才能保证该地区的安全?</p>
<p>立陶宛总统瑙塞达接受了半岛电视台记者的采访。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>加拿大外长：芬兰、瑞典应迅速加入北约</strong></p>
<p>加拿大外交部长指出，虽然在瑞典和芬兰加入北约方面应该已经达成了共识，但是两国的入盟进程还应当加速。</p>
<p>梅勒妮·乔利在德国北部举行的七国集团外长会议间隙对记者们表示，“重要的是我们要达成共识。”</p>
<p>“我们希望芬兰和瑞典不仅能够加入，而且能够尽快加入。在芬兰和瑞典寻求安全保障的情况下，这是至关重要的问题。”</p>

<p><strong>芬兰电网运营商：俄罗斯暂停向芬兰供电</strong></p>
<p>芬兰电网运营商的一名官员告诉法新社记者，在能源公司“RAO Nordic”因欠款问题而威胁要切断电力供应之后，俄罗斯已于当夜暂停对芬兰的电力供应。</p>
<p>该运营商的项目计划经理蒂莫·考科嫩表示，“目前的供应为零，并且已经按照计划从午夜开始了。”</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>“回家”: 格鲁吉亚分裂地区将对加入俄罗斯举行投票</strong></p>
<p>格鲁吉亚分离地区南奥塞梯的领导人将今年7月17日定为是否加入俄罗斯联邦的公投日期。</p>
<p>比比洛夫办公室在一份声明中指出，“阿纳托利·比比洛夫签署了一项法令，要求在南奥塞梯共和国举行全民公投”，并引述了“他的人民”加入俄罗斯的“历史性愿望”。</p>

<p><strong>美俄国防部长自入侵以来首次举行会谈</strong></p>
<p>俄罗斯国防部长绍伊古在13日与美国国防部长劳埃德·奥斯汀进行了会谈，在此前数月内，俄罗斯防长拒绝与美国防长进行直接接触。</p>
<p>不过，官员们表示，这次通话似乎并不意味着莫斯科在乌克兰的战争发生了任何变化。</p>
<p>美国国防部的一名高级官员表示，尽管奥斯汀认为这场长达一小时的对话，对保持沟通渠道畅通的努力非常重要，但它却没有解决任何“尖锐的问题”，也没有导致俄罗斯在战争进入第12周之际的所作所为发生任何改变。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>德国总理：自战争开始以来，普京没有任何改变</strong></p>
<p>德国总理奥拉夫·舒尔茨表示，自战争开始以来，俄罗斯领导人普京的决心没有发生任何改变。</p>
<p>舒尔茨在接受媒体采访时表示，很明显，俄罗斯没有实现其声明的任何战争目标，其中之一便是确保乌克兰领土作为莫斯科和北约扩张之间的缓冲地带。</p>
<p>舒尔茨表示，“为了普京想要扩张俄罗斯帝国的疯狂想法，俄罗斯和整个世界现在都在付出高昂的代价。”</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>库列巴：如果打破针对俄罗斯的进口禁令，那么欧盟的“团结”就将面临风险</strong></p>
<p>乌克兰外交部长库列巴在接受半岛电视台外交编辑詹姆斯·贝斯的采访时表示，如果匈牙利阻止欧盟禁止进口俄罗斯石油的提议，那么欧盟在俄罗斯问题上的“团结”就将宣告破裂。</p>
<p>库列巴在德国魏森豪斯举行的七国集团峰会上强调，“我相信这将对欧盟造成很大的损害。”</p>

<p><strong>当地媒体：俄罗斯部长称，一旦北约更靠近边境，莫斯科将作出回应</strong></p>
<p>俄新社援引俄罗斯副外长格鲁什科的话报道称，如果北约在俄边境附近部署核力量和基础设施，那么莫斯科就将采取充分的预防措施。</p>
<p>格鲁什科表示，莫斯科对芬兰和瑞典没有敌对意图，也没有看到这两个国家加入北约的“真正”理由。</p>

<p><strong>俄新社：俄罗斯国家杜马副主席访问被俄罗斯占领的乌克兰赫尔松地区</strong></p>
<p>根据俄罗斯国家通讯社的报道，俄罗斯资深议员安娜·库兹涅佐娃访问了乌克兰的赫尔松地区，以讨论当地人口的社会和医疗需求。</p>
<p>俄罗斯在今年4月表示完全取得了对赫尔松地区的控制权，这也将是莫斯科要吞并的第一个地区。该地区曾发生过零星的反俄抗议活动。</p>
<p>根据俄新社的报道，俄罗斯国家杜马(即下议院)副议长库兹涅佐娃就当地所需的食品、医疗和其他产品的供应问题举行了讨论。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>芬兰执政党将宣布加入北约的决定</strong></p>
<p>预计芬兰执政党社会民主党即将宣布支持加入北约的决定。</p>
<p>芬兰总统绍利·尼尼斯托和总理桑娜·马林在本周早些时候表示支持“毫不拖延地”加入北约，并认为这将加强该北欧国家的安全。</p>
<p>来自马林所在的社会民主党的支持，将意味着这项决定将在芬兰议会内获得更为广泛的支持，并为芬兰在未来几天内提出入盟申请铺平道路。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>匈牙利反对欧盟禁止进口俄罗斯石油的计划</strong></p>
<p>外交官员们表示，欧盟迅速禁止进口俄罗斯石油的希望可能会落空，因为匈牙利要求为自身的燃料供应提供昂贵的保障。</p>
<p>欧盟外交政策负责人博雷尔13日在德国召开的七国集团峰会期间对记者们表示，“我相信我们会达成协议，我们需要这项协议，我们会达成它的。”</p>
<p>但他补充称，如果欧盟外交官员无法克服来自某些成员国的阻力，那么下周一将在布鲁塞尔开会的外交部长们就将需要“提供政治动力”。</p>

<p><strong>报告：超过70多万乌克兰战争难民身处德国境内</strong></p>
