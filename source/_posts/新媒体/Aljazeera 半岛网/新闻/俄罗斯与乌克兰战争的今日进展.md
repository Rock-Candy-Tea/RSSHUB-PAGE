<li>乌克兰总统泽连斯基透露，在基辅等待欧盟决定授予其候选国家地位的同时，预计俄罗斯将加强针对乌克兰的攻击。</li>
<li>卢甘斯克州长海岱表示，随着俄罗斯军队加强对东部城市的攻击，他正在为“最坏的情况”做准备。</li>
<li>俄罗斯声称对北顿涅茨克的进攻取得了成功，此前，它控制了位于该市东郊的一个名为梅约金的定居点。</li>
<li>受俄罗斯支持的分离主义分子表示，梅约金的部分乌克兰士兵及其指挥官已经投降，现在正在作证以指控他们仍在位于北顿涅茨克地区的战友。</li>

<h3>以下是这场战争进入今日以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>本·斯蒂勒出现在利沃夫</strong></p>
<p>根据路透社的报道，好莱坞演员本·斯蒂勒18日被发现出现在乌克兰西部利沃夫市中心，并且边走边打电话。</p>
<p>根据社交媒体上发布的一篇帖子，他正在与联合国难民机构合作，以支持乌克兰难民。</p>
<p>联合国难民署在推特上指出，斯蒂勒是联合国难民署亲善大使，他曾前往波兰和乌克兰边境，会见被迫逃离乌克兰的难民，“分享他们的故事，并加强团结的呼吁”。后来还有人看到他穿过利沃夫的中心广场。</p>
<p>这位演员还在推特上宣布，他“在世界难民日之前抵达波兰，与生活受到乌克兰战争影响的人们见面”。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-292035" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/hollywood-star-in-ukraine.png?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国陆军司令：英国必须拥有能够击败俄罗斯的军事力量</strong></p>
<p>当地媒体援引英国新任陆军司令对军队的讲话称，英国必须拥有一支能够在欧洲作战并击败俄罗斯的军队。</p>
<p>根据英国《独立报》19日的报道，本月上任的英国陆军司令帕特里克·桑德斯向其部队表示，“我是1941年以来，第一位在欧洲大陆大国卷入陆战的阴影下接管陆军的总参谋长。”</p>
<p>他还表示，“俄罗斯入侵乌克兰凸显了我们的核心目标——通过准备战斗并赢得陆地战争来保护英国。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯军队炮轰乌克兰边境地区</strong></p>
<p>地区州长表示，俄罗斯军队19日多次炮击乌克兰边境地区苏米的一座城镇，摧毁了至少10栋私人建筑。</p>
<p>这位州长在Telegram上表示，“他们向瑟雷迪纳-布达镇（Seredyna Buda）的居民区发射了迫击炮。至少有10座平民私人住宅和附属建筑遭到破坏。一座房子被大火烧毁。”</p>
<p>当天晚些时候，这位官员还表示，初步数据显示没有人员伤亡，但这一点仍在核实中。</p>
<figure id="attachment_292037" aria-describedby="caption-attachment-292037" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-292037" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/fire-on-house.png?w=770&resize=770%2C440" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-292037" class="wp-caption-text">2022年6月19日发布的一段视频截图显示，乌克兰苏米地区瑟雷迪纳-布达镇的一处建筑遭到炮击后起火 (路透社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯保证继续向匈牙利输送天然气</strong></p>
<p>匈牙利外交部长彼得·西亚尔托在接受公共服务电台采访时表示，俄罗斯已承诺继续向匈牙利输送天然气，莫斯科国有能源巨头俄罗斯天然气工业股份公司将会履行对匈牙利的合同义务。</p>
<p>西亚尔托１９日表示，俄罗斯天然气工业股份公司首席执行官与俄罗斯副总理诺瓦克都在电话中向他保证了这一点。但是这位部长没有透露相关电话通话是在什么时候进行的。</p>
<p>根据去年与俄罗斯天然气工业股份公司签署的一项协议，匈牙利每年通过保加利亚和塞尔维亚获得35亿立方米天然气，并通过奥地利的管道获得另外10亿立方米的天然气。</p>
<p>俄罗斯天然气工业股份公司已经切断了通过丹麦奥斯特和壳牌能源以向德国供应天然气的合同，以回应西方在俄罗斯对乌克兰开展军事行动后对俄罗斯实施的制裁。此外，它还切断了对荷兰天然气贸易商“Terra”以及对保加利亚、波兰和芬兰的供应，原因是它们拒绝根据一项新的要求而使用卢布支付俄罗斯天然气货款。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟高级人权官员在乌克兰进行事实调查</strong></p>
<p>欧盟一名高级人权官员表示，将对乌克兰犯下的战争罪行进行彻底调查。</p>
<p>欧盟特别人权代表埃蒙·吉尔摩(Eamon Gilmore)指出，“当我们谈论战争罪行时，我们谈论的不仅仅是那些犯下罪行的人……他们当然要承担责任。但我们也在谈论那些处于指挥链中的人员，如果有必要的话，甚至是最高领导层。”</p>
<p>吉尔摩在基辅附近的伊尔宾镇的废墟和受损汽车周围走动后发表了讲话，而该镇在军事行动初期曾发生过激烈的战斗。</p>
<p>这场伊尔宾之行旨在强调乌克兰及其支持者所说的俄罗斯军队犯下的大规模暴行，德国总理舒尔茨称现场是“难以想象的残酷”和“毫无意义的暴力”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报道：日本首相着眼于与韩国、澳大利亚、新西兰领导人接触</strong></p>
<p>据日本《读卖新闻》报道，日本首相岸田文雄正考虑在今年的6月北约会议期间，与韩国、澳大利亚和新西兰领导人举行峰会，以展现四国对华的团结立场。</p>
<p>上述四个亚太国家的领导人被邀请参加今年6月在马德里举行的北约会议，而该联盟成员国召开这场会议的目标是在乌克兰危机上传递国际团结的信号。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>受俄罗斯支持的武装分子：投降的乌克兰战士指证其战友</strong></p>
<p>根据俄罗斯国家通讯社塔斯社的报道，在梅约金(Metyolkine)定居点向受俄罗斯支持的部队投降的乌克兰武装分子，正在指证其躲藏在东部城市北顿涅茨克的化工厂内的战友。莫斯科方面称，梅约金目前已被处于其控制之下。</p>
<p>根据俄罗斯新闻机构的报道，受俄罗斯支持的“卢甘斯克人民共和国”的民兵武装19日表示，当俄罗斯在北顿涅茨克郊区占领梅约金期间，乌克兰驻守卢甘斯克的埃达尔营（Aidar）士兵已于18日向该民兵武装投降，但是并未透露乌克兰军队中有多少人投降，只是声称其中包括该部队的指挥官。</p>
<p>塔斯社20日援引亲近该民兵武装的消息人士的话报道称，受俄罗斯支持的分裂主义分子现在正在使用埃达尔营提供给他们的信息，以与仍在Azot化工厂内乌克兰士兵进行谈判。</p>
<p>数百名平民和部分乌克兰士兵一直藏身北顿涅茨克的这座化工厂内。卢甘斯克州长表示，俄罗斯军队每天都在对这家工厂进行轰炸。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯官员、叶利钦的亲密助手布尔布利斯去世、享年76岁</strong></p>
<p>俄罗斯前总统叶利钦的高级助手根纳季·布尔布利斯（Gennady Burbulis）去世，享年76岁，他曾帮助制定和签署导致前苏联于1991年正式解体的条约。</p>
<p>作为1991年至1992年的俄罗斯国务卿和第一副主席，布尔布利斯在领导这个新成立的后苏联国家方面发挥了重要的作用。</p>
<p>他与叶利钦，是俄罗斯在1991年12月8日与乌克兰、白俄罗斯达成的解散苏联的协议的签字人。这项协议是在别洛韦日签署的，也就是现在的白俄罗斯境内。</p>
<p>布尔布利斯是这项协议的第三位关键人物，他已在过去几周内去世。乌克兰前总统克拉夫丘克和白俄罗斯前总统舒什科维奇也都已在今年5月去世。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰调查俄罗斯士兵的性暴力罪行</strong></p>
<p>乌克兰副检察长指出，基辅对俄罗斯士兵提起了19项刑事诉讼，因为俄罗斯士兵在临时占领的领土上强奸了至少14名妇女。</p>
<p>这位乌克兰官员在推特上表示，“每名罪犯都必须受到制裁。”</p>
<p>联合国驻乌克兰人道主义协调员乌斯娜特·鲁布拉尼18日表示，联合国人权事务高级专员办事处(OHCHR)已经核实了在乌克兰发生的针对女性和男性的性暴力案件。</p>
<p>鲁布拉尼表示，“由于积极的敌对行动、大规模的国内流离失所、与性暴力有关的耻辱感以及转送途径的中断，幸存者往往无法或不愿向执法当局或服务提供者报告此类事件。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基访问前线，英国、北约警告乌克兰战争将长期持续</strong></p>
<p>英国首相鲍里斯·约翰逊、北约秘书长延斯·斯托尔滕贝格警告西方盟国称，要为乌克兰的长期作战做好准备。</p>
<p>当约翰逊与斯托尔滕贝格在18日分别发出警告之际，乌克兰总统泽连斯基视察了该国南部的尼古拉耶夫和敖德萨地区的前线，并在当地宣称，乌克兰人“一定会”战胜入侵的俄罗斯军队。</p>
<p>约翰逊在《星期日泰晤士报》上撰文，呼吁继续对乌克兰提供支持，并称该国的外国支持者应该保持勇气，确保乌克兰拥有“生存下来并最终获胜的战略耐力”。</p>

<p><strong>欧盟寻求释放因俄罗斯的海上封锁而被困的乌克兰粮食</strong></p>
<p>欧盟外长将于２０日在卢森堡举行会议，讨论如何释放因俄罗斯黑海港口封锁而滞留在乌克兰境内的数百万吨粮食。</p>
<p>欧盟支持由联合国斡旋以达成一项协议，恢复乌克兰的海上出口，以换取俄罗斯的食品和化肥出口，但这需要得到莫斯科的批准。</p>
<p>土耳其与基辅和莫斯科都有良好的关系，它表示，如果达成协议，土耳其准备在伊斯坦布尔的“观察机制”中发挥作用。目前尚不清楚欧盟是否会参与军事行动以确保达成这样一项协议。</p>
<p>一名欧盟官员告诉路透社记者，“未来是否会有为这些商船护航的需求，这还是一个问号，我认为我们还没有走到那一步。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯在争夺乌克兰东部关键城市的战斗中推进</strong></p>
<p>俄罗斯方面表示，俄罗斯军队占领了乌克兰工业城市北顿涅茨克附近的一个村庄，这是莫斯科控制乌克兰东部的主要目标。</p>
<p>日本国防部１９日表示，他们已经夺得了梅约金(Metyolkine)的胜利——在这场战争爆发之前，梅约金只是一个居民人数不足800的定居点。俄罗斯国家通讯社塔斯社报道称，许多乌克兰武装分子在当地投降。</p>
<p>莫斯科１９日表示，其夺取北顿涅茨克的攻势取得了成功。</p>
<p>卢甘斯克州长海岱向乌克兰电视台记者表示，战斗使得人们不可能从该市撤离，但是“俄罗斯人声称他们控制了该市的所有说法都是谎言。他们控制了城镇的主要部分，但并非整个城镇。”</p>

<p><strong>泽连斯基预计俄罗斯将会加强对乌克兰的攻击</strong></p>
<p>泽连斯基表示，在基辅等待欧盟授予其候选国家地位的决定之际，预计俄罗斯将加强针对乌克兰的攻击。</p>
<p>泽连斯基在其夜间视频讲话中表示，“显然，本周我们应该预料到俄罗斯的敌对活动会加剧。”</p>
<p>“而且不仅仅是针对乌克兰，也针对其他的欧洲国家。我们正在做准备。我们已经准备好了。”</p>
