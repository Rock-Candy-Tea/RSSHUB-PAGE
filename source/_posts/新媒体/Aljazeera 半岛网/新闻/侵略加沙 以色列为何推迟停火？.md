<div>   
<p>自对加沙地带实施军事侵略第一周结束以来，以色列推迟达成停火协议，并拒绝联合国旨在让加沙地带恢复平静的所有倡议——其中包括美国倡议，与此同时，随着战争进入第二周，以色列计划加快轰炸步伐，加紧突袭并瞄准居民区，并强调排除实现长期停火的可能性。</p>
<p>尽管埃及采取了主动行动，并在侵略开始两天后进行了区域停火调解，但在抵抗派系于以色列所谓的耶路撒冷“统一日”纪念日向被占领的耶路撒冷发射火箭弹袭击之后，以色列坚持不指定目标的继续侵略，这与以前的军事行动形成了鲜明对比。</p>
<p>在以前加沙地带采取的军事行动结果，以及巴勒斯坦抵抗运动在“耶路撒冷之剑”战役中所施加的新等式之间，本次冲突有所不同，很明显，以色列总理本杰明·内塔尼亚胡致力于通过推迟停火来构筑“胜利画面”，并挫败伊斯兰抵抗运动（哈马斯）的任何战略成就。</p>
<ul>
<li>
<h3>加沙战线宣传的以色列战略目标是什么？是否了解“城墙护卫”？</h3>
</li>
</ul>
<p>根据以色列政府的定义，自2014年“铸铅行动”结束以来宣布的战略目标是：加强对加沙地带哈马斯的威慑力，在该领域恢复和平与安全稳定及长期平静，保持这种状况而无需了解加沙地带局势的根本性变化。</p>
<p>在“城墙护卫”战役第一周战斗结束后，以色列开始计算自2014年以来一直在寻找的目标，与此同时，以色列——在军事战役进入第十天之后——坚持在寻找“胜利画面”，以色列研究中心达成共识强调称，特拉维夫已经耗尽了针对另一方使用武力的效力，其并没有采取地面机动，没有对加沙地带发动地面入侵。</p>
<p>随着在加沙战斗继续进行，内塔尼亚胡的困境再次出现：军事行动何时，如何以及在什么条件下全面结束军事战役和实现停火？</p>
<p>但是，在“城墙护卫”——以色列为这一军事行动命名——期间，以色列并未摆脱指导其在先前与哈马斯对抗中进行军事行动的逻辑，其中最重要的是实现威慑。再一次，很明显，以色列的做法和行为没有试图产生和创造建设性的视野和政策选择。</p>
<p><img class="aligncenter size-arc-image-770 wp-image-194422" src="https://chinese.aljazeera.net/wp-content/uploads/2021/05/image-1-16.jpg?w=770&resize=770%2C433" data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul>
<li>
<h3>哈马斯和抵抗运动从“耶路撒冷之剑”中获得了什么？</h3>
</li>
</ul>
<p>哈马斯及其派系实现了（耶路撒冷之剑）战斗目标，因为这是其首先对以色列打响了战争，而此前的军事战役中，以色列是引发军事战役第一方。</p>
<p>因此，哈马斯已将其定义为阿克萨清真寺和耶路撒冷的捍卫者，并向以色列领土深处发射了火箭榴弹，造成巨大破坏，并造成数亿美元的巨大损失，因为火箭弹袭击造成12人死亡，数百人受伤，此外，这给以色列人的生活造成破坏和影响，火箭弹降落至特拉维夫，以色列没有成功拦截火箭弹或击败哈马斯的能力，而哈马斯正致力于巩固其在巴勒斯坦和阿拉伯人中的成就意识。</p>
<p>哈马斯——以及抵抗力量——能够为所有巴勒斯坦人提供一个受欢迎的孵化器，而不仅仅是在被围困长达15年之久的加沙地带。在以色列城镇遭到火箭弹袭击的同时，巴勒斯坦内部以及被占领约旦河西岸和耶路撒冷对加沙地带表示支持，这延长了以色列和哈马斯之间的军事对抗，哈马斯似乎在领导巴勒斯坦阵营，从而也暴露了巴勒斯坦权力机构的软弱。</p>
<ul>
<li>
<h3>以色列为什么不回应联合国的停火要求？</h3>
</li>
</ul>
<p>根据后备役军官奥迪·德克尔——他曾在以色列前总理埃胡德·奥尔默特政府中担任与巴勒斯坦谈判局局长职位——说服称，以色列政府应该制定和宣布的战略目标是，掌控和控制与哈马斯交战的战区，防止紧张局势升级并蔓延到约旦河西岸其他地区和内陆其他地区，着重于削弱哈马斯力量，停止其对巴勒斯坦地区的控制，削弱哈马斯的军事能力，将其倒退回至少十年前水平，并致力于在巴勒斯坦竞技场制定新规格，重点在阿巴斯之后，将巴勒斯坦权力机构恢复至巴勒斯坦专属代表权手中。</p>
<p>由于所有这些都没有实现，而且以色列的军事解决方案牵强附会，内塔尼亚胡拒绝停止对加沙的侵略，特别是因为他除了摧毁基础设施和住房项目外，甚至没有取得任何成就。因此，尽管受到欧洲施压以及来到美国政府及其总统乔·拜登的要求，内塔尼亚胡仍将推迟停火，旨在通过暗杀一些哈马斯或伊斯兰圣战组织领导人来赢得时间，并构筑胜利舞台，以色列情报部长埃里·科恩承认，加沙地带哈马斯领导人以及卡桑旅指挥官穆罕默德·迪夫是袭击目标，以色列军方承认，在“城墙护卫”爆发第一周内，两次暗杀迪夫的行动失败。</p>
<p><img class="aligncenter size-arc-image-770 wp-image-194424" src="https://chinese.aljazeera.net/wp-content/uploads/2021/05/image-48.jpg?w=770&resize=770%2C433" data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul>
<li>
<h3>内塔尼亚胡推迟停火和坚持轰炸加沙的目标是什么？</h3>
</li>
</ul>
<p>根据内塔尼亚胡愿望以及以色列安全和军事层面官员指示，对加沙的侵略将在几天之内结束，这只是单方面停火，没有达成任何协议，也没有关于长期和解的任何谅解，而埃及将确定并宣布停火及其生效的时间。</p>
<p>有关确定结束战役和单方面停火机制事宜，以色列试图单方面确定战役结束的日期和条件，试图以此推销其拥有决定话语权的说辞，并防止哈马斯控制停火进程。从前几轮战斗中吸取的教训是，以色列力求避免在停火之前起草具有调解原则的文件。</p>
<p>尽管诸如以色列国防军参谋长阿维夫·科查维（Aviv Kochavi）代表的军事层面官员，以及以内塔尼亚胡为代表的政治层面官员都试图结束军事行动，但很明显，结束在加沙地带的战争令内塔尼亚胡感到不舒服，因为当前没有胜利的画面，足以说服以色列的舆论。</p>
<p>值得注意的是，被迫回应巴勒斯坦抵抗运动的内塔尼亚胡，是出于个人政治动机而发动的这场战争，以逃避对腐败指控的起诉，并试图阻止“拥有未来”党领袖拉皮德的组阁进程，拉皮德被授权组建新政府，任期将在两周内结束，这意味着内塔尼亚胡将担任总理职位，直到举行第五次选举为止。</p>
<ul>
<li>
<h3>尽管如此，以色列即将实现停火吗？</h3>
</li>
</ul>
<p>以色列第13频道援引以色列军事问题专家阿隆·本·戴维说法称，在埃及调解下，以色列正准备大幅度缓解在加沙地带的军事活动，以试图在本周末结束军事行动，但条件是在接下来24小时内逐步促进停火，5月20日将是一个考验。</p>
<p>值得注意的是，以色列安全和政治事务部长级内阁推迟了原定于当地时间5月19日晚召开的会议，而本次会议上原本应该就停止军事行动达成共识，这表明以色列尚未做出任何决定。尽管如此，以色列估计，停火将在当地时间5月21日中午生效。</p>
<p>在停火生效前48小时之前，以色列国防部长本尼·甘茨指示军队19日晚加强对加沙地带目标的突袭和袭击，以实现最大目标，特别是摧毁以色列军方所谓的“加沙地带的哈马斯地铁”，指的是哈马斯在地下建立的隧道网络和城市。</p>
<ul>
<li>
<h3>军队是否用尽了军事行动并朝着停火方向迈进？</h3>
</li>
</ul>
<p>随着军事行动可能即将结束，以色列军队承认哈马斯正在努力增加成就记录，并致力于巩固巴勒斯坦意识，因此，其应该力图挫败这些成就，并且军队将大部分精力集中在瞄准火箭弹和迫击炮发射台上。</p>
<p>根据以色列估计，停火协议将在接下来24小时内生效。与此同时，安全机构正在寻求用尽其所谓的“军事成就”，高级军事官员呼吁加强进攻，直到停火为止，这是要向哈马斯传达一个信息，即尽管受到国际压力，以色列仍会毫不犹豫地采取有力行动。</p>
<p>因此，预计以色列军队将继续轰炸哈马斯位于加沙地带的基础设施，以色列政治和安全机构担心局势将进一步升级，从而可能会阻止实现停火的努力，无论是由于以色列袭击加沙地带造成的平民伤亡，还是由于哈马斯主动向特拉维夫及以色列中心地带频繁发射火箭弹袭击的举动，此举使以色列感到尴尬，并迫使其做出回应。</p>
<p>军事分析家阿米尔·布赫布特（Amir Bukhbout）在“online”网站上发表声明称，以色列军队提出了一种务实方法，根据这种方法，在发动军事行动第二天后，重启平静局势，以防止加沙地带爆发人道主义灾难。布赫布特估计，很可能会在谈判中提出加沙地带的恢复与重建问题，安全机构官员认为，要求将组织、恢复和重建加沙地带的责任移交给巴勒斯坦权力机构是正确之举，旨在加强其地位，以与哈马斯抗衡。</p>
