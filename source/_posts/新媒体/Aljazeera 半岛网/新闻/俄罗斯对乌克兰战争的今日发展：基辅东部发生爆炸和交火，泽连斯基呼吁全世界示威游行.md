<div>   
<p>俄罗斯对乌克兰的战争已经进入第 29 天，鉴于各条战线的战斗仍在继续，外交努力持续不断，美国倾向于在周四对俄罗斯立法者实施更多制裁，与此同时，预计美国总统乔·拜登在访问布鲁塞尔期间将发起一项向欧洲供应天然气的倡议，旨在帮助其减少对俄罗斯能源的依赖。</p>
<p>《纽约时报》透露，在战争开始 4 天后，美国总统乔·拜登政府组建了一个名为“老虎队”的国家安全官员工作组，每周秘密会面 3 次，其任务之一是研究如何应对俄罗斯在乌克兰使用化学、生物或核武器，以及如何应对战争扩大到邻国以及在北约国家领土内运送武器的车队遭到袭击的事态发展。</p>
<h3>以下是这场战争第29天的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>敖德萨市当局：</h3>
<ul>
<li>乌克兰海军对别尔江斯克港的袭击造成 3 名俄罗斯士兵死亡和11 人受伤。</li>
<li>俄罗斯大型登陆舰“萨拉托夫”号因袭击而起火。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰海军：在扎波罗热市亚速海的别尔江斯克港附近摧毁了一艘大型俄罗斯登陆舰。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰国防部长奥列克西·列兹尼科夫表示，自战争开始以来，他的部队已经击落了 100 多架战斗机，摧毁了 520 多辆坦克和 1500 辆装甲车，并强调，我们仍然面临着非常困难的时期，我们将把俄罗斯军队从我们的土地上驱逐出去，列兹尼科夫补充说，国际支持正在逐渐增加，敌人的损失是4万多名士兵死亡、受伤、被俘和失踪。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>俄罗斯国防部：我们获得了新信息，证明五角大楼参与了乌克兰生物武器部件的研发。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰副总理：我们同意从马里乌波尔和基辅各个郊区开放7条安全走廊。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>欧盟的立场</h3>
<p>欧盟外交政策负责人何塞普·博雷利表示，俄罗斯军方尚未实现其在乌克兰的目标。</p>
<p>博雷利补充说，俄罗斯试图将乌克兰孤立起来，在这种情况发生之前对谈判不感兴趣。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>俄罗斯控制</h3>
<p>俄罗斯国防部表示，其部队完全控制了哈尔科夫省的伊久姆地区。</p>
<p>俄罗斯国防部补充说，昨晚它用远程导弹轰炸了乌克兰的军事设施，并指出，它摧毁了首都基辅以南的13个防空导弹系统。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总检察长办公室：自俄罗斯对乌克兰发动战争以来，已有128名儿童死亡，超过172人受伤。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>英国外交大臣：七国集团对俄罗斯的制裁行动完全一致，我们将继续努力结束这场战争。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>顿涅茨克分离主义领导人：马里乌波尔的乌克兰军队估计有数千人，弹药即将耗尽。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰海军：在被占领的别尔江斯克港摧毁黑海的俄罗斯海军舰艇“奥尔斯克”号。</p>

<ul style="list-style-type:square">

<h3>北约优先事项</h3>
<p>北约秘书长延斯·斯托尔滕贝格表示，北约加强了对乌克兰的财政和军事支持，并指出，他们为基辅提供了先进的防空系统和反装甲导弹。</p>
<p>斯托尔滕贝格补充说，北约有责任确保这场冲突不会扩大范围，并指出，北约不会将其部队派往乌克兰。</p>
<p>根据北约秘书长说法称，北约飞机不会进入乌克兰领空。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>俄罗斯的批评</h3>
<p>俄罗斯驻华盛顿大使阿纳托利·安东诺夫表示，美国国务院有关乌克兰涉嫌战争罪的声明超出了常识范围。</p>
<p>安东诺夫还表示，美国试图赋予自己确定谁是“战犯”的权利令人反感，并表示，一个摧毁了许多国家并夺走了数十万人生命的国家，无法发挥治理的作用。</p>
<p>俄罗斯驻华盛顿大使补充说，美国正试图通过其指控妖魔化莫斯科。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>塔斯社援引俄罗斯航空局说法报道称：将飞往俄罗斯南部和中部的航班禁令延长至4 月 1 日。</p>

<h3>谷歌的困难：</h3>
<p>路透社援引谷歌说法报道称，俄罗斯访问其新闻应用程序和网站存在困难，访问新闻应用程序和网站的困难与其本身的任何技术问题无关。</p>
<p>早些时候，俄罗斯国际文传电讯社报道称，俄罗斯通信监管机构 (Roskomnadzor) 阻止了Alphabet 旗下谷歌提供的新闻服务，指责它允许访问所谓的关于俄罗斯在乌克兰军事行动的虚假消息。</p>
<p>一项新的俄罗斯法律规定，以一种所谓的有损俄罗斯军队名誉的方式报道事件是犯罪行为。</p>

<li><strong>更新</strong></li>
