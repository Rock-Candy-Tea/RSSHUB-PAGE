<p>以下是所有最新更新：</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>埃尔多安助手：土耳其提议为受伤的马里乌波尔战士提供海上疏散</strong></p>
<p>土耳其总统雷杰普·塔伊普·埃尔多安的发言人说，安卡拉已提议对躲在乌克兰南部城市马里乌波尔一家钢铁厂的受伤战士进行海上疏散。</p>
<p>易卜拉欣·卡林告诉路透社，两周前他曾在基辅与乌克兰的泽连斯基亲自讨论过该提议，尽管莫斯科尚未同意，但该提议仍“摆在桌面上”。</p>
<p>他说，根据该计划，从庞大的亚速钢铁厂撤离的人员将通过陆路被带到别尔江斯克港，该港口和马里乌波尔一样位于亚速海，一艘土耳其船只将带他们穿越黑海到达伊斯坦布尔。</p>
<p>“如果可以，我们很乐意这样做。我们准备好了。事实上，我们的船已经准备好将受伤的士兵和其他平民带到土耳其，”同时也是埃尔多安最高外交政策顾问的卡林说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基庆祝乌克兰赢得欧洲歌唱大赛冠军，承诺有一天将在马里乌波尔举办比赛</strong></p>
<p>乌克兰总统称赞卡卢什乐团在欧洲歌唱大赛中获胜。</p>
<p>“我们的勇气打动了世界，我们的音乐征服了欧洲！”泽连斯基在Telegram帖子中说。</p>
<p>作为今年欧洲歌唱大赛的获胜者，乌克兰将主办明年的比赛。泽连斯基说，他希望有一天能在“自由、和平和重建”的马里乌波尔接待欧洲歌唱大赛的参与者和嘉宾。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>布林肯抵达德国参加北约会议</strong></p>
<p>美国国务卿安东尼·布林肯在柏林举行的北约外长非正式会议开幕前抵达德国。</p>
<p>会议将考虑芬兰和瑞典加入军事联盟北约的举措，以及北约在不卷入与俄罗斯冲突的情况下支持乌克兰的方式。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>米奇·麦康奈尔向乌克兰保证“持续支持”</strong></p>
<p>美国参议院共和党领袖在突然访问基辅期间重申了华盛顿对乌克兰的支持。</p>
<p>米奇·麦康奈尔在一份声明中说，他向泽连斯基保证，美国“坚定地支持乌克兰，并将维持我们的支持，直到乌克兰赢得这场战争”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰赢得欧洲歌唱大赛</strong></p>
<p>乌克兰赢得了欧洲歌唱大赛，在整个欧洲为陷入困境的国家赢得了公众支持的浪潮，并使它受到具有感染力的民间嘻哈旋律的鼓舞。</p>
<p>卡卢什乐团的歌曲“Stefania”在周六世界上最大的现场音乐盛会的决赛中击败了24名参赛者。获奖歌曲以乌克兰语演唱，将说唱与传统民间音乐融为一体，向乐队主唱Oleh Psiuk的母亲致敬。</p>
<p>乌克兰总统泽连斯基在当晚早些时候对乐队发去祝福，称如果卡卢什乐团获胜，这一胜利将具有巨大的象征意义。</p>
<p>“对今天的我们来说，任何胜利都非常重要，”他在晚间讲话中说。</p>
<p>由于入侵乌克兰，俄罗斯被排除在都灵的比赛之外。</p>

<p><strong>马里乌波尔车队抵达安全，难民讲述“毁灭性”逃亡</strong></p>
<p>在等待俄罗斯军队允许他们离开数天后，一支载有从马里乌波尔废墟中逃离的难民的大型汽车和货车车队抵达乌克兰控制的扎波罗热市。</p>
<p>周六抵达的难民首先必须离开马里乌波尔，然后以某种方式前往别尔江斯克（沿海岸向西约80公里）和其他定居点，然后再向西北行驶200公里到达阿波罗热。</p>
<p>74岁的尼古拉·巴甫洛夫说，他的公寓被毁后，他在地下室住了一个月。一位亲戚使用“秘密绕道”的方式，设法将他从马里乌波尔带到别尔江斯克。</p>
<p>“我们勉强做到了，我们中间有很多老人。这次旅行是毁灭性的，但是值得的，”他在车队在黑暗中抵达后说道。</p>
<p>马里乌波尔市长的一名助手早些时候表示，车队里有500至1000辆汽车，是自俄罗斯入侵乌克兰以来该市最大的一次撤离。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>特使说俄罗斯在美国的外交官受到联邦调查局和中央情报局的威胁和诱惑</strong></p>
<p>据塔斯社报道，俄罗斯驻美国特使表示，俄罗斯驻华盛顿外交官正受到暴力威胁，美国情报部门正试图与他们取得联系。</p>
<p>“这就像一个被围困的堡垒。基本上，我们的大使馆是在一个充满敌意的环境中运作的。大使馆的员工正在受到威胁，包括人身暴力攻击的威胁，”塔斯社援引阿纳托利·安东诺夫大使周六的话说。</p>
<p>“来自美国安全部门的特工在俄罗斯大使馆外闲逛，分发中央情报局和联邦调查局的电话号码，以便打电话建立联系，”大使告诉塔斯社。</p>
<p>自俄罗斯入侵乌克兰之前，俄罗斯和美国就各自外交使团的规模和运作情况一直存在争议。</p>

<p><strong>美国的米奇·麦康奈尔在基辅会见泽连斯基</strong></p>
<p>美国参议院共和党领袖米奇·麦康奈尔与其他共和党参议员对基辅进行了未公开的访问，并与乌克兰总统进行会谈。</p>
<p>麦康奈尔由参议员苏珊·柯林斯、约翰·巴拉索和约翰·科宁陪同。</p>
<p>泽连斯基称赞这次访问是美国两党支持乌克兰和加强两国关系的有力信号。</p>
<p>泽连斯基在视频讲话中说，“我们讨论了支持我们国家的许多领域，包括国防和金融，以及加强对俄罗斯的制裁，”并补充说，他向参议员们强调，有必要将俄罗斯指定为恐怖主义国家。</p>
<p>麦康奈尔正在敦促共和党参议员兰德·保罗结束他对乌克兰400亿美元一揽子援助计划的反对，该计划得到美国两大政党的压倒性支持。</p>

<p><strong>乌克兰乐队在欧洲电视网为马里乌波尔求助</strong></p>
<p>乌克兰卡卢什管弦乐团在欧洲歌唱大赛上的表演结束时，为在马里乌波尔市亚速钢铁厂坚持的战士发出援助请求。</p>
<p>“请帮助乌克兰，帮助马里乌波尔。立即帮助亚速钢铁厂，”乐队演唱了歌曲“Stefania”后，主唱Oleh Psiuk在意大利城市都灵的舞台上喊道。</p>
<p>俄罗斯军队轰炸了南部港口马里乌波尔的钢铁厂，这是数百名乌克兰守军在这座城市的最后堡垒。经过两个多月的围攻，这座城市几乎完全被俄罗斯控制。</p>

<p><strong>普京告诉芬兰总统：加入北约将是“一个错误”</strong></p>
<p>俄罗斯总统弗拉基米尔·普京告诉芬兰总统，加入北约将是“一个错误”，莫斯科早些时候切断了对改北欧国家的电力供应。</p>
<p>克里姆林宫周六在一份声明中说：“普京强调，结束传统的军事中立政策将是一个错误，因为芬兰的安全没有受到威胁。”</p>
<p>“该国政治取向的这种变化可能会对多年来本着睦邻友好和伙伴合作精神发展起来的俄芬关系产生负面影响，”它说。</p>
