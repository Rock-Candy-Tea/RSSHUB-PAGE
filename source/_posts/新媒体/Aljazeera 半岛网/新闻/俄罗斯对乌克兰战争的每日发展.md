<p>乌克兰陆军总参谋部22日表示，该国东部多地的战争节奏加快，并称俄罗斯部队加强了顿涅茨克地区前线的作战行动。</p>
<p>就在一天之前，俄罗斯总统普京宣布俄军已经控制了马里乌波尔，并称之为俄军取得的一项成就。</p>
<p>普京21日在与俄罗斯国防部长绍伊古举行的会议期间下令取消进攻工业区的行动，并要求继续对马里乌波尔的这个钢铁厂实施包围，而不对其发动攻击。</p>
<p>马里乌波尔市长瓦迪姆·博伊琴科表示，从该市撤离平民面临着困难，目前约有200人正在等待撤离，但却没有足够的大巴车抵达以运送这些人员。</p>
<p>博伊琴科补充称，在20日大约有80人乘坐4辆大巴车离开了这座城市，其中3辆目前已经抵达了扎波罗热。</p>
<p><strong>以下是这场战争进入第58天以来的最新进展：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>马克龙：没有直接的解决方案可以让我们立即取代俄罗斯的天然气，我们正在寻找替代途径。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>法新社：英国预计乌克兰战争将持续至2023年底。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>英国首相：我们正在考虑向波兰派遣坦克。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰国防部长：我们需要重型武器来解放被俄罗斯占领的土地。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>法新社：联合国方面表示，已经记录了发生在乌克兰布查镇的50位平民被杀的事件。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯国防部：</strong></p>
<ul>
<li>我们使用“卡利布”（Kalibr）导弹摧毁了梅利乌拉提维火车站地区的乌克兰武器和设备。</li>
<li>我们的部队摧毁了当地的S-300防空系统、数十部车辆、燃料库以及指挥中心。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>英国情报机构：俄罗斯人正在遭受先前的损失，并且正试图重组他们已经力竭的部队。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>彭博社援引美国商务部长的话报道：受制裁影响，俄罗斯进口的零部件和技术减少了50%。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总参谋部：在过去的几个小时内，我们的部队击退了俄罗斯在顿涅茨克和卢甘斯克州发动的10次袭击。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>五角大楼（美国国防部）发言人约翰·柯比向CNN记者表示：</p>
<ul>
<li>俄罗斯人认为，控制马里乌波尔对向顿巴斯地区进军非常重要。</li>
<li>从我们的角度来看，俄罗斯人在顿巴斯地区并没有取得任何领土收益。</li>
<li>俄罗斯人近来的策略是非常残酷的，他们犯下了暴行及战争罪行。</li>
<li>我们仍然相信马里乌波尔的乌克兰人正在抵制俄罗斯控制这座城市的企图。</li>
<li>显然，俄罗斯人为控制马里乌波尔投入了大量的精力。</li>
<li>我们额外提供给基辅的系统适用于乌克兰人即将在顿巴斯地区进行的战斗。</li>
</ul>
