<li>51岁的希琳在报道以色列军队对杰宁难民营的突袭时，尽管身穿印有“新闻”字样的背心，但被一颗子弹击中脸部。</li>

<li>半岛电视台在一份声明中说，阿布·阿格莱被“冷血暗杀”，并呼吁国际社会追究以色列军队的责任。</li>
</ul>

<p>以下是最新更新：</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>大赦：以色列必须结束非法杀戮</strong></p>
<p>人权组织大赦国际在一份声明中呼吁结束以色列对巴勒斯坦人的“非法杀戮”和其他种族隔离罪行。</p>
<p>大赦国际负责中东和北非事务的副主任萨利赫·希加齐说：“资深记者希琳·阿布·阿格莱被杀事件血腥地提醒人们，以色列锁定巴勒斯坦人的致命系统。”</p>
<p>声明继续说，以色列军队在对抗议活动进行监管或搜查、逮捕突袭时过度使用武力，导致巴勒斯坦人死亡或受伤，在这种情况下构成法外处决——这是国际法下的罪行。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>另一位记者和目击者讲述了枪击时刻</strong></p>
<p>穆贾希德·萨迪是一名巴勒斯坦记者，他是进入杰宁难民营的记者团中的一员，他说他当时正在等待同事，之后枪击便发生了。</p>
<p>“我们正在等待我们的同事进入难民营，”他说。“我们选择了一个以色列军队和武装战士之间没有对抗的地点。我们等着希琳穿上她所有的安全装备。然后她加入我们，我们移动了几米。</p>
<p>“几秒钟内就响了第一枪。我告诉他们我们正在成为目标，我们被枪对准了。我转身发现希琳倒在地上。我发现沙塔（哈尼什）躲在树后尖叫。枪击持续了三分钟多。（半岛电视台记者）阿里（萨穆迪）受伤了，他穿过马路得以到达安全地点。沙塔站在树后时，朝着她的射击仍在继续。</p>
<p>“我们无法为希琳提供急救。街上的年轻人来找我们，想把希琳拉出来，但也被枪瞄准了。每当有人向前移动时，他们就会被抢威胁。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>以色列警方突袭阿布·阿克莱在耶路撒冷的家</strong></p>
<p>以色列军队袭击了阿布·阿克莱在被占领的东耶路撒冷的家，没收了巴勒斯坦国旗，并阻止播放民族主义歌曲。</p>
<p>半岛电视台看到的视频显示，朋友和家人高呼让以色列警察离开房子。</p>
<p>现场的一名记者说，送葬者设法将部队赶出房屋，但仍留在那里。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>半岛电视台耶路撒冷分社社长：阿布·阿格莱是“直接射击”的目标</strong></p>
<p>半岛电视台巴勒斯坦分社社长瓦利德·奥马里表示，打死阿布·阿克勒的子弹很明显是被一名以色列狙击手发射的。</p>
<p>“目击者告诉我们，子弹是从以色列占领军所在的地方射出的，”他说。</p>
<p>“这是一个巨大的冲击，因为记者们身处一片开阔的区域，远离（以色列）与巴勒斯坦抵抗力量的军事对抗，”他说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>巴勒斯坦权力机构拒绝以色列参与调查阿布·阿格莱的死因</strong></p>
<p>巴勒斯坦权力机构发言人易卜拉欣·米利姆表示，他的政府拒绝以色列在调查阿布·阿格莱被杀事件中发挥任何作用。</p>
<p>“让我问一下，罪犯什么时候有权参与对受害者的调查？”米利姆告诉半岛电视台。</p>
<p>“我们拒绝并拒绝任何以色列人参与此类调查。他们必须被带到国际法庭。我们呼吁国际法院对希琳被杀害和其他巴勒斯坦人被针对的罪行展开调查。”</p>
<p>米利姆还敦促国际社会停止对巴勒斯坦事业和以色列罪行的双重标准。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>世界对以色列军队杀害半岛电视台记者的反应</strong></p>
<p>阿布·阿格莱遇害引发了强烈的哀悼和谴责，以及对进行彻底调查以追究相关人员的责任呼声。</p>

<p><strong>希琳·阿布·阿格莱：为巴勒斯坦人发声的开拓者</strong></p>
<p>周三被以色列军队杀害的巴勒斯坦半岛电视台记者希琳·阿布·阿克莱是一名资深的电视记者，她因大胆报道巴以冲突而在整个阿拉伯世界家喻户晓。</p>
<p>朋友和同事将阿布·阿格莱描述为一位勇敢而善良的记者，带着“感染力的笑声”，他为巴勒斯坦人在近30年的抵抗事业中的挣扎发声。</p>
<figure id="attachment_279697" aria-describedby="caption-attachment-279697" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-279697" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/12-7.png?w=770&resize=770%2C514" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-279697" class="wp-caption-text">希琳·阿布·阿克莱 (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>国际记者联盟将阿布·阿克莱的案件提交国际刑事法院</strong></p>
<p>国际记者联盟秘书安东尼·贝兰格表示，阿布·阿格莱被杀是“蓄意、系统地针对一名记者”。</p>
<p>他在一份声明中说：“又一次，穿着新闻背心的记者成为以色列狙击手的目标。我们将寻求将此案添加到国际记者联盟提交到国际刑事法院的投诉中，详细说明此类系统性的攻击。”</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>巴勒斯坦记者协会抨击“蓄意暗杀”阿布·阿格莱</strong></p>
<p>巴勒斯坦记者协会表示，阿布·阿克莱的死是“以色列占领军的蓄意暗杀”。</p>
<p>该协会还认为以色列占领军“对这一危害新闻自由的令人发指的罪行负有全部责任”，特别是在5月8日世界新闻自由日刚刚过后。</p>
<p>巴勒斯坦记者协会强调，占领军针对阿布·阿格莱的罪行是“蓄意和有预谋的行为，是真正的暗杀行动”。</p>
<p>该协会呼吁采取明确的行动，保护记者同行免受“以色列占领者及其所有组成部分的持续煽动和杀戮”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟敦促对阿布·阿格莱遇害进行“迅速和独立”的调查</strong></p>
<p>欧盟驻巴勒斯坦代表团对阿布·阿格莱被杀表示“震惊”，呼吁“迅速进行独立调查，将肇事者绳之以法”。</p>
<p>“当务之急是确保报道冲突的记者的安全和获得的保护，”声明补充说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>目击者：以色列军队开枪杀人</strong></p>
<p>目击现场的同行记者沙塔·哈尼什说，当阿布·阿格莱被枪杀时，现场的记者被狙击手压制着。</p>
<p>她告诉半岛电视台：“我们作为记者集合，然后开始活动。 我们被迎面发射的实弹所震惊，而处于一个无法撤退的地方。”</p>
<p>“我们正面狙击手。如果他们不是真的愿意杀死我们中的一些人，他们可能在我们到达这个狭窄区域之前就开始射击了。我认为这是对记者的明显暗杀，”她说。</p>
<p>“杀死希琳的人是故意杀死她的，因为他向她身体没有受到保护的区域开枪，” 哈尼什说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>哈马斯：杀害阿布·阿格莱是以色列的最新罪行</strong></p>
<p>加沙地带的统治者哈马斯表示，以色列占领军杀害阿布·阿格莱是他们犯下的最新罪行。</p>
<p>杀戮是以色列犯下的“一系列罪行”，特别是针对巴勒斯坦的新闻业的罪行中“最新的一项”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>巴勒斯坦议员：阿布·阿格莱曾是我们的声音</strong></p>
<p>巴勒斯坦议员哈利达·贾拉尔表示，阿布·阿格莱曾代表着巴勒斯坦人的声音，但被“以色列殖民主义和占领的怪物”杀害。</p>
<p>她说，“当我在牢房里时，希琳一直替我传达声音。”她补充说，在她上次被以色列拘留一个月后，希琳是她在法庭听证会上见到的第一个人。</p>
<p>“希琳是我们的声音。她的死令人难以置信。这是一种犯罪行为，这一切都很清楚，故意和直接针对。她被盯上了。这很清楚。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>以色列总理称“可能”是巴勒斯坦人开枪打死了阿布·阿格莱</strong></p>
<p>以色列总理纳夫塔利·贝内特表示，“很可能”是巴勒斯坦人开枪打死了半岛电视台的记者。</p>
<p>“根据我们收集的信息，武装的巴勒斯坦人当时不分青红皂白地开枪，他们似乎对这名记者的不幸死亡负有责任，”贝内特在一份声明中说。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>联合国特使敦促追究杀害阿布·阿格莱的责任</strong></p>
<p>联合国中东和平进程特别协调员托尔·温内斯兰谴责在被占领的约旦河西岸杀害阿布·阿格莱。</p>
<p>“我强烈谴责半岛电视台记者希琳·阿布·阿格莱被杀的事件，今天早上，她在被占领的约旦河西岸报道以色列安全部队在杰宁的行动时被实弹击中，”温内斯兰在推特上写道。</p>
<p>“我呼吁立即进行彻底调查，并追究相关人员的责任。媒体工作者永远不应该成为目标。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>美国驻以色列大使呼吁调查</strong></p>
<p>美国驻以色列特使汤姆·奈德斯呼吁对持有美国公民身份的阿布·阿格莱遇害事件进行调查。</p>
<p>“很难过得知巴勒斯坦裔美国记者希琳·阿布·阿格莱的死讯，”他在推特上写道。 “我呼吁对她今天在杰宁的死亡和至少另一名记者受伤的情况进行彻底调查。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>人权观察：阿布·阿格莱的死亡不是“偶然事件”</strong></p>
<p>人权观察以色列和巴勒斯坦地区主任奥马尔·沙基尔说，阿布·阿格莱被以色列军队枪击，这种情况并不罕见。</p>
<p>“我们知道以色列军队有系统地过度使用武力，”他告诉半岛电视台。</p>
<p>“这是一个需要在系统性做法和许多巴勒斯坦记者被杀的背景下理解的事件。”</p>
<p>沙基尔将以色列的调查描述为“洗白机制”。</p>
<p>“这是包括以色列首屈一指的B’Tselem在内的人权组织做出的评估，”他说。“当犯罪新闻被报道时，以色列军队经常说他们会进行调查。现实情况是，当涉及到以色列当局的行动时，这些滥用职权的行为不会受到追究。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>记者称巴勒斯坦战士不在现场</strong></p>
<p>半岛电视台记者阿里·萨穆迪当时在阿布·阿克来旁边，也被以色列军队击中，但目前情况稳定，他说现场没有巴勒斯坦武装战士。</p>
<p>“我们正要拍摄以色列军队的突袭行动，他们在没有要求我们离开或停止拍摄的情况下突然向我们开枪，”他说。</p>
<p>“第一颗子弹击中了我，第二颗子弹击中了希琳。他们冷血地杀死了她，因为他们是杀手，他们专门杀害巴勒斯坦人。”</p>
<p>他补充说，“现场根本没有巴勒斯坦的军事人员。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>以色列军队否认针对记者，提议进行联合调查</strong></p>
<p>以色列军队证实，周三早些时候在杰宁难民营进行了一次行动，但否认故意针对记者。</p>
<p>“（军队）当然不会针对记者，”一名军方官员说。</p>
<p>军方表示，嫌疑人与安全部队发生了交火，目前正在“调查此事，以及调查记者被巴勒斯坦枪手击中的可能性”。</p>
<p>以色列外交部长亚尔·拉皮德表示，以色列已提议与巴勒斯坦人进行联合调查。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>希琳·阿布·阿格莱的“专业且坚持不懈”</strong></p>
<p>半岛电视台记者向阿布·阿格莱的家人表示慰问，并对她的逝世表示哀悼。</p>
<p>“希琳，对于每一位巴勒斯坦记者和每一位阿拉伯记者来说，都是一个榜样，然而我们今天失去了这个榜样，”半岛电视台记者塔玛尔·米沙尔说。“我们在社交媒体上看到的头部中弹的悲惨照片表明，一名以色列狙击手直接向她开了一枪。”</p>
<p>“直到最后一秒，希琳·阿布·阿格莱都很专业且坚持不懈，”他补充道。</p>
<p>阿布·阿格莱发送给半岛电视台的最后一条信息是早上6点13分的一封电子邮件，她在邮件中写道：“占领军袭击了杰宁并围攻了贾布里亚特社区的一所房子。正在去的路上，等情况明朗，我会第一时间给您带来消息。”</p>
