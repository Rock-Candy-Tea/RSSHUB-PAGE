<p>乌克兰首都基辅北部的社区在早晨受到俄罗斯的轰炸，而美国国务院则警告俄罗斯不要使用化学武器。</p>
<p>波兰、捷克共和国和斯洛文尼亚总理在基辅与乌克兰总统弗拉基米尔·泽连斯基会晤时承诺继续支持乌克兰，美国总统乔·拜登宣布向乌克兰提供价值136亿美元的人道主义和军事援助。</p>
<p>乌克兰总统顾问米哈伊洛·波多利亚克周二表示，乌克兰与俄罗斯的谈判仍存在“深刻矛盾”，谈判将于周三继续，并强调“解决”分歧仍有可能。</p>
<p>周二，欧盟宣布了针对俄罗斯对乌克兰发动战争的第四套制裁措施。</p>
<p>联合国难民事务高级专员宣布，逃离祖国的乌克兰人超过300万人。</p>
<p><strong>以下是这场战争21日的最新进展：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>在俄罗斯对乌克兰开展军事行动的背景下，俄罗斯驻华盛顿大使馆为在美国受到歧视的俄罗斯人开通了一条热线。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>美国参议院一致通过一项谴责俄罗斯总统弗拉基米尔·普京为战争犯的决议。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>白宫官员：预计拜登周三将宣布向乌克兰追加8亿美元的安全援助。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>美国国务卿：我们担心中国可能向俄罗斯提供物质支持或破坏对莫斯科实施的制裁。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总统：</p>
<ul style="list-style-type:disc">
<li>昨天我们设法从哈尔科夫、基辅和苏梅撤离了大约29000名公民，但情况非常困难。</li>
<li>应该对俄罗斯实施更多制裁。</li>
<li>我呼吁对基辅进行更多正式访问，空域仍然开放。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>《华尔街日报》援引军方官员：预计拜登此时不会下令在欧洲增兵。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>美国人的担忧</h3>
<p>美国国务卿安东尼·布林肯表示，他确实担心俄罗斯会在乌克兰使用化学武器，并指出华盛顿对莫斯科表示得非常清楚，并告诉它，如果使用化学武器，将会有回应和后果。</p>
<p>他在向美国有线电视新闻网（CNN）发表的声明中补充说，俄罗斯谈论其核武器反映了不负责任的态度，并强调拜登总统明确了避免美国与俄罗斯发生冲突，从而使美国陷入第三次世界大战的立场。</p>
<p>布林肯认为，普京的目的是剥夺乌克兰作为独立国家存在的权利，普京对乌克兰加入北约的担忧是没有根据的。</p>
<p>他补充说，“即使在俄罗斯入侵之前，乌克兰也没有即将加入北约。”</p>

<h3>欧洲制裁</h3>
<p>欧盟针对俄罗斯富人的第四套制裁措施，包括禁止从欧盟进口奢侈品，于周二在欧盟官方公报上公布后生效。</p>
<p>欧盟外交政策负责人何塞普·博雷尔在宣布此举的声明中表示，那些促成入侵乌克兰的人正在为他们的行为付出代价。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<li>我向乌克兰外长重申，美国将继续支持乌克兰人民，并承诺让普京为他对乌克兰无端和不人道的轰炸负责。</li>
<li>自2月24日以来，我们已提供2.93亿美元帮助乌克兰人民。</li>
</ul>
