<li>乌克兰总统泽连斯基表示，赢得对俄罗斯的战争，对全球稳定“至关重要”，他还呼吁获得更多的武器。</li>
<li>被炸毁的港口城市马里乌波尔市长表示，霍乱爆发可能会导致数千人死亡，并呼吁剩余的居民撤离该市。</li>
<li>爱沙尼亚召见俄罗斯大使，谴责俄罗斯总统普京对彼得大帝“完全不可接受”的赞扬。</li>
<li>乌克兰一名地区州长表示，乌克兰与俄罗斯进行了第11次囚犯交换。</li>
<li>在乌克兰东部的激烈战争下，300名乌克兰人乘坐专门的撤离列车离开波克罗夫斯克，并前往第聂伯罗和其他位于该国西边的城市。</li>
</ul>

<h3>以下是这场战争进入今日以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国国防部：俄罗斯在战争中使用更致命的武器</strong></p>
<p>乌克兰和英国官员警告称，俄罗斯军队在试图占领乌克兰东部时，依靠的是能够造成大规模伤亡的武器，而激烈、旷日持久的战斗耗尽了双方的资源。</p>
<p>英国国防部表示，俄罗斯轰炸机可能在乌克兰发射了上世纪60年代的重型反舰导弹。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>世卫组织：马里乌波尔处于霍乱的高风险之下</strong></p>
<p>世界卫生组织表示，被摧毁的乌克兰城市马里乌波尔爆发霍乱的风险系数“极高”，但到目前为止还没有确诊病例。</p>
<p>一位发言人在日内瓦表示，“世卫组织尚未收到任何疑似或确诊病例的警报或通知。”</p>
<p>已经不在俄罗斯占领的港口城市马里乌波尔的市长瓦迪姆·博伊琴科，以及英国国防部警告称，马里乌波尔面临着迫在眉睫的霍乱爆发。</p>
<figure id="attachment_289526" aria-describedby="caption-attachment-289526" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-289526" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/2022-06-07T083833Z_913583198_RC2ACU9HJAFH_RTRMADP_3_UKRAINE-CRISIS-AZOV-1.jpg?w=770&resize=770%2C512" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-289526" class="wp-caption-text">乌克兰南部港口城市马里乌波尔的亚速钢铁厂内，在乌克兰和俄罗斯战争中被摧毁的建筑与设施 (路透社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>当地媒体：一位韩国公民将在东乌克兰地区受审</strong></p>
<p>国际文传电讯社援引分离主义官员纳塔莉亚·尼科诺洛娃的话报道称，一名被抓获的韩国武装分子将在亲俄分裂分子控制的顿涅茨克人民共和国内接受审判。</p>
<p>根据国际文传电讯社的报道，“对一名来自韩国的雇佣兵的审判正在准备中。他是帮助乌克兰武装力量作战的韩国公民。”</p>
<p>9日，在为乌克兰作战期间被捕的两名英国人和一名摩洛哥人被当地法院判处死刑。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>冯德莱恩：欧盟对乌克兰成员国身份的评估将于下周出台</strong></p>
<p>欧盟委员会主席乌苏拉·冯德莱恩在基辅向泽连斯基表示，欧盟执委会将在下周末之前就乌克兰加入欧盟的申请发表意见。</p>
<p>冯德莱恩在新闻发布会上表示，与泽连斯基的讨论“将使我们能在下周末之前完成评估”，即是否批准乌克兰成为欧盟成员国的候选国。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-289531" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/EU-3.png?w=770&resize=770%2C384" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>一名在顿巴斯面临死刑的英国男子的家属呼吁释放这名男子</strong></p>
<p>英国男子肖恩·平纳在顿巴斯地区被俄罗斯代理当局判处死刑，其家属在听到这一消息后表示震惊，并要求交换或释放他。</p>
<p>肖恩·平纳的家属在一份声明中指出，“首先，我们全家对所谓的顿涅茨克人民共和国进行非法审判的结果感到震惊和难过。”</p>
<p>“根据《日内瓦公约》的规定，肖恩应享有战俘的所有权利，包括拥有充分独立的法律代表。我们真诚希望各方尽快合作，以确保肖恩的安全释放或交换。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>一名韩国人将在俄罗斯控制下的东乌克兰地区受审</strong></p>
<p>根据国际文传电讯社的报道，一名被俘的韩国武装人员将在亲俄分裂主义分子控制下的顿涅茨克人民共和国内接受审判。</p>
<p>国际文传电讯社援引分离主义官员纳塔莉亚·尼科诺洛娃的话报道称，“对一名来自韩国的雇佣兵的审判正在准备中。他是帮助乌克兰武装力量作战的韩国公民。”</p>
<p>她并没有透露这名武装人员的名字。</p>
<p>9日，在为乌克兰作战期间被捕的两名英国人和一名摩洛哥人被当地法院判处死刑。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>部长：多达30万吨的粮食处于乌克兰被摧毁的仓库中</strong></p>
<p>乌克兰农业部副部长塔拉斯·维索茨基表示，乌克兰仓库中储存的谷物可能多达30万吨，基辅方面表示，这些仓库在上周末被俄罗斯的炮击摧毁。</p>
<p>维索茨基在乌克兰国家电视台上表示，根据记录，在2月24日战争开始之时，乌克兰最大的农产品码头之一——位于黑海海岸的尼古拉耶夫港，其仓库内储存着25万至30万吨谷物，主要是小麦和玉米。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>马克龙表示，法国不愿向俄罗斯让步</strong></p>
<p>法国总统马克龙的一名官员表示，法国不愿向俄罗斯做出让步，并且希望乌克兰在与莫斯科侵略势力的战争中赢得胜利，以恢复其领土完整。巴黎方面正寻求缓解外界对其在这场冲突中的立场的担忧。</p>
<p>法国总统马克龙在一次采访中表示，要避免“羞辱”俄罗斯，这一点至关重要，只有这样，才能在战争结束时找到外交解决方案。但是这一言论遭到了乌克兰及其东欧盟友的批评。</p>
<p>这位官员在被问及马克龙的言论时，向记者评论称，“正如总统所说的那样，我们希望乌克兰获胜。我们希望乌克兰恢复领土完整。”</p>
<p>他还表示，“总统并没有要对普京或俄罗斯让步的意思。他直接表示，这并不是妥协，而是表明我们如何看待问题。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>部长：捷克公民在乌克兰的战争中丧生</strong></p>
<p>捷克外长利帕夫斯基表示，一名捷克公民在乌克兰死亡，这证实了媒体有关在乌克兰作战的捷克志愿军出现第一例伤亡的报道。</p>
<p>利帕夫斯基向记者们表示，“我可以确认一位捷克公民在乌克兰顿涅茨克地区死亡的事情”，“遗体将在明天被运往哈尔科夫地区。”</p>
<p>在当地政府的同意下，有100多位捷克人加入了乌克兰武装部队，以帮助乌克兰抵御俄罗斯在今年2月24日发起的军事行动。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基对粮食危机发出警告并敦促俄罗斯结束封锁</strong></p>
<p>乌克兰总统泽连斯基敦促国际社会施加压力，以结束俄罗斯海军对黑海港口的封锁。这种封锁阻断了乌克兰的粮食出口，并构成了出现全球粮食危机的威胁。</p>
<p>泽连斯基在新加坡香格里拉对话安全峰会上发表的视频讲话中指出，“食品短缺将不可避免地导致政治混乱，这可能会导致许多政府的崩溃，以及许多政界人士的下台。”</p>
<p>他还表示，“只要看看世界市场和某些国家的基本产品价格的暴涨，就能清楚地看到这种迫在眉睫的威胁。这是俄罗斯政府行为所导致的直接后果。”</p>
<p>乌克兰正在与联合国、土耳其和其他国家讨论开辟一条允许粮食出口的通道。泽连斯基表示，会谈的重点是关于这条走廊的“形式”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：乌克兰控制了数百人避难其中的北顿涅茨克工厂</strong></p>
<p>北顿涅茨克化工厂“Azot”仍处在乌克兰的控制之下，该地区州长表示，有数百名平民在其中避难。此前，一名受俄罗斯支持的分裂主义分子声称，共有300至400名乌克兰武装分子被困在其中。</p>
<p>卢甘斯克地区州长谢尔盖·盖代在消息应用Telegram上表示，“有关封锁Azot工厂的消息是谎言”。</p>
<p>他还写道，“我们的军队控制了北顿涅茨克的一个工业区，并且正在摧毁该镇内的俄罗斯军队。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯声称已击落3架乌克兰战斗机</strong></p>
<p>俄罗斯国防部称其防空部队击落了3架乌克兰战斗机。</p>
<p>俄罗斯国防部在一份声明中指出，俄罗斯军方在尼古拉耶夫地区击落了两架米格-29飞机，在哈尔科夫地区击落了一架苏-25战斗机。</p>
<p>对此，乌克兰方面没有立即发表评论。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基在亚洲会议上强调，阻止俄罗斯对整个世界至关重要</strong></p>
<p>乌克兰总统泽连斯基在新加坡举行的香格里拉对话会议上发表远程讲话称，乌克兰战争的结果不仅仅会对乌克兰造成影响，还将影响到国际秩序的未来。</p>
<p>他还表示，“我感谢你们的支持……但是这种支持不仅是对乌克兰的，也是对你们的”，“正是在乌克兰的战场上，这个世界的未来规则的边界正在被决定。”</p>
<p>泽连斯基指出，俄罗斯封锁了黑海和亚速海的港口，阻止乌克兰的食品出口至世界市场。他还表示，“如果……由于俄罗斯的封锁，我们无法出口粮食，那么在亚洲和非洲的许多国家都面临严重的粮食危机和饥荒。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟主席访问乌克兰以讨论其加入欧盟的申请</strong></p>
<p>欧盟主席乌苏拉·冯德莱恩访问乌克兰，并与该国总统泽连斯基讨论乌克兰加入欧盟的候选国资格问题。</p>
<p>冯德莱恩抵达基辅后在推特上写道，“我将与泽连斯基总统一起评估重建所需要的联合工作，以及乌克兰在走向欧洲的道路上所取得的进展。”</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯与乌克兰战争第108天的关键事件清单</strong></p>
<p>乌克兰陆军司令部表示，俄罗斯军队已经在北顿涅茨克附近的两个社区内占据了阵地，而英国国防部则表示，“激烈的巷战正在进行”，“双方都可能遭受大量伤亡”。</p>
<p>泽连斯基呼吁欧盟“迈出历史性的一步”，接受乌克兰成为欧盟成员国候选国。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯表示将对北约在波兰的部署做出回应</strong></p>
<p>国际文传电讯社援引一名俄罗斯外交官的话报道称，俄罗斯外交部表示，莫斯科将对北约在波兰集结部队的行动作出恰当反应。</p>
<p>该媒体援引俄罗斯外交部俄欧关系负责人奥列格·泰普金的话称，“我们一如既往地将会作出恰当回应，旨在消除针对俄罗斯联邦安全的潜在威胁。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰报告马里乌波尔又有24名儿童死亡</strong></p>
<p>乌克兰总检察长办公室表示了解到马里乌波尔又有24名儿童死亡，自俄罗斯发起军事行动以来，死亡的儿童总数达到了287名。</p>
<p>该办公室在即时通讯应用Telegram上表示，“在记录刑事犯罪的过程中，我们发现，由于俄罗斯军队的肆意炮击，顿涅茨克地区的马里乌波尔又有24名儿童死亡。”</p>
<p>“这些数字并非最终数字，在敌对行动最为活跃的地方、临时被占领的地方和解放区内，统计相关数据的工作正在进行中。”</p>
<p>俄罗斯否认袭击平民，并否认有关战争罪行的指控。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰官员称东部的战争将依靠西方的武器供应</strong></p>
<p>乌克兰官员表示，俄罗斯关注的东部战争目前主要是一场火炮之战，基辅在武器装备上严重落后。这就意味着，只有美国和其他国家履行承诺，向乌克兰提供包括导弹系统在内的更多、更好的武器，形势才有可能扭转。</p>
<p>乌克兰军事情报部副部长斯基比茨基向英国《卫报》的记者表示，“现在，这就是一场火炮之战。”</p>
<p>“现在的一切都将取决于(西方)给我们提供什么。俄罗斯的火炮数量是乌克兰的10到15倍。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国称俄罗斯军队没有在北顿涅茨克南部取得进展</strong></p>
<p>英国国防部在其每日情报更新中表示，乌克兰北顿涅茨克附近的俄罗斯军队未能在该市南部推进。</p>
<p>“激烈的巷战正在进行，双方可能都遭受了大量伤亡”，英国国防部在其推特官方账号上这样写道。</p>
<p>该部还补充称，俄罗斯的中型轰炸机可能发射了几十枚上世纪60年代的Kh-22空射重型反舰导弹，但这些导弹“非常不精确，并且因此可以造成重大的附带损害和平民伤亡”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
