<p>俄罗斯国防部长谢尔盖·绍伊古周四宣布，俄罗斯军队已经控制了乌克兰东部的马里乌波尔市，与此同时，俄罗斯军队继续围攻亚速斯塔尔工厂内的约 2000 名士兵。</p>
<p>绍伊古——他在向总统弗拉基米尔·普京提交的关于乌克兰军事行动报告时——补充说，有 142000 名平民从马里乌波尔市撤离。</p>
<p>另一方面，普京祝贺他的国防部长在马里乌波尔的行动取得了成功，并下令在不攻击情况下包围亚速斯塔尔工厂。</p>
<p>基辅曾重申，如果俄罗斯军队控制了马里乌波尔，与俄罗斯的谈判将停止，而俄罗斯外交部长谢尔盖·拉夫罗夫则表示，如果基辅不考虑俄罗斯的要求，这些谈判就没有未来。</p>
<h3>以下是这场战争第57天的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰副总理：与俄罗斯交换战俘，将带19人回国，其中包括10名士兵。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>克里姆林宫：</h3>
<ul>
<li>普京取消入侵亚佐夫斯塔尔的决定并不意味着在乌克兰的军事行动计划有任何变化。</li>
<li>俄罗斯正在等待乌克兰方面在双边谈判框架内回应其提议。</li>
<li>俄罗斯尚未回应联合国秘书长会见普京总统的要求。</li>
<li>乌克兰军队有可能交出武器并通过人道主义走廊离开马里乌波尔。</li>
<li>乌克兰宣布可能袭击克里米亚大桥，旨在为恐怖行为做准备，这是不可接受的。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>法新社援引乌克兰官员的说法报道称：基辅地区的太平间里有一千多具平民尸体。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>哈尔科夫市长：俄罗斯轰炸哈尔科夫市和舒戈耶夫镇，造成5人死亡。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰副总理：我呼吁世界领导人和国际社会现在将精力集中在亚速斯塔尔上。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<ul>

<li>我们的军队和卢甘斯克的分裂分子已经完全控制了克列梅纳亚镇。</li>
</ul>

<li>俄罗斯军队围攻了亚速斯塔尔工厂，并给予被围攻者最后一次投降的机会。</li>
