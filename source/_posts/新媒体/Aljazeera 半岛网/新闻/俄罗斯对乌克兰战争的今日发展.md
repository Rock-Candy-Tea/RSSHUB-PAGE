<p><strong>以下是这场战争最新的情况更新：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称乌克兰向布良斯克边境哨所发射了迫击炮</strong></p>
<p>俄罗斯安全部门表示，布良斯克地区的一个边境哨所遭到了来自乌克兰的迫击炮袭击，这是当地报告的一系列跨境袭击事件的最新一起。</p>
<p>俄罗斯联邦安全局发言人向俄罗斯国家电视台表示，没有人员在这起事故中受伤，但是有部分车辆受损。</p>
<p>乌克兰内政部顾问安东··赫拉什申科在社交网站“Telegram”上发帖称，边境附近布良斯克的一处军事设施有物体“坠落并着火”。</p>
<p>该帖并未明确否认乌克兰应为此事负责。俄罗斯国防部13日表示，乌克兰军队继续实施的“破坏和攻击”，可能引发针对基辅的打击。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>荷兰政府指示企业不要使用卢布购买俄罗斯天然气</strong></p>
<p>荷兰政府将指示企业不要按照莫斯科方面的提议，以卢布购买俄罗斯天然气，因为这样做会违反欧盟对俄罗斯实施的制裁。</p>
<p>该国经济事务部发言人表示，在欧盟委员会13日做出决定后，该国也遵循了同样的立场，目前正在向能源公司通报这一决定。</p>
<p>这位发言人表示，“发生的变化是，欧盟委员会在昨天得出结论称，支付卢布将会违反制裁，而与俄罗斯天然气工业银行进行的建设也将违反制裁。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称“莫斯科”号导弹巡洋舰上的火势已经得到控制</strong></p>
<p>俄罗斯国防部表示，水手们已经控制住了“莫斯科”号导弹巡洋舰上的火势，这艘巡洋舰是俄罗斯黑海舰队的主力战舰，目前，俄方正在采取措施将其拖回港口。</p>
<p>俄罗斯国防部表示，这艘巡洋舰的主要武器系统没有受损，船员们已被疏散至黑海水域的其他舰艇上。</p>
<p>莫斯科声称，这场火灾是因船上的弹药爆炸而引发的，而基辅方面则表示，这场火灾是该舰被两枚乌克兰制造的“海王星”反舰巡航导弹击中而引起的。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>哈尔科夫州长：当地受到的炮击造成4位平民死亡、10人受伤</strong></p>
<p>乌克兰哈尔科夫地区州长奥列格·辛耶胡博夫表示，俄罗斯对哈尔科夫市发动的炮击造成4位平民死亡，另有10人受伤。</p>
<p>该州长还在一份声明中敦促部分城镇内的居民撤离，因为预计该地区将会发生军事行动。</p>
<figure id="attachment_272427" aria-describedby="caption-attachment-272427" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-272427" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/2022-04-13T121417Z_1658255571_RC29MT909HFU_RTRMADP_3_UKRAINE-CRISIS-KHARKIV.jpg?w=770&resize=770%2C514" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-272427" class="wp-caption-text">在乌克兰哈尔科夫受到的一次火炮袭击之后，紧急救援人员将41岁的当地居民谢尔盖的尸体从其被摧毁的公寓中抬出 (路透社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>马里乌波尔市长否认了俄罗斯关于该港口已被占领的报道</strong></p>
<p>马里乌波尔市市长瓦迪姆·博伊琴科向德国广播公司表示，“马里乌波尔在过去、现在和将来都将是乌克兰的城市。”</p>
<p>博伊琴科表示，俄罗斯有关该市港口已被占领以及超过1000多名乌克兰战士投降的报道是“假新闻”。</p>
<p>几周以来，这座位于亚速海岸边的城市一直被俄罗斯军队和亲俄分裂武装包围。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>立陶宛称俄罗斯威胁要在波罗的海地区增加军事力量“并不新鲜”</strong></p>
<p>立陶宛总理希莫尼特表示，俄罗斯威胁要在波罗的海地区增加军事力量，甚至包括核力量，但这“并不新鲜”。</p>
<p>她向记者表示，“俄罗斯的威胁，并不是什么新鲜事”,“加里宁格勒多年来一直是一个军事化程度很高的地区，而且它位于波罗的海地区。”</p>
<p>俄罗斯的飞地加里宁格勒位于波罗的海岸边，而且被夹在北约成员国立陶宛和波兰之间。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯对其士兵疑似受到乌克兰酷刑的案件展开刑事调查</strong></p>
<p>俄罗斯调查委员会表示，将对乌克兰军人涉嫌对俄罗斯军人实施酷刑的案件展开刑事调查。</p>
<p>这个负责调查重大犯罪的委员会表示，一些俄罗斯士兵在扎波利日支亚和尼古拉耶夫地区被乌克兰军队俘获，并被乌克兰安全部门非法关押。</p>
<p>该委员会在一份声明中指出，“俄罗斯人遭受了身体上的暴力和酷刑，以迫使他们对其在（俄罗斯的）特别军事行动中被乌克兰安全机构非法拘留的实际情况作出虚假的解释。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯就瑞典和芬兰的加入北约向该联盟发出警告</strong></p>
<p>俄罗斯警告北约，如果瑞典和芬兰加入该军事联盟，那么俄罗斯将不得不加强其防御，并且不会再有波罗的海“无核化”谈判。</p>
<p>俄罗斯安全委员会副主席德米特里·梅德韦杰夫表示，“不能再讨论波罗的海的任何无核化状态，这种平衡必须得到恢复。”</p>
<p>他补充称，“直到今天，俄罗斯还没有采取这样的措施，而且它也不愿这样做。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：克拉马托尔斯克和康斯坦丁诺夫卡或将面临“无差别攻击”</strong></p>
<p>英国国防部表示，乌克兰城镇克拉马托尔斯克和康斯坦丁诺夫卡或将成为“无差别攻击”的目标，背景是俄罗斯军队正准备在乌克兰东部发动新的进攻。</p>
<p>英国国防部在最新的情报信息中指出，“在整个冲突过程中，城市中心不断受到俄罗斯方面的无差别攻击”，它还补充称，这两个城镇或将面临“程度类似的暴力”。</p>
<p>它还在推特上写道，乌克兰对港口城市马里乌波尔的防御，“目前牵制了俄罗斯大量的军队和装备”，而在东部地区发动进攻将“需要大量兵力”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰正在调查近6500起战争罪行</strong></p>
<p>乌克兰检察官办公室表示，他们正在调查俄罗斯军队在乌克兰犯下的近6500起战争罪行。</p>
<p>该办公室在推特上表示，共报告了6492起案件，已有197名儿童被证实死亡。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰声称撤离进程在中断后将重新开始</strong></p>
<p>乌克兰方面表示，将会重新开放人道主义走廊，以便从该国饱受战争创伤的地区撤离平民。基辅方面认为，该通道在前一天的暂停，是因俄罗斯违反停火规定而造成的。</p>
<p>乌克兰副总理伊琳娜·韦列修克（Iryna Vereshchuk）在社交媒体上发布的一份声明中指出，该国东部和南部的9条路线将在因“太危险”而关闭的一天之后重新投入运营。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧洲航天局暂停了与俄罗斯在登月计划上的合作</strong></p>
<p>由于俄罗斯入侵乌克兰，欧洲航天局终止了与俄罗斯在3项登月任务上的合作。此前，欧洲航天局曾决定对火星任务采取同样的举措。</p>
<p>欧洲航天局在一份声明中表示，“与ExoMars项目一样，俄罗斯对乌克兰的侵略以及由此产生的制裁意味着形势发生了根本性的变化，并使欧洲航天局无法与俄方进行计划中的登月项目合作。”</p>
<p>欧洲航天局表示，它将“中止”与俄罗斯在Luna-25、Luna-26和Luna- 27这三大项目上的“合作活动”，这些是俄罗斯的一系列探月任务，而欧洲航天局原计划通过这些任务测试其新的设备和技术。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>印孚瑟斯技术公司撤离俄罗斯</strong></p>
<p>根据《印度时报》的报道，由于与乌克兰之间的持续冲突，全球技术公司印孚瑟斯(Infosys)正将其业务从俄罗斯转移至“另一个地点”。</p>
<p>在此之前，甲骨文公司等其他大型技术公司也宣布暂停其在俄罗斯的所有业务。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>斐济警方讯问受制裁的俄罗斯亿万富翁的超级游艇船长</strong></p>
<p>根据路透社的报道，斐济警方对一艘抵达的豪华游艇进行了调查，而这艘游艇的主人是一名受到制裁的俄罗斯寡头，斐济警方讯问其船长是如何在没有通关的情况下，将这艘游艇带至这个太平洋岛国的。</p>
<p>斐济媒体在当日早些时候报道称，警方查获了俄罗斯亿万富翁苏莱曼·克里莫夫(Suleiman Kerimov)拥有的超级游艇“阿玛蒂亚”号(Amadea)，并拘留了该游艇上的船员。需要指出的是，克里莫夫因俄罗斯入侵乌克兰而受到美国、英国和欧盟的制裁。</p>
<p>根据路透社的报道，美国大使馆表示，他们知道“阿玛蒂亚”号停靠在斐济的劳特卡港，而且美国大使馆正在就此事与斐济当局进行合作。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰议员：“如果俄罗斯人夺权，我们将无法生存”</strong></p>
<p>一名乌克兰国会议员表示，除了赢得对俄罗斯的战争，她想不出还有什么办法能让乌克兰继续生存下去。</p>
<p>乌克兰议会入欧事务委员会主席伊万娜·克里普什向CNN记者指出，“如果俄罗斯人接管乌克兰，我不认为我还能在这个国家内生存。”</p>
<p>她还表示，“不仅是我，还有我的所有朋友，或者所有在这30年的独立过程中尽力建设一个民主、自由和独立国家的人，均是如此。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>有报道称赫尔松的乔尔诺巴伊夫卡机场发生爆炸</strong></p>
<p>相关报道和视频显示，赫尔松的乔尔诺巴伊夫卡机场发生了爆炸。</p>
<p>消息尚未得到证实。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>联合国难民署：“有报道称乌克兰妇女在英国感到来自男性赞助者的威胁”</strong></p>
<p>联合国难民署(UNHCR)对英国一项允许英国居民为逃离俄罗斯入侵的乌克兰人提供住所的计划中，可能存在对乌克兰妇女的剥削的情况表示担忧。</p>
<p>该机构指出，它发现越来越多的报告显示，乌克兰女性觉得存在来自男性赞助者的风险，该机构认为，女性以及带孩子的母亲应当与夫妇或家庭配对，而不是与单身男性。</p>
<p>联合国难民署在一份声明中指出，“除了流离失所、家庭分离和已经经历过的暴力所造成的创伤外，缺乏适当监督的配对可能会增加妇女面临的风险。”</p>
<p>英国政府在上个月推出了一项名为“乌克兰之家”的计划，允许英国人资助乌克兰人，并为他们提供至少为期6个月的住所。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>美国将扩大为乌克兰东部战争提供的情报</strong></p>
<p>根据《华尔街日报》的报道，美国计划加强向乌克兰提供的情报信息，以便其军队能够打击俄罗斯在顿巴斯和克里米亚地区的军事单位。</p>
<p>这篇报道指出，俄罗斯从首都城市内撤出军队，集中进攻顿巴斯和乌克兰东部的其他地区，从而“促成了(拜登)政府在上周晚些时候关于与乌克兰加强情报共享的决定”。</p>
<p>但是这项声明补充称，官员们还强调，美国会“避免提供让乌克兰人能够打击俄罗斯境内目标的情报”。</p>

<p><strong>泰国向乌克兰提供救援物资，包括食品、毯子和急救箱</strong></p>
<p>根据泰国外交部发布的一份声明，泰国驻华沙大使馆13日向波兰红十字会运送了第一批救援物资，以便在乌克兰分发。</p>
<p>这批物资包括7.1万件物品，其中包括食品、毯子、睡袋、婴儿和儿童用品、卫生用品和急救箱。</p>
<p>这些救援物资是泰国政府向乌克兰提供的价值200万泰铢(约合6万美元)的人道主义援助的一部分。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
