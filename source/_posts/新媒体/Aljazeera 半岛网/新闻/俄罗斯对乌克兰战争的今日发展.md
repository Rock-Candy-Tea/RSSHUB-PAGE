<div>   
<p>基辅当局表示，俄罗斯对乌克兰战争的第二阶段已经打响，莫斯科将其部队集中在该国东部的顿巴斯地区，旨在完全控制该地区。而在被围困的乌克兰南部城市马里乌波尔，受莫斯科支持的分离主义武装开始猛攻乌克兰军队在该市内坚守的最后几处据点。</p>
<p>由于乌克兰军队在东部地区与俄罗斯军队和分裂主义武装部队之间的战斗愈演愈烈，今天（19日）成为了连续第3个没有平民从战区撤离的日子。</p>
<p>在政治层面上，美国总统乔·拜登19日将与美国的盟友召开电话会议，以讨论向乌克兰提供支持以应对这场战争的问题，此外，还将讨论西方因这场自2月24日一直持续的战争而制裁莫斯科所付出的努力。</p>
<p><strong>以下是这场战争进入第55天以来的最新进展：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰副总理：我们与俄罗斯交换了俘虏，其中包括60名士兵和16位平民。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯外交部：莫斯科驱逐了15名荷兰外交官，作为对荷兰驱逐18名俄罗斯外交官的回应。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>美国有线电视新闻网（CNN）援引哈尔科夫市长的话报道：</p>
<ul>
<li>自17日以来，该市的各大平民社区一直受到俄罗斯持续轰炸的压力。</li>
<li>在过去的36个小时内，由于俄罗斯对该市实施的轰炸，已至少有15人死亡，50多人受伤。</li>
</ul>
<p>医务消息人士向半岛电视台记者透露：俄罗斯轰炸了哈尔科夫（乌克兰东部）的萨尔托夫卡居民区，造成至少3人死亡，超过12人受伤。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯国防部：对于所有选择在马里乌波尔（乌克兰南部）亚速斯塔尔工厂内放下武器的人员，我们将遵守日内瓦公约的规定，给予他们相应的战俘待遇。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯媒体：俄罗斯军队在马里乌波尔的亚速斯塔尔工厂内，为希望投降的乌克兰士兵开辟了一条通道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>英国首相府：一位安全官员告诉我们，乌克兰冲突的下一阶段将是一场持续数月的消耗战。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-273753" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/image-2-34.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰谈判代表团成员米哈伊洛·波多利亚克：</strong></p>
<ul>
<li>马里乌波尔事件进一步复杂化了与俄罗斯的谈判。</li>
<li>我们无法确定与俄罗斯举行下一轮直接谈判的日期。</li>
<li>莫斯科正寄希望于其第二阶段的军事行动。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>国际文传电讯社援引俄罗斯中央银行的消息报道：莫斯科计划采取法律措施，以反对针对其境外储备的史无前例的冻结措施。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯国防部长：</strong></p>
<ul>
<li>美国及其控制下的西方国家正竭尽全力地拖延乌克兰的特别行动。</li>
<li>越来越多的武器供应表明，西方有意推动基辅政权一直战斗到最后一人。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>法国财政部长：我们正努力说服我们在欧洲的合作伙伴，以禁止进口俄罗斯石油。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>日本防卫省：我们将向乌克兰提供包括无人机、口罩和化学武器防护服在内的援助物资。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>雅典通讯社：希腊东南部卡里斯托斯港当局扣押了悬挂俄罗斯国旗的油轮“Pegas”，以执行欧盟和北约（北大西洋公约组织）采取的制裁措施。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯国防部：我们呼吁位于马里乌波尔市亚速斯塔尔钢铁厂内的乌克兰军队立即交出武器。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯外交部长：</strong></p>
<ul>
<li>乌克兰成为了北约的平台，其军队在加强其东部存在之时违背了承诺。</li>
<li>我们正在对乌克兰极端主义军营及乌克兰武装部队犯下的罪行展开调查。</li>
<li>西方对犯罪和事故的调查通常不会产生任何结果。</li>
<li>俄罗斯军人被禁止实施任何犯罪行为，并且将对违法行为予以记录并展开调查。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯国防部：</strong></p>
<ul>
<li>我们的空军轰炸了乌克兰军队的60个军事设施，并摧毁了其中3个指挥中心。</li>
<li>摧毁了乌克兰的两个用于储存战术导弹弹头的仓库。</li>
<li>摧毁了乌克兰军队的13个据点，以及步兵部队和导弹装备的集中地。</li>
<li>在马里诺瓦地区击落了乌克兰的米格-29战斗机，并且在彼得罗波尔、马拉亚卡米什瓦卡拦截了两枚导弹。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>顿涅茨克地区分离主义者：</strong></p>
<ul>
<li>特种军事集团开始袭击马里乌波尔的亚速斯塔尔工厂。</li>
<li>在俄罗斯军队的支持下，突袭乌克兰军队在马里乌波尔的最后一处据点——亚速斯塔尔工厂——的行动正在进行。</li>
</ul>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰副总理：鉴于对顿巴斯地区的猛烈炮击仍在持续，今天没有开放人道主义走廊。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰总参谋部：</strong></p>
<ul>
<li>俄罗斯军队继续部分包围哈尔科夫市（东部），并将防空导弹小组转移到了顿巴斯地区。</li>
<li>俄罗斯将武器从其中部和东部地区运往乌克兰。</li>
<li>俄罗斯人将主要努力集中在突破我们部队在卢甘斯克和顿涅茨克地区的防御之上。</li>
<li>我们的部队在过去24小时内击退了俄军在顿涅茨克和卢甘斯克地区发动的7次袭击。</li>
<li>我们的空军击落了俄罗斯的7个空中目标，包括一架战斗机和一枚巡航导弹。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯塔斯社：新西兰因乌克兰战争对俄罗斯银行和金融机构实施制裁。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>白宫：拜登将于19日与盟友通话，以讨论继续支持乌克兰以及追究俄罗斯责任的问题。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰外交部：16个外交使团已经恢复了它们在基辅的活动。</p>
