<li>随着对爆发全球粮食危机的担忧加剧，欧盟最高外交官员表示，俄罗斯封锁乌克兰港口并阻止其粮食出口，加剧了人们对全球粮食危机的担忧，这相当于一桩“战争罪行”。</li>
<li>俄罗斯国防部表示，俄罗斯军队袭击了乌克兰西南部敖德萨地区的一座机场，并且摧毁了多架无人机和一座无人机控制站。</li>
</ul>

<h3>以下是这场战争进入今日以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰官员：争夺北顿涅茨克的“决定性”战斗仍在继续</strong></p>
<p>乌克兰国防部副部长安娜·马里亚尔表示，针对北顿涅茨克的“决定性”战斗正在进行，俄罗斯将资源集中在控制这座具有重要战略意义的城市上。</p>
<p>安娜·马里亚尔在电视讲话中指出，“毫不夸张地说，针对北顿涅茨克的决定性战斗正在进行，敌人的计划和目标是在6月26日前到达卢甘斯克地区边界。”</p>
<p>她还补充称，俄罗斯军队“投入了所有的力量和资源”，以“袭击城市周围的居民区”。</p>
<p>半岛电视台记者自基辅报道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>立陶宛称只是停止向加里宁格勒运送受到制裁的货物</strong></p>
<p>立陶宛外交部向俄罗斯驻维尔纽斯外交使团负责人表示，立陶宛禁止俄罗斯本土与其飞地加里宁格勒之间过境的货物只是受欧盟制裁的货物。</p>
<p>立陶宛外交部在一份声明中指出，立陶宛没有实施“单方面的、个别的或额外的”限制措施。被禁止的商品包括煤炭、金属、建筑材料和先进技术。</p>
<p>据悉，俄罗斯大陆和其位于波罗的海的加里宁格勒飞地之间，唯一的铁路线路必须经过立陶宛。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报道：有关被抓获的美国公民的视频片段似乎揭示了他们的行踪</strong></p>
<p>美国有线电视新闻网(CNN)的一篇报道称，两名曾与乌克兰军队并肩作战的美国公民似乎被拘留在自行宣布成立的“顿涅茨克人民共和国”境内。</p>
<p>据信，俄罗斯军队逮捕了今年27岁的曾在美国海军陆战队服役的安迪·黄恩(Andy Huynh)和今年39岁的前美国陆军军官亚历山大·德鲁伊克(Alexander Drueke)。</p>
<p>二者于6月9日在哈尔科夫失踪。</p>
<p>CNN在20日报道称，在“YouTube”上一个亲俄的塞尔维亚民族主义频道周末播出的对上述两人的采访中，可以听到镜头之外的一个人在提问时说道，“这里是顿涅茨克”。</p>
<p>而另外3名外国武装分子——包括两名英国人和一名摩洛哥人，最近被受俄罗斯支持的分裂主义分子设立的法院判处死刑。</p>
<figure id="attachment_292266" aria-describedby="caption-attachment-292266" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-292266" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/AP22163637248597.jpg?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-292266" class="wp-caption-text">俄罗斯军方认为，与乌克兰作战的外国人员属于雇佣军，并声称他们作为武装人员并不受日内瓦公约的保护 (美联社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>柏林将举办粮食安全会议</strong></p>
<p>德国政府发言人称，德国将就乌克兰战争引发的日益严重的粮食危机召开会议，美国国务卿安东尼·布林肯将出席这场会议。</p>
<p>这位发言人在柏林举行的新闻发布会上表示，将在本周五以“团结起来保障全球粮食安全”为主题，召开会议以解决当前冲突导致的粮食短缺问题。</p>
<p>他还表示，“我国政府在担任七国集团轮值主席国期间，特别致力于寻求被俄罗斯与乌克兰战争引发的日益逼近的全球饥饿危机的共同解决方案。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰官员：俄罗斯人在马里乌波尔“将死者当作垃圾”</strong></p>
<p>一名乌克兰地方官员称，俄罗斯军队已经开始使用拖拉机挖掘和运输在乌克兰被占领的东南部港口城市马里乌波尔内因炮击而身亡的平民的尸体。</p>
<p>马里乌波尔市长的顾问彼得罗·安德里尤琴科在Telegram上发帖称，“尸体被挖出来，装上拖拉机，并被运至停尸房。”需要指出的是，他的工作地点在城外。</p>
<p>他还声称，其中一些尸体随后被埋入了乱葬岗。</p>
<p>他还表示，“将死者当作垃圾对待，已经成为一种常态。”半岛电视台记者无法独立证实他的说法。</p>
<p>半岛电视台记者自基辅报道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称其部队袭击了敖德萨机场</strong></p>
<p>俄罗斯国防部表示，俄罗斯军队袭击了乌克兰西南部敖德萨地区的一个机场，并且摧毁了两架“Bayraktar”无人机和一个无人机控制站。</p>
<p>俄罗斯国防部发言人表示，一枚高精度的“onik”导弹被用于袭击当地的“Artsyz”机场。乌克兰军方早些时候表示，其防空系统阻止了针对敖德萨地区的两次未遂袭击，并且摧毁了来袭的导弹。</p>
<p>半岛电视台记者无法独立核实双方的说法。</p>
<figure id="attachment_292129" aria-describedby="caption-attachment-292129" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-292129" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/2022-06-08T200351Z_1842176048_RC2NNU9QY33Y_RTRMADP_3_UKRAINE-CRISIS-DONETSK-REGION.jpg?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-292129" class="wp-caption-text">俄罗斯在乌克兰的攻势正在阻止粮食离开乌克兰这座“世界粮仓”，并导致全球粮食价格上涨 (路透社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：乌克兰军队失去了对北顿涅茨克附近村庄的控制</strong></p>
<p>卢甘斯克州长海岱称，乌克兰军队已经“失去”对东南部村庄梅约金的控制，该村庄靠近被围攻的北顿涅茨克市。</p>
<p>海岱在Telegram上表示，“北顿涅茨克和利西昌斯克周围的许多城镇都处于激烈的战斗之下。不幸的是，现在我们会失去了对该地区中心附近的梅约金的控制。”</p>
<p>海岱补充称，俄罗斯军队加强了对北顿涅茨克工业区的炮击，而数百名乌克兰士兵和平民在其中避难。</p>
<p>半岛电视台记者无法独立证实这一说法。</p>
<p>半岛电视台记者自基辅报道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯要求立陶宛立即解除加里宁格勒的过境禁令</strong></p>
<p>俄罗斯要求立陶宛立即解除针对俄罗斯飞地加里宁格勒部分货物的过境禁令。</p>
<p>俄罗斯外交部向立陶宛驻莫斯科特使表示，如果在加里宁格勒地区和俄罗斯其他地区之间经立陶宛的货物运输无法恢复，那么莫斯科将对此作出回应，以保护自身的利益。</p>
<p>立陶宛当局禁止受到欧盟制裁的货物过境，而该国境内存在一条连接俄罗斯大陆和俄罗斯位于立陶宛与波兰之间的加里宁格勒飞地的铁路，而这也是连接这两个地区的唯一一条铁路线路。</p>
<p>被禁止过境的商品包括煤炭、金属、建筑材料和先进技术。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰在赫尔松突破了俄罗斯的“第一道防线”</strong></p>
<p>据当地官员称，乌克兰军队在被占领的南部赫尔松地区“突破”了俄罗斯军队的“第一道防线”。</p>
<p>赫尔松州长助理谢尔盖·哈兰在电视讲话中表示，乌克兰军队正在第聂伯河的三角洲内推进，并且摧毁了俄罗斯位于新卡霍夫卡镇的军火库。</p>
<p>半岛电视台记者无法独立证实这一说法。</p>
<p>进入克里米亚的关键门户赫尔松地区的大片领土在莫斯科发起这场军事行动初期便已被俄军占领。</p>
<p>半岛电视台记者自基辅发回报道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>亲俄地区负责人表示，乌克兰军队袭击了克里米亚的石油钻井平台</strong></p>
<p>乌克兰被俄吞并地区负责人谢尔盖·阿斯基诺夫表示，乌克兰军队袭击了克里米亚半岛附近属于“Chernomorneftegaz”石油和天然气公司的钻井平台。</p>
<p>谢尔盖·阿斯基诺夫在Telegram上发帖称，有3人受伤，针对另外7名工作人员的搜救工作仍在进行中。</p>
<p>半岛电视台记者无法独立证实这一说法。</p>
<p>需要指出的是，2014年初，俄罗斯从乌克兰手中夺取了克里米亚半岛。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄乌战争如何引发了全球粮食危机？</strong></p>
<p>俄罗斯与乌克兰的战争正在阻止粮食离开乌克兰这座“世界粮仓”，并卖到全球粮食价格上涨，并且有可能加剧发展中国家的粮食短缺、饥饿和政治不稳定。</p>
<p>这场冲突导致乌克兰向中东、北非和亚洲部分地区约2000万吨粮食的出口中断。</p>
<p>与此同时，有关建立安全走廊、将食品运出乌克兰黑海港口的谈判已进行了数周的时间，但却进展甚微。随着夏季收获季节的到来，谈判的紧迫性越来越大。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>梅德韦杰夫：俄罗斯不应就核问题与美国谈判</strong></p>
<p>俄罗斯前总统梅德韦杰夫认为，莫斯科不应该与华盛顿就核裁军问题进行谈判，除非美国主动回归谈判。</p>
<p>梅德韦杰夫在Telegram上发布的一篇帖子中指出，“我们现在和美国没有任何关系。”</p>
<p>他还表示，“目前还没有必要(就核裁军)与他们谈判。这对俄罗斯不利。让他们自己回去吧，并对此发出呼喊。”</p>
<p>甚至在莫斯科发起军事行动之前，俄美关系就已经处于冷战结束以来的最低水平。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>分裂主义分子声称控制了北顿涅茨克南部的村庄</strong></p>
<p>乌克兰东部受俄罗斯支持的分裂主义势力表示，他们已经控制了通往北顿涅茨克的南部主要公路旁边的一个村庄。</p>
<p>俄罗斯塔斯社援引所谓的“卢甘斯克人民共和国”内政部长助理维塔利·基谢列夫的话报道称，位于北顿涅茨克以南约25公里的托什基夫卡村(Toshkivka)已被占领。</p>
<p>卢甘斯克州长谢尔盖·海岱此前曾对乌克兰电视台表示，亲俄势力正试图突破托什基夫卡的防线。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>德国声称正在与波兰、罗马尼亚合作，以释放被困的乌克兰粮食</strong></p>
<p>德国外长贝尔伯克表示，柏林支持波兰和罗马尼亚改造铁路，以使数百万吨因俄罗斯的海上封锁而滞留在乌克兰境内的粮食得以出口。</p>
<p>贝尔伯克在抵达卢森堡参加与欧盟各国外长举行的会议时指出，“铁路轨道需要现代化，我们需要合适的货运马车——德国政府正在与许多其他方面展开合作。”</p>
<p>“显然，我们最终肯定无法把所有的粮食都运出，但是，如果我们设法通过各种途径把其中一部分粮食运出，这在我们面临全球挑战的时候，都将是很有帮助的。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟高级外交官：俄罗斯封锁乌克兰粮食出口是一项战争罪</strong></p>
<p>欧盟外交政策负责人博雷利认为，俄罗斯封锁乌克兰数百万吨粮食的出口，是一桩战争罪行。</p>
<p>他向记者们表示，“我们呼吁俄罗斯解除对(乌克兰)港口的封锁……这是不可思议之事，人们无法想象，当世界其他地方的人们正在挨饿时，乌克兰的数百万吨小麦仍然处于被封锁的状态。”</p>
<p>他补充称，“这是真正的战争罪行，所以我无法想象这种情况还会持续更久”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：北顿涅茨克的战争昼夜不停</strong></p>
<p>卢甘斯克州长海岱表示，乌克兰军队在东部城市北顿涅茨克坚守阵地，战斗昼夜不停。</p>
<p>海岱在Telegram上写道，“敌人经常使用违禁弹药。”</p>
<p>市长亚历山大·斯特吕克表示，当俄罗斯军队获得对北顿涅茨克全部居民区的控制权后，他们正在驱逐市内居民。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰称有323名儿童在战争中丧生</strong></p>

<p>欧盟外长将于２０日在卢森堡举行会议，讨论如何释放因俄罗斯黑海港口封锁而滞留在乌克兰境内的数百万吨粮食。</p>
<p>欧盟支持由联合国斡旋以达成一项协议，恢复乌克兰的海上出口，以换取俄罗斯的食品和化肥出口，但这需要得到莫斯科的批准。</p>
<p>土耳其与基辅和莫斯科都有良好的关系，它表示，如果达成协议，土耳其准备在伊斯坦布尔的“观察机制”中发挥作用。目前尚不清楚欧盟是否会参与军事行动以确保达成这样一项协议。</p>
<p>一名欧盟官员告诉路透社记者，“未来是否会有为这些商船护航的需求，这还是一个问号，我认为我们还没有走到那一步。”</p>

<p><strong>俄罗斯在争夺乌克兰东部关键城市的战斗中推进</strong></p>
<p>俄罗斯方面表示，俄罗斯军队占领了乌克兰工业城市北顿涅茨克附近的一个村庄，这是莫斯科控制乌克兰东部的主要目标。</p>
<p>日本国防部１９日表示，他们已经夺得了梅约金(Metyolkine)的胜利——在这场战争爆发之前，梅约金只是一个居民人数不足800的定居点。俄罗斯国家通讯社塔斯社报道称，许多乌克兰武装分子在当地投降。</p>
