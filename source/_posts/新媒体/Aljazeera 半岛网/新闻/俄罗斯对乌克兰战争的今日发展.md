<li>乌克兰总统泽连斯基表示，顿巴斯战役的代价对乌克兰而言“非常之高”，而且“非常可怕”。</li>
<li>他还表示，这将作为欧洲最暴力的战争之一而被载入军事史。</li>
<li>克里姆林宫发言人佩斯科夫表示，俄罗斯在乌克兰的主要目标是保护独立的“顿涅茨克人民共和国”和“卢甘斯克人民共和国”。</li>
<li>乌克兰国家警察局长透露，当局正在调查超过1.2万名乌克兰人在这场战争期间被杀害的事件。</li>
<li>北顿涅茨克地区州长表示，当地的所有桥梁都已被摧毁，无法运送人道主义物资或撤离市民。</li>

<h3>以下是这场战争进入今日以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯天然气工业股份公司表示，俄罗斯将大幅削减通过北溪管道的天然气输送</strong></p>
<p>俄罗斯能源巨头俄罗斯天然气工业股份公司表示，由于需要德国西门子公司“修复”压缩机单元，该公司将减少通过北溪管道(Nord Stream)向德国输送的天然气。</p>
<p>俄罗斯天然气工业股份公司在Telegram上的一份声明中表示:“目前通过北溪天然气管道的天然气供应量可达每天1亿立方米。”该公司补充称，预期的日供应量为1.67亿立方米。</p>
<p>该公司还表示，由于西门子推迟交付零部件，目前在西北部城市维堡附近的“Portovaya”压缩站内只有3台天然气压缩设备尚在运行。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称会向躲在“Azot”化工厂内的乌克兰武装分子提供投降机会</strong></p>
<p>根据国际文传电讯社的报道，俄罗斯国防部表示，将为那些在乌克兰东部城市北顿涅茨克“Azot”化工厂内躲避的乌克兰武装分子提供在6月15日投降的机会。</p>
<p>俄罗斯国防部表示，乌克兰已经要求俄罗斯建立一条撤离通道，以帮助平民离开这座化工厂，背景是连接北顿涅茨克和乌克兰控制地区的所有桥梁目前都已被摧毁。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>法国称马克龙访问乌克兰只是一个可能的“选择”</strong></p>
<p>法国政府发言人表示，法国总统马克龙访问乌克兰首都基辅，是当前讨论的“几个选项”之一，尽管尚未就此采取决定。</p>
<p>德国媒体在本月早些时候报道称，德国总理舒尔茨将于本周四与法国总统马克龙、意大利总理马里奥·德拉吉一起前往基辅。</p>
<p>自这场战争开始以来，马克龙一直寻求与俄罗斯总统普京保持对话，但是这一立场遭到了部分东欧和波罗的海国家的欧洲伙伴的批评，他们认为这破坏了将普京推至谈判桌前的努力。</p>
<figure id="attachment_290244" aria-describedby="caption-attachment-290244" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-290244" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/000_32AT236.jpg?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-290244" class="wp-caption-text">连接乌克兰东部顿巴斯地区的利西昌斯克市和北顿涅茨克市之间的桥梁已经倒塌 (法国媒体)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>部长：德国榴弹炮即将在乌克兰投入使用</strong></p>
<p>德国国防部长克里斯汀·兰布雷希特表示，乌克兰军队对德国榴弹炮的训练将很快完成，从而为在乌克兰战争中使用这种武器铺平道路。</p>
<p>她在参观德国西部莱茵巴赫镇的一个军事基地时向记者表示，“PzH-2000型榴弹炮的训练即将完成，以便在乌克兰的战斗中使用。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基：弹药就绪，但需要远程武器</strong></p>
<p>泽连斯基在新闻发布会上告诉丹麦记者，乌克兰军队拥有足够的弹药和武器，但却需要更多的远程武器。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰主要城市州长：有70人被撤离</strong></p>
<p>卢甘斯克地区州长海岱表示，另有70人从利西昌斯克及其周边定居点撤离。</p>
<p>海岱在一份电报中称，该地区的状况是“真正的地狱”。</p>
<p>他还解释称，居民们面临着严重的威胁，只有在夜幕降临时才能出行。</p>
<p>“炮火如此猛烈，人们无法再藏身掩体之内”，“但是只要有可能再挽救一条生命，我们就不会放弃，我们就会去营救。”</p>
<p>受俄罗斯支持的分裂分子已经控制了卢甘斯克的大部分地区。</p>
<figure id="attachment_290359" aria-describedby="caption-attachment-290359" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-290359" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/INTERACTIVE-Russia-Ukraine-War-Who-controls-what-in-Donbas-DAY-111.png?w=770&resize=770%2C770" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-290359" class="wp-caption-text"><strong>俄乌战争</strong><br><strong>顿巴斯各地处于哪方势力的控制之下？</strong><br>北顿涅茨克地区州长表示，当地的所有桥梁都已被摧毁，无法运送人道主义物资或撤离市民。克里姆林宫发言人佩斯科夫表示，俄罗斯在乌克兰的主要目标是保护独立的“顿涅茨克人民共和国”和“卢甘斯克人民共和国”。 (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟机构警告称，乌克兰冲突可能加剧毒品问题</strong></p>
<p>总部位于里斯本的欧盟毒品监管机构警告称，俄罗斯与乌克兰的战争可能会引发走私路线的转变，并使更多的人接触到毒品，从而给欧洲打击毒品的进程制造“新的漏洞”。</p>
<p>欧洲毒品和毒瘾监测中心(EMCDDA)在其年度报告中指出，许多在冲突期间遭受“严重心理压力”的人可能会在未来更容易出现药物滥用的问题。</p>
<p>报告指出，毒贩可能会转向其他路线，以避开加强安全措施的地区，而欧洲国家的医疗服务——尤其是在与乌克兰接壤的国家内，可能会变得更加紧张，因为逃离冲突的吸毒者将寻求支持。</p>
<p>报告指出，“持续的治疗、语言服务、提供住宿和社会福利支持可能是基本的需求”，并补充称，即使那些非吸毒人员也面临风险。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官员：俄罗斯将重新开放马里乌波尔剧院</strong></p>
<p>一名乌克兰官员批评俄罗斯军队在乌克兰南部城市马里乌波尔的行动——据称，俄罗斯当局计划重新开放曾有“数百名”乌克兰人在内被轰炸的那家剧院。</p>
<p>一项媒体调查发现，有证据表明，这场袭击导致大楼内外近600人死亡。</p>
<p>马里乌波尔市长顾问安德里·尤先科在Telegram上表示，俄罗斯计划于今年9月1日重新开放这家剧院。</p>
<p>他在谈到这项计划时表示，“这是在骨头上跳舞，在墓地上办展。”</p>
<p>半岛电视台记者自乌克兰基辅发回报道。</p>
<figure id="attachment_290110" aria-describedby="caption-attachment-290110" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-290110" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/Severodonetsk.jpg?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-290110" class="wp-caption-text">2022年6月8日，乌克兰卢甘斯克地区的北顿涅茨克前线，被毁坏的汽车残骸被遗弃在路边 (美联社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>哈尔科夫州长称，俄罗斯的炮击造成1人死亡、5人受伤</strong></p>
<p>哈尔科夫州长谢尼胡波夫表示，在东部地区，俄罗斯人击打一名妇女致死，并导致其3个月大的孩子受伤。</p>
<p>他在Telegram上表示，俄罗斯的炮击破坏了房屋和公寓楼，并引发了火灾，整个地区新增4人受伤。</p>
<p>半岛电视台记者自乌克兰基辅发回报道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官员称俄罗斯军队在扎波罗热绑架3人</strong></p>
<p>当地官员表示，俄罗斯军人在乌克兰东南部扎波罗热地区被占土地上绑架了3名乌克兰人。</p>
<p>当地政府在Facebook上表示，其中一名是在扎波罗热核电站所在的埃涅尔戈达尔镇上被绑架的，而另外两名则是从梅利托波尔区被带走的。</p>
<p>自俄罗斯军队占领该地区一半以上的面积以来，该核电站的数十名员工和亲乌克兰的同情者遭到了绑架。</p>
<p>半岛电视台记者自乌克兰基辅发回报道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>教皇批评俄罗斯在乌克兰的行动，但他表示这场战争可能是被挑起的</strong></p>
<p>教皇方济各对俄罗斯在乌克兰的军事行动发起了一系列新的抨击，并称俄罗斯军队残忍、凶猛，同时赞扬了为生存而战的、“勇敢的”乌克兰人。</p>
<p>但是在上个月与教会媒体编辑的对话中，方济各表示，当前的局势并非黑白分明，而且这场战争“或许在某种程度上是被挑起的”。</p>
<p>方济各指出，“如果我们希望这些问题得到解决，我们就不能忘记真正的问题所在”——他将军备工业列为这场战争的诱因之一。</p>
<p>“的确，俄罗斯人原认为在一周之内就会结束这场军事行动。但是他们错了。他们遇到了一个勇敢的民族，一个为生存而奋斗的民族，一个有着斗争历史的民族。”</p>
<figure id="attachment_290210" aria-describedby="caption-attachment-290210" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-290210" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/AP22114368327508.jpg?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-290210" class="wp-caption-text">教皇方济各表示，就在普京派兵进入乌克兰的几个月前，他曾会见了一位国家元首，后者对北约“在俄罗斯家门口的狂吠”可能导致战争提出了担忧 (美联社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>新的制裁措施出台后，莫斯科交易所暂停瑞士法郎交易</strong></p>
<p>莫斯科证交所表示，在瑞士采取欧盟针对俄罗斯的新制裁措施后，将暂停瑞士法郎兑卢布和美元的交易。</p>
<p>俄罗斯最大的交易所莫斯科交易所表示，由于瑞士上周实施新的交易限制，该交易所难以进行瑞士货币交易。</p>
<p>莫斯科证交所在一份声明中表示，“由于瑞士6月10日实施的限制性措施，市场参与者和金融业在用瑞士法郎进行结算时遇到了困难，因此暂停相关交易。”</p>
<figure id="attachment_290215" aria-describedby="caption-attachment-290215" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-290215" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/AP22061654356046.jpg?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-290215" class="wp-caption-text">不属于欧盟成员国的瑞士在上周五更新了制裁方案，以配合欧盟针对俄罗斯及白俄罗斯的企业、银行和个人的最新限制 (美联社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官员称有4人在与乌克兰接壤的俄罗斯城镇受到的炮击中受伤</strong></p>
<p>地区官员表示，在与乌克兰接壤的布良斯克地区，有4人在俄罗斯城镇受到的炮击中受伤。</p>
<p>该事件发生在距离乌克兰边境约50公里的克林特西镇。</p>
<p>当地官员亚历山大·博格玛兹在即时通讯应用Telegram上写道，“根据初步信息，有多座房屋被毁，共4人受伤。”</p>
<p>当地居民们在社交媒体上发布了一段视频，视频显示一架俄罗斯军用直升机在炮击后盘旋在该镇上空，此外，还有报告称水电已被切断。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：北顿涅茨克的情况“极度艰难”</strong></p>
<p>卢甘斯克州长海岱称北顿涅茨克的情况“极度艰难”。</p>
<p>他在Telegram上写道，“针对这座城市的进攻已经持续了好多天。敌人使用大炮摧毁了高层建筑和工业设施。”</p>
<p>他还表示，每天都有人员死亡，但是他们的尸体“由于密集的炮击而难以接近”，他还补充称，“有许多受损的高层建筑……俄罗斯军队开枪摧毁了其中部分基础”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：俄罗斯国防工业可能难以满足乌克兰战争的要求</strong></p>
<p>英国国防部表示，俄罗斯国防工业可能难以进一步满足乌克兰战争的需求，部分原因是制裁的影响和专业知识的缺乏。</p>
<p>俄罗斯军事工业委员会的一名高级官员预测，国防开支可能会使俄罗斯的国防预算增加20%。</p>
<p>英国国防部表示，“该行业可能难以满足其中许多要求。俄罗斯的高质量光学和先进电子产品生产可能仍存在问题，可能会破坏其替换在乌克兰损失的设备的努力。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>教皇方济各表示拒绝在乌克兰战争中区分“好与坏”</strong></p>
<p>意大利《新闻报》(La Stampa)报道了教皇方济各与欧洲教会文化杂志编辑之间的对话，并称方济各拒绝在乌克兰战争中区分“好与坏”。</p>
<p>当被问及是否支持俄罗斯总统普京时，教皇方济各回答称，“不，我不支持，我只是反对将复杂的问题变成简单的好与坏的区分。”</p>
<p>他还表示，教皇希望今年9月能在哈萨克斯坦会见俄罗斯东正教大牧首基里尔。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>智库：俄罗斯军队将乌克兰军队赶出了北顿涅茨克市中心</strong></p>
<p>智库战争研究所表示，俄罗斯军队13日将乌克兰军队从北顿涅茨克市中心赶出，但尚未完全占领这座城市。</p>

<p>分离主义官员和俄罗斯新闻机构13日报道了乌克兰境内发生的多次炮击，其中包括针对一个市场的炮击。俄罗斯新闻机构随后报道称，一枚炮弹落在顿涅茨克市的一家妇产医院内，并引发火灾，从而促使该医院的工作人员将病人送往地下室。</p>

<p>半岛电视台记者无法独立核实该报道，基辅方面也没有立即对此作出回应。乌克兰经常否认对顿涅茨克和卢甘斯克人民共和国这两个自主独立地区发动过任何袭击。2014年，分裂分子在这两个地区占领了大片土地。</p>

<p><strong>调查：乌克兰战争和成本的飙升动摇了澳大利亚农民的信心</strong></p>
<p>一项调查发现，近一半的澳大利亚农民认为乌克兰战争将损害农业事务。</p>
<p>只有28%的当地农民预计未来12个月的经营状况会有所改善，而在上一季度，这项比例为31%。从总体而言，当地农民预计在未来12个月内，他们的收入将保持稳定。</p>
<p>拉博银行（Rabobank）表示，尽管乌克兰战争推高了农场的销售价格，尤其是谷物的销售价格，但是这些价格的上涨也是抵消投入成本上涨的必需因素。</p>
<p>该银行还指出，化肥、燃料、货运和机械的成本不断上升(其中一些也是由战争推动的)，再加上澳大利亚经济面临更广泛的通胀压力，这些因素都令市场承压。</p>

<p><strong>泽连斯基承诺要“解放”被占领的乌克兰领土</strong></p>

<p>“我们将去往赫尔松。赫尔松地区的普通居民将在城市的街道上与我们的军队相遇，我们将去往梅利托波尔。我们要让梅利托波尔的所有居民都有机会生活在没有恐惧的环境中。”</p>
<p>“我们会去往马里乌波尔。我们将第3次解放这座城市”，他还解释称，这座城市先是在1943年从纳粹手中解放出来，然后又于2014年6月13日从受俄罗斯支持的分裂分子手中再次解放出来。</p>
<p>“我们要去往埃涅尔戈达尔。我想对这座城市内所有走上街头反对俄罗斯军队、拒绝与占领者合作、今天还在等待我们的人们重复这句话。我想重申，我们不曾忘记我们的埃涅尔戈达尔，一天都不曾。”</p>
