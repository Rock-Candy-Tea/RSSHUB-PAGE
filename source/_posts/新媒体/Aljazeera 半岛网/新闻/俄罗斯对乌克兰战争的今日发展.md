<div>   
<p>乌克兰当局今天周三表示，他们已与俄罗斯达成初步协议，以建立一条人道主义走廊，从被围困的南部城市马里乌波尔撤离妇女、儿童和老人；与此同时，该国东部顿巴斯地区发生激战，俄军和亲俄分裂势力正试图突破乌克兰军队的防线。</p>
<p>另一方面，挪威今天表示，已向乌克兰运送了约100枚防空导弹，而美国等西方国家则承诺向基辅提供更多武器，特别是重型火炮，以应对俄罗斯对东部地区的袭击，英国表示正在研究向乌克兰军队提供反舰导弹的可能性。</p>
<p><strong>以下是这场战争第56天的最新进展：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>国际货币基金组织总裁：</p>
<ul style="list-style-type:disc">
<li>我们必须尽我们所能帮助乌克兰和其他受影响严重的国家。</li>
<li>欧洲的战争简单地转化为非洲的饥荒。</li>
<li>非洲国家受到能源和食品价格上涨的打击。</li>
<li>我们正在重组大约20个撒哈拉以南非洲国家的债务。</li>
<li>我与乌克兰总统讨论了他国家的重建工作。</li>
<li>我们向乌克兰提供了14亿美元的紧急融资。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄外交部：美国向乌克兰运送的直升机可用于瞄准我国领土。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯外交部发言人玛丽亚·扎哈罗娃：</p>
<ul style="list-style-type:disc">
<li>新西兰向乌克兰提供武器弹药并为其提供财政支持。</li>
<li>乌克兰谈判代表拖延，但与基辅的谈判仍在继续，谈判内容包括克里米亚、顿涅茨克和卢甘斯克。</li>
<li>华盛顿向基辅提供了之前要求阿富汗军队使用的俄罗斯Mi-17直升机。</li>
<li>任何关于援助基辅的言论都是谎言，这笔钱是用来杀死乌克兰和俄罗斯人民的。</li>
<li><strong>更新</strong></li>
</ul>
<p>乌克兰议会人权专员：</p>
<ul style="list-style-type:disc">
<li>自军事行动开始以来，乌克兰已有205名儿童丧生，373名儿童受伤。</li>
<li>俄罗斯军队破坏了1141所教育机构，其中99所被完全摧毁。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯外交部：如果资助我们驻外使团的可能性受阻，我们将采取报复措施。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>路透社援引乌克兰总统顾问：我们的部队击退了俄罗斯对斯洛维扬斯克（东部）的袭击，俄罗斯人尚未成功占领它。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>英国首相：</p>
<ul style="list-style-type:disc">
<li>我敦促俄罗斯以人道主义和仁慈对待囚犯，并积极努力防止个人进入战区。</li>
<li>我告诉俄罗斯官员，他们将被国际刑事审判追究责任，就像那些参与波斯尼亚大屠杀的人一样。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>普京：</p>
<ul style="list-style-type:disc">
<li>我们的军事行动将继续进行，直到卢甘斯克和顿涅茨克的居民得到充分的安全保障。</li>
<li>顿涅茨克和卢甘斯克的局势在过去几年中变得更糟，那里的公民生活在艰难的条件下。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>塔斯社援引俄罗斯外交部：</p>
<ul style="list-style-type:disc">
<li>莫斯科继续致力于在卢甘斯克和顿涅茨克开设大使馆。</li>
<li>俄罗斯不打算对乌克兰公民施加签证限制。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>克里姆林宫：</p>
<ul style="list-style-type:disc">
<li>乌克兰在和谈期间违背了其承诺，这对谈判产生了负面影响。</li>
<li>与乌克兰的谈判节奏已经不够良好。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>德国外长：柏林没有透露其发往乌克兰的所有武器。</p>
<p>路透社援引拉脱维亚外长：欧盟正在准备新措施防止俄罗斯逃避制裁。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>路透社援引德国外长：布查发生的事情清楚地表明，只有乌克兰的胜利才能结束这场战争。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>联合国难民署：自俄罗斯入侵开始以来，已有超过500万乌克兰人离开该国。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>克里姆林宫：我们向乌克兰递交了一份格式清晰的草案文件，我们正在等待答复。球现在在基辅的球场上。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>卢甘斯克分离主义者：乌克兰军队离开了乌克兰东部卢甘斯克地区的罗伊别日内、北顿涅茨克和莱西昌斯克。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯国家杜马议长：欧洲国家应赔偿俄罗斯因制裁而遭受的经济损失。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯外交部发言人：</p>
<ul style="list-style-type:disc">
<li>乌克兰总统不是独立的，北约和美国控制着他。</li>
<li>俄罗斯对乌克兰谈判代表失去了信任。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>顿涅茨克分裂分子：我军控制了城市西南的4个新城镇，在我们控制下的城镇数量达到167个。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯国防部：</p>
<ul style="list-style-type:disc">
<li>昨晚，我们的部队瞄准了乌克兰的73个军事设施。</li>
<li>我们在乌克兰的不同地区击落了6架无人机。</li>
<li>在对新龙佐瓦和基塞尔夫卡地区的导弹袭击中，40名乌克兰士兵丧生，装备被毁。</li>
</ul>

<ul style="list-style-type:square">

</ul>
<p>马里乌波尔市长：我们希望今天用90辆公共汽车从被围困的城市疏散6000人。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
