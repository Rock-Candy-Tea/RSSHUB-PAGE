<h3>以下是这场战争进入今日以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>流亡海外的霍多尔科夫斯基呼吁西方向乌克兰提供重型武器</strong></p>
<p>克里姆林宫流亡批评人士、前石油大亨霍多尔科夫斯基敦促西方国家向乌克兰提供重型武器。</p>
<p>他对德国媒体表示，“如果乌克兰人得不到他们所要求的武器，基辅附近很快就会再次爆发战斗。”</p>
<p>霍多尔科夫斯基目前正式居住在伦敦。他认为，西方政客害怕普京。他批评他们认为自己没有与俄罗斯开战。</p>
<p>霍多尔科夫斯基表示，这是一种“非常愚蠢的态度”，并且认为从普京的角度来看，西方政客和国家已经与俄罗斯处于战争状态。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>普京警告舒尔茨和马克龙不要向基辅运送重型武器</strong></p>
<p>普京在长达80分钟的电话通话中，警告德国总理舒尔茨和法国总统马克龙不要向乌克兰提供重型武器。</p>
<p>普京在通话结束后发表的声明中表示，向乌克兰交付更强大的武器可能会进一步破坏局势稳定，并加剧人道主义危机。</p>
<p>德国政府发言人表示，舒尔茨和马克龙呼吁结束冲突。他还指出，“德国总理舒尔茨和法国总统马克龙要求立即停火，并与乌克兰总统进行直接对话，寻求通过外交途径解决冲突。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报告：西班牙向北约拉脱维亚增派导弹和军队</strong></p>
<p>根据西班牙《国家报》的报道，西班牙向北约驻拉脱维亚前线部队派遣了一组地对空导弹和近100名士兵，以加入已经驻扎在这个波罗的海国家的近500名西班牙士兵的行列。</p>
<p>《国家报》指出，西班牙国防部“计划在拉脱维亚部署一组NASAMS地对空导弹，即挪威的先进地对空导弹系统。该媒体称，北约加强了它在波罗的海地区的存在，以应对俄罗斯对乌克兰的进攻。</p>
<p>西班牙国防大臣在谈到这份报告时对西班牙电视台表示，“我们对北约的承诺是百分之百的”，“我们目前在拉脱维亚和立陶宛都部署了军队，并且还在地中海部署了我们的船只。”</p>
<figure id="attachment_285343" aria-describedby="caption-attachment-285343" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-285343" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/INTERACTIVE-NATO-in-Europe-map-updated-1.png?w=770&resize=770%2C770" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-285343" class="wp-caption-text"><strong>北约</strong><br><strong>北约在欧洲的情况</strong><br>北约成立于1949年，在成立之初共有12个成员国。如今，北约已经发展成一个拥有30个成员国的大型军事联盟。俄罗斯向来反对乌克兰加入北约，并指责北约的扩张削弱了地区的安全和稳定。 (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>马克龙、舒尔茨敦促普京释放2500名亚速钢铁厂的武装人员</strong></p>
<p>法国总统马克龙和德国总理舒尔茨要求普京释放2500名被俄罗斯俘虏的乌克兰武装人员，此前，他们藏身在马里乌波尔的亚速钢铁厂内。</p>
<p>法国总统府在3位领导人进行电话通话后表示，“法国总统和德国总理要求释放被俄罗斯军队俘虏的近2500名亚速钢铁厂内的守卫者。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>舒尔茨、马克龙要求普京与泽连斯基进行“认真的直接谈判”</strong></p>
<p>德国总理舒尔茨和法国总统马克龙，要求普京与泽连斯基进行“认真的直接谈判”。</p>
<p>德国总理办公室表示，在与俄罗斯总统长达80分钟的谈话中，两位欧盟领导人“坚持要求立即停火，并撤离俄罗斯军队”。</p>
<p>马克龙和舒尔茨敦促普京“与乌克兰总统进行认真的直接谈判，并找到通过外交途径解决当前冲突的办法”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>评论：非盟能成为乌克兰和平的中立仲裁者吗?</strong></p>
<p>在5月22日与德国总理舒尔茨在达喀尔举行的联合新闻发布会上，塞内加尔总统马基·萨勒表示，他将在未来几周以非洲联盟主席的身份访问莫斯科和基辅，并希望双方能够通过对话而实现“和平”。</p>
<p>萨勒表示，“我们不想在这场冲突上达成一致，很明显，我们想要和平”，“尽管我们谴责入侵，但我们正在努力缓和紧张局势，我们正在努力寻求停火并进行对话……这就是非洲的立场。”</p>
<p>但是非洲大陆真的在乌克兰问题上保持“中立”吗?它能以不结盟实体的立场在交战各方之间开始对话，从而真正地促进和平吗?</p>

<li><strong>更新</strong></li>
</ul>
<p><strong>英国首相约翰逊：七国集团正努力恢复粮食出口</strong></p>
<p>英国首相鲍里斯·约翰逊告诉乌克兰总统泽连斯基，国际合作伙伴正在加紧努力，以寻找恢复乌克兰粮食出口的方法，旨在避免全球粮食危机。</p>
<p>英国方面对此次通话的内容解读称，“约翰逊表示，英国将与七国集团(G7)内的伙伴合作，推动在粮食出口方面取得紧急进展。”</p>
<p>“领导人们就下一步行动达成了一致，包括要求俄罗斯放松封锁，并允许开放安全航运通道等等。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰前总统波罗申科声称被禁止出境</strong></p>
<p>乌克兰前总统波罗申科声称自己被禁止离开乌克兰，并指责政府违反了自俄罗斯入侵以来实施的所谓政治停火协议。</p>
<p>在俄罗斯对乌克兰发起军事行动之后，乌克兰议会取缔了多个亲俄政党，并允许其他政党在所谓的政治停火的协议下继续运作。这是各方达成的一种默契，旨在让所有政党搁置国内的政治分歧，以团结起来反对这场战争。</p>
<p>然而，波罗申科的办公室表示，他“被拒绝离开乌克兰的边境”，并指责政府违反了协议。</p>

<p><strong>半岛电视台记者：莱曼可被用作“发射场”</strong></p>
<p>半岛电视台记者自基辅报道称，俄罗斯在乌克兰东部和南部重新部署军队的计划似乎正在奏效，俄罗斯声称他们已经完全控制了莱曼市。</p>
<p>记者补充称，“莱曼并不是一个很大的地方。它是一个村庄，在这场军事行动之前，大约有2万人生活在那里，但它的确是一个重要的战略位置。”</p>
<p>“如果莱曼真的像俄罗斯人所说的那样处于他们的完全控制之下,那么，他们可以做的是利用莱曼作为一个平台，以便向南部和东部地区进一步推进，而这些战线上目前正在为争夺北顿涅茨克而激战。在这样的情况下，他们可能会切断超过8000名乌克兰士兵的战术撤退。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰国防部长：收到鱼叉导弹和榴弹炮</strong></p>
<p>乌克兰国防部长宣布，基辅已经开始接收来自丹麦的“鱼叉”反舰导弹和来自美国的榴弹炮，并表示这些武器将加强其对抗俄罗斯的力量。</p>
<p>他还在其Facebook页面上写道，“我们国家的海岸防御将会被鱼叉导弹加强，将由训练有素的乌克兰队伍来使用这些导弹。”</p>
<p>他还表示，“鱼叉”反舰导弹将与乌克兰的“海神”导弹一起部署，以防御包括南部港口敖德萨在内的海岸地区。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯声称进行了高超音速导弹试验</strong></p>
<p>俄罗斯军队宣布其“锆石”高超音速巡航导弹的最新成功测试。</p>
<p>俄罗斯国防部称，该导弹从巴伦支海的戈尔什科夫“海军上将”号护卫舰上发射，并“成功击中”位于1000公里以外的北极白海的目标。</p>
<p>俄罗斯国防部补充称，这次试验是正在进行的“新武器试验”的一部分。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>

<p><strong>塔斯社：一艘船只抵达马里乌波尔以将金属运往俄罗斯</strong></p>
<p>根据塔斯社的报道，自俄罗斯占领乌克兰港口马里乌波尔以来，一艘船只首次进入该港口，装载了金属并将其向东运往俄罗斯。</p>
<p>该港口发言人告诉塔斯社记者，这艘船将装载2700吨金属，然后在于下周一向东行驶160公里，以抵达俄罗斯城市顿河畔罗斯托夫。</p>
<p>这位发言人没有透露这些金属是在哪里生产的。</p>
<p>乌克兰人权监察员德尼索娃表示，这批货物相当于俄罗斯的“抢劫”。</p>
<p>“在盗窃乌克兰的粮食之后，占领者又开始从马里乌波尔出口金属产品。”</p>

<p><strong>半岛电视台记者：乌克兰的排雷工作可能需要5年时间</strong></p>
<p>半岛电视台记者从乌克兰北部城市布查报道称，俄罗斯在乌克兰留下了很多尚未爆炸的弹药和地雷，专家估计，完全清理被乌克兰政府重新占领的地区，可能需要“长达5年”的时间。</p>
<p>他还表示，“俄罗斯军队留下的东西继续造成生命和财产损失。事实上，布查的每一个国家公园和森林地区目前都是禁止进入的，因为那里可能也存在地雷的威胁。”</p>
