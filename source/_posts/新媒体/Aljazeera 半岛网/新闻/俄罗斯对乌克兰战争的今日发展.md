<h3>以下是这场战争进入今日以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰通过国际货币基金组织的账户获得第一笔资金</strong></p>
<p>乌克兰总理表示，乌克兰以优惠条件获得了价值10亿加元(约合7.7亿美元)的贷款，这是该国通过国际货币基金组织(IMF)设立的“管理账户”获得的第一笔资金。</p>
<p>乌克兰总理杰尼斯·什米加尔在推特上表示，“感谢加拿大副总理克里斯蒂娅·弗里兰为乌克兰反击侵略者所提供的支持和全面援助。”其所指的是俄罗斯。</p>
<p>国际货币基金组织在今年4月设立了这项管理账户，为捐助方以赠款和贷款的形式向乌克兰提供财政援助提供了一条安全的途径。</p>
<p>该组织发言人加里·赖斯在上周表示，德国也已承诺提供资金，其他国家也表达了类似的意象。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>丹麦指责俄罗斯军舰侵犯其领海</strong></p>
<p>丹麦军方表示，当天早些时候，一艘俄罗斯军舰两次侵犯了波罗的海伯恩霍尔姆岛北部的丹麦领海。</p>
<p>丹麦外交部长科福德抨击称，这类行动是不可接受的。这项受到指控的行动的发生，正值在伯恩霍尔姆庆祝其民主节之际，资深议员和商界人士都参加了这一活动。</p>
<p>科福德在推特上写道，“俄罗斯在丹麦的民主节期间发起了极其不负责任、粗暴和完全不可接受的挑衅。”</p>
<p>他还补充称，“欺凌手段对丹麦不起作用”，并指出俄罗斯驻这个北约成员国的大使已经因为这个问题而被召见。</p>
<p>但莫斯科方面没有立即对这些指控作出回应。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>普京称莫斯科经受住了西方的制裁</strong></p>
<p>普京表示，俄罗斯经济经受住了西方针对莫斯科的军事行动而实施制裁的“闪电战”。</p>
<p>普京在圣彼得堡国际经济论坛上发表了充满斗志的演讲，他还表示，西方对俄罗斯经济的悲观预测没有实现。</p>
<p>普京还发表了一篇谴责美国及其盟友的长篇文章，指责华盛顿将其他国家视为“殖民地”，并警告称，“在全球政治中，一切都不会再像过去那样了”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>莫斯科指责欧盟在乌克兰的成员国资格问题上“操纵”乌克兰</strong></p>
<p>俄罗斯外交部指责布鲁塞尔“操纵”乌克兰，此前，欧盟委员会建议给予乌克兰加入该27国集团的候选国资格。</p>
<p>俄罗斯新闻机构援引该国外交部发言人玛丽亚·扎哈罗娃的话报道称， “我们看到，西方社会多年来一直在操纵乌克兰以某种方式参与其一体化结构的想法。”</p>
<p>然而，她补充称，乌克兰正变得“越来越糟糕”。</p>
<p>扎哈罗娃指出，乌克兰“因为某些原因，无法获得一个光明的未来，尽管这些承诺正变得越来越甜蜜和诱人”。</p>
<figure id="attachment_291448" aria-describedby="caption-attachment-291448" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-291448" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/EU-4.png?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-291448" class="wp-caption-text">欧盟委员会主席乌苏拉·冯德莱恩在宣布决定时，身穿与乌克兰国旗颜色相同的蓝色衬衫外搭黄色夹克 (路透社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>国际专家称“指挥系统”似乎将所谓的”战争罪行”与俄罗斯领导人直接联系起来</strong></p>
<p>一名帮助乌克兰调查战争罪行的国际专家表示，所有的“指挥系统特征”都将冲突期间在当地犯下的暴行与俄罗斯总统普京本人联系起来。</p>
<p>“克里姆林宫发出的声音和言论，克里姆林宫所说的与地面上发生的事情明显相符，从地面犯罪再到普京本人，你已经看到了一系列指挥系统的所有特征”，正在协调流动司法小组以帮助乌克兰调查人员的的英国律师韦恩·约尔达什，在访问基辅附近的一个村庄时这样说道。</p>
<p>约尔达什及其他人参观了位于乌克兰首都东北约50公里(30英里)的博达尼夫卡，他们参观了一所被烧毁的学校和一所幼儿园，以及4月初俄罗斯军队从该地区撤出时变成废墟的房屋。</p>
<p>俄罗斯否认其军队在这场“特别军事行动”中犯下了战争罪，并声称该行动旨在使其邻国乌克兰“去军事化”和“去纳粹化”。而乌克兰及其盟国认为，这是一场无端的侵略战争。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧洲核研究理事会因乌克兰战争暂停与俄罗斯和白俄罗斯之间的合作</strong></p>
<p>欧洲核研究理事会表示，其决策机构计划在2024年它与俄罗斯和白俄罗斯之间的合作协议到期后不再续期，原因是乌克兰的局势。</p>
<p>该机构在一份声明中表示，其理事会在16日召开的一次会议上宣布，“将继续密切关注”局势，并“随时准备根据这个饱受战争蹂躏的国家的事态发展，采取任何进一步的决定”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰将从7月1日起取消俄罗斯人的免签旅行</strong></p>
<p>泽连斯基宣布，乌克兰将取消对俄罗斯人的免签入境，从下个月初开始，他们将需要获得签证才能进入乌克兰。</p>
<p>这位乌克兰领导人表示，“根据内阁发布的一项决定，乌克兰将从2022年7月1日起对俄罗斯公民实行签证制度。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>记者日志：在哈尔科夫目睹战争罪行</strong></p>
<p>点击<a href="https://www.aljazeera.com/news/2022/6/17/russia-ukraine-live-news-gross-violations-in-mariuopol-un-says-liveblog">此处</a>阅读半岛电视台记者对他在乌克兰第二大城市哈尔科夫所目睹的悲惨场景的记述，其中包括双方实施的明显的虐待行为。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>半岛电视台记者：如果乌克兰申请加入欧盟失败，俄罗斯可能会“更加胆大”</strong></p>
<p>半岛电视台记者从基辅发回报道称，乌克兰及其西方盟友有一种“普遍的感觉”，那就是如果乌克兰加入欧盟的候选国推荐没有得到欧盟委员会的通过，那么俄罗斯总统普京可能会“更加胆大”。</p>
<p>记者指出，“因此，这是一项积极的举措，但是在打击腐败和法治方面，欧盟对乌克兰提出了非常严格的条件。”</p>
<p>记者补充称，“乌克兰的整个法律体系将受到布鲁塞尔的审查，基本上必须遵守布鲁塞尔制定的所有标准。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基：欧盟候选国身份将帮助乌克兰击败俄罗斯</strong></p>
<p>泽连斯基表示，欧盟委员会建议给予乌克兰欧盟候选者地位的决定，将有助于基辅击败俄罗斯的入侵。</p>
<p>这位乌克兰领导人在推特上写道，“这是乌克兰加入欧盟道路上的第一步，这肯定会让我们更接近胜利。感谢冯德莱恩及每一个欧盟成员国采取这一历史性的决定。”</p>
<p>他还补充称，预计欧盟各国政府领导人将在下周于布鲁塞尔举行的峰会上批准这项提议。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>半岛电视台记者：乌克兰申请加入欧盟的“细节问题”</strong></p>
<p>半岛电视台记者从柏林发回报道称，对于欧盟委员会建议乌克兰在申请加入欧盟时获得候选国地位的举动，他评价称，“细节决定成败”。</p>
<p>这位记者指出，“如果是候选国，乌克兰人必须符合几个不同的标准，但最重要的是，他们必须证明他们拥有一个稳定的民主制度，一个正常运转的市场经济，以及适用于整个国家的法治。”</p>
<p>他还表示，“他们必须签署协议，接受欧盟立法的所有部分，包括在他们加入欧盟时或加入欧盟后成为欧元市场。”</p>
<p>记者还指出，其他一些国家“已经作为候选国而等候了很长一段时间，以正式成为欧盟成员国”。</p>
<p>他还补充称，“因此，这意味着乌克兰向前迈进了一步……但毫无疑问，前方还有很长的路要走。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>主办方：“2023欧歌赛”无法在乌克兰举行</strong></p>
<p>欧洲广播联盟宣布，由于战争的影响，2023年的欧洲歌唱大赛(欧歌赛)将无法在乌克兰举行。</p>
<p>该赛事委员会在推文中援引乌克兰公共广播公司的话称，“主办、组织和制作欧洲歌唱大赛所需的安全和运营保障无法得到满足。”</p>
<p>声明补充称，“欧洲广播联盟感谢乌克兰公共广播公司在卡卢什乐团（Kalush Orchestra）今年5月14日在都灵获胜之后的几周内，全心全意的配合并致力于探索所有的场景，并分享他们对明年的比赛无法在乌克兰举行而感到的悲伤和失望”，并指出目前正在讨论改在英国举办这一年度活动。</p>
<p>乌克兰的卡卢什管乐团在上个月赢得了今年的比赛，而英国的参赛乐队排名第二。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟委员会推荐乌克兰成为欧盟“候选国”</strong></p>
<p>欧盟委员会表示，已建议正式提名乌克兰为加入欧盟的“候选国”，此举可能为乌克兰加入欧盟开启一条长达数年的道路。</p>
<p>欧盟执行机构负责人乌苏拉·冯德莱恩表示，“是的，乌克兰作为候选国家应该受到欢迎——这是基于这样一种理解，即我们已经做了很好的工作，但还有重要的工作有待完成。”</p>
<p>与此同时，俄罗斯表示将密切关注乌克兰加入欧盟的努力，特别是考虑到欧盟内部防务合作的加强。</p>
<p>克里姆林宫发言人德米特里·佩斯科夫表示，这一问题“需要我们高度关注，因为我们都意识到，欧洲关于加强欧盟防务部门的讨论正在加剧”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>克里姆林宫：俄罗斯在乌克兰的主要目标仍然是保护顿巴斯地区</strong></p>
<p>克里姆林宫表示，其在乌克兰的“特别军事行动”的目标仍然是保护乌克兰东部顿巴斯地区的居民。</p>
<p>克里姆林宫发言人佩斯科夫在与记者通话时表示，俄罗斯在乌克兰的主要目标是“拯救”顿巴斯人，以使他们免遭基辅武装部队的“野蛮袭击”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>法国“不再通过管道接收俄罗斯天然气”</strong></p>
<p>天然气网络运营商“GRT”表示，自今年6月15日以来，法国没有通过管道从俄罗斯获得任何天然气。此前，俄罗斯能源巨头俄罗斯天然气工业股份公司在本周警告称，将会大幅削减针对欧洲的天然气供应。</p>
<p>俄罗斯天然气工业股份公司表示，北溪管道的供应减少，是维修工作的结果，但是欧盟官员认为，俄罗斯此举是为了制裁乌克兰的盟友。</p>
<p>法国近17%的天然气通过与德国连接的网络而从俄罗斯获得，德国严重依赖俄罗斯的天然气供应，并批评俄罗斯天然气工业股份公司的举动是“政治性”的。自今年年初以来，相关的天然气供应已经减少了60%，从而导致其价格飙升。</p>
<p>但是“GRT”集团表示，目前法国的供应并不存在风险，其储备达56%，而通常的6月期间为50%。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>斯洛伐克天然气进口商预计俄罗斯的天然气供应将会减少一半</strong></p>
<p>根据报道，斯洛伐克天然气进口商“SPP”首席执行官表示，俄罗斯通知其天然气供应量将会削减一半。</p>
<p>“目前，减少一半的供应量对我们来说没有影响。斯洛伐克新闻网站援引该集团负责人理查德·普罗基帕克的话报道称，“我们面临着被完全切断供应的现实风险。”</p>
<p>“SPP”并未立即就此置评。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>马克龙对将粮食运出乌克兰敖德萨港的协议持怀疑态度</strong></p>
<p>法国总统埃马克龙表示，他愿意与俄罗斯总统普京进行谈判，以达成一项协议，将粮食运出乌克兰港口敖德萨，但他认为达成协议的可能性很小。</p>
<p>马克龙在从乌克兰返回途中对BFM电视台记者表示，“我们必须与俄罗斯就粮食安全问题进行谈判，直接与联合国秘书长联系，将粮食运出敖德萨，这是合理的。”</p>
<p>“但是我对这条道路并无把握，因为几周前我已经与普京总统进行了会谈，但他不想接受联合国关于这个问题的决议。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰：从北顿涅茨克化工厂撤离是“不可能的”</strong></p>
<p>乌克兰卢甘斯克地区州长表示，由于炮火和激烈的战斗，目前不可能疏散躲藏在北顿涅茨克Azot化工厂地下掩体中的568位平民。</p>
<p>他在Telegram messenger上发帖称，有38名儿童在化工厂的掩体中避难。</p>
