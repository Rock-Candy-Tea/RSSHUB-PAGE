<p><strong>俄罗斯支持的官员：火车从乌克兰南部到达克里米亚</strong></p>
<p>据莫斯科支持的一名官员称，俄罗斯已经恢复了克里米亚半岛和乌克兰南部之间的铁路连接。</p>
<p>莫斯科任命的克里米亚领导人的顾问奥列格·克里乌奇科夫告诉俄罗斯商业咨询通讯社RBC，一列载有谷物的火车从俄罗斯占领的乌克兰南部城市梅利托波尔抵达克里米亚，这是克里米亚半岛于2014年初被俄罗斯占领以来的首次此类运输。</p>
<p>在他发表上述言论之前，俄罗斯国防部长周二表示，军方与铁路公司合作，已在乌克兰东南部恢复了约1200公里（745英里）的火车轨道，并开通道路以允许在俄罗斯与乌克兰东部顿巴斯地区，乌克兰南部被占领地区和克里米亚之间的“全面通行”。</p>
<p><em>曼苏尔·米罗瓦廖夫在基辅报道。</em></p>

<p><strong>经合组织警告乌克兰战争的“沉重”经济代价</strong></p>
<p>经济合作与发展组织警告称，世界经济将为乌克兰的战争付出“沉重的代价”，包括增长放缓、通胀加剧以及供应链可能遭受的长期损害。</p>
<p>在最新的经济展望中，该组织将今年全球经济增长预测从12月的4.5%下调至3%，并将38个成员国的通胀预测提高了一倍，接近9%。</p>

<p><strong>乌克兰和俄罗斯在前线交换阵亡士兵遗体</strong></p>
<p>乌克兰重返社会部表示，乌克兰和俄罗斯各自交出了50名阵亡士兵的遗体，其中包括在马里乌波尔亚速钢铁厂遇难的37名乌克兰士兵。</p>
<p>该部在官网上的一份声明中表示，此次交换发生在乌克兰东南部扎波罗热地区的前线，并表示，这种交换将继续下去。</p>

<li><strong>更新</strong></li>
</ul>
<p><strong>莫斯科支持的官员：俄罗斯将乌克兰小麦运往土耳其和中东</strong></p>
<p>在乌克兰被部分占领的东南部扎波罗热地区，一名莫斯科支持的官员表示，俄罗斯已开始向土耳其和中东国家运送小麦。</p>
<p>“我们通过俄罗斯运送小麦，大部分合同都是与土耳其签订的，”莫斯科在被占地区任命的军民行政部门负责人叶夫根尼·巴利茨基告诉俄罗斯24电视新闻频道。</p>
<p>他没有具体说明将向中东哪些国家供应小麦。</p>
<p>莫斯科、安卡拉或基辅没有立即对巴利茨基的言论发表评论。</p>

<p><strong>“空话”：乌克兰否认俄罗斯对粮食运输的保证</strong></p>
<p>乌克兰驳斥了俄罗斯的保证，称其是“空话”。俄罗斯表示，即如果基辅允许粮食运输通过黑海安全离开，它不会利用这种情况为自己谋利。</p>
<p>“需要军事装备来保护海岸线，并且需要海军巡逻保障黑海出口路线的安全。不能允许俄罗斯利用粮食走廊袭击乌克兰南部，”外交部发言人奥列格·尼科连科在推特上写道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>莫斯科威胁回应法国对俄罗斯一些电视频道的禁令</strong></p>
<p>俄罗斯外交部发言人表示，莫斯科将对法国封禁俄罗斯某些电视频道的决定作出回应。</p>
<p>“外交部将对此类行动做出回应，”玛丽亚·扎哈罗娃告诉记者，但没有具体说明将采取哪些措施。</p>
<p>据报道，俄罗斯周一警告美国新闻机构，除非俄罗斯记者在美国的待遇有所改善，否则他们可能会被剥夺资格认证。</p>
<figure id="attachment_288533" aria-describedby="caption-attachment-288533" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-288533" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/Russia-vs-Ukraine-The-fog-of-propaganda-and-disinformation.png?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-288533" class="wp-caption-text">俄罗斯和乌克兰：宣传的迷雾 (半岛电视台)</figcaption></figure>

<li><strong>更新</strong></li>
</ul>
<p><strong>意大利警告粮食危机可能导致“数百万人”丧生</strong></p>
<p>意大利外交部长警告说，除非俄罗斯解除对乌克兰港口的封锁，否则数百万人可能死于饥饿，目前乌克兰有数千万吨谷物储存在粮仓中。</p>
<p>“接下来的几周对于解决局势至关重要，” 路易吉·迪马约在罗马举行的粮食危机会议上说。</p>
<p>“我想明确地说，我们期待俄罗斯发出明确而具体的信号，因为阻止粮食出口意味着挟持数百万儿童、妇女和男子的生命并将他们处以死刑。”</p>
<p>乌克兰是世界上最大的小麦、玉米和葵花籽油出口国之一，但战争和俄罗斯对其港口的封锁阻止了大部分出口。许多国家的粮食供应受到危及。此外，许多港口目前分布着大量水雷。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>克里姆林宫表示必须取消制裁，俄罗斯谷物才能进入市场</strong></p>
<p>莫斯科表示，如果要向国际市场运送粮食，就必须取消对俄罗斯的制裁。</p>
<p>克里姆林宫发言人德米特里·佩斯科夫在与记者的电话中表示，关于解除制裁的“没有实质性的讨论”仍在进行。</p>
<p>佩斯科夫还表示，俄罗斯没有理由拖欠其债务，只是由于西方国家实施的一系列措施，该国难以向债券持有人支付利息。</p>
<p>他指责制裁导致该国近一半的外汇储备被冻结，“将俄罗斯推入人为的人为债务违约”。</p>
<figure id="attachment_288531" aria-describedby="caption-attachment-288531" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-288531" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/How-is-Russias-economy-performing-under-Western-sanctions.png?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-288531" class="wp-caption-text">俄罗斯处于经济制裁中 (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰情报机构称已没收普京盟友拥有的公司资产</strong></p>
<p>乌克兰国家安全局（SBU）表示，它已经没收了俄罗斯总统弗拉基米尔·普京最亲密盟友拥有的一家大型混凝土公司的资产。</p>
<p>SBU在Telegram帖子中表示，该公司是乌克兰最大的混凝土建筑材料生产商之一，其所有者是为顿巴斯地区分离主义分子提供资金的俄罗斯商人。</p>
<p>“其中一位所有者是俄罗斯总统核心圈子的一部分，是（执政党）统一俄罗斯党竞选活动的赞助商，”乌克兰情报机构补充说，但没有具体说明此人的姓名</p>
<p>该机构说，被扣押的资产已被转移到乌克兰国家机构的管理部门，该机构负责寻找、追踪和管理腐败和其他犯罪资产，是一个特殊的政府机构。</p>
<p><em>曼苏尔·米罗瓦廖夫在基辅报道。</em></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基在与朔尔茨的电话中提及俄罗斯对待战俘的问题</strong></p>
<p>乌克兰总统泽连斯基说，他在与德国总理奥拉夫·朔尔茨的电话中提出了俄罗斯遵守战俘待遇相关国际规则的问题。</p>
<p>乌克兰领导人在推特上说，两人还讨论了加强对乌克兰和全球粮食安全的国防支持。他没有进一步说明囚犯事项。</p>
<p>据报道，东南部港口城市马里乌波尔投降的1000多名乌克兰士兵被转移到俄罗斯接受调查，之后，泽连斯基发表了上述言论。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报告：扎波罗热地区将签发俄罗斯护照</strong></p>
<p>俄罗斯塔斯社援引俄罗斯任命的官员的话报道，莫斯科指定的当局将于周日开始向乌克兰部分占领的东南部扎波罗热地区的居民发放俄罗斯护照。</p>
<p>莫斯科在该地区设立的军民行政当局负责人叶夫根尼·巴利茨基表示，此举将与俄罗斯6月12日的独立日庆祝活动相吻合。</p>
<p><em>曼苏尔·米罗瓦廖夫在基辅报道。</em></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>总检察长：乌克兰向法院提起了八起战争罪案件</strong></p>
<p>乌克兰总检察长说，除了已经对俄罗斯士兵做出判决的三起审判外，乌克兰还向法院提起了八起战争罪案件。</p>
<p>伊琳娜·维涅迪克托娃在电视讲话中说，乌克兰对俄罗斯在入侵期间可能犯下的战争罪行展开了共16000多项调查。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>梅利托波尔市长指责俄罗斯军队绑架了数百名居民</strong></p>
<p>俄罗斯占领的乌克兰南部城市梅利托波尔市市长声称，500多名当地居民因支持乌克兰而被绑架。</p>
<p>伊万·费奥多罗夫在电视讲话中说：“占领者肯定希望确保每个想法不同的人都离开（被占）地区，从而不干涉他们的宣传和政治制度。”</p>
<p>莫斯科没有立即回应费奥多罗夫的言论，半岛电视台也无法独立核实他的说法。</p>
<p><em>曼苏尔·米罗瓦廖夫在基辅报道。</em></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>拉夫罗夫称乌克兰必须对港口进行排雷以允许粮食出口</strong></p>
<p>俄罗斯外交部长表示，乌克兰有责任对其港口进行排雷，以解决恢复粮食运输的问题。</p>
<p>拉夫罗夫在与土耳其外长梅夫吕特·恰武什奥卢举行了会谈，之后在安卡拉对记者发表讲话时表示，俄罗斯方面不需要采取任何行动，因为它已经做出了必要的承诺。</p>
<p>“我们每天都表示，我们准备好保证离开乌克兰港口前往（博斯普鲁斯海峡）海湾的船只安全，我们已准备好与我们的土耳其同事合作，”他说。</p>
<p>“要解决这个问题，乌克兰人唯一需要做的就是让船只离开他们的港口，无论是通过排雷还是划定安全走廊，其他都不需要。”</p>
<p>拉夫罗夫称，主要问题是乌克兰总统泽连斯基“断然拒绝”解决港口地雷的问题。他还对土耳其努力寻找解决方法的努力表示了赞赏。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>土耳其称联合国对乌克兰的粮食出口计划是合理的</strong></p>
<p>土耳其外交部长称联合国计划开辟走廊以重启乌克兰粮食出口是合理的，但表示该提议需要与各方进行更多谈判，以确保船舶安全。</p>
<p>恰武什奥卢在安卡拉举行的新闻发布会上告诉记者，他与正在土耳其首都访问的俄罗斯外长拉夫罗夫就这个问题进行了富有成果的讨论。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰总理宣布世界银行14.9亿美元的一揽子支持计划</strong></p>
<p>乌克兰总理表示，总部位于美国的世界银行已批准向乌克兰提供14.9亿美元的财政支持，以帮助支付社会工作者和公务员的工资。</p>
<p>“恢复和胜利将是民主和整个文明世界的胜利，”杰尼斯·什米加尔在推特上写道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：乌克兰军队可能不得不在北顿涅茨克撤退</strong></p>

<p>“目前，我们不会在国外销售我们的天然气和煤炭。所有国内生产都将针对我们公民的内部需求。”他还表示，部长们正在努力修复在战争中受损的火力发电厂、热电联产厂和锅炉房。</p>

<p><strong>布林肯：俄罗斯将信息用作“战争武器”</strong></p>
<p>美国国务卿安东尼·布林肯在一次关于数字民主的在线峰会上表示，俄罗斯正在将信息用作战争武器。</p>
<p>布林肯在与菲律宾记者玛丽亚·雷萨的一次谈话中说，他相信信息的民主化是积极的，但技术也允许以没有人完全预料到的方式滥用和传播错误信息。</p>
<p>“所以我们看到威权政府正在使用它。例如，我们在俄罗斯对乌克兰的侵略中看到了这一点。我们在2014年看到了这一点，那是俄罗斯最初进入乌克兰并将信息用作战争武器的时候，”他说。</p>
<p>“因此，在那个特定的例子和在这个例子中，我们实际上已经通过使用信息、真实信息来准确地扭转了这一点，唤起我们看到的他们正在准备和努力做的事情，”他补充道。</p>
