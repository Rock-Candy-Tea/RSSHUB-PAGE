<div>   
<p>乌克兰“国际文传电讯社”援引乌克兰总统泽连斯基的话报道称，由于民众对“俄罗斯军队犯下的暴行”的“普遍愤怒”，该国与莫斯科之间的谈判很有可能会暂停。与此同时，战斗集中在该国的东部地区和东南部地区——俄罗斯军队正寻求控制卢甘斯克和顿涅茨克的更多地区。</p>
<p>乌克兰总统办公室顾问米哈伊洛·波多利亚克在电台发布的声明中指出，俄罗斯总统普京与乌克兰总统泽连斯基之间的会晤并不会像土耳其所说的那样很快发生，而是需要进一步的等待。</p>
<p>另一方面，俄罗斯外交部发言人玛丽亚·扎哈罗娃宣布，俄罗斯没有义务配合国际刑事法院在乌克兰的调查。</p>
<p>实地方面，半岛电视台驻乌克兰记者报道称，鉴于俄罗斯军队继续尝试控制卢甘斯克和顿涅茨克的更多地区，俄罗斯军队和乌克兰军队之间的战斗在过去24小时内继续集中在该国东部和东南部地区。</p>
<p>顿涅茨克分离主义领导人丹尼斯·普希林表示，他的部队正与马里乌波尔市亚速钢铁厂内的乌克兰军方保持着联系，他还补充说，根据分离主义当局掌握的信息，在被困在该园区内的部队中约有400名外国雇佣军。</p>
<p><strong>以下是这场战争进入第66天以来的最新进展：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰核能机构：基辅地区111个城镇的天然气供应得到了恢复。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>法国总统府：马克龙在电话中向泽连斯基承诺，将会加强对乌克兰的军事和人道主义支持。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>基辅警察局长：在布查斯基地区发现了3具被俄罗斯士兵受到酷刑和枪杀的男子尸体。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰顿涅茨克当局：俄罗斯轰炸了顿涅茨克的一处居民区，并造成7人受伤，其中包括3名儿童。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>塔斯社援引俄罗斯外交部的话报道：西方选择了对俄罗斯发动代理人战争的战略，但是这条路线的后果非常严重。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总统：俄罗斯军队对我国人民犯下的罪行正在调查中，必须将每一个俄罗斯罪犯绳之以法并追究其责任。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>马里乌波尔市长：</strong></p>
<ul>
<li>俄军在两个月内杀死了马里乌波尔的两万多平民，并强行驱逐了超过4万人。</li>
<li>俄罗斯军队在现代历史上最为严重的一场种族灭绝罪行中摧毁了这座城市及其居民。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>泽连斯基办公室主任：</strong></p>
<ul>
<li>我们为我们的盟友制定了一项计划，以便对俄罗斯核能公司“Rosatom”实施制裁。</li>
<li>俄罗斯人不断尝试使用核勒索，而我们必须作出强烈反应。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>泽连斯基的顾问：俄罗斯外交部宣布了其在乌克兰的破坏性行动的新目标，即将世界从西方的控制之下解放出来。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰紧急事务局：俄罗斯昨晚对哈尔科夫的多个居民区及医院实施了轰炸，造成一位平民受伤、许多建筑物受损。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>美联社援引一名美国高级国防官员的话报道：俄罗斯入侵乌克兰的进展较其计划要慢得多。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>俄罗斯国防部：</strong></p>
<ul>
<li>俄罗斯防空系统在昨晚击落了18架乌克兰无人机。</li>
<li>昨晚使用高精度导弹摧毁了4个弹药库和一个燃料库。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰外交部：我们呼吁荷兰政府要求一艘已经进入阿姆斯特丹港的、运载俄罗斯石油的船只离开当地。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰国防部：自在乌克兰展开军事行动以来，已经有超过2.3万名俄罗斯士兵丧生。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>国际文传电讯社：俄罗斯国防部表示，其部队在昨晚袭击了乌克兰的389个军事设施，并且摧毁了乌克兰的35个控制中心和15个武器库。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>俄罗斯国家杜马议长：</strong></p>
<ul>
<li>停止向拒绝支付卢布的国家供应天然气的决定是正确的，有必要对那些不友好的国家采取同类措施。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>俄新社：针对俄罗斯布良斯克地区的轰炸击中了一个油站。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>俄罗斯布良斯克州州长：两场空袭击中了与乌克兰接壤的布良斯克地区一个村庄，但是没有造成任何人员伤亡。</p>

<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>俄新社：俄罗斯防空力量阻止乌克兰的飞机进入布良斯克地区。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰卢甘斯克政府：俄罗斯军队在过去24小时内摧毁了卢甘斯克地区几个城镇内的两所学校和20所房屋。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰农业部副部长：俄罗斯在其控制的乌克兰地区内夺取了150万吨粮食。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>乌克兰国防部：</strong></p>
<ul>
<li>摧毁了11辆坦克、7个火炮系统、27辆装甲车和14辆敌方车辆。</li>
<li>防空部队在顿巴斯的领空内击落了7架无人机。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>新华社援引俄罗斯外长拉夫罗夫的话报道：</strong></p>
<ul>
<li>西方以牺牲他人为代价来解决其问题的尝试注定要失败。</li>
<li>华盛顿及其盟友忽略了现代世界已经变得多极化的事实。</li>
<li>俄罗斯在乌克兰的军事行动有助于将世界从新殖民主义的压迫中解放出来。</li>
<li>华盛顿及其盟友正在走一条长期性的破坏道路，其要素之一就是北约不计后果地东扩。</li>
<li>如果美国和北约真的想要解决乌克兰危机，就必须停止向基辅提供武器。</li>
</ul>
