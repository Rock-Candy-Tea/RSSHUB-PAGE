<h3>以下是这场战争今日的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯的空袭在乌克兰哈尔科夫造成7人受伤</strong></p>
<p>据急诊医生维克托·扎巴什塔称，俄罗斯军队袭击了哈尔科夫地区洛佐瓦镇的一栋建筑，造成至少7人受伤。</p>
<p>该地区应急服务机构在即时通讯应用Telegram上表示，其团队成员正在当地灭火和清理残骸。</p>
<p>洛佐瓦镇靠近顿涅茨克地区，而当地正是俄罗斯的进攻重点所在。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>柏林将于7月向乌克兰交付首批15辆Gepard坦克</strong></p>
<p>德国国防部发言人表示，德国将于7月向乌克兰交付首批15辆Gepard坦克。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>泽连斯基网站上张贴请愿书，要求改变以俄罗斯诗人命名的乌克兰街道</strong></p>
<p>泽连斯基的网站上张贴了一份请愿书，要求将以俄罗斯著名诗人普希金命名的乌克兰街道重新命名为“史蒂文·金”大道。</p>
<p>这份请愿书已经收到了1500多个签名。如果它能在一个月内得到25000个签名，那么泽连斯基的政府将不得不考虑这项要求。</p>
<p>亚历山大·普希金是埃塞俄比亚出身的贵族，在被沙皇尼古拉一世驱逐出圣彼得堡后，于1820年访问了乌克兰东南部城市马里乌波尔。尽管普希金批评俄罗斯的独裁统治，但是一些乌克兰人仍想把他的名字从他们国家的地图上抹去。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>CEZ以欧元向俄罗斯支付天然气款项</strong></p>
<p>捷克公用事业公司CEZ的发言人表示，该公司已经用欧元支付了俄罗斯天然气的款项，但是拒绝提供任何进一步的细节。</p>
<p>“按照欧盟委员会的建议，我们已经用欧元支付了这笔款项。我们不会对细节发表评论”，该公司拒绝具体说明该公司是否同意了俄罗斯所要求的计划，其中包括通过俄罗斯天然气工业股份公司的账户而将外币付款转换为卢布。该公司仅从俄罗斯天然气工业股份公司获得一小部分的天然气。</p>
<figure id="attachment_282818" aria-describedby="caption-attachment-282818" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-282818" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/INTERACTIVE-Europes-reliance-on-Russian-gas.png?w=770&resize=770%2C769" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-282818" class="wp-caption-text"><strong>能源</strong><br><strong>俄罗斯向欧洲的天然气出口</strong><br>俄罗斯是欧洲天然气的最大供应商，这些天然气用于供暖和发电等方面。在2020年，欧盟从俄罗斯获得了占其总需求量40%的天然气 (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>加拿大对俄罗斯寡头施加更多制裁并禁止一些奢侈品贸易</strong></p>
<p>加拿大表示，将对俄罗斯寡头实施更多制裁，并禁止从俄罗斯进口和出口目标奢侈品。</p>
<p>根据一份官方声明，新措施将对包括俄罗斯寡头、他们的家庭成员以及总统普京亲信在内的14人实施限制。</p>
<p>进口禁令将针对包括酒精饮料、海鲜和非工业钻石在内的俄罗斯商品，而出口禁令将针对鞋类、奢侈服装和珠宝等等。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>舒尔茨：卡塔尔是德国未来能源战略的关键</strong></p>
<p>德国总理舒尔茨表示，卡塔尔将在德国摆脱对俄罗斯天然气的依赖、实现其天然气来源多元化的战略中发挥核心作用。</p>
<p>舒尔茨与卡塔尔埃米尔在柏林举行的联合新闻发布会上告诉记者，“能源安全问题对我们而言至关重要。德国将发展其基础设施，以便能够通过船只进口液化天然气”，他还补充称，“这将是一大进步，卡塔尔在我们的战略中扮演着重要的角色。”</p>
<p>谢赫塔米姆证实，卡塔尔计划在2024年开始向德国供应液化天然气。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称“即将”完全控制乌克兰的卢甘斯克地区</strong></p>
<p>俄罗斯国防部长绍伊古表示，莫斯科即将完全控制乌克兰东部的分裂地区卢甘斯克。</p>
<p>俄罗斯在发动军事行动前不久承认了该分离主义共和国的独立。</p>
<p>绍伊古在通过电视转播的内阁会议上表示，“卢甘斯克人民共和国的解放即将完成。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>七国集团募集198亿美元援助乌克兰</strong></p>
<p>七国集团的财政部长和央行行长表示，正在为乌克兰筹集198亿美元，并承诺会在必要时提供更多的财政支持。</p>
<p>美国、加拿大、日本、德国、法国、意大利和英国组成的七国集团的财政领导人在德国举行会谈，旨在为乌克兰提供资金。乌克兰正在努力抵御俄罗斯的军事行动，其资金正在耗尽。</p>
<p>七国集团在其公报中表示，“在2022年，我们筹集了198亿美元的预算支持——包括最近作出的95亿美元的承诺，以帮助乌克兰弥补其融资缺口，并继续确保向乌克兰人民提供基本服务。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯将于21日停止向芬兰输送天然气</strong></p>
<p>据芬兰国有天然气商Gasum称，俄罗斯天然气工业股份公司已经通知芬兰，将从本月21日上午开始将停止对该国的天然气供应。</p>
<p>该公司拒绝向俄罗斯天然气工业股份公司支付卢布，尽管俄罗斯曾要求欧洲国家这样做。</p>
<p>该公司首席执行官在一份声明中表示，“非常遗憾的是，仍然处于我们有效合同范围内的天然气供应，现在将被暂停。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>市长：马里乌波尔将开设“俄罗斯化”的学校</strong></p>
<p>一名乌克兰官员称，俄罗斯当局在被占领的马里乌波尔市强行驱赶公立学校教师，以推出基于俄语的新课程。</p>
<p>该城市的逃亡官员彼得罗·安德里·尤先科在社交媒体Telegram上表示，他们计划在今年5月底之前，在这个城市破败、人口减少的地区内开设4所学校，并在9月份让这些学校符合“俄罗斯标准”。</p>
<p>他还表示，这些学校只会教授俄罗斯文学、历史和数学。成千上万的平民仍然留在这座缺乏电力、水和天然气供应的城市内。</p>
<p>半岛电视台记者报道。</p>
<figure id="attachment_282822" aria-describedby="caption-attachment-282822" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-282822" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/ukraine-5.png?w=770&resize=770%2C769" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-282822" class="wp-caption-text"><strong>俄罗斯-乌克兰战争</strong><br><strong>马里乌波尔处于哪方势力的控制之下？</strong><br>根据研究研究所提供的消息，俄罗斯当局正设法在马里乌波尔建立对该市的有效控制。 (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官员：俄罗斯在欧洲最大的核电站附近驱散抗议人群</strong></p>
<p>据一名乌克兰官员透露，俄罗斯军队殴打了乌克兰的消防队员，这些队员们抗议俄罗斯在其占领的Enerhodar镇内绑架了他们的上司。</p>
<p>这个南部城镇坐拥欧洲最大的扎波罗热核电站，为乌克兰提供近四分之一的电力。俄罗斯人在今年3月初夺取了该地区，以确保对能源匮乏的克里米亚的电力供应。</p>
<p>消防队长维塔利·特罗安19日因拒绝与俄罗斯人合作而被强行带走。</p>
<p>该市的逃亡市长在社交媒体Telegram上表示，俄罗斯军队“使用武力”驱散了20日为反对其绑架行动而举行的集会，并且破坏了消防设备。</p>
<p>半岛电视台记者报道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官员：尼古拉耶夫地区有5人受伤</strong></p>
<p>据一名官员透露，俄罗斯炮击了乌克兰南部与俄罗斯占领的赫尔松接壤的尼古拉耶夫地区，并造成5名平民受伤。</p>
<p>该地区议员加林娜·扎马泽耶娃在Telegram上表示，“幸运的是，受害者中没有儿童。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国称乌克兰阻止了俄罗斯在顿巴斯渡河的企图</strong></p>
<p>根据英国国防部在推特上发布的公告，乌克兰军队已经成功阻止了俄罗斯在顿巴斯地区的一次渡河企图。</p>
<p>英国国防部在其最新情报通告中指出，图像显示，俄罗斯在穿越北顿涅茨克西部的一条河流时，至少损失了一个营的战术队伍的装甲机动成分，以及部署的架桥设备。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>12人在俄罗斯的袭击中丧生</strong></p>
<p>据乌克兰东部卢甘斯克地区州长海岱称，俄罗斯军队袭击了卢甘斯克地区的利西昌斯克和北顿涅茨克。</p>
<p>海岱在社交媒体Telegram上表示，袭击造成北顿涅茨克地区12人死亡，整个地区有60多座房屋被毁。</p>
<p>他还补充称，针对北顿涅茨克的攻击“没有成功——俄罗斯人遭受了人员损失，并在随后撤退”。但是他的言论无法得到独立消息来源的证实。</p>
<p>据报道，在北顿涅茨克以南30公里的吉尔斯克地区，还有一人死于俄罗斯的袭击。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官员：1.5万人藏身于被围困的北顿涅茨克的地下室内</strong></p>
<p>一名官员表示，多达1.5万名无法离开被围困的乌克兰东部城市北顿涅茨克的平民，正试图在俄罗斯军队的猛烈炮击中求生。</p>
<p>当地行政长官亚历山大·斯特雷克在电视讲话中指出，“这些人住在防弹场所内，住在公寓楼的地下室内，他们可以在那里躲起来，并找到躲避炮击的避难场所。”</p>
<p>他还表示，炮击摧毁了该地区“多达70%”的房屋和公寓楼。该地区位于反政府武装控制的卢甘斯克西北方向仅100公里处，是俄罗斯最近几周内进攻的重点。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>斯洛伐克公司将支付欧元购买俄罗斯天然气</strong></p>
<p>斯洛伐克国家天然气进口商SPP公司首席执行官在接受斯洛伐克公共电视台现场采访时表示，SPP公司已经用欧元支付了俄罗斯天然气的账单，认为将资金转换为卢布不在其控制范围内。</p>
<p>他还表示，”外汇兑换并不受公司控制，对于我们而言，兑现承诺的真正重要时刻，就是支付欧元。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：哈尔科夫有3人受伤，俄罗斯人撤退</strong></p>
<p>哈尔科夫地区州长表示，俄罗斯的炮击导致该市3人受伤。与此同时，俄罗斯地面部队继续从这座乌克兰第二大城市周边撤退。</p>
<p>他在社交媒体Telegram上表示，俄罗斯人使用多管火箭发射器袭击了哈尔科夫的多个地区及其郊区。</p>
<p>他还表示，乌克兰军队在俄罗斯边境以南几公里处的多个村庄附近与俄罗斯军队作战。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰声称从世界银行获得了美国和英国的5.3亿美元赠款</strong></p>
<p>乌克兰财政部表示，乌克兰已从世界银行设立的一个基金中，获得了约5.3亿美元的美国和英国赠款，该基金旨在俄罗斯军事行动的背景下向基辅政府捐款。</p>
<p>乌克兰财政部声明称，美国捐赠了近5000万美元，英国捐赠了2400万英镑(约合3000万美元)，这些钱现在已经进入了乌克兰的国家预算。</p>
<p>声明指出，“这些资金将直接用于提供紧急的社会、人道主义和卫生支出，以及支持国内的流离失所者。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>

<p>比斯利在联合国安理会会议上表示，43个国家内的4900万人已经处于“饥荒边缘”。</p>

<p><strong>泽连斯基：顿巴斯地区已经完全被摧毁</strong></p>
<p>乌克兰总统泽连斯基表示，俄罗斯军队已经“完全摧毁”了该国东部的顿巴斯地区，并且指责莫斯科在加强进攻的同时进行了无差别的轰炸。</p>
<p>“在顿巴斯地区，占领者们试图施加更大的压力。那里是地狱——这绝非夸张”，泽连斯基在深夜的视频讲话中这样说道。</p>
<p>“这是一场蓄意的犯罪企图，目的是杀死尽可能多的乌克兰人，摧毁尽可能多的房屋、社会设施和企业。”</p>

<p><strong>阿列克谢·纳瓦尔尼的团队敦促美国实施更多制裁</strong></p>
<p>俄罗斯反对派领袖阿列克谢·纳瓦尔尼的团队敦促美国就乌克兰战争实施新一轮制裁，以使制裁超越富裕的寡头，并将这种金融痛苦扩散到俄罗斯政府官员、中层政客和公众人物身上。</p>
<p>访问过美国国会的该组织正在与美国参议员讨论一份可能受到制裁的6000人名单，其中包括俄罗斯国防和安全官员、行政雇员、州长、议会成员，甚至还有与政府结盟的媒体业务的编辑和经理。</p>
<p>纳瓦尔尼反腐基金会执行董事弗拉基米尔·阿舒尔科夫表示，西方迄今为止的“雪崩式制裁”正在对俄罗斯产生影响。</p>
<p>阿舒尔科夫表示，“让我们继续，或者至少宣布开始下一波浪潮。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯前总统谴责“疯狂”制裁</strong></p>
<p>俄罗斯前总统梅德韦杰夫表示，如果西方国家因战争而对莫斯科实施毁灭性制裁，那么就不要指望俄罗斯会继续供应粮食。</p>
<p>“我国准备充分履行其义务。但也希望得到贸易伙伴的帮助，包括在国际平台上的帮助”，目前担任俄罗斯安全委员会副主席的梅德韦杰夫在通讯应用平台“Telegram”上这样说道。</p>
<p>他还表示，“否则，这就毫无逻辑：一方面，他们对我们实施疯狂的制裁；另一方面，他们又要求获得粮食供应。事情不会像这样发展，我们又不是白痴。”</p>
<p>就在梅德韦杰夫发表上述言论之际，包括联合国秘书长安东尼奥·古特雷斯在内的世界各国领导人对全球粮食供应表示严重关切，他们警告称，限制乌克兰港口的粮食出口可能会导致食品短缺甚至饥荒。</p>

<p><strong>华盛顿宣布向基辅额外提供1亿美元的军事援助</strong></p>

<p>拜登在一份声明中表示，这项援助包括额外的大炮、雷达和其他设备。</p>

<p>拜登在白宫与瑞典和芬兰领导人会晤后表示，瑞典和芬兰的加入将加强北约的集体安全。</p>
<p>白宫在讨论结束后的一份声明中表示，美国总统“热烈欢迎(瑞典和芬兰)申请加入北约，这将加强我们的集体安全”。</p>

<p><strong>拜登称美国的援助计划将帮助乌克兰“捍卫民主”</strong></p>
<p>美国总统乔·拜登对美国国会通过向基辅提供近400亿美元军事和人道主义援助的方案表示赞赏，并称这“向世界传递了一个明确的来自民主和共和两党的信息，即美国支持乌克兰”。</p>
<p>此前由众议院通过的这项法案，在参议院以86票赞成对11票反对的压倒性优势获得通过。</p>
<p>“这项一揽子计划将允许我们向乌克兰运送更多的武器和弹药，因为他们需要捍卫自己的民主和自由。”</p>
