<h3>以下是俄罗斯与乌克兰战争今日的最新发展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>舒尔茨：乌克兰将承受战争余波长达“100年”</strong></p>
<p>德国总理舒尔茨表示，乌克兰可能会因为与俄罗斯之间的战争而在未来“100年”内感受到战争的余波，因为城市里到处都是未爆炸的军火。他补充称，盟国将帮助乌克兰进行重建。</p>
<p>舒尔茨告诉记者，“那些生活在德国的人都知道，二战时期的炸弹仍然至今经常被发现。”</p>
<p>“乌克兰应该做好准备，以与这场战争的后果进行长达100年的斗争。这就是为什么我们还必须在重建方面付出共同努力。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>塞尔维亚被敦促对乌克兰战争采取立场：“游戏即将结束”</strong></p>
<p>塞尔维亚与俄罗斯关系友好，但是最近几周，亲政府的小报却将矛头指向普京，另一方面，西方官员则谴责贝尔格莱德对俄罗斯入侵持观望态度。</p>
<p>点击<a href="https://www.aljazeera.com/news/2022/5/11/this-game-is-up-how-vucic-balances-his-ties-amid-ukraine-war">此处</a>以了解更多内容。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报道：俄罗斯的炮击阻止了卢甘斯克的撤离进程</strong></p>
<p>根据乌克兰一家新闻频道的报道，在过去3天内，俄罗斯的炮击使得乌克兰东部卢甘斯克地区的平民无法撤离。</p>
<p>这家乌克兰媒体播放了据称是被俄罗斯袭击摧毁的房屋，以及该地区因爆炸而产生浓烟的画面。</p>
<p>报道称，由于轰炸几乎持续不断，连接卢甘斯克的吕先尚斯克(Lysychansk)和邻近的顿涅茨克地区的巴赫穆特(Bakhmut)的高速公路已经不再是撤离的安全途径。</p>
<p>半岛电视台记者无法独立证实该报道的真实性。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-279856" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/kherson-1.png?w=770&resize=770%2C586" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>地方当局：乌克兰储存硝酸铵的仓库遭到炮击</strong></p>
<p>乌克兰东部城市斯洛夫扬斯克的地方当局要求居民们尽可能减少出门，并关闭窗户，因为俄罗斯的炮击破坏了附近存放硝酸铵的仓库。</p>
<p>硝酸铵是氮肥最常用的一种来源，但是如果人体大量吸入，则会导致呼吸问题。</p>
<p>该市当局表示，“我们强调，斯洛夫扬斯克社区居民的生命没有受到直接威胁”，并补充称该仓库位于克拉马托尔斯克地区。</p>
<p>半岛电视台记者无法独立证实该报道的真实性。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>决定在即：瑞典议会将举行北约入盟辩论</p>
<p>瑞典执政的社会民主党呼吁该国议会就是否加入北约举行辩论。在俄罗斯入侵乌克兰之后，瑞典准备放弃其坚持了数十年的军事不结盟政策，并且准备作出加入北约的决定。</p>
<p>预计瑞典及其邻国芬兰都将在未来几天内申请加入北约，此前，莫斯科的攻势引发了两国对各自政策立场的彻底反思。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰称已经重新控制了大片边境地区</strong></p>
<p>一名官员称，在将莫斯科军队赶出之后，乌克兰已经重新控制了与俄罗斯和白俄罗斯接壤的约长1200公里的领土范围。</p>
<p>边境地区官员莱昂尼德·巴兰在电视讲话中表示，收复的地区有三分之二直接与俄罗斯接壤。</p>
<p>乌克兰方面表示，其军队最近在该国东北部城市哈尔科夫周围发动反攻，并解放了该地区的几个重要城镇和村庄。</p>
<p>半岛电视台记者自基辅发回报道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯在华沙反战抗议后召见波兰大使</strong></p>
<p>根据俄罗斯官方媒体报道，俄罗斯外交部召见了波兰驻俄罗斯大使，就在两天前，俄罗斯驻波兰大使被当地的反战抗议者泼红漆。</p>
<p>波兰国家通讯社援引该国外交部长兹比格涅夫·劳的话证实了俄罗斯方面的这一决定。</p>
<figure id="attachment_279743" aria-describedby="caption-attachment-279743" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-279743" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/h_57666061.jpg?w=770&resize=770%2C514" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-279743" class="wp-caption-text">俄罗斯驻波兰大使谢尔盖·安德烈耶夫9日在华沙一个纪念二战红军阵亡士兵的公墓前遭到了反战抗议者的袭击 (法国媒体)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>瑞典首相称与英国之间的安全协议可能包括军事资源</strong></p>
<p>瑞典首相玛格达莱娜·安德森表示，瑞典和英国已经同意在发生危机或袭击时相互援助，这些支持可能包括提供军事资源。</p>
<p>她在斯德哥尔摩南部的政府乡间别墅与约翰逊举行会谈后表示，“在危机时期，合作变得更加重要。”</p>
<p>“如果对方国家遭受灾难或袭击，英国和瑞典将以各种方式相互帮助。援助将应受影响国家的要求提供，其中可能包括军事资源。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基：乌克兰加入北约就能避免战争</strong></p>
<p>乌克兰总统泽连斯基表示，如果乌克兰事先成为北约成员国，就可以避免与俄罗斯的战争。</p>
<p>泽连斯基通过视频链接向法国巴黎政治大学的学生们表示，“如果乌克兰在战前是北约的一部分，就不会有这场战争”。</p>
<p>泽连斯基补充称，他希望在冲突结束之前恢复乌克兰的领土，并表示他仍然愿意与莫斯科进行对话。</p>
<p>普京一再表示，看到乌克兰可能成为北约成员国的风险，正是俄罗斯入侵乌克兰的理由。</p>
<figure id="attachment_279748" aria-describedby="caption-attachment-279748" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-279748" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/INTERACTIVE-NATO-in-Europe-map-updated.png?w=770&resize=770%2C770" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-279748" class="wp-caption-text"><strong>北约</strong><br><strong>北约在欧洲的势力范围</strong><br>北约成立于1949年，当时拥有12个成员国。此后，该联盟不断发展并拥有了30个成员国。俄罗斯反对乌克兰加入该联盟，并称北约的扩张对地区的安全构成了威胁 (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯表示正在密切关注北约在其边境地区的部署</strong></p>
<p>克里姆林宫发言人说，俄罗斯正在密切关注任何可能影响北约在其边境地区部署的事情。</p>
<p>在佩斯科夫发表上述言论之际，瑞典和芬兰预计将于本月就是否申请加入这个由美国领导的跨大西洋军事联盟而采取决定。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国与瑞典和芬兰达成新的安全协议</strong></p>
<p>英国首相鲍里斯·约翰逊表示，英国已与瑞典和芬兰达成新协议，以加强欧洲安全，并承诺会在这两个国家的武装力量受到攻击时提供支持。</p>
<p>“我们坚定不移地、毫不含糊地支持瑞典和芬兰，签署这些安全声明是我们两国之间永久承诺的象征”，英国首相在访问瑞典和芬兰期间发表的一份声明中这样说道。</p>
<p>英国政府在一份声明中表示，约翰逊将在访问期间“阐明英国在两国军队面临危机或受到攻击时支持两国军队的意愿”。</p>
<figure id="attachment_279752" aria-describedby="caption-attachment-279752" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-279752" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/2022-05-11T114613Z_1974703230_RC2Z4U9JAVTE_RTRMADP_3_UKRAINE-CRISIS-SWEDEN-BRITAIN.jpg?w=770&resize=770%2C514" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-279752" class="wp-caption-text">约翰逊表示，英国对瑞典和芬兰的支持是“坚定的” (路透社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>国际劳工组织：战争重创经济导致近500万个工作岗位流失</strong></p>
<p>国际劳工组织(ILO)表示，自今年2月底俄罗斯开始对乌克兰发起军事行动以来，乌克兰已经失去了大约480万个工作岗位，这场冲突导致企业关闭，并且抑制了出口，迫使数百万人逃离该国。</p>
<p>国际劳工组织在一项研究中表示，在莫斯科发动攻势之前，乌克兰的失业人数约占劳动力的30%，如果敌对行动继续下去，失业人数可能攀升至700万。</p>
<p>研究补充称，一旦停火，将会迅速恢复340万个工作岗位。</p>
<p>国际劳工组织表示，这场战争还可能推高拥有数百万难民的邻国的失业率，并打击中亚经济体，因为这些国家在俄罗斯境内的移民劳工将会失去工作并返回。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>克里姆林宫称是否加入俄罗斯将取决于赫尔松居民</strong></p>
<p>克里姆林宫发言人佩斯科夫表示，在被俄罗斯占领的乌克兰南部赫尔松地区，将由当地居民决定是否加入俄罗斯。</p>
<p>早些时候，俄罗斯塔斯社援引受莫斯科控制的赫尔松地区的一名官员的话称，当地计划要求普京将其并入俄罗斯。</p>
<p>但是佩斯科夫告诉记者，任何此类决定都必须具有明确的法律基础，“克里米亚的情况就是如此”。</p>
<p>莫斯科在2014年初占领了克里米亚，一个月后在这个黑海半岛组织了一场公投，决定它是否应该被俄罗斯吞并。这次投票以压倒性多数支持吞并进程，但却被乌克兰和西方国家视为非法行径。</p>
<figure id="attachment_279561" aria-describedby="caption-attachment-279561" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-279561" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/kherson-1.jpg?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-279561" class="wp-caption-text">俄罗斯已经在赫尔松地区引入了卢布，以取代乌克兰的货币格里夫纳。图为该地区居民为抗议莫斯科的占领而举行示威(美联社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：哈尔科夫在乌克兰反攻后“相对平静”</strong></p>
<p>乌克兰东北部哈尔科夫地区的州长称，在乌克兰军队报告在反攻行动中从俄罗斯军队手中夺回了几个城镇和村庄之后，该地区目前“相对平静”。</p>
<p>但是该州长警告称，在莫斯科继续进攻之际，“绝不能失去警惕”，他还警告公民不要回到“近期解放的居住区”。</p>
<p>他还指出，“敌人在所有地方都埋下了地雷，包括学校、幼儿园和私人住宅。”</p>
<p>半岛电视台记者自基辅发回报道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>联合国秘书长并不认为乌克兰会很快进行和平谈判</strong></p>
<p>联合国秘书长安东尼奥·古特雷斯表示，他认为短期之内不会就结束战争举行任何和平谈判，但他补充称，这类谈判迟早会举行。</p>
<p>古特雷斯在维也纳与奥地利总统亚历山大·范德贝伦共同举行的新闻发布会上指出，“这场战争不会永远持续下去。总有一天会进行和平谈判。”</p>
<p>“我认为短期内不会出现这种情况。但我可以说，我们绝不放弃。”</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报道:被占领的乌克兰赫尔松的亲莫斯科领导人寻求加入俄罗斯</strong></p>
<p>塔斯社援引处于莫斯科控制下的赫尔松地区一名当局官员的话报道称，该地区计划要求普京在今年年底前将其并入俄罗斯。</p>
<p>塔斯社援引该地区军事和民政部门副主管基里尔·斯特雷穆索夫在新闻发布会上的话报道称，“将会提出要求让赫尔松成为俄罗斯联邦的一个正式组成部分。”</p>
<p>他还表示，“这将是一项基于赫尔松地区领导人对俄罗斯总统普京的呼吁的单一法令。”</p>
<p>俄罗斯在今年4月表示，已经完全控制了赫尔松地区。该地区是位于俄罗斯在2014年从乌克兰手中夺取的克里米亚半岛和受俄罗斯支持的乌克兰东部分裂地区之间的陆上通道的一部分，具有重要的战略意义。</p>
<p>乌克兰总统泽连斯基在4月23日表示，如果俄罗斯利用“伪公投”作为吞并被占领的赫尔松和扎波罗热的领土的幌子，那么基辅和莫斯科之间的谈判就将面临风险。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>

<p><strong>乌克兰国防部：乌克兰在哈尔科夫附近击退俄罗斯军队</strong></p>
<p>乌克兰军队从俄罗斯军队手中夺回了哈尔科夫北部和东北部的一些村庄，这一反攻可能标志着战争势头的转变，并且危及俄罗斯的主要进展。</p>
<p>乌克兰军队的主要新闻官员特蒂安娜·阿帕钦科在该地区报道称，乌克兰军队最近几天收复了位于乌克兰第二大城市北部的切尔卡斯基·蒂什基(Cherkaski Tyshky)、鲁斯基·蒂什基(Ruski Tyshki)、博尔什霍瓦(Borshchova)和斯洛博赞斯克(slobozhanke)等地点。</p>
<p>乌克兰国防部顾问尤瑞·萨克斯（Yuriy Saks）还表示，乌克兰正在将俄罗斯军队赶出位于该国东北部的哈尔科夫，而自这场战争开始以来，哈尔科夫一直处于无休止的轰炸之下。</p>
<p>萨克斯告诉路透社记者，“乌克兰武装部队在哈尔科夫周围的军事行动，特别是哈尔科夫北部和东北部的军事行动，是成功的行动。”</p>

<p><strong>乌克兰独立后的第一任总统列昂尼德·克拉夫丘克去世</strong></p>
<p>乌克兰官员称，在苏联解体期间领导乌克兰独立并出任该国首任总统的列昂尼德·克拉夫丘克去世，享年88岁。</p>
<p>泽连斯基办公室负责人安德里·叶尔马克在社交平台“Telegram”上证实了克拉夫丘克的去世的消息。克拉夫丘克健康状况不佳，并在去年接受了心脏手术。</p>
<p>在1991年至1994年担任乌克兰总统之前，克拉夫丘克在苏联衰落时期领导了乌克兰共产党。</p>
<p>他是1991年宣布乌克兰从苏联独立的推动力量，同年晚些时候，他与俄罗斯和白俄罗斯领导人一起签署了一项协议，正式宣布了苏联的解体。</p>

<p><strong>意大利收养了几十名乌克兰孤儿</strong></p>

<p>这场行动是由教皇约翰二十三世协会，以及意大利驻乌克兰和波兰的外交官员组织的。</p>

<p><strong>佩洛西：美国众议院将就400亿美元的乌克兰援助方案举行投票</strong></p>
<p>美国众议院议长南希·佩洛西表示，美国众议院将于10日晚上就向乌克兰提供400亿美元军事和人道主义援助的一揽子计划举行投票。</p>
<p>该法案预计将在未来几天内在众议院和参议院获得通过。</p>
<p>美国总统乔·拜登曾要求国会批准对乌克兰的价值330亿美元的额外援助，并警告称，此前批准的资金即将用完，但美国议员们决定将援助总额增至398亿美元。</p>
<p>佩洛西在写给众议院议员的一封信中指出，“这一方案建立在国会已经获得的强有力支持的基础上，帮助乌克兰，不仅仅是捍卫乌克兰这个国家，而且对全世界的民主都至关重要。”</p>
