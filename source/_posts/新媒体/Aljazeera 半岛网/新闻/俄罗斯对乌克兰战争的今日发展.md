<li>地方当局说，在俄罗斯军队连夜炮击多个地点时，哈尔科夫东北地区至少有3人丧生，30多人受伤。</li>
<li>俄罗斯国防部发言人说，基辅附近的一家弹药厂被摧毁。</li>
<li>俄罗斯表示，其军队已经“彻底清理”关键城市马里乌波尔，只有少数乌克兰军队留在一家钢铁厂内。</li>
<li>它补充说，如果马里乌波尔的乌克兰军队从莫斯科时间早上6点（格林威治标准时间3:00）开始放下武器，他们将得以幸免。</li>
<li>乌克兰总统泽连斯基威胁说，如果马里乌波尔的乌克兰武装分子被打死，他将退出与俄罗斯的和平谈判。</li>
<li>他还称马里乌波尔的局势“不人道”，并敦促盟国为乌克兰军队提供更多武器。</li>
</ul>

<p><strong>以下是这场战争进入第53天以来的最新进展：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报道：打击在乌克兰东部城市哈尔科夫造成5人死亡</strong></p>
<p>一名地区卫生官员告诉法新社，哈尔科夫遭受的一系列打击造成至少5人死亡、13人受伤。</p>
<p>哈尔科夫地区卫生部门负责人马克西姆·哈斯托夫证实了在一系列袭击事件发生后的死亡人数。法新社记者在现场表示，这些袭击引发了整个城市的大火，并烧毁了被击中建筑物的屋顶。</p>
<p>乌克兰公共广播公司援引当地卫生当局的话报道了同样的伤亡情况，并补充说救援工作仍在进行中。</p>
<p><strong>人们正在逃往那里？</strong></p>
<figure id="attachment_273223" aria-describedby="caption-attachment-273223" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-273223" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/31.png?w=770&resize=770%2C770" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-273223" class="wp-caption-text"><strong>俄乌战争</strong><br><strong>人们正在逃往哪里？</strong><br>美国称约480万人，大多数是妇女和儿童，自俄罗斯入侵以来逃离了乌克兰。大部分人在周边国家寻求庇护 (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称北约在北极地区增加军事活动令人担忧</strong></p>
<p>据报道，俄罗斯无任所大使尼古拉·科尔丘诺夫表示，俄罗斯对北约部队在北极的活动增加感到担忧，并认为存在“意外事件”的风险。</p>
<p>“最近北约在北极活动的增加令人担忧。该联盟的另一次大规模军事演习最近在挪威北部举行。在我们看来，这无助于该地区的安全，”科尔丘诺夫说。</p>
<p>这位外交官表示，此类活动会增加“意外事件”的风险，除了安全风险外，还会对北极生态系统造成严重破坏。他没有具体说明指的是哪种类型的事件。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>“战争中的复活节”：教皇敦促停止敌对行动</strong></p>
<p>教皇方济各呼吁世界领导人在“战争中的复活节”期间倾听和平的呼声，包括来自饱受冲突蹂躏的乌克兰的呼声。</p>
<p>“愿饱受战争蹂躏的乌克兰获得和平，它被拖入的残酷和毫无意义的战争，被暴力和破坏如此痛苦地考验着，”教皇在梵蒂冈圣彼得广场举行的传统复活节星期天Urbi et Orbi演讲中说。</p>
<p>“我们的眼睛也对这次战争中的复活节表示怀疑，”他说。“愿各国领导人听到人们对和平的呼吁。”</p>
<figure id="attachment_273196" aria-describedby="caption-attachment-273196" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-273196" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/25.png?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-273196" class="wp-caption-text">在梵蒂冈庆祝复活节弥撒后，教皇弗朗西斯在圣彼得广场的教皇专车上向信徒致意时，波兰和乌克兰国旗在人群中飘扬 (路透)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称已从马里乌波尔清真寺“释放人质”</strong></p>
<p>俄罗斯表示，已经释放了一些被关押在马里乌波尔清真寺的人质。</p>
<p>俄罗斯国防部长伊戈尔·科纳申科夫将军表示，此次行动是应土耳其总统雷杰普·塔伊普·埃尔多安的要求进行的。他补充说，“29名武装分子，包括外国战士”被击毙。</p>
<p>3月初的报道表明，一些在清真寺内的土耳其公民不是俘虏，而是为了躲避俄罗斯的炮击。</p>
<p>科纳申科夫补充说，“人质”来自独联体国家。独联体国家包括阿塞拜疆、亚美尼亚、白俄罗斯、哈萨克斯坦、吉尔吉斯斯坦、摩尔多瓦、俄罗斯、塔吉克斯坦、土库曼斯坦、乌兹别克斯坦和乌克兰。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官方：乌克兰要求G7提供500亿美元以弥补预算赤字</strong></p>
<p>总统的经济顾问奥列格·乌斯坚科表示，乌克兰已向G7国家寻求500亿美元的财政支持，并正在考虑发行零息债券，以帮助其弥补未来六个月与战争相关的预算赤字。</p>
<p>乌斯坚科在国家电视台发表讲话说，这些选择正在积极讨论中。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>从东部撤离平民的行动停止</strong></p>
<p>乌克兰副总理伊琳娜·韦列舒克表示，由于乌克兰和俄罗斯军队未能承诺停火，今天从该国东部撤离平民的人道主义走廊无法开放。</p>
<p>“截至4月17日今天早上，我们未能与占领者就撤离路线上的停火达成一致，”她在网上发帖说。 “我们正在努力让人道主义走廊尽快重回正轨，”她补充说。</p>
<p>韦列舒克敦促开放走廊，让南部城市马里乌波尔的平民，特别是妇女、儿童和受伤士兵能够逃离暴力。</p>
<figure id="attachment_273124" aria-describedby="caption-attachment-273124" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-273124" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/19.png?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-273124" class="wp-caption-text">俄乌战争：泽连斯基称马里乌波尔的局势“不人道” (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟将拨款5400万美元用于人道主义援助</strong></p>
<p>欧盟正在分配5000万欧元（5400万美元）的人道主义援助，以支持乌克兰受俄罗斯入侵影响的人们。</p>
<p>欧盟委员会发表的一份声明称，大部分资金将用于乌克兰境内的项目，而500万资金将用于摩尔多瓦。</p>
<p>这笔钱“将帮助援助难以到达的地区的人们，他们无法获得医疗保健、水和电，以及那些被迫逃离并将一切抛下的人，”危机管理专员珍妮兹·莱纳西奇说。 “乌克兰，我们与你同在，”她补充道。</p>
<p>最新的资金加入了委员会为应对战争而分配的约1.546亿美元。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>地方当局：3人在哈尔科夫遇难</strong></p>
<p>地区州长奥列格·西涅古波夫在Telegram频道上说，俄罗斯军队连夜炮击哈尔科夫东北部的多个地点，造成至少3人死亡。</p>
<p>西涅古波夫说，袭击发生在萨尔蒂夫卡北部、皮亚季哈特基和阿列克西伊夫卡等地区，并补充说30多人受伤，包括4名儿童。</p>
<p>“我呼吁哈尔科夫人民：不要不必要地走上街头，”他说。</p>
<figure id="attachment_273126" aria-describedby="caption-attachment-273126" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-273126" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/20.png?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-273126" class="wp-caption-text">俄罗斯炮击哈尔科夫后，一栋被毁建筑物外被烧毁的汽车 (路透)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>半岛电视台记者：“警报声一次又一次地响起”</strong></p>
<p>半岛电视台的拉塞尔·塞勒达尔说，针对基辅及其周边军事基础设施的袭击一直在加剧，人们担心未来几天袭击可能会进一步增加。</p>
<p>“夜间警报声在城市周围一次又一次地响起，不仅在基辅，而且在利沃夫和其他一些地方，” 塞勒达尔从乌克兰首都报道说。</p>
<p>塞勒达尔指出，在过去两天里，俄罗斯袭击了两家军工厂，一家负责修理坦克，另一家负责修理和生产导弹。他补充说，总体而言，有1人丧生，4人受伤。</p>
<p>“这里的担忧是，如果战争升级，首都基辅将目睹更多俄罗斯远程、高精度导弹袭击，”他补充说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>导弹袭击了布罗瓦里的基础设施</strong></p>
<p>一枚导弹击中首都基辅东郊的布罗瓦里，破坏了基础设施，市长伊戈尔·萨波日科在社交媒体上表示。</p>
<p>格林威治标准时间2:30左右，布罗瓦里在Telegram频道上发布了一条“空袭预警！”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰军方：俄罗斯军队“供应方面存在重大问题”</strong></p>
<p>乌克兰军方表示，俄罗斯军队在供应方面存在严重问题，部署在战线上的士兵的不满情绪越来越高。</p>
<p>乌克兰武装部队总参谋部代表亚历山大·斯普顿说：“俄罗斯军人不断抱怨缺乏轮换、设备不断损坏、燃料供应质量和食品质量糟糕。”</p>
<p>斯普顿在每日简报会上说，俄罗斯士兵继续对被占领土平民进行抢劫和暴力行动。</p>
<p>“他们这些行为得到了军事指挥的鼓励，”他补充说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：俄罗斯继续向乌克兰东部重新部署作战装备</strong></p>
<p>英国国防部表示，俄罗斯军队继续从白俄罗斯向乌克兰东部重新部署战斗和支援设备。</p>
<p>它在最新的情报更新中说，目标地点包括靠近哈尔科夫和北顿涅茨克的地方。</p>
<p>“虽然俄罗斯的行动重心已经转移到乌克兰东部，但俄罗斯的最终目标还是一样。它致力于迫使乌克兰放弃其欧洲-大西洋定位并维护自己的地区主导地位，” 英国国防部补充说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>意大利禁止俄罗斯船只驶入港口</strong></p>
<p>作为本月早些时候宣布的欧盟扩大制裁的一部分，意大利从周日开始禁止所有俄罗斯船只进入其港口。</p>
<p>根据发送给港口当局的通知，已经在意大利港口的船只必须“在完成商业活动后”立即离开。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>更新：基辅听到的爆炸声</strong></p>
<p>据当地媒体报道，周日凌晨，乌克兰首都基辅听到爆炸声。</p>
<p>一夜之间，几乎整个乌克兰都响起了空袭警报。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官方：在苏梅村发现的化学武器遗骸</strong></p>
<p>乌克兰北部苏梅州特罗斯蒂亚涅茨市市长说，当局在一个以前被俄罗斯军队占领的村庄里发现了沙林毒气的残骸。</p>
<p>“我们在比尔卡村发现了化学武器的残骸——沙林和其他物质。我们发现了安瓿。乌克兰安全局目前正在为此努力。占领者可能想用这种化学物质袭击基辅、波尔塔瓦或其他城市，”尤里·博瓦告诉乌克兰电台。</p>
<p>莫斯科没有立即对最新指控发表评论，半岛电视台也无法独立核实该报道。</p>
<p>博瓦还说，俄罗斯军队在占领特罗斯蒂亚涅茨期间对墓地、行政大楼、森林和公园进行了地雷攻击。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰首富承诺重建被围困的马里乌波尔</strong></p>
<p>乌克兰首富承诺帮助重建被围困的马里乌波尔市，称这是一个靠近他心脏的地方，他在这里拥有两家大型钢铁厂，并表示它们将再次有能力在全球范围内竞争。</p>
<p>里纳特·艾哈迈托夫见证了他的商业帝国在乌克兰东部八年的战斗中支离破碎，但他仍坚持反抗，并坚信他所谓的“我们勇敢的士兵”将保卫亚速海城市；目前，这座城市在七周的轰炸中沦为一片荒地。</p>
<p>“马里乌波尔是一场全球悲剧，也是全球英雄主义的典范。对我来说，马里乌波尔曾经并将永远是一座乌克兰城市，”艾哈迈托夫在对路透社问题的书面答复中说。</p>
<p>“我相信我们勇敢的士兵会保卫这座城市，我理解这对他们来说是多么艰难和困苦，”他说。</p>
<p>“我的目​​标是回到乌克兰的马里乌波尔并实施我们的（新生产）计划，以便马里乌波尔生产的钢铁能够像以前一样在全球市场上竞争。”</p>

<p><strong>对哈尔科夫的新袭击造成1死2伤</strong></p>
<p>据该市市长称，俄罗斯军队周六晚上再次炮击了乌克兰的哈尔科夫市，造成1人死亡、2人受伤。</p>
<p>“哈尔科夫的一个地区刚刚遭到炮火袭击。 1人死亡，2人受伤。 目前，他们在医院，我们的卫生工作者正在为他们的生命而战。 俄罗斯入侵者发动炮击是为了制造混乱，恐吓人民，”当地媒体援引伊戈尔·捷列霍夫的话说。</p>

<p><strong>网红主厨的厨房在哈尔科夫被炸毁</strong></p>
<p>俄罗斯周六对乌克兰周边城市的轰炸包括哈尔科夫，当地发生的爆炸摧毁了一个社区厨房。</p>
<p>厨房由名厨何塞·安德烈斯经营的“世界中央厨房”（World Central Kitchen）设立，旨在为灾区和战区建立供餐系统。安德烈斯在推特上说，该非政府组织的工作人员受到了震动，但是安全的。</p>
<p>该组织表示，目前已覆盖全国30个城市，每天提供近30万份餐食。</p>
<p>安德烈斯说，哈尔科夫的袭击表明“在一场毫无意义的战争中提供食物是一种勇气、韧性和抵抗的行为”，他团队的厨师将继续为乌克兰做饭。</p>
