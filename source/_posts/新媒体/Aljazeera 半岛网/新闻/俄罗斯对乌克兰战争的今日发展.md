<li><strong>更新</strong></li>
</ul>
<p><strong>奥地利呼吁乌克兰成为欧盟“中间”成员</strong></p>
<p>奥地利总理卡尔·内哈默呼吁乌克兰和摩尔多瓦等国家在合作和成为欧盟正式成员之间建立一个中间阶段。</p>
<p>内哈默说，所谓的“准备空间”将允许各国达到欧盟的标准，​​类似于欧洲经济区（EEA）或欧洲自由贸易联盟（EFTA）的情况。</p>
<p>内哈默尔在总理办公室发表的一份声明中说：“我们为了同一个目标而团结在一起，我们都想要一个强大、独立和经济成功的乌克兰。”</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报告：德国每年因俄罗斯制裁而面临50亿欧元的损失</strong></p>
<p>《周日世界报》（Welt am Sonntag）援引行业代表的话说，俄罗斯对俄罗斯天然气工业股份公司Gazprom的德国分公司Gazprom Germania及其子公司实施的制裁，可能使德国纳税人和天然气用户每年额外支付50亿欧元（54亿美元）来使用替代天然气。</p>
<p>5月，由于俄罗斯入侵乌克兰，柏林将Gazprom Germania置于受托管理之下后，俄罗斯决定停止向其提供天然气。</p>
<p>从那时起，作为受托人的德国联邦能源监管机构不得不在市场上购买替代天然气，以履行与德国市政公用事业公司和地区供应商的供应合同。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>莫斯科导弹袭击基辅是“恐怖主义行为”</strong></p>
<p>乌克兰总统顾问米哈伊洛·波多利亚克称莫斯科对基辅的导弹袭击是“恐怖主义行为”，并要求对俄罗斯实施新的制裁。</p>
<p>波多利亚克还呼吁向乌克兰提供重型武器，以帮助其保卫领土免受俄罗斯的袭击。</p>
<p>“今天对基辅的导弹袭击只有一个目标；杀死尽可能多的乌克兰人，”他在推特上写道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯在基辅“摧毁”欧洲国家派出的坦克</strong></p>
<p>俄罗斯国防部表示，俄罗斯的空袭在乌克兰基辅郊区摧毁了欧洲国家提供的坦克和其他装甲车辆。</p>
<p>该部发表声明前，乌克兰首都周日早些时候发生多起爆炸事件。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>普京：如果向乌克兰提供射程更远的导弹，俄罗斯将加大打击力度</strong></p>
<p>据塔斯社报道，俄罗斯总统弗拉基米尔·普京警告西方，如果美国开始向乌克兰提供远程导弹，俄罗斯将打击新目标。</p>
<p>如果提供这种导弹，“我们将打击那些我们尚未打击的目标，”普京在接受俄罗斯国家电视台“俄罗斯1”频道采访时说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>法国部长：正在与阿联酋谈判以取代俄罗斯的石油供应</strong></p>
<p>法国财政部长布鲁诺·勒梅尔表示，巴黎正在与阿拉伯联合酋长国进行谈判，以取代将在欧盟禁令实施后停止的俄罗斯石油采购。</p>
<p>“正在与阿拉伯联合酋长国进行讨论。我们必须找到俄罗斯石油的替代品，”勒梅尔告诉欧洲1台。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰电力运营商：俄罗斯导弹以“极其低的距离”飞越核电站</strong></p>
<p>乌克兰国有核电运营商Energoatom表示，一枚俄罗斯巡航导弹以“极其低的距离”从一座大型核电站上空飞过。</p>
<p>“这很可能是朝基辅方向发射的导弹，”Pivdennoukrainska核电站（也称为南乌克兰核电站）的运营商在Telegram消息应用程序上的一篇帖子中说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>导弹袭击了基辅的“主要火车枢纽”</strong></p>
<p>半岛电视台的泽因·巴斯拉维从基辅报道说，俄罗斯的导弹击中了一个主要的火车枢纽，一个铁路线维修中心。</p>
<p>“导弹袭击了他们所说的维修中心，但是过去当俄罗斯导弹袭击这样的地区时，无论是在基辅还是在该国其他地方，他们都说俄罗斯的目标是他们认为的军事目标，”巴斯拉维说。</p>
<p>“未经证实的当地报道表明，目标地点可能有一个军火库。总参谋部说没有人员伤亡，但有一人受伤并被送往医院，”他补充说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：北顿涅茨克在乌克兰和俄罗斯军队之间分裂成两半</strong></p>
<p>北顿涅茨克所在的卢甘斯克地区州长谢尔盖·海代说，乌克兰和俄罗斯军队对这座城市的控制权一分为二。</p>
<p>“这是一个困难的局面，俄罗斯人控制了该市70%的地区，但在过去的两天里，他们被推回去了，”海代在乌克兰电视台上说。</p>
<p>“这座城市现在或多或少地被分成了两半。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：乌克兰在北顿涅茨克发起反击</strong></p>
<p>英国国防部表示，乌克兰军队在乌克兰东部有争议的城市北顿涅茨克进行了反击。</p>
<p>该部在推文中表示，此举可能会削弱俄罗斯军队通过集中作战部队和火力获得的作战势头。</p>
<p>“使用代理步兵部队进行城市清理行动是俄罗斯先前在叙利亚观察到的策略，”英国国防部补充说，这种做法可能表明俄罗斯希望限制俄方正规部队的伤亡。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>市长：一人在基辅爆炸中受伤</strong></p>
<p>基辅市长表示，在首都发生多起爆炸事件后，至少有一人住院。</p>
<p>“目前没有导弹袭击基础设施导致的死亡情况，” 克利钦科说。 “基础设施仍在受影响的地区工作。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报告：基辅听到巨大爆炸声</strong></p>
<p>据路透社报道，乌克兰首都基辅听到了强烈的爆炸声。</p>
<p>该机构援引一名目击者的话说，爆炸后在该市看到烟雾。</p>
<p>报告补充说，早些时候，包括基辅地区在内的乌克兰大部分地区响起了空袭警报。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>美国国防部长将在布鲁塞尔主持乌克兰援助会议</strong></p>
<p>美国国防部长劳埃德·奥斯汀说，他将在下周主持一个联络小组会议，该联络小组旨在协调援助，帮助乌克兰抵御俄罗斯的袭击。</p>
<p>据乌克兰新闻网站《基辅独立报》报道，乌克兰国防联络小组会议将于6月15日在布鲁塞尔举行。</p>
<p>劳埃德在推文中表示，“在俄罗斯无端入侵之后，来自世界各地的国防领导人将齐聚一堂，讨论乌克兰目前的危机”。</p>
<p>在布鲁塞尔期间，国防部长还将出席北约会议，讨论瑞典和芬兰加入该安全联盟的申请。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>美国特使访问博罗江卡，誓言追究责任</strong></p>
<p>美国驻乌克兰特使布里奇特·布林克访问了基辅西北部的小镇博罗江卡，俄罗斯军队被指控犯有战争罪。</p>
<p>“博罗江卡遭受了可怕的痛苦，”布林克在周六晚些时候的一条推文中说。</p>
<p>“目睹在俄罗斯残酷战争中发生的暴行，包括在自己家中遇难的家庭，只会更加坚定我的决心，我们会竭尽全力追究这些可怕罪行制造者的责任。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>德国因俄罗斯天然气制裁而面临每年50亿欧元的损失</strong></p>
<p>《周日世界报》（Welt am Sonntag）援引行业代表的话说，俄罗斯对俄罗斯天然气工业股份公司Gazprom的德国分公司Gazprom Germania及其子公司实施的制裁，可能使德国纳税人和天然气用户每年额外支付50亿欧元（54亿美元）来使用替代天然气。</p>
<p>由于俄罗斯入侵乌克兰，柏林将Gazprom Germania置于受托人管理之下。5月，俄罗斯决定停止向分公司供应。从那时起，作为受托人的德国联邦能源监管机构不得不在市场上购买替代天然气，以履行与德国市政公用事业公司和地区供应商的供应合同。</p>
<p>《周日世界报》报道说，经济部长罗伯特·哈贝克估计每天需要额外的1000万立方米天然气，目前每年将花费约35亿欧元，进一步的成本来自哈贝克周三订购的填充Rehden储存设施的天然气。</p>
<p>该报还表示，额外成本将从10月起以天然气税的形式转嫁给能源供应商和终端客户。</p>
<p>《周日世界报》说，经济部拒绝就所产生的成本规模发表评论，该部没有人立即回复电子邮件的置评请求。</p>

<p><strong>乌克兰计划从8月起重启足球联赛</strong></p>
<p>乌克兰足协主席表示，尽管受到俄罗斯的攻击，基辅仍计划在8月恢复该国的竞技足球。</p>
<p>安德烈·帕维尔科告诉美联社通讯社，乌克兰总统已经批准了这一决定。</p>
<p>帕维尔科在加的夫说：“我与我们的总统弗拉基米尔·泽连斯基谈论了足球分散注意力的重要性。”</p>
<p>“我们谈到足球具有的强大力量能够帮助人们思考未来，因为当前，当然，人们的心态并不好。他们处于最糟糕的状态。我们谈论了足球如何帮助我们思考未来。”</p>
<p>这是一个球员们能够再次在乌克兰上场，提供令人振奋的运动快乐的未来。</p>
<p>“所以我们与总统做出决定，我们将在8月恢复乌克兰联赛，”帕维尔科通过翻译说。 “在乌克兰，我们将在各个层面上进行比赛，包括超级联赛，甲级和以及职业联赛，以及女子联赛。</p>
<p>“在各个层面上，我们将从8月开始。该决定是与乌克兰总统一起做出的。”</p>

<p><strong>泽连斯基说，北顿涅茨克的巷战仍在继续</strong></p>
<p>乌克兰总统称北顿涅茨克的局势“极其困难”，东部城市的巷战仍在继续。</p>
<p>泽连斯基在晚间讲话中说：“俄罗斯军队再次向苏梅边境、尼古拉耶夫、扎波罗热的城市和人民以及哈尔科夫地区开火。”</p>
<p>“北顿涅茨克的巷战仍在继续，局势仍然非常困难。在利西昌斯克、马林卡、库拉霍沃等顿巴斯的其他城市和社区，情况也很困难。不断的空袭、炮击和火箭袭击。到今天早上，俄罗斯对乌克兰使用的各种导弹已经达到2503枚。”</p>

<p><strong>泽连斯基抨击俄罗斯炮击东正教修道院</strong></p>
<p>乌克兰总统说，该国最神圣的东正教圣地之一斯维亚托希尔斯克修道院的主教堂在俄罗斯炮击后被烧毁。</p>
<p>修道院部分建筑被毁的消息传出时，俄罗斯军队正在升级对顿涅茨克地区的袭击。</p>
<p>“在这一天，全面战争的第101天，在顿涅茨克地区的斯维亚托希尔斯克修道院，万圣神殿被烧毁了。由于俄罗斯的炮击，它起火了，”泽连斯基在晚间讲话中说。</p>
<p>“他们准备烧毁一切：东正教教堂以及乌克兰的其他任何教堂。”</p>
<p>他声称斯维亚托希尔斯克修道院的三名僧侣在俄罗斯炮击中丧生，自2月入侵开始以来，已有113座东正教教堂被俄罗斯摧毁或损坏。</p>
<p>俄罗斯和乌克兰军官互相指责对方应对斯维亚托希尔斯克修道院的破坏负责。</p>
