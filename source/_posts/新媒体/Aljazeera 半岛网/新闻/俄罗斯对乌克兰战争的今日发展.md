<h3>以下是这场战争进入今日的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>俄罗斯抨击制裁并推卸粮食危机的责任</strong></p>
<p>俄罗斯向西方国家施压，要求解除因乌克兰战争而对俄罗斯实施的制裁，并试图将日益严重的粮食危机归咎于西方，乌克兰因当前的冲突而无法运送数百万吨粮食及其他农产品，从而加剧了全球的粮食危机。</p>
<p>英国立即指责莫斯科“试图勒索全世界”，并坚称不会放松制裁。</p>
<p>乌克兰是世界上最大的小麦、玉米和葵花籽油出口国之一，但是战争以及俄罗斯对其港口的封锁，使得其大部分产品无法离开乌克兰，从而危及世界粮食供应。其中许多港口现在布满了地雷。</p>
<p>克里姆林宫发言人佩斯科夫将这场危机归咎于西方的制裁。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>乌克兰军方称俄罗斯在卢甘斯克地区占有优势</strong></p>
<p>乌克兰军方表示，俄罗斯在乌克兰东部卢甘斯克地区的战斗中拥有军事优势，但他们也在尽一切努力。</p>
<p>与此同时，卢甘斯克地区州长海岱承认，乌克兰军队在俄罗斯进攻东部顿巴斯地区之前就已经撤退，但他表示，离开利西昌斯克和北顿涅茨克的最后一条道路仍处在俄罗斯的控制之外。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>泽连斯基：金钱让部分国家容忍俄罗斯</strong></p>
<p>乌克兰总统泽连斯基表示，俄罗斯资金的吸引力让一些国家能够容忍其侵略行为。</p>
<p>他拒绝为了安抚莫斯科而接受领土让步的呼吁。</p>
<p>泽连斯基在向拉脱维亚议会发表的视频讲话中指出，“今天我们听到，据说俄罗斯应当得到它想要的东西，据说有些民族有必要同意被剥夺一些外交政策权利。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官员：土耳其与俄罗斯、乌克兰正就粮食出口走廊问题进行谈判</strong></p>
<p>路透社援引土耳其一名高级官员的话报道称，土耳其正与俄罗斯和乌克兰就开通一条经博斯普鲁斯海峡的走廊，以便从乌克兰出口粮食的问题进行谈判。</p>
<p>乌克兰的黑海港口自今年2月俄罗斯对乌克兰发起军事行动以来，便一直受到封锁，约有2000万吨谷物被困在当地的仓库内。</p>
<p>俄罗斯和乌克兰的小麦供应占到全球小麦供应总量的近三分之一，而乌克兰的出口不足正在加剧全球范围内的粮食危机。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟希望中东能够取代俄罗斯以供应天然气</strong></p>
<p>俄罗斯对乌克兰的战争迫使欧盟寻找俄罗斯天然气的替代品，而俄罗斯的天然气进口量每年约占欧洲天然气消费量的40%。卡塔尔和其他中东国家可以在这个过程中发挥关键的作用。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯：如果YouTube屏蔽外交部新闻发布会，那么将会驱逐西方记者</strong></p>
<p>俄罗斯外交部表示，如果YouTube阻止用户访问其发言人的简报会，那么西方国家的记者们就将被驱逐出俄罗斯。</p>
<p>俄罗斯外交部发言人玛利亚·扎哈罗娃每周都会就俄罗斯外交政策(包括该国对乌克兰的军事行动)举行简报会。她指出，俄罗斯外交部已经警告YouTube不要屏蔽相关内容。</p>
<p>塔斯社援引她的话报道称，“我们只是要告诉他们：‘如果你们屏蔽另一场简报会，那么就会有另一名记者或者美国媒体机构可以回去了’。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：普京通过食物而勒索世界</strong></p>
<p>英国外交大臣特拉斯在回答是否支持解除对俄制裁以换取乌克兰粮食出口的问题时表示，俄罗斯总统普京正通过粮食问题而勒索全世界。</p>
<p>特拉斯26日在访问波斯尼亚时表示，“普京试图勒索全世界，这完全是令人震惊的事实，他实际上是在将世界上最贫穷的人受到的饥饿和食物缺乏当作一种武器。”</p>
<p>“我们绝不能允许这种情况发生。普京需要解除对乌克兰粮食的封锁。”</p>
<figure id="attachment_284708" aria-describedby="caption-attachment-284708" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-284708" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/UK-FM.png?w=770&resize=770%2C517" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-284708" class="wp-caption-text">特拉斯认为普京的态度“完全是令人震惊的事实” (路透社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄新社：俄罗斯公布了伊斯坎德尔导弹的发射画面</strong></p>
<p>根据俄新社的报道，俄罗斯国防部公布了一段视频，其中显示了一枚伊斯坎德尔-K型导弹向乌克兰一个未具名的“军事目标”发射的画面。</p>
<p>“伊斯坎德尔”是一种短程弹道导弹系统，自2月24日俄罗斯向乌克兰派遣数万名士兵以开展其“特别军事行动”以来，俄罗斯军队已经部署了该系统以打击乌克兰的城市、弹药库与其他的军事目标。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>分离主义共和国内“关押了8000名乌克兰战俘”</strong></p>
<p>塔斯社援引卢甘斯克地区官员罗迪翁·米罗什尼克的话报道称，被关押在受俄罗斯支持的卢甘斯克和顿涅茨克“人民共和国”内的乌克兰战俘约有8000人。</p>
<p>米罗什尼克表示，“有很多的囚犯。当然，在顿涅茨克人民共和国境内还有更多的囚犯，但我们这里也有很多，现在这些囚犯的总数大约在8000人。这是一个庞大的数字，而且每天都会新增数百人。”</p>
<p>半岛电视台记者无法独立证实这些说法。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>基辅：乌克兰东部19座高层建筑被毁</strong></p>
<p>乌克兰总统泽连斯基的办公室表示，在北顿涅茨克地区有11座高层建筑被毁，而在利西昌斯克地区也有8座高层建筑被毁。</p>
<p>泽连斯基表示，在东部的一些地区内，俄罗斯军队的人数远远超过了乌克兰军队，基辅一直试图与莫斯科安排囚犯交换的行动，但却没有成功。</p>
<figure id="attachment_284619" aria-describedby="caption-attachment-284619" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-284619" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/zelensky.png?w=770&resize=770%2C528" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-284619" class="wp-caption-text">泽连斯基表示，俄罗斯军队在东部一些地区内的人数远远超过了乌克兰军队 (美联社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>爱沙尼亚总理称必须避免乌克兰出现糟糕的和平</strong></p>
<p>爱沙尼亚总理卡娅·卡拉斯25日表示，乌克兰必须能够站在强势立场上与俄罗斯谈判，这样莫斯科才不会受到鼓励，以进一步采取咄咄逼人的行动。</p>
<p>她在斯德哥尔摩发表的讲话中指出，“我们必须避免糟糕的和平，乌克兰被迫在谈判中接受的和平，对我们所有人来说，都将是糟糕的和平。”</p>
<p>“向普京屈服要比激怒他危险得多。所有这些对侵略者看似微不足道的让步都会导致大规模的战争。我们已经犯了3次这样的错误——格鲁吉亚、克里米亚和顿巴斯。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄新社：顿巴斯地区分裂主义领导人希望俄罗斯加快行动</strong></p>
<p>根据俄新社的报道，受俄罗斯支持的顿涅茨克分裂分子领袖呼吁俄罗斯加快在乌克兰东部顿巴斯地区的军事行动。</p>
<p>自行宣布成立的顿涅茨克人民共和国领导人丹尼斯·普希林表示，基辅已经封锁了该地区北部主要城市内的供水，他还呼吁加强军事行动。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯预计2022年的石油产量将减少8.4%</strong></p>
<p>俄罗斯副总理诺瓦克表示，俄罗斯的石油产量预计将从2021年的5.24亿吨下降至今年的4.8亿吨至5亿吨。</p>
<p>诺瓦克26日在德黑兰向记者们表示，这一预测可能会根据情况而发生改变。</p>
<p>俄罗斯经济部曾表示，在基本条件不变的情况下，俄罗斯今年的石油产量将下降9.3%至4.753亿吨。</p>
<p>“我认为收缩幅度会小得多。只有一个月的日产量下降超过100万桶，但现在已经没有那么严重了。所以，我认为未来的产量将会复苏。”</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：战争造成顿涅茨克地区至少431位平民死亡</strong></p>
<p>顿涅茨克州长表示，自俄罗斯开始入侵以来，该地区至少已有431名平民遇害，另有1168人受伤。但他补充称，马里乌波尔和沃尔诺瓦哈镇的伤亡人数目前尚不清楚。</p>
<p>帕夫洛·基里连科在Telegram上表示，“俄罗斯武装部队正在杀害和摧毁顿涅茨克定居点内的民用基础设施，并导致数十万人流离失所。”</p>
<p>马里乌波尔市长的一名顾问在25日表示，据该市官员估计，在俄罗斯开展军事行动的前3个月内，至少已有2.2万名平民死亡。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯支持的分裂分子称已经关押了8000名乌克兰战俘</strong></p>
<p>塔斯社援引卢甘斯克地区官员罗迪翁·米罗什尼克的话报道称，被关押在受俄罗斯支持的卢甘斯克和顿涅茨克“人民共和国”内的乌克兰战俘约有8000人。</p>
<p>米罗什尼克表示，“有很多的囚犯。当然，在顿涅茨克人民共和国境内还有更多的囚犯，但我们这里也有很多，现在这些囚犯的总数大约在8000人。这是一个庞大的数字，而且每天都会新增数百人。”</p>
<p>半岛电视台记者无法独立证实这些说法。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰声称有240名儿童在战争中丧生</strong></p>
<p>乌克兰方面已经证实，在俄罗斯对乌克兰发起军事行动之后，已经有240名儿童死亡，另有436名儿童受伤。</p>
<p>人权监察员表示，这些数字来自审前调查登记册以及“其他需要证实的来源”。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯官员：乌克兰“永远”失去了进入亚速海的通道</strong></p>
<p>根据俄新社的报道，俄罗斯在被占领的扎波罗热地区的一名代表称，乌克兰永远失去了通往亚速海的通道。</p>
<p>早些时候，在被吞并的克里米亚地区的另一名俄罗斯代表向该媒体记者表示，在马里乌波尔“被解放”之后，亚速海成为了俄罗斯和“顿涅茨克人民共和国”的共同海域。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：俄罗斯的专业空军可能管理不善</strong></p>
<p>英国国防部表示，尽管俄罗斯空降部队（VDV）享有精英地位、高薪，并且由专业合同士兵组成，但他们却在乌克兰遭遇了重大的战术失误。该部补充称，这可能反映了对这股原本优越的力量的战略管理不善。</p>
<p>英国国防部在最新的情报简报中指出，该部队涉及的战术失败包括“在今年3月份试图通过戈斯托梅利机场向基辅推进，4月以来在伊久姆战线上的进展停滞，以及最近在试图横跨北顿涅茨河过程中的失败并且付出了高昂的代价”。</p>
<p>英国国防部认为，“该部队被用于更适合重型装甲步兵的任务，并且在战役中承受了严重的伤亡。”</p>
<p>该部补充称，“其好坏参半的表现，可能反映了对这股力量的战略管理不善，以及俄罗斯未能确保其空中优势。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：利西昌斯克及其周边地区有3人在炮击中丧生</strong></p>
<p>卢甘斯克地区州长海岱表示，25日有3人在利西昌斯克及其周边地区的炮击中丧生。</p>
<p>海岱表示，空袭“摧毁”了利西昌斯克附近的一个村庄，造成两人死亡，而利西昌斯克当地也有一人死亡。</p>
<p>他还补充称，一枚炮弹击中了北顿涅茨克的一个人道主义中心。</p>

<p>易卜拉欣·卡林在安卡拉进行了近5个小时的会谈后表示，“我们已经非常明确地表示，如果土耳其的安全担忧不能在特定时间框架内得到具体措施的解决，那么这项进程将不会取得进展。”</p>
<p>土耳其指责这两个欧洲国家窝藏“恐怖分子”，即库尔德工人党和居伦运动。</p>
<p>卡林表示，安卡拉提出的要求两国取消针对土耳其武器出口限制的建议得到了“积极的回应”。他还补充称，土耳其还预计将从瑞典引渡28名“恐怖主义”嫌疑人，同时还将从芬兰引渡12名相关嫌疑人。</p>
