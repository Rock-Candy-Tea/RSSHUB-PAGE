<p>以下是最新更新：</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基总统首次访问基辅地区以外的前线</strong></p>
<p>乌克兰总统府称，泽连斯基视察了哈尔科夫东北地区前线的部队。</p>
<p>此次访问标志着他自2月24日俄罗斯全面入侵乌克兰开始以来首次在基辅地区以外的地方正式露面。</p>
<p>总统办公室网站援引了他对士兵的话说，“你们为我们所有人和我们的国家冒着生命危险，”声明补充说他分发了表彰和礼物。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯宗主教：我理解乌克兰分会切断联系的决定</strong></p>
<p>俄罗斯东正教教会领袖基里尔一世宗主教表示，他理解乌克兰教会分会与莫斯科教会断绝关系的决定。</p>
<p>据法新社报道，俄罗斯东正教教会负责人在这一消息公布后的首次评论中说：“我们完全理解乌克兰东正教教会今天遭受的苦难。”</p>
<p>曾经与莫斯科教会结盟的乌克兰东正教最近宣布，由于对乌克兰的战争，它决定与俄罗斯教会断绝关系，宣布“完全独立”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯大使驳回战争罪指控</strong></p>
<p>俄罗斯驻英国大使表示，称俄罗斯军队在乌克兰布查镇犯下战争罪的指控是“捏造的”。</p>
<p>安德烈·克林告诉BBC，俄罗斯军队没有炮击平民区。</p>
<p>莫斯科拒绝为其部队犯下的罪行承担责任，声称有关滥杀滥伤、万人坑和强迫失踪的报道是虚假的。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>普京与塞尔维亚总统就天然气供应达成一致</strong></p>
<p>据路透社报道，克里姆林宫表示，普京和塞尔维亚总统亚历山大·武契奇同意俄罗斯将继续向塞尔维亚供应天然气，两国将加强伙伴关系。</p>
<p>武契奇说，两位领导人在周日的电话中就一项为期三年的天然气供应合同达成一致，进一步的细节将与能源巨头俄罗斯天然气工业股份公司敲定。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>立陶宛人筹款为乌克兰购买战斗无人机</strong></p>
<p>数百名立陶宛人汇集资金为乌克兰购买了一架先进的军用无人机，以表达对以前在莫斯科统治下的同胞国家的声援。</p>
<p>据发起该活动的立陶宛广播公司Laisves TV称，在不到四天的时间里筹集了540万美元，由于金额很小，这笔资金将用于购买一架拜拉克塔尔 TB2军用无人机。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯特使排除使用核武器</strong></p>
<p>俄罗斯驻英国大使周日排除了克里姆林宫在乌克兰使用战术核武器的可能性，称此类武器“根本不会用于那样的冲突”。</p>
<p>安德烈·克林在周日早上在BBC播出的一次采访中说，俄罗斯“在使用战术核武器的问题上有非常严格的规定”，并补充说，俄罗斯的原则主要是在“国家存在受到威胁”时使用它们。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称摧毁了乌克兰军队的武器库</strong></p>
<p>据路透社报道，俄罗斯国防部表示，它在乌克兰中部的克里维里赫市摧毁了乌克兰军队的一个大型武器库。</p>
<p>据俄罗斯塔斯社报道，该部还表示，俄罗斯防空系统在第聂伯地区击落了一架乌克兰苏-25战斗机。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰东正教与莫斯科宗主教区决裂</strong></p>
<p>乌克兰东正教中央分支的领导人与莫斯科的等级制度正式决裂，形成了自2014年俄罗斯吞并克里米亚以来一直在酝酿的分裂。</p>
<p>曾隶属于俄罗斯东正教莫斯科宗主教区的乌克兰东正教理事会表示，它正在寻求“完全独立和自治”。</p>
<p>乌克兰东正教理事会还表示，由于莫斯科宗主教区领导人基里尔一世对战争的支持，它正在切断与他的联系。基里尔是两国教会的领袖，但他避免谴责对乌克兰人的袭击。</p>
<figure id="attachment_285477" aria-describedby="caption-attachment-285477" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-285477" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/15-12.png?w=770&resize=770%2C555" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-285477" class="wp-caption-text">俄罗斯总统弗拉基米尔·普京与莫斯科宗主教基里尔（右）握手 (路透)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>北顿涅茨克的情况“更加困难”</strong></p>
<p>卢甘斯克地区政府负责人周日表示，乌克兰东部北顿涅茨克的战斗仍在继续，当地的局势“更加困难”，“敌人不断进攻”。</p>
<p>谢尔盖·海代在一份声明中说，该地区有60所房屋被毁，两人被发现死在瓦砾下，其中包括一名女孩。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰指责俄罗斯掠夺金属</strong></p>
<p>乌克兰指责莫斯科从被占领的马里乌波尔市偷走金属，并将其装载到一艘开往俄罗斯的船上。</p>
<p>据《基辅独立报》报道，乌克兰议会人权事务专员柳德米拉·德尼索娃表示，俄罗斯在第一艘船上向俄罗斯顿河畔罗斯托夫运送了3000吨金属产品。</p>
<p>她补充说，“为了更方便地清扫战利品，占领者已经开始恢复马里乌波尔和沃尔诺瓦卡的铁路连接。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯天然气工业股份公司仍在通过乌克兰向欧洲运输天然气</strong></p>
<p>俄罗斯天然气生产商俄罗斯天然气工业股份公司证实，俄罗斯将继续通过乌克兰向欧洲运送天然气。</p>
<p>这家俄罗斯天然气生产商表示，尽管发生了战争，但其通过乌克兰（Sudzha入口点）对欧洲天然气供应仍在继续。</p>
<p>据路透社报道，乌克兰拒绝了通过另一个入口点Sokhranovka供应天然气的申请。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰有682名儿童死亡或受伤</strong></p>
<p>该国总检察长办公室在Telegram上表示，乌克兰有超过682名儿童受伤或死亡。</p>
<p>声明说，有242人死亡，440人受伤，并补充说这些数字不是最终数字，因为很难证实在发生战斗的地方的报道。人数最多的是顿涅茨克（153）、基辅（116）和哈尔科夫（108）。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯取消新兵年龄限制</strong></p>
<p>据俄罗斯国家通讯社塔斯社报道，俄罗斯取消了俄罗斯人和外国人参军的年龄上限，使40岁以上的人能够入伍。</p>
<p>此前，军队对俄罗斯公民的年龄限制为18至40岁，对于希望入伍的外国人，年龄限制为18至30岁。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>“没有讨论”公投</strong></p>
<p>俄罗斯在占领的赫尔松设立的政府副局长表示，在该地区以及敖德萨和尼古拉耶夫附近地区的战斗停止之前，不会就正式加入俄罗斯进行公投。</p>
<p>基里尔·斯特雷穆索夫表示，“没有关于公投的讨论”，但“我们将在稍后进行某种投票或公民投票时宣布”。乌克兰官员此前警告称，俄罗斯任命的行政人员正准备进行虚假投票以接管乌克兰的部分地区。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰呼吁远程武器</strong></p>
<p>乌克兰总统的一位顾问呼吁西方国家向乌克兰提供远程武器，如果西方真的希望基辅战胜俄罗斯的话。</p>
<p>“当你在70公里外受到攻击而没有什么可以反击的时候，你很难反抗。乌克兰可以将俄罗斯带回铁幕后，但我们需要有效的武器，”米哈伊洛·波多利亚克在推特上写道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰军队：为北顿涅茨克的战斗仍在继续</strong></p>
<p>乌克兰军方表示，乌克兰东部城市北顿涅茨克的战斗仍在继续，俄罗斯军队周六进行了突击行动。</p>
<p>乌克兰武装部队总参谋部周日在Facebook上发表声明说：“通过使用大炮，俄罗斯军队在北顿涅茨克市地区进行了突击行动。”</p>
<p>“战斗还在继续。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官方：数十名乌克兰运动员在战争中丧生</strong></p>
<p>据乌克兰国家通讯社报道，乌克兰青年和体育部长表示，超过50名乌克兰运动员在保卫国家对抗俄罗斯时丧生。</p>
<p>“这些都是我们从事不同运动的人。目前仍有许多运动员在乌克兰武装部队服役，其中一些是在领土防御部队，”瓦迪姆·赫特萨尔特说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基说他期待武器交付的“好消息”</strong></p>
<p>乌克兰总统表示，他期待下周有关伙伴国家运送武器的好消息。</p>
<p>“每天我们都在接近我们的军队在技术和火力上超越占领者。当然，这在很大程度上取决于合作伙伴是否愿意为乌克兰提供捍卫自由所需的一切。我预计下周就会有好消息，”他在晚间讲话中说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>日本承诺提供170万美元用于运送乌克兰援助物资</strong></p>
<p>据NHK广播公司报道，日本已宣布提供170万美元的援助，以帮助向乌克兰运送援助物资。</p>
<p>据NHK报道，官员们表示，这笔钱将捐给联合国项目服务办公室。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>大坝被毁以阻止俄罗斯前进数月后，乌克兰部分村庄仍被洪水覆盖</strong></p>
<p>官员们说，在一座大坝被摧毁，该地区被洪水淹没以阻止俄罗斯军队向基辅推进后，乌克兰德米迪夫村的数十座房屋仍然在水中。</p>
<p>“此时，德米迪夫村大约50所房屋仍然泡在水中，”地区州长奥列克西·库列巴在Telegram消息应用程序上的一篇帖子中写道。</p>
<p>“人们正在了解情况。我们会尽一切努力解决这个问题。”</p>
<p>2月，乌克兰军队炸毁了伊尔平河上的大坝，淹没了有1000多年历史的德米迪夫村庄的房屋和田地。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基说，俄罗斯阻止乌克兰人离开赫尔松</strong></p>
<p>乌克兰总统表示，俄罗斯军队正试图阻止乌克兰人离开赫尔松地区南部被占领地区。</p>
<p>“他们不提供任何人道主义走廊。 并封闭了个人的离境通道。对自己的立场有信心的人，绝对不会做出这样的决定。这显然是软弱的表现，”泽连斯基在晚间讲话中说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基向欧洲理事会发表讲话</strong></p>
<p>乌克兰总统表示，他正准备在5月30日至31日举行的欧洲理事会会议上发表讲话。</p>
<p>“特别是，我将谈论以下内容：恐怖，这实际上已成为俄罗斯对欧洲采取行动的唯一形式。乌克兰土地上的恐怖。欧洲能源市场上的恐怖，不仅仅是我们国家。食品市场和全球范围内的恐怖活动。那么接下来会出现什么样的恐怖呢？”</p>
<p>他在晚间讲话中表示，下周他将继续在欧洲国家议会发表讲话。</p>
<p>“将有更多国际活动举行，加强我们的国家并增加整个自由世界对俄罗斯的共同压力。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰东部局势“非常困难”：泽连斯基</strong></p>
<p>乌克兰总统泽连斯基表示，俄罗斯对顿巴斯和哈尔科夫地区前线的进攻使局势“非常艰难”。</p>
<p>泽连斯基穿着一件写有“我是乌克兰人”的T恤，他说俄罗斯的重点仍然集中在北顿涅茨克、利西昌斯克、巴赫穆特和波帕斯纳等城市。</p>
<p>他补充说：“我将提醒世界，俄罗斯最终必须被正式认定为恐怖主义国家和支持恐怖主义的国家。”</p>

<p><strong>ISW：乌克兰在赫尔松地区的反攻正在进行</strong></p>

<p>该政策研究组织周六早些时候表示，俄罗斯军队在赫尔松-尼古拉耶夫附近发动的地面攻击没有成功。</p>

<p><strong>乌克兰接收鱼叉反舰导弹、榴弹炮</strong></p>
<p>乌克兰国防部长阿列克谢·列兹尼科夫表示，乌克兰已开始接收来自丹麦的鱼叉反舰导弹和来自美国的自行榴弹炮。</p>
<p>雷兹尼科夫说，乌克兰收到了一系列重型火炮，包括改进的美国制造的M109自行榴弹炮，这将使乌克兰军队能够从更远的距离打击目标。</p>
<p>这位部长在他的Facebook页面上写道，鱼叉岸对舰导弹将与乌克兰海王星导弹一起用于保卫该国海岸，包括南部港口敖德萨。</p>
<p>乌克兰南部敖德萨地区军事管理局发言人塞尔吉奥·布拉楚克在网上发帖称，“我们获得了如此多的鱼叉导弹，我们可以击沉整个俄罗斯黑海舰队。为什么不呢？”</p>

<p><strong>乌克兰总统顾问称与莫斯科的协议一文不值</strong></p>
<p>乌克兰总统顾问兼和平谈判谈判代表米哈伊洛·波多利亚克表示，与俄罗斯达成的任何协议都不可信，并补充说，阻止莫斯科入侵的唯一方法是武力。</p>
<p>“与俄罗斯的任何协议都不值一分钱，”波多利亚克在Telegram消息应用程序上写道。 “有可能与一个总是见利忘义撒谎和宣传的国家谈判吗？”</p>
<p>“俄罗斯已经证明它是一个威胁世界安全的野蛮国家，”波多利亚克说。 “野蛮人只能靠武力阻止。”</p>
<p>在和平谈判陷入僵局后，俄罗斯和乌克兰互相指责，最后一次已知的面对面谈判是在3月29日。</p>

<p><strong>普京表示愿意讨论将粮食运往乌克兰港口的方式</strong></p>
<p>俄罗斯总统弗拉基米尔·普京告诉法国和德国领导人，莫斯科准备寻找方法运送滞留在乌克兰港口的粮食，但要求西方解除制裁。</p>
<p>普京表示，向世界市场供应粮食的困难是“西方国家错误的经济和金融政策”的结果。</p>
<p>克里姆林宫称，普京告诉马克龙和朔尔茨，“俄罗斯准备帮助寻找粮食出口不受阻碍的选择，包括从黑海港口出口乌克兰粮食。”</p>
<p>“俄罗斯化肥和农产品供应的增加也将有助于缓解全球食品市场的紧张局势，当然，这需要取消相关制裁。”</p>
