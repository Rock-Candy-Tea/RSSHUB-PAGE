<li>克里姆林宫表示，普京告诉埃尔多安，俄罗斯准备与土耳其协调，以便为乌克兰港口畅通的粮食出口提供便利。</li>

<h3>以下是这场战争进入今日后的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>北顿涅茨克市长称俄罗斯军队占领了这座城市的一半面积</strong></p>
<p>北顿涅茨克市长奥莱克桑德尔·斯特里克表示，俄罗斯军队已经占领了东部城市北顿涅茨克的一半面积。</p>
<p>奥莱克桑德尔·斯特里克向美联社记者表示，“这座城市基本上正在被一个街区一个街区地无情地摧毁。”</p>
<p>他还表示，激烈的巷战仍在继续，炮火威胁着仍在废墟中避难的近1.3万平民的生命。这座城市曾经居住着超过10万人。</p>
<p>由于炮击，撤离工作已经停止。他还表示，“粮食供应还可以维持几天，但问题在于如何分配。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>检察官：乌克兰已确认了600名俄罗斯的战争罪嫌疑人</strong></p>
<p>据基辅最高检察官说，乌克兰已经确认了600多名俄罗斯战争罪嫌疑人，并将开始起诉其中近80人。</p>
<p>乌克兰总检察长伊琳娜·韦涅季克托娃在海牙举行的新闻发布会上表示，嫌疑人名单包括“俄罗斯的高级军事人员、政治人物和宣传人士”。</p>
<p>韦涅季克托娃指出，爱沙尼亚、拉脱维亚和斯洛伐克已经决定加入乌克兰的国际调查小组。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯议员提议绑架北约国防部长</strong></p>
<p>俄罗斯高级议员奥列格·莫洛佐夫建议绑架北约驻乌克兰的一名国防部长，并将他们带至莫斯科，询问西方对基辅下达了什么“命令”。</p>
<p>奥列格·莫洛佐夫于1993年首次当选俄罗斯议会议员，是俄罗斯执政党统一俄罗斯党的成员。他认为，西方向乌克兰提供武器对俄罗斯构成了直接威胁，可能会要求莫斯科重新审视其军事目标。</p>
<p>莫洛佐夫30日晚间在俄罗斯媒体的一档脱口秀节目中表示，“你知道,也许这只是一个我想象出来的情节……在不久的将来的某个阶段内，某个北约国家的战争部长将乘火车前往基辅去跟泽连斯基（乌克兰总统）会谈。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟将对俄罗斯实施为期数月的石油制裁</strong></p>
<p>欧盟委员会的一名发言人表示，欧盟对从海上进口俄罗斯石油的制裁将分阶段实施，其中，原油进口为6个月，成品油进口为8个月。</p>
<p>欧盟领导人在30日举行的峰会上，从原则上同意实施制裁措施，而这一时间表也将在制裁措施正式实施后生效，欧盟各国大使计划在本周实施这些制裁措施。</p>
<p>这项时间表将遵循欧盟委员会最初提出的最后期限，需要指出的是，该委员会在本月早些时候提出了石油制裁。</p>
<figure id="attachment_286031" aria-describedby="caption-attachment-286031" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-286031" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/irpin-ukraine.png?w=770&resize=770%2C511" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-286031" class="wp-caption-text">2022年5月30日，乌克兰基辅伊尔宾的郊区，一位女孩坐在被摧毁的建筑物外的秋千上 (美联社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>土耳其与俄罗斯将就乌克兰出口走廊问题于6月8日举行会谈</strong></p>
<p>土耳其外交部长恰武什奥卢表示，俄罗斯外交部长拉夫罗夫将于6月8日率领一个军事代表团访问土耳其，以讨论为乌克兰农产品出口建立一个潜在的海上走廊的问题。</p>
<p>俄罗斯总统普京30日在与土耳其总统埃尔多安的电话中表示，俄罗斯准备与土耳其协调，为恢复乌克兰港口的粮食出口畅通而提供便利。</p>
<p>恰武什奥卢表示，与联合国的合作仍在进行中，以就建立从黑海出发的走廊达成协议，但是莫斯科和基辅之间的问题仍然存在。他还表示，联合国已经提议建立一个联合观察机制来监督相关的航运路线，而土耳其对此持开放态度。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰高级官员称欧盟对俄罗斯的制裁仍然“不够”</strong></p>
<p>乌克兰总统办公室的一名高级官员表示，欧盟对俄罗斯的最新制裁——禁止从俄罗斯进口大部分石油——的力度仍然“不够”，并且认为欧盟迄今为止的制裁步伐过于缓慢。</p>
<p>当地时间5月30日，在布鲁塞尔举行的欧盟峰会上，欧盟对莫斯科采取了自3个月来最为严厉的制裁措施，官员们表示，这些措施将立即削减从俄罗斯进口的石油总量的三分之二以上，并且将在今年年底之前削减90%。</p>
<p>俄罗斯总统泽连斯基办公室副主任却表示，“如果你问我的意见的话，我会说这太慢了，太晚了，而且肯定力度还不够。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧安组织称，被关押在乌克兰东部的4名乌克兰工作人之一获释</strong></p>
<p>欧安组织（欧洲安全与合作组织）秘书长表示，在乌克兰东部被拘留的4名乌克兰人中，已有一人获释。这4人主要负责该组织的监督工作。</p>
<p>赫尔加·施密德在推特上表示，“对一名任务成员的释放感到非常宽慰”，“我敦促释放仍被关押在顿涅茨克和卢甘斯克地区的另外3名同事。”</p>
<p>该特派团的任务期限已于今年3月底到期。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰法院判处两名俄罗斯士兵入狱</strong></p>
<p>乌克兰一家法院判处两名被逮捕的俄罗斯士兵11年半监禁，原因是他们炮轰了乌克兰东部一个城镇，这是自今年2月份俄罗斯对乌克兰发起军事行动以来，第二次有俄罗斯士兵被判处战争罪。</p>
<p>这两名士兵——亚历山大·波比金(Alexander Bobikin)与亚历山大·伊万诺夫(Alexander Ivanov)上周在乌克兰中部科泰弗斯卡地区法院听取了判决，而两人已经在上周“认罪”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称击落了乌克兰的战斗机</strong></p>
<p>俄罗斯国防部表示，其部队在乌克兰尼古拉耶夫地区击落了一架乌克兰苏-25战斗机，并且炸毁了乌克兰东部的一个雷达站和两个弹药库。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-286099" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/zelensky-1.png?w=724&resize=724%2C535" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>拉脱维亚欢迎欧盟针对俄罗斯大部分石油进口的禁令</strong></p>
<p>拉脱维亚总理克里贾尼斯·卡林斯赞扬欧盟达成禁止从俄罗斯进口大部分石油的协议，并称欧盟国家共同采取了第6轮制裁措施，以使莫斯科没有资金对乌克兰继续发动战争。</p>
<p>欧盟27个国家的领导人在克服来自匈牙利的抵制并达成协议之后，卡林斯表示，“我们会尽可能快地行动起来，让俄罗斯更难资助这场针对乌克兰的战争。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>博雷尔：欧盟石油禁令将削减俄罗斯在乌克兰战争上的开支</strong></p>
<p>欧盟最高外交官员博雷尔表示，欧盟削减俄罗斯大部分石油进口的协议，将迫使莫斯科以更低的价格向其他国家供应原油。</p>
<p>“我们是俄罗斯最重要的客户”，博雷尔在抵达并参与欧盟领导人关于俄罗斯对乌克兰最近战争的第二天的讨论时指出，“这样做的目的是让俄罗斯没有足够的财力来供养其战争机器。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>塔斯社：俄罗斯支持的分裂分子拥有留在马里乌波尔港的船只</strong></p>
<p>据俄罗斯国家通讯社塔斯社的报道，“顿涅茨克人民共和国”的分裂主义领导人丹尼斯·普希林表示，包括一些外国船只在内的停靠在马里乌波尔港的船只，将被移交给该政权，从而使其拥有自己的“商船队伍”。</p>
<p>丹尼斯·普希林还指出，“部分船只将被转移到该共和国的管辖范围内。有关决定已经作出。这些船只将会悬挂什么旗帜也已非常清晰。”</p>
<p>早些时候，俄罗斯国防管理中心的负责人说，共有6艘外国船只留在马里乌波尔港，其中包括“公主”号(保加利亚)、“阿兹堡”号(多米尼加共和国)、“斯玛特”号(利比里亚)、“蓝星”号(巴拿马)、“亚速康科德”号(土耳其)和“奥古斯塔夫人”号(牙买加)。</p>
<p>塔斯社报道称，俄罗斯陆军上校米哈伊尔·米津采夫表示，拥有这些船只的国家尚未采取措施以撤回它们。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯天然气工业股份公司称已切断对荷兰天然气公司的天然气供应</strong></p>
<p>俄罗斯能源巨头俄罗斯天然气工业股份公司表示，由于荷兰天然气贸易商“GasTerra”未能支付4月份交付的天然气款项，因此，俄方公司已经完全切断了对该荷兰公司的天然气供应。</p>
<p>路透社报道称，这些款项应当按照俄罗斯总统下令的“天然气换卢布”计划完成支付。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：顿涅茨克地区有3人死亡、4人受伤</strong></p>
<p>乌克兰顿涅茨克地区州长表示，该地区在30日共有3位平民丧生，另有4人受伤。</p>
<p>他还指出，在托雷茨克镇有两人死亡，另外在阿维迪夫卡镇也有一人死亡。</p>
<p>他在Telegram上表示，“目前无法确定马里乌波尔和沃尔诺瓦卡地区确切的受害者人数。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：俄罗斯需要北顿涅茨克以外的领土来完全控制顿巴斯地区</strong></p>
<p>英国国防部表示，俄罗斯如果想完全占领卢甘斯克和顿涅茨克地区，就需要占领北顿涅茨克以外的领土，包括主要城市克拉马托尔斯克和第聂伯罗-顿涅茨克M04主干道。</p>
<p>英国国防部在最新的情报简报中指出，俄罗斯占领莱曼镇支持了其包围北顿涅茨克的主要行动，并且封锁了卢甘斯克地区的乌克兰军队。</p>
<p>简报指出，“猛烈的炮击仍在继续，而北顿涅茨克的郊区可能会发生巷战。”</p>
<p>这份简报还补充称，俄罗斯的进展缓慢，但它保持着进展，这可能是因为它在相对较小的地区集结了部队与火力，从而迫使它在被占领领土的其他地方承受风险。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>分离主义领导人：自俄罗斯占领马里乌波尔后有第一艘船离开该港</strong></p>
<p>国际文传电讯社援引受俄罗斯支持的乌克兰分裂地区顿涅茨克分裂主义领导人的话称，自俄罗斯占领马里乌波尔以来，首次有一艘船离开该港口，并朝东驶向俄罗斯。</p>
<p>该港口发言人在上周表示，这艘船将在马里乌波尔装载2700吨金属，然后向东前往俄罗斯城市顿河畔罗斯托夫。</p>
<p>乌克兰表示，从马里乌波尔向俄罗斯运送金属的行为相当于抢劫。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯军队正向北顿涅茨克市中心挺进</strong></p>
<p>卢甘斯克州长海岱在接受乌克兰电视台采访时表示，俄罗斯军队正在缓慢向北顿涅茨克的中心地区推进。</p>
<p>海岱指出，保卫北顿涅茨克的乌克兰军队并没有被包围的危险，因为他们可以撤退到河对岸的利西昌斯克。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>车臣领导人发布了军队在北顿涅茨克市中心的视频</strong></p>

<p><strong>泽连斯基警告不要屈服于俄罗斯在食品供应上的“勒索”</strong></p>
<p>乌克兰总统办公室表示，乌克兰总统泽连斯基在与法国外交部长卡特琳·科隆纳会晤时，要求法国不要在食品供应问题上屈服于俄罗斯的“勒索”。</p>
<p>乌克兰总统在此次会谈的总结中，指责俄罗斯试图利用粮食安全问题来缓解其国际制裁。基辅表示，莫斯科从乌克兰军队控制的地区掠夺了粮食和农业设备，并阻挠了乌克兰的粮食出口。</p>
<p>泽连斯基办公室表示，他与科降纳讨论了制裁、武器供应的问题，以及乌克兰加入欧盟的愿望。</p>
<p>乌克兰战争导致全球食品价格上涨，人们担心全球许多地方的饥饿危机会进一步恶化。</p>

<p><strong>德国政界同意了1070亿美元的军事建设协议</strong></p>
<p>德国政府和保守反对派已就一项协议达成一致，同意释放1000亿欧元(约合1070亿美元)用于德国军队的现代化建设，以应对俄罗斯。</p>
<p>双方还达成协议，为军事采购设立一项特别基金，从而使柏林方面能够实现北约为其设定的目标，即将其GDP（国内生产总值）的2%用于国防目标。</p>
<p>这些团体的代表告诉法新社记者，这项协议涉及修改国家宪法中的预算规则，并且是在执政联盟中的政党和前总理默克尔的保守派进行了为期数周的艰难谈判后达成的。</p>

<p><strong>埃尔多安分别与普京、泽连斯基举行会谈</strong></p>
<p>土耳其总统埃尔多安分别与乌克兰总统泽连斯基、俄罗斯总统普京举行会谈，并提出为乌克兰港口重新开放提供便利。</p>
<p>在谈到普京与埃尔多安之间的通话时，克里姆林宫表示，“在讨论乌克兰局势时，重点放在了确保黑海和亚速海的安全航行，以及消除这两个海域内的水雷威胁。”</p>
<p>安卡拉方面表示，埃尔多安告诉普京，需要尽快建立和平，如果达成协议，土耳其愿意在莫斯科、基辅和联合国之间的“观察机制”中发挥作用。</p>
<p>据埃尔多安办公室透露，埃尔多安后来表示，他“特别重视为出口乌克兰农产品创建一条安全海上航线的项目”，并补充称，土耳其领导人原则上欢迎将伊斯坦布尔作为“观察机制”总部的想法。</p>

<p><strong>欧盟没有就俄罗斯石油禁运达成协议</strong></p>
<p>欧盟各国在布鲁塞尔举行会议，讨论对莫斯科实施更多制裁，但未能就禁止从俄罗斯进口石油达成一致，尽管在为期两天的峰会召开之前进行了最后一刻的磋商。</p>
<p>自从俄罗斯对乌克兰发起军事行动以来，欧盟已经对俄罗斯实施了5轮制裁措施，但是由于许多国家依赖俄罗斯的原油，石油禁运协议一直难以达成。</p>
<p>匈牙利总理欧尔班表示，“目前根本没有妥协的余地。”匈牙利一直是反对达成相关协议的主要国家。</p>
