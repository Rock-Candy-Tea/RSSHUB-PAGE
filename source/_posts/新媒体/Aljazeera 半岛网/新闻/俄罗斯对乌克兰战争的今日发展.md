<p>英国首相鲍里斯·约翰逊在基辅表示，英国将继续对俄罗斯实施制裁，并将向乌克兰提供更多援助，使其日后不受任何侵略。</p>
<p>乌克兰总统弗拉基米尔·泽连斯基赞扬英国对乌克兰的支持，并呼吁对俄罗斯石油实施全面禁运。</p>
<p>另一方面，俄罗斯驻华盛顿大使阿纳托利·安东诺夫表示，这次军事行动是由于基辅不愿停止对俄罗斯人的种族灭绝，以及未能遵守国际协议。</p>
<p>在经济和人道主义方面，欧盟委员会主席表示，在波兰首都举行的捐助者会议筹集了90亿欧元，用于帮助乌克兰难民和流离失所者。</p>
<p>波兰总统说他的国家和西方支持乌克兰，因为乌克兰同时捍卫了大家的自由。</p>
<p>以下是这场战争第46天的最新进展：</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>第聂伯罗市官员：</p>
<ul style="list-style-type:disc">
<li>俄罗斯军队瞄准了该市的机场和一些民用设施。</li>
<li>该市的机场在俄罗斯的轰炸中被完全摧毁，目前还没有关于人员伤亡的信息。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<p>我与德国总理讨论了对俄罗斯的制裁以及对我国的军事和财政支持。</p>

<p>我与乌克兰官员讨论了对俄罗斯实施第6套欧洲制裁方案的提议。</p>

<li><strong>更新</strong></li>

<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰总统顾问：</strong></p>
<p>马里乌波尔的战斗仍在继续，我们的部队正在保卫这座城市。</p>
<p>我们的部队正在进行反攻行动并控制乌克兰北部。</p>

</ul>

<li><strong>更新</strong></li>

<li><strong>更新</strong></li>

<ul style="list-style-type:disc">
<li>由于损失逐渐累积，莫斯科正在寻求召回自2012年以来复员的军事人员，以加强其部队。</li>
<li>莫斯科加强其部队的计划包括从摩尔多瓦的德涅斯特河沿岸地区招募人员。</li>

<li><strong>更新</strong></li>

<p>一名州长周六晚间通过Telegram表示，俄罗斯轰炸了乌克兰东部两个城市，造成5人死亡，另有5人受伤，乌克兰东部即将遭到俄罗斯的大规模袭击。</p>
<p>顿涅茨克地区州长帕夫洛·基里连科说：“俄罗斯在顿涅茨克地区的炮击导致5人死亡，另有5人受伤，”并补充说，伏格利达尔市有4人遇难，新米哈伊洛夫卡有1人遇难。</p>
<p>该地区的激烈战斗升级，乌克兰当局敦促其居民在俄罗斯军队即将到来的袭击前离开。</p>
