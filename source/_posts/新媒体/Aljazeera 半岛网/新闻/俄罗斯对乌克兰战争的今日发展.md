<li>乌克兰总统泽连斯基在复活节的讲话中呼吁和平。</li>
<li>乌克兰总统泽连斯基批评了联合国秘书长安东尼奥·古特雷斯在前往基辅前周二访问莫斯科的决定。</li>
<li>泽连斯基表示，乌克兰希望周日在基辅与美国国务卿和国防部长的会谈中获得重型武器，但再次呼吁与普京会面以“结束战争”。</li>
<li>欧洲安全与合作组织表示，它正在努力确保一些被拘留在乌克兰东部的特别监察团成员获释。</li>
<li>一系列巡航导弹袭击在乌克兰敖德萨地区造成至少8人死亡，其中包括一名三个月大的婴儿。</li>
</ul>

<p><strong>以下是这场战争的最新进展：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>冯德莱恩在印度讨论乌克兰问题</strong></p>
<p>欧盟委员会主席乌尔苏拉·冯德莱恩抵达印度进行贸易、安全和气候谈判后，并将就其对乌克兰战争的中立立场向印度施压。</p>
<p>总理纳伦德拉·莫迪的政府没有公开谴责克里姆林宫，也没有支持联合国安理会对莫斯科“侵略”的谴责。</p>
<p>冯德莱恩的行程从参观当地气候变化倡议开始，她在那里赞扬了印度“雄心勃勃”的可再生能源目标，并表示乌克兰战争已将能源安全问题推到了聚光灯下。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>教皇再次呼吁在乌克兰实现复活节休战</strong></p>
<p>教皇方济各再次呼吁在东正教复活节周末在乌克兰停火，“以减轻疲惫不堪的人们的痛苦”。</p>
<p>“战争不但没有停止，反而变得更糟，”当乌克兰的蓝黄旗在圣彼得广场聚集的信徒中飘扬时，罗马天主教会领袖说。</p>
<p>“可悲的是，在基督徒最神圣和最庄严的日子里，我们听到的武器杀戮的声音比宣布基督复活的钟声更多”。</p>
<p>“我再次呼吁实现复活节休战，这是和平意愿的最小的有形标志，”他恳求道。</p>
<figure id="attachment_275055" aria-describedby="caption-attachment-275055" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-275055" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/28-1.png?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-275055" class="wp-caption-text">教皇方济各利用东正教复活节呼吁乌克兰停火 (美联社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>埃尔多安告诉泽连斯基土耳其准备协助谈判</strong></p>
<p>土耳其总统府表示，土耳其总统埃尔多安在电话中告诉泽连斯基，土耳其准备在乌克兰和俄罗斯的谈判过程中提供一切可能的帮助。</p>
<p>埃尔多安表示，必须确保乌克兰马里乌波尔的伤员和平民撤离，并补充说，土耳其原则上积极看待担保人问题。乌克兰在会谈期间向各国寻求安全保证。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯驻美国大使称代表团工作“受阻”</strong></p>
<p>俄新社援引大使阿纳托利·安东诺夫的话说，俄罗斯驻华盛顿大使馆的工作“受到阻碍”，其银行账户被关闭，工作人员受到威胁。</p>
<p>“大使馆实质上被美国政府实体封锁了。我们在休斯敦和纽约的两个领事馆的账户已被美国银行关闭，”安东诺夫说。</p>
<p>“我们收到了电话和信件的威胁。某些时候，甚至大使馆的出口都被封锁了，”他补充道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>联合国呼吁“立即停止”马里乌波尔战斗以允许平民撤离</strong></p>
<p>联合国乌克兰危机协调员阿明·阿瓦德呼吁“立即停止”马里乌波尔的战斗，以便“今天”疏散受灾城市中被困的平民。</p>
<p>“成千上万的生命，包括妇女、儿童和老年人，在马里乌波尔受到威胁，”阿瓦德在一份声明中说。</p>
<p>“我们现在需要暂停战斗以挽救生命。我们等待的时间越长，就会有更多的生命处于危险之中。现在，今天，必须允许他们安全撤离。明天可能为时已晚。”</p>
<p>这一呼吁得到了人道主义组织的响应，包括红十字国际委员会在内，该委员会为平民的安全通行提供了便利。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基在复活节谈及希望</strong></p>
<p>乌克兰总统泽连斯基在复活节的讲话中谈到了他的国家的希望和胜利。</p>
<p>泽连斯基在古老的圣索菲亚大教堂发表讲话时说：“今天这个伟大的节日给了我们巨大的希望和坚定的信念，那就是光明会战胜黑暗，善良会战胜邪恶，生命会战胜死亡，因此乌克兰一定会赢！”</p>
<p>“主和神圣的天堂之光在我们这一边，”他说。“我们正在经历非常艰难的考验。让我们在这条道路上达到一个正义的终点——乌克兰幸福生活和繁荣的开始。”</p>
<p>泽连斯基说：“在复活节，我们祈求上帝赐予我们伟大的恩典，让我们的梦想成真——这是另一个伟大的日子——是乌克兰将迎来伟大和平的一天。”</p>
<figure id="attachment_275028" aria-describedby="caption-attachment-275028" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-275028" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/22-1.png?w=770&resize=770%2C514" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-275028" class="wp-caption-text">信徒们在乌克兰佩雷莫哈村受损的圣母诞生教堂外参加东正教复活节礼拜 (欧洲通讯社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称其袭击了哈尔科夫地区的武器库</strong></p>
<p>俄罗斯国防部表示，其精准导弹在夜间击中了乌克兰9个军事目标，其中包括哈尔科夫地区储存火炮的4个武器库。</p>
<p>该部还表示，其导弹和炮兵部队在同一地区摧毁了另外4个这样的武器库，并袭击了第聂伯罗彼得罗夫斯克地区的一个为乌克兰军队生产炸药的设施。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：乌克兰顿涅茨克地区两名儿童在炮击中丧生</strong></p>
<p>乌克兰顿涅茨克州州长表示，两名儿童在俄罗斯军队的炮击中丧生，并敦促人们撤离战斗附近的地区。</p>
<p>保罗·基里连科在Telegram上说，两个5岁和14岁的女孩在奥切列京斯克社区死亡，她们居住的建筑物被毁。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰谈判代表称俄罗斯不断炮击马里乌波尔</strong></p>
<p>乌克兰高级谈判代表兼总统顾问米哈伊洛·波多利亚克称俄罗斯军队不断炮击马里乌波尔，并敦促莫斯科同意“真正的复活节休战”。</p>
<p>“俄罗斯不断攻击马里乌波尔的亚速钢铁厂。我们的平民和军队所在的地方遭到了重型航空炸弹和大炮的轰炸，”波多利亚克在推特上说。</p>
<p>他敦促俄罗斯“考虑维护其仅存的声誉”，并呼吁“在马里乌波尔实现真正的复活节休战”，同时为平民提供直接的人道主义走廊，并举行一轮特别会谈，以促进军民交流。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>基督教东正教精神领袖呼吁结束乌克兰“自相残杀的战争”</strong></p>
<p>普世牧首巴塞洛缪是全球东正教基督徒的精神领袖，他于夜间呼吁在乌克兰开放人道主义走廊，称“一场难以形容的人类悲剧正在上演”。</p>
<p>巴塞洛缪说：“我们再次呼吁立即结束自相残杀的战争，与任何战争一样，这场战争损害了人类尊严。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报告：瑞士拒绝德国向乌克兰再出口弹药的请求</strong></p>
<p>据瑞士报纸《周日报》报道，瑞士拒绝了再出口基辅想要获得的用于黄鼠狼步兵战车的瑞士制造弹药，以阻止德国向乌克兰运送武器。</p>
<p>该报称，由德国武器制造商莱茵金属公司制造的黄鼠狼步兵战车使用的是瑞士制造的弹药。瑞士限制将此类战争物资再出口到冲突地区。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰称200多名儿童在俄罗斯入侵中丧生</strong></p>
<p>乌克兰总检察长办公室表示，俄罗斯入侵乌克兰已造成213名儿童死亡，另有389名儿童受伤。</p>
<p>该办公室在推特上表示，这一数据不包括冲突活跃地区的人员伤亡。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国称“俄罗斯士气低落”阻碍了顿巴斯的推进</strong></p>
<p>英国国防部发推文在一份定期公告中称，尽管俄罗斯在领土上取得了一些进展，但乌克兰的抵抗力量在各个方面都很强大，并给俄罗斯军队造成了重大损失。</p>
<p>最新公告补充说：“俄罗斯士气低落，以及从先前的进攻中重新装备和重组部队的时间有限，这可能会阻碍俄罗斯的战斗力发挥。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰称俄罗斯在边境附近部署了伊斯坎德尔-M发射器</strong></p>
<p>据乌克兰军方称，俄罗斯在距离乌克兰边境60公里（40英里）的范围内部署了伊斯坎德尔-M移动战场导弹发射器。</p>
<p>乌克兰武装部队在每日早间更新中说：“然后敌人通过转移和集中更多部队来增加别尔哥罗德地区的部队人数。”</p>
<p>“根据现有信息，伊斯坎德尔-M发射器已部署在距乌克兰边境60公里处，”它说，但没有提供有关系统位置的更多细节。</p>
<p>伊斯坎德尔是一种机动弹道导弹系统，北约代号为SS-26 Stone，取代了苏联的飞毛腿导弹。它的两枚制导导弹射程可达500公里（300英里），可以携带常规弹头或核弹头。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>波兰对乌克兰的国防援助超过16亿美元</strong></p>
<p>波兰总理马泰乌什·莫拉维茨基表示，自俄罗斯入侵开始以来，波兰已向乌克兰提供了价值超过16亿美元的武器。</p>
<p>乌克兰人“正在为我们、为欧洲、为自由、为欧洲的和平而战。每个人都应该意识到这一点”，他告诉记者。</p>
<p>波兰媒体报道称，援助中包括40辆坦克和大约60辆装甲车。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>市议会：两枚俄罗斯导弹在敖德萨被拦截</strong></p>
<p>据官员称，乌克兰军队拦截了两枚飞往南部城市敖德萨附近港口的俄罗斯巡航导弹。</p>
<p>敖德萨市议会在Telegram上发文称，这些导弹“由一艘俄罗斯船只从黑海地区发射，正飞向皮夫登尼港”。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>部长：德国必须在不危及自身安全的情况下支持乌克兰</strong></p>
<p>德国财政部长克里斯蒂安·林德纳表示，柏林必须尽其所能帮助乌克兰赢得对俄罗斯的战争，但不得危及自身安全和北约的防御能力。</p>
<p>“我们必须竭尽全力帮助乌克兰获胜，但道德责任的极限是我们自身的安全、北约对领土的防御能力受到威胁，”林德纳周六在柏林的一次党内会议上说。</p>
<p>“但可能的事情，必须与我们的欧洲伙伴一起采取务实而迅速的措施，”他说。</p>
<p>林德纳表示，他赞成用重型武器支持乌克兰，但德国绝不能成为战争的一方。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>多伦多为举行乌克兰守夜活动</strong></p>
<p>加拿大多伦多市数十人为乌克兰举行了烛光守夜活动。</p>
<p>多伦多内森菲利普斯广场的机会于东正教圣周六举行。</p>

<p><strong>乌克兰称两名俄罗斯将军在赫尔松阵亡</strong></p>
<p>乌克兰军方表示，它摧毁了位于赫尔松的俄罗斯指挥所。赫尔松是一座在战争初期被俄罗斯军队攻陷的南部城市。</p>
<p>乌克兰军事情报机构发表声明称，指挥所周五遭到袭击，两名将军丧生，一名受重伤。</p>
<p>乌克兰总统顾问奥列克西·阿列斯托维奇在接受在线采访时表示，当指挥中心遭到袭击时，有50名俄罗斯高级军官在指挥中心。他说，他们的命运未知。</p>
