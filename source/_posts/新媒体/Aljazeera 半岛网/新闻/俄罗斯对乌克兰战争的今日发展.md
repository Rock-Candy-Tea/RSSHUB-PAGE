<li>非洲联盟主席表示，非洲大陆国家是俄乌冲突中的“经济受害者”，因为乌克兰无法向全球买家出口粮食及其他食品，从而加剧了一场迫在眉睫的粮食危机。</li>
<li>联合国乌克兰危机协调员警告称，这场战争“没有也不会产生赢家”。</li>
<li>联合国方面已经证实，已有超过9000位平民在这场冲突中丧生，并补充称，由于持续的敌对行动使统计死亡人数变得困难，真实的伤亡数据可能会“相当高”。</li>

<h3>以下是这场战争进入今日以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟禁止俄罗斯的大部分石油出口并制裁普京女友</strong></p>
<p>欧盟正式通过一项禁令，禁止从俄罗斯进口大部分石油。在与匈牙利的分歧持续数周之后，欧盟最终因乌克兰战争而对莫斯科实施了最为严厉的制裁。</p>
<p>欧盟官方发布的文本指出，这是自今年2月克里姆林宫对乌克兰发动入侵以来，欧盟这一27国集团对俄罗斯实施的第6轮制裁，其中包括将俄罗斯最大的银行“Sberbank”从全球SWIFT通讯系统中删除。</p>
<p>俄罗斯总统普京的“女友”、俄罗斯前体操运动员卡巴耶娃也被欧盟列入了冻结资产和禁止签证的黑名单，此外，还有涉嫌在乌克兰布查镇犯下战争罪行的俄罗斯军方人员。</p>
<figure id="attachment_287196" aria-describedby="caption-attachment-287196" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-287196" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/INTERACTIVE-Which-countries-have-sanctioned-Russia-May-17.png?w=770&resize=770%2C770" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-287196" class="wp-caption-text"><strong>俄罗斯-乌克兰战争</strong><br><strong>制裁俄罗斯的国家有哪些？</strong><br>至少有46个国家或地区对俄罗斯实施了制裁，或是承诺遵守美国和欧盟所实施的制裁。自今年2月22日以来，已有超过7782项制裁措施被强加给俄罗斯，从而使之成为全球受到制裁最多的国家。 (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>半岛电视台记者：基辅的“重大胜利”在于乌克兰“依然屹立”</strong></p>
<p>半岛电视台记者自乌克兰首都基辅报道称，“有两种方式来看待”这次进攻。</p>
<p>记者指出，“为了抵抗俄罗斯的入侵，乌克兰失去了20%的领土。”</p>
<p>他还补充称，“另一方面，在战争开始100天之后，乌克兰仍在战斗。我们必须记住一点，俄罗斯似乎原本想在(发动进攻后的)几天内占领基辅。”</p>
<p>“专家们估计，一旦基辅沦陷，乌克兰的其他地区也将在几周内相继沦陷，但现在，这个国家仍然屹立不倒，这本身就是在这场冲突中取得的重大胜利。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰授予谴责战争的俄罗斯高级记者公民身份</strong></p>
<p>据一名高级官员称，乌克兰已经授予著名的俄罗斯记者亚历山大·涅夫佐夫乌克兰公民身份，这名记者在谴责克里姆林宫入侵乌克兰后，与妻子逃离了俄罗斯。</p>
<p>俄罗斯一直寻求逮捕涅夫佐夫，并指控他在俄罗斯对乌克兰的“特别军事行动”中散布虚假信息。而涅夫佐夫及其妻子已于今年3月逃离俄罗斯。</p>
<p>乌克兰内政部长顾问安东·格拉什琴科表示，基辅当局已授予涅夫佐夫夫妇乌克兰公民身份。</p>
<p>涅夫佐夫在社交应用Telegram上发表声明，证实自己已经获得了乌克兰国籍。他还在声明中指出，俄罗斯的战争是一种罪行，而乌克兰是其罪行的受害者。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯指责德国“再度军事化”危及欧洲安全</strong></p>
<p>俄罗斯指责德国“再度军事化”的举措，会将欧洲安全置于不平衡的状态。需要指出的是，柏林方面正采取措施增加军费开支，以应对莫斯科的军事行动。</p>
<p>德国总理舒尔茨本周在德国媒体上发表评论称，柏林很快就会拥有北约欧洲成员国中规模最大的常规军队。</p>
<p>俄罗斯外交部发言人玛丽亚·扎哈罗娃表示，“我们认为，德国总理的言论再次证实，柏林已经为该国加速再度军事化的进程确定了方向。这怎么可能结束呢?这在历史上是众所周知的事情。”</p>
<p>今年2月，舒尔茨承诺大幅增加国防开支，并向德国军队注入1000亿欧元(约合1070亿美元)，这标志着德国自冷战结束后执行数十年的军队减员政策的重大转变。议员们预计将在3日就这项支出计划举行投票。</p>
<p>扎哈罗娃表示，“在有必要寻找机会减少共同威胁的时候，德国却恰恰相反，并且走上了加剧欧洲大陆军事政治局势的道路，投入数百亿欧元以增加武器的临界规模。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>非盟领导人告诉普京，非洲国家是乌克兰冲突的“经济受害者”</strong></p>
<p>非洲联盟主席敦促俄罗斯总统普京，切实考虑非洲国家因乌克兰冲突造成粮食短缺而遭受的痛苦。</p>
<p>塞内加尔总统麦基·萨勒在访问俄罗斯时告诉普京，“你要意识到，我们的国家即使远离战场，但在经济层面上，却是这场冲突的受害者”。</p>
<p>他还补充称，食品供应应当被排除在西方因乌克兰问题而对莫斯科实施的制裁之外。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基在战争的第100天誓争胜利</strong></p>
<p>乌克兰总统泽连斯基表示，乌克兰将赢得它与俄罗斯的战争。</p>
<p>“乌克兰武装部队就在这里。最重要的是，我们的人民也在这里。我们已经保卫乌克兰100天了”，泽连斯基在首都基辅中心的乌克兰总统办公室前向全国发表讲话时这样说道。</p>
<p>他还补充称，“胜利将会属于我们。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>瑞典国防部长敦促继续支持乌克兰</strong></p>
<p>瑞典国防部长胡尔特奎斯特表示，瑞典和其他欧洲国家必须准备向乌克兰提供包括武器运输在内的持续支持。</p>
<p>胡尔特奎斯特3日表示，“我认为这将是一场长期的冲突”，他是在一场外交政策集会——2022年布拉迪斯拉发全球金融安全论坛的间隙上向记者发表的言论。</p>
<p>“我们必须保证交货的可持续性，这样我们才能在之后提供更多。不同的国家必须相互补充。”</p>
<p>由于乌克兰冲突改变了斯堪的纳维亚地区的安全格局，瑞典正在寻求加入北约。瑞典2日宣布，将向乌克兰提供更多的经济援助和军事装备，包括反舰导弹、步枪和反坦克武器。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>图集：战争的100天</strong></p>
<p>经过几个月的紧张局势，俄罗斯军队在乌克兰的军事行动已经进入了第100天。</p>
<p>这场战争引发了二战以来欧洲最大规模的人口流动——超过1400万人流离失所，共690万乌克兰人越过边境，并以难民的身份生活在国外。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>德国称非洲粮食危机的罪魁祸首是普京而非西方</strong></p>
<p>德国外交部发言人表示，柏林强烈谴责俄罗斯总统普京试图制造的一种说法，即西方应该为非洲迫在眉睫的粮食危机负责。</p>
<p>德国政府发言人在例行新闻发布会上表示，“我们必须强调，世界部分地区存在饥荒的风险，部分国家被切断了粮食出口，造成这些事实的原因是俄罗斯的侵略战争，而不是西方实施的制裁。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>逃离战争：乌克兰的孩子们带走了什么？</strong></p>
<p>数以百万计的乌克兰儿童被迫背井离乡，并离开自己的家园、朋友、亲人和生活，以逃避战争。</p>
<p>在罗马尼亚，半岛电视台记者见到了这些逃离家园的孩子们，他们讲述了他们随身携带的珍贵物品，以及有关他们失去的东西的记忆。</p>
<p>点击<a href="https://www.aljazeera.com/features/longform/2022/6/3/escaping-war">此处</a>以了解他们的故事。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯将继续“行动”直至达成所有目标</strong></p>
<p>克里姆林宫发言人佩斯科夫表示，俄罗斯将继续其在乌克兰的“特别军事行动”，直至实现其所有目标。</p>
<p>佩斯科夫表示，“行动的主要目标之一，是保护顿涅茨克人民共和国和卢甘斯克人民共和国内的人们。我们已经采取措施以确保他们受到保护，并且取得了一定的成果”，而这两个国家是自行宣布成立的。</p>
<p>自2014年初以来，乌克兰的这两个分离地区一直处于受俄罗斯支持的分裂主义分子的控制之下。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>联合国官员：需要更多的谈判来允许俄罗斯的出口</strong></p>
<p>联合国乌克兰危机协调员阿明·阿瓦德表示，为恢复乌克兰粮食出口而设想的协议，还需要进行更多的谈判，其中包括达成允许俄罗斯出口的协议。</p>
<p>同时还担任联合国秘书长助理的阿明·阿瓦德在日内瓦举行的在线新闻发布会上表示，“俄罗斯原则上同意，但是还需要更多的谈判，以促进俄罗斯的出口。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰称俄罗斯试图将战争拖入“长期阶段”</strong></p>
<p>乌克兰国防部长表示，克里姆林宫正试图通过在该国南部被占领地区建造多层防御工事，而将这场战争拖入“长期阶段”。</p>
<p>“克里姆林宫正试图将这场战争拖入长期阶段”，这位国防部长通过视频方式向2022年布拉迪斯拉发论坛发表讲话时这样说道。</p>
<p>他还表示，俄罗斯军队在南部占领地区(主要是赫尔松地区)“建造多层防御工事，而不是继续向前推进”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯将传唤美国媒体负责人，并警告将采取“严厉措施”</strong></p>
<p>俄罗斯外交部表示，将于下周一召集在莫斯科的美国媒体负责人召开会议，并通知他们将采取严厉措施，以回应美国对俄罗斯媒体的限制措施。</p>
<p>俄罗斯外交部发言人玛丽亚·扎哈罗娃表示，“如果俄罗斯媒体——运营商与记者——在美国的工作变得非正常化，那么最严厉的措施将不可避免地随之而来。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>卢卡申科声称准备讨论经白俄罗斯运输乌克兰粮食的可能性</strong></p>
<p>据当地媒体报道，白俄罗斯总统卢卡申科表示，准备讨论经白俄罗斯运输乌克兰粮食的可能性。</p>
<p>白俄罗斯媒体援引卢卡申科的话报道称，“现在，所有人都在寻找后勤保障……好吧，我们可以谈谈。我们并不介意通过白俄罗斯进行运输，但是必须达成一些妥协。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰防长：北约应该考虑让乌克兰成为“事实上”的成员国</strong></p>
<p>乌克兰国防部长瑞兹尼科夫表示，当北约在即将于马德里举行的峰会上讨论其未来10年的战略时，它应当考虑给予乌克兰“事实上”的成员国资格，而不是“法律上”的成员国资格。</p>
<p>瑞兹尼科夫通过视频方式在全球安全委员会2022年布拉迪斯拉发论坛上表示，“我认为，如果我们谈论的是乌克兰在事实上而非法律上的北约成员国资格，那么这可能会是这项战略中的一个好主意。”</p>
<p>“乌克兰也将是这项战略中的一部分，因为我们也是欧洲东侧的一部分，我们是北约国家的东翼，也是欧盟的东翼。我认为，这对所有国家来说，都是一种双赢的局面。”</p>
<p>在俄罗斯发起的军事行动期间，北约盟国向乌克兰提供了价值数十亿美元的武器。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰特使称土耳其也在购买被盗粮食的国家之列</strong></p>
<p>乌克兰驻安卡拉大使指责土耳其和其他一些国家，购买了俄罗斯在入侵乌克兰期间从该国窃取的粮食。</p>
<p>这位大使还向记者们表示，他已经向土耳其当局和国际刑警组织寻求帮助，以调查涉嫌通过土耳其水域运输这些粮食的人员。</p>
<p>安卡拉和莫斯科都没有立即对他的言论作出回应，半岛电视台记者目前无法独立证实这一说法。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
