<li>随着莫斯科继续推进以完全控制东南部港口城市，乌克兰官员警告马里乌波尔的情况将回到“中世纪”。</li>

<p>以下是所有最新更新：</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>活动人士：5名克里米亚穆斯林被判最高14年监禁</strong></p>
<p>一名社区活动人士表示，俄罗斯军事法庭周四对五名来自被吞并的克里米亚的穆斯林男子因他们加入所谓的“伊斯兰”组织而处以刑法。</p>
<p>她告诉半岛电视台，顿河畔罗斯托夫市南区军事法庭判处比利亚尔·阿迪洛夫14年监禁，而伊泽特·阿卜杜拉耶夫、托菲克·阿卜杜勒加齐耶夫、弗拉德伦·阿卜杜勒卡德罗夫和梅吉特·阿卜杜拉赫马诺夫被判12年有期徒刑。</p>
<p>“俄罗斯媒体不会对此进行报道，法院也不会公布交给律师的（量刑信息），” 穆明·萨利瓦说。</p>
<p>这些人被指控是伊扎布特（Hizb-ut Tahrir）的成员，该组织在乌克兰自由活动，但在俄罗斯被禁止。</p>
<p>克里姆林宫多年来一直试图压制克里米亚鞑靼人的语言和文化，克里米亚鞑靼人是一个以穆斯林为主的少数民族，曾统治黑海半岛，并强烈抵制2014年吞并克里米亚。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>在乌克兰控制运输系统之前，关键的天然气路线不会重新开放</strong></p>
<p>该系统运营商GTSOU的负责人告诉路透社，在基辅获得对其天然气运输系统的控制权之前，乌克兰不会重新开放暂停的从俄罗斯到欧洲客户的Sokhranovka天然气运输路线。</p>
<p>天然气管道穿过乌克兰的卢甘斯克地区，该地区的一部分自2014年以来一直处于俄罗斯支持的分裂分子的控制之下。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯警告对芬兰加入北约作出回应</strong></p>
<p>在赫尔辛基领导人支持加入以美国为首的联盟后，俄罗斯发出警告称，它必须采取“军事技术”措施来应对邻国芬兰加入北约。</p>
<p>克里姆林宫发言人德米特里·佩斯科夫对记者说：“北约的扩张和联盟接近我们的边界并没有使世界和我们的大陆更加稳定和安全。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>五角大楼：将芬兰纳入北约并不难</strong></p>
<p>五角大楼发言人约翰·柯比告诉美国媒体MSNBC，将芬兰纳入北约并不难，它的加入将具有历史意义。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟冯德莱恩谴责俄罗斯“对乌克兰的野蛮战争”</strong></p>
<p>欧盟委员会主席乌尔苏拉·冯德莱恩在日本东京表示，由于俄罗斯入侵乌克兰，俄罗斯是对国际秩序的“最直接威胁”。</p>
<p>她在和欧洲理事会主席查尔斯·米歇尔与日本首相岸田文雄和会面后表示，俄罗斯“今天对乌克兰的野蛮战争及其与中国的令人担忧的协议是对世界秩序的最直接威胁”。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>联合国：俄罗斯对大多数平民伤亡负责</strong></p>
<p>联合国人权事务负责人表示，她的办公室发现俄罗斯军队及其附属武装团体应对乌克兰战争期间大多数平民死亡负责。</p>
<p>“根据我们的信息，虽然此类事件可归咎于冲突双方，但大多数伤亡似乎是由俄罗斯武装部队及其附属武装团体造成的，”米歇尔·巴切莱特告诉人权理事会。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>耐克终止与俄罗斯足球俱乐部的合同</strong></p>
<p>俱乐部表示，服装巨头耐克已经终止了与莫斯科斯巴达克的赞助协议，因为该球队下赛季将不会参加欧洲比赛。</p>
<p>欧洲足球管理机构欧足联此前宣布，由于莫斯科入侵乌克兰，2022-23赛季，俄罗斯球队失去了参加俱乐部比赛的资格。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：一名乌克兰人在哈尔科夫死亡，三人受伤</strong></p>
<p>俄罗斯炮击在哈尔科夫东北部地区造成1人死亡，3名平民受伤。</p>
<p>奥列·西尼胡博夫在Telegram上说，该地区首府哈尔科夫西北部的德哈切村遭到了炮击袭击。</p>
<p>最近几天，乌克兰人解放了哈尔科夫周围的几个主要城镇，并将俄罗斯军队推向边境。</p>
<p><em>曼苏尔·米罗瓦廖夫在基辅报道。</em></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>州长：乌克兰的打击在俄罗斯西部造成1人受伤，7人受伤：</p>
<p>一名地区州长声称，乌克兰的一次打击在俄罗斯与乌克兰接壤的一个村庄造成1名俄罗斯人死亡，另外7人受伤。</p>
<p>维亚切斯拉夫·格拉德科夫在Telegram上说，周三晚间，打击还摧毁了别尔哥罗德州索洛基村的17座房屋和6辆汽车。</p>
<p>他称，此次袭击是“自四月份炮击开始以来最可怕的一次”。</p>
<p>俄罗斯指责乌克兰袭击了俄罗斯西部几个存放燃料和武器的村庄，但基辅经常否认这些说法。</p>
<p><em>曼苏尔·米罗瓦廖夫在基辅报道。</em></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>瑞典部长：将考虑芬兰的北约评估</strong></p>
<p>瑞典外交部长安·林德表示，瑞典在决定申请加入时将考虑芬兰对北约的评估。</p>
<p>“芬兰是瑞典最亲密的安全和国防合作伙伴，我们需要考虑芬兰的评估，”林德在推特上写道。</p>

<p><strong>俄罗斯天然气工业股份公司：从俄罗斯通过乌克兰到欧洲的天然气供应下降</strong></p>
<p>俄罗斯天然气工业股份公司表示，一条穿过乌克兰达到欧洲的关键路线的天然气输送量下降了三分之一。</p>
<p>据国际文传电讯社报道，俄罗斯天然气工业股份公司表示，周四过境乌克兰的总供应量为5060万立方米，而前一天为7200万立方米。</p>
<p>乌克兰的管道运营商GTSOU宣布，由于俄罗斯占领军正在干扰运营，它从周三开始停止在Sokhranivka中转站的天然气运输。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>联合国人权事务负责人：乌克兰的侵权行为可能构成战争罪</strong></p>
<p>联合国人权事务负责人表示，最近几周在基辅附近发现了数千具尸体，并补充说，自俄罗斯入侵以来，许多被报告的虐待行为正在等待核实，它们可能构成战争罪。</p>
<p>“非法杀戮的规模，包括基辅北部地区的即决处决迹象，令人震惊，”米歇尔·巴切莱特在联合国人权理事会的视频讲话中说。</p>
<p>人权高专办将于周四决定是否对2月和3月在基辅和其他地区发生的事件进行正式调查。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯军队试图“封锁”亚速钢铁厂的地下通道</strong></p>
<p>一名乌克兰官员表示，俄罗斯军队试图占领马里乌波尔被围困的亚速钢铁厂时，试图封锁这座巨大建筑群下方的地下通道。</p>
<p>在遭受两个多月袭击几乎被摧毁的亚速海港口城市，占地11平方公里（4.25平方英里）的亚速钢铁厂是乌克兰的唯一一个据点。</p>
<p>俄罗斯人的“主要目标是封锁地下通道系统的出口，这些出口被叛徒指出，”市政府官员彼得·安德里申科在Telegram上说。</p>
<p>然而，数百名躲在工厂里的乌克兰军人“冒着一切风险试图反击，”他补充说。</p>
<p><em>曼苏尔·米罗瓦廖夫在基辅报道。</em></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>时间表：俄罗斯在乌克兰战争进入第11周</strong></p>
<p>俄罗斯入侵乌克兰的第11周，基辅的军队发起了反攻。乌克兰军队已经收复了乌克兰东部第二城市哈尔科夫北部和东部的城镇。</p>
<p>数百名平民已从俄罗斯占领的港口城市马里乌波尔被围困的亚速钢铁厂撤离。由于匈牙利对潜在的石油禁运感到愤怒，欧盟难以就第六轮制裁达成一致。</p>

<li><strong>更新</strong></li>
</ul>
<p><strong>州长：俄罗斯人在卢甘斯克河“喂鱼”</strong></p>
<p>地区州长说，俄罗斯军队试图穿越卢甘斯克东南部地区的一条战略河流，用他们的身体“喂鱼”。</p>
<p>俄罗斯军队一直试图在比洛戈里夫卡镇附近用浮桥架起横跨西维尔斯基顿涅茨河的桥梁。然而，乌克兰军队炮击了浮桥并击垮了渡河的俄罗斯部队。</p>
<p>俄罗斯人“试图再次渡河，但他们只能喂鱼，”谢尔盖·海代在Telegram上打趣道。</p>
<p>5月8日，俄罗斯在比洛戈里夫卡的炮击恐造成至少60名平民丧生。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>芬兰总统、总理赞成加入北约联盟</strong></p>
<p>芬兰总统和总理表示，他们的国家愿意加入北约，并将在本周晚些时候做出正式决定。</p>
<p>“芬兰必须毫不拖延地申请加入北约，”紹利·尼尼斯托总统和桑娜·马林总理在一份联合声明中说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官方：在乌克兰排雷可能需要10年</strong></p>
<p>据报道，一名官员称，清除俄罗斯军队在乌克兰8个地区埋设的所有地雷最长需要10年的时间。</p>
<p>“根据国际经验判断，排雷将需要5到10年的时间，”乌克兰国家通讯社周四援引国家紧急情况服务处的奥列·邦达尔的话说。</p>
<p>乌克兰军方和官员说，俄罗斯人将地雷、诱杀装置和爆炸装置留在公立学校、房屋和尸体旁边。</p>
<p><em>曼苏尔·米罗瓦廖夫在基辅报道。</em></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>“乌克兰人民”获得大西洋理事会奖</strong></p>
<p>大西洋理事会向乌克兰人民颁发了杰出的领导奖，这是该理事会首次授予整个国家这一荣誉。</p>
<p>据理事会称，意大利总理马里奥·德拉吉和乌克兰歌手兼词曲作者贾马拉也属于今天的获奖者，他们被视为“跨大西洋关系的支柱”。</p>
<p>泽连斯基在视频中感谢了理事会，称“勇气是我们的民族性格特征”，并仅列出了在战争期间做出牺牲的少数人的名字。</p>
<p>其中包括：维塔利·斯卡昆“为了阻止俄罗斯坦克纵队而牺牲自己炸毁了一座桥”；“21岁的维塔利·萨皮洛中尉，摧毁了敌人的30件装备并死于空袭”；“马里乌波尔的护理人员塞尔吉奥·乔诺布里维茨，自战争开始以来已经值班22天”；“11岁的马克西姆什么都没有告诉母亲，离家出走，要求参与领土防御。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：俄罗斯从哈尔科夫撤军表明其无法占领关键城市</strong></p>
