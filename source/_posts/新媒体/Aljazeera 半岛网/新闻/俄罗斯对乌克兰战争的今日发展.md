<h3>以下是这场战争进入今日以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报道：俄罗斯法院因报道乌克兰新闻对某电台罚款32.5万美元</strong></p>
<p>据俄罗斯国际文传电讯社报道，莫斯科一家法院对受美国支持的自由欧洲广播电台处以2000万卢布(约合325214美元)的罚款，原因是该电台散布了有关俄罗斯在乌克兰军事行动的“虚假”内容。</p>
<p>今年3月初，俄罗斯通信监管机构屏蔽了该电台以及其他部分外国媒体的网站。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基：乌克兰拥有“机会”在北顿涅茨克发动反击</strong></p>
<p>乌克兰总统泽连斯基表示，俄罗斯军队在北顿涅茨克的战斗中拥有人数上的优势，但是基辅军队仍然拥有“机会”发动反击。</p>
<p>泽连斯基在基辅举行的新闻发布会上对记者们表示，如果俄罗斯在顿巴斯东部地区取得突破，那么乌克兰的局势将会变得非常困难。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰官员称有数十名记者在俄罗斯的军事行动期间丧生</strong></p>
<p>乌克兰文化和信息政策部长特卡琴科表示，在俄罗斯的攻势中，至少已有32名记者遇害。</p>
<p>在乌克兰庆祝一年一度的记者日时，特卡琴科称，所有仍坚持在这个饱受战争蹂躏的国家工作的记者都是“英雄”。</p>
<p>“永远记住我们工作在新闻前线的战士们。现在所有进行全天候工作的记者都是英雄。感谢你们所有人，我们的新闻战线非常强大。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>市长：乌克兰可以击退俄罗斯对北顿涅茨克的攻击</strong></p>
<p>北顿涅茨克市长表示，乌克兰已经在当地集中了足够的兵力以击退俄罗斯的攻击，他还补充称，在激烈的巷战中，双方都不准备撤退。</p>
<p>他向乌克兰电视台表示，“(我们)在当地集中了足够的力量和资源，以击退对这座城市的攻击。”</p>
<p>俄罗斯国防部发言人莫图赞扬克在新闻发布会上表示，俄罗斯在夺取卢甘斯克地区最大城市北顿涅茨克的过程中，不会空出任何军队或装备。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>一项令人痛心的任务：挖掘乌克兰死者以进行战争罪调查</strong></p>
<p>在基辅地区当局调查俄罗斯军队可能犯下的战争罪行时，志愿者们肩负着挖掘尸体的艰巨任务。</p>
<p>点击<a href="https://www.aljazeera.com/features/2022/6/6/exhuming-ukraines-dead-for-war-crimes-investigations">此处</a>以了解更多内容。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>拉夫罗夫：在远程武器的威胁下，俄罗斯仍将击退乌克兰军队</strong></p>
<p>俄罗斯外长拉夫罗夫表示，对于西方国家向乌克兰运送远程武器一事，莫斯科将作出的回应，就是将乌克兰军队击退，并进一步使之远离俄罗斯边境。</p>
<p>他在新闻发布会上表示，“这些系统的射程越远，我们就会把纳粹主义分子从可能会对讲俄语的人员和俄罗斯联邦构成威胁的战线上击退得更远。”拉夫罗夫引用了俄罗斯官方一再重复的论调，即将乌克兰当局称为新纳粹主义势力和极端民族主义势力，而基辅方面认为，这些标签被俄罗斯方面用来宣传这场冲突的正当性。</p>
<p>拉夫罗夫指出，保加利亚、北马其顿和黑山阻止其乘坐的飞机前往塞尔维亚的举动是“前所未有的”，他还补充说，他尚未收到对这些决定的解释。</p>
<p>他还表示，他将邀请塞尔维亚外长到莫斯科访问。拉夫罗夫补充称，“最重要的是，没有人能够破坏我们与塞尔维亚之间的关系。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基称英国首相在通话中证实了新的军事援助计划</strong></p>
<p>乌克兰总统泽连斯基表示，在他与英国首相鲍里斯·约翰逊进行的电话通话中，后者证实了英国将向乌克兰提供的新一揽子军事援助。</p>
<p>泽连斯基在推特上补充称，他和约翰逊正在“寻求避免粮食危机和疏通(乌克兰)港口的方案”，他指的是俄罗斯对乌克兰实施的海上封锁，这举使得乌克兰无法出口其大部分农产品。</p>
<p>在泽连斯基发表上述言论之前，英国在6日早些时候表示，英国承诺向乌克兰提供的多管火箭发射系统将为该国抵御俄罗斯入侵的努力带来“显著的能力提升”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称与美国进行核谈判是必要的，但是目前“不太可能”</strong></p>
<p>克里姆林宫表示，俄罗斯有兴趣与美国就核武器问题进行谈判，但他警告称，这类谈判在目前不太可能进行。</p>
<p>克里姆林宫发言人佩斯科夫向记者表示，“鉴于我们看到的结构性变化，我们重视并且相信能够在这个话题上继续谈判和讨论……整个世界都需要这类谈判。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>国际原子能机构总干事正在处于俄罗斯控制下的扎波罗热核电站执行任务</strong></p>
<p>国际原子能机构总干事表示，该联合国核监督机构正在为一个国际专家组“制定模式”，希望向处于俄罗斯控制下的乌克兰扎波罗热核电站派遣专家。而该核电站是欧洲最大的核电站。</p>
<p>“我们正在制定派遣这样一个特派团的模式，而其他的考虑不应该阻止这项重要的国际任务的进行”，国际原子能机构总干事拉斐尔·格罗西在一份向由35个成员国组成的机构理事会提交的声明中这样说道，他还补充称，乌克兰已经提出了这样的要求。</p>
<p>“乌克兰扎波罗热核电站仍然处于俄罗斯军队的控制之下。我多次对乌克兰管理层以及员工在极其紧张和富有挑战的工作条件下运营这家核电站的情况提出严重关切”，他还表示，这种情况对当地的“安全、保障和安保措施构成了明确而现实的风险”。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称对其外长专机实施禁飞是一种“敌对行动”</strong></p>
<p>莫斯科谴责3个东欧国家对俄罗斯外长拉夫罗夫前往塞尔维亚的飞机关闭领空，是一项“敌对行动”。</p>
<p>塞尔维亚周边国家——保加利亚、北马其顿和黑山，对俄罗斯外交部长6日飞往贝尔格莱德的一架官方飞机关闭了领空。</p>
<p>克里姆林宫发言人佩斯科夫向记者们表示，这样的行动可能会给高层外交会议的时间表造成问题。但他补充称，这并不会阻止莫斯科与友好国家保持联系。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>向乌克兰提供武器：哪些国家提供了什么?</strong></p>
<p>自2月24日俄罗斯对乌克兰发起军事行动以来，许多国家都向乌克兰提供了武器。</p>
<p>点击<a href="https://www.aljazeera.com/news/2022/6/5/weapons-to-ukraine-which-countries-sent-what">此处</a>以查看有关这些武器供应的情况。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰军队称俄罗斯军队沿着顿涅茨克的整条前线发动进攻</strong></p>
<p>乌克兰军队表示，俄罗斯军队沿着该国东南部顿涅茨克地区的整条前线发动进攻，并在邻近的卢甘斯克地区发动进攻。</p>
<p>乌克兰武装部队发言人奥列克桑德尔·什图潘表示，莫斯科军队正在继续施压，要求完全控制卢甘斯克的北顿涅茨克市，还在斯拉维扬斯克和利西昌斯克发起进攻。</p>
<p>什图潘还指责俄罗斯军队，将乌克兰北部的切尔尼哈夫和东北部苏米地区的民用基础设施作为袭击目标。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>美国驻俄罗斯大使表示，莫斯科和华盛顿必须保持大使馆开放</strong></p>
<p>美国驻莫斯科大使表示，尽管乌克兰战争引发了危机，但俄罗斯和美国应该力求各自在对方国家的大使馆保持开放，因为世界上两个最大的拥核国必须继续进行谈判。</p>
<p>沙利文向俄罗斯塔斯社表示，华盛顿和莫斯科不应该简单地切断外交关系。</p>
<p>沙利文强调，“我们必须保持彼此沟通的能力。”塔斯社用俄语报道了沙利文的讲话，而路透社将其翻译成英语。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄乌战争：拉夫罗夫“被迫取消塞尔维亚之行”</strong></p>
<p>根据媒体报道，俄罗斯外长拉夫罗夫取消了对塞尔维亚的访问，因为塞尔维亚周边国家对他所乘坐的飞机关闭了领空。</p>

<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰南部城市尼古拉耶夫传来爆炸声</strong></p>
<p>乌克兰市长奥列克桑德尔·森科维奇在Telegram上发帖称，乌克兰南部城市尼古拉耶夫响起了爆炸声。</p>
<p>但是森科维奇又补充称，目前还没有关于这起爆炸事件的进一步消息。</p>

<p><strong>乌克兰表示，该国在北顿涅茨克的局势“略有恶化”</strong></p>
<p>卢甘斯克地区州长海岱表示，在北顿涅茨克作战的乌克兰军队的局势“有所恶化”。</p>
<p>海岱向国家电视台表示，“我们的防守方成功地发动了反击，并且持续了一段时间，他们几乎解放了半个城市。但是现在，我们的情况又恶化了一些。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄州长：俄罗斯库尔斯克地区遭到炮击</strong></p>
<p>与乌克兰接壤的俄罗斯库尔斯克州州长罗曼·斯达洛沃特表示，当地的泰特基诺村在6日上午遭到了炮击。</p>
<p>斯达洛沃特在Telegram上表示，“没有人员受伤或死亡。主要的攻击发生在当地的一座桥梁上。”</p>
<p>俄罗斯几个边境地区的州长多次指责乌克兰向他们的定居点开火。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰：俄罗斯损失了超过3.1万名士兵</strong></p>
<p>乌克兰军方表示，俄罗斯军队自其2月24日入侵乌克兰以来，已经损失了31250名人员。</p>
<p>乌克兰武装部队总参谋部还表示，俄罗斯方面损失了1386辆坦克、3400辆装甲车辆、690套火炮系统、209套多管火箭发射系统、551枚巡航导弹、96套防空系统、211架飞机、176架直升机和13艘船只。</p>
<p>半岛电视台记者无法独立核实这些数据。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：哈尔科夫地区共6人因炮击受伤</strong></p>
<p>俄罗斯军队5日继续炮击乌克兰东北部的哈尔科夫地区，并造成6人受伤。</p>
<p>哈尔科夫地区州长奥列格·辛涅胡波夫表示，俄罗斯的炮击导致楚胡伊夫镇（Chuhuiv）一名52岁的男性受伤，另外，科罗博奇基涅村和查卡洛夫斯克村有两人受伤，马利诺夫卡村有两人受伤，还有巴拉克利亚镇的一位老年妇女受伤。</p>
