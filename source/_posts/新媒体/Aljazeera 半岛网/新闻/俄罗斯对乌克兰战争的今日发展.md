<div>   
<p>乌克兰总统泽连斯基在11日表示，俄罗斯即将实施的计划，是摧毁该国其他地区并攻击其他邻国，他还强调，乌克兰当前承受的是一场俄罗斯已为之筹备十年的战争。</p>
<p>另一方面，克里姆林宫在评论将瑞典和芬兰纳入北约（北大西洋公约组织）的努力时表示，该联盟的扩大不会给欧洲带来稳定，也不会为欧洲大陆带来更多的安全。</p>
<p>预计欧盟国家外交部长将在11日讨论有关对俄罗斯实施第6套制裁措施的问题，另一方面，美国总统乔·拜登打算与印度总理纳伦德拉·莫迪举行视频会议，以讨论乌克兰战争的事态发展。</p>
<p><strong>以下是这场战争进入第47天以来的最新进展：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>克罗地亚外交部：驱逐18名俄罗斯外交官员和俄罗斯大使馆内的6名行政人员。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>加拿大外交部：对33家俄罗斯国防企业实施新的制裁。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯外交部长：</strong></p>
<ul>
<li>在下一轮会谈之前不会停止在乌克兰的军事行动。</li>
<li>乌方将继续挑衅俄罗斯，而莫斯科将以事实予以回应。</li>
<li>没有理由阻止与乌克兰之间的谈判。</li>
<li>俄罗斯在乌克兰的特别军事行动旨在结束美国的全球霸权。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>奥地利外交部长：俄罗斯总统在道德上输掉了这场战争。</p>
<p>欧盟委员会：由于不符合国际安全标准，有20家俄罗斯航空公司被列入欧盟航空安全黑名单。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>法新社援引乌克兰海军的消息报道：我们正在为马里乌波尔的最后一战作准备。</p>
<p>俄新社援引俄罗斯国防部的消息报道：我们的部队已经控制了卡缅卡地区，这是乌克兰军队在该国东部伊久姆地区的防御线。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>克里姆林宫评论瑞典和芬兰加入北约的努力称：北约的扩张不会给欧洲带来稳定，也不会给欧洲大陆带来更多的安全。</p>
<p><strong>克里姆林宫：</strong></p>
<ul>
<li>俄罗斯总统与奥地利总理在莫斯科的会谈将以闭门形式举行，双方将讨论乌克兰问题和能源问题。</li>
<li>俄罗斯有能力履行其债务义务。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>顿涅茨克分离主义领导人：我们将进一步加强行动以解放该地区。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-271308" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/image-2-17.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰总统泽连斯基：</strong></p>
<ul>
<li>乌克兰当前承受的是一场俄罗斯已为之筹备十年的战争。</li>
<li>俄罗斯军队破坏了乌克兰，并实施了抢劫和强奸的行为。</li>
<li>俄罗斯希望通过这场战争征服乌克兰，然后再攻击我们的其他邻国。</li>
<li>俄罗斯下一步的计划是摧毁我国的其他地区，然后再攻击我们的其他邻国。</li>
<li>自从对我们发动战争以来，俄罗斯军队已经摧毁了超过900所大学和超过300家药店。</li>
<li>俄罗斯袭击了那些集中了来自全国各地的平民的地点。</li>
<li>理智必须赢得这场战争的胜利，我们必须迫使俄罗斯寻求和平。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>德国外交部长：</strong></p>
<ul>
<li>乌克兰需要更多的武器，尤其是重型武器。</li>
<li>有迹象表明乌克兰发生过战争罪行，应当为此提供证据。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>哈尔科夫应急管理局：由于​​俄罗斯的轰炸，该市在过去24小时内共有6位平民丧生。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰副总理：开放9条人道主义走廊，以便从包括马里乌波尔在内的各个地区撤离平民。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-271411" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/image-8-3.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯国防部：</strong></p>
<ul>
<li>一个欧洲国家送给乌克兰的S-300系统在第聂伯罗彼得罗夫斯克郊区遭到破坏。</li>
<li>两架乌克兰的苏-25战斗机在伊久姆市附近被击落。</li>
<li>在过去的24小时内，我们已经袭击了78个乌克兰军事设施。</li>
<li>一架乌克兰军用米-24直升机在赫尔松被击落。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰外交部长：莫斯科正在竭尽所能地破坏我国的武器供应。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟外交政策负责人：</strong></p>
<ul>
<li>欧盟国家外交部长11日将开会讨论如何加强对乌克兰人民的援助。</li>
<li>制裁俄罗斯的问题将一直摆在桌面上。</li>
<li>在未来几天内，顿巴斯地区的战斗将会升级。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总检察长：自这场战争开始以来，已有183名儿童遇难、938所教育机构受损，其中87所教育机构遭到了彻底的摧毁。</p>

<ul style="list-style-type:square">

<ul>
