<p><strong>以下是最新更新：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>亲俄分裂分子维持对外国人的死刑判决</strong></p>
<p>顿涅茨克的亲俄分离主义领导人杰尼斯·普希林表示，他不会改变两名英国人和一名摩洛哥人因为乌克兰军队作战而被处于死刑的判决。</p>
<p>“他们来乌克兰是为了钱而杀害平民。这就是为什么我认为不存在任何减轻或修改判决的条件，”普希林告诉记者。</p>
<p>普希林说，法院对这三名战士“做出了完全公平的惩罚”。</p>
<p>他还指责英国首相鲍里斯·约翰逊无视他们的命运，没有与分离主义当局取得联系。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>前指挥官：乌克兰亚速营战士的遗体仍在马里乌波尔</strong></p>
<p>乌克兰亚速国民警卫队前指挥官说，在南部城市马里乌波尔的亚速钢铁厂围攻中丧生的数十名乌克兰战士的遗体仍有待找回。</p>
<p>马克西姆·若林说，根据最近的交换条款，大约220具亚速钢铁厂遇难者的遗体已被送往基辅，但“还有很多遗体仍留在马里乌波尔”。</p>
<p>“关于进一步交换的谈判仍在继续，以将所有遗体送回家。绝对必须归还所有遗体，这是我们将要做的事情，”若林在Telegram上发布的视频中补充道。</p>
<p>他说，三分之一的死者属于亚速营，而其他人则属于边境巡逻队、海军以及警察部队。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：俄罗斯利用其兵力和火炮优势夺取领土</strong></p>
<p>英国国防部表示，俄罗斯正在利用其在兵力和火炮方面的优势，逐步占领乌克兰北顿涅茨克及其周边地区的领土。</p>
<p>英国国防部在推特上发布的最新情报更新中表示，俄罗斯最近几周可能已经开始准备从一些战斗编队部署第三营。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯“摧毁”乌克兰存放西方武器的大型仓库</strong></p>
<p>国际文传电讯社援引俄罗斯国防部的话称，俄罗斯军队发射了“口径”巡航导弹，摧毁了乌克兰捷尔诺波尔州一个储存美国和欧洲武器的大型仓库。</p>
<p>国际文传电讯社报道称，俄罗斯军队还在乌克兰东部顿涅茨克和哈尔科夫附近击落了三架乌克兰苏-25战斗机。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：俄罗斯在乌克兰西部的打击造成至少22人受伤</strong></p>
<p>据捷尔诺波尔州州长称，俄罗斯袭击乌克兰城镇乔特基夫造成至少22人受伤，这是该国西部罕见的袭击事件。</p>
<p>“昨天19:46（格林威治标准时间16:45），乔特基夫被4枚从黑海发射的导弹击中，”沃拉迪米尔·特鲁什在Facebook的帖子中说，并补充到，所有伤员都住院了。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯的麦当劳以新的所有权重新开业，更名为“Vkusno & tochka”</strong></p>
<p>该公司表示，曾属于麦当劳公司的15家餐厅将于周日在莫斯科以新的所有权和新名称“Vkusno & tochka”重新开业，新名称意思是“美味，就是这样”。</p>
<p>Vkusno & tochka 说，另外50家餐厅将于周一营业。</p>
<p>周日标志着俄罗斯快餐爱好者的新曙光，先前由广受欢迎的西方快餐连锁店经营的餐厅以新品牌和更名的汉堡重新开业，这一天距麦当劳在莫斯科首次开业已有30多年。</p>

<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯通过乌克兰向欧洲的天然气出口保持不变</strong></p>
<p>俄罗斯天然气工业股份公司表示，周日通过Sudzha泵站过境乌克兰向欧洲供应的天然气为4190万立方米 (mcm)，与周六持平。</p>
<p>俄罗斯天然气工业股份公司表示，通过另一个主要泵站Sokhranovka供应天然气的申请被乌克兰拒绝。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：北顿涅茨克工厂仍在乌克兰军队控制下</strong></p>
<p>据该地区州长称，乌克兰仍然控制着位于北顿涅茨克的阿佐特化工厂，数百名平民正在那里避难。</p>
<p>“阿佐特没有被封锁，工厂旁边的街道上正在进行战斗，”谢尔盖·海代在乌克兰的电视上说。</p>
<p>他补充说，他预计俄罗斯军队在周日或周一竭尽全力试图占领这座城市。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>德国总统批评前总理施罗德与俄罗斯的关系</strong></p>
<p>德国总统弗兰克-瓦尔特·施泰因迈尔批评了前总理格哈德·施罗德与俄罗斯的商业关系。</p>
<p>施罗德在担任总理期间所做的大部分事情都因这一行为而黯然失色，施泰因迈尔告诉《图片报》小报的周日版。</p>
<p>“格哈德·施罗德与俄罗斯能源公司的合作给我们国家，尤其我们的东欧邻国，留下了疑问，”施泰因迈尔说。</p>
<p>施泰因迈尔过去曾与社民党同事施罗德密切合作，并于1999年至2005年担任施罗德总理办公室主任。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰恢复被占领核电站与国际原子能机构之间的互联网连接</strong></p>
<p>乌克兰国家核能发电公司表示，它帮助恢复了国际原子能机构与被俄罗斯军队占领的乌克兰扎波罗热核电站服务器之间的互联网连接。</p>
<p>该公司在一份声明中表示，与工厂服务器的连接于5月30日中断，但已于6月10日恢复，从而使国际原子能机构能够恢复对工厂核材料的监测数据。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>《图片报》周日版：朔尔茨、马克龙和德拉吉将在G7峰会举行前访问基辅</strong></p>
<p>据德国报纸《图片报》的周日版报道，德国总理奥拉夫·朔尔茨将在6月底七国集团峰会举行前与法国总统和意大利总理一起前往基辅。</p>
<p>该报引用了法国和乌克兰政府的消息来源。</p>
<p>德国政府发言人告诉路透社：“我们无法证实这一点。”在路透社的询问下，爱丽舍宫拒绝证实这一信息。意大利政府没有立即回应置评请求。</p>
<p>自2月莫斯科入侵乌克兰以来，三位领导人都没有去过基辅。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>斯里兰卡总理表示对购买俄罗斯石油持开放态度</strong></p>
<p>斯里兰卡可能被迫从俄罗斯购买更多石油，新上任的总理表示，在史无前例的经济危机中，该国拼命寻找燃料。</p>
<p>拉尼尔·维克拉马辛哈告诉美联社通讯社，他将首先寻找其他来源，但愿意从莫斯科购买更多原油。西方国家在很大程度上切断了从俄罗斯进口的能源，以保持对俄罗斯发动乌克兰战争的制裁。</p>
<p>“如果我们可以从任何其他来源获得，我们将从那里获得。否则（我们）可能不得不再次转向俄罗斯，”他说。</p>
<p>“当然，我们将海湾国家视为我们的主要供应。”</p>
<p>维克拉马辛哈说，俄罗斯也向斯里兰卡提供了小麦。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>中国支持俄罗斯与乌克兰之间的谈判</strong></p>
<p>中国国防部长魏凤和表示，北京对乌克兰的事件感到难过，并表示他支持莫斯科和基辅之间进行和平谈判。</p>
<p>他还表示，中国反对西方向乌克兰提供武器以及对俄罗斯实施制裁。</p>
<p>“这场危机的根本原因是什么？这背后的主谋是谁？谁输得最多？谁将获得最大的收益？谁在促进和平，谁在火上浇油？我想我们都知道这些问题的答案，”他在新加坡香格里拉对话会上说。</p>

<p><strong>日本向敖德萨捐赠净水器</strong></p>
<p>乌克兰驻日本大使谢尔盖·科尔孙斯基表示，东京正在向乌克兰南部城市敖德萨捐赠33台便携式净水器。</p>
<p>“在不久的将来，33台便携式净水器将在紧急情况下被送往敖德萨。我们希望敖德萨与尼古拉耶夫分享（这些设备）。我想相信它们不会被需要，但我们应该拥有它们以防万一，”他在Facebook上写道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰第一夫人在立陶宛开设难民中心</strong></p>
<p>乌克兰第一夫人奥莱娜·泽伦斯卡在立陶宛首都维尔纽斯开设了一个乌克兰难民中心。</p>
<p>“我们希望因战争而被迫来到立陶宛的乌克兰人拥有一个真正的故乡。因此，该中心将成为可以解决任何问题的地方，帮助或沟通。人们能够在这里学习，度过休闲时光，参加文化活动或教育课程，”泽伦斯卡通过来自基辅的视频链接说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基声称俄罗斯32000名人阵亡</strong></p>
<p>泽连斯基总统表示，北顿涅茨克的激烈巷战仍在继续，他为数周来阻止俄罗斯前进的乌克兰捍卫者感到自豪。</p>
<p>他说，自2月24日战争开始以来，俄罗斯人遭受了重大损失。</p>
<p>“你们还记得俄罗斯是如何希望在五月初占领整个顿巴斯的吗？现在是战争的第108天。顿巴斯坚持了下来。占领者遭受的损失，包括在这一地区的损失是极其巨大的。今天的俄罗斯军队总共有大约32000人阵亡。这为了什么？这给你俄罗斯带来了什么？”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>五角大楼负责人斥俄罗斯的“帝国胃口”</strong></p>
<p>美国国防部长劳埃德·奥斯汀说，莫斯科对乌克兰的入侵“是压迫者践踏保护我们所有人的规则”。</p>
<p>奥斯汀在访问亚洲时说：“当大国认为他们的帝国胃口比他们和平邻国的权利更重要时，就会发生这种情况。这是一个可能的混乱和动荡世界的预演，我们谁都不想生活在其中。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>分离主义者声称乌克兰捍卫者“将平民扣为人质”</strong></p>
<p>一些非战斗人员设法逃离饱受战争蹂躏的北顿涅茨克市的工业区，俄罗斯军队在那里与乌克兰捍卫者作战。</p>
<p>自称卢甘斯克人民共和国的俄罗斯支持的代表罗迪翁·米罗什尼克说，平民开始离开工厂，但他没有说有多少人。</p>
<p>“偶尔会发生交火。他们（乌克兰捍卫者）可能仍将数百名平民扣为人质，”他在网上发帖说</p>
<p>乌克兰表示，约有800人躲在阿佐特工厂下面，包括约 200 名员工和600名北顿涅茨克居民。</p>
<p>米罗什尼克估计有300至400名乌克兰战士被封锁在该地。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>“非常不准确”：俄罗斯使用苏联时期的弹药</strong></p>
<p>俄罗斯军队在试图占领乌克兰东部地区时，正在依靠会造成大量平民伤亡的旧武器。</p>
<p>英国国防部表示，俄罗斯轰炸机可能在乌克兰的地面攻击中发射了数十枚1960年代的重型反舰导弹。</p>
<p>报告称，当此类带有常规弹头的导弹用于地面攻击时，它们“非常不准确，因此可能造成严重的附带损害和人员伤亡”。</p>

<p><strong>随着与俄罗斯军队的激烈战斗继续，乌克兰军队“弹药不足”</strong></p>
<p>在乌克兰南部前线附近的尼古拉耶夫地区，该地区州长强调迫切需要国际军事援助。</p>
<p>“俄罗斯的军队更强大，他们有很多火炮和弹药。目前，这是一场炮火战争。我们的弹药已经用完了，”维塔利·金说。“欧洲和美国的帮助非常非常重要，我们只需要弹药来保卫我们的国家。”</p>
<p>泽连斯基总统在讲话中说，乌克兰“不能让世界把注意力从战场上发生的事情上转移开”。</p>

<p><strong>乌克兰军队：俄罗斯准备进攻斯拉维扬斯克</strong></p>
<p>乌克兰军方表示，俄罗斯军队正准备对斯拉维扬斯克市发起新的进攻。</p>
<p>乌克兰武装部队总参谋部表示，莫斯科设法在斯洛文斯克西北24公里（15英里）的 Bohorodychne 村站稳脚跟，并准备进攻该市。</p>
<p>陆军发言人亚历山大·斯图彭表示，俄罗斯军队在向城市进发的作战中“取得了成功”。 斯图彭还声称俄罗斯在黑海有6艘航母“准备好使用导弹武器”。</p>
<p>他还表示，乌克兰军队“击退”了顿涅茨克和卢甘斯克地区的14次袭击，摧毁了4辆坦克、2个火炮系统和1个弹药库。</p>

<p><strong>州长：乌克兰控制北顿涅茨克工厂庇护数百人</strong></p>
<p>该地区州长说，乌克兰仍然控制着位于北顿涅茨克的阿佐特化工厂，那里有数百名平民避难。</p>
<p>之前，俄罗斯支持的分离主义领导人称，300至400名乌克兰战士也被困在那里。</p>
<p>“有关封锁阿佐特工厂的信息是谎言，”卢甘斯克州州长谢尔盖·海代说。“我们的部队控制着北顿涅茨克的工业区，并正在摧毁镇上的俄罗斯军队。”</p>
<p>乌克兰表示，约有800人躲在阿佐特工厂下方的几个防空洞中，其中包括约200名员工和北顿涅茨克的600名居民。</p>
<p>自称为卢甘斯克人民共和国的俄罗斯支持的代表罗迪翁·米罗什尼克表示，300至400名乌克兰战士与平民一起在工厂内被封锁，他们试图通过谈判进入利西昌斯克市。</p>
<p>“这样的要求是不可接受的，不会被讨论，”米罗什尼克说，并补充说正在与战斗人员就工厂里的平民进行谈判。</p>

<p>国际文传电讯社援引负责俄罗斯与欧洲关系的外交部负责人奥列格·季亚普金的话说：“一如既往，回应将是相称和适当的，旨在消除对俄罗斯联邦安全的潜在威胁。”</p>

<p><strong>议员：乌克兰希望拯救被判处死刑的外国士兵</strong></p>
<p>一位议员说，乌克兰正在尽一切可能挽救三名因为乌克兰而战而被顿巴斯代理当局判处死刑的外国人。</p>
<p>被捕后，两名英国人和一名摩洛哥人被自称为顿涅茨克人民共和国（DPR）的一家法院判定犯有“雇佣军活动”罪；该共和国的分离主义领导人得到莫斯科的支持。</p>
<p>“国防部和负责交换囚犯的主要情报局都在采取一切必要措施，以确保这些外国公民获救，”费迪尔·韦尼斯拉夫斯基议员在国家电视台上说。</p>
<p>副总理伊琳娜·韦列舒克表示，她相信分离主义当局最终会采取理性行动，“因为他们非常清楚，对这三名士兵采取任何错误措施都将对他们和俄罗斯人造成不可挽回的影响”。</p>
<p>“有件事告诉我，最终，这样或那样，迟早这三名军人会以某种方式交换（或以其他方式回家）。”</p>

<p><strong>俄罗斯炮击在北顿涅茨克引发火灾</strong></p>
<p>该地区州长说，俄罗斯炮击了位于北顿涅茨克的阿佐特化工厂，数吨石油泄漏引发了一场大火。</p>
<p>谢尔盖·海代州长没有透露有数百名平民避难的工厂里的大火是否已经扑灭。他说，俄罗斯军队控制了该市的大部分地区，但乌克兰武装分子控制了阿佐特化工厂。</p>
<p>北顿涅茨克战役及其破坏使人们回忆起数周以来对南部港口城市马里乌波尔的轰炸。</p>
<p>在俄罗斯军队上个月控制这座城市之前，它变成了一片废墟，最后的乌克兰防御者从他们在亚速钢铁厂的堡垒投降。</p>

<p><strong>乌克兰领导人誓言“将取得胜利”</strong></p>
<p>泽连斯基总统发出反抗的声音，誓言要打败俄罗斯的入侵。</p>
<p>“我们肯定会在俄罗斯发起的这场战争中获胜，”泽连斯基通过视频链接在新加坡的一次会议上说。“正是在乌克兰的战场上，这个世界的未来规则正在被决定。”</p>
<p>俄罗斯的打击摧毁了顿涅茨克乌克兰控制的两个最大城市——克拉马托尔斯克和斯拉维扬斯克——的电力供应，地区州长帕夫洛·基里连科在Telegram上写道。</p>
<p>他后来在国家电视台发表讲话，称此举属于蓄意切断仍然在乌克兰手中的顿涅茨克城镇的电力的战略。 他说：“敌人知道他在打击什么，目的是什么。”</p>
