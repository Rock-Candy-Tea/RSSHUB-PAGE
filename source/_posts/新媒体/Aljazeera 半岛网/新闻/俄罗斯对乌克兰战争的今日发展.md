<p>乌克兰首都基辅周四晚上遭到轰炸，此举恰逢联合国秘书长安东尼奥·古特雷斯对该市进行访问期间，这是自 4 月中旬以来针对乌克兰首都的首次此类袭击。</p>
<p>五角大楼发言人约翰·柯比表示，美国国防部仍在评估和分析俄罗斯对基辅的袭击，他在接受美国有线电视新闻网采访时补充说，俄罗斯最近对乌克兰西部的军事目标发动袭击，以打击乌克兰的补给线。</p>
<p>欧盟外交政策负责人何塞普·博雷利表示，俄罗斯在古特雷斯和保加利亚总理基里尔·佩特科夫在场时轰炸基辅，再次表明其公然无视国际法。</p>
<h3>以下是这场战争第65天的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>乌克兰外交部长：</h3>
<ul>
<li>我与英国外交大臣讨论了向乌克兰运送更多武器事宜。</li>
<li>我和英国外交大臣一致认为，需要对俄罗斯石油进口实施真正的禁运。</li>
<li>我与英国外交大臣讨论了安全保障问题。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>克里姆林宫：美国国会批准一项允许使用俄罗斯资产援助乌克兰的法案是一个危险的先例。</p>
<p>博雷利：我们必须团结一致，以应对普京切断对欧盟成员国供气的敲诈。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>哈尔科夫省省长：乌克兰防空系统击落了一架俄罗斯Su-34战机。</p>
<p>波兰媒体：华沙交付基辅200辆T-72坦克和数十辆装甲车。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰南部空军司令部：</p>
<ul>
<li>我们从一个友好国家收到了 S-300 防空系统。</li>
<li>S-300防空系统大大加强了我们在该地区的防空能力。</li>
</ul>
<p>英国政府：我们将派专家组帮助乌克兰调查俄罗斯的战争罪行。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>基辅市长：昨天首都的一个居民区遭遇袭击后，救援队在瓦砾下发现了一具尸体。</p>
<p>俄罗斯国防部：我们摧毁了基辅的导弹和航天工业设施。</p>
<p>欧盟外交政策专员：摩尔多瓦的破坏稳定行动损害了该国的安全与稳定。</p>
<p>博雷利：欧盟支持摩尔多瓦，并重申对其主权和领土完整的坚定支持。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>泽连斯基：感谢拜登和美国人民在与俄罗斯的战争中支持我们，我们今天比以往任何时候都更需要帮助。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰媒体援引第聂伯罗彼得罗夫斯克州长的话报道称：俄罗斯使用“格拉德”导弹袭击了韦利卡科斯特罗姆卡和马林斯基村。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总统办公室：计划于周五从马里乌波尔的亚速钢铁厂撤离平民。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>英国国防部：派出约8000名英国士兵到东欧各地参加演习。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>博雷利：俄罗斯在古特雷斯和保加利亚总理基里尔·佩特科夫在场时轰炸基辅，再次表明其公然无视国际法。</p>
<p>美国有线电视新闻网CNN援引五角大楼发言人的话报道称：我们仍在评估和分析俄罗斯对基辅的袭击。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>克里姆林宫发言人：普京总统强调，亚速钢铁厂综合体的士兵需要投降，并允许平民离开。</p>
<p>白宫：我们欢迎泽连斯基应邀出席在印度尼西亚举行的二十国集团峰会，但我们认为俄罗斯不应参加。</p>
