<p>乌克兰一家通讯社援引一名乌克兰官员的话报道称，在首都基辅附近布查的一个万人坑中发现了 57 具尸体。</p>
<p>乌克兰国防部公布了据称在布查镇的照片，显示尸体散落在街上，乌克兰国防部指责控制布查的俄罗斯军队对民众进行处决和屠杀。</p>
<p>乌克兰国防部将布查的情况比作波斯尼亚“斯雷布雷尼察”发生的情况，后者发生在1990 年代中期的战争中，数千人丧生。</p>
<p>有关战地方面的发展，乌克兰空军司令部宣布，在过去 24 小时内击落两架俄罗斯苏-34和苏-35战斗机、两枚巡航导弹和一架直升机，并击落一架侦察机。</p>
<p>有关政治方面消息，俄罗斯“塔斯社”通讯社援引波兰副总理雅罗斯瓦夫·卡钦斯基的话报道称，提议向乌克兰派遣一个北大西洋公约组织（北约）的武装维和特派团。</p>
<p>关于俄乌谈判，克里姆林宫发言人德米特里·佩斯科夫在俄罗斯国家频道的一个节目中表示，俄罗斯总统弗拉基米尔·普京可以在双方达成明确、具体和书面协议后会见乌克兰总统泽连斯基。</p>
<p>佩斯科夫强调，俄罗斯天然气的欧洲进口商可以继续以欧元支付天然气订单，但补充说，他们还必须在俄罗斯开立卢布账户，才能将卢布收入转换为俄罗斯天然气工业股份公司。</p>
<p>佩斯科夫表示，俄罗斯不想让欧洲人的生活复杂化，但买家应该遵守新规则。</p>
<h3>以下是这场战争第40天的最新进展：</h3>

</ul>
<h3>乌克兰总检察长：</h3>
<ul>
<li>我们将对在伊尔平、布查和其他地方发生的事件展开调查。</li>
<li>俄罗斯军队在乌克兰犯下的罪行被列为危害人类罪。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>俄罗斯外交部长：</h3>
<ul>
<li>英国必须履行其作为联合国安理会成员的义务。</li>
<li>英国反对就布查召开安理会会议，俄罗斯将再次要求就此召开会议。</li>
<li>乌克兰及其西方赞助商在媒体和社交网络上散布有关布查的虚假视频片段。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>乌克兰国防部：</h3>
<ul>
<li>我们已经对俄罗斯的第一波入侵做出了回应，但可能还有其他攻击，我们正在为所有情况做准备。</li>
<li>在我们击退了俄罗斯的大规模干预 39 天后，基辅地区获得了解放。</li>
<li>自战争开始以来，我们击落了 147 架飞机和 134 架直升机。</li>
<li>自战争开始以来，已有 18300 名俄罗斯士兵丧生。</li>
<li>尽管从基辅和切尔尼戈夫撤军，俄罗斯仍试图夺取哈尔科夫、马里乌波尔和敖德萨。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>波兰边防部队宣布，自俄对乌开战以来，已有248.1万乌克兰难民进入该国，乌克兰总检察长办公室宣布，自“俄罗斯侵略”开始以来，已有161名儿童被杀。</p>
<p>法国总统：我们需要新的制裁措施，包括煤炭和燃料，作为谈判的筹码，在乌克兰存在明显的战争罪行证据，俄罗斯军队似乎对此负有责任。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>美国国务卿安东尼·布林肯在推特上发表推文称：</h3>
<ul>
<li>我们强烈谴责俄罗斯军队在布查镇和整个乌克兰犯下的明显暴行。</li>
<li>我们通过所有可用的工具和文件来追究责任，并共享信息以追究肇事者的责任。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰空军指挥部：我们击落了两架俄罗斯战斗机苏-34和苏-35战斗机、一架直升机、一架侦察机和两枚巡航导弹。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<p>我认为，乌克兰极端分子应对在布查对平民犯下的罪行负责，我们已要求在周一召开安理会会议，讨论乌克兰军方和极端分子在布查的挑衅行为。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<p>马里乌波尔遭到俄罗斯猛烈的狂轰滥炸，乌克兰军队进行了激烈的抵抗。马里乌波尔是俄罗斯入侵的主要目标，因为它确保了从俄罗斯到克里米亚的陆地走廊。</p>
<ul style="list-style-type:square">

</ul>
<p>英国国防部：随着俄罗斯军队试图占领整个城市，马里乌波尔爆发了激烈的战斗。</p>
