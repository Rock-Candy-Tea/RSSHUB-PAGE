<p>俄罗斯对乌克兰的战争已经进入第45天，与此同时，西方国家继续对莫斯科实施更多制裁，而乌克兰总统弗拉基米尔·泽连斯基再次要求与俄罗斯总统弗拉基米尔·普京会面。</p>
<p>面对俄罗斯的袭击，乌克兰呼吁其盟国在军事上为其提供支持，而现场报道显示，俄罗斯军队正计划向哈尔科夫挺进。</p>
<p>顿巴斯地区（东部）克拉马托尔斯克市火车站遭到导弹袭击，造成数十名平民丧生，基辅指责俄罗斯军队实施了袭击，但莫斯科对此予以否认。</p>
<p>另一方面，乌克兰宣布重新夺回了苏梅市（东北部）和尼古拉耶夫（南部）的大部分地区，而英国情报部门报告称，俄罗斯军队完全从乌克兰北部撤退到白俄罗斯和俄罗斯。</p>
<h3>以下是这场战争第45天的最新动态：</h3>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>红十字会：许多平民仍被困在马里乌波尔，他们迫切需要安全通道，以免为时已晚。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>CNN援引一名美国军方官员说法报道称：</h3>
<ul>
<li>俄罗斯军队尚未解决其在乌克兰的后勤问题。</li>
<li>俄罗斯不太可能在乌克兰东部增援部队。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰国防部：发现了132具尸体，有乌克兰公民遭受酷刑的痕迹，基辅马卡里夫区一半被摧毁。</p>
<p>乌克兰总统：我们相信我们军队的实力，并正在与外国领导人进行谈判，以向我们的国家提供武器。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>俄罗斯国防部：</h3>
<ul>
<li>使用高精度导弹武器摧毁了属于乌克兰军队的弹药库。</li>
<li>基辅的乌克兰军队正准备挑衅和指责俄罗斯在伊尔平对平民进行大屠杀。</li>
</ul>

<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>英国总理鲍里斯·约翰逊：</h3>
<ul>
<li>我们将与七国集团合作伙伴合作，瞄准俄罗斯经济每一个支柱，以确保普京失败。</li>
<li>我们将向乌克兰运送更多的防御武器。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>俄罗斯中央银行：3 月起暂停交易后，自 4 月 18 日起允许外汇交易。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>俄罗斯国防部：Su-30cm 和 Su-35 战斗机摧毁了乌克兰的一个雷达站和 Buk-M1 防空系统。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总统顾问：俄罗斯人试图隐瞒马里乌波尔的战争罪行，世界还不了解那里悲剧的严重程度。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰卢甘斯克市市长：对该地区的轰炸有所增加，我们需要进一步疏散民众。</p>
<p>乌克兰副总理：同意周六开放 10 条人道主义走廊，可以通过私人交通工具撤离，其中包括马里乌波尔。</p>

<li><strong>更新：</strong></li>
</ul>
<h3>苏梅地区行政主管：</h3>
<ul>
<li>俄罗斯军队撤离该地区后，有118平民和70名士兵死亡。</li>
<li>已经部分控制了边境，我们看到俄罗斯军队在我们边境附近进行大量军事集结。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>英国陆军情报：</h3>
<ul>
<li>俄罗斯继续轰炸克拉曼斯克火车站等非军事场所。</li>
<li>俄罗斯的行动继续集中在巡航导弹支持的顿巴斯、马里乌波尔和尼古拉耶夫。</li>
<li>乌克兰的抵抗继续挫败莫斯科在克里米亚和顿巴斯之间建立陆地走廊的野心。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>《新闻周刊》援引俄罗斯驻华盛顿大使说法报道称：</h3>
<p>军事行动的目的是结束基辅政权的种族灭绝。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总统弗拉基米尔·泽连斯基预计国际社会将做出果断反应，此前，乌克兰火车站遭到袭击，造成 50 多人死亡，他表示，“我们期待全球对这场战争罪行做出强有力的反应。”</p>
<p>乌克兰当局表示，俄罗斯导弹袭击东部城市克拉马托尔斯克的一个火车站，造成 52 人死亡，另有 109 人受伤。</p>
<p>成千上万的人正在等待撤离，而另一方面，莫斯科否认参与了这次袭击。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>

<ul>
<li>我们呼吁我们的盟友提供军事支持，帮助我们对抗俄罗斯的进攻。</li>
<li>俄军正在乌克兰东部重新集结，并计划向哈尔科夫挺进。</li>
<li>俄罗斯军队在进入切尔诺贝利地区的红森林后受到核辐射。</li>
</ul>
