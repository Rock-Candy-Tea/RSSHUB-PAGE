<h3>以下是这场战争进入今日以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国、立陶宛同意加强安全合作</strong></p>
<p>英国和立陶宛签署了一项联合声明，以加强防务和安全合作。</p>
<p>英国表示，该宣言将建立在两国作为北约盟国的防务合作基础上，并将增强对包括俄罗斯等国在内的威胁的抵抗力。但它没有透露更多细节。</p>
<p>自从俄罗斯于今年2月底发起军事行动以来，立陶宛等波罗的海国家越来越担心自己可能会成为下一个面临俄罗斯入侵的国家。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰第一夫人寻求心理健康支持</strong></p>
<p>乌克兰第一夫人敦促世界卫生组织帮助解决该国巨大的心理健康危机，并警告称，俄罗斯战争所产生的影响可能会持续几十年。</p>
<p>奥莱娜·泽连斯基在联合国卫生组织的年度大会上发表视频讲话称，“乌克兰人经历了占领时期，在前线、在防空洞里、在炮击下……他们需要像那些身体受伤的人一样得到心理上的康复。”</p>
<p>她还指出，“不幸的是，这场战争的后果将持续多年甚至几十年的时间”，“俄罗斯的战争呈现出了我们无法想象的恐惧。”</p>
<figure id="attachment_283608" aria-describedby="caption-attachment-283608" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-283608" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/ukraine-first-lady.png?w=770&resize=770%2C518" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-283608" class="wp-caption-text">乌克兰第一夫人警告称，俄罗斯战争的“后果”将会持续“几十年”的时间 (法国媒体)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>随着外国公司的继续撤离，星巴克将退出俄罗斯</strong></p>
<p>在俄罗斯经营了近15年的时间后，咖啡连锁店星巴克也加入了大批西方公司的行列，并宣布将会退出俄罗斯。</p>
<p>这家总部位于美国西雅图的咖啡公司在俄罗斯拥有130家门店，由获得许可的阿尔沙亚集团(Alshaya)全资持有和运营，在该国拥有近2000名员工。</p>
<p>该公司在今年3月初关闭了它在俄罗斯的所有门店，并且暂停了所有的商业活动，但是没有提供退出俄罗斯将对公司财务产生的影响细节。然而，该公司表示将继续支持其在该国内的员工，包括向他们支付6个月的工资。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯常驻联合国日内瓦外交官员因乌克兰战争而辞职</strong></p>
<p>俄罗斯常驻联合国日内瓦办事处的一名资深外交官员表示，他在递交辞呈之前，还向外国的同事们发出了一封措辞严厉的信，以谴责俄罗斯总统普京在乌克兰发动的“侵略性战争”。</p>
<p>今年41岁的鲍里斯·邦达列夫在一封于23日上午递交给俄罗斯外交使团的信件中证实了自己的辞职。此前，一名外交官员将他的英文声明转交给了美联社。</p>
<p>他在信中写道，“在我长达20年的外交生涯中，我看到过我们外交政策上的不同转折点，但是我从未像今年2月24日那样为我的国家感到羞耻。”而这正是俄罗斯对乌克兰发起特别军事行动的日期。</p>
<p>这项辞职，意味着俄罗斯外交使团内罕见地出现了公开承认对俄罗斯战争不满的情况。目前，普京政府正在打压外界对这场军事行动的异议。</p>

<li><strong>更新</strong></li>

<p><strong>“犯罪冒险”：乌克兰战争加剧俄罗斯人才流失</strong></p>
<p>在2022年的前3个月，共有近400万人离开了俄罗斯，其中包括IT领域的专家、记者、研究人员和分析人士，背景是俄罗斯在其军事行动的问题上面临着西方大国越来越大的外交和经济压力。</p>

<p><strong>乌克兰战争背后的谈判</strong></p>
<p>在这期节目中，半岛电视台的“The Take”播客重点讨论了这场冲突的外交背景，包括基辅和莫斯科之间目前陷入僵局的关于结束这场战争的谈判。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>

<p><strong>半岛电视台记者:泽连斯基利用达沃斯的演讲呼吁重建援助</strong></p>
<p>半岛电视台外交编辑詹姆斯·贝斯自达沃斯报道称，泽连斯基利用他在世界经济论坛上发表的视频演讲，“呼吁国际社会、政治领袖和商业领袖努力重建乌克兰”。</p>
<p>贝斯指出，“泽连斯基表示，世界正处在一个转折点，并将其与第一次世界大战和第二次世界大战的爆发相提并论。他还表示，他的国家迫切需要更多的资金。”</p>
<p>据贝斯报道，泽连斯基已经明确表示，在俄罗斯发起军事行动之后，“国际秩序需要得到改变”，需要“不仅仅对此类事件作出反应，还要采取行动以确保它们不会再次发生”。</p>
<p>贝斯表示，“在这里，以及在达沃斯之外，人们还将听到泽连斯基关于对俄罗斯实施更多制裁的强烈呼吁，以及禁止与俄罗斯进行任何贸易的呼吁。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>麦当劳俄罗斯分公司开始重塑品牌：莫斯科附近的“金拱门”标志被拆除</strong></p>
<p>据路透社报道，莫斯科北部一家麦当劳餐厅的工作人员拆除了“金拱门”标志，这是该快餐公司在俄罗斯重塑品牌的第一阶段。</p>
<p>据路透社报道，这家餐厅位于莫斯科附近的希姆基镇，餐厅外的地上矗立着黄色的拱门。目前尚不清楚是否还要在全国其他分店移除该标志。</p>
<p>世界上最大的汉堡连锁店麦当劳，已经将其在俄罗斯的餐厅出售给当地的一家特许经营商，后者将用一个尚未公布的新名字重新命名这些餐厅，从而结束了麦当劳在俄罗斯经营30多年的历史。麦当劳表示，将保留其商标。</p>

<p><strong>半岛电视台记者：关于亚速钢铁厂投降武装人员命运的报道相互矛盾</strong></p>
<p>半岛电视台记者从莫斯科报道称，关于在马里乌波尔的亚速钢铁厂投降的乌克兰武装人员的命运，存在“相互矛盾的报道”。</p>
<p>记者指出，“顿涅茨克人民共和国领导人丹尼斯·普希林在23日早些时候表示，这个自封的共和国正在努力建立一个国际法庭，并将在那里审判这些武装分子。”</p>
<p>记者还表示，“但是我们也听到俄罗斯的一位副外长透露……乌克兰和俄罗斯官员正在就可能的囚犯交换进行讨论。”</p>
<p>记者指出，目前并不清楚在撤离该厂的2478名武装人员中，有多少人是“亚速营”成员，而该组织的极右翼势力正是克里姆林宫进攻乌克兰以打击该国境内纳粹主义分子的目标。</p>
<p>记者补充称，“我们所知道的是, 俄罗斯首都内拥有不同背景的议员们，都呼吁那些“亚速营”成员不应成为任何囚犯交换项目的一部分，并且渴望建立一个法庭来对他们的战争罪进行审判。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>卢卡申科指责西方试图“肢解”乌克兰</strong></p>
<p>白俄罗斯总统卢卡申科表示，西方国家试图“肢解”乌克兰，并指责波兰试图夺取乌克兰的西部地区。</p>
<p>卢卡申科在与长期盟友俄罗斯总统普京举行的电视会议上表示，基辅最终将不得不寻求帮助，以防止乌克兰的西部被占领。</p>
<p>但他没有为这种论断提供任何证据。</p>

<p><strong>德国宣布乌克兰难民货币兑换计划</strong></p>
<p>德国财政部和德国各大银行宣布，从本周二开始，逃离乌克兰战争的难民可以在德国将手中的乌克兰格里夫纳兑换成欧元。</p>
<p>相关机构在一份联合声明中表示，德国财政部与德国央行、乌克兰央行以及德国银行业委员会对这项兑换计划达成了一致。</p>
<p>商定的外汇交易量最初设定为15亿格里夫纳(约合5078万美元)，该计划将至少持续3个月的时间。</p>
<p>根据这项协议，难民可以在参与协议的德国银行内，以当前汇率兑换1万格里夫纳(约合338美元)的欧元。这种兑换可以分多期进行，并且是免费的。</p>
<ul style="list-style-type:square">

<p><strong>泽连斯基称有87人在切尔尼哈夫的袭击中丧生</strong></p>
<p>泽连斯基表示，在俄罗斯上周二针对德斯纳村的空袭中，共有87人丧生。</p>
<p>这位乌克兰领导人在达沃斯世界经济论坛上发表讲话时，作出了上述声明。此前，乌克兰地方当局在上周表示，共有8人在针对乌克兰北部切尔尼哈夫地区的空袭中丧生。</p>
<p>半岛电视台无法独立证实这些说法的真实性。</p>

<p><strong>德国部长：匈牙利必须在俄罗斯石油禁运问题上与欧盟合作</strong></p>
<p>德国经济部长罗伯特·哈贝克警告匈牙利称，不要试图阻止在欧盟范围内对俄罗斯石油进口实施禁运的努力。</p>
<p>哈贝克在达沃斯世界经济论坛上表示，“不同的国家拥有不同的解决方案”，“我希望包括匈牙利在内的所有人都能找到解决方案。”</p>
<p>作为俄罗斯在欧盟最亲密的盟友，匈牙利政府已表示希望得到欧盟的补偿，以降低放弃俄罗斯原油或在拟议的禁运中获得某些豁免。欧盟需要其27个成员国的一致同意，才能使这项禁运措施生效。</p>

<p><strong>乌克兰称俄罗斯在顿巴斯地区使用“焦土”战术</strong></p>
<p>乌克兰和俄罗斯军队在乌克兰东部的顿巴斯地区继续发生激烈战斗。与此同时，莫斯科军队继续向北顿涅茨克推进。当地官员指责莫斯科使用“焦土战术”。</p>

<p><strong>克里姆林宫对乌克兰的战争罪审判表示警惕</strong></p>
<p>克里姆林宫方面表示，莫斯科对一名在基辅被控犯有战争罪的俄罗斯军人的受审问题感到担忧。</p>
<p>今年21岁的俄罗斯坦克指挥官瓦迪姆·希什马林在上周认罪，并承认曾在2月28日于乌克兰东北部的楚帕克希维卡村（Chupakhivka）杀害了一名手无寸铁的老年平民。他的案子将是第一起针对在乌克兰参与进攻的俄罗斯士兵的战争罪审判。</p>
<p>克里姆林宫发言人佩斯科夫在与记者进行的电话会议上表示，“我们当然担心我们公民的命运，但我重申，我们没有能力单独保护他的利益。”</p>

<p><strong>泽连斯基在达沃斯论坛的讲话中要求对俄罗斯实施“最大限度”的制裁</strong></p>
<p>乌克兰总统泽连斯基在达沃斯世界经济论坛上发表的一次视频讲话中，呼吁对俄罗斯实施包括石油禁运在内的“最大限度”的制裁。</p>

<p><strong>乌克兰终止与俄罗斯的税收协议</strong></p>
<p>根据国际文传电讯社的报道，乌克兰议会投票决定终止与俄罗斯之间的一项双重征税协议。该协议自1995年开始实施，根据这项协议，在乌克兰活动的俄罗斯居民无需缴纳基辅方面的税费，而只需向俄罗斯方面缴税。</p>
<p>乌克兰财政部表示，现在，“俄罗斯联邦居民从乌克兰获得的所有收入都将按照乌克兰税法规定的15%的一般税率完成缴纳”，而不是按照双重征税协议规定的优惠税率来计算。</p>
<p>该部门还表示，在俄罗斯生活的乌克兰居民也将不再向莫斯科纳税。</p>
