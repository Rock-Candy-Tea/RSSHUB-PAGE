<li>乌克兰总统泽连斯基表示，乌克兰准备用俄罗斯囚犯换取在马里乌波尔亚速钢铁厂投降的军队。</li>
<li>一位俄罗斯谈判代表表示，莫斯科将考虑用乌克兰亚速营的战俘换取与总统弗拉基米尔·普京关系密切的乌克兰富商维克托·梅德韦丘克。他没有具体说明考虑交换的囚犯人数。</li>
<li>俄罗斯能源巨头俄罗斯天然气工业股份公司表示，由于没有收到卢布付款，它已停止向芬兰供应所有天然气。</li>

<p><strong>以下是所有最新更新：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>波兰总统在基辅：乌克兰人是我们的客人</strong></p>
<p>根据乌克兰议员的说法，波兰总统安杰伊·杜达周日告诉乌克兰议会成员，“今天的自由世界有乌克兰的面孔”。</p>
<p>罗曼·赫里舒克在推特上发布了一张杜达在会议厅发表讲话的照片，并引用他的话说：“亲爱的乌克兰人，你们的亲戚、妻子、父母和孩子被迫离开波兰，他们在我们国家不是难民。他们是我们的客人。”</p>
<p>波兰允许大约350万乌克兰人入境，是接收乌克兰难民最多的国家。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>朔尔茨的非洲之行重点关注俄乌战争</strong></p>
<p>奥拉夫·朔尔茨将作为德国总理以俄罗斯对能源和食品价格以及安全的影响为主题，开始了他的第一次非洲之行。</p>
<p>周日开始的为期三天的塞内加尔、尼日尔和南非之旅，正值德国在入侵乌克兰后寻求减少对俄罗斯天然气的严重依赖之际。</p>
<p>据路透社报道，一名政府官员周五表示，这也可能有助于探索塞内加尔的一个气田。塞内加尔拥有数十亿立方米的天然气储量，有望成为该地区主要的天然气生产国。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>波兰总统称乌克兰必须决定自己的命运</strong></p>
<p>周日，波兰总统安杰伊·杜达在基辅对议员们说，只有乌克兰有权决定自己的未来。</p>
<p>“出现了令人担忧的声音，称乌克兰应该屈服于普京的要求。只有乌克兰有权决定自己的未来，”他说</p>
<p>杜达是自俄罗斯入侵以来第一位在乌克兰议会发表面对面演讲的外国领导人。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>拜登会见莫迪以孤立俄罗斯</strong></p>
<p>美国总统乔·拜登本周将在日本与印度总理纳伦德拉·莫迪会面，国家安全顾问杰克·沙利文在美​​国总统从韩国飞往日本时告诉记者。</p>
<p>美国正在努力说服印度与俄罗斯保持距离。</p>
<p>印度是俄罗斯武器的主要购买者，迄今为止，印度一直抵制谴责莫斯科入侵乌克兰或加入孤立俄罗斯的国家行列的压力。</p>
<p>拜登和印度总理将在美国、印度、日本和澳大利亚之间的非正式联盟Quad峰会期间会面。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>立陶宛将削减俄罗斯能源进口</strong></p>
<p>从周日开始，立陶宛将完全减少对俄罗斯能源的进口，包括石油、电力和天然气。</p>
<p>立陶宛能源部周五在一份声明中表示，能源交易商Nord Pool已决定停止与其在波罗的海国家的唯一进口商俄罗斯公用事业公司Inter RAO进行俄罗斯电力交易，这意味着该国将不再进口任何俄罗斯能源.</p>
<p>“这不仅是立陶宛走向能源独立的一个极其重要的里程碑，而且也是我们与乌克兰团结一致的体现，”能源部长代纽斯·克雷维斯说。 “我们必须停止资助俄罗斯的战争机器。”</p>
<p>乌克兰议会第一副议长亚历山大·科尔尼延科对此举表示欢迎。</p>
<ul style="list-style-type:disc">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯人袭击北顿涅茨克但被击退</strong></p>
<p>乌克兰总统泽连斯基办公室周日表示，东部城市北顿涅茨克一夜之间遭到了四个不同方向的袭击，但俄罗斯人被击退。</p>
<p>总统早间简报说：“北顿涅茨克同时遭到来自四个方向的袭击，但敌人被击退并撤退到先前的阵地。”</p>
<p>声明说，北顿涅茨克的7座房屋、周边城镇和村庄的至少27座房屋遭到破坏。</p>
<p>北顿涅茨克和斯拉维扬斯克对于控制卢甘斯克地区至关重要。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>市长：马里乌波尔面临健康和卫生“灾难”</strong></p>
<p>马里乌波尔市长瓦迪姆·博伊琴科表示，在大规模浅坟埋葬和下水道系统崩溃可能导致乌克兰港口城市出现健康危机。</p>
<p>他补充说，夏季降雨有可能污染水源，因为他敦促莫斯科允许居民安全离开城市。</p>
<p>他在Telegram上说：“除了占领者和合作者造成的人道主义灾难外，这座城市还处于传染病爆发的边缘”。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯禁止960多名美国人入境</strong></p>
<p>俄罗斯公布了一份963名美国人的名单，他们被“永久禁止”进入该国，以回应美国对莫斯科入侵乌克兰的制裁。</p>
<p>位居榜首的是美国总统乔·拜登。</p>
<p>名单上还有84岁的好莱坞演员摩根弗里曼，他曾于2017年出现在一段视频中，指责俄罗斯政府干涉美国民主</p>
<p>俄罗斯外交部表示，名单上的人“煽动俄罗斯恐惧症”，其中包括已故参议员约翰·麦凯恩等几位死去的政客。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯官员：乌克兰可能成为“联邦”</strong></p>
<p>据俄新社报道，俄罗斯政治家、普京任命的克里米亚地区代表表示，乌克兰不太可能以目前的形式继续存在。</p>
<p>格奥尔基·穆拉多夫表示乌克兰可能会成为一个联邦或一组国家。</p>
<p>他补充说，任何尊重自己的国家都不会容忍在邻国公然侵犯本国人民的权利，援引莫斯科通常用来入侵乌克兰的论点。</p>
<p>“如果这些企图导致人们彻底灭绝，那就更是如此，就像近年来生活在乌克兰的俄罗斯人和讲俄语的人遭受的一样，他们在他们的祖国生活了几个世纪，”他说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰第一夫人罕见地与泽连斯基一同接受采访</strong></p>
<p>乌克兰第一夫人叶莲娜·泽连斯卡娅在乌克兰电视台罕见地与她的丈夫一同接受了采访，这是自战争开始以来这对夫妇第二次被看到在一起。</p>
<p>泽连斯卡娅描述了她醒来听到“外面有奇怪的声音”并发现丈夫不在她身边的那个晚上。她说她走进隔壁房间，“他已经穿着西装，但没有打领带”。</p>
<p>“我问他发生了什么事，他说，‘已经开始了’，”泽连斯卡娅回忆道。</p>
<p>“我们的家庭和其他乌克兰家庭一样支离破碎，”她说，并补充说两人在此次采访之前已经有两个半月没见面了，只能通过电话交谈。</p>

<p><strong>波兰杜达将向乌克兰议会发表演讲</strong></p>
<p>波兰总统将成为战争爆发以来第一位直接与乌克兰议会对话的外国国家元首。</p>
<p>据国际文传电讯社报道，安杰伊·杜达周六抵达乌克兰，将于周日向乌克兰最高拉达发表演讲。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>部长：制裁“几乎破坏了”俄罗斯物流</strong></p>
<p>据塔斯社报道，俄罗斯交通部长表示，国际制裁“几乎破坏了”该国的物流。</p>
<p>“对俄罗斯实施的制裁，几乎破坏了我们国家的所有物流。 我们必须寻找新的物流走廊，”维塔利·萨维利耶夫在访问位于里海的俄罗斯南部港口城市阿斯特拉罕时说</p>
<p>新的货物运输走廊包括一条穿过里海两个港口Olya和马哈奇卡拉的南北路线。</p>
<p>这位部长的评论是克里姆林宫罕见地承认，旨在削弱俄罗斯经济的制裁正在产生重大影响。</p>

<p><strong>俄罗斯将两名备受瞩目的批评者称为“外国代理人”</strong></p>
<p>俄罗斯在其长长的“外国代理人”名单中增加了两名克里姆林宫批评者，前国际象棋冠军加里·卡斯帕罗夫和前石油大亨米哈伊尔·霍多尔科夫斯基。</p>
<p>该名称适用于许多独立媒体公司、记者和非政府组织。名单上的每个人都有义务在他们的出版物上标明“外国代理人”身份的免责声明。</p>
<p>出生于苏联的前国际象棋冠军卡斯帕罗夫是普京的长期反对者，在美国生活了近十年。</p>
<p>霍多尔科夫斯基是1990年代俄罗斯最有权势的商人之一。在流放之前，他在俄罗斯监狱中因许多人认为是伪造的指控而被关押了十年。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯再次指责乌克兰向库尔斯克地区开火</strong></p>
<p>据塔斯社报道，俄罗斯库尔斯克州州长再次指责乌克兰向其定居点开火。</p>
<p>“Tetkino和附近的居民区再次遭受乌克兰的袭击，” 罗曼·斯塔罗沃伊周六表示，并补充说他将在稍后提供有关情况的更多细节。</p>
<p>州长说，这次袭击没有造成人员伤亡或基础设施损坏。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基：乌克兰军队正在阻止俄罗斯袭击斯洛维扬斯克和北顿涅茨克</strong></p>
<p>泽连斯基曾表示，乌克兰军队几天来一直在阻止俄罗斯在斯洛维扬斯克和北顿涅茨克的进攻。</p>
<p>“顿巴斯的情况极其困难。与前几天一样，俄罗斯军队正试图攻击斯洛维扬斯克和北顿涅茨克。乌克兰武装部队正在阻止这种进攻，”泽连斯基在夜间讲话中说。</p>
<p>俄罗斯国防部长周五表示，莫斯科军队几乎完全控制了卢甘斯克。俄罗斯正在加强对北顿涅茨克的攻势，这是乌克兰在该地区的最后一个据点。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯分裂分子说，6名男子在亚速钢铁厂投降期间死亡</strong></p>
<p>一名俄罗斯分离主义领导人表示，在亚速钢铁厂的一次撤离行动中，6名乌克兰战士死亡，当时被撤离的战士集体向俄罗斯人投降。</p>
<p>自称为顿涅茨克人民共和国的领导人杰尼斯·普希林说，这是因为乌克兰男子炸毁了自己的弹药库。</p>
<p>“不清楚是谁干的，没有人承担责任，但在主力部队走出后，有人下令炸毁弹药库，6人当场死亡，据我所知，4人受伤，”普希林周六在YouTube Soloviev Live频道上说道。</p>
<p>普希林还表示，未知数量的乌克兰军人可能仍留在亚速钢铁厂，并补充说他们有一些食物和水库存，但缺乏药品。</p>

<p><strong>泽连斯基：俄罗斯阻止了乌克兰2200万吨食品的出口</strong></p>
<p>泽连斯基曾表示，俄罗斯已阻止乌克兰2200万吨食品的出口。</p>
<p>泽连斯基在与葡萄牙总理安东尼奥·科斯塔会晤后接受媒体采访时表示，如果国际社会不帮助乌克兰解除对港口的封锁，能源危机之后将是粮食危机。</p>
<p>“你可以通过不同的方式解除对它们的封锁。其中一种方法是军事解决方案。这就是我们向我们的合作伙伴询问相关武器的原因，”他补充说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：卢甘斯克地区近60人撤离</strong></p>
<p>卢甘斯克州长表示，周六有57人从该地区撤离，并补充说，北顿涅茨克、利西昌斯克和比洛戈里夫卡村的局势非常“危急”。</p>
<p>“炮击甚至一个小时都没有停止。俄罗斯人日夜使用火炮，”谢尔盖·海代说。</p>
<p>“今天，从这些社区获救的57人的每一条生命对我们都很重要。他们没有受伤，已经安全，”他补充说。</p>

<p><strong>乌克兰表示同意与俄罗斯停火只会升级战争</strong></p>
<p>乌克兰总统顾问驳斥了西方的“非常奇怪”的呼吁，即与俄罗斯谈判紧急停火，这将涉及俄罗斯军队留在他们在乌克兰南部和东部占领的领土上。</p>
<p>米哈伊洛·波多利亚克告诉路透社，做出让步将对乌克兰来说适得其反，因为俄罗斯会在战斗中断后进行更猛烈的反击。</p>
<p>“对俄罗斯联邦的任何让步都会立即导致战争升级。所以战争不会停止。它只会暂停一段时间，”他说。</p>
<p>“一段时间后，俄罗斯人会以新的强度加强他们的武器、人力和纠正错误，进行一些现代化改造，解雇许多将军。他们将开始新的进攻，更加血腥，规模更大，将所有错误考虑在内，”波多利亚克补充道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>智库：俄罗斯军队加紧攻占北顿涅茨克</strong></p>
<p>战争研究所表示，俄罗斯军队已加紧包围和占领乌克兰卢甘斯克州的北顿涅茨克市，未来几天可能会继续这样做。</p>
<p>“卢甘斯克的俄罗斯军队可能会利用最近在鲁比日内-北顿涅茨克-卢甘斯克-波帕斯纳弧线取得的进展，包围和围攻北顿涅茨克——乌克兰在卢甘斯克州的最后一个据点，”这家总部位于美国的智库表示。</p>
<p>据战争研究所称，俄罗斯的军事博主正在假设俄罗斯在该地区的战术如何取得成功，并将其称为“北顿涅茨克之战”。</p>

<p><strong>乌克兰导演谴责俄罗斯在戛纳的存在</strong></p>
<p>乌克兰导演德米特罗·苏霍伊基-索布楚克批评戛纳电影节将俄罗斯导演纳入其阵容。</p>
<p>电影节禁止俄罗斯官方代表团参加，但公开反对入侵乌克兰的俄罗斯持不同政见者基里尔·谢列布连尼科夫周三在电影节首映了他的竞赛电影《柴可夫斯基的妻子》。</p>
<p>“当他在这里时，他是俄罗斯宣传的一部分，他们可以使用他，” 苏霍伊基-索布楚克告诉路透社。</p>
<p>俄罗斯导演谢列布连尼科夫本周早些时候曾表示，不应抵制俄罗斯文化，并表示俄罗斯的文化“一直在促进人类价值观”。</p>
<p>乌克兰导演苏霍伊基-索布楚克称，他的国家在与俄罗斯入侵作斗争时同处戛纳电影节的感觉“非常怪异”。</p>

<p><strong>塔斯社：亚速战士中的女性现在是俄罗斯的囚犯</strong></p>
<p>一位亲俄分裂主义领导人说，俄罗斯军队从马里乌波尔被围困的亚速钢铁厂抓获的人中有78名妇女。</p>
<p>俄罗斯塔斯社报道，顿涅茨克分离主义领导人杰尼斯·普希林说，从亚速钢铁厂被俘的人中也有外国人。他没有说明有多少外国人被俘。</p>
<p>“他们有足够的食物和水，他们也有足够的武器，”普希林告诉塔斯社。</p>
<p>“问题是缺乏药物，”他说，指的是在钢铁厂坚守的乌克兰军队。</p>

<p><strong>谈判代表：莫斯科可能用乌克兰囚犯换普京盟友梅德韦丘克</strong></p>
<p>俄罗斯谈判代表表示，莫斯科将考虑用乌克兰亚速营的战俘换取与总统弗拉基米尔·普京关系密切的乌克兰富商维克多·梅德韦丘克。</p>
<p>据俄新社报道，俄罗斯乌克兰问题谈判小组的高级成员列昂尼德·斯卢茨基说：“我们将研究这种可能性。”</p>
<p>斯卢茨基没有透露莫斯科考虑交换的亚速作战人员的数量。乌克兰东部的一名分离主义领导人表示，近2500名乌克兰武装分子被拘留，他们肯定会面临法庭审判。</p>
<p>现年67岁的梅德韦丘克是一名政治家，也是乌克兰最富有的人之一，并以与普京的密切关系而闻名。他在2月俄罗斯入侵乌克兰后逃脱了软禁，但在4月中旬被乌克兰军队重新逮捕。</p>

<p><strong>泽连斯基：乌克兰准备用俄罗斯战俘交换其士兵</strong></p>
<p>泽连斯基说，他的国家准备用俄罗斯囚犯换取在马里乌波尔的亚速钢铁厂投降的部队。</p>
