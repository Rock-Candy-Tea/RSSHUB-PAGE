<li>乌克兰国防部指责俄罗斯在马里乌波尔使用远程轰炸机，当地的一家钢铁厂和港口周边正发生着激烈的战斗。</li>

<h3>以下是这场战争进入第51天以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>地区州长：乌克兰哈尔科夫地区的导弹袭击造成1人死亡、18人受伤</strong></p>
<p>乌克兰东北部哈尔科夫地区州长在“Telegram”上发表声明称，一枚俄罗斯导弹击中了哈尔科夫地区中部的一个地区，造成1人死亡、18人受伤。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>瑞士希望制裁俄罗斯和白俄罗斯的体育官员</strong></p>
<p>瑞士体育和国防部长维奥拉·阿姆赫德在致国际奥委会的一封信中指出，瑞士希望将俄罗斯和白俄罗斯的官员排除在国际体育联合会的高级职位之外。</p>
<p>信中说道，鉴于当前的乌克兰战争，将这两个国家的运动员排除在国际比赛之外的措施已经远远不够。</p>
<p>需要指出的是，包括国际奥委会、国际足联、欧足联和国际篮球联合会(FIBA)在内的多个体育协会的总部都设在瑞士。</p>
<p>根据瑞士联邦运动办公室发言人的说法，他们通常以私营业务的形式组织起来，但是瑞士政府认为，国际奥委会可以对这些协会施加压力，例如发出将其排除在奥林匹克大家庭之外的威胁。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>基辅市长称导弹袭击造成一人死亡、数人受伤</strong></p>
<p>基辅市长维塔利·克利钦科表示，乌克兰首都遭受了导弹袭击，造成1人死亡、数人受伤。</p>
<p>克利钦科补充称，医护人员正在抢救伤者的生命。</p>
<p>他还强调，“基辅过去是，现在也仍然是侵略者们的目标。”</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>官员：俄罗斯在乌克兰各地发动的袭击至少造成两人死亡</strong></p>
<p>当地官员表示，俄罗斯在乌克兰各地发动袭击，并造成至少2位平民死亡，另有4人受伤。</p>
<p>卢甘斯克州长谢尔盖·盖达在网上发帖称，东部地区的炮击已造成1人死亡、3人受伤。盖达在消息应用“Telegram”上发布的一篇帖子中指出，吕尚斯克和西维耶罗顿涅茨克的一条天然气管道被毁，而当地已经断水断气。</p>
<p>盖达在随后发布的一篇帖子中指出，“我们需要在还有可能的时候完成撤离”，他还补充称，已经为那些愿意撤离该地区的人们准备好了大巴车。</p>
<p>波尔塔瓦州州长德米特雷·鲁宁在“Telegram”上表示，波尔塔瓦中部首府波尔塔瓦附近的一个小村庄在夜间遭到袭击，并造成1人死亡、1人受伤。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>莫斯科禁止英国首相鲍里斯·约翰逊入境俄罗斯</strong></p>
<p>俄罗斯外交部表示，已经禁止英国首相鲍里斯·约翰逊、外交大臣特拉斯、国防大臣本·华莱士以及其他10名英国政府成员及政界人士入境俄罗斯。</p>
<p>俄罗斯外交部在一份声明中表示，此举是“鉴于英国政府对俄罗斯采取了前所未有的敌对行动，特别是针对俄罗斯高级官员实施的制裁”，声明补充称，还将很快扩大制裁名单。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>英国国防部表示，撤退的俄罗斯军队对乌克兰道路基础设施的蓄意破坏，阻碍了向之前被占领的地区提供人道主义援助。</p>
<p>英国国防部在其发布一条推文中指出，“俄罗斯军队从乌克兰北部撤出时，破坏桥梁、埋设地雷，并在关键道路上遗弃车辆，从而加剧了这种情况。”</p>
<p>在切尔尼维夫，只有一座横跨德斯纳河的人行桥还得以保留。而在战争爆发之前，当地约有28.5万居民。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>国际文传电讯社：俄罗斯袭击装甲车工厂和军事维修设施</strong></p>
<p>俄罗斯国际文传电讯社援引俄罗斯国防部的话报道称，俄罗斯摧毁了位于基辅的一家装甲车工厂的生产大楼，以及位于尼古拉耶夫市的一家军事维修设施。</p>
<p>声明指出，这些攻击是由高精度远程武器实施的。</p>
<p>国际文传电讯社援引俄罗斯国防部的消息称，俄罗斯还在乌克兰东部哈尔科夫州伊久姆市附近击落了一架乌克兰的苏-25战斗机。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>德国部长：交付坦克并不代表该国参战</strong></p>
<p>德国司法部长表示，向乌克兰运送坦克等重型武器，并不构成对俄开战。</p>
<p>德新社援引马可·布施曼的评论报道称，“因此，如果(乌克兰)行使其合法的自卫权，那么，通过提供武器支持它不能导致我们成为战争一方。”</p>
<p>他还表示，这不仅是他个人的观点，也是德国政府的观点。</p>
<p>自今年2月底的冲突开始以来，德国政界一直忙于解决向乌克兰提供何种武器，以及以多快的速度交付这些武器的问题。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>乌克兰副总理:同意开通9条人道主义走廊</strong></p>
<p>乌克兰副总理伊琳娜·韦列修克表示，双方16日同意通过9条人道主义走廊撤离平民，包括用私家车撤离被围困城市马里乌波尔内的平民。</p>
<p>韦列修克在一份声明中说，在9条人道主义走廊中，有5条来自乌克兰东部的卢甘斯克地区。地方官员表示，当地正在遭受猛烈的炮击。</p>

<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>欧洲对俄罗斯煤炭实施进口禁令导致德国处于十字路口</strong></p>
<p>乌克兰战争带来了一些意想不到的后果。</p>
<p>德国一直在逐步停止采矿，但是在欧盟禁止进口俄罗斯煤炭之后，德国现在推翻了这一决定。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-272883" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/hqdefault.jpg?w=480&resize=480%2C360" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>基辅市长表示，救援人员和医务人员出现在城市郊区的爆炸现场</strong></p>
<p>基辅市长维塔利·克利钦科表示，救援人员和医务人员正在基辅郊区发生爆炸的地点开展工作。</p>
<p>克利钦科在消息应用“Telegram”上发帖称，爆炸发生在基辅的达尔尼茨基区。达尔尼茨基区位于基辅东南部、第聂伯河左岸。</p>
<p>克利钦科补充称，有关伤者的信息正在确认中。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>波罗申科呼吁援助：“武器，武器，武器”</strong></p>
<p>乌克兰前总统波罗申科向半岛电视台记者表示，如果想要结束这场战争，那么乌克兰就需要更多的武器。</p>
<p>波罗申科在一次独家采访中表示，“实现和平的捷径是向乌克兰提供更多的武器”，“这就是为什么我们需要向国际社会索要3样东西：武器、武器和更多的武器。”</p>
<p>他还表示，关键是提供“改变游戏规则”的设备，并称该国需要300辆坦克、1000辆武装运兵车和100架喷气式战斗机。</p>
<p>他还强调，“这些将会彻底改变战争的性质。”</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>基辅、利沃夫发生爆炸，乌克兰全境响起了空袭警报</strong></p>
<p>据报道，基辅和利沃夫在16日凌晨发生了爆炸。</p>
<p>根据路透社的报道，乌克兰大部分地区响起了空袭警报。</p>
<p>目前还没有证实这些爆炸的官方消息。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>在美乌网球赛前向乌克兰致敬</strong></p>
<p>伴随着激动人心的乌克兰国歌旋律，美国和乌克兰之间的比利简金杯（Billie Jean King Cup）预选赛开幕式在眼泪、国旗和比利·简·金本人的烘托下，极度感染人心。</p>
<p>在乌克兰出生的北卡罗来纳州居民尤利娅·卡希列茨演唱了乌克兰的国歌，而包括身披乌克兰国旗的世界第93号种子选手达亚娜·雅斯特雷姆斯卡（Dayana Yastremska）在内的乌克兰队员，都流下了泪水。</p>
<p>这场仪式还包括为“乌克兰人民”默哀。预选赛的获胜者将进入今年11月的决赛。</p>

<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>泽伦斯基估计有2500到3000名乌克兰士兵在战争中丧生</strong></p>
<p>乌克兰总统泽连斯基向美国有线电视新闻网(CNN)表示，到目前为止，估计有2500到3000名乌克兰士兵在战争中死亡。</p>
<p>他还表示，大约有1万人受伤，“很难说有多少人能够活下来”。</p>
<p>泽连斯基还表示，俄罗斯估计损失了1.9万至2万士兵，尽管克里姆林宫认为这一数字为1351人。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>报告：英国特种部队在乌克兰提供训练</strong></p>
<p>英国《泰晤士报》报道称，乌克兰特种部队一直在为驻扎在基辅附近的两个当地营提供训练。</p>
<p>该报援引乌克兰官员的话报道称，这是自俄罗斯7周前入侵以来，他们首次这样做。</p>
<p>乌克兰人正在学习如何使用英国提供的反坦克导弹，而这些导弹是在今年2月份交付的。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>乌克兰总理和高级官员将于下周访问美国</strong></p>
<p>据路透社报道，乌克兰总理杰尼斯·什梅加尔和该国一些高级财政官员下周将在国际货币基金组织和世界银行春季年会期间访问美国。</p>
<p>知情人士向路透社记者指出，预计什梅加尔、乌克兰财政部长马尔琴科、央行行长基里洛·舍甫琴科将与来自七国集团的财政官员和其他官员举行双边会谈，此外，预计还将参加世界银行于下周四举行的有关乌克兰问题的圆桌会议。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>泽连斯基:乌克兰将为马里乌波尔“竭尽所能”</strong></p>
<p>乌克兰总统泽连斯基表示，他在15日与该国军方领导人和情报机构负责人举行的会议上讨论了被围困的港口城市马里乌波尔的命运。</p>
<p>泽连斯基在向全国发表的夜间视频讲话中指出，“目前还不能公布细节，但我们正尽一切努力拯救我们的人民。”</p>
<p>马里乌波尔在这场长达7周的战争中被夷为平地。乌克兰国防部发言人在早些时候表示，该市的局势“非常艰难”。</p>

<p><strong>自战争开始以来，已有7000多名乌克兰人登陆墨西哥</strong></p>
