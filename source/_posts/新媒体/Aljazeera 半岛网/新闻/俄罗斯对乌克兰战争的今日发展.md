<li>卢甘斯克州州长谢尔盖·海代表示，随着俄罗斯军队加强对东部城市的攻击，他正在为“最坏的情况”做准备。</li>

<p><strong>以下是所有最新更新：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称东部袭击进展顺利</strong></p>
<p>俄罗斯表示，在控制了该市郊区的一个地区后，其对乌克兰东部北顿涅茨克的进攻正在顺利进行。</p>
<p>俄罗斯国防部发言人伊戈尔·科纳申科夫在视频声明中说：“北顿涅茨克方向的攻势正在顺利展开。”他补充道，位于城市东郊的Metyolkine居住区已被占领。</p>
<p>“俄罗斯联邦武装部队继续打击乌克兰境内的军事目标，”他说。</p>
<p>科纳申科夫说，远程“口径”巡航导弹袭击了第聂伯罗彼得罗夫斯克州的一个指挥中心，杀死了包括总参谋部人员在内的乌克兰将军和军官。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称袭击了哈尔科夫坦克修理厂</strong></p>
<p>俄罗斯国防部表示，俄罗斯的伊斯坎德尔导弹击中了乌克兰的哈尔科夫坦克修理厂。</p>
<p>该部还表示，在过去10天里，它在乌克兰城市尼古拉耶夫摧毁了西方国家提供的10门榴弹炮和20辆军用车辆。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>意大利埃尼集团入股卡塔尔大型天然气项目</strong></p>
<p>在俄罗斯削减对意大利的供应几天后，有消息称埃尼集团成为卡塔尔能源项目的合作伙伴，该项目旨在扩大世界上最大天然气田的产量。</p>
<p>埃尼集团加入法国公司道达尔能源，后者在价值280亿美元的North Field East项目中拥有6.25%的股份，价值约20亿美元。更多合作伙伴有待宣布。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国情报：俄罗斯、乌克兰军队可能会逃跑</strong></p>
<p>英国国防部根据情报部门提供的信息报告称，最近几周，在东顿巴斯地区作战的乌克兰和俄罗斯部队可能出现逃兵现象，并补充说俄罗斯的士气“很可能依旧特别糟糕”。</p>
<p>声明中写道：“整个俄罗斯部队拒绝服从命令以及军官和他们的部队进行武装对峙的情况继续发生。俄罗斯当局可能难以对持不同政见的军事人员施加法律压力，因为入侵的官方说法是‘特殊军事行动’而不是战争。”</p>
<p>该部还表示，双方继续在北顿涅茨克的北部、东部和南部的轴线上进行猛烈炮击，但前线局势几乎没有变化。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>德国计划采取进一步措施弥补俄罗斯天然气</strong></p>
<p>根据德新社看到的计划，鉴于来自俄罗斯的天然气供应减少，德国经济部长罗伯特·哈贝克正在计划采取额外措施来减少天然气消耗并更换供应。</p>
<p>发电和工业用气将被减少，在冬季之前优先填充储存设施。</p>
<p>德新社从政府圈子获悉，政府正在提供数十亿美元的资金来资助这些措施。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰官员：俄罗斯军队向哈尔科夫挺进</strong></p>
<p>乌克兰内政部一名官员表示，乌克兰第二大城市哈尔科夫以北的局势相当困难，因为俄罗斯军队一直试图再次炮击该市。</p>
<p>“俄罗斯正试图让哈尔科夫成为前线城市，”内政部长顾问瓦迪姆·德尼森科告诉乌克兰国家电视台。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基誓言夺回乌克兰南部</strong></p>
<p>乌克兰总统承诺重新夺回被俄罗斯军队占领的乌克兰南部地区。</p>
<p>“我们不会将南部交给任何人，”泽连斯基在访问南部前线后不久，在夜间视频讲话中说。他发誓要收回“属于乌克兰人的一切”。</p>
<p>他说，乌克兰还将在此过程中恢复出海安全通道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯在乌克兰战场上的袭击越来越激烈</strong></p>
<p>乌克兰军方表示，工业城市北顿涅茨克是莫斯科攻势全面控制卢甘斯克（顿巴斯的两个州之一）的主要目标，它再次面临猛烈的火炮和火箭弹袭击。</p>
<p>“北顿涅茨克的局势非常困难，”乌克兰任命的卢甘斯克州州长谢尔盖·盖代补充说，俄罗斯军队正在使用无人机进行空中侦察，迅速调整打击以应对防御变化。</p>
<p>“桥梁附近地区再次遭到猛烈炮击，”盖代在网上发帖称，并补充说，数百人避难的阿佐特化工厂遭到两次袭击。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：新莫斯科斯克市2人死亡15人受伤</strong></p>
<p>州长瓦伦丁·雷兹尼琴科在Telegram上说，救援人员试图扑灭俄罗斯火箭袭击造成的大火时，新莫斯科斯克地区的一个油箱爆炸，造成至少1人死亡，2人受伤。</p>
<p>雷兹尼琴科说，周六，袭击本身造成至少13人受伤，1人死亡。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯天然气生产商通过乌克兰向欧洲出口的天然气略有上升</strong></p>
<p>俄罗斯天然气工业股份公司表示，通过Sudzha站点过境乌克兰向欧洲供应的天然气从周六的4140万立方米增加到4170万立方米。</p>
<p>该公司表示，通过另一个主要站点Sokhranovka供应天然气的申请被乌克兰拒绝。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报告：亚速高级指挥官被转移到俄罗斯接受调查</strong></p>
<p>据俄罗斯国家通讯社塔斯社报道，两名守卫马里乌波尔亚速斯塔尔钢铁厂的乌克兰高级指挥官已被转移到俄罗斯接受调查。</p>
<p>塔斯社周六援引一位不愿透露姓名的俄罗斯执法消息人士的话说，亚速营副司令斯维亚托斯拉夫·帕拉马尔和第36旅海军陆战队代理旅长谢尔伊·沃伦斯基已被转移到俄罗斯。</p>
<p>塔斯社援引消息人士的话说，特种部队将他们从顿涅茨克转移以便“进行调查”，“乌克兰各部队的其他军官也被运送到俄罗斯。”</p>
<p>在对马里乌波尔进行长达数月的围攻后，俄罗斯军队在 5 月俘虏了数百名战士。莫斯科当时表示，他们被转移到乌克兰东部脱离俄罗斯支持的实体。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>北约斯托尔滕贝格警告乌克兰将发生长达数年的战争</strong></p>
<p>北约秘书长延斯·斯托尔滕贝格警告说，乌克兰的战争可能会持续数年，并呼吁为乌克兰军队配备最先进的武器。</p>
<p>“我们必须为战争可能持续数年时间的事实做好准备。我们不能放弃对乌克兰的支持，”他告诉德国《图片报》。</p>
<p>“即使成本很高，不仅是为了军事支持，还因为能源和食品价格上涨。”</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰在北顿涅茨克附近遭受军事挫折</strong></p>
<p>乌克兰武装部队总参谋部表示，乌克兰军队在北顿涅茨克市东南部的Metolkine遭遇军事挫折。</p>
<p>“由于炮火和攻击，敌人在Metolkine村取得了部分成功，并试图获得立足点，”乌克兰总参谋部在Facebook帖子中说。</p>
<p>但军方表示，乌克兰军队已经在Syrotyne村附近阻止了俄罗斯的进攻。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>卢甘斯克州州长为“最坏的情况”做准备</strong></p>
<p>乌克兰东部卢甘斯克州州长谢尔盖·海代告诉法新社，俄罗斯军队“每天24小时炮击乌克兰部队的阵地”。</p>
<p>“有一种说法：为最坏的情况做好准备，最好的自己会来，”海代在利西昌斯克镇说，那里正在进行街头战斗的准备工作：士兵在挖战壕，架起带刺铁丝网，警察使用烧毁的车辆减缓道路上的交通。</p>
<p>海代呼吁“使远程武器供应尽快送达”，并补充说：“西方帮助我们的事实是好的，但为时已晚。”</p>
