<li>克里姆林宫声称，这些武装人员将受到“人道”待遇，而一名俄罗斯议员认为他们“不配活着”。与此同时，俄罗斯杜马发言人呼吁对他们进行审判。</li>
<li>俄罗斯表示，其部队在乌克兰西部的利沃夫地区通过导弹袭击摧毁了美国和欧洲的武器运输。</li>
<li>俄罗斯副外长表示，莫斯科和基辅之间关于结束战争的谈判已经停止。</li>

<h3>以下是这新战争的今日最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>联合国核监督机构将派遣专家小组前往切尔诺贝利</strong></p>
<p>国际原子能机构总干事表示，联合国核监督机构计划在“未来几周”向乌克兰废弃的切尔诺贝利核电站派遣另外一组专家。</p>
<p>国际原子能机构总干事拉斐尔于上个月底率领第一个代表团前往这座位于乌克兰北部的核电站，他发布推文称，该组织将“在辐射防护、废物管理安全以及核安全方面提供支持”。</p>
<p>俄罗斯军队在今年2月底莫斯科发起军事行动之初，就控制了曾在1986年发生核灾难的切尔诺贝利核电站，并在一个月后撤出。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧盟将批准针对乌克兰的新军事援助</strong></p>
<p>欧盟外交政策负责人博雷尔表示，欧盟国家的国防部长们将批准向乌克兰提供另外5亿欧元(约合5.27亿美元)的军事援助。</p>
<p>博雷尔在前往出席欧盟国防部长会议的途中向记者们指出，“我们必须继续为乌克兰提供武器，这就是我们将会从欧洲和平基金再拨出5亿欧元的原因。”</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>拜登将在白宫会见瑞典和芬兰领导人</strong></p>
<p>白宫方面表示，美国总统乔·拜登本周四将在白宫接待瑞典和芬兰领导人，以讨论其加入北约的申请。</p>
<p>白宫发言人卡琳·让-皮埃尔表示，拜登、瑞典首相安德松和芬兰总统尼尼斯托还将讨论“欧洲安全，加强我们在一系列全球问题上的密切合作以及对乌克兰的支持”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>国际刑事法院向乌克兰派出了有史以来规模最大的调查小组</strong></p>
<p>国际刑事法庭最高检察官表示，该法庭已向乌克兰派遣了一支由42名调查人员、法医专家和后勤人员组成的团队，以调查可能在乌克兰境内犯下的战争罪和反人类罪。</p>
<p>卡里姆·汗在一份声明中指出，“这是自我的办公室成立以来，发起的最大规模的一场现场部署。”</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-281758" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/bucha.png?w=770&resize=770%2C578" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>芬兰议会批准加入北约</strong></p>
<p>芬兰议会以压倒性多数投票赞成该国加入北约。</p>
<p>芬兰议会发言人马蒂·万哈宁表示，已有188名议员投票支持此举。只有8人投了反对票。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>普京称一些欧盟国家不可能放弃俄罗斯的石油</strong></p>
<p>俄罗斯总统普京曾表示，一些严重依赖俄罗斯能源的欧洲国家不可能完全避开俄罗斯的石油供应。</p>
<p>普京在与国内石油经理和政府官员举行的电视会议上表示，西方的制裁以及欧盟可能对俄罗斯石油实施的禁运，都导致了全球石油价格的上涨。</p>
<figure id="attachment_281760" aria-describedby="caption-attachment-281760" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-281760" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/INTERACTIVE-Which-countries-have-the-most-oil-AJLABS.png.png?w=770&resize=770%2C770" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-281760" class="wp-caption-text"><strong>能源</strong><br><strong>哪些国家拥有最多的石油？</strong><br>委内瑞拉、沙特及伊朗拥有全球总量达1.55万亿桶石油储量中的近半数。在根据国际能源署公布的数据，2019年全球的日均石油消耗量为9970万桶。 (半岛电视台)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>紧急服务机构：俄罗斯空袭在切尔尼哈夫导致8人死亡</strong></p>
<p>乌克兰国家紧急服务机构称，俄罗斯对乌克兰北部切尔尼哈夫地区的德斯纳村发动的空袭，造成至少8人死亡、12人受伤。</p>
<p>该地区州长表示，俄罗斯军队在17日凌晨发射了4枚导弹。他还补充称，其中两枚导弹击中了该村内的建筑物。</p>
<p>该州长曾在早些时候表示，该地区“不再存在(俄罗斯)占领者”，但他又警告称，“他们很容易就能找到我们”。</p>
<p>他还警告称，“不要忽视空袭警报。”</p>
<p>半岛电视台记者无法独立证实上述报道的真实性。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>德国外长希望芬兰和瑞典能够尽快加入北约</strong></p>
<p>德国外长表示，尽管土耳其对芬兰和瑞典的加入持保留态度，但她相信芬兰和瑞典会迅速加入北约。</p>
<p>贝尔伯克表示，如果这两个国家申请成为正式成员之间的过渡阶段再延长一些，北约成员国将给予这两个北欧国家适当的安全保障。</p>
<p>贝尔伯克17日在柏林表示，“土耳其方面存在一些悬而未决的问题”，“他们正在讨论这些问题，但是我非常有信心，这两个国家很快就会加入，因为所有人都知道，这将是在非常戏剧性的局势中具有决定性和历史性的时刻。”</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-281764" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/maxresdefault.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰官员称马里乌波尔的防御改变了战争进程</strong></p>
<p>乌克兰总统泽连斯基的一名顾问表示，在俄罗斯的进攻下保卫马里乌波尔的乌克兰士兵坚持了82天，从而改变了战争进程。</p>
<p>米哈伊洛·波多里亚克在电视讲话中表示，关于从该市亚速钢铁厂撤离更多人员的谈判非常困难，但是仍有取得成功的希望。亚速钢铁厂是乌克兰在俄罗斯围攻和轰炸数周后仅剩的最后一个堡垒。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>克里姆林宫：投降的亚速钢铁厂武装人员将会得到“人道”待遇</strong></p>
<p>克里姆林宫发言人德米特里·佩斯科夫表示，在亚速钢铁厂投降的乌克兰武装人员将会受到“人道”待遇以及“按照国际标准”的对待。</p>
<p>佩斯科夫表示，俄罗斯总统普京曾亲自作出这样的保证。</p>
<p>一名参加与基辅和谈的俄罗斯议员在17日早些时候指出，俄罗斯应该考虑对躲藏在亚速钢铁厂内的乌克兰“亚速营”的“民族主义”武装分子处以死刑。</p>
<p>列奥尼德·斯卢茨基在俄罗斯国家杜马（议会下院）举行的一次辩论中表示，“他们犯下了而且还在继续对囚犯实施可怕的反人类罪行，他们不应该继续活下去。”</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯：采取等制措施并驱逐两名芬兰外交官</strong></p>
<p>俄罗斯外交部表示，将驱逐芬兰驻莫斯科大使馆的两名外交官，以报复赫尔辛基决定驱逐俄罗斯两名特使的决定。</p>
<p>俄罗斯外交部还在一份声明中表示，对“芬兰对俄罗斯的敌对路线”提起抗议，这里显然是指芬兰申请加入北约的决定。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>报道：俄罗斯官员呼吁审判“亚速营”武装人员</strong></p>
<p>根据报道，俄罗斯国家杜马发言人表示，从马里乌波尔亚速钢铁厂内撤离的“亚速营”武装人员不应该被用来交换俄罗斯战俘，而是应该作为“战犯”而接受审判。</p>
<p>俄罗斯塔斯社援引该发言人的话报道称，“纳粹罪犯不应该被交换。这些是战犯，我们必须尽一切努力确保他们受到审判。”</p>
<p>莫斯科一直声称“亚速营”是“新纳粹主义组织”，需要指出的是，“亚速营”是一个富有争议的志愿组织，后来转变为乌克兰国民警卫队，并与极右翼势力存在联系。</p>
<p>半岛电视台记者自基辅报道。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>芬兰总统：与土耳其之间的讨论可以“解决”北约内的反对问题</strong></p>
<p>芬兰总统尼尼斯托表示，芬兰和瑞典将能够与土耳其达成协议，尽管土耳其反对这两个北欧国家加入北约这个由30个国家组成的军事联盟。</p>
<p>尼尼斯托在瑞典议会发表的一次讲话中指出，“在过去几天内，土耳其的声明迅速发生了变化，并且变得更加艰难”，“但我相信，在具有建设性讨论的帮助下，我们将解决这个问题。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰称莫斯科因战争失败而处罚其军事领导人</strong></p>
<p>乌克兰国防部称，克里姆林宫已将至少6名俄罗斯军事要员降职、解职或逮捕，原因是他们在莫斯科攻势中的失败。</p>
<p>乌克兰国防部情报部门在一份电报中指出，“为了减轻未来的政治和刑事责任后果，克里姆林宫试图将其所有的失败归咎于其部队的指挥官。”</p>
<p>报道指出，俄罗斯黑海舰队司令伊戈尔·奥西波夫是克里姆林宫处罚的目标之一。俄罗斯国防部指出，奥西波夫因俄罗斯“莫斯科”号导弹巡洋舰今年4月在黑海海域被击沉，而遭到解雇和逮捕。</p>
<p>半岛电视台记者无法独立证实该部门的说法。</p>
<p>半岛电视台自基辅发回报道。</p>

<p><strong>美国敦促盟国加大对乌克兰的资助</strong></p>
<p>美国财政部长珍妮特·耶伦呼吁华盛顿的盟友加大对乌克兰的财政支持，并称目前宣布的资金不足以满足乌克兰抗击俄罗斯攻势的“基本需求”。</p>
<p>耶伦在准备向布鲁塞尔经济论坛发表的讲话中指出，“乌克兰的融资需求很大。”</p>
<p>她还补充称，“在税收恢复正常之前的几个月内，乌克兰需要预算资金来支付士兵、雇员和退休人员的工资，并运行一个满足公民基本需求的经济”，“短期内，它将需要转向修复和恢复关键的公用事业及服务。”</p>
<p>她还指出，尽管乌克兰最终将需要“大规模的支持”来重建和恢复，就像二战后欧洲的马歇尔计划那样，但是乌克兰将不得不“一步一步地”采取行动。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称摧毁了乌克兰西部的武器运输</strong></p>
<p>俄罗斯国防部表示，俄罗斯军队在乌克兰西部利沃夫地区向包括美国在内的基辅西方盟友运送的武器发射导弹，并摧毁了这些武器。</p>
<p>利沃夫地区州长表示，俄罗斯军队再次袭击了该地区的一处主要军事设施。基辅方面没有立即作出回应。</p>
<p>半岛电视台无法独立证实俄罗斯国防部的报道。</p>

<p><strong>俄罗斯官员称与乌克兰的谈判已经停止</strong></p>
<p>俄罗斯国际文传电讯社援引俄罗斯副外长的话报道称，莫斯科和基辅没有就结束战争举行“任何形式”的会谈。</p>
<p>“不，谈判没有进行。乌克兰实际上已经退出了谈判进程。”</p>
<p>双方的前几轮面对面会谈以及网上讨论都未能就停止冲突达成任何协议。</p>
