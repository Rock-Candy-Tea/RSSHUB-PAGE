<li>据俄罗斯国家通讯社报道，两名英国人和一名摩洛哥人在为乌克兰作战期间被捕，他们可能会在乌克兰东部的一个俄罗斯代理人法庭内被判处死刑。</li>
<li>俄罗斯和土耳其表示支持在黑海建立一条安全走廊，允许乌克兰的出口粮食，但是基辅拒绝了这个建议，认为它并不可信。</li>

<h3>以下是这场战争进入今日以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>舒尔茨重申了德国对北约的承诺</strong></p>
<p>德国政府发言人表示，鉴于俄罗斯对乌克兰的战争，德国总理舒尔茨与北约秘书长斯托尔滕贝格通电话，并讨论了更新北约战略目标的问题。</p>
<p>这位发言人表示，这场电话会议旨在为计划于6月底在马德里举行的北约峰会奠定基础。舒尔茨在通话中重申了德国政府为北约作贡献的决心，以帮助北约实现其未来10年的目标。</p>
<p>两人原定9日在柏林举行会晤，但是在斯托尔滕贝格被诊断出患上带状疱疹后，双方不得不改为电话通话。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>意大利总理：乌克兰需要保证港口以运输粮食</strong></p>
<p>意大利总理德拉吉表示，乌克兰需要保证其港口不会受到攻击，才能运出粮食货物。</p>
<p>德拉吉在巴黎举行的经合组织部长级会议上发表讲话时表示，“我们需要疏散因冲突而困在当地的数百万吨粮食。联合国和土耳其的调解努力将是非常重要的步骤。”</p>
<p>他还补充称，“我们必须向乌克兰总统泽连斯基保证，这些港口不会受到攻击。”</p>
<figure id="attachment_288992" aria-describedby="caption-attachment-288992" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-288992" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/EU-2.png?w=770&resize=770%2C515" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-288992" class="wp-caption-text">意大利总理马里奥·德拉吉发表讲话 (美联社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯称并未与土耳其就乌克兰粮食出口问题达成协议</strong></p>
<p>克里姆林宫方面表示，并没有与土耳其就乌克兰粮食出口问题达成协议。</p>
<p>克里姆林宫发言人佩斯科夫向记者们表示，相关工作仍在继续。</p>
<p>土耳其一直在推动俄罗斯和乌克兰就恢复从乌克兰港口出口粮食的计划达成协议，不过由于俄乌两国都指责对方破坏了全球粮食供应，达成协议的前景目前看来非常渺茫。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>大众汽车公司为退出俄罗斯主要工厂的员工提供奖金</strong></p>
<p>大众汽车公司表示，该公司将向俄罗斯下诺夫哥罗德一家汽车装配厂的员工提供奖金，条件是他们自愿辞职。</p>
<p>大众公司发言人证实了早前的一篇媒体报道，报道声称下诺夫哥罗德工厂的近200名员工将获得经济补偿和医疗保险直至2022年底。</p>
<p>今年3月，大众汽车公司宣布，由于西方制裁，该公司位于卡卢日加和下诺夫哥罗德的工厂生产将会暂停，直至另行通知，并将立即停止向俄罗斯出口汽车。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>克里姆林宫：预计不会再削减对欧洲客户的天然气供应</strong></p>
<p>克里姆林宫方面曾表示，预计俄罗斯天然气工业股份公司不会再切断对欧洲客户的天然气供应，并补充称，该公司让买家使用卢布支付天然气费用的计划正在发挥预期的作用。</p>
<p>俄罗斯天然气工业股份公司已经切断了对部分欧洲国家的天然气供应，原因是这些国家拒绝以卢布支付俄罗斯的天然气款项。这是克里姆林宫为应对西方制裁而制定的一项新的计划。</p>
<figure id="attachment_288917" aria-describedby="caption-attachment-288917" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-288917" src="https://chinese.aljazeera.net/wp-content/uploads/2022/06/AP22117532119709.jpg?w=770&resize=770%2C592" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-288917" class="wp-caption-text">俄罗斯天然气工业股份公司已经切断了对部分欧洲国家的天然气供应，原因是这些国家拒绝用卢布支付俄罗斯的天然气款项 (美联社)</figcaption></figure>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>泽连斯基：由于俄罗斯封锁港口，数百万人可能会挨饿</strong></p>
<p>乌克兰总统泽连斯基警告称，由于俄罗斯对乌克兰黑海港口的封锁，数百万人可能会挨饿，他还表示，这种封锁让世界“处于可怕的粮食危机的边缘”。</p>
<p>俄罗斯占领了乌克兰大部分沿海地区，封锁了农产品出口，从而推高了粮食价格。</p>
<p>泽连斯基表示，乌克兰现在无法出口大量小麦、玉米、植物油和其他“在全球市场上起到稳定作用”的产品。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>西方武器将在“2至3天”内结束北顿涅茨克的战斗</strong></p>
<p>乌克兰一名地区官员表示，西方的远程火炮将使乌克兰能够击退俄罗斯军队，并在几天内收复北顿涅茨克。</p>
<p>卢甘斯克地方长官海岱在社交媒体的官方频道上接受采访时表示，“只要我们拥有远程火炮，便能够与俄罗斯炮兵进行决斗，而我们的特种部队也就能够在2至3天内清理这座城市。”</p>
<p>莫斯科军队正将火力集中在这个具有重要战略意义的工业中心上，这是夺取乌克兰东部大片地区的努力之一。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰在俄罗斯的战争中看到了水域污染的威胁</strong></p>
<p>乌克兰总统泽连斯基曾表示，俄罗斯入侵乌克兰，对包括亚速海在内的水域造成了巨大的污染威胁。</p>
<p>泽连斯基在经济合作与发展组织(OECD)部长级会议上发表讲话时强调，在俄罗斯于今年2月24日对乌克兰发起军事行动之后，世界需要考虑用绿色能源取代俄罗斯的天然气供应。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>北顿涅茨克市长：不可能进行撤离，近一万人被困</strong></p>
<p>北顿涅茨克市长表示，乌克兰军队仍然控制着北顿涅茨克市的工业区和邻近地区，形势“艰难但仍然可控”。</p>
<p>他还表示，尽管俄罗斯的炮火猛烈，但乌克兰的防线仍在坚守，然而，现在无法从北顿涅茨克疏散民众。他还表示，大约有一万平民仍滞留在这座城市内，这里现在仍是俄罗斯对乌克兰的进攻重点。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>州长：俄罗斯炮轰北顿涅茨克并陷入激烈巷战</strong></p>
<p>乌克兰东部的北顿涅茨克巷战激烈，据该地区州长表示，俄罗斯军队摧毁了“一切可以用于防卫的东西”。</p>
<p>卢甘斯克州长表示，俄罗斯正在炮击仍由基辅控制的卢甘斯克地区，但是乌克兰军队一旦收到远程火炮，就能够“清理”该市。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：俄罗斯恢复对伊久姆的攻势</strong></p>
<p>英国国防部表示，俄罗斯军队已恢复向伊久姆镇南部推进的努力，这场攻势自4月份以来一直处于停滞状态。</p>
<p>英国国防部表示，俄罗斯可能试图从伊久姆的行动中重组其东部地面部队，因为他们“在向基辅推进的过程中伤亡惨重，但是他们部队的兵力可能仍然不足”。</p>
<p>英国国防部表示，“俄罗斯可能寻求在这一地区恢复势头……进一步向北顿涅茨克施加压力，并让它有机会向顿涅茨克深处推进。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英美烟草公司称正就转移俄罗斯业务而进行谈判</strong></p>
<p>英美烟草公司表示仍在努力转让其俄罗斯的业务，并且有信心实现其财务目标，无论需要多长时间才能完成出售。</p>
<p>这家生产香烟的公司坚持其2022年的收入和收益预期，并称由于乌克兰和俄罗斯冲突中的不确定性，全球烟草行业的年度销量预计会下降3%。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>塔斯社：俄罗斯副总理称尽管受到制裁，但经济仍在增长</strong></p>
<p>根据塔斯社的报道，俄罗斯副总理阿列克谢·奥弗丘克表示，尽管受到制裁，但该国经济一直在增长。</p>
<p>阿列克谢·奥弗丘克在努尔苏丹举行的俄罗斯与哈萨克斯坦合作会议上表示，“多亏俄罗斯总统、政府和俄罗斯银行采取的措施，针对我们国家的经济战并没有按照那些发动这场战争的人的计划进行。”</p>
<p>他还表示，俄罗斯经济在过去4个月内增长了近1.7%，“在今年5月，通货膨胀几乎停止了，5月份最后一个星期的通货膨胀率为0，在上周，我们进入了通货紧缩。”</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>官员：俄罗斯没有法律义务帮助乌克兰出口粮食</strong></p>
<p>俄罗斯宪法立法和国家建设委员会的一名高级官员表示，俄罗斯没有法律义务为乌克兰的粮食出口提供便利。</p>
<p>这位名叫安德烈·克里沙斯的官员在Telegram上写道，“俄罗斯根本没有为从乌克兰出口粮食建立物流链，或者以某种方式为这种出口提供便利的法律义务。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>顿涅茨克州长：当地4人死亡、11人受伤</strong></p>
<p>顿涅茨克州长帕夫洛·基里连科表示，俄罗斯军队8日在顿涅茨克杀害了4位平民，另有11人受伤。</p>
<p>基里连科在Telegram上表示，库拉霍夫镇有3位平民死亡，阿夫迪夫卡镇有一位平民死亡，“马里乌波尔还有4名受害者，其中包括两名儿童”，但没有说明这些受害者是否死亡。</p>
<p>他还补充称，被占领的马里乌波尔和沃尔诺瓦卡的确切受害者人数目前尚不清楚。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>卢甘斯克州长：4人死亡，至少6人受伤</strong></p>
<p>卢甘斯克地区州长海岱表示，俄罗斯在卢甘斯克地区发动的袭击造成至少4人死亡。</p>
<p>海岱在Telegram上表示，俄罗斯人向北顿涅茨克的Azot化工厂开火，他此前曾表示，有很多平民在厂区内避难。</p>
<p>海岱指出，在北顿涅茨克，有一名男子在6日受伤后于8日不治身亡。而在利西昌斯克，俄罗斯向该镇及其周边地区发射了“迫击炮、大炮和榴弹”，造成3人死亡。他还补充称，至少有6人在袭击中受伤。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄新社：华为公司关闭在俄罗斯的零售店</strong></p>
<p>俄新社援引与华为公司关系密切的消息人士的话报道称，电信巨头华为已开始关闭其在俄罗斯的零售门店，部分原因在于需求下降。</p>
<p>根据俄新社的报道，华为在俄罗斯的19家门店中，已有4家关闭，其余门店可能很快也会关闭。</p>
<p>消息人士给出的另一个原因是仓库产品短缺。</p>
