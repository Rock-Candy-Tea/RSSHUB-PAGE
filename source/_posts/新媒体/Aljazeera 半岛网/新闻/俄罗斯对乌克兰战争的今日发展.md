<div>   
<p>乌克兰当局否认被围困的马里乌波尔市已落入俄军之手，并表示，在南部沿海城市剩余的乌克兰军队将战斗到最后，与此同时，乌克兰总统泽连斯基表示，与俄罗斯对话以结束战争的机会正在逐渐消失。</p>
<p>另一方面，乌克兰外长德米特罗·库莱巴在接受美国CBS网络采访时预计，乌克兰军队与俄罗斯军队在乌克兰东部顿巴斯地区的战斗将愈演愈烈。</p>
<h3>以下是这场战争第53天的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>半岛电视台记者：南赫尔松阵线目睹了俄罗斯和乌克兰军队之间的持续战斗。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰通讯社：俄军第49防空师司令伊万·格拉茨上校阵亡。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰安全部门播放了一段政治家维克托·梅德韦丘克的视频，他在视频中要求俄罗斯总统普京在马里乌波尔用他交换囚犯。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯电视台播放了两名英国人的录音，称他们在乌克兰被捕，并要求英国首相鲍里斯·约翰逊就释放他们进行谈判。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>半岛电视台记者：克拉马托尔斯克和乌克兰东部顿涅茨克省所有城市都响起了警笛。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>马里乌波尔市长顾问：</h3>
<ul>
<li>卡尔米乌斯河左岸地区的战斗仍在继续，俄罗斯军队使用各种武器轰炸亚速斯塔尔综合体（城市东部）。</li>
<li>城市中没有医生，这使人道局势恶化。</li>
<li>平民死亡人数已达2.2万人左右，预计还会增加。</li>
</ul>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-273433" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/image-61.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>卢甘斯克军事管理当局：</h3>
<ul>
<li>俄罗斯军队控制了克雷梅纳，撤离行动成为不可能。</li>
<li>4 名平民在克雷梅纳的撤离行动中被俄罗斯军队杀害。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总参谋部：自战争开始以来，俄罗斯军队总损失约为20600人。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰副总理：出于安全原因，今天不会开放人道主义走廊。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>卢甘斯克军事管理当局：俄罗斯人正在攻击卡里米纳，该市目前正在目睹街头战斗，不可能进行撤离行动。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总统府：从今天开始，所有欧盟国家的港口都对俄罗斯船只关闭。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>利沃夫军事管理局局长：在俄罗斯对该市的轰炸中，包括一名儿童在内的 6 人死亡，8 人受伤。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>俄罗斯国防部：</h3>
<ul>
<li>我们昨晚用高精度导弹摧毁了乌克兰军队的16个军事设施。</li>
<li>高精度制导导弹摧毁了普文科沃市和多布卢布利（东部）附近的燃料和弹药储存设施。</li>
<li>我们在波帕斯尼亚、扬博尔和克拉马托尔斯克（东部）用导弹袭击摧毁了 4 个武器和军事装备仓库。</li>
<li>两架乌克兰战斗机和一架苏-25攻击机被炸，11架无人机被击落。</li>
<li>向切尔尼戈夫（乌克兰与白俄罗斯边境附近）发射的 10 枚导弹被拦截。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>利沃夫市长安德烈·萨多维：五枚导弹袭击了利沃夫市，并封锁了多条街道。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯国防部：乌克兰军队使用Tochka-U 导弹系统瞄准了扎波罗热（东南部）的托克马克地区。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯中央银行允许外汇交易，此前，由于西方对莫斯科的制裁，俄罗斯中央银行在 3 月初暂停外汇交易。</p>
<p>卢甘斯克当局（东部地区）：俄罗斯昨晚轰炸该市警察大楼，造成6名警察受伤。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>第聂伯罗彼得罗夫斯克军事管理局：俄罗斯导弹袭击摧毁了该市的铁路基础设施。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯航空局：将俄罗斯南部和西南部11个机场的关闭时间延长至4月25日。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>英国国防部：乌克兰在马里乌波尔的强烈抵抗减缓了俄罗斯军队在乌克兰其他地区的进展。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总统顾问在推特上发文称：利沃夫市（西部）的民用基础设施遭到五枚俄罗斯导弹的袭击。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰媒体：俄罗斯对基辅西南部瓦西尔科夫镇的目标进行了轰炸袭击。</p>
<p>乌克兰媒体援引卢甘斯克市长的话报道称：在俄罗斯对佐洛特市的一个体育场馆的轰炸中，至少有两人死亡，另有多人受伤。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>苏梅地区（东部）行政长官：俄罗斯军队仍在城市边界，我们不建议居民返回。</p>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰媒体援引第聂伯罗彼得罗夫斯克市（东部）市长说法报道称：</p>
<ul>
<li>俄罗斯军队今天早上用导弹轰炸了这座城市。</li>
<li>60%的俄罗斯导弹被乌克兰的防御系统击落。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰媒体援引敖德萨地区军事管理局发言人说法报道称：</p>
<ul>
<li>俄罗斯军队计划轰炸赫尔松，随后指责乌克兰军队对此负责。</li>
<li>俄罗斯正试图利用轰炸赫尔松作为掩护，旨在那里举行公投。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>半岛电视台记者：乌克兰南部敖德萨市的警笛声响起。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总理丹尼斯·什米加尔告诉美国广播公司新闻网说：马里乌波尔被围困，但并未沦陷，那里的乌克兰军队将战斗到最后一口气。</p>
