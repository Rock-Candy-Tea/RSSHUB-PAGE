<p>以下是所有最新更新：</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>北约希望在候选期间为芬兰、瑞典提供安全保证</strong></p>
<p>北约秘书长延斯·斯托尔滕贝格表示，北约将考虑在芬兰和瑞典从申请成员资格到加入的过渡期间为他们提供安全保障。</p>
<p>“芬兰和瑞典对过渡时期感到担忧。我们将努力加快过渡进程，”北约秘书长说。</p>
<p>“我们将研究提供安全保证的方法，包括增加北约在该地区的存在。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>芬兰、瑞典寻求加入北约，土耳其提出要求</strong></p>
<p>土耳其外交部长表示，瑞典和芬兰在寻求加入北约时，必须停止支持本国的“恐怖分子”，提供明确的安全保障，并解除对土耳其的出口禁令。</p>
<p>梅夫吕特·恰武什奥卢说，安卡拉没有威胁任何人，也没有寻求影响力，只是特别谈到瑞典对库尔德工人党的支持。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>一个“历史性的决定”</strong></p>
<p>在芬兰宣布有意申请加入北约后，芬兰外交部长佩卡·哈维斯托表示，芬兰正在做出“历史性决定”。</p>
<p>“通过加入北约，芬兰将加强自身和整个欧洲的安全。我们正在为后代做出这一历史性决定，” 外交部长哈维斯托说。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>芬兰愿意与土耳其讨论其加入北约的问题</strong></p>
<p>芬兰总统绍利·尼尼斯托表示，芬兰准备与安卡拉就其提出的北约成员国问题进行对话。</p>
<p>土耳其一再批评瑞典和芬兰对其视为“恐怖组织”的处理，包括库尔德工人党和叙利亚库尔德人民保护部队。</p>
<figure id="attachment_281006" aria-describedby="caption-attachment-281006" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-281006" src="https://chinese.aljazeera.net/wp-content/uploads/2022/05/14-8.png?w=770&resize=770%2C514" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-281006" class="wp-caption-text">土耳其外交部长梅夫吕特·恰武什奥卢抵达德国柏林参加为期两天的北约外长会议 (路透)</figcaption></figure>
<p>阿纳多卢通讯社援引他的话说，在北约最高外交官会议开始之前，土耳其外交部长梅夫吕特·恰武什奥卢强调，两国对库尔德工人党的支持“对我们人民的感情产生了负面影响”。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>北约认为俄罗斯的“野蛮入侵”步履蹒跚</strong></p>
<p>一名北约高级官员表示，俄罗斯在乌克兰的军事进展似乎步履蹒跚，他表示基辅有望赢得这场战争。</p>
<p>北约高级外交官周日在柏林开会，讨论向乌克兰提供进一步支持，以及芬兰、瑞典和其他国家在面对俄罗斯威胁时加入西方联盟的举措。</p>
<p>“俄罗斯的野蛮入侵正在失去动力，”北约副秘书长米尔恰·吉阿纳告诉记者。“我们知道，凭借乌克兰人民和军队的勇敢，在我们的帮助下，乌克兰能够赢得这场战争。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯邻国芬兰宣布打算加入北约</strong></p>
<p>芬兰宣布，这个北欧国家打算申请加入北约。此举为拥有30个成员的西方军事联盟在俄罗斯与乌克兰的战争中扩大势力范围铺平道路。</p>
<p>绍利·尼尼斯托总统和桑娜·马林总理在赫尔辛基总统府举行的联合新闻发布会上宣布了这一消息。</p>
<p>预计芬兰议会将在未来几天内批准该决定，但这被认为是一种形式。然后，最有可能在下周的某个时候，正式的成员资格申请将提交给布鲁塞尔的北约总部。</p>

<li><strong>更新</strong></li>

<p><strong>针对粮食短缺风险的慈善战争</strong></p>
<p>德国慈善机构Welthungerhilfe的负责人马蒂亚斯·莫格警告世界各地粮食短缺的后果。</p>
<p>“埃及、肯尼亚、南苏丹、黎巴嫩和许多其他国家直接或间接地严重依赖俄罗斯和乌克兰的出口，”他告诉RND媒体集团。“这些国家无法再收到他们订购的货物，或者不得不为此支付更多费用。”</p>
<p>莫格说，由于气候变化、冲突和大流行等因素，食品价格已经升至战前的历史新高。</p>
<p>“当联合国世界粮食计划署不得不在2015年削减叙利亚难民难民营的口粮时，这是大规模难民流向欧洲的触发因素之一。 我们不应该忘记这一点。”</p>

<p><strong>北约副手称赞欧洲歌唱大赛获胜者</strong></p>
<p>欧洲歌唱大赛和北约通常不会联系在一起，但该军事联盟的副秘书长米尔恰·吉阿纳祝贺乌克兰用一首“优美的歌曲”赢得了年度音乐比赛，并称这首歌曲证明了乌克兰与俄罗斯作战时的勇敢。</p>
<p>“我要祝贺乌克兰赢得欧洲歌唱大赛。这不是一个轻松的声明。我们昨天看到了整个欧洲和澳大利亚对勇敢的巨大公众支持，这首歌当然很美，它很美，”这位外交官说。</p>

<p><strong>库列巴：更多武器前往乌克兰</strong></p>
<p>乌克兰最高外交官德米特罗·库列巴在德国首都柏林与美国外交官安东尼·布林肯会面后表示，乌克兰将获得更多武器和支持。</p>
<p>“更多的武器和援助正在运往乌克兰，”库列巴在推特上说。</p>
<p>他补充说，两国还承诺密切合作，以确保“乌克兰食品出口抵达非洲和亚洲的消费者手中”。</p>

<p><strong>利沃夫遇袭事件更新</strong></p>
<p>利沃夫地区州长马克西姆·科齐茨基提供了有关俄罗斯凌晨在该地区发动袭击的更多信息。</p>
<p>他在Telegram上说，“四枚敌方导弹击中了靠近波兰边境的雅沃里夫地区的军事基础设施，该设施被完全摧毁，”并补充说没有受害者。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>德国表示已准备好迅速批准芬兰和瑞典的北约成员国资格</strong></p>
<p>德国外交部长安娜莱娜·贝尔博克表示，如果芬兰和瑞典决定申请加入北约，德国已为快速批准程序做好一切准备，同时强调两国都需要安全保障。</p>
<p>“如果他们决定加入，他们可以迅速加入。我们必须确保给他们安全保障，不能有他们的身份不明的过渡期和灰色地带，”她在德国首都柏林对记者说。</p>
<p>她指的是可能需要长达一年的批准期，在此期间，两国将不受北约第5条的保护，该条款规定对一个盟友的攻击就是对所有人的攻击。</p>

<p><strong>北约副总干事对芬兰、瑞典成员身份有信心达成共识</strong></p>
<p>北约副部长米尔恰·吉阿纳表示，他相信土耳其对芬兰和瑞典加入防御性军事联盟的担忧能够得到解决。</p>
<p>吉阿纳说，“土耳其是一个重要的盟友，并表达了朋友和盟友之间的担忧，”并补充说，如果两国决定申请加入，他相信土耳其盟友能找到“达成共识的所有条件”。</p>
<p>他在发表此评论前，瑞典和芬兰准备加入北约，而这一举动遭到北约成员国土耳其的反对。土耳其总统雷杰普·塔伊普·埃尔多安指责斯堪的纳维亚国家是“恐怖组织的旅馆”。</p>

<p>别尔哥罗德州州长维亚切斯拉夫·格拉德科夫在Telegram上说，乌克兰袭击俄罗斯村庄塞雷达后，一个人因弹片受伤。</p>
<p>该镇位于舍别金斯基区，毗邻乌克兰边境。</p>

<p><strong>官方：导弹击中利沃夫的军事基础设施</strong></p>
<p>乌克兰利沃夫州州长报告说，西部地区发生了导弹袭击。</p>
<p>马克西姆·科齐茨基说，周日早些时候发射的导弹袭击了该地区的一些军事基础设施。</p>
<p>他表示，“目前没有关于死亡或受伤的信息，”并补充说，“正在查明破坏的程度。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>英国：俄罗斯的顿巴斯攻势“失去动力”</strong></p>
<p>英国国防部表示，俄罗斯在乌克兰顿巴斯地区的进攻“已经失去动力，大大落后于计划”。</p>
<p>在最新的情报简报中，该部表示，俄罗斯“在过去一个月内未能取得实质性的领土收益，同时持续保持高水平的人员损失”。</p>
<p>简报称，俄罗斯现在很可能损失了其在2月份投入的地面作战力量的三分之一。</p>
<p>该部补充说：“俄罗斯军队越来越受到能力下降、士气持续低迷和战斗力下降的限制。在目前的情况下，俄罗斯不太可能在未来30天内大幅加快推进速度。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>欧洲歌唱大赛冠军准备回家战斗</strong></p>
<p>卡卢什乐团的主唱说，他和他的乐队成员回国后，“将准备再次与俄罗斯作战”。</p>
<p>乌克兰在欧洲歌唱大赛上的胜利在周日凌晨得到证实后，奥莱·普修克（Oleh Psiuk）在都灵举行的新闻发布会上发表了讲话。</p>
<p>普修克说：“我们有临时授权在这里，有效期两天后就结束了，两天后我们将回到乌克兰。很难说我到底要做什么，因为这是我第一次赢得欧洲歌唱大赛，但就像每个乌克兰人一样，我们已经准备好尽可能多地战斗，直到最后。”</p>
<p>目前乌克兰政府禁止18至60岁的男性离开该国，但全男性乐队的6名成员获得了特别许可，可以在音乐比赛中代表乌克兰和乌克兰文化。</p>
<p>其中1名最初的成员​​留在了乌克兰继续战斗。</p>

<p><strong>亚速营部队的亲属向中国习近平寻求帮助</strong></p>
<p>被困在马里乌波尔亚速钢铁厂的乌克兰士兵亲属呼吁中国国家主席习近平“拯救”被围困的军队，称他是最后一位莫斯科听从的世界领导人。</p>
<p>五名乌克兰士兵的妻子和一位父亲周六在基辅举行了新闻发布会，再次绝望地呼吁帮助躲在被俄罗斯军队围困数周的巨大工厂的地下隧道中的士兵。</p>
<p>斯塔夫·维奇尼亚克，一名被困士兵的父亲说：“世界上只有一个人可以帮主我们，那就是中国领导人。中国对俄罗斯和（俄罗斯总统）普京个人都有很大影响。我们要求他进行干预，”</p>

<p>在等待俄罗斯军队允许他们离开数天后，一支载有从马里乌波尔废墟中逃离的难民的大型汽车和货车车队抵达乌克兰控制的扎波罗热市。</p>

<p>“我们勉强做到了，我们中间有很多老人。这次旅行是毁灭性的，但是值得的，”他在车队在黑暗中抵达后说道。</p>
<p>马里乌波尔市长的一名助手早些时候表示，车队里有500至1000辆汽车，是自俄罗斯入侵乌克兰以来该市最大的一次撤离。</p>
