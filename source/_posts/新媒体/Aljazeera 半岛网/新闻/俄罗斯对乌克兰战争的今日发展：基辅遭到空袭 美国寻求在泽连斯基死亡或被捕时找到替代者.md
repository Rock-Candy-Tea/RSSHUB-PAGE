<p>俄罗斯对乌克兰的战争已经进入第11天，有迹象表明俄罗斯军队可能会发动大规模进攻以占领乌克兰首都基辅。</p>
<p>乌克兰军队证实击落俄罗斯飞机并在尼古拉耶夫南部）和切尔尼戈夫（北部）俘虏了一些飞行员；基辅和莫斯科互相指责对方阻挠了从被围困的马里乌波尔市（南部）撤离平民的计划。</p>
<p>俄罗斯总统弗拉基米尔·普京似乎对其部队发起的军事行动将实现目标充满信心，乌克兰总统弗拉基米尔·泽连斯基呼吁俄罗斯控制地区的公民进行示威和抵抗；此时，有消息称双方将周一举行新一轮谈判。</p>
<p>以下是俄罗斯对乌克兰开战第11天的最新进展：</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>《纽约时报》援引西方政府消息来源：华盛顿及其盟友正在寻求在泽连斯基死亡或被捕的情况下找到替代者。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰国防部：俄军正寻求控制首都基辅以南的卡涅夫水电站。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>维萨和万事达卡制裁</h3>
<p>白宫宣布，乔·拜登总统对维萨和万事达卡暂停在俄罗斯业务的决定表示欢迎。</p>
<p>路透社援引包括俄罗斯最大银行在内的俄罗斯联邦储蓄银行称，维萨和万事达卡的服务在俄罗斯仍然可用，两家公司的制裁不会影响其在俄罗斯境内的客户。</p>
<p>同时，《华尔街日报》报道称，维萨和万事达卡在2020年俄罗斯所有支付交易中的占比为74%。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>白宫：拜登正在与国会协调，以确保向乌克兰提供额外的财政援助。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总参谋部：敌人正在逼近基辅和哈尔科夫，并试图控制卡涅夫水电站。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>向波兰提供战斗机</h3>
<p>Politico网站援引美国官员：如果波兰决定将其米格-29战斗机转移到乌克兰，华盛顿正在与华沙讨论向其提供美国战斗机的可能性。</p>
<p>该网站称，白宫承诺讨论向波兰提供美国战机的可能性，并不反对华沙向乌克兰派遣战机的想法。</p>
<p>一位白宫发言人告诉Politico，如果决定将战斗人员转移到乌克兰，提供可填补波兰空缺的工作正在进行中。</p>
<p>波兰曾询问拜登政府，如果其飞机被派往乌克兰，是否有可能获得美国战斗机。</p>
