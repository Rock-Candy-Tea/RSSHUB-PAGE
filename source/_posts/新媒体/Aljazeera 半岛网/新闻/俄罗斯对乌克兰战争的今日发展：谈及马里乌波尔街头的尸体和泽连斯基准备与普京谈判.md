<p>有证词报告说，被围困的沿海城市马里乌波尔的街道上到处都是尸体，这座城市正在遭遇俄罗斯军队轰炸和围攻，与此同时，乌克兰总统泽连斯基表示，愿意与俄罗斯总统普京进行会晤。</p>
<p>乌克兰当局早些时候表示，俄罗斯轰炸了马里乌波尔的一所学校，这所学校有 400 人，包括儿童、妇女和老人。</p>
<p>马里乌波尔市议会在Telegram上写道：“我们知道，这座建筑已被摧毁，和平的平民仍在废墟中，正在等待死亡人数统计。”</p>
<p>随着多条战线战斗的继续，俄罗斯军队继续轰炸乌克兰首都基辅，并有伤亡报告。</p>
<h3>以下是这场战争第26天的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>白宫：拜登总统将于周五前往波兰讨论乌克兰的人道主义危机。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰军队：</strong></p>
<p>在最初的损失之后，俄罗斯空军改变了进攻策略。</p>
<p>乌克兰空军发言人尤里·伊格纳特表示，“在 25 天内，俄罗斯占领者在地面和空军方面遭受了重大损失：96 架飞机、118 架直升机、数十枚巡航导弹和无人机。”</p>
<p>伊格纳特表示，为了应对这些损失，出现了一项新战略，在两天的时间里，越来越多地使用侦察无人机来监测导弹袭击的结果。</p>
<p>该信息无法独立核实。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>俄罗斯无人机在乌克兰南部港口城市敖德萨市及周边地区的活动有所增加。</p>
<p>地区军事指挥官马克西姆·马里琴科表示，“乌克兰空军向敌人的无人机开火……并证实其中一些无人机被击落。”</p>
<p><img loading="lazy" class="size-arc-image-770 wp-image-265954" src="https://chinese.aljazeera.net/wp-content/uploads/2022/03/image-1-65.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>三名美国人表示，他们与乌克兰士兵一起对抗俄罗斯军队。</p>
<p>三人谈及他们奇迹般地逃脱，此前，周日，他们乘坐的一辆车在首都基辅附近的前线撞上了地雷。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>周一晚些时候，乌克兰总统府宣布恢复与俄罗斯的停火谈判。</p>
<p>乌克兰总统府顾问米哈伊尔·波多利亚克表示，谈判将通过视频会议进行。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰总统：</strong></p>
<p>以色列正在为安排我们与俄罗斯之间的高级别和谈做出很多努力，可能在耶路撒冷举行该会谈。</p>
<p>在通过视频向以色列议会发表讲话后，泽连斯基表示，以色列总理贝内特正试图充当基辅和莫斯科之间的调解人。</p>
<p>泽连斯基补充说，“当然，以色列有自己的利益和保护其公民的战略。我们了解这一切。”</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-265945" src="https://chinese.aljazeera.net/wp-content/uploads/2022/03/image-96.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰副总理伊琳娜·韦列舒克表示，乌克兰放弃马里乌波尔市并放下武器是不可能的。</p>
<p>新闻门户《乌克兰真理报》援引韦列舒克的说法报道称，“任何投降或放下武器都是不可能的，”她并补充说，“我们已经就此告知了俄罗斯方面。”</p>
<p>上周日，俄罗斯呼吁乌克兰军队在被围困的东部沿海城市放下武器。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总统表示愿意与俄罗斯总统就停战进行谈判。</p>
<p>在美国有线电视新闻网播出的采访中，泽连斯基表示，“我已准备好与（弗拉基米尔·普京）谈判，过去两年，我已经做好了准备，我相信，没有谈判，战争就不会停止。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>基辅市市长维塔利·克里琴科宣布，一枚炮弹在首都基辅一栋公寓楼前爆炸，造成至少 5 人受伤，其中两人被送往医院。</p>
