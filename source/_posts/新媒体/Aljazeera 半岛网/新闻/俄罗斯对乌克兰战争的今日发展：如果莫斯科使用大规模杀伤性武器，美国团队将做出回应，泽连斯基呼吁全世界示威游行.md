<p>俄罗斯对乌克兰的战争已经进入第 29 天，鉴于各条战线的战斗仍在继续，外交努力持续不断，美国倾向于在周四对俄罗斯立法者实施更多制裁，与此同时，预计美国总统乔·拜登在访问布鲁塞尔期间将发起一项向欧洲供应天然气的倡议，旨在帮助其减少对俄罗斯能源的依赖。</p>
<p>《纽约时报》透露，在战争开始 4 天后，美国总统乔·拜登政府组建了一个名为“老虎队”的国家安全官员工作组，每周秘密会面 3 次，其任务之一是研究如何应对俄罗斯在乌克兰使用化学、生物或核武器，以及如何应对战争扩大到邻国以及在北约国家领土内运送武器的车队遭到袭击的事态发展。</p>
<h3>以下是这场战争第29天的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>半岛电视台记者：今天早上，乌克兰首都基辅警笛声响起。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>谷歌的困难：</h3>
<p>路透社援引谷歌说法报道称，俄罗斯访问其新闻应用程序和网站存在困难，访问新闻应用程序和网站的困难与其本身的任何技术问题无关。</p>
<p>早些时候，俄罗斯国际文传电讯社报道称，俄罗斯通信监管机构 (Roskomnadzor) 阻止了Alphabet 旗下谷歌提供的新闻服务，指责它允许访问所谓的关于俄罗斯在乌克兰军事行动的虚假消息。</p>
<p>一项新的俄罗斯法律规定，以一种所谓的有损俄罗斯军队名誉的方式报道事件是犯罪行为。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>法新社援引德国议会消息人士的说法报道称：柏林拟向乌克兰追加两千枚反坦克导弹。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>法新社援引瑞典外长的说法报道称：我们将向乌克兰提供5000件反装甲武器。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>乌克兰总统泽连斯基：</h3>
<ul>
<li>乌克兰没有对俄罗斯的安全构成威胁，并且正在尽一切努力结束战争。</li>
<li>我呼吁世界各地举行示威游行，反对俄罗斯在乌克兰的军事行动。</li>
<li>俄罗斯使用各种武器，并试图从世界各地引进雇佣军。</li>
<li>我们正在等待北约和七国集团就俄罗斯入侵我国采取严肃措施。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>《华盛顿邮报》援引五角大楼的说法报道称：</h3>
<ul>
<li>自从对乌克兰的战争开始以来，美国国防部长劳埃德·奥斯汀和参谋长马克·米利一直试图联系俄罗斯同行，但没有成功。</li>
<li>到目前为止，俄罗斯国防部长和陆军参谋长一直拒绝与奥斯汀和米利进行交流。</li>
</ul>

<li><strong>更新</strong></li>

<h3>《纽约时报》：</h3>
<ul>
<li>拜登组建了一个由国家安全官员组成的乌克兰问题工作组。</li>
<li>如果俄罗斯在乌克兰使用化学、生物或核武器，该小组将研究如何应对。</li>
<li>白宫根据美国国家安全顾问在战争开始 4 天后签署的备忘录组建了该小组。</li>
<li>《纽约时报》：这个被称为“老虎队”的小组，也在考虑如果武器车队在北约领土内遭到袭​​击，则将进行报复。</li>
<li>“老虎队”每周秘密会面 3 次，还讨论了如果俄罗斯将战争扩大到邻国将如何应对事宜。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>《华盛顿邮报》援引一位美国官员的话报道称：</h3>
<ul>
<li>拜登将在周四宣布新的制裁措施，其中一些针对俄罗斯立法者。</li>
<li>预计拜登将在访问布鲁塞尔期间宣布一项向欧洲供应天然气的倡议。</li>
<li>拜登的倡议涉及将液化天然气运输到欧洲。</li>
<li>拜登的倡议旨在帮助欧洲减少对俄罗斯能源的依赖。</li>
</ul>
