<p>乌克兰总统泽连斯基表示，南部和顿巴斯地区的局势仍然极其困难，并重申，俄罗斯正在被围困的马里乌波尔附近集结军队。</p>
<p>泽连斯基在深夜视频中补充道，“未来会有战斗，我们仍然需要走一条非常艰难的路才能得到我们想要的一切。”</p>
<p>与此同时，乌克兰官员周四晚间宣布，俄罗斯军队已离开自 2 月 24 日乌克兰战争开始以来他们一直占领的切尔诺贝利核电站，并带走了人质。</p>
<p>乌克兰政府机构“Energoatom”援引员工的话报道称：“俄罗斯占领者在离开切尔诺贝利核电站时带走了国民警卫队成员，将他们扣为人质。”目前尚不清楚被俄罗斯军队控制的乌克兰士兵人数。</p>
<p>该机构指责俄罗斯军队在离开核电站时“抢劫建筑物并偷窃设备和其他贵重物品”，而乌克兰专家将在核电站内搜索任何“简易爆炸装置”。</p>
<h3>以下是这场战争第37天的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰布罗瓦里市长：俄军已部分撤出基辅郊区布罗瓦里。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>国际文传电讯社：在针对别尔哥罗德储油设施背景下，与乌克兰接壤的库尔斯克地区已提高警戒状态。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>俄罗斯国防部：</h3>
<ul>
<li>我们的部队正在进行搜索和侦察行动，并在基辅附近地区进行梳理，目的是驻扎在这些地区。</li>
<li>我们阻止了乌克兰军队向戈斯托梅利-布查-奥泽拉移动。</li>
<li>我们的部队控制了从伊尔平到基辅的 5 公里地带。</li>
<li>摧毁了 6 个乌克兰军事设施，包括 5 个弹药库、导弹库和火炮武器库。</li>
<li>俄罗斯空军对 52 个乌克兰军事目标发动了空袭，其中包括 3 个指挥点。</li>
<li>自战争开始以来，我们已经摧毁了 353 架乌克兰无人机、1839 辆坦克和 52 座乌克兰军事设施。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>与中国的峰会：</h3>
<p>随着人们越来越担心北京与莫斯科的持续和解，及不愿谴责俄罗斯对乌克兰的袭击，欧盟将于周五与中国召开一次虚拟峰会，此前，乌克兰总统泽连斯基警告俄罗斯不要在他的国家南部蔓延。</p>
<p>法国外交部主管欧洲事务的国务秘书克莱门特·伯恩表示，此次会议“将重点关注我们敦促中国发挥的作用，即毫不含糊地站在国际法原则一边，对俄罗斯施加一切必要的影响和压力。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>路透社援引马里乌波尔市助理市长说法报道称：俄罗斯军队正在阻止援助物资抵达该市，离开该市变得非常危险。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总参谋部：俄罗斯军队继续从基辅西部向白俄罗斯边境撤军。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>半岛电视台记者：炮击的目标是乌克兰外国情报局在基辅的大楼。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>俄罗斯紧急情况部：</h3>
<ul>
<li>别尔哥罗德的燃料库爆炸造成两人受伤，消防队正在努力防止火势蔓延。</li>
<li>乌克兰的轰炸导致 8 个别尔哥罗德油箱起火，并有可能蔓延到其他 8 个油箱。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>质的发展：</h3>
<p>一位地区负责人表示，周五，两架乌克兰战机在低空越过边境后，轰炸了俄罗斯别尔哥罗德市的一个燃料储存设施。</p>
<p>该地区的州长维亚切斯拉夫·格拉德科夫补充说，爆炸导致火灾，造成两名工人受伤，同时该市靠近乌克兰边境的一些地区的居民被疏散。</p>
<figure id="attachment_269068" aria-describedby="caption-attachment-269068" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-269068" src="https://chinese.aljazeera.net/wp-content/uploads/2022/04/1-1.png?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-269068" class="wp-caption-text">乌克兰对俄罗斯别尔哥罗德燃料库的轰炸是同类事件中的首次 (社交网站)</figcaption></figure>
<p>此事态发展两天前，武器库爆炸震动了该地区，格拉德科夫表示，据信这是由另一场火灾引起的，并补充说，地方当局正在等待国防部的确认。</p>

<li><strong>更新</strong></li>
</ul>
<p>英国国防部：俄罗斯正在从格鲁吉亚重新部署其部队，以加强对乌克兰的入侵。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>永久存在：</h3>
<p>乌克兰总参谋部周五早些时候在一份公报中表示，俄罗斯军队希望在乌克兰东部和南部保持军事存在。</p>
<p>军方领导人谈到俄罗斯试图在其占领的顿涅茨克、卢甘斯克、扎波罗热和赫尔松地区的领土上建立政府，预计这些地区的战斗将继续。</p>
<p>乌克兰总统泽连斯基已经警告称，俄罗斯打算在赫尔松（乌克兰南部）举行“虚假”公投，以便在那里建立一个“人民共和国”，类似于它在乌克兰东部分离地区宣布的那样。</p>
