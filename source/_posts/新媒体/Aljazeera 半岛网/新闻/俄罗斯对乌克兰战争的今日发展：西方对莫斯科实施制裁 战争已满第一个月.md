<p>俄罗斯对乌克兰发动的战争在今天（25日）已满第一个月，而背景是乌克兰境内多条战线上的战斗仍在持续、对俄罗斯及其领导人的制裁进一步收紧，以及西方召开峰会承诺对莫斯科进一步升级。</p>
<p>比利时首都布鲁塞尔在一天之内召开了北约（北大西洋公约组织）、七国集团和欧盟的3场峰会，这些会议都认为有必要继续对俄罗斯施压，以停止其针对乌克兰的战争，此外，这些会议还强调了西方对乌克兰的支持。</p>
<p>实地方面，乌克兰宣布在基辅周边的部分地区击退了俄罗斯部队，并指责俄军阻止平民从马里乌波尔撤离，与此同时，俄罗斯对哈尔科夫（乌克兰东北部）的轰炸造成了人员伤亡。</p>
<h3>以下是这场战争进入第30天以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>《华盛顿邮报》：美国情报部门得出的结论是，俄罗斯军队上个月参与了针对乌克兰通信设备发动的黑客攻击。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总统泽连斯基：</p>
<ul>
<li>乌克兰守军在各处阻挡了敌人，我感谢所有人在战争开始以来的一个月内对俄罗斯占领者发起的抵抗。</li>
<li>自2月24日与俄罗斯爆发战争以来，已经有超过2000名乌克兰儿童遭到绑架。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>美国负责政治事务的国防部副部长科林·卡尔向路透社表示：</p>
<ul>
<li>俄罗斯已经在对乌克兰的战争中耗尽了精确制导弹药，而且很可能会开始依靠所谓的哑弹（非制导）和火炮。</li>
<li>我并不认为俄罗斯总统普京想要与北约发生全面冲突。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>CNN援引五角大楼的消息称：</p>
<ul>
<li>无论乌克兰战争如何结束，欧洲的安全环境都发生了变化。</li>
<li>北约领导人正在讨论欧洲大陆未来的安全环境会是什么样子的。</li>
<li>在过去一周内，我们与俄罗斯国防部长和俄罗斯陆军总参谋长进行联络的尝试，均未取得成功。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>美国国务院：</p>
<ul>
<li>对俄罗斯、朝鲜和中国的5个实体及个人实施制裁。</li>
<li>实施制裁是由于相关实体及个人与平壤的核活动相关。</li>
<li>这些制裁凸显了俄罗斯作为这项令人不安的计划推动者所发挥的负面作用。</li>
</ul>
