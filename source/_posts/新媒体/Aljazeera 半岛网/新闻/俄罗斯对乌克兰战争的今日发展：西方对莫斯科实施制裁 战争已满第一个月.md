<div>   
<p>俄罗斯对乌克兰发动的战争在今天（25日）已满第一个月，而背景是乌克兰境内多条战线上的战斗仍在持续、对俄罗斯及其领导人的制裁进一步收紧，以及西方召开峰会承诺对莫斯科进一步升级。</p>
<p>比利时首都布鲁塞尔在一天之内召开了北约（北大西洋公约组织）、七国集团和欧盟的3场峰会，这些会议都认为有必要继续对俄罗斯施压，以停止其针对乌克兰的战争，此外，这些会议还强调了西方对乌克兰的支持。</p>
<p>实地方面，乌克兰宣布在基辅周边的部分地区击退了俄罗斯部队，并指责俄军阻止平民从马里乌波尔撤离，与此同时，俄罗斯对哈尔科夫（乌克兰东北部）的轰炸造成了人员伤亡。</p>
<h3>以下是这场战争进入第30天以来的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>哈尔科夫市警方：俄罗斯导弹袭击了乌斯诺维扬斯基区一家诊所，造成4位平民死亡和另外3人受伤。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<ul>
<li>哈尔科夫当局：俄罗斯军队在过去24小时内对该市的不同街区发动了55次空袭和240场炮击。</li>
<li>哈尔科夫当局：我们击落了两架俄罗斯飞机和一枚弹道导弹，并摧毁了西线上的一些坦克。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>法新社：美国和欧盟宣布成立工作组以降低对俄罗斯天然气的依赖。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<ul>
<li>俄罗斯国防部：使用“Kalibr”巡航导弹轰炸了基辅郊区的一个燃料库。</li>
<li>俄罗斯国防部：自军事行动开始以来，已经摧毁了204个防空系统、261架乌克兰乌克人和1587辆坦克。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>俄罗斯国防部：在昨天夜间摧毁了51处隶属乌克兰武装部队的军事设施。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰副总理：从马里乌波尔、梅利托波尔到扎波罗热的人道主义走廊今天将会开放。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>顿涅茨克地区分裂主义分子：在过去24小时内，共有58名乌克兰士兵在前线战斗中死亡。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>第聂伯军管局局长：俄罗斯军队使用两枚导弹轰炸了一个军事单位，我们正继续搜寻幸存者。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>俄罗斯安全委员会副主席德米特里·梅德韦杰夫：美国不再是全球霸主，当它认为自己没有竞争对手时，它就错了。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>中国外交部发言人：乌克兰局势对全球供应网络造成了严重影响。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰副总理：从马里乌波尔开启的人道主义走廊今天将向私家车开放。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>英国交通大臣：我们正尽一切努力帮助乌克兰，但是我们不想扩大这场战争。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>波兰边防部队：自俄罗斯发动进攻以来，已经有223.7万乌克兰难民进入了我们的领土。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰紧急服务部：在过去24小时内，我们已经分别在基辅和哈尔科夫扑灭了因俄罗斯的轰炸引发的13起火灾和6起火灾。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>俄罗斯国家安全委员会副主席：当欧洲讨论拒绝购买俄罗斯天然气的可能性时，他们只是在假装勇敢。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰通讯社：在基辅巴里雪夫卡地区的抵抗行动中，击毙了40名俄军士兵，并摧毁了多部军事车辆。</p>

<h3>英国的估计</h3>
<p>英国国防部表示，乌克兰军队已经重新控制了基辅以东35公里处的城市和防御阵地，并且指出，这些部队可能会继续推动俄罗斯人从基辅向戈斯托梅尔机场的方向撤退。</p>
<p>英国国防部表示，俄罗斯军队正试图绕过尼古拉耶夫，因为他们希望向西往敖德萨方向转移。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>俄新社援引俄罗斯安全委员会副主席的话报道称：任何俄罗斯人对乌克兰军事行动的反对，都是叛国之举，并且大多数俄罗斯人都支持这场行动。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总检察长：自俄罗斯入侵乌克兰开始以来，已有135名儿童丧生，另有184名儿童受伤。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰新闻社援引梅利托波尔市市长的话报道：今天将从该市撤离350人，并将送入人道主义援助。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>俄罗斯的目标</h3>
<p>俄罗斯卫星通讯社援引俄罗斯安全委员会副主席德米特里·梅德韦杰夫的话报道称，西方试图通过实施经济制裁来影响莫斯科是愚蠢之举。</p>
<p>梅德韦杰夫表示，西方对俄罗斯实施的限制，旨在让俄罗斯人民反对他们的政府，并推动该国公民反对普京总统的政策。</p>
<p>这位俄罗斯安全委员会副主席还强调，俄罗斯的军事行动将会继续进行，直至实现其目标——使乌克兰去军事化并使其成为中立国。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-267232" src="https://chinese.aljazeera.net/wp-content/uploads/2022/03/image-1-75.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<h3>取得部分成功</h3>
<p>乌克兰总参谋部表示，分离主义分子部分成功地保护了他们在顿涅茨克和卢甘斯克的控制区，并且指出，“敌人”控制该国东部的企图没有成功，与此同时，分离主义分子正试图包围首都。</p>
<p>乌克兰总参谋部表示，俄罗斯军队正准备在基辅以东的布罗瓦里和普罗斯波尔战线上重新发起进攻，它还指出，乌克兰军队继续在顿涅茨克、斯洛博赞斯基和特瓦里亚的部分地区开展其防御行动。</p>
<p>该部补充称，乌克兰部队昨天击落了一架俄罗斯战斗机、一架无人机和4枚巡航导弹，此外，乌克兰军队仍然控制着切尔尼戈夫市，并且阻碍着俄罗斯军队向基辅方向进军。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总统与埃及总统通了电话，双方一致认为有必要恢复乌克兰的和平。</p>

<li><strong>更新：</strong></li>
</ul>

<li><strong>更新：</strong></li>
