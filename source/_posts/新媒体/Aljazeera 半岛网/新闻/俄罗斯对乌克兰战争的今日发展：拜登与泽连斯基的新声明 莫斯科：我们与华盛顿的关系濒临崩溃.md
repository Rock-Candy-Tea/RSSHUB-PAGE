<p>乌克兰总统泽连斯基重申了会见俄罗斯总统普京的意愿，而美国总统拜登则再次对俄罗斯人发起抨击，指责他们为在乌克兰使用违禁武器而寻找借口，对此，莫斯科表示，它与华盛顿之间的关系已处于崩溃边缘。</p>
<p>美国已承诺向乌克兰提供它在几十年前秘密获得的、由苏联制造的防御系统，而乌克兰总统则强调，该国的战士正奋勇抵抗“俄罗斯的入侵”。</p>
<p>在人道主义层面，乌克兰当局表示，在21日共有8057人通过7条人道主义走廊撤离了乌克兰境内正发生战争的多个城镇。</p>
<h3>以下是这场战争进入第27天后的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>法国外交部：</strong></p>
<p>勒德里昂与布林肯在电话通话中同意加强乌克兰核设施安全的必要性。</p>
<p>勒德里昂与布林肯同意继续加强针对俄罗斯的制裁。</p>
<p>勒德里昂与布林肯强调，有必要在乌克兰达成全面的停火协议。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰政府21日指责俄罗斯军队在乌克兰东南部城市梅利托波尔绑架了3名以色列人。</p>
<p>乌克兰副总理伊琳娜·韦列舒克在Facebook上发帖称，“俄罗斯占领者今天在梅利托波尔绑架了3名以色列公民——塔添娜·库穆克及其父母维拉、米哈伊尔·库穆克。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>联合国大会将于23日召开特别会议，以讨论有关乌克兰人道主义局势的决议草案。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-266258" src="https://chinese.aljazeera.net/wp-content/uploads/2022/03/image-3-28.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<p><strong>乌克兰总统：</strong></p>
<ul>
<li>俄罗斯敌人试图向前推进，但是我们仍奋勇抵抗。俄罗斯部队手段残忍，并且轰炸了一个旨在撤离妇女和儿童的人道主义车队。</li>
<li>在这样的背景下，乌克兰总统表示，如果不与俄罗斯总统普京举行会晤，就不可能通过谈判结束发生在本国的这场战争。</li>
<li>此外他还重申，他在本月早些时候便承认，乌克兰目前可能无法获得北约（北大西洋公约组织）的成员国资格。泽连斯基表示，“我认为，在我们与俄罗斯联邦总统举行会晤之前，你们无法真正了解他们愿意为停止这场战争做些什么，也无法真正了解在我们未能接受这样或那样的解决方案的情况下，他们又准备做些什么。”这位乌克兰总统近一年来一直寻求会见俄罗斯总统普京，但是普京却表示拒绝，并坚持要求泽连斯基结束该国境内与俄罗斯相关的分裂主义地区的“内战”。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>俄罗斯外交部：</strong></p>
<p>我们召见了美国大使约翰·沙利文并告诉他，美国总统乔·拜登将俄罗斯总统普京称为“战争犯”的措辞，将两国关系推向了崩溃边缘。</p>
<p>拜登曾在上周表示，普京是一名“战争犯”，因为他向乌克兰派遣了数十万士兵并以平民作为袭击目标。</p>
<p>俄罗斯方面强调，针对它的敌对行动将受到“果断且坚决的回击”。</p>
<p>另一方面，美国国务院发言人内德·普赖斯表示，美国认为有必要与俄罗斯保持沟通渠道，尤其是在“冲突时期”。</p>
<p>他还补充称，“我们试图在莫斯科维持我们的外交存在。”</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>

<p><strong>美国总统乔·拜登：</strong></p>
<ul>
<li>俄罗斯人正在谈论乌克兰拥有生化武器的问题，而这表明他们打算使用这类武器。</li>
<li>受到网络攻击威胁的不仅仅是美国的公司，还有其国家利益。</li>
<li>早些时候，我曾就任何针对我们或我们的盟友发动网络攻击的后果，向俄罗斯总统普京发出警告。</li>
<li>当普京开始在乌克兰边境集结军队时，我曾呼吁北约国家团结起来。</li>
<li>我要强调的是，在俄罗斯对乌克兰发动战争的情况下，北约从未像现在这般团结。</li>
</ul>
