<div>   
<p>一位美国官员宣布，华盛顿及其盟国将把苏联制造的坦克转移至乌克兰顿巴斯地区，以支持该国的防御行动，而英国军方情报部门则发布了针对多条战线的战争发展评估。</p>
<p>另一方面，基辅宣布了俄罗斯军队在过去一周内的巨大死亡人数，与此同时，俄罗斯驻伦敦大使就英国可能向乌克兰运送武器的问题向该国发出警告。</p>
<p>乌克兰总统泽连斯基警告称，乌克兰东部局势依然十分严峻，俄罗斯正准备在顿巴斯和哈尔科夫实施新的打击，并要求西方国家为乌克兰提供战斗机以保卫其国家。</p>
<p>另一方面，半岛电视台记者报道称，在哈尔科夫市东北部萨尔托夫卡地区发生强烈爆炸和火灾之后，有部分人员受伤。</p>
<p>乌克兰首都基辅市长维塔利·克里琴科在4月1日表示，乌克兰军队和俄罗斯军队在基辅北部和东部爆发了“大规模”的战斗，并警告那些已经逃离了首都的居民不要急于返回。</p>
<p><strong>以下是这场战争进入第38天的最新进展：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>车臣领导人在社交网站上表示：我们的战士已经开始袭击马里乌波尔地区，我们不会让恐怖分子活着。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>第聂伯罗市市长：俄罗斯对乌克兰中部第聂伯罗的巴甫洛格勒地区实施了轰炸。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰尼古拉耶夫紧急事务部门：俄罗斯对该市行政大楼的轰炸造成的死亡人数已经上升至35人。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰总统顾问：我们的部队已经重新控制了基辅范围内30多个人口稠密的地区。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰通讯社：乌克兰摄影记者马克斯·莱文被发现在基辅附近的一个小镇上遇害。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>模棱两可的评论</strong></p>
<p>乌克兰总统泽连斯基表示，乌克兰不会接受与俄罗斯军队的战斗出现除了胜利之外的任何结果。</p>
<p>泽连斯基在接受美国“福克斯新闻”采访时表示，保卫该国土地的问题不容讨论。</p>
<p>当被问及他是否使用了直升机对俄罗斯境内的一个俄罗斯军用仓库发动袭击时，泽连斯基回答称，“对不起，我不会谈论我作为陆军总司令所下达的任何命令。”</p>
<p>当地时间4月1日，在两架乌克兰直升机成功越过边境并轰炸了俄罗斯的一处石油设施后，莫斯科指责基辅攻击了俄罗斯城市别尔哥罗德。但是乌克兰国家安全委员会秘书奥列克西·丹尼洛夫否认该国对此次袭击负有责任。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>土耳其国防部长：我们可以派遣船只从乌克兰的马里乌波尔撤离平民和伤员。</p>

<p><strong>俄罗斯的袭击</strong></p>
<p>俄罗斯在最近一次针对乌克兰的导弹袭击中，宣布摧毁了另外67个军事目标。</p>
<p>俄罗斯国防部表示，在2日早上，距乌克兰首都基辅东南300公里的商业和工业城市克列门丘克附近的一个汽油和柴油仓库被摧毁。</p>
<p>该部还表示，在波尔塔瓦市附近和第聂伯罗市附近的两个军用机场已经停止运行。</p>
<p>俄罗斯国防部发言人伊戈尔·科纳申科夫表示，导弹是从军舰和飞机上发射的。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
