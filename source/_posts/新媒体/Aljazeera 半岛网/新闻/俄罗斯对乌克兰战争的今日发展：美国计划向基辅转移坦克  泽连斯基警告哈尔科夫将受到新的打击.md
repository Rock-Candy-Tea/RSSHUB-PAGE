<div>   
<p>乌克兰总统泽连斯基警告称，乌克兰东部局势依然十分严峻，俄罗斯正准备在顿巴斯和哈尔科夫实施新的打击，并要求西方国家为乌克兰提供战斗机以保卫其国家。</p>
<p>另一方面，半岛电视台记者报道称，在哈尔科夫市东北部萨尔托夫卡地区发生强烈爆炸和火灾之后，有部分人员受伤。</p>
<p>乌克兰首都基辅市长维塔利·克里琴科在4月1日表示，乌克兰军队和俄罗斯军队在基辅北部和东部爆发了“大规模”的战斗，并警告那些已经逃离了首都的居民不要急于返回。</p>
<p>以下是这场战争进入第38天的最新进展：</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>德国武装部队监察长：</strong></p>
<ul>
<li>在任何情况下，我们都不会在乌克兰领土上部署我们的部队。</li>
<li>如果伊斯坦布尔谈判没有取得成功，那么我们将看到乌克兰境内的游击战增多。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>乌克兰总统顾问：</strong></p>
<ul>
<li>并不会像一些人预测的那样出现耗尽俄罗斯能力的长期冲突。</li>
<li>俄罗斯将退出除南部和东部以外的所有地区，并且试图强加其条件。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰副总理：已经同意开放人道主义走廊，以疏散马里乌波尔和别尔江斯克地区的平民。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>乌克兰总参谋部：</strong></p>
<ul>
<li>俄罗斯军队在过去一周内损失了第二旅内的近800名士兵。</li>
<li>俄罗斯军队正在进行重新部署，以便在斯洛博赞斯基、顿涅茨克和卢甘斯克开展进一步的行动。</li>
<li>俄罗斯正令其部队在摩尔多瓦的德涅斯特河分离地区开展行动。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>英国军方情报机构：</strong></p>
<ul>
<li>随着俄罗斯军队继续从基辅郊区撤出，乌克兰军队继续向前推进。</li>
<li>经过激烈的战斗，乌克兰军队设法在该国东部的哈尔科夫确保了一条主要道路。</li>
<li>俄罗斯军队从戈斯托梅尔机场撤出，该机场自冲突爆发以来一直是双方的交战场。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>路透社：美国军方取了消洲际弹道导弹试验，以缓解与俄罗斯之间的核紧张局势。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>路透社援引中国外交部欧洲司司长的话报道：中方无意规避对俄罗斯实施的制裁。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>五角大楼（美国国防部）的援助</strong></p>
<ul>
<li>五角大楼：美国将向乌克兰提供额外价值3亿美元的安全援助。</li>
<li>五角大楼：向基辅提供额外安全援助的决定，强调了我们对维护乌克兰主权和领土完整的承诺。</li>
<li>五角大楼：对乌克兰的安全援助包括激光制导导弹系统、无人机和弹药。</li>
<li>五角大楼：美国继续与盟国及其伙伴合作，提供额外援助以支持乌克兰的作战能力。</li>
<li>五角大楼：面对俄罗斯的侵略，我们将继续动用所有可用的工具，来支持乌克兰的武装部队。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>乌克兰的坦克</strong></p>
<ul>
<li>《纽约时报》援引一名美国官员的话报道：拜登政府及其盟友将把苏联制造的坦克运往顿巴斯地区，以支持乌克兰的防御。</li>
<li>《纽约时报》援引一名美国官员的话报道：将坦克运送至顿巴斯地区的行动即将开始。</li>
<li>《纽约时报》援引一名美国官员的话报道：这些坦克将使乌克兰能够对顿巴斯地区的俄罗斯目标发动远程炮击。</li>
<li>《纽约时报》：美国官员拒绝透露即将运往顿巴斯地区的坦克数量，或者这些坦克来自哪些国家。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总统：我们成功营救了6266人，其中包括在马里乌波尔营救的3071人。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总统：人道主义走廊1日在3个地区内运行，即顿涅茨克、卢甘斯克和扎波罗热。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总统：马里乌波尔市共有超过3000人获救。</p>

<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>路透社：美国商务部将俄罗斯和白俄罗斯的120家实体列入了制裁名单。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p><strong>泽连斯基的声明</strong></p>
<ul>
<li>泽连斯基向“福克斯”新闻表示：乌克兰东部的局势仍然非常困难，俄罗斯正准备对顿巴斯和哈尔科夫发动新的打击。</li>
<li>乌克兰总统向“福克斯”新闻表示：莫斯科拥有更多的武器，应当再为我们提供战斗机以实施防御。</li>
<li>乌克兰总统向“福克斯”新闻表示：必须通过外交途径来结束对克里米亚半岛的占领。</li>
<li>乌克兰总统向“福克斯”新闻表示：我们需要主要国家的安全保障，而俄罗斯清楚这一点。</li>
<li>乌克兰总统向“福克斯”新闻表示：我们已准备好讨论克里米亚半岛的问题。</li>
</ul>

<li><strong>更新：</strong></li>
</ul>
<p>半岛电视台记者：哈尔科夫市东北部的萨尔托夫卡地区发生强烈爆炸和火灾，并造成人员受伤。</p>
