<p>俄罗斯宣布自今日（3月31日）开始在乌克兰东南部的马里乌波尔停火，旨在从这座被围困的城市内撤离平民。</p>
<p>与此同时，一名乌克兰谈判代表证实，与俄罗斯举行的和平谈判将在本周五（4月1日）通过线上方式恢复。</p>
<p>另一方面，乌克兰总统泽连斯基表示，“我们绝不会做出任何让步，我们将为我们的每一寸土地和每一位同胞而战。”</p>
<h3>以下是这场战争进入第36天的最新进展：</h3>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>CNN援引美国参议院情报委员会主席的话报道：</strong></p>
<ul>
<li>普京夺取乌克兰主要城市并推翻该国政权的企图已被挫败。</li>
<li>必须向乌克兰提供武器，并避免北约分裂或爆发世界大战。</li>
<li>应当为乌克兰配备远程防空导弹系统。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>路透社援引乌克兰谈判代表的话报道：</strong></p>
<p>与俄罗斯之间的和平谈判将于本周五（4月1日）通过线上方式恢复。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p><strong>乌克兰总统：</strong></p>
<ul>
<li>乌克兰和俄罗斯之间的和谈仍在继续，但是目前仍只是空话。</li>
<li>基辅正在与华沙进行谈判，以推动其关闭它与白俄罗斯之间的边界。</li>
<li>我们召回了驻摩洛哥和格鲁吉亚的大使，因为二者未能说服摩洛哥和格鲁吉亚支持乌克兰。</li>
</ul>

<ul style="list-style-type:square">

<p><strong>乌克兰总统顾问：</strong></p>
<ul>
<li>担保国机制包括在任何侵略发生后的3天内提供军事和人道主义援助。</li>
<li>作为联合国安理会的常任理事国，俄罗斯将成为担保国之一，但却没有一票否决权。</li>
