<div>   
<p>基辅指责俄罗斯军队开始破坏乌克兰城市的油库和食品商店。与此同时，卢甘斯克分离主义领导人表示，他们正计划就加入俄罗斯进行公投。</p>
<p>乌克兰总统弗拉基米尔·泽连斯基今天周日黎明时要求西方国家向他的国家提供一小部分军事装备，称这对维护“欧洲的自由”是必要的；同时华盛顿宣布在一个月内对俄罗斯的470多名个人和196个实体实施了制裁。</p>
<p>另一方面，英国指责俄罗斯军队将目标对准乌克兰人口稠密的地区，并表示轰炸从俄罗斯领空内进行，因为担心这些飞机暴露在乌克兰的防御系统之下。</p>
<p><strong>以下是这场战争第32天的最新进展：</strong></p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>塔斯社援引俄罗斯国家杜马：现在不是就卢甘斯克加入俄罗斯举行公投的时候。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>路透社援引乌克兰军事情报部门负责人：</p>
<ul style="list-style-type:disc">
<li>游击战可能很快在俄罗斯占领的地区开始。</li>
<li>俄罗斯在未能完全占领乌克兰后试图分裂乌克兰。</li>
<li>俄罗斯正试图将其占领的地区统一为一个类似国家的实体。</li>
<li>俄罗斯正试图在乌克兰建立朝鲜和韩国的模式。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>波兰边防卫队：自俄罗斯发动进攻以来，已有230万乌克兰难民进入波兰。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰国防部：俄罗斯职业杀手目前正在马里乌波尔对乌克兰人进行种族灭绝。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>卢甘斯克分裂主义领导人：我们计划就加入俄罗斯举行全民公投。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>卡塔尔外交部发言人：</p>
<ul style="list-style-type:disc">
<li>我们的立场从一开始就很明确，拒绝俄罗斯在乌克兰的军事升级。</li>
<li>我们准备为促成俄罗斯和乌克兰之间的解决方案做出任何努力。</li>
</ul>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>英国外交大臣：在俄罗斯完全撤出乌克兰之前不会解除制裁。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3>美以会面</h3>
<p>美国国务卿安东尼·布林肯在与以色列外长亚尔·拉皮德在被占领的耶路撒冷举行的联合新闻发布会上说，华盛顿赞赏以色列总理纳夫塔利·贝内特为解决乌克兰危机而进行的调解。</p>
<p>拉皮德重申以色列对俄罗斯军事行动表示谴责，并表示特拉维夫寻求以一切可能的方式帮助难民。此外，布林肯还对以色列谴责“俄罗斯入侵”乌克兰的立场表示赞赏。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<h3><strong>俄罗斯的声明</strong></h3>
<p>俄新社报道称，俄罗斯国防部表示，其部队使用精准武器摧毁了一个油箱，以及位于乌克兰利沃夫的一个无线电维修厂。</p>
<p>该部表示，俄罗斯空军瞄准了乌克兰的67个军事设施，包括两个指挥所。</p>
<p>该部补充说：我们摧毁了3个武器库、11个乌克兰军队的目标和20个军事装备储存区。</p>

<p>英国国防部的声明称，乌克兰北部的战场基本上保持静止，乌克兰的反击正在阻碍俄罗斯重组部队的尝试。</p>
