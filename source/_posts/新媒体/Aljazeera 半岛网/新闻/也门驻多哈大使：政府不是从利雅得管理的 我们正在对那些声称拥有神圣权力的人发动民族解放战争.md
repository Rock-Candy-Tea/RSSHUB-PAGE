<div>   
<p>在担任也门政府官方发言人超过7年，熟悉该国经历的许多事件，尤其是政府在沙特-阿联酋联盟的支持下与2014年9月控制了首都萨那的胡塞武装作战多年，他开始了作为也门共和国特别大使在卡塔尔首都多哈的旅程。</p>
<p>在这次会面中，我们与海湾危机后第一任也门驻多哈大使拉杰·巴迪讨论了也门战争的前景及其持续7年多的原因，以及与胡塞武装谈判的命运，沙特首都利雅得管理国家政府的现实，还有观察家所说的“酒店政府”等。</p>
<ul>
<li>也许我们可以从实地发展开始。沙特-阿联酋联盟进入也门境内保卫合法政府，但似乎联盟现在的态度是保卫重要设施，特别是胡塞武装袭击了阿联酋和持续瞄准沙特之后，这是否表明各方力量的不平衡吗？</li>
</ul>
<p>一开始，我们知道这场战争是胡塞民兵发动的，它反对民选的合法权威，反对其参与的全国对话的结果，反对联合国主持的海湾倡议及其执行机制，他们是这场战争的主要原因。</p>
<p>现在，经过7年的战争，胡塞武装成为该地区一些国家的工具已经变得很明显。</p>
<p>7年前，胡塞出现在临时首都亚丁的总统府，而现在也门各省大片地区被解放，我们说，这场战争，也门人付出了沉重的代价；不幸的是，这其中的原因是胡塞人的统治。3位联合国特使的努力都失败了，每个人都在关注这些特使在任务结束时所说的话，谁是导致战争继续的一方。</p>
<ul>
<li>但是，根据解放萨那和塔伊兹等一些地区的红线，政府及其背后的联盟发动这场战争的战略是什么，是解放也门所有地区的战略，还是仅仅扩大对尽可能多的省份的控制，以便将力量筹码放在谈判桌上？</li>
</ul>
<p>我可以保证，合法政府没有红线。我们正在与一个带有宗教幌子的奴役项目进行民族解放斗争。也门人仍然对伊玛目政权记忆犹新，1962年9月，革命爆发反对该政权，革命得到了我们的阿拉伯兄弟，尤其是埃及共和国的支持。</p>
<p>“也门人反对的这个项目现在得到了地区支持，这使它更强、不断游走并延长战争。作为一个合法政府，我们不希望我们的国家成为冲突的舞台，这一点得到了胡塞武装袭击历史和政治首都萨那时在那里的总统阿卜杜拉布·曼苏尔·哈迪的证实，他说：“我将用和平之剑战斗直到最后一刻，”但胡塞武装并没有考虑国家利益和理性的声音，也没有考虑对话声音。</p>
<figure id="attachment_262102" aria-describedby="caption-attachment-262102" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-262102" src="https://chinese.aljazeera.net/wp-content/uploads/2022/03/image-41.jpg?w=770&quality=80&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-262102" class="wp-caption-text">(半岛电视台)</figcaption></figure>
<ul>
<li>但在许多观察家看来，这场战争已经变得荒谬。联盟既不能，也不愿意，扩大对所有省份的控制，而胡塞武装也不能统治所有省份？</li>
</ul>
<p>我不是说战局已经趋于正常，而是地区性的干预导致了战争的延续，一些大国和国家想在也门清算账目和利益。但我向您们保证，所有也门人都相信这是一场解放我们国家的战争，而且这样的战争不会很快结束。</p>
<ul>
<li>这意味着也门人将无限期地参与到战争之中？</li>
</ul>
<p>我们捍卫自己的民族原则，捍卫共和、自由和民主，同时在政治轨道上做出一个又一个让步，在科威特、日内瓦和皮尔进行了不止一轮谈判。我们做出了所有让步，但没有决定权的是对方，对方的支持者似乎还没有就结束战争并达成政治共识感到满意。</p>
<ul>
<li>提到这些谈判，您经历了很多，最近在阿曼首都马斯喀特有外交动作。合法政府取得了什么，为什么停滞不前？</li>
</ul>
<p>由于胡塞民兵的不妥协，有运动，但结果为零，代表团前往萨那，也与他们的不妥协发生冲突。</p>
<ul>
<li>但在政治上，胡塞反对什么？例如，他们是否就总统职位或他们在政府中的份额与合法政府发生争执？或者究竟是什么？</li>
</ul>
<p>我的意思是，例如，在科威特谈判中，我们达成了一项全面、完整和书面的协议，我们作为一个政府代表团，当时由前外交部长阿卜杜勒-马利克·米赫拉菲率领，我们签署了最终协议，这是马拉松式协商的结果，但胡塞人在最后一刻退缩了。尽管我们签署了我们并不相信的该协议的许多条款，但我们渴望停止战争，希望每个人都朝着共存、和平和建设国家的方向前进，胡塞方面在最后一刻撤回了签署决定，尽管他们自己为起草该协议的条款做出了贡献。</p>
<ul>
<li>为什么，在您看来，这是胡塞拒绝，还是您所说的“不妥协”？胡塞想要什么？</li>
</ul>
<p>胡塞自称是有权统治的政党，因为他们拥有权力和武器，不想屈服于自1990年也门统一以来开始哪怕相对持续下去的民主进程，应由也门人民选举他们的领导人，但胡塞希望用枪来选择。因此，除其他任何因素外，我再次重申，也门人正在与一个认为有一个拥有神圣统治权的势力进行民族解放斗争。</p>
<h3>也门和地区问题</h3>
<ul>
<li>似乎也门问题因与沙特-伊朗会谈等地区性问题或伊朗核文件等国际性问题的联系变得更加复杂。也门人能否在他们之间进行谈判并找到摆脱这些问题的和平方式？</li>
</ul>
<p>由于地区干涉，也门问题在伊朗核问题、叙利亚等多个问题中都被摆上了谈判桌。至于我们，政府和合法当局，我们确定战争会强加于我们，如果导致这场战争爆发的原因被消除，我们准备立即停止战争。</p>
<figure id="attachment_262096" aria-describedby="caption-attachment-262096" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-262096" src="https://chinese.aljazeera.net/wp-content/uploads/2022/03/image-1-27.jpg?w=770&quality=80&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-262096" class="wp-caption-text">(半岛电视台)</figcaption></figure>
<ul>
<li>如果地区和国际大国承认萨那胡塞武装的合法性怎么办？</li>
</ul>
<p>世界上没有一个国家可以承认武装民兵的合法性，现在以及在其控制历史和政治首都萨那7年后，一直没有人承认它，因为承认任何民兵的权威都将打开世界混乱的大门，我们必须如实看待事情的本质，远离任何政治考虑，这是一个武装民兵，控制了国家武器，推翻了权威，如果世界承认这个民兵的武装政变，明天世界其他任何地方的任何政变都将获得承认，这将打开世界混乱的大门。</p>
<ul>
<li>但是华盛顿从恐怖主义名单中删除了胡塞武装？</li>
</ul>
<p>华盛顿是合法政府的主要支持者。美国新政府的打算可能是希望向胡塞发出鼓励信息，以便他们转向对话，特别是因为乔·拜登总统是第一位任命也门问题特别代表的美国总统，但到目前为止，经过一年多的时间，胡塞人拒绝与这位美国特使会面。</p>
<p>胡塞武装误解了这一信息，并升级了他们在马里布、沙布瓦和荷台达的敌对行为，因此我们听说美国政府正寻求审查其决定，努力将胡塞武装重新归类为恐怖组织。</p>
<ul>
<li>与您所说的相反，在将胡塞从恐怖分子名单中删除后，美国的立场是否可能演变为给他们事实上的权威？</li>
</ul>
<p>不，这不会发生。世界有政治和外交规范。一个背叛合法当局的武装团体不可能得到承认。与此类团体打交道将在世界多个地区引发混乱。</p>
<ul>
<li>如果大国与伊朗就核问题达成协议，能否以某种方式与胡塞武装往来，并承认他们为萨那的执政当局？</li>
</ul>
<p>如果主要大国与伊朗就其核问题达成协议，可能会以一种或另一种方式处理，但不会承认武装组织的权威。</p>
<ul>
<li>您现在是也门驻卡塔尔大使，多哈能否发挥政治作用，拉近也门各方的观点？</li>
</ul>
<p>卡塔尔是本地区举足轻重的重要国家，拥有成功的外交政策，在多个问题上取得不止一次的成功。也门与卡塔尔的关系具有历史意义，自1990年也门统一以来，卡塔尔一直支持也门的领土统一和国家稳定，受到也门人民和也门领导人的广泛尊重。</p>
<p>以埃米尔谢赫塔米姆·本·哈马德·阿勒萨尼为首的卡塔尔领导层申明支持也门的合法政府。</p>
<p>就我们而言，我们欢迎任何国家，无论是卡塔尔还是其他国家，为制止流血和结束针对合法当局的政变并恢复我们在战前达成的目标所做的任何努力和任何作用。</p>
<figure id="attachment_262098" aria-describedby="caption-attachment-262098" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-262098" src="https://chinese.aljazeera.net/wp-content/uploads/2022/03/image-2-18.jpg?w=770&quality=80&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-262098" class="wp-caption-text">(半岛电视台)</figcaption></figure>
<ul>
<li>卡塔尔方面没有给您任何角色吗？</li>
</ul>
<p>到目前为止，没有，但我们在卡塔尔外交政策中遵循了我可以形容的专业精神，无论是在阿富汗文件还是在其他文件中，而且我们，正如我告诉您的那样，对寻求恢复正常、结束这场胡塞政变的各方力量和努力持开放态度。</p>
<h3>政府的存在</h3>
<ul>
<li>对于从外部管理国家的政府已经有很多讨论和批评，而且政府似乎并不打算改变其管理危机的方式，特别是考虑到它的决定取决于联盟之手，我们可以知道这背后的原因吗？</li>
</ul>
<p>我不知道为什么这种错误信息在许多媒体和社交媒体上重复出现；政府自《利雅得协定》成立以来，一直在亚丁，总理莫恩·阿卜杜勒-马利克先生和他的政府所有成员都在亚丁，尽管他们的飞机抵达时成为胡塞武装的目标。</p>
<ul>
<li>在亚丁存在但不稳定？</li>
</ul>
<p>不，那不是真的。当我担任政府发言人时，在我被任命为驻卡塔尔大使之前，政府一直在在亚丁，总理只用过几天的时间闪电访问了沙特首都利雅得，会见了哈迪总统和联盟领导人。</p>
<ul>
<li>但是当权者哈迪总统在利雅得？</li>
</ul>
<p>这是另一个问题，但政府及其所有成员、总理和办公室都在亚丁，尽管存在可能使政府无法有效和按要求开展工作的问题和障碍。但是，政府留在亚丁将克服这些困难和障碍。我们看到欧洲国家的大使和美国大使前往亚丁与政府会晤。</p>
<ul>
<li>您的意思是，您确认政府的所有成员都常驻亚丁，所谓的“酒店政府”不正确？</li>
</ul>
<p>是的，政府在亚丁，政府在亚丁别无选择，哈迪总统在对宪法宣誓后的第一次政府会议上向大家保证，任何政府官员都没有理由留在也门以外，指示很明确。</p>
<p>政府在亚丁，尽管这可能不会满足或对当地和非当地各方的利益构成威胁，而在亚丁的决定是关于合法性的不可逆转的决定。</p>
<figure id="attachment_262100" aria-describedby="caption-attachment-262100" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-262100" src="https://chinese.aljazeera.net/wp-content/uploads/2022/03/image-3-11.jpg?w=770&quality=80&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"><figcaption id="caption-attachment-262100" class="wp-caption-text">(半岛电视台)</figcaption></figure>
<ul>
<li>总统以及副总统回国怎么办，有安全隐患吗？</li>
</ul>
<p>可能有。</p>
<ul>
<li>如果部分地区掌握在胡塞武装手中，而其他地区掌握在非政府力量手中，政府如何重新获得在也门的影响力？就像在由巨人部队控制的萨赫勒地区以及由由南方过渡委员会管理的南部地区，那么政府管理什么？如何管理？</li>
</ul>
<p>所有这些矛盾都是政府管理的，政府留在亚丁就是为了解决这些矛盾，把这块马赛克统一起来。而《利雅得协定》就是为我提出的所有这些问题找到解决方案，我是说，《利雅得协定》必须得到充分执行，并作为一个综合的军事和政治体系。</p>
<ul>
<li>谁在阻挠《利雅得协定》的执行？</li>
</ul>
<p>也许有一些地方和地区力量的说法，当也门解放区有一个真正的国家并启动国家机构、监督机构、警察和法院时，所有这一切都不符合一些势力的利益，他们认为这威胁到他们的小型地方项目或区域层面的大型项目。</p>
<ul>
<li>您说的是南方过渡委员会吗？</li>
</ul>
<p>我不是在谈论南方过渡委员会或其他任何一方。我说，“有一些力量”，而不是一个力量，不寻求执行《利雅得协定》，我无法具体说明，但也门人有敏锐和智慧，他们知道那些当事人是谁。</p>
<h3>目前解决也门危机的前景如何？</h3>
<p>不幸的是，我对也门的局势感到痛苦和悲伤，我想说，直到这一刻，我们在隧道尽头找不到地平线或光明，因对这种情况和这种挫败负责的一方，即使联合国特使以及国际和地区大国也有这种挫败感，是想要表现得好像这场战争中唯一的一方或胜利者的胡塞武装。</p>
<p>尽管如此，我们希望也门人能够在他们的阿拉伯兄弟和国际社会的合作下克服这一巨大的考验。</p>
