<p>俄罗斯对乌克兰的战争已经进入第九天，乌克兰当局周五黎明时分证实，乌克兰东南部扎波罗热州的核电站因俄罗斯轰炸而着火，与此同时，巨大的爆炸声响彻首都基辅。</p>
<p>乌克兰外交部长德米特罗·库莱巴警告说，扎波罗热核电站的爆炸规模将是 1986 年切尔诺贝利反应堆爆炸的 10 倍。</p>
<p>以下是俄罗斯对乌克兰开战第九天的最新进展：</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>国际原子能机构：乌克兰未发现扎波罗热站辐射水平发生变化。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>法新社援引乌克兰当局说法报道称：扎波罗热核电站的核安全得到了保障。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>路透社援引乌克兰 24 频道报道称：扎波罗热核电站负责人表示，该核电站的辐射安全已得到保障。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>乌克兰紧急服务部门：扎波罗热核电站的一座培训大楼发生火灾。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>法新社援引美国官员的话报道称：拜登刚刚与乌克兰总统就俄罗斯轰炸扎波罗热核电站一事进行了交谈。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>扎波罗热军管局局长：允许消防员进入核电站。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>CNN援引美国国防部（五角大楼）发言人的话报道称：据我们估计，俄罗斯军队距离基辅市中心仍有 25 公里。</p>
<ul>
<li>CNN援引五角大楼发言人的话报道称：我们仍然认为，俄罗斯军队的意图是包围并最终占领基辅。</li>
<li>CNN援引五角大楼发言人的话报道称：由于乌克兰的强烈抵抗，俄罗斯军队一直处于混乱和缓慢的状态。</li>
<li>CNN援引五角大楼发言人的话报道称：没有我们的空军对抗俄罗斯，乌克兰就没有禁飞区。</li>
<li>CNN援引五角大楼发言人的话报道称：有迹象表明，俄罗斯正在控制乌克兰的赫尔松市。</li>
</ul>

<li><strong>更新</strong></li>
</ul>
