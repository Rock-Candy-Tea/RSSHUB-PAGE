<div>   
<p>美国加入欧洲国家共同对俄罗斯总统弗拉基米尔·普京实施制裁，并对周四开始攻击乌克兰的普京实施了其他制裁，与此同时，乌克兰多地战争愈演愈烈，逼近首都基辅，而后者宣布同意与俄罗斯进行谈判。</p>
<p>与此同时，美国强调，将保卫北大西洋公约组织（北约）的每一寸领土。</p>
<p>至于安理会，在俄罗斯使用否决权、3个国家投弃权票后，安理会未能通过谴责俄罗斯袭击乌克兰的决议草案。</p>
<p>要了解俄罗斯入侵乌克兰第二天（周五）的事态发展，请<a href="https://www.aljazeera.net/news/politics/2022/2/25/%D8%AA%D8%B7%D9%88%D8%B1%D8%A7%D8%AA-%D8%AD%D8%B1%D8%A8-%D8%B1%D9%88%D8%B3%D9%8A%D8%A7-%D8%B9%D9%84%D9%89-%D8%A3%D9%88%D9%83%D8%B1%D8%A7%D9%86%D9%8A%D8%A7">单击此处</a>。</p>
<p>以下是本周六（俄罗斯入侵第三天）的最新进展：</p>
<h3><strong>28 分钟前</strong></h3>
<p>半岛电视台记者：基辅发生两起爆炸，首都北部阿波罗地区发生战斗的报道。</p>
<h3><strong>38 分钟前</strong></h3>
<p>CNN援引美国官员说法称：</p>
<ul>
<li>美国军方寻求与俄罗斯建立沟通渠道，以避免在乌克兰及其周边地区出现任何误判。</li>
<li>与俄罗斯的沟通旨在提醒该地区可能的空中行动。</li>
<li>华盛顿仍在制定与俄罗斯军方建立联系的建议。</li>
</ul>
<h3><strong>一小时前</strong></h3>
<p>半岛电视台记者：乌克兰首都基辅不同地区都听到了爆炸声。</p>
<p>《华盛顿邮报》援引美国官员：</p>
<ul>
<li>我们准备帮助（乌克兰总统）泽连斯基离开基辅，避免他被俘或被杀。</li>
<li>我们与泽连斯基讨论了对他来说最安全的地方，以确保乌克兰政府的连续性。</li>
</ul>
<p>路透社：白宫要求国会拨款 64 亿美元，以解决乌克兰的人道主义危机和安全危机。</p>
<h3>两小时前</h3>
<figure id="attachment_259347" aria-describedby="caption-attachment-259347" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-259347" src="https://chinese.aljazeera.net/wp-content/uploads/2022/02/27059199-1.jpg?w=770&resize=770%2C481" alt data-recalc-dims="1" referrerpolicy="no-referrer">北约秘书长延斯·斯托尔滕贝格：对俄罗斯的制裁将产生长期后果（阿纳多卢通讯社）</figure>
<p>北约秘书长延斯·斯托尔滕贝格告诉 PBS说：</p>
<ul>
<li>我们不会派遣盟军在乌克兰进行实地作战。</li>
<li>如果北约前往乌克兰，情况会比我们今天看到的更糟。</li>
<li>我们正在通过陆、空和海加强我们在北约东部地区的力量。</li>
<li>如果诸如波兰或波罗的海任何成员国受到攻击，北约将做出回应。</li>
<li>北约是历史上最强大的军事联盟，我们将保卫它的每一寸土地的安全，但我们不会在乌克兰部署军队。</li>
<li>我理解乌克兰的沮丧和痛苦，但北约拥有应该履行的责任。</li>
<li>对俄罗斯的制裁将产生长期后果。</li>
</ul>
<h3>两小时前</h3>
<p>欧盟驻联合国代表：我们现在就采取行动，要求联合国呼吁俄罗斯立即撤军。</p>
<p>美国驻联合国大使：</p>
<ul>
<li>俄罗斯不能对乌克兰人民施加否决权。</li>
<li>俄罗斯不能继续不被问责，也不能阻止《联合国宪章》。</li>
<li>俄罗斯今天滥用权力否决我们的强硬决定。</li>
</ul>
<h3>两小时前</h3>
<p>乌克兰军队：我们在瓦西尔科夫地区击落了一架俄罗斯飞机。</p>
<h3>两小时前</h3>
<p>乌克兰驻联合国代表：</p>
<ul>
<li>现在决定了乌克兰的命运。</li>
<li>在俄罗斯入侵的第一天，乌克兰约有 140 人死亡，316 人受伤。</li>
<li>即使俄罗斯成功地暂时占领了我们领土的更多部分，我们也不会投降。</li>
<li>我们呼吁我们的伙伴断绝与俄罗斯的外交关系。</li>
<li>我们呼吁国际组织禁止或暂停俄罗斯的会员资格。</li>
</ul>
<h3>3小时前</h3>
<p>美国总统欧洲事务助理：弹药继续流入乌克兰及其军队。</p>
<p>推特：我们暂时停止在俄罗斯和乌克兰投放广告。</p>
<h3>3小时前</h3>
<p>乌克兰媒体：首都基辅发生暴力冲突。</p>
<h3>3小时前</h3>
<p>美国财政部：华盛顿对俄罗斯总统、外交部长和国防部长以及总参谋长实施制裁。</p>
<h3>3小时前</h3>
<p>在俄罗斯代表否决后，联合国安理会未能通过谴责俄罗斯入侵乌克兰的决议草案。</p>
<p>对俄罗斯入侵乌克兰国际决议草案投弃权票的国家是：中国、印度和阿联酋。</p>
<p>白宫：财政部将对俄罗斯直接投资基金实施全面禁令制裁。</p>
<h3>4小时前</h3>
<p>美国国务院：国务卿布林肯与英国外交大臣讨论持续协调，敦促俄罗斯停止军事行动。</p>
<p>乌克兰文尼察市长：一架俄罗斯飞机在卡利诺夫卡附近被击落。</p>
<p>法新社：惠誉将乌克兰的主权评级从“B”下调三档至“CCC”。</p>
<h3>4小时前</h3>
<p>英国驻联合国代表：</p>
<ul>
<li>我们今天投票表决的决议是向乌克兰人民发出的信息，即我们与他们站在一起。</li>
<li>我们今天投票的决议要求结束俄罗斯对乌克兰的战争。</li>
<li>普京的战争目标是推翻乌克兰政府，这个事实无法掩盖。</li>
</ul>
<h3>4小时前</h3>
<p>乌克兰总统：敌人正在竭尽全力破坏我们，今晚它将以武力进攻，今晚将决定乌克兰的命运。</p>
<p>英国国防部：俄军正从多条轴线向基辅推进，企图包围该市。</p>
<h3>4小时前</h3>
<figure id="attachment_259350" aria-describedby="caption-attachment-259350" style="width:770px" class="wp-caption aligncenter"><img loading="lazy" class="size-arc-image-770 wp-image-259350" src="https://chinese.aljazeera.net/wp-content/uploads/2022/02/RTS5T5T5-1.jpg?w=770&resize=770%2C513" alt data-recalc-dims="1" referrerpolicy="no-referrer">美国代表：俄罗斯袭击乌克兰威胁到我们的国际秩序（路透社）</figure>
<p>美国驻联合国大使：</p>
<ul>
<li>俄罗斯选择给自己的公民和乌克兰人民带来巨大的痛苦。</li>
<li>让我们永远不要忘记，俄罗斯选择发动这场战争。</li>
<li>俄罗斯对我们原则发起公然攻击，我们绝不能对此保持沉默。</li>
<li>我们面前的决议草案谴责俄罗斯的侵略并申明乌克兰的领土完整。</li>
<li>俄罗斯对乌克兰的袭击威胁到我们的国际秩序。</li>
<li>我们必须捍卫《联合国宪章》，而俄罗斯是侵略者。</li>
<li>俄罗斯威胁芬兰和瑞典，负责任的国家不会侵略邻国。</li>
<li>我请安理会成员对该决议草案投赞成票。</li>
</ul>
<h3>4小时前</h3>
<p>拜登：我们继续向乌克兰提供经济、人道主义和安全支持，我们继续动员其他国家提供类似援助。</p>
<h3>4小时前</h3>
<p>半岛电视台记者：基辅警笛响起。</p>
<h3>4小时前</h3>
<p>俄罗斯中央银行：</p>
<ul>
<li>如果客户要求，受制裁的银行愿意将存款退还给客户。</li>
<li>我们准备使用其他工具来维持银行的稳定性并保护客户。</li>
</ul>
<p>路透社记者：加拿大将对俄罗斯总统普京及其外长拉夫罗夫以及白俄罗斯及其领导人实施制裁。</p>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-259353" src="https://chinese.aljazeera.net/wp-content/uploads/2022/02/image-110.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<h3>5小时前</h3>
<p>五角大楼发言人：我们将向乌克兰提供额外的国防援助，我们正在研究如何做到这一点。</p>
<p>NBC：美国国家安全委员会正在考虑，如果华盛顿向乌克兰提供致命援助，是否会成为冲突的一方。</p>
<p>半岛电视台记者：</p>
<ul>
<li>美国向安理会成员分发了一份关于俄罗斯袭击乌克兰的修正决议草案。</li>
<li>美国修订后的决议草案对俄罗斯袭击乌克兰表示遗憾而不是谴责。</li>
<li>修正后的决议草案从序言段落中删除了对普京总统的谴责，取而代之的是，对俄罗斯的谴责。</li>
<li>美国方面修改后的决议草案不再适用于《联合国宪章》第七章。</li>
</ul>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-259355" src="https://chinese.aljazeera.net/wp-content/uploads/2022/02/image-1-76.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<h3>5 小时前</h3>
<p>NBC 原因国会消息人士称：拜登政府正在研究向乌克兰提供武器和情报的法律问题。</p>
<p>泽连斯基总统新闻秘书：</p>
<ul>
<li>我们驳斥关于我们拒绝谈判的说法，并准备讨论停火与和平。</li>
<li>我们同意俄罗斯的谈判建议。</li>
<li>越早开始谈判，恢复正常生活的机会就越大。</li>
</ul>
<p>白宫：</p>
<ul>
<li>制裁对俄罗斯金融体系和经济产生巨大影响。</li>
<li>尽管乌克兰人进行了抵抗，但基辅很有可能落入俄罗斯军队手中。</li>
<li>拜登总统仍然愿意与普京总统接触，但现在不是时候。</li>
<li>当另一个国家的领导人入侵时，外交似乎不合适，但这并不意味着可以永远排除外交。</li>
<li>经济合作与国际发展组织：我们决定取消俄罗斯的入会程序，并关闭我们在莫斯科的办事处。</li>
</ul>
<h3>5小时前</h3>
<p>路透社援引乌克兰总统发言人：基辅和莫斯科正在讨论确定双方会谈的地点和时间问题。</p>
<p>俄罗斯国防部：我们的部队在两栖登陆后毫无抵抗地进入了乌克兰城市梅利托波尔。</p>
<h3>5小时前</h3>
<p>加拿大总理：世界将追究俄罗斯和白俄罗斯的责任，我与北约领导人确认我们对乌克兰主权的坚定支持。</p>
<p>乌克兰驻美大使：俄罗斯军队扣押切尔诺贝利核电站92名员工为人质。</p>
<h3>5小时前</h3>
<p>白宫：</p>
<ul>
<li>美国加入欧洲盟友共同对普京总统和外交部长拉夫罗夫实施制裁。</li>
<li>我们正在与泽连斯基总统和他的国家安全团队联系，讨论额外援助的选择。</li>
<li>几周来，我们一直在警告普京总统正致力于改变乌克兰领导层。</li>
<li>美国将保卫北约的每一寸领土，我们将严格遵守北约第5条承诺。</li>
<li>我已下令部署更多部队，以增强我们在欧洲支持北约盟国的能力。</li>
<li>普京总统未能分裂西方，北约比以往任何时候都更加团结和坚决。</li>
</ul>
<p><img loading="lazy" class="aligncenter size-arc-image-770 wp-image-259357" src="https://chinese.aljazeera.net/wp-content/uploads/2022/02/image-2-55.jpg?w=770&resize=770%2C433" alt data-recalc-dims="1" referrerpolicy="no-referrer"></p>
<p>美国国务院：</p>
<ul>
<li>华盛顿目前没有与俄罗斯进行战略稳定对话的计划。</li>
<li>我们短期内的首要任务是结束俄罗斯对乌克兰国家和人民的暴力和暴行。</li>
<li>我们不想惩罚俄罗斯人民，而且，我们声援他们。</li>
</ul>
