<p>俄罗斯对乌克兰发动的战争进入了第16天，欧盟领导人峰会继续在法国首都巴黎的凡尔赛宫举行，以讨论这场战争的后果。</p>
<p>欧洲领导人正在讨论全球市场能源价格持续上涨的可能解决方案，并且讨论了在能源领域内独立的可能性，特别是在严重依赖俄罗斯能源进口的情况下。</p>
<p>在政治方面，乌克兰与俄罗斯的外交部长在土耳其南部城市安塔利亚举行的会谈，并未朝着可能实现全面停火的方向取得任何进展，与此同时，从一些冲突地区撤离平民的进程仍在继续。</p>
<p>俄罗斯已开始采取措施以应对西方制裁的影响，包括暂停粮食和其他材料的出口，与此同时，俄罗斯总统普京承诺将对制裁作出回应，并称俄罗斯将从当前的危机中变得更加强大。</p>
<p>以下是这场战争的最新进展：</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>乌克兰总统经济顾问：我们因俄罗斯的袭击而遭受的损失估计为1000亿美元。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>白宫：拜登将在11日宣布采取措施，以继续追究俄罗斯对乌克兰发动无理战争的责任。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>美国参议院批准向乌克兰提供价值136亿美元的援助。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>彭博社：拜登今天将呼吁停止与俄罗斯的正常贸易关系，而这将与七国集团以及欧盟共同完成。</p>
<p>美国银行摩根大通宣布暂停其在俄罗斯市场内的业务。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>半岛电视台记者：乌克兰城市利沃夫拉响警报，警告居民可能发生空袭。</p>
<ul style="list-style-type:square">
<li><strong>更新：</strong></li>
</ul>
<p>中国国务院总理：</p>
<ul>
<li>支持俄罗斯和乌克兰的停火谈判非常重要。</li>
<li>各国的主权必须得到尊重，正当的安全关切必须得到重视。</li>
<li>中方有自身的观点，并愿与国际社会一道，为恢复和平发挥积极作用。</li>
<li>中方已提出应对乌克兰局势的倡议，并将继续向该国提供人道主义援助。</li>
<li>中方呼吁保持最大限度的克制，以防止出现大规模的人道主义危机。</li>
<li>我们支持一切旨在和平解决危机的努力。</li>
<li>当务之急是防止局势升级，以免局势失控。</li>
</ul>

</ul>
<p>路透社：世界卫生组织表示已建议乌克兰消灭其实验室内存在的高危细菌。</p>

<li><strong>更新：</strong></li>
</ul>
<p>白宫：我们支持美国公司，它们正就其在俄罗斯的业务未来采取艰难的决定，而俄罗斯没收这些公司资产的任何决定，都意味着俄罗斯并非一个安全的投资场所。</p>

</ul>

<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>拜登：普京曾下注于分裂北约与西方，但是他的赌注并不正确，我们目前看到的是西方出现了前所未有的团结，而美国人也意识到，如果不采取行动以反抗这位专制者，那么就将付出更大的代价。</p>
<ul style="list-style-type:square">
<li><strong>更新</strong></li>
</ul>
<p>泽连斯基：俄罗斯正试图夺下马里乌波尔市并摧毁当地人民的决心</p>
<p>乌克兰总统泽连斯基表示，俄罗斯对马里乌波尔儿童医院的轰炸造成3人死亡，其中包括一名儿童，他还指责莫斯科所谓的“医院里没有病人”是在撒谎。</p>
<p>这位乌克兰总统还表示，莫斯科声称他的国家想要恢复其核能力，这可能是它将会使用核弹的前奏。</p>
<p>泽连斯基补充称，乌克兰拥有用于科学目的的和平性质的核反应堆，他还强调，俄罗斯正试图夺取马里乌波尔市并摧毁当地人民的决心。</p>
<p>泽连斯基表示，基辅已向马里乌波尔派出了几辆人道主义援助巴士，他还对俄罗斯“想要误导我们公民的虚假宣传”发出了警告。</p>

<h3>武装乌克兰人</h3>
<p>美国常驻安理会代表琳达·托马斯·格林菲尔德在致半岛电视台的一份声明中指出，美国将继续通过北约为乌克兰人民提供武装。</p>
<p>这位美国代表补充称，美国将继续尽力孤立俄罗斯政权，并迫使其立即停止战斗。</p>
<p>关于制裁方面，格林菲尔德表示，我们的欧洲盟友应努力减少对俄罗斯石油的依赖，以使西方对俄罗斯实施的制裁更加有效。</p>
