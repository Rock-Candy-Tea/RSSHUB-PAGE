【BTFS已上线NFT分布式存储解决方案NFT Hub最新版本V1.1】据官方消息，BitTorrent File System（BTFS）已上线NFT分布式存储解决方案NFT Hub最新版本V1.1。该版本新增Justin@BTFS页面，用户通过NFT hub平台可一键查看波场TRON创始人兼BitTorrent CEO 孙宇晨在BTFS上分享的收藏、作品、报道等精彩内容。同时，该版本可支持NFT铸造，进一步为全网用户提供铸造NFT的平台级服务。 
NFT Hub为BitTorrent File System（BTFS）近期上线的NFT分布式存储解决方案，旨在进一步打通BTFS去中心化网络与波场TRON生态, 为NFT免费提供分布式存储支持。
