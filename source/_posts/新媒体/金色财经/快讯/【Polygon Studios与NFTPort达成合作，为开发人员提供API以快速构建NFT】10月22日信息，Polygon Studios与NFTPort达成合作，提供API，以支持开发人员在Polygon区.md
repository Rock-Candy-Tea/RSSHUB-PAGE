【Polygon Studios与NFTPort达成合作，为开发人员提供API以快速构建NFT】10月22日信息，Polygon Studios与NFTPort达成合作，提供API，以支持开发人员在Polygon区块链上构建NFT应用程序。开发人员可以通过API轻松找到Polygon和Ethereum NFT数据，包括合约、代币、资产、元数据和所有权信息。借助NFTPort的铸造端点，开发人员可以在不到5分钟的时间内通过单个API调用将任何东西变成NFT。NFTPort 还提供增强的API来构建高级NFT搜索和基于AI的推荐引擎。
