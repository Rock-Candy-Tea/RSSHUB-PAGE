【Polygon Studios 计划筹集 1 亿美元以支持区块链游戏开发商和 NFT 创建者】以太坊扩容方案 Polygon 用于发展区块链游戏和 NFT 的新部门 Polygon Studios ，正在计划筹集 1 亿美元资金以支持区块链游戏开发商和 NFT 创建者，同时与游戏公司雅达利（Atari）合作，旨在成为 Web2.0 和 Web3.0 之间的桥梁。为了实现此愿景，Polygon Studios 将为区块链游戏和 NFT 团队提供360° 构建和发布支持，促进成熟品牌的发布并提供play-to-earn机会。此前报道，2 月份，游戏公司雅达利接入 Polygon 网络，将其 NFT 和 Atari Token 带到了 Layer 2。 
7 月份，Polygon 宣布推出新部门 Polygon Studios，用于发展区块链游戏和 NFT，帮助开发人员创建支持区块链的游戏及自定义 NFT 模型和市场，促使 Polygon 网络吸引更多游戏和 NFT 的优质品牌和特许。
