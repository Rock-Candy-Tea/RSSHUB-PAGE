【Saito Network已成功完成Web 3 Fundation Grant的第一个里程碑交付】近日，Saito Network宣布已成功完成Web 3 Fundation Grant的第一个里程碑交付。这一里程碑将Saito的去中心化游戏引擎更好地带到了Web 3社区。该引擎能让开发人员以真正去中心化的方式构建回合制游戏。它消除了对复杂游戏流程（如洗牌、掷骰子和回合管理）的中心化、可信任的第三方需求，这是通过实施标准化的、基于区块链的堆栈和复杂的加密技术（如心理扑克）来实现的。 
Saito联合创始人指出：“我们感谢web3基金会在这方面的支持，并很高兴向社区展示Saito如何为web3项目提供开放基础设施。第二个也是最后一个里程碑的工作正在顺利进行。这增加了代币转移功能、开发者文档和其他游戏。”
