【Security Token Group完成300万美元的A轮融资】5月19日消息，专注于构建金融基础设施以支持安全令牌行业增长和发展的风险投资工作室Security Token Group确认完成300万美元的A轮融资，Blue Bay Ventures 牵头，Blizzard Fund（Ava Labs / Avalanche 的风险投资基金）、Exodus和miamitech领投。凭借其 A 轮融资的承诺，STG 还将寻求扩展其其他业务垂直领域，即 Security Token Advisors（STA）和 Security Token Capital（STC）。
