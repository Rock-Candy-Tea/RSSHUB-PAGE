【Github调查滥用其基础架构进行加密挖矿的攻击活动】4月3日消息，代码托管平台Github对一系列攻击进行了调查，这些攻击旨在滥用其基础设施来非法挖掘加密货币。攻击者执行自己的恶意代码，以在Github的基础结构上挖掘加密货币，在某些情况下，攻击者可能在一次攻击中部署数百名矿机。这种攻击对被滥用的基础架构的计算能力具有重大影响。（SecurityAffairs）
