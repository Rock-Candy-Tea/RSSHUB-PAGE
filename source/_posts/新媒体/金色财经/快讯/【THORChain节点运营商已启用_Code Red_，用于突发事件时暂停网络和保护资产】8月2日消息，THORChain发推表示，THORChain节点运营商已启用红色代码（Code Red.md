【THORChain节点运营商已启用“Code Red”，用于突发事件时暂停网络和保护资产】8月2日消息，THORChain发推表示，THORChain节点运营商已启用红色代码（Code Red）。如果发现任何可疑情况，每个节点都可以暂停网络1小时。任何其他节点都可以取消暂停。该功能旨在24/7全天候监控网络，保护用户资金。
