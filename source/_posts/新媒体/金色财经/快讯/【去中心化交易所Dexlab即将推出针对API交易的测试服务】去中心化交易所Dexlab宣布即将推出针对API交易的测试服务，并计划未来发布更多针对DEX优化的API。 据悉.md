【去中心化交易所Dexlab即将推出针对API交易的测试服务】去中心化交易所Dexlab宣布即将推出针对API交易的测试服务，并计划未来发布更多针对DEX优化的API。 
据悉，Dexlab是基于Solana的去中心化跨链交易平台，提供交易、兑换、流动性提供等服务，此前获得Solana生态基金战略投资。
