【固定利率借贷协议Notional发布NOTE质押模块】3月15日消息，固定利率借贷协议 Notional 发布其协议 Token NOTE 的质押模块。用户可将 NOTE 质押进 Balancer 的 NOTE/WETH 流动性池，并获得 sNOTE。sNOTE 作为 Balancer LP Token 获得收益，同时为 Notional 用户提供保障，即在抵押品短缺的情况下，通过治理可使用 50% 的 sNOTE 资产来为协议重新注资。
