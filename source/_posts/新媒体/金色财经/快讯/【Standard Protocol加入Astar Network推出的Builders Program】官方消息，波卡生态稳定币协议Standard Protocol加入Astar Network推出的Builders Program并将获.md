【Standard Protocol加入Astar Network推出的Builders Program】官方消息，波卡生态稳定币协议Standard Protocol加入Astar Network推出的Builders Program并将获得一笔grant，现已经准备好在Shiden Network主网上部署智能合约。Standard Protocol将推出他们的DApp，并尝试将Polkadot中的wrapped原生代币带到Shiden和Astar Network。此外，Astar希望提供合成加密资产，支持用户对某些资产开启看涨期权。
