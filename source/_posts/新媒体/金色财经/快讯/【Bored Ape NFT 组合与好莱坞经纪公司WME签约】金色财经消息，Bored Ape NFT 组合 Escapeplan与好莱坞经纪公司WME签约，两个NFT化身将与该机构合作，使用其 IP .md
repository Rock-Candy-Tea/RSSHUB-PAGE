【Bored Ape NFT 组合与好莱坞经纪公司WME签约】金色财经消息，Bored Ape NFT 组合 Escapeplan与好莱坞经纪公司WME签约，两个NFT化身将与该机构合作，使用其 IP 进行音乐创作、MV制作等。据悉，Escapeplan 计划与 Dillon Francis 合作多首歌曲，并与说唱歌手 Rich the Kid 合作制作音乐视频。 
ESCAPPLAN 由 Bored Apes #309 和 #4321 组成，分别被称为 zETH 和 ETHan。Yuga Labs 是 Bored Apes 的创建者，以拥有非限制性 IP 政策而闻名，允许 Bored Ape 持有者将其 NFT 用于商业目的。此前，环球音乐集团也曾购买无聊猿NFT，用于元宇宙组合中的音乐创作等。（decrypt）
