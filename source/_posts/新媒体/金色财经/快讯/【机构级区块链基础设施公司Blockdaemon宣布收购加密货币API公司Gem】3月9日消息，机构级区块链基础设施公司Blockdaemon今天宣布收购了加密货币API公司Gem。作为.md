【机构级区块链基础设施公司Blockdaemon宣布收购加密货币API公司Gem】3月9日消息，机构级区块链基础设施公司Blockdaemon今天宣布收购了加密货币API公司Gem。作为协议的一部分，Blockdaemon将利用Gem的所有旗舰产品，包括Onramp解决方案、Connect API、KYC Passport等。据介绍，Gem拥有一个法币-加密货币入口和全面的交易数据聚合服务，用于从单个位置跟踪加密货币投资。 
此前消息，Blockdaemon宣布以32.5亿美元估值完成2.07亿美元C轮融资，Sapphire和Tiger Global领投。（BusinessWire）
