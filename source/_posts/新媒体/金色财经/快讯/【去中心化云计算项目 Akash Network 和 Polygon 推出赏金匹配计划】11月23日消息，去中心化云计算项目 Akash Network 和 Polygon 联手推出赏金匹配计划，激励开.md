【去中心化云计算项目 Akash Network 和 Polygon 推出赏金匹配计划】11月23日消息，去中心化云计算项目 Akash Network 和 Polygon 联手推出赏金匹配计划，激励开发人员从传统的云提供商迁移到 Akash 网络。Akash 的点对点市场为 Polygon 开发人员提供了一个选项来部署和扩展应用程序和服务，而 Polygon 为开发人员提供一个入门套件，包括在 Akash Network 上托管的说明。Akash 向 Polygon 生态项目提供 20 美元的 AKT，以支付在 Akash 上的托管费用。Polygon 应用程序可在 Akash 赏金委员会上发布赏金，以聘请工程师帮助其在 Akash 上进行迁移和托管。
