【API3联盟联合逾125家API提供商直接向Web3提供预言机服务】据官方消息，Web 3去中心化API服务API3宣布推出API3联盟，联合逾125家API提供商直接向Web3提供预言机服务。API3联盟成立之初，参与者包括FTX、Benzinga Kaiko、Finage、World Weather Online、Zabo、Deribit、Data Sports Group、1forge、Alameda Research、Ex Machina、Coinranking、Algoseek、Wikiroutes、dxFeed、Finnhub等。 
API3开发的Airnode架构是API3联盟中各API提供商的交互中心，允许API提供商快捷地创建第一方预言机，使他们能够将其提供的所有形式的API数据和服务带到Web3应用程序中，并在区块链和其他数字世界之间形成重要桥梁。API3表示，该生态系统将由新成立的API3去中心化自治组织（DAO）进行规模化管理。
