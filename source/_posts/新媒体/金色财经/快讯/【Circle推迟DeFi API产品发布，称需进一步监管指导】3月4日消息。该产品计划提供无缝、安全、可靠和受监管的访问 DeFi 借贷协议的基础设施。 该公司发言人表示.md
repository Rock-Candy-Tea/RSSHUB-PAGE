【Circle推迟DeFi API产品发布，称需进一步监管指导】3月4日消息。该产品计划提供无缝、安全、可靠和受监管的访问 DeFi 借贷协议的基础设施。 
该公司发言人表示，DeFi API 产品推出的时间将取决于进一步监管指导、合规工具增强和区块链身份协议的可用性发展程度，并且暂时无法提供确切的推出时间表。（THE BLOCK）
