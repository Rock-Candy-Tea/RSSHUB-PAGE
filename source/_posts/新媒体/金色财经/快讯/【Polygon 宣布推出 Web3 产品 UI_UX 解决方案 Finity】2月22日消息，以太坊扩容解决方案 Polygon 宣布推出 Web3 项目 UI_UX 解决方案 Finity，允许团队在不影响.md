【Polygon 宣布推出 Web3 产品 UI/UX 解决方案 Finity】2月22日消息，以太坊扩容解决方案 Polygon 宣布推出 Web3 项目 UI/UX 解决方案 Finity，允许团队在不影响美观和可用性的情况下快速原型化、发布和扩展他们的产品。Finity 提供久经考验的资产、元素和模板，专注于 3D 设计。使用 Finity，开发人员可以轻松自定义样式和符号，添加自己的元素并缩放设计，组合符号以创建多个 UI 元素。
