【BTFS数据浏览器BTFS Scan已正式升级为BTSCAN，并启用新域名】据官方消息，BitTorrent File System(BTFS) 数据浏览器BTFS Scan已升级为BTSCAN，并启用全新域名scan.bt。BTSCAN在原有数据基础上，新增了BT Speed产品关键数据展示，包括BT Speed钱包地址数、BT Speed链上被激活地址数、BT Speed空投总数等。 
据BTSCAN浏览器2021年7月28日数据显示，BT Speed产品钱包账户数为266,178,530，被激活的波场链上地址数为2,715,415，累计空投达5,535,028 BTT。
