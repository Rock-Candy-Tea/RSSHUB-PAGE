【微软宣布支持 Astar 孵化计划】3月10日消息，据波卡平行链 Astar Network 发布的文章，微软将通过 Astar 孵化计划支持 Astar Network 构建和实施 Web3 未来。作为此次合作的一部分，微软和 Astar Network 将在 Astar 孵化计划上展开合作，具体来说，微软将为 Astar 孵化计划中选定的初创公司提供资源，包括营销支持和指导网络。
