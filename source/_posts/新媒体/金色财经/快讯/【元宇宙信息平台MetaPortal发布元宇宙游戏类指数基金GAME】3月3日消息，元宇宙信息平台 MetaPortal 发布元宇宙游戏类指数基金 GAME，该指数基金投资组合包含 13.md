【元宇宙信息平台MetaPortal发布元宇宙游戏类指数基金GAME】3月3日消息，元宇宙信息平台 MetaPortal 发布元宇宙游戏类指数基金 GAME，该指数基金投资组合包含 13 个元宇宙游戏类项目 Token，主要包括：AXS（占比 22.6%）、SAND（占比 20.18%）和 GALA（占比 14.7%）。该指数基金是 MetaPortal 继元宇宙指数基金 MVI 后发布的第二个指数基金，均发行在去中心化金融协议 Set Protocol 上。
