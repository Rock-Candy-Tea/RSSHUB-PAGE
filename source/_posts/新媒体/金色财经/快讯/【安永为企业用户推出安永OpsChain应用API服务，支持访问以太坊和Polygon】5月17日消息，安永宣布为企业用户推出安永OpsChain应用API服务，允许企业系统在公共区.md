【安永为企业用户推出安永OpsChain应用API服务，支持访问以太坊和Polygon】5月17日消息，安永宣布为企业用户推出安永OpsChain应用API服务，允许企业系统在公共区块链上进行交易，目前允许企业访问以太坊主网和Polygon网络。EY OpsChain API服务包括部署代币合约、铸造、转移或销毁代币的能力。据The Block称，OpsChain托管在Nightfall上，这是EY与Polygon共同开发的Layer2 Rollup区块链。（prnewswire）
