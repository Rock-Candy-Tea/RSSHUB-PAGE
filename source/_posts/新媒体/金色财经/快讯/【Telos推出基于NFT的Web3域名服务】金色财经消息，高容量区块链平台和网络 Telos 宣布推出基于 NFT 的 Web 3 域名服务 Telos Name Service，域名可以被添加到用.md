【Telos推出基于NFT的Web3域名服务】金色财经消息，高容量区块链平台和网络 Telos 宣布推出基于 NFT 的 Web 3 域名服务 Telos Name Service，域名可以被添加到用户配置文件中，域名持有者可以发送和接收具有唯一性的“.tlos”NFT，其中可以包含个人资料、头像和社交信息等。Web 3 域名本质上可以看做是一种数字真实性证书 NFT，用户还可以在 NFT 市场上进行交易，Telos 表示，用户可以将 Telos Domains 集成到 DApp、钱包或 Web 3 项目中为用户提供更好的整体体验。<br><a href="https://twitter.com/TelosDomains/status/1508349495599194113" target="_blank">原文链接</a>
