【支持Vasil协议升级Cardano Rosetta 1.8.0版本已经发布】金色财经报道，根据Cardano 更新分享的 Github 数据，支持即将到来的 Vasil 协议升级的全新 Cardano Rosetta 1.8.0 已经发布。正如开发人员说明中所述，Dockerfile 捆绑了最新的 Vasil 节点 1.35.3 和新发布的 Cardano DB sync v13.0.4。 
Cardano Rosetta既是一个规范，也是一套工具，可简化与 Cardano 的集成过程。Rosetta 的目标是通过作为通用集成框架使集成过程更容易、更快和更可靠。与 Rosetta 集成的基本原理是，所有采用接口的区块链都可以通过单个接口与其通信，而不会出现任何问题。
