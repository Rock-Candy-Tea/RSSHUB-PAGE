【NFT平台bitsCrunch加入万事达卡的Start Path加密计划】4月7日消息，NFT平台bitsCrunch已加入万事达卡的Start Path加密计划。今年第二季度，bitsCrunch将获得万事达卡的咨询、营销和咨询支持。此前去年7月份消息，万事达卡为初创企业推出Start Path加密货币和区块链计划。（U.today）
