【Astar Network与波卡生态Web 3多链钱包Talisman达成合作并将推出“Spirit Key”NFT】金色财经报道，多链智能合约平台Astar Network官方宣布已经与波卡生态Web 3多链钱包Talisman达成合作并完成集成，这意味着 Astar 社区用户现在恶意直接使用 Talisman 钱包探索 Web 3 应用世界，并在波卡 Polkadot 和 Kusama 上存储、发送和接收资产。此外，Astar Network还将推出“Sprint Key”NFT，据悉这是一种特殊的 RMRK NFT，发行量为 3333 枚，持有者将获得特殊权限，比如参与未来 Alpha 版钱包使用、抢先体验 Talisman Web 应用新功能，在 Discord 频道中获得特殊许可、赢取赠品等。
