【Overledger通过新的标准化API支持Polkadot主网】4月29日消息，面向企业的DLT网关Overledger宣布通过新的标准化API支持Polkadot主网（中继链），可以在Polkadot主网上执行的OverledgerAPI功能包括创建交易、搜索交易或区块、监控地址上的活动等。 
Overledger表示，该版本为2.2.5版，在上一个版本2.2.4中，Overledger提供了对PolkadotWestendTestNet的支持。
