【BitTorrent Speed钱包地址数突破两亿六千万】据BitTorrent File System(BTFS)官方数据浏览器BTSCAN显示，截止2021年8月3日，BT Speed钱包地址数已突破两亿六千万（具体数据：267,215,695）。更多数据，用户可登陆官网查询。 
据悉，BitTorrent Speed致力于使用加密货币BTT奖励用户参与做种与带宽分享，为全球下载用户提供更优质的下载体验，全球超过一亿BitTorrent生态用户将有机会加入到去中心化互联网的货币化进程中，支持内容创作者。
