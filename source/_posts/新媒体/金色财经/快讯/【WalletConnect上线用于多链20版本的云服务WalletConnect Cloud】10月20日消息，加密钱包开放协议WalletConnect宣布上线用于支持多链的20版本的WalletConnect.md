【WalletConnect上线用于多链2.0版本的云服务WalletConnect Cloud】10月20日消息，加密钱包开放协议WalletConnect宣布上线用于支持多链的2.0版本的WalletConnect Cloud，任何开发人员或项目都可以轻松注册一个账户来生成 API 密钥以使用 WalletConnect 的新基础设施。WalletConnect Cloud 是免费的，引入了 API 密钥。WalletConnect 此前发布针对 v2.0 beta 的 JavaScript SDK，即将发布 Swift 和 Kotlin SDK alpha 版。
