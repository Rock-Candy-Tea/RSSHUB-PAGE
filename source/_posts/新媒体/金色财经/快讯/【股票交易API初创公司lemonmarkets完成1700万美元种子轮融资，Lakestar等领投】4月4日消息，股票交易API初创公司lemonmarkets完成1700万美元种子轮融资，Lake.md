【股票交易API初创公司lemon.markets完成1700万美元种子轮融资，Lakestar等领投】4月4日消息，股票交易API初创公司lemon.markets完成1700万美元种子轮融资，Lakestar和Lightspeed Venture Partners领投。创始人Max Linden表示，未来将整合加密，希望让人们能够获得最好的投资组合，设想十年后lemon.markets API将能够访问资本和数字资产市场。（TheBlock）
