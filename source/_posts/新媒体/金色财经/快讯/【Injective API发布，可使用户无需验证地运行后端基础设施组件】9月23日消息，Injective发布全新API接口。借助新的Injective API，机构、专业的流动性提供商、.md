【Injective API发布，可使用户无需验证地运行后端基础设施组件】9月23日消息，Injective发布全新API接口。借助新的Injective API，机构、专业的流动性提供商、算法交易员和个人都可以无需验证地运行后端基础设施的所有组件。这使任何人都可以获取市场数据并在完全去中心化的Injective订单簿上进行交易，而不会面临任何中心化的瓶颈或限制。 
Injective API包括 SDK 和多语言支持，包括：SDK-Python、SDK-Go 和 SDK-Typescript
