【IDEX已将V3沙盒环境部署至Polygon测试网】10月5日消息，去中心化交易所 IDEX 宣布其 V3 版本的沙盒环境已经在 Polygon 的测试网上线。IDEX 团队表示，V3 的沙盒环境是一个全面的测试网解决方案，可以通过 API 访问，该沙盒支持所有的 API 接入，包括 REST 和 WebSocket，以及各种交易功能，比如高级的订单类型。另外，IDEX 的 AMM 流动性池已经被部署并增加了初始流动性。
