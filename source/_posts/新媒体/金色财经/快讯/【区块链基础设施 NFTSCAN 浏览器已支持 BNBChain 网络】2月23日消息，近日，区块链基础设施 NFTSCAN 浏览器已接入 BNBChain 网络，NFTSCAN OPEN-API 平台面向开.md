【区块链基础设施 NFTSCAN 浏览器已支持 BNBChain 网络】2月23日消息，近日，区块链基础设施 NFTSCAN 浏览器已接入 BNBChain 网络，NFTSCAN OPEN-API 平台面向开发者推出 BNBChain NFT API 服务。这是继以太坊之后，NFTSCAN 团队支持的第 2 条公链，并计划于今年 3 月初上线 BNBChain 版 NFTSCAN 浏览器子站，为 BNBChain 生态 NFT 用户提供更加便捷的 NFT 数据搜索查询服务。
