【Ankr Network 发布一系列 SDK，将允许开发者为其用户提供代币质押和挖矿服务】8月2日消息，加密基础设施提供商 Ankr Network 发布一系列软件开发工具包（SDK），允许开发人员为其项目和平台用户提供代币质押和收益挖矿服务。这些 SDK 最初将首先在以太坊、Polygon、BNB Chain、Avalanche 和 Fantom 网络上提供。（coindesk）
