【State Machine联合著名拳王穆罕默德·阿里知识产权方推出Web3游戏】据官方消息，AI元宇宙项目Altered State Machine（ASM）宣布与Non-Fungible Lab（NF Labs）和著名拳击运动员穆罕默德·阿里知识产权方Muhammad Ali Enterprises共同开发Web3游戏 “Muhammad Ali — The Next Legends”，由ASM的人工智能协议驱动，由NF Labs领导。 
该游戏的灵感将汇集一组元宇宙拳击手，他们将在游戏世界中训练和争夺冠军。
