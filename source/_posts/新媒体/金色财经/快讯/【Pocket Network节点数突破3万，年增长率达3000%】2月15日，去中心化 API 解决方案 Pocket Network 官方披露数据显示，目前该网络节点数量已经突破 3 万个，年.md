【Pocket Network节点数突破3万，年增长率达3000%】2月15日，去中心化 API 解决方案 Pocket Network 官方披露数据显示，目前该网络节点数量已经突破 3 万个，年增长率达到 3000%。 
Pocket Network 是一个针对 Web3 应用程序的区块链云计算生态系统，利用 Web3 RPC 基础设施中间件协议提供丰富的区块链带宽，目前其节点能为比特币、以太坊、Solana、Avalanche 和 Harmony 等其他区块链提供基础设施服务。
