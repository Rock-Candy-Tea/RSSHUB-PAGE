【Algorand Name Service获Algorand基金会资助支持】3月16日消息，Algorand基金会宣布通过审查和选择决定对域名服务Algorand Name Service提供资助支持，Algorand Name Service主要为Algorand社区提供去中心化域名服务并使用Algorand智能合约来存储域名和域名管理策略。 
ANS为Algorand用户提供了工具和基础设施，以便为他们的钱包地址注册一个公开的.algo域名（例如bob.algo或alice.algo）并在后续交互中使用.algo的域名地址来引用自己的真实钱包地址。
