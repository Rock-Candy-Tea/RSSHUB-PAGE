【0x API已上线Arbitrum】9月20日消息，去中心化交易协议0x的API现已在Arbitrum上线，开发者现在可以用0x API来聚合流动资金和智能订单路由（routing），包括执行价格、深度流动性等等，目前已支持Matcha、Brave、DeFi Saver、DexKit、DODO、Frontier、Nested、Tally、Zapper和Zerion，预计在未来几周会有更多的集成。
