【Keen Technologies完成2000万美元融资】金色财经报道，3D游戏之父约翰·卡马克在社交媒体宣布创立新公司Keen Technologies并完成2000万美元融资， GitHub 前首席执行官 Nat Friedman、 Cue 创始人 Daniel Gross，以及 Stripe 联合创始人兼首席执行官 Patrick Collison、Shopify 联合创始人兼首席执行官 Tobi Lütke 、红杉资本参投。关于这轮投资，约翰·卡马克称自己并不缺钱，完全可以自己开一张 2000 万美元的支票，但为了让自己在创业这件事上下定决心并更加自律，他才选择花投资人的钱，约翰·卡马克还表示，他将继续担任 Meta 的 VR 事务顾问，但只会占去自己 20% 的时间。
