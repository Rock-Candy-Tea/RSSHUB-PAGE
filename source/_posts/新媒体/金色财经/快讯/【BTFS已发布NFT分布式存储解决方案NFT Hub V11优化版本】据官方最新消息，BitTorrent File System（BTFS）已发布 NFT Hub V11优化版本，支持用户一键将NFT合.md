【BTFS已发布NFT分布式存储解决方案NFT Hub V1.1优化版本】据官方最新消息，BitTorrent File System（BTFS）已发布 NFT Hub V1.1优化版本，支持用户一键将NFT合约添加至TronLink钱包，进而可以转出其铸造的NFT给任意地址。此外，NFT Hub V1.1优化版在用户体验方面也做了升级。 
NFT Hub为BTFS近期上线的NFT分布式存储解决方案，旨在进一步打通BTFS去中心化网络与波场TRON生态, 为NFT免费提供分布式存储支持。NFT Hub是由BTFS团队为NFT数据存储量身打造的专业化服务，开发者可以使用内容寻址和去中心化存储保护NFT元数据，确保所有NFT遵循最佳实践方案并能够进行长期稳定访问。 
 BTFS既是一种协议，也是一种网络应用，它提供了一种内容可寻址的点对点机制，用于在去中心化的文件系统中存储和共享数字化内容，同时，它也为去中心化应用提供了一个基础平台。
