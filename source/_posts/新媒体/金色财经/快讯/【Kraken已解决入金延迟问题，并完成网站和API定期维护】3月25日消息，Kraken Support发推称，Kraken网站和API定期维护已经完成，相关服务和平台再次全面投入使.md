【Kraken已解决入金延迟问题，并完成网站和API定期维护】3月25日消息，Kraken Support发推称，Kraken网站和API定期维护已经完成，相关服务和平台再次全面投入使用。 
今日凌晨，Kraken曾表示，一些客户报告称遇到入金延迟问题；目前正在调查问题，并正在寻找解决方案。根据status.kraken.com网站，入金延迟问题已经解决。 
根据公告，由于进行网站维护，Kraken Futures平台将在UTC时间3月25日11:00-11:15 （北京时间）期间不可用。一旦平台重新上线并完成检查，将会有一段“仅发布”期。
