【跨链通信协议 Movr Network 将推出跨链聚合器 FundMovr】10月9日消息，跨链通信协议 Movr Network 计划推出首款无缝跨链聚合器 FundMovr，它可将所有跨链桥、DEX 和 DEX 聚合器聚合在一起，并找到所有可用的路线，然后根据目的地链上的最大输出、交易和转账的最低 Gas 费用、最低桥接时间等因素，帮助用户以最优途径在不同的区块链之间移动资金。同时 FundMovr 会推出点对点结算层，以实现更低廉的跨链成本，并通过 SDK 和 API 让 FundMovr 可用，从而允许开发者轻松构建自己的用例。
