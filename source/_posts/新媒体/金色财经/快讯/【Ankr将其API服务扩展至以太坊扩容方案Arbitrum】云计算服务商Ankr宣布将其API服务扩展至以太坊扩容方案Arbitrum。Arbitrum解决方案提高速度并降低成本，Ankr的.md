【Ankr将其API服务扩展至以太坊扩容方案Arbitrum】云计算服务商Ankr宣布将其API服务扩展至以太坊扩容方案Arbitrum。Arbitrum解决方案提高速度并降低成本，Ankr的API端点使开发人员能够轻松地与该网络进行交互并在其上构建。据悉，Ankr API服务此前已扩展至Polygon、xDAI和Fantom。（Medium）
