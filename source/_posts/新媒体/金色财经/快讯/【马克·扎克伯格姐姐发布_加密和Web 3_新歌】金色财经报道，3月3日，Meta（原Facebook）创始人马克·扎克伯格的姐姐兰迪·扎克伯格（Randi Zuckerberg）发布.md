【马克·扎克伯格姐姐发布“加密和Web 3”新歌】金色财经报道，3月3日，Meta（原Facebook）创始人马克·扎克伯格的姐姐兰迪·扎克伯格（Randi Zuckerberg）发布了一首献给Web 3和加密行业的新歌，并声称这首歌将成为 Web 3 女性的“战斗号角”。兰迪·扎克伯格在推特上写道：“我在朋友的簇拥下创作了这首歌，作为 Web 3 女性的战斗口号，在一起，我们可以完成任何事情，玩得开心！ #WAGMI。”
