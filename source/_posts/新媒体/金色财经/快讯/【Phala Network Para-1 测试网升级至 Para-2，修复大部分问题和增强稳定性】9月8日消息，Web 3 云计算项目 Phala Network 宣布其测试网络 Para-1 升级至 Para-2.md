【Phala Network Para-1 测试网升级至 Para-2，修复大部分问题和增强稳定性】9月8日消息，Web 3 云计算项目 Phala Network 宣布其测试网络 Para-1 升级至 Para-2。 Para-2 测试网修复了 Para-1 中临近 Vmax 时奖励骤降、Gatekeeper Master Key 分发逻辑、pRuntime 代码质量等多个问题，稳定性得到进一步提升。用户可在 Para-2 上完整体验 Secure Worker Mining 全流程，模拟真实 Mining 环境，也可通过 Console 面板进行抵押设置、查看收益。 
此外，Khala 首个公投投票期即将结束，该公投通过后，Khala 将正式上线Secure Worker Mining。
