<p><span class="text-remarks" label="备注">本文来自微信公众号：</span><a href="https://mp.weixin.qq.com/s?__biz=MjM5MjAyNDUyMA%3D%3D&mid=2650733755&idx=1&sn=a6b63ebda14f10cc7d5acda568abe818&chksm=bea6ce1c89d1470af82af4bdc7e5a538a7b2bcdb042a15d60c1ca35189806cc0c80b0e21b03b&mpshare=1&scene=1&srcid=1219TgTXsPzlT2dKVq6tuqKX&sharer_sharetime=1639868382032&sharer_shareid=5ec3942ede706772360a252a7c21f39b&version=3.1.18.70072&platform=mac#rd" target="_blank" rel="nofollow" style="text-decoration: none;"><span class="text-remarks">APPSO（ID：appsolution）</span></a><span class="text-remarks">，作者：陆新宇，头图来自：视觉中国</span></p><p>今年 6 月，微软为我们带来了 Windows 11 操作系统，全新的设计语言、全面优化的触控体验、可直接运行 Android app……每一个新特性都让我们直呼微软“杀疯了”。</p><p>但没想到，升级体验后的用户，在另外一个方面直呼微软“杀疯了”：救命，我的右键刷新去哪里了？</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074539510200.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="649" src="https://img.huxiucdn.com/article/content/202112/19/074539510200.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>好像没什么用，又被无数用户视为必备功能的“刷新”，它到底有什么用呢？</p><h3 label="大标题" class="text-big-title">一、刷新真的能让电脑变快吗？</h3><p>周末的下午，我正在房间里和喜欢的女生聊着 QQ，突然房门被妈妈打开了，我赶忙缩小对话框，手指快速的刷新桌面，然后不经意的自言自语：啊，电脑好卡呀。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074541203773.gif" data-w="640" data-h="736" src="https://img.huxiucdn.com/article/content/202112/19/074541203773.gif" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">▲在你的记忆中有这样的类似经历吗？</p><p>不知道用电脑做什么时，我们总会不断的刷新桌面，而且，虽然不论校内校外，似乎都没有哪个教程明确告诉我们“刷新桌面”有什么用，但“刷新桌面会让电脑变快”这种说法，还是成为了我们的“共识”。</p><p>那么，<strong>电脑在刷新时，到底做了什么？</strong></p><p>电脑桌面看起来很特殊，但它其实只是存储在硬盘中的一个文件夹。刷新的过程，就是重新读取文件夹中的内容，并将它们再次显示的过程。所以你在点击“刷新”时，会看到桌面上的文件闪一下，这就是桌面文件进行了重新读取。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074543473457.gif" data-w="640" data-h="474" src="https://img.huxiucdn.com/article/content/202112/19/074543473457.gif" referrerpolicy="no-referrer"></p><p>这个操作有用么？当然。在你将桌面上的文件删除、重命名、移动，或者刚刚安装好一个软件并建立桌面快捷方式时，这些变化有可能不会第一时间反应到<span class="text-remarks" label="备注">（自动更新到）</span>你的桌面上。</p><p>通过刷新重新读取，修改后的名称就会显示，删除的文件就会消失，安装好的软件快捷方式就会出现。最常见的情况，可能就是你将一个文件<span class="text-remarks" label="备注">（图片或者 Word）</span>拖动到桌面上，刷新后出现文件的预览图。</p><p>如果你使用过更早的 Windows 系统，比如 Windows XP，你可能会对这个画面印象深刻。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074544419977.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="653" label="图片备注" src="https://img.huxiucdn.com/article/content/202112/19/074544419977.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">▲示意图</p><p>在你移动窗口时，桌面上会出现你移动的那个窗口的残留。这是因为窗口失去响应时，系统会接管窗口绘制工作，直到软件响应<span class="text-remarks" label="备注">（或者被结束）</span>。</p><p>在当年的微机课上，这甚至成为了一种不想认真听课的摸鱼方式。<strong>当老师要过来看到我电脑屏幕时，只需“刷新”，桌面就会恢复正常。</strong></p><p>在 Windows 95 上，有一个名为活动桌面<span class="text-remarks" label="备注">（Microsoft Active Desktop）</span>的功能，他可以显示时间、待办、股票、新闻、搜索框等功能信息。<strong>刷新桌面，可以让小组件上的信息更新。</strong></p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074545847920.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="755" src="https://img.huxiucdn.com/article/content/202112/19/074545847920.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">▲Windows 95</p><p>更新 Windows 95 活动桌面的信息、清除 Windows XP 桌面的窗口残留、刷新桌面的文件显示情况，这基本就是“刷新”操作的所有功能了。</p><p>这些功能都有一个共同点，那就是今天几乎都用不到。</p><p>在 Netmarketshare 的网站中，我们可以看到近期电脑系统的市场占比，其中 Windows 7 及以后发布的 Windows 系统占比接近 90%，另外的 10% 中还有 mac OS 等系统的身影，仍在使用 Windows XP 及之前系统的个人用户，只有不到 2%。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074546788887.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="381" src="https://img.huxiucdn.com/article/content/202112/19/074546788887.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>系统的升级、电脑性能逐渐增强，像文件显示延迟、名称修改延迟、预览显示延迟等问题，其实已经越来越少出现在我们的电脑上了，“刷新”的作用自然就越来越小了。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074547002173.gif" data-w="640" data-h="322" src="https://img.huxiucdn.com/article/content/202112/19/074547002173.gif" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">▲第一时间出现预览才是正常情况</p><p>另外，我也看到有人说刷新可以清除电脑文件资源管理器<span class="text-remarks" label="备注">（explorer.exe）</span>的缓存，从而让电脑变得更快一些。但我并未在官方资料中找到相关介绍。而且性能越来越强的电脑，清除文件资源管理器缓存得到的提升，意义也并不大。</p><p>不过我也知道，大家也并不那么在意“刷新”能否加快电脑运行速度，能否更新桌面内容。毕竟对于很多人来说，刷新桌面作为一个在无聊时可以随便进行而且几乎不会出现任何错误、不会对电脑运行造成任何影响的事情，早已成为了数字时代的抖腿。</p><h3 label="大标题" class="text-big-title">二、左右横跳的课不只是“刷新”</h3><p>在 Windows 11 中，虽然仍有概率会出现桌面更新延迟的问题，但刷新桌面这种低频操作，完全没有必要给它右键菜单的重要位置：不能删掉，放到二级菜单似乎也是一个不错的选择。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074549122964.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="592" src="https://img.huxiucdn.com/article/content/202112/19/074549122964.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">▲早期的 Windows 11，我们只能通过点击“显示更多选项”来刷新</p><p>而这一改变，就引起了文章开端提到的问题，喜欢“刷新”功能的朋友们，纷纷在社交平台里询问：我的刷新去哪里了？</p><p>不只是国内，国外也有许多用户对这一设定感到不满，他们不光想在桌面上看到刷新功能，还希望能在系统的任何地方都能看到这一功能。当然了，如何在 Windows 11 上刷新系统的教程也不会少。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074550107335.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="951" src="https://img.huxiucdn.com/article/content/202112/19/074550107335.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>似乎是用户的反馈让微软意识到了严重性，在 Windows 11 的 7 月的更新中，“刷新”又被移到了桌面右键菜单那个熟悉的位置。我理解微软把“刷新”放到二级菜单的缘由，也能预见微软把功能放回原位的情况，毕竟这种事，微软也不是第一次做了。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074551682671.gif" data-w="640" data-h="820" src="https://img.huxiucdn.com/article/content/202112/19/074551682671.gif" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">▲现在，我们又可以愉快的刷新了</p><p>1995 年 8 月，随着滚石乐队震耳欲聋的《Start Me Up》，比尔·盖茨向全球用户介绍了全新的 Windows 95。而在系统界面的左下角，首次出现了那个可以“Start Windows up”的 Start 键，而这个“开始”按钮，从那天起就在电脑的左下角陪伴了我们快 30 年。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074552056657.gif" data-w="640" data-h="360" src="https://img.huxiucdn.com/article/content/202112/19/074552056657.gif" referrerpolicy="no-referrer"></p><p>有人说，放在左下角的开始按钮，是为了方便我们在任何界面都能把鼠标移动到角落打开它，但事实上，Windows 95 的开始按钮并不能在屏幕角落唤醒，因为微软为了让它看起来像按钮，留出了边缘间隔。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074553000056.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="525" src="https://img.huxiucdn.com/article/content/202112/19/074553000056.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>有人说，开始按钮放在左下角时当年技术力不足，只能固定在屏幕边缘，后来懒得改了，但微软并没有“懒得改”，它一直在尝试。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074554244025.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="622" src="https://img.huxiucdn.com/article/content/202112/19/074554244025.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>在 Windows 8 上，微软引入了一个最大化利用屏幕空间的“开始屏幕”，来替换掉 Windows 95 以来的那个适用于小屏幕的“开始菜单”。结果我们也知道，Windows 10 上，开始菜单又回来了。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074554142955.gif" data-w="800" data-h="600" src="https://img.huxiucdn.com/article/content/202112/19/074554142955.gif" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">Internet Explorer、磁贴、Cortana、平板模式、时间线、控制面板。删除、添加、保留、整合、全新设计。<br label="图片备注" class="text-img-note"></p><p>可能是因为用户反馈，可能是因为内部优化，在最新的 Windows 11 上，我们依然可以看到不同时代的界面、不同时代的字体、不同时代的操作逻辑，以及那个作用不大，却依然占领右键 C 位的“刷新”。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074555478568.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="561" src="https://img.huxiucdn.com/article/content/202112/19/074555478568.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><h3 label="大标题" class="text-big-title">三、我们距离宣传片中的 Windows 还有多远？</h3><p>相比占领了桌面系统大片江山的 Windows，macOS 在系统更新的统一性上，要比 Windows 坚决得多。</p><p>在系统字体更新到 San Francisco 和苹方后，我们几乎就看不到其他老版字体<span class="text-remarks" label="备注">（比如华文黑体）</span>出现在 macOS 的系统中；系统全面转向 64 位后，32 位的软件就无法开启、遭到清理。</p><p>而 Windows 走向了完全不同的方向，即使在最新的 Windows 11 中，我们依然可以运行 32 位的软件<span class="text-remarks" label="备注">（比如微信）</span>，依然可以用 Windows Vista<span class="text-remarks" label="备注">（2006 年）</span>的兼容模式运行软件。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074559472638.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="712" src="https://img.huxiucdn.com/article/content/202112/19/074559472638.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>在微软的宣传片中，打字、触屏、绘画，Windows 总是那么优雅和流畅，但全球庞大的用户群体、从几百到几万元的硬件配置、几乎覆盖所有领域的专业用途，让 Windows 无法像 macOS 那么坚定的断舍离。</p><p><strong>我们距离宣传片中的 Windows 还有很远的距离。</strong></p><p>不过，Windows 11 中，开始菜单终于离开了它呆了快 30 年的左下角，居中的它不仅更适合触屏，对超宽屏用户也更加友善；Microsoft 365 采用了全新设计，与系统设计更加和谐；控制面板的功能逐步迁移到设置中，距离我们告别它又近了一步。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202112/19/074601461755.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="667" src="https://img.huxiucdn.com/article/content/202112/19/074601461755.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>可能在下次微软取消“刷新”功能时，我们就能看到宣传片中那个优雅又现代的 Windows 了吧。</p><p><span class="text-remarks" label="备注">本文来自微信公众号：</span><a
