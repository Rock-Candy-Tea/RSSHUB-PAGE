class="text-remarks">，作者：郑丽媛，题图来自：unsplash</span></p><p>两周前，我们还在就下一代 Windows 的名称争论不休：</p><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p>“Windows 10 党”认为：下一代 Windows 肯定是延续 Windows 10 的叫法，毕竟微软曾说过：“Windows 10 将是 Windows 的最后一个版本。”</p></li></ul><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p>“Windows 11 党”觉得：微软官宣 6 月 24 号的活动宣传图和活动时间都疯狂暗示“11”了，肯定叫 Windows 11。</p></li></ul><p>不过，现在我们不用吵了：昨天下一代 Windows 的预览版已曝光，显眼的“Windows 11”已然宣布了答案，猛一看居然还有点 macOS 那味儿。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/16/185305946630.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="514" data-h="410" src="https://img.huxiucdn.com/article/content/202106/16/185305946630.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">（图片来自 Tom Warren 推特）</p><h3 label="大标题" class="text-big-title">一、“苹果”味儿的 Windows 11？</h3><p>从目前网络上流出的 Windows 11 预览版镜像来看，其开始菜单和任务栏都经过了全新的 UI 设计：<strong>应用图标居中显示的任务栏、新的开始按钮和菜单、清理过的托盘区域</strong>。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/16/185308948370.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="769" data-h="616" src="https://img.huxiucdn.com/article/content/202106/16/185308948370.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>怎么样，乍一看是不是有点“苹果”味儿了？不过不用担心，如果你不喜欢居中的应用图标和开始菜单，可以进入设置将所有任务栏图标像现在一样移动到左侧。另外，Windows 11 还支持切换为深色模式。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/16/185310990570.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="1000" data-h="604" src="https://img.huxiucdn.com/article/content/202106/16/185310990570.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>有人可能要开始嘀咕了：怎么不见我的动态磁贴？别紧张，虽然许多人对这项功能并不在意，但也有人非常喜欢，因此 Windows 11 中还是保留了动态磁贴，需要的用户可以重新启用。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/16/185313019744.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="804" data-h="617" src="https://img.huxiucdn.com/article/content/202106/16/185313019744.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>另外，微软还在整个 Windows 11 中都采用圆角设计。不仅开始菜单本身是圆角，应用程序、文件资源管理器、控制面板也全部都是圆角。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/16/185316206410.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="767" data-h="615" src="https://img.huxiucdn.com/article/content/202106/16/185316206410.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>说来也巧，曾经 Windows 7 和 Vista 等旧版 Windows 采用的就是圆角设计，但 Windows 8 之后再无踪影。如今圆角却再度复兴，时尚果然是个圈。</p><p>据 The Verge 高级编辑 Tom Warren 发现，Windows 11 还在所有应用程序的最大化按钮中内置了一个新的快照功能，用户可以快速并排对齐窗口，或在桌面上按比例有序排列。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/16/185318797400.gif?imageView2/2/w/1000/format/gif/interlace/1/q/85" data-w="600" data-h="479" src="https://img.huxiucdn.com/article/content/202106/16/185318797400.gif?imageView2/2/w/1000/format/gif/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>此外，Windows 11 还有全新的启动音效、Xbox 体验以及两个新的默认壁纸：</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/16/185328594777.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="986" data-h="616" src="https://img.huxiucdn.com/article/content/202106/16/185328594777.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">（浅色模式）</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/16/185331885445.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="984" data-h="616" src="https://img.huxiucdn.com/article/content/202106/16/185331885445.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">（深色模式）<br label="图片备注" class="text-img-note"></p><p>据了解，<strong>这次泄露的 Windows 11 预览版镜像是早期版本</strong>，许多例如全新的 Windows 应用商店等功能可能尚未完全开发，其中还能看到许多 Windows 10 的影子，因此总体看来它更像是精致简约版的 Windows 10。</p><h3 label="大标题" class="text-big-title">二、Flag “屡立屡倒”</h3><p>不过，分明说好 Windows 10 是最后一个版本，怎么如今 Windows 11 却又“实锤”了？对此我们只能说：不要轻信微软立的 Flag，会倒。</p><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p><strong>Flag 1：Windows 10X</strong></p></li></ul><p>两年前微软宣布的 Windows 10X 为双屏设备而生，针对双屏场景对开始菜单、搜索栏等图标都进行了重新设计，提供了非常灵活的配件和硬件形态支持。</p><p>Windows 10X 一经面世就吸引了众人的眼球，而微软也因此立下一个 Flag：要在 2021 年春季发布首个正式版的 Windows 10X。</p><p>结果这个 Flag 不仅倒了，甚至还没了：上个月微软宣布，未来将不再继续 Windows 10X 的开发。</p><p>不过微软也表示，<strong>过往基于 Windows 10X 研发的技术不会直接弃用，而是会集成到 Windows 10 及其他产品中</strong>。如今看来，比起说 Windows 11 中有 Windows 10 的影子，说它结合了 Windows 10X 的设计可能更为贴切。</p><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p><strong>Flag 2：Windows 10</strong></p></li></ul><p>正如文章开头所说，2015 年微软开发者业务高管 Jerry Nixon 曾在微软 Ignite 大会上强调：“Windows 10 将是 Windows 的最后一个版本。”六年过去了，原以为这个 Flag 能成，但这次 Windows 11 的泄露又“打脸”了。</p><p>行吧，毕竟有了 Windows 10X 的前科，要说 Windows 11 是 Windows 10 之后的“最最后一个版本”倒也不是不能接受。可是微软连曾经声称是“Windows 最后一个版本”的 Windows 10 都没放过：近日，据 Windows 10 家庭版和专业版生命周期页面显示，微软计划于 2025 年 10 月 14 日结束对 Windows 10 的支持。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/16/185337734866.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="987" data-h="425" src="https://img.huxiucdn.com/article/content/202106/16/185337734866.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><h3 label="大标题" class="text-big-title">三、并不买账的网友</h3><p>Windows 11 突然的提前泄露也吸引了一众网友的关注，但从评论看来，大多数网友对 Windows 11 并不买账。</p><blockquote><p>“如果我想要一个 Mac 的设计外观，我会买一台 Mac，但我根本不想要啊。”</p></blockquote><blockquote><p>“微软只是移植了越来越多的 UI，但他们完全没有更新 Win32 的基础。”</p></blockquote><blockquote><p>“真想知道他们拿着数十亿美元到底在做什么？”</p></blockquote><blockquote><p>“我可以保留 Windows 10 吗？”</p></blockquote><blockquote><p>“Windows 7 是最好的！”</p></blockquote><p>不过正如上文所说，目前泄露的 Windows 11 预览版是一个早期版本，功能尚不齐全。微软随后也对这次泄露发推回应道：“这只是开始。”下一代 Windows 的真面目还需待到 6 月 24 日由其亲自揭开。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/16/185341133424.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="587" data-h="168" src="https://img.huxiucdn.com/article/content/202106/16/185341133424.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>那么对于 Windows 11 你怎么看呢？</p><p><span
