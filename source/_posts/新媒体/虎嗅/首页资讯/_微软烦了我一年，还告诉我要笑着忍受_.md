target="_blank" rel="nofollow" style="text-decoration: none;"><span class="text-remarks">CSDN（ID：CSDNnews）</span></a><span class="text-remarks">，作者：郑丽媛，题图来自：视觉中国</span></p><p>相信绝大多数 Windows 用户都有过被“催”的经历：用着 Windows XP 催 Windows 7，用着 Windows 7 催 Windows 10，现在用着 Windows 10 已经被还没正式推出的 <a href="https://mp.weixin.qq.com/s?__biz=MjM5MjAwODM4MA==&mid=2650860699&idx=1&sn=277b753d01c27e0d97d7e813a194ff4a&chksm=bd589d488a2f145e467defa4b4d276b1a93ad8fc6b29ff7cf10d24f1bead45c76281a6a35d79&scene=21#wechat_redirect" target="_blank" rel="nofollow">Windows 11 最低硬件要求催着换电脑了。</a></p><p>当然了，用户自然可以想办法关闭系统升级提醒和自动更新，但这不论对安全性还是系统性能而言都存在一定弊端，<strong>因此多数用户还是习惯于放任微软的各种提醒，然后在不胜其烦的情况下，遂了微软的意。</strong></p><p>但长此以往，微软这种不达目的就反复弹窗、提醒兼威胁的催法，令许多用户都抱怨不已，更何况微软催的还不仅仅只是操作系统的升级。</p><p>近来，一位知名科技博客作者 Chris Matyszczyk 就在科技网站 ZDNet 上分享了他的内心独白<strong>：“微软烦了我一年，还告诉我要笑着忍受。”</strong><span class="text-remarks" label="备注">（注：为更具代入感，下文将保留第一人称的叙述手法。）</span></p><h3 label="大标题" class="text-big-title">一、我行我素的微软</h3><p>我想成为一个好人，但我发现这越来越难。</p><p>你是否经历过当朋友或家人给你打电话诉苦时，忍不住点开社交软件查看各类新闻？你是否在浏览到一些求助文章时却被旁边的趣闻推荐吸引住了目光？你是否在无意旁观到别人的危急情况时第一反应是拿出手机拍摄而并非上前帮忙？</p><p>这一切的背后都归咎于数字技术，它让我们的注意力下降，让我们容易分心，更让我们似乎逐渐失去了应该有的道德感。</p><p>但这些对我来说都不是主要因素，<strong>我更厌烦的是如今科技公司不断唠叨、唠叨再唠叨直到你完全按照他们的要求去做的情形，而微软就是罪魁祸首之一。</strong></p><p>毫无疑问，在某些方面微软可以称得上是一家好公司，但它却秉持着一个核心想法：想让用户服从你的意愿的最佳方式就是气他们。</p><p>举个例子，当微软想把它号称全新且极好的 Edge 浏览器进行推广时，选择的方式是：毫无预告地强加给用户，根本不管许多 Windows 用户因此被激怒。现在也是，即便很多用户都在抗议 Windows 11 的最低硬件条件，但微软依旧我行我素。</p><h3 label="大标题" class="text-big-title">二、广告其实不是广告？</h3><p>除了操作系统，我们铁杆的 Hotmail 用户也是受害者之一。我知道它现在叫做 Outlook，但我的邮箱地址还是 Hotmail，我已经习惯了。</p><p>不知道为什么，微软老是在我的 Hotmail 页面上方插入小广告。过去一年中，这些广告一直催我下载 Edge 浏览器，可是我明明在它发布不久后的 2020 年 4 月就已经下载过了，我也的确挺喜欢 Edge 浏览器的。所以如果一定要说其中原因的话，<strong>我猜是因为我没有将 Edge 浏览器设置为默认浏览器</strong>。</p><p>最近这些广告出现的频率有所下降，但也没有完全消失，我还是需要时不时地点击“X”使其关闭。所以我向微软提问：“我要怎样才能停止下载一个我已经下载了一年多的产品？”</p><p>然而，微软的一位发言人却告诉我：<strong>“那些让你下载 Edge 或添加 Outlook 应用的提醒，实际上根本不是广告。”</strong></p><p>这个回答令我困惑，毕竟在我看来这就是一种持续并烦人的广告啊。那位发言人接着说：<strong>“这是所谓的‘Biz Bar’，它位于 Outlook.com 的顶部，用来告诉消费者有新的功能和应用可供他们使用。”</strong></p><h3 label="大标题" class="text-big-title">三、无法阻止的通知</h3><p>对于这个说法，我还是不能接受，毕竟我早就在一年前就下载过 Edge 了，我也挺喜欢的，为什么 Biz Bar 还要提醒我下载？</p><p>那位微软发言人接着说：“它主要是让客户了解 Edge 和 Outlook 的功能，比如新的 Outlook 扩展之类。<strong>它还会根据使用的浏览器，出现不同的提醒。”</strong></p><p>这么说的话，是不是因为我的默认浏览器是 Firefox，所以微软不知道我下载了 Edge，以后还会一直缠着我下载 Edge？</p><p>不行，我想想都受不了，于是我对那位发言人说：“拜托，我已经下载过 Edge 了，我也很喜欢 Edge。但 Firefox 目前是我的默认浏览器，有没有什么办法可以阻止这些你口中的‘非广告’、这些 Biz Bar 的骚扰出现在我的 Outlook 页面？”</p><p><strong>“Biz Bar 推送的通知都是可以一一关闭的，但目前还没有办法完全禁用它们。”微软发言人如此说道。</strong></p><p>但在我看来，这完全没有正面回答我的问题，我猜有些人肯定会把这句话翻译成：:“我永远、永远、永远都不会禁用它们。”</p><h3 label="大标题" class="text-big-title">四、用 Edge 打开，问题迎刃而解</h3><p>未来微软是否有办法完全禁止 Biz Bar 的通知我们不得而知，但我现在担心的是，微软是不是会一直跟我纠缠关于把 Edge 设置为默认浏览器的问题，之后我是否会一直收到这种他们口中“精选的、个性化”的信息，直到我屈服于这种繁杂的生活，最后毫无灵魂地妥协。</p><p>可悲的是，通过验证，我证明了这个可怕的猜想：<strong>当我改用 Edge 登录 Hotmail 时，Biz Bar 也立刻亮了起来，但不是之前的下载推荐，而是“你的浏览器支持将 Outlook.com 设置为默认邮件处理程序”。</strong></p><p>而这句话，在我用 Firefox 打开 Hotmail 时从未看到过。虽然 Firefox 也支持将 Outlook.com 作为我的默认邮件处理程序，但 Biz Bar 从未提醒过。</p><p>我很疑惑，为什么微软就不能设置一个广告阈值......不对，是 Biz Bar 的阈值，类似于我关闭了这个通知 20 次，那么之后就可以不再推送同样通知的设定？</p><p>我猜肯定有人说这是因为我没有充 Hotmail 会员，但实际上，<strong>我早就为了增强版的 Hotmail 服务花了 19.95 美元，而这钱并没有让我免于这些毫无意义的信息。</strong></p><p>但不论如何，我还是 Hotmail 的忠实用户，而我在说的，也只是微软公司。</p><p>最近微软为了鼓励用户续订 Microsoft 365，用一杯免费的星巴克咖啡作为赠品。那么我在想，如果我把 Edge 设为默认浏览器，有一天它会不会给我提供一辈子的免费星巴克星冰乐？</p><p>最后，你是否也曾被微软“催”得不行？</p><p><span class="text-remarks" label="备注">参考链接：</span></p><p><span class="text-remarks"
