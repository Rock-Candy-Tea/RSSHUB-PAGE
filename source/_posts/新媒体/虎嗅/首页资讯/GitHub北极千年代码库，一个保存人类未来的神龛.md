<p>GitHub Arctic Code Vault是一个数据仓库，保存在北极世界档案馆（AWA）中，这是一个长期储存的档案机构，位于北极深250米永久冻土层。该档案馆位于斯瓦尔巴群岛上一个只有几千人的小镇朗伊尔城。这里人迹罕至、气温寒冷，有几百米厚的冻土层，非常适宜存放胶片。</p><p>GitHub在2020年2月2日捕获每个活跃公共存储库的快照，将开源代码作为人类文明的火种留给后代，放在一个环境稳定、远离人类战火的地方。如果你的代码库被收录，在头像下方会有一颗雪花。</p><p label="正文" class="text-normal"><span class="text-remarks" label="备注">（via GitHub）</span></p>
