<p><span class="text-remarks" label="备注">本文来自微信公众号：</span><a href="https://mp.weixin.qq.com/s/tV8yJUO4P_lcw4xD7sVR9Q" target="_blank" rel="nofollow" style="text-decoration: none;"><span class="text-remarks">ZEALER（ID：zealertech）</span></a><span class="text-remarks">，作者：ZEALER，头图来自：unsplash</span></p><p>Windows 升级就像一座围城，升级不了的人想升级，升级上去的人想回退。</p><p>预览版实装的新功能，琳琅满目，吸引着用户更新；当用户升级之后又发现，新系统充斥着这样那样的问题，让人后悔不已。好在，Windows 提供了 7 天内可回退的功能，给了尝鲜党一份后悔药。</p><p>不幸的是，我就是对升级一时的沉醉，而错过了后悔药有效期的那个人。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192802108158.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="563" src="https://img.huxiucdn.com/article/content/202202/21/192802108158.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><h3 label="大标题" class="text-big-title">一、后磁贴时代，果里果气的 Windows </h3><h3 label="大标题" class="text-big-title"></h3><p>关于 Windows 的缔造者微软，乔布斯的一句话相信还印刻在很多人的心中：“微软最大的问题就是他们没有品位。”乔帮主认为，微软“问题很大”，他们不重视原创想法，也没有为产品注入太多文化。</p><p>乔布斯之言并不尽然。我们刚刚见证了微软最为创造力迸发的时代，就在大约 10 年前。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192804031509.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="563" src="https://img.huxiucdn.com/article/content/202202/21/192804031509.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p><strong>2011 年前后诞生的 Windows 8、Windows Phone，是一个“圆滑当道时代的锐利异类”。它们</strong>将 Windows XP Media Center、ZUNE 播放器上的 Metro UI 从幕后带到前台，方格磁贴从此成为 Windows 的一个标志。这套 UI 秉承平面设计原则，视效清爽，其模糊了图标与小组件的界限的理念，无比前卫。</p><p>十年之后，Windows 11 到来，Metro UI 逐渐从微软的舞台中央退下。Windows 11 是 Windows 设计史上的一次重大变革，动态磁贴这一锐利异类被抛弃，开始菜单采用居中布局，应用窗口与对话框全面圆角化，Vista、Win7 时代的小组件“文艺复兴”。一夜之间，Windows 11 以大变活人之姿亮相，让许多 Windows 老用户眼前一亮。</p><p><strong>总的来说，果里果气的。</strong></p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192804430902.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="1000" data-h="681" src="https://img.huxiucdn.com/article/content/202202/21/192804430902.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>微软的创造力，就如同 Windows 11 的圆角窗口一样，被磨平了棱角。<strong>Windows 11 抛弃了 Metro UI，砍掉了 Windows 手机，停止了 UWP，推翻了 Windows 8、10 上的种种前卫理念。</strong>另一边，微软的友商们尽情“二创”，把动态磁贴换成圆角，摇身一变成为新时代的小组件；Windows PC、手机、平板大一统的设想，被苹果重新拾起，练就独家的跨平台大一统生态。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192807200954.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="563" src="https://img.huxiucdn.com/article/content/202202/21/192807200954.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>理念重塑之下，Windows 11 是否如微软所愿变得诱人了呢？至少现在，还不是。</p><h3 label="大标题" class="text-big-title">二、bug，bug，bug！</h3><p>谈起 Windows 升级，最让人揪心的问题莫过于种种 bug，让人抓狂不已。Windows 历史上有着各种升级出问题的先例，Vista 这种大翻车且不提，去年四月 Windows 10 的一次更新，就导致了用户游戏帧数暴跌、蓝屏等问题。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192808117076.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="671" data-h="243" src="https://img.huxiucdn.com/article/content/202202/21/192808117076.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>Windows 11 这种大版本更新又如何呢？不出意外地，出意外了。Windows 11 正式推送不久，就有一个恶性 bug 降到了 AMD 用户头上：<strong>L3 缓存大幅减速、延迟大增</strong>。相对的，英特尔处理器没有遇到这个问题，恰巧 12 代酷睿几乎与 Windows 11 同期推出，有网友吐槽微软是不是故意当了一把“A 黑”。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192809389038.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="1000" data-h="481" src="https://img.huxiucdn.com/article/content/202202/21/192809389038.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">AMD 处理器 L3 缓存大幅降速</p><p label="图片备注" class="text-img-note">（图自 Notebookcheck）</p><p>好在微软和 AMD 很快注意到了这个问题，AMD 表示“问题不大”，对性能的影响大概只有“3%-5%”。Windows 11 正式推送几个星期之后，就发布了相应的补丁修复。</p><p>AMD 专属 bug 至少是得到了修复，可是呢，Windows 11 还有大大小小数量众多的 bug 有待修复。其中<strong>输入法切换时屏幕闪烁</strong>的问题，几乎从初始版本延续到最新的预览版，至今仍待修复。</p><h3 label="大标题" class="text-big-title">三、画张大饼慢慢烙，功能依然砍砍砍</h3><p>除了各种 bug 依旧，Windows 11 正式版还缺席了一项最重点的功能：<strong>安卓子系统</strong>。</p><p>除了大量 UI 更新之外，安卓子系统是 Windows 11 最受人瞩目的功能，可以补足 Windows 在平板电脑等触控设备上的生态缺陷。然而，Windows 11 正式推送之后，安卓子系统功能却宣告迟到，最初仅向部分美国的 Beta 测试用户开放。直到本月的最新 Windows 11 Build 22000.527<span class="text-remarks" label="备注">（KB5010414）</span>体验更新上，该功能才开启全面公测——但是，同样仅对美国用户开放。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192811056446.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="728" src="https://img.huxiucdn.com/article/content/202202/21/192811056446.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>让人揪心的远不止于此。<strong>新功能迟到也就算了，一些昔日功能竟也不知所踪，集中反映在开始菜单和任务栏上。</strong></p><p>Windows 11 的开始菜单可能是近几代 Windows 版本中最糟糕的一个，引得各大媒体与网友纷纷吐槽。Windows 11 的开始菜单不支持尺寸调节也不能全屏，无论你使用多大的显示屏，它都只显示为一小块。<br></p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192812913746.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="419" src="https://img.huxiucdn.com/article/content/202202/21/192812913746.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>开始菜单支持的单屏应用数量是固定的 3 行 6 列，总共 18 个，菜单下半部分的“推荐的项目”不允许关闭，在你关闭推荐应用显示的情况下，微软宁可让这一栏空出来，也不允许你用来显示其他应用。</p><p>不好用的同时还不让你选，这不和某些我们日常吐槽的国产软件差不多吗？</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192813280423.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="604" src="https://img.huxiucdn.com/article/content/202202/21/192813280423.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p><strong>任务栏也有着类似的设定：只能固定在屏幕底部，不能贴靠至左右侧或者顶部。</strong>任务栏拖放功能丢失，无法通过拖拽文件到任务栏程序打开，也无法拖拽程序到任务栏快速固定。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192814640008.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="596" data-h="328" src="https://img.huxiucdn.com/article/content/202202/21/192814640008.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>显而易见，<strong>Windows 11 的开始菜单和任务栏遭到史诗级削弱</strong>，不仅破坏了用户的使用习惯，而且刻意地增加了操作步骤。我们原以为微软是有意采用这样的设定，直到我们看到，微软在最新的 Dev 预览版 Build 22557 中又把任务栏拖放功能加了回来。早知如此，何必当初呢？</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192815927633.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="340" src="https://img.huxiucdn.com/article/content/202202/21/192815927633.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">Windows Dev 预览版 Build 22557，任务栏拖放打开功能复活<br label="图片备注" class="text-img-note"></p><p>先砍后加、反复横跳，这样的事情，微软一直没少做。最为经典案例是，微软在 Windows 8 上砍掉了开始菜单/屏幕上的电源按钮，当你需要关机的时候，需要用手势划出菜单、依次点击“设置-电源-关机”，造就了史上最复杂的关机操作。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192816602939.gif" data-w="619" data-h="347" src="https://img.huxiucdn.com/article/content/202202/21/192816602939.gif" referrerpolicy="no-referrer"></p><p>诚然，借助“Alt+F4”组合键以及自定义命令行+快捷方式等，你依然可以快速关闭电脑，而我们提到的各种问题，也可以借助第三方工具补救。但是，有多少人能顺利的解决这所有的问题呢？就拿砍掉电源按钮的 Windows 8 来说吧，那些高度依赖鼠标、不懂快捷键的普通用户，他们在找不到关机按钮的时候，除了直接拔电源线还有什么选择吗？</p><p>这就是为什么，油管上一个“如何关闭你的 Windows 8 电脑”的视频能收获近 16 万的播放量。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192817688974.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="217" src="https://img.huxiucdn.com/article/content/202202/21/192817688974.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><h3 label="大标题" class="text-big-title">四、Windows 11 仍是半成品</h3><p>微软对 Windows 11 的态度有些矛盾，<strong>一方面以限期免费升级催促用户，另一方面又通过限定处理器、必须有 TPM 模块等条件，设立硬件门槛，限制用户更新</strong>。当然这些限制条件也可能是出于 OEM 压力，要借 Windows 11 提高 PC 硬件销售。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192818742887.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="554" data-h="364" src="https://img.huxiucdn.com/article/content/202202/21/192818742887.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">微软暗示 Windows 11 更新可能收费<br label="图片备注" class="text-img-note"></p><p><strong>那么 Windows 11 到底好不好呢？总的来说是好的。</strong>拿开始菜单来说，微软大刀阔斧砍掉了设计前卫但上手难度高的 Metro UI，回到了更易用的抽屉式、启动器式设计。另外，微软虽然砍掉平板模式，但宜人的文字与图形元素间距、直观的触摸手势，依然很好地照顾了触屏设备的体验。Windows 11 的快捷分屏操作，也堪称是时下桌面系统之中最简洁高效的。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192821222413.gif" data-w="600" data-h="479" src="https://img.huxiucdn.com/article/content/202202/21/192821222413.gif" referrerpolicy="no-referrer"></p><p>但如今的 Windows 11，bug 不断，重磅功能还在打磨，各种迷惑设定……正式版推送至今已有四个月，Windows 11 依然像个半成品。至少从我的角度来说，升级 Windows 11 让我懊悔不已。升级到 Windows 11 的小伙伴，是否也和我一样呢？</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202202/21/192823718681.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" data-w="1000" data-h="419" src="https://img.huxiucdn.com/article/content/202202/21/192823718681.jpg?imageView2/2/w/1000/format/jpg/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>即便如此，系统当然还是新的最好，况且 Windows 10 系统会在 2025 年停止支持，我们迟早都得用上 Windows 11。我们只希望微软的计划，不是耗尽这三年的时间，才把 Windows 11 打磨成真正的正式版。</p><p><span class="text-remarks" label="备注">本文来自微信公众号：</span><a href="https://mp.weixin.qq.com/s/tV8yJUO4P_lcw4xD7sVR9Q"
