10 的延续？》，头图来自：视觉中国</span></p><p>“Windows 10 将是 Windows 的最后一个版本。”这是 2015 年微软开发者业务高管 Jerry Nixon 在微软 Ignite 大会上重申的一句话。</p><p>彼时，许多人认为 Windows 10 将成为 Windows 系统的终结版：<strong>未来不会再有新的版本号问世，只有定期的更新与改进。</strong>而纵观 Windows 10 陪伴我们的这 6 年，确实如此。</p><p>这 6 年，Windows 10 没有焕然一新的大改革，微软也没有举办有关其操作系统的重要活动，Windows 10 的活跃似乎只存在于定期的小更新和用户不间断的“吐槽”之中。</p><p>但在今年 5 月的微软 Build 开发者大会上，微软 CEO Staya Nadella 却给出了一个令人期待的信号：“不久之后，下一代 Windows 即将到来，而这也是十年来最重要的更新之一。”</p><p>Staya Nadella 没有食言。近日微软官网更新了活动页面，<strong>宣布将于美国东部时间 6 月 24 日上午 11 点举行 Windows 特别活动，揭晓有关下一代 Windows 的真面目。</strong></p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/07/114006125539.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="1000" data-h="647" src="https://img.huxiucdn.com/article/content/202106/07/114006125539.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p label="图片备注" class="text-img-note">（图片来自微软官网）<br></p><h3 label="大标题" class="text-big-title">全新的 Windows 11 还是 Windows 10 的延续？</h3><p>这个消息一经宣布就引起了众多网友的注意，甚至还有许多“福尔摩斯”从中推测出了下一代 Windows 的名字是 Windows 11：</p><blockquote><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p>活动邀请中那张疑似下一代 Windows 标志的图片中，光线透过窗户落到地面的投影没有该有的横杠，看起来非常像数字“11”；</p></li></ul><p><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/07/114009700352.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="363" data-h="200" style="text-align: center;" src="https://img.huxiucdn.com/article/content/202106/07/114009700352.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p>活动开始时间是上午“11”点。</p></li></ul></blockquote><p>其实仅靠这两点推测下一代 Windows 是 Windows 11 有些牵强，但微软似乎是注意到了网友这番推测，也不知是出于“恶作剧”心态还是在认真剧透，微软在推特上发布了该图像的动态版本，证明投影的确是有意没有横杠的。</p><p><video src="https://video.huxiucdn.com/mda-mf6gn76fuirpu1cd/mda-mf6gn76fuirpu1cd.mp4" poster="https://v2-img.huxiucdn.com/mda-mf6gn76fuirpu1cd/mda-mf6gn76fuirpu1cd.jpg" controls="controls" width="100%" data-video="0">您目前设备暂不支持播放</video><br></p><p>微软这番疑似暗示的操作令网友更加笃定了 Windows 11 即将到来。</p><p>不仅如此，今日国外的爆料大神 @evleaks<span class="text-remarks" label="备注">（Evan Blass）</span>也加入了“Windows 11 党”，他发布了一则推特：</p><blockquote class="js_blockquote_wrap" data-type="2" data-url data-author-name data-content-utf8-length="42" data-source-title><p>来自“不要对此版本进行截图”部门的消息：即将发布的微软操作系统叫 Windows 11。</p></blockquote><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/07/114012982176.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="527" data-h="262" src="https://img.huxiucdn.com/article/content/202106/07/114012982176.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>种种迹象似乎都在指向微软即将发布的下一代 Windows 就是 Windows 11，但这样的话又与 2015 年微软坚持“Windows 10 将是 Windows 的最后一个版本”的主张相悖。</p><p>因此下一代 Windows 的名字究竟是 Windows 10 的延续还是全新的 Windows 11 目前还无法得知，不过这个号称是“十年来最重要的更新之一”的内容我们可以预测一波。</p><h3 label="大标题" class="text-big-title">下一代 Windows 会有哪些更新？</h3><p>既然是“十年来最重要的更新之一”的下一代 Windows，那微软此前必定做足了准备，因此在 6 月 24 日到来之前，我们可以根据微软近来的举措从中找出一些“预告”。</p><p label="小标题" class="text-sm-title">重大 UI 变化：“Windows 的全面视觉复兴”</p><p>本次重大更新在微软内部被称为“太阳谷”项目，而在视觉方面，微软将对 Windows 10 的基本设计进行许多改动。据了解，<strong>这些视觉变化中许多设计都来源于微软上个月宣布放弃的 Windows 10X。</strong></p><p>通过太阳谷更新，微软预计将带来重新设计的开始菜单、操作中心、文件资源管理器和任务栏，使其具有更现代的外观。其中最为显著的变化应该是微软将抛弃一直以来采用的直角边框设计，并将圆角与渐变色、浮动效果等元素添加至系统菜单。此外，为了改善圆角体验，微软还计划更改最小化、最大化和关闭窗口周围的填充格式。</p><p>其实 Windows 7 和 Vista 等旧版 Windows 曾经采用的就是圆角设计，不过自从 Windows 8 开始就不见圆角的踪影了。</p><p label="小标题" class="text-sm-title">新的 Windows 应用商店</p><p>最近几个月微软一直在为 Windows 开发一个新的应用商店，有传言称这将与如今的应用商店大不相同。微软 CEO Staya Nadella 曾承诺“Windows 将为开发者和创作者给予更多的经济机会”，因此新的应用商店似乎就是以此为目的在进行改革。</p><p>据了解，<strong>新的应用商店将面向所有类型的应用程序和游戏开放。</strong>此外，未来微软也将允许开发者向应用商店提交标准的 EXE 或 MSI 包，并且可以通过开发者自己的内容交付网络<span class="text-remarks" label="备注">（CDN）</span>管理更新。</p><p label="小标题" class="text-sm-title">其他值得关注的更新</p><p>除了以上两点，下一代 Windows 预计还支持运行 Linux GUI 应用程序；修复多台显示器上重新排列应用程序的问题；即将推出 Xbox Auto HDR 功能；对蓝牙音频支持的改进。</p><h3 label="大标题" class="text-big-title">6 月 24 日见！</h3><p>目前已经有多家媒体收到了来自微软官方的邀请函，据这些媒体透露，微软 CEO Staya Nadella 和首席产品官 Panos Panay 都将出席这场为下一代 Windows 举办的特别活动。这场活动令微软高管 Yusuf Mehdi 都十分期待：“我自 Windows 95 以来从未对新版本的 Windows 感到如此兴奋！”</p><p>不过也有许多网友对下一代 Windows 并不感兴趣：</p><blockquote><p>“我今天刚买了 Windows 10 的密钥，请不要发布 Windows 11 让我再花 6 英镑买升级密钥。”</p><p>“臃肿的操作系统会带来更多的问题。”</p><p><strong>“它最好是好的，而不只是新 UI 的变化。”</strong></p></blockquote><p>但不管怎样，这是微软在 6 年之后首次对操作系统的大动作，并且 Staya Nadella 主张的“十年来最重要的更新之一”也充满噱头，令人充满期待。</p><p><span class="text-remarks" label="备注">参考链接：</span></p><p><span
