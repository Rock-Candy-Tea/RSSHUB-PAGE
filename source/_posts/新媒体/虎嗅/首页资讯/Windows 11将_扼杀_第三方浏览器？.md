<p><span class="text-remarks" label="备注">本文来自微信公众号：</span><a href="https://mp.weixin.qq.com/s/EBOUVUHzFop9KQd5DBBrdQ" target="_blank" rel="nofollow" style="text-decoration: none;"><span class="text-remarks">CSDN（ID：CSDNnews）</span></a><span class="text-remarks">，作者：郑丽媛，原文标题：《Windows 11 将“扼杀”第三方浏览器？Firefox 绝地反击！》</span></p><p>相信大多 Windows 10 用户都有这样一个体会：默认浏览器可一键设置成微软自家的 Edge，但若想将其换成别的浏览器，步骤则稍显繁多，同时会弹出建议保持 Edge 作为默认浏览器的提示弹窗。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202109/17/160605115681.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="361" data-h="234" src="https://img.huxiucdn.com/article/content/202109/17/160605115681.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>在此基础上，许多人为图省事，会长期放任 Windows 10 的默认设置。尽管 Firefox 或 Chrome 等其他浏览器也会提示将其设置为默认浏览器，可这个步骤每多一步，我们的耐心就会被多磨灭一分，直至放弃。</p><p>因此，众多浏览器厂商都对微软在 Windows 10 中默认捆绑 Edge 的行为十分不满，控诉微软没有给他们一个公平竞争的环境，但这并没有令微软做出任何改变。</p><p>俗语有言“山不就我，我来就山”，近日 Firefox 浏览器的开发公司 Mozilla 便自己行动起来了：据外媒 The Verge 报道，在上个月发布的 Firefox 91 中，<strong>Mozilla 对微软在 Windows 10 中将 Edge 设置为默认浏览器的方式进行了逆向工程，使用户也可以一键将 Firefox 设置为默认浏览器</strong>。</p><h3 label="大标题" class="text-big-title">一、6 年前没有回应的诉求</h3><p>早在 6 年之前，也就是 Windows 10 正式发布的第二天，Mozilla 就已明确表明立场：<strong>希望微软改变 Windows 10 的默认浏览器设置。</strong></p><p>为此，彼时 Mozilla 的 CEO Chris Beard 给微软 CEO Satya Nadella 写了一封公开信《致微软 CEO 的一封公开信：不要在选择和控制上倒退》。<span class="text-remarks" label="备注">（注：Mozilla 方面表示，此前他们已专门就该问题与微软团队联系，但并未取得进展，所以才有了这封公开信。）</span></p><p>彼时许多将系统升级至 Windows 10 的用户发现，他们无法在安装过程中将 Firefox、Chrome 等第三方浏览器设置为默认浏览器，因为 Edge 会自动成为 Windows 10 的默认浏览器。因此，Chris Beard 表示，Windows 10 的更新“似乎是微软为了发展自己的浏览器而迫使用户放弃他们对其他浏览器的选择”。</p><p>Chris Beard 在公开信中说道：“有时我们会看到巨大的进步，比如消费产品尊重个人及其选择。但是，<strong>在 Windows 10 发布以后，我们非常失望，因为微软采取了一个严重倒退的举措</strong>。”</p><p>Chris Beard 明确指出，明明微软可以保留以前那样简单设置默认浏览器的方式，但 Windows 10 的整个升级体验与默认设置 API 的设计都改变了，它将第三方浏览器设置为默认浏览器的方法隐去，并且整个操作也变得复杂很多，“现在人们需要两倍多的鼠标点击次数、滚动浏览内容和一些复杂技术才能重新选择默认浏览器。”</p><p>因此，Mozilla 方面对微软提出诉求：“强烈敦促微软重新考虑其商业策略，尊重人们选择和控制其在线体验的权利，让人们更容易、更明显、更直观地自行选择，而不是仅仅提供特权给微软自己的产品。”</p><p>在公开信最后，Chris Beard 呼吁：“请在 Windows 10 中为你的用户提供他们应得的选择和控制权。”</p><p>可惜，这个公开的诉求并没有改变什么。</p><h3 label="大标题" class="text-big-title">二、Mozilla：那我自己来</h3><p>既然微软不为所动，那 Mozilla 只好“自力更生”了。</p><p>外媒 The Verge 报道，Mozilla 在今年 8 月 10 日发布的 Firefox 91 中，对微软在 Windows 10 中将 Edge 设置为默认设置的方式进行了逆向工程，使 Firefox 也能像  Edge 那样可以快速简便地设置为默认浏览器。</p><p>据了解，<strong>Mozilla 进行的逆向工程可确保用户在 Firefox 浏览器中即可将其设置为默认浏览器，并且不会收到系统弹出建议保持 Edge 的提示</strong>，因为它“绕过了微软为了确保恶意软件无法劫持默认应用而在 Windows 10 中内置的反劫持保护机制”。</p><p>Mozilla 方面对 The Verge 解释道：“人们本来应该能简单轻松地设置的，但这在 Windows 10 并未实现。所有操作系统都应为默认设置提供官方支持，以便用户可轻松将他们的应用设置为默认状态。”</p><p>截至目前，仅 Mozilla 一家采取了这一举措，谷歌、Vivaldi、Opera 等其他浏览器厂商还未表明态度，同时微软也尚未对此事有所回应或抵制，因此这一问题在 Windows 10 中暂时得到了解决。但令 Mozilla 更为担忧的是，<strong>在十月即将发布的 Windows 11 中，切换默认浏览器的难度将再度升级</strong>。</p><h3 label="大标题" class="text-big-title">三、令众多浏览器厂商担忧的 Windows 11</h3><p>据了解，在目前公开的 Windows 11 预览版中，有两种改变默认浏览器的方式。</p><p>一种是在安装新浏览器并首次打开 Web 链接时，会出现一个提示：</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202109/17/160609924887.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="280" data-h="313" src="https://img.huxiucdn.com/article/content/202109/17/160609924887.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>原始的默认浏览器是 Edge，如果用户想要切换，需点击想换的浏览器并勾选“始终使用此应用”。但很多情况下，用户其实很容易忘记点击“始终使用此应用”，而只要错过第一次，这一提示就再也不会自动弹出。</p><p>如果用户无意中错过简便的第一种方式，那么只能用既繁琐又混乱的第二种方式来更换默认浏览器了：需按照文件或链接类型一个一个来设置默认浏览器。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202109/17/160611142839.gif?imageView2/2/w/1000/format/gif/interlace/1/q/85" data-w="600" data-h="650" src="https://img.huxiucdn.com/article/content/202109/17/160611142839.gif?imageView2/2/w/1000/format/gif/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>以 Chrome 为例，用户如要在  Windows 11 将其设置为默认浏览器，需把 HTM、HTML、PDF、SHTML、SVG、WEBP、XHT、XHTML、FTP、HTTP 和 HTTPS 的默认打开方式全部更改一遍，麻烦指数爆表，简直就在跟用户说：“放弃吧，不要改了。”</p><p>对此，<strong>多家浏览器厂商都很担忧，因为这无疑极大削弱了旗下浏览器成为 Windows 11 默认浏览器的可能</strong>。</p><p>谷歌 Android、Chrome 和 Chrome OS 负责人 Hiroshi Lockheimer 发推表示：“这家公司自称最开放、拥有‘最多选择’，我希望这只是开发者预览版，正式版 Windows 11 能像他们说的那样‘开放’。因为如今看来，这根本没有‘选择’可言。”</p><p>Opera 浏览器主管 Krystian Kolondra 也指出：“当平台供应商为了提高自己产品的地位而掩盖其他选项时，是非常不幸的。我们希望鼓励所有平台供应商尊重用户选择并允许公平竞争，而取消用户选择也是一种倒退的象征。”</p><h3 label="大标题" class="text-big-title">四、网友：怎么没人告微软？</h3><p>这一事件在众多资讯网站都登上了热榜，多数网友对微软的做法均持反对态度。</p><p>@Strom：</p><blockquote><p>“Windows 11 默认管理器真的对用户有敌意吧！微软居然认为让非技术用户手动调整 15 种不同的文件和关联协议以更改默认浏览器是一个好主意？这让我想起了糟糕的 Windows XP 时代，当时也是如此，而我不希望非技术用户再次遭受这种体验。”</p></blockquote><p>@tgtweak：‍‍‍</p><blockquote><p>每次 Windows 更新后整个屏幕都是微软写的废话，还要求用户“使用微软推荐的浏览器设置”，理由是“让我们一起把 Windows 做得更好”。但最令人失望的不是微软这样的做法，而是居然没有人以有效的方式将其告上法庭。</p></blockquote><p>@_fat_santa：</p><blockquote><p>“有趣，我想看看 MS 会如何回应：他们是会屈服并让用户轻松设置他们的默认浏览器，还是会变成猫捉老鼠的游戏？最后，我认为 MS 对默认浏览器所做的事愚不可及。他们难道没有从 2000 年代的反垄断案中学到什么吗？”</p></blockquote><p><span class="text-remarks" label="备注">参考链接：</span></p><p><span class="text-remarks"
