<p><span class="text-remarks" label="备注">本文来自微信公众号：</span><a href="https://mp.weixin.qq.com/s/WVr_rahLAGSUWC-osq9OEw" target="_blank" rel="nofollow" style="text-decoration: none;"><span class="text-remarks">CSDN（ID：CSDNnews）</span></a><span class="text-remarks">，作者：郑丽媛，原文标题：《Windows 11 再惹“众怒”！网友：微软就是逼我去买新电脑！》</span></p><p>一般来说，不论是移动还是桌面操作系统，如若要升级版本，大多用户都不会产生过大的抵触情绪，毕竟更新往往都是为了确保用户获得最佳体验。但近来用户对微软推出 Windows 11 的态度却是“反其道而行之”。</p><p>自 6 月 24 日微软正式官宣 Windows 11 之后，随之而来的不是期待与支持，而是用户从未平息的“抱怨”：<strong>升级 Windows 11 的硬件要求太高了</strong>！</p><p>面对争议，微软于 7 月 21 日特地进行了一个名为“Windows 11 升级路径和部署工具”的主题直播。</p><p>微软项目经理 Aria Carley 在直播中回应了有关升级到 Windows 11 的硬件要求问题，可她的回答却激起了又一轮的“众怒”：“<strong>我们知道这对受影响的用户来说很糟糕，但为了确保设备的安全，我们还是会阻止不符合硬件标准的设备升级系统</strong>。”</p><h3 label="大标题" class="text-big-title">一、硬件“高门槛”引发众怒</h3><p>Windows 11 刚官宣时，它的最低硬件要求就已被众多网友“吐槽”，甚至还有国外网友认为这是 Linux 崛起的最佳时机。在这之中，<strong>绝大多数无法达到硬件要求的设备都卡在了“TPM 2.0”这一点上</strong>。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202108/06/185809270794.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="1000" data-h="491" src="https://img.huxiucdn.com/article/content/202108/06/185809270794.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>2014 年才正式发布的 TPM 2.0，到广泛使用也花费了一定时间，因此<strong> Windows 11 的这一硬件要求等于将许多 5~6 年前生产的计算机都拒之门外</strong>。</p><p>对于这点，Aria Carley 在直播中表示，虽然目前 Windows 11 的 Alpha 版本并没有对设备进行严格的硬件要求<span class="text-remarks" label="备注">（包括但不限于支持 TPM 2.0）</span>，但她明确指出，<strong>Windows 11 最终版本的升级将严格遵守最低硬件要求。</strong></p><p>Aria Carley 说：“我们知道有些设备不符合 Windows 11 的标准，这很糟糕。但这是为了保持设备更高效，让用户有更好的体验。最重要的是，这会比以前更安全。”</p><p>因此，虽然 Aria Carley 对用户的抱怨表示理解，但可以看出微软对 Windows 11 的硬件要求还是很坚持。</p><p>Aria Carley 在回答随后的一个问题中也表现出了这一态度：“即便你利用组策略<span class="text-remarks" label="备注">（Group Policy）</span>也无法绕过 Windows 11 强制执行的硬件要求。此外，为了确保你的设备得到足够的支持和安全，我们会阻止你升级不符合要求的设备。”</p><p>这种可谓强硬的姿态不出所料地再次引发众怒。据外媒 Windows Central 报道，微软这次的直播上传到了 YouTube 平台，而这则视频最热的评论是：</p><blockquote><p>“很多问题的答案听起来<span class="text-remarks" label="备注">（微软）</span>就像是个‘聋子’……看来 Windows 11 将成为另一个 Windows 8 了。”<span class="text-remarks" label="备注">（注：Windows Vista 是微软举世公认的败笔之一，而 Windows 8 实际比 Windows Vista 的市场占有率还低。）</span></p></blockquote><p>还有用户留言评论称，微软这些看似不必要的硬件要求实际是一种几乎不加掩饰的策略，目的就是为了推动新电脑及 Windows 许可证的销售。</p><h3 label="大标题" class="text-big-title">二、微软不予回应，删评禁言冷处理</h3><p>面对 YouTube 视频下清一色的负面评论，微软选择的解决方法尤为“简单粗暴”：删除所有现有评论，并关闭了网友评论功能。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202108/06/185811527609.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="736" data-h="186" src="https://img.huxiucdn.com/article/content/202108/06/185811527609.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>尽管如此，微软却无法控制点赞功能——截至发稿，该视频的点赞数仅有 178，踩的人数却高达 3784。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202108/06/185812671990.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="744" data-h="526" src="https://img.huxiucdn.com/article/content/202108/06/185812671990.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>虽然 YouTube 的评论功能已被关闭，但网友们还是在许多外媒对此事件的报道下留下了自己的态度：</p><p>“我确定我可以在我的 CPU 中启用 TPM 2.0，但我不会启用的。”</p><p>“微软是在说：‘我们将通过不支持你的硬件逼你<span class="text-remarks" label="备注">（买新电脑）</span>！’的意思吗？”</p><p>“无论如何，我都不急于尝试 Windows 11，因为我对目前 Windows 11 的一切都不感兴趣。Windows 10 将支持到 2025 年，届时我的电脑应该将在 Mac 或 Linux 上运行。”</p><p><span class="text-remarks" label="备注">参考链接：</span></p><p><span class="text-remarks"
