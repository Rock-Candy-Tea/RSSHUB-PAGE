href="https://mp.weixin.qq.com/s/Y7Sqvf5dKP10eyQrqQHPlQ" target="_blank" rel="nofollow" style="text-decoration: none;"><span class="text-remarks">CSDN（ID：CSDNnews）</span></a><span class="text-remarks">，作者：苏宓，题图来自：Microsoft</span></p><p>时钟拨回 2019 年 10 月 2 日，微软在 Surface 新品发布会上重磅发布了多款产品，其中压轴出场的是微软推出的首款 Surface 折叠屏电脑——Surface Neo，该设备搭载了全新设计的操作系统 Windows 10X，这款操作系统利用容器技术支持运行所有的 Win32 应用，除此之外，它也特别针对了双屏场景进行优化，并对开始菜单、搜索栏等图标进行了重新设计，提供了非常灵活的配件和硬件形态支持。彼时，一经发布，在赚足了用户眼球的同时，不少人评价，凭借这款操作系统，微软将再次颠覆 PC 的形态。</p><p>然而，在短短两年不到的时间之后，微软于近日宣布，<strong>将不再继续 Windows 10X 的开发</strong>，但是过往基于 Windows 10X 研发的技术也不会直接弃用，而是会集成到 Windows 10 及其他产品中。</p><h3 label="大标题" class="text-big-title">一、Windows 10 当道，微软为什么还要开发 Windows 10X？</h3><p>众所周知，微软凭借 Windows 操作系统树立了自己独一无二的核心竞争力与品牌，也引领了 PC 时代。在进入数字化经济时代之初，微软在 2015 年宣布 Windows 10 将作为最后一个 Windows 独立版本，以 Windows 即服务的方式进行后续的更新和维护。那 Windows 10X 又是如何开了头？</p><p>还记得 2019 年初，有不少媒体爆料，微软正在开发一款 Windows 10 的轻量级版本，内部代号为 Lite，它旨在运行 PWA 和 Windows 10 应用商店的应用。同年 10 月，微软官宣 Windows 10X，也有不少人认为它是微软内部对 Lite 项目的延续。</p><p>事实证明，的确如此。</p><p>从一定程度上来看，<strong>Windows 10X 是简化版本的 Windows 10</strong>。它是基于 Windows Core OS<span class="text-remarks" label="备注">（WCOS）</span>构建而成的。</p><p>那么 WCOS 又是什么呢？据<a href="https://www.windowscentral.com/windows-core-os" target="_blank" rel="nofollow"> WindowsCentral 报道</a>，2017 年初以来，微软内部开始研发一款 Windows Core OS<span class="text-remarks" label="备注">（WCOS）</span>，并且已经投入使用了很长时间，它采用了 OneCore 的共享代码并在其之上构建了一个现代、无遗留的操作系统。Windows Core OS 与 CShell 一起使微软可以快速创建 Windows 10 的新版本，并可以快速、轻松地在不同设备类型和 WCOS 版本之间共享标准组件和功能。对此，微软将 WCOS 称之为是 Windows 的未来。</p><p>因此，<strong>基于 Windows 10 和 WCOS，微软简化了许多 Windows 10 的功能，如文件资源管理器等等，带来了 Windows 10X</strong>。</p><p>从 Windows 10X 的命名上我们不难看出，X 在罗马数字中代表的也是 10，“Windows 10 10”，顾名思义，<strong>该系统诞生的初衷就是为了双屏设备而生，这也意味着其并不是面向普通设备的系统</strong>。据悉，Windows 10X 只会搭载在新的硬件设备上发布，像 Windows 10 等系统并不能通过升级的方式更新成 Windows 10X。</p><h3 label="大标题" class="text-big-title">二、Windows 10X：从延期到宣告停止研发</h3><p>不过，随着时间的推移，Windows 10X 的发展似有些“雷声大雨点小”。</p><p>记得在 2019 年 Windows 10X 诞生伊始，微软表示该系统可以在双屏和可折叠设备上运行，还特地对系统进行模块化设计，在软件设计层面也特地针对这些设备做了优化，与此同时，其表示，首款 Windows 10X 设备将在 2020 年秋季上市。</p><p>然而，在 2020 年 5 月 4 日，微软 Windows 设备首席产品官 Panos Panay 发布<a href="https://blogs.windows.com/windowsexperience/2020/05/04/accelerating-innovation-in-windows-10-to-meet-customers-where-they-are/" target="_blank" rel="nofollow">公告</a>称：</p><blockquote><p>将加速 Windows 10 的创新，以确保 Windows 设备是工作、学习和游戏的最佳方式。另外，随着外界关注度的增加，Windows 的优先级也发生了转变。与去年 10 月我们对于新型双屏 Windows 设备的愿景大不相同，我们将继续把客户的需求放在首位，且集中精力满足用户的需求，我们的客户比以往更加需要使用云力量，现在是时候以不同的方式来推进这一需求。</p><p>借助 Windows 10X 的灵活性，我们将会把精力转向单屏 Windows 10X 设备，这些设备会利用云的力量帮助客户以新的方式工作、学习和游戏。这些单屏设备将是我们交付给客户的 Windows 10X 的首批产品。未来，我们将继续与 OEM 合作伙伴一起寻找合适的时机，将双屏设备推向市场。</p></blockquote><p>到了 7 月，微软再次宣布 Windows 10X 操作系统以及第三方 OEM 设备将延期。与此同时，有消息表示，单屏 Windows 10X 或要等到 2021 年春季才会发布，而对双屏支持的版本甚至可能要等到 2022 年。</p><p>如今 2021 年春季已过，微软 Windows 服务和交付负责人 John Cable 于 5 月 18 日发布公告中直接宣告了 Windows 10X 最终的结局，告知用户无须再等待了，因为“Windows 10X 已被融入到其他产品里面”：</p><blockquote><p>经过长达一年的探索，并与客户交流讨论，我们意识到 Windows 10X 技术可以比我们最初想象的更有用，它可以为更多的客户提供服务。因此我们达成共识，Windows 10X 技术不应仅局限于一部分客户。</p><p>我们没有像最初打算的那样在 2021 年将 Windows 10X 产品推向市场，而是利用我们迄今为止的经验教训，加速将基础关键的 10X 技术集成到 Windows 其他部分和公司的产品中。事实上，其中一些技术已经体现在 Windows Insider 预览版本的 Windows 核心中。例如，我们正在将新的容器技术集成到 Microsoft Defender Application Guard 之类的产品中，从而增强 Voice Typing 体验等等。</p></blockquote><p>这一公告发布，也意味着被视为是 Google ChromeOS 强劲竞争对手的 Windows 10X “已死”。</p><h3 label="大标题" class="text-big-title">三、Windows 10X 为什么会失败？</h3><p>如今回首看 Windows 10X 的发展，从高调入场到低调退出，究其失败的原因，主要有几点：</p><p><strong>其一，定位不明。</strong>从最初搭载到双屏设备中，到中间主要目标是单屏设备，再到当前融入其他产品中，不难看出，微软内部对于 Windows 10X 的定位一直不太明确，而这或许也是造成 Windows 10X 最终以失败告终的真正原因所在。</p><p><strong>其二，生态的缺失。</strong>操作系统得以发展起来，最为关键的要素就是生态。此前微软承诺 Windows 10X 利用容器技术支持运行所有的 Win32 应用，不过在外媒测试了 Windows 10X 的 RTM 候选版本之后发现，该系统并不能运行 Win32 应用，其没有命令行、控制面板等等。因此如果推翻 Win32 生态系统，重建一个全新的，并没有说说那么简单，这也是为何现在数百种操作系统中，仅有几款能够成为主流的重要原因。</p><p><strong>其三，当前的时机和市场环境或并不适合 Windows 10X 生存。</strong>最初在微软的构想中，Windows 10X 系统将会搭载双屏电脑等新型设备进入市场，然而在如今 PC 端 Windows、macOS 当道、普通的电脑设备已经满足了用户的日常需求情况下，需要有硬需应用场景才能催生 Windows 10X 快速落地，而 Windows 10X 想要挑战 Windows 10 的意义也并不大。</p><h3 label="大标题" class="text-big-title">四、结语</h3><p>因此，Windows 10X 虽然具备一定的创新性，但是其适用的场景不会是现在，或许是未来，也或许永远不会出现。</p><p>当下，微软在宣布停止 Windows 10X 开发之后，也将目光与精力重新聚焦于 Windows 10。据外媒报道，Windows 10 即将到来的“太阳谷”更新，将带来重大的 UI 更新，从中，我们或许能看到一些从 Windows 10X 移植过去的技术。</p><p><span
