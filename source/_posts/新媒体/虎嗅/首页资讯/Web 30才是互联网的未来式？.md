<p><span class="text-remarks" label="备注">本文来自微信公众号：</span><a href="https://mp.weixin.qq.com/s/f2B32Uo5T8gPuI-HDDh99w" target="_blank" rel="nofollow" style="text-decoration: none;"><span class="text-remarks">AI前线（ID：ai-front）</span></a><span class="text-remarks">，作者：Josh Drake（DFINITY首席运营官），译者：王强，策划：凌敏，原文标题：《元宇宙之后又一概念出圈，Web 3.0来了，这才是互联网的未来式？》，头图来自：unsplash</span></p><p>Web 已经发展了超过三十年。我们从早年间以静态文本和图像为主的 Web 1.0 时代一点点向前迈进，直到 Web 2.0 的崛起带来了我们今天熟悉的互联网——一个由中心化平台主导的网络世界。<br></p><p>今天，我们正处于 Web 3.0 时代的开端。Web 3.0 的愿景是让价值和数据在去中心化的平台上无缝移动，所有权和控制权也都是分散的。然而，要实现这个新 Web 世界的愿景，我们仍有一些障碍需要克服。</p><h3 label="大标题" class="text-big-title">一、从 Web 1.0 到 Web 2.0</h3><p>尽管 Web 1.0 的世界本质上是很粗糙的，<strong>但它是一个由去中心化、开放协议定义的时代</strong>。最早在互联网上运行的平台是由开源代码驱动的，人们通过论坛、公告板系统和邮件组共享内容。那些代码中大部分都是没有专利的，而且是免费的。</p><p>然后，在互联网泡沫崩溃<span class="text-remarks" label="备注">（约 2000 年）</span>之后，一批新的在线服务出现了。随着互联网传输速度的提高，流媒体网站<span class="text-remarks" label="备注">（如 YouTube 和 Netflix）</span>开始脱颖而出。与此同时，社交媒体应运而生；很快，通过分享视频或直播将各种体验通过媒体传递出去的能力得到了广泛普及。这个时代也见证了移动互联网的兴起，由于手机和平板电脑的出现，人们有了更多机会接触这些平台。</p><p>然而，这些新的平台是由企业和公司打造的。<strong>曾经定义 Web 1.0 世界的开源代码和平台被种种专利替代。</strong>没有人可以随意复制和修改这些模式，因为这样做会招来严重的法律后果。</p><p>此外，这些平台的崛起将用户聚集到了中心化的中心，这些中心最后被 Facebook 和谷歌等大型科技公司所控制。不久之后，收集用户数据成为了标准实践——这种做法往往掩盖在表面上的“免费”服务幌子后面。不幸的是，这就是 Web 2.0 时代的整体样貌。</p><h3 label="大标题" class="text-big-title">二、去中心化的 Web 3.0</h3><p>不管互联网迄今为止的发展轨迹如何，它还是在不断发展当中的。由区块链驱动的去中心化应用<span class="text-remarks" label="备注">（DAPP）</span>有望再次将控制权从少数中心化的守门人手中夺回来。一个“货币互联网”正在崛起，伴随而生的还有分布式数据托管和众多由用户控制的平台。</p><p><strong>去中心化的金融</strong>，或称 DeFi，已经为全世界数百万人提供了交换资产、赚取被动收入、贷款等等能力，所有这些服务都没有中间人的参与。这意味着用加密货币为难以获得银行服务的人们提供金融服务的理念已经走入现实。</p><p>这种技术也不仅被用于个人资金管理。这个新世界为 SocialFi 这样的事物提供了可能性——这是社交媒体和金融的混合体，旨在准确提供你所期望的事物。</p><p>如果没有区块链，通过社交应用发送金钱的选项会更加繁琐，也不那么安全，并且需要独立的第三方来提供信任，而不是像区块链那样提供一个自动和不可改变的支付层。然而有了这一层之后，用户就可以进行无信任的交易——踢走成本高昂的中间商，提供更高效的支付能力。</p><p><strong>还有一个蓬勃发展的 GameFi 世界，与即将到来的元宇宙愿景紧密相连。</strong>游戏中的经济将由加密货币资产和不可伪造的代币<span class="text-remarks" label="备注">（NFTs）</span>驱动。游戏玩家遵循“边玩边赚”的模式，在游戏中投入的时间可以用来赚取真正能被他们拥有的，具有真实价值的资产。</p><p>此外，除了提供无信任的交易模式和去中介化的实体外，去中心化技术还提供了一种全新的商业模式，以取代互联网的主流原型。这种模式的形式是<strong>去中心化自治组织</strong><span class="text-remarks" label="备注">（DAO）</span>，它将控制权分配给一个用户集体，允许他们对平台的发展方向掌握发言权，通过有形的所有权和治理奖励来激励用户使用。</p><p>这意味着初创企业现在可以在一个公平的环境中与现有的服务竞争，通过激励措施吸引用户群，并推动网络传播效应以扩大用户群。</p><p>这种新的互联网有很大可能完全改变用户、开发者和品牌的互动方式，但要达成这一目标仍有很多工作要做。</p><h3 label="大标题" class="text-big-title">三、离真正的 Web 3.0 还有多远？</h3><p>具体而言，阻碍 Web 3.0 发展的因素主要有两点。</p><p><strong>首先，许多自称是去中心化的应用远远没有达到真正的去中心化状态。 </strong>这些服务的前端运行在云服务器上的情况并不少见，这意味着对它们的访问仍然依赖于传统的基础设施，而区块链只是偶尔被用来发送或接收数据而已。</p><p>即使是旨在实现去中心化愿景的区块链网络本身，如果它们中的大多数都运行在亚马逊 Web 服务或类似的中心化企业云上，也会丧失这种特性。这种情况越来越多，甚至在以太坊的例子中也是如此。诸如此类的单点故障正是区块链设计时就要避免的状况，所以目前的情况既不理想，也不可持续。</p><p><strong>另一个阻碍进展的共同问题是，许多 Web 3.0 服务目前运行在以太坊上，交易费用高昂、吞吐量低下，且没有外部基础设施就无法扩展</strong>，这是众所周知的。以太坊作为 Web 3.0 的骨干，其存在的这些固有缺陷将影响其潜力发挥。为了让元宇宙真正为每个人服务，像这样的门槛和障碍都需要尽快解决。</p><h3 label="大标题" class="text-big-title">四、写在最后</h3><p>综上所述，说我们已经实现了 Web 3.0 的愿景还为时过早。但这并不意味着行业没有打下基础。</p><p>例如，新的平台正在出现，希望完全摆脱 Web 2.0 标准的束缚。这包括了互联网计算机——一个以 Web 速度运行的区块链，为 Web 提供服务，并为智能合约提供一个可无限扩展的平台。互联网计算机已经克服了扩展、速度和费用问题，为既不慢也不贵的 DeFi 体验创造了实现条件——允许 DAPP 充分利用去中心化架构的全部潜力。</p><p>为了实现 Web 3.0 的愿景，在线服务的每个部分<span class="text-remarks" label="备注">（从前端到后端）</span>都需要在链上托管。Web 3.0 不需要传统的基础设施，这意味着没有守门员和最小停机时间。这将带来 100% 真实的 Web 3.0 体验，同时提供明显更好的性能。</p><p>然而，这个新技术时代最重要的因素之一是，它必须从头开始正确构建。这是一个重新想象互联网的机会，重要的是要确保它是为了最终用户的利益而生，而不是以公司的利益为重。</p><p><span class="text-remarks" label="备注">原文链接：</span></p><p><span class="text-remarks"
