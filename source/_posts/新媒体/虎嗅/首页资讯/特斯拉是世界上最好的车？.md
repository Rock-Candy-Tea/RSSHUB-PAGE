<img alt="特斯拉是世界上最好的车？" data-v-f85d0f0e src="https://www.huxiu.com/article/undefined" referrerpolicy="no-referrer"><p>标题是用来钓鱼的，点进来的同学可以在评论区说说，自己觉得，世界上最好的车是哪辆？</p><p>马斯克说世界上最好的车是今儿刚开始在美国交付的 Model S Plaid 版本。</p><p>尽管发布会时间一再延迟，甚至略微有点儿让人怀疑延迟多次的发布会主持人会是罗永浩，但马斯克还是用非常快的速度，介绍了下今天交付的首批 25 辆 Model S Plaid。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/11/150146193339.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="1000" data-h="665" src="https://img.huxiucdn.com/article/content/202106/11/150146193339.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>按照马斯克在发布会上的表述，这是世界上最好的车，比任何保时捷都快，比任何沃尔沃都安全。</p><p>确实，Model S Plaid 版本在参数上看起来非常强。前后三电机，最大功率1020马力，四轮驱动。百公里加速时间 2.1 秒，最高时速 322 公里，EPA 续航 628 公里，15 分钟快充可以补充 300 公里续航。全新的热泵空调让冬季续航增加 30%，制冷和制热的能耗都比之前降低了 50%。散热器尺寸加大一倍，让 Model S Plaid 可以多次全力弹射加速。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/11/150304069692.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="1000" data-h="421" src="https://img.huxiucdn.com/article/content/202106/11/150304069692.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>这确实是这个世界上，综合性能最好的纯电动车，也是世界上百公里加速能力最好的量产车，之前这个记录的保持者是布加迪 Chiron，售价 260 万美元，从 2016 年发布到去年，一共交付了 250 辆车。</p><p>而特斯拉 Model S，美国市场的售价 122,990 美元，中国市场售价 105.99 万元。从今天开始美国交付 25 辆，随后每周交付 100 辆，中国的预定车主在明年第一季度就能提车，甚至更快。</p><p>一百万人民币出头买全世界最快量产车，这事儿香吗？确实香。</p><p>甚至新款 Model S 中控的那块儿大屏，算力比 PS5 还要高，基本与 Xbox Series X 持平，可以直接用手柄在中控屏上以 60 帧的刷新率玩儿《赛博朋克 2077》。基本相当于买车送了个游戏机，对中年已婚晚上回家不想上楼面对家庭的男性非常友好。</p><p class="img-center-box"><img class="lazyImg" _src="https://img.huxiucdn.com/article/content/202106/11/150349742615.png?imageView2/2/w/1000/format/png/interlace/1/q/85" data-w="1000" data-h="514" src="https://img.huxiucdn.com/article/content/202106/11/150349742615.png?imageView2/2/w/1000/format/png/interlace/1/q/85" referrerpolicy="no-referrer"></p><p>方向盘也经过了重新设计，类似赛车的共轭式方向盘，砍掉方向盘上半部分的设计更加有利于在辅助驾驶时，给驾驶员提供一个开阔的视线。当然，比较让人担心的是，赛车用这样的方向盘主要是因为转相比与家用车不一样，不需要转很多圈，Model S 现在这个共轭式方向盘在掉头打方向时候会不会麻烦还不确定。</p><p>另外挡杆被取消了，特斯拉号称 Model S 可以自动换挡，当jiashiz额坐进车内踩下制动踏板后，车辆会自动判断行进方向并挂入相应档位。</p><p>无论在车辆动力，还是车辆内饰里，特斯拉都做了非常多的调整，然而，Model S Plaid 的外观没怎么变。</p><p>Model S Plaid 的外观只做了一些小调整，让整车风阻只有 0.208。</p><p>也许，特斯拉觉得 Model S 的外观已经足够出色了。</p><p>回想起去年连线采访特斯拉首席设计师弗朗茨·冯·霍尔茨豪森（Franz von Holzhausen），他对于 Model S 的设计是这样描述的：</p><p>Model S要养眼高级，但不该一味追求酷；产品得让人觉得舒服，有亲和力，感觉是自己能够长期拥有的东西。这样的设计，是以环法自行车赛手为灵感的，因为这些自行车赛手，都不是肌肉型猛男。他们的身体线条紧致流畅，没有一点赘肉。正是因为这个体型，让赛手们既有惊人的爆发力, 又有很强的耐力（平均赛程超过3500公里）。</p><p>当时我觉得，这样的设计明显是一种权衡策略，Model S 的定位是对标宝马 7 系和奔驰 S 级的四门豪华轿车，售价不低，目标消费者年龄也并不特别年轻。这部分消费者，想要不一样的豪华车，但对特别酷的车型其实并不是特别买账。</p><p>像兰博基尼、法拉利那样造型出挑的超跑，平均消费者年龄层要比选择大型豪华轿车的消费者小很多。更重要的是，特斯拉的目标是让人们都开上电动车，上来就用出挑的造型去敲富豪中都算少数的超跑消费者，显然是不合理的（在 Model S 之前，特斯拉也尝试过 Roadster 1），也更加困难。</p><p>所以 Model S 保持了相对保守的外观设计，当然，相对保守不代表 Model S 的设计循规蹈矩，即使在 2021 年，看老款的 Model S，也不会有特别落后的感觉。</p><p>但是，时代变了，特斯拉也不是当年那个特斯拉了。</p><p>Model S 刚发布的时候，特斯拉还没有现在那么多超级充电站，还没有 Model 3，那会儿的特斯拉更像个新锐的豪华品牌，但现在的特斯拉，更像丰田。</p><p>特斯拉品牌形象的转变，可能意味着新款 Model S 的市场需求，不会复现当年那种极为好看且体面的光景。</p><p>毕竟当年的 Model S 车主，身上挂着“百万豪车”“环保先锋”“尊享充电”的标签，如今的特斯拉 Model S，依然售价百万，但车主要想使用超级充电站，还得去和 Model 3 车主们一起排队，还得担心自己的车会不会很快降价，还得琢磨琢磨身边的亲朋好友会不会嘲讽自己可能会成为韭菜。</p><p>Model S Plaid 性价比高吗？确实高。但很明显，这个价位的消费者，其实并没有那么车辆参数上的性价比。因为汽车领域发展百年，所谓豪华品牌或者是豪华车，车辆性能优异是最基本的标签，车给人提供的，其实是一种感觉。</p><p>前段时间看到这么一段话：一辆车能够透露关于这个人的一切。虽然它只占据我们大概8%的时间，却几乎承载了100%的生活，人的远行都是从车开始的。在这个钢板构建的盒子里，藏着每个人最本质的秘密。</p><p>你看，车给人的感觉，其实某种意义上和车没有关系，主要看人。</p><p>我自己身边有一些朋友，是中国第一批 Model S 车主，他们再换车时候，有人还是买了特斯拉，有的人，选了保时捷 Taycan，对，就是那辆无论是续航还是加速都不如特斯拉的保时捷电动爹。</p><p>原因其实特别简单，保时捷中国说，一个典型的保时捷客户在家庭中大约还有2辆车。 例如，我们的 Taycan 客户在家庭中平均拥有两辆半汽车。</p><p>我也确实相信买 100 万售价区间纯电动的车主，是不需要一辆车全能的，因为他们总有别的代步选择，而且翻看历史上那些让人印象深刻的车型，绝大多数也不是什么高性价比车型。</p><p>当然，话虽然这么说，我还是希望身边能有朋友第一波购车，让我有机会体验下全世界最快加速的量产车。</p><p>我自己挺喜欢 Model S Plaid，就是如果能外观更反常规一点，就更好了。</p>
