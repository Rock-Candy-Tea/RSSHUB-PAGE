<p data-vmark="fc27"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 13 日消息，本周初，微软将 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 预览版 Build 22499 发布给了 Dev 频道的内部人员。该版本具有从打开的应用程序窗口直接从任务栏快速分享内容到 Teams 会议电话的功能。此外，时钟应用程序支持使用微软的工作和学校账户进行登录。</p><p data-vmark="0497">今天，微软发布了 Windows 11 累积更新 Build 22499.1010（KB5008400）。该更新不包括任何新功能，旨在测试微软在 Dev 频道中构建的服务管道。</p><h2 data-vmark="06ab">新增功能：</h2><p data-vmark="b993"><span class="accentTextColor">用户能够将打开的应用程序窗口中的内容直接从任务栏快速分享到会议呼叫中</span>。这种体验将从 Microsoft Teams 开始，消除为了分享窗口而在应用程序之间来回切换的需要，用户在屏幕上看到的内容不会受到干扰。</p><p data-vmark="3fee"><img src="https://img.ithome.com/newsuploadfiles/2021/11/10eb4e9d-f16f-419d-9cf3-2c86561cf503.png" w="895" h="594" title="微软 Win11 预览版 Build 22499.1010 (KB5008400) 发布" width="895" height="544" referrerpolicy="no-referrer"></p><p data-vmark="1161">▲ 通过将鼠标悬停在任务栏上打开的窗口上，就可以看到快速分享按钮</p><p data-vmark="623c">IT之家了解到，该功能首先推广到安装了 Microsoft Teams for Work 或 Schools 的 Windows Insiders 用户群中，其他用户将在后续获得该功能。</p><h2 data-vmark="114d">改进与提升：</h2><ul class=" list-paddingleft-2"><li><p data-vmark="81a4">根据反馈，微软使任务视图和 Alt + Tab 中的键盘焦点视觉效果更加突出，以便更容易看到。</p></li></ul><h2 data-vmark="685a">修复：</h2><p data-vmark="5645">输入：</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="9451">使用此版本后，剪贴板历史记录应备份并正常运行。</p></li><li><p data-vmark="a3df">点击表情符号面板中的 gif 现在会将它们插入支持的应用程序中。</p></li><li><p data-vmark="45ee">更新了多种语言的后端词典 —— 触摸键盘的文本建议和自动更正现在应该更准确。</p></li><li><p data-vmark="87c5">如果用户更改强调色，表情符号面板中的强调色将立即生效，而不是停留在旧颜色上。</p></li><li><p data-vmark="81aa">修复了选择使用以前版本的拼音输入法时发生输入法崩溃问题。</p></li><li><p data-vmark="85c0">缓解了最近有时发生的与触摸键盘使用相关的 explorer.exe 崩溃问题。</p></li></ul><p data-vmark="b36a">窗口：</p><ul class=" list-paddingleft-2"><li><p data-vmark="65f3">修复了在 Alt + Tab 处于打开状态时按 Alt + F4 会导致 explorer.exe 崩溃的问题。</p></li></ul><p data-vmark="966f">设置：</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="a2b1">修复了在通过远程桌面访问 PC 时尝试检查远程声音属性时可能发生的设置崩溃。</p></li></ul><p data-vmark="3256">其他：</p><ul class=" list-paddingleft-2"><li><p data-vmark="df3b">修复了在某些超宽显示器上使用 OOBE 时导致一些剪裁/意外缩放的问题。</p></li><li><p data-vmark="46bb">当用户的指纹在登录屏幕上无法识别时，错误消息中的撇号现在应该可以正确显示。</p></li><li><p data-vmark="3962">在通过按下截图工具中的新建按钮截取 UWP 应用的屏幕截图时，截图工具应在截图完成后进入前台。</p></li><li><p data-vmark="f840">修复了导致某些电脑最近在从睡眠中醒来时进行错误检查的问题，错误消息为“SYSTEM_THREAD_EXCEPTION_NOT_HANDLED”。</p></li><li><p data-vmark="94cc">修复了可能与 MediaPlaybackCommandManager 相关的死锁，导致某些应用有时无法播放媒体。</p></li><li><p data-vmark="fa67">解决了在查看更多信息时导致可靠性监视器中的报告意外为空白且只有一个空矩形的问题。</p></li><li><p data-vmark="67c2">解决了导致某些游戏有一些延迟的问题，这种延迟只有在窗口聚焦时才明显。</p></li></ul>
