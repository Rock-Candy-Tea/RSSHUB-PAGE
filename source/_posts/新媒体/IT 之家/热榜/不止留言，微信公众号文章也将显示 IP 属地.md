            <p data-vmark="238c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 29 日消息，今日晚间，微信官方发布《微信公众平台即将展示用户 IP 属地的公告》称，<strong>将显示用户发布内容时的 IP 属地</strong>。该功能将于近日进行测试，后续将根据测试情况陆续对用户开放。</p><p data-vmark="01f7" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/59985d8d-8392-4980-8ec9-fe7a155fddcd.png" alt="微信" title="不止留言，微信公众号文章也将显示 IP 属地" referrerpolicy="no-referrer"></p><p data-vmark="0920">据IT之家网友反馈，微信公众平台展示用户 IP 属地将<strong>涉及公众号文章、公众号留言等</strong>。其中，公众号文章 IP 属地将阅读量上方显示，公众号留言 IP 属地将显示在用户名后。</p><p data-vmark="8dde" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/e0509f1a-416a-4744-95d3-79550ababb3d.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xZC9xZDkxLnBuZw==,g_3,x_9,y_9,a_0,t_100" w="593" h="179" alt="微信公众号文章IP属地" title="不止留言，微信公众号文章也将显示 IP 属地" width="593" height="179" referrerpolicy="no-referrer"></p><p data-vmark="c610" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/8de2e134-a3eb-4294-aa9b-8d9c5a4e6759.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xZC9xZDEwMi5wbmc=,g_9,x_10,y_10,a_0,t_100" alt="微信公众平台即将展示用户 IP 属地" title="不止留言，微信公众号文章也将显示 IP 属地" referrerpolicy="no-referrer"></p><p data-vmark="53ff">根据微信官方公告，<strong>微信公众平台境内账号展示到省</strong>（自治区、直辖市），<strong>境外账号展示到国家</strong>（地区），账号 IP 属地以运营商提供信息为准，<strong>用户暂时无法主动开启或关闭相关展示</strong>。</p><p data-vmark="cdff">IT之家了解到，微博今年 3 月宣布测试开放部分国内用户的 IP 归属地展示，并于 4 月 28 日全量开放评论展示发评 IP 属地小尾巴功能、上线个人主页一级页面展示 IP 属地。本月，今日头条、抖音、百度百家号、知乎、快手等均宣布拟在个人主页等位置展示账号 IP 属地。</p><p data-vmark="ae71"><strong>相关阅读：</strong></p><p data-vmark="9796">《<a href="https://www.ithome.com/0/616/019.htm" target="_blank">公众号留言将显示位置，微信公众平台宣布即将展示用户 IP 属地</a>》</p>
