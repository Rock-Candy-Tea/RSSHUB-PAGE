<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 14 日消息 外媒 MSPoweruser 报道，微软在博客上最近更新了 Chromium 版 Edge 的功能路线图，有几个新功能将在未来几个月内出现。IT之家获悉，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 和 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 系统上的 Edge 浏览器即将推出一些新功能，包括标签组、购物体验中价格历史、改进的画中画功能、PDF 自由格式高亮等。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/346b2e4d-e475-4f4c-98d2-4215e4959b3f.png" w="904" h="508" title="抢先看，2021 年微软 Win11/Win10 的 Edge 浏览器还将更新什么内容" width="904" height="461" referrerpolicy="no-referrer"></p><h2>下面是 Edge 2021 新功能预览</h2><p><strong>在 PDF 文档中增加自由形式文本框。</strong>从 Edge 94 版本开始，Edge 支持在 PDF 文档中添加自由表格文本框，你可以用它来填写表格和添加可见的注释。</p><p><strong>用 URL 链接启动 PWA 而不是浏览器。</strong>从 Microsoft Edge 97 版本开始，你可以选择在 PWA 中启动浏览器，而不是完整的本地浏览器。</p><p><strong>标签组。</strong>将标签归类为用户定义的组能力有助于你在多个工作流中更有效地寻找、切换和管理标签。为了实现这一点，微软将从 Edge 93 版本开始开启标签分组功能。</p><p><strong>购物体验中的价格历史。</strong>价格历史帮助你决定现在是否是购买产品的最佳时机。你现在可以看到感兴趣的产品在 Amazon.com 和其他 9 家零售商的价格变化，让你有快速、无压力的购物体验。</p><p><strong>收藏中加入引文支持。</strong>从微软 Edge 95 版本开始，已经改进了收藏的体验，特别是对学生和研究人员。收藏将开始支持引文和阅读列表。</p><p><strong>新政策支持 Microsoft Edge 启动时更多灵活性</strong>。从 Microsoft Edge 94 版本开始，企业管理员在启动策略中选择的选项将更加灵活。这将包括能够指定起始页、添加/删除额外的页面，以及在你离开的地方继续，同时确保政策指定的一组页面也被打开。</p><p><strong>从悬停工具条进入视频画中画（PiP）</strong>。从 Edge 93 版本开始，进入画中画（PiP）模式将变得更加容易。当你将鼠标悬停在一个支持的视频上时，会出现工具栏，允许你在画中画窗口中查看该视频。</p><p><strong>在 PDF 上自由突出显示。</strong>从微软 Edge 94 版本开始，PDF 浏览和标记体验将通过增加自由格式高亮显示而得到改善。这将使你能够突出显示用户可能无法评论的 PDF 文件部分，以及扫描的文件。</p><p><strong>来自网络应用通知显示为来自 PWA。</strong>从微软 Edge 94 版本开始，来自渐进式网络应用 PWA 的通知将不再显示为来自微软 Edge，而是显示网络应用的名称。</p><p><strong>在使用垂直标签时隐藏标题栏。</strong>在垂直标签中，通过隐藏浏览器的标题栏，获得额外的几个像素。从微软 Edge 93 版本开始，你可以进入 edge://settings/appearance，在自定义工具栏部分选择在垂直标签模式下隐藏标题栏的选项。</p><p><strong>对未知语言进行自动翻译。</strong>从微软 Edge 94 版本开始，当你导航到一个显示语言，但不是你设置的已知语言的页面时，微软 Edge 会自动将该页面的内容翻译成用户的主要语言。然后，微软会让你选择返回原始页面或禁用进一步自动翻译。</p><p><strong>微软 Edge 初始偏好。</strong>从 93 版本开始，在企业中部署 Edge 浏览器将变得更加容易，因为增加了初始偏好。</p><p><strong>改进 IE 模式和现代浏览器之间的衔接。</strong>从 Edge 95 版本开始，现代浏览器体验和 IE 模式体验之间的过渡将得到改善。推荐人标题、帖子数据、表单数据和请求方法现在将在两种体验之间正确转发。</p><p><strong>家庭安全 — 改进对 YouTube.com 等网站的保护。</strong>使用微软家庭安全的家庭将很快看到内容过滤器的改进。家长们也将更加放心，因为允许的网站和阻止的网站过滤设置将被改进，例如 YouTube.com，以防止儿童浏览不允许的视频。</p><p><strong>家庭安全 — 只需一次点击就可以允许多个 EDU 教育网站。</strong>使用微软家庭安全的家庭将很快看到内容过滤器的改进。当使用“只使用允许的网站”模式时，父母将看到一个选项，可以通过一次点击将教育网站列表添加到允许列表中。</p>
