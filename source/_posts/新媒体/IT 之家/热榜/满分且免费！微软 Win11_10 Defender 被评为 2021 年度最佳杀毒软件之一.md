<p data-vmark="132f"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 27 日消息，据 Windows Latest 报道，AV-TEST 发布了最新报告，预装在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 和 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 中的 Windows Defender 是 2021 年最好的反病毒软件之一。AV-TEST 是一家独立的 IT 安全机构，进行各种测试，根据性能、安全和其他因素对反病毒软件进行排名。</p><p data-vmark="4cfe"><img src="https://img.ithome.com/newsuploadfiles/2021/11/318cfba2-c553-4917-aa2e-2d48f2b31089.jpg" w="696" h="385" title="满分且免费！微软 Win11/10 Defender 被评为 2021 年度最佳杀毒软件之一" width="696" height="385" referrerpolicy="no-referrer"></p><p data-vmark="da1c">总部位于德国的 AV-TEST 研究所已经发布了 2021 年 10 月的最佳防病毒程序评估报告。值得指出的是，该结果是针对 Windows 10 家庭用户的，该安全机构测试了来自不同供应商的 21 个安全程序，包括诺顿、卡巴斯基和微软。</p><p data-vmark="22e8">根据报告，AV-TEST 对 Windows Defender、Avira、McAfee、Avast、AVG 和其他几个反病毒应用程序进行了各种安全方面的测试。例如，该组织针对零日恶意软件、网络攻击、受感染的电子邮件等测试了这些防病毒解决方案。</p><p data-vmark="a523">“我们一直使用所有产品的最新公开版本进行测试。他们被允许在任何时候自我更新，并查询他们的云端服务，”安全报告中写道。</p><p data-vmark="1d9e">AV-TEST 有严格的测试程序，Windows 10 上的 Windows Defender 在所有保护、性能和可用性方面都获得了满分。McAfee 和 Avast 等厂商的其他安全解决方案也获得了满分，但 Windows Defender 是唯一的免费防病毒解决方案。</p><p data-vmark="4295"><img src="https://img.ithome.com/newsuploadfiles/2021/11/2e4fa785-62d7-4b78-a7f0-2ae57c676d1e.jpg@s_2,w_820,h_631" w="1300" h="1000" title="满分且免费！微软 Win11/10 Defender 被评为 2021 年度最佳杀毒软件之一" srcset="https://img.ithome.com/newsuploadfiles/2021/11/2e4fa785-62d7-4b78-a7f0-2ae57c676d1e.jpg 2x" width="1300" height="631" referrerpolicy="no-referrer"></p><p data-vmark="bc88">在测试中，安全软件产品在保护、性能和可用性类别中最多可以各得到 6 分。</p><p data-vmark="f788">从 AV-TEST 结果截图中可以看到，Windows Defender 获得了 18 分，该组织宣布微软 Defender 是 Windows 10 的顶级安全解决方案之一，估计也是 Windows 11 的顶级安全解决方案。</p><h3 data-vmark="d6c9">用于 Windows 11 的新 Defender 应用程序</h3><p data-vmark="d400">IT之家获悉，作为改善跨平台安全解决方案的努力的一部分，微软显然正在开发新的安全中心。该应用程序被称为“Microsoft Defender Preview”，它将允许用户监控他们的 Windows 11 PC、Mac 和移动平台（<a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 和 iOS）。</p><p data-vmark="94d0"><img src="https://img.ithome.com/newsuploadfiles/2021/11/efb755e2-7430-4f75-b4a0-683852778e4a.jpg@s_2,w_820,h_461" w="1440" h="810" title="满分且免费！微软 Win11/10 Defender 被评为 2021 年度最佳杀毒软件之一" srcset="https://img.ithome.com/newsuploadfiles/2021/11/efb755e2-7430-4f75-b4a0-683852778e4a.jpg 2x" width="1440" height="461" referrerpolicy="no-referrer"></p><p data-vmark="020b">通过新的 Windows 安全应用程序，你将能够在桌面和移动设备上监控防病毒、网络钓鱼、密码和盗窃警报。</p><p data-vmark="e855">微软 Defender 将支持新的个性化仪表板，允许用户在一个地方查看 Windows 设备和其他连接设备的安全状态。</p><p data-vmark="eae5">据报道，全新 Windows 11 Defender 应用程序的代号为 Gibraltar，目前正在开发中。</p>
