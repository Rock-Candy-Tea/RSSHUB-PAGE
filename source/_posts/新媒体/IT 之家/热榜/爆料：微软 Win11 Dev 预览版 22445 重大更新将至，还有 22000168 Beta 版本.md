<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 27 日消息 微软最近警告 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Insiders 内测用户，微软将很快开始面向 Dev 开发频道推送不太稳定和更具实验性的预览版，微软将开始测试不打算在今年晚些时候全面发布的功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/109a2aef-e58e-483f-bf00-dc9d27953a37.png" w="1440" h="692" title="爆料：微软 Win11 Dev 预览版 22445 重大更新将至，还有 22000.168 Beta 版本" width="1440" height="394" referrerpolicy="no-referrer"></p><p>现在，爆料人士 Albacore 暗示 Windows 11 Insider 预览版重大更新要来了，其中微软将面向 Dev 频道用户推送 Build 22445 版本，面向 Beta 频道用户推送 Build 22000.168 版本更新。</p><p><img src="https://qimg.ithome.com/newfiles/2021/8/b2a1a83e-8f13-4538-a916-196623eada99.jpg" title="爆料：微软 Win11 Dev 预览版 22445 重大更新将至，还有 22000.168 Beta 版本" referrerpolicy="no-referrer"></p><p>IT之家获悉，微软此前表示，Windows 11 每年只会进行一次大版本更新，今年下半年会带来一些功能更新，例如<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a> App 支持。</p><p>值得一提的是，目前微软 Dev 和 Beta 两个通道的用户目前都在测试 Windows 11 Build 22000.160，拥有完全相同的功能。</p><p><strong>相关阅读：</strong></p><p>《<a href="https://www.ithome.com/0/571/073.htm" target="_blank">Bugs 即将来袭，大量 Win11 内测用户被困在 Dev 开发频道中，无法切换到 Beta，修改注册表都没用</a>》</p>
