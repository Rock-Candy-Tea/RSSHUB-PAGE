<p data-vmark="7e2d"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 10 日消息，苹果本周推出了第三代 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> SE，带来了包括 A15 芯片、5G 支持、更长的电池寿命、相机升级和更坚固耐用的玻璃等方面的升级。虽然苹果官方并未高调宣传，但开发者发现新款 iPhone SE 还配备了更大的 RAM。</p><p data-vmark="f692" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/1b8b1cc9-2360-44e7-b163-912cd1edb089.jpg" w="400" h="225" alt="iPhone SE 3 堆叠" title="苹果 iPhone SE 3 确认配备 4GB 内存，前代仅有 3GB" width="400" height="225" referrerpolicy="no-referrer"></p><p data-vmark="bc3b">开发者 Moritz Sternemann 发现，苹果第三代 iPhone SE 配备了 4GB 内存，而之前的型号仅有 3GB（由 Apple“Peek Performance”后发布的 Xcode 13.3 RC 版本中的字符串确认）。</p><p data-vmark="1002">IT之家了解到，之前 Xcode 字符串也准确地揭示了好几代 iPhone 和 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 中的 RAM 量。例如，去年的字符串就明确显示了苹果第六代 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad mini</a> 也增加到了 4GB 的 RAM。</p><p data-vmark="76ba">也就是说，虽然新机外观没什么变化，但内部配置还是有一定升级的。也得益于此，新款 iPhone SE 可以更流畅地使用照片和视频编辑应用，而且额外的 RAM 还可以让 Safari 后台保持活动状态。</p><p data-vmark="5ede">苹果在昨日的新品发布会上正式推出了 <span class="accentTextColor">iPhone SE 3</span>，搭载 A15 处理器。</p><p data-vmark="8455" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/09b5eb55-06f2-4946-966d-f1d2d57bacfe.png" w="941" h="405" title="苹果 iPhone SE 3 确认配备 4GB 内存，前代仅有 3GB" width="941" height="353" referrerpolicy="no-referrer"></p><p data-vmark="11d7">据介绍，iPhone SE 3 采用了性能强大的 CPU，<span class="accentTextColor">性能可达 iPhone 8 的 1.8 倍</span>，6 核中央处理器（2 个性能核心和 4 个能效核心），4 核图形处理器，16 核神经网络引擎，<span class="accentTextColor">支持 5G 网络</span>。</p><p data-vmark="82a3" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/b2b83059-711c-4e77-8cdf-bd434c7aaf4d.png" w="932" h="526" title="苹果 iPhone SE 3 确认配备 4GB 内存，前代仅有 3GB" width="932" height="463" referrerpolicy="no-referrer"></p><p data-vmark="5b6b">iPhone SE 3 拥有<span class="accentTextColor">黑、白、红</span>三款配色，搭载 4.7 英寸 LCD 显示屏，保留了经典的 Touch ID，续航也更好。此外，iPhone SE 3 搭载 1200 万像素后置镜头，搭配 A15 仿生芯片可以实现更强的拍照功能。</p><p data-vmark="e8e3">此外，iPhone SE 3 采用了更多环保元素，抛弃了塑料包装盒等等。</p><p data-vmark="9f01" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/c0d8c5b0-d559-4efa-9a90-8d455e4b240d.png" w="938" h="532" title="苹果 iPhone SE 3 确认配备 4GB 内存，前代仅有 3GB" width="938" height="465" referrerpolicy="no-referrer"></p><p data-vmark="0c46">IT之家了解到，iPhone SE 3 起价 <span class="accentTextColor">429 美元</span>（3499 元人民币），拥有 64GB、128GB、256GB 三种存储版本，将于 3 月 18 日正式开售。</p><p data-vmark="cca1" height="459" referrerpolicy="no-referrer"></p>
