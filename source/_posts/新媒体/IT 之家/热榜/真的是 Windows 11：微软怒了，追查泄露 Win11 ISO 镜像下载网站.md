<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 19 日消息 外媒 Windows Central 报道，本周，某个公共论坛上泄露了所谓的 Windows 11 ISO 镜像系统。该版本的 Windows 在整个 WinVer、安装屏幕、开箱设置体验等方面都表明是 Windows 11，还带来了全新的 UI 和一些隐藏功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/6a34afea-31f1-4ae5-ba6d-f8f709203705.png" w="752" h="560" title="真的是 Windows 11：微软怒了，追查泄露 Win11 ISO 镜像下载网站" width="752" height="560" referrerpolicy="no-referrer"></p><p>虽然我们从来没有怀疑过泄漏的真实性或该操作系统被标记为 Windows 11 的事实，但有些人仍然认为这是微软的一些邪恶的诡计。换句话说，有人认为这次泄漏是微软故意的，Windows 11 的 Logo 是障眼法，本月晚些时候，微软将揭示真正的操作系统和新名称。</p><p>但是根据微软的行动来看，这家软件巨头对 Windows 11 提前泄露十分不满。</p><p>微软对位于印度的技术出版物 Beebom 发出了正式的数字千年版权法（DMCA）投诉。Fossbytes 今天早上首先发现了这一抗议，DMCA 投诉可在网上查看。申诉是昨天发出的，描述很简短但很有趣。</p><p>“Beebom.com 的文章正在分发 Windows 11 ISO（版权归微软所有）。请从搜索中删除他们的文章。这是一个未发布的 Windows 11 的泄露副本。”</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/3478609f-a33a-492a-a884-4c8d1c4a9ed8.png" w="1204" h="848" title="真的是 Windows 11：微软怒了，追查泄露 Win11 ISO 镜像下载网站" width="1204" height="578" referrerpolicy="no-referrer"></p><p>目前还不清楚其他直接分发泄露的 ISO 镜像的网站是否也收到了撤下通知，但一旦被微软发现，似乎很有可能会被要求执行删除操作。</p><p>然而，更重要的消息是，微软否认了这个 ISO 是其故意泄露的，也间接确认了新操作系统的名称是 Windows 11。</p><p>IT之家获悉，被泄露的 ISO 镜像是 5 月份 Windows 11 的早期构建版本，并不是将在今年 10 月晚些时候作为免费更新发布的完整操作系统。</p>
