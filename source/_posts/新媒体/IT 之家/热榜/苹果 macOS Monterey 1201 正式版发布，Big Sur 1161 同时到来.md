Monterey 带来多项开创性功能，助力用户以全新方式沟通交流、完成更多任务、使用多台 Apple 设备顺畅开展工作。macOS Monterey 今日起作为免费软件更新发布，供搭载 Apple 芯片和基于 Intel 芯片的 Mac 下载。</p><p data-vmark="9599"><img data-vmark="a871">通过 Facetime 通话沟通联系的新方式</h2><p data-vmark="0432">FaceTime 通话的音质更加自然逼真，同播共享功能助用户以全新方式和亲友增进感情、同享美好时光。依托空间音频技术，FaceTime 通话的语音听起来如同来自发言者在屏幕上所处的方位。两种新的麦克风模式让用户进一步控制自己的语音传递效果：“语音突显”模式可消除背景噪声，确保用户的声音清晰通透；“宽谱”模式则可传递所有环境音，让对方听到一切。FaceTime 通话新增的人像模式利用 M1 芯片的神经网络引擎虚化背景，让焦点落在人物身上。人像模式同样适用于 Webex 与 Zoom 等第三方 App 中。</p><p data-vmark="38ea">同播共享包括一组强大功能，让用户在 FaceTime 通话中共享精彩体验。用户可以和亲友实时同步欣赏音乐、电视剧、电影等更多内容。同播共享的参与者可以通过共享播放控制来控制播放、暂停或快进。用户可以共享整个屏幕上的内容，也可以仅共享某个 App，轻松通过 FaceTime 通话与好友或同事展开协作。同播共享适用于 Apple Music 以及多个其它第三方 App。</p><p data-vmark="487f"><img src="https://img.ithome.com/newsuploadfiles/2021/10/c9532a52-8a39-4ee4-9802-69bff88e1af0.jpg@s_2,w_820,h_533" w="980" h="637" title="苹果 macOS Monterey 12.0.1 正式版发布，Big Sur 11.6.1 同时到来" srcset="https://img.ithome.com/newsuploadfiles/2021/10/c9532a52-8a39-4ee4-9802-69bff88e1af0.jpg 2x" width="980" height="533" referrerpolicy="no-referrer"></p><h2 data-vmark="e327">使用通用控制和隔空播放到 Mac 跨 Apple 设备工作（未发布）</h2><p data-vmark="54aa">通用控制功能让用户只使用一套鼠标和键盘就可在 Mac 和 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 中无缝切换。用户只需将两台设备靠近，就能立刻将指针从一台设备移到另一台，且无需进行设置。用户可以使用 Mac 键盘在任意一台设备上输入文本，或者在两台设备间来回拖放内容，还可以用 Apple Pencil 在 iPad 上画好图后，将图片直接拖放至 Mac 的 App 内，同时利用两台设备的强大功能。借助隔空播放到 Mac 功能，用户可以轻松在 Mac 上播放、展示和共享来自 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 或 iPad 的内容。使用隔空播放，无论是最新电影、游戏、照片还是演示文稿，都可共享至 Mac 令人惊艳的视网膜显示屏上；用户也可以在房间内的任意地方通过 Mac 的高保真音响系统播放音乐。</p><p data-vmark="ee0e"><img src="https://img.ithome.com/newsuploadfiles/2021/10/fd03c3b2-1c97-4cf9-af46-24901013be84.gif" w="340" h="246" title="苹果 macOS Monterey 12.0.1 正式版发布，Big Sur 11.6.1 同时到来" width="340" height="246" referrerpolicy="no-referrer"></p><h2 data-vmark="74f9">实况文本</h2><p data-vmark="734b">实况文本可以检测照片中的文本，包括电话号码、网页、地址等等，方便用户复制和粘贴、呼叫号码、打开网页和查找更多信息。实况文本可在 macOS 系统各处使用，依托强大的设备端机器学习发挥作用，保护用户数据隐私。</p><p data-vmark="31bf"><img src="https://img.ithome.com/newsuploadfiles/2021/10/1e434cc0-7dc4-4572-aea0-4081ad76bb90.jpg@s_2,w_820,h_533" w="980" h="637" title="苹果 macOS Monterey 12.0.1 正式版发布，Big Sur 11.6.1 同时到来" srcset="https://img.ithome.com/newsuploadfiles/2021/10/1e434cc0-7dc4-4572-aea0-4081ad76bb90.jpg 2x" width="980" height="533" referrerpolicy="no-referrer"></p><h2 data-vmark="dedd">专注模式帮助用户全心投入，避免分心</h2><p data-vmark="c91d">专注模式可根据用户当下希望专注投入的内容，筛选来自 App 和联系人的通知。用户在 Mac 上设置专注模式后，其他 Apple 设备也将自动同步此模式。使用专注模式时，信息 App 和 Slack 等支持此功能的第三方通讯 App 会自动显示专注状态，让其他人明白用户正在全心工作或享受假期，此时不宜打扰。</p><p data-vmark="097d"><img src="https://img.ithome.com/newsuploadfiles/2021/10/e5e19c20-783d-4ac5-b698-392d3679446b.jpg" w="653" h="653" title="苹果 macOS Monterey 12.0.1 正式版发布，Big Sur 11.6.1 同时到来" width="653" height="653" referrerpolicy="no-referrer"></p><h2 data-vmark="2924">备忘录带来管理与协作的更多方式</h2><p data-vmark="26a4">Monterey 的备忘录 App 为用户提供全新方式轻松整理、优化协作，并随时随地创建新的备忘录。快速备忘录让用户在任何 App 内都能记录笔记，确保在灵感到来时及时轻松捕捉想法。用户还可以将 App 内链接添加到快速备忘录，便于在使用 Safari <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>时记录网址，或在使用地图 App 时查找地址。新的协作工具帮助用户添加提及对象，并在全新“活动”视图中查看共享备忘录的更新内容。在 macOS Monterey 中可以使用标签对备忘录进行分类，以便快速对备忘录进行分类，并在全新标签浏览器中轻松查找备忘录。</p><p data-vmark="fb99"><img src="https://img.ithome.com/newsuploadfiles/2021/10/d043c9de-33e6-4382-9604-873a12599ae2.jpg" w="653" h="522" title="苹果 macOS Monterey 12.0.1 正式版发布，Big Sur 11.6.1 同时到来" width="653" height="522" referrerpolicy="no-referrer"></p><h2 data-vmark="e82b">Safari 浏览器提升标签页体验</h2><p data-vmark="3373">Safari 浏览器帮助用户进一步掌控浏览网络的方式，包括提供全新标签页组功能，让用户轻松保存和管理多个标签页，十分适用于用户计划旅行、进行项目研究，或者保存常用标签页。标签页组还可在 Mac、iPhone 和 iPad 上同步，用户可以随时随地继续完成项目，并且轻松与亲朋好友共享标签。Safari 浏览器还带来体验网络的新方式，包括新增的可选紧凑标签页栏设计，为用户浏览网页留出更多空间；集成“与您共享”功能，让用户轻松访问通过信息 App 发来的链接；支持快速备忘录功能，方便用户及时记录并跟踪在网站上发现的信息。Safari 浏览器还提供了更强大的隐私保护，通过智能防跟踪功能，阻止跟踪者通过用户 IP 地址来建立用户的信息档案。</p><p data-vmark="b012"><img src="https://img.ithome.com/newsuploadfiles/2021/10/eb55c72f-1e01-4ae1-93c4-5c9b6e5eaa5b.jpg@s_2,w_820,h_533" w="980" h="637" title="苹果 macOS Monterey 12.0.1 正式版发布，Big Sur 11.6.1 同时到来" srcset="https://img.ithome.com/newsuploadfiles/2021/10/eb55c72f-1e01-4ae1-93c4-5c9b6e5eaa5b.jpg 2x" width="980" height="533" referrerpolicy="no-referrer"></p><h2 data-vmark="c2bf">Mac 无缝集成快捷指令</h2><p data-vmark="36d3">Mac 现已支持快捷指令功能，帮助用户更方便地使用最常用的 App，最大化提升效率。用户可以通过丰富的预设快捷指令，轻松自动化日常任务，比如瞬间打开常用 App 让工作日高效开始，或者直接用 Mac 向购物清单添加提醒，在用户抵达商场时发出通知。快捷指令编辑器可帮助用户定制快捷指令，以适配自己的工作流。快捷指令集成在 macOS 系统各处，包括菜单栏、“访达”、“聚焦”和 Siri，方便用户随时调用。用户还可以将 Automator App 中的工作流转化为快捷指令，即刻运行。</p><p data-vmark="caba"><img src="https://img.ithome.com/newsuploadfiles/2021/10/71d16f16-e203-4e4b-8ee0-7a4eddbbab68.jpg@s_2,w_820,h_533" w="980" h="637" title="苹果 macOS Monterey 12.0.1 正式版发布，Big Sur 11.6.1 同时到来" srcset="https://img.ithome.com/newsuploadfiles/2021/10/71d16f16-e203-4e4b-8ee0-7a4eddbbab68.jpg 2x" width="980" height="533" referrerpolicy="no-referrer"></p><h2 data-vmark="66ad">其他功能</h2><p data-vmark="4475">“与您共享”让用户直接在照片 App、Safari 浏览器、Apple 播客等 App 中轻松欣赏通过信息 App 共享的精彩内容。用户可以轻松发现共享内容，看到是谁向自己推荐了它们，并在原对话内直接回复。</p><p data-vmark="0e94">地图 App 通过丰富的道路、街区、树木、建筑等城市细节带来全新沉浸式体验，通过 Mac 绚丽的显示屏生动呈现。</p><p data-vmark="9e20">iCloud+ 不仅集合了用户喜欢的 iCloud 功能，还推出了全新的高级功能，包括隐藏邮件地址和进一步拓展的 HomeKit 安防视频支持，无需额外费用。所有 iCloud+ 方案都可与家人共享群组成员共享，每位成员皆可享受服务提供的新功能、储存方案，以及升级体验。现有 iCloud 订阅用户将自动升级至 iCloud+，无需额外付费。</p><p data-vmark="5bb0">搭载 M1 芯片的 Mac 迎来空间音频功能，搭配 AirPods（第三代）、AirPods Pro 和 AirPods Max，为用户提供剧场般的聆听体验。动态头部追踪功能让用户在移动头部时，可将声音始终置于设备方位，即便转动头部也分毫不差。</p><p data-vmark="9911">邮件隐私保护功能可防止发件人获知用户是否打开了电子邮件，并隐藏用户的 IP 地址，让发件人无法获知用户位置信息，或利用此信息建立用户档案。</p><p data-vmark="58a8">新的辅助功能选项让任何人都可直接用“标记 " 添加用于旁白功能的图像描述，改进后的全键盘控制和新的自定义光标选项让用户在导览 Mac 时享有更高的灵活性。</p>
