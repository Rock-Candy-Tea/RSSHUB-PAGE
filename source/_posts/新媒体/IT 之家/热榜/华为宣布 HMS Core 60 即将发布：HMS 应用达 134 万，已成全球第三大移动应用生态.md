<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 10 日消息 据华为开发者联盟宣布，今日，HDD 北京站・暨 DIGIX 创新大赛全球发布会上，华为正式宣布 HMS Core 6.0 即将发布，进一步加码应用生态升级。</p><p>华为消费者云服务 HMS Core 平台表示，得益于 HMS Core 丰富的开放能力及完整的技术方案、以及全方位全流程的服务支持，全球集成 HMS Core 的应用达到 13.4 万，已成为全球第三大移动应用生态。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/2693b702-df4a-4fee-992c-814bf84f0c17.jpg" w="690" h="395" title="华为宣布 HMS Core 6.0 即将发布：HMS 应用达 13.4 万，已成全球第三大移动应用生态" width="690" height="395" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/15c7e88f-094b-4f33-989f-060ca9f2f5e2.jpg" w="690" h="385" title="华为宣布 HMS Core 6.0 即将发布：HMS 应用达 13.4 万，已成全球第三大移动应用生态" width="690" height="385" referrerpolicy="no-referrer"></p><p>另外，2021 华为 HMS 全球应用创新大赛已全球启动，华为表示，未来的全场景智慧时代需要更多的创新，借助此次华为 HMS 全球应用创新大赛，华为希望继续汇聚全球开发者力量，激发更大的创新，创造更多的美好，共筑 HMS 全场景智慧生态。</p><p>2021 华为 HMS 全球应用创新大赛特设科技女性奖，以鼓励优秀的女性开发者及团队。</p><p>华为云 CloudIDE 支持终端云后端服务开发。华为终端云所有业务都构建在华为云基础设施之上，华为云提供高效的资源底座，帮助终端云触达全球 170 多个国家和地区的开发者。这两朵云在能力开放方面深度协同，为开发者提供统一线上开发体验和 DevOps 开发平台，以及统一的应用分发及运营服务，提升开发者的开发体验和质量效能。</p><p>IT之家获悉，去年 8 月，华为 HMS Core 5.0 面向全球开发者上线，HMS Core 5.0 能力开放覆盖七大领域的服务，包括应用服务（App Services）、图形（Graphics）、媒体（Media）、人工智能（AI）、智能终端（Smart Device）、安全（Security）、系统（System），更全面地开放华为“芯 - 端 - 云”能力。</p>
