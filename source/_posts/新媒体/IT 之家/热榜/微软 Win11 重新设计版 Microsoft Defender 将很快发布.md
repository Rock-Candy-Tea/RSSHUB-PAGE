<p data-vmark="0355"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 8 日消息，微软一直在为 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>、<a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a>、macOS、<a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 和 iOS 开发跨平台的 Microsoft Defender 客户端。这款基于 Web 组件（WebView）的改造后的应用程序将帮助消费者掌握防病毒、恶意软件扫描、网络钓鱼、泄露密码等信息。</p><p style="text-align: center;" data-vmark="b2fc"><img src="https://img.ithome.com/newsuploadfiles/2022/2/366e8c8c-d638-4c68-a60b-648f25e315f9.jpg" w="696" h="392" title="微软 Win11 重新设计版 Microsoft Defender 将很快发布" width="696" height="392" referrerpolicy="no-referrer"></p><p data-vmark="b00f">该应用的官方商店描述称，Microsoft Defender 是一款让你安心的安全应用。微软表示，新的应用程序带有个性化的仪表板，你可以在一个地方查看你的 Windows 设备和其他连接设备的安全状态。</p><p data-vmark="05d8">微软似乎已经抢先一步，提前发布了 Windows Defender 的早期预览版，并有效地确认了一旦该安全应用程序到达 Windows Insider 计划--很可能在未来几周内--我们可以期待什么。</p><p style="text-align: center;" data-vmark="3725"><img src="https://img.ithome.com/newsuploadfiles/2022/2/0c7e1135-3673-4ead-8926-d84722c65a75.jpg@s_2,w_820,h_547" w="1137" h="759" title="微软 Win11 重新设计版 Microsoft Defender 将很快发布" srcset="https://img.ithome.com/newsuploadfiles/2022/2/0c7e1135-3673-4ead-8926-d84722c65a75.jpg 2x" width="1137" height="547" referrerpolicy="no-referrer"></p><p data-vmark="3440">根据测试，Microsoft Defender 预览版在今天早些时候进行了更新，支持使用个人微软账户登录。目前，Microsoft Defender Preview 在美国以外的地区无法使用，而且大部分功能目前都无法使用。</p><p data-vmark="4a5a">根据代码参考，我们知道 Microsoft Defender 预览版应用程序的代号为 Gibraltar，它已经为微软员工提供了一段时间。</p><p style="text-align: center;" data-vmark="ae7f"><img src="https://img.ithome.com/newsuploadfiles/2022/2/27870da6-34b3-44ec-a9aa-a8cd8c95f3e2.jpg" w="672" h="420" title="微软 Win11 重新设计版 Microsoft Defender 将很快发布" width="672" height="420" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="3c20"><img src="https://img.ithome.com/newsuploadfiles/2022/2/6af422a3-ea38-4682-91b9-9730101c1018.jpg" w="691" h="420" title="微软 Win11 重新设计版 Microsoft Defender 将很快发布" width="691" height="420" referrerpolicy="no-referrer"></p><p data-vmark="8fe5">我们还做了一些调查，发现有迹象表明这个新的应用程序不会取代 Windows 中现有的 Microsoft Defender 解决方案。与现有的 Defender 客户端不同，这个新的应用程序将只涵盖各种基本的安全威胁，它应该是作为一个仪表板，而不是恶意软件清除工具。</p><p data-vmark="2ee1">正如我们一开始提到的，Microsoft Defender 预览版可以让你用任何账户登录，因此将有可能把家庭成员添加到仪表板上，并获得所有连接设备的安全概述。你将能够使用二维码邀请家庭成员，用户将被要求安装移动或桌面应用程序。</p><p data-vmark="31bc">当然，如果你的连接设备运行 Windows，安全仪表板会更全面。对于其余的设备，你可以监控密码泄露、恶意软件警报、盗窃问题等警报通知。</p><p data-vmark="e088">Microsoft Defender 预览版还没有正式发布日期，但它将在几周或几个月后登陆 Windows 10 和 Windows 11 内测预览版。</p>
