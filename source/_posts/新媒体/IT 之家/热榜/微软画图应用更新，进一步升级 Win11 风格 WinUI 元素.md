<div class="tougao-user">感谢IT之家网友 <a href="https://m.ithome.com/html/app/open.html?url=ithome%3A%2F%2Fuserpage%3Fid%3D1065563" rel="nofollow">Eggiverse</a> 的线索投递！</div>
            <p data-vmark="3b9e"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 19 日消息，去年 12 月，微软测试了新的画图应用程序。新版应用在“编辑颜色”和“调整大小和倾斜”对话框中获得了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 风格的设计主题。</p><p data-vmark="f6b1">现在在最新的预览版 Windows 11 中，微软进一步优化了画图的整体界面，将更多界面和设计升级到了 WinUI 风格，尤其是圆角和 Mica 之类的设计。</p><p data-vmark="9d21">最新版的画图主要是 UI 视觉上的更新，但其中一部分仍依赖于 Windows 8 时代的代码。微软确认，并不是每个对话框都得到更新，只有重要的对话框，如“取色器”或“调整大小”工具仍然使用旧的设计，没有升级 Fluent 和 WinUI 3.0。</p><p data-vmark="4b6c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/d4b35b23-d845-49b2-9625-22e1891113cb.jpg@s_2,w_820,h_430" w="1440" h="756" title="微软画图应用更新，进一步升级 Win11 风格 WinUI 元素" srcset="https://img.ithome.com/newsuploadfiles/2022/3/d4b35b23-d845-49b2-9625-22e1891113cb.jpg 2x" width="1440" height="430" referrerpolicy="no-referrer"></p><p data-vmark="f849">不过，最新的画图在测试通道中进一步启用了 WinUI 元素。除了“编辑颜色”、“调整大小和倾斜”外还有其他对话框，都更新了 WinUI 以匹配 Windows 11 的设计风格。</p><p data-vmark="c96f">此外，你还可以在选色板上通过按住 Shift 键来选取某种颜色作为你的第二选色。</p><p data-vmark="0130" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/39af085b-b83c-46bf-aaf1-d438b71aa95b.png" w="913" h="901" title="微软画图应用更新，进一步升级 Win11 风格 WinUI 元素" width="913" height="809" referrerpolicy="no-referrer"></p><p data-vmark="3e6f">新版本还修复了多个 bug。例如，修复了在使用 IME 时文本框意外移动的问题，还有在某些区域语言中对话框无法显示本地化语言的问题。</p><p data-vmark="1f57" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2021/12/0af08e37-2198-46f5-ab5f-b3dd684c962f.png" title="微软画图应用更新，进一步升级 Win11 风格 WinUI 元素" referrerpolicy="no-referrer"></p><p data-vmark="5014" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2021/12/95a23064-565d-4d79-9087-3ceff5270807.png" alt="New Resize and Skew Dialog in Paint." title="微软画图应用更新，进一步升级 Win11 风格 WinUI 元素" referrerpolicy="no-referrer"></p><p data-vmark="3e7e">最后，新版画图还更新了对屏幕阅读器的支持。</p><p data-vmark="37f9">IT之家了解到，新的画图应用将在今年晚些时候通过稳定频道向 Windows 11 正式版用户推出（未链接到任何内部程序)，至少会在今年晚些时候预装在 Windows 11 22H2 版本中。</p>
