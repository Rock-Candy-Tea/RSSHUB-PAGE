            <p data-vmark="73a8"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 13 日消息，微软近期宣布<span class="accentTextColor">推出 Windows 预览体验计划的全新图标</span>。微软表示，新图标代表热爱并通过 Dev、Beta 和 Release 预览 3 个内部通道帮助塑造 Windows 未来的用户们。</p><p data-vmark="51b4" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/4fc8bf34-bde9-4692-af75-9a6bdcf29c9c.png" w="752" h="434" title="微软公布 Win11 预览体验计划全新图标：代表帮助塑造 Windows 系统未来的用户们" width="752" height="434" referrerpolicy="no-referrer"></p><p data-vmark="598f">IT之家了解到，Dev 通道的 Insider 成员很快就会在“设置> Windows 更新 > Windows 预览体验计划”下看到该图标。新图标也将出现在电子邮件和其他区域中。</p><p data-vmark="474d">此前，微软还公布了最新的 Windows 预览体验计划变动。Dev 通道将成为实验性功能出现的地方，而 Beta 通道则包含更接近所有用户的功能。这可能意味着新功能如果接近公布的话，甚至会首先出现在 Beta 通道中。</p><p data-vmark="21b8">根据微软的描述，微软还计划为 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 测试人员提供一个选项，可以从 Dev 通道切换到 Beta 通道，来避免更多正在进行的实验性功能。</p>
