<p data-vmark="61a1">微软在 10 月 5 日发布了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 正式版，带来了大量外观方面的改进，例如右键快捷菜单，就和旧款完全不同。</p><p data-vmark="f766">Win11 的新右键菜单使用了 Fluent Design 设计，颜值更高，布局也作了大幅调整，将一些常用的功能，隐藏了起来，需要点击“显示更多选项”才能找到。</p><p data-vmark="3ae1"><img src="https://img.ithome.com/newsuploadfiles/2021/10/85450001-4d80-4856-9be1-9df4c9d85b40.jpg" w="411" h="383" title="完美契合 Win11 右键菜单，这款压缩软件推荐给你" width="411" height="383" referrerpolicy="no-referrer"></p><p data-vmark="1c7e">和 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 相比，Win11 的右键菜单更加简洁，但却在很多场合中使用不便。例如安装了压缩软件后，呼出右键菜单后，就需要多点击一次“显示更多选项”才能找到压缩相关选项，不像 Win10 那样直接在右键菜单就能操作。</p><p data-vmark="675a">怎么办？其实这很大程度上是因为当前的压缩软件，并没有适配 Win11 造成的。适配了 Win11 后，在右键菜单也可以直接找到压缩相关选项！例如这款“NanaZip”，就可以解决相关问题。</p><p data-vmark="fb28">NanaZip：<span class="link-text-start-with-http">https://www.microsoft.com/zh-cn/p/nanazip/9n8g7tscl18r</span></p><p data-vmark="71df">NanaZip 可以直接通过微软商店安装，它基于大名鼎鼎的“7-Zip”，目前在 Github 上开源。如果你是开发者，也可以点击下面的链接进入 NanaZip 的 Github 项目主页。</p><p data-vmark="a9dd">NanaZip Github 主页：<span class="link-text-start-with-http">https://github.com/M2Team/NanaZip</span></p><p data-vmark="d453">由于 NanaZip 基于 7-Zip，因此功能和性能都和 7-Zip 差不多，甚至连界面也和 7-Zip 一个简陋风格，除了提供最基础的交互，没有什么精美的设计。</p><p data-vmark="8462"><img src="https://img.ithome.com/newsuploadfiles/2021/10/00a98f89-81b3-42ee-bbd8-084ee9409bb1.png" w="700" h="394" title="完美契合 Win11 右键菜单，这款压缩软件推荐给你" width="700" height="394" referrerpolicy="no-referrer"></p><p data-vmark="b5f9">不过，NanaZip 的亮点在于对 Win11 的适配。安装了 NanaZip 后，即可设置将其中的功能集成到 Win11 的右键菜单当中。接着，右键点击某个文件或者文件夹，就可以看到 NanaZip 的二级菜单，展开压缩相关的各种功能了。</p><p data-vmark="1ae3"><img src="https://img.ithome.com/newsuploadfiles/2021/10/ff86b223-24ca-45c4-a441-4e9714e892ef.png" w="700" h="394" title="完美契合 Win11 右键菜单，这款压缩软件推荐给你" width="700" height="394" referrerpolicy="no-referrer"></p><p data-vmark="cd18">总的来说，这是一款率先适配 Win11 的压缩工具。随着 Win11 的普及，未来应该还会有更多软件会对 Win11 作适配。如果你想要在 Win11 中获得更方便的压缩体验，不妨尝试一下 NanaZip 吧。</p>
