<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 8 日消息 外媒 MSPoweruser 报道，当第一个 <a target="_blank" href="https://win11.ithome.com/" class="s_tag">Windows 11</a> ISO 镜像泄露时，该操作系统被认为是 <a target="_blank" href="https://win10.ithome.com/" class="s_tag">Windows 10</a> 的一个令人失望的更新，除了新的“开始”菜单外，其他与 Windows 10 相比基本没有变化。</p><p>然而，当微软正式发布首个 Windows 11 Insider 预览版发布时，该操作系统得到了重大的视觉更新，改进了操作和通知中心，重新设计了现代设置，新的微软商店，更多的圆角 UI，还有新的文件资源管理器等等。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/ee9e3370-bf05-47d7-9833-02dd66217d61.jpg" w="680" h="383" title="网友吐槽 Win11 预览版 22000 部分界面无圆角，无新文件资源管理器，微软回应：会推出越来越多功能和调整" width="680" height="383" referrerpolicy="no-referrer"></p><p>然而，Windows 11 操作系统的一些界面设计仍然有些粗糙，但好消息是微软表示，仍有大量的开发工作要进行。</p><p>今天，Windows Dev 推特账户证实，在谈到 Windows 11 的功能时，“还有很多东西要做”。</p><p>有网友吐槽，没有累积更新补丁的 Windows 11 预览版 22000 ，没有圆角，没有新的文件资源管理器，没有新的操作中心，没有新的设置，仍然是 Windows 10 的登录屏幕。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/daebd131-4517-43b9-bf99-b8a30e2cf140.png" w="688" h="739" title="网友吐槽 Win11 预览版 22000 部分界面无圆角，无新文件资源管理器，微软回应：会推出越来越多功能和调整" width="688" height="739" referrerpolicy="no-referrer"></p><p>微软 Windows Developer 回应称，在接下来的几个月里，我们将推出越来越多的 Windows 11 Insider build 功能，所以我们可以尝试正确地调整每个功能。</p><p>该网友继续表示，我只是表明有多少新的东西是随着累积更新补丁发布的。微软 Windows Developer 回应称，我们知道了！只是想让你知道，还有更多的东西要来。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/5ef421b9-4d44-4b41-955b-a00eabf8aedc.png" w="687" h="397" title="网友吐槽 Win11 预览版 22000 部分界面无圆角，无新文件资源管理器，微软回应：会推出越来越多功能和调整" width="687" height="397" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/d2161b86-deee-42c0-964c-9f67924ae72e.jpg" 更加现代化，整洁且美观。无论是全新的“开始”键和任务栏，还是每一种声音、字体和图标，这一切都经过精心设计，让你可以自由掌控。微软把“开始”键放在中心位置，让你更容易地快速找到所需的内容。凭借云和 Microsoft 365 的强大性能，“开始”菜单会向你显示最近文档，无论你之前在哪个平台或设备上查看过，即使是在 <a target="_blank" href="https://android.ithome.com/" class="s_tag">Android</a> 或 iOS 设备上浏览过这些文档，“开始”菜单都能让你一览无余。</p>
