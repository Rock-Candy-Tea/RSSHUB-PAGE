<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 16 日消息  微软即将推出的 <a href="https://www.ithome.com/0/557/468.htm" target="_blank">Windows 11 操作系统镜像今天在网上泄露</a>了，新的操作系统采用了全新的用户界面、开始菜单、任务栏，以及更多更现代化的界面。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/d330b277-538a-4599-a969-ebd2d95a44cc.jpg@s_2,w_820,h_547" w="1200" h="800" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" srcset="https://img.ithome.com/newsuploadfiles/2021/6/d330b277-538a-4599-a969-ebd2d95a44cc.jpg 2x" width="1200" height="547" referrerpolicy="no-referrer"></p><p>新的 Windows 11 用户界面和开始菜单看起来与 Windows 10X 中的非常相似，<span class="accentTextColor">微软已经取消了 Windows 10X，其部分界面、功能已经移植到了 Windows 11 中。</span></p><h2>任务栏</h2><p>从视觉上看，最大的变化在任务栏上，微软将应用图标放在了任务栏的中心位置，清理了托盘区域，并采用了一个新的开始按钮和菜单。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/4ec2715d-4ff0-433a-8f15-0de5b563a03e.jpg" w="700" h="560" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" width="700" height="560" referrerpolicy="no-referrer"></p><h2>开始菜单</h2><p>Windows 11 中新的“开始”菜单是 Windows 10 中的一个简化版本，没有动态磁贴，里面有钉住的应用程序、最近的文件，以及快速关闭或重新启动功能，确实比 Windows 10 中的简化了很多。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/f58e5e5b-fbba-4b26-9134-443bdd4fde77.jpg@s_2,w_820,h_547" w="1200" h="800" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" srcset="https://img.ithome.com/newsuploadfiles/2021/6/f58e5e5b-fbba-4b26-9134-443bdd4fde77.jpg 2x" width="1200" height="547" referrerpolicy="no-referrer"></p><p><span class="accentTextColor">如果用户不希望应用图标和开始菜单居中，有一个选项可以将它们全部移回左侧</span>。再加上同样可用的黑暗模式，Windows 11 开始看起来像 Windows 10 的一个更精致的版本，而不是一个全新的东西。</p><h2>圆角设计</h2><p>微软还在整个 Windows 11 中大量使用圆角设计。这些在上下文菜单、应用程序和文件资源管理器中都可以看见。开始菜单本身也采用了圆角设计。由于目前这仍然是一个泄露的早期版本的 Windows 11，所以并非所有设计都已完成。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/002b6f9e-814f-41a3-8ded-ecf45cd47105.png" w="1280" h="1024" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" width="1280" height="656" referrerpolicy="no-referrer"></p><p>▲Windows 11 的搜索界面</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/6c83a7ca-5a5e-476d-92a6-ffe86bf98972.png" w="1280" h="1024" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" width="1280" height="656" referrerpolicy="no-referrer"></p><p>▲Windows 11 中的文件资源管理器</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/f0369f90-6c36-42de-9ee1-d71313195e4d.jpg" w="700" h="476" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" width="700" height="476" referrerpolicy="no-referrer"></p><p>▲右键菜单</p><h2>小工具</h2><p>Windows 11 任务栏中有一个新图标，被命名为 Widgets（插件）。有传言说，微软正在带回 Windows 小工具，这个早期 Windows 11 版本似乎包含了小工具。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/567d0051-3a37-4d51-b1b9-a2a55f3fbfd2.jpg@s_2,w_820,h_655" w="1200" h="959" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" srcset="https://img.ithome.com/newsuploadfiles/2021/6/567d0051-3a37-4d51-b1b9-a2a55f3fbfd2.jpg 2x" width="1200" height="655" referrerpolicy="no-referrer"></p><p>由于这显然是 Windows 11 的一个非常早期的版本，<span class="accentTextColor">这些小工具并没有在操作系统中完全加载，它们被设计成滑出</span>，并提供对新闻、天气和其他网络内容的快速访问。</p><p>虽然微软取消了其 Windows 10X 操作系统，但微软显然将该系统的很多特色功能放到了 Windows 11 中。Windows 10X 最初是为双屏设备设计的，后来转向传统笔记本电脑，然后被取消。Windows 10X 包括对 Windows 的一些改进和简化，而这些工作的大部分都存在于这个泄露的 Windows 11 版本中。</p><h2>快速布局应用窗口</h2><p>Windows 11 还包括新的快速布局应用窗口功能，<span class="accentTextColor">可以使用所有应用程序都有的最大化按钮来实现（如下方的动图）</span>，用户可以快速地将窗口放在左侧、右侧，或者四个角落，或在桌面上将它们排列成若干部分。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/c99ed2fb-e109-44f9-9e07-7c60c9c602c8.gif" w="600" h="479" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" width="600" height="479" referrerpolicy="no-referrer"></p><h2>Windows 商店</h2><p>遗憾的是，Windows 商店似乎还没有大的更新。由于这是一个早期泄露的版本，很可能只是还没有给予提供。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/e9deabcf-f680-4021-9a11-948a8a2651b9.jpg@s_2,w_820,h_656" w="1200" h="960" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" srcset="https://img.ithome.com/newsuploadfiles/2021/6/e9deabcf-f680-4021-9a11-948a8a2651b9.jpg 2x" width="1200" height="656" referrerpolicy="no-referrer"></p><p>Windows 商店的界面与目前 Windows 10 中存在的界面基本相同，可以使用目前可用的所有相同的应用程序。</p><p>据报道，最近几个月，微软一直在为 Windows 开发一个新的应用商店，将有很大的改变。微软首席执行官萨提亚 - 纳德拉（Satya Nadella）承诺，将通过 Windows“为开发者和创造者释放更大的经济机会”，而这很可能会以新商店的形式出现。据报道，微软正在全面改革其 Windows 应用商店，允许开发者提交任何 Windows 应用，包括 Chrome 或 Firefox 等浏览器。微软可能还在考虑允许应用程序中的第三方商务平台，让开发者避开微软通常从应用内购系统中获取的分成。</p><h2>文件管理器</h2><p>Windows 11 内置的文件管理器改动幅度并不大，改善了对触控的支持，升级了默认的文件类型图标，窗口和导航按钮也有圆角。不过，文件资源管理器的属性标签不支持深色模式。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/36c3c979-002a-4699-9080-3487a29d84f0.jpg" alt="i0a747l3.jpg" w="700" h="487" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" width="700" height="487" referrerpolicy="no-referrer"></p><h2>其它方面</h2><p>微软还在 Windows 11 中加入了一个新的初始化设置体验，与 Windows 10X 非常相似，通过一系列步骤引导新硬件用户或升级到 Windows 11 的用户来配置 Windows。<span class="accentTextColor">新体验还包括一个新的 Windows 11 启动声音，会在每次启动时触发</span>。</p><p>微软还在改进 Windows 11 中的 Xbox 体验。新的 Xbox 应用程序现在集成到 Windows 11 中，提供快速访问 Xbox Game Pass 游戏、Xbox 网络的社交部分和 Xbox 商店。在这个早期泄露的版本中，Xbox 游戏栏和 Windows 游戏模式都与 Windows 10 保持一致。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/b7bf34a7-1b69-461e-8bd5-3477c80206e7.png" w="1280" h="1024" title="Win11 上手初体验：任务栏和开始菜单等迎来大改" width="1280" height="656" referrerpolicy="no-referrer"></p><p><img srcset="https://img.ithome.com/newsuploadfiles/2021/6/868b0519-4737-400b-a326-e7aa0a83cfe9.jpg 2x" width="1280" height="656" referrerpolicy="no-referrer"></p><p>Windows 11 上的新 Xbox 应用。</p><p>微软一直在暗示已经准备好推出 Windows 11，这家软件巨头将在 6 月 24 日举行一次特别的 Windows 活动，以揭示其下一代操作系统。</p><p>附外媒 Windows Central 上手体验视频：</p><p><a class="ithome_super_player" contenteditable="false" target="_blank"
