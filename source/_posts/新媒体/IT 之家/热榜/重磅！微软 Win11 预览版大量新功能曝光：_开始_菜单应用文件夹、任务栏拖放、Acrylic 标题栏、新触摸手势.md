<p data-vmark="2a9a"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 15 日消息，微软正准备在 Dev 开发频道为 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Insiders 发布一些新的功能。有消息称，有许多最初发布版本中没有的功能将被重新引入 Windows 11 系统。自上一次 Windows 11 Dev 版本发布以来，已经过去了两个星期，这让一些人怀疑微软内部是否发生了什么事情，导致版本发布的暂停。</p><p data-vmark="46e3" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/340108dc-1d54-4e9c-925c-95bef1ae6eda.jpg" w="760" h="428" title="重磅！微软 Win11 预览版大量新功能曝光：“开始”菜单应用文件夹、任务栏拖放、Acrylic 标题栏、新触摸手势..." width="760" height="428" referrerpolicy="no-referrer"></p><p data-vmark="7f94">然而，有消息称，微软一直在利用额外的一周时间为 Windows 11 的公共预览版分支（rs_prerelease）添加全新的功能并实施一些改变。这是最近缺乏新的 Dev 开发构建版本的一个可能原因。然而，同样值得注意的是，构建版本号出现了巨大的跳跃（高达 Build 250xx），这表明一个新的开发周期正在开始。</p><p data-vmark="17e5">其中一些变化包括以下内容：</p><ul class=" list-paddingleft-2"><li><p data-vmark="7c1c">在“开始”菜单上创建应用程序文件夹</p></li><li><p data-vmark="40c0">在任务栏上支持拖放</p></li><li><p data-vmark="8368">传统标题栏的新 Acrylic 或模糊效果</p></li><li><p data-vmark="428c">新的 snapping UX 用户体验，称为“Snap Bar”</p></li><li><p data-vmark="0844">打开“开始”菜单的新手势</p></li><li><p data-vmark="ba24">对快速设置面板的触摸支持</p></li></ul><p data-vmark="8d0e">下面是一些功能解析或者介绍。</p><h3 data-vmark="b46d">“开始”菜单中的应用程序文件夹</h3><p data-vmark="28ca">Windows 11 用户高度期盼这个即将到来的功能，它的功能正如你所期望的那样。当你点击并将一个应用程序拖到另一个上时，这将创建一个新的文件夹，你可以点击。这项新功能允许用户在他们的“开始”菜单上有更多的组织。</p><h3 data-vmark="035f">Snap Bar</h3><p data-vmark="5fd7">新的 Snap Bar 功能将引入一种新的机制，用于在 Windows 11 的内置移动网格系统中占据多个应用程序窗口。目前，有两种不同的方式来抓取应用程序窗口：当你将鼠标悬停在窗口控件上时出现的下拉式，或将一个窗口拖到屏幕的角落里。随着新的 Snap Bar 功能的出现，将有第三个选择。具体来说，当你在屏幕上点击并拖动一个窗口时，用户将看到一个条形面板出现在屏幕的顶端部分。</p><h3 data-vmark="54c1">新的 Acrylic / 模糊效果</h3><p data-vmark="3598">你会发现传统的标题栏有轻微的 UI 变化，因为新的 Acrylic 或模糊效果将取代我们熟悉的白色标题栏设计。如果你在一月份观看了 Windows 直播，你应该对这个新功能的外观有一个概念，尽管它的质量不高。</p><p data-vmark="db75" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/b3d9dd99-f9b0-4ef0-a784-e3cabeaf9c30.png" w="1440" h="873" title="重磅！微软 Win11 预览版大量新功能曝光：“开始”菜单应用文件夹、任务栏拖放、Acrylic 标题栏、新触摸手势..." width="1440" height="497" referrerpolicy="no-referrer"></p><h3 data-vmark="f626">“开始”菜单和快速设置变化</h3><p data-vmark="36e2">平板电脑用户会很高兴，一些重大的改进也正通过新的手势到来。这些手势将允许通过从屏幕底部向上滑动来更容易地访问“开始”菜单和快速设置面板。如果你从系统托盘向上轻扫，这将启动快速设置。另一方面，在屏幕上其他地方向上轻扫将打开“开始”菜单。</p><h3 data-vmark="fead">任务栏功能</h3><p data-vmark="25bf">最后，Windows 11 用户将最终体验到任务栏中缺少的功能。这些功能包括支持拖放，更好的 UI，以及在使用设备的平板电脑模式时自动隐藏任务栏的能力。</p><p data-vmark="1e72">当然，我们极有可能错过了其他几个功能，或者微软将在一段时间内宣布更多的功能。以下是预计将在 Dev 开发频道中为内测人士提供的所有新功能：</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="e1a5">“开始”中的应用程序文件夹</p></li><li><p data-vmark="0935">任务栏上的拖放功能</p></li><li><p data-vmark="594d">快速设置 / 通知中心的改进</p></li><li><p data-vmark="538c">文件资源管理器中固定文件</p></li><li><p data-vmark="d2d4">Acrylic 标题栏</p></li><li><p data-vmark="e878">Snap Bar 的用户体验</p></li><li><p data-vmark="749c">使用触摸“开始”/快速设置的手势</p></li><li><p data-vmark="9b0f">新的实时字幕功能</p></li></ul><p data-vmark="0dd4">Windows 11 的下一个主要版本的代号为“太阳谷 2”，这些将是塑造它的功能。与往常一样，微软仍可能取消或推迟其中一些功能的出现，我们可以拭目以待。</p>
