<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 29 日消息 今天凌晨，微软面向 Dev 开发频道发布了第一个 Windows 11 Insider Preview 版本，即 Build 22000.51!  微软表示，随着我们在未来几个月内完成产品的定型，我们将与你一起验证这一体验。在这个早期预览中，<a href="https://www.ithome.com/0/559/232.htm" target="_blank">你将可以尝试上周展示的许多新功能</a>，但不是全部。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/bf37a4e2-09b8-42be-9e11-03a43144c98e.png" w="811" h="516" title="微软：Win11 首个预览版 22000.51 不需要 TPM 2.0 或特定 CPU" width="811" height="516" referrerpolicy="no-referrer"></p><p>在今天的一篇博文中，微软再次尝试澄清 Windows 11 的最低配置要求，表示 Windows 11 将需要英特尔第八代 Coffee Lake 或 AMD Zen 2 处理器及以上，但英特尔第七代 Kaby Lake 和 AMD Zen 1 用户还有一些希望。</p><p>微软表示，Windows 11 被设计和构建为一套完整的体验，释放客户所依赖的 PC 的全部力量，包括在安全、可靠性、兼容性、视频会议、多任务、游戏、创建、构建、学习等方面。微软称需要一个最低的系统要求，使得能够调整软件和硬件，以跟上人们的期望、需求，并利用 PC 的真正价值和力量，在现在和未来提供最佳体验。</p><p>但微软称今天推出的 Windows 11 的首个测试版本 22000.51 将不需要 TPM 2.0 或特定 CPU。（<a href="https://www.ithome.com/0/560/005.htm" target="_blank">不过仍有一大部分用户显示硬件过低无法 OTA 升级</a>）微软认为，通过向 Windows Insider 计划中的不同系统提供预览版，将更全面地了解 Windows 11 在不同 CPU 型号上的表现，为今后对最低系统要求的调整提供依据。</p><p>微软将格外关注使用英特尔第七代 CPU 和 AMD Zen 1 CPU 的用户，看看他们在可靠性和安全性方面是否获得微软所寻求的性能。</p><p>在博文中，微软声称，Windows 11 不会在此前的一些旧 CPU 上运行的原因是为了保持更高水平的安全性、稳定性和可靠性。</p><p>IT之家获悉，微软还将“暂时删除”PC 健康检查应用程序，该应用旨在告诉运行 Windows 10 的用户他们的电脑是否能够更新到 Windows 11。微软表示，该应用程序“没有充分准备好分享用户所期望的细节或准确性水平”，但其表示，该应用程序将在今年秋季（可能是 10 月份）Windows 11 正式版发布时将再次可用。</p><p>《<a href="https://www.ithome.com/0/559/243.htm" target="_blank">微软 Win11 运行硬件最低需求配置公布：1GHz 64 位处理器，4GB 内存 / 64GB 存储</a>》</p>
