            <p data-vmark="b55f"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 7 日消息，谷歌<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a> Chrome 97 正式版已经发布，新版本的变化大多与开发者有关，但日常用户也会注意到一些有趣的功能。</p><p data-vmark="d74f"><img src="https://img.ithome.com/newsuploadfiles/2022/1/d098b966-5da0-4b57-a295-e13fc7a9b5f1.png" w="1200" h="675" alt="Chrome 97" title="Chrome 97 浏览器正式版发布下载：遭苹果、火狐等抵制的功能上线" width="1200" height="461" referrerpolicy="no-referrer"></p><p data-vmark="fe82">之前的桌面版 Chrome 支持记住缩放设置，就是如果用户在某个网站上选择了缩放，那么在用户下次访问该网页时将保持这种状态。然而此前手机版并不支持这个功能，在 Chrome 97 中谷歌已经解决这个问题，不过需要启用该功能的 flag。</p><p data-vmark="f043">IT之家了解到，网站存储了我们的很多数据，但是在这之前，Chrome 浏览器只允许用户删除个别的 cookies。<span class="accentTextColor">Chrome 97 的一个重大隐私升级是能够删除网站在用户访问时存储的所有数据和信息</span>，位置在：设置 > 隐私和安全 > 网站设置 > 查看权限和跨网站存储的数据。</p><p data-vmark="6e6f">如果用户正在浏览一个有 HDR 视频的网站，网站现在可以检查用户的显示器是否支持 HDR。如果不支持，该网站就不需要用 HDR 显示。</p><p data-vmark="b655">网络应用程序现在可利用顶部的应用栏了，更像真正的、受支持的应用。</p><p data-vmark="aa5d">值得一提的是，此次更新还添加了一个备受争议的键盘 Map API（keyboard MAP API ），允许 Web 应用程序识别特殊的键盘布局。Mozilla、苹果、Brave 及其他浏览器的开发者们都对此事表示了担忧。这些公司反对浏览器集成的一个关键论点是：网站或可将该功能用于指纹识别目的。谷歌方面宣布将在 Chrome 97 浏览器中使用该 API，但许多其他基于 Chrome 的浏览器却表示不会支持该 API，或直接禁用该 API。</p>
