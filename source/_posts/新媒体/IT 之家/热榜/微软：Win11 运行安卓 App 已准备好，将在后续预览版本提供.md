<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 29 日消息 今天凌晨，面向 Dev 开发频道发布了第一个 Windows 11 Insider Preview 版本，即 Build 22000.51!  微软表示，随着我们在未来几个月内完成产品的定型，我们将与你一起验证这一体验。在这个早期预览中，<a href="https://www.ithome.com/0/559/232.htm" target="_blank">你将可以尝试上周展示的许多新功能</a>，但不是全部。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/4b3b2a9a-fc2a-43f5-b59c-96ef89f095cc.png" w="1440" h="960" title="微软：Win11 运行安卓 App 已准备好，将在后续预览版本提供" width="1440" height="547" referrerpolicy="no-referrer"></p><p>微软在博客中明确表示，在未来几个月里，我们将为你带来更多的功能，如与微软 Teams 聊天和微软商店中的安卓 App，因为它们已经准备好了 -- 我们在这个旅程中才刚刚开始。</p><p>6 月 24 日晚，微软举行了下一代 Windows 的发布会，<a href="https://www.ithome.com/0/559/232.htm" target="_blank">正式宣布名称为 Windows 11</a>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/d33e0817-856d-4677-8338-d642c2b2cef3.png" w="924" h="614" title="微软：Win11 运行安卓 App 已准备好，将在后续预览版本提供" width="924" height="545" referrerpolicy="no-referrer"></p><p>IT之家获悉，微软正在将安卓应用引入 Windows 11。安卓应用将在 Windows 11 上原生运行，并可从操作系统中包含的新 Windows 11 商店下载。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/129a39eb-28ac-4cdd-9d9d-64aa5afccb7f.png" w="922" h="523" title="微软：Win11 运行安卓 App 已准备好，将在后续预览版本提供" width="922" height="465" referrerpolicy="no-referrer"></p><p>通过使用亚马逊的应用商店，安卓应用将可以直接从新的 Windows 商店获得，并由英特尔 Bridge 技术驱动。Windows 11 上的安卓应用可以被固定在任务栏上，或与传统的 Windows 应用一起使用。</p><p>《<a href="https://www.ithome.com/0/559/246.htm" target="_blank">微软发布全新 Win11 应用商店：美观易用 UI，支持安卓 App</a>》</p>
