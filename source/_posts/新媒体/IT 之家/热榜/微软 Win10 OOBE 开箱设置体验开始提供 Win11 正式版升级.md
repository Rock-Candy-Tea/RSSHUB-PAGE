<p data-vmark="1152"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 9 日消息 据 Windows Latest 报道，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 正式版现在可用于支持的 PC，但只有一些用户在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 上检查更新时才能看到下载新操作系统的选项。这是因为 Windows 11 正在分阶段推出，预计推送工作将在明年夏天结束。</p><p data-vmark="1339"><img src="https://img.ithome.com/newsuploadfiles/2021/10/639264c7-a155-4b59-9897-82bd6a91215e.jpg" w="696" h="392" title="微软 Win10 OOBE 开箱设置体验开始提供 Win11 正式版升级" width="696" height="392" referrerpolicy="no-referrer"></p><p data-vmark="881f">根据支持文件，微软已经发布了一个名为“KB5005716”的 Windows 10 OOBE（开箱体验，初始设置体验）更新，以实现对 Windows 11 家庭版或专业版的新升级体验。然而，有一个问题是，这个过程需要用户重置设备或纯净安装 Windows 10。</p><p data-vmark="537a">事实证明，Windows 11 OOBE 升级过程是为运行最近的 Windows 10 版本 2004、20H2、21H1，甚至 21H2（预计本月晚些时候或下个月推出）的新设备设计的。当然，你需要互联网连接来下载 KB5005716 补丁更新。</p><p data-vmark="8f08">KB5005716 补丁适用于最近的 Windows 10 版本 2004 或更高版本，它只在 OOBE 屏幕上下载，这意味着你不能从微软更新目录中手动下载它。</p><p data-vmark="06f6"><img src="https://img.ithome.com/newsuploadfiles/2021/10/4392ef0d-e1af-4241-82fd-57b61a3fcf5d.jpg@s_2,w_820,h_343" w="1280" h="536" title="微软 Win10 OOBE 开箱设置体验开始提供 Win11 正式版升级" srcset="https://img.ithome.com/newsuploadfiles/2021/10/4392ef0d-e1af-4241-82fd-57b61a3fcf5d.jpg 2x" width="1280" height="343" referrerpolicy="no-referrer"></p><p data-vmark="2401">正如你在截图中所看到的，当你保持互联网连接并安装了 KB5005716 累积更新时，你就可以在 OOBE 中选择 Windows 10 和 Windows 11。</p><p data-vmark="c452">IT之家提醒，这种方法只在支持的 PC 设备上工作，它对 Surface Pro X 等产品特别有帮助。</p><p data-vmark="6652">这是因为 ARM CPU 上不支持 Windows 11 升级助手，使用 Media Creation Tool 工具生成的 ISO 镜像不兼容 ARM64，所以只有两种方法在 ARM PC 上安装新操作系统 ——Windows  Insider 计划和 OOBE（需要重置）。</p><p data-vmark="2cb9">因此，如果你重置任何支持的 PC 硬件，OOBE 体验将下载一个累积更新，其中包括“升级到 Windows 11”功能。这将允许设备在初始设置体验期间下载 Windows 11，设备将直接启动到桌面。</p><p data-vmark="e322">KB5005716 将应用于 Windows 10 OOBE 过程中，需满足以下条件：</p><ul class=" list-paddingleft-2"><li><p data-vmark="7a46">你的设备得到官方支持。</p></li><li><p data-vmark="7af9">在 OOBE 过程中，互联网连接是可用的。</p></li><li><p data-vmark="464b">你的 Windows 10 版本 2004 伴随 KB4586781（Build 19041.630）和可选更新 KB4580364（Build 19041.610）。如果你的版本是 20H2 或更高版本，不需要额外的更新。</p></li></ul><p data-vmark="2506">当然，Windows 11 OOBE 升级过程不一定对每个用户都有效，所以如果你的 PC 不是 ARM 系统，想要更快安装 Windows 11，可以立即下载 Windows 11 ISO 镜像并执行原地升级或纯净安装。</p>
