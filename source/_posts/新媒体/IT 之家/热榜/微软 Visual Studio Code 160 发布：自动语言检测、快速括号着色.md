<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 4 日消息 外媒 MSPoweruser 报道，微软最新发布了 Visual Studio Code v1.60（2021 年 8 月版）。本次更新带来了几个新功能，包括自动语言检测、括号着色、JavaScript/TypeScript 内嵌提示等。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/84dff400-eb5a-40f1-a3f2-90a6047a2049.png" w="906" h="568" title="微软 Visual Studio Code 1.60 发布：自动语言检测、快速括号着色..." width="906" height="514" referrerpolicy="no-referrer"></p><p><strong>下面是 Visual Studio Code 新功能：</strong></p><ul class=" list-paddingleft-2"><li><p><a href="https://code.visualstudio.com/updates/v1_60?WT.mc_id=devcloud-00000-cxa#_automatic-language-detection" target="_blank">自动语言检测</a> —— 当你粘贴到 VS Code 时会检测到编程语言。自动语言检测功能使用机器学习来猜测语言。</p></li><li><p><a href="https://code.visualstudio.com/updates/v1_60?WT.mc_id=devcloud-00000-cxa#_high-performance-bracket-pair-colorization" target="_blank">内置快速括号着色</a> —— 对大文件进行快速括号匹配和着色。</p></li><li><p><a href="https://code.visualstudio.com/updates/v1_60?WT.mc_id=devcloud-00000-cxa#_syntax-highlighting-of-codeblocks-in-the-settings-editor" target="_blank">设置编辑器语法高亮</a> —— 设置描述代码块的丰富语法高亮。</p></li><li><p><a href="https://code.visualstudio.com/updates/v1_60?WT.mc_id=devcloud-00000-cxa#_terminal" target="_blank">自定义终端字形渲染</a> —— 更好地显示框图和块元素字符。</p></li><li><p><a href="https://code.visualstudio.com/updates/v1_60?WT.mc_id=devcloud-00000-cxa#_set-value-in-watch-view" target="_blank">设置调试观察值</a> —— 在调试会话中改变观察值。</p></li><li><p><a href="https://code.visualstudio.com/updates/v1_60?WT.mc_id=devcloud-00000-cxa#_notebooks" target="_blank">笔记本改进</a> —— Markdown 链接导航，更快地渲染大型输出。</p></li><li><p><a href="https://code.visualstudio.com/updates/v1_60?WT.mc_id=devcloud-00000-cxa#_inlay-hints-for-javascript-and-typescript" target="_blank">JavaScript/TypeScript 内嵌提示</a> —— 参数名称和类型的内嵌提示，以及更多。</p></li><li><p><a href="https://code.visualstudio.com/updates/v1_60?WT.mc_id=devcloud-00000-cxa#_locked-editor-groups" target="_blank">锁定的编辑器组预览</a> —— 通过锁定编辑器组来保持一个首选的编辑器布局。</p></li><li><p><a href="https://code.visualstudio.com/updates/v1_60?WT.mc_id=devcloud-00000-cxa#_python" target="_blank">Python 扩展测试更新</a> —— 更好地支持测试发现、导航和状态。</p></li><li><p><a href="https://code.visualstudio.com/updates/v1_60?WT.mc_id=devcloud-00000-cxa#_web-extensions" target="_blank">Web 扩展作者指南</a> —— 了解如何在<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>中为 VS Code 更新你的扩展。</p></li></ul>
