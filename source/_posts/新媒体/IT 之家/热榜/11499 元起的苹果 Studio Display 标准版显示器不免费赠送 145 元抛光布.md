<p data-vmark="170c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 9 日消息，据 MacRumors 报道，苹果全新的标准 Studio Display 显示器在包装盒中没有附带抛光布，这与 Pro Display XDR 不同，后者在标准和纳米纹理型号中都提供抛光布。</p><p style="text-align: center;" data-vmark="ab53"><img src="https://img.ithome.com/newsuploadfiles/2022/3/6a5c144e-0127-4815-a24a-e90b9b04f6a2.jpg" w="800" h="450" title="11499 元起的苹果 Studio Display 标准版显示器不免费赠送 145 元抛光布" width="800" height="450" referrerpolicy="no-referrer"></p><p data-vmark="10cc">在其网站上，苹果公司表示，只有纳米纹理型号的 Studio Display 才会在包装盒中配备抛光布，以及显示器本身和一条 1 米长的 Thunderbolt 线缆。</p><p style="text-align: center;" data-vmark="e870"><img src="https://img.ithome.com/newsuploadfiles/2022/3/b110d86f-c42d-433a-bcde-fcce48327fb6.jpg" w="800" h="401" title="11499 元起的苹果 Studio Display 标准版显示器不免费赠送 145 元抛光布" width="800" height="401" referrerpolicy="no-referrer"></p><p data-vmark="41b3">苹果最高端的 Pro Display XDR，无论何种配置，都会在包装盒中为用户提供一块抛光布。购买标准的 Studio Display（国行 11499 元起）的客户需要额外购买苹果 145 元的抛光布，或者从其他地方寻找一款合适的显示器清洁布。</p><p style="text-align: center;" data-vmark="b5f4"><img src="https://img.ithome.com/newsuploadfiles/2022/3/2ee1fc57-f59f-4611-95e3-ab78b4718662.png" w="1278" h="668" title="11499 元起的苹果 Studio Display 标准版显示器不免费赠送 145 元抛光布" width="1278" height="429" referrerpolicy="no-referrer"></p><p data-vmark="0d19">苹果今日推出了全新的 Mac Studio 台式电脑和 Studio Display 显示器，宣称“充分满足用户的各项需求，帮助用户打造梦寐以求的工作室。”Mac Studio 搭载了 M1 Max 或最新的 M1 Ultra，这款电脑采用紧凑设计，可轻松地置于桌面上进行操作，提供丰富的连接方式和各种全新功能。Mac Studio 能让用户完成使用其他台式电脑无法完成的任务，例如渲染大规模的 3D 场景，或者同时播放 18 条 ProRes 视频流。Studio Display 则是 Mac Studio 的完美伴侣，搭配其他 Mac 电脑也同样合拍。它拥有宽大的 27 英寸 5K 视网膜显示屏，支持人物居中的 1200 万像素超广角摄像头，以及支持空间音频的高保真六扬声器音响系统。</p><p data-vmark="3989">苹果表示，Mac Studio 与 Studio Display 能够完美搭配，相得益彰，可以将任何工作空间变为强大的创意工作室。两款产品作为 Apple 迄今最强大的 Mac 产品系列的新成员，于今日开放预购，并将于 3 月 18 日（星期五）起开始向顾客发货。</p>
