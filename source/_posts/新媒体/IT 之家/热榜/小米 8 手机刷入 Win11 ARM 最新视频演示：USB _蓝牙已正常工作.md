<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 16 日消息 此前，微软面向 Dev 开发频道发布了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Insider Preview 版本，微软 Windows 11 Insider Preview 版本 Build 22000.51 镜像也已流出，其中包括 Windows 11 ARM 预览版本，有开发者已经在 Lumia 950 XL 刷入了 Windows 11 ARM 预览版本，当时有网友表示，搭载骁龙 845 处理器的小米 8 手机也已成功刷入 Windows 11 ARM64 完整系统，但该机型上的 USB、触屏还不能正常工作，无法控制。</p><p>现在 B 站 UP 主 @5 超级菜鸟 5 表示，小米 8 手机刷 Windows 11 ARM64 系统迎来了新的进展，USB 和蓝牙功能已经正常工作。</p><p>下面是小米 8 手机进入 Windows 11 ARM64 系统后的桌面，“开始”菜单新 Logo，还有新的任务栏。</p><p><img src="https://qimg.ithome.com/image/2021/07/16/6376200515283526322064993.jpg" title="小米 8 手机刷入 Win11 ARM 最新视频演示：USB /蓝牙已正常工作" referrerpolicy="no-referrer"></p><p>下面是视频：</p><p><iframe class="ithome_video bilibili" src="https://player.bilibili.com/player.html?bvid=bv14b4y1k7kg&page=1" frameborder="no" width="100%" allowfullscreen="true"> </iframe></p><p><img src="https://qimg.ithome.com/image/2021/07/16/6376200515854676364116512.jpg" title="小米 8 手机刷入 Win11 ARM 最新视频演示：USB /蓝牙已正常工作" referrerpolicy="no-referrer"></p><p><img src="https://qimg.ithome.com/image/2021/07/16/6376200516330394068798693.jpg" title="小米 8 手机刷入 Win11 ARM 最新视频演示：USB /蓝牙已正常工作" referrerpolicy="no-referrer"></p><p>另外小米 8 刷入 Windows 11 ARM 系统还有个 Bug，重启后会蓝屏。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/f7a74379-4c4b-4c41-b726-8fa258e969ac.png" w="619" h="645" title="小米 8 手机刷入 Win11 ARM 最新视频演示：USB /蓝牙已正常工作" width="619" height="645" referrerpolicy="no-referrer"></p><p>项目地址：https://github.com/edk2-porting/edk2-sdm845</p><p>驱动下载地址：https://github.com/edk2-porting/WOA-Drivers</p><p>微软官方此前早已经对 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> Mobile 停止支持，不过在 WoA 项目开发者的努力下，仍然会有开发者去为旧机型刷入新系统。</p><p>由于微软提供了 ARM 设备兼容的 Windows 10 版本，一群独立开发人员在过去几年里一直在维护一个非官方的 Lumia Windows on ARM 项目，这就是 Lumia WOA。现在微软发布了 Windows 11 ARM 预览版本，又有新的系统可以尝试了。</p>
