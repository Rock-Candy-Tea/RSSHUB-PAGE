<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 7 日消息 微软对 Windows 7 和 Windows 8.1 用户的免费升级福利早在 2016 年就结束了，但用户仍然可以升级到 Windows 10。微软表示，将继续支持从 Windows 7 或 Windows 8.1 升级到 Windows 11，只要他们满足最低系统要求。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/f1a88e9e-f6fb-4474-9a85-3464b562f747.jpg" w="696" h="365" title="Win7/8.1 可免费升级到 Win11，但应用、设置等内容会被删除" width="696" height="365" referrerpolicy="no-referrer"></p><p>然而，Windows 7 升级到 Windows 11 后，用户的应用程序、设置和自定义功能可能会被删除。这是因为根据联想 6 月 24 日发布的一份支持文件，<span class="accentTextColor">Windows 7/8.1 用户没有合适的直接升级路径。</span></p><p>联想印度公司的一份新的支持文件（该文件基于微软的说法）表明，Windows 7 和 Windows 8/8.1 的用户也可以升级到 Windows 11，<span class="accentTextColor">但只能清洁安装（clean install）或重新 reimage（重新安装）。</span></p><p>对于那些使用 Windows 10 的用户来说，如果硬件符合要求的话，他们可以通过传统的更新途径--Windows Update、WSUS、媒体创建工具和更新助手，免费升级到 Windows 11。</p><p>支持文件中写道：“现在可以购买的大多数设备将可以升级到 Windows 11。你将可以选择升级、清洁安装或重新安装 Windows 10 设备，以转移到 Windows 11。<span class="accentTextColor">对于符合硬件要求的 Windows 7 设备，需要进行清洁安装或重新安装，以使用 Windows 11</span>。”</p><p>Windows 11 的早期预览版本已经包括了 Windows 7、8 和 8.1 的配置密钥，这将使用户能够免费升级。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/5ab92b31-92b2-4925-9b3f-2d71b1ff4de6.jpg@s_2,w_820,h_526" w="1058" h="679" title="Win7/8.1 可免费升级到 Win11，但应用、设置等内容会被删除" srcset="https://img.ithome.com/newsuploadfiles/2021/7/5ab92b31-92b2-4925-9b3f-2d71b1ff4de6.jpg 2x" width="1058" height="526" referrerpolicy="no-referrer"></p><p>当然，如果用户不喜欢新的 Windows 11，可以随时降级到 Windows 10。要降级到 Windows 10，并保留文件和数据，用户需要在 10 天的窗口期移回 Windows 10。如果在安装 10 天后降级，则需要手动备份你的应用程序并进行清洁安装。</p><p>Windows 11 和 Windows 7 操作系统有很大的不同，但微软承诺，现有的应用程序将继续工作。该公司说：“Windows 11 的构建考虑到了兼容性。因此，如果应用程序在 Windows 7/8.1/10 上正常工作，它们也将继续在 Windows 11 上正常工作。”</p><p>“如果你有 App Assure [企业专用服务]，微软将免费帮助你修复它们，”微软在同一文件中指出。</p><p>微软目前正计划在今年下半年推出 Windows 11，我们将在未来几周/几个月内了解更多关于硬件兼容性的信息。</p>
