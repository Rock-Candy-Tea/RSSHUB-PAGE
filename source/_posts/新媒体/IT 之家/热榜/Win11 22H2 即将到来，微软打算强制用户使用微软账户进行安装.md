<p data-vmark="cf40"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 9 日消息，Windows 11 22H2 即将到来，而从最近的更新来看它显然已经删除了为设备提供本地帐户的选项，即使你安装的是 Windows 11 专业版。</p><p data-vmark="0e18">在此之前，微软已经针对 Windows 11 家庭版用户进行了调整，取消了在 OOBE 界面使用本地账户的选项，强迫用户使用微软账户进行设置、注册。</p><p data-vmark="1bd3">从 Windows 11 22H2 版本开始，微软希望用户都使用在线登录微软账户来进行安装，不过“工作”或“学校”用户仍然不受影响，用户可以按照自己的方式进行设置。</p><p data-vmark="24dc">当你登录你的账户时，微软会自动为你的设备配置 OneDrive，并开始在你的设备之间同步数据，包括设置和偏好。虽然将微软账户链接到 Windows 11 是一件很实用的事，但并不是每个人都喜欢微软强迫他们做事。</p><p data-vmark="49b3" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/1aef0e1c-7383-4fa0-9af7-8c7fb3a577de.jpg@s_2,w_820,h_615" w="1024" h="768" alt="Windows 11 Microsoft account" title="Win11 22H2 即将到来，微软打算强制用户使用微软账户进行安装" srcset="https://img.ithome.com/newsuploadfiles/2022/5/1aef0e1c-7383-4fa0-9af7-8c7fb3a577de.jpg 2x" width="1024" height="615" referrerpolicy="no-referrer"></p><p data-vmark="8d47">在最新的 Build 22616 中，Windows 11 Pro 版本在安装时已经强制要求用户设置一个 Microsoft 帐户，而且你必须提供互联网连接。</p><p data-vmark="4dfa">当然，只有当你选择将设备设置为个人使用时才会出现这种情况，而如果您选择了“工作”或“学校”账户，那么系统将提示您使用组织的域登录注册。</p><p data-vmark="a650">IT之家了解到，目前仍然可以通过输入不存在的微软账户电子邮件和密码来绕过账户要求，不过这么做的话会导致一个 bug，虽然对用户来说不是特别方便，但这似乎是最简单的解决方案。</p><p data-vmark="013f">如果你安装 Windows 10 系统，机器如果没有联机则可以为你设置一个本地账户，而在 Windows 11 上却要求用户必须连接到互联网。当然，你也可以通过注册表之类的方法禁用连网，但这个方法很复杂，对于大部分用户而言并不实用。</p><p data-vmark="fcbd">从目前情况来看，强制使用网络 ID 登录新设备这已经是整个行业的趋势，无论是手机还是 PC。而且，登录微软账户对于日常中的 Windows 操作来说是有好处的，特别是你已经在微软的生态中投入了大量资金或精力的情况下，包括 Xbox、Your Phone、OneDrive 等服务都可以帮你更好的工作和生活。</p><p data-vmark="62e3">然而这里还是要提一句，虽然大部分 Windows 用户不反对这种行为，但相信用户还是喜欢能有选择的余地，毕竟也有许多人都会觉得使用一个本地账户更舒服，更能确保隐私。</p><p data-vmark="9fb9">《<a href="https://www.ithome.com/0/616/811.htm" target="_blank">微软发布 Win11 Dev & Beta 22616 更新：带来全新的 Xbox 功能栏，任务栏托盘弹窗图标拖放功能回归</a>》</p>
