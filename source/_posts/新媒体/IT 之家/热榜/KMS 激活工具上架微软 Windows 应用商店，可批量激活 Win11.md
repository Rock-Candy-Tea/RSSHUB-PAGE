            <p data-vmark="320b"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 2 日消息，据网友投稿，微软 Windows 商店竟然上架了以破解 Windows 系统和 Office 而知名的“KMS 激活工具”。</p><p data-vmark="15a6">该应用介绍页面显示，“本应用可用于激活您的 Windows 操作系统和 Office 桌面应用，KMS 是一种组织内部批量激活 Windows 和 Offce 系列软件的服务系统，您需要向您所在组织的有关部门获取一个 KMS 激活服务器地址，将该地址提供给本应用，然后选择您的 Windows 操作系统版本，就可以实现一键激活，对于 Office，2019 以前 (不含) 的版本未经测试，请谨慎使用。对于 Windows11，可选择对应的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 版本，二者完全通用。如果您所在组织提供了特殊的激活密钥，也可以直接手动输入。只要服务器的运营者向 Microsoft 购买了合法的授权，您激活的软件就是 100% 完全正版的，不必担心法律问题。”</p><p data-vmark="7ae9">介绍页面还称，<span class="accentTextColor">本应用不适用于个人用户及未获得 Microsoft 授权的组织用户激活操作需要管理员权限。如果您不是管理员用户，本应用将无法正常工作</span>。</p><p data-vmark="eafb">从介绍来看，该工具本身没有提供非法破解的方案，更像是给企业进行多设备的批量激活。</p><p data-vmark="2e54">IT之家了解到，Key Management Service 即简称 KMS，这是微软提供的一种新型的产品激活机制，更加高效，更加灵活，可以实现大批量地激活 windows 操作系统以及 office 办公软件。</p><p data-vmark="f888" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/7687cad2-d07a-4935-bf99-0a75ad065121.png" w="1440" h="774" title="KMS 激活工具上架微软 Windows 应用商店，可批量激活 Win11" width="1440" height="441" referrerpolicy="no-referrer"></p><p data-vmark="7a91" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/f2472cce-1559-4dcc-a840-92718cf795c0.png" w="1440" h="774" title="KMS 激活工具上架微软 Windows 应用商店，可批量激活 Win11" width="1440" height="441" referrerpolicy="no-referrer"></p>
