<p data-vmark="eb57"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 30 日消息，微软已为测试人员推送 Windows 11 Build 25151 Dev 更新，除了海量 Bug 修复之外，新版本同时还带来一个新的隐藏功能。</p><p data-vmark="c207">从下图中我们可以看到，如果你在 Win11 中右键选择“打开方式”，然后选择其他应用，会出现一种更贴合 Win11 现代化 UI 设计的新外观。</p><p style="text-align: center;" data-vmark="efa1"><img src="https://img.ithome.com/newsuploadfiles/2022/6/56230dd7-67b1-464b-93be-d37ab0635d14.png" w="427" h="322" title="Win11 UI 焕新：微软为最古老的 Windows 功能之一带来新外观" width="427" height="322" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="7c1b"><img src="https://img.ithome.com/newsuploadfiles/2022/6/f04fbeea-1d38-4257-a339-eb8e3cb612d7.jpg@s_2,w_820,h_500" w="1079" h="658" alt="Windows 11 Build 25151 打开方式" title="Win11 UI 焕新：微软为最古老的 Windows 功能之一带来新外观" srcset="https://img.ithome.com/newsuploadfiles/2022/6/f04fbeea-1d38-4257-a339-eb8e3cb612d7.jpg 2x" width="1079" height="500" referrerpolicy="no-referrer"></p><p data-vmark="8199">实际上，该功能在今年早些时候曾被曝光，但微软从后来的版本中已将其删除。</p><p data-vmark="e2da">简单来说，Windows 11 Build 25151 没有任何特定的功能更新，但它被认为是明年 23H2 功能更新的一部分。但目前 23H2 尚处于早期开发阶段，因此没有带来任何新功能。</p><p data-vmark="dfa5">IT之家了解到，微软上一次为右键“选择其他应用打开”对话框更新外观还是在 Windows 8 中，现在微软显然已为 Windows 11 进行了更新。</p><p data-vmark="7e70">从图中我们可以看到，Windows 11 新的对话框已经支持浅色和深色主题。如果您的 PC 主题设置为深色，则“打开方式”对话框将自动以深色显示。</p><p data-vmark="bcb8">此外，微软还通过一键更新默认应用程序来简化体验。当然，此功能并非面向所有用户推出，目前仅在部分被选中的机器上才会出现。</p>
