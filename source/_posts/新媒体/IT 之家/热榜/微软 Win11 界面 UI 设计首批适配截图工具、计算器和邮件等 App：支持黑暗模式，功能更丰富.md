<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 13 日消息 今天<a href="https://www.ithome.com/0/569/033.htm" target="_blank">微软发布了 Windows 11 预览版 Build 22000.132</a>，累积更新补丁为 KB5005190，面向 Dev 开发和 Beta 测试频道的内测人员发布。这个预览版本带来了全新的微软 Teams 聊天，升级后的截图工具、计算器应用和邮件和日历应用等。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/b539766d-32c9-4471-8392-6cf266eca093.png" w="919" h="525" title="微软 Win11 界面 UI 设计首批适配截图工具、计算器和邮件等 App：支持黑暗模式，功能更丰富" width="919" height="468" referrerpolicy="no-referrer"></p><p>微软表示，面向 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 内测用户们，正在推出第一组应用更新，这些更新包括作为 Windows 11 一部分的几个应用程序。以下应用程序的更新将首先在 Dev 频道中向 Windows Insiders 推出。</p><h2>截图工具（Snipping Tool）</h2><p><img referrerpolicy="no-referrer"></p><p>▲ Windows 11 中新的 Snipping Tool 图标</p><p>在 Windows 11 中，经典的 Snipping Tool 和 Snip & Sketch 应用都被一个新的 Snipping Tool 应用所取代，它代表了这两个应用在 Windows 屏幕截图的下一次进化中的最佳体验。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/6d953024-9e02-4dac-aea5-cbcc79b6eb41.png" w="722" h="435" title="微软 Win11 界面 UI 设计首批适配截图工具、计算器和邮件等 App：支持黑暗模式，功能更丰富" width="722" height="435" referrerpolicy="no-referrer"></p><p>▲ Windows 11 中 Snipping Tool 新视觉效果</p><p>Windows 11 的 Snipping Tool 包括新的视觉效果，它建立在经典应用程序的基础上，并增加了一些功能，如来自 Snip & Sketch 的 WIN + SHIFT + S 键盘快捷方式和更丰富的编辑。微软还为 Snipping Tool 引入了新的设置页面。</p><p>使用“WIN + SHIFT + S”键盘快捷键，截取屏幕快照变得超级简单！这将弹出带有选项的剪裁菜单，包括矩形剪裁、自由格式剪裁、窗口剪裁和全屏剪裁。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/29906082-bcf8-4be6-8a33-6c5dea04a185.png" w="1426" h="917" title="微软 Win11 界面 UI 设计首批适配截图工具、计算器和邮件等 App：支持黑暗模式，功能更丰富" width="1426" height="527" referrerpolicy="no-referrer"></p><p>▲ 带有剪裁选项的截图菜单</p><p><strong>注意：</strong>如果你关闭了通知功能或开启了专注模式功能，你在截图后将不会收到弹出的通知。然而，你的截图仍然会被复制到剪贴板。这将在未来的更新中被修复。</p><p>一旦你获取了屏幕截图，Snipping Tool 将提供用于注释的编辑工具，改进剪裁，以及更多的编辑屏幕截图的方法。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/f6465fd1-f361-4bb1-8619-a3c118825709.png" w="972" h="778" title="微软 Win11 界面 UI 设计首批适配截图工具、计算器和邮件等 App：支持黑暗模式，功能更丰富" width="972" height="656" referrerpolicy="no-referrer"></p><p>▲ 编辑工具包括注释、改进的裁剪，以及更多按你想要的方式编辑你的屏幕截图</p><p>最后，Snipping Tool 现在支持你的 Windows 主题! 如果 Windows 11 主题是黑暗模式，Snipping Tool
