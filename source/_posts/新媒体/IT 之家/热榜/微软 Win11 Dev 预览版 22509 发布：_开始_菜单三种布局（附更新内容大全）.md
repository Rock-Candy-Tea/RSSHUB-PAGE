<p data-vmark="fac2"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 2 日消息，今天微软官方发布了最新的 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Dev 预览版 22509 系统更新，这个新版本为“开始”菜单提供了新的布局选项，改进了 Edge + 讲述人的体验，并修复了一系列的错误 Bug。</p><p data-vmark="4c33"><img src="https://img.ithome.com/newsuploadfiles/2021/12/37c0ecac-020b-48aa-b71e-0d0493d762f3.png" w="1440" h="967" title="微软 Win11 Dev 预览版 22509 发布：“开始”菜单三种布局（附更新内容大全）" width="1440" height="551" referrerpolicy="no-referrer"></p><h2 data-vmark="6c7e">更新摘要</h2><ul class=" list-paddingleft-2"><li><p data-vmark="8047">该版本包括一系列改进，使在 Microsoft Edge 中使用 Narrator 讲述人浏览网页更加容易。</p></li><li><p data-vmark="ee5b">该版本包括一系列良好的改进，包括“开始”菜单的新布局选项，时钟和日期现在将显示在辅助显示器上，以及设置的变化。</p></li></ul><h2 data-vmark="a529">下面是 Windows 11 Build 22509 中的具体更新内容</h2><h3 data-vmark="fcdd">使用讲述人在 Microsoft Edge 中浏览网页现在更容易了</h3><p data-vmark="99d5">微软一直在努力对使用 Microsoft Edge 和 Narrator 浏览网页进行一系列改进。具体来说，现在在编辑字段中打字的速度应该更快，在网络上导航时，会提供更多有用的信息，最后，你将拥有与 Narrator 更一致的导航体验。</p><p data-vmark="7734"><img src="https://img.ithome.com/newsuploadfiles/2021/12/134e45d4-af98-446f-a335-a6c9fe055e3e.png" w="1440" h="970" alt="Narrator Home in Windows 11." title="微软 Win11 Dev 预览版 22509 发布：“开始”菜单三种布局（附更新内容大全）" width="1440" height="552" referrerpolicy="no-referrer"></p><p data-vmark="cb16">微软使在编辑字段（如地址栏）中的输入变得更加容易，因为 Narrator 扫描模式现在会更快地关闭。具体来说，如果你按下 Ctrl + L 跳到地址栏，你应该能够立即开始输入，因为扫描模式将更快关闭。这一改进在填写表格字段时也会有帮助，比如在不同的编辑框中输入你的名字和地址。除了使在编辑栏中打字更容易外，微软还使在编辑栏中删除更容易。讲述人现在会在删除一个字符后读取光标位置所在的字符。换句话说，如果光标在 "hello" 的 "h" 上，你按下删除键，Narrator 会说 "e"。</p><p data-vmark="8caa">IT之家获悉，在网页上导航时，你还会得到更多的上下文信息。例如，所需的单选按钮和编辑字段现在将被读出，关于列表的更多信息，包括列表级别和准确的列表指数现在将被读出。</p><p data-vmark="4483">除了获得更多的信息读出外，你将有一个与 Narrator 更一致的导航体验。这包括标题导航的改进，更一致的扫描模式导航体验，其中扫描模式不太可能在项目之间循环，以及向前和向后导航时更一致的超链接阅读体验。</p><p data-vmark="7ad4">注意：需要 Microsoft Edge 97.0.4683.0 或更高版本，以利用 Windows 11 Insider Preview Build 22509 及以上版本的这些改进。</p><h2 data-vmark="ce55">变化和改进</h2><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="4942">从这个版本开始，你可以右击“开始”菜单来快速访问“开始”设置，并选择“更多固定”或“更多推荐”布局选项来分别显示额外的固定图标或推荐行。</p></li></ul><p data-vmark="6f28"><img src="https://img.ithome.com/newsuploadfiles/2021/12/241e070f-017c-44a2-80ff-93e17728e188.png" w="1363" h="546" title="微软 Win11 Dev 预览版 22509 发布：“开始”菜单三种布局（附更新内容大全）" width="1363" height="328" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="0e94">当第二台显示器被连接时，时钟和日期现在也会显示在第二台或几台显示器的任务栏上，以便于浏览。[微软正在开始推广这一变化，所以它还没有向所有的内测人员提供，因为微软计划监测反馈，并在向所有人推送之前看看它的效果。］</p></li><li><p data-vmark="66fa">对于在操作系统中利用 Windows 通知发送电话、提醒或警报通知的应用程序，3 个高优先级的通知现在将被显示为堆叠并同时显示。这意味着在任何时候，你可能同时看到多达 4 个通知 ——3 个高优先级通知和 1 个普通优先级通知。[微软正在开始推出这一变化，所以它还没有向所有的内测人员提供，因为微软计划监测反馈，并在向所有人推送之前看看它的效果。］</p></li><li><p data-vmark="5498">作为微软正在进行的将“设置”从控制面板引入“设置”应用程序的努力的一部分：</p></li></ul><p data-vmark="e718">・ 微软已将高级共享设置（如网络发现、文件和打印机共享以及公共文件夹共享）移至“设置”应用程序中高级网络设置下的一个新页面。</p><p data-vmark="90e3">・ 微软对“设置”中的“打印机和扫描仪”下的设备特定页面进行了一些更新，以便在“设置”中直接显示有关你的打印机或扫描仪的更多信息。</p><p data-vmark="3f8c">・ 控制面板中的一些网络和设备设置的入口现在将重定向到“设置”中的相应页面。</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="e7ce">在“设置”中的“已安装的应用程序”页面增加了新的选项，从名称（Z 到 A）对列表进行排序，并相应地将之前的“按字母顺序”选项更新为现在的名称（A 到 Z）。</p></li><li><p data-vmark="2664">微软现在会记住你在飞行模式下是否打开蓝牙或 Wi-Fi。下次你使用飞行模式时，该模块将反映你的偏好，并保持开启状态，以方便你在旅行时继续倾听耳机并保持连接。</p></li><li><p data-vmark="7628">Windows Sandbox 沙盒现在支持在其虚拟化环境中重启（例如，如果你点击“开始”中电源按钮下的重启选项）。</p></li></ul><h2 data-vmark="e31c">修复</h2><p data-vmark="228a"><strong>[任务栏]</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="bcde">当系统缩放比例设置为 125% 时，任务栏中的“开始”、搜索、任务视图、小组件和 Chat 聊天图标应该不再意外放大。</p></li><li><p data-vmark="691d">如果你重复按 Windows 键 +（对应于应用程序图标在任务栏中的位置的数字），现在应该循环浏览应用程序的窗口，而不是停在最后一个窗口。</p></li><li><p data-vmark="40e2">当 Narrator 的焦点被设置为任务视图按钮并出现桌面弹窗时，Narrator 现在会让你知道有关桌面的信息。</p></li><li><p data-vmark="ec6e">修复了一些内测人员在最近的版本中遇到的与更新任务栏中的标记有关的 explorer.exe 崩溃问题。</p></li></ul><p data-vmark="c819"><strong>[输入]</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="86c9">当语言设置为中文时，手写现在应该可以重新工作。</p></li><li><p data-vmark="f076">缓解了一个与鼠标有关的问题，该问题在最近的版本中有时会导致一些 Insiders 在休眠期间遇到 bugcheck 故障检测（使其看起来像计算机已重新启动）。</p></li><li><p data-vmark="f68e">更新了中文简体 IME 候选，使其与候选窗口高度居中对齐，并已进行了定制。</p></li><li><p data-vmark="e3a3">修正了 PowerShell 中的触摸键盘调用，所以现在应该可以正常工作。</p></li></ul><p data-vmark="003f"><strong>[窗口]</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="fdb0">在任务视图中的不同桌面之间来回悬停鼠标，将不再导致显示的缩略图和内容区域意外地缩小。</p></li><li><p data-vmark="7f30">CompactOverlay 在这个版本中应该可以重新工作（也就是一些应用程序用来创建一个总是在顶部的小窗口的选项）。</p></li><li><p data-vmark="96b6">修正了一个竞争条件，当在任务视图中关闭 3 个或更多窗口的快照组时，该条件有时会导致 explorer.exe 崩溃。</p></li><li><p data-vmark="5fdf">删除了当点击新桌面按钮的一个角落时，意外出现倾斜动画的问题。</p></li><li><p data-vmark="17cb">进行了调整，以解决一个潜在的问题，该问题导致在任务视图中右键单击桌面时，上下文菜单会向鼠标下方出现动画，而不是远离鼠标。</p></li><li><p data-vmark="691e">修正了最近版本中高命中的 DWM 崩溃问题。</p></li></ul><p data-vmark="c593"><strong>[设置]</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="52d7">修正了在“设置”中的已安装应用程序页面上某些语言的文本重叠问题。</p></li><li><p data-vmark="d827">在网格视图中使用安装的应用程序页面时，点击卸载按钮现在将显示确认对话框。</p></li><li><p data-vmark="4c5d">将新的 ms-settings:installed-apps URI 改为现在的 ms-settings:appsfeatures，以解决现有链接希望后者能指向已安装应用列表的情况，例如，在“开始”菜单中右击 UWP 应用后选择应用设置选项。</p></li></ul><p data-vmark="4dd4"><strong>[其他］</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="6c37">修正了导致一些内测人士在试图升级到最近的 Dev 频道构建时看到错误 0x8007001f 的问题。</p></li><li><p data-vmark="9098">修正了一些 PC 无法安装新的构建版本，或其他更新的问题，错误代码为 0x80070002。</p></li><li><p data-vmark="7064">修正了导致一些设备在安装新的构建版本时出现错误代码 0xc1900101-0x4001c 的回滚问题。</p></li><li><p data-vmark="6947">做了一些工作，以帮助解决导致 Wi-Fi 连接在从睡眠恢复后丢失的问题。</p></li></ul><p data-vmark="95c3">注意：这里提到的来自活跃 Dev 分支的 Insider Preview 构建中的一些修复可能会进入 10 月 5 日开始普遍提供的 Windows 11 发布版本的服务更新。</p><h2 data-vmark="47f0">已知问题</h2><p data-vmark="dc43"><strong>[一般]</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="6700">用户使用最新的 Dev 频道 ISO 镜像从 22000.xxx 或更早的版本更新到较新的 Dev 频道版本，可能会收到以下警告信息：你试图安装的版本是飞行签名的。要继续安装，请启用飞行签名。如果你收到这条消息，请按下启用按钮，重新启动电脑，并重新尝试更新。</p></li></ul><p data-vmark="ff23"><strong>[开始]</strong></p><ul class=" list-paddingleft-2"><li><p list-paddingleft-2"><li><p data-vmark="beb7">切换输入法时，任务栏有时会闪动。</p></li></ul><p data-vmark="ae99"><strong>[搜索]</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="dc5f">点击任务栏上的搜索图标后，搜索面板可能无法打开。如果出现这种情况，请重新启动“Windows Explorer”进程，并再次打开搜索面板。</p></li></ul>
