<p data-vmark="0313"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 3 日消息，据 Windows Latest 报道，Windows Store 存在应用程序问题并不是什么秘密。事实上，著名的“App 差距”是 Windows Phone 垮台的原因之一。在 Windows 10 Mobile 时代，许多开发人员忽略了他们的 Windows Store 应用，而专注于流行的移动商店，如谷歌 Play Store 或苹果 App Store。</p><p style="text-align: center;" data-vmark="1b04"><img src="https://img.ithome.com/newsuploadfiles/2022/5/772f0e47-ef43-4286-ae02-e799f3008c71.jpg" w="696" h="365" title="不怕尴尬了，微软 Win11 商店 Web 版终于显示应用“最后更新”日期" width="696" height="365" referrerpolicy="no-referrer"></p><p data-vmark="662a">为了掩饰尴尬，微软在所有 Windows 设备上的 Microsoft Store 中删除了“最后更新”日期。用户几乎不可能知道特定应用程序的最新版本何时在 Microsoft Store 中发布。</p><p data-vmark="8ea5">微软没有在任何公共平台上承认做出了这一改变，人们普遍认为这家科技巨头想要隐藏商店中的许多应用程序很久未更新的事实。这些理论当然是有道理的，尤其是考虑到 Windows Store 上的应用程序与 Android、iOS 相比或多或少已经过时。</p><p style="text-align: center;" data-vmark="a0c9"><img src="https://img.ithome.com/newsuploadfiles/2022/5/f4388096-8ae3-4222-8352-992d7fd1cbf8.jpg@s_2,w_820,h_510" w="1440" h="895" title="不怕尴尬了，微软 Win11 商店 Web 版终于显示应用“最后更新”日期" srcset="https://img.ithome.com/newsuploadfiles/2022/5/f4388096-8ae3-4222-8352-992d7fd1cbf8.jpg 2x" width="1440" height="510" referrerpolicy="no-referrer"></p><p data-vmark="d5f9">不用说，这种做法违反了微软自己的透明度政策，该公司终于朝着恢复“最后更新日期”迈出了一步。</p><p style="text-align: center;" data-vmark="334a"><img src="https://img.ithome.com/newsuploadfiles/2022/5/0b8f71fe-4047-4ac6-9499-24ee228233b5.jpg@s_2,w_820,h_510" w="1440" h="896" title="不怕尴尬了，微软 Win11 商店 Web 版终于显示应用“最后更新”日期" srcset="https://img.ithome.com/newsuploadfiles/2022/5/0b8f71fe-4047-4ac6-9499-24ee228233b5.jpg 2x" width="1440" height="510" referrerpolicy="no-referrer"></p><p data-vmark="747e">正如我们最近报道的那样，微软已经开始向 Windows 10 和 Windows 11 的用户推出其新的 Web 应用商店。</p><p data-vmark="7769">现在，微软发布了另一项服务器端更改，恢复了“上次更新”部分，该部分最终显示了应用程序的更新时间。对一些人来说，它仍然是一个重要且必要的“透明度”功能。</p><p data-vmark="c338">这也可以鼓励对应用商店感兴趣的开发者保持他们的应用程序最新状态，<span class="accentTextColor">还可以帮助用户在购买之前确定应用程序是否得到了开发者的良好维护</span>。</p><h3 data-vmark="d2b9">Windows 应用商店 Web 版介绍</h3><p data-vmark="6406">正如一开始提到的，Windows 应用商店 Web 版有新的设计。</p><p data-vmark="f2c8">如果你无法访问桌面上的原生应用商店，你可以随时从 Web 商店中查找应用。</p><p style="text-align: center;" data-vmark="91b9"><img src="https://img.ithome.com/newsuploadfiles/2022/5/3d5bf375-417b-402c-818e-65bc8c6e05ec.jpg@s_2,w_820,h_483" w="1440" h="848" title="不怕尴尬了，微软 Win11 商店 Web 版终于显示应用“最后更新”日期" srcset="https://img.ithome.com/newsuploadfiles/2022/5/3d5bf375-417b-402c-818e-65bc8c6e05ec.jpg 2x" width="1440" height="483" referrerpolicy="no-referrer"></p><p data-vmark="b5c7">微软的 Web 商店已经更新，焕然一新，新设计符合 Windows 11 的流畅设计。这个新的 Web 商店与 Windows 11 的原生应用商店相匹配，它还提供了新的类别，使浏览应用和游戏变得更加容易。</p><p data-vmark="f58b">你将在 Web 商店中找到所有熟悉的应用程序，例如记事本和 Office。此外，微软更新的 Web 应用商店还可以索引桌面 Win32 应用。</p><p data-vmark="dd5a">你可以单击“安装”按钮，通过新的商店弹出窗口立即安装应用程序，这意味着你不必手动打开本机商店。</p>
