<p data-vmark="ae7f"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 4 日消息，去年 12 月，微软最新版本的 Surface Go 3 现在有了深颜色，以前只有铂金色，新增了纯黑色。但国行版本的更新晚了一些，今日微软中国官方商城终于推出全新 Surface Go 3 纯黑色版的预售，售价 3988 元起。</p><p data-vmark="d497">现在微软中国官方商城全新 Surface Go 3 共有 5 款上架，它们分别是：</p><ul class=" list-paddingleft-2"><li><p data-vmark="1c84">英特尔奔腾 6500Y / 4GB / 64GB，亮铂金，售价 2988 元</p></li><li><p data-vmark="22e0">英特尔奔腾 6500Y / 8GB / 128GB，亮铂金 / 典雅黑，售价 3988 元</p></li><li><p data-vmark="9c99">英特尔酷睿 i3/8GB / 128GB，亮铂金 / 典雅黑，售价 4588 元</p></li></ul><p data-vmark="3830"><img src="https://img.ithome.com/newsuploadfiles/2021/12/d379604e-caa1-4e6e-b95d-32f345c06063.png" w="660" h="371" alt="Surface Go 3纯黑色版" title="3988 元起，微软 Surface Go 3 纯黑色版国行预定：搭载 Win11 + 英特尔奔腾 6500Y / 酷睿 i3" width="660" height="371" referrerpolicy="no-referrer"></p><p data-vmark="9497">在许多国家和地区，可能只有少数不同配置的 Surface Go 3 打算用于零售。其中一个例子是英特尔奔腾金 6500Y、8GB 内存和 128GB 存储版本。预装了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> S 操作系统登场，起步价为 599 欧元（约 4312.8 元人民币）。</p><p data-vmark="6edd">还会有其他几个版本，包括配备英特尔酷睿 i3-10100Y、更多存储和可选 LTE 的版本。就供应情况而言，预计黑色款 Surface Go 3 将在明年 1 月期间将在越来越多的零售商和微软官方零售店开售。</p><p data-vmark="64e0"><img src="https://img.ithome.com/newsuploadfiles/2021/12/760e5af4-d6de-4243-bf55-e51463323a80.png" w="1440" h="810" alt="Surface Go 3纯黑色版" title="3988 元起，微软 Surface Go 3 纯黑色版国行预定：搭载 Win11 + 英特尔奔腾 6500Y / 酷睿 i3" width="1440" height="461" referrerpolicy="no-referrer"></p><p data-vmark="3caa">IT之家了解到，Surface Go 3 轻至 544 克，配备 1080p 摄像头、矩阵式远场双麦克风、杜比音效和 10.5 英寸 1920 x 1280 分辨率触控屏幕。Surface Go 3 处理器可选英特尔奔腾金 6500Y 或英特尔酷睿 i3-10100Y 处理器。该产品提供可选的 LTE 增强版、长效电池续航、内置 Microsoft 安全防护，兼具从平板电脑到笔记本电脑的多功能性，针对触控笔和触控屏进行了优化。</p><p data-vmark="cff3">・ 点击访问：<a href="https://c.duomai.com/track.php?site_id=157062&aid=2649&euid=storynew&t=https://www.microsoftstore.com.cn" target="_blank">微软中国官方商城</a></p>
