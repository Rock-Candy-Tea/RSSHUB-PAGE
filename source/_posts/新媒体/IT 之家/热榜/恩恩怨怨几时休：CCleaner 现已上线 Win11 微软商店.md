<p data-vmark="e226"><a class="s_tag link-instanted" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 9 日消息，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 中的微软应用商店对该公司来说可能是一个新的开始。为了建设自家生态，微软终于放下了它的傲慢，同时取消了一些限制，从而使得 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 用户都可以通过商店下载到最新的 win32 应用程序和 UWP 程序。</p><p data-vmark="7e6a">同时，这也意味着我们将看到一些与微软存在纠纷的产品或竞品出现在微软商店中，如 Firefox 和 Opera 等<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>，还有著名的 Windows 优化应用程序 ——CCleaner。</p><p data-vmark="0dd5"><img src="https://img.ithome.com/newsuploadfiles/2022/1/4875b447-236f-4a80-93d6-065f15081c98.jpg@s_2,w_820,h_518" w="1440" h="909" title="恩恩怨怨几时休：CCleaner 现已上线 Win11 微软商店" srcset="https://img.ithome.com/newsuploadfiles/2022/1/4875b447-236f-4a80-93d6-065f15081c98.jpg 2x" width="1440" height="518" referrerpolicy="no-referrer"></p><p data-vmark="e852">微软与 CCleaner 的恩怨情仇由来已久。2020 年，微软在其安全网站上将 CCleaner 标为“潜在的不需要的应用程序”。有消息称，这主要是由于 CCleaner 加入了一些微软认为“额外、不必要的应用程序”。</p><p data-vmark="29bd">除了去年出现在 Windows 设备上的 PUA 警告外，微软 Windows Defender 实际上还会变相阻止该应用程序运行。</p><p data-vmark="410b">此外，CCleaner 曾被入侵，从而向 Windows PC 传递了一些恶意软件，而且 CCleaner 也有一些 与 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 不兼容的问题。</p><p data-vmark="d974">IT之家了解到，虽然 CCleaner 中的大部分功能都可以用 Windows 内置工具来实现，但我们还是很高兴看到微软愿意放开一些限制，从而将选择权交还给用户。</p>
