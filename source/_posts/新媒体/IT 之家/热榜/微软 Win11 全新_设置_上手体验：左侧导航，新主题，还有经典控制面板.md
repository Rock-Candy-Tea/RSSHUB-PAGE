<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 29 日消息 今天凌晨，微软面向 Dev 开发频道发布了第一个 Windows 11 Insider Preview 版本，即 Build 22000.51!  微软表示，随着我们在未来几个月内完成产品的定型，我们将与你一起验证这一体验。在这个早期预览中，<a href="https://www.ithome.com/0/559/232.htm" target="_blank">你将可以尝试上周展示的许多新功能</a>，但不是全部。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/a425c03b-ac1b-473d-8a59-e4ef3edc71b7.png" w="1440" h="1128" title="微软 Win11 全新“设置”上手体验：左侧导航，新主题，还有经典控制面板" width="1440" height="642" referrerpolicy="no-referrer"></p><p>在 6 月 24 日的下一代 Windows 活动中，微软重点介绍了所有华丽且令人兴奋的新功能和生产力工具，但新的设置应用程序是另一个让人对 Windows 11 感到兴奋的理由。这个最初代号为“太阳谷”的新操作系统包括重新设计的“设置”应用程序，基于控制面板和 Windows 10 设置的设计原则重新打造。</p><p>在 Windows 11 上，“设置”应用程序与我们在 Windows 10 上的应用程序有很大的不同。新的“设置”重新设计解决了空白空间的问题，改善了搜索体验，它还在左侧增加了新的导航窗格。侧边栏与我们在控制面板中的侧边栏相似，所有选项都采用了流畅设计图标。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/c0e97673-a824-494a-9f82-6c81e862aa51.jpg@s_2,w_820,h_607" w="1119" h="829" title="微软 Win11 全新“设置”上手体验：左侧导航，新主题，还有经典控制面板" Windows Update 或蓝牙等子页面时，轻松回到个性化和网络等主要选项。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/ff7ff78a-25da-4851-a633-a635d646954c.jpg@s_2,w_820,h_640" w="1052" h="821" title="微软 Win11 WinUI 面包屑设计，以帮助你在深入浏览 Windows 设置时知道你处于哪里。Windows 11 还在顶部引入了新的页眉或“hero 控件”，将突出显示桌面背景和常用设置等信息，以帮助你根据需要定制操作系统。</p><p>这种新的标题设计在几个类别的页面上都可以看到。Windows Update、个性化、网络和互联网、系统等。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/83c1d877-df94-4a5f-8411-63ba13295ffd.jpg@s_2,w_820,h_491" w="1282" h="768" title="微软 Win11 全新“设置”上手体验：左侧导航，新主题，还有经典控制面板" 有更大的控制权。例如，你现在可以看到为累积或安全更新重新启动需要多长时间的估计时间。当你的设备等待重启时，这个新状态将出现在 Windows Update 页面内。</p><p><img 主题之间切换。使用触摸键盘的用户可以用背景图片对键盘进行个性化设置。此外，你现在可以改变按键的大小或颜色。</p><h2>控制面板和新“设置”将并存</h2><p>Windows 设置应用程序是在几年前推出的，它原本应该在 Windows 10 中取代控制面板。但随着 Windows 11 的推出，控制面板仍然不会消失。</p><p>如果你使用控制面板，Windows 11
