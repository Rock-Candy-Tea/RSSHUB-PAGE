<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 10 日消息 今日凌晨，面向 Windows 预览体验成员，微软为 Beta 通道和商用 PC <span class="accentTextColor">发布了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Insider Preview Build 22000.184 更新</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/0fc673f4-a419-4ce0-b3e0-77beccc37071.jpg@s_2,w_820,h_461" title="微软 Win11 测试版 Build 22000.184 发布，带来任务栏、开始菜单 Bug 解决方法" referrerpolicy="no-referrer"></p><h2>修复</h2><ul class=" list-paddingleft-2"><li><p>修复了 OOBE 中 Windows Hello 的了解更多信息未翻译为非英语语言的问题。</p></li><li><p>修复了在 Windows 11 中 UI 一小部分缺少语言翻译的问题。</p></li></ul><h2>已知的问题</h2><ul class="ai-word-checked list-paddingleft-2"><li><p>微软正在调查测试用户在 Beta 频道中的报告。在升级到 Windows 11 后，有测试用户无法看到全新任务栏，且开始菜单不工作。</p></li></ul><p>IT之家了解到，微软表示，要解决这一问题，受影响的用户可尝试进入 Windows Update-Update history，卸载 Windows 的最新累积更新，并通过检查更新重新安装。</p><ul class=" list-paddingleft-2"><li><p>此外，微软正在修复一个问题，导致部分 Surface Pro X 出现 WHEA_UNCORRECTABLE_ERROR 错误检查的问题。</p></li></ul><p><strong>开始</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>在部分情况下，用户从开始或任务栏搜索时可能无法输入文本。如果遇到该问题，用户可通过 WIN + R 启动运行对话框，然后将其关闭。</p></li><li><p>右键单击“开始”按钮 (WIN + X) 时缺少系统和 Windows 终端。</p></li></ul><p><strong>任务栏</strong></p><ul class=" list-paddingleft-2"><li><p>切换输入法时，任务栏有时会闪烁。</p></li></ul><p><strong>搜索</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>单击任务栏上的搜索图标后，搜索面板可能无法打开。如果发生这种情况，用户需重新启动“Windows 资源管理器”进程，然后再次打开搜索面板。</p></li><li><p>搜索面板可能显示为黑色，并且在搜索框下方不显示任何内容。</p></li></ul><p><strong>小部件</strong></p><ul class=" list-paddingleft-2"><li><p>小部件小部件面板可能显示为空。要解决该问题，用户可以注销然后重新登录。</p></li><li><p>小部件可能会在外部显示器上显示错误的尺寸。如果遇到这种情况，用户可以先在 PC 显示器上通过触摸或 WIN + W 快捷方式启动小部件，然后在辅助显示器上启动。</p></li></ul><p><strong>微软商店</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>微软将继续努力提高商店中的搜索相关性。</p></li></ul><p><strong>Windows 沙盒</strong></p><ul class=" list-paddingleft-2"><li><p>在 Windows Sandbox 中，单击任务栏上的切换器图标后，语言输入切换器不会启动。一种解决方法是，用户可以通过以下键盘快捷键切换输入语言：Alt + Shift、Ctrl + Shift 或 Win + Space（第三个选项仅在 Sandbox 全屏时可用）。</p></li><li><p>在 Windows Sandbox 中，单击任务栏中的 IME 图标后，IME 上下文菜单不会启动。作为变通方法，用户可以使用以下任一方法访问 IME 上下文菜单的功能：</p></li></ul><p>・ 通过设置 > 时间和语言 > 语言和区域 > 三点 > 语言选项 >（如 Microsoft IME）三点 > 键盘选项访问 IME 设置。或者，用户还可以启用 IME 工具栏（一个替代 UI）来快速调用特定的 IME 功能。从上面继续，键盘选项 > 外观 > 使用 IME 工具栏。</p><ul class="ai-word-checked list-paddingleft-2"><li><p>使用与每种 IME 支持的语言相关联的一组键盘快捷键。</p></li></ul>
