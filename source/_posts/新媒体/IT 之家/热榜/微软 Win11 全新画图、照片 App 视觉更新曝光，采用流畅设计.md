<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 2 日消息 外媒 Windows Latest 报道，微软画图和照片 App 的视觉更新已经公布，展示了一个与 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 设计语言相匹配的新的现代界面。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/4d33a100-555f-4195-a4e3-cb871b8adaf2.jpg@s_2,w_820,h_590" w="1229" h="884" title="微软 Win11 全新画图、照片 App 视觉更新曝光，采用流畅设计" srcset="https://img.ithome.com/newsuploadfiles/2021/8/4d33a100-555f-4195-a4e3-cb871b8adaf2.jpg 2x" width="1229" height="590" referrerpolicy="no-referrer"></p><p>对于 Windows 操作系统和微软的应用程序来说，早就应该有全新的外观了，特别是由于 Metro 界面和 Ribbon 菜单已经变味了。值得庆幸的是，微软正在为 Windows 11 的几乎所有库应用进行新的设计。列表中包括时钟、录音机、计算器、Office、记事本、照片，甚至是微软画图应用。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/d9009ea3-d39d-453e-84cd-68af49ecbe58.jpg" w="768" h="432" title="微软 Win11 全新画图、照片 App 视觉更新曝光，采用流畅设计" width="768" height="432" referrerpolicy="no-referrer"></p><p>IT之家获悉，微软现在已经悄悄地发布了两个最重要的第一方 Windows 应用程序的新外观--画图和照片。微软画图抛弃了传统的 Ribbon 菜单，采用了类似于新的文件资源管理器的流畅标头，而且与 Windows 11 操作系统的其他部分一致。</p><p>正如你在图片中所看到的，Windows 11 的画图应用现在具有简化的工具栏。现有的功能区界面已被新的流畅标头所取代，它带有所有重要的选项，如“文件”、“编辑”、画笔等。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/1039d08d-2969-45a6-95d1-98b64dbd37a7.jpg@s_2,w_820,h_542" w="1280" h="846" title="微软 Win11 全新画图、照片 App 视觉更新曝光，采用流畅设计" srcset="https://img.ithome.com/newsuploadfiles/2021/8/1039d08d-2969-45a6-95d1-98b64dbd37a7.jpg 2x" width="1280" height="542" referrerpolicy="no-referrer"></p><p>在标头中，你可以找到简单的图标来执行基本任务（如在画笔和铅笔之间切换、复制、粘贴等）。撤销/重做按钮现在出现在页眉的“文件”和“查看”选项旁边。当然，作为 Windows 11 改版的一部分，画图现在具有圆角和 WinUI 控件。</p><p>对于微软画图来说，看起来不会有新的功能或改进，更新的目的是为了获得全新的外观。然而，我们可能会在今年晚些时候了解到更多关于该更新的信息。</p><p>下面是当前版本的 Windows 11 上的 画图：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/7b6549db-c7b6-403b-95be-579b481804c3.png" w="921" h="725" title="微软 Win11 全新画图、照片 App 视觉更新曝光，采用流畅设计" width="921" height="645" referrerpolicy="no-referrer"></p><p>正如一开始提到的，我们还将得到新的微软照片应用程序，它有一个新的界面，以及改进，和更好的性能。照片应用将获得新的编辑体验，编辑工具将漂浮在图像上方，类似于<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>或 iOS 移动应用。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/25341aa6-0c5e-48ab-bec0-3f6427ca9b91.jpg@s_2,w_820,h_603" w="1253" h="922" title="微软 Win11 全新画图、照片 App 视觉更新曝光，采用流畅设计" srcset="https://img.ithome.com/newsuploadfiles/2021/8/25341aa6-0c5e-48ab-bec0-3f6427ca9b91.jpg 2x" width="1253" height="603" referrerpolicy="no-referrer"></p><h2>新的 Windows 11 应用程序即将推出</h2><p>微软将在未来几周内开始为 Windows 11 推出更新的库应用程序，以获得消费者的反馈。</p><p>除了这些应用，微软正在为 Windows 11 开发新的 Outlook 和 Microsoft Teams 客户端，这些应用将基于 Microsoft Edge WebViewer。</p><p>微软 Teams 2.0 应用程序已经提供给 Windows Insiders，微软重新设计的 Outlook 应用程序（预览版）将于今年晚些时候发布。</p><p>微软还计划在 Insider 计划的 Dev 开发频道发布 Windows 11 22H2（明年的功能更新），但新功能可能不包括在初始构建版本中。</p>
