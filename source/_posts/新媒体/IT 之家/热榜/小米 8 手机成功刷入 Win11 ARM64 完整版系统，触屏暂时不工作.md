<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 29 日消息 今天凌晨，微软面向 Dev 开发频道发布了第一个 Windows 11 Insider Preview 版本，即 Build 22000.51!  微软表示，随着我们在未来几个月内完成产品的定型，我们将与你一起验证这一体验。在这个早期预览中，<a href="https://www.ithome.com/0/559/232.htm" target="_blank">你将可以尝试上周展示的许多新功能</a>，但不是全部。</p><p>微软 Windows 11 Insider Preview 版本 Build 22000.51 镜像也已流出，其中包括 Windows 11 ARM 预览版本，有开发者已经在 Lumia 950 XL 刷入了 Windows 11 ARM 预览版本，现在有IT之家网友表示，搭载骁龙 845 处理器的小米 8 手机也已成功刷入 Windows 11 ARM64 完整系统，不过该机型上的 USB、触屏还不能正常工作，无法控制。</p><p>下面是小米 8 手机进入 Windows 11 ARM64 系统后的桌面，“开始”菜单等在任务栏居中显示。</p><p><img src="https://qimg.ithome.com/image/2021/06/29/6376060405020178756528607.jpg" title="小米 8 手机成功刷入 Win11 ARM64 完整版系统，触屏暂时不工作" referrerpolicy="no-referrer"></p><p><img src="https://qimg.ithome.com/image/2021/06/29/6376060405354718962318179.jpg" title="小米 8 手机成功刷入 Win11 ARM64 完整版系统，触屏暂时不工作" referrerpolicy="no-referrer"></p><p>下面是视频：</p><p><iframe class="ithome_video bilibili" src="https://player.bilibili.com/player.html?bvid=bv1M44y1q7kc&page=1" frameborder="no" width="100%" allowfullscreen="true"> </iframe></p><p>项目地址: https://github.com/edk2-porting/edk2-sdm845</p><p>微软官方此前早已经对 Windows 10 Mobile 停止支持，不过在 WoA 项目开发者的努力下，仍然会有开发者去为旧机型刷入新系统。</p><p>由于微软提供了 ARM 设备兼容的 Windows 10 版本，一群独立开发人员在过去几年里一直在维护一个非官方的 Lumia Windows on ARM 项目，这就是 Lumia WOA。现在微软发布了 Windows 11 ARM 预览版本，又有新的系统可以尝试了。</p>
