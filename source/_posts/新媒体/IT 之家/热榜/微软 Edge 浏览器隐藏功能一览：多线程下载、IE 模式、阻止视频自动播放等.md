<p data-vmark="86d0">不知不觉，微软 Edge 浏览器的版本号也跟随着 Chromium，狂奔到 100 + 了。<strong>Edge 浏览器抛弃自研内核，转用 Google 主导的 Chromium</strong>，这一行为自然颇具争议，但不可否认的是 Edge 的功能终于变得可以和 Chrome 平起平坐。不过 Edge 换用了 Chromium 这么久，大家真的有把它的功能都发掘出来吗？其实 Edge 有很多深藏其中却又相当实用的小功能，今天带大家一起来发掘一下吧！</p><p data-vmark="0c3c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/fe0ca529-6129-4682-9ffc-c084d346a67a.png" w="674" h="415" alt="微软Edge浏览器" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="674" height="415" referrerpolicy="no-referrer"></p><h2 data-vmark="2700">标签页预览</h2><p data-vmark="76ce">使用 Chromium 内核的浏览器，无论是 Chrome 还是 Edge，都有一个硬伤，那就是浏览器的标签页开启多个后，这些标签页有可能会挤压在一起，网页标题完全看不清，效果大家可以从下面的图片感受一下。</p><p data-vmark="4da3" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/4a9976ef-e8f5-4e9f-89a0-6d0e5f6e64d0.png" w="700" h="472" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="472" referrerpolicy="no-referrer"></p><p data-vmark="12a5">要如何解决这个问题？实际上 Edge 就带有一个自带的解决方案 —— 标签页预览。</p><p data-vmark="0ba3">顾名思义，标签页预览这个特性在开启后，<strong>只要将鼠标悬停在标签页上，就会出现标签页所开启网页的预览图</strong>。无论是网站的标题还是网页内容，都看得一清二楚，无需再担心分不清哪个标签页了！</p><p data-vmark="7061" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/10d39855-2d9c-47c4-bf6f-fab1d471b634.png" w="700" h="474" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="474" referrerpolicy="no-referrer"></p><p data-vmark="1223">Edge 默认并没有打开这个功能，要开启的方法也很简单。在 Edge 的地址栏输入：</p><pre class="brush:javascript;toolbar:false">Edge://flags</pre><p data-vmark="ad27">然后找到“Tab Hover Card Images”这个特性，改为“Enable”即可。</p><p data-vmark="f851" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/e0882053-3cae-4776-880f-689910cfc048.png" w="700" h="472" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="472" referrerpolicy="no-referrer"></p><h2 data-vmark="0aea">多线程下载</h2><p data-vmark="f2b1">Edge 有一点让很多朋友都感到不太满意 —— 下载速度太慢了！这主要是由于 Edge 浏览器默认使用单线程下载，只对资源发起单个连接，速度自然上不去。有的朋友可能会说了，用迅雷之类的第三方下载工具，不就解决了么？但某些网站是禁止下载器的，而且保存图片之类的操作，调用的也是 Edge 的下载模块，所以在很多情况下，Edge 下载速度慢这点，还是会带来使用上的困扰的。</p><p data-vmark="9767">怎么办？其实 Edge 是支持多线程下载模式的！但同样，Edge 默认也没有开启这个功能，我们<strong>需要进入到 Edge://flags 当中开启它</strong>。</p><p data-vmark="ab4c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/8fe8ef91-9cef-45d1-9e09-7beb149788a5.png" w="700" h="472" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="472" referrerpolicy="no-referrer"></p><p data-vmark="457b">操作和上文类似，在地址栏输入 edge://flags 后进入到实验功能界面，找到“Parallel downloading”的选项，打开“Enable”即可。之后，Edge 的下载速度就会有明显提升了。</p><h2 data-vmark="a470">Win11 样式滚动条</h2><p data-vmark="7f92">微软已经推出 Windows 11 一段时间了，很多软件都已经适配了 Win11 的外观，但 Edge 浏览器却并不在此列。例如，Edge 浏览器的滚动条仍然是 Win10 的样式，方方正正，和 Win11 的画风迥然不同。</p><p data-vmark="a16a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/049296e6-db70-4143-9288-703597f25813.png" w="700" h="569" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="569" referrerpolicy="no-referrer"></p><p data-vmark="95b3">看看 Win11 设置面板的滚动条，这才是 Win11 的样式。</p><p data-vmark="46af" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/150ba50b-473e-4556-8753-72b5f126ee42.png" w="700" h="485" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="485" referrerpolicy="no-referrer"></p><p data-vmark="71a4">不过，<strong>Edge 浏览器其实是支持 Win11 样式滚动条的</strong>，不过默认并没有开启这一特性。进入到 edge://flags 界面后，查找“Windows style overlay scrollbars”，改成“Enable”，滚动条就变成了细长的现代 Windows UI 风格了！</p><p data-vmark="0fa4" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/67dfdd1d-3c9f-401d-a923-7ec490fef886.png" w="700" h="472" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="472" referrerpolicy="no-referrer"></p><p data-vmark="f342" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/fdd72c8e-bdf4-4a14-b059-d6e4c0db6ddf.png" w="700" h="572" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="572" referrerpolicy="no-referrer"></p><h2 data-vmark="6325">平滑滚动</h2><p data-vmark="0835">旧版 Edge 和 Chrome 相比的一大优势，就在于平滑滚动，而<strong>新版的 Edge 由于使用 Chromium 的代码</strong>，因此微软也将平滑滚动的算法贡献了一些给 Chromium，改善了新版 Edge 乃至 Chrome 的平滑滚动体验。但是如果你觉得现在的 Edge 依然不够平滑，怎么办？</p><p data-vmark="6780" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/3c20dc94-a2f6-44d0-83e0-33f06b6fcf1b.png" w="700" h="457" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="457" referrerpolicy="no-referrer"></p><p data-vmark="cbb5">在 edge://flags 当中，可以找到“Smooth Scrolling”这个选项，<strong>将其设置为“Enable”即可</strong>。开启后，按理来说网页滑动会更加顺畅，但也有朋友反馈说没什么区别，大家可以自己试试。</p><h2 data-vmark="d94c">阻止视频自动播放</h2><p data-vmark="ae3a">某些网页视频非常烦人，会自动播放，猛一下被视频声音吓一跳，实在令人抓狂。其实 Edge 浏览器暗藏了阻止视频自动播放的功能，相当实用。</p><p data-vmark="2a14">Edge 的阻止视频播放功能开关默认可能并不显示（也有可能显示），我们需要先进入到 edge://flags 页面，找到“Show block option in autoplay settings”，将其设置为“Enable”。</p><p data-vmark="508f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/92086861-3300-4879-a5f2-da2b11cf8e01.png" w="700" h="457" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="457" referrerpolicy="no-referrer"></p><p data-vmark="a923">重启浏览器后，在 Edge 的“设置-Cookie 和网站权限”页面中，找到“站点权限”下的“媒体自动播放”，将其设置为“阻止”即可。</p><p data-vmark="aaf3" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/9a6ec3cf-1917-4092-88e4-8584d9651eba.png" w="700" h="406" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="406" referrerpolicy="no-referrer"></p><h2 data-vmark="6e25">数学求解器</h2><p data-vmark="a801">作为一个浏览器，Edge 还可以算数学题，没想到吧？这可不是简单的加减乘除，<strong>是可以解方程、算积分的超强功能</strong>！</p><p data-vmark="6181">在 Edge 浏览器的设置中，进入“外观”页面，找到“选择要在工具栏上显示的按钮”，其中有“数学求解器按钮”选项，打开它，数学求解器就会出现在工具栏了。</p><p data-vmark="7c3b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/2a2fc751-fc04-4aeb-a1b1-61fe5501d3bf.png" w="700" h="406" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="406" referrerpolicy="no-referrer"></p><p data-vmark="9015">这个数学求解器，可以截屏网页中的数学算式并进行求解。点击后，用方框选择网页中的数学式，Edge 就会自动解出答案，甚至还会列出步骤！可以说相当黑科技了。</p><p data-vmark="a45a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/7135b75c-8509-47be-8ac0-25cf18dfa4e0.png" w="700" h="503" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="503" referrerpolicy="no-referrer"></p><p data-vmark="2d1b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/eae1bda7-0835-4494-98ae-db13adabe585.png" w="700" h="503" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="503" referrerpolicy="no-referrer"></p><p data-vmark="aa99" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/9df7605d-517a-440f-807a-d3aeb688217f.png" w="700" h="503" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="503" referrerpolicy="no-referrer"></p><h2 data-vmark="7b19">IE 模式</h2><p data-vmark="b5d7">Win11 已经将 Edge 设置为了默认浏览器，并去除了 IE 的入口。但问题来了，时至今日，仍有部分网站只兼容 IE，怎么办？其实 <strong>Edge 内置了 IE 模式</strong>，可以解决这个难题。</p><p data-vmark="0a1c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/cdc16372-edd4-4697-aff2-61937e7b9c10.png" w="700" h="483" alt="Edge网银 IE模式" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="483" referrerpolicy="no-referrer"></p><p data-vmark="4a36">首先，我们需要 Edge 浏览器的设置中，找到“默认浏览器”的大项，即可看到“允许在 Internet Explorer 模式下重新加载网站”的选项，将其设置为“允许”。</p><p data-vmark="3034" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/926ada79-70e1-4799-9ebb-c03c3aadc59c.png" w="700" h="468" alt="Edge网银 IE模式" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="468" referrerpolicy="no-referrer"></p><p data-vmark="df63">接着，不要直接在这个设置页面中直接设置 IE 模式的页面，因为这里的可选项不多，而且域名指定不够宽泛。我们需要打开 Edge 浏览器的企业网站列表管理器，具体方式是在地址栏输入以下字符串并进入。</p><pre class="brush:javascript;toolbar:false ai-word-checked">edge://compat/sitelistmanager</pre><p data-vmark="fb38">在其中，<strong>我们就可以配置启用 IE 模式的站点了</strong>。例如想要让建设银行使用 IE 模式，那么添加“ccb.com”后，此后无论建设银行的网址的前缀是 http 还是 https，无论网址的后缀是哪个子页面，IE 模式都可以生效。</p><p data-vmark="f008" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/3b7a1373-8048-47df-8070-3baf372a2f5c.png" w="700" h="472" alt="Edge网银 IE模式" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="472" referrerpolicy="no-referrer"></p><p data-vmark="fbbc" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/9e6ac89d-01a1-4ab4-acd4-213c01a49bd9.png" w="700" h="490" alt="Edge网银 IE模式" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="490" referrerpolicy="no-referrer"></p><p data-vmark="abcd">Edge 的 IE 模式非常实用，但设置的确不够方便，希望之后 Edge 能将其设为一个一键切换的按钮，体验会更好。</p><h2 data-vmark="0078">强制阅读模式</h2><p data-vmark="e1fb">我们知道 Edge 是支持阅读模式的，而且效果还相当不错。但很多时候，这个阅读模式需要 Edge 自己判断是否能打开，与此同时 Edge 的判断标准又过于严格，这导致很多网页其实在阅读模式下效果很不错，但偏偏无法被 Edge 识别开启。</p><p data-vmark="2d71">这个问题有个很简单的解决方法，那就是在 Edge 地址栏中，<strong>为网页 URL 前面加上一个“read:”前缀</strong>，然后按下回车键，阅读模式就强制打开了！</p><p data-vmark="ca5a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/93c786e9-0c0f-482f-adf4-f45d76e2ab97.png" w="700" h="503" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="503" referrerpolicy="no-referrer"></p><p data-vmark="3e2f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/752e0ef7-eb37-4b50-a26a-dae853500c52.png" w="700" h="503" alt="Edge隐藏功能 Edge隐藏设置" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="503" referrerpolicy="no-referrer"></p><h2 data-vmark="17f4">让笔记本电脑更省电</h2><p data-vmark="9ebf">和台式机不同，笔记本电脑会工作在电池模式下，省电就必须重视起来了。Edge 上有一个开关，可以让笔记本工作在电池模式下变得更加省电。</p><p data-vmark="be9b">首先，<strong>Edge 支持标签页睡眠</strong>，因此我们首先需要在设置中打开“使用标签页休眠功能节约资源”以及“淡出睡眠标签页”功能，并设置指定一段时间后，标签页进入睡眠的时间。如此一来，当标签页持续一段时间不活动后，即会进入到睡眠状态，大大减少 CPU 的使用，节省电量。</p><p data-vmark="6c0d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/6019ab87-70dc-44e5-9046-090285f607e9.png" w="700" h="535" alt="Edge浏览器耗电 Edge耗电" title="微软 Edge 浏览器隐藏功能一览：多线程下载、IE 模式、阻止视频自动播放等" width="700" height="535" referrerpolicy="no-referrer"></p><p data-vmark="f4d0">关键的一步来了，我们还需要设置“在以下情况下启用效率模式”为“已拔出”。如此设置后，当笔记本使用电池时，处于非活动状态的标签页在 5 分钟内就会进入到睡眠，对于笔记本电脑的续航有显著好处。</p><h2 data-vmark="0861">总结</h2><p data-vmark="5475">总的来说，Edge 浏览器的很多功能都做得相当不错，整体体验笔者认为是比 Chrome 要更好的，在国内尤其如此。上文的这些 Edge 隐藏的小特性，可以让浏览器的使用体验更上一层楼，有需要的朋友不妨折腾一下吧！</p>
