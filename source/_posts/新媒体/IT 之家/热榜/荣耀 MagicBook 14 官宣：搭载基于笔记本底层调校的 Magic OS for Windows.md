<p data-vmark="c432"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 9 日消息，荣耀方面现宣布，荣耀笔记本新品发布会将于 5 月 16 日 14:30 举行，包括即将到来荣耀 MagicBook 14。</p><p data-vmark="50ed">据介绍，这是第一款搭载 Magic OS for Windows、PC 端首秀 OS Turbo 技术的全新荣耀笔记本！“每一次突破都是自我成长，每一次升级都在变得更强！畅快体验，性能时刻在线”。</p><p data-vmark="f25e">荣耀还表示，Magic OS for Windows 基于笔记本底层调校！搭载 Magic OS for Windows 的荣耀笔记本会在主动服务、协同服务、基础能力三个方面带来更惊喜的使用体验，敬请期待！</p><p data-vmark="6c14">此外，未来荣耀也将以 Magic OS 为基石，给大家带来更优秀的智慧生活产品、更多的智慧生活新体验。</p><p style="text-align: center;" data-vmark="d0ea"><img src="https://img.ithome.com/newsuploadfiles/2022/5/3de8151c-7ef7-4068-acf6-8a8d2734a978.jpg@s_2,w_820,h_1458" w="1440" h="2560" title="荣耀 MagicBook 14 官宣：搭载基于笔记本底层调校的 Magic OS for Windows" srcset="https://img.ithome.com/newsuploadfiles/2022/5/3de8151c-7ef7-4068-acf6-8a8d2734a978.jpg 2x" width="1440" height="1458" referrerpolicy="no-referrer"></p><p data-vmark="ea52">此外，“荣耀老熊”对 Magic OS for Windows 进行了进一步的解释，称其 Magic OS for Windows 大致由主动服务的场景化体验、智能协同的跨设备 OS、个性化的基础 OS 构成，将助力 PC 产品性能更强、功耗更低。“荣耀老熊”还表示，未来荣耀将以赋能 Windows 为起点，继续秉持“双轮驱动”开发理念……</p><p data-vmark="4d9c">IT之家此前报道，据华为 / 荣耀系爆料博主“勇气数码君”、“厂长是关同学”消息，荣耀新款笔记本定位“小旗舰”，将在不久后发布。预计荣耀新款笔记本将可选 12 代酷睿处理器或锐龙 6000 系列处理器。</p>
