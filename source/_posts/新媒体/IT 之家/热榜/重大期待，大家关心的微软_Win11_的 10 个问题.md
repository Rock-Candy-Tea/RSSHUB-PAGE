<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 5 日消息 微软 Windows 11 正在向我们走来吗？两天前，微软宣布将在在 6 月 24 日举行活动，公布“下一代 Windows”。到目前为止，还不能 100% 确定微软将发布 Win11，但一些迹象标明，这将比目前的任何 Windows 10 都令人感到期待。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/18a1ca18-1fab-4766-8a88-36e167950f1a.jpg@s_2,w_820,h_384" w="1440" h="675" title="重大期待，大家关心的微软“Win11”的 10 个问题" srcset="https://img.ithome.com/newsuploadfiles/2021/6/18a1ca18-1fab-4766-8a88-36e167950f1a.jpg 2x" width="1440" height="384" referrerpolicy="no-referrer"></p><p>IT之家获悉，关于 Windows 新版本的消息最初是通过泄露的 Windows 10X 开始的：这是一款轻量级的、更新的 Windows 10 版本，首先是为 Surface Neo 等双屏笔记本或平板电脑准备的。Windows 10X 起初只限于双屏使用。</p><p>此后因为 Windows 10 ARM 可靠运行（指完全支持 Win32 应用）方面的问题持续存在，加上疫情办公影响，导致 Windows 10X 和 Surface Neo 设备开发方面大幅度跳票。Windows 10X 即使是在单屏幕上，也从未想过要成为 Windows 10 的“替代品”：它将只在新硬件上出现，并被定位为在教育领域与 Chromebook 竞争。但最终微软做了决定，“砍掉”Windows 10X，并将其新特性添加到普通版 Windows 10 上，这也许就是将要发布的 Windows 11。</p><p>Windows 10“太阳谷”是 Windows 10 21H2 版本的代号，将作为 Windows 10X 的修订版，有全新的图标，新的“开始”菜单，以及更加一致的用户界面。</p><p><strong>1. 名字到底是什么？</strong></p><p>微软预告 GIF 动图中的“Windows 11”的影子已经引起了很多人的注意，把“下一版本的 Windows”称为 Windows 11 是想当然的。其他人则倾向于只叫它“Windows”。当然，即使有这么多猜测，最终仍可能只是“Windows 10”的一些变体。它应该是 Windows 的最后一个版本，对吗？不过，在 2015 年 Ignite 会议上，已经有人说过 Windows 10 是“最后一个版本”。</p><p><strong>2. 完全不同的东西？</strong></p><p>暗示我们将迎来“新一代的 Windows”，但我们应该期待多大的变化？上周发现的一份招聘显示，微软承诺“一个彻底改变 Windows 用户体验平台的多年旅程”，虽然微软对 6 月 24 日的公告大做文章，但认为我们在 21H2 的时间框架内就拥有全新的、不同的 Windows 似乎很牵强。到目前为止，我们听到的关于“太阳谷”的一切消息是，虽然它有新的、刷新的外观设计，但其核心基础还是 Windows 10 系统。</p><p>这更多来自 Windows 10X 中诞生的新鲜干净的外观设计，但是我们真的能期待完全统一协调的的 UI/UX 改造吗？</p><p><strong>3. 企业怎么办？</strong></p><p>长期以来，微软一直坚持忠于传统的 Windows，这是有原因的。企业一般都讨厌变化，因为要重新制作培训材料、帮助服务台和 IT 管理以适应 Windows 的变化，这涉及到真正的经济痛苦。</p><p>作为消费者，我们都想要最新、最新鲜的外观，但这正是企业不想要的。</p><p><strong>4. 一条新路，还是一条岔路？</strong></p><p>在本周的 Windows 周刊播客中，Mary Jo Foley 和 Paul Thurrott 对刚刚发布的 6 月 24 日即将发布的消息有很多话要说。不过，Mary Jo Foley 表示，“这里面有很多障眼法，加上一个新的商店”，她还猜测预计今年秋天会看到 21H2 以及新的 Windows“11”。她的理由是，企业并不关心任何这些用户界面的东西，他们会想坚持使用 Windows 10。</p><p>法国博客 Frandroid 发布了一张据说是 Windows 11 系统菜单的截图，并推测会有两个版本，但在这一点上只是猜测而已。有可能新版本会优先面向 seeker 尝鲜测试者推送。</p><p><strong>5. 功能体验包是答案吗？</strong></p><p>虽然从技术上讲，目前的 Windows Insider Dev 开发频道正在向太阳谷方向准备，但微软正在努力使新的用户体验功能成为“秘密”武器。这些功能可能会作为 Windows 功能体验包的一部分，尽可能长时间地保持新内容的保密性。</p><p>然而，这也可能意味着，内幕人士和消费者吵着要的太阳谷 / Windows 10X/Windows 11 功能，企业几乎肯定会讨厌，可能会作为模块化 Windows 的一部分出现。想要最新和最棒的功能吗？安装功能体验包就可以了。而企业 IT 部门则不需要安装这个体验包，或者推出简化版的企业 Windows 功能体验包。</p><p><strong>6. Windows“11”会销售吗？</strong></p><p>有一件事几乎是每个人都同意的：Windows 的新命名方案对试图销售新设备的 OEM 厂商来说将是一个福音。OEM 更喜欢能够在他们的市场推广和宣传页中写上“全新的 Windows 11 !!!”，而且随着个人电脑的销售在疫情的推动下重新崛起，有新的和闪亮的东西可以销售将使 OEM 厂商非常高兴。</p><p><strong>7. 太阳谷就是 Windows 11 的团队整合？</strong></p><p>Mary Jo Foley 提到，微软已经透露了更多关于 6 月 24 日的预期内容。微软称，“新的软件将包括设计上的变化，通过 Windows 应用商店为创作者和开发者增加机会，以及在用户和他们关心的社区之间更容易建立联系的方法”。</p><p><strong>8. Windows 内测人员是否可以测试新的 Windows 11？什么时候？</strong></p><p>微软再次提供了一些提示，称“该软件将在活动结束后向注册测试新产品的所谓 Windows Insiders 推出”。</p><p><strong>9. (就像 Windows 一样，我们这次也跳过了 9)</strong></p><p><strong>10. 还有一件事，关于新的微软商店是怎么回事？</strong></p><p>萨提亚・纳德拉在 Build 大会上提到“下一代 Windows”的一分钟时间里，其强调，“很快，我们将分享过去十年中对 Windows 最重要的更新之一，为开发者和创作者释放更大的经济机会。”</p><p>当然，你可以把这句话理解为“过去十年中对 Windows 最重要的更新之一”，但听起来纳德拉肯定是在具体谈论微软商店的变化。早在今年 4 月，微软宣布它将把 Windows 10 商店中的游戏销售提成从 30% 降至 12%，更多关于商店即将发生的变化的信息已经浮出水面，远不止是太阳谷用户界面的更新。据 Zac Bowden 表示，新的商店将有以下大的变化：</p><ul class=" list-paddingleft-2"><li><p>允许开发者向商店提交未打包的 Win32 应用程序</p></li><li><p>允许开发者在自己的内容交付网络（CDN）上托管应用程序和更新</p></li><li><p>允许开发者在应用程序中使用第三方商务平台</p></li></ul><p>现在，微软曾经试图重塑应用商店的模式，最引人注目的是 Windows 10 Mobile 的 UWP 应用，但结果我们已经知道了。最近，微软还试图推动 PWA（网络渐进式应用），作为突破苹果和谷歌商店围墙的一种方式，但没有取得多大成功。然而，即使没有新的销售模式，微软商店也急需改革，所以说这只会是一个好消息。</p>
