<p data-vmark="755a"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 18 日消息，蔚来 NIO Day 2021 将于今晚 19:00 在苏州举办。届时，蔚来备受关注的第二款轿车（或定名 ET5）将迎来正式发布。</p><p data-vmark="3789"><img src="https://img.ithome.com/newsuploadfiles/2021/12/bc05a42f-e019-45b8-b700-b2c2aeeb714b.jpg" w="630" h="476" alt title="蔚来 ET5 今晚发布，高管暗示新车将强于特斯拉 Model 3 / 宝马 3 系" width="630" height="476" referrerpolicy="no-referrer"></p><p data-vmark="c6e0">蔚来企业传播高级总监马麟在活动前夕发文预告称，“今夜，NIO Day 点亮苏州夜空。期待大家的莅临。明晚，蔚来第五款量产车发布后，XXXXXX 市场就要变天了。”<span class="accentTextColor"> 同时强调“#5 大于 3”，或暗示新车将超越宝马 3 系或特斯拉 Model 3</span>。</p><p data-vmark="b0ae"><img src="https://img.ithome.com/newsuploadfiles/2021/12/f0cc383b-4198-444f-b708-0fe2f0a54cc5.png" w="833" h="558" alt="ET7 " title="蔚来 ET5 今晚发布，高管暗示新车将强于特斯拉 Model 3 / 宝马 3 系" width="833" height="549" referrerpolicy="no-referrer"></p><p data-vmark="73f6">在今年年初举办的 NIO Day 期间，蔚来首款轿车产品 ET7 正式发布并公布售价。新车价格定为 44.8-52.6 万元，定为一款中大型车，预计会在 2022 年一季度正式交付。</p><p data-vmark="25fc"><img src="https://img.ithome.com/newsuploadfiles/2021/12/bca642b2-9ecb-4f00-9ba2-772a748a281e.jpg" w="630" h="840" alt="ET5" title="蔚来 ET5 今晚发布，高管暗示新车将强于特斯拉 Model 3 / 宝马 3 系" width="630" height="840" referrerpolicy="no-referrer"></p><p data-vmark="4c40">今晚蔚来将发布的新车很可能是此前传闻的 ET5，相比定位豪华中大型纯电轿车的蔚来 ET7，蔚来 ET5 车身尺寸要略小一些，预计车长在 4800mm-4900mm 之间，轴距在 2800mm-2900mm 之间，价格预计直指特斯拉 Model 3 所在区间。</p><p data-vmark="c960">IT之家了解到，从 2017 年至今，蔚来以每年一款新车的稳定频率在 NIO Day 上推出当年主打车型。</p>
