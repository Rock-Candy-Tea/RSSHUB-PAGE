<p>不得不说，此次 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 的发布还真是戳中了很多人关注点，虽然一直被吐槽果里果气的，可…… 大家似乎都挺受用。其实此次 Win11 升级，除了统一外观 UI 外，一个很重要的目标就是提高平板用户的操作体验。众所周知，微软一直都是生产力的代言人，而苹果则是人性化交互的倡导者，那么此次 Win11 提升了交互界面，它和苹果的巅峰对决，又将鹿死谁手呢！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/a33a213a-a8f2-40ae-8e5a-8504ff6acaee.jpg" w="700" h="394" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="700" height="394" referrerpolicy="no-referrer"></p><h2>1. 更不容易误点的“任务栏”</h2><p>任务栏此次被吐槽得不轻，一个很重要原因，是它更改了很多人们之前习惯的操作逻辑。不过很多人并未发现，当我们将键盘与平板电脑分离时，任务栏上的图标也会自动扩大一丢丢。正是这一丢丢的距离，让点按时更不容易误触了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/4344f7db-9117-4db3-a67f-8d86e5c06034.gif" w="601" h="426" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="601" height="426" referrerpolicy="no-referrer"></p><p>不过相比苹果的 Dock 栏，Win11 的任务栏还是显得小了一些，特别是没法修改大小这个“Bug”，着实让人难受。好在，这只是个对比，如果你一直都是微软的 Fans，慢慢也就适应了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/4eb69be6-b928-4bf1-9d6d-7ce7386ce88f.jpg" w="663" h="195" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="663" height="195" referrerpolicy="no-referrer"></p><h2>2. 操作中心</h2><p>操作中心的改进同样明显，不得不说 Win11 这一次的确带给了我们不一样的体验。和之前的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 通知中心相比，新操作中心不光结构紧凑，运行逻辑也要明显强于老版。特别是新增加的开关按钮，更加适合触控。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/133cb9c0-25fd-4339-85c1-0b340dcda585.jpg" w="454" h="765" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="454" height="765" referrerpolicy="no-referrer"></p><p>不过与苹果的控制中心相比，Win11 的操作中心能够控制的功能并不多，可新增选项也仅限于之前 Win10 的那几个（甚至还少了个截图）。相比之下，苹果的逻辑更像是通常的手机系统，除了各种控制功能外，你还能根据喜好调出计时器、备忘录、秒表、扫码、手电筒等，在日常生活中也是十分方便的。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/b8df2a34-6464-4c9c-9cf2-87659652fbbe.jpg" w="436" h="713" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="436" height="713" referrerpolicy="no-referrer"></p><h2>3. 夜间模式</h2><p>微软的夜间模式一直都是老大难问题，即便到了 Win11，这种情况也没有好哪儿去！Win11 的夜间模式依旧采用了之前设计，虽然换装了新版资源管理器和新版操作中心，可效果…… 依旧难以直视。而且微软还面临着一个比苹果更大的阻碍。前者开发环境开放，海量的存量应用难以更新，明显限制住了夜间模式推行。而后者则被一直限定在苹果的开发框架内，无论外观还是夜间效果，都要比微软更统一。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/4ce243c2-f2be-4d9d-9b97-071c72e66bc2.jpg" w="700" h="293" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="700" height="293" referrerpolicy="no-referrer"></p><p>此外，苹果还在传统夜间模式基础上，开发出了定时换壁纸和定时切换夜间模式等小功能。这些变化虽然都不足挂齿，可的确能有效提升操作体验。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/4faf794d-1b17-4274-9537-b850c4738751.jpg" w="652" h="638" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="652" height="638" referrerpolicy="no-referrer"></p><h2>4. 触屏手势</h2><p>Win 11 在触屏手势上增加了很多，比如三指下滑显示桌面、三指上滑多任务管理、三指左右滑动切换软件、四指左右滑动切换桌面等。特别是在新动画的加持下，整体效果要强于 Win10。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/822b353a-3e15-467d-93f3-f660bf2b10e4.gif" w="601" h="426" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="601" height="426" referrerpolicy="no-referrer"></p><p>不过和深耕触屏多年的苹果相比，Win11 的手势还是显得羸弱了些。除了手势数量更加丰富外（比如一个动作可以有多组手势），<a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 还将很多系统功能也融入到日常的手势操作里。比如桌面下滑打开快速搜索栏、顶部下滑调出通知中心、三指捏合/松开完成复制粘贴等，几乎覆盖到了日常使用的各个方面。当然这也和两家的发展方向有关，一个更注重于生产力体验（都生产力了当然没必要玩手势了），一个更注重于用户操作逻辑。从这一点上说，Win11 在触屏方面的改进仍然任重而道远。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/11f69165-96ca-4fd3-8606-08245147e519.jpg" w="700" h="218" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="700" height="218" referrerpolicy="no-referrer"></p><h2>5. 动画效果</h2><p>作为系统 UI 的重要组成部分，动画对于现代操作系统来说，已经不单单是好不好看，或者卡不卡顿的问题了。举个最简单例子，同样是滑动多任务窗口，iPad 的动画效果就要明显流畅和跟手，这其中一个奥秘，就是屏幕刷新率和动画惯性在作怪。不过给人的最直观感受，就是苹果系统更流畅，而 Windows 太生硬。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/47d0ade0-7a16-493a-a308-70f545e74348.gif" w="405" h="197" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="405" height="197" referrerpolicy="no-referrer"></p><h2>6. 窗口布局</h2><p>Win11 在原有 4 分屏基础上，增加了随屏幕尺寸自动调整的新分屏方案。在我们之前的测试中，发现在一些超宽屏幕显示器上，Win11 可以自动激活“左/中/右”、“左辅/中主/右辅”两组全新分屏布局。同时此次新增加在最大化按钮上的布局控件，也明显是为触屏用户准备的，除了更加直观外，使用起来也会更便捷。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/0e222bf9-aadc-428c-9543-e7bbbbe0460e.jpg" w="521" h="320" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="521" height="320" referrerpolicy="no-referrer"></p><p>相比之下，iPad 的分屏逻辑就要明显落后于 Win11。先不说沿用多年的双分屏策略，基本就是 Win8 时代玩剩下的。即使是目前最新的 M1 芯片，也最多支持三分屏。当然像 Windows 那样的窗口布局也是可以了，只是…… 需要你自己慢慢去挪。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/d2fbeae6-0abe-459e-b76e-bcc04e26684c.gif" w="599" h="330" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="599" height="330" referrerpolicy="no-referrer"></p><h2>7. App 使用</h2><p>自打 Win8 正式转向平板模式后，微软已经历经了 N 次应用迭代，之前也出现过 Metro、UWP 等不同版本的新应用形式。然鹅…… 最终结果大家也看到了，应用商店不死不活，开发商依旧青睐于为传统应用添砖加瓦。实话说，这的确是微软的一个尴尬，因为传统软件中能够适配触屏的数量极少，而且与 Win11 的 UI 也不是很搭。这也就意味着微软首先要面对的，其实就是自己阵营里的那部分人。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/7ca4f5e0-80e4-4e1d-bb7d-f536717bda90.jpg" w="563" h="498" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="563" height="498" referrerpolicy="no-referrer"></p><p>反观苹果，封闭的生态链体系保证了自家应用，无论在风格还是操作逻辑上都极为统一。即便是个别应用由于历史原因没有得到适配，也可以强制拉伸至全屏使用。当然这里还有苹果完善的生态链保障，一款应用从 iPad 到 macOS，并不会有明显的外观差异。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/5b798052-e4aa-4d64-8e40-fa74ae23601c.gif" w="630" h="609" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="630" height="609" referrerpolicy="no-referrer"></p><p>当然还有一点也是不容忽视的，那就是微软此次在发布会上宣称的兼容 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 应用一事。虽然不能解决数据同步与多端共享问题，但却相当于短时间内上架了海量的可支持触屏的应用。当然这项功能目前也仅限于发布会视频，由于预览版尚未开放此功能，实际体验未知。但对于 Win11 来说，这仍然是一件值得夸耀的事。</p><h2>8. 手写笔</h2><p>微软对于手写笔的定位，仅仅是简单的标注与绘图。一个最明显例子，就是 Apple Pencil 可以直接将书写转为文字。相对而言，微软认为搞定生产力的家伙还得靠键鼠，至于手写笔么，日常玩玩就可以了，不必太认真。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/09d61298-30af-4194-bf9b-9fa491b220bc.jpg" w="700" h="524" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="700" height="524" referrerpolicy="no-referrer"></p><h2>9. 多屏幕管理</h2><p>Win11 对于多屏幕管理进步明显，一个最显著变化，就是当我们将外接显示器移除后，Win11 不会自动回收这些窗口。甚至当显示器重连后，窗口还会自动恢复到以前的位置。这样的表现相信大家也能想到有什么用了，没错！做演示简直爽呆了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/c36c9142-93e4-4d29-bcb6-6cec763da301.gif" w="599" h="355" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="599" height="355" referrerpolicy="no-referrer"></p><h2>10. 小组件</h2><p>Win11 的小组件摆明了就是为了踢苹果场子，此次更新的小组件，无论从外观还是功能使用上，都和苹果极为相似。即便是第一个测试版，也安排上了天气、待办、日程这些与生产力有关系的模块。同时信息流的加入，也可看作是小部件的一大看点。除了支持点赞互动以外，用户感兴趣的内容还可以收藏起来稍后待看。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/463a3310-a535-4fa5-83d3-c95bcba73497.jpg" w="700" h="1012" title="UI 大革新：微软 Win11 界面设计能扳回苹果一局吗" width="700" height="1012" referrerpolicy="no-referrer"></p><p>不过目前 Win11 的小组件使用的是云端数据，并没有和本地 App 打通。比方说你在本地日历建立好的日程，在小组件中是看不到的。这样的逻辑与 iPad 完全不同，因为在苹果中，小部件其实就是 App 的代言人，如果本尊没有了，小部件自然也就不存在了。</p><h2>写在最后</h2><p>总体来说，Win11 此次在触屏方面的改进是比较大的，特别是它没有一味地顺从苹果的思路，依旧保持了熟悉的 Windows 味道，必须给个大大的赞。未来的世界一定是生态链的竞争，这一点上苹果做到了，那么微软呢？拭目以待吧。</p>
