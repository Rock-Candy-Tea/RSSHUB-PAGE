<p data-vmark="feef">很多电脑用户开机后，都习惯打开 Windows 的任务管理器，这并不是为了要随时管理进程，而是任务管理器的小窗可以显示 CPU 的使用率。</p><p data-vmark="f00d">任务管理器会在任务栏显示一个动态的 CPU 占用率图标，非常直观。不过，你可知道查看 CPU 占用还有更好玩的方法！近日笔者发现了一款名叫“<span class="accentTextColor">RunCat</span>”的 CPU 占用查看开源工具，相当的萌，一起来看看！</p><p data-vmark="46d6"><strong>RunCat：</strong></p><p data-vmark="1e95"><a href="https://github.com/Kyome22/RunCat_for_windows" target="_blank"><span class="link-text-start-with-http">https://github.com/Kyome22/RunCat_for_windows</span></a></p><p data-vmark="9f52">这是一款开源的小工具，顾名思义，<span class="accentTextColor">它就是用跑动的小猫来显示 CPU 占用情况的，CPU 占用越高，猫猫跑得越快</span>！RunCat 的体积非常小，只有数百 K，而且是完全绿色无后台的小工具，可以放心使用。</p><p data-vmark="4979">RunCat 的运行需要依托微软的 .NET 5.0 运行环境，如果没有安装，开启 RunCat 后会提示去下载，点击即可跳转到微软官方页面，下载用于桌面应用的版本即可，主要要选择对应的系统版本下载。</p><p data-vmark="fe84"><img src="https://img.ithome.com/newsuploadfiles/2021/12/213eb04b-cbc0-4cd8-9ff4-c40df0078683.png" w="700" h="357" title="在任务栏显示猫猫，超萌的 Win11/Win10 CPU 占用率工具你用过吗" width="700" height="357" referrerpolicy="no-referrer"></p><p data-vmark="68b7">如果打开报错，则选择“是”去下载.NET，下载 desktop App 的版本即可，注意平台选择</p><p data-vmark="340f">安装了.NET 5.0 后，就可以运行 RunCat 了。注意，<span class="accentTextColor">如果你用的是 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 系统，想要在任务栏一直看到猫，那么还需要在系统设置中，将 RunCat 图标设为任务栏常驻</span>。</p><p data-vmark="3bdd"><img src="https://img.ithome.com/newsuploadfiles/2021/12/1b4b06fb-db08-44a5-a577-960152d4ffbe.png" w="700" h="532" alt="任务栏CPU占用" title="在任务栏显示猫猫，超萌的 Win11/Win10 CPU 占用率工具你用过吗" width="700" height="532" referrerpolicy="no-referrer"></p><p data-vmark="e75f">直接打开 RunCat，任务栏就会显示一只正在跑步的猫猫了。当 CPU 占用不高的时候，猫猫的跑步速度是这样的。</p><p data-vmark="8c62"><img src="https://img.ithome.com/newsuploadfiles/2021/12/9eb645ee-5d17-4ec7-b962-7306adc6c882.gif" w="680" h="86" alt="任务栏CPU占用" title="在任务栏显示猫猫，超萌的 Win11/Win10 CPU 占用率工具你用过吗" width="680" height="86" referrerpolicy="no-referrer"></p><p data-vmark="59fd">CPU 占用有所升高了，小猫也哒哒哒地快起来了！</p><p data-vmark="82d4"><img src="https://img.ithome.com/newsuploadfiles/2021/12/f449f8db-9319-443f-8564-48b0335610b3.gif" w="680" h="86" alt="任务栏CPU占用" title="在任务栏显示猫猫，超萌的 Win11/Win10 CPU 占用率工具你用过吗" width="680" height="86" referrerpolicy="no-referrer"></p><p data-vmark="c7e3">当 CPU 占用非常高的时候，小猫狂奔！</p><p data-vmark="f83d"><img src="https://img.ithome.com/newsuploadfiles/2021/12/a05494bf-94d5-45a8-91a8-29b0317cedb5.gif" w="680" h="86" alt="任务栏CPU占用" title="在任务栏显示猫猫，超萌的 Win11/Win10 CPU 占用率工具你用过吗" width="680" height="86" referrerpolicy="no-referrer"></p><p data-vmark="6494">RunCat 的功能就是这么简单，它的卖点在于萌，实际上的确也非常萌。<span class="accentTextColor">除了猫猫外，RunCat 还提供了鹦鹉的样式</span>。</p><p data-vmark="f8a3"><img src="https://img.ithome.com/newsuploadfiles/2021/12/0fd6425f-46a3-4848-bea7-16cffacf09e9.gif" w="846" h="331" 还支持系统明亮和暗黑主题的切换</span>，还是很不错的。</p><p data-vmark="6ff0">总的来说，这就是一款简简单单的、但却又特别可爱的小工具，感兴趣的朋友不妨尝试一下！</p>
