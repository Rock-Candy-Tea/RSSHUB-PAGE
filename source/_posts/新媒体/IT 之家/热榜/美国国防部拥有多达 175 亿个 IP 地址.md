<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 25 日消息 近年来我国 IPv4 地址资源十分紧张，因此一直在从运营商以及企业方面推动 IPv6 的建设，目前已经取得初步成效。根据外媒 TheVerge 消息，近日美国国防部（五角大楼）委托一家位于弗洛里达州的小公司寻找网络漏洞，并同时交出了 <span class="accentTextColor">1.75 亿个 IP 地址的控制权</span>，但并没有公布是 IPv4 还是 IPv6 地址。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/1ef008d0-6e22-46ab-942b-35f7fa5af33c.png" w="498" h="380" title="美国国防部拥有多达 1.75 亿个 IP 地址" width="498" height="380" referrerpolicy="no-referrer"></p><p>这家公司名为全球资源系统 (Global Resource Systems), 该公司于 2021 年 1 月 20 日起便开始管理这些 IP 地址。五角大楼发言人告诉《华盛顿邮报》称，该公司的将识别潜在的漏洞，以便防止未经授权的用户使用美国国防部的 IP 地址空间。</p><p>外媒表示，全球资源系统公司成立于去年 9 月，没有任何公开网站，目前也找不到该公司签订的合同。这项关于 IP 地址的计划是由五角大楼一个名为“国防数字服务”的小组负责的。</p><p>IT之家了解到，目前的互联网就是由美国国防部在上世纪发明的，因此 IP 地址资源牢牢掌握在美国手中。目前 IPv4 理论有 42 亿多个，但是公网 IP 地址范围仅有 <span class="accentTextColor">36.47 亿个</span>，虽然有一个国际组织负责分配 IP 地址资源，但美国仍牢牢掌控着大量 IP 地址，因此需要使用 IPv6 来逐步替代 IPv4。</p>
