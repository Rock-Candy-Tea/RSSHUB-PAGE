<p data-vmark="1eb7"><span class="font-color-greytip">原文标题：《微软见了都得点赞！Windows 桌面的最高级玩法！》</span></p><p data-vmark="980a">这几天，Win11 22H2 发布的消息又掀起了网络上有关 Windows 更新的一系列讨论。</p><p data-vmark="8ddb">每当 Windows 发布一版大更新，网上有关的讨论最多的无非就是这个版本更新了什么内容，和老版本有什么区别，以及好用不好用之类的话题。</p><p data-vmark="de38">作为一个经历和使用过多个 Windows 历史版本的用户来说，有的时候，却并不关心这些。</p><p data-vmark="b005">记得最早接触电脑，我们还使用的是 XP 系统，蓝天白云草地壁纸和深蓝色风格的窗口，足以让许多人回忆起以往的点点滴滴。</p><p data-vmark="6537" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/61e8ddf6-aaf9-4d56-a1eb-68607d80de2b.png" w="800" h="600" title="梦回 Windows  XP，微软 Win10 / Win11 桌面的高级玩法" width="800" height="600" referrerpolicy="no-referrer"></p><p data-vmark="25af">但是 XP 等老旧系统已经被微软抛弃多年，如今如果想重新启用，可能不太现实，虽然老系统用不了，但是体验一把老版系统的感觉还是能够做到的。</p><p data-vmark="b162">今天推荐的这两款软件可谓是满满的回忆杀，能够瞬间让你有种返回到 20 年前的感觉。</p><p data-vmark="6143" style="text-align: center;"><img src="http://ss1.sinaimg.cn/large/95e190b7ly1h6jvax0f51g205w06g40i.gif" w="212" h="232" class="lazy" title="梦回 Windows  XP，微软 Win10 / Win11 桌面的高级玩法" referrerpolicy="no-referrer" width="212" height="232"></p><h2 data-vmark="f6ac">RetroBar</h2><p data-vmark="202b">RetroBar 是一款让任务栏瞬间恢复为 XP 以及 9X 系统风格的小软件，软件免安装，直接运行就可以使用。</p><p data-vmark="5e7a">首次运行 RetroBar，默认语言是英文，需要设置为中文才方便使用，设置方式是在任务栏上单击右键，然后点击<strong>「Properties」</strong>，最后进入语言选项将语言更改成简体中文。</p><p data-vmark="cd00" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/226c873f-fcc7-487e-aed0-5403488f903b.png" w="514" h="678" title="梦回 Windows  XP，微软 Win10 / Win11 桌面的高级玩法" width="514" height="678" referrerpolicy="no-referrer"></p><p data-vmark="ab52" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/0e545817-5d84-4ab4-9338-e7c29699ae22.png" w="494" h="670" title="梦回 Windows  XP，微软 Win10 / Win11 桌面的高级玩法" width="494" height="670" referrerpolicy="no-referrer"></p><p data-vmark="6de3">除了语言，RetroBar 支持的主题非常丰富，包括 Windows 95-98、Windows 2000、Windows XP 等十几种皮肤。</p><p data-vmark="b25f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/b5abb5ff-3d05-451e-8656-020636cb113c.png" w="495" h="660" title="梦回 Windows  XP，微软 Win10 / Win11 桌面的高级玩法" width="495" height="660" referrerpolicy="no-referrer"></p><p data-vmark="b9a7">每切换一种主题，窗口最上方都会预览出任务栏的最终效果，随之任务栏同时也会相应地变化。</p><p data-vmark="c96c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/a4ce0e5c-eeea-4099-af0d-8f69388bfc8e.png" w="938" h="70" title="梦回 Windows  XP，微软 Win10 / Win11 桌面的高级玩法" width="938" height="61" referrerpolicy="no-referrer"></p><p data-vmark="17f9">RetroBar 还能更改任务栏位置，或上或左或右，都能根据需要随意变换，这对 Win11 来说可以说是一款福利工具，毕竟原版 Win11 的任务栏无非更改位置。</p><p data-vmark="f3a8">RetroBar 也支持开机自启，不用每次开机都去重新手动运行，省时又省力！</p><p data-vmark="2757">但大家要清楚，RetroBar 只能恢复任务栏的旧版效果，却不能变换开始菜单的样式，如果需要，建议搭配使用 Open-Shell 等这一类开始菜单美化工具，美化效果会更好！</p><p data-vmark="2c5d">为了正常运行 RetroBar，请首先保证你的电脑中已经安装了.NET 6 或更高版本。</p><p data-vmark="f568">RetroBar 开源地址：<a href="https://github.com/dremin/RetroBar" target="_blank">点此</a></p><h2 data-vmark="edfc">XP Shutdown</h2><p data-vmark="7c95">准确来说这是两款程序，都属于免安装绿色小工具，直接双击运行。</p><p data-vmark="7747">XP_Settings.exe 这个程序文件用来调整软件的基础设置选项。</p><p data-vmark="7910" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/cbafb5af-72f9-4222-baf6-0a3aabf43e23.png" w="392" h="281" title="梦回 Windows  XP，微软 Win10 / Win11 桌面的高级玩法" width="392" height="281" referrerpolicy="no-referrer"></p><p data-vmark="df98">XP_Shutdown.exe 是关机程序，XP_Logoff.exe 是注销程序，每次运行这两款程序，系统桌面都会出现经典的 XP 风格关机菜单，甚至连桌面黑白特效也没有放过，简直满满的都是回忆！</p><p data-vmark="406b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/4fc2b266-cda8-4e71-a736-904eea9ef5f2.png" w="535" h="443" title="梦回 Windows  XP，微软 Win10 / Win11 桌面的高级玩法" width="535" height="443" referrerpolicy="no-referrer"></p><p data-vmark="8b22" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/ba3accfb-9d06-42f2-adfc-42e37b58a031.png" w="541" h="372" title="梦回 Windows  XP，微软 Win10 / Win11 桌面的高级玩法" width="541" height="372" referrerpolicy="no-referrer"></p><p data-vmark="e06a">如果我们开始菜单使用的 Open-Shell，那么还可以将 XP_Shutdown.exe 链接到开始菜单的关机按钮，每次关机时都将默认采用这种 XP 关机风格菜单。</p><p data-vmark="7787" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/18456a73-a832-4ae5-aea0-4d343d404bbd.png" w="770" h="783" title="梦回 Windows  XP，微软 Win10 / Win11 桌面的高级玩法" width="770" height="783" referrerpolicy="no-referrer"></p><p data-vmark="52fe">链接方式是在关机菜单项中右键之后编辑选择菜单项，然后连接 XP_Shutdown.exe 程序。</p><p data-vmark="d97b">这是在 Win10 上的最终效果图，感觉还不错吧！要是再将桌面也一并跟换成 XP 风格的蓝天白云草地壁纸，那就简直太完美了！</p><p data-vmark="14b5" style="text-align: center;"><img src="http://ss1.sinaimg.cn/large/95e190b7ly1h6jvaxm8h6g20m00gme82.gif" w="792" h="598"
