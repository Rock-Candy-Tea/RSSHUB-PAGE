<p data-vmark="7c8c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 11 日消息，很长一段时间以来，用户对 AMD 驱动程序和性能的评价一直低于对英伟达的 OpenGL 驱动程序的评价，尤其是 Win11 发布后。因此，AMD 开发了一套 Mantle API，试图在底层图形领域取得先机。</p><p data-vmark="fbbe">从现在的情况来看，AMD 在 Win11 上的性能有望得到大幅度提升。</p><p data-vmark="e906">在 Windows 11 最新的 22H2 内部版本中，微软提供了 AMD 预发布的 31.0.12000.20010 UWP 驱动程序，其中附带了 22.05 OpenGL beta 驱动程序，可以大幅提高 AMD 显卡在 OpenGL 应用中的性能。</p><p data-vmark="ca7d">Guru3D 论坛有人发现，这个版本的驱动程序将基于下一代 Windows 显示驱动程序模型 (WDDM) v3.1 而设计。有人使用该驱动配合 OpenGL API 渲染器在 GPU-Z 中进行了多次测试，性能差异非常显著。此外，AMD 最近推出了 22.5.2 驱动也为几款 DirectX 11 游戏性能带来了大幅提升，对于 Win11 的兼容性也有改进。</p><p data-vmark="56e3">例如，在 Unigine 2013 的显卡测试程序“<span class="qkunPe">Valley</span>”(山谷) 中，该指数的表现提升了近 55%；而在更现代的 Unigine Superposition 测试中，性能差异超过 33%，这对于即将推出的 22.05 OpenGL 驱动来说无疑是个好消息。</p><p data-vmark="4bff"><img src="https://img.ithome.com/newsuploadfiles/2022/6/8d8964b1-28b1-487e-aff2-f0135862c3a7.jpg" w="760" h="527" alt="显示 WDDM 31 的 AMD OpenGL beta 驱动程序的 GPU-Z 屏幕截图" title="AMD 战未来，微软发布 Win11 22H2 WDDM 3.1 驱动：大幅提高 Radeon 显卡 OpenGL 性能" width="760" height="527" referrerpolicy="no-referrer"></p><p data-vmark="806c">IT之家了解到，在更老的 Heaven 基准测试中，新版本驱动程序性能提升并不明显，这表明新版本的 OpenGL 驱动对于老应用程序和游戏来说优化可能不太到位。</p><p data-vmark="3d20">当然，上述测试是在基于 RDNA 2 的 Radeon RX 6800 XT 上进行的，但第一代 RDNA 显卡也可以获得不俗的提升。</p>
