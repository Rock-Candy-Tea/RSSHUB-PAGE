            <p data-vmark="7009"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 14 日消息，MagiskOnWSA 是一个可以在微软 Windows 11 安卓子系统上获取 root 权限，并安装谷歌服务框架的项目，在 GitHub 上有 1.7 万次 Star。</p><p data-vmark="86cd" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/8/0ac2bbb6-ec0e-4a38-888a-d27ef88f7b18.png" w="1111" h="135" title="Win11 安卓子系统 root 项目 MagiskOnWSA 被微软 GitHub 封禁" width="1111" height="100" referrerpolicy="no-referrer"></p><p data-vmark="cc6d">然而，这个项目已被微软 GitHub 官方封禁。当前访问 <a href="https://github.com/LSPosed/MagiskOnWSA" target="_blank">MagiskOnWSA 存储库</a>只能看到一条通知，<span class="accentTextColor">由于违反了 GitHub 的服务条款，GitHub 员工已禁止访问此存储库</span>。</p><p data-vmark="33ee" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/8/fa383952-2506-4480-a947-8760ee34f9c6.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzIwNi5wbmc=,g_7,x_21,y_21,a_0,t_100" w="1337" h="394" title="Win11 安卓子系统 root 项目 MagiskOnWSA 被微软 GitHub 封禁" width="1337" height="242" referrerpolicy="no-referrer"></p><p data-vmark="23df">IT之家在去年还发布过使用该项目对 Win11 安卓子系统进行 root 的教程，<span class="accentTextColor">其中需要通过 Fork 该项目，并在 GitHub 上构建 artifact</span>，然后安装专用的面具 Magisk App，因此该项目的 Fork 数也达到了<span class="accentTextColor">十</span><span class="accentTextColor">万多次</span>。</p><p data-vmark="1ddb">《<a href="https://www.ithome.com/0/583/221.htm" target="_blank">微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk</a>》</p><p data-vmark="6e2d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2021/10/8ba41867-a665-4d62-8a0d-4c168680a26c.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzE2OC5wbmc=,g_2,x_0,y_273,a_0,t_100" w="1440" h="765" title="Win11 安卓子系统 root 项目 MagiskOnWSA 被微软 GitHub 封禁" width="1440" height="436" referrerpolicy="no-referrer"></p><p data-vmark="1ad8" style="text-align: justify;">目前尚不清楚该项目被封禁的具体原因，而 Magisk 的开发者和 LSPosed 的开发者都暂未在社交媒体上给出回应。</p><p data-vmark="be37" style="text-align: justify;"><span class="accentTextColor">Windows 11 安卓子系统 (Windows Subsystem for Android) 包括 Linux 内核和基于 Android 开源项目（AOSP）版本的 Android 操作系统。</span>该子系统在 Hyper-V 虚拟机中运行，就像 Linux 子系统一样，可以将 AOSP 环境中 App 的运行时和 API 映射到 Windows 图形层、内存缓冲区、输入模式、物理和虚拟设备以及传感器，可以在英特尔、AMD、高通的 CPU 上运行。</p>
