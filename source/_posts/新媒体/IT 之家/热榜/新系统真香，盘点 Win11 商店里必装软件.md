<p data-vmark="b995">很快我们就会迎来 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 正式发布的日子，可新系统的表现…… 还是差那么一点点。特别是很多刚加入的模块，经常会遇到这样或那样的问题。其实我们在关注 Win11 本身的同时，却一直忽略掉应用商店这块宝地。</p><p data-vmark="a7cd">随着微软对商店支持力度的不断提升，里面各种好玩的东东也在慢慢丰富起来。就比如说接下来这几个，堪称 Win11 里的真香现场！</p><p data-vmark="ffcd"><img src="https://img.ithome.com/newsuploadfiles/2021/10/a42717d5-6ce5-4c8a-937c-714f2a017891.jpg" w="700" h="429" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="429" referrerpolicy="no-referrer"></p><h2 data-vmark="0151">1. Microsoft PowerToys</h2><p data-vmark="3bf0">Microsoft PowerToys 是微软推出的一款辅助工具集，内置有很多功能模块。比如你可以用它批量调整图像尺寸、通过拾色器读取屏幕任意区域上的颜色、按下 Shift 键实现窗口的快速布局（相比 Win11 的窗口布局更高效）、通过映射器修改键盘快捷键（方便苹果系统的小伙伴使用）等，可最厉害的还得数它的搜索功能。</p><p data-vmark="c304"><img src="https://img.ithome.com/newsuploadfiles/2021/10/6b0da6a5-9630-4cde-bc1c-7be3c23327c8.gif" w="774" h="269" title="新系统真香，盘点 Win11 商店里必装软件" width="774" height="269" referrerpolicy="no-referrer"></p><p data-vmark="45e2">在 PowerToys 中，我们可以通过“Alt + 空格”调出快速搜索栏。这里的搜索类似于 Windows，但更快也更实用。你可以用它搜索文件、定位程序、数学运算、筛选文件夹。除了没有语音搜索，它几乎搞定了所有用户在搜索方面的诉求。再加上这是一款完全没有广告的官方工具，大大提高了它的实用性。</p><p data-vmark="8d8a"><img src="https://img.ithome.com/newsuploadfiles/2021/10/823ca0dd-02eb-4776-9189-f061a4875cb6.gif" w="774" h="269" title="新系统真香，盘点 Win11 商店里必装软件" width="774" height="269" referrerpolicy="no-referrer"></p><h2 data-vmark="b510">2. Snipaste</h2><p data-vmark="d761">Win11 的截图工具历经多次改版，可依旧难用，这款 Snipaste 除了拥有一键截图、一键存图、一键编辑三组传统功能外，还特别加入了“贴图”这一绝技。举个例子，比如你正在编写一份文件，突然觉得参考一下“前人”的东东更有效率，这时就可以借助贴图功能，将别的资料“粘贴”在旁边对照编写。和传统的窗口排列相比，贴图功能不仅用起来方便，其自动置顶机制还有效避免了程序遮挡对操作造成的影响。</p><p data-vmark="4f2b"><img src="https://img.ithome.com/newsuploadfiles/2021/10/bf720989-ac02-4ac4-afc6-d40fbe2c27d6.jpg" w="700" h="393" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="393" referrerpolicy="no-referrer"></p><p data-vmark="54df">此外，Snipaste 还带有丰富的图片编辑功能，比方说日常用到的马赛克涂抹、箭头标注、文字标签等，都可以在软件内直接搞定。</p><p data-vmark="eef8"><img src="https://img.ithome.com/newsuploadfiles/2021/10/b9d13a79-56ee-487c-8d53-b2d6891b8dd9.jpg" w="700" h="463" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="463" referrerpolicy="no-referrer"></p><h2 data-vmark="a312">3. 微软必应词典</h2><p data-vmark="2bec">词典的重要性在日常工作中不言而喻，微软必应词典是一款集翻译、阅读、背单词、口语训练于一体的综合类词典软件。传统功能并不出众，但它的背单词功能却十分专业。词典内置有英语四级、英语六级、GRE、GMAT、TOEFL 等专业考试词汇，以及国外生活、高考词汇、考研词汇等一些实用生词本，点击旁边的“背”即可进入背单词模式。</p><p data-vmark="91f4"><img src="https://img.ithome.com/newsuploadfiles/2021/10/92606763-f496-4005-9e59-5aee1adf2c9f.gif" w="429" h="600" title="新系统真香，盘点 Win11 商店里必装软件" width="429" height="600" referrerpolicy="no-referrer"></p><p data-vmark="8140">必应词典的背词机制很有趣，当屏幕出现英语单词时，按住下方按钮开始思考。如果在规定时间内未松手，则代表该单词不会。反之松开按钮的时间越短，代表对该单词越熟悉。之后词典会自动记录这些数据，并生成图表，便于你下次使用。同时你也可以将不熟悉的单词添加到生词本里，对疑难词重点记忆。</p><h2 data-vmark="350e">4. Files – File Manager for Windows</h2><p data-vmark="7ece">Files 是一款功能足以碾压原版的第三方资源管理器，除了内置很多官方没有的功能外，在整个外观和操作方法上也极尽模仿 Windows，尽最大程度降低用户的学习成本。</p><p data-vmark="a125"><img src="https://img.ithome.com/newsuploadfiles/2021/10/1d8ade33-8ca9-4137-a526-580bcef6fc73.jpg" w="700" h="412" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="412" referrerpolicy="no-referrer"></p><p data-vmark="8e36">Files 的功能主要分为两块，一是类似于 Windows 的传统功能，二是它自己的特色功能。基本功能这里就不多说了，反正 Windows 能做的它几乎都能搞定。特色功能方面，除了大家期待已久的多标签面板、自动打开上一次位置、双窗格视图、文件 Tag、简洁权限面板外。还由于其基于 Fluent Design 设计，因此也具备了官方版特有的夜间模式、新折叠右键等属性。同时 Files 还在官方资源管理器基础上，对图标分组、布局排列、可视化效果等方面，做出了自己的创新！毫不夸张地说，在目前这个时间点上，Files 已经远远将官方版落在后面了。</p><p data-vmark="6afa"><img src="https://img.ithome.com/newsuploadfiles/2021/10/1e73e37d-5d97-43ce-a6e5-018ef3f00edb.jpg" w="700" h="391" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="391" referrerpolicy="no-referrer"></p><h2 data-vmark="d7c3">5. 官方精简版 QQ</h2><p data-vmark="1ee8">很多软件都有单独的 UWP 版，但不是体验太差，就是功能更新慢。QQ 桌面版同样来自于腾讯，但与普通 UWP 版不同，它的整体框架与我们常见的普通版没什么两样。</p><p data-vmark="b287"><img src="https://img.ithome.com/newsuploadfiles/2021/10/28098b71-d5ac-4a84-baf6-2dbbaf43788a.jpg" w="700" h="403" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="403" referrerpolicy="no-referrer"></p><p data-vmark="5fd4"><img src="https://img.ithome.com/newsuploadfiles/2021/10/ea5df591-3c4b-4516-82a4-ab0f7a3f5c47.jpg" w="660" h="470" title="新系统真香，盘点 Win11 商店里必装软件" width="660" height="470" referrerpolicy="no-referrer"></p><p data-vmark="7fe6">“商店桌面版”使用了和普通版一样的外观，功能上的区别也相差无几。由于是商店里安装，因此它可以直接享受微软的一键更新与一键卸载服务。特别是由于微软商店的限制，这个版本还专门去掉了广受诟病的“QQ Protect”，而后者恰恰是很多老电脑运行 QQ 卡顿的元凶。</p><h2 data-vmark="d36a">6. Microsoft Whiteboard</h2><p data-vmark="c736">如果你的 Windows 设备是一台平板，那么 Whiteboard 将是最不可或缺的应用之一。正如它的名字所言，这是一款 Windows 下的白板软件。你可以通过手指或触控笔，在平板上任意书写、绘画，或者将自己的想法记录下来，与他人一同协作分享。和 OneNote 一样，Whiteboard 也没有边框与区域的限制，这使得我们在创作时，可以拥有无限的自由度。此外和其他 Microsoft 工具一样，Whiteboard 也支持云端实时备份与同步，大大提高了用户的日常体验。</p><p data-vmark="8ff5"><img src="https://img.ithome.com/newsuploadfiles/2021/10/461ba4e2-c100-4a34-8320-58c38de26e64.jpg" w="700" h="468" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="468" referrerpolicy="no-referrer"></p><p data-vmark="fbe8"><img src="https://img.ithome.com/newsuploadfiles/2021/10/5fc5479a-8145-4841-aca6-ac37eb63f76f.jpg" w="700" h="478" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="478" referrerpolicy="no-referrer"></p><h2 data-vmark="19de">7. Sleep Waves Free</h2><p data-vmark="3e8f">“白噪音”这个词相信大家并不陌生，在自己心烦意乱或者睡不着觉时，打开白噪音就能快速平静下来。Sleep Waves 是一款功能实用的白噪音软件，除了可以像其他软件一样，发出柔风、细雨、火车、风铃等让人宁神的声音外，它的最大特点是能够将不同音效混合在一起播放。软件使用很简单，点击图片播放音效，拖动滑杆调整音量。尽管是纯英文界面，但上手并没有什么难度。</p><p data-vmark="98f4"><img src="https://img.ithome.com/newsuploadfiles/2021/10/b241f144-197f-4bdd-9eb6-7634dbcd079c.jpg" w="700" h="475" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="475" referrerpolicy="no-referrer"></p><h2 data-vmark="59b9">8. Microsoft To Do</h2><p data-vmark="48ae">对于做事没有计划的人来说，一款优秀的 To-Do List 是十分有必要的。Microsoft To Do 是微软出品的一款待办清单软件，首先它是一款免费应用，甚至免费的功能比很多收费软件都强。其次它的功能支持全面，重复、提醒、细化步骤、建立任务……，基本上你能想到的功能，它这里都能实现。同时作为一款 Microsoft 套件，它与其他微软组件的配合也很完美，比如你刚刚在 Outlook 里标记了一封邮件，转眼间 Microsoft To Do 就会自动生成一份任务。而在这个过程中，你是全程不用动手的。</p><p data-vmark="ac2e"><img src="https://img.ithome.com/newsuploadfiles/2021/10/8d21a44b-c5b1-4e08-b68a-9d9b9fd01ca1.jpg" w="700" h="399" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="399" referrerpolicy="no-referrer"></p><p data-vmark="9f30">顺道说一句，Microsoft To Do 还是目前少有的 Win、<a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a>、iOS 全平台同步软件。全免费 + 全平台同步，我想这一点就足够让人吹爆了吧！</p><h2 data-vmark="d9a8">9. Lively Wallpaper</h2><p data-vmark="8213">厌倦了一成不半的桌面壁纸，不妨换一个心情。Lively Wallpaper 是一款简单实用的动态壁纸软件，具有两大功能。一是为桌面添加高大上的动态桌面背景，二是加入了很多小伙伴日思夜盼的 —— 任务栏变透明！</p><p data-vmark="5c44"><img src="https://img.ithome.com/newsuploadfiles/2021/10/fb55d158-da70-41cb-a3eb-74da5b5b2d64.jpg" w="700" h="522" title="新系统真香，盘点 Win11 商店里必装软件" width="700" height="522" referrerpolicy="no-referrer"></p><p data-vmark="1551">Lively Wallpaper 内置有十二组壁纸，虽然数量不多，但质量普遍很高。与其他动态桌面相比，Lively Wallpaper 除了资源占用极低以外（除非是那些很老很老的老爷机），高分辨率是它的最大亮点。即便是在主流的 4K 显示器上，也能很清晰地展现。同时 Lively Wallpaper 还为绝大多数壁纸提供了自定义选项，炫酷的同时还支持手工 DIY，想想是不是都感觉很幸福！</p><p data-vmark="d1aa"><img src="https://img.ithome.com/newsuploadfiles/2021/10/f0a2e394-1ab8-473a-be1a-e3ba9fbbc725.gif" w="699" h="442" title="新系统真香，盘点 Win11 商店里必装软件" width="699" height="442" referrerpolicy="no-referrer"></p><h2 data-vmark="e47c">写在最后</h2><p data-vmark="f25f">总的来说，应用商店就是一个大宝库，无论你对现行 Win11 的设计满不满意，都能在这里找到令自己心动的那一个。顺道说一句，除了上面提到的这些小工具外，微信、B站、爱奇艺等也都推出过自己的 UWP 版本，虽然功能不及桌面版全面，但全程没有一丁点广告就是它的最大看点。好了，这就是本期要和大家分享的几款小工具，喜欢的话赶紧试一试吧！</p>
