<p data-vmark="2527"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 9 日消息，微软近日向 Dev 预览版用户推送了 <span class="accentTextColor"><a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> Build 22526 版本更新</span>，对 Alt+Tab UI 进行了改进，大家发现了吗？</p><p data-vmark="a914">Windows 的“任务切换器”允许用户使用 Alt+Tab 键盘快捷键在打开的应用之间快速切换，在 Windows 11 中，Alt+Tab 显示为全屏毛玻璃背景，应用选项卡堆叠排列。</p><p data-vmark="1703"><img src="https://img.ithome.com/newsuploadfiles/2022/1/b1b93c30-ea35-43be-8d59-9585f350ca79.jpg" w="696" h="354" alt="Windows 11 Alt Tab feature" title="微软 Win11 更新 Alt+Tab UI：毛玻璃横向排列，教你强制开启" width="696" height="354" referrerpolicy="no-referrer"></p><p data-vmark="4d9e">而在最新的 Dev 预览版中，<span class="accentTextColor">Alt+Tab UI 将显示为窗口化的毛玻璃，且变成了横向排列</span>，看起来更加清新。</p><p data-vmark="fbce"><img src="https://img.ithome.com/newsuploadfiles/2022/1/775c0351-d51c-4633-828e-dc0d29497f17.jpg@s_2,w_820,h_289" w="1428" h="503" alt="Windows 11 Alt Tab UI" title="微软 Win11 更新 Alt+Tab UI：毛玻璃横向排列，教你强制开启" srcset="https://img.ithome.com/newsuploadfiles/2022/1/775c0351-d51c-4633-828e-dc0d29497f17.jpg 2x" width="1428" height="289" referrerpolicy="no-referrer"></p><p data-vmark="3039">IT之家了解到，<span class="accentTextColor">这一新 UI 目前只向部分 Dev 预览用户提供</span>，毛玻璃效果更加自然，同时也能显示窗口预览，而非应用图标，大家觉得怎么样？</p><h2 data-vmark="feae">强制开启方法：</h2><ul class=" list-paddingleft-2"><li><p data-vmark="eba4">1. 下载 <a href="https://github.com/thebookisclosed/ViVe/releases" target="_blank">ViveTool</a></p></li><li><p data-vmark="c97a">2. 在下载到的目录以管理员方式执行指令：</p></li></ul><pre class="brush:javascript;toolbar:false ai-word-checked">vivetool addconfig 36226836 2</pre><ul class=" list-paddingleft-2"><li><p data-vmark="f383">3. 重启电脑</p></li></ul><p data-vmark="8054">微软已经开始着手开发下一个“<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 22H2”大版本，计划通过将 Mica+Fluent 引入更多使用不同框架的应用程序来改进现有操作系统的界面，预计这一新 Alt+Tab UI 将在新版本上线。</p>
