<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 14 日消息 在 6 月 24 日的 Windows Sun Valley 发布会之前，微软正在加快节奏以争取敲定 Windows Sun Valley 或者是说“下一代 Windows”的正式版。</p><p>作为 Sun Valley 中的一部分，目前预览版中最大的改变还是一些小的设计调整，比如圆角 UI，所以看起来好像微软设计师的使命就只是“使 Windows UX 的现有部分更加现代化”。</p><p>IT之家了解到，Windows 10 早在 2015 年发布，至今已延续 12 个版本，但实际上后续版本界面与 初代 1507 相比没有任何改变。此外，Win10 仍然在使用 Win8、Win7、WinXP 甚至 Win95 的 UI 特性 / 元素。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/2f6b2aba-bd51-4189-a415-eb880bb76c5b.png" w="1041" h="663" title="大升级还是改 UI，微软正在对 Win10 UX 部分进行现代化改造" width="1041" height="522" referrerpolicy="no-referrer"></p><p>界面经过重新设计的 Windows 11（可能）将于晚些时候发布，而微软之前透露的信息表面它将会是“革命性的”一代产品。</p><p>根据外媒 Windows Latest 今天放出的另一份清单，微软现在急切想要开创一种新的体验，而这会涉及到现有界面的现代化内容。</p><p>据悉，微软正在努力打造“Windows UX 的新部件和更新现有部件”。且为了改善用户体验，他们正在内部进行用户调查，以便更好地了解用户的需求，并在最终定稿前完成各种解决方案的测试。</p><p>此外，微软还聘请了一位高级项目经理来“定义、完善和提供世界级的（Windows）体验”。值得一提的是，这项工作清单要求项目经理“与微软的各个团队协作，建立对拟议解决方案的支持”。</p><h2>微软真的在更新 Windows 10 吗？<br></h2><p>答案是：视情况而定。</p><p>表面上，Windows 11 或太阳谷更新将 Windows 10 显得更加“现代”或者说清新 / 新奇。微软也认为“这将是‘令人愉快的和标志性的’”。</p><p>就目前已知的 UI 变更来看，微软将更新大部分图标、属性标签、右键菜单和其他重要功能，使其更显现代，还会提供黑色模式。例如文件资源管理器将采用圆角、新图标和与其他 Windows 应用程序设计相匹配的暗黑模式。</p><p>不过，Windows Latest 也认为，太阳谷更新将包含一些旧的功能，或者说遗留组件，如图标或弹出窗口等。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/12c15655-015d-4159-9044-541a57dca7ee.jpg" alt="Windows 10 modern interface" w="696" h="365" title="大升级还是改 UI，微软正在对 Win10 UX 部分进行现代化改造" width="696" height="365" referrerpolicy="no-referrer"></p>
