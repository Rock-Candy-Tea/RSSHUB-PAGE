<p data-vmark="127a"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 26 日消息，据 Windows Latest 报道，KB5012643 现在可用于 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>，其中包含一些值得注意的更改和一系列额外的错误 Bug 修复。</p><p data-vmark="42ee">Windows 11 KB5012643 是可选的累积更新，旨在测试与 2022 年 5 月补丁星期二更新一起发布的更改。<span class="accentTextColor">与其他可选更新一样，此累积更新不会自动下载和安装，除非检查更新并手动开始下载</span>。</p><p data-vmark="304e" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/a5d145fb-75f7-4f25-90e8-7cf1404b5364.jpg@s_2,w_820,h_430" w="1440" h="756" title="微软 Win11 Build 22000.652（KB5012643）正式版发布：任务栏天气图标顶部显示温度（附更新内容大全）" srcset="https://img.ithome.com/newsuploadfiles/2022/4/a5d145fb-75f7-4f25-90e8-7cf1404b5364.jpg 2x" width="1440" height="430" referrerpolicy="no-referrer"></p><p data-vmark="082e">与 2022 年 3 月的可选更新不同，此可选累积更新不是大版本，但有几处质量改进。例如，微软添加了一项新功能，该功能将在天气图标顶部显示温度，该图标已通过 2022 年 4 月更新添加到任务栏。</p><p data-vmark="24eb">微软修复了操作系统错误对齐视频字幕的问题，并修复了另一个错误 Bug，即视频字幕将被部分切断。微软表示，还修复了阻止用户使用 Windows 窗口控件，例如最小化、最大化和关闭按钮问题。</p><h3 data-vmark="dde7">Windows 11 KB5012643（版本 22000.652）改进内容</h3><ul class=" list-paddingleft-2"><li><p data-vmark="9bf4">Windows 11 任务栏现在可以在任务栏上的天气图标顶部显示温度。</p></li><li><p data-vmark="b1c9">微软进行了更改，以改善 Windows 11 安全启动组件的服务。</p></li><li><p data-vmark="006e">微软修复了视频字幕被部分截断的问题。</p></li><li><p data-vmark="aa11">微软修复了导致操作系统错误对齐视频字幕的问题。</p></li><li><p data-vmark="0c02">已修复用户无法单击最小化、最大化和关闭按钮的重大错误。</p></li></ul><p data-vmark="1d5a">根据发行说明，微软已经修复了启动过程可能导致停止错误的竞争条件，即在启动操作系统时出现蓝屏死机。</p><p data-vmark="2a8a">该公司修复了安装 MSIX 应用时 AppX 部署服务 (AppXSvc) 停止工作的问题。该公司还对支持自我部署和预配置场景的 Autopilot 客户端和 TPM 进行了改进。</p><p data-vmark="a1b4">由于内存泄漏错误，Windows 报告高内存使用率的另一个错误已得到修复。微软还修复了影响 Microsoft Edge IE 模式下 Title 属性的问题，即 Windows Enterprise 版本中不允许移动设备管理策略的错误。</p><p data-vmark="1010">微软表示，还修复了一个 Windows 可能在服务更新后进入 BitLocker 恢复的问题。另一个错误已修复，其中操作系统可能无法复制组策略的安全部分。</p><h3 data-vmark="4b02">Build 22000.652 改进和修复</h3><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="5b0e">微软已修复 Netdom.exe 或 Active Directory 域和信任管理单元过程可能失败的问题。</p></li><li><p data-vmark="16aa">对于企业，微软修复了根域的主域控制器 (PDC) 在系统日志中生成警告和错误事件的问题。</p></li><li><p data-vmark="6ce9">微软还修复了减少了每秒高输入 / 输出操作 (IOPS) 场景中资源争用开销。</p></li></ul><h3 data-vmark="12ac">Windows 11 更新中的已知问题</h3><p data-vmark="0682">微软目前只知道更新中的一个已知问题。根据发行说明，如果使用 <a class="s_tag" href="https://www.win7china.com/" target="_blank">Windows 7</a> 的“备份和还原”功能创建了恢复磁盘，恢复将无法进行。</p><p data-vmark="d960">此错误 Bug 不会影响第三方备份或恢复应用程序。</p><h3 data-vmark="d4fe">微软开始为“太阳谷 2”更新做准备</h3><p data-vmark="61de">微软计划在秋季推出代号为“太阳谷 2”的 Windows 11 版本 22H2，有报道称该更新最早可能在 8 月推出。</p><p data-vmark="8a08">除了可选更新外，微软表示还捆绑了一项新更改，以提高服务堆栈的质量。服务堆栈更新包括对安装 Windows 更新的组件的质量改进。</p><p data-vmark="7b00">它确保堆栈更新具有稳健和可靠的服务堆栈，以确保顺利安装 Windows 更新。服务堆栈更新通常在主要功能版本发布前几个月发布。</p><h3 data-vmark="a79c">下面是完整更新日志：</h3><ul class=" list-paddingleft-2"><li><p data-vmark="1522">新的！添加了为 Windows 的安全启动组件提供服务的改进。</p></li><li><p data-vmark="4c48">解决了在安装某些 MSIX 应用后导致 AppX 部署服务 (AppXSvc) 停止工作的问题。</p></li><li><p data-vmark="77bd">解决了在启动过程早期发生的可能导致停止错误的竞争条件。</p></li><li><p data-vmark="37fa">改进了 Autopilot 客户端以处理支持自我部署和预配置方案的更新的可信平台模块 (TPM) 功能。</p></li><li><p data-vmark="1015">将已加入 Azure AD 的混合 Autopilot 方案的 Azure Active Directory (Azure AD) 注册超时从 60 分钟更改为 90 分钟。这也解决了在超时时导致异常的竞争条件。</p></li><li><p data-vmark="fa9e">解决了某些 POS 终端在重新启动长达 40 分钟期间偶尔会遇到操作系统启动延迟的问题。</p></li><li><p data-vmark="21c9">解决了影响一周中每天 24 小时都在使用的 Windows 系统的内存泄漏问题。</p></li><li><p data-vmark="0745">通过阻止使用连接特定的 DNS 后缀搜索列表，解决了影响动态主机配置协议 (DHCP) 选项 119（域搜索选项）的问题。</p></li><li><p data-vmark="ab38">解决了在 Microsoft Edge IE 模式下影响 Title 属性的问题。</p></li><li><p data-vmark="3294">解决了使用 Azure AD 加入订阅权利升级到企业版的 Windows 企业版上不允许移动设备管理 (MDM) 策略的问题。</p></li><li><p data-vmark="c18e">解决了可能导致视频字幕被部分截断的问题。</p></li><li><p data-vmark="54ef">解决了错误对齐视频字幕的问题。</p></li><li><p data-vmark="9b48">解决了导致 Kerberos 身份验证失败的问题，错误为“0xc0030009 (RPC_NT_NULL_REF_POINTER)”。当客户端计算机在启用远程凭据保护时尝试使用远程桌面协议 (RDP) 连接到另一台计算机时，会发生这种情况。</p></li><li><p data-vmark="b5dd">解决了导致 Windows 在服务更新后进入 BitLocker 恢复的问题。</p></li><li><p data-vmark="3aed">解决了阻止从 TPM 设备检索认可密钥 (EK) 证书的问题。</p></li><li><p data-vmark="49a3">解决了可能无法将组策略的安全部分复制到计算机的问题。</p></li><li><p data-vmark="3226">解决了阻止 Microsoft RDP 客户端控件（版本 11 及更高版本）在 Microsoft 基础类 (MFC) 对话框中实例化的问题。</p></li><li><p data-vmark="1206">在任务栏上的天气图标顶部显示温度。</p></li><li><p data-vmark="c343">解决了阻止在最大化的应用程序窗口上使用最小化、最大化和关闭按钮的问题。出现此问题是因为通知中心保持输入焦点。</p></li><li><p data-vmark="9477">解决了使用 Netdom.exe 或 Active Directory 域和信任管理单元列出或修改名称后缀路由时可能出现的问题。这些过程可能会失败。错误消息是“系统资源不足，无法完成请求的服务”。在主域控制器模拟器 (PDCe) 上安装 2022 年 1 月安全更新后会出现此问题。</p></li><li><p data-vmark="3933">解决了导致根域的主域控制器 (PDC) 在系统日志中生成警告和错误事件的问题。当 PDC 错误地尝试扫描仅传出信任时，会出现此问题。</p></li><li><p data-vmark="3f27">解决了将网络驱动器映射到服务器消息块版本 1 (SMBv1) 共享时出现的问题。重新启动操作系统后，将无法访问该网络驱动器。</p></li><li><p data-vmark="cdd2">解决了影响 SMB 多通道连接并可能生成 13A 或 C2 错误的问题。</p></li><li><p data-vmark="b728">解决了当客户端缓存 (CSC) 清理方法无法删除已创建的资源时损坏池的问题。</p></li><li><p data-vmark="838d">解决了由于非分页池增长并用完所有内存而可能导致服务器锁定的问题。重新启动后，当尝试修复损坏时再次出现相同的问题。</p></li><li><p data-vmark="783c">减少在具有多个线程争用单个文件的高每秒输入 / 输出操作 (IOPS) 场景中的资源争用开销。</p></li></ul>
