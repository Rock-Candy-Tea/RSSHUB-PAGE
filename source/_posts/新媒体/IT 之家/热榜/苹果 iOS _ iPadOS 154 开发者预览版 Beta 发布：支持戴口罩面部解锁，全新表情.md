<p data-vmark="ef3b"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 28 日消息，苹果今日向 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 和 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 用户推送了 iOS / iPadOS 15.4 开发者预览版 Beta 更新（内部版本号：19E5209h），本次更新距离上次发布隔了 2 周时间，带来了 Universal Control 全局控制、全新的 emoji 表情、苹果钱包 Apple Card 小部件等新功能。</p><p data-vmark="1709" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/1/76ae8374-7eb8-476a-83bd-9a64d8ab0fac.jpg" w="800" h="450" title="苹果 iOS / iPadOS 15.4 开发者预览版 Beta 发布：支持戴口罩面部解锁，全新表情" width="800" height="450" referrerpolicy="no-referrer"></p><h2 data-vmark="edaa">Face ID 识别戴口罩的人脸</h2><p data-vmark="5d3f">iOS 15.4 系统的 Face ID 人脸识别解锁功能得到了升级，用户无需摘下口罩依旧可以解锁 iPhone。<span class="accentTextColor">这项功能仅支持 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 12</a> 及以上的手机</span>，同时包括 iPhone 12 mini、<a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 13 mini</a>。在系统设置中，可以看到新功能的开关。介绍中指出，Face ID <span class="accentTextColor">可以通过分析人眼周围独特的特征来进行识别</span>。用户佩戴口罩解锁手机时，双眼必需直视传感器。</p><p data-vmark="e8f2" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/1/097ca7f9-7f1f-4595-8ffc-85c1963ef63c.jpg" w="800" h="784" title="苹果 iOS / iPadOS 15.4 开发者预览版 Beta 发布：支持戴口罩面部解锁，全新表情" width="800" height="784" referrerpolicy="no-referrer"></p><h2 data-vmark="0e23">Universal Control 全局控制</h2><p data-vmark="9a3f">这项新功能为 iPad OS 15.4 Beta 版推出，同时支持 macOS Monterey 12.3 Beta 版。用户可以通过 Mac 电脑上的一套鼠标（触摸板）和键盘，操控多台身边的电脑、iPad 平板，前提是设备登录了同一个苹果账号。</p><p data-vmark="e29f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/1/dea29363-114d-4e3a-a00a-8e20eb638fb4.jpg" w="800" h="480" title="苹果 iOS / iPadOS 15.4 开发者预览版 Beta 发布：支持戴口罩面部解锁，全新表情" width="800" height="480" referrerpolicy="no-referrer"></p><h2 data-vmark="88ed">全新 Emoji 表情：</h2><p data-vmark="818b">iOS 15.4 Beta 本次更新后，<span class="accentTextColor">带来了对于 Emoji 14 版本的支持</span>，新增了 37 款新表情、75 个皮肤颜色变种，共计 112 个。新增的皮肤包括融化的黄脸、双手比心手势、偷看表情、咬一边嘴唇、敬礼、电池电量不足、气泡等等。</p><p data-vmark="8be5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/1/7a1799b3-c2d7-4bf5-aea9-25bc82cd4f05.jpg" w="800" h="450" title="苹果 iOS / iPadOS 15.4 开发者预览版 Beta 发布：支持戴口罩面部解锁，全新表情" width="800" height="450" referrerpolicy="no-referrer"></p><h2 data-vmark="c59f">Apple Card 小部件</h2><p data-vmark="7914">苹果 iOS 本次更新，带来了“钱包”应用的 Apple Card 小部件，可以添加到主屏幕，方便显示当前余额等。</p><p data-vmark="1c96" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/1/b0dbbfa6-ad4e-4afd-8fe6-d01df03c6fa3.jpg" w="800" h="1606" title="苹果 iOS / iPadOS 15.4 开发者预览版 Beta 发布：支持戴口罩面部解锁，全新表情" width="800" height="1606" referrerpolicy="no-referrer"></p><h2 data-vmark="9c19">iPad 磁吸键盘背光亮度调节功能</h2><p data-vmark="582e">iPadOS 15.4 版本新增了一个键盘背光亮度调节功能，在控制中心可访问。</p><p data-vmark="d01c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/1/ef79ee31-fc0e-434b-aca1-3dc7097cbcfa.jpg" w="800" h="582" title="苹果 iOS / iPadOS 15.4 开发者预览版 Beta 发布：支持戴口罩面部解锁，全新表情" width="800" height="582" referrerpolicy="no-referrer"></p><h2 data-vmark="b941">iCloud 钥匙链密码注释功能。</h2><p data-vmark="4154">苹果的 Keychain 钥匙链可以帮助用户记录第三方应用、网站的账户密码，如今用户可以在每一项存储记录中，添加注释，更方便之后的管理。</p><p data-vmark="a65a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/1/3732f1b9-fceb-4259-8bda-72700b516e38.jpg" w="800" h="784" title="苹果 iOS / iPadOS 15.4 开发者预览版 Beta 发布：支持戴口罩面部解锁，全新表情" width="800" height="784" referrerpolicy="no-referrer"></p><h2 data-vmark="07a2">通过 Passkey 证书登录网站</h2><p data-vmark="f657">iOS 15.4、iPadOS 15.4 beta 版新增了这项功能，用户可以使用已保存的 Passkey 登录网站或者 App。这意味着用户可以通过 i 设备进行 Face ID、Touch ID 验证，从而登录第三地方服务，无需手动输入账号密码。</p><h2 data-vmark="3966">支持第三方应用解锁 120Hz 显示</h2><p data-vmark="2b9a">iOS 本次更新后，为 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 13 Pro</a> 系列手机解锁第三方应用 120Hz 显示功能，充分利用屏幕刷新率。</p><p data-vmark="fa90"><img src="https://img.ithome.com/newsuploadfiles/2022/1/4b7e6352-a5a6-486f-9477-765c481546c5.jpg" w="800" h="448" title="苹果 iOS / iPadOS 15.4 开发者预览版 Beta 发布：支持戴口罩面部解锁，全新表情" width="800" height="448" referrerpolicy="no-referrer"></p><p data-vmark="2446">IT之家了解到，苹果本次系统更新还新增了对于<span class="accentTextColor">索尼 PS5 手柄自适应扳机功能的支持</span>。在健康 App 中，用户可以查看新冠疫苗接种记录，支持欧盟的证书。疫苗接种卡，也可以添加到钱包应用当中，方便使用。</p>
