            <p data-vmark="9597"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 17 日消息，现有情报表明，微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 版本 22H2 更新将在 9 月 20 日前后到来，目前已有微软内部员工收到 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 22H2 推送。</p><p data-vmark="260f">微软博客今年 5 月透露 Insider build 22621 被选为 Windows 11 22H2（Sun Valley 2）的 RTM 候选版本，只要满足最低版本号为 Build 22621 的系统都可以作为最终版本交付，而且后续还可以在这个基础上进一步更新。</p><p data-vmark="68c2">此前有消息称，微软打算把最近的 22621.382 (KB5016632) 版本设定为 Windows 11 22H2 普遍可用版本 (GA)，但现在看来版本号可能还要更新一些。</p><p data-vmark="3228"><img src="https://img.ithome.com/newsuploadfiles/2022/8/ae151cb2-34f7-4b70-a3f0-70c1973451a7.jpg" w="820" h="547" title="有人买到预装 Win11 22H2 的新电脑，正式版本号为 Build 22621.521" width="820" height="547" referrerpolicy="no-referrer"></p><p data-vmark="206f">国外论坛 Thurott 上的网友 @WILL 近期购买了一台戴尔笔记本，不过令他感到惊讶的是该笔记本出现了全新的启动画面，最后发现该机内置系统为 Windows 11 22H2 Build 22621.521。</p><p data-vmark="efe6"><img src="https://img.ithome.com/newsuploadfiles/2022/9/2df3d836-55af-4c59-9eb6-35b943f2a3b1.jpg@s_2,w_820,h_322" w="923" h="363" alt="QQ截图20220917064307.webp" title="有人买到预装 Win11 22H2 的新电脑，正式版本号为 Build 22621.521" srcset="https://img.ithome.com/newsuploadfiles/2022/9/2df3d836-55af-4c59-9eb6-35b943f2a3b1.jpg 2x" width="923" height="322" referrerpolicy="no-referrer"></p><p data-vmark="471f">他表示：“我不确定这是不是真的，但我刚刚收到了一台装有 Windows 11 22H2 内部版本 22621.521 的戴尔笔记本电脑。我注意到它有些不同，因为它在启动时出现了新的圆形动画。有没有其他人拿到了预装 Windows 11 22H2 的新硬件？”</p><p data-vmark="ccc7">IT之家获悉，微软周二面向 Release Preview 通道用户发布了 KB5017321 累积更新，用户安装之后版本号升至 Build 22621.521。包括微软官方支持论坛在内的多个社区和社交平台上都有用户反馈成功安装该更新。</p><p data-vmark="9c89"><img src="https://img.ithome.com/newsuploadfiles/2022/9/0cb1e379-981e-4375-a89d-c586a51a3ced.png" w="459" h="422" alt="Win11 KB5017321(22621.521)" title="有人买到预装 Win11 22H2 的新电脑，正式版本号为 Build 22621.521" width="459" height="422" referrerpolicy="no-referrer"></p><p data-vmark="d773">近期，微软“入门”应用还无意中透露了 22H2 功能更新名称，被简单地称为“Windows 11 2022 更新”。然而，在意识到名称泄露后，微软公司已迅速将其删除。</p><p data-vmark="a944">对于运行原始版本的 Windows 11 21H2 用户，将可以无缝升级到 Windows 11 22H2，因为系统要求没有改变。在功能方面，Windows 11 22H2 将恢复备受期待的任务栏“拖放”功能支持。<span class="accentTextColor">它还将附带重新设计的任务管理器，该管理器将使用 WinUI 和 Fluent Design ，包括 Mica 和 Acrylic 材料</span>。</p><p data-vmark="2bb3">此功能更新还包括“开始”菜单改进、更快的上下文菜单、桌面版 Windows Spotlight 以及文件资源管理器中的标签支持（预计将在 22H2 之后发布）。</p>
