<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 17 日消息 腾讯安全应急响应中心昨日发布公告，<span class="accentTextColor">谷歌 Chrome 使用的 V8 引擎存在一处安全问题</span>，该问题可能会影响 Windows 微信及使用该引擎的软件。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/0efba657-6c21-46de-8c82-a788cee8de72.png" id="1618638467452-1" title="腾讯紧急发布 Window 版微信更新，修复谷歌 Chrome V8 引擎安全问题" referrerpolicy="no-referrer"></p><p>腾讯安全应急响应中心表示，现已紧急发布新版本微信修复该问题，微信会自动升级，用户也可以通过“设置”-“关于微信”-“升级版本”升级到最新版（<span class="accentTextColor">3.2.1.141 及以上</span>）。</p><p>IT之家的小伙伴可以检查一下自己的 Windows 版微信，尽快升级到新版本。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/8255b881-72d8-46a7-85e4-091c433afbde.png" id="1618638539656-2" title="腾讯紧急发布 Window 版微信更新，修复谷歌 Chrome V8 引擎安全问题" referrerpolicy="no-referrer"></p>
