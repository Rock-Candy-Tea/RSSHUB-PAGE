            <p data-vmark="991f"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 19 日消息，微软刚推送了一个新的 Windows 11 预览版。各位 Windows Insider Preview 的 Dev 注册用户已经可以接收到更新。</p><p data-vmark="8392">Windows 11 25120 拥有一项堪称“巨大”的变化，因为它标志着微软实现了对交互式桌面内容的支持。</p><p data-vmark="0041">简单来说，这是一种新的桌面集成工具。最直观的感受是，微软现在给你的桌面上增添了一个大大的搜索框。你更新完之后可以一眼看到这个搜索框，从而实现“在桌面上快速搜索网络内容”。</p><p data-vmark="3af6" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/65a6432b-c90a-4698-aa6f-5ced4b09e96c.png" w="1440" h="810" title="微软发布 Win11 Dev 预览版 25120：带来全新交互式桌面内容，支持 ARM64 设备（附下载）" width="1440" height="461" referrerpolicy="no-referrer"></p><p data-vmark="fa7a">“Dev Channel 的 Windows  预览体验成员可能更容易接受一些新的想法，更希望自己领先他人一步尝试新特性，以及帮助验证新概念的体验过程。从这个预览版开始，我们将开始探索 Windows 桌面的轻量级交互内容，为一些 Insider 人士带来这些概念特性之一。今天，Windows 在 Widgets 面板中实现了这种类型的内容。为了探索这个非同一般的想法和交互设计，我们在这个领域的做的第一步就是在桌面上添加一个搜索框，使您能够搜索网络。”</p><p data-vmark="1fb7">IT之家提醒，这个新功能还处于开发早期阶段，所以目前微软只能将搜索框放在桌面上。但不出意外的话，微软未来将为大家带来更多类似的“小部件”。</p><p data-vmark="209a">当然，微软也说了，如果你不喜欢的话仍然可以禁用这一功能，并鼓励每个人向其反馈，以帮助微软进一步完善 Win11 早期开发阶段的功能体验。</p><p data-vmark="ff5d">“如果您想删除此搜索框，可以右键单击桌面，选择“显示更多选项”，然后找到“显示搜索”选项并关闭。我们很希望了解您对此交互设计的反馈，因此，如果您收到此实验特性，请使用反馈中心提供反馈”。</p><p data-vmark="6d5d">除了这个新功能之外基本都是无关紧要的 Bug 修复内容，不过微软也提供了这一版本的<a href="https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewiso" target="_blank"> ISO 下载</a>（包括 ARM64 设备）。原文点此：<a href="https://blogs.windows.com/windows-insider/2022/05/18/announcing-windows-11-insider-preview-build-25120/" target="_blank">微软博客</a>。</p>
