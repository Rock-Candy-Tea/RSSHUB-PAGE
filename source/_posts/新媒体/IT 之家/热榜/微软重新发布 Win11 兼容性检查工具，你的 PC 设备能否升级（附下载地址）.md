<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 28 日消息 外媒 Windows Latest 报道，想把你的设备升级到 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>，但不确定 Windows 11 的兼容性？微软终于重新发布了 Windows 11 兼容性检查工具，现在可以很容易地弄清最新版本的操作系统是否与你的设备兼容。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/8c62e815-fb51-49df-9489-d81ecd982cf1.jpg@s_2,w_820,h_461" w="1440" h="810" title="微软重新发布 Win11 兼容性检查工具，你的 PC 设备能否升级（附下载地址）" srcset="https://img.ithome.com/newsuploadfiles/2021/8/8c62e815-fb51-49df-9489-d81ecd982cf1.jpg 2x" width="1440" height="461" referrerpolicy="no-referrer"></p><p>Windows 11 于 2021 年 6 月宣布，其新的设计功能将使其成为对一些用户有吸引力的升级。微软后来宣布了对其下一个操作系统的严格要求，很多用户不能从他们现有的 PC 免费升级到 Windows 11--因为它不符合必要的硬件要求。</p><p>英特尔第八代核心 CPU、Ryzen 2000 系列及以上的 PC 正式支持 Windows 11。根据 Windows Insiders 遥测数据，微软现在已经<a href="https://www.ithome.com/0/572/066.htm" target="_blank">增加了对部分英特尔第七代芯片的支持</a>。根据更新的列表，Windows 11 将支持在以下英特尔第七代芯片上运行。</p><ul class=" list-paddingleft-2"><li><p>英特尔酷睿 X 系列芯片。</p></li><li><p>英特尔 Xeon W 系列处理器。</p></li><li><p>英特尔酷睿 i7-7820HQ（曾用于 Surface Studio 2 等设备）。</p></li></ul><p>换句话说，微软正在放弃对旧的和广泛使用的英特尔第七代酷睿处理器的支持。英特尔第六代和其他处理器也不被支持。事实上，微软自己的 Surface 阵容中也有很多设备不符合 Windows 11 升级资格。</p><p>微软在硬件要求上立场坚定，看起来旧的处理器不会被正式支持。</p><h2>检查 Windows 11 的兼容性</h2><p>为了确保你的系统能够升级运行 Windows 11，微软开发了一款名为“PC Health Check”的工具。使用这个工具，用户可以检查他们的硬件和软件是否能与最新版本的 Windows 一起工作。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/60bd38cc-9f02-4398-9021-96ecab1755e9.jpg@s_2,w_820,h_680" w="1121" h="930" title="微软重新发布 Win11 兼容性检查工具，你的 PC 设备能否升级（附下载地址）" 健康检查工具，该工具此前在用户抱怨不能正常工作后被移除。</p><p>IT之家获悉，PC 健康检查工具作为微软 Windows Insider 预览版计划的一部分可供下载，现在可以安装它来检查你的硬件和软件是否能与 Windows 11 一起工作。如果你的设备配备了较新的处理器、TPM 2.0 和 UEFI 安全启动，则正式支持 Windows 11。</p><h2>如何使用 PC
