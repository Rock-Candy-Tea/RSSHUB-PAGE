<p data-vmark="834d"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 11 日消息 据网友反馈，腾讯 START 云游戏平台开始采用收费模式，此前免费的时候大部分游戏最高画质可以达到 1080P 120Hz。现在开始收费后只有部分游戏支持高帧率。</p><p data-vmark="ca15"><img src="https://img.ithome.com/newsuploadfiles/2021/9/9ee7fa0e-a600-4759-8372-20ae62cb7e02.png" w="1440" h="740" title="腾讯 START 云游戏开始收费：会员 251 元/年，仅《英雄联盟》等部分游戏支持高帧率" width="1440" height="421" referrerpolicy="no-referrer"></p><p data-vmark="39f8">根据腾讯 START 云游戏平台，会员可以享受不限时长、排队优先、跨端同享、专属福利等权益，套餐包括 251 元（12 个月，原价 449 元）、149 元（6 个月，原价 249 元）、29 元（1 个月，原价 49 元）、78 元（3 个月，原价 129 元）等选择。</p><p data-vmark="b3fe"><img src="https://img.ithome.com/newsuploadfiles/2021/9/ddacdb9b-c6fe-4cb0-aa26-c8b487f7dc0a.jpg@s_2,w_820,h_1093" w="1440" h="1919" title="腾讯 START 云游戏开始收费：会员 251 元/年，仅《英雄联盟》等部分游戏支持高帧率" srcset="https://img.ithome.com/newsuploadfiles/2021/9/ddacdb9b-c6fe-4cb0-aa26-c8b487f7dc0a.jpg 2x" width="1440" height="1093" referrerpolicy="no-referrer"></p><p data-vmark="25c9">在下面的几款游戏中，只有英雄联盟显示了支持高帧率。</p><p data-vmark="19cd"><img src="https://img.ithome.com/newsuploadfiles/2021/9/f0b97c98-04e2-4d8b-815c-e738d85a3004.jpg@s_2,w_820,h_601" w="1440" h="1056" title="腾讯 START 云游戏开始收费：会员 251 元/年，仅《英雄联盟》等部分游戏支持高帧率" srcset="https://img.ithome.com/newsuploadfiles/2021/9/f0b97c98-04e2-4d8b-815c-e738d85a3004.jpg 2x" width="1440" height="601" referrerpolicy="no-referrer"></p><p data-vmark="cd45">IT之家获悉，早在 2019 年的英伟达 GTC 大会上，英伟达 CEO 黄仁勋宣布与腾讯合作推出 START 云游戏服务。英伟达显卡将为腾讯的 START 云游戏平台提供支持，基于 PC 和主机游戏，支持 macOS / Windows / <a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a> TV 等平台，包括多款网游和端游。</p><p data-vmark="d00c">官网介绍：随时随地享受主机级云游戏体验！英雄联盟、拳皇 14、地下城与勇士等游戏无需下载，一键秒玩。现已支持 PC/TV/ 手机三端互通。</p>
