<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 30 日消息 微软此前发布了 Windows 11 系统，已经针对多种形式的因素进行优化，由于新的内置平板电脑模式和明显改善的手势体验，它在带有触摸屏的笔记本电脑/平板电脑上运行效果似乎很好。</p><p>Windows 11 的第一个预览版 Build 22000.51 现在已经在 Windows Insider Dev 频道中正式提供给配备英特尔、AMD 和 ARM（骁龙）处理器的 PC 设备。由于有了这个测试版，有开发者 Gustave Monce 现在已经成功地在 Lumia 950 XL 上安装了 Windows 11 ARM 预览版，以展示它在手机设备上的扩展性。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/65e5d7f6-478a-45b9-8570-57723457cbc6.jpg@s_2,w_820,h_891" w="992" h="1078" title="没有 Mobile 手机版了，微软官方 Win11 只适配 9 英寸屏幕及以上设备" srcset="https://img.ithome.com/newsuploadfiles/2021/6/65e5d7f6-478a-45b9-8570-57723457cbc6.jpg 2x" width="992" height="891" referrerpolicy="no-referrer"></p><p>虽然 Windows 11 为平板电脑/台式机设计，但即将推出的这款操作系统在智能手机上的扩展性依然很好。拥有 5.7 英寸显示屏的 Lumia 950 XL 能够启动新的 Windows 11，而且看起来很不错。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/8d055e74-fbe9-42e6-b15a-9c05417e374f.jpg" w="236" h="420" title="没有 Mobile 手机版了，微软官方 Win11 只适配 9 英寸屏幕及以上设备" width="236" height="420" referrerpolicy="no-referrer"></p><p>开发者还通过手动移植 Windows 10 Mobile 和 Windows 10 on ARM 项目的驱动程序，成功地在 Windows 11 中启用蜂窝网络连接。该黑客为 Windows 11 桌面操作系统带来了通话和短信支持。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/a89da1ed-e67d-4b6b-8839-664c47de1029.jpg" w="236" h="420" title="没有 Mobile 手机版了，微软官方 Win11 只适配 9 英寸屏幕及以上设备" width="236" height="420" referrerpolicy="no-referrer"></p><p>虽然这个项目可能并不理想，但这是一个令人印象深刻的壮举，它证明了 Lumia 950XL 仍然可以运行新 Windows 11 操作系统。</p><p>Windows 11 本质上是 Windows 10，它从微软被砍的 Windows 10X 中获得了很多灵感，同时仍然保留了 Windows 的外观和感觉。动态磁贴没有了，但图标已经针对触摸输入进行了优化。</p><p>另外值得注意的是，微软并没有在开发 Windows 11 Mobile 移动版。外媒 Windows Latest 报道，微软官方文件已经确认该操作系统只适用于 9 英寸显示屏及以上设备，这可能意味着微软不打算将其用于移动外形的触摸屏设备。</p>
