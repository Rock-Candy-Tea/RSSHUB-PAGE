            <p data-vmark="b401"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 24 日消息，本月初，微信 Windows 版迎来了 3.4.5 正式版，仅表示可以把免打扰的群收进「折叠的群聊」；置顶聊天过多时，可以将其折叠；修复了一些已知问题。</p><p data-vmark="e29c">IT之家网友现发现，微信 Windows 版本近日开始内测一项新功能 —— 网页版的文件传输助手，适合临时传输文件，比如打印店。</p><p data-vmark="4abe">如果你足够幸运，那么就可以在登录界面看到这个入口。</p><p data-vmark="961a">据称，他目前使用的版本是 Windows 客户端正式版 3.4.5.27，也就是说不需要最新版本也可以使用。</p><p data-vmark="3663">那么说了这么久，这个“网页版文件传输助手”究竟长什么样呢？我们一起来看一下。</p><p data-vmark="6c04">首先打开 Windows 版本微信的登录界面，你会发现右下角出现了一个“仅传输文件”的选项，如下图。</p><p data-vmark="8938"><img src="https://img.ithome.com/newsuploadfiles/2021/12/9054de84-f487-46b8-870c-b0838b7351d1.jpg" w="350" h="475" title="微信内测新功能：PC 网页版文件传输助手，不占用 Windows 版本在线名额" width="350" height="475" referrerpolicy="no-referrer"></p><p data-vmark="0aba">点击该按钮后跳转到网页，然后是常规的扫码登录微信账号，如下图。</p><p data-vmark="23c6"><img src="https://img.ithome.com/newsuploadfiles/2021/12/93393b6c-0392-465e-9834-b2830de7ba65.jpg" w="800" h="602" title="微信内测新功能：PC 网页版文件传输助手，不占用 Windows 版本在线名额" width="800" height="602" referrerpolicy="no-referrer"></p><p data-vmark="a64d">当你用手机扫码时，手机端界面显示正在打开“网页版文件传输助手”，你这时可选择打开或取消打开，避免被其他不法分子利用。</p><p data-vmark="2a0b"><img src="https://img.ithome.com/newsuploadfiles/2021/12/47f41692-6e31-49a4-ba86-3db67900db83.jpg@s_2,w_820,h_1756" w="1080" h="2313" title="微信内测新功能：PC 网页版文件传输助手，不占用 Windows 版本在线名额" 以及平板电脑上登录微信后会在手机端顶部有所提示，那么“网页版文件传输助手”也是同样，只是不占用你 PC 端的登陆名额罢了。</p><p data-vmark="b49c"><img src="https://img.ithome.com/newsuploadfiles/2021/12/5a4d9249-00e1-4362-a192-adc3de197ce4.jpg@s_2,w_820,h_1746" w="1080" h="2299" title="微信内测新功能：PC 网页版文件传输助手，不占用 Windows 版本在线名额"
