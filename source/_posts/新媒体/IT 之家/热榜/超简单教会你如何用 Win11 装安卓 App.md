<p data-vmark="9a12">用着 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 系统的网友们，你装上 Windows 11 <a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>子系统（Windows Subsystem for <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a>）没？有了它的帮助，让你的 Windows 也能运行安卓应用，真是太爽了。</p><p data-vmark="c918">不过目前 Windows 11 安卓子系统（Windows Subsystem for Android）的安装和使用有点复杂，这不许多网友就给复杂的操作劝退了。不要紧，这就来个简单的教程，让你轻松安装与使用安卓子系统，还有两个可以帮助你直接装安卓应用的小软件也一并奉上！</p><h2 data-vmark="3db3">三步快捷安装 Windows 11 安卓子系统</h2><p data-vmark="dd2a">怎么安装 Windows 11 安卓子系统呢？以前比较的复杂，现在可简单了，只需要三步即可完成安装。</p><p data-vmark="15c1">步骤一 首先需要将你的 Windows 11 升级到最新版（正式版、预览版均可），然后微软应用商店也升级到最新。</p><p data-vmark="bb95">步骤二 设置 → 时间和语言 → 语言 & 区域 → 国家或地区 → 切换到美国。安装完毕可以切换回中国。</p><p data-vmark="7fd1">步骤三 打开微软应用商店，搜索“Amazon Appstore”，然后下载安装。安装好 Amazon Appstore 后，会自动安装安卓子系统 Windows Subsystem for Android。</p><p data-vmark="b719">（懒得搜索的用户可以使用浏览器打开 <span class="link-text-start-with-http">http://aka.ms/</span> AmazonAppstore 这个网址，将直接跳转到应用商店）</p><p data-vmark="d0eb" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/85993678-1ca5-4c2c-979f-5ddbc7840a26.png" w="700" h="572" alt="Amazon Appstore" title="超简单教会你如何用 Win11 装安卓 App" width="700" height="572" referrerpolicy="no-referrer"></p><p data-vmark="c4f5">图 1 安装 Amazon Appstore</p><p data-vmark="8b63">对的，就是这么的简单就能完成 Windows 11 安卓子系统的安装操作了。</p><p data-vmark="4134">Windows Subsystem for Android 安卓子系统安装好了，那么怎么样往里边安装安卓应用呢？原本呢，咱就可以通过 Amazon Appstore 来安装安卓应用的，只是目前 Amazon Appstore 不支持中国区域，也就不能靠这个方案了。（当然你也可以注册个美国 Amazon 来体验。）</p><p data-vmark="d1ab">还有一种方式是需要用到 ADB 工具包以及命令行，太复杂了，装个应用这么复杂多麻烦啊。不要紧，这不，有大神们就给我们打造出了简单的安装工具，让你可以像安装 Windows 软件那样安装安卓应用，例如 WSA Pacman 或 WSA Tool。</p><p data-vmark="3eef">WSA Pacman 是一款傻瓜式的安卓应用安装程序，下载回来安装后，它就会自动关联安卓应用安装包，这时候用户只需要双击安卓应用安装包，就会出现安装提示，这里还可以看到该应用所需要的权限。像安装 Windows 软件一样安装安卓应用就这么简单。</p><p data-vmark="c7ca" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/6e3aed1e-a19f-44d7-a790-c6f8a2283ed7.png" w="486" h="328" alt="WSA-Pacman" title="超简单教会你如何用 Win11 装安卓 App" width="486" height="328" referrerpolicy="no-referrer"></p><p data-vmark="a8b1">图 2 安装 WSA-Pacman 后的安卓应用安装界面</p><p data-vmark="3c8c">如果你的 WSA-Pacman 在安装应用时“安装”按钮是灰色的不可按下，那你就得去检查一下下边的设置。</p><p data-vmark="8768" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/4707e939-2bd3-4c23-963f-e18f5082a2c4.png" w="486" h="328" alt="WSA-Pacman" title="超简单教会你如何用 Win11 装安卓 App" width="486" height="328" referrerpolicy="no-referrer"></p><p data-vmark="1a6b">图 3 “安装”按钮不可按下</p><p data-vmark="73b5">运行“适用于 Android 的 Windows 子系统设置”（开始菜单里找），会进入安卓子系统的设置界面，我们需要开启安卓子系统的“开发人员模式”。</p><p data-vmark="3e2c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/df1b982d-094e-4644-a3ef-50aa14c6e972.png" w="700" h="544" alt="WSA-Pacman" title="超简单教会你如何用 Win11 装安卓 App" width="700" height="544" referrerpolicy="no-referrer"></p><p data-vmark="9476">图 4 开启开发人员模式</p><p data-vmark="ef5d" style="text-align: center;"><img referrerpolicy="no-referrer"></p><p data-vmark="b8f8">图 5 成功安装安卓应用</p><p data-vmark="81b6">成功安装完毕，WSA-Pacman 还会提醒用户是否在桌面创建快捷方式。</p><p data-vmark="35f3" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/1a3fed7a-f362-45a8-859c-c3803be0e200.png" w="486" h="328" alt="WSA-Pacman" title="超简单教会你如何用 Win11 装安卓 App" width="486" height="328" referrerpolicy="no-referrer"></p><p data-vmark="b245">图 6 可以创建桌面快捷方式</p><p data-vmark="6fbd">WSA-Pacman 的设置选项也比较的简单，可以设置端口、语言、主题模式等。</p><p data-vmark="6f15" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/2b1f7cd6-7a0d-4ef7-ac7e-208b2fda8188.png" w="700" h="514" alt="WSA-Pacman" title="超简单教会你如何用 Win11 装安卓 App" width="700" height="514" referrerpolicy="no-referrer"></p><p data-vmark="5c94">图 7 WSA-Pacman 设置</p><p data-vmark="3023">第二款是来自 @Makazeu 的“WSA 工具箱”，它集成了安卓子系统的安卓应用安装管理功能，让用户可以轻松安装管理安卓应用。用户目前可以在微软商店中下载到它。</p><p data-vmark="b453" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/d82b063e-0ac6-42de-bcd0-aaa743dc7fb6.png" w="700" h="571" alt="WSA 工具箱" title="超简单教会你如何用 Win11 装安卓 App" width="700" height="571" referrerpolicy="no-referrer"></p><p data-vmark="9555">图 8 下载安装 WSA 工具箱</p><p data-vmark="2519">打开 WSA 工具箱可以看到上边一排的功能选项，有安装 APK 的，有管理 APP（可启动、关闭、创建快捷方式及卸载 APP）的，还有显示安卓子系统进程的，界面简单粗暴，非常易于使用。</p><p data-vmark="ae0f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/00ac1452-5244-43af-b7e8-c0405983aa5d.png" w="700" h="481" referrerpolicy="no-referrer"></p><p data-vmark="6812">图 10 已安装 App 管理</p><p data-vmark="d2dd" style="text-align: center;"><img
