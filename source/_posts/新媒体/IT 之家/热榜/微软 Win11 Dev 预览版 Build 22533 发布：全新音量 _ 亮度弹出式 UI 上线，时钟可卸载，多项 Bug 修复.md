            <p data-vmark="2222"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 13 日消息，微软今日面向 Dev 预览用户推送了 <span class="accentTextColor"><a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> Build 22533</span> 预览版本。微软表示，该版本重置了任务栏上的天气小部件，还进行了多项改进，ISO 文件<a href="https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewiso" target="_blank" title="点此下载">点此下载</a>。</p><p data-vmark="9e58">IT之家了解到，你的手机应用推出了全新的通话体验，包括一个新的正在进行的通话窗口，其中包含更新的图标、字体和其他 UI 更改，与 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的改进设计相一致。</p><p data-vmark="97a9"><img src="https://img.ithome.com/newsuploadfiles/2022/1/a4f6ccc9-0cc9-416e-9475-adbea3cda86d.png" w="337" h="370" alt="New in-progress call window with updated visuals in the Your Phone app." title="微软 Win11 Dev 预览版 Build 22533 发布：全新音量 / 亮度弹出式 UI 上线，时钟可卸载，多项 Bug 修复" width="337" height="370" referrerpolicy="no-referrer"></p><h2 data-vmark="eb77">变化和改进：</h2><ul class=" list-paddingleft-2"><li><p data-vmark="51f3">更新了亮度、音量、相机隐私、相机开 / 关和飞行模式的<span class="accentTextColor">硬件指示器的弹出设计</span>，以符合 Windows 11 设计原则。当用户按下笔记本电脑上的音量或亮度键时，这些新的弹出按钮将出现，并且将支持亮 / 暗模式，提供更连贯的 Windows 体验。</p></li></ul><p data-vmark="49a8"><img src="https://img.ithome.com/newsuploadfiles/2022/1/2c7e843c-a160-41b0-ac80-f87dec2c64a5.png" w="1440" h="161" alt="The redesigned hardware indicator for volume." title="微软 Win11 Dev 预览版 Build 22533 发布：全新音量 / 亮度弹出式 UI 上线，时钟可卸载，多项 Bug 修复" width="1440" height="92" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="1376">用户可以从任务栏搜索语音访问并将语音访问固定到任务栏，或像其他应用程序一样开始以及打开 / 关闭它。</p></li><li><p data-vmark="033d">正在将 13 种触摸键盘主题扩展到 IME、表情符号面板和语音输入（在 Build 22504 中首次引入）的推广范围，以向 Dev 频道中的所有 Windows 预览体验成员扩展。</p></li><li><p data-vmark="3f94">当用户按 WIN + X 或右键点击开始图标时，菜单中会显示“已安装的应用”而不是“应用和功能”。</p></li><li><p data-vmark="183e">时钟应用已支持卸载。</p></li></ul><h2 data-vmark="4447">修复：</h2><p data-vmark="25c0"><strong>常规：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="22f4">修复了内部人员在驱动程序或固件更新期间可能看到错误 0x8007012a 的问题。</p></li><li><p data-vmark="d47d">解决了导致内部人员有时无法登录某些应用程序的问题，例如反馈中心。</p></li><li><p data-vmark="78a4">修复了 Windows 安全应用中的漏洞利用保护描述中的文本，使其仅指 Windows 而不是 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a>。</p></li><li><p data-vmark="a3e4">修复了导致无法将某些相机和手机中的照片导入照片应用的问题（会永远循环显示到目前为止找到的 0 个项目）。</p></li><li><p data-vmark="480e">启动 Windows 沙盒，将其关闭，然后再次启动，不会再导致任务栏中出现两个 Windows 沙盒图标（其中一个无法正常工作）。</p></li></ul><p data-vmark="d2bc"><strong>任务栏：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="cbdb">Wi-Fi 图标现在应该更可靠地出现在任务栏中。</p></li><li><p data-vmark="cbe5">如果用户有多个显示器连接到 PC，并右键单击主显示器任务栏上的日期和时间，explorer.exe 将不再崩溃。</p></li><li><p data-vmark="c3e2">按住 CTRL 并将鼠标悬停在任务栏中的任务视图图标上不再导致 explorer.exe 崩溃。</p></li></ul><p data-vmark="10db"><strong>设置：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="dd17">缓解了与在“设置”中使用云母相关的潜在问题，该问题影响了近期版本中“设置”应用程序的整体可靠性。</p></li><li><p data-vmark="6eb3">修复了影响某些内部用户的问题，该问题导致在尝试访问已安装的应用、启动应用和默认应用页面时设置崩溃。</p></li><li><p data-vmark="b02c">解决了在为应用添加操作时导致设置中的鼠标页面崩溃的问题。</p></li><li><p data-vmark="d024">如果播放音频并反复点击“快速设置”中的音量滑块来更改音量，用户不会再听到噼啪声。</p></li></ul><p data-vmark="1e30"><strong>窗口：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="60e1">如果将鼠标悬停在 ALT + Tab 或任务视图中的窗口标题上，现在将出现显示完整窗口名称的工具提示。</p></li></ul><p data-vmark="66d3"><strong>输入：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="670a">改进了在候选窗口、表情符号面板和剪贴板上应用主题的文本颜色和按钮的外观（在此之前，某些按钮 / 文本在某些自定义背景颜色下很难看到）。</p></li><li><p data-vmark="4f18">点击麦克风图标调用语音输入后，语音输入启动器不应再意外重新出现。</p></li><li><p data-vmark="d231">对于具有更新的输入切换器体验的内部用户，放大镜和讲述人等辅助工具现在应该可以更好地使用。</p></li></ul><h2 data-vmark="4cf7">已知问题：</h2><p data-vmark="dd07"><strong>开始菜单：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="5857">在某些情况下，从开始菜单或任务栏进行搜索时可能无法输入文本。如果遇到此问题，请按键盘上的 WIN + R 启动“运行”对话框，然后将其关闭。</p></li></ul><p data-vmark="d35e"><strong>任务栏：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="e11f">切换输入法时，任务栏有时会闪烁。</p></li></ul><p data-vmark="d34c"><strong>搜索：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="f47c">在点击任务栏上的搜索图标后，搜索面板可能无法打开。如果发生这种情况，请重新启动“Windows Explorer”进程，并再次打开搜索面板。</p></li></ul><p data-vmark="a00c"><strong>设置：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="d67d">在查看可用的 Wi-Fi 网络列表时，信号强度指标不能反映正确的信号强度。</p></li><li><p data-vmark="a7e5">进入系统 > 显示 > HDR 时，设置可能会崩溃。</p></li><li><p data-vmark="3933">在蓝牙和设备下有一个空白条目。</p></li></ul><p data-vmark="a26b"><strong>小组件：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="e81b">改变任务栏的对齐方式会导致小组件按钮从任务栏上消失。</p></li><li><p data-vmark="cd33">当有多个显示器时，任务栏上的小组件内容可能会在显示器之间不同步。</p></li><li><p data-vmark="2710">任务栏左对齐时，不显示温度等信息，这将在未来的更新中修复。</p></li></ul>
