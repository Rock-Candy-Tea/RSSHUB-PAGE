<p data-vmark="9afa"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 16 日消息 在昨天的 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 13</a> 发布会上，苹果公司没有详细说明其新的 A15 仿生芯片所带来的相对性能提升，这在苹果的重大硬件发布会上是很罕见的。一份报告认为，这并不是苹果的疏忽，因为 A15 CPU 核心性能提升并不大，是苹果有意为之。</p><p data-vmark="fc55"><img src="https://img.ithome.com/newsuploadfiles/2021/9/05f14eb5-0dc3-4e75-9edd-81ef8186179e.jpg@s_2,w_820,h_461" w="1280" h="720" title="报告：iPhone 13 系列 A15 仿生芯片 CPU 性能缺乏重大提升是因苹果人才流失" srcset="https://img.ithome.com/newsuploadfiles/2021/9/05f14eb5-0dc3-4e75-9edd-81ef8186179e.jpg 2x" width="1280" height="461" referrerpolicy="no-referrer"></p><p data-vmark="4245">虽然苹果提到了其新的 A15 仿生芯片设计的某些改进，特别是与图形功能有关的改进，但高管们对 CPU 性能却保持沉默。在 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad mini</a> 的发布会上，<span class="accentTextColor">苹果说 A15 的 CPU 比该平板电脑上代的 A12 处理器快 40%，但比较到此为止</span>。</p><p data-vmark="57cc">在 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 13 和 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> mini 发布后不久发表的一份报告中，SemiAnalysis 认为，<span class="accentTextColor">苹果人才流失导致关键高管和人员从 A15 等项目中抽走后，苹果的芯片团队被内部问题困扰</span>。在对 A15 进行了基于总晶体管、估计芯片尺寸和其他指标的初步芯片分析后，该报告发现 CPU 没有重大改进，<span class="accentTextColor">并认为苹果的下一代核心被推迟到 2022 年。</span></p><p data-vmark="1e5b">IT之家了解到，虽然没有什么证据表明人才外流对苹果产生了影响，但该公司近年来确实失去了一些关键人物。苹果从 A7 到 A12X 的主要 A 系列芯片设计师杰拉德-威廉姆斯三世于 2019 年离开公司，创办了 Nuvia 公司，该公司后来被高通以 14 亿美元收购。苹果起诉了威廉姆斯，称他在苹果工作期间创办了 Nuvia，并从其队伍中招募人员，该芯片架构师否认了这些指控。</p><p data-vmark="0823">今天的报告还提到了 Rivos，这是一家 RISC-V 初创公司，该公司拥有一些前苹果高级工程师。“我们认为，由于苹果一直在经历的人员流动，苹果不得不推迟下一代 CPU 核心，”该报告说，“他们没有使用新的 CPU 核心，而是使用去年核心的修改版。”</p><p data-vmark="0823"><img src="https://img.ithome.com/newsuploadfiles/2021/9/eb10fc94-d994-4dc9-b938-f4dc1a5bc6f8.jpg@s_2,w_820,h_547" w="1024" h="683" alt="A15 CPU" title="报告：iPhone 13 系列 A15 仿生芯片 CPU 性能缺乏重大提升是因苹果人才流失" srcset="https://img.ithome.com/newsuploadfiles/2021/9/eb10fc94-d994-4dc9-b938-f4dc1a5bc6f8.jpg 2x" width="1024" height="547" referrerpolicy="no-referrer"></p><p data-vmark="966b">另外，苹果可以将其晶体管预算投入到改进 GPU 内核和硅设计中，以增强图形和相机相关功能。<span class="accentTextColor">SemiAnalysis 同意 GPU 性能的提升收益“相当可观”，但坚持认为缺乏 CPU 性能提升源于苹果内部问题。</span></p><p data-vmark="7054">初步的 <a href="https://www.ithome.com/0/575/798.htm" target="_blank">Geekbench 跑分</a>表明，与 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 12</a> 系列中的 A14 仿生处理器相比，<span class="accentTextColor">A15 CPU 单核性能大约提升 10%，多核性能提升 18%</span>，<span class="accentTextColor">GPU 性能则提升巨大，大约在 55% 左右</span>，符合苹果的说法。对 A15 的更多详细分析应该在下周 iPhone 13 和 iPad mini 发售后出现。</p>
