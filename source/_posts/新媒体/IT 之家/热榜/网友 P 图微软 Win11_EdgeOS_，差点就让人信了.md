            <p data-vmark="550a"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 12 日消息，Twitter 用户 FireCube 经常会泄露 Windows 即将推出的功能或发布一些有趣的内容，虽然很多时候是真的，但今天这张 “EdgeOS”Beta 截图欺骗了许多人。</p><p data-vmark="8f8b">FireCube 发布了一张基于 Windows 11 的“EdgeOS”Beta 的假想截图。这是故意伪造的，微软实际上并没有打造一款谷歌 Chrome OS 的竞争对手系统。</p><p data-vmark="df6c">在 Twitter 帖子中，FireCube 为其添加了一些描述，称“泄漏 微软计划在切换到新的基于 Web 技术驱动的 shell 后重新塑造 Windows 11 品牌”，并附有以下图片：</p><p data-vmark="590b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/828cdd87-bacc-4f8b-9bb8-558684970ceb.jpg?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzE0Mi5wbmc=,g_7,x_14,y_14,a_0,t_100" w="922" h="1920" title="网友 P 图微软 Win11“EdgeOS”，差点就让人信了" width="922" height="1708" referrerpolicy="no-referrer"></p><p data-vmark="54f4">屏幕截图是编辑过的 winver 窗口，但顶栏显示的还是“关于 Windows”，而不是“关于 EdgeOS”，而且 Build 版本号甚至没有正确排列或间隔。足以证明这是假冒的。</p><p data-vmark="3128" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/520cde69-b06c-4188-906e-8cd075bb811e.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzgyLnBuZw==,g_7,x_8,y_8,a_0,t_100" w="531" h="517" title="网友 P 图微软 Win11“EdgeOS”，差点就让人信了" width="531" height="517" referrerpolicy="no-referrer"></p><p data-vmark="09c0" style="text-align: center;">▲ 微软 Windows 10 版本的关于窗口</p><p data-vmark="2ab5">虽然这只是为了开个玩笑，但微软最近在 Edge 浏览器功能集相关的行动方面确实让“EdgeOS”变得非常可信。在过去的几个月里，<span class="accentTextColor">微软一直在向 Edge 浏览器添加大量额外的、甚至一些不必要的功能，使其本身成为一款 mini OS。</span></p><p data-vmark="c553">到目前为止，微软 Edge 浏览器已添加了完整的侧边栏，其中包含计算器、单位转换器以及照片应用，甚至还有指向整个 Office 套件和 Outlook 日历的链接。而且 Edge 现在也是一款游戏浏览器，微软称 Edge 是“最适合游戏玩家的浏览器”。微软以后也可能会添加更多此类功能。</p><p data-vmark="0cc7">《<a href="https://www.ithome.com/0/626/154.htm" target="_blank">微软自夸 Edge 是“最适合游戏玩家的浏览器”</a>》</p>
