            <p data-vmark="f741"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 21 日消息，据 MacRumors 报道，分析师郭明錤今天在推文中表示，苹果可能会在 2024 年推出其首款全面屏智能手机，其中“<a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 16 Pro”将是第一款配备屏下 Face ID 和屏下前置摄像头的苹果设备。不过，郭明錤也只是预测，两年三年后的事情苹果自己可能都没法确定是否采纳量产……</p><p data-vmark="0551" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/54f8bf4a-1b18-48f6-b552-c027a62f2721.jpg@s_2,w_820,h_461" w="1440" h="809" alt title="郭明錤：苹果 iPhone 16 / Pro 真全面屏手机将首发搭载屏下 Face ID 和前置摄像头" srcset="https://img.ithome.com/newsuploadfiles/2022/4/54f8bf4a-1b18-48f6-b552-c027a62f2721.jpg 2x" width="1440" height="461" referrerpolicy="no-referrer"></p><p data-vmark="d44e">郭明錤称，“我认为真正的全面屏 iPhone 将在 2024 年问世。2024 年的高端 iPhone 将采用屏下前置摄像头和屏下 Face ID。弱光条件不利于前置摄像头质量，而 ISP 和算法对于质量改进至关重要。”</p><p data-vmark="680b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/06c49500-f870-4752-ab28-541e742dc68e.jpg@s_2,w_820,h_1773" w="902" h="1950" title="郭明錤：苹果 iPhone 16 / Pro 真全面屏手机将首发搭载屏下 Face ID 和前置摄像头" srcset="https://img.ithome.com/newsuploadfiles/2022/4/06c49500-f870-4752-ab28-541e742dc68e.jpg 2x" width="902" height="1773" referrerpolicy="no-referrer"></p><p data-vmark="3578">郭明錤的评论建立在他之前的预测之上，即苹果短期内不会推出搭载屏下 Touch ID 的 iPhone 。郭明錤同意显示分析师 Ross Young 的说法，即屏幕下的 Face ID 将出现在 iPhone 16 上，而 Touch ID 不会出现。</p><p data-vmark="5b29">此前，郭明錤表示，苹果在 2024 年 iPhone 采用屏下 Face ID 的路线图“不是技术问题”，更可能是营销决策。但最新想法似乎表明，当前的硬件和软件限制也可能是苹果致力于完善屏下技术的一个因素。</p><p data-vmark="ecab">今年的高端 iPhone 14 Pro 机型预计将放弃刘海凹槽，取而代之的是药丸 Face ID 和单打孔前置相机。苹果的发展方向尚不清楚，但 2019 年的传言称，苹果已经制作了至少一款具有无边框屏幕的 iPhone 原型，而用于 Face ID 的 TrueDepth 摄像头传感器则安装在显示屏上方的薄边框中。</p>
