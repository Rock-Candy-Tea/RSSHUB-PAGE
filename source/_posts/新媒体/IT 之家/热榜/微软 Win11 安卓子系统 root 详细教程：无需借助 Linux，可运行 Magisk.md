<p data-vmark="39bd"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 28 日消息，前段时间 Magisk、LSPosed 等的开发者们表示正在为 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> <a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>子系统（WSA）研究 root。</p><p data-vmark="3782">据IT之家网友 @liyi820 投稿，近日 LSPosed 的 <a href="https://github.com/LSPosed/MagiskOnWSA" target="_blank">GitHub 页面</a>已经放出了安装教程，IT之家测试可<span class="accentTextColor">正常运行</span>，有一定技术能力的可以看 GitHub 页面的教程自行安装，下面为普通用户带来详细安装教程。</p><p data-vmark="9121"><img src="https://img.ithome.com/newsuploadfiles/2021/10/8ba41867-a665-4d62-8a0d-4c168680a26c.png" w="1440" h="765" title="微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk" width="1440" height="436" referrerpolicy="no-referrer"></p><p data-vmark="ac54">该方法是直接修改安卓子系统，因此<span class="accentTextColor">需要先卸载掉已经安装过的 WSA</span>。</p><p data-vmark="f781">接着下载IT之家已经构建好的 artifact，分为 <a href="https://pan.baidu.com/s/10fqynUOv944p90nv3BRvkg" target="_blank">x64 版</a>（提取码：xyhm）和 <a href="https://pan.baidu.com/s/1Tb5ZmdDdOgTQ8UFNfn1WGA" target="_blank">arm64 版</a>（提取码：r8ir），大家根据自己的处理器选择下载。</p><p data-vmark="c552">下载完成后，解压该 zip 包到一个合适的位置（不要随便放），然后在解压好的文件夹上右键，点击“<span class="accentTextColor">复制文件地址</span>”。</p><p data-vmark="04af"><img src="https://img.ithome.com/newsuploadfiles/2021/10/679f0e62-e929-4435-9e5c-24f475ffb882.png" w="447" h="475" title="微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk" width="447" height="475" referrerpolicy="no-referrer"></p><p data-vmark="08ea">在 Win11 的开始菜单徽标上右键，点击 Windows 终端（管理员），会弹出 PowerShell 终端。</p><p data-vmark="6c5c"><img src="https://img.ithome.com/newsuploadfiles/2021/10/706936a9-321d-4513-aec6-f3d7c080ec8d.png" w="681" h="433" title="微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk" width="681" height="433" referrerpolicy="no-referrer"></p><p data-vmark="35be">在 PowerShell 终端中输入<span class="accentTextColor">“</span><span class="accentTextColor">cd”+“空格”</span>，接着按 Ctrl + V 粘贴之前复制的地址，然后点回车。</p><p data-vmark="6073">接着输入以下命令行：</p><pre class="brush:javascript;toolbar:false">Add-AppxPackage -Register .\AppxManifest.xml</pre><p data-vmark="4128">然后回车，Win11 安卓子系统就安装完成了。</p><p data-vmark="01b8"><img src="https://img.ithome.com/newsuploadfiles/2021/10/9844e22b-5fb1-43e2-8bcc-4c9a8f82de04.png" w="1440" h="763" title="微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk" width="1440" height="434" referrerpolicy="no-referrer"></p><p data-vmark="4786">接下来，在开始菜单中找到 Windows Subsystem for <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a>，打开后启用开发者模式，并点击右上角的箭头图标以运行安卓子系统。</p><p data-vmark="b31b"><img src="https://img.ithome.com/newsuploadfiles/2021/10/f3d44886-ae04-4cbc-9348-b9b3a429b4df.png" w="357" h="79" title="微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk" width="357" height="79" referrerpolicy="no-referrer"></p><p data-vmark="a134"><img src="https://img.ithome.com/newsuploadfiles/2021/10/5235936a-ca50-4156-a68f-ca68ce516b5b.png" w="1280" h="986" title="微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk" width="1280" height="632" referrerpolicy="no-referrer"></p><p data-vmark="b9b0">如果提示无法启动安卓子系统：</p><p data-vmark="8c15"><img src="https://img.ithome.com/newsuploadfiles/2021/10/8dbe55ab-a284-4f77-8712-55e8d269ac44.png" w="512" h="253" title="微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk" width="512" height="253" referrerpolicy="no-referrer"></p><p data-vmark="42bc">▲ 来自IT之家之前的文章，作者：CraigTaylor</p><p data-vmark="fe7c">那么前往系统设置界面，"应用" -> "可选功能" -> "更多 Windows 功能" => 勾选 "Hyper-V" 和 "虚拟机平台"，确定重启系统即可。</p><p data-vmark="6686"><img src="https://img.ithome.com/newsuploadfiles/2021/10/49abf739-40a9-4df6-86f5-2163d4fdd23c.png" w="1323" h="810" title="微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk" width="1323" height="502" referrerpolicy="no-referrer"></p><p data-vmark="83e2">▲ 来自IT之家之前的文章，作者：CraigTaylor</p><p data-vmark="b583"><img src="https://img.ithome.com/newsuploadfiles/2021/10/6999c938-4224-4654-b7d5-355493e50ddc.png" w="412" h="415" title="微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk" width="412" height="415" referrerpolicy="no-referrer"></p><p data-vmark="69d6">▲ 来自IT之家之前的文章，作者：CraigTaylor</p><p data-vmark="d7b1">系统运行成功后，<span class="accentTextColor">需要本地安装 Magisk App</span>，我们用到的是经过预构建的版本（<a href="https://wwa.lanzoui.com/iMXLuvuxznc" target="_blank">点此下载</a>），请不要下载官方版 Magisk App，否则无法运行。</p><p data-vmark="08bf"><img src="https://img.ithome.com/newsuploadfiles/2021/1/20210123_074840_528.png" title="微软 Win11 安卓子系统 root 详细教程：无需借助 Linux，可运行 Magisk" referrerpolicy="no-referrer"></p><p data-vmark="ad77">接着使用 adb 命令行安装 Magisk App，或者其他工具也可以，不懂的话可以看IT之家之前由 CraigTaylor 发表的文章。</p><p data-vmark="ae16">《<a href="https://www.ithome.com/0/581/826.htm" target="_blank">微软 Win11 安卓子系统已支持运行IT之家、支付宝等 APK 应用（附手动侧载安装教程攻略）</a>》</p><p data-vmark="c4a6">安装好 Magisk 后，点击图标打开，<span class="accentTextColor">App 会提示需要修复运行环境，点击确定</span>，等待一会儿，窗口将会关闭。然后，我们重新打开 Magisk，就可以正常使用了。</p><p data-vmark="57b5"><img referrerpolicy="no-referrer"></p><p data-vmark="0ea7"><img src="https://img.ithome.com/newsuploadfiles/2021/10/13d78492-2437-41d5-876a-19a8e0b0ad53.png" w="772" h="696" title="微软 Win11 安卓子系统 root
