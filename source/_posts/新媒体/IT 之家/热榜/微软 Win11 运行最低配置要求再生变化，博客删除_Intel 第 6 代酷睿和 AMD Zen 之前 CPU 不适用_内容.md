<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 29 日消息 今天凌晨，微软面向 Dev 开发频道发布了第一个 Windows 11 Insider Preview 版本，即 Build 22000.51!  微软表示，随着我们在未来几个月内完成产品的定型，我们将与你一起验证这一体验。在这个早期预览中，<a href="https://www.ithome.com/0/559/232.htm" target="_blank">你将可以尝试上周展示的许多新功能</a>，但不是全部。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/7351ff96-2bc4-462d-9363-a1ffd95197c7.png" w="1440" h="960" title="微软 Win11 运行最低配置要求再生变化，博客删除“Intel 第 6 代酷睿和 AMD Zen 之前 CPU 不适用”内容" width="1440" height="547" referrerpolicy="no-referrer"></p><p>在今天的一篇博文中，微软再次尝试澄清 Windows 11 的最低配置要求，这让很多人感到困惑，特别是关于新操作系统将支持哪一代的 CPU。<span class="accentTextColor">该博文此前明确指出，英特尔第六代 Skylake 及更早的 CPU，以及非 AMD Zen 处理器，将不符合“围绕安全性和可靠性的原则以及 Windows 11 的最低系统要求”。</span>这与微软最初的声明一致，即 Windows 11 将需要英特尔第八代 Coffee Lake 或 AMD Zen 2 处理器及以上，但英特尔第七代 Kaby Lake 和 AMD Zen 1 用户还有一些希望。</p><p>在博文中，微软声称，Windows 11 不会在这些 CPU 上运行的原因是为了保持更高水平的安全性、稳定性和可靠性。</p><p>但需要注意的是，随后更新的博客文章，<span class="accentTextColor">微软删除了“Intel 第 6 代酷睿和 AMD  Zen 之前 CPU 不适用 ”的内容，但也没有再特别说明</span>。</p><p><img src="https://qimg.ithome.com/image/2021/06/29/6376058837877663542587872.jpg" title="微软 Win11 运行最低配置要求再生变化，博客删除“Intel 第 6 代酷睿和 AMD Zen 之前 CPU 不适用”内容" referrerpolicy="no-referrer"></p><p><img src="https://qimg.ithome.com/image/2021/06/29/6376058838458776258564537.png" title="微软 Win11 运行最低配置要求再生变化，博客删除“Intel 第 6 代酷睿和 AMD Zen 之前 CPU 不适用”内容" referrerpolicy="no-referrer"></p><p>▲此前的博文内容</p><p><img src="https://qimg.ithome.com/image/2021/06/29/6376058838210768555904861.png" title="微软 Win11 运行最低配置要求再生变化，博客删除“Intel 第 6 代酷睿和 AMD Zen 之前 CPU 不适用”内容" referrerpolicy="no-referrer"></p><p>▲更改后的博文内容</p><p>外媒表示，尽管出现了这种混乱，但微软显然试图在 Windows 更安全和其开放性之间走一条细线。很明显，安全是推动这些要求的动力，而且确实感觉微软想要砍掉那些不支持其安全推动的旧 CPU 型号设备。</p><p>IT之家获悉，微软还将“暂时删除”PC 健康检查应用程序，该应用旨在告诉运行 Windows 10 的用户他们的电脑是否能够更新到 Windows 11。微软表示，该应用程序“没有充分准备好分享用户所期望的细节或准确性水平”，但其表示，该应用程序将在今年秋季（可能是 10 月份）Windows 11 正式版发布时将再次可用。</p><p>《<a href="https://www.ithome.com/0/559/243.htm"
