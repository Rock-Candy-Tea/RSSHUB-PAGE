<p data-vmark="8adb"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 23 日消息，微软拒绝为基于 Arm 架构的苹果 M1、M1 Pro 和 M1 Max Mac 提供 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 版本，现在我们可能知道了原因 —— 微软与高通公司达成了一项秘密独家协议。</p><p data-vmark="8158">根据 XDA 开发者的说法，由于两家公司之前未知的交易，<span class="accentTextColor">基于 Arm 的 Windows 只能在高通 SoC 的设备上提供</span>。</p><p data-vmark="18d9"><img src="https://img.ithome.com/newsuploadfiles/2021/11/7b925166-4b28-4774-a408-d7ed433d5460.jpg" w="810" h="298" alt="Qualcomm has an exclusivity deal with Microsoft for Windows on ARM" title="为什么 Win11 ARM 不支持 M1 Mac，消息称微软与高通达成秘密独家协议" width="810" height="298" referrerpolicy="no-referrer"></p><p data-vmark="00d9">两位熟悉该交易的人士告诉 XDA，<span class="accentTextColor">该协议“将很快到期”，但没有具体说明何时结束</span>。届时，其他芯片供应商将可以生产 Arm Windows 设备，并且可能会让微软在苹果的 M 系列 Mac 上提供 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> ARM。</p><p data-vmark="c7bd">IT之家了解到，搭载 M 系列芯片的 Mac 不支持 Boot Camp，微软也表示无法安装 Win11 ARM。但目前来看，搭载 M 系列芯片的 Mac 可以通过虚拟机的方式体验 Win11 ARM。</p><p data-vmark="0574"><img src="https://img.ithome.com/newsuploadfiles/2021/11/ed3defd0-3697-423b-8629-a3eabfba95ca.png" w="776" h="430" title="为什么 Win11 ARM 不支持 M1 Mac，消息称微软与高通达成秘密独家协议" width="776" height="430" referrerpolicy="no-referrer"></p><p data-vmark="a1c2">另一个佐证是联发科的表态，上周，<span class="accentTextColor">联发科宣布将推出面向 Windows on ARM 市场的芯片，虽然计划处于早期阶段</span>，但也在一定程度上表明，未来 Win11 ARM 市场将迎来一系列变动。</p>
