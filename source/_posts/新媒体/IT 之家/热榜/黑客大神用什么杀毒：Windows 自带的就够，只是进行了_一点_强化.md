<p data-vmark="ca0e">这年头 Windows 电脑还需要杀毒软件吗？现在搜索这个问题，会发现很多建议都是“裸奔就行”。也就是说对于普通人，只用微软出厂自带的免费 Windows Defender 足够了。</p><p data-vmark="bed6">那么非普通人，比如网络安全工程师、黑客们自己用什么？没想到，他们也说 Windows 自带的就够了，只是需要再强化一下。一位之前在公司里做网络安全主管，现在自己单干漏洞猎人的黑客大神 h0ek 抛出这个观点，在 Hacker News 上引起热议。</p><p data-vmark="40da" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/042b012d-ed95-4c72-99ce-2a9cb420fbe5.png" w="962" h="106" alt="在Hacker News上引起热议" title="黑客大神用什么杀毒：Windows 自带的就够，只是进行了“一点”强化" width="962" height="90" referrerpolicy="no-referrer"></p><p data-vmark="4fc3">大神所说的强化也不复杂，就是开启了微软本来给企业客户准备的隐藏功能。以前他们公司用的方案就是 Windows Defender 配合微软 Azure 上的云原生防护服务 Sentinel。他发现一些高级功能其实在个人版和家庭版操作系统中也能开启，让个人电脑也拥有企业级防护。</p><h2 data-vmark="1a55">开启隐藏功能</h2><p data-vmark="e652">h0ek 所说的隐藏功能，全称为微软高级保护服务 MAPS。这是一种云安全防护，由知识图谱和机器学习驱动，用于识别病毒库里没出现过的全新威胁。</p><p data-vmark="0355">要启用这个功能需要用到组策略编辑器。在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a>/11 的专业版和企业版上，<span class="accentTextColor">按下 Win+R→输入”gpedit.msc”即可</span>。</p><p data-vmark="ff23" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/88bc5f54-04b2-4eab-af63-ab9c8e934c35.png" w="413" h="237" alt="输入”gpedit.msc”" title="黑客大神用什么杀毒：Windows 自带的就够，只是进行了“一点”强化" width="413" height="237" referrerpolicy="no-referrer"></p><p data-vmark="1659">在组策略编辑器中找到计算机配置模版→管理模版→Windows 组件→Windows Defender 防病毒→MAPS。进入条目并选择启用。下面的选项可以选基本或高级，选择基本只会上传少量必要的数据。</p><p data-vmark="2998" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/8e151ca9-1c06-4a1e-9aa4-2eb1b7691f7c.png" w="1080" h="713" alt="组策略编辑器" title="黑客大神用什么杀毒：Windows 自带的就够，只是进行了“一点”强化" width="1080" height="541" referrerpolicy="no-referrer"></p><p data-vmark="c06e">在个人版和家庭版上，组策略编辑器默认是用不了的，需要用脚本开启 MAPS 功能。以管理员权限打开 PowerShell, 输入“Get-MpRreference”查看安全设置。其中 MAPSReporting 控制 MAPS 的设置，0 是关闭，1 是基本模式，2 是高级模式。</p><p data-vmark="449f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/c3bf7c14-193f-4a23-96d3-e0635af41b85.png" w="1080" h="909" alt="控制 MAPS 的设置" title="黑客大神用什么杀毒：Windows 自带的就够，只是进行了“一点”强化" width="1080" height="690" referrerpolicy="no-referrer"></p><p data-vmark="339f">输入“Set-MpPreference -MAPSReporting 2”就可以开启高级模式。</p><p data-vmark="9c76" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/b00c39ab-a95f-40be-bd70-718bb3dec85d.png" w="848" h="424" alt="开启高级模式" title="黑客大神用什么杀毒：Windows 自带的就够，只是进行了“一点”强化" width="848" height="410" referrerpolicy="no-referrer"></p><p data-vmark="9bae">在这里还可以进行一些其他设置，比如 SignatureUpdateInterval 控制签名更新的间隔，h0ek 推荐设置成每小时 1 次。</p><p data-vmark="de85">CloudBlockLevel 控制云端阻拦级别，可以设置成 0，1，2，4，6，最高为 6，会阻拦所有未知的可执行程序。</p><p data-vmark="e73e" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/65f08902-8ec9-4725-9ab9-e768958bc6cf.png" w="1080" h="429" title="黑客大神用什么杀毒：Windows 自带的就够，只是进行了“一点”强化" width="1080" height="326" referrerpolicy="no-referrer"></p><p data-vmark="77b7">下一步是开启勒索软件专项防护。这个就更简单了，直接在 Windows 安全控制面板里就有。</p><p data-vmark="3a7c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/ec8b9aa8-076f-4b5c-a686-32fcb4808c19.gif" w="900" h="697" alt="开启勒索软件专项防护" title="黑客大神用什么杀毒：Windows 自带的就够，只是进行了“一点”强化" width="900" height="635" referrerpolicy="no-referrer"></p><p data-vmark="7486">最后，h0ek 觉得网上总有人吹嘘自己不用杀毒软件、以“裸奔”为荣，还有人觉得免费软件都是垃圾，这都没必要。在他看来，<span class="accentTextColor">Windows Defender 现在比最早刚推出时候进步了不少</span>，加上强化以后，对普通用户来说足够了。</p><h2 data-vmark="2503">有杀软也要带脑子上网</h2><p data-vmark="dedb">有一位微软安全中心的员工看到这个分享，也跑来自夸一波。作为工程师，我自己也承认这个产品真的很好。他还爆料，产品的进步来自微软这些年并购了很多小型安全公司。比如最开始他负责的项目只有自己一个人在干，现在团队在全球有数百名成员。</p><p data-vmark="a022">这下可好，逮着一个内部员工，吸引了更多人来提问和提意见。比如 Nim 编程语言的开发者就来求助。因为使用 Nim 的黑客太多了，现在 Windows Defender 直接把 Nim 的安装程序都误报成了病毒，提交了误报核查目前还没有结果。</p><p data-vmark="e9da" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/04c2d666-4f7c-4b6a-b54d-cc579c050587.png" w="1006" h="576" alt="更多人来提问和提意见" title="黑客大神用什么杀毒：Windows 自带的就够，只是进行了“一点”强化" width="1006" height="470" referrerpolicy="no-referrer"></p><p data-vmark="f7d2">也有 Web 开发者来吐槽，说 Windows Defender 的安全防护能力确实够用，就是硬盘读写性能太差。尤其是用 npm 安装的包文件太多，这是他还使用付费杀毒软件的唯一原因了。</p><p data-vmark="0a8b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/ff5e41e7-258b-425c-9828-e4590fb908a7.png" w="982" h="272" alt="网友评论" title="黑客大神用什么杀毒：Windows 自带的就够，只是进行了“一点”强化" width="982" height="227" referrerpolicy="no-referrer"></p><p data-vmark="65f8">对于这个情况，有评论建议他把开发用的目录从杀毒扫描范围中排除出去，反正恶意程序也不会在那里出现。他认为不行，因为现在 npm 包里藏病毒这种事也不少见了。</p><p data-vmark="b025" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/421dc2e4-0838-40c9-b121-db9497529284.png" w="908" h="184" alt="网友评论" title="黑客大神用什么杀毒：Windows 自带的就够，只是进行了“一点”强化" width="908" height="166" referrerpolicy="no-referrer"></p><p data-vmark="52b0">除了 Web 开发常用的 npm，机器学习常用的 Python 包管理工具 PyPI 里也藏有病毒，伪装成常见软件包，不小心输错名字就会中招。</p><p data-vmark="f124">正如 h0ek 最后提醒大家的那样，没有完美的解决方案。开车需要考驾照，上网不需要考试但也得自己注意，不要点击来历不明的链接，保持警惕。人类自己比他的电脑更容易攻破。</p><p data-vmark="b3b3"><strong>参考链接：</strong></p><p data-vmark="a12c">[1]<span
