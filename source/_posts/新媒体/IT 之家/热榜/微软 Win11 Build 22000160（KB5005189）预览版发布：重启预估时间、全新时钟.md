referrerpolicy="no-referrer"></p><p>《<a href="https://www.ithome.com/0/570/410.htm" target="_blank">微软 Win11 全新时钟应用发布：视觉 UI 更漂亮，支持 Focus Sessions</a>》</p><ul class="ai-word-checked list-paddingleft-2"><li><p>微软一直在测试在“开始”菜单下的电源菜单、重启通知、Windows Update 设置页面以及显示在任务栏右下角的 Windows Update 图标内查看重新启动更新所需时间的估计能力。微软正在为这个功能做一个小的调整，以便它只显示在有 SSD 固态硬盘的电脑上。如果你的电脑是普通的标准的 HDD 机械硬盘，你将不再看到估算时间。IT之家获悉，微软希望，一旦解决了一些更多的错误 Bug，就能将估算时间功能带回配置 HDD 机械硬盘的电脑上。</p></li></ul><p><img src="https://qimg.ithome.com/newfiles/2021/8/3112343b-c8c0-4deb-8881-70471fba7e68.jpg" title="微软 Win11 Build 22000.160（KB5005189）预览版发布：重启预估时间、全新时钟" referrerpolicy="no-referrer"></p><h2>修复和改进</h2><p><strong>任务栏：</strong></p><ul class=" list-paddingleft-2"><li><p>修复了“使用中的位置”图标有时会显示在任务栏上的问题，尽管你的偏好设置阻止了使用。</p></li></ul><h2>已知问题</h2><ul class="ai-word-checked list-paddingleft-2"><li><p>[提醒] 当从 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 升级到 Windows 11 或安装 Windows 11 的更新时，一些功能可能被废弃或删除。<a href="https://www.ithome.com/0/559/247.htm" target="_blank">请看这里的细节</a>。</p></li><li><p>微软正在调查一个问题，在某些设备上，当进入设置 > Windows Update>Windows Insider Program 时，只有“停止获取预览版”选项是可见的。这使得内测用户无法选择一个通道。微软已经在 Answers 上发布了解决方法。</p></li><li><p>[BETA 通道] 微软正在调查来自 Beta 通道的内部人员的报告，他们在升级到 Windows 11 后，没有看到新的任务栏，而且“开始”菜单也不工作。为了解决这个问题，如果你受到了影响，请尝试进入 Windows Update > 更新历史，卸载 Windows 最新累积更新补丁，然后通过检查更新重新安装。</p></li></ul><p><strong>开始：</strong></p><ul class=" list-paddingleft-2"><li><p>在某些情况下，在使用“开始”或“任务栏”中的搜索时，你可能无法输入文本。如果你遇到这个问题，按键盘上的 WIN + R 键，启动“运行”对话框，然后关闭它。</p></li><li><p>右键单击“开始”按钮（WIN + X）时，系统和 Windows Terminal 会丢失。</p></li></ul><p><strong>任务栏：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>切换输入法时，任务栏有时会闪动。</p></li></ul><p><strong>搜索：</strong></p><ul class=" list-paddingleft-2"><li><p>点击任务栏上的搜索图标后，搜索面板可能无法打开。如果发生这种情况，请重新启动“Windows Explorer”进程，并再次打开搜索面板。</p></li><li><p>搜索面板可能显示为黑色，在搜索框下面不显示任何内容。</p></li><li><p>在支持手写笔的设备上，应用程序可能无法从搜索面板上启动。如果你遇到这个问题，请退出并重新登录以解决这个问题。</p></li></ul><p><strong>设置：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>使用“设置”中的搜索框进行的某些搜索可能会崩溃掉设置。</p></li><li><p>微软正在调查有配对蓝牙 LE 设备的用户报告，遇到了蓝牙可靠性问题增加，以及从休眠状态恢复后或蓝牙关闭后的错误检查。</p></li></ul><p><strong>小组件：</strong></p><ul class=" list-paddingleft-2"><li><p>小组件面板可能显示为空。为了解决这个问题，你可以退出，然后再次登录。</p></li><li><p>小组件可能在外部显示器上以错误的尺寸显示。如果你遇到这种情况，可以通过触摸或 WIN + W 快捷方式先在你的实际 PC 显示器上启动小组件，然后在你的辅助显示器上启动。</p></li><li><p>[家庭小组件] 一些用户可能会看到“连接设备以查看屏幕时间活动”的消息，即使屏幕时间设置已启用。</p></li><li><p>[家庭小组件] 位置信息可能对 iOS 上的某些用户不可用。</p></li></ul><p><strong>微软商店：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>微软正在努力改善商店的搜索相关性，包括解决在某些情况下搜索结果排序不准确的问题。</p></li><li><p>在某些有限的情况下，安装按钮可能还不能发挥作用。</p></li><li><p>一些应用程序的评级和评论不可用。</p></li></ul><p><strong>Windows Sandbox：</strong></p><ul class=" list-paddingleft-2"><li><p>在 Windows Sandbox 中，点击任务栏上的切换器图标后，语言输入切换器不会启动。作为一种解决方法，用户可以通过以下任何一个硬件键盘快捷键来切换他们的输入语言。Alt + Shift、Ctrl + Shift 或 Win + Space（第三个选项只有在 Sandbox 是全屏情况下才可用）。</p></li><li><p>在 Windows Sandbox 内，点击任务栏上的 IME 图标后，IME 上下文菜单不会启动。作为变通方法，用户可以通过以下任一方法访问 IME 上下文菜单的功能。</p></li></ul><p>通过设置 > 时间和语言 > 语言和地区 >（如日语）...> 语言选项 >（如微软 IME）...> 键盘选项访问 IME 设置。</p><p>作为选择，你也可以启用 IME 工具栏，这是一个快速调用特定 IME 功能的替代用户界面。从上面继续，导航到键盘选项 > 外观 > 使用 IME 工具条。</p><ul class="ai-word-checked list-paddingleft-2"><li><p>使用与每个 IME 支持的语言相关的独特的硬件键盘快捷键。</p></li></ul><p><strong>本地化问题:</strong></p><ul class=" list-paddingleft-2"><li><p>有一个问题，即一些内测用户在运行最新的 Insider Preview 构建的一小部分语言的用户体验中可能会有一些翻译缺失。</p></li></ul><p><strong>微软 Teams 的聊天：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>体验仅针对英语（美国）进行了本地化。其他语言和地区即将推出。</p></li><li><p>当你拨打电话时，虽然你没有听到铃声，但用户界面显示电话正在接通。</p></li><li><p>在视频通话中，有时人们的视频会冻结或显示一个黑色图像。这个问题有一个解决方法，就是把冻结的视频固定住，然后解开固定来解决这个问题。</p></li><li><p>在呼叫之间切换时，前一个呼叫不会自动搁置，因此音频和视频流在两个呼叫中都会继续。请确保在接听另一个电话之前完成通话。</p></li></ul>
