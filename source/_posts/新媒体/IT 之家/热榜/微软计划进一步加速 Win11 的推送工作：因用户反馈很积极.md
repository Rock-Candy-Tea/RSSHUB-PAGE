<p data-vmark="23c1"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 20 日消息，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 遵循微软公司典型的分阶段式推出方式，并不是所有人第一时间都会收到该更新。微软公司认为，在向所有人推出之前，先向一小部分人推送 Windows 11 或其它大的更新，是最安全的选择。</p><p data-vmark="2a37"><img src="https://img.ithome.com/newsuploadfiles/2021/11/000cf2fd-7a08-4246-b3f1-bd48fcf009fb.jpg" w="696" h="365" title="微软计划进一步加速 Win11 的推送工作：因用户反馈很积极" width="696" height="365" referrerpolicy="no-referrer"></p><p data-vmark="c8e9">Windows 11 目前还是存在不少 Bug，例如，微软最近对使用某些版本的英特尔 SST 音频控制器驱动程序的设备设置了升级障碍，因为其会导致蓝屏错误。</p><p data-vmark="8e71">不过，由于整体反馈并不负面，微软正在扩大 Windows 11 的推送范围。在一份新的支持公告中，<span class="accentTextColor">微软解释说，Windows 11 已经为更多支持的设备做好准备。</span></p><p data-vmark="0a4e"><span class="accentTextColor">“我们推进推送的速度比我们之前预期的要快，”</span> 微软说，推送经验和用户反馈都是积极的，该公司现在正将 Windows 11 广泛提供给更多符合条件的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 硬件。</p><p data-vmark="f137">值得指出的是，微软只在符合条件的设备上提供 Windows 11。如果你有不支持的硬件，需要通过修改 Windows 注册表来手动绕过这些限制。</p><p data-vmark="22c4">Windows 11 带来了大量的设计改进，如新的开始菜单、任务栏通知中心，以及微软商店、Windows 设置等功能，为混合处理器提供更好的资源管理，改进操作系统的各个领域等等。</p><p data-vmark="7f00">当然，Windows 11 也带来了自己的错误、问题和限制，其中一些将在未来的版本中修复。然而，Windows 11 是一个可选的升级，用户可以继续使用 Windows 10，后者将每年更新一些改进。</p><p data-vmark="f30b">IT之家了解到，微软对 Windows 功能更新的推出越来越谨慎，但由于用户的积极反馈，微软还计划将 Windows 10 功能更新推送到旧版本的 Windows 10。值得注意的是，Windows 10 的 2004 版将很快停止支持。</p>
