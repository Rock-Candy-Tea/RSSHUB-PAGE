<p data-vmark="1116"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 2 日消息，据 Windows Latest 报道，Windows 11 太阳谷 2 更新设置为以现代外观刷新任务管理器界面，这更符合整个操作系统的 WinUI 实现的其余部分。在最新的 Windows 11 预览版中包括一个新的彩色任务管理器。</p><p style="text-align: center;" data-vmark="543f"><img src="https://img.ithome.com/newsuploadfiles/2022/5/aa6d8264-02ea-4406-9000-192cae10fd64.jpg" w="696" h="365" title="微软 Win11 全新彩色任务管理器上手体验：更现代，支持黑暗模式和强调色" width="696" height="365" referrerpolicy="no-referrer"></p><p data-vmark="075d">与从头开始重写的任务栏不同，微软并没有为 Windows 11 创建新的 UWP 任务管理器。相反，该公司为任务管理器的现有 Win32 框架带来了 WinUI 风格外观。这也是微软计划改造 Windows 系统的一种方式。</p><p data-vmark="0d72">正如图片所示，微软的新任务管理器终于感觉是 Windows 11 风格了。<span class="accentTextColor">该公司已经用汉堡菜单替换了用于在进程、性能和其他部分之间切换的选项卡</span>。汉堡菜单出现在左侧。</p><p style="text-align: center;" data-vmark="d11e"><img src="https://img.ithome.com/newsuploadfiles/2022/5/3231445a-2ef0-4e5e-82b1-b31aa9a85b8d.jpg@s_2,w_820,h_616" w="1376" h="1033" title="微软 Win11 全新彩色任务管理器上手体验：更现代，支持黑暗模式和强调色" srcset="https://img.ithome.com/newsuploadfiles/2022/5/3231445a-2ef0-4e5e-82b1-b31aa9a85b8d.jpg 2x" width="1376" height="616" referrerpolicy="no-referrer"></p><p data-vmark="3dd9">左侧滑动菜单使任务管理器更像大多数现代 Windows 11 应用程序。</p><p data-vmark="d15f">任务管理器还获得了对黑暗模式和系统强调色的支持。</p><p data-vmark="1413">与 Windows 10 一样，Windows 11 也为各种屏幕元素使用“强调色”。你需要从个性化选项卡中选择强调色，或者你也可以选择“自动从我的背景中选择强调色”以自动将颜色与背景匹配。</p><p style="text-align: center;" data-vmark="1efa"><img src="https://img.ithome.com/newsuploadfiles/2022/5/866622ef-cb00-48f6-a6f2-698f88eee94e.jpg@s_2,w_820,h_613" w="1292" h="966" title="微软 Win11 全新彩色任务管理器上手体验：更现代，支持黑暗模式和强调色" srcset="https://img.ithome.com/newsuploadfiles/2022/5/866622ef-cb00-48f6-a6f2-698f88eee94e.jpg 2x" width="1292" height="613" referrerpolicy="no-referrer"></p><p data-vmark="c9fc">强调色适用于“开始”菜单、任务栏、通知中心、按钮和操作系统的其他区域。相同的强调色功能现在正在前往任务管理器，它支持大多数颜色，但该公司表示它将仅适用于进程选项卡，至少目前如此。</p><p style="text-align: center;" data-vmark="985a"><img src="https://img.ithome.com/newsuploadfiles/2022/5/8910d01b-ae74-4502-aadd-9723cc26a9ee.jpg@s_2,w_820,h_625" w="1321" h="1007" title="微软 Win11 全新彩色任务管理器上手体验：更现代，支持黑暗模式和强调色" srcset="https://img.ithome.com/newsuploadfiles/2022/5/8910d01b-ae74-4502-aadd-9723cc26a9ee.jpg 2x" width="1321" height="625" referrerpolicy="no-referrer"></p><p data-vmark="cc52">由于可访问性和可读性问题，强调色目前不适用于性能等选项卡。</p><p data-vmark="e8c8">微软没有提到任务管理器的其他调整，而且 Windows 11 Build 22610 预览版中的版本似乎还处于早期，因为还没有完全准备好。据报道，微软计划在该应用程序中引入一些附加功能。</p><p data-vmark="c76a">例如，任务管理器正在获得新部分来显示有关“应用程序健康”和“电池健康”的专用信息。</p><p data-vmark="33f1">鉴于操作系统的模块化特性，如果在面向公众发布功能更新时会添加其他功能，也在预期之中。</p><p data-vmark="ec19">在网络直播中，微软开发人员表示这不是最终设计，因此 Windows 11 新的任务管理器将获得更多的设计改进和功能。</p>
