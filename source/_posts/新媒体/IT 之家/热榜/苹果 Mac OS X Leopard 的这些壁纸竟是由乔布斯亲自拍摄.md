<p>IT之家 4 月 10 日消息 一名前苹果软件设计师声称，乔布斯亲自拍摄了 2007 年 Mac OS X Leopard 的几张壁纸。</p><p>以 “Cricket”为名的博主发布的一篇新博客文章称，<span class="accentTextColor">包括草丛、岩石花园和金色宫殿在内的 OS X Leopard 壁纸或桌面图片都是乔布斯拍摄的</span>。该博主表示，他在苹果工作了近 20 年。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210410_152620_33.jpg@base@tag=imgScale&w=850&h=512&q=90" w="850" h="531" title="苹果 Mac OS X Leopard 的这些壁纸竟是由乔布斯亲自拍摄" srcset="https://img.ithome.com/newsuploadfiles/2021/4/20210410_152620_33.jpg 2x" width="850" height="512" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210410_152830_924.jpg@base@tag=imgScale&w=1280&h=514&q=90" w="1280" h="802" title="苹果 Mac OS X Leopard 的这些壁纸竟是由乔布斯亲自拍摄" srcset="https://img.ithome.com/newsuploadfiles/2021/4/20210410_152830_924.jpg 2x" width="1280" height="514" referrerpolicy="no-referrer"></p><p>“Cricket”展示了五张乔布斯拍摄的 OS X Leopard 壁纸图片，他还展示了另外三张乔布斯拍摄的照片，但 “无法确认 (或记得)这些照片是否是 Mac OS X 的壁纸”。</p><p>IT之家了解到，外媒 AppleInsider 证实 “Cricket”确实曾在苹果工作过，并获悉，<span class="accentTextColor">史蒂夫 - 乔布斯并没有因为在 OS X Leopard 中使用他的摄影作品而获得功劳。</span></p>
