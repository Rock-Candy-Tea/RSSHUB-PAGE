            <p data-vmark="ad31"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a>7 月 9 日消息，据 SamMobile 报道，三星 One UI 5.0 Beta 将于 7 月的第三周在三星 Galaxy S22 系列上测试，之后扩展到其他机型。同时，三星计划于 10 月推送正式版。而在官方推送前，9to5Googole 今日曝光了 One UI 5.0 的大量截图。</p><p data-vmark="b98e" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/50c7348a-49a4-4c10-9acf-ea30761dcb0e.png" w="688" h="738" alt="图片 1" title="三星基于 Android 13 的 One UI 5.0 大量截图曝光，可提取图片中文字" width="688" height="738" referrerpolicy="no-referrer"></p><p data-vmark="2bac">9to5Googole 称，该版本是早期测试版并且没有更新日志，正式版更新内容可能会有所不同。对比三星 One UI 4.0，在通知页面，One UI 5.0 将会拥有新的通知图标，并且通知设置和清理两个选项的透明度有所降低，方便用户交互。</p><p data-vmark="5715" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/f0214c84-5e74-4415-bc70-7875d730c461.png" w="689" h="747" alt="图片 2" title="三星基于 Android 13 的 One UI 5.0 大量截图曝光，可提取图片中文字" width="689" height="747" referrerpolicy="no-referrer"></p><p data-vmark="00a3">在权限弹窗界面，One UI 5.0 采用了<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>默认的权限对话框，弹窗位置从手机底部改为了画面正中央。但 9to5Googole 称，这也有可能是三星为了加快 One UI 5.0 的开发进度，未曾改动这些相对不重要的界面。</p><p data-vmark="c50f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/76aa7e98-883d-420d-ac92-c8ffbcfa28b4.jpeg" w="473" h="1024" alt="One UI 5 Gallery OCR" title="三星基于 Android 13 的 One UI 5.0 大量截图曝光，可提取图片中文字" width="473" height="1024" referrerpolicy="no-referrer"></p><p data-vmark="fdc0"><span class="accentTextColor">One UI 5.0 比较重要的升级是三星在相册中增加了 OCR 功能</span>。当系统检测到图片中的文字时，它会在图片底部出现一个按钮，提醒用户可以提取图片中的文字。这曾经是 Bixby Vision 的功能，但在 One UI 5.0 中被整合在了系统底层。</p><p data-vmark="0db5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/d3d4fed4-984a-41d5-ae6d-d29ea92b274a.png" w="700" h="759" alt="图片 4" title="三星基于 Android 13 的 One UI 5.0 大量截图曝光，可提取图片中文字" width="700" height="759" referrerpolicy="no-referrer"></p><p data-vmark="6d60">而键盘 OCR 的工作方式与苹果 iOS 相似，在短信界面，用户可以用相机识别文字并提取到输入框中，方便用户交流。</p><p data-vmark="f4f5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/ec1f4e47-c272-4330-a276-6c715f691eba.jpeg" w="473" h="1024" alt="One UI 5 Security and Privacy Hub" title="三星基于 Android 13 的 One UI 5.0 大量截图曝光，可提取图片中文字" width="473" height="1024" referrerpolicy="no-referrer"></p><p data-vmark="5693">三星拥有定制版安全和隐私中心，它将用户的所有账户、密码、安全和隐私功能放在同一界面上，使用户能够快速访问。但相对于 One UI 4.0，其并没有什么新功能，所有功能在 One UI 4.0 中都能够找到。</p><p data-vmark="f5e1" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/68d2d256-4edd-4248-91d2-72967f2c567a.png" w="698" h="758" alt="图片 6" title="三星基于 Android 13 的 One UI 5.0 大量截图曝光，可提取图片中文字" width="698" height="758" referrerpolicy="no-referrer"></p><p data-vmark="585e">在实验室界面，One UI 5.0 显示用户可以通过双指从屏幕底部上滑来开启分屏，或者从右上角滑入屏幕开启小窗。但该功能在当前版本并不能使用，仅为演示页面，有待后续更新。</p><p data-vmark="511b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/c384f411-1ffe-4b30-8815-725536456c5b.png" w="688" h="359" alt="图片 7" title="三星基于 Android 13 的 One UI 5.0 大量截图曝光，可提取图片中文字" width="688" height="359" referrerpolicy="no-referrer"></p><p data-vmark="3008">其他方面，One UI 5.0 还更新了三星笔记的协作功能、快速设置当前活动 App、关于手机界面和 UWB 切换设置。</p><p data-vmark="a041">IT之家了解到，谷歌此前已经发布了 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android 13</a> 开发者预览版，包括外观改进、新的媒体播放器和输出选择器以及 Google Material You 配色方案扩展。同时，<a href="https://www.ithome.com/0/628/195.htm" target="_blank">有爆料称三星已经提供 One UI 5.0 Beta 测试版给内部员工</a>。</p>
