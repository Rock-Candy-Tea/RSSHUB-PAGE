<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 14 日消息 根据 SellCell.com 的一项调查，大多数苹果用户期待的功能是 Touch ID 的回归。苹果此前一直在逐步淘汰指纹传感器，转而使用 Face ID 人脸识别。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/350a8c02-906c-4944-8748-9dfb101e711b.jpg@s_2,w_820,h_461" w="961" h="540" title="调查：Touch ID 回归是用户对苹果 iPhone 13 最期待的功能" srcset="https://img.ithome.com/newsuploadfiles/2021/5/350a8c02-906c-4944-8748-9dfb101e711b.jpg 2x" width="961" height="461" referrerpolicy="no-referrer"></p><p>Touch ID 于 2013 年在 iPhone 5s 上首次亮相，随后又在 iPad 和 Mac 上出现。<span class="accentTextColor">苹果在 2017 年的 iPhone X 中开始逐步淘汰 Touch ID</span>，转而采用 Face ID 面部识别功能。用户不需要触摸他们的手机，就能解锁或验证他们的身份。</p><p>然而，取代 Touch ID 的更重要原因，<span class="accentTextColor">也许是为了让苹果能够摆脱 Home 键，并采用全面屏</span>。</p><p>今年，有传言称苹果将带回 Touch ID，<span class="accentTextColor">但将被内置到 iPhone 显示屏中，该功能也不会取代 Face ID</span>。相反，Touch ID 将作为一种替代方案，例如在戴着口罩时解锁 iPhone。</p><p>根据 SellCell.com 的数据，在其采访的 3000 名用户户中，21% 的人将 Touch ID 的回归列为他们最渴望的功能。以下是具体结果：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/1ffc5f7c-5258-4496-98d0-e4c21c2567fa.jpg@s_2,w_820,h_383" w="981" h="458" title="调查：Touch ID 回归是用户对苹果 iPhone 13 最期待的功能" srcset="https://img.ithome.com/newsuploadfiles/2021/5/1ffc5f7c-5258-4496-98d0-e4c21c2567fa.jpg 2x" width="981" height="383" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p>Touch ID ID 的回归 --21%</p></li><li><p>更高刷新率的显示屏（120Hz）--17%</p></li><li><p>更小的刘海 / 无刘海设计 --15%</p></li><li><p>包装盒带充电器 --12%</p></li><li><p>更大的电池 - 11%</p></li><li><p>更好的相机功能 - 8.2%</p></li><li><p>更快的处理器 - 6.8%</p></li><li><p>紧凑的手机 - 5%</p></li><li><p>更好的防水性 - 2.2%</p></li><li><p>可折叠设计 - 1%</p></li><li><p>去掉闪电接口 - 0.8%</p></li></ul><p>IT之家需要提醒的是，这只是一个相对较小的样本量。不过有趣的是，Touch ID 比下一个最令人期待的功能 --ProMotion 显示屏还要多出几个百分点。此外，很少有人对可能出现的折叠 iPhone 感兴奋，但这可能是因为很少有人认为今年会推出折叠 iPhone。</p>
