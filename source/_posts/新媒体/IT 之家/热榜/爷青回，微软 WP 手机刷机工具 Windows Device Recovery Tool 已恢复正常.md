<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 7 日消息 近期有网友称 Windows Phone 手机恢复工具 Windows Device Recovery Tool 已无法检测更新并进行使用，软件提示“应用程序更新检查失败”，疑似服务器已关停。</p><p>但现在情况似乎有了转机，“应用程序更新检查失败”的问题微软不到一周就修复了，Windows Device Recovery Tool 可正常下载固件刷机。</p><p><img src="https://qimg.ithome.com/image/2021/05/07/6375600858179772812818523.png" title="爷青回，微软 WP 手机刷机工具 Windows Device Recovery Tool 已恢复正常" referrerpolicy="no-referrer"></p><p><img src="https://qimg.ithome.com/image/2021/05/07/6375600858384530116460086.png" title="爷青回，微软 WP 手机刷机工具 Windows Device Recovery Tool 已恢复正常" referrerpolicy="no-referrer"></p><p><img src="https://qimg.ithome.com/image/2021/05/07/6375600858575036514143926.png" title="爷青回，微软 WP 手机刷机工具 Windows Device Recovery Tool 已恢复正常" referrerpolicy="no-referrer"></p><p>IT之家了解到，Windows Device Recovery Tool（或简称 WDRT）此前已经存在了很长的时间，其不仅支持包括 Lumia、HTC、惠普等在内的 Windows Phone 手机恢复系统，还支持了微软的 HoloLens 等新设备恢复固件的安装。</p><p><img src="https://img.ithome.com/newsuploadfiles/2020/5/20200510_142756_115.jpg" title="爷青回，微软 WP 手机刷机工具 Windows Device Recovery Tool 已恢复正常" referrerpolicy="no-referrer"></p><p>微软宣布在 2019 年底停止对 Windows 10 Mobile 的支持，但支持工具可能还会继续存活一段时间。</p><p><strong>Windows Device Recovery Tool 下载</strong>，<a href="https://repairavoidance.blob.core.windows.net/packages/WindowsDeviceRecoveryToolInstaller.exe" target="_blank">点此链接</a>。</p>
