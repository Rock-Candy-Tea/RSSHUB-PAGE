<p data-vmark="8268"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 11 日消息，WoA 项目主要创建者 Gustave Monce 发布了一组新的驱动程序，允许在 Surface Duo 初代上运行 Windows 10 和 Windows 11 ARM 系统。<span class="accentTextColor">最新版本引入了无线连接支持，成功启用了 Wi-Fi、蜂窝网络、GPS 和 eSIM。</span></p><p data-vmark="dee2" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/ffa5e28b-ff31-4d39-9ced-bf6b3d6bfbf3.png" w="1200" h="1221" title="刷了 Win11 的微软 Surface Duo 获重要突破：Wi-Fi、GPS、eSIM、蜂窝网络可正常工作" width="1200" height="834" referrerpolicy="no-referrer"></p><p data-vmark="f4b4">版本 2207.32 的更新内容如下：</p><ul class=" list-paddingleft-2"><li><p data-vmark="690e">Wi-Fi 现在可以正常工作（2.4Ghz / 5Ghz）。但是，还不支持 Mac 地址检索。</p></li><li><p data-vmark="dcb0">蜂窝数据现在可以正常工作（LTEA / LTE / HSDPA+/HSDPA/ EDGE / GPRS）。</p></li><li><p data-vmark="2a5b">蜂窝文本现在可以使用。使用聊天应用程序发送和获取消息。</p></li><li><p data-vmark="7ec2">通话、VoLTE、RCS 等功能即将在以后的版本中推出。</p></li><li><p data-vmark="49ca">已可启用 eSIM 支持。</p></li><li><p data-vmark="da6a">GPS 已可正常工作。</p></li></ul><p data-vmark="5a86">尽管该项目已获得多项改进和增强，但仍远未准备好替代原来的 Android 以供日常使用。<span class="accentTextColor">除了缺少呼叫支持之外，用户还会遇到相机无法正常工作、图形渲染问题、自动定位问题、损坏的 USB dongles 和其他错误 Bug。</span></p><p data-vmark="909e">微软官方此前早已经对 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> Mobile 停止支持，不过在 WoA 项目开发者的努力下，仍然会有开发者去为 Lumia 旧机型和新安卓机型刷入新系统。</p><p data-vmark="1f06">由于微软提供了 ARM 设备兼容的 Windows 10 版本，一群独立开发人员在过去几年里一直在维护一个非官方的 Windows on ARM 项目，这就是 WOA。微软发布 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> ARM 版本后，又有新的系统可以尝试了。</p><p data-vmark="afc1">从 2019 年开始，第三方开发者集合的 Lumia 手机刷 Windows 10 ARM（WOA）项目此前在 GitHub 上线，包含了所有有关 LumiaWOA 的项目。</p><p data-vmark="f546" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2021/12/0fc1da5e-53fc-4dfb-9bd9-a61159d7b149.png" w="1440" h="595"
