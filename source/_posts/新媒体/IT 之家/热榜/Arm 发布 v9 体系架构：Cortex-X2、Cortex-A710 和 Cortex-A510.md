<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 25 日消息 又到了每年一度的 Arm 架构更新的时候。在上个月 Arm 发布了最新的基础架构 Neoverse V1 和 Neoverse N2 CPU IP 之后，现在官方终于推出了移动端新架构。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/033e0863-a240-4253-8490-9b8a1c5e5849.jpg" w="678" h="373" title="Arm 发布 v9 体系架构：Cortex-X2、Cortex-A710 和 Cortex-A510" width="678" height="373" referrerpolicy="no-referrer"></p><p>今年，Arm 推出的东西比往年更多，包括面向移动和客户端的三种新一代微架构：旗舰级的 Cortex-X2，A78 继任者 Cortex-A710、全新的 Cortex-A510，延续多年的小核 A55 终于更新了。</p><p>Cortex-X2：相比 X1 性能提高 16%。<br>Cortex-A710：相比 Cortex-A78 提高 30% 的功率效率和 10% 的性能。<br>Cortex-A510：取代了 Cortex-A55，性能提高 35%，能效提高 20%，接近 A73。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/c23a40a4-65cb-4980-88d2-8c1ec213ff91.jpg@s_2,w_820,h_424" w="1400" h="724" title="Arm 发布 v9 体系架构：Cortex-X2、Cortex-A710 和 Cortex-A510" srcset="https://img.ithome.com/newsuploadfiles/2021/5/c23a40a4-65cb-4980-88d2-8c1ec213ff91.jpg 2x" width="1400" height="424" referrerpolicy="no-referrer"></p><p>IT之家了解到，这三款新的 SoC CPU 架构均基于 Armv9 兼容设计，旨在引领更强的架构 / ISA 变革，这在业界是很少见的。但考虑到这只是 PPT 效果，实际效用可能还要再打一层折扣，颇有黄家刀法传承之志。</p><p>除了新的 CPU 核心，我们还看到 DSU-110 采用了新的 L3 和集群设计，Arm 也在 interconnect IP 上进行了大量升级改进，包括采用了高速缓存一致性 CI-700 网状网络和 NI-700 片上网络 IP，但这些跟各位关系不大。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/755599e3-0412-4391-abfa-f1213834e8fd.png" w="678" h="381" title="Arm 发布 v9 体系架构：Cortex-X2、Cortex-A710 和 Cortex-A510" width="678" height="381" referrerpolicy="no-referrer"></p><p>相对往年的设计，这些新架构在迭代改进 IPC 和效率的同时，还以 Armv9 和新扩展（例如 SVE2）的形式构建了一些新功能。</p><p>此外，Arm 新的小核心 Cortex-A510 是一个相对更大的微架构跃进，，因为它代表了 Arm 剑桥 CPU 设计团队的全新 CPU 设计。A510 在改进 IPC 的同时仍保证了高能效比，并且它还保留了其有序微架构的特征。</p><p>新的 CPU 系列标志着 Arm 多年来最大的架构飞跃之一，新的 Arm v9 架构不再多说，新 ISA 的基础功能包括此前版本可选 / 缺失的 Armv8.2 + 支持，并引入了新的 SVE2 SIMD 和矢量扩展。</p><p>值得一提的是，Cortex-X2 超大核和 Cortex-A510 小核都是仅限 AArch64 的微体系结构，它们无法再执行 AArch32 代码，因此无法再运行 64 位兼容设备提供的 32 位应用程序。</p><p>然后，Arm 表示，为了满足中国市场的需求（缺乏谷歌 Play 生态），Cortex-A710 仍然支持 AArch32 编码，中国厂商及其我们的各大应用商店需要更多时间来推进 64 位 App 的普及。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/49592f08-1d59-4f83-99ad-e44525a60b72.png" w="678" h="381" title="Arm 发布 v9 体系架构：Cortex-X2、Cortex-A710 和 Cortex-A510" width="678" height="381" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/3578aa7d-6f82-4af6-ad23-916f0d6b9e2e.png" w="678" h="381" title="Arm 发布 v9 体系架构：Cortex-X2、Cortex-A710 和 Cortex-A510" width="678" height="381" referrerpolicy="no-referrer"></p>
