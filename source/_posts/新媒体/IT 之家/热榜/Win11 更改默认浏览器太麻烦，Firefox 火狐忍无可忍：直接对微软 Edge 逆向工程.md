<p data-vmark="29cd"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 14 日消息 Mozilla 绕过了微软在 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 上设置默认<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>的限制，让 Firefox 火狐浏览器成为默认浏览器变得更加容易。</p><p data-vmark="4bd3">众所周知，微软目前正在主推自家的 Edge 浏览器，并且在系统设置中将 Edge 设为默认浏览器。在 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 中，<span class="accentTextColor">默认浏览器选项直接分成了多个后缀链接格式</span>，用户需要自己手动一个一个修改，而且会遭到微软的一再挽留。</p><p data-vmark="b28d"><img src="https://img.ithome.com/newsuploadfiles/2021/9/1c8b5bcb-75a7-4cbe-bde8-367acdc085a3.png" w="1104" h="625" title="Win11 更改默认浏览器太麻烦，Firefox 火狐忍无可忍：直接对微软 Edge 逆向工程" width="1104" height="464" referrerpolicy="no-referrer"></p><p data-vmark="81a9">面对这一情况，Mozilla 决定反击。据 The Verge 报道，在 Firefox 91 版本及之后，火狐浏览器再也不需要通过系统默认应用设置来成为默认浏览器。根据官方的描述，<span class="accentTextColor">Mozilla 对微软将 Edge 浏览器设置为 Windows 默认浏览器的方式进行了逆向工程</span>，从而将自己的浏览器设置为默认浏览器。</p><p data-vmark="bbf8">▼ 无需在系统设置中更改，浏览器点击按钮直接修改完成</p><p data-vmark="ee62"><img src="https://img.ithome.com/newsuploadfiles/2021/9/0ac1c351-f7e2-4847-b051-4e8e41e59608.gif" w="800" h="450" title="Win11 更改默认浏览器太麻烦，Firefox 火狐忍无可忍：直接对微软 Edge 逆向工程" width="800" height="450" referrerpolicy="no-referrer"></p><p data-vmark="e5c5">IT之家了解到，在给 The Verge 的一份声明中，Mozilla 的一位发言人重申，在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a>/11 上设置默认浏览器如此困难是微软的错：</p><blockquote><p data-vmark="4a5e">所有操作系统都应为默认状态提供官方开发人员支持，以便人们可以轻松地将他们的应用程序设置为默认状态。由于在 Win10/11 上没有发生这种情况，因此当用户选择 Firefox 火狐作为默认浏览器时，Firefox 火狐将采用其他途径，为人们提供类似于 Edge 浏览器的体验。</p></blockquote>
