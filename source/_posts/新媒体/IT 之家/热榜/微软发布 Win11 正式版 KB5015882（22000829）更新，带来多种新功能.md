            <p data-vmark="0a0d"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 22 日消息，微软今天发布了最新的 Windows 11 KB5015882 更新补丁，升级之后系统内部版本号升至 22000.829。实际上，这一版本已在上周的发布预览（RP）频道中推送，两个版本完全相同。</p><p data-vmark="73df">微软强调，你可以在这个新版本中开启专注模式的情况下同时选择接收紧急通知，而且如果你的设备符合条件，就可以在第一次启动 Windows 时选择更新到最新版本。</p><p data-vmark="a4cf">此外，微软还解决了在某些设备上使用播放和暂停键盘按钮时导致文件资源管理器崩溃的问题、解决了当用户使用开始菜单的右键菜单 (Win+X) 和外部显示器时导致文件资源管理器崩溃的问题、解决了将鼠标悬停在任务栏上的搜索图标上时无法关闭空白窗口的问题，以及解决了某些导致故障排除工具无法打开的问题。</p><p data-vmark="bb7b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/dd250b52-d7ed-4728-b58d-060480d2a831.jpg" w="680" h="357" title="微软发布 Win11 正式版 KB5015882（22000.829）更新，带来多种新功能" width="680" height="357" referrerpolicy="no-referrer"></p><p data-vmark="b02b">IT之家了解到，今天的更新包括以下内容，主要是 Bug 修复：</p><ul class=" list-paddingleft-2"><li><p data-vmark="11e6">我们为您带来了在开启“专注助手”时接收紧急通知的功能。</p></li><li><p data-vmark="bf31">当符合条件的新设备第一次登录时，现已支持在 OOBE 阶段直接更新到最新的系统版本。如果您在开机时选择“更新到较新的版本”，则将会在不久后自动开始更新。</p></li><li><p data-vmark="db1f">我们恢复了 Windows Autopilot 部署场景的功能，这些功能会受到硬件重用安全性的影响。此次更新取消了自部署模式 (SDM) 和预配置模式 (PP) 的一次性使用限制。此更新还为已批准的制造商重新启用了用户驱动模式 (UDM) 部署下的任何用户主体名 (UPN) 显示。</p></li><li><p data-vmark="ccd3">我们修复了 UIAutomation () 中导致应用程序停止工作的问题。</p></li><li><p data-vmark="dd12">我们修复了一个问题，该问题会导致 Startup Task API 无法对某些应用程序按预期工作。</p></li><li><p data-vmark="d3ee">我们改进了操作系统升级后“一键重置”的可靠性。</p></li><li><p data-vmark="17b5">我们修复了一个问题，如果删除 EN-US 语言包，租户限制事件日志记录通道将不可访问。</p></li><li><p data-vmark="beca">我们修复了在域控制器上安装 2022 年 5 月 10 日的安全更新后，在某些情况下导致基于证书的计算机帐户认证失败的问题。</p></li><li><p data-vmark="2859">我们修复了一个影响使用 Windows 11 软件开发工具包 (SDK) 构建的 Arm64EC 代码的问题。</p></li><li><p data-vmark="6ead">我们更新了 Remove-Item cmdlet，以正确地与 Microsoft OneDrive 文件夹交互。</p></li><li><p data-vmark="df33">我们修复了导致疑难解答程序无法打开的问题。</p></li><li><p data-vmark="037d">我们修复了导致容器端口映射冲突的问题。</p></li><li><p data-vmark="aa6f">我们修正了导致代码完整性在文件被修改后继续信任文件的问题。</p></li><li><p data-vmark="888c">我们修复了在启用智能安全图功能的情况下启用 Windows Defender 应用程序控制时可能导致 Windows 停止工作的问题。</p></li><li><p data-vmark="9928">我们在设备上部署了“搜索聚焦”功能。为方便起见，我们将很快向 Microsoft 下载中心发布适用于 Windows 11 版本 21H2 的管理模板 (. admx) 的更新版本。</p></li><li><p data-vmark="8ae8">我们修复了当您在某些设备上使用播放和暂停键盘按钮时导致.exe 停止工作的问题。</p></li><li><p data-vmark="d454">我们修复了当您使用“开始”菜单的右键菜单 (Win + X) 并且外接显示器连接到您的设备时导致.exe 停止工作的问题。</p></li><li><p data-vmark="16eb">我们修复了当您将鼠标悬停在任务栏上的搜索图标上时显示无法关闭的空白窗口的问题。</p></li><li><p data-vmark="665b">在 IOPS 的场景中经常会有许多线程争用一个文件，我们减少了资源争用的开销问题。</p></li><li><p data-vmark="11de">我们修复了一个导致 Windows 配置文件服务偶尔失败的问题。当你登录时可能会看到报错:“gpsvc service failed to sign in。拒绝访问”。</p></li></ul>
