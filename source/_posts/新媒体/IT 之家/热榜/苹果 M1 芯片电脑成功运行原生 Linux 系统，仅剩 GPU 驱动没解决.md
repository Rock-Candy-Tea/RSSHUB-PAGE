<p data-vmark="e835"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 7 日消息 苹果自研的 M1 芯片于 2020 年正式发布，从产品发布起，一些开发者便开始了将 Linux 系统适配 M1 芯片的项目。根据外媒 TomsHardware 报道，目前 Linux 系统已经能够在搭载 M1 芯片等 MacBook 电脑上原生运行，<span class="accentTextColor">并且电脑能够满足基本操作</span>。</p><p data-vmark="a290"><img src="https://img.ithome.com/newsuploadfiles/2021/10/a5545a0b-b76c-4c33-94ea-13adcf0f45ac.jpg@s_2,w_820,h_461" w="970" h="545" title="苹果 M1 芯片电脑成功运行原生 Linux 系统，仅剩 GPU 驱动没解决" srcset="https://img.ithome.com/newsuploadfiles/2021/10/a5545a0b-b76c-4c33-94ea-13adcf0f45ac.jpg 2x" width="970" height="461" referrerpolicy="no-referrer"></p><p data-vmark="af3a">这一系统的开发团队名叫 Asahi Linux，目前该项目已经能够保证系统界面正常显示。<span class="accentTextColor">该系统基于 Linux 5.16 内核打造，开发者合并了 PCIe、USB-C PD 等多种驱动</span>，还设法集成了 I2C、电源管理、NVMe 固态硬盘、显示控制芯片等驱动。只不过，M1 芯片集成的 GPU 为苹果自研，没有任何 Linux 驱动可用，因此系统没法调用 GPU 来实现加速渲染等功能。</p><p data-vmark="19b2">Asahi Linux 项目负责人表示，M1 芯片 Mac 电脑目前运行 Linux 的可用性十分高。由于处理器性能非常强大，以至于仅靠 CPU 渲染，速度也比任何具有硬件加速功能的 Rockchip ARM64 机型要快。</p><p data-vmark="6ec2">IT之家了解到，苹果 M1 芯片的集成显卡预计采用 PowerVR 或自研架构，只要苹果不推出 Linux 驱动，第三方很难独立开发。这一项目团队成员表示，一旦适配 M1 芯片的 Linux 系统能够稳定运行，<span class="accentTextColor">团队将发布正式的安装程序</span>，以便部分用户在 M1 芯片电脑上体验原生 Linux。</p>
