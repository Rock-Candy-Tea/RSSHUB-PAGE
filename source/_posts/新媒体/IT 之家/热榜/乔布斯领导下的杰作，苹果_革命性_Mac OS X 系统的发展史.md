<p data-vmark="5d57"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 25 日消息，据 Apple Insider 报道，2001 年 3 月 24 日，苹果公司推出了 Mac OS X 的第一个官方公开版本。它一开始并不成功，但为 Mac 提供了支持，一直坚持了 19 年，直到被 macOS Big Sur 替代。</p><p data-vmark="d569" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/3460fb69-0818-49a7-af2a-c918a7b325ba.jpg@s_2,w_820,h_461" w="1280" h="720" title="乔布斯领导下的杰作，苹果“革命性”Mac OS X 系统的发展史" srcset="https://img.ithome.com/newsuploadfiles/2022/3/3460fb69-0818-49a7-af2a-c918a7b325ba.jpg 2x" width="1280" height="461" referrerpolicy="no-referrer"></p><p data-vmark="cc07">OS X 刚推出时带有救世主色彩，如果你是长期的 Mac 用户，感觉会是一种成就。尽管第一个“Cheetah”版本，即 10.0，缺乏功能，也包括很多错误 Bug，但它确实是苹果操作系统的跳跃。</p><p data-vmark="e666">这不仅仅是因为 1999 年的 Mac OS 9 与 1997 年的 OS 8 相比，几乎没有显著的改进。而且，在整个 90 年代，苹果公司一直在努力开发革命性的新操作系统。</p><p data-vmark="3bf9">从 Pink 和 Taligent，到 Copland 和 Gershwin，苹果公司曾多次试图制作新的操作系统来保持 Mac 的更新。通常情况下，它有多个版本的作品，有些是为了解决短期问题，有些是为了让 Mac 为未来做好准备。</p><p data-vmark="9d5a">当这些长达数年的项目都没有产生一款可供销售的产品时，苹果选择去购买操作系统，并最终将史蒂夫・乔布斯带回了公司。</p><p data-vmark="8b70">收购 NeXT 并因此收购其 NeXTSTEP 操作系统，是为 OS X 铺平了最终的道路。在这一过程中，人们曾短暂地期待它能带来 Rhapsody—— 这是在 NeXTSTEP 中运行经典 Mac 应用程序的操作系统的代号。</p><p data-vmark="1d3d">它的消亡恰好与苹果 CEO 吉尔-阿梅里奥的去职相吻合，随着时间的推移，他被史蒂夫・乔布斯取代。在 OS X Cheetah 发货前一年多，乔布斯概述了从经典 Mac 操作系统“温和迁移”到新操作系统的整个计划。他结束了长期以来不同版本的操作系统在同一时间工作的问题。</p><p data-vmark="4585" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/9b3788a2-9455-492a-aeae-3cb0697a5bad.png" w="640" h="483" title="乔布斯领导下的杰作，苹果“革命性”Mac OS X 系统的发展史" width="640" height="483" referrerpolicy="no-referrer"></p><p data-vmark="8c35">乔布斯强调，OS X 将是一个操作系统。他说：“我们将在苹果公司实行单一的操作系统战略，”“我们不会像其他一些公司那样有双重或三重或四重的操作系统战略。我们将拥有一个操作系统，这对我们非常重要。”</p><p data-vmark="d09c">他总结了苹果公司在 OS X 上的目标，但随后又将其进一步分解。他说，OS X 的唯一真正目标是“制造下一个伟大的个人电脑操作系统”。</p><p data-vmark="8fb7">正如当时在 2000 年 1 月所概述的那样，该计划将有一年的时间来进行推广。OS X 的测试版将在该年春季开始，然后在夏季上市销售。最后，OS X 将于 2001 年 1 月开始预装在新的 Mac 上。</p><p data-vmark="1cae">但事情并非是这样发展的。在乔布斯宣布的时候，Mac OS X 服务器 1.0 已经发货，操作系统的开发者预览版也已经发货。2000 年 9 月，一个公开测试版问世，但 OS X Cheetah 直到 2001 年 3 月 24 日才发布。</p><h3 data-vmark="c79f">Mac OS X Cheetah 终于来了</h3><p data-vmark="3f4f">“Mac OS X 是 Mac 的未来，我们希望它能以其无与伦比的功能和易用性使我们的客户感到高兴，”史蒂夫・乔布斯在苹果的官方公告中说。“公开测试版从 Mac 用户和开发者那里获得了难以置信的反馈和支持，这帮助我们将 Mac OS X 打造成有史以来最先进的操作系统”。</p><p data-vmark="dd79">“[如果] 你是一个 Mac 粉丝，”大卫-波格在《纽约时报》上写道，“担心已经结束。对 Mac OS X 的未来不再有任何怀疑，它是一个稳定、坚固、好看的平台，将在未来 17 年内支持苹果的增强功能”。</p><p data-vmark="ca12">到那时，最初的、经典的 Mac OS 已经运行了 17 年。Mac OS X 将持续更长的时间，达到 19 年，但它如此变化，以至于在大部分时间里它不再被看作是新的操作系统。</p><p data-vmark="9356">尽管他总体上称赞，但大卫-波格和当时所有的评论家一样，期待着即将到来的更新，“可能会更快、更顺畅、更精炼”。</p><p data-vmark="23d5">这被证明是 2001 年 9 月的 Mac OS X Puma。与 OS 9 相比，它的速度仍然很慢，但它正在逐步实现目标。然后在 2002 年 8 月，迎来了 Mac OS X Jaguar。</p><p data-vmark="cca5">一年多以后，轮到了 Mac OS X Panther。除了每年的重大更新外，还有定期的小更新，解决错误和关键问题。</p><h3 data-vmark="9165">OS X 给 Mac 带来了什么</h3><p data-vmark="eb21">史蒂夫-乔布斯为第一个 OS X 制定的目标清单一直是操作系统的核心。他说它必须拥有所有操作系统中“最好的内部管道”，而且需要“杀手级的图形”。</p><p data-vmark="d6ed">“我们需要从一开始就为互联网设计，”他继续说。“我们需要以一种方式来设计它，让大多数总是泡在互联网上的用户得到充分的好处。”</p><p data-vmark="a062">OS X 伸向了互联网，它也成为了乔布斯所说的日益增长的网络生活的“数字中心”。它使 Mac 用户需要的运行多个应用程序的方式成为可能，并且不会让其中任何一个应用程序使得整个系统崩溃。</p><p data-vmark="5d34">当 OS X 问世时，这些功能已经不再是“下一个伟大的个人电脑操作系统”应该具备的功能，而是它们都已具备的功能。新的东西包括 Quartz，OS X 的“杀手级图形”引擎，这是基于 PDF 的。</p><p data-vmark="9481">“那意味着什么？”乔布斯问。“你知道当你去网络上看到 PDF 文件时？嗯，这项技术现在是 Mac OS X 图形的核心。所以你可以立即对 PDF 文件进行成像。所以现在所有的应用程序都能免费获得这个”。</p><p data-vmark="ad63">就是这样，时至今日，每个 Mac 应用程序都可以制作 PDF 文档，而不需要第三方应用程序。</p><p data-vmark="f7b7" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/6413baad-ef90-47dc-a391-54414138d05a.jpg@s_2,w_820,h_547" w="900" h="600" title="乔布斯领导下的杰作，苹果“革命性”Mac OS X 系统的发展史" srcset="https://img.ithome.com/newsuploadfiles/2022/3/6413baad-ef90-47dc-a391-54414138d05a.jpg 2x" width="900" height="547" referrerpolicy="no-referrer"></p><p data-vmark="b4ac" style="text-align: center;">▲ Mac OS X Leopard</p><p data-vmark="4bd7">Cheetah 最明显的图形的其余部分，即 Aqua 界面，随着时间的推移逐渐消失。照片般逼真的图标、色彩鲜艳的按钮和金属拉丝的外观都使经典的 Mac OS 发生了惊人的巨变，但通过逐步的新的更新而平静了下来。</p><p data-vmark="857e">OS X 还引入了 Dock，也许是操作系统中最明显受 NeXT 影响的元素。此外，它还为我们带来了邮件、TextEdit 和所谓的地址簿（现在被称为联系人）。</p><h3 data-vmark="f90c">OS X 最初是如何发展的</h3><p data-vmark="d9d8">最初发布的 OS X 一半是为了展示一项全新的技术，一半是为了追赶经典的 Mac。视觉效果立即不同，但需要时间来匹配功能集--而且较新的操作系统要足够快，才能真正可用。</p><p data-vmark="de81">直到 OS X 10.1 版本，Puma，新的操作系统才可以播放 DVD 或刻录 CD。但是，该版本也是首次捆绑 iTunes，这使 OS X 为音乐繁荣做好了准备，它既从中获利，也帮助创造了音乐繁荣。</p><p data-vmark="dd75">如果你是当时的 Mac 用户，那么一些功能的缺失和其他功能的缓慢让人感到非常沮丧。这是在 2002 年的 OS X Jaguar 之前，后来它的速度明显更快。</p><p data-vmark="9bf3">随着 2001 年 12 月 OS 9 的最终发布，Jaguar 成为苹果唯一的操作系统。OS X 自立门户，从那时起，该操作系统获得了更多的功能，淡化了更多的“杀手级图形”，也加强了安全性。</p><p data-vmark="a01d">OS X 一直专注于为其用户和开发者提高安全性。Jaguar 之后，苹果在 OS X 10.3，即 Panther 中为开发者引入了 Xcode。在此之前，开发者使用任何数量的不同工具来创建他们的软件，而现在他们都有同一款工具。</p><p data-vmark="7151">然后在 2004 年，OS X Tiger 增加了 Spotlight（我们仍然拥有）和仪表盘部件（现在已没有）。也是在这个版本的 OS X 中，苹果完成了从 PowerPC 到 Intel 处理器的过渡。</p><h3 data-vmark="47af">OS X 的细化</h3><p data-vmark="dc13">在最初的 Mac 推出七年后，随着 System 7 的发布，预示着最大的操作系统更新到来。OS X 实际上没有一个非常大的变化，但在其推出六年左右，它确实到达了 OS X Leopard。</p><p data-vmark="830a">它于 2007 年到来，带来了主要的新功能，以及对当时熟悉的中坚力量的修订。一个显而易见的视觉变化是 Dock 被改造成了 3D 架子。</p><p data-vmark="d72d">然后，在你不小心调出这些功能之前，不太明显的是 Expose 和 Spaces。前者帮助注重视觉效果的用户看到他们想切换到哪些应用程序和窗口。后者帮助不整洁的人在多个不同的桌面上保存文件。</p><p data-vmark="6faa">OS X 并不总是有苹果至今仍为人所知的无休止的年度更新，但 Leopard 之后有了新的东西。2009 年的 OS X Snow Leopard 被认为是苹果暂停新功能，转而努力巩固所有现有功能。</p><p data-vmark="4793">这是一种方法，一种短暂的放缓，许多人希望苹果能重复这种做法。该公司确实重复了类似名称的想法，2011 年的 Lion 之后是 2012 年的 Mountain Lion。但是，如果说从 Leopard 到 Snow Leopard 是巧妙地使用名字来显示进步而不是替换，那么到了 Mountain Lion 时，苹果已经没有新名字了。</p><p data-vmark="bce5">从第二年，即 2013 年开始，OS X 将转向以加州的地方命名。Mavericks 开始了这种趋势，然后是 Yosemite、El Capitan、Sierra、High Sierra、Mojave 和 Catalina。</p><p data-vmark="8ce7">一路走来，OS X 的名字被改成了 macOS。然后，苹果的 Big Sur 既延续了 macOS 的名字，也延续了以加州地点命名的惯例。但它也打破了一个惯例。</p><p data-vmark="7d12">经过 19 年的时间，每次更新都是 10 左右的版本 —— 一直到 10.15——Big Sur 是 macOS 11 版本。</p><h3 data-vmark="6d80">OS X 的终结</h3><p data-vmark="1d41">苹果的 macOS Catalina 是近 20 年的开发成果，但它仍然是熟悉的 OS X。就其所有的视觉变化而言，它在 2020 年的替代品 macOS Big Sur 仍然是熟悉风格的 Mac。</p><p data-vmark="23dc">这是视觉上变化最大的 macOS 版本，但同样是融合，所以在使用 Big Sur 几个月后，感觉差异不那么明显了。这就是 Mac，甚至在苹果内部也有关于它是否应该是 11 版本的争论。</p><p data-vmark="72ee">“是时候了，”克雷格-费德里吉在《脱口秀》上说，在 2020 年 6 月的 WWDC 上揭晓后不久。“我们当然对 [改成 11] 的正确时机进行了大量的内部讨论。”</p><p data-vmark="95f5">“我们觉得这个操作系统在支持 Apple Silicon 方面为 Mac 实现下一阶段的一些方式，那是一个重要的时刻，”他继续说。“我们认为，为实现这一转变而发生的一些深层次的基础设施变化也是相当根本的，所以这是一个象征性地转向 11 的正确时刻。我们觉得这是当之无愧的。”</p><p data-vmark="b894">包括在 macOS 名称下的时间，OS X 从 2001 年 3 月 24 日到 2020 年 11 月 12 日为 Mac 服务的日子长达 19 年 7 个月又 19 天。</p><p data-vmark="56ed">相比之下，从 Mac 推出到 2001 年 12 月 5 日最后一次发布 Mac OS 9.2.2 的时间是 17 年 10 个月又 11 天。</p><p data-vmark="1e08">对于今天大多数 Mac 用户来说，OS X 是他们成长的计算平台，它不会被遗忘。请注意，经典的 Mac 操作系统也仍然活在模拟器中。</p>
