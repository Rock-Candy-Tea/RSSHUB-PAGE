<p data-vmark="d27e"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 3 日消息 微软将于 10 月 5 日推送 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 更新，但是正式版有一些严格的硬件要求，包括 2 代锐龙/8 代英特尔 CPU 以及几款 7 代型号，TPM 2.0 支持等。这意味绝大部分 PC 甚至虚拟机都将无法用 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 升级成 Windows 11。</p><p data-vmark="2b0b">除了升级以外，如果您已经安装了 Windows 11 测试版但不满足硬件条件，您可能在正式版到来之后也依然无法更新到最新版本。</p><p data-vmark="3507">幸运的是，外媒提出了一种简单的方法可以绕过 TPM 要求，仅需一个简单的开源脚本即可。</p><p data-vmark="0a57"><img src="https://img.ithome.com/newsuploadfiles/2021/10/f1cfa2f9-e652-496d-9d27-7639a54f1bdd.jpg@s_2,w_820,h_461" w="1440" h="809" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" srcset="https://img.ithome.com/newsuploadfiles/2021/10/f1cfa2f9-e652-496d-9d27-7639a54f1bdd.jpg 2x" width="1440" height="461" referrerpolicy="no-referrer"></p><h2 data-vmark="58ee">如何绕过 Windows 11 的 TPM 要求</h2><p data-vmark="3a89">1.利用 Aveyo 开源工具 Universal MediaCreationTool（<a href="https://gist.github.com/AveYo/c74dc774a8fb81a332b5d65613187b15#file-skip_tpm_check_on_dynamic_update-cmd" target="_blank">Github</a>），不过我们这里仅需要其中跳过 TPM 检查的脚本。</p><p data-vmark="7e67">2.找到“Skip_TPM_Check_on_Dynamic_Update.cmd”</p><p data-vmark="8df8">3. 仅选择并复制该文件的所有代码。</p><p data-vmark="4c43"><img src="https://img.ithome.com/newsuploadfiles/2021/10/4653bf41-a5db-4072-ae1b-0d4d1431b862.png" w="970" h="499" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" width="970" height="422" referrerpolicy="no-referrer"></p><p data-vmark="4be2">4.打开记事本，并把代码粘贴到其中。</p><p data-vmark="6fb0"><img src="https://img.ithome.com/newsuploadfiles/2021/10/62353b63-de16-4f2a-a611-cb4fa195b738.png" w="970" h="445" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" width="970" height="376" referrerpolicy="no-referrer"></p><p data-vmark="61ed">5.将文件以后缀名 .cmd 为格式保存。</p><p data-vmark="7324"><img src="https://img.ithome.com/newsuploadfiles/2021/10/b628ebf9-9064-4468-94f3-50d528e7d0f9.png" w="274" h="213" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" width="274" height="213" referrerpolicy="no-referrer"></p><p data-vmark="c1a0">6.双击并运行该脚本。</p><p data-vmark="9a55">7.当询问是否允许 Windows PowerShell 对您的设备进行更改时，单击是。</p><p data-vmark="6bce">然后程序会打开 PowerShell 并显示“Skip TPM Check on Dynamic Update [Installed]”。IT之家提醒，如果你之后想重新启用 TPM 检查再次运行即可。</p><p data-vmark="149b"><img src="https://img.ithome.com/newsuploadfiles/2021/10/c5069d1c-70fa-4efe-8815-98a24bb8a878.png" w="679" h="382" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" width="679" height="382" referrerpolicy="no-referrer"></p><p data-vmark="ab54">完成后，如果您现处于 Insider Program 中，Windows 将自动开始下载 Windows 11 并安装。如果您已经安装了 Windows 11 但无法接收更新，则系统会下载更新。</p><p data-vmark="a53f"><img src="https://img.ithome.com/newsuploadfiles/2021/10/8dd2598d-8fa2-4793-a7de-f1b01eb20176.png" w="809" h="636" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" width="809" height="636" referrerpolicy="no-referrer"></p><p data-vmark="799e">此外，如果您不在测试通道中，您也可以通过下载 Windows 11 的 ISO 文件、将其写入启动盘并从 Windows 10 中运行 setup.exe 文件来升级到新版操作系统。</p><h2 id="how-to-bypass-tpm-when-clean-installing-windows-11" data-vmark="2556">全新安装 Windows 11 时如何绕过 TPM</h2><p data-vmark="dab1">如果您计划进行全新安装而不是升级到 Windows 11 而且同意需要绕过 TPM，您需要将安装盘或 ISO 文件也禁用 TPM 检测。</p><p data-vmark="b54e">幸运的是，刚才我们提到的 Universal MediaCreationTool 工具也有这一功能，它可以自动生成不检查 TPM 状态的 Windows 11 ISO 或启动盘。当然，相信很多小伙伴之前已经用其他方法，例如修改 ISO 镜像来达到同样的目的。</p><p data-vmark="dee8">1.打开 MediaCreationTool 的 <a href="https://gist.github.com/AveYo/c74dc774a8fb81a332b5d65613187b15#file-mediacreationtool-zip" target="_blank">Github </a>页面并下载。</p><p data-vmark="32cc">2.从 zip 文件中提取 MediaCreationTool.bat 并将其保存到某个位置。</p><p data-vmark="a0ee">3.运行 MediaCreationTool.bat。如果 Windows 提醒该文件具有潜在危险，则还需要点击“仍然运行”。</p><p data-vmark="e392">4.选择 11 作为 MCT 版本。</p><p data-vmark="0823"><img src="https://img.ithome.com/newsuploadfiles/2021/10/5ee3e6b7-d675-493e-96b5-4781c6745f5d.jpg@s_2,w_820,h_562" w="970" h="665" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" srcset="https://img.ithome.com/newsuploadfiles/2021/10/5ee3e6b7-d675-493e-96b5-4781c6745f5d.jpg 2x" width="970" height="562" referrerpolicy="no-referrer"></p><p data-vmark="6d74">5.选择“创建 USB”或“创建 ISO”。如果您在虚拟机上安装，则需要 ISO 文件。</p><p data-vmark="f931"><img src="https://img.ithome.com/newsuploadfiles/2021/10/633a5fe6-d437-4b36-ad45-ab499f66dea4.jpg" w="509" h="340" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" width="509" height="340" referrerpolicy="no-referrer"></p><p data-vmark="1f0f">6.当 Windows 询问是否允许 Power Shell 打开应用程序时，选择“是”。Windows MCT 工具将自行启动。当然，这里的标题栏依然是显示“Windows 10”，但只是需要借用 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> MCT 工具罢了。</p><p data-vmark="406b">7.出现提示时请自行选择 USB 驱动器或 ISO 文件。</p><p data-vmark="aba1"><img src="https://img.ithome.com/newsuploadfiles/2021/10/79d1cda3-bb16-4e89-bf88-39c6269394c2.jpg@s_2,w_820,h_710" w="970" h="840" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" srcset="https://img.ithome.com/newsuploadfiles/2021/10/79d1cda3-bb16-4e89-bf88-39c6269394c2.jpg 2x" width="970" height="710" referrerpolicy="no-referrer"></p><p data-vmark="ddd1">8.选择所需的 USB 驱动器或 ISO 文件位置。</p><p data-vmark="f65d"><img src="https://img.ithome.com/newsuploadfiles/2021/10/23d03ad8-fc48-4c58-af33-93f58646c27f.jpg@s_2,w_820,h_710" w="970" h="840" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" srcset="https://img.ithome.com/newsuploadfiles/2021/10/23d03ad8-fc48-4c58-af33-93f58646c27f.jpg 2x" width="970" height="710" referrerpolicy="no-referrer"></p><p data-vmark="35d4">9.MCT 工具需要几分钟时间从微软服务器下载 Windows 11 镜像并创建 USB 启动盘或 ISO 文件。当然，该工具这里依然显示的是“正在下载 Windows 10”，但没必要因为这个怀疑自己。</p><p data-vmark="c668"><img src="https://img.ithome.com/newsuploadfiles/2021/10/2dc3fb64-6909-498a-be34-c244f11179f7.jpg@s_2,w_820,h_710" w="970" h="840" title="微软 Win11 正式版即将到来，一个小工具帮你绕过 TPM 升级要求" srcset="https://img.ithome.com/newsuploadfiles/2021/10/2dc3fb64-6909-498a-be34-c244f11179f7.jpg 2x" width="970" height="710" referrerpolicy="no-referrer"></p><p data-vmark="f17d">10.从启动盘或 ISO 文件启动安装程序即可。现在即使您的 PC 或 VM 不支持 TPM，你应该也可以顺利安装 Windows 11。</p>
