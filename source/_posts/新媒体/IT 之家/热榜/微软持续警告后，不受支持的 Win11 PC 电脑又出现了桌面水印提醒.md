            <p data-vmark="c085"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 22 日消息，据 Windows Latest 报道，去年，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 推出时公布的配置要求引起了一些严重的争论，微软表示，这些限制要求是出于安全考虑而制定的。微软的决定激怒了许多期待 Windows 11 的用户，他们还探索发现了一些绕过限制的变通方法。</p><p data-vmark="b6d4" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/0c1362e8-bcf7-4412-b774-db3a5d04598d.jpg" w="696" h="392" title="微软持续警告后，不受支持的 Win11 PC 电脑又出现了桌面水印提醒" width="696" height="392" referrerpolicy="no-referrer"></p><p data-vmark="8856">即使你的设备不符合要求，也有可能将 Windows 11 操作系统强制安装到设备上。事实上，微软有一个官方注册表破解，可以绕过安装助手中的 TPM 和 CPU 检查，在任何能够运行 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 的硬件上安装 Windows 11。</p><p data-vmark="e863">在同一份文件中，微软警告说，未能满足要求的设备将不再保证能收到更新。微软还总结说，不“建议”强行安装 Windows 11，因为这些配置要求能确保更好的性能和安全。</p><p data-vmark="fcc0" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/9c0212b5-f30b-4a3c-a3f7-3ac2f630f8af.jpg@s_2,w_820,h_444" w="1397" h="757" title="微软持续警告后，不受支持的 Win11 PC 电脑又出现了桌面水印提醒" srcset="https://img.ithome.com/newsuploadfiles/2022/2/9c0212b5-f30b-4a3c-a3f7-3ac2f630f8af.jpg 2x" width="1397" height="444" referrerpolicy="no-referrer"></p><p data-vmark="c99a">如果你已经在不支持的设备上安装了 Windows 11，你可能很快会注意到桌面上有新的水印“System requirements not met”（未达到系统要求）。这个水印出现在任务栏时钟上方，类似于“Windows 未激活”的错误，但它不会影响应用程序、窗口或网络浏览器。</p><p data-vmark="b711" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/a4029ae2-8c72-41de-a640-db245f03aaf7.jpg" w="402" h="152" title="微软持续警告后，不受支持的 Win11 PC 电脑又出现了桌面水印提醒" width="402" height="152" referrerpolicy="no-referrer"></p><p data-vmark="b0b4">桌面上的水印只是写着“未达到系统要求”，但可能会刺激一些用户，微软之前警告过用户可能遭遇到“损害”。</p><p data-vmark="649d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/c61ef034-10b9-4685-99b8-088660e204d3.jpg@s_2,w_820,h_440" w="1048" h="562" title="微软持续警告后，不受支持的 Win11 PC 电脑又出现了桌面水印提醒" srcset="https://img.ithome.com/newsuploadfiles/2022/2/c61ef034-10b9-4685-99b8-088660e204d3.jpg 2x" width="1048" height="440" referrerpolicy="no-referrer"></p><p data-vmark="2b15">正如你在截图中所看到的，该警告也将出现在设置应用程序内，它将提醒用户，他们的设备没有得到官方支持。</p><p data-vmark="dd6d">目前，你会在两个地方发现这些警告 —— 桌面和设置应用程序。但这些通知可以被安全地忽略，它们不会影响整个用户体验，因为水印只限于桌面或设置页面。</p><p data-vmark="c0ad">没有迹象表明，微软正计划在打开的应用程序上面显示“不支持的设备”警告。</p><p data-vmark="4e67">值得注意的是，微软正在 Windows 11 Build 22557 中对新的水印进行 A / B 测试，它可能会随着太阳谷 2 的更新出现在生产构建版本中。</p><h3 data-vmark="d3fe">不受支持 Windows 11 电脑的未来</h3><p data-vmark="c80c">微软还实施了一个弹出窗口，在安装操作系统时出现，并告诉用户在未能满足最低规格的 PC 上安装 Windows 11 的缺点。</p><p data-vmark="874a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/4798ed07-9b63-4af4-b10e-57c9dda6108c.jpg@s_2,w_820,h_461" w="970" h="545" title="微软持续警告后，不受支持的 Win11 PC 电脑又出现了桌面水印提醒" srcset="https://img.ithome.com/newsuploadfiles/2022/2/4798ed07-9b63-4af4-b10e-57c9dda6108c.jpg 2x" width="970" height="461" referrerpolicy="no-referrer"></p><p data-vmark="8783">该弹出窗口明确表示，如果你在不支持的电脑上继续安装 Windows 11，可能会出现兼容性问题、“可能的损害”和其他问题。</p><p data-vmark="d159">此外，微软已经证实，不会为不支持的 Windows 11 设备提供任何定期，甚至是关键的安全更新保证。</p>
