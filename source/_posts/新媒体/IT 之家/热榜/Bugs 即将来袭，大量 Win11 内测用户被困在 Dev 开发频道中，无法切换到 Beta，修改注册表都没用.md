<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 24 日消息 外媒 HTNovo 报道，微软最近警告 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Insiders 内测用户，微软将很快开始面向 Dev 开发频道推送不太稳定和更具实验性的预览版，微软将开始测试不打算在今年晚些时候全面发布的功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/5c2623be-aafe-4696-9818-66eba60a2d6d.png" w="1440" h="692" title="Bugs 即将来袭，大量 Win11 内测用户被困在 Dev 开发频道中，无法切换到 Beta，修改注册表都没用" width="1440" height="394" referrerpolicy="no-referrer"></p><p>微软建议那些愿意坚持使用目前非常稳定的版本的用户转到 Windows 11 Beta 频道，但报道称，对许多用户来说这似乎是不可能的。</p><p>许多内测人士报告说，他们无法从 Dev 开发频道切换到 Beta 测试频道，因为这些选项在 Windows Update 设置中是灰色的。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/4e0bf0e5-a6bc-4593-9f05-7314e8d030c5.png" w="920" h="730" title="Bugs 即将来袭，大量 Win11 内测用户被困在 Dev 开发频道中，无法切换到 Beta，修改注册表都没用" width="920" height="651" referrerpolicy="no-referrer"></p><p>此外，旧的注册表技巧似乎也无法起效了，虽然你可以从 Dev 开发频道切换出来，但你最终会进入 Release Preview 发布预览版而不是 Beta 测试版频道。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/510649ab-715d-4a92-a1ba-0b909fae5947.png" w="924" h="472" title="Bugs 即将来袭，大量 Win11 内测用户被困在 Dev 开发频道中，无法切换到 Beta，修改注册表都没用" width="924" height="419" referrerpolicy="no-referrer"></p><p>IT之家获悉，目前似乎还没有彻底解决这个问题的办法，进行切换的窗口正面临着短时间内关闭。</p>
