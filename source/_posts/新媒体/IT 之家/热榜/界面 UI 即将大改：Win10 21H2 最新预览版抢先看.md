<p>近日微软面向所有 Dev 频道用户，发布了最新 Windows 10 Build 21337 更新。相比之前的小打小闹，这一次的变化可谓亮点多多，除了先前泄露过的部分改进外，还有一些则是首次出现在 Win10 中。<span class="accentTextColor">这个预览版隶属于 21H2，这也是 Win10 将会进行界面 UI 全新升级的版本</span>。接下来，就让我们一起来看一看。</p><h2>1. 图标更新</h2><p>设置面板启用了新图标，除了手机、网络、游戏、更新等几个固定版块外，其余图标均做了一些调整。此外，部分应用程序也更换了新 ICON，这里就包括大家耳熟能详的记事本、电影和电视、讲述人、Windows 安全中心等，基本覆盖了日常的绝大多数应用。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063303_9502.jpg" w="700" h="308" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="700" height="308" referrerpolicy="no-referrer"></p><p>设置面板中的新图标</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063304_6460.jpg" w="309" h="222" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="309" height="222" referrerpolicy="no-referrer"></p><p>新旧版记事本对比</p><h2>2. 资源管理器</h2><p>资源管理器的变化主要体现在两点：一是项目间距被拉大，视觉感更舒服，更适合高分屏及触摸屏使用。另一点就是当开启 “使用复选框以选择项”功能后，默认会在所有内容前显示复选框，而不像现版这样，仅当鼠标划过才会显示，视觉逻辑上更清晰。同时小编注意到，还有一个比较有意思的地方，即微软在设置中预留了一项 “恢复选项”。如果实在对新外观不感冒，也可以点击它随时恢复回老样子。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063304_3418.jpg" w="700" h="349" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="700" height="349" referrerpolicy="no-referrer"></p><p>新旧版资源管理器对比</p><h2>3. 虚拟桌面</h2><p>虚拟桌面在这一版中变化较多，其中最明显的，就是用户可以为不同桌面设置不同的壁纸了。相比现版只能通过桌面名区分，改壁纸大法无疑能够大大提高桌面的查找效率。位置调整是新桌面的另一大看点，通过快捷菜单或鼠标拖拽，我们可以快速修改桌面左右顺序，方便实现某些特殊需要。除此之外，新系统还增加了几组快捷键，包括：Alt+Shift+→（向右移动）、Alt+Shift+←（向左移动）、Del（删除当前桌面）。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063304_6146.jpg" w="510" h="249" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="510" height="249" referrerpolicy="no-referrer"></p><p>虚拟桌面菜单增加的新功能</p><h2>4. 自动 HDR</h2><p>自动 HDR 是新版中一项人性化改进，通俗来说，就是当你的设备支持 HDR，且所玩游戏也在微软的支持列表中时（微软已提供 1000 余款支持 DirectX11 和 DirectX12 的游戏），系统便会在游戏时自动开启 HDR。这项功能位于 “设置”→“应用”→“视频播放”，当你的设备（主要是显示器）支持 HDR 功能后，就能在下方看到 “Auto HDR”这个选项，勾选即代表开启。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063304_7402.jpg" w="584" h="800" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="584" height="800" referrerpolicy="no-referrer"></p><p>新增加的自动 HDR 功能</p><h2>5. 内置应用</h2><p>除了先前被移除的 “3D 对象”文件夹外，这一版还对其他内置应用进行了调整，其中最有代表性的就是 “记事本”。新版不再集成记事本，转而通过应用商店下载与更新。Microsoft To Do（微软待办应用）、Windows Terminal（Windows 终端）、Power Automate Desktop，则成为新的默认应用内置于系统中。</p><p>其中 “Power Automate Desktop”大家可能比较陌生，这是一款能够模仿用户键鼠操作（如点击、输入等）的应用，类似于之前的计划任务。可以让用户以较低的门槛，实现个别应用程序的自动化运行。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063304_4360.jpg" w="700" h="469" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="700" height="469" referrerpolicy="no-referrer"></p><p>新内置应用——Power Automate Desktop</p><h2>6. 桌面壁纸聚焦</h2><p>自动换壁纸终于在这一版中出现，该功能类似于现版的 “锁屏聚焦”，能够每天自动下载 Bing 首图作为桌面壁纸。生成的壁纸会在右下角显示一个 “Learn about this picture”链接，点击即可弹出壁纸名称及版权信息。不过这项功能目前还属于 A/B 测试阶段，仅有部分用户能够体验到。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063304_1317.jpg" w="500" h="361" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="500" height="361" referrerpolicy="no-referrer"></p><p>类似于 “锁屏聚焦”的 “桌面壁纸聚焦”</p><h2>7. 手写支持自定义字体</h2><p>手写功能新增自定义字体，当使用手写输入时，可以自行选择喜欢的字体进行显示。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063304_5803.jpg" w="684" h="441" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="684" height="441" referrerpolicy="no-referrer"></p><p>手写支持自定义字体</p><h2>8. “设置”面板重新编组</h2><p>新版 “设置”在排列上更集中，除了部分新功能外，很多老功能也被重新编组。比如 “输入”和 “任务栏”，除了功能上更加丰富以外，条理的设计也更明晰。之前很多分散于多处的设置，现在都被重新编排到一起，更加方便用户一站式操作。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063305_1288.jpg" w="700" h="894" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="700" height="894" referrerpolicy="no-referrer"></p><p>新增加与合并的部分设置</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063305_4988.jpg" w="695" h="918" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="695" height="918" referrerpolicy="no-referrer"></p><p>新增加与合并的部分设置</p><h2>9. 剪贴板</h2><p>剪贴板（Win+V）的变化主要体现在：表情、GIF、特殊符号、字符画几个方面。特别是特殊符号，日常中实用性很高。此外新版终于可以随意拖拽位置了，这也是旧版中一个被诟病比较多的问题。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063305_9473.jpg" w="420" h="454" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="420" height="454" referrerpolicy="no-referrer"></p><p>新版剪贴板</p><h2>10. 字幕设置</h2><p>“字幕模板”是新版提供的一项字幕新功能，和旧版完全依赖手工设置不同，新版提供了 “黑底白字”、“大型文字”、“蓝底黄字”三组默认效果。用户只要选择或者在此基础上简单修改，即可得到想要的效果。而且它的自定义页面也要比以前更有条理，分成文本、背景、窗口三大标签。预览图尺寸大了很多，以方便用户更直观地了解到最终效果。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063305_6431.jpg" w="586" h="800" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="586" height="800" referrerpolicy="no-referrer"></p><p>新旧版字幕界面对比</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327063305_7687.jpg" w="477" h="804" title="界面 UI 即将大改：Win10 21H2 最新预览版抢先看" width="477" height="804" referrerpolicy="no-referrer"></p><p>自定义设置更有条理</p><h2>后记</h2><p>除了上面这些大的变化外，Windows 10 Build 21337 也增加了很多小调整。比方说蓝牙配对的浮窗显示时间就要比现版更长，以方便用户执行配对操作。全新的 IME 候选窗口，从 UI 层面开始与整个系统匹配。触控板比起现版有着更详细选项…… 总体来说，这一版的变化还是比较丰富的，且实用性普遍不错。后期逐步稳定后，将是一个值得期待的版本。</p>
