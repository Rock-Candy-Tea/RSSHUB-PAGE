<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 20 日消息 贪玩蓝月官方宣布，由于游戏中出现了对 ID 含“吴亦凡”的玩家进行围攻和追杀，以及被踢出公会的情况。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/23ad1d05-00e8-451c-af7e-5cbc52cf4cfb.jpg" w="503" h="390" title="贪玩蓝月：所有 ID 含“吴亦凡”的玩家可免费改名" width="503" height="390" referrerpolicy="no-referrer"></p><p>官方表示，为保证各位玩家的游戏体验，将为所有 ID 含“吴亦凡”的玩家提供免费改名机会一次，具体操作方式请留意游戏内通知。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/a7211e3f-736c-47be-97f2-61a7e4a53c73.png" w="583" h="339" title="贪玩蓝月：所有 ID 含“吴亦凡”的玩家可免费改名" width="583" height="339" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/0028af0e-7da6-408e-a7c5-a8f57fdc8b62.jpg@s_2,w_820,h_1093" w="1080" h="1440" title="贪玩蓝月：所有 ID 含“吴亦凡”的玩家可免费改名" srcset="https://img.ithome.com/newsuploadfiles/2021/7/0028af0e-7da6-408e-a7c5-a8f57fdc8b62.jpg 2x" width="1080" height="1093" referrerpolicy="no-referrer"></p><p>IT之家获悉，今天下午，游戏厂商育碧（UBISOFT）发微博称，决定从《舞力全开》国行游戏本体和舞力无限曲库中下架吴亦凡先生的《大碗宽面》及相关版本，并将为玩家提供其它中文歌曲作为补偿。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/72be8d1f-8ad5-4972-9660-e7337c4b08e5.png" w="714" h="213" title="贪玩蓝月：所有 ID 含“吴亦凡”的玩家可免费改名" width="714" height="213" referrerpolicy="no-referrer"></p><p>近期，艺人吴亦凡被都美竹曝出私生活事件。相关内容在周一爆发，多条话题占据微博热搜榜。</p><p>本周一开始，多家与其合作品牌发布声明表示，已终止或结束与吴亦凡方面的合作。在科技行业，云听 App、华帝、王者荣耀、腾讯视频等平台已先后发布解约声明。</p>
