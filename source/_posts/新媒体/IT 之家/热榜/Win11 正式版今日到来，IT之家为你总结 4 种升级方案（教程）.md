<p data-vmark="231c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 5 日消息 今日早间，微软推出了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 正式版，支持满足硬件条件的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 用户升级。考虑到微软此次采用分阶段推出的政策，目前仍有相当多的用户未收到推送升级。</p><p data-vmark="f805">念及于此，有需要升级 Windows 11 并且满足硬件条件的小伙伴可以参考以下 4 个方法来尝试！值得一提的是，如果你不满足最低硬件条件例如 CPU 哪怕你偷渡安装了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 也可能无法收到后续更新。四种升级方案如下：</p><ul class=" list-paddingleft-2"><li><p data-vmark="7672">通过更新推送升级 Windows 11</p></li><li><p data-vmark="84e2">通过 Windows 11 安装助手升级</p></li><li><p data-vmark="2099">通过下载 Windows 11 镜像升级</p></li><li><p data-vmark="a545">通过创建 Windows 11 安装媒体工具制作 Windows 11 启动盘全新安装 Windows11</p></li></ul><p data-vmark="837f">Windows 11 <a href="https://docs.microsoft.com/zh-cn/windows-hardware/design/minimum/supported/windows-11-supported-amd-processors">支持的 AMD 处理器</a>、<a href="https://docs.microsoft.com/zh-cn/windows-hardware/design/minimum/supported/windows-11-supported-intel-processors">支持的 Intel 处理器</a>、<a href="https://docs.microsoft.com/zh-cn/windows-hardware/design/minimum/supported/windows-11-supported-qualcomm-processors">支持的高通处理器</a>，下载<a href="https://aka.ms/GetPCHealthCheckApp" target="_blank">电脑健康状况检查应用</a>检查硬件。</p><p data-vmark="cea6"><img src="https://img.ithome.com/newsuploadfiles/2021/10/aeee0d08-80fa-4c04-8e4e-dc6ad2812093.png" w="1003" h="664" title="Win11 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 设备推送 Windows 11 升级更新包。</p><p data-vmark="ea93">所以，如果你希望通过 Windows 官方更新程序来升级，却暂时没有收到更新提示也没有关系，只要你确保当前所用的是 Win10 最新的 21H1 版本，然后耐心等待即可，微软最晚会在明年准备好统一推送 Win11 更新。</p><p data-vmark="3d22">如果你通过 Windows Update 收到了系统更新推送，那么恭喜你可以尝试安装，但这里也提醒大家最好提前备份重要数据，以防出现更新导致数据丢失或损坏的情况。</p><p data-vmark="c5d1"><img
