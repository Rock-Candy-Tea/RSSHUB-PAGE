<p data-vmark="231c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 5 日消息 今日早间，微软推出了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 正式版，带来大量新特性，支持满足硬件条件的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 用户现可升级。</p><h2 data-vmark="7ec4">Win11 升级的必要性、新系统的亮点</h2><ul class=" list-paddingleft-2"><li><p data-vmark="95d7">即将到来的 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 应用支持，预计会在几个月后上线</p></li><li><p data-vmark="95d7">全新的 UI 和声音设计，系统更加现代化、整洁且美观，颇有种令人耳目一新的感触。</p></li><li><p data-vmark="228e">在“开始”菜单及其中，Win11 凭借云成功同步了所有设备上你查看过的文档。</p></li><li><p data-vmark="1187">贴靠布局、贴靠群组和虚拟桌面功能可以同时处理多个任务，从而优化屏幕空间。</p></li><li><p data-vmark="dcf9">带来了支持 AI 的个性化提示功能小部件，快速查看各种你关心的信息。</p></li><li><p data-vmark="04fc">凭借 DirectStorage API 和 Auto HDR 等技术支持，Win11 能够充分发挥系统硬件潜力，提供更好的 Windows PC 游戏体验。</p></li><li><p data-vmark="e98d">全新设计的 Microsoft Store，让你更简便地搜索和发现想要的应用、游戏、节目和电影。</p></li><li><p data-vmark="1896">Windows 11 专门为残障人士进一步改进了辅助（无障碍）功能。</p></li><li><p data-vmark="65d5">新系统对开发者更为友好，包括提供新的开发工具，改进本地和网页应用程序的开发流程，让开发者更快捷地将自己的应用界面适配新系统等。</p></li><li><p data-vmark="4aae">对 PC 速度、效率进行了优化，并改进了触控屏、触控笔和语音输入的体验。</p></li><li><p data-vmark="760d">Windows 11 从设计源头确保了安全性，并拥有既简易又熟悉的 IT 部署和管理，带来更加安全且舒适的办公方式新体验。</p></li></ul><h2 data-vmark="9e2a">Win11 是否支持安卓 App，何时支持</h2><p data-vmark="231c">目前 Android 应用支持暂未实装，但相关的工具已经出现在了 Microsoft Store 中。该功能会在随后的几个月中，面向 Windows 预览体验成员推出预览版本。届时，微软可通过 OTA 提供<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>应用支持，用户也可通过系统商店下载并使用各种移动应用。</p><h2 data-vmark="332f">Win11 支持哪些设备升级，不支持升级的设备能否强行安装</h2><p data-vmark="231c">微软对于 Windows 11 的要求主要以处理器为主，运行条件是 1GHz 或更快的 64 位处理器（双核或多核）或系统单芯片 SoC，但实际上仅有名单内的部分机型可升级。</p><p data-vmark="231c">虽然其他机型仍可通过某些方式进行安装，但后续无法收到 Win11 安全更新等内容。也正因如此，IT之家并不推荐低配机型升级到 Windows 11，目前大家使用的 Windows 10 支持将一直持续到 2025 年 10 月 14 日，相比偷渡 Win11 更加安全。</p><p data-vmark="837f">这里也给出微软官方的支持清单：Windows 11 <a href="https://docs.microsoft.com/zh-cn/windows-hardware/design/minimum/supported/windows-11-supported-amd-processors">支持的 AMD 处理器</a>、<a href="https://docs.microsoft.com/zh-cn/windows-hardware/design/minimum/supported/windows-11-supported-intel-processors">支持的 Intel 处理器</a>、<a href="https://docs.microsoft.com/zh-cn/windows-hardware/design/minimum/supported/windows-11-supported-qualcomm-processors">支持的高通处理器</a>。各位小伙伴还可通过下载<a href="https://aka.ms/GetPCHealthCheckApp" target="_blank">电脑健康状况检查应用</a>检查自己的硬件是否受支持。</p><p data-vmark="cea6"><img src="https://img.ithome.com/newsuploadfiles/2021/10/aeee0d08-80fa-4c04-8e4e-dc6ad2812093.png" w="1003" h="664" title="Win11 正式版今日到来，IT之家为你总结 4 种升级方案（教程）" width="1003" height="543" referrerpolicy="no-referrer"></p><p data-vmark="cea6">目前来看，处理器受限是无法突破 Win11 限制的，除非后续微软放开硬件标准线。如果你通过微软电脑健康状况检查应用发现自己是 TPM 2.0 受限，可查看 PC 主板是否带有 fTPM 支持，如果有将安全启动选项设为“开启 (enable)”即可，但如果没有可能需要各位小伙伴自另行购买相关产品。</p><p data-vmark="cea6">除此之外，UEFI 启动、4GB 内存、64GB 硬盘相对升级难度并不高，而显卡和显示器要求并不设限，但仍推荐使用支持 DX12、WDDM2.0 驱动的显卡以及 9 英寸 720p 分辨率 8bit 通道以上的显示器。</p><p data-vmark="cea6">其他方面，Windows 11 家庭版用户在初次开机进行设置时，须连接到网络，否则无法完成 OOBE 过程。微软近日还为 Win10 2004、20H2、21H1 和 21H2 家庭版和专业版推送更新，若你在完成 OOBE 前选择了升级 Win11 则会自行启动升级。</p><h2 data-vmark="7af7">如何激活 Win11</h2><p data-vmark="cea6">我们这里推荐各位小伙伴为获得 Windows 许可证的 PC 升级，完成升级之后依然可以获得 Win11 许可证。而跨版本升级的小伙伴可能需要另行购买产品密钥。</p><h2 data-vmark="2269">关于 Dev、Beta 预览版和正式版的关系</h2><p data-vmark="cea6">简单来说，Windows Insider 项目共有快速、慢速和 Release Preview 三个通道，前两者即为 Dev 开发者预览版和 Beta 测试版，后者则与正式发行版几乎没有区别，除更新频率依次降低外稳定性也相应地依次增强，用户可根据自己的偏好进行选择加入。</p><p data-vmark="cea6">除 Dev、Beta、Release Preview 三个预览版通道外，就是大家常说的半年通道 (SAC) 和长期维护通道 (LTSC)，目前 Windows 已经从半年更改为年更，所以大家下一次再看到 Win11 正式版就是明年秋季的 22H2 了；长期支持版每二至三年会发布一次新版本，而 Win11 的 LTSC 可能还需要等上几年。</p><p data-vmark="ab27"><img src="https://img.ithome.com/newsuploadfiles/2021/10/717a7872-3843-42a7-bd51-1568f44b2f8c.png" w="769" h="218" title="Win11 正式版今日到来，IT之家为你总结 4 种升级方案（教程）" width="769" height="218" referrerpolicy="no-referrer"></p><h2 data-vmark="dd2c">如何升级到 Win11</h2><p data-vmark="6492">Windows 11 升级并不难，难的是满足硬件条件。从微软官方数据来看，全球似乎只有 13 亿 Win10 用户可以升级到 Windows 11。</p><p data-vmark="ff0a">微软此次发布采用分阶段推出的政策，因此目前仍有相当多的用户未收到推送升级。</p><p data-vmark="f805">念及于此，有需要升级 Windows 11 并且满足硬件条件的小伙伴可以参考以下 4 个方法来尝试！值得一提的是，如果你不满足最低硬件条件例如 CPU 哪怕你偷渡安装了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 也可能无法收到后续更新。四种升级方案如下：</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="7672">通过更新推送升级 Windows 11</p></li><li><p data-vmark="84e2">通过 Windows 11 安装助手升级</p></li><li><p data-vmark="2099">通过下载 Windows 11 镜像升级</p></li><li><p data-vmark="a545">通过创建 Windows 11 安装媒体工具制作 Windows 11 启动盘全新安装 Windows11</p></li></ul><h2
