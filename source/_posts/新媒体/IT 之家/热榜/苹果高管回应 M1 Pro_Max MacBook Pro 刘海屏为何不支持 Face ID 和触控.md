<p data-vmark="2b7e"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 30 日消息，据 9to5 Mac 、MacRumors 报道，虽然新的 14 英寸和 16 英寸 MacBook Pro 的更新已经广泛受到评论家和消费者的欢迎，但仍有关于缺少触控屏和 Face ID 等功能的问题。在接受《华尔街日报》采访时，苹果公司高管为该公司抵制在 Mac 上添加这些功能的做法提供了“借口理由”。</p><p data-vmark="6b72"><img src="https://img.ithome.com/newsuploadfiles/2021/10/f2319c76-a54c-4ac8-bf33-1856990f31ce.jpg@s_2,w_820,h_533" w="980" h="637" title="苹果高管回应 M1 Pro/Max MacBook Pro 刘海屏为何不支持 Face ID 和触控" srcset="https://img.ithome.com/newsuploadfiles/2021/10/f2319c76-a54c-4ac8-bf33-1856990f31ce.jpg 2x" width="980" height="533" referrerpolicy="no-referrer"></p><p data-vmark="a3aa">虽然 Chromebook 和 Windows PC 制造商已经在笔记本电脑中广泛采用触控屏，但苹果没有。相反，苹果公司正专注于 Mac 的“间接输入”，并将触控屏技术留给 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a>。有一种观点认为，如果 Mac 获得触控屏输入功能，就会蚕食 iPad 的销售。苹果公司负责硬件工程的高级副总裁 John Ternus 在采访中说。</p><blockquote><p data-vmark="3f6e">“我们在 iPad 上制造了世界上最好的触控屏电脑。它完全为此进行了优化。而 Mac 则完全为间接输入而优化。我们还没有真正感到有理由改变这一点。”</p></blockquote><p data-vmark="5e68"><img src="https://img.ithome.com/newsuploadfiles/2021/10/b52bfb05-da93-4b9d-8f34-40bb406a6420.jpg@s_2,w_820,h_410" w="1440" h="720" title="苹果高管回应 M1 Pro/Max MacBook Pro 刘海屏为何不支持 Face ID 和触控" srcset="https://img.ithome.com/newsuploadfiles/2021/10/b52bfb05-da93-4b9d-8f34-40bb406a6420.jpg 2x" width="1440" height="410" referrerpolicy="no-referrer"></p><p data-vmark="b3e0">对 Mac 的最大批评之一是缺乏 Face ID。自从 Face ID 在 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone X</a> 上推出以来，一些人想知道苹果是否会把它带到 Mac 上。今年早些时候的一份报告显示，这将在未来发生，但新的 14 英寸和 16 英寸 MacBook Pro 的刘海缺口，以及缺乏 Face ID，使辩论重新成为主流。</p><p data-vmark="eee7">键盘右上角的 Touch ID 让用户只需将手指放在传感器上就能轻松认证。不过，Mac 上的 Face ID 可能会更容易，因为一旦用户看向显示屏，Mac 就会解锁，这与 Face ID 在 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 和 iPad 上的行为相似。</p><blockquote><p data-vmark="f8e3">“那么 Face ID 呢？当我盯着笔记本电脑的巨大凹槽时，我想知道为什么我不能用脸来解锁机器。Tom Boger 先生说，Touch ID 在笔记本电脑上更方便，因为你的手已经在键盘上了。”</p></blockquote><p data-vmark="92f4"><img src="https://img.ithome.com/newsuploadfiles/2021/10/7815459f-86bc-4919-8e33-73b6dfafa4b6.png" w="867" h="573" title="苹果高管回应 M1 Pro/Max MacBook Pro 刘海屏为何不支持 Face ID 和触控" width="867" height="542" referrerpolicy="no-referrer"></p><p data-vmark="c2fb">Ternus 和 Boger 还回答了关于新推出的 14 英寸和 16 英寸 MacBook Pro 以及苹果芯片的更多具体问题。最近的 MacBooks 的特点是用户无法升级内存，因此如果用户发现他们需要比电脑出厂时更多的内存，但内存数量无法自行升级。</p><p data-vmark="6954">IT之家获悉，两位高管都表示，苹果芯片的“统一内存架构”使采用 Apple Silicon 芯片的 Mac 电脑具有更高的性能，这表明如果没有 UMA，类似的性能水平是无法实现的。</p><p data-vmark="59ab">新的 MacBook Pro 2021 带回了 Mac 用户享有的许多功能，这些功能在 2016 年笔记本重新设计期间被移除。最值得注意的是，苹果带回了额外的端口，如 HDMI 和 MagSafe 磁吸端口，取消了 Touch Bar，改进了显示屏，等等。</p><p data-vmark="90f5">在谈到今年的逆转变化时，Tom Boger 表示，苹果一直在“倾听客户的意见”，这意味着它最终需要撤销之前的一些 Mac 设计决定。</p><blockquote><p data-vmark="ea47">“苹果公司 Mac 和 iPad 产品营销副总裁 Tom Boger 表示：我们一直在倾听客户的意见，在这个新的 MacBook Pro 系列中，我们决定做出了一些改变，因为我们在 Mac 上做了很多。”</p></blockquote><p data-vmark="8e93">其他花絮还包括苹果在与独立维修店合作方面的持续“工作”。</p><blockquote><p data-vmark="b691">“正如我几个月前发现的那样，如果你把水洒满了 MacBook，而且没有 AppleCare，你可以通过去独立维修店节省数百美元--但前提是该维修店有正确的零件和信息来修复它们。他们往往没有。Tom Boger 先生说，该公司将继续 在这一领域开展工作。”</p></blockquote><p data-vmark="9f58">目前苹果还没有推出防水型 MacBook 的计划。</p><blockquote><p data-vmark="d8e2">“当我问及防水笔记本电脑时，他们两人都在 Zoom 上茫然地盯着我。Tom Boger 先生说：这并不在许多人的意愿名单上。”</p></blockquote>
