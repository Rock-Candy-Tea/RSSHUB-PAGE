<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 19 日消息 外媒 Neowin 报道，<a href="https://www.ithome.com/0/569/037.htm" target="_blank">微软已经慢慢开始向 Windows 11 截图工具、计算器等内置应用推出了一堆 App 更新</a>，作为这些应用更新的一部分。这些并不代表只是视觉 UI 上的更新，因为微软正在为其增加新的功能。就截图工具而言，微软正在将该应用统一为一款单一产品。微软首席产品官 Panos Panay 本月早些时候预告的另一款应用程序包括<a href="https://www.ithome.com/0/567/540.htm" target="_blank">新的闹钟和时钟应用程序</a>，该应用程序具有“Focus Sessions”和 Spotify 集成功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/9e6ef32d-55eb-40e3-8340-5f77228ce4ad.jpg" w="760" h="428" 中占有一席之地。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/846617b4-7b1f-4e35-b832-876341517f25.png" w="742" h="537" title="微软官方预热展示 Win11 画图全新设计：现代图标、圆角 UI，浅/深色主题" width="742" class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 画图升级的用户界面，包括现代图标、圆角 UI，以及对浅色和深色主题的支持。特别是主题选项，似乎是整个内置应用程序的重点，甚至计算器现在也有这个功能。选择字体风格和笔刷类型的方法也变得更容易了，特别适合使用手写笔的用户。</p><p><img referrerpolicy="no-referrer"></p><p>与 Windows 文件资源管理器一样，微软画图应用将获得一个新的标题菜单，称为“命令栏”，将取代传统的 Ribbon 菜单。新的画图感觉更现代，并为基于触摸的设备进行了优化。</p><p>当然，新版画图会自动遵循 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 1</a>1 系统的主题。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/94cffc7b-f2f4-4538-9f36-e13eef17e7f2.jpg@s_2,w_820,h_292" w="1196" h="426" title="微软官方预热展示 Win11 referrerpolicy="no-referrer"></p><p>微软正在为平板电脑和桌面用户优化画图的控件。例如，你可以点击“A”来直接在图片上插入文字，并从新的下拉菜单中改变对齐方式或字体风格。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/21aef3a5-e1be-4c0c-96dd-2cd72fe5eb35.jpg@s_2,w_820,h_272" w="1148" h="381" title="微软官方预热展示 Win11 画图全新设计：现代图标、圆角 UI，浅/深色主题" referrerpolicy="no-referrer"></p><p>画图从来都不是一款非常全面的编辑工具，在这个版本中它不会获得高级编辑功能。值得庆幸的是，微软画图并没有失去功能，如果你想对图片进行微小的调整，它是理想的选择。</p><p>在新版画图中，你仍然可以调整图像的大小和裁剪。你还可以使用“选择”工具（星形、矩形等）、橡皮擦、填充工具、挑选颜色、铅笔和画笔等等。微软似乎也在测试一种选择画笔类型的新方法，而且已经为基于触摸的用户进行了优化。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/17286f5a-093b-4d76-8f44-38677469e0de.jpg" w="696" h="578" title="微软官方预热展示 Win11 画图全新设计：现代图标、圆角 UI，浅/深色主题" width="696" height="578" referrerpolicy="no-referrer"></p><p>IT之家获悉，经典的“画图”应用经历了一些坎坷，微软曾计划用“画图 3D”来取代这一产品。然而，今年早些时候，画图 3D 不再与 Windows Insider 版本的新安装捆绑在一起，这标志着微软对画图应用的关注以及 Windows 本身的 3D 模式的结束。现在，这款流行的经典产品复兴，对那些仍然依赖这个简单而实用的工具的人来说是个好消息。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/b2d31adf-1f63-45e2-866b-7ba568fdcfa2.jpg" w="760" h="428" title="微软官方预热展示 Win11 画图全新设计：现代图标、圆角 UI，浅/深色主题" width="760" height="428" referrerpolicy="no-referrer"></p><p>就像其他的 Windows 11 的内置应用程序一样，更新的画图应用程序也可能通过微软商店提供。除了“画图”，微软预计还将对“照片”应用进行改进，这是它在 Windows 11 公告中预告的。由于这些应用程序的更新是通过微软商店推出的，它们不需要与 Windows 11 的构建进行特别的绑定联系。</p>
