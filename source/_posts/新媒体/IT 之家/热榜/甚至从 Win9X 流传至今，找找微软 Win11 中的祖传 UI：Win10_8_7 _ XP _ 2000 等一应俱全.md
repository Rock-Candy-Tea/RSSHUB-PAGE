<p data-vmark="a11d">众所周知，虽然微软在 Windows 11 中，对系统 UI 作了大刀阔斧的修改，试图将系统 UI 变得统一化、现代化，<span class="accentTextColor">但 Win11 仍不可避免残留着大量的祖传 UI</span>。</p><p data-vmark="a064">在某些显而易见的地方，我们就可以观测到 Win11 系统界面的割裂，例如同时存在控制面板和设置 App；但对于 Win11 的 UI 到底残留了多少祖传的玩意儿，大部分朋友依然没个概念。</p><p data-vmark="6772" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/ab52ace0-4464-46c6-a922-bb7d7ba164aa.jpg" w="700" h="368" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="368" referrerpolicy="no-referrer"></p><p data-vmark="e69a">今天，就让我们一起来找找 Win11 中残存的祖传 UI，来看看一套设计能在微软系统中流传多少年吧！</p><p data-vmark="233b">注意：Win11 仍在不断更新，界面会持续有变化，本文以 <span class="accentTextColor">Windows 11 Build 22000.613</span> 为对象。</p><h2 data-vmark="1ba9">Win11 中的 Win10、Win8 UI</h2><p data-vmark="f31f">Metro / Modern 磁贴元素，是 Win10、Win8 中最标志性的设计语言。这套设计已经被微软所淘汰，但事实上是淘汰了又没完全淘汰，Win11 中，依然可以轻易淘到很多源自 Win10、Win8 的元素。</p><ul class=" list-paddingleft-2"><li><p data-vmark="ff0a"><strong>音量、亮度控制条</strong></p></li></ul><p data-vmark="3824">使用键盘快捷键调节系统音量或者亮度的时候，浮现出的控制台其实和 Win8、Win10 中的一模一样，方方正正，和 Win11 的画风不能说一模一样吧，只能说毫不相干了。</p><p data-vmark="3065" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/b35d158f-08e9-4675-ad3e-7b2a6fe3254f.jpg" w="700" h="467" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="467" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="c68f"><strong>添加新用户界面</strong></p></li></ul><p data-vmark="e9de">运行“netplwiz”命令，打开用户账户管理页面，点击下方的“添加”，进入到添加新用户的流程，可以看到弹出的设置用户信息窗口中，完全就是 Win8 / Win10 的风格，方正得就如同从未被磨平过棱角。</p><p data-vmark="c7af" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/8d1552e2-417d-4885-97f5-f25791d75984.jpg" w="700" h="438" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="438" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="31f4"><strong>应用程序选择器</strong></p></li></ul><p data-vmark="3eeb">右键点击某个文件，选择文件的打开方式，点击“选择其他应用”后出现的应用选择器浮窗中，也可以领略到原汁原味的 Win10 方块风尚。</p><p data-vmark="fc33" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/ce009f18-e920-40f7-a9a5-9e5656344606.jpg" w="700" h="438" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="438" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="a4e0"><strong>高级启动</strong></p></li></ul><p data-vmark="0ad2">在设置中，进入系统菜单，找到高级启动选项，所进入的界面，简直就是 Win8 再生、Win10 再临。</p><p data-vmark="1d2d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/f2b74b56-13bf-4fcc-b2e8-d5b4881a1c2c.png" w="700" h="525" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="525" referrerpolicy="no-referrer"></p><p data-vmark="6fb7">……</p><p data-vmark="14e1">Win10 仅仅是 Win11 的前一代，因此在 Win11 中残留的元素，其实远不止上文所提到的，限于篇幅，权作抛砖引玉吧。</p><h2 data-vmark="dca2">Win11 中的 Win7、Vista UI</h2><p data-vmark="6b49">在 Win11 中，经典的 Win7、Vista 界面可谓是无处不在、数不胜数。</p><ul class=" list-paddingleft-2"><li><p data-vmark="3569"><strong>控制面板</strong></p></li></ul><p data-vmark="ce59">Win11 的控制面板和 Win7、Vista 几乎是一脉相承的。</p><p data-vmark="5d74" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/43ac310e-bbed-458a-9842-4e31914f4cdd.jpg" w="700" h="438" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="438" referrerpolicy="no-referrer"></p><p data-vmark="249f">而在控制面板中不断点击“往上”的按钮，还可以看到当年经典的 Ribbon 菜单，在很多人眼中，搭载 Ribbon 菜单的文件资源管理器，可比现在 Win11 的这套好用多了。</p><p data-vmark="e920" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/07f5a9c8-a9ce-48f9-88a6-ef5d1e31cc6f.png" w="700" h="527" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="527" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="68b1"><strong>Windows Media Player 等预装程序</strong></p></li></ul><p data-vmark="c92a">Win11 大量的自带程序都沿用自 Win7、Vista，例如 Windows Media Player，浓浓的 Aero 风格设计，令人怀念不已。而除了 WMP 外，远程桌面、写字板等自带程序也是同样的作风。</p><p data-vmark="4b04" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/6b4839a5-a1d8-4f84-9da4-b9cee17ca869.jpg" w="700" h="438" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="438" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="e35d"><strong>Windows 系统安装</strong></p></li></ul><p data-vmark="c489">Vista 奠定了 Windows 安装管理器 WIM 的基础，直到 Win11，在安装新系统的时候，尽管可能改了个 Logo，但其实调用的依然是从 Vista 沿袭过来的 WIM。</p><p data-vmark="5c83" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/d816749b-688d-41d8-be71-24ddcd681e28.png" w="700" h="531" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="531" referrerpolicy="no-referrer"></p><p data-vmark="270a">……</p><p data-vmark="ef0c">Win11 中的 Win7、Vista 元素实在是太多了，本文不可能一一列举。之所以如此，是因为 Windows 面向消费市场的版本，从 Vista 开始改用了 Windows NT 6.x 内核，奠定了此后的 Windows 系统的基础。虽然从 Win10 开始，系统内核被命名为“NT 10.x”，但本质仍是从 Vista 开始的“NT 6.x”系列（Win8.1 内核仍被命名为“NT 6.3”）。</p><p data-vmark="6ebb">Win7 可以看作是 Vista 的小改版，这两个系统对后续 Windows 的影响，是难以估量的，Win11 中仍会出现如此多 Vista 和 Win7 的元素，也就在情理之中了。</p><h2 data-vmark="7d9e">Win11 中的 Windows XP UI</h2><p data-vmark="eaca">说实话，Windows XP 的元素在 Win11 中并不算多 —— 但不意味着没有。</p><ul class=" list-paddingleft-2"><li><p data-vmark="e8c4"><strong>驱动安装对话框</strong></p></li></ul><p data-vmark="8d61">在设备管理器中，手动为某个硬件安装驱动程序，选择了本地驱动文件后，所弹出的对话框，就是经典的 XP 范式。</p><p data-vmark="ed12" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/2ab960bd-4000-4bd7-8699-c2dd27bd6763.jpg" w="700" h="333" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="333" referrerpolicy="no-referrer"></p><p data-vmark="2ce1">Windows XP 所使用的内核是 NT 5.x，经过 Vista 的大改后，很多 XP 的 Luna 界面元素都被剔除，在 Win11 中 XP UI 的残留量的确不可和 Vista、Win7 同日而语。</p><h2 data-vmark="5e94">Win11 中的 Windows 2000 UI</h2><p data-vmark="27a2">作为使用 NT 内核的最早的 Windows 系统之一，尽管 Windows 2000 隶属于服务器产品线，但很多组件也深刻影响到了现在的 Win11.</p><ul class=" list-paddingleft-2"><li><p data-vmark="03c9"><strong>Window 控制台 MMC</strong></p></li></ul><p data-vmark="cb4e">这是从 Windows 2000 以来，就引入系统的组件。时至今日，隶属于 MMC 的组件，例如计算机管理、设备管理器等，仍保留着 Windows 2000 相若的界面和交互（基本就换了换图标），二十年不变。</p><p data-vmark="3528" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/1806f2df-685e-42b1-9ea8-f8e97781e215.jpg" w="700" h="438" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="438" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="8f66"><strong>Windows 安装器</strong></p></li></ul><p data-vmark="3dd8">当你安装软件的时候，会调用 Windows 安装器，这也是传承自 Windows 2000 的沉淀，从中，我们甚至还可以看到和二十年前一样的图标。</p><p data-vmark="ed00" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/adb2d735-7320-43f6-be8c-7824b2f761e0.png" w="700" h="544" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="544" referrerpolicy="no-referrer"></p><p data-vmark="cf97">……</p><p data-vmark="c82c">Windows 2000 对于后续 Windows 版本的影响是巨大的，Windows XP 中也存在很多源自 Windows 2000 的痕迹，这些痕迹直到 Win11 都依旧没有去除，或者得以更新。</p><h2 data-vmark="1463">Win11 中的 Windows 9x、Windows 3.x 元素</h2><p data-vmark="77e1">轮到熬了三四十年的精华现身了！由于这些元素的确比较远古，要找到它们需要一点小技巧。</p><ul class=" list-paddingleft-2"><li><p data-vmark="5860"><strong>CHM 阅读器</strong></p></li></ul><p data-vmark="4326">不知道大家有没有接触过 CHM 这种文件格式，以前很多电子书会制作成 CHM，一些软件乃至 Windows 系统本身的说明文件，也会用 CHM 格式保存。而如果你在 Win11 中打开 CHM 文件，那么则会看到<span class="accentTextColor">源自 Windows 9x 系统</span>的 CHM 阅读器，连图标都没改过，可谓源远流长。</p><p data-vmark="c7b6" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/c6a818eb-04bf-47d2-9f45-3e37441c1203.png" w="700" h="469" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="469" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="3997"><strong>ODBC</strong></p></li></ul><p data-vmark="e009">这是 Windows 系统中，用来连接不同数据库的接口，它的管理界面相当复古，当你添加数据库的时候，甚至会看到<span class="accentTextColor">来自 Windows 3.x 的 UI 元素</span>。</p><p data-vmark="1496" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/42080452-ad8c-4c13-8651-e2c71d72da0c.png" w="700" h="492" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="492" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="ac9b"><strong>图标</strong></p></li></ul><p data-vmark="9a98">其实 Win11 中，藏有丰富的历代 Windows 的各式图标，上文没有拿出来说事，是因为像 Win10、Win8、Win7 等历代 Windows 的图标都有，大家一眼就可以看到，没什么好说的。但大家或许缺乏一点想象力 ——Win11 甚至还藏有 Windows 3.x 的图标，这就很夸张了。</p><p data-vmark="98df">例如，右键点击某个快捷方式，查看它的属性，之后点击“更改图标”。在路径中，输入“moricons.dll”，就会看到来自于 Windows 3.x 年代的图标，一股古朴的气息扑面而来……</p><p data-vmark="a008" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/41e0b51e-6313-4d0a-ba8f-2d603120fc42.png" w="700" h="519" alt="Win11界面 Win11 UI" title="甚至从 Win9X 流传至今，找找微软 Win11 中的祖传 UI：Win10/8/7 / XP / 2000 等一应俱全" width="700" height="519" referrerpolicy="no-referrer"></p><p data-vmark="4416">实际上，Windows 的整体操作逻辑很早就固定下来了，一些不影响使用的组件功能，的确没有必要进行改动。不过从最新版的 Win11 中，能找到几十年前的图标，还是很令人惊讶的。</p><h2 data-vmark="19e4">总结</h2><p data-vmark="cf6b">Windows 是历史悠久的产品，而强大的兼容性，是其多年来在市场上屹立不倒的杀手锏。而为了向前兼容，Windows 显然做了大量工作，不少老旧的组件不得不保留。然而，在比较表层的地方，Windows 至今仍未能呈现统一的风格设计，这就很让人感到困扰了。本文所找到的祖传 UI，只是冰山一角，大家都可以在 Win11 找到诸多不和谐因素。</p><p data-vmark="d536">Windows 并非没有大改版过，例如 Win7，尽管 Win7 中也存在诸如 MMC 等源自更早 Windows 的工具，但系统看起来还是非常和谐统一的。为何从 Win8 开始，微软就再也做不到这点了呢？希望微软能够仔细打磨一下 UI，适当翻新、淘汰一些老组件，为用户带来更好的体验吧。</p>
