<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 29 日消息 在上周微软发布 Windows 11 后公布了最低系统要求，微软决定将老一代 CPU 从 Windows 11 兼容性列表中取消，引发网友声讨。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/6baa9875-e6af-4e91-9aa6-b47a244a4286.png" title="微软解释 Win11 最低系统要求的原因：可实现 99.8% 的无崩溃体验" referrerpolicy="no-referrer"></p><p>今天，微软推送了 Win11 的第一个预览版，之后更新了最新系统要求，微软表示他们在测试第 7 代英特尔处理器和 AMD 初代锐龙的兼容性，以决定 Windows 11 正式版是否加入此类 CPU 支持。</p><p>从微软的最低硬件要求来看，Windows 11 要求硬件能够提供 Windows Hello、设备加密、基于虚拟化的安全 (VBS)、管理程序保护的代码完整性 (HVCI) 和安全引导等保护，从而提高了安全性。</p><p>微软对此也分享更多关于“我们为什么更新 Windows 11 系统要求”的细节。</p><p>IT之家了解到，微软解释道，他们的最低要求遵循三个原则：安全、可靠性、兼容性。</p><ul class=" list-paddingleft-2"><li><p>安全：这些新特性已经被证明可以在测试机型上减少 60% 的恶意软件侵害。为满足该原则，所有支持 Windows 11 操作系统的 CPU 都拥有嵌入式的 TPM，支持安全引导，支持 VBS 和特定的 VBS 能力。</p></li><li><p>可靠性：升级到 Windows 11 的设备将处于受支持和受支持且可靠的状态。通过选择采用新的 Windows 驱动的 CPU，并得到 OEM 及其合作伙伴的支持，他们在这些机型上实现了 99.8% 的无崩溃体验。</p></li><li><p>兼容性：Windows 11 具有较高的兼容性，它要求用户使用 1GHz 以上的双核处理器、4GB 内存和 64GB 存储的基础，而这正是微软 Office 和 Microsoft Teams 的最低系统。</p></li></ul><p>《<a href="https://www.ithome.com/0/559/885.htm" target="_blank">微软降低 Win11 的 CPU 要求：七代酷睿、初代锐龙有救了</a>》</p>
