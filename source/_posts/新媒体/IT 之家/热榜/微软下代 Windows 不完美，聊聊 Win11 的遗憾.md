<p>微软已经在 6 月 24 日确认了下一代 Windows 系统 ——<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>，并在近日于 Insider 通道推送了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 的预览版。和 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 相比，Win11 更新的系统界面，带来了覆盖面更广的 Fluent Design 界面设计，并且更新了一系列的系统组件，并在 WDDM 等内部机制上，迭代了技术。</p><p>可以说，Win11 解决了一些在 Win10 中长期存在的问题，并引入了一些激动人心的新技术，不过这并不意味着 Win11 就已经趋近完美。我们已经试用了 Win11 了一段时间，在实际体验中，可以感受到 Win11 仍存在一些遗憾，这就来简单说说吧。</p><h2>仍存在大量未经改进的旧组件</h2><p>Win11 的一大卖点，在于系统更广泛地使用了微软近年来力推的全新设计语言 Fluent Design。实际上，Fluent Design 在 Win10 中就已经开始推广，然而对于 Win10 来说，这是一个在版本生命周期中，中途突然插入的新特性 ——Win10 刚发布时，主推的还是 Modern 磁贴界面。加上 Win10 本身残留了大量传统组件，使得系统中存在严重的界面割裂问题，你可以在 Win10 中轻松找到三种设计风格、交互方式大相径庭的 UI。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/512a68f7-9b21-4235-8c4a-06f32487aba2.jpg" w="640" h="400" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="640" height="400" referrerpolicy="no-referrer"></p><p>Window 10 的 UI 这么多年过去，仍是个半成品，连界面和图标风格都尚未统一，Win11 就做到了吗？</p><p>Win11 作出了很大的努力，以解决相关问题，但 Win11 解决得显然不够完美。尽管 Win11 重新设计了设置面板、开始菜单、资源管理器等组件的外观，但系统中实际上仍包含着大量未经修改的老旧组件，最典型的例子就是控制面板。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/47688b9c-ff9a-4afe-b721-b5b71180ddd0.jpg" w="640" h="429" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="640" height="429" referrerpolicy="no-referrer"></p><p>Win11 中仍存在传统控制面板</p><p>长年以来，控制面板是 Windows 系统功能配置的入口，然而在 Win10 后，它和新引入的设置面板同时存在，这让系统同时拥有了两个功能近似的入口。微软承诺将会逐步将控制面板的功能迁移到设置面板中，让设置面板渐渐成为系统唯一的设置入口。但在 Win11 中，这显然落空了。</p><p>除了控制面板，Win11 也仍旧存在大量未经修改的旧版组件，例如设备管理器、任务管理器等等，这和系统主推的 UI 格格不入。相比之下，命令行等工具则更新到了 Fluent Design 的新版。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/5b1afd36-1731-44a3-a752-e3ec4be19162.jpg" w="640" h="350" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="640" height="350" referrerpolicy="no-referrer"></p><p>任务管理器也是旧版的 UI</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/ae439742-ed0a-4999-af63-9ffb7f3b3e6d.jpg" w="640" h="570" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="640" height="570" referrerpolicy="no-referrer"></p><p>一些弹窗甚至是四四方方的 Modern 风格，没有圆角</p><p>简而言之，Win11 中仍存在明显的界面割裂问题，希望 Win11 的正式版能够更彻底地走向新设计吧。</p><h2>细节欠缺打磨</h2><p>Windows 的界面 UI 并不以细节见长，Win11 也是如此。我们不难发现，在一些系统的角落中，Win11 依然欠缺细节的打磨，这就举几个例子。</p><p>・窗口圆角存在锯齿。</p><p>Win11 的窗口统一使用圆角渲染，但是效果却不甚如人意，有些圆角存在明显的锯齿，看着颇为不精致。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/7326f6fe-0912-4816-af8e-9cff0f268c83.jpg" w="640" h="375" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="640" height="375" referrerpolicy="no-referrer"></p><p>有时候某些圆角渲染不够精致</p><p>・Windows 商店图标。</p><p>尽管 Win11 中 Windows 商店作了改版，但其中 App 的图标仍是磁贴样式，和 Fluent Design 设计语言格格不入。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/cb3f16a2-a626-4a26-9035-d163b414023d.jpg" w="640" h="446" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="640" height="446" referrerpolicy="no-referrer"></p><p>商店中 App 图标仍是磁贴</p><p>・快捷菜单过于粗糙。</p><p>Win11 对右键快捷菜单进行了改版，然而排布却非常不讲究。为了简洁，快捷菜单隐藏起了所有第三方软件的选项，需要多点击一次“Show more options”才能看到这些选项，但此时菜单又回到了原先的样子，使用起来并不简洁。另外，菜单还经常出现残缺不全、阴影裁切、仅有一个选项等情况，给人一种半成品的感觉。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/7118f3eb-691d-47eb-80e2-b2b5eb953e32.jpg" w="640" h="430" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="640" height="430" referrerpolicy="no-referrer"></p><p>可以看到大的那块右键菜单右侧缺了一部分</p><p>・字符排版出错。</p><p>例如弹出的通知，有时候间隔会异常的大；又例如在某些地方，字符并不居中，这让强迫症很不舒服。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/fa96ed35-a518-4d28-b80c-cb6410117b3a.jpg" w="537" h="313" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="537" height="313" referrerpolicy="no-referrer"></p><p>数字不居中</p><p>Win11 细节上的问题并不止上文中提到的，这令人感到非常膈应 ——Windows 本来就并不以细节见长，这次 Win11 大改 UI 的契机，本是一个能够好好修补细节上不足的机会，但现在看来 Win11 的细节做得也并不怎么样。希望在正式版发布前，微软能进一步查缺补漏，带来一个更加精致的 Win11 吧。</p><h2>消失的好用功能</h2><p>Win11 对一些系统功能进行了重新设计，但也淘汰了一些 Win10 中原本备受好评的功能设计，下面举几个简单的例子。</p><h2>任务栏位置调整。</h2><p>Win11 对系统任务栏进行了改版，但与此同时任务栏失去了可以调整位置的特性。以往很多朋友习惯于让任务栏靠在侧边或者顶部，但在 Win11 中，任务栏只能够固定在底部。目前通过修改注册表还能够移动 Win11 的任务栏，不知道 Win11 正式版中是否还能有这样的操作。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/eb1eca1b-d97e-4bf9-9007-7c6e5ed948da.jpg" w="640" h="376" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="640" height="376" referrerpolicy="no-referrer"></p><p>通过修改注册表把任务栏固定到屏幕上方，但弹出开始菜单的位置就错乱了</p><ul class=" list-paddingleft-2"><li><p>任务栏无法拖放图标。</p></li></ul><p>在之前的 Windows 系统中，可以将程序或者快捷方式的图标直接拖到任务栏固定，但 Win11 却取消了这点，如此便捷的操作被删掉令人百思不得其解。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/ac571524-30d0-4047-b1c4-314071f00606.jpg" w="640" h="242" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="640" height="242" referrerpolicy="no-referrer"></p><p>无法直接将软件或者快捷方式直接拖到任务栏固定</p><ul class="ai-word-checked list-paddingleft-2"><li><p>多任务时间线被砍。</p></li></ul><p>Win10 的多任务界面带有时间线功能，简而言之就是可以让你多设备在不同时间段开启过的应用记录，都同时显示在多任务界面中。对于用多部设备办公的朋友来说，这可以提供更加顺畅的工作流信息，很遗憾 Win11 取消了这个功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/d17078db-65f8-4ad8-9be5-093aa8f9a781.jpg" w="640" h="420" title="微软下代 Windows 不完美，聊聊 Win11 的遗憾" width="640" height="420" referrerpolicy="no-referrer"></p><p>多任务界面没有了时间线</p><ul class=" list-paddingleft-2"><li><p>不能使用本地账户登录。</p></li></ul><p>进入系统的时候，必须使用微软账户登录，但由于种种原因，微软账户在国内的连接其实并不算很稳定，死活登录不上的时候会令人抓狂。好消息是，这个限制只存在于家庭版，而且只会在首次配置全新的 Win11 的时候才会出现，专业版的 Win11 或者从旧系统升级上去的 Win11，还是可以使用本地账户的。</p><h2>总结</h2><p>除了上文中提到的，Win11 的一些改变也令人感到不适，例如硬件必须支持 TPM 2.0 才能安装等等。但无论如何，Win11 对比 Win10 的改变仍是巨大的，从中我们可以看到微软对用户体验的思考，以及对系统革新的决心。希望在 Win11 正式版中，有更多的遗憾可以被弥补，让我们期待 Win11 正式版的到来吧。</p>
