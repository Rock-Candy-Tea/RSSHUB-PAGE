<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 23 日消息 根据外媒 VideoCardz 消息，AMD 未来的锐龙 7000 系列桌面处理器将不再使用 AM4 接口，更新为 AM5。今日一位爆料者 @ExecuFix 在推特称，<span class="accentTextColor">AM5 接口将取消 CPU 上的针脚</span>，由 PGA 转向 LGA（land grid array），使得针脚放置在主板上，减小损坏的风险。此外，新处理器将有 1718 个针脚，LGA1718 规格。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/2ff601ee-601a-4a7c-aa82-fe92033ef9c0.png" w="814" h="548" title="针脚要消失了：AMD 下一代 AM5 CPU 接口曝光，LGA1718 规格" width="814" height="548" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/bbdab6fb-90bc-4fa7-b4c8-3d4e5bc886ed.png" w="894" h="564" title="针脚要消失了：AMD 下一代 AM5 CPU 接口曝光，LGA1718 规格" width="894" height="517" referrerpolicy="no-referrer"></p><p>爆料者还透露，<span class="accentTextColor">AMD 这一代桌面处理器还会支持双通道 DDR5 内存，但是仅仅支持 PCIe 4.0</span>。PCIe 5.0 的支持仅限代号为“Genoa”的 EPYC 霄龙服务器处理器。尽管 CPU 接口更换为 LGA1718，但是 CPU 的大小并没有改变，依旧为 40mm×40mm。</p><p><img src="https://www.techpowerup.com/img/16-11-29/17bee3dd3d3d.jpg" title="针脚要消失了：AMD 下一代 AM5 CPU 接口曝光，LGA1718 规格" referrerpolicy="no-referrer"></p><p>▲ AM4 接口的 CPU，本体搭载密集的针脚</p><p>目前 AMD 最新的桌面芯片组为 500 系列，下一代为 600 系列。这一代锐龙 5000 系处理器已经支持 PCIe 4.0 通道，足以满足大部分性能需求。外媒表示英特尔第 12 代 Alder Lake 平台确定将同时支持 PCIe 5.0 和 DDR5，将是首个同时支持两种新协议的 CPU 平台。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/2849dba0-d53f-4f04-a682-fa68b0e3cab6.png" w="897" h="621" title="针脚要消失了：AMD 下一代 AM5 CPU 接口曝光，LGA1718 规格" width="897" height="568" referrerpolicy="no-referrer"></p><p>▲ 图片来自 VideoCardz</p><p>IT之家此前报道，路线图显示采用 AM5 接口的 AMD 桌面锐龙处理器代号为“Raphael”（拉斐尔），<span class="accentTextColor">将采用 5nm 制程工艺制造，升级为 Zen4 内核架构</span>，集成 Navi2 核显，该产品预计为锐龙 7000 系列。而在这之前，锐龙 6000 系列处理器代号为“Warhol”（沃霍尔），采用 Zen3 + 架构、6nm 制程工艺，预计将于今年发布。</p>
