            <p data-vmark="66df"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 16 日消息，据 Windows Latest 报道，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> KB5010414 累积补丁更新现在正向生产频道的用户推出，它是带有几个新功能的大版本。与典型的月度更新不同，今天的可选更新是微软在主要功能更新之外用新功能升级 Windows 11 计划的一部分。</p><p data-vmark="82a1" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/099b3d2b-0405-4125-9c55-0849a3c62a2c.jpg" w="768" h="432" title="微软 Win11 Build 22000.527 (KB5010414) 正式发布：商店运行安卓 App，任务栏天气小组件...（附更新内容大全）" width="768" height="432" referrerpolicy="no-referrer"></p><p data-vmark="64c8">Windows 11 KB5010414 是一个可选的更新，所以它不会主动下载或安装，除非你明确点击“获取更新”按钮。2022 年 2 月的可选更新带来了对新任务栏功能的支持，如“AirDrop”和多显示器上的时钟。此外，还有一堆一般性的改进和修复，使其成为重量级更新。</p><p data-vmark="ae06">这是一个可选的更新，选择安装它的用户将间接帮助微软测试下个月的“星期二补丁”。安装可选更新的用户被认为是“探索者”。请记住，你以后可以在 2022 年 3 月的“星期二补丁”获得同样的一组变化更新。</p><p data-vmark="c03d">Windows 11 KB5010414（Build 22000.527）的新功能如下：</p><ul class=" list-paddingleft-2"><li><p data-vmark="9cf8">通过微软商店推出的<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a> App。</p></li><li><p data-vmark="8e64">任务栏现在可以在多个显示器上显示时钟。</p></li><li><p data-vmark="403d">从任务栏轻松切换麦克风，并在 Teams 中分享应用程序的窗口。</p></li><li><p data-vmark="8a39">任务栏现在可以显示天气信息。</p></li><li><p data-vmark="8a21">“开始”菜单的推荐部分得到了 Office Web 整合。</p></li></ul><p data-vmark="2daf">我们大多数人都熟悉 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 的《新闻和兴趣》小组件，它出现在任务栏的右侧，根据你的个人兴趣显示本地天气更新和新闻头条。</p><p data-vmark="6b2b">另一方面，微软对 Windows 11 的做法略有不同。在 Windows 11 上，我们有一个小组件面板，它包含了天气、新闻、交通、微软待办事项和其他小组件（如照片）等细节的信息流。与安卓小组件一样，Windows 11 的小组件面板也有小的应用程序，为你提供快速信息。</p><p data-vmark="95c0">Windows 11 小组件面板支持《新闻和兴趣》中提供的所有功能，但它不能在任务栏上显示天气。这一功能在 Windows 11 中被从任务栏中抛弃，但在 Build 22000.527 中得到了恢复。</p><p data-vmark="c903" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/1daf4828-9657-4a99-beb5-fc7887bb2d2e.jpg@s_2,w_820,h_548" w="1424" h="952" title="微软 Win11 Build 22000.527 (KB5010414) 正式发布：商店运行安卓 App，任务栏天气小组件...（附更新内容大全）" srcset="https://img.ithome.com/newsuploadfiles/2022/2/1daf4828-9657-4a99-beb5-fc7887bb2d2e.jpg 2x" width="1424" height="548" referrerpolicy="no-referrer"></p><p data-vmark="307b">正如你在截图中所看到的，小组件现在位于默认任务栏的最左边，并显示天气更新。当你将鼠标悬停在任务栏上的天气按钮上时，带有新闻源和其他小组件的小组件面板将自动启动。</p><p data-vmark="9953">今年晚些时候，小组件面板也将获得对第三方应用程序的支持。</p><h3 data-vmark="5ffe">任务栏获得新功能</h3><p data-vmark="c3fe">Windows 11 的新功能称为“分享窗口”，当你想在微软 Teams 会议期间分享一个应用程序的窗口时，可以减少在应用程序之间来回移动的需要。“分享窗口”也将支持其他视频会议客户端，但目前仅限于 Microsoft Teams。</p><p data-vmark="ba1f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/cdc3fefc-346c-49ca-a1e9-b40d5c582a23.jpg@s_2,w_820,h_461" w="1280" h="720" title="微软 Win11 Build 22000.527 (KB5010414) 正式发布：商店运行安卓 App，任务栏天气小组件...（附更新内容大全）" srcset="https://img.ithome.com/newsuploadfiles/2022/2/cdc3fefc-346c-49ca-a1e9-b40d5c582a23.jpg 2x" width="1280" height="461" referrerpolicy="no-referrer"></p><p data-vmark="64e9">根据微软的说法，Windows 11 任务栏的新窗口共享体验可以使在线会议期间的中断次数减少，前提是你能正确利用它。</p><p data-vmark="d75b">要开始使用，你需要将鼠标悬停在任务栏的应用程序上，并点击新的按钮，允许你在会议中与其他人共享一个窗口。当你分享屏幕时，你可以随时点击“停止分享”按钮，或者通过点击“分享此窗口”切换到另一个应用程序。</p><p data-vmark="c22c">微软表示，这项新功能对于使用任何视频会议应用程序共享 PowerPoint 幻灯片将非常有用（对第三方应用程序的支持即将推出）。</p><p data-vmark="846e">此外，微软正在为 Windows 11 任务栏引入对新的静音和取消静音按钮的支持。这项新功能对于经常在 Teams 上进行会议的用户来说将非常方便。</p><p data-vmark="b9f5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/d362a1ae-f046-4ba3-9325-a3aff0e24a9e.jpg@s_2,w_820,h_273" w="840" h="280" title="微软 Win11 Build 22000.527 (KB5010414) 正式发布：商店运行安卓 App，任务栏天气小组件...（附更新内容大全）" srcset="https://img.ithome.com/newsuploadfiles/2022/2/d362a1ae-f046-4ba3-9325-a3aff0e24a9e.jpg 2x" width="840" height="273" referrerpolicy="no-referrer"></p><p data-vmark="1f52">正如你在截图中看到的，你将能够直接从任务栏访问新的静音或取消静音切换。该功能在整个通话过程中持续存在，即使屏幕上有几个窗口打开，也能让你从任何地方对 Teams 麦克风进行静音 / 取消静音。</p><p data-vmark="1dec">目前，只支持 Teams，但微软也可能通过 API 为第三方应用程序引入支持。</p><h3 data-vmark="3873">Windows 11 更新中的其他改进</h3><p data-vmark="1718">KB5010414 还带来了其他一些改进。例如，你现在可以直接在浏览器中从“开始”菜单启动 Office 文件。</p><p data-vmark="cf23">当你通过“开始”菜单的推荐部分访问 Office 文件，而设备没有相应的 Office 许可证或文件在 OneDrive 或 SharePoint 中时，这将发挥作用。</p><p data-vmark="d6cf">如果你有一个有效的许可证，该文件将像往常一样在桌面应用程序中打开。</p><p data-vmark="fdfe">同样，现在可以在 Microsoft Edge 的 Internet Explorer 模式和 Microsoft Edge 之间共享 cookies。</p><p data-vmark="0f5d">其他改进包括：</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="f495">修复了沉默的 BitLocker 启用策略不起作用的问题。</p></li><li><p data-vmark="aaa6">修复了一个影响远程桌面应用程序的问题。</p></li><li><p data-vmark="983d">修复了从右到左（RTL）语言文本在上下文菜单上显示为左对齐的问题。</p></li><li><p data-vmark="d4bf">修复了任务栏工具提示的问题，即应用程序图标出现在任务栏上的一个空区域。</p></li><li><p data-vmark="ce65">修复了网络驱动器上的离线文件的问题。</p></li></ul>
