<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a>4月11日消息 GPU 虚拟化是一种将显卡硬件资源分发到许多实例的方式，该技术<span class="accentTextColor">允许多个终端用户使用同一张显卡进行各种任务</span>，例如 3D 建模或科学工作，目前英伟达显卡仅在 Grid、Tesla、Quadro 显卡上支持该技术。</p><p>然而，一群技术爱好者发布了一个名叫 “vgpu_unlock”的破解补丁，可<span class="accentTextColor">使家用的 GeForce 显卡也支持 GPU 虚拟化</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210411_092615_96.jpg@base@tag=imgScale&w=882&h=518&q=90" w="882" h="557" title="英伟达 GeForce 显卡已被民间支持 GPU 虚拟化：一张显卡可在多台电脑上使用" srcset="https://img.ithome.com/newsuploadfiles/2021/4/20210411_092615_96.jpg 2x" width="882" height="518" referrerpolicy="no-referrer"></p><p>▲ 图自外媒 videocardz</p><p>IT之家了解到，该破解补丁的原理其实很简单，就是通过<span class="accentTextColor">欺骗驱动程序，使其读取到 Grid、Tesla、Quadro 显卡的 PCI 设备 ID</span>，从而开启 GPU 虚拟化。</p><p>按理说，该技术在 GeForce 显卡上实现后，最主要的场景就是网吧，可以用一张显卡带动多台机器的游戏。然而，<span class="accentTextColor">该技术目前不支持 Windows</span>，仅可在 Linux 和 KVM 虚拟机上使用，最多也只能在 WSL 上跑。</p><p>“vgpu_unlock”补丁已在 GitHub 上发布，感兴趣的IT之家小伙伴可以<a href="https://github.com/DualCoder/vgpu_unlock" target="_blank">点此查看</a>。</p><h2>以下为支持的显卡列表：</h2><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210411_092554_18.png" w="628" h="912" title="英伟达 GeForce 显卡已被民间支持 GPU 虚拟化：一张显卡可在多台电脑上使用" width="628" height="912" referrerpolicy="no-referrer"></p>
