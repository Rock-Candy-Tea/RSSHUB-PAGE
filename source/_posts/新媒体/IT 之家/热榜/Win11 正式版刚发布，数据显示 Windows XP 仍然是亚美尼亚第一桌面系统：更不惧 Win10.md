<p data-vmark="6ead"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 30 日消息，今年是 2021 年，微软已经迎来了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>。虽然很多很多人仍然在使用旧的 Windows 操作系统，这可能并不令人震惊，但有一个国家或地区仍然主要使用 Windows XP，这是相当不寻常的情况了。</p><p data-vmark="8d76"><img src="https://img.ithome.com/newsuploadfiles/2021/10/58fbecb3-a476-444f-863c-6137170b7ff9.png" w="934" h="583" title="Win11 正式版刚发布，数据显示 Windows XP 仍然是亚美尼亚第一桌面系统：更不惧 Win10" width="934" height="512" referrerpolicy="no-referrer"></p><p data-vmark="3181">据 Windows Central 报道，对许多人来说，2021 年 10 月是一个变革的时间，其中 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 开始被抛弃，而改用 Windows 11。然而，不是每个人都那么热衷于进行操作系统的更换。事实上，有一个国家二十年来都没有大规模变动过。</p><p data-vmark="ad04">IT之家获悉，根据 StatCounter 数据，亚美尼亚的台式机电脑仍然主要使用 Windows XP。截至 2021 年 9 月，53.5% 的台式机在使用 2001 年发布的这款操作系统。Windows 10 排在第二位，份额占 32.8%。鉴于 10 月份的数据还没有包括在图表中，目前还不清楚亚美尼亚已经有多大份额的 Windows 11，但可以安全地假设，这些设备远远不足以推翻 Windows XP 的山头之王地位。</p><p data-vmark="41d6"><img src="https://img.ithome.com/newsuploadfiles/2021/10/cb96c298-ca36-4325-b1d4-7d4f7b12271b.png" w="1440" h="725" title="Win11 正式版刚发布，数据显示 Windows XP 仍然是亚美尼亚第一桌面系统：更不惧 Win10" width="1440" height="413" referrerpolicy="no-referrer"></p><p data-vmark="6b7a">目前还不清楚为什么亚美尼亚的 Windows XP 和 Windows 10 的市场份额值仍然每隔几个月就互相争夺主导权，而 Windows 10 偶尔会打成平手或夺冠。更不清楚为什么 Windows XP 还能反扑增长回来。</p><p data-vmark="4db3">如果你不想或不能升级到 Windows 11，仍然可以愉快地使用 <a class="s_tag" href="https://www.win7china.com/" target="_blank">Windows 7</a>、Windows 8、Windows 10 或其他旧的操作系统，你要知道你并不孤单。世界上有很多人还在使用 20 年前的系统软件。不过现在的问题是，2041 年，你还会使用 Windows 11 吗？</p><p data-vmark="e97b">《<a href="https://www.ithome.com/0/583/034.htm" target="_blank">微软 Windows XP 诞生 20 周年，Win11 继续前进</a>》</p>
