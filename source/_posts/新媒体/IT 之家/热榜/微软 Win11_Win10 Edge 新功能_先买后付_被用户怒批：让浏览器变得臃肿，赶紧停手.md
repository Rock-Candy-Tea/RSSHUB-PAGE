<p data-vmark="7170"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 28 日消息，微软 Edge <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>最近新增了一项功能，允许人们分期支付网上购物的费用，被称为先买后付（BNPL），目前正在 Microsoft Edge Dev 和 Canary 上进行测试。该功能引起了 Edge 浏览器的粉丝和用户的强烈批评，他们在官方宣布该功能的帖子的评论部分表达了不满。</p><p data-vmark="86bb"><img src="https://img.ithome.com/newsuploadfiles/2021/11/68cb3d92-09f9-421d-a8c9-c6f7d701f82a.jpg" w="800" h="600" title="微软 Win11/Win10 Edge 新功能“先买后付”被用户怒批：让浏览器变得臃肿，赶紧停手" width="800" height="600" referrerpolicy="no-referrer"></p><p data-vmark="d13c">大多数用户的抱怨认为，<span class="accentTextColor">微软 Edge 正在变得臃肿，充满了购物功能，而不是提供一个纯粹的浏览器体验</span>。BNPL 功能虽然是可选的，但其反对者认为 Edge 不应该内置购物功能。</p><p data-vmark="6915"><img src="https://img.ithome.com/newsuploadfiles/2021/11/40114072-7b44-422a-a84b-acbad22fa0da.png" w="763" h="701" title="微软 Win11/Win10 Edge 新功能“先买后付”被用户怒批：让浏览器变得臃肿，赶紧停手" width="763" height="701" referrerpolicy="no-referrer"></p><p data-vmark="3cec">用户 vyrotek 说：“令人印象深刻的是，你可以用这样一个可笑的功能把多年的努力和善意丢掉。Edge 团队需要暂停一下，想想他们怎么可能认为这是一个好主意。即使是 Bing 的功能也变得太激进了。”</p><p data-vmark="4329">用户 Cameron_Bush 表示，要求微软重新考虑这个新增功能，“这听起来是一个可怕的想法，只会被媒体视为一个无耻的现金掠夺和/或臃肿。我请求你们重新考虑将其推向市场。这个功能将收到负面新闻，是不值得的。”</p><p data-vmark="5202">用户 GNS722 也表达了类似的看法：“正如上面许多人提到的，请不要继续增加这种臃肿。Edge 已经成为一个非常好的 Chrome 替代品，但如果你要走这条路，它就不会再是了。”</p><p data-vmark="3b62">用户 examinedliving 的评论将微软某些部分的开放性与 Edge 中增加的 BNPL 进行了对比：“似乎微软已经被划分为竞争的圣经实体。一个是纯粹的，创造了像 VS Code 和 Powertoys 这样的东西。另一个是恶魔般的野兽，他想出了这些可笑的、侮辱性的垃圾。”</p><p data-vmark="10a5">IT之家了解到，总共有几页的评论，其中大部分是负面的。Edge 提供了一些帮助人们省钱的功能，批评者似乎认为，像这样的功能应该留给扩展程序和专门的网站，而不是内置于一个浏览器中。</p>
