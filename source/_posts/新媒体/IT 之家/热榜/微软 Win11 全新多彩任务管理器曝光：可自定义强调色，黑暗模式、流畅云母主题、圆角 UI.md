<div class="tougao-user">感谢IT之家网友 <a href="https://m.ithome.com/html/app/open.html?url=ithome%3A%2F%2Fuserpage%3Fid%3D1948668" rel="nofollow">Terrence</a> 的线索投递！</div>
            <p data-vmark="082d"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 27 日消息，作为太阳谷 2（22H2）的一部分，微软正在对 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 进行许多亮点改进，其中就包括<span class="accentTextColor">全新 UI 的任务管理器</span>。</p><p data-vmark="568c">微软正在开发一个基于现有 Win32 框架的新任务管理器，但它支持 WinUI 3.0 设计组件，包括 Windows 11 的 Mica 云母效果、Fluent Design 的 Acrylic 亚克力和其他设计改进。此外，它还支持了深色模式。</p><p style="text-align: center;" data-vmark="a5f5"><img src="https://img.ithome.com/newsuploadfiles/2022/2/3385b181-d54d-4390-bcf3-0f03006b7dc1.jpg" title="微软 Win11 全新多彩任务管理器曝光：可自定义强调色，黑暗模式、流畅云母主题、圆角 UI" referrerpolicy="no-referrer"></p><p style="text-align: justify;" data-vmark="eaa8">而近日，微软在 Windows Insider 播客展示了 Windows 11 任务管理器的<span class="accentTextColor">全新多彩设计</span>，用户可以调整 Windows 11 主题色，<span class="accentTextColor">让任务管理器自定义显示强调色</span>，而非之前让人焦虑的黄色。</p><p style="text-align: center;" data-vmark="3174"><img src="https://img.ithome.com/newsuploadfiles/2022/4/ac8d8415-f0ca-4708-95a0-6016bf6c2fab.jpg@s_2,w_820,h_462" w="1278" h="720" title="微软 Win11 全新多彩任务管理器曝光：可自定义强调色，黑暗模式、流畅云母主题、圆角 UI" srcset="https://img.ithome.com/newsuploadfiles/2022/4/ac8d8415-f0ca-4708-95a0-6016bf6c2fab.jpg 2x" width="1278" height="462" referrerpolicy="no-referrer"></p><p data-vmark="a36d">微软官方首先展示了此前历代任务管理器的设计，可以看到 UI 越来越现代化，色彩也越来越丰富。</p><p style="text-align: center;" data-vmark="adbd"><img src="https://img.ithome.com/newsuploadfiles/2022/4/0adbcfb5-1187-4deb-ad16-fd4c3f654d50.png" w="882" h="494" title="微软 Win11 全新多彩任务管理器曝光：可自定义强调色，黑暗模式、流畅云母主题、圆角 UI" width="882" height="459" referrerpolicy="no-referrer"></p><p data-vmark="ca01">而到了 Windows 11 的最新版本中，任务管理器的显示界面与整个 Windows 11 主题更加统一，虽然可以看到依然会出现部分老 UI 的弹窗，但整体色彩是一致的。</p><p style="text-align: center;" data-vmark="ebcc"><img src="https://img.ithome.com/newsuploadfiles/2022/4/20c2bf96-6480-4b33-ab51-cb297e45f844.png" w="894" h="495" title="微软 Win11 全新多彩任务管理器曝光：可自定义强调色，黑暗模式、流畅云母主题、圆角 UI" width="894" height="454" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="357c"><img src="https://qimg.ithome.com/newfiles/2022/4/91cb1465-43cb-4471-b2ec-01d368cf69a7.jpg" title="微软 Win11 全新多彩任务管理器曝光：可自定义强调色，黑暗模式、流畅云母主题、圆角 UI" referrerpolicy="no-referrer"></p><p data-vmark="63e0">IT之家了解到，微软计划在<span class="accentTextColor">未来几天内</span>为 Windows Insider 人员推出该设计，并且会在 Windows 11 版本 22H2 发布之前继续开发任务管理器。</p><p data-vmark="9790">微软还在为任务管理器开发一个电池和应用程序健康功能，该功能目前尚未在预览版本中启用。</p>
