            <p data-vmark="6358"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 23 日消息，近期，华为移动服务 HMS Core Windows 版迎来了 6.2.0.301 更新，新增支持 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 系统，支持 Windows 远程登录使用 HMS Core，另外还优化了 HMS Core 的用户体验。</p><p data-vmark="7124"><img src="https://img.ithome.com/newsuploadfiles/2021/11/4ea0f517-bdac-45bc-a7ca-b48724f1accf.jpg@s_2,w_820,h_557" w="895" h="608" title="华为 HMS Core PC 版 6.2.0.301 更新：支持 Win11 系统" srcset="https://img.ithome.com/newsuploadfiles/2021/11/4ea0f517-bdac-45bc-a7ca-b48724f1accf.jpg 2x" width="895" height="557" referrerpolicy="no-referrer"></p><p data-vmark="6a1e">IT之家获悉，在今年 10 月份的华为开发者大会上，华为正式发布全新的 HMS Core 6，向全球开发者开放 7 大领域的 69 个 Kit 和 21738 个 API，包括 13 个跨操作系统能力。</p><p data-vmark="7eb3">HMS Core 6 不仅支持 <a class="s_tag" href="https://hmos.ithome.com/" target="_blank">HarmonyOS</a> 和<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>，还可以支持其它更多操作系统，如 OpenHarmony、Windows、iOS 等，可在平板、智慧屏、智能手表及车机等多设备调用。HMS Core 还支持第三方开发框架，如 React Native、Cordova 等。</p><p data-vmark="0b62"><img src="https://img.ithome.com/newsuploadfiles/2021/10/f2b1743a-0461-4d23-b48c-9714bddcf303.jpg@s_2,w_820,h_273" title="华为 HMS Core PC 版 6.2.0.301 更新：支持 Win11 系统" referrerpolicy="no-referrer"></p><p data-vmark="ba8e">在图形领域，HMS Core 6 新增 3D 建模服务，并增强了 AR Engine、图形计算服务、图形引擎服务等多项开放能力。</p><p data-vmark="f7f2">在媒体领域，基于 AI 技术的加持，华为全新开放了视频编辑服务（Video Editor Kit）和音频编辑服务（Audio Editor Kit），并能提供覆盖 70 + 种语言的全球化服务，支持全球调用。</p><p data-vmark="4dfa">在人工智能领域，Machine Learning Kit 作为已开放的能力，有将近一万个应用集成了 ML Kit，月调用量超过 82.5 亿次，现已扩展到其他操作系统应用上。</p><p data-vmark="c9d5">在应用服务领域，HMS Core 6 新增和增强的开放能力达到 13 项，跨 OS 能力 12 项。</p><p data-vmark="f367">在系统领域，华为将核心的连接与通信能力开放出来，包含 9 项能力，通过 Wireless Kit、Network Kit 开放给应用开发者。</p>
