<p data-vmark="b73c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 15 日消息，据 Windows Latest 报道，作为 2021 年 12 月星期二补丁周期的一部分，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> KB5008215 现在已经推出，它改进了任务栏的动画。</p><p data-vmark="08cf"><img src="https://img.ithome.com/newsuploadfiles/2021/12/d51d5c9a-67f6-4f78-af44-1c96e2c1f24c.jpg" w="768" h="432" alt="搭载 Windows 11 的 Surface Laptop Studio " title="微软 Win11 Build 22000.376（KB5008215）正式发布：修复 NVMe / SSD / HDD 性能变慢、任务栏图标动画等问题" width="768" height="432" referrerpolicy="no-referrer"></p><p data-vmark="8700">随着今天星期二补丁的发布，Windows 11 终于有了一些新的表情符号，与微软的流畅风格兼容。新的表情符号将使操作系统与微软的其他产品，如 Skype、微软 Teams 等保持一致。</p><p data-vmark="d78b">这些表情符号有一个现代的外观，它们与 Windows 11 的整体设计相一致，而且这个系列在整体上是相当不错的。当然，这些是 2D 风格的表情符号，尽管此前微软宣传为 Windows 11 提供 3D 表情符号。不出所料，Windows 11 的新表情符号集还包括 Clippy 作为回形针表情符号的替代品。</p><p data-vmark="52bc">Clippy 是 21 世纪初微软 Office 套件的一部分，它因大众需求而回归。如果你还记得，有相当多的人投票赞成这个标志性助手的表情符号。</p><p data-vmark="88c0">IT之家获悉，Windows 11 KB5008215 还修复了性能问题，包括一个可能降低存储驱动器的写入 / 读取速度的错误。受影响的存储驱动器包括 SSD、HDD 和 NVMe。</p><p data-vmark="eecc">用户检查更新会出现一个新的补丁：</p><blockquote><p data-vmark="cda3">2021-12 Cumulative Update for Windows 11 for x64-based Systems (KB5008215)</p></blockquote><h2 data-vmark="9613">下面是 Windows 11 Build 22000.376（KB5008215） 更新日志</h2><ul class=" list-paddingleft-2"><li><p data-vmark="ed9e">修正了一个可能阻止文件资源管理器和桌面快捷菜单的问题。</p></li><li><p data-vmark="2643">修复了任务栏图标的动画。</p></li><li><p data-vmark="e46a">修复了蓝牙音频问题。</p></li><li><p data-vmark="903f">修复了文件资源管理器的各种问题。</p></li><li><p data-vmark="b05b">修复了一个影响“开始”菜单的性能和可靠性的问题。</p></li></ul><p data-vmark="feb8">Windows 11 有一个错误 Bug，可能会降低固态硬盘和其他驱动器的速度。在某些情况下，Windows 11 的运行速度可能会慢 50% 以上。</p><p data-vmark="3680">自 6 月 24 日 Windows 11 操作系统推出以来，这个问题一直困扰着一些用户，上个月有更多用户提出了这个问题。几个月后，这个问题得到了承认，微软现在推出的是 Windows 11 Build 22000 累积更新补丁。</p><p data-vmark="b508">根据发布说明，Windows 11 Build 22000 修复了一个问题，即当你试图复制和粘贴文本时，会导致 Internet Explorer 停止响应。</p><p data-vmark="72fd">另一个错误 Bug 是，当用户试图使用单次点击打开一个项目时，文件资源管理器和桌面快捷菜单无法使用。</p><p data-vmark="0799">微软还进行了修改，以改善任务栏图标的动画和蓝牙音频设备的音量控制。</p><p data-vmark="66b3">文件资源管理器也得到了一些重要的错误 Bug 修复。例如，微软已经修复了一个问题，该问题导致文件资源管理器在浏览目录后关闭窗口时停止响应。同样，另一个 Bug 也得到了修复，当你试图使用 Directory / Background 为应用程序创建快捷方式时，会影响桌面上下文菜单。</p><p data-vmark="142d">遗憾的是，Windows 11 被强烈要求的任务栏拖放功能并不包括在更新中。然而，微软已经修复了一个问题，即当你在任务栏上的图标上悬停时导致闪烁。</p><p data-vmark="575b"><strong>最新更新中的其他错误修复包括：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="25a7">修复了导致一些视频中的闭合字幕的渲染问题。</p></li><li><p data-vmark="730e">修复了可能影响任务视图、Alt-Tab 或 Snap 辅助的性能的问题。</p></li><li><p data-vmark="0014">修复了当你关闭耳机的存在传感器选项时，可能会导致 Windows 混合现实失败的问题。</p></li><li><p data-vmark="30ec">修复了打印问题。</p></li><li><p data-vmark="aa70">修复了可能导致音频暂时丢失的错误。</p></li><li><p data-vmark="c383">修复了字体渲染的问题。</p></li><li><p data-vmark="3569">修复了可能导致有触摸板的设备上的某些应用程序崩溃或阻塞的问题。</p></li></ul>
