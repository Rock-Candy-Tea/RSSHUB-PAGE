<p data-vmark="55f3"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 11 日消息，据 Windows Latest 报道，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 太阳谷 2 更新预计将对 UI 进行大修，并在 WinUI 设计原则的基础上使界面看起来更加现代化。此前最初的 Windows 11 版本仍保留了自旧 Windows 98 时代以来的旧界面。</p><p style="text-align: center;" data-vmark="1b28"><img src="https://img.ithome.com/newsuploadfiles/2022/3/77951dcd-6eea-4943-8979-9a0799ee608f.jpg" w="696" h="365" title="焕然一新！微软 Win11 22H2“太阳谷 2”重大版本将全面改造升级传统经典 UI" width="696" height="365" referrerpolicy="no-referrer"></p><p data-vmark="b39e">Windows 11 版本 22H2“太阳谷 2”预计将通过 WinUI 大改版刷新很多旧版 UI，并改变 Windows 界面的很多主要元素。所以 2022 年对 Windows 粉丝来说也将是令人感到兴奋的年份。</p><p data-vmark="1286">在 Windows 11 预览版 Build 22572 中，微软正在测试内置打印功能的现代对话框 UI，该功能可通过记事本等本地应用程序启动。当用户使用记事本或写字板打印文档时，会看到基于 WinUI 和流畅设计原则的现代界面，与 Windows 11 的其他 UI 保持一致。</p><p data-vmark="ff62">与 Word 不同，像记事本这样的应用程序依赖于 Windows 内置的“打印”对话框，允许用户使用连接的硬件创建 PDF 或打印文本。</p><p style="text-align: center;" data-vmark="24d5"><img src="https://img.ithome.com/newsuploadfiles/2022/3/9295b075-c039-4aac-bf61-f979136ebf9a.png" w="903" h="618" title="焕然一新！微软 Win11 22H2“太阳谷 2”重大版本将全面改造升级传统经典 UI" width="903" height="561" referrerpolicy="no-referrer"></p><p data-vmark="14cd">正如上面的现代“打印”对话框截图，微软通过使用 WinUI 2.6 已经更新了“打印机”等按钮。然而，这次设计更新删除了一些功能，如“打印机偏好”、“位置”和其他高级选项，这些功能在传统版本中是可用的。</p><p data-vmark="ada9">重新设计的对话框是 Windows 预览计划的一部分。除了现代打印对话框，微软还在测试新版 Windows  <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 子系统和一堆任务栏的改进，包括浮动的系统托盘图标。</p><p data-vmark="b705">这些功能目前只在 Insider 预览计划中提供，但它们预计在今年晚些时候作为太阳谷 2 重大更新的一部分对所有人开放。</p>
