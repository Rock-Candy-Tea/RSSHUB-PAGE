<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 4 日消息 微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 正式版将于 10 月 5 日开始向符合条件的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 个人电脑推出，但对<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>应用的支持仍需一段时间。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/f4891e3b-6180-45ff-beaf-ac9dbbef8a85.jpg@s_2,w_820,h_461" title="微软 Win11 Windows 安卓子系统应用最低系统版本要求变更为 22000.63" referrerpolicy="no-referrer"></p><p>昨日， Windows 安卓子系统（Windows Subsystem for <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a>）已现身微软商店，需要 8GB 内存并推荐 16GB 配置，可在 ARM64 或 x64 处理器上运行。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/228316d9-d0e4-4dc9-8b94-3672e0f6420d.png" w="270" h="270" title="微软 Win11 Windows 安卓子系统应用最低系统版本要求变更为 22000.63" width="270" height="270" referrerpolicy="no-referrer"></p><p>据IT之家网友反馈，Windows 安卓子系统应用最低系统版本要求已<span class="accentTextColor">从 Windows 10 版本 22000.0 变更为 Windows 10 版本 22000.63</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/ac10dd41-3bb1-4a05-a888-f0b75a830786.png" w="1206" h="951" title="微软 Win11 Windows 安卓子系统应用最低系统版本要求变更为 22000.63" width="1206" height="647" referrerpolicy="no-referrer"></p><p>不过，目前打开微软商店提供的 Windows 安卓子系统应用，仍然只能看到一个空白的窗口。Windows 安卓子系统何时上线尚不清楚。</p><p>据悉，微软商店中的 Android 应用将从亚马逊应用商店（Amazon Appstore）下载和安装，这将允许用户在不借助任何第三方软件的情况下运行 Android 应用。同时，用户也可使用从互联网上下载的 APK 文件来侧载 Android 应用。</p><p>此外，今年 6 月，谷歌表示将要求应用开发者使用新的 Android App Bundle（AAB）标准，而不是 Android Application Package （APK）。亚马逊已经确认，其应用商店将支持 Android App Bundle（AAB）。</p>
