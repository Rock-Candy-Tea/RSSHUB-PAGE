            <p data-vmark="5269"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 15 日消息，今日晚间，知乎发布公告，宣布<strong>即将上线显示用户 IP 属地</strong>。</p><p data-vmark="3bcf">知乎表示，近期注意到，在国内外热点事件的讨论中，<strong>出现个别用户冒充当地网友，发布、传播不实信息</strong>，干扰正常讨论秩序的情况。与此同时，众多用户也向其反映，建议知乎公开展示用户 IP 属地信息。</p><p data-vmark="1de4" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/9e106e61-6b7a-4387-9fb0-d631ca891753.png" alt="知乎" title="知乎宣布即将上线显示用户 IP 属地：个别用户冒充国内外热点事件当地网友" referrerpolicy="no-referrer"></p><p data-vmark="d8df">知乎在公告中指出，<strong>社区拟逐步开放“用户 IP 属地”的公开显示</strong>。届时，问答页面将显示用户发布内容时的 IP 属地。据介绍，<strong>接下来相关测试将逐步展开</strong>，用户会感受到如下变化：</p><ul class=" list-paddingleft-2"><li><p data-vmark="89e5">部分问题下将展示回答的评论 IP 属地；</p></li><li><p data-vmark="c02e">目前采用的 IP 对应属地，境内互联网用户账号 IP 地址属地信息标注到省（区、市），境外账号 IP 地址属地信息标注到国家（地区）（ IP 属地以运营商提供信息为准），用户暂时无法主动开启或关闭相关展示。</p></li></ul><p data-vmark="9838" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/64454be8-0427-4a84-a9a0-5d428fec29b9.jpg@s_2,w_820,h_1216" w="1440" h="2136" alt="知乎部分问题下将展示回答的评论 IP 属地" title="知乎宣布即将上线显示用户 IP 属地：个别用户冒充国内外热点事件当地网友" srcset="https://img.ithome.com/newsuploadfiles/2022/4/64454be8-0427-4a84-a9a0-5d428fec29b9.jpg 2x" width="1440" height="1216" referrerpolicy="no-referrer"></p><p data-vmark="cb88">IT之家了解到，今年 3 月，微博宣布测试开放部分国内用户的 IP 归属地展示，测试正常后陆续对其他用户开放相关功能。</p><p data-vmark="f84f">今日早些时候，今日头条、抖音、百度百家号宣布拟在个人主页等位置展示账号 IP 属地。其中，今日头条相关功能即日起在网页版进行测试，抖音相关功能在 4 月底进行测试，百度百家号则将于近期在个人主页中新增“发帖所在地”的展示功能。</p>
