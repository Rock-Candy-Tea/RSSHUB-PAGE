<p data-vmark="835b"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 5 日消息，今日下午，DxOMark 公布了苹果 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> SE 3 的音频得分。<strong>苹果 iPhone SE 3 音频得分为 70</strong>，位列 DXOMARK 中高端机型音频排名第三。</p><p style="text-align: center;" data-vmark="23a8"><img src="https://img.ithome.com/newsuploadfiles/2022/4/2526dd57-3db8-4fcf-875b-f104e6fa0f96.png" w="1200" h="675" alt="苹果iPhone SE 3 DxOMark音频得分" title="70 分，苹果 iPhone SE 3 DxOMark 音频得分出炉" width="1200" height="461" referrerpolicy="no-referrer"></p><p data-vmark="c3e2">IT之家了解到，其中，苹果 <strong>iPhone SE 3 播放得分为 69 分，录制得分为 74 分</strong>。作为苹果推出的全新机型，iPhone SE 3 于 3 月 9 日发布，采用 4.7 英寸屏幕，搭载 A15 仿生芯片，支持 5G，售价 3499 元。</p><p data-vmark="600d">DxOMark 表示，作为苹果发售价格最亲民的机型，iPhone SE 3 无论是在播放还是录音测试的结果中，都几乎没有音损。它在最大音量下的播放效果出色，可以为用户带来良好的游戏和观影体验。</p><p style="text-align: center;" data-vmark="7d65"><img src="https://img.ithome.com/newsuploadfiles/2022/4/b5312cf5-5732-4349-9e92-1b75a1f91597.png" w="1275" h="581" alt="苹果iPhone SE 3 DxOMark音频得分" title="70 分，苹果 iPhone SE 3 DxOMark 音频得分出炉" width="1275" height="374" referrerpolicy="no-referrer"></p><p data-vmark="ce23">此外，DxOMark 称 iPhone SE 3 在录音方面的性能也很突出，在不同使用场景中的测试结果都能保持一致。</p><p style="text-align: center;" data-vmark="63e1"><img src="https://img.ithome.com/newsuploadfiles/2022/4/0e72c286-e95a-4d22-96b0-21b9b19e1f6c.png" w="1349" h="658" alt="苹果iPhone SE 3 DxOMark音频得分排名" title="70 分，苹果 iPhone SE 3 DxOMark 音频得分出炉" width="1349" height="400" referrerpolicy="no-referrer"></p>
