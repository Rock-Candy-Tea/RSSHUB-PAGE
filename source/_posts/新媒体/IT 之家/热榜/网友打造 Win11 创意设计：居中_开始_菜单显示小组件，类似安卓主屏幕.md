<p data-vmark="f532"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 3 日消息 微软此前已经宣布，10 月 5 日起，符合条件的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> PC 将可免费升级到 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>，预装 Windows 11 的各种全新 PC 设备也将开始上市发售。Windows 11 将带来全新的 Windows 体验，旨在让用户与之所爱，更近一步。</p><p data-vmark="8bd0">在 Windows 11 在“开始”菜单中，微软表示将以用户及其内容为中心。凭借云和 Microsoft 365 的强大性能，“开始”菜单会向用户显示其最近在各个设备上查看过的文档。</p><p data-vmark="be6a">不过鉴于微软马上就要发布 Windows 11 正式版，已经不可能改变新的“开始”菜单的工作方式了。话虽如此，到目前为止，新的“开始”菜单还饱受用户争议，原因有很多，包括可定制性有限且难以定制。Windows 11 的小组件系统也显得非常俗套，很像 Windows 10 的新闻和兴趣弹出窗口。</p><p data-vmark="3bbc">IT之家获悉，微软小组件（Widgets）是一项由 AI 驱动的个性化提示功能，它提供了更加快捷的方式来获取你所关心的信息。借助于 Microsoft Edge <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>卓越的性能、速度和生产力，你可以在网页上完成更多的工作。</p><p data-vmark="a372">现在网友 Gab Leo 提出了进一步的设想，如果将小组件整合到 Windows 11 的“开始”菜单中会怎么样？</p><p data-vmark="1c75"><img src="https://img.ithome.com/newsuploadfiles/2021/10/40894302-ab6a-4072-9c23-7bdda570a3d6.png" w="1440" h="810" title="网友打造 Win11 创意设计：居中“开始”菜单显示小组件，类似安卓主屏幕" width="1440" height="461" referrerpolicy="no-referrer"></p><p data-vmark="f829">与目前的系统相比，这个想法有几个优点。第一，它是 Windows 10“开始”菜单的自然过渡，其中包括应用图标和动态磁贴。第二，它对 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 和<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>用户（世界上大多数智能手机用户）来说也非常熟悉，他们也习惯于混合小组件和图标，最后，它将把小组件暴露给更多用户，而不是把它放在单独的任务栏按钮栏里。</p><p data-vmark="9427">另外这个概念还可以拓展，那就是使“开始”菜单支持水平滚动，这将再次使其与数十亿人熟悉的安卓手机主屏幕非常相似。</p><p data-vmark="d798">目前尚不清楚微软下一步是否会继续改革 Windows 11“开始”菜单设计。</p>
