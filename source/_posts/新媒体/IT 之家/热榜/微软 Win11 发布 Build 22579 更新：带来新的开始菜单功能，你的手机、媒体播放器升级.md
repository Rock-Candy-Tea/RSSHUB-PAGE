            <p data-vmark="44dc"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 19 日消息 微软现已发布 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Build 22579 预览版，它带来了一个新的开始菜单功能，可以让你为文件夹自定义一个名称。</p><p data-vmark="3d9a"><img src="https://img.ithome.com/newsuploadfiles/2022/3/823c1563-22cf-42cb-937b-717a67511cdb.gif" w="1011" h="1125" alt="开始时的文件夹。" title="微软 Win11 发布 Build 22579 更新：带来新的开始菜单功能，你的手机、媒体播放器升级" width="1011" height="912" referrerpolicy="no-referrer"></p><p data-vmark="18d6">此外，从现在开始到 3 月 22 日，微软将在开发频道为 Windows Insiders 举办 2022 年三月 Bug 大扫除，详情请查看反馈中心的任务部分，微软一般会在那里发布各种功能通告。</p><p data-vmark="2ca2">今天的 Windows 11 预览版增加了对新的组策略的支持，你现在可以将 USB 驱动器（U 盘）从 BitLocker 加密中排除等等。</p><p data-vmark="6104">当你检查更新时，可能会看到 "Windows 11 Insider Preview 22579 (NI_RELEASE)"，它来自全新的 Nickel 分支。微软表示，新版本中的变化不会与任何特定的版本产生关联，但大多数优化和功能更新将在今年晚些时候作为 22H2 版本或功能体验包发布。</p><p data-vmark="87d3">目前来看，Build 22579 也不是一次特别大的功能更新，主要是基于过去几周内刚出现的功能进行了优化和改进。例如，"开始" 菜单现已支持文件夹 "重命名" ，因此你可以用任意喜欢的名称创建开始菜单文件夹。</p><p data-vmark="e5a9" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/5a22bbbb-771b-46e2-bad5-d15f22e4b969.jpg@s_2,w_820,h_787" w="845" h="811" alt="Windows 11 Build 22579 开始菜单" title="微软 Win11 发布 Build 22579 更新：带来新的开始菜单功能，你的手机、媒体播放器升级" srcset="https://img.ithome.com/newsuploadfiles/2022/3/5a22bbbb-771b-46e2-bad5-d15f22e4b969.jpg 2x" width="845" height="787" referrerpolicy="no-referrer"></p><p data-vmark="eb39">▲ Windows 11 Build 22579 开始菜单</p><p data-vmark="ce35">大家可能知道，微软在 Windows 11 开始菜单砍掉了实时磁贴设计，能够自定义的部分也很少。不过，微软正计划通过 Sun Valley 2 为用户提供一些新的方式来实现个性化菜单，可能包括文件夹的变化和在菜单中实现更多图标或推荐文件的功能。</p><h2 data-vmark="efc7">Windows 11 Build 22579 中的新内容</h2><p data-vmark="f48f">微软正在开发一项新的组策略，旨在帮助用户将 USB 可移动驱动器排除在 BitLocker 加密之外。这个新策略主要针对的是相关专业人士，但实际上任何人都可以用，只是有点麻烦。</p><p data-vmark="6797">顾名思义，新的 BitLocker 策略旨在解决自动或意外加密的问题。</p><p data-vmark="9965">当你启用这一项组策略时，用户将无法对“黑名单”中的存储器进行加密。首先，你需要找到想要排除的设备的硬件 ID，或者在 Intune 中配置 BitLocker 排除列表策略。</p><p data-vmark="bd94">此外，这次更新也有一些设计上的改进。例如微软最近为打印队列体验添加了新的 Fluent 和 WinUI 设计。</p><p data-vmark="284f">今天更新完之后，你会发现微软将 Win32 程序的打印对话框更新成了新的样式，而使用 CPrintDialog 的桌面应用将以现代打印对话框的形式出现在你的面前，更加契合 Windows 11 的外观基调。</p><p data-vmark="7037"><img src="https://img.ithome.com/newsuploadfiles/2022/3/a2604ebe-aa45-449f-ab40-63de011d9003.png" w="903" h="618" alt="Windows 11 用户界面刷新" title="微软 Win11 发布 Build 22579 更新：带来新的开始菜单功能，你的手机、媒体播放器升级" width="903" height="561" referrerpolicy="no-referrer"></p><p data-vmark="53a1">IT之家了解到，微软还对 Windows 11 打印功能进行了改进。如果你有一个新的打印机，微软会自动帮你安装。</p><p data-vmark="7cec">如果你的 Windows Terminal 已经卸载，那么 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 也可以帮你更方便地进行命令行操控。如果任务栏菜单（Win+X）中没有 Windows Terminal，则将自动重定向到 PowerShell。</p><h2 data-vmark="f549">Windows 11 Build 22579 的改进</h2><p data-vmark="9aca">微软正在优化 Win11 的“入门”应用，允许用户将网站固定到任务栏上。</p><p data-vmark="7806"><img
