<p data-vmark="adf7"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 11 日消息 据 MSPoweruser 报道，微软已经发现了一个新问题，可能会给升级到 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的用户带来影响。</p><p data-vmark="4eba"><img src="https://img.ithome.com/newsuploadfiles/2021/10/735098f0-d4bc-4227-bfc1-fe0e11fcdb37.png" w="1440" h="692" title="微软 Win11 正式版发现新问题：不兼容注册表中带有非 ASCII 字符的应用程序" width="1440" height="394" referrerpolicy="no-referrer"></p><p data-vmark="40df">事实证明，Windows 11 并不能很好地兼容注册表中带有非 ASCII 字符的应用程序。</p><p data-vmark="0e8c"><img src="https://img.ithome.com/newsuploadfiles/2021/10/269d198e-b7a2-45d1-b93e-0eb7367e90ec.png" w="1227" h="402" title="微软 Win11 正式版发现新问题：不兼容注册表中带有非 ASCII 字符的应用程序" width="1227" height="269" referrerpolicy="no-referrer"></p><p data-vmark="523d">IT之家获悉，微软已经发现在注册表键或子键中使用一些非 ASCII 字符的应用程序与 Windows 11 之间存在兼容性问题。受影响的应用程序可能无法打开，并可能导致 Windows 其他问题或错误 Bug，包括可能的蓝屏死机。带有非 ASCII 字符的受影响的注册表键可能无法被修复。</p><p data-vmark="099d">非 ASCII 字符通常用于中文或泰米尔语等非罗马字母。</p><p data-vmark="773b">由于这个问题，如果用户的注册表里有使用非 ASCII 字符的应用程序，微软将不会向你推送升级 Windows 11 。微软还建议不要尝试使用“立即更新”按钮或 Media Creation Tool 工具强制升级 Windows 11。</p><p data-vmark="99b2">微软将继续调查这个问题，并在恰当的时候提供解决方案。</p>
