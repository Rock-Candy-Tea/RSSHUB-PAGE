data-vmark="2432"><img src="https://img.ithome.com/newsuploadfiles/2021/11/a4b1b7cb-1f85-4a0a-90f9-29dea8209197.png" w="1440" h="1063" title="微软 Win11 安卓子系统 1.8.32828 更新" width="1440" height="605" 1.8.32828 更新" width="1440" height="637" referrerpolicy="no-referrer"></p><p data-vmark="4992">在上一个版本的更新中，Win11 安卓子系统迎来了全屏壁纸填充功能，本次更新版本号提升较小，可能只是修复 Bug，官方未列出具体更新内容，IT之家网友体验后发现有哪些改变可以投稿告诉我们。</p><p data-vmark="554d"><img src="https://img.ithome.com/newsuploadfiles/2021/11/3c07b48a-324d-422f-80e8-52d5ff79ac85.jpg@s_2,w_820,h_547" title="微软 Win11 安卓子系统 1.8.32828 更新" referrerpolicy="no-referrer"></p><p data-vmark="e4f2">IT之家获悉，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 安卓子系统 (Windows Subsystem for <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a>) 包括 Linux 内核和基于 Android 开源项目（AOSP）版本 11 的 Android 操作系统。该子系统在 Hyper-V 虚拟机中运行，就像 Linux 子系统一样，可以将 AOSP 环境中 App 的运行时和 API 映射到 Windows 图形层、内存缓冲区、输入模式、物理和虚拟设备以及传感器，可以在英特尔、AMD、高通的 CPU 上运行。</p><p data-vmark="4c72">Windows 11 安卓子系统支持以下新特性：</p><ul class=" list-paddingleft-2"><li><p data-vmark="53da">用户可以将安卓 App 固定到开始菜单或任务栏，并通过鼠标、触摸或笔输入与它们交互；</p></li><li><p data-vmark="ab73">安卓 App 还可以集成到 Alt + Tab 选项卡和任务视图中，帮助用户在 App 之间快速切换；</p></li><li><p data-vmark="e8b3">用户可以在操作中心中查看来自安卓 App 的通知，或在 Windows 应用程序和安卓 App 之间共享剪贴板；</p></li><li><p data-vmark="b211">微软还添加了无障碍体验，许多 Windows 辅助功能设置都适用于安卓 App。</p></li></ul><p data-vmark="eb29">安装 Windows 11 安卓子系统 (Windows Subsystem for Android)：</p><p data-vmark="71d3"><span
