<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 31 日消息 微软现宣布 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 正式版将于 2021 年 10 月 5 日开始为符合条件的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 个人电脑推出免费升级服务（具体时间将依据不同设备而有所不同），预装 Windows 11 的 PC 也将开售。</p><p>微软强调，<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a> App 支持将在几个月之后的预览版之中到来，具体信息则未透露。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/78dc2644-b02c-492a-aa2d-887655f6584c.png" w="568" h="796" title="微软：Win11 正式版将于 10 月 5 日到来，安卓 App 支持还要等几个月" width="568" height="796" referrerpolicy="no-referrer"></p><p>IT之家了解到，微软此次免费升级将于 10 月 5 日开始，他们预计到 2022 年年中可使所有符合条件的设备免费升级到 Windows 11，现阶段将通过分阶段和可评估的形式推出 Windows 11 免费升级活动。</p><p>如果您拥有符合升级条件的 Windows 10 电脑，Windows 更新会在那天到来之时通知您。您现在可以通过“设置”>“Windows 更新”>“检查更新”来检查 Windows 11 兼容性。</p><p>当然，届时你也可以选择购买全新的 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> PC ，可能微软也更希望你这么干。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/c36f04dc-b5a0-469d-a498-f816810ee8e9.jpg@s_2,w_820,h_512" w="1440" h="900" title="微软：Win11 正式版将于 10 月 5 日到来，安卓 App 支持还要等几个月" srcset="https://img.ithome.com/newsuploadfiles/2021/8/c36f04dc-b5a0-469d-a498-f816810ee8e9.jpg 2x" width="1440" height="512" 强大的性能，“开始”菜单会向用户显示其最近在各个设备上查看过的文档。</p></li><li><p>贴靠布局、贴靠群组和虚拟桌面功能为您提供了更加有力的支持，帮助您同时处理多个任务及优化屏幕空间。</p></li><li><p>基于 Microsoft Teams 技术的聊天（Chat） 被整合到了任务栏中，可以让您更快捷地与您所关心的人保持联系。</p></li><li><p>小组件（Widgets）是一项由 AI 驱动的个性化提示功能，它提供了更加快捷的方式来获取您所关心的信息。借助于 Microsoft Edge <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>卓越的性能、速度和生产力，您可以在网页上完成更多的工作。</p></li><li><p>Windows 11 提供了有史以来最好的 Windows PC 游戏体验，凭借 DirectX12 Ultimate、直通存储（DirectStorage）以及 Auto HDR 等技术，它能够充分发挥系统硬件的潜力。通过 Xbox Game Pass for PC  或 Xbox Game Pass Ultimate，玩家仅需支付很低的订阅费，即可在 Windows 11 上畅玩超过百款优秀的 PC 游戏（Xbox Game Pass 单卖）。</p></li><li><p>Windows 11 带来了全新的 Microsoft Store，它采用全新的设计，让您在这个值得信任的平台上可以更轻松地搜索和发现喜爱的应用、游戏、节目和电影。我们期待通过与亚马逊和英特尔的合作，继续将 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 应用程序带到 Windows 11 和 Microsoft Store，在随后的几个月中，该功能将首先面向 Windows 预览体验成员推出预览版本。</p></li><li><p>Windows 11 是极具包容性的 Windows 版本，身有残障的工程师也参与到了其开发过程当中，他们专门为残障人士进一步改进了辅助功能。</p></li><li><p>Windows 11 为开发者和创作者解锁新的机会。我们持续扩展 Microsoft Store 的开放性，让越来越多的开发者和独立软件供应商（ISV）能够将他们的应用程序带到 Microsoft Store，我们通过新的开发者工具改进本地和网页应用程序的开发，并让您更轻松地刷新对于我们所有应用程序设计和体验的外观和感受。</p></li><li><p>Windows 11 对速度、效率进行了优化，并改进了触控、触控笔和语音输入体验。</p></li><li><p>Windows 11 是为混合办公而生的操作系统，带来适合您办公方式的新体验；它从设计源头确保了安全性，并拥有既简易又熟悉的 IT 部署和管理。今日起，企业用户还可以在 Azure 虚拟桌面中提前测试 Windows 11 预览版，或在新的 Windows 365 中完整体验 Windows 11 的使用感受。</p></li></ul><p>硬件要求相信大家都已经了解过了，下面简单列一下微软给出的 Win11 支持的 CPU 名单：</p><p>Windows 11 <a DirectX 12+WDDM 2.0 要求 ，而 TPM 2.0 则需要自行查看是否支持（曾因我国法律原因被罚的戴尔和惠普有部分型号不支持，其他大都支持）。</p><p>《<a href="https://www.ithome.com/0/559/673.htm" target="_blank">中国可能是例外：虽然微软强烈要求 TPM 2.0，但 OEM 仍可推出无 TPM 的 Win11
