<p data-vmark="8575"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 11 日消息，现有网友发现，微软新版 <a href="https://apps.microsoft.com/store/detail/outlook-for-windows/9NRX63209R7B" target="_blank">Outlook for Windows</a> 已经上架了 Windows 应用商店，开发者可进行尝鲜。新版本 Outlook 应用适配了 Windows 11 的 fluent Design，支持 Mica 和 Acrylic 材质的透明 / 半透明效果。</p><p data-vmark="6d85" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/72802281-bc49-4258-bde7-e6cf4b14e63a.png" w="963" h="712" title="微软全新 Outlook for Windows 上架 Win11 商店，现已开启开发者预览" width="963" height="606" referrerpolicy="no-referrer"></p><p data-vmark="42ec" style="text-align: left;">除了新的外观设计之外，微软还为此应用添加了“我的一天”等功能，让您可以在工作的同时帮助您与重要话题保持联系。此外，您还可以轻松地将消息作为任务拖放到待办事项中，或者更好地管理日历中的任务。</p><p data-vmark="8775" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/94baa991-fcc7-48eb-8355-f803cac0fda6.jpg@s_2,w_820,h_487" w="1142" h="678" alt="产品图片" title="微软全新 Outlook for Windows 上架 Win11 商店，现已开启开发者预览" srcset="https://img.ithome.com/newsuploadfiles/2022/6/94baa991-fcc7-48eb-8355-f803cac0fda6.jpg 2x" width="1142" height="487" referrerpolicy="no-referrer"></p><p data-vmark="35f1" style="text-align: left;">IT之家曾报道，该项目被称为“Project Monarch”，这个新 Outlook 已经开发了一年多。这是对 Web 应用的重新启动，其目的是统一所有现有的 Windows 电子邮件客户端，如《邮件和日历》以及桌面版 Outlook。</p><p data-vmark="3aad" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/e3af5977-a911-47d0-92bd-d3bc61a66214.jpg" w="696" h="388" title="微软全新 Outlook for Windows 上架 Win11 商店，现已开启开发者预览" width="696" height="388" referrerpolicy="no-referrer"></p><p data-vmark="cd49">《<a href="https://www.ithome.com/0/621/101.htm" target="_blank">微软全新 Win11 Outlook 电子邮件应用抢先看：采用 Mica 设计，快速整理好“我的一天”</a>》</p>
