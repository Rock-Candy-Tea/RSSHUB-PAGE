<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 26 日消息 外媒 MacRumors 报道，据爆料人士 Jon Prosser 称，苹果目前正在测试新的 Face ID 硬件，可以让用户在戴着面具口罩或雾状眼镜时解锁他们的 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/f88d3b3f-3a16-445b-b365-3801a1513c0f.jpg@s_2,w_820,h_461" w="900" h="506" title="苹果 iPhone 13/Pro 系列 Face ID 重磅升级曝光：刘海凹槽减小，戴面具口罩/雾状眼镜也能解锁" srcset="https://img.ithome.com/newsuploadfiles/2021/8/f88d3b3f-3a16-445b-b365-3801a1513c0f.jpg 2x" width="900" height="461" referrerpolicy="no-referrer"></p><p>▲ 升级后的 Face ID 原型渲染图</p><p>Prosser 解释说，苹果正在使用一款紧贴 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 12</a> 的外壳测试更先进的 Face ID 硬件。该保护套允许 iPhone 绕过其内置的 Face ID 系统，而使用该保护套的系统。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/98c97d5b-0307-4f3f-ae76-eab7f72038c6.png" w="757" h="829" title="苹果 iPhone 13/Pro 系列 Face ID 重磅升级曝光：刘海凹槽减小，戴面具口罩/雾状眼镜也能解锁" width="757" height="829" referrerpolicy="no-referrer"></p><p>Prosser 认为，Face ID 原型阵列与今年早些时候 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 13</a> 系列的 CAD 文件中描述的布局完全一致。这是因为外壳中的 Face ID 原型硬件比 iPhone 12 的硬件窄得多，这与 iPhone 13 机型上预计的更细的凹槽相一致。</p><p>由于正在进行大规模测试以收集尽可能多的数据，显然需要外部 Face ID 硬件。据称，这使得苹果可以在不分发未发布的 iPhone 的情况下，给大量员工提供新硬件。</p><p>参与的员工被要求在一系列条件下佩戴面罩和眼镜的组合来测试新硬件。员工只是被要求进行测试，但没有被告知案例中的 Face ID 传感器阵列有什么具体变化。</p><p>IT之家获悉，据报道，新硬件允许 iPhone 在戴着面具、不借助 Apple Watch 和戴着有雾的眼镜时用 Face ID 解锁。</p><p>一种可能性是，苹果正在专门为 Face ID 重新训练其神经网络，以根据人体脸部哪些部分没有暴露来识别。鉴于 Face ID 作为生物识别安全替代密码的重要性，这项努力可能是一项特别具有挑战性的工作。</p><p>虽然 Prosser 强调了该手机壳的 Face ID 硬件与即将推出的 iPhone 13 上预期的更小的凹槽有惊人的相似之处，但目前还不清楚这种改进的 Face ID 是否会随着 iPhone 13 或未来的 iPhone 机型而出现。</p><p>这些测试显然是最近进行的，因此 Prosser 表示，如果改进后的 Face ID 不能立即与 iPhone 13 系列一起使用，它可能会在以后的软件更新中推出。</p><p>Prosser 选择不分享所谓的原型机的图片，因为在上面发现了识别标记，而是分享了一些基于图片的渲染图。</p><p>COVID-19 疫情使得戴口罩成为全球欢迎的做法，苹果之前已经设计了一些功能来缓解 Face ID 无法识别口罩面部的问题。如果带有密码的 Apple Watch 检测到在手腕上的存在，戴 Apple Watch 的 iPhone 用户可以选择绕过 Face ID。然而，并不是每个 iPhone 用户都拥有或者戴着 Apple Watch，所以重新设计 Face ID 如何为新的蒙面世界工作，对全世界的用户来说都是一种胜利。</p>
