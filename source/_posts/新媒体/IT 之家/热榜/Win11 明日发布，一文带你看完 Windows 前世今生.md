<p data-vmark="e885"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 4 日消息 几个月前，微软正式宣布 <span class="accentTextColor">Win 11 的发售时间是 2021 年 10 月 5 日</span>，也就是明天。</p><p data-vmark="ecac">本次升级，微软承诺会<span class="accentTextColor">为符合硬件条件的 Win 10 正版用户提供免费升级</span>。</p><p data-vmark="cf3d"><img src="https://img.ithome.com/newsuploadfiles/2021/10/4e0a1744-9930-4537-a116-5c0215438ae5.jpg" w="553" h="592" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="553" height="592" referrerpolicy="no-referrer"></p><p data-vmark="6e2e">不过，<span class="accentTextColor">对于万众瞩目的<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a> App 支持功能，微软官方表示还需要等几个月才能提供</span>。</p><p data-vmark="ab0a">纵观 Windows 三十多年的发展历程，从一个黑白界面演化成现在这样一个功能丰富，操作形式多样的系统，令我们不得不好奇，Windows 究竟经历了怎样的演变？</p><p data-vmark="2e36">现在，让IT之家带你来看看 Windows 系统的前世今生。</p><h2 data-vmark="e690">Windows 1.0</h2><p data-vmark="099f">Windows 1.0 是微软在 1985 年年底发布的，售价 99.9 美元。这是微软发布的第一个 Windows 操作系统版本。</p><p data-vmark="bf55"><img src="https://img.ithome.com/newsuploadfiles/2021/10/5d35c33c-76e7-459e-8d43-b8bdcd600f29.png" w="256" h="192" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="256" height="192" referrerpolicy="no-referrer"></p><p data-vmark="bcc9">严格上讲，Windows 1.0 并不能算作是一款“操作系统”。因为 <span class="accentTextColor">Windwos 1.0 的运行必须依托 MS-DOS 操作系统</span>。因此 Windows 1.0 更像是一款软件。</p><p data-vmark="6826"><img src="https://img.ithome.com/newsuploadfiles/2021/10/6ccaca64-fdbc-411c-a1f4-d6d668b83ed0.png" w="271" h="167" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="271" height="167" referrerpolicy="no-referrer"></p><p data-vmark="df6d">Windows 1.0 提供的功能也比较少，自带了一些简单的应用程序，包括日历、记事本、计算器等等。无法安装程序，像是一个 PDA 系统。</p><p data-vmark="fba5"><img src="https://img.ithome.com/newsuploadfiles/2021/10/d1d80dfd-09cf-45c9-bbdf-813844ee0564.png" w="553" h="323" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="553" height="323" referrerpolicy="no-referrer"></p><p data-vmark="99b4">当然，Windows 1.0 也有其先进之处。在 Windows 1.0 中，微软第一次对个人电脑操作平台进行用户图形界面进行尝试，<span class="accentTextColor">多软件并行在当时也是跨时代的功能</span>。同时也是当时少有的支持鼠标的“操作系统”。控制面板、驱动程序、虚拟内存等概念也是从 Windwos 1.0 开始的。</p><h2 data-vmark="38ba">Windows 2.0</h2><p data-vmark="2cc2">Windows 2.0 发布于 1987 年年底，售价同样是 99.9 美元，也依然是依托于 MS-DOS 操作系统才能运行。</p><p data-vmark="c2a5"><img src="https://img.ithome.com/newsuploadfiles/2021/10/52626393-9719-40ce-8e41-60a53b085ff5.png" w="553" h="301" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="553" height="301" referrerpolicy="no-referrer"></p><p data-vmark="f7ba">IT之家注意到，相较于 1.0 版本，Windows 2.0 的进步并不算大，仅仅是增加了更大的内存支持，并增强了图形功能，支持了更多的色彩。用户可以叠加窗口，控制屏幕布局，可以用组合键快速使用 Windows 的功能。</p><p data-vmark="05cb">另外，<span class="accentTextColor">现在我们耳熟能详的 Word 和 Excel</span>，就是从这一代 Windows 开始推出的。</p><p data-vmark="97e8">不过，Windows 2.0 的推出，激怒了苹果的乔布斯。乔布斯认为比尔盖茨剽窃了他的创意，把微软告上法庭。</p><p data-vmark="c442"><img src="https://img.ithome.com/newsuploadfiles/2021/10/e98e90b6-5871-4b48-b614-10f079944bdf.png" w="552" h="388" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="552" height="388" referrerpolicy="no-referrer"></p><p data-vmark="cdf3">微软则表示早先他们曾和 Apple 有过一个协议，该协议让微软有权使用 Apple 界面特征。4 年之后，微软赢了官司。</p><p data-vmark="7c28">实际上，不管是苹果还是微软，都是抄袭了当时的美国电脑巨头施乐推出的世界第一款图形化界面系统 Xerox 8010 Star。</p><p data-vmark="e37c"><img src="https://img.ithome.com/newsuploadfiles/2021/10/0349746b-5641-44b7-9935-1a49222fff32.png" w="492" h="389" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="492" height="389" referrerpolicy="no-referrer"></p><p data-vmark="3db3">▲ 施乐 Xerox 8010 Star</p><p data-vmark="6b30">而在乔布斯与比尔盖茨争执的时候，比尔盖茨就是拿着施乐怼乔布斯：“我们都有个有钱的邻居，叫施乐，我闯进他们家准备偷电视机的时候，发现你已经把它盗走了。”</p><p data-vmark="7b3f">书归正题。</p><p data-vmark="1564">不管是 Windows 1.0 还是 Windows 2.0，都没有获得用户认同，销量惨淡。微软真正的翻身仗，是 Windows 3.0 的发布。</p><h2 data-vmark="ed7d">Windows 3.0</h2><p data-vmark="5589">1990 年 5 月 22 日，微软发布了 Windows 3.0。</p><p data-vmark="a798">由于提供了多媒体，网络等众多最先进技术的支持，从而被称为软件技术的一场革命，Windows 3.0 也一度大卖，两个月的销售量就超过了一百万份。</p><p data-vmark="7025"><img src="https://img.ithome.com/newsuploadfiles/2021/10/b9de5517-6dc1-49c8-9a42-e119711b213d.png" w="514" h="330" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="514" height="330" referrerpolicy="no-referrer"></p><p data-vmark="3483">在这一版 Windows 中，微软开始提供 16 位色的支持，显示效果大有改观。另外，IT之家注意到，在随后的 3.1 版本中，微软添加了 CD 播放器和多媒体管理功能，随后又在 3.2 版本中增加了以太网连接和工作组支持。</p><p data-vmark="cc05"><img src="https://img.ithome.com/newsuploadfiles/2021/10/d18f53bf-e98a-43fc-9b6b-78576a238337.png" w="553" h="418" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="553" height="418" referrerpolicy="no-referrer"></p><p data-vmark="137e">另外，微软为 Windows 3.2 增加了简体中文支持，使其成为第一款支持简体中文的 Windows 系统。中国第一批微软用户也是从 Windows 3.2 开始了解微软的。</p><p data-vmark="207d"><img src="https://img.ithome.com/newsuploadfiles/2021/10/c9cbff82-ad11-40d5-87fe-e8d953e271d5.png" w="306" h="367" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="306" height="367" referrerpolicy="no-referrer"></p><h2 data-vmark="1329">Windows 95</h2><p data-vmark="572e">作为 Windows 3.2 的继任者，Windows 95 发布于 1995 年 8 月 24 日，是一个混合的 16 位 / 32 位的 Windows 操作系统。</p><p data-vmark="a418"><img src="https://img.ithome.com/newsuploadfiles/2021/10/4f23f528-a2c1-46fb-a6f5-b39cf4a3308b.png" w="400" h="300" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="400" height="300" referrerpolicy="no-referrer"></p><p data-vmark="00c5">在 Windows 95 中，<span class="accentTextColor">微软第一次抛弃了对 MS-DOS 的依赖</span>，用户不必向之前那样先安装 MS-DOS 再安装 Windows 了。</p><p data-vmark="2199">这一代 Windows 增加了海量的自带软件，例如画图、电话拔号程序、磁盘碎片整理程序、磁盘扫描程序、磁盘空间管理、软件备份等等。</p><p data-vmark="5d20">另外 UI 界面也做了较大程度的调整，看起来更加美观和人性化。</p><p data-vmark="bb39"><img src="https://img.ithome.com/newsuploadfiles/2021/10/cda50d84-355e-4ed1-aa99-8e1dc03f73a2.png" w="399" h="300" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="399" height="300" referrerpolicy="no-referrer"></p><p data-vmark="1936">Windows 95 的推出彻底引爆了市场，消费者排队购买 Windows 光盘，其盛况不亚于现在的抢鞋。</p><p data-vmark="b6b6"><img src="https://img.ithome.com/newsuploadfiles/2021/10/a09e55f9-3fe7-4f01-bc4a-f383eace73fe.png" w="400" h="300" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="400" height="300" referrerpolicy="no-referrer"></p><p data-vmark="b236">当时有评论家认为，通过 Windows 95，微软已经彻底终结了操作系统之战，将竞争对手 macOS、Solaris 、OpenBSD 打的满地找牙。<span class="accentTextColor">Windows 成为了这个市场唯一的霸主</span>。</p><h2 data-vmark="1a0d">Windows 98</h2><p data-vmark="5c9c">作为 Windows 95 的改进版，Windows 98 于 1998 年 6 月份发布。</p><p data-vmark="25eb"><img src="https://img.ithome.com/newsuploadfiles/2021/10/4de11b73-4d86-4ef0-8425-0d01ba193851.png" w="500" h="374" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="500" height="374" referrerpolicy="no-referrer"></p><p data-vmark="92ea">这一版 Windows 增加了电源管理和多个功能总线，例如 USB 总线和 AGP 显卡总线。它首次发布了 DirectX，将混乱的 3D 图形 API 接口进行统筹规划。现在 DirectX 已经是游戏党必备的接口程序了。</p><p data-vmark="6237"><img src="https://img.ithome.com/newsuploadfiles/2021/10/4bcffb01-9d59-48b7-9e44-fd7b6800d352.png" w="487" h="190" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="487" height="190" referrerpolicy="no-referrer"></p><p data-vmark="d364">同时，微软为 Windows 98 捆绑搭售了 IE4 <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>，随着 Windows 市场占有率增加，IE4 也成为了当时最流行的浏览器。</p><p data-vmark="db06">而正是因为 IE，做浏览器的网景公司和微软打了一场世纪大战，一度让微软陷入被拆分的窘境。</p><p data-vmark="afdd"><img src="https://img.ithome.com/newsuploadfiles/2021/10/6279621b-fccc-453c-b5b4-7e5183bba140.jpg@s_2,w_820,h_222" w="1080" h="293" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" srcset="https://img.ithome.com/newsuploadfiles/2021/10/6279621b-fccc-453c-b5b4-7e5183bba140.jpg 2x" width="1080" height="222" referrerpolicy="no-referrer"></p><p data-vmark="f7f4">1994 年，<span class="accentTextColor">网景公司发布了导航者浏览器（Netscape Navigator）</span>，作为世界上第一款民用浏览器，迅速占领了 90% 以上的市场份额。</p><p data-vmark="4299"><img src="https://img.ithome.com/newsuploadfiles/2021/10/ccb6a04d-ca7a-4fd5-a98f-c3e26070706d.jpg" w="447" h="288" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="447" height="288" referrerpolicy="no-referrer"></p><p data-vmark="1dc3">而微软看到网景的成功，迅速推出 IE 浏览器抢夺市场。通过捆绑 Windows 的策略，IE 浏览器大获成功，将网景的浏览器逼上死路。</p><p data-vmark="405b">网景公司不甘示弱，将微软告上法庭，声称其通过垄断地位打压其他公司。</p><p data-vmark="f10f"><img src="https://img.ithome.com/newsuploadfiles/2021/10/e645945d-4ac8-4bda-a496-999b36b63444.png" w="874" h="527" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="874" height="494" referrerpolicy="no-referrer"></p><p data-vmark="985c">▲ 网景公司创始人马克・安德森</p><p data-vmark="62e4">而其他被微软“霸王条款”欺压的多家公司纷纷支援网景，比如康柏等电脑制造商就向司法部提交了微软强制要求捆绑 IE 浏览器的证据。</p><p data-vmark="96fc">最终，经过三年审理，美国司法部在 2000 年宣布微软违反《反垄断法》，<span class="accentTextColor">强制拆分微软为两家公司，并且十年内不得合并</span>。</p><p data-vmark="de19"><img src="https://img.ithome.com/newsuploadfiles/2021/10/9e742fba-4757-49d0-a9bb-d426177096db.png" w="672" h="355" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="672" height="355" referrerpolicy="no-referrer"></p><p data-vmark="9cf2">而微软并不甘心就这么被拆分了，他们向总统候选人小布什捐献了 460 万美元的政治献金，成为小布什最大金主之一。在小布什当上总统后，微软向法院上诉，并在六个月后与美国司法部达成协议，支付 18 亿美元和解费，以换取美国政府的“既往不咎”。</p><p data-vmark="fa38"><img src="https://img.ithome.com/newsuploadfiles/2021/10/92d0eecc-1a78-4d43-802e-8bdc7dbb6574.png" w="493" h="273" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="493" height="273" referrerpolicy="no-referrer"></p><p data-vmark="f1b2">▲ 比尔盖茨和小布什一起看球</p><p data-vmark="3997">在微软解除了拆分危机的 2001 年，影响一代人的 Windows XP 发布了。</p><h2 data-vmark="51c5">Windows XP</h2><p data-vmark="d954">Windows XP 的中文全称为“视窗操作系统体验版”，字母 XP 代表英文单词“体验”（experience）。</p><p data-vmark="3e0f"><img src="https://img.ithome.com/newsuploadfiles/2021/10/27b663b8-6825-4109-9b23-dcbf41b1a596.jpg" w="472" h="296" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="472" height="296" referrerpolicy="no-referrer"></p><p data-vmark="9ff3">从 Windows XP 开始，微软会为同一款系统准备多个版本发布，XP 就有家庭版（Home）和专业版（Professional）两个版本，后者具备更强的专业功能，例如增加了对双处理器的支持。</p><p data-vmark="80f7">Windows XP 的外观和体验发生的天翻地覆的变化。绿色的开始按钮、蓝色的任务栏，以及蓝天白云大草原的桌面，是一代人对 XP 的记忆。</p><p data-vmark="81ed"><img src="https://img.ithome.com/newsuploadfiles/2021/10/9eb58444-ccae-4e13-b469-fc51eb72b31d.png" w="553" h="415" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="553" height="415" referrerpolicy="no-referrer"></p><p data-vmark="5469">由于微软后续的 Windows Vista 操作系统过于拉垮（性能要求高，操作体验差），消费者纷纷把系统回退到 XP，<span class="accentTextColor">微软迫不得已给 XP 这位老将续命十三年</span>，直到 2014 年才停止技术支持。这也让 XP 成为了史上最长寿系统。</p><p data-vmark="cfd9">IT之家注意到，<span class="accentTextColor">时至今日，依然有很多电脑或工控机运行在 XP 系统上</span>，例如超市收银机和银行提款机。</p><h2 data-vmark="34ce">Windows 7</h2><p data-vmark="e027"><img src="https://img.ithome.com/newsuploadfiles/2021/10/9e9d064e-2f8c-4e20-9ad0-02246ec3b05b.png" w="554" h="344" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="554" height="344" referrerpolicy="no-referrer"></p><p data-vmark="2be7"><a class="s_tag" href="https://www.win7china.com/" target="_blank">Windows 7</a> 发布于 2009 年 10 月 22 日，其精美的 Aero 效果让人以耳目一新。Windows 7 也是微软第一款支持触控操作的系统，不过在当时配备触控屏幕的电脑并不多，因此并没多少人使用这个功能。</p><p data-vmark="2724">相较于中道崩殂的 Vista，Windows 7 的资源消耗相当低，对配置要求不高。而其运行效率却比 Vista 更快，配合固态硬盘可以实现七秒开机，笔记本的电池续航能力也得到了大幅增加。</p><p data-vmark="9cfa">根据微软官方的调查报告，Windows 7 面世不到两年的时间，<span class="accentTextColor">就成为了最主流的 Windows 操作系统</span>，<span class="accentTextColor">消费者满意度高达 95%</span>。</p><h2 data-vmark="1f61">Windows 8</h2><p data-vmark="0bb1">2012 年 10 月 26 日，微软正式推出 Windows 8。</p><p data-vmark="1dfa"><img src="https://img.ithome.com/newsuploadfiles/2021/10/93b34173-9fc5-4ba4-afdc-3be5fd942225.png" w="531" h="298" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="531" height="298" referrerpolicy="no-referrer"></p><p data-vmark="e5f0">Windows 8 是微软第一款支持 ARM 架构的操作系统，同时为了应对 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 等平板电脑的挑战，微软为 Windows 8 进行了大刀阔斧的改革，界面采用方块化布局，将其命名为 Metro 风格。</p><p data-vmark="b808"><img src="https://img.ithome.com/newsuploadfiles/2021/10/e757ea1d-02e8-4938-9c74-09522ea9e885.png" w="553" h="311" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="553" height="311" referrerpolicy="no-referrer"></p><p data-vmark="61d9">同时微软为触控功能增加了大量优化，以至于使用逻辑都发生了较大变化，例如取消开始菜单按钮，取而代之的是用手或鼠标下滑来弹出开始菜单。</p><p data-vmark="1ec3"><span class="accentTextColor">然而操作逻辑的巨大改变引发了灾难性的后果</span>，大量用户不知道如何使用 Windows 8，甚至连关机都做不到，因为找不到开始按钮。</p><p data-vmark="a261"><img referrerpolicy="no-referrer"></p><p data-vmark="1f65">迫不得已，<span class="accentTextColor">微软推出了 Win 8.1，恢复了开始菜单按钮</span>，并调整了一些使用逻辑。</p><p data-vmark="34f9">即使微软努力迎合大众需求，这一代 Windows 依然<span class="accentTextColor">口碑崩坏</span>，大量用户选择退回 Windows 7。</p><h2 data-vmark="165d">Win 10</h2><p data-vmark="5c3c">2015 年 7 月 29 日，微软正式发布 Win 10 系统。</p><p data-vmark="83b0">早在系统发布之初，就有网友按照之前的命名习惯，猜测新系统会命名为 Windows 9。还有网友建了 Windows 9 贴吧。</p><p data-vmark="532b"><img src="https://img.ithome.com/newsuploadfiles/2021/10/6ba11110-6c13-45b4-8475-6f45ba2af473.png" w="402" h="598" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="402" height="598" referrerpolicy="no-referrer"></p><p data-vmark="0a38">微软之所以要跳过“9”直接发布 Win 10，部分网友认为是因为 Windows 95 和 Windows 98 的系统代码简称是 Win 9 ，采用 Windows 9 命名可能会造成程序运行出错，因此直接跳过。</p><p data-vmark="8575"><img src="https://mmbiz.qpic.cn/mmbiz_png/YTI67xfsN52PZ0tAicq6og0TvYICTxszWaGCro1LmeibcXpxN1McibcAEPG8QSV2GibJG1pia3ItxIVmjGS1X5FIGCg/640?wx_fmt=png&tp=webp&wxfrom=5&wx_lazy=1&wx_co=1" alt="图片" title="Win11 明日发布，一文带你看完 Windows 前世今生" referrerpolicy="no-referrer"></p><p data-vmark="e957">另外一种说法是，<span class="accentTextColor"><a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 是微软第十款操作系统</span>，所以得此名字。</p><p data-vmark="ab26"><img src="https://img.ithome.com/newsuploadfiles/2021/10/e21bb7da-2d00-4a73-9770-aaf56f7ae7ab.jpg" w="319" h="421" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="319" height="421" referrerpolicy="no-referrer"></p><p data-vmark="cd33">不过微软官方也曾经对 Win 10 命名有过表态。</p><p data-vmark="05b8">微软发言人表示：</p><blockquote><p data-vmark="24ef">Windows 10 代表着 Windows 转向新的做事方式，这不仅仅是名称上的变化，而是将被下一个 10 亿用户使用的新 Windows。</p></blockquote><p data-vmark="ed81">不仅命名上有变化，微软在操作系统的盈利渠道方面，也正在改变。</p><p data-vmark="4522">Win 10 推出时，微软就表示要向 Windows 7 和 Windows 8 的正版用户免费推送 Win 10。用户再也不用向之前那样购买新系统的激活码了。</p><p data-vmark="4c3d">微软也不再依赖销售 Windows 系统来赚钱，而是以配套服务作为盈利重点，例如 Office 365 和 OneDrive 网络存储等服务。</p><p data-vmark="30e9"><img src="https://img.ithome.com/newsuploadfiles/2021/10/402ec136-e619-434a-a322-84899e566ef5.jpg@s_2,w_820,h_512" w="825" h="515" title="Win11 明日发布，一文带你看完 Windows 前世今生" data-vmark="0511">Win 10 一个巨大的提升就是网络支援。任何一台能够联网的电脑，安装 Win 10 后会自动连接微软服务器来添加驱动，免去了过往手动安装驱动的麻烦。</p><p data-vmark="3504"><img src="https://img.ithome.com/newsuploadfiles/2021/10/d7b72bac-b101-4d0c-b6f1-b86fcaa71674.png" w="553" h="311" title="Win11 明日发布，一文带你看完 Windows 前世今生" width="553" height="311"
