<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 25 日消息 根据外媒 WindowsLatest 消息，微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 目前最新版本为 Build 22000.100，该版本带来了全新的应用商店 Microsoft Store。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/de2b480b-b38e-4eb3-baff-c4bbe537c571.jpg" w="696" h="392" title="Win11 预览版应用商店采用全新动画效果，分类更科学" width="696" height="392" referrerpolicy="no-referrer"></p><p>在 Win11 之前的预览版，应用商店与 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 一致。Win11 Build 22000.100 带来了新版应用商店，在界面动画上有了大幅改善，<span class="accentTextColor">响应更快更流畅</span>。具体来看，当用户单击某一个项目时，不同元素将以独立的动画效果进行展示，最终排列成新界面。</p><p>以下为视频演示：</p><p><a class="ithome_super_player" contenteditable="false" target="_blank" data-timestamp="1627203529308" data-vpreview-url="https://f.video.weibocdn.com/jdY5ThIwlx07OvoiPi2A010412006Wky0E010.mp4?label=mp4_720p&template=1080x720.25.0&trans_finger=1f0da16358befad33323e3a1b7f95fc9&ori=0&ps=1BThihd3VLAY5R&Expires=1627207143&ssig=X6uBit684I&KID=unistore,video" href="https://weibo.com/1740418805/KqnTw9OHn"></a></p><p>IT之家了解到，新版应用商店在界面排布上也进行了优化，左侧边栏分别为主页、应用、游戏、电影和电视按钮。</p><p>除此之外，<span class="accentTextColor">微软还将优化应用商店的搜索功能</span>，以便更好地提供用户想要的结果，同时对应用程序进行排名。根据微软在 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 发布会的消息，未来应用商店将正式支持运行<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a> App，与亚马逊应用商店合作。届时，通过 Microsoft Store 即可访问安卓应用商店，目前亚马逊的 App Store 约有 50 万个应用。</p>
