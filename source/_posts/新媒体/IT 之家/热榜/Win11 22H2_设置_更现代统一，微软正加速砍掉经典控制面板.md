<p data-vmark="fa1b"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 20 日消息，据 Windows Latest 报道，Windows 控制面板已经存在了相当长的时间，它是 <a class="s_tag" href="https://www.win7china.com/" target="_blank">Windows 7</a> 或更早版本中管理设置的默认应用。自 Windows 8 发布以来，微软开始关注控制面板的现代替代方案，并决心在未来发布的 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 中把遗留功能迁移到“设置”应用程序中。</p><p data-vmark="e3c0"><img src="https://img.ithome.com/newsuploadfiles/2021/12/618a78cc-d103-4d13-b4bf-3e328f130199.jpg" w="696" h="357" alt="经典控制面板" title="Win11 22H2“设置”更现代统一，微软正加速砍掉经典控制面板" width="696" height="357" referrerpolicy="no-referrer"></p><p data-vmark="7519">IT之家获悉，在过去的几年里，微软公司一直在将控制面板的部分功能迁移到现代的设置中，在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 版本 2004 发布后，过渡变得缓慢。然而，自 Windows 11 以来，迁移似乎正在以相当快的速度进行。</p><p data-vmark="8245">微软在控制面板迁移方面的做法相当直接。该公司希望只有在设置应用程序中准备好该页面的替代品时，才会删除控制面板页面。从 Windows 11 的 22H2 版本开始，试图点击进入控制面板的“程序和功能”页面将自动将用户重定向到“设置”。</p><p data-vmark="3c7f"><img src="https://img.ithome.com/newsuploadfiles/2021/12/c17ce49c-679a-4483-98a9-aa266192d2a6.jpg@s_2,w_820,h_530" w="1104" h="713" alt="经典控制面板" title="Win11 22H2“设置”更现代统一，微软正加速砍掉经典控制面板" srcset="https://img.ithome.com/newsuploadfiles/2021/12/c17ce49c-679a-4483-98a9-aa266192d2a6.jpg 2x" width="1104" height="530" referrerpolicy="no-referrer"></p><p data-vmark="5913">控制面板中“程序和功能”的链接将在 Windows 11 Build 22523 或更高版本中打开 Windows 设置应用。虽然没有全面消亡的截止时间，但控制面板很可能仍然是下一个大型 Windows 11 功能更新的一部分，但也可能会失去一些功能。</p><p data-vmark="02c2"><img src="https://img.ithome.com/newsuploadfiles/2021/12/1b8f28d5-90b8-4b4c-a76c-6880ede7d800.jpg@s_2,w_820,h_564" w="1107" h="761" alt="Windows 11设置" title="Win11 22H2“设置”更现代统一，微软正加速砍掉经典控制面板" srcset="https://img.ithome.com/newsuploadfiles/2021/12/1b8f28d5-90b8-4b4c-a76c-6880ede7d800.jpg 2x" width="1107" height="564" referrerpolicy="no-referrer"></p><p data-vmark="f559">早在 2020 年 9 月，媒体就首次发现了禁用“程序和功能”控制面板页面的实验性 flag。微软已经测试这个特殊的变化一年多了，以便用户适应迁移到 Windows 设置中的适当页面。</p><p data-vmark="b1ee">让你管理已安装应用程序的“程序和功能”页面已被移至“设置”中的“应用和功能”，该页面已经拥有控制面板提供的所有功能。</p><h3 data-vmark="1aa9">Windows 设置正在变得更好</h3><p data-vmark="d19b">此外，控制面板的 Windows 更新页面现在已被移至设置应用程序中一个名为“卸载更新”的新部分。正如你在截图中所看到的，Windows 设置终于可以让你卸载 Windows 更新，而无需导航到控制面板。</p><p data-vmark="4174"><img src="https://img.ithome.com/newsuploadfiles/2021/12/3130d6e1-9efc-4616-8b1a-3b456c218ab7.jpg@s_2,w_820,h_573" w="1121" h="784" alt="Windows 11设置" title="Win11 22H2“设置”更现代统一，微软正加速砍掉经典控制面板" srcset="https://img.ithome.com/newsuploadfiles/2021/12/3130d6e1-9efc-4616-8b1a-3b456c218ab7.jpg 2x" width="1121" height="573" referrerpolicy="no-referrer"></p><p data-vmark="30df">请注意，“设置”应用程序拥有这些控制面板页面提供的所有功能，但这一变化可能会让那些宁愿使用控制面板等传统功能或者仅仅是习惯了操作系统的旧布局来完成这些任务的人感到失望。</p><p data-vmark="d9e1">事实上，与控制面板应用相比，“设置”应用有更多高级功能。</p><h3 data-vmark="fa4b">控制面板的日子已经不多了</h3><p data-vmark="c280">这一变化是另一个迹象，表明控制面板的日子可能已经不多了。程序和功能页面是控制面板中最常用的功能之一，它最终被迁移到了 Windows 11 设置中。</p><p data-vmark="0765">但完全扼杀经典的控制面板仍然需要一些时间，最晚会在未来几年内完成。</p><p
