<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 19 日消息 即将发布的 Windows 11 将改进操作系统的 UI，将放弃 Windows 95 时代的图标，拥抱 Fluent 图标，但根据泄露的预览版，微软也在为传统的功能进行其他 UI 改进。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/5170fe19-0ad1-4d38-afcd-84869e2d6d34.jpg" w="696" h="365" title="老式对话框在 Win11 中得到 UI 重塑，圆角窗口已运用于多处" width="696" height="365" referrerpolicy="no-referrer"></p><p>根据泄露的预览版，Windows 11 仍然使用传统的一些元素，包括 Windows 3.1 的对话框。这证实了微软没有完全重新设计 Windows 11，但微软在此基础上做了一些现代化的改进。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/5ab07ae8-0c2a-45b7-83b4-0ec1f40e8578.jpg@s_2,w_820,h_499" w="982" h="597" title="老式对话框在 Win11 中得到 UI 重塑，圆角窗口已运用于多处" srcset="https://img.ithome.com/newsuploadfiles/2021/6/5ab07ae8-0c2a-45b7-83b4-0ec1f40e8578.jpg 2x" width="982" height="499" referrerpolicy="no-referrer"></p><p>例如，<span class="accentTextColor">微软在 ODBC 数据源管理员中启用了圆角和阴影</span>，这是一个非常古老的传统工具，允许管理员将 Microsoft Access 数据库连接到外部数据源，如 Microsoft SQL Server。</p><p>ODBC 数据源管理员工具显然是自 Windows 7 以来第一次获得明显的 UI 改进。当然，大多数人并不使用 ODBC 数据源工具，但这证实了微软也正将其现代圆角处理应用于最古老的工具之一。</p><p>像设备管理器和磁盘管理工具这样的传统应用程序现在也在使用圆角设计。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/677cc11d-e509-458b-adb3-9d268694bd7f.jpg@s_2,w_820,h_570" w="973" h="676" title="老式对话框在 Win11 中得到 UI 重塑，圆角窗口已运用于多处" srcset="https://img.ithome.com/newsuploadfiles/2021/6/677cc11d-e509-458b-adb3-9d268694bd7f.jpg 2x" width="973" height="570" referrerpolicy="no-referrer"></p><p>这些是很小的变化，但也是肉眼可见的，而且似乎确实改善了整体 UI 的一致性。</p><p>正如我们在一开始提到的，微软也在改变图标，其中一些图标包括诸如软盘驱动器、网络（芯片和调制解调器）、全球（互联网）等的图标。此外，在 Windows 11 上仍然存在的控制面板，也在使用新的图标和圆角。</p><p>IT之家了解到，任务管理器也有新的图标和圆角。这些图标是流畅设计改造的一部分，以前只限于现代应用程序，如 Windows Store、Edge 和 Microsoft 365 应用程序。</p>
