<p data-vmark="3eeb"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 9 日消息，荣耀笔记本新品发布会将于 5 月 16 日 14:30 举行，发布会的主角是搭载 Magic OS for Windows 的荣耀 MagicBook 14。</p><p data-vmark="f5ab">荣耀赵明日前已经宣布，Magic OS for Windows 基于笔记本底层调校，主打三大功能，分别为<strong>主动服务、协同服务、基础能力</strong>。</p><p data-vmark="ac18">今天，赵明介绍了 Magic OS for Windows 的基础能力。</p><p style="text-align: center;" data-vmark="3b7c"><img src="https://img.ithome.com/newsuploadfiles/2022/5/57751ad0-ddb7-4ea5-a5bc-542a720a02ab.png" w="1264" h="878" title="荣耀预热 Magic OS for Windows：OS Turbo 加持，新款 MagicBook 14 整体功耗下降 2X%" width="1264" height="570" referrerpolicy="no-referrer"></p><p data-vmark="d838">赵明称，全新荣耀 MagicBook 14 是第一款搭载 Magic OS for Windows 的新品，<strong>荣耀 OS Turbo 技术也将首次在 PC 端亮相</strong>。在这项技术的优化加持下，<strong>全新荣耀 MagicBook 14 在保持性能的同时，整体功耗将大幅下降</strong>。荣耀的海报显示，这款笔记本在 OS Turbo 加持下，功耗最高下降 2X%。</p><p style="text-align: center;" data-vmark="8958"><img src="https://img.ithome.com/newsuploadfiles/2022/5/3de8151c-7ef7-4068-acf6-8a8d2734a978.jpg@s_2,w_820,h_1458" w="1440" h="2560" title="荣耀预热 Magic OS for Windows：OS Turbo 加持，新款 MagicBook 14 整体功耗下降 2X%" width="1440" height="1458" referrerpolicy="no-referrer"></p><p data-vmark="22c6">昨天，“荣耀老熊”对 Magic OS for Windows 进行了解释，称其大致由主动服务的场景化体验、智能协同的跨设备 OS、个性化的基础 OS 构成，<strong>将助力 PC 产品性能更强、功耗更低</strong>。“荣耀老熊”还表示，未来荣耀将以赋能 Windows 为起点，继续秉持“双轮驱动”开发理念……</p><p data-vmark="6dac">目前，我们尚不知道 Magic OS for Windows 是一款性能调优 / 跨端互联软件，还是荣耀真的与微软合作，对 Windows 11 进行了“底层定制”。 Magic OS for Windows 到底为何物，将在 5 月 16 日 14:30 正式揭晓。</p>
