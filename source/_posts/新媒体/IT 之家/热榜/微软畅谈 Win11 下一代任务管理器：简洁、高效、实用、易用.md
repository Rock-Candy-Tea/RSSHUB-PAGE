<p data-vmark="199d"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 5 日消息，随着 Windows 11 22H2 版本的推出，我们将看到一个全新的任务管理器。在最近的一次广播中，微软向大家阐述了关于新任务管理器的理念和新设计，以及为什么他们放弃了基于标签页的 UI 而转而采用汉堡（hamburger）菜单。</p><p data-vmark="8be9">（科普：hamburger menu 是由交互设计师 Norm Cox 于 1981 年为 Xerox Star 个人工作站创建的一种风格，你可以当成抽屉式菜单，整体更加简洁。）</p><p data-vmark="bd7e">微软在解释这一设计时说道，他们希望任务管理器可以遵循现代 UI 框架和 Windows 11 设定的设计原则。这个新的界面在很多方面都是“简洁”的，微软希望开发任务管理器时也可以遵循同样的原则。</p><p data-vmark="fb7b">通过切换到汉堡菜单，微软希望帮助导航系统进一步简化，使传统视觉区域与 Windows 11 的外观和感觉完美融合在一起。你可以通过点击菜单来翻阅其页面，也可以使用 Ctrl + tab 键翻阅页面，所以新的设计相对更适合键盘党和无障碍人士使用。</p><h2 data-vmark="f275">新的任务管理器将提供更好的易用性<br></h2><p data-vmark="8fa2">原始选项卡 / 部分区域现在包括了任务管理器所有页面中特有的常用操作。你可以直接通过命令栏创建一个新的任务，而无需从一系列繁琐的菜单中找到对应的功能按钮。</p><p data-vmark="15a7"><img src="https://img.ithome.com/newsuploadfiles/2022/5/dc313523-f45d-420c-a6c0-716d3aebe56f.jpg" w="720" h="420" title="微软畅谈 Win11 下一代任务管理器：简洁、高效、实用、易用" width="720" height="420" referrerpolicy="no-referrer"></p><p data-vmark="4484">另外一个明显的变化是，新的任务管理器设置页面。你可以在黑暗 / 光明模式之间切换，也可以设置默认页面。</p><p data-vmark="d7b5"><img src="https://img.ithome.com/newsuploadfiles/2022/5/e4976428-b0c4-4a0d-9ca3-44b4185b3487.jpg" w="722" h="420" alt="任务管理器黄色" title="微软畅谈 Win11 下一代任务管理器：简洁、高效、实用、易用" width="722" height="420" referrerpolicy="no-referrer"></p><p data-vmark="4575">关于任务管理器中的状态图标，微软说它决定用暂停（pause）图标取代挂起（suspended）图标，以尽可能减少混乱。</p><p data-vmark="5b0b"><img src="https://img.ithome.com/newsuploadfiles/2022/5/3f5becc9-e2ae-472f-9af8-cface0343948.jpg" w="686" h="420" alt="任务管理器进程选项卡" title="微软畅谈 Win11 下一代任务管理器：简洁、高效、实用、易用" width="686" height="420" referrerpolicy="no-referrer"><img src="https://img.ithome.com/newsuploadfiles/2022/5/b5d008c8-9d06-4fd0-a5ec-bc0f0fe2972f.jpg" w="701" h="420" alt="任务管理器设计更新" title="微软畅谈 Win11 下一代任务管理器：简洁、高效、实用、易用" width="701" height="420" referrerpolicy="no-referrer"></p><h2 data-vmark="b4ab">任务管理器的新功能可提高 UI 响应速度</h2><p data-vmark="5663">在功能方面，新的任务管理器有一个新增功能叫“效率模式”，它可以帮助你限制某一个特定的进程的 CPU 资源占用。</p><p data-vmark="d807"><img src="https://img.ithome.com/newsuploadfiles/2022/5/ea85f441-93aa-47d7-b1ec-38470e171fc1.jpg@s_2,w_820,h_536" w="1440" h="941" alt="任务管理器效率模式" title="微软畅谈 Win11 下一代任务管理器：简洁、高效、实用、易用" srcset="https://img.ithome.com/newsuploadfiles/2022/5/ea85f441-93aa-47d7-b1ec-38470e171fc1.jpg 2x" width="1440" height="536" referrerpolicy="no-referrer"></p><p data-vmark="46eb">“当你通常在使用 Windows 时，一定会有一个进程或应用程序占用大量的资源（CPU）。面对这种情况，我们之前往往只能选择通过任务管理器去结束进程，” 微软负责任务管理器的项目经理说，“你实际上可以用“效率模式”来帮你，它不仅可以节约你的 CPU 资源，还可以提高你的 UI 响应速度”。</p><p data-vmark="214a">IT之家了解到，目前任务管理器的高效模式只能限制应用程序的 CPU 占用率，但微软也在探索对内存和网络方面的支持，这可能会在未来某个版本中启用。</p>
