            <p data-vmark="4d6c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 6 日消息，微软刚刚向 Beta 和 Dev 频道的测试用户推送了 Windows 11 Insider Preview Build 22616 版本。该版本主要是一些小更新和 bug 修复方面的内容，例如新的 Xbox 功能栏。</p><p data-vmark="6a57">新的 Xbox 手柄功能栏看上去也没什么太意外的地方，你可以在这里快速启动最近玩过的游戏。</p><p data-vmark="a6d5" style="text-align: center;"><img 更新：带来全新的 Xbox 功能栏，任务栏托盘弹窗图标拖放功能回归" width="1024" height="372" referrerpolicy="no-referrer"></p><p data-vmark="a8bc">根据微软的说法，当您将 Xbox 手柄配对或连接到您在开发或测试版通道中注册的 Windows 11 PC 时，系统将自动为你打开这一功能。您可以在这里看到最近玩过的三款游戏，以及 Steam 或 EA Origin 游戏启动器，当然你也可以使用 Xbox 按钮手动启用。</p><p data-vmark="16af" style="text-align: center;"><img 功能栏，任务栏托盘弹窗图标拖放功能回归" width="1024" height="519" referrerpolicy="no-referrer"><img src="https://img.ithome.com/newsuploadfiles/2022/5/0f3012cf-dab8-477c-ab17-6cc70461706d.png" w="1024" h="150" data-vmark="75af">除此之外，此版本还修复了系统托盘中的“显示隐藏图标”弹出窗口问题，小部件、设置、窗口、任务管理器和 Windows 沙箱的崩溃问题也已得到修复，微软更新公告如下。</p><ul class=" list-paddingleft-2"><li><p data-vmark="8d35">修复了在使用日语输入法时按下半角 / 全角键导致某些应用程序崩溃的问题。</p></li><li><p data-vmark="4f5c">修复了一个问题，这一问题会导致您在尝试使用手势打开小部件栏时出现闪退的问题。</p></li><li><p data-vmark="f432">修复了用户无法使用滑动功能稳定地启用小部件的问题。</p></li><li><p data-vmark="9e7f">修复了投影到某些无线设备时可能导致快速设置崩溃的问题。</p></li><li><p data-vmark="17eb">解决了使用 3 指触屏手势进行最小化可能导致动画在整个系统中停止工作的问题。</p></li><li><p data-vmark="db2c">修复了影响近期飞行中 DWM 可靠性的问题。</p></li><li><p data-vmark="451a">修复了导致某些应用程序（例如设置）在重新打开升级或重启时打开空白的问题。</p></li><li><p data-vmark="8409">修复了从命令栏中单击效率模式时导致进程列表在应用程序和背景组之间波动的问题</p></li><li><p data-vmark="7520">修复了一个潜在的问题，该问题可能导致 Windows 沙盒中的某些文本在某些情况下显示为黑框。</p></li><li><p data-vmark="358e">修复了导致 Windows 更新通知显示它们是从“Windows.SystemToast.WindowsUpdate.MoNotification”而不是“Windows 更新”发送的问题。</p></li></ul><p data-vmark="a29a" referrerpolicy="no-referrer"></p><p data-vmark="6e8b">IT之家了解到，此版本还显示了两个新的已知问题。例如，某些全屏应用可能无法显示实时字幕；部分启用顶部实时字幕的程序也可能在字幕窗口后重启。据称，您可以使用系统菜单（ALT + 空格键）来解决这个问题。</p><p data-vmark="b1b5">值得一提的是，现在 Dev 和 Beta 频道的 Win11 版本号相同，微软表示此次窗口期很快就会关闭！所以如果你要想使用稳定一点系统的话可以趁现在无损切换到 Beta 版本。</p><p data-vmark="7192">其他方面，此次更新还有两大功能特性。微软决定取消对 Build 22581 中刚引入的系统托盘的更改，特别是“显示隐藏的图标”。</p><p data-vmark="471d">此外，Windows 11 专业版上的 Windows 预览体验成员现在将在初始设备设置 (OOBE) 期间要求连接到 Internet，不过这一条件仅当你设置为个人使用时才会触发。如果您选择为工作或学校设备则没有任何变化。</p><p data-vmark="d216" style="text-align: center;"><img height="576" referrerpolicy="no-referrer"></p><p data-vmark="2383">微软表示，虽然最新版本中没有桌面右下角的版本水印，但这并不意味着已经完成了新版本的开发，他们表示 Win11 开发版水印将在未来的版本中回归。</p>
