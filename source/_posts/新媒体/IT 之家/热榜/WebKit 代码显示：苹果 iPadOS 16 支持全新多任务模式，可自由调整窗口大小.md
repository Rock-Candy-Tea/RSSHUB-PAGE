<p data-vmark="a1af"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 27 日消息，苹果将于 6 月 6 日至 10 日通过线上形式举行年度全球开发者大会，届时预计将发布全新 iOS 16、iPadOS 16、macOS 13、watchOS 9、tvOS 16 系统。</p><p data-vmark="4a42">据 9to5Mac 报道，WebKit 最近的变化<strong>暗示了 iPad 可能在窗口大小调整方面会有很大的改变</strong>。IT之家了解到，WebKit 是苹果的浏览器引擎，为 Safari 和其他浏览器提供支持。由于 WebKit 是一个开源引擎，代码公开，任何人都可以在 GitHub 上进行访问。</p><p data-vmark="dd1d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2021/5/7f13fd8a-989c-41a0-93c7-dd4aa2fe552e.jpg@s_2,w_820,h_547" alt="苹果iPad Pro" title="WebKit 代码显示：苹果 iPadOS 16 支持全新多任务模式，可自由调整窗口大小" referrerpolicy="no-referrer"></p><p data-vmark="0fa7">开发者 Steve Troughton-Smith 在 Twitter 上首次指出，最近在 GitHub 上添加的 WebKit 代码<strong>提及了一种全新“多任务模式”</strong>，<strong>允许在 iPadOS 上自由调整窗口大小</strong>。WebKit GitHub 的更新表明，<strong>iPadOS 将为启用该功能增加一个新的“多任务模式”切换</strong>。</p><p data-vmark="7b75">另一个来自苹果工程师的 GitHub 更新证实了这一变化，他再次提到了 WebKit 更新，旨在提高这种“多任务模式”的兼容性。</p><p data-vmark="2c1c">此外，Smith 解释了这一功能的含义，并将其与当前 Windows 设备的“平板模式”进行了比较：“‘多任务模式’看起来是可以在运行时改变的，所以应用可在运行时进行切换。可以想象为类似 Windows 10 的‘平板模式’，用户能随时切换以调整为触控或键鼠的 UI。”</p>
