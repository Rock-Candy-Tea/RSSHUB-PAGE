<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 21 日消息 微软将于 6 月 24 日正式发布全新的 Windows 11 以及其应用程序商店，但实际上该公司之前从未正式公布新系统名称，也没有说明它打算在发布会上透露什么内容。</p><p>但现在，微软已经在日本一份支持文法中明确表示 Win11 的存在，确定其所谓的“下一代操作系统”就是之前泄露的 Windows 11。</p><p>IT之家了解到，尽管 Windows 11 预告片和爆料已多到无可计数，甚至连开发镜像都已经泄漏出去，但还是有很多人不相信这一事实，仍有大量用户认为泄露是假的，Windows 11 只是微软放出来的烟雾弹，下一代操作系统不可能只有这么点东西。</p><p>微软现已通过 DMCA 删除通知和最新的 GitHub 支持文档确认 Windows 11 ，证实泄露非虚。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/8518e2bc-9ca7-4b7c-afe1-226112e03c51.jpg@s_2,w_820,h_695" w="1118" h="947" title="Win11 绝非烟雾弹，微软官方已肯定 Windows 11 确实存在" srcset="https://img.ithome.com/newsuploadfiles/2021/6/8518e2bc-9ca7-4b7c-afe1-226112e03c51.jpg 2x" width="1118" height="695" referrerpolicy="no-referrer"></p><p>▲微软发给印度网站 Beebom 的通知，要求停止分发泄露的 ISO，并明确表示“这是未发布的 Windows 11 的泄露副本”。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/6126dd5f-904e-46db-af19-f07a897e1921.jpg@s_2,w_820,h_482" w="1193" h="701" title="Win11 绝非烟雾弹，微软官方已肯定 Windows 11 确实存在" srcset="https://img.ithome.com/newsuploadfiles/2021/6/6126dd5f-904e-46db-af19-f07a897e1921.jpg 2x" width="1193" height="482" referrerpolicy="no-referrer"></p><p>▲微软的 Azure 文档已经确认了 Windows 10 继任者即为 w11/Windows 11</p><p>此外，这次泄漏还证实了 Windows 11 将与 Windows 10 共存，但微软没有透露任何与其发布会有关的内容。</p><p>值得一提的是，微软现已将该页面清空删除，再次肯定泄露的真实性。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/56a64582-c1a3-47a5-add6-8dfaf5ede421.png" w="1145" h="541" title="Win11 绝非烟雾弹，微软官方已肯定 Windows 11 确实存在" width="1145" height="387" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/3f701d5c-19c8-4f23-9054-0173d7846acf.jpg@s_2,w_820,h_461" title="Win11 绝非烟雾弹，微软官方已肯定 Windows 11 确实存在" referrerpolicy="no-referrer"></p><p>《<a href="https://www.ithome.com/0/557/468.htm" target="_blank">微软惊天泄漏，Win11 预览版镜像提前出世（附 ISO 下载）</a>》</p>
