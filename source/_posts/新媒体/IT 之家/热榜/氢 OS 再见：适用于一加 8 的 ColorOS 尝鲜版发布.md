<p>IT之家 3 月 22 日消息 今天，一加手机 CEO 刘作虎终于正式宣布，一加 9 系列将会出厂搭载 ColorOS 11，取代氢 OS，而过往机型的氢 OS 还会持续维护更新。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210322_101349_159.jpg" w="690" h="920" title="氢 OS 再见：适用于一加 8 的 ColorOS 尝鲜版发布" width="690" height="920" referrerpolicy="no-referrer"></p><p>此外，来自一加论坛的消息，<span class="accentTextColor">今天适用于一加 8 的 ColorOS 尝鲜版已发布</span>。官方表示，一加帐号与 OPPO 帐号暂不互通，数据或资料无法共享。</p><p><strong>IT之家附官方关于本次更新的详细内容：</strong></p><p>亲爱的加油们，</p><p>你们好，我们收到部分加油希望体验 ColorOS 的想法，于是与 ColorOS 团队一起合作推出了这次的 ColorOS 尝鲜版，欢迎喜欢尝鲜的加油们体验。也期待大家试用后给我们提出宝贵的意见和建议。</p><p>由于此版本仍在开发中，可能存在不稳定及兼容性问题，建议有刷机经验的用户进行体验，并提前做好手机数据备份。同时，这个版本保留了原汁原味的 ColorOS 体验，请大家根据自己的喜好进行选择。</p><p>注：此版本目前仅支持 OnePlus 8 国内版本，其他机型适配计划请关注 OnePlus 9 系列新品发布会。</p><p><strong>【温馨提示】</strong></p><p>1. 请注意，此版本为 ColorOS 尝鲜版，系统稳定性可能有一定潜在风险和体验不佳之处。</p><p>2. 此版本目前仅支持 OnePlus 8 国内版本，后续可刷回一加的系统。</p><p>3. 更新此版本和回退版本均会导致手机数据全部清除，请务必提前做好数据备份！详细方式请见【备份建议】。</p><p>4. 此版本需要通过电脑刷机，请确保电脑操作系统为 Windows 10。</p><p>5. 升级到此 ColorOS 尝鲜版本后，后续无法通过 OTA 升级，需要通过工具再次刷机。</p><p><strong>【已知问题】</strong></p><p>由于此版本为尝鲜版，版本体验中可能出现以下问题。请知悉：</p><p>1. 相机与相册有一定的兼容性问题，包括概率性相机闪屏、相册部分信息显示存在误差、偶发性无法查看图片。</p><p>2. 一加帐号与 OPPO 帐号暂不互通，数据或资料无法共享。如需使用云服务、软件商店、游戏中心等应用，可在一加官网（ <a href="https://www.oneplus.com/cn" _src="https://www.oneplus.com/cn">https://www.oneplus.com/cn</a>  ）进行一次登陆，完成帐号升级，升级后可使用一加帐号。</p><p>3. 此版本没有 “钱包”应用，且不支持 NFC 相关功能。</p><p>4. 王者荣耀、和平精英等游戏可能存在掉帧、卡顿及无法分享至微信、QQ 等问题。</p><p>5. 三段式开关可能存在适配问题。</p><p>6. 部分三方应用可能存在适配问题。</p><p>7. 黑屏手势不支持上滑、左滑和右滑手势。</p><p><strong>【更新方法】</strong></p><p>更新前请保证手机电量充足和做好重要数据备份，备份方式详见【备份建议】。</p><p>1. 此版本需要通过电脑刷机，请确保电脑操作系统为 Windows 10。</p><p>2. 获取刷机包后进行解压，刷机包已附于 “刷机教程”说明帖中。</p><p>3. 解压后，请根据此 “ 刷机教程”说明贴，按照步骤进行刷机。</p><p><strong>【回退方法】</strong></p><p>更新前请保证手机电量充足和做好重要数据备份，备份方式详见【备份建议】。</p><p>1. 此版本需要通过电脑刷机，请确保电脑操作系统为 Windows 10。</p><p>2. 下载回退包后进行解压，回退包已附于 “ 刷机教程”说明帖中。（点击刷机教程即可跳转）</p><p>3. 回退版本的刷机方式与更新版本的方式一致。请根据 “ 刷机教程”，按照步骤进行手机刷机。</p><p><strong>【备份建议】</strong></p><p>1）在更新版本前，建议将整机数据备份到电脑上保存，以便后续回退版本时恢复数据。可使用一加搬家备份至电脑，建议操作如下：</p><p>* 请注意此备份和恢复方法仅适用于一加系统。</p><p>a、如需备份文件管理 - 保险柜中的文件，需要先将文件移除保险柜，一加搬家才能识别并将其备份，建议操作路径为：文件管理 -> 保险柜 -> 移出保险柜；</p><p>b、选择一加搬家 -> 备份与恢复 -> 新建备份 -> 备份完成；</p><p>c、将手机连接电脑 -> 进入手机文件夹 -> 内部存储空间 -> 复制 opbackup 文件夹 -> 将该文件夹复制并保存至电脑或其它存储设备。</p><p>d、成功回退到一加系统后将 opbackup 文件夹复制回手机根目录 -> 一加搬家 -> 备份与恢复 -> 恢复备份即可。</p><p>2）可通过三方备份工具将电话、短信、图片等数据备份到云端并进行迁移，后续更新至 ColorOS 后通过三方备份工具恢复。</p><p><strong>【反馈渠道】</strong></p><p>如加油在使用过程中遇到问题，可通过以下方式进行问题反馈：</p><p>方法一：找到 “反馈工具箱”应用，使用该应用进行问题反馈；</p><p>方法二：拨号盘输入 *#800# 后可直接跳转至反馈工具箱。</p><p>关于 “反馈工具箱”的更多介绍和使用指南可参考此链接： https://www.oppo.cn/thread-393320869-1</p><p>此外，如加油在使用过程中遇到操作相关疑问，可在桌面找到 “使用技巧”应用，了解更多功能与使用技巧。</p><p><strong>【问题答疑】</strong></p><p>Q：为什么有些三方应用、相机偶尔会出现闪退或无响应的情况？</p><p>A：由于此版本为尝鲜版。部分三方应用尚未完全适配，以及功能仍有待优化，可能会出现部分使用体验不佳情况。</p><p>Q：一加帐号和 OPPO 帐号共通吗？</p><p>A：一加帐号与 OPPO 帐号暂不互通，数据或资料无法共享。如需使用云服务、软件商店、游戏中心等应用，可在一加官网（ https://www.oneplus.com/cn ）进行一次登陆，完成帐号升级。升级后可使用一加帐号。</p><p>Q：为什么 NFC 功能无法使用？</p><p>A：此版本暂无 “钱包”App，所以暂时无法使用 NFC 相关功能。</p><p>Q：氢 OS 的功能在此版本上会有么？</p><p>A：此版本为 ColorOS 尝鲜版，暂不会有关于氢 OS 的相关功能。</p><p>Q：此版本会多久进行一次更新？</p><p>A：我们会根据用户反馈进行版本更新。</p><p>Q：其他机型什么时候能体验 ColorOS 尝鲜版？</p><p>A：请关注 OnePlus 9 系列新品发布会。</p><p>Q：刷了此版本后可以刷回到氢 OS 吗？</p><p>A：可以支持刷回至氢 OS。具体刷回氢 OS 的方式请参考【回退方法】。请注意更新此版本和回退版本均会导致手机全部数据清除，请加油们务必提前做好数据备份</p><p>Q：为什么游戏无法开启高帧率？</p><p>A：此问题已知，我们会在后续版本进行适配。</p><p>Q：线性马达效果没有 OnePlus 8 效果体验好。</p><p>A：由于 ColorOS 与氢 OS 的振动场景存在差异，所以体验效果上会有些许差别。</p><p>Q：“腾讯视频”无法分享 GIF 动图至微信。</p><p>Q：“天猫”使用过程中无法对商品进行评论。</p><p>Q：“手机淘宝”扫一扫结果异常。</p><p>Q：“抖音火山版”无法分享二维码至微信。</p><p>A：由于部分三方应用尚未完全适配，以及功能仍有待优化，可能会出现部分使用体验不佳情况。</p><p>感谢各位加油参与此次的尝鲜体验！</p>
