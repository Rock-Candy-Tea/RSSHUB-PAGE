data-vmark="2199">不卖关子，直接揭晓“答案”：</p><p data-vmark="d36a"><img src="https://img.ithome.com/newsuploadfiles/2021/10/2a5efe99-c4b7-4926-892a-24be93b45067.gif" w="406" h="375" Gonna Give You Up》的 MV！</p></blockquote><p data-vmark="de81">没错，所有人都被这位黑客 <span class="accentTextColor">Rickroll</span> 了。</p><p data-vmark="2451">此处需要“插播”一下，Rickroll 在国外大致就是一个被整蛊了的梗。</p><p data-vmark="2acb">举个例子，你看到一个文件标题叫做《日本老师 800G 合集》，通宵达旦下载了一晚上，结果里面全是葫芦娃。</p><p data-vmark="028b">恭喜你，这时你就被 Rickroll 了。</p><p data-vmark="43d7">面对此情此景，被整蛊的师生们也是非常无语了：</p><blockquote><p data-vmark="12bd">好吧~ 大家周末愉快！</p></blockquote><p data-vmark="25c0"><img src="https://img.ithome.com/newsuploadfiles/2021/10/6d341d83-74e3-41f6-a1e9-08523dd5d493.png" w="590" h="292" data-vmark="141b">来到另外一间教室，得知被整蛊的师生沉浸在笑声中。</p><p data-vmark="521c">走廊里，放学的学生们也在笑着窃窃私语：</p><p data-vmark="6643">我们被玩了。</p><p data-vmark="f43c">不仅是教室，就连学校机房的屏幕也被稳稳地拿捏。</p><p data-vmark="1918"><img src="https://img.ithome.com/newsuploadfiles/2021/10/12c8cf83-0616-4235-bf5d-9f280183b1c8.gif" w="406" h="375" title="国外高中生黑了学校所有的电子屏，结果老师转发点赞，当地政府还上门求合作" width="406" height="375" referrerpolicy="no-referrer"></p><p data-vmark="232f">就这样，在这位高中生一波“神操作”之下，整个学校都被 Rickroll 了。</p><p data-vmark="8896">所到之处，完美同步，皆是《Never Gonna Give You Up》的魔性音乐。</p><p data-vmark="96f4"><img src="https://img.ithome.com/newsuploadfiles/2021/10/1373ab9d-254f-419c-b916-65d3ed48c9fb.png" w="1080" h="801" title="国外高中生黑了学校所有的电子屏，结果老师转发点赞，当地政府还上门求合作" width="1080" height="608" referrerpolicy="no-referrer"></p><h2 data-vmark="3f43">“蓄谋已久”的实施计划</h2><p data-vmark="0cc7">这次恶搞可以说是相当的成功，那么问题来了：</p><blockquote><p data-vmark="a5c8">他们为什么要侵入整个学校？又是如何一步步实现如此大规模的攻击的呢？</p></blockquote><p data-vmark="a45a">这不，在事情结束之后，“领头人”自己发了篇博客，对这些问题一一做了解答。</p><p data-vmark="3b80"><img src="https://img.ithome.com/newsuploadfiles/2021/10/b997745b-763e-4af2-99d1-69c5f2d2580c.png" w="1080" h="317" title="国外高中生黑了学校所有的电子屏，结果老师转发点赞，当地政府还上门求合作" width="1080" height="241" referrerpolicy="no-referrer"></p><p data-vmark="33a4">故事还要从这位小黑客高一的时候开始讲起。</p><p data-vmark="a557">当时的他，对于“黑客攻击”方面的技术道德、责任等东西，可以说是一无所知。</p><p data-vmark="64ef">于是，只要逮到个机会，他就会“肆意破坏、攻击”。</p><p data-vmark="3b4d">久而久之，小黑客就心生一个想法：</p><blockquote><p data-vmark="7140">我们学校的技术水平咋样呢？</p></blockquote><p data-vmark="db9b">说干就干。</p><p data-vmark="b720">他招呼了两三好友，开始对学校局域网范围内的<span class="accentTextColor">所有 IP 进行扫描</span>。</p><p data-vmark="ed81">不扫不知道，一扫吓一跳，连他们自己都没想到能一下子扫出来这么多 IP。</p><p data-vmark="cb33">多到甚至惊动了学校技术主管，一度要求他们马上停止。</p><p data-vmark="877f">虽然小团队按要求停了下来，但截至当时，<span class="accentTextColor">就已经扫出来了 8388606 个 IP</span>！</p><p data-vmark="50f9">而从拿到的这些结果，他们发现学校的各种设备都暴露在了局域网络内。</p><p data-vmark="c7e7">例如打印机、IP 电话…… 甚至还包括没有任何密码认证的安全摄像头。</p><p data-vmark="8b58">瞧，下面这张图，就是 14 岁的小黑客，用 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 远程访问摄像头的截图。</p><p data-vmark="5090"><img src="https://img.ithome.com/newsuploadfiles/2021/10/8d0790ea-322c-40b3-9dfa-2977f2a66c55.png" w="1080" h="593" title="国外高中生黑了学校所有的电子屏，结果老师转发点赞，当地政府还上门求合作" width="1080" height="450" referrerpolicy="no-referrer"></p><p data-vmark="988c">这波操作又惊动校方的技术团队，他们马上对摄像头加上了 ACL 限制，把它们保护了起来。</p><p data-vmark="7948">然而，真正的博弈才刚刚开始。</p><p data-vmark="f86c">小黑客想到了另外一个“突破口”—— <span class="accentTextColor">IPTV 系统</span>。</p><p data-vmark="5cf6">简单来说，IPTV 就是交互式网络电视，主要包含三个部分：</p><ul class=" list-paddingleft-2"><li><p data-vmark="673a">接收器（AvediaPlayer）</p></li><li><p data-vmark="86fd">编码器（AvediaStream）</p></li><li><p data-vmark="f99d">管理（AvediaServer）</p></li></ul><p data-vmark="ed3c">首先是 AvediaPlayer 接收器，它是连接投影仪和电视的蓝色小盒子。</p><p data-vmark="1bf6">它们可以向各自的设备发送串行命令，来打开/关闭显示器，改变输入/音量，切换通道等等。</p><p data-vmark="0f76">这些接收器包括一个 web 接口和一个 SSH 服务器来执行串行命令。</p><p data-vmark="5d17">其次是 AvediaStream 编码器可以连接到直播视频的设备上。</p><p data-vmark="1fcb">它们将来自这些设备的实时消息，编码到 AvediaPlayer 接收器，从而显示流。</p><p data-vmark="cdb2">而 AvediaServers 则允许管理员同时控制所有接收器和编码器。</p><p data-vmark="d547">它们有典型的 x8664 处理器，并运行企业级 Linux 发行版 CentOS。</p><p data-vmark="0dc9">与接收器和编码器一样，AvediaServers 也有 web 接口和 SSH 服务器。</p><p data-vmark="2348">而这一整套系统，小黑客在高一的时候就已经玩儿的非常溜了，用他自己的话就是：</p><blockquote><p data-vmark="66fc">只是随便玩了几次，就可以完全驾驭了。</p></blockquote><p data-vmark="5a50">而这，也为他们刚刚完成的大规模整蛊买下了伏笔。</p><h2 data-vmark="a61d">是时候展示真正的技术了</h2><p data-vmark="4921">通过 IPTV 整蛊的想法，一直在小黑客的脑中盘旋。</p><p data-vmark="2fba">直到高三下学期，因为疫情久久上网课的学生们，终于迎来了大规模返校的机会。</p><p data-vmark="f77b">此时的他突然想到 ——“是时候搞点大动静了”。</p><p data-vmark="f8e3">于是火速把小伙伴们召集在了一起，开始制定整蛊计划，还起了个响亮的代号 ——“the Big Rick”。</p><p data-vmark="c267">首先要解决的一个问题就是：</p><blockquote><p data-vmark="c05a">如何同时控制所有的投影仪。</p></blockquote><p data-vmark="3b6c">虽然可以使用 web 接口向每个接收器发送命令，但他们发现，同时向每个接收器发送 HTTP 流并不是个理想的做法。</p><p data-vmark="e67d">因此，他们使用每个接收器上的 SSH 访问，作为命令和控制（C2）通道。</p><p data-vmark="730f">团队开发了一个简单的 shell 脚本，可以作为分阶段的有效负载，提前上传到每个接收器。</p><p data-vmark="d339">这个脚本包含各种函数，可以在接收器本地执行对 web 接口的请求。</p><p data-vmark="31c0">而由于增加了有效负载的灵活性，他们还可以在整蛊结束后，备份和恢复接收器的设置到文件系统。</p><p data-vmark="f445">为了防止有人在整蛊过程中关闭投影仪或静音，团队还专门搞了循环命令，每隔 10 秒钟就会打开电源并把音量调到最大。</p><p data-vmark="ea46">他们处理的第二大难题，就是建立一个定制的视频流，来实时播放 Rickroll。</p><p data-vmark="6da5">但由于 ACL 的限制，只有 AvediaStream 编码器或 AvediaServers 管理可以做到这一点。</p><p data-vmark="c098">这个部分可以说是最痛苦的，因为他们白天在学校没法执行。</p><p data-vmark="e476">于是，挪到了晚上进行测试！</p><p data-vmark="a3c2">具体来说，就是他们通过远程连接到计算机实验室的一台电脑，让前置摄像头对着投影仪。</p><p data-vmark="d150">然后他们就会录制一段视频，来测试投影机是否正确地显示了流。</p><p data-vmark="218f">而就在这次整蛊计划实施的前三天，意想不到的事情发生了。</p><p data-vmark="ced1">团队在扫描 IP 的时候，突然发现了几个新的物联网设备。</p><p data-vmark="e086">后来才知道，原来是学校最近安装了新的铃声系统 （bell system），叫做教育寻呼和对讲机通信（EPIC）。</p><p data-vmark="fad5">这些设备大多都被安装在了教室或走廊，当扬声器用。</p><p data-vmark="58e4">“咦？要不把铃声全部改成《Never Gonna Give You Up》？”</p><p data-vmark="c8de">突然的灵感，成功地勾起了团队的兴趣，于是，说干就干。</p><p data-vmark="7407"><img src="https://img.ithome.com/newsuploadfiles/2021/10/b6049381-bb63-417b-ab87-e977322253a6.png" w="1080" h="576" data-vmark="470b">但他们还有一些顾虑，例如造成课堂过于混乱，再例如期末考试也将来临等。</p><p data-vmark="7887">所以团队还针对整蛊，定制了缜密的时间表：</p><p data-vmark="68ca"><img 亿美元的无人车公司，图森未来的 CTO 的身份，再一次站到了聚光灯中心。</p><p data-vmark="802f"><img src="https://img.ithome.com/newsuploadfiles/2021/10/af23ce54-1c39-41f0-860b-aba13bf35ae1.gif" w="720" h="405"
