<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 27 日消息 外媒 Windows Latest 报道，微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 于今年 6 月底发布，其整体界面的重大变化证明了操作系统版本号的跳跃。Windows 11 结合了 Fluent Design 和 WinUI 的优点，为满足最低要求的设备提供了现代和触摸友好的体验。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/5c544eb6-a07f-404f-a5a6-da56e392b03a.png" w="1440" h="960" title="微软 Win11 一批原生应用将获得视觉 UI 更新：画图、截图工具、记事本、照片、录音机、计算器..." width="1440" height="547" referrerpolicy="no-referrer"></p><p>随着 Windows 11 的推出，“Fluent Design 流畅设计”变得更好，微软还推出了一种名为“Mica 云母”的新设计材料，将允许应用程序和桌面元素与主题壁纸相匹配。然而，这些变化主要影响系统元素，这意味着应用程序开发人员需要更新他们的应用程序，以获得 Windows 11 的最佳效果。</p><p>值得庆幸的是，<a href="https://www.ithome.com/0/569/037.htm" target="_blank">微软正计划用 Windows 11 的外观设计来更新几乎所有的原生应用程序</a>。在这篇文章中，你会发现由微软开发的应用列表，这些应用将在不久的将来更新为 Windows 11 的外观设计。</p><p><strong>微软画图</strong></p><p>微软终于对最受欢迎的应用程序之一 —— 微软画图进行了重大的视觉改造。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/9465289b-d060-4800-8ed6-9b23f25050bc.jpg@s_2,w_820,h_522" w="1194" h="760" title="微软 Win11 一批原生应用将获得视觉 UI 更新：画图、截图工具、记事本、照片、录音机、计算器..." srcset="https://img.ithome.com/newsuploadfiles/2021/8/9465289b-d060-4800-8ed6-9b23f25050bc.jpg 2x" width="1194" height="522" referrerpolicy="no-referrer"></p><p>新的微软画图截图首先出现在 Unsplash Windows 页面上，后来微软的 Windows 主管 Panos Panay 分享了更多细节。正如你所看到的，画图将最终与 Windows 11 的流畅设计语言和新的标题菜单相匹配。</p><p>这种视觉上的刷新使微软画图的外观更加现代，这些变化将在未来几周内开始向 Windows Insider 计划的测试者推出。</p><p>《<a href="https://www.ithome.com/0/570/161.htm" target="_blank">微软官方预热展示 Win11 画图全新设计：现代图标、圆角 UI，浅/深色主题</a>》</p><p><strong>记事本</strong></p><p>在 6 月 24 日的活动中，微软预告了 Windows 11 记事本的新外观，支持 Office 或类似画图的标题菜单。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/1cf781d8-4456-4e15-9b5f-11deb3e83b74.jpg@s_2,w_820,h_461" w="1344" h="756" title="微软 Win11 一批原生应用将获得视觉 UI 更新：画图、截图工具、记事本、照片、录音机、计算器..." srcset="https://img.ithome.com/newsuploadfiles/2021/8/1cf781d8-4456-4e15-9b5f-11deb3e83b74.jpg 2x" width="1344" height="461" referrerpolicy="no-referrer"></p><p>这个紧凑的标题菜单使用了触摸式控制。</p><p><strong>微软照片</strong></p><p>微软也在用 Windows 11 更新照片应用程序。作为更新的一部分，你可以期待新的界面和新的控件，这将与 WinUI 的视觉更新保持一致。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/e63f206b-eebf-4b07-8af9-153e2b68e9e4.jpg@s_2,w_820,h_504" w="1190" h="731" title="微软 Win11 一批原生应用将获得视觉 UI 更新：画图、截图工具、记事本、照片、录音机、计算器..." referrerpolicy="no-referrer"></p><p>正如你所看到的，微软照片应用程序似乎得到了新的浮动菜单，其中将包括编辑选项，如裁剪、旋转和其他。</p><p><strong>录音机</strong></p><p>在发布会上，微软还预告了语音记录器的新外观。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/e93f3fe9-7707-488c-8de3-e6156f2ccd9a.jpg@s_2,w_820,h_504" w="1280" h="787" title="微软 Win11 一批原生应用将获得视觉 UI 更新：画图、截图工具、记事本、照片、录音机、计算器..." 计算器</strong></p><p>Windows 计算器应用程序也将得到一个小的重新设计，使计算工具与流畅设计方法的其他部分保持一致。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/20d96248-97b0-46df-8065-323cc5e756a5.jpg@s_2,w_820,h_628" w="871" referrerpolicy="no-referrer"></p><p>IT之家获悉，计算器应用程序已经用 C#重新编写，WinUI 2.6 的圆角现在更加明显，而且还支持黑暗模式。正如你所看到的，计算器的整体界面在 Windows 11 中没有什么变化。</p><p><strong>截图工具</strong></p><p>Windows 11 的应用程序生态系统的另一个重大变化涉及经典的 Snipping Tool（截图工具）和 Snip & Sketch（截图和草图）应用程序。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/d6e71a0e-783f-42f9-b59f-cf2515ebaa84.jpg" w="798" h="435" title="微软 Win11 一批原生应用将获得视觉 UI 更新：画图、截图工具、记事本、照片、录音机、计算器..." width="798" height="435" referrerpolicy="no-referrer"></p><p>改良后的截图工具是用经典 Snipping Tool 和 Snip & Sketch 创建的。你仍然可以使用 WIN + SHIFT + S 键盘快捷键访问它，或者你也可以在“开始”菜单中搜索该应用程序。</p><p>更新后的截图工具将支持矩形剪贴、自由剪贴、窗口剪贴和全屏剪贴，”微软解释说。</p><p><strong>时钟应用</strong></p><p>时钟应用程序正在获得一个更新的外观，支持 Windows 11 和 <a class="s_tag" href="https://win10.ithome.com/" 音乐</a>》</p><p><strong>邮件和日历</strong></p><p>邮件和日历应用程序也得到了视觉上的改进（圆角）。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/72e53951-360e-409a-83c2-9ed75451dad5.jpg@s_2,w_820,h_439" w="1280"
