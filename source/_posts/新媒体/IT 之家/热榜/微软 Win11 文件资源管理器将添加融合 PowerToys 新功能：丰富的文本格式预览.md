<div>   
<p data-vmark="4655"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 7 日消息，据 Windows Latest 报道，Windows 文件资源管理器集成了用户的照片、视频、文档、电子表格等。在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 和 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 上，我们可以快速、轻松地查看其中的一些文件，而不必打开其链接的应用程序。这可以通过文件资源管理器的预览面板实现，它出现在屏幕的右侧。</p><p data-vmark="cef6"><img src="https://img.ithome.com/newsuploadfiles/2021/12/0a72a48c-b00c-4417-936c-372ee814fc82.jpg" w="696" h="365" title="微软 Win11 文件资源管理器将添加融合 PowerToys 新功能：丰富的文本格式预览" width="696" height="365" referrerpolicy="no-referrer"></p><p data-vmark="9f80">你可以通过点击“查看”选项卡来尝试 Windows 文件管理器中的预览面板。当你在“查看”选项卡中时，你需要选择预览窗格，然后点击希望查看的文件。如果该文件格式被支持，你将能够在资源管理器本身中查看该文件的预览。</p><p data-vmark="529b">通过 PowerToys，微软一直在测试文件管理器的附加组件，以扩展预览窗格的默认功能。现有的 PowerToys 扩展添加了预览 SVG 和 markdown 文件的能力，这对程序员和 Web 开发人员来说特别方便。</p><p data-vmark="221c">微软现在正计划通过一个新的扩展将文件资源管理器的预览体验提升到新的水平。</p><p data-vmark="bb78">目前，文件资源管理器对文本文件格式的支持有限。有许多文本文件不能在预览窗格中预览，这尤其影响到开发人员使用的文件。</p><p data-vmark="8ce3"><img src="https://img.ithome.com/newsuploadfiles/2021/12/bf644c45-2bb0-4215-bc6a-b4ed82805902.jpg@s_2,w_820,h_394" w="1391" h="669" title="微软 Win11 文件资源管理器将添加融合 PowerToys 新功能：丰富的文本格式预览" srcset="https://img.ithome.com/newsuploadfiles/2021/12/bf644c45-2bb0-4215-bc6a-b4ed82805902.jpg 2x" width="1391" height="394" referrerpolicy="no-referrer"></p><p data-vmark="8979">正如在截图中看到的，微软正计划将 Monaco 编辑器的文件预览功能整合到文件资源管理器的预览面板中。Monaco 是 Visual Studio Code 的一个流行的代码编辑器，PowerToys 将使用它来轻松地将文本预览应用于大量的文件类型。</p><p data-vmark="e1c8"><img src="https://img.ithome.com/newsuploadfiles/2021/12/080645d0-684f-42b0-90fa-b8daa69b471f.jpg" w="736" h="758" title="微软 Win11 文件资源管理器将添加融合 PowerToys 新功能：丰富的文本格式预览" width="736" height="758" referrerpolicy="no-referrer"></p><p data-vmark="072a">由于独立开发者 Aaron Junker 和微软之间的合作，这项功能已经在开发之中。这将允许你在 Windows 11 或 Windows 10 的文件资源管理器中预览几乎所有流行的文本相关文件格式。</p><p data-vmark="211b">此外，微软正计划在 Windows 中添加右键菜单项“对该类型的文件使用文本预览”。对于开发者，微软表示将使用 Visual Studio Code 进行预览服务，以实现语法突出显示。</p><p data-vmark="ce8c">除了帮助你预览自己的文件外，这个新的 PowerToys 模块可能为高级预览面板的整合铺平道路。在某一点上，你将能够查看未知的文件，当你不清楚哪些 Windows 应用程序包括对这种文件的支持时，这特别有帮助。</p><p data-vmark="14f8">除了新的文件资源管理器扩展，微软 PowerToys 最近也得到了支持，用户可以使用键盘快捷键轻松地将麦克风 / 摄像机静音和取消静音操作。</p><p data-vmark="953c">IT之家获悉，微软还在为 PowerToys 的不同区域试验类似 Windows 11 的外观设计。像 Power Rename 和 PowerToys 设置这样的工具已经得到改造，预计在下一个主要版本中还会有更多的设计改进。</p>
