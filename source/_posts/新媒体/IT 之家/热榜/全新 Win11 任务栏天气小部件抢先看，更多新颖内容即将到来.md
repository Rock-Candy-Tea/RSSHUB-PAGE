<p data-vmark="2380"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 12 日消息，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的任务栏拖放预计还需要几个月的时间才能正式支持，但在此期间，微软提供了多种小功能更新并进行测试，例如任务栏功能其中一个新特性是对天气小部件的支持，它现在已经在开发频道中上线。</p><p data-vmark="1e85">IT之家了解到，Windows 11 配备了新的集成小工具，样式类似于 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 此前已有的“新闻和兴趣”功能。<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 小部件位于屏幕左侧，目前仅有一些用于本地功能的小部件，如微软待办、Outlook 日历、照片、交通和天气等等。</p><p data-vmark="f2af"><img src="https://img.ithome.com/newsuploadfiles/2021/12/6638f021-850b-47ba-8163-1e11b0f882f0.jpg@s_2,w_820,h_430" w="1440" h="756" title="全新 Win11 任务栏天气小部件抢先看，更多新颖内容即将到来" srcset="https://img.ithome.com/newsuploadfiles/2021/12/6638f021-850b-47ba-8163-1e11b0f882f0.jpg 2x" width="1440" height="430" referrerpolicy="no-referrer"></p><p data-vmark="2e4f">当你点击 Windows 小部件时会弹出一个大型信息面板，其中包括最新的头条新闻和 MSN (微软的新闻源) ，包括体育事件、政治新闻、相关股票信息、即将发生的事件等等，您还可以在面板中滚动并与新闻条目进行交互。</p><p data-vmark="2b8a">预计微软之后会进一步放开对小程序的限制，外媒认为微软将会在 2022 年支持第三方小部件，并在 2022 年上半年对该功能进行有效测试。</p><p data-vmark="b83c"><img src="https://img.ithome.com/newsuploadfiles/2021/12/0b80755a-c365-4d23-8814-79b883f786b5.jpg" w="816" h="898" title="全新 Win11 任务栏天气小部件抢先看，更多新颖内容即将到来" width="816" height="898" referrerpolicy="no-referrer"></p><p data-vmark="2ff9">如果你对这种新的功能设计感兴趣的话，可以通过加入 Dev Channel 来尝试一下。除了最新版 Windows 系统，你还可以从微软商店下载新版 Windows Web 体验包。</p><p data-vmark="706c">据介绍，新版小部件可能类似于此前支持第三方桌面小工具的 Windows Vista，但区别在于用户将无法将这些 Win11 小工具固定到桌面。而第三方小部件似乎仅可通过新的 api 为开发者提供集成支持。</p><p data-vmark="024c">目前还不清楚微软之后否会允许用户将这些小部件固定到桌面上。但目前还没有迹象表明 Windows 11 Dev 测试版本有这种设计。</p>
