<p>IT之家 3 月 27 日消息 Windows 95 是微软公司的一个爆款操作系统，发售初期人们都要排队购买该系统。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327084330_1696.jpg" w="640" h="473" title="时隔26年，黑客在 Windows 95 邮件应用中发现彩蛋" width="640" height="473" referrerpolicy="no-referrer"></p><p>开发这款操作系统及其应用程序的开发人员有很多值得骄傲的地方，他们决定在操作系统中的一个应用程序中留下他们的印记。</p><p>Windows 爱好者和逆向工程师 Albacore 一直在研究 Windows 95，并在 IE4 Internet Mail 应用中发现了一个至今未知的彩蛋。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210327084501_1757.jpg@base@tag=imgScale&w=1440&h=612&q=90" w="1440" h="1074" title="时隔26年，黑客在 Windows 95 邮件应用中发现彩蛋" srcset="https://img.ithome.com/newsuploadfiles/2021/3/20210327084501_1757.jpg 2x" width="1440" height="612" referrerpolicy="no-referrer"></p><p>当该打开应用程序时，转到 “关于”界面，选择其中一个文件，并键入 “Mortimer”，<span class="accentTextColor">你将看到一个滚动窗口，会列出应用程序的开发人员的名称</span>。这个彩蛋此前并没有被发现过，其并没有出现在维基百科上的 Windows 95 彩蛋列表中。</p><p>用户可以在浏览器窗口中运行 Windows 95，感兴趣的IT之家小伙伴可<a href="https://archive.org/details/win95_in_dosbox" target="_blank">点此尝试</a>。</p>
