<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 15 日消息 微软本周推出了新版本的 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 预览版，版本号为 Build 22000.132。然而新的 Windows 11 预览版仍不能支持 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 应用，而这恰恰是 Windows 11 的关键功能之一，大量开发者不禁好奇微软到底什么时候才能上线这一功能。</p><p>据悉，微软此前表示 Windows 11 可支持<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a> App，但后者必须从 Amazon Appstore 中下载，而亚马逊商店将会在之后的版本中集成到 Microsoft Store 之中。</p><p>除此之外，此前微软描绘的一些其他功能和改进目前同样仍未实装，例如收件箱还没有采用 Windows 11 全新的设计元素。</p><p>微软此前仅表示 Windows 11 正式版将于今年晚些时候推出，将会带来重大更新。根据微软的说法，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 的设计看起来比 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 要更加淡雅，这主要归功于他们微妙的圆角，Mica 主题，以及更加一致的 Fluent 设计。</p><p>IT之家了解到，在最新的开发者的问答环节中，微软解释了他们对 Fluent Design 的愿景，并确认名为 Mica 的“新设计”不会影响软件或系统的性能。此外，微软还承诺将为更多 Windows 应用引入 Win11 元素。</p><p>微软澄清道，他们不可能在第一天就拿出所有内容，因为“这些都是不同的团队（负责的），他们都有自己的时间规划和目标。”</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/cf7f2197-daeb-47d6-83cf-938df1acf607.jpg" w="696" h="392" title="仍未支持安卓 App、Fluent 设计尚未展开，微软承诺将对 Win11 进行大改造" width="696" height="392" referrerpolicy="no-referrer"></p><p>微软为 Windows 11 引入了一种新的 Mica 效果，并将以此取代此前的 Fluent Design 亚克力/毛玻璃效果，展现出一种“颜色层次感”，以此帮助用户在多个窗口并列时可专心于其主要目标窗口。</p><p>根据微软一份新的声明，这种新的设计和其他改进将很快出现在更多的收件箱应用中。</p><p>“我们正努力协调整个公司（人员），因为我们希望将新的 Windows 11 Fluent 外观引用于每个应用中，但他们（开发者）只能按自己的节奏来。”</p><p>据悉，Fluent Design 和 Mica 材料可以用于所有应用程序，包括计算器和一些高级应用程序，如 Office。</p><p>“我们的设计非常流畅，而且适用于许多类型的 UI，包括超复杂的密集型 UI。例如 Office 有一个相当复杂的用户界面，但它采用了 Fluent，因此它支持各种交互方式 —— 触摸、键盘、鼠标和手写笔。”</p><p>微软还表示，微软 Teams 也将很快得到 Windows 11 元素的改善，但我们不知道它何时开始推出。</p>
