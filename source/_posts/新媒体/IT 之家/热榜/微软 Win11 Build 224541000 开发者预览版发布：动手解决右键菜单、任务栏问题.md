<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 10 日消息 微软向 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> Dev 通道的用户推送了 <span class="accentTextColor">Build 22454.1000</span> 版本更新，带来了一些新变化。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/d61cddd7-99b8-4a1b-8665-2f90bb7b7a89.png" w="1120" h="314" title="微软 Win11 Build 22454.1000 开发者预览版发布：动手解决右键菜单、任务栏问题" width="1120" height="230" referrerpolicy="no-referrer"></p><p>首先，22454.1000 版本带来了<span class="accentTextColor">改进的回收站右键菜单</span>，从图中可以看到，新版的回收站右键菜单更加契合 Win11 的设计，对触摸用户和性能进行了优化，隐藏了部分选项。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/f9eec7a7-18d8-46d9-b333-96ed780c2cf1.png" w="959" h="455" title="微软 Win11 Build 22454.1000 开发者预览版发布：动手解决右键菜单、任务栏问题" width="959" height="389" referrerpolicy="no-referrer"></p><p>▲ 右侧为新版回收站右键菜单</p><p>此外，Win11 Build 22454.1000 还添加了一个新选项，允许用户将网络共享固定到快速访问部分，而无需单击“显示更多选项”。</p><p>微软还为韩国用户推出了更可靠输入和新的亚克力视觉设计的输入法。输入法菜单具有了亚克力、新的选择视觉、暗黑模式，并且还提供更好的性能和兼容性。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/1db7a385-4b63-4930-a12f-2d579c155d9a.png" w="321" h="723" alt="The updated Korean IME candidate window in dark mode." title="微软 Win11 Build 22454.1000 开发者预览版发布：动手解决右键菜单、任务栏问题" width="321" height="723" referrerpolicy="no-referrer"></p><h2>以下为其他细节改动：</h2><ul class=" list-paddingleft-2"><li><p>修复了 Windows Defender 应用程序防护 (WDAG) 功能可能导致持续错误检查的问题。</p></li><li><p>当用户右键点击开始菜单时，会正确出现 Windows 终端选项。</p></li></ul><p><strong>任务栏：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>现在，在使用任务视图按钮上的项目导航进行扫描时，“讲述人”会正确关闭桌面弹出按钮。</p></li><li><p>修复了桌面的预览缩略图在某些纵横比下无法正确呈现的问题。</p></li><li><p>修复了在某些情况下音量图标工具提示显示错误数字的问题。</p></li><li><p>输入指示器、快速设置和通知中心图标工具提示在弹出窗口打开时将不再被遮挡。</p></li><li><p>修复了打开应用程序时，图标会从随机位置进入任务栏的错误动画。</p></li><li><p>修复了与任务栏预览交互后意外卡在全屏应用程序（例如 PowerPoint 演示文稿）顶部的问题。</p></li><li><p>修复了当用户鼠标悬停在任务栏图标上时，任务栏闪烁的错误。</p></li><li><p>修复了应用程序关闭后，图标依然以红色方式显示在任务栏中的问题。</p></li><li><p>Shift + 点击应用程序图标现在可以正常工作了。</p></li></ul><p><strong>输入：</strong></p><ul class=" list-paddingleft-2"><li><p>修复了第一次调用手写面板后候选文本不出现的问题。</p></li><li><p>减轻了点击文本字段以调用触摸键盘时动画中的停顿。</p></li><li><p>修复了剪贴板历史无法显示的错误。</p></li><li><p>优化输入指示器中显示的第三方输入法图标。</p></li><li><p>解决了在使用触摸键盘时更改窗口焦点时可能发生的 explorer.exe 崩溃的问题。</p></li><li><p>修复了选择使用以前版本的日语输入法导致某些游戏崩溃的问题。</p></li><li><p>修复了使用触摸键盘时语音输入提示无法连接到麦克风的问题。</p></li><li><p>解决了触摸键盘可能因使用错误的背景颜色而进入按键标签不可见状态的问题。</p></li><li><p>缓解了导致触摸键盘设置弹出有时在白色文本上显示白色的问题。</p></li><li><p>修复了触摸键盘中输入按钮交互可能导致候选区域布局损坏的问题。</p></li></ul><p><strong>文件资源管理器：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>提高了文件资源管理器中右键菜单的可靠性。</p></li><li><p>缓解了使用文件资源管理器时导致泄漏的几个问题。</p></li><li><p>当在文件资源管理器中启用单击打开内容的选项时，右键菜单现在不会立即关闭。</p></li><li><p>如果按 F11 全屏文件资源管理器，然后使用 WIN + Shift + Left/Right 更改窗口显示的屏幕，再次按 F11 将不再将窗口跳回原始屏幕。</p></li></ul><p><strong>设置：</strong></p><ul class=" list-paddingleft-2"><li><p>缓解了在“设置”中的搜索框中使用某些第三方输入法输入时可能导致候选窗口在屏幕上的其他位置呈现（未附加到搜索框）和/或插入到搜索框中的字符不显示的问题。</p></li><li><p>修复了设置中的 Windows 预览体验计划页面有时显示为空白。</p></li><li><p>对于阿拉伯语和希伯来语显示语言，辅助功能 > 鼠标指针和触摸下的鼠标指针可以使用了。</p></li><li><p>系统 > 存储 > 显示更多类别 > 其他不再显示“由组策略管理”。</p></li><li><p>在“设置”中添加了“查找我的设备”的一些链接，以帮助用户了解有关隐私资源的更多信息。</p></li><li><p>当使用对比主题设置焦点时，现在可以看到焦点辅助 > 时间段选择的开始时间和结束时间选择器。</p></li><li><p>解决了使用声音设置时可能发生的崩溃。</p></li><li><p>修复了快速设置中音量滑块的问题，该问题导致音量有时保存在与实际设置略有不同的级别。</p></li></ul><p><strong>登录与认证：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>修复了当网络图标更新登录屏幕上的状态时可能发生的崩溃。</p></li></ul><p><strong>应用窗口：</strong></p><ul class=" list-paddingleft-2"><li><p>修复了在应用程序最大化时将鼠标移动到屏幕顶部时，标题栏选项（包括关闭、最小化和最大化）在某些应用程序中未按预期显示的问题。</p></li></ul><p><strong>其他：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>当用户将鼠标悬停在搜索图标上时，改进了弹出浮窗的应用程序图标呈现样式。</p></li><li><p>解决了导致某些语言的共享窗口中文本重叠的问题。</p></li><li><p>切换到任务管理器的“详细信息”选项卡时进行了一些性能改进。</p></li><li><p>如果在切换到暗黑模式时 Windows 安全中心处于打开状态，UI 元素现在能更好地响应并且不会出现不可读的文本。</p></li><li><p>如果 Windows 安全中心的病毒和威胁防护下列出了大量排除项，现在将在加载它们时显示进度指示器。</p></li><li><p>修复了使用辅助功能设置中的选项增加文本大小时，Windows 安全中心的一些文本无法完全显示。</p></li><li><p>缓解了导致某些应用程序中使用的 WM_CTLCOLORSTATIC 消息被忽略的问题，因此某些地方的颜色无法正确显示。</p></li><li><p>修复了当桌面背景设置为幻灯片时出现的问题，该问题在重新启动 explorer.exe 之前会随着时间的推移影响性能。</p></li><li><p>修复了在启用 Hyper-V 并创建外部 V-Switch 后导致 Wi-Fi 速度变慢的问题。</p></li><li><p>缓解了导致某些 PC 在现代待机状态下进行错误检查的问题。</p></li><li><p>当系统中禁用动画时，从启动画面过渡到应用程序内容时，UWP 应用程序（如设置或反馈中心）中不再出现淡入淡出动画。</p></li></ul><h2>以下为已知问题：<br></h2><ul class=" list-paddingleft-2"><li><p>正在修复导致某些 Surface Pro X 出现 WHEA_UNCORRECTABLE_ERROR 错误检查的问题。</p></li><li><p>在某些情况下，使用开始或任务栏搜索时可能无法输入文本。如果遇到此问题，请按键盘上的 WIN + R 启动“运行”对话框，然后将其关闭。</p></li><li><p>右键点击“开始”按钮 (WIN + X) 时系统设置丢失。</p></li><li><p>切换输入法时，任务栏有时会闪烁。</p></li><li><p>任务栏中的应用程序图标被“显示隐藏图标”按钮截断，并且在使用多个显示器时未居中。（正在调查）</p></li><li><p>点击任务栏上的搜索图标后，搜索面板可能无法打开。如果发生这种情况，请重新启动“Windows 资源管理器”进程，然后再次打开搜索面板。</p></li><li><p>搜索面板可能显示为黑色，并且在搜索框下方不显示任何内容。</p></li><li><p>如果用户在文件资源管理器中右键点击 OneDrive 中的文件，当用户将鼠标悬停在打开子菜单的条目上时，上下文菜单将意外关闭，例如“打开方式”。</p></li><li><p>小部件板可能显示为空。要解决此问题，用户可以注销后重新登录。</p></li><li><p>小部件可能会在外部显示器上以错误的尺寸显示。如果遇到这种情况，用户可以先在实际 PC 显示器上通过触摸或 WIN + W 快捷方式启动小部件，然后再使用辅助显示器。</p></li><li><p>某些预览体验成员可能无法启动 Windows Sandbox。</p></li><li><p>将继续努力提高应用商店中的搜索相关性。</p></li></ul><p>IT之家的小伙伴可以在设置 —> Windows 更新中查找并下载。如果没有查找到，请先加入“Windows 预览体验计划”，选择 Dev 通道即可。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/df3debc6-9922-4c85-9ae1-dab26e839006.png" w="600" h="249" title="微软 Win11 Build 22454.1000 开发者预览版发布：动手解决右键菜单、任务栏问题" width="600"
