<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 16 日消息 今天<a href="https://www.ithome.com/0/563/378.htm" target="_blank">微软发布了 Windows 10 21H1 Build 19043.1147 (KB5004296) 更新</a>，面向 Release Preview 发布预览频道推送。</p><p>在这篇关于 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 21H1 博客文章中，微软还透露，他们将很快向 Beta 频道的 Windows Insiders 用户推送 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 预览版版本。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/33d87b51-9126-4465-9cd1-fb0e8e56a0ae.png" w="1440" h="960" title="微软：将很快发布 Win11 Beta 预览版，Win10 21H1 Beta 频道可直接升级" width="1440" height="547" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/44b460c1-9793-49dc-8925-452928c5725d.png" w="1062" h="120" title="微软：将很快发布 Win11 Beta 预览版，Win10 21H1 Beta 频道可直接升级" width="1062" height="93" referrerpolicy="no-referrer"></p><p>微软表示，</p><p>“注意：21H1 的 Beta 通道中的 Windows Insiders 将不会收到这个更新。我们将很快向这些内测人员推送 Windows 11 Insider Preview 构建版本。”</p><p>微软指出，Beta 频道是早期采用者的理想选择。Beta 频道中的内测人员和 IT 专业人士可以检查即将到来的 Windows 功能，同时仍然获得由微软验证的相对可靠的更新。</p><p>IT之家获悉，据传闻，微软 Windows 11 将在今年 10 月推向硬件 PC 设备，微软将需要努力工作（并需要大量的协助），以验证该操作系统在这么短的时间内准备好 RTM 正式版，首个 Windows 11 Beta 预览版预计将在 7 月底之前发布。</p>
