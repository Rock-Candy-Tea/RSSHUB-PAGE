<p data-vmark="a430">北京时间 9 月 18 日上午消息，据报道，苹果 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 13</a> 系列有一项变化尤其引人关注。它是自 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone X</a> 首次引入 Face ID 以来，苹果对 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 正面最大幅度的一次调整：那个引发争议的<span class="accentTextColor">刘海终于小了一点</span>。</p><p data-vmark="9f17"><img src="https://img.ithome.com/newsuploadfiles/2021/9/8b6710c2-b64c-498f-aef5-0e92266f4991.jpg" w="630" h="393" title="国外科技博主谈“苹果 iPhone 13 为何还要保留刘海屏”" width="630" height="393" referrerpolicy="no-referrer"></p><p data-vmark="e7cf">苹果称，新的刘海大约较旧机型小了 20%，但眼尖的人却发现，<span class="accentTextColor">苹果依然为此做出了一些妥协，把刘海上面的黑边做大了</span>。</p><p data-vmark="9355">刘海屏再小，也还是刘海屏，跟三星和一加等竞争对手采用的打孔屏和水滴屏相比，依然很大。</p><p data-vmark="4594">所以，我们很自然地会想到一个问题：都已经 2021 年了，是否还有必要继续使用刘海屏？这种设计是否已经过时？</p><p data-vmark="3f94"><strong>以下是科技博客 The Verge 的哈依姆・加藤伯格（Chaim Gartenberg）和肖恩・霍利斯特（Sean Hollister）的观点</strong>：</p><p data-vmark="e8ac"><strong>哈依姆</strong>：我从 iPhone X 发布以来一直使用 iPhone 的刘海屏，我目前还是想替刘海屏辩护一下。跟之前的边框式设计相比，这显然是一种提升；现在很难回到 iPhone SE 那种风格的设计了。虽然很多基于 Face ID 的功能都是些华而不实的东西，但它的核心功能依然很好。</p><p data-vmark="499b">当我要解锁手机时，Face ID 就开始工作了，效果很好。这主要是因为它确实需要苹果融合进来的各种技术（包括各种传感器、红外发射器和前置摄像头）。<a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 手机确实没有 iPhone 这种刘海，<span class="accentTextColor">但人脸解锁系统的效果也没有 iPhone 好用。如果刘海是为这种便利付出的代价，我可以接受</span>。</p><p data-vmark="5fc2"><strong>肖恩</strong>：当然，<span class="accentTextColor">我承认 Face ID 很好</span>。<span class="accentTextColor">这是我从 Android 转用 iPhone 后最好用的功能</span>，没有之一。我无法想象竟然能这么轻松地登录银行账号，查看病例记录，或者管理密码。但这既是对 Android 的嘲讽，也是对 iPhone 的赞美 —— 因为我真正喜欢的是速度，跟人脸无关。</p><p data-vmark="74be">Android 为什么不能在优秀的快速指纹识别功能上实现标准化？设计一个精确地放置在我手指位置的传感器？苹果为什么总是坚持把指纹传感器放在下面？我觉得这样很别扭。为什么不把指纹传感器放在背面的苹果商标上？</p><p data-vmark="5701"><strong>哈依姆</strong>：把指纹传感器放在背面很容易破坏苹果那漂亮、易碎的玻璃背板。如果他们能使用屏下指纹识别器，那最好设计在正面的屏幕下方。</p><p data-vmark="e511"><span class="accentTextColor">我觉得确实应该支持苹果重新启用 Touch ID</span>，将其与电源按钮整合（就像新的 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> Mini 和最近发布的 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad Air</a> 一样）。但即使这样，苹果仍然得找地方安装前置摄像头。藏在边框里面的设计已经不可接受了，所以根本无处可放。要么刘海小，功能少，要么刘海大，功能多，Face ID 是个显而易见的选择。</p><p data-vmark="2988"><img src="https://img.ithome.com/newsuploadfiles/2021/9/1745c764-ffbf-488d-bf45-233ce9aa7220.png" w="630" h="417" title="国外科技博主谈“苹果 iPhone 13 为何还要保留刘海屏”" width="630" height="417" referrerpolicy="no-referrer"></p><p data-vmark="12d2"><strong>肖恩</strong>：<span class="accentTextColor">完全可以效仿其他厂商的做法，打一个小孔</span>，而不用非得用那么大的刘海。或者采用屏下自拍摄像头，这种技术已经流传很久了。说到这里，苹果究竟是为什么不采用屏下指纹传感器，也没有采用打孔屏来安装 Faceb ID？我倒不是说这样的刘海让我每次看到屏幕都心生厌烦（OLED 效果很赞），但确实没有必要为了几个花哨的红外发射器设计这么大的刘海。</p><p data-vmark="018b"><strong>哈依姆</strong>：现在看来，屏下摄像头似乎还有一些问题。假设苹果也准备采用这种技术，可以想象这项技术要真正达到人们的想象，还需要一段时间。</p><p data-vmark="4fad">但即使达到预期效果，Face ID 也不只是几个红外发射器那么简单。还记得苹果发布 iPhone X 时的这张幻灯片吗？</p><p data-vmark="3fb6"><img src="https://img.ithome.com/newsuploadfiles/2021/9/4de51a80-2bb3-4ee6-bd08-5b12d5443f8f.png" w="630" h="408" title="国外科技博主谈“苹果 iPhone 13 为何还要保留刘海屏”" width="630" height="408" referrerpolicy="no-referrer"></p><p data-vmark="3fc2">这似乎很难用一个打孔屏来实现 —— 即使苹果愿意像三星 Galaxy S10 Plus 那样采用双镜头打孔屏。</p><p data-vmark="5f47">当然，支持恢复指纹传感器最重要的理由都源自 2020 和 2021 年的魔幻现实：我们都因为疫情而整天带着口罩，导致 Face ID 的识别效果大打折扣。但等到疫情减轻的时候，希望这个问题能有所改善，那个时候出门在外就更容易解锁手机了。</p><p data-vmark="5f97"><strong>肖恩</strong>：我准备把这件事情留到后面说，结果被你抢先一步。那么你是否同意速度比 Face ID 更重要？</p><p data-vmark="1d80">就说上面那张图吧，里面的扬声器、麦克风、环境光传感器和距离传感器其实在普通手机里面都有，很容易安装在手机边框上。退一万步讲，就算 Face ID 真的必须把一连串的组件排在一起，像 Pixel 4 那样用一个小小的“前额”代替“刘海”，又有什么错呢？</p><p data-vmark="e9f3"><img src="https://img.ithome.com/newsuploadfiles/2021/9/2215a8fc-876d-4c5b-a0f5-362453225063.png" w="630" h="419" title="国外科技博主谈“苹果 iPhone 13 为何还要保留刘海屏”" width="630" height="419" referrerpolicy="no-referrer"></p><p data-vmark="0122">如果能在额外多出的这部分地方增加一点电池，那这种变化完全可以接受，或者也可以设计一个不错的耳机插孔。Pixel 4 之所以失败，肯定不是因为设计，而是因为它的电池续航不好，噱头不多，而且没有足够的营销。</p><p data-vmark="855a"><strong>哈依姆</strong>：我觉得可能还是跟设计有一定的关系，Pixel 4 反而证明了我的观点：谷歌是为了融合 Soli 手势这个噱头，而融合了比 Face ID 更多的技术进去。</p><p data-vmark="75f2">为了摄像头把整个手机上面加一个黑边，感觉比刘海更糟糕。刘海屏至少可以让你在名义上使用手机的最顶部空间，屏幕两边的“兔耳朵”可以用来显示状态栏。</p><p data-vmark="823c"><span class="accentTextColor">苹果未来有可能在 iPhone 上恢复 Touch ID</span>，可能是按钮形式，也可能是屏下形式。但即便如此，我也很难想象他们目前会放弃 Face ID。也就是说，<span class="accentTextColor">无论你喜欢与否，在可预见的未来恐怕还是得忍受一段时间的刘海屏</span>。</p>
