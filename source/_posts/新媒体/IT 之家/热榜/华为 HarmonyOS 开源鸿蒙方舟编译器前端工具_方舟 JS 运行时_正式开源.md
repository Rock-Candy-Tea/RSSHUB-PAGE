<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 6 日消息  感谢IT之家网友热心线索投递，OpenHarmony 的“方舟 JS 运行时”现已开源。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/f1efee28-1434-4ab5-8029-709e4d6e1d09.jpg" w="720" h="156" title="华为 HarmonyOS 开源鸿蒙方舟编译器前端工具“方舟 JS 运行时”正式开源" width="720" height="156" referrerpolicy="no-referrer"></p><h2><a href="https://gitee.com/openharmony/ark_js_runtime" target="_blank">方舟 JS 运行时组件 </a></h2><p>方舟 JS 运行时（ARK JavaScript Runtime）是 OpenHarmony 上 JS 应用使用的运行时。包含 JS 对象的分配器以及垃圾回收器（GC）、符合 ECMAScript 规范的标准库、用于运行 ARK 前端组件生成的方舟字节码（ARK Bytecode，abc）的解释器、用于存储隐藏类的内联缓存、方舟 JS 运行时对外的函数接口（AFFI）等模块。</p><h2><a href="https://gitee.com/openharmony/ark_ts2abc" target="_blank">ts2abc 组件 </a></h2><p>ts2abc 组件是方舟平台的一个组件，其作为方舟编译器中 JavaScript 语言的前端工具，支持将 JavaScript 文件转换为方舟字节码文件。</p><h2><a href="https://gitee.com/openharmony/ark_runtime_core" target="_blank">方舟公共组件</a></h2><p>Runtime 组件是方舟运行时的公共组件，主要包括一些语言无关的基础运行库，包含承载字节码以及执行字节码所需要相关信息的 ARK File、支持 Debugger 的 Tooling、负责对应系统调用的 ARK Base 等。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/1da7d073-b3f6-4d12-b976-bb47e137fc83.jpg" w="808" h="399" title="华为 HarmonyOS 开源鸿蒙方舟编译器前端工具“方舟 JS 运行时”正式开源" width="808" height="399" referrerpolicy="no-referrer"></p><p>IT之家了解到，OpenHarmony 是由开放原子开源基金会（OpenAtom Foundation）孵化及运营的开源项目，目标是面向全场景、全连接、全智能时代，搭建一个智能终端设备操作系统的框架和平台。</p><p><a href="https://gitee.com/openarkcompiler" target="_blank">方舟编译器</a>（OpenArkCompiler）是为支持多种编程语言、多种芯片平台的联合编译、运行而设计的统一编程平台，包含编译器、工具链、运行时等关键部件。</p>
