<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a>4月9日消息 昨日，微软官方宣布，已经发布了 OneDrive 64 位版的早期预览版 Windows 客户端，这是 Windows 版 OneDrive 自 2014 年发布以来的首个 64 位版本，<span class="accentTextColor">将比 32 位版提供更好的性能和更高的大文件同步效率</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210409123146_2832.jpg" title="时隔 7 年，微软 Windows 版 OneDrive 终于发布 64 位版本" referrerpolicy="no-referrer"></p><p>IT之家了解到，<span class="accentTextColor"> 64 位版 OneDrive </span><span class="accentTextColor">只支持 64 位版 Windows</span><span class="accentTextColor">，而 Win10 ARM 只支持 32 位版 OneDrive</span>，并且 64 位版 OneDrive 占用内存更多。</p><p>微软表示，64 位版本对于普通用户来说体验差别不大，主要区别在于<span class="accentTextColor">同步以 GB 为单位以上的大文件</span>时，64 位版 OneDrive 能提供更好的性能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210409_124416_407.png" w="1189" h="768" title="时隔 7 年，微软 Windows 版 OneDrive 终于发布 64 位版本" width="1189" height="530" referrerpolicy="no-referrer"></p><p><strong>下载地址：</strong><a href="https://support.microsoft.com/zh-cn/office/%e5%9c%a8-64-%e4%bd%8d%e5%92%8c-32-%e4%bd%8d%e7%89%88%e6%9c%ac%e7%9a%84-onedrive-%e4%b9%8b%e9%97%b4%e9%80%89%e6%8b%a9-9d36d262-4fc2-4019-a8f6-314ef41a29d1?ui=zh-CN&rs=zh-CN&ad=CN" target="_blank">点此查看</a></p>
