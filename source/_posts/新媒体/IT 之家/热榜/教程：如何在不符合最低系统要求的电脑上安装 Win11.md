<p data-vmark="f35c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 7 日消息 早在 8 月，微软就披露了运行 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的最低系统要求。由于可信平台模块（TPM）2.0 版本的要求，一些比较新的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 电脑变得不符合 Windows 11 的升级要求。</p><p data-vmark="f597"><img src="https://img.ithome.com/newsuploadfiles/2021/10/54141f6b-c798-4c52-b551-ae1ee4fd4a0f.jpg@s_2,w_820,h_539" w="1164" h="765" alt="Microsoft Windows 11 PCs" title="教程：如何在不符合最低系统要求的电脑上安装 Win11" srcset="https://img.ithome.com/newsuploadfiles/2021/10/54141f6b-c798-4c52-b551-ae1ee4fd4a0f.jpg 2x" width="1164" height="539" referrerpolicy="no-referrer"></p><p data-vmark="f5a3">IT之家了解到，<span class="accentTextColor">如果你可以接受遇到兼容性问题或其他问题的风险</span>，你仍然可以使用以下方法在不符合 Windows 11 最低系统要求的 PC 上安装 Windows 11。</p><p data-vmark="7868">如何在不符合最低系统要求的 PC 上安装 Windows 11：</p><blockquote><p data-vmark="ce8c">1）在你的 Windows 10 电脑上添加以下注册表键：</p><p data-vmark="6b61">注册表键：HKEY_LOCAL_MACHINE\SYSTEM\Setup\MoSetup</p><p data-vmark="611f">名称：AllowUpgradesWithUnsupportedTPMOrCPU</p><p data-vmark="c0f8">类型：REG_DWORD</p><p data-vmark="7d9c">值：1</p><p data-vmark="6b02">注意：如果你通过使用注册表编辑器或使用其他方法错误地修改注册表，可能会发生严重的问题，这些问题可能需要重新安装操作系统。</p><p data-vmark="b01a">2）你可以下载媒体创建工具来制作 Windows 11 可启动的 USB 或 DVD，可以在<a href="https://go.microsoft.com/fwlink/?linkid=2156295" target="_blank">这里</a>下载 Windows 11 媒体创建工具。</p><p data-vmark="cab4">3）在你的 Windows 10 电脑上启动你在步骤 2 中创建的媒体上的设置，进行升级。</p><p data-vmark="0565">4) 你可以选择：</p><ul class=" list-paddingleft-2"><li><p data-vmark="4fa6">执行完全升级，这将保留个人文件（包括驱动程序）、应用程序和 Windows 设置，这是默认的设置。</p></li><li><p data-vmark="770a">只保留数据将只保留个人文件（包括驱动程序），不保留应用程序和 Windows 设置。</p></li><li><p data-vmark="bc71">清洁安装 Windows 11，将不保留 Windows 10 安装时的任何内容。</p></li></ul></blockquote><p data-vmark="ce0f">微软表示，不建议在不符合 Windows 11 最低系统要求的设备上安装 Windows 11。如果用户选择在不符合条件的硬件上安装 Windows 11，应该承担遇到兼容性问题的风险。</p><p data-vmark="d6c8">《<a href="https://www.ithome.com/0/579/082.htm" target="_blank">Win11 正式版今日到来，IT之家为你总结 4 种升级方案（教程）</a>》</p>
