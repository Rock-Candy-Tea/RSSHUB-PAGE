<p data-vmark="6f35"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 29 日消息，微软通过将基于 EdgeHTML 的 Edge 迁移到 Chromium 成功解决了 Windows 上的 Web 浏览器问题。</p><p data-vmark="bd75">该公司现在对 Outlook 应用采取了类似的策略，并开始通过 Office Insider 渠道推出适用于 Windows 11 和 Windows 10 的新版本电子邮件客户端。</p><p data-vmark="83b0">简单来说，Windows 目前支持两种桌面电子邮件客户端：预装的基于 UWP 的邮件应用和与 Microsoft Office 捆绑的 Outlook Win32 程序。微软正在努力为 Windows 和其他平台维护两个版本的 Outlook。</p><p data-vmark="c175">与电子邮件客户端的完整版相比，UWP 版缺少很多功能。有消息人士透露，微软已经将 UWP 应用程序改为“维护模式”，这意味着你不需要再指望微软为旧版应用带来大的功能改进。</p><h2 data-vmark="c674">Windows 11 的新电子邮件客户端体验</h2><p data-vmark="999c">新版 Outlook 应用基于 Chromium 的 Edge WebView，简单来说这就是一个 Web 壳。但是，Outlook 是一个设计非常好的 Web 应用，大多数用户甚至不会注意到它是一个 Web 应用。</p><p data-vmark="be1e">它具有现代应用程序所期望的所有功能，包括使用 @files 和 @documents 定位文档的能力。这类似于让您 @提及某人以将其添加到电子邮件中的现有功能。</p><p data-vmark="ae02">例如：当 Outlook 认为您错过了一条对你来说很重要的消息时，它会通过另一个新功能提醒您。</p><p style="text-align: center;" data-vmark="053f"><img src="https://img.ithome.com/newsuploadfiles/2022/5/b41c097f-9fd9-4501-b482-76eeac558c1d.jpg@s_2,w_820,h_566" w="1440" h="994" alt="Windows 11 Outlook 日历" title="微软全新 Win11 Outlook 电子邮件应用抢先看：采用 Mica 设计，快速整理好“我的一天”" srcset="https://img.ithome.com/newsuploadfiles/2022/5/b41c097f-9fd9-4501-b482-76eeac558c1d.jpg 2x" width="1440" height="566" referrerpolicy="no-referrer"></p><p data-vmark="781b">此外，它还集成了新的日历功能，虽然这有点背刺日历应用，但可以说他们基本功能相同。在这个界面中，你可以组织和维护你那些很重要的东西，并可以在一个视图中进行管理。</p><p style="text-align: center;" data-vmark="fef0"><img src="https://img.ithome.com/newsuploadfiles/2022/5/7ac9814a-948e-40a2-bef3-a030d00073cd.jpg@s_2,w_820,h_475" w="1440" h="834" alt="Outlook 云母设计" title="微软全新 Win11 Outlook 电子邮件应用抢先看：采用 Mica 设计，快速整理好“我的一天”" srcset="https://img.ithome.com/newsuploadfiles/2022/5/7ac9814a-948e-40a2-bef3-a030d00073cd.jpg 2x" width="1440" height="475" referrerpolicy="no-referrer"></p><p data-vmark="fa70">有趣的是，微软似乎也在为 Outlook 开发新的 Mica 材质设计。如上图所示，你可以通过修改注册表调出 Mica 样式的 Outlook 界面。</p><p data-vmark="c0f3">当然，微软也有可能采用 Fluent Design 的 Acrylic 材质。IT之家了解到，Microsoft Edge 的右键菜单以及主菜单已经支持类似 Acrylic 的透明 / 半透明效果。</p><p data-vmark="704e">除了新的外观设计之外，微软还为此应用添加了“我的一天”功能，让您可以在工作的同时帮助您与重要话题保持联系。此外，您还可以轻松地将消息作为任务拖放到待办事项中，或者更好地管理日历中的任务。</p><p data-vmark="844c">值得注意的是，新 Outlook 应用的其他 UI 改进有望在今年晚些时候出现，微软可以通过服务器端更新为你添加新功能，敬请期待。</p>
