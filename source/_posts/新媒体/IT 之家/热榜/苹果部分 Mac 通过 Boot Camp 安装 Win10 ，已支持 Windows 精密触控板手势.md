<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 11 日消息 外媒 9to5 Mac 报道，触控板手势是一个备受用户喜爱的 macOS 功能，但如果你想在安装了 Windows10 的 MacBook 笔记本上使用 Mac 触控板，很多手势操作并不支持，或者体验很差。</p><p>微软早在 2013 年就增加了触控板手势，而且有各种非官方驱动程序允许它们在 Mac 上使用，但直到现在，苹果官方才通过 Boot Camp（6.1.15 版本）为它们提供了标准支持。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/a26edcfe-bb1d-4638-a950-f1d719d7326f.jpg@s_2,w_820,h_410" w="1440" h="720" title="苹果部分 Mac 通过 Boot Camp 安装 Win10 ，已支持 Windows 精密触控板手势" srcset="https://img.ithome.com/newsuploadfiles/2021/6/a26edcfe-bb1d-4638-a950-f1d719d7326f.jpg 2x" width="1440" height="410" referrerpolicy="no-referrer"></p><p>网友报告称，它们比非官方的驱动程序工作得更好。</p><p>这个更新提供了：</p><p>- 支持 Windows Precision 精密触控板</p><p>- 支持单点点击，右下角点击右键，向下滑动向上滚动，以及三个和四个手指手势</p><p>苹果公司的一份支持文件解释说，精密手势只适用于装有 T2 芯片的 Mac 设备。</p><p>在基于英特尔的 Mac 上的 Windows 10 中，你可以设置触控板选项，包括点击、二次点击、拖动等。根据你的触控板内容，你只看到这些配置选项中的一部分。</p><p>IT之家获悉，Windows 10 中对微软精密触控板驱动程序的支持适用于装有苹果 T2 芯片的 Mac 电脑。如果你的 Mac 支持这些驱动程序，你可以使用 Windows 设置精密触控板选项；你也可以使用 Boot Camp 控制面板设置触控板选项。</p><p><strong>包括针对以下设备进行了适配：</strong></p><ul class=" list-paddingleft-2"><li><p>iMac (Retina 5K，27 英寸，2020 年)</p></li><li><p>iMac Pro</p></li><li><p>Mac Pro (2019 年)</p></li><li><p>Mac Pro (机架，2019 年)</p></li><li><p>Mac mini (2018 年)</p></li><li><p>MacBook Air (Retina，13 英寸，2020 年)</p></li><li><p>MacBook Air (Retina，13 英寸，2019 年)</p></li><li><p>MacBook Air (Retina，13 英寸，2018 年)</p></li><li><p>MacBook Pro (13 英寸，2020 年，两个 Thunderbolt 3 端口)</p></li><li><p>MacBook Pro (13 英寸，2020 年，四个 Thunderbolt 3 端口)</p></li><li><p>MacBook Pro (16 英寸，2019 年)</p></li><li><p>MacBook Pro (13 英寸，2019 年，两个 Thunderbolt 3 端口)</p></li><li><p>MacBook Pro (15 英寸，2019 年)</p></li><li><p>MacBook Pro (13 英寸，2019，四个 Thunderbolt 3 端口)</p></li><li><p>MacBook Pro (15 英寸，2018 年)</p></li><li><p>MacBook Pro (13 英寸，2018 年，四个 Thunderbolt 3 端口)</p></li></ul><p>你可以直接在 Windows 中定制选项，或者在 Boot Camp 控制面板中定制。苹果为这两者提供了说明，并指出以下内容是默认设置的。</p><blockquote><p>“用单个手指点击来单点”</p><p>“按下触控板的右下角来右击”</p><p>“捏一下来缩放”</p><p>默认的滚动方向是“向下滑动向上滚动”（自然滚动方向）。</p><p>默认的触控板灵敏度设置是“中等灵敏度”。如果你使用触控板进行游戏，请将设置改为更高的敏感度水平。</p></blockquote>
