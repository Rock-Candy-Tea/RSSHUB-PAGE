<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 8 日消息 <a target="_blank" href="https://win11.ithome.com/" class="s_tag">Windows 11</a> 已经到来，正式版将在今年晚些时候或明年年初开始向消费者免费推出。如果你遇到了硬件、应用程序的问题，或者就是不喜欢 Windows 11，微软称都可以轻松回滚到 <a target="_blank" href="https://win10.ithome.com/" class="s_tag">Windows 10</a>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/8b7891e9-035b-41ef-8b2d-0b6140dad3e3.jpg" w="696" h="392" title="微软：升级 Win11 十天内可轻松降级到 Win10" width="696" height="392" referrerpolicy="no-referrer"></p><p>与 Windows 10 以前的功能更新相比，Windows 11 有着很大的视觉升级，包括新的开始菜单、通知中心、系统托盘、重新设计的任务栏和圆角。根据联想和微软的说法，如果你想恢复到 Windows 10，<span class="accentTextColor">你将有 10 天的时间进行原状态（in-place）降级。</span></p><p>如果用户在过去 10 天内升级到 Windows 11，降级过程是非常直接的。如果超过了 10 天，那么这个过程就稍微复杂一些，因为需要清洁安装之前的操作系统（即 Windows 10）来卸载 Windows 11。</p><p>“一旦你升级到 Windows 11，你有 10 天时间利用回滚功能移回 Windows 10，同时保留之前的文件和数据。如果在 10 天之后，你需要备份你的数据，并进行‘清洁安装’以移回 Windows 10，”联想的支持文件中写道，该文件是根据微软的意见编写的。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/debca395-f94a-444d-93b7-d99f38f6c62d.jpg@s_2,w_820,h_596" w="968" h="704" title="微软：升级 Win11 十天内可轻松降级到 Win10" srcset="https://img.ithome.com/newsuploadfiles/2021/7/debca395-f94a-444d-93b7-d99f38f6c62d.jpg 2x" width="968" height="596" referrerpolicy="no-referrer"></p><p>如果你升级到 Windows 11 后不到 10 天，可以很容易地降级到 Windows 10。<span class="accentTextColor">该功能是通过 Windows 内置的恢复和备份镜像实现的，用户可以通过设置 > 更新和安全 > 恢复来进行降级。</span></p><p>然而，如果已经超过了 10 天，那么将无法进入恢复选项，但仍然可以恢复旧的操作系统。正如微软和联想所指出的，<span class="accentTextColor">可以通过执行清洁安装来恢复到 Windows 10。</span></p><p>IT之家了解到，要进行 Windows 10 的清洁安装，可以使用微软的媒体创建工具，这个工具将允许你重新安装 Windows 10 并制作一个可启动的 U 盘。然而，这个过程将删除你的应用程序和设置，这意味着你需要手动备份文件和应用程序。</p><p>如果你不喜欢媒体创建工具，也可以手动下载 Windows 10 ISO，并手动创建自己的 USB 驱动器。你也可以在另一个分区上重新安装 Windows 10，并在需要时重新激活许可证。</p>
