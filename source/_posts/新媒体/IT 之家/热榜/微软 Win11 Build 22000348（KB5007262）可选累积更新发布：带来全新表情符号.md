<p data-vmark="79a8"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 23 日消息，今天微软为 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 发布了一个新的可选累积性更新（KB5007262，版本号 22000.348）。然而，<span class="accentTextColor">与大多数累积性更新不同，这个更新有新功能</span>，带来了早在 7 月宣布的新的表情符号。</p><p data-vmark="c020"><img src="https://img.ithome.com/newsuploadfiles/2021/11/5750bff0-0749-4da5-827b-f72fd0ad2745.png" w="640" h="461" title="微软 Win11 Build 22000.348（KB5007262）可选累积更新发布：带来全新表情符号" width="640" height="461" referrerpolicy="no-referrer"></p><p data-vmark="da86">Fluent 风格 emoji 是此次更新的主角，微软公司解释说，通过这次更新，<span class="accentTextColor">所有 Segoe UI 表情符号都将转为美丽的 Fluent 2D 风格，同时还引入了 Emoji 13.1。</span></p><p data-vmark="c67b">Windows 设计团队的项目经理 Judy Safran-Aasen 说，“今天在 Windows 11 中发布的这些新的 Fluent 风格的表情符号给人以个性化和熟悉的感觉，这是我们 Windows 11 的两个设计原则。一个由表情符号设计师、项目经理、字体专家和开发人员组成的团队专注于创建这个新的表情符号系统。从调色板到模块化功能，设计师们建立了一个可以扩展到 Unicode 表情符号扩展集的系统。”</p><p data-vmark="c9b7">IT之家了解到，安装更新后，按下 Windows 键 + 句号键，就可以在表情符号选择器内使用新的表情符号。</p><p data-vmark="e436">除了新的表情符号，此次更新还带来了大量修复内容，部分亮点如下：</p><ul class=" list-paddingleft-2"><li><p data-vmark="a2ad">更新了一个问题，当你在使用输入法编辑器（IME）复制和粘贴文本时，会导致 Internet Explorer 停止工作。</p></li><li><p data-vmark="7abe">更新了一个问题，即在通知区显示 iFLY 简体中文 IME 图标的背景不正确。</p></li><li><p data-vmark="1233">更新了一个阻止显示文件资源管理器和桌面快捷方式菜单的问题。当你选择使用单次点击来打开一个项目时，这个问题经常发生。</p></li><li><p data-vmark="9e6a">改进了任务栏上图标的动画性能。</p></li><li><p data-vmark="33f4">更新影响蓝牙音频设备的音量控制问题。</p></li><li><p data-vmark="9a7b">更新了一个导致文件资源管理器在你关闭文件资源管理器窗口后停止工作的问题。</p></li><li><p data-vmark="4d7f">更新了一个对某些视频显示不正确的隐藏式字幕阴影的问题。</p></li><li><p data-vmark="0db1">更新了一个自动从设备上删除塞尔维亚语（拉丁语）Windows 显示语言的问题。</p></li><li><p data-vmark="36e7">更新了一个问题，当你悬停在任务栏上的图标上时，会导致闪烁；如果你应用了高对比度的主题，这个问题会发生。</p></li><li><p data-vmark="4ea8">更新了一个问题，在某些条件下，当你使用任务视图、Alt-Tab 或 Snap Assist 时，键盘焦点矩形会被阻止可见。</p></li><li><p data-vmark="84c6">更新了一个问题，该问题可能导致 Windows 混合现实在你戴上耳机时启动。即使你关闭了“当我的耳机的存在传感器检测到我正在佩戴时启动混合现实门户”选项，这个问题也会发生。</p></li><li><p data-vmark="ecea">更新了一个问题，该问题可能导致你的设备在你插入打印机后报告说它没有检测到打印机。</p></li><li><p data-vmark="d485">更新了一个可能导致您的设备暂时失去音频的问题。</p></li><li><p data-vmark="bc4e">更新了一个可能导致某些可变字体显示不正确的问题。</p></li><li><p data-vmark="b450">更新了一个问题，当你使用 Meiryo UI 字体和其他垂直字体时，显示字母或字符的角度不对。这些字体在日本、中国或亚洲其他国家经常使用。</p></li><li><p data-vmark="43d7">更新了一个导致某些应用程序对输入停止响应的问题。这个问题发生在有触摸板的设备上。</p></li><li><p data-vmark="9085">增加了一个选项，让你选择是否在 Windows 功能更新后的第一个小时内自动打开焦点辅助功能。</p></li><li><p data-vmark="871f">更新了一个影响 Xbox One 和 Xbox 系列音频外设的音频失真问题，并在你使用它们的空间音频时发生。</p></li><li><p data-vmark="f8f5">更新了 Windows 表情符号的几个方面。作为迭代和持续工作的一部分，我们为这个版本做了以下改进：1、将所有表情符号从 Segoe UI 表情符号字体更新为 Fluent 2D 表情符号样式；2、包括对 Emoji 13.1 的支持，其中：更新了表情符号字典；增加了在所有支持的语言中搜索 Emoji 13.1 的功能；更新了 Emoji 和更多面板，因此你可以在你的应用程序中输入表情符号。</p></li></ul><p data-vmark="6a07">IT之家了解到，用户可以通过 Windows Update 检查更新安装，该更新是可选的，意味着不会自动安装。如果你不接受，这些修复和变化将被纳入下个月的“补丁星期二”更新，这是强制性的。</p>
