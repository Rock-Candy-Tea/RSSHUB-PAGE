<p data-vmark="d0c0"><img src="https://img.ithome.com/newsuploadfiles/2021/10/d556df7c-c4fb-451f-8a85-61c8bd048978.png" w="1080" h="540" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="410" referrerpolicy="no-referrer"></p><p data-vmark="24cd">十一假期期间，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 正式版如期发布，包括笔者在内的不少之前的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 老用户，都第一时间更新到了新版 Windows。</p><p data-vmark="b9f9">或许是因为 Windows 10 是很多人在过去六年一直在用着的操作系统，所以当 Windows 11 发布之后，就像 Windows 10 刚发布时很多 <a class="s_tag" href="https://www.win7china.com/" target="_blank">Windows 7</a> 系统用户困惑的，如今很多用户开始关心这个问题：「Windows 11 值得升级吗」？</p><p data-vmark="2f6c">在一周的 Windows 11 首个正式版使用体验过后，我也在这篇文章中整理了目前 Windows 11「感知很强」的各项新变化，看完这篇长期使用体验之后，或许关于上面的那个问题，你也可以得出自己的结论。</p><p data-vmark="8c2e"><img src="https://img.ithome.com/newsuploadfiles/2021/10/60eefc1e-1e95-4a31-8b75-8305ab13630d.png" w="1080" h="813" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="617" referrerpolicy="no-referrer"></p><h2 data-vmark="f111">任务栏：不只是「居中」</h2><p data-vmark="891b">任务栏样式设计肯定是 Windows 11 更新最明显的设计 变化之一；与 Windows 11 系统本身强调的简洁设计相同，图标居中的任务栏也给人一种更加熟悉的感觉，操作逻辑上也更接近目前其他主流操作系统的「桌面 - 应用菜单」的关系。</p><p data-vmark="d7a9"><img src="https://img.ithome.com/newsuploadfiles/2021/10/3f7cd2d5-6512-42b8-ae15-b3fe48bf49cd.png" w="1080" h="568" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="431" referrerpolicy="no-referrer"></p><p data-vmark="7652">在 Windows 11 中，<span class="accentTextColor">微软为几乎所有系统图标都重新绘制了动效</span>，包括「开始」「任务视图」「搜索」等已经在 Windows 10 中非常熟悉的图标都有适配，同时应用窗口缩放也全局支持非线性动画；考虑到近年来 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 系统非线性动画已经成为了新的竞争点，这些有让 Windows 11 新的视觉体验加分不少：毕竟在很多时候，动画效果的优劣都是系统流畅性最直观的反馈。</p><p data-vmark="b4a5"><img src="https://img.ithome.com/newsuploadfiles/2021/10/a0f7424a-5493-4753-9033-13b22acf2cca.gif" w="736" h="148" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="736" height="148" referrerpolicy="no-referrer"></p><p data-vmark="61a0">▲ Windows 11 任务栏新增交互动画｜图源：Windows Central</p><p data-vmark="1ed9">虽然新的任务栏样式让 Windows 看起来更具设计感了不少，但就功能性而言，Windows 11 的任务栏相比于 Windows 10 确实有着明显倒退，<span class="accentTextColor">包括快捷方式，工具栏等在 Windows 10 中常见的任务栏设置</span>，在目前的 Windows 11 中都不见踪影。</p><h2 data-vmark="22d5">开始菜单：经典「磁贴」不再，简洁第一</h2><p data-vmark="4b61"><img src="https://img.ithome.com/newsuploadfiles/2021/10/162dcf79-4cfa-4591-8af5-4ce17475a03e.png" w="1080" h="720" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="547" referrerpolicy="no-referrer"></p><p data-vmark="5042">上面这张图中展示的是 Windows 10 一直以来沿用的开始菜单样式，其中最具标志性的就是延续自 Window 8 系统的「磁贴」应用图标设计。这一设计到了 Windows 10 时代一开始是一种跨平台通用设计 —— 当时还是「第三大手机操作系统」的 Windows 10 Mobile 桌面同样是这样的设计，而适配这一特性的 UWP 应用则可以让磁贴同时在包括手机/PC 等一众 Windows 设备上显示同样的效果。</p><p data-vmark="6c63">但到了 Windows 11 发布的 2021 年，不仅微软力推的 UWP 应用生态不见起色。连 Windows 10 Mobile 手机系统本身都已经被微软放弃，这套标志性的图标设计在 Windows 11 更新中被取消也算是情理之中。</p><p data-vmark="0e9c"><img src="https://img.ithome.com/newsuploadfiles/2021/10/e7df9b09-51b3-4016-98d2-a6d74d843227.png" w="1080" h="981" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="745" referrerpolicy="no-referrer"></p><p data-vmark="3622">▲ 搭载 Windows 10 Moblie 的 Lumia 手机</p><p data-vmark="f7b4">在 Windows 11 中，微软并没有给出比这一套系统启动/查找应用效率更高的方案：查找应用被收纳在占据屏幕空间不到四分之一的应用菜单中，相比于 Windows 10 并没有明显提升。平心而论，如果是从 Windows 10 升级而来的老用户，对于 Windows 11 这套新的开始菜单一开始很有可能会陷入混乱，相对较多的改动也意味着，你可能需要重新花上几个小时，去适应一下这种新的设计。</p><p data-vmark="f5ae"><img src="https://img.ithome.com/newsuploadfiles/2021/10/96615dac-95cd-46de-af60-8c44db3ce3d4.png" w="800" h="450" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="800" height="450" referrerpolicy="no-referrer"></p><p data-vmark="478b">但如果是之前没有用过 Windows 操作系统的用户，对于 Windows 11 更加简洁的开始菜单设计，可能会更加容易上手，掌握对应的应用查找方式也并不难。</p><h2 data-vmark="f6dd">应用商店：更开放，也更「混乱」</h2><p data-vmark="6866">Windows 自带的应用商店在 Windows 10 时代一直是比较尴尬的存在：主要原因并非不好用，而是因为其收纳的应用太少，日常常用的应用很少能在 Windows 自带应用商店中找到的，往往只有在下载微软第一方游戏或是一些 Windows 小工具时才才会启动，这也是 Windows 应用商店对于很多用户来讲「存在感稀薄」的原因。</p><p data-vmark="8c79"><img src="https://img.ithome.com/newsuploadfiles/2021/10/87912ba2-0a2a-409e-aab9-e9e48b22cd7d.png" w="800" h="533" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="800" height="533" referrerpolicy="no-referrer"></p><p data-vmark="c873">而在 Windows 11 中，应用商店开放了很多之前严苛的应用上架限制，允许开发者将大多数 Windows 应用提交到商店。之后的 Windows 11 中不仅会有包括第三方<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>以及创意/开发工具上架，甚至还加入了对于第三方应用商店的下载支持。换言之，在不远的 Windows 11 更新中，你可以在自带应用商店中下载到包括 Steam、Epic 等游戏应用商店。虽然这事听起来很「套娃」，但确实可以有效降低因为直接在互联网下载安装包，导致电脑中病毒的风险，Windows 用户也确实很需要一个能够「发现好应用」的系统平台。</p><p data-vmark="6881"><img src="https://img.ithome.com/newsuploadfiles/2021/10/88567d55-9b57-4f73-927d-edd960eacb78.png" w="1080" h="720" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="547" referrerpolicy="no-referrer"></p><p data-vmark="4675">虽然长远来看，这一改变当然是好事，但目前版本下的 Windows 应用商店，给人的感觉仍然是「混乱」：不仅有之前的 UWP 应用，也有 win32 转制应用、PWA 应用、以及数量仍然不多的 Windows 原生应用，按照微软官方公布的时间线，近期 Windows 11 也将在应用商店中整合 Android 应用支持，应用的分类以及整合，都需要微软在之后的版本更新中陆续解决。</p><p data-vmark="6788"><img src="https://img.ithome.com/newsuploadfiles/2021/10/a3e1e015-8c11-46cd-b9e5-406957ec58a5.png" w="1080" h="720" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="547" referrerpolicy="no-referrer"></p><h2 data-vmark="beb3">通知中心/小组件</h2><p data-vmark="553a">通知中心在 Windows 11 中也做了一些微小的改进工作：Windows 11 版本的通知中心被分为了通知与系统设置两部分，设置部分可以直接通过点击 Wi-Fi、音量/亮度等按钮调节，Wi-Fi 则会展开一个新的选择界面，不必再跳转到设置中设置。</p><p data-vmark="ae58">虽然这个改进相对并不是太大，但都更接近智能手机系统中「控制中心」的定位，这也反映出 Windows 11 试图简化系统操作的努力，将常用的系统设置选项整合在一个界面中，会让系统体验不那么混乱和复杂，而这也正是 Windows 11 设计初衷想要改变的。</p><p data-vmark="9fc6"><img src="https://img.ithome.com/newsuploadfiles/2021/10/0f2a9dfd-c78a-40db-b56c-d98a3ba21da4.png" w="1080" h="1005" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="763" referrerpolicy="no-referrer"></p><p data-vmark="bfde">除了通知中心变得更像手机操作系统之外，在 Android/iOS 都重新设计桌面小组件系统之后，微软也开发了自己的小组件系统并将其带到了 Windows 11 上。但这个新的小组件系统并不能直接添加到桌面上，而是在屏幕左侧的一个专门用于收纳小组件的页面中展示，当你按下 Win + W 快捷键或是在触摸屏向右滑动时，这个界面就会直接从侧边划出，这样的设计与 macOS 中通过滑动呼出的控制中心也很雷同。</p><p data-vmark="31b4"><img src="https://img.ithome.com/newsuploadfiles/2021/10/7629063e-3396-4a2e-8a49-ad28a486c51b.png" w="1080" h="608" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="462" referrerpolicy="no-referrer"></p><p data-vmark="f8db">不过和微软其他第一方适配特性一样，这套小组件系统目前也只有数款 Windows 11 自带的系统服务支持，虽然你可以在这里查看到包括天气、待办事项以及新闻等颇为实用的小组件，但很难期待未来会有更多第三方应用去适配这一新特性，微软也没有苹果在 iOS/macOS 上通用小组件设计的开发者生态，去快速推动让这个功能更加实用。</p><p data-vmark="4539">实际上在体验中，除了一开始的新鲜感使然，会让我会去刻意启动它之外，就再也没有开启过这个小组件界面了，一部分原因是这些小组件目前的实用性确实不高，另一方面，这套小组件系统其实与微软的账号同步体系高度融合，如果你不是微软系服务的重度用户，那这些小组件功能对于你而言，很可能也不会太重要。</p><p data-vmark="0289"><img src="https://img.ithome.com/newsuploadfiles/2021/10/d218a2e3-3406-4984-b364-01819b248f98.png" w="800" h="533" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="800" height="533" referrerpolicy="no-referrer"></p><h2 data-vmark="7147">多任务操作优化</h2><p data-vmark="2ac5">虽然上手第一时间很多用户可能不会注意到，但在一段时间的体验之后，Windows 11 给我留下印象最深的功能点，其实是对于多任务操作加入的新优化：在 Windows 10 中，想要在同一个屏幕上打开多个窗口铺满，用户需要将应用窗口拖拽到桌面侧边才能实现，但在 Windows 11 中，这一操作可以直接在应用窗口的最大化按钮上悬停就能实现。</p><p data-vmark="ea0d">同时，微软还在这里提供了六种不同的窗口排布样式，你可以视情况选择其中一种快速给应用分组显示，对于大屏用户来讲，这一改进能让原本就很方便的 Windows 分屏操作效率进一步提升。</p><p data-vmark="c42c"><img src="https://img.ithome.com/newsuploadfiles/2021/10/07632a78-1d74-41f2-9303-85660bc36df6.png" w="1080" h="660" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="501" referrerpolicy="no-referrer"></p><p data-vmark="ccb3">当然，Windows 10 中的应用贴边分屏这一操作还是被保留了下来，而且应用窗口预览动画也变成了 Arcrylic 风格模糊的非线性动画，在契合 Windows 11 新设计语言的同时，看起来也现代了不少。</p><p data-vmark="882c"><img src="https://img.ithome.com/newsuploadfiles/2021/10/3c81b08c-d02a-4a93-aeb5-a17a7bfb16d5.gif" w="800" h="532" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="800" height="532" referrerpolicy="no-referrer"></p><p data-vmark="d03e">至于在 Windows 10 中不怎么实用的时间轴功能，到了 Windows 11 中被一个改进的多任务后台界面取而代之。除此之外多个虚拟桌面这一概念在 Windows 11 中也被重新设计，而且一些新增功明显能看出 macOS 的虚拟桌面的影子：你还可以通过给每个虚拟桌面单独设置不同的壁纸来区别用途，这些一次设置完成之后即使重启电脑也仍然会被保留下来。相比 Windows 10 来讲无论是使用效率还是易用性上来讲都提升不少。</p><p data-vmark="1d3d"><img src="https://img.ithome.com/newsuploadfiles/2021/10/eda0ab62-4ddf-4fb4-b667-1211470b73fb.png" w="800" h="533" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="800" height="533" referrerpolicy="no-referrer"></p><h2 data-vmark="e389">Windows 11 值得升级吗？</h2><p data-vmark="2ade">2015 年的微软 Ignite 开发者大会上，微软开发者杰里-尼克松（Jerry Nixon）曾公开表示：Windows 10 会是 Windows「最后一个版本」；未来的 Windows 系统，将作为一个不断迭代的平台保持更新，保证其能顺应时代潮流，跟上最新的系统发展趋势。</p><p data-vmark="8a30">虽然最后这被证明只是一次媒体的过度解读，但将 Windows 系统更新视为一种服务来运营，却是微软在过去数年中一直坚持的战略，Windows 11 的发布，也并未改变这一策略。</p><p data-vmark="9379">实际上，即使当下 Windows 11 已经是微软宣传中的「正式版」，但在很多细节上看起来都更像是一个尚未准备万全的开发版本。这其实正是 Windows 11 的矛盾之处：微软希望让 Windows 重新跟上时代，但又想避免重蹈之前各种因为「超前设计」导致的问题。</p><p data-vmark="de64">「升级了，但又没升级」这就是 Windows 11 带给我的感觉，毕竟，让一个有着近 40 年沉积历史的操作系统一直跟上最新的时代变化，同时避免让老用户感觉过于陌生，对于微软来讲，确实是一个复杂且棘手的挑战。</p><p data-vmark="e51f"><img src="https://img.ithome.com/newsuploadfiles/2021/10/749bcd6f-3b77-40fc-8592-f49da3fb69b5.png" w="1080" h="608" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="462" referrerpolicy="no-referrer"></p><p data-vmark="394c">▲ Windows 11 UI 设计语言「四代同堂」｜ 图源 @libxzr</p><p data-vmark="12a4">除了主观升级意愿之外，很多用户在尝试升级到 Windows 11 时，还遇到了由于 TPM 2.0 硬件导致无法安装更新的问题，这个问题一度让 Windows 11 冲上微博热搜，以及 Windows 11 官方不再支持搭载酷睿七代处理器以及 AMD Zen 第一代的设备。这些由于设备原因导致的问题也提高了很多用户升级至新系统的难度。</p><p data-vmark="9bed">此外，随着 Windows 11 正式发布，之后出厂预装 Windows 11 的设备也会越来越多，但这并不意味着 Windows 10 的时代就此结束，在 2025 年之前，微软都会为 Windows 10 提供支持，所以你大可放心地继续使用你现在熟悉的系统。</p><p data-vmark="3e72"><img src="https://img.ithome.com/newsuploadfiles/2021/10/ce486725-4124-4363-8e51-cc3b8384435b.png" w="1080" h="540" title="微软 Win11 正式版一周体验：升还是不升，可以再等等" width="1080" height="410" referrerpolicy="no-referrer"></p><p data-vmark="cdaf">诚然，Windows 11 中很多新变化都是 WIndows 用户一直以来呼声很高的功能，生产力的提升，加上新的系统级主题引擎，都让 Windows 11 更加适合当前用户的使用需求，但如果你对于上述这些升级点都不是很感冒，或你是将系统稳定性放在第一位的用户，也可以暂时不升级 Windows 11 ，等待之后的季度更新，让系统本身的各项优化完善，或许不失为一种更为稳妥的选择。</p>
