<p data-vmark="4cde">不久前，微软发布了新版操作系统 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>，其中的开始菜单又一次让大家“上了头”。遥想当年的 Win8、<a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 刚刚发布时也是各种的不适应，最后还是一大波菜单定制工具面世，才解了微软的围。</p><p data-vmark="45f3">日前著名菜单定制软件公司 Stardock 发布了最新版 Start 11，其中最大的看点不是帮 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 找回经典菜单，而是让 Win10 用户也能体验上和 Win11 一样的新菜单！</p><blockquote><p data-vmark="f75d">Start 11：<span class="link-text-start-with-http">https://www.stardock.com/products/start11/</span></p></blockquote><p data-vmark="a445"><img src="https://img.ithome.com/newsuploadfiles/2021/11/579fba04-4891-416b-b521-29f7d3760673.jpg" w="700" h="704" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="704" referrerpolicy="no-referrer"></p><p data-vmark="fa39">Start11 的界面延续了先前风格，进入后会首先打开一个风格页面。前几种大家应该都很熟悉了，倒是最后一个“Windows 11 风格”比较吸睛，OK！就点它了。</p><p data-vmark="852c"><img src="https://img.ithome.com/newsuploadfiles/2021/11/0f1e9e3b-a643-415f-b807-1d7c00c49ce3.jpg" w="700" h="497" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="497" referrerpolicy="no-referrer"></p><p data-vmark="79a4">选好风格后，接下来就是居中任务栏的选项。可以看出，作者是在尽可能地模仿 Win11 的调调。这里特别说一句，如果你的 Win10 已经打开了搜索栏，Start 11 还会同时提醒你是否将搜索栏隐藏，以便更好地适配新菜单风格。</p><p data-vmark="a539"><img src="https://img.ithome.com/newsuploadfiles/2021/11/e1733675-9029-4701-b585-192d5063a1b4.jpg" w="700" h="363" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="363" referrerpolicy="no-referrer"></p><p data-vmark="f998">在每个菜单项的右侧，都会有一个小小的设置按钮，点击后可以修改当前菜单里的一些细节。以我们目前选择的 Windows 11 为例，打开后会呈现一个功能丰富的配置选项。大到最近使用文件、快捷方式尺寸，小到菜单透明度、菜单背景图，里面的内容几乎应有尽有，甚至比原版 Win11 还要全面。</p><p data-vmark="cfde"><img src="https://img.ithome.com/newsuploadfiles/2021/11/e7bfaf0b-82cd-4be6-86bf-c4e8e3e10acf.jpg" w="700" h="470" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="470" referrerpolicy="no-referrer"></p><p data-vmark="a2d9"><img src="https://img.ithome.com/newsuploadfiles/2021/11/138f3cd7-c533-4c78-9806-56d4bfec3fa7.jpg" w="700" h="470" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="470" referrerpolicy="no-referrer"></p><p data-vmark="b471">这里单独说一下右键，在原版 Win11 中，开始菜单的右键已经被微软屏蔽。比如当我们右击菜单里的程序或文档时，是无法看到里面的 Jumplist，这无疑会给日常操作带来很多不便。而在 Start 11 中，右键功能同样也能正常使用，这也是相比原版一处比较不错的设计。</p><p data-vmark="59a5"><img src="https://img.ithome.com/newsuploadfiles/2021/11/79ffb5ae-1d0f-4d27-8800-d3130adba0fd.jpg" w="700" h="690" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="690" referrerpolicy="no-referrer"></p><p data-vmark="2d5c">开始按钮没什么可说的，支持用户手工设置一个自己喜欢的开始图标。旁边的“下载”按钮应该是个新选项，点击后可以到网站上挑选更多喜欢的图标。</p><p data-vmark="6b87"><img src="https://img.ithome.com/newsuploadfiles/2021/11/c73d391e-8d07-4dbb-bc3d-10f31221bfea.jpg" w="700" h="849" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="849" referrerpolicy="no-referrer"></p><p data-vmark="9da3">任务栏的设置不是很多，除了刚刚提到的居中式任务栏外，还有一个看点就是它的透明任务栏了。当然这个任务栏还是基于 Win10 基础之上，像 Win11 中无法拖拽快捷方式、无法使用右键的问题是不存在的。</p><p data-vmark="9eaa"><img src="https://img.ithome.com/newsuploadfiles/2021/11/04f01dcf-b2e4-4591-abc9-5267f81b732c.jpg" w="700" h="470" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="470" referrerpolicy="no-referrer"></p><p data-vmark="c3e1">搜索是 Start 11 的另一大看点，和之前版本直接调用 Windows 搜索不同。此次 Start 11 带来了一个自家的搜索工具（像 Win11 一样直接显示在开始菜单上）。和 Win10 内置搜索器一样，Start 11 的搜索器是分门类进行显示，实际使用中会区分为“应用程序”、“设置”、“文档”、“图片”、“音乐”、“网络”等多个门类，每个门类右侧会设有一个展开按钮，以便更好地查看该类别的结果。</p><p data-vmark="f62a">设置面板中，用户可根据需要开启是否搜索文件内容以及是否搜索 Fences（Stardock 的一款桌面管理软件），同时 Start 11 的搜索效果也算不错，在精准度方面并不输于原版 Windows。</p><p data-vmark="197e"><img src="https://img.ithome.com/newsuploadfiles/2021/11/f9380361-5bfb-45b5-a841-cd962305e303.jpg" w="700" h="808" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="808" referrerpolicy="no-referrer"></p><p data-vmark="66f0"><img src="https://img.ithome.com/newsuploadfiles/2021/11/6e4e5e60-4358-47b6-be95-29af103fe4be.jpg" w="700" h="543" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="543" referrerpolicy="no-referrer"></p><p data-vmark="0eec"><img src="https://img.ithome.com/newsuploadfiles/2021/11/4a28ec5c-feda-4afb-a7c0-b8272e307ecf.jpg" w="700" h="339" title="还没收到 Win11 推送？这款小工具让你 Win10 先用上新版菜单" width="700" height="339" referrerpolicy="no-referrer"></p><p data-vmark="7eea">和先前版本一样，Start 11 也提供了一些与原生菜单的共存选项，用户可分别设定 Start 11 与原生菜单的激活热键。甚至还单独分出了一个选项，用便控制全屏模式下这些热键是否有效。</p><p data-vmark="bf68"><img referrerpolicy="no-referrer"></p><h2 data-vmark="b11f">写在最后</h2><p data-vmark="3904">和之前的系统升级相比，此次 Win11 对于系统硬件要求较高，很多老电脑已经无缘 Win11。而 Start 11 此次推出的 Win11 风格开始菜单，算是一定程度上解决了这部分用户的痛点。同时由于它还顺带解决了部分 Win11 原版菜单里的 Bug，对于新版用户来说，其实也是一个不错的选择。</p><p data-vmark="18c1"><img src="https://img.ithome.com/newsuploadfiles/2021/11/63eb34ed-1214-4e7b-bdb6-bec49d780113.jpg" w="700" h="776" title="还没收到 Win11 推送？这款小工具让你
