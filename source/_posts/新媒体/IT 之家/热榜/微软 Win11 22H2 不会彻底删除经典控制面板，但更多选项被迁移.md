<p data-vmark="8eef"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 1 日消息，微软一直在慢慢淘汰经典控制面板，更多页面正在转移到 Windows 现代设置应用程序。看起来下一个 Windows 11 更新 22H2 —— 累积或次要功能更新会将更多页面从控制面板重定向到设置应用程序。</p><p data-vmark="2dbf" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/8/67a8c9b8-5d1e-4c72-96e6-50ee4a2a0d90.jpg" w="820" h="547" title="微软 Win11 22H2 不会彻底删除经典控制面板，但更多选项被迁移" width="820" height="547" referrerpolicy="no-referrer"></p><p data-vmark="14bf"><span class="accentTextColor">在 Windows 11 中，微软重新设计了“设置”应用并开始向新体验过渡。</span>在许多方面，Windows 11 功能升级为用户简化了许多功能，因为它将控制面板中可用的每一个小功能都迁移到 Windows 设置中。</p><p data-vmark="d303" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/8/ed7afa70-3568-40da-b66b-a6ef50413cd3.png" w="1120" h="632" title="微软 Win11 22H2 不会彻底删除经典控制面板，但更多选项被迁移" width="1120" height="463" referrerpolicy="no-referrer"></p><p data-vmark="4d7b">设置应用程序现在使用 WinUI，并为涉及到大量设计更改的现代页面做好准备。据熟悉开发情况的消息人士透露，作为微软公司长期迁移工作的一部分，微软计划在未来某个时候将所有内容从控制面板转移到设置应用程序。</p><p data-vmark="2562">在最近更新中，用户现在可以删除设备上安装的更多应用程序。当时，某些应用程序只能通过控制面板删除，因为设置应用程序无法识别某些 Windows 应用程序，包括 Win32 桌面软件。</p><p data-vmark="1f28">用户终于可以使用设置应用程序删除所有 Win32 应用程序。如前所述，此选项以前仅在经典控制面板中可用。用户还可以删除具有相互依赖关系的应用程序。例如，Steam 和在 Steam 上运行的游戏应用程序，或修复和修改 Win32 应用程序。</p><p data-vmark="6250">当然，这并不是经典控制面板的终结，因为迁移工作仍在进行中。许多关键页面都存在于控制面板中，当大多数任务栏不需要控制面板时，微软的迁移工作将接近尾声。</p><p data-vmark="d8c1">但是，整个过渡过程进展缓慢，控制面板和设置将继续在 Windows 11 中并存。</p><p data-vmark="69f3">报告显示，微软已经取消 Windows 11 23H2 版本，因为该公司计划在开发下一代操作系统（即 Windows 12），此前将专注于较小的功能更新。</p>
