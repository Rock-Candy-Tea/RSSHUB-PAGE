            <p data-vmark="7402"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 8 日消息，微软 .NET 官网已推出<span class="accentTextColor">简体中文版</span>，目前支持英语、中文、日语三种语言。</p><p data-vmark="09fb"><img src="https://img.ithome.com/newsuploadfiles/2021/12/93b246f5-7514-435f-8389-59b7f3c89970.png" w="1440" h="506" title="微软 .NET 中文官网正式上线" width="1440" height="288" referrerpolicy="no-referrer"></p><p data-vmark="34cd">.NET 是一个统一平台，使开发人员能够跨操作系统和平台构建世界一流的应用程序。</p><p data-vmark="b79a"><span class="accentTextColor">.NET 中文官网汉化的内容包括功能介绍、学习内容、文档等</span>，.NET 新手终于能够获得更正式的、更官方的学习内容，.NET 开发⼈员，则能通过官网学习更多.NET 的体系结构，企业也能够更好地了解 .NET 平台。</p><p data-vmark="1fee"><img src="https://img.ithome.com/newsuploadfiles/2021/12/4c433f5a-c7cb-43d3-9adf-98dce3bfc363.png" w="1440" h="686" title="微软 .NET 中文官网正式上线" width="1440" height="391" referrerpolicy="no-referrer"></p><p data-vmark="b3c6"><img src="https://img.ithome.com/newsuploadfiles/2021/12/7e103d79-0cc0-46f8-b970-9ab51a4f4347.png" w="1440" h="621" title="微软 .NET 中文官网正式上线" width="1440" height="354" referrerpolicy="no-referrer"></p><p data-vmark="6dbe">IT之家了解到，微软近期还推出了 <a href="https://techcommunity.microsoft.com/t5/net/ct-p/dotnet" target="_blank">.NET 官方社区论坛</a>，不过该论坛的界面还没有汉化。</p><p data-vmark="d23c"><img src="https://img.ithome.com/newsuploadfiles/2021/12/3269d12a-c3e6-4981-9351-8870bc43616f.png" w="1440" h="682" title="微软 .NET 中文官网正式上线" width="1440" height="388" referrerpolicy="no-referrer"></p><p data-vmark="e405">目前微软已推出 .NET 6 正式版，该版本对 .NET 的所有部分都进行了重大改进，包括对云、桌面和移动应用的改进，开启了 .NET 的统一。</p><p data-vmark="7a8c"><img src="https://img.ithome.com/newsuploadfiles/2021/11/399c0960-0bcd-4678-9a54-5a6b6afbf34d.png" title="微软 .NET 中文官网正式上线" referrerpolicy="no-referrer"></p><p data-vmark="c5d7"><strong>微软 .NET 中文官网：</strong><a href="https://dotnet.microsoft.com/zh-cn/?ocid=_reactor_dec_shanghai_social_organic_youtube_dotnet-klo" target="_blank">点此查看</a></p>
