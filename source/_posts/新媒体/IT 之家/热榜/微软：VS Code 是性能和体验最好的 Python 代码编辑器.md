<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 16 日消息 微软近日发布了 Visual Studio Code Python 扩展的 2021 年 5 月更新，本次更新<span class="accentTextColor">引入了 Pylisce </span>作为 Python 的默认语言服务器，以及用于 Python 脚本的自动重新加载调试器。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/c5ecebb0-4828-414b-9984-40531ef28ca2.png" w="1380" h="600" title="微软：VS Code 是性能和体验最好的 Python 代码编辑器" width="1380" height="357" referrerpolicy="no-referrer"></p><p>微软 Visual Studio 的 Pylance 和 Python 项目经理 Savannah Ostrowski 表示：<span class="accentTextColor">VS Code 是有史以来性能最强、用户体验最好的 Python 编辑器</span>。</p><p>IT之家了解到，Pylance 是微软开发的 Python 语言支持扩展，可提供智能建议和类型信息，旨在帮助开发者更快速地编写代码。该扩展基于微软 2019 年发布的 Python Pyright 静态类型检查工具，可帮助开发者快速清理并自动补全代码片段。 </p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210305_195859_561.png" title="微软：VS Code 是性能和体验最好的 Python 代码编辑器" referrerpolicy="no-referrer"></p><p>微软官方表示，本次更新后，<span class="accentTextColor">Windows 和 macOS 版本的启动时间缩短了 10％，Linux 版的启动时间缩短了 20％</span>。</p><p>此外，新的 Python 扩展已支持实时调试重载功能，用户在调试时可以实时修改代码，无需重载调试器即可生效。</p>
