<p data-vmark="9336"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 22 日消息，“环境变量”是 Windows 系统中<span class="accentTextColor">简化命令行</span>的一项重要工具，然而，该工具一直都不怎么方便，需要用户打开重重界面，然后在一个很复古的窗口中进行设置。</p><p data-vmark="4f5f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2015/9/20150920_161514_469.jpg" title="微软 Win11 / Win10 终于要简化“环境变量”设置了" referrerpolicy="no-referrer"></p><p data-vmark="22a8">在 PowerToys 的 <a href="https://github.com/microsoft/PowerToys/issues/16424" target="_blank">GitHub 页面</a>，微软已经承认环境变量页面的<span class="accentTextColor">“可读性”非常差</span>，初学者很难理解路径或指定文件夹等内容。</p><p data-vmark="dc80" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/e2993af4-2a5c-4aef-8cf5-66199ae0202f.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzE3NS5wbmc=,g_3,x_17,y_17,a_0,t_100" w="1136" h="677" title="微软 Win11 / Win10 终于要简化“环境变量”设置了" width="1136" height="489" referrerpolicy="no-referrer"></p><p data-vmark="bd8d">于是，微软打算在 PowerToys 中添加一个名为“<span class="accentTextColor">环境管理器</span>”的工具，让管理环境变量变得更容易。</p><p data-vmark="574b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/c862c678-fd4d-47cd-846c-aafd050178d1.gif" w="779" h="716" alt="Command Prompt tool" title="微软 Win11 / Win10 终于要简化“环境变量”设置了" width="779" height="716" referrerpolicy="no-referrer"></p><p data-vmark="fb78">从图中可以看到，<span class="accentTextColor">该工具允许用户像管理列表一样插入环境变量</span>，并帮助用户定义运行可执行文件的路径。用户可在“用户变量”和“系统变量”下直接点击“添加变量”按钮，看起来是不是很方便。</p><p data-vmark="bd15">IT之家了解到，该工具的环境变量编辑规则与原版相同，如果需要添加多个值，则需要用分号来隔开。</p><p data-vmark="feef">微软名为 PowerToys 的免费系统实用工具套件从 Windows XP 时代复活，并于 2019 年 5 月引入 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 系统。可以用于高级用户调整和简化 Windows 操作，以提高效率。通过与 Windows 10/<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> shell 搭配使用，同时让用户可以自定义各个工作流。</p><p data-vmark="0786">目前该工具仍处在开发中，我们可以期待一下在 PowerToys 之后的版本中看到它。</p>
