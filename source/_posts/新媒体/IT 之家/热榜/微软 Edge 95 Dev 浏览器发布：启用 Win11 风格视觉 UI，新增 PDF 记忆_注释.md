<p data-vmark="8450"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 25 日消息 据 onMSFT 报道，近期，微软发布了 Edge <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a> Dev Build 95.0.1020.0 版本更新，它带来了各种 PDF 改进以及受 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 启发的小型设计 UI 变化。Edge 团队表示，这一更新将很快进入 Beta 频道。</p><p data-vmark="6424"><img src="https://img.ithome.com/newsuploadfiles/2021/9/a5fabe47-f80c-49e7-aa2c-0b6ccf0aa3c1.png" w="1440" h="714" title="微软 Edge 95 Dev 浏览器发布：启用 Win11 风格视觉 UI，新增 PDF 记忆/注释" width="1440" height="407" referrerpolicy="no-referrer"></p><p data-vmark="215f">关于微软 Edge 的 Windows 11 风格的视觉更新，该团队表示，它们仍在开发中。到目前为止，我们只注意到重新设计的最小化/最大化/关闭按钮在应用窗口的右上角，这三个图标在垂直方向上的排列也更好。</p><p data-vmark="33e5"><img src="https://img.ithome.com/newsuploadfiles/2021/9/785c0f44-ec2e-4b03-ad69-8a7c9d2c0761.png" w="515" h="152" title="微软 Edge 95 Dev 浏览器发布：启用 Win11 风格视觉 UI，新增 PDF 记忆/注释" width="515" height="152" referrerpolicy="no-referrer"></p><p data-vmark="e40f">至于 PDF 的改进，这个新的 Edge Dev 更新带来了当你在网络浏览器中重新打开 PDF 文档时，可以从你离开的地方继续进行。微软 Edge 会记住你在 PDF 中的位置，以及之前的缩放状态和布局，但如果你不喜欢，可以在设置中关闭这个功能。</p><p data-vmark="b74c">IT之家获悉，该最新版本还允许 Edge 内测人员用自由格式的文本来注释 PDF，这个功能可以用来快速填写表格或向你的 PDF 文档添加可见的注释。</p><p data-vmark="4e83">这次更新中另一个值得注意的新功能是可以用墨水注释网络截图。</p>
