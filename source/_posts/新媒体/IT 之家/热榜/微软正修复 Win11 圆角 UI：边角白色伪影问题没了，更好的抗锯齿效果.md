<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 7 日消息 外媒 Windows Latest 报道，随着 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的推出，大量的 UI 元素和菜单将采用 WinUI 2.6 或 WinUI 3 的更新。作为设计更新的一部分，Windows 11 将为桌面操作系统带回圆角 UI。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/5780627a-2f33-40ca-965c-8436df2cf6d3.jpg" w="768" h="432" title="微软正修复 Win11 圆角 UI：边角白色伪影问题没了，更好的抗锯齿效果" width="768" height="432" referrerpolicy="no-referrer"></p><p>大量最突出的 UI 元素、菜单和按钮，包括上下文菜单或弹窗都将在 Windows 11 中获得圆角 UI。微软在 Windows 11 中不再采用通常的尖锐边缘外观 —— 这是 Windows 8 操作系统的一个关键组成部分。</p><p>IT之家获悉，Windows 11 中的圆角 UI 甚至已经延伸到了传统的应用程序和页面，如控制面板和磁盘管理工具。你可能知道，Windows 11 中此前的窗口框架圆角有一些意外的白色像素/工件。因此，圆角看起来参差不齐。</p><p>值得庆幸的是，微软终于通过 Windows 11 Build 22000.120 预览版解决了这个问题。这个问题在更新中大部分得到了解决，圆角在黑暗模式下终于看起来效果不错，但在某些情况下，边框可能仍然显得有点奇怪，比如反馈中心的左上方。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/beb9144d-4204-4baf-9268-f151c4375e84.jpg@s_2,w_820,h_525" w="860" h="551" title="微软正修复 Win11 圆角 UI：边角白色伪影问题没了，更好的抗锯齿效果" srcset="https://img.ithome.com/newsuploadfiles/2021/8/beb9144d-4204-4baf-9268-f151c4375e84.jpg 2x" width="860" height="525" referrerpolicy="no-referrer"></p><p>在 Feedback Hub 上的一篇文章中，微软确认它修复了窗口边角出现白色伪影的问题。更新后，像 Steam 的聊天窗口这样的桌面应用程序现在在 Windows 11 上有了更好的抗锯齿效果。</p><p>“感谢你提交的反馈意见！我们修复了一个白色伪影的问题。我们在 Build 22000.120 中修复了窗口边角出现白色伪影的问题，该版本现在可以在 Dev 频道中使用，”微软说道。</p><h2>Windows 11 改造后的界面</h2><p>Windows 11 配备了几个新的设计功能。这包括圆角、云母主题、由 Fluent 流畅设计提供的新设计元素、新的桌面背景、新的彩色主题，以及新的图标。</p><p>据微软称，这些设计上的改进使操作系统在不同的外形因素上比以往更加协调。Windows 11 改版的目的是使界面更柔和、更友好、更不令人生畏。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/d05017c2-67cc-4691-8536-eac690ce4465.jpg@s_2,w_820,h_430" w="1200" h="630" title="微软正修复 Win11 圆角 UI：边角白色伪影问题没了，更好的抗锯齿效果" srcset="https://img.ithome.com/newsuploadfiles/2021/8/d05017c2-67cc-4691-8536-eac690ce4465.jpg 2x" width="1200" height="430" referrerpolicy="no-referrer"></p><p>因此，微软在所有应用程序中都采用了圆角的设计，以建立一款新颖的设计，并且不妨碍客户的生产力。</p><h2>Windows 11 正式版今年晚些时候发布</h2><p>Windows 11 仍处于早期预览阶段，但根据多个文件，该操作系统将在 2021 年 10 月面向消费者发布。</p><p>Windows 11 将在 2022 年初开始向满足最低配置要求的设备广泛推出。如果你的设备未能满足要求，你可以继续使用 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 系统，或者购买一台预装 Windows 11 的全新 PC。</p><p>虽然居中的“开始”菜单和圆角将是 Windows 11 系统独有的，但微软将很快把新的应用商店和更新的内置应用带到 Windows 10 系统上。</p>
