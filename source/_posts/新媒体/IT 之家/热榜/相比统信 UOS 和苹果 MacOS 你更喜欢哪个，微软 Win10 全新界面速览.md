<p>原定于今年下半年发布的 Win10 21H2（Sun Valley），再有几个月就要与我们见面了。之前我们已经陆续介绍过新版在开始菜单、窗口动效、通知中心等方面的变化。</p><p>近日，在 Windows 10 Dev 预览版 21343 中，我们意外地发现资源管理器开始更换了新图标。打开 “此电脑”的一瞬间，一种焕然一新的感觉扑面而来。接下来，就请随小编一起来看一看吧。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210329_002825_67.jpg" title="相比统信 UOS 和苹果 MacOS 你更喜欢哪个，微软 Win10 全新界面速览" referrerpolicy="no-referrer"></p><p>“太阳谷”图标首次曝光？</p><p>新图标主要体现在桌面、计算机、文件夹、我的文档、我的图片、我的音乐、我的视频、回收站、驱动器等几个方面。也就是说，当你打开 “此电脑”的时候，基本上就能看到所有的变化了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210329_003004_427.jpg" title="相比统信 UOS 和苹果 MacOS 你更喜欢哪个，微软 Win10 全新界面速览" referrerpolicy="no-referrer"></p><p>搭载新 ICON 的 “计算机”面板</p><p>和现版图标相比，新图标在色彩表现上有了很大升级。尤其是像音乐、下载、图片、视频等一些特殊图标，不再基于文件夹进行简单修改，而是分别根据各自的属性进行改造。举个例子，当你打开现版 “此电脑”时，满眼都是黄黄的颜色。而在新版中，窗口将变得更加五彩缤纷，同时异形化设计也为图标带来了更多辨识感。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210329_003017_318.jpg" title="相比统信 UOS 和苹果 MacOS 你更喜欢哪个，微软 Win10 全新界面速览" referrerpolicy="no-referrer"></p><p>新图标在小尺寸模式下效果不错</p><p>角度是新版图标的另一处改变，之前微微倾斜的 45&deg;，已经全部改成了正面。特别是驱动器和回收站，几乎原样未动，仅仅是更换了一个角度而已。我们不清楚这种角度调整究竟有何用意。但不得不说，作为一名很久以前就将 Win10 作为主力系统的老用户来说，新图标带给我的感受聊胜于无，甚至还有一些怪怪的味道。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210329_003027_271.jpg" title="相比统信 UOS 和苹果 MacOS 你更喜欢哪个，微软 Win10 全新界面速览" referrerpolicy="no-referrer"></p><p>这张图很好地展现了新旧两版图标上的异同</p><p>其实纵观 Win10 的其他对手，无论是苹果的 MacOS，还是统信的 UOS 2.0。尽管设计理念各不相同，但其实都遵循了一个最基础原则，即图标要与系统 UI 自成一派。换句话说，系统之美不光是你将图标单独拿出来没有问题，更重要的，它要和整个系统 UI 相互协调后，才能达到最终的效果。这就像去年 Win10 首次更新 Fluent Design 图标之后，由于系统 UI 未做调整，实际效果只能说是勉强。只有当几个月后，Win10 逐步推送新 UI 后，整个界面才算上了一个新台阶。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210329_003101_537.jpg" title="相比统信 UOS 和苹果 MacOS 你更喜欢哪个，微软 Win10 全新界面速览" referrerpolicy="no-referrer"></p><p>统信 UOS 2.0 图标设计</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210329_003108_896.jpg" title="相比统信 UOS 和苹果 MacOS 你更喜欢哪个，微软 Win10 全新界面速览" referrerpolicy="no-referrer"></p><p>macOS 图标设计</p><p>写在最后</p><p>当然这一次的调整，显然只是太阳谷的一个 “前奏”，后续应该会有更多经典应用换上新图标（比如上版中刚刚换上的 “记事本”）。而至于最终效果，我们暂且不做评论，一切还是等尘埃落定后再说吧！</p><p><img
