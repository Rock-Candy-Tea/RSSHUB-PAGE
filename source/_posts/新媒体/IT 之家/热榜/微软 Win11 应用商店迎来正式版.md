<p data-vmark="5e81"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 30 日消息 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的微软应用商店一直处于预览版或测试版阶段，用户可以在标题栏左上角看到“预览”两个字。</p><p data-vmark="288f">据 Windows Latest 报道，在 Windows 11 正式发布之前，微软应用商店终于迎来了正式版，<span class="accentTextColor">第一个稳定版本现在已面向 Windows Insider 用户推出</span>。</p><p data-vmark="04d7"><img src="https://img.ithome.com/newsuploadfiles/2021/9/090ba986-b80b-4555-8182-5d1071aaf572.jpg@s_2,w_820,h_512" w="986" h="616" title="微软 Win11 应用商店迎来正式版" srcset="https://img.ithome.com/newsuploadfiles/2021/9/090ba986-b80b-4555-8182-5d1071aaf572.jpg 2x" width="986" height="512" referrerpolicy="no-referrer"></p><p data-vmark="4607">微软商店的新设计与 Windows 11 升级带来的 UI 变化一致。例如，商店不再使用 WebView，而使用本机 XAML 组件进行了更新，性能更好，对触摸屏的支持也更好。</p><p data-vmark="9234">Windows 11 应用商店还提供了新的开发者策略，现在对所有第三方应用商店开放。开发者可以向市场提交未打包的桌面应用程序（Win32），支持 MSI 软件包。Epic Games 和亚马逊等已能够绕过微软的商业平台，不被微软进行抽成。</p><p data-vmark="81ef">IT之家了解到，微软官方在政策上为第三方开发者给予了极大的“自由”，允许上架 Win32、.NET、UWP、Xamarin、Electron、React Native、Java 甚至 Progressive Web Apps，而且几乎不进行抽成，使用第三方支付的开发者可得到 100% 的收入。</p>
