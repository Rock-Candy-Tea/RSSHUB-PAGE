            <p data-vmark="5763">笑不活了家人们，微博、抖音等平台展示 IP 属地后，上演了一出出好戏。今天不如一起做个乐子人，看看谁露馅谁翻车，谁能保持住人设？</p><p data-vmark="153d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/0544553c-4f2d-4bf4-ac82-689355e41509.png" w="550" h="368" title="“人在美国刚下飞机”，IP 属地显示湖南" width="550" height="368" referrerpolicy="no-referrer"></p><p data-vmark="8aa8">首先被注意到的是本地生活大 V 们。没办法，谁让他们账号名字里就带地名呢，太显眼了。</p><p data-vmark="5fdc">一看好家伙，无论是“西安吃喝玩乐资讯”还是“广州人的广州生活”又或者“重庆吃喝生活”。这些名字相似、粉丝数量几百万 + 的大号，他们都在同一个地方 —— 湖南。</p><p data-vmark="3472" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/e82858f2-fa4b-4ef3-aa35-2e01872461aa.png" w="818" h="1242" title="“人在美国刚下飞机”，IP 属地显示湖南" width="818" height="1242" referrerpolicy="no-referrer"></p><p data-vmark="4643">难道湖南人最懂世界美食？</p><p data-vmark="3e2d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/90b4c3e9-f42b-4e57-8cc4-6ee75b56c4d3.png" w="1080" h="142" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="108" referrerpolicy="no-referrer"></p><p data-vmark="6aa8">其实呢，很多这类账号会相互关注，并经常在各自评论区串门互动，抱团取暖。</p><p data-vmark="02aa" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/f63f4ccd-a802-42ba-b534-2ae00cad929f.png" w="1080" h="800" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="607" referrerpolicy="no-referrer"></p><p data-vmark="063a">不难看出，<span class="accentTextColor">这些账号背后很可能由同一 MCN 机构集中运营</span>。</p><p data-vmark="4410">MCN 指多频道网络 (Multi-Channel Network)，是通过资本和团队的支持批量生产内容, 从而实现商业变现的运作模式。根据前瞻产业研究院统计，截至 2020 年湖南 MCN 机构数量占全国 4.7%。</p><p data-vmark="b2a2" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/5a1b9325-67e3-4bf5-a50f-feec507cde62.png" w="1080" h="483" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="367" referrerpolicy="no-referrer"></p><p data-vmark="763a">对于本地生活类账号，虽然运营者不在本地，只要确实能提供本地吃喝玩乐资讯，似乎勉强可以接受。但对于那些打造定居在海外人设，主打分享海外日常生活的博主来说，就真的可以算是翻车了。</p><p data-vmark="c13f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/8d2086b8-d481-4437-b7e6-0351e94701db.png" w="1080" h="1095" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="831" referrerpolicy="no-referrer"></p><p data-vmark="a248" style="text-align: center;">△ 来自微博用户 @大风吹奏</p><p data-vmark="dc73">当然，也有 IP 属地与人设相符的，为他们点赞。</p><p data-vmark="ef2a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/1fbe65a2-6477-4394-8ee8-23d9a0aa5e7f.png" w="1080" h="978" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="743" referrerpolicy="no-referrer"></p><h3 data-vmark="4daf">主流社交平台基本开放</h3><p data-vmark="fe33">首先来看看微博的 IP 属地功能。</p><p data-vmark="8e5c">微博是最早一批测试“展示用户 IP 属地”功能的平台，今年 3 月就已经上线。4 月 28 日，微博又针对这一功能进行了迭代升级，具体包括两点：</p><p data-vmark="ff8a">其一，全量开放评论展示发评 IP 属地小尾巴功能；其二，上线“个人主页一级页面展示 IP 属地”功能。</p><p data-vmark="8f48">“全量”是有多全量？例如“人在泰国，正在拍戏”的吴京，也逃不过 IP 属地展示（doge）：</p><p data-vmark="cc64" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/22647b40-aa17-4990-b8c6-4368e2ec79b2.png" w="624" h="464" title="“人在美国刚下飞机”，IP 属地显示湖南" width="624" height="464" referrerpolicy="no-referrer"></p><p data-vmark="98c0">值得注意的是，<span class="accentTextColor">这并不是一个可以关闭的功能，它是强制开启的。</span></p><p data-vmark="2311">所以，IP 属地是如何判定的呢？从规则来说，它根据近期博主发博、发评、投票时对应的 IP 属地进行判断，在国内则显示省份 / 地区，在国外就显示国家 / 地区。</p><p data-vmark="ee5a">除了微博以外，4 月 29 日，抖音、今日头条、小红书也都已经上线“展示用户 IP 属地”功能。其中，抖音 IP 属地同样是全量显示，判断规则是发文和发评时用户对应的 IP 属地。</p><p data-vmark="0de7">如果你在抖音是个“路人”，即从来没在抖音上发文或发评，则不会显示 IP 属地；如果还没更新到 20.5.0 版本，就连别人的 IP 属地也不会显示……</p><p data-vmark="c8a2" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/6cc4a67e-1d1f-43da-ab3d-3de134dd00a2.png" w="512" h="646" title="“人在美国刚下飞机”，IP 属地显示湖南" width="512" height="646" referrerpolicy="no-referrer"></p><p data-vmark="4f23" style="text-align: center;">△ 图源 @抖音安全中心</p><p data-vmark="dce5">今日头条则是通过 @头条安全中心在 4 月 15 日就官宣了这件事情，同样是“全量上线”。</p><p data-vmark="8359" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/04a9453a-e910-477d-b5c5-18143c768e26.png" w="1080" h="598" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="454" referrerpolicy="no-referrer"></p><p data-vmark="27be" style="text-align: center;">△ 图源 @头条安全中心</p><p data-vmark="78d9">而小红书的功能也类似，同样是境内展示到省（区、市），境外展示到国家（地区）。</p><p data-vmark="2087" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/ffe1a24b-821a-4ed9-b75f-c0aed5347911.png" w="1080" h="286" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="217" referrerpolicy="no-referrer"></p><p data-vmark="4a38" style="text-align: center;">△ 图源 @小红书</p><p data-vmark="e5c7">此外，知乎、百家号、豆瓣和快手等多家平台，也都已经宣布逐步开放“IP 属地展示”的功能。值得一提的是，微信公众平台昨晚发布公告后立即进行了更新，效果 be like：</p><p data-vmark="0200" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/139d3543-43b7-4d48-842e-fcaf5c14ef2a.png" w="448" h="208" title="“人在美国刚下飞机”，IP 属地显示湖南" width="448" height="208" referrerpolicy="no-referrer"></p><p data-vmark="1909" style="text-align: center;">△ 图源公众号 @英国那些事儿</p><h3 data-vmark="48a8">IP 属地准吗？</h3><p data-vmark="382b">这么多平台开放 IP 属地展示后，也有一些网友发现自己定位不太对劲。对此，微博官方表示这锅不背，原因是“定位根据运营商 IP 信息解析”。</p><p data-vmark="d7d3" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/0a494e33-0973-4212-b71e-13d86f73f537.png" w="1080" h="437" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="332" referrerpolicy="no-referrer"></p><p data-vmark="2914">实际上，IP 定位数据库的创建和维护也是一项技术活，属于网络空间测绘的一部分。</p><p data-vmark="a217">在这次大规模开放 IP 属地展示之前，各大互联网公司就已经用来做用户画像，定制个性化内容，以及自身业务防流量攻击、防欺诈等安全防护。</p><p data-vmark="f2b2">一般由专业的第三方技术公司提供，他们会与电信运营商和云计算厂商合作获得数据并提供更新维护。而互联网公司从他们手中直接购买数据库，或按次数租用查询 API 接口。这方面国内做的较大的有 IPIP.NET 和纯真 IP 数据，根据公开信息，即将开放展示的知乎和豆瓣都是 IPIP.NET 的客户。</p><p data-vmark="523c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/730b7be0-5140-4966-ad61-f3e937c9b74d.png" w="1080" h="284" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="216" referrerpolicy="no-referrer"></p><p data-vmark="6bc8">据 IPIP.NET 官网介绍，他们提供的数据库最高可精确到市一级，而经纬度数据统一成该市的中心点，并非 IP 地址对应的确切经纬度。</p><p data-vmark="f5ee">对于展示精确到市一级的 IP，到底算不算泄露隐私？大家也有不同观点。</p><p data-vmark="86ae">老网民通常会提起，互联网早期的论坛聊天室都有展示 IP 功能，一般会对最后一段打码，自己去查询的话也可以得到市一级定位。</p><p data-vmark="b394">比如早期的百度贴吧，如果不登录账号直接发言，就会直接显示打过码的 IP 地址。如果现在去考古，还能看到这些“遗迹”。</p><p data-vmark="c525" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/abd4d5a5-abec-4404-93f7-4f85bdf796c2.png" w="1080" h="458" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="348" referrerpolicy="no-referrer"></p><p data-vmark="1e6a">到了移动互联网和 App 的时代，还在运营的老式论坛 BBS 很多也保留了这个功能，如清华的水木社区。</p><p data-vmark="97a5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/f2179fed-e028-4984-b11d-abe48aebb31c.png" w="1080" h="688" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="522" referrerpolicy="no-referrer"></p><p data-vmark="f70f">所以这一波啊，这一波只是回归传统，不算泄露新的隐私。</p><p data-vmark="e15f">反对一方则认为，精确到省虽然看起来足够模糊，但结合发布的内容还是能推测出更多信息。</p><p data-vmark="803a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/d1553abb-cbdb-4ce7-8ebf-b78cbbd3de79.png" w="1080" h="339" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="257" referrerpolicy="no-referrer"></p><p data-vmark="47dd">还有人提出折中的解决方案：后台强制显示给网站管理者能看，前台自愿。</p><p data-vmark="d7d2" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/e3fa9725-ef89-4bb4-ac9c-affa8b26df1b.png" w="1080" h="300" title="“人在美国刚下飞机”，IP 属地显示湖南" width="1080" height="228" referrerpolicy="no-referrer"></p><p
