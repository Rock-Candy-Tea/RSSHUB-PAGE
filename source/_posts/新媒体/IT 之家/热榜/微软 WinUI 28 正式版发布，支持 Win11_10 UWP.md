            <p data-vmark="0e74"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 19 日消息，据微软 Windows UI 官方宣布，Windows UI Library 2.8 正式发布。</p><p style="text-align: center;" data-vmark="f4f6"><img src="https://img.ithome.com/newsuploadfiles/2022/7/287d40dd-82a5-4e0b-bf40-f5eba1fd8f1c.png" w="1268" h="819" title="微软 WinUI 2.8 正式版发布，支持 Win11/10 UWP" width="1268" height="530" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="795b"><img src="https://img.ithome.com/newsuploadfiles/2022/7/9f75d950-641b-4388-8b36-abfff471c5fb.png" w="1259" h="778" title="微软 WinUI 2.8 正式版发布，支持 Win11/10 UWP" width="1259" height="507" referrerpolicy="no-referrer"></p><p data-vmark="6caa"><span class="accentTextColor">WinUI 2.8 是适用于 UWP 应用程序（以及使用 XAML Islands 的桌面应用程序）的 Windows UI 库 (WinUI) 的最新稳定版本。</span></p><p data-vmark="1051">有关使用最新版本的 WinUI 3 构建 Windows 桌面和 UWP 应用程序的详细信息，请参阅 Windows UI 库 3。</p><p data-vmark="fb63">WinUI 托管在 GitHub 上的 Windows UI 库存储库中，微软欢迎在其中报告错误报告、功能请求和社区代码贡献。</p><p data-vmark="cab0">WinUI 包可以通过 NuGet 包管理器添加到 Visual Studio 项目中。</p><p data-vmark="483d"><strong>下面是更新内容：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="4953">WebView2 现在可在 WinUI 2 中使用。WebView2 使你能够使用 Microsoft Edge (Chromium) 在本机应用程序中嵌入 Web 内容。</p></li><li><p data-vmark="d436">WinUI 2 最低支持版本现在是 Windows 10 版本 1809 - Build 17763。</p></li><li><p data-vmark="f6dc">针对包括 TabView、NavigationView、InfoBadge 和 ProgressRing 在内的一系列控件的若干错误修复和改进。</p></li><li><p data-vmark="268d">可访问性和高对比度模式的改进。</p></li><li><p data-vmark="31b8">AnimatedVisualPlayer.AnimationOptimization 属性配置动画在 AnimatedVisualPlayer 中的缓存方式。</p></li></ul><p data-vmark="9b73">WinUI 3 Gallery 和 WinUI 2 Gallery 应用程序包括大多数 WinUI 3 和 WinUI 2 控件、特性和功能的交互式示例。用户可以从 Microsoft Store 下载 <a href="https://apps.microsoft.com/store/detail/winui-3-gallery/9P3JFPWWDZRC?hl=en-us&gl=US" target="_blank">WinUI 3 Gallery</a> 和 <a href="https://apps.microsoft.com/store/detail/winui-2-gallery/9MSVH128X2ZT?hl=en-us&gl=US" target="_blank">WinUI 2 Gallery</a>。</p><p data-vmark="0c01">用户还可以从 GitHub 获取两者的源代码（WinUI 3 使用 main 分支，WinUI 2 使用 winui2 分支）。</p>
