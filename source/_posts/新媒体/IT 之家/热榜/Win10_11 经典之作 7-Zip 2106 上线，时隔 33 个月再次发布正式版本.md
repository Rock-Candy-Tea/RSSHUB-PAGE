<p data-vmark="ec40"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 28 日消息，<a href="https://github.com/sparanoid/7z/tree/master/zh-cn" target="_blank">7-Zip</a> 是一款拥有极高压缩比的的开源压缩软件，支持 LZMA2 等 7 种开源算法，以及支持较广泛的压缩格式 .7z，体积不大（1.4MB）、功能完善，除了默认 UI 不符合国内审美外几乎没什么可挑剔的地方，堪称“小而美”。</p><p data-vmark="eba7">7-Zip 大多数源代码都基于 GNU LGPL 许可协议下发布。AES 代码基于 BSD 许可下发布。unRAR 代码基于两种许可：GNU LGPL 和 unRAR 限制许可。而且用户可以在任何一台计算机上使用 7-Zip ，包括用在商业用途的计算机，就算不捐赠也不影响使用。</p><p data-vmark="b22b">IT之家提醒，7-Zip 主要用在 Windows 平台上，不过 Unix-like 的操作系统如 Linux 与 FreeBSD 也有 7-zip 的移植版本 p7zip 可以使用。</p><p data-vmark="28e1">7-Zip 支持众多的压缩/未压缩的文件格式，包含 ZIP、CAB、RAR、ARJ、Z、gzip、bzip2、LLHA、tar、cpio、RPM。从 4.42 版开始支持 CD/DVD 映像文件格式 ISO，但因为版权问题不再支持 ACE 格式。此外，从 7-Zip 9.14 测试版开始，7-Zip 也已支持开启和解压缩 VHD。</p><p data-vmark="b4b6">自该公司在 2019 年 2 月份发布上一个正式版本也就是 19.00 之后， 7-Zip 就再也没有发布正式版本，倒是测试版到一直更新。现有网友发现， 7-Zip 刚刚推出了 <a href="https://sparanoid.com/lab/7z/" target="_blank">21.06 </a>正式版本，支持简体中文，各位有需要的小伙伴可以尝试升级。</p><p data-vmark="bcab"><img src="https://img.ithome.com/newsuploadfiles/2021/11/fe673df6-f5b8-47c3-a3a9-52629a44298c.png" w="596" h="94" title="Win10/11 经典之作 7-Zip 21.06 上线，时隔 33 个月再次发布正式版本" width="703" height="281" referrerpolicy="no-referrer"></p><blockquote><h2 data-vmark="326b">7-Zip 主要特征</h2><ul class=" list-paddingleft-2"><li><p data-vmark="425e">使用了 LZMA 与 LZMA2 算法的 7z 格式 拥有极高的压缩比</p></li></ul><h3 data-vmark="471b">支持格式：</h3><ul class=" list-paddingleft-2"><li><p data-vmark="10a6">压缩 / 解压缩：7z, XZ, BZIP2, GZIP, TAR, ZIP and WIM</p></li><li><p data-vmark="ab69">仅解压缩：ARJ, CAB, CHM, CPIO, CramFS, DEB, DMG, FAT, HFS, ISO, LZH, LZMA, MBR, MSI, NSIS, NTFS, RAR, RPM, SquashFS, UDF, VHD, WIM, XAR, Z</p></li></ul><p data-vmark="e98e">对于 ZIP 及 GZIP 格式，7-Zip 能提供比使用 PKZip 及 WinZip 高 2-10% 的压缩比</p><p data-vmark="375b">为 7z 与 ZIP 提供更完善的 AES-256 加密算法</p><p data-vmark="c418">7z 格式支持创建自释放压缩包</p><p
