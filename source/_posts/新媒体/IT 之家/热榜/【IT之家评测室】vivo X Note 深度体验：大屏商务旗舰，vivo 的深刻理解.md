<p data-vmark="8ab9">vivo X Note 是笔者今年来最期待的一款旗舰手机，作为 vivo 手机产品的又一个新的形式存在，vivo 基因里对新体验，新形态的探索在这款产品上展现的淋漓尽致。</p><p data-vmark="0d8a">4 月 11 日，这款蓝厂的顶级旗舰 vivo X Note 正式发布。</p><p data-vmark="2713">骁龙 8 Gen1、7 英寸 2K+ E5 超感宽幕、3D 大面积超声指纹、GN1 超大底主摄、800 万 OIS 潜望、IMX663 专业人像（2X）、IMX598 超广角、自研芯片 V1、5000mAh 电池、80W 双电芯闪充、50W 无线闪充、10W 无线反向充、高通 SPU 芯片级安全防护、IP68 防尘防水 、X 轴线性马达、独立 Hi-Fi 芯片、立体声双扬声器、三麦电影级立体录音、气压计、红外遥控、物理静音键、全场景 NFC... 从配置到设计，从功能到交互，不夸张的说，vivo X Note 几乎全部拉满，让人挑不出任何毛病。</p><p data-vmark="a7cb">这近半个月来，笔者一直拿 vivo X Note 作为主力机使用，那么这款旗舰新机的体验如何？现在第一时间为大家送上 vivo X Note
