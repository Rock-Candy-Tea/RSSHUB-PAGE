<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 6 日消息 据外媒 Windows Latest 报道，Windows 10 的大型界面更新“太阳谷（Sun Valley）”将于今年晚些时候发布，在最新的预览版本中，可以看到 Windows 95 时代功能的新图标。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/79d83d95-1054-4644-92d7-a4b7b4994093.jpg" w="696" h="365" title="微软 Win10“太阳谷”更新了 Windows 95 时代的图标" width="696" height="365" referrerpolicy="no-referrer"></p><p>▲图自 Windows Latest，下同</p><p>除了为常用的文件夹和系统应用添加新图标，微软还更新了 Windows 95 时代的图标，例如休眠模式。这些新图标更加类似于 Microsoft 365、Microsoft Launcher 和其他产品的图标。</p><p>IT之家了解到，预览版为 Shell32.DLL 中的文件夹 / 选项 / 按钮提供了一组新图标。在下一个功能更新中，其中一些图标将得到改进。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/5369e35d-4520-4f51-be9d-654c358f40cb.png" title="微软 Win10“太阳谷”更新了 Windows 95 时代的图标" referrerpolicy="no-referrer"></p><p>▲Preview builds 预览版本（左）与 Production 版本（右）</p><p>截图显示，对比 Windows 10 太阳谷更新和 Windows 10 的 2020 年 10 月更新发现，微软更新了 Shell32.DLL 内几个文件夹的图标，包括休眠模式、地球仪、网络等。</p><p>这都是“太阳谷”更新的一部分，UI 大型变动将于下半年开始向消费者推出。</p>
