<p data-vmark="0216"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 5 日消息，微软每月都会发布累计更新以保证 Windows 系统的安全和稳定。</p><p data-vmark="451f">但由于 Windows 的使用基础之庞大，世界上可能有数以千计的硬件正在运行着 Windows 系统，因此微软每月发布的更新也偶尔会导致一些异常或 bug。</p><p data-vmark="4f8a">在某些情况下，微软会将存在缺陷的 Windows 更新自动回滚到上一个版本中。但在大部分情况下，用户都需要自己去卸载导致 Bug 的更新补丁来修复问题。自从 22H2 版本的出现，每个 Win11 用户都可以体验到更简单的操作体验。</p><h2 data-vmark="ae80">如何卸载 Windows 11 更新</h2><p data-vmark="70ff">卸载 / 回滚 Windows 更新的方式取决于当前使用的 Windows 版本。Windows 11 的初始版本（21H2）可以从控制面板中删除更新，而在新的 22H2 版本中则可以直接从设置中删除更新。</p><p data-vmark="d83c">IT之家了解到，微软不允许用户删除包括 Surface 固件在内的一些 Windows 11 更新，也不能卸载驱动程序或 BIOS 更新。</p><p data-vmark="ac21">下面是如何在 Windows 11 21H2 (build 22000) 上卸载 Windows 11 更新:</p><p data-vmark="507c">打开“设置”，进入“Windows 更新”>“更新历史记录”。</p><p data-vmark="ec9c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/156c08f5-c463-48ed-99c4-d5522b47c3f3.jpg" w="760" h="554" alt="显示如何卸载 Windows 11 更新的 Windows 设置应用" title="微软进一步削弱 Win11 控制面板，22H2 卸载更新补丁更简单了" width="760" height="554" referrerpolicy="no-referrer"></p><p data-vmark="e6b0">找到并单击卸载更新。值得一提的是，点击该按钮后系统将为你打开旧的控制面板。</p><p data-vmark="0e88" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/6ac68869-dbb7-4100-920a-e52d6d981687.jpg" w="760" h="554" alt="显示如何卸载 Windows 11 更新的 Windows 设置应用" title="微软进一步削弱 Win11 控制面板，22H2 卸载更新补丁更简单了" width="760" height="554" referrerpolicy="no-referrer"></p><p data-vmark="9566">弹出控制面板后，你只需要选择要在 Windows 11 中删除的更新，然后单击“卸载”即可。</p><p data-vmark="7536" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/f7533891-b815-4536-93d4-81b0229a4733.jpg@s_2,w_820,h_386" w="1440" h="678" alt="显示如何卸载 Windows 11 更新的 Windows 设置应用" title="微软进一步削弱 Win11 控制面板，22H2 卸载更新补丁更简单了" srcset="https://img.ithome.com/newsuploadfiles/2022/6/f7533891-b815-4536-93d4-81b0229a4733.jpg 2x" width="1440" height="386" referrerpolicy="no-referrer"></p><p data-vmark="7f17">随后，你需要确认卸载并等待完成。值得注意的是，在 Windows 11 中删除已安装的更新需要重新启动系统，因此请务必保存你已经打开的工作并关闭所有应用。</p><h2 data-vmark="3b7f">如何卸载 Windows 11 更新（22H2 及更高版本）</h2><p data-vmark="f88d">按 Win + I 或使用任何方法打开“设置”应用，例如右击“开始”菜单，然后选择“设置”。</p><p data-vmark="625d">同样需要你转到 Windows 更新 > 更新历史记录。</p><p data-vmark="b8d6" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/fcd0ba66-ee11-4a8a-a22e-c08631ceafe0.jpg" w="760" h="564" alt="Windows 11 设置应用，显示如何在版本 22H2 及更高版本中卸载 Windows 11 更新" title="微软进一步削弱 Win11 控制面板，22H2 卸载更新补丁更简单了" width="760" height="564" referrerpolicy="no-referrer"></p><p data-vmark="8dd5">在更新历史记录下方，你可以找到“卸载更新”按钮。点击即可。</p><p data-vmark="c589" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/d3eb2667-0e1a-4c8f-b9e9-68ea900b60e4.jpg" w="760" h="564" alt="Windows 11 设置应用，显示如何在版本 22H2 及更高版本中卸载 Windows 11 更新" title="微软进一步削弱 Win11 控制面板，22H2 卸载更新补丁更简单了" width="760" height="564" referrerpolicy="no-referrer"></p><p data-vmark="762d">找到要删除的更新，然后单击旁边的“卸载”按钮。</p><p data-vmark="c11f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/55d2b2a7-3d15-41c9-b814-c2d60b61df69.jpg" w="760" h="564" alt="Windows 11 设置应用，显示如何在版本 22H2 及更高版本中卸载 Windows 11 更新" title="微软进一步削弱 Win11 控制面板，22H2 卸载更新补丁更简单了" width="760" height="564" referrerpolicy="no-referrer"></p><p data-vmark="79c5">通过单击新弹出窗口上的“卸载”来确认删除 Windows 11 更新。</p><p data-vmark="c8f6" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/c2b72394-c3f0-4b3d-87bf-f80048f1dbc6.jpg" w="760" h="564" alt="Windows 11 设置应用，显示如何在版本 22H2 及更高版本中卸载 Windows 11 更新" title="微软进一步削弱 Win11 控制面板，22H2 卸载更新补丁更简单了" width="760" height="564" referrerpolicy="no-referrer"></p><p data-vmark="3f97">完成后，只需要等待 Windows 11 完成然后重启就好了。</p>
