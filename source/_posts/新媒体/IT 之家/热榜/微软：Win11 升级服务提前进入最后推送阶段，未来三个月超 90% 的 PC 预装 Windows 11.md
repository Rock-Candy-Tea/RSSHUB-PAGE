<p data-vmark="efb4"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 29 日消息，近期，微软首席产品官 Panos Panay 发布了《欢迎来到 PC 新纪元》文章，其中提到了当考虑是什么推动了 PC 的需求和使用方式发生如此结构性变化时，看到了三个持续发展的趋势：混合办公及学习的兴起、娱乐习惯和发行模式的转变，以及日常消费习惯的改变。</p><p data-vmark="66c1" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/1/8c77b2a6-6456-47e6-9281-9c7927292480.png" w="1080" h="868" alt="Windows 11 " title="微软：Win11 升级服务提前进入最后推送阶段，未来三个月超 90% 的 PC 预装 Windows 11" width="1080" height="659" referrerpolicy="no-referrer"></p><p data-vmark="b074">微软表示，PC 市场迎来了近 10 年以来幅度最大的一次增长，2021 年全球 PC 出货量超过 3.4 亿台，据 Canalys 统计，与 2019 年相比，增幅达 27%。如今，采用 Windows 系统的设备月活数量超 14 亿台，相比于疫情爆发之前，这些设备的 Windows 运行总时长提升了 10%。2021 年第四季度，根据用户洞察研究显示，Windows 在整个市场中占据了重要的一席之地，我们看到计划使用 PC 进行创作、游戏或办公的用户数量增长了近 50%。同时，我们还看到，各个组织机构、学校和家庭成员都意识到了 PC 对于每个人的价值。</p><p data-vmark="7997">微软表示，为应对持续的供应链挑战，在推出 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 时，我们动员了整个行业的芯片厂商、OEM、零售商及其他合作伙伴，以确保让包括商业及教育类客户在内的更多想要更换新 PC 用户，在这个冬季都可以得偿所愿。截止到发稿时间，中国消费市场上预装 Windows 11 的 PC 已经超过 200 款，达到销售数量的 50%。在未来的 3 个月中，我们预计消费者会看到超过 90% 的 PC 都会预装 Windows 11。</p><p data-vmark="f7f8">微软表示，自 Windows 11 于去年 10 月开始推送以来，我们看到对 Windows 11 强劲的需求与喜爱，随着大家纷纷接受了 Windows 11 的升级服务，此次 Windows 11 的升级进程比此前的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 升级快了一倍。Windows 11 在我们所有 Windows 的历史版本中，也得到了最高的质量评分和产品满意度。产品质量是微软的一大关注点，我们在经过一系列深思熟虑之后，采取了分阶段式的升级方式。今天，我们很高兴地告诉大家，Windows 11 的升级服务已经提前进入到最后的推送阶段，而原定计划是于 2022 年年中达成该目标。与 Windows 10 相比，人们使用 Windows 11 PC 的时间增加了 40%。Windows PC 的多任务处理能力和生产效率得到了充分释放，近一半 Windows 11 用户使用了全新的“贴靠布局”。</p><p data-vmark="6c00">微软表示，Windows 11 还为重新设计的 Microsoft Store 吸引了 3 倍的流量。基于开放的平台，我们去年宣布了新的 Microsoft Store 政策，为创作者和开发者创造更多的盈利机会，这不仅为新的应用程序提供了机会，还为亚马逊和 Epic Games 等新的商店铺平了道路。客户与合作伙伴对全新 Microsoft Store 的反馈让我们倍感振奋。</p><p data-vmark="d1db">微软表示，Windows 平台的一大优势是它提供了一个广阔而开放的生态系统，为人们提供了丰富的选择。如今，Windows 生态系统涵盖了硬件、软件、应用程序、周边设备等，比以往任何时候都更强大。</p><p data-vmark="7e01">微软表示，自 Windows 11 推出以来，我们的 OEM 和芯片合作伙伴们推出了一系列充满活力的 Windows 11 PC 产品，Surface 也迎来了有史以来最大规模的产品更新。最近在 CES 上，我们的合作伙伴 AMD、戴尔、高通、宏碁、华硕、惠普、雷蛇、联想、英特尔展示出了令人心潮澎湃的 PC 和芯片创新成果，AMD 和联想还发布了首款 Microsoft Pluton PC 设备，为我们的客户提供更强的安全性和更多的选择。</p><p data-vmark="9650">微软称，我们将继续升级 Windows 来满足客户不断变化的需求。借助于 Windows 365，我们将 Windows 迁移到云端，让员工可以在任意设备上享受到安全、流畅的 Windows 体验。Windows 11 SE 能够让学校拥有一个简单、纯净的环境，以满足 K-8 学生的独特需求。</p><p data-vmark="8338">微软称，下个月，我们将为 Windows 带来全新体验，其中包括一个公开预览版本，让用户可以在微软与亚马逊、英特尔的合作加持下，通过 Microsoft Store 在 Windows 11 上使用 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 应用程序 ；任务栏也将得到更新，增加语音静音和取消静音功能，以及更便捷的窗口共享功能，和天气服务。此外，我们还将推出两个经重新设计的应用程序 —— 记事本和媒体播放器。</p><p data-vmark="1178">Windows 始终是微软创新的核心，是游戏、创作和探索的目的地，同时也是通往元宇宙的大门。</p>
