<p data-vmark="edde"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 15 日消息，虽然这么多的原生应用程序和服务在 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 甚至 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 中都得到了改造，但任务管理器的现代外观早就应该出现了。通过 Windows 11 Build 22543，微软悄悄地对任务管理器做了一个重大的改变 —— 它仍然是 Windows 11 上最有用的工具。</p><p data-vmark="e5ba" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/dcb4219e-0eb1-42a8-94f9-51357554200d.jpg" w="696" h="365" title="微软 Win11 全新现代任务管理器更多曝光功能体验：支持 App 健康、电池健康、新启动项" width="696" height="365" referrerpolicy="no-referrer"></p><p data-vmark="ec35">微软正在为 Windows 11 试验重新设计的任务管理器，并在预览版中被发现。这个新的外观只对微软内部开放，但通过修改系统文件可以尝试半生不熟的新任务管理器。</p><p data-vmark="b783">正如你在截图中所看到的，任务管理器的刷新设计与 Windows 11 和其他本地应用程序（如 Microsoft Edge 或设置）的界面相匹配。更新后，任务管理器使用 WinUI、Fluent Design 和 Mica 云母材料（这是一种新的半透明设计元素）。</p><p data-vmark="faf6" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/e7e0d5ff-07ab-4a24-bbad-a2f5b5937103.jpg@s_2,w_820,h_428" w="1440" h="751" title="微软 Win11 全新现代任务管理器更多曝光功能体验：支持 App 健康、电池健康、新启动项" srcset="https://img.ithome.com/newsuploadfiles/2022/2/e7e0d5ff-07ab-4a24-bbad-a2f5b5937103.jpg 2x" width="1440" height="428" referrerpolicy="no-referrer"></p><p data-vmark="0ee2">此外，微软还调整了任务管理器的基本界面，将进程等选项移到一个新的菜单上。这些选项之前是在任务管理器的顶部提供的。在新的设计中，你将在左侧的菜单中找到这些选项，类似于“设置”和其他现代 Windows 11 应用程序。</p><p data-vmark="9f52" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/7948cb2c-0a9a-468b-9bb9-a0870cc6c9ae.jpg@s_2,w_820,h_553" w="1403" h="947" title="微软 Win11 全新现代任务管理器更多曝光功能体验：支持 App 健康、电池健康、新启动项" srcset="https://img.ithome.com/newsuploadfiles/2022/2/7948cb2c-0a9a-468b-9bb9-a0870cc6c9ae.jpg 2x" width="1403" height="553" referrerpolicy="no-referrer"></p><p data-vmark="a889">Windows 11 预览版中一个有趣的发现是使用任务管理器监测应用程序的健康状况的新功能。一个看似名为“应用程序健康”的功能将帮助用户知道他们的应用程序何时需要修复（如果它没有响应）。</p><p data-vmark="dd1f">同样，将有一个新的“启动应用程序”部分，以帮助用户识别在启动期间拖慢设备的应用程序。</p><p data-vmark="f195">对于笔记本电脑用户，任务管理器的更新将包括另一个名为“电池健康”的功能，这将帮助用户在运行资源密集型应用程序时了解电池的当前状态。</p><p data-vmark="cf5d">在过去几年中，笔记本电脑的电池健康问题一直是用户关注的焦点，尤其是在 Windows ARM 和苹果 M1 发布之后，所以这个新功能对一些用户来说可能是有用的。</p><p data-vmark="60a5">从趋势上来看，微软正在为 Windows 11 添加更多类似智能手机上的移动功能。</p><p data-vmark="eaf2">值得一提的是，微软公司还没有宣布新的任务管理器，泄露的版本也没有完全发挥作用。</p><p data-vmark="85af">任务管理器在 Windows 11 预览版中有相当多的错误，所以如果你经常使用任务管理来关闭无反应的程序，跳过 Dev 开发频道的版本可能是个好主意。</p>
