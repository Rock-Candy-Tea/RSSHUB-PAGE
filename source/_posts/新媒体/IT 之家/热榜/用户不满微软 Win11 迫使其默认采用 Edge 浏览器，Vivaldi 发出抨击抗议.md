<div>   
<p data-vmark="8e50"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 13 日消息，据 Windows Latest 报道，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的设置和广告迫使用户将 Microsoft Edge 作为默认<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>，这让 Vivaldi 的一些用户感到不满。Vivaldi 公司联合创始人 Jon von Tetzchner 撰写了一篇博文，抨击微软严重的反竞争做法。</p><p data-vmark="ee3e"><img src="https://img.ithome.com/newsuploadfiles/2021/12/e20ed722-1a3f-498c-be20-eac3a1e7fd6c.jpg@s_2,w_820,h_547" w="1200" h="800" title="用户不满微软 Win11 迫使其默认采用 Edge 浏览器，Vivaldi 发出抨击抗议" srcset="https://img.ithome.com/newsuploadfiles/2021/12/e20ed722-1a3f-498c-be20-eac3a1e7fd6c.jpg 2x" width="1200" height="547" referrerpolicy="no-referrer"></p><p data-vmark="c4a6">最近，微软基于 Chromium 的浏览器因不太光彩的原因成为头条新闻。在许多方面，Microsoft Edge 与 Chrome 或其他基于 Chromium 的浏览器相似，但该公司一直在使用新的伎俩来说服用户在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 或 Windows 11 上给 Edge 一个机会。</p><p data-vmark="3cae"><img src="https://img.ithome.com/newsuploadfiles/2021/12/7708e5e6-052e-433f-80d5-5a87fad31bfa.jpg" w="696" h="392" title="用户不满微软 Win11 迫使其默认采用 Edge 浏览器，Vivaldi 发出抨击抗议" width="696" height="392" referrerpolicy="no-referrer"></p><p data-vmark="8049">IT之家获悉，微软最新的尝试之一是关于 Microsoft Edge 浏览器内的新横幅，当用户试图下载 Chrome 浏览器时，该横幅称“Chrome 是 2008 年的浏览器”。此外，微软还让人难以将 Vivaldi 或 Chrome 设置为 Windows 11 的默认浏览器。（相关阅读：《<a href="https://www.ithome.com/0/590/001.htm" target="_blank">微软 Win11 Edge 弹窗警告，试图阻止用户下载 Chrome 浏览器</a>》）</p><p data-vmark="6f82">Vivaldi 显然对微软说服用户将 Chromium Edge 作为 Windows 11 的默认浏览器的最新尝试不满意。</p><p data-vmark="bf1e">“微软的举动似乎令人绝望，而且很熟悉。很明显，他们不希望你使用其他浏览器，”Vivaldi 联合创始人兼 CEO Jon von Tetzchner 说。</p><p data-vmark="2a43">Vivaldi 表示，微软的一些做法阻碍了其他浏览器的业务，包括 Vivaldi。例如，如果你使用微软 Edge 在 Bing 上搜索 Vivaldi，你会被要求继续使用 Chromium Edge，以获得“快速、安全和现代的网络体验”。</p><p data-vmark="afb9">如果你无视建议在 Windows 上安装 Vivaldi，Windows 10 会再次要求你保留 Microsoft Edge，因为它更好。在 Windows 11 上，“[这种做法] 甚至更糟糕”，Jon von Tetzchne 说。</p><p data-vmark="46af"><img src="https://img.ithome.com/newsuploadfiles/2021/12/cd7797dc-dc5a-43f4-849b-5cfde336173d.jpg@s_2,w_820,h_537" w="1063" h="696" title="用户不满微软 Win11 迫使其默认采用 Edge 浏览器，Vivaldi 发出抨击抗议" srcset="https://img.ithome.com/newsuploadfiles/2021/12/cd7797dc-dc5a-43f4-849b-5cfde336173d.jpg 2x" width="1063" height="537" referrerpolicy="no-referrer"></p><p data-vmark="4d8e">正如 Vivaldi 所强调的，Windows 11 让改变默认浏览器这样的基本事情变得困难。与 Windows 10 不同，Windows 11 要求你首先选择浏览器，并将其指定为所有文件扩展的默认浏览器。</p><p data-vmark="4599">你需要为各种 Web 相关的文件扩展名手动选择第三方浏览器，包括 HTTP、HTTPs、htm、HTML 或文件。而如果你真的试图改变所有这些文件扩展名的默认值，Windows 11 会显示又一个弹出窗口，强调 Edge 浏览器的功能。</p><p data-vmark="dccc">“他们甚至提出要通过他们的微软奖励计划向你支付使用该浏览器的费用。这不是一个开发卓越浏览器的自信公司的行为，”Vivaldi 创始人说。“这是一家公司公开滥用其强大的地位，推动人们使用其劣质产品的行为，仅仅是因为它能够 [这样做]。”</p><p data-vmark="abb3">Vivaldi 创始人告诉用户，在社交媒体平台上投诉 Windows 11 的反竞争行为。该公司还建议美国和欧洲的用户给反垄断监管机构写信或打电话，对微软的这些反竞争行为进行调查。</p><p data-vmark="fce6">“最重要的是，不要让微软在这场堆砌的游戏中获胜。坚持下去，直到你能在 Windows 上使用你选择的浏览器 —— 并帮助你的朋友或同事也这样做。”Vivaldi 说。</p>
