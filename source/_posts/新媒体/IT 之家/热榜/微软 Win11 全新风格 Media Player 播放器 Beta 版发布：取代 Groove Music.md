            <p data-vmark="0915"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 5 日消息，去年 11 月，微软宣布将使用 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Media Player 媒体播放器取代 Groove Music 音乐。这一变化首先是为 Windows 11 Dev 开发频道用户提供，但现在该公司正在扩大其可用性，包括 Beta 频道的内测用户。</p><p style="text-align: center;" data-vmark="900a"><img src="https://img.ithome.com/newsuploadfiles/2022/2/bfc221ea-e8d2-4a12-adab-63db633f4148.jpg@s_2,w_820,h_461" w="1440" h="810" title="微软 Win11 全新风格 Media Player 播放器 Beta 版发布：取代 Groove Music" srcset="https://img.ithome.com/newsuploadfiles/2022/2/bfc221ea-e8d2-4a12-adab-63db633f4148.jpg 2x" width="1440" height="461" referrerpolicy="no-referrer"></p><p data-vmark="1fb8">近期，微软对其关于 Windows 11 Media Player 的博客进行了小幅更新，称新的媒体播放器应用现在正向 Windows 11 Beta 通道测试用户推出。然而，值得注意的是，它不是现有《电影和电视》应用的替代品。换句话说，《电影和电视》应用仍然能够串流传输你通过微软商店购买的内容。</p><p data-vmark="5859">同时，新的 Media Player 媒体播放器应用将包括全功能的音乐库，允许你快速浏览和播放音乐，以及创建和管理播放列表。它还具有带有专辑封面和丰富的艺术家图像的专用播放视图，全屏和 mini 播放器选项，以及图形均衡器。此外，媒体播放器包括对浏览、管理和观看本地视频集的全面支持。</p><p data-vmark="8f15">如果你是 Groove 音乐应用程序的活跃用户，你不必担心这一变化，因为你在该应用程序中的音乐收藏将自动迁移到这个新的 Media Player 应用程序。新应用的设计也与 Windows 11 的设计一致。</p><p data-vmark="4137">如果你是 Beta 频道的内测人员，你可以去微软商店检查更新，以获得新的 Windows 11 Media Player 应用。一旦你安装了新的应用程序，它将取代 Groove 音乐应用程序。</p>
