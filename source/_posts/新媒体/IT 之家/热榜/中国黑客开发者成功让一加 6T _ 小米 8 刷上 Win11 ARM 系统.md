<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 2 日消息 外媒 Windows Latest 报道，在智能手机上运行 Windows 系统一直是微软爱好者的梦想，特别是在该公司停止支持 Windows 10 Mobile 之后。虽然我们可能永远看不到微软推出 Windows 11 Mobile 手机设备，但一些年轻的开发者已经向我们展示了在安卓手机上运行 Windows 11 的预览。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/3c82a5f1-d57c-487f-9366-1e223ddc4e08.jpg" w="315" h="420" title="中国黑客开发者成功让一加 6T / 小米 8 刷上 Win11 ARM 系统" width="315" height="420" referrerpolicy="no-referrer"></p><p>微软最近开始向内测人员推出第一个 Windows 11 预览版 Build 22000.51。自然，一些技术爱好者决定尝试在一加和小米等公司的安卓手机上运行桌面版 Windows 11 系统。</p><p>这些努力还处于早期阶段，但“叛逆者”（Renegade）黑客项目背后的主要开发者 Xilin Wu 告诉我们，Windows 11 ARM64 可以安装在骁龙 845 手机上，而骁龙 855 则由 edk2-sm8150 port 部分支持。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/303d8fe2-94ee-4f7d-b010-8cba2cf98333.png" w="1440" h="1026" title="中国黑客开发者成功让一加 6T / 小米 8 刷上 Win11 ARM 系统" width="1440" height="584" referrerpolicy="no-referrer"></p><p>由于 Renegade 团队所做的工作，一些开发者已经在一加 6T 和<a href="https://www.ithome.com/0/560/109.htm" target="_blank">小米 8 上成功启动了 Windows 11 操作系统</a>。然而，该操作系统（OS）的大部分内容是无法使用的。例如，据开发者称，目前不支持拨打电话。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/1829beac-8d25-4ad8-ae05-0f3a3db5a38d.jpg" w="637" h="719" title="中国黑客开发者成功让一加 6T / 小米 8 刷上 Win11 ARM 系统" width="637" height="719" referrerpolicy="no-referrer"></p><p>▲一加 6T</p><p>为了在安卓手机上运行 Windows 11 桌面操作系统，开发者已经创建了自己的工具/驱动程序，并为骁龙 845 启用了自定义的 UEFI 环境。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/58cf87a1-c527-40b2-bc31-d512d6da91ef.jpg@s_2,w_820,h_720" w="1062" h="933" title="中国黑客开发者成功让一加 6T / 小米 8 刷上 Win11 ARM 系统" srcset="https://img.ithome.com/newsuploadfiles/2021/7/58cf87a1-c527-40b2-bc31-d512d6da91ef.jpg 2x" width="1062" height="720" referrerpolicy="no-referrer"></p><p>▲小米 8</p><p>这是因为高通公司的 xbl（UEFI 固件）和 abl（加载 Linux 内核所需）在零售终端上进行了签名，它们不能被修改。因此，不可能使用现存引导程序启动 Windows。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/203e8e02-aacc-45bd-b126-fb61dc9cf10e.jpg@s_2,w_820,h_452" w="1024" h="565" title="中国黑客开发者成功让一加 6T / 小米 8 刷上 Win11 ARM 系统" srcset="https://img.ithome.com/newsuploadfiles/2021/7/203e8e02-aacc-45bd-b126-fb61dc9cf10e.jpg 2x" width="1024" height="452" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/9a1873a5-1117-4e3f-9100-bf421fa7d0f0.jpg" w="315" h="420" title="中国黑客开发者成功让一加 6T / 小米 8 刷上 Win11 ARM 系统" width="315" height="420" referrerpolicy="no-referrer"></p><p>IT之家获悉，微软并不打算让桌面版的 Windows 11 进入智能手机。Windows 11 对 Arm 架构的支持以及由这些有才华的开发者创建的定制工具和驱动将使其成为可能。</p><p>到目前为止，我们已经看到 Windows 11 在 Lumia 950 XL、一加 6T、小米 8 和一些骁龙原型机上得以运行。</p><p>除了安卓手机，也有可能在树莓派 4、Lumia 950 XL 上运行新的 Windows 11，其中 Lumia 950 XL 已支持通话和数据流量上网。</p><p>不过，大家先不要抱太大希望，因为 Windows 11 ARM 移植项目在安卓手机上远非完美，目前主要是概念验证，触摸屏、手机网络连接不能正常工作等多个问题还有待解决。</p>
