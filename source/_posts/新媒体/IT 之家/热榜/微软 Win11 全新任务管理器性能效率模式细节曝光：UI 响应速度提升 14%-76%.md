<p data-vmark="aa07"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 22 日消息，据 Windows Latest 报道，在任何版本的 Windows 上，你都会发现有少数进程在后台运行，即使在空闲状态下也会活跃使用系统资源。用户可以用任务管理器监控活跃的进程或程序，如果该进程拖累了系统，可以使用“结束任务”功能将其终结。</p><p data-vmark="feff">任务管理器是为了控制性能，可以显示后台进程占用了多少资源。这些系统资源包括 CPU、内存、GPU 或甚至网络带宽。</p><p data-vmark="21c5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/9979079c-3c4f-4b95-bee0-c46bdd4cb54c.jpg" w="696" h="365" title="微软 Win11 全新任务管理器性能效率模式细节曝光：UI 响应速度提升 14%-76%" width="696" height="365" referrerpolicy="no-referrer"></p><p data-vmark="4f1b">在 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Build 22557 预览版中，微软正试图实现一项功能，允许用户限制“基于每个应用进程的资源分配”。之此前不同的是，“结束任务”功能会杀死进程，并可能导致不良问题，而效率模式将对进程进行节流，并将对前台体验的干扰降至最低。</p><p data-vmark="3b7c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/cad8b3dc-878c-45ff-a054-ae2daa5af72c.jpg@s_2,w_820,h_526" w="929" h="596" title="微软 Win11 全新任务管理器性能效率模式细节曝光：UI 响应速度提升 14%-76%" srcset="https://img.ithome.com/newsuploadfiles/2022/2/cad8b3dc-878c-45ff-a054-ae2daa5af72c.jpg 2x" width="929" height="526" referrerpolicy="no-referrer"></p><p data-vmark="a231">Windows 11 任务管理器的效率模式将进程的基本优先级降低到最低，并将 QoS 模式设置为 EcoQoS。</p><p data-vmark="f09e">要使用这个新功能，你需要右击任何进程并选择该选项，如下面的截图所示。</p><p data-vmark="c548" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/df6d52de-1031-41f4-b3cc-7e53ab3ae506.jpg@s_2,w_820,h_624" w="1073" h="817" title="微软 Win11 全新任务管理器性能效率模式细节曝光：UI 响应速度提升 14%-76%" srcset="https://img.ithome.com/newsuploadfiles/2022/2/df6d52de-1031-41f4-b3cc-7e53ab3ae506.jpg 2x" width="1073" height="624" referrerpolicy="no-referrer"></p><h3 data-vmark="3af7">Windows 11 的效率模式如何优化进程</h3><p data-vmark="4b98">在 Windows 上，可以根据调度优先级来运行线程。根据微软更新的文档，所有线程都有一个分配的调度优先级，其范围从零（最低优先级）到 31（最高优先级）。</p><p data-vmark="8bb7">效率模式将基本优先级设置为“THREAD_PRIORITY_LOWEST”，以确保它们 [进程] 在必要时可以被抢占。根据官方文档，这是为“后台线程，特别是那些处理器密集型的线程 ”所做的努力。</p><p data-vmark="6343" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/acfd2695-3220-4494-ba5b-5b0dbc49c29a.jpg@s_2,w_820,h_635" w="1057" h="819" title="微软 Win11 全新任务管理器性能效率模式细节曝光：UI 响应速度提升 14%-76%" srcset="https://img.ithome.com/newsuploadfiles/2022/2/acfd2695-3220-4494-ba5b-5b0dbc49c29a.jpg 2x" width="1057" height="635" referrerpolicy="no-referrer"></p><p data-vmark="5141">当你将多个进程设置为消耗较少资源时，Windows 会根据情况自动将空闲资源分配给优先级较高的进程。</p><p data-vmark="dcb1">“低优先级确保该进程不会干扰用户正在活跃使用的高优先级进程，”微软指出。</p><h3 data-vmark="8fd0">EcoQoS 在 Windows 11 CPU 优化中的作用</h3><p data-vmark="b32e">Effiecienly 模式的第二步是调用 EcoQoS。术语“EcoQoS”是生态服务质量（QoS）级别的标准，它于 2021 年首次推出，是一项可选功能，供那些希望有效运行其应用程序的某些进程以降低功耗的开发者使用。</p><p data-vmark="9b71">如果使用得当，EcoQoS 可以延长电池续航，带来更好的能源效率，减少风扇噪音和改善热节流。</p><p data-vmark="7b45">“这个新的 QoS 级别对于那些没有显著的性能或延迟要求的工作负载来说是很有价值的，使它们总是以节能的方式运行，”微软在去年发表的一份文件中解释道。</p><p data-vmark="2822">当你为一个进程启用效率模式时，任务管理器也会触发 EcoQoS，以确保该进程以最省电的方式执行。因此，处理器将能够以较低的频率运行，以节省电量，提高 UI 响应速度，以及 CPU 的热量性行踪。</p><h3 data-vmark="ebcd">效率模式保证 UI 响应速度</h3><p data-vmark="f384">微软已经对效率模式进行了一年的测试，该公司表示，此功能还可以使应用程序或“开始”菜单的启动在典型的“繁忙”系统上受益。</p><p data-vmark="b8f5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/ed741c71-b4b7-4996-a181-c990f822de0d.jpg@s_2,w_820,h_430" w="836" h="438" title="微软 Win11 全新任务管理器性能效率模式细节曝光：UI 响应速度提升 14%-76%" srcset="https://img.ithome.com/newsuploadfiles/2022/2/ed741c71-b4b7-4996-a181-c990f822de0d.jpg 2x" width="836" height="430" referrerpolicy="no-referrer"></p><p data-vmark="f317">正如在上图中看到的，微软已经能够将 Windows 11 的响应速度提升 14%-76%。</p><p data-vmark="b087">根据这些文件，以下是将效率模式和 EcoQoS 整合到任务管理器的主要好处：</p><ul class=" list-paddingleft-2"><li><p data-vmark="f3ca">允许用户手动启用效率模式（EcoQoS），而不是依赖应用程序开发人员。目前，微软 Edge 等少数应用程序包括对这些改进的内置支持。</p></li><li><p data-vmark="385f">提供 CPU 耗电量减少高达 90%。</p></li><li><p data-vmark="1e0c">减少热量和风扇噪音。</p></li><li><p data-vmark="ba6d">提高并发工作负载的性能。</p></li><li><p data-vmark="9e26">减少过热降频保护。</p></li><li><p data-vmark="7a17">注重能源的可持续性。</p></li></ul><p data-vmark="bd2e">目前，微软专注于 CPU 优化，因为它只想减少功耗。在即将到来的 Windows 11 版本中，你可以期待其他系统资源的类似技术，比如内存甚至是 GPU 显卡。</p><p
