<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 29 日消息 今天凌晨，微软面向 Dev 开发频道发布了第一个 Windows 11 Insider Preview 版本，即 Build 22000.51!  微软表示，随着我们在未来几个月内完成产品的定型，我们将与你一起验证这一体验。在这个早期预览中，<a href="https://www.ithome.com/0/559/232.htm" target="_blank">你将可以尝试上周展示的许多新功能</a>，但不是全部。</p><p>微软 Windows 11 Build 22000.51 包括居中的“开始”菜单、任务栏，以及操作中心 / 任务栏弹出的新体验。除了这些大的改进之外，Windows 11 还为文件资源管理器提供了现代界面。</p><p>自 Windows 7 以来，文件资源管理器的整体界面一直保持不变。微软在 Windows 8 中增加了一个功能区，但它大部分看起来都是一样的。这将在 Windows 11 中得到改变。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/f090a1e0-4102-4408-a145-c6ee3998a6f1.jpg@s_2,w_820,h_624" w="958" h="729" title="微软 Win11 新变化：文件资源管理器“命令栏”和上下文菜单上手体验" srcset="https://img.ithome.com/newsuploadfiles/2021/6/f090a1e0-4102-4408-a145-c6ee3998a6f1.jpg 2x" width="958" height="624" referrerpolicy="no-referrer"></p><p>在 Windows 11 中，微软不会取消经典或当前的文件资源管理器应用程序。同样的文件资源管理器更新将作为今年秋天 Windows 更新的一部分发布，但它似乎已经在现有应用程序的基础上更新了类似于现代流畅设计的用户界面 UI，而且看起来与以前不同。</p><p>正如我们在图片中看到的，文件资源管理器有新的简化菜单，称为“命令栏”，看起来是基于 WinUI 的。它有简洁的界面，在资源管理器窗口的顶部可见。</p><p>微软已经删除了 2012 年首次推出的 Ribbon UI 界面。新的“命令栏”带有这些操作：创建一个新的文件夹 / 文件或快捷方式，剪切、复制、粘贴、重命名、共享、删除、应用过滤器和切换布局。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/48713c0a-0c94-4daf-8826-8ed08a04429b.jpg" w="768" h="523" title="微软 Win11 新变化：文件资源管理器“命令栏”和上下文菜单上手体验" width="768" height="523" width="768" height="481" referrerpolicy="no-referrer"></p><p>你还可以点击新的 eclipse 菜单，该菜单包含更多的功能，如选项，让你自定义资源管理器的高级功能。</p><p>微软还在为文件资源管理器和桌面开发新的现代右键或上下文菜单。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/a6ac1150-26b8-446d-bfbd-824ff2093835.jpg@s_2,w_820,h_581" w="1078" h="764" title="微软 Win11 新变化：文件资源管理器“命令栏”和上下文菜单上手体验" referrerpolicy="no-referrer"></p><p>目前，上下文菜单的功能有限，但用户可以点击“显示更多选项”按钮，访问传统的上下文菜单，该菜单也已更新为圆角和阴影效果。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/5f85e183-c14d-4c5f-9062-7b6b919ab6b2.jpg@s_2,w_820,h_574" w="1018" h="713" title="微软 Win11 新变化：文件资源管理器“命令栏”和上下文菜单上手体验" referrerpolicy="no-referrer"></p><p>微软正在为应用开发者编写文档，以便在新的上下文菜单中整合他们的应用。当所有开发者开始使用新的上下文菜单时，旧的上下文菜单有可能会在未来退役。</p><p><img 版本要干净得多，也不那么杂乱，而且还针对触摸屏设备进行了优化。</p><p>文件资源管理器的整体界面并没有大的变化。它看起来很熟悉，但现在界面更干净，而且在某些区域可以看到 acrylic 透明效果。</p><p>带有新文件资源管理器的 Windows 11 将在今年秋天开始向主流用户推出。如果你不能等待 Windows 11 的全面上市，<a
