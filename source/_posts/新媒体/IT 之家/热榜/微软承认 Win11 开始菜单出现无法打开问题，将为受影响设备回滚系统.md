<div>   
<p data-vmark="3d76"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 23 日消息，今天早些时候，微软针对导致某些 Windows 11 PC 上的<span class="accentTextColor">开始菜单无法打开</span>的问题发布了一份<a href="https://docs.microsoft.com/en-us/windows/release-health/status-windows-11-21h2#start-menu-might-not-open" target="_blank">公告</a>，确定在 KB5014668 更新（内部版本 22000.778）及之后的版本会受影响。</p><p data-vmark="5b4e" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/65594b69-b6f4-4a60-b82b-f1c301e6fae0.jpg" w="700" h="394" alt="Win11 不再有磁贴了，界面布局似乎不再以移动平台为基准" title="微软承认 Win11 开始菜单出现无法打开问题，将为受影响设备回滚系统" width="700" height="394" referrerpolicy="no-referrer"></p><p data-vmark="3a94">公告称：</p><blockquote><p data-vmark="aeb0">安装 KB5014668 更新（内部版本 22000.778）或更高版本后，我们收到报告称，少数设备可能无法打开开始菜单。在受影响的设备上，点击或选择开始按钮，或使用键盘上的 Windows 按键都可能无效。</p></blockquote><p data-vmark="2f7b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/1bbda7d2-e5ea-4672-9f00-762bfd3b5186.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzEzNS5wbmc=,g_3,x_14,y_14,a_0,t_100" w="879" h="818" title="微软承认 Win11 开始菜单出现无法打开问题，将为受影响设备回滚系统" width="879" height="763" referrerpolicy="no-referrer"></p><p data-vmark="83df">IT之家了解到，微软已使用 <span class="accentTextColor">Known Issue Rollback（KIR，已知问题回滚）方案</span>解决了这个问题，该方案是微软在 2021 年推出的功能，如果在 Windows 更新期间出现问题，它可以快速将受影响的设备恢复到生产使用状态。</p><blockquote><p data-vmark="0129">解决方案最多可能需要 24 小时才能自动传播到消费者设备和非托管业务设备，重新启动 Windows 设备可能有助于解决方案更快地应用于设备。</p><p data-vmark="ec56">对于已安装受影响更新并遇到此问题的企业管理设备，可以通过安装和配置特殊组策略来解决此问题（<a href="https://download.microsoft.com/download/a/3/9/a39e5797-119b-4d83-b778-f1227243baa0/Windows%2011%20(original%20release)%20KB5014668%20220721_04201%20Known%20Issue%20Rollback.msi" target="_blank">点此下载</a>）。特殊组策略可在计算机配置 -> 管理模板 -> KB5014668 220721_04201 已知问题回滚 -> Windows 11（original release）中找到。</p></blockquote><p data-vmark="34b9">值得一提的是，<span class="accentTextColor">KB5014668 补丁此前也存在过问题</span>，该补丁修复了阻止 Windows 10 用户升级到 Windows 11（初始版本）的问题，然而尴尬的是，根据微软 Windows 反馈中心的内容，这个补丁本身似乎也遇到了安装问题。</p><p data-vmark="5357"><strong>相关阅读：</strong></p><p data-vmark="b392">《<a href="https://www.ithome.com/0/628/180.htm" target="_blank">微软 Win11/10 KB5014668 修复无法升级系统问题，但用户称补丁本身也无法安装</a>》</p>
