<p data-vmark="b5a6"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 25 日消息，据 Windows Latest 报道，与 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 不同，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 拥有全新的、现代的“Fluent 流畅滚动条”，当用户与之互动时，其形状会发生变化。流畅滚动条是动态的，它们在不同的外形因素上或当改变窗口的大小时自动缩放，它目前在设置、媒体播放器等应用程序中使用。</p><p style="text-align: center;" data-vmark="96ae"><img src="https://img.ithome.com/newsuploadfiles/2022/3/c6bcc7bc-b236-46e4-88b9-36284d7f900f.jpg" w="696" h="365" title="微软：Win11 流畅设计滚动条将引入到谷歌 Chrome 浏览器" width="696" height="365" referrerpolicy="no-referrer"></p><p data-vmark="5bfa">根据微软的一项新提议，谷歌 Chrome 浏览器可能很快就会拥有流畅滚动条功能。在一份提案中，微软表示，他们希望对 Chrome 浏览器中的旧滚动条进行现代化改造，使谷歌浏览器与 Windows 11 设计标准保持一致。</p><p data-vmark="d813">该提案旨在使 Chromium 的滚动条（包括覆盖和非覆盖）现代化，以适应 Windows 11 流畅的设计语言。此外，微软还在考虑对用户与 Chrome 浏览器中的滚动条的互动方式进行一些视觉上的改变，采用不同的模式。</p><ul class=" list-paddingleft-2"><li><p data-vmark="f54c">极简模式：这将是滚动条的默认设计。</p></li></ul><p style="text-align: center;" data-vmark="c1b4"><img src="https://img.ithome.com/newsuploadfiles/2022/3/6aca3d39-76b3-4661-9e9d-e66b97b0cb98.jpg" w="713" h="362" title="微软：Win11 流畅设计滚动条将引入到谷歌 Chrome 浏览器" width="713" height="362" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="8ec6">完全模式：完全模式滚动条是当光标在滚动条区域上时，所有的部分如拇指或按钮都会被描画出来。</p></li></ul><p style="text-align: center;" data-vmark="9669"><img src="https://img.ithome.com/newsuploadfiles/2022/3/378061c0-7462-473a-a085-5fd26548a3a4.jpg" w="690" h="322" title="微软：Win11 流畅设计滚动条将引入到谷歌 Chrome 浏览器" width="690" height="322" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="4fb6">隐藏模式：这是一个新功能，滚动条将完全淡出，可能无法找到页面上的确切位置。谷歌对隐藏模式表示担忧，微软正在积极探索一个更好的方法来处理视觉上的改进。</p></li></ul><p data-vmark="0d71">为了更好地理解这些差异，下面是微软 Edge 中的新滚动条和 Chrome 中的经典滚动条的比较。</p><p style="text-align: center;" data-vmark="da79"><img src="https://img.ithome.com/newsuploadfiles/2022/3/0a1e1d37-3882-4f8c-9638-8599b26bf25b.jpg@s_2,w_820,h_522" w="1162" h="740" title="微软：Win11 流畅设计滚动条将引入到谷歌 Chrome 浏览器" srcset="https://img.ithome.com/newsuploadfiles/2022/3/0a1e1d37-3882-4f8c-9638-8599b26bf25b.jpg 2x" width="1162" height="522" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="a5d4">▲ Chrome 和 Edge 的滚动条</p><p data-vmark="986b"><strong>其他特点 / 变化：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="2c9b">根部覆盖的滚动条将始终存在于最小模式中。</p></li><li><p data-vmark="86f4">当指针直接在轨道上时，滚动条会过渡到完整模式。</p></li><li><p data-vmark="81bd">在启动时，所有的滚动条都会被描画出来，根滚动条仍然可见（在最小模式下），其余的则会淡出。</p></li><li><p data-vmark="ba83">当指针进入一个非根滚动节点时，滚动条会以最小模式绘制。</p></li><li><p data-vmark="5c90">当指针离开节点的边界时，最小模式的滚动条就会淡出。</p></li><li><p data-vmark="6605">通过 Ctrl+F 进行的文本匹配会使滚动条立即以完整模式绘制，并在匹配位置的轨道上出现勾选标记。</p></li></ul><p data-vmark="1094">微软还指出，可以通过 Windows 11 的原生解决方案“始终打开滚动条”绕过“隐藏模式”。与 macOS 一样，可以从设置-> 辅助功能-> 视觉效果中“始终显示滚动条”。新的滚动条将意识到本地设置。</p><p data-vmark="8c5e">“当启用该功能时，滚动条将以完整模式 绘制。在这种操作系统设置下，滚动条将占用布局空间，”微软指出。</p><p style="text-align: center;" data-vmark="4792"><img src="https://img.ithome.com/newsuploadfiles/2022/3/b055f692-7ce5-40f3-878a-5e38a38a6b0a.jpg" w="668" h="781" title="微软：Win11 流畅设计滚动条将引入到谷歌 Chrome 浏览器" width="668" height="781" referrerpolicy="no-referrer"></p><p data-vmark="a04e">正如上面截图显示，微软提议的滚动条将表现得更像覆盖式滚动条。它们将更薄，有内嵌和圆角的边缘。如果谷歌开发者拒绝覆盖式滚动条，微软还计划重新设计非覆盖式滚动条，以使 Windows 11 上的 Chrome 浏览器现代化。</p><p data-vmark="3229">自 2022 年 1 月以来，微软一直在考虑为基于 Chromium 的网络浏览器提供这一功能。</p><p data-vmark="11ff">早在 2021 年，微软的一名开发人员就已经证实，该公司打算把 Edge 的一些设计改进也带到 Chromium 开源项目中。</p>
