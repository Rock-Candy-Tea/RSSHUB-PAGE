<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 3 日消息 此前，微软发布了 Windows 11 系统，简化了界面设计和用户操作。全新 Windows 更加现代化，整洁且美观。无论是全新的“开始”键和任务栏，还是每一种声音、字体和图标，这一切都经过精心设计，让你可以自由掌控。微软把“开始”键放在中心位置，让你更容易地快速找到所需的内容。凭借云和 Microsoft 365 的强大性能，“开始”菜单会向你显示最近文档，无论你之前在哪个平台或设备上查看过，即使是在 Android 或 iOS 设备上浏览过这些文档，“开始”菜单都能让你一览无余。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/fb5727cc-429f-4693-bc2e-6bb7aa54bcc1.png" w="1440" h="960" title="爆料：微软 Win11 Cloud PC 将在 7 月 14 日 Inspire 大会期间发布" width="1440" height="547" referrerpolicy="no-referrer"></p><p>不过除了消费者版和企业版 Windows 11 外，爆料称，微软还将发布一款 Windows 11 Cloud PC。预计将在 7 月 14 -15 日的 Inspire 大会期间公开。</p><p>微软 Inspire 2021 大会将于 7 月 14-15 日举行，鉴于全球疫情危机尚未解决，本次大会将采用线上数字活动方式进行。此前 6 月份的 Build 2021 开发者大会也是采用了线上模式。</p><p><img src="https://qimg.ithome.com/image/2021/07/03/6376094052107275063430460.png" title="爆料：微软 Win11 Cloud PC 将在 7 月 14 日 Inspire 大会期间发布" referrerpolicy="no-referrer"></p><p><img src="https://qimg.ithome.com/image/2021/07/03/6376094051798313963204407.png" title="爆料：微软 Win11 Cloud PC 将在 7 月 14 日 Inspire 大会期间发布" referrerpolicy="no-referrer"></p><p>IT之家获悉，此前网上已经泄露了 Cloud PC 服务的一些细节。</p><p>首先，Cloud PC 是一项基于 Azure 的新服务，建立在“Windows 虚拟桌面”之上，它旨在重新定义“现代工作场所”，并彻底改变计算环境。</p><h2>什么是 Cloud PC？</h2><p>通过 Cloud PC，你可以在云端拥有自己的个人桌面，就像你的 OneDrive 和 Google Drive 文件一样，可以从任何地方访问。Cloud PC 将托管在微软 Azure 服务器上，用户可以使用 Windows、Android 和 iOS 的微软远程桌面应用访问它。</p><p>目前，你可以使用远程桌面应用将桌面屏幕镜像到手机或设备上。Cloud PC 电脑将把事情提升到一个新的水平，在云端提供虚拟化版本的 Windows 10。这将允许你通过互联网流式传输桌面应用程序，以提高性能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/1/20210103_103047_406.jpg@s_2,w_820,h_324" w="1345" h="531" title="爆料：微软 Win11 Cloud PC 将在 7 月 14 日 Inspire 大会期间发布" width="1345" height="324" referrerpolicy="no-referrer"></p><p>Cloud PC 电脑至少会有三种不同的配置 —— 中级（通用计算）、重型（性能更好）和高级（商业客户）。</p><p>根据一份新的支持文件，Cloud PC 还将使用微软 Graph 技术，这是一个用于实时协作、Azure 集成和其他功能的工具。微软 Graph v1.0 已经更新，包括 Cloud PC 的新 API，这表明这项新服务可能会在不久的将来推出。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/1/20210103_103059_375.jpg@s_2,w_820,h_513" w="1029" h="644" title="爆料：微软 Win11 Cloud PC 将在 7 月 14 日 Inspire 大会期间发布" width="1029" height="513" referrerpolicy="no-referrer"></p><p>微软在文件中指出：“Cloud PC API 处于预览阶段，目前仅对特定客户群体开放”。</p>
