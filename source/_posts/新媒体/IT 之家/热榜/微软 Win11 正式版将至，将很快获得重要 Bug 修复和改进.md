<p data-vmark="123b"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 1 日消息 据 Windows Latest 报道，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 是微软桌面操作系统的最新版本，它将于 10 月 5 日开始向兼容硬件推出。虽然微软已经宣布了 Windows 11 的发布日期，但该更新直到 2022 年初才会在大多数设备上正式推出。</p><p data-vmark="ed7d"><img src="https://img.ithome.com/newsuploadfiles/2021/10/9f65f49f-5b69-4125-95a9-99110c99607a.jpg@s_2,w_820,h_461" w="1440" h="810" title="微软 Win11 正式版将至，将很快获得重要 Bug 修复和改进" srcset="https://img.ithome.com/newsuploadfiles/2021/10/9f65f49f-5b69-4125-95a9-99110c99607a.jpg 2x" width="1440" height="461" referrerpolicy="no-referrer"></p><p data-vmark="c741">越来越多的设备将在每周或每月获得 Windows 11 更新，使用户能够获得新的功能和改进。你可以通过安装 PC 健康检查工具查看哪些设备与 Windows 11 兼容。</p><p data-vmark="9c28">在推出之前，微软已经确认将在未来几周内发布额外的错误 Bug 修复和改进。在 Windows 11 Build 22468 的发布说明中，微软指出，修复措施不与特定的版本挂钩，因此一些改进和修复措施可能会出现在以后的累积更新补丁中。</p><p data-vmark="aadc">例如，微软正在测试一项新功能，允许用户看到关于虚拟专用网络连接的一些统计数据。同样，我们也会得到一个新的设置选项，以关闭当你悬停在 Windows 任务栏的搜索图标上时出现的最近搜索弹窗。</p><p data-vmark="b616">这个选项将在任务栏设置中的任务栏行为下提供。此外，微软正在测试许多错误 Bug 修复，这些修复将在今年晚些时候发布给生产渠道的用户。</p><p data-vmark="ec36">例如，微软正在研究一个新的变化，当你悬停在搜索图标上时，将提高飞出弹窗的可靠性。同样，微软已经修复当你搜索 Word 等应用程序时，最近的文件无法出现的问题。</p><p data-vmark="f7aa">对于文件资源管理器，微软做了一个改变，当你右键点击 OneDrive 位置的文件时，将减少上下文菜单崩溃。我们还得到了即耸人听闻的 kaomoji¯\_(ツ)_/¯ 会显示在不正确的位置的修复。</p><p data-vmark="a8ff">微软也在进行这些错误 Bug 的修复工作：</p><ul class=" list-paddingleft-2"><li><p data-vmark="3b1a">一个可能导致文本渲染问题的错误已被修复。</p></li><li><p data-vmark="0e94">一个可能在系统重启后带回被删除的应用程序的 Bug 已被修复。</p></li></ul><p data-vmark="fb29">请注意，微软仍在测试这些改进，大部分变化将在今年移植到 Windows 11 版本 21H2 中。</p><p data-vmark="1ce2">IT之家获悉，如果你已经等不了 Windows 11 推送，你将能够在 10 月 5 日使用 Media Creation Tool 和 ISO 镜像来安装它。你也可以购买到大量预装 Windows 11 的 PC 设备，尽管该更新最终将提供给世界各地的旧款兼容 PC 设备。</p><p data-vmark="b105">根据你使用的 OEM 制造商和驱动程序不同，你可能还需要等待一段时间。</p>
