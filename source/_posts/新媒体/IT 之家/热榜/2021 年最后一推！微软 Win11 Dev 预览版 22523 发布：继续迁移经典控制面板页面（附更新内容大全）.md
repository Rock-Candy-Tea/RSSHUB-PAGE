target="_blank">Windows 11</a> Dev 预览版 22523 更新，该版本包括改进的“设置”应用程序、若干错误 Bug 修复和其他内容等。另外微软还发布了<a href="https://www.ithome.com/0/592/660.htm" target="_blank"> Windows 11 Dev 预览版 22523 官方 ISO 镜像下载</a>。</p><p data-vmark="3476"><img src="https://img.ithome.com/newsuploadfiles/2021/12/66115ed8-df11-45b4-b3bc-d3b3ac059c65.jpg@s_2,w_820,h_482" w="1440" h="847" alt="Windows 11系统" title="2021 年最后一推！微软 Win11 Dev 预览版 22523 发布：继续迁移经典控制面板页面（附更新内容大全）" srcset="https://img.ithome.com/newsuploadfiles/2021/12/66115ed8-df11-45b4-b3bc-d3b3ac059c65.jpg 2x" width="1440" height="482" referrerpolicy="no-referrer"></p><p data-vmark="ae6d">IT之家获悉，微软表示，在 ARM64 电脑上将提供此版本更新。这将是微软在假期中的最后一次推送版本。2022 年见!</p><h2 data-vmark="8af5">下面是 Windows 11 Build 22523 更新内容</h2><h3 data-vmark="d63e">变化和改进</h3><ul class=" list-paddingleft-2"><li><p data-vmark="6f0a">微软已经在 ALT + TAB 和任务视图中推出了快照组，就像你在任务栏上悬停打开的应用程序一样，你可以在那里看到它们，面向 Dev 频道中的所有内测人士。</p></li><li><p data-vmark="4ab3">当文件资源管理器同通往本机时，添加媒体服务器和（如果合适）删除媒体服务器选项现在可以在点击命令栏的“...”时使用。</p></li><li><p data-vmark="0244">作为微软正在进行的努力的一部分，将更多控制面板中的设置引入到“设置”应用程序中。</p></li></ul><p data-vmark="5a01">・ 控制面板中的“程序和功能”链接现在将通往设置 > 应用程序 > 已安装的应用程序。（换句话说，链接到卸载或更改电脑上的程序的页面。）</p><p data-vmark="3b3d">・ 微软正在将卸载更新（用于累积更新等）从控制面板移到“设置”中的一个新页面，即设置 > Windows Update > 更新历史。</p><h3 data-vmark="fb81">修复</h3><p data-vmark="7bd4">[任务栏]</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="5a61">修复了一个与文本输入初始化有关的问题，该问题可能导致 shell（例如，“开始”菜单和搜索）在 ARM64 PC 上变得无响应性。</p></li><li><p data-vmark="4c08">电池图标工具提示不会再意外地显示高于 100 的百分比。</p></li><li><p data-vmark="5186">当有大量打开的应用程序时，应用程序图标不会再与二级显示器上的日期和时间重叠。</p></li></ul><p data-vmark="f700">[文件资源管理器]</p><ul class=" list-paddingleft-2"><li><p data-vmark="0dff">做了一些工作来解决一个问题，即在使用 F2 重命名 OneDrive 文件时，有时按回车键后会导致失去键盘焦点。</p></li></ul><p data-vmark="5a15">[Spotlight 集合]</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="9657">启用 Spotlight 集合后，你的第一张图片（在 Whitehaven Beach 之后）现在应该更快地到达。</p></li><li><p data-vmark="760a">为 Spotlight 集合的上下文菜单条目添加了图标。</p></li></ul><p data-vmark="ca53">[输入]</p><ul class=" list-paddingleft-2"><li><p data-vmark="3c12">提高了调用语音输入的可靠性。</p></li><li><p data-vmark="6ff3">修复了当启用对比主题时，微软的文本输入体验（语音输入、表情符号面板等）的边界不能正确绘制的问题。</p></li><li><p data-vmark="464f">缓解了笔菜单进程的间歇性崩溃，如果它被启动，然后在启动之前立即关闭。</p></li></ul><p data-vmark="42c6">[小组件]</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="c793">微软修复了在使用悬停打开小组件面板时导致链接无法正常打开的问题。</p></li></ul><p data-vmark="da9f">[设置]</p><ul class=" list-paddingleft-2"><li><p data-vmark="0907">当设置窗口变小时，设置内容不会再被截断在窗口的一侧。</p></li><li><p data-vmark="57be">在打开组合框时，“设置”不会再零星地崩溃，这影响了某些设置，如为笔设置自定义点击动作的能力。</p></li><li><p data-vmark="6bf4">解决了蓝牙和设备中的“添加设备”选项在尝试连接新的蓝牙设备时悄悄崩溃的问题。</p></li><li><p data-vmark="7221">增加了一些关键词，使语音访问控制功能出现在设置搜索结果中。</p></li></ul><p data-vmark="f3a5">[其它]</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="c393">修复了导致 ARM64 电脑在前次版本中出现引用内存管理错误的错误检查问题。</p></li><li><p data-vmark="0f0d">修复了导致 DWM 在尝试使用某些应用程序时崩溃（导致屏幕反复闪烁）的问题。</p></li><li><p data-vmark="51d1">缓解了导致某些应用程序在 Narrator 讲述人运行时挂起的问题。</p></li><li><p data-vmark="cb93">在检查 narratorquickstart.exe 属性中的细节时，增加了一些缺失的信息。</p></li><li><p data-vmark="9288">解决了 Narrator 讲述人不会响应 UIA 事件的问题，如通知、实时区域或文本事件。</p></li></ul><p data-vmark="7b9a">注意：这里提到的来自活跃开发分支的 Insider Preview 构建中的一些修复可能会进入 10 月 5 日普遍可用的 Windows 11 发布版本的服务更新中。</p><h3 data-vmark="0148">已知问题</h3><p data-vmark="f82d">[一般]</p><ul class=" list-paddingleft-2"><li><p data-vmark="294b">你可能无法登录到某些应用程序，如 Feedback Hub。重新启动你的电脑应该可以纠正这个问题。</p></li><li><p data-vmark="af11">微软正在调查关于一些内测人士在最近的构建中看到驱动程序和固件更新失败的报告，错误为 0x8007012a。</p></li></ul><p data-vmark="62d3">[开始]</p><ul class=" list-paddingleft-2"><li><p data-vmark="dff3">在某些情况下，当从“开始”或任务栏使用搜索时，你可能无法输入文本。如果你遇到这个问题，请按键盘上的 WIN + R 来启动运行对话框，然后关闭它。</p></li></ul><p data-vmark="0ae8">[任务栏]</p><ul class=" list-paddingleft-2"><li><p data-vmark="4f15">切换输入法时，任务栏有时会闪动。</p></li><li><p data-vmark="67f6">网络图标有时会在任务栏中消失，而它本应在那里。如果你遇到这种情况，请尝试使用任务管理器来重新启动 explorer.exe。</p></li><li><p data-vmark="80cd">如果你有多个显示器连接到你的电脑，并在主显示器的任务栏上右键点击日期和时间，它将使 explorer.exe 崩溃。</p></li></ul><p data-vmark="54cf">[搜索]</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="f689">在点击任务栏上的搜索图标后，搜索面板可能无法打开。如果发生这种情况，请重新启动“Windows Explorer”进程，并再次打开搜索面板。</p></li></ul><p data-vmark="c0b6">[设置]</p><ul class=" list-paddingleft-2"><li><p data-vmark="cbad">在查看可用的 Wi-Fi 网络列表时，信号强度指标不能反映正确的信号强度。</p></li><li><p data-vmark="c955">进入系统 > 显示 > HDR 时，设置可能会崩溃。</p></li><li><p data-vmark="84c6">在蓝牙和设备下有一个空白条目。</p></li></ul><p data-vmark="be9d">[Spotlight 集合］</p><ul class=" list-paddingleft-2"><li><p data-vmark="bd33">如果你使用 Spotlight 集合，当前的图像目前没有在升级时迁移，这可能使你在升级到这个版本后有一个黑色的桌面背景。这个问题应该在下一次版本中解决。</p></li></ul><p data-vmark="63c0">[小组件]</p><ul class=" list-paddingleft-2"><li><p data-vmark="e836">改变任务栏的排列方式会导致小组件按钮从任务栏上消失。</p></li><li><p data-vmark="c673">当把入口点悬停在辅助显示器上时，小组件面板可能没有采用正确的分辨率。</p></li><li><p data-vmark="3486">小组件面板可能暂时是空白的。</p></li><li><p data-vmark="f18f">当有多个显示器时，任务栏上的小组件内容可能在显示器之间不同步。</p></li><li><p data-vmark="a9e5">在任务栏左对齐的情况下，温度等信息不会被显示。这将在未来的更新中被修复。</p></li></ul><p data-vmark="4c8a">[语音访问控制]</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="9854">一些文本创作命令，例如“选择那个”或“删除那个”，可能无法在 Windows 应用程序中按预期运行。</p></li><li><p data-vmark="69c7">对一些标点符号和符号的识别不准确，如 @符号。</p></li></ul>
