<p data-vmark="1540"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 8 日消息，据微软 Windows 博客发布，今天开始面向 Dev 频道中的所有 Windows Insiders 推出重新设计的 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 版记事本。</p><p data-vmark="c383"><img src="https://img.ithome.com/newsuploadfiles/2021/12/6667e8cd-d60e-488c-bde5-6a4112c29653.png" w="192" h="192" title="微软 Win11 Dev 全新记事本发布：圆角 / 云母 UI 现代设计，黑暗模式，多级撤销" width="192" height="192" referrerpolicy="no-referrer"></p><p data-vmark="7698">微软非常兴奋地向大家介绍重新设计的 Windows 11 记事本，其中包括一些其认为社区会喜欢的变化。首先，你会注意到一个完全更新的用户界面 UI，与 Windows 11 的新视觉设计相一致，包括圆角、云母等。微软表示，知道记事本对用户的许多日常工作流程有多重要，所以对这个经典的应用程序进行了现代化的设计，带来新鲜感觉，但又熟悉。</p><p data-vmark="b65a"><img src="https://img.ithome.com/newsuploadfiles/2021/12/e5195cee-fd95-4c70-a664-6a763c4ed0dc.png" w="1440" h="1257" title="微软 Win11 Dev 全新记事本发布：圆角 / 云母 UI 现代设计，黑暗模式，多级撤销" width="1440" height="716" referrerpolicy="no-referrer"></p><p data-vmark="07da">Windows 11 笔记本新功能是黑暗模式。这一直是社区的最高要求，微软希望你会喜欢这个华丽的新主题。默认情况下，记事本将适应你的系统主题偏好，但你可以在全新的设置页面中自行改变这一选项，该页面也是字体选项的新设置之处。</p><p data-vmark="45fe"><img src="https://img.ithome.com/newsuploadfiles/2021/12/c5fe3222-2cee-4d6b-ade6-fcf8efe8fa95.png" w="1440" h="1222" title="微软 Win11 Dev 全新记事本发布：圆角 / 云母 UI 现代设计，黑暗模式，多级撤销" width="1440" height="696" referrerpolicy="no-referrer"></p><p data-vmark="c80d">IT之家获悉，为了帮助你提高工作效率，微软引入了重新设计的查找和替换体验，并增加了对多级撤销的支持 —— 这是社区的另一个主要功能请求。</p><p data-vmark="3406"><img src="https://img.ithome.com/newsuploadfiles/2021/12/3c04a04c-0f41-441a-b50f-8305af8b5f13.png" w="1440" h="663" title="微软 Win11 Dev 全新记事本发布：圆角 / 云母 UI 现代设计，黑暗模式，多级撤销" width="1440" height="378" referrerpolicy="no-referrer"></p><p data-vmark="f891">生产力、性能和可靠性在记事本中是最重要的。无论你如何将记事本纳入你的工作流程，微软表示，都将确保记事本在这些方面继续保持出色。</p><p data-vmark="07b5">微软称，在今天发布的预览版中，有一些已知的问题，将在未来的更新中解决。这包括在不同的输入语言之间切换时或在使用日语 IME 输入法时影响键盘访问键的使用问题，以及在某些情况下，当使用 Shift-click 选择文本或滚动时，你可能注意到意外的行为。微软还在继续提高性能，以应对打开非常大的文件的情况。</p>
