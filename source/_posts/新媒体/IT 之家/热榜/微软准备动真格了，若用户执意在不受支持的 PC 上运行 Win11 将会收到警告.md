<p data-vmark="4e82"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 13 日消息，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 现已广泛推出，但要想升级正版 Win11 则用户还需满足官方的一系列要求。如果你的设备不符合要求，你仍然可以通过修改注册表等方式来进行偷渡。</p><p data-vmark="f55e">如果你在不符合要求的机器上安装了 Win11 系统，你将在接下来一段时间内看到 Windows 设置带来的一个新警告（提醒）。</p><p data-vmark="cc3a">在预览版中，微软已经在实装了一个警告性的通知，提醒用户“他们的设备不符合最低规格”要求。</p><p data-vmark="6469">目前，硬件不支持的警告已经出现在了 Win11 设置主页，看上去类似 Windows 激活错误的提醒。但通知可能会包括更多信息，例如警告用户“执意在不支持的设备上运行 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的后果”，包括缺少更新。</p><p data-vmark="7c37"><img src="https://img.ithome.com/newsuploadfiles/2022/2/c834058f-65b5-48ba-a519-7bd0a9915006.jpg@s_2,w_820,h_440" w="1048" h="562" alt="Windows 11 警告" title="微软准备动真格了，若用户执意在不受支持的 PC 上运行 Win11 将会收到警告" srcset="https://img.ithome.com/newsuploadfiles/2022/2/c834058f-65b5-48ba-a519-7bd0a9915006.jpg 2x" width="1048" height="440" referrerpolicy="no-referrer"></p><h2 data-vmark="692e">Windows 11 警告</h2><p data-vmark="c34d">微软将通过该提醒引导用户进入一个解释条款页面，强调它们为什么会不支持部分老电脑运行 Windows 11 以及可能导致的不兼容问题，而这些问题将不在任何制造商的保修范围内。</p><p data-vmark="2115">IT之家了解到，微软此前已经确认，在不支持的设备上运行 Windows 11 可能会由于“这些兼容性或其他问题”而遇到严重的问题。</p><p data-vmark="f06c">“不符合这些系统要求的设备将不再被保证接收更新，包括但不限于安全更新，”微软警告说。</p><p data-vmark="e718">众所周知，微软早在 Win11 测试初期就明确指出，用户虽然可以在不支持的硬件上运行 Windows 11（够正常运行 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 的设备大概能正常开机），但这是因为 Win11 本质上还是换用了新 UI 设计的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 系统，它在大多数现有硬件上都能启动，只是不那么完美。</p><p data-vmark="ae99">而且微软之前也警告称，偷渡用户将不会收到系统更新，虽然这也引起部分用户的欢呼但仍强行安装存在风险，只不过微软一直对这种现象睁一只眼闭一只眼，直到如今都没有采取实际措施进行制裁。</p>
