<p data-vmark="3599"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 14 日消息，据 Windows Central 报道，微软正在转向新的 Windows 计划，并恢复到更传统的 Windows 主要版本的三年发布周期，同时增加交付到当前 Windows 版本的新功能。</p><p data-vmark="5405">Windows Central  的消息来源称，<span class="accentTextColor">微软现在打算每三年发布一次 Windows 的主要版本</span>，目前计划在 2024 年发布下一个版本（暂称 Windows 12），比 2021 年发布 Windows 11 晚三年。这意味着最初计划的 <span class="accentTextColor">2023 年 Windows 版本（代号 Sun Valley 3）已被取消</span>。随着这一新的开发计划的推进，微软还计划增加向最新版 Windows 用户推出的新功能。</p><p style="text-align: center;" data-vmark="512d"><img src="https://img.ithome.com/newsuploadfiles/2022/7/3b6ac7d5-f40b-41d3-84b2-3650ee2db1e5.png" w="727" h="483" title="2024 年发布 Win12？消息称微软 Windows 开发周期大改：每三年发布一次主要版本，Sun Valley 3 已被取消" width="727" height="483" referrerpolicy="no-referrer"></p><p data-vmark="3e90">从 Windows 11 22H2（Sun Valley 2）开始，微软正在启动一项新的“<span class="accentTextColor">Moments</span>”工程，旨在让公司能够在全年的关键点推出新的功能和体验，而不是主要的操作系统版本。消息称微软打算从 2023 年开始，<span class="accentTextColor">每隔几个月向市场上的 Windows 版本发布新功能，每年最多四次</span>。</p><p data-vmark="96f1">微软已经在今年早些时候在 Windows 11 上推出了任务栏天气按钮来测试这个系统，同样的方法也将用于新的 Moments，该公司将把该系统与 Insiders 测试的新功能组合在一起，并将它们推广到面向所有用户最新发布的 Windows 版本。</p><p data-vmark="1e02">现已废弃的 Windows Sun Valley 3 版本计划的许多功能将作为其中一个 Moments 的一部分发布在 Windows 11 22H2（Sun Valley 2）之上。</p><p data-vmark="7b02">IT之家了解到，报道称关于 2024 年版本的 Windows 目前知之甚少，仍处于早期规划和工程阶段，微软内部人员在对话中称其为“<span class="accentTextColor">Next Valley</span>”，但不确定这是否是该项目的最终代号。</p><p data-vmark="92e1">总体而言，Windows 路线图的变化意义重大，将允许微软像过去一样每隔几年发布一次爆炸式的 Windows 新版本更新，同时保持市面上的 Windows 版本可以获得定期而频繁的新功能。</p>
