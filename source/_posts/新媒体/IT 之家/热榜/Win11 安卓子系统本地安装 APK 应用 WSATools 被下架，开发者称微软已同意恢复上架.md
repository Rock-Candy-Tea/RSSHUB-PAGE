<p data-vmark="e007"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 13 日消息，据 MSPoweruser 报道，几天前，微软已经从 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 应用商店中删除了 WSATools 工具 —— 这是为 Windows 11 <a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>子系统提供本地 APK 安装的第三方程序 —— 没有明确的原因。</p><p data-vmark="0c02"><img src="https://img.ithome.com/newsuploadfiles/2021/11/f23e2d3c-a59b-46ee-906a-1cc612b609f4.png" w="808" h="555" title="Win11 安卓子系统本地安装 APK 应用 WSATools 被下架，开发者称微软已同意恢复上架" width="808" height="555" referrerpolicy="no-referrer"></p><p data-vmark="0525">现在，经过一些争议后，微软似乎已经努力与 WSATools 开发者解决了这个问题（他称这是“正当的”），而且这个应用程序很快就可以上架使用。</p><p data-vmark="52fc"><img src="https://img.ithome.com/newsuploadfiles/2021/11/25c2a996-adf7-4a62-a3d8-887fd2a33f29.png" w="690" h="423" title="Win11 安卓子系统本地安装 APK 应用 WSATools 被下架，开发者称微软已同意恢复上架" width="690" height="423" referrerpolicy="no-referrer"></p><p data-vmark="76d1"><img src="https://img.ithome.com/newsuploadfiles/2021/11/dedd6e6c-9e43-4e7c-8b7c-b7f596af703d.png" w="684" h="429" title="Win11 安卓子系统本地安装 APK 应用 WSATools 被下架，开发者称微软已同意恢复上架" width="684" height="429" referrerpolicy="no-referrer"></p><p data-vmark="4b8d"><img src="https://img.ithome.com/newsuploadfiles/2021/11/250cf989-0384-44f2-acb9-12afce2a2ad8.png" w="683" h="520" title="Win11 安卓子系统本地安装 APK 应用 WSATools 被下架，开发者称微软已同意恢复上架" width="683" height="520" referrerpolicy="no-referrer"></p><p data-vmark="9c68">意大利开发者 Simone Franco 开发了一个微软商店的应用程序，使其像双击 APK 安装一样容易。WSATools 工具有如下描述。</p><blockquote><p data-vmark="6ad3">“易于 Windows Subsystem for Android™使用的 apk 安装程序。</p><p data-vmark="47d2">WSATools 是一个简单的 apk 安装程序，适用于 Windows 11 的 Android 子系统，设计得很直观，让用户立刻就有了家的感觉。”</p></blockquote><p data-vmark="0829"><img src="https://img.ithome.com/newsuploadfiles/2021/11/bd2a1beb-7aed-45e9-b81b-9417b789556d.png" w="795" h="557" title="Win11 安卓子系统本地安装 APK 应用 WSATools 被下架，开发者称微软已同意恢复上架" width="795" height="557" referrerpolicy="no-referrer"></p><p data-vmark="99cc">Franco 指出，该应用程序已经可以通过一个私人链接再次使用，并希望能很快恢复上架到应用商店中。</p><p data-vmark="42df">IT之家获悉，Windows 11 安卓子系统 (Windows Subsystem for <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a>) 包括 Linux 内核和基于 Android 开源项目（AOSP）版本 11 的 Android 操作系统。该子系统在 Hyper-V 虚拟机中运行，就像 Linux 子系统一样，可以将 AOSP 环境中 App 的运行时和 API 映射到 Windows 图形层、内存缓冲区、输入模式、物理和虚拟设备以及传感器，可以在英特尔、AMD、高通的 CPU 上运行。</p>
