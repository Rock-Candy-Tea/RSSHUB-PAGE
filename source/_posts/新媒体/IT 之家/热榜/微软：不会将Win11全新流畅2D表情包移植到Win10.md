<p data-vmark="12f8"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 27 日消息，据 Windows Latest 报道，微软终于开始在生产渠道面向 Windows 11 用户推出基于 Fluent Design 的新表情符号。新的表情包体验是基于微软流畅设计，它与 Windows 11 的现代外观和感觉相匹配。这个新的 Fluent 表情包还支持 Unicode 表情符号 13.1 版或更高版本。</p><p data-vmark="453f"><img src="https://img.ithome.com/newsuploadfiles/2021/11/4d52e7c0-2fbb-4d09-857f-17bb6a56a896.jpg" w="696" h="392" title="微软：不会将Win11全新流畅2D表情包移植到Win10" width="696" height="392" referrerpolicy="no-referrer"></p><p data-vmark="16cf">IT之家获悉，新的表情包体验与 Windows 11 KB5007262（Build 22000.348）一起推出，这是新的可选累积更新，未经用户许可，它不会下载或安装。</p><p data-vmark="bd22">KB5007262 是可选的累积性更新，但微软表示，很快表情符号体验将作为 Windows 11 强制更新上线。换句话说，你可以期待这些新的表情符号在 2021 年 12 月的“星期二补丁”中发布。</p><p data-vmark="b35f">不过，看起来微软并没有计划将这些现代表情符号带到 Windows 10。全新表情符号仍将是新 Windows 11 操作系统的专属，而 Windows 10 用户将不会得到任何新的表情符号，至少目前是这样。然而，这可能会在未来发生变化。</p><p data-vmark="07ce"><img src="https://img.ithome.com/newsuploadfiles/2021/11/cdb9dfca-7ee7-45e1-a0ad-6231b9063b01.jpg@s_2,w_820,h_473" w="928" h="535" title="微软：不会将Win11全新流畅2D表情包移植到Win10" srcset="https://img.ithome.com/newsuploadfiles/2021/11/cdb9dfca-7ee7-45e1-a0ad-6231b9063b01.jpg 2x" width="928" height="473" referrerpolicy="no-referrer"></p><p data-vmark="3545">与以前 Windows Phone 时代的表情符号不同，全新的表情符号看起来很现代，色彩丰富，更有趣。然而，新的表情符号看起来与微软之前预告的 3D 刷新版有很大不同。Windows 11 的新表情符号实际上是 2D 的，而不是 3D 的。</p>
