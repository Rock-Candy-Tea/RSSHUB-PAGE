<p data-vmark="f45b"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 29 日消息，据 Windows Latest 报道，微软终于开始向更多 PC 电脑推送 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>。我们的测试和微软声明表明，该公司在对一部分用户进行了一段时间的测试后，已经开始向更多符合最低硬件要求的设备推送 Windows 11。</p><p data-vmark="19ce"><img src="https://img.ithome.com/newsuploadfiles/2021/10/750341b7-4732-440a-825a-d26d889921e6.png" w="1213" h="1104" title="微软 Win11 正式版开始大范围推送，面向更多 PC 电脑" width="1213" height="746" referrerpolicy="no-referrer"></p><p data-vmark="5666">Windows 11 的广泛推送已经开始，用户必须满足最低硬件要求才有资格。然而，似乎有些用户尽管满足了最低要求，却无法立即下载 Windows 11。</p><p data-vmark="f9f0">Windows 11 是一个全新新版本的 Windows，具有现代设计和一些新功能，如 Windows 的 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 子系统和改进的内置应用程序。IT之家获悉，<a href="https://www.ithome.com/0/581/826.htm" target="_blank">新的 Windows 的 Android 子系统正在测试中</a>，已允许用户快速下载和侧载 Android 应用程序。</p><h2 data-vmark="6f9c">Windows 11 目前正在广泛推送</h2><p data-vmark="2af3">微软在今年 7 月开始测试 Windows 11，一些用户在 10 月 5 日收到了 Windows 11 正式版升级。</p><p data-vmark="76fe">在其管理中心发布的消息中，微软解释说，现在当更多用户检查更新时，Windows 11 将提供给他们。微软指出，Windows 11 的可用性已经提高，推送的方式是基于该公司最新版本的机器学习模型。</p><p data-vmark="fd30">“我们建议你将设备升级到 Windows 11，以利用最新的功能和对安全威胁的高级保护，”微软说。</p><p data-vmark="5eed">微软相信，其基于机器学习的推广将使更多的设备安全升级到 Windows 11。微软建议用户下载并安装 Windows 11 的安全功能，该公司将继续提供每月更新，以确保平稳的升级体验。</p><p data-vmark="df79"><img src="https://img.ithome.com/newsuploadfiles/2021/10/2e9f9811-5a2e-4d16-8a32-bcc6324d9100.jpg@s_2,w_820,h_652" w="1027" h="817" title="微软 Win11 正式版开始大范围推送，面向更多 PC 电脑" srcset="https://img.ithome.com/newsuploadfiles/2021/10/2e9f9811-5a2e-4d16-8a32-bcc6324d9100.jpg 2x" width="1027" height="652" referrerpolicy="no-referrer"></p><p data-vmark="7e5a">如果你使用的是 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a>，你可以按照以下步骤检查与 Windows 11 的兼容状态。</p><ul class=" list-paddingleft-2"><li><p data-vmark="4e2d">下载并安装 PC 健康检查应用程序。</p></li><li><p data-vmark="60c4">运行该工具，它将报告状态。</p></li><li><p data-vmark="9a5f">你也可以打开 Windows Update 设置，选择检查更新，以了解更多关于更新的信息。</p></li><li><p data-vmark="f296">如果升级对你的设备已经准备好了，你会看到下载和安装的选项。</p></li></ul><p data-vmark="b7a4">正如我们所指出的，Windows 11 主要推送对象将是满足最低要求的设备，其中包括 TPM 2.0 和安全启动。如果你想立即安装 Windows 11，在满足最低要求的的情况下，你不需要等待 —— 可以尝试下载 Windows 11 的 ISO 镜像，并使用媒体创建工具或升级助手手动安装。</p>
