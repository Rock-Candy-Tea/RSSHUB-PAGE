<p data-vmark="a942"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 26 日消息，微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> <a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>子系统可以让用户在电脑上运行安卓 App，不过侧载的时候有些麻烦，许多开发者都做出了更加方便的工具。</p><p data-vmark="cb21">据 Windows Latest 报道，又出现了一款微软 Win11 安卓子系统 WSA 开源工具，名为 <a href="https://github.com/alesimula/wsa_pacman/releases/" target="_blank">WSA PacMan</a>，采用 Win11 的原生 UI 风格开发，可替代 ADB 安装 App 方式，还能打开安卓应用设置。</p><p data-vmark="f4f1"><img src="https://img.ithome.com/newsuploadfiles/2021/12/de5e7f2e-02de-4367-b8cd-7d1810e955e3.png" w="755" h="557" title="微软 Win11 安卓子系统 WSA 全新开源工具上线：原生 UI，支持双击 APK 安装" width="755" height="557" referrerpolicy="no-referrer"></p><p data-vmark="83c3">从图中可以看到，WSA PacMan 拥有云母效果，支持亮 / 暗显示主题，并且可以绑定 APK 后缀文件。<span class="accentTextColor">Win11 用户可以像 .exe 应用那样，直接双击 APK 文件就可以进行安装</span>。</p><p data-vmark="73c5"><img src="https://img.ithome.com/newsuploadfiles/2021/12/ba0cf3f7-fcc1-48e6-a8f7-60332f7df909.png" w="94" h="87" title="微软 Win11 安卓子系统 WSA 全新开源工具上线：原生 UI，支持双击 APK 安装" width="94" height="87" referrerpolicy="no-referrer"></p><p data-vmark="1a4b"><img src="https://img.ithome.com/newsuploadfiles/2021/12/429ad105-c211-4b66-a7c3-8df5e8fb1caf.png" w="822" h="547" title="微软 Win11 安卓子系统 WSA 全新开源工具上线：原生 UI，支持双击 APK 安装" width="822" height="546" referrerpolicy="no-referrer"></p><p data-vmark="30ad">IT之家了解到，安装界面可显示版本、包名、权限信息等，当用户的 WSA 没有启动时，<span class="accentTextColor">支持</span><span class="accentTextColor">双击安装自动启动 WSA</span>。</p><p data-vmark="241a"><img src="https://img.ithome.com/newsuploadfiles/2021/12/983a287b-9f75-4c9b-800e-9b0e022b1f27.png" w="722" h="555" title="微软 Win11 安卓子系统 WSA 全新开源工具上线：原生 UI，支持双击 APK 安装" width="722" height="555" referrerpolicy="no-referrer"></p><p data-vmark="d019"><strong>WSA PacMan：</strong><a href="https://github.com/alesimula/wsa_pacman/releases/" target="_blank">点此下载</a></p>
