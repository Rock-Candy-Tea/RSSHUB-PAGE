<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 26 日消息 屏下摄像头一直是智能手机发展的目标之一，近年来也出现了一些实验性的产品，比如努比亚、OPPO 等都推出了屏下摄像头样机，中兴还推出过一款 Axon 20 量产机型，然而效果都不太令人满意。</p><p><img src="https://img.ithome.com/newsuploadfiles/2020/12/20201230233333_1449.jpg" title="OPPO 适配屏下摄像头新动向，系统 UI 可巧妙遮挡色差" referrerpolicy="no-referrer"></p><p>▲ 中兴 Axon 20</p><p>今日，数码博主 @数码闲聊站 曝光了一张 <span class="accentTextColor">OPPO 的系统 UI 图</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/2628d1ab-47f5-4cbe-86db-e72d6d80f4dd.png" w="912" h="880" title="OPPO 适配屏下摄像头新动向，系统 UI 可巧妙遮挡色差" width="912" height="791" referrerpolicy="no-referrer"></p><p>从图中可以看出，OPPO 为了适配屏下摄像头，<span class="accentTextColor">在摄像头的位置巧妙地放置了通知、信号图标</span>，外层的一圈可能是电量，这样就可以避免摄像头位置的屏幕色差导致的显示突兀。</p><p>IT之家了解到，今年 4 月份，数码博主 @数码闲聊站 爆料，OPPO 正在研发一款全无孔手机，<span class="accentTextColor">该机的前置摄像头采用了屏下设计，后摄则采用电致变色技术</span>，为了达到全无孔设计，该机还采用了侧边压感按键。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210406_091753_562.jpg@s_2,w_820,h_250" w="1047" h="319" title="OPPO 适配屏下摄像头新动向，系统 UI 可巧妙遮挡色差" width="1047" height="250" referrerpolicy="no-referrer"></p>
