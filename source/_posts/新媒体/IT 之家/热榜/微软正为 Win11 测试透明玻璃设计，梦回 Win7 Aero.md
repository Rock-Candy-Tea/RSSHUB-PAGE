<p data-vmark="8fd9"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 14 日消息，在 Windows 8 及其继任者 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 中，微软放弃了对圆角和 Aero Glass 的支持，转而采用更加扁平和中性色彩的用户界面。</p><p data-vmark="d838" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/c1bb0683-ec2a-4ef7-9952-7c6ac7abfc85.png" w="1440" h="900" title="微软正为 Win11 测试透明玻璃设计，梦回 Win7 Aero" width="1440" height="512" referrerpolicy="no-referrer"></p><p data-vmark="3b98"><a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 已经恢复了对圆角的支持，现在看起来下一个重大设计更新将与 <a class="s_tag" href="https://www.win7china.com/" target="_blank">Windows 7</a> 时代的 Aero 透明玻璃效果略有相似。</p><p data-vmark="870c">据 WindowsLatest 报道，经典的 Windows 应用程序的透明标题栏将卷土重来，可能是亚克力或新材料。目前，云母的效果由开发者决定，所以只在某些应用中拥有云母效果，其余的应用窗口都是简单的设计。</p><p data-vmark="7a2c">根据 1 月份 Windows 开发者计划网络广播期间意外曝光的 Windows Run 截图显示，<span class="accentTextColor">该应用的标题栏是透明的，微软似乎正在回归 Windows 7 和 Windows Vista 时代采用的 Windows Aero 风格。</span></p><p data-vmark="7e0e"><img src="https://img.ithome.com/newsuploadfiles/2022/2/d3ec0a49-a620-4cb3-832e-92e00b47f5e1.jpg" w="578" h="442" alt="Windows Run" title="微软正为 Win11 测试透明玻璃设计，梦回 Win7 Aero" width="578" height="442" referrerpolicy="no-referrer"></p><p data-vmark="3cb4">Windows 7 / Vista 的 Aero Glass 主题给了应用程序窗口标题栏（包括最小化、最大化和关闭等按钮的菜单）一个柔和的半透明外观。</p><p data-vmark="b192">IT之家了解到，微软有可能将亚克力效果用于所有经典应用的标题栏，而将云母效果保留给窗口。与亚克力效果不同的是，云母是一种不透明的效果，它适用于应用程序窗口、标题栏甚至背景。</p><p data-vmark="a280" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/6dce3b27-2c24-40d0-93a4-76db522dfdf9.png" w="745" h="420" alt="Windows 11 Acrylic" title="微软正为 Win11 测试透明玻璃设计，梦回 Win7 Aero" width="745" height="420" referrerpolicy="no-referrer"></p><p data-vmark="977d">▲ Windows 11 亚克力效果</p><p data-vmark="cdd2" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/44af2294-0142-4031-a348-c3d36f26ed72.png" w="745" h="420" alt="Windows 11 Mica" title="微软正为 Win11 测试透明玻璃设计，梦回 Win7 Aero" width="745" height="420" referrerpolicy="no-referrer"></p><p data-vmark="1002">▲ Windows 11 云母效果</p><p data-vmark="7c24">亚克力是一种花哨的、更耗费资源的类似于 Windows Aero 的效果，可以显示桌面壁纸和活动应用程序后面的其他窗口。</p><p data-vmark="7a44">目前还不清楚微软何时应用这一效果，据悉 Windows 11 的 22H2 版本，也就是所谓的太阳谷 2，可能会为应用程序的标题栏重新引入这种外观。</p>
