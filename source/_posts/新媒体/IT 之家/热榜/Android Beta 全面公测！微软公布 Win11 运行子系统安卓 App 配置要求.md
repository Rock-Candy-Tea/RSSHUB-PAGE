<p data-vmark="4abe"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 18 日消息，据 Windows Latest 报道，微软的 Windows Subsystem for <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 经过几个月的“炒作”和用户的期待，终于推出了。虽然该功能已经在 Insider 设备上已使用了近 6 个月，但它现在已经向拥有运行 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 21H2 版本的支持设备用户全面开放。</p><p data-vmark="9ef1" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/d33e2731-3413-4c01-80fd-185f7bb87c18.jpg" w="696" h="392" title="Android Beta 全面公测！微软公布 Win11 运行子系统安卓 App 配置要求：要有 SSD" width="696" height="392" referrerpolicy="no-referrer"></p><p data-vmark="5db3">在微软的 FAQ 网站上列出了<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a> Beta 测试版的支持，有近五个主要要求，你的设备必须满足这些要求，才能正常运行移动应用。目前支持安卓版本的 PC 的完整列表还没有出来，但你的设备如果满足 Windows 11 的要求，应该就可以运行移动应用。</p><p data-vmark="691b">具体来说，你需要一台运行 Windows 11 21H2 或更新版本的设备，至少拥有有 8GB 的内存（而 Windows 11 本身需要 4GB 内存），并且有一款微软正式支持的处理器。此外，你还需要从可选功能页面启用虚拟机平台功能。</p><p data-vmark="fd7d">下面是完整的要求配置清单：</p><ul class=" list-paddingleft-2"><li><p data-vmark="f7b9">内存：8GB（最低）和 16GB（推荐）。</p></li><li><p data-vmark="3ecd">存储类型：固态驱动器（SSD）。</p></li><li><p data-vmark="df97">处理器：你的设备应满足 Windows 11 的要求，即 Core i3 8th Gen，Ryzen 3000，Snapdragon 8c 或以上。</p></li><li><p data-vmark="1c31">处理器类型：x64 或 ARM64。</p></li><li><p data-vmark="1e20">虚拟机平台：这个可选的设置是用于虚拟化的，你需要从控制面板上启用该设置。</p></li></ul><p data-vmark="15a9">此外，如果你想从亚马逊 AppStore 下载并运行 1000 多个安卓应用程序，请确保设备的地区设置为美国地区，并使用英语语言。（实际上该限制已被破解，可以安装 APK 文件）</p><p data-vmark="f600">如果你满足上述要求，你可以下载并安装 Windows 11 最新的可选更新（KB5010414），其中包括安卓应用的公开预览版。对于美国的用户来说，他们可以在 Windows 11 上访问超过 1000 个亚马逊 AppStore 项目，包括流行的应用程序，甚至是游戏。</p><p data-vmark="d6e9">《<a href="https://www.ithome.com/0/603/227.htm" target="_blank">微软：全新 Win11 体验已发布，亚马逊应用商店预览版新增 1000 多个安卓 App，任务栏支持天气 / 静音 / 共享</a>》</p><p data-vmark="44b6">《<a href="https://www.ithome.com/0/603/200.htm" target="_blank">微软 Win11 Build 22000.527 (KB5010414) 正式发布：商店运行安卓 App，任务栏天气小组件...（附更新内容大全）</a>》</p><p data-vmark="9fdb">这包括 Audible、Kindle、Subway Surfers 和更多。微软商店中的亚马逊商店集成让你在 Windows 11 上下载安卓应用，就像从微软商店下载其他应用一样。</p><p data-vmark="5112">你将能够通过微软商店发现这些应用程序，但下载是由亚马逊应用商店提供的。</p><p data-vmark="8faf">Windows 11 的安卓子系统支持是来自于微软对 Windows Linux 子系统的贡献。</p><p data-vmark="c722" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/bf9a459d-8465-4c1e-ace5-956061d3a886.png" w="924" h="614" title="Android Beta 全面公测！微软公布 Win11 运行子系统安卓 App 配置要求：要有 SSD" width="924" height="545" referrerpolicy="no-referrer"></p><p data-vmark="9701">Android 集成是建立在开源安卓平台之上的，它得到了 Intel Bridge 技术的支持。安卓支持不限于亚马逊 AppStore，所以你也可以手动侧载应用程序，而不需要将地区和语言切换到美国（英语）。</p>
