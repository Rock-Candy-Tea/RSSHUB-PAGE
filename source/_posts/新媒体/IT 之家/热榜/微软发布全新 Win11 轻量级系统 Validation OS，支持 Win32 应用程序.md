<p data-vmark="286a"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 15 日消息，微软近期悄然发布了一款新的轻量级 Windows 11 版本。但是，<span class="accentTextColor">这款新的轻量级操作系统并不适合普通用户。</span>该系统基于命令行，并被称为“Microsoft Validation OS”，专为硬件或软件供应商、开发人员和技术人员设计，以帮助诊断和缓解问题。“Microsoft Validation OS”系统本质上有助于围绕 Windows 11 构建硬件和软件，而无需安装完整的东西。</p><p data-vmark="10b7" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/1aaf06fe-b07e-41b1-b7be-eba55e3ee16c.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzEzOS5wbmc=,g_3,x_14,y_14,a_0,t_100" w="902" h="1116" title="微软发布全新 Win11 轻量级系统 Validation OS，支持 Win32 应用程序" width="902" height="1015" referrerpolicy="no-referrer"></p><p data-vmark="1fc5">下面是微软定义其新 Validation OS 的方式：</p><blockquote><p data-vmark="48a7">“<span class="accentTextColor">Microsoft Validation OS 是一款轻量级、快速且可自定义的基于 Windows 11 的操作系统，你可以在工厂车间使用它来诊断、缓解和修复 Windows 设备制造过程中的硬件缺陷。</span>Validation OS 启动到命令行环境以提高工厂车间的可靠性并支持运行 Win32 应用程序，支持从早期硬件启动到零售操作系统和应用程序开发的平滑过渡。”</p></blockquote><p data-vmark="6d41">Validation OS 支持 Win32 应用程序，旨在帮助诊断基于 Windows 11 的开发中项目。这是因为 Win32 API 提供对系统硬件的直接访问，并具有各种诊断工具。</p><p data-vmark="de22">用户可以在以下找到有关 Validation OS 的更多详细信息和资源：</p><ul class=" list-paddingleft-2"><li><p data-vmark="d7e7"><a Validation OS 镜像）</p></li><li><p data-vmark="a0c6"><a href="https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/validation-os-boot?view=windows-11" target="_blank">Setup a bootable Validation OS installation</a>（设置可引导的 Validation OS 安装）</p></li><li><p data-vmark="bee9"><a href="https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/validation-os-optional-packages?view=windows-11" target="_blank">Validation OS feature packages reference</a>（Validation OS 功能包参考）</p></li><li><p data-vmark="5da8"><a href="https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/validation-os-run-an-app-on-boot?view=windows-11" target="_blank">Launching an App or command when Validation OS starts</a>（Validation OS 启动时，启动应用程序或命令）</p></li><li><p data-vmark="c533"><a href="https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/validation-os-debug-apps?view=windows-11" target="_blank">Debugging Apps in Validation OS</a>（在 Validation OS 中调试应用程序）</p></li></ul><p data-vmark="ef24">如果你想尝鲜 Microsoft Validation OS ，可通过<a href="https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/validation-os-overview?view=windows-11" target="_blank">该链接</a>下载 Windows 镜像 (.WIM) 文件，适用于 ARM64 和 AMD64 架构。</p>
