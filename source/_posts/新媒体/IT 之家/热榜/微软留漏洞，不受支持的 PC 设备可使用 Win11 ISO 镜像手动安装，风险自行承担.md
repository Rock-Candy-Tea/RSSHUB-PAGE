<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 28 日消息 外媒 Windows Latest 报道，微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 带来了两个重大变化：设计上的更新和对新老用户的严格硬件要求。</p><p>今年晚些时候，Windows 11 将开始向微软、惠普、联想、戴尔和其他合作伙伴的兼容 PC 硬件推出。新的操作系统是可选的更新，如果你选择升级现有的设备（只要它满足必要的硬件要求），你在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 上安装的所有东西都可以在 Windows 11 上正常运行。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/78d18c7c-5bdd-4525-af63-d81e6445256f.png" w="1440" h="960" title="微软留漏洞，不受支持的 PC 设备可使用 Win11 ISO 镜像手动安装，风险自行承担" width="1440" height="547" referrerpolicy="no-referrer"></p><p>严格来说，Windows 11 是采用新设计的 Windows 10，但运行它的硬件要求相对更加严格。对于 Windows 10 来说，具有 1GHz 处理器、1GB 内存和 6GB 存储空间的设备被认为是“兼容”的。在 Windows 11 上，你将需要现代的硬件。</p><p>根据微软及其合作伙伴发布的官方文件，你需要英特尔第八代或更新的处理器来运行新的操作系统。微软目前正计划支持英特尔的第 8 代和 AMD 的 Ryzen 2000 系列芯片，<a href="https://www.ithome.com/0/572/066.htm" target="_blank">还有少量的英特尔第七代芯片</a>，而对于旧硬件来说，看起来一切都结束了。</p><p>值得庆幸的是，微软并不打算禁用那些允许任何人运行 Windows 11 的“漏洞”。</p><h2>Windows 11 可安装在不受支持的硬件上</h2><p>为了帮助企业和 IT 管理员为新的体验评估他们现有的配置，Windows 11 将通过 Media Creation Tool 工具或镜像安装媒介在不支持的设备上提供系统。</p><p>但微软不会在不受支持的设备上推广或建议使用 Windows 11，并且驱动支持是有限的。旧设备不会得到支持，因为微软的内部遥测数据显示，由于内核模式崩溃和其他性能问题，旧硬件无法可靠地运行 Windows 11。</p><p>微软的一份声明表示，</p><blockquote><p>Windows 11 不会通过 Windows Update 或 WSUS 为不支持的设备提供业务。你将能够使用媒体创建工具或安装镜像（.ISO）在“不支持的”设备上安装 Windows 11。</p></blockquote><p>或者你也可以使用 Windows Insider 计划来运行新的操作系统。</p><p>换句话说，<span class="accentTextColor">你其实不必担心你的 PC 是否包括较新的处理器或 TPM 2.0</span>。只要你能接受问题 Bug 的存在和微软相对有限的支持，你就可以在自己的设备上下载并运行 Windows 11，但需要注意的是，造成的风险后果需要自行承担。</p><p>IT之家获悉，微软还发布了一个改进版的 Windows 11 兼容性检查工具 <a href="https://www.ithome.com/0/572/070.htm" target="_blank">“PC 健康检查”预览版</a>，方便用户检测自己的设备是否适合通过微软支持渠道升级到 Windows 11。</p><p><strong>扩展阅读：</strong></p><p>《<a href="https://www.ithome.com/0/572/064.htm" target="_blank">微软 Win11 Build 22000.168 (KB5005191) 预览版发布：Teams 更多语言，全新 Microsoft 365 小组件，应用商店库界面更新</a>》</p><p>《<a href="https://www.ithome.com/0/570/399.htm" target="_blank">纯净安装，微软 Win11 Build 22000.132 预览版 ISO 官方镜像下载</a>》</p>
