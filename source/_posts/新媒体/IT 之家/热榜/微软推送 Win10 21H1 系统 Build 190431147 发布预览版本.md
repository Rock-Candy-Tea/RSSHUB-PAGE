<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 16 日消息 今天微软发布了 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 21H1 Build 19043.1147 (KB5004296) 更新，面向 Release Preview 发布预览频道推送。</p><p>微软表示，“注意：21H1 的 Beta 通道中的 Windows Insiders 将不会收到这个更新。<a href="https://www.ithome.com/0/563/353.htm" target="_blank">我们将很快向这些内测人员推送 Windows 11 Insider Preview 构建版本</a>。”</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/f850caf4-b3b1-42fb-9f47-13a230d1512c.png" w="1024" h="683" title="微软推送 Win10 21H1 系统 Build 19043.1147 发布预览版本" width="1024" height="547" referrerpolicy="no-referrer"></p><p><strong>Windows 10 Build 19043.1147 (21H1) 预览版包括以下改进修复：</strong></p><ul class=" list-paddingleft-2"><li><p>修复了搜索索引器的一个问题。在你退出后，搜索索引器继续持有每个用户搜索数据库的句柄，在配置文件路径“C:\Users\username\AppData\Roaming\Microsoft\SearchData\Applications\”上存在。因此，搜索索引器停止工作，并创建了重复的配置文件名称。</p></li><li><p>修复了阻止游戏服务为桌面用户打开某些游戏的问题 Bug。</p></li><li><p>修复了使你无法使用输入法编辑器（IME）输入文本的问题。例如，在启动后，如果你设置了电源选项，通过关闭笔记本电脑的盖子来关闭笔记本电脑，这可能会发生问题。</p></li><li><p>改变了将新活动上传至时间线的功能。如果你使用微软账户（MSA）在设备上同步活动历史，你不能把新的活动上传到时间线。但你仍然可以使用时间线，在本地设备上看到活动历史（关于最近的应用程序、网站和文件的信息）。这并不影响 Azure Active Directory（AAD）账户。要查看网络历史，微软 Edge 和其他<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>提供了查看最近网络活动的选项。你也可以使用微软 OneDrive 和 Office 查看最近使用的文件。</p></li><li><p>修复了当你在映射网络驱动器时，可能会导致文件资源管理器窗口失去焦点的问题。</p></li><li><p>修复了当你删除映射的网络驱动器上的许多文件时，会导致文件资源管理器在达到 99% 的完成度后停止工作的问题。</p></li><li><p>修复了组策略注册表遥测中的时间问题，会导致组策略扩展处理失败。</p></li><li><p>修复了重复重建 Windows 过滤平台（WFP）过滤器的问题。当设备被注册到移动设备管理（MDM）服务并且且“MDMWinsOverGP”被设置时，这个问题就会发生。</p></li><li><p>修复了 MDM 服务的问题，未能正确应用某些垃圾邮件规则。</p></li><li><p>修复了在向 MDM 服务注册期间，设备上的更新构建修订（UBR）总是报告为零（0）的问题。</p></li><li><p>修复了导致椭圆曲线数字签名算法（ECDSA）证书注册失败，出现错误“0x80090027 NTE_INVALID_PARAMETER”的问题。这个问题发生在可信平台模块（TPM）提供者（微软软件密钥存储提供者）存储密钥时。</p></li><li><p>修复了审核事件 4624 和 5142 的问题，当荷兰语是显示语言时，会显示错误的事件模板。</p></li><li><p>修复了导致 System Integrity 泄漏内存的问题。</p></li><li><p>修复了当你按下游戏控制手柄上的触发按钮时，会大声播放在游戏中选择东西的声音的问题。</p></li><li><p>修复了使电源计划和游戏模式无法按预期工作的问题。这将导致游戏时帧率降低和性能下降。</p></li><li><p>修复了即在从某些域访问互联网的系统上，任务栏网络图标上出现“网络内部访问”的问题。</p></li><li><p>修复了即在你连接到虚拟专用网络后，网络连接状态指示器（NCSI）无法检测到互联网连接的问题。</p></li><li><p>修复了导致打印停止或打印错误输出的问题。这个问题发生在更新到 Windows 10 2004 版本或更高版本后使用 USB 连接打印时。</p></li><li><p>修复了可能会降低应用程序的性能的罕见问题，这些应用程序在每次调用 Gdiplus.dll!GdipMeasureString 时都会有一个新字体的紧密循环。这个问题发生在安装 2021 年 2 月及以后发布的 Windows 更新之后。</p></li><li><p>修复了即当使用某些固定的通道布局进行流媒体时，会错误地路由一些音频通道的问题。</p></li><li><p>修复了即 Set-RDSessionCollectionConfiguration 没有设置 camerastoredirect:s:value 自定义属性的问题。</p></li><li><p>修复了特权访问管理（PAM）部署中报告的本地安全授权子系统服务（LSASS）域控制器内存泄漏问题。</p></li><li><p>修复了在签出后无法访问映射到分布式文件系统（DFS）根的网络驱动器的问题。</p></li><li><p>修复了在登录后无法重新连接到映射的网络驱动器，并显示拒绝访问的错误的问题。如果你使用 net use /deep 选项在同一个加密文件共享上创建多个驱动器映射到不同的路径，就会出现这个问题。</p></li><li><p>修复了 Windows 网络文件系统（NFS）客户端中中可能会阻止你在挂载 NFS 共享后重命名文件的问题。如果你使用文件资源管理器重命名文件，这个问题会发生，但如果你使用命令行重命名文件，则不会发生。</p></li><li><p>修复了关于未处理的“打开文件”对话框关键异常的问题。因此，微软基础类（MFC）应用程序可能会意外地关闭。</p></li><li><p>修复了即设置中的“存储感知”页面可能会错误地报告某些使用 GUID 分区表（GPT）的存储设备的大小的问题。受影响的设备会在 Storage Sense 中错误地报告，其大小是文件资源管理器中报告大小的两倍。注意：这个问题不影响使用主引导记录（MBR）的存储设备。</p></li></ul>
