<p><strong>更新：</strong></p><blockquote><p>经过网友指正，以及IT之家测试发现，iCloud for Windows 11 指的是 iCloud 的版本号，而非 Windows 11，在此指正道歉，抱歉发布误导文章。</p></blockquote><p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 7 日消息 微软将于 6 月 24 日发布“下一代 Windows”，之前已经有多个迹象表明，下一代 Windows 可能被称为“Windows 11”。</p><p>IT之家网友 @yang_洋 最新发现，<span class="accentTextColor">苹果在 iCloud 的<a href="https://support.apple.com/en-us/HT204363#ICW10" target="_blank">支持页面</a>已经出现了 Win11</span>，并且将其和 2019 年 5 月更新后的 Win10 并列说明。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/a8fd3870-32aa-4cea-a068-cc22a03e6633.png" w="1159" h="601" title="苹果 iCloud 支持页面出现 Windows 11 字样，但与微软无关" width="1159" height="425" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/3d600a84-9f06-4e25-b364-d1c6d7b737ea.png" w="1139" h="540" title="苹果 iCloud 支持页面出现 Windows 11 字样，但与微软无关" width="1139" height="389" referrerpolicy="no-referrer"></p><p>而图中的 Win7.x，则是指 Win7、Win8、Win8.1，以及早期版本的 Win10。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/b50267e1-9d9b-4252-9aa7-7a4cf7fb5023.png" w="1127" h="177" title="苹果 iCloud 支持页面出现 Windows 11 字样，但与微软无关" width="1127" height="129" referrerpolicy="no-referrer"></p><p>IT之家此前报道了微软在最新海报中隐藏的种种细节，暗示“下一代 Windows”为 Win11，而苹果的这一“助攻”，则是 Win11 存在的又一有力证据。</p><p>在过去的六年里，微软一直在通过软件更新对 Windows 10 进行定期改进，但这可能很快就会改变。在 5 月 25 日的微软 Build 开发者大会上，微软首席执行官萨提亚 - 纳德拉说，微软正在计划“过去十年中最重要的 Windows 更新之一”，证实了 Windows 操作系统的重大重新设计即将到来的传言。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/18a1ca18-1fab-4766-8a88-36e167950f1a.jpg@s_2,w_820,h_384" title="苹果 iCloud 支持页面出现 Windows 11 字样，但与微软无关" referrerpolicy="no-referrer"></p><p>虽然“太阳谷更新”将是一次大的升级，但它仍将被视为另一个版本的 Windows 10。如果微软想在下一个 Windows 更新中大做文章，需要从简单的“Windows 10 + 月份”的名称品牌中走出来。</p>
