<p data-vmark="09fb"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 10 日消息，微软在最新的教育活动中正式发布了 <span class="accentTextColor"><a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> SE 系统</span>，将搭载于多个品牌的教育类笔记本中。</p><p data-vmark="2769"><img src="https://img.ithome.com/newsuploadfiles/2021/11/4267270c-6554-48a7-b574-442cdcf9f8e9.png" w="1062" h="590" title="微软 Win11 SE 系统正式发布：面向教育市场用户，无法升级到 Win11" width="1062" height="456" referrerpolicy="no-referrer"></p><p data-vmark="b16b"><span class="accentTextColor">Windows 11 SE 针对最常用的教育体验进行了优化</span>，包括 Microsoft 365 应用程序，支持在线和离线功能。Microsoft Office 应用程序（包括 Word、PowerPoint、Excel、OneNote 和 OneDrive）可作为 Microsoft 365 许可证的一部分在 Windows 11 SE 设备上脱机使用。OneDrive 在每个设备上本地存储文件，供学生离线时访问。一旦他们通过互联网访问回到学校，Windows 11 SE 将自动同步离线更改。</p><p data-vmark="de0e">Windows 11 SE 还广泛支持基于 Web 的学习应用程序。借助 Edge <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>，最流行的扩展程序和其他基于 Web 的工具都能很好地工作。Edge 浏览器还带来了 Immersive Reader 等内置工具，让更多学生可以访问互联网内容。Windows 11 SE 还支持包括 Zoom 和 Chrome 在内的第三方应用程序，并致力于继续扩展第三方应用程序支持，以包括最常见的考试、可访问性、内容过滤和课堂编排应用程序。</p><p data-vmark="b224">对于 IT 管理员，Windows Autopilot 提供零接触设备部署，而 Intune for Education 提供简单且安全的云管理解决方案，允许 IT 管理员从任何地方开箱即用地部署和管理 Windows 11 SE 设备。通过云进行管理时，设备始终处于最新状态。它们通过 Intune for Education 中设置的策略在课外静默、自动地更新，以最大限度地减少中断，同时提供最新的安全性和可靠性更新。Intune for Education 还允许 IT 管理员向每个学生设备安全地提供正确的应用程序和策略。为了帮助确保设备和学生的安全，只有 IT 管理员可以直接将应用程序和服务下载并安装到设备上。</p><p data-vmark="97c1"><img src="https://img.ithome.com/newsuploadfiles/2021/11/1cb64742-e5bd-4050-b8fa-5d50582dd073.png" w="1423" h="794" title="微软 Win11 SE 系统正式发布：面向教育市场用户，无法升级到 Win11" width="1423" height="458" referrerpolicy="no-referrer"></p><h2 data-vmark="d173">主要特性：</h2><ul class=" list-paddingleft-2"><li><p data-vmark="bb63">Windows 11 SE 拥有更加简化的用户界面，可最大程度地减少干扰并专注于教育；</p></li><li><p data-vmark="6467">没有适用于 Windows 11 SE 的 Microsoft Store，因此 IT 教育工作者可以控制学生下载哪些应用和工具；</p></li><li><p data-vmark="39bd">Windows 11 SE 具有内置增强功能，可优化低成本硬件的性能；</p></li><li><p data-vmark="d92b">无法从 Windows 11 SE 升级或切换到 Windows 11；</p></li><li><p data-vmark="f5b4">无法通过 ISO 镜像本地安装 Windows 11 SE；</p></li><li><p data-vmark="8610">Windows 11 SE 的系统要求与 Windows 11 基本相同，但内存和硬盘容量要求更低。</p></li></ul><p data-vmark="2374"><img src="https://img.ithome.com/newsuploadfiles/2021/11/0a0ce345-1a3a-4374-9925-e880e8c7cae1.png" w="1440" h="810" title="微软 Win11 SE 系统正式发布：面向教育市场用户，无法升级到 Win11" width="1440" height="461" referrerpolicy="no-referrer"></p><p data-vmark="accf">IT之家了解到，微软表示，<span class="accentTextColor">Windows 11 SE 中的“SE”实际上并不是任何事物的缩写</span>，只是为了与其他版本的操作系统区分开来。Windows 11 SE 本身针对教育环境，尤其是 K-8 市场。</p><p data-vmark="db3b">宏碁、华硕、戴尔、Dynabook、富士通、惠普、JP-IK、联想和 Positivo 等品牌都将推出搭载 Windows 11 SE 的笔记本电脑，<span class="accentTextColor">预计大部分设备将在今年晚些时候和 2022 年通过教育渠道订购</span>。</p><p data-vmark="9159"><img src="https://img.ithome.com/newsuploadfiles/2021/11/91741d2a-1b90-4481-bd8d-5b7a0c110b98.png" w="1440" h="629" title="微软 Win11 SE 系统正式发布：面向教育市场用户，无法升级到 Win11" width="1440" height="358" referrerpolicy="no-referrer"></p><p data-vmark="569b">不过，<span class="accentTextColor">Windows 11 SE 将在除中国外的全球范围内推出</span>。微软表示，仍在研究如何在中国推出下一版本的 Windows 11 SE。</p>
