class="accentTextColor"><strong>F2</strong></span>：<strong>重命名</strong>。鼠标也可以进行重命名，具体操作为：先<strong>单击选中</strong>文件，然后<strong>停顿一下</strong>，<strong>再次单击</strong>文件即可进行重命名。</p></li></ul><ul class=" list-paddingleft-2"><li><p data-vmark="0307"><span class="accentTextColor"><strong>Alt + Enter</strong></span>：快速打开文件或文件夹的<strong>属性对话框</strong>，方便查看文件或文件夹的属性信息。</p></li></ul><ul class=" 文件资源管理器" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="0e5b"><span class="accentTextColor"><strong>Alt + ←</strong></span> 或 <span class="accentTextColor"><strong>Backspace</strong></span>：后退至<strong>上一次</strong>访问的目录。</p></li><li><p data-vmark="8939"><span class="accentTextColor"><strong>Alt + →</strong></span> 或 <span class="accentTextColor"><strong>Enter</strong></span>：前进至<strong>后一次</strong>访问的目录。</p></li><li><p data-vmark="f4dc"><span class="accentTextColor"><strong>Alt + data-vmark="9508"><span class="accentTextColor"><strong>Ctrl + A</strong></span>：按下该组合键即可全部选中所有文件或文件夹。另外，利用鼠标也可以完成全选。操作方式：<strong>按住鼠标不放</strong>，<strong>按对角线方向拖动鼠标</strong>（例如从右下方拖动至左上方），该操作适合于<strong>文件较少</strong>的场景。</p></li></ul><p data-vmark="d537"><img src="https://img.ithome.com/mpimg/content/11653976/2021/11/20211127_172101_328.gif" title="Win10/Win11学院：玩转 Windows 文件资源管理器" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="4928"><span class="accentTextColor"><strong>Shift + 鼠标两次单击</strong></span>：具体操作为：按住 Shift 键不放，首先单击第一个文件，然后单击最后一个文件，即可快速选中多个连续的文件，又称<strong>连续选中</strong>。</p></li></ul><ul class=" list-paddingleft-2"><li><p data-vmark="35f7"><span class="accentTextColor"><strong>Ctrl + 鼠标多次单击</strong></span>：具体操作为：按住 Ctrl 键不放，依次单击要选中的文件，即可快速选中多个不连续的文件，又称<strong>不连续选中</strong>。</p></li></ul><h2 文件资源管理器" referrerpolicy="no-referrer"></p><p data-vmark="bbe4">当然，还可以右击该库将其固定至 “快速访问”，这样访问就更加方便了。</p><h2 data-vmark="2b8e">五、第三方工具增强文件资源管理器</h2><p data-vmark="e109"><strong>1.预览文件的方法</strong></p><p data-vmark="0104">首先来看预览文件的<strong>使用场景</strong>：</p><p data-vmark="6cfc">(1) 我们需要快速找到自己所需的文件，若可以预览，则可以快速查看其缩略图或内容；</p><p data-vmark="c593">(2) 我们只是查看或复制文档的内容，并不需要编辑文档内容。</p><p data-vmark="12bd">我们可以在文件资源管理器的<strong>预览窗格</strong>中预览文件，但预览窗格默认是关闭的，这里推荐使用 <span class="accentTextColor"><strong>Alt+P</strong></span> 组合键打开预览窗格，<strong>操作方法</strong>：打开文件资源管理器后，按下该组合键即可打开预览窗格，再次按下即可关闭。<strong>记忆方法</strong>：预览的英文为<strong>P</strong>review。下图中红框标记的区域即为预览窗格。</p><p data-vmark="b973"><img src="https://img.ithome.com/mpimg/content/11653976/2021/11/20211121_232010_156.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzEwMC5wbmc=,g_7" title="Win10/Win11学院：玩转 Windows 文件资源管理器" referrerpolicy="no-referrer"></p><p data-vmark="da37">默认情况下，文件资源管理器可以预览的文件格式有（仅列举最常见的）：</p><ul class=" list-paddingleft-2"><li><p data-vmark="5636"><strong>txt </strong>格式的文本文件，预览效果如下图所示；</p></li></ul><p data-vmark="b3a0"><img src="https://img.ithome.com/mpimg/content/11653976/2021/11/20211121_232039_493.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzEwMC5wbmc=,g_7" title="Win10/Win11学院：玩转 Windows 文件资源管理器" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="14c2"><strong>jpg </strong>和 <strong>png </strong>格式的的图片；</p></li><li><p data-vmark="ce1d"><strong>MP4</strong>、<strong>RMVB</strong>、<strong>AVI</strong>、<strong>3GP </strong>等格式的视频（仅预览缩略图）；</p></li></ul><p data-vmark="bf1f">像常见的如 <strong>pdf </strong>文件以及 Office 格式（如 <strong>docx、xlsx、pptx </strong>等）的文档则无法在预览窗格进行预览<strong>。</strong>下面提供<strong>两种方案来解决这个问题。</strong></p><p data-vmark="e824"><span class="font-color-red">方案1：安装 Microsoft Office + Sumatra PDF </span></p><p data-vmark="09b1">就目前而言，绝大多数的新电脑都会预装正版的 Microsoft Office，则文件资源管理器自然可以预览 Office 格式的文档。</p><p data-vmark="09b1">预览 pdf 文件可以安装 <a href="https://www.sumatrapdfreader.org/download-free-pdf-viewer" target="_blank">Sumatra PDF</a> 软件。这里需要注意的是：在安装软件时，需<strong>勾选上【让 Windows 显示 PDF 文件的预览】选项</strong>，具体如下图所示。</p><p data-vmark="39c1"><img src="https://img.ithome.com/mpimg/content/11653976/2021/11/20211120_211314_457.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzEwMC5wbmc=,g_7" id="1637413994845-2" title="Win10/Win11学院：玩转 Windows 文件资源管理器" referrerpolicy="no-referrer"></p><p data-vmark="3b93"><span class="font-color-red">方案2：安装 Microsoft Office + PowerToys</span></p><p 文件，还可以预览 Markdown(<strong>.md</strong>)、SVG(<strong>.svg</strong>)格式的文件。</p><p data-vmark="9d4c"><img src="https://img.ithome.com/mpimg/content/11653976/2021/11/20211121_201156_507.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzEwMC5wbmc=,g_3" title="Win10/Win11学院：玩转 Windows 文件资源管理器" referrerpolicy="no-referrer"></p><p data-vmark="034a"><span class="font-color-red">方案3：安装 QuickLook 及其 OfficeViewer 插件</span></p><p data-vmark="ecfb">若办公软件使用的是 WPS Office，则可以安装 <strong>QuickLook</strong>，它也是开源免费的，下载地址：<a href="https://github.com/QL-Win/QuickLook/releases" target="_blank">msi下载</a> 或 <a class="font-color-red"><strong>还需额外安装该软件的插件</strong></span>，插件下载地址：<a href="https://github.com/QL-Win/QuickLook/wiki/Available-Plugins">Available Plugins · QL-Win/QuickLook Wiki · GitHub</a> 。</p><p data-vmark="6c33">插件的安装：</p><p data-vmark="6c33">(1) 安装并运行 QuickLook ，</p><p data-vmark="6c33">(2)下载 .qlplugin 格式的插件（这里下载的 OfficeViewer v4），用鼠标单击选中下载好的插件，按下<strong>空格键</strong>，在弹出的对话框（如下图），点击【Click here to install this plugin】按钮安装即可，最后<strong>重启</strong> QuickLook 软件。</p><p data-vmark="8994"><img title="Win10/Win11学院：玩转 Windows 文件资源管理器" referrerpolicy="no-referrer"></p><p data-vmark="939c">(3) <strong>预览文件</strong>：打开 QuickLook 软件并使其驻留后台，然后选中文件后按下<strong>空格键</strong>即可预览文件，按下 <strong>Esc</strong> 键（或点击关闭按钮）即可取消并关闭预览窗口。</p><p data-vmark="ab9a"><img src="https://img.ithome.com/mpimg/content/11653976/2021/11/20211122_091338_304.gif" title="Win10/Win11学院：玩转 Windows 文件资源管理器" data-vmark="e750">方案3：<strong>预览的文件格式更加丰富</strong>，可以满足不同职业人士的需求，但需要 QuickLook <strong>在后台保持运行</strong>（默认开机自启）。</p><p data-vmark="5390"><strong>2.多标签页管理文件和多列浏览文件</strong></p><p
