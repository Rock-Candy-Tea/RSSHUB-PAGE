<p>从未见过微软对一件事能捂得这么“严”，2021 年 6 月 3 日，微软宣布将于 6 月 24 日发布下一代 Windows（Next generation of Windows），很多媒体都将其理解为 Windows 11 将至。纳尼…… 有些怪是吗？六年前，同样也是这家公司，昭告天下说 Win10 将是微软的最后一版操作系统！！！这…… 似乎让事情变得有些复杂。同样另一个疑惑也随之而来，距离 6 月 24 日已经不足 20 天了，传说中的“下一代操作系统”依旧没有任何消息。按照微软从前的套路，改个图标都得昭告天下大庆三天，这么大的改变竟然没有一丁点消息，着实有些让人琢磨不透！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/5904724a-f719-43c6-a28d-054f01d0cd02.jpg" w="700" h="525" title="下代 Windows 将至？有关微软“Win11”的畅想" width="700" height="525" referrerpolicy="no-referrer"></p><p>▲传说中的“Windows 11”是啥样？</p><p>好吧，不管基于什么原因，6 月 24 日已经离我们很近了，到底 Next generation of Windows 是 Win11，还是，之前发展路线上早已明昭的 Win10 21H2“太阳谷”，届时也将有个定论。以下这篇文章，既包含了一部分“爆料”，也有我们关于这款即将到来的新操作系统的猜测。甭管结果如何，这些都代表着大家对于这款新操作系统的期待，一起来看吧！</p><h2>1. 新版 UI</h2><p>在之前已经爆料的“太阳谷”中，微软宣称要给 Windows 一套崭新的 UI，试图解决现版 Win10 中一个棘手的问题 —— 多风格 UI 混搭。其实有关微软的审美，业界早有定论，概括起来就是两句话“生不逢时，老的不愿换！“。在下一版 Windows 的猜测中，我们也很难绕过一个已经被微软判过死刑的系统 ——Win10X。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/ab415e88-b946-4919-9747-32115e0cf28e.jpg" w="700" h="394" title="下代 Windows 将至？有关微软“Win11”的畅想" width="700" height="394" referrerpolicy="no-referrer"></p><p>▲下版 Windows 概念图</p><p>Win10X 曾是微软专为双屏设备打造的一款操作系统，在 Win10 推出四年后的 2019 年推出。作为一款全新的 UI 体系，Win10X 解决掉了 Win10 在 UI 层面上的很多问题，比如一体式窗口、更加和谐统一的外观等等。而这恰恰与微软，关于下一代 Fluent Design 的论述不谋而合。事实上从最近几次 Win10 Dev 的更新也能看出，微软正有计划地将所有界面全部统一到一个框架，更现代化外观 + 圆角设计 + 悬浮式选单 + 毛玻璃。而这种变化可能不再局限于操作系统本身，功能模块、内置组件、第三方软件都有囊括。未来 Windows 或许将成为一个类似于 MacOS 的统一化 UI 体系，这才是微软带给我们的最大惊喜！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/77442f11-ae44-43ef-a51a-cdb1ce2732c9.jpg" w="700" h="395" title="下代 Windows 将至？有关微软“Win11”的畅想" width="700" height="395" referrerpolicy="no-referrer"></p><p>▲下版 Office UI 概念图（图片来自微软）</p><h2>2. 开始菜单</h2><p>作为 Windows 中倍受关注的功能组件，开始菜单一直都是大家关心的焦点。在 Win10X 中，微软试图通过一个独立式菜单来替代现有的开始菜单。这个颇有些 Android、iOS 味道的家伙，自然不能得到全体桌面用户的认可。事实上对于 Windows 而言，最好的菜单其实已经由 Win10 给出，这就是 传统菜单 + 全屏菜单 的组合。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/07ab2ee2-6543-40b7-88c3-60b817bb90b3.jpg" w="373" h="471" title="下代 Windows 将至？有关微软“Win11”的畅想" width="373" height="471" referrerpolicy="no-referrer"></p><p>▲下版 Windows 开始菜单概念图</p><p>除了与系统 UI 相匹配的圆角和毛玻璃外，开始菜单最有可能发生的改变，就是 Win10X 里的图标布局了。这一点从现版 Win10 的变化中也能看出，虽然冠以磁贴的大名，但绝大多数磁贴其实早已失去原本的意义，而仅仅只是一个图标。那磁贴呢？看看现版“资讯和兴趣”吧，后者除了在信息展示上更直观以外，还很好地借助网络实现了数据的及时更新，而现版磁贴其实也基本上只服务于以上两组应用了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/e96221a9-7c1a-4f5a-b6f4-f4978c30d694.jpg" w="700" h="697" title="下代 Windows 将至？有关微软“Win11”的畅想" width="700" height="697" referrerpolicy="no-referrer"></p><p>▲Win10“资讯和兴趣”或许将成为磁贴的继任者</p><p>除了图标与磁贴外，可定制性或将再次成为开始菜单的主流。一边是常用程序，一边是常用功能，新版开始菜单将带来我们更多的灵活性。而新增加的“快速访问“文件夹，也将成为一个不错的点子。配合悬浮在菜单顶端的搜索栏（几乎可以照搬 Win10X 的设计），新版开始菜单或许才真正成为 Windows 的中心！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/13af1e8a-7fb4-49ae-9108-db65dc9ff2f7.jpg" w="700" h="507" title="下代 Windows 将至？有关微软“Win11”的畅想" width="700" height="507" referrerpolicy="no-referrer"></p><p>▲早期 Win10X 开始菜单</p><h2>3. Microsoft Store</h2><p>未来 Microsoft Store（应用商店）将同时提供 Modern 应用和传统桌面应用，这或许是新版操作系统带给我们的最大期待。众所周知，现行 Modern 应用已经苟延残喘，而作为这一组件的唯一“载体”，应用商店自然也就乏人问津。而 Microsoft Store 同时提供 Modern 应用和传统桌面应用这一变化，除了可以有效提高商店的访问率以外，另一方面同时也是微软最期望的，则是将所有软件入口全部统一到 Microsoft Store 中来。彻底解决现版中软件搜索、下载、安装、升级、安全，和规范化等一系列问题。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/55deea4d-5577-44cd-bc7b-6b08417aebc5.jpg" w="700" h="457" title="下代 Windows 将至？有关微软“Win11”的畅想" width="700" height="457" referrerpolicy="no-referrer"></p><p>▲现版 Win10 应用商店</p><p>如果这个想法得以实现，那么目前困扰我们（当然还有微软）的有关软件方面的所有问题，几乎都有了解决途径。而这恰恰也是建立 Windows 生态中最重要的一环！</p><h2>4. 全新设置面板</h2><p>微软拖拖拉拉的习惯，从设置面板设计就可见一斑。和图标相比，控制面板对于我们的影响甚至还更甚一些。现版设置面板在 Win10 发布之初就已确定，目前已经沿用五年之久，之前的“太阳谷”泄露图也仅仅是在现有面板的顶端增加一个小标题，并没有什么实质性改变。但从微软关于设置面板的最新表述看，新面板将基于新的流畅设计语言，布局和现版也将出现一个很大的不同。除了引入全新的彩色图标外，还会在左侧增加一个导航栏，以方便用户使用。无论身处哪个界面，使用者都可以轻松跳转到自己所需的功能。而这对于触屏用户，也有很强的吸引力。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/314f46e7-152e-4bf8-8c3a-e62a705a5d71.jpg" w="700" h="462" title="下代 Windows 将至？有关微软“Win11”的畅想" width="700" height="462" referrerpolicy="no-referrer"></p><p>▲下版 Windows 设置面板概念图</p><h2>5. 虚拟桌面</h2><p>Win10 很早便引入了虚拟桌面功能，但使用率嘛…… 多少和最开始的呼声成反比。当然这或许与微软的一贯思路有关，别人都是先设计框架，再向里面添功能。而微软呢…… 恰恰是先有了功能，再想办法塞到系统里。尽管我们不知道下一版 Windows 是否会对这一问题做一点点修改，但可以确认的是，微软将会在虚拟桌面与多显示器间做一个均衡。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/31a4cd35-5ef9-410e-a732-6de64118424d.jpg@s_2,w_820,h_353" w="1344" h="579" title="下代 Windows 将至？有关微软“Win11”的畅想" srcset="https://img.ithome.com/newsuploadfiles/2021/6/31a4cd35-5ef9-410e-a732-6de64118424d.jpg 2x" width="1344" height="353" referrerpolicy="no-referrer"></p><p>▲现版 Win10 的虚拟桌面似乎和多显示器有点“冲突”</p><p>目前有关这方面的内容，微软方面还透露得较少。但一个显而易见的问题是，现版“虚拟桌面”已经和“实体桌面”（多显示器）存在了一定冲突，由于 Win10 虚拟桌面先天设计的不足（比如缺乏类似 MacOS 的鼠标拖拽管理），这种与实体桌面分离的设计显然没有达到 1+1>2 的效果。之前有消息称，微软正计划为 Windows 引入基于 Azure 的云端操作系统，这或许就是未来 Windows“虚拟桌面”的最好出路。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/da146ad8-aca7-459b-bf4a-59b0abed8468.jpg@s_2,w_820,h_516" w="995" h="626" title="下代 Windows 将至？有关微软“Win11”的畅想" srcset="https://img.ithome.com/newsuploadfiles/2021/6/da146ad8-aca7-459b-bf4a-59b0abed8468.jpg 2x" width="995" height="516" referrerpolicy="no-referrer"></p><p>▲微软或将引入基于 Azure 的云端操作系统替代现有的虚拟桌面</p><h2>6. 通知面板</h2><p>早在 Win10X 发布之初，观众便被全新的通知面板所吸引。的确，现版通知面板不仅臃肿且效率很低。在一张据称是新版 Windows 概念图中，我们看到了一个与 Win10X 极为接近的全新通知面板。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/b7b6bb60-5945-4a12-aa40-8d5f28594dc3.gif" w="700" h="394" title="下代 Windows 将至？有关微软“Win11”的畅想" width="700" height="394" referrerpolicy="no-referrer"></p><p>▲下版 Windows 通知面板概念图</p><p>新通知面板采用的是通知与快速设置分离的设计，这也正是 Win10X 一直所倡导的。这样做的好处有两点，一是功能与功能间的界限更加明晰，二是操作一个模块时也不会影响另一个模块。从细节上说，概念图在易用性方面也比现版表现得更好，一个最明显变化就是直接显示出来的亮度调节杆（现版默认是隐藏状态）。而右下角的快速设置区，显然也能跟随用户喜好自行定义，这一点和现版 Win10 无异。</p><h2>7. 动画质感</h2><p>如果说 UI 决定了一款操作系统的“本色”，那么动画则是整个 UI 体系中不可或缺的“润滑剂”。MacOS 的动画一直是业界比较公认的“优等生”，无论效果还是流畅度方面，都要比现版 Win10 强很多。而随着新一代 Fluent Design 的铺开，动画或许会迎来更多崭新变化。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/111be01c-bfb1-43ae-b6df-4b8377911dc3.gif" w="700" h="430" title="下代 Windows 将至？有关微软“Win11”的畅想" width="700" height="430" referrerpolicy="no-referrer"></p><p>▲基于 Fluent Design 的动画演示</p><p>从官方反馈来看，新版 UI 动画将涉及到窗口、开始菜单、最大 / 最小化、上下文菜单，后期还会出现在一些细节层面（比如鼠标悬停在某个元素上）。目前我们已经可以通过最新版 Win10 Dev 观察到一些改变，但个人认为，这仅仅是 Windows UI 漫长改进的一部分。换言之，剑已经有了，但如何使用，还得看微软自己的想法！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/31cc5a5f-4f75-4eb8-8f75-a42b9779e78d.gif" w="700" h="525" title="下代 Windows 将至？有关微软“Win11”的畅想" width="700" height="525" referrerpolicy="no-referrer"></p><p>▲目前存在于 Win10 Dev 版本中的新过渡动画</p><h2>8. 小组件崛起</h2><p>无论是 14 年前的 Vista 小部件首秀，还是后来的 Win10 磁贴，微软似乎都在思考这样一个问题，如何让用户在低负载情况下，尽可能快地完成某些简单操作（如天气查询）。尽管小部件、磁贴后来都被证实不适合现在的操作系统，但并不意味着它已经过时。事实上无论是新版 Win10 中刚刚加入的“资讯和兴趣”，还是近期热度飞涨的 PowerToys（微软开发的一款系统增强工具），所有一切都在暗示这样一个道理 —— 微软仍在这条路上不断探索。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/b409f92b-8190-42be-96c1-8f8977c938f3.jpg" w="246" h="800" title="下代 Windows 将至？有关微软“Win11”的畅想" width="246" height="800" referrerpolicy="no-referrer"></p><p>▲14 年前的 Vista 小部件</p><p>据外媒报道，微软正计划为 Windows 开发一套基于 Edge 的小部件。与上一代小部件不同，新组件将完全依赖于网络（也可以认为是 PWA 的一种），后者无论从升级还是资源占用上，都要比前者更具优势（“资讯和兴趣”就是一个典型例子）。最关键的是，它拥有着和传统 App 一样的特点，却多出了很多对方没有的优势。目前有关这项功能的介绍还不多，但现在看来，微软正打算将小部件更深入地集成到 Windows 系统中。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/ad33526e-2a6e-4b95-b560-aeb3e16aadbb.jpg" w="538" h="329" title="下代 Windows 将至？有关微软“Win11”的畅想" width="538" height="329" referrerpolicy="no-referrer"></p><p>▲下版 Windows Edge 小组件概念图</p><h2>9. 统一大生态</h2><p>尽管从诞生伊始就被赋予了通用系统的光环，但从 Win10X 的诞生来看，微软似乎放弃过 Win10 的这一身份。然而近期的一些变化，似乎让软件巨人又一次回归了这个想法。从公开的消息看，微软正打算对 Win10 的平板模式给予更多关注，比方说通过屏幕边缘扫出小部件，就是近期可能出现的一个变化。而这，似乎又让我们回到了之前那个 Win8 时代（类似于 Win8 的 Charm 栏）。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/4bef7fc9-e654-49be-8994-74ab23b891cc.jpg" w="264" h="348" title="下代 Windows 将至？有关微软“Win11”的畅想" width="264" height="348" referrerpolicy="no-referrer"></p><p>▲Win8 中曾经出现过的 Charm 栏</p><p>其实对于任何一家操作系统厂商来说，单纯注重某一平台都是不正确的。苹果不仅让 MacOS 顺利过渡到 ARM，同时也和 iOS、iPAD 等形成了一个巨大的生态系统。此外华为的鸿蒙也是同样的套路，尽管暂时还没有前面两者强大，但也开始整合手机、电视、平板、汽车、桌面等诸多场景。从这个角度上看，微软重视平板模式，或许正是它庞大计划的开始而已，毕竟“统一大生态”才是未来能和其他对手抗衡的最大胜算。</p><h2>写在最后</h2><p>说了这么多，最后再说一下我的观点吧。其实对于微软而言，目前所面临的问题远不止更新界面，或者重构用户体验这样简单。在苹果、Google、安卓、鸿蒙…… 等一众新生力量的狂追不舍下，微软几乎已经到了腹背受敌的艰苦边缘。尽管有关 6 月 24 日还有众多猜测，但我想这可能仅仅是微软开创新纪元的一个开始，而“太阳谷”—— 不过是这千里之行的第一步，两者间并无冲突！</p>
