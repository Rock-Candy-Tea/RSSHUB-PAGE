<p>太难了！太难了！</p><p>清华电子系，只学 3 节 Python 课，然后…… 然后……</p><p>就直接要求“手撸”一个 AI 算法！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/06b6d26d-468d-4577-91ad-b43bfaf9f074.gif" w="308" h="225" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="308" height="225" referrerpolicy="no-referrer"></p><p>就这样，清华再一次因为难，冲上了知乎热榜。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/6db1e62b-ba66-427d-9073-1a262790758a.png" w="1080" h="217" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="1080" height="165" referrerpolicy="no-referrer"></p><h2>这作业有多难？</h2><p>先来聊聊这个作业的背景。</p><p>根据清华计算机系 00 后本科生、知乎用户“孙恒”的介绍：</p><p>这门 Python 课，是在清华电子系大一《计算机程序设计基础 (2)》附带的小学期中进行。</p><p>课程的时间一共是 2 周。</p><p>其中，会抽出一周的时间来上 3 节 Python 课（共 9 学时，1 学时是 45 分钟），</p><p>除此之外，还设有 Python 三次小作业和一次大作业。</p><p>但值得一提的是，这个小学期的内容是今年课改后的结果。</p><p>往年其实是大一下 C++ 课的附带小学期，往往用来完成一个 C++ 大作业。</p><p>但今年却直接删掉了 C++ 的自主选题。</p><p>这不禁让网友感慨道：</p><p>和“计算机程序设计基础”几个字渐行渐远了。</p><p>接下来，再来看下作业本身内容。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/1905c3e0-a6f6-47ee-b8b8-6f05f3160d9d.png" w="1080" h="833" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="1080" height="632" referrerpolicy="no-referrer"></p><p>△ 图源：知乎</p><p>从布置作业的内容来看，大致就是一个人脸性别识别任务。</p><p>好了，那要完成这么一个大作业，你都得会些啥？</p><p>起码以下这些问题得 hold 住喽：</p><ul class=" list-paddingleft-2"><li><p>什么是 Python？</p></li><li><p>什么是训练集？</p></li><li><p>什么是分类问题？</p></li><li><p>什么是计算机视觉（CV）？</p></li><li><p>……</p></li></ul><p>上面的这些问题，<span class="accentTextColor">可能只是完成这个大作业，必须要掌握技能的一隅</span>。</p><p>但让网友们觉得“不可思议”的是，<span class="accentTextColor">你可能必须在 1 周时间内去了解它们</span>：</p><p>“太难了吧！”</p><p>“太离谱！”</p><p>而更“离谱”的还在后边，请看“作业要求”第三条：</p><p><span class="accentTextColor">不允许直接使用开源项目提供的已训练好的模型或已写好的现有代码。</span></p><p>什么概念？</p><p>可不仅是不能用 torch 是问题啊，这是连 numpy、pandas 都不让用啊！</p><p>纯手撸啊！</p><p>这个时间节奏、这个“细节要求”，可算是把网友惊到了：</p><p>可真是地狱难度！</p><p>更有网友爆料道 ——“大一学生没接触过机器学习……”</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/6b91dec0-dc16-4966-ab0d-a8f1cc7a6320.gif" w="491" h="244" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="491" height="244" referrerpolicy="no-referrer"></p><h2>网友：离本科毕设只差一篇万字论文</h2><p>“难”上热榜的清华电子系大作业，自然引来了众多网友的热议。</p><p>“不合理”、“离谱”，成了评论区的热词。</p><p>知乎网友“婷小婷”见此话题，特意跑去问了国防七子某航电子信息工程的同学，得到了这样的评价：</p><p>这个题目的难度基本上等于他本科划水毕设的难度水平。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/711913ec-29d7-48a5-9155-09242276fd47.png" w="1080" h="250" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="1080" height="190" referrerpolicy="no-referrer"></p><p>而对于这样的课程时间和作业布置，她直言道：</p><p>对于原本有基础的学生，这 9 课时的课程纯粹属于“拿个必修学分”，课后作业也能正常完成。</p><p>对于原本零基础的学生，这 9 课时的课程同样没有任何营养，课后作业也只能跟着“大佬”凑个数。</p><p>9 个学时，从 python 开始讲到深度学习，对于新生来说正常也就能讲一遍综述，这能写出个锤子！自学也不是这样学的！</p><p>同为清华学生，计算机系本科生“孙恒”也在这个问题下给出了他的看法。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/a409534f-9cc9-4dc3-85f2-7f3aa56ee118.png" w="1080" h="524" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="1080" height="398" referrerpolicy="no-referrer"></p><p>他认为，这个大作业之所以让很多同学无法接受，总共有五点。</p><p>首先，是对同学们基础能力预估过高。</p><p>电子系本非以对软件编程高度依赖的院系，同学们大一时大多只接触过 C/C++ 的基础版，没有培养大家自学、自主找资料的能力，现在突然速成 Python 并完成大作业，对很多同学来说是非常困难的。</p><p>第二，是忽略了 AI 领域前置知识确缺失问题。</p><p>虽然“调 pytorch 随便写几层 CNN，不追求准确率”这件事对 AI 圈的人来说好像确实只是个简单的 demo 水平，可是别忘了，大一小朋友们并没有接触过 AI 这个领域，他们不知道什么是 pytorch 什么是 CNN。</p><p>倘若一门课在毫无理论的情况下直接让人直接实践，这已经不是 fly bitch 了，这听上去更像是教学事故。</p><p>第三，是忽略了算法知识缺失问题。</p><p>有同学指出题面上也没有非要炼丹，用 SVM、最近邻之类的也可以，这一点我认可，但这并不代表这个作业就合理了，因为无论是 SVM 还是最近邻这些都属于需要了解过才能知道的算法，我们不能指望多数人具备“自己拍拍脑袋就创造出了简单算法”的能力。</p><p>第四，是需求理解难度高。</p><p>老师选择了多数同学毫无经验的人工智能领域的问题，同学们既不熟悉 Python，也不熟悉任务，更不熟悉任务所需要的其他工具，很有可能即便最后磕磕绊绊写了份代码，也不知道这份代码到底在干啥。</p><p>最后，孙恒还认为，“老师对大作业的布置没有足够重视，缺少调研和实践”。</p><p>那毕业于清华的“老哥”们，怎么看？</p><p>清华硕士毕业、前旷视手机端识别算法老哥 ——“水哥”回忆道：</p><p>不得不表示，黑人，我们当时也没做过。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/cce1fad2-2cbd-4857-9afc-ba6cee817c75.png" w="1080" h="249" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="1080" height="189" referrerpolicy="no-referrer"></p><p>当然，评论不近乎全是反对的声音。</p><p>也有网友认为：</p><p>学好了能受用一辈子，这老师直接授人以渔，值得嘉奖。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/9c8e7553-ec09-48d1-ae48-52f97e10c8be.png" w="920" h="172" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="920" height="153" referrerpolicy="no-referrer"></p><h2>离谱作业，不止这一个</h2><p>关注这个话题的朋友都会知道，清华大学因“作业/考试难”上热榜，已经不是第一次了。</p><p>除了这次电子系的作业外，此前还有自动化系等。</p><p>例如“自动化系大一 C++ 大作业写一个功能更强大的雨课堂”：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/7b045731-aba3-46a4-aea5-ddee021ac6c8.png" w="1080" h="208" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="1080" height="158" referrerpolicy="no-referrer"></p><p>还有“计算机系大一下学期考试要求学生在 3 小时内完成三道码量较大的工程题”：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/486547f5-077c-4ad5-8ae7-13604587b1a5.png" w="1080" h="188" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="1080" height="143" referrerpolicy="no-referrer"></p><p>如此现象，同样也引发了网友对于课程设置合理性、教育等方面的讨论。</p><p>例如知乎网友“小朱”，便用自身的经历做了比较：</p><p>感觉现在国内高校尤其是头部学校，普遍有个错觉，认为现在学生很强，就可以提高难度，拔苗助长。殊不知这样不仅没有育人，反而还成了变相劝退。</p><p>我在国外上课的体验是，虽然课程也很虐，但他们的作业和考试都是比较循序渐进的。</p><p>比如一道手写机器学习的题，要用 numpy 写 MLP 的反向传播，那一定会先铺垫一道 MLP 反传的推导，同时提示可以用的公式结论。</p><p>后来自己做了助教才知道，出这样结合课程难度，又有区分度的题是非常费时费力的。归根结底，还是我们在大学教学上花的时间太少了，苦了学生。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/dcebf527-22f0-4590-a1ff-385c64517131.png" w="1080" h="168" title="清华大一 Python 作业太难上热榜：只上 3 节课，手撸 AI 算法" width="1080" height="128" referrerpolicy="no-referrer"></p><p>这是出现在头部高校的问题。</p><p>而也有网友认为，与之形成鲜明对比的其它高校，也存在着教学等方面的问题。</p><p>最为明显的一点，便是“很不注重实践”：</p><p>实验、作业布置得很多；大部分时间都花在了写 word 报告上。</p><p>考试内容都在 PPT 上，背书不如背 PPT。</p><p>……</p><p>最后，如果你也是正在经历或经历过“恐怖大作业”，也欢迎留下你的回忆、故事。</p><p>评论区见～</p>
