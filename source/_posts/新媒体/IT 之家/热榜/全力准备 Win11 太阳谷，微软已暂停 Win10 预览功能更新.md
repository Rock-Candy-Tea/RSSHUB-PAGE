<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 8 日消息 微软此前宣布，将于北京时间 6 月 24 日晚 23 点发布“下一代 Windows”，外界普遍预测为 Win11。</p><p>今日据外媒 Windows Latest，<span class="accentTextColor">微软已经暂停了 Win10 DEV 预览的功能更新，新发布的 Build 21390.1000 没有任何新功能</span>，只是为任务管理器、MSI 安装包启用了新的 Fluent 图标，允许 Windows Terminal Preview 作为默认终端模拟器，以及进行修复 Bug 的累积更新。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/b08cec6a-f3e6-4d2b-86f8-18dca527a98c.jpg" w="696" h="392" title="全力准备 Win11 太阳谷，微软已暂停 Win10 预览功能更新" width="696" height="392" referrerpolicy="no-referrer"></p><p>微软官方表示，<span class="accentTextColor">正在测试累积更新，以解决准正式版的一些问题</span>。</p><p>IT之家了解到，Win11 或 Win10 太阳谷更新即将到来，所有的证据都指向了重大的变化，“开始”菜单的 UI 改造，新的 Windows 商店，以及更多改进。</p><p>此外，最新的微软文档显示，Win11 太阳谷 / Win10 有两个不同版本，一个针对消费者，一个针对商业和企业客户。</p>
