<p data-vmark="21a3"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 19 日消息，微软已经开始着手开发下一个“<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 22H2”大版本，预计将于 2022 年秋季发布。</p><p data-vmark="b60d">作为 Windows 11 22H2 版本的一部分，微软计划通过将 Mica+Fluent 引入更多使用不同框架的应用程序来改进现有操作系统的界面。</p><p data-vmark="7fc1">Fluent Design 是 Windows 11 和 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 的设计语言的一部分，它采用了光和景深等元素来取代 Windows 8 单调的外观。它拥有类似 Acrylic“亚克力”这般的设计，这是一种用于菜单、对话框背景或整个应用程序窗口的半透明材质。</p><p data-vmark="cee5">在 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 中，微软为其 Fluent 设计语言添加了一种新的材质，即 Mica“云母”。这也是一种半透明的模糊设计，可以让你的桌面壁纸在应用程序窗口中隐隐发光。在某种程度上，它也可以实现类似于“亚克力”或 <a class="s_tag" href="https://www.win7china.com/" target="_blank">Win7</a> 的 Aero“毛玻璃”效果。</p><p data-vmark="7527">云母的透明度效果可以在省电模式或者当设备速度过慢（硬件过老）时自动调整，所以它在保持亚克力材质的优势的同时实现了更快更流畅的效果。</p><p data-vmark="695b">IT之家了解到，Win11 的 Mica 材质目前还无法与所有现代化应用程序兼容，但这种情况可能很快就会改变。</p><p data-vmark="43f3">根据 WindowsLatest 发现的新参考内容，微软正计划将 Mica 扩展到现代应用程序甚至传统应用程序的“FrameHostTitlebar”标签。</p><p data-vmark="1f7a"><img src="https://img.ithome.com/newsuploadfiles/2021/12/4fa3ce9b-4191-4cf0-a220-5ba603f4522f.png" w="940" h="640" alt="云母暗模式" title="Win11 22H2 正在路上，微软将会在不影响性能的情况下进行更多 UI 设计优化" width="940" height="558" referrerpolicy="no-referrer"></p><p data-vmark="ff7c">▲ 云母黑暗模式</p><p data-vmark="fdc2">据悉，Mica 已经出现在 Build 22509 或更高版本的 Win11 预览版核心应用中，但部分现代程序无法使用 Mica 构建。不过微软似乎正在测试“MicaBackdropInApplicationFrameHostTitlebar”标签（这在更新的版本中已经被证实）。</p><p data-vmark="351c">顾名思义，这个标签将 Win11 的 Mica 效果覆盖到了反馈中心 (Feedback Hub) 等应用中，例如标题栏。得益于此，用户无需从微软商店下载更新包就可以获得一些额外的 UI 优化。</p><h2 data-vmark="a965">新材质</h2><p data-vmark="0f06">有趣的是，微软似乎正在试验另一种名为“Tabbed”的设计材料，这种材料似乎是以云母为基础进行优化的产物，也可能是云母的一种版本。</p><p data-vmark="278d">在 Windows 11 Build 22523 SDK 的 Win32 API 中，用户发现了新的和奇怪的“标签”设计材料。</p><p data-vmark="5286"><img src="https://img.ithome.com/newsuploadfiles/2021/12/f0eda22f-1175-440c-a160-9d79b41d7435.png" w="745" h="420" alt="丙烯酸纤维" title="Win11 22H2 正在路上，微软将会在不影响性能的情况下进行更多 UI 设计优化" width="745" height="420" referrerpolicy="no-referrer"><img src="https://img.ithome.com/newsuploadfiles/2021/12/2e6f9890-1305-49a5-aff6-aa67be681247.png" w="745" h="420" alt="云母" title="Win11 22H2 正在路上，微软将会在不影响性能的情况下进行更多 UI 设计优化" width="745" height="420" referrerpolicy="no-referrer"><img src="https://img.ithome.com/newsuploadfiles/2021/12/b964637c-23bd-455b-b871-6b9a09a0abfe.png" w="745" h="420" alt="标签式" title="Win11 22H2 正在路上，微软将会在不影响性能的情况下进行更多 UI 设计优化" width="745" height="420" referrerpolicy="no-referrer"></p><p data-vmark="0a8f">如上图所示，“系统背景类型”中的“Tabbed”设计材质似乎是现有“云母”材质的暗色优化版本，但我们目前还缺乏更进一步的信息。</p>
