<p data-vmark="ab72"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 28 日消息，据 Windows Latest 报道，微软最初在今年 6 月宣布了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的全新应用商店，从那时起，该公司一直在努力提高其应用商店的质量。作为 Windows Insider Program 的一部分，微软已经发布了一些服务更新，用户已经在他们的设备上测试了三个多月的全新应用商店。</p><p data-vmark="96c8"><img src="https://img.ithome.com/newsuploadfiles/2021/10/fcb7e059-edc1-4b8e-87aa-4d7b1a13e568.jpg" w="800" h="420" title="Win11 全新微软应用商店 App 登陆 Win10，快速上手体验：性能更高，下载更快" width="800" height="420" referrerpolicy="no-referrer"></p><p data-vmark="787d">IT之家获悉，在“What's new for Windows”活动中，微软解释说，它重建了微软商店，采用了新的设计，提高了性能，并为开发者提供了更好的支持。正如预期的那样，微软终于将其 Windows 11 的全新应用商店带到了 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> PC 上，而且似乎正在分阶段推出。</p><p data-vmark="49c8">如果你在 Windows Insiders 计划的 Release 预览通道中，并且运行 Windows 10 版本 21H2，你现在可以通过检查微软商店内的更新来获得全新的应用商店。对于那些运行非 Insider 版本的用户，可以通过以下步骤手动安装新的微软商店。</p><ul class=" list-paddingleft-2"><li><p data-vmark="5b88">从<a href="https://wwa.lanzoui.com/iJVTtvuyuuf" target="_blank">这里</a>下载微软商店的更新版本。（获取解压后的文件）</p></li><li><p data-vmark="1f58">部分用户需先下载 Microsoft.UI.Xaml.2.7，<a href="https://wwa.lanzoui.com/iXYVavv6owj" target="_blank">点此链接</a>。（获取解压后的文件）</p></li><li><p data-vmark="97c8">打开 PowerShell 管理员权限。</p></li><li><p data-vmark="3474">使用 cd 命令，导航到 Microsoft.UI.Xaml.2.7 文件位置，例如，如果下载的文件移动后的位置是 C 盘，输入 cd C:\ 。</p></li></ul><p data-vmark="fdb8">安装命令为</p><pre class="brush:javascript;toolbar:false ai-word-checked">Add-AppxPackage Microsoft.UI.Xaml.2.7_7.2109.13004.0_x64__8wekyb3d8bbwe.appx</pre><ul class=" list-paddingleft-2"><li><p data-vmark="57d8">使用 cd 命令，导航到你保存微软商店 App 更新的位置。例如，如果下载的文件移动后的位置是 C 盘，输入 cd C:\ 。</p></li><li><p data-vmark="6eff">输入 Add-AppxPackage <package name.msixbundle>。</p></li></ul><p data-vmark="7ea9">在这种情况下，最后的命令将是，</p><pre class="brush:javascript;toolbar:false ai-word-checked">Add-AppxPackage Microsoft.WindowsStore_22110.1401.10.0_neutral___8wekyb3d8bbwe.Msixbundle</pre><h2 data-vmark="1f75">近距离观察 Windows 10 的全新微软商店</h2><p data-vmark="48bd">新的微软商店在 Windows 10 上采用了重新设计的界面。</p><p data-vmark="7e6e">与目前的应用商店相比，它看起来不那么杂乱，而且新的商店是基于 XAML（UWP）代码，而不是 WebView，所以你可以期待更好的性能，特别是在网络连接缓慢的时候。</p><p data-vmark="5746"><img referrerpolicy="no-referrer"></p><p data-vmark="e95e"><img src="https://img.ithome.com/newsuploadfiles/2021/10/dd6c7674-61b1-4bf6-aef3-f501bea3e5af.jpg" w="768" h="512" title="Win11 全新微软应用商店 App 登陆 Win10，快速上手体验：性能更高，下载更快" width="768" height="512" referrerpolicy="no-referrer"></p><p data-vmark="44bd">我们不会剖析微软商店的每一个小变化，但值得指出的是，Windows 10 的新应用商店比旧应用商店更快，用户现在可以在不同页面之间快速导航。</p><p data-vmark="e9ed">微软还提高了下载速度，你将能够从新的库中轻松管理你已安装/拥有的应用程序。</p><p data-vmark="b075">此外，微软还在为应用商店开发一项名为“故事”的新功能。这将允许开发者创建编辑内容来推广他们的应用和游戏，并帮助用户发现那些能激发他们使用设备实现更多目标的应用。</p><p data-vmark="db77">这个新的微软商店支持更多类型的应用程序，它对所有 Windows 开发者开放。</p><p data-vmark="df07">遗憾的是，微软不打算将 Windows <a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>子系统带到 Windows 10 应用商店。目前，对安卓子系统的支持仍将是 Windows 11 系统的专属，因为它是新操作系统的旗舰级功能之一。</p>
