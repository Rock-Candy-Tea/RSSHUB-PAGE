<p data-vmark="c87f"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 27 日消息，据 Neowin 报道，今年 10 月早些时候，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 正式向消费者发布，但推送是分阶段的。新的功能，重新设计的图形 UI，以及对升级的强制性要求（如 TPM 2.0）的挫败感，让人们对 Windows 操作系统的一些“美好的过去”更加欣赏。</p><p data-vmark="0ccb"><img src="https://img.ithome.com/newsuploadfiles/2021/10/566eaba8-f279-4236-a59e-d5d71069a55e.jpg" w="760" h="405" title="微软 Windows XP 诞生 20 周年，Win11 继续前进" width="760" height="405" referrerpolicy="no-referrer"></p><p data-vmark="708a">2021 年 10 月 25 日是 Windows XP 诞生 20 周年纪念日，延续到未来的 Windows 也正在迭代。Windows XP 建立在 Windows NT 内核而不是 Windows 95 内核上，是 Windows Me 和 Windows 2000 的进化版本。</p><p data-vmark="e907">随着人们对 Windows Me 的广泛不满，Windows XP 在可靠性和性能方面有了明显的改进。虽然“开始”菜单界面早在几年前就出现在 Windows 95 上了，但它在定制和功能方面有了明显的改进。“开始”菜单被改为包括两栏；左边侧重于已安装的软件，右边则提供对文档、最近文件/文档、图片、音乐和“我的电脑”等文件夹的便捷访问，以及其他重要区域，如控制面板、搜索和运行。</p><p data-vmark="0a61"><img src="https://img.ithome.com/newsuploadfiles/2021/10/1a909091-6cd1-4b94-add7-e957b93ce12b.gif" w="376" h="516" title="微软 Windows XP 诞生 20 周年，Win11 继续前进" width="376" height="516" referrerpolicy="no-referrer"></p><p data-vmark="a80d">尽管 Windows XP“开始”菜单和任务栏的默认颜色略显两极分化，但可以选择将“开始”菜单恢复到更经典的外观。当然，“开始”菜单将在 Windows XP 的未来迭代和版本中继续存在，采取不同的形式因素，如在 Windows 8 中非常大胆，在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 中又恢复到更传统的外观。除了“开始”菜单，任务栏还能首次将从同一应用程序打开的窗口分组，并提供访问上下文菜单的机会，以显示被分组的每个项目。</p><p data-vmark="cd4d">IT之家获悉，在 Windows XP 中，首次实现了快速用户切换。这允许多个用户切换活动账户，而不必关闭所有打开的应用程序或注销，在返回会话时创造更好的工作流程。</p><p data-vmark="fa65"><img src="https://img.ithome.com/newsuploadfiles/2021/10/cbd06e16-8357-4012-bc6b-8c311e28f90b.jpg" w="760" h="428" title="微软 Windows XP 诞生 20 周年，Win11 继续前进" width="760" height="428" referrerpolicy="no-referrer"></p><p data-vmark="6a56">在性能改进方面，Windows XP 允许通过使用预取数据来减少启动和软件启动时间。其他值得注意的改进包括 DirectX 8.1，在 LCD 显示器上实施 ClearType 字体渲染，以及通过系统还原等功能提高系统可靠性。</p><p data-vmark="a15c">微软多年来以各种版本销售 Windows XP。从家庭版、专业版和 Windows XP Media Center 版，微软试图根据预期的消费者或企业来满足更多特定的使用情况。例如，一个名为 Windows XP Tablet PC 版本包含了针对手写笔/笔输入的功能。此外，为了迎合 64 位架构，微软发布了 Windows XP Professional x64 版本。</p><p data-vmark="b9ed"><img src="https://img.ithome.com/newsuploadfiles/2021/10/22f2b70b-e205-485e-96b4-a6a810742cfb.jpg" w="760" h="428" title="微软 Windows XP 诞生 20 周年，Win11 继续前进" width="760" height="428" referrerpolicy="no-referrer"></p><p data-vmark="d3bb">更好的 Windows 版本的蓝图可以从 Windows XP 的产品架构中体现。虽然它现在已经有 20 年的历史了，但它仍然为微软提供了很好的参考点来建立和改进 Windows 。</p>
