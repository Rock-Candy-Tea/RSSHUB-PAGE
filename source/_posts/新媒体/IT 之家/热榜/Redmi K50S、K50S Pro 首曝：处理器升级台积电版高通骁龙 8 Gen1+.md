            <p data-vmark="5213"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 25 日消息，xiaomiui 发现，Redmi K50S / Pro 和小米 12T / Pro 现已连同他们的 SoC 一起出现在 Mi Code 和 IMEI 数据库中，外媒预计它们会在 8 月和 10 月推出。</p><p data-vmark="9ff0">从代码来看，Redmi K50S 系列将分为 3 款设备，Redmi K50S / Pro 与 K50 系列基本相同，将采用最新的高通骁龙和联发科天玑芯片，不过这款高通芯片组将由台积电而非三星生产，也就是说 Redmi K50S 系列或将成为 Redmi 冲击高端的机型。</p><p data-vmark="70aa">IT之家了解到，Redmi K50S 系列之前已经出现在 IMEI 数据库中。从型号来看，Redmi K50S 系列似乎是全球市场上推出的的小米 12T。</p><h2 data-vmark="3396">Redmi K50S / Pro 出现在小米 Mi Code 上</h2><p data-vmark="09ec">借助 Mi Code 上发现的信息，我们基本可以确定 Redmi K50S Pro 和小米 12T Pro / 12T Pro HyperCharge 将搭载台积电生产的 sm8475（或命名为高通骁龙 8 Gen 1+），似乎是小米在全球市场上唯一的骁龙 8 Gen 1+ 机型。</p><p data-vmark="9c4e" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/914eb577-51f2-471d-b5c3-059d3f76dc60.png" w="403" h="80" alt title="Redmi K50S、K50S Pro 首曝：处理器升级台积电版高通骁龙 8 Gen1+" width="403" height="80" referrerpolicy="no-referrer"></p><p data-vmark="1ddf">在小米代码的这一行中，我们可以看到型号为 L12A（Redmi K50S，小米 12T）的设备的代号是“柏拉图”，而且“柏拉图”完全基于联发科移动平台。</p><p data-vmark="ea7b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/23147299-ea58-4c98-9b3e-62845afffc4b.jpg" w="558" h="181" alt title="Redmi K50S、K50S Pro 首曝：处理器升级台积电版高通骁龙 8 Gen1+" width="558" height="181" referrerpolicy="no-referrer"></p><p data-vmark="6e01">在下图中的代码行中，我们可以看到小米部分新机将搭载高通新一代 SoC，也就是是骁龙 8 Gen1+。</p><p data-vmark="7606">结合此前的爆料来看，小米 12S、小米 12S Pro、MIX FOLD 2 将会搭载台积电版本的骁龙 8 Gen1+，今年最后推出的旗舰手机将是 L12。</p><p data-vmark="a83f"><img src="https://img.ithome.com/newsuploadfiles/2022/4/5718e031-d6af-4816-85a2-d9083ca93869.jpg" w="721" h="320" alt title="Redmi K50S、K50S Pro 首曝：处理器升级台积电版高通骁龙 8 Gen1+" width="721" height="320" referrerpolicy="no-referrer"></p><p data-vmark="87cd">Mi Code 中的这些代码还证明小米新机将搭载骁龙 x475（可能是 8 Gen 1+ 的调制解调器）平台的代号为“蜉蝣”。这证实了蜉蝣就是 L12，即小米 12T / Redmi K50 Pro。</p><p data-vmark="d4be">按照常规的地区型号来区别：</p><ul class=" list-paddingleft-2"><li><p data-vmark="1035">22081212CL12“蜉蝣”Redmi K50S Pro（中国）</p></li><li><p data-vmark="44ec">22071212ACL12A “柏拉图” Redmi Pro（全球）</p></li><li><p data-vmark="5090">22081212RL12“蜉蝣”小米 12T Pro（日本）</p></li></ul><p data-vmark="cd91">也就是说，小米 12T / Pro 和 Redmi K50S / Pro 系列机型有可能暂时不会在印度发布（印度字母是 I）。</p><p data-vmark="7de4">外媒认为，小米 12T 系列将采用联发科天玑旗舰处理器，而小米 12T Pro 将搭载骁龙 8 Gen 1+ 旗舰平台，有望在今年 8/10 月发布。</p>
