<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 6 日消息 此前，微软面向 Dev 开发频道发布了第一个 Windows 11 Insider Preview 版本，即 Build 22000.51!  微软表示，随着我们在未来几个月内完成产品的定型，我们将与你一起验证这一体验。在这个早期预览中，<a href="https://www.ithome.com/0/559/232.htm" target="_blank">你将可以尝试此前展示的许多新功能</a>，但不是全部。</p><p>外媒 Windows Latest 报道，如果你足够幸运，符合 Windows 11 的要求，那么当你升级到新的操作系统时，你将失去一些重要的任务栏功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/485d2125-f5a6-450c-84ea-9b1ec84e6b26.jpg" w="768" h="432" title="Win11 任务栏基于 UWP（XAML）打造，但不少功能退化或缺失，微软回应" width="768" height="432" referrerpolicy="no-referrer"></p><p>微软 Windows 11 采用了基于现代 WinUI、XAML 和 UWP 的组件。它将采用新的文件资源管理器、“开始”菜单、操作中心、Windows 桌面应用程序界面的增强、圆角、新的上下文菜单、更快的性能、强化的安全以及居中的任务栏。</p><p>这当然是巨大的 Windows 更新，但它确实也有一些小的注意事项。首先，你将会失去一些重要的任务栏功能。在 Windows 11 中，任务栏已被现代化，基于 UWP（XAML）打造。因此，任务栏的默认功能目前是有限制的，其排列方式也不能改变。</p><p>Windows 11 的任务栏功能被限制位于显示器的底部，这在微软的文档中得到了正式确认。</p><p>同样，Windows 11 也不允许用户将应用程序/快捷方式拖放到任务栏上。在 Windows 10 和更早的版本中，你可以从桌面或资源管理器拖放应用程序到任务栏，Windows 会自动生成一个快捷方式。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/4412b9ff-f67e-43bb-8df8-5c36f057bd5c.jpg@s_2,w_820,h_309" w="1007" h="380" title="Win11 任务栏基于 UWP（XAML）打造，但不少功能退化或缺失，微软回应" srcset="https://img.ithome.com/newsuploadfiles/2021/7/4412b9ff-f67e-43bb-8df8-5c36f057bd5c.jpg 2x" width="1007" height="309" referrerpolicy="no-referrer"></p><p>遗憾的是，这项功能不再被支持，至少目前是这样。</p><p>Windows 任务栏现在包括对名为“快照组”的新功能支持，它可以让你轻松地切换回被快照的窗口。当你将两个或更多的应用程序列在一起，并将鼠标悬停在任务栏上打开的应用程序上时，这种新的多任务体验是可见的。</p><p>如果你经常把应用抓在一起，快照组可能很好的功能，但它也可能是令人讨厌的。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/2c9e9f0f-5c68-43c8-bd98-eaf733118681.jpg" w="638" h="238" title="Win11 任务栏基于 UWP（XAML）打造，但不少功能退化或缺失，微软回应" width="638" height="238" referrerpolicy="no-referrer"></p><p>目前，Windows 不允许取消对任务栏图标的分组。</p><p>同样，微软称，一些图标（取决于应用程序）在更新到 Windows 11 后可能不再出现在系统托盘中，而且某些应用程序不能再自定义任务栏的区域。</p><p><strong>任务栏上下文菜单</strong></p><p>在 Windows 11 预览版中，微软正在试验将所有任务栏上下文菜单（右键菜单）选项移到设置应用程序中。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/3a4f634e-fd69-452f-ae65-dc6086955fb8.jpg" w="729" h="273" title="Win11 任务栏基于 UWP（XAML）打造，但不少功能退化或缺失，微软回应" width="729" height="273" referrerpolicy="no-referrer"></p><p>此举将进一步简化体验，减少杂乱，但对于没有激活 Windows 的用户来说，这可能是个问题。在未激活的 Windows 11 副本中，用户不可能访问个性化选项。</p><p>在未激活的 PC 上，你仍然可以使用注册表编辑器定制 Windows，尽管不能保证这种情况会一直存在。</p><p><strong>微软正在听取反馈意见</strong></p><p>微软官方证实，他们正在听取 Windows Insiders 的反馈意见，因此，这些缺失的一些功能很可能在 Windows 11 操作系统的最终版本中得到恢复，但不能完全保证。</p><p>Windows 11 目前正处于开发阶段，它将在今年晚些时候开始向符合条件的消费者推出。对于那些不支持的设备，微软表示，将继续支持 Windows 10 21H2 版本和其他更新，但重大的变化仍将是 Windows 11 新操作系统的专属。</p>
