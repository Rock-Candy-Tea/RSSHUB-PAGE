<p data-vmark="64cd"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 24 日消息，早在 2021 年 10 月，Windows 11 上的 Android 应用程序终于进入了 Windows Insiders。经过数月的测试，它们也以操作系统的预览形式提供，但仅限于美国地区。今天，微软宣布将把该计划扩展到更多国家地区。</p><p data-vmark="1c18">现在，另外五个国家 / 地区的 Windows 11 用户将能够使用 Android 应用程序，其中包括法国、德国、意大利、日本和英国。尽管微软没有列出具体的时间表，但它指出这应该在今年年底之前登陆。微软还指出，亚马逊 Appstore 预览版将继续每月发展。但是，如果你正在等待 Google Play 的官方支持，你将不得不等待更长的时间 - 或者可能永远等待中。</p><p data-vmark="aa95">微软也强调了其 Microsoft Store 的早期成功。在 2022 年第一季度，<span class="accentTextColor">与去年同期相比，微软商店新桌面应用程序和游戏的数量增加了 50%。</span></p><p data-vmark="21e0">微软还宣布提高 Windows 11 中应用程序可发现性的新方法。第一种方法是 Windows Search 显示来自 Microsoft Store 的搜索结果，并直接从其中安装应用程序。由于超过 50% 的用户“定期”使用“开始”菜单，微软正依靠这一点成为应用程序的主要获取渠道。</p><p data-vmark="4400">微软正在测试一种跨 Windows 设备恢复应用程序的功能，以便在 PC 之间进行无缝转换。上述两个功能都将很快提供给 Windows 预览体验成员。</p><p data-vmark="cc00">同时，开发人员现在可以利用弹出式商店和应用程序徽章，使他们的应用程序在网站上更加突出。在接下来的几个月内，他们还将能够通过名为 Microsoft Store Ads 的新产品在 Microsoft Store 中投放广告活动。这些活动将被情境化，目前，感兴趣的开发者可以注册候补名单。</p><p data-vmark="af0d">此外，将 Win32 应用程序发布到商店的候补名单不再存在。任何开发人员现在都可以将他们的 Win32 包发布到 Microsoft Store 并利用新功能，例如通过 CI / CD GitHub 操作或 Rest API 管道自动提交、测试应用可见性、查看应用遥测以及直接响应用户评论。</p><p data-vmark="66e8">最后，渐进式网页应用程序 (PWA) 的开发人员可以利用以下功能：</p><ul class=" list-paddingleft-2"><li><p data-vmark="fe28">PWA Starter：此功能可帮助开发人员引导他们的 PWA 开发，提高性能、与主机操作系统的集成以及提供出色用户体验的指南。</p></li><li><p data-vmark="e75b">PWA Studio：这是一个新的 Visual Studio Code 扩展，可帮助构建、打包 PWA 并将其发布到 Microsoft Store，而无需离开 Visual Studio Code。</p></li><li><p data-vmark="0f1f">Meta Quest 支持：HoloLens 设备已经支持 Microsoft Store 中的 PWA，Meta 最近宣布在其 VR 设备上支持 PWA。Microsoft 和 Meta 合作使 PWABuilder 成为为虚拟和混合现实打包 PWA 的终极工具，使开发人员能够使用 Meta Quest 和 HoloLens 的旁加载包。</p></li></ul><p data-vmark="5c0b">微软强调，未来它将继续让其 Microsoft Store 商店成为消费者和开发者的更好场所。</p>
