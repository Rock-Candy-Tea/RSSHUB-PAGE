<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 10 日消息 微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 首个预览版发布后，微软于 7 月 9 日推送了 <span class="accentTextColor">KB5004745</span> 功能更新，将系统升级为 Dev 预览版 Build 22000.65。本次更新带来了众多功能方面的改善，同时修复了诸多 Bug。IT之家安装了这一更新，并进行了体验。</p><h2>一、右键菜单、开始菜单</h2><p>更新完之后，最先感知到的便是开始菜单的变化。由于 Win11 取消了之前的磁贴设计。尽管“已固定”应用列表支持上下滚动，但一些用户在寻找应用时仍会感到不便。</p><p>新版开始菜单顶部增加了搜索框，但是点击后会自动跳转到任务栏的“搜索”工具，不仅可以搜索本地应用，还支持搜索网页或文档。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/a7da258f-b959-4b40-8059-f41598ce8dbc.png" w="1440" h="959" title="Win11 Build 22000.65 体验：汉化更完善，右下角返回桌面回归" width="1440" height="546" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/0d546c0b-8066-444f-90ed-5c67fce1c7f0.png" w="950" h="845" title="Win11 Build 22000.65 体验：汉化更完善，右下角返回桌面回归" width="950" height="729" referrerpolicy="no-referrer"></p><p>本次更新为用户带来了新版右键菜单，<span class="accentTextColor">加入了“刷新”按钮</span>。这一功能满足一些用户使用右键进行刷新的习惯（也可以按键盘上的 F5）。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/ca2546b0-6a5a-479c-9eed-3e325c897021.png" w="778" h="496" title="Win11 Build 22000.65 体验：汉化更完善，右下角返回桌面回归" width="778" height="496" referrerpolicy="no-referrer"></p><h2>二、任务栏微调</h2><p>在更新完 KB5004745 后，任务栏右下角的返回桌面按钮终于恢复正常，鼠标直接拖到屏幕右下角，单击即可返回桌面。此前版本的 Win11，这一按钮莫名与屏幕边缘有一段距离，导致用户不能够通过鼠标盲点来快速回到桌面。除了这种方法，用户还可以使用 Win+D 快捷键快速回到桌面。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/9777ea68-c065-41f4-a248-d6c11ea1e47a.png" w="403" h="170" title="Win11 Build 22000.65 体验：汉化更完善，右下角返回桌面回归" width="403" height="170" referrerpolicy="no-referrer"></p><p>除此之外，更新完后的首次开机，用户会发现任务栏右下角的多个按钮、时间区域均变得更小，并且位置偏向上方。此外，IT之家还遇到了安全删除硬件按钮、显示隐藏的图标按钮出现重影的 Bug。</p><p>不过，在短暂使用并且再次重启系统后，这两项 Bug 均会自动消失。目前尚不清楚这属于偶发故障还是新特性的一部分。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/0c928161-3527-4be6-921b-40c2ddc94bbf.png" w="402" h="226" title="Win11 Build 22000.65 体验：汉化更完善，右下角返回桌面回归" width="402" height="226" referrerpolicy="no-referrer"></p><h2>三、电源模式</h2><p>新版系统可以直接在设置→系统→电源中切换三种电源模式，分别为：<span class="accentTextColor">最佳电源效率、已平衡、最佳性能</span>。</p><p>值得注意的是，在该界面切换电源模式后，传统控制面板里的“电源选项”计划不会随之更改。可以看出这两种电源模式相互独立，新功能并不依赖传统的电源计划。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/8929c83a-f5e4-4eea-86e6-b6d4dba28b8f.png" w="965" h="415" title="Win11 Build 22000.65 体验：汉化更完善，右下角返回桌面回归" width="965" height="353" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/d689b16b-c74f-4b36-a9f0-2bb70015b50d.png" w="930" h="681" title="Win11 Build 22000.65 体验：汉化更完善，右下角返回桌面回归" width="930" height="600" referrerpolicy="no-referrer"></p><h2>四、汉化</h2><p>尽管 Win11 首个预览版在系统安装界面、文件资源管理器等环境均支持简体中文，但是新版设置应用的汉化还不完全，大量选项依旧采用全英文标题以及说明。</p><p>全新的 Win11 Build 22000.65
