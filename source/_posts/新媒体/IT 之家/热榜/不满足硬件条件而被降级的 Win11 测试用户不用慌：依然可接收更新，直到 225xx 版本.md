<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 29 日消息 据德国媒体 Deskmodder，由于不符合硬件要求而被移至发布预览通道 (Release Preview) 的 Dev 测试用户仍然可接收到后续的 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 更新，直到明年 Nickel 版本（225xx）的到来。</p><p>IT之家了解到，微软曾宣布，不符合硬件要求的 Insiders 测试人员将从 Dev 频道移到 Release Preview 频道。</p><p>（科普：Windows Insiedr 预览体验计划分为三种通道，虽然大部分 Win11 用户都已经知道了，但还是要给少部分小伙伴解释一下：Dev 开发版适合高技术性用户、Beta 渠道则适合大部分喜欢尝鲜的用户，而 Release Preview 频道是稳定性最高的预览版，更适合商业用户）</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/f967f930-f44b-4450-8d18-c11e715e6bee.png" w="1026" h="581" title="不满足硬件条件而被降级的 Win11 测试用户不用慌：依然可接收更新，直到 225xx 版本" width="1026" height="464" referrerpolicy="no-referrer"></p><p>如果你现在发现自己已经处于 Release Preview 频道也无需担心，因为这一频道推送内容是分开的。如果您使用的是 Windows10，您将在此处获得 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 高级更新；如果您使用的是 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>，您将在此处收到 Win11 高级更新。</p><p>也就是说，原则上来讲，你现在切换到 Release Preview 频道后，似乎可以不必重装 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 而继续使用 Win11，哪怕现在 Win11 还没有加入 Release Preview 频道，哪怕你的 PC 硬件不满足 Windows 11 要求，你依然可以在 2023 年 10 月之前收到 Win11 更新，不过明年无法收到新版 Windows 11（22？Xx）（代号 Nickel）更新。</p><p>不过，微软将进一步收紧 Dev 渠道即将推出的 225xx 版本限制条件，以便帮助更多不满足升级条件的设备在没有指定支持的情况下退出预览更新（即不再接收新版本）。此外，微软也准备进一步限制 Win11 的注册表等偷渡方式，目前来看似乎仅可通过镜像文件安装可绕过限制，但即使这样做也无法得到微软认可且无法接收安全更新。</p>
