<p data-vmark="8ab2"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 2 日消息，天风国际分析师郭明錤发布推文表示，“显示屏下的 Face ID 将在 2024 年（<a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 16）出现，这个时间安排与其说是一个技术问题，不如说是一个营销目的。”不过，郭明錤也只是预测，两年三年后的事情苹果自己可能都没法确定是否采纳量产……</p><p data-vmark="676a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/cf9b9003-717f-4b89-8f44-f2dbd3195d9c.jpg@s_2,w_820,h_738" title="郭明錤：苹果 iPhone 16 / Pro 将搭载屏下 Face ID" referrerpolicy="no-referrer"></p><p data-vmark="ba72">此前，显示器分析师 Ross Young 称，“显示屏下的 Face ID 将出现在 iPhone（16）上，而显示屏下的 Touch ID 则不会出现...”</p><p data-vmark="4278" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/cc9e61c4-6c63-4570-9053-18cc2acd9ead.png" w="582" h="260" title="郭明錤：苹果 iPhone 16 / Pro 将搭载屏下 Face ID" width="582" height="260" referrerpolicy="no-referrer"></p><p data-vmark="1690">近期，郭明錤表示，<a href="https://www.ithome.com/0/610/569.htm" target="_blank">苹果 iPhone 15/16 系列不会采用屏下 Touch ID 指纹识别</a>，因为 iPhone 上支持戴口罩的 Face ID 已经是一个很好的生物识别技术解决方案。</p><p data-vmark="7c8c">苹果在 iOS 15.4 系统更新中支持 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 12</a> 及更新机型在戴口罩下的 Face ID 解锁和使用。</p>
