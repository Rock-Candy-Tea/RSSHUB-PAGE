<p data-vmark="e850"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 2 日消息，在预览版本中，微软为 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 增加了新的 Windows 更新功能，称为“更新堆栈包”（Update Stack Packages）。与功能体验包或功能更新不同，这些堆栈包包括一系列关键的改进和 shell 组件，使主要的 Windows 版本更新不那么具有破坏性。</p><p style="text-align: center;" data-vmark="4099"><img src="https://img.ithome.com/newsuploadfiles/2022/2/4cc1064b-6f80-46f7-8405-9ad1fe1da340.jpg@s_2,w_820,h_429" w="1440" h="753" title="推出“更新堆栈包”，微软希望 Win11 主要大更新减少破坏性" srcset="https://img.ithome.com/newsuploadfiles/2022/2/4cc1064b-6f80-46f7-8405-9ad1fe1da340.jpg 2x" width="1440" height="429" referrerpolicy="no-referrer"></p><p data-vmark="94f9">微软没有给出关于这些堆栈更新包如何工作的详细信息，但它可以使操作系统更加模块化。利用“更新堆栈包”和“功能体验包”等功能，微软可以在常规的操作系统更新之外为某些 Windows 模块推送新功能或改进。</p><p data-vmark="af71">Windows 更新堆栈包将在典型的预览版或安全版之外自动安装，而用户在检查更新时将注意到它的存在。今天，微软发布了一个新版本的堆栈包，版本为 822.124.211.0，它并没有带来新的功能。</p><p data-vmark="a72e">该 Windows 堆栈包的目的还没有明确，但它可能被用来提供更新体验的改进。</p><p data-vmark="26a6">这些软件包将在每月的累积更新或功能更新之前发布，微软此前表示，该功能将有助于确保你的 PC 保持稳定。</p><p data-vmark="635c">目前，“更新堆栈包”仅限于一小部分与更新有关的改进，但你可以期待在未来的版本中进行更多的改进。目前，微软只想在主要的 Windows 版本之外测试与更新有关的小系统文件。</p><p data-vmark="92a3">“更新堆栈包”通过 Windows Update 交付，就像累积更新一样。换句话说，你会在设置 > Windows Update 中找到这些堆栈包。</p><p data-vmark="bc72">除了 Windows 更新堆栈包和 Windows Update 的其他几个变化，Windows 11 的下一个功能更新也包含了其通常的新功能和改进。例如，“设置”应用程序已经更新，为管理 Windows 更新提供了更多选项，你通常会在控制面板中找到。</p><h3 data-vmark="e040">Windows 更新的卸载按钮</h3><p data-vmark="bacf">现在，如果你想卸载更新，你需要浏览控制面板中的 Windows Update 页面，自 Windows 1.0 以来，这一直是 Windows 的一部分。未来随着 Windows 11 22H2（又称太阳谷 2）的推出，微软终于将“卸载 Windows Update”功能迁移到了现代设置应用程序中。</p><p style="text-align: center;" data-vmark="4a9b"><img src="https://img.ithome.com/newsuploadfiles/2022/2/9f8c639e-433b-46cf-9ccb-43aab2ddf70a.jpg@s_2,w_820,h_573" w="1121" h="784" title="推出“更新堆栈包”，微软希望 Win11 主要大更新减少破坏性" srcset="https://img.ithome.com/newsuploadfiles/2022/2/9f8c639e-433b-46cf-9ccb-43aab2ddf70a.jpg 2x" width="1121" height="573" referrerpolicy="no-referrer"></p><p data-vmark="fc32">从截图可以看出，Windows 11 即将推出的“卸载更新”功能并不像控制面板那样全面，而且缺少过滤器或栏目等功能，但微软已经承诺在未来的版本中进行改进。</p>
