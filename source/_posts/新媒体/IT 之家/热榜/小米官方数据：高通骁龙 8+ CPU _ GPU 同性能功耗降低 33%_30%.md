<p data-vmark="b139"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 2 日消息，随着小米 12S 系列发布会的临近，小米的预热也越来越频繁。</p><p data-vmark="4dd4">从小米官方给出的数据来看，采用台积电 4nm 工艺的高通骁龙 8+ Gen1 芯片性能和功耗比上一代都有了很大进步，在同样的场景相比之前更加省电，相比骁龙 8 的 CPU 在相同性能表现下的前提功耗足足降低了 33%，而 GPU 也降低了 30%。</p><p data-vmark="be4c"><img src="https://img.ithome.com/newsuploadfiles/2022/7/cd02688c-6f2f-4404-a4f3-cfc5e1a2dbdd.jpg@s_2,w_820,h_1093" w="1080" h="1440" title="小米官方数据：高通骁龙 8+ CPU / GPU 同性能功耗降低 33%/30%" srcset="https://img.ithome.com/newsuploadfiles/2022/7/cd02688c-6f2f-4404-a4f3-cfc5e1a2dbdd.jpg 2x" width="1080" height="1093" referrerpolicy="no-referrer"></p><p data-vmark="7c1f">值得一提的是，海报下面还有一行小字，表明此次 CPU 测试用的是 SpecINT2K6，可能是特定情境下的功耗测试，而 GPU 是 GFXBench 测的 AztecRuins 工作负载。另外，从走线图来看这个时间点上骁龙 8 的 CPU 并未达到最大性能，可能是某个功耗差值较大的时刻。</p><p data-vmark="7cc6">IT之家了解到，骁龙 8 + 平台由三星 4nm 工艺转为台积电 4nm 工艺，基于 Arm v9 架构的全新 Kyro CPU，超大核由 3.0GHz 提升到 3.2GHz，CPU 性能也提升了 10%；此外 GPU 虽然依旧是同款 Adreno730，GPU 性能也提升了 10%。</p><p data-vmark="1c80"><span class="accentTextColor">高通表示，骁龙 8+ Gen1 的 CPU 以及 GPU 功耗降低都高达 30%，SoC 整体功耗降低了 15%</span>，而且第七代高通 AI 引擎能效提升 20%。</p><p data-vmark="01ed">博主 @数码闲聊站 还放出了小米 12S Ultra 跑《原神》的测试，同时还对比了天玑 9000 和骁龙 8，该博主称三款机型温度都是 40° 左右，但从其放出的帧数表格来看，<span class="accentTextColor">小米 12S Ultra 平均帧率更高，达到了 59.5fps，天玑 9000 和骁龙 8 则分别为 54.6fps 和 49.5fps。</span></p><p data-vmark="6be8"><span class="accentTextColor">《<a href="https://www.ithome.com/0/627/521.htm" target="_blank">小米 12S Ultra《原神》实测出炉：平均帧率达 59.5fps，骁龙 8 + 不“发烧”</a>》</span></p>
