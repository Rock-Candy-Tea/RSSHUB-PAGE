<p data-vmark="28a3"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 7 日消息，微软将在一周年之际发布代号为“Sun Valley 2”的 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 重大更新，该版本正在开发中，我们已经从 Dev 预览版了解到了部分新内容。</p><p data-vmark="f44f">而在上周发布的 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Build 22567.100 (KB5012427）的 Dev 预览版中，微软明确提到了“<span class="accentTextColor">22H2</span>”这个字样，说明新版本已经不远了。</p><p style="text-align: center;" data-vmark="9cb5"><img src="https://img.ithome.com/newsuploadfiles/2022/3/e8b5b5fd-0eb2-40f3-a5bf-a743745d9c5b.png" w="866" h="112" title="微软确认 Win11 下个大版本 22H2 将至，已出现在更新名称中" width="866" height="106" referrerpolicy="no-referrer"></p><p data-vmark="5c68">IT之家了解到，从目前的更新内容来看，22H2 版本将更侧重于改善 Windows 11 的现有界面，如开始菜单、控制中心 / 通知、MS Paint 等原生应用、Windows 设置、任务栏桌面和操作系统的其他区域。</p><p style="text-align: center;" data-vmark="b2f7"><img src="https://img.ithome.com/newsuploadfiles/2022/2/9979079c-3c4f-4b95-bee0-c46bdd4cb54c.jpg" title="微软确认 Win11 下个大版本 22H2 将至，已出现在更新名称中" referrerpolicy="no-referrer"></p><p data-vmark="0f06">此外，该更新还将重新支持任务栏拖放功能，并为 Windows 11 添加了新的任务管理器。</p><p data-vmark="cf55"><a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Build 22567.100 (KB5012427）的 Dev 预览版并没有什么大的改动，因为它是一个累积性的更新，微软正在测试其服务管道。</p><p data-vmark="285f"><img src="https://img.ithome.com/newsuploadfiles/2022/3/36492df5-fa21-4769-bf2f-3209695d5a25.jpg@s_2,w_820,h_900" title="微软确认 Win11 下个大版本 22H2 将至，已出现在更新名称中" referrerpolicy="no-referrer"></p>
