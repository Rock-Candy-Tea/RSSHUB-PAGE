<p data-vmark="228a"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 4 日消息，据 mspoweruser 报道，微软一直在慢慢升级 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 上的内置应用程序，其风格更适合于操作系统。</p><p data-vmark="db0d"><img src="https://img.ithome.com/newsuploadfiles/2021/12/cea78867-61b5-4f4b-8fa4-2ecb156d410e.png" w="466" h="741" title="微软正推送 Win11 风格 UI 版 OneDrive" width="466" height="741" referrerpolicy="no-referrer"></p><p data-vmark="73d0">▲ 图片来自 Florian</p><p data-vmark="dfac">最新得到处理的应用程序似乎是 Windows 11 上的 OneDrive 同步客户端。具体版本号为 Version 2021 (Build 21.249.1202.0001)（64 位）。</p><p data-vmark="cf4f"><img src="https://img.ithome.com/newsuploadfiles/2021/12/6773a02c-a52a-4507-8835-1526c9069684.png" w="445" h="105" title="微软正推送 Win11 风格 UI 版 OneDrive" width="445" height="105" referrerpolicy="no-referrer"></p><p data-vmark="79dc">除了明显的圆角 UI 外，该应用现在还清晰地浮现在任务栏上方。它现在还支持黑暗模式。</p><p data-vmark="7407">IT之家获悉，今年 8 月份，微软 OneDrive 64 位版本同步客户端面向 Windows 11、<a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> PC 发布，64 位 OneDrive 同步应用程序并没有带来任何新的功能，但它在 64 位版本操作系统的电脑上运行得更有效率。</p><p data-vmark="02d3">另外，微软近期宣布，OneDrive ARM64 预览版本将在 12 月中旬适配 Windows 11 ARM 和苹果 M1 Mac 设备。</p>
