<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 29 日消息 微软今天发布了第一个适用于 Windows 11 的 Insider 预览版，随后微软更新了 Windows 11 的最低系统要求。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/dc69e7c2-ab81-45b5-a07c-d423a677128a.png" w="1440" h="960" title="微软降低 Win11 的 CPU 要求：七代酷睿、初代锐龙有救了" width="1440" height="547" referrerpolicy="no-referrer"></p><p>IT之家了解到，目前微软 Win11 的最低要求是英特尔第 8 代 CPU 和 AMD Zen+ 以及 Qualcomm 7 和 8 系列及以上的设备。也就是说，基于“一系列因素”，微软放弃了对第 7 代酷睿及更旧处理器的支持。</p><p>微软现在表示，随着向 Windows Insiders 发布更新，以及与 OEM 合作，他们将测试 Win11 在英特尔第 7 代和 AMD Zen 1 设备上运行的结果。</p><p>此外，今天的预览版显然也适用于第 7 代酷睿的用户，微软则将根据此次测试的数据修改一些 Windows 11 的 CPU 要求。如果你的电脑性能够好，今年晚些时候有望获取到 Win11 正式版的支持。</p><p>以下为微软一位员工 @anduin17 在推特公布的 7 代酷睿的 Surface Studio 2 升级 Win11 商业版的图，可以看到新版 Win11 将到来更现代的资源管理器/设置界面 UI 设计。</p><p><img src="https://qimg.ithome.com/image/2021/06/28/6376050826584911316606007.jpg" title="微软降低 Win11 的 CPU 要求：七代酷睿、初代锐龙有救了" referrerpolicy="no-referrer"></p><p>《<a href="https://www.ithome.com/0/559/679.htm" target="_blank">Win11 预览版到来：即使不满足最低硬件要求，你仍可升级 Dev（附方法）</a>》</p><p>《<a href="https://www.ithome.com/0/559/331.htm" target="_blank">微软放弃支持旧机型，Win11 需要英特尔 8 代 CPU 起步</a>》</p><p>《<a href="https://www.ithome.com/0/559/572.htm" target="_blank">微软不给力，开发者于是自制了一款开源的 Win11 升级检测工具</a>》</p><p>《<a href="https://www.ithome.com/0/559/915.htm" target="_blank">微软解释 Win11 最低系统要求的原因：可实现 99.8% 的无崩溃体验</a>》</p><p>《<a href="https://www.ithome.com/0/559/959.htm" target="_blank">微软承认其 Win11 升级电脑健康状况检测工具存在 bug，现已下架</a>》</p>
