<p data-vmark="dc63"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 13 日消息，据 Windows Latest 报道，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> KB5012592 更新补丁已经面向公众发布，带有来了一长串的改进。在新功能方面，用户现在将获得搜索亮点。</p><p style="text-align: center;" data-vmark="2743"><img src="https://img.ithome.com/newsuploadfiles/2022/4/cf8b675f-aa6d-4400-aaf6-a5c70eed8d06.jpg" w="800" h="420" title="微软 Win11 Build 22000.613（KB5012592）正式版发布：搜索亮点、改进通知（附更新内容）" width="800" height="420" referrerpolicy="no-referrer"></p><p data-vmark="df6b">Windows 11 KB5012592 是一个强制性的安全更新，不能被跳过，除非暂停更新长达七天。但它也会自动安装，这取决于 Windows 更新设置，或者用户试图将其推迟到微软提供的七天宽限期之外。</p><p data-vmark="30b8">与上个月的安全更新不同，2022 年 4 月的安全更新并没有带来大量的新功能，但微软表示已经修复了许多错误 Bug，并启用了对 Windows 搜索亮点等功能的支持。此外，微软还在通知中心增加了对最多三个高优先级 toast 通知的支持。</p><p data-vmark="b1e1">如果用户经常收到来自使用 Windows 通知的应用程序的电话、信息、提醒或警报通知，这个新的通知功能可能很有用。更新后，通知将得到特殊处理，这些提醒将同时出现。</p><h3 data-vmark="af62">Windows 11 KB5012592（Build 22000.613）重要更新内容</h3><p data-vmark="6f28">微软表示，修复了一个当重命名 OneDrive 文件并按下回车键时，该文件可能失去焦点的问题。如果一个文件失去了焦点，你将被要求再次选择它，否则无法执行操作。</p><p style="text-align: center;" data-vmark="ee83"><img src="https://img.ithome.com/newsuploadfiles/2022/4/1f2988be-6f78-4671-9cf2-cfdd86d41903.jpg@s_2,w_820,h_820" w="920" h="920" title="微软 Win11 Build 22000.613（KB5012592）正式版发布：搜索亮点、改进通知（附更新内容）" srcset="https://img.ithome.com/newsuploadfiles/2022/4/1f2988be-6f78-4671-9cf2-cfdd86d41903.jpg 2x" width="920" height="820" referrerpolicy="no-referrer"></p><p data-vmark="2853">正如截图显示，微软正在为 Windows 11 引入新的搜索界面。当点击任务栏的搜索框，不输入任何东西时，这就会出现亮点功能帮助发现趋势性搜索，了解更多关于户外活动、当前日期的事件等。</p><p data-vmark="f7a6">除了新功能和改进之外，Windows 11 2022 年 4 月更新中还有许多错误 Bug 修复。</p><p data-vmark="77ec">例如，微软已经修复了一个错误，当搜索 widgets（小组件）这个词时，设置页面不会出现在结果中。微软已经修复了一个操作系统意外地关闭通用 Windows 平台（UWP）的自动启动功能，而且只有在使用最新版本的操作系统时才会发生的问题。</p><p data-vmark="b95e">同样，微软说修复了 Windows 设置和搜索的潜在性能问题，包括 SystemSettings.exe 可能停止响应和 searchindexer.exe 会阻止 Outlook 的离线搜索返回最近的电子邮件的错误。</p><p data-vmark="d8f2">正如一开始提到的，Windows 11 2022 年 4 月更新包括几个错误修复和质量改进。根据发布说明，微软修补了一个可能导致操作系统启动时间延长的问题。当网络 API 中的 UI 线程停止响应时，设备上会出现这个问题。</p><p data-vmark="857d"><strong>Windows 11 Build 22000.613 改进和修复：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="5224">修复了 wmipicmp.dll 模块导致大量错误警报和增加内存使用的问题。</p></li><li><p data-vmark="b633">修复了执行策略设置的问题，现代浏览器无法渲染由 gpresult / h 生成的 HTML，以及在对 AppLocker 进行 PowerShell 测试时对文件出现“拒绝访问”错误的问题。</p></li><li><p data-vmark="ee41">微软已经解决了 Kerberos.dll 可能在本地安全授权子系统服务（LSASS）中停止工作的问题。</p></li><li><p data-vmark="d6b9">修复了一个防止用户账户控制（UAC）对话框显示要求提升权限的应用程序的问题。例如，如果用管理员权限启动游戏，屏幕上会出现请求访问的弹出窗口，但不会显示游戏的名称。</p></li><li><p data-vmark="47e1">修复了一个阻止用户访问 SMB 共享的问题。</p></li></ul><p data-vmark="4546">截至目前，微软还没有确认任何新的已知问题。</p><p data-vmark="4599">然而，据报道，上次的可选更新对一些用户造成了性能和任务栏问题，类似的问题也可能出现在这次安全更新中。</p>
