等虚拟机在 Mac 上安装 Linux 的方法。但到了苹果自研的 M1 处理器，一切又得重头开始。</p><p data-vmark="57ed">而 Asahi Linux 的目标，就是让苹果 M1 系列 Mac 用户能真正拥抱 Linux。</p><p style="text-align: center;" data-vmark="b126"><img src="https://img.ithome.com/newsuploadfiles/2022/4/3a6c80be-a627-4ded-9856-9d5547efe303.gif" w="1079" h="530" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1079" height="403" referrerpolicy="no-referrer"></p><p data-vmark="3b10">所以在苹果电脑上装一个原生 Linux 系统，究竟是一种怎样的体验？一位有 30 万 + 粉丝的 YouTuber 在他的 M1 Mac 上亲测了一下。</p><p style="text-align: center;" data-vmark="c49c"><img src="https://img.ithome.com/newsuploadfiles/2022/4/0472b9ad-c835-4805-bc6a-d22e7332b5f5.png" w="1080" h="233" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="177" referrerpolicy="no-referrer"></p><p data-vmark="44bf">经过一系列测评，小哥表示：<span class="accentTextColor">真香</span><span class="accentTextColor">！</span></p><p data-vmark="b86e">在完全相同的硬件上运行某些程序，Linux 竟然比 macOS 还快！</p><p style="text-align: center;" data-vmark="0181"><img src="https://img.ithome.com/newsuploadfiles/2022/4/d59fe835-b2ae-4ea2-924d-3978bc98a3c7.png" w="1080" h="610" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="463" referrerpolicy="no-referrer"></p><p data-vmark="e79f">而且 Mac 电脑看起来并不像 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 手机和 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 系统那样封闭。证据是，苹果官方还会积极为用户修复 Bootloader 启动器上发现的 Bug。</p><p data-vmark="8caa">那么小哥到底是怎么搞定 M1 双系统的呢？</p><h2 data-vmark="9582">安装前的准备工作</h2><p data-vmark="3cbb">磨刀不误砍柴工。博主提醒，想尝试 Asahi Linux 的朋友，一定要先看清楚安装要求，并最好选择一台没有重要文件的计算机来操作。</p><p data-vmark="7e2c">Asahi Linux 官网也给出了详细安装须知：</p><p style="text-align: center;" data-vmark="18e8"><img src="https://img.ithome.com/newsuploadfiles/2022/4/1c27b7cd-e9cc-4d1f-b652-ac7f6a543b7d.png" w="1080" h="462" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="351" referrerpolicy="no-referrer"></p><p data-vmark="e06a">否则，如果出问题了，你的电脑很可能就无法启动了。</p><p style="text-align: center;" data-vmark="d3fc"><img Desktop 等虚拟机上安装这个操作系统不就行了？</p><p data-vmark="0832">Asahi 官方对此回复，Asahi Linux 是为苹果硬件量身打造的，而 Parallels Desktop 虚拟的是一个通用 ARM 架构，适合安装其他 Linux 发行版。</p><p style="text-align: center;" data-vmark="7df2"><img Plasma</span> 桌面环境。</p><p style="text-align: center;" data-vmark="ca2e"><img src="https://img.ithome.com/newsuploadfiles/2022/4/3c207b46-f99d-4a7f-8d96-d3a395d60a06.jpg@s_2,w_820,h_459" w="1080" referrerpolicy="no-referrer"></p><h2 data-vmark="99c8">Asahi Linux 香在哪儿？</h2><p data-vmark="a158"><strong>视频播放性能</strong></p><p data-vmark="cb56">在没有使用 GPU 加速器的情况下，博主在 M1 Mac mini 上的 Asahi Linux 系统中播放了一个 YouTube 视频。</p><p data-vmark="2d13">他欣喜地发现，在这里可以播放 0 掉帧的全屏 HD 视频，且 UI 也一直很灵敏，而不像其他许多基于 ARM 的单板机那样迟钝。</p><p data-vmark="ac4f"><strong>Bug 修复速度</strong></p><p data-vmark="4bfc">接着，博主使用 Iperf 测试了网络性能，并发现了 Asahi Linux 的一个 bug——</p><p data-vmark="2208">由于 Asahi 将网络适配器设置为使用单一的 PCI Express 通道，导致 10 千兆的超快网络<span class="accentTextColor">被限制到仅 1.5 千兆</span>。</p><p data-vmark="3f79">小哥将这个漏洞反馈到了 Asahi 的聊天区里。<span class="accentTextColor">几分钟后，开发者就修复了这个 bug</span>，并表示这将适用于所有 Asahi 用户。</p><p style="text-align: center;" data-vmark="7e00"><img src="https://img.ithome.com/newsuploadfiles/2022/4/db61b721-ce48-49ed-a341-a641f66c5a77.png" w="1080" h="675" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="512" referrerpolicy="no-referrer"></p><p data-vmark="ab9d">官方这认真的态度也是没谁了！</p><p data-vmark="cb62"><strong>降低 CPU 负载</strong></p><p data-vmark="1b08">博主发现，只要打开图形性能监视器，CPU 的负载会增加 25% 左右；但如果执行 Asahi Linux 给出的命令行’htop’，CPU 的的负载将会<span class="accentTextColor">直线下降到 1%</span>。</p><p style="text-align: center;" data-vmark="919e"><img src="https://img.ithome.com/newsuploadfiles/2022/4/687f4240-6ee9-47a5-a0fb-ad339245e2ae.png" w="1080" h="675" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="512" referrerpolicy="no-referrer"></p><p data-vmark="b0a0"><strong>运行速度</strong></p><p data-vmark="4747">博主以“重新编译树莓派 Linux 内核”为基准，对不同系统进行了测试。</p><p data-vmark="b790">结果发现，<span class="accentTextColor">M1 mini 上 Linux 的速度比 macOS 足足快了 40%</span>！这个速度与在有两倍性能核心的 Mac Studio 上跑 macOS 的速度不相上下。</p><p style="text-align: center;" data-vmark="62a5"><img src="https://img.ithome.com/newsuploadfiles/2022/4/f80216bd-854b-42fb-a304-aea35e9d2f9b.png" w="1080" h="675" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="512" referrerpolicy="no-referrer"></p><p data-vmark="9b65">价格方面，M1 Mac mini 最低价格是 699 美元，而配置豪华点的 Arm 开发版如 HoneyComb LX2，光一块板子就要 750 美元了。</p><p style="text-align: center;" data-vmark="7e42"><img src="https://img.ithome.com/newsuploadfiles/2022/4/0ee7fd1d-51d8-4477-bb2a-f6899e8fadc4.png" w="1080" h="535" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="406" referrerpolicy="no-referrer"></p><p data-vmark="d15c">所以，如果你想要一台能运行 Linux 的 Arm 电脑，M1 Mac mini 本身也不失为一个好选择。</p><p data-vmark="2ff3">另外，博主的一个朋友 Michael Larabel 还使用 Phoronix 套件，对 M1 mini 上的 macOS 和 Asahi Linux 进行了测试。</p><p data-vmark="700c">对于一些检测计算机性能的基准，如：LevelDB，macOS 表现得更优秀。</p><p style="text-align: center;" data-vmark="35de"><img src="https://img.ithome.com/newsuploadfiles/2022/4/fc8133a5-df17-409d-9f38-d0faef22a236.png" w="1080" h="350" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="266" referrerpolicy="no-referrer"></p><p data-vmark="c41a">但在一些别的方面，如：WebP 图像编码，Asahi Linux 具有优势。</p><p style="text-align: center;" data-vmark="dfc3"><img src="https://img.ithome.com/newsuploadfiles/2022/4/e92d2846-5537-4424-980a-4dfdd7b1a879.png" w="1080" h="380" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="289" referrerpolicy="no-referrer"></p><p data-vmark="c5f8">值得注意的是，目前 Asahi Linux 还处于测试阶段，未来 Asahi Linux VS macOS 会有怎样的改变，还得交给时间来证明。</p><h2 data-vmark="1746">Asahi Linux 到底是什么来头？</h2><p data-vmark="8a5e">所以 Asahi Linux 是怎么来的？</p><p data-vmark="7f3b">2020 年末，Asahi Linux 由程序员 <span class="accentTextColor">Hector Martin</span> 众筹发起。</p><p data-vmark="61be">Hector Martin 是一位网安高手，也是资深的操作系统移植专家，曾为各种设备提供非官方的开源支持来移植 Linux 系统，包括任天堂 Wii、索尼 PS 系列游戏主机等。</p><p data-vmark="1640">2021 年 1 月份，Asahi Linux 项目众筹完毕，正式启动；<span class="accentTextColor">今年 3 月份上线了测试版</span>。</p><p data-vmark="3933">据其官网称，Asahi 这个名字也是有一定来历的：它在日语中的意思是“<span class="accentTextColor">朝日</span>”，同时也是代表着苹果品种“旭”，这正是 Mac 系统名字的来源 <span class="accentTextColor">Mcintosh 苹果的日语名字</span>。</p><p style="text-align: center;" data-vmark="c819"><img src="https://img.ithome.com/newsuploadfiles/2022/4/a70fa5b3-4783-42c0-8d8d-8711a3e3e92c.png" w="1080" h="707" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="537" referrerpolicy="no-referrer"></p><p data-vmark="7b62">Asahi Linux 由开源社区的工程师们开发，<span class="accentTextColor">可免费使用</span>。</p><p data-vmark="dfb8">目前 Asahi Linux 支持的功能如下，包括 CPU 频率转换、RTC 等。</p><p style="text-align: center;" data-vmark="1d27"><img src="https://img.ithome.com/newsuploadfiles/2022/4/fdaac513-0623-447d-aae6-9640bf2bdf21.png" w="862" h="1326" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="862" height="1261" referrerpolicy="no-referrer"></p><p data-vmark="6483">当然，还在开发阶段的 Asahi Linux 还有许多地方需要完善，例如：不能使用蓝牙、没有 GPU 加速等。</p><p style="text-align: center;" data-vmark="61b9"><img src="https://img.ithome.com/newsuploadfiles/2022/4/7df3824b-5682-4fee-8f07-a4723b97c121.png" w="1080" h="549" title="真香警告！在苹果 M1 芯片上测试跑原生 Linux：编译速度比 macOS 还快 40%" width="1080" height="417" referrerpolicy="no-referrer"></p><p data-vmark="58ce">开发团队希望，未来 Linux 不仅能在 M1 系列机器上运行，而且能作为一个日常的操作系统在各个 Mac 机器上运行。</p><h2 data-vmark="9550">还未正式发布就被寄予厚望</h2><p
