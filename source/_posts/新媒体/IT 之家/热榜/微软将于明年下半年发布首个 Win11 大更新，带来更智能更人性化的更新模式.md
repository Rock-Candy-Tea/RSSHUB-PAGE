<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 4 日消息 微软在 6 月 24 日正式发布下一代桌面操作系统 Windows 11，并表示 Win11 将于今年晚些时候开始向用户推出。</p><p>众所周知，在 Windows 10 发布后，微软一直按照春季和秋季的双节点提供大版本功能更新，而随着 Windows 11 将取代 Windows 10，微软的版本更新节奏也将发生改变。</p><p>在 Win11 正式到来后，微软将转向每年发布一次功能更新，且微软官方表示 Win11 第一次重大更新将于明年下半年发布。</p><p>此外，微软将为每个新版本提供 24 个月 (消费者版) 时间或 36 个月（企业版和教育版）的支持时间。</p><p>IT之家了解到，和 Windows 10 一样，Win11 也将继续向用户推送累积更新，也就是所谓的周二更新或“B”版本。对于部分有需求的用户，微软还将发布 Windows 11 的可选累积更新。</p><h2>更小更智能的改进</h2><p>微软表示，之后的 Windows 更新将会像现在的 Win10 一样提供，但这些累计更新将会变得更小（体积小 40%），用户也可以期待更快的更新过程，更稳定和有效的 Windows 体验，微软也将持续致力于 Win11 的建设。</p><p>据外媒 WindowsLatest 报道，除了这些质量改进之外，微软还正在试验一个新功能，你的电脑可以通过这个新功能告诉系统更新重启和安装所需要的时间。</p><p>而且，这个新功能适用于所有类型的 Windows 更新，包括 .net 框架、周二更新、可选更新等。当然，对于那种不需要重启系统的更新，它也不会去刻意找存在感。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/0d7a1d78-b300-4d94-a12f-7d13f16a9be7.jpg" alt="Windows 更新估计时间" w="292" h="158" title="微软将于明年下半年发布首个 Win11 大更新，带来更智能更人性化的更新模式" width="292" height="158" referrerpolicy="no-referrer"></p><p>▲ 在当前的预览版本中 Windows 更新预估时间固定显示为 5 分钟</p><p>由图可知，微软会从更新内容和 PC 性能预计其 Windows 更新时间，并将其置于开始菜单和 Windows 更新设置界面中。预估时间将取决于 PC 综合性能，例如你的处理器、内存和硬盘等。</p>
