<p data-vmark="2d2d"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 29 日消息，JetBrains 在官方博客宣布，<span class="accentTextColor">推出一款有点不一样的轻量级编辑器 Fleet，并称其为“下一代 IDE”</span>。</p><p data-vmark="9d74"><img src="https://img.ithome.com/newsuploadfiles/2021/11/3b563e38-bcb5-463e-a5bd-2201436fe012.png" w="1440" h="763" title="微软 VS Code 新对手？JetBrains 发布全新轻量编辑器 Fleet，号称“下一代 IDE”" width="1440" height="434" referrerpolicy="no-referrer"></p><p data-vmark="4167">据介绍，当用户第一次启动 Fleet 时，<span class="accentTextColor">它会作为一个功能齐全的编辑器启动</span>，具有语法高亮显示、简单的代码补全以及用户对一个编辑器期待的所有功能。</p><p data-vmark="339c"><img src="https://img.ithome.com/newsuploadfiles/2021/11/3b2a9215-7b28-4eff-b554-4c3c27d57000.png" w="1293" h="960" title="微软 VS Code 新对手？JetBrains 发布全新轻量编辑器 Fleet，号称“下一代 IDE”" width="1293" height="609" referrerpolicy="no-referrer"></p><p data-vmark="d937">Fleet 还是一个功能齐全的 IDE，具有智能补全、重构、导航、调试以及 IDE 中一直存在的所有功能。更重要的是，点击一键即可使用所有这些功能。</p><h2 data-vmark="e753">Fleet 是适用于多语言的</h2><p data-vmark="36e7">开发者通常在不同的项目中会使用到不同的技术，有时在单个项目中也会使用不同的技术。<span class="accentTextColor">有了 Fleet，用户不再需要打开不同的 IDE 来获得特定技术所需的功能</span>。</p><p data-vmark="5d8b"><img src="https://img.ithome.com/newsuploadfiles/2021/11/f26537be-35a3-4f49-a487-1462c06d43e2.png" w="1075" h="817" title="微软 VS Code 新对手？JetBrains 发布全新轻量编辑器 Fleet，号称“下一代 IDE”" width="1075" height="623" referrerpolicy="no-referrer"></p><h2 data-vmark="59b8">Fleet 是分布式的</h2><p data-vmark="68bd">Fleet 是从头开始构建的，以满足那些不一定在本地项目上工作的开发者的需求。因为 Fleet 有一个分布式架构，无论项目是在本地、在容器中，还是在千里之外的国家，都没有关系。<span class="accentTextColor">Fleet 使用虚拟文件系统，用户可以使用 Fleet 在本地和远程以相同的方式处理项目</span>。</p><p data-vmark="bdf9">Fleet 架构是分布式和前端、后端、工作区服务器和文件系统观察程序。 这使得 Fleet 本身可以以分布式方式运行，例如，允许在远程机器上运行分析和其他操作的场景。</p><h2 data-vmark="a543">Space 中的 Fleet</h2><p data-vmark="73ec">JetBrains Space 添加了编排支持，可从源仓库轻松启动远程服务器实例，支持使用 Dockerfile 进行自定义。</p><p data-vmark="721a"><img src="https://img.ithome.com/newsuploadfiles/2021/11/248a3fed-4ee2-493f-ae1e-fde68ae85ad7.png" w="1288" h="951" title="微软 VS Code 新对手？JetBrains 发布全新轻量编辑器 Fleet，号称“下一代 IDE”" width="1288" height="605" referrerpolicy="no-referrer"></p><h2 data-vmark="4931">Fleet 是协作性的</h2><p data-vmark="de73">Fleet 的构建连接了协作的思想。无论是本地还是远程，用户可以轻松进行项目协作。它能够同时处理相同或不同的文件、运行测试、访问终端以及执行协作 IDE 所期望的其他功能。</p><p data-vmark="91ef"><img src="https://img.ithome.com/newsuploadfiles/2021/11/09dadfc9-3044-4463-a681-2dac8d4d421c.png" w="1440" h="610" title="微软 VS Code 新对手？JetBrains 发布全新轻量编辑器 Fleet，号称“下一代 IDE”" width="1440" height="347" referrerpolicy="no-referrer"></p><p data-vmark="1ab9">JetBrains 表示，他们将在开发工具方面，尤其是 IDE 开发，20 年的知识和经验应用于开发 Fleet。不仅如此，它还融合了当下的技术。事实上，<span class="accentTextColor">用户在 Fleet 中看到的所有智能技术都使用 IntelliJ 代码处理引擎作为后端</span>。</p><p data-vmark="e116">Fleet 的目标是为了满足那些可能只需要一个编辑器，但也同时需要 IDE 中的强大功能的场景，想要使用单一工具而非多个专用工具的用户提供不同的体验。当然，Fleet 也可以针对现有的 IDE 在分布式开发时可能无法满足的特定场景。此外，非常重要的一点是 <span class="accentTextColor">Fleet 不会替换现有工具，而是添加了更多选项</span>。与往常一样，JetBrains 将继续投资于所有的 IDE，而 Fleet 也会继续利用这些进展。</p><p data-vmark="aabe">IT之家了解到，Fleet 仍处于起步阶段，用户可以访问早期版本的 Fleet，请填写<a href="https://www.jetbrains.com/zh-cn/fleet/" class="ek-link">申请表</a>加入 Fleet Explorer 计划。</p>
