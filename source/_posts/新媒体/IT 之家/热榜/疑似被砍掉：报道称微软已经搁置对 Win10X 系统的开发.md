<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 8 日消息 在 2019 年秋天的一次活动中，微软详细介绍了 Windows 10X。与 Surface Duo 和 Surface Neo（第一款 Windows 10X 旗舰机）一起，该操作系统本应将微软带向一个新的方向。该公司最初对 Windows 10X 的宣传是为双屏设备优化的更现代、更轻便的 Windows 版本，甚至一度被认为要与 Chrome OS 抗衡。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/f3a72b19-d0dc-4e1b-b74a-04318661efc5.jpg@s_2,w_820,h_547" w="1200" h="800" title="疑似被砍掉：报道称微软已经搁置对 Win10X 系统的开发" srcset="https://img.ithome.com/newsuploadfiles/2021/5/f3a72b19-d0dc-4e1b-b74a-04318661efc5.jpg 2x" width="1200" height="547" referrerpolicy="no-referrer"></p><p>然而，在推迟和将重点转向单屏设备之后，现在看来 Windows 10X 可能永远不会正式推出。根据外媒 The Verge 的说法，<span class="accentTextColor">微软今年不会推出 Windows 10X，而且该操作系统可能永远不会以目前的形式出现</span>。据报道，微软公司计划将其为该操作系统开发的一些功能整合到 Windows 10 中，这些元素可能包括用户界面和应用程序容器。</p><p>IT之家了解到，微软拒绝对 Windows 10X 的状态发表评论。</p><p>至于决定背后的原因，似乎大多数微软给予机会尝试 Windows 10X 的人都喜欢该公司开发的功能，但对在一个与他们每天使用的操作系统完全不同的操作系统中使用它们并不感兴趣。</p>
