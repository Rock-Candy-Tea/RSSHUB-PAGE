            <p data-vmark="4fd3"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 6 日消息，据 Windows Latest 报道，Windows 文件资源管理器的标签支持是反馈中心中要求最多的功能之一。过去，微软曾试图通过 Windows Sets 将其带给用户。关于 Windows Sets 的工作最终被放弃，但是微软并没有放弃在文件资源管理器中设置标签页的想法，微软再次启动了文件资源管理器标签的工作。</p><p data-vmark="697c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/1bc7a546-3119-4745-a79c-13c389b7b177.png" w="924" h="612" title="不会砍掉，微软宣布 Win11 全新设计版文件资源管理器支持“标签页”，支持右键扩展" width="924" height="543" referrerpolicy="no-referrer"></p><p data-vmark="a06f">在 Windows 混合办公发布会上，微软正式确认未来的 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 预览版将会有新的文件资源管理器。备受期待的标签页功能现已经在内部构建的资源管理器中启用，预计今年晚些时候会公开推出，测试人员很快就能获得标签页功能。</p><p data-vmark="6522">微软尚未在 Windows 11 预览版中添加新的文件资源管理器，但该公司在活动中预告了其新设计。文件资源管理器的标签与微软 Edge 标签类似。用户可以创建新的标签，拖放标签，而且现在可以在右键单击时选择标签或窗口。</p><p data-vmark="941d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/698762d8-0213-4bc2-80d2-e094123bad83.jpg@s_2,w_820,h_443" w="1440" h="778" title="不会砍掉，微软宣布 Win11 全新设计版文件资源管理器支持“标签页”，支持右键扩展" srcset="https://img.ithome.com/newsuploadfiles/2022/4/698762d8-0213-4bc2-80d2-e094123bad83.jpg 2x" width="1440" height="443" referrerpolicy="no-referrer"></p><p data-vmark="3875" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/cb087cfb-0862-4608-9448-d546aeb53926.png" w="920" h="516" title="不会砍掉，微软宣布 Win11 全新设计版文件资源管理器支持“标签页”，支持右键扩展" width="920" height="460" referrerpolicy="no-referrer"></p><p data-vmark="f5d6">标签页是默认启用的，目前尚不清楚用户是否可以禁用该功能。</p><p data-vmark="6e1c">微软表示，文件资源管理器已经设计好，并提供云端支持，所以用户将从 Microsoft 365 账户获得文件建议。可以更容易找到用户要找的东西，并且如果你购买了 Microsoft 365 订阅，还可以在一个地方看到所有的文件。</p><p data-vmark="4952">微软还在增加新的文件资源管理器主页，包括典型的快速访问文件夹、最近的文件和新的收藏夹选项。用户很快就能右键单击一个文件，将其添加到收藏夹，它将出现在文件资源管理器的主页上。甚至还有改进的文件共享选项，更新的共享对话框可以让用户发送至最近的联系人或 Teams、Outlook 和 OneDrive 等应用程序。</p><p data-vmark="8473">这些改进预计将与 Windows 11 版本 22H2（太阳谷 2）一起推出，微软还在为任务栏和“开始”菜单开发一些新功能。例如，“开始”菜单将获得文件夹支持，任务栏更新后将支持拖放。</p>
