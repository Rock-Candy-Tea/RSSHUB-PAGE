<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 17 日消息 外媒 MacRumors 报道，苹果公司今天发布了新版本的 iCloud for Windows 应用，支持 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>、<a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 系统等，iCloud  12.5 版本的更新为 Windows 用户增加了新的 iCloud 钥匙链密码管理器应用。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/c1586bb5-0f3b-46ee-9438-cccb78c1b05a.png" w="864" h="700" title="苹果 iCloud Win11/Win10 桌面版 12.5 正式版发布：支持全新钥匙链密码管理器" width="864" height="664" referrerpolicy="no-referrer"></p><p>有了新的密码管理选项，运行 Windows 的用户可以访问 iCloud 钥匙链密码，并可以添加、编辑、复制和粘贴、删除，以及查询用户名或密码。苹果在今年 1 月份发布了 Windows 版 iCloud 的更新版本，暗示了新的密码应用，但苹果直到现在才实施。</p><p>微软 Edge <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>和谷歌 Chrome 浏览器的新 iCloud 密码扩展与密码功能协同工作，让 iCloud 密码在 Windows 设备上自动填写，就像在 Safari 浏览器中一样。为网站创建的用户名和密码会自动添加到密码应用程序中。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/92baeaae-5c42-4b90-bb50-c62309394cf6.png" w="872" h="701" title="苹果 iCloud Win11/Win10 桌面版 12.5 正式版发布：支持全新钥匙链密码管理器" width="872" height="659" referrerpolicy="no-referrer"></p><p>IT之家获悉，Windows 版 iCloud 钥匙链密码存储在一个加密的数据库中，并使用加密通道传输到浏览器扩展，密码不以明文存储。</p><p>iCloud for Windows 密码是 iCloud for Windows 12.5 的一部分，Windows 用户可以从<a href="https://www.microsoft.com/zh-cn/p/icloud/9pktq5699m62?rtc=1" target="_blank">微软商店</a>下载它。适用于 Windows 的 iCloud 应用程序旨在让 Windows 用户从 Windows 设备访问他们的 iCloud 文件、照片、电子邮件等。</p>
