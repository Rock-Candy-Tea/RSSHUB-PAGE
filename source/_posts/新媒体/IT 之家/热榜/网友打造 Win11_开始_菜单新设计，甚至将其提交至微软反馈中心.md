<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 19 日消息 外媒 Neowin 报道，自从微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>“开始”菜单的首批构建版本可供 Windows Insiders 使用以来，其实施的变化一直是相当热门的话题。事实上，它已经被简化为光荣的“最近”列表，人们已经在寻找替代和解决方案。就在一个多星期前，Stardock 推出了 Start11 工具，将以前的 Windows 版本的“开始”菜单带回，然而 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 的风格和动态磁贴在测试版中明显缺席。</p><p>现在，一位名叫 Dot Matrix 的网友决定分享他自己对 Windows 11 中“开始”菜单的看法和设计，甚至还将其作为<a href="https://insider.windows.com/en-us/feedbackhub/fb?contextid=49&feedbackid=4b140c2c-ae26-4b7b-9f2a-af13ef6303f9&form=1&utm_source=product-placement&utm_medium=feedback-hub&utm_campaign=feedback-hub-redirect" target="_blank">建议</a>提交到微软的反馈中心。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/00636b1f-dbff-4f53-b5a8-15d53d9129df.jpg" w="760" h="506" title="网友打造 Win11“开始”菜单新设计，甚至将其提交至微软反馈中心" width="760" height="506" referrerpolicy="no-referrer"></p><p>正如你看到的，这款 Windows 11“开始”菜单回到了 Windows 10 的布局，左边是“所有应用程序”，右边是固定的应用程序组。我们看到的是应用程序的图标，而不是动态磁贴。Dot Matrix 继续说，“没有更多的推荐的项目部分。这个完全丢弃了。”</p><p>我们可以理解他的想法。“推荐的项目”看起来很酷，但它不是很注重隐私。在许多方面，它比跳转列表建议更糟糕，跳转列表建议可以在以前的 Windows 版本中被关闭。IT之家获悉，<a href="https://www.ithome.com/0/568/574.htm" target="_blank">“推荐的项目”也可以在 Windows 11 中关闭</a>，但它是一个全局设置，所以如果禁用它，你将得到一个半空白的“开始”菜单，没有跳转列表或文件资源管理器的建议。(所以为什么不把这些选项分开呢，微软？)</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/a286115d-76c6-4dce-a106-e3a14b314342.jpg" w="760" h="506" title="网友打造 Win11“开始”菜单新设计，甚至将其提交至微软反馈中心" width="760" height="506" referrerpolicy="no-referrer"></p><p>在这里我们可以看到一个全屏版的“开始”菜单，可以通过平板电脑模式调用。总而言之，它已经比目前微软设计的 Windows 11 更有用了，即使所显示的白色空间仍需要改进。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/7b6e3436-1bc1-489f-93ec-ac46c66b4597.jpg" w="760" h="506" title="网友打造 Win11“开始”菜单新设计，甚至将其提交至微软反馈中心" width="760" height="506" referrerpolicy="no-referrer"></p><p>最后，如图所示，把整个小组件窗格和动态磁贴的精神继承者移回“开始”菜单，而不是弹出窗格。</p><p>外媒认为，微软 Windows 11 中的小部组件窗格并没有经过深思熟虑。微软说，要想获得天气和新闻的“一目了然的信息”，需要从任务栏中打开窗格，完全错过了将这些小工具固定在桌面上的机会。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/b6c523a0-99fd-44c7-9df1-f1315613b18b.jpg@s_2,w_820,h_615" w="1400" h="1050" title="网友打造 Win11“开始”菜单新设计，甚至将其提交至微软反馈中心" srcset="https://img.ithome.com/newsuploadfiles/2021/8/b6c523a0-99fd-44c7-9df1-f1315613b18b.jpg 2x" width="1400" height="615" referrerpolicy="no-referrer"></p><p>随着我们的屏幕越来越大，越来越宽（比如拥有一款 5180x1440 的超宽屏幕），能够直接在桌面上放置小组件是有意义的。设置一个新闻提醒？除非你特意决定查看小组件窗格，否则你不会看到它，如果它被固定在桌面上，你看到它的机会就大得多了。微软曾经把很好的想法用在了动态磁贴上，但在 Windows 10 中实施得很糟糕，以至于现在已经完全停用了。</p>
