<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a>3月26日消息 外媒 Windows Latest 报道，Windows 10 的任务栏可能会从 Windows 文件资源管理器 (explorer.exe)中分离出来，未来该功能可能会得到其独立的进程。在 Windows 10 的太阳谷更新（也就是 21H2 版本）的预览版中发现了这些变化。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210326_071552_875.png" w="1024" h="683" title="曝微软正将 Win10 「任务栏」从 Explorer.exe 中移出：响应将更流畅稳定" width="1024" height="547" referrerpolicy="no-referrer"></p><p>IT之家获悉，直到现在，在所有最新版本的 Windows 10 和其他版本中，任务栏和 explorer.exe 一直共存。<span class="accentTextColor">“Explorer.exe”进程负责文件资源管理器，并显示大部分的用户界面（UI），其中包括任务栏、操作中心等等</span>。</p><p>通常情况下，Explorer.exe 的工作没有任何故障，但当它冻结时，可能会使你的桌面失去作用，并将任务栏也隐藏起来。当发生这种情况时，你无法与任务栏或其他体验进行交互，这包括固定在任务栏或跳转列表（右键菜单）上的应用程序。</p><p>在 Windows 10 上，你可以通过重新启动操作系统或任务管理器的 Windows 资源管理器进程来解决这些问题。</p><p>为了彻底解决这些问题，微软现在计划将 “任务栏”从 Explorer.exe 中移出。</p><p>在最新的预览版中，已经发现了对 “taskbar.dll”的引用，这似乎表明微软希望在未来为任务栏保留一个单独的进程。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210326_071614_502.jpg" w="401" h="505" title="曝微软正将 Win10 「任务栏」从 Explorer.exe 中移出：响应将更流畅稳定" width="401" height="505" referrerpolicy="no-referrer"></p><p>值得注意的是，目前只是将任务栏的部分内容从 Explorer.exe 中移出，任务栏并不会在短期内获得自己的进程。资源管理器仍然包含任务栏代码，但将来可以改用 DLL 实现，这样可以减少对 Explorer.exe 的依赖。</p><p>这也可以让任务栏运行得更加流畅。通过减少对资源管理器的依赖，微软应该能够保护它免受文件资源管理器可能引起的潜在问题。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210326_071627_109.jpg" w="540" h="39" title="曝微软正将 Win10 「任务栏」从 Explorer.exe 中移出：响应将更流畅稳定" width="540" height="39" referrerpolicy="no-referrer"></p><p>对于消费者来说，最终的结果是更可靠的任务栏体验和更灵敏的响应，因为它将不再依赖于文件资源管理器。</p><p>如截图所示，新的 taskbar.dll 似乎并不能正常工作，它渲染的搜索栏也是损坏的。</p><p>目前，还不清楚这一变化是否也能使跳转列表（右键菜单）的启动时间稍快，这个问题在 2020 年被谷歌工程师指出，并得到微软的承认。</p><p><strong>新任务栏功能</strong></p><p>除了性能改进之外，微软一直在测试任务栏的《新闻和天气》小工具，它旨在使用 Microsoft Edge 和 Windows 搜索基础设施为你提供最新的更新。</p><p>这个概念类似于 Google Discover，它基本上是在桌面右侧添加了一个新闻源，如下图所示。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210326_071642_484.jpg@base@tag=imgScale&w=827&h=657&q=90" w="827" h="663" title="曝微软正将 Win10 「任务栏」从 Explorer.exe 中移出：响应将更流畅稳定" srcset="https://img.ithome.com/newsuploadfiles/2021/3/20210326_071642_484.jpg 2x" width="827" height="657" referrerpolicy="no-referrer"></p><p>它将根据你的兴趣或在微软 MSN 账户中选择的主题显示头条新闻、天气、体育、金融和新闻。</p><p><strong>文件浏览器的其他改进</strong></p><p>除此之外，<a href="https://www.ithome.com/0/542/071.htm" target="_blank">微软还在为 Windows 文件资源管理器开发新的彩色图标</a>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210326_071657_641.jpg@base@tag=imgScale&w=1440&h=464&q=90" w="1440" h="814" title="曝微软正将 Win10 「任务栏」从 Explorer.exe 中移出：响应将更流畅稳定" srcset="https://img.ithome.com/newsuploadfiles/2021/3/20210326_071657_641.jpg 2x" width="1440" height="464" referrerpolicy="no-referrer"></p><p>这些新的现代图标基于微软的 Fluent Design，这是该公司改造操作系统视觉界面计划的一部分。</p>
