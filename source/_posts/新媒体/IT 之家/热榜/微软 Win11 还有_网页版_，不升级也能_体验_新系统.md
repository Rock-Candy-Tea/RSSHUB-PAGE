<p>距离 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 发布已经过去一个多月时间，可由于 TPM 以及其他方面的缘故，并不是每个人都有机会体验新系统。对于那些老爷机用户，如何一睹 Win11 的真容呢？日前，一位名叫 Blueedge 的国外大神打造了这样一款“网页版 Win11”。只要一款<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>，在任何一台电脑上，都能随时随地体验微软的最新操作系统，甚至…… 你的手机也不例外！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/b35d221c-dac2-4b3a-bd96-5641b9b02ea3.jpg" w="700" h="454" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="700" height="454" referrerpolicy="no-referrer"></p><p class>网址：<a href="https://win11.blueedge.me/" target="_blank">https://win11.blueedge.me/</a></p><p>网页版 Win11 是 Github 上的一个开源项目，打开浏览器访问上述网址即可。整个界面高度模仿了 Win11，包括居中式任务栏、开始菜单、圆角 UI、Win11 壁纸、分屏模式等都被 1:1 还原出来。甚至你还可以通过 F11 键将浏览器设为全屏状态，沉浸式体验这款“模拟版 Win11”。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/2dfb166c-7c1b-412b-90a1-334aeeafe31c.jpg" w="626" h="633" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="626" height="633" referrerpolicy="no-referrer"></p><p>除了外观以外，“网页版 Win11”对于组件的还原程度也很高。比如开始菜单，除了外观上与实机版相差无几外，里面的功能也大抵和实机版一样。比如你可以在里面直接打开某个程序，或者点击“All apps”进入全部应用列表。同时由于是网页模拟的缘故，它的动画过渡甚至比实机版还要流畅，整个体验非常顺滑。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/f717d2f3-efa3-40e6-b8dd-a437378a2cf6.gif" w="610" h="620" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="610" height="620" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/ebf6e8d7-055d-4a67-a890-f626f508822e.jpg" w="387" h="406" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="387" height="406" referrerpolicy="no-referrer"></p><p>软件方面，作者加入了 Edge、电邮、应用商店、记事本、白板、计算器、Twitter、Terminal 终端、VS Code 等等（开始菜单中带“红点”的项目）。除了外观以外，这些功能也都是可以正常使用的。不过由于不是虚拟机，整个过程我们只能体验一下，无法生成任何真实文件。至于想通过商店安装应用的想法，当然也是不可行的，仅仅是看看就好…… 看看就好。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/7d6cf236-9ecb-4ecc-b958-5b473a9c89a0.jpg" w="352" h="434" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="352" height="434" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/51f95734-2520-4e7a-849a-36f7acd416dc.jpg" w="700" h="404" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="700" height="404" referrerpolicy="no-referrer"></p><p>唯一有点变化的就是浏览器了，确确实实是真实可用的，包括打开网址、搜索引擎都没问题。甚至你还可以通过鼠标右键，将当前网页打印出来。在浏览器中打开浏览器，这大概也算是一个比较有意思的体验吧！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/ca70602b-8a2d-4daf-a3ed-e870fe97753b.jpg" w="700" h="453" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="700" height="453" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/ecae37aa-f606-49cd-abc8-ca2d3b9612ad.jpg" w="700" h="422" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="700" height="422" referrerpolicy="no-referrer"></p><p>窗口分屏也是真正存在的，操作体验也和实机版一致。只是所有的快捷键在这里统统无效，必须要用鼠标操作。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/b3aad86d-c4cf-4dd8-bc20-c6ed086b2ac9.jpg" w="700" h="404" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="700" height="404" referrerpolicy="no-referrer"></p><p>这是小部件界面，但仅仅是演示了一个样子。除了翻页功能外，里面的所有内容均无法点击，包括天气与新闻。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/96f1e11c-75f3-44ac-a3cc-f4ea38af8b75.gif" w="705" h="786" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="705" height="786" referrerpolicy="no-referrer"></p><p>接下来是 Win11 标志性的日历面板，不过作者在这里仅仅还原了日历面板，操作中心和通知面板并没有合并，依旧还是之前的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 样子，稍显遗憾！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/fdd439b1-a996-4fea-ac66-7999e6a5f6a4.jpg" w="392" h="500" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="392" height="500" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/43353e95-cc9b-4f72-bb9f-8f78109373fe.jpg" w="458" h="794" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="458" height="794" referrerpolicy="no-referrer"></p><p>Windows 终端也有点意思，内置了十几组常用命令，包括显示文件夹内容的 DIR、运行程序或命令的 START、显示系统信息的 SYSTEMINFO 等。虽然与真正的 Windows 终端没法相比，但确实给这款“网页版”带来了一些新玩法。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/2ed19f33-42ed-4f6c-87b9-74d477388243.jpg" w="700" h="404" title="微软 Win11 还有“网页版”，不升级也能“体验”新系统" width="700" height="404" referrerpolicy="no-referrer"></p><h2>写在最后</h2><p>从体验上讲，这款“网页版 Win11”还是可圈可点的。最大优势就是效果逼真，随便一台电脑甚至手机都能体验。特别是对那些没办法升级新系统的老爷机用户来说，不失为一个不错的选择。不过问题也是不少，比如没有设置面板、没法修改当前界面、没有文件管理器、桌面图标不支持点击等…… 特别是它不是一台虚拟机，无法生成真实文件，也无法与主机交换信息等，让它仅仅只能作为一个尝鲜的手段。当然即便如此，这款“Win11”也是相当有新意的，感兴趣的小伙伴不妨一试！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/95d9fd3c-7f33-418f-b65a-f29009bdb16e.jpg" w="349"
