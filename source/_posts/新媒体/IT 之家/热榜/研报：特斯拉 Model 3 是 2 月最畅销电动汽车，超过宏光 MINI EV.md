<p>IT之家 4 月 2 日消息 ，据国外媒体报道，特斯拉 2016 年 3 月底推出、次年 7 月开始交付的 Model 3，是他们旗下目前起售价最低的电动汽车，也是产量、交付量最高的一款，销量在主要市场也位居前列。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210402_182321_720.jpg@base@tag=imgScale&w=1300&h=284&q=90" w="1300" h="450" title="研报：特斯拉 Model 3 是 2 月最畅销电动汽车，超过宏光 MINI EV" srcset="https://img.ithome.com/newsuploadfiles/2021/4/20210402_182321_720.jpg 2x" width="1300" height="284" referrerpolicy="no-referrer"></p><p>ev-sales 最新的数据显示，全球 2 月份共交付约 270000 辆新能源汽车增长 136%，特斯拉和中国厂商们表现亮眼。特斯拉 Model 3 是 2 月份最畅销的电动汽车，其次是五菱宏光 Mini EV，第三名是特斯拉 Model Y，第五名是比亚迪的汉 EV。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210402_182831_435.jpg" w="280" h="137" title="研报：特斯拉 Model 3 是 2 月最畅销电动汽车，超过宏光 MINI EV" width="280" height="137" referrerpolicy="no-referrer"></p><p>▲仅 2 月</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210402_182527_13.jpg" w="409" h="495" title="研报：特斯拉 Model 3 是 2 月最畅销电动汽车，超过宏光 MINI EV" width="409" height="495" referrerpolicy="no-referrer"></p><p>▲年初至今</p><p>此外，特斯拉终于在总交付量上重返第一，而特斯拉+通用五菱的市占率达到了 23%，近乎整个市场的四分之一。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210402_182723_60.jpg" w="365" h="489" title="研报：特斯拉 Model 3 是 2 月最畅销电动汽车，超过宏光 MINI EV" width="365" height="489" referrerpolicy="no-referrer"></p><p>从研究机构公布的数据来看，2 月份特斯拉在全球范围内共向消费者交付 28014 辆 Model 3，也是欧洲最畅销的新能源汽车。</p><p>IT之家了解到，五菱宏光 MINI EV 是 2 月份全球销量仅次于特斯拉 Model 3 的电动汽车，，销售 20167 辆，较 Model 3 低 7847 辆。</p><p>除了 Model 3，特斯拉在 2019 年 3 月份推出、去年开始交付的跨界运动型多用途电动汽车 Model Y，交付量在 2 月份也位居前列，达到了 13971 辆，仅次于特斯拉 Model 3 和五菱宏光 MINI EV。</p>
