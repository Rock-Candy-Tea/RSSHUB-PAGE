<p data-vmark="cf74"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 7 日消息，据彭博社记者 Mark Gurman 称，苹果的第一款配备 Face ID 的 Mac 不太可能是 MacBook，因为将身份验证硬件嵌入到笔记本显示器部分<span class="accentTextColor">所需的技术仍然不存在</span>。</p><p style="text-align: center;" data-vmark="b193"><img src="https://img.ithome.com/newsuploadfiles/2021/10/374b821a-9c55-4678-b49a-d0c7e5afed03.png" title="Gurman：MacBook 太薄了，苹果还没有把 Face ID 放到里面的技术" referrerpolicy="no-referrer"></p><p data-vmark="eb31">Gurman 关于这个主题的回答出现在他最新的“Power On”时事通讯的问答部分。关于 Face ID 是否会出现在 Mac 上，Gurman 写道：</p><blockquote><p data-vmark="70f5">Face ID 最初将被用于 M1 iMac。因为 iMac 是最厚的带有内置显示屏的 Mac，而 MacBook 的屏幕相当薄。在这一点上，将 Face ID 嵌入到薄薄的 MacBook 显示器部分的技术还不存在。因此，如果 Face ID 出现在 Mac 上，我认为它将首先出现在 iMac 或外接显示器上。苹果肯定一直在努力，但时间会证明他们是否会推出它。</p></blockquote><p data-vmark="9e2e">苹果在 2017 年首次展示 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone X</a> 时，就为其搭载了 Face ID 面部识别系统，此后，该系统在大多数 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 和 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 中取代了 Touch ID，并且有传言称未来的 Mac 也将采用 Face ID 。</p><p data-vmark="ab0c">IT之家了解到，Gurman 在 2021 年 7 月谈到这个话题时表示，他认为苹果打算“在几年内”推出其首款配备 Face ID 的 Mac，而且苹果最初计划在 2021 年的 24 英寸 iMac 中加入 Face ID。</p><p data-vmark="2a6b">除了 Mac 上的 Face ID 问题外，Gurman 的最新通讯重申，苹果计划在 3 月 8 日星期二举行一场虚拟活动，推出新的 iPhone SE 和 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad Air</a> 机型，预计这两款机型都将配备 A15 芯片和 5G 支持。</p>
