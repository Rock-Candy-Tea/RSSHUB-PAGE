<p>Win11 已经发布开发者预览版了，很多朋友都通过 Insider 通道体验到了这个下一代的 Windows 系统。和 Win10 相比，Win11 在很多地方都发生了剧变，界面交互都焕然一新。不过这些改变是否都广受欢迎？起码从用户反馈来看，Win11 的任务栏就拥有很多槽点，今天就一起来吐槽一下吧。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/a11ff5f3-777e-476d-a3ae-fe9dd0f996c7.jpg" w="640" h="331" title="比旧版 Windows 更难用？吐槽 Win11 的任务栏设计" width="640" height="331" referrerpolicy="no-referrer"></p><p>Win11 这个新任务栏被吐槽得不轻</p><p>首先我们回顾一下 Win10 的任务栏。尽管和苹果的 Dock 相比，Windows 的任务栏或许没有这么美观，但在功能性、实用性方面，Windows 任务栏可谓是好评如潮。而 Win10 的任务栏继承了前代 Windows 任务栏的种种特性，除了外观有所不同，其余特性悉数保留。但这一切却在 Win11 中，却变得大为不同。</p><p>新版的 Win11 基于全新的组件构造，它的任务栏被 UWP 化，基于 XAML 构成，按理来说这将会带来更多的先进特性。然而，人们发现 Win11 的任务栏非常难用，具体包括但不仅限于下面几点。</p><p>・Win11 任务栏位置被固定在底部，无法更改。之前的 Windows 系统，任务栏可以设置在屏幕四边，很多用户已经养成了任务栏在屏幕侧边或者上方的习惯。而 Win11 的任务栏只能固定在底部，尽管可以通过修改注册表来变更位置，但一来这并不是系统默认开放的设置项，二来将任务栏移到其他地方后，点击开始菜单仍会从屏幕下方弹出，使用上存在很多问题，这就引发了很多用户的不满。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/52b7fc05-8880-4193-a1e4-59346e7fb7b1.jpg" w="640" h="376" title="比旧版 Windows 更难用？吐槽 Win11 的任务栏设计" width="640" height="376" referrerpolicy="no-referrer"></p><p>通过修改注册表把任务栏固定到屏幕上方，但弹出开始菜单的位置就错乱了</p><p>・Win11 任务栏右键菜单大缩水。点击 Win10 的任务栏，会弹出丰富的功能选项，可以直接进行呼出任务管理器、调节窗口、调节任务栏显示内容等操作。但在 Win11 中，在任务栏的大部分位置点击右键，仅仅会显示一个“设置”的选项，只有在开始菜单按钮上点击右键，才会呼出功能菜单。据悉，任务栏的右键菜单之所以缩水，是为了让相应功能整合进设置面板，从而减少杂乱简化体验，但事实上这带来了更繁琐的操作步骤。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/66da8f29-3798-405d-843b-8d05a146a3b7.jpg" w="640" h="240" title="比旧版 Windows 更难用？吐槽 Win11 的任务栏设计" width="640" height="240" referrerpolicy="no-referrer"></p><p>右键点击任务栏，只会弹出一个“设置”</p><p>・显示桌面按钮过小。点击任务栏最右方的竖条显示桌面，是 Windows 任务栏的一个巧妙设计。但在 Win11 中，这个设计变得不好用了。Win11 任务栏最右方的小竖条按钮大为缩水，不少用户都反馈说很难点中，操作效率大大降低。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/a5730f99-ebfc-4e2d-9d62-a920760537a8.jpg" w="640" h="373" title="比旧版 Windows 更难用？吐槽 Win11 的任务栏设计" width="640" height="373" referrerpolicy="no-referrer"></p><p>右侧显示桌面的按钮只有一条线，实测无法盲点</p><p>・无法将应用图标拖放到任务栏。在之前的 Windows 版本中，我们可以将应用程序本身，或者它的快捷方式，直接拖到任务栏中固定。但在 Win11 中，这个操作不再有效，这改动令人摸不着头脑。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/1471d6fc-0165-42e5-af54-40306e0de4c2.jpg" w="640" h="242" title="比旧版 Windows 更难用？吐槽 Win11 的任务栏设计" width="640" height="242" referrerpolicy="no-referrer"></p><p>无法直接将软件或者快捷方式直接拖到任务栏固定</p><p>・无法关闭 snap groups。Win11 的任务栏并非只是砍功能，也引入了一些新特性，例如引入了一个名叫“snap groups”的特性，这可以让用户将不同的应用图标抓到一起，在任务栏中显示为一个图标，切换应用的时候只需要将鼠标悬浮在这个图标上即可，更加方便快捷。但问题在于，Win11 并不支持关闭这个功能，如果你不习惯，那也只能被迫接受。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/0c1ec816-8a55-475b-9dfa-688b8fa661fa.jpg" w="638" h="238" title="比旧版 Windows 更难用？吐槽 Win11 的任务栏设计" width="638" height="238" referrerpolicy="no-referrer"></p><p>不是所有人都习惯 snap groups，但无法关闭</p><p>总的来说，Win11 的任务栏对比 Win10 乃至前几代 Windows，可谓是焕然一新，这必然会改变用户的使用习惯。目前不少用户都并不满足新的任务栏设计，所幸的是微软表示正在听取用户的反馈，希望在 Win11 的正式版中，任务栏能变得更完善、更好用吧。</p>
