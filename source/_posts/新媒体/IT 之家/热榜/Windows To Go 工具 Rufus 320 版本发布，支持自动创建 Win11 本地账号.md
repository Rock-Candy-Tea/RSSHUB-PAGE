<p data-vmark="f03f"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 4 日消息，经典的 Windows To Go 工具 Rufus 迎来 <span class="accentTextColor">3.20 版本更新</span>，除了此前支持的绕过 Windows 11 TPM、SecureBoot 和微软账户要求外，新版本带来了更多功能优化。</p><p data-vmark="9119" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/8/f9c9a340-31dd-4c9a-adfc-f0e6e12d0fe8.png" w="733" h="789" title="Windows To Go 工具 Rufus 3.20 版本发布，支持自动创建 Win11 本地账号" width="733" height="789" referrerpolicy="no-referrer"></p><p data-vmark="06c5">Rufus 3.20 已支持为用户<span class="accentTextColor">自动创建 Windows 11 的本地账户</span>，并支持记住上一次安装时选择的用户体验选项。</p><p data-vmark="dfdb"><strong>更新日志：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="8ba9">为 Windows 10 启用适用的 Windows 用户体验选项。</p></li><li><p data-vmark="38c2">记住上一次选择的 Windows 用户体验选项。</p></li><li><p data-vmark="c9a0">添加自动本地账户创建和区域选项复制。该功能仅限于创建一个与当前用户同名且密码为空的账户，用户将在第一次重新启动后被提示更改。</p></li><li><p data-vmark="cb18">为具有指向 /isolinux/ (Knoppix) 的 syslinux 符号链接的 ISO 添加解决方法。</p></li><li><p data-vmark="9529">尽可能为绕过 TPM / SB / RAM 的注册表项恢复为脱机插入。</p></li><li><p data-vmark="0e18">在显示“仅 UEFI 引导”警报消息时提高 BIOS 兼容性。</p></li><li><p data-vmark="85f6">修复 Windows To Go 出现两次 Windows 用户体验对话框的问题。</p></li><li><p data-vmark="4f1e">修复 Windows 用户体验选项不适用于 ARM64 的问题。</p></li><li><p data-vmark="96fc">修复除非选择了绕过 TPM / SB / RAM，否则无法绕过微软账户的问题。</p></li><li><p data-vmark="29e1">修复了对带有非标准前缀的 GRUB2 引导加载程序的过度检测。</p></li></ul><p data-vmark="6adf">IT之家小伙伴可以在 <a href="https://rufus.ie/zh/#" target="_blank">Rufus 官网</a>或者 <a href="https://github.com/pbatard/rufus/releases/tag/v3.20" target="_blank">GitHub</a> 下载 3.20 版本。Rufus 是一个可以帮助格式化和创建可引导 USB 闪存盘的工具，比如在 U 盘中安装 Windows 11 操作系统。</p>
