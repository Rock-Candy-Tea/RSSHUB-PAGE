<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 6 日消息 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Build 22000.120 现已上线 Windows 预览体验计划的 DEV 和 Beta 版频道，相关用户现可体验最新版的功能。</p><p>IT之家了解到，这是 Windows 11 RTM Build 22000 的第五个累积更新版本，带来了多项质量增强功能，但没有任何新功能或重大改进。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/12d3c9f8-4ac7-4f9b-86cd-e8cf65f8cc5a.png" w="1015" h="273" title="微软发布 Win11 Build 22000.120 (KB5005188) 更新：精简右键菜单，大量细节优化" width="1015" height="221" referrerpolicy="no-referrer"></p><p>你现在可以通过 Windows 检查更新下载并安装新补丁“Windows 11 累积更新 (10.0.22000.120) (KB5005188)”。</p><p>值得一提的是，当您更新到 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 最新版本时，它还会自动下载并安装新的 Defender 安全更新，贯彻微软所宣称的 安全主题。</p><p>Windows 11 KB5005188 为 PC 用户提供了部分功能性的改进，例如微软更新了文件资源管理器的右键菜单功能，为桌面用户提供了一个相比此前更显紧凑的界面。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/29e71a60-0de5-481c-a72e-cd3acccfa34c.jpg@s_2,w_820,h_476" w="1213" h="704" title="微软发布 Win11 Build 22000.120 (KB5005188) 更新：精简右键菜单，大量细节优化" srcset="https://img.ithome.com/newsuploadfiles/2021/8/29e71a60-0de5-481c-a72e-cd3acccfa34c.jpg 2x" width="1213" height="476" referrerpolicy="no-referrer"></p><p>此外，微软正在通过集成下拉菜单改进文件资源管理器中的“新建”按钮。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/61159ec9-28d4-4845-8bca-08d7a920401e.jpg" w="781" h="494" title="微软发布 Win11 Build 22000.120 (KB5005188) 更新：精简右键菜单，大量细节优化" width="781" height="494" referrerpolicy="no-referrer"></p><p>从实际体验来看，这将允许用户查看列表中的所有选项，而非嵌套列表。该公司还解决了在长时间使用右键菜单后导致文件资源管理器卡顿的问题。</p><h2>其他方面的更新：</h2><p>Windows 11 带来了一个新的小部件面板，虽然它本质上依然是一组小程序。你可以直接从任务栏查看这些小部件，包括天气、运动、交通情况更新等。</p><p>Build 22000.120 还为用户引入了一个新的家庭小部件，需要登录 Microsoft 帐户，支持所有语言和地区，它可以帮助您查看 Microsoft 家庭组的近期活动。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/19d0f9e1-eecd-44cd-a053-5823d116f7cc.png" w="620" h="844" alt="新的家庭小部件可让您查看 Microsoft 家庭组成员的近期活动。" title="微软发布 Win11 Build 22000.120 (KB5005188) 更新：精简右键菜单，大量细节优化" width="620" height="844" referrerpolicy="no-referrer"></p><p>据悉，微软正准备为任务栏上的聊天软件图标推出通知标记，代表用户可直接从任务栏查看未读的提醒内容。此外，微软还优化了任务栏预览窗口中的关闭按钮、实装了显示设置中的“识别”功能，以提高多显示器体验。</p><p>以下是 Build 22000.120 (KB5005188) 中其他改进的列表：</p><ul class=" list-paddingleft-2"><li><p>改进了 ALT + Tab、任务视图和快照辅助中的间距和缩略图大小。</p></li><li><p>文件资源管理器在黑暗模式下会拥有新的滚动条。</p></li><li><p>Microsoft 已解决影响文件资源管理器的内存溢出错误。</p></li><li><p>文件资源管理器的命令栏已更新，从而减少资源使用。</p></li><li><p>Microsoft 已修复任务资源管理器崩溃的 bug。</p></li><li><p>右键菜单将不再闪烁或被固定在某一边。</p></li><li><p>滚动条问题已修复。</p></li><li><p>文件资源管理器中的单选按钮现在在处于选中状态时突出显示。</p></li><li><p>Microsoft 已修复影响文件操作对话框的问题。</p></li></ul><h2>微软商店</h2><p>Microsoft Store 同时也获得了具有多项质量改进的新更新。例如 Microsoft Store 现获得了对聚焦下“自动滚动”的支持，允许用户无需手动滚动列表即可发现下一个喜爱的应用、电影或游戏。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/a82af9c1-64b6-400d-9ee1-8c8bab045710.jpg@s_2,w_820,h_592" w="977" h="705" title="微软发布 Win11 Build 22000.120 (KB5005188) 更新：精简右键菜单，大量细节优化" srcset="https://img.ithome.com/newsuploadfiles/2021/8/a82af9c1-64b6-400d-9ee1-8c8bab045710.jpg 2x" width="977" height="592" referrerpolicy="no-referrer"></p><p>此外，微软还引入了游戏页面的详情，以便用户可以查看游戏的更多信息和截图。据悉，微软还提供了一个新的评论对话框，新版界面终于是基于 XAML 而不是 WebView。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/967e59b2-99ab-4fe3-b860-8e9a4483ce4e.png" w="1440" h="1043" alt="我们更新了评分和评论表单，让您可以更轻松地提供有关您使用的产品的反馈。" title="微软发布 Win11 Build 22000.120 (KB5005188) 更新：精简右键菜单，大量细节优化" width="1440" height="594" referrerpolicy="no-referrer"></p><h2>Build 22000.120 中的已知问题</h2><p>Windows 11 Build 22000.120 可能会损坏搜索栏并导致以下问题：</p><ul class="ai-word-checked list-paddingleft-2"><li><p>Windows Search 检测不到用户输入的文本。</p></li><li><p>右击“开始”按钮 (WIN + X) 时，不会弹出系统和 Windows Terminal。</p></li><li><p>任务栏闪烁。</p></li><li><p>当您将鼠标悬停在任务栏的搜索上时，可能不会出现最近的搜索历史。</p></li><li><p>Windows 沙盒中的任务栏崩溃。</p></li><li><p>搜索框可能显示为黑色或空白。</p></li><li><p>对于某些用户，Windows 小部件将显示为空白，不过这可以通过重新登录微软帐户来解决。</p></li></ul><p>值得注意的是，这一版本的 Beta 版似乎还不如 Dev 版稳定。如果你通过 Beta 通道更新后发现无法访问任务栏或开始菜单，请手动卸载更新。</p><p>当然，如果遇到 bug 你还可以通过回滚到以前版本的 Windows 来解决。更多更新内容请参见<a href="https://blogs.windows.com/windows-insider/2021/08/05/announcing-windows-11-insider-preview-build-22000-120/" target="_blank">微软官方博客</a>。</p>
