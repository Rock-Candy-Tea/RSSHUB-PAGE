<p data-vmark="86a7"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 30 日消息，据 Windows Latest 报道，微软已经确认强大的小组件将通过应用商店进入 Windows 11，但与许多人想要的相反，微软仍然对将小组件转变为 Windows Vista 和 Windows 7 的经典小组件进化版本的计划保持沉默。</p><p data-vmark="96e8">Windows 7 的桌面小组件允许用户通过快速浏览固定的小组件直接在桌面上阅读股票更新和获取天气预报。小组件后来从操作系统中删除，它们再也没有卷土重来，尽管许多用户希望将类似的功能添加到 Windows。</p><p data-vmark="6956">Windows 11 的小组件方法完全不同，它基于“新闻和兴趣”体验。此外，仅支持使用微软自适应卡片的基于 Web 的小组件，因此总体而言，操作系统中的新小组件提要只是改进了已经可以在任务栏上使用的“新闻和兴趣”。</p><p data-vmark="9df1">然而，<span class="accentTextColor">微软现在正在探索对基于桌面的小组件的支持，该小组件具有称为“搜索栏”的功能。</span></p><p data-vmark="9a83">在 Windows 11 Build 25120 中，桌面上出现了一个新的浮动搜索栏，用户可以访问 Bing 的搜索栏。</p><p style="text-align: center;" data-vmark="0f1a"><img src="https://img.ithome.com/newsuploadfiles/2022/5/abcc0ca8-9a43-4b05-9843-decd75c6f063.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzk0LnBuZw==,g_3,x_9,y_9,a_0,t_100" w="609" h="473" title="微软 Win11 第一个桌面小组件上手体验：并非 Windows Vista / 7 进化版" width="609" height="473" referrerpolicy="no-referrer"></p><p data-vmark="d5b1">正如在上面的截图中看到的，微软正在直接向桌面添加一个“搜索栏”。与经典小组件不同，搜索栏不提供任何其他选项来个性化搜索栏或其功能。</p><p data-vmark="61f6">搜索栏也是静态的，它的位置不能改变。</p><p data-vmark="9ee2">与 Bing 类似，当在搜索栏中输入关键字时，最多会出现六个建议。</p><p data-vmark="3bb9">微软在官方公告中表示：“为了开始评估这种总体思路和交互模型，该领域的第一次探索在桌面上添加了一个显示的搜索框，可以让你搜索网络。”</p><p data-vmark="fff3">但是，有一个问题 —— 搜索栏不遵循用户配置的默认浏览器。换句话说，如果你的默认浏览器是 Chrome，搜索栏将忽略你的选择并将搜索查询定向到微软 Edge 浏览器。</p><p data-vmark="6e2e">如果你不喜欢搜索栏，可以随时通过上下文菜单禁用它。</p><p data-vmark="5e5c">好消息是，这可能是某种桌面小组件集成的开始，小组件功能有可能演变成 Windows 7 时代的经典小组件。</p><p data-vmark="d9e5">值得注意的是，根据 Build 2022 开发者大会上的公告，微软正在努力为小组件面板提供第三方小组件支持。第三方 API 支持将允许开发人员将他们的应用程序提交到微软商店并有助于提高用户参与度。</p><p data-vmark="a63a">此外，微软还在内部测试对动态小组件和平板电脑用户全屏模式的支持。</p>
