<div class="tougao-user">感谢IT之家网友 <a href="https://m.ithome.com/html/app/open.html?url=ithome%3A%2F%2Fuserpage%3Fid%3D1883497" rel="nofollow">sitinpeace</a> 的线索投递！</div>
            <p data-vmark="e736"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 2 日消息，今年早些时候，微软为 Windows 11 推出了<span class="accentTextColor">适用于平板用户的任务栏</span>，然而在 Windows 11 Build 22610 中被取消了。</p><p data-vmark="7f25" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/907b3b81-0f50-43e0-a132-019ac672b177.jpg@s_2,w_820,h_544" w="820" h="544" alt="Taskbar tablet UI" title="微软 Win11 任务栏平板 UI 回归：图标更大，可自动隐藏成细条" width="820" height="544" referrerpolicy="no-referrer"></p><p data-vmark="e550">该任务栏设计就像<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>或 iOS，用户可以从任务栏向上滑动来扩展它并启动钉住的应用程序。当用户从平板电脑的桌面底部向上滑动时，<span class="accentTextColor">会出现一个更宽的任务栏，上面有大图标</span>，以帮助平板电脑用户轻松点击图标。</p><p data-vmark="32d9">微软当时的说法是，根据早期测试人员的反馈，效果并不理想，所以该功能不会很快回归。然而四个月后，<span class="accentTextColor">这个设计又回来了</span>。</p><p data-vmark="3cf4">据 @PhantomOfEarth 发现的信息，在今日发布的 <a href="https://www.ithome.com/0/638/535.htm" target="_blank">Windows 11 Dev 预览版 25193</a> 中，用户可手动开启一项这一任务栏平板 UI。</p><p data-vmark="0c91" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/a8877411-63ae-444d-b3cb-92128bec61b9.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzEwMy5wbmc=,g_3,x_10,y_10,a_0,t_100" w="669" h="773" title="微软 Win11 任务栏平板 UI 回归：图标更大，可自动隐藏成细条" width="669" height="773" referrerpolicy="no-referrer"></p><p data-vmark="5e76">该设置在 Windows 11 任务栏设置下的“任务栏行为”中，当用户勾选该选项后，将在平板形态下自动显示优化后的任务栏，也就是我们上面提到过的 UI。</p><p data-vmark="5d8a">要想体验该设置，IT之家小伙伴需要使用工具手动开启，不建议在主力机上测试，可能会存在不稳定的情况。</p><ul class=" list-paddingleft-2"><li><p data-vmark="48e8">从 <a href="https://github.com/thebookisclosed/ViVe/releases" target="_blank">GitHub 存储库</a>下载 Vive Tool</p></li><li><p data-vmark="f67d">提取 Vive Tool 文件到某个目录</p></li><li><p data-vmark="e545">以管理员身份启动 Windows 终端，切换到命令提示符</p></li><li><p data-vmark="d1df">使用 CD 命令导航到 Vive Tool 的文件夹。例如，CD C:\Vive</p></li><li><p data-vmark="d117">输入 <span class="accentTextColor">vivetool /enable /id:26008830</span> 并按回车</p></li><li><p data-vmark="b1ca">重启电脑</p></li></ul><p data-vmark="c276">如果想要关闭可以输入 <span class="accentTextColor">vivetool /disable /id:26008830</span>。</p><p data-vmark="51d5">此外，开启该功能后，会出现一个<span class="accentTextColor">新的任务栏启动动画</span>。</p><p data-vmark="2155" style="text-align: center;"><a class="ithome_super_player" contenteditable="false"
