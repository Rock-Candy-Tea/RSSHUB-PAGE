<p data-vmark="0c75"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 7 日消息 如果你有一台受 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 支持的设备，但却在 Windows Update 页面上看到错误“这台电脑目前不符合 Windows 11 的所有系统要求”，<span class="accentTextColor">微软已经证实这是一个错误。</span></p><p data-vmark="f9f1">对于 Windows Insiders 用户，微软一直在测试一项新功能，在 Windows Update 中直接告诉用户 Windows 11 的兼容状态：“此电脑不能运行 Windows 11”或“此电脑可以运行 Windows 11”。</p><p data-vmark="a420">该信息是 PC 健康检查应用程序和其他第三方工具的一个方便和更直接的替代，然而在这一过程中也出现了一些 bug，目前困扰 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 上的 Windows Update 的问题是准确报告 Windows 11 兼容性状态的能力。</p><p data-vmark="ea6e">正如下面的截图所见，当拥有兼容 Windows 11 设备的用户检查更新时，Windows 10 给出了错误信息：“这台电脑目前不符合 Windows 11 的所有系统要求。了解详情，看看是否有你可以在 PC 健康检查应用程序中做的事情”。</p><p data-vmark="af8a"><img src="https://img.ithome.com/newsuploadfiles/2021/10/bb195bce-b1c7-400b-97ae-9922ee7548ba.jpg@s_2,w_820,h_533" w="1091" h="709" title="微软证实：“这台电脑无法运行 Win11”报错有时会错误弹出" srcset="https://img.ithome.com/newsuploadfiles/2021/10/bb195bce-b1c7-400b-97ae-9922ee7548ba.jpg 2x" width="1091" height="533" referrerpolicy="no-referrer"></p><p data-vmark="523f">据悉，<span class="accentTextColor">该错误信息偶尔会出现在符合 Windows 11 升级条件的 PC 上，似乎并不普遍。</span></p><p data-vmark="1dc1">电脑健康检查显示电脑可以运行 Windows 11，但 Windows Update 仍然称“这台电脑目前不符合 Windows 11 的所有系统要求”，重新启动和检查更新都没有帮助。</p><p data-vmark="2e49">错误的报告阻止在支持的硬件上升级 Windows 11，但用户仍然可以选择通过安装助手或 Windows 11 ISO 下载和安装新的操作系统。</p><p data-vmark="5d9a">值得庆幸的是，对于那些被这个问题影响的用户，<span class="accentTextColor">微软已经确认这是一个已知的问题</span>，公司正在积极调查此事，并正在研究解决方法。“我们意识到了这个问题，我们目前正在努力进行修复，”微软的支持人员称。</p><p data-vmark="b784">值得注意的是，错误的兼容性状态不仅出现在 Windows Update 中，而且在微软的规格页面上也会出现。</p><p data-vmark="c66a">微软建议用户在反馈中心提交报告，以帮助公司确定根本原因。</p><p data-vmark="4df9">IT之家了解到，除了新的处理器和 TPM 2.0 要求外，用户还需要 Windows 10 2004 版或更高版本才能看到 Windows 11 更新通知。此外，必须安装 9 月的累积更新。如果你没有最新版本的 Windows 10，Windows 11 通知就不会在 Windows Update 上显示出来。在升级到 Windows 11 之前，确保关闭 VBS 功能，以减少性能和游戏问题。</p>
