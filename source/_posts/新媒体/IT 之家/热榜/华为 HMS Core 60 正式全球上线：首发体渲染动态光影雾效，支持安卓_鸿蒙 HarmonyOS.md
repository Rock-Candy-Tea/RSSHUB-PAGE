<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 15 日消息 华为官方宣布，HMS Core 6.0 于 7 月 15 日正式全球上线，为应用开发者带来了多项全新的开放能力，并对已有的特性及服务进行了升级。目前，用户设备内置的 HMS Core 6.0 APK 已全面完成更新升级，开发者可登录华为开发者联盟官网下载使用各项服务。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/47c624de-0ed2-4a1f-9bd7-f10f01731ec0.jpg@s_2,w_820,h_462" w="1080" h="608" title="华为 HMS Core 6.0 正式全球上线：首发体渲染动态光影雾效，支持安卓/鸿蒙 HarmonyOS" srcset="https://img.ithome.com/newsuploadfiles/2021/7/47c624de-0ed2-4a1f-9bd7-f10f01731ec0.jpg 2x" width="1080" height="462" referrerpolicy="no-referrer"></p><p>HMS Core 6.0 将华为在媒体应用、图形渲染、网络加速等多个优势领域的先进技术进一步开放。在媒体领域，新增了多媒体管线服务（AV Pipeline Kit），通过音视频开发框架及视频超分、声音事件检测等插件，帮助开发者解决音视频应用开发难、功耗压力大的痛点。在图形领域开放了 3D 建模能力（3D Modeling Kit），为开发者提供基于 AI 技术的物体建模、材质制作等能力，用户仅需普通 RGB 相机，通过手机便可高效生成 3D 模型，让电商类、动画制作类 App 获得趣味交互，目前该能力支持 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 全平台。</p><p>IT之家获悉，HMS Core 6.0 在现有的图形计算服务（CG Kit）中，业界首次在移动端实现了基于体渲染技术的动态光影雾效“体积雾”，帮助手游提升室内场景画面的真实感和表现力。该特性也已面向 Android 所有支持 Vulkan 的设备开放。</p><p>在通信领域，Network Kit 升级了 AI 网络环境预测，可基于业务的网络访问规律以及差异化网络需求，进行网络参数调优，让应用的网络时延降低超过 20%。</p><p>HMS Core 6.0 还开放了包括会员、视频剪辑、音频剪辑等多项服务，开发者可以登录华为开发者联盟官网查看更多详细信息。</p><p>华为 HMS Core 为跨平台、跨设备的应用提供了一致性体验。HMS Core 支持包括 Web 应用、快应用，以及 Android、<a class="s_tag" href="https://harmonyos.ithome.com/" target="_blank">HarmonyOS</a> 等应用和服务开发，同时也支持在平板、智慧屏、智能手表及车机等设备调用。此外，HMS Core 还支持丰富的第三方开发框架，如 React Native、Cordova 等。</p>
