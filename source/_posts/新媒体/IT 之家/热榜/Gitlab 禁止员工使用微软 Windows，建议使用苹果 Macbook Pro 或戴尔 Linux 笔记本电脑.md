<p data-vmark="bde3"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 9 日消息，Gitlab 公司最新宣布了 IT 团队计算机管理规定，其中禁止员工使用微软 Windows 系统，并且建议使用 macOS 或 Linux。Gitlab 公司列举了诸多原因，包括授权许可证成本和安全因素。由于 Gitlab 是一个基于 Web 的平台，因此矛盾与 IT 团队成员在包括微软 Edge 在内的各种浏览器中可用的测试可能性有关。</p><blockquote><p data-vmark="c2e9">“由于 Windows 在桌面操作系统中的主导地位，Windows 是间谍软件、病毒和勒索软件最具针对性的平台。macOS 预装在苹果计算机上，Linux 可免费使用。”</p></blockquote><p data-vmark="50ed"><span class="accentTextColor">要批准使用 Windows，GitLab 必须购买 Windows Professional 许可证，因为 Windows 家庭版不符合 GitLab 的安全准则。</span>由于许多笔记本电脑是由后来 GitLab 员工报销购买的，因此远程办公员工通常会购买预装 Windows Home 家庭版的笔记本电脑。众所周知，Windows Home 家庭版难以提供可靠防护。”</p><p data-vmark="3df5">一些人认为 Gitlab IT 团队成员需专注于工作而不是安全方面，因此新的举措是有意义的。其他人则更多地认为 Gitlab 也可以选择（由其 IT 团队成员提供）更适合商业用途的计算机。</p><blockquote><p data-vmark="68f9">“我们目前唯一获得批准的 Linux 笔记本电脑供应商是戴尔。这些笔记本电脑通常预装 Ubuntu Linux，以节省未使用的 Windows 许可证的费用。戴尔目前不在澳大利亚和新西兰销售预装 Linux 的笔记本电脑；员工必须自己安装 Linux。”</p></blockquote><p data-vmark="0a04">GitLab 批准其员工使用 Mac 或 Linux，包括笔记本电脑的使用配置情况等。</p><blockquote><p data-vmark="0133"><strong>苹果硬件</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="a46c">Macbook Pro 14 英寸和 16 英寸- M1 Max / 10 核 CPU / 24 核 GPU / 32GB 统一内存 / 512GB 或 1TB 存储。性能型号</p></li><li><p data-vmark="525d">MacBook Pro 16 英寸 - Intel i7 或 i9 / 512 SSD / 32 内存 - 传统性能型号 - 供应有限；我们可能会根据新 M1 Max 型号的可用性提供此型号。</p></li><li><p data-vmark="c173">Macbook Pro - 13 英寸 / M1 / 16GB 内存 / 512GB - 标准型号</p></li></ul><p data-vmark="c300"><strong>注意：</strong>大多数需要更高性能设备的角色都被批准使用 14 英寸或 16 英寸 MacBook Pro M1 Max。</p><p data-vmark="0c71"><strong>Linux 硬件</strong></p><p data-vmark="f91e">IT 强烈鼓励团队成员选择 Mac；如果你有 Linux 方面经验并且能够自我支持，请仅申请 Linux 笔记本电脑。</p><ul class=" list-paddingleft-2"><li><p data-vmark="1ac9">工程师、支持工程师、数据分析师、技术营销经理、产品设计师、UX 经理、产品经理、技术和数字生产人员符合申请戴尔 Precision 移动工作站 5500 或 5700 系列笔记本电脑的资格。由于供应限制，这些提供的特定型号可能会有所不同。IT 将与大家一起寻找满足以下规格的可用型号：15.6 英寸显示屏 / 512GB SSD / 32GB 内存 / Intel i9 或 i7 CPU。</p></li><li><p data-vmark="e47a">其他所有人都有资格使用戴尔 Latitude 7300 系列笔记本电脑。由于供应限制，这些提供的特定型号可能会有所不同。IT 将与大家一起寻找满足以下规格的可用型号：13.3 英寸显示屏 / 256GB SSD / 16GB 内存 / Intel Quad-Core i5 CPU</p></li></ul><p data-vmark="934c"><strong>注意：</strong>Linux 笔记本电脑最高价格不超过同等 MacBook Pro  16 英寸笔记本电脑价格。</p></blockquote><p data-vmark="e0c8">当前的发展还提到 Gitlab IT 团队成员在 Linux 主机内使用 Windows 虚拟机的能力。</p><p data-vmark="f11a">实际上，Gitlab 作出的决定并不新鲜。谷歌过去以同样的方式面向 macOS 和 Linux 开放。</p><blockquote><p data-vmark="c479">“在 2010 年黑客攻击后，我们从 Windows PC 切换到 macOS 操作系统，”一位经理说，并补充“员工可以选择使用 Linux 作为他们的计算机操作系统。”</p></blockquote><p data-vmark="f56a">在网络浏览器方面，目前也有不同的观点，但基本上都是基于 Chrome、Chromium 和 Firefox 的浏览器。浏览器的问题是，例如，Chrome 在不同的操作系统上显示的菜单项不同。此外，某些选择样式规则的解释取决于浏览器运行的操作系统平台。</p><p style="text-align: center;" data-vmark="e68e"><img src="https://img.ithome.com/newsuploadfiles/2022/8/92e7b9a8-7ca4-4c9e-b9fe-43994d19462a.png" w="1399" h="1266" title="Gitlab 禁止员工使用微软 Windows，建议使用苹果 Macbook Pro 或戴尔 Linux 笔记本电脑" width="1399" height="742" referrerpolicy="no-referrer"></p>
