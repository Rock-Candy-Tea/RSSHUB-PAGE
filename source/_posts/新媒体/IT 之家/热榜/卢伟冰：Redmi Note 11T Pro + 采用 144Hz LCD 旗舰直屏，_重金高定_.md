<p data-vmark="646b"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 20 日消息，今日上午，小米集团合伙人、中国区、国际部总裁、Redmi 品牌总经理卢伟冰通过社交媒体表示，Redmi Note 11T Pro+“重 / 金 / 高 / 定”，打造“LCD 旗舰直屏”。</p><p style="text-align: center;" data-vmark="b495"><img src="https://img.ithome.com/newsuploadfiles/2022/5/6516e22e-e973-4eab-950e-6a7efac07be9.jpg" w="690" h="920" alt="Redmi Note 11T Pro +官方海报" title="卢伟冰：Redmi Note 11T Pro + 采用 144Hz LCD 旗舰直屏，“重金高定”" width="690" height="920" referrerpolicy="no-referrer"></p><p data-vmark="4b4c">据卢伟冰透露，Redmi Note 11T Pro + 屏幕<strong>支持 144Hz 七挡变速与全程 DC 调光</strong>，拥有定制比例、定制面板，带来高配色准、高配色温、高配体验。</p><p style="text-align: center;" data-vmark="b940"><img src="https://img.ithome.com/newsuploadfiles/2022/5/02b3b9dd-4516-4298-b19d-613546f38639.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzE0Mi5wbmc=,g_9,x_14,y_14,a_0,t_100" w="925" h="857" alt="卢伟冰微博" title="卢伟冰：Redmi Note 11T Pro + 采用 144Hz LCD 旗舰直屏，“重金高定”" width="925" height="760" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="821d"><img src="https://wx1.sinaimg.cn/mw690/003itXZnly1h2e49oqe2tj60u0140drd02.jpg" alt="Redmi Note 11T Pro +官方海报" class="lazy" title="卢伟冰：Redmi Note 11T Pro + 采用 144Hz LCD 旗舰直屏，“重金高定”" referrerpolicy="no-referrer"></p><p data-vmark="2ce8">IT之家了解到，<strong>Redmi Note 11T 系列将于 5 月 24 日发布</strong>，新机定位“性能小金刚”。根据此前信息，Redmi Note 11T 系列大杯起步，<strong>提供 Pro、Pro + 两款机型</strong>。</p><p style="text-align: center;" data-vmark="d660"><img src="https://img.ithome.com/newsuploadfiles/2022/5/e00a5cd3-4c92-4478-862d-b0974f558feb.jpg@s_2,w_820,h_1093" alt="Redmi Note 11T Pro +官方海报" title="卢伟冰：Redmi Note 11T Pro + 采用 144Hz LCD 旗舰直屏，“重金高定”" referrerpolicy="no-referrer"></p><p data-vmark="101d">其中，Redmi Note 11T Pro + 将<strong>搭载天玑 8100 芯片</strong>，配备 3.5mm 耳机孔，后置三摄。天玑 8100 采用台积电 5nm 制程，其八核 CPU 架构由 4 个 2.85GHz Cortex-A78 核心和 4 个 Cortex-A55 能效核心组成，搭配六核 Arm Mali-G610。</p>
