<p data-vmark="d03c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 7 日消息，在 2021 年 9 月，也就是 Windows 11 正式发布前一个月，微软解释了新的操作系统设计能够充分利用硬件。在 2021 年 11 月底，微软公司承诺，到 2022 年，Windows 11 将比当时更快。</p><p style="text-align: center;" data-vmark="9907"><img src="https://img.ithome.com/newsuploadfiles/2022/7/e8945ec5-645d-4124-9654-a8933f91e3b4.jpg" w="820" h="547" title="最新测试：微软 Win11 性能速度已赶上 Win10，前者运行虚幻引擎优势巨大" width="820" height="547" referrerpolicy="no-referrer"></p><p data-vmark="85d4">看起来微软并没有夸大其词，定制计算机和工作站供应商 Puget Systems 的最新测试数据发现，<span class="accentTextColor">Windows 11 性能现在确实比以前更快，尽管 Windows 11 与 Windows 10 相比还没有明显优势，两者似乎在硬件性能方面互有胜负。</span></p><p data-vmark="cfa2">Puget Systems 高级实验室技术员 Matt Bach 写道：</p><blockquote><p data-vmark="8b5c">“上次我们在视频编辑、渲染和游戏开发方面将 Windows 11 与 Windows 10 进行了比较，我们发现 Windows 10 通常能够提供比 Windows 11 更高的性能。然而，在 Windows 11 正式版发布之后，并且从那时起，微软推出了大量更新，而且全新一代 CPU 也推出了。</p><p data-vmark="51b9">好消息是，这次我们发现 Windows 11 没有出现太多的性能问题，而且此前发现的许多问题似乎都已经得到解决。另一方面，在大多数情况下，Windows 11 也没有明显胜出 Windows 10 。</p><p data-vmark="199d">[..] 总体而言，只有在少数情况下，使用一种操作系统比另一种操作系统具有明显的性能优势：</p><ul class=" list-paddingleft-2"><li><p data-vmark="615f">Premiere Pro - 在任何情况下，Windows 10 速度都稍快于 Windows 11，有一定的明显差异。</p></li><li><p data-vmark="c30c">V-Ray CUDA 模式 - Windows 11 始终比 Windows 10 快约 20%。</p></li><li><p data-vmark="db62">虚幻引擎 - 可能是 Windows 10 的错误 Bug，英特尔酷睿 i9-12900K 在 Windows 11 中显示出巨大的性能提升。”</p></li></ul></blockquote><p data-vmark="d6f9">Puget 指出虚幻引擎 4.26 版本中的 Bug 似乎影响到了 Intel Alder Lake-S i9-12900K 处理器的性能表现。也有可能是 Intel 芯片的高性能混合架构在 Windows10 上没有针对 UE 4.26 进行充分优化，这也是 Windows 10 和 Windows 11 系统存在巨大差异的因素之一：</p><p style="text-align: center;" data-vmark="59cf"><img src="https://img.ithome.com/newsuploadfiles/2022/7/a6920ab0-fe1b-4a7f-82a6-1945e6cd5d59.jpg@s_2,w_820,h_643" w="1196" h="938" title="最新测试：微软 Win11 性能速度已赶上 Win10，前者运行虚幻引擎优势巨大" srcset="https://img.ithome.com/newsuploadfiles/2022/7/a6920ab0-fe1b-4a7f-82a6-1945e6cd5d59.jpg 2x" width="1196" height="643" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="d9b9"><img src="https://img.ithome.com/newsuploadfiles/2022/7/ea1854ee-8c7d-487b-8e3a-8e015a307918.jpg@s_2,w_820,h_643" w="1196" h="938" title="最新测试：微软 Win11 性能速度已赶上 Win10，前者运行虚幻引擎优势巨大" srcset="https://img.ithome.com/newsuploadfiles/2022/7/ea1854ee-8c7d-487b-8e3a-8e015a307918.jpg 2x" width="1196" height="643" referrerpolicy="no-referrer"></p>
