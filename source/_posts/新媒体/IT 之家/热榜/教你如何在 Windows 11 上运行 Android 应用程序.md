<p data-vmark="491e"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 21 日消息，微软今天宣布在 Windows 上为预览用户提供“<a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> Subsystem for Windows”的第一个预览版。美国地区的小伙伴可通过在运行 Intel、AMD 和 Qualcomm 平台且符合条件的设备上加入 Beta Channel 体验新功能。</p><p data-vmark="7bdd">随后 ，微软为大家提供了一些更详细的说明以帮助用户了解更多有关于在 Windows 上运行<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a> App 的方法。</p><p data-vmark="3738">需要注意的是，适用于 Android 的 Windows 子系统目前只能通过 Windows 预览体验计划中的预览获得。您的设备还必须满足特定的 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 要求。</p><p data-vmark="c5b8">IT之家了解到，要运行这些移动应用程序，您需要 Windows 11 版本 21H2 Build 22000 .xxx 或更高版本，并将地区设置为美国。</p><p data-vmark="766c"><img src="https://img.ithome.com/newsuploadfiles/2021/10/db91e667-c1be-4d48-b495-9ed331e4406a.png" w="1440" h="810" title="教你如何在 Windows 11 上运行 Android 应用程序" width="1440" height="461" referrerpolicy="no-referrer"></p><section><h2 data-vmark="9a33">设备要求</h2><p data-vmark="e4df">要查看您的 PC 所需的最低要求，首先需要满足 Windows 11 <a href="https://www.microsoft.com/zh-cn/windows/windows-11-specifications" target="_blank">规格</a>和系统要求，然后还需要满足 Windows 11 部分功能的特定要求。</p></section><section><h2 data-vmark="b003">安装 Android 子系统</h2><p data-vmark="62d3">有两种方式可以安装 Android 子系统：</p><ul class=" list-paddingleft-2"><li><p data-vmark="87e6">打开 Microsoft Store 应用程序并搜索 <span class="ocpLegacyBold">Amazon Appstore</span>。选择 <span class="ocpLegacyBold">获取</span> 并安装它。在安装过程中，适用于 Android 的 Windows 子系统将安装在您的 PC 上。</p></li><li><p data-vmark="c5a4">在 Microsoft Store 上搜索移动应用。Amazon Appstore 和适用于 Android 的 Windows 子系统将在安装移动应用程序之前安装。</p></li></ul><p data-vmark="4309">安装完成后，Windows Subsystem for Android Settings 应用程序和 Amazon Appstore 将出现在“开始”菜单和您的应用程序列表中。</p></section><section><h2 data-vmark="74de">安装移动应用</h2><p data-vmark="cbc5">从亚马逊应用商店下载移动应用程序需要亚马逊账户。准备就绪后，打开 Amazon Appstore 并使用您的 Amazon 帐户登录。</p><p data-vmark="8522">安装移动 App 后，您可以直接从“开始”菜单和应用程序列表中打开运行。</p></section><section><h2 data-vmark="8d29">访问子系统中的 Windows 文件</h2><p data-vmark="d405">该子系统使用独立于 Windows 的文件系统 —— 这些文件无法传输到 Windows。此外，无法在子系统和任何移动应用程序中访问 Windows 中可用的文件。如果需要，以下是查看子系统文件的方法：</p><ul class=" list-paddingleft-2"><li><p data-vmark="2025">打开 <span class="ocpLegacyBold">适用于 Android 设置</span> 应用程序的<span class="ocpLegacyBold"> Windows 子系统</span>。</p></li><li><p data-vmark="d138">选择 <span class="ocpLegacyBold">文件</span>。</p></li><li><p data-vmark="e1cf">您在子系统中的文件将打开，您可以浏览保存在子系统中的任何图像、视频、音频、文档和下载。</p></li></ul></section><section><h2 data-vmark="a373">移动应用程序的操控</h2><p data-vmark="8df6">在您的 PC 上浏览移动应用程序是一种不同的体验。以下是一些有关日常使用的提示：</p><ul class=" list-paddingleft-2"><li><p data-vmark="eacc">使用鼠标上的滚轮或触控板手势滚动浏览应用程序。您还可以使用鼠标或触控板在应用程序中单击并向上、向下、向左和向右拖动。</p></li><li><p data-vmark="1742">选择移动应用程序标题栏中的后退按钮可返回上一页。</p></li><li><p data-vmark="b66d">一些游戏支持通过方向键走动，有些可能还支持鼠标来移动。</p></li></ul><header class="ocpArticleTitleSection"><h2 data-vmark="a81c">适用于 Android 子系统的辅助功能</h2></header><section><p data-vmark="a9c5">适用于 Android 的 Windows 子系统中使用的应用需要额外的屏幕阅读器。Windows 中的大多数辅助功能可应用于安卓子系统中的 App，例如文本缩放、隐藏式字幕、夜间模式深色主题和滤色器。以下是您还需要了解的内容。</p></section><article class="ocpArticleContent"><section><h3 data-vmark="5685">使用子系统屏幕阅读器<br></h3><section><p data-vmark="a924">有两种方法可以打开子系统屏幕阅读器</p><ul class=" list-paddingleft-2"><li><p data-vmark="d496">选择“开始”，输入“适用于 Android 的 Windows 子系统”，然后从结果中选择应用程序。确保子系统屏幕阅读器已打开。</p></li><li><p data-vmark="1aed">在键盘上，按 Windows 徽标键 + Ctrl  + T 打开或关闭子系统屏幕阅读器。您可以在子系统中使用应用程序时随时执行此操作。</p></li></ul></section><section><h3 data-vmark="cb04">切换应用 </h3><p data-vmark="9fc7">当您从 Windows 应用程序切换到移动应用程序时，请注意当前的 Windows 屏幕阅读器窗口的标题。当它宣布 Subsystem Screen Reader 时，按键盘快捷键（Windows 徽标键 + CTRL + T）可切换到它。</p><p data-vmark="2ee1">有关键盘快捷键列表，请参阅<a href="https://support.microsoft.com/zh-cn/windows/windows-subsystem-for-android-subsystem-screen-reader-keyboard-shortcuts-f18a072e-22dd-4c8b-aa25-c6562bb409bc" target="_blank">适用于 Android 子系统屏幕阅读器键盘快捷键</a>。</p><p data-vmark="3ff6">注意： 当您使用子系统屏幕阅读器时，触摸输入将无法使用。</p></section></section><section><h3 data-vmark="4572">使用放大镜</h3><p data-vmark="107e">如果您使用 Windows 的放大镜时仍会看到放大的缩放区域，当前不支持文本光标放大。</p><h2 data-vmark="c39d">微软收集的数据</h2><section><p data-vmark="fe6a">当您使用安装在子系统中的移动应用程序时，Microsoft 可以收集应用程序崩溃、有关子系统的性能数据以及您的应用程序使用情况。</p><p data-vmark="3588">该数据用于提高子系统的可靠性和性能以及子系统中运行的应用程序的性能。您可以选择是否允许 Microsoft 在您首次安装适用于 Android 的 Windows 子系统时或在适用于 Android 的 Windows 子系统设置应用程序中的任何时间收集这些数据。</p></section><section><h2 data-vmark="7d9c">与开发人员分享数据</h2><p data-vmark="40ac">Microsoft 不会直接与应用程序开发人员共享它收集的任何数据。</p></section><section><h2 data-vmark="a187">与亚马逊分享数据</h2><p data-vmark="2a62">如果您选择允许 Microsoft 收集数据，Microsoft 可以与 Amazon 共享聚合和匿名的应用程序使用数据以及应用程序故障转储信息。</p></section><section><h2 data-vmark="8867">更改您的数据共享方式</h2><p data-vmark="d6fa">如果您想更改数据的共享方式，可执行以下操作：</p><ul class=" list-paddingleft-2"><li><p data-vmark="4e38">选择开始 并输入 Windows Subsystem for Android，然后从结果中打开应用程序。</p></li><li><p data-vmark="07b6">在设置应用程序中，确保 关闭发送可选诊断数据。</p></li></ul></section><section><h2 data-vmark="095b">移动应用程序权限</h2><p data-vmark="f6e4">您可以从 Amazon Appstore 为每个移动应用程序设置权限。以下是调整应用权限和设置的方法：</p><ul class=" list-paddingleft-2"><li><p data-vmark="160d">选择开始 > 设置  > 应用和功能，找到您要调整的移动应用，然后选择 修改。</p></li><li><p data-vmark="ec4d">该应用程序的设置将在适用于 Android 的 Windows 子系统中打开，并允许您调整相机、位置和通知等功能的隐私设置。</p></li></ul></section><h2 data-vmark="b261">对 Windows 上的移动应用程序进行故障排除</h2><h3 data-vmark="4d1b">重启大法</h3><p data-vmark="3ed5">大多数问题将通过重新启动安卓子系统来解决。</p><ul class=" list-paddingleft-2"><li><p data-vmark="cbf8">开始，搜索适用于 Android 的 Windows 子系统，然后打开。</p></li><li><p data-vmark="6692">在这个应用程序中，选择 Shutdown Subsystem。</p></li><li><p data-vmark="3002">打开任何移动应用程序以自动重启子系统。</p></li></ul><h3 data-vmark="e5b5">检查设置</h3><p data-vmark="4e19">适用于 Android 设置的 Windows 子系统应用程序可通过调整内存使用情况、屏幕分辨率、可调整大小的窗口等提高性能：</p><ul class=" list-paddingleft-2"><li><p data-vmark="9dec">子系统屏幕阅读器</p></li><ul class=" list-paddingleft-2"><li><p data-vmark="1cd4">如果触摸输入不起作用，请确保子系统屏幕阅读器已关闭。</p></li></ul><li><p data-vmark="8279">子系统资源：</p></li></ul><p data-vmark="ccec">选择此项后，将在打开移动应用程序时打开子系统。由于子系统需要先打开，因此移动应用程序可能需要更长的时间才能打开。在第一个之后打开的移动应用程序可能不受影响。</p><p data-vmark="14f3">子系统随时准备打开应用程序。由于它始终在后台打开，因此会占用更多 PC 的内存和处理能力。</p><h3 data-vmark="ae6c">卸载</h3><p data-vmark="1374">卸载子系统时，您还将卸载 Amazon Appstore 和 PC 上安装的所有移动应用，而且所有直接保存到 PC 子系统中的文件也将被删除。如果你备份完之后，请按以下步骤进行卸载：</p><ul class=" list-paddingleft-2"><li><p data-vmark="f104">选择开始 > 应用程序 > 应用程序和功能。</p></li><li><p data-vmark="ca91">查找 适用于 Android 的 Windows 子系统，选择 更多 > 卸载。卸载子系统还将卸载您 PC 上的 Amazon Appstore 和所有移动应用程序。</p></li></ul><p data-vmark="deca">卸载子系统时，请注意以下几点：</p><ul class=" list-paddingleft-2"><li><p data-vmark="91af">卸载 Amazon Appstore 也将卸载适用于 Android 的 Windows 子系统。所有移动应用程序都将从您的 PC 中删除。</p></li><li><p data-vmark="82fb">卸载移动应用程序只会删除该应用程序以及与其相关的任何数据。</p></li></ul><h3 data-vmark="abec">反馈</h3><p data-vmark="72e7">如果子系统或移动应用程序的问题无法解决，您可以向微软反馈以帮助改善：</p><ul class=" list-paddingleft-2"><li><p
