<p data-vmark="a916"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 13 日消息，微博博主 @数码闲聊站 于 10 月 11 日透露，红米 Redmi 的 MIUI 系统的部分 UI 界面，可能要和小米做一些细节上的小区分，<span class="accentTextColor">主要是增添自己品牌的元素</span>。这一界面变化将不会十分明显，目的是提高 Redmi 品牌的辨识度。</p><p data-vmark="591a">目前多家国内智能手机品牌都有自己的子品牌，针对不同的价位段和用户群体。IT之家了解到，拆分前华为与荣耀子品牌的系统 UI 界面有着不同的设计，同时 iQOO 与 vivo 品牌的系统 UI 界面也不完全相同。</p><p data-vmark="9cd6"><img src="https://img.ithome.com/newsuploadfiles/2021/10/620538cd-ed21-4eec-bc97-a8e4a8393396.png" w="802" h="391" title="爆料：红米 Redmi 部分 MIUI UI 界面将与小米区分，提高品牌辨识度" width="802" height="391" referrerpolicy="no-referrer"></p><p data-vmark="f348">爆料者在微博还表示，红米 Redmi 新款手机将采用高通骁龙 870 SoC，<span class="accentTextColor">项目正常推进中</span>。此外，他还透露 10 月份将有一款“越级”的中端机即将发布，品牌未知。</p><p data-vmark="112f">目前小米最新的手机系统为 MIUI 12.5 增强版，口号为“触碰想象，感受真实”。该系统具有光锥动效设计，众多系统 App 的图标，在操作时会有细微的动画效果。此外，系统的触控手感也有拟真设计，并开放兼容第三方应用。</p><p data-vmark="67cc"><img src="https://img.ithome.com/newsuploadfiles/2021/10/78bf5b9f-36a6-4f82-9454-57035ee8c25a.png" w="1260" h="628" title="爆料：红米 Redmi 部分 MIUI UI 界面将与小米区分，提高品牌辨识度" width="1260" height="409" referrerpolicy="no-referrer"></p><p data-vmark="35c5"><img src="https://img.ithome.com/newsuploadfiles/2021/10/f2d64d50-103a-4f73-98e4-60df41107a9b.png" w="747" h="751" title="爆料：红米 Redmi 部分 MIUI UI 界面将与小米区分，提高品牌辨识度" width="747" height="751" referrerpolicy="no-referrer"></p>
