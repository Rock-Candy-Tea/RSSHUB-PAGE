<p data-vmark="76b9"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 22 日消息，华为开发者大会 2021（Together）于今天正式开幕，华为在主题演讲中正式发布全新的 HMS Core 6，向全球开发者开放 7 大领域的 69 个 Kit 和 21738 个 API，包括 13 个跨操作系统能力。</p><p data-vmark="6889"><img src="https://img.ithome.com/newsuploadfiles/2021/10/58bbf85f-220e-4172-afba-785bc4ca1a02.jpg@s_2,w_820,h_273" w="1440" h="480" title="华为发布全新 HMS Core 6，支持 Windows、iOS 等更多操作系统" srcset="https://img.ithome.com/newsuploadfiles/2021/10/58bbf85f-220e-4172-afba-785bc4ca1a02.jpg 2x" width="1440" height="273" referrerpolicy="no-referrer"></p><p data-vmark="d9c5">IT之家了解到，华为消费者业务云服务总裁、华为云 CEO 张平安表示，HMS 生态目前拥有 510 万开发者，HMS Core 全球应用集成数增长 81% 达 17.3 万，全球开发者收入增长 62%。</p><p data-vmark="4bd3"><img src="https://img.ithome.com/newsuploadfiles/2021/10/0ff7e97b-e6f5-430d-90fe-6e5eb7e5b230.jpg@s_2,w_820,h_273" w="1440" h="480" title="华为发布全新 HMS Core 6，支持 Windows、iOS 等更多操作系统" width="1440" height="273" referrerpolicy="no-referrer"></p><p data-vmark="19fb">数据显示，<span class="accentTextColor">全球 7.3 亿用户正在使用华为 AppGallery</span>。2021 年前三季度应用分发量达 3322 亿，同比增长 13%；2021 年前三季度快应用分发量达 115 亿，同比增长 188%。</p><p data-vmark="0ab4"><img src="https://img.ithome.com/newsuploadfiles/2021/10/33759f7d-f5bd-46cd-b736-7e5ce8e799ad.jpg@s_2,w_820,h_273" w="1440" h="480" title="华为发布全新 HMS Core 6，支持 Windows、iOS 等更多操作系统" width="1440" height="273" referrerpolicy="no-referrer"></p><p data-vmark="0427">张平安表示，本次发布的全新多终端、跨 OS、全场景的 HMS Core 6，新增开放能力 13 个，跨操作系统开放 kit 13 个，能力增强 18 个。</p><p data-vmark="4899">据介绍，本次升级 HMS Core 6 不仅支持 <a class="s_tag" href="https://hmos.ithome.com/" target="_blank">HarmonyOS</a> 和<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>，还可以支持其它更多操作系统，<span class="accentTextColor">如 OpenHarmony、Windows、iOS 等</span>，可在平板、智慧屏、智能手表及车机等多设备调用。HMS Core 还支持第三方开发框架，如 React Native、Cordova 等。</p><p data-vmark="9fa7"><img src="https://img.ithome.com/newsuploadfiles/2021/10/f2b1743a-0461-4d23-b48c-9714bddcf303.jpg@s_2,w_820,h_273" w="1440" h="480" title="华为发布全新 HMS Core 6，支持 Windows、iOS 等更多操作系统" srcset="https://img.ithome.com/newsuploadfiles/2021/10/f2b1743a-0461-4d23-b48c-9714bddcf303.jpg 2x" width="1440" height="273" referrerpolicy="no-referrer"></p><p data-vmark="f6cf">在图形领域，HMS Core 6 新增 3D 建模服务，并增强了 AR Engine、图形计算服务、图形引擎服务等多项开放能力。</p><p data-vmark="193c">在媒体领域，基于 AI 技术的加持，华为全新开放了视频编辑服务（Video Editor Kit）和音频编辑服务（Audio Editor Kit），并能提供覆盖 70 + 种语言的全球化服务，支持全球调用。</p><p data-vmark="9b18">在人工智能领域，Machine Learning Kit 作为已开放的能力，有将近一万个应用集成了 ML Kit，月调用量超过 82.5 亿次，现已扩展到其他操作系统应用上。</p><p data-vmark="f72b">在应用服务领域，HMS Core 6 新增和增强的开放能力达到 13 项，跨 OS 能力 12 项。</p><p data-vmark="ebab">在系统领域，华为将核心的连接与通信能力开放出来，包含 9 项能力，通过 Wireless Kit、Network Kit 开放给应用开发者。</p><p data-vmark="3c1b"><img src="https://img.ithome.com/newsuploadfiles/2021/10/fe8fc5cd-24ed-4614-8d87-33842704efa6.jpg@s_2,w_820,h_273" w="1440" h="480" title="华为发布全新 HMS Core 6，支持 Windows、iOS 等更多操作系统"
