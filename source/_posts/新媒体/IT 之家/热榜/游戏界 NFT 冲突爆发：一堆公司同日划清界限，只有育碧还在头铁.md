<p data-vmark="1a63">游戏行业发生一件怪事：一群开发商突然排队表态与 NFT 做切割。</p><p data-vmark="ad69">先是一家工作室公开谴责《百战天虫》开发商 Team17 的 NFT 计划，并终止合作关系，此举收获网友 10 万点赞。</p><p data-vmark="4354">随后两家游戏公司承诺永不涉足 NFT 的声明也收到大量点赞，其中包括《胡闹厨房》开发商。</p><p data-vmark="1dcd" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/24fee63b-dced-48e4-8311-102241d60702.png" w="1080" h="1002" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="1080" height="761" referrerpolicy="no-referrer"></p><p data-vmark="d0d9">压力之下，被谴责的 Team17 也终于宣布放弃 NFT 计划。</p><blockquote><p data-vmark="a329">我们听取了团队成员、合作伙伴和游戏社区的意见，因此决定退出 NFT 领域。</p></blockquote><p data-vmark="396d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/2fccf460-c50f-40da-95f6-23b37447b529.png" w="1080" h="315" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="1080" height="239" referrerpolicy="no-referrer"></p><p data-vmark="b6d3">不止这些小型公司，就连游戏巨头 EA 公司的 CEO，在最新一次电话会议上也表达了对 NFT 的观望和犹豫。</p><blockquote><p data-vmark="8d09">我相信，收藏价值将继续成为游戏体验的重要组成部分。但是否需要作为 NFT 区块链的一部分还有待观察。</p></blockquote><p data-vmark="87a8">要知道，同一位 CEO 可是在上一次电话会议上说过“NFT 和区块链游戏是行业的未来”这种话。</p><p data-vmark="bc98">NFT 即非同质化代币，一些游戏公司希望引入这项技术，让玩家可以在区块链上买卖游戏道具。</p><p data-vmark="3366">近几个月对区块链原生游戏和 NFT 相关游戏公司的投资正在大幅增加，育碧、SE、卡普空等传统游戏公司也纷纷开始布局。</p><p data-vmark="dd81">怎么一夜之间，香饽饽突然不香了？</p><h2 data-vmark="340c">玩家疯狂点踩</h2><p data-vmark="d82b">与区块链原生游戏社区不同，传统游戏社区的玩家会把怒火烧向任何想涉足 NFT 的游戏公司。</p><p data-vmark="dd8a">这次的《百战天虫》开发商 Team17 是在宣布 NFT 计划 24 小时之内，就被玩家 + 合作伙伴一起骂到不得不取消。</p><p data-vmark="afd0">网友们还开辟了一个新话题标签#TeamNoNFT#来集中表达不满。</p><p data-vmark="3fdf" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/fcd3f9c1-03c6-448b-9bdf-4e35ea9c09d7.png" w="1080" h="358" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="1080" height="272" referrerpolicy="no-referrer"></p><p data-vmark="96d8">再往前数，《潜行者》系列的乌克兰开发商坚持时间更短，仅仅 90 分钟之后就取消了新作中的计划。</p><p data-vmark="9742" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/967f01c2-354c-4aaa-8a99-35e80fa71dfd.png" w="1080" h="1214" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="1080" height="922" referrerpolicy="no-referrer"></p><p data-vmark="8cf0">除了游戏公司以外，游戏配音演员要搞 NFT，玩家也不答应。</p><p data-vmark="7847">曾出演《生化奇兵无限》《最后生还者》等众多大作的动捕配音一哥 Troy Baker，半月前宣布与 VoiceVerseNFT 合作探索声音 NFT 新项目。</p><p data-vmark="126f">还说了“你可以选择去痛恨，也可以选择去创造”这样容易引起误解的话。</p><p data-vmark="7a75" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/66cde7e2-04a4-4fbf-83d8-1a6cf3e2beae.png" w="496" h="166" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="496" height="166" referrerpolicy="no-referrer"></p><p data-vmark="27e5">结果评论区里，点赞靠前的都是选择痛恨的。</p><p data-vmark="97e1" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/2d9e1114-3431-4b6c-94a9-78eac76dd079.png" w="1048" h="1268" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="1048" height="992" referrerpolicy="no-referrer"></p><p data-vmark="207a">近日，他也宣布了终止与区块链公司 VoiceVerseNFT 的合作关系。</p><p data-vmark="5dbe" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/f6211751-ec31-4514-8d03-ff15e6bc45c4.png" w="1080" h="291" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="1080" height="221" referrerpolicy="no-referrer"></p><p data-vmark="b9d3">为什么游戏玩家会如此痛恨 NFT？</p><p data-vmark="8536">海量批评声音中可以看到以下几种观点。</p><p data-vmark="2613">出于兴趣玩游戏的人认为，引入 NFT 会让游戏变味。为了赚钱而玩，会让游戏不再有趣，变成另一种“打工”。</p><p data-vmark="469f">比如育碧公司在《幽灵行动：断点》中发行的 NFT 道具可以按游戏等级和游玩时间领取。</p><p data-vmark="1062">其中一款限量的头盔皮肤需要玩上 600 小时才能解锁。每天雷打不动玩 2 小时也要玩 10 个月才能获得。</p><p data-vmark="81d6" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/f85f456d-5a6d-4398-b90e-5494c3a59e8a.png" w="959" h="534" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="959" height="457" referrerpolicy="no-referrer"></p><p data-vmark="da91">最终育碧 NFT 平台 Quartz 的宣传视频在油管上的点踩率一度到达 96% 破了历史记录，这条视频也被隐藏起来。</p><p data-vmark="d05a">区块链对环境的破坏，以及金融诈骗猖獗也是不少玩家反感之处。</p><p data-vmark="3319">有人表示“我不愿意用毁掉整整一片森林的代价来换取游戏里一个道具”。</p><p data-vmark="8418">Aggro Crab Games 工作室公开谴责 Team17 的主要原因也是基于此。</p><blockquote><p data-vmark="1350">我们相信 NFT 不可能做到环境友好，或有价值，而且整个就是个骗局。</p></blockquote><p data-vmark="e1e7">更何况，挖矿热正是很多玩家认为他们买不到显卡的“罪魁祸首”。</p><p data-vmark="57dc">即使是喜欢收藏和道具交易的玩家也不欢迎 NFT。</p><p data-vmark="d1a3">在《CS：GO》等游戏中，道具交易系统不靠任何区块链技术也可以运转的很好。</p><p data-vmark="caef" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/5c81208a-435b-4bb4-9f4c-9b2f8faa4967.png" w="1080" h="493" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="1080" height="374" referrerpolicy="no-referrer"></p><p data-vmark="bb07">除了玩家以外，个人游戏开发者和独立游戏工作室总体上来说也对 NFT 不感冒。</p><p data-vmark="747f">在最近一次全球游戏开发者大会 GDC 上作了相关调查，70% 的开发者表示对 NFT 没有兴趣。</p><p data-vmark="4335" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/377f4a25-5b16-446e-9eee-646097e751da.png" w="1080" h="300" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="1080" height="228" referrerpolicy="no-referrer"></p><h2 data-vmark="04d5">NFT 与手游相性更高？</h2><p data-vmark="baba">与育碧 NFT 平台 Quartz 无人问津，发行一个月销量不到 1% 还受玩家强烈抵制不同。</p><p data-vmark="5a42">手机游戏公司，特别是亚洲手游公司进军 NFT 取得了不少成功。</p><p data-vmark="26b1">早些时候，日本公司 Square Enix 发行的《资产性百万亚瑟王》受到市场欢迎，前两期全部售罄。</p><p data-vmark="307b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/0f981120-1c4c-4a6e-a574-e0db60bcddff.png" w="750" h="750" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="750" height="750" referrerpolicy="no-referrer"></p><p data-vmark="3e42">同样日本公司卡普空的《街头霸王》NFT 交易卡牌在 24 小时内全部售罄，销售额超 200 万美元，在二手市场上交易量又增加了 100 万美元。</p><p data-vmark="b286" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/be2bf863-1bbc-41ab-b0df-f2d76780e674.png" w="1080" h="356" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="1080" height="270" referrerpolicy="no-referrer"></p><p data-vmark="1a55">字节跳动收购的中国游戏公司沐瞳科技，在海外发售的《无尽对决》NFT 盲盒，总计 2.5 万个每个 25 美元，2 分钟之内就被抢购一空。</p><p data-vmark="fdf3" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/2889e18d-5520-45a2-9fa8-06e0c2a5257b.png" w="1080" h="608" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="1080" height="462" referrerpolicy="no-referrer"></p><p data-vmark="a6b6">最新的消息是，韩国手游第一大厂 Netmarble 宣布计划推出 20 款新游戏，其中 70% 将使用区块链技术。</p><p data-vmark="ae59" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/99592d28-df85-46ae-8864-4573e305ccb2.png" w="960" h="640" title="游戏界 NFT 冲突爆发：一堆公司同日划清界限，只有育碧还在头铁" width="960" height="547" referrerpolicy="no-referrer"></p><p data-vmark="e718">另一面，大型 PC 游戏和主机游戏平台对 NFT 的态度比较保守。</p><p data-vmark="5bb8">Steam 平台直接推出新规定，禁止 NFT 和区块链游戏上架。</p><p data-vmark="b5d2">微软 Xbox 负责人 Phil Spencer 也不看好，他认为“目前 NFT 游戏的剥削性大于娱乐性”。</p><p data-vmark="c01d">拥有《堡垒之夜》的 Epic Games 态度也比较谨慎，只允许 NFT 游戏在满足一定条件下上架旗下游戏商店。</p><p data-vmark="5749">头最铁的还是育碧，尽管受到大量抵制而且销量惨淡，育碧还是计划继续推进 NFT 项目。</p><p data-vmark="15d4">育碧战略创新实验室副总裁 Nicolas Pouard 在最近一次采访中表示：</p><blockquote><p data-vmark="3bf5">数字藏品二级市场对玩家有好处，只是他们现在还不明白。</p></blockquote>
