<p>微软已经公布了下一代 Windows 系统 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a>，不知不觉 Windows 系统也已经走过了三十多近四十个年头。此前微软曾经表示，<a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 将会是最后一代 Windows，但没想到这款身经百战的产品还是迭代了下去，这让很多老玩家都百感交集。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/baaea8b8-71cd-4507-99bb-f454359566b0.jpg" w="700" h="437" title="微软 Win11 都来了它们还活着，盘点老树常青的小软件" width="700" height="437" referrerpolicy="no-referrer"></p><p>Windows 已经成为了老将，而在 Windows 的麾下，也有很多伴随我们共同成长的小软件。即使迈入到了移动互联网时代，连 Windows 都已经经过重新设计，这些老软件们仍在发光发热，一如既往地为老玩家勤勤恳恳服务。</p><p>能一路走过时代的惊涛骇浪实属不易，在今时今日仍大有用武之地更是难得可贵。Windows 是如此，这些远没有 Windows 家大业大的小软件们，更是如此。今天，就让我们一起来盘点那些在 Win11 时代依然能老树常青的小软件们吧。</p><h2>Firefox 火狐浏览器</h2><p>Firefox <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>算不算是小软件？或许这是有争议的。浏览器是个极其庞杂的系统工程，涉及的软硬件知识面极广，但尽管如此，Firefox 的安装包体积至今也保持在 50 多 M。对比起现在动不动占数百 M 乃至上 G 的聊天软件，Firefox 作为拥有自己独立渲染引擎的浏览器，这不可谓不小。</p><p>体积虽小，但论资排辈起来，Firefox 就显得厚重多了，它的历史甚至可以追溯到上世纪 90 年代和 IE 掀起浏览器大战的网景浏览器。在微软祭出 Windows 捆绑 IE 这招后，网景浏览器落败，但网景遗留下了 Mozilla 开源项目及基金会，让火种得以延续。此后，网景最宝贵的技术和精神，都在 Mozilla 中大体延续了下去，Firefox 火狐浏览器即是 Mozilla 最弥足珍贵的产品。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/9c9ae63c-4c79-4183-9af3-31788ae0fc9e.jpg" w="700" h="525" title="微软 Win11 都来了它们还活着，盘点老树常青的小软件" width="700" height="525" referrerpolicy="no-referrer"></p><p>▲ Firefox 的经典版本，用过的都是老网友了，4.0 版本后 Firefox 开始大改，刷起版本号来了</p><p>千禧年后，Firefox 火狐浏览器曾一度再次走上和 IE 搏击的擂台，靠开源开放的力量，赢得了众多用户的心。时至今日，尽管 Firefox 被 Chrome 抢夺了不少风头，但在 Opera、Edge 等各大浏览器纷纷向 Chrome 投降、换用 Chromium 内核的今天，Firefox 仍坚持使用自主研发的 Gecko 引擎，这种不对商业软件低头、对自由软件的坚持与执着，这也让它始终拥有不少死忠粉。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/7360c144-edf8-464f-882e-a57136ffe894.jpg" w="700" h="453" title="微软 Win11 都来了它们还活着，盘点老树常青的小软件" width="700" height="453" referrerpolicy="no-referrer"></p><p>▲ 现在 Firefox 的版本号已经刷到了 90+，界面更加现代化</p><p>和 Windows 一样，Firefox 也历经了数次大改。速度更快、界面更现代的 Firefox，在 Win11 中又会展现出怎样的姿态？我们拭目以待。</p><h2>Foobar 2000</h2><p>谈起最好的音乐播放器，Foobar 2000 一定会获得提名。Fooobar 界面简单，但在 PC 上玩 HiFi 的发烧友都知道这简单的皮囊下是多么强大的架构。不过，却很少有人知道，看上去简简单单的 Foobar 2000 有多么源远流长。</p><p>如果你比较早接触电脑，一定听说过 Winamp 这个名字，在上世纪 90 年代，Winamp2 这是毫无争议的音乐播放器之王。但 Winamp 被 AOL 收购后，改版不当且运营失败，使得 Winamp 3 口碑大崩坏。原 Winamp 开发公司 Nullsoft 的成员 Peter Pawlowski 愤然出走，另起炉灶，其新作品就是另一大名鼎鼎的音乐播放器 Foobar2000—— 从某种意义来说，Foobar 2000 才是 Winamp 的精神续作。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/16f832d4-1966-4453-a26c-07d433dba3b6.jpg" w="548" h="234" title="微软 Win11 都来了它们还活着，盘点老树常青的小软件" width="548" height="234" referrerpolicy="no-referrer"></p><p>▲ Foobar 2000 的起源是经典的 Winamp 2</p><p>Foobar 2000 之所以能历经这么多年大风大浪的考验，和其开放架构带来极高的可玩性是密不可分的。Foobar2000 继承了 Winamp2 的大量优点，例如可自由更换皮肤定制 UI，支持丰富的 DSP 插件，通过插件可以轻而易举支持现在 HiFi 圈子里流行的高规格 PCM 音频和 DSD 音频，也可以支持在线音乐和歌词服务。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/43c1fb27-91e2-4644-99bf-e32dad8fa88c.jpg" w="700" h="419" title="微软 Win11 都来了它们还活着，盘点老树常青的小软件" width="700" height="419" referrerpolicy="no-referrer"></p><p>▲ Foobar 2000 界面虽然简单，但功能架构却无比强大，可谓是 PC HiFi 发烧友必备</p><p>事实证明，Winamp2 的设计思路时至今日仍未过时，其灵魂继承者 Foobar2000 依然是极其出色的音乐播放器。虽然在互联网时代，在线音乐已然成为主流，但只要你追求更高品质的 PC 音源，Foobar 2000 就仍会是最佳的选择之一。</p><h2>7-Zip</h2><p>压缩软件一直以来都属于装机必备品，大家熟悉的压缩软件有很多，从以前流行的 WinZip、WinRAR，到现在国内常见的 XX 好压、XX 快压等等。但如果要问到，当今什么压缩软件影响力最大？恐怕这名头还是得让给 7-Zip。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/861f2a8c-e1fe-4e7b-bd77-7e101d077c86.png" w="700" h="441" title="微软 Win11 都来了它们还活着，盘点老树常青的小软件" width="700" height="441" referrerpolicy="no-referrer"></p><p>▲ 7-Zip 长久以来广泛地影响着市面上的各种压缩软件</p><p>作为一款 1999 年诞生的压缩软件，7-Zip 目前已经广为人知。7-Zip 的算法非常先进，支持 LZMA 等算法，压缩效率极高，对比 ZIP 和 RAR 在压缩率方面都有优势。不仅如此，7-Zip 还支持 AES 加密、固实压缩等特性，并且广泛支持多种压缩格式，表现可以用惊艳来形容。当然，对于一些需要商业授权的压缩格式例如 RAR，就只能解压不能压缩，但瑕不掩瑜，7-Zip 仍可以称得上是良心之选。</p><p>更为良心的是，7-Zip 以及其压缩格式 7Z 都是免费开源的，因此它的内核被众多其他压缩软件所使用 —— 可以说，国内的一大票压缩软件，都是靠 7-Zip 的代码养活的。</p><p>开源代码可以借用，这无可厚非，但国内某些压缩软件，却抄得毫无底线 —— 使用了 7-Zip 的源码后，非但没有按照 LGPL 协议开源，还自称是自主研发的压缩技术，但实质是在压缩文件的文件头中故意加入无助于压缩的私货，让其他压缩软件无法解压，可谓是相当不厚道了。</p><p>时至今日，7-Zip 依然兢兢业业地为广大用户服务。尽管它的界面简陋，但先进的压缩算法、无私的开源奉献，让它安然走过了这二十多个年头。在 Win11 中，尽管系统已经自带解压缩支持，但如果你想要更强大的压缩算法和功能，7-Zip 依然是值得信赖的方案。</p><h2>VLC</h2><p>如果你入手了一部电脑，想要看视频，那么应该安装什么播放器？或许有的朋友会第一时间想到 XX 影音，但如果这电脑不是 Windows 系统，是 macOS 呢？甚至是 Linux 呢？甚至这不是 PC，而是 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 或者 iOS 设备呢？能解决所有问题的，只有一个答案，那就是 VLC。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/4cdd6800-dad4-4a32-8777-d5f290a5a2c0.jpg" w="700" h="438" title="微软 Win11 都来了它们还活着，盘点老树常青的小软件" width="700" height="438" referrerpolicy="no-referrer"></p><p>▲ 可以支持多平台的 VLC</p><p>作为一款在 2001 年诞生的开源产品，VLC 已经成为了很多平台上万能播放器的首选。如果你切换到陌生的系统，不懂得要如何播放视频，那么找 VLC 准没错。而作为一款老软件，VLC 也相当与时俱进，目前它已经上架了 App Store、Google Play、Windows 商店等软件商店，并加入了对触控的支持，这让它的产品里历久弥新，吸引到了为数不少的新用户。</p><p>当然，VLC 最为核心的卖点，除了跨平台外，还在于强大的播放能力。VLC 能够调用 GPU 对各类视频编码进行硬解码，还可以结合 FFmpeg 提供对先进视频编码的支持，对付当前的 AVC、HEVC 乃至最先进的 VVC 编码完全不在话下。</p><p>对于 VLC 来说，Win11 的发布只意味着多了一个需要兼容的平台，而兼容多平台一直是 VLC 的传统。如果你想要一个什么平台都能用、什么视频都能播的万能播放器，VLC 很值得你考虑，无论你是否使用 Win11 都是如此。</p><h2>Everything</h2><p>如果说 Windows 什么最让人抓狂，搜索功能绝对算得上是一个。尽管 Windows 最近几代都在优化搜索功能，但其速度、资源占用以及搜索结果，都依然难以令人满意。在 Windows XP 之前的时代，Windows 的搜索功能更是犹如残废，那么人们是如何忍受过来的呢？答案就是依靠 Everything 这款神器。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/7d6885b3-fa49-4390-b00c-4750ef96d1fd.png" w="700" h="494" title="微软 Win11 都来了它们还活着，盘点老树常青的小软件" width="700" height="494" referrerpolicy="no-referrer"></p><p>▲ 搜索神器 Everything</p><p>Everything 在 2004 年发布了第一个版本，在随后几年加入了图形界面和更多功能，很快就成为了人们填补 Windows 搜索硬伤的神兵利器。Everything 的优点很多，它体积非常小，仅有 1M 左右，功能和性能却强得超乎人们想象，使用 Everything 搜索几乎瞬时就可以得出结果，通过一些设置还可以搜索文档内的内容，体验要比 Windows 自带的搜索好得多。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/8b06d809-af9f-4386-96fa-96c5ce6bfc16.gif" w="700" h="702" title="微软 Win11 都来了它们还活着，盘点老树常青的小软件" width="700" height="702" referrerpolicy="no-referrer"></p><p>▲ 通过“Everything Toolbar”小工具，可以将 Everything 集成到 Windows 的搜索栏，体验更完美</p><p>时至今日，Everything 仍非常受欢迎。虽然 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 再次改进了搜索功能，但和系统自带搜索相比，没有网络内容的、速度更快、资源占用更少的 Everything，仍是上佳之选。可以预见，只要 Windows 仍存在搜索方面的短板，Everything 就仍会流行很多年。</p><h2>总结</h2><p>实际上，历久弥新的小软件远不止上文所提到的。Windows 已经成为了经典，但缔造这份经典的，又远不止 Windows 本身。在移动互联网新时代，Windows 和其上的一众小软件们，还会一直陪伴着人们吗？我们拭目以待吧。</p>
