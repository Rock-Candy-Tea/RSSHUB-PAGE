<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 16 日消息 传闻成真，Windows 11 预览版镜像已经在网上流出，证实了 Windows 11 将是下一个 Windows 版本的名称，拥有全新的开始菜单和任务栏。不过，Windows 10 不是应该是最后一个 Windows 版本吗？</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/32f2b1e7-0498-4de9-83ad-d05ccb3d2001.jpg@s_2,w_820,h_445" w="1200" h="651" title="Win11 镜像泄露，微软曾称 Win10 是最后一个 Windows 版本" srcset="https://img.ithome.com/newsuploadfiles/2021/6/32f2b1e7-0498-4de9-83ad-d05ccb3d2001.jpg 2x" width="1200" height="445" referrerpolicy="no-referrer"></p><p>当微软发布 Windows 10 时，微软公司高管曾解释说，这将是 Windows 的最后一个版本。微软前高级技术布道师杰里 - 尼克森早在 2015 年就表示：“<span class="accentTextColor">现在我们正在发布 Windows 10，因为 Windows 10 是 Windows 的最后一个版本</span>。”</p><p>但现在泄露的 Windows 11 镜像似乎证实了 Windows 10 不是 Windows 的最后一个版本，这是什么原因呢？</p><p>目前微软还未给出任何官方回应，不过我们可以猜测一下微软这样做的原因。简单来说，最有可能的原因是微软改变想法了，<span class="accentTextColor">新版本的 Windows 变化之大足以值得微软为其改名。</span></p><p>IT之家了解到，微软将在 2021 年 6 月 24 日举行官方活动，届时会展示 Windows 11。在这次活动中，我们应该会了解到关于微软为什么决定为其最新版本的 Windows 使用 Windows 11 名字的官方解释。</p><p>《<a href="https://www.ithome.com/0/557/485.htm" target="_blank">微软 Win11 意外泄露：全新 UI 可有心动？</a>》</p><p><a href="https://www.ithome.com/0/557/468.htm" target="_blank">微软惊天泄漏，Win11 预览版镜像提前出世（附 ISO 下载）</a></p>
