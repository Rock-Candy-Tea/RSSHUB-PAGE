<p data-vmark="e1b3"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 4 日消息，在微博、微信等各大互联网平台相继展示 IP 属地信息后，<span class="accentTextColor">手游方面也将显示 IP 属地了</span>。</p><p style="text-align: center;" data-vmark="7e77"><img src="https://img.ithome.com/newsuploadfiles/2022/8/a2e9745e-feee-4a8e-9b55-dad3499efa45.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzIyMi5wbmc=,g_9,x_22,y_22,a_0,t_100" w="1440" h="610" title="国内运营部分手游已显示 IP 属地" width="1440" height="347" referrerpolicy="no-referrer"></p><p data-vmark="f621">据<span class="accentTextColor">雷霆游戏</span>官网 7 月 22 日公告，为维护网络空间的健康秩序，防范网络诈骗、恶意炒作、散布谣言等违规行为，雷霆游戏平台将依据相关法律法规要求，自本月起，<span class="accentTextColor">在旗下部分游戏开放试行“展示账号 IP 属地”功能</span>。</p><p data-vmark="adfe">该功能上线后，玩家个人主页、部分信息发布页将展示账号 IP 属地，账号 IP 属地信息以网络运营商提供信息为准。</p><p data-vmark="3edf"><strong>Q1: IP 属地显示可以关闭吗？</strong></p><p data-vmark="0cbe">A1: 该功能旨在更透明地向广大玩家呈现真实情况，无法由玩家主动开启或关闭。</p><p data-vmark="cc28"><strong>Q2: 个人主页 IP 属地根据什么显示？</strong></p><p data-vmark="b1e1">A2: 根据最近您游戏上线、发言、朋友圈发布的 IP 属地判定所属地区。</p><p data-vmark="867f"><strong>Q3: IP 属地显示规则是什么？</strong></p><p data-vmark="7d7f">A3: 显示规则为境内展示到省 (自治区、直辖市)，境外展示到国家 (地区)，账号 IP 属地信息以网络运营商提供信息为准。</p><p data-vmark="334e"><strong>Q4: 为什么定位不准？</strong></p><p data-vmark="7854">A4: 只展示对应 IP 属地位置，并不获取其他信息，定位根据运营商 IP 信息解析，如有问题可咨询对应运营商。</p><p style="text-align: center;"
