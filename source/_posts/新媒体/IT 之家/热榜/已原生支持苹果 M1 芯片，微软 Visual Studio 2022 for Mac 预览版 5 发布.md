style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/ab71b46d-3743-4214-bae6-ed9c4ede2a7e.png" w="528" h="576" title="已原生支持苹果 M1 芯片，微软 Visual Studio 2022 for Mac 预览版 5 list-paddingleft-2"><li><p data-vmark="9ef3">移动了文档大纲窗口以使用原生 macOS UI</p></li><li><p data-vmark="a862">文档切换器对话框（使用 Ctrl+Tab 快捷键显示）现在已移植到原生 macOS UI</p></li><li><p data-vmark="7b8c">修复了拖放窗口将它们放置在屏幕错误一侧的问题</p></li><li><p data-vmark="fead">修复了“在文件中查找”和“Command-.Search”不搜索解决方案文件的问题</p></li><li><p data-vmark="6fce">修复了 F# interactive 不执行任何命令的问题</p></li><li><p data-vmark="0d11">修复与 Git 状态视图和历史视图相关的几个错误</p></li><li><p data-vmark="538e">修复了 Git 比较不起作用的问题</p></li><li><p data-vmark="2ef9">修复了测试结果输出视图不滚动到选定测试的问题</p></li></ul><p data-vmark="5a4d">IT之家了解到，在 .NET 6 CLR 上运行 Visual Studio for Mac 是一项重要的改进，从 Mono 迁移到 .NET 6 的首要好处之一是提高了 I / O 场景的性能。</p><p data-vmark="8c71" style="text-align: center;"><img referrerpolicy="no-referrer"></p><p data-vmark="dfe5">此外，新版本已从 Visual Studio 2019 for Mac 迁移了<span class="accentTextColor">近 95% 的用户体验</span>，包括编辑器、所有常用工具窗口以及大多数首选项和项目设置，都采用了原生 macOS UI。</p><p
