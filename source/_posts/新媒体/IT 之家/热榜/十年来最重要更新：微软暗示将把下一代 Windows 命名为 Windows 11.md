<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 4 日消息 2015 年，微软领导层表示，Windows 10 将是其标志性桌面操作系统的最终版本，不会有 Windows 11 或 Windows 12，而是会发布后续的 Windows 10 版本。</p><p>在过去的六年里，微软一直在通过软件更新对 Windows 10 进行定期改进，但这可能很快就会改变。在 5 月 25 日的微软 Build 开发者大会上，微软首席执行官萨提亚 - 纳德拉说，微软正在计划“过去十年中最重要的 Windows 更新之一”，证实了 Windows 操作系统的重大重新设计即将到来的传言。</p><p>虽然“太阳谷更新”将是一次大的升级，但它仍将被视为另一个版本的 Windows 10。如果微软想在下一个 Windows 更新中大做文章，需要从简单的“Windows 10 + 月份”的名称品牌中走出来。</p><p>微软已经安排在 6 月 24 日举行 Windows 活动，这家软件巨头将推出新版桌面操作系统，微软将其称为“下一代 Windows”，其市场名称将在活动期间揭晓。</p><p>然而，有迹象表明，下一代 Windows 可能被称为“Windows 11”。如果仔细看下面微软发布的预告，<span class="accentTextColor">似乎是一个新的 Windows 标志，其阴影形成了一个数字“11”的轮廓。</span></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/4d333eea-85f6-48af-9b71-6e6adef256c9.jpg@s_2,w_820,h_384" w="1440" h="675" title="十年来最重要更新：微软暗示将把下一代 Windows 命名为 Windows 11" srcset="https://img.ithome.com/newsuploadfiles/2021/6/4d333eea-85f6-48af-9b71-6e6adef256c9.jpg 2x" width="1440" height="384" referrerpolicy="no-referrer"></p><p><span class="accentTextColor">微软有意去掉了水平阴影，因此这应该暗示了什么</span>。此外，<span class="accentTextColor">该活动在不寻常的美东时间上午 11 点开始</span>，这也暗示了“Windows 11”的可能性。</p><p>另外，今天早些时候，爆料大神 @evleaks （Evan Blass）也表示，<span class="accentTextColor">微软即将推出的桌面操作系统版本可能被称为“Windows 11”。</span></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/34ce43a9-abbe-462f-9756-bcc44a352e63.jpg" w="706" h="298" title="十年来最重要更新：微软暗示将把下一代 Windows 命名为 Windows 11" width="706" height="298" referrerpolicy="no-referrer"></p><p>虽然我们只有三周的时间等待，因为该公司将揭示 "下一代 Windows" 的名称和功能，但猜测总是很有趣的。</p>
