<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 20 日消息 微软今天面向 Dev 和 Beta 通道用户发布了 <a href="https://www.ithome.com/0/570/404.htm" target="_blank">Windows 11 Build 22000.160 预览版（KB5005189）系统更新</a>，带来了电脑重启更新升级的预估时间、全新的时钟应用等。</p><p>微软表示，上周，我们开始发布作为 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 一部分的应用程序的更新。<a href="https://www.ithome.com/0/569/037.htm" target="_blank">第一组更新包括新的截图工具以及计算器、邮件和日历应用等更新</a>。今天，微软面向 Dev 开发频道向 Windows Insiders 发布 Windows 11 的全新时钟应用，该应用程序的更新已经开始推送，如果你没有立即看到它，请继续检查。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/72ed1b73-a768-4422-867e-c8ad142418be.png" w="1387" h="947" title="微软 Win11 全新时钟应用发布：视觉 UI 更漂亮，支持 Focus Sessions" width="1387" height="560" referrerpolicy="no-referrer"></p><p>全新的时钟应用程序不仅有新的视觉效果，使其在 Windows 11 中看起来很漂亮，它还包括一个非常令人兴奋的主要新功能：Focus Sessions。</p><p>《<a href="https://www.ithome.com/0/567/540.htm" target="_blank">微软 Panos Panay 自曝 Win11 全新功能：Focus Sessions，属于闹钟和时钟一部分</a>》</p><p>保持专注和高效真的很有挑战性。微软表示，随着混合工作和学习成为常态，专注于数字健康从未像现在这样重要，而新的 Focus Sessions 功能就是为了帮助大家。</p><p>这里有一些功能供你尝试，当你使用 Focus Sessions 来保持对你想要完成的事情的专注。</p><ul class=" list-paddingleft-2"><li><p><strong>Focus 计时器：</strong>通过 Focus Sessions 实现你的目标并完成更多工作。告诉你有多少时间，该功能就会安排剩下的时间。</p></li></ul><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/ac0d7254-5bce-45e1-b063-43974673b833.png" w="653" h="619" title="微软 Win11 全新时钟应用发布：视觉 UI 更漂亮，支持 Focus Sessions" width="653" height="619" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p><strong>用 Spotify 找到完美音频来集中注意力：</strong>研究表明，音频可以减少被其他事情分散注意力的机会。Focus Sessions 支持连接你的 Spotify 账户，以帮助你找到完美的音频来帮助保持专注。</p></li></ul><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/3d3246ec-bb57-43f7-b0ae-122c4d78c533.png" w="771" h="897" title="微软 Win11 全新时钟应用发布：视觉 UI 更漂亮，支持 Focus Sessions" width="771" height="897" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p><strong>使用微软 To Do 待办，保持你的工作状态：</strong>选择你想在 Focus Sessions 期间进行的任务，并在完成后将其勾销。Focus Sessions 与 Microsoft To Do 集成，使你能够管理你的 To Do 任务，而不必切换上下文。利用 Focus Sessions 来处理你当天计划的任务，并使其发挥最大的作用。</p></li></ul><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/697c5013-66cc-4b39-8e67-f1519e4295a2.png" w="771" h="496" title="微软 Win11 全新时钟应用发布：视觉 UI 更漂亮，支持 Focus Sessions" width="771" height="496" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p><strong>习惯培养：</strong>为了帮助你建立健康的数字习惯，你可以设定一个每日目标，并挑战自己，不断打破你的连胜纪录。</p></li></ul><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/de003bcd-59d0-46dc-a2b8-248661629690.png" w="765" h="495" title="微软 Win11 全新时钟应用发布：视觉 UI 更漂亮，支持 Focus Sessions" width="765" height="495" referrerpolicy="no-referrer"></p><p>IT之家获悉，微软表示，我们还有更多的 Windows 11 中的应用程序的更新在路上 —— 敬请关注。</p><p><strong>扩展阅读：</strong>《<a href="https://www.ithome.com/0/570/635.htm" target="_blank">微软 Win11 全新时钟 App
