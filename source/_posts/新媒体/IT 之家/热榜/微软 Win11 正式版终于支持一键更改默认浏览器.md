<p data-vmark="6767"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 29 日消息，据 Neowin 报道，微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 中最有争议的变化之一是操作系统对浏览器的操作方式。在 Windows 11 中改变默认浏览器是繁琐而恼人的过程，而且微软有时还会在未经用户同意的情况下通过重置配置来给系统带来麻烦。经过开发者和客户的多次投诉，微软终于“修复”了 Windows 11 中最恼人的设置之一，恢复了一键改变默认浏览器的功能。</p><p style="text-align: center;" data-vmark="46cb"><img src="https://img.ithome.com/newsuploadfiles/2022/3/44bd5c83-1042-447a-a50a-3074725b59ad.png" w="920" h="606" title="微软 Win11 正式版终于支持一键更改默认浏览器" width="920" height="540" referrerpolicy="no-referrer"></p><p data-vmark="bcdb">近期，微软发布了稳定版 Windows 11 的可选累积更新 KB5011563。除了修复大量的错误 Bug 外，该更新还从 Dev 预览版中回传了对设置应用程序的改进。Windows 11 现在允许用户导航到默认应用程序设置部分，选择最喜欢的浏览器，并将其作为默认浏览器。操作系统不再需要手动改变十多个文件关联，为而不是费力找到谷歌 Chrome 浏览器。</p><p style="text-align: center;" data-vmark="3b70"><img src="https://img.ithome.com/newsuploadfiles/2022/3/4afe3eb0-8704-4985-bca1-c2db1d462486.jpg@s_2,w_820,h_599" w="1150" h="840" title="微软 Win11 正式版终于支持一键更改默认浏览器" srcset="https://img.ithome.com/newsuploadfiles/2022/3/4afe3eb0-8704-4985-bca1-c2db1d462486.jpg 2x" width="1150" height="599" referrerpolicy="no-referrer"></p><p data-vmark="73dc">以下是安装 Windows 11 Build 22000.593 版本后如何改变默认浏览器的方法：</p><ul class=" list-paddingleft-2"><li><p data-vmark="06b6">安装喜欢的浏览器。</p></li><li><p data-vmark="5b61">打开“设置”应用程序（Win + I），导航到“应用程序”>“默认应用程序”。</p></li><li><p data-vmark="f2e1">在应用程序列表中找到浏览器并打开它。</p></li><li><p data-vmark="9ad1">点击“设置默认”按钮。</p></li></ul><p data-vmark="eb36">当然，微软仍然是微软，这意味着 Windows 11 将继续尝试以其他方式强加 Edge 浏览器。在 Windows 11 中使用更新的设置应用改变默认浏览器并不影响特定的格式。例如，Edge 浏览器仍将是打开 PDF、SHTML、SVG、FTP 和其他网络相关文件的默认应用。另外，用户仍然需要第三方调整器来在微软 Edge 以外的浏览器中打开 Windows 小组件中的链接。</p>
