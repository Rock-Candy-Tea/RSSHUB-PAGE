<p data-vmark="73e5"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 21 日消息，据 Windows Latest 报道，在许多方面，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 是一款非常漂亮的操作系统，但新的设计大修是以功能改变为代价的。在开始使用后，不少用户抱怨了 Windows 11 的多处不佳，特别是如果你是一名专业用户，并且依赖任务栏拖放或文件资源管理器中的文件夹预览等功能时。</p><p data-vmark="1268" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/c4549641-25cc-4604-9aab-5843d91250eb.jpg" w="696" h="363" title="微软 Win11 新版文件资源管理器上手体验：文件夹预览、快速访问固定、整合 OneDrive 网盘" width="696" height="363" referrerpolicy="no-referrer"></p><p data-vmark="585f">微软最近向 Dev 开发频道的测试用户员推送了 Windows 11 Build 22557 预览版更新，其中一个大的变化是能够将图像或文本等文件拖放到任务栏中的应用程序上，以打开突出显示的应用程序中的文件。例如，你可以在 Outlook 中拖放图片，只需将其拖到 Outlook 图标上。</p><p data-vmark="7e20">除了任务栏的改进之外，微软还恢复了对文件资源管理器中“文件夹预览”的支持。文件夹预览是以前 Windows 所有版本中都存在的方便功能，许多用户每天都在使用，所以当这个功能被 Windows 11 删除后确实激怒了众多用户。</p><p data-vmark="b7af" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/3333bde3-9759-463a-b868-c8f724cdc3b0.jpg@s_2,w_820,h_569" w="1000" h="694" title="微软 Win11 新版文件资源管理器上手体验：文件夹预览、快速访问固定、整合 OneDrive 网盘" srcset="https://img.ithome.com/newsuploadfiles/2022/2/3333bde3-9759-463a-b868-c8f724cdc3b0.jpg 2x" width="1000" height="569" referrerpolicy="no-referrer"></p><p data-vmark="e72f">从 Windows 11 Build 22557 开始，用户现在可以在资源管理器中预览文件夹的文件，而无需打开该文件夹。</p><p data-vmark="ff3e">你可以尝试 Windows 11 即将推出的功能，在虚拟机中安装预览版，并导航到文件资源管理器中的一个目录，该目录中有你想预览的包含图片或文档等媒体的文件夹。</p><p data-vmark="b2e0">如截图所示，如果媒体内容已被 Windows 索引，你将立即看到一个文件夹预览缩略图。</p><h3 data-vmark="ff92">文件资源管理器的快速访问现在允许固定文件</h3><p data-vmark="8adc">文件资源管理器的快速访问视图现在允许你固定文件。以前微软允许用户固定文件夹，但此次更新将支持范围扩大到了文件。</p><p data-vmark="e29e" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/f98f2143-e007-40f0-b279-02e141494ff2.jpg@s_2,w_820,h_586" w="981" h="701" title="微软 Win11 新版文件资源管理器上手体验：文件夹预览、快速访问固定、整合 OneDrive 网盘" srcset="https://img.ithome.com/newsuploadfiles/2022/2/f98f2143-e007-40f0-b279-02e141494ff2.jpg 2x" width="981" height="586" referrerpolicy="no-referrer"></p><p data-vmark="54ff">微软在最近文件上方创建了一个新的部分，以突出固定的文件。</p><p data-vmark="9104">如果你的微软账户连接到设备上，文件资源管理器还将从 Office.com 同步固定的和最近的文件。这包括 OneDrive、SharePoint 和 Teams。</p><p data-vmark="11cb">这个功能支持实时同步，所以如果你在快速访问中固定或取消固定住 Office.com 文件，它也将反映在 Office.com 和 Office 应用程序中。</p><h3 data-vmark="6e9e">向 Outlook 分享内容的新方法</h3><p data-vmark="15b2">微软正在测试一项名为“Outlook 桌面集成”的功能，将允许本地文件分享到 Outlook，并直接从文件资源管理器中撰写电子邮件。电子邮件撰写框和附件将出现在资源管理器的共享窗口内，因此你可以轻松地从资源管理器发送电子邮件。</p><p data-vmark="fe5e" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/bccfbeb8-8795-40eb-8897-085220a96bb3.jpg@s_2,w_820,h_603" w="1291" h="950" title="微软 Win11 新版文件资源管理器上手体验：文件夹预览、快速访问固定、整合 OneDrive 网盘" srcset="https://img.ithome.com/newsuploadfiles/2022/2/bccfbeb8-8795-40eb-8897-085220a96bb3.jpg 2x" width="1291" height="603" referrerpolicy="no-referrer"></p><p data-vmark="2feb">目前，文件资源管理器的 Outlook 集成仅适用于本地文件，不可能共享存储在 OneDrive 文件夹中的文件。</p><p data-vmark="53e8">此外，Windows 11 的共享窗口还包括对来自 Microsoft 365 的联系用户支持，并且列表会自动同步，这使得通过资源管理器的 Outlook 共享文件更加容易。</p><h3 data-vmark="4e9f">文件资源管理器中的 OneDrive 整合</h3><p data-vmark="8364">微软正在使直接从文件资源管理器管理 OneDrive 变得更加容易。如截图所示，你可以浏览 OneDrive 文件夹，在资源管理器窗口内观看同步状态和配额使用情况。</p><p data-vmark="26a8" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/e228dbb4-2598-484b-8ddc-6df022a6a19c.jpg" w="446" h="420" title="微软 Win11 新版文件资源管理器上手体验：文件夹预览、快速访问固定、整合 OneDrive 网盘" width="446" height="420" referrerpolicy="no-referrer"></p><p data-vmark="2590">现在文件资源管理器中显示了存储容量，甚至不用打开 OneDrive 文件夹。</p><p data-vmark="e511">这些变化目前只对测试用户开放。普通公众将通过 Windows 11 太阳谷 2 获得新的文件资源管理器功能，该版本预计将在今年秋季开始推出。</p><p
