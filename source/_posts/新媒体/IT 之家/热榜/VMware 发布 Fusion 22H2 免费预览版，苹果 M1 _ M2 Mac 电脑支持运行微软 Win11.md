<p data-vmark="e2fe"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 29 日消息，VMWare 为 Mac 用户发布了 Fusion 22H2 技术预览版。适用于 macOS 的免费虚拟化软件为 Apple Silicon 和 Intel 带来了期待已久的对微软 Windows 11 系统的支持，允许使用 macOS 的 Mac 并行运行微软最新的操作系统。</p><p style="text-align: center;" data-vmark="d6b9"><img src="https://img.ithome.com/newsuploadfiles/2022/7/30934f2c-77a8-4e40-8ef0-c5fb4251379e.png" w="1440" h="1135" title="VMware 发布 Fusion 22H2 免费预览版，苹果 M1 / M2 Mac 电脑支持运行微软 Win11" width="1440" height="646" referrerpolicy="no-referrer"></p><p data-vmark="bf08">下面是 VMWare Fusion 22H2 技术预览版的新增功能：</p><ul class=" list-paddingleft-2"><li><p data-vmark="799c">Intel 和 Apple Silicon 上的 Windows 11，具有 2D 图形和网络支持。</p></li><li><p data-vmark="ccff">VMTools 安装支持基于 M1  Mac 上的 Windows 11 guest 操作系统。</p></li><li><p data-vmark="d5fa">具有快速加密支持的虚拟 TPM 设备。</p></li><li><p data-vmark="67e2">改进 M1 上的 Linux 支持。</p></li><li><p data-vmark="46c4">Linux 虚拟机中的 3D 图形硬件加速和 OpenGL 4.3。</p></li><li><p data-vmark="9191">适用于 Apple Silicon 和 Intel Mac 的通用二进制文件。</p></li></ul><p data-vmark="ebfb"><span class="accentTextColor">借助 VMWare Fusion 22H2 技术预览版，Mac 用户可以创建符合 Windows 11 硬件要求的虚拟机。</span> Fusion 现在支持具有快速加密功能的 Virtual TPM 2.0，为敏感数据提供改进的性能和安全区域（可以选择完全加密 VM）。</p><p data-vmark="1ecb">除了 TPM 2.0 支持外，VMWare Fusion 22H2 还为 2D 图形带来了 WDDM 驱动程序，允许用户以高屏幕分辨率操作 Windows 虚拟机。</p><p data-vmark="db21">值得注意的是，VMWare Fusion 22H2 目前处于公测阶段（<a href="https://customerconnect.vmware.com/downloads/get-download?downloadGroup=FUS-PUBTP-22H2" target="_blank">点此下载</a>），因此在使用该软件时可能会出现错误 Bug 和粗糙设计。此外，Windows 11 支持仍需要一些额外的工作和润色。</p><p style="text-align: center;" data-vmark="f7bd"><img src="https://img.ithome.com/newsuploadfiles/2022/7/34206e77-de21-49d2-910c-721ab28490df.png" w="1440" h="928" title="VMware 发布 Fusion 22H2 免费预览版，苹果 M1 / M2 Mac 电脑支持运行微软 Win11" width="1440" height="528" referrerpolicy="no-referrer"></p>
