<p data-vmark="ecf7"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 23 日消息，苹果将于下周一正式推送 macOS Monterey，<span class="accentTextColor">不过新系统的一些功能只适用于苹果自研 M 系列芯片</span>，搭载英特尔芯片的 Mac 则会受到一些限制。</p><p data-vmark="53a5"><img src="https://img.ithome.com/newsuploadfiles/2021/10/4e85eaa1-d0cc-4893-889c-72ac2aa7a2d9.png" w="688" h="365" title="苹果 macOS Monterey 下周到来，这些功能不适用于英特尔芯片的 Mac" width="688" height="365" referrerpolicy="no-referrer"></p><h2 data-vmark="19c0">FaceTime 的人像模式</h2><p data-vmark="4fb4">通过 macOS Monterey 系统中的人像模式，用户可以在 FaceTime 通话期间，模糊背景，让焦点集中在自己身上。但是如果用户的 Mac 为英特尔芯片，则不享有该功能。</p><h2 data-vmark="dad7">地图互动地球仪</h2><p data-vmark="8d01">在配备 M1、M1 Pro 和 M1 Pro Max 芯片的 Mac 上，苹果增强了地图应用程序，包括一个新的地球视图，用户可以旋转世界并放大地球上的不同区域，还可以从太空中看到地球的 3D 视图，导航更有趣。</p><p data-vmark="433c"><img src="https://img.ithome.com/newsuploadfiles/2021/10/b3c3f853-b1f8-41c0-944c-8fecb657c575.png" w="687" h="437" title="苹果 macOS Monterey 下周到来，这些功能不适用于英特尔芯片的 Mac" width="687" height="437" referrerpolicy="no-referrer"></p><h2 data-vmark="8ed8">对象捕捉</h2><p data-vmark="e1fc">Object Capture 允许用户通过将一系列照片拼接在一起来创建逼真的、经过 AR 优化的 3D 对象。该技术称为摄影测量，以前需要专门的软件才能完成，但苹果 已将 Object Capture API 集成到 macOS Monterey 系统中。</p><p data-vmark="be87"><img src="https://img.ithome.com/newsuploadfiles/2021/10/54bf5bbb-4814-478b-a8f6-f06f1fa460e4.png" w="636" h="392" title="苹果 macOS Monterey 下周到来，这些功能不适用于英特尔芯片的 Mac" width="636" height="392" referrerpolicy="no-referrer"></p><p data-vmark="c67b">例如，使用像 PhotoCatch 这样的应用程序，用户可以从多个角度导入一个视频或几张对象的照片，可将它们转换为逼真的 3D 模型，然后可以轻松地将其集成到任何对象中。所有 M 系列芯片都支持 Object Capture，但至少具有 16GB RAM 和 4GB VRAM 的英特尔芯片 Mac 才支持该功能。</p><h2 data-vmark="49dc">Siri 文字转语音</h2><p data-vmark="e991">相比于英特尔芯片 Mac ，M 系列芯片 Mac 的语音转换功能，支持更多的语言文本，包括瑞典语、丹麦语、挪威语和芬兰语等。</p><p data-vmark="163f"><img src="https://img.ithome.com/newsuploadfiles/2021/10/26396da9-d85a-4639-81d3-995c58304180.png" w="697" h="511" title="苹果 macOS Monterey 下周到来，这些功能不适用于英特尔芯片的 Mac" width="697" height="511" referrerpolicy="no-referrer"></p><h2 data-vmark="ede0">设备上的键盘听写</h2><p data-vmark="c09e">使用键盘听写，用户可以在任何可以输入的地方听写文本，而且该功能会随着用户使用次数的增加而改进。在 M 系列 Mac 上，键盘听写现在通过在设备上执行所有处理，来保护用户隐私，它将完全离线，用户还可以在没有超时的情况下听写任何长度的文本。但是，在 英特尔 Mac 上，有 60 秒的时间限制。</p><p data-vmark="791d">IT之家了解到，苹果在 10 月 19 日发布了全新的 14 英寸和 16 英寸的 MacBook Pro，配备专为 Mac 设计的新一代突破性芯片 M1 Pro 和 M1 Max。两者均在 M1 的开创性架构基础上进一步拓展，M1 Pro 带来了令人惊叹的性能表现和业界领先的能效，M1 Max 更是在这两个方面达到了全新高度。</p><p data-vmark="ae8d"><img src="https://img.ithome.com/newsuploadfiles/2021/10/fbb8036f-3ea1-4f22-b4ae-28465cfc7cde.png" w="779" h="396" title="苹果 macOS Monterey 下周到来，这些功能不适用于英特尔芯片的 Mac" width="779" height="396" referrerpolicy="no-referrer"></p>
