<p data-vmark="2336"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 27 日消息，微软今日凌晨在 Windows Insider 播客展示了 <a href="https://www.ithome.com/0/615/331.htm" target="_blank">Windows 11 任务管理器的全新多彩设计</a>，用户可以调整 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 主题色，让任务管理器自定义显示强调色。</p><p data-vmark="d2c6">然而有网友发现了一个有趣的细节，微软官方人员使用的电脑搭载了<span class="accentTextColor">英特尔 i7-7660U 处理器</span>，这明显是一款不被 Windows 11 所支持的 CPU。</p><p data-vmark="8e3d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/654723d5-dc70-4b8d-b28f-f3030f08b60b.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xdy9xdzE4Ny5wbmc=,g_7,x_19,y_19,a_0,t_100" w="1214" h="686" title="尴尬，微软设计团队使用不被支持的 CPU 展示 Win11" width="1214" height="463" referrerpolicy="no-referrer"></p><p data-vmark="5380">IT之家了解到，Windows 11 升级要求兼容 64 位处理器、4GB 内存、64GB 存储空间、UEFI secure boot、图形要求和 TPM 2.0 的设备，英特尔处理器大部分是八代及以上，只有少部分七代处理器受支持，比如：</p><ul class=" list-paddingleft-2"><li><p data-vmark="37b8">英特尔酷睿 i5-7640X</p></li><li><p data-vmark="0ddf">英特尔酷睿 i7-7740X</p></li><li><p data-vmark="653d">英特尔酷睿 i7-7800X</p></li><li><p data-vmark="3ddf">英特尔酷睿 i7-7820HQ</p></li><li><p data-vmark="4ccc">英特尔酷睿 i7-7820X</p></li><li><p data-vmark="f922">英特尔酷睿 i7-7900X</p></li><li><p data-vmark="de94">英特尔酷睿 i7-7920X</p></li><li><p data-vmark="a2da">英特尔酷睿 i9-7940X</p></li><li><p data-vmark="73c6">英特尔酷睿 i9-7960X</p></li><li><p data-vmark="4c46">英特尔酷睿 i9-7980XE</p></li></ul><p data-vmark="cd2d">微软此前表示，不受支持的 PC 设备<span class="accentTextColor">可使用 Windows 11 ISO 镜像手动安装升级</span>，不过需要自行承担风险。微软官方人员使用了不受支持的处理器，看来也是官方默认了这一做法。</p><p data-vmark="66a3">此前，微软还发布了一个改进版的 Windows 11 兼容性检查工具 <a href="https://www.ithome.com/0/576/648.htm" target="_blank" title="“PC 健康检查”">“PC 健康检查”</a>，方便用户检测自己的设备是否适合通过微软支持渠道升级到 Windows 11。</p>
