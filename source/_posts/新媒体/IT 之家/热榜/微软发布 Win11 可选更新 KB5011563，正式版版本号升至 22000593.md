            <p data-vmark="74bb"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 29 日消息，微软刚发布了一个新的 Build 22000.593（KB5011563）作为 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的可选 "C" 更新下，修复大量 bug。</p><p data-vmark="54ae">在新的版本中有一个已知的问题需要注意。微软指出：</p><blockquote><p data-vmark="3fd9">在受影响的 Windows 上安装 2022 年 1 月 11 日发布的 Windows 更新或更高的 Windows 版本后，通过使用控制面板中的备份和还原（Windows 7）程序创建的恢复光盘（CD 或 DVD）可能将无法启动。</p><p data-vmark="d3ba">在安装了 2022 年 1 月 11 日之前发布的 Windows 更新的设备上，通过使用备份和还原（Windows 7）应用程序创建的恢复光盘不受此问题影响，系统将按预期进行启动。</p></blockquote><p data-vmark="ae16">IT之家了解到，微软也表示没有任何第三方的备份或恢复应用程序被反馈称有受到这个问题的影响，而且目前还没有针对上述问题的官方解决方法，不过微软表示他们正在努力解决这个问题。</p><p data-vmark="408c">在这个版本中，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 最多同时显示三个高优先级的 Toast 通知窗口。这个功能主要是针对那些使用系统中的 Windows 通知发送电话、提醒或报警通知的应用程序。因此，系统最多可能会同时出现四个通知--三个高优先级通知和一个普通优先级通知。</p><p data-vmark="4d80">下面是微软给出的 bug 修复清单：</p><blockquote><p data-vmark="1e77">我们修复了一个问题，即在你升级操作系统后关闭通用 Windows 平台 UWP 应用程序的自动启动。</p><p data-vmark="5b6c">我们修复了一个导致 SystemSettings.exe 停止工作的问题。</p><p data-vmark="c238">我们修复了一个影响 searchindexer.exe 的问题，它使 Microsoft Outlook 的离线搜索无法返回最近的电子邮件。</p><p data-vmark="a07b">我们修复了一个增加 Windows 启动时间的问题。这个问题的出现是因为当 5G 无线广域网（WWAN）设备缓慢唤醒时，网络 API 中的一个 UI 线程停止响应。</p><p data-vmark="eb26">我们修复了 wmipicmp.dll 模块中的一个内存泄漏，该模块在 System Center Operations Manager（SCOM）数据中心监控系统中导致了大量的错误警报。</p><p data-vmark="9eda">我们修复了一个问题，即从注册表中错误地返回执行策略设置。</p><p data-vmark="7170">我们修复了一个问题，即现代浏览器无法正确呈现由 gpresult / h 生成的 HTML。</p><p data-vmark="8702">我们修复了一个问题，在对 AppLocker 进行 PowerShell 测试时，该问题会导致一个文件出现 "拒绝访问" 的异常。</p><p data-vmark="929d">我们修复了一个问题，该问题导致远程桌面服务（RDS）服务器在登录用户数超过 100 时变得不稳定。这使您无法在 Windows Server 2019 上使用 RDS 访问已发布的应用程序。</p><p data-vmark="3eac">我们修复了一个问题，即当您浏览域或组织单元（OU）时返回错误信息。这个问题是由于内存清零不当而发生的。</p><p data-vmark="f231">我们修复了一个问题，该问题导致组策略管理控制台在您关闭后停止工作。系统记录应用程序错误事件 ID 1000 和错误，0xc0000005（STATUS_ACCESS_VIOLATION）；失败的模块是 GPOAdmin.dll。</p><p data-vmark="be84">我们修复了一个可能导致组策略服务停止处理组策略注册表偏好的遥测信息的问题。</p><p data-vmark="8307">我们修复了 DirectX 内核组件中的一个停止错误（0xD1，DRIVER_IRQL_NOT_LESS_OR_EQUAL）。</p><p data-vmark="0bef">我们修复了一个可能导致 Kerberos.dll 在本地安全授权子系统服务（LSASS）中停止工作的问题。当 LSASS 为同一客户用户同时处理用户服务（S4U）用户对用户（U2U）请求时，会发生这种情况。</p><p data-vmark="054f">我们修复了一个影响到密钥分发中心（KDC）代理的问题。KDC 代理不能正确获得 Kerberos 票据，以签署 Key Trust Windows Hello for Business。</p><p data-vmark="e6f0">我们在 Azure Active Directory (AAD) 网络账户管理器 (WAM) 中增加了对微软账户 (MSA) 穿透场景的支持。</p><p data-vmark="e645">我们修复了在某些密码更改场景中记录事件 ID 37 的问题，包括故障转移集群名称对象（CNO）或虚拟计算机对象（VCO）密码更改。</p><p data-vmark="e6b0">我们修复了一个问题，该问题使用户账户控制（UAC）对话框无法正确显示请求提升权限的应用程序。</p><p data-vmark="4fad">我们修复了一个问题，该问题可能导致微软 OneDrive 文件在你重命名并按下 Enter 键后失去焦点。</p><p data-vmark="555f">当你搜索 widgets 这个词时，我们返回了相应的设置页面。</p><p data-vmark="34f9">我们修复了一个问题，该问题使事件 4739 在策略改变后无法显示某些属性的新值。</p><p data-vmark="fd68">我们修复了一个问题，当你跨域移动计算机账户时，会导致 Move-ADObject 命令失败。错误信息是："为一个只能有一个值的属性指定了多个值"。</p><p data-vmark="1c10">我们修复了一个问题，当启用 SMB 加固时，你无法使用 IP 地址访问服务器信息块（SMB）共享。</p><p data-vmark="cdfc">我们修复了一个导致 SMB 服务器（srv2.sys）中停止错误 0x1E 的问题。</p><p data-vmark="765a">我们修复了一个问题，当你创建一个集群时，会导致 NetBIOS 和 DNS Active Directory 域名不匹配。</p></blockquote>
