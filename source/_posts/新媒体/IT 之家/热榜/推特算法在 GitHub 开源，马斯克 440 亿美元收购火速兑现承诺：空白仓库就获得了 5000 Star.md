            <p data-vmark="5f23"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 26 日消息，马斯克今日以 440 亿美元（约 2873.2 亿元人民币）买下推特 Twitter，成为了 IT 行业的重磅消息。在本月早些时候，<span class="accentTextColor">马斯克还承诺如果收购推特，将开源其算法</span>。</p><p data-vmark="fd51">IT之家小伙伴 @Ryen_D 发现，推特今日在 GitHub 创建了一个名为“<span class="accentTextColor">the-algorithm</span>（算法）”的<a href="https://github.com/twitter/the-algorithm" target="_blank">空白仓库</a>，仅推出六小时就引来了<span class="accentTextColor">五千多个 Star</span>。</p><p data-vmark="9a74" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/ddb1299d-2420-4fcc-b450-225fca4f51f2.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xZC9xZDE3NS5wbmc=,g_9,x_17,y_17,a_0,t_100" w="1137" h="536" title="推特算法在 GitHub 开源，马斯克 440 亿美元收购火速兑现承诺：空白仓库就获得了 5000 Star" width="1137" height="387" referrerpolicy="no-referrer"></p><p data-vmark="b0db" style="text-align: justify;">IT之家了解到，马斯克此前表示，他相信“给言论自由提供一个包容性舞台是极为重要的”。<span class="accentTextColor">他认为 Twitter 算法应该变成开源算法</span>，应该将后台代码放在 Github 平台。</p><p data-vmark="bbae" style="text-align: justify;">看来马斯克确实说到做到，收购消息刚公布就跟进开源，目前尚不清楚算法文件何时将上线。</p><p data-vmark="a21a" style="text-align: justify;">马斯克在新闻稿中的一份声明中称：“言论自由是民主运作的基石，而推特是一个数字城镇广场，人们在这里讨论对人类未来至关重要的问题，我还希望通过使用新功能增强产品、使算法开源以增加信任、击败垃圾邮件机器人以及对所有人进行真人验证，让推特比以往任何时候都更好。”</p>
