<p data-vmark="8e6e"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 14 日消息，据 Electrek 报道，过去几天，特斯拉交付了一些新的 Model 3 和 Model Y 汽车，但它们<span class="accentTextColor">没有 USB 接口</span>。</p><p data-vmark="90fb">据报道，有一些特斯拉车主称，收到的汽车的中控 USB 接口没有了，只留下一个小洞，<span class="accentTextColor">无线充电也不起作用</span>，车主想要给手机充电结果傻眼了。</p><p data-vmark="44af"><img src="https://img.ithome.com/newsuploadfiles/2021/11/ee39e0a5-b315-44d8-b509-dfdd29ebe087.png" w="1440" h="643" title="部分特斯拉 Model 3/Y 新车没有 USB 接口，官方称零部件短缺" width="1440" height="366" referrerpolicy="no-referrer"></p><p data-vmark="abfb"><img src="https://img.ithome.com/newsuploadfiles/2021/11/a5d5f063-db98-4853-8bd5-5bace328c5c2.png" w="545" h="849" title="部分特斯拉 Model 3/Y 新车没有 USB 接口，官方称零部件短缺" width="545" height="849" referrerpolicy="no-referrer">.</p><p data-vmark="4ff0"><img src="https://img.ithome.com/newsuploadfiles/2021/11/59d13110-ca7b-457d-98f9-5512e1711be5.png" w="1153" h="530" title="部分特斯拉 Model 3/Y 新车没有 USB 接口，官方称零部件短缺" width="1153" height="377" referrerpolicy="no-referrer"></p><p data-vmark="98e5">特斯拉服务部门将该缺失归咎于零部件短缺，并且告诉一些受影响的车主，<span class="accentTextColor">他们可以打电话安排服务预约，以后再装 USB 接口</span>，预计会在 12 月份。</p><p data-vmark="d9ee"><img src="https://img.ithome.com/newsuploadfiles/2021/11/d706fab3-bed6-4c2c-bbd2-8edccb90e169.png" w="960" h="240" title="部分特斯拉 Model 3/Y 新车没有 USB 接口，官方称零部件短缺" width="960" height="205" referrerpolicy="no-referrer"></p><p data-vmark="a9fd">IT之家了解到，全球芯片短缺极大地影响了汽车行业，而且已经持续了很长时间。尽管特斯拉避免了最糟糕的情况，但也不得不取消了新 Model 3 和 Model Y 汽车的乘客腰部支撑调整。宝马还从新车上移除了触摸屏，并给予了 500 美元的优惠。</p>
