<p data-vmark="9c39"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 20 日消息，微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> <a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>子系统在官宣后经过多次曝光，现在终于要来了。据IT之家网友 @扶摇直上九万里 投稿，<span class="accentTextColor">Win11 安卓子系统将面向 Beta 通道的美国用户开启测试</span>。</p><p data-vmark="1bac"><img src="https://img.ithome.com/newsuploadfiles/2021/10/f51203e7-e61f-4f86-895e-cda6f52fa169.png" w="1440" h="322" title="Win11 安卓子系统来了，微软宣布面向 Beta 通道开启测试（附攻略）" width="1440" height="183" referrerpolicy="no-referrer"></p><p data-vmark="872d">微软表示，在 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的安卓 App 将拥有熟悉、轻松、集成的体验：</p><ul class=" list-paddingleft-2"><li><p data-vmark="824c">用户可以将安卓 App 固定到开始菜单或任务栏，并通过鼠标、触摸或笔输入与它们交互；</p></li><li><p data-vmark="8909">安卓 App 还可以集成到 Alt + Tab 和任务视图中，帮助用户在 App 之间快速切换；</p></li><li><p data-vmark="9a1f">用户可以在操作中心中查看来自安卓 App 的通知，或在 Windows 应用程序和安卓 App 之间共享剪贴板；</p></li><li><p data-vmark="7918">微软还添加了无障碍体验，许多 Windows 辅助功能设置都适用于安卓 App。</p></li></ul><p data-vmark="e543"><img src="https://img.ithome.com/newsuploadfiles/2021/10/847c78c2-3e32-4d78-a6c1-b7ae48aa5731.png" w="938" h="522" title="Win11 安卓子系统来了，微软宣布面向 Beta 通道开启测试（附攻略）" width="938" height="456" referrerpolicy="no-referrer"></p><p data-vmark="5d90">▼ 在 Surface Pro X 上运行 Kindle App</p><p data-vmark="b024"><img src="https://img.ithome.com/newsuploadfiles/2021/10/84f44e5c-69b1-41c8-af12-4e6692cb8db3.png" w="940" h="620" title="Win11 安卓子系统来了，微软宣布面向 Beta 通道开启测试（附攻略）" width="940" height="541" referrerpolicy="no-referrer"></p><p data-vmark="c23a">微软与亚马逊合作，<span class="accentTextColor">为 Windows 内部人员首先推出了 50 款安卓 App</span>，以便在一系列硬件上进行测试和验证。微软将在未来几个月通过 Windows Insider 程序发布更多的安卓 App。</p><p data-vmark="5858">这 50 款 App 中包含：</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="e372">手机游戏：《王国纪元》《琼的旅程》《 Coin Master》等；</p></li><li><p data-vmark="fdd2">阅读：Kindle App、漫画 App 等；</p></li><li><p data-vmark="36bf">儿童内容：Khan Academy Kids、Lego Duplo World 等。</p></li></ul><p data-vmark="d885">IT之家了解到，Win11 的安卓子系统包括 Linux 内核和<span class="accentTextColor">基于 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 开源项目（AOSP）版本 11 的 Android 操作系统</span>。它作为 Amazon Appstore 安装的一部分通过 Microsoft Store 分发。该子系统在 Hyper-V 虚拟机中运行，就像 Linux 子系统一样，可以将 AOSP 环境中 App 的运行时和 API 映射到 Windows 图形层、内存缓冲区、输入模式、物理和虚拟设备以及传感器，可以在英特尔、AMD、高通的 CPU 上运行。</p><p data-vmark="cd7c"><img src="https://img.ithome.com/newsuploadfiles/2021/10/e54f2ee1-f986-4482-af92-e407dd717342.png" w="1440" h="810" title="Win11 安卓子系统来了，微软宣布面向 Beta 通道开启测试（附攻略）" width="1440" height="461" referrerpolicy="no-referrer"></p><h2 data-vmark="f4f4">使用方法：</h2><ul class=" list-paddingleft-2"><li><p data-vmark="7bec">电脑需要安装在 Windows 11（Build 22000.xxx series builds）上，并满足硬件要求；</p></li><li><p data-vmark="2206">用户需要为电脑的 BIOS/UEFI 启用虚拟化；</p></li><li><p data-vmark="c369">电脑的地区必须设置为美国；</p></li><li><p data-vmark="3e83">电脑必须处于 Win11 Beta 通道（Dev 通道将在后续支持）；</p></li><li><p data-vmark="6f40">用户需要有一个美国亚马逊帐户才能使用亚马逊应用商店；</p></li><li><p data-vmark="e73c">然后<a href="http://aka.ms/AmazonAppstore" target="_blank">点击链接</a>前往微软应用商店进行安装使用。</p></li></ul><p data-vmark="b980"><img src="https://img.ithome.com/newsuploadfiles/2021/10/c1d7c976-e752-44d2-bca2-7e3d95f7fd2d.png" w="1417" h="730" title="Win11 安卓子系统来了，微软宣布面向 Beta 通道开启测试（附攻略）" width="1417" height="422" referrerpolicy="no-referrer"></p><p data-vmark="a87d">此外，Win11 安卓子系统的版本号为 1.7.32815.0，用户可以在开始菜单中找到该子系统，并且调整设置。</p><h2 data-vmark="87e8">已知问题：</h2><p data-vmark="1978"><strong>微软应用商店：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="54bd">应用商店不会显示亚马逊应用商店 App 的审查评级。</p></li></ul><p data-vmark="a807"><strong>亚马逊应用商店：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="f94a">如果遇到亚马逊应用商店登录问题，请关闭广告拦截器。</p></li><li><p data-vmark="5bff">使用亚马逊应用商店时，辅助功能选项尚未启用。</p></li></ul><p data-vmark="8d6c"><strong>安卓子系统：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="5253">ARM64 上的蜂窝 LTE 网络接入有问题。</p></li><li><p data-vmark="0d2c">ARM64 上的某些 App 可能会渲染错误（display visual artifacts）。</p></li><li><p data-vmark="9627">运行多个 App 时，性能可能会有所不同。</p></li><li><p data-vmark="8cf0">某些 App 存在已知的摄像头问题。</p></li><li><p data-vmark="18a9">现代待机模式下的 App 将遇到问题。</p></li></ul>
