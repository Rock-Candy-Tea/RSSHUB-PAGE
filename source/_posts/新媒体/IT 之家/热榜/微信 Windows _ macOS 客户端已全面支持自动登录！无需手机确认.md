            <p data-vmark="ae1a"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 22 日消息，据多名网友反馈，微信 PC 客户端在升级到最新版本后，包括 Windows 和 macOS 版，已经支持自动登录该设备，可以选择在常用设备登录后不需要手机确认。目前微信已支持同时登录电脑、平板电脑、手表等设备。</p><p data-vmark="fdb1"><img src="https://img.ithome.com/newsuploadfiles/2021/12/3e177c36-d457-42f4-873b-fc45ab62fe6a.jpg@s_2,w_820,h_1773" w="937" h="2026" alt="微信自动登录" title="微信 Windows / macOS 客户端已全面支持自动登录！无需手机确认" srcset="https://img.ithome.com/newsuploadfiles/2021/12/3e177c36-d457-42f4-873b-fc45ab62fe6a.jpg 2x" width="937" height="1773" referrerpolicy="no-referrer"></p><p data-vmark="d3b5"><img src="https://img.ithome.com/newsuploadfiles/2021/12/bcfc9f11-a73f-4c78-832e-09a06bc305fe.jpg@s_2,w_820,h_1773" w="937" h="2026" alt="微信自动登录" title="微信 Windows / macOS 客户端已全面支持自动登录！无需手机确认" srcset="https://img.ithome.com/newsuploadfiles/2021/12/bcfc9f11-a73f-4c78-832e-09a06bc305fe.jpg 2x" width="937" height="1773" referrerpolicy="no-referrer"></p><p data-vmark="f747">IT之家获悉，登录 Windows / macOS 微信后，在手机端的微信中，点击“微信已登录”条幅，在出现的页面中，会新增显示“自动登录该设备”选项，用户只需要打勾即可。</p><p data-vmark="9e74">此外，“自动登录”可以在手机端打开或者关闭，在 PC 端，仅有关闭选项。PC 端设置页面显示：“开启后，在本机登录微信将无需手机确认，可在手机和电脑上关闭。”</p><p data-vmark="0dee">用户在 PC 端登录微信后，手机也会发送通知提醒。</p><p data-vmark="a338">各位小伙伴可通过关注IT之家官方微信公众账号并回复“微信”两个字即可获取最新版。</p><p data-vmark="cc2b"><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210316222136_9513.jpg" alt="image.png" title="微信 Windows / macOS 客户端已全面支持自动登录！无需手机确认" referrerpolicy="no-referrer"></p>
