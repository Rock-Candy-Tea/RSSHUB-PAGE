<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 2 日消息 近日微软 Edge 92 版开始向开发频道的测试人员推出，带来了两个新功能：Office 查看器和 Windows 搜索集成。这两个新功能已经测试了几个月，现在更多的用户可以通过安装 Edge Dev 92 来测试它们。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/e701ee95-1d28-4ab2-9396-a4f73b0c1b34.jpg" w="696" h="365" title="微软 Edge 浏览器整合 Office：一键打开 Word、Excel 和 PowerPoint 文档" width="696" height="365" referrerpolicy="no-referrer"></p><p>随着 92 版的推出，微软 Office 文档将整合进入 Edge 浏览器。<span class="accentTextColor">这项名为“Office Viewer”的新功能将允许用户在浏览器中一键打开微软 Word、Excel 和 PowerPoint 文档</span>。这项功能处于早期开发阶段，与 Windows 和 Mac 版本的 Edge 浏览器兼容。</p><p>要启用和使用 Office Viewer，打开 Edge 设置 > 下载，并启用“使用 Office Viewer 在网上快速查看 Office 文件”选项。启用后，用户可以在 Edge 内查看 Office 文件，而不用下载。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/2ead4a1b-b1b2-49bb-ac34-9b2c873b9dfd.jpg@s_2,w_820,h_455" w="1082" h="601" title="微软 Edge 浏览器整合 Office：一键打开 Word、Excel 和 PowerPoint 文档" srcset="https://img.ithome.com/newsuploadfiles/2021/5/2ead4a1b-b1b2-49bb-ac34-9b2c873b9dfd.jpg 2x" width="1082" height="455" referrerpolicy="no-referrer"></p><p>考虑到用户目前需要下载文件并在 Office 应用中打开它们，Edge 中的 Microsoft 365 整合应该能为用户节省一些时间。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/d0965c52-2477-4884-8831-75821701fcc1.jpg" w="589" h="420" title="微软 Edge 浏览器整合 Office：一键打开 Word、Excel 和 PowerPoint 文档" width="589" height="420" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/aa5b7a10-3bb6-4915-b625-45cdb524733a.jpg" w="590" h="420" title="微软 Edge 浏览器整合 Office：一键打开 Word、Excel 和 PowerPoint 文档" width="590" height="420" referrerpolicy="no-referrer"></p><p>另外，Edge 的 Office 查看器还可以保护用户免受通过 Office 文件传递的恶意软件的威胁。</p><p>第二个新功能是 Windows 搜索集成，但它似乎没有在 Windows 10 的新版本中启用。微软正在寻求将 Chromium Edge 与任务栏中的搜索框进行整合。一旦启用，Windows 搜索将从 Microsoft Edge 获取信息。例如，用户将能够启动任务栏搜索，发现用户的浏览历史、最近的标签、书签（收藏夹）等等。</p><p>Windows 搜索目前可以让用户寻找存储在本地存储中的文件和文件夹。用户可以搜索 Bing，企业客户也可以搜索他们组织内的文件。有了这个新功能，微软正在为 Windows 10 增加 Edge 集成。要启用搜索集成，打开 Microsoft Edge > 设置 > 配置文件。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/c9c7515c-44b2-4a54-82b1-8b217a59898e.jpg@s_2,w_820,h_584" w="923" h="657" title="微软 Edge 浏览器整合 Office：一键打开 Word、Excel 和 PowerPoint 文档" srcset="https://img.ithome.com/newsuploadfiles/2021/5/c9c7515c-44b2-4a54-82b1-8b217a59898e.jpg 2x" width="923" height="584" referrerpolicy="no-referrer"></p><p>该功能目前还不能使用，可能需要较新的 Windows 10 版本。</p><p>IT之家了解到，微软也有可能将 Edge 的 Windows 搜索集成作为 Windows 时间轴跨设备同步功能的替代品，该功能将随着 Windows 10 太阳谷更新而消失。</p><p>好的是，这项功能是可选的，如果用户不想让 Windows Search 监控 Edge 的历史，可以把它关掉。</p>
