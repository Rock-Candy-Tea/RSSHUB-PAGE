<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 6 日消息 早在 6 月份，就有消息称微软将在下半年发布两个版本的 Windows，一个是全新的 Win11，另一个是面向企业的 Win10 更新。</p><p>据外媒 Windows Latest 报道，Win10 21H2 也就是下一个 Win10 大版本更新，有望在 <span class="accentTextColor">10 月份发布</span>。该版本除了带来更多<span class="accentTextColor">圆角 UI</span>，还会添加一些小的功能更新，以及一些补丁修复。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/11a864db-4166-4a2e-84fb-b59bc77bb368.jpg" w="696" h="365" title="微软 Win10 21H2 版本号将是 Build 19044，更多新功能曝光" width="696" height="365" referrerpolicy="no-referrer"></p><p>在 Windows 2021 年 5 月 10 日更新（版本 21H2）中，当设备上存在支持 Windows Hello 的外部和内置摄像头时，Microsoft 启用了对多个 Windows Hello 摄像头的支持。</p><p>根据更新的支持文档，从版本 21H2 开始，用户可以在笔记本电脑<span class="accentTextColor">合盖时使用外部 Windows Hello 兼容相机</span>。在另一份文件中，微软表示，英特尔 TPM Tiger Lake 平台将在 Win10 21H2 或更高版本的系统中支持 <span class="accentTextColor">TPM 认证</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/154cb8f3-0623-4952-8354-64632001c9bf.jpg@s_2,w_820,h_409" w="836" h="417" title="微软 Win10 21H2 版本号将是 Build 19044，更多新功能曝光" srcset="https://img.ithome.com/newsuploadfiles/2021/7/154cb8f3-0623-4952-8354-64632001c9bf.jpg 2x" width="836" height="409" referrerpolicy="no-referrer"></p><p>对于企业用户，当使用 WSUS 或 Configuration Manager 时，Win10 21H2 将能够按需管理功能 (FOD) 和语言包。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/59d69a89-5ccb-472c-9089-ffc28a666faa.gif" w="915" h="672" title="微软 Win10 21H2 版本号将是 Build 19044，更多新功能曝光" width="915" height="602" referrerpolicy="no-referrer"></p><p>在 21H2 中，非管理员还可以添加显示语言及其相应的语言功能。此外，消息称微软也在改进<span class="accentTextColor">通用打印和部署功能</span>，如 Windows Autopilot。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/ac062d94-ffc5-4f4f-8c93-55bb4eb33a96.jpg@s_2,w_820,h_641" alt="Windows 10 21H2 Build 19044" w="1026" h="802" title="微软 Win10 21H2 版本号将是 Build 19044，更多新功能曝光" srcset="https://img.ithome.com/newsuploadfiles/2021/7/ac062d94-ffc5-4f4f-8c93-55bb4eb33a96.jpg 2x" width="1026" height="641" referrerpolicy="no-referrer"></p><p>IT之家了解到，Win10 21H2 的版本号将是 <span class="accentTextColor">Build 19044</span>，也是通过 Windows 更新来进行推送。微软已经承诺，将会继续支持 Win10，直到 <span class="accentTextColor">2025 年 10 月 14 日</span>。</p>
