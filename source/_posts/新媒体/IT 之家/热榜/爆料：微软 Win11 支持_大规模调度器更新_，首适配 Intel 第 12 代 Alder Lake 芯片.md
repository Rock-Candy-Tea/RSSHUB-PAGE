<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 5 日消息 外媒 Windows Latest 报道，微软将于 6 月 24 日公布下一版 Windows，有报道称，下一代桌面操作系统将被称为“Windows 11”。</p><p>虽然微软曾在内部将其下一代操作系统称为代号 Sun Valley 和 Cobalt，但在 6 月 24 日正式发布时，它将被简单地称为 Windows 11 或 Windows 10 + something。对于那些已经运行 Windows 10 的用户来说，这将是免费的可选升级，它预计将在今年秋天向主流受众发布。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/ed6a7f63-6bd2-4122-ab0d-1601d8b99c37.png" w="804" h="598" title="爆料：微软 Win11 支持“大规模调度器更新”，首适配 Intel 第 12 代 Alder Lake 芯片" width="804" height="598" referrerpolicy="no-referrer"></p><p>我们对未宣布的 Windows 11 或 Windows Sun Valley 知之甚少，然而，据说它具有全新的“开始”菜单、操作中心、任务栏，以及其他核心组件的视觉改造，如弹窗、上下文菜单、应用程序菜单等。除了用户界面的变化，下一代 Windows 还将配备大规模的调度器更新。</p><p>微软 Windows 配备了系统调度器，这是操作系统的一个模块，它能够选择将被系统处理的下一个任务，或执行的下一个线程。通过确定获得下一个处理线程，控制多任务，提高平台的整体性能。但目前的系统调度器没有针对混合 CPU 进行适当的优化。</p><p>据报道，即将推出的 Windows 大版本将具有“大规模调度更新”，以支持英特尔即将推出的 CPU 芯片。</p><p>爆料人士 Moore's Law is Dead 还声称，英特尔期待已久的第 12 代“Alder Lake”芯片将在今年 10 月与微软的下一代 Windows 一起推出。</p><p>据报道，英特尔 Alder Lake 芯片将引入全新的插槽，它将同时使用普通的 CPU 内核和高效 / 低功耗内核，为笔记本电脑提供更好的性能和更长的电池续航。一些报道称，Alder Lake CPU 将比 Rocket Lake 提供更大的性能升级。</p><p>微软也在与英特尔合作，为“Alder Lake-S”台式机处理器进行性能优化。据报道，Windows 11 或太阳谷将更新调度器，更好地支持混合处理器（为不同任务设计的 CPU 内核）。</p><p>IT之家获悉，微软 CEO 萨蒂亚・纳德拉和 Panos Panay 已经证实，公司很快将分享“Windows 最重要的更新之一”，他们对下一代 Windows 感到无比兴奋。</p><p>随着 Windows 10X 现在被取消，微软的重点已经转移到 Windows 10 桌面系统，以及下一代只针对主流客户的 Windows 版本。</p>
