<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 19 日消息 微软今天宣布，其已不再计划发布 Windows 10X 系统，这一信息是在微软宣布推出新的 Windows 10 21H1 更新的博文中披露的。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/737e6c8a-995b-4ea1-bfa1-5d9998908e37.jpg@s_2,w_820,h_750" w="958" h="876" title="微软官宣：Win10X 系统已死" srcset="https://img.ithome.com/newsuploadfiles/2021/5/737e6c8a-995b-4ea1-bfa1-5d9998908e37.jpg 2x" width="958" height="750" referrerpolicy="no-referrer"></p><p>微软 Windows 服务和交付主管约翰・凯布尔（John Cable）周二写道：“我们没有像最初计划的那样，在 2021 年将名为 Windows 10X 的产品推向市场，而是利用我们迄今旅程中的经验教训，加快将关键的底层 10X 技术整合到 Windows 其他版本和产品中。”</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/da5f36e1-26b6-4732-a99e-66b14849fbdd.jpg@s_2,w_820,h_540" w="1200" h="790" title="微软官宣：Win10X 系统已死" srcset="https://img.ithome.com/newsuploadfiles/2021/5/da5f36e1-26b6-4732-a99e-66b14849fbdd.jpg 2x" width="1200" height="540" referrerpolicy="no-referrer"></p><p><span class="accentTextColor">微软将在 Windows 及其他产品中整合 Windows 10X 的功能</span>，该公司已经开始在 Windows 10 Insider 预览版上添加其中的一些功能，如 Microsoft Defender Application Guard 中的新应用容器技术，增强的语音打字体验，以及具有优化的按键大小、声音、颜色和动画的现代化触摸键盘。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/32643122-f8be-4432-b09b-60745bda4493.jpg@s_2,w_820,h_461" w="1200" h="675" title="微软官宣：Win10X 系统已死" srcset="https://img.ithome.com/newsuploadfiles/2021/5/32643122-f8be-4432-b09b-60745bda4493.jpg 2x" width="1200" height="461" referrerpolicy="no-referrer"></p><p>IT之家了解到，Windows 10X 本来是专门为双屏设备（如 Surface Neo）设计的 Windows 10 的精简版。在今年早些时候，微软宣布其将转而为单屏幕的经济型笔记本电脑设计，甚至在 1 月份推出了其工作演示。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/ed25ab01-12e0-44ee-a1e8-ce58a69a2174.jpg@s_2,w_820,h_547" w="1200" h="800" title="微软官宣：Win10X 系统已死" srcset="https://img.ithome.com/newsuploadfiles/2021/5/ed25ab01-12e0-44ee-a1e8-ce58a69a2174.jpg 2x" width="1200" height="547" referrerpolicy="no-referrer"></p><p>目前，微软正在为 Windows 10 推出最新的 21H1 更新，这是一个相对较小的更新，微软计划在 10 月份推出更大的“太阳谷”更新。</p>
