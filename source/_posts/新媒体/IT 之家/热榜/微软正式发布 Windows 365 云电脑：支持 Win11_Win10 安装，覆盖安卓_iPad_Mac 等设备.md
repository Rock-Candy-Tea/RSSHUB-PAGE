<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 14 日消息 微软今天宣布了云电脑服务，正式名称为 Windows 365，允许客户在云中创建 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 或 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的系统安装，并将操作系统流转向任何设备。Windows 365 服务将于 8 月 2 日上线，旨在将最好的 Windows 带到任何设备上，包括你的 Mac、<a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a>、Linux PC 和 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 设备。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/2f27356d-e6d7-4b3e-abff-e50485ac4299.png" w="936" h="590" title="微软正式发布 Windows 365 云电脑：支持 Win11/Win10 安装，覆盖安卓/iPad/Mac 等设备" width="936" height="517" referrerpolicy="no-referrer"></p><p>IT之家获悉，Windows 365 将允许用户在云端安装他们喜爱的 Windows 应用和体验，并从他们选择的任何设备上访问它们。云电脑将支持“即时启动”，并将通过订阅以统一价格提供。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/90397bbf-19ba-4c98-aa65-a1e39ca3903b.png" w="804" h="435" title="微软正式发布 Windows 365 云电脑：支持 Win11/Win10 安装，覆盖安卓/iPad/Mac 等设备" width="804" height="435" referrerpolicy="no-referrer"></p><p>当然，Windows 365 并不是要取代 Windows 11 或 Windows 10。本地版本的 Windows 仍然被认为是最新的，并将一直可以在本地安装而不需要订阅。Windows 365 只是一项服务，它允许用户通过云端串联 Windows 10 或 Windows 11 的实例，就像你自己的 PC 一样。</p><p>Windows 365 建立在 Azure 虚拟桌面的基础上，其工作方式与微软 XCloud 服务类似，后者允许你从云端串流游戏。用户将能够选择他们的 Windows 365 安装的大小。微软还没有宣布 Windows 365 的计划或价格，但预计该公司将在 8 月 2 日服务上线时宣布。微软在其 Inspire 2021 主题演讲中公布了 Windows 365。</p><p>Windows 365 服务将首先面向企业推出，目前还没有关于 Windows 365 何时或是否推出消费者版本的消息。</p><p>《<a href="https://www.ithome.com/0/562/916.htm" target="_blank">微软发布 Windows 365 云电脑，随时随地用上 Win11/Win10 PC，几分钟就能创建</a>》</p>
