<p data-vmark="9590">北京时间 7 月 9 日消息，韩国是世界上科技最发达的国家之一。在这里，你可以在网上任意冲浪，几乎没有限制。除非，你使用了错误的浏览器。</p><p style="text-align: center;" data-vmark="4b9e"><img src="https://img.ithome.com/newsuploadfiles/2022/7/f08a8345-8a55-4dd2-80f4-ca844a90fd97.png" w="1192" h="791" alt="韩国开发者做的IE墓碑" title="韩国网速全球第一，却对“老掉牙”的 IE 浏览器恋恋不舍" width="1192" height="544" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="edc6">韩国开发者做的 IE 墓碑</p><p data-vmark="47a6">在谷歌 Chrome 上，如果你是韩国大型外资银行的企业客户，你将无法进行在线商业支付。如果你使用的是苹果 Safari，你就无法通过国家文化艺术网站申请艺术家资助。如果你是一家托儿所的经营者，那么你无法通过 Mozilla 的火狐浏览器在卫生福利部的网站办理营业资质。</p><p data-vmark="310b">但是，所有以上情况，你都可以使用微软的 IE 或 IE 内核浏览器完成。</p><p data-vmark="a10e">微软已经宣布，将于 6 月 15 日彻底关闭 IE 浏览器，会在未来几个月开始把用户重定向至更新的 Edge 浏览器。这一声明引发了怀旧情绪。一些用户通过笑话和表情包的形式来纪念互联网过去的岁月。</p><p data-vmark="0627">但是在韩国，IE 并没有成为网络古迹。<span class="accentTextColor">虽然 IE 已经关闭，但仍有少数关键的银行和政府相关任务需要使用它，而这些任务是许多人的生活中不可或缺的。</span></p><h2 data-vmark="0ca5">离不开的 IE</h2><p data-vmark="07b6">IE 从推出到退出已有 27 年的历史。韩国对它的忠诚带有一种严重的讽刺意味：一个以超快宽带和创新设备闻名的国家，却被一款世界上大多数国家早已抛弃、漏洞百出而且不安全的软件所束缚。</p><p data-vmark="a32c">韩国的网速世界领先。根据网速测试网站 Ookla 在 2020 年发布的《速度测试全球指数》，韩国的平均 5G 网速为 121 Mbps，是全球平均网速 35.96 Mbps 的 3.4 倍，位居全球第一。</p><p style="text-align: center;" data-vmark="5725"><img src="https://img.ithome.com/newsuploadfiles/2022/7/5716d5b4-be1e-4bef-ad6b-a528aa02f1f3.jpg" w="300" h="490" alt="韩国5G网速全球第一" title="韩国网速全球第一，却对“老掉牙”的 IE 浏览器恋恋不舍" width="300" height="490" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="b9c4">韩国 5G 网速全球第一</p><p data-vmark="ae19">大多数韩国网站可以在各种浏览器上运行，包括谷歌的 Chrome，该浏览器约占韩国互联网使用量的 54%。根据研究公司 Statcounter 的数据，IE 浏览器的占比不到 1%。然而，在微软宣布关闭 IE 后，一些重要网站在最后一刻展开了争夺，为 IE 之后的日子做准备。</p><p data-vmark="d3e0">今年 5 月，英国渣打银行的韩国分行警告企业客户，他们将需要开始通过“IE 模式”使用 Edge 浏览器，以访问其 Straight2Bank 互联网银行平台。同时，多家韩国政府网站告诉用户，如果他们不切换到 Edge，一些服务可能会面临中断。</p><p data-vmark="36a5">同月，韩国互联网巨头 Naver 着重强调了其鲸鱼浏览器的一个功能：允许访问需要使用 IE 浏览器的网站。Naver 鲸鱼浏览器团队负责人金孝 (Kim Hyo，音译) 透露，该公司最初是在 2016 年添加这一选项的。他原本认为，当微软关闭 IE 后，它就不再被需要了。</p><p data-vmark="499b">然而，随着 IE 关闭的日子越来越近，他意识到一些韩国网站无法及时切换到其他浏览器，所以保留了这个功能，并将其名称改为“IE 模式”。他说，让那些迎合 IE 几十年的网站现代化是一项“艰巨任务”，一些网站“恰好错过了最后期限”。</p><h2 data-vmark="782b">历史因素</h2><p data-vmark="9e16"><span class="accentTextColor">韩国对 IE 浏览器的依赖可以追溯到上世纪 90 年代</span>。当时，韩国成为了使用互联网从事银行业务和购物的先行者。为了保护在线交易，政府在 1999 年通过了一项法律，要求之前需要签名的任何事项都需要加密的数字证书。</p><p data-vmark="4d9a">验证一个人的身份需要连接到浏览器的附加软件，也就是插件。韩国政府授权 5 家公司使用微软的“ActiveX”插件颁发数字证书，但是这个插件只能在 IE 上使用。</p><p data-vmark="74fb">当时，使用微软的插件似乎是一个顺理成章的选择。微软的 Windows 系统在 20 世纪 90 年代统治了 PC 市场，而 IE 利用它与 Windows 的捆绑成为了主导浏览器。由于韩国主要网站需要 IE，其他网站也开始迎合微软浏览器，强化了它的重要性。根据互联网趋势 (Internet Trend) 发布的报告，从 2004 年到 2009 年，IE 在韩国拥有 99% 的市场份额。</p><p data-vmark="c42d">“我们真的是唯一选择。”詹姆斯・金 (James Kim) 这么说，他在 2009 年至 2015 年领导微软韩国业务，现在是首尔美国商会的会长。他说，微软并没有试图阻挠竞争，但是很多事情没有 IE 就“行不通”。</p><p data-vmark="8e37">韩国高丽大学法学教授金基昌 (Kim keecchang) 认为，IE 在 21 世纪初对韩国的控制是如此彻底，以至于大多数韩国人“无法说出其他浏览器的名字”。</p><p style="text-align: center;" data-vmark="a97a"><img src="https://img.ithome.com/newsuploadfiles/2022/7/83f9f45b-cb9f-40a7-9610-c378bb8e1a04.png" w="596" h="394" alt="金基昌为弃用IE发起诉讼" title="韩国网速全球第一，却对“老掉牙”的 IE 浏览器恋恋不舍" width="596" height="394" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="c356">金基昌为弃用 IE 发起诉讼</p><p data-vmark="e64a">2002 年，当金教授在国外教完书回到韩国后，他发现自己的那台运行 Linux 系统和火狐浏览器的电脑根本没法用。Linux 是一款旨在取代 Windows 的开源替代系统。每一年，为了在政府网站上报税，他都会去网吧用 IE 上网。</p><p data-vmark="5c07">2007 年，金教授对韩国金融电信和清算研究所提起了诉讼，后者是政府批准的 5 家发行数字证书的私人企业之一。他认为，该公司不允许使用其他浏览器，这是对他的非法歧视。韩国大约 80% 的数字证书由韩国金融电信和清算研究所颁发。</p><p data-vmark="0b43">在 3 年时间里，金教授接连经历了败诉、上诉失败以及在最高法院的败诉。但他在法庭上开展的诉讼让人们更广泛地注意到韩国系统的缺陷，尤其是在 2009 年的一次网络攻击之后，该攻击利用 ActiveX 在韩国人的电脑上传播恶意软件。</p><h2 data-vmark="5845">智能机时代带来改变</h2><p data-vmark="9f80">随着智能手机的出现，一个以苹果、谷歌系统为基础的行业油然而生。和世界上许多国家一样，韩国开始减少对微软的依赖。2010 年，韩国发布了政府网站应兼容三种不同浏览器的指导方针。但是，<span class="accentTextColor">改变韩国的互联网管道系统并不容易，尤其是在银行和信用卡公司坚守现有系统的情况下。</span></p><p data-vmark="d8cb">随着公众舆论的转变，用户对需要使用 ActiveX 进行网上购物的不便感到愤怒。批评者认为，这个插件未能达到目的，因为外挂软件使用户更不安全。</p><p data-vmark="939c">微软在 2015 年推出了 IE 的替代品 Edge，不再支持 ActiveX。实际上，Chrome 早在三年前就成为了韩国第一大浏览器。</p><p data-vmark="8b2f">2020 年，韩国修改了 1999 年的法律，撤销了对数字证书的要求，此举似乎结束了 ActiveX 和 IE 的历史。同一年，微软开始在其部分在线服务中取消对 IE 的支持。一年后，微软宣布计划让 IE 彻底退出历史舞台。</p><p data-vmark="80fd">尽管世界上很多人都在拿 IE 的退出开玩笑，但一位韩国工程师却用一种更严肃的方式来纪念这一时刻。</p><p style="text-align: center;" data-vmark="e0ea"><img src="https://img.ithome.com/newsuploadfiles/2022/7/8ae3d42f-6e8b-4f21-b1b6-2a71b232279f.png" w="591" h="394" alt="郑基荣为IE立碑" title="韩国网速全球第一，却对“老掉牙”的 IE 浏览器恋恋不舍" width="591" height="394" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="5900">郑基荣为 IE 立碑</p><p data-vmark="d6de">郑基荣 (Jung Ki-young，音译) 今年 39 岁，是一名软件开发者。他在其哥哥位于庆州市的咖啡馆的屋顶上为 IE 立了一座墓碑。庆州位于韩国东南沿海，距离首尔约 170 英里。为此，他花了 330 美元。墓碑上刻着 IE 的标志性“e”标志，还有一句铭文：“它是下载其他浏览器的好工具。”</p><p data-vmark="c366">他说，他也曾对 IE 感到失望，但他觉得这个让这么多韩国人接触到网络的浏览器值得一个体面的告别。</p><p data-vmark="bbb5">“使用 IE 是困难和令人沮丧的，但它也作出了贡献，”郑基荣表示，“抱着‘我们不再需要你了’的态度让它退休，我感觉不太好。”</p>
