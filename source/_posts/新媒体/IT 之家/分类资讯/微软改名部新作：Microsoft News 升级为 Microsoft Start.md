<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 8 日消息 据IT之家网友 @软媒用户 118844 投稿，微软近日推出了<span class="accentTextColor">个性化新闻动态应用 Microsoft Start</span>，该应用是 Microsoft News 和 MSN News 的更名，并且在功能上进一步提升。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/6f7cba39-1afb-4267-ab8c-0ad9bf4f7a2e.jpg@s_2,w_820,h_300" w="1218" h="446" title="微软改名部新作：Microsoft News 升级为 Microsoft Start" srcset="https://img.ithome.com/newsuploadfiles/2021/9/6f7cba39-1afb-4267-ab8c-0ad9bf4f7a2e.jpg 2x" width="1218" height="300" referrerpolicy="no-referrer"></p><p>IT之家了解到，Microsoft Start 将<span class="accentTextColor">聚合来自新闻出版商提供的内容</span>，然后根据用户选择按个性化方式提供内容，目前已推出<a href="http://microsoftstart.com/" target="_blank">网页版</a>和手机 App 版本。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/fb352973-36ff-4713-9240-8b09faa454a8.jpg@s_2,w_820,h_461" w="1280" h="720" title="微软改名部新作：Microsoft News 升级为 Microsoft Start" srcset="https://img.ithome.com/newsuploadfiles/2021/9/fb352973-36ff-4713-9240-8b09faa454a8.jpg 2x" width="1280" height="461" referrerpolicy="no-referrer"></p><p>此外，<span class="accentTextColor">该应用与 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 中的小部件功能相似</span>，几乎相当于是把该功能单独摘了出来。除了新闻，Microsoft Start 还会像以前一样，为用户提供天气、财经、体育、导航等方面的卡片。预计用户在 Win11 或网站上设置的首选项可同步到 App 中。</p><p>▼ 网页版 Microsoft Start</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/f6e46c6a-0b4c-4c83-8a13-79981ffb5c5b.gif" w="800" h="450" alt="Microsoft Start service showing news interests on an Edge new tab page" title="微软改名部新作：Microsoft News 升级为 Microsoft Start" width="800" height="450" referrerpolicy="no-referrer"></p><p>▼ Win11 小部件</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/14db6ae0-fe85-465f-914a-d98c4c642474.png" title="微软改名部新作：Microsoft News 升级为 Microsoft Start" referrerpolicy="no-referrer"></p>
