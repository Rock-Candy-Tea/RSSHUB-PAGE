<p data-vmark="2f57"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 8 日消息，据外媒 TheVerge 报道，谷歌今日发布了名为 Ripple 的开源雷达 API 标准，<span class="accentTextColor">这项标准由福特汽车参与制定，可以用于电子设备以及汽车等产品</span>。</p><p data-vmark="8dcc"><img src="https://img.ithome.com/newsuploadfiles/2022/1/eeef7755-f9d1-4004-ab87-a43a3ebba3af.jpg@s_2,w_820,h_547" w="1200" h="800" title="谷歌发布 Ripple 开源雷达标准，可增强汽车感知力" srcset="https://img.ithome.com/newsuploadfiles/2022/1/eeef7755-f9d1-4004-ab87-a43a3ebba3af.jpg 2x" width="1200" height="547" referrerpolicy="no-referrer"></p><p data-vmark="d931">自从 2015 年起，谷歌便开始了微型雷达芯片的研发，它不仅能够与智能手表通信，还可以计算纸张页数、模拟乐器弹奏等。</p><p data-vmark="2b08">Ripple 项目是由消费者技术协会 (CTA) 赞助的，<span class="accentTextColor">目的是创造一款通用雷达标准，让更多人受益</span>。该项目已经在 GitHub 上线，包含几个实例。想要为项目做贡献的用户，必须与谷歌签署开源许可协议才行。</p><p data-vmark="6d66">IT之家了解到，Ripple 项目的雷达，可以用于室内环境，进行人体检测和避障等。福特表示，雷达可以成为智能辅助驾驶系统的一部分，该公司正在使用先进的外置雷达来研究相关功能。福特研究人员表示，正在使用雷达传感器，<span class="accentTextColor">以增强 Co-Pilot360 驾驶辅助技术</span>。标准 API 可以方便开发者不针对特定硬件开发软件，并使得软件团队的创新能够应用到多个雷达平台。</p><p data-vmark="518b">CTA Ripple 项目 GitHub 页面：<a href="https://github.com/CTA-Ripple" target="_blank">点击打开</a></p>
