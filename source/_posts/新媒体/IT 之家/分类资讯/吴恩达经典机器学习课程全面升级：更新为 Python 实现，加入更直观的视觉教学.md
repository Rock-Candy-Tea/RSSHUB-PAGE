data-vmark="db2a">就在几个月前，吴恩达得了新冠，所幸很快康复；一周多前，他还注册了知乎账号，回答了第一个提问“如何系统学习机器学习？”。</p><h3 data-vmark="ef8f">六月开讲，全部用 Python 实现</h3><p data-vmark="f33d">如果谁要问如何入门机器学习，十个人里有八个都得向他推荐吴恩达的机器学习。这门课程从 2012 年在 Coursera 推出至今，已有来自全球近 500 万的学习者注册学习。</p><p data-vmark="e0fc">它在 Coursera 上的评分高达 4.9 分（一共有近 17 万人打分），好评如潮。而该平台上别的课程要么没有这么高的分数，要么没有这么多的评分人数。</p><p style="text-align: center;" data-vmark="7cd0"><img src="https://img.ithome.com/newsuploadfiles/2022/4/d875eefe-74ec-4334-b206-f81362b2258e.png" w="680" h="368" title="吴恩达经典机器学习课程全面升级：更新为 Python 实现，加入更直观的视觉教学" width="680" height="368" referrerpolicy="no-referrer"></p><p data-vmark="62f1">吴恩达作为机器学历领域的大牛，理想就是让全世界每个人能够接受高质量且免费的教育。他表示，许多学生从该门课程中的收获也让他非常非常开心和感谢，所以，是时候“回馈”下一代学习者了。</p><p data-vmark="f062">于是，他对这门无数人奉为经典的课程进行了重新设计，与斯坦福免费在线课程平台 StanfordOnline 以及 DeepLearningAI 一同推出全新的 Machine Learning Specialization。</p><p data-vmark="79fb">新课程比以往的版本更具有“相关性”（more relevant than ever），并扩展为 3 门，教学和作业全部用 Python 实现代替以往的 Octave。</p><p style="text-align: center;" data-vmark="b586"><img src="https://img.ithome.com/newsuploadfiles/2022/4/e911bfc6-38d0-47b2-875c-9ddc760a779f.png" w="986" h="405" title="吴恩达经典机器学习课程全面升级：更新为 Python 实现，加入更直观的视觉教学" width="986" height="337" referrerpolicy="no-referrer"></p><p data-vmark="8c77">并且还会先用更直观的视觉方法教授大家基本概念后，再引出实现代码，相关基础数学则放在最后，并且为可选课程。</p><p style="text-align: center;" data-vmark="15ac"><img src="https://img.ithome.com/newsuploadfiles/2022/4/705ec91a-08e8-4cfd-b4f5-53466ac0bf9d.png" w="912" h="460" title="吴恩达经典机器学习课程全面升级：更新为 Python 实现，加入更直观的视觉教学" width="912" height="414" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="c48b"><img src="https://img.ithome.com/newsuploadfiles/2022/4/6052496a-a528-4c51-858e-d305d0d3a7af.png" w="1080" h="550" center;" data-vmark="48e0"><img src="https://img.ithome.com/newsuploadfiles/2022/4/4f62e440-1480-456f-b5c7-776b52e7e18b.gif" w="1079" h="529" title="吴恩达经典机器学习课程全面升级：更新为 Python 实现，加入更直观的视觉教学" width="1079" height="402" referrerpolicy="no-referrer"></p><p data-vmark="4a44">对于纯小白来说，没有先验数学知识和系统的编程背景也能听。</p><p data-vmark="faa4">对于学过那门原始机器学习课程但没学完的人来说，它可以在简单有效的学习体验中加强你的代码实践和数学理论，并提供了分难度等级的练习。而已经学完机器学习全部课程的人，也能从这门课中刷新你对基本概念的认知，还可以看一看全新的 Python 实现。</p><p
