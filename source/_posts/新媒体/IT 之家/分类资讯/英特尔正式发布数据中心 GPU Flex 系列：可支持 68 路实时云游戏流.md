<p data-vmark="21ad"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 25 日消息，今日，英特尔宣布推出数据中心 GPU Flex 系列，加速智能视觉云应用，<strong>可提供更好的媒体转码吞吐性能和支持多达 68 路实时云游戏流</strong>。</p><p data-vmark="369e">英特尔表示，Flex 系列 GPU 可满足图像质量、部署密度和时延方面的要求。在<strong>集成了 AV1 编码器</strong>的基础上，Flex 系列数据中心 GPU 可以提供更好的媒体转码吞吐性能和解码吞吐性能。它<strong>还实现了 30% 以上的带宽增幅</strong>，降低了总体拥有成本。在软件层面，它可广泛支持主流媒体工具、API、框架和最新的编解码器。</p><p style="text-align: center;" data-vmark="6433"><img src="https://img.ithome.com/newsuploadfiles/2022/8/1ab65a70-9ea1-4457-9cd0-2fd88f4c150f.png" w="1080" h="609" alt="英特尔数据中心GPU Flex系列" title="英特尔正式发布数据中心 GPU Flex 系列：可支持 68 路实时云游戏流" width="1080" height="462" referrerpolicy="no-referrer"></p><p data-vmark="c970">基于英特尔的 X<sup>e</sup>-HPG 微架构，并由生态系统中的广大硬件供应商和软件开发者提供支持，英特尔数据中心 GPU Flex 系列可以为诸如媒体分析、智慧城市、医疗影像等场景的 AI 推理工作负载提供在 CPU 和 GPU 之间灵活扩展的能力。</p><p data-vmark="74c0">IT之家了解到，英特尔数据中心 GPU Flex 系列<strong>将首先提供媒体处理和传输以及 Android 云游戏相关能力</strong>。</p><p data-vmark="1c2a">媒体处理和传输方面，Flex 系列 GPU 的<strong>媒体架构包含四个 X<sup>e</sup> 媒体引擎</strong>，专注于满足现代数据中心对于高密度视频流的需求：</p><ul class=" list-paddingleft-2"><li><p data-vmark="0f81">单卡能够支持多达 36 路视频流的 1080p60 转码吞吐量。</p></li><li><p data-vmark="14ba">单卡能够支持 8 路视频流的 4K60 转码吞吐量。</p></li><li><p data-vmark="668a">在 4U 服务器中扩展到 10 卡配置时，它可以支持多达 360 路视频流的 HEVC-HEVC 1080p60 转码吞吐量。</p></li><li><p data-vmark="b634">Flex 系列 GPU 基于硬件的开源 AV1 编码器可节省 30% 以上的带宽。Flex 系列 GPU 的媒体引擎还支持包括 HEVC、AVC 和 VP9 在内的业界广泛使用的编解码器并可以提高其性能。</p></li><li><p data-vmark="ff51">利用英特尔 Deep Link 超级编码功能，英特尔数据中心 GPU Flex 系列 140 可在单卡上配备两套编解码设备，可以满足业界的“一秒时延”要求，同时提供 8K60 实时转码。此功能适用于 AV1 和 HEVC HDR 格式。</p></li></ul><p style="text-align: center;" data-vmark="7ed9"><img src="https://img.ithome.com/newsuploadfiles/2022/8/277e9703-5a22-4ef8-a2f5-c900a3b14481.jpg@s_2,w_820,h_375" w="1080" h="494" data-type="jpeg" alt="英特尔数据中心GPU Flex系列" title="英特尔正式发布数据中心 GPU Flex 系列：可支持 68 路实时云游戏流" srcset="https://img.ithome.com/newsuploadfiles/2022/8/277e9703-5a22-4ef8-a2f5-c900a3b14481.jpg 2x" width="1080" height="375" referrerpolicy="no-referrer"></p><p data-vmark="d7c6">Android 云游戏方面，英特尔称 Flex 系列 GPU 可在各种设备上带来畅爽的游戏体验，这一点已经在近 90 款时下热门的 Google Play Android 游戏中得到验证。</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="965c">单个 Flex 系列 170 GPU 可以支持多达 68 路 720p30 数据流，而单个 Flex 系列 140 GPU <strong>可以支持多达 46 路 720p30 数据流</strong>（基于特定游戏的测试结果）。</p></li><li><p data-vmark="5aad">当使用 6 个 Flex 系列 140 GPU 进行扩展使用时，它<strong>可以支持多达 216 路 720p30 数据流</strong>。</p></li></ul><p style="text-align: center;" data-vmark="012d"><img src="https://img.ithome.com/newsuploadfiles/2022/8/6c001c21-8c13-43b9-a0fc-546d1ebb02e9.png" w="1080" h="608" alt="英特尔数据中心GPU Flex系列" title="英特尔正式发布数据中心 GPU Flex 系列：可支持 68 路实时云游戏流" width="1080" height="462" referrerpolicy="no-referrer"></p><p style="text-align: left;" data-vmark="58d6">英特尔指出，包括戴尔科技集团、HPE、新华三、浪潮、联想和超微在内的<strong>全球系统提供商将推出搭载 Flex 系列 GPU 的系统</strong>。在未来几个月里，从媒体传输和 Android 云游戏工作负载开始，搭载 Flex 系列 GPU 的解决方案将陆续面市。随后将是面向 Windows 云游戏、人工智能和虚拟桌面基础架构工作负载的系统和解决方案。</p>
