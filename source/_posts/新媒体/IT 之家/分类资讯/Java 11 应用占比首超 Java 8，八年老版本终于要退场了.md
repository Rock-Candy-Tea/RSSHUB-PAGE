<p data-vmark="6493"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 29 日消息，New Relic 发布了一份《2022 年 Java 生态系统状况报告》，公布了 Java 生态的最新统计数据。</p><p data-vmark="56be">据报告，Java 11 在推出一年多后，终于成为了 Java 生态中应用占比第一的版本，<span class="accentTextColor">超过了八年的经典版本 Java 8</span>。有超过 48% 的应用在生产环境中使用 Java 11（2020 年仅为 11.11%），Java 8 目前占比 46.45%。</p><p data-vmark="0285" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/a21b006f-aa85-44ef-ae12-2d35164c24fb.png" w="877" h="552" title="Java 11 应用占比首超 Java 8，八年老版本终于要退场了" width="877" height="516" referrerpolicy="no-referrer"></p><p data-vmark="89b4">此外，Java 7 将于 2022 年结束支持，但仍有 1.71% 的应用在生产环境中使用 Java 7。同时，更早的 Java 6 仍有 0.27% 的应用正在使用。</p><p data-vmark="f196">与 LTS 版本相比，非 LTS Java 版本的使用率仍然非常低，只有 2.7% 的应用在使用，其中 Java 14 最受欢迎。</p><p data-vmark="cbb4" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/2b345867-32ac-458c-839c-ea0fc553ffc1.png" w="839" h="470" title="Java 11 应用占比首超 Java 8，八年老版本终于要退场了" width="839" height="459" referrerpolicy="no-referrer"></p><p data-vmark="0c36">IT之家了解到，报告称近年来，使用的 Java 开发工具包 (JDK) 发行版的源代码发生了变化，人们越来越倾向于使用 OpenJDK。</p><p data-vmark="98d7">2020 年，甲骨文是最受欢迎的供应商，约占 Java 市场的 75%。而到了今年，<span class="accentTextColor">这一数字降到了 34.48%</span>，亚马逊的市场份额已大幅攀升至 22%（2020 年为 2.18%）。</p><p data-vmark="ce13" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/8d4dfe54-1690-41a4-94f5-e84436b50f73.png" w="878" h="393" title="Java 11 应用占比首超 Java 8，八年老版本终于要退场了" width="878" height="367" referrerpolicy="no-referrer"></p><p data-vmark="a0eb">此外，容器化应用已经成为市场主流，有超过 70% 的 Java 应用运行在容器中。</p><p data-vmark="882d"><strong>《2022 年 Java 生态系统状况报告》：</strong><a href="https://newrelic.com/resources/report/2022-state-of-java-ecosystem" target="_blank">点此查看</a></p>
