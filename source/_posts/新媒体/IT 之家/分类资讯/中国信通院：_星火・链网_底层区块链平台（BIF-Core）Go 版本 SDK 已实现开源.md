<p data-vmark="0dac"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 11 日消息，从中国信通院获悉，近日，<strong>“星火・链网”底层区块链平台（BIF-Core） Go 版本 SDK 已实现开源</strong>，可支持账户服务、合约服务、交易服务和区块服务。</p><p data-vmark="9816" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2021/6/0d3119cb-2601-475e-b655-2d8c7ce6b8f6.png" alt title="中国信通院：“星火・链网”底层区块链平台（BIF-Core）Go 版本 SDK 已实现开源" referrerpolicy="no-referrer"></p><p data-vmark="27c7">中国信通院表示，开发者使用 Go 语言即可与“星火・链网”底层区块链平台（BIF-Core）进行交互，<strong>实现查询交易、部署合约、新建账户等功能</strong>。</p><p data-vmark="10d0">据介绍，<strong>Go 版本 SDK 采用模块服务架构</strong>。业务应用可以集成 SDK，实现与“星火・链网”底层区块链平台 BIF-Core 交互。Go 版本 SDK 包含账户地址和公私钥生成、签名验签、提交交易、部署和调用智能合约、查询区块和链信息等服务接口。</p><p data-vmark="2ef1" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/3567a620-f300-4d48-aa9b-a181ee7e8d11.jpg@s_2,w_820,h_645" w="1080" h="850" alt="Go 版本 SDK 采用模块服务架构" title="中国信通院：“星火・链网”底层区块链平台（BIF-Core）Go 版本 SDK 已实现开源" srcset="https://img.ithome.com/newsuploadfiles/2022/4/3567a620-f300-4d48-aa9b-a181ee7e8d11.jpg 2x" width="1080" height="645" referrerpolicy="no-referrer"></p><p data-vmark="084d" style="text-align: center;">▲ 图源：中国信通院</p><p data-vmark="9953">此外，“星火・链网”底层区块链平台（BIF-Core） Go 版本 SDK 提供了详细的测试示例，开发者可以根据示例，在“星火・链网”平台测试网上完成相应的开发和测试。</p><p data-vmark="dfce">目前，中国信通院已将“星火・链网”底层区块链平台（BIF-Core） Go 版本 SDK 在 Github 上发布，正式面向社区和公众开源。</p><p data-vmark="8b55">IT之家了解到，“<strong>星火・链网”底层区块链平台（BIF-Core）是中国信通院自主研发的许可公有链</strong>，是“星火・链网”核心组成部分。具有自主可控、强安全、高性能、可扩展的技术特点。“星火・链网”底层区块链平台（BIF-Core）于 2021 年 8 月 3 日正式发布，<strong>标志“星火・链网”主链正式启动运行并开始向全球进行服务</strong>。</p><p data-vmark="3c5a">Github 代码库地址：</p><p data-vmark="d28f"><span
