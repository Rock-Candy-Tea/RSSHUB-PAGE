            <p data-vmark="7fa2">IT之家8 月 4 日消息，在微博、微信等各大互联网平台相继展示 IP 属地信息后，小米社区也已经于 7 月 27 日开始展示用户 IP 属地信息。</p><p style="text-align: center;" data-vmark="8b07"><img src="https://img.ithome.com/newsuploadfiles/2022/8/5caaa0c5-bcae-4093-bf13-953481220a99.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzk3LnBuZw==,g_3,x_10,y_10,a_0,t_100" w="631" h="487" alt="图形用户界面, 图表, 应用程序" title="小米社区已上线 IP 属地功能，包括个人主页和评论区" width="631" height="487" referrerpolicy="no-referrer"></p><p data-vmark="41b8">据小米员工介绍，小米社区上线 IP 属地功能是根据互联网用户账号信息管理规定 2022 版的有关规定。该规定指出，互联网信息服务提供者应当在互联网用户账号信息页面展示合理范围内的互联网用户账号的互联网协议（IP）地址归属地信息，便于公众为公共利益实施监督。</p><p style="text-align: center;" data-vmark="960f"><img src="https://img.ithome.com/newsuploadfiles/2022/8/f245076f-30fc-4aee-95b5-b1ba65a7b980.png" w="900" h="804" alt="图形用户界面, 应用程序, 网站" title="小米社区已上线 IP 属地功能，包括个人主页和评论区" width="900" height="733" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="1d85"><img src="https://img.ithome.com/newsuploadfiles/2022/8/d63bc16e-996b-4804-9108-6eac725a8c50.png" w="858" h="1207" alt="图表

低可信度描述已自动生成" title="小米社区已上线 IP 属地功能，包括个人主页和评论区" width="858" height="1154" referrerpolicy="no-referrer"></p><p data-vmark="9706">从图片中可以看出，在小米社区的个人主页中，IP 属地会展示在用户昵称下方。同时，<strong>用户发帖、回复时都会展示用户的 IP 属地。</strong></p><p data-vmark="162c">IT之家了解到，今年 3 月，微博宣布测试开放部分国内用户的 IP 归属地展示。截至目前，今日头条、抖音、知乎、快手、豆瓣、B站等都在个人主页等位置展示了账号 IP 属地。其中微博、抖音、B站还在用户账号主页显示了账号所属 MCN 机构。</p>
