<p data-vmark="c3a1"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 15 日消息，抖音安全中心今日宣布，为维护真实有序的讨论氛围，减少冒充热点事件当事人、恶意造谣、蹭流量等不良行为，<span class="accentTextColor">抖音拟在个人主页等位置展示帐号 IP 属地</span>，相关功能在 <span class="accentTextColor">4 月底</span>进行测试，将根据测试情况逐步在其他场景全量上线。</p><p style="text-align: center;" data-vmark="381d"><img src="https://img.ithome.com/newsuploadfiles/2022/4/8952ea2b-b743-437e-abd4-e9a09c110ab3.png" w="897" h="383" title="抖音拟在个人主页等位置展示帐号 IP 属地，4 月底开启测试" width="897" height="350" referrerpolicy="no-referrer"></p><p data-vmark="296e">抖音安全中心表示，平台展示的帐号 IP 属地为最近一次发文或评论时的网络位置，境内展示到省（区、市），境外展示到国家（地区）。帐号 IP 属地以运营商提供信息为准，<span class="accentTextColor">相关展示不支持手动开启或关闭</span>。</p><p data-vmark="b212">IT之家此前报道，今日头条也宣布拟在个人主页等位置展示帐号 IP 属地，相关功能即日起在网页版进行测试，将根据测试情况逐步在其他场景全量上线。</p><p style="text-align: center;" data-vmark="55b6"><img src="https://img.ithome.com/newsuploadfiles/2022/4/68d758c7-93e5-4a71-8cc8-a151101898ac.png" title="抖音拟在个人主页等位置展示帐号 IP 属地，4 月底开启测试" referrerpolicy="no-referrer"></p><p data-vmark="a16f">目前，字节跳动旗下的另一大平台西瓜视频还没有宣布跟进这一功能。</p><p data-vmark="3ffb">而微博在 3 月份就宣布开测“用户个人资料页展示近期发帖所在地”的功能。将在“个人主页-个人资料”中展示用户最近一个月内最后一次发布博文或评论的 IP 归属地域。IP 归属地域显示规则与评论区当前显示规则保持一致。此功能在测试阶段，仅对部分用户开启测试，主动开启 / 关闭功能目前暂不开放。</p><p style="text-align: center;" data-vmark="682a"><img src="https://img.ithome.com/newsuploadfiles/2022/3/74f0197b-ae37-4043-a834-a052c066d4a3.jpg" title="抖音拟在个人主页等位置展示帐号 IP 属地，4 月底开启测试" referrerpolicy="no-referrer"></p><p style="text-align: justify;" data-vmark="3556">《<a href="https://www.ithome.com/0/613/185.htm" target="_blank">微博之后，今日头条也将在个人主页等位置展示帐号 IP 属地</a>》</p>
