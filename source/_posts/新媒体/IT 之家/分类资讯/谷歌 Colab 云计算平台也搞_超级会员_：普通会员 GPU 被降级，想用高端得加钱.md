<p>在 Colab 成为付费会员后，就一定能用上 V100 和 P100？</p><p>Nonono，现在你可能会“碰巧”用上 Tesla T4！</p><p>前段时间，<span class="accentTextColor">Colab 上线了一个 50 美元/月（约 325 元/月）的“超级会员”Pro+</span>，和 10 美元/月的“普通会员”Pro 相比，Pro + 能优先用上 V100 和 P100 GPU。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/0d95853c-8de1-4098-9466-94076d915b42.png" w="1080" h="582" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱" width="1080" height="442" referrerpolicy="no-referrer"></p><p>现在，有 Reddit 网友发现，以前总能抢到 V100 和 P100 的 Pro 会员，甚至可能被分配到 T4。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/5ce6734e-d201-432b-9037-9b51cd578092.png" w="718" h="750" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱" width="718" height="750" referrerpolicy="no-referrer"></p><p>▲ 图源：Reddit</p><p>要知道，Tesla T4 可是 Colab 免费玩家也能“碰运气”获得的算力！</p><p>对于 Colab 用户来说，充钱买 Pro 本身就是想要更好的算力，T4 显然不符合他们对于 GPU 的期待。</p><p>话题一上 Reddit，立即炸出了一波网友。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/37327762-e4eb-4369-b1e4-7091fd06dcab.png" w="1080" h="159" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱" width="1080" height="121" referrerpolicy="no-referrer"></p><h2>使用优先级的问题</h2><p>网友们在登录 Colab 后发现，事情没有想象得这么糟糕。</p><p>不少人去试了试，发现搞到一个 P100 GPU 还是很容易的，应该不是想象中的“官方 Pro 降级”。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/6634a680-d42b-401c-bc49-1484b20916e1.png" w="1080" h="780" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱" width="1080" height="592" referrerpolicy="no-referrer"></p><p>“Pro 会员拿到 T4”的具体原因，可能只是恰好赶上了使用高峰期；此外，如果经常在高峰期大量使用 GPU，会员的优先级也会下降。</p><p>具体来说，现在谷歌 Colab 上有几种 GPU 可供选择：K80、T4、P100、V100……</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/c5a8f3e4-5290-4768-a5d3-63deacb61b26.png" w="1034" h="808" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱" width="1034" height="641" referrerpolicy="no-referrer"></p><p>但这并不意味着某个会员就一定能用上某个型号的显卡。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/0928da71-6e0b-4ba8-838f-a9888ff25a6d.png" w="1080" h="186" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱" width="1080" height="141" referrerpolicy="no-referrer"></p><p>例如，免费玩家通常会被分配到 K80，但运气好的话也能搞到 Tesla T4；</p><p>Pro 玩家可以被分配到 P100，偶尔也会分配到 V100 或者 T4；</p><p>Pro + 玩家可以被分配到 V100，偶尔也会分配到 P100（目前还没有 Pro + 用户表示自己被分配到 T4）。</p><p>不过，现在 Pro 会员想要搞到 V100，确实没那么容易了。</p><p>在 V100 刚出来的时候，Pro 会员基本都能用上，Colab 也一度被认为是“真香”的云计算资源。</p><p>随着用户数量增多、前段时间又升级了“超级会员”Pro+，现在 Pro 用户基本抢不到 V100 了，正常就只能用 P100 甚至偶尔用上 T4。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/5fac8377-9425-4e85-979c-db822ca26a33.png" w="1080" h="184" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱" width="1080" height="140" referrerpolicy="no-referrer"></p><p>对于这种现状，有人替谷歌说话，认为 Colab 现阶段能提供的所有 GPU 都已经很不错了。要是真的有商业项目需要用到更多 GPU 的话，肯定是花钱更好。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/0b53570e-edde-454e-85bf-ba65d744971d.png" w="1080" h="218" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱" width="1080" height="166" referrerpolicy="no-referrer"></p><p>但也有不少用户抱怨称，原来免费玩家甚至能获得 P100，现在基本只能用上 K80 了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/343c769b-a846-47fa-830c-13e6110eb05c.png" w="1080" h="282" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱" width="1080" height="214" referrerpolicy="no-referrer"></p><h2>Colab 还香不香？</h2><p>除了 Colab 以外，现阶段确实还有不少免费的 GPU 资源提供。</p><p>例如，Kaggle 每周提供 30 小时的 TPU 资源，以及至少 30 小时的 GPU 资源；</p><p>又例如，AI Studio 每天免费提供 8 张算力卡，具体消耗情况来看，16GB 显存每小时消耗 0.5 张；32GB 显存每小时消耗 1 张，4×32GB 显卡每小时消耗 4 张。</p><p>至于付费的 GPU 资源，同样也有不少。</p><p>有网友已经改用了 Saturn Cloud，表示它“虽然有点贵，但更可靠也更易用”。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/1be807f9-e8a4-4c95-bb40-a42d4fbdbcbd.png" w="1080" h="241" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱" width="1080" height="183" referrerpolicy="no-referrer"></p><p>除此之外，类似于 Sagemaker、Paperspace Gradient 也是不错的选择。</p><p>例如，在 Paperspace Gradient 上，G1 会员 8 美元/月，免费租用 6 小时 GPU，或是以 2.3 美元/小时的价格租用 V100，同时提供 200GB 存储和 5 个并行 notebook。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/4306cb68-113a-40fd-877f-59ea826cafee.png" w="508" h="488" title="谷歌 Colab 云计算平台也搞“超级会员”：普通会员 GPU 被降级，想用高端得加钱"
