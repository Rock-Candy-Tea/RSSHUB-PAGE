<p>IT之家 7 月 1 日消息 据 Macworld 报道，<span class="accentTextColor">苹果的 OS X 10.7 Lion 和 OS X 10.8 Mountain Lion 系统原本需要付费 19.99 美元才能下载</span>，<span class="accentTextColor">不过苹果最近取消了收费</span>，使用旧版 Mac 电脑的用户现在可以免费下载。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/88e2e0ad-1005-49e9-9526-46ab0757e1d4.png" w="687" h="373" title="苹果对 OS X Lion 和 Mountain Lion 系统取消了收费：现可免费下载" width="687" height="373" referrerpolicy="no-referrer"></p><p>苹果一直为不能更新系统的旧 Mac 电脑提供 OS X 10.7 Lion 和 OS X 10.8 Mountain Lion 系统，有需要的用户可以从苹果官网 OS X Lion 和 OS X Mountain Lion <a href="https://support.apple.com/kb/DL2076?locale=en_US" target="_blank">支持文档</a>中下载。</p><p>IT之家了解到，Mac OS X Lion 系统与配备 Intel Core 2 Duo、Core i3、Core i5、Core i7 或 Xeon 处理器，至少 2GB RAM 和 7GB 存储空间的 Mac 兼容。</p><p>而 OS X Mountain Lion 系统与以下 Mac 兼容：</p><ul class=" list-paddingleft-2"><li><p>iMac（2007-2020 年中）</p></li><li><p>MacBook（2008 年末铝制，或 2009 年初或更新）</p></li><li><p>MacBook Pro（2007 年中/末或更新机型）</p></li><li><p>MacBook Air（2008 年末或更新机型）</p></li><li><p>Mac mini（2009 年初或更新）</p></li><li><p>Mac Pro（2008 年初或更新）</p></li><li><p>Xserve（2009 年初）</p></li></ul><p>苹果对早期的 Mac 系统一直收费 19.99 美元，<span class="accentTextColor">直到 OS X 10.9 Mavericks 的推出，苹果才开始免费提供 Mac 系统更</span>新。</p>
