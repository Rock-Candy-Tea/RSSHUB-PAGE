            <p data-vmark="47b0"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 6 日消息，在 12 月 3 日的“为未来而同心 —— 操作系统生态大会 2021”上，统信软件正式推出“<span class="accentTextColor">毕昇驱动引擎方案</span>”。</p><p data-vmark="8bfc">据介绍，目前，统信软件已经与国内外各大打印机厂商形成了长效对接机制并持续推进适配计划，但是在个别项目中由于部分国外机型适配排期比较长、旧款打印机无原厂适配驱动等原因，无法满足客户使用需求。</p><p data-vmark="b81b"><span class="accentTextColor">毕昇驱动引擎方案可以直接在统信 UOS 上运行 Windows 的打印机驱动</span>，而无需安装 Windows 虚拟机或者采购虚拟化解决方案，极大地扩展了统信 UOS 的驱动适配生态，在国内和国际上均属于首创。</p><p data-vmark="c438">IT之家了解到，<span class="accentTextColor">毕昇驱动引擎方案现已支持 2000 余款 Windows 打印驱动</span>，覆盖佳能、惠普、爱普生、兄弟、奔图等 20 多个主流打印机品牌，加上原生厂商适配打印机驱动，统信 UOS 操作系统可满足 90% 以上打印机新旧设备型号。</p><p data-vmark="5f36">技术上，毕昇驱动引擎不但实现了 Windows 驱动运行，<span class="accentTextColor">还可以同时支持 x86、arm 和 loongarch 三种架构</span>，兼容统信 UOS 操作系统所有版本，后续新发系统版本默认集成毕昇引擎环境，历史系统版本提供毕昇引擎环境包进行支持。</p><p data-vmark="2149"><img src="https://img.ithome.com/newsuploadfiles/2021/12/f06f5ebb-d461-468f-93fe-1eff880797f9.jpg" w="750" h="1334" title="统信 UOS“毕昇驱动引擎”正式发布，可直接运行 Windows 的打印机驱动" width="750" height="1334" referrerpolicy="no-referrer"></p><p data-vmark="8bd4">▲ 毕昇驱动引擎</p><p data-vmark="dd83">通过毕昇引擎适配的 Windows 打印驱动进入驱动仓库后，可以通过系统的打印管理器自动识别安装，可以支持 USB 连接打印机直接打印，也支持网络打印机进行打印，使用体验与 Windows 原生驱动无差异。</p><p data-vmark="76fa"><img src="https://img.ithome.com/newsuploadfiles/2021/12/7f0ab404-f51b-46bf-b31e-f56cc077f20a.png" w="1080" h="752" title="统信 UOS“毕昇驱动引擎”正式发布，可直接运行 Windows 的打印机驱动" width="1080" data-vmark="4669">毕昇驱动引擎将在 <span class="accentTextColor">12 月 31 日</span>随着统信桌面操作系统 V20（1043）正式推送。</p>
