<p data-vmark="36b4">昨夜，<span class="accentTextColor">Rust 编程语言的审核团队突然宣布辞职，且即刻生效</span>。他们在 GitHub 的 pull request 上发出了该声明。</p><p data-vmark="4130"><img src="https://img.ithome.com/newsuploadfiles/2021/11/aca35175-3eee-4731-88e7-091c40e12124.png" w="1080" h="540" title="Rust 程序员一觉醒来都懵了，审核团队集体辞职" width="1080" height="410" referrerpolicy="no-referrer"></p><p data-vmark="4260">审核团队成员 Andrew Gallant 在声明中写道：<strong>审核</strong><span class="accentTextColor">团队辞职是为了抗议核心团队，后者认为自己不对除自己以外的任何人负责</span>。</p><p data-vmark="36a4"><img src="https://img.ithome.com/newsuploadfiles/2021/11/9521a64a-a0da-4d90-8c03-e78988208848.png" w="1080" h="462" title="Rust 程序员一觉醒来都懵了，审核团队集体辞职" width="1080" height="351" referrerpolicy="no-referrer"></p><p data-vmark="67c3">辞职声明来得如此突然，措辞又比较严厉，广大程序员看到这则声明后“一脸懵逼”，而又不免好奇。</p><p data-vmark="4226">由于目前该 pull request 页面已锁定，他们只能移步至 Reddit 的 Rust 子版块发帖讨论。</p><p data-vmark="4e35">这一突如其来的事件迅速成了国外程序员圈最热话题。</p><p data-vmark="6c65"><img src="https://img.ithome.com/newsuploadfiles/2021/11/90ef30df-744f-4b3b-83fd-be5c98f0be81.png" w="1080" h="290" title="Rust 程序员一觉醒来都懵了，审核团队集体辞职" width="1080" height="220" referrerpolicy="no-referrer"></p><p data-vmark="0321">Rust，这个连续 6 年获得程序员最喜爱的编程语言，其内部团队出现了哪些裂痕？</p><h2 data-vmark="610f">到底发生什么事了？</h2><p data-vmark="fb54">“求内幕”成了社区吃瓜群众最关心的话题，但没有人在帖子中透露细节。</p><p data-vmark="a05f">不过，从撰写辞职信的 Andrew Gallant 社交网络上能找到一点蛛丝马迹。</p><p data-vmark="11d2">原来双方的冲突由来已久，并不是一朝一夕间突然爆发的。<span class="accentTextColor">尤其是亚马逊 AWS 的介入，让双方冲突进一步加剧</span>。</p><p data-vmark="5db6">今年 2 月，Rust 基金会成立，亚马逊 AWS、谷歌、华为、微软、Mozilla 五家公司为创始成员。</p><p data-vmark="666f">今年 9 月，AWS 团队开源战略与营销负责人 Matt Asay 在 InfoWorld 上发表了一篇文章《Rust 的未来》。</p><p data-vmark="e872"><img src="https://img.ithome.com/newsuploadfiles/2021/11/2e79741d-6c66-4a9b-a856-6a75993f2f41.png" w="1080" h="649" title="Rust 程序员一觉醒来都懵了，审核团队集体辞职" width="1080" height="493" referrerpolicy="no-referrer"></p><p data-vmark="b08b">Rust 核心团队成员 Steve Klabnik 转发了这篇文章，他写道：</p><blockquote><p data-vmark="b975">我拒绝让亚马逊定义 Rust。</p></blockquote><p data-vmark="e7de"><img src="https://img.ithome.com/newsuploadfiles/2021/11/f4580886-5383-45ed-877f-adab93f852a1.png" w="1080" h="1011" title="Rust 程序员一觉醒来都懵了，审核团队集体辞职" width="1080" height="768" referrerpolicy="no-referrer"></p><p data-vmark="fba7">Steve Klabnik 还认为，亚马逊正在逐渐侵蚀 Rust 团队，主要的表现是：</p><ul class=" list-paddingleft-2"><li><p data-vmark="6c57">1、亚马逊雇佣了 Rust 团队的几名成员，其中包括编译器团队负责人、语言团队负责人。</p></li><li><p data-vmark="cf52">2、亚马逊决定不设立 Rust 基金会 ED，这意味着主席将在 Rust 基金会中拥有巨大的权力。</p></li></ul><p data-vmark="f104"><span class="accentTextColor">他认为，亚马逊正在采取措施将核心团队边缘化</span>。</p><p data-vmark="e847">后来文章被 Matt Asay 修改，并在末尾附注声明：</p><p data-vmark="1b8f">这篇文章的前一个版本正确地指出 Rustacean 原则是以亚马逊原则为蓝本的，但无意中可能暗示亚马逊在某种程度上对 Rust 的开发负责。亚马逊雇佣了多名 Rust 维护者和贡献者，但它只是众多有员工参与的公司之一。</p><p data-vmark="d20b">Rust 库团队负责人 Mara Bos 的观点却与之相反，她认为 Steve 的观点“简直是胡说八道”。</p><blockquote><p data-vmark="64fa">Steve 说的“核心团队”变得不那么重要了，这是绝对正确的。但这并不是因为亚马逊正在接管。</p><p data-vmark="e1ba">我认为，既不该让核心团队，也不该让亚马逊团队来领导或定义 Rust。</p></blockquote><p data-vmark="11f8">Andrew Gallant 最后两条 Twitter 便是转发 Mara Bos 的观点。</p><p data-vmark="f055"><img src="https://img.ithome.com/newsuploadfiles/2021/11/0ae58ee1-c718-433e-8768-bcb5e5bce017.png" w="1080" h="824" title="Rust 程序员一觉醒来都懵了，审核团队集体辞职" width="1080" height="626" referrerpolicy="no-referrer"></p><p data-vmark="9e67">根据 Rust 官方的《行为守则》，Rust 审核团队的作用是帮助“遵守行为守则和社区标准”，而现在他们无法做到这一点，因为核心团队的权利似乎超出了界限。</p><p data-vmark="314b">既然如此，审核团队自然也就无法约束核心团队。看来是双方在权利和责任问题上发生了争吵。而 AWS 的问题不过是双方开始角力的导火索。</p><p data-vmark="fe1b">另一位“局外人”System Initiative 公司 CEO 也发表了自己的看法：</p><p data-vmark="c62b">他认为 Rust 核心团队权力过大，核心团队管理着社区的各个方面，但是他们之外的人却无法决定谁加入核心团队，或者被踢出核心团队，从而在 Rust 社区形成了一种“寡头政治”。</p><p data-vmark="77d5"><img src="https://img.ithome.com/newsuploadfiles/2021/11/537d97be-398f-4db5-a1cf-8a94099bf579.png" w="1080" h="522" title="Rust 程序员一觉醒来都懵了，审核团队集体辞职" width="1080" height="396" referrerpolicy="no-referrer"></p><p data-vmark="c2b4">截至目前，<span class="accentTextColor">Rust 临时团队紧急成立，接管辞职三人的工作</span>。</p><p data-vmark="64d8"><img src="https://img.ithome.com/newsuploadfiles/2021/11/24939e76-cda7-442a-b0a2-a426cb143037.png" w="1080" h="434" title="Rust 程序员一觉醒来都懵了，审核团队集体辞职" width="1080" height="330" referrerpolicy="no-referrer"></p><p data-vmark="6263">由于涉事双方都没有给出事件的细节，网友只能凭借靠猜测来“吃瓜”。</p><p data-vmark="fa34">一些程序员认为，Rust 团队的“瓜”貌似比其他编程语言社区多出不少。</p><p data-vmark="7dd8">从 2016 年开始，Rust 在每年的 Stack Overflow 年度评选中都当选为最受欢迎编程语言，而太多的冲突戏份对于技术团队而言可能并不是什么好事。</p><p data-vmark="9dd7"><strong>参考链接：</strong></p><p data-vmark="2e54">[1]<span
