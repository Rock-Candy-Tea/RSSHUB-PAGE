<p>Gurman：苹果的最终目标是将所有产品都引入 Face ID 功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/ad4bddc4-f2a5-429f-a310-0c92d2b9d0f5.png" w="900" h="383" title="苹果大力推广，Face ID 真的是“终极方案”吗" width="900" height="349" referrerpolicy="no-referrer"></p><p>7 月 26 日消息，据外媒报道，昨天，彭博社记者 Mark Gurman 在其最新一期的 Power On 通讯中表示，苹果计划在未来 "几年内" 将 Face ID 引入 Mac。</p><p>Gurman 在通讯中说，他相信苹果的最终目标是将所有产品都引入 Face ID 的功能，包括 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> SE 和 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad Air</a> 等目前仍采用 Touch ID 功能的低端机型。</p><p>同时他也强调，在当前这个阶段，Touch ID 仍然是苹果产品的重要组成部分，特别是对于低端机型，因为它是相对于 Face ID 的“更便宜的选择”，同时也能继续为用户提供安全保障。</p><h2>01.“Face ID 上 Mac”已经流传了 4 年</h2><p>自从 2017 年 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone X</a> 携同 Face ID 出现以来，就一直存在“Face ID 明年就能上 Mac”的流言。这个流言也代表了部分“果粉”们的心愿，毕竟与 Touch ID 相比，Face ID 最大的优点之一是安全。据苹果官方称 Touch ID 的安全性为五万分之一，而 Face ID 的安全性高达百万分之一。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/be446b66-518f-4deb-9726-3143ce4a628c.jpg" w="400" h="300" title="苹果大力推广，Face ID 真的是“终极方案”吗" width="400" height="300" referrerpolicy="no-referrer"></p><p>▲ Face ID</p><p>2018 年虽然 Face ID 仍旧没有上 Mac，但是出现在了 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad Pro</a> 上，这让部分“果粉”很兴奋，觉得 Mac 也不远了。但这之后就再也没有消息。</p><p>2020 年，外媒“9To5Mac”在 macOS Big Sur beta 3 的系统中发现了“PearlCamera”的代码，该字符是苹果 TrueDepth 摄像头和 Face ID 的内部代号。除此之外，他们还发现了诸如“FaceDetect”和“BioCapture”这样的代码，这些字符都印证了苹果正在尝试将 Face ID 系统引入到 Mac 设备上。</p><h2>02.2023 年的 iPhone“没刘海”？</h2><p>Gurmam 在通讯中表示，今年 Face ID 仍旧不会登入 Mac，但他“打赌”将在几年内实现。他表示，面部识别传感器为苹果提供了两个核心功能：安全性和增强现实。Touch ID，只能做到第一点，而做不到第二点。</p><p>他预计在几年内，所有 iPhone 和 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 也将全部搭载 Face ID。而且，会采用内置在屏幕上的摄像头（embed Face ID into the screens）并消除顶部“刘海”。苹果分析师郭明錤预测，iPhone 明年可能将采用挖孔屏，后年才会采用屏下镜头，也就是说 2023 年苹果的“刘海”才可能完全消失。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/cf582243-3435-4c6c-93b9-a1ffb3856008.jpg" w="576" h="300" title="苹果大力推广，Face ID 真的是“终极方案”吗" width="576" height="300" referrerpolicy="no-referrer"></p><p>▲“挖孔屏”iPhone（图源自网络）</p><p>Gurman 此前曾报道称，苹果原本计划在近期推出包含 Face ID 的 24 英寸 iMac，但因为 Face ID 实现困难并没有成行。Gurman 指出，与 iPhone 和 iPad 不同，Mac 笔记本电脑的屏幕会薄得多，因此更难安装面部识别所需的深度传感器。</p><p>至于 Mac 台式机，苹果公司正在开发一款替代 27 英寸 iMac 的产品，这款产品可能拥有更大的屏幕和“更强大”的 M2X 芯片。今年早些时候，苹果暂停了更大尺寸的 iMac 的研发工作，转而专注于今年 4 月发布的 24 英寸 iMac。</p><h2>03. 结语：Face ID 会是“终极方案”吗？</h2><p>Gurmam 在通讯中表示，今年的 Mac 仍旧没有 Face ID 的功能，但他“打赌”几年内就能实现，并且几年内手机和 iPad 将全面搭载 Face ID。</p><p>Face ID 相比于 Touch ID 确实有着更好的安全性，但其是否会是手机解锁的完美方案？拍摄时姿态的变化、服装的遮挡因素（例如口罩）和对计算能力要求高，都有可能给 Face ID 识别带来难题。是否会有一种生物体感的识别，能够更简化的进行用户辨认？</p>
