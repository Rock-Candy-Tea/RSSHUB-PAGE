<p data-vmark="3378">IT之家8 月 29 日消息，成立 47 年以来，微软发布了许多重量级软件，但就物理重量来说，微软发布的最大的软件是一款编译器。</p><p style="text-align: center;" data-vmark="05e8"><img src="https://img.ithome.com/newsuploadfiles/2022/8/9683be83-8e8a-4f89-86a8-42eb4d349761.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzgyLnBuZw==,g_3,x_8,y_8,a_0,t_100" w="533" h="688" alt="图片 1" title="微软历史上发布的“最重磅”软件：重量超过 36 斤的编译器" width="533" height="688" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="47e7"><iframe class="ithome_video bilibili" src="https://player.bilibili.com/player.html?bvid=bv1DV4y1H7N8&page=1" frameborder="no" width="100%" allowfullscreen="true"> </iframe></p><p data-vmark="5aa2">8 月 19 日，微软 Windows 开发者文档团队向微软档案管理员艾米・史蒂文森（Amy Stevenson）问到：“微软历史上发布的最大软件是什么？” 史蒂文森表示，微软最大的软件是 1992 年发布的带有 Windows SDK 的 Microsoft C / C++ 编译器，该编译器让开发人员能够开发 MS-DOS、Windows 和 OS / 2 应用程序。<strong>该编译器装在一个超过 2 英尺（约 61 厘米）长的盒子中</strong>，<strong>重量超过 40 磅（约 18 千克）</strong>。</p><p style="text-align: center;" data-vmark="4662"><img src="https://img.ithome.com/newsuploadfiles/2022/8/858a04e4-4f25-45ef-ad21-70af1fe6cfc1.png" w="1440" h="790" alt="图片 2" title="微软历史上发布的“最重磅”软件：重量超过 36 斤的编译器" width="1440" height="450" referrerpolicy="no-referrer"></p><p data-vmark="4be3">目前，我们使用的软件都是在网络中下载来的，可以说是没有重量。但在此前互联网并不发达时，用户想要获得软件，需要通过购买软盘或者 CD，并且官方会在其包装盒中附上详细的说明书。据介绍，微软的 Microsoft C / C++ 编译器包装中除了 20 多张存放软件的软盘外，还有 20 余本说明书。史蒂文森表示，微软此后再没有发布过如此重的软件。</p><p data-vmark="48fe">IT之家了解到，1993 年，微软推出了 Visual C++ 来取代 C / C++ 编译器。据悉，该编译器重量大大减少，“仅”重 33 磅（约 14 千克）。</p>
