<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 1 日消息 上个月，外媒 Windows Latest 报道微软正在为 Win10 任务栏测试推荐 Edge PWA 应用，用户可在任务栏看到一个新的弹窗，鼓励用户尝试 Edge 驱动的 PWA 网页应用。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210411_210610_701.jpg" title="微软 Edge 浏览器将推荐用户固定 PWA 应用到 Win10 任务栏" referrerpolicy="no-referrer"></p><p>在最新的 Edge 浏览器 Canary 版本中，微软添加了一项名为“Web Apps Post Install Dialog”的功能，该功能可在用户打开最近安装的 PWA 应用时，<span class="accentTextColor">自动在右上角弹出一个窗口，建议用户将该 PWA 应用添加到任务栏和开始菜单</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/806be3bd-ded7-47c2-adf3-82da6962364c.jpg@s_2,w_820,h_632" w="1053" h="812" title="微软 Edge 浏览器将推荐用户固定 PWA 应用到 Win10 任务栏" srcset="https://img.ithome.com/newsuploadfiles/2021/5/806be3bd-ded7-47c2-adf3-82da6962364c.jpg 2x" width="1053" height="632" referrerpolicy="no-referrer"></p><p>IT之家了解到，微软还允许 Edge 浏览器 PWA 应用<span class="accentTextColor">创建桌面快捷方式，并设置开机自启动</span>。</p><p>此外，PWA 应用的体验将向传统桌面应用靠拢，微软正在测试为 PWA 应用的标题栏进行简化，使其更像桌面应用。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/96255863-1c1a-40db-b1f1-a7525f972cba.jpg@s_2,w_820,h_148" w="923" h="167" title="微软 Edge 浏览器将推荐用户固定 PWA 应用到 Win10 任务栏" srcset="https://img.ithome.com/newsuploadfiles/2021/5/96255863-1c1a-40db-b1f1-a7525f972cba.jpg 2x" width="923" height="148" referrerpolicy="no-referrer"></p>
