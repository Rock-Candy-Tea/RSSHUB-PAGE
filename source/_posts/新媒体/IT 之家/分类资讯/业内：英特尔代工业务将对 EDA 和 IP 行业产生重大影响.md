<p data-vmark="ab7a">近日，海外电子工程类媒体 edn.com 对英特尔代工厂将如何影响 EDA 和 IP 行业做了分析。</p><p data-vmark="67c0" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/65d85545-cdc8-4f96-af24-6abe70df6287.png" w="425" h="239" title="业内：英特尔代工业务将对 EDA 和 IP 行业产生重大影响" width="425" height="239" referrerpolicy="no-referrer"></p><p data-vmark="4194">文章指出，英特尔曾表示，一个开放、繁荣的生态系统对其“IDM 2.0”战略至关重要。因此，英特尔代工服务总裁 Randhir Thakur 阐述，为了加速创新，IDM 2.0 旨在汇集业界顶端人才、发挥广泛能力，实现与英特尔的工艺和封装技术的无缝对接。</p><p data-vmark="3a03">随着晶圆厂业务的竞争不断升温，Pedestal Research 研究总监 Laurie Balch 表示，行业内部竞争越激烈，产品的稳定性和客户的确定性越强：“<span class="accentTextColor">半导体生态系统无疑将从英特尔的代工业务中受益</span>。”</p><p data-vmark="83c5">她说，首先，英特尔对代工模式的涉足将促进集成电路设计和制造的不同要素之间的整合。“英特尔加强其代工厂业务，为各种综合服务产品打开了大门，因为从设计到制造的技术对英特尔来说都游刃有余。”</p><p data-vmark="55f1">Balch 补充说，尽管英特尔一直与 EDA 工具供应商、制造设备公司、IP 供应商和其他半导体技术企业保持合作，但这种联盟和合作关系肯定会越来越多，越来越有价值。</p><p data-vmark="0077">随着英特尔的加入，代工服务上升到了新高度，业界可能会看到 EDA 行业与代工厂的关系发生微妙的转变。Balch 指出，英特尔是 EDA 供应商的顶级客户和开发伙伴，与 EDA 业务紧密交织在一起。此外，英特尔内部有 CAD 团队，甚至自己也销售一些利基型 EDA 工具。</p><p data-vmark="e101">Mentor Graphics Calibre 设计解决方案公司高级总监 Michael Buehler-Garcia 也认为，为了应对市场挑战，需要 EDA、IP 和代工厂公司之间进行更多的合作和沟通：“三方都能从中受益，帮助他们的共同客户更快地推出创新”。</p><p data-vmark="dc22">“我们相信，将有办法与我们的代工伙伴分享硅片性能的相关验证数据，使他们能够进一步调整和改进制造工艺”，Garcia 补充说。</p><p data-vmark="ecfc">代工厂对支持越来越多的工艺节点的 IP 的需求将变得更加迫切。在过去的十年里，正如 Arteris IP 公司的首席运营官 Laurent Moll 所指出：“IP 生态系统将随着代工厂数量的增长而轻松扩展。”</p><p data-vmark="cc79">随着代工厂数量的增加，<span class="accentTextColor">各 IP 供应商之间的合作也可能越来越密切</span>，Moll 说：“一系列代工工艺节点服务将有可能导致一些 IP 供应商获得部分投资组合，与合作伙伴一起为客户提供完整的解决方案。特别是 28 纳米以下节点，优化 IP、调整 EDA 产品和流程与开发新节点一样重要”。</p><p data-vmark="cc47">Balch 也认为：“随着英特尔作为代工企业的地位不断提升，<span class="accentTextColor">EDA 和 IP 供应商将被迫与更多的代工伙伴建立密切的关系</span>。”她指出，另一方面，这也为英特尔提供了发展自己的目标 IP 和 EDA 业务的重要机会，可能会威胁到这些行业的现有公司。“不过从长远来看，通过促进良性竞争和技术进步，它实际上将有利于整个半导体生态系统的发展。代工厂、EDA 和 IP 行业之间紧密合作可以带来宝贵的生态系统创新。”</p>
