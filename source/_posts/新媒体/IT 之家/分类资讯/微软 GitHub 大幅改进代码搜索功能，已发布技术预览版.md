<p data-vmark="9b90"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 9 日消息，微软 GitHub 在<a href="https://github.blog/2021-12-08-improving-github-code-search/" target="_blank">博客</a>宣布，<span class="accentTextColor">将大幅改进代码搜索功能</span>，已推出技术预览版。</p><p data-vmark="ef25"><img src="https://img.ithome.com/newsuploadfiles/2021/12/55e26ba6-3e8e-4aac-83f9-6050af5f99b3.png" w="956" h="497" title="微软 GitHub 大幅改进代码搜索功能，已发布技术预览版" width="956" height="426" referrerpolicy="no-referrer"></p><p data-vmark="3b09">GitHub 为代码搜索创建了一个单独的界面，<span class="accentTextColor">索引涵盖了超过 500 万个最受欢迎的公共存储库</span>，用户还可以搜索自己的私人存储库。</p><p data-vmark="51e3">该搜索功能有以下特性：</p><ul class=" list-paddingleft-2"><li><p data-vmark="5914">借助智能排名和针对代码优化的索引，轻松在最佳结果中找到要查找的内容。</p></li><li><p data-vmark="3c4d">搜索精确字符串，支持子字符串匹配和特殊字符，或使用正则表达式（包含在 / 分隔符中）。</p></li><li><p data-vmark="81e9">使用 org: 或 repo: 限定符确定搜索范围，并在搜索框中提供自动完成建议。</p></li><li><p data-vmark="94c4">使用诸如“language:”“path:”“extension:”和布尔运算符（OR、NOT）等过滤器来优化结果，使用“symbol:”搜索符号的定义。</p></li><li><p data-vmark="9f58">使用附加功能快速了解位置，例如目录树视图、活动范围的符号信息、跳转到定义、选择搜索等。</p></li></ul><p data-vmark="5e23">IT之家了解到，GitHub 将增加索引，直到涵盖用户可以在 GitHub 上访问的每个存储库，并且将试验评分和排名。</p>
