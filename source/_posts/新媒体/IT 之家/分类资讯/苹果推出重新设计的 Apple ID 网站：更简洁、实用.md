<p data-vmark="d48d"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 5 日消息，据 MacRumors 报道，<span class="accentTextColor">苹果重新设计了其 Apple ID 网站</span>，引入了全新的外观，和之前的设计风格相比，更具有现代感、更简洁。</p><p data-vmark="c3f9">《<a href="https://appleid.apple.com/" target="_blank">Apple ID 网站</a>》</p><p data-vmark="140e"><img src="https://img.ithome.com/newsuploadfiles/2021/11/933aee7b-96e7-4ad7-a4cb-d10313ac5ae6.png" w="1003" h="737" title="苹果推出重新设计的 Apple ID 网站：更简洁、实用" width="1003" height="603" referrerpolicy="no-referrer"></p><p data-vmark="e1fa">该 Apple ID 登陆页面更换一个新的圆点和苹果标志设计，同时还提供了 Apple ID 网站可以用来做什么的信息。之前的设计是以图形为主，有一个大横幅，上面有使用各种苹果设备的人。</p><p data-vmark="6a77"><img src="https://img.ithome.com/newsuploadfiles/2021/11/1059872e-14f7-41ce-b03c-c83dbaf926b9.png" w="1056" h="399" title="苹果推出重新设计的 Apple ID 网站：更简洁、实用" width="1056" height="310" referrerpolicy="no-referrer"></p><p data-vmark="4147">登录 Apple ID 网站后，<span class="accentTextColor">所有可用的 Apple ID 管理选项都以卡片的形式呈现</span>，此外，左边还有导航选项。</p><p data-vmark="3b41">IT之家了解到，<span class="accentTextColor">卡片式视图提供了一目了然的信息</span>，如你的苹果 ID、你的通知邮件、你是否启用了双重认证，以及你的密码最后一次被更新的时间。</p><p data-vmark="7dfa"><img src="https://img.ithome.com/newsuploadfiles/2021/11/3237f86b-6bb5-44f3-9636-7ffe3e2b6804.png" w="642" h="458" title="苹果推出重新设计的 Apple ID 网站：更简洁、实用" width="642" height="458" referrerpolicy="no-referrer"></p><p data-vmark="d98d"><img src="https://img.ithome.com/newsuploadfiles/2021/11/2e397897-751e-4679-8e60-d756556e02a0.png" w="684" h="482" title="苹果推出重新设计的 Apple ID 网站：更简洁、实用" width="684" height="482" referrerpolicy="no-referrer"></p><p data-vmark="5784">此外，点击导航选项会出现更多信息，如你的姓名、生日、可联系的电子邮件地址、设备列表、支付选项等。</p><p data-vmark="b1e3">总的来说，新的 Apple ID 网站设计简单，易于浏览，而且比之前的网站更快。对于那些需要在网络上管理他们的 Apple ID 的人来说，新的外观设计有明显的改进。</p>
