<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 2 日消息 感谢IT之家网友热心投稿，<span class="accentTextColor">Windows Phone 手机恢复工具 Windows Device Recovery Tool 目前已无法检测更新并进行使用</span>，软件提示“应用程序更新检查失败”，疑似服务器已关停。</p><p><img src="https://qimg.ithome.com/image/2021/05/01/6375547984720971638280500.png" title="爷青结，WP 手机恢复工具 Windows Device Recovery Tool 已无法使用" referrerpolicy="no-referrer"></p><p><img src="https://qimg.ithome.com/image/2021/05/02/6375555160294521622900724.png" title="爷青结，WP 手机恢复工具 Windows Device Recovery Tool 已无法使用" referrerpolicy="no-referrer"></p><p>IT之家了解到，Windows Device Recovery Tool（或简称 WDRT）此前已经存在了很长的时间，其不仅支持包括 Lumia、HTC、惠普等在内的 Windows Phone 手机恢复系统，还支持了微软的 HoloLens 等新设备恢复固件的安装。</p><p><img src="https://img.ithome.com/newsuploadfiles/2020/5/20200510_142756_115.jpg" title="爷青结，WP 手机恢复工具 Windows Device Recovery Tool 已无法使用" referrerpolicy="no-referrer"></p><p>微软宣布在 2019 年底停止对 Windows 10 Mobile 的支持，Windows Device Recovery Tool 也曾遭遇服务器连接失败的问题。</p><p>目前，Windows Device Recovery Tool 的下载链接已不可使用，<span class="accentTextColor">网站提示文件已不存在</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/4b9a4ca2-0beb-4230-8f5c-adfe7ec264b6.png" title="爷青结，WP 手机恢复工具 Windows Device Recovery Tool 已无法使用" referrerpolicy="no-referrer"></p>
