data-vmark="429f"><strong>以下为演讲全文：</strong></p><p data-vmark="8eb8">各位开发者、各位朋友们：</p><p data-vmark="228d">大家好。欢迎参加首期 TechoDay，“腾讯技术开放日”。</p><p data-vmark="0958">我是 Dowson，曾经也是一名程序员。我加入腾讯已经 17 年了，最开始在架构部担任架构师。现在，也与卢山一起主持腾讯技术委员会和腾讯技术通道的工作。</p><p data-vmark="c9ee">过去二十年，诞生了很多 C 端应用，比如大家熟悉的 QQ、微信、QQ 音乐；而腾讯的创始人和高管，比如 Pony、Tony，他们本身也是程序员出身，不少大家用过的产品就是他们亲手敲出来的。在消费互联网时代，无数程序员出身的创业者，或者单枪匹马，或者用小团队的模式，做出了很多新奇的应用，改变和丰富了我们的娱乐和生活。</p><p data-vmark="65c8">现在，我们进入产业互联网时代，更加波澜壮阔，但需求的场景也更加复杂。开发者更需要有行业的 KnowHow，需要上下游的协作。一个产品，从想法到落地，需要更多的流程工具、协作平台，还有技术的积累。</p><p data-vmark="9624">所以，我们发起了“腾讯技术开放日”，希望提供一个真正懂程序员需求的交流平台。腾讯云也在过去这几年，不断推出更多的研发工具，期望通过易用、前沿的技术和工具，帮助开发者解决遇到的难题，低门槛实现对数字世界的创想。</p><p data-vmark="5c4b"><strong>首先，我们希望通过轻量化的工具，降低开发门槛，让每个人都可以成为“数字建筑师”。</strong></p><p data-vmark="d2e2">一方面，针对通用性需求，提供模版级工具，实现一键开发；另一方面，也为定制化需求，提供可用的 API 和低代码工具，降低开发门槛。</p><p data-vmark="8568">过去，在搭建 Web 应用、小程序、APP 等场景中，需要创建服务器、配置网络、安装应用软件、数据库、Web 服务器等，再进行各种环境配置，其实还蛮花时间的。</p><p data-vmark="2d10">针对这类高频开发场景，我们推出了轻量应用服务器 Lighthouse，把 IaaS 资源、应用软件和各种配置都统一打包好，实现开箱即用。就算是刚入门的开发者，也只需要 1 分钟不到的时间，就能够完成一个网站的搭建。</p><p data-vmark="21d5">我们的一家合作伙伴“深圳网商天下”主要服务于电商出海业务，基于 Lighthouse 的服务，在不到 3 个月时间里，就帮助数百个卖家成功搭建跨境电商独立站和 APP。</p><p data-vmark="0688">我们提供的微搭低代码服务，可以用“搭积木”的方式，构建企业 OA 门户和审批管理系统。同时与小程序流程深度打通，方便构建和发布小程序，快速触达客户。微搭通过丰富的开发模板，减轻了重复和定制化的工作，开发者可以聚焦在业务逻辑本身。同时又打通了企业微信、腾讯文档和腾讯会议等丰富的 SaaS 应用，开发者可以一站式完成应用的集成和开发，快速优化企业各种业务流程，提升管理流转的效率。目前，微搭已经服务了 10 万名开发者，也构建了 30 万的应用。</p><p data-vmark="eff7">此外，以小程序为代表的微信生态，是开发者关注的技术热点。腾讯云和微信联合推出了云开发，服务了超过 200 万的开发者，成为微信开发者首选的一站式 Serverless 开发平台，日调用量超过 30 亿。随着平台的成长，很多企业开发者也提出了更深度的诉求，为此，我们原子化输出云开发网关，将边缘加速、安全风控、流量治理等功能整合起来，让开发者可以更加轻便的接入。帮助像泡泡玛特等客户在弱网的情况下稳定性提升到 99.9%，同时也抵御了黑产刷单、DDoS 攻击、数据爬虫等多类安全风险。</p><p data-vmark="33fe"><strong>其次，我们希望通过自动化流程，提升研发效率，让“建筑师”们可以在云端快速构建和交付应用。</strong></p><p data-vmark="768e">传统开发模式之下，项目管理、编译、构建、部署等流程环节各自独立进行，就像一座座“孤岛”，需要大量的人工在中间协调配置，很难满足快速迭代的业务需求。</p><p data-vmark="d91b">云的出现，为软件开发提供了标准化的基础设施、统一的架构，也让开发工具链统一和开发工程化，有了实现的基础。</p><p data-vmark="f746">腾讯云的 Coding DevOps 平台，也在助力研发和运维的自动化，帮助产品发布时间，从过去以“季度”、“年”为单位，缩短到以“天”或者“周”为单位。在 Coding 平台上，敏捷项目管理、测试管理、持续集成、制品库、持续部署、应用生命周期管理等功能，在同一个平台上实现，这让团队研发工具建设成本下降 82.7%，研发效能提升 75%，产品交付效率提升了 68%。</p><p data-vmark="8ae7">同时，Coding DevOps 平台和腾讯云的监控能力无缝集成到一起，也实现了秒级监控、精准告警、可观测能力，帮助开发者轻松应对云端的各种突发情况，增强业务稳定性，提升运维效率，减少成本。</p><p data-vmark="5f13">今天，我们也将发布 Orbit 云原生应用部署平台，简化应用部署和运维工作，支持应用运维工作左移，开发人员可以低门槛的观测应用的运行状态，进行扩缩容，监控日志报警等各种配置，完成整个应用的生命周期管理。</p><p data-vmark="2c1a"><strong>最后，我们还期望提供模块化的产品与能力，让 AI、音视频、虚拟仿真等前沿技术，为每个开发者所用。</strong></p><p data-vmark="cd8a">数字技术正在发生巨大的变革。音视频成为新一代的人机交互语言，我们向计算机输入的信息内容，从文字、图片拓展为声音、动作，越来越符合人体自然感官的习惯，创造的可能性也更多。AI 通过革新运算处理，成为无处不在的数据驱动力，将很多过去高度依赖人工的重复性工作，通过建立模型进行自动化的处理。</p><p data-vmark="410d">但这些技术门槛较高，从 0 到 1 的搭建耗费大量的时间与成本。现在，我们把多年积累的 AI、音视频等能力模块化，统一放到腾讯云上，让开发者一键调用。</p><p data-vmark="c052">例如，我们将支撑腾讯会议的音视频能力，封装为腾讯云视立方音视频终端 SDK，让开发者可以一键集成快速稳定的视频、直播能力。以我们的合作伙伴小鹅通为例，这是一款集品牌营销、知识产品交付、用户管理等为一体的数字化工具，拥有百万级客户，借助腾讯云视立方，小鹅通只需要一个 SDK 一套接口，就能拥有全栈视频需求的能力，投入的成本也缩减了 2/3。</p><p data-vmark="4098">接下来的议程里，我的同事们也将为大家介绍更多的产品与工具。例如，机器学习平台“腾讯云 TI 平台公有云版”，让个人开发者也能用数据训练出自己的 AI 模型。</p><p data-vmark="694e">创新突破、开源共享、求精务实是开发者的精神，同样也是腾讯人的坚持。我们相信，代码能传递思想，技术可以创造未来。在过去 20 多年里，腾讯在全球主要国家和地区申请的专利总数超过 5.5 万件，授权数量超过 2.5 万件，是国内互联网公司最多的，在全球也仅次于 Google。在开源方面，我们在 Github 上贡献了超过 140 个项目，贡献者人数超过 3000 名，获得的全球 Star 数超过 40 万。</p><p data-vmark="5815">在数实一体的新世界建设中，代码是基础的建筑材料，开发者是最重要的“建筑师”。未来，腾讯也将持续贡献自己的能力，探索前沿技术，做好工具和产品，助力开发者用得更好、迭代更优、创新更前沿！一起携手“筑梦云端”！</p><p data-vmark="80d0">谢谢大家！</p>
