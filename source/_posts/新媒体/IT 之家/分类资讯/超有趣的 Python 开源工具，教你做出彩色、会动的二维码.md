data-vmark="5bc8">如何给二维码换装？</h2><p data-vmark="2315">要想给二维码换装首先我们得准备一个二维码还有一件“衣服”。</p><p data-vmark="2323">其中二维码可以通过函数生成，只需要填上你希望扫描二维码之后显示出的文字或者网址之类的信息。</p><p data-vmark="955a">“衣服”就是背景图啦，多种格式的图片和鬼畜动图都可以。</p><p data-vmark="8663">具体到操作层面应该怎么做呢？</p><p data-vmark="686c">Lars heuer 准备了两个库：<span class="accentTextColor">qrcode-artistic 和 Pillow</span>。</p><p data-vmark="c0d8">我们先把这两个库装上：</p><pre class="brush:javascript;toolbar:false">pip install qrcode-artistic
pip install Pillow</pre><p data-vmark="3265">然后用 segno.make () 生成一个二维码，在函数中填上扫描二维码显示出的内容。</p><p data-vmark="ab9e">有了生成函数之后，在转化函数 qrcode.to_artistic，background 之后输入你想要替换的背景，可以输入.png，.jpg，.gif 等格式的图片。</p><p data-vmark="dbfd">最后在 target 之后给输出的二维码命名，方便在文件夹中找到。</p><pre class="brush:javascript;toolbar:false ai-word-checked">>>> import segno
>>> qrcode = segno.make('1', error='h')
>>> qrcode.to_artistic(background='2', target='3',
...                    scale=8)</pre><p data-vmark="0370">等运行完毕，你就能在项目文件中找到带有各种背景的二维码了~</p><h2 data-vmark="0732">什么原理？</h2><p data-vmark="d957">其实道理也很简单。</p><p data-vmark="ad19">二维码能修改部分背景是因为二维码中不同的区域，承担不同的功能，并且可以自定义颜色。</p><p data-vmark="9793">例如很明显的一点，常见的二维码在三个角上有较大的像素块（标红），用来定位：</p><p data-vmark="12e4"><img src="https://img.ithome.com/newsuploadfiles/2021/12/d8abd45d-310c-4e61-b689-845e94bc35a4.png" w="212" h="212" title="超有趣的 Python 开源工具，教你做出彩色、会动的二维码" width="212" height="212" referrerpolicy="no-referrer"></p><p data-vmark="b58a">在三个大像素块旁边的黑色像素块（标红）用来设置格式信息：</p><p data-vmark="adb7"><img src="https://img.ithome.com/newsuploadfiles/2021/12/bcb44231-41c3-4a6d-9ac1-8021925c9299.png" w="212" h="212" title="超有趣的 Python 开源工具，教你做出彩色、会动的二维码" width="212" height="212" referrerpolicy="no-referrer"></p><p data-vmark="0f29">而左下和右上两个大像素块旁边的黑色像素块（标红）用来确定版本信息等等：</p><p data-vmark="41d0"><img src="https://img.ithome.com/newsuploadfiles/2021/12/b56a2e05-57d8-4411-92ac-6fb5e07f83f5.png" w="212" h="212" title="超有趣的 Python 开源工具，教你做出彩色、会动的二维码" width="212" height="212" referrerpolicy="no-referrer"></p><p data-vmark="cfbb">因此，当我们自定义这些区域的颜色时，就可以改变二维码的背景了。</p><p data-vmark="e02c">大开脑洞来试试吧，更细节的信息可以在文末链接中找到~</p><p data-vmark="eb68"><strong>参考链接：</strong></p><p data-vmark="67d0">[1]<span class="link-text-start-with-http">https://github.com/heuer/segno</span></p><p data-vmark="f509">[2]<span class="link-text-start-with-http">https://segno.readthedocs.io/en/stable/colorful-qrcodes.html</span></p>
