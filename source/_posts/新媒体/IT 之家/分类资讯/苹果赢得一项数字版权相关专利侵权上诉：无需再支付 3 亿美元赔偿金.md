<p>北京时间 8 月 7 日消息，据报道，苹果公司今日赢得一项裁决，<span class="accentTextColor">无需再向 Personalized Media Communications 公司支付 3.085 亿美元的赔偿金</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/c41b5b8e-37cd-433a-9261-0edfc405c661.png" w="1094" h="720" title="苹果赢得一项数字版权相关专利侵权上诉：无需再支付 3 亿美元赔偿金" width="1094" height="540" referrerpolicy="no-referrer"></p><p>得克萨斯州地区法官罗德尼・吉尔斯特拉普（Rodney Gilstrap）今日裁定，Personalized Media Communications 公司的专利无法强制执行，<span class="accentTextColor">因为该公司故意推迟向美国专利商标局（PTO）提出申请，以便日后能获得更多资金</span>。</p><p>该案始于 2015 年，Personalized Media Communications 当时起诉苹果，指控苹果有数项服务侵犯其专利，<span class="accentTextColor">包括用于在 iTunes、App Store 和 Apple Music 中传输加密内容的 FairPlay 服务</span>。</p><p>今年 3 月，得克萨斯州的一个联邦陪审团裁定，苹果对 Personalized Media Communications 的一项数字版权管理相关的专利构成侵权，应赔偿该公司 3.085 亿美元。</p><p>对于该裁决，苹果表示不满，并提出上诉。今日，得克萨斯州地区法官吉尔斯特拉普驳回了这一裁决。</p><p>Personalized Media Communications 是一家持有专利组合、并通过专利诉讼获得收入的非执业实体。<span class="accentTextColor">除了苹果，Netflix、谷歌和亚马逊等科技公司也被 Personalized Media Communications 起诉过</span>。</p>
