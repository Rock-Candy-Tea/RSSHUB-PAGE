<p>北京时间 5 月 7 日上午消息，据报道，育碧游戏（Epic Games）诉苹果案件进入庭审第四日，庭审中分享的文件让人们更多了解 App Store 和苹果的商业实践。与此同时，第四日也是是苹果高管首次出庭作证的日子。App Store‌副总裁马特・费舍尔（Matt Fischer）在法庭上回答了苹果和 Epic 律师提出的问题，详细介绍了他参与苹果应用商店的工作内容，以及苹果指导方针的制定过程。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/7f2fffdc-84f4-4314-a720-16572500040c.jpg" w="400" h="239" title="庭审第四日苹果高管出庭作证：Hulu 有 App Store‌白名单特权" width="400" height="239" referrerpolicy="no-referrer"></p><p>费舍尔自 2010 年起担任 App Store 副总裁。在今天的证词中，费舍尔被问及了一系列关于他在 App Store 中扮演角色的问题。这位 App Store 副总裁解释说，<span class="accentTextColor">苹果公司长期以来一直关注 App Store 的安全性，并驳斥了有关替代支付方法和侧加载应用程序的观点。</span></p><p>费舍尔还讨论了 Apple Arcade，App Store 账户系统和欺诈等问题。这位副总裁向 Epic 的律师表示，苹果并未就游戏公司在应用内使用第三方支付平台是否会导致安全问题进行研究。</p><p>与此同时，苹果的律师还问了费舍尔一系列关于他参与重新设计 iOS 12 应用商店的问题。费舍尔还详细介绍了苹果和 Epic 之前合作将《堡垒之夜（Fortnite）》和《无尽之剑（Infinity Blade）等游戏带到 iOS 平台，以及在 App Store 中获得突出推荐的例子。</p><p>苹果的律师还询问了费舍尔一些内容友好的邮件，这些邮件来自育碧游戏，内容主要是感谢苹果公司在 App Store 对其应用的推广。当费舍尔被问及 Epic 绕过 App Store 内支付系统部署服务器端更新时的反应，费舍尔回答说：“我愣住了。”</p><p>费舍尔在作证期间，关于苹果公司将内部沟通作为证据的问题，也出现了一些戏剧性的变化。罗杰斯法官似乎反对 Epic 基于相关性和引用第三方的文件等因素继续将某些文件作为证据的尝试。</p><p>其中，<span class="accentTextColor">一封作为证据被披露的电子邮件证实苹果公司已经与 Hulu 等主要的应用程序开发商达成了特殊协议。</span></p><p>2018 年，开发者大卫・巴纳德（David Barnard）在推特上评论了‌App Store‌的订阅将通过 StoreKit API 自动取消，并质疑为什么没有更多来自‌App Store‌的交换计费服务。马特・费舍尔向辛迪・林（Cindy Lin）询问了这个问题，她解释道，Hulu 是一个开发者，拥有订阅取消 / 退款 API 的特殊访问权限。<span class="accentTextColor">“Hulu 是拥有订阅取消 / 退款 API 的白名单开发者之一。早在 2015 年，他们就使用这一功能来支持即时升级，当时我们还没有内置订阅升级 / 降级功能。”</span></p><p>苹果并没有详细说明还有哪些其他拥有特殊权限的开发者，但我们很清楚的是，<span class="accentTextColor">这些功能并不是所有开发者都能访问的</span>。苹果一直声称 App Store 为行业提供了一个“公平的竞争环境”，它对‌App Store‌中的所有应用都一视同仁，对每个人都有一套规则，没有特殊交易或特殊条款。但事实很明显，有些应用确实享有特殊特权。</p><p>尼克・斯塔特（Nick Statt）在推特上分享了一些苹果员工谈论诸如 Hulu 这样的白名单公司特权使用‌App Store ‌API（比如取消 / 退款 API）的文件。</p><p>当被问及苹果是否给予了某些开发者一些特殊权限，允许他们做一些其他开发者不能做的事情时，马特・费舍尔回答说没有。但他表示，苹果有时会在将某个功能提供给所有开发者之前，先与一小部分开发者进行测试。</p><p>大卫・巴纳德（David Barnard）称：“Hulu 是拥有订阅取消 / 退款 API 的白名单开发者之一，当然我们希望苹果公司能在将来的某一天将这一功能赋予我们。Google Play 允许开发者触发取消计划，这对于客户支持来说非常重要。”</p><p>另一封邮件透露，菲舍尔通常希望避免在 App Store 中推广竞争对手。在一封电子邮件中提到的一个具体例子是，苹果的易访问性团队正计划在实现语音支持的应用上开发一个应用商店编辑功能。易访问性团队希望将谷歌和亚马逊应用纳入这一编辑功能中，但一位苹果高管写道：“马特强烈反对在 App Store 中推荐我们的竞争对手。”</p><p>在‌Epic Games‌推出自己的购买选项并发起这场法律大战之前，Epic 首席执行官蒂姆・斯威尼（Tim Sweeney）曾向苹果提出一项特殊协议，要求允许其绕过应用内置购买系统，但遭到苹果的拒绝。就在本周，斯威尼还表示，他将接受苹果的特殊条款，降低‌App Store‌佣金。</p>
