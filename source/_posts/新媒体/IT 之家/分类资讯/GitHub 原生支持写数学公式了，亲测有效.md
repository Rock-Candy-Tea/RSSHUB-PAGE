<p data-vmark="887b">提问：在 GitHub 的 README 文件中插入数学公式有多少种办法？</p><p data-vmark="edb9">做成图片？复制公式去 codecogs 生成一个 svg 图片的外链？还是用插件？</p><p data-vmark="bd83">这些方法有的麻烦，有的需要收费，有的还会出 bug。就不能原生支持一下吗？</p><p data-vmark="28df" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/b44b8247-310b-465c-9624-161c83305f1f.png" w="312" h="146" title="GitHub 原生支持写数学公式了，亲测有效" width="312" height="146" referrerpolicy="no-referrer"></p><p data-vmark="579b">大家吐槽了这么多年，就在最近，<span class="accentTextColor">GitHub 终于宣布可以用 LaTeX 语法直接写公式了</span>！</p><p data-vmark="380a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/d749fed7-33f2-4866-b6e0-aab3cd67bc09.png" w="1080" h="565" title="GitHub 原生支持写数学公式了，亲测有效" width="1080" height="429" referrerpolicy="no-referrer"></p><h2 data-vmark="0418">$ 符号搞定，亲测有效</h2><p data-vmark="9959">GitHub 是用 <span class="accentTextColor">MathJax</span> 实现的这个功能。</p><p data-vmark="3847">MathJax 是一个基于 JavaScript 的开源显示引擎，能够支持非常多的 LaTeX 宏包。</p><p data-vmark="9d81">很多人应该对它不陌生，用过它的插件来在 GitHub 中加入公式。</p><p data-vmark="cc89" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/8a79f48c-c3d8-44d7-99f7-806d1aa1a62c.png" w="900" h="202" title="GitHub 原生支持写数学公式了，亲测有效" width="900" height="184" referrerpolicy="no-referrer"></p><p data-vmark="0c92">废话不多说，直接看方法：</p><h3 data-vmark="d04a">1、编写行内公式</h3><p data-vmark="e7d7">如果你想在行内插公式，只需用一个“$”进行分隔，就像这样：</p><pre class="brush:javascript;toolbar:false">This sentence uses $ delimiters to show math inline: $\sqrt&#123;3x-1&#125;+(1+x)^2$</pre><p data-vmark="ae0b">然后就能直接正确显示公式了：</p><p data-vmark="4871" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/9ae0b604-b533-4df6-8788-6d99c212127d.png" w="1080" h="146" title="GitHub 原生支持写数学公式了，亲测有效" width="1080" height="111" referrerpolicy="no-referrer"></p><h3 data-vmark="e41b">2、编写行间公式</h3><p data-vmark="0fca">如果你想在行间插入公式，只需另起一行并用两个“$”进行分隔：</p><pre class="brush:javascript;toolbar:false ai-word-checked">The Cauchy-Schwarz Inequality
$$\left( \sum&#123;k=1&#125;^n a_k b_k \right)^2 \leq \left( \sum&#123;k=1&#125;^n ak^2 \right) \left( \sum&#123;k=1&#125;^n b_k^2 \right)$$</pre><p data-vmark="1184">效果如下：</p><p data-vmark="a9b1" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/ab062ad8-fc56-450e-a209-04a74d834199.png" w="1080" h="167" title="GitHub 原生支持写数学公式了，亲测有效" width="1080" height="127" referrerpolicy="no-referrer"></p><p data-vmark="42c7">需要注意的是：</p><p data-vmark="8921">（1）如果你的<span class="accentTextColor">公式包含“$”</span>，则需要在前面加上“\”进行转义；</p><p data-vmark="3a31" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/8128fafa-86c0-48cc-a248-e118834e761f.png" w="1080" h="224" title="GitHub 原生支持写数学公式了，亲测有效" width="1080" height="170" referrerpolicy="no-referrer"></p><p data-vmark="288f">（2）如果在公式之外你的<span class="accentTextColor">正文也包含“$”</span>，则需要用“<span></span>”标记；</p><p data-vmark="4923" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/0986517e-da3d-42ab-ad4b-9a281947893e.png" w="1080" h="210" title="GitHub 原生支持写数学公式了，亲测有效" width="1080" height="159" referrerpolicy="no-referrer"></p><p data-vmark="2df0">是不是非常简单？我们亲自试一下看看。</p><p data-vmark="2519">首先新建一个仓库，然后打开 README.md 文件，往里面扔入一些带 markdown 格式的公式和文字。</p><p data-vmark="75ee" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/522ec0d9-4f3c-4a82-9c81-5e4b7d2f853f.png" w="908" h="360" title="GitHub 原生支持写数学公式了，亲测有效" width="908" height="325" referrerpolicy="no-referrer"></p><p data-vmark="6364">然后点击“预览”：</p><p data-vmark="f5cd" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/41039bb6-8c0d-4822-88b2-6401626b3302.png" w="1080" h="391" title="GitHub 原生支持写数学公式了，亲测有效" width="1080" height="297" referrerpolicy="no-referrer"></p><p data-vmark="0461">成功！（行间公式自动居中了。）</p><p data-vmark="696f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/e65bd22a-8b0d-440e-9df3-071972e7d7d1.jpg" w="640" h="425" title="GitHub 原生支持写数学公式了，亲测有效" width="640" height="425" referrerpolicy="no-referrer"></p><p
