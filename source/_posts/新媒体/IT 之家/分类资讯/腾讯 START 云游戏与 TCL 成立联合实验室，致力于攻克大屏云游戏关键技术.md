<p data-vmark="723d"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 1 日消息，今天，腾讯 START 云游戏宣布与 TCL 成立联合实验室，双方将以提升用户的大屏云游戏体验为目标，并专注于核心技术的研发升级，全面覆盖大屏云游戏相关的运行环境、网络、解码、渲染、视频后处理、输入输出外设及安全等模块。</p><p data-vmark="9025"><img src="https://img.ithome.com/newsuploadfiles/2021/12/109b5fb3-97f5-4eee-a7c8-0ba3afb81028.png" w="554" h="369" title="腾讯 START 云游戏与 TCL 成立联合实验室，致力于攻克大屏云游戏关键技术" width="554" height="369" referrerpolicy="no-referrer"></p><p data-vmark="e87e">▲ 图自 START 云游戏微信公众号</p><p data-vmark="31a6">腾讯 START 云游戏表示，START 云游戏平台自 2020 年下半年公开亮相以来，凭借较为扎实的技术基础，以及云端存档互通，原生触控操作支持，3A 大作的持续引进，良好的大屏游戏体验等特色，已发展成为国内云游戏领域头部平台之一。而 TCL 则在智能终端产业深耕多年，其在智屏设备的外观设计，画质引擎，LED 显示技术等方面掌握有大量发明专利与自主知识产权，自与 START 云游戏合作一年来，也在不断攻坚大屏云游戏领域的关键技术，并已建立起相当的技术壁垒。</p><p data-vmark="d856">官方表示，在<strong> TCL 品牌已认证机型上，通过 START 云游戏平台运行游戏，全链路延时已降低至 68.8ms</strong>，也即从按下手柄的按钮到 TCL 电视机上游戏画面出现改变的时间间隔仅有不到 70 毫秒 —— 这一数据事实上已经达到本世代游戏主机在手柄使用无线连接，并接入主流品牌电视时的水准。</p><p data-vmark="698a">未来腾讯 START 云游戏与 TCL 双方还将继续发挥各自所长，尝试攻克更多大屏云游戏领域关键技术，如当前业界呼声较高的 4K/8K 显示技术，手柄的震动反馈功能实现与优化，陀螺仪、重力感应等当代输入外设技术在云游戏体验上的应用，双 WiFi 信道择优机制，QoE 监控，网络抗干扰等网络技术的提升，以及蓝牙手柄优化，端、云、外设三方通信协议优化定制等等。</p>
