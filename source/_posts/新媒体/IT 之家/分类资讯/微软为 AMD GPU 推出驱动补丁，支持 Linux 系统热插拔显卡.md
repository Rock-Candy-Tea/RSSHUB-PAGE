            <p data-vmark="2671"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 8 日消息，微软凭借其 Azure 服务器技术在全球多家企业中获得了云计算业务。目前，微软在其服务器中使用 AMD 数据中心 GPU 和 Linux。但是，当需要更换或安装新的 GPU 时，则需要关闭服务器，这样显然是不方便的。</p><p data-vmark="c2b2">于是，微软打算为 AMD GPU 创建了一个独特的驱动程序，可以在他们的 Linux 服务器上启用 <span class="accentTextColor">AMD GPU 的“热插拔”功能</span>。顾名思义，热插拔是指可以在系统处于活动状态时，将显卡从 PCIe 接口中移除并用另一个替换。</p><p style="text-align: center;" data-vmark="5716"><img src="https://img.ithome.com/newsuploadfiles/2022/4/dea570a8-3b2a-4cdd-9dbb-137d0d5245b1.png" w="843" h="463" title="微软为 AMD GPU 推出驱动补丁，支持 Linux 系统热插拔显卡" width="843" height="450" referrerpolicy="no-referrer"></p><p data-vmark="e1bd">微软研究院的工程师 Shuotao Xu 发布了<a href="https://lists.freedesktop.org/archives/amd-gfx/2022-April/077447.html" target="_blank">以下请求</a>，要求对 AMD GPU 热插拔支持进行代码审查，用于 Linux 操作系统。它专注于 Microsoft Azure 系统，以在需要时帮助支持基于 GPU 的热插拔的功能。微软研究小组也在 GitHub 上提出了<a href="https://github.com/RadeonOpenCompute/ROCK-Kernel-Driver/pull/131" target="_blank">类似的请求</a>。</p><p style="text-align: center;" data-vmark="89e0"><img src="https://img.ithome.com/newsuploadfiles/2022/4/60a58b22-1007-4455-8884-4a29ed4d096f.png" w="1405" h="647" title="微软为 AMD GPU 推出驱动补丁，支持 Linux 系统热插拔显卡" width="1405" height="378" referrerpolicy="no-referrer"></p><p data-vmark="9b7b">IT之家了解到，微软提供的关于新 GPU 技术的信息很少，该驱动程序是微软专有的，旨在允许 Azure 系统将 GPU 加速包含到尚未安装显卡的服务器中。</p><p data-vmark="66d7">通过 PCIe 热插拔显卡和加速卡是一个新概念。最初的热插拔用于一些消费类系统，例如 eGFX，允许显卡通过雷电接口进行热插拔。随着数据中心在市场上变得越来越普遍，这项新技术将使微软的 Azure 系统、AMD 和公司的 GPU 产品线受益。</p><p style="text-align: center;" data-vmark="6c36"><img src="https://img.ithome.com/newsuploadfiles/2022/4/550c95c5-ba07-4651-baa9-9c709ee1c35d.png" w="844" h="422" title="微软为 AMD GPU 推出驱动补丁，支持 Linux 系统热插拔显卡" width="844" height="410" referrerpolicy="no-referrer"></p>
