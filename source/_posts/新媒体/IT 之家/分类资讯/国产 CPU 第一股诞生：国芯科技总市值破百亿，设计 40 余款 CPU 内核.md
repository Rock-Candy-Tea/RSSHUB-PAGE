<p data-vmark="92f7">1 月 6 日报道，今天，嵌入式 CPU 设计公司苏州国芯科技股份有限公司（以下简称国芯科技）以发行价 41.98 元 / 股登陆科创板。</p><p data-vmark="bb7c"><img src="https://img.ithome.com/newsuploadfiles/2022/1/b35e0820-2edb-4846-9fcc-a8e913a06a05.jpg@s_2,w_820,h_349" w="900" h="383" alt="国芯科技" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" srcset="https://img.ithome.com/newsuploadfiles/2022/1/b35e0820-2edb-4846-9fcc-a8e913a06a05.jpg 2x" width="900" height="349" referrerpolicy="no-referrer"></p><p data-vmark="93d8">国芯科技开盘价为 44 元 / 股，开盘后股价有所波动，最低跌破发行价为 41.26 元 / 股，最高为 45.92 元 / 股。截至芯东西成文，国芯科技股价 42 元 / 股，涨幅 1.29%，<span class="accentTextColor">总市值 101 亿元</span>。</p><p data-vmark="d597"><img src="https://img.ithome.com/newsuploadfiles/2022/1/cd35a36b-df9b-4c48-9c43-03f5cfc94fd9.jpg" w="800" h="442" alt="国芯科技" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="442" referrerpolicy="no-referrer"></p><p data-vmark="498f">国芯科技在 CPU IP 方面以摩托罗拉的“M*Core 指令集”、IBM 的“PowerPC 指令集”和开源的“RISC-V 指令集”为基础，设计了 8 个系列 40 余款 CPU 核，且具有 14nm 芯片流片经验。</p><p data-vmark="8ac8">国芯科技起源于工信部和摩托罗拉在 2000 年签订的技术转让备忘录，其中摩托罗拉将 32 位微处理器指令集 M-Core 转让给中国。<span class="accentTextColor">2001 年 6 月，上海科技、南京斯威特、神州信息共同出资设立国芯科技</span>。</p><p data-vmark="ba86">国芯科技营收呈递增趋势，其 2018 年-2021 年上半年各期营收分别为 1.95 亿元、2.32 亿元、2.59 亿元和 1.40 亿元。</p><p data-vmark="7bf2">本次 IPO，国芯科技计划公开发行不超过 6000 万股，所募资金共计 6.0251 亿元。这笔资金将被用于“云-端信息安全芯片设计及产业化”、“基于 C*Core CPU 核的 SoC 芯片设计平台设计及产业化”、“基于 RISC-V 架构的 CPU 内核设计”三个项目。</p><p data-vmark="d213"><img src="https://img.ithome.com/newsuploadfiles/2022/1/7e21ec3f-591d-473e-9679-05142877c6dd.png" w="800" h="247" alt="国芯科技募集资金安排" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="247" referrerpolicy="no-referrer"></p><p data-vmark="c0e0">▲ 国芯科技募集资金安排</p><h2 data-vmark="b25e">一、董事长曾为东南大学博导，2003 年发起中国 C*Core 产业联盟</h2><p data-vmark="579c">谈到国芯科技的创建，离不开现任国芯科技董事长的郑茳。郑茳 1966 年生人，1985 年于东南大学博士毕业后留校任职。1994 年，郑茳被破格提升为教授，1996 年他成为博士生导师，并任东南大学无锡分校副校长。</p><p data-vmark="4d0f">在东南大学任教多年后，郑茳发现中国集成电路的设计水平和外企相比有很大差距。恰巧当时摩托罗拉想要在中国苏州建设一个集成电路设计中心，因此郑茳就加入了摩托罗拉。1999 年，摩托罗拉在苏州设立了中国首间国家级 IC 设计中心，郑茳任该设计中心的经理。</p><p data-vmark="ca92"><img src="https://img.ithome.com/newsuploadfiles/2022/1/7dcec207-4b35-4cf5-a48c-bdf6de0b8de9.jpg" w="800" h="456" alt="国芯科技董事长郑茳" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="456" referrerpolicy="no-referrer"></p><p data-vmark="42c1">▲ 国芯科技董事长郑茳</p><p data-vmark="4c89">后来郑茳接受采访称，在摩托罗拉工作时，他需要获得英国 Arm 的 CPU 授权，这让他进一步看到了 CPU 国产化的难关，决心打造国产处理器。2000 年 8 月，摩托罗拉与工信部签订技术转让备忘录，同意将 32 位微处理器 M*Core 及相关技术完全转让给中国。</p><p data-vmark="6ba4">2001 年 6 月，上海科技、南京斯威特、神州信息各出资 500 万元设立国芯有限，郑茳任董事长。基于摩托罗拉的 M*Core，国芯有限研发设计了 C*Core（32 位嵌入式 CPU）技术。</p><p data-vmark="4010">2003 年 4 月，国芯科技、中芯国际、上海宏力、国家集成电路产业化基地等 60 余家单位共同发起组建了“中国 C*Core 产业联盟”，其中国芯科技是该产业联盟的核心发起单位。</p><p data-vmark="10b3">2019 年，国芯有限变更为股份公司，国家集成电路基金持股 1553.06 万股，占 8.63%，是当时国芯科技的第三大股东。</p><h2 data-vmark="5898">二、营收稳步增长，中云信安为金融安全芯片大客户</h2><p data-vmark="4c4b">报告期内，国芯科技营收呈递增趋势，其 2018 年-2021 年上半年各期营收分别为 1.95 亿元、2.32 亿元、2.59 亿元和 1.40 亿元，2018 年-2020 年 3 年的复合增长率为 15.42%。</p><p data-vmark="6de5">利润方面，2018 年国芯科技净利润为 319.66 万元，2019 年这一数字涨到了 3113.64 万元；2020 年国芯科技净利润为 4574.46 万元；2021 年上半年国芯科技净利润为 317.39 万元。</p><p data-vmark="a8b8"><img src="https://img.ithome.com/newsuploadfiles/2022/1/8902f81c-a8f6-451b-8f71-e84556eede90.png" w="800" h="393" alt="国芯科技 2018 年-2021 年上半年营收及净利润变化" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="393" referrerpolicy="no-referrer"></p><p data-vmark="2ff9">▲ 国芯科技 2018 年-2021 年上半年营收及净利润变化</p><p data-vmark="b7e6">国芯科技的主要产品和服务为 IP 授权、芯片定制服务和自主芯片及模组产品，其中芯片定制服务包括定制芯片设计服务和定制芯片量产服务。</p><p data-vmark="1af2">报告期内，自主芯片及模组产品和芯片定制服务是国芯科技的主要收入来源，两项收入相加占国芯科技营收的七成以上。</p><p data-vmark="f08a"><img src="https://img.ithome.com/newsuploadfiles/2022/1/b47a20d8-1dc1-487d-932b-01a6d380d0ca.png" w="800" h="424" alt="国芯科技 2018 年-2021 年上半年各业务收入占比变化" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="424" referrerpolicy="no-referrer"></p><p data-vmark="3575">▲ 国芯科技 2018 年-2021 年上半年各业务收入占比变化</p><p data-vmark="2d32">从客户情况来说，中云信安在 2018 年-2021 年上半年一直是国芯科技的前五大客户，主要采购国芯科技的金融安全芯片。</p><p data-vmark="0c52">此外，深圳华智融科技股份有限公司目前也通过中云信安在国芯科技的金融芯片基础上配套模组后进行采购。</p><p data-vmark="d061"><img src="https://img.ithome.com/newsuploadfiles/2022/1/34885971-aaf0-45c6-a015-b4ed65210001.png" w="800" h="980" alt="国芯科技 2018 年-2021 年上半年前五大客户情况" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="980" referrerpolicy="no-referrer"></p><p data-vmark="8fa7">▲ 国芯科技 2018 年-2021 年上半年前五大客户情况</p><p data-vmark="6eea">作为一家采用 Fabless 模式的公司，国芯科技主要采购项目为晶圆、封装测试、电子元器件等。报告期内，台积电是国芯科技的主要晶圆供应商，华天科技则是其最主要封装测试供应商。</p><p data-vmark="0182"><img src="https://img.ithome.com/newsuploadfiles/2022/1/4f4484d4-a14a-4069-8852-8cd7b4b52d74.png" w="800" h="1080" alt="国芯科技 2018 年-2021 年上半年前五大供应商情况" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="1080" referrerpolicy="no-referrer"></p><p data-vmark="da45">▲ 国芯科技 2018 年-2021 年上半年前五大供应商情况</p><h2 data-vmark="4162">三、基于三大指令集打造国产 CPU</h2><p data-vmark="1135">作为中国老牌的芯片设计厂商，国芯科技的定位是聚焦于国产自主可控嵌入式 CPU 技术研发和产业化应用的芯片设计公司。</p><p data-vmark="4936">根据招股书，国芯科技目前以摩托罗拉授权的“M*Core 指令集”、IBM 授权的“PowerPC 指令集”和开源的“RISC-V 指令集”为基础，实现了 8 个系列 40 余款 CPU 内核。</p><p data-vmark="7453"><img src="https://img.ithome.com/newsuploadfiles/2022/1/0560cff0-f7ed-4135-a7b4-1f6391f80855.png" w="800" h="763" alt="国芯科技 CPU 研发历程" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="763" referrerpolicy="no-referrer"></p><p data-vmark="7af2">▲ 国芯科技 CPU 研发历程</p><p data-vmark="1bab">虽然国芯科技此前的产品基于摩托罗拉和 IBM 的授权，但 2019 年 IBM 已将 PowerPC 指令集开源，国芯科技产品不再受限于指令和架构授权；摩托罗拉的微处理架构则主要用于国芯科技现有的成熟产品。</p><p data-vmark="998a">而自 2017 年起，国芯科技基于开源“RISC-V 指令集”的 CPU 内核已经逐步研发完成，能够满足未来的国产替代需求。国芯科技还计划持续开发基于“RISC-V 指令集”的 CPU，实现 RISC-V CPU 对物联网节点、金融安全及端安全应用芯片产品的全面覆盖。</p><p data-vmark="42d1">国芯科技的主要竞争目标是全球 CPU 架构龙头 Arm。与之相比，国芯科技在市场占有率、历史积淀、经营规模、产品丰富性和技术水平方面均存在一定差距。但国芯科技凭借自主研发的 CPU 技术，在国家重大需求领域和信息安全领域拥有一定的市场份额，并已在汽车电子领域实现了突破。</p><p data-vmark="5724"><img src="https://img.ithome.com/newsuploadfiles/2022/1/f17e0e0b-a2d8-4c56-b490-14f737f1ec00.png" w="800" h="535" alt="国芯科技的自主芯片及模组研发情况" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="535" referrerpolicy="no-referrer"></p><p data-vmark="62ab">▲ 国芯科技的自主芯片及模组研发情况</p><p data-vmark="2c38">当前，国芯科技<span class="accentTextColor">已拥有 14nm FinFET 流片经验和 40nm eFlash / RRAM 等工艺节点芯片的规模量产经验</span>。截至 2021 年 6 月 30 日，国芯科技累计为超过 90 家客户提供了超过 130 次的 CPU IP 授权，累计为超过 70 家客户提供了 140 余次的芯片定制服务。</p><p data-vmark="10e1">相较行业可比上市公司，国芯科技三大业务的毛利率都略高于平均水平。</p><p data-vmark="bd78"><img src="https://img.ithome.com/newsuploadfiles/2022/1/5391f37f-9c83-43ea-be9d-ac938fc5eaa9.png" w="800" h="746" alt="国芯科技各业务毛利率与上市公司对比" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="746" referrerpolicy="no-referrer"></p><p data-vmark="623b">▲ 国芯科技各业务毛利率与上市公司对比</p><p data-vmark="5ed1">报告期内，国芯科技累计研发投入 2.52 亿元，占营收的 30.45%。截至 2021 年 6 月 30 日，国芯科技已获授权专利 116 项，拥有 126 项软件著作权和 33 项集成电路布图设计。</p><p data-vmark="43c0">截至 2020 年末，国芯科技研发人员为 137 人，占公司总人数比例为 55.47%，其核心技术人员共有 6 人，公司实际控制人郑茳、肖佐楠、匡启和都在其中。</p><p data-vmark="bcdc">具体来说，郑茳为东南大学出身，在 1985 年至 1997 年期间曾任东南大学讲师、副教授、教授、博士生导师等；1998 年-2002 年曾任摩托罗拉中国集成电路设计中心经理。</p><p data-vmark="8e2b">肖佐楠曾入选国家“万人计划”，1994 年-1998 年任中国华大集成电路设计公司工程师；1998 年至 2003 年历任摩托罗拉苏州设计中心工程师、部门经理；此后在国芯科技担任 IC 设计部经理、总经理、董事等职务。</p><p data-vmark="41b0">匡启和 1999 年至 2002 年于南京航空航天大学攻读博士学位；2002 年至 2003 年任江苏意源科技有限公司部门经理；2003 年至 2019 年历任国芯科技有限部门经理、副总经理；现任国芯科技董事、副总经理。</p><p data-vmark="f0df"><img src="https://img.ithome.com/newsuploadfiles/2022/1/ba9d5d8c-67eb-4503-8cdf-d38696feb08e.png" w="800" h="194" alt="国芯科技董事长郑茳（左一）、董事总经理肖佐楠（左二）、董事副总经理匡启和（左三）" title="国产 CPU 第一股诞生：国芯科技总市值破百亿，设计 40 余款 CPU 内核" width="800" height="194" referrerpolicy="no-referrer"></p><p data-vmark="35e5">▲ 国芯科技董事长郑茳（左一）、董事总经理肖佐楠（左二）、董事副总经理匡启和（左三）</p><h2 data-vmark="7189">四、大基金为第三大股东，三大核心技术人员为实际控制人</h2><p data-vmark="a4bd">由于国芯科技第一大股东宁波麒越股权投资基金合伙企业（有限合伙）持有 13.38% 的股份，第二大股东西藏津盛泰达创业投资有限公司持有 10.79% 的股份。因为两大股东持股比例相近，且均未超过 30%，故国芯科技无控股股东。</p><p data-vmark="dbc0">此外，国家集成电路基金也以 1529.75 万元的金额，持股 8.63%，是国芯科技的第三大股东。国芯科技董事长郑茳、董事兼总经理肖佐楠和联创投资也持有 5% 以上的公司股份，其持股比例分别为 7.34%、5.14% 和 5.13%。</p><p data-vmark="db38"><img width="800" height="1447" referrerpolicy="no-referrer"></p><p data-vmark="a17f">▲ 国芯科技股本情况</p><h2 data-vmark="b3e9">结语：国芯科技上市或扩充国产 CPU 生态</h2><p data-vmark="2772">作为国产 CPU 开发商，国芯科技在市场份额或技术水平上无法与英特尔或 生态，为汽车电子、信息安全等关键领域提供新的选择。</p>
