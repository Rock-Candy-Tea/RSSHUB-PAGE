<p data-vmark="b1ce">10 月 15 日，亚马逊（AMZN-US）网络服务业务（AWS）新上任 CEO 赛利普斯基（Adam Selipsky）接受 CNBC 采访时表示，<span class="accentTextColor">亚马逊计划未来将设计更多自研芯片</span>，为客户带来更多效益。</p><p data-vmark="2ca2"><img src="https://img.ithome.com/newsuploadfiles/2021/10/ac1e6694-4661-47b7-b51c-9958c3b956f3.png" w="588" h="416" title="亚马逊 AWS CEO：未来将推出更多自研芯片产品" width="588" height="416" referrerpolicy="no-referrer"></p><p data-vmark="8b7c">赛利普斯基表示：“到目前为止，我们已经自主设计了几种不同的芯片，未来还会有更多。最新的芯片名为 Graviton2，与基于 x86（英特尔处理器的关键指令集）的同类芯片相比，其性价比实际上提高了 40%。”</p><p data-vmark="4c3b">目前 AWS 已开发三个系列自研芯片，包括 ARM 架构 Graviton 2 CPU，可用于 AWS EC2 服务，此芯片为数据中心专用处理器，试图挑战英特尔数据中心市场的龙头地位。</p><p data-vmark="9037">其次为 AWS Inferentia，主要是针对 EC2 Inf1 机器学习应用；第三类则是 AWS Nitro System 芯片，主要用来支持 EC2 instant 底层管理平台，可将网络储存、安全性卸载到专用硬件及软件设备。</p><p data-vmark="9591">亚马逊云科技致力于联合半导体产业上下游，建立互信融合的协同设计开发平台，提供集成的设计、验证、流片及计算资源服务。降低设计公司建立 EDA 开发环境难度，提升设计效率和流程成功率。</p><p data-vmark="6eac">2014 年，亚马逊成立芯片研发部；次年通过收购的以色列芯片设计公司 Annapurna  Labs，在本地数据中心上做一些设计和开发应用。从 2016 年开始到如今，亚马逊芯片设计逐步实现从混合架构到完全基于云端设计，并在云上端到端实现了第一颗 7nm 服务器 CPU  Graviton 2 的全流程设计。</p><p data-vmark="25b4">与此同时，亚马逊云科技还通过定制化芯片开发，持续优化云端基础架构。首先在虚拟化、存储、网络及安全领域进行变革，推出新的虚拟化平台 Amazon  Nitro System，在安全、网络和虚拟化性能有效提升。同时也开发基于 Arm 架构服务器芯片，提供更好性价比，开发机器学习推理芯片，加速计算。</p>
