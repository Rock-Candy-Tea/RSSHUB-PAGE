<p data-vmark="dbf1">任何 Linux 用户，几乎都无法绕开命令行，也绕不开 Vim 编辑器。就在今天，Vim 之父 Bram Moolenaar 传来了一个沉痛的消息：他的挚友、也是 Vim 的主要维护者之一 Sven Guckes 去世了。</p><p style="text-align: center;" data-vmark="dc72"><img src="https://img.ithome.com/newsuploadfiles/2022/2/a3cd7c99-c02b-410a-ba77-6d12a2beb673.jpg" w="627" h="383" title="Vim 主推广者程序员 Sven Guckes 去世了，Vim 编辑器之父：我要把 9.0 版本献给他" width="627" height="383" referrerpolicy="no-referrer"></p><p data-vmark="90cf">Sven Guckes 因患脑瘤于 2 月 21 日在柏林逝世，终年 55 岁。</p><p data-vmark="aba0">Bram Moolenaar 决定，将 Vim 的下一个大版本 9.0 献给这位挚友。Sven 不仅是 Bram 的好友，也为 Vim 的推广做出了重大贡献，甚至 Vim 的官方主页 vim.org 也是他注册的。</p><h3 data-vmark="8826">Vim 推广者</h3><p data-vmark="2f1f">Vim 刚问世时，很多程序员觉得难用，即使到今天上手门槛依然很高，连如何 Vim 在网上都成了段子。程序员圈里经常会存在“鄙视链”，用 Vim 的瞧不起用图形界面的。但 Sven Guckes 并没有这种“精英思维”，反而是致力于 Vim 的推广。让其他人更容易上手 Vim，成了他的重要使命。</p><p style="text-align: center;" data-vmark="569c"><img src="https://img.ithome.com/newsuploadfiles/2022/2/1d3854e8-ac9e-43b1-b90d-2464b4a976bb.png" w="1080" h="608" title="Vim 主推广者程序员 Sven Guckes 去世了，Vim 编辑器之父：我要把 9.0 版本献给他" width="1080" height="462" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="e67f">△ Sven Guckes 在 2015 年 Kiel Linux Days 上讲解命令行</p><p data-vmark="d516">1994 年，当时的 Sven 还是柏林自由大学的学生，首次接触到了 Vim，他感叹这个比 Vi 要好用得多。自此他专心研究 Vim，因此收获了爱情。1994 年夏天，Sven 遇到了一个女孩，正在写论文。最后 Sven 帮助她终于用 Vim 写完了论文。在九十年代，Sven 活跃于 Vim 开发团队中，也正是在这段时间，他与 Bram Moolenaar 结下了深厚的友谊。Sven 还注册了沿用至今的主页域名 vim.org。</p><p style="text-align: center;" data-vmark="f8df"><img src="https://img.ithome.com/newsuploadfiles/2022/2/5ed1b3fc-e4fb-434d-98b0-a20a19bd03f7.jpg@s_2,w_820,h_1238" w="848" h="1280" title="Vim 主推广者程序员 Sven Guckes 去世了，Vim 编辑器之父：我要把 9.0 版本献给他" srcset="https://img.ithome.com/newsuploadfiles/2022/2/5ed1b3fc-e4fb-434d-98b0-a20a19bd03f7.jpg 2x" width="848" height="1238" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="a7d5">△ Sven Guckes 摆出 Vim 手势</p><p data-vmark="7ede">后来，他诞生了“什么是 Vim？”的想法，并用 6KB 文件来解释 Vim，并帮助撰写了一些 Vim 版本的指导文件。</p><p data-vmark="e591">2000 年，Sven 找到了 O’Reilly，希望能在 Vi 的书籍中更新 Vim 的内容，但是遭到了拒绝，因为 O’Reilly 不觉得有利可图。但随后 Vim 团队出版教程大受欢迎，令 O’Reilly 之后加入了 Vim 的内容。</p><p data-vmark="b89e">二十多年来，Sven 参加各种开源社区会议，不厌其烦地解答初学者的问题，甚至还在火车上帮助其他程序员解决 Vim 问题，也因此获得了“命令行解释者”的称号。不仅如此，他还致力于推广各种命令行工具，例如 tmux、Mutt 等等。Sven 本人说，他几乎不怎么使用图形界面，因为他打字效率更高。</p><h3 data-vmark="ebd8">带着笑容离开</h3><p data-vmark="7e18">2021 年底，Sven Guckes 被检测出脑瘤，医生也无能为力。今年 1 月底，他被转移到临终关怀中心。Sven 最后的时光是和朋友一起度过的，直到生命最后时刻他们还在交谈，聊完后 Sven 还笑了。后来 Sven 安静下来，渐渐停止了呼吸。</p><p data-vmark="209c">Sven Guckes 乐于助人的性格，帮他在程序员中赢得了良好的口碑，许多程序员虽然和他只有一面之缘，但对他的评价都很高。</p><p style="text-align: center;" data-vmark="45f9"><img src="https://img.ithome.com/newsuploadfiles/2022/2/4dc4f35b-5c3b-4fc7-9beb-bb6381fae099.png" w="1080" h="988" title="Vim 主推广者程序员 Sven Guckes 去世了，Vim 编辑器之父：我要把 9.0 版本献给他" width="1080" height="750" referrerpolicy="no-referrer"></p><p data-vmark="c989">有些网友虽然与 Sven 素未谋面，但 Vim 作为了他们重要的生产力工具陪伴多年。在 Hacker News 论坛上，网友以推出 Vim 命令的方式，来悼念 Sven。</p><p style="text-align: center;" data-vmark="7120"><img src="https://img.ithome.com/newsuploadfiles/2022/2/d636737a-4e6a-4420-800f-3350f2cb71ec.png" w="462" h="182" title="Vim 主推广者程序员 Sven Guckes 去世了，Vim 编辑器之父：我要把 9.0 版本献给他"
