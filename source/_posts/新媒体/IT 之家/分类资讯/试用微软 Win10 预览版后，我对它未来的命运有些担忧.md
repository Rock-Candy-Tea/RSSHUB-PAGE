<p>前段时间，极客之选介绍了 <a href="https://www.ithome.com/0/549/719.htm" target="_blank">Win10 发布 7 年来 12 个大版本的变化</a>，在这 7 年里，虽然功能增加了不少，但「感知」似乎并不强烈，毕竟 UI 设计才是一款操作系统最直观的水平体现，而与常用常新的手机系统和 macOS 相比，Win10 本来就不出彩的设计也没有太大进步。</p><p>甚至在今天，还有很多人会怀念 Win7 精美的毛玻璃拟物界面，这无不从设计的角度反映出了 Windows 的停滞不前。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/b5371f4c-a00d-4d69-8d5d-ef8a71db6313.png" w="1024" h="683" title="试用微软 Win10 预览版后，我对它未来的命运有些担忧" width="1024" height="547" referrerpolicy="no-referrer"></p><p>也许是微软自己都审美疲劳了，最近终于<span class="accentTextColor">宣布将在下个 Win10 大版本中（21H2，也称太阳谷）带来全新的设计语言</span>，并宣称这将是史上最好用的 Win10。抱着满怀期待的心情，我加入了 Windows 预览体验计划，提前用上了半年后的 Win10，不过在一段时间体验后，我不仅对全新的设计有点失望，更对 Windows 之后的命运开始有些担忧。</p><h2>理想很美好现实很骨感的 Fluent UI</h2><p>Windows 是个历史悠久的系统，你甚至能在 2021 年的 Win10 中找到上世纪 Win95 留下的岁月痕迹，不过 Win10 同时还有最新的 Modern UI 元素，这种新老设计相互杂糅的怪状，<span class="accentTextColor">高情商的说法可以是「赛博朋克」，实在点说就是「割裂感」严重</span>。这也遭到了很多用户的吐槽，所以统一设计语言成为了 21H2 的首要任务。</p><p>熟悉 Windows 的朋友对 Fluent UI 设计风格一定不陌生，这是 Windows 设计的风向标，在扁平化成为主流后，<span class="accentTextColor">Fluent UI 进一步将其发扬光大，加入了光线（Light），深度（Depth），材质（Material）等元素</span>，提升了设计的质感和细节。如果你看过微软的宣传片，一定会被 Fluent UI 的精美程度所震撼，它相比于 macOS 也是有过之而无不及。但可惜的是，Fluent UI 的精美仅止步于宣传片中，从未完全应用至到 Win10 系统上。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/e8a59981-71fd-498c-ba0f-14759d2d57fe.png" w="1440" h="810" title="试用微软 Win10 预览版后，我对它未来的命运有些担忧" width="1440" height="461" referrerpolicy="no-referrer"></p><p>更让人失望的是，<span class="accentTextColor">在最新的 21H2 中，Fluent UI 似乎「跑偏了」</span>，当我打开资源管理器的那一刻，都有些不敢相信自己的眼睛。与之前相比，资源管理器的整体排布并没有明显变化，也依然没有加入亚克力材质的透明设计，而且经典的黄色文件夹变得五颜六色，失去了以往的秩序感，图标设计风格也缺少 Fluent UI 在「深度」上的理解，远不如 Office 系列精致。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/9b1204b2-f808-423a-a335-7144c1cef6ef.png" w="1440" h="968" title="试用微软 Win10 预览版后，我对它未来的命运有些担忧" width="1440" height="551" referrerpolicy="no-referrer"></p><p>▲ Win10 最新预览版 资源管理器</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/e41a2f8f-a950-46b4-b693-b38749f22b80.png" w="1200" h="800" title="试用微软 Win10 预览版后，我对它未来的命运有些担忧" width="1200" height="547" referrerpolicy="no-referrer"></p><p>▲ Fluent UI 风格的资源管理器 网友假想图</p><h2>系统视觉界面「小打小闹」</h2><p>在最新的预览版中，除了新设计的图标，系统 UI 也有一些细微变化，其中圆角菜单的引入是最明显的。</p><p>如今，无论是移动端的 Android、iOS，还是桌面端的 macOS，这些系统似乎都越来越像了，比如所有的系统都非常喜欢圆角的设计，Windows 反而成了唯一的异类。<span class="accentTextColor">不过在 21H2 中也将迎来改变，目前的预览版已经出现了一些圆角菜单</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/dde914a0-7cae-469d-a619-bbfd20f9c460.png" w="946" h="614" title="试用微软 Win10 预览版后，我对它未来的命运有些担忧" width="946" height="532" referrerpolicy="no-referrer"></p><p>圆角设计深得乔布斯喜爱，从视觉角度上来说，弧形线条确实更容易被人眼接受，而且圆角矩形有明确的中心指向性，会让阅读信息更加专注。其实早在 Edge For Chromium 浏览器中，圆角元素就有体现，<span class="accentTextColor">不过目前 Win 10 预览版中圆角的应用还非常少，也没有形成统一风格的迹象</span>，只希望微软这次能继续优化下去，做得彻底一些。</p><p>相比于 macOS 上炫酷的过渡动画效果，Win10 的窗口动画可谓是连「5 毛特效」都算不上，甚至连 Win7 都不如。不过在最新的预览版中加入了一些很浅层的过渡动画，目前只局限在打开和关闭窗口上。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/824abbad-c079-4ce7-9a17-a74aad1aac69.gif" w="1440" h="930" title="试用微软 Win10 预览版后，我对它未来的命运有些担忧" width="1440" height="530" referrerpolicy="no-referrer"></p><p>整体来看，<span class="accentTextColor">目前 Win10 太阳谷的视觉设计依然没有达到质的进步</span>，目前所做的图标重绘、UI 变动以及过渡动画仍然是比较基础的体验改进，甚至是「小打小闹」，与 macOS 和一些精美的 Linux 发行版比起来都相去甚远。现在距离 21H2 大版本推送已经不到半年时间，微软能否大胆甩掉历史包袱，将系统 UI 做到宣传片级水准，希望似乎不大。</p><h2>内容生态、协同办公将成未来隐患</h2><p>除了系统界面的持续完善，在最新版系统中，「新闻和兴趣」功能的加入也引起了我的关注，它被放置在任务栏上，入口是一个实时展示天气变化的图标，只需要将鼠标放置其上，就会弹出信息流资讯窗口。</p><p>该功能的入口优先级非常高，不难看出，Windows 中加入该类功能反映出了微软在重塑内容生态上的焦虑，但实际体验下来，<span class="accentTextColor">「新闻和兴趣」所提供的内容质量却良莠不齐，多为标题党类新闻，使用感受并不好</span>。猜测在正式推送后，该功能很有可能被用户默认关闭。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/817b2a24-0b3b-4f95-bc01-523a62048465.png" w="1440" h="900" title="试用微软 Win10 预览版后，我对它未来的命运有些担忧" width="1440" height="512" referrerpolicy="no-referrer"></p><p>基于 Windows 操作系统，微软早年间曾经建立了丰富的软件内容生态，但在移动端的失利和其他业务的频繁变动下，最近几年，微软在大众心中似乎变成了一家工具类软件提供商，而并非像苹果一样软硬件和内容生态并驾齐驱的大厂。</p><p>另一方面，让微软引以为傲的 Office 套件，如今也受到了来自各种云文档的冲击。总而言之，微软基于 Window 和 Office 所打造的强大优势，看似坚不可摧，实则已开始被一步步蚕食了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/5f4053ba-7e11-4320-b9e6-24066f1666ba.png" w="767" h="431" title="试用微软 Win10 预览版后，我对它未来的命运有些担忧" width="767" height="431" referrerpolicy="no-referrer"></p><p>另一方面，在多端融合的大环境下，跨平台协同办公很快会成为未来操作系统必不可少的功能，在这方面，缺少移动设备和移动操作系统的微软几乎没有多少优势，并且在当前版本的预览版 Win10 中，依然没有看到在多设备协同方面实质性的功能进展。</p><p>对于微软来说，最方便的一条「捷径」便是兼容 Android 运行环境，但具体的实施可能会遇到不少困难，以 Windows 系统现在的开发能力来看，我不得不对它的前景有所担忧。</p>
