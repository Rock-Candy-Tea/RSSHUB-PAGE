<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 8 日消息 前几日，腾讯会议、钉钉等协作办公软件都推出了 Linux 版本，并上架优麒麟、统信 UOS 等多个操作系统的应用商店。</p><p>昨日，优麒麟操作系统宣布，字节跳动旗下企业协作与管理平台-- <span class="accentTextColor">飞书 for Linux 上架优麒麟</span>，原生应用接踵而至，Linux 用户高效协作的春天已然来临。</p><p>IT之家了解到，飞书 for Linux 有多种安装方式，首先可在麒麟软件商店搜索“飞书”，单击进入软件详情页面，点击“下载”并完成安装。随后即可在开始菜单打开，还可以右键选择“发送快捷方式至桌面”。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/15340772-1f9e-4f7f-a343-e8be8240c5fc.png" w="1080" h="608" alt="图片" title="协作办公软件齐开花，字节跳动飞书推出 Linux 版" width="1080" height="462" referrerpolicy="no-referrer"></p><p>其次，可通过命令行进行安装：</p><pre class="brush:javascript;toolbar:false">sudo apt install bytedance-feishu-stable</pre><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/f5721121-54c7-483b-bb2c-b134ab49d3b1.png" w="1080" h="608" alt="图片" title="协作办公软件齐开花，字节跳动飞书推出 Linux 版" width="1080" height="462" referrerpolicy="no-referrer"></p><p>此外，<a href="https://www.feishu.cn/download" target="_blank">飞书官网</a>已上线 Linux 版下载地址，<span class="accentTextColor">用户可手动下载 deb 或 rpm 安装包</span>，适用于 Ubuntu 系列与 CentOS 系列的多个操作系统，<span class="accentTextColor">支持 X64、ARM64 与龙芯 CPU</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/8e8190ea-d0b8-4383-af51-21ce6e17e14d.jpg@s_2,w_820,h_454" w="1060" h="587" alt="图片" title="协作办公软件齐开花，字节跳动飞书推出 Linux 版" srcset="https://img.ithome.com/newsuploadfiles/2021/9/8e8190ea-d0b8-4383-af51-21ce6e17e14d.jpg 2x" width="1060" height="454" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/a2aeb729-5d67-4604-a645-2140d227bb1b.png" w="968" h="622" title="协作办公软件齐开花，字节跳动飞书推出 Linux 版" width="968" height="527" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/46b222fd-efed-4cf8-b5b1-15b7c60fa10d.png" w="983" h="639" title="协作办公软件齐开花，字节跳动飞书推出 Linux 版" width="983" height="533" referrerpolicy="no-referrer"></p>
