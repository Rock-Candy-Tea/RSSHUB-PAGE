<p>北京时间 7 月 2 日上午消息，据报道，亚马逊的下一任 CEO 安迪・贾西（Andy Jassy）面临的最大挑战可能来自他最熟悉的业务：<span class="accentTextColor">云计算服务</span>。</p><p>从下周开始，亚马逊这家成立已经 27 年的互联网巨头将迎来历史上第一次，由除了创始人杰夫・贝索斯（Jeff Bezos）之外的另一个人发号施令。贝索斯即将让位给亚马逊网络服务（AWS）现任负责人贾西，而对于后者来说，这并不是一个轻松的任务。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/3efdbcba-2d79-4d80-87ab-abb5eb341b8b.jpg" w="550" h="412" title="贝索斯卸任在即，亚马逊 AWS 云服务能否延续“商业奇迹”" width="550" height="412" referrerpolicy="no-referrer"></p><p>通过创造各种各样的新产品（例如 Kindle 电子书和 Alexa 智能音箱等），贝索斯为投资人们创造了 <span class="accentTextColor">1.7 万亿美元</span>的价值。并在电子商务、线上市场和云服务等广阔的新增长市场中处于领先地位。但是对于贾西来说，在上任之后，他将立刻面临许多需要他去担心的事情。在解决了这些事情之后，他才能够有精力去考虑如何在创新方面赶上前任的脚步。</p><p>对于亚马逊来说，本次权力转移是一个关键时刻。该公司的商业行为正在面临来自监管机构越来越多的审查。与此同时，美国国会也正在考虑针对反垄断进行立法，而这一立法的针对对象之一，正是亚马逊这个庞大的商业帝国。而且，在疫情全球大流行期间，由于人们不得不留在家中躲避疫情，亚马逊电子商务迎来了爆发式的增长。而随着疫情的缓解，其电子商务销售速度也将快速放缓。但这些问题的重要性与另一个关键问题相去甚远：<span class="accentTextColor">亚马逊网络服务（AWS）能否保持其在云计算领域的领导地位？</span>关于这个问题，谁也不敢打包票。</p><h2>云服务之王</h2><p>在云计算服务领域，<span class="accentTextColor">AWS</span> 依然是领头羊，然而如今保持其领先地位正在变得更加困难。</p><p>几乎所有分析师都一致同意，AWS 是亚马逊最重要、也最值钱的业务。该部门的高利润率和增长前景，正是最近几年来股东回报的主要驱动力。而且这项面向初创企业和大型企业出租计算能力的业务，未来的机会仍然巨大。Wedbush Securities 的数据显示，未来十年，随着企业继续将支出转向公共云供应商，全球云服务支出将接近 1 万亿美元。</p><h2>竞争逐渐激烈</h2><p>然而，去年一年中，AWS 的<span class="accentTextColor">增长速度</span>却落后于其他一些竞争对手。</p><p>有迹象表明，亚马逊的主导地位正在减弱。Gartner 周一发布的一份报告显示，AWS 去年的增长率明显落后于竞争对手。报告显示，该部门 2020 年基础设施云服务收入增长 29%，而整个市场增长幅度为 41%，微软旗下 Azure 和 Alphabet 旗下谷歌云（Google cloud）的涨幅大约为 60%。</p><p>尽管 AWS 在云服务领域仍然占据着绝对的领先地位，但竞争对手正在迎头赶上。如今的现实情况是，市场已经不再是 AWS 一家的天下了，而变成了<span class="accentTextColor">三国鼎立</span>的势头。而客户选择与 AWS 的竞争对手合作是有原因的。</p><p>以微软为例，该公司提供了与传统的办公服务器更好的集成服务，同时为已经使用其企业软件的公司提供了一站式的合作关系。另外，谷歌凭借其在人工智能工具和能力方面的强大实力获得了市场份额，这些工具和能力使用了开源自由软件技术（包括 Kubernetes），使公司能够更好地扩展和管理云端应用程序。</p><p>但是亚马逊最大的阻力可能来自于其<span class="accentTextColor">四处出击的战略</span>，即尝试在几乎所有经济领域日益提升自己竞争实力。亚马逊已经是零售、娱乐、物流和广告领域的一支生力军，专家预测它将扩展到新的领域，包括医疗保健、食品杂货和金融服务。这种策略导致亚马逊与其 AWS 客户的目标变得越来越不一致，而美国的企业们已经注意到了这一点。</p><p>今年 4 月，塔吉特（Target）首席信息官迈克・麦克纳马拉（Mike McNamara）表示，四年前亚马逊在收购了全食超市之后，塔吉特决定退出亚马逊平台。这位高管还透露，塔吉特现在已经不再使用 AWS，转而开始使用谷歌云服务和微软的 Azure。其他一些企业也将效仿塔吉特，抛弃亚马逊 AWS。</p><p>此前，贝索斯经常讲述亚马逊 AWS 成功背后的故事，称其他企业在 AWS 面世整整 7 年之后才推出了自己的云服务业务，贝索斯将其称为一场“商业奇迹”，并称这是商业史上很少见的一件事情，是一次不可多得的好运。从某种意义上说，他是在取笑微软、谷歌和其他公司完全忽视了新云市场的重要性。现在，考虑到云计算玩家之间的竞争日益激烈，竞争对手的威胁不断上升，要想保持领先，需要的不仅仅是运气。但这已经成为了贾西要解决的问题，与贝索斯无关。</p><p>如今贝索斯正忙着飞向太空，亚马逊的业务正面临着前所未有的威胁。新任 CEO 不太走运，在上任之初，就赶上了竞争对手的完全觉醒。</p>
