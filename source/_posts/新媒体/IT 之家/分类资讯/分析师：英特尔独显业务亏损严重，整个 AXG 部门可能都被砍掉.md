<p data-vmark="3d95">业界分析师 Jon Peddie 近撰文分析了英特尔的 GPU 业务。他认为英特尔已在其离散型 GPU 开发上投资了约 35 亿美元 —— 这些投资尚未得到回报。</p><p data-vmark="c046" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/8/049fb814-960b-4eaa-b300-215820b4ab63.jpg" w="416" h="234" title="分析师：英特尔离散 GPU 业务亏损严重，整个 AXG 部门可能都被砍掉" width="416" height="234" referrerpolicy="no-referrer"></p><p data-vmark="f201">自 2021 年第一季度正式成立以来，英特尔的 AXG（加速计算和图形部门）亏损了 21 亿美元。鉴于英特尔首席执行官 Pat Gelsinger 自 2021 年初以来取消了六项业务的业绩记录，JPR 认为 AXG 可能是下一个被取消的业务。</p><p data-vmark="0c7a">Peddie 写道：“Pat Gelsinger 不怕做出艰难的决定，如果重点项目不成功，他就会放弃, 即使是他个人可能喜欢的项目，有谣言一直在暗示 AXG 将是下一个被抛弃的事业部。”</p><p data-vmark="029c">英特尔在 2017 年公开其开发离散绘图解决方案的计划时，宣布了利用其 GPU 为客户端和数据中心应用解决计算，图像和机器智能等功能的计划。核心和视觉计算事业部是一个额外的辅助有利因素，旨在应对新兴的边缘计算市场。</p><p data-vmark="0bd1">在其离散型 GPU 发展的五年里，该公司发布了两款低端独立的 GPU，用于处理廉价个人电脑和一些数据中心应用；推出了用于集成 GPU 的低功耗绘图架构；交付了 oneAPI，可用于编程 CPU、GPU、FPGA 和其他计算单元；取消了数据中心 GPU 的 Xe-HP GPU 架构；多次推迟了用于人工智能和高性能计算应用的 Ponte Vecchio 计算 GPU 的发货（最近一次是由于 Intel 4 节点的延迟），并将 Xe HPG ACM-G11 游戏 GPU 的发布推迟了约一年。</p><p data-vmark="c1ea">Jon Peddie 认为，考虑到英特尔的 Arc Alchemist 500 和 700 系列 GPU 很晚才上市，而且它们将不得不与 AMD 和英伟达的下一代 Radeon RX 7000 和 GeForce RTX 40 系列产品竞争，它们很可能会失败，这显然会增加英特尔的损失。</p><p data-vmark="3527">Jon Peddie 称，考虑到英特尔 AXG 的业绩记录，该公司迄今已花费 35 亿美元，但没有取得任何实质性的成功。对于英特尔来说，离散型 GPU 是一个全新的市场，需要大量投资，因此有亏损并不奇怪。与此同时，英特尔自己的 Habana Gaudi2 深度学习处理器在人工智能方面表现出比英伟达 A100 更明显的性能优势。Peddie 表示：“英特尔是否会止步并退出，可能性只有一半，如果他们不这样做，该公司将面临多年亏损，因为它试图进入一个不友好、难以应付的市场。”</p><p data-vmark="b878">虽然对英特尔来说，取消 AXG 事业部和放弃离散型 GPU 开发以减少损失可能是合理的，但应该注意到英特尔的 AXG 部门总体上追求几个具有战略意义的重要方向，特别是离散型 GPU 的开发。</p><p data-vmark="2f6e">Peddie 认为，到目前为止，离散型 GPU 开发本身确实给英特尔带来了损失，但应该注意到，如果没有一个类似 GPU 的竞争架构，其服务范围包括从低端笔记本到超级计算机，英特尔将无法解决许多新的增长机会。</p><p data-vmark="b505">Habana Gaudi2 看起来是一个有竞争力的深度学习解决方案，但它不能用于超级计算应用。此外，如果不进一步改进英特尔 Xe-HPC 数据中心 GPU 架构，该公司将无法为人工智能 / 机器学习和高性能计算应用构建混合处理单元。如果没有这样的 XPU，英特尔 2027 年的每秒 1021 次浮点运算计划将变得越来越不现实。</p><p data-vmark="5270">虽然英特尔的离散型 GPU 努力没有达到预期，但英特尔需要一个明确的并行计算架构来应对即将到来的应用负载。如果英特尔停止独立 GPU 开发，它将不得不在产品和架构上完全重新设计其路线图。例如，它必须为其客户端处理器找到一家具有竞争力的 GPU 架构供应商，因为英特尔内部的小型 iGPU 开发团队很难能与 AMD 和苹果为其客户端 SoC 提供的解决方案竞争。</p><p data-vmark="24f0">取消 AXG 部门似乎是一个越来越有吸引力的管理决策。然而，GPU 和衍生混合架构对于英特尔服务的许多市场和未来几年要服务的应用具有重要的战略意义，因此取消 AXG 事业部似乎适得其反。</p>
