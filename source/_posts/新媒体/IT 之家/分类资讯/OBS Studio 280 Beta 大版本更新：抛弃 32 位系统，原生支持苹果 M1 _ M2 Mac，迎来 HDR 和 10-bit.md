大版本更新</span>，带来了多项新功能，UI 框架升级到 Qt6。</p><p style="text-align: center;" data-vmark="0ff2"><img Linux）</p></li><li><p data-vmark="7eec">原生支持苹果 M1 / M2 芯片（macOS）：注意，第三方插件也需要适配 M1 / M2 芯片才能使用</p></li><li><p data-vmark="7ef0">将 UI 框架升级到 Qt6（所有平台），意味着 OBS Studio 不再支持以下操作系统，许多依赖于 Qt5 28.0 Beta 1 已在 <a href="https://github.com/obsproject/obs-studio/releases/tag/28.0.0-beta1" target="_blank">GitHub 发布</a>，还有更多修复内容可以在日志中查看。该版本可以通过 GitHub 下载，也能在 <a
