<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 11 日消息 近期，腾讯旗下 Discuz! 宣布 DNSPod 公共 DNS 专业版上线了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/60523418-1200-4085-8901-0083a82b5f7e.png" w="1080" h="328" title="腾讯 DNSPod 公共 DNS 专业版上线：每人每月 300 万次查询免费额度" width="1080" height="249" referrerpolicy="no-referrer"></p><p>IT之家获悉，官方介绍，和公共 DNS 免费版（直接配置 119.29.29.29）相比，DNSPod 公共 DNS 专业版提供个性化的 DNS 安全配置和管控功能，通过几步简单的设置，您即可获得以下能力：</p><ul class=" list-paddingleft-2"><li><p>高风险网站拦截</p></li><li><p>黑名单、白名单管理</p></li><li><p>网络访问分析数据</p></li></ul><p>而且还可以根据不同的场景（公司、家里、小朋友的电脑...）进行不同的配置，在不同的设备上面（电脑、手机、路由器）进行 DNS 设置，即刻生效。</p><p><strong>如何使用？</strong></p><p>1. 访问 DNSPod 控制台 https://console.dnspod.cn/publicdns/，选择开始使用；如果还没有腾讯云账号，请先行注册。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/70e4c361-3dcc-4f42-babb-471a7c0b6688.png" w="1080" h="503" title="腾讯 DNSPod 公共 DNS 专业版上线：每人每月 300 万次查询免费额度" width="1080" height="382" referrerpolicy="no-referrer"></p><p>2. 根据场景选择一种配置，获得特定的拦截规则。当然，这个以后也可以自行调整。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/ab69febf-8bd9-4b97-8fc6-3656dacbd91e.png" w="1080" h="587" title="腾讯 DNSPod 公共 DNS 专业版上线：每人每月 300 万次查询免费额度" width="1080" height="446" referrerpolicy="no-referrer"></p><p>3. 查看配置项，在这里选择一种接入方式来配置到你的设备上，推荐使用 DOH 方式，直接配置到浏览器上即可；如果还想使用传统的 DNS 方式，则需要先将你的公网 IP 进行绑定，然后再使用系统提供的 DNS 服务器 IP 进行配置。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/ae99bff6-cb59-425e-99a4-3baf9f2701a3.png" w="1080" h="566" title="腾讯 DNSPod 公共 DNS 专业版上线：每人每月 300 万次查询免费额度" width="1080" height="430" referrerpolicy="no-referrer"></p><p>4. 使用预置的一些规则，做个性化的配置。通过跟腾讯安全团队深入合作，将持续更新发现的危险网站数据。除了预置规则，你可以可以使用黑 / 白名单功能做更个性的配置。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/bebb5ab4-d9cd-4d15-847a-09015c12bdbf.png" w="1080" h="669" title="腾讯 DNSPod 公共 DNS 专业版上线：每人每月 300 万次查询免费额度" width="1080" height="508" referrerpolicy="no-referrer"></p><p>5. 通常配置成功后 1 个小时后即可看到统计数据。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/fe02504a-6da8-4987-8ebc-2a92153baede.png" w="1080" h="676" title="腾讯 DNSPod 公共 DNS 专业版上线：每人每月 300 万次查询免费额度" width="1080" height="513" referrerpolicy="no-referrer"></p><p>至此配置工作就完成了。</p><p><strong>功能总结</strong></p><p>以下是专业版和免费版的区别：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/a5668ce3-89eb-49ae-b0a3-08c83e52af33.png" w="1080" h="495" title="腾讯 DNSPod 公共 DNS 专业版上线：每人每月 300 万次查询免费额度" width="1080" height="376" referrerpolicy="no-referrer"></p><p><strong>常用问题</strong></p><p>Q：这个服务收费吗？</p><p>A：我们为每位用户每月提供 300 万次查询的免费额度（相信绝大部分用户用不到这么多），超过部分在公测期也不会收费。一旦公测结束，我们会向所有用户发布公告。</p><p>Q：它和防火墙的区别是什么？</p><p>A：两者追求的目标都一样，即替用户过滤危险的网站 。但公共 DNS 专业版提供了更轻量化解决方案，你不用安装客户端，也不会消耗系统资源，如果你想查看拦截效果，访问控制台即可。</p><p>Q：预置的拦截规则会更新吗？</p><p>A：是的，我们和腾讯安全团队合作，会获得最新的风险网站数据，用于更新拦截规则。当然，我们也会提供一些网络流行的网站过滤规则，比如本次提供的 Easylist，后续还会持续新增。</p>
