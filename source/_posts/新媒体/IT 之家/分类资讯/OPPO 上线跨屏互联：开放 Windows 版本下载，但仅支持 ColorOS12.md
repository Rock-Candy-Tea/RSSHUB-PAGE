<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 5 日消息 OPPO 近日悄然上线了一个「<a href="https://pc.heytap.com/zh-CN" target="_blank">跨屏互联</a>」功能，现已开放 Windows 客户端下载，支持靠近即连、跨设备文件传输、剪贴板共享、拍照截图同步和消息同步等功能，但目前似乎仅支持 ColorOS12。</p><p>PC 版系统要求如下：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/1e5374fd-9541-4a34-9082-b886df7a14bf.png" w="224" h="175" title="OPPO 上线跨屏互联：开放 Windows 版本下载，但仅支持 ColorOS12" width="224" height="175" referrerpolicy="no-referrer"></p><p>从官网来看，这一功能主打靠近即连、跨设备传输和编辑文件、共享剪切板、电脑上接收并处理手机的微信、短信等消息、拍照截图快速同步等特性。</p><p>以靠近即连为例，用户只要在支持的手机下拉控制中心开启“跨屏互联”开关并在电脑打开“跨屏互联”客户端，然后将手机靠近电脑便会自动弹出连接提示，从而实现快速连接。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/95933c9e-a7a3-447c-9fac-05c28d35551a.png" w="1440" h="749" title="OPPO 上线跨屏互联：开放 Windows 版本下载，但仅支持 ColorOS12" width="1440" height="427" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/09347be0-ee63-44f1-ae64-1b71233caa74.png" w="426" h="1013" title="OPPO 上线跨屏互联：开放 Windows 版本下载，但仅支持 ColorOS12" width="426" height="1013" referrerpolicy="no-referrer"></p><p>数码博主 @数码闲聊站 体验发现，该功能目前功能性和交互流畅度已经不输其他厂商。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/8da7fbf0-9529-4292-ad67-a27211242db9.png" w="587" h="304" title="OPPO 上线跨屏互联：开放 Windows 版本下载，但仅支持 ColorOS12" width="587" height="304" referrerpolicy="no-referrer"></p>
