<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 19 日消息 近期，微信全新功能“微信客服”功能内测，不过这并非是普通消费者所想的那个客服，其定位和公众号、视频号一样，拥有独立的业务属性。</p><p>目前“微信客服”官网（<a href="https://work.weixin.qq.com/kf" target="_blank">https://work.weixin.qq.com/kf</a>）已上线，用户可以使用企业微信扫码登录。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/11df5888-643b-4b00-bab0-f1bc19875086.png" w="1172" h="646" title="「微信客服」来了，官网已上线，但不是普通用户想要的那个" width="1172" height="452" referrerpolicy="no-referrer"></p><p>IT之家获悉，据官方介绍，用户可以在微信内、外各个场景中接入微信客服，用户可以发起咨询，企业可通过 API 接口回复消息，做好客户服务。从这里可以看出微信客服功能目前是面向企业级用户推出的服务功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/6490f73f-d099-41b2-b4ad-a769b0fd0a91.png" w="1184" h="354" title="「微信客服」来了，官网已上线，但不是普通用户想要的那个" width="1184" height="245" referrerpolicy="no-referrer"></p><p>微信客服功能有以下三个亮点：</p><p><strong>丰富的接入口</strong></p><ul class=" list-paddingleft-2"><li><p>可以在视频号、搜一搜、支付凭证、网页等微信内和微信外场景接入微信客服，接待用户咨询。</p></li></ul><p><strong>微信官方消息通道</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p>咨询记录将出现在微信的「客服消息」中，用户更易发现。</p></li></ul><p><strong>API 收发消息</strong></p><ul class=" list-paddingleft-2"><li><p>可通过 API 收发消息，实现多坐席协作、自动回复等功能。</p></li></ul><p>IT之家官方微信公众账号</p><p>爱科技，爱这里。</p><p>“扫一扫”二维码来关注IT之家，或者微信搜索“IT之家”并关注。</p><p><img src="https://img.ithome.com/newsuploadfiles/2015/4/20150402_160646_229.png" title="「微信客服」来了，官网已上线，但不是普通用户想要的那个" referrerpolicy="no-referrer"></p><p>▲ 移动客户端用户：点击二维码图片并保存到手机，然后使用微信的扫一扫功能，选择相册，找到此二维码即可扫一扫。</p><p>各位小伙伴可通过关注IT之家官方微信公众账号并回复“微信”两个字即可获取最新版。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210316222136_9513.jpg" title="「微信客服」来了，官网已上线，但不是普通用户想要的那个" referrerpolicy="no-referrer"></p>
