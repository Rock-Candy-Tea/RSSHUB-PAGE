<p data-vmark="66ea">近日，工信部发布了一组数据，2022 年第一季度中国新建 5G 基站达到了 13.4 万个，新增 5G 移动电话用户数达到 4811 万户，总数累计达到了 4.03 亿户。</p><p data-vmark="0f50">在 5G 技术的广泛普及下，智能移动终端市场无论是入门级别的设备，还是旗舰级别的设备，其性能正在变得越来越高端，比如高品质摄影、4K 视频录制、游戏竞技等功能，甚至能够与独立相机、流媒体设备、计算机分庭抗礼，这得益于嵌入其中的高性能闪存芯片。</p><p style="text-align: center;" data-vmark="e164"><img src="https://img.ithome.com/newsuploadfiles/2022/5/64e7dca6-1a44-4955-aa90-d9b63d038c55.jpeg" w="1440" h="1756" title="江波龙旗下 FORESEE MCP 系列重构智能移动终端存储组合" width="1440" height="1000" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="f6fa">（FORESEE eMCP）</p><p data-vmark="1e79">江波龙嵌入式存储团队的“明星”系列产品是能够使移动终端更具智能化、集成化、节能化的 MCP 系列，公司旗下行业类存储品牌 FORESEE 将 MCP 系列产品细分为 eMCP 和 uMCP，这是一种将 DRAM 与 NAND 堆叠到一个封装中的存储组合，巧妙地集成了 eMMC / UFS 与 LPDDR3/4x 的复合产品，目前最大可提供 128GB+32Gb 的存储容量，为智能移动终端赋予高效生产力。</p><p style="text-align: center;" data-vmark="a128"><img src="https://img.ithome.com/newsuploadfiles/2022/5/0968e4a7-beeb-4efb-a8c8-2e56d26ff45c.jpeg" w="1440" h="1675" title="江波龙旗下 FORESEE MCP 系列重构智能移动终端存储组合" width="1440" height="954" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="7cca">（FORESEE uMCP）</p><p data-vmark="1ff0">除了国内市场，江波龙嵌入式存储团队在海外市场依然有较大的投入与产出，目前 FORESEE eMCP / uMCP 已广泛应用在欧美、亚非等国家及地区的智能移动终端上，覆盖不同市场定位。</p><p style="text-align: center;" data-vmark="10c5"><img src="https://img.ithome.com/newsuploadfiles/2022/5/01706f71-4427-4373-b00a-39fd37824732.jpeg" w="1440" h="900" title="江波龙旗下 FORESEE MCP 系列重构智能移动终端存储组合" width="1440" height="512" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="baa6">（MCP 产品应用覆盖全球市场）</p><p data-vmark="671d">优化 PCB 设计</p><p data-vmark="c873">赢在起跑线</p><p data-vmark="9755">MCP 系列存储产品显而易见的优势是二合一的多芯片堆叠封装，可在电路设计的源头上简化走线，从而为主板其他零部件释放电路布局空间，实现更灵活的系统设计，同时也有利于终端厂商优化 BOM 清单，缩短出货周期，确保生产 → 销售过程中，销量与质量两不误。</p><p style="text-align: center;" data-vmark="b11d"><img src="https://img.ithome.com/newsuploadfiles/2022/5/7979f2aa-0ff2-4ffc-bdd8-05003df7c010.jpeg" w="1440" h="1706" title="江波龙旗下 FORESEE MCP 系列重构智能移动终端存储组合" width="1440" height="971" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="ca8f">（堆叠封装成市场趋势）</p><p data-vmark="809c">自研固件程序</p><p data-vmark="c5f7">强化数据安全</p><p data-vmark="1b03">江波龙拥有自己的技术研发团队，自主研发所有固件程序，支持软硬件定制。</p><p data-vmark="3893">◆FBA 空闲块加速</p><p data-vmark="bf75">当空闲块充足时，数据以 pSLC 模式写入到空闲块，可以提升瞬时写入性能。</p><p data-vmark="a24b">◆预替换</p><p data-vmark="826c">在数据出错之前先进行数据替换。当识别到存储数据即将损坏，对存储数据进行搬移重写操作，可以提升数据存储的稳定性。</p><p data-vmark="a02c">◆紧急断电保护</p><p data-vmark="d389">通过主控检测低电压行为，实现提前停止操作 Flash，避免丢失用户数据。</p><p data-vmark="4335">◆全局磨损平衡管理</p><p data-vmark="c9ca">通过监控 Flash Block 的磨损次数，实现动态调节全局磨损，避免局部 Block 磨损异常，以提升产品寿命。</p><p data-vmark="5bdc">◆LDPC 纠错算法</p><p data-vmark="d757">通过使用 LDPC 纠错算法，实现存储在极限温度环境下能恢复数据，提升了数据存储能力。</p><p data-vmark="c73a">◆支持 FFU 升级</p><p data-vmark="cdb2">通过标准的 eMMC / UFS 接口，实现对 eMMC / UFS 固件进行更新，以加强对产品生命周期的维护</p><p style="text-align: center;" data-vmark="e613"><img src="https://img.ithome.com/newsuploadfiles/2022/5/8cc4ecc7-26c4-4c77-a17a-84863f626c77.jpeg" w="1440" h="687" title="江波龙旗下 FORESEE MCP 系列重构智能移动终端存储组合" width="1440" height="391" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="3cac">（江波龙自研固件）</p><p data-vmark="6fe0">SLT 芯片测试能力</p><p data-vmark="9a6b">匹配市场质量要求</p><p data-vmark="b61a">FORESEE MCP 系列存储产品采用江波龙自研 SLT 芯片测试方案，创新地达成在同一个产品里，eMMC（FT、RDT）和 LPDDR 同时进行大规模测试的能力，实现了批量自动测试机台，并自主开发测试程序，同测数可达 256 颗且可拓展，全方位保障产品质量。</p><p style="text-align: center;" data-vmark="fbf5"><img src="https://img.ithome.com/newsuploadfiles/2022/5/cbe629ec-72ea-4e60-a290-62138ddbdf9a.jpeg" w="1080" h="707" alt="图片" title="江波龙旗下 FORESEE MCP 系列重构智能移动终端存储组合" width="1080" height="537" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="d708">（自动化测试机台）</p><p data-vmark="98bc">展望</p><p data-vmark="f277">近年来，智能移动终端设备迭代迅速，以智能手机为例，用户追求新形态、便捷性、大容量、高性能等多个维度，这也让终端厂商对存储器的需求不断发生变化。比如，折叠屏手机采用柔性电路板（FPC），更小更薄的电路板体积要求走线设计更加灵活，同时也要求闪存芯片在集成度上有更大的突破，这也是堆叠封装的存储产品受到市场青睐的原因之一。</p><p data-vmark="8ce1">江波龙嵌入式存储团队也在创新形态产品上做足了功夫，FORESEE ePOP、FORESEE Subize eMMC 等产品均能够为特定的细分市场定制精准的存储方案，满足全球众多智能终端企业稳定可靠的产品供应需求。</p>
