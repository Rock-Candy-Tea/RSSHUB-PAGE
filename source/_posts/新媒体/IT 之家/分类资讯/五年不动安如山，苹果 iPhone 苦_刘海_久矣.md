<p>苹果 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone</a> 祖传四年的刘海今年将迎来最大变化，缩小三分之一！是不是很惊喜？惊的是还会保留，喜的是终于要变小了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/3288124f-b853-4ad6-b56e-cde0dc6c2726.jpg@s_2,w_820,h_349" w="900" h="383" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" href="https://iphone.ithome.com/" target="_blank">iPhone X</a> 全面屏设计后，无论<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>厂商的水滴屏、打孔屏、屏下摄像等新技术层出不穷，苹果手机仍头顶“刘海”不动如山。（还被三星的广告嘲笑了一番）</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/75050d19-4c97-4e40-ad33-6aa60ba8c743.jpg@s_2,w_820,h_512" w="1000" h="625" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" srcset="https://img.ithome.com/newsuploadfiles/2021/9/75050d19-4c97-4e40-ad33-6aa60ba8c743.jpg 2x" width="1000" height="512" referrerpolicy="no-referrer"></p><p>▲ 2018 年三星宣传 Galaxy S9 时，暗中嘲讽苹果刘海屏</p><p>但是这几天，业内接二连三爆出苹果“刘海”将缩小的消息，按照 iPhone 前几次发布前的预测和曝光来看，这次 <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 13</a> 的“小刘海”设计基本上是板上钉钉了。</p><p>此前，天风国际的苹果分析师郭明錤就曾发布报告表示，苹果的刘海将缩小，并且这将成为 iPhone 13 的主要硬件规格改变之一。</p><p>随后，知名爆料博主乔恩・普罗瑟也通过一系列渲染图表示 iPhone 13 的“刘海”将缩小三分之一。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/adb2167a-82ac-48f2-b8bd-2684f264227a.jpg@s_2,w_820,h_462" w="1000" h="563" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" srcset="https://img.ithome.com/newsuploadfiles/2021/9/adb2167a-82ac-48f2-b8bd-2684f264227a.jpg 2x" width="1000" height="462" referrerpolicy="no-referrer"></p><p>▲ <a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 13 Pro</a> CAD 图</p><p>一些国内外的手机配件制造商也同样证实了这一点，他们还透露“刘海”处的前置摄像头可能会从左边移到右边。</p><p>据 Counterpoint 调研机构报道，<a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 12</a> 的全球销售量已经超过一亿部。有供应链消息称，iPhone 13 的备货目标将达到 9000 万台，较 iPhone 12 提高了 20%。</p><p>iPhone X 的刘海设计，开创了用前置双目结构光模组做人脸识别的手机交互时代。保留刘海设计说明苹果仍然坚持 Face ID 解锁的交互路线不变。但缩小刘海，则需要解决大量的工程技术难题。</p><p>苹果将如何缩小“刘海”，对供应链企业将提出怎样的挑战？“小刘海”是否会对苹果的前摄或者 Face ID 功能造成影响？苹果的“刘海”什么时候能消失？我带着这些问题深入采访和调查，试图找到问题的答案。</p><h2>01.五年不动如山，苹果为何钟爱“刘海屏”？</h2><p>被用户又爱又恨的“刘海”里到底有什么让苹果无法舍弃？</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/00376ac5-c0dd-45fa-bc7f-6a629fe2a294.png" w="1000" h="454" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" width="1000" height="372" referrerpolicy="no-referrer"></p><p>▲ 从左到右分别为 iPhone X、<a class="s_tag" href="https://iphone.ithome.com/" target="_blank">iPhone 11</a>、iPhone 12 以及 iPhone 13 的渲染图</p><p>我们常说的苹果“刘海”其实是指 iPhone 手机顶端的深感摄像头阵列。</p><p>谈到这个深感摄像头阵列的相关技术研发我们就不得不提到一家以色列 3D 传感器公司 PrimeSense。</p><p>该公司曾与微软共同研发的 Kinect 1 外设，这款外设同样拥有用于面部表情和动作识别的摄像头，深度传感器、麦克风等器件。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/734ffc01-7b29-49be-80ac-98d537cca366.jpg@s_2,w_820,h_547" w="1000" h="667" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" srcset="https://img.ithome.com/newsuploadfiles/2021/9/734ffc01-7b29-49be-80ac-98d537cca366.jpg 2x" width="1000" height="547" referrerpolicy="no-referrer"></p><p>▲ PrimeSense 与微软共同研发的 Kinect 1</p><p>PrimeSense 后于 2013 年被苹果以 3.6 亿美元收购，四年后，苹果推出了 iPhone X 的深感摄像头阵列。</p><p>该阵列里包含了听筒、麦克风、前置摄像头、红外摄像、距离感应器、环境光感应器、泛光感应器、点阵投射器等八个重要组件。这些深感阵列主要涉及 Face ID 解锁、听筒、前置摄像三大功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/79a6b1d2-dc99-4e39-86a4-3131f6fd3420.jpg@s_2,w_820,h_386" w="1000" h="471" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" srcset="https://img.ithome.com/newsuploadfiles/2021/9/79a6b1d2-dc99-4e39-86a4-3131f6fd3420.jpg 2x" width="1000" height="386" referrerpolicy="no-referrer"></p><p>▲ iPhone X“刘海”处的深感摄像头阵列</p><p>在这三大功能里，Face ID 一直是苹果推崇的生物特征识别方式之一。库克曾亲自为 Face ID 技术“站台”，表示 Face ID 的破解难度比 Touch ID 高出二十倍，更为安全。</p><p>苹果也逐渐将 Face ID 的产品应用范围扩大，据最新的 Mac OS 测试文件显示，即将推出的 Macbook 中可能将搭载 Face ID 功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/05cab167-487e-42f1-8d19-445ce1d73f56.png" w="1000" h="678" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" width="1000" height="556" referrerpolicy="no-referrer"></p><p>▲ 苹果发布会上介绍 Face ID 功能</p><p>Face ID 主要涉及 3D 结构光技术，苹果需要在发射端通过点阵投射器和泛光感应器，提升 3D 识别和红外拍摄能力，这是导致“刘海”厚重的原因之一。</p><p>与此同时，3D 视觉领域创企光鉴科技 CEO 朱力接受智东西的采访表示，如果将苹果的“刘海”缩小，可能会牺牲精度，但由于 Face ID 的识别距离要求较近，不超过半米，因此苹果可能会通过优化算法来保证 Face ID 的精度。</p><h2>02.将“刘海”缩小，苹果将有何妙计？</h2><p>尽管 Face ID 再好，但“刘海”存在仍是阻碍了苹果迈向全面屏的道路。目前，业内盛传两种有关 iPhone 13 新“刘海”的设计方案。</p><p>一种是向安卓厂商学习，将扬声器从“刘海”处移出，并且“置顶”到手机边框处，从而减小“刘海”处的面积。</p><p>早在 2019 年，三星就在 Galaxy S10 上就采用了这种边框听筒的设计。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/93bbebf3-2c90-451c-8fe1-e23648344a73.png" w="1000" h="517" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" width="1000" height="424" referrerpolicy="no-referrer"></p><p>▲ 三星 Galaxy S10 将“听筒”放在边框处</p><p>此前，日媒 Mac Otakarac 曾传出一张 3D 打印模拟图，并表示 iPhone 13 Pro 的刘海高度将为 5.35 毫米，较 iPhone 12 增加了 0.05 毫米，宽度则从 34.83 毫米降至 26.80 毫米。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/59d80369-eb3c-4ae2-b67e-7c7186f9de08.jpg@s_2,w_820,h_461" w="1000" h="562" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" srcset="https://img.ithome.com/newsuploadfiles/2021/9/59d80369-eb3c-4ae2-b67e-7c7186f9de08.jpg 2x" width="1000" height="461" referrerpolicy="no-referrer"></p><p>另一种方案则是将“刘海”处的零部件都优化。巴克莱银行分析师安德鲁・加德纳透露，苹果此次的“小刘海”主要通过更加紧密集成 3D 结构光仪器。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/8fe53ee7-6448-4c33-bbb1-ef40adab4a68.png" w="1000" h="1000" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" width="1000" height="820" referrerpolicy="no-referrer"></p><p>▲ iPhone X 与 iPhone 12“刘海”处零部件对比；上为 iPhone X，下为 iPhone 12</p><p>同时，中国台湾媒体 DigiTimes 也表示，苹果已经决定将 3D Face ID 传感器中使用的 VCSEL 芯片尺寸缩小 40-50%，此举将帮助供应商大幅削减生产成本。</p><p>在 VCSEL 芯片制造上，由于 VCSEL 芯片的产业链比较复杂，VCSEL 芯片升级可能涉及外延代工厂、FAB 工厂等多个环节。</p><p>据报道，2020 年 iPhone 12 的 VCSEL 芯片的工具是由测试解决方案提供商 Elite Advanced Laser 和 Chroma Ate 研发。而全球砷化镓晶圆代工龙头台湾稳懋半导体公司负责苹果的 VCSEL 芯片代工。中国台湾的精材科技公司为苹果提供 3D CMOS 的 DOE 封装服务。</p><p>但对于 VCSEL 芯片的尺寸变小从而导致“刘海”也会缩小这种猜测，朱力博士表示 VCSEL 芯片的尺寸跟“刘海”的大小并无直接影响。</p><p>另外，行业内还推测苹果还可能对八大部件中的一些零部件进行了“集成化”处理。目前，国内创企光鉴科技通过 WFP 纳米光子芯片来控制光线投射的切换，从而实现点阵和泛光发射器件二者合一。</p><h2>03.告别“刘海屏”，果粉还要等多久？</h2><p>每次临近苹果新品发布会时，果粉们大多都会期待一块全面屏的出现。事实上，苹果也已经迫不及待地“甩掉”这个厚重的“刘海”了。</p><p>近日，苹果的“可调节显示窗口的电子设备”的专利获得美国专利与商标局的批准。</p><p>该专利将刘海处的摄像矩阵全部都藏在屏幕下，当用户需要使用相关功能时，柔性屏将会露出一条缝隙，通过相关仪器协助完成人脸识别或者拍摄功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/960a570f-b1d8-415d-b414-b8db9cb7d6a9.jpg@s_2,w_820,h_922" w="1000" h="1124" title="五年不动安如山，苹果 iPhone 苦“刘海”久矣" srcset="https://img.ithome.com/newsuploadfiles/2021/9/960a570f-b1d8-415d-b414-b8db9cb7d6a9.jpg 2x" width="1000" height="922" referrerpolicy="no-referrer"></p><p>▲ 苹果的“可调节显示窗口的电子设备”的专利</p><p>苹果还申请了有关屏下 Face ID 以及 Touch ID 有关的专利，该专利也同样将光学成像仪器位于屏幕后面，接收定向传输的屏幕光，并将其处理成像数据。</p><p>值得一提的是，彭博社的马克・古尔曼和华尔街日报的乔安娜・斯特恩都曾表示，苹果正在 iPhone 13 上测试了屏下 Touch ID 技术。</p><p>不仅如此，苹果还有“面部热成像技术”、“用于困难的生物识别认证案例的静脉匹配”等诸多专利，来准备替换 Face ID 或者将 Face ID 的相关仪器置于屏幕后方。</p><p>尽管这些专利可能最后不会出现 iPhone 上，但也体现出苹果苦“刘海”久矣。</p><p>据郭明錤预测，苹果最快将在 2023 年推出新品高端 iPhone 手机上搭载真正的全面屏，彻底和“刘海屏”告别。而在 2022 年，苹果将根据生产良率的情况在部分机型上搭载类似“打孔屏”的屏幕设计，“刘海”处将进一步缩小。</p><h2>04.结语：苹果 Face ID 里的“秘密”，能否给行业带来新玩法</h2><p>iPhone 13 看似只是缩小了的“刘海”，但其实这里涉及深感摄像头阵列、3D 人脸识别等多项技术。尽管近年来苹果想出了各种方法提高其屏占比，但基于 Face ID 等多重考虑，苹果依然选择了“刘海屏”设计。</p><p>在全面屏刚刚兴起的时代，安卓机普遍选择“大下巴”来实现全面屏，而苹果却用更先进的封装技术实现了超窄边框，只在额头留下了一个“刘海”，随后这样的设计曾一度成为主流。</p><p>然而今天，对比安卓机普遍通过屏下摄像头实现“完美屏”，苹果似乎慢了一步。所以这次苹果能否在“小刘海”上玩出新花样，又或者给 VCSEL 等芯片带来新的变革？我们拭目以待。</p>
