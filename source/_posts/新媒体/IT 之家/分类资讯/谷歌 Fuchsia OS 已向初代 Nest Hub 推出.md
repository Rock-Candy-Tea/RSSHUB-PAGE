<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 18 日消息 今年 5 月份，谷歌正式确认 <span class="accentTextColor">Fuchsia OS 将向 2018 年发布的初代 Nest Hub 智能显示器推出</span>，更新后的 Nest Hub 将不会有功能改变，但系统底层将由基于 Linux 的 Cast OS 变更为独立的 Fuchsia OS。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/a597c113-5319-4052-975b-0980f2f11569.jpg@s_2,w_820,h_410" title="谷歌 Fuchsia OS 已向初代 Nest Hub 推出" referrerpolicy="no-referrer"></p><p>▲ 图自 9TO5Google，下同</p><p>据外媒 9To5Google，上周晚些时候，谷歌更新了一个支持页面，初代 Nest Hub 正在接收固件版本 <span class="accentTextColor">1.52.260996</span> 更新。谷歌之后向其证实，此次更新确实包括<span class="accentTextColor">升级到 Fuchsia OS</span>。</p><p>外媒表示，如果一切顺利，在未来几天之内，世界各地的所有初代 Nest Hub 设备都将运行 Fuchsia OS 系统。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/e4a48317-475e-4db1-9555-ea1376a33fb6.png" w="1440" h="608" title="谷歌 Fuchsia OS 已向初代 Nest Hub 推出" width="1440" height="346" referrerpolicy="no-referrer"></p><p>IT之家了解到，经外媒测试，更新后的初代 Nest Hub 在<span class="accentTextColor">功能和 UI 上并没有什么变化</span>，但底层确实更换了。</p><p>按照谷歌给出的查看方法，如果用户可以在设置 —> 关于设备中找到“<span class="accentTextColor">Operating System Version</span>”，那就表明设备已经在运行 Fuchsia OS 了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/bdb30b13-642d-44b9-b230-f5108227b4d5.png" w="837" h="235" title="谷歌 Fuchsia OS 已向初代 Nest Hub 推出" width="837" height="230" referrerpolicy="no-referrer"></p>
