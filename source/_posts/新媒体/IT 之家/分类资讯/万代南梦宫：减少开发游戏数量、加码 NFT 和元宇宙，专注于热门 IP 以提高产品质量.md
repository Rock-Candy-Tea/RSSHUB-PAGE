<p data-vmark="6907"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 19 日消息，TweakTown 报道称，万代南梦宫高管表示公司将缩小开发的游戏数量，以专注于热门 IP，提高产品的质量。</p><p data-vmark="a1bc">“为了提高开发中的游戏质量，我们将通过高级制作人组成的委员会进一步加强我们的评审体系。为了稳步开发高质量的游戏，我们将比以往更多地减少游戏数量，同时提高游戏的质量。我们将根据市场水平和粉丝期望对必要的游戏进行更扎实的投资。”</p><p style="text-align: center;" data-vmark="7ab6"><img src="https://img.ithome.com/newsuploadfiles/2022/8/cec71474-7e42-4162-bad7-631f272d48ea.png" w="620" h="348" alt="Bandai Namco 将“缩小标题数量”以专注于热门 IP 1 |  TweakTown.com" title="万代南梦宫：减少开发游戏数量、加码 NFT 和元宇宙，专注于热门 IP 以提高产品质量" width="620" height="348" referrerpolicy="no-referrer"></p><p data-vmark="19f9">此外，万代南梦宫提供了有关其即将推出的 NFT 和 metaverse 计划的更多细节，并确认第一个 metaverse 将设置在 Gundam 专营权中。</p><p style="text-align: center;" data-vmark="3aa0"><img src="https://img.ithome.com/newsuploadfiles/2022/8/5363cfcc-ad36-46b0-90e8-7df59a3ee00b.png" w="620" h="348" alt="Bandai Namco 巩固 NFT 和 metaverse 计划 22 |  TweakTown.com" title="万代南梦宫：减少开发游戏数量、加码 NFT 和元宇宙，专注于热门 IP 以提高产品质量" width="620" height="348" referrerpolicy="no-referrer"></p><p data-vmark="97e2">第一个元宇宙将基于高达 IP。它被称为 Side-G，是一种具有多个“殖民地”的星系用于不同的活动和内容；其中一个将用于电子竞技，另一个用于高达相关产品；还有一个用于游戏。</p><p style="text-align: center;" data-vmark="47fe"><img src="https://img.ithome.com/newsuploadfiles/2022/8/59c99083-3c24-436f-bf6f-491b23103b09.png" w="620" h="348" alt="Bandai Namco 将“缩小标题数量”以专注于热门 IP 3 |  TweakTown.com" title="万代南梦宫：减少开发游戏数量、加码 NFT 和元宇宙，专注于热门 IP 以提高产品质量" width="620" height="348" referrerpolicy="no-referrer"></p><p data-vmark="bb6c">万代告诉投资者，它的元宇宙以及纳入元宇宙的入 NFT 并非用于投机目的。</p><p data-vmark="8c53">“最终通过连接已经推出的多个殖民地，我们将与粉丝建立强大的联系。我们将成为 NFT，引入区块链等新技术不是为了投机，而是为了粉丝，我们会考虑连接的必要性. [我们的目标是] 与粉丝联系并扩大社区。我想创造一个粉丝可以联系的世界”，万代南梦宫高管在第一季度财报电话会议上的问答环节中表示。</p><blockquote><p data-vmark="34fb">“以 IP 为轴，万代南梦宫将为每个 IP 开发一个元宇宙，作为与粉丝联系的新框架。在这个 IP mettaverse 中，我们预计虚拟空间将使客户在 IP 轴上享受广泛的娱乐，以及利用万代南梦宫的独特优势，将物理产品和场馆与数字元素融合在一起。</p><p data-vmark="3b62">“我们的目标是建立一个开放的框架，为粉丝和商业伙伴之间的联系提供场所。通过 IP Metaverse，我们将在万代南梦宫和粉丝之间，以及粉丝之间建立社区。</p><p data-vmark="8796">“通过这些社区和内容，我们将建立深入、广泛、多方面的联系，并长期持续下去，我们将关注这些联系的质量。”</p></blockquote>
