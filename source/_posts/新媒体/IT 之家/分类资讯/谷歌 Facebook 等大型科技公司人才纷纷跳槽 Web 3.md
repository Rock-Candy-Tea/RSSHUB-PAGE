<p data-vmark="d37c"> 北京时间 2 月 23 日下午消息，据报道，Polygon 和 Circle 等区块链平台正从谷歌和 Facebook 等大型科技公司聘请顶尖人才，主要通过 Web 3 来引他们。</p><p data-vmark="3a4b"><img src="https://img.ithome.com/newsuploadfiles/2022/2/615ba0d1-f91d-47b8-804e-1dc87fad822d.jpg" w="550" h="309" alt title="谷歌 Facebook 等大型科技公司人才纷纷跳槽 Web 3" width="550" height="309" referrerpolicy="no-referrer"></p><p data-vmark="e3a0">本月早些时候，瑞安・怀亚特（Ryan Wyatt）离开了 YouTube，领导 Polygon 旗下一家新的游戏工作室。怀亚特早在 2014 年就加盟 YouTube，负责视频游戏内容，与亚马逊 Twitch 平台展开更积极的竞争。</p><p data-vmark="e4f0">怀亚特在接受媒体采访时表示：“大约 8 年前，当我开始在 YouTube Gaming 部门工作时，我是第一个来到这里的人。当时，我们还没有一支团队，但人们已经开始对游戏视频感兴趣了。”</p><p data-vmark="f9f5">对于 Web 3，怀亚特称：“与当初的游戏视频一样，我对这个机会（Web 3）的看法非常类似。它目前尚处于早期阶段，令人兴奋。”</p><p data-vmark="bb1b">Web 3 被视为科技市场的“下一件大事”，旨在对当前的互联网进行彻底改革，将流行的在线服务转移到区块链等“去中心化”技术上。如今，它正在吸引科技界一些最具前沿意识的人们。</p><p data-vmark="f340">除了怀亚特，跳槽至数字加密货币领域（区块链）的大型科技公司高管还包括 Facebook 数字加密货币和支付部门 Novi 的前首席营销官（COO）谢丽丝・托雷斯（Sherice Torres），她于今年 1 月加盟 Circle。</p><p data-vmark="388d">此外，亚马逊云（Amazon Cloud）高管普拉夫吉特・蒂瓦纳（Pravjit Tiwana）也已离职，加盟数字加密货币交易所 Gemini，担任首席技术官（CTO）一职。</p><p data-vmark="c995">Facebook 数字货币加密业务前负责人大卫・马库斯（David Marcus）去年年底辞职。虽然还没有公布下一步行动，但他已经对 Web 3 赞不绝口。</p><p data-vmark="689e">业内专家表示，科技高管之所以被吸引到这个新兴行业，部分原因是由于它的快速发展。</p><p data-vmark="207c">软件公司 Deel 的 CEO 兼联合创始人亚历克斯・布阿齐兹（Alex Bouaziz）表示：“很容易理解，对于那些被视为科技领域最令人兴奋和最具创新性的趋势，每个人都想涉足。而当前，数字加密货币和 Web 3 就是趋势。许多人把它视为科技行业的未来，就像 Facebook 和亚马逊过去很有吸引力一样。”</p><p data-vmark="1d7d">此外，还有一个因素能让 Web 3 吸引到大型科技公司的人才，那就是资金。有数据显示，比特币交易所 Coinbase 每年为软件工程师提供高达 90 万美元的薪水。</p><p data-vmark="7eb0">对数字加密货币公司投资的激增，也意味着它们有更多的现金，可以用来支付员工薪水。CB Insight 数据显示，区块链初创公司去年筹集了创纪录的 250 亿美元风险投资。</p><p data-vmark="0f50">另外，还有股权。随着数字加密货币公司估值的飙升，如果这些公司将来被收购或进行 IPO（首次公开招股），则早期员工可能会获得巨额回报。</p><p data-vmark="5cb6">当然，Web 3 也有一些令人怀疑的地方。目前，它仍是一个定义松散的术语，广泛地包括一些初始行动，旨在围绕数字加密货币网络建立一个去中心化版本的互联网。</p><p data-vmark="b01e">Twitter 联合创始人杰克・多尔西（Jack Dorsey）就认为，Web 3 过于集中，由少数几个风险资本家控制。而特斯拉 CEO 埃隆・马斯克（Elon Musk）表示，它更像是一个“营销流行语”，而不是现实。</p><p data-vmark="ed99">但怀亚特称，当他开始在 YouTube 工作时，人们对观看其他人玩视频游戏（游戏直播）的想法也持怀疑态度。但如今，游戏是 YouTube 上的第二大垂直市场。怀亚特认为，随着更丰富的体验的推出，人们对数字加密货币和 Web 3 的怀疑将会消退。</p><p style="text-align: center;" data-vmark="6cfe"><img
