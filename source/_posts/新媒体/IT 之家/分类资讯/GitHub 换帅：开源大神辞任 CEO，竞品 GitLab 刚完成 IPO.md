<p data-vmark="6a20">11 月 4 日消息，GitHub 换帅了。一句“Thank you, GitHub”，纳特・弗里德曼（Nat Friedman）—— 原 GitHub CEO，正式宣布卸任。</p><p data-vmark="4275"><img src="https://img.ithome.com/newsuploadfiles/2021/11/089283bd-01b8-4123-889a-6399ae45ab70.png" w="1080" h="510" title="GitHub 换帅：开源大神辞任 CEO，竞品 GitLab 刚完成 IPO" width="1080" height="387" referrerpolicy="no-referrer"></p><p data-vmark="4860">他在官方发布的文章中表示：</p><blockquote><p data-vmark="1789">考虑到我们已经取得的所有成就，以及我在微软五年多的美好时光，我决定是时候回到我的创业初心了。</p><p data-vmark="4595">功成圆满，只留任名誉主席的纳特・弗里德曼要去追寻他的初心了。</p></blockquote><p data-vmark="aa43">而 GitHub 的大旗，将于 11 月 15 日交到现任首席产品官 （CPO）<span class="accentTextColor">托马斯・多姆克</span>（Thomas Dohmke）手里。</p><h2 data-vmark="03e1">“我将继续我的下一次冒险”</h2><p data-vmark="fe9a">纳特・弗里德曼，1999 年毕业于麻省理工学院计算机和数学专业。</p><p data-vmark="6bff"><img src="https://img.ithome.com/newsuploadfiles/2021/11/bd5a93fe-eea1-4a0e-a637-5bcdcc548dd9.png" w="500" h="500" title="GitHub 换帅：开源大神辞任 CEO，竞品 GitLab 刚完成 IPO" width="500" height="500" referrerpolicy="no-referrer"></p><p data-vmark="bce7">2018 年，也就是在微软收购 GitHub 后上任 GitHub CEO。在此之前，他是 Xamarin 的联合创始人兼 CEO。</p><p data-vmark="0b09">Xamarin 创立于 2011 年，是一家构建跨平台移动开发工具的初创公司，2016 年它被微软收购，奈特随即被任命为微软开发者服务副总裁。</p><p data-vmark="9019">在他坐镇 GitHub CEO 的这 5 年，GitHub 已经从一个存储和共享代码的地方变成了一个完整的开发者平台：</p><ul class=" list-paddingleft-2"><li><p data-vmark="6a41">GitHub Actions，如今已成为最流行的 CI 服务。</p></li><li><p data-vmark="9154">AI 编程工具 Copilot 与网页版开发环境 Codespaces，让我们看到了软件开发云原生（cloud-native）与 AI 融合的未来。</p></li><li><p data-vmark="a289">GitHub Discussions 功能，在 GitHub 上前 100 个开源社区中，已经有一半在使用。</p></li><li><p data-vmark="a315">新产品比如 GitHub Advanced Security、CodeQL、private repo secret scanning、ReliateOT 以及公共咨询数据库，大力保护了 GitHub 所依赖的软件供应链</p></li><li><p data-vmark="d085">……</p></li></ul><p data-vmark="64b0">如今，<span class="accentTextColor">已有超过 7300 万开发人员在使用 GitHub</span>，而在微软宣布收购 GitHub 时，这一数字还是 2800 万。</p><p data-vmark="ee24">可以说自收购以来，在奈特的带领下，GitHub 业务每年都比上一年增长的快。</p><p data-vmark="b823">因此，奈特也表示，作为 GitHub 的首席执行官，为这个全球开发者社区服务是一生的荣幸。</p><blockquote><p data-vmark="9f6e">衷心感谢每一位 Hubber 和每一位使 GitHub 成为今天这样的开发者，每天都要感谢你们。</p></blockquote><p data-vmark="9856">口碑不错的奈特要走了，有网友非常好奇他接下来会去干什么。</p><p data-vmark="f542"><img src="https://img.ithome.com/newsuploadfiles/2021/11/7fc17cfb-43c6-414b-a81d-91680a763b02.png" w="1080" h="455" title="GitHub 换帅：开源大神辞任 CEO，竞品 GitLab 刚完成 IPO" width="1080" height="345" referrerpolicy="no-referrer"></p><p data-vmark="8fe7">其实，取得了这么多成就的奈特早已“心在别处”，在告别博客上，他透露道：</p><blockquote><p data-vmark="e1e9">我要进行下一次冒险了，回到我的创业初心：支持、建议和投资用技术创造未来并抓住某些重大机遇的创始人和开发人员。</p></blockquote><p data-vmark="a526">具体要创什么业，倒是可以期待一波。</p><h2 data-vmark="9b2e">托马斯：没想到这么快就换新角色了</h2><p data-vmark="c1a9">另一边，即将上任的新 CEO 托马斯・多姆克，<span class="accentTextColor">今年 8 月份才成为 GitHub 的 CPO</span>，他似乎没想到自己这么快就迎来了新角色。</p><p data-vmark="d9d0"><img src="https://img.ithome.com/newsuploadfiles/2021/11/2c0e081c-b9a9-4dbd-b1ff-f3fc90431339.png" w="272" h="272" title="GitHub 换帅：开源大神辞任 CEO，竞品 GitLab 刚完成 IPO" width="272" height="272" referrerpolicy="no-referrer"></p><p data-vmark="8da1">托马斯毕业于德国工业大学计算工程专业（博士），曾是移动崩溃分析和应用发布服务 <span class="accentTextColor">HockeyApp 的联合创始人兼 CEO</span>。</p><p data-vmark="7dc0">2015，HockeyApp 也被微软收购了，他随之进入微软工作，而他在开发部门的职位一路上升，做到了项目经理。</p><p data-vmark="b2ce">2018，GitHub 被收购后，奈特邀请他转入 GitHub 工作。</p><p data-vmark="1bf4">而面对即将到来的 CEO 大任，托马斯没有说太多雄心壮志的话，也并不打算立即做什么重大改变，只是表示先继续稳固 GitHub 的四大支柱，以便能够进军 1 亿用户规模。</p><p data-vmark="148d">他说的这四大支柱，<span class="accentTextColor">首先就是继续打造像 Copilot 这样的 AI 项目</span>。</p><p data-vmark="0471"><span class="accentTextColor">其次是继续开发 Codespaces</span>，让开发人员不必为每个新项目都设置复杂的开发环境。</p><p data-vmark="a3cb"><span class="accentTextColor">接着就是 GitHub 社区了</span>，多姆克表示，GitHub 在 2008 年成立之初就是最早的一波创作社区之一。</p><p data-vmark="3747">最后，在托马斯看来，<span class="accentTextColor">GitHub 的第四大支柱是安全</span>，这也是该公司这一段时间以来的重点。他曾助成了该领域的一些收购，包括 2019 年的 Dependabot 和 Semmle。</p><p data-vmark="f936">值得一提的是，GitHub 最有力的竞争对手之一 ——GitLab，几周前刚在纳斯达克上市，<span class="accentTextColor">市值 165 亿美元</span>，是微软在 2018 年收购 GitHub 时的两倍之多。</p><p data-vmark="83b5"><span class="accentTextColor">GitLab 目前已拥有 3000 多万用户</span>，截至今年 7 月 31 日，季度收入同比增 69%。</p><p data-vmark="4415">另一边，来自澳大利亚的项目开发和协作工具 Atlassian，表示其 Bitbucket Cloud 服务已达到 1000 万注册用户。</p><p data-vmark="b22c">可以说 GitHub 的竞争对手都在变强。</p><p data-vmark="2988">而网友们对新帅托马斯・多姆克寄予的一点期望是：</p><p data-vmark="9996">希望下一任 CEO 可以像奈特一样愿意倾听用户意见。</p><p data-vmark="3d86"><img src="https://img.ithome.com/newsuploadfiles/2021/11/4038c7b2-026d-4874-abc7-f14e4092bf27.png" w="1080" h="95" title="GitHub 换帅：开源大神辞任 CEO，竞品 GitLab 刚完成 IPO" width="1080" height="72" referrerpolicy="no-referrer"></p><p data-vmark="c5ab">那么，接下来就看新帅托马斯・多姆克如何施展拳脚吧。</p>
