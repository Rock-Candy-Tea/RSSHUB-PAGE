<p data-vmark="f26b">1 月 12 日消息，最近微软前高管本・斯里夫卡 (Ben Slivka) 在接受采访时表示，<span class="accentTextColor">微软应该剥离旗下 Windows 和 Office 业务，专注于日益增长的云计算业务</span>。</p><p data-vmark="43ae">“正确做法可能是把未来押在云上，”曾任微软消费者和商务部门总经理的斯里夫卡说。</p><p data-vmark="e2ee">如今，很多机构使用微软的 Azure 云计算服务为应用程序提供支持，其已经成为微软的明星业务，也是目前亚马逊云服务 AWS 最大的竞争对手。</p><p data-vmark="e821"><img src="https://img.ithome.com/newsuploadfiles/2022/1/d5e370c3-dec5-4ce0-b566-9eabde1c46fb.jpg" w="660" h="441" alt title="微软前高管：公司应剥离 Windows 和 Office 业务，专注云计算" width="660" height="441" referrerpolicy="no-referrer"></p><p data-vmark="d2d2">数十年前，微软曾在操作系统和生产力软件领域占据市场主导地位。但 Azure 业务的不断增长重新激起投资者对微软的兴趣。微软股价在 2021 年应声上涨 51%，同期标准普尔 500 指数上涨约 27%。</p><p data-vmark="d3da">斯里夫卡仍持有微软的少量股票。他在采访中提到微软在移动设备领域的失败，当时公司输给了苹果和谷歌。斯里夫卡说自己不希望内部分歧妨碍 Azure 业务的持续增长。</p><p data-vmark="27d4">“运营 Windows 业务的人束缚了微软移动操作系统团队的手脚，限制了他们能做的事情，”斯里夫卡说。“他们的原型同样有一个小的’开始’按钮，还有其他各种各样没用的东西。微软曾三次重启其移动业务战略，最终手机制造商和开发商都离微软而去。”</p><p data-vmark="913e">2015 年，微软对斥资 95 亿美元从诺基亚收购来的手机设备和服务进行 76 亿美元的减记。四年后，由于市场份额已经下降到 1% 以下，微软停止了对 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> Mobile 移动操作系统的支持。</p><p data-vmark="4ac9">过去几年，微软一直在主动出售部分业务。2015 年，公司将必应地图资产出售给叫车服务公司 Uber；次年，微软同意以 3.5 亿美元的价格将诺基亚功能手机资产出售给富士康和 HMD Global，并剥离图像公司 Vexcel。</p><p data-vmark="ea37">斯里夫卡于 1985 年加入微软，并于 1999 年离职，曾负责 Windows 95 和 IE <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>等项目。</p><p data-vmark="9c08">“我明白 Windows 对微软有多重要，”斯里夫卡曾在 1997 年向联合创始人比尔・盖茨（Bill Gates）发电邮。这封电子邮件成为美国司法部反垄断案中的呈堂证据。</p><p data-vmark="67e3">但斯里夫卡还表示，微软“不会明天就倒闭”，不需要马上就剥离 Windows 和 Office 业务制定交易计划。</p><p data-vmark="4487">然而，似乎很少有分析人士同意斯里夫卡的观点。</p><p data-vmark="42a4">分析人士说，如今 Windows 和 Office 业务仍在各自市场占据领导地位，这些产品有助于吸引客户继续使用 Azure 云服务。科技行业研究机构 Gartner 在其最新的市场年度报告中表示，大公司通过多年时间建立对微软品牌的信任之后，选择继续使用 Azure。</p><p data-vmark="5498">富国银行分析师迈克尔・特林（Michael Turrin）在接受采访时表示：“微软长期以来建立的良好信誉，为 Azure 带来了一个非常有利可图的未来。”“从某个角度来说，Office 也是微软在生产力工具领域的护城河，把这些东西结合在一起更强大。”</p><p data-vmark="eb88">特林说，当然人们也希望能够投资一家更精简的公共云服务公司。例如，科技行业和华尔街的许多业内人士都曾猜测亚马逊会剥离 AWS。但亚马逊曾多次强调，没有类似计划。</p><p data-vmark="ba75">特林在去年 11 月份发布的一份报告中给予微软股票“买入”评级。他估计，到 2028 年，Azure 市场份额将超过 AWS。特林认为，到 2023 年底，微软市值将达到 3 万亿美元，其中 Azure 业务占到其中一半。</p><p data-vmark="afb3">去年 9 月份，富国银行公布自家使用 Azure 和谷歌云服务的计划。微软首席商务官贾德森・阿尔托夫 (Judson Althoff) 称，微软与成立 169 年的富国银行有“长期合作关系”。</p><p data-vmark="71ae">放弃 Windows 和 Office 业务将对微软营收前景产生重大影响。就 2021 财年第三季度而言，微软仍有超过三分之一的当季营收来自 Office 产品和 Windows 业务。</p><p data-vmark="fb8e">Windows 和 Office 业务的利润也很高。瑞银分析师去年 11 月份估计，微软总营收的 12% 来自 Windows，这一业务还贡献了总毛利的 17%。多年来，虽然微软一直在扩大 Azure 业务的毛利率，但分析师认为还没有接近 Windows 业务水准。</p><p data-vmark="86a6">微软拒绝就是否会剥离 Windows 和 Office 进行置评。</p>
