<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 12 日消息 微软 GitHub 团队昨日发文，宣布已将开发环境从苹果 macOS 迁移到了自家的 <span class="accentTextColor">Codespaces 云端环境</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/54050d9a-34f8-4dd6-9034-dd5021939702.png" w="1440" h="746" title="微软 GitHub 工程团队：已将开发环境从苹果 macOS 迁移至 Codespaces" width="1440" height="425" referrerpolicy="no-referrer"></p><p>从图标就可以看出来，Codespaces 与微软 Visual Studio 系列产品有点关系。IT之家了解到，事实上，Codespaces 是微软收购 GitHub 之后的一项成果，由原来的 Visual Studio Online 与 GitHub 服务相结合，打造了完全云端的开发环境。</p><p>开发者们可以通过任何环境的<span class="accentTextColor"><a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a></span>，在云端打开 Visual Studio Code 等开发工具，<a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 编程也不是问题。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/6be3bff0-0543-4ede-8a67-7409a1aa7c2b.png" w="719" h="509" title="微软 GitHub 工程团队：已将开发环境从苹果 macOS 迁移至 Codespaces" width="719" height="509" referrerpolicy="no-referrer"></p><p>据介绍，GitHub.com 代码库已有近 14 年的历史。在这 14 年中，支持 GitHub.com (github/github) 的核心存储库已经收到了超过一百万次提交，这些提交中的绝大多数来自在苹果 macOS 上构建和测试的开发人员。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/4a73e1c9-d737-4f89-aa61-4143a7bfa6ab.png" w="1272" h="147" title="微软 GitHub 工程团队：已将开发环境从苹果 macOS 迁移至 Codespaces" width="1272" height="95" referrerpolicy="no-referrer"></p><p>GitHub 官方表示，在过去的几个月里，他们将 macOS 环境抛之脑后，转而使用 Codespaces 进行 GitHub.com 的大部分开发，这是日常开发流程的根本转变。</p><blockquote><p>Codespaces 产品更强大，我们为 GitHub.com 的未来发展做好了准备。</p></blockquote><p>官方也承认这一迁移遇到了不少困难，比如 GitHub.com 存储库在磁盘上几乎有 13 GB；简单地克隆存储库需要 20 分钟。结合依赖设置，引导一个 GitHub.com 代码空间<span class="accentTextColor">需要 45 分钟以上的时间</span>。</p><p>不过，官方通过一系列改进将 45 分钟缩短到了 5 分钟，又从 5 分钟缩短到了 10 秒，大家可以<a href="https://github.blog/2021-08-11-githubs-engineering-team-moved-codespaces/" target="_blank">点此查看文章</a>，了解他们是如何解决这一问题的。</p>
