<p>Docker 官方宣布：自 2021 年 8 月 31 日起，<span class="accentTextColor">Docker 桌面将不再对企业用户免费了</span>。</p><p>对于员工数量超过 250 名或收入超过 1000 万美元的公司，他们必须使用付费订阅。</p><p><span class="accentTextColor">费用最高为每月 21 美元</span>（折合人民币约 136 元）。</p><p>对于个人用户，Docker 桌面则仍旧可以免费使用，只是 Docker 把它的订阅名称从“Free”换成了“Personal”。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/37788ca5-ed8f-46b8-b442-64240e02cc08.png" w="1080" h="552" title="Docker 桌面不再对企业用户免费，每月订阅费最高 21 美元" width="1080" height="419" referrerpolicy="no-referrer"></p><p>这次收费只针对 Docker 桌面，在 Docker Engine、上游开源 Docker 和 Moby 项目上，则没有什么变化。</p><p>Docker 官方表示，<span class="accentTextColor">这次调整是他们对可持续商业模式的探索</span>。</p><p>但这种说法并不能让广大开发者买账，有人直接给 Docker 官方留言：</p><p>你们这是想给对家 Podman、Buildah 留机会吧！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/1c4407e9-58ba-473a-8e79-c3bfe8b85d2e.png" w="1080" h="305" title="Docker 桌面不再对企业用户免费，每月订阅费最高 21 美元" width="1080" height="232" referrerpolicy="no-referrer"></p><h2>想要开拓新的商业模式</h2><p>针对企业，Docker 给出了三种订阅方案：<span class="accentTextColor">Pro、Team、Business</span>。</p><p>其中收费最高的 Docker Business 可以提供更多的服务内容。</p><p>主要包括集中管理、单点登录和增强安全性这几项功能。</p><p>用户可以在集中控制面板上设置允许开发人员访问的内容。</p><p>同时还有基于 SaaS 的集中管理，能够控制 CPU 使用率、内存、端口和防火墙访问的配置。</p><p>总的来看，Docker Business 主要是想要帮助企业更好管理 Docker 开发环境。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/889a4f93-f006-4459-be10-5d9abbeabafc.jpg" w="770" h="578" title="Docker 桌面不再对企业用户免费，每月订阅费最高 21 美元" width="770" height="578" referrerpolicy="no-referrer"></p><p>针对此次变动，Docker 官方主要给出了两点理由。</p><p>Docker 的 CEO Scott Johnston 表示：</p><blockquote><p>一方面，现在公司想要解决软件供应链上存在的安全隐患；另一方面，公司需要开拓一个可行的商业模式。</p></blockquote><p>目前，<span class="accentTextColor">全球有 55% 的专业开发人员在使用 Docker</span>。</p><p>但是对供应链攻击的速度也在增加，这导致对 Docker 可信内容（包括 Docker Official Images 和 Docker Verified Publisher images）的需求也在加速增长。</p><p>再加上全球开发人员数量也在飞速增长，<span class="accentTextColor">Johnston 表示这一数字在 2030 年将会增长到 4500 万</span>。为了满足更大的需求，Docker 需要做出一些调整。</p><p>事实上，Docker 非常受开发者们欢迎。</p><p>它能够相对轻松地使用容器在一台机器上开发和测试多个程序，而且程序之间共享操作系统的内核不会互相干扰。</p><p>Docker Desktop 是其中的一部分，它可以管理各种 Docker 组件和功能，包括容器、图像、本地 Kubernetes 等等。</p><h2>网友炸锅了：这不是钱的事</h2><p>虽然 Docker 官方已经给出了充分的理由，但是许多人对此并不买账。</p><p>不少人已经在 Docker 的推特下嚷嚷着要换平台了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/493acef1-ccb0-4290-93a4-4a2e7cccf623.png" w="1080" h="206" title="Docker 桌面不再对企业用户免费，每月订阅费最高 21 美元" width="1080" height="156" referrerpolicy="no-referrer"></p><p>有人直言，这不就和之前只有订阅后才能使用 Java SE 一样么，最后有大量公司都选择了 FOSS 替代方案。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/6a53ad64-4c59-49d7-a161-ce29dc0d4169.png" w="1080" h="243" title="Docker 桌面不再对企业用户免费，每月订阅费最高 21 美元" width="1080" height="184" referrerpolicy="no-referrer"></p><p>也有人担心，这不只是钱的问题，更是一种“钓鱼”策略。等到有一天每个人都使用它时，然后他们再毫无征兆地调整收费模式。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/dfe18729-bb47-4658-b342-ff059b401e2f.png" w="1080" h="248" title="Docker 桌面不再对企业用户免费，每月订阅费最高 21 美元" width="1080" height="188" referrerpolicy="no-referrer"></p><p>但现实问题是，如果大企业想要更换平台，也不是件容易事。</p><p>相比之费九牛二虎之力迁移平台，一个月 21 美元的收费，真的微不足道。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/5a6f4426-d2ed-4355-9120-a168eda630ac.png" w="1080" h="137" title="Docker 桌面不再对企业用户免费，每月订阅费最高 21 美元" width="1080" height="104" referrerpolicy="no-referrer"></p><p>所以“妥协”，或许是更多用户会选择的方案。</p><p>而且相较于收费的问题，更多人想吐槽的是：</p><blockquote><p>如果你想要我付钱，那先把优化做好再说吧！不要总是当我工作中的绊脚石。</p></blockquote><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/e511e0da-b2ee-4077-9d6b-88597841d284.png" w="1080" h="121" title="Docker 桌面不再对企业用户免费，每月订阅费最高 21 美元" width="1080" height="92"
