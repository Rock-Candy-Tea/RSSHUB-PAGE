<p data-vmark="886e"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 26 日消息，2021 年初，谷歌提供了群组联合学习 (FLoC)，以替代网站上投放相关广告的第三方 Cookie。在过去一年的多方反对后，谷歌近日提出了一个新的解决方案 ——<span class="accentTextColor">Topics API</span>。</p><p data-vmark="3563">谷歌希望到 2023 年逐步停止在 Chrome 中使用第三方 Cookie。为此，它正在隐私沙盒下开发一系列隐私保护替代方案。FLoC 是其中一项重要工作。</p><p data-vmark="d77a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210410152614_2390.jpg" title="谷歌宣布放弃 FLoC：推出全新的 Topics API，以取代第三方 Cookie" referrerpolicy="no-referrer"></p><p data-vmark="b82a">根据谷歌官方的说法，FLoC 技术使用机器学习算法来分析用户数据，然后根据个人访问的站点创建人群的集合。广告商不会获得用户的本地数据，而是直接获得更广泛的人群画像，从而进行广告发放。</p><p data-vmark="bf6c">然而，FLoC 也被担心仍可用于跟踪用户而受到批评。<span class="accentTextColor">WordPress 和亚马逊将其关闭，也没有其他<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>承诺将采用 FLoC</span>。</p><p data-vmark="98d5">谷歌现在正在用 Topics API 替换其 FLoC 方案。Topics API 的工作原理是让浏览器 (Chrome) 在本地设备上确定“<span class="accentTextColor">根据浏览历史代表用户本周最感兴趣的主题（Topics）</span>”。Chrome 将利用分类器模型，把站点主机名映射到 Topics。</p><p data-vmark="bc20" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/1/7b50fda6-3b2b-4fde-b3d6-9d95a0aef698.png" w="955" h="416" title="谷歌宣布放弃 FLoC：推出全新的 Topics API，以取代第三方 Cookie" width="955" height="357" referrerpolicy="no-referrer"></p><p data-vmark="f054">IT之家了解到，<span class="accentTextColor">主题包括健身、旅行和新闻，最初提供大约 350 个</span>。主题在被自动删除之前会保留三周，Chrome 将允许用户随意查看和删除他们的主题 —— 或者完全禁用该功能。谷歌表示，主题是由人类“精心策划”的，不会包括潜在的敏感类别，例如性别或种族。</p><p data-vmark="cc6b">当用户访问请求主题的网站时，<span class="accentTextColor">浏览器会选择其中三个与出版和广告合作伙伴分享</span>。这些第三方将使用该信息来投放与兴趣 / 主题相关的广告。</p><p data-vmark="77fd">总体而言，Topics API 似乎比 FLoC 更通用。与第三方 Cookie 相比，谷歌希望“让用户更容易理解和管理主题”。</p>
