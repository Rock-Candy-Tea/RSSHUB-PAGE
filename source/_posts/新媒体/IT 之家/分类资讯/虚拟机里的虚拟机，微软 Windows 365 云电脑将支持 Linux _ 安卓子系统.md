<p data-vmark="0549"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 17 日消息，去年 7 月，微软发布了其虚拟化云操作系统产品 Windows 365 云电脑，企业能够在云端获得完整的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a>/11 体验，价格为 20 美元到 162 美元不等。</p><p data-vmark="2c46" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2021/8/443922e4-3380-49b2-a50d-f1c645ed1057.png" title="虚拟机里的虚拟机，微软 Windows 365 云电脑将支持 Linux / 安卓子系统" referrerpolicy="no-referrer"></p><p data-vmark="e6f8">现在，微软宣布正在为 Windows 365 云电脑带来<span class="accentTextColor">嵌套虚拟化</span>的支持，包括支持 Linux 和 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 的 Windows 子系统的预览版。这意味着企业可以在 Hyper-V Windows 365 虚拟机中运行 Hyper-V，并直接在其中运行 <span class="accentTextColor">Linux 和 WSA</span>。</p><p data-vmark="37b2" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/74e2ceb2-d811-428f-81af-2b0ade13d916.png" w="1440" h="342" title="虚拟机里的虚拟机，微软 Windows 365 云电脑将支持 Linux / 安卓子系统" width="1440" height="195" referrerpolicy="no-referrer"></p><p data-vmark="4c22">IT之家了解到，该功能只会为“<span class="accentTextColor">某些许可证类型</span>”启用，可能是更昂贵的 Windows 365 企业版，但目前还不清楚具体区分。该功能在 <a href="https://www.microsoft.com/en-us/microsoft-365/roadmap?filters=New%20Last%20Week&searchterms=89004" target="_blank">Microsoft 365 路线图</a>中被跟踪为功能 ID 89004，预计将在下个月发布预览，<span class="accentTextColor">2022 年 12 月</span>全面上线。</p>
