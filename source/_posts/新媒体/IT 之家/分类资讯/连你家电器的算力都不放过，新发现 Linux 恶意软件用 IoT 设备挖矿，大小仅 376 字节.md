<p data-vmark="82e2">继电脑和手机后，<strong>挖矿病毒</strong>也盯上了 IoT 设备。</p><p data-vmark="22c6">无论是智能冰箱、彩电还是洗衣机，但凡有点算力的（物联网和端侧）设备都可能被这种病毒感染，用于挖掘加密货币等。</p><p style="text-align: center;" data-vmark="656d"><img src="https://img.ithome.com/newsuploadfiles/2022/9/58d00f81-e89d-4bbb-bf19-df75814ff492.png" w="386" h="305" title="连你家电器的算力都不放过，新发现 Linux 恶意软件用 IoT 设备挖矿，大小仅 376 字节" width="386" height="305" referrerpolicy="no-referrer"></p><p data-vmark="4db4">AT&T Alien Labs 新发现的 Linux 恶意软件 <strong>Shikitega</strong> 就是一例。</p><p data-vmark="c6a5">相比之前的一些 IoT 设备，Shikitega 更加隐蔽，总共只有 <strong>376 字节</strong>，其中代码占了 300 字节。</p><p data-vmark="f7e7">那么，这个新型恶意软件究竟是如何感染设备的？</p><h2 data-vmark="c6de">利用加壳技术“隐身”</h2><p data-vmark="2033">具体来说，Shikitega 核心是一个很小的 ELF 文件（Linux 系统可执行文件格式）。</p><p style="text-align: center;" data-vmark="ebd4"><img src="https://img.ithome.com/newsuploadfiles/2022/9/a9a5b657-18f8-42cb-9ebc-636d614e34aa.png" w="1080" h="557" title="连你家电器的算力都不放过，新发现 Linux 恶意软件用 IoT 设备挖矿，大小仅 376 字节" width="1080" height="423" referrerpolicy="no-referrer"></p><p data-vmark="97cd">这个 ELF 文件加了动态壳，以规避一些安全防护软件的查杀。</p><p data-vmark="251d"><strong>加壳</strong>，指利用特殊算法压缩可执行文件中的资源，但压缩后的文件可以独立运行，且解压过程完全隐蔽，全部在内存中完成。</p><p data-vmark="a99b">动态壳则是加壳里面更加强力的一种手段。</p><p data-vmark="d0ac">从整体过程来看，Shikitega 会对端侧和 IoT 设备实施多阶段感染，控制系统并执行其他恶意活动，包括加密货币的挖掘（这里 Shikitega 的目标是门罗币）：</p><p style="text-align: center;" data-vmark="018a"><img src="https://img.ithome.com/newsuploadfiles/2022/9/ea74ca6d-0ce2-43cb-b35e-f0f01b2ab8ff.png" w="1080" h="871" title="连你家电器的算力都不放过，新发现 Linux 恶意软件用 IoT 设备挖矿，大小仅 376 字节" width="1080" height="661" referrerpolicy="no-referrer"></p><p data-vmark="8160">通过漏洞利用框架 Metasploit 中最流行的编码器 Shikata Ga Nai（SGN），Shikitega 会运行多个解码循环，每一个循环解码下一层。</p><p style="text-align: center;" data-vmark="de84"><img src="https://img.ithome.com/newsuploadfiles/2022/9/6a8a7bfa-f1ed-45a6-966c-53f27bd1149f.png" w="1080" h="588" title="连你家电器的算力都不放过，新发现 Linux 恶意软件用 IoT 设备挖矿，大小仅 376 字节" width="1080" height="446" referrerpolicy="no-referrer"></p><p data-vmark="fee7">最终，Shikitega 中的有效载荷（恶意软件的核心部分，如执行恶意行为的蠕虫或病毒、删除数据、发送垃圾邮件等的代码）会被完全解码并执行。</p><p data-vmark="db2e">这个恶意软件利用的是 CVE-2021-4034 和 CVE-2021-3493 两个 Linux 漏洞，虽然目前已经有修复补丁，但如果 IoT 设备上的旧版 Linux 系统没更新，就可能被感染。</p><p data-vmark="aa6e">事实上，像 Shikitega 这样感染 IoT 设备的恶意软件已经很常见了。</p><p data-vmark="c1a8">例如在今年三月，AT&T Alien Labs 同样发现了一个用 Go 编写的恶意软件 <strong>BotenaGo</strong>，用于创建在各种设备上运行的僵尸网络（Botnets）。</p><p data-vmark="8542">对此有不少网友吐槽，IoT 设备的安全性堪忧：</p><p style="text-align: center;" data-vmark="27a1"><img src="https://img.ithome.com/newsuploadfiles/2022/9/08ee53d9-a664-451d-adcf-4ed511e3a00b.png" w="1080" h="265" title="连你家电器的算力都不放过，新发现 Linux 恶意软件用 IoT 设备挖矿，大小仅 376 字节" width="1080" height="201" referrerpolicy="no-referrer"></p><p data-vmark="8f1f">也有网友认为，IoT 设备应该搞 WiFi 隔离，不然就会给病毒“可乘之机”：</p><p style="text-align: center;" data-vmark="6831"><img src="https://img.ithome.com/newsuploadfiles/2022/9/1da45aa3-1cca-47c7-bd89-cb76995f70f3.png" w="1080" h="401" title="连你家电器的算力都不放过，新发现 Linux 恶意软件用 IoT 设备挖矿，大小仅 376 字节" width="1080" height="304" referrerpolicy="no-referrer"></p><p data-vmark="6db6">而除了 IoT 设备，更多人的关注点则放在了 Linux 系统的安全上。</p><h2 data-vmark="509c">Linux 恶意软件数量飙升 650%</h2><p data-vmark="5946">这几年来，Linux 恶意软件的多样性和数量都上升了。</p><p data-vmark="6669">根据 AV-ATLAS 团队提供的数据，新的 Linux 恶意软件的数量在 2022 年上半年达到了历史新高，发现了近 170 万个。</p><p data-vmark="97b5">与去年同期（226324 个恶意软件）相比，新的 Linux 恶意软件数量飙升了近 <strong>650%</strong>。</p><p data-vmark="b42b">除了 Shikitega，近来发现的流行 Linux 恶意软件也变得更加多样，已知的包括 BPFDoor、Symbiote、Syslogk、OrBit 和 Lightning Framework 等。</p><p style="text-align: center;" data-vmark="2a5d"><img src="https://img.ithome.com/newsuploadfiles/2022/9/6e60e34e-ffa0-40a2-894a-db3040389c73.png" w="1080" h="605" title="连你家电器的算力都不放过，新发现 Linux 恶意软件用 IoT 设备挖矿，大小仅 376 字节" width="1080" height="459" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="e31d"><strong>△</strong> 图源 AV-ATLAS</p><p data-vmark="032f">对此有网友提出疑惑，正因为 Linux 开源，它似乎无论如何都会面临病毒和恶意软件的泛滥？</p><p style="text-align: center;" data-vmark="e339"><img src="https://img.ithome.com/newsuploadfiles/2022/9/18e5836d-2cfa-46f2-9849-173b6a5c9307.png" w="1080" h="225" title="连你家电器的算力都不放过，新发现 Linux 恶意软件用 IoT 设备挖矿，大小仅 376 字节" width="1080" height="171" referrerpolicy="no-referrer"></p><p data-vmark="319d">有网友回应称，一方面，虽然旧的 Linux 系统可能充满漏洞、成为病毒的“温床”，但它在经过升级、打了补丁之后就会变好。</p><p data-vmark="8e7e">另一方面，开发恶意软件本身也不是“有手就能做”的事情。</p><p data-vmark="e734">毕竟安全研究人员会不断修复并堵上所有漏洞，而恶意软件开发者必须在他们修复前找到漏洞、开发出恶意软件，还得让它们“大流行”，最终实现自己的目的。</p><p style="text-align: center;" data-vmark="0925"><img src="https://img.ithome.com/newsuploadfiles/2022/9/0bb1ed4d-552d-430a-85b2-bb359b17c6ed.png" w="1080" h="432" title="连你家电器的算力都不放过，新发现 Linux 恶意软件用 IoT 设备挖矿，大小仅 376 字节" width="1080" height="328" referrerpolicy="no-referrer"></p><p data-vmark="5d8c">要是你家还有在用老旧 Linux 系统的设备，要注意及时升级 or 做好网络隔离等安全措施~</p><p data-vmark="eee0"><span class="referenceTitle">参考链接：</span></p><ul class="custom_reference
