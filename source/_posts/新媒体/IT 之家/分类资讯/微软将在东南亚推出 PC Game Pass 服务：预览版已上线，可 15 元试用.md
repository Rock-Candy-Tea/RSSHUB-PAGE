<p data-vmark="7691"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 29 日消息，PC Game Pass 是微软 Xbox Game Pass 订阅在 PC 端的名称，今日微软宣布，该订阅将进入<span class="accentTextColor">东南亚的五个国家</span>。</p><p data-vmark="0f3f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/46449a0c-3e04-4167-9896-b122dea7730d.jpg@s_2,w_820,h_418" w="1280" h="653" title="微软将在东南亚推出 PC Game Pass 服务：预览版已上线，可 1.5 元试用" srcset="https://img.ithome.com/newsuploadfiles/2022/3/46449a0c-3e04-4167-9896-b122dea7730d.jpg 2x" width="1280" height="418" referrerpolicy="no-referrer"></p><p data-vmark="8bde">PC Game Pass 服务即将来到印度尼西亚、马来西亚、菲律宾、泰国和越南，目前已开放<span class="accentTextColor">预览计划</span>，玩家可以在预览期间以 1 马来西亚林吉特（<span class="accentTextColor">约 1.5 元人民币</span>）试用 PC Game Pass。</p><p data-vmark="33f6">微软将测试东南亚本地的支付系统，并在该地区寻找玩家对 Xbox 功能进行反馈。</p><p data-vmark="daa2">IT之家了解到，PC Game Pass 预览版包含 100 多种高品质游戏的库，其中包括 Xbox Game Studios 旗下首发的新游戏、B 社的经典游戏，以及每月 EA Play 会员低价资格等，而且还将不断地添加新游戏。</p><p data-vmark="5fab">有兴趣的玩家可以在微软应用商店下载 <a href="https://www.microsoft.com/en-us/p/xbox-insider-hub/9pldpg46g47z" target="_blank">Xbox Insider 应用</a>，并选择 PC Game Pass 东南亚市场拓展。</p><p data-vmark="ad87" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/e59dc504-7bf1-4a6a-8e70-3e88fc4a9a5f.png" w="1440" h="643" title="微软将在东南亚推出 PC Game Pass 服务：预览版已上线，可 1.5 元试用" width="1440" height="366" referrerpolicy="no-referrer"></p>
