<p data-vmark="f7bf">随着 Win11 的开发重点由 22H2 转移到 23H2，一些系统级组件也逐步得到更新。近日 Microsoft Edge 正式升级到 <span class="accentTextColor">102 正式版</span>，尽管官方并没有大张旗鼓地宣传，但我们仍能从中挖掘到很多不一样的变化。都有什么呢？一起来看吧。</p><p style="text-align: center;" data-vmark="d526"><img src="https://img.ithome.com/newsuploadfiles/2022/6/5deb0a6c-a63a-4b04-8323-6be687ba0820.jpg" w="700" h="398" alt="Edge浏览器 Edge新功能" title="微软 Edge 浏览器 102 悄悄更新三项功能：视频实时字幕、可变刷新率、新版 Win11 UI" width="700" height="398" referrerpolicy="no-referrer"></p><h2 data-vmark="179f">1.视频实时字幕</h2><p data-vmark="7648">Win11 22H2 新增加了一项实时字幕功能，不过只能用于本地视频。而在新版 Microsoft Edge 中，<span class="accentTextColor">这项功能开始被正式应用到在线视频中</span>。</p><p style="text-align: center;" data-vmark="b696"><img src="https://img.ithome.com/newsuploadfiles/2022/6/95d8de5f-a091-4ac2-bf5c-fc635dc27aa0.jpg" w="700" h="373" alt="Edge浏览器 Edge新功能" title="微软 Edge 浏览器 102 悄悄更新三项功能：视频实时字幕、可变刷新率、新版 Win11 UI" width="700" height="373" referrerpolicy="no-referrer"></p><p data-vmark="f98d">点击“<span class="accentTextColor">设置-辅助功能-辅助字幕</span>”，即可看到新增加的“实时字幕（预览）”开关。开启该功能后，当检测到网页上有视频（音频）播放时，Edge 即会进行字幕呈现。总体来说，它与 Win11 的实时字幕功能同根同源，在速度与翻译质量上也相差无几，非常不错。</p><p style="text-align: center;" data-vmark="1cc8"><img src="https://img.ithome.com/newsuploadfiles/2022/6/adf3a4dc-a6a9-4806-b1e1-91be355610f0.gif" w="807" h="512" alt="Edge浏览器 Edge新功能" title="微软 Edge 浏览器 102 悄悄更新三项功能：视频实时字幕、可变刷新率、新版 Win11 UI" width="807" height="512" referrerpolicy="no-referrer"></p><p data-vmark="9ca8">遗憾的是，和系统实时字幕功能一样，Edge 的这项功能目前也<span class="accentTextColor">仅能支持英语</span>。因此最多就是便宜了那些“生肉党”，不过随着微软陆续增加多语种支持，相信这项功能的前景将更为广阔。</p><h2 data-vmark="a370">2.可变刷新率</h2><p data-vmark="b909">很早以前，Win11 就已经支持可变刷新率技术。通过让电脑在 60Hz 与高刷新率（如：120Hz、144Hz、165Hz 等）之间自动变化，来实现电量与流畅度间的一个平衡。而这一次，<span class="accentTextColor">可变刷新率开始出现在 Microsoft Edge 中</span>。</p><p style="text-align: center;" data-vmark="0765"><img src="https://img.ithome.com/newsuploadfiles/2022/6/919414cf-2bb3-42d9-82e5-553fa919eb0b.jpg" w="700" h="351" alt="Edge浏览器 Edge新功能" title="微软 Edge 浏览器 102 悄悄更新三项功能：视频实时字幕、可变刷新率、新版 Win11 UI" width="700" height="351" referrerpolicy="no-referrer"></p><p data-vmark="0e3e">想要体验这项功能，首先要确保你的电脑支持可变刷新率技术（显示器刷新率高于 120Hz、WDDM 3.0）。通过在地址栏中输入“edge://flags/”，搜索“Boost”，将“<span class="accentTextColor">Boost screen refresh rate when scrolling</span>”设为“Enable”，即可打开新版 Edge 的可变刷新率功能。</p><p data-vmark="e2cf">目前 Edge 的这项功能，主要用于页面滚动，即在阅读文字时采用低刷新率，页面滚动时采用高刷新率。这样的设计一方面可以让页面滚动更流畅，另一方面也能让电脑的续航时间更长。</p><h2 data-vmark="9c0a">3.新版 UI</h2><p data-vmark="f842">Edge 的 UI 设计一直都是大家吐槽的焦点，毕竟 Win11 都已经发布很长时间了，这 UI 还是当年的 Win10 风格。不过从这一版开始，我们终于可以开启全新的 <span class="accentTextColor">Win11 UI</span> 了。</p><p style="text-align: center;" data-vmark="bffb"><img src="https://img.ithome.com/newsuploadfiles/2022/6/fa51b8ac-c606-4ea9-9112-85a8dbeccc5f.jpg" w="700" h="263" alt="Edge浏览器 Edge新功能" title="微软 Edge 浏览器 102 悄悄更新三项功能：视频实时字幕、可变刷新率、新版 Win11 UI" width="700" height="263" referrerpolicy="no-referrer"></p><p data-vmark="fa75">新版 UI 采用 Mica 效果设计，因此一个很明显的变化，就是整体的色彩明快度更高。同时新版大量采用<span class="accentTextColor">圆角化元素</span>，标题栏、选项卡，甚至收藏夹中都能看到大量圆角加入。总体来说和现版相比，新 UI 在整体感觉上更加现代，同时也更能匹配 Win11 本身的调调。</p><p style="text-align: center;" data-vmark="3fc6"><img src="https://img.ithome.com/newsuploadfiles/2022/6/173ba1e1-49b0-4c94-a220-211df4aa59c8.jpg" w="700" h="340" alt="Edge浏览器 Edge新功能" title="微软 Edge 浏览器 102 悄悄更新三项功能：视频实时字幕、可变刷新率、新版 Win11 UI" width="700" height="340" referrerpolicy="no-referrer"></p><p data-vmark="dd8a">开启方法共分三步，首先打开地址栏，输入“edge://flags/”进入实验功能面板，然后搜索“show experimental”，将“<span class="accentTextColor">Show experimental appearance settings</span>”设为“Enable”。重启 Edge 后，继续打开“设置-外观”，勾选右侧的“在标题栏和工具栏中显示 Windows 11 视觉效果（预览）”和“使用圆角浏览器选项卡（预览）”。重启浏览器后，新 UI 便生效了。</p><p style="text-align: center;" data-vmark="c013"><img src="https://img.ithome.com/newsuploadfiles/2022/6/8e01289a-a84a-4844-854b-0e63ca831d67.jpg" w="700" h="419" alt="Edge浏览器 Edge新功能" title="微软 Edge 浏览器 102 悄悄更新三项功能：视频实时字幕、可变刷新率、新版 Win11 UI" width="700" height="419" referrerpolicy="no-referrer"></p><p data-vmark="93d4">这里需要注意的是，由于新版 UI 不兼容之前的浏览器主题，<span class="accentTextColor">因此需要首先将“主题”修改为“默认值”</span>，否则将无法看到效果。</p><h2 data-vmark="1ddd">写在最后</h2><p data-vmark="7653">总体来说，在版本号突破 100 这一整数关口后，Edge 又一次开始发力了。特别是在和系统功能的匹配中，新版本表现出了更加友好的态势。目前这些功能已经可以在正式版（102.0.1245.39）中体验，感兴趣的小伙伴不妨一试。</p>
