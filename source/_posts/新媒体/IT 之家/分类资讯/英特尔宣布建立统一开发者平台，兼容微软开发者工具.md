<p data-vmark="1061">10 月 28 日上午消息，英特尔 On 创新技术峰会今日召开，英特尔宣布建立统一的开发者平台（Developer Zone），推出最新的 oneAPI 2022 工具包和 oneAPI 卓越中心，<span class="accentTextColor">旨在让开发者通过一个开放且标准统一的编程环境，便可获取人工智能、云、5G/边缘和游戏领域的参考设计、工具包等资源。</span></p><p data-vmark="8811"><img src="https://img.ithome.com/newsuploadfiles/2021/10/8563f25f-8dc8-4c07-9292-e560c0431116.jpg@s_2,w_820,h_328" w="1000" h="400" alt="ICS&#39; Contributions to the Intel Developer Zone | ICS" title="英特尔宣布建立统一开发者平台，兼容微软开发者工具" srcset="https://img.ithome.com/newsuploadfiles/2021/10/8563f25f-8dc8-4c07-9292-e560c0431116.jpg 2x" width="1000" height="328" referrerpolicy="no-referrer"></p><p data-vmark="dc06">据介绍，oneAPI 是英特尔推出的统一编程模型，能够简化跨不同架构的开发工作，让开发者摆脱专有语言和编程模型的束缚。峰会上，英特尔推出了统一的英特尔开发者目录，其中包含英特尔关键软件产品，以及优化后的英特尔 DevCloud 开发环境，用于测试和运行于英特尔最新硬件上的工作负载和多种软件工具。</p><p data-vmark="42e2">最新推出的 oneAPI 2022 工具包拥有 900 项新功能，并且通过统一的 C++/SYCL/Fortran 编译器和 Data Parallel Python，为 CPU 和 GPU 增加了跨架构的软件开发能力，同时通过扩展 Microsoft Visual Studio Code 集成和 Microsoft WSL2 支持，提高了开发者的生产力。</p><p data-vmark="5f5f">“oneAPI 是通向加速计算的一条高效智能路径，能够有效避免目前专有编程模型的经济和技术负担。让开发者自由做决策、释放硬件全部性能以及安心部署和开发软件。”英特尔方面表示。</p>
