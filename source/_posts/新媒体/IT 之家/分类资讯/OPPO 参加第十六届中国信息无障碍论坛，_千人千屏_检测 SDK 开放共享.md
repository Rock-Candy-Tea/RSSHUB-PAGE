<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a>7 月 29 日消息 第十六届中国信息无障碍论坛暨全国无障碍环境建设成果展示应用推广活动于昨日在哈尔滨举办，<span class="accentTextColor">本次论坛与活动的主题为“智能・无障碍”</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/2c8451e8-3bf8-4425-b24e-418dce1bd7e9.jpeg" w="1272" h="850" title="OPPO 参加第十六届中国信息无障碍论坛，“千人千屏”检测 SDK 开放共享" width="1272" height="548" referrerpolicy="no-referrer"></p><p>本次中国信息无障碍论坛倡导更多的企业关注色弱人群以及更多的有障碍人群。</p><p>相关数据显示，全球有 16% 的人群为色弱色盲，他们眼中的世界因为缺失一些色彩，给工作和生活带来很多不便，甚至失去了一些好的工作机会、兴趣爱好。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/c6d20c5e-97f2-4f30-8c86-937a0152e5f5.jpeg" w="1440" h="962" title="OPPO 参加第十六届中国信息无障碍论坛，“千人千屏”检测 SDK 开放共享" width="1440" height="548" referrerpolicy="no-referrer"></p><p>OPPO 应邀参加第十六届中国信息无障碍论坛，分享了其对于色觉无障碍的思考，<span class="accentTextColor">并表示“千人千屏”技术检测能力 SDK 已经开发完成并将面向全行业共享</span>。</p><p><span class="accentTextColor">OPPO 在 Find X3 系列手机首发“千人千屏”的色彩视觉增强技术</span>，希望帮助色盲色弱人群“看”到接近真实的色彩，缓解生活中的不便。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/a8846414-1179-4c1d-a809-72ac292fd408.jpeg" w="1440" h="962" title="OPPO 参加第十六届中国信息无障碍论坛，“千人千屏”检测 SDK 开放共享" width="1440" height="548" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/ce5c8b13-444b-4341-b0cf-06beafaab3b5.jpeg" w="1440" h="962" title="OPPO 参加第十六届中国信息无障碍论坛，“千人千屏”检测 SDK 开放共享" width="1440" height="548" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/baf75742-f6d9-4564-b190-c8eba33252b1.jpeg" w="874" h="1610" title="OPPO 参加第十六届中国信息无障碍论坛，“千人千屏”检测 SDK 开放共享" width="874" height="1511" referrerpolicy="no-referrer"></p><p>OPPO Find X3 系列首发的“色彩视觉增强”可以<span class="accentTextColor">帮助色觉障碍人群更准确、更高效地识别、获取画面中的信息</span>，解决生活和工作中因为色彩偏差造成障碍。</p><p>未来，OPPO 还将融入更多的色彩偏好维度，并将“色彩视觉增强”打包成 APK，让更多的用户在各种场景中使用到 OPPO 专业的视觉检测及获得属于自己的一套屏幕色彩能力。</p><p>相较于此前行业采用的假同色图检测法，OPPO 自研的“千人千屏”技术，<span class="accentTextColor">不仅可以检测用户是属于哪一类型的色弱色盲，还可以进一步检测用户色弱色盲的轻重程度</span>，多达 766 种色彩视觉方案可以与用户检测结果精确匹配，让用户辨色能力从原有 20% 左右的准确率提升到了 90%。</p><p>IT之家了解到，OPPO 的“色彩视觉增强”有两大核心技术：</p><ul class=" list-paddingleft-2"><li><p>专业检测方法：通过孟塞尔色棋检测方法帮助用户检测自己色觉是否正常，解决了很多人对自己色觉认知不清晰的问题。</p></li><li><p>色彩维度更多：目前提供 700 + 种调试方案，更有针对性，未来还将持续增加。</p></li></ul><p>此外，“千人千屏”的色彩视觉增强技术只是 OPPO 众多无障碍技术中的一种，在 2017 年，OPPO 与信息无障碍研究会合作，针对视障人群进行手机使用交互习惯研究。同年 12 月，<span class="accentTextColor">OPPO 开始针对 ColorOS 系统进行无障碍优化</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/4a655123-3628-4e22-bc03-3645804aa131.jpeg" w="803" h="1478" title="OPPO 参加第十六届中国信息无障碍论坛，“千人千屏”检测 SDK 开放共享" width="803" height="1478" referrerpolicy="no-referrer"></p><p>截至目前，<span class="accentTextColor">OPPO 已经提供了 21 个谷歌原生无障碍功能，和 3 个特色功能，包括千人千屏、语音字幕和无障碍颜色模式</span>。例如面向视听障碍人群的 TalkBack，开启后系统会加入绿色外框，根据用户手势来进行跳动，在视觉显示的同时，还加入了语音朗读，使得视听障碍用户能够迅速知道自己的操作位置，简单直白的操作逻辑和贴心的语音提示，以及触觉、音效等多个方面共同来提升视听障碍人士的用机体验。</p>
