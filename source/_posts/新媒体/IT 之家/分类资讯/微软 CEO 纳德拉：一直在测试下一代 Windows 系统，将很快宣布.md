<p>北京时间 5 月 26 日早间消息，在本周的 Build 2021 大会上，微软没有花大多时间谈论 Windows 发展计划，为什么呢？因为微软准备在另一个单独场合介绍操作系统的发展。在 Build 主题演讲中，微软 CEO 萨提亚・纳德拉（Satya Nadella）告诉听众，说他最近几个月一直在测试下一代 Windows 系统。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/764fa6d3-319c-4a87-b4b4-cb0afe4370dc.jpg" w="630" h="419" title="微软 CEO 纳德拉：一直在测试下一代 Windows 系统，将很快宣布" width="630" height="419" referrerpolicy="no-referrer"></p><p>纳德拉称：“很快，我们就会分享 Windows 更新，它是过去 10 年最重要的一次更新，新系统会为开发者、创作者带来更大的经济机会。在过去几个月我一直在测试系统，对于下一代 Windows 我很是期待。我们的承诺是这样的：微软将为今天的每一位 Windows 开发者创造机会，欢迎每一位寻找最有创造力、最新、最开放平台的创作者，希望他们开发、分发应用并获得收入。很快我们就会分享更多细节。”</p><p>最近几个月，微软一直在为 Windows 开发新程序店，操作系统的 UI 也会有明显变化。例如，Windows Store 将会允许开发者提交任何 Windows 应用，包括 Chrome、Firefox 浏览器。还有传闻暗示，微软将会允许第三方商务平台型 App 进驻，也就是说开发者可以避开微软分成，针对 App 微软会分走 15% 的收入，游戏分走 12%。</p><p>纳德拉提到“下一代 Windows”，这样的提法很有趣，因为过去微软总是说 Windows 10 就是一切，现在口吻变了，这说明除了用户界面会有变化，Windows 品牌也会有很大调整。</p><p>上周微软发布消息称，不会推出 Windows 10X，这款操作系统是针对双屏设备开发的。不过微软会将 Windows 10X 的优秀元素植入 Windows 10。</p>
