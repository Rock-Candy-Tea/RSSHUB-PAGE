<p data-vmark="08df">北京时间 3 月 18 日早间消息，据报道，流媒体音乐服务 Spotify <span class="accentTextColor">正计划在其流媒体服务中融合区块链技术和非同质化代币（NFT）</span>，此举令加密货币和音乐行业都颇感兴奋，认为有望借助 NFT 来提升艺人的收入。</p><p style="text-align: center;" data-vmark="ef63"><img src="https://img.ithome.com/newsuploadfiles/2022/3/a06bf3bf-cd28-415e-8837-f66a78447f51.jpg" w="550" h="309" alt title="流媒体音乐巨头 Spotify 拟试水 NFT：招募 Web3 人才" width="550" height="309" referrerpolicy="no-referrer"></p><p data-vmark="8707">最近的两则招聘启事显示，<span class="accentTextColor">Spotify 计划为跟 Web3 有关的项目招募人才</span>。所谓 Web3，<span class="accentTextColor">指的是一个由区块链支持的网络</span>，加密货币支持者希望这个网络能够从科技巨头的手中重新夺回互联网的控制权。</p><p data-vmark="377e">在 Spotify 之前，已经有不少科技公司也加入了 NFT 浪潮，从而应对来自加密货币创业公司的挑战。Meta CEO 马克・扎克伯格（Mark Zuckerberg）本周也证实，该公司旗下的 Instagram 很快也将支持 NFT。包括 Twitter 和 Reddit 在内的其他社交媒体公司也在为展示或交易 NFT 开发新的功能。</p><p data-vmark="122c">NFT 使用区块链技术来确认数字资产的所有权。根据市场追踪公司 Nonnfungible.com 的数据，去年的 NFT 交易额达到 177 亿美元，其中绝大多数都是虚拟艺术品、游戏和收藏品。</p><p data-vmark="5628">尽管呈现泡沫化趋势的 NFT 市场最近几周增速放缓，但加密货币行业的许多人是都认为音乐有望成为 NFT 行业新应用模式的爆发点。例如，可以借此出售数字唱片，或者使用 NFT 来解锁演唱会的相关业务，包括特许商品和后台通行证。</p><p data-vmark="2a11">Snoop Dogg 和史蒂夫・青木（DJ Steve Aoki）等音乐人都也已经成为著名的 NFT 收藏家，而 Kings of Leon 和 Grimes 也以 NFT 方式出售了自己的音乐作品。华纳和环球等大型唱片公司也与 NFT 项目建立合作关系，包括带有 Bored Ape Yacht Club 角色的虚拟品牌。</p><p data-vmark="88f4">包括 Royal 和 Catalog 在内的创业公司已经开始出售由音乐人制作的 NFT，他们通常将加密货币宣传成一种可以替代 Spotify 等流媒体服务的商业模式，宣称这种模式的利润更高，不必向平台支付佣金。</p><p data-vmark="a332">Spotify 在 NFT 领域的招聘似乎还处在探索阶段。该公司在其“实验性增长”团队的工程师招募启事中提到了 Web3：“这个小而完整的团队负责通过 Web3 等新技术推动增长。”</p><p data-vmark="f8a4">Spotify 的另外一份招聘启事是为“创新和市场情报”部门招募一名管理者，表明这家流媒体音乐服务计划寻找一名在“内容、创作、媒体 Web 和新兴技术行业”都有经验的人才，帮助其定义“Spotify 的登月计划”。在科技行业，“登月计划”指的是野心勃勃的新项目。</p><p data-vmark="21d6">Spotify 拒绝对 NFT 计划发表评论。</p><p data-vmark="8fd6">此次招募 Web3 人才并非 Spotify 首次进军加密货币领域：该公司曾是 Facebook 命运多舛的加密货币项目 Diem 的早期合作者。</p><p data-vmark="15f5">当这个项目在 2019 年以“天秤币”（Libra）的名字推出时，Spotify CEO 丹尼尔・艾克（Daniel Ek）通过一档播客节目表示，加密货币和区块链可以让“Spotify 等服务的用户直接向艺人支付费用”，尤其是为跨境支付和传统银行账号不普及的地区提供便利。</p><p data-vmark="734a">“这将打开巨大的机会。一夜之间，日本的用户或许就能为阿根廷的创作者付费。这还会为我们继续推进自己的使命创造了巨大机会。”艾克当时说。</p>
