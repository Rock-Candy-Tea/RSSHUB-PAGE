referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/6ed403b9-cddb-4966-87a9-413a8b15b1cb.jpg" w="690" h="1221" title="2021 腾讯视频大剧片单公布：《三体》系列、《英雄联盟》电竞剧、《鬼吹灯》系列" width="690" height="1221" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/315317f3-01ac-4467-a3b4-05ef970a58d0.jpg" w="690" h="1221" title="2021 腾讯视频大剧片单公布：《三体》系列、《英雄联盟》电竞剧、《鬼吹灯》系列" width="690" height="1221" referrerpolicy="no-referrer"></p><p>会上，腾讯视频介绍了 IP 精品系列内容规划，包括国民级科幻 IP《三体》，并宣布将对三体系列进行持续开发；以《司藤》作者尾鱼为核心的“尾鱼宇宙”，将开启《四月间事》《西出玉门》《三线轮回》《龙骨焚香》《枭起青壤》等剧集的联合开发；已有多部佳作诞生的“鬼吹灯宇宙”也公布了后续作品的最新进展。此外，腾讯视频还发布了《精锐行动》《英雄联盟电竞剧》等热血竞技系列内容，并表示将持续针对优质 IP《斗罗大陆》系列进行全产业链开发。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/2d6cf63c-730c-47a2-b9ab-4257882200e3.jpg" w="690" h="937" title="2021 腾讯视频大剧片单公布：《三体》系列、《英雄联盟》电竞剧、《鬼吹灯》系列" width="690" height="937" referrerpolicy="no-referrer"></p><p>会上，腾讯视频公布了精品电影合作计划第一批片单，其中包括已进入前期开发的《鬼吹灯》系列《三线轮回》等冒险类网络电影；通过小人物的喜怒哀乐来打动观众的《环球迪斯科》《夜市风云》《四平警事》等喜剧类佳作；由《战狼》动作指导团队打造的《断金》等动作类作品；《斗破苍穹》等融合了独特东方美学和震撼视效的玄幻类影片；《狐妖小红娘・月红 2》等动画电影；“神话传说系列”的女性题材影片及展示时代精神的《幸存者 1937》《林海雪原》等主旋律影片，实现了多类别覆盖。另外，腾讯视频还宣布了正与星辉公司进行自制电影战略合作，周星驰导演也期待与腾讯视频合作，在网络电影这个新的领域给大家带来更多好的作品。</p><p>IT之家获悉，在 2021 腾讯视频影视年度发布活动上，腾讯视频副总裁、企鹅影视高级副总裁韩志杰透露，为去除内容注水，腾讯视频将试点尝试改变“按集采购”的行业通行法则。腾讯副总裁、腾讯视频首席执行官孙忠怀在中国网络视听大会表示，正是长视频内容注水，造成切条搬运式短视频横行。“包括高热剧集在内，节奏太慢、剧情拖沓、注水戏太多的质疑持续存在，用户已经习惯倍速看剧，对注水作不再买单，这些都给行业敲响了警钟。”</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/c897553f-bebe-4083-b970-80ed53ee548c.jpg" w="690" h="10745" title="2021 腾讯视频大剧片单公布：《三体》系列、《英雄联盟》电竞剧、《鬼吹灯》系列" width="690" height="10745"
