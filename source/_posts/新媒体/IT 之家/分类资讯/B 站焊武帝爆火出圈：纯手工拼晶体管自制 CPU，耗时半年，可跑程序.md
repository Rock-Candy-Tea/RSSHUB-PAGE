<p>一个人，到底能肝到什么程度？</p><p>最近B站上大火的一个视频，或许给了这个问题一个完美诠释：</p><p><strong>纯！手！工！自制 CPU！</strong></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/18757241-1f18-440c-b4af-1989dbd63bda.png" w="1080" h="585" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="1080" height="444" referrerpolicy="no-referrer"></p><p>这位叫做“奶味的”Up 主，耗时整整半年，用他那双勤劳的双手，“逐点”焊接，最终打造出了一个完整 CPU！</p><p>手工做一个 CPU 能用么？</p><p>当然可以，他还做了一个 demo 展示，来看一下效果吧。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/29c6ee8e-66ea-4d8b-af3e-a1900a6c278c.gif" w="731" h="392" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="731" height="392" referrerpolicy="no-referrer"></p><p>纯手工“肝”出一个 CPU，还成功运行了流水灯效果，直接引发了一大波网友们的惊呼：</p><blockquote><p>肝帝！焊武帝！</p><p>人长在肝上了！</p></blockquote><p>（感受下这满屏的 Respect）</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/24da0d93-3591-46b7-befd-b7e35afecbee.gif" w="700" h="271" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="700" height="271" referrerpolicy="no-referrer"></p><p>接下来，让我们一同看下“肝帝”自制 CPU 的完整过程。</p><h2>纯手工自制 CPU</h2><p>其实在发布这个视频之前，Up 主便在贴吧中“开玩笑”的说了句：</p><blockquote><p>CPU 太贵了，买了显卡就买不起 CPU，手工捏个出来……</p></blockquote><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/a1c5e867-9bda-42bc-a5d1-6df065f69d49.png" w="1080" h="533" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="1080" height="405" referrerpolicy="no-referrer"></p><p>万万没想到，真的是说干就干。</p><p>据这位 Up 主介绍，他主要采用了 3 个原材料：</p><p><span class="accentTextColor">二极管、三极管和电阻</span>。</p><p>通常来讲，CPU 的结构可以大致分为运算逻辑部件、寄存器部件和控制部件等。</p><p>而他最先“下手”的，就是 CPU 的寄存器部分。</p><p>Up 主设计了一个 <span class="accentTextColor">6 位的移位寄存器</span>：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/8f62980a-6dba-4367-afa4-7a889a320fc6.png" w="936" h="526" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="936" height="461" referrerpolicy="no-referrer"></p><p>它的作用不仅能是用来存储，还能在时钟信号的控制下，将数据进行进行逐次右移或左移。</p><p>简单来说就像是一个交警叔叔，当数据从一个方向进来的时候，这个移位寄存器可以指挥它什么时候该往哪里走。</p><p>Up 主处理的第二个结构，是<span class="accentTextColor">程序计数器 （PC）</span>。</p><p>它作用简单来说，就是记录程序运行的位置。</p><p>而这也是整个项目下来最耗时、最复杂的部分，花了整整 3 个月之久。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/152810ef-9e7b-4153-9f57-f6619710491c.png" w="1080" h="434" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="1080" height="330" referrerpolicy="no-referrer"></p><p>程序计数器涉及的功能那可就多了。</p><p>像最基本的就是挨个字节读完指令后，计数要自动 + 1；而 CPU 重启之后，计数便会清零。</p><p>而且在不同的条件之下，还要能实现直接跳转、调用函数、函数返回等功能。</p><p>听着就有够复杂的了。</p><p>奈何，除此之外，还时常伴随着各种各样的“玄学问题”。</p><p>Up 主就举了个例子：</p><blockquote><p>花了一个多星期才在最深处找到一只焊反的二极管……</p></blockquote><p>但功夫不负有心人，在经历 3 个月令人头秃的时光之后，最复杂的模块还是被他搞定了。</p><p>上电测试也没有问题：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/7336d1f9-515f-41aa-811c-895293337f2c.gif" w="640" h="360" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="640" height="360" referrerpolicy="no-referrer"></p><p>然后就是硬盘（ROM）和内存（RAM）。</p><p>这是 CPU 外的比较庞大两个部分要手搓一个不太现实。</p><p>因此，Up 主用上了比较容易操作的 hm628512 来做 ROM 和 RAM：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/78688bb8-234c-4e51-a730-4d31610d5d73.png" w="1080" h="327" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="1080" height="248" referrerpolicy="no-referrer"></p><p>▲ 左：RAM 和指针 右：ROM</p><p>再将 ROM 和 RAM 组装上去，现在 CPU 的基础模块已经基本完成了。</p><p>接下来就是打造<span class="accentTextColor">指令译码器</span>。</p><p>它主要是用于把传到这里的 CPU 指令，进行解析运行：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/887bb0f5-90c6-4d12-b884-2b0de53c574a.gif" w="640" h="360" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="640" height="360" referrerpolicy="no-referrer"></p><p>再把做加减乘除的运算器（ALU）加上去：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/42dc091a-8bb4-4ac4-82f1-e9200f231d47.gif" w="640" h="360" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="640" height="360" referrerpolicy="no-referrer"></p><p>最后还得再焊一个<span class="accentTextColor">通用缓存</span>上去：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/e61b8b91-4271-4fa2-8b25-bd12a0e93685.png" w="1080" h="475" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="1080" height="361" referrerpolicy="no-referrer"></p><p>现在，这个全部由三极管、二极管和电阻焊接而成的 CPU 就完成了！</p><p>看到这密密麻麻的元件，工程量的浩大就不用多说。</p><p>也难怪能让一众网友直呼“肝帝”了。</p><h2>还有纯手写最原始代码</h2><p>以为纯手工焊接就完了？</p><p>不不不，还有更硬核的。</p><p>因为这个 CPU 的指令集和架构都是自主研发的，所以没有适配的编程语言。</p><p>那怎么能让它跑起来呢？</p><p>这位 Up 主的对策是：</p><p>纯手写最原始代码 —— <span class="accentTextColor">二进制编程</span>！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/96a3a040-c3e7-4ef6-ac84-7e8855a37758.gif" w="702" h="348" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="702" height="348" referrerpolicy="no-referrer"></p><p>噫吁嚱！噫吁嚱！</p><p>这一手露的，直接引发了网友们的第二波高潮：</p><blockquote><p>手写指令集，牛皮！这才是真・写代码！</p><p>直接上机械码，太狠了吧！</p></blockquote><p>然后…… 然后……</p><p>Up 主竟然就开启了<span class="accentTextColor">上古编程模式 ——“扣”程序</span>！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/8fef7f83-84eb-48bb-89a3-8ee08c733077.gif" w="691" h="376" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="691" height="376" referrerpolicy="no-referrer"></p><p>面对此情此景，怎么一句“绝绝子”了得。</p><p>……</p><p>一切准备工作就绪。</p><p>接下来，便是见证奇迹的时刻。</p><h2>亮灯，跑起！</h2><p>但最开始，程序的运行并不是一帆风顺。</p><p>即便大神重启了几次，跑马灯的效果偶尔还是出现问题。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/68546262-5c08-41c2-9780-30485ad0fce1.gif" w="718" h="391" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="718" height="391" referrerpolicy="no-referrer"></p><p>但 bug 很快被找到了：</p><blockquote><p>有个地方断开了。</p></blockquote><p>与此同时，因为还没有 I/O 接口，所以测试使用的灯是临时寄存器的灯。</p><p>而且左移指令和跳转指令都使用了同一组灯。</p><p>因此，流水灯的效果就不是非常明显。</p><p>于是，大神重新编程，使 CPU 跳转到 0X0F 处运行。</p><p>如此一来，在跳转时灯就是灭的，给左移指令让出了一条路。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/83b6656e-3ca4-404e-a4a2-16bfcb551502.gif" w="719" h="392" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="719" height="392" referrerpolicy="no-referrer"></p><p>一切修改完毕，重新“扣”程序，启动！</p><p>这一次，就没有任何的异常了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/bce5bbf3-05a2-41ef-8869-7c1166482b2e.gif" w="694" h="342" title="B 站焊武帝爆火出圈：纯手工拼晶体管自制 CPU，耗时半年，可跑程序" width="694" height="342" referrerpolicy="no-referrer"></p><h2>走进“肝帝”大神</h2><p>在看完这波“神级”操作之后，想必大家都想了解这位大神。</p><p>量子位帮你实现这个梦想。</p><p><span class="accentTextColor">大神原名林乃卫，广西北海人</span>。</p><p>做这个项目的灵感，来源于在他写代码时分析可执行文件中的二进制。</p><p>那时候大神便萌生出了一想法：</p><blockquote><p>CPU 是怎么执行这一串 0101 的？</p></blockquote><p>在结合数电课程的知识之后，大神猜测到了大致的原理，于是就想试一试。</p><p>最初他想用门级电路来做的，但是感觉难度不是很高，又想把整个过程从零开始理解透。</p><p>于是，便决定从最基础的<span class="accentTextColor">模拟电路</span>开始。</p><p>而整个过程最难的部分，便是“器件每一级的连接”和“CPU 的运行速度”：</p><blockquote><p>在此之前没有看过关于 CPU 原理方面的书籍，是靠自己对门电路的理解而设计的一套电路 ，想设计一套属于自己的架构。</p><p>从最基础的三极管开关到门级电路，网上找的电路图根本没法用，需要自己设计。虽然设计好后理论通过了，但是门级之间组合成功能级器件就有新的问题，功能级的电路再多个 组合起来成为功能模块又会产生新的问题，模块与模块之间连接更会产生新的问题.</p><p>这每一级的问题都很有可能要修改基础电路，然后又会从头开始产生新的问题。</p><p>因为没有专业的设备去调试，我用的是一盏 LED 和蜂鸣器，想尽办法也将速度提高到 100kHz 左右 而已，这是十分慢的，过程也十分艰难。</p></blockquote><p>至于器件方面，大神均是从网上购得，然后再将它们一点一点地焊接成为模块。</p><p>这个 CPU 大致耗费了 <span class="accentTextColor">1000 多个三极管、2000 多个二极管，电阻数量也达到了 2000 多，焊点近万</span>。</p><p>目前的费用大概花了 1000 多元。</p><p>而网友在弹幕中调侃居多的，还有制程方面的问题。</p><p>据大神介绍：</p><blockquote><p>要是非得给它定个工艺制程，那就是 <span class="accentTextColor">2.54mm</span>，比先进的 3 纳米大了将近 1 百万倍。</p></blockquote><p>最后，Up 主还表示，将在接下来的工作中，持续完善 CPU 的功能，让它能够运行更加复杂的程序。</p><p>对此，你期待了吗？</p><p><iframe class="ithome_video bilibili" src="https://player.bilibili.com/player.html?bvid=bv1sy4y1j7Ue&page=1" frameborder="no" width="100%"
