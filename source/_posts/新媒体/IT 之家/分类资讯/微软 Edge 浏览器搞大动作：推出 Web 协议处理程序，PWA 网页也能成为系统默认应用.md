<p data-vmark="0b85"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 21 日消息，微软<a href="https://blogs.windows.com/msedgedev/2022/01/20/getting-started-url-protocol-handlers-microsoft-edge/" target="_blank">宣布</a>，从 Edge 96 版本开始，<span class="accentTextColor">Web 应用将能够在<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>中使用协议处理程序</span>。最新功能将允许已安装的 Web 应用程序（或 PWA）导航预设或自定义协议。</p><p data-vmark="34b9">已安装的 Web 应用程序也将能够向操作系统注册为协议处理程序，并在调用特定协议后启动。换句话说，<span class="accentTextColor">用户可以将网页设置为默认处理程序</span>，比如用户想创建电子邮件，系统将默认打开浏览器中的某个网站。</p><p data-vmark="1a2b"><img src="https://img.ithome.com/newsuploadfiles/2022/1/856f1c16-3b3f-40e4-8a86-87990fde1e88.png" w="1245" h="1346" title="微软 Edge 浏览器搞大动作：推出 Web 协议处理程序，PWA 网页也能成为系统默认应用" width="1245" height="887" referrerpolicy="no-referrer"></p><p data-vmark="9b97">此外，开发者可以通过注册<span class="accentTextColor">以 web+ 为前缀的协议</span>来生成 Web 应用程序自定义方案。协议处理程序可用于 Web 应用通信，其中一个应用直接调用另一个应用并通过自定义协议链接传递数据。</p><p data-vmark="5293">IT之家了解到，微软演示了一个名为 PWinter 的 PWA 网页，该网页可以响应 web+pwinter:// 格式开头的链接。用户点击网页中的链接后，就可以像调用桌面应用那样调用 PWinter。</p><p data-vmark="c25d"><img src="https://img.ithome.com/newsuploadfiles/2022/1/7077c8c1-56b7-4cd5-83a4-f3c670d6bf5e.gif" w="640" h="360" alt="Animation showing a PWA registering for and responding to protocol handlers" title="微软 Edge 浏览器搞大动作：推出 Web 协议处理程序，PWA 网页也能成为系统默认应用" width="640" height="360" referrerpolicy="no-referrer"></p><p data-vmark="9338">微软表示，协议处理程序在 <span class="accentTextColor">Windows、Linux 和 Mac</span> 中受支持，并在 Edge 96 浏览器中发布。微软已将这项工作贡献给 Chromium 开源项目，现在其他基于 Chromium 的浏览器也可以使用该功能。</p>
