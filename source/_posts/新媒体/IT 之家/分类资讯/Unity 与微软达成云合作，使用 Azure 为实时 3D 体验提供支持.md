<p data-vmark="0dc8"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 8 日消息，据微软官方博客消息，<span class="accentTextColor">微软 Azure 成为 Unity 的云合作伙伴</span>，以通过 Unity 引擎构建和运营实时 3D (RT3D) 体验。</p><p style="text-align: center;" data-vmark="d5ae"><img src="https://img.ithome.com/newsuploadfiles/2022/8/50cf807c-7209-44cb-8b1d-01f6424f32ff.png" w="963" h="535" title="Unity 与微软达成云合作，使用 Azure 为实时 3D 体验提供支持" width="963" height="456" referrerpolicy="no-referrer"></p><p data-vmark="72f2">Unity 正在构建一个<span class="accentTextColor">与平台无关的云原生解决方案</span>，以满足从企业到个人开发者的广泛需求。通过让创作者轻松访问 RT3D 模拟工具以及创建真实世界地点和对象的数字复制品的能力，Unity 为创作者提供了制作 RT3D 资产的简单途径，无论是用于游戏还是非游戏世界。</p><p data-vmark="0945">为了支持这种演变，创作者需要一个技术基础设施，而 Unity 选择了微软 Azure。微软称，Azure 专为安全性和全球可扩展性而构建，已经支持一些世界上最大的游戏，并将这些久经考验的学习成果为所有行业的 RT3D 体验提供动力。</p><p data-vmark="71d4">随着对实时模拟的需求成为从电子商务到能源、制造到医疗等各个行业的核心，Unity 和微软正在构建云创作，使 3D 艺术家能够在 Azure 上构建和运行这些体验。</p><p data-vmark="74c0">此外，微软与 Unity 之间的合作伙伴关系，还将使 Unity 游戏开发者能够<span class="accentTextColor">更轻松地通过 Windows 和 Xbox 设备接触到玩家</span>。</p><p data-vmark="59d8">IT之家了解到，2021 年底，Unity 中国团队宣布<a href="https://www.ithome.com/0/592/529.htm" target="_blank">正式推出部署在中国本地云端的云渲染服务</a>，<span class="accentTextColor">免费开放给所有开发者使用</span>。Unity 云渲染几乎不需要任何下载时间，突破终端硬件的限制，可以被应用于建筑、施工、工业产品设计、房地产、市场与广告行业。<span class="accentTextColor">客户端支持 Chrome / Firefox / Safari 以及国内主流<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a></span>，在微信也可以直接打开。</p><p style="text-align: center;" data-vmark="3310"><img src="https://img.ithome.com/newsuploadfiles/2021/12/021b56e5-c6a5-48a5-88f3-20e1de90a8c3.png" w="962" h="469" alt="Unity" title="Unity 与微软达成云合作，使用 Azure 为实时 3D 体验提供支持" width="962" height="400" referrerpolicy="no-referrer"></p>
