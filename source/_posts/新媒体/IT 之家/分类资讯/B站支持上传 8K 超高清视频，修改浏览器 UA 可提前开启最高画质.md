<p data-vmark="d521">最近，<span class="accentTextColor">哔哩哔哩将网站视频的规格升级到了 8K 分辨率</span>，这可谓是国内最强的线上视频画质了。但遗憾的是，B站这次 8K 视频升级并没有全面铺开，一方面仅对少数 Up 主开放 8K 视频规格上传，现在B站也仅有为数不多的演示视频；另一方面<span class="accentTextColor">并不是所有设备都可以观看到 8K 视频</span>！例如手机 App 乃至 PC <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>，都无法开启B站8K 视频的选项。</p><p data-vmark="52e0"><img src="https://img.ithome.com/newsuploadfiles/2021/12/9b54c830-5273-41a4-b6dd-833f7dc62235.png" w="700" h="534" title="B站支持上传 8K 超高清视频，修改浏览器 UA 可提前开启最高画质" width="700" height="534" referrerpolicy="no-referrer"></p><p data-vmark="d0ef">▲ 用浏览器兴冲冲开启B站，发现竟然没有 8K 选项？</p><p data-vmark="4732">根据B站的官方说法，当前开启 8K 画质的方法有两种，一种是在电脑上用 Safari 浏览器（版本 11 以上）打开，另一种则是在支持 8K 解码的智能电视机上打开云视听小电视应用。经过笔者测试，<a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a> 和 Mac 的 Safari 浏览器的确是可以支持 8K 视频的，这意味着，通过一些特殊的方法，广大的 Windows 其他浏览器用户，其实也是可以开启 8K 画质的！今天，就来分享一下方法吧。</p><p data-vmark="c817">这个方法相信很多熟悉电脑的朋友已经猜到了，那就是改 UA 大法。所谓 UA，全称 User Agent，是浏览器标识自己身份的方式。我们<span class="accentTextColor">只需要修改浏览器的 UA</span>，让它伪装成为 Mac 上的 Safari，就可以开启 8K 了。</p><p data-vmark="ee03">浏览器修改 UA 的方法有很多，下面就以 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 上的 Edge 浏览器为例。</p><p data-vmark="19a6">首先，我们安装一款修改 UA 的浏览器扩展，地址如下。</p><p data-vmark="0f04">User-Agent Switcher and Manager：</p><p data-vmark="fcd3"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://microsoftedge.microsoft.com/addons/detail/useragent-switcher-and-m/cnjkedgepfdpdbnepgmajmmjdjkjnifa</span></span></p><p data-vmark="edf2">安装后，浏览器的工具栏就会出现对应的按钮。打开一个B站支持 8K 视频的页面，然后点击该工具的按钮，选择“Safari”和“Mac OS”，接着选择最新版的 Safari 浏览器，最后点击“Apply”。</p><p data-vmark="c0cd"><img src="https://img.ithome.com/newsuploadfiles/2021/12/e910842b-31b0-4801-8b41-e4400dd6ba1d.png" w="700" h="588" title="B站支持上传 8K 超高清视频，修改浏览器 UA 可提前开启最高画质" width="700" height="588" referrerpolicy="no-referrer"></p><p data-vmark="95ba">之后刷新页面，可以发现，B站视频中就出现 8K 视频的选项了！切换后，即可播放。</p><p data-vmark="0b90"><img src="https://img.ithome.com/newsuploadfiles/2021/12/04a89114-3ad5-4c20-9387-612bc3fd740d.png" w="700" h="534" title="B站支持上传 8K 超高清视频，修改浏览器 UA 可提前开启最高画质" width="700" height="534" referrerpolicy="no-referrer"></p><p data-vmark="b76e">▲ 改了 UA 后，8K 超高清的画质选项出现了</p><p data-vmark="74d9">从视频信息可以看到，目前正在播放的的确是 8K 规格的视频，该视频使用了 HEVC 编码，规格在国内网络视频中还是首屈一指的，观感也的确非常高清。</p><p data-vmark="d6bb"><img src="https://img.ithome.com/newsuploadfiles/2021/12/ba9683ab-afa1-493a-8949-1cc61154a5a0.png" w="700" h="522" title="B站支持上传 8K 超高清视频，修改浏览器 UA 可提前开启最高画质" width="700" height="522" referrerpolicy="no-referrer"></p><p data-vmark="d9b3">▲ B站的 8K 视频使用 HEVC 编码</p><p data-vmark="ef61">不过需要注意的是，这个方法属于取巧。或许是由于 Chromium 对 HEVC 的支持不够好，在播放的时候会出现明显的卡顿，实用性不佳。</p><p data-vmark="74df"><img src="https://wx2.sinaimg.cn/large/005OlC5vly1gx9iyrhcx8g30jg0f4kjo.gif" class="lazy" title="B站支持上传 8K 超高清视频，修改浏览器 UA 可提前开启最高画质" referrerpolicy="no-referrer"></p><p data-vmark="3c65">根据B站官方说法，很快B站就会正式加入对 8K 视频的支持，例如 Chrome 和 Edge 浏览器等 8K 标准的视频，将会逐步完成。或许在那时候，Chrome 和 Edge 等播放 8K 也会有比较流畅的体验了。</p><p data-vmark="dca0">总的来说，B站上线 8K 视频的确是一件好事，尽管大家未必有设备分辨率达到 8K，但高分辨率的画面经过缩小后，细节仍会更多。希望国内的视频站再接再厉，为用户带来更好的画质吧。</p>
