<p>7 月 27 日消息，对于久坐的上班族来说，按钮一按，桌面上升，这种电动升降桌听起来真的很美好。</p><p>但慢慢的就有人不满足了：不想按按钮，让桌子自己上升行不行？</p><p>诶，这不就来了。<span class="accentTextColor">有一位美国小哥用树莓派黑入自己的升降桌，最终得到了一台能在随机时间内自动升降的升降桌</span>。</p><p><span class="accentTextColor">该升降桌每隔 45 到 60 分钟桌面就会自动上升</span>，你不想站也得站。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/7508bbea-a594-4e5e-9ebb-e8d16e99dafb.jpg@s_2,w_820,h_614" w="1080" h="809" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" srcset="https://img.ithome.com/newsuploadfiles/2021/7/7508bbea-a594-4e5e-9ebb-e8d16e99dafb.jpg 2x" width="1080" height="614" referrerpolicy="no-referrer"></p><h2>从懒癌开始的开发</h2><p>这位小哥名叫 David Kong，是一位产品经理。</p><p>和很多上班族一样，他拥有一台电动升降桌，美名其曰“养成良好的站立习惯”。</p><p>不过因为沉迷工作而忘记站立这种事还是常常发生。</p><p>甚至就算设置了手机提醒，也常常由于“啊我现在好累要不再坐一会儿”的心理而在升降桌前坐了一整天……</p><p>于是，在人类科技发展的第一驱动力 —— 懒的作用下，小哥决定：要让升降桌不用按按钮就能自动升降。</p><p>说干就干，小哥马上拆开了升降桌的按钮控制盒：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/e908c370-0b73-4327-82eb-658e91fc9a5d.jpg@s_2,w_820,h_442" w="1080" h="582" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" srcset="https://img.ithome.com/newsuploadfiles/2021/7/e908c370-0b73-4327-82eb-658e91fc9a5d.jpg 2x" width="1080" height="442" referrerpolicy="no-referrer"></p><p>好耶！发现一排整整齐齐的引脚：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/04aa964e-a723-4e34-855d-fda81ccb4956.jpg@s_2,w_820,h_415" w="1080" h="546" alt="图片" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" srcset="https://img.ithome.com/newsuploadfiles/2021/7/04aa964e-a723-4e34-855d-fda81ccb4956.jpg 2x" width="1080" height="415" referrerpolicy="no-referrer"></p><p>黑客 DNA 立马就动了！</p><p>小哥找来了两根电线，把它们焊在了按钮“1”和“2”的引脚上：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/f82023f7-5414-456f-b126-3d5b53b43e0a.jpg@s_2,w_820,h_365" w="1080" h="481" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" srcset="https://img.ithome.com/newsuploadfiles/2021/7/f82023f7-5414-456f-b126-3d5b53b43e0a.jpg 2x" width="1080" height="365" referrerpolicy="no-referrer"></p><p>这两个按钮是他之前已经预设好的，一坐一站两个固定高度的按钮。</p><p>然后小哥选择用一种自动化的方式来处理这两个按钮，把两根电线连到了继电器上：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/22bcd5fb-326e-4f18-ab84-6541f6a59049.jpg" w="300" h="293" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" width="300" height="293" referrerpolicy="no-referrer"></p><p>接下来就是<span class="accentTextColor">有规律地发送电流</span>，来激活这两个按钮了。</p><p>小哥第一个想到的就是设计一个定时电路。</p><p>比如使用这种叫做 555 定时器的芯片，可以以固定间隔连接电路：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/bb28e0a5-8504-459c-8bbf-ef1cc00b3205.jpg" w="300" h="225" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" width="300" height="225" referrerpolicy="no-referrer"></p><p>但问题就是它的间隔可能还不到 1 秒。</p><p>如果要把这个数字变为 1000 甚至更大，就需要一系列的定时器。</p><p>初次开发的复杂度就不用说了，后续哪怕是做一些微小的时间改动也会非常麻烦。</p><p>于是 David 小哥果断拒绝，转头就想起了自己老树莓派粉的身份。</p><p>不如连接树莓派写一个简单的定时脚本？这样以后就可以直接从代码层面调整桌子的移速了。</p><p>好，就决定是你了，树莓派！</p><h2>树莓派再立大功</h2><p>小哥买了一个最基础的 Raspberry Pi Zero，又订购了一个外壳，以及一张预装了 rasbian OS 的 Micro SD 卡。</p><p>总共花了大约 20 美元 （折合人民币约 130 块）。</p><p>然后用 USB 数据线把 Pi Zero 和电脑连接起来，开始在终端上进行编程：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/f1182bde-1cf3-44bc-b9b3-1c423c8bab39.png" w="1080" h="671" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" width="1080" height="509" referrerpolicy="no-referrer"></p><p>小哥编写了一个 Python 脚本，功能是能在树莓派上激发一个 GPIO（通用输入输出）引脚：</p><pre class="brush:javascript;toolbar:false">from gpiozero import LED # The LED library allows easy pin control
from time import sleep
import random
relay = LED(17) # I connected the relay to pin 17 and ground
while True:
relay.on()
sleep(1)
relay.off()
sleep(random.randint(45, 60) * 60)</pre><p><span class="accentTextColor">因为希望桌子能以一种更随机、更无意识的方式来升降</span>，所以小哥选了一个 45 到 60 分钟的间隔来作为激发时间。</p><p>这一脚本需要保存到/home/pi/Documents/moveDesk.py 中。</p><p>然后，再将这一行添加到/etc/rc.local：</p><pre class="brush:javascript;toolbar:false ai-word-checked">python /home/pi/Documents/moveDesk.py</pre><p>这样只要树莓派一启动，脚本就能随之运行。</p><p>再把连接着按钮的继电器的输出引脚焊到树莓派上：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/b3a032f9-717f-4502-9085-2ae9ed072096.jpg@s_2,w_820,h_452" w="1080" h="595" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" srcset="https://img.ithome.com/newsuploadfiles/2021/7/b3a032f9-717f-4502-9085-2ae9ed072096.jpg 2x" width="1080" height="452" referrerpolicy="no-referrer"></p><p>最后把这些杂乱的线板一股脑都塞到桌子底下：</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/545a5ea3-0cd1-4a4e-afd4-1442ac1018ef.jpg@s_2,w_820,h_483" w="1080" h="636" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" srcset="https://img.ithome.com/newsuploadfiles/2021/7/545a5ea3-0cd1-4a4e-afd4-1442ac1018ef.jpg 2x" width="1080" height="483" referrerpolicy="no-referrer"></p><p>成了！</p><h2>已成功运行几个月</h2><p>现在，这张自动升降桌已经用了好几个月了。</p><p>David 小哥站立的时间规律了许多，腰背疼痛也得到了缓解。</p><p>网友表示：方法虽然糙了点，但是很管用！从无到有的过程也非常有价值！</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/a86c20e1-7622-4cfa-9283-c4bae1991142.png" w="1080" h="58" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" width="1080" height="44" referrerpolicy="no-referrer"></p><p>而 David Kong 小哥作为一个就职于 Frame.io，业余当当黑客的产品经理，也觉得很有意义。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/0c36bd1d-95fa-42d6-bb62-13a040215b4d.jpg" w="450" h="450" title="美国小哥用树莓派爆改办公桌：可在随机时间内自动升降" width="450" height="450" referrerpolicy="no-referrer"></p><p>毕竟，桌子会在一个随机间隔的时间内自动上升，这就意味着再也不用额外分出注意力来提醒自己应该站立了。</p><p>在无意识间，就保持了一种健康的上班姿势。</p><p>小哥也对此表示表示：</p><blockquote><p>现在已经形成了一种自然而然的健康节奏。有些时候，当桌子开始向上移动时，我就会不自觉地站起来，甚至不知道发生了什么。</p></blockquote>
