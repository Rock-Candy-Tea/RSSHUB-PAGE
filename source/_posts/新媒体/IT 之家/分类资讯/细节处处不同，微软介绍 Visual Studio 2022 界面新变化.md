<p data-vmark="e09b"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 19 日消息，微软预计将于今年 11 月 8 日发布 Visual Studio 的下一个迭代版本，即 Visual Studio 2022。该 IDE 对上一代 Visual Studio 2019 进行了许多改进，包括 64 位版本，支持.NET 6 和 C++ 20，为核心调试器提供了更好的性能等。</p><p data-vmark="e369">除了功能方面的改进，Visual Studio 2022 在外观界面上也有一些显而易见的改动，微软发布了一篇<a href="https://devblogs.microsoft.com/visualstudio/weve-upgraded-the-ui-in-visual-studio-2022/" target="_blank">博客文章</a>，<span class="accentTextColor">介绍了新 IDE 在 UI 上都有哪些不同</span>。</p><p data-vmark="4d9a"><img src="https://img.ithome.com/newsuploadfiles/2021/10/98036c6c-20f9-47db-8266-d934af652700.png" w="898" h="678" title="细节处处不同，微软介绍 Visual Studio 2022 界面新变化" width="898" height="619" referrerpolicy="no-referrer"></p><p data-vmark="2edf">首先，<span class="accentTextColor">Visual Studio 2022 将获得全新图标</span>，如下图所示。微软表示，这些图标是在与开发人员社区讨论并在整个预览过程中审查他们的反馈后最终确定的，由此产生的图标既现代又直观。</p><p data-vmark="00a8">▼ Visual Studio 2022 为右侧图标</p><p data-vmark="ff72"><img src="https://img.ithome.com/newsuploadfiles/2021/10/7d61da6f-11ed-43ae-b54f-294850cde844.jpg@s_2,w_820,h_388" w="1024" h="484" title="细节处处不同，微软介绍 Visual Studio 2022 界面新变化" srcset="https://img.ithome.com/newsuploadfiles/2021/10/7d61da6f-11ed-43ae-b54f-294850cde844.jpg 2x" width="1024" height="388" referrerpolicy="no-referrer"></p><p data-vmark="54a6">微软指出，其对 Visual Studio 界面进行改造的验收标准是一致性、易读性和熟悉性。这些措施包括将图标放置在正确的位置，使用颜色识别参考点，纠正对比度，并在熟悉的地方保留颜色和符号。</p><p data-vmark="7ec3"><img src="https://img.ithome.com/newsuploadfiles/2021/10/a43616dc-5240-433d-a9eb-384fda9d0e58.jpg@s_2,w_820,h_605" w="1009" h="744" title="细节处处不同，微软介绍 Visual Studio 2022 界面新变化" srcset="https://img.ithome.com/newsuploadfiles/2021/10/a43616dc-5240-433d-a9eb-384fda9d0e58.jpg 2x" width="1009" height="605" referrerpolicy="no-referrer"></p><p data-vmark="6b3f">此外，微软发现深色主题已经取代浅色主题，成为 Visual Studio 开发者最流行的选择。因此，<span class="accentTextColor">微软调整了整个 IDE 的颜色对比度，使设计更易于访问</span>，特别是对于使用深色主题的人，其中的一个关键改进是高亮显示的边框。</p><p data-vmark="a00b">▼ Visual Studio 2022 为右侧，选中项的边框将高亮，更易于区分</p><p data-vmark="ff57"><img src="https://img.ithome.com/newsuploadfiles/2021/10/56fcd05b-7ae0-4888-94c5-0cc3767b4f34.png" w="1395" h="742" title="细节处处不同，微软介绍 Visual Studio 2022 界面新变化" width="1395" height="436" referrerpolicy="no-referrer"></p><p data-vmark="6342">最后，<span class="accentTextColor">Visual Studio 2022 减少了强调色的强度</span>，以减少开发者的眼睛疲劳，而默认字体改为了 Cascadia Mono。</p><p data-vmark="1b30">▼ Visual Studio 2022 为右侧，蓝色的强调色改为了紫色</p><p data-vmark="0382"><img src="https://img.ithome.com/newsuploadfiles/2021/10/fdc4c4a3-0f6e-43d8-aa5f-41995717e7b7.png" w="1398" h="724" title="细节处处不同，微软介绍 Visual Studio 2022 界面新变化" width="1398" height="425" referrerpolicy="no-referrer"></p><p data-vmark="e06b">IT之家了解到，不习惯新暗黑模式界面的用户可以下载 <a href="https://marketplace.visualstudio.com/items?itemName=MadsKristensen.DarkTheme2019" target="_blank">Visual Studio 2019 扩展</a>来改回原来的设计。</p>
