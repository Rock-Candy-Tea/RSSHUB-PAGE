<p data-vmark="6621"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 7 日消息，近日有消息称，Systemd 开发者 Lennart Poettering 已离开工作了 15 年的红帽，据 Poronix 报道，他已经加入了微软。</p><p data-vmark="5b7b">Lennart Poettering 是 Linux 系统 Systemd 服务的开发者，对于现代 Linux 桌面版起到了重要作用。Poronix 获得的消息证实，<span class="accentTextColor">Lennart Poettering 已经加入了微软，将继续专注于 Systemd 开发</span>。</p><p data-vmark="6735">IT之家了解到，微软此前已经雇佣了不少 Linux 开发者和著名的开源开发者，比如 Python 之父 Guido van Rossum、GNOME 之父 Miguel de Icaza，Xamarin 开发者 Nat Friedman 还担任了 GitHub 首席执行官。</p><p data-vmark="a2b5">Lennart 在红帽工作了 15 年之久，除了著名的 Systemd 服务，<span class="accentTextColor">他还领导了 PulseAudio、Avahi 项目</span>。Systemd 是 Linux 系统的一个引导程序，可取代传统的 init 进程，实现系统初始化时服务的并行启动，同时达到降低 Shell 的系统开销的效果。</p><p data-vmark="e14b">微软目前在 Linux 的兼容方面持续发力，为 Windows 11/10 系统推出了 Linux 子系统，甚至也为 Linux 系统本身作出了贡献。</p>
