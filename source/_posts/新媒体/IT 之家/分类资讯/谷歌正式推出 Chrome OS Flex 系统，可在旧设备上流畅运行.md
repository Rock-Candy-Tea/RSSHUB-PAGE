            <p data-vmark="6c72"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 15 日消息，谷歌今天正式发布了 Chrome OS Flex，这是为企业和学校设计的新版本 Chrome OS，可在旧 PC 和 Mac 上安装和运行。谷歌今年早些时候首次开始测试 Chrome OS Flex，该公司现在已经解决了 600 个错误，今天向企业和学校推出了稳定版。</p><p style="text-align: center;" data-vmark="3cf3"><img src="https://img.ithome.com/newsuploadfiles/2022/7/6db36311-fa2c-4c9d-bce2-a48f50a9c334.png" w="746" h="478" title="谷歌正式推出 Chrome OS Flex 系统，可在旧设备上流畅运行" width="746" height="478" referrerpolicy="no-referrer"></p><p data-vmark="be01">Chrome OS Flex 主要是为使用旧 Windows PC 的企业设计的，谷歌一直在测试和验证来自宏基、华硕、戴尔、惠普、联想、LG、东芝和更多 OEM 的设备。Flex 甚至会在一些旧的 Mac 上运行，包括一些有 10 年历史的 MacBooks。</p><p data-vmark="b6fa">对旧硬件的支持是 Chrome OS Flex 的最大卖点，因为企业不必为了获得最新的现代操作系统而抛弃现有硬件。<span class="accentTextColor">超过 400 台设备经过认证可以使用</span>，而安装也很简单，<span class="accentTextColor">使用 USB 驱动器就可以安装</span>。</p><p data-vmark="067c">“我们每天都在进行更多的认证，即使你的设备还没有得到认证，你仍然可以尝试 Chrome OS Flex，”谷歌产品、企业和教育总监 Thomas Riedl 解释说。在没有得到官方支持的设备上，用户可能会遇到一些小问题，例如不稳定，或启动问题。</p><p data-vmark="a2c0">谷歌 2020 年收购了 Neverware 公司，之后研发了 Chrome OS Flex 系统，Neverware 公司之前销售一种名为 CloudReady 的应用程序，允许用户在旧 PC 上运行 Chrome OS 系统。</p><p data-vmark="3b84">谷歌公司在一篇博文中说：“通过在你现有的硬件上安装 ChromeOS Flex，你不仅获得了良好的体验，而且还为一项重要的事业做出了贡献。在全球范围内，每年有 4000 万吨电子垃圾产生--这相当于每秒钟扔掉 800 台笔记本电脑。将设备升级到 ChromeOS Flex，而不是完全替换它们，是减少浪费和防止这一日益严重的问题的一种有影响的方式。此外，运行 ChromeOS Flex 的设备比其他设备平均少消耗 19% 的能源。”</p><p data-vmark="8cc9">谷歌表示，除了通过赋予旧机器新的生命来提高可持续性之外，该操作系统还能保护用户免受勒索软件和恶意软件的威胁，并在后台进行更新，因此用户不必浪费停机时间。值得注意的是，Chrome OS Flex 与 Chrome OS 处于相同的发布周期，因此它可以即时接收所有的更新。</p><p data-vmark="e54d">并非所有的企业和学校都能轻松转换到 Chrome OS Flex，特别是如果他们依赖现有的 Windows 应用和系统，而这些应用和系统是以 Windows 为基础设计的。</p><p data-vmark="85a1">Chromebook 无疑向世界展示了 Windows 的强大替代方案，尤其是在教育领域，Chrome OS Flex 是那些想远离 Windows 的人的又一选择。</p><p data-vmark="a4f3">谷歌为用户提供了在 50 台设备上试用 30 天的 Chrome OS Flex 的机会，然后才需要购买许可证，目前关于其定价还未知。</p>
