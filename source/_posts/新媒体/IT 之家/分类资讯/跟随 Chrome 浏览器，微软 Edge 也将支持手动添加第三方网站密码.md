<p data-vmark="28fd"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 4 日消息，去年 11 月份，谷歌 Chrome Canary <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>引入了手动向密码管理器添加密码的功能。现在，Redditor Leopeva64-2 注意到该功能也可以在微软 Edge Canary 中使用。</p><p data-vmark="f289"><img src="https://img.ithome.com/newsuploadfiles/2022/1/5c39b001-2f0a-4c4b-8c0c-8bab8b4b1618.jpg" w="760" h="428" alt="You can now manually add passwords to Edge Canary" title="跟随 Chrome 浏览器，微软 Edge 也将支持手动添加第三方网站密码" width="760" height="428" referrerpolicy="no-referrer"></p><p data-vmark="a18a">据介绍，<span class="accentTextColor">用户可以直接在 Edge Canary 中使用该功能</span>，不需要调整任何实验选项来开启，软件版本为 Build 99.0.1124.0。</p><p data-vmark="c8a5">IT之家了解到，在过去的几个月里，微软一直在为 Edge Canary 启用新功能。11 月，Edge Canary 默认开启标签搜索，12 月，Edge Dev for <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 启用了要求身份验证以自动填充内容的选项。</p><p data-vmark="e1ec">而 Chrome 也在学习 Edge 的一些功能，比如 Chrome Canary 最新版工具栏中加入下载图标，与微软 Edge 浏览器类似，<span class="accentTextColor">下载时在工具栏弹出下载图标，下载完成后消失</span>。</p><p data-vmark="8c4c"><img src="https://img.ithome.com/newsuploadfiles/2021/12/2e62dac5-eda3-4549-a6ed-3f33c03607f5.png" title="跟随 Chrome 浏览器，微软 Edge 也将支持手动添加第三方网站密码" referrerpolicy="no-referrer"></p>
