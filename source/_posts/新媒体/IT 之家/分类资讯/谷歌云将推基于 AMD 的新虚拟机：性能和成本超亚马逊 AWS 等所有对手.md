<p>北京时间 6 月 22 日早间消息，谷歌云将于本周四推出一系列<span class="accentTextColor">基于 AMD 技术的高性能虚拟机</span>。谷歌表示，对于需要扩大云计算基础设施的互联网大公司来说，新虚拟机服务的速度更快、成本更低。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210323_101926_703.png" title="谷歌云将推基于 AMD 的新虚拟机：性能和成本超亚马逊 AWS 等所有对手" referrerpolicy="no-referrer"></p><p>虚拟机是在云端运行程序、存储数据的软件。谷歌的 8 号员工、首位工程副总裁乌尔斯・赫兹勒（Urs Holzle）在接受采访时表示，将于周四推出的新的云计算技术“无论是<span class="accentTextColor">在性能上还是在成本上</span>，都超越了当前云计算行业<span class="accentTextColor">任何竞争对手</span>的产品”。他特别指出，这些竞争对手也包括亚马逊 AWS。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/ab24290e-503d-4ec1-b784-c5aa1912a098.png" w="550" h="277" title="谷歌云将推基于 AMD 的新虚拟机：性能和成本超亚马逊 AWS 等所有对手" width="550" height="277" referrerpolicy="no-referrer"></p><p>谷歌此前已经与两家知名公司 Twitter 和 Snap 合作，测试这个新的“Tau 虚拟机”服务。Twitter 平台负责人尼克・托尔诺（Nick Tornow）在客户证言中表示：“初步测试显示，<span class="accentTextColor">性能有</span><span class="accentTextColor">两位数比例的提升</span>，我们对此感到很兴奋。”Snap 也发表了类似观点。</p><p>Tau 虚拟机针对大量基础功能对性价比进行了优化，这些功能包括 Twitter 的内容排序及 Snapchat 的搜索。谷歌表示，新的虚拟机服务将于<span class="accentTextColor">第三季度</span>在谷歌云平台上推出。这款新服务比竞争对手产品快 56%，在同样性能下成本低 42%。</p><p>谷歌此次与 AMD 合作开发了新的虚拟机服务，采用了 AMD 的<span class="accentTextColor">第三代 EPYC 处理器</span>。负责该项目的 AMD 高级副总裁弗雷斯特・诺罗德（Forrest Norrod）表示：“这种价格优化会立即对利润产生影响。”</p><p>赫兹勒也表示：“很明显，我们拥有了云计算行业速度最快的虚拟机。看看其他云计算公司会如何反应很有趣。”</p><p>在此前云计算市场的竞争中，谷歌远远落后于对手。市场分析公司 Canalys 估计，2021 年 4 月，亚马逊 AWS 的市场份额为 32%，微软 Azure 为 19%，而<span class="accentTextColor">谷歌云只有 7%</span>。不过，谷歌云的增长速度很快，2021 年第一季度营收为 40.2 亿美元，高于分析师预计的 39.9 亿美元。去年第四季度，谷歌云的营收为 38.3 亿美元。这是谷歌首次在财报中单独公布云计算业务的业绩。</p><p>业内专家认为，新的 Tau 虚拟机有助于谷歌云保持快速增长的势头。</p><p>Constellation Research 首席分析师霍尔格・穆勒（Holger Mueller）表示：“谷歌正在取得进展，实现对 AWS 和微软等主要竞争对手的差异化。AWS 和微软将不得不追赶，而谷歌每次取得进展都会有助于市场份额提升。”</p><p>Moor Insights & Strategy 创始人及首席分析师帕特里克・莫尔海德（Patrick Moorhead）表示：“我认为这会引起很大的关注。虽然这个产品单独来看不会提升谷歌云的市场份额，但总体而言谷歌云与亚马逊 AWS 和微软 Azure 的竞争会越来越激烈。”</p><p>赫兹勒在谷歌供职已有 22 年，见证了许多技术上的变化。他表示，在过去 5 年中，还没有见过科技产品在价格和性能上出现“如此大的飞跃”。关于云计算市场，他表示：“我们都在相互竞争，这是一场永远不会结束的比赛。”</p><p>关于新的虚拟机服务是否也适用于 AWS 的母公司亚马逊，赫兹勒说：“我们会给他们有吸引力的报价。”</p>
