<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a>3月24日消息 微软在 2019 年将自家的安全防护软件 Defender 带到了 macOS 中。</p><p>近日微软宣布，Defender Mac 版已支持 <span class="accentTextColor">USB 设备访问管理</span>，可对外部 USB 设备（包括 SD 卡）的<span class="accentTextColor">读取、写入、执行等权限进行自定义策略</span>，以保障用户的系统安全。</p><p>当用户在 Mac 电脑上连接 USB 设备时，Defender 将会在<span class="accentTextColor">屏幕右上角进行提醒</span>，用户可在软件中进一步配置权限。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210324_194814_178.png" w="1286" h="859" title="微软 Defender Mac 版已支持 USB 防护：可自定义访问权限，防止恶意文件传输" width="1286" height="548" referrerpolicy="no-referrer"></p><p>微软表示，该功能支持产品 ID、供应商 ID 和序列号规则，对不同的产品进行分层控制，<span class="accentTextColor">如果当前 USB 设备与规则不匹配时，访问权限将被设置为顶级，以免恶意文件通过 USB 传输</span>。</p><p>IT之家了解到，该功能已在<span class="accentTextColor">预览版 Defender 软件</span>上线，用户可在安全中心启用预览功能，以开启 USB 防护。</p>
