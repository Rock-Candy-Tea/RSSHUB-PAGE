<p data-vmark="7f53">要说苹果 Macbook 最忠实的大神用户，<strong>Linus Torvalds</strong> 当属其一。</p><p data-vmark="5f06">当支持 M2 芯片的几个 Linux 项目还在测试时，Linus 就已经想办法给自己的 <strong>M2 Macbook Air</strong> 移植了 Fedora Linux。</p><p data-vmark="fc9b">Linux Kernel 5.19 正式版，就是他在自己的 M2 Macbook Air 上发布的。</p><p data-vmark="25ea">今年的 Linux Plumbers 大会（LPC，Linux 内核开发者峰会）上，Linus 在接受 ZDNet 采访时，谈到了在 M2 Mac 上使用 Linux 的感受、Rust 引入 Linux 的进度、以及一些工作上的事情：</p><p data-vmark="cae8">我真不是一个工作狂。</p><p data-vmark="f621">但同时他也透露，自己“<strong>并不需要游戏</strong>”。</p><p style="text-align: center;" data-vmark="7e23"><img src="https://img.ithome.com/newsuploadfiles/2022/9/a1a2f832-e2d4-40eb-b3e4-e68112341f01.png" w="1080" h="712" title="Linux 之父是个果粉：给自用 M2 Mac 装上 Fedora Linux，除了图形加速和 Chrome 都搞定了" width="1080" height="541" referrerpolicy="no-referrer"></p><p data-vmark="9c44">一起来看看他这次访谈透露的几点核心内容。</p><h2 data-vmark="e3de">给 M2 Mac 移植 Fedora Linux</h2><p data-vmark="71cd">Linux 有好几个发行版（Fedora、Ubuntu、Asahi 等），Linus 通常用的是 Fedora。</p><p data-vmark="e0c8">因此，他在 M2 Macbook Air 出来后，立刻就入手了一台，并成功向这台电脑上移植了 Fedora Linux。</p><p data-vmark="3bca">无论是编译内核、还是给内核打补丁，都可以正常工作，而且 Macbook 的 16GB 内存 + 256GB 硬盘配置也还不错。</p><p style="text-align: center;" data-vmark="382e"><img src="https://img.ithome.com/newsuploadfiles/2022/9/f10068c4-8b4f-48cb-b006-fc2c5f73b264.png" w="1080" h="721" title="Linux 之父是个果粉：给自用 M2 Mac 装上 Fedora Linux，除了图形加速和 Chrome 都搞定了" width="1080" height="547" referrerpolicy="no-referrer"></p><p data-vmark="b384">但值得一提的是，Linus 并不推荐其他人这么做（高端玩家可以上手一试）。</p><p data-vmark="2c75">不过，这个“Linus 版”Fedora Linux 系统也有一些限制。</p><p data-vmark="5002">一方面，它<strong>不能处理 3D 图形</strong> （不支持 M2 FPU 图形加速），无法使用屏幕调光等功能；另一方面，<strong>Chrome</strong> 也还不能在 Fedora Linux 上运行。</p><p data-vmark="0d52">对于不支持 M2 FPU 的问题，Linus 的回应是“反正我也不需要游戏”。（手动狗头）</p><p style="text-align: center;" data-vmark="cc27"><img src="https://img.ithome.com/newsuploadfiles/2022/9/c5ed1f55-a1a2-451d-97aa-1baa21ddbd40.png" w="556" h="518" title="Linux 之父是个果粉：给自用 M2 Mac 装上 Fedora Linux，除了图形加速和 Chrome 都搞定了" width="556" height="518" referrerpolicy="no-referrer"></p><p data-vmark="1ee2">但不能用 Chrome 会有点麻烦，因为 Linus 习惯用 Chrome 保存密码，现在只能通过手机端 Chrome 同步。</p><p data-vmark="8a82">不过整体来说问题不大，毕竟 Chromium 可以正常运行。</p><h2 data-vmark="223f">“我不是一个工作狂”</h2><p data-vmark="e558">虽然给 Linux 内核做了不少贡献，但提及工作量，Linus 表示自己并不是一个“工作狂”：</p><p data-vmark="c8ae">我之所以现在还在做它，是因为我随时可以抽身。但一段时间后我又会觉得无聊，所以回来继续工作。</p><p data-vmark="c384">事实上，相比参加这次的 LPC 2022，Linus 坦言自己更想潜水。就在会议召开前，他已经在博奈尔（Bonaire，西印度群岛中荷属安的列斯群岛的岛屿）潜了 6 天水。</p><p data-vmark="c77d">对于“真正的工作狂”这一称号，Linus 颁给了 Greg Kroah-Hartman。Greg 同样是 Linux 核心开发者之一，目前在 Linux 基金会工作，是 Linux 内核稳定分支（-stable）的维护者：</p><p data-vmark="ce94">虽然很多任务可以自动化完成，但总归是无穷无尽的。他却每周都能完成。</p><p style="text-align: center;" data-vmark="2594"><img src="https://img.ithome.com/newsuploadfiles/2022/9/627c6f35-a408-4171-8824-dfcff420ca73.png" w="1080" h="702" title="Linux 之父是个果粉：给自用 M2 Mac 装上 Fedora Linux，除了图形加速和 Chrome 都搞定了" width="1080" height="533" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="ae2d"><strong>△</strong> 强者的发型</p><h2 data-vmark="94d9">Linux 内核啥时候引入 Rust？</h2><p data-vmark="fe28">虽然早在几个月前，Linus 就透露过会在 Linux 内核中引入 Rust。</p><p data-vmark="a325">BUT，在 8 月发布的 Linux 6.0 第一个候选版本中，Rust for Linux 的补丁却并没有被合并……</p><p data-vmark="c03a">被问及合并时间时，Linus 表示“不敢保证”，甚至在 10 月份的 6.1 版本也不一定出炉。</p><p style="text-align: center;" data-vmark="3eda"><img src="https://img.ithome.com/newsuploadfiles/2022/9/235b2380-1b01-45de-8a2c-c4d0400a1371.png" w="730" h="384" title="Linux 之父是个果粉：给自用 M2 Mac 装上 Fedora Linux，除了图形加速和 Chrome 都搞定了" width="730" height="384" referrerpolicy="no-referrer"></p><p data-vmark="f0a0">毕竟，为了确保能在 Linux 内核中工作，必须对 Rust 进行非官方扩展，例如 Linux Rust NVMe 驱动就需要 70 多项扩展。</p><p data-vmark="8eba">Linus 表示，最重要的是 Rust 编译器的稳定性，目前 GCC Rust 还不稳定，还需要使用 Clang。</p><p data-vmark="788e">但对于合并 Rust for Linux 这件事，Linus 在访谈中一再保证：</p><p data-vmark="3e99">这一定会成功的。</p><p data-vmark="798b"><span class="referenceTitle">参考链接：</span></p><ul
