<p data-vmark="ccef"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 21 日消息，尽管微软会定期为其支持的软件发布质量和安全更新，但作为消费者，尽快安装更新也是很重要的。</p><p data-vmark="2933">今日，微软发布了一份关于已经修补但在尚未更新的配置上被利用的漏洞的公告。</p><p data-vmark="5acd">早在 11 月，微软就将两个漏洞标记为<span class="accentTextColor"> CVE-2021-42287 和 CVE-2021-42278</span>，将它们描述为“Windows Active Directory 域服务提权漏洞”。这些问题允许恶意行为者在破坏常规用户帐户后轻松获得 Active Directory 中的域管理员权限。</p><p data-vmark="eb29">微软发布了三个补丁可立即部署在域控制器上：</p><ul class=" list-paddingleft-2"><li><p data-vmark="7518"><a href="https://support.microsoft.com/en-us/topic/kb5008102-active-directory-security-accounts-manager-hardening-changes-cve-2021-42278-5975b463-4c95-45e1-831a-d120004e258e">KB5008102—Active Directory Security Accounts Manager hardening changes (CVE-2021-42278)</a></p></li><li><p data-vmark="dd7f"><a href="https://support.microsoft.com/en-us/topic/kb5008380-authentication-updates-cve-2021-42287-9dafac11-e0d0-4cb8-959a-143bd0201041">KB5008380—Authentication updates (CVE-2021-42287)</a></p></li><li><p data-vmark="42d4"><a href="https://support.microsoft.com/en-us/topic/november-14-2021-kb5008602-os-build-17763-2305-out-of-band-8583a8a3-ebed-4829-b285-356fb5aaacd7">KB5008602(OS Build 17763.2305) Out-of-band</a></p></li></ul><p data-vmark="3ebd">IT之家了解到，上述补丁已经发布了数周，但是利用这些漏洞的概念验证工具于 12 月 12 日公开披露。黑客可以利用它通过针对未打补丁的域控制器对 Active Directory 执行权限提升攻击。</p><p data-vmark="8e03">因此，微软发布公告，<span class="accentTextColor">要求客户尽快修补适用的系统</span>。在其技术博客文章中，微软还深入探讨了有关如何检测入侵指标的详细信息，并附上了一些 Advanced Hunting 查询，大家可以<a href="https://techcommunity.microsoft.com/t5/security-compliance-and-identity/sam-name-impersonation/ba-p/3042699" target="_blank">点此查看</a>更多详细信息。</p>
