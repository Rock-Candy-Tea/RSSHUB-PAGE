<p data-vmark="1bf2"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 3 日消息，微软于北京时间 11 月 2 日 0 点的 Ignite 2021 开发大会上公布了几个和 OneDrive 有关的公告，宣布 OneDrive 将原生支持苹果 M1 芯片以及基于 ARM 平台的 Windows 设备。</p><p data-vmark="9838"><img src="https://img.ithome.com/newsuploadfiles/2021/11/bce43d15-554a-4fdd-b1d8-27e16f9e6ed1.png" w="1149" h="383" title="微软官宣 OneDrive 将原生支持苹果 M1 芯片以及 ARM 平台的 Windows 设备" width="1149" height="273" referrerpolicy="no-referrer"></p><p data-vmark="35d5">微软表示欢迎大家加入开发者预览通道以获得原生支持 ARM 处理器架构的全新预览版 One Drive。<span class="accentTextColor">并宣布面向苹果 M1 芯片用户的原生预览版 OneDrive 将于今年 12 月发布</span>，用户也可以通过开发者预览通道进行申请。该原生版本将充分调用苹果 M1 芯片，让用户在使用上感受到和之前版本之间的性能差异。</p><p data-vmark="9db6"><img src="https://img.ithome.com/newsuploadfiles/2021/11/08818d42-8f43-454f-8774-2ce22b8bda78.jpeg" w="700" h="378" title="微软官宣 OneDrive 将原生支持苹果 M1 芯片以及 ARM 平台的 Windows 设备" width="700" height="378" referrerpolicy="no-referrer"></p><p data-vmark="56e2">IT之家了解到，新版本的 OneDrive 还针对 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 系统优化了使用体验，<span class="accentTextColor">微软将 OneDrive 直接添加到 Windows 11 的文件资源管理器中</span>，在用户登录的同时文件资源管理器顶部功能区的文件共享选项将会默认为 OneDrive 共享，方便用户传输和分享文件。</p><p data-vmark="4f83"><img referrerpolicy="no-referrer"></p><p data-vmark="8687">作为微软 Windows 生态的一部分，<span class="accentTextColor">微软最后表示 OneDrive 网页版现可在 Windows 和 Mac 的<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>上作为渐进式网络应用程序（PWA）使用</span>，用户的大部分工作无需使用客户端即可在 OneDrive 网页版上轻松完成，同时此次更新还为 OneDrive 网页版增加了照片编辑功能。</p><p data-vmark="7c09"><img
