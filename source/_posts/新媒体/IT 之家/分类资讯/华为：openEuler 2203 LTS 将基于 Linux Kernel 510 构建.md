<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 23 日消息 Linux Kernel 5.10 维护周期已确定从 2 年延长至 6 年。华为本周表示，按照版本计划，openEuler 22.03 将于明年发布，并将基于 Linux Kernel 5.10 构建。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/d104448b-7c83-4b1b-a8c8-b5cece45737c.jpg" title="华为：openEuler 22.03 LTS 将基于 Linux Kernel 5.10 构建" referrerpolicy="no-referrer"></p><p>华为数据显示，华为在 Linux Kernel 5.10 补丁贡献排名 Top1，Linux Kernel 5.6-5.11 代码总贡献量占比 5.8%、排名 Top3。</p><p>据介绍，根据 openEuler 社区的版本生命周期，openEuler 22.03 LTS 版本将于明年发布，届时将会使用 Linux Kernel 5.10 作为该版本的内核。</p><p>IT之家了解到，Linux 5.10 LTS 原本只计划维护到 2022 年底。然而，5 月上旬的公告显示，该版本的维护时间将会延长至 2026 年底。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/a34e6e26-e080-471f-b765-8bdd38d4e84c.png" title="华为：openEuler 22.03 LTS 将基于 Linux Kernel 5.10 构建" referrerpolicy="no-referrer"></p>
