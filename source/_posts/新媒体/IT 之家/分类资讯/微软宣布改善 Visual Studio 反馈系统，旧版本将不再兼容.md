<p data-vmark="cf30"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 19 日消息，微软近日发表博客宣布，<span class="accentTextColor">将改善 Visual Studio 的反馈系统</span>，旧版本的 Visual Studio 将不再兼容提供反馈。</p><p data-vmark="c71b"><img src="https://img.ithome.com/newsuploadfiles/2021/12/1015e7cf-4ecd-476d-be0f-b6a01d050a49.png" w="891" h="172" alt title="微软宣布改善 Visual Studio 反馈系统，旧版本将不再兼容" width="891" height="158" referrerpolicy="no-referrer"></p><p data-vmark="1ddf">微软 Visual Studio 团队每个月都会收到 3000 条反馈项目，这是直接联系到微软工程团队并确保反馈项目会被看到、阅读并采取行动的最佳方式。</p><p data-vmark="d6dc">微软希望在提供反馈时提供最佳的客户体验，并确保可以听到用户的声音。出于这个原因，微软打算升级帮助菜单和反馈中心中可用的发送反馈选项：</p><p data-vmark="95a2"><img src="https://img.ithome.com/newsuploadfiles/2021/12/7cb52b84-b9ac-44ea-9978-29c74e92fe91.png" w="906" h="675" alt title="微软宣布改善 Visual Studio 反馈系统，旧版本将不再兼容" width="906" height="611" referrerpolicy="no-referrer"></p><p data-vmark="32ec">据介绍，<span class="accentTextColor">新的反馈系统将使工程团队更容易跟踪、组织和处理反馈</span>，提高微软对问题的响应能力，进而为用户提供更好的体验。</p><p data-vmark="4ece">但是，新的系统也要付出代价，那就是旧版本的 Visual Studio 将无法发送反馈，其中包含：</p><ul class=" list-paddingleft-2"><li><p data-vmark="e591">15.9.0 – 15.9.34</p></li><li><p data-vmark="778c">16.4.0 – 16.4.20</p></li><li><p data-vmark="8472">16.5.0 – 16.6.X</p></li></ul><p data-vmark="e398">IT之家了解到，微软表示，如果你想继续提交反馈，<span class="accentTextColor">请升级到 16.7 或 2021 年 4 月之后的任何 LTSC 服务版本</span>。</p>
