<p data-vmark="32e1">10 月 21 日凌晨，特斯拉公布了截止 9 月 30 日的 2021 财年第三季度财报。</p><p data-vmark="e48f">财报显示，<span class="accentTextColor">特斯拉汽车第三季度总营收达到创纪录的 137.57 亿美元</span>，同比与环比均保持增长；归属于普通股股东的净利润为 16.18 亿美元，<span class="accentTextColor">同比增长 389%</span>，比较上一季度的 11.42 亿美元增长了 41.7%。</p><p data-vmark="05ae"><img src="https://img.ithome.com/newsuploadfiles/2021/10/f20f3608-c960-41e5-b5c9-d0495dab26f5.png" w="630" h="398" title="特斯拉 Q3 财报解读：营收净利创纪录，中国新造车压力来了" width="630" height="398" referrerpolicy="no-referrer"></p><p data-vmark="9f4c">连续九个季度盈利，连续五个季度实现创纪录的营收，连续两个季度盈利突破 10 亿美元，此次财报堪称特斯拉历史上最好的财报，并且对外释放了继续保持高速增长的可能性。</p><p data-vmark="8899">新能源汽车的先驱，再次为“特斯拉门徒”们验证了良性的财务模型，这对中国的新造车企业来说是一件好事，也不是一件好事。</p><h2 data-vmark="d84c">Model 3 和 Model Y 成为增长动力</h2><p data-vmark="8005">2019 财年第三季度，是特斯拉录得第一次盈利，与 Model 3 是密不可分的。从 2017 年 10 月第一辆 Model 3 开始交付到 2019 年 2 月在中国市场首次交付 Model 3，再到 2020 年 1 月在中国交付第一批中国制造 Model 3，这款车已经成为特斯拉的主销车型。</p><p data-vmark="511f">如今，这款车已经成为电动车历史上最畅销的车型。LeaseLoco 的数据显示，<span class="accentTextColor">特斯拉每小时可以售出约 24.5 辆 Model 3</span>。每年约可以售出 21.5 万辆 Model 3。这个数字在 2021 年还在保持增长，因为特斯拉在 2021 年有望销售近 90 万辆汽车，其中大约一半是 Model 3。</p><p data-vmark="739b">第三季度财报显示，特斯拉的总汽车产量为 237823 辆，与去年同期的 145036 辆相比增长 64%。其中，Model S 和 Model X 的产量同比下降了 47%；而 Model 3 和 Model Y 车型的产量为 228882 辆，与去年同期的 128044 辆相比增长 79%。</p><p data-vmark="04e1">特斯拉汽车第三季度的总汽车交付量为 241391 辆，与去年同期的 139593 辆相比增长 73%。其中，Model S 和 Model X 的交付量同比下降了 39%；而 Model 3 和 Model Y 车型的交付量为 232102 辆，与去年同期的 124318 辆相比增长 87%。</p><p data-vmark="25a5">在产销两端，特斯拉 Model 3 和 Model Y 的组合保持全面的领涨。</p><p data-vmark="da75">Model Y 是继 Model 3 之后特斯拉的第二款畅销车型，这款跨界车型的销售时间只有 18 个月，但是它的月销量已经超过了 Model 3 成为当前特斯拉最受欢迎的车辆，目前该款车型的每年平均销量达到了 10 万辆，相当于每小时可以售出 11.4 辆。</p><p data-vmark="5b60">在 Model 3 和 Model Y 两款畅销车型的刺激下，特斯拉主要收入来源 —— 汽车销售收入首次跨过百亿美元大关，达到了 116.72 亿美元，高于去年同期的 73.46 亿美元和上一季度的 98.74 亿美元。</p><p data-vmark="d732">特斯拉已经决定要继续扩大畅销车型的产量。特斯拉表示，在未来几年时间里，预计车辆交付量将可每年平均增长 50%。为此，正继续扩大上海超级工厂的产能，并在美国得克萨斯州和德国柏林超级工厂建设新的产能。上海工厂主要生产 Model 3 和 Model Y，今年底以前，特斯拉将在得州奥斯汀和德国柏林的工厂生产第一批 Model Y 车型。</p><p data-vmark="4626">这也意味着，特斯拉在未来会继续把增长的希望放在 Model 3 和 Model Y 两款车型上。</p><p data-vmark="86de">有汽车业内人士分析，特斯拉的例子已经很好地说明，一家新造车企业必须有一款畅销车型作为支撑，才能保证正向的运转，不管是在品牌上还是财务模型上，都可以产生正向的影响。特斯拉的经历，对于中国的新造车企业也是一个很好的启示。</p><h2 data-vmark="9df2">中国新造车能学到什么</h2><p data-vmark="6fc3">同样是依靠造车卖车起家，但特斯拉在汽车销售之外的业务已经颇具规模。</p><p data-vmark="d72d">财报显示，在汽车销售业务之外，第三季度特斯拉来自于汽车租赁业务的营收为 3.85 亿美元，来自于发电和储能业务的营收为 8.06 亿美元，来自于服务及其他业务的营收为 8.94 亿美元。</p><p data-vmark="b7bb">汽车销售业务之外的三个业务板块的收入在季度营收中的占比为 15.16%。而当前，中国新造车多依赖于汽车销售收入，比如蔚来在第二季度中汽车销售收入的占比为 93.65%，小鹏汽车为 95.3%，理想汽车则高达 97.2%。</p><p data-vmark="ae5c">收入不够多元化，是目前中国新造车的问题之一，其二则是老生常谈的亏损，目前中国新造车依然没有看到盈利的可能。</p><p data-vmark="7be2">特斯拉此前也是处于连年的亏损，但自从 2019 年第三季度实现盈利后，特斯拉在过去的九个季度都实现了盈利。2021 年第三季度，特斯拉实现了创纪录的盈利。</p><p data-vmark="db7c">财报显示，2021 年第三季度特斯拉 GAAP 营业利润为 20 亿美元，实现了 14.6% 的营业利润率。非 GAAP 净利润 (未计股份支付费用) 为 21 亿美元，达到历史新高。</p><p data-vmark="e4d4">对于中国新造车来说，这说明了盈利并非不可能。但如今依然持续亏损的中国新造车势力，面对特斯拉连续九个季度的盈利，也逐渐失去了继续亏损的理由。</p><p data-vmark="8b58">特斯拉提到，营业利润同比大幅增长，主要得益于汽车产销规模的提升和成本的降低。而产销规模的提升和成本的降低，则是相辅相成的事，规模提升有助于降低成本，降低成本反哺规模提升。</p><p data-vmark="9871">自中国制造 Model 3 开启交付以来，经过连续降价，价格下降了约 10 万元，最新售价为 23.59 万元；Model Y 自今年开始在国内交付不到一年时间里，特斯拉推出了 27 万元的版本。</p><p data-vmark="f1e8">按照年化销量，特斯拉今年的销量有望突破 100 万辆，约为 2020 年的两倍，这意味着特斯拉又跨过了一个门槛，也和很多追随者拉开了更大的距离。</p>
