<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 20 日消息 微软在上个月正式发布了 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 操作系统，为游戏玩家带来了重磅技术 <span class="accentTextColor">DirectStorage API</span>。这个新的 API 改变了游戏从 NVMe SSD 驱动器读取数据的方式，从而实现<span class="accentTextColor">更快的读取速度</span>和更短的加载时间。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/bfd86537-a898-437c-9ea3-b1fb980dcec9.jpg@s_2,w_820,h_461" alt="Windows 11: The Best Windows Ever for Gaming Hero" title="微软：Win11 的 DirectStorage API 将下放到 Win10，大大加快游戏加载速度" referrerpolicy="no-referrer"></p><p>近日，微软 DirectX 博客官方人员宣布，适用于 Win11 的 DirectStorage SDK 已经推出，同时官方确认，DirectStorage API 将适用于 <span class="accentTextColor"><a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 1909 版本及以上</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/48101741-e14a-4fed-98b3-4da5344cf46f.png" w="1210" h="472" title="微软：Win11 的 DirectStorage API 将下放到 Win10，大大加快游戏加载速度" width="1210" height="320" referrerpolicy="no-referrer"></p><p>不过按照官方解释，由于 Win11 在设计之初就考虑到了 DirectStorage API ，因此将采用全新升级的存储堆栈；而 Win10 的 DirectStorage API 会在原来的基础上<span class="accentTextColor">优化传统存储堆栈</span>，但并未指出两者的性能差距有多少。</p><p>IT之家了解到，该技术只在 <span class="accentTextColor">NVMe SSD 硬盘</span>上可用，其余硬盘将以原有速度加载游戏，所以最好把游戏放在 NVMe SSD 中。</p>
