<p data-vmark="8b3d"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 20 日消息，今日凌晨，苹果面向开发者宣布，<strong>即将从 XML Feed 过渡到 App Store Connect API</strong>。</p><p data-vmark="e58f">苹果表示，App Store Connect REST API <strong>让开发者可以跨各种开发者工具自定义任务并实现任务自动化</strong>，使工作流程更灵活、更高效。</p><p style="text-align: center;" data-vmark="10a9"><img src="https://img.ithome.com/newsuploadfiles/2021/12/292c2d22-f87a-4736-9344-e30942e542d8.png" w="720" h="477" alt="苹果 App Store" title="苹果：即将从 XML Feed 过渡到 App Store Connect API" width="720" height="477" referrerpolicy="no-referrer"></p><p data-vmark="ef75">IT之家了解到，<strong>从 2022 年 11 月开始</strong>，开发者将需要使用这一 API 来代替 XML Feed，从而实现对 App 内购买项目、订阅、元数据和 App 定价的自动化管理。<strong>XML Feed 将继续为现有的 Game Center 管理功能提供支持</strong>。</p><p data-vmark="db38">苹果 API 概览显示，借助 App Store Connect API 可实现工作流程自动化。这个基于标准规范的 REST API 让开发者可以跨各种开发者工具，如 App Store Connect、Xcode 以及“Certificates, Identifiers & Profiles”（证书、标识符和描述文件）。</p>
