<p data-vmark="638c">3 月 26 日消息，据彭博社援引知情人士的消息，索尼集团正准备最早于下周为 PlayStation 推出新的视频游戏订阅服务。</p><p data-vmark="7833">这项服务代号为 Spartacus，索尼已经开发了很长时间，以此与微软 Xbox Game Pass 展开竞争。Xbox Game Pass 已经积累了超过 2500 万用户。</p><p data-vmark="8c1c">知情人士说，Spartacus 首次亮相时，索尼将推出一系列近年来引人注目的热门游戏。索尼的新服务将结合其目前的两项服务，即 PlayStation Now 和 PlayStation Plus。</p><p data-vmark="2d1c">消费者将有多个层级的套餐进行选择，包括近年来开发的新款游戏以及老版经典游戏。索尼发言人没有立即回复置评请求。</p><p data-vmark="fd5c">目前还不清楚 Spartacus 的收费标准。目前，PlayStation Now 和 PlayStation Plus 的年费都为 60 美元，加起来是 120 美元。这个价格与标准 Game Pass 订阅版差不多，但比 Game Pass Ultimate 版略低，所以索尼在定价方面应该有一定的回旋余地。</p><p data-vmark="aa4e">还有账号迁移的问题，许多用户购买了几个月甚至几年的 PlayStation Plus 订阅服务。如果这项服务被取代，索尼将如何帮助这些付费用户切换到 Spartacus 上？Spartacus 可能会有类似服务，但价格可能会有所变化。</p><p data-vmark="337c">近年来，PlayStation 游戏机的销量超过了 Xbox，但索尼在流媒体游戏市场上的努力落后于微软。Xbox Game Pass 提供数百种游戏，月费仅为 10 美元。</p><p data-vmark="084f">Xbox Game Pass 最大的卖点之一是，当微软工作室旗下游戏在全球发售当天，便会加入到 Xbox Game Pass 服务中，订阅用户可立即畅玩，这包括今年备受期待的微软科幻游戏《星空》（Starfield）。</p><p data-vmark="f99e">然而，索尼在这方面可能会处于劣势，其新服务预计不会在新游戏推出当天就允许订阅用户访问。一位知情人士表示，索尼 Spartacus 平台上不太可能马上推出即将上市的《战神 3：诸神黄昏》（God of War Ragnarök）。</p><p style="text-align: center;" data-vmark="f3c3"><img src="https://img.ithome.com/newsuploadfiles/2022/3/22d9d7d2-9e16-4d20-b808-08135fc1db5a.jpg@s_2,w_820,h_461" w="1440" h="810" title="消息称索尼最早下周推出 PS 游戏订阅服务 Spartacus，对标微软 XGP" srcset="https://img.ithome.com/newsuploadfiles/2022/3/22d9d7d2-9e16-4d20-b808-08135fc1db5a.jpg 2x" width="1440" height="461" referrerpolicy="no-referrer"></p>
