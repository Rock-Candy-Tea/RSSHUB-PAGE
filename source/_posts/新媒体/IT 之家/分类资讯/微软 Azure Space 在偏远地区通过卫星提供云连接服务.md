<p data-vmark="0746"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 19 日消息，微软为 Azure Space 平台提供了更新，该平台在偏远地区提供云托管服务。该计划现在提供 Azure Orbital Cloud Access 的预览，并确认 Azure Orbital Ground Station 的普遍可用性。</p><p data-vmark="f822">IT之家获悉，通过 Azure Orbital Cloud Access，微软确保通过卫星互联网服务提供商访问微软云。<span class="accentTextColor">该平台本质上将连接从 Azure 云中继到地球上的数据终端</span>。此外，微软还确认了 Azure Orbital Ground Station 的全面可用性，这是一项完全托管的地面站服务。</p><p data-vmark="58a9">微软在世界卫星商业周期间正式宣布了这些新产品。</p><blockquote><p data-vmark="38ad">“首先，我们致力于通过解锁与微软云的连接和数据，使空间的可能性民主化。其次，我们还可以通过使用 Azure 中灵活、可扩展的计算能力来帮助我们的客户和合作伙伴在航天工业中实现数字化转型。”</p></blockquote><p data-vmark="34ba">微软的 Azure Orbital Cloud Access 服务现在是私密预览版。这意味着面向消费者的公司和最终用户尚无法使用它。微软正在向美国政府机构提供该平台。该公司希望在连接选项稀少或故障转移功能至关重要的地区能够访问云服务。</p><p data-vmark="5148">Azure Orbital Cloud Access 本质上搭载了 SpaceX 卫星连接，并与 Azure 边缘设备配合使用。该公司可以“优先考虑和提供跨光纤、蜂窝和卫星网络的流量，同时还可以与瞻博网络的 SD-WAN 技术进行本地集成”。</p><p data-vmark="65c4">Azure Orbital 地面站本质上为卫星运营商提供了地面站即服务选项。尽管该服务现已普遍可用，但 Pixxel、Muon Space 和 Loft Orbital 等卫星运营商应该会在不久的将来从中受益。</p>
