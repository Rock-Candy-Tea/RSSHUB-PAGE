<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 2 日消息 微软近日在官方博客发布了一篇《<a href="https://devblogs.microsoft.com/visualstudio/lets-make-visual-studio-even-more-accessible-together/" target="_blank">让我们一起使 Visual Studio 更易用</a>》的文章，宣布将与<span class="accentTextColor">残障人士</span>积极沟通，为 Visual Studio 带来更多<span class="accentTextColor">辅助功能</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/539a01c9-dc44-4ef9-b21c-bf6565eaf859.png" w="1049" h="499" title="微软 Visual Studio 将获得多项辅助功能，对残障人士更友好" width="1049" height="390" referrerpolicy="no-referrer"></p><p>官方表示，不会优先考虑特定的某一类残障，希望所有使用 Visual Studio 的残障人士都能够与他们沟通，说出自己对产品的改进需求。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/13461881-468c-4cb5-ab2b-972075e3ec58.png" w="560" h="350" title="微软 Visual Studio 将获得多项辅助功能，对残障人士更友好" width="560" height="350" referrerpolicy="no-referrer"></p><p>IT之家了解到，微软已经列出了多项<span class="accentTextColor">新的辅助功能</span>：</p><h2>Whitespace Rendering（空白字符渲染）</h2><p>该功能将允许视障人群自定义由空格填充的字符空格数，从而在使用特大字体编程时，可以将缩进改为单个字符宽度，获得更好的体验。</p><h2>Option Collecting（精简选项）</h2><p>Visual Studio 有数百个选项，允许用户配置和自定义体验以更好地满足他们的需求。然而，过多的选项杂烩在一起，有的是能更改的，有的是不能更改的，这对残障人士来说并不友好。</p><p>所以，微软打算精简 Visual Studio 的选项，只显示用户可用的几个。该功能还将扩展到隐私或安全等其他领域。</p><h2>Focus Mode（专注模式）</h2><p>微软在 Win10 中推出了“专注模式”，允许用户过滤掉一些通知，防止受到打扰。微软打算在 Visual Studio 里也采用这个功能，用户可以自由选择哪些通知是可以被显示的，以防止在写代码时受到干扰。</p><h2>Audio Cues（音频提示）</h2><p>微软将把音频提示连接到 Visual Studio 中的事件，以语音的形式播报事件，以帮助视障人群更好地写代码。</p>
