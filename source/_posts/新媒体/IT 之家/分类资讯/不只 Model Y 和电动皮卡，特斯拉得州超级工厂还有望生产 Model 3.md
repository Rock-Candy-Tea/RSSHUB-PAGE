<p data-vmark="a472">10 月 20 日消息，据国外媒体报道，自特斯拉确定在得克萨斯州奥斯汀建设超级工厂以来，特斯拉在财报中就表示，这一工厂在建成之后<span class="accentTextColor">将用于生产跨界运动型多用途电动汽车 Model Y 和电动皮卡 Cybertruck</span>。</p><p data-vmark="10e6"><img src="https://img.ithome.com/newsuploadfiles/2021/10/0979db6a-0f11-45d4-9cc7-e15fd66d7132.jpg" w="500" h="279" title="不只 Model Y 和电动皮卡，特斯拉得州超级工厂还有望生产 Model 3" width="500" height="279" referrerpolicy="no-referrer"></p><p data-vmark="40ae">▲ 正在建设的特斯拉得克萨斯超级工厂</p><p data-vmark="0fbd">但从外媒最新的报道来看，这一正在建设、准备开始生产 Model Y 的特斯拉超级工厂，<span class="accentTextColor">还可能会用于生产 Model 3</span>。</p><p data-vmark="de2f">外媒报道特斯拉得克萨斯超级工厂有望用于生产 Model 3，源自特斯拉官网新推出的制造页面，披露了目前旗下 6 座主要工厂建成之后将生产的车型。</p><p data-vmark="1a2f">对于正在建设的得克萨斯超级工厂，外媒发现特斯拉最开始披露的是预计 2021 年建成之后将用于生产 Cybertruck、Model 3 和 Model Y。但很快特斯拉就对页面进行了更新，显示的得克萨斯超级工厂只用于生产 Cybertruck 和 Model Y，没有了 Model 3 的身影。</p><p data-vmark="ac83">目前还不清楚得克萨斯超级工厂计划生产车型中出现 Model 3，是特斯拉方面网站更新人员的失误造成的，还是特斯拉有意为之。</p><p data-vmark="aebf">外媒在报道中表示，特斯拉官网制造页面的这一变化，预示着得克萨斯超级工厂，可能会用于生产 Model 3。</p><p data-vmark="b4e9">此外，外媒在报道中还提到，在 2020 年二季度的财报分析师电话会议上，特斯拉 CEO 马斯克曾提到，他们将在北美东部建设一座大型工厂，这一工厂将用于生产电动皮卡 Cybertruck、电动卡车 Semi，也将为北美东部生产生产 Model 3 和 Model Y。按他当时的表态，他们在北美的下一座工厂，就将用于生产 Model 3，只不过当时特斯拉还未确定在北美的下一座工厂会建在奥斯汀。</p>
