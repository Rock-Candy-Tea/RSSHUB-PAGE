<p data-vmark="7832"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 3 日消息，在北京时间 11 月 2 日 0 点的微软 Ignite 2021 开发者大会上，微软宣布推出 Azure OpenAI 服务，<span class="accentTextColor">该服务允许通过 Azure 平台访问 OpenAI 的 API，且允许开发者访问 GPT-3 模型以及微软 Azure 内置的其他企业级功能</span>。</p><p data-vmark="07c7"><img src="https://img.ithome.com/newsuploadfiles/2021/11/c0f41408-22d1-4f95-8d75-fa01e5175c46.png" w="1440" h="687" title="微软推出 Azure OpenAI 服务，可为开发者提供 GPT-3 语言模型" width="1440" height="391" referrerpolicy="no-referrer"></p><p data-vmark="56d2">IT之家通过微软官方博客了解到，微软还为使用 Azure OpenAI 的开发者提供新的开发工具，以确保模型输出的结果符合他们工作预期。</p><p data-vmark="4027"><img src="https://img.ithome.com/newsuploadfiles/2021/11/af7b72df-3af6-43bc-8c1c-60bf2685bee3.png" w="1440" h="748" title="微软推出 Azure OpenAI 服务，可为开发者提供 GPT-3 语言模型" width="1440" height="426" referrerpolicy="no-referrer"></p><p data-vmark="bc26">微软 Azure AI 副总裁 Eric Boyd 表示：“我们在现阶段需要明确 GPT-3 的水平和潜力，这会非常有趣。”OpenAI 首席执行官 Sam Altman 也表示：“随着越来越多的开发者开始访问并使用这些 AI 模型，这些 AI 模型的能力也会越来越强。期待有一天，开发者使用简单的编程语言就可以告诉计算机开发目标，即使开发目标可能并不明确”。</p><p data-vmark="b0b6"><img src="https://img.ithome.com/newsuploadfiles/2021/11/f0d065d4-3529-4e7e-bbf6-e5a607778eb3.png" w="1440" h="807" title="微软推出 Azure OpenAI 服务，可为开发者提供 GPT-3 语言模型" width="1440" height="460" referrerpolicy="no-referrer"></p><p data-vmark="3d19">Sam Altman 还表示：“GPT-3 已经证明了作为第一款功能强大且可以通用的自然语言模型，开发者可以用它去做任何事。一段时间以来，我们一直想找出一种方法来尽可能广泛地拓宽它的应用面，这也是我们与微软合作真正兴奋的原因。”</p><p data-vmark="f05e">“我们希望与我们的开发者一同学习，希望在 AI 领域了解到我们在哪些地方还需要打磨，”微软在官方博客上表示：“这对 AI 领域来说是一个非常关键的点，随着 GPT-3 推动了 AI 的发展，我们需要微软一直处于最前沿，才能负责任地去使用和发展 AI 技术。”</p>
