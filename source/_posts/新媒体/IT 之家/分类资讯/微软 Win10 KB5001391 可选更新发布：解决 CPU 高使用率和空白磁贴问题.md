<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 3 日消息 微软近日发布了 Win10 KB5001391 可选更新，对之前累积更新的一些问题进行了解决。</p><p>首先解决的就是 <span class="accentTextColor">CPU 高使用率问题</span>，据外媒 Windows Latest 报道，该问题似乎是多个进程竞争 CPU 而造成死锁，从而引发 CPU 使用率飙升。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/c2b0e1ac-9721-45a7-9b0a-9c67b7032422.jpg" w="696" h="365" title="微软 Win10 KB5001391 可选更新发布：解决 CPU 高使用率和空白磁贴问题" width="696" height="365" referrerpolicy="no-referrer"></p><p>其次，KB5001391 可选更新<span class="accentTextColor">修复了开始菜单偶现的空白磁贴问题</span>，该问题造成一些应用的磁贴图标消失，且名称变为 ms-resource:AppName 或 ms-resource:appDisplayName。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/36605046-ae67-488f-b679-4ba1c48a6665.jpg@s_2,w_820,h_676" w="991" h="817" title="微软 Win10 KB5001391 可选更新发布：解决 CPU 高使用率和空白磁贴问题" srcset="https://img.ithome.com/newsuploadfiles/2021/5/36605046-ae67-488f-b679-4ba1c48a6665.jpg 2x" width="991" height="676" referrerpolicy="no-referrer"></p><p>IT之家了解到，该功能已向 <span class="accentTextColor">Win10 20H2 和 2004 </span>大版本用户推出，点击“下载并安装”即可更新。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/5f701e9c-fb63-496c-b8c9-98db03b0de32.png" title="微软 Win10 KB5001391 可选更新发布：解决 CPU 高使用率和空白磁贴问题" referrerpolicy="no-referrer"></p>
