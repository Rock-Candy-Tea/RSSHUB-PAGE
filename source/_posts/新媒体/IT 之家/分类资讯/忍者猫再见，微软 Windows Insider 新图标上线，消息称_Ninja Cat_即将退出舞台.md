<p data-vmark="4153"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 15 日消息，微软似乎将用一个新的图标来代替 Windows Insider 标志性的忍者猫 Ninja Cat。</p><p data-vmark="3b6e">据 The Register 报道，微软上周发布了<span class="accentTextColor">新的 Windows Insider 图标</span>，采用了类似三个人在一起的标志。接近微软的消息人士告诉 The Register，<span class="accentTextColor">这意味着忍者猫的终结</span>。</p><p data-vmark="105c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/2e215ff8-833e-4985-8bc9-166a8b43c015.jpg" w="700" h="394" title="忍者猫再见，微软 Windows Insider 新图标上线，消息称“Ninja Cat”即将退出舞台" width="700" height="394" referrerpolicy="no-referrer"></p><p data-vmark="6eac">IT之家了解到，忍者猫（Ninja Cat）是微软在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 预览版计划中推出的非官方吉祥物，骑着独角兽，手举微软旗帜。</p><p data-vmark="dc6c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2015/3/20150312_201939_482.jpg" title="忍者猫再见，微软 Windows Insider 新图标上线，消息称“Ninja Cat”即将退出舞台" referrerpolicy="no-referrer"></p><p data-vmark="0ea9" style="text-align: justify;">之后，忍者猫进行了多次进化，成为了主题壁纸、Edge 浏览器彩蛋等等。</p><p data-vmark="987e" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/389fa260-af83-47a8-ad51-63c0d09767f9.png" w="1408" h="879" title="忍者猫再见，微软 Windows Insider 新图标上线，消息称“Ninja Cat”即将退出舞台" width="1408" height="512" referrerpolicy="no-referrer"></p><p data-vmark="5671" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2020/3/20200301205453_2063.jpg" title="忍者猫再见，微软 Windows Insider 新图标上线，消息称“Ninja Cat”即将退出舞台" referrerpolicy="no-referrer"></p><p data-vmark="a3c7" style="text-align: justify;">直到现在，<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的预览体验计划设置项仍然采用了忍者猫的 emoji 符号，不过很快就会被替换。</p><p data-vmark="d6f5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/9c6be9a7-e241-47cc-9531-3288091a1d71.png" justify;">微软在博客中表示，非常高兴地推出 Windows 预览体验计划的新图标，它代表了热爱并帮助塑造 Windows 未来的人们社区。开发频道的预览体验成员很快就会在“设置”>“Windows 更新”>“Windows 预览体验计划”下看到此图标。该图标也将出现在电子邮件和其他区域中</p><p data-vmark="8602" style="text-align:
