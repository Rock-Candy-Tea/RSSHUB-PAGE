referrerpolicy="no-referrer"></p><p data-vmark="d291">没错，项目如其名，「HowToCook」它真的没在标题党，就是一大份正经家常菜谱。</p><p style="text-align: center;" data-vmark="bd66"><img src="https://img.ithome.com/newsuploadfiles/2022/2/38914aff-906d-48f3-ae6f-d79c22b38ad1.gif" w="720" h="403" title="「程序员做饭指南」霸榜 GitHub，还用数学公式解决「吃什么」世纪难题，微软程序员出品" width="720" height="403" referrerpolicy="no-referrer"></p><p data-vmark="d6aa">当然，细节之处，可能还是跟普通菜谱略有不同……</p><p style="text-align: center;" data-vmark="c395"><img src="https://img.ithome.com/newsuploadfiles/2022/2/bd5a4458-81b0-4145-afcb-ffc68fc1753f.png" w="1028" h="982" title="「程序员做饭指南」霸榜 GitHub，还用数学公式解决「吃什么」世纪难题，微软程序员出品" width="1028" height="783" referrerpolicy="no-referrer"></p><p data-vmark="521d">这计算公式，这 1cm*1cm，真的很精准了 2333。</p><p data-vmark="5db7">仓库作者就开门见山地说了：</p><p data-vmark="54e7">网上的菜谱对习惯了形式语言的程序员太不友好，所以他计划结合自己的经验，用更清晰精准的描述整理常见菜的做法，方便程序员们在家做饭。并且，与许多开源项目一样，作者也非常欢迎更多小伙伴加入近来，共同维护、驱动这样一个别样的仓库。</p><p data-vmark="03aa">还真别说，目前，该项目已经吸引到 60 + 贡献者，而 Pull request 里也已经堆满了新菜谱：</p><p style="text-align: center;" data-vmark="cdca"><img src="https://img.ithome.com/newsuploadfiles/2022/2/66be4ce5-80f8-4f63-bbc8-eb2f8f4a1576.png" w="1080" h="578" title="「程序员做饭指南」霸榜 GitHub，还用数学公式解决「吃什么」世纪难题，微软程序员出品" width="1080" height="439" referrerpolicy="no-referrer"></p><h3 data-vmark="9e37">从厨房准备到“中午吃什么”，一站搞定</h3><p data-vmark="5f11">能受到这么多程序员的热捧，可见同是教做饭，还是这份指南最得程序员心。</p><p data-vmark="703e">毕竟在最开始的“厨房准备”这个序章里，除了常规的锅碗瓢盆柴米油盐，「程序员做饭指南」还提供了一个非常具有仪式感的工具方案：天平、游标卡尺、量筒、停表、烧杯、测温枪、移液器。</p><p style="text-align: center;" data-vmark="4596"><img src="https://img.ithome.com/newsuploadfiles/2022/2/ac84dce1-bf38-47e2-aa8f-6f88668a9044.png" w="690" h="690" title="「程序员做饭指南」霸榜 GitHub，还用数学公式解决「吃什么」世纪难题，微软程序员出品" width="690" height="690" referrerpolicy="no-referrer"></p><p data-vmark="c52a">而如果你觉得以上都还不够“程序员”，那么不妨看看仓库里附带的“中午吃什么”这个世纪难题解决方案。</p><p data-vmark="6674">正如仓库作者所说：</p><blockquote><p data-vmark="ed9f">如何决策吃什么也是我做菜之前一大难题，所以只能用数学描述一下了。</p></blockquote><p data-vmark="b41d">计算方法大致是这样的：</p><blockquote><p data-vmark="bac7">当有人数 N 时，设素菜数为 a，荤菜数为 b。N，a，b 均为整数。</p><p data-vmark="5db0">此时有下列不等式组：</p><p data-vmark="7aab">a + b = N + 1</p><p data-vmark="ce54">a ≤ b ≤ a + 1</p><p data-vmark="1263">解得：</p><p data-vmark="b7e2">a=floor(N/2)</p><p data-vmark="c112">b=cell(N/2)</p></blockquote><h3 data-vmark="f90d">Pull Request 指南</h3><p data-vmark="bb68">这个项目的发起者 Auduin Xue，是一名微软程序员，base 苏州。前面已经提到，Auduin 希望更多人一起来建设「程序员做饭指南」这个有趣的仓库。那么最后，如果你感兴趣，就一起来看看怎么把你手上的独门菜谱加入菜单吧~</p><p data-vmark="cef8">需要注意的是，写新菜谱时，需要复制并修改已有的菜谱模板。模板包括这几个部分：必备原料和工具、计算、操作、附加内容。其中计算主要介绍计算公式，以便求得原料的量、重要时间参数、混合比例等。而操作部分，不允许使用“适量”、“少量”等不精准描述的词汇。</p><p style="text-align: center;" data-vmark="6fc8"><img src="https://img.ithome.com/newsuploadfiles/2022/2/3969e401-1c05-482e-902b-3af648251b78.png" w="1080" h="790" title="「程序员做饭指南」霸榜 GitHub，还用数学公式解决「吃什么」世纪难题，微软程序员出品" width="1080" height="600" referrerpolicy="no-referrer"></p><p data-vmark="9493">另外，在提交 Pull request 之前还需要同步更新一下 README.md 里的菜谱目录。</p><p data-vmark="0891">项目传送门：</p><p data-vmark="8f6f"><span class="link-text-start-with-http">https://github.com/Anduin2017/HowToCook</span></p>
