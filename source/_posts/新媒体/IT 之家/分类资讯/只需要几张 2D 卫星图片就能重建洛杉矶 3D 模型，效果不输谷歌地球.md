<p data-vmark="2710">看到下面这张动图，你会想到什么？是谷歌地球，还是苹果自带的 3D 地图？</p><p data-vmark="caa8"><img src="https://img.ithome.com/newsuploadfiles/2021/12/5fa7ede4-999d-496d-aee1-8e4157d91164.gif" w="480" h="276" alt title="只需要几张 2D 卫星图片就能重建洛杉矶 3D 模型，效果不输谷歌地球" width="480" height="276" referrerpolicy="no-referrer"></p><p data-vmark="9d7b">其实都不是，它是用卫星和航拍图片直接渲染生成的洛杉矶。很难想象，如此精细的城市 3D 模型，竟然是用几张不同角度和高度的 2D 图片重建的。</p><p data-vmark="9915">这项研究来自香港中文大学多媒体实验室团队，叫做 CityNeRF。</p><p data-vmark="05ff">说到这里，有人应该想到了这两年大热的“神经辐射场”（NeRF），它可以用多张角度照片重建 3D 对象，性能出色。量子位之前对此进行了相关报道和解读。</p><p data-vmark="a6a7"><img src="https://img.ithome.com/newsuploadfiles/2021/12/07654c5b-ca35-45b6-b829-b030daad12cb.gif" w="635" h="386" alt title="只需要几张 2D 卫星图片就能重建洛杉矶 3D 模型，效果不输谷歌地球" width="635" height="386" referrerpolicy="no-referrer"></p><p data-vmark="839c">NeRF 虽然恢复室内场景效果惊艳，但是直接用到城市级的卫星地图上，却面临着巨大的挑战。</p><p data-vmark="145d">首先是拍摄相机有很大的运动自由度。随着相机的上升，场景中的地物外观越来越粗糙，几何细节越来越少，纹理分辨率越来越低。</p><p data-vmark="631a">同时，随着空间覆盖范围的扩大，来自外围区域的新对象会加入到视图中。</p><p data-vmark="0c42"><img src="https://img.ithome.com/newsuploadfiles/2021/12/d99c583d-960a-4cd5-8d06-ac3d0702a753.gif" w="640" h="360" alt title="只需要几张 2D 卫星图片就能重建洛杉矶 3D 模型，效果不输谷歌地球" width="640" height="360" referrerpolicy="no-referrer"></p><p data-vmark="e33f">相机在这个一系列场景中，产生了具有不同细节级别和空间覆盖范围的多尺度数据。</p><p data-vmark="7c57"><img src="https://img.ithome.com/newsuploadfiles/2021/12/7317c9a0-90cb-462d-9475-0078d8e397d1.png" w="800" h="595" alt title="只需要几张 2D 卫星图片就能重建洛杉矶 3D 模型，效果不输谷歌地球" width="800" height="595" referrerpolicy="no-referrer"></p><p data-vmark="70f2">如果使用原来的 NeRF 渲染，那么生成的远景往往不完整，周边场景区域存在伪影，近景总是具有模糊的纹理和形状。</p><p data-vmark="a5b8"><img src="https://img.ithome.com/newsuploadfiles/2021/12/ad7ad8b6-c26f-4380-8df5-3a87b56e4c92.jpg@s_2,w_820,h_540" w="1080" h="711" alt title="只需要几张 2D 卫星图片就能重建洛杉矶 3D 模型，效果不输谷歌地球" srcset="https://img.ithome.com/newsuploadfiles/2021/12/ad7ad8b6-c26f-4380-8df5-3a87b56e4c92.jpg 2x" width="1080" height="540" referrerpolicy="no-referrer"></p><p data-vmark="9a42">什么原理？</p><p data-vmark="7e5f">针对上述问题，作者提出了采用多阶段渐进式学习范式的 CityNeRF。</p><p data-vmark="ff41">作者根据相机距离将整个训练数据集划分为预定义数量的尺度。从最远的尺度开始，每个训练阶段逐渐将训练集扩大一个更近的尺度，并同步增长模型。</p><p data-vmark="5f24">通过这种方式，CityNeRF 可以稳健地学习跨场景所有尺度的表示层次结构。</p><p data-vmark="8246">CityNeRF 引入了两个特殊的设计：</p><p data-vmark="c859">1、具有残差块结构的生长模型：</p><p data-vmark="05f4">在每个训练阶段附加一个额外的块来扩展模型。每个块都有自己的输出 head，用于预测连续阶段之间的颜色和密度残差，促使块在近距离观察中关注新兴细节；</p><p data-vmark="143e">2、包容的多级数据监督：</p><p data-vmark="ab08">每个块的输出 head 由从最远尺度到其对应尺度的图像联合监督。</p><p data-vmark="e6b8">换句话说，最后一个块接受所有训练图像的监督，而最早的块只暴露于最粗尺度的图像。通过这样的设计，每个块模块都能够充分利用其能力，在更近的视图中对复杂的细节进行建模，并保证尺度之间一致的渲染质量。</p><p data-vmark="368d"><img src="https://img.ithome.com/newsuploadfiles/2021/12/3444b197-22d9-49f4-a714-0293b9ea9b3a.png" w="800" h="254" alt title="只需要几张 2D 卫星图片就能重建洛杉矶 3D 模型，效果不输谷歌地球" width="800" height="254" referrerpolicy="no-referrer"></p><p data-vmark="114c">总体来说，CityNeRF 是一种渐进式学习范式，可同步增长 NeRF 模型和训练集。从用浅基块拟合远景开始，随着训练的进行，添加新的块以适应越来越近的视图中出现的细节。</p><p data-vmark="5155">该策略有效地激活了位置编码中的高频通道，并随着训练的进行展开更复杂的细节。</p><p data-vmark="57ca">简而言之，使用基本神经网络多层感知器的权重，NeRF 将提前处理所有图像，知道其观点位置。NeRF 将使用相机的光线找到每个像素的颜色和密度。</p><p data-vmark="c85f">因此，它知道相机的方向，并可以同时使用所有数组来了解深度和相应的颜色。然后，使用损失函数优化了神经网络的收敛性，</p><p data-vmark="d3db">模型训练数据数据来自 Google Earth Studio 中的 12 个城市图像。结果显示在几种常见重建模型中达到了最佳的效果。</p><p data-vmark="2050"><img src="https://img.ithome.com/newsuploadfiles/2021/12/4c7ad0c8-a5a2-462e-81ad-d18635294206.png" w="662" h="375" alt title="只需要几张 2D 卫星图片就能重建洛杉矶 3D 模型，效果不输谷歌地球" width="662" height="375" referrerpolicy="no-referrer"></p><p data-vmark="30b2">最后，作者又将该模型用于重建无人机拍摄的空中图像，依然收到了更佳的效果。</p><p data-vmark="753a"><img src="https://img.ithome.com/newsuploadfiles/2021/12/169f5144-10e3-455c-8480-7e3a7f0aff4b.png" w="676" h="351" alt title="只需要几张 2D 卫星图片就能重建洛杉矶 3D 模型，效果不输谷歌地球" width="676" height="351"
