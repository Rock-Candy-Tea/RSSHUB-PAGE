<p data-vmark="81d9">北京时间 11 月 4 日消息，据报道，周三<span class="accentTextColor">微软宣布 GitHub CEO 奈特・弗里德曼（Nat Friedman）卸任</span>，他的职位<span class="accentTextColor">由 GitHub 产品主管托马斯・多梅克（Thomas Dohmke）接替</span>。</p><p data-vmark="faa2"><img src="https://img.ithome.com/newsuploadfiles/2021/11/2767178c-0511-4ea8-b488-6c2e40100aee.jpg" w="630" h="354" title="微软旗下 GitHub 更换 CEO，由产品主管接手" width="630" height="354" referrerpolicy="no-referrer"></p><p data-vmark="3b51">就在几周前 GitHub 的主要竞争对手 GitLab 在纳斯达克上市，上市后它的估值达到 165 亿美元。2018 年微软收购 GitHub 时它的估值为 75 亿美元。</p><p data-vmark="30bc">多梅克将于 11 月 15 日从弗里德曼手中接过工作。</p><p data-vmark="0a6b">GitHub 于 2008 年创建，2009 年多梅克成为 GitHub 注册用户。他曾是 App 测试软件创业公司 HockeyApp 的联合创始人、CEO，2014 年这家公司被微软收购。2018 年微软完成对 GitHub 的收购，于是多梅克加入 GitHub。</p>
