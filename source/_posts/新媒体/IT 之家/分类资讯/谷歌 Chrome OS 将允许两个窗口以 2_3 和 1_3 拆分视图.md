<p data-vmark="3679"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 14 日消息，据 XDA 报道，谷歌的 Chrome OS 操作系统在过去几年中好评如潮，操作系统开发团队正通过新功能使其更加强大。谷歌目前正在改善其桌面操作系统的狭缝屏幕体验。</p><p data-vmark="a9db">据 Chrome Story 发现，Chromium Gerrit 上的一项新提交表明，将两个窗口对齐为 2/3 和 1/3 以用于拆分视图的功能正在开发中。目前，Chrome OS 允许移动贴靠两个大小相同的窗口，但这种限制有时会令人感到沮丧。虽然用户仍将被限制为移动贴靠两个窗口，但通过允许用户使用更多的窗口比例配置，这将让操作变得更方便。</p><p data-vmark="72fc">比如，<span class="accentTextColor">一旦该功能上线，用户的社交媒体订阅源将可能占据屏幕空间的 1/3，而工作应用程序将占据 2/3 的区域。</span>新的移动贴靠体验目前隐藏在一个 flag 后面，其描述如下：</p><blockquote><p data-vmark="5ca8">“启用移动贴靠两个窗口到 2/3 和 1/3 以进行拆分视图的选项。”</p></blockquote><p data-vmark="cf98">移动贴靠在 Windows 11 上非常有用，当然比 Chrome OS 更好。前者允许贴靠多个窗口，而不仅仅是两个。准确地说，你可以在拆分视图屏幕中移动贴靠多达四个窗口。Chrome OS 操作系统在这方面有很多工作要做，以追赶 Windows 11 的卓越多任务处理功能。</p><p data-vmark="3c88">目前该功能还未面向 Chrome OS 任何频道版本推出，但首先应该会出现在 Canary 频道中进行测试。</p>
