<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 18 日消息 JetBrains 近日发布了第五次年度开发者生态系统调查报告《<a href="https://www.jetbrains.com/lp/devecosystem-2021/" target="_blank" title="The State of Developer Ecosystem 2021">The State of Developer Ecosystem 2021</a>》。IT之家了解到，本次调查基于来自 183 个国家或地区的 <span class="accentTextColor">31743 名开发者</span>，旨在展示开发者社区的最新趋势。</p><p>报告指出，过去一年里，<span class="accentTextColor">最流行的编程语言是 JavaScript</span>，有 69% 的开发者都在使用；第二名是 HTML/CSS，占比 60%（虽然不是编程语言）；第三名是 SQL 数据库语言，占比 54%；<span class="accentTextColor">Python 再一次领先 Java 来到第四</span>，占比 52%，后者占比 49%。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/8464c933-3646-4a2f-9f5f-d39bd23faa61.png" w="1440" h="1105" title="JetBrains 调查：JavaScript 最流行，Python 再次领先 Java" width="1440" height="629" referrerpolicy="no-referrer"></p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/3229dbca-8eda-47cd-8e2f-95ef96cea3fd.png" w="738" h="821" title="JetBrains 调查：JavaScript 最流行，Python 再次领先 Java" width="738" height="821" referrerpolicy="no-referrer"></p><p>然而，在<span class="accentTextColor">主要使用的编程语言中，Java 比 Python 要高</span>，占比 32%。说明更多 Java 开发者会使用 Python 语言开发，但主力还是使用 Java。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/f2a55491-f225-42a7-8ccf-bf8d77195abb.png" w="642" h="525" title="JetBrains 调查：JavaScript 最流行，Python 再次领先 Java" width="642" height="525" referrerpolicy="no-referrer"></p><p>此外，<span class="accentTextColor">Windows 仍是最主要的开发环境</span>，占比 61%，其次分别是 Linux（47%）、macOS（44%）和其他（1%）。</p><p>关于其他具体的调查数值，大家可以<a href="https://www.jetbrains.com/lp/devecosystem-2021/" target="_blank">点此查看详细报告</a>获得。</p>
