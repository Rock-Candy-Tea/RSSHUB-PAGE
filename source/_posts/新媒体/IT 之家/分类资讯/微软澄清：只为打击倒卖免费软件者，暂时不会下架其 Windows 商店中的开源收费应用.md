<p data-vmark="f064"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 9 日消息，微软上个月发布了 Microsoft Store Policies 的修订版，这是每个开发者在 Microsoft Store 中发布项目时必须遵循的一组规则。</p><p data-vmark="3129">在最新的 <span class="accentTextColor">7.16 版本</span>中带来了一些变化，其中一项会让不良开发者更难使用免费开源软件来欺骗人们付费。<span class="accentTextColor">新规则现在禁止销售“开源或免费”应用，同时商店将不允许过度收费或设置不合理的高价，也就是避免有人倒卖免费软件</span>。</p><p data-vmark="1e42">微软出发点是好的，但这一决策影响了一些开发者，从而引发了不小的争议。后续就是，多位在微软应用商店中上架开源应用程序的开发人员提出投诉，毕竟自由和开源的软件（FOSS）也不全都是可以免费下载和使用的。</p><p data-vmark="1483">当然，目前大部分 Linux 和 OSS 是完全免费的，但也有许多基于开源影应用的商业项目产生了附加价值，例如基于 Apple WebKit 的应用，所以业界此前针对开发人员面向开源应用程序收费一事没有任何限制。</p><p data-vmark="9c3e">微软应用程序、合作伙伴、商店总经理 Giorgio Sardo 解释道，微软的目标并不是阻止开源软件 (OSS) 的分发。该公司只是为了减少“被误导购买免费应用”。</p><p data-vmark="6277">不过，Sardo 提供的解释很是模糊。因此一些开发人员要求他澄清公司在 Microsoft Store 上销售应用程序（基于 FOSS）的立场。</p><p data-vmark="c919">他表示，微软已决定推迟该政策的实施，而且微软需要澄清其修订草案背后的意图。这意味着微软不会禁止 Microsoft Store 中的收费 OSS 应用。但目前该公司也未取消此前更新的政策，目前微软仍未给出一个正式回应。</p><p data-vmark="c2bb"><img src="https://img.ithome.com/newsuploadfiles/2022/7/a84f29e6-f574-4bf7-a1ae-9c5ab906d485.png" w="487" h="396" title="微软澄清：只为打击倒卖免费软件者，暂时不会下架其 Windows 商店中的开源收费应用" width="487" height="396" referrerpolicy="no-referrer"></p><p data-vmark="b3ac"><img src="https://img.ithome.com/newsuploadfiles/2022/7/67fe7744-24fd-4cb2-aad6-03b5c3c8a803.png" w="473" h="702" title="微软澄清：只为打击倒卖免费软件者，暂时不会下架其 Windows 商店中的开源收费应用" width="473" height="702" referrerpolicy="no-referrer"></p><p data-vmark="9ab4">IT之家了解到，Microsoft Store 还将禁止仿冒软件，这意味着任何第三方应用都不能模仿 Google Chrome、Microsoft Edge 或其他开发者的产品。这条规则与新的定价政策相结合，预计会打击一些想走歪门邪道赚钱的不良开发者。</p><p data-vmark="fd8c">Microsoft Store 上架规则：<a href="https://docs.microsoft.com/en-us/windows/uwp/publish/store-policies-change-history" target="_blank">点此查看</a></p>
