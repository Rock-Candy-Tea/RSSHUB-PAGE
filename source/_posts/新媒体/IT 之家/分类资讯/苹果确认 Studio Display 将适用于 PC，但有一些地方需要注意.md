<p data-vmark="e3df"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 10 日消息，苹果已向 MacRumors 证实，全新的 Studio Display 可支持连接到 PC 工作，但与在 Mac 上使用时相比会缺少部分显示器关键新功能，而且体验会有所欠缺。</p><p style="text-align: center;" data-vmark="b017"><img src="https://img.ithome.com/newsuploadfiles/2022/3/68a21ea8-3ebe-49c9-9ece-a6b4895dd919.jpg" w="400" h="225" alt="演播室展示" title="苹果确认 Studio Display 将适用于 PC，但有一些地方需要注意" width="400" height="225" referrerpolicy="no-referrer"></p><p data-vmark="9bfc">据称，需要 macOS 的功能（例如 Center Stage 和 True Tone）在连接到 PC 时将无法使用。正如苹果在其网站上指出的那样，你若想在 Studio Display 中使用网络摄像头，则需要使用运行最新 macOS 系统的 Mac 电脑才行。</p><p data-vmark="799f">其他方面，连接到 PC 时该机需要固件更新的功能将无法运行。此外，苹果还表示 Studio Display 的屏幕分辨率将因系统而异。</p><p data-vmark="6654">IT之家了解到，Studio Display 是 Apple 新推出的外接显示器，搭载了一块 <span class="accentTextColor">27 英寸 5K 视网膜显示屏</span>，以及出色的摄像头与音频系统。起价 11499 元，还可选配更高配置的版本。</p><p data-vmark="abd9">该显示器尺寸为 27 英寸，采用与 Pro Display XDR 类似的设计，但边框更大且没有后面板图案。Studio Display 还配备了 A13 仿生芯片，可为 Center Stage、Spatial Audio、True Tone 等提供支持。</p><p data-vmark="c04a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/2aca1166-f6ea-417b-9775-3bc3c58a41f2.png" w="667" h="466" title="苹果确认 Studio Display 将适用于 PC，但有一些地方需要注意" width="667" height="466" referrerpolicy="no-referrer"></p><p data-vmark="d17e">Studio Display 采用<span class="accentTextColor">窄边框全面屏设计</span>，在精致纤薄的全铝机身内整合了多种功能。自带支架可让用户对显示器进行调节，最大角度可至 30 度。</p><p data-vmark="fdbc" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/eca7657c-2010-4d71-8ad5-d5f265815cc7.png" w="468" h="347" title="苹果确认 Studio Display 将适用于 PC，但有一些地方需要注意" width="468" height="347" referrerpolicy="no-referrer"></p><p data-vmark="fb8a">Studio Display 出人意料地搭载了 <span class="accentTextColor">A13 仿生芯片</span>，配备 1200 万像素的<span class="accentTextColor">超广角摄像头</span>，通过摄像头和音频系统，支持人物居中，可在用户移动的时候自动将他们置于画面中央，加强视频通话的交互性。</p><p data-vmark="1a4c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/7c019d40-3013-4539-a8f7-6ea66478d943.png" w="638" h="383" title="苹果确认 Studio Display 将适用于 PC，但有一些地方需要注意" width="638" height="383" referrerpolicy="no-referrer"></p><p data-vmark="82d2">Studio Display 还配备录音棚级的<span class="accentTextColor">三麦克风阵列</span>，以及 Mac 上迄今最高配置的高保真六扬声器系统，四个振动抵消低音单元可以最大限度地减少失真，生成浑厚有力的低音；而两个高性能高音单元则能生成精准饱满的中音和清澈透亮的高音。</p><p data-vmark="5571" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/479da59a-ca07-4773-a6e9-f3f13576da43.png" w="677" h="338" title="苹果确认 Studio Display 将适用于 PC，但有一些地方需要注意" width="677" height="338" referrerpolicy="no-referrer"></p><p data-vmark="3210">这套六扬声器系统也支持空间音频，可以播放杜比全景声的音乐和视频。</p><p data-vmark="7fe0" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/77780d63-b5db-4e47-81ed-e175137b8e42.png" w="544" h="314" title="苹果确认 Studio Display 将适用于 PC，但有一些地方需要注意" width="544" height="314" referrerpolicy="no-referrer"></p><p data-vmark="a8a7">接口方面，Studio Display 提供 3 个传输速度最高均可达 10Gb / s 的 USB-C 端口，可让用户为这台显示器连接高速外接设备、存储设备或者将它接入各类网络。</p><p data-vmark="c4ac" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/3862b3a5-87be-486a-91d5-be566d37ff40.png" w="801" h="377" title="苹果确认 Studio Display 将适用于 PC，但有一些地方需要注意" width="801" height="377" referrerpolicy="no-referrer"></p><p data-vmark="5d30">它还提供 1 个雷电接口，让用户只用一根连接线就能将可能已连接若干外接设备的 Studio Display 与一台 Mac 连接起来。不仅如此，这根连接线还能以 96W 的功率向 Mac 笔记本电脑供电，从而使得 Studio Display 甚至能为一台 14 英寸的 MacBook Pro 快速充电。一台 MacBook Pro 最多可以同时连接三台 Studio Display。</p><p data-vmark="1539" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/e3183e99-425b-46ad-9945-8e9cde1897ee.png" w="663" h="420" title="苹果确认 Studio Display 将适用于 PC，但有一些地方需要注意" width="663" height="420" referrerpolicy="no-referrer"></p>
