<p data-vmark="7b14"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 20 日消息，今天 JetBrains 带来了对最新成立的 PHP 基金会的介绍：PHP 焕然新生 —— PHP 基金会成立！</p><p data-vmark="3ed7"><img src="https://img.ithome.com/newsuploadfiles/2021/12/98a80d5e-eacc-4404-b69f-0fb9376818a3.png" w="1440" h="356" alt="PHP基金会" title="JetBrains 详解最新成立的 PHP 基金会，公司每年捐赠 100000 美元" width="1440" height="203" referrerpolicy="no-referrer"></p><p data-vmark="a726">IT之家获悉，26 年来，PHP 语言由大量人员积极开发，包括 Rasmus Lerdorf、Zeev Suraski、Andi Gutmans、Nikita Popov 等。2021 年，PHP 将迎来新一轮进化。</p><h3 data-vmark="c2be">Nikita Popov 不再专注于 PHP</h3><p data-vmark="24a9">Nikita 早在 2011 年就开始开发 PHP，当时他还在上高中。他在 PHP 5.5 中做出了自己的第一个贡献。在他与 PHP 一起走过的 10 年里，Nikita 实现了无数功能、错误修正和语言开发流程改进。</p><p data-vmark="6866">Nikita 在 JetBrains 从事 PHP 工作近三年，在此期间取得了诸多成就。PHP 7.4、PHP 8.0 和 PHP 8.1 这三大主要版本都有他的大量贡献。</p><p data-vmark="dec0">除了 PHP，Nikita 长期以来也一直是 Rust 和 LLVM 的贡献者。就像 PHP 一样，随着爱好发展成了真正的工作，Nikita 决定将专业活动的重点放在 LLVM 上。</p><p data-vmark="eab4">Nikita 于 12 月 1 日离开 JetBrains，并大幅减少在 PHP 上投入的时间。尽管我们为他的离开感到难过，但我们仍要祝贺 Nikita，希望他在新的旅程中一切顺利！</p><p data-vmark="fd0c">感谢 Nikita，感谢你为 PHP 所做的一切！你的成就激励着数百万的 PHP 开发者。</p><h3 data-vmark="4b14">关键因素</h3><p data-vmark="dbf1">失去 PHP 的主要贡献者对社区来说无疑是一个打击，因为这意味着失去大量知识和专长。这将一种为 78% 的网络提供支持的语言置于脆弱的境地。更不用说维护者要承受的沉重负担，在 OSS 领域，这常常会导致人们筋疲力尽。</p><p data-vmark="9bcd">2021 年 5 月，就在 Joe Watkins 发布 Avoiding Busses 博文之后，我们开始了对 PHP 基金会这一想法的讨论。这并不是什么新鲜事，并且已经流传了很长时间。我们与 Joe 和 PHP 社区的其他几位成员进行了交谈，包括 Sebastian Bergmann 和 Stefan Priebsch（PHPUnit，thePHP.cc），他们曾在 2019 年尝试建立过一个类似的基金会。</p><p data-vmark="1d2e">此外，Zend by Perforce 在过去 20 多年一直关注着 PHP 并为 PHP 语言贡献关键举措，包括 PHP 5 对象模型、操作码缓存、PHP 7 引擎和 JIT 编译器。通过继续资助 PHP 项目的全职和兼职开发者，Zend 帮助确保了 PHP 语言的持续繁荣和现代化，延续了其在任务关键型应用程序开发中的作用。</p><p data-vmark="8ce1">我们当时并不着急，没有意识到问题的严重性。Nikita 的决定迫使我们加快了基金会的进度。</p><h3 data-vmark="c375">PHP 基金会</h3><p data-vmark="6a27">在与社区成员以及一些在 PHP 有既得利益的公司展开多次对话后，我们高兴地宣布以下公司将联手组建 PHP 基金会：</p><ul class=" list-paddingleft-2"><li><p data-vmark="045b">Automattic</p></li><li><p data-vmark="e08b">Laravel</p></li><li><p data-vmark="c5fd">Acquia</p></li><li><p data-vmark="d686">Zend</p></li><li><p data-vmark="ce23">Private Packagist</p></li><li><p data-vmark="dbcc">Symfony</p></li><li><p data-vmark="d102">Craft CMS</p></li><li><p data-vmark="2abc">Tideways</p></li><li><p data-vmark="2ebe">PrestaShop</p></li><li><p data-vmark="a7f8">JetBrains</p></li></ul><p data-vmark="ca8e">PHP 基金会将是一个非营利组织，其使命是确保 PHP 语言的存续和繁荣。</p><p data-vmark="7a4d">为了更快启动和发挥基金会的作用，我们选择了 Open Collective：<a href="https://opencollective.com/phpfoundation" target="_blank"><span class="no-auto-spacing">opencollective.com/ phpfoundation</span></a><span class="no-auto-spacing">。</span></p><p data-vmark="22ec">Open Collective 是一家处理法律和会计事务的基金会即服务提供商。Open Collective 受到许多开源项目的信任，包括 Vue.js、Open Web Docs、webpack、Yii Framework 等。</p><p data-vmark="2777">根据迄今为止所有参与公司的预计捐款，我们预计每年可筹集约 300,000 美元。JetBrains 决定每年捐赠 100,000 美元。</p><p data-vmark="d4eb">我们希望能够向 PHP 核心开发者支付市场薪酬。我们筹集的资金越多，就有越多的开发者能够全职开发 PHP。</p><p data-vmark="9788">请考虑成为赞助者，帮助 PHP 蓬勃发展！</p><h3 data-vmark="baac">基金会管理</h3><p data-vmark="c362">最初将设立一个临时管理机构，在基金会正式成立并启动后，将决定常设管理机构的组成。临时管理机构将推进基金会规则和章程的制定。</p><p data-vmark="79c7">临时管理机构将包括赞助者代表和一组 PHP 人员：Rasmus Lerdorf、Dmitry Stogov、Joe Watkins、Sara Golemon、Derick Rethans、Sebastian Bergmann 和 Nikita Popov。</p><h3 data-vmark="bb83">基金会活动</h3><p data-vmark="555c">基金会筹集的资金可能用于吸引 PHP 贡献者的服务，以进一步推动语言发展。基金会的主要任务是资助开发者从事 PHP 工作。</p><p data-vmark="35ee">基金会和任何从基金会获得资金的人员都必须遵守行为准则 (<span class="link-text-start-with-http">https://www.contributor-covenant.org/</span>)。</p><h3 data-vmark="4bc7">资助选择</h3><p data-vmark="c48b">任何 php-src 的贡献者都可以向基金会申请资助。</p><p data-vmark="ed7d">基金会的目的是为兼职和全职开发者提供资金，其他收入来源并不妨碍你与基金会签订合同。这些职位并非常设，可能会在经基金会管理机构同意后取消。</p><p data-vmark="1948">申请期从 11 月 22 日开始，持续 28 天。</p><p data-vmark="0fec">请使用此表格申请：</p><p data-vmark="94a1"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://forms.gle/mk52v6FphaYr6yMS8</span>。</span></p><p data-vmark="734f">最初，临时管理机构将协同采用专业知识决定谁应该获得资金。</p><h3 data-vmark="e396">未来计划</h3><p data-vmark="9d6c">选定开发者后，临时管理机构将在 PHP 社区成员的帮助下正式确定和记录工作流程。</p><p data-vmark="7277">当前的 RFC 流程不会改变，语言决定将始终由 PHP Internals 社区做出。</p><p data-vmark="e604">你可以在</p><p data-vmark="8a7c"><span class="no-auto-spacing">o<a href="https://opencollective.com/phpfoundation" target="_blank">pencollective.com/ phpfoundation</a></span> 和 @ThePHPF 下关注 PHP 基金会的动态。</p><p data-vmark="47aa">如需详细了解 PHP 基金会的工作、参与其中或询问其他贡献方式，请发送电子邮件至</p><p data-vmark="302b"><a href="https://www.ithome.com/cdn-cgi/l/email-protection" class="__cf_email__" data-cfemail="40302830262f352e242134292f2e2f2626292329212c00272d21292c6e232f2d">[email protected]</a>。</p><p data-vmark="f75b">你可以如何提供帮助</p><p data-vmark="c650">如果你想帮助基金会和 PHP 开发，请考虑以下选项：</p><p data-vmark="208f">1.如果你是 PHP 核心开发者，请提交申请。</p><p data-vmark="bc5b">2.在 OpenCollective</p><p data-vmark="ee05"><a href="https://opencollective.com/phpfoundation" target="_blank"><span class="no-auto-spacing">opencollective.com/ phpfoundation</span></a><span class="no-auto-spacing"> </span>上向基金会捐款。请注意，除了支持 PHP 基金会和 PHP 核心开发之外，捐款目前还不会带来其他收益。</p><p data-vmark="e6dc">3.如果你的雇主正在使用 PHP，请让管理层了解支持 PHP 的机会。</p><p data-vmark="e118">4.使用 #ThePHPFoundation 话题标签并提及 @ThePHPF，在社交媒体上传播消息。</p><h3 data-vmark="7504">常见问题解答</h3><p data-vmark="91b2">问：为什么好的语言会遭遇这样不好的事？</p><p data-vmark="bc08">答：每个参与者都会对语言主要贡献者的离开感到难过。但 PHP 将屹立不倒，而基金会则是确保这一点的最佳途径。前景仍然光明。</p><p data-vmark="7869">问：我可以在哪里购买 PHP 大象？</p><p data-vmark="a48f">答：待定 🙂</p><p data-vmark="1230">问：我可以通过 GitHub Sponsors 捐款吗？</p><p data-vmark="1e86">答：暂时不可以。我们计划近期添加此类选项。</p><p data-vmark="193d">问：我可以申请多少资金？</p><p data-vmark="6160">答：你可以申请的范围是占用 20% 精力的兼职职位到全职职位。</p><p data-vmark="fed5">问：基金会的运营范围是什么？</p><p data-vmark="47bb">答：我们最初的运营范围很窄，目的是全力解决 PHP 核心开发者不足这一最紧迫的问题。</p><p data-vmark="e1e7">在头一两年，我们唯一的优先事项是招聘核心开发者。最终，在紧迫问题得到解决后，基金会将准备扩大运营范围，雇用致力于非核心项目的开发者。除此之外，我们还可能扩大关注领域，包括社区导向的项目。</p><p data-vmark="b8bc">问：理事会需要多长时间来正式制定运营程序？</p><p data-vmark="4658">答：基金会的目标是在 6 个月内将运营程序正规化。</p><p data-vmark="6c04">问：这和 Rasmus 的生日有关系吗？</p><p data-vmark="95af">答：没有，11 月 22 日纯属巧合。也与 Nikita 的生日无关，Nikita 的生日是 11 月 23 日。</p><p data-vmark="0508">谢谢！</p><p data-vmark="43f5">特别感谢帮助实现这一切的 PHP 社区成员 – Joe Watkins、Stefan Priebsch、Sebastian Bergmann、Sara Golemon、Larry Garfield、Alexander Makarov、Nicolas Grekas、Ben Ramsey 等。谢谢你们！</p>
