<p>据以色列媒体 globes 报道，瑞典视频分析和安全系统公司 Irisity AB（ST：IRIS）宣布<span class="accentTextColor">以 6750 万美元的现金和股票收购以色列视频分析公司 Agent Vi</span>，后者成立于 2003 年，是一家人工智能视频分析解决方案提供商。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/25577482-d739-4f08-b267-12ae96e45d35.png" w="242" h="289" title="行业加速整合，瑞典安防公司 Irisity 收购以色列视频分析公司 Agent Vi" width="242" height="289" referrerpolicy="no-referrer"></p><p>Agent Vi 开发的智能系统，可利用视频摄像头来识别可疑行为或不规则物体，运营方可以设置相关条件来定义可疑行为，然后系统会在相关条件触发时发出警报。比如，在街道这类场景，系统可根据颜色或服装对过往行人的异常行为进行监控。</p><p>Agent Vi 将产品直接销售给电力和基础设施公司，这些公司将 Agent Vi 的系统集成到其指挥和控制系统中。截至目前，Agent Vi 已经与西门子、Convergint 和 Securitas 等达成合作，业务遍及 90 多个国家和地区。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/cc414a9b-d6fe-4f26-adc4-ccc5dd4d25bd.png" w="1024" h="519" title="行业加速整合，瑞典安防公司 Irisity 收购以色列视频分析公司 Agent Vi" width="1024" height="416" referrerpolicy="no-referrer"></p><p>此前，Agent Vi 已从摩托罗拉、21 Ventures、Indigo Global 和中国科技公司光驰等投资者筹集了 3000 万美元。其财务数据显示，2020 年的的总营收为 610 万美元，净利润 30 万美元。</p><p>“我们相信两家公司在管理、产品、市场和技术等所有重要方面都可以互补。”Irisity 董事长 Lennart Svantesson 表示。</p><p>“安防行业正在经历一场基于技术的重大转型，Irisity 和 Agent Vi 的结合将创建一个支持 AI 的视频分析引擎。”</p>
