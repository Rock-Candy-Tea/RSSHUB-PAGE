            <p data-vmark="4b36"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 8 日消息， 全球开放硬件标准组织 RISC-V International 今天宣布，英特尔公司已加入 RISC-V International 高级会员级别。此举展示了英特尔在开放式协作和支持开放式 RISC-V 架构方面的战略投资，并突显了众多 RISC-V 成员与英特尔的深度合作伙伴关系。英特尔代工服务客户解决方案工程副总裁 Bob Brennan 将加入 RISC-V 董事会和技术指导委员会。</p><p style="text-align: center;" data-vmark="3bf4"><img src="https://img.ithome.com/newsuploadfiles/2022/2/d6a71ce6-eda1-454d-b583-4db27665fb11.jpg" w="680" h="318" title="Intel 正式加入 RISC-V 国际基金会，成为高级会员" width="680" height="318" referrerpolicy="no-referrer"></p><p data-vmark="e2d5">开放的 RISC-V 生态系统提供了现代计算必不可少的开放模块化构建块。作为高级成员加入 RISC-V，英特尔正在为行业利益相关者引领和扩展 RISC-V 生态系统的潜力。RISC-V 的开放式协作模式与跨领域和跨行业的设计自由相结合，促进了硬件社区和 RISC-V 成员的技术创新。英特尔计划投资和对 RISC-V 开放合作的承诺将加强整个生态系统并推动 RISC-V 的商业化。</p><p data-vmark="92d6">英特尔称，很高兴支持免费和开放的 RISC-V 指令集架构的发展。期待针对英特尔工艺技术优化 IP，以确保 RISC-V 在 IFS 芯片上跨所有类型的内核（从嵌入式到高性能）运行最佳。</p><p data-vmark="11b0">英特尔是微处理器行业的先驱，制造了第一个商用微处理器。该公司已经提供基于 RISC-V 的 Nios V 处理器。英特尔对 RISC-V 不断增长的投资将与 RISC-V 社区一起加速开放 RISC-V IP 和其他可交付成果的开发。今天，英特尔宣布了由英特尔代工服务 (IFS) 主导的多项即时 RISC-V 社区福利。IFS 将赞助一个开源软件开发平台，该平台允许实验自由，包括整个生态系统、大学和财团的合作伙伴。IFS 战略将提供针对英特尔工艺技术优化的范围广泛的领先知识产权 (IP)。IFS 是唯一一家提供针对所有三种行业领先 ISA（x86、Arm 和 RISC-V）优化的 IP 的代工厂。英特尔已经看到代工客户对支持更多 RISC-V IP 产品的强烈需求。</p><p data-vmark="301f">RISC-V 社区正在与英特尔合作，为 RISC-V 内核构建必要的 IP，为各个细分市场开发创新，并作为一个整体培养支持软件社区。英特尔还表示通过在技术协同优化、优先考虑晶圆穿梭、支持客户设计、构建开发板和软件基础设施等方面的合作来支持 RISC-V 生态系统。</p>
