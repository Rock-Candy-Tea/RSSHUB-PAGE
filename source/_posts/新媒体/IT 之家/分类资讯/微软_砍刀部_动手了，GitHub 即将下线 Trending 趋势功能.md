<p data-vmark="7406"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 2 日消息，微软 GitHub 在 <a href="https://github.com/trending" target="_blank">Trending 趋势版块</a>下发布提醒，<span class="accentTextColor">从 2022 年 9 月 30 日开始</span>，Trending 趋势将不再可用。</p><p data-vmark="7256" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/9/fcaa5acd-f741-450a-b605-4178d3ab044b.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xZC9xZDgwLnBuZw==,g_5,x_0,y_103,a_0,t_100" w="518" h="580" title="微软“砍刀部”动手了，GitHub 即将下线 Trending 趋势功能" width="518" height="580" referrerpolicy="no-referrer"></p><p data-vmark="2e6f">Trending 趋势是微软 GitHub 于 2018 年推出的一项功能，<span class="accentTextColor">可以查看 GitHub 社区中热门的存储库和开发者</span>。</p><p data-vmark="b6f3">微软 GitHub 官方给出的下线理由是 Trending Repositories（热门存储库）和 Trending Developers（热门开发者）两项功能的<span class="accentTextColor">使用率较低</span>。</p><p data-vmark="8cf6">IT之家了解到，Trending Repositories 展示了前 25 个有潜力的开源项目，用户可对项目进行赞助，还能按照语言、编程语言、每日 / 每周 / 每月进行筛选。</p><p data-vmark="c157">Trending 会根据 star、fork、提交、follow 数等进行统计，每天更新 8 次。</p><p data-vmark="b4c2">大家平时有在看 GitHub 的 Trending 吗？</p>
