<p><img src="https://img.ithome.com/newsuploadfiles/2021/5/9a35b942-c7c7-4147-92f3-9ec6809b0b28.png" w="900" h="383" title="Arm 亮出多款大小核 CPU：AI 算力翻倍，剑指明年安卓旗舰机" width="900" height="349" referrerpolicy="no-referrer"></p><p>5 月 26 日消息，昨日，英国芯片设计公司 Arm 公布了其最新 3 款 CPU 和 3 款 GPU 核心设计。</p><p>三款新 CPU 分别是旗舰核心 Cortex-X2、高性能核心 Cortex-A710、高能效核心 Cortex-A510 CPU，三款新 GPU 核心则覆盖高中端和入门级。</p><p>这是继今年 3 月推出十年来最重要的创新 —— 全新 64 位指令集 Armv9、4 月推出基于 Armv9 的首个产品 —— 面向数据中心的 Neoverse N2 之后，Arm 首次展示基于 Armv9 设计的一系列面向消费级移动设备市场的新核心，这些新设计不仅性能的大幅提升，而且增加新的安全性和人工智能（AI）功能。</p><p>也许很多消费者对 Arm 核心并不了解，简单来说，Arm 的设计几乎是每部安卓手机芯片的共同配置。而此次推出的旗舰 CPU 核心，可以说预告了 2022 年旗舰安卓手机的 CPU 性能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/dae2b076-013b-4e05-b65a-1316dd4f92e1.png" w="1000" h="516" title="Arm 亮出多款大小核 CPU：AI 算力翻倍，剑指明年安卓旗舰机" width="1000" height="423" referrerpolicy="no-referrer"></p><h2>新 CPU 核心：旗舰核机器学习性能提升 2 倍</h2><p>Arm 今年将推出三款基于 Armv9 指令集的新 CPU 设计：X2、A710、A510，分别对标现有的 X1、A78、A55。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/867c42ad-ca3b-455b-8310-4f6332522239.png" w="1000" h="562" title="Arm 亮出多款大小核 CPU：AI 算力翻倍，剑指明年安卓旗舰机" width="1000" height="461" referrerpolicy="no-referrer"></p><p>首先是 Cortex-X2，它是 Arm Cortex-X 定制项目的一部分，允许合作伙伴帮助为其特定用例设计专用核心。</p><p>作为去年 Cortex-X1 的继任者，它也是 Arm CPU 产品线中最强大的设计，可以用于笔记本电脑等大屏设备。Arm 称，与上一代相比，Cortex-X2 的性能将提高 16%，机器学习性能提升 2 倍。</p><p>新款“big”核心 Cortex-A710 则有望比去年推出的 Cortex-A78 提高 10% 的性能、30% 的能效、2 倍的机器学习性能。</p><p>相较“big”核心系列一年一迭代，“LITTLE”核心上一次发布还是在 2017 年。</p><p>时隔四年，Arm 首次推出了新的“LITTLE”高能效核心 Cortex-A510，该核心将取代自 2017 年推出以来一直用于主要手机的 Cortex-A55 设计。与老款 A55 相比，其性能提升 35%，能效提高 20%，机器学习性能提升 3 倍。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/5009aafe-4d7c-44ce-b702-f1ea23731a08.png" w="1000" h="516" title="Arm 亮出多款大小核 CPU：AI 算力翻倍，剑指明年安卓旗舰机" width="1000" height="423" referrerpolicy="no-referrer"></p><p>当这些新核心进入芯片后，新的 Armv9 设计将带来性能的大幅提升。例如，用在许多最新安卓旗舰手机的高通骁龙 888 芯片，即采用去年旗舰 Arm Cortex-X1 和 Cortex-A78 的部分定制版本作为其四个“大”核心，并使用大约四年历史的 Cortex-A55 设计作为其“LITTLE”核心。三星旗舰芯片 Exynos 2100 也采用类似的配置，并采用了 Arm 的 Mali-G78 GPU 设计。不出意外，这两款芯片的下一代都有望用上新的 Arm CPU 核心设计。</p><p>根据 Arm 公布的信息，相较 Armv8.2，由 Armv9 设计（单个 Cortex-X2、三个 Cortex-A710 核心和四个 Cortex-A510 核心）的 CPU 丛簇可以提升高达 30% 的峰值性能（得益于 Cortex-X2）、30% 的整体效率（得益于 Cortex-A710）和 35% 的“LITTLE”性能（得益于 Cortex-A510）。</p><h2>A710 保留兼容 32 位，A510 采用新型微架构</h2><p>三款新 CPU 核心中，X2、A510 都将是纯 64 位，不再兼容 32 位，而 A710 会继续支持 OL0 AArch32。</p><p>其中，X2 和 A710 的前端都改进了分支预测，精度更高，错误更少。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/7d79718f-691d-4372-b19a-11c4472f459c.png" w="1000" h="523" title="Arm 亮出多款大小核 CPU：AI 算力翻倍，剑指明年安卓旗舰机" width="1000" height="429" referrerpolicy="no-referrer"></p><p>凭借 L3 缓存和丛簇设计 DSU-110 的特性，单一丛簇（cluster）最多可容纳 8 个 X2 核心，并具有最大 16MB 的 L3 快取能力，使得笔记本电脑 SoC 的设计更具弹性，也为其带来跨入高效能 PC 的可能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/9c04a3d6-a23a-4dc2-8f66-4a6861e8c384.png" w="1000" h="580" title="Arm 亮出多款大小核 CPU：AI 算力翻倍，剑指明年安卓旗舰机" width="1000" height="476" referrerpolicy="no-referrer"></p><p>A710 核心设计也优化了与 DSU 的联系，与 DSU、内存之间的延迟更低。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/4e3b1f04-d4c4-4d27-960d-3168b1af85c0.png" w="1000" h="533" title="Arm 亮出多款大小核 CPU：AI 算力翻倍，剑指明年安卓旗舰机" width="1000" height="437" referrerpolicy="no-referrer"></p><p>A510 则采用了一个混合核心微架构（merged core microarchitecture）新设计，可将 2 个 A510 组合成一个群组，单一 CPU 可由多个群组构成，从而实现更加弹性化的结构设计。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/a314d233-dcba-4a7f-a020-cdeef4160ee8.png" w="1000" h="533" title="Arm 亮出多款大小核 CPU：AI 算力翻倍，剑指明年安卓旗舰机" width="1000" height="437" referrerpolicy="no-referrer"></p><h2>新 GPU 核心：旗舰级游戏性能提升 20%<br></h2><p>Arm 还将推出 3 款新 GPU：旗舰产品 Mali-G710、中端产品 Mali-G510、入门级 Mali-G310。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/23d586af-f5b8-4081-9981-b56db57d3e89.png" w="1000" h="516" title="Arm 亮出多款大小核 CPU：AI 算力翻倍，剑指明年安卓旗舰机" width="1000" height="423" referrerpolicy="no-referrer"></p><p>其中，旗舰产品 Mali-G710 将带来 20% 的游戏性能提升、20% 的能效提升、35% 的机器学习性能提升。</p><p>Mali-G510 为更高性价比设备的中档选择，可将性能提升 100%，能效提升 22%，机器学习性能提升 100%。</p><p>入门级 Mali-G310 可将纹理处理能力提升高达 6 倍，Vulkan 性能提升 4.5 倍，安卓 UI 内容处理能力提升 2 倍。</p><h2>结语：最早预计 2022 年初落地<br></h2><p>Arm 的总体目标是为广泛的用例提供各种设计。计算机可能更多地依赖 Arm 的 Cortex-X2 CPU 和用于图形处理的独立 GPU 解决方案，先进的智能手机将采用基于 Arm 系列 CPU 设计的 CPU 丛簇和 Mali-G710 GPU，对性能需求较低的智能手表可能采用 Cortex-A510 和 Mali-G310。</p><p>新的 Arm 设计还需要一段时间才能出现在手机或设备上：Arm 仍然需要将设计交给合作伙伴，然后合作伙伴需将它们引入自己的芯片产品，然后将这些芯片提供给手机制造商。</p><p>因此，新的 Arm CPU 和 GPU 设计可能要到 2022 年初才会应用在新款手机上，并且前提是持续发酵的全球芯片短缺不会影响明年产品的推出。</p>
