            <p data-vmark="f0e8"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT 之家</a> 4 月 28 日消息，微博管理员今天上午发布了 IP 属地功能升级公告。</p><p data-vmark="8003">公告称，微博一直致力于维护健康有序的讨论氛围，保障用户第一时间获取真实有效信息的权益。为减少冒充热点事件当事人、恶意造谣、蹭流量等不良行为，确保传播内容的真实、透明，站方于今年 3 月上线展示用户“IP 属地”功能。</p><p data-vmark="33a9">为进一步保障用户权益、提升用户体验，站方将在此前基础上于 4 月 28 日进行产品升级和新功能上线，具体如下：</p><p data-vmark="d585">一、全量开放评论展示发评 IP 属地小尾巴功能；</p><p data-vmark="c439" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/9832fec1-24a6-43fd-8547-36b4f1e12df6.jpg" w="444" h="657" title="微博 IP 属地功能大升级：全量强制开放评论展示小尾巴，上线个人主页一级页面展示" width="444" height="657" referrerpolicy="no-referrer"></p><p data-vmark="bb48">二、上线“个人主页一级页面展示 IP 属地”功能。</p><p data-vmark="685d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/e587b7e8-31ba-44d7-a02b-95e5ccbadfda.jpg" w="408" h="619" title="微博 IP 属地功能大升级：全量强制开放评论展示小尾巴，上线个人主页一级页面展示" width="408" height="619" referrerpolicy="no-referrer"></p><p data-vmark="ab47">下面是问答：</p><p data-vmark="63a5">Q1: 评论区 IP 属地显示可以关闭吗？</p><p data-vmark="6c2f">A1: <span class="accentTextColor">该功能旨在更透明地向广大用户呈现真实情况，无法由用户主动开启或关闭</span>。</p><p data-vmark="5c0b">Q2: 个人主页 IP 属地根据什么显示？</p><p data-vmark="af36">A2: 根据最近您发博、发评、投票的 IP 属地判定所属地区。</p><p data-vmark="e492">Q3: IP 属地显示规则是什么？</p><p data-vmark="c1f6">A3: 显示规则为国内显示到省份 / 地区，国外显示到国家。</p><p data-vmark="da77">Q4: 为什么定位不准？</p><p data-vmark="bd36">A4: 站方只展示对应 IP 属地位置，并不获取其他信息，定位根据运营商 IP 信息解析，如有问题可咨询对应运营商。</p><p data-vmark="fc4b" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/3962c8fc-45aa-4b1f-bcd4-feed4d30d401.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xZC9xZDExNC5wbmc=,g_7,x_11,y_11,a_0,t_100" w="744" h="958" title="微博 IP 属地功能大升级：全量强制开放评论展示小尾巴，上线个人主页一级页面展示" width="744" height="958" referrerpolicy="no-referrer"></p>
