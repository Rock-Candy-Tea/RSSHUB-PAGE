<p data-vmark="21a9"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 11 日消息，据 MSPoweruser 报道，今年年初，Twitter 因一种似乎有偏见的裁剪算法而受到批评，该公司在 iOS 和 <a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 上的解决方案相当简单有效 —— 停止裁剪图片的显示方式。</p><p data-vmark="c967">现在，Twitter 公司终于把这个解决方案带到了他们的 Web 网页应用中。</p><p data-vmark="cfe5"><img src="https://img.ithome.com/newsuploadfiles/2021/11/145b1b05-ab41-414b-9265-5c2ec309d0db.png" w="688" h="1034" title="Twitter Web 版不再显示“裁剪”的图片" width="688" height="1034" referrerpolicy="no-referrer"></p><p data-vmark="bb5f"><img src="https://img.ithome.com/newsuploadfiles/2021/11/cfb2aeba-fa3d-4db9-81ac-3df5f7b00a15.png" w="689" h="371" title="Twitter Web 版不再显示“裁剪”的图片" width="689" height="371" referrerpolicy="no-referrer"></p><p data-vmark="fac2">这意味着用户将不再需要点击图片就能看到完整的图片，这是一个非常受欢迎的改进，那些在 Twitter 上发布图片的用户将清楚地知道他们的粉丝会看到什么。</p><p data-vmark="79c0">IT之家获悉，现在，这项功能似乎正在向所有人推出。</p>
