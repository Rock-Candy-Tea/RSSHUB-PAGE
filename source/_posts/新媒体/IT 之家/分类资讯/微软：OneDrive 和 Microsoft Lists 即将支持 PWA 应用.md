<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 12 日消息 据外媒 MSPoweruser 报道，微软宣布很快将把 OneDrive 和 Microsoft Lists web 应用作为 PWA 应用 （Progressive web apps）提供。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/2/20210207_112227_540.png" title="微软：OneDrive 和 Microsoft Lists 即将支持 PWA 应用" referrerpolicy="no-referrer"></p><p>在用户将 Microsoft Lists 和 OneDrive 的 Web 版本安装为 PWA 应用后，其将与任何独立桌面应用的工作方式相似。</p><p>这也意味着用户可以从桌面启动应用。应用的工作原理与在浏览器的原理相同，包括能够从电脑的主屏幕或任务栏锁定和启动应用，使用应用程序快捷方式，并执行与它们 Web 版本应用相同的操作。</p><p>Microsoft Edge、Google Chrome、Firefox 或其他支持渐进式网页应用的浏览器（Safari 除外）将支持 PWA。</p><p>IT之家了解到，该功能将在 7 月初推出目标版本，在 7 月中旬推出标准版本，并应在 8 月初完成。</p>
