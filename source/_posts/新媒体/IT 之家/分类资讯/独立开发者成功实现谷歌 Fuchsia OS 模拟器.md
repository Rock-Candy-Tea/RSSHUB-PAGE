<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 31 日消息 谷歌在上周确认，将为初代 Nest Hub 智能显示器更新 Fuchsia OS，正式使这一等待了数年的系统问世。</p><p><img src="https://img.ithome.com/newsuploadfiles/2019/10/20191016081258_4549.jpg" title="独立开发者成功实现谷歌 Fuchsia OS 模拟器" referrerpolicy="no-referrer"></p><p>然而，普通用户想要体验到该系统可能有点复杂，虽然 Fuchsia OS 是开源的，但一般只能在 Pixelbook 等兼容设备上运行。</p><p>近日，Dahlia OS 的独立开发者宣布，<span class="accentTextColor">已</span><span class="accentTextColor">经实现了 Fuchsia OS 的模拟器 Flmage，现已在 GitHub 开放</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/85830030-2767-4681-b6c8-af0cd31e9f16.png" w="892" h="661" title="独立开发者成功实现谷歌 Fuchsia OS 模拟器" width="892" height="608" referrerpolicy="no-referrer"></p><p>IT之家了解到，<span class="accentTextColor">Flmage 目前仅支持 Linux 平台</span>，最好在 Ubuntu 20.04 或以上。如果要获得良好的体验，电脑硬件需要 <span class="accentTextColor">8 GB 以上内存</span>、第四代及以上英特尔处理器、最新的图形驱动程序。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/6308b378-9b47-43d7-80c8-73273f9b1ac4.png" w="1440" h="800" title="独立开发者成功实现谷歌 Fuchsia OS 模拟器" width="1440" height="456" referrerpolicy="no-referrer"></p><p>Dahlia OS 的开发者们表示，他们打算定期更新 FImage 项目到 Fuchsia OS 的最新版本，以帮助 Fuchsia OS 的爱好者更好地体验这一系统。</p><p>具体使用方法都在 GitHub 的 README.md
