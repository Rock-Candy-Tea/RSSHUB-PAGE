<p data-vmark="aaa5">嗨胖友们，今儿 IE 浏览器就正式退出历史舞台了。尽管你可能早八百年就只用 IE 来下 Chrome 了，不过作为“童年回忆”，网友们对于这位老同志，感情还是非常深的。</p><p data-vmark="e129">这不，梗图排着队就来了。</p><p data-vmark="28d6" style="text-align: center;"><img referrerpolicy="no-referrer"></p><blockquote><p data-vmark="4475">死神：IE，是时候上路了。</p><p data-vmark="d840">IE：Internet Explorer 已停止工作。</p></blockquote><p data-vmark="792b">有人觉得 IE 靠着魂器仍存于世（手动狗头）。</p><p data-vmark="ea56" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/6e928530-c85a-44ed-84f4-a527346e0103.png" w="868" h="1020" alt="IE寿终正寝，网友们搞起了真·墓碑…" title="微软 IE 浏览器寿终正寝，网友们搞起了真・墓碑" width="868" height="964" referrerpolicy="no-referrer"></p><p data-vmark="daba">但也有人迫不及待给 IE P 起了墓碑。</p><p data-vmark="f93c" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/e5d82910-a9e1-468d-807e-ad44d61058e5.png" w="798" h="622" alt="IE寿终正寝，网友们搞起了真·墓碑…" title="微软 IE 浏览器寿终正寝，网友们搞起了真・墓碑" width="798" height="622" referrerpolicy="no-referrer"></p><p data-vmark="105a">而据韩国网友透露，在韩国庆州，还有实体版……</p><p data-vmark="63db" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/63bef136-fe72-4895-851d-af2a65f94d50.png" w="1074" h="1244" alt="IE寿终正寝，网友们搞起了真·墓碑…" title="微软 IE 浏览器寿终正寝，网友们搞起了真・墓碑" width="1074" height="950" referrerpolicy="no-referrer"></p><p data-vmark="49e5">墓志铭写的是：他曾经是个下载其他浏览器的好工具。</p><p data-vmark="80b4" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/cc0ae6bf-77d7-4602-95ee-0865e83fb05d.jpg@s_2,w_820,h_777" w="1196" h="1134" alt="IE寿终正寝，网友们搞起了真·墓碑…" title="微软 IE 浏览器寿终正寝，网友们搞起了真・墓碑" srcset="https://img.ithome.com/newsuploadfiles/2022/6/cc0ae6bf-77d7-4602-95ee-0865e83fb05d.jpg 2x" width="1196" height="777" referrerpolicy="no-referrer"></p><h2 data-vmark="7a56">从 96% 到 0.64%</h2><p data-vmark="9087">值此送别之际，我们还是来回顾一下 IE 这位曾经的浏览器老大哥波澜起伏的一生。</p><p data-vmark="d042">IE 的第一个版本 Internet Explorer 1 诞生于 1995 年 8 月。第一轮网页浏览器“大战”，也就此拉开序幕：当时，网景（Netscape）作为浏览器界一哥，市场占有率超过 70%。</p><p data-vmark="7525">值得注意的是，那时候苹果的默认浏览器就是网景，而作为竞争对手，在 IE 之前微软并没有自己的默认浏览器。</p><p data-vmark="1a49" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/b1d486cd-ead6-4d46-9ce5-5a5adc053a17.png" w="1080" h="676" alt="IE寿终正寝，网友们搞起了真·墓碑…" title="微软 IE 浏览器寿终正寝，网友们搞起了真・墓碑" width="1080" height="513" referrerpolicy="no-referrer"></p><p data-vmark="f53c">通过与 Windows 系统捆绑的方式，IE 很快就给网景造成了冲击。特别是在 1996 年 Internet Explorer 3—— 首款支持编程语言及 CSS 的商用浏览器推出之后，IE 的市场占有率开始紧追网景。</p><p data-vmark="51f5">而随着两者竞争的白热化，当时的网页设计者们还会把“用网景可获得最佳效果”、“用 IE 可获得最佳效果”的标志放在主页上，甚至由此触发了名为 Viewable With Any Browser（可用任何浏览器浏览）的运动。</p><p data-vmark="1cc8">1998 年，背靠财大气粗的微软，IE 正式斩网景于马下。后者在这一年年底被美国网络公司美国在线（AOL）收购。此后，IE 一路高歌。到了 2002 年，其市场份额达到了惊人的 96%，可以说占据了浏览器领域的绝对统治地位。</p><p data-vmark="2bd2" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/3543344f-5f32-457a-8a00-755b0e7d604a.png" w="1080" h="675" alt="IE寿终正寝，网友们搞起了真·墓碑…" title="微软 IE 浏览器寿终正寝，网友们搞起了真・墓碑" width="1080" height="512" referrerpolicy="no-referrer"></p><p data-vmark="bcb2">只是 IE 成功了，却也懈怠了。</p><p data-vmark="17b8">IE 6.0 版本 2001 年推出，而其下一代版本却直到 2005 年才与用户见面，IE6 也成为该系列产品中生命周期最长的一个版本。并且在这长达 5 年的时间内，IE6 不断受到用户诟病 —— 运行速度慢不说，安全漏洞还层出不穷……</p><p data-vmark="d01f">而网景虽败，却仍留下了一点星星之火。</p><p data-vmark="3c79">就在败退的 1998 年，在网景资助下，Mozilla 组织成立。没错，就是火狐浏览器（Mozilla Firefox）名字里的那个 Mozilla。这也是为什么，火狐被认为是网景的“精神续作”。</p><p data-vmark="49a3">2004 年，火狐推出 1.0 版本。到了 2005 年，IE 市占率就在火狐的冲击之下，跌至 85%。如此竞争压力之下，微软也终于开始重拾第一轮浏览器之战时的创新动力，加快 IE 7.0 版本的研发。</p><p data-vmark="322f">但更强大的竞争对手很快就出现了 ——2008 年，谷歌推出 Google Chrome 浏览器，同时推出对应开源版本 Chromium。Chromium 对现今浏览器的影响不需要多说：如今替代了 IE Windows 系统默认浏览器之位的 Microsoft Edge，在 2020 年也已经改为基于 Chromium 开发。</p><p data-vmark="0f8d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/8c90518e-ea0d-4041-aba5-a04ad65eb52b.png" w="1080" h="738" alt="IE寿终正寝，网友们搞起了真·墓碑…" title="微软 IE 浏览器寿终正寝，网友们搞起了真・墓碑" width="1080" height="560" referrerpolicy="no-referrer"></p><p data-vmark="1782" style="text-align: center;">△ 来自 StarCounter</p><p data-vmark="f02f">Chrome 的强势登场，成为了压死骆驼的最后一根稻草。</p><p data-vmark="cfad">2015 年，IE 的市场占有率已经跌破 20%。这一年的微软 Build 开发者大会上，微软也开始舍弃 IE，宣布用 Microsoft Edge 替代 IE，成为新的 Windows 系统默认浏览器。</p><p data-vmark="8287">2016 年，微软宣布将会停止发布 Internet Explorer 11 之前版本的安全更新。</p><p data-vmark="ea87">2020 年，微软宣布将陆续停止对 IE 的支持。</p><p data-vmark="9d6e">2021 年，微软预告了 IE 的“寿终正寝”。</p><p data-vmark="9f2a">而现在，告别时刻已至。</p><p data-vmark="339a">数据显示，在上个月，IE 全球市场份额仅余 0.64%。</p>
