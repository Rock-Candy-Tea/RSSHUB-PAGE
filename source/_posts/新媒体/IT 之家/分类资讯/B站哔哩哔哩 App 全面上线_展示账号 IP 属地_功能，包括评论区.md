            <p data-vmark="879d"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 25 日消息，此前B站官方宣布：为保障社区真实有序的讨论氛围，持续防治冒充热点事件当事人、恶意造谣、蹭流量等不良行为，社区将依据相关法律法规要求逐步开放“展示账号 IP 属地”功能，用户暂时无法主动开启或关闭相关展示。今天开始，B站全面上线“展示账号 IP 属地”功能，包括评论区。不过目前B站网页版还未上线“展示账号 IP 属地”。</p><p style="text-align: center;" data-vmark="49df"><img src="https://img.ithome.com/newsuploadfiles/2022/7/550e5557-6a1d-467a-8c38-29cdd002d657.png" w="1079" h="1922" title="B站哔哩哔哩 App 全面上线“展示账号 IP 属地”功能，包括评论区" width="1079" height="1461" referrerpolicy="no-referrer"></p><p data-vmark="05ef">B站表示，该功能上线后，将在用户个人主页、信息发布页等位置展示 IP 属地，境内展示到省 (自治区、直辖市)，境外展示到国家 (地区)，账号 IP 属地信息以网络运营商提供信息为准，用户暂时无法主动开启或关闭相关展示。</p><p style="text-align: center;" data-vmark="89d1"><img src="https://img.ithome.com/newsuploadfiles/2022/7/f917e005-5a0e-4c7e-89e4-b03005c5b12f.jpg?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzE2Ni5wbmc=,g_7,x_17,y_17,a_0,t_100" w="1080" h="2400" title="B站哔哩哔哩 App 全面上线“展示账号 IP 属地”功能，包括评论区" width="1080" height="1822" referrerpolicy="no-referrer"></p><p data-vmark="100f">值得一提的是，今年 3 月开始，微博、抖音、快手、今日头条、知乎、百度、微信等平台逐步开放“展示 IP 属地”功能。</p>
