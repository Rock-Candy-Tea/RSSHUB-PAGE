<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 18 日消息 微软 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a>/<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 的预览渠道分为三种，按照更新速度分为 Dev 渠道、Beta 渠道、Release Preview 渠道三种，目前 Win11 已向 Dev 渠道推送了多个版本更新。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/ee9b310c-fe78-4854-81b5-af9cd07dd9ec.png" w="1076" h="375" title="微软：Win11 Beta 渠道首个预览版将在月底发布" width="1076" height="286" referrerpolicy="no-referrer"></p><p>据 Windows Insider 官推，Win11 的首个 Beta 渠道预览版将在<span class="accentTextColor">月底推出</span>，相比 Dev 渠道更加稳定，Bug 更少，IT之家小伙伴可以耐心等待。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/b4c31fca-2641-470d-b046-24402c448ac0.png" w="738" h="226" title="微软：Win11 Beta 渠道首个预览版将在月底发布" width="738" height="226" referrerpolicy="no-referrer"></p><p>微软近日确定了 Win11 <span class="accentTextColor">Build 22000</span> 作为 RTM 候选版本，正在测试累积更新。</p><p>此外，微软近日向 Dev 渠道用户推送了 Win11 Build 22000.71 更新（KB5004252），带来了全新的亚克力菜单、任务栏预览窗口，以及大量 Bug 修复。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/f783659c-57cc-4c4d-adb6-a8aa15a066ad.png" title="微软：Win11 Beta 渠道首个预览版将在月底发布" referrerpolicy="no-referrer"></p>
