<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 6 日消息 微软将于 6 月 24 日发布“下一代 Windows”，之前已经有多个迹象表明，下一代 Windows 可能被称为“Windows 11”。</p><p>昨日，IT之家网友发现了一个隐藏的新信息，那就是<span class="accentTextColor">在微软发布的预告图片中，微软 Logo 和 Windows Logo 之间有一个夹角，而这个夹角就是 11°</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/18a1ca18-1fab-4766-8a88-36e167950f1a.jpg@s_2,w_820,h_384" title="Win11 又一铁证！微软 Windows 海报新发现：Logo 夹角 11°" referrerpolicy="no-referrer"></p><p><img src="https://qimg.ithome.com/image/2021/06/05/6375845025102170847755877.png" title="Win11 又一铁证！微软 Windows 海报新发现：Logo 夹角 11°" referrerpolicy="no-referrer"></p><p><img src="https://qimg.ithome.com/image/2021/06/05/6375845025502176405067188.jpg" title="Win11 又一铁证！微软 Windows 海报新发现：Logo 夹角 11°" referrerpolicy="no-referrer"></p><p>此外，微软有意去掉了水平阴影，使其像 11 数字，同时该活动在不寻常的美东时间上午 11 点开始，这也暗示了“Windows 11”的可能性。</p><p>另外，今天早些时候，爆料大神 @evleaks （Evan Blass）也表示，<span class="accentTextColor">微软即将推出的桌面操作系统版本可能被称为“Windows 11”</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/34ce43a9-abbe-462f-9756-bcc44a352e63.jpg" title="Win11 又一铁证！微软 Windows 海报新发现：Logo 夹角 11°" referrerpolicy="no-referrer"></p><p>IT之家了解到，2015 年，微软领导层表示，Windows 10 将是其标志性桌面操作系统的最终版本，不会有 Windows 11 或 Windows 12，而是会发布后续的 Windows 10 版本。</p><p>在过去的六年里，微软一直在通过软件更新对 Windows 10 进行定期改进，但这可能很快就会改变。在 5 月 25 日的微软 Build 开发者大会上，微软首席执行官萨提亚 - 纳德拉说，<span class="accentTextColor">微软正在计划“过去十年中最重要的 Windows 更新之一”</span>，证实了 Windows 操作系统的重大重新设计即将到来的传言。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/bf6e372b-c3e2-4dae-9ce3-08dc4aba54dd.gif" title="Win11 又一铁证！微软 Windows 海报新发现：Logo 夹角 11°" referrerpolicy="no-referrer"></p><p>虽然“太阳谷更新”将是一次大的升级，但它仍将被视为另一个版本的 Windows 10。如果微软想在下一个 Windows 更新中大做文章，需要从简单的“Windows 10 + 月份”的名称品牌中走出来。</p>
