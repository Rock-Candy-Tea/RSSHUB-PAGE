<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 2 日消息 在最新的 Edge 浏览器 Canary 版本中，微软带来了一项实验功能，可使管理菜单界面更易用。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/7c5f2863-7bb8-4d52-9e08-7ba69defeae4.jpg" alt="Microsoft Edge flyout menu" w="696" h="365" title="微软 Edge 浏览器将优化管理菜单界面，可永久固定历史记录等" width="696" height="365" referrerpolicy="no-referrer"></p><p>据外媒 Windows Latest 报道，微软 Edge 浏览器的下载、历史记录、收藏夹等功能目前可在弹窗中打开，也可以单次固定到侧边栏，然而却不能永久固定。</p><p>在最新的 Canary 版本中，微软添加了一项名为“Persist pin state for menus”的实验选项，<span class="accentTextColor">用户可选择将这些管理菜单进行永久固定</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/5b9b0ed1-00e3-443b-b8d1-45bc2ba213f7.jpg@s_2,w_820,h_360" alt="Edge flags menu" w="1322" h="580" title="微软 Edge 浏览器将优化管理菜单界面，可永久固定历史记录等" srcset="https://img.ithome.com/newsuploadfiles/2021/5/5b9b0ed1-00e3-443b-b8d1-45bc2ba213f7.jpg 2x" width="1322" height="360" referrerpolicy="no-referrer"></p><p>第一种是添加一个取消固定按钮，用户可以点击该按钮将管理菜单进行取消固定。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/09476c08-a2d6-435f-99d0-d2c27ee2fcf8.jpg" alt="Edge menu unpin" w="634" h="284" title="微软 Edge 浏览器将优化管理菜单界面，可永久固定历史记录等" width="634" height="284" referrerpolicy="no-referrer"></p><p>第二种是添加一个取消固定按钮和关闭按钮。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/05f154aa-4444-4070-bf61-8f8acb211c0f.jpg" alt="Edge menu unpin and close" w="608" h="310" title="微软 Edge 浏览器将优化管理菜单界面，可永久固定历史记录等" width="608" height="310" referrerpolicy="no-referrer"></p><p>第三种是永久固定，没有取消固定按钮，收藏夹、集锦等将始终作为侧边栏打开。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/1ece990b-0991-4d65-938b-ba764c1ee4d2.jpg" alt="Edge menu always pinned" w="671" h="570" title="微软 Edge 浏览器将优化管理菜单界面，可永久固定历史记录等" width="671" height="570" referrerpolicy="no-referrer"></p><p>IT之家了解到，该实验选项已出现在 <span class="accentTextColor">Canary 版本 Build 92.0.874.0 + 中</span>，感兴趣的IT之家小伙伴可以尝试下载体验，不过不推荐作为日常使用。</p>
