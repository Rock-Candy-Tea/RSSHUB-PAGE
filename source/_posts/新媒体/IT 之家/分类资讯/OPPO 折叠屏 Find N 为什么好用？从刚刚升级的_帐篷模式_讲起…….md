<p data-vmark="951e">8 月 5 日，OPPO 折叠屏 Find N 迎来了 ColorOS 新版本（11-A-22）更新，此次更新除了新增 Android 2022 年 7 月份安全补丁之外，还迎来了两项重磅功能 ——Omoji 表情和帐篷模式。</p><p data-vmark="2af8">Omoji 大家应该很熟悉了，在 Find N 上，Omoji 能够跟踪每一丝表情，定制用户的专属虚拟形象。</p><p data-vmark="e41d">至于另一项功能“帐篷模式”，不少用户可能就摸不着头脑了，“帐篷模式”又是何方神圣？今天上午，OPPO 官方分享了一张海报，进一步详解了这个“帐篷模式”。就如同字面意思 ——“帐篷模式”是指，在亮屏解锁状态下时，将折叠屏 Find N 倒扣放置在桌面上，此时外屏朝外，呈现折叠开合角度在 10°～120° 之间的“帐篷”状态，画面和操作会锁定在外屏上。</p><p style="text-align: center;" data-vmark="3cc8"><img src="https://img.ithome.com/newsuploadfiles/2022/8/a78ed872-a79a-43b8-b39f-7a729cc55cb7.png" w="810" h="1440" alt="图片" title="OPPO 折叠屏 Find N 为什么好用？从刚刚升级的“帐篷模式”讲起……" width="810" height="1440" referrerpolicy="no-referrer"></p><p data-vmark="e11a">“帐篷模式”的加入，进一步解锁了折叠屏手机全新的交互方式，在“帐篷模式”下，你可以更加轻松自如的使用外屏，比如通过外屏看电影，或者是使用一些横屏应用，此时在“帐篷模式”下，外屏以更自然、舒适的角度呈现在你眼前，而此时整个手机也兼顾了手机支架的功能，甚至比“悬停”还要稳定，实用性极强。</p><p style="text-align: center;" data-vmark="f972"><img src="https://img.ithome.com/newsuploadfiles/2022/8/b7e54363-fc76-4dce-ada0-6f9d3b13fbe6.jpeg" w="1210" h="807" alt="图片" title="OPPO 折叠屏 Find N 为什么好用？从刚刚升级的“帐篷模式”讲起……" width="1210" height="547" referrerpolicy="no-referrer"></p><p data-vmark="97f8">“帐篷模式”可以理解为另一种形态的“悬停交互”，它实现了手机在折叠、展开之外的第三种使用形态，更是给折叠屏手机留下了很大的想象空间，而如果想在普通直板手机上实现类似功能，除非使用外置支架，不然根本没可能。</p><p style="text-align: center;" data-vmark="5e32"><img src="https://img.ithome.com/newsuploadfiles/2022/8/e9b85b24-722e-42e1-b5af-3d2db45e4714.jpeg" w="1210" h="807" alt="图片" title="OPPO 折叠屏 Find N 为什么好用？从刚刚升级的“帐篷模式”讲起……" width="1210" height="547" referrerpolicy="no-referrer"></p><p data-vmark="4e05">作为 OPPO 的首款折叠屏产品，OPPO Find N 一发布便喊出了“从尝鲜到常用”的口号，背后的底气，不仅仅是超高的性价比，还有着强大的设计和交互能力所支撑，尤其是系统设计，很多地方都与普通直板手机不同，下面我们就来讲讲折叠屏 Find N 系统设计的那些“巧心思”吧。</p><p data-vmark="1f38">近些年来，折叠屏已经成为各大手机产商在高端市场的重要突破口，但 OPPO 没有跟风强上折叠品手机，而是甘愿做一个厚积薄发的“笨鸟”，耐心打磨体验，从尺寸形态到转轴、折痕，从应用交互的实用性再到悬停的创新体验，Find N 的每一处细节，都隐藏着 OPPO 多年来对折叠屏手机的观察和思考。</p><p data-vmark="f737">所以，呈现在我们面前的 OPPO Find N，可谓是折叠屏手机市场的瑰宝。在外观上，它精致小巧，采用左右翻折的设计方案，外屏 5.49 英寸，长宽比 18:9 的黄金折叠比例，展开后的内屏则是 7.1 英寸的横屏，加上外屏边缘 3D 小微曲设计，让折叠屏摆脱了粗犷厚重的标签。同时搭载 OPPO 自研精工拟椎式铰链，拥有近乎无折痕的全新设计，并支持多角度悬停功能，增加了用户的使用场景。</p><p data-vmark="252b">在交互上，Find N 独创双指下滑分屏、屏幕悬停、自由拖拽、定制化输入法等便捷交互体验，在折叠屏的交互体验上下了很多功夫。</p><p data-vmark="6fc0">以相机悬停为例，拍摄一些长曝光样张，或者是延时摄影，往往需要配合三脚架，而在悬停模式下，OPPO Find N 下半部分就可以作为“三脚架”固定在平面上。</p><p style="text-align: center;" data-vmark="a473"><img src="https://img.ithome.com/newsuploadfiles/2022/8/ddbddc96-4c75-4f60-af83-50f832138fa5.gif" w="656" h="368" alt="图片" title="OPPO 折叠屏 Find N 为什么好用？从刚刚升级的“帐篷模式”讲起……" width="656" height="368" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="57c5">（悬停摄影）</p><p data-vmark="7d3c">再比如悬停观影，将屏幕调整至任意角度放在桌面，就可以一半屏幕播放影片，一半屏幕进行操控。</p><p style="text-align: center;" data-vmark="3a5b"><img src="https://img.ithome.com/newsuploadfiles/2022/8/b0406304-6770-41fe-8c9b-454e3f00ff70.gif" w="656" h="368" alt="图片" title="OPPO 折叠屏 Find N 为什么好用？从刚刚升级的“帐篷模式”讲起……" width="656" height="368" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="5e64">（悬停观影）</p><p data-vmark="2485">悬停视频通话，把手机“立”在桌面，就能解放双手，一边通话一边做别的事了。开启后，上屏通话界面五五开左右分布画面，下屏显示控制按钮，双方都看得清对方。</p><p style="text-align: center;" data-vmark="faa8"><img src="https://img.ithome.com/newsuploadfiles/2022/8/8902d78b-bd0a-44c6-9d31-e788a4015550.gif" w="656" h="368" alt="图片" title="OPPO 折叠屏 Find N 为什么好用？从刚刚升级的“帐篷模式”讲起……" width="656" height="368" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="639c">（悬停视频通话）</p><p data-vmark="0feb">可见，OPPO Find N 不仅在折叠屏的硬件品质上做了创新与打磨，更没有浪费折叠屏的天然优势，挖掘了很多与折叠相关的细节功能，与直板手机相比，在很多方面的体验都有所不同。</p><p data-vmark="54a6">在大屏状态下，OPPO 还优化了输入法使用体验。比如输入法布局上，OPPO 设计了分离式键盘，同时还精密测量了上百位用户握持姿势，对输入区域进行了优化，让打字体验更自然、舒适。</p><p style="text-align: center;" data-vmark="359a"><img src="https://img.ithome.com/newsuploadfiles/2022/8/b6c13a66-312e-4f1b-9405-60ab809be575.gif" w="656" h="368" alt="图片" title="OPPO 折叠屏 Find N 为什么好用？从刚刚升级的“帐篷模式”讲起……" width="656" height="368" referrerpolicy="no-referrer"></p><p data-vmark="6cc1">此外，OPPO Find N 的“双指分屏功能”只需一步即可分屏，大大提升了折叠屏分屏的效率和便捷性，是个很实用的交互创新，也大大提升了折叠屏的使用体验。这项功能，更是获得了全球知名度非常高的 2022 年 iF 国际设计奖。</p><p style="text-align: center;" data-vmark="cf87"><img src="https://img.ithome.com/newsuploadfiles/2022/8/a78b9cd6-bfdb-44d2-be1d-a4a076f5fc6a.png" w="1210" h="1151" alt="图片" title="OPPO 折叠屏 Find N 为什么好用？从刚刚升级的“帐篷模式”讲起……" width="1210" height="780" referrerpolicy="no-referrer"></p><h2 data-vmark="979b">小结：</h2><p data-vmark="2fe3">来自知名国际数据公司（IDC）手机季度跟踪报告显示，折叠屏细分领域保持了高速增长态势，其中 OPPO 凭借 Find N 显著的差异化产品设计，在上半年折叠屏市场份额位列第二名。另一则数据显示，2022 年 1 月 1 日-6 月 20 日近半年时间内，OPPO Find N 荣获京东自营平台 2022 上半年折叠品类销量榜首，好评率达高达 98%。</p><p data-vmark="f966">销量和消费者口碑的双赢，是对 OPPO Find N 的最大认可。</p><p data-vmark="fbbe">不断优化升级的 OPPO Find N，让我们更清晰的看到，其实折叠屏手机在使用体验上还有很多跃升的可能，而 OPPO Find N 所带来的这些差异化体验，在逆风市场环境中，正加速折叠屏从尝鲜走向常用，而且折叠屏已经处于了一个比较成熟的阶段了，折叠屏更是将成为我们选购手机时一个重要选项，那么问题来了，你的下一步手机会选择折叠屏吗？欢迎留言交流。</p>
