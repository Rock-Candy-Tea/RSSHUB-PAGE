<p data-vmark="269c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 14 日消息，微软近日在博客平台宣布，<span class="accentTextColor">将停止对旧版 Visual Studio 的支持</span>，官方建议升级到 Visual Studio 2022。</p><p data-vmark="c577" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/e592d101-a5cc-4827-b9e3-fb9accffbe80.png" w="1440" h="436" title="微软：旧版 Visual Studio 即将停止支持，2012/2017/2019 都受影响" width="1440" height="248" referrerpolicy="no-referrer"></p><p data-vmark="f5e3">据介绍，多个版本的  Visual Studio 将在不久后停止支持，包含：</p><ul class=" list-paddingleft-2"><li><p data-vmark="bbf2"><strong>Visual Studio 2012：</strong>对 IDE 及其相关产品、运行时和组件的支持将于 2023 年 1 月 9 日结束，微软建议用户升级到较新版本的 Visual Studio。</p></li><li><p data-vmark="4414"><strong>Visual Studio 2017：</strong>主流支持将于 2022 年 4 月 12 日结束，产品将过渡到扩展支持，直到 2027 年 4 月。在扩展支持期间，微软将仅针对安全问题提供修复，建议用户迁移到受支持的 15.9 版本。</p></li><li><p data-vmark="f59f"><strong>Visual Studio 2019 版本 16.7：</strong>支持将于 2022 年 4 月 12 日结束，微软建议用户迁移到版本 16.11 或 Visual Studio 2022。</p></li><li><p data-vmark="20b3"><strong>Visual Studio 2019 预览频道：</strong>2022 年 4 月之后，微软将不再为 Visual Studio 2019 的预览频道提供更新，建议用户迁移到 Visual Studio 2019 发布频道或 Visual Studio 2022 预览频道，以确保安全并接收最新的功能更新。</p></li></ul><p data-vmark="2a76" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/dd87d9a6-9115-405a-98e2-d618ad959780.png" w="971" h="520" title="微软：旧版 Visual Studio 即将停止支持，2012/2017/2019 都受影响" width="971" height="439" referrerpolicy="no-referrer"></p><p data-vmark="3c7f">IT之家了解到，微软 Visual Studio 2022 已于去年发布正式版，终于升级到了 64 位，可以更好地利用大内存设备进行开发。</p><p data-vmark="8414">Visual Studio 2022 还包括对 <a href="https://devblogs.microsoft.com/dotnet/announcing-net-6/" target="_blank">.NET 6</a> 的支持，Windows 和 Mac 开发人员都可以使用它来构建 Web、客户端和移动应用程序，并为开发 Azure 应用程序提供了更好的支持。</p><p data-vmark="857d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2021/4/508d08f1-5c8d-4790-bd7a-c8ef84cb1a6a.png" title="微软：旧版 Visual Studio 即将停止支持，2012/2017/2019 都受影响" referrerpolicy="no-referrer"></p>
