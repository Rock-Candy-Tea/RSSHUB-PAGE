<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 12 日消息 WinUI 是微软用于构建 Windows 应用程序的最先进的用户界面技术。在微软 Build 2020 开发者大会上，WinUI 团队正式公布了 <span class="accentTextColor">WinUI 3</span>，并发布了 WinUI 3 的预览版。</p><p><img src="https://img.ithome.com/newsuploadfiles/2020/11/20201118_233518_261.png" title="微软 Win11 将重点优化 Win32 应用，为其带来现代化界面" referrerpolicy="no-referrer"></p><p>WinUI 3 是适用于 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a> 和 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 的新用户界面，它继承了 WinUI 和 UWP XAML 的传统，提供了最新的图形功能和 <span class="accentTextColor">Fluent Design 样式</span>，这些样式包含了当今的现代设备、硬件和输入。</p><p><img src="https://img.ithome.com/newsuploadfiles/2020/5/20200521_150149_474.png" title="微软 Win11 将重点优化 Win32 应用，为其带来现代化界面" referrerpolicy="no-referrer"></p><p>微软当时表示，WinUI 3 将支持 Win32 应用和 UWP 应用。</p><p>然而，近日微软宣布，暂时不推出 UWP 的 WinUI 3 稳定版，而将<span class="accentTextColor">重点放在 Win32 应用的界面改进</span>。</p><p>IT之家了解到，根据微软官方的说法，他们将专注于“让 WinUl 3 为 Win32 开发人员工作”。作为新战略的一部分，微软表示将接触庞大的 Win32 社区，<span class="accentTextColor">说服开发人员为 Win11 更新他们的应用程序</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/b9fa1c6e-5f9f-434a-bb05-9048f90df6fb.jpg@s_2,w_820,h_460" alt="WinUI roadmap" w="902" h="506" title="微软 Win11 将重点优化 Win32 应用，为其带来现代化界面" srcset="https://img.ithome.com/newsuploadfiles/2021/7/b9fa1c6e-5f9f-434a-bb05-9048f90df6fb.jpg 2x" width="902" height="460" referrerpolicy="no-referrer"></p><p>对于 UWP 开发人员，WinUI 项目经理 Ryan Demopoulos 表示：并不是 WinUI 3 永远不支持 UWP，而是目前<span class="accentTextColor">暂时没有计划</span>。</p><p>这对 Win32 开发人员来说是一个好消息，但对 UWP 开发人员来说，WinUI 2 并不支持 Chromium 的 WebView2 控件或 NET 5.0，所以开发上可能会有些限制。</p><p>对用户来说，一大批老旧的 Win32 应用将披上现代化的 UI，使用体验上将有质的飞跃。</p>
