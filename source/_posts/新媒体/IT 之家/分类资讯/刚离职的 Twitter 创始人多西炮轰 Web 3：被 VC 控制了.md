<p data-vmark="e30c">北京时间 12 月 21 日下午消息，据报道，刚刚辞去 Twitter CEO 一职的杰克・多西（Jack Dorsey）今日表达了他对所谓的“Web 3”技术，以及安德森・霍洛维茨（Andreessen Horowitz）等风险投资公司参与其中的不满。</p><p data-vmark="08cb"><span class="accentTextColor">Web 3 指的是基于区块链技术的去中心化系统和技术，旨在取代当前的互联网（Web 2）</span>。今年，Web 3 获得了许多关注，也赢得了不少投资，投资者包括安德森・霍洛维茨（Andreessen Horowitz）等。在以太坊（Etherum）和 Solana 区块链上交易 NFT（非同质化代币）就是最明显表现。此外，还有许多公司正为这些平台开发去中心化应用程序和游戏。</p><p data-vmark="5e5e">而多西今日则在 Twitter 上称：“你们并不拥有 Web 3，风投（VC）和他们的有限合伙人拥有。Web 3 永远不会逃脱他们的激励。”（You don't own ‘web3’。The VCs and their LPs do. It will never escape their incentives.）</p><p data-vmark="704c">多西的这篇推文获得了超过 1.6 万个赞和数千次转发，<span class="accentTextColor">许多人用“高度不同意”和“大错特错”之类的评论予以回击，但也有许多人表示支持。</span></p><p data-vmark="4433">特斯拉 CEO 埃隆・马斯克（Elon Musk）也加入其中，询问是否有人看过 Web 3？而多西回答说，“它介于 a 和 z 之间”，<span class="accentTextColor">暗示 Web 3 是由硅谷知名投资者马克・安德森（Marc Andreessen）和本・霍洛维茨（Ben Horowitz）创立的风险投资公司“a16z”控制的。</span></p><p data-vmark="6140">多西的这一系列推文和回复，引发了人们对 Web 3 的分歧和辩论。对此，a16z 的合伙人克里斯・迪克森（Chris Dixon）则向多西伸出了橄榄枝。迪克森说：“我是多西的超级粉丝，希望我们最终能让他接受以太坊和其他区块链。”</p><p data-vmark="11f1">迪克森还称，在 Web 3 中，所有代码、数据和所有权都是开源的，风投（包括 a16z）拥有的很少。</p><p data-vmark="3d55">而多西回应说：“重要的是，我们要将精力集中在真正安全和有弹性的技术上，这些技术要属于大众，而不是个人或机构。”</p><p data-vmark="10a5">11 月底，多西刚刚宣布辞去 Twitter CEO 一职，由 Twitter 首席技术官（CTO）帕拉格・阿格拉瓦尔（Parag Agrawal ）继任。</p>
