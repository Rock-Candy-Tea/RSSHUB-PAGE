<p>深圳 RISC-V 创企睿思芯科已完成数千万美元 A + 轮融资，<span class="accentTextColor">持续加码自主可控的高端 CPU 研发</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/bf48a05b-581d-4e75-b144-7077e9f657c3.jpg@s_2,w_820,h_349" w="900" h="383" title="RISC-V 创企睿思芯科完成数千万美元融资：加码高端 CPU 研发" srcset="https://img.ithome.com/newsuploadfiles/2021/9/bf48a05b-581d-4e75-b144-7077e9f657c3.jpg 2x" width="900" height="349" referrerpolicy="no-referrer"></p><p>▲ RISC-V 已经不再只是 IoT 处理器的候选者</p><p>本轮融资由<span class="accentTextColor">字节跳动、高瓴创投领投</span>，联想创投、双湖资本、水木投资集团、真格基金等跟投，北极光创投等老股东持续跟投，义柏资本担任独家财务顾问。</p><p>RISC-V，这个被视作<span class="accentTextColor">有望与 x86、Arm 三分 CPU 天下的新兴指令集架构</span>，今年以来因科技巨头的不断加码正显得冲劲十足。</p><p>先是“女版巴菲特”凯瑟琳・伍德在其 1 月发布的 2021 年技术趋势报告中，<span class="accentTextColor">判断 Arm、RISC-V、GPU 将成为主流的处理器</span>。随后到年中，英特尔被爆料提出超过以 20 亿美元的交易价格收购明星 RISC-V 创企 SiFive。近日，苹果公司也开始在官网张贴 RISC-V 高性能工程师的招聘岗位。</p><p>在睿思芯科本轮融资的投资阵容中，我们亦看到字节跳动、联想等科技巨头的身影。</p><p>为何多家科技巨头盯上 RISC-V？包括苹果招聘 RISC-V 工程师，这些巨头的新举动透露哪些信号？睿思芯科创始人兼 CEO 谭章熹围绕睿思芯科的布局与 RISC-V 的最新发展趋势分享了许多见解。</p><p>谭章熹透露道，<span class="accentTextColor">目前睿思芯科正与一家国际芯片巨头企业联合打造未来高端智能可穿戴芯片</span>，从定义指令集出发，提供了基于 RISC-V 先进的微架构芯片解决方案，<span class="accentTextColor">其计算性能/能效远高于业界现有基于 Arm 的 DSP/AI 解决方案</span>。这一芯片将在近期进行大规模量产。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/4ac3c7c7-5284-4c6f-b328-4b4578602d64.jpg@s_2,w_820,h_461" w="1000" h="562" title="RISC-V 创企睿思芯科完成数千万美元融资：加码高端 CPU 研发" srcset="https://img.ithome.com/newsuploadfiles/2021/9/4ac3c7c7-5284-4c6f-b328-4b4578602d64.jpg 2x" width="1000" height="461" referrerpolicy="no-referrer"></p><p>▲ 睿思芯科创始人兼 CEO 谭章熹</p><h2>缘起加州伯克利，让 CPU 设计不再是少数人的游戏</h2><p>创办睿思芯科的谭章熹，是 RISC-V“嫡系”出身，也是“接触 RISC-V 最早且距离最近的中国人”。</p><p>谭章熹是在清华园中长大的，父母都是清华大学电子与计算机教授，他的本科和硕士研究生也分别就读于清华电子工程系和清华计算机系，后赴加州大学伯克利分校攻读 EECS 博士，与第五代精简指令集 RISC-V 的缘分便始于此。</p><p>读博士期间，谭章熹师从 2017 年图灵奖得主、知名计算机体系结构宗师、RISC-V 奠基人 David Patterson 教授，是 RISC-V 原创项目组成员。可以说，RISC-V 是谭章熹看着成长起来的。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/f93e129b-c991-4d1f-8bcf-64a08329bd9f.jpg@s_2,w_820,h_494" w="1000" h="602" title="RISC-V 创企睿思芯科完成数千万美元融资：加码高端 CPU 研发" srcset="https://img.ithome.com/newsuploadfiles/2021/9/f93e129b-c991-4d1f-8bcf-64a08329bd9f.jpg 2x" width="1000" height="494" referrerpolicy="no-referrer"></p><p>▲ David Patterson</p><p>从加州大学伯克利分校毕业后，谭章熹先是在美国创办了一家激光雷达芯片公司 OURS，到 2018 年又选择回国创业，在深圳成立主攻 RISC-V 芯片研发的睿思芯科。今年年初，OURS 被美国自动驾驶企业 Aurora 以 1 亿美元价格收购。</p><p>也是在 2018 年，RISC-V 中国联盟正式成立。国内外多家芯片公司开始公开布局这一领域。<span class="accentTextColor">尤其在国内，</span><span class="accentTextColor">RISC-V 被视作助力中国企业在 CPU 领域弯道超车的一大机遇</span>。</p><p>不同于存在授权限制的 x86 和 Arm，<span class="accentTextColor">RISC-V 具有精简、开源、中立、反应速度快等特性</span>，可开发更适应特定需求的独特芯片，而且不会受到美国管制的影响。</p><p>尤其对于有定制化需求并有庞大应用规模的厂商来说，RISC-V 芯片带来的高效能优势还有助于节省成本，转换成经济上的收益。</p><p>谭章熹认为，RISC-V 具有 Arm 不可比拟的优势，绝非 Arm 的廉价替代，甚至在一些高端垂直领域，由于定制化、模块化等特性带来的强大产品优化和创新能力，RISC-V 很可能全面替代 Arm。</p><p>通过架构自由创新，基于 RISC-V 设计的芯片可以在芯片层面结合子集的算法和应用需求，定制专属场景最优化的命令，从而达到针对特定场景低能耗、高性能的效果。</p><p>“RISC-V 是实现国产高端 CPU 自主可控的唯一路径。”谭章熹说。</p><h2>研发完全本地化，向高端芯片进发</h2><p>“<span class="accentTextColor">我们的研发是绝对的本地化</span>，”谭章熹谈道，睿思芯科的主要研发、制造、销售都围绕中国市场开展。</p><p>创立数年，睿思芯科已明确针对高端场景开发芯片的目标，基于 RISC-V 架构开发了高端标量与矢量处理器核，同时通过自主开发芯片产品、合作开发及提供技术服务等多种方式完成商业落地，其产品已经应用在智能穿戴、边缘计算、传感器融合和 AI 加速等领域。</p><p>目前，睿思芯科已自主设计了超低功耗 AI 处理器 IP 核、超低功耗 32 位处理器、高能效 64 位多核异构 AI 处理器等先进 RISC-V 芯片方案，并配以完善的端到端开发工具平台，方便用户快速部署应用。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/11b11966-53f1-4be2-ad84-0233c8865f97.jpg@s_2,w_820,h_615" w="1000" h="750" title="RISC-V 创企睿思芯科完成数千万美元融资：加码高端 CPU 研发" srcset="https://img.ithome.com/newsuploadfiles/2021/9/11b11966-53f1-4be2-ad84-0233c8865f97.jpg 2x" width="1000" height="615" referrerpolicy="no-referrer"></p><p>▲ 高能效 64 位多核异构 AI 处理器 Pygmy</p><p>在落地方面，睿思芯科同多家国际芯片巨头达成合作，并成为大客户主流产品核心供应商，<span class="accentTextColor">也是中国目前唯一在核心高性能处理器/DSP 领域与国际大厂合作的 RISC-V 厂商</span>。</p><p>此次 A + 轮融资，睿思芯科计划将其用于继续提升其优势，包括进一步扩大人才池，以及进行中高端线 RISC-V 芯片产品研发及落地。</p><p>联想集团高级副总裁、联想创投总裁贺志强表示：“睿思芯科是目前中国市场上，少有的拥有全栈、端到端能力的芯片公司，其产品应用在多个智能计算场景中。联想在智能计算领域积累了丰富的产品经验和客户服务经验，未来将在技术研发、人才对接、商业化落地等方面提供助力。”</p><p>睿思芯科 A + 轮融资的领投方之一高瓴创投的执行董事吾雪飞认为，睿思芯科团队在中国乃至世界范围内都具有强大的技术优势，再加上睿思芯科已与众多芯片头部厂商合作落地，高瓴创投非常看好他们的技术实力与市场前景。</p><h2>RISC-V 逐渐走向服务器等主流计算场景</h2><p>睿思芯科的发展方向，也折射出 RISC-V 领域的发展进程 —— 向着更先进的制造工艺、更强劲的性能、更高端的应用演进。</p><p>近年来，RISC-V社区软硬件多方面不断突破，不仅算力持续提升，软件、编译器等生态组件也逐渐成熟。如今，RISC-V 的布局正从中低端芯片逐渐延展向服务器、PC 等高端芯片领域。</p><p>谭章熹认为，RISC-V 如果真正取得成功，还要在主流的计算领域甚至是通用计算，包括数据中心、自动驾驶等高端场景有所突破。睿思芯科也在战略布局这些领域。</p><p>睿思芯科此次投资方中有字节跳动、联想等多家科技巨头，足见下游厂商正赋予 RISC-V 更多的关注和期待。</p><p>在业界，<span class="accentTextColor">苹果最近开始招聘 RISC-V 工程师，计划用在一些比较主力的计算场景</span>，以及对 RISC-V 向量进行一些扩展，和对 RISC-V 可定制化的深入探索和产品研发。</p><p>国际市场分析机构 Semico Research 预测，2025 年全球市场的 RISC-V CPU 核心数将达到 624 亿颗，而中国将拥有全球最大的市场空间；另一家市场调研机构 Tractica 预测，RISC-V 的 IP 和软件工具市场将在 2025 年达到 10.7 亿美元。</p><p>此外，多家科技巨头均预言，RISC-V 将闯入数据中心。ARK Big Ideas 系列年度报告对 RISC-V 未来在云计算领域的发展亦给出了积极的预测。</p><p>该报告预计 Arm/RISC-V 组合所占服务器市场份额，将从 2020 年的 0% 增至 2030 年的 71%，并有可能在云业务领域取代英特尔 x86。到 2030 年，它们的合计市场规模将以每年 45% 的速度增长，CPU 收入有望达到 190 亿美元、服务器收入达到 1000 亿美元。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/20dd6306-56e8-4643-831c-e36be51a67fa.png" w="1000" h="263" title="RISC-V 创企睿思芯科完成数千万美元融资：加码高端 CPU 研发" width="1000" height="216" referrerpolicy="no-referrer"></p><p>▲ 2000-2030 年数据中心服务器 CPU 架构分布及市场规模变化与预测</p><p>RISC-V 国际基金会在近日表示，未来，RISC-V 指令集架构可以作为处理器和加速器的底层指令集，为最强大的计算机提供动力。</p><p>“我认为 RISC-V 已经不再只是 IoT 处理器的候选者了，”谭章熹相信，在通用计算、有差异化的计算场景下 RISC-V 芯片将越来越多，围绕 RISC-V 的定制化扩展，也应是未来的一个发展方向。</p><h2>结语：高层次人才缺口仍待填补</h2><p>谈及当前国内 RISC-V 领域发展面临的挑战，谭章熹认为其中的一大难点是人才稀缺。</p><p><span class="accentTextColor">中国半导体的人才储备与创新程度远远不够</span>，目前做合作集成的公司较多，但真正能够做到底层核心架构技术研发、做出差异化产品的研究团队和企业屈指可数。而解决国内产业需求与人才供给不平衡的问题，依然任重而道远。</p><p>背靠清华-Berkeley 的睿思芯科，拥有从 RISC-V 国际基金会、国内外名校到半导体及科技巨头公司等深厚的产学界资源。同时，它也是清华-Berkeley RIOS 实验室（RISC-V 国际开源实验室）的紧密合作伙伴，正与 RIOS 实验室一起在培养更多的半导体高级人才方面贡献一份力量。</p>
