<p data-vmark="7a02">北京时间 8 月 15 日晚间消息，据报道，游戏软件平台 Unity Software 今日拒绝了游戏软件公司 AppLovin 175.4 亿美元的收购要约。同时，Unity Software 还表示，将继续执行之前所宣布的“收购移动应用货币化软件公司 ironSource”交易。</p><p data-vmark="de04"><img src="https://img.ithome.com/newsuploadfiles/2022/8/1fca4ba4-a82f-4edd-a85e-789bab475e1f.jpg" w="630" h="378" title="175 亿美元看不上，Unity 拒绝 AppLovin 收购要约" width="630" height="378" referrerpolicy="no-referrer"></p><p data-vmark="c9d3">当前，Unity Software 提供了一个全球领先的游戏软件平台，用于创建和运营交互式、实时 3D 内容。而 AppLovin 为游戏开发者提供必要的服务和工具，允许开发者获取新用户，获得游戏变现收入，并利用分析工具优化应用商店排名。</p><p data-vmark="2dec">上周三，AppLovin 向 Unity Software 发出收购要约，提议以 175.4 亿美元的全股票交易收购 Unity Software。交易完成后，Unity Software 将持有新公司流通股的 55%，代表 49% 的投票权。</p><p data-vmark="664f">Unity Software 当时表示，公司董事会将评估 AppLovin 的收购邀约。而今日，Unity Software 拒绝了 AppLovin 的提议。</p><p data-vmark="44c3">Unity Software 称，AppLovin 的提议不符合公司股东的最佳利益。而且，该提议还将阻止 Unity Software 将来会接到一个“更优越的报价”。</p>
