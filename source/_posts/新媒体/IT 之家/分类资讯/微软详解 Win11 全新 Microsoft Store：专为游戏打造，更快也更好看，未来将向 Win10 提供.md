正式上市，而符合条件的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 电脑也可逐步获得升级推送。</p><p data-vmark="9621"><img data-vmark="7640">也就是说，从今天开始，世界各地的 Windows 11 用户都可以访问全新的 Microsoft Store。微软现为大家介绍了有关于 Windows 11 上的新 Microsoft Store 需要了解的 11 件事。</p><h2 data-vmark="67af">适用于 Windows 设备的应用和娱乐</h2><p data-vmark="4e5f"><img src="https://img.ithome.com/newsuploadfiles/2021/10/59466c89-64d7-46f3-9c36-d9554138a44e.gif" w="1440" h="810" title="微软详解 Win11 全新 Microsoft Store：专为游戏打造，更快也更好看，未来将向 Win10 提供" width="1440" height="461" referrerpolicy="no-referrer"></p><h2 data-vmark="88e8">专为游戏打造</h2><p data-vmark="bedc">游戏一直是 Windows 的基础，全新的 Microsoft Store 可带来 PC 生态中的各种游戏。</p><p data-vmark="2684">除了一系列休闲游戏以及《我的世界》等经典游戏外，微软还提供了 Xbox 应用程序的无缝支持，让游戏玩家可以轻松访问社区并选择超过 100 款可通过 Xbox Game Pass 获得的一流 PC 游戏，其中包括 10 月 28 日的帝国时代 IV、极限竞速地平线 5，11 月 9 日和 12 月 8 日的 Halo Infinite。</p><p data-vmark="ad04">此外，Windows 11 用户还可以在 Microsoft Store 内找到第三方店面，例如 Epic Games Store。</p><h2 data-vmark="e079">更快、更便捷、也更好看！</h2><blockquote><p data-vmark="b5ae">我们从头开始重新设计了 Windows 11 上的 Microsoft Store，关注每一个细节，并倾听客户的反馈。</p></blockquote><p data-vmark="3a68">Microsoft Store 现在打开和在页面之间跳转的速度比以往快了 35%。它采用全新、精致的设计，用流畅的动画引导您从一页到下一页，以及微妙的聚光灯效果以为你展示细节。</p><p data-vmark="41c9">微软商店产品详细信息页面也已更新，从而突出显示每位开发人员的独特内容，同时保持跨内容类型的一致性。</p><p data-vmark="89ac"><img src="https://img.ithome.com/newsuploadfiles/2021/10/5c354893-231f-410c-8980-025f114ca950.gif" w="1399" h="816" title="微软详解 Win11 全新 Microsoft Store：专为游戏打造，更快也更好看，未来将向 Win10 提供" width="1399" height="478" referrerpolicy="no-referrer"></p><h2 data-vmark="b091">新的搜索体验</h2><p data-vmark="6425">微软正在改进搜索功能，以确保您可以在 Windows 上的 Microsoft Store 中获得最佳体验。</p><p data-vmark="b1bd">这包括诸如新的搜索结果页面之类的功能，该页面将您搜索的内容放在前面和居中。微软还让过滤搜索结果变得更加容易，例如，通过设置年龄分级或按类别过滤，而且微软表示还有更多计划，但现在不能说，将在未来几个月内与大家分享更多改进。</p><p data-vmark="d739"><img OTT 流媒体服务，旨在对标 Netflix、亚马逊等流媒体。适用于 Windows 的 Disney+ PC 应用程序现已在受支持的地区推出（中国没有）。</p><p data-vmark="f53f">Disney+ 是播放迪士尼、皮克斯、漫威、星球大战、国家地理等内容的必备工具。在部分国际市场，它还包括新的通用娱乐内容品牌 Star，提供原创剧集和特辑、最新电影、经典冒险、纪录片、怀旧电视节目等等。</p><h2 data-vmark="fabc"> 娱乐小工具<br></h2><p data-vmark="10bf">Windows 11 包含一些出色的小部件，其中包括一款新的娱乐小部件，可让您随时了解热搜看点和热门的新游戏。这也是 Windows Insiders 互动最多的小部件之一，微软计划在接下来的几个月中添加更多内容。</p><p data-vmark="2df5"><img src="https://img.ithome.com/newsuploadfiles/2021/10/5b5ad5fa-f4b1-432c-89d9-5797b7defbd9.jpg@s_2,w_820,h_461" w="1024" h="576" title="微软详解 referrerpolicy="no-referrer"></p><h2 data-vmark="0094">帮助您发现最佳应用和游戏的编辑故事</h2><p data-vmark="5b5e">Windows 上的 Microsoft Store 不仅是安装您最喜爱的应用程序的地方，还是一个了解您可能还不知道的新体验的地方。</p><p data-vmark="84e4">从本月开始，您将看到精选的故事和应用程序集，以帮助您充分利用 Windows PC，并让您了解 Windows 应用程序和游戏的最新趋势和发展。</p><h2 data-vmark="6ab2">快速安装</h2><p data-vmark="f9e4">Microsoft Store 的主要目标是始终让用户能够轻松方便地安装最好的 Windows 软件 —— 即使您不使用 Microsoft Store 应用程序本身。</p><p data-vmark="c617">如果您直接从网站安装应用程序，微软提供了新的弹出式商店。当您单击网页上新的“从 Microsoft 获取”时，会弹出一个“迷你”版本的微软商店来代理安装，因此您知道您正在安装来自受信任服务的软件。</p><p data-vmark="b63f"><img src="https://img.ithome.com/newsuploadfiles/2021/10/b18238c6-b9ae-44e3-97a9-efb5142ee9b3.png" w="864" h="312" title="微软详解 Win11 全新 Microsoft Store：专为游戏打造，更快也更好看，未来将向 Win10 提供" width="864" height="296" referrerpolicy="no-referrer"></p><p data-vmark="fdfc"><img src="https://img.ithome.com/newsuploadfiles/2021/10/df271bef-dc56-475f-8d04-e7425675e144.jpg" w="300" h="166" title="微软详解 Win11 全新 Microsoft Store：专为游戏打造，更快也更好看，未来将向 Win10 提供" width="300" height="166" referrerpolicy="no-referrer"></p><h2 data-vmark="94a8"> 集成 Windows 工作环境</h2><p data-vmark="1ff2">Windows 11 中的众多改进之一是它如何让您与来自 Microsoft Store 的相关应用和体验保持联系。</p><p data-vmark="ba66">新的“入门”应用程序有助于提升当前合适的应用程序，包括来自商店的建议资源。Windows 11 还可以将您与有助于特定工作流程的应用程序联系起来 —— 例如，如果您正在照片应用程序中查看照片，您现在只需单击一下即可轻松跳转到更强的图像编辑器中。如果您希望对桌面的外观和感觉进行深度个性化，Windows 的设置会将您连接到 Microsoft Store，以便为您的设备浏览和安装自定义主题。</p><p data-vmark="82e4"><img src="https://img.ithome.com/newsuploadfiles/2021/10/666f45d4-f8e3-4443-9b38-0d097d1d9b6a.png" w="1024" h="587" title="微软详解 Win11 全新 Microsoft Store：专为游戏打造，更快也更好看，未来将向 Win10 提供" width="1024" height="470" referrerpolicy="no-referrer"></p><p data-vmark="21e8">▲ 从 Microsoft 照片快速跳转到 Store 中的照片编辑应用程序，例如 PicsArt。</p><h2 data-vmark="cfd6">即将上线 Windows 10</h2><p data-vmark="5a6c">IT之家知悉，新版 Microsoft Store 现已在 Windows 11 上提供，微软还表示它也将在未来几个月内向 Windows 10 客户提供，后续公布更多细节。</p><h2 data-vmark="fd6e">向开发者开放，听取用户意见</h2><p data-vmark="03b3">微软表示他们已经向使用不同类型框架、打包技术和商业平台的开发人员开放了 Windows 上的 Microsoft Store。</p><p data-vmark="47cd">例如，当应用程序管理自己的支付系统时，Windows 上的 Microsoft Store 不再要求应用程序开发人员与 Microsoft 分享收入。</p><p data-vmark="f985">如果您是 Windows 开发人员，您可以将您的应用发布到新的 Microsoft Store，而如果您的应用已经在商店中，您可以将快速安装所需的新 logo 和弹出式商店添加到网站中。</p>
