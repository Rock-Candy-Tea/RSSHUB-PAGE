显卡芯片明年上市：主攻高性能图形渲染" width="900" height="349" referrerpolicy="no-referrer"></p><p data-vmark="a701">芯东西 2 月 23 日报道，今日早间，南京 GPU 创企砺算科技宣布完成数亿元天使轮融资。</p><p data-vmark="23bc">达泰资本、将门创投、万物创投、海松资本、协立创投等机构参与了本轮融资，融资资金被用于其首款 GPU 产品开发及团队扩张。</p><p data-vmark="69db">砺算科技成立于 2021 年 8 月，是国内 GPU 创业热潮中最年轻的生力军之一。但与多数聚焦 GPGPU 的创企不同，它要做“唯一一家可实现从端 + 云 + 边的高性能图形渲染的中国公司”。</p><p data-vmark="db40">“唯一”，从何而谈？砺算科技联合创始人、联席 CEO 宣以方告诉芯东西，据他观察，当前国内还没有第二家企业，真正有实力实现基于自研架构、全自有知识产权（IP）的高性能图形渲染 GPU 产品。</p><p data-vmark="5ede">这家成立仅半年的创企目标明确，计划在 2023 年将国内第一颗 6nm GPU 推向市场，目标是“性能对标国际一线企业产品”，也就是英伟达安培架构 RTX 系列 GPU。</p><p data-vmark="baaf">“我们跟行业客户讨论时，他们会很有信心地说，你们做出来，我们就用。”客户的认可令联合创始人、联席 CEO 孔德海深感自豪。首次公布融资之际，芯东西同砺算科技的几位核心创始成员进行了深入交流。</p><p style="text-align: center;" data-vmark="cafa"><img src="https://img.ithome.com/newsuploadfiles/2022/2/feac65c0-947a-493b-be55-2a0a5c7cbdfe.jpg@s_2,w_820,h_462" w="1080" h="608" title="砺算科技完成数亿元天使轮融资，自研 6nm TrueGPU center;" data-vmark="41f0">▲ 砺算科技创始团队（从左至右：孔德海、宣以方、牛一心）</p><h3 data-vmark="b380">01. 近 30 年 GPU 研发经验，创业是水到渠成</h3><p data-vmark="41f9">创办 GPU 公司，对于砺算科技创始团队而言，是一件水到渠成的事。</p><p data-vmark="4f75">砺算公司创始人均曾在老牌硅谷 GPU 公司 S3 沉淀了深厚的技术能力及工程经验。</p><p data-vmark="ae5a">在 20 世纪 90 年代，刚刚诞生的 GPU 领域呈现群雄混战之景，1989 年成立的 S3 即是早期的头部 GPU 玩家之一。它成立 18 个月后推出的 86C911 产品，正式开启了 2D 图形硬件加速时代。</p><p data-vmark="3a8a">砺算科技联合创始人、联席 CEO 宣以方，曾是 1992 年 S3 公司 GPU 创始团队成员，也是年销售 2000 万颗的 Trio 图形芯片及第二代 3D 显示芯片 Savage 野人的负责人。</p><p style="text-align: center;" data-vmark="6cdf"><img src="https://img.ithome.com/newsuploadfiles/2022/2/860c7e43-f81b-40cd-a048-71f6bae71331.jpg@s_2,w_820,h_443" w="1000" h="540" title="砺算科技完成数亿元天使轮融资，自研 6nm TrueGPU 显卡芯片明年上市：主攻高性能图形渲染" data-vmark="f101">▲ S3 Trio64V + 显卡</p><p data-vmark="57d7">Trio 芯片是 S3 第一个辉煌的缔造者，由它构成的图形加速卡一度风靡市场。在鼎盛时期，S3 公司曾占据 90% 以上的 GPU 芯片市场，并给 DirectX 贡献了许多技术标准。</p><p data-vmark="0928">宣以方毕业于中国台湾交通大学，拥有 28 年 GPU 研发经验，曾领导量产 15 代 GPU 芯片。此前 22 年，他一直担任高级副总裁或副总裁，领导 GPU 开发，包括架构、ASIC、软件、系统、量产、市场等。</p><p style="text-align: center;" data-vmark="d2ce"><img src="https://img.ithome.com/newsuploadfiles/2022/2/78030829-f055-4ef5-866e-85d60e1f9972.png" w="1000" h="491" title="砺算科技完成数亿元天使轮融资，自研 6nm TrueGPU 显卡芯片明年上市：主攻高性能图形渲染" width="1000" height="403" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="1d8d">▲ S3 Savage3D 显卡</p><p data-vmark="631f">1984 年，孔德海考入清华大学无线电系本科专业，从 1992 年起从事 GPU 芯片研发，是中国第一代超大规模集成电路（VLSI）设计师。</p><p data-vmark="0810">他在硅谷有 20 多年芯片研发、管理、投资经验，曾在中美 GPU 公司任管理职务，还参与创立 3 个硅谷天使基金。</p><p data-vmark="06ea">孔德海参与过 4 家初创公司，其中 1 家在纳斯达克上市，另外 3 家被上市公司并购，也曾任中国第二家纳斯达克上市芯片公司炬力集成（上海）研发副总经理，并担任数家芯片创企的顾问或董事。</p><p data-vmark="bf75">砺算科技 CTO 牛一心在 1994 年加入 S3，是首个 S3D 引擎的研发者，也是全球第一代 3D 加速 GPU 芯片 ViRGE（总销售超 1 亿颗）的负责人，此前 18 年担任 GPU 研发副总经理，带领 GPU 硬件设计，支持 DirectX、OpenGL、OpenCL 全部标准。</p><p style="text-align: center;" data-vmark="48fc"><img src="https://img.ithome.com/newsuploadfiles/2022/2/1b333bef-f2a2-4233-9e37-2ce2dd802ddf.jpg@s_2,w_820,h_364" w="1080" h="479" title="砺算科技完成数亿元天使轮融资，自研 6nm TrueGPU 显卡芯片明年上市：主攻高性能图形渲染" srcset="https://img.ithome.com/newsuploadfiles/2022/2/1b333bef-f2a2-4233-9e37-2ce2dd802ddf.jpg 2x" width="1080" height="364" referrerpolicy="no-referrer"></p><p style="text-align: center;" data-vmark="1027">▲ S3 ViRGE 显卡</p><p data-vmark="6b1c">当图形技术进入 3D 主导时代，曾经显赫一时的 S3，开始跟不上技术发展节奏，在市场竞争中渐趋下风。世纪之交，英伟达凭借革命性的 GeForce 256 图形处理芯片大放异彩，经过激烈的竞争缠斗，将 S3、3Dfx 等昔日霸主甩在身后。</p><p data-vmark="e44f">S3 则几番易主，先是 2000 年被威盛电子（VIA）以 3.22 亿美元收购，后于 2011 年被 HTC 以 3 亿美元买下。如今 S3 的市场地位早已不复当年，但若论人才输送，S3 可以说是培育 GPU 专业人才的核心基地之一，不少国内外 GPU 玩家的技术带头人，都曾在 S3 从事过 GPU 研发。</p><p data-vmark="efac">正是因为经历多代 S3 经典产品从设计到落地的磨炼，砺算科技创始团队掌握了从 0 到 1 做出大规模销售的高性能 GPU 的经验，以及国内少有的大芯片研发、量产的经验。</p><p data-vmark="9316">在嗅到市场需求后，三位共事十余载的芯片老兵一拍即合，回到国内，打造真正自主研发的 GPU！</p><h3 data-vmark="640d">02. 主攻高性能图形渲染，不做 AI 加速 GPGPU</h3><p data-vmark="c68e">砺算科技的 Slogan 是“砥砺算心，不负时代”，这也是该公司名字的由来。</p><p data-vmark="8305">“宝剑锋从磨砺出。”孔德海说，创始团队相信做芯片是一个长期的事，不是砸了几十亿，就能一定做出一个优秀的芯片。他们的长期愿景是持续静下心来认真打磨产品，做出一系列锋利的“宝剑”。</p><p data-vmark="cbf1">自主研发的 TrueGPU，就是砺算科技正在打磨的核心利器。</p><p data-vmark="830b">目前，砺算科技正在开发基于原创自研架构盘古架构的 GPU 产品，对标国际一线企业产品。据称，这是国内第一颗 6nm GPU，将于明年正式进入市场。</p><p data-vmark="7390">作为一款专攻图形渲染的芯片，其 GPU 既覆盖桌面 PC、笔记本电脑等端侧设备，也包含汽车等边缘产品，以及能够满足数字内容、建筑设计、智慧城市、数字孪生、云游戏、元宇宙等高性能云端渲染需求。</p><p style="text-align: center;" data-vmark="9962"><img src="https://img.ithome.com/newsuploadfiles/2022/2/3c508562-4650-4ae8-8235-55bb5cafe160.jpg@s_2,w_820,h_373" w="1000" h="455" title="砺算科技完成数亿元天使轮融资，自研 6nm TrueGPU 显卡芯片明年上市：主攻高性能图形渲染" srcset="https://img.ithome.com/newsuploadfiles/2022/2/3c508562-4650-4ae8-8235-55bb5cafe160.jpg 2x" width="1000" height="373" referrerpolicy="no-referrer"></p><p data-vmark="9aa6">如今全球 GPU 市场由英伟达、AMD、英特尔三大巨头主导，近两年，国内 GPU 创业公司涌现近 20 家，不过截至目前，市面上多数创企的 GPGPU 专注于 AI 计算，不具备渲染能力。</p><p data-vmark="4436">那么，主攻渲染的 GPU 与专注 AI 加速的 GPGPU，究竟有哪些区别？</p><p data-vmark="4757">孔德海给的答案可概括为：GPU 是“难做好卖”，而 GPGPU 是“好做难卖”。</p><p data-vmark="ef25">GPU 早期主做图形渲染，2007 年开始走向通用计算，到 2016 年第三次 AI 浪潮兴起，擅长 AI 计算和高性能计算的 GPGPU 逐渐成为数据中心的芯片“宠儿”。</p><p data-vmark="2dbd">砺算科技又为何坚定选择了更难做的图形 GPU 赛道？</p><p data-vmark="a5c5">从技术角度来看，砺算科技的 GPU 在功能上是 GPGPU 的超集，结合了图形渲染视觉计算、科学计算、AI 计算三个模块，管理复杂度和设计难度，都要比仅涉及 AI 计算的 GPGPU 更高。</p><p data-vmark="55c2">孔德海说，GPGPU 的 AI 加速运算是做大量矩阵乘加，运算方式相对单纯，而 GPU 渲染需将物理世界的东西分割成很多小三角形在数字世界重现，涉及几何、像素、颜色、光照、纹理映射、环境映射等，运算复杂，每个功能都要单独实现。“很多人可能有兴趣想做，但是做不出来，并没有这个能力做。”</p><p data-vmark="ec16">从生态角度来看，英伟达花了十几年、投入百亿美金，在 AI 计算、高性能计算领域打造了壁垒森严的 CUDA 软件生态。如果做 GPGPU，则要么兼容 CUDA 生态，要么自建生态，这极具挑战性。</p><p data-vmark="f5b0">而图形渲染 GPU 的生态较为成熟，是以 DirectX、OpenGL、Vulkan 为主的开放、通用的生态，如果能做出高性能的产品，商业化相对容易。</p><p data-vmark="e551">从市场规模来看，孔德海提到图形渲染 GPU 的 2021 年全球数据达 200 亿美元，几乎是 GPGPU 市场规模的 4 倍。到 2027 年，中国 GPU 市场规模预计达到 1800 亿人民币，是 GPGPU 市场的 2 倍。</p><p data-vmark="a0bf">随着自动驾驶、智慧城市、元宇宙中的虚拟数字人、云端游戏等应用不断拓展，图形渲染能力难以被替代的 GPU，预计市场需求将持续增长。</p><p data-vmark="183c">因此，砺算科技选择专攻自研架构、自主 IP，打造能完成复杂渲染任务的高性能 GPU 产品。</p><h3 data-vmark="6a6f">03. 真正全自研：自研架构 + 自有 IP</h3><p data-vmark="83cc">“GPU 是一个硬骨头，没有很大的决心和推动力，很难去做好这个事情。”</p><p data-vmark="7751">全自研，是砺算科技的一大核心优势。联合创始人、CTO 牛一心强调说，砺算科技应该是“国内唯一自研 GPU 架构、自主开发 GPU、自有 GPU 知识产权的团队”。</p><p data-vmark="bb37">GPU 的系统设计复杂而庞大，技术涉及面广，想要从零打造完全自主的 GPU 芯片并非易事。据孔德海观察，目前国内渲染 GPU 公司基本都是从第三方采购 IP 核，几乎没有一家真正做到全自主研发。</p><p data-vmark="077d">购买 IP 核的 GPU 开发方式，可以说是起步容易、兑现难、进阶难。</p><p data-vmark="1441">一方面，嵌入式 IP 性能低，只能覆盖低端市场；另一方面，由于 IP 核已固定，扩展性较差，且仅获授权，自主性较低，生态支持也不完整。</p><p data-vmark="fe18">相比之下，自研 IP 虽然难度大、更具挑战性，却从长远来看拥有更好的发展路径。</p><p data-vmark="f9d2">比如自研 IP 可以打造高性能 GPU，覆盖国内全方位、全生态市场，并且可扩展，可客制、定制，能无限扩展到云游戏、元宇宙应用。</p><p data-vmark="376d">另外，自研 IP 拥有 DirectX、OpenGL、Vulkan 全生态支持，商业化路径也更为顺畅。</p><p data-vmark="7ff6">砺算科技创始团队已深耕 GPU 数十年，积累了方法论、顶层设计、最优化调度、高运算单元利用率等诸多工程经验，能兼顾大芯片量产对设计实力、know-how 和经验传承的要求。</p><p data-vmark="a613">因此，砺算科技不是购买 GPU IP，而是确立了自研架构自有 IP 的路线，自己掌握整体芯片设计。</p><p data-vmark="581a">技术创新方面，砺算科技在国内率先使用 AI Powered GPU 技术，通过 AI 方法多快好省地增强其 GPU 的图形渲染功能，实现性能成倍提升。除了优化渲染功能外，这里的 AI 技术也可以用于视频处理、识别等方面的加速任务。</p><p data-vmark="d775">此外，通过国际领先的线程调度管理技术，砺算科技提高了 GPU 渲染和运算单元的实际利用率。</p><h3 data-vmark="a4b4">04. 两种商业落地模式，发挥本土定制化优势</h3><p data-vmark="4118">在孔德海看来，国内大芯片创业分为三个阶段，第一阶段 2016~2018 年是 AI 芯片，第二阶段 2019~2021 年主要是 GPGPU 芯片，而第三阶段就是主攻图形渲染的 GPU 芯片。</p><p data-vmark="f308">至于选址南京的原因，砺算科技创始团队主要考虑到南京高校资源丰富，电子相关专业学生多、水平高，并且能给予创业公司很好的支持。</p><p data-vmark="a328">尽管去年 8 月才创立，但孔德海认为砺算科技的入局时间并不算晚，反而因为团队技术和工程经验足够丰富，其首款芯片明年进入市场的时间不会输于其他 GPU 企业。</p><p data-vmark="68a4">考虑到芯片人才的稀缺，在创立之初，砺算科技已经搭建了整建制研发和量产团队。</p><p data-vmark="281d">孔德海透露道，过去半年，砺算科技持续推进研发以及与行业伙伴的合作，团队规模已经扩展至数百人，其中研发团队占比超过 90%。</p><p data-vmark="8041">据悉，许多团队成员毕业于加州大学、清华、北大、复旦、上海交大等知名学府的微电子专业和计算机专业，并具有平均 15 年的 GPU 从业经历，包括 S3、泰鼎、高通、英特尔、英伟达、AMD 以及国内 GPU 或 GPGPU 创业公司。</p><p data-vmark="6443">当前，砺算科技有两种商业落地模式：</p><p data-vmark="6c80">一是研发和销售自研 TrueGPU 图形芯片，在端、云、边等多领域应用。</p><p data-vmark="f71e">二是配合客户，定制优化，提供高性能 GPU 板卡和系统，直接解决客户的技术和新产品商业化问题。这也是本土企业贴近市场需求的一大优势。</p><p data-vmark="04cf">至于如何吸引更多人才？宣以方说，国内公司要达到英伟达的高度，必须团队整体朝着同一个目标团结一致，并且从上到下有长期艰苦奋斗的恒心，并谈到砺算科技主要从三个方面来提高团队凝聚力：一是做有意义的事情，二是实现整个工作环境的愉悦性，三是与员工共同创业，给员工的股份在业内是很大方的。</p><h3 data-vmark="86ff">05. 结语：数字经济时代，GPU 成关键算力资源</h3><p data-vmark="e84f">如今，在发展数字经济的主旋律下，不断提升的算力需求已成为拉动 GPU 市场增长的重要引擎。</p><p data-vmark="2bcd">根据市场研究机构 Market Study Report 数据，全球 GPU 市场在 2020 年达到 224.1 亿美元，预计将在 2021-2028 年间以 31.87%
