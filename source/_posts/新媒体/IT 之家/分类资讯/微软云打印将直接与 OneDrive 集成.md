<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 17 日消息 早在 2020 年 3 月，微软就宣布了其基于云的打印解决方案 Universal Print 的私人预览版。它本质上是将 Windows Server 打印功能移至云端，只要用户拥有支持 Universal Print 的打印机，就无需操心其他基础设施甚至打印机驱动程序更新。</p><p>几个月前，该服务正式面向 Microsoft 365 客户提供，现在，微软宣布 <span class="accentTextColor">Universal Print 将很快与 OneDrive 集成</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/8c50be51-0163-4b40-88c9-be6096fbcd8c.jpg" w="760" h="428" title="微软云打印将直接与 OneDrive 集成" width="760" height="428" referrerpolicy="no-referrer"></p><p>IT之家了解到，用户可以<span class="accentTextColor">直接从 OneDrive 打印文档</span>，而无需下载或要求 IT 管理员在其组织管理的设备上安装打印机驱动程序。</p><p>用户只需在选择文档后单击“打印”，选择具有 Universal Print 功能的打印机，即可进行打印。如果用户更愿意使用本地打印机，也可以相应地选择不同的配置。</p><p>微软表示，该功能将于<span class="accentTextColor">本周向企业推出</span>。打印体验在所有设备和<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>中都是通用的，目前支持的文件类型有 <span class="accentTextColor">PDF、XPS、Word、Excel 和 PowerPoint</span>。</p>
