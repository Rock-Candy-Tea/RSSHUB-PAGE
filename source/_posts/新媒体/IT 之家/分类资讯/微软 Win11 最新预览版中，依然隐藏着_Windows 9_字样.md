<p data-vmark="b10e"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 7 日消息，微软 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> 最新预览版已经到了 Build 22567 版本，正在为 22H2 做准备。然而，Twitter 用户 Xeno 偶然发现，Build 22567 中仍然存在的对 <span class="accentTextColor">Windows 9</span> 的引用。</p><p data-vmark="be26" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/9be24735-2e35-46cd-ae8d-1ee5f10e071f.png" w="1440" h="658" title="微软 Win11 最新预览版中，依然隐藏着“Windows 9”字样" width="1440" height="375" referrerpolicy="no-referrer"></p><p data-vmark="fc07">该引用可以在 System32 文件夹下找到，完整路径如下：</p><blockquote><p data-vmark="ecaa">C:\Windows\System32\migwiz\replacementmanifests\CommandPrompt-Win8-Replacement.man</p></blockquote><p data-vmark="ab1b">该文件谈到了从包括 Windows 9 在内的旧 Windows 系统收集数据。文件还提到了 Windows Blue，是 Windows 8.1 的代号。</p><p data-vmark="b815" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/3/1a8e197a-4f97-4fae-89e5-6d7fbbbdf455.png" w="840" h="439" title="微软 Win11 最新预览版中，依然隐藏着“Windows 9”字样" width="840" height="429" referrerpolicy="no-referrer"></p><p data-vmark="e203">IT之家了解到，关于 Windows 9 的表述，自 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 技术预览版 9860 以来就已经存在了，不过该系统从未正式发布。有传言称，Windows 9 由于与 Windows 95 和 Windows 98 混淆，因此微软将新一代系统改名为了 Windows 10。</p>
