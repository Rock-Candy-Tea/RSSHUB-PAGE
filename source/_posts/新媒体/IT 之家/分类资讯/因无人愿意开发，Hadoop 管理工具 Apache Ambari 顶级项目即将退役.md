<p data-vmark="8d01"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 18 日消息，Apache Ambari 是一个基于 Web 的 Apache Hadoop 集群的供应、管理和监控工具，曾是 Apache Software Foundation 的顶级项目。</p><p data-vmark="4cbe"><img src="https://img.ithome.com/newsuploadfiles/2022/1/0523ba40-6bb7-4910-a7c0-cd28ee7041a3.png" w="621" h="450" title="因无人愿意开发，Hadoop 管理工具 Apache Ambari 顶级项目即将退役" width="621" height="450" referrerpolicy="no-referrer"></p><p data-vmark="405f">近日，Apache Ambari 的开发者之一 Jayush Luniya 提议将该项目的开发搁置，原因是<span class="accentTextColor">大部分提交者和 PMC 成员没有积极参与该项目</span>，过去两年来，只发布了 Ambari 2.7.6 这一个版本。</p><p data-vmark="a535">Jayush Luniya 发布了一个 72 小时的投票，如果同意项目退役，就回复 +1。<span class="accentTextColor">目前该投票已获得全票通过</span>。</p><p data-vmark="7ded"><img src="https://img.ithome.com/newsuploadfiles/2022/1/a97dea29-751b-48e1-81d4-4a22c4724900.png" w="734" h="884" title="因无人愿意开发，Hadoop 管理工具 Apache Ambari 顶级项目即将退役" width="734" height="884" referrerpolicy="no-referrer"></p><p data-vmark="8008">IT之家了解到，该项目尚未出现在退役名单中，不过预计很快就将正式退役。</p><p data-vmark="214c"><img src="https://img.ithome.com/newsuploadfiles/2022/1/2efc18d8-afda-46cd-affe-8870ef6651d5.png" w="808" h="569" title="因无人愿意开发，Hadoop 管理工具 Apache Ambari 顶级项目即将退役" width="808" height="569" referrerpolicy="no-referrer"></p><p data-vmark="3029">Ambari 已支持大多数 Hadoop 组件，包括 HDFS、MapReduce、Hive、Pig、 Hbase、Zookeeper、Sqoop 和 Hcatalog 等，其提供了一个直观、易于使用的 Hadoop 管理 Web UI，由其 RESTful API 提供支持。</p>
