<p data-vmark="ae15">集微网消息，5 月 3 日，美国卫星通讯商 DISH Network 正式对外宣布，将采用三星 5G O-RAN 的虚拟化无线接入网络 (vRAN) 与无线电单元 (RU) 设备。其中，RU 将支持大规模 MIMO，以及 DISH 的频分双工 (FDD) 和时分双工 (TDD) 两种模式。</p><p style="text-align: center;" data-vmark="91b0"><img src="https://img.ithome.com/newsuploadfiles/2022/5/48539e60-d85d-4630-9f90-679633c3bbf2.png" w="594" h="402" title="三星正式打入 DISH 卫星通讯商 5G O-RAN 供应链，使用 Galaxy S22 测试网络" width="594" height="402" referrerpolicy="no-referrer"></p><p data-vmark="8ab5">据悉，DISH Network 即将推出基于 O-RAN 架构的 5G 网络服务，DISH Network 首席运营官 John Swieringa 表示，三星的软硬件解决方案将为 DISH Network 在 5G 网络领域的拓展提供多方位助力，是 DISH Network 网络领域拓展中不可或缺的一环，三星在 vRAN、O-RAN 技术上的创新可以帮助 DISH Network 向消费者、企业提供开放、可互相操作的 5G 云端服务。</p><p data-vmark="4d49">除此之外，三<span class="accentTextColor">星还将为 DISH Network 提供 5G 终端设备</span>。目前，DISH Network 正在使用三星 Galaxy S22 测试 5G 网络服务，DISH Network 计划在整个网络布署过程中继续使用三星手机作为测试平台。</p><p data-vmark="f583">事实上，三星并非 DISH vRAN 的独家供应商。据悉，目前除了三星以外，DISH Network 早在 2020 年便与英特尔展开了合作，并采用英特尔 Xeon 可扩充处理器、以太 800 网卡、vRAN 加速器 ACC100 与相关软件服务。</p><p data-vmark="1825">公开资料显示，O-RAN 即开放式无线电接入网 (Open Radio Access Network)，是一个基于 RAN 元素的互操作性和标准化的概念，包括针对不同供应商的白盒硬件和开源软件元素的统一互连标准。O-RAN 架构将模块化基站软件堆栈集成在现成的硬件上，允许来自分立供应商的基带和无线电单元组件无缝地一起运行。</p><p data-vmark="bd7b">O-RAN 通过高效、智能和通用性的共同属性，强调了精简的 5G RAN 性能目标。部署在网络边缘的开放式 RAN 将使 5G 应用受益，例如自动驾驶车辆和物联网，有效支持网络切片用例，并支持安全高效的无线固件升级等。</p>
