<p data-vmark="15fe"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 8 日消息，对于在不同 Web 浏览之间做出决定的消费者来说，性能通常是一个非常重要的指标。微软和谷歌都在不断寻找分别提高 Edge 和 Chrome <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>性能的方法。</p><p data-vmark="1b33">近日，微软推出了一套名为 Microsoft-Performance-Tools for Linux-<a class="s_tag" href="https://android.ithome.com/" target="_blank">Android</a> 的开源工具，用于<span class="accentTextColor">测量各种系统上的浏览器性能</span>。</p><p data-vmark="641a"><img src="https://img.ithome.com/newsuploadfiles/2021/12/8b78a439-fb95-4a38-981f-d8d694a6298f.jpg@s_2,w_820,h_462" w="1430" h="805" title="微软推出安卓 / Linux 平台的浏览器性能测试工具，开源免费使用" srcset="https://img.ithome.com/newsuploadfiles/2021/12/8b78a439-fb95-4a38-981f-d8d694a6298f.jpg 2x" width="1430" height="462" referrerpolicy="no-referrer"></p><p data-vmark="bcaf">微软强调，该工具可用于监视和测量浏览器性能，与过去几年中用于提高 Windows 性能的方法相同，<span class="accentTextColor">它可用于提供有关操作系统和应用程序代码在某个时间点正在执行的操作的更多见解</span>，如果需要，用户还可以将跟踪与 Windows 性能分析器 (WPA) 相关联。</p><p data-vmark="f796">IT之家了解到，该工具是基于 .NET Core 和 microsoft-performance-toolkit-sdk 构建的，这意味着它理论上在 .NET Core 支持的任何操作系统上都受支持。</p><p data-vmark="b51c">对于 Linux 系统，该工具包括 LTTng、perf 和 Perfetto。而在 Chromium 和 Android 环境中，仅支持 Perfetto。每个单独工具支持的跟踪指标如下：</p><ul class=" list-paddingleft-2"><li><p data-vmark="117f"><strong>LTTng：</strong>Linux 内核 CPU 调度、进程、线程、块 IO / 磁盘、系统调用、文件事件等</p></li><li><p data-vmark="fbdf"><strong> perf：</strong>Linux CPU 采样（cpu-clock）</p></li><li><p data-vmark="4456"><strong>Perfetto：</strong>Android 和 Chromium CPU 调度、CPU 采样、CPU 频率、FTrace、Android 日志、通用事件 / 默认轨道、GPU 计数器</p></li></ul><p data-vmark="4177"><strong>Microsoft-Performance-Tools for Linux-Android：</strong></p><p data-vmark="4021"><a href="https://github.com/microsoft/Microsoft-Performance-Tools-Linux-Android" target="_blank">点此查看</a></p>
