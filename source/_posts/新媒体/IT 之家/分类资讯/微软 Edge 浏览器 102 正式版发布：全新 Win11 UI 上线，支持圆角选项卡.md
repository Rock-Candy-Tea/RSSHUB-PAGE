            <p data-vmark="0597"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 3 日消息，微软 Edge 浏览器稳定版（正式版）于近日推出了 <span class="accentTextColor">102 版本</span>，官方更新日志没有公布新内容，然而该版本隐藏了多项重要的 UI 改进。</p><p data-vmark="3004">用户需要在地址栏输入：</p><pre class="brush:javascript;toolbar:false">edge://flags/#edge-visual-rejuv-show-settings</pre><p data-vmark="5303">然后启用该实验选项，接着就可以在浏览器设置中看到两个预览功能，一个是<span class="accentTextColor">在标题栏和工具栏中显示 Windows 11 视觉效果</span>，另一个是<span class="accentTextColor">使用圆角浏览器选项卡</span>。</p><p data-vmark="99f5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/5c1e6592-7dfa-47ea-b30f-5cc2c0468d5b.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzE5NS5wbmc=,g_8,x_0,y_19,a_0,t_100" w="1266" h="229" title="微软 Edge 浏览器 102 正式版发布：全新 Win11 UI 上线，支持圆角选项卡" width="1266" height="148" referrerpolicy="no-referrer"></p><p data-vmark="fbe5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/1bb540c8-7f82-41f0-b8de-a3548b063ff5.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xdy9xdzE1Ny5wbmc=,g_9,x_16,y_16,a_0,t_100" w="1018" h="622" title="微软 Edge 浏览器 102 正式版发布：全新 Win11 UI 上线，支持圆角选项卡" width="1018" height="501" referrerpolicy="no-referrer"></p><p data-vmark="2326" style="text-align: justify;">IT之家了解到，重新设计的 Edge 浏览器 UI 仅适用于默认的黑白主题，由于是预览功能，微软警告用户可能会遇到一些<span class="accentTextColor">性能问题</span>。</p><p data-vmark="5387" style="text-align: justify;">此外，也有网友称更新到 Edge 102 正式版后，<span class="accentTextColor">浏览器打印功能可能出现了问题</span>，可以通过退回到 Edge 101 来解决。</p><p data-vmark="bfb2" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/731cf9cf-5658-4396-ae8b-fa2d3224648e.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzEwOS5wbmc=,g_9,x_11,y_11,a_0,t_100" w="708" h="438" title="微软 Edge 浏览器 102 正式版发布：全新 Win11 UI 上线，支持圆角选项卡" width="708" height="438" referrerpolicy="no-referrer"></p>
