<p data-vmark="5d3c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 12 月 22 日消息，根据 The Loadout 的计算，整个 2021 年添加到 Xbox Game Pass 的所有游戏的总价值达到了惊人的 <span class="accentTextColor">6317.35 美元（约 40241.52 元人民币）</span>。</p><p data-vmark="f098"><img src="https://img.ithome.com/newsuploadfiles/2021/2/20210201_090434_808.jpeg" title="微软“钞能力”：Xbox Game Pass 今年新增游戏价值超 4 万元" referrerpolicy="no-referrer"></p><p data-vmark="3be3">在 The Loadout 的报告中，他们表示他们得出的这个数字是基于 Microsoft Store 上每款游戏的基本价格，因此没有考虑任何销售或折扣。</p><p data-vmark="a199">根据 The Loadout 的报告称，由于 Xbox 对贝塞斯达及其母公司 Zemimax 媒体的收购，<span class="accentTextColor">Xbox Game Pass 在今年“最有价值”的月份是 3 月份</span>，《上古卷轴》《辐射》等 IP 的重磅加盟使该月的价值达到了 964.67 美元（约 6144.95 元人民币）。</p><p data-vmark="5f8a"><img src="https://img.ithome.com/newsuploadfiles/2021/3/20210312_100731_440.jpeg" title="微软“钞能力”：Xbox Game Pass 今年新增游戏价值超 4 万元" referrerpolicy="no-referrer"></p><p data-vmark="003d">IT之家了解到，Xbox Game Pass 是微软推出的一项游戏订阅服务，玩家在订阅期内可以畅玩该服务包含的数百款游戏，但除了部分微软第一方游戏，其余游戏都有上架期限，当游戏被移出 Xbox Game Pass 服务后，玩家就不再能免费畅玩了，因此该服务更适用于游玩一次通关类型的游戏。</p><p data-vmark="25f0">微软 Xbox 部门负责人 Phil Spencer 此前表示，Xbox Game Pass 服务当前的状态为”可持续“，而且将来只会变得更强大，<span class="accentTextColor">并没有在烧钱</span>。</p><p data-vmark="bf84">在今年的 TGA 上，微软宣布 Xbox Game Pass 在 PC 平台更名为 PC Game Pass，但 Logo 仍将包括一个 Xbox 符号。</p><p
