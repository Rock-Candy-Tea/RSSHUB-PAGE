<p data-vmark="1481"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 11 日消息，昨日，微软宣布，计划在 2022 年上半年，<span class="accentTextColor">面向中国市场推出 Azure Arc 混合云与多云管理解决方案</span>，将进一步增强在中国市场运营的云服务能力。</p><p style="text-align: center;" data-vmark="d961"><img src="https://img.ithome.com/newsuploadfiles/2022/3/04833b4e-c6c9-4db2-917b-8d86406aebc0.jpg@s_2,w_820,h_601" w="1200" h="879" alt="微软" title="微软：计划今年上半年面向中国市场推出 Azure Arc 混合云方案" srcset="https://img.ithome.com/newsuploadfiles/2022/3/04833b4e-c6c9-4db2-917b-8d86406aebc0.jpg 2x" width="1200" height="601" referrerpolicy="no-referrer"></p><p data-vmark="4d07">据介绍，微软智能云 Azure 混合云让企业<span class="accentTextColor">能够使用本地、多云或边缘技术来开发、部署、管理和保护应用程序基础结构</span>，将各种技术集成到可扩展、可靠和高效的体系结构中。</p><p data-vmark="a595">微软表示，微软智能云 Azure Arc 混合云与多云管理解决方案是微软智能云 Azure 混合云的核心管理方案。Azure Arc 提供了统一的管理平面，可以在一个地点管理所有的混合计算工作负载，并提供标准化的视图、运营和合规管理。</p><p data-vmark="a4d5">IT之家了解到，Azure Arc 是微软混合云解决方案的最新进展。此前，微软已经推出了 Azure Stack HCI 混合云解决方案。</p>
