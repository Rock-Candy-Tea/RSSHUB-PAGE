<p data-vmark="cfd0">韩国产业通商资源部与中小创投企业部近日共同宣布，<span class="accentTextColor">将于明年第二季度正式启动半导体 IP 银行平台</span>，下月起开始进行平台经营者选拔程序。</p><p data-vmark="f8eb"><img src="https://img.ithome.com/newsuploadfiles/2021/12/85bb1a18-0fee-4880-9851-ddcab139ce05.png" w="496" h="256" alt title="半导体 IP 持续涨价，韩国明年 Q2 启动首个 IP 交易平台" width="496" height="256" referrerpolicy="no-referrer"></p><p data-vmark="d43e">据 BusinessKorea 报道，目前，<span class="accentTextColor">韩国本土 IP 发展基础薄弱，Fabless 对海外 IP 依赖程度较高</span>，而海外 IP 售价往往非常昂贵。该平台旨在开发半导体 IP，并帮助 Fabless 更容易获得半导体行业的先进 IP。</p><p data-vmark="91ef">此消息发布之际，ETNews 上周最新报道指出，海外半导体 IP 售价持续上涨。在 10nm 以下微处理领域，单个 IP 的价格超过 80 亿韩元的事例也时有发生，缺乏资金支持的 Fabless 将不得不放弃尖端工艺设计。</p><p data-vmark="dbf1">半导体业界透露，<span class="accentTextColor">用于 10nm 以下工艺的高速连接技术 SerDes IP 价格为 500 万-700 万美元，USB IP 的价格为 100 万-130 万美元</span>，构建计算环境的主接口 IP 的价格，如“PCIe”（主板接口）和“SATA”（存储设备连接标准），也在类似的范围内。</p><p data-vmark="2ba8">大多数昂贵的 IP 由 ARM 和 Synopsys 等全球 IP 公司拥有。为了提供稳定的服务，代工厂必须为现有的海外 IP 提供优化的流程。韩国本土 Fabless 不得不承受并支付昂贵的许可证费用。没有足够的资金，芯片开发就无法开始。</p><p data-vmark="ade4">Fabless 相关人士表示，“为了设计 10nm 以下的汽车用人工智能（AI）半导体或应用处理器（Aps），需要购买的 IP 价格合计将超过数百亿韩元。”</p><p data-vmark="af65">韩国系统集成电路革新中心所长 Hwee-won Kim 强调，“随着半导体微缩持续推进，IP 的重要性将不可避免地提高。虽然困难重重，但从长远来看，有必要营造有利于 IP 研发投资和国内优质 IP 发展的环境。我们还需要一个短期应对策略。”</p>
