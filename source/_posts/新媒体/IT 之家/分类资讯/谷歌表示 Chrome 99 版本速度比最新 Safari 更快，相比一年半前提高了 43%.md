<p data-vmark="0b6f"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 3 月 8 日消息，谷歌表示，最新版本的 Chrome M99 在苹果自己的 Speedometer 网络基准测试中创造了记录，并且在苹果自己的 Mac 平台上比 Safari 运行更快、响应更快。</p><p data-vmark="f0b5">谷歌 Chromium 博客上的一篇文章详细介绍了 Chrome 在过去一年左右在编译 JavaScript 和渲染图形时对速度的改进。</p><p data-vmark="1c7a">根据谷歌的说法，它的浏览器在 M1 Mac 上是最快的，光看基准测试的话比 Safari 快约 7%；而 Chrome M99 在 Apple WebKit 团队创建的基准测试中也创下了 300 分的速度记录。</p><p data-vmark="6e21"><img src="https://img.ithome.com/newsuploadfiles/2022/3/5b8759db-7b7f-4488-a0fb-31ac0392d4eb.jpg" alt="Chrome 100 Beta marks the beginning of the end for user agent strings" title="谷歌表示 Chrome 99 版本速度比最新 Safari 更快，相比一年半前提高了 43%" referrerpolicy="no-referrer"></p><p data-vmark="3ff3">IT之家了解到，Speedometer 基准测试旨在模拟使用使用各种技术运行的 Web 应用程序的体验，以反馈更快的响应速度为用户带来的感受。</p><p data-vmark="90a7">谷歌指出，它在配备 10 核 M1 Max 芯片和 64 GB RAM 的 14 英寸 MacBook Pro 上进行了测试，后续还在配备 16 GB RAM 的 13 英寸 M1 MacBook Pro 上重复测试。</p><p data-vmark="9422">最终数据显示：Chrome 每分钟运行 252 次，正负 8.6，而 Safari 则为 185，正负 46。差不多平均有 30% 的差异，但 Safari 也存在很大的可变性。</p><p data-vmark="f97a">当然，对于一般的浏览器来说，性能是很重要的，毕竟没人喜欢在上网时等待。但就它的价值而言，Chrome 与 Safari 各有千秋，我们不能单纯拿响应速度作比较判一个高低。</p><p data-vmark="0196">而且部分同时使用 Chrome 和 Safari 的小伙伴也反馈称，两者之间没有太明显的速度差异，可能这个数据也是因人而异，IT之家不做评论。</p>
