<p data-vmark="11da"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 1 日消息，GitHub 在 6 月份联合 OpenAI，推出了一款“GitHub Copilot”工具，可以根据上下文自动写代码，包括文档字符串、注释、函数名称、代码，只要用户给出提示，就可以写出完整的函数。</p><p data-vmark="bc5d"><img src="https://img.ithome.com/newsuploadfiles/2021/6/5416188a-f440-4769-9099-fbe742b36dfd.png" title="GitHub AI 编程工具 Copilot 已支持 IntelliJ IDEA、PyCharm 等" referrerpolicy="no-referrer"></p><p data-vmark="2eee">该工具此前仅支持 Visual Studio Code，在近日的  GitHub Universe 2021 开发者大会上，GitHub 官方宣布了 Copilot 的更新，<span class="accentTextColor">已支持 Neovim 和 JetBrains IDE，包含开发者常用的 IntelliJ IDEA、PyCharm 等</span>。</p><p data-vmark="f838">要想使用 GitHub Copilot，用户需将 IntelliJ IDEA 和 PyCharm 升级到 2021.2 或以上版本；而 Neovim 的版本则需为 0.6 或以上版本，并且需要安装 Node.js v12 或以上版本。</p><p data-vmark="374d"><img src="https://img.ithome.com/newsuploadfiles/2021/11/afec61b9-f95c-4c1c-a13d-a088bd950bb2.png" w="800" h="422" title="GitHub AI 编程工具 Copilot 已支持 IntelliJ IDEA、PyCharm 等" width="800" height="422" referrerpolicy="no-referrer"></p><p data-vmark="d83c">IT之家了解到，<span class="accentTextColor">GitHub Copilot 还增加了对 Java 中多行补全的支持</span>，并在未来几个月内支持更多语言。目前，GitHub Copilot 支持 Python、JavaScript、TypeScript、Ruby 和 Go 等编程语言。</p><p data-vmark="5fa1"><strong>GitHub Copilot 官网：</strong><a href="https://copilot.github.com/" target="_blank">点此查看</a></p>
