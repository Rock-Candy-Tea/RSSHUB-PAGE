            <p data-vmark="d41e"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 30 日消息，4 月 20 日，知乎发布了《关于即将上线显示用户 IP 属地的公告》，针对站内部分问题的回答展示评论 IP 属地。今日上午，知乎官方宣布<strong>全面上线显示用户 IP 属地</strong>。</p><p data-vmark="5097">IT之家了解到，知乎表示，其采纳了部分网友提出的有效建议，进一步优化升级了显示 IP 地址属地功能。自本公告发布起，<strong>升级后的版本会陆续上线</strong>。</p><p data-vmark="b6b4">公告显示，知乎改进了评论区“评论回复”的 IP 地址属地信息显示问题，<strong>评论区将全面显示 IP 地址属地信息</strong>。</p><p data-vmark="a87d">此外，知乎新增<strong>账号个人主页显示最后一次发布内容对应的 IP 地址属地信息</strong>。回答、文章、视频、想法等多个环节将逐步开放显示用户发文时对应的 IP 地址属地信息。</p><p data-vmark="5d0b">知乎还表示，因过往历史内容无法追溯属地，所以<strong>历史内容无法显示属地信息</strong>。接下来知乎将继续加强对违法违规信息的识别和处置能力。</p><p data-vmark="ee4e" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/4/6b300d02-bad5-4a16-8621-fd52640b78d0.jpg@s_2,w_820,h_595" w="1440" h="1045" alt="知乎全面上线显示用户 IP 属地" title="知乎宣布全面上线显示用户 IP 属地，涉及个人主页、回答、评论等" srcset="https://img.ithome.com/newsuploadfiles/2022/4/6b300d02-bad5-4a16-8621-fd52640b78d0.jpg 2x" width="1440" height="595" referrerpolicy="no-referrer"></p><p data-vmark="3b4c" style="text-align: center;">▲ IP 地址属地信息显示 示意图 | 图源：知乎</p>
