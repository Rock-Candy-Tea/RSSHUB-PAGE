<p data-vmark="785c">10 月 12 日消息，近日，有位外国网友在网上发布了一张特斯拉 Model S Plaid 的电池组的照片。</p><p data-vmark="a1f7">从图片中可以发现，该电池组里使用的仍是 18650 电池，外媒称特斯拉未来一段时间内可能不会完全放弃 18650 电池，并会为 18650 电池保持技术更新。</p><p data-vmark="3c69">目前，特斯拉正在努力改进其电池技术。未来，特斯拉的 4680 电池组将会用于 Semi 卡车、Cybertruck 和新 Roadster 等需要强大电能的车型上，而特斯拉的高端车型在近两年可能还会使用 18650 电池组。</p><h2 data-vmark="6238">一、电池组模块减少 18650 电池终要谢幕</h2><p data-vmark="84cc">该名网友表示，新款 Model S Plaid 的电池组与老款车型有所区别，新的电池模组采用了 5 模块布局方式，而原来为 16 模块。</p><p data-vmark="b6d8">模块数量的减少也就意味着车辆的零件数量减少，进而降低了车辆的生产成本。马斯克曾表示，降低车辆的生产成本是特斯拉的首要任务。</p><p data-vmark="91f0">根据这名外国网友的描述，测量后发现图片中的电池直径为 18mm，长度为 65mm，这也就证明了新款的 Model S Plaid 车型还在使用来自松下的 18650 电池。</p><p data-vmark="f297"><img src="https://img.ithome.com/newsuploadfiles/2021/10/9d3c28c5-9f1a-4659-87db-96c61cb3e488.jpg@s_2,w_820,h_615" w="850" h="638" title="18650 电池将成历史，特斯拉新款 Model S 陪它“退役”" srcset="https://img.ithome.com/newsuploadfiles/2021/10/9d3c28c5-9f1a-4659-87db-96c61cb3e488.jpg 2x" width="850" height="615" referrerpolicy="no-referrer"></p><p data-vmark="646e">▲ 外国网友所拍照片</p><p data-vmark="8fcc">特斯拉之前曾表示，新款的电池组结构可以进一步提升车辆的动力以及能源利用效率。</p><p data-vmark="2175">外媒预测，特斯拉 Model S Plaid 和未来将要发布的 Model X Plaid 可能是特斯拉产品阵列中，<span class="accentTextColor">最后两款使用 18650 电池的车型</span>。未来，随着特斯拉不断加大 4680 电池的生产规模，18650 电池终将退出历史舞台。</p><p data-vmark="baa0">从 2008 年发布其第一辆电动车 Roadster 开始，特斯拉就一直在他们的车辆上使用 18650 电池，现在已经超过 13 年了，18650 电池也是时候退役了。</p><p data-vmark="5dd0"><img src="https://img.ithome.com/newsuploadfiles/2021/10/f1fcfc6c-b24f-4f0f-963c-cfd259ebbcf0.jpg@s_2,w_820,h_495" w="850" h="513" title="18650 电池将成历史，特斯拉新款 Model S 陪它“退役”" srcset="https://img.ithome.com/newsuploadfiles/2021/10/f1fcfc6c-b24f-4f0f-963c-cfd259ebbcf0.jpg 2x" width="850" height="495" referrerpolicy="no-referrer"></p><p data-vmark="6094">▲ 2008 款 Roadster</p><h2 data-vmark="03a2">二、4680 将分镍基和铁基 Model Y 率先上车</h2><p data-vmark="d9ad">未来，特斯拉的 4680 电池的电芯分为镍基和铁基两种正极材料，更少的电池化学种类将会减轻特斯拉内部的开发难度和后期维护的复杂度。</p><p data-vmark="ab73">根据之前的报道，特斯拉有望成为第一家使用来自 LG 能源的新型 NCMA 电池的汽车制造商，而 LG 能源的新 NCMA 电池中，使用了 90% 的镍基正极材料，减少钴元素的使用，并且电池的能力密度也得到提升。</p><p data-vmark="1107">这种镍基电池将会被应用于特斯拉的高功率车型或瞬时耗电量大的车型，例如特斯拉的 Semi 卡车、Cybertruck 皮卡和新款 Roadster 跑车，这三款车型分别需要强大的拖拽能力、强大的装载能力和强大的加速能力。</p><p data-vmark="079f">未来，特斯拉的高端车型 Model S 和 Model X 也将会使用这种电池，进一步提升车辆的性能和续航能力，保证在其纯电豪华轿车和纯电豪华 SUV 市场中的竞争力。</p><p data-vmark="9af4"><img src="https://img.ithome.com/newsuploadfiles/2021/10/1a81fb17-4d0d-4b58-9de8-30fc84caaca6.jpg@s_2,w_820,h_434" w="850" h="450" title="18650 电池将成历史，特斯拉新款 Model S 陪它“退役”" srcset="https://img.ithome.com/newsuploadfiles/2021/10/1a81fb17-4d0d-4b58-9de8-30fc84caaca6.jpg 2x" width="850" height="434" referrerpolicy="no-referrer"></p><p data-vmark="00df">▲ 特斯拉 Model S Plaid</p><p data-vmark="0119">而像 Model 3、Model Y 以及未来要发布的入门级车型这类量大价低的车型，将会使用铁基正极材料的 4680 电池。</p><p data-vmark="171a">今年 8 月，特斯拉在美国的库存中出现了使用宁德时代磷酸铁锂电池的标准续航增强版 Model 3，该车的续航里程比三元锂电池版本下降 10 英里（约合 16 公里），但价格保持不变。</p><p data-vmark="d8fc">马斯克之前曾表示，他本人更喜欢磷酸铁锂电池，因为它可以充电至 100%，而三元锂电池只推荐充到 90%。</p><p data-vmark="73d1">昨天，特斯拉在柏林超级工厂举办开放日活动，在其展示的产品中，就已经可以看到 4680 电芯出现在特斯拉 Model Y 的车身中，而特斯拉柏林工厂或将会是最先用上这一产品的工厂。</p><p data-vmark="6537"><img src="https://img.ithome.com/newsuploadfiles/2021/10/8e71c237-343a-448f-b2a3-751662b23e31.jpg@s_2,w_820,h_615" w="850" h="638" title="18650 电池将成历史，特斯拉新款 Model S 陪它“退役”" srcset="https://img.ithome.com/newsuploadfiles/2021/10/8e71c237-343a-448f-b2a3-751662b23e31.jpg 2x" width="850" height="615" referrerpolicy="no-referrer"></p><p data-vmark="557d">▲ 特斯拉展示了 4680 电芯</p><h2 data-vmark="ed1a">结语：4680 电池成特斯拉“新宠”</h2><p data-vmark="9e2b">相比于 18650 电池以及 2170 电池来说，4680 电池的能量密度、输出功率都有大幅提升，生产成本也进一步降低。而搭载 4680 电池的车辆将会拥有更高的续航里程以及更低的使用成本</p><p data-vmark="180c">目前，整个动力电池行业都在加速研发或生产 4680 电池，但是目前还没有一款量产电动汽车使用 4680 电池。</p><p data-vmark="6d04">特斯拉的柏林工厂将于今年年底正式投产，4680 电池也将会迎来首次上车，如果说 18650 电池帮特斯拉发家成名，那么 4680 电池将成为未来特斯拉的金字招牌。</p>
