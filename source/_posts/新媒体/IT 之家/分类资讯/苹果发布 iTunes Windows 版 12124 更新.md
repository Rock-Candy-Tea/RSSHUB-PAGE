<p data-vmark="b480"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 19 日消息，今日，除 iOS 15.6、macOS 12.5 开发者预览版 Beta 外，苹果还<strong>发布了 iTunes Windows 版 12.12.4 更新</strong>，为新版本加入安全修复程序以解决漏洞。</p><p style="text-align: center;" data-vmark="3beb"><img src="https://img.ithome.com/newsuploadfiles/2022/5/bffe6ef9-b3ec-4084-885a-743683b973ed.png" w="579" h="380" alt="iTunes" title="苹果发布 iTunes Windows 版 12.12.4 更新" width="579" height="380" referrerpolicy="no-referrer"></p><p data-vmark="8a0a">IT之家了解到，根据 Windows 版 iTunes 的安全支持文档，新版本 iTunes <strong>包括对 AppleGraphicsControl、ImageIO、移动设备服务、WebKit 等问题的修复</strong>。</p><p style="text-align: center;" data-vmark="83db"><img src="https://img.ithome.com/newsuploadfiles/2022/5/9b9111d6-6521-438f-ba43-a4e11e8fea5d.png" w="1440" h="810" alt="iTunes" title="苹果发布 iTunes Windows 版 12.12.4 更新" width="1440" height="461" referrerpolicy="no-referrer"></p><p data-vmark="b954">苹果并未提及漏洞是在自然环境中发现的，但 Windows 用户仍应尽可能将 iTunes 更新至最新版本。iTunes 12.12.4 没有其他明显变化，主要集中在底层方面的改进。</p><p style="text-align: center;" data-vmark="e33a"><img src="https://img.ithome.com/newsuploadfiles/2022/5/88cdc085-e54e-4f36-a96a-e78b31ffe77e.png" w="1440" h="810" alt="iTunes" title="苹果发布 iTunes Windows 版 12.12.4 更新" width="1440" height="461" referrerpolicy="no-referrer"></p><p data-vmark="8312">没有安装 iTunes 的用户，可从 <a href="https://apps.microsoft.com/store/detail/itunes/9PB2MZ1ZMB1S?hl=zh-cn&gl=CN" target="_blank">Microsoft Store</a> 下载 iTunes（Windows 10 或更高版本）应用，也可以从 <a href="https://support.apple.com/zh-cn/HT210384" target="_blank">Apple 网站</a>下载最新版本的 iTunes。</p>
