<p data-vmark="1c37">本周五一家爱尔兰非执业实体 (NPE) Sonrai Memory 在美国得克萨斯州西区地方法院提起诉讼，<span class="accentTextColor">指控苹果侵犯了其拥有的两项专利</span>。</p><p data-vmark="c216">Patently Apple 报道称，其中一项专利是 6,874,014，标题是 "具有多个操作系统的芯片多处理器，可追溯到 2005 年。另一项是 6,724,241，被称为" 具有动态负载的可变电荷泵电路 "，首次授权于 2004 年。侵权涉及众多苹果产品，可追溯到 iPhone 6 和 iPhone 6 Plus、第五代 iPad、所有 iPad Pro 代以及各种 MacBook Air 和 MacBook Pro 型号。</p><p data-vmark="184d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/5cbd37da-d757-4e38-9b77-9c1527faa876.png" w="700" h="447" title="又被专利流氓盯上？爱尔兰 Sonrai Memory 对苹果提起专利侵权诉讼" width="700" height="447" referrerpolicy="no-referrer"></p><p data-vmark="e2f5">Sonrai Memory 称，苹果 A 系列、M 系列和 T 系列芯片组使用的技术侵犯了其 6,874,014 专利。同时，6,724,241 涉及 SK 海力士 NAND 闪存芯片。该公司主张赔偿损失、成本、费用、利息、对持续许可费的判决以及“合理的律师费”。</p><p data-vmark="4c6a">据悉，Sonrai Memory 从惠普和 Atmel（已被 Microchip 收购）获得了诉讼涉及的相关专利。自 2021 年 7 月以来，Sonrai Memory 还对西部数据、Google、三星、LG、联想和其他公司提起了类似的专利侵权诉讼。</p>
