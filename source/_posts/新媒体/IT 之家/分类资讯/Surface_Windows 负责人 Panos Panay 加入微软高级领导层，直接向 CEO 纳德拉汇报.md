<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 26 日消息 外媒 onMSFT、Windows Central 报道，彭博称，周三晚上微软高管迎来调整：首先，Surface 主管、现在负责所有 Windows 和设备的 Panos Panay 已被提升到公司的高级领导层团队，担任 EVP 执行副总裁。</p><p>此项举措是在微软准备推出 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 的时候进行的，该系统是对 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 的巨大改造和重塑。这项任命显示了微软对 Panos Panay 领导下 Windows 11 和 Surface 愿景的巨大信心，后者在 Panos Panay 的领导下已经成长为另一个价值 10 亿美元的微软业务。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/441d1066-6170-4604-ba6b-fed47f118271.jpg@s_2,w_820,h_615" w="1024" h="768" title="Surface/Windows 负责人 Panos Panay 加入微软高级领导层，直接向 CEO 纳德拉汇报" srcset="https://img.ithome.com/newsuploadfiles/2021/8/441d1066-6170-4604-ba6b-fed47f118271.jpg 2x" width="1024" height="615" referrerpolicy="no-referrer"></p><p>Panos Panay 将与 Xbox 负责人 Phil Spencer 一起在高级领导层任职，直接向微软 CEO 兼董事会董事萨提亚・纳德拉（Satya Nadella）汇报工作。</p><p>IT之家获悉，多年来，Panos Panay 对 Surface 的设想是以创造者和消费者为目标，而 Windows 11 似乎也在效仿。微软承诺将<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>应用引入 Windows 本身，在社交媒体上的营销也越来越倾向于对 Windows 的怀旧，展示了振兴的 Clippy 和微软 Paint。希望我们能看到 Windows 和 Surface 享受到类似于在 Xbox 上看到的投资促进。</p><p>此外，亚马逊 AWS 资深人士 Charlie Bell 将加入微软，担任公司副总裁。据 CNBC 报道，目前还不清楚贝尔将在微软担任什么角色，他目前在微软的企业目录中列在人力资源主管 Kathleen Hogan 之下，但这可能只是暂时的。</p><p>亚马逊最近有一些霸道地使用非竞争条款来反对其他前亚马逊高管转投其他公司的历史，最值得注意的是（曾是微软 Surface 营销主管和 20 年微软老员工）Brian Hall 从亚马逊最终转投谷歌。然而，根据 Corey Quinn 的说法，这次可能不是这种情况。</p><blockquote><p>“是的，亚马逊和微软将理顺非竞争条款。亚马逊 AWS 有很多非常了不起的人在那里工作，但你们中没有人敢于尝试起诉 Charlie Bell。”</p></blockquote><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/76f16fe7-6e99-4213-8896-641e3234c7e9.png" w="681" h="1039" title="Surface/Windows 负责人 Panos Panay 加入微软高级领导层，直接向 CEO 纳德拉汇报" width="681" height="1039" referrerpolicy="no-referrer"></p><p>主导 Windows 系统的 Panos Panay 在微软高级领导层有了一席之地，另外还有 Charlie Bell 的加入，即使微软 Windows 11 将和亚马逊合作支持安卓应用，也不妨碍微软继续用 Azure 对抗亚马逊 AWS。</p>
