<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 1 日消息 随着 Win11 的重磅推出，微软正在对应用商店进行了一些期待已久的更改，比如全新的设计、搜索体验、新的应用下载/安装体验、通过集成的 Amazon AppStore 安装 Android 应用等等。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/75dc41ff-0f52-499a-ae70-16dc836d6ef5.png" title="微软 Win11 应用商店已上线 Edge 浏览器扩展" referrerpolicy="no-referrer"></p><p>除此之外，微软在政策上为开发者给予了极大的“自由”，允许上架 Win32、.NET、UWP、Xamarin、Electron、React Native、Java 甚至 Progressive Web Apps。</p><p>今日，微软兑现承诺，已将 <span class="accentTextColor">Edge 浏览器扩展</span>上架了 Win11 应用商店，用户可以像搜索应用一样，直接在 Win11 应用商店里浏览、查找 Edge 浏览器扩展，就像老版本的 Edge 浏览器那样。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/39422950-d1be-4bc1-8506-d9918e2ec80f.jpg@s_2,w_820,h_472" alt="Microsoft Store Edge extensions" w="1080" h="621" title="微软 Win11 应用商店已上线 Edge 浏览器扩展" srcset="https://img.ithome.com/newsuploadfiles/2021/7/39422950-d1be-4bc1-8506-d9918e2ec80f.jpg 2x" width="1080" height="472" referrerpolicy="no-referrer"></p><p>从图中可以看到，当用户搜索 Microsoft Edge 时，将自动匹配 Edge 浏览器扩展，包括 Outlook、LastPass for Edge 等。</p><p>目前尚不清楚微软将如何上架 Edge 浏览器扩展，也许是自动的，不需要开发者单独上架。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/6793b9ce-fbdc-4b71-80a6-10f6aaa60b6a.png" w="1180" h="349" title="微软 Win11 应用商店已上线 Edge 浏览器扩展" width="1180" height="243" referrerpolicy="no-referrer"></p><p>而 Edge 浏览器端的扩展商店也会进行保留，毕竟 Edge 浏览器作为跨平台浏览器，不是所有系统都有一个 Win11 应用商店。</p><p>IT之家了解到，微软还将改进 Win11 的搜索算法，并支持从浏览器网页端安装应用。微软表示，应用商店的目录和搜索质量将在<span class="accentTextColor">未来几个月内“大大提高”</span>。</p>
