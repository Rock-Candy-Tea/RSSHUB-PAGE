<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 8 日消息 苹果今日举行了 WWDC21 全球开发者发布会，发布了 iOS/iPadOS 15、macOS 12 Monterey 等系统大版本更新。</p><p>据外媒 MacRumors，苹果面向开发者们宣布了一系列新的开发功能，比如<span class="accentTextColor">向第三方开发者开放了屏幕时间 API</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/6/6ca1b976-a199-4445-bc80-47faa1b00c58.jpg" alt="apple screen time screen icons" w="400" h="329" title="苹果向第三方开发者开放“屏幕时间 API”，孩子再也不能偷偷玩 iPad 了" width="400" height="329" referrerpolicy="no-referrer"></p><p>苹果的屏幕时间是在 2018 年的 iOS 12 中首次推出的，用户可以查看自己在各个 App 中使用了多少时间。然而，当初的屏幕时间是直接集成在 iPhone 中的，也就是只能用户本人看到。</p><p>现在，苹果将屏幕时间 API 开放给了第三方开发者，以<span class="accentTextColor">帮助 App 建立家长模式，以后的家长监控将可以精细到各个 App</span>，孩子即使偷偷看到了 iPad 密码，也不能偷偷玩游戏了。</p><p>此外，苹果在开放该 API 时<span class="accentTextColor">特别提到了隐私</span>，预计会有一些上架审查。</p><p>IT之家了解到，苹果当年推出屏幕时间功能后，禁用了一些专门做家长模式的 App，由此<span class="accentTextColor">引来了反垄断调查</span>，本次开放 API，也可能是为了缓解受到的反垄断压力。</p>
