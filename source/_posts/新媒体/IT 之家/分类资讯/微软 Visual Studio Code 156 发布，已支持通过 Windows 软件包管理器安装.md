<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 9 日消息 微软 Visual Studio Code 1.56 版本已经发布，本次更新有几大亮点。</p><p>首先，Visual Studio Code 1.56 版本可以<span class="accentTextColor">通过 Windows 软件包管理器 WinGet 安装</span>了，x86、x64、arm64 版本都可以安装，用户还可以通过 --scope 参数来选择用户安装和系统安装。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/99ab5815-0d2c-4c5b-8ee5-21bd3eccf915.png" id="1620549594324-1" title="微软 Visual Studio Code 1.56 发布，已支持通过 Windows 软件包管理器安装" referrerpolicy="no-referrer"></p><p>其次，Visual Studio Code 1.56 版本的<span class="accentTextColor">调试器 UI 支持了新的 Inline values 扩展 API</span>，用户安装诸如 Debugger for Java 扩展之后，就可以<span class="accentTextColor">在调试时在代码右侧看到变量的值、类型等数据</span>，非常方便。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/1ae6bfd7-f157-4db7-ba6e-98e8b97fa34b.png" w="897" h="470" title="微软 Visual Studio Code 1.56 发布，已支持通过 Windows 软件包管理器安装" width="897" height="430" referrerpolicy="no-referrer"></p><p>IT之家了解到，Visual Studio Code 1.56 版本还有许多其他功能改进：</p><ul class=" list-paddingleft-2"><li><p>鼠标悬停反馈改进，更有标识性；</p></li></ul><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/690ae46e-464a-4fab-8dfa-4bf97fc37e13.gif" alt="Actions hover feedback is improved with a background color." w="1144" h="886" title="微软 Visual Studio Code 1.56 发布，已支持通过 Windows 软件包管理器安装" width="1144" height="635" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p>终端配置文件的改进：创建自定义默认终端配置文件；</p></li><li><p>Notebook KaTeX 支持：notebook Markdown cells 中的数学支持；</p></li></ul><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/574fff9d-31cd-4257-948a-84b9c0c298f1.png" w="934" h="358" title="微软 Visual Studio Code 1.56 发布，已支持通过 Windows 软件包管理器安装" width="934" height="314" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p>Remote - Containers volumes view：管理 Docker 容器中已安装的卷；</p></li><li><p>新的入门视频：关于开始使用 VS Code 以及使用 C++ 的视频；</p></li></ul><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/364bca2f-7124-40e3-a5c3-93710d5fb028.png" w="644" h="430" title="微软 Visual Studio Code 1.56 发布，已支持通过 Windows 软件包管理器安装" width="644" height="430" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p>Terminal tabs 预览：通过新的 tabs 视图初步了解管理 open terminals 的情况。可以通过以下设置启用：</p></li></ul><blockquote><p>"terminal.integrated.tabs.enabled": true</p></blockquote><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/e17c6d34-0d47-4fd1-bcba-b4aa4d407e92.png" w="607" h="254" title="微软 Visual Studio Code 1.56 发布，已支持通过 Windows 软件包管理器安装" width="607" height="254" referrerpolicy="no-referrer"></p><p>完整的更新内容可以<a href="https://code.visualstudio.com/updates/v1_56#_windows-installers-consolidated-in-windows-package-manager" target="_blank">点此查看</a>，全平台下载页面<a href="https://code.visualstudio.com/Download" target="_blank">点此进入</a>。</p>
