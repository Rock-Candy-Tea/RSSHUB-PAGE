<p data-vmark="e2b0"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 3 日消息，微软 GitHub 是现有最受欢迎的软件存储库之一，为无数软件项目提供支持，并为许多产品托管关键基础设施。</p><p data-vmark="d7ed">GitHub 一直在鼓励用户从经济上支持喜欢的软件项目，现在，GitHub 为项目推出一项<span class="accentTextColor">捐赠功能</span>。</p><p data-vmark="91c0" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/24c7eb7f-7eb3-4b47-8426-2fa16bb5d02e.png" w="1440" h="749" title="微软 GitHub 推出赞助功能：可为喜欢的项目打赏，还能设置付费访问" width="1440" height="427" referrerpolicy="no-referrer"></p><p data-vmark="9db2">GitHub 已经允许开发者将<span class="accentTextColor">捐赠链接</span>添加到他们的项目中，或者开发者可以注册公司，添加“GitHub 赞助”功能。</p><p data-vmark="b897">GitHub 在周三的一篇<a href="https://github.blog/2022-02-02-new-sponsors-only-repositories-custom-amounts-and-more/" target="_blank">博客</a>中写道，“启用 GitHub 赞助的开发者和组织可以将私人存储库附加到每个赞助层，<span class="accentTextColor">可授予不同的资助者访问存储库的不同权限</span>。这些邀请由 GitHub 自动管理，用户在设置后无需管理任何内容。”</p><p data-vmark="0fd9" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/036dfb45-9d92-4f03-bf6c-9078683c699d.png" w="999" h="343" title="微软 GitHub 推出赞助功能：可为喜欢的项目打赏，还能设置付费访问" width="999" height="282" referrerpolicy="no-referrer"></p><p data-vmark="d022">GitHub 表示，仅限赞助后访问的存储库可用于早期访问或 beta 测试，或者是类似论坛讨论的入场券，开发者可以<span class="accentTextColor">自行设定金额</span>。</p><p data-vmark="21c1" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/2/7fe14abc-9626-4cc5-ac60-c199f4e5df9f.png" w="903" h="399" title="微软 GitHub 推出赞助功能：可为喜欢的项目打赏，还能设置付费访问" width="903" height="362" referrerpolicy="no-referrer"></p>
