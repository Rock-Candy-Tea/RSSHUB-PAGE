<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 13 日消息 时隔近一个月，微软终于发布了 <span class="accentTextColor">Visual Studio 2022 Preview 3 版本</span>，本次更新带来了关于个人和团队生产力、现代开发和不断创新为主题的新功能。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/a693d20c-23bb-41c3-afd4-1ef4a9a25b1e.jpg" alt="Visual Studio 2022 Preview 2 is now available with new icons and features" title="微软 Visual Studio 2022 Preview 3 发布" referrerpolicy="no-referrer"></p><h2>个人和团队生产力</h2><p>在 Visual Studio 2022 Preview 3 中，微软为一些不太常用但有时很有用的功能进行了改进，例如附加到进程。</p><ul class=" list-paddingleft-2"><li><p>附加到进程改进</p></li></ul><p>一个例子是对附加到进程的对话框的改进。该对话框现在是异步的、显示进程的命令行参数、w3wp.exe 进程的 IIS 信息，而且该对话框还有一个可选的树状视图模式，用于显示 parent-child 进程关系。这些功能减少了在高级场景中决定调试哪个进程的很多摩擦。</p><ul class="ai-word-checked list-paddingleft-2"><li><p>新项目设计器</p></li></ul><p>在 Preview 3 中，为 .NET SDK 项目提供了一个全新的项目属性设计器。新的设计器更容易使用和浏览，单列的选项有清晰的描述。最重要的是，新的设计器有内置的搜索功能，用户可以轻松找到需要的属性。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/66b9c05d-b378-4b9e-9ee8-6c8ae6c33a92.gif" w="960" h="540" title="微软 Visual Studio 2022 Preview 3 发布" width="960" height="461" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p>Dark 主题改进</p></li></ul><p>在 Preview 3 中，Dark 主题将会拥有巨大变化，以提高 Visual Studio 的可用性。新的 Dark 主题有一个新的强调色，旨在减少分心和眼睛疲劳。该颜色与最新的产品视觉识别相匹配，可帮助用户在多个工具之间导航时快速找到正确的窗口。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/938e5c09-86b3-42cd-bba2-a4737c7900c2.gif" w="960" h="540" title="微软 Visual Studio 2022 Preview 3 发布" width="960" height="461" referrerpolicy="no-referrer"></p><h2>开发现代应用</h2><p>在 Preview 3 中，Visual Studio 2022 添加了在 Linux 环境中运行测试的新功能，以及用于使用 TypeScript/JavaScript 的 React 和 Vue.js 应用程序的前端开发的新项目类型。</p><ul class="ai-word-checked list-paddingleft-2"><li><p>远程测试</p></li></ul><p>通过远程测试，用户可以从跨平台测试中获得反馈，甚至可以在 Visual Studio 中进行调试。该功能适用于一系列的远程环境，如 Linux 容器、WSL 和 SSH 连接。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/267efa5a-9fb8-4985-b445-8a8801fd4583.gif" w="960" h="540" title="微软 Visual Studio 2022 Preview 3 发布" width="960" height="461" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p>新的 JavaScript/TypeScript 项目类型</p></li></ul><p>增加用于 Angular、Vue 和 React 的 JavaScript/TypeScript 项目类型，使用户可以轻松地将前端应用程序合并到 Visual Studio 解决方案中，并能够从 Visual Studio 启动和调试整个应用程序。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/8d38b54b-f8b8-47b8-b634-f68be5783dc8.gif" w="960" h="540" title="微软 Visual Studio 2022 Preview 3 发布" width="960" height="461" referrerpolicy="no-referrer"></p><h2>创新触手可及</h2><p>微软表示，Visual Studio 2022 愿景的另一个关键部分是创新功能，可释放用户作为开发者的潜力。</p><ul class="ai-word-checked list-paddingleft-2"><li><p>诊断分析</p></li></ul><p>Preview 3 中的诊断分析器可用于内存转储，以识别 .NET 应用程序中的常见问题，用户能在短时间内找到复杂问题的根源。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/6ae7cdd7-98d3-497a-85cd-d4fe155ed59a.gif" w="960" h="540" title="微软 Visual Studio 2022 Preview 3 发布" width="960" height="461" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p>同时使用多个 Git 存储库</p></li></ul><p>Preview 3 还获得了新的预览功能，例如多个 Git 存储库，用户能够使用具有多个存储库中的项目的单个解决方案，并从 Visual Studio 的单个实例为它们做出贡献。</p><ul class="ai-word-checked list-paddingleft-2"><li><p>一键发布到 Azure DevOps</p></li></ul><p>在 Preview 3 中，通过更新的创建 Git 存储库体验，用户可以更轻松地将代码发布到 Azure DevOps。只需单击一下，Visual Studio 就可以初始化一个新的 Git 存储库，并将它推送到 Azure DevOps。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/8/c8bc6f13-aa88-4115-ada3-d7ccdf953f91.png" w="600" h="442" title="微软 Visual Studio 2022 Preview 3 发布" width="600" height="442" referrerpolicy="no-referrer"></p><p>IT之家了解到，上面的功能只是 Visual Studio 2022 改进的几个例子。在 Preview 3 中还有更多功能可以探索。在接下来的几周内，官方将分享更多 Visual Studio 2022 的新功能。</p><p><strong>Visual Studio 2022 Preview 3：</strong><a target="_blank">点此查看</a></p>
