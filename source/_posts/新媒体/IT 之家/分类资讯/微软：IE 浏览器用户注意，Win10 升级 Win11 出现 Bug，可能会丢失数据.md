<p data-vmark="e075"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 2 日消息，微软通知了 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 用户一个新的已知问题，该问题可能会阻止一些用户更新到 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>。据介绍，<span class="accentTextColor">该问题会影响坚持使用 Internet Explorer 浏览器</span>，并且不愿意换到 Edge 的用户。</p><p data-vmark="4758">根据微软官方 <a href="https://docs.microsoft.com/en-us/windows/release-health/status-windows-11-21h2#ie11-data-might-not-be-accessible-after-upgrading-to-windows-11" target="_blank">Windows 11 Health Dashboard 文档</a>，微软为部分 Internet Explorer 用户设置了更新模块，如果用户在安装 Windows 11 之前<span class="accentTextColor">不接受将浏览数据导入 Edge 的提示</span>，升级到 Windows 11 后可能将无法访问 Internet Explorer 的数据。</p><p style="text-align: center;" data-vmark="9cb3"><img src="https://img.ithome.com/newsuploadfiles/2022/4/21b528f1-861d-4a1a-afff-d3731ccf1f74.png" w="1239" h="727" title="微软：IE 浏览器用户注意，Win10 升级 Win11 出现 Bug，可能会丢失数据" width="1239" height="481" referrerpolicy="no-referrer"></p><p data-vmark="b579">微软表示，目前正在努力修复可能导致潜在数据丢失的错误。<span class="accentTextColor">即将发布的补丁</span>将允许 Windows 10 用户为已经更新到 Windows 11 的用户更新和恢复无法访问的数据。同时，想要使用 Windows 11 的 Internet Explorer 用户可以手动将数据从 Internet Explorer 导入 Edge 浏览器，从而绕过更新模块。</p><p data-vmark="7f7f">IT之家了解到，微软计划在 <span class="accentTextColor">2022 年 6 月 15 日</span>停用 Internet Explorer 并停止支持该浏览器。另一方面，Edge 中的 IE 模式将至少支持到 2029 年。</p>
