<p data-vmark="b57b"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 13 日消息，默认情况下，出于隐私原因，微软 Edge 和谷歌 Chrome 等浏览器在隐身模式下会禁用扩展程序。该功能并不能针对某个特定网站生效，而微软打算补全这一功能。</p><p data-vmark="1f41">据 Windows Latest 报道，从 Edge 浏览器即将推出的版本开始，<span class="accentTextColor">用户将能够限制某些网站的扩展程序访问</span>，即阻止扩展程序读取所选网站上的数据。</p><p style="text-align: center;" data-vmark="ec13"><img src="https://img.ithome.com/newsuploadfiles/2022/4/86a6bdb3-bd38-498d-a7cb-cac5599c8b66.jpg@s_2,w_820,h_212" w="1406" h="363" alt="Edge pause extensions" title="微软 Edge 浏览器将允许在特定网站上停用扩展" srcset="https://img.ithome.com/newsuploadfiles/2022/4/86a6bdb3-bd38-498d-a7cb-cac5599c8b66.jpg 2x" width="1406" height="212" referrerpolicy="no-referrer"></p><p data-vmark="1267">从上图可以看到，该实验性功能可让用户选择对特定网站暂停使用扩展。暂停之后，扩展列表中会有一个蓝色的提醒内容。当然，用户也可以在同一个菜单中选择恢复运行扩展。</p><p style="text-align: center;" data-vmark="cf66"><img src="https://img.ithome.com/newsuploadfiles/2022/4/22d668a7-d634-4e76-9325-1ce079477265.jpg@s_2,w_820,h_523" w="1255" h="801" alt="Microsoft Edge extensions paused" title="微软 Edge 浏览器将允许在特定网站上停用扩展" srcset="https://img.ithome.com/newsuploadfiles/2022/4/22d668a7-d634-4e76-9325-1ce079477265.jpg 2x" width="1255" height="523" referrerpolicy="no-referrer"></p><p data-vmark="3b87">微软正在 Edge Canary 中进行 <span class="accentTextColor">A / B 测试</span>，也就是说一部分用户可以体验到。当用户暂停扩展时，<span class="accentTextColor">网站会进行重新加载</span>。扩展程序暂停后，网站上将出现一条横幅，说明“我们已暂停此站点上的扩展程序以尽量减少您的数据共享”。</p><p data-vmark="8bb0">IT之家此前报道，微软在 Edge 100 中对浏览器的睡眠标签页进行了重大改进，用户可最多可以节省 8% 的额外资源。</p><p style="text-align: center;" data-vmark="45eb"><img src="https://img.ithome.com/newsuploadfiles/2022/4/5c697314-cfda-4a41-9839-8c33bbac55a0.png" w="609" h="601" title="微软 Edge 浏览器将允许在特定网站上停用扩展" width="609" height="601" referrerpolicy="no-referrer"></p>
