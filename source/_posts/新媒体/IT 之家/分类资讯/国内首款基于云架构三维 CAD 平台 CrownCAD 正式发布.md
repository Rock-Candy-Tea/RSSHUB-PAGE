<p>2021 年 9 月 8 日 14：00，山东山大华天软件有限公司（简称：华天软件）召开“云端设计 协同分享 —— 国内首款基于云架构的三维 CAD 平台 CrownCAD”云端发布会。此次发布会通过新华社、央视新闻移动网、中国工业新闻网、中国网、凤凰新闻、数字化企业网、机械工业出版社、抖音等 20 家直播平台实时直播。除此之外，人民网、新华网、科技日报、大众日报、中国新闻网、山东广播电视台、新浪新闻、<a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a>、科创板日报、界面新闻、澎湃新闻等 25 家媒体代表出席现场，共同见证这款历经十年核心技术攻关、五大版本迭代、两轮全国公测的国内首款云化工业软件的成功面世。发布会由华天软件首席技术官、华云三维总经理梅敬成博士主持。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/51a04083-14fa-4493-8217-f99197c94a33.png" w="1437" h="800" title="国内首款基于云架构三维 CAD 平台 CrownCAD 正式发布" width="1437" height="457" referrerpolicy="no-referrer"></p><p>发布会伊始，山东省济南市副市长孙斌、中国工程院院士/华中科技大学教授李培根、e-works CEO 黄培博士分别从政府政策、专业视角、行业趋势等多维度对 CrownCAD 进行解析，他们指出 CrownCAD 在 CAD 关键技术上实现国产自主，为中国制造业的创新提供坚实保障，其对于中国制造业具有里程碑意义。</p><p>作为智能制造的重要基础和核心支撑，工业软件对推动中国制造业转型升级、建设制造强国具有重要的意义。当前，工业软件核心技术被国外垄断，加快攻克“卡脖子”问题成为迫切需求。随着互联网、云计算技术的快速发展，制造业设计模式由传统单机向多人协同设计转变，CAD 也需要顺应技术变革为多人异地协同研发提供支撑，CAD 云化就成了必然趋势。</p><p>国内首款基于云架构的三维 CAD 平台 CrownCAD 包含 CrownCAD、CrownCAD APP、三维几何建模引擎 DGM 和几何约束求解器 DCS 四大产品，发布会上梅敬成博士对四大产品进行了详细介绍。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/52e7aa89-61a0-4c73-be73-f457c75106d5.jpg@s_2,w_820,h_547" w="1440" h="960" title="国内首款基于云架构三维 CAD 平台 CrownCAD 正式发布" srcset="https://img.ithome.com/newsuploadfiles/2021/9/52e7aa89-61a0-4c73-be73-f457c75106d5.jpg 2x" width="1440" height="547" referrerpolicy="no-referrer"></p><p>针对高端制造业的“卡脖子”需求，CrownCAD 提供了完全自主研发的两大 CAD 核心技术：三维几何建模引擎 DGM 和几何约束求解器 DCS。三维几何建模引擎是三维 CAD 软件的核心和基础组成部分。DGM（ Diamond Geometry Modeler ）是完全自主研发且能够独立对外集成的三维几何建模引擎，提供与国际主流建模引擎兼容的 API 接口。几何约束求解器是三维 CAD 参数化设计的基础组件，用于求解尺寸和几何约束，实现二维、三维的参数化设计，在草图、工程图、三维装配、运动仿真中广泛应用。DCS（ Diamond Constraint Solver）是完全自主研发的二维、三维约束求解引擎，提供与国际商用约束求解器兼容的 API 接口。两大核心技术可以让企业独立使用，并按照企业的业务特点进行深度定制。</p><p>CrownCAD 产品，首先是一款 CAD 产品，它包含数据转换、零件设计、装配、工程图等 CAD 软件功能，具备多用户在线协同设计、版本管理、项目分享等云架构带来的优势。用户在任意地点和终端打开<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>（www.crowncad.com）即可进行产品设计和协同分享。</p><p>CrownCAD 云架构具有安全完整的版本管理和创新便捷的协同设计两大优势。安全完整的版本管理通过对云端存储、传输数据的加密，确保数据的安全性，这就为安全性要求高的企业提供安全可控的国产化应用环境。创新便捷的协同设计基于云技术，提供创新、安全、高效、多终端的协同设计功能，支持团队协同、安全协同、高效协同、多终端协同，真正实现一站式研发设计和管理服务。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/f3b44642-04dc-4035-afa4-3591dff7c999.jpg@s_2,w_820,h_547" w="1440" h="960" title="国内首款基于云架构三维 CAD 平台 CrownCAD 正式发布" srcset="https://img.ithome.com/newsuploadfiles/2021/9/f3b44642-04dc-4035-afa4-3591dff7c999.jpg 2x" width="1440" height="547" referrerpolicy="no-referrer"></p><p>CrownCAD App 是一款面向通用制造、设计行业用户的移动端应用程序，可配合 CrownCAD Web 端进行在线浏览、协同设计。CrownCAD App 提供用户登录、模型导入、精确测量、显示渲染、第三方分享等功能，目前已在主流应用商店上架。</p><p>CrownCAD 可以在小到手机、大到超级服务器上运行，可以满足个人用户、中小企业、高端制造的需求，如此强大的功能如何让更多人知道，市场推广显得尤为重要。梅敬成博士从“高端制造、中小企业、合作共赢、生态建设”四个方面介绍 CrownCAD 的应用和推广。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/906b2ef7-c7a0-4871-a133-9a9b39e5c84e.png" w="1439" h="814" title="国内首款基于云架构三维 CAD 平台 CrownCAD 正式发布" width="1439" height="464" referrerpolicy="no-referrer"></p><p>对于高端制造，着重攻克核心技术“卡脖子”、大场景模型“转不动”、协同设计“缺平台”等关键问题。中国航空制造技术研究院 、航天一院、商飞集团下的多个单位对 CrownCAD 的“数据交换、模型创建”能力给予了积极的评价，对“基于唯一数据源的协同设计”非常认可，对“基于服务器的大型场景渲染”非常期待。</p><p>对于中小企业，CrownCAD 提供性价比最优的国产自主三维 CAD 软件，满足企业的正版化需求，助力广大的制造业企业创新和转型升级。多维激光总经理黄国斌表示：CrownCAD 的零件、装配、工程图等传统设计模块满足我们设计的需求，同时，不同于传统的 PC 软件，CrownCAD 提供的协同设计功能，更加便于我们直接与供应商、制造商之间的交流。CrownCAD 操作界面十分友好，简单易上手，关键是功能与国外软件一样但价格适宜。CrownCAD，作为一款国产自主的三维 CAD，为我们中小制造企业带来了希望。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/67fe4b97-975b-418b-94f5-5707c99724b4.png" w="1438" h="810" title="国内首款基于云架构三维 CAD 平台 CrownCAD 正式发布" width="1438" height="462" referrerpolicy="no-referrer"></p><p>对于 CAE/CAM/BIM 领域的开发者，CrownCAD 的 DGM 功能和性能对标国际主流建模引擎，并提供与它们兼容的 API 接口，实现合作共赢。中国航空发动机研究院周帅工程师、西安电子科技大学张玉教授、苏州千机智能赵建军总经理纷纷表达了 CrownCAD 在科研、教学、高科技企业带来的惊喜，并期待能够持续密切合作，共同打造世界级的国产 CAD/CAM 工业软件。</p><p>另外，CrownCAD 基于云架构的三维参数化建模、多终端运行；无需安装、协同交流、集中管理；自主三维 CAD 平台、云端运行等特征便于形成生态建设圈。中源广科的张麟博士称自己是 CrownCAD 的忠实粉丝，在 CrownCAD 的平台上，各个行业的工程师，通过 API 打造自己的细分需求的工具，并进行细分市场的服务；华为云张亮从工业互联网生态共建角度表达了与 CrownCAD 合作带来的共赢效应；仲恺农业工程学院翟橙老师、中国海洋大学刘永辉博士表达了无需安装的 CrownCAD 进入课堂带来的“轻应用”优势。</p><p>对于云化 CAD 产品，发布会还对外宣布价格政策：个人用户标准版、企业用户标准版永久免费，CrownCAD App 永久免费, 这预示着 CrownCAD 支持正版化并真正成为人人用得起的软件。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/9/908bb606-dc5e-430a-b01d-094b03404314.jpg@s_2,w_820,h_547" w="1440" h="960" title="国内首款基于云架构三维 CAD 平台 CrownCAD 正式发布" srcset="https://img.ithome.com/newsuploadfiles/2021/9/908bb606-dc5e-430a-b01d-094b03404314.jpg 2x" width="1440" height="547" referrerpolicy="no-referrer"></p><p>华天软件创始人/董事长杨超英表示：工业软件已被列为当前科技攻关最紧急、最迫切的问题，关乎国家急迫需要和长远需求，与基础原材料和高端芯片等并列，彰显了国家对于工业软件的高度重视。CrownCAD 建立在我们自己的几何造型引擎上，承担了国家的云三维核心引擎课题，CrownCAD 不仅是服务模式的改变，更能够解决传统 CAD 解决不了的问题，CrownCAD 实现的科技突破在国际上也是独一无二的成就。</p><p>梅敬成博士用“无处不在，人人能用”八个字来总结 CrownCAD。梅敬成博士希望 CrownCAD 成为中国工程师、设计师的设计利器，也希望中国的制造业不再承受来自三维 CAD 软件的版权、价格、运行设备等因素的压力，放心地、无所顾及地只专注于产品设计本身。</p><p>CrownCAD 这颗中国工业软件新星的成功面世，注定为中国制造业注入崭新活力，正如梅敬成博士所言：“发布会只是一个起点，我们的目标是三年做到三维 CAD 用户数全国第一。”</p>
