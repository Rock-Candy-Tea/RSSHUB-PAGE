class="accentTextColor">自己做了个新版本并开源</span>，受到了其他家长们的一致好评，安卓商店评分 4.2 星。</p><p data-vmark="6093"><img src="https://img.ithome.com/newsuploadfiles/2021/11/b2e09770-24e1-4b9f-a78f-4318e08e50ce.png" class="accentTextColor">不仅说家长们涉嫌非法访问信息数据</span>，并且要警方介入调查此事。</p><p data-vmark="dcc2">还是那句话：</p><p data-vmark="b9d3">离谱他妈给离谱开门，离谱到家了。</p><h2 data-vmark="7ea5">码农爸妈手动自制新版本</h2><p data-vmark="0103">如上提到的这个学校 App，<span class="accentTextColor">是瑞典首都斯德哥尔摩政府推出的一个平台 Skolplattformen</span>。</p><p data-vmark="ff91"><img src="https://img.ithome.com/newsuploadfiles/2021/11/688effd0-57c5-41bd-993d-20d37ff664d3.png" w="1080" h="640" title="学校花重金打造的 App 太难用：国外程序员爸妈们自己做出开源程序，学校却称其会非法访问个人信息" width="1080" height="486" referrerpolicy="no-referrer"></p><p data-vmark="54f7">它主打的功能就是让家长们更好了解自己娃在学校的生活，比如学校最近有什么活动、明天要上什么课、老师的一些通知等等，包括请假一些事宜也是在这个 App 上来办。</p><p data-vmark="07fa">官方声称打造该平台耗资约 1.17 亿美元（人民币约 7.5 亿）。</p><p data-vmark="9dbc">它覆盖了斯德哥尔摩市 600 所幼儿园、177 所中小学，老师、学生、家长都可以登录使用。</p><p data-vmark="a285">听起来这个 App 非常智能，但实际用起来却相当智障。</p><p data-vmark="925c">从家长们的吐槽中就能感受一二：</p><blockquote><p data-vmark="969b">导航很难用，菜单栏没有用。老师让我们打开通知提醒，但是我不知道怎么打开。</p></blockquote><p data-vmark="a718"><img src="https://img.ithome.com/newsuploadfiles/2021/11/031c5ee6-1845-4b76-a1c5-48ee1f5ebda9.png" w="1080" h="229" title="学校花重金打造的 App 太难用：国外程序员爸妈们自己做出开源程序，学校却称其会非法访问个人信息" width="1080" height="174" referrerpolicy="no-referrer"></p><blockquote><p data-vmark="8d41">UI 非常差，查找数据的逻辑有问题，最大的问题是它太卡了。</p></blockquote><p data-vmark="9cc3"><img src="https://img.ithome.com/newsuploadfiles/2021/11/94770340-ee6c-492f-963b-a7ba35a129fe.png" w="1080" h="182" title="学校花重金打造的 App 太难用：国外程序员爸妈们自己做出开源程序，学校却称其会非法访问个人信息" width="1080" height="138" referrerpolicy="no-referrer"></p><p data-vmark="77fb">显然，是平台的前端做的太差了。</p><p data-vmark="96f8">有一位家长甚至发推特说，自己要戴着印有“垃圾 Skolplattformen”的棒球帽去接孩子放学。</p><p data-vmark="629d"><img src="https://img.ithome.com/newsuploadfiles/2021/11/a5f608eb-5f12-4100-b031-8fe148593876.png" w="1080" h="1252" title="学校花重金打造的 App 太难用：国外程序员爸妈们自己做出开源程序，学校却称其会非法访问个人信息" width="1080" height="951" referrerpolicy="no-referrer"></p><p data-vmark="fdef">也正是这位老哥，打响了改造 Skolplattformen 的“第一枪”。</p><p data-vmark="c766">他叫 Landgren，是一位程序员，也是 3 个孩子的父亲，还是瑞典创新咨询公司 Iteam 的 CEO。</p><p data-vmark="98df">在去年 10 月，他给市政府写信，请求查看该平台的 API。</p><p data-vmark="b853">在等待回信的期间，老哥登录了自己的账户，想看看这个系统是不是可以逆向工程。</p><p data-vmark="e51a">没想到鼓捣了几个小时，就已经能看到学校平台上的一些信息了。</p><p data-vmark="da69">干脆一不做二不休，老哥打算在原有的垃圾 API 上，构建一个新的 API 出来。</p><p data-vmark="8cb6">在之后的几周里，他联合了另外两位码农家长 Johan Öbrink 和 Erik Hellman，三人决定创建一个 Skolplattform 的新版本，并将其作为应用程序发布，让更多的家长使用。</p><p data-vmark="aef0">通过 Chrome 开发者工具，他们记下了 Skolplattform 的所有 URL 和有效负载，并获取了调用该平台的 API 代码并构建了一个包，以便它可以在手机上运行。</p><p data-vmark="e94f">简而言之，<span class="accentTextColor">就是在现有的 Skolplattform 上再创建一个层</span>。</p><p data-vmark="a74b">Öppna Skolplattformen 就这样诞生了。</p><p data-vmark="dfc8">它于今年 2 月份发布，所有代码都已在 GitHub 上开源。</p><p data-vmark="7b97">目前，只有家长能使用这个新程序，功能和原有平台一致，用原有 ID 登录即可。</p><p data-vmark="8c1f">其售价为 <span class="accentTextColor">1 欧元 （折合人民币约 7.4 元）</span>，下载量达 12500 次。</p><h2 data-vmark="e1bb">官方要报警</h2><p data-vmark="cc69">可是这事儿，却让当地政府不开心了。</p><p data-vmark="bb30">政府表示，“你们这是非法行为！”</p><p data-vmark="0ecc">接下来的 8 个月里，市政府都在试图破坏或关闭这个开源程序，并多次警告家长们不要再使用它，并声称该程序会非法访问个人信息。</p><p data-vmark="565f">于是，<span class="accentTextColor">一场市政府和家长们的猫鼠游戏就开始了</span>。</p><p data-vmark="5e99">2 月底，官方版本更新，阻止任何潜在个人数据被访问，让家长自制版 API 一度不能工作。</p><p data-vmark="830a">之后自制版也开始更新回应“阻击”，3 月份更新了 7 次。</p><p data-vmark="81dd">官方这边当然也没歇着，他们在继续改进自己的底层系统。</p><p data-vmark="3bd5">4 月份，甚至连警方都介入了此事。</p><p data-vmark="ee03">官方以数据泄露为由，要求警方调查 Landgren 等开发人员及开源程序的工作原理。</p><p data-vmark="ed48">Landgren 则表示他们访问的数据都是<span class="accentTextColor">公开信息</span>，“<span class="accentTextColor">所显示的学生成绩来自程序中的内嵌<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>，而程序本身是无法访问任何数据的</span>。”</p><p data-vmark="9b93">并且老哥还表示，他们之前与市政府沟通时表示，官方可以使用新程序来代替原有版本。不过市政府却对此置之不理，依旧选择报警处理。</p><p data-vmark="2c9c">但这事儿似乎是官方自己打脸了。</p><p data-vmark="e8ca">8 月份，警方证实了 Landgren 一行人的清白。</p><p data-vmark="80d6">他们表示，Öppna Skolplattformen 使用的所有信息都是公开信息，也未对用户隐私构成威胁。</p><p data-vmark="df37">9 月，市政府发布声明表示，Öppna Skolplattformen 可以建立许可证，这相当于对它已经认可了。</p><p data-vmark="1e4c">但在具体执行上，似乎还没什么动静，甚至就在不久前再次阻止了 Öppna Skolplattformen 的访问。</p><p data-vmark="e16c">对此，Landgren 老哥还是希望能够与市政府达成协议，让政府支付程序的许可费用，最终让所有家长都能免费使用这个 App。</p><h2 data-vmark="1344">One More Thing</h2><p data-vmark="0cf3">说到这种学校 App，国内其实也有不少学校在使用，而且也有类似令人诟病的情况。</p><p data-vmark="a1ac">比如今年 9 月南方都市报曝出，广东阳江的一所寄宿制民办小学要求家长必须安装一个需要支付年费的 App，才能了解孩子在学校的情况。</p><p data-vmark="82d0">但问题是，许多家长反映这个 App 用了一年，很多功能点进去还是没反应，比如校园相册、作业管理、课程中心等十余种功能。</p><p data-vmark="4549">而能使用的只有“饭堂缴费”，并且如果不交 198 元的年费，这些功能也用不了。</p><p data-vmark="cdb1">嗯，还真是世界之大无奇不有。</p>
