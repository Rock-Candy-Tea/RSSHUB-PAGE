<p data-vmark="8a22"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 28 日消息，RISC-V 软硬件厂商赛昉科技近期发布两款新品：全球首款量产高性能 RISC-V 多媒体处理器 —— 昉・惊鸿 7110（JH7110），和全球性能最高的量产 RISC-V 单板计算机 —— 昉・星光 2（VisionFive 2）。</p><p data-vmark="b669">官方介绍，作为一款高性能、低功耗、多功能的基于 RISC-V 的应用处理器芯片，JH7110 采用成熟台积电 28nm 工艺，搭载 64 位高性能四核 RISC-V CPU，工作频率 1.5GHz，2MB 的二级缓存。<span class="accentTextColor">JH7110 集成 3DGPU、H.264 / H.265 视频编解码 IP 及 ISP IP，是一款功能强大的多媒体处理平台。</span></p><p data-vmark="daac">赛昉科技根据实测，JH7110 CPU 稳定工作频率达 1.5GHz，Coremark 跑分达到 5.09，是目前市场上 RISC-V 量产芯片中性能最优的产品。此外，JH7110 提供更加强大的 GPU 处理能力，在 400MHz 的 GPU 工作主频下，使用业界标准的 GPU benchmark – GLmark2 测试，相比单板计算机领域应用广泛的主控芯片，JH7110 的实测数据达到其 4 倍以上。</p><p data-vmark="c609">赛昉科技表示，功耗方面，JH7110 被划分为 8 个可独立开关的电源域，CPU 频率可通过软件调节，客户可完全按照应用场景和性能需求设置最有效的芯片工作场景，以实现 PPA 平衡。休眠状态下，JH7110 功耗为 120mW；在单板计算机应用场景，JH7110 满负荷工作，动态功耗为 4100mW；在软路由、NAS 的应用场景，用户不需要 GPU 和视频编码，而需要双网口工作，用户可通过软件控制模块开关，此时实际功耗会下降到 3300mW。</p><p data-vmark="3a9a">赛昉科技官方介绍，JH7110 拥有强大的多媒体支持能力。JH7110 集成了赛昉科技自研的低功耗 ISP，提供强大的 H.264 / H.265 编解码能力，支持双路 1080P@30fps 编码，1 路 4K@60fps 或 4 路 1080P@30fps 解码；支持 JPEG 编解码；显示输出可支持 2 路 4K 双屏显，6 种图层叠加显示；集成的 GPU 渲染能力可达 2400MPixels / s。通过更快的响应速度和更流畅的图像处理，JH7110 可以给游戏玩家带来好的游戏体验，另外，普通用户也能够在常用的桌面环境，如 Debian、OpenHarmony 中获得流畅的使用体验。</p><p data-vmark="8191">基于 JH7110，赛昉科技发布了全球首款高性能、低成本且集成 3D GPU 的量产 RISC-V 单板计算机 ——VisionFive 2。</p><p data-vmark="ff00">赛昉科技表示，VisionFive 2 搭载 JH7110 多媒体处理器，拥有较强的计算能力和图形处理能力；配有完整的音视频接口：DVP、MIPI、HDMI、音频立体声接口；两个网络接口；40-pin 全功能 GPIO，可兼容各种主流开发板的通用外设模组；同时提供 PCIE、USB、SDIO 外设的扩展接口。</p><p data-vmark="8686">据赛昉科技介绍，目前，在 VisionFive 系列开发板上，已移植完成 Linux 官方发行版 Ubuntu、Fedora、OpenSUSE、Debian、ArchLinux、AltLinux、Gentoo、Lubuntu 等操作系统，正在移植的有 OpenHarmony、Deepin、优麒麟等。此外，VisionFive 系列开发板已全面支持 Python，JAVA 等高级语言，同时，支持 Javascript 编程的方舟编译器很快将在 OpenHarmony 操作系统下问世。</p><p data-vmark="464e">IT之家获悉，VisionFive 2 涵盖 2GB、4GB、8GB 三个版本，对应价格分别为 55 美元、65 美元、85 美元。此外，赛昉科技还将开启为期两个月的众筹让利活动，用户最高可享七五折优惠。</p><p data-vmark="1a78" style="text-align: center;"><img
