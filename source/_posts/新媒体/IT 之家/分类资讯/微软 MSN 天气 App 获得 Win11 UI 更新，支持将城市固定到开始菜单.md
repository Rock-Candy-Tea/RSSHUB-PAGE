<p data-vmark="19be"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 3 日消息，微软已为 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> Insiders Dev 频道发布了 <span class="accentTextColor">MSN 天气 App 的 4.53.33420.0 版本</span>，并带来了改进的设计和开始菜单的改进。</p><p data-vmark="fca9"><img src="https://img.ithome.com/newsuploadfiles/2022/1/fa1ebe0a-f21b-4923-807c-2ee048d43f58.jpg@s_2,w_820,h_546" w="1440" h="959" title="微软 MSN 天气 App 获得 Win11 UI 更新，支持将城市固定到开始菜单" srcset="https://img.ithome.com/newsuploadfiles/2022/1/fa1ebe0a-f21b-4923-807c-2ee048d43f58.jpg 2x" width="1440" height="546" referrerpolicy="no-referrer"></p><p data-vmark="1a6c">从截图可以看到，新版的 MSN 天气采用了<span class="accentTextColor">更符合 Windows 11 风格的 UI</span>，在左侧边栏的变化比较明显。</p><p data-vmark="5f41"><img src="https://img.ithome.com/newsuploadfiles/2022/1/56aa3cbb-4354-4d12-b6b0-0d5fe8a021ce.png" w="1282" h="1002" title="微软 MSN 天气 App 获得 Win11 UI 更新，支持将城市固定到开始菜单" width="1282" height="641" referrerpolicy="no-referrer"></p><p data-vmark="c7f0">此外，新版本支持将某个城市的天气图标固定到 Windows 11 开始菜单，<span class="accentTextColor">用户点击就可以直接查看该城市的天气</span>。</p><p data-vmark="2e24"><img src="https://img.ithome.com/newsuploadfiles/2022/1/893e710a-cf40-4b21-bd59-75c8a49af29b.jpg@s_2,w_820,h_365" w="1384" h="616" title="微软 MSN 天气 App 获得 Win11 UI 更新，支持将城市固定到开始菜单" srcset="https://img.ithome.com/newsuploadfiles/2022/1/893e710a-cf40-4b21-bd59-75c8a49af29b.jpg 2x" width="1384" height="365" referrerpolicy="no-referrer"></p><p data-vmark="7f34">IT之家了解到，新版本更新日志如下：</p><ul class=" list-paddingleft-2"><li><p data-vmark="4857">新设计：天气 App 已更新，现在使用更符合 Windows 11 界面的图形样式。按钮现在具有圆形边缘，字体变得更大更粗，并且设置页面上的控件已更新。</p></li><li><p data-vmark="0934">开始菜单中的位置：将其他位置固定到开始菜单时，将显示相应的城市名称，以便用户更轻松地识别天气 App 所指的链接。</p></li><li><p data-vmark="d809">各种错误修复和改进。</p></li></ul>
