<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 7 日消息 微软上周向所有 Windows 用户发布了一则警告，称其 Windows 后台打印程序存在一个未修补的严重 bug，这一漏洞被称为 PrintNightmare，而且正在被广泛利用。</p><p>IT之家提醒，该漏洞正在被一些恶意攻击者恶意利用 ，他们可藉此实现本地特权和远程代码执行，甚至可能会远程向你 PC 上安装程序、修改数据或创建具有最高管理员权限的帐户，危险程度拉满。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/5d0b9424-5315-45c3-9bc2-9fc5bdd62819.png" w="436" h="327" title="微软现发布紧急更新，力求修复 Win7/8/8.1/10 零日漏洞" width="436" height="327" referrerpolicy="no-referrer"></p><p>工作日来临后，微软开发人员开始奋力对其进行修复，微软今日向 Win10 用户推送了 KB5004945 紧急更新，其他版本代码如下：</p><ul class=" list-paddingleft-2"><li><p>版本 21H1、20H1、2004 – KB5004945。</p></li><li><p>版本 1909 – KB5004946。</p></li><li><p>版本 1809 和 Windows Server 2019 – KB5004947。</p></li><li><p>版本 1803 – KB5004949。</p></li><li><p>版本 1507 – KB5004950。</p></li><li><p>Windows 8.1 和 Windows Server 2012 – KB5004954 和 KB5004958（仅限安全补丁）。</p></li><li><p>Windows 7 SP1 和 Windows Server 2008 R2 SP1 – KB5004953 和 KB5004951（仅安全）</p></li><li><p>Windows Server 2008 SP2 – KB5004955 和 KB5004959（仅安全）。</p></li><li><p>Windows 10 KB5004945 下载直链：<a href="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5004945" target="_blank">64 位和 32 位 (x86)</a>。</p></li></ul><p>这里建议各位小伙伴尽快更新，安装后系统内部版本号将提升至 19043.1083（19042.1083 或 19041.1083）。如果你打定主意必须要跳过今天的更新，你还可以在下周的周二更新（7 月 13 日）中收到相同的推送。</p><p>对于其余受支持的 Windows 版本，微软计划在未来几天内发布紧急补丁。若实在担心可参考以下步骤（禁用 Windows Print Spooler 服务）：</p><ol class="ai-word-checked list-paddingleft-2"><li><p>打开 Windows 搜索。</p></li><li><p>输入 PowerShell 并以“管理员”身份运行它。</p></li><li><p>输入以下命令：</p></li><li><pre class="brush:javascript;toolbar:false">Stop-Service -Name Spooler -Force</pre></li><li><p><strong><img src="https://img.ithome.com/newsuploadfiles/2021/7/fb253a55-c483-4d5d-93ca-a86c12da9843.jpg@s_2,w_820,h_174" alt="Windows PrintNightmare 错误" w="981" h="208" title="微软现发布紧急更新，力求修复 Win7/8/8.1/10 零日漏洞" srcset="https://img.ithome.com/newsuploadfiles/2021/7/fb253a55-c483-4d5d-93ca-a86c12da9843.jpg 2x" width="981" height="174" referrerpolicy="no-referrer"></strong></p></li><li><p>按下回车键。</p></li><li><p>输入以下命令：</p></li><li><pre class="brush:javascript;toolbar:false">Set-Service -Name Spooler -StartupType Disabled</pre></li><li><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/11a5bf92-076f-440b-a2f4-48f2d044aa54.jpg@s_2,w_820,h_160" alt="禁用打印后台处理程序" w="981" h="191" title="微软现发布紧急更新，力求修复 Win7/8/8.1/10 零日漏洞" srcset="https://img.ithome.com/newsuploadfiles/2021/7/11a5bf92-076f-440b-a2f4-48f2d044aa54.jpg 2x" width="981" height="160" referrerpolicy="no-referrer"></p></li><li><p>按回车键。</p></li></ol><p>当你成功运行上述两个命令后，Windows 即可禁用 Print Spooler 服务。当然，我们还是建议你更新一下，毕竟禁用系统服务后偶尔会出现一些玄学问题。</p><p>值得一提的是，Windows 后台打印处理程序服务的漏洞一直是令系统管理员最为头疼的问题之一。例如历史上最臭名昭著的例子 Stuxnet 病毒。</p><p>十多年前，Stuxnet 就曾利用多个零日漏洞 (包括 Windows Print Spooler 漏洞) 摧毁了伊朗 60% 的计算机，并以之摧毁其工业基础设施，打击核设施建设，几乎以一己之力拉响安全网络战和新型军备竞赛。</p><p><img
