<p data-vmark="a547"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 21 日消息，微软现已推出了可完全在<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>中运行的轻量级 VS Code 网页预览版，<span class="accentTextColor">该程序可以允许开发者直接使用浏览器中的 VS Code，无需使用本地安装版本进行开发。</span></p><p data-vmark="6289"><img src="https://img.ithome.com/newsuploadfiles/2021/10/c75a8eb0-dce4-498e-912b-7d97fe822f01.png" w="1440" h="885" title="微软发布可完全在浏览器中运行的轻量级 VS Code 工具" width="1440" height="504" referrerpolicy="no-referrer"></p><p data-vmark="51cd">微软负责人提出了几种开发者可以使用的功能，其中包括：本地文件的查看和编辑，以及快速笔记和 Markdown 预览；构建 HTML、JavaScript 和 CSS 应用程序的客户端，并可结合浏览器工具进行调试；在不方便安装 VS Code 的机器上进行代码编辑，比如 Chromebook 甚至是 <a class="s_tag" href="https://ipad.ithome.com/" target="_blank">iPad</a>。</p><p data-vmark="fbfa">VS Code 网页预览版可以在支持系统文件访问 API 的浏览器上运行，<span class="accentTextColor">比如微软的 Edge 浏览器以及谷歌的 Chrome 浏览器</span>。而如果该用户使用的浏览器暂不支持系统文件访问 API 的特性，那用户可以选择将文件上传或下载至浏览器以进行开发工作。</p><p data-vmark="a61a"><img src="https://img.ithome.com/newsuploadfiles/2021/10/6a8de7be-361e-48f7-a8f7-5d12b7c80fe4.jpeg" w="974" h="701" title="微软发布可完全在浏览器中运行的轻量级 VS Code 工具" width="974" height="590" referrerpolicy="no-referrer"></p><p data-vmark="2d7c">IT之家了解到，微软在与网站版 VS Code 有关的博客文章中总结道：“将 VS Code 成功植入浏览器是对该开发套件最初愿景的实现。对于 VS Code 也是一个全新的开始。人们只需要一台能联网的电脑和一个浏览器就可以在未来实现开发，在那种环境下人们可以随时随地的开发代码。”</p>
