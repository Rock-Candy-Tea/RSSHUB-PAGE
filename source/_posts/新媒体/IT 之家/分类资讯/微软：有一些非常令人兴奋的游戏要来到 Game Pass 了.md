<p data-vmark="f845"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 2 月 23 日消息，在哥伦比亚波哥大最近的一次活动中，Xbox 游戏营销总经理 Aaron Greenberg 回答了粉丝关于 Xbox 品牌的问题。其中之一涉及动视暴雪，它们被微软收购，并将成为 Xbox 游戏工作室的一部分。</p><p style="text-align: center;" data-vmark="7cc9"><img src="https://img.ithome.com/newsuploadfiles/2022/1/01baecd1-674b-4576-87b3-927e43b8239b.jpg@s_2,w_820,h_461" alt="收购动视暴雪" title="微软：有一些非常令人兴奋的游戏要来到 Game Pass 了" referrerpolicy="no-referrer"></p><p data-vmark="2654">虽然 Greenberg 表示“由于保密问题”不能透露太多，但他承诺 <span class="accentTextColor">Game Pass 将有一些“非常令人兴奋的事情”</span>。</p><blockquote><p data-vmark="9f05">由于保密问题，我对此不能说太多。但我可以告诉你，Xbox 和 Xbox Game Pass 即将推出一些令人兴奋的事情。</p></blockquote><p style="text-align: center;" data-vmark="408d"><img src="https://img.ithome.com/newsuploadfiles/2022/2/df996972-5e18-48f4-921f-67687c9d2be0.png" w="1440" h="807" title="微软：有一些非常令人兴奋的游戏要来到 Game Pass 了" width="1440" height="460" referrerpolicy="no-referrer"></p><p data-vmark="f0f3">从对动视暴雪的收购来看，微软的 Game Pass 大概率会将《使命召唤》《暗黑破坏神》等包揽其中，也可能会让 Xbox 提前测试《守望先锋 2》等。</p><p data-vmark="204a">IT之家了解到，Greenberg 还证实，《战争机器》工作室 The Coalition 正在开展“<span class="accentTextColor">几个未宣布的项目</span>”。</p><p data-vmark="10e1">▼ Aaron Greenberg</p><p style="text-align: center;" data-vmark="b30b"><img src="https://img.ithome.com/newsuploadfiles/2022/2/a52c24f1-f0ca-429e-991e-ba2ba635875e.png" w="609" h="540" alt="Aaron Greenberg" title="微软：有一些非常令人兴奋的游戏要来到 Game Pass 了" width="609" height="540" referrerpolicy="no-referrer"></p><p data-vmark="7dc5">此前，微软 Xbox 业务负责人菲尔・斯宾塞（Phil Spencer）谈到与动视的交易时表示：“我们将竭尽所能，将尽可能多的动视暴雪添加到 Xbox Game Pass 和 PC Game Pass 之中，包括新游戏和动视暴雪的经典游戏。”</p>
