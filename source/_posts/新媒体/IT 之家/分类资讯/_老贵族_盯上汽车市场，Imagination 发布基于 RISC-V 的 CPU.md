<p data-vmark="1a4b">12 月 7 日消息，曾经的移动 GPU 大厂 Imagination Technologies <span class="accentTextColor">正式发布基于 RISC-V 的 Catapult（中文是投石机的意思）系列 CPU</span>，其是 Imagination 的异构计算解决方案之一，目标是适应 5G 调制解调器、存储、先进驾驶辅助系统（ADAS）/自动驾驶车辆、数据中心和高性能计算等各种异构计算场景。</p><p data-vmark="2e79">Catapult 系列 CPU 现在有 4 款：</p><ul class=" list-paddingleft-2"><li><p data-vmark="532f">动态微控制器，已经出货，已被客户用在 SoC 中的高性能汽车 GPU 上；</p></li><li><p data-vmark="af74">实时嵌入式 CPU，已上市；</p></li><li><p data-vmark="d6d0">高性能应用处理器 CPU，2022 年上市；</p></li><li><p data-vmark="6b2e">支持汽车功能安全的 CPU，2022 年上市，根据 ISO 26262 车规标准开发，带有功能安全性和安防性能。宣称可为客户去达到每个汽车安全完整性等级（Automotive Safety Integrity       Level，ASIL）而提供一系列 CPU 解决方案。</p></li></ul><p data-vmark="2294"><img src="https://img.ithome.com/newsuploadfiles/2021/12/ddbc105b-cafa-4453-8e93-213251684de6.jpg" w="740" h="742" title="“老贵族”盯上汽车市场，Imagination 发布基于 RISC-V 的 CPU" width="740" height="742" referrerpolicy="no-referrer"></p><p data-vmark="0d21">▲ 图源 Imagination 官网</p><p data-vmark="936a">Catapult 系列 CPU 基于 RISC-V 开源架构，全系均是多线程架构，有 32 位和 64 位版本，提供了大量的可选项，能根据应用需求具体配置，从而调整性能、能效比。极限规格可以扩展到每个集群 8 个非对称相干内核，加速器也能自定义添加。</p><p data-vmark="594a">Imagination 还发布了与 Catapult CPU 配套的 SDK 软件开发工具包和 Catapult  Studio。SDK 包括了 GCC、LLVM 和 GDB、优化的 C 库等构建和调试工具，有 Windows、Ubuntu、CentOS、MacOS、FreeRTOS 等操作系统，也可支持 Linux 的引导加载程序、内核和基于 Yocto 的文件系统。</p><p data-vmark="430b">而 Catapult Studio 是 Imagination 的 IDE 集成开发环境，以 Visual Studio  Code 为基础，用于嵌入式、RISC-V 开发，有测试功耗和能效的评价模型，可提供交互式调试，兼容 gem5 仿真器。</p><p data-vmark="c1ea">Imagination 是家总部在英国的半导体 IP 公司，它最为人熟知的，应该是它在苹果 A 系列处理器中的 PowerVR 系列 GPU（<a class="s_tag" href="https://android.ithome.com/" target="_blank">安卓</a>阵营的 SoC 也有使用，但比例远不如当年）。苹果在 2017 年走上自研 GPU 的道路，但 2020 年初又和苹果达成多年期授权协议。Catapult 系列 CPU 的新闻稿中还有 RISC-V 国际基金会、瑞萨电子汽车 SoC 开发部、林利集团分析师的站台。这次算是“老贵族”盯上“新市场”了。</p>
