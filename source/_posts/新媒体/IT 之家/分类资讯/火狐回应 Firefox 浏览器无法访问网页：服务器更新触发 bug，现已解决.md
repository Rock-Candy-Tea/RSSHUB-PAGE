            <p data-vmark="871a"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 1 月 13 日消息，火狐官方今日发布了 Firefox 96 版本，但有大批量网友反馈称更新后出现无法上网的情况，目前官方表示该问题已解决。</p><p data-vmark="b817"><img src="https://img.ithome.com/newsuploadfiles/2022/1/ef826db7-e9c9-474f-b5da-e733d10991c8.png" w="654" h="362" alt="截图" title="火狐回应 Firefox 浏览器无法访问网页：服务器更新触发 bug，现已解决" width="654" height="362" referrerpolicy="no-referrer"></p><p data-vmark="d119"><img src="https://img.ithome.com/newsuploadfiles/2022/1/90b48078-6e9b-4381-832b-44905af33c42.png" w="642" h="80" alt="截图" title="火狐回应 Firefox 浏览器无法访问网页：服务器更新触发 bug，现已解决" width="642" height="80" referrerpolicy="no-referrer"></p><p data-vmark="e96f">官方解释称：Firefox <a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>下午无法访问网页的问题，是与 http / 3 相关的 bug 被某些服务器的更新触发所导致，现已解决。</p><p data-vmark="37de"><img src="https://img.ithome.com/newsuploadfiles/2022/1/3328c092-c337-4f0a-aac3-94818e33517f.png" w="581" h="318" title="火狐回应 Firefox 浏览器无法访问网页：服务器更新触发 bug，现已解决" width="581" height="318" referrerpolicy="no-referrer"></p><p data-vmark="cb4d">IT之家了解到，HTTP / 3 是第三个 HTTP 协议主要版本。该版本与其前任 HTTP / 1.1 和 HTTP / 2 不同，在 HTTP / 3 中将弃用 TCP 协议，改为使用基于 UDP 协议的 QUIC 协议实现。</p>
