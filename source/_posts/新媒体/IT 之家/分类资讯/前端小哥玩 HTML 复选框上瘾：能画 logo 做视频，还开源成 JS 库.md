data-vmark="4ead">1.做出原本的图像。</p><p data-vmark="ffaa">2.将图像转化为 ASCII 文本输出。</p><p data-vmark="ff0b">以水波为例，首先要生成这样动态的水波。</p><p data-vmark="1a5b"><img src="https://img.ithome.com/newsuploadfiles/2021/10/2a7e057f-1920-4070-89a5-5c4e3223b3a7.gif" w="594" h="596" title="前端小哥玩 HTML 复选框上瘾：能画 logo 做视频，还开源成 JS 库" width="594" height="596" referrerpolicy="no-referrer"></p><p data-vmark="3e01">想要生成它，需要以中心为原点，在 xy 平面上建立正弦函数。</p><p data-vmark="aceb">z 轴垂直屏幕向外，把 z 轴的数值转化为灰度，白色为波峰，黑色为波谷。</p><p data-vmark="7033"><img src="https://img.ithome.com/newsuploadfiles/2021/10/fe784944-0912-4e10-80be-86b812c3ed1f.png" w="784" h="786" title="前端小哥玩 HTML 复选框上瘾：能画 logo 做视频，还开源成 JS 库" width="784" height="786" referrerpolicy="no-referrer"></p><p data-vmark="f959">然后在图形计算器 desmos 上让水波动起来，这样第一步就完成了。</p><p data-vmark="0c19"><img src="https://img.ithome.com/newsuploadfiles/2021/10/61373d59-5201-49d4-964e-7952caaf6573.gif" w="727" h="533" title="前端小哥玩 HTML 复选框上瘾：能画 logo 做视频，还开源成 JS 库" width="727" height="533" referrerpolicy="no-referrer"></p><p data-vmark="6ed0">第二步，将第一步的成果转化为 ASCII 码输出。</p><p data-vmark="6a55">这一步的转化主要涉及到将彩色对应灰度。</p><p data-vmark="692a">采用这个公式，即使是彩色的图片，也只不过是五彩斑斓的灰罢了~</p><p data-vmark="3a71">GrayScale = 0.21 R + 0.72 G + 0.07 B</p><p data-vmark="204e">提取原图的 RGB 色彩，输出为灰度：</p><pre class="brush:javascript;toolbar:false">const toGrayScale = (r, g, b) => 0.21 * r + 0.72 * g + 0.07 * b;
const convertToGrayScales = (context, width, height) => &#123;
const imageData = context.getImageData(0, 0, width, height);
const grayScales = [];
for (let i = 0 ; i < imageData.data.length ; i += 4) &#123;
const r = imageData.data[i];
const g = imageData.data[i + 1];
const b = imageData.data[i + 2];
const grayScale = toGrayScale(r, g, b);
imageData.data[i] = imageData.data[i + 1] = imageData.data[i + 2] = grayScale;
grayScales.push(grayScale);
&#125;
context.putImageData(imageData, 0, 0);
return grayScales;
&#125;;</pre><p data-vmark="9ec6">然后为每个像素赋灰度值：</p><pre class="brush:javascript;toolbar:false ai-word-checked">const grayRamp = '<a href="https://www.ithome.com/cdn-cgi/l/email-protection" class="__cf_email__" data-cfemail="65412527">[email protected]</a>%8&WM#*oahkbdpqwmZO0QLCJUYXzcvunxrjft/|()1&#123;&#125;[]?-_+~<>i!lI;:,"^`\'. ';
const rampLength = grayRamp.length;
const getCharacterForGrayScale = grayScale => grayRamp[Math.ceil((rampLength - 1) * grayScale / 255)];
const asciiImage = document.querySelector('pre#ascii');
const drawAscii = (grayScales) => &#123;
const ascii = grayScales.reduce((asciiImage, grayScale) => &#123;
return asciiImage + getCharacterForGrayScale(grayScale);
&#125;, '');
asciiImage.textContent = ascii;
&#125;;</pre><p data-vmark="00e9">最后调整一下图片大小就大功告成了~ 更多详细内容见文后链接~</p><p data-vmark="0761"><img src="https://img.ithome.com/newsuploadfiles/2021/10/74586952-f3cf-4cb5-a6af-3c1f10e322ee.gif" w="1064"
