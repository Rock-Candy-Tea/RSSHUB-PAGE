<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 5 月 1 日消息 在今年一月份的时候，谷歌推出了一项取代第三方 Cookie 的技术，名叫 Federated Learning of Cohorts (FLoC)，谷歌表示，该技术将解决第三方 Cookie 困扰人们已久的隐私问题。然而，这项技术的推进可能没那么顺利。</p><p>近日，GitHub 宣布<span class="accentTextColor">在所有 GitHub Pages 网站上推出一个新的 HTTP 标头，该标头可使网站所有者禁止谷歌 FLoC 追踪</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2020/11/20201118074201_3550.jpg" title="又添一家：GitHub 宣布禁用谷歌 FLoC 新隐私追踪技术" referrerpolicy="no-referrer"></p><p>IT之家了解到，根据谷歌官方的说法，FLoC 技术使用机器学习算法来分析用户数据，然后根据个人访问的站点创建人群的集合。<span class="accentTextColor">广告商不会获得用户的本地数据，而是直接获得更广泛的人群画像，从而进行广告发放</span>。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/20210410152614_2390.jpg" title="又添一家：GitHub 宣布禁用谷歌 FLoC 新隐私追踪技术" referrerpolicy="no-referrer"></p><p>该技术推出以来，已被 Opera、微软 Edge、苹果 Safari 以及 Firefox 几家主流浏览器拒绝，同样拒绝的还有著名博客网站 WordPress，而 GitHub 的加入，意味着谷歌 FLoC 的推行道路上又多了一个阻碍。</p>
