<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 4 月 18 日消息 微软近日向 Android 版 Edge 浏览器推送了新 UI 更新，接着又为 <span class="accentTextColor">Android 版 OneDrive 和 Bing 更新了 UI</span>。</p><p>OneDrive 应用的 Home 选项卡现在突出显示用户互动的最后一个文件的最新视图，用户可以快速返回离开的位置。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/059f3676-6dfe-4c43-acdb-4b8f396c668d.jpg" title="Android 版微软 OneDrive 和 Bing 获得 UI 更新" referrerpolicy="no-referrer"></p><p>Bing 应用则在 UI 上进行了大改，在主页上添加了许多选项，整体风格与新版本的 Edge 浏览器靠近。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/ac80af3e-4a2c-4adc-8eb7-25cec217b8d2.jpg" title="Android 版微软 OneDrive 和 Bing 获得 UI 更新" referrerpolicy="no-referrer"></p><p>IT之家了解到，新版 OneDrive 还支持了谷歌的 Chromecast 连接器，该连接器可插在电视的 HDMI 接口上，用户可推送视频到 Chromecast 进行播放。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/4/b2a8e219-c665-4484-b1e9-27934439b426.png" id="1618699175475-1" title="Android 版微软 OneDrive 和 Bing 获得 UI 更新" referrerpolicy="no-referrer"></p>
