<p data-vmark="f707"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 6 日消息，微软 Edge 浏览器的全新 Drop 功能已面向 Canary 预览用户推出，该功能可以帮助用户使用浏览器跨设备共享文件和笔记，类似于 Telegram 的已保存消息功能。</p><p data-vmark="05bf">今日，外媒 Windows Latest 对该功能进行了上手体验，<span class="accentTextColor">该功能几乎可以传送所有内容</span>，包括应用程序、文本、笔记、文件和其他类型的附件等。</p><p data-vmark="027a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/38580768-8838-483e-a109-61d01c3bc8ac.png" w="861" h="633" title="微软 Edge 浏览器全新 Drop 功能上手体验，电脑、手机传文件的又一种方式" width="861" height="603" referrerpolicy="no-referrer"></p><p data-vmark="d301">据测试，微软没有为 Edge 开发新的云存储服务，<span class="accentTextColor">而是使用 OneDrive</span>。为了访问该功能，用户必须登录到 Microsoft 帐户，并且存储空间取决于 OneDrive 的空间。</p><p data-vmark="7fb5">想要使用该功能，用户需要点击浏览器<span class="accentTextColor">右上角工具栏的纸飞机图标</span>，然后就可以在一个类似聊天窗口的界面中输入文本，或者发送文件。</p><p data-vmark="cd03" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/a642e9f0-fc21-4c67-b9ef-6a37595f2ce1.png" w="451" h="679" title="微软 Edge 浏览器全新 Drop 功能上手体验，电脑、手机传文件的又一种方式" width="451" height="679" referrerpolicy="no-referrer"></p><p data-vmark="b777">在测试发布一条文本消息后，其他所有安装了 Edge 浏览器且支持该功能的设备都能够收到这条消息。</p><p data-vmark="9a43">此外，<span class="accentTextColor">Drop 可以共享大多数格式文件内容</span>。例如，外媒上传了一个名为“MPV 播放器”（mpv.exe）的应用，并在另一台设备上下载了该播放器，可以正常播放视频。</p><p data-vmark="81cb" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/bc87ce74-3e90-470c-b5eb-328eb985ed28.png" w="860" h="635" title="微软 Edge 浏览器全新 Drop 功能上手体验，电脑、手机传文件的又一种方式" width="860" height="605" referrerpolicy="no-referrer"></p><p data-vmark="e604">在 Edge Drop 的其中一个选项卡中，我们可以看到 <span class="accentTextColor">OneDrive 存储容量数据</span>，这表明 Drop 传输内容将计入 OneDrive 存储空间。</p><p data-vmark="4ffc" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/f4b7a5a4-6e72-4053-a040-6044fb0820cf.png" w="510" h="744" title="微软 Edge 浏览器全新 Drop 功能上手体验，电脑、手机传文件的又一种方式" width="510" height="744" referrerpolicy="no-referrer"></p><p data-vmark="b588">IT之家了解到，该功能也是可以隐藏的。如果你不需要的话，可以在右上角的设置 —> 外观中自定义工具栏图标，从而关闭 Drop 图标。</p>
