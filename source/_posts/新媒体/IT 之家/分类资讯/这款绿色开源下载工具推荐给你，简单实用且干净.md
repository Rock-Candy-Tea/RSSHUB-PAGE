<p>下载工具哪家强？论功能，很多下载软件都可以满足需求，但要找出一款干干净净的、规规矩矩的下载器，可能就不那么容易了。迅雷的广告多已经为人吐槽已久，而一些传统的 BT 下载工具，例如比特彗星、μtorrent 等，推广内容所占篇幅也是与日俱增。难道真的没有老实好用的下载器了嘛？今天就来给大家介绍一款吧。</p><p>LIII BitTorrent Client 是一款开源的 BT 下载工具，下面就简称它为 LIII 吧。LIII 的亮点主要有这么几个，首先它非常干净，完全没有广告和多余的后台；其次它的性能可圈可点，作为一款 BT 下载工具在连接 DHT 网络方面的表现很优秀；最后它还是开源的，有安装版和绿色版可选，用起来令人异常放心。</p><p>如果你是一名开发者，可以点击下面的链接进入它的 GitHub 主页查看代码。</p><p>LIII BitTorrent Client GitHub 主页：<a href="https://github.com/aliakseis/LIII" target="_blank" class="texttolink">https://github.com/aliakseis/LIII</a></p><p>LIII 的使用很简单，安装后即可开启进入主界面。虽然 LIII 并不支持中文，但其实还是非常好上手的，点击界面中的“+”按键或者直接在输入框，使用种子文件或者磁链添加任务即可。</p><p><img src="https://i0wssl.pconline.com.cn/pconline/2104/16/spcgroup/width_640-qua_50/spcgroup/width_640,qua_50/14144866_1.png" title="这款绿色开源下载工具推荐给你，简单实用且干净" referrerpolicy="no-referrer"></p><p>下载速度相当不错，可以连接 DHT 网络</p><p>LIII 的性能是相当不错的。它可以自动连接 DHT 网络，搜寻网络上活跃的 Tracker，这对于 BT 下载的速度来说尤为重要。就笔者的测试而言，如果下载的是比较热门的资源，速度完全可以达到满速。</p><p><img src="https://i0wssl.pconline.com.cn/pconline/2104/16/spcgroup/width_640-qua_50/spcgroup/width_640,qua_50/14144866_4a.png" title="这款绿色开源下载工具推荐给你，简单实用且干净" referrerpolicy="no-referrer"></p><p>和其他 BT 客户端连接顺畅，不会像迅雷那样被反吸血插件封杀</p><p>LIII 虽然看起来简洁，但功能上却完全不弱。它带有混淆流量、设置优先级等老司机喜闻乐见的功能，完全可以满足日常的 BT 下载需求。不过 LIII 也并非完全没有缺点，它并不支持 RSS 订阅，也就是说无法通过 RSS 来获得下载站的第一手资源发布信息，希望日后能加上这样的功能吧。</p><p><img src="https://i0wssl.pconline.com.cn/pconline/2104/16/spcgroup/width_640-qua_50/spcgroup/width_640,qua_50/14144866_2.png" title="这款绿色开源下载工具推荐给你，简单实用且干净" referrerpolicy="no-referrer"></p><p>一些设置选项</p><p>总的来说，LIII 是一款功能比较完善、性能表现出色的 BT 下载器。它的可贵之处，在于干净开源，如果你对软件有洁癖，那么 LIII 应该会是你的菜。不过，如果你需要 RSS 订阅这类功能，笔者会更推荐 qBittorrent，大家酌情选择吧。</p>
