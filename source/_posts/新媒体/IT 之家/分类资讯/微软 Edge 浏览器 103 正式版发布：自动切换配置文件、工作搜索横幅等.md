            <p data-vmark="9d7c"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 24 日消息，微软 Edge 浏览器 103 正式版发布，具体版本号为 <span class="accentTextColor">103.0.1264.37</span>，包含一些更改、安全性升级和小幅改进。</p><p data-vmark="bde6">Edge 103 包含以下改进：</p><ul class=" list-paddingleft-2"><li><p data-vmark="16a5"><strong>能够控制自动切换配置文件：</strong>GuidedSwitchEnabled 策略允许 Microsoft Edge 在检测到链接是个人链接或工作链接时提示用户切换到适当的配置文件。</p></li><li><p data-vmark="e0a6"><strong>客户端证书切换器：</strong>此功能将为用户提供一种在访问需要 http 证书身份验证的站点时，清除记住的证书并重新显示证书选择器的方法。无需手动退出 Microsoft Edge 即可完成切换。</p></li><li><p data-vmark="4f9c"><strong>更可靠的网络防御：</strong>借助针对 Windows 上的 Microsoft Edge 重写的 Microsoft Defender SmartScreen 库，用户浏览 Web 时将提供更可靠的保护。NewSmartScreenLibraryEnabled 策略将允许企业客户继续使用该库的旧版本，直到它在 Microsoft Edge 版本 105 中被弃用。</p></li><li><p data-vmark="badb"><strong>Edge 地址栏中的工作搜索横幅：</strong>此横幅通过将搜索焦点缩小到仅限工作的结果来帮助用户保持工作流程。用户可以在搜索开始时选择横幅，并可使用 AddressBarMicrosoftSearchInBingProviderEnabled 策略打开或关闭此功能。</p></li></ul><p data-vmark="c3f4">IT之家了解到，Edge 103 还添加了几个新的策略：</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="327e">GuidedSwitchEnabled - 启用引导开关</p></li><li><p data-vmark="ca3f">InternetExplorerZoomDisplay - 在 IE 模式选项卡中显示缩放，其中包含 DPI 比例，就像在 Internet Explorer 中一样</p></li><li><p data-vmark="a062">LiveCaptionsAllowed - 允许实时字幕</p></li><li><p data-vmark="9ec2">OriginAgentClusterDefaultEnabled - 默认启用 Origin-keyed 代理集群</p></li><li><p data-vmark="1942">SleepingTabsTimeout - 为睡眠标签设置后台标签不活动超时。注意：此策略中添加了 30 秒不活动的超时。</p></li></ul><p data-vmark="3c01">IT之家小伙伴可以在 Edge 浏览器的设置 —> 关于 Microsoft Edge 中查找更新并下载 Edge 103。</p>
