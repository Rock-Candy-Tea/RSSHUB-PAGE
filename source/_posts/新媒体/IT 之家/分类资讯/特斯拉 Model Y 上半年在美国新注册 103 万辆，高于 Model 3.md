<p data-vmark="195e">8 月 16 日消息，据国外媒体报道，研究机构最新的数据显示，特斯拉今年上半年已主导美国豪华汽车市场，新注册近 23 万辆，比宝马、梅赛德斯-奔驰等一众豪华汽车厂商的新注册量都要高。</p><p style="text-align: center;" data-vmark="88cd"><img src="https://img.ithome.com/newsuploadfiles/2022/8/4f4000f7-6b58-4eb7-8697-4d0dd491133d.jpg" w="500" h="286" title="特斯拉 Model Y 上半年在美国新注册 10.3 万辆，高于 Model 3" width="500" height="286" referrerpolicy="no-referrer"></p><p data-vmark="f18d">从研究机构的数据来看，特斯拉在上半年能主导美国豪华汽车市场，是得益于 Model Y 和 Model 3 这两款畅销电动汽车。</p><p data-vmark="6a41">机构的数据显示，特斯拉 Model Y 上半年在美国市场新注册 103215 辆，是新注册量最高的电动汽车。Model 3 紧随其后，上半年新注册 97075 辆。</p><p data-vmark="e4fd">而由于得克萨斯超级工厂在今年上半年就已投产，在生产的 Model Y 的产能还在不断提升，随着这一工厂产量的不断提升，Model Y 在美国市场的交付能力就会更强，届时比 Model 3 就会更多。</p><p data-vmark="2f69">特斯拉目前在交付的电动汽车，共有 4 款，除了 Model Y 和 Model 3，还有推出时间更早、售价更高的 Model S 和 Model X，这两款今年上半年在美国市场分别新注册 15317 辆、13382 辆，分别是上半年美国市场新注册量第 4、第 5 高的电动汽车。</p><p data-vmark="f0be">《<a href="https://www.ithome.com/0/635/370.htm" target="_blank">特斯拉已主导美国豪华汽车市场：2022 上半年新注册近 23 万辆</a>》</p>
