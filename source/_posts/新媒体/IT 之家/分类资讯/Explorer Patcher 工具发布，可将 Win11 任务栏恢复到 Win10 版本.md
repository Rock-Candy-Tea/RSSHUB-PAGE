<p data-vmark="eb2a"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 10 月 16 日消息，据 Neowin 报道，如果你正在寻找一种方法，简单地将 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> 任务栏和文件资源管理器恢复到你在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 中习惯的行为，并且无需付费，第三方工具 Explorer Patcher 工具可能是比较好的选择。只需将一个.DLL 添加到 C:\Windows，微软就会下载其余的文件，使之成为可能。</p><p data-vmark="1200"><img src="https://img.ithome.com/newsuploadfiles/2021/10/415e0d3d-1537-49f3-a4df-f79ae06a4134.jpg" w="760" h="475" title="Explorer Patcher 工具发布，可将 Win11 任务栏恢复到 Win10 版本" width="760" height="475" referrerpolicy="no-referrer"></p><p data-vmark="37bc">开发者表示，其目的是将富有成效的工作环境带回到 Windows 11 上。或者更确切地说，恢复到 Windows 10 版本的任务栏等。</p><h2 data-vmark="4830">下面是 Explorer Patcher 软件功能</h2><p data-vmark="60d3"><strong>来自 Windows 10 的任务栏：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="889e">主任务栏和副任务栏“总是合并”/“满时合并”/“从不合并”选项</p></li><li><p data-vmark="94a6">搜索按钮</p></li><li><p data-vmark="fa33">任务视图按钮</p></li><li><p data-vmark="9dcf">可选的皮肤、居中和切换的弹出式菜单，或无皮肤（适合经典主题）</p></li><li><p data-vmark="6711">当点击系统托盘网络图标的弹出式菜单中的“打开网络和互联网设置”时，打开“网络和共享中心”</p></li><li><p data-vmark="ed39">隐藏控制中心按钮</p></li><li><p data-vmark="ff24">在任务栏上显示人脉</p></li><li><p data-vmark="4d84">显示触摸键盘按钮</p></li></ul><p data-vmark="ef9a"><strong>文件资源管理器：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="2e30">禁用 Windows 11 的命令栏</p></li><li><p data-vmark="1e28">禁用 Windows 11 的上下文菜单</p></li><li><p data-vmark="a0ab">甚至禁用 Windows 10（沉浸式）上下文菜单（对经典主题用户很有用）</p></li><li><p data-vmark="9602">禁用现代搜索栏（恢复到早期 Windows 10 构建或 <a class="s_tag" href="https://www.win7china.com/" target="_blank">Windows 7</a>/8 的搜索栏）。</p></li><li><p data-vmark="3206">完全禁用搜索栏</p></li></ul><p data-vmark="613b"><strong>来自 Windows 11 的“开始”菜单：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="9560">在包含光标的显示器上打开“开始”菜单</p></li><li><p data-vmark="c7be">在登录时打开“开始”菜单</p></li><li><p data-vmark="cdaf">默认在“所有应用程序”中打开“开始”菜单</p></li><li><p data-vmark="6368">定位屏幕</p></li><li><p data-vmark="9c7e">显示频繁使用的应用程序最大数量</p></li></ul><p data-vmark="8528"><strong>来自 Windows 10 的窗口切换器（Alt-Tab）：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="9331">选择是否在窗口列表中包括桌面</p></li><li><p data-vmark="72c5">能够设置窗口列表的不透明度</p></li><li><p data-vmark="5946">设置行的大小，最大的范围等</p></li></ul><p data-vmark="9c30"><strong>其他功能：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="6728">当按下 Win+C（而不是 Microsoft Teams）时，打开时间和日期弹窗。</p></li><li><p data-vmark="309f">为桌面上的 Alt+F4 设置默认关机动作</p></li><li><p data-vmark="64a8">在 Win+X 菜单中显示命令提示符而不是 PowerShell</p></li></ul><h2 data-vmark="1a8d">安装</h2><p data-vmark="30e6">下载最新版本的 <a href="https://github.com/valinet/ExplorerPatcher/releases" target="_blank">Explorer Patcher</a>，你会发现一个名为 dxgi.dll 的 DLL 下载。它已经在最新的 Windows 11 稳定版（22000.258）上进行了测试，但开发者声称它在较新的版本上也能正常工作。这是应用程序的运行时部分。你需要把这个 DLL 放在以下位置。C:\Windows，这也意味着你将需要管理员权限。</p><p data-vmark="bbf1">完成上述工作后，确保 PC 连接到网络，并使用任务管理器或使用以下命令重新启动资源管理器进程。</p><blockquote><p
