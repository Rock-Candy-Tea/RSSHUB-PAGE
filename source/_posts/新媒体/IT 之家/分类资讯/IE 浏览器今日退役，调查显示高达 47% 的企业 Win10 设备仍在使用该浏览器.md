            <p data-vmark="0678"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 16 日消息，微软的 IE 浏览器今日正式退役，但新的调查显示，高达 47% 的 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 设备仍在使用 IE 浏览器作为其浏览器。</p><p style="text-align: center;" data-vmark="214b"><img src="https://img.ithome.com/newsuploadfiles/2022/6/1f83a05d-9491-4ac6-b288-1df75a73ef55.png" w="500" h="353" title="IE 浏览器今日退役，调查显示高达 47% 的企业 Win10 设备仍在使用该浏览器" width="500" height="353" referrerpolicy="no-referrer"></p><p data-vmark="5c1a">根据 Lansweeper 的调查，<span class="accentTextColor">他们调查了 33000 个不同组织所使用的高达 900 万台设备</span>，结果令人震惊，似乎许多公司还不想和 IE 浏览器说再见，<span class="accentTextColor">高达 47% 的 Windows 10 设备被发现仍在使用 IE 浏览器作为其主要浏览器</span>。</p><p data-vmark="17d1">Lansweeper 的调查还发现，多达 79% 的被调查电脑运行的是过时的 Windows 10 版本，这意味着公司不仅决定不升级到 <a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a>，而且也没有把设备更新到最新版本的 Windows 10。根据统计结果，企业电脑上最流行的 Windows 10 版本是 2004 版。</p><p data-vmark="1918">这将使得企业处于风险之中，随着微软对 IE 浏览器的停止支持，对于那些仍然坚持使用 IE 浏览器的用户来说，网络攻击的风险会大大增加。</p><p data-vmark="19f3">新的 Edge 浏览器明明更加好用，但为什么这么多企业仍然在使用 IE 浏览器？Lansweeper 的首席战略官 Roel Decneut 称，这并不完全令人惊讶，企业可能有很多理由选择不升级其设备上的浏览器或 Windows 版本。根据企业的规模，进行全公司的软件或操作系统升级可能是一个大的工程，公司可能有更紧迫的事情要处理，因此会尽可能地推迟更新。削减成本往往也是一个因素，此外没有强大的 IT 部门的小型企业可能根本没有意识到是时候升级了。</p><p data-vmark="7f1f">IT之家了解到，新的基于 Chromium 的 Edge 浏览器更快、更安全，甚至有一个内置的 IE 浏览器模式，帮助用户浏览那些还没有适应新技术的网站。</p>
