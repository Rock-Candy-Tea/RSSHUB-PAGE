<p data-vmark="f8a7"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 23 日消息，微软 Edge 浏览器近日测试了一项全新 Drop 功能，该功能可以帮助用户使用浏览器跨设备共享文件和笔记，类似于 Telegram 的已保存消息功能。</p><p style="text-align: center;" data-vmark="c943"><img src="https://img.ithome.com/newsuploadfiles/2022/6/38580768-8838-483e-a109-61d01c3bc8ac.png" w="861" h="633" title="微软 Edge 浏览器跨设备文件传输功能 Drop 征集中文名：拖拖、无边、闪推等" width="861" height="603" referrerpolicy="no-referrer"></p><p style="text-align: justify;" data-vmark="3d9b">昨日晚间，微软 Edge 浏览器官方微博<a href="https://weibo.com/7732075754/M2feenHQc" target="_blank">发起投票</a>，向网友征集 Drop 功能的中文名，列出了几个候选项，包括“拖拖”“无边”“易投（e 投）”“爱传”“闪推”，也可以留言写出觉得更好的中文名。</p><p style="text-align: center;" data-vmark="7941"><img src="https://img.ithome.com/newsuploadfiles/2022/8/c20f760f-b3f4-48e6-8d87-f66f13d4c52c.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzExOC5wbmc=,g_3,x_12,y_12,a_0,t_100" w="765" h="719" title="微软 Edge 浏览器跨设备文件传输功能 Drop 征集中文名：拖拖、无边、闪推等" width="765" height="719" referrerpolicy="no-referrer"></p><p style="text-align: justify;" data-vmark="6797">IT之家了解到，外媒 Windows Latest 对该功能进行了<a href="https://www.ithome.com/0/622/495.htm" target="_blank">上手体验</a>，<span class="accentTextColor">该功能几乎可以传送所有内容</span>，包括应用程序、文本、笔记、文件和其他类型的附件等。</p><p data-vmark="d301">据测试，微软没有为 Edge 开发新的云存储服务，<span class="accentTextColor">而是使用 OneDrive</span>。为了访问该功能，用户必须登录到 Microsoft 帐户，并且存储空间取决于 OneDrive 的空间。</p><p data-vmark="7fb5">想要使用该功能，用户需要点击浏览器<span class="accentTextColor">右上角工具栏的纸飞机图标</span>，然后就可以在一个类似聊天窗口的界面中输入文本，或者发送文件。</p><p data-vmark="cd03" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/a642e9f0-fc21-4c67-b9ef-6a37595f2ce1.png" w="451" h="679" title="微软 Edge 浏览器跨设备文件传输功能 Drop 征集中文名：拖拖、无边、闪推等" width="451" height="679" referrerpolicy="no-referrer"></p>
