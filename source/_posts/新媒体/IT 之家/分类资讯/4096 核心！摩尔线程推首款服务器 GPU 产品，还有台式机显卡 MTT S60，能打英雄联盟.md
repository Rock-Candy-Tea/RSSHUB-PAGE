data-vmark="5a97">在发布会开场，摩尔线程先展示了一段颇具视觉冲击力的视频，包含虚拟城市、数字工厂、虚拟人等“元宇宙”相关元素。紧接着，摩尔线程创始人兼 CEO 张建中登台，首先谈到“元计算”概念，即元宇宙计算的简称。</p><p style="text-align: center;" data-vmark="89a5"><img src="https://img.ithome.com/newsuploadfiles/2022/3/e7276716-a68a-457b-a798-485313d35c38.jpg@s_2,w_820,h_615" w="1000" h="750" title="4096 核心！摩尔线程推首款服务器 GPU 产品，还有台式机显卡 MTT S60，能打英雄联盟" 张建中进行演讲</p><p data-vmark="5f9e">元计算涉及整个物理世界数字化与数字世界物理化的计算过程。</p><p data-vmark="0281">物理世界数字化涉及 3D 建模、数字人制作、数字孪生、AI 模型训练、IoT 传感器、CAD / GIS、知识图谱、脑机接口、低延时高速通信、区块链 NFT 等技术。</p><p data-vmark="e0d4">数字世界物理化需要 3D 图形渲染、物理仿真、AI 训练与推理、科学计算、AR / VR / XR 交互、AI 人机交互、大规模分布式并行计算、超高清视网膜成像现实技术、生物仿真等技术。</p><p data-vmark="e40c">张建中希望，“元计算”能成为赋能下一代互联网的基础算力。为此，摩尔线程打造了统一系统架构 MUSA（MT Unified System Architecture）。</p><p style="text-align: center;" data-vmark="0da8"><img src="https://img.ithome.com/newsuploadfiles/2022/3/584d7a65-6619-42dc-b823-7a5bf7858af3.jpg@s_2,w_820,h_615" w="1000" data-vmark="1cc2">张建中在接受采访时谈道，设计 MUSA 架构的目的，是减少软件重复劳动，把不同引擎的核心能力释放出来，MUSA 底层提供不同 runtime，因此能将业界标准 API 做好。</p><p data-vmark="bb8c">据他分享，摩尔线程 GPU 产品的 PPA 能够满足各行业的需要，同时软件驱动也对产品性能进步至关重要，没有软件的芯片只是芯片，好的 GPU
