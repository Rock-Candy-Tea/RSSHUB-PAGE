<p data-vmark="3ac7">2021 年里 AI 飞速发展，优秀算法层出不穷，令我们大开眼界。而真正要将这些算法实现，一定少不了 Python 这个主流机器学习语言的加持。</p><p data-vmark="9a2c">今年各大公司和研究小组都推出了一系列<span class="accentTextColor">方便使用的高性能开源库</span>，其中还不乏一些国产的优秀作品。</p><p data-vmark="d632">在这里将盘点出最佳的 10 名，致力于帮你解决各种问题。</p><p data-vmark="dfd6">如果你还在为模型训练中的“杂务”头疼，那就快来看看吧。</p><h2 data-vmark="d2c5">1. Awkward Array</h2><p data-vmark="b1ee">根据官方介绍，<span class="accentTextColor">Awkward Array 用于嵌套的、大小不一的数据</span>，包括任意长度的列表、记录、混合的类型和缺失数据，使用起来类似 NumPy。</p><p data-vmark="99a7">看起来像是升级版的 NumPy 呀。</p><p data-vmark="5030"><img src="https://img.ithome.com/newsuploadfiles/2021/12/d6f45655-3f9e-4bac-b6e3-2d6acf4de4bf.png" w="1080" h="748" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="1080" height="568" referrerpolicy="no-referrer"></p><p data-vmark="ffc8">果然，不同长度的数组可以直接放在一起运算。</p><p data-vmark="b33e"><img src="https://img.ithome.com/newsuploadfiles/2021/12/2085d251-aa8c-4050-814c-a7d8c565279a.png" w="1080" h="458" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="1080" height="348" referrerpolicy="no-referrer"></p><p data-vmark="95d1">并且，官方表示 Awkward Array 不仅使用起来更简便，在速度和内存上也有量级的优势。</p><p data-vmark="c45c">看看是不是可以安排上了～</p><p data-vmark="6e4e"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://pypi.org/project/awkward/</span></span></p><h2 data-vmark="b9e6">2. Jupytext</h2><p data-vmark="4a4f">相信大家对 Jupyter Notebook 都不陌生。</p><p data-vmark="8951">当你有了 Jupytext 这个小插件就可以将 Jupyter Notebook 和 IDE 完美结合，听起来是不是很棒！</p><p data-vmark="fdc5">从此 Jupyter Notebook 可以被存储为 Markdown 文件或多种语言的脚本文件。</p><p data-vmark="f72a">Jupytext 可以做的事主要有：</p><ul class=" list-paddingleft-2"><li><p data-vmark="1817">Jupyter Notebook 的版本控制；</p></li><li><p data-vmark="fd26">在你喜欢的文本编辑器中编辑、合并或重构 Notebook；</p></li><li><p data-vmark="ee16">在 Notebook 上使用 Q&A 检查。</p></li></ul><p data-vmark="44d2">在 Python 中使用的样子：</p><p data-vmark="8d67">此项目在 Github 上已有 5k+star。</p><p data-vmark="b85b"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://github.com/mwouts/jupytext</span></span></p><h2 data-vmark="f750">3. Gradio</h2><p data-vmark="feb2">比 Streamlit 还轻量的 UI 设计库，Gradio 让你轻松在<a class="s_tag" href="https://qiyu.ruanmei.com/" target="_blank">浏览器</a>中“玩转”你的模型，可以直接在浏览器中拖放图片，粘贴文字，录制声音，等等。</p><p data-vmark="f163"><img src="https://img.ithome.com/newsuploadfiles/2021/12/b7003b52-1066-44c6-bf0b-39287a9de62f.png" w="1080" h="227" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="1080" height="172" referrerpolicy="no-referrer"></p><p data-vmark="18bd"><img src="https://img.ithome.com/newsuploadfiles/2021/12/13361f2f-0cef-4c79-815c-2752af532272.gif" w="960" h="161" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="960" height="138" referrerpolicy="no-referrer"></p><p data-vmark="95dd">只要将 launch () 函数中的参数设置为 share=True，还能得到一个可分享的网址，拿到链接的朋友在电脑和手机端都能打开，活脱脱就是一个小程序。</p><p data-vmark="77a1">时常需要做 Demo 的小伙伴快看起来吧，此项目在 Github 上已有 4.5k+star。</p><p data-vmark="1eec"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://github.com/gradio-app/gradio</span></span></p><h2 data-vmark="2a5c">4. Hub</h2><p data-vmark="efcf">这个 Hub 在数据管理和数据预处理上可是一把好手。</p><p data-vmark="cec2"><img src="https://img.ithome.com/newsuploadfiles/2021/12/8232c7c7-bb7e-4d36-a22d-b39759bd7d66.png" w="584" h="285" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="584" height="285" referrerpolicy="no-referrer"></p><p data-vmark="054b">它可以处理任何类型，任何大小的数据，并且因为数据储存在云端上，所以可以无缝在任何机器上访问。</p><p data-vmark="d0ca">被压缩为二进制字节的数据可以被存储在任何地方，并且只有在需要的时候才会被获取，所以没有 TB 级硬盘也可以处理 TB 级数据。</p><p data-vmark="99ea">Hub 贴心地提供了重要 API，支持数据在常用工具（PyTorch 等）上的使用，数据版本控制，数据转换等功能。</p><p data-vmark="112e">此项目在 github 上已有 4.1k+star。</p><p data-vmark="cfda"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://github.com/activeloopai/Hub</span></span></p><h2 data-vmark="3e09">5. AugLy</h2><p data-vmark="28e5">AugLy 是 facebook 最新推出的数据增强库，同时支持语音，文本，图像和视频类型的数据，包含了 100 多种增强方式。</p><p data-vmark="9f72">数据对于模型训练至关重要，而标注大规模数据十分困难。由于人力资源，和模型特性的限制，数据增强的应用越来越广泛。</p><p data-vmark="58f9">AugLy 的优点：</p><p data-vmark="04aa">处理类型更为全面。其他的数据增强库，例如 Albumentations 和 NVIDIA DALI，主要负责图像相关数据的处理，文字数据不支持。</p><p data-vmark="3151">处理方式十分人性化。AugLy 可以将一张图片做成备忘录，在图片 / 视频上叠加文字 / Emojis，转发社交媒体上的截图，还可以帮助你处理诸如拷贝检测、仇恨言论检测或版权侵权等问题。</p><p data-vmark="1329">此项目在 Github 上已有 4.1k+star。</p><p data-vmark="08fe"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://github.com/facebookresearch/AugLy</span></span></p><h2 data-vmark="f258">6. Evidently</h2><p data-vmark="24ce">Evidently 是用来监测模型效果的工具，可从 Pandas DataFrame 或 csv 文件中生成交互式可视化报告和 JSON 格式的效果简介。在 Jupyter Notebook 中可以使用。</p><p data-vmark="6b57">目前可以提供 6 种报告：数据漂移、数值目标漂移、分类目标漂移、回归模型性能、分类模型性能和概率分类模型性能。</p><p data-vmark="1a7a">此项目在 Github 上已有 1.8k+star。</p><p data-vmark="9e1b"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://github.com/evidentlyai/evidently</span></span></p><h2 data-vmark="1d69">7. YOLOX</h2><p data-vmark="6d7c">如果你熟悉 YOLO 的话，那你或许会对旷视今年推出的 YOLOX 感兴趣。</p><p data-vmark="24c2">YOLO 就是那个目标检测算法，可以被使用在汽车自动驾驶等前沿技术中。</p><p data-vmark="5614">而 YOLOX 是 YOLO 的无锚版本，设计更简单，但性能更好！它的目标是在研究界和工业界之间架起一座桥梁，同时弥合两方之间的差距。</p><p data-vmark="870f"><img src="https://img.ithome.com/newsuploadfiles/2021/12/4d2cd208-a8ad-4d42-a689-ce05a2b94049.png" w="427" h="147" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="427" height="147" referrerpolicy="no-referrer"></p><p data-vmark="81af"><img src="https://img.ithome.com/newsuploadfiles/2021/12/7fa9b714-35cc-4075-a163-0895ea7e7b8b.png" w="1080" h="187" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="1080" height="142" referrerpolicy="no-referrer"></p><p data-vmark="0856">这个 Github 上的开源项目在短短半年内已获得 5.2k+star。</p><p data-vmark="2c24"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://github.com/Megvii-BaseDetection/YOLOX</span></span></p><h2 data-vmark="88f6">8. LightSeq</h2><p data-vmark="3936">正如它的名字一样，LightSeq 是一款由字节跳动开发的支持 BERT、GPT、Transformer 等众多模型的超快推理引擎。</p><p data-vmark="9ae3"><img src="https://img.ithome.com/newsuploadfiles/2021/12/360fb0e7-1b4f-42af-b781-2c45e0410ef2.png" w="1080" h="240" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="1080" height="182" referrerpolicy="no-referrer"></p><p data-vmark="f592">可以看到它的表现，比 FasterTransformer 还要 Fast。</p><p data-vmark="04e8"><img src="https://img.ithome.com/newsuploadfiles/2021/12/b45714d1-7803-4100-937b-6b9cccde54f5.png" w="1080" h="446" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="1080" height="339" referrerpolicy="no-referrer"></p><p data-vmark="b962">LightSeq 支持的模型也是非常全面。</p><p data-vmark="8894"><img src="https://img.ithome.com/newsuploadfiles/2021/12/831f4b97-1996-45d2-a23a-5f5b7fd25852.png" w="1080" h="170" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="1080" height="129" referrerpolicy="no-referrer"></p><p data-vmark="e919">总之就是两个字“好用”。此项目在 Github 上已有 1.9k+star。</p><p data-vmark="453e"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://github.com/bytedance/lightseq</span></span></p><h2 data-vmark="0ea5">9. Greykite</h2><p data-vmark="d341">想预测 COVID-19 的恢复速度吗？那就来看看 LinkedIn 为了自家时间序列预测需求开发的 Greykite 吧。</p><p data-vmark="1180"><img src="https://img.ithome.com/newsuploadfiles/2021/12/baf922f9-5f8c-4fd5-8114-c8bd01e81fe0.png" w="1080" h="831" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="1080" height="631" referrerpolicy="no-referrer"></p><p data-vmark="d09a">功能全面（多种时间趋势），界面直观，预测速度快和可扩展性强是它最大的亮点。</p><p data-vmark="a580"><img src="https://img.ithome.com/newsuploadfiles/2021/12/63ac8ddd-ac4a-4c64-8787-f7eaef6f4f6b.png" w="1080" h="508" alt="图片" title="2021 年 Python 十佳 ML 库大盘点：国产选手 GitHub 半年获 5k+star，第一名是升级版 NumPy" width="1080" height="386" referrerpolicy="no-referrer"></p><p data-vmark="aae6">被应用在上面的三大算法：</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="bbbc">Silverkite (Greykite’s flagship algorithm)</p></li><li><p data-vmark="fec0">Facebook Prophet</p></li><li><p data-vmark="d646">Auto Arima</p></li></ul><p data-vmark="d09d">感兴趣的话就去研究看看吧，此项目在 Github 上已有 1.4k+star。</p><p data-vmark="246b"><span class="no-auto-spacing"><span class="link-text-start-with-http">https://github.com/linkedin/greykite</span></span></p><h2 data-vmark="aff8">10.
