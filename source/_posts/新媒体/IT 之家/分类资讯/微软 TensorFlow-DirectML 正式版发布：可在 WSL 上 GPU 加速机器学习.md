<p data-vmark="4e06"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 12 日消息 微软近日发布了 <span class="accentTextColor">TensorFlow-DirectML 的正式版</span>，该工具可让用户在 Windows 的 Linux 子系统（WSL）上 GPU 加速机器学习（ML）模型训练。</p><p data-vmark="ed67"><img src="https://img.ithome.com/newsuploadfiles/2021/9/6ead2003-ff8f-4dec-9345-26e8aad78194.png" w="1194" h="586" title="微软 TensorFlow-DirectML 正式版发布：可在 WSL 上 GPU 加速机器学习" width="1194" height="402" referrerpolicy="no-referrer"></p><p data-vmark="b46d">顾名思义，TensorFlow-DirectML 用于将谷歌 TensorFlow 机器学习平台与微软 DirectML 机器学习 API 适配，以<span class="accentTextColor">支持在 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Win10</a>/WSL 上进行机器学习模型训练时获得 GPU 加速</span>。</p><p data-vmark="ac9e">IT之家了解到，用户可以在 Python 环境下（Python 3.8 及以上暂不支持）输入 <strong>pip install tensorflow-directml</strong> 进行安装，支持在各种 DirectX 12 兼容硬件上训练和推理复杂的机器学习模型。</p><h2>Win10 软硬件要求：</h2><ul class=" list-paddingleft-2"><li><p data-vmark="ee74"><a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 1709，64 位（Build 16299 或更高）；</p></li><li><p data-vmark="157d">Python x86-64 3.5、3.6 或 3.7；</p></li></ul><p data-vmark="81ec">以下支持的 GPU 之一：</p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="e3a6">AMD Radeon R5/R7/R9 2xx 系列或更新版本；</p></li><li><p data-vmark="b1ea">英特尔 HD Graphics 5xx 或更新版本；</p></li><li><p data-vmark="f0f8">英伟达 GeForce GTX 9xx 系列 GPU 或更新版本；</p></li></ul><h2>WSL 软硬件要求：</h2><ul class=" list-paddingleft-2"><li><p data-vmark="bd13">Windows 10 Insider Preview，64 位（Build 20150 或更高）；</p></li><li><p data-vmark="d022">Python x86-64 3.5、3.6 或 3.7；</p></li></ul><p data-vmark="6fc6">以下支持的 GPU 之一：</p><ul class=" list-paddingleft-2"><li><p data-vmark="cb53">AMD Radeon R5/R7/R9 2xx 系列或更新版本，以及 20.20.01.05 WSL 驱动；</p></li><li><p data-vmark="92b9">英特尔 HD Graphics 6xx 或更新版本，以及 28.20.100.8322 WSL 驱动；</p></li><li><p data-vmark="f093">英伟达 GeForce GTX 9xx 系列或更新版本，以及 460.20 WSL 驱动；</p></li></ul><p data-vmark="0815">具体使用方法，大家可以查看 TensorFlow-DirectML 的 <a href="https://github.com/microsoft/tensorflow-directml" target="_blank">GitHub 页面</a>。</p>
