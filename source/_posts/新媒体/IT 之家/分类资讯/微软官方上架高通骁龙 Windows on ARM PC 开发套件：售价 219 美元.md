<p data-vmark="e172"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 15 日消息，微软近日在美区微软官方线上商城上架了一款基于高通骁龙平台的 Windows on ARM PC 开发套件，型号为 ECS LIVA Mini Box QC710 Desktop，售价 219 美元（约合人民币 1397 元）。</p><p data-vmark="a17a"><img src="https://img.ithome.com/newsuploadfiles/2021/11/f91216b5-0c58-499d-b297-e9a88e40fcbd.jpeg" w="700" h="394" title="微软官方上架高通骁龙 Windows on ARM PC 开发套件：售价 219 美元" width="700" height="394" referrerpolicy="no-referrer"></p><p data-vmark="6065">IT之家从美区微软官方线上商城了解到，该套件的整体外观为黑色，尺寸为 35.05 mm x 119.13mm x 116.58 mm，重量 0.23kg，在设备的顶部中心位置有一个大大的内凹开关按钮。<span class="accentTextColor">该套件采用高通骁龙 7c 计算单元（Compute）SC7180</span>，微软表示该款芯片可为平台提供快速的相应能力，该芯片发热量低，无需风扇即可高效运行。其他硬件方面该套件搭配 4GB 内存以及 64GB 存储空间，随机附带 64 位 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 家庭版操作系统。</p><p data-vmark="1975"><img src="https://img.ithome.com/newsuploadfiles/2021/11/179017fe-2427-4ef2-89c8-68d4d508aec8.jpeg" w="700" h="394" title="微软官方上架高通骁龙 Windows on ARM PC 开发套件：售价 219 美元" width="700" height="394" referrerpolicy="no-referrer"></p><p data-vmark="24a1">接口配置方面，ECS LIVA Mini Box QC710 Desktop 开发套件拥有一个 USB Type – C 接口（支持 PD 充电）、一个支持 USB 3.2 Gen 1 的 USB Type – A 接口、一个支持 USB 2.0 的 USB Type – A 接口、一个 MicroSD 卡插槽、一个 HDMI 视频输出接口（规格暂时未知）以及一个以太网接口。无线连接方面该开发套件支持 Wi-Fi 5 且支持蓝牙（规格暂时未知）。</p><p data-vmark="52f9"><img src="https://img.ithome.com/newsuploadfiles/2021/11/b7b5f48a-97f0-4334-a18c-6c54a64aa918.png" w="700" h="394" title="微软官方上架高通骁龙 Windows on ARM PC 开发套件：售价 219 美元" width="700" height="394" referrerpolicy="no-referrer"></p><p data-vmark="e753">ECS LIVA Mini Box QC710 Desktop 开发套件目前可在美区微软官方线上商城购买，<span class="accentTextColor">但现阶段仅面向开发人员出售</span>。</p>
