<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 6 月 12 日消息 英伟达方面今天确认，从 8 月 31 日起，它将不再支持基于 Kepler 架构的 GeForce 显卡和 非 Windows 10 操作系统（Windows 7/8/8.1）。</p><p>此外，英伟达将发布支持这些显卡和操作系统的最后一版驱动。当然，这并不意味着开普勒显卡将不再收到针对关键错误和安全更新的补丁，再但这些显卡将不有 Game Ready 驱动程序。</p><p>IT之家了解到，英伟达在 2012 年推出了 GeForce GTX 600 系列显卡，随后又推出了 GeForce GTX 700 系列卡。</p><p>值得一提的是，700 系列卡是首个采用 TITAN 级 GPU 的代系，虽然其中一些卡仍然能够在一些较旧的游戏中提供不错的性能，但这些卡确实该归入历史了。</p><p>此外，英伟达似乎确实犯了一个小错误，因为他们公布的 Kepler GPU 名单包括 GTX 750 和 GTX 750 Ti，而这两款显然是麦克斯韦架构卡。因此我们不确定是官方写错了还是故意的（更新：官方已删）。</p><p>如果确实取消 GTX750/750Ti 支持的话，这个名单不如说是 GeForce 600 和 700 系列更准确。</p><blockquote><p>自 2021 年 10 月起，Game Ready 驱动程序升级（包括性能增强、新功能和错误修复）将适用于使用 Maxwell、Pascal、Turing 和 Ampere 系列 GPU 的系统。到 2024 年 9 月，将在使用桌面 Kepler 系列 GPU 的系统上提供关键安全更新。可在此处找到桌面 Kepler 系列 GeForce GPU 的完整列表。</p><p>自 2021 年 10 月起，Game Ready 驱动程序升级（包括性能增强、新功能和错误修复）将专门用于使用 Windows 10 作为其操作系统的系统。到 2024 年 9 月，将在使用 Windows 7、Windows 8 或 Windows 8.1 的系统上提供关键安全更新。</p><p>— 英伟达</p></blockquote><p>NVIDIA 还提供了有关这两项更改的简短常见问题解答。</p><blockquote><p>问：为什么 NVIDIA 不再支持 Kepler 桌面 GPU？<br>答：基于 Kepler 的桌面 GPU 最初于 2012 年 3 月推出。从那时起，随着 DirectX 12 Ultimate 和 NVIDIA DLSS（深度学习超级采样）等技术的出现，游戏技术发生了巨大的发展。展望未来，NVIDIA 的软件 QA 团队将专注于支持新技术的硬件。</p><p>问：为什么 NVIDIA 不再支持 Windows 7、Windows 8 和 Windows 8.1？<br>答：Microsoft 已正式终止对 Windows 7 和 Windows 8 的支持，Windows 8.1 的生命周期也接近尾声。我们的绝大多数 GeForce 客户已迁移到 Windows 10 操作系统。为了确保 GeForce 用户体验到最佳的安全性、支持和功能，NVIDIA 现在将专注于 Windows 10 操作系统。</p><p>问：支持基于 Kepler 的桌面 GPU 的最终 Game Ready 驱动程序何时上市？<br>问：支持 Windows 7、Windows 8 和 Windows 8.1 的最终 Game Ready 驱动程序何时公开？<br>答：2021 年 8 月 31 日（R470 GA5）</p><p>问：第一个不支持基于 Kepler 的桌面 GPU 的 Game Ready 驱动程序什么时候上市？<br>问：第一个不支持 Windows 7、Windows 8 和 Windows 8.1 的 Game Ready
