<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 8 日消息 早在 2017 年，微软就宣布了 Windows Containers Beta 计划中的 SQL Server，使开发人员能够从同一容器内部或外部连接到 SQL Server 实例。</p><p>近日，微软宣布暂停该计划。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/badfd4af-f5f8-4e61-94a3-664dea0934ca.png" w="1440" h="659" title="微软宣布暂停自家 Windows 容器的 SQL Server：去用 Linux 版吧" width="1440" height="375" referrerpolicy="no-referrer"></p><p>在一篇简短的博客文章中，微软强调了 SQL Server on Windows Containers Beta 计划由于<span class="accentTextColor">生态系统挑战和“使用模式”</span> 而被暂停。微软在声明中措辞谨慎，表示在可预见的未来<span class="accentTextColor">没有恢复该计划的准备</span>。</p><p>微软建议用户迁移到 <span class="accentTextColor">Linux Containers 上的 SQL Server</span>，该服务已经在生产环境中得到支持。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/774b6898-03a6-4d31-b5b0-7529b1fbb0cf.jpg" alt="Microsoft and Linux written on the left and right respectively with a heart icon in the middle" w="760" h="428" title="微软宣布暂停自家 Windows 容器的 SQL Server：去用 Linux 版吧" width="760" height="428" referrerpolicy="no-referrer"></p><p>IT之家了解到，微软表示 SQL Server Express 和 Developer for Windows Containers Docker 存储库将很快被删除。这些存储库中的标签也将被删除，并且 Docker 镜像将无法下载。</p>
