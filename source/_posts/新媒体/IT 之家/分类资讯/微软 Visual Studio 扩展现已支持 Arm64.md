            <p data-vmark="1ecd"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 9 月 25 日消息，继微软 Visual Studio 2022 支持 Arm64 之后，<span class="accentTextColor">Visual Studio 的扩展现在也支持 Arm64 了</span>。</p><p style="text-align: center;" data-vmark="c297"><img src="https://img.ithome.com/newsuploadfiles/2022/9/17684fba-20cc-44a0-a721-d76134eda084.png" w="1428" h="572" title="微软 Visual Studio 扩展现已支持 Arm64" width="1428" height="328" referrerpolicy="no-referrer"></p><p data-vmark="2c83"><strong>为新扩展或现有扩展添加 Arm64 支持：</strong></p><p data-vmark="9152">1、在扩展解决方案中，通过 NuGet 包管理器将 VSSDK 构建工具包更新到最新版本。</p><p data-vmark="198b">2、在 VSIX 清单中，添加指定 arm64 的 ProductArchitecture 标记，并确保 Prerequisites 标记包含 VS 2022。</p><p style="text-align: center;" data-vmark="f1fc"><img src="https://img.ithome.com/newsuploadfiles/2022/9/77f5bbad-7eba-44dc-a93a-76ab6fdbccdb.png" w="994" h="126" alt="ProductArchitecture Tag" title="微软 Visual Studio 扩展现已支持 Arm64" width="994" height="104" referrerpolicy="no-referrer"></p><p data-vmark="62e1">3、在项目的 Properties > Build 下，将构建目标设置为 AnyCPU 或 Arm64。</p><p style="text-align: center;" data-vmark="5737"><img src="https://img.ithome.com/newsuploadfiles/2022/9/5b85b35d-893d-4f68-92c9-509a860b71db.png" w="573" h="226" alt="Arm64 Build Settings" title="微软 Visual Studio 扩展现已支持 Arm64" width="573" height="226" referrerpolicy="no-referrer"></p><p data-vmark="ba0b">▼ 带有 Arm64 安装目标的 VSIX 清单</p><p style="text-align: center;" data-vmark="cecb"><img src="https://img.ithome.com/newsuploadfiles/2022/9/686e86b8-863c-4fe2-84c9-00c747e37a30.png" w="1123" h="555" alt="Arm64 VSIX Manifest Installation Target" title="微软 Visual Studio 扩展现已支持 Arm64" width="1123" height="405" referrerpolicy="no-referrer"></p><p data-vmark="8ef2">此外，扩展开发者现在可以在单个 Marketplace 条目上发布 Arm64 扩展以及扩展的 amd64（64 位）版本了。</p><p style="text-align: center;" data-vmark="cf4a"><img src="https://img.ithome.com/newsuploadfiles/2022/9/c721a313-5302-42d2-9503-577bcb36b263.png" w="604" h="369" title="微软 Visual Studio 扩展现已支持 Arm64" width="604" height="369" referrerpolicy="no-referrer"></p><p data-vmark="3287">IT之家了解到，今年 6 月份，微软 Visual Studio 2022 首次支持了 Arm64，开发者可以在 ARM 设备上原生开发 Arm64 应用。</p><p data-vmark="2baf">此外，Visual Studio 2022 也支持了苹果 Apple Silicon (Arm64) 处理器，打开大型解决方案等操作的速度比 Visual Studio 2019 for Mac 快 50%。</p>
