属地功能上线之后，不少博主的秘密被网友发现，直呼翻车。相关话题 <span class="accentTextColor">#显示 IP 属地后 IP 代理畸形产业爆火# </span>登上微博热搜榜...</p><p data-vmark="695d" style="text-align: center;"><img referrerpolicy="no-referrer"></p><p data-vmark="0dd1">据悉，自从多个平台上线 IP 属地功能后，网络上出现了一个个翻车现场，某些海外代购账号，其实 IP 归属地在国内；一些本地的网红账号，运营地却相隔十万八千里。IP 归属地功能，让不少大 V 都心慌慌。记者了解到，随着 IP 归属地功能的上线，<span class="accentTextColor">付费 IP 代理</span>的生意红火了起来。</p><p data-vmark="8f8f" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/96045a71-9b1f-4bd5-82fb-ddec2e662f28.png" w="658" h="481" title="显示 IP 属地后有人慌了，IP 代理畸形产业爆火：最低 6 元就能更改" width="658" height="481" referrerpolicy="no-referrer"></p><p data-vmark="478b">根据澎湃新闻报道，这也让付费 IP 代理的生意红火了起来，然而这本就一个黑灰产业。在一些电商平台上，已经出现了付费 IP 代理的业务，有的目的性非常明显，<span class="accentTextColor">直接指明可以更改微博、抖音的 IP 归属地</span>。</p><p data-vmark="9fd7">有卖家称，国内的服务器价格<span class="accentTextColor">从 6-8 元不等</span>，不过这只是短效 IP，且可供选择地点比较少。最近找 IP 代理的人确实增多，自己出售的服务，可更改手机或电脑的 IP 地址，并且支持所有的平台，所有品牌的手机。卖家还透露，自己是计算机专业，技术还是过硬的，因此他所出售的 IP 代理服务，都是自己破解的。目前，某平台上“IP 代理”等关键词搜索结果已经被屏蔽。</p><p data-vmark="95da" style="text-align: center;"><img referrerpolicy="no-referrer"></p><p data-vmark="5231">据悉，目前大多数的卖家所提供的更换 IP 归属地的服务，一般服务器都在国内，提供国外服务器的商家较少，且要价较高。</p><p data-vmark="71f3">此功能开放后，许多网友发现，不少大 V 都出现了 IP 属地与资料地址不相符的情况。对此，网友们纷纷表示：“ 一个 ip 显示养活了一个行业，都有光明的未来”、“其实说白了还是隐私问题，有人就是不喜欢显示啊。网上肯定有人会捣乱，但也有普通人不想显示的啊”... 大家怎么看？</p><p data-vmark="672c" style="text-align: center;"><img
