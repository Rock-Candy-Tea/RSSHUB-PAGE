            <p data-vmark="aed0"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 2 日消息，微软 GitHub AI 编程工具 Copilot 在经过了近一年的免费测试后<a href="https://www.ithome.com/0/625/623.htm" target="_blank">正式上线</a>，定价每月 10 美元或每年 100 美元，这一做法引发了不小的争议，甚至引来了行业的抵制。</p><p data-vmark="cf86">专注于自由和开源软件 (FOSS) 的非营利组织<span class="accentTextColor">软件自由保护协会 (SFC) </span>表示，它已停止使用微软的 GitHub 进行项目托管，并敦促其他软件开发商也这样做。</p><p data-vmark="8228" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/7/ce37e8dc-6336-45d3-8842-bc8ac2d77306.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzkxLnBuZw==,g_2,x_0,y_14,a_0,t_100" w="937" h="1006" title="微软 AI 编程工具 Copilot 转付费引争议，开源组织号召退出 GitHub" width="937" height="880" referrerpolicy="no-referrer"></p><p data-vmark="d309">在博客中，SFC 称与 GitHub 的决裂正是由于 AI 编程工具 Copilot 的收费。SFC 表示，<span class="accentTextColor">GitHub 决定发布源自 FOSS 代码的营利性产品，这一做法“令人难以接受”</span>。</p><p data-vmark="34fa">IT之家了解到，Copilot 基于 OpenAI 的 Codex，可以在开发者工作时向他们推荐代码。据 GitHub 称，Copilot  接受了“来自公开来源的自然语言文本和源代码的训练，包括 GitHub 上公共存储库中的代码”。</p><p data-vmark="c438" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/6/86ecb364-569c-48dd-b922-7782de69521f.png?x-bce-process=image/watermark,image_aW1nL3dhdGVybWFyay9xYy9xYzE2NS5wbmc=,g_3,x_16,y_16,a_0,t_100" w="1072" h="652" title="微软 AI 编程工具 Copilot 转付费引争议，开源组织号召退出 GitHub" width="1072" height="499" referrerpolicy="no-referrer"></p><p data-vmark="8a14">SFC 称，微软和 GitHub 未能提供以下说明：</p><ul class=" list-paddingleft-2"><li><p data-vmark="90c1">有关在公共代码上训练其 AI 系统的版权后果</p></li><li><p data-vmark="e18d">Copilot 为何接受 FOSS 代码而不是受版权保护的 Windows 代码的培训</p></li><li><p data-vmark="230c">是否可以指出训练集代码附带的所有软件许可和版权所有者。</p></li></ul><p data-vmark="c67c">SFC 表示，虽然目前不会强制现有的成员项目迁移，<span class="accentTextColor">但将不再接受没有长期计划从 GitHub 迁移的新成员项目</span>。</p>
