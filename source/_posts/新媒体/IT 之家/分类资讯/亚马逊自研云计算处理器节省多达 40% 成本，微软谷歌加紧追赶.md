<p data-vmark="c6f6">北京时间 7 月 20 日晚间消息，据报道，在过去的十年里，亚马逊 AWS 云服务通过技术和价格而占据市场主导地位。而在未来的十年，亚马逊自主研发的“<span class="accentTextColor">Graviton</span>”处理器将成为制胜武器。</p><p data-vmark="dc56">过去十年，亚马逊云计算服务部门 Amazon Web Services（AWS）通过快速的技术升级和降低价格，在云计算服务市场领先于竞争对手微软和谷歌。未来十年，AWS 的一个关键优势在于其 Graviton 处理器。Graviton 由 AWS 自主开发，致力于为互联网上的应用程序（App）提供动力，或帮助客户训练机器学习（ML）模型。</p><p data-vmark="cca6">AWS 的六家客户向媒体表示，与基于英特尔和 AMD 处理器的服务器相比，<span class="accentTextColor">使用 Graviton 处理器的云服务器的能耗更低，运算速度更快</span>。这些客户称，通过租用 Graviton 服务器，他们节省了 <span class="accentTextColor">10% 到 40%</span> 的计算成本。</p><p data-vmark="5d9f">知情人士称，Twitter、Snap、Adobe 和 SAP 等公司均已成为 Graviton 服务器的客户。在推出仅三年后，Graviton 服务器就发展成为一项年营收达数十亿美元的业务。今年 5 月，亚马逊又推出了更具成本效益的第三代 Graviton 处理器，让竞争对手们感受到了更大的追赶压力。</p><p data-vmark="92dd">基于 Graviton 处理器的服务器的速度和效率，为 AWS 客户节省了大量资金。与此同时，旺盛的市场需求也推动亚马逊的云服务营收迅速增长。知情人士称，在去年秋季，Graviton 服务器（2018 年底推出）的年收入规模达到约 50 亿美元。这意味着，如果 Graviton 服务器继续保持该增长势头，则今年的营收有望达到亚马逊 Elastic Compute Cloud（弹性计算云，EC2）营收的 10% 以上。去年，EC2 营收约占亚马逊 620 亿美元的 AWS 营收的一半。</p><p data-vmark="0665">亚马逊预计，随着时间的推移，将有更多的客户涌向 Graviton 服务器以节省成本。例如，Graviton 服务器已经成功吸引了 Twitter，后者是世界上最大的云计算客户之一。最终，这些客户还可能选择其他 AWS 服务，从而进一步刺激亚马逊营收增长。</p><p data-vmark="598a">对此，AWS、英特尔和 AMD 尚未发表评论。</p><h2 data-vmark="eca6">竞争对手的追赶</h2><p data-vmark="3fc5">Annapurna Labs 基于 Arm 的芯片设计开发了 Graviton 处理器。ARM 的芯片设计常用于智能手机中，但由于其性能优势，如今在数据中心和云服务器市场受到了青睐。据报道，<span class="accentTextColor">微软正在设计自己的基于 ARM 的服务器芯片，而谷歌去年也公布了类似的计划</span>。</p><p data-vmark="d9ce">与此同时，两家公司目前都通过与英特尔前高管瑞妮・詹姆斯（Renée James）领导的芯片初创公司 Ampere Computing 合作，提供“预发布版本”的 Arm 云服务器。上周，谷歌推出了其首款基于 Ampere Computing 云服务器的预览版，并表示将在今年晚些时候面向客户广泛推出该服务器。此外，甲骨文公司（Oracle）已经在向其云客户提供基于 Ampere Computing 服务器。</p><p data-vmark="5186">数据分析初创公司 Starburst Data 工程副总裁肯・皮克林（Ken Pickering）称，基于 Arm 芯片设计的服务器的一个优势在于，它们在并行化（同时处理多个不同的计算任务）方面要优于英特尔和 AMD 服务器。</p>
