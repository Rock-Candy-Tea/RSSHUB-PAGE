<p data-vmark="9cfa"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 30 日消息，根据甲骨文在官网列出的<a href="https://www.oracle.com/java/technologies/java-se-support-roadmap.html" target="_blank">时间表</a>，Java 7 将在 <span class="accentTextColor">2022 年 7 月底</span>结束生命周期，正式停止对该版本的扩展支持。</p><p style="text-align: center;" data-vmark="130f"><img src="https://img.ithome.com/newsuploadfiles/2022/7/ee4fcfc3-235c-478f-a757-be783f7a4251.png" w="1440" h="543" title="Java 7 生命周期结束" width="1440" height="309" referrerpolicy="no-referrer"></p><p data-vmark="097e">根据官方的说法，当生命周期结束后，该产品将进入持续支持模式，<span class="accentTextColor">不会提供进一步的补丁更新</span>（不提供错误修复、安全修复或功能实施），仅提供有限的支持。</p><p data-vmark="4d5a">此外，随着月底对 Java SE 7 的扩展支持结束，各种甲骨文融合中间件产品的某些旧版本将不再提供经过认证的 JDK。</p><p data-vmark="af9c">IT之家了解到，甲骨文<a href="https://support.oracle.com/knowledge/Middleware/2809708_1.html" target="_blank">建议</a>用户升级到受支持的 Java SE 版本，例如 Java SE 8 或 Java SE 11，以获得对其 Java 运行时的全面支持。</p><p style="text-align: center;" data-vmark="3d50"><img src="https://img.ithome.com/newsuploadfiles/2022/7/ea3fbebd-8b1d-41b5-81d3-e4b5d6d1eb01.png" w="1105" h="430" title="Java 7 生命周期结束" width="1105" height="319" referrerpolicy="no-referrer"></p><p data-vmark="5ed1">此外，Java SE 8 的支持生命周期直到 <span class="accentTextColor">2030 年底</span>才会结束，这个版本是国内不少软件公司、开发者都在用的，还不用太过于担心升级问题。</p><p style="text-align: center;" data-vmark="72a5"><img src="https://img.ithome.com/newsuploadfiles/2022/7/be0a148b-03bb-46c5-96c3-99430d04967d.jpg@s_2,w_820,h_492" w="1440" h="864" title="Java 7 生命周期结束" srcset="https://img.ithome.com/newsuploadfiles/2022/7/be0a148b-03bb-46c5-96c3-99430d04967d.jpg 2x" width="1440" height="492" referrerpolicy="no-referrer"></p>
