<p data-vmark="ee30">根据世界半导体贸易协会（WSTS）统计的数据，继 2020 年芯片设计 IP 销售额同比增长 16% 之后，<span class="accentTextColor">2021 年设计 IP 销售额同比增长了 19.4%，达到 54.5 亿美元</span>。与此同时，2021 年半导体销售额同比增长了 26.2%。基于此，IPnest 于 2022 年 5 月发布了《设计 IP 报告》，按各类别对 IP 供应商进行了排名。</p><p data-vmark="f5c5" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/e6df1244-9864-4cd8-ae1c-fd9e13c81725.png" w="816" h="324" title="WSTS：2021 年全球芯片设计 IP 销售额同比增长 19.4% 至 54.5 亿美元" width="816" height="324" referrerpolicy="no-referrer"></p><p data-vmark="a704">2021 年设计 IP 的主要趋势对大多数 IP 供应商来说都非常积极，特别是 Synopsys 增长了 21.7%，Imagination Technologies（IMG）增长了 43.4%，闪存编译器供应商（超捷半导体，力旺电子）和 Alphawave 增长达 100% 以上。</p><p data-vmark="108a">显然，Synopsys 和 Alphawave 的增长证实了<span class="accentTextColor">有线接口 IP 市场</span>的重要性 (该类别的增长为 22.7%)，与以数据为中心的有线接口应用程序、超标量、数据中心、网络或 IA 相一致。但是，ARM 和 IMG 的良好表现证明了智能手机行业的复苏，以及汽车行业的兴起是一个增长载体。</p><p data-vmark="df6a" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/49e009f8-0104-4ae3-b5f2-f251c650ee37.png" w="821" h="274" title="WSTS：2021 年全球芯片设计 IP 销售额同比增长 19.4% 至 54.5 亿美元" width="821" height="274" referrerpolicy="no-referrer"></p><p data-vmark="52a2">回顾 2016 至 2021 年 IP 市场的发展历程，可以从中获得 IP 市场主要趋势的信息。<span class="accentTextColor">全球 IP 市场增长了 59.3%</span>，而前三大供应商的增长并不平衡。排名第一的 ARM 增长了 33.7%，排名第二的 Synopsys 增长了 140.9%，排名第三的 Cadence 则增长了 167.2%。</p><p data-vmark="93c6">相对而言，市场份额信息更为重要。ARM 的市场份额从 2016 年的 48.1% 下降到 2021 年的 40.4%，Synopsys 的市场份额从 2016 年的 13.1% 增长到 2021 年的 19.7%（或从 2016 年到 2021 年市场份额增长了 50%），而 Cadence 的市场份额从 2016 年的 3.4% 增长到 2021 年的 5.8%。</p><p data-vmark="14c4">此外，还可以通过对比 2016 年至 2021 年的复合年均增长率来进行综合分析。其中，一个强劲的信息是，2016 年至 2021 年设计 IP 市场的复合年均增长率接近 10%。同样值得注意的是，Synopsys 的复合年均增长率为 19.2%，是 ARM（复合年均增长率为 6%）的三倍多。</p><p data-vmark="1d31">IPnest 还根据授权和版税 IP 收入计算了 IP 供应商的排名：</p><p data-vmark="447d" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/0592630b-6296-4fda-b765-65678ff5b5d1.png" w="830" h="334" title="WSTS：2021 年全球芯片设计 IP 销售额同比增长 19.4% 至 54.5 亿美元" width="830" height="330" referrerpolicy="no-referrer"></p><p data-vmark="7178"><span class="accentTextColor">按 IP 授权收入计算，2021 年 Synopsys 以 31.2% 的市场份额排名第一</span>，而 ARM 则以 25.6% 的市场份额排名第二。Alphawave 创建于 2017 年，现在排名第四，仅次于 Cadence，这表明高性能 SerDes IP 对于以数据为中心的现代应用非常重要。</p><p data-vmark="bba6">事实上，Synopsys 的良好表现在一定程度上与他们对有线接口的高度关注有关，他们在 13 亿的市场中占有 55.6% 的份额，而高性能 SerDes 是互连市场的主要支柱。Synopsys 采用了“一站式”战略，支持几乎所有协议（USB、PCIe、以太网、SATA、HDMI、MIPI、DDR 内存控制器），并在每个协议中占有领先的市场份额。</p><p data-vmark="88e6">Alphawave 在某种意义上是互补的，因为他们的战略更倾向于“Stop-For-Top”，限制了他们对前沿技术节点上最先进产品的支持。但如果着眼于 2021 年的设计 IP 结果，便会发现这两者都能够获得成功，并遵循不同的策略和市场定位。</p><p data-vmark="a963" style="text-align: center;"><img src="https://img.ithome.com/newsuploadfiles/2022/5/1f316ed8-8ea9-40b6-adc8-bf70da74d14d.png" w="829" h="343" title="WSTS：2021 年全球芯片设计 IP 销售额同比增长 19.4% 至 54.5 亿美元" width="829" height="339" referrerpolicy="no-referrer"></p><p data-vmark="7edf"><span class="accentTextColor">2021 年的版税排名显示，ARM 以 60.8% 的市场份额占据主导地位</span>，如果考虑到他们的客户安装基数和他们在智能手机行业的强大地位，这并不令人惊讶。而更令人惊讶的是，超捷半导体和 Imagination Technologies（IMG）的复苏，使他们在前五名中分别排名第二和第三。</p><p data-vmark="40c0">超捷半导体正受益于微控制器市场的回暖，因为他们配备了大多数售出的微控制器产品。而 IMG 已经能够克服几年前苹果公司带来的寒流，并在智能手机之外的各个细分市场重新定位为现代 GPU 供应商，如汽车娱乐、智能电视或平板电脑。</p><p data-vmark="0da5">2021 年设计 IP 行业同比增长了 19.4%，证明了这一细分市场在半导体市场中非常强劲，同时过去 2016 年至 2021 年间 9.8% 的复合年均增长率是一个很好的衡量标准。IPnest 还对设计 IP 的未来 5 年进行了预测（尚未公布），即 2026 年设计 IP 市场将达到 110 亿美元，而复合年均增长率（2021 年至 2026 年）为 15%。</p>
