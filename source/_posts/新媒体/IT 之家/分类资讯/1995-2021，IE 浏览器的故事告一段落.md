<p>现在我们的上网方式，对比 10 年前有着特别明显的变化：那就是能用手机做的事儿，绝不用电脑。刷微博？谁用电脑看网页版呀，还是用手机微博刷对味儿。打游戏？以前玩电脑游戏的那批玩家，现在估计都跑去玩王者荣耀和吃鸡了。</p><p>相比不便携的电脑，随时都可以开黑的手机游戏不香吗？</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/4d1514ee-4db1-4850-a6d6-62d301998ef8.jpg" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>其实从娱乐方面来看，手机已经在很大程度上代替电脑了。但是，要是把时间调回十多年前，电脑才是每家每户上网冲浪必备的神器。</p><p>因为那时候也没啥精品 3A 大作可玩，小雷联网后，大部分时间都泡在 IE 浏览器里访问各种网页。而在当时被用来访问网页和玩小游戏的「IE 浏览器」，最近再次被微软官方宣布“死刑”。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/e58ff149-9eca-4ecf-b131-78fb1cc84e42.png" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>虽然在很长一段时间里，咱们早已默认 IE 凉凉，但由微软官方来给 IE 浏览器亲自盖上棺材，还是有点唏嘘。根据微软官方的计划，Internet Explorer 11 的桌面版本将在 2022 年 6 月 15 日正式“死亡”。</p><p>除了企业专用的 LTSC 版，咱们平时自用的 Win 10 将不再内置 IE 浏览器。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/9d36de50-ff5f-43f0-812c-fbd7a38c085b.png" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>取而代之的，是微软另一款搭载了 Chromium 内核，并混得风生水起的 Edge 浏览器。甚至连微软 Edge 首席项目经理肖恩・林德赛都发话了：你的下一个 IE，将以 Edge 呈现。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/05d44a73-64b2-4669-a7fc-6eb2e54294c2.png" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>可是问题来了，现在很多政府和银行网站还很依赖 IE 浏览器，一些年久失修的学校系统也只支持 IE 浏览器访问。如果 IE 就这么凉了，我们岂不是打不开学校和银行的网站？</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/2abc1fe0-b3e7-442b-a646-50a21afe8371.png" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>别慌，其实微软早就考虑到这个问题了，所以在新版 Edge 中增加了 IE 兼容模式。在设置中打开这个开关后，以后遇到不兼容的网页，就可以兼容模式来访问。</p><p>当然，微软最后还是会完全去 IE 化，只是按目前的情况来看，IE 兼容模式至少会被保留到 2029 年。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/0f511e0d-25d2-4519-978c-c6f08d867973.png" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>其实从微软的这几年的操作就能看出，它早就让 IE 浏览器退出历史舞台了。</p><p>杀死 IE，复兴 Edge，这就是微软目前最想做的事儿。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/fa7ab43a-7b93-46f3-b80b-de00f732e9ad.jpg" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>这就很奇怪喽，IE 在当年可是微软最亲的儿子，还生怕自己喂不饱它呢。</p><p>可二十多年过去，微软却要扼杀亲自养肥的 IE，这中间到底经历了啥？</p><p>这就是小雷今天要和大伙聊的一个重点了。</p><p>从 1995 年，IE 浏览器诞生开始，它就享受着微软爸爸毫不吝啬的资源倾斜。</p><p>比如，它刚出生就被内置在市场份额巨大的 Windows 系统上，成为每个 Windows 用户第一个使用的浏览器。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/6efdf1e6-3ff3-4ab9-b90d-8772a0eb5a30.jpg" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>靠着捆绑安装和免费使用的优势，IE 浏览器在 2002 年 - 2003 年拿下高达 95% 的市场份额，成功干掉了最大的竞争对手 Netscape。</p><p>要知道，巅峰时的 Netscape 最高拥有 90% 以上的市场占有率，新功能加得还快，吸引了不少用户使用。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/aa36a0ad-3f35-4115-b39f-6e46d4912c94.png" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>可惜人家微软财力雄厚，有着当时最好的技术人员，背后还有垄断了市场的操作系统。</p><p>最后网景 Netscape 被 IE 捶破头，便退出浏览器市场，IE 一家独大的时代正式开启。</p><p>但事实证明，一个产品得有竞争对手，才能有卓越的进步。</p><p>毕竟后来处于垄断阶段的 IE 活成啥样，大伙都看得到...</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/0e83d02a-20a6-439e-971a-e0c525b42456.jpg" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>微软当时做 IE 浏览器这个产品的初衷，就是为了干翻 Netscape。</p><p>因为对手 Netscape 经常在浏览器中加入新功能，微软也不敢懈怠，在刚开始投入了不少精力去写代码加功能。</p><p>期间不断地投入和创新，让 IE 逐渐成为竞争力十足的浏览器。</p><p>比如在 1996 年推出的 IE 3，它是市面上第一款支持编程语言和 CSS 的浏览器，甚至能直接显示 GIF 和播放歌曲。</p><p>也是从这个版本开始，IE 才真正成为 Netscape 最大的竞争对手。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/2a8e8db5-e7a0-42ce-becc-3b19e4fad752.png" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>随后在 1997 年发布的 IE 4.0，更是让 IE 拿下超过 60% 的市场份额，超越 Netscape，成为浏览器一哥。</p><p>因为，这个版本的 IE 首次使用了自研的 Trident 排版引擎，这个引擎被沿用到 IE 11，被俗称为 IE 内核。</p><p>大伙别小看这玩意儿，这个引擎对于当时的浏览器来说算得上革命性创新。</p><p>一来，Trident 允许开发者调用，让开发者的软件有了浏览网页的能力。</p><p>二来，搭载了 Trident 引擎的 IE，有了加载动态网页的能力，让浏览网页的体验得到质的提升。</p><p>光是这两个特性，就让老前辈 Netscape 看得脑壳痛，毕竟自己还真不够技术去跟 IE 硬碰硬了。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/7f06a0cc-ae9b-4c95-bff1-c27356ea54e2.png" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>你瞧，有竞争对手的 IE 就是这么给力，一代比一代强。</p><p>但自从把 Netscape 干掉后，IE 前进的脚步肉眼可见地放缓，微软也不再像以前那样投入大量资源更新维护。</p><p>自己不花心思维护，软件的漏洞就会被黑客大佬一个一个地挖出来。</p><p>从 IE 5 开始，IE 几乎每一年都会被曝出有严重的安全漏洞，一点也不让用户省心。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/7fcf207e-0ed1-46dd-99e6-21a360a13f38.png" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>IE 的口碑也是从这时候开始崩坏，连小雷这个老用户都觉得 IE 兼容性越来越差，安全性也越来越低。</p><p>与此同时，各种吐槽 IE 的梗图开始流传在网上。</p><p>比如世界上最慢的四个东西，其中一个就是 IE...</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/70bff2b0-c4e6-4b06-beca-48a3b70f6e13.jpg" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>还有这个堪称经典的浏览器对比图。</p><p>人家谷歌浏览器都说了好几句话，IE 居然隔了好久才反应过来。</p><p>卡、慢、崩溃、不安全成为了 IE 撕不掉的标签。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/55a5532c-144c-4dcb-8001-a43bb3b781ca.jpg" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>虽然微软在后续对 IE 进行了很多修复和优化，但问题还是没得到解决。</p><p>时代在前进，更何况是日新月异的互联网时代。</p><p>不求进步的 IE 很开被 Opera、Firefox 和 Chrome 这几家新兴的浏览器吃掉市场份额。</p><p>这几个浏览器，要么加载网页快，要么功能丰富，支持多样的拓展，要么安全性高，能有效保护用户隐私。</p><p>而 IE 呢？不好意思，它啥也没有，啥也不是。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/8c9d829c-73d3-4666-b075-4c2ba32a5311.jpg" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>现在的 IE，是连微软自己都嫌弃的软件...</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/04dfe421-7e77-40dc-a6ef-82735126c2e3.png" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p><p>也许在某些学校和政务系统里，IE 还能活一段时间，但在大部分用户眼里，IE 浏览器早就是个名存实亡的软件了。</p><p>都啥年代了，还用 IE？搭载了 Chromium 的 Edge 不香吗？</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/5/e741d830-6561-4c52-9816-54c232503115.jpg" title="1995-2021，IE 浏览器的故事告一段落" referrerpolicy="no-referrer"></p>
