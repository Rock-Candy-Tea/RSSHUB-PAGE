            <p data-vmark="88eb"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 8 月 3 日消息，微软名为 PowerToys 的免费系统实用工具套件从 Windows XP 时代复活，并于 2019 年 5 月引入 <a class="s_tag" href="https://win10.ithome.com/" target="_blank">Windows 10</a> 系统。可以用于高级用户调整和简化 Windows 操作，以提高效率。通过与 Windows 10/<a class="s_tag" href="https://win11.ithome.com/" target="_blank">Windows 11</a> shell 搭配使用，同时让用户可以自定义各个工作流。</p><p data-vmark="6b6a">近日，PowerToys 迎来 <span class="accentTextColor">0.61.1 版本</span>，包含始终置顶、FancyZones 和 PowerToys Run 等方面的多项错误修复。</p><p style="text-align: center;" data-vmark="29e9"><img src="https://img.ithome.com/newsuploadfiles/2022/8/d2e3243f-f131-430c-9056-53dc23ced36a.png" w="918" h="708" title="微软 Win11/10 免费工具集 PowerToys 0.61.0 发布：修复窗口置顶、查找鼠标等功能的 Bug" width="918" height="632" referrerpolicy="no-referrer"></p><p data-vmark="cbda"><strong>主要更新内容：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="0342">修复了导致更新 PowerToys 时未创建 PowerToys 开始菜单快捷方式的问题。如果用户手动将其删除，则不会在更新时创建新的 PowerToys 快捷方式。</p></li><li><p data-vmark="ba66">将 Windows App SDK 运行时升级到 1.1.2。</p></li><li><p data-vmark="6edc">修复了始终置顶功能激活时边框闪烁、出现黑色边缘、CPU 占用 100%、退出时错误挂起的问题。</p></li><li><p data-vmark="514e">修复了 FancyZones 中窗口布局在多显示器、虚拟显示器、竖向显示器中的错误。</p></li><li><p data-vmark="8c54">修复了查找鼠标功能中的聚光灯在屏幕左上角挂起的问题。</p></li><li><p data-vmark="9136">修复了 PowerToys Run 中 WindowWalker 插件 UI 的错字，支持查找 win32 程序的本地化名称。</p></li><li><p data-vmark="1af5">将 .NET 依赖项更新到 6.0.7。</p></li></ul><p data-vmark="5993">IT之家小伙伴可以在<a href="https://aka.ms/getPowertoys" target="_blank">微软 Windows 11/10 应用商店</a>或 <a href="https://github.com/microsoft/PowerToys/releases/tag/v0.61.0" target="_blank">GitHub</a> 更新 PowerToys，不过由于数据源都是 GitHub，所以后者更方便一些。</p><p
