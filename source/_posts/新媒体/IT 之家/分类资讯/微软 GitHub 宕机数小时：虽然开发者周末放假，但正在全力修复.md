<p data-vmark="1ae0"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 28 日消息，据 mspoweruser 报道，<span class="accentTextColor">微软 GitHub 已经宕机了数小时</span>，虽然开发者将在本周末放假休息，但他们已经在修复了。</p><p data-vmark="2a4e"><img src="https://img.ithome.com/newsuploadfiles/2021/11/3d031e09-2cfb-48f9-a53f-8a6f06896eca.png" w="948" h="363" title="微软 GitHub 宕机数小时：虽然开发者周末放假，但正在全力修复" width="948" height="314" referrerpolicy="no-referrer"></p><p data-vmark="f6bd">IT之家查看 GitHub 显示，目前部分地区网站服务仍无法连接，显示 500 错误，也就是服务器内部错误；部分地区的服务已经可用。</p><p data-vmark="7a18"><img src="https://img.ithome.com/newsuploadfiles/2021/11/e1908ea1-cc4b-42cf-b4af-e9b9cd7ec0b1.png" w="1440" h="669" title="微软 GitHub 宕机数小时：虽然开发者周末放假，但正在全力修复" width="1440" height="381" referrerpolicy="no-referrer"></p><p data-vmark="1e06">微软在 2018 年 10 月以 75 亿美元的价格购买了代码存储库网站 GitHub，一些用户表示该服务自那以后变得越来越不可靠。然而，GitHub 在微软的管理下继续保持强劲增长，从购买时的不到 4000 万用户增长到现在的超过 7300 万用户。</p><p data-vmark="26ce">目前官方已经知晓了这一问题，正在全力修复中，部分服务仍处于 Degraded 和 Incident 状态。</p><p data-vmark="05d9"><img src="https://img.ithome.com/newsuploadfiles/2021/11/e79616cb-2816-4a33-865b-bf5ccbc99830.png" w="1255" h="845" title="微软 GitHub 宕机数小时：虽然开发者周末放假，但正在全力修复" width="1255" height="552" referrerpolicy="no-referrer"></p><p data-vmark="5d4c"><img src="https://img.ithome.com/newsuploadfiles/2021/11/08808fdf-1d35-414c-a926-2e138f3e879a.png" w="1289" h="663" title="微软 GitHub 宕机数小时：虽然开发者周末放假，但正在全力修复" width="1289" height="422" referrerpolicy="no-referrer"></p>
