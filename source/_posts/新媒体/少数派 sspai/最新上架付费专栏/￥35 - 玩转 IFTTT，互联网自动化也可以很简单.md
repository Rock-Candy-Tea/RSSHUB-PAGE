<img src="https://cdn.sspai.com/article/dd3858a1-720e-c955-495c-cb4698ad068b.jpg" style="max-width: 100%;" referrerpolicy="no-referrer"><p><strong>购前须知：</strong>本教程基于 2018 年版本的 IFTTT 写作，后 IFTTT 更改了付费机制，因此部分内容可能不适用于 IFTTT 最新版本。请读者在购买前考虑到这一点。</p><p>进入过互联网时代，我认为的满足以下两点，才算真正的实现「智能生活」：</p><ol><li>让复杂的事情简单化</li><li>让重复的事情自动化</li></ol><p>只要满足了这两点，就一定会大大提高我们的工作和生活效率，而 IFTTT 恰恰是这方面的专家，也是最容易入门的服务。IFTTT 自 2010 年创办至今，不停地维护和升级，对互联网上比较常见的服务都做了相应支持，目前已经支持 44 个分类里的 1120 个服务，并且这个数字还在不断的增加中。</p><p>IFTTT 作为「任务自动化服务」的鼻祖，其 If This Then That 的模式已经成为行业标杆。IFTTT 在产品设计上符合人性，界面也相对简洁，但是纯英文的界面，使得许多人望而却步。本系列教程通过初识篇、基础篇、本地应用篇、进阶篇、硬件篇的顺序，以及通俗易懂的语言和大量的配图说明，带领大家循序渐进地掌握并玩转 IFTTT。</p><ul><li><strong>初识篇</strong>：通过梳理信息孤岛是如何产生的，引出如何打破服务和服务之间的壁垒，再讲到 IFTTT 产生的必然性，充分了解 IFTTT 的前世今生，知其然才能知其所以然；</li><li><strong>基础篇</strong>：分别通过提醒类、定时类、社交类、任务管理类、网络存储类等实例，了解 IFTTT 的基础用法；</li><li><strong>本地应用篇</strong>：通过 IFTTT 应用结合手机本地的应用服务，自动同步照片，自动备份通讯录，快速拍照记录灵感灯操作，让我们的手机更加智能；</li><li><strong>进阶篇</strong>：本篇分两部分，第一部分串联多个 IFTTT 任务，组成一个大的自动化任务集合，来完成较为复杂的任务；第二部分介绍了 Webhooks 的用法，我们知道 IFTTT 的核心是 API 的调用，对于那些没有公开 API 的服务，我们依然可以自己构建 API，使其自动化；</li><li><strong>硬件篇</strong>：本篇提供了大量的智能硬件和 IFTTT 结合的例子，类似如果降温将灯的颜色变成蓝色，如果打开房门，自动点亮玄关灯之类。从线上到线下，让我们的生活更智能。</li></ul><h2>作者介绍</h2><p>王掌柜，少数派效率类文章作者，熟悉 Workflow、IFTTT、Zapier 等效率类软件。代表作品有《王掌柜带你玩转 Zapier - Zapier VS.
