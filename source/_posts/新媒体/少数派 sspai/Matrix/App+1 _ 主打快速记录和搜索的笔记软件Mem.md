<div class="articleWidth-content" data-v-1eaafa2a><div class="content wangEditor-txt minHeight" data-v-1eaafa2a><h2>前言</h2><p>笔记软件层出不穷，作为一个工具党，根本无法停止试用新软件的手。这次，我想介绍一款较为轻量但功能不少的笔记软件。看Mem是怎么在Obsidian、Notion、Roam Research 和logseq 之类的热门软件之中显现出自己的特色。</p><h2>什么是Mem</h2><figure class="image ss-img-wrapper image_resized" style="width:579px;"><img src="https://cdn.sspai.com/2021/12/04/576f97121eddc933036ece0c259ae3bb.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/576f97121eddc933036ece0c259ae3bb.png" referrerpolicy="no-referrer"></figure><blockquote><p>什么是 Mem？Mem 是捕获，连接和共享信息的最快方法。它与 Apple Notes 一样简单，并由协作图形数据库提供支持。它具有闪电般的快速捕获，始终在线的搜索和无缝协作的功能 - 全部集中在一处。</p></blockquote><h2>Mem Spotlight</h2><p>Mem Spotlight 是 Men 的主打功能，可以用 Command + Shift + Space 快速启动，很像Apple 的 Spotlight 或者 Windows 的 Everything。</p><figure class="image ss-img-wrapper image_resized" style="width:584px;"><img src="https://cdn.sspai.com/2021/12/04/8d3ccfa42d05208eecce8b0d0629c8c8.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/8d3ccfa42d05208eecce8b0d0629c8c8.png" referrerpolicy="no-referrer"></figure><p>这个功能大大加快了我的信息记录速度和信息检索速度，这种流畅感是很多笔记软件无法带给我的，下面看看 Mem 是怎么实现这种流畅感的：</p><h3>搜素</h3><p>Mem 的搜索非常强大，举个例子：</p><figure class="image ss-img-wrapper image_resized" style="width:583px;"><img src="https://cdn.sspai.com/2021/12/04/36ca5fb4d6e6ae89f3ae4f2ed4687d03.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/36ca5fb4d6e6ae89f3ae4f2ed4687d03.png" referrerpolicy="no-referrer"></figure><p>如果你建立了一个联系人的 Mem 卡片，里面记录了邮箱、手机号、公司等信息，你要做的就是在电脑任何地方，按下快捷键，然后搜素联系人+邮箱/手机，你就会得到你想要的信息</p><figure class="image ss-img-wrapper image_resized" style="width:579px;"><img src="https://cdn.sspai.com/2021/12/04/2e8efd664bdbbff5a33130dc5286a345.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/2e8efd664bdbbff5a33130dc5286a345.png" referrerpolicy="no-referrer"></figure><p>如果你建立了一个会议记录的 Mem 卡片，你可以直接搜素标题+想法/总结，信息就会自动显示出来</p><figure class="image ss-img-wrapper image_resized" style="width:573px;"><img src="https://cdn.sspai.com/2021/12/04/f6ce6f79ec6bd43792a553a7b2d08295.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/f6ce6f79ec6bd43792a553a7b2d08295.png" referrerpolicy="no-referrer"></figure><p>如果你忘记了某项事情的具体时间，只要你在卡片里提到了时间，就可以通过搜索检索出来</p><h3>剪藏</h3><p>如果你在浏览器的某个页面，只要唤起 Spotlight，就可以直接收藏网址，并添加笔记</p><figure class="image ss-img-wrapper image_resized" style="width:561px;"><img src="https://cdn.sspai.com/2021/12/04/2fcfd49fbc88e11245de9a103cfeb1bd.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/2fcfd49fbc88e11245de9a103cfeb1bd.png" referrerpolicy="no-referrer"></figure><h2>记录</h2><p>通过快速的信息记录和检索，我可以很快的把他融入进我的工作流里并且不打断我的思绪，Mem 很现代的摒弃了传统的文件夹整理，用反向链接和标签来连接起笔记（新工具标配），排版也用了Markdown，鉴于我的主力笔记软件是 obsidian 和 logseq，上手完全是没有什么困难的。</p><p>值得注意的是，它的反向链接的语法不是大部分软件用的[[]]，而是用 <strong>+ 加 页面名称</strong>的语法。</p><p>还有一个让我很惊喜的特色是沉浸模式，输入的时候自动进入沉浸模式，有意思的是先隐藏掉所有的操作栏，如果持续编辑的情况下，会过几秒隐藏掉右侧的关联列表。当你停止编辑了之后，它又会出来。方便了我在沉浸写作的时候会想参考其他文章的习惯，很好的设计。<br> </p><figure class="image ss-img-wrapper image_resized" style="width:570px;"><img src="https://cdn.sspai.com/2021/12/04/7582e44eb8eabff2d71e3b2d4cb2ed72.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/7582e44eb8eabff2d71e3b2d4cb2ed72.png" referrerpolicy="no-referrer"><figcaption>刚开始写作时</figcaption></figure><figure class="image ss-img-wrapper image_resized" style="width:571px;"><img src="https://cdn.sspai.com/2021/12/04/7528e6cee2cfed85b725b482244bf15a.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/7528e6cee2cfed85b725b482244bf15a.png" referrerpolicy="no-referrer"><figcaption>写作几秒后</figcaption></figure><h2>Mem Timeline</h2><figure class="image ss-img-wrapper image_resized" style="width:565px;"><img src="https://cdn.sspai.com/2021/12/04/10be1b5c3bc041db4e0be1aadd943fad.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/10be1b5c3bc041db4e0be1aadd943fad.png" referrerpolicy="no-referrer"></figure><p>Timeline 可以让你写 Daily Review，管理任务，还可以看到你当天那个时间浏览了什么笔记，通过自然语言搜索可以随意回顾每一天的记录情况。同时也提供了一个Dailynote功能，可以通过模板创建每一天的笔记。</p><h2>Mem Inbox</h2><p>所有自动创建的笔记都会进入 Inbox，自己手动创建的也可以添加到 Inbox，官方的建议是将重要的还没处理完的信息放到这里，和邮箱的 Inbox 差不多吧。</p><figure class="image ss-img-wrapper image_resized" style="width:575px;"><img src="https://cdn.sspai.com/2021/12/04/d208e02eaecbc2aa51630f2f8771860f.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/d208e02eaecbc2aa51630f2f8771860f.png" referrerpolicy="no-referrer"></figure><p>Mem 可以通过 Gmail 保存邮件生成笔记，只要你通过自己的 Gmail 转发邮箱到 save@mem.ai，邮件就会自动出现在 Inbox 里。</p><figure class="image ss-img-wrapper image_resized" style="width:575px;"><img src="https://cdn.sspai.com/2021/12/04/57dc866310cbed332baae06efb005264.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/57dc866310cbed332baae06efb005264.png" referrerpolicy="no-referrer"></figure><h2>Mem Flow</h2><figure class="image ss-img-wrapper image_resized" style="width:581px;"><img src="https://cdn.sspai.com/2021/12/04/1ca3b50c8d19b197412ecd74963ad811.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/1ca3b50c8d19b197412ecd74963ad811.png" referrerpolicy="no-referrer"></figure><p>Mem Flow 是我最喜欢的功能之一，它可以通过 <a href="https://zapier.com/" target="_blank">Zapier</a> 连接各种软件，包括但不限于 Gmail、Outlook、Twitter、Discord、Notion、Airtable 和 Spotify 等等。</p><figure class="image ss-img-wrapper image_resized" style="width:571px;"><img src="https://cdn.sspai.com/2021/12/04/18e49922fbad7ec18493968e3f70225b.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/18e49922fbad7ec18493968e3f70225b.png" referrerpolicy="no-referrer"></figure><p>不通过 Zapier，它还能用 SMS<sup class="ss-footnote" href title="不支持中国手机号" footnote-id="1">1</sup>、WhatsApp 和 TG 来快速输入想法。对于从Telegram这里输入的内容来说，有个很贴心的选项是多少分钟内的信息会合并到一条，毕竟很多时候短时间频繁记录的内容大多都是一个主题，也便于后期维护。</p><figure class="image ss-img-wrapper image_resized" style="width:567px;"><img src="https://cdn.sspai.com/2021/12/04/8e5d9cf2a0488daa0eed892140ca043b.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/04/8e5d9cf2a0488daa0eed892140ca043b.png" referrerpolicy="no-referrer"></figure><h2>更多功能</h2><p>Mem 还包括任务管理功能和在线协作功能，我暂时用的不多（毕竟一个学生根本用不到多人协作功能啊😂）</p><h2>总结</h2><p>总的来说，我还是非常喜欢这款笔记软件的，但是我并没有把它当作我的主力软件使用，它有一些软件没有的优点，整体完成度也不错，但还是功能不够充足。</p><p>但是，对于融入工作流，这款软件是一个不错的选择，快速的输入和官方 Api 加持的连接性足以连接起原本工作流之间的小缺憾。</p><p>目前 <a href="https://get.mem.ai/" target="_blank">Mem</a> 支持 Windows、Mac 和 网页版，大家可以去尝试一下</p><p> </p><p><br><br> </p></div><!----></div><div style="border:1px solid
