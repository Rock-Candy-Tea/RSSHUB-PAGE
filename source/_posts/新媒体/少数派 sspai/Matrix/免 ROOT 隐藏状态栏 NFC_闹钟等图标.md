<div class="articleWidth-content" data-v-557a067a><div class="update-wrap" data-v-557a067a></div><div class="content wangEditor-txt minHeight" data-v-557a067a><p>因为最新的 MIUI 测试版 /其它部分 UI 会始终显示 NFC 图标，所以写一下这个图标如何隐藏。</p><h3>准备工作：</h3><ol><li>USB 数据线</li><li>电脑端 adb</li><li>手机打开 USB 调试，部分设备需要打开<strong>USB 调试（安全模式）</strong> 一般会在 USB 调试开关的下方</li></ol><h3>操作步骤：</h3><pre class="language-shell"><code>adb shell settings put secure icon_blacklist nfc</code></pre><p>nfc 可以替换成下面的内容，多个的话可以用,分割。</p><figure class="table"><table><thead><tr><th>图标</th><th>图标描述</th></tr></thead><tbody><tr><td>volume</td><td>静音 /震动图标</td></tr><tr><td>location</td><td>定位图标</td></tr><tr><td>speakerphone</td><td>麦克风使用中图标</td></tr><tr><td>wifi</td><td>wifi</td></tr><tr><td>hotspot</td><td>热点</td></tr><tr><td>alarm_clock</td><td>闹钟</td></tr><tr><td>bluetooth</td><td>蓝牙</td></tr><tr><td>airplane</td><td>飞行模式</td></tr><tr><td>headphone</td><td>耳机</td></tr><tr><td>cast</td><td>录屏状态</td></tr><tr><td>nfc</td><td>nfc</td></tr></tbody></table></figure><p>如果需要恢复，需要使用：</p><pre class="language-shell"><code>adb shell settings put secure icon_blacklist null</code></pre><blockquote><p>注意一：上面这条命令会一次性恢复所有图标</p><p>注意二：如果你需要屏蔽多个图标，那么你需要使用英文逗号“,”将图标名称隔离开来，一次性输入。</p></blockquote><p> </p></div><div class="update-details-wrap"
