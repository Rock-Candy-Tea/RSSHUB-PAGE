<div class="articleWidth-content" data-v-6a669db8><div class="content wangEditor-txt minHeight" data-v-6a669db8><p style="margin-left:0px;"><strong>木及简历</strong>是一个免费的在线 Markdown 简历制作网站，致力于<strong>让你能以更简单的方式写好简历</strong>。木及简历上线 5 个月到目前累计已导出 10000+ 份简历。尽管可能还是一个不够成熟的产品，但在大量用户的反馈和建议中不断成长着，一直在保持着稳定的更新迭代。</p><p style="margin-left:0px;">在线地址：<a href="https://resume.mdedit.online/">https://resume.mdedit.online/</a></p><figure class="image ss-img-wrapper image_resized" style="width:259px;"><img src="https://cdn.sspai.com/2021/09/15/article/131e20aba66174e2f94876b52fe1ac00?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="image-20210911213706595" data-original="https://cdn.sspai.com/2021/09/15/article/131e20aba66174e2f94876b52fe1ac00" referrerpolicy="no-referrer"></figure><h2><strong>诞生的契机</strong></h2><p style="margin-left:0px;">在之前我们找工作写简历时，就发现<strong>「找模板 -> 改格式 -> 编写内容 -> 排版 」</strong>这一套流程繁琐复杂。简历的核心本应该是写好自己的履历技能等内容，却把大量的时间花费在了格式排版上。</p><p style="margin-left:0px;">到后来看候选人的简历，里面不乏有各种花里胡哨让人看不清重要信息的模板，甚至还有各种错乱的 Word 格式文件。</p><p style="margin-left:0px;">我们在认真体验了现在市场上的各种产品来写简历后，发现：</p><p style="margin-left:0px;">Word 排版操作复杂，找模板五花八门。</p><p style="margin-left:0px;">在线简历网站在写好简历后不支持内容的平滑迁移(一键切换其他模版)，版本管理不够方便。</p><p style="margin-left:0px;">复杂的交互会打扰书写的思路。</p><p style="margin-left:0px;"><strong>总结起来就是：现有的产品并无法满足我们现有的需求。</strong></p><h2><strong>思考</strong></h2><p style="margin-left:0px;">有问题就得有思考。</p><p style="margin-left:0px;">由于我们平时写项目文档、总结文章和接口API文档都是用的 Markdown，其语法非常简洁且易上手。使得我们可以专注于文本的内容编辑，不必在文本与工具栏频繁切换，只需要键盘输入就可以完成一份具有各类标题与标注的文档，让我们的思绪得以延续。</p><p style="margin-left:0px;">也正是由于 Markdown 的这些特性，Github上很多程序员都在使用 Markdown 写简历，但是大多数用的是默认的 Markdown 渲染方式，虽然足够简洁，但他看上去并不像一份简历。<strong>只解决温饱问题并不够，我们还想要简历「好看与专业」。</strong></p><p style="margin-left:0px;">至此，木及简历已经有了初步核心的想法理念。我们要做一款以<strong>「简历 = Markdown文本 + 主题」</strong>为核心的简历制作工具，并提供好主题，目的是为了<strong>让你只需专注于自己的内容，就能写好一份「好看与专业」的简历。</strong></p><figure class="image ss-img-wrapper image_resized" style="width:249px;"><img src="https://cdn.sspai.com/2021/09/15/article/0d7a9c5b7c619e309f17e3bc367650e7?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="image-20210911221817881" data-original="https://cdn.sspai.com/2021/09/15/article/0d7a9c5b7c619e309f17e3bc367650e7" referrerpolicy="no-referrer"></figure><h2><strong>不一样的简历体验</strong></h2><h3><strong>01 专注于内容，轻量级 Markdown 书写</strong></h3><p style="margin-left:0px;">我们选用 Markdown 来书写简历，同时支持导入导出 <code>.md</code> 文件。你可以随时随地编写，并可以很方便的对自己的简历进行版本管理，只需维护 Markdown 即可。</p><figure class="image ss-img-wrapper image_resized" style="width:765px;"><img src="https://cdn.sspai.com/2021/09/15/article/8861d294f42099aaaebc7b5fafab9896?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="image-20210912003253922" data-original="https://cdn.sspai.com/2021/09/15/article/8861d294f42099aaaebc7b5fafab9896" referrerpolicy="no-referrer"></figure><h3><strong>02 双栏实时显示文本与简历</strong></h3><p style="margin-left:0px;">我们采用<strong>「左侧 Markdwon 编辑器，右侧简历预览」</strong>的双栏模式，使你能实时预览自己的简历现在是什么样子，做到心中有数。</p><figure class="image ss-img-wrapper image_resized" style="width:765px;"><img src="https://cdn.sspai.com/2021/09/15/article/adf5e0744f6aef7538952190f5ec63dc?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="2021-09-07-00.47.58_macbookgold_front" data-original="https://cdn.sspai.com/2021/09/15/article/adf5e0744f6aef7538952190f5ec63dc" referrerpolicy="no-referrer"></figure><h3><strong>03 一份文本 + 多样主题，构建无限可能</strong></h3><p style="margin-left:0px;">对于传统方式来写简历，如果在写完内容后想要更换一份模板会比较费时费力。</p><p style="margin-left:0px;">而我们提供了<strong>「主题」</strong>，对于同一份 Markdown，你只需简单<strong>选择并切换主题</strong>即可更换为不同的简历样式，构建无限可能。</p><figure class="image ss-img-wrapper image_resized" style="width:765px;"><img src="https://cdn.sspai.com/2021/09/15/article/de2da2498b77419ae5c3ad8556842281?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="2021-09-07-00.50.17_macbookgold_front" data-original="https://cdn.sspai.com/2021/09/15/article/de2da2498b77419ae5c3ad8556842281" referrerpolicy="no-referrer"></figure><h3><strong>04 智能一页，自动识别，完美排版</strong></h3><p style="margin-left:0px;">我们还提供了智能一页的功能，能够帮助你更好地应对以下尴尬的场景，使你的简历变得更加<strong>「精致」</strong>。</p><figure class="image ss-img-wrapper image_resized" style="width:765px;"><img src="https://cdn.sspai.com/2021/09/15/article/a180073f4c081751d29eb04a2ef71c0c?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="mujijianli-zhineng" data-original="https://cdn.sspai.com/2021/09/15/article/a180073f4c081751d29eb04a2ef71c0c" referrerpolicy="no-referrer"></figure><h3><strong>05 智能分割</strong></h3><p style="margin-left:0px;">通过我们的智能分割算法，能够自动识别简历的边距，以最合适的距离进行排版。</p><figure class="image ss-img-wrapper image_resized" style="width:765px;"><img src="https://cdn.sspai.com/2021/09/15/article/7b8f004488d49f4daadec23e2c82f667?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="image-20210910014404617" data-original="https://cdn.sspai.com/2021/09/15/article/7b8f004488d49f4daadec23e2c82f667" referrerpolicy="no-referrer"></figure><h3><strong>06 自动云端存储，数据永不丢失</strong></h3><p style="margin-left:0px;">书写简历时，我们会帮你自动存储 Markdown 到云端，保证数据不会丢失。同时可以不受任何地点/时间的限制，随写随关，下次登录后即可接着继续写。</p><figure class="image ss-img-wrapper image_resized" style="width:765px;"><img src="https://cdn.sspai.com/2021/09/15/article/916b24464210c77a107f857af7bcc678?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="yuncunchu" data-original="https://cdn.sspai.com/2021/09/15/article/916b24464210c77a107f857af7bcc678" referrerpolicy="no-referrer"></figure><h2><strong>为什么叫木及</strong></h2><p style="margin-left:0px;">木和及两个字拼在一起就是"极"，汉字释义为"顶点;尽头"，与简历的"简"字组合在一起就是极简。</p><p style="margin-left:0px;">为的是想<strong>传达我们"极简"的理念</strong>，正如著名设计师迪特·拉姆斯(Dieter Rams )的设计理念"<strong>Less, but better</strong>"，我们希望剔除复杂的流程，使木及简历看起来简洁，使用起来简单。</p><p style="margin-left:0px;">因此我们不管是在「自身网站的设计」还是「简历模板的提供」，一如既往的追求着极简的风格。</p><figure class="image ss-img-wrapper image_resized" style="width:765px;"><img src="https://cdn.sspai.com/2021/09/15/article/afce40db9fd238abf977a65aa83e8fe5?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="2021-09-11-22.58.40_macbookgold_front" data-original="https://cdn.sspai.com/2021/09/15/article/afce40db9fd238abf977a65aa83e8fe5" referrerpolicy="no-referrer"></figure><h2><strong>日拱一卒</strong></h2><p style="margin-left:0px;">木及简历从最初只是一个雏形(开源于<a href="https://github.com/hua1995116/react-resume-site">Github</a>)到现在，经历了构思、设计、技术选型、代码编写、测试、收集反馈、深挖需求、更新迭代这一系列过程。</p><p style="margin-left:0px;">上线 5 个月，经历了<a href="https://resume.mdedit.online/changelog/">15 次版本迭代</a> ，上线了 9 大核心功能点，累计解决了 30+ 的线上 BUG。我们不断优化用户体验，积极响应用户的反馈，形成了良好的社群氛围。例如智能一页、证件照、滚动跟随、字体切换等功能，都是由用户的建议中发展而来的。</p><h2><strong>未来的木及简历</strong></h2><p style="margin-left:0px;">一个产品的价值永远应该建立在<strong>「如何帮助使用它的人实现价值」</strong>之上。</p><p style="margin-left:0px;">我们不希望在简历制作过程中，你被一些复杂的交互所干扰，或是为设计而焦虑。而是希望帮助你聚焦核心，更多地专注于自己的<strong>思考与总结</strong>，写出满意的简历，走好职场的每一步。</p><p style="margin-left:0px;">未来我们还将会在以下几个方向发力：</p><h3><strong>01 共建</strong></h3><p style="margin-left:0px;">提供模板开发生态工具链，每一个人都可以参与木及简历模板的设计和开发。满足自己的同时，还可以将自己的模板分享给他人，共建木及简历生态。</p><h3><strong>02 分享</strong></h3><p style="margin-left:0px;">提供在线简历预览，能够让大家自主地分享在线简历。</p><h3><strong>03 快捷</strong></h3><p style="margin-left:0px;">优化编辑器的使用，提供快捷命令，让你拥有更沉浸式的简历书写体验。</p><h3><strong>04 多端</strong></h3><p style="margin-left:0px;">提供移动端 APP，更加便捷地随时随地修改自己的简历。</p><h2><strong>鸣谢</strong></h2><p style="margin-left:0px;">排名不分先后：</p><p style="margin-left:0px;">感谢阮一峰周刊、Hello Github、玩转 GitHub、不死鸟、Java Guide、木子淇、EatmyMark 等众多知名网站/博主对木及简历的推荐。</p><p style="margin-left:0px;">感谢目前两个共建群里的朋友们积极反馈问题与提供建议。</p><p style="margin-left:0px;">感谢每个推荐和分享过木及简历的你。</p><p style="margin-left:0px;">投入大量业余时间来维护是件很不容易的事，但有了你们的支持与鼓励，木及简历一定会走的越来越好。</p><h3 style="margin-left:0px;">项目成员</h3><figure class="ss-imgRows" figcaption><img
