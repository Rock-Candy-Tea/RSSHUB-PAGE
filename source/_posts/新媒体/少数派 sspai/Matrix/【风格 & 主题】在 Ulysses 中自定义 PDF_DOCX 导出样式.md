style="margin-left:0px;"><strong>适用于初学者的 Ulysses 样式表</strong></h2><p>以下会为你逐步讲解 Ulysses 样式表，且不需要你预先掌握任何其他样式表语言的知识。</p><p>唯一的要求是需要你有一个文本编辑器。为了实现自动完成和语法彩色高亮，我们建议使用 <a href="https://macromates.com/" data-original="https://cdn.sspai.com/2021/11/02/article/fb44e6e9146276012c33d9d66f157484" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">如果你浏览整个文件，你会发现几乎所有的 Ulysses 标记语言都有相应的区段。例如：<a href="https://ulysses.app/styles/reference/paragraph" target="_blank">heading-all</a> 区段描述了标题的通用样式。另一个区段，<a href="https://ulysses.app/styles/reference/inline" Ulysses 何时应该用怎样的样式级别。选择符后面紧跟着一个被花括号 &#123; &#125; 括起来的”代码块”。这个代码块涵盖了实际的样式设置，包括设置名称与数值。比如在上面的例子中，规定所有的标题都应使用 “Futura” 字体。</p><p style="margin-left:0px;">现在，让我们练习一下吧：将你样式表中 <a 。创建一个新文稿，并包含以下内容：</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/11/02/45a58509c6534bf5db0c9e6e18427b95.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" Next” 字体。</p><p style="margin-left:0px;"> </p><h4 style="margin-left:0px;"><strong>无需重复！</strong></h4><p style="margin-left:0px;">当创建一个样式时，你无需不断重复编辑操作：Ulysses 不会为每个标记单独设置字体系列或大小，而是自动选择正确的默认值。这样，你只需在一个地方改变设置即可，而不会因此改变其他设置。当你仔细查看你的样式表时，你会发现所有的标题都共享一个字体，只是字体大小有所不同而已：</p><figure class="image ss-img-wrapper"><img 会从上到下扫描你所有的样式级。只要有一个样式级与你的内容相匹配，它将会被自动选定为你的样式。例如，对于二级标题的文本可能会与 <a href="https://ulysses.app/styles/reference/paragraph" target="_blank"><code><strong>heading-all</strong></code></a> 和 <a href="https://ulysses.app/styles/reference/paragraph"><code><strong>heading-all</strong></code></a><code> </code>的<a 的 <a href="https://ulysses.app/styles/reference/font-size"><code><strong>font-size</strong></code></a> 相匹配。如果两个样式包含相同的设置，将自动使用后一个样式的设置。</p><h4 style="margin-left:0px;"> </h4><h4 src="https://cdn.sspai.com/2021/11/04/0facee69b00eac3e85f2fe23ac6368d8.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" 在一种情况下，这个文本是标题的一部分，而在另一种情况下，它属于一个普通段落。在你的导出结果中，你可以看到，无论段落类型如何，强标记文本均为粗体字。</p><figure class="image ss-img-wrapper"><img target="_blank"><code><strong>inline-strong</strong></code></a>）和外部标记的样式（如<code> </code><a href="https://ulysses.app/styles/reference/paragraph" style="margin-left:0px;">然而，在某些情况下，你可能想根据文本的位置来确定其样式。让我们假设你不想在标题内导出任何粗体字。如何防止这两种样式被混合在一起呢？只需一个<i>条件选择器</i>。只需在你的<a href="https://ulysses.app/styles/reference/inline"><code><strong>inline-strong</strong></code></a> 级别之后添加另一个级别。</p><figure class="image ss-img-wrapper"><img  出现<i>之后</i>（在样式表内），所以标题的样式级将覆盖掉它。</p><p>当然，对于条件样式也有其他情况。例如，你想让一个段落的第一行缩进，这取决于它是否是标题后的第一段。在这种情况下，你只需通过 "+" 关系来创建一个条件样式：</p><figure class="image ss-img-wrapper"><img 。第二个样式级为<strong>紧跟标题的所有段落</strong>创建了无需缩进的设置。</p><p>还有更多设定你内容的可能性；你可以在<a href="https://ulysses.app/styles/reference/definingstyles" target="_blank">语法概述</a>中，以浏览关于样式选择器更完整的概述。</p><h4 style="margin-left:0px;"><strong>如何继续？</strong></h4><p>到目前为止，你已经改变了一些标题的样式。在阅读样式表时，你会发现还有许多样式级。其中许多与你的 Ulysses 标记的定义名称相似：例如，<a href="https://ulysses.app/styles/reference/inline"><code><strong>inline-emphasis</strong></code></a>样式强调文本，<a href="https://ulysses.app/styles/reference/documentclasses"><code><strong>document-settings</strong></code></a>级包含了适用于整个导出文档的细节，如论文格式或脚注列举的方式。</p><p>我们已经整理了几个常见的样式调整样式的方法。这主要还是要根据你想要实现的效果，自己多多练手试一试，就会让你在之后的使用中更加游刃有余。</p><p>要了解更多关于所有现有的风格级和它们的样式设置，你可以查阅我们的综合<a href="https://ulysses.app/styles/reference" target="_blank">参考目录</a>。如果想深入了解 ULSS 的一些高级功能，我们推荐你看一下我们为大家准备的<a href="https://ulysses.app/styles/reference/definingstyles" target="_blank">语法概述</a>。</p><h2 style="margin-left:0px;"><strong>适用于已熟悉 CSS 用户的 Ulysses 样式表</strong><br> </h2><p style="margin-left:0px;">若你很熟悉CSS， ULSS 样式表对你而言将会非常简单。 只需通过双击复制其中一个内置样式，然后用你喜欢的文本编辑器进行编辑。我们为<a class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/11/02/article/6afbed33c6a8eec7f7b6bcd895ce8eb2?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" 中一些不太相关的功能。这一章将为你简要介绍两者主要的区别和相似之处。要全面了解所有可用的样式设置，请看<a href="https://ulysses.app/styles/reference/quickreference">完整的参考资料。​</a></p><p style="margin-left:0px;"> </p><h4 style="margin-left:0px;">通用选择器</h4><p style="margin-left:0px;">除了特定标记的选择器（如 <code>inline-strong</code>），ULSS 还支持通用选择器。通用选择器就是匹配整个类别的标记。例如，你可以使用 <code>heading-all </code>级为所有的标题创建一个样式。目前，有以下的通用级可供使用：</p><ul><li><a href="https://ulysses.app/styles/reference/paragraph"><code><strong>defaults</strong></code></a>用作所有文本的基础样式。</li><li><a href="https://ulysses.app/styles/reference/paragraph"><code><strong>heading-all</strong></code></a>设定任何标题的样式</li><li><a href="https://ulysses.app/styles/reference/list"><code><strong>list-all</strong></code></a>设定任何文本列表的样式</li><li><a href="https://ulysses.app/styles/reference/block"><code><strong>block-all</strong></code></a>设定任何代码块的样式 (代码块引用 等)</li></ul><h4 style="margin-left:0px;"><strong>相对选择器</strong></h4><p style="margin-left:0px;">与 CSS 类似，样式选择器不仅仅可以选择单个标记，还可以通过一个条件来关联多个样式。例如：选择器 <code>heading-all + paragraph</code>规定，只有当段落在标题之后时，才应该应用某个段落级。 ULSS 支持以下 CSS 的关系：</p><ul><li><code>parent child</code></li><li><code>parent > immediate-child</code></li><li><code>predecessor + successor</code></li><li>一些 <a href="https://ulysses.app/styles/reference/definingstyles#pseudoclasses" target="_blank"><code><strong>伪级别</strong></code></a> 比如<code>:first</code> 或 <code>:last</code></li></ul><h4 style="margin-left:0px;">样式继承</h4><p style="margin-left:0px;">如同 CSS，Ulysses 样式表是层叠的：内部标记的样式覆盖外部标记的样式。</p><p style="margin-left:0px;">与 CSS不同的是，对于一个样式级的特殊性并无等级之分。如果多个样式级可以应用到一个标记（例如 <code>heading-all</code> 和 <code>heading-1</code>)，样式在样式表中的顺序是优先的。样式表中之后的设定将覆盖之前的设定。</p><h4 style="margin-left:0px;"><strong>混入(Mixins)</strong></h4><p style="margin-left:0px;">如果你想在多个样式级之间共享样式设置，你可以使用样式混入。<a href="https://ulysses.app/styles/reference/definingstyles#mixins" style="margin-left:0px;"><strong>变量和表达式</strong></h4><p style="margin-left:0px;">为了提高样式的可读性，且更易创建变体，你可以使用<a href="https://ulysses.app/styles/reference/usingstylesettings#variables">变量</a> </i>应用程序中获得所有字体系列及字体样式的概览。在字体列表中，只要展开字体系列条目，你就会得到所有可用的字体样式列表。</p><p>设置标题的<a href="https://ulysses.app/styles/reference/text-alignment"
