<div class="articleWidth-content" data-v-e1f0100a><div class="content wangEditor-txt minHeight" data-v-e1f0100a><p style="margin-left:0px;">Typora 是我最爱的本地 Markdown 编辑器，几乎所有的文字输出都是在上面进行。专注模式、实时预览、图床等等功能，很大程度上提高了自己的效率。我曾经写过一些关于它的文章：</p><p style="margin-left:0px;"><a href="https://mp.weixin.qq.com/s/xQPQDt62_QwWRibdSc7DTA">Typora 使用小技巧</a></p><p style="margin-left:0px;"><a href="https://mp.weixin.qq.com/s/98jYHOsKGkY0JNDficGrmw">Typora 画图小技巧</a></p><p style="margin-left:0px;"><a href="https://mp.weixin.qq.com/s/aHxLbNdpULGSgo1jeJ4K7w">公众号写作排版指南</a></p><p style="margin-left:0px;">昨天，Typora 迎来了一次较大的版本更新，重新设计了 icon 图标以及一些 UI，并在导出、表格、流程图绘制等等功能上，有了不少的改进。</p><h3 style="margin-left:0px;"><strong>UI 更新</strong></h3><p style="margin-left:0px;">更换了图标，改成了下面这种更接近于 macOS Big Sur UI 的设计风格，大量运用圆角，看起来更加的活泼了。</p><figure class="image ss-img-wrapper image_resized" style="width:800px;"><img src="https://cdn.sspai.com/2021/05/13/article/f02b6c10371b86ed3b75fe29fd88c430?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="Artboard" data-original="https://cdn.sspai.com/2021/05/13/article/f02b6c10371b86ed3b75fe29fd88c430" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">Windows 版本更新了右键菜单，现在菜单上可以有更多的快捷方式。</p><figure class="image ss-img-wrapper image_resized" style="width:800px;"><img src="https://cdn.sspai.com/2021/05/13/article/32c8eec18d1aa374e9e1c284920b5445?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="CleanShot 2021-04-19 at 00.18.11@2x" data-original="https://cdn.sspai.com/2021/05/13/article/32c8eec18d1aa374e9e1c284920b5445" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">更新了代码选择的自动补全菜单，新增了语言图标。</p><figure class="image ss-img-wrapper image_resized" style="width:800px;"><img src="https://cdn.sspai.com/2021/05/13/article/7262ff0a779f913494b14a76539bad10?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="CleanShot 2021-04-19 at 00.17.32@2x" data-original="https://cdn.sspai.com/2021/05/13/article/7262ff0a779f913494b14a76539bad10" referrerpolicy="no-referrer"></figure><h3 style="margin-left:0px;"><strong>图表</strong></h3><p style="margin-left:0px;">表格支持使用 Tab 键来增加新的一行</p><figure class="image ss-img-wrapper image_resized" style="width:800px;"><img src="https://cdn.sspai.com/2021/05/13/article/3e573e83fa8b8d780ca264e44dd37bd5?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="CleanShot 2021-04-05 at 23.43.07" data-original="https://cdn.sspai.com/2021/05/13/article/3e573e83fa8b8d780ca264e44dd37bd5" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">图表现在可以使用 CSS 变量，来配置样式。你可以修改主题文件夹中的 <code>base.css</code> 中的配置。如果找不到个文件，可以自己新建一个 <code>base.user.css</code>，使用下面的配置进行覆盖。</p><pre class="language-css"><code>:root &#123;
  --mermaid-theme: default;
  --mermaid-sequence-numbers: off;
  --mermaid-flowchart-curve: linear;
  --mermaid--gantt-left-padding: 75;
  --sequence-theme: simple;
&#125;</code></pre><p style="margin-left:0px;">下图中左边是默认样式，右边是新支持的样式。</p><figure class="image ss-img-wrapper image_resized" style="width:800px;"><img src="https://cdn.sspai.com/2021/05/13/article/1c9c2597bd1d07e6cdbbeae322713926?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="image-20210513160345865" data-original="https://cdn.sspai.com/2021/05/13/article/1c9c2597bd1d07e6cdbbeae322713926" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">需要注意的是，<code>–sequence-theme</code> 变量是作用在 <code>sequence</code> 语法，而不是 <code>mermaid</code> 语法。</p><h3 style="margin-left:0px;"><strong>文档导出</strong></h3><p style="margin-left:0px;">单独新增了导出到配置菜单，可以到设置中找到，针对不同的文件格式有不同的配置选项，并可以设置默认的导出文件夹。</p><figure class="image ss-img-wrapper image_resized" style="width:800px;"><img src="https://cdn.sspai.com/2021/05/13/article/f6f02d6ca406078324fd61aff50c521e?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="Screen Shot 2021-01-18 at 22.51.30" data-original="https://cdn.sspai.com/2021/05/13/article/f6f02d6ca406078324fd61aff50c521e" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">PDF 导出支持对页面大小、边距、页头、页脚等进行配置。</p><figure class="image ss-img-wrapper image_resized" style="width:800px;"><img src="https://cdn.sspai.com/2021/05/13/article/7cf764f4fd4f2e32c1aae36af69b5441?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="Screen Shot 2021-01-20 at 00.07.01" data-original="https://cdn.sspai.com/2021/05/13/article/7cf764f4fd4f2e32c1aae36af69b5441" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">图片的导出也支持设置导出大小，并且新增了一些自定义的命令导出。</p><h3 style="margin-left:0px;"><strong>其他</strong></h3><ul><li>Big Sur 中重新启用了经典模式</li><li>Windwos 安装程序现在支持用户级别的安装</li><li>添加了希伯来语翻译</li><li>一些 Bug 的修复</li></ul><p style="margin-left:0px;">Typora 作为一款免费的软件，能做到如此用心，并长期坚持更新维护，真心非常不错。</p><p style="margin-left:0px;">读者可以关注 Typora 的 <a href="https://twitter.com/typora">Twitter</a> 或者订阅 <a href="https://typora.io/#subscribe">Newsletter</a> 来关注最新的更新消息。</p></div><!----></div><div style="border:1px solid transparent;" data-v-e1f0100a></div><div class="article-side sideTop"
