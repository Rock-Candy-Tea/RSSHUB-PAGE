<div class="articleWidth-content" data-v-e1f0100a><div class="content wangEditor-txt minHeight" data-v-e1f0100a><h2>前言</h2><p>上次发布还是半年多以前的事情了，那时候也没信心移植 MarkText 的笔记编辑器，所以笔记编辑的体验着实一般，后来突然有了一大段的时间可以用来写代码，正好拿来改进改进，发个新版本。</p><p>先上 GitHub 地址： <a href="https://github.com/TankNee/Memocast" target="_blank">https://github.com/TankNee/Memocast</a></p><p>前后花了大概二十天，完善了革新了绝大多数的功能，以前的版本可以参照我写的这篇文章：</p><span class="ss-linkCard" link-card-href="https://sspai.com/post/63020"> </span><h2>更新内容</h2><p>这次的 2.0 版本有如下的大更新！</p><ol><li>全新的 Logo 。</li><li>从 Neeto-Vue 更名为 Memocast，意为 Cast Your Memo ，将你的思想碎片汇集成笔记，一如思考一般轻松简单。</li><li>全新的所见即所得编辑器，来自开源社区的优秀产品 MarkText，向他的开发者致以敬意。</li><li>全新的源码编辑器， Monaco，也就是vscode使用的编辑器，非常好用，应该是当下最好的源码模式了。</li><li>使用 GitHub 搭建了更新服务器，Windows 下自动更新测试通过，如果你是 Windows 系统，那么就可以方便的将软件更新到最新版本。</li><li>为知笔记图片服务的支持，现在可以不需要考虑什么图床不图床的了，直接使用内置的图片服务，也不需要担心图片保存的问题，同样支持为知笔记 Docker 部署的私有服务。</li><li>大量快捷键的引入，国际化多语言增强等。</li></ol><h2>操作动图</h2><h3>暗色模式与国际化</h3><figure class="image
