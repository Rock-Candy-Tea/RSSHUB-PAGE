<div class="articleWidth-content" data-v-6a669db8><div class="content wangEditor-txt minHeight" data-v-6a669db8><p style="margin-left:0;">记录一下 MIUI 12.5 海外版刷机过程。</p><h2><strong>写在前面</strong></h2><p style="margin-left:0;">我手里这台 K30 Pro 是 5G 版本，去年年中买回来后，刷了海外版的 MIUI 作为<strong>备机使用</strong> ，整体体验还是不错的。</p><p style="margin-left:0;">之所以选择海外版，首先是因为广告确实少了不少、也没有了茫茫多的内置应用，其次因为 GPDR 的原因，某些地区的固件版本迫于合规要求，对于数据采集也更加的收敛。</p><p style="margin-left:0;">加上在主力机苹果信号不好的时候，一台安卓备机能“挺身而出”承担提供网络热点的职责。平时多数时候，备机还能作为导航专用设备使用，不用担心导航过程中突然来电话，界面和外放提醒受到干扰，对我个人来说，还是挺香的。</p><p style="margin-left:0;">在刷 MIUI 11 和 MIUI 12 海外版时，参考两年前写的一篇记录（<a href="https://soulteary.com/2019/09/08/xiaomi-k20pro-experience-android-10.html"><strong>《小米 K20Pro 体验 Android 10》</strong></a>）进行操作，并不会出现什么意料之外的情况 。然而在进行 12.5 版本的刷机过程中，发现原来的方法不奏效了，这或许和我之前使用了 TWRP 作为引导工具有关系，也有可能是和“官方并未直接发布 12.5 刷机固件有关”。此外，在刷机过程中，也遇到了一些之前没有遇到的情况。</p><p style="margin-left:0;">故记录下来，希望能帮到有相同需求的同学。</p><h2><strong>第一关：准备刷机资料</strong></h2><p style="margin-left:0;">打开 <a href="https://en.miui.com/"><strong>MIUI 英文页面</strong></a>，我们会发现相比较中文页面，导航栏多了一个 “Download” 按钮，访问按钮，我们会被重定向到一个新的页面：<a href="https://c.mi.com/global/miuidownload/index"><strong>https://c.mi.com/global/miuidownload/index</strong></a>。</p><figure class="image ss-img-wrapper image_resized" style="width:100%;"><img src="https://cdn.sspai.com/2021/10/22/article/6a37609170c424ec3f3029f808fcfd4a?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="先在海外站点选择合适的机型" data-original="https://cdn.sspai.com/2021/10/22/article/6a37609170c424ec3f3029f808fcfd4a" referrerpolicy="no-referrer"><figcaption>先在海外站点选择合适的机型</figcaption></figure><p> </p><p style="margin-left:0;">在页面中选择你的设备型号，有一些设备在海外发行的时候会改名字，比如 K30 Pro 在海外的发行名称是“POCO F2 Pro”。当选择完毕设备型号后，我们会来到另外一个新页面：<a href="https://c.mi.com//miuidownload/detail?device=1900380"><strong>https://c.mi.com//miuidownload/detail?device=1900380</strong></a>。</p><figure class="image ss-img-wrapper image_resized" style="width:100%;"><img src="https://cdn.sspai.com/2021/10/22/article/1f65f49ce5d5d3cea8594634408c71c2?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="打开匹配型号的页面" data-original="https://cdn.sspai.com/2021/10/22/article/1f65f49ce5d5d3cea8594634408c71c2" referrerpolicy="no-referrer"><figcaption>打开匹配型号的页面</figcaption></figure><p> </p><p style="margin-left:0;">在页面左侧选择你想要刷入的某个发行版固件，我这里选择的是“POCO F2 Pro EEA”版本。然而，如果你直接使用 “Download Full Rom” 固件下载按钮进行了固件下载，那么大概率在使用 TWRP 刷机时，会得到固件包有问题的错误提示，而导致不能继续进行刷机操作。哪怕这个压缩包可以被测试正确解压缩...</p><figure class="image ss-img-wrapper image_resized" style="width:100%;"><img src="https://cdn.sspai.com/2021/10/22/article/97d2bfd1090b88177409f3325c82b789?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="使用 TWRP 刷机会出现问题" data-original="https://cdn.sspai.com/2021/10/22/article/97d2bfd1090b88177409f3325c82b789" referrerpolicy="no-referrer"><figcaption>使用 TWRP 刷机会出现问题</figcaption></figure><p> </p><p style="margin-left:0;">我们不去纠结为什么会产生这个原因，避开这个问题的解决方案是，点击“Flashing Guide”按钮，进入 <a href="https://c.mi.com/oc/miuidownload/detail?guide=1"><strong>https://c.mi.com/oc/miuidownload/detail?guide=1</strong></a> 教程页面，选择侧边栏左侧的 “Fastboot Update” 分类，然后选择我们需要的固件，我这里选择的是 “POCO F2 Pro Latest Europe Stable Version Fastboot File Download” 版本，最终下载完毕，会得到一个名称类似 “lmi_eea_global_images_V12.5.2.0.RJKEUXM_20210811.0000.00_11.0_eea_413f2f329c.tgz” 的文件。</p><h2><strong>第二关：刷入设备</strong></h2><p style="margin-left:0;">准备一台 Windows 设备，将刚刚准备好的固件传到设备中解压缩。接着下载 <a href="https://cdn.alsgp0.fds.api.mi-img.com/micomm/MiFlash2020-3-14-0.rar"><strong>MiFlash 刷机软件</strong></a>。</p><figure class="image ss-img-wrapper image_resized" style="width:100%;"><img src="https://cdn.sspai.com/2021/10/22/article/59cea51d4a54a5d7e84984f5d290a839?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="刷机软件首次打开需要先安装驱动" data-original="https://cdn.sspai.com/2021/10/22/article/59cea51d4a54a5d7e84984f5d290a839" referrerpolicy="no-referrer"><figcaption>刷机软件首次打开需要先安装驱动</figcaption></figure><p> </p><p style="margin-left:0;">为了简化操作，建议在数据备份之后，可以将手机密码等要素都先移除，并删除手机上关联的账号，比如 Google账号和小米账号。</p><figure class="image ss-img-wrapper image_resized" style="width:100%;"><img src="https://cdn.sspai.com/2021/10/22/article/95ca5e1ee36ddce4bb590786c8a1b10c?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="使用 TWRP 刷机会出现问题" data-original="https://cdn.sspai.com/2021/10/22/article/95ca5e1ee36ddce4bb590786c8a1b10c" referrerpolicy="no-referrer"><figcaption>使用 TWRP 刷机会出现问题</figcaption></figure><p> </p><p style="margin-left:0;">将手机关机，长按音量键和电源键，进入 FastBoot 模式，然后将手机连接电脑。先点击软件的加载设备按钮，当软件设备列表中展示出了我们的设备之后，点击选择按钮，加载刚刚解压缩好的文件目录，最后点击刷机，耐心等待设备刷入新的固件。</p><figure class="image ss-img-wrapper image_resized" style="width:100%;"><img src="https://cdn.sspai.com/2021/10/22/article/db10378e6dec98fddff0888b6542ec40?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="刷机过程比较漫长，需要几分钟" data-original="https://cdn.sspai.com/2021/10/22/article/db10378e6dec98fddff0888b6542ec40" referrerpolicy="no-referrer"><figcaption>刷机过程比较漫长，需要几分钟</figcaption></figure><p> </p><p style="margin-left:0;">以往刷入系统之后，我们只需要等待机器重启即可，最多进行一次双清操作即可，然而在刷入系统之后，会发现根本无法进入系统。在引导界面中会提醒我们 <strong>“This MIUI version can't be installed on this device”</strong> 。</p><h2><strong>第三关：解除引导限制</strong></h2><p style="margin-left:0;">这个时候，我们和传统刷机一样，再次下载 MiUnlock 软件。先需要登陆你的小米账号，验明你有解锁资格。</p><p style="margin-left:0;">不过即使你的账号拥有解锁资格，第一次解锁也可能会遇到失败，不过没有关系，我们可以多试几次。解锁成功之后，重启手机，“一路 Next” 就能进入系统了。</p><figure class="image ss-img-wrapper image_resized" style="width:100%;"><img src="https://cdn.sspai.com/2021/10/22/article/dd83974c375149fbbd0953dc6d1d282f?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="成功解锁的界面" data-original="https://cdn.sspai.com/2021/10/22/article/dd83974c375149fbbd0953dc6d1d282f" referrerpolicy="no-referrer"><figcaption>成功解锁的界面</figcaption></figure><p> </p><p style="margin-left:0;">不得不说这个版本的开机欢迎界面还是比较好看的，不过默认屏幕并没有打开 DC 调光，降低闪烁的功能，所以拍出来的照片会有比较多抖动“干扰”。</p><figure class="image ss-img-wrapper image_resized" style="width:100%;"><img src="https://cdn.sspai.com/2021/10/22/article/5d74b459a3b8f40e04f8a2d5ac3a4805?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="MIUI 12.5 欢迎界面" data-original="https://cdn.sspai.com/2021/10/22/article/5d74b459a3b8f40e04f8a2d5ac3a4805" referrerpolicy="no-referrer"><figcaption>MIUI 12.5 欢迎界面</figcaption></figure><p> </p><p style="margin-left:0;">经过层层设置，最终我们进入到的系统界面是这样的，可以看到谷歌全家桶都在，综合来说，还是比较省心的。</p><figure class="image ss-img-wrapper image_resized" style="width:100%;"><img src="https://cdn.sspai.com/2021/10/22/article/0768b60717c6b547dac4d1bb9ac3e74a?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="顺利进入系统界面" data-original="https://cdn.sspai.com/2021/10/22/article/0768b60717c6b547dac4d1bb9ac3e74a" referrerpolicy="no-referrer"><figcaption>顺利进入系统界面</figcaption></figure><p> </p><h2><strong>其他</strong></h2><p style="margin-left:0;">相比较 Pixel Experience 的固件，MIUI 12.5 固件大小足足大了 3～4 个G，所以如果追求绝对轻量的体验，或许也可以试试 <a href="https://download.pixelexperience.org/picasso"><strong>Pixel Experience</strong></a>。</p><h2><strong>最后</strong></h2><p style="margin-left:0;">我最早的一台小米手机是一台比较有棱角的 MI3，后续虽然主力设备都是苹果，但是备机的选项里，一直会为小米留一个名额。</p><p style="margin-left:0;">或许是因为需要使用热点；或许是作为开发需要有一台安卓测试机；也可能是在使用苹果作为主力设备，缺少 SIM 卡槽得多带一台机器的尴尬时期；再或者我想要有一个便携的阅读器；想要有一台没有前置挖空屏幕的设备等等。</p><p style="margin-left:0;">但是，这些设备几乎在第一个伴随机器的 MIUI（Android）大版本之后，就都没有再收到过新版本的 Android 适配，除非使用非官方适配的版本，不然很快就变成了电子垃圾（开始卡顿、无法体验新特性、残值迅速降低），除非使用者小心翼翼的进行呵护（<strong>当然，这也和之前的安卓市场生态有关，不完全是小米的问题</strong>）。</p><p style="margin-left:0;">相比较隔壁G家自己同时代、甚至早些时候发布的机器可以被一直升级支持，作为小米用户，还真是蛮羡慕的。出于早期的 MIUI 确实做了比较多体验上的优化，使用起来蛮方便的，鱼和熊掌不可兼得嘛，咱也就接受了。</p><p style="margin-left:0;">然而，在最近几年里，不论是软件、还是硬件都频频被网友发现质量出现下滑，对于国内的支持者来说，心里总归不是那么舒服。之前还有网友整理了长达几个小时的 BUG 视频，不时就会出来几篇“爱之深、责之切”的内容，或许小米真的到了该重视产品研发的时刻了，因为再这么下去，可能发烧友们甚至会连“精神股东”都不愿再当下去了。</p><p
