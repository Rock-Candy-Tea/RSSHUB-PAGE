<div class="articleWidth-content" data-v-6a669db8><div class="content wangEditor-txt minHeight" data-v-6a669db8><p>项目仓库：<a href="https://github.com/Benbinbin/TagDown" target="_blank">https://github.com/Benbinbin/TagDown</a></p><h2><strong>简介</strong></h2><p style="margin-left:0px;"><a href="https://github.com/Benbinbin/TagDown" target="_blank">TagDown</a> 是一款开源的书签管理插件， 您可以使用扩展程序<i>浏览</i>、<i>新增</i>、<i>修改</i>书签，它也支持以不同方式<i>导出</i>书签。</p><iframe class="ss-videoIframe" src="//player.bilibili.com/player.html?bvid=BV1up4y1s7VP"> </iframe><p> </p><p style="margin-left:0px;">🔗 <a href="https://github.com/Benbinbin/TagDown/blob/main/README.md">Introduction English Version</a></p><h2><strong>特点</strong></h2><ul><li>支持 🎬 <a href="https://www.bilibili.com/video/BV1up4y1s7VP?p=4">新增</a>书签，并附加额外的信息，例如 <code>tags</code>、<code>groups</code> 等</li><li>支持 🎬 <a href="https://www.bilibili.com/video/BV1up4y1s7VP?p=7">导出</a>任意书签为 <code>json</code> 文档</li><li>以 🎬 <a href="https://www.bilibili.com/video/BV1up4y1s7VP?p=3">树图</a>的形式浏览层级结构的书签数据</li><li>一键打开多个书签，支持在 🎬 <a href="https://www.bilibili.com/video/BV1up4y1s7VP">标签组</a>内打开书签</li></ul><p style="margin-left:0px;">数据保存在本地，额外附加的信息是「非破坏性」的，即卸载扩展程序后，不会影响浏览器的书签数据</p><h2><strong>安装</strong></h2><p style="margin-left:0px;">1.下载最新版本的 <a href="https://github.com/Benbinbin/TagDown/releases">TagDown</a> <code>zip</code> 文件并解压</p><p style="margin-left:0px;">2.根据官方<a href="https://developer.chrome.com/docs/extensions/mv2/getstarted/#:~:text=added%20as%20an%20extension%20in%20developer%20mode%20in%20its%20current%20state">指南</a>「加载已解压的扩展程序」：</p><ul><li>打开 Chrome 浏览器并访问 <code>chrome://extensions</code> 网址</li><li>在页面的右上角激活 <code>开发者模式</code></li><li>点击左上角 <code>加载已解压的扩展程序</code> 选择解压后的文件夹以安装扩展程序</li></ul><p style="margin-left:0px;">3.然后您可以使用此扩展程序和 Chrome 作为您的书签管理器。请尝试使用！</p><p style="margin-left:0px;">💡 🎬 <a href="https://www.bilibili.com/video/BV1up4y1s7VP">此系列视频</a>是关于扩展程序的使用说明</p><h2><strong>数据</strong></h2><p style="margin-left:0px;">TagDown 使用 Chrome 提供的 APIs 与书签和标签进行交互，它不依赖于任何后台服务器，所以<strong>所有数据都存储在本地浏览器中</strong>。</p><p style="margin-left:0px;">💡 如果您的 Chrome 浏览器登录账号，并<a href="https://support.google.com/chrome/answer/185277">打开同步选项</a>，书签将由 Google 进行同步，这样您可以使用同一账户在不同设备之间共享书签（如果您使用微软推出的 Edge 浏览器，它也是基于 Chromium 内核进行开发的，也可以安装该插件，书签将由微软进行<a href="https://support.microsoft.com/en-us/microsoft-edge/sign-in-to-sync-microsoft-edge-across-devices-e6ffa79b-ed52-aa32-47e2-5d5597fe4674">同步</a>）。</p><p style="margin-left:0px;">除了通过调用 Chrome 的 APIs 来使用书签外，TagDown 还将一些数据保存在 <a href="https://developer.mozilla.org/zh-TW/docs/Web/API/IndexedDB_API"><strong>IndexedDB</strong></a> 中，这是一个由浏览器提供的数据库，用于持久性地存储数据。当您 🎬 <a href="https://www.bilibili.com/video/BV1up4y1s7VP?p=4"><strong>通过 TagDown 添加新书签</strong></a>，可以为新书签提供额外的信息，如 <code>tags</code>、<code>groups</code> 和 <code>description</code> 等，这些数据将保存在 IndexedDB 中，它们<strong>不会</strong>被 Google 同步，它总是保存在本地。</p><p style="margin-left:0px;">TagDown 提供了 🎬 <a href="https://www.bilibili.com/video/BV1up4y1s7VP?p=10"><code>export</code> 和 <code>import</code> 功能</a>，您可以将整个 IndexedDB 数据库导出为 <code>json</code> 文件并保存在您指定的地方作为备份，并通过导入该文件恢复数据库。</p><p style="margin-left:0px;">💡 Chrome 会为每个书签分配一个唯一的 ID，TagDown 使用这个唯一的 ID 作为 IndexedDB 的索引，将数据库中不同部分的数据与不同的书签相匹配。</p><p style="margin-left:0px;">⚠️ 如果您想在 IndexedDB 为书签创建一条相应记录，请在新增书签时优先使用 TagDown 插件。话虽如此，如果在安装 TagDown 之前，浏览器中已经存在很多书签，或者您顺手<a href="https://support.google.com/chrome/answer/188842">用 Chrome 的默认工具创建书签</a>也是没问题，因为之后您还可以通过修改书签的方式，手动添加数据到 IndexedDB 数据库中。</p><h2><strong>常见疑问解答</strong></h2><h3><strong>TagDown 为什么需要使用 IndexedDB 数据库？</strong></h3><p style="margin-left:0px;">虽然书签都保存在浏览器中，但它们只包含 <code>id</code>、<code>title</code>、<code>url</code> 和文件夹信息，TagDown 希望给书签附加更多的信息，如 <code>tags</code>、<code>description</code> 等，因此使用一个内置在浏览器中的本地数据库来保存这些数据。</p><h3><strong>如果卸载/移除扩展程序会丢失哪一部分数据？</strong></h3><p style="margin-left:0px;">书签保存在浏览器中并由 Google 同步（如果您使用 Chrome 并打开同步选项），<strong>因此如果您卸载扩展程序，请放心，所有书签都不会受影响</strong>。</p><p style="margin-left:0px;">但是 TagDown 创建并保存在 <strong>IndexedDB 中的相关数据将会丢失</strong>。 您可以在卸载扩展前 🎬 <a href="https://www.bilibili.com/video/BV1up4y1s7VP?p=10"><strong>导出</strong></a> 这些数据，获得的 <code>json</code> 文件可以作为备份数据库，下次重新安装扩展时可以导入以恢复 IndexedDB 相关数据。</p><p style="margin-left:0px;">顺便说一句，JSON 格式的文件是很常见的，您可以使用其他文本编辑器打开它，例如 <a href="https://code.visualstudio.com/">Visual Studio Code</a>，直接浏览文档内容。</p><h3><strong>如果重新安装扩展程序会丢失哪一部分数据？</strong></h3><p style="margin-left:0px;">当安装解压后的扩展时，Chrome 会给每个扩展分配一个唯一的 ID，而 IndexedDB 遵循<a href="https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API/Basic_Terminology">同源策略</a>限制，如果Chrome 为 TagDown 分配了另一个 ID，那么在重新安装扩展后，**可能会导致 IndexedDB 数据丢失。</p><p style="margin-left:0px;">您可以在卸载扩展之前<strong>导出</strong>这些数据，这个 <code>json</code> 文件可以作为一个备份数据库，您可以在下次重新安装扩展时导入它。</p><h3><strong>如何升级 TagDown？</strong></h3><p style="margin-left:0px;">如果从 TagDown v1.0.0 升级到 v2.0.0 可以直接卸载原有旧版本，安装新版本。但是从 v2.0.0 版本开始 TagDown 使用数据库，而重新安装扩展程序可能会导致 IndexedDB 数据丢失，所以之后的升级，您只需要直接<strong>用新版本的文件夹覆盖替换</strong>原来的扩展的文件夹。</p><h2><strong>在浏览器中保存的书签数据与 IndexedDB 中保存的数据的区别</strong></h2><p style="margin-left:0px;">IndexedDB 数据是浏览器书签数据的副本，它在书签上都附加有其他信息。</p><p style="margin-left:0px;"><strong>但事实上，它并</strong><i><strong>不是</strong></i><strong>一个完整拷贝</strong>。</p><p style="margin-left:0px;"><strong>只有通过 TagDown 创建或修改过的书签，才会在 IndexedDB 中有相应的记录</strong>。因此，导出的 Indexed 数据库所获得的 JSON 文件可能并不包含所有书签。</p><p style="margin-left:0px;">如果您想得到所有的书签数据，请使用浏览器提供的默认导出功能，您将会得到一个 HTML 文件：</p><p style="margin-left:0px;">进入书签管理器：<code>chrome://bookmarks/</code></p><p style="margin-left:0px;">点击页面右上角的汉堡菜单（三个小点）</p><p style="margin-left:0px;">从下拉菜单中选择 <code>导出书签</code></p><h2><strong>如何为所有标签页批量创建书签？</strong></h2><p style="margin-left:0px;">Chrome 提供了「为所有标签页添加书签」的功能，以批量创建书签，您可以使用快捷键 <code>Ctrl + Shift + D</code> 来触发该功能。</p><p style="margin-left:0px;">强烈推荐使用另一个扩展程序 <a href="https://www.one-tab.com/">OneTab</a>，它关闭所有标签，并返回一个带有相应链接列表的页面，也可以将该页面开启在线功能，共享给其他人查看。</p><h2><strong>版本</strong></h2><ul><li><a href="https://github.com/Benbinbin/TagDown/releases/tag/v2.0.0">v2.0.0</a> 提供两种模式，<strong>树图模式</strong>和<strong>网格模式</strong>，以<strong>浏览器和可视化</strong>层次化的书签数据，您可以在 2.0.0 版本中<strong>添加</strong>新书签和<strong>编辑</strong>现有书签，并添加 <code>tags</code> 和 <code>groups</code> 等额外信息。 通过附加到书签的 tag 和 group，您可以<strong>以多种方式导出书签</strong>。 所有数据都将保存在本地，您可以随时<strong>导出整个数据库或导入备份以恢复它</strong>，因此请尝试使用该扩展程序和 chrome 作为您的书签管理器。</li><li><a href="https://github.com/Benbinbin/BookDown/releases/tag/v1.0.0">v1.0.0</a> 提供两种模式，<strong>浏览和可视化</strong>层次化的书签数据。</li></ul></div><!----></div><div style="border:1px solid transparent;" data-v-6a669db8></div><div class="article-side sideTop" style="display:none;left:0;"
