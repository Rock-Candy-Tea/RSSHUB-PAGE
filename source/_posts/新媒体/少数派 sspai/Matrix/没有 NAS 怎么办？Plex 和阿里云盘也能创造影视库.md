<div class="articleWidth-content" data-v-0b37afcb><div class="update-wrap" data-v-0b37afcb></div><div class="content wangEditor-txt minHeight" data-v-0b37afcb><h2>起因</h2><p>周五一个晴朗的下午，正打着哈欠对着电脑屏幕无所事事，突然想看 Mark McMorris 的记录片《Unbroken》，由于在红牛官网看有点累，所以下载了下来。看完之后就按照习惯随手上传到了阿里云盘，看着阿里云盘9.5T的容量和保存丰富的影音库，突然想到阿里云盘支持 WebDAV，能挂载到本地，就想能不能试试把阿里云盘里面的电影、美剧和动漫接入我的本地影音管理工具Plex（Emby、Jellyfin 和 Infuse 同理，操作稍微有一点不同）。</p><h2>准备工作</h2><p>我尝试了两种把阿里云盘创建 WebDAV 的方式，分别是：</p><ol><li>利用 docker 创建阿里云盘的 WebDAV 服务</li><li>利用 RaiDrive 创建阿里云盘的 WebDAV 服务</li></ol><p>在这之前，我们需要打开浏览器来获取阿里云盘的 refreshToken</p><p>第一步，打开浏览器，打开 <a href="https://www.aliyundrive.com/drive/" target="_blank">阿里云盘</a> 官网并登录</p><p>第二部，按F12打开开发者工具，点击 Application（应用程序），在打开的菜单里点击 Local Storage（本地储存），点击 Local Storage 下的 https://www.aliyundrive.com/，然后在二级菜单里点击右边的 token，就可以看到下面的 refreshToken 了。</p><figure class="image ss-img-wrapper image_resized" style="width:441px;"><img src="https://cdn.sspai.com/editor/u_/c901dblb34tdg623e260.jpeg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/editor/u_/c901dblb34tdg623e260.jpeg" referrerpolicy="no-referrer"></figure><h2>利用 docker 创建阿里云盘的 WebDAV 服务</h2><p>在这里我利用的是 <a href="https://registry.hub.docker.com/r/mumiehub/rclone-mount" target="_blank">rclone-mount</a> 这个 docker 镜像来实现，但是我后来有找到了更好上手的方法，如下：</p><ol><li>利用 <a href="https://github.com/zxbu/webdav-aliyundriver" target="_blank">zxbu/webdav-aliyundriver</a> 来实现 WebDAV</li><li>下载 Jar 包运行</li><li>Docker run 运行</li><li>Docker compose</li></ol><h3>补充</h3><p>上面的折腾主要是因为我一个穷学生没有 NAS，如果你有群晖 NAS 的话，也可以利用 docker 部署到本地后，直接在群晖内挂载此 WebDAV 就可以在群晖的文件管理里随时访问。</p><ol><li>docker - 注册表 - 搜索 <code>webdav-aliyundriver</code></li><li>下载结果的第一个 <code>zx5253/webdav-aliyundriver</code></li><li>启动此镜像</li><li>在常规设置里勾选 <code>使用高级权限执行容器</code></li><li>在高级设置的装载路径填写 <code>/etc/aliyun-driver</code></li><li>容器端口设置默认为8080</li><li>高级设置 - 环境 - 添加以下变量</li><li><code>ALIYUNDRIVE_REFRESH_TOKEN</code> 为上文获取的 <code>refresh_token</code></li><li><code>ALIYUNDRIVE_AUTH_USER-NAME</code> 为 WebDAV 设置用户名</li><li><code>ALIYUNDRIVE_AUTH_PASSWORD</code> 为 WebDAV 设置密码</li><li>最后在群晖里直接挂载 WebDAV</li></ol><h2>利用 RaiDrive 创建阿里云盘的 WebDAV 服务</h2><p>如果你觉得用 docker 太难，<a href="https://www.raidrive.com/" target="_blank">RaiDrive</a> 绝对是更为简单的选择，这里我们则需要准备几个文件：</p><ol><li>首先下载 <a href="https://www.raidrive.com/" target="_blank">RaiDrive</a></li><li>下载 <a href="https://github.com/winsw/winsw/releases" target="_blank">Winsw</a> 的 exe 文件</li><li>下载 <a href="https://github.com/zxbu/webdav-aliyundriver" target="_blank">zxbu/webdav-aliyundriver</a> 里的 Jar 文件</li></ol><p>到这里，整个流程已经完成了一半，我们剩下要做的是创建一个 Winsw 的 xml 文件，内容如下：</p><pre class="language-htmlembedded"><code><service>
  
  <!-- ID of the service. It should be unique across the Windows system-->
  <id>webdav-aliyundriver</id>
  <!-- Display name of the service -->
  <name>webdav-aliyundriver</name>
  <!-- Service description -->
  <description>webdav-aliyundriver</description>
  
  <!-- Path to the executable, which should be started -->
  <executable>java</executable>

  <!-- 
       aliyundrive.refresh-token 填写TOKEN
       server.port 填写端口
       这里出于避免与其它服务冲突将其默认8080修改为8181，当然你也可以修改成自己的端口 
  -->
  <arguments>-jar webdav-aliyundriver-2.4.2.jar --aliyundrive.refresh-token="填写TOKEN"  --server.port="8181" --aliyundrive.auth.enable=true --aliyundrive.auth.user-name="admin" --aliyundrive.auth.password="admin"</arguments>
   <!--
    OPTION: onfailure
    Defines a sequence of actions, which should be performed if the managed executable fails.
    Supported actions: restart, reboot, none
  -->
  <onfailure action="restart" delay="10 sec"/>
  <onfailure action="restart" delay="20 sec"/>
  <!--
    OPTION: resetfailure
    Time, after which the Windows service resets the failure status.
    Default value: 1 day
  -->
  <resetfailure>1 hour</resetfailure>
  <!--
    OPTION: priority
    Desired process priority.
    Possible values: Normal, Idle, High, RealTime, BelowNormal, AboveNormal
    Default value: Normal
  -->
  <priority>Normal</priority>
  
  <!-- 
    OPTION: stoptimeout
    Time to wait for the service to gracefully shutdown the executable before we forcibly kill it
    Default value: 15 seconds
  -->
  <stoptimeout>15 sec</stoptimeout>
    
  <!--
    OPTION: stopparentprocessfirst
    If set, WinSW will terminate the parent process before stopping the children.
    Default value: false
  -->
  <stopparentprocessfirst>false</stopparentprocessfirst>
 
    <!--
      OPTION: startmode
      Defines start mode of the service.
      Supported modes: Automatic, Manual, Boot, System (latter ones are supported for driver services only)
      Default mode: Automatic
    -->
    <startmode>Automatic</startmode>
    <waithint>15 sec</waithint>
    <sleeptime>1 sec</sleeptime>

    <log mode="roll-by-size">
        <sizeThreshold>10240</sizeThreshold>
        <keepFiles>8</keepFiles>
    </log>
</service></code></pre><p>做完所有步骤后，我们可以看到文件夹里有三个文件：</p><ol><li>Winsw 的 exe 文件</li><li>Winsw 的 xml 文件</li><li>Webdav-aliyundrive 的 jar 文件</li></ol><p>此时，我们将第一个和第三个文件命名保持一致，然后在 xml 文件里修改 refreshToken（就是我们在前面得到的）。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/editor/u_/c901dbtb34tdhva438qg.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/editor/u_/c901dbtb34tdhva438qg.png" referrerpolicy="no-referrer"></figure><p>剩下就可以开始启动服务了：</p><ol><li>以管理员身份打开 cmd</li><li>使用 cd 命令进入进入下载的软件包解压后放置的位置，例如 <code>cd aliyundriver</code></li><li>执行命令 <code>webdav-aliyundriver-2.4.2.exe install</code></li><li>若提示安装成功，执行命令 <code>net start webdav-aliyundriver</code> ，如果这一步显示 1064 报错，安装 jdk1.8 可解决，版本过高可能会出现问题。</li><li>若提示请求的服务已经启动，打开 RaiDrive，点击 Add</li></ol><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/03/27/db7caae1419705c982ac1aa43c9cf3ac.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/03/27/db7caae1419705c982ac1aa43c9cf3ac.png" referrerpolicy="no-referrer"></figure><p>选择服务 WebDAV，盘符和名称自定义，端口8181，账号和密码都是 admin，记得去掉地址后面的那个选框。</p><p>配置完成以后，点击连接，就可以看到阿里云盘已经挂载到本地了。<sup class="ss-footnote" href="https://blog.csdn.net/weixin_42795002/article/details/121864236" title="注：以上方案根据 https://blog.csdn.net/weixin_42795002/article/details/121864236 实现，已获得作者同意转载。" footnote-id="1">1</sup></p><h2>连接 Plex</h2><p>剩下的就是将本地资料库添加到 Plex 进行刮削了，由于 Plex 自带的刮削器不太给力，比如说我就遇到了有些动漫和美剧无法找到的情况，我们可以用两个插件来解决这个问题。</p><p><a href="https://github.com/ZeroQI/Absolute-Series-Scanner" target="_blank">Absolute-Series-Scanner</a> 可以扫描且补全剧集的缺失信息</p><p><a href="https://github.com/ZeroQI/Hama.bundle" target="_blank">Hama.bundle</a> 用来从 Anidb 刮削数据，补全动漫刮削的缺失</p><p>The Movie Database (TMDb) 用来补全电影和剧集刮削的缺失</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/03/27/96503651017805b541bbeeba5f6916e0.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/03/27/96503651017805b541bbeeba5f6916e0.png" referrerpolicy="no-referrer"></figure><h3>Absolute Series Scanner</h3><p>我们先从 GitHub 上下载 Scanner，然后找到 Plex Media Server 的文件夹，位置在 <code>C:\Users\用户名\AppData\Local\Plex Media Server\</code>，将 <code>Scanners</code> 目录和子目录下的所有<code>.py</code> 文件放置于 Plex 服务器文件夹内即可生效。<sup class="ss-footnote" href title="注：Scanner 是包含对剧集 (Series) 和音乐 (Music) 的两种扫描器的，音乐扫描器亦可保留。" footnote-id="2">2</sup></p><h3>Hama bundle</h3><p>直接从 Github 下载最新的代码，解压于 <code>Plex Media Server</code>内的 <code>Plug-ins</code> 子目录下即可生效。</p><p>确定插件下载完成后，我们需要启用插件。打开 Plex 的媒体库设置，然后进入高级设置。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/03/27/6c75530f4a86ee8f3bd7d377a4810967.jpeg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/03/27/6c75530f4a86ee8f3bd7d377a4810967.jpeg" referrerpolicy="no-referrer"></figure><p>将扫描器选为 Absolute Series Scanner，代理选为 HamaTV。在 HamaTV 设置部分中：</p><ul><li>将 <code>Serie Lanaguge Priority</code> 设置为 <code>zh-Hans, zh, zh-Hant, ja, main, en</code></li><li>将 <code>Episode Language Priority</code> 设置为 <code>zh-Hans, zh, zh-Hant, ja, main, en</code></li><li>将 <code>TheTVDB Poster Language Priority</code> 设置为 <code>zh, ja</code></li></ul><p>如果你有其他亚洲语言需求，可以参考 GitHub 库里的语言参考表：</p><figure class="table"><table><tbody><tr><td style="border:1px solid rgb(221,
