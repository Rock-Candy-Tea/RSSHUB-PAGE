<div class="articleWidth-content" data-v-e1f0100a><div class="content wangEditor-txt minHeight" data-v-e1f0100a><p>像是会员，软件，点卡这类数字商品，倘若官方渠道支付不太方便或者淘宝上价格更加优惠，我倾向于直接在淘宝上购买。淘宝上不乏经营此类商品的正规店铺，但也有一些赚黑心钱的店家。他们来也匆匆去也匆匆，等你意识到上当时，却连店铺都消失不见。</p><p>这类虚拟商品的买卖监管并不严格，淘宝的开店门槛不高，卖家的货源不透明，想要买得放心还得消费者自己费心思。要筛选出合适且可信的店家，本文建议你事先了解官方的购买渠道及售价，再使用工具查看商品上架时间，最后通过咨询客服确认信息。三步走下来，很大程度上能规避因冲动或不了解产品而造成的损失。</p><h2 style="margin-left:0px;">了解官方购买渠道和售价</h2><p>有不少消费者在连官方的购买渠道都不知道的情况下直奔淘宝，看到某商品价格合适就下单，这样无异于羊入虎口。目前，所有数字商品的官方售卖渠道都是在自己的官网、官方 App 或者平台指定商城。决定在淘宝购买相关商品前，一定要事先去官网了解此商品的三类信息：</p><ol><li>数字商品的内容：是一年的订阅还是终身的授权？会员能解锁那些权益？</li><li>商品「寄送」方式：正版软件是和通过帐号激活还是激活码？数字会员是需要邮件确认还是直接订阅成功？</li><li>售价及支付方式：官方给出了哪些购买的平台？官方使用什么支付？年付还是月付？官方价格是怎样的？</li></ol><p>了解这三类信息，我们至少心理有个底，甚至能发现更好的购买途径。当店铺给出的信息和我们在官方渠道获得的信息不一致时，要么是我们有遗漏的信息没看到，要么是这个商品有问题。</p><p>以 <a href="https://affinity.serif.com/en-us/"><u>Affinity</u></a> 这个套件为例，假如前几个月我们想要购买时，事先检查了它的官网，发现官方做活动，官网半价反而比淘宝上便宜不少，就可以在官网或 Windows Store 直接购买。</p><p>再如，通过 <a href="https://secure.internetdownloadmanager.com/buy_idm.html"><u>IDM</u></a> 官网，我们得知有一年和永久两种套餐，价格分别为 76 和 160 元，通过激活码激活。</p><figure class="image ss-img-wrapper image_resized" style="width:789.167px;"><img src="https://cdn.sspai.com/2021/07/20/article/248f9577045101d2fb46de0ba24cf543?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="202107200013" data-original="https://cdn.sspai.com/2021/07/20/article/248f9577045101d2fb46de0ba24cf543" referrerpolicy="no-referrer"></figure><p>此时再对比一些淘宝店家 3 元 IDM 免激活码终身激活的描述，就不难发现，他们大概率在卖盗版。</p><h2 style="margin-left:0px;">查看商品的初次上架时间</h2><p>选择了一个看起来和官网信息一致的商品后，我们还可以通过其上架时间确认其是否可信。想要在淘宝代理商购买时长一年的软件订阅，那么当此商品已经正常售卖了超过一年时，显然可信度就很高。毕竟坏事做不长久。查看商品的初次上架时间可以使用一个 <a href="https://taodaxiang.com/shelf/index/init"><u>淘大象</u></a> 这个特殊的网页工具，只需要提供商品链接即可。</p><p>譬如我们想要买一个一年的 Microsoft 365，复制网址进淘大象后，发现他的创建时间是 2019 年，距离今天已经将近 2 年，那么我们就知道这个店家突然跑路的可能性不大。</p><figure class="image ss-img-wrapper image_resized" style="width:789.167px;"><img src="https://cdn.sspai.com/2021/07/20/article/48b4fc32ffd0654656cb4ca80c030678?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/07/20/article/48b4fc32ffd0654656cb4ca80c030678" referrerpolicy="no-referrer"></figure><p>另一个商家售卖某音乐软件的会员，声称自己是正规充值，长期经营，甚至有评论称自己已经第二次购买，但他的商品上架时间却说不了慌——是上个月，那么一般说来可信度不高。</p><figure class="image ss-img-wrapper image_resized" style="width:789.167px;"><img src="https://cdn.sspai.com/2021/07/20/article/c8ed13173e2c392e073cdb91b66ba3e4?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/07/20/article/c8ed13173e2c392e073cdb91b66ba3e4" referrerpolicy="no-referrer"></figure><p>其实我们还可以在网页版淘宝中看到一个店铺的注册时间，可以作为参考，但并不如商品上架时间有价值。</p><h2 style="margin-left:0px;">和客服沟通</h2><p>假的东西经不起追问，就算不是假的，问一问也没有坏处。问的内容可以是第一步中我们从官方了解到的消息，一些信息商家可能没有写在详情页面，我们可以确认一下这些信息是和官方一致的。如果遇到和官方消息不一样的地方，就可以以此为切入点继续追问下去：官方不是只能在应用商店里面购买吗？为什么这里卖的订阅软件需要我的帐号和密码？</p><p>只要店家是从正规渠道获得数字商品的，他们就一定能给出令人信服的解释。会员便宜可能是有什么官方活动，软件便宜可能是因为店铺促销……含糊其辞，顾左右而言他的回复必定有猫腻。与其冒风险占一点点小便宜，不如买个安心和保障。</p><p>最后，也希望数字商品的买卖体系能愈加完善，数字版权的严格保护能落实，只有这样，消费者才能放心，开发者也能从中获得实打实的收益。</p><p>题图：<a
