<div class="articleWidth-content" data-v-0b37afcb><div class="update-wrap" data-v-0b37afcb></div><div class="content wangEditor-txt minHeight" data-v-0b37afcb><p style="margin-left:0px;">众所周知，Chrome 浏览器是基于 Chromium 这个开源浏览器项目，但两者有着不小的区别，比如 Chrome 支持的系统平台其实是很有限的（这一点有点反常识），虽然目前主流的 Mac、Windows 等桌面平台都有对应的安装包，但 Linux 上只有 RPM 和 Deb 格式的安装包，且 Linux 安装包仅支持 X86-64 结构系统。因此如果你使用非 Debian/RHEL 发行版的 Linux（比如 Acrh Linux）或者是树莓派这样的 ARM 架构的设备，就只能安装 Chromium 浏览器——他基本上涵盖了大部分的系统平台。但唯一的问题是，从 2021 年 3 月开始 Google 开始限制 Chromium 浏览器登录 Google 账户，现在的 Chromium 浏览器中点击账户登录是没有 Google 账户登录入口的，好在 Chromium 官方也给出了相应的方法来教我们一步步的找回 Google 账户登录功能。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/01/31/dfd00509c85d703070ddb252f258029c.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/01/31/dfd00509c85d703070ddb252f258029c.png" referrerpolicy="no-referrer"></figure><h2 style="margin-left:auto;"><strong>加入相应的 Chrome 群组来获得相应权限</strong></h2><p style="margin-left:0px;">根据 Chromium 有关获取相应 Google API 的要求，我们首先要加入 <a href="https://groups.google.com/a/chromium.org/forum/?fromgroups#!forum/chromium-dev">chromium-dev</a> 这个 Google 群组来获取相应的 API 权限，这里你可以选择加入群组但不订阅群组邮件，因为我们只需要获得相应的 API 权限就行。</p><h2 style="margin-left:auto;"><strong>创建恢复 Google 账户登录项目</strong></h2><p style="margin-left:0px;">这里打开 <a href="https://console.cloud.google.com/">Google Cloud</a>（没有注册的话用 Google 账号注册一个），点击左上角项目选择「新建项目」。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/01/31/b10092c7a04212b959051cd04cb6a896.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/01/31/b10092c7a04212b959051cd04cb6a896.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">项目名称最好有所区别，这里我特意新建一个单独项目并命名，然后点击「创建」。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/01/31/22bec63e6210dad7ce998b3f21f4ef32.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/01/31/22bec63e6210dad7ce998b3f21f4ef32.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">然后回到当前项目的控制台，这里展开左边的汉堡菜单，找到「API 和服务」-「OAuth 同意屏幕」，选择「外部」然后再点击「创建」。</p><figure class="ss-imgRows" figcaption><img src="https://cdn.sspai.com/2022/01/31/9413d49f4d035b32c69a2e9785876790.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/01/31/9413d49f4d035b32c69a2e9785876790.png" referrerpolicy="no-referrer"><img src="https://cdn.sspai.com/2022/01/31/57cb799f8abbfe76e311b763ccc9a62f.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/01/31/57cb799f8abbfe76e311b763ccc9a62f.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">下面就是要创建一个信息，设定「应用名称」（我这里命名为 chromium login），然后「用户支持电子邮件」这里输入后面需要登录的 Google 邮箱。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/01/31/bde79404b59b1fe7e1e343dad182847b.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/01/31/bde79404b59b1fe7e1e343dad182847b.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">在第二步「范围」这里，选择「添加测试用户」，然后输入测试用户的邮箱，这里填写后面那个需要登录的 Google 邮箱。点击「添加」然后「保存并继续」。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/01/31/df933c797ad491a120c2e98644514738.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/01/31/df933c797ad491a120c2e98644514738.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">OAuth 同意屏幕创建好之后，我们回到控制台，然后还是从汉堡菜单中选择「API 和服务」-「库」。开始启用各种 API 库，这里我们需要启动的库包括：</p><ul><li>Cloud Search API</li><li>Google Drive API</li><li>Safe Browsing API</li><li>Time Zone API</li><li>Admin SDK</li><li>Chrome Sync API</li><li>Chrome Web Store API</li><li>Chrome Spelling API</li></ul><p style="margin-left:0px;">你可以搜索关键词然后点击「启用」即可，开启之后无需进一步的操作。</p><figure class="ss-imgRows" figcaption><img src="https://cdn.sspai.com/2022/01/31/543c80a8fd692bdb0408e49f3a5a6d56.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/01/31/543c80a8fd692bdb0408e49f3a5a6d56.png" referrerpolicy="no-referrer"><img src="https://cdn.sspai.com/2022/01/31/c7971db8f55b7d72c3f3b3bb623ea58a.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/01/31/c7971db8f55b7d72c3f3b3bb623ea58a.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">再次回到控制台，在汉堡菜单中找到「API 和服务」-「凭据」，这里需要创建可以调用这些 API 的关键 key，在凭据页面中点击「创建凭据」。</p><figure class="ss-imgRows" figcaption><img src="https://cdn.sspai.com/2022/01/31/29d3e94c487ed9697506c9bf54c15fa2.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/01/31/29d3e94c487ed9697506c9bf54c15fa2.png" referrerpolicy="no-referrer"><img src="https://cdn.sspai.com/2022/01/31/df772adabbf728b354555e2fbe9f04bc.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/01/31/df772adabbf728b354555e2fbe9f04bc.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">首先我们需要创建「API 密钥」，这里会直接生成一段 键值，记得将其保存到相应的位置；然后我们需要创建「OAuth 客户端 ID」，这里记得选择「桌面应用」，然后会生成对应的 客户端 ID 以及对应的客户端密钥，同样复制并保存下来。</p><figure class="ss-imgRows" figcaption><img src="https://cdn.sspai.com/2022/01/31/a01015492c301aab39388589011be349.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/01/31/a01015492c301aab39388589011be349.png" referrerpolicy="no-referrer"><img src="https://cdn.sspai.com/2022/01/31/3cd3634fdc59e9261deff0a3cf62f425.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/01/31/3cd3634fdc59e9261deff0a3cf62f425.png" referrerpolicy="no-referrer"></figure><h2 style="margin-left:auto;"><strong>将密钥加入到系统环境变量中</strong></h2><p style="margin-left:0px;">这里我们需要将上面获得的值、密钥等加入到你的系统环境变量中，打开系统终端，使用 vim 编辑环境变量文件（无需提权）：</p><pre class="language-shell"><code>vim ~/.xprofile</code></pre><p style="margin-left:0px;">按键盘的 <code>i</code> 进入编辑模式，然后输入下面的环境变量内容：</p><pre class="language-shell"><code>export GOOGLE_API_KEY=API 密钥「键值」
export GOOGLE_DEFAULT_CLIENT_ID=OAuth 客户端 ID
export GOOGLE_DEFAULT_CLIENT_SECRET=OAuth 客户端 ID 对应的密钥</code></pre><p style="margin-left:0px;">编辑完成之后按 <code>esc</code>退出编辑模式，在输入<code>:wq</code> 保存，然后直接注销当前的系统再重新登录，打开你的 Chromium 的账户入口，就可以看到 Google 账户登录入口了。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/01/31/6a82ce6aefb793e952271edc0ff78052.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/01/31/6a82ce6aefb793e952271edc0ff78052.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">参考资料：<a href="https://www.chromium.org/developers/how-tos/api-keys/">API Keys - The Chromium Projects</a></p></div><div class="update-details-wrap" data-v-0b37afcb></div><!----></div><div style="border:1px
