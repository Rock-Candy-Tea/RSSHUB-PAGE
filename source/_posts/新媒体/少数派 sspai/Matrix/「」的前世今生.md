<div class="articleWidth-content" data-v-6dd0eb6a><div class="update-wrap" data-v-6dd0eb6a></div><div class="content wangEditor-txt minHeight" data-v-6dd0eb6a><p style="margin-left:0px;"><strong>Matrix 首页推荐</strong> </p><p style="margin-left:0px;"><a href="https://sspai.com/matrix">Matrix</a> 是少数派的写作社区，我们主张分享真实的产品体验，有实用价值的经验与思考。我们会不定期挑选 Matrix 最优质的文章，展示来自用户的最真实的体验和观点。 <br>文章代表作者个人观点，少数派仅对标题和排版略作修改。</p><hr><h2>千变万化的「#」</h2><p>当看到「#」时，你会想到什么？</p><p>如果你下国际象棋，它可以表示对局的终结<sup class="ss-footnote" href title="国际象棋中，# 表示将军（checkmate）" footnote-id="1">1</sup>；如果你是程序员，它可以是注释的开始；如果你刷微博，它可以传递当前最热的话题。中文里，「#」通常称为「井字」或「井号」，在日语中称为「番号记号」或「井桁」，英式英语中则通常称为「hash」。</p><p>「#」的大部分名称来源于其形状，含义却随着上下文屡屡发生变化。对大部分人而言，与「#」的第一次会面，大概是电话拨号盘角落中那个不起眼的方块。</p><h2>libra pondo 的千年的漫步</h2><p>先将时间拨回罗马时代，那时「一磅重」被称为「<i>libra pondo</i>」，其中 <i>libra</i> 指「尺度」或「平衡」；<i>pondo</i> 则来自于动词 <i>pendere</i>，表示「称重」。从这两个词根中，「#」得到了它最初的形状与名称。</p><p>十四世纪晚期，来源于 <i>libra</i> 的「lb」进入了英语中，同时配上了一条表明其缩写身份的短线「~」，即「℔」。</p><p>由于「℔」在当时使用非常广泛，部分早期印刷机开始把它作为一个单独的字符进行印刷。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/05/21/772c28af9ae4d229d51b2343efd1ac48.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/05/21/772c28af9ae4d229d51b2343efd1ac48.png" referrerpolicy="no-referrer"><figcaption>左图是 Isaac Newton 笔下的 ℔ 符号，右图是它出现在 1698 年印刷品中的样子。</figcaption></figure><p>随后，「℔ 」逐渐演变成了两个不同的形式：</p><ul><li>来源于抄写员潦草笔记的「#」</li><li>没有短线修饰代表磅的「lb」</li></ul><p>而这个符号本身，却则淹没在时间长河中。与此同时，<i>libra </i>的老搭档 <i>pondo</i> 也开始了独自的演变之路。它先以 <i>pund</i> 的形式出现在古英语中，又在现代英语中成为 <i>pound</i>。历经数千年，<i>libra</i> 与 <i>pondo</i>，这一对古老的搭档，最终又重新连接在一起<sup class="ss-footnote" href title="在美式英语中，# 被称为 pound sign" footnote-id="2">2</sup>。</p><p>如果不是偶然的经历，也许「#」将一直存在于生活不起眼的角落，默默无闻地行使它最初的职责。</p><h2>拨号盘一隅</h2><p>在电话出现的早期，拨号数字利用脉冲拨号（Pulse dialing）进行传输。在这种技术中，每个数字对应一到十次脉冲信号。上世纪40年代末，贝尔实验室发明了一种新的拨号系统。新系统在每个数字被按下的时候，发出一个独特的声音信号，被称为双音多频（Dual-Tone Multi-Frequency）。然而直到1968年，电话上还只有 0-9 的拨号键。</p><figure class="ss-imgRows" figcaption="左图，早期使用脉冲拨号的转拨盘式电话；右图，贝尔实验室于 1963 年推出的世界第一台按键电话 WE 1500"><img src="https://cdn.sspai.com/2022/05/21/a90eb794ae6bddb8b85ad0aaa75cdfd1.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/05/21/a90eb794ae6bddb8b85ad0aaa75cdfd1.png" referrerpolicy="no-referrer"><img src="https://cdn.sspai.com/2022/05/21/f593b597421dfe4d15db67ff1e84b711.jpeg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/05/21/f593b597421dfe4d15db67ff1e84b711.jpeg" referrerpolicy="no-referrer"></figure><p>1968年，为了实现菜单选择等更多通话功能，贝尔实验室计划在数字 0 按键的两侧添加两个新的按键。在拨号盘新样式的设计过程中，曾诞生过一系列尝试，比如在 0 的两侧分别增加带有五角星与菱形符号的按键。然而这些字符在当时标准的打字机键盘上并不存在，给记录带来了障碍。此时，负责设计新拨号系统的贝尔实验室工程师 Douglas A. Kerr，被任命选择更加合适的符号替代它们。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/05/21/72482806e50428c8505f785630e13ac5.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/05/21/72482806e50428c8505f785630e13ac5.png" referrerpolicy="no-referrer"><figcaption>Douglas A. Kerr</figcaption></figure><p>Douglas Kerr 曾经担任设计美国信息交换标准代码（ASCII）的委员会负责人。基于过去的工作经验，也考虑到打字机的键盘设计，他与团队最终选择了包含在 ASCII 字符集中的「*」与「#」作为新的按键符号。</p><p>这个决定，使得「#」以前所未有的姿态进入了世界上绝大多数人的生活，它也将比历史上任何时候都更为人所熟悉。</p><h2>来「#」聊天吧</h2><p>1988年，Jarkko Oikarinen 创建 Internet Relay Chat (IRC)，这是一个基于文本的即时聊天系统。它被设计用于以组群的形式进行多人聊天，许多人可以加入同一个组群中，称为频道（channel）。频道的名称以「#」开头，用户可以用 <code>/join #channelname</code> 的方式加入某个频道。</p><figure class="image ss-img-wrapper image_resized" style="width:424px;"><img src="https://cdn.sspai.com/2022/05/21/354f6753b4f984b446c521dac65c9297.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/05/21/354f6753b4f984b446c521dac65c9297.png" referrerpolicy="no-referrer"><figcaption>IRC 客户端 HexChat 截图</figcaption></figure><p>2007年，曾经在 Google 和 Uber 担任开发主管的 Chris Messina 在 Twitter 上提议，可以用类似的系统来标记网络上感兴趣的主题，并发出了第一条倡议使用「#标签」的推文。然而该建议并没有立刻得到广泛认可，Twitter 联合创始人 Evan Williams 甚至告诉 Messina，这个概念永远都不会流行。</p><figure class="image ss-img-wrapper image_resized" style="width:472px;"><img src="https://cdn.sspai.com/2022/05/21/631bedaa3eedd9bba9a58d08052a2bb0.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/05/21/631bedaa3eedd9bba9a58d08052a2bb0.png" referrerpolicy="no-referrer"></figure><p>Messina 没有放弃这个概念，他甚至写了一篇 <a href="https://factoryjoe.com/2007/08/25/groups-for-twitter-or-a-proposal-for-twitter-tag-channels/">Groups for Twitter; or A Proposal for Twitter Tag Channels</a> 来详细阐释其使用方式。同年的圣地亚哥大火期间，他联系了在 Twitter 上直播事件的加州居民 Nate Ritter，并建议后者在所有相关推文中带上 <code>#SanDiegoFire</code> 的标签。Twitter 这才逐渐认识到到这是对内容进行分组的有效方式。终于，2009 年 Twitter 官方将标签系统加入搜索工具中。次年，热门话题功能出现，将时下最流行的话题展示在所有用户面前。</p><p>2014 年，牛津词典将其作为一个新的条目，添加到当年的更新中：</p><blockquote><p><strong>hashtag</strong> <i>n.</i> (on social media web sites and applications) a word or phrase preceded by a hash and used to identify messages relating to a specific topic; (also) the hash symbol itself, when used in this way.<br>在推特等社交网络媒介中，出现在 # 符号后用来标注主题的词语。</p></blockquote><h2>「#」的今天与明天</h2><p>如今，「#」已经很少用来表示重量，也只有偶然间才在通话时被按下，但它依旧活跃在互联网弄潮儿的指尖。如果要用一个词形容现在的它，那或许是「主题」。在用 Markdown 写作时，它界定着层级的标题，在开头结尾打下标签，以最简洁的方式记载着文章内容；在媒体上，一个个标签引导着时下最流行的趋势，却也在看不见的地方暗流涌动。相同的标签，连接不同的内容，把彼此独立的信息编织成网，同一个标签下，聚集着相似的人们。那么，是否某天一切都可以打上标签？标签网络之中，是否也隐藏着我们的未来？</p><figure class="image ss-img-wrapper image_resized" style="width:617px;"><img src="https://cdn.sspai.com/2022/05/21/db29f1ade7211361f6fe166ef9d7fda3.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" 年英文文献中出现的频率</figcaption></figure><blockquote><p>在你眼中，「#」代表什么？</p></blockquote><p><strong>图片来源</strong></p><ul><li><a
