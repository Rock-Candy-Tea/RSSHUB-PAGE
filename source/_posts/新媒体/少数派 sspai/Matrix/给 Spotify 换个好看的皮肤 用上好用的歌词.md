<div class="articleWidth-content" data-v-557a067a><div class="update-wrap" data-v-557a067a></div><div class="content wangEditor-txt minHeight" data-v-557a067a><p><strong>Matrix 首页推荐</strong></p><p><a target="_blank" rel="noopener noreferrer" href="https://sspai.com/matrix">Matrix</a> 是少数派的写作社区，我们主张分享真实的产品体验，有实用价值的经验与思考。我们会不定期挑选 Matrix 最优质的文章，展示来自用户的最真实的体验和观点。</p><p>文章代表作者个人观点，少数派仅对标题和排版略作修改。</p><hr><p style="margin-left:0px;">几年前有写过用 Spicetify 修改 Spotify 桌面客户端主题的<a target="_blank" rel="noopener noreferrer" href="https://sspai.com/post/58174">教程</a>，这几天再次重新装上发现有了挺多大的改进：lyric-plus 支持了网易云和 MusixMatch 歌词；fullAppDisplay 也能实现优雅全屏的播放；安装简化了很多主题也更为丰富。目前的状态已经让我觉得十分好看好用了，就特地再写篇 Spicetify 的教程。</p><h2 style="margin-left:auto;"><strong>安装 </strong>Spicetify<strong> 客户端</strong></h2><p style="margin-left:0px;">Spotify 在 Linux、Mac、Windows 桌面端都是 CEF (Chromium Embedded Framework) 架构，这给对客户端修改的修改提升了很大的便利和可行性。Spicetify 则是一款开源免费强大能修改 Spotify 客户端的主题、添加插件的命令行工具。Spicetify<a target="_blank" rel="noopener noreferrer" href="https://spicetify.app/"> 官网</a> 有安装教程，这里也是以官网的教程作为参考的。</p><h3 style="margin-left:0px;">Mac 或 Linux</h3><pre class="language-shell"><code>curl -fsSL https://raw.githubusercontent.com/spicetify/spicetify-cli/master/install.sh | sh
curl -fsSL https://raw.githubusercontent.com/spicetify/spicetify-marketplace/main/resources/install.sh | sh</code></pre><p style="margin-left:0px;">复制命令在终端里执行就能完成安装。</p><h3 style="margin-left:0px;">Windows</h3><pre class="language-shell"><code>iwr -useb https://raw.githubusercontent.com/spicetify/spicetify-cli/master/install.ps1 | iex
iwr -useb https://raw.githubusercontent.com/spicetify/spicetify-marketplace/main/resources/install.ps1 | iex</code></pre><p style="margin-left:0px;">复制命令到 powershell 里执行就能完成安装。</p><h2>使用内置的 Markertplace</h2><p>新版本的 Spicetify 安装完成后就自带有 Markertplace 功能在左侧的菜单：可以直接点击选择主题、插件、和修改界面的脚本，使用起来便利很多。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/09/07/36d0af91ffb48c9c6ef4ea7e217e914e.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/09/07/36d0af91ffb48c9c6ef4ea7e217e914e.png" referrerpolicy="no-referrer"><figcaption>App 已经内置了插件商店，从主题到功能拓展都有</figcaption></figure><h2 style="margin-left:auto;"><strong>添加主题</strong></h2><p style="margin-left:0px;">如果要使用第三方主题，我们要将主题文件添加到 Spicetify 的指定路径。GitHub 上有挺多 Spicetify 主题，也有整理好的 Spicetify 主题的库，我们仅仅需要将主题下载下来放到指定的路径下就能够使用了，这里推荐下载 Spicetify <a target="_blank" rel="noopener noreferrer" href="https://github.com/spicetify/spicetify-themes">官方主题库</a>，复制下面的命令就能完成添加主题。注意：下面的命令会用到 git，如果你的 Mac 和 Windows 没有安装过需要先<a target="_blank" rel="noopener noreferrer" href="https://git-scm.com/downloads">下载安装 git</a>。</p><h3 style="margin-left:0px;">Linux</h3><pre class="language-shell"><code>git clone https://github.com/spicetify/spicetify-themes.git
cd spicetify-themes
cp -r * ~/.config/spicetify/Themes</code></pre><h3 style="margin-left:0px;">Mac</h3><pre class="language-shell"><code>git clone https://github.com/spicetify/spicetify-themes.git
cd spicetify-themes
cp -r * ~/.config/spicetify/Themes</code></pre><h3 style="margin-left:0px;">Windows</h3><pre class="language-shell"><code>git clone https://github.com/spicetify/spicetify-themes.git
cd spicetify-themes
cp * "$(spicetify -c | Split-Path)\Themes\" -Recurse</code></pre><h2 style="margin-left:auto;"><strong>切换主题</strong></h2><p style="margin-left:0px;">官方仓库提供了 BurntSienna、Default、Dreary、Dribbblish、Flow、Glaze、Onepunch、Sleek、Turntable、Ziro 这几个主题，我们只需要在终端里执行下命令就能切换主题。</p><h3 style="margin-left:0px;">比如切换到 Ziro 主题</h3><pre class="language-shell"><code>spicetify config current_theme Ziro
spicetify apply</code></pre><p style="margin-left:0px;">我替换了不同的主题可以参看下：</p><figure class="image ss-img-wrapper"><img style="margin-left:auto;"><strong>设置插件功能</strong></h2><h3 style="margin-left:0px;">fullAppDisplay</h3><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/09/06/article/b3dad18c8a3d8512045daca173ad5046?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/09/06/article/b3dad18c8a3d8512045daca173ad5046" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">Spotify 官方客户端并没有窗口全屏播放的模式，我个人是非常喜欢这种简洁的播放界面的，有了 Spicetify 在终端里启用 fullAppDisplay 插件就能实现。</p><pre class="language-shell"><code>spicetify config extensions fullAppDisplay.js
spicetify apply</code></pre><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/09/06/article/a367fd4169ad047dc8143daead7c2e73?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/09/06/article/a367fd4169ad047dc8143daead7c2e73" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">安装好插件后，在左上角点击箭头图标就能启用。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/09/06/article/405a8e3a61d81e03e8b244b87eb18371?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/09/06/article/405a8e3a61d81e03e8b244b87eb18371" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">在窗口全屏播放里，右键可以控制各个元素的显示，自定义程度非常高。</p><h3 style="margin-left:0px;">lyrics-plus</h3><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/09/06/article/a5dbe54fd4d8a394bc56f5193bd6169c?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/09/06/article/a5dbe54fd4d8a394bc56f5193bd6169c" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">两三年前，Spotify 在少部分国家地区推出了歌词的功能，除了支持的给国家地区太少以外，不少朋友都抱怨 Spotify 的支持的歌词太少很多歌词也不能同步。而 Spicetify 的 lyric-plus 支持网易云和 MusixMatch 歌词体验真的会好很多，在终端执行即可开启 lyric-plus 功能：</p><pre class="language-"><code>spicetify config custom_apps lyrics-plus
spicetify apply</code></pre><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2022/09/06/dfd93f767d8ae89fda736dd5fbcbaa51.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/09/06/dfd93f767d8ae89fda736dd5fbcbaa51.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">另外 fullAppDisplay 和 lyric-plus 可以一起使用，看起来非常优雅。</p><h2 style="margin-left:auto;"><strong>最后</strong></h2><p style="margin-left:0px;">如果你想恢复 Spotify 默认的状态，执行 restore 命令即可：</p><pre class="language-"><code>spicetify restore</code></pre><p style="margin-left:0px;">几年前的 Spicetify 还非常简陋，仅仅是对 Spotify 主题配色进行少部分的修改；今天 Spicetify 我个人倒觉得挺实用，能提供相当美观简洁的播放界面的插件，和远比官方歌词更靠谱实用的歌词插件，安装和使用也简单便利了很多，对 Spotify 用户来说是个不错的选择。</p><p>>
