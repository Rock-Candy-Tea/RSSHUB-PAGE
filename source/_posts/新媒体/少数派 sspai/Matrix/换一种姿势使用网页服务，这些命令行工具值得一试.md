<div class="articleWidth-content" data-v-e1f0100a><div class="content wangEditor-txt minHeight" data-v-e1f0100a><h3><strong>Matrix 首页推荐</strong></h3><p><a href="https://sspai.com/matrix">Matrix</a> 是少数派的写作社区，我们主张分享真实的产品体验，有实用价值的经验与思考。我们会不定期挑选 Matrix 最优质的文章，展示来自用户的最真实的体验和观点。</p><p>文章代表作者个人观点，少数派仅对标题和排版略作修改。</p><hr><p>虽然不如具有图形界面的应用容易上手，命令行工具（CLI）往往有着更高的效率，这一点在通过浏览器提供的网页服务上尤其明显。如果你已经厌烦为了使用一个网页服务而一遍又一遍地打开浏览器 - 打开官网 - 选择文件 - 下载文件，或为了查个单词、测个网速而不得不承受各类无关推广的打扰，那么是时候换一种姿势使用这些服务了。</p><p>本文将要介绍的是一些网页服务的命令行版本，脱离了浏览器外壳的它们，保留了网页服务跨平台的特性，却更加简洁高效，还能与各种脚本或自动化联动。</p><p>要使用本文推介的应用，请首先安装 <a href="https://nodejs.org/"><u>Node.js</u></a>，并在使用中保证网络连接正常，这些应用大部分都不支持脱机使用。</p><h2 style="margin-left:0px;">TinyPNG CLI</h2><p>🌐项目地址：<a href="https://github.com/websperts/tinypng-cli"><u>GitHub</u></a> | 📦安装方式: <code>npm install -g tinypng-cli</code></p><p><a href="https://tinypng.com/"><u>TinyPNG</u></a> 是一个提供 PNG 和 JPEG 格式图片无损压缩的网站，压缩率高，并且基本不会丢失图片细节。TinyPNG CLI 则是利用其开发者接口制作的命令行工具。</p><p>使用 npm 安装 CLI 工具后，进入 TinyPNG 的 <a href="https://tinypng.com/developers"><u>开发者界面</u></a> 申请 API Key，在系统的用户目录创建名为「.tinypng」的文件并将获得的 Key 写入此文件。接着，我们就能在终端中使用<code>tinypng [filepath]</code>来压缩图片了，压缩后的图片将直接替换原图片。</p><figure class="image ss-img-wrapper image_resized" style="width:716px;"><img src="https://cdn.sspai.com/2021/04/07/article/7d4702f113aad19b9e6e643f31714e99?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/04/07/article/7d4702f113aad19b9e6e643f31714e99" referrerpolicy="no-referrer"></figure><p>用户每个月能免费压缩 500 张图片，每张图片不能超过 5 MB。</p><figure class="image ss-img-wrapper image_resized" style="width:716px;"><img src="https://cdn.sspai.com/2021/04/07/article/1119d8c33ce04533fd7e57b6792b325c?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/04/07/article/1119d8c33ce04533fd7e57b6792b325c" referrerpolicy="no-referrer"></figure><h2 style="margin-left:0px;">Fanyi</h2><p>🌐项目地址：<a href="https://github.com/afc163/fanyi"><u>GitHub</u></a> | 📦安装方式: <code>npm install -g fanyi</code></p><p>Fanyi 这款命令行小工具能够在终端中进行中文和英语的互相翻译。由于已经不再需要常备词典，我偶尔碰到不认识的词，还不想打开广告丛生的在线词典，此时就直接在终端中敲一行<code>fanyi [xxx]</code>。</p><p>在终端中输入<code>fanyi</code>，后面加上需要翻译的中文或者英文词句，句子的翻译或者是词的解释用法就会立马出现在你的面前。</p><figure class="image ss-img-wrapper image_resized" style="width:716px;"><img src="https://cdn.sspai.com/2021/04/07/article/8bea0b1929b8035c6d194ba31bf2ba00?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/04/07/article/8bea0b1929b8035c6d194ba31bf2ba00" referrerpolicy="no-referrer"></figure><h2 style="margin-left:0px;">Splash CLI</h2><p>🌐项目地址：<a href="https://github.com/splash-cli/splash-cli"><u>GitHub</u></a> | 📦安装方式: <code>npm install --global splash-cli</code></p><p>Splash CLI 是图片网站 <a href="https://unsplash.com/"><u>Unsplash</u></a> 的命令行客户端，它不仅能够管理和下载 Unsplash 账户下的图片，还能够从网站中获取图片设置为电脑壁纸。</p><figure class="image ss-img-wrapper image_resized" style="width:716px;"><img src="https://cdn.sspai.com/2021/04/07/article/9452ba8ef30fc883aca5ddd3b0ef4c27?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/04/07/article/9452ba8ef30fc883aca5ddd3b0ef4c27" referrerpolicy="no-referrer"><figcaption>图：GitHub</figcaption></figure><p>我们可以使用<code>splash --id [photo_id]</code> 下载指定图片并设置为壁纸，也可以使用<code>splash --collection [collection_id]</code>从某收藏集中随机获取图片。例如，想要获取 <a href="https://unsplash.com/collections/9717149/coronavirus-covid-19-pandemic"><u>COVID-19</u></a> 主题的壁纸，我们可以输入<code>splash --collection 9717149</code>。</p><figure class="image ss-img-wrapper image_resized" style="width:716px;"><img src="https://cdn.sspai.com/2021/04/07/article/e95b2bad91d230df5749caa6735619fb?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/04/07/article/e95b2bad91d230df5749caa6735619fb" referrerpolicy="no-referrer"><figcaption>Unsplash 的「收藏集」页面</figcaption></figure><p>配合 Windows 下的任务计划程序等自动化软件，就能实现每周自动更换一次壁纸的效果。</p><h2 style="margin-left:0px;">Speed-test</h2><p>🌐项目地址：<a href="https://github.com/sindresorhus/speed-test"><u>Speed-test</u></a> | 📦下载地址: <code>npm install -g speedtest-net</code> & <a href="https://www.speedtest.net/apps/cli"><u>Speedtest 官网</u></a></p><p>5G 时代下出镜非常频繁的测速服务 Speed-test 也能够在终端中被调用。其官方直接在官网提供了各个平台的 CLI 版本下载地址，将下载后的文件解压，在终端中运行相应文件便能获得结果。</p><figure class="image ss-img-wrapper image_resized" style="width:716px;"><img src="https://cdn.sspai.com/2021/04/07/article/46429c657cdb59cf9e8c9c7eb67883f8?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/04/07/article/46429c657cdb59cf9e8c9c7eb67883f8" referrerpolicy="no-referrer"></figure><p>同时，GitHub 上也非官方的 Node 包可供使用。<code>npm</code> 安装成功后，运行<code>speed-test</code>即可。</p><figure class="image ss-img-wrapper image_resized" style="width:716px;"><img src="https://cdn.sspai.com/2021/04/07/article/8b8d160a5912d922ee4c15227312f2bc?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/04/07/article/8b8d160a5912d922ee4c15227312f2bc" referrerpolicy="no-referrer"></figure><h2 style="margin-left:0px;">Weather-CLI</h2><p>🌐项目地址：<a href="https://github.com/riyadhalnur/weather-cli"><u>GItHub</u></a> | 📦安装方式: <code>npm install -g weather-cli</code></p><p>安装 Weather-cli 后，使用 <code>weather config -c [City] -C [Country]</code> 提前配置好城市和国家，再输入 <code>weather</code> 便可得到当前的天气，气温和空气质量。</p><figure class="image ss-img-wrapper image_resized" style="width:716px;"><img src="https://cdn.sspai.com/2021/04/07/article/34dd1a1b28cbdb4a2752c01a72b827b2?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/04/07/article/34dd1a1b28cbdb4a2752c01a72b827b2" referrerpolicy="no-referrer"></figure><h2 style="margin-left:0px;">Emoji-CLI</h2><p>🌐项目地址：<a href="https://github.com/watilde/emoji-cli"><u>GItHub</u></a> | 📦安装方式: <code>npm install -g emoji-cli</code></p><p>在文章之中添加一些 emoji 😃 多少可以让白纸黑字更加活泼一点，只是国内许多输入法的 emoji 搜索功能并不那么好用。关键时刻，还是要寻求 <a href="https://emojipedia.org/"><u>Emojipedia</u></a> 之类的网站的帮助。Emoji-CLI 则提供了在终端中搜索复制 emoji 的功能。</p><p>输入<code>emoji [keywords]</code>然后按回车，工具会输出匹配条件的 emoji，如果只有一个匹配项，会自动复制到剪贴板中；如果有多个匹配项，则会让用户使用箭头选择一项复制。<strong>搜索不支持中文</strong>，但工具可以离线使用。如果出现乱码的情况，则说明需要更换终端的字体。</p><figure class="image ss-img-wrapper image_resized" style="width:1140px;"><img src="https://cdn.sspai.com/2021/04/07/article/f57f186f17e53732faa239272c9ebf0e?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/04/07/article/f57f186f17e53732faa239272c9ebf0e" referrerpolicy="no-referrer"></figure><p>此外，<a href="https://github.com/sindresorhus/emoj"><u>emoj</u></a> 这个类似的工具能够实时搜索 emoji，可惜只有在 macOS 和 Linux 上效果才比较好。</p><h2 style="margin-left:0px;">PicGo-Core</h2><p>🌐项目地址：<a href="https://github.com/PicGo/PicGo-Core"><u>GItHub</u></a> | 📦安装方式: <code>npm install -g picgo</code></p><p>现在很少有人会去公共或私人图床手动上传图片并复制链接了，PicGo 等工具的出现大大简化了这一过程，而 PicGo-Core 是 PicGO 的 CLI 版本。使用 CLI 版本而不是官方给出的客户端是为了在脚本和其他程序中上传图片更加方便无感，比如 Typora 就集成了 PicGo-Core 以自动上传图片。</p><p>PicGo-Core 支持各类插件，例如笔者经常使用的自动复制和重命名插件，只需要输入<code>picgo install autocopy rename-file</code> 便能安装成功。除了通过修改 config 文件，PicGo-Core 还支持直接输入<code>picgo set</code>的方式配置程序本身和插件，用户体验比较优秀。</p><p>输入<code>picgo u</code>会直接上传剪贴板图片到默认图床，输入<code>picgo u [imagepath]</code>可以上传指定图片。</p><figure class="image ss-img-wrapper image_resized" style="width:716px;"><img src="https://cdn.sspai.com/2020/06/23/0bebb2a4b76a56a12fcb8899eb723aa3.gif" data-original="https://cdn.sspai.com/2020/06/23/0bebb2a4b76a56a12fcb8899eb723aa3.gif" referrerpolicy="no-referrer"></figure><p> </p><p>> 下载少数派 <a href="https://sspai.com/page/client">客户端 </a>、关注 <a href="https://sspai.com/s/J71e">少数派公众号 </a>，了解更妙的数字生活 🍃</p><p>> 想申请成为少数派作者？<a
