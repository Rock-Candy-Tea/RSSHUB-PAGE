<div class="articleWidth-content" data-v-1eaafa2a><div class="content wangEditor-txt minHeight" data-v-1eaafa2a><p style="margin-left:0px;">Typora 是多年以来我一直非常喜欢用的 Markdown 编辑器。</p><p style="margin-left:0px;">然而进入1.0版本之后，Typora 开始收费了。</p><p style="margin-left:0px;">89元的买断价格（3台设备），不算太贵。然而考虑到我本身就已经购买了挺多付费服务的（微信阅读、思源笔记、滴答清单 ......)，再投入89元到一个编辑器上我觉得有点心疼。</p><p style="margin-left:0px;">我对于 Markdown 编辑器有以下几点需求：</p><p style="margin-left:0px;">1.保存本地化。在网上冲浪这么多年之后，我开始反思 all in cloud 是否正确。至少从数据安全的角度来说，all in cloud 是不可靠的。所以文档存储本地化是我非常看重的一点。</p><p style="margin-left:0px;">2.支持云。这与上一条并不矛盾。不要把鸡蛋放在同一个篮子里。本地备份固然重要，再加一层云备份来增加保险又有何不可呢？而且云同步的便利体验过之后确实没法拒绝。</p><p style="margin-left:0px;">3.轻量化。这是我迭代自己的知识管理体系之后提出的需求。我需要的是一款能无压力输入的编辑器，不需要考虑配置、路径、环境，不用考虑要把文章归于哪一个文件夹，打开就能直接写作。</p><p style="margin-left:0px;">坦白来说我的这种需求不是特别迫切，因为思源笔记的功能已经足够满足我的创作功能。但是思源毕竟是一个功能非常全面的笔记软件，是我用于知识管理的主力，每次打开思源选择在哪个笔记本新建文档是一件需要思考的事情。</p><p style="margin-left:0px;">直到我遇见 <a href="http://bangle.io/"><u>Bangle.io</u></a> 。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/12/14/29e5104f0b38ad19c4eb68991b9659ab.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/14/29e5104f0b38ad19c4eb68991b9659ab.png" referrerpolicy="no-referrer"><figcaption><strong>面向下一个十年的笔记工具，开发者很有精神！</strong>😄</figcaption></figure><p style="margin-left:0px;">Powerful、Simple、locally，官网的简介已经很明确地表明了 Bangle的特点:</p><p style="margin-left:0px;">√ No sign up</p><p style="margin-left:0px;">√ Fully offline</p><p style="margin-left:0px;">√ No install</p><p style="margin-left:0px;">√ No data hostage</p><p style="margin-left:0px;">你不需要注册账号，不需要安装程序，不需要担心网络，不用担心文章被“劫持”，只需要打开浏览器就可以使用。</p><p style="margin-left:0px;"><a href="http://bangle.io/"><strong><u>Bangle.io</u></strong></a><strong> 麻雀虽小，五脏俱全</strong>。</p><p style="margin-left:0px;">Bangle 虽然是一个网页应用，但是它紧跟潮流的支持了" / "功能键：</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/12/14/5f2292da3e045cb5213953328db5d2ac.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/14/5f2292da3e045cb5213953328db5d2ac.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">无序/有序列表、层级、To-do等时下Markdown编辑器流行的功能它一应俱全，甚至广受喜爱的“ ：" 呼出表情功能 <a href="http://bangle.io/"><u>Bangle.io</u></a> 同样具备，如果你想到你只是打开了一个网页，你会为这些功能而感到惊喜。</p><p style="margin-left:0px;"><a href="http://bangle.io/"><u>Bangle.io</u></a> 还有自己的特色组合键：“Ctrl+shift+P” 可以呼出功能面板，“Ctrl+P”呼出工作空间（work space）面板，“Ctrl+R“ 切换工作空间。</p><p style="margin-left:0px;">另外<a href="http://bangle.io/"><u>Bangle.io</u></a> 对图片的支持可以说是意外之喜，复制粘贴即可，而且可以控制25~100的比例来控制宽度。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/12/14/42058b5e74f32ef9da2118b6d5ba7270.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/14/42058b5e74f32ef9da2118b6d5ba7270.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">另一个让人惊喜的是，<a href="http://bangle.io/"><u>Bangle.io</u></a> 支持正反链功能，紧跟潮流。其他功能或许都不如Typora完整，但是这一个正反链可以说解决了使用Typora的一大痛点：如何让过去的作品与当下结合。发现这个功能之后，我觉得 <a href="http://bangle.io/"><u>Bangle.io</u></a>真有「面向下一个十年」那味儿了。</p><figure class="image ss-img-wrapper image_resized" style="width:347px;"><img src="https://cdn.sspai.com/2021/12/14/b1a620fba9328c39562c8bfdce4e9231.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/14/b1a620fba9328c39562c8bfdce4e9231.png" referrerpolicy="no-referrer"><figcaption><strong>简单的正反链功能</strong></figcaption></figure><p style="margin-left:0px;"><a href="http://bangle.io/"><u>Bangle.io</u></a> 的文件虽然存储在本地，但你仍然可以使用坚果云之类的云盘实现同步功能，而且同步过程非常简单，你不需要像webDAV那样找地址和密码，直接用坚果云创建同步文件夹即可。</p><p style="margin-left:0px;"><a href="http://bangle.io/"><strong><u>Bangle.io</u></strong></a><strong> 功能虽全，但仍保持克制</strong>。</p><p style="margin-left:0px;">它只支持简单的文字编辑，不要奢望它能像wolai、思源、notion那样有全功能的编辑功能，它不能更改字体颜色，不支持表格（简单表格也不行），不能嵌套网页，可以用 “#” 来打标签，但不要指望它有database这种事情，<a href="http://bangle.io/"><u>Bangle.io</u></a> 目前只是一个功能尚可的Markdown编辑器，它的所有功能，恰好能最低限度地满足你的写作需求。</p><p style="margin-left:0px;">更难能可贵的是，它免费，而且在国内是可用的。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/12/14/1aca9539fb4d2ad4273475d1928ae2d0.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/12/14/1aca9539fb4d2ad4273475d1928ae2d0.png" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;">如果你也在寻找 Typora 的替代品，追求无压力写作，那么我强烈推荐你打开浏览器，输入 <a href="http://bangle.io/"><u>https://Bangle.io</u></a> 去尝试一下。</p></div><!----></div><div style="border:1px solid transparent;" data-v-1eaafa2a></div><div class="article-side sideTop"
