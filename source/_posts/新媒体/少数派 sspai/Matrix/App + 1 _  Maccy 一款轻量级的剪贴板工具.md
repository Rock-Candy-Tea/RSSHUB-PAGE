<div class="articleWidth-content" data-v-6a669db8><div class="content wangEditor-txt minHeight" data-v-6a669db8><p>最近在GitHub上发现了一款为macOS开发的剪贴板工具，名字叫做「<a href="https://maccy.app/" target="_blank">Maccy</a>」。至于推荐的原因是它的特点：<strong>开源、原生、轻量。</strong></p><h2><strong>安装</strong></h2><h3>HomeBrew</h3><p>首先最推荐方式就是通过安装，仅需通过brew命令即可安装更新：</p><p><code><s>brew install --cask maccy </s></code><sup class="ss-footnote" href="https://sspai.com/u/limestone" title="经评论区“锦瑟无端五十弦”提醒现在的 Homebrew 已再不需要额外使用 --cask 参数安装 Cask 程序，所以直接使用 brew install maccy 就好了" footnote-id="1">1</sup></p><p><code>brew install maccy </code></p><h3>官网/Apple Store</h3><p>「Maccy」的官方网址是 <a href="https://maccy.app/" target="_blank">https://maccy.app</a> ，在官网可以任意价格购买App，无论是0元使用或是支持作者。同时「Maccy」在Apple Store售价为68¥。</p><span class="ss-application" app-id="16214"> </span><h2>开源+轻量+原生</h2><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/10/08/57c7987faed0308bfb671c75f8cfb13a.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2021/10/08/57c7987faed0308bfb671c75f8cfb13a.png" referrerpolicy="no-referrer"></figure><p>在「<a href="https://github.com/p0deje/Maccy" target="_blank">Maccy」的GitHub页面</a>上可以看到这个项目是使用Swift开发的，最低适配系统为macOS Mojave 10.14。</p><p>在系统占用方便也可以体现出「Maccy」的轻量，得益于Swift的原生开发使得无论是在CPU还是内存占用都足够低；</p><figure class="ss-imgRows" figcaption><img src="https://cdn.sspai.com/2021/10/08/cef11e700edb4aa5ba6c744b2705426d.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2021/10/08/cef11e700edb4aa5ba6c744b2705426d.png" referrerpolicy="no-referrer"><img src="https://cdn.sspai.com/2021/10/08/ad8d757aa759b9e1f041b27b1550cfe2.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2021/10/08/ad8d757aa759b9e1f041b27b1550cfe2.png" referrerpolicy="no-referrer"></figure><p>在功能方面“麻雀虽小，但五脏俱全”，像搜索功能、Pin单条记录、无格式粘贴忽略钥匙串等基础功能都具备。值得一提的是App奉行「keyboard-first」一切操作都可以组合键来处理，最大程度减少移动鼠标的时间。</p><figure class="image ss-img-wrapper image_resized" style="width:556px;"><img src="https://cdn.sspai.com/2021/10/08/7a581a7a17529aec4b2f3744b18d8475.gif" data-original="https://cdn.sspai.com/2021/10/08/7a581a7a17529aec4b2f3744b18d8475.gif" referrerpolicy="no-referrer"><figcaption>官网搜索功能示例</figcaption></figure><h2>使用感受</h2><p>对于轻量用户来说，这款App无疑是最好的选择，免费开源和最接近macOS的原生界面，以及足够的功能。在我个人使用上设置为通过快捷键唤醒窗口同时取消菜单栏图标以及底部菜单，在使用体验下十分接近原生的功能。非常推荐大家尝试一下！</p><figure
