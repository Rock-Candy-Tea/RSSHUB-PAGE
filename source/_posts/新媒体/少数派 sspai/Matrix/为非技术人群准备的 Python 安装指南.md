<div class="articleWidth-content" data-v-6a669db8><div class="content wangEditor-txt minHeight" data-v-6a669db8><p style="margin-left:0px;text-align:justify;">通常一个没有接触计算机的新人打算学编程，问我学什么语言，我都会告诉他「学 Python」，Python 很容易学习又非常的强大，对于非专业编程人士而言可谓是最佳选择，对于专业人员而言掌握一门脚本语言也可以很大程度提高编程效率。</p><blockquote><p style="margin-left:0px;">该文章的唯一受众是完全不会配置 Python 环境的人，因此一切以简单为第一要义而不会过于考虑最优解。如果你已经有了适合你的 Python 环境那么本文并不适合你，当然，也欢迎你写一篇新的文章（而非在本文下评论，因为这可能会对完全的新人造成困惑）更进一步的分享你的环境搭建方式让大家学习</p></blockquote><h2 style="margin-left:0px;text-align:justify;"><strong>选择：Miniconda</strong></h2><p style="margin-left:0px;text-align:justify;">对于任何人而言，我都会推荐使用 Miniconda 来安装 Python，对于初级开发者而言，它可以最大限度的降低安装的难度，而对于高级开发者而言则可以很方便的创建多个不同 Python 版本的虚拟环境。</p><p style="margin-left:0px;text-align:justify;">Miniconda 是 Anaconda 的简化版，或者更准确地说，Anaconda 不过是 Miniconda + 一些数据科学需要用到的库而已。相比于 Anaconda 1G+ 的体积，Miniconda 50M 的大小格外友好，因此对于不需要进行科学分析的我们而言使用 Miniconda 才是最优选择。</p><p style="margin-left:0px;text-align:justify;">另外，由于 Miniconda 的诞生是在 Anaconda 之后的、是 Anaconda 决定精简后推出的，因此在网络上你或许搜到的绝大多数问题都是 Anaconda 而非 Miniconda（甚至可能某些在用 Anaconda 的人都不知道 Miniconda 的存在），不过绝大多数情况下他们都是通用的，所以当你遇到问题时利用 Miniconda 搜索不到时可以换成 Anaconda 碰碰运气。</p><p style="margin-left:0px;text-align:justify;">另外，Anaconda 和 Miniconda 的版本号其实让人有些困惑。截止本文写作之时，最新的 Anaconda 版本名字叫做 Anaconda3 4.10.3。事实上，目前所有的 Anaconda 都是叫做「Anaconda3」，因为这里的 3 是对应的其附带的 Python 版本的，因此 Anaconda 3 所附带的是 Python 3，而随着 Python 2 正式退出舞台，Anaconda2 目前已经正式消失。至于 4.10.3 才是 Anaconda 的真正版本，这也是 Anaconda 安装后对应的 conda 命令（提供了 Anaconda 环境的管理）的版本。</p><p style="margin-left:0px;text-align:justify;">好了，关于 Anaconda/Miniconda 的布道与讲解就这么多，下一步就是正式开始安装了。对了，如果你只有 Mac / Windows 二者中的一个，可以只看相关章节，我在撰写时特意没有合并重复的内容以最大化降低你出错的可能。而如果你用的是 Linux，我相信你肯定有自己安装的能力：）</p><h2 style="margin-left:0px;text-align:justify;"><strong>Mac</strong></h2><h3 style="text-align:justify;"><strong>安装 Miniconda</strong></h3><p   - defaults
show_channel_urls: true
default_channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/msys2
custom_channels:
  conda-forge: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  msys2: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  bioconda: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  menpo: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  pytorch: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  simpleitk: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud</code></pre><p style="margin-left:0px;text-align:justify;">按 Control + X 保存（并在后续询问时输入 <code>y</code> 确认修改、回车确认路径）并执行一次 <code>conda data-original="https://cdn.sspai.com/2021/10/31/article/c60b5ce2499ac738af1d200adf8d8daa" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;text-align:justify;">然后安静的等待安装进度条走完，等待时你可以考虑阅读下 Python 的<a href="https://docs.python.org/zh-cn/3/tutorial/index.html">官方教程</a> 并考虑放弃学习编程（bushi</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/10/31/article/6832f2674a5583100fb2fa16a8b42c3a?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2021/10/31/article/6832f2674a5583100fb2fa16a8b42c3a" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;text-align:justify;">安装完成，有两个勾可以选择，主要是教你如何使用 Anaconda 的，感兴趣可以看看，点击 Finish 即安装完成</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/10/31/article/233d5bd1cbd020486204ebd62538a6ce?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2021/10/31/article/233d5bd1cbd020486204ebd62538a6ce" referrerpolicy="no-referrer"></figure><h3 style="text-align:justify;"><strong>测试安装情况</strong></h3><p style="margin-left:0px;text-align:justify;">这时候你看你的开始菜单，会发现里面多了两个和 Miniconda 有关的内容：Anaconda Powershell Promt (miniconda3) 和 Anaconda Promt (miniconda3)，这两个的区别主要是一个依托于 Cmd 一个依托于 Powershell，这属于一个 Windows 系统的历史问题，严格上 Powershell 比 Cmd 更先进强大</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/10/31/article/cb501fc9a346db07127fa9e2d2cd8d88?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2021/10/31/article/cb501fc9a346db07127fa9e2d2cd8d88" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;text-align:justify;">Windows 7 系统上可能没有 Powershell，因此这里我使用「Anaconda Promt (miniconda3)」做演示。点击打开它可以发现它就是一个「黑乎乎的窗口」</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/10/31/article/7b697c91d98f16b68301148cd852090e?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2021/10/31/article/7b697c91d98f16b68301148cd852090e" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;text-align:justify;">这个窗口未来会非常常用，一切和 Python 打交道的事情几乎都会在这里完成，你可以依次输入以下命令（每行是一个命令，注意输入时的空格、拼写、大小写不要出错，所有的 version 前面都是有两个 <code>-</code> 的）验证下是否安装成功</p><pre class="language-shell"><code>conda --version
python --version
pip --version</code></pre><p style="margin-left:0px;text-align:justify;">如果你的输出和我的类似（版本号可能因时间而变化，但整体应该是差不多的，至少不会有什么「找不到」「错误」「异常」「Error」「Exception」的字样出现），那么恭喜你，已经安装成功了</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/10/31/article/647bd43b62d8c4cad848e67e0bb0f43a?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2021/10/31/article/647bd43b62d8c4cad848e67e0bb0f43a" referrerpolicy="no-referrer"></figure><h3 style="text-align:justify;"><strong>配置 conda 和 pip 的镜像加速</strong></h3><p style="margin-left:0px;text-align:justify;">这时 Python 已经可以正常使用了，但为了加快在国内的访问速度，我们可以更进一步的配置相关包的镜像服务器，我们需要分别配置 conda（Miniconda 的包管理与环境管理工具）和 pip（Python 的包管理工具）的镜像，这里也依然使用清华镜像站为例，官方文档分别位于 <a href="https://mirror.tuna.tsinghua.edu.cn/help/anaconda/">Anaconda 镜像使用帮助</a> 和 <a href="https://mirrors.tuna.tsinghua.edu.cn/help/pypi/">pypi 镜像使用帮助</a> 。</p><p style="margin-left:0px;text-align:justify;">首先在刚刚的窗口内执行 <code>conda config --set show_channel_urls yes</code>，不输出任何内容即为成功。</p><p style="margin-left:0px;text-align:justify;">然后输入 <code>notepad %HOMEPATH%\.condarc</code> （注意使用英文输入法输入，condarc 前面有个点），将打开的记事本内的内容用下面的覆盖</p><pre class="language-line_char"><code>channels:
  - defaults
show_channel_urls: true
default_channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/msys2
custom_channels:
  conda-forge: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  msys2: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  bioconda: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  menpo: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  pytorch: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  simpleitk: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud</code></pre><p style="margin-left:0px;text-align:justify;">保存后再执行一次 <code>conda clean -i</code> 清除已有缓存即可</p><p style="margin-left:0px;text-align:justify;">以上是配置了 conda 的镜像，配置 pip 的就简单多了，执行 <code>pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple</code> 就可以成功设置</p><h2 style="margin-left:0px;text-align:justify;"><strong>第一个脚本</strong></h2><blockquote><p style="margin-left:0px;">这里开始就不分操作系统啦，如果有差异我会说的</p></blockquote><p style="margin-left:0px;text-align:justify;">安装成功以后，让我们执行我们的第一个 Python 脚本以验证一切 OK（记得顺便阅读下输出的 Python 哲学以感受 Python 的魅力）</p><p style="margin-left:0px;text-align:justify;">输入 <code>python -c "import this"</code> （注意在英文输入法下输入，注意空格、引号及拼写）来查看执行结果</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/10/31/article/158dcabeb73de736b5019fa58c9779a5?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2021/10/31/article/158dcabeb73de736b5019fa58c9779a5" referrerpolicy="no-referrer"></figure><p style="margin-left:0px;text-align:justify;">这个命令和你写一个内容为 <code>import this</code> 的脚本然后执行效果是一样的，下面我们就来尝试写下这个脚本。</p><p style="margin-left:0px;text-align:justify;">编写代码可以使用 Windows 系统自带的记事本工具或 Mac 系统自带的文本编辑工具，但我更推荐你使用专为编程所设计的编辑器或是 IDE（老规矩，好奇区别的话记得搜索，多了解点总没坏处），这是为了避免一些空格、编码、BOM 等问题。编辑器和 IDE 的选择每个人都有不同的偏好，我个人推荐你如果想要轻量简单可以下载 <a href="https://www.sublimetext.com/">Sublime Text</a> 编辑器，而想要真的步入 Python 编程的世界那么推荐使用 <a href="https://www.jetbrains.com/pycharm/">PyCharm</a> IDE。多说一句，虽然这两个都是付费软件，但 Sublime Text 可以无限制试用只是会在一段时间内提醒你购买一次，PyCharm 则有免费的 Community 版本，而 Professional 多的则主要是和科学计算网站开发有关的内容，对于大多数用户而言，免费的 Community 版本已经够用了，而如果你是学生的话则可以免费使用 Professtional 版本，完全没有必要去寻找使用盗版（特别是我认识一个学生竟然还找盗版，根本不知道学生可以使用免费的正版……）。</p><p style="margin-left:0px;text-align:justify;">为了保持简单，我们这里以 Sublime Text 4 为例，进入<a href="https://www.sublimetext.com/download">下载页面</a>，根据你的操作系统选择下载。</p><figure class="image ss-img-wrapper"><img src="https://cdn.sspai.com/2021/10/31/article/08b9283f34eece6b9fba017b22b8b3a2?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt 用户直接将下载好的文件双击解压（如果是 Safari 下载的则自动解压好了）并将解压得到的 .app 文件移动到「应用程序」文件夹下。</p><p style="margin-left:0px;text-align:justify;">安装成功后，打开，直接输入 <code>import this</code> （输入的过程注意不要有多余的空格）并选择一个合适的路径保存。</p><p style="margin-left:0px;text-align:justify;">为了简单，如果是 Windows 请保存在根目录下并命名为 <code>first.py</code>，然后进入到刚刚的 Anaconda Promt 窗口输入 <code>cd C:\</code> 切换到根目录、输入 <code>python first.py</code>，输出内容应当是与最开始的完全相同的</p><figure class="image ss-img-wrapper"><img 请保存在家目录（可以在访达中进入到桌面并按 Command + 上箭头进入）下并命名为 <code>first.py</code>，然后打开终端输入 <code>cd ~</code> 和 <code>python first.py</code>，输出内容应当是与最开始的完全相同的</p><figure class="image ss-img-wrapper"><img
