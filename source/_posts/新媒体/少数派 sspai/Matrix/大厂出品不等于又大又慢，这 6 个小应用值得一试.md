<div class="articleWidth-content" data-v-557a067a><div class="update-wrap" data-v-557a067a></div><div class="content wangEditor-txt minHeight" data-v-557a067a><p>「大厂」应用在你的想象中是什么样子？动辄 10G 的占用空间，还是让老旧设备突发高热的迟缓动效？</p><p>我们总是看到这样的例子——当一家软件公司开始做大做强，用户量飞速上涨之后，他们的产品往往就走向一条不可挽回的庞杂之路。开发团队试图用更多的新功能和更大的饼圈住更多用户，越来越臃肿的界面和逐渐扩充的体积也让用着老设备的用户更加痛苦。</p><p>不过大厂也并非所有产品都如此不堪，总有一些优质应用，因为顶着「大厂」的帽子让用户望而却步。这篇文章就简单介绍几个由头部开发商开发的小应用，希望这些好产品能得到它们应有的认识。</p><h2>微软数学：你手机里的数学老师</h2><p>平台：<a target="_blank" rel="noopener noreferrer" href="https://apps.apple.com/us/app/microsoft-math-solver/id1483962204?_branch_match_id=680484335697283429">iOS</a> / <a target="_blank" rel="noopener noreferrer" href="https://www.coolapk.com/apk/com.microsoft.math">Android</a> / <a target="_blank" rel="noopener noreferrer" href="https://math.microsoft.com/zh">网页</a></p><figure class="ss-imgRows" figcaption><img src="https://cdn.sspai.com/2022/09/09/5e92b8f76abe6cfc44f00b6de5594341.JPEG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/09/09/5e92b8f76abe6cfc44f00b6de5594341.JPEG" referrerpolicy="no-referrer"><img src="https://cdn.sspai.com/2022/09/09/71bcf0903325d02473e933448b5c9362.JPEG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/09/09/71bcf0903325d02473e933448b5c9362.JPEG" referrerpolicy="no-referrer"></figure><p>微软数学是一款用来求解数学方程式的 App——但它不止于此。通过拍照、手写或者是键盘输入待求解的方程式，它不仅可以帮你求出结果，还可以查看详细求解过程，必要时还会附加图片，再也不怕答案里的「略」。</p><p>书签、历史记录功能都很完善，App 还提供了测验功能，可以根据运算过的方程式生成类似的题目，帮助你巩固知识点。详细的功能我们在之前的 <a target="_blank" rel="noopener noreferrer" href="https://sspai.com/post/57557">App+1</a> 文章中有过介绍，如果你常常被数学困扰，微软数学可以让你的学习之路更加轻松。此外，Edge 浏览器已经内置了微软数学，你可以在打开含有数学公式的页面时尝试这一功能。</p><h2>Layout：快捷图像拼贴工具</h2><p>平台：<a target="_blank" rel="noopener noreferrer" href="https://apps.apple.com/us/app/instagram/id389801252">iOS</a> / <a target="_blank" rel="noopener noreferrer" href="https://play.google.com/store/apps/details?id=com.instagram.layout">Android</a></p><figure class="image ss-img-wrapper image_resized" style="width:336px;"><img src="https://cdn.sspai.com/2022/08/21/f9b58f579a6605640f5868e45efbb59b.JPEG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/08/21/f9b58f579a6605640f5868e45efbb59b.JPEG" referrerpolicy="no-referrer"></figure><p>Layout 是一款用来拼接图片的 App，开发者是当时推出了多款独立图像编辑应用的 Instagram。Layout 功能简单，操作也很直观，选择图片与给定的拼接模板即可。不过在 Facebook（Meta）再次改变策略后，这些独立应用又都纷纷整合回了 Instagram。2020 年 Instagram 上推出的 Layout 功能就是这款 App 的延伸。</p><p>Layout 上次更新停留在 2018 年末，但其体积小巧，功能粗暴，成图迅速，到今天还有不错的下载量。缺点是对屏幕尺寸的适配有些弱，部分机型可能无法全屏显示。</p><h2>汽水音乐：字节跳动的音乐流产品</h2><p>平台：<a target="_blank" rel="noopener noreferrer" href="https://apps.apple.com/cn/app/%E6%B1%BD%E6%B0%B4%E9%9F%B3%E4%B9%90/id1605585211">iOS</a> / <a target="_blank" rel="noopener noreferrer" href="https://www.coolapk.com/apk/com.luna.music">Android</a></p><p>汽水音乐是字节跳动面向在线音乐市场交出的答卷。在音乐版权方面，相比起家大业大的 TME，汽水音乐距离它们都还有着不小的差距。但它的推广优势也在于此——汽水音乐的音乐库与抖音关联紧密，无论你喜欢与否，对于当下热衷于短视频的年轻人，甚至年轻音乐人来说，汽水音乐是个非常有效的收听/推广平台。</p><p>和其他音乐 App 最大的不同是，汽水音乐没有上一首/下一首按钮，而是和抖音类似，依靠滑动播放页面来切换歌曲。可以说，汽水音乐延续了「流」的概念处理音乐，更接近热门歌曲电台，而非音乐播放器。</p><figure class="ss-imgRows" figcaption><img src="https://cdn.sspai.com/2022/09/09/6a47f49bc2bd68838322ad8b0bfbd888.JPEG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/09/09/6a47f49bc2bd68838322ad8b0bfbd888.JPEG" referrerpolicy="no-referrer"><img src="https://cdn.sspai.com/2022/09/09/c06c542d4c76709d3a8b4f14f830407e.JPEG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/09/09/c06c542d4c76709d3a8b4f14f830407e.JPEG" referrerpolicy="no-referrer"></figure><p>App 的设计总让人感觉看到了别人的影子，也有一些或大或小的问题，比如播放界面风格不统一，歌单界面与主界面风格割裂等。但是对比其他国内竞品，在设计上真的保持了足够的克制。除了听歌，没有多余的功能。</p><p>没有开屏广告，没有「运营推广」，没有弹框，没有小红点，没有推广 banner，没有自家其他业务的引流入口。如此纯净的音乐体验，在当下显得十分可贵。就是不知道汽水音乐这样的态度还能延续多久，祝它好运。</p><h2>腾讯日历：能与腾讯会议联动的优质日历</h2><p>平台：<a target="_blank" rel="noopener noreferrer" href="https://apps.apple.com/cn/app/id1578243689">iOS</a> / <a target="_blank" rel="noopener noreferrer" href="https://www.coolapk.com/apk/com.tencent.calendar.app">Android</a> / <a target="_blank" rel="noopener noreferrer" href="https://rili.tencent.com/">Web</a> / 微信小程序</p><figure class="image ss-img-wrapper image_resized" style="width:334px;"><img src="https://cdn.sspai.com/2022/08/21/e27c57124f2f872fee1859c59b5b2932.JPEG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/08/21/e27c57124f2f872fee1859c59b5b2932.JPEG" referrerpolicy="no-referrer"></figure><p>腾讯日历是腾讯推出的日历 App。在 2.0 版本，它们将整体色调进行了更换，并加强了社交生活与互联互通的属性。详细内容在《<a target="_blank" rel="noopener noreferrer" href="https://sspai.com/post/74256">腾讯日历 2.0：焕新的日程管理工具</a>》一文中有介绍。</p><p>腾讯日历的优势非常明显，它可以与微信、企业微信、腾讯会议等服务打通，在国内目前的应用场景下，其实便利性不错。而且腾讯日历目前的版本在功能设置上比较克制，着重强调了作为一个「好用的工具」的场景。工作上可以快速同步会议等日程，生活上又可以与微信好友联动，或是快速安排探索发现中附近好玩的活动。对于与微信生态绑定颇深的用户来说，与其为了日历的同步头疼，不如真的试试它们自家产的东西。</p><h2>乘车码：不再被在闸机口卡死的微信拖后腿</h2><p>平台：<a target="_blank" rel="noopener noreferrer" href="https://apps.apple.com/cn/app/%E4%B9%98%E8%BD%A6%E7%A0%81/id1446262556">iOS</a> / <a target="_blank" rel="noopener noreferrer" href="https://a.app.qq.com/o/simple.jsp?pkgname=com.tencent.fit.ccm">Android</a></p><figure class="image ss-img-wrapper image_resized" style="width:336px;"><img src="https://cdn.sspai.com/2022/08/22/485cfc43d2a585b73cb2646214fa569f.JPEG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" data-original="https://cdn.sspai.com/2022/08/22/485cfc43d2a585b73cb2646214fa569f.JPEG" referrerpolicy="no-referrer"></figure><p>乘车码 App 是微信乘车码服务剥离出的独立 App。相比起在微信里启动乘车码小程序，做一个单独的 App 要快那么亿点点。App 登录时会提示需要小程序授权，跳转微信授权后即可使用。如果多退出几次程序，它也会提供账号密码登录的选项。对于一些老旧机型，这样功能单一的应用显然比在地铁闸机口被卡死的微信拖后腿划算太多了。</p><p>不过考虑到微信账号系统的复杂性，如果你采用账号密码登录，建议先尝试几次，再只带着这个应用乘车，防止出现账号掉线的问题。</p><h2>布丁扫描：vivo 出品的 OCR 扫描工具</h2><p>平台：<a target="_blank" rel="noopener noreferrer" href="https://apps.apple.com/cn/app/id1626815079">iOS</a> / <a target="_blank" rel="noopener noreferrer" href="https://www.coolapk.com/apk/com.wibo.bigbang.ocr">Android</a></p><p>布丁扫描的前身是 vivo 系统内置的琥珀扫描。在跟随 vivo X Fold / Note 发布之后，原本等着提取安装包的爱好者等来了官方的开放下载，不过名字从琥珀最终定成了「布丁」。和市面上常见的 OCR 应用相比，布丁扫描在保持成品优秀的前提下做到了功能免费，可以说是 vivo 软件方面送来的一份大礼。</p><figure class="ss-imgRows" figcaption><img src="https://cdn.sspai.com/2022/09/09/734c570e9b07a1a7a3958361666d6dbf.jpg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/09/09/6b9362ddef789b5003ba27d40e9988c8.jpg" referrerpolicy="no-referrer"></figure><p>应用提供了文档扫描、证件扫描、文字提取、表格识别等四大功能，同时可以提供了照片修复与口算批改等附加功能。除去纸面文字的扫描，识别电脑屏幕时，布丁扫描还提供了摩尔纹消除等图片处理，实测效果不错。不过 iOS 端上诸如照片修复等附加功能尚未上线。</p><figure class="ss-imgRows" figcaption><img src="https://cdn.sspai.com/2022/09/09/b451724603388116d6c334ab0b4a8440.jpg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt data-original="https://cdn.sspai.com/2022/09/09/b4fc1f36f2677a120738067d59eaaa98.jpg" referrerpolicy="no-referrer"></figure><p>坊间有一个奇妙的传闻：Xbox 游戏机在起名时没有带上「微软」，是因为「微软」会对产品的印象带来负分。在大厂的主力产品逐渐被负面口碑淹没时，它们的优质产品往往也会因为这些刻板印象被掩盖。虽然大厂的产品更容易被用户与增量拖累，但它们中还有不少的闪光点。此外，还有一些尚未公开，或是曾经很优秀的作品，比如腾讯的「电视版」腾讯视频「片多多」，曾经的便签王者锤子便签，以及局限在 ColorOS 内部尚未对外公开下载的 OPPO Relax 等。希望这些还藏在家里的好应用可以有开放给大众的打算，也希望有更多的大厂可以重视一下产品的体验，带来更多更新的好应用。</p><h4>关联阅读</h4><ul><li><a target="_blank" rel="noopener noreferrer" href="https://sspai.com/post/74256">腾讯日历 2.0：焕新的日程管理工具</a></li><li><a
