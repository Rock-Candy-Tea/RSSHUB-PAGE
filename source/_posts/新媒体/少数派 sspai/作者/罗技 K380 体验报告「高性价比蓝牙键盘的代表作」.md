<ul>
<li><strong>设备</strong><ul>
<li>罗技 K380</li>
<li>iPad pro 10.5寸</li>
<li>iPhone 8</li>
<li>Windows 笔记本电脑</li>
</ul>
</li>
<li><strong>使用场景</strong><ul>
<li>生活</li>
<li>课堂</li>
<li>作业</li>
</ul>
</li>
<li><strong>在读大学生</strong></li>
</ul>
<h2 id="ss-2-1524609467980">购买蓝牙键盘的原因</h2>
<ul>
<li>长时间使用 iPad pro 进行学习的过程中日益觉得效率不足，而键盘或许可以给 iPad 带来新生。</li>
<li>如果在只使用 iPhone 进行大段的文本编辑的情况下，那块小屏幕可以让人戳到怀疑人生，这个时候如果有一个蓝牙键盘，或许可以解放辛苦的眼睛和疲惫的手指。</li>
</ul>
<h2 id="ss-2-1524609467980">筛选键盘的过程</h2>
<ol>
<li>支持多设备连接</li>
<li>性价比要高  <s>原装的 smart keyboard 有些累觉不爱</s></li>
<li>手感和颜值兼顾</li>
</ol>
<p>在京东徘徊一段时间后，锁定了今天的主角。<strong>罗技 K380</strong><br></p><figure tabindex="0" class="ss-img-wrapper"><img src="https://i.loli.net/2018/04/24/5adf228c0b923.jpg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="开箱" title="开箱" data-original="https://i.loli.net/2018/04/24/5adf228c0b923.jpg" referrerpolicy="no-referrer"><figcaption class="ss-image-caption">开箱</figcaption></figure><p></p>
<hr>
<h2 id="ss-2-1524609467980">使用报告</h2>
<h3 id="ss-3-1524609467980">单纯键盘的体验</h3>
<p><strong>连接过程</strong></p>
<p>罗技确实不愧是外设的大厂之一，可以在 200 的价位中制作出基本无可挑剔的 K380。</p>
<p><strong>拆开包装->拿出键盘->长按蓝牙匹配键开始匹配->设备中操作->匹配成功。</strong></p>
<p>过程一气呵成，没有阻断感。</p>
<p><s>在 Windows 笔记本中比较奇怪，需要输入设备码，然后再敲 enter ，对于没有屏幕的键盘而言这个操作有些匪夷所思。</s></p>
<p><strong>手感方面</strong></p>
<p>手感这种因人而异的东西，不好以一个量值去衡量。</p>
<p>我就单纯说一下自己的感受吧。</p>
<ul>
<li>优点<ul>
<li>全键盘的长度，使用时没有紧迫感。</li>
<li>几个重要的常用按键做了加长处理，可以盲按。</li>
</ul>
</li>
<li>因人而异的点<ul>
<li>罗技 K380 采用圆键，初上手可能很不适应。</li>
<li>键程略有些短，敲击感不够强烈  <s>不过一个高性价比的蓝牙键盘也确实不应该要求这么多</s></li>
</ul>
</li>
</ul>
<hr>
<h3 id="ss-3-1524609467982">连接到 iPad 使用体验</h3>
<p>首先不得不说，iPad 在连键盘后的确获得新生。在 iPad pro 生产力被夸大的现实中，得到了显著的提升。</p>
<p><strong>日常</strong></p>
<ul>
<li>比较轻的码字活动交给配备了键盘的 iPad</li>
<li>使用 iWork 完成一些资料处理</li>
<li>聊天</li>
</ul>
<p><strong>学习</strong></p>
<ul>
<li>使用 xMind 完成思维导图绘制</li>
<li>使用 Draft 速记</li>
<li>使用 Notability 记录上课笔记，在老师飞速的教案前，只有录音和键盘才能给我带来安全感</li>
</ul>
<p><strong>使用心得</strong></p>
<p>长按 cmd 键出现应用快捷方式提示框。<br>例如 QQ 中</p>
<ul>
<li>cmd+↑ 选择上一个聊天框</li>
<li>cmd+↓ 选择下一个聊天框</li>
<li>cmd+p 发送照片</li>
<li>ctrl+cmd+a QQ内截图</li>
</ul>
<p><strong>关于 Apple Pencil</strong></p>
<p>笔者在购买iPad的同时购置了 Apple Pencil。综合对比两者相对于学习的帮助，只能说<strong>各有千秋</strong>，不相伯仲。</p>
<p>键盘拥有笔远不及的速度，笔有键盘达不到的仪式感。</p>
<p><strong>两者针对不同领域，都为翘楚(键盘以记录为优，笔以阅读为主) </strong></p>
<p>如果您仍在观望，个人建议您可以选择先购入键盘再考虑考虑Pencil。</p>

<h3 id="ss-3-1524609467982">连接到iPhone使用体验</h3>
<p>绝大多数情况下确实使用不到，在此就不多赘述了。</p>
<hr>
<h3 id="ss-3-1524609467982">连接到Windows笔记本电脑使用体验</h3>
<p>可能因为个人原因，或者设备原因，此处只作为参考<br>结论:<strong>用户体验极差！</strong><br>1.连接过程诡异<br>2.首次连接之后就不能正常连接<br><s>3.笔记本有一个键盘干嘛还要连接键盘呢</s></p>
<hr>
<h2 id="ss-2-1524609467982">总结</h2>
<p><strong>物超所值</strong></p>
<p>如果你需要一个蓝牙键盘提高自己的效率，或者为了无线桌面环境添一员大将。罗技K380是一个值得购买的好物。在200的价位确实没有对手。</p>
