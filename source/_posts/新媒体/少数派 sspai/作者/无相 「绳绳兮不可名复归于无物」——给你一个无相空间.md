<div class="articleWidth-content" data-v-e1f0100a><div class="content wangEditor-txt minHeight" data-v-e1f0100a><blockquote>
<p>《老子》:"绳绳兮不可名复归于无物。是谓无状之状无象之象是谓忽恍。"</p>
<p>无相——指没有形迹，没有具体形象、概念。原为道家形容道玄虚无形之语后亦泛指诸种义理的玄微难测或玄微难测的义理。</p>
</blockquote>
<p><a href="https://sspai.com/post/42134" title="熊猫吃短信">熊猫吃短信</a>的作者 baye 最近推出了一个新的浏览器 app 。名为 <a href="https://itunes.apple.com/cn/app/%E6%97%A0%E7%9B%B8-%E4%B8%8D%E4%B8%80%E6%A0%B7%E7%9A%84%E6%B5%8F%E8%A7%88%E5%99%A8/id1331533746?mt=8&ign-mpt=uo%3D4" title="「无相」">「无相」</a>。</p>
<p>这是一个只要经过简单配置，就可以完全掌握于手的<strong>无相空间</strong>。</p>
<p>仿佛一股清流，冲击着传统浏览器固守的最后一处高地。</p>
<hr>
<h1 id="ss-1-1524714044071">应用简介</h1>
<blockquote>
<h3 id="ss-3-1524714044071">摘抄自 App Store 应用介绍页</h3>
<p>无相是一个与众不同的浏览器，它可以将一些静态网站格式化为美观的原生界面，让您在浏览一些网站时感觉就像在使用它们的App一样。可以极大的提高浏览器性能和提升浏览体验。</p>
</blockquote>
<p><strong>无相</strong>，如它名字一般。</p>
<ul>
<li><p>极简，「没有具体形象、概念」。</p>
</li>
<li><p>隐私，「没有形迹」。</p>
</li>
</ul>
<p>这就是<strong>无相</strong>最基本的两个功能。</p>
<h2 id="ss-2-1524714044071">极简</h2>
<p></p><figure tabindex="0" class="ss-img-wrapper"><img src="https://i.loli.net/2018/04/25/5ae08b3c51f70.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="应用打开页" title="应用打开页" data-original="https://i.loli.net/2018/04/25/5ae08b3c51f70.png" referrerpolicy="no-referrer"><figcaption class="ss-image-caption">应用打开页</figcaption></figure><p></p>
<p>在你设置 「相空间」之前，无相极简的就像一只白纸。<strong>充满无限可能</strong>。</p>
<p>这个词也不是随便提及的，打开它的「相空间」，里面就充满着<strong>无限可能</strong>。</p>
<h3 id="ss-3-1524714044071">咱们先加个搜索引擎呗</h3>
<p>轻点搜索栏，就进入了一个新的空间。</p>
<ul>
<li>我们可以进入「相空间」</li>
<li>可以选择搜索引擎</li>
<li>可以快速选择搜索引擎进行搜索</li>
</ul>
<p></p><figure tabindex="0" class="ss-img-wrapper"><img src="https://i.loli.net/2018/04/25/5ae08ff8af851.jpg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="添加过程" title="添加过程" data-original="https://i.loli.net/2018/04/25/5ae08ff8af851.jpg" referrerpolicy="no-referrer"><figcaption class="ss-image-caption">添加过程</figcaption></figure><p></p>
<h3 id="ss-3-1524714044071">点开搜索看看呗</h3>
<blockquote>
<p> 它可以将一些静态网站格式化为美观的原生界面</p>
</blockquote>
<p></p><figure tabindex="0" class="ss-img-wrapper"><img src="https://i.loli.net/2018/04/25/5ae08e44a67b0.jpg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="左边无相、右边safari" title="左边无相、右边safari" data-original="https://i.loli.net/2018/04/25/5ae08e44a67b0.jpg" referrerpolicy="no-referrer"><figcaption class="ss-image-caption">左边无相、右边safari</figcaption></figure><p></p>
<p>无相在此展现了它独一无二的能力——<strong>强大的页面转换能力</strong> </p>
<p>基于页面转换能力，它达到了其他浏览器很难达到的高度</p>
<ul>
<li>去除网页中不重要的元素以及广告，<strong>只展示最重要的信息</strong></li>
<li>为进一步打造<strong>原生 app 界面</strong>特供基础。</li>
<li>只加载 HTML 就可以马上渲染，快人一步，更可以<strong>节约大约4/5的流量</strong></li>
<li>不加载 JS ，保护您的<strong>隐私</strong>。</li>
<li>可以自定义「相」配置，让无相知道如何格式化一个网站。<strong>打造属于你自己的小空间</strong> </li>
</ul>
<h3 id="ss-3-1524714044071">瞅瞅转换的原生页面呗</h3>
<p>基于页面转换能力，在浏览器中使用各个 app 的原生界面。</p>
<p></p><figure tabindex="0" class="ss-img-wrapper"><img src="https://i.loli.net/2018/04/25/5ae0963f93d23.jpg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="左边无相、右边官方 app" title="左边无相、右边官方 app" data-original="https://i.loli.net/2018/04/25/5ae0963f93d23.jpg" referrerpolicy="no-referrer"><figcaption class="ss-image-caption">左边无相、右边官方 app</figcaption></figure><p></p>
<p>乍一看，咦！这不是小程序吗？</p>
<p>非也非也，像也不像。</p>
<p>小程序目前只有微信一个入口（此处不讨论其他厂家的小程序），而浏览器却是世界的入口。</p>
<p>随着无相的发展，将会有越来越多的网站被重新编译，逐渐扩充它的独有生态系统。</p>
<p>或许不能和小程序覆盖方方面面相较。</p>
<p>但在这个基础上，无相可以成为你的 RSS 订阅器，杂志，iBook……</p>
<h3 id="ss-3-1524714044071">再谈谈它的阅读模式~</h3>
<p>无相通过静态网页格式化，可以高度定制网页格式，例如形成 app 原生界面这种操作不在话下。也可以实现更多页面适配阅读模式，还你一个清爽的空间。</p>
<p><strong>它的覆盖范围，可持续发展能力，格式统一化，相较于 PWA ，或许更有优势。</strong></p>
<p></p><figure tabindex="0" class="ss-img-wrapper"><img src="https://i.loli.net/2018/04/26/5ae14948ca6e9.jpg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="依次为：无相阅读界面、safari中原网页、safari阅读模式" title="依次为：无相阅读界面、safari中原网页、safari阅读模式" data-original="https://i.loli.net/2018/04/26/5ae14948ca6e9.jpg" referrerpolicy="no-referrer"><figcaption class="ss-image-caption">依次为：无相阅读界面、safari中原网页、safari阅读模式</figcaption></figure><p></p>
<h3 id="ss-3-1524714044071">相的秘密</h3>
<p>无相如此神奇的一个秘密，或许就在于它是可编程的。</p>
<p>对于互联网玩家来说，可编程，可自定义，这简直就是理想的利器呀。</p>
<p><strong>而无相也提供了这个功能</strong></p>
<p>由于关于网页开发知识贫瘠，不发表个人意见。</p>
<p>详情请见 <a href="https://github.com/waylybaye/docs.wuxiang.io" title="官方文档">官方文档</a></p>
<p> <a href="https://wuxiang.io/dev#/" title="编辑页">编辑页</a></p>
<p></p><figure tabindex="0" class="ss-img-wrapper"><img src="https://i.loli.net/2018/04/26/5ae140d397d33.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1" alt="编辑页简图" title="编辑页简图" data-original="https://i.loli.net/2018/04/26/5ae140d397d33.png" referrerpolicy="no-referrer"><figcaption class="ss-image-caption">编辑页简图</figcaption></figure><p></p>
<h2 id="ss-2-1524714044071">使用体验</h2>
<blockquote>
<p>我绝对低估了它。如果您现在感觉不如您意，请麻烦给它一些发展的时间。</p>
</blockquote>
<p>经过一段时间体验，以个人感受做个总结。</p>
<p>优点</p>
<ul>
<li>快！</li>
<li>极简却不失功能。</li>
<li>交互逻辑合理，舒适。</li>
</ul>
<p>缺点</p>
<ul>
<li>部分页面没有适配极简相风格页面（作者提及到的小说转iBook，也只有部分匹配了此功能）。</li>
<li>在英语环境中，中文和英文交叉显示，有些了破坏一体性。</li>
<li>暂时没有从safari快速转移到无相的方法（例如利用分享页面跳转到无相，也比手动复制粘贴来的好）。</li>
</ul>
<hr>
<p>无相，我还理解成无我之相。让自己沉浸到一个无我的世界，去慢慢感受它的静谧~</p>
