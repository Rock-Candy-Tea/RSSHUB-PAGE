<div>   
早些时候，英特尔在 Hot Chips 34 大会期间，介绍了与下一代 CPU 有关的一些关键细节。<strong>可知 14 代 Meteor Lake、15 代 Arrow Lake 和 16 代 Lunar Lake 将采用的 3D Foveros 封装技术，很有乐高积木的风格。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0824/4085d7b84c4282f.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/4085d7b84c4282f.png" alt="34.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（via <a href="https://wccftech.com/intel-lego-like-chiplet-design-for-next-gen-meteor-lake-arrow-lake-lunar-lake-3d-foveros-cpus-detailed/" target="_self">WCCFTech</a>）</p><p><strong>据悉，Foveros 是一种先进的芯片间封装技术，并且细分为三种。</strong></p><blockquote><p>● 首先是适用于量产的 Foveros 标准设计，其凸点间距在 50-25 μm，密度 > 400-1600/m㎡，功率 0.156 pJ/bit 。</p><p>● 其次是 Foveros Omni 设计，其旨在匹配连接基础芯片复合体中的瓦片，提供高达 EMIB 四倍的互连凸点密度 —— 凸点间距 25 μm、密度 1600 / m㎡、功率 < 0.15 pJ/bit 。</p><p>● 然后是 Foveros Direct 设计，其提供 16 倍互连密度，具有更低延迟、更高带宽、以及更低的功率 / 裸片要求 —— 凸点间距 < 10 μm、密度  > 10000 m㎡、功率 < 0.05 pJ/bit 。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0824/13a9b9f88b5861d.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/13a9b9f88b5861d.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>在 12 代 Alder Lake 与 13 代 Raptor Lake CPU 率先采用了混合式核心架构设计之后，英特尔还计划利用 3D Foveros 封装迎接多芯片时代。</p><p><strong>14 代 Meteor Lake、15 代 Meteor Lake 和 16 代 Lunar Lake CPU 的亮点有：</strong></p><blockquote><p>● Intel 下一代 3D 客户端平台；</p><p>● 具有 CPU、GPU、SOC 和 IO 块的分解式结构；</p><p>● 采用 Foveros 互连的 Meteor / Arrow Lake 基础块；</p><p>● 拥抱开放式的 UCIe 通用小芯片互连生态系统。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0824/e0df95fbe0b13d6.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/e0df95fbe0b13d6.png" alt="8.png" referrerpolicy="no-referrer"></a></p><p>先聊聊 Meteor Lake，该公司展示了一种全新的芯片布局，让我们更好地了解了具有各种 IP 块的小芯片，主要拆分为 CPU、图形块、片上系统、以及 IOE 块。</p><p>其中主 CPU 块将采用 Intel 4（7nm EUV）工艺节点，而 SoC 与 IOE 小芯片采用了台积电 6nm 工艺（N6）制造。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/4a316515176b8bd.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/4a316515176b8bd.png" alt="12.png" referrerpolicy="no-referrer"></a></p><p>英特尔宣称 Meteor Lake 是其迈入小芯片生态的第一步，但据业内人士所述，情况并非如此，称 Meteor Lake 的 tGPU 一直定的台积电 5nm（N5）设计。</p><p>计算图块可在各种核心数量、世代、节点与缓存之间完全扩展，且英特尔不仅能够在 Foveros 3D 封装 CPU（如 Meteor Lake）中混搭不同核心架构、还可向上或向下扩展至不同节点。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/560a53800a3329d.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/560a53800a3329d.png" alt="13.png" referrerpolicy="no-referrer"></a></p><p>图形块也是如此，英特尔可根据不同的核心、节点和缓存规模进行缩放。图表中的 tGPU 从 4~12 个 Xe 内核（64~192 EU）不等，但同一照片似乎也展示了 8 Xe / 128 EU 的版本。</p><p>片上系统块亦可根据 SKU 放大或缩小，主要模块是低功耗 IP（特指 VPU）、SRAM、IO 和可扩展电压设计。最后是 I/O 扩展块，其在通道数量、带宽、协议和速率等方面，都是完全可扩展的。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/a7d2dbe7bfa70ad.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/a7d2dbe7bfa70ad.png" alt="14.png" referrerpolicy="no-referrer"></a></p><p>接着，英特尔展示了如何将各 IP 块排列到一起。金属化的顶背，也是 Foveros 无源芯片所在的位置，正下方就是上述各个小芯片块。</p><p>这些瓦片之间通过 36 μm 间距（芯片-芯片）互连方案，与基础块实现连接。Base Tile 带有大电容，辅以 IO / 供电 / D2D 路径的金属层。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/90da8411c3585d7.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/90da8411c3585d7.png" alt="15.png" referrerpolicy="no-referrer"></a></p><p>每个金属层都是模块化的设计，带有用于逻辑 / 存储器的有源硅片。顶部和底部具有用于顶 / 底层互连的封装凸点。</p><p>示例中的移动芯片，采用了 6P+8E 的混合式架构设计。另有 CPU / IOE 块，且通向 SOC 块的图形块之间有两个 Die-To-Die 连接。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/0bd33e009ab3a6c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/0bd33e009ab3a6c.png" alt="21.png" referrerpolicy="no-referrer"></a></p><p>作为 Foveros 3D 封装的一部分，英特尔表示在主小芯片顶部有一个无源中介层，其基于英特尔自家的 22nm（FFL）工艺制造。</p><p>现阶段它并没有被分配任何用途，Meteor Lake CPU 也暂时用不到 EMIB 技术。但该公司计划在未来使用更先进的封装技术，并将之换成有源小芯片。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/a63996e3a873554.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/a63996e3a873554.png" alt="24.png" referrerpolicy="no-referrer"></a></p><p><strong>Foveros 芯片互连（FDI）的技术特点如下：</strong></p><blockquote><p>● 低压 CMOS 接口；</p><p>● 高带宽、低延迟；</p><p>● 支持同步与异步信令；</p><p>● 低面积开销；</p><p>● 2 GHz 主频下的操作功耗在 0.15-0.3 pJ/bit 左右。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0824/ded238c116e00f9.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/ded238c116e00f9.png" alt="30.png" referrerpolicy="no-referrer"></a></p><p>CPU 与 SOC 之间的互连，具有大约 2K（2×IDI）的主带宽，图形与 SOC 块也具有大约 2K（2×iCXL）的互连主带宽，而 SOC 和 IOE 块则在 1K（IOSF，4×DisplayPort）左右。</p><p>不过 Meteor Lake CPU 的另一关键改进，就是提升了最大睿频性能。得益于 Intel 4 工艺的协同优化，其有望带来较 Alder Lake CPU 更高的睿频潜力，且 Base Tile 的总电流达到了 500 。</p><p>I/O 能力方面，英特尔搬出了 Haswell 来比较。成本方面，随着新一代晶圆成本的迭代上涨，开发单芯片将变得更加不换算。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/b4830e48f8d2a67.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/b4830e48f8d2a67.png" alt="31.png" referrerpolicy="no-referrer"></a></p><p>若英特尔坚持按老方法造新芯片，其实也不是不可以。毕竟新制程节点不是摆设，只是难以将成本压低到合理水平。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/a7a72b9a243b244.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/a7a72b9a243b244.png" alt="32.png" referrerpolicy="no-referrer"></a></p><p>该公司称，与单片式芯片制造方案相比，Meteor Lake 等分解式设计可提供更高的晶体管性能，以及跨各种工艺节点的更好 IP 刷新速度和更高能效。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/fdd97717e679c8c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/fdd97717e679c8c.png" alt="33.png" referrerpolicy="no-referrer"></a></p><p>英特尔透露，其 Meteor Lake SKU 的功耗设计从 <10W 到 >100W 不等，但采用小芯片设计的 CPU 性能仍与单片式设计相当。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/7d18c58d654a250.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/7d18c58d654a250.png" alt="35.png" referrerpolicy="no-referrer"></a></p><p>此外该公司称，14 代 Meteor Lake / 15 代 Arrow Lake CPU，确实正在走向桌面和移动平台。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/242b0a80406745c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/242b0a80406745c.png" alt="36.png" referrerpolicy="no-referrer"></a></p><p>按照计划，英特尔计划在 2023 / 2024 年发布 Meteor / Arrow Lake CPU，且两者都采用下一代 LGA 1851 插槽。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/01ef5075c5f2cfb.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/01ef5075c5f2cfb.png" alt="37.png" referrerpolicy="no-referrer"></a></p><p>至于 16 代 Lunar Lake CPU，据说该系列最初瞄向 15W 低功耗移动 CPU 细分市场，但鉴于距离正式上市还有几年时间，我们仍不能排除未来有生变的可能。</p><p><a href="https://static.cnbetacdn.com/article/2022/0824/75aaac2a923fc24.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0824/75aaac2a923fc24.png" alt="38.png" referrerpolicy="no-referrer"></a></p><p>最后，这并不是我们首次看到英特尔在移动 / 部分桌面 SKU 之间划出明确界限。除了历史上的 Broadwell，近些年的 Ice Lake / Tiger Lake 系列也是如此。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a
