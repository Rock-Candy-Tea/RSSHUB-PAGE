早在 6 月，就有人发现微软正在测试一款类似 Spotlight 的启动器，用以在 Edge 浏览器中快速执行相关命令。<strong>近日，有眼尖的人们在 Canary 分支中发现，该功能正以开发人员工具中的“命令面板”的名义推出。</strong>Neowin 指出，DevTools Command Palatte 已在 Microsoft Edge Canary 105 版本中可用。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0720/49ccce62694b70b.png" alt="0.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Edge <a href="https://techcommunity.microsoft.com/t5/articles/devtools-command-palette-now-available-for-testing-in-canary/m-p/3576618" target="_self">Insider</a>）</p><p>微软表示，Command Palatte 旨在帮助用户更轻松地访问 Microsoft Edge 的浏览器管理和开发人员工具。感兴趣的朋友，可通过 Ctrl+Shift+Space 组合键来调用。</p><p><a href="https://static.cnbetacdn.com/article/2022/0720/e179ddee7852f0a.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0720/e179ddee7852f0a.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>通过该快捷方式，你将能够访问选项卡（Tabs）、书签（Bookmarks）和收藏管理（Collection Management）选项，甚至召唤其它开发人员工具（比如模拟一款移动设备）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0720/3ca286aac206059.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0720/3ca286aac206059.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>以下是 DevTools 中 Command Palatte 的启用方法：</p><blockquote><p>（1）点击多功能地址栏，输入 edge://flags 并跳转。</p><p>（2）在搜索标记文本字段，输入 DevTools Toaster 或 Command Palette 关键词。</p><p>（3）在 DevTools Toaster 或 Command Palette 下拉菜单中，选择 Enabled 启用。</p><p>（4）点击右下角出现的‘重新启动’按钮，以刷新浏览器设置。</p><p>（5）此时便可在 Microsoft Edge 浏览器中，使用 Ctrl+Shift+Space 组合键打开 Command Palatte 命令面板。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0720/29938e2eee86572.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0720/29938e2eee86572.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>启用后，你可输入“选项卡”（Tabs）来管理标签页，或输入“书签”来管理书签收藏。</p><p>另外用户可以点击 ↑ ↓ 箭头以选择命令，然后按回车键（Enter）确认。以及点击工具栏底部的 →，来访问开发人员选项。</p><p>最后，感兴趣的朋友，也可移步至微软官网（<a href="https://docs.microsoft.com/en-us/microsoft-edge/devtools-guide-chromium/experimental-features/edge-command-palette" target="_self">传送门</a>），以了解该工具的完整功能。</p>
