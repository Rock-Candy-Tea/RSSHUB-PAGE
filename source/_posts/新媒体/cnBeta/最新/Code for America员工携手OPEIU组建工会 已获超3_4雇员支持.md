Code for America 员工今日宣布，他们将与办公室和专业雇员国际工会（OPEIU）的 Local 1010 共同组建一支工会。<strong>组织者表示，已有 62 人签署了授权，意味着 77% 符合资格的员工对这项工作表示了支持。</strong>据悉，这家全国性的非盈利组织，致力于为政府机构打造科技产品，包括帮助县法院在符合条件后自动清除当事人记录、以及运营加州食品券的最大接收点之一。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0805/e8c9ae2f319e7b5.jpg" referrerpolicy="no-referrer"></p><p>员工们表示，他们对这项工作基本上感到满意，且薪酬较其它非盈利组织的同事更高。此外他们认为，工会将使之在商业决策中拥有更大的发言权。</p><p>CFA 员工兼组织者 Ash Campo 指出：“工会为独立贡献者提供了一个席位，加入工会使我们能够重新思考员工与管理层之间的隔阂，以便正式员工能够在影响他们、且让组织也受益的决策中拥有根据意义的发言权”。</p><blockquote><p>据悉，本次公告也是更大的白领工人工会运动的一部分。2020 年的时候，Kickstarter 工人与 OPEIU 成立了首批由科技行业全职白领工人组成的工会之一。</p><p>今年早些时候，Glitch 员工也签署了这家科技公司的第一份集体谈判协议。至于 CFA 的组织工作，其实始于将近一年之前。</p><p>起初一些员工对此持怀疑态度，指出该组织已就政策决策向员工提供了意见，但组织者认为这还不够。</p></blockquote><p>软件工程师兼工会组织者 Ben Calegari 解释称：“人们普遍对事情的进展方式感到满意，并谈到了管理的协作性质。但我认为的一个问题是，其依赖与管理者的的关系来创造变革”。</p><p>另一位要求匿名的工会组织者也赞同这一点，表示工会给予了他们集体的力量，但现在他们必须依靠个人力量来改变一些事情。</p><p>员工称，其计划倡导更好的薪酬与晋升政策，并有望在董事会中占有一席之地。Calegari 补充道：</p><blockquote><p>作为非营利组织的工作人员，我们有机会较其他人赚得更高、可用资源也更多。在工人的认可与投入下，将没有人可以不顾尊严地工作。</p><p>其实有许多人的情况比我们更加糟糕，但它们没有像我们一样拥有自由行动和说话的能力。正因如此，我们才更有责任将我们的自由付诸于集体利益。</p></blockquote><p>Ash Campo 指出，工会是 CFA 充分实现其目标的一个重要机遇：“我们的使命和准则是为大众创造技术，而加入工会就是在内部实现这一使命。我们希望提出由民众为了他人而制定的政策，而工会就是让我们在这张桌子上有一席之地的唯一途径”。</p><p>周一的时候，工会已要求 Code for America 管理层接受这一点。OPEIU 主要组织者 Grace Reckers 表示，他们有信心顺利推进这项工作。但若遇阻，他们其实也已经做好了相应的准备。</p>
