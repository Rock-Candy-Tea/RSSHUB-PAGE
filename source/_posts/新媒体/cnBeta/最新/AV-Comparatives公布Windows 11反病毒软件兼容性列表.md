尽管对于大多数 Windows 用户来说，系统自带的 Microsoft Defender 安全软件已经足够应付日常使用。但随着 Windows 11 即将于 10 月 5 日正式发布，<strong>独立安全测试实验室 AV-Comparatives（简称 AVC）检查并发布了适用于 Windows 11 的消费级反病毒软件名单。</strong><br><br>
<p><img src="https://static.cnbetacdn.com/article/2021/0929/3e8a078e0f8f8bb.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://techdows.com/2021/09/windows-11-compatible-antivirus.html" target="_self">Techdows</a>）</p><p>作为微软打造的最新一代桌面操作系统，<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 引入了居中放置的“开始”菜单、任务栏、靓丽的现代 UI + Mica 设计元素、情境（上下文）菜单，且文件资源管理器也迎来了重新设计。</p><p>硬件方面，想要正式用上 Windows 11 操作系统，用户需检查设备是否基于 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 2000 / <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 8 代处理器或更高版本，且可信任平台模块（TPM 2.0）也成为了一项强制性要求 —— 甚至虚拟机也无法豁免。</p><p>如果你的设备不满足上述条件，即使能够通过“旁门左道”（比如 ISO 镜像 / 媒体创建工具）运行 Windows 11，微软也不保证后续的相关支持和安全更新。</p><p>此外需要指出的是，无论你使用了那种反病毒软件，Windows Update 自动更新都将是强制性的。</p><p>微软最近表示，其已在 Windows 11 上测试了数千款应用程序。不出所料的是，大多数可在 Windows 10 上运行的应用程序，都能够在 Windows 11 上良好运行。</p><p><img src="https://static.cnbetacdn.com/article/2021/0929/11d3116fd34befb.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://www.av-comparatives.org/av-comparatives-releases-list-of-working-consumer-av-programs-for-windows-11/" target="_self">AV-Comparatives</a>）</p><p>上月，AVC 表示流行的反病毒软件在 Windows 11 操作系统中表现良好啊。现在，该实验室又在 Build 22454.1000 内部编译版本上，<strong>分享了 17 款与之兼容的消费级反病毒软件列表。</strong></p><blockquote><p>● Avast Free Antivirus 21.7.2481</p><p>● AVG Free Antivirus 21.8.3202</p><p>● Avira Antivirus Pro 1.1.54.22911</p><p>● Bitdefender Internet Security 25.0.26.89</p><p>● ESET Internet Security 14.2.24.0</p><p>● G Data Total Security 25.5.11.316</p><p>● K7 Total Security 16.0.0556</p><p>● Kaspersky Internet Security 21.3.10.391</p><p>● Microsoft Defender Antivirus, as in Build 22454.1000 of Windows 11</p><p>● Malwarebytes Premium 4.4.6</p><p>● McAfee Total Protection 16.0</p><p>● Norton LifeLock Norton 360 22.21.8.62</p><p>● Panda Free Antivirus 21.00.00</p><p>● Total AV Total Security 5.15.69</p><p>● Total Defense Essential Antivirus 13.0.0.545</p><p>● Trend Micro Internet Security 17.0.1181</p><p>● VIPRE Advanced Security 11.0.6.22</p></blockquote><p><strong>而 AVC 定下的兼容性测试基准，则包含了以下几点：</strong></p><blockquote><p>● 无需专业知识或解决方案，即可顺利完成安全。</p><p>● 无需用户干预，即可激活实时保护。</p><p>● 与 Windows 安全性良好集成。</p><p>● 可（自动或手动）成功更新恶意软件签名。</p><p>● 可在实时防护被禁用时发出警告，并且能够轻松重新激活。</p><p>● 提供至少与 Windows 10 平台上相当的恶意软件检测能力。</p><p>● 可在遇到恶意软件时采取适当的阻止措施，确保系统能够受到保护。</p><p>● 没有误报或疏漏。</p><p>● 可彻底卸载并清理其在 Windows 安全中心中留下的条目。</p></blockquote>
