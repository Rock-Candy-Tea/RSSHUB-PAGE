早在 2019 年，微软就趁着 .dev 顶级域名开放注册时，申请了 vscode.dev、并将之重定向到 code.visualstudio.com 。<strong>然而直到本周三，微软才正式发布了可完全在浏览器中运行的轻量级 VS Code 。</strong>这意味着广大开发者无需任何安装，即可在基于浏览器的开发环境中、打开本地机器上的一个文件夹并开始工作。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1021/3e04ce0d7881a05.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1021/3e04ce0d7881a05.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：VS Code <a href="https://code.visualstudio.com/blogs/2021/10/20/vscode-dev" target="_self">官网</a>）</p><p>据悉，VS Code for the Web 主要提供了以下体验。首先时结合浏览器工具，构建客户端 HTML、JavaScript 和 CSS 应用程序并进行调试。</p><p><a href="https://static.cnbetacdn.com/article/2021/1021/870d45c85110697.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1021/870d45c85110697.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>其次可在 Chromebook 等低性能设备上轻松编程，而无需安装完整版 VS Code 集成开发环境。</p><p><img src="https://static.cnbetacdn.com/article/2021/1021/3cb1ad602c8b926.png" alt="3.png" referrerpolicy="no-referrer"></p><p>此外你也可以在 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a> 上做开发，支持上传 / 下载文件（甚至使用 Files 应用程序将之存储在云端），以及使用内置 GitHub 存储库扩展，远程打开存储库。</p><p><a href="https://static.cnbetacdn.com/article/2021/1021/41ecfe1c2620001.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1021/41ecfe1c2620001.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p>值得一提的是，VS Code for the Web 还内置了存储库（GitHub Repositories）、代码空间（Codespaces）和查询请求（Pull Request）扩展。</p><p><a href="https://static.cnbetacdn.com/article/2021/1021/41bab07aaca913c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1021/41bab07aaca913c.png" alt="5.png" referrerpolicy="no-referrer"></a></p><p>至于 VS Code for the Web 与 GitHub.dev 在线编辑器的不同之处，主要是 github.dev 为深度集成到 GitHub 中的 Web 版 VS Code 定制实例。</p><p><a href="https://static.cnbetacdn.com/article/2021/1021/92633d0b583b304.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1021/92633d0b583b304.png" alt="6.png" referrerpolicy="no-referrer"></a></p><p>github.dev 可以自动登录，且网址（URL）遵循该代码托管网站的 /organization/repo 格式。</p><p><a href="https://static.cnbetacdn.com/article/2021/1021/8a8cd19da3152ec.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1021/8a8cd19da3152ec.png" alt="7.png" referrerpolicy="no-referrer"></a></p><p>允许用户简单地改变 .com 或 .dev 域名后缀来编辑某个 repo，辅以 GitHub 定制的浅色与暗色主体。</p><p><a href="https://static.cnbetacdn.com/article/2021/1021/e3182efc7218ced.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1021/e3182efc7218ced.png" alt="8.png" referrerpolicy="no-referrer"></a></p><p>最后，除了 GitHub 上的存储库，VS Code for the Web 还支持 Azure Repos（作为 Azure DevOps 的一部分）。</p><p><a href="https://static.cnbetacdn.com/article/2021/1021/079dc472d11208f.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1021/079dc472d11208f.png" alt="9.png" referrerpolicy="no-referrer"></a></p>
