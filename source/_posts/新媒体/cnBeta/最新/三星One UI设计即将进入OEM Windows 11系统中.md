和Android手机品牌不同，PC厂商不能随意调整Windows操作系统的界面设计，不过这种局面很快就会有所改变。10月30日消息，据XDA报道，在今年的三星开发者大会上，<strong>三星宣布与微软的合作进一步深入，三星笔记本用户可以在PC端体验到One UI的设计风格，以此来保证手机和PC端的体验一致性。</strong><br>
 <p>据悉，三星很快会在自家笔记本电脑上带去One UI的设计。和<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11系统相比，经过三星调整的Windows系统主界面风格更像One UI，<strong>三星称之为“One UI Book 4”。</strong>PS：三星最新版<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>操作系统命名为One UI 4。</p><p>这次三星Galaxy Book Pro、Galaxy Book 360、Galaxy Book Flex2、Galaxy Book Odyssey等Galaxy Book笔记本上都会体验到这一全新的操作系统。</p><p style="text-align: center;"><a href="https://img1.mydrivers.com/img/20211030/cf1ac3a8d2264b81a67fbf38aec3e46f.png" target="_blank"></a><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1030/b794ae2d3170c97.png"><img data-original="https://static.cnbetacdn.com/article/2021/1030/b794ae2d3170c97.png" src="https://static.cnbetacdn.com/thumb/article/2021/1030/b794ae2d3170c97.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><a href="https://img1.mydrivers.com/img/20211030/1969ca97cf734d149679dace2ccfebaa.jpg" target="_blank"></a><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1030/17a47080ee30b11.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1030/17a47080ee30b11.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1030/17a47080ee30b11.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">三星One UI Book 4</p>
