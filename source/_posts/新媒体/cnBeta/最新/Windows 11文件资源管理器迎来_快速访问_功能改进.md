在上一个 Windows 版本中，文件资源管理器（Explorer）并没有迎来大刀阔斧的改革。时至今日，广大 Windows 用户仍在等待文件资源管理器的“选项卡”支持。<strong>即便如此，微软还是有着眼于细小处的改进，以带来更现代的体验 —— 比如 Windows 11 Build 22557 预览版本中的快速访问支持。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0217/284b1fe8e6763c5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0217/284b1fe8e6763c5.jpg" referrerpolicy="no-referrer"></a></p><p>与旧版 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 操作系统一样，从 Build 22557 预览版本开始，Windows 11 的快速访问还支持文件、而不仅仅是文件夹。</p><p><strong>换言之，如果你需要将特定文件固定到快速访问，只需右键单击即可轻松完成，就像尝试固定文件夹时那样。</strong></p><blockquote><p>微软写道，该公司正在对文件资源管理器的快速访问视图进行刷新。</p><p>首先是此前仅支持文件夹的‘固定到快速访问’功能，现也支持特定的文件。</p><p>固定后的文件，会显示在快速访问中‘最近文件’上方的新部分中。</p></blockquote><p><strong>此外微软计划为快速访问引入对 <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a>.com 文件的固定支持，</strong>以便登录了 Microsoft 账户的用户可以获得更直接方便的体验。</p><blockquote><p>接下来，使用 Microsoft Account 主账号 / 辅助账号登陆 Windows 的工作和<a data-link="1" href="https://c.duomai.com/track.php?k=yUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmYiJ05WZkVHdzZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyUiR" target="_blank">教育</a>客户，将能够固定 Office.com 中的文件，且最近文件也会显示在快速访问中。</p><p>在 OneDrive、SharePoint 和 Teams 中固定 / 取消固定的文件，其更改也将同步反映在 Office.com 和 Office 应用程序中。</p><p>来自协作者的文件活动更新（例如编辑和评论），还可显示在瓷贴（Tiles）和详情（Details）视图中。</p></blockquote><p>需要指出的是，目前上述“快速访问”改进仅适用于开发通道的 Insider 测试者们。</p><p>微软尚未提供任何有关此功能何时可在生产设备上启用的预估到达时间，但若一切顺利，其很有望作为即将到来的操作系统更新的一部分，而与广大 Windows 用户正式见面。</p>
