<strong>微软刚刚宣布了针对 Web 开发的一些改进，其中最吸引我们的，莫过于 Microsoft Edge 的 3D 视图工具。</strong>对于开发者来，这有助于他们更轻松高效地发现、解决并改进网站上的诸多元素。在解决常见 Web 开发挑战时 —— 比如 DOM 复杂性、不需要滚动条、或 z-index 堆叠等问题 —— 3D 可视化工具都会派上极大的用场。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0622/37959e634e14ae7.webp" alt="1.webp" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Edge <a href="https://blogs.windows.com/msedgedev/2022/06/21/debug-the-web-in-3d-with-the-3d-view-tool/" target="_self">Blog</a>）</p><p>虽然 3D 视图等 DevTools 并不适合普通网络用户，但<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还是在努力给 Web 开发者带来新功能和相关体验改进。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=358834074&autoplay=false&disablePlaylist=true" width="720" height="480" frameborder="0"></iframe></p><p style="text-align: center;">Debug the web in 3D - Microsoft Edge（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzU4ODM0MDc0LnNodG1s.html" target="_self">via</a>）</p><p>最新加入的功能包括快速查看页面复杂度、调试 z-index 堆叠问题、以及调试复合层的性能问题。</p><p><img src="https://static.cnbetacdn.com/article/2022/0622/40888e40902b254.webp" alt="2.webp" referrerpolicy="no-referrer"></p><p>如果你想要在深度嵌套的容器中展开快速查找，3D 视图工具显然可以提供最佳体验。</p><p><img src="https://static.cnbetacdn.com/article/2022/0622/1344f4a2a4b6475.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>其 DOM 选项卡中能够快速展示三维树，并帮助 Web 开发者发现页面的哪部分可能存在过多的嵌套。</p><p><img src="https://static.cnbetacdn.com/article/2022/0622/7a5c41d140141f6.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>DOM 选项卡还可为开发者提供网页的整体 3D 视图，允许缩放平移，直到超出视口的元素也变得可见。只需点击其中一个元素，便可直接跳转。</p><p><img src="https://static.cnbetacdn.com/article/2022/0622/5c29ee93900c0ef.jpg" alt="5.jpg" referrerpolicy="no-referrer"></p><p>此外 z-index 选项卡能够让调查更加简洁直观，该工具允许查看哪些元素正在堆叠上下文、以及元素如何沿着 z-index 轴堆叠，有助于快速定位常见问题。</p><p><img src="https://static.cnbetacdn.com/article/2022/0622/e3634fe89933566.jpg" alt="6.jpg" referrerpolicy="no-referrer"></p><p>接着是调试复合层的性能问题，这项新功能支持以适当或优化的层数来分解网页、进而提升 Web 性能体验 —— 尤其当某些组件独立于其它组件，来处理动画或变更时。</p><p><img src="https://static.cnbetacdn.com/article/2022/0622/15120a5bb832263.jpg" alt="7.jpg" referrerpolicy="no-referrer"></p><p>最后，微软添加了一个新的复合层选项卡，以便开发者在左侧栏查看已创建的图层列表。当鼠标悬停在每个图层上时，就会在 3D 视图中高亮显示重要信息，比如渲染图层所需的内存大小。</p>
