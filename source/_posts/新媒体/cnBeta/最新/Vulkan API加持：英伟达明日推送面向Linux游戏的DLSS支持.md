<div>   
<strong>本月初，英伟达宣布了要将 GeForce RTX 系列显卡的“深度学习超级采样”（简称 DLSS）技术引入 Linux / Steam Play 平台，而现在正是时候。</strong>即使当前尚无任何原生支持 DLSS 的 Linux 游戏，但该项目主要是给基于 Steam Play（Proton 项目）的游戏提供尝鲜，后者旨在让 Windows 游戏在 Linux 上顺畅运行。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0622/a2b0b2460530dde.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0622/a2b0b2460530dde.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：Nvidia <a href="https://www.nvidia.com/en-gb/geforce/news/june-2021-rtx-dlss-game-update/" target="_self">官网</a>）</p><p>在即将推出的最新版 Linux 驱动程序中，英伟达将通过 Vulkan API 来实现对 Linux 游戏的 DLSS 支持。</p><p><a href="https://static.cnbetacdn.com/article/2021/0622/7ead1634ea90d50.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0622/7ead1634ea90d50.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>英伟达指出，对采用 Vulkan API 的 DLSS 游戏支持，将于本周二正式上线。</p><p><a href="https://static.cnbetacdn.com/article/2021/0622/c36365ddfb2a2d1.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0622/c36365ddfb2a2d1.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>届时 Linux 平台玩家也将能够充分利用 GeForce RTX GPU 专用的 AI Tensor Core 核心来加速《毁灭战士：永恒》、《无人深空》、以及《德军总部：新血脉》等游戏的帧率。</p><p><a href="https://static.cnbetacdn.com/article/2021/0622/02bab23d89dbd22.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0622/02bab23d89dbd22.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>不过要让基于 Direct3D 的 DLSS 游戏在 Steam Play 下调用 DXVK / VKD3D-Proton 功能，所需部分仍要等到今秋才能准备就绪。</p><p style="text-align: center;"><iframe width="640" height="480" src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=266535986&autoplay=false&disablePlaylist=true" frameborder="0"></iframe></p><p style="text-align: center;">NVIDIA DLSS vs Traditional Upscaling（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMjY2NTM1OTg2LnNodG1s.html" target="_self">via</a>）</p><p>最后，除了对 Linux 平台的 DLSS 测试版驱动支持，470 系列驱动还有望改善对 Wayland 的支持。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://hot.cnbeta.com/articles/game/1137009.htm" target="_blank">英伟达携手Valve将DLSS技术引入Linux</a></p></div>
