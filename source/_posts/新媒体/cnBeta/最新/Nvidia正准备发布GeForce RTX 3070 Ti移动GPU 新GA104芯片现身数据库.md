消息称 NVIDIA 正为笔记本市场开发两款新的移动 GPU，其中一款就是最近传闻中的 GeForce RTX 3070 Ti。<strong>近日网友在 PCI Device ID 数据库中发现了一个新的 GA104 芯片</strong>。
根据网友 Faintsnow 爆料评论，科技媒体 VideoCardz 在  PCI Device ID 数据库上发现了一个特别的 ID-- ID 24A0，指的是 NVIDIA 尚未发布的移动 GPU--NVIDIA GN20-E6。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1104/ca2c1585c1d170a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1104/ca2c1585c1d170a.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1104/7c3a23db7ee78c4.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">据了解，GN20-E6 是 NVIDIA GeForce RTX 3070 Ti 笔记本 GPU 的名称。目前，英伟达笔记本 GPU 并未与NVIDIA Ampere 系列的其余产品一起上市。</p><p style="text-align: left;">如果这一信息属实，那么英伟达开始考虑为移动设备（如笔记本电脑和笔记本）制造RTX 3080 Ti和RTX 3070 Ti GPU，这些GPU将提供比非Ti型号更多的核心数。</p><p style="text-align: left;">据推测，RTX 3080 Ti 将使用较新的图形处理器 GA103，该处理器已被展示为拥有高达 7680 个 CUDA 核心。这种特殊图形处理器的移动版本更有可能产生比桌面版本更少的 CUDA 核心。然而，NVIDIA RTX 3070 Ti 可能会高于非 Ti 型号，后者有高达 5120 个 CUDA 核心，因为目前领先的 NVIDIA SKU 提供 6144 个 CUDA 核心。</p>
