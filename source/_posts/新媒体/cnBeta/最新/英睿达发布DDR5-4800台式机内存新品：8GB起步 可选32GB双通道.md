<strong>美光科技（Micron）刚刚发布了英睿达（Crucial）系列 DDR5 台式内存新品，并且可选单条 8GB / 16GB / 32GB、以及容量 × 2 的双通道套装。</strong>据悉，与 DDR4 DRAM 相比，DDR5 内存具有更高的电源效率、标准工作电压降低到了 1.1V，且存储密度也达到了上一代的四倍。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1028/9b7a23e73308847.png" alt="1.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">DDR5 DRAM 将 PMIC 电源管理芯片，从主板端转移到了内存 PCB 上。</p><p>首批亮相的 DDR5 台式内存新品，普遍采用了 JEDEC 标准的 DDR5-4800 传输速率（较上一代 DDR4 DRAM 提升 50%），能够为主流 PC 用户提供发烧级别的性能体验。</p><p>美光表示，得益于内存技术的进步，DDR5 可为每个 CPU 核心提供更高的带宽（有效提升近一倍），同时满足多核 CPU 的多任务处理和其它计算密集型 PC 应用程序的高速数据传输需求。</p><p><img src="https://static.cnbetacdn.com/article/2021/1028/01a9585cf4d4a5f.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">DDR5-4800 vs DDR4-3200</p><p>无论是商务、学习、创意、互动娱乐、还是基础的个人应用，DDR5 都可兼顾效能与灵活性。这为支持 DDR5 内存的系统，带来了空间和面向未来的可扩展性。</p><p>结合改进的总线效率（更高的有效带宽）、更低的能耗（1.1V）、以及更高的存储密度，DDR5 内存平台还可进一步增强新型 PC 应用的性能 —— 比如 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_110018%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">4K</a> / 8K 内容创建、交互式娱乐、个人 / 企业生产力、以及虚拟现实等体验。</p><p><img src="https://static.cnbetacdn.com/article/2021/1028/22d09695db69f9d.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">严格的兼容性测试保障</p><p>通过为饥饿的多核心 CPU 提供支持，DDR5 内存还有助于在不影响系统性能的情况下，大幅提升多任务处理的效率。</p><p>兼容性方面，美光宣称已同各大主机板制造商达成了合作，支持<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>最新推出的 12 代 Alder Lake 台式处理器、并且具有预定义的 XMP 配置文件。</p><p><img src="https://static.cnbetacdn.com/article/2021/1028/fa5ca08b16535b8.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">合作主机板厂商</p><p>最后，英睿达 DDR5-4800 系列台式机内存模组，现已陆续抵达世界各地的零售商、商业合作伙伴、以及系统集成商渠道。</p><p>单条 <a href="https://www.crucial.com/memory/ddr5/ct8g48c40u5" target="_self">8GB</a> / <a href="https://www.crucial.com/memory/ddr5/ct16g48c40u5" target="_self">16GB</a> / <a href="https://www.crucial.com/memory/ddr5/ct32g48c40u5" target="_self">32GB</a> 的 DDR5-4800 SKU，首发指导价为 57.99 / 105.99 / 200.99 美元（约 371 / 678 / 1286 RMB）。</p><p>另有 <a href="https://www.crucial.com/memory/ddr5/ct2k8g48c40u5" target="_self">8G×2</a> / <a href="https://www.crucial.com/memory/ddr5/ct2k16g48c40u5" target="_self">16GB×2</a> / <a href="https://www.crucial.com/memory/ddr5/ct2k32g48c40u5" target="_self">32GB×2</a> 的 DDR5-4800 双通道内存套装，建议零售价为 115.99 / 211.99 / 400.99 美元（约 742 / 1357 / 2566 RMB） 。</p>
