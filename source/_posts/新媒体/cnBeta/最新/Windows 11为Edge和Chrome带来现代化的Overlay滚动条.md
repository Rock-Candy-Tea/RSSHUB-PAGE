通过微软的努力，所有基于 Chromium 的浏览器在 Windows 11 平台上都能获得更现代化、更优秀的滚动条体验。近日，微软为 Edge 浏览器带来契合 Windows 11 的视觉变化，带来了透明效果、Mica 和圆角变化。<br>
 <p style="text-align: left;">在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11/10 系统中，微软重新调整了 Chromium Scrollbars 滚动条特性。微软希望 Chromium Scrollbars 在视觉上像其他 Windows 应用程序一样吸引人，并使其性能与 Chromium 的平滑滚动行为相匹配。</p><p style="text-align: left;">在 Windows 11 中为 Microsoft Edge 启用 Overlay Scrollbars</p><blockquote style="text-align: left;"><p style="text-align: left;">1. 关闭微软 Edge 浏览器的所有实例和频道，如果正在运行的话</p><p style="text-align: left;">2. 在"运行"对话框或命令提示符中键入以下内容，然后按回车键</p><p style="text-align: left;">"%LOCALAPPDATA%\Microsoft\Edge SxS\Application\msedge.exe"--enable-features=OverlayScrollbar, OverlayScrollbarWinStyle, OverlayScrollbarWinStyleAnimation</p><p style="text-align: left;">3. 这样打开 Edge Canary，访问一个网站，看一下滚动条。</p></blockquote><p style="text-align: left;">使用 Overlay Scrollbar，Scrollbar 只在你悬停时出现，当你移开鼠标时就会消失。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0827/5f68612b13f7861.gif" referrerpolicy="no-referrer"></p><p style="text-align: left;">微软 Edge 高级产品经理 Scott Low 说：“为了与 Windows 11 的外观和感觉保持一致，我们已经开始在 Edge 中再次尝试使用覆盖式滚动条（打算在chromium 中也将其推向上游）”。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0827/e3e36c71fe1cfc1.jpg" referrerpolicy="no-referrer"></p><p>目前的 Edge 滚动条</p>
