<strong>为帮助缓解共享电动滑板车乱停放的问题，Bird 和 Lime 正在推出基于谷歌“ARCore Geospatial”增强现实地理围栏 API 的新工具。</strong>两家共享出行初创企业提出的解决方案，分别叫做“视觉驻车系统”（Bird Visual Parking System）和“视觉位置服务”（Lime Visual Positioning Service）。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0512/81ac02ea4e7c8e9.jpg" referrerpolicy="no-referrer"></p><p>在今日的 I/O 2022 开发者大会期间，我们得知 API 的定位识别精度可达 1 米以下。</p><p>虽然一些研究表明，只有不到 1% 的用户会乱停放。但考虑到共享载具的庞大基数，此类状况还是会极大拉低各品牌在公众眼中的正面形象。</p><p>Google 指出，Bird 和 Lime 有幸获得了该公司 ARCore 新技术的早期访问。不过后续该 API 也会向全球推广开来，以帮助实现更精确的位置、方向、以及增强现实等体验。</p><p>据 Bird 和 Lime 所述，与最近被共享出行运营商 Tier 收购的 Fantasmo 摄像头定位技术类似，VPS 服务会提示骑手通过智能机扫描周围环境。</p><p>然后系统会将骑手拍摄的照片与 Google 庞大的街景数据库进行比较、同时借助 Google 的 3D 扫描与增强现实技术，来确定骑手的位置和停放是否得当。</p><p>目前 Bird 已在纽约、旧金山、圣迭戈开启 VPS 功能的随机试点，并计划在未来几周至数月内推广到更多城市 —— 该公司宣称又在全球 400+ 城市开展业务、而 Google 有在 87 个市场区域提供了街景覆盖。</p><p>此外 Lime 也有在全球 200+ 城市开展业务，且其 VPS 已在伦敦、巴黎、特拉维夫、马德里、圣地亚哥和波尔多开启飞行测试。结果发现，新工具有助于将乱停车现象减少 26% 。</p>
