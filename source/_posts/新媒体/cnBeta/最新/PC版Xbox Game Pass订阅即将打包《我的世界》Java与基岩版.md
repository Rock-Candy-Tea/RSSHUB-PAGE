作为世界上最受欢迎和易上手的游戏之一，《我的世界》（Minecraft）几乎在你可以想到的任何平台上体验。从移动设备、PC、到游戏主机，玩家都可在基岩和 Java 两大版本种进行选择。<strong>不过用不了多久，Xbox Game Pass for PC 订阅服务，将很快把它们都捆绑于其中。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1020/88123667dc44bfe.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1020/88123667dc44bfe.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>一方面，基岩版主要面向普通用户，提供了付费皮肤、世界、甚至光线追踪等附加体验。</p><p>尽管你也可以在 Java 版本种获得类似的内容，但通常需要付出一些额外的努力。</p><p>此外基岩版还支持跨平台游戏体验，因而对主机 / 手游党更具吸引力。</p><p><a href="https://static.cnbetacdn.com/article/2021/1020/c58849c97da8652.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1020/c58849c97da8652.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p>考虑到受众群体的不同，《我的世界》基岩版和 Java 版之间总是存在着一些分歧。但到了 11 月 2 日，事情将迎来一些转机。</p><p><a href="https://static.cnbetacdn.com/article/2021/1020/3b16a3ab581c0e1.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1020/3b16a3ab581c0e1.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>届时微软将在 PC 版的 XGP 订阅服务中，同时将两个版本都收纳于其中，并统一归于《Minecraft PC Bundle》这个名号之下。</p><p><a href="https://static.cnbetacdn.com/article/2021/1020/e6ee5c13b4fc7eb.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1020/e6ee5c13b4fc7eb.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>此外 Java 和基岩版《我的世界》都将能够在 XGP for PC 下，使用全新且统一的 Minecraft Launcher 启动器进行访问，且允许在两者之间随意切换。</p><p><a href="https://static.cnbetacdn.com/article/2021/1020/88d5627f45e1c4f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1020/88d5627f45e1c4f.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>微软官方还在公告种提到了跨平台游戏体验，或许意味着 Java 与基岩版《我的世界》也终于被打通。至于真相究竟如何，仍有待 Minecraft 团队给予明确的答复。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=296833314&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">Get Minecraft with Game Pass for PC this November（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMjk2ODMzMzE0LnNodG1s.html" target="_self">via</a>）</p><p>最后，现有的 Game Pass 订户，可以直接通过 PC 平台加载《我的世界》。若尚未订阅 XGP，也可趁着微软搞活动时，体验一把限时免费优惠。</p>
