<strong>Build 2022 开发者大会期间，微软证实了 Windows 11 应用商店将很快允许用户下载第三方小部件。</strong>用过 WinKey + W 快捷键的朋友，应该不会对 Windows Widgets 的概览界面感到陌生 —— 比如显示来自 Microsoft MSN 的新闻头条、或天气更新。<br><br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0526/f4c61b42ab7a81f.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://www.windowslatest.com/2022/05/26/microsoft-teases-interesting-3rd-party-widgets-for-windows-11/" target="_self">Windows Latest</a>）</p><p>如果一切顺利，开发者们可于今年晚些时候向 Microsoft Store 提交他们的小部件 / 微应用。审核通过之后，用户便可通过官方应用商店来轻松获取来自第三方的原生小部件。</p><p>显然，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>一直想要积极拓展 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 的这一板块。只是目前仅提供了少量的第一方小部件，比如 Microsoft To Do、Microsoft Stores 和 Movies & TV 等。</p><p><img src="https://static.cnbetacdn.com/article/2022/0526/53a1ff3abd3a930.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>为吸引广大开发者进入小部件生态，微软正在酝酿第三方应用程序接口（API）。不过在 Build 2022 的一场会议上，我们还看到了微软展示的新面板。</p><p>如图所示，用户可通过第三方小部件直接访问相关服务。比如一个类似 Slack 的小部件，就能够查看来自不同频道或对话的消息。相信不久后，我们还有望获得更多增强体验。</p>
