<strong>上月，System76 宣布了一个让开源社区兴奋不已的好消息，因为惠普即将推出预装 Pop!_OS 的 Dev One 笔记本电脑。现在，这款机型已在惠普官网正式上架。</strong>据悉，Pop!_OS 由 System76 基于 Ubuntu Linux 改造，而新推出的 14 英寸 HP Dev One 机型也仅提供了一种配置。当然，这并不妨碍消费者在后续自行升级替换部分组件。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0603/0e7930967872ee5.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0603/0e7930967872ee5.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：HP <a href="https://hpdevone.com/" target="_self">官网</a>）</p><p>这款以开发者为中心的笔记本电脑，选用了 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 7 PRO 5850U 处理器。该 APU 集成了 8C /16T 的 CPU + Radeon 核显，并且预装了 16GB RAM + 1TB NVMe <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 。</p><p>端口方面，该机提供了两个 USB-A 和两个 USB-C，一个全尺寸 HDMI 视频输出 + 3.5mm 复合音频插孔，以及圆孔（DC）电源输入和防盗锁孔。遗憾的是，该机缺乏 SD 读卡器、且不支持雷电。</p><p><img src="https://static.cnbetacdn.com/article/2022/0603/446a7b71e4ea006.png" alt="2.png" referrerpolicy="no-referrer"></p><p>网络连接方面，HP Dev One 也仅限于 802.11ac（Wi-Fi 5），而不是较新的 Wi-Fi 6 或 Wi-Fi 6E 。</p><p>即便如此，System76 首席执行官 Carl Richell 还是就与<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://mall.jd.com/index-1000000155.html" target="_blank">惠普</a>的这项合作感到激动不已，双方正努力将开源 Linux 体验提供给更广泛的受众。</p><p><a href="https://static.cnbetacdn.com/article/2022/0603/e6dcceb4ee48f55.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0603/e6dcceb4ee48f55.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p><strong>以下是 HP Dev One 的主要特点：</strong></p><blockquote><p>● 预装 Pop!_OS 22.04 LTS Linux 发行版</p><p>● 搭载 AMD 锐龙 7 PRO 5850U 低压移动 APU，拥有 8C / 16T + 16MB L3 缓存，基础频率 1.9 GHz / 加速可达 4.4 GHz 。</p><p>● 集成支持硬件解码、DX12、HDMI 2.0 / HDCP 2.2 的 AMD Radeon 核显</p><p>● 板载 2×8GB @ DDR4-3200 SO-DIMM 内存</p><p>● 预装 1TB PCIe 3.0 x4（M.2 2280）NVMe SSD</p><p>● 采用 14 英寸 @ 1080p LED 背光显示屏（1000 尼特）</p>● 内置双立体声扬声器 + 双阵列麦克风</blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0603/c2852c315d598d0.png" alt="4.png" referrerpolicy="no-referrer"></p><p>HP Dev One 还内置了 53 Wh 锂离子电池，标配 HP Smart 65W 电源适配器。配备瑞昱 RTL 8822CE 无线适配器（Wi-Fi 5 + 蓝牙 5.0）。</p><p>两个 USB-C 端口为 10 Gbps 的 SuperSpeed 速率，支持 PD 供电 + DP1.4 视频输出。两个 USB-A 端口为 5 Gbps 速率，且其中一个支持充电。</p><p><img src="https://static.cnbetacdn.com/article/2022/0603/d8b6b1b7ffd369a.png" alt="5.png" referrerpolicy="no-referrer"></p><p>另有一个 HDMI 2.0 视频输出 + DC 电源输入，内嵌 720p 高清网络摄像头。整机大小为 32.34×21.46×1.91 厘米（12.73×8.44×0.75 英寸），重 3.24 磅（1.47 kg）、辅以矿物银配色。</p><p>感兴趣的朋友，现可通过惠普官网选购 HP Dev One 笔记本电脑，售价为 1099 美元（约 7320 RMB）。至于何时推向美国之外的市场，目前暂不得而知。</p>
