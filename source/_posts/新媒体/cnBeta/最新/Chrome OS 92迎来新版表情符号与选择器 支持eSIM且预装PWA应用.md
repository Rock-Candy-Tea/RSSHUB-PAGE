<strong>即两周前的桌面更新后，谷歌又于今日推出了 Chrome OS 92，为用户带来了一些重要的功能更新和其它增强。</strong>其中包括将近 1000 个重新设计的谷歌表情符号（作为 Unicode 13.1 的一部分），且选择器也迎来了 UI 刷新。<br>
<p><a href="https://static.cnbetacdn.com/article/2021/0528/0c0ac444f68eb46.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0528/0c0ac444f68eb46.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">资料图（来自：Acer）</p><p>其次是添加了搜索（英语）功能，用户可通过启动键（或搜索）+ Shift + Space 的组合键来唤出。或右键点击文本字段，接着选择菜单顶部的“表情符号”来打开。</p><p><a href="https://static.cnbetacdn.com/article/2021/0803/cab34082dc30ca4.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0803/cab34082dc30ca4.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>字符将显示在具有九个类别 + 最近使用的密集网格中，然后滚动到特定的部分。具有多种肤色的表情符号位于右下角，右键点击还可显示所有可用选项。</p><p><a href="https://static.cnbetacdn.com/article/2021/0803/09bf35f3af739d4.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0803/09bf35f3af739d4.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>在 Chrome OS 92 中，刚发布的 Google Meet PWA（渐进式 Web 应用程序）和 Chat 应用，其图标现已默认出现在 Chrome OS 的启动器上。</p><p>谷歌还宣称改进了性能体验，让通话适应不同的网络条件，并且可在屏幕分享时调节视频性能。</p><p><a href="https://static.cnbetacdn.com/article/2021/0803/dfd4a99bb7b61f6.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0803/dfd4a99bb7b61f6.png" alt="0.png" referrerpolicy="no-referrer"></a></p><p>值得一提的是，与 Android 智能机一样，Chrome OS 现也支持 eSIM 功能，支持更快地注册蜂窝移动网络服务。</p><p>有需要的朋友，可移步至“设置 -> 网络 -> 移动数据”页面，然后通过空中写号来替代实体 SIM 卡，并且用户能够在提供二维码设置的运营商配置文件之间快速切换。</p><p>这项功能可在兼容的设备上启用，比如<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://acer-pc.jd.com/" target="_blank">宏碁</a>（Acer）的Chromebook Spin 513 和 Chromebook 511 机型。</p><p>如果你需要经常连接到蜂窝移动网络，但又没空跑道商店去购买实体 SIM 卡；或者是经常国际漫游的旅行者，这项功能都显得相当实用。</p><p><img src="https://static.cnbetacdn.com/article/2021/0803/9efdec919ca28e1.png" alt="3.png" referrerpolicy="no-referrer"></p><p>最后，Chrome OS 92 还带来了键盘方面的几项调整，首先是能够通过虚拟键盘来访问剪贴板历史。谷歌同样改进了美式 / 国际键盘的快捷键，并支持连续听写功能。</p><p>此外用户可在 Chrome OS 92 中右键点击 Android / Linux 应用程序窗口，并将其移动到虚拟桌面。其它补充包括：</p><blockquote><p>● 架子上的手提包，现可显示从 Android 应用下载的内容，以及打印到 PDF 功能。</p><p>● Chrome OS 的相机应用，现支持在兼容硬件上的平移、倾斜或缩放操作。</p><p>● 新增名为《团结和睦》（Togetherness）的壁纸集。</p><p>● 启动器的搜索结果，现已包括 Chrome OS 的系统设备帮助。</p><p>● 在《探索》（Explorer）应用中，有为启用 Family Link 功能的小朋友带来新的数字杂志。</p></blockquote>
