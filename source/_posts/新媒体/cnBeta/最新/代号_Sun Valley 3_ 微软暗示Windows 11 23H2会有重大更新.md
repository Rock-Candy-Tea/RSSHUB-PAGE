在多份微软官方支持文档中提及“Version 23H2”功能更新之后，在最新微软网络直播中提及了“Sun Valley 3”、“SV 3”字眼。正如下方截图所示，PowerPoint Live 笔记提到了“SV3 Refined Investments Next Step”，证实了我们的猜测，即该公司暂时不打算放弃 Sun Valley 的代号。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0418/c9fbfb4944e389e.webp" alt="k3na7mub.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Windows 11 系统中的“refined investments”（精炼投资），表明在明年下半年推出的 Windows 11 23H2 上将会有明显的功能改进。</p><p style="text-align: left;">日前，在 Windows Server build 25075 中的 appraiserres DLL 文件文件中，将 Windows 11 23H2被称呼为Copper（CU）版本，而第一行字眼中明确显示 Windows Insider Dev频道中的当前版本是代号镍（NI），也就是22H2。</p><p style="text-align: left;">现在谈论明年的新功能还为时过早，但 Sun Valley 3 很可能会解决新任务栏和平板电脑体验的不足之处。你可能知道，Windows 11 没有一个成熟的任务栏或平板电脑模式，但这可能在明年改变。</p>
