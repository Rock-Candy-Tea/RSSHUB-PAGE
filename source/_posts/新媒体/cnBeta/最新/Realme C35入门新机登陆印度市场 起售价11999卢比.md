<strong>以实惠为主打的 Realme C35 新机，刚刚登录了印度市场。该机配备了 6.6 英寸 @ FHD+ 显示屏，分辨率 2408×1080（刷新率 60Hz）、屏占比 90.7%、峰值亮度 600 nits 。</strong>处理器为紫光展锐 T616（12nm 芯片组），内置 5000 mAh 电池（支持 18W 快充），且预装了基于 Android 11 移动操作系统定制的 Realme UI 2.0 软件。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0307/5fa2f590c01d097.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0307/5fa2f590c01d097.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://www.91mobiles.com/hub/realme-c35-launch-india-price-specifications-sale-date/" target="_self">91Mobiles</a>）</p><p>上月发布的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://realme.jd.com/" target="_blank">Realme</a> C35，现终于正式在印度市场开售。其采用了侧面指纹方案，可选亮绿 / 亮黑配色。机身大小为 164.4×75.6×8.1 毫米，重 189 克。</p><blockquote><p>● 紫光展锐 T616 SoC 集成了 Mali-G57 GPU，且在 4GB RAM + 64 / 128GB ROM 之外支持 microSD 存储扩展。</p><p>● 水滴型刘海屏内嵌了一枚 8MP 前置自拍摄像头，背部为 50MP 主摄 + 2MP 微距 + VGA 黑白人像镜头的三摄组合。</p><p>● 连接方面，该机支持 4G 双卡 VoLTE、蓝牙 5.0 + 802.11ac Wi-Fi、GPS / GLONASS 定位，辅以 3.5mm 音频插孔和 USB-C 充电 / 数据端口。</p></blockquote><p>其中4GB RAM + 64GB ROM 版本的起售价为 11999 卢比（约 986 RMB），4GB RAM + 128GB ROM 版本则是 12999 卢比（约 1068 RMB）。</p><p>此外 Realme C35 首销定于印度标准时 3 月 12 日 12 点整，感兴趣的消费者可通过官网或 Flipkart 选购。</p>
