自森海塞尔（Sennheiser）于 2018 年发布初代 Momentum True Wireless 以来，该系列真无线耳机的音质表现就一直广受好评。2020 年，该公司推出了它的续作。<strong>而现在，我们又迎来了具有重大设计更改、且增强了主动式降噪（ANC）体验的 Momentum True Wireless 3 。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0426/1ffa6116d08aa17.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>据悉，Momentum True Wireless 3 可选黑（black）、白（white）、石墨（graphite）三种配色。定于 5 月 10 日上市，售价为 249.95 美元（约 1640 RMB）。</p><p>外观方面，新款真无线耳机看起来较前代更接近于 CX 和 CX Plus True Wireless，体现了 Sennheiser 已经为该系列产品线确定了一种更加通用的家族式设计语言。</p><p>便携式充电收纳盒方面，Momentum True Wireless 3 已将 USB-C 端口挪到了正前方（Jabra 等越来越多的竞品厂商也在这么做），但织物覆盖的外壳在形状和尺寸上并没有太大变化。</p><p>虽然乍看之下有些奇怪，但在某些场景下，正面连线充电确实要方便许多。不过更让我们感到欣喜的是，Momentum True Wireless 3 新增了无线充电功能。</p><p><img src="https://static.cnbetacdn.com/article/2022/0426/7d6b5056c2d616c.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>在焕然一新的风格之外，Sennheiser 还改进了 Momentum True Wireless 3 的佩戴舒适度、主动式降噪（ANC）、以及通话品质。</p><p>遗憾的是，在音频质量和电池续航这两个关键领域，它并没有迎来重大的升级 —— 新品延续了该公司在德国制造的 7mm 驱动单元。</p><p>软件方面，配套的移应用程序现支持开展听力测试，以带来更加个性化的高中低音体验。不过至少从新闻稿来看，我们可推测整体声音配置文件是相似的。</p><p>编解码器方面，Momentum True Wireless 3 支持 AAC、SBC 和 aptX，后者可在搭配 aptX Adaptive 兼容设备时减少可察觉的延迟、同时达到较 AAC 更高的比特率。</p><p><img src="https://static.cnbetacdn.com/article/2022/0426/57a74a0d5451346.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>此外 Momentum True Wireless 3 能够根据不断变化的环境自动调节降噪等级（自适应降噪），系统会持续监听环境噪声以实时抑制。当然，如有需要，你仍可根据自己的偏好来手动调节。</p><p>值得一提的是，新款真无线耳机在每个耳塞上都集成了 3 个麦克风，官方宣称可为语音和视频通话带来更显著的改进和清晰的音质。</p><p>该耳机还通过了 IPX4 级防水认证，能够在锻炼或小雨环境下佩戴。单体续航为 7 小时，加上充电收纳盒可补充 21 小时，可惜的是不支持同时连接多个蓝牙设备（Bluetooth Multipoint）。</p><p>最后，Sennheiser 最近还推出了售价 199.95 美元（约 1312 RMB）的 Sport True Wireless 真无线耳机，对标的是 Beats Fit Pros 等竞品。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://hot.cnbeta.com/articles/music/1242689.htm" target="_blank">森海塞尔预告第三代Momentum真无线耳机将于4月到来</a></p><p><a href="https://www.cnbeta.com/articles/tech/1255763.htm" target="_blank">森海塞尔Momentum True Wireless 3的首次泄漏由汉莎航空完成</a></p></div>
