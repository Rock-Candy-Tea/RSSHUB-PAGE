<strong>在今年 4 月的公开预览之后，微软终于在今日宣布了将向所有 Windows 10 PC 用户推送 64-bit 版本的 OneDrive 同步客户端的消息。</strong>在此之前，该公司仅为 Windows 用户提供了 32-bit 版本的 OneDrive 网盘客户端。然而在同步大量文件的时候，许多用户都抱怨受到了性能问题的困扰。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0828/a591cb351b36f26.png" alt="0.png" referrerpolicy="no-referrer"></p><p>虽然这款 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 64 位 OneDrive 同步应用没有带来任何新功能，但它至少可以在 Windows 64-bit 操作系统上运行得更加高效。</p><p><img src="https://static.cnbetacdn.com/article/2021/0828/44fbc7e6e9455d6.png" alt="1.png" referrerpolicy="no-referrer"></p><p>OneDrive 团队在<a href="https://techcommunity.microsoft.com/t5/microsoft-onedrive-blog/onedrive-sync-64-bit-for-windows-now-in-public-preview/ba-p/2260619" target="_self">博客文章</a>中解释到：</p><blockquote><p>如果你有许多大文件，且计算机运行着 Windows 64-bit 操作系统，那 64 位版本的 OneDrive 客户端就是最正确的选择。</p><p>它较之前的 32 位应用程序拥有更多资源，可调用的内存（高达 1840 万 PB）和处理能力都更强。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0828/8a6f0234d8ac5d4.png" alt="2.png" referrerpolicy="no-referrer"></p><p>据悉，无论是工作、学校、还是个人用户，大家对 OneDrive 64 位版本的呼声都一直很高。<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>表示，若 Windows 设备满足要求，系统也将自动以 64 位版本来替换现有的 32 位版本。</p><p>最后，尽管目前新版 OneDrive 64 位同步客户端仅能安装在基于 x86_64 处理器的 Windows PC 上，但不久后它也将扩展到 覆盖广大 ARM64 设备。</p>
