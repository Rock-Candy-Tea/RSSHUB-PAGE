<strong>Canonical 刚刚发布了 Ubuntu Core 22，这是一款完全容器化的 Ubuntu 22.04 LTS 版本，专为物联网和嵌入式应用而设计。</strong>通过使用 snap 快照包，Ubuntu Core 上的所有软件都基于容器化，允许内核、操作系统、应用程序的无缝无线更新。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0616/e5bc780442690e4.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Canonical <a href="https://ubuntu.com/blog/canonical-ubuntu-core-22-is-now-available-optimised-for-iot-and-embedded-devices" target="_self">Blog</a>）</p><p>由于软件已经事先完成了打包，因而快照可避免应用程序遇到任何依赖问题。若更新期间遇到任何麻烦，系统都可自动将之回滚到以往的可工作版本。</p><p>Canonical 对 Ubuntu Core 22 的官方描述为“低接触”，理由是它具有开箱即用的增强安全特性 —— 包括安全启动、全盘加密、安全恢复，以及对操作系统和应用程序的严格限制。</p><p>更棒的是，官方承诺为 Ubuntu Core 22 提供长达十年的更新支持。</p><p><strong>Canonical 首席执行官 Mark Shuttleworth 说到：</strong></p><blockquote><p>从云环境开发、到边缘计算设备，我们致力于在任何地方都提供安全、可靠的开源产品与服务。</p><p>随着新版本的发布，Canonical 已做好将 Ubuntu Core 的优势，扩展到整个嵌入式领域的准备。</p></blockquote><p>最后，如果你使用的是台式计算机，那 Ubuntu Core 22 并不适合。相反，4 月发布的 Ubuntu 22.04 LTS（长期支持版本），才是更好的选项。</p><p><strong>Ubuntu for IoT 下载地址：</strong></p><blockquote><p><a href="https://ubuntu.com/download/iot" _src="https://ubuntu.com/download/iot" target="_blank">https://ubuntu.com/download/iot</a></p></blockquote>
