经过一段时间的测试，新版网页端 Microsoft Store 在本月初悄然上线。与嵌套在 <a href="https://www.microsoft.com/zh-cn/store/apps/windows" target="_blank">Microsoft.com 内部的经典网络商店</a>不同，这个新的网络商店已经被转移到 <a href="https://apps.microsoft.com/store/apps" target="_blank">apps.microsoft.com</a>，除了应用之外还允许浏览和安装 Chromium Edge 的扩展。<br>
<p style="text-align:center"><iframe class="juxtapose" src="https://cdn.knightlab.com/libs/juxtapose/latest/embed/index.html?uid=059054aa-c76a-11ec-b5bb-6595d9b17862" width="720" height="515" frameborder="0"></iframe></p><p style="text-align: left;">微软并未正式宣布将会迁移到新版本中，目前经典网页端 Microsoft Store 仍可访问。</p><p style="text-align: left;">该商店名为“<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Apps”，与原生商店应用程序的目标相同，帮助你在离开主设备时探索内容。如果你喜欢通过搜索引擎或平台发现链接，它也很有用。</p><p style="text-align: left;"><img src="https://static.cnbetacdn.com/article/2022/0429/f528cde3a7eb279.webp" alt="QQ截图20220429111246.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">正如你在上面的截图中看到的，新版网页端 Microsoft Store 的新设计更接近于 Windows 11，它似乎遵循了 Fluent Design 的原则。它使用了圆角、极简主义图标等。然而，它并不支持黑暗模式，但这可能会在未来的版本中添加。</p><p style="text-align: left;">与经典的网络商店不同，新的微软商店不能让你查看应用的最后更新日期或发布日期等信息，但你仍然可以查看应用描述、屏幕截图预览以及评论和评级。你还可以点击"安装"按钮，通过一个新的弹出窗口下载应用，这使得安装过程甚至不需要打开应用商店。</p>
