<div>   
<strong>AMD 刚刚发布了基于 Aldebaran GPU 架构的 Instinct MI200 加速卡，同时它也是该公司旗下首款 6nm 多芯片封装（MCM）产品。</strong>规格方面，可知基于 CDNA 2 架构的 Instinct MI200 拥有 580 亿个晶体管、超过 14000 个内核、以及高达 128GB 的 HBM2e 显存，FP32 性能更是达到了惊人的 95 TFLOPs 。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1109/5cfc27dc0250c97.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/5cfc27dc0250c97.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/amd-unveils-instinct-mi200-aldebaran-gpu-first-6nm-mcm-product-with-58-billion-transistors-over-14000-cores-128-gb-hbm2e-memory/" target="_self">WCCFTech</a>）</p><p>AMD 指出，CDNA 2 内核可加速 FP64 和 FP32 矩阵运算，FP64 理论峰值性能最高可达上一代的 4 倍。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=301776547&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">Introducing the AMD Instinct MI200 Series Accelerator（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzAxNzc2NTQ3LnNodG1s.html" target="_self">via</a>）</p><p>得益于业内领先的 2.5D Elevated Fanout Bridge（EFB）封装技术，AMD 首创的多芯片 GPU 设计，在内核数量上达到了上一代的 1.8 倍、内存带宽达到了 2.7 倍，聚合理论峰值内存带宽也高达 3.2 TB/s 。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/afc79112da30d22.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/afc79112da30d22.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>第三代 Infinity Fabric 技术可管理多达 8 条 Infinity Fabric 链路，将 AMD Instinct MI200 计算节点与三代霄龙（EPYC）处理器和其它 GPU 连接起来，以实现统一的 CPU / GPU 一致性、并最大限度地提升系统吞吐量。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/1301e2d7e7c8c3c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/1301e2d7e7c8c3c.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>据悉，AMD Instinct MI200 内部是一块 Aldebaran GPU，并且可进一步细分为主 / 副两部分芯片。每个芯片有 8 组着色器引擎（总共 16 组 SE），且每 SE 包含 16 个具有全速率 FP64 / FP32 / 二代矩阵引擎的 CU（用于 FP16 / BF16 操作）。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/eea208408dd8bef.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/eea208408dd8bef.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>每个芯片 die 上拥有 128 计算单元（CU）/ 8192 个流处理器，总计可轻松达成 220 计算单元 / 14080 个留处理器，辅以新颖的 XGMI 内部互联设计、且每个小芯片具有 VCN 2.6 引擎 / 主 IO 控制器。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/781d648bd076a1c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/781d648bd076a1c.jpg" alt="6.jpg" referrerpolicy="no-referrer"></a></p><p>AMD Instinct MI200 系列加速器基于 AMD CDNA 2 架构，可为广泛的高性能计算（HPC）工作负载提供业内领先的应用程序性能。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/5fd584312011940.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/5fd584312011940.jpg" alt="7.jpg" referrerpolicy="no-referrer"></a></p><p>其中 MI250X 型号提供 4.9X 倍于竞品的 FP64 性能，以及超过 380 万亿次的理论峰值半精度（FP16）浮点运算性能，可极大推动 AI 等数据驱动的研究工作。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/fa047624774f3b6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/fa047624774f3b6.jpg" alt="8.jpg" referrerpolicy="no-referrer"></a></p><p>性能方面，AMD Instinct MI200 也超越英伟达 A100 解决方案，并打破了多项纪录，比如高达 3X 的 AMG 性能。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/74ba834c273b184.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/74ba834c273b184.png" alt="5-1.png" referrerpolicy="no-referrer"></a></p><p>DRAM 方面，AMD 采用了 1024-bit @ 8 通道接口，总计 8192-bit 总线位宽，且每个接口可支持 2GB HBM2e 缓存。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/d05d4788f933ebd.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/d05d4788f933ebd.png" alt="5-2.png" referrerpolicy="no-referrer"></a></p><p>每个堆栈具有高达 16GB 的 HBM2e 缓存，最大可达 128GB 。相比之下，英伟达 A100 平台只有 80GB HBM2e 缓存。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/aa34e2625a296f7.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/aa34e2625a296f7.png" alt="5-3.png" referrerpolicy="no-referrer"></a></p><p>AMD Instinct MI200 的显存速率也达到了 3.2 Gb/s，总带宽 3.2 TB/s，较 2TB/s 的英伟达 A100 80GB 型号还领先 1.2 TB/s 。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/ae15cb98392780c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/ae15cb98392780c.png" alt="5-4.png" referrerpolicy="no-referrer"></a></p><p>目前已知的是，AMD Instinct MI200 将为三台顶级超算提供支持，包括美国百亿亿级 Frontier 系统、欧盟 pre-exascale LUMI 系统、以及澳大利亚千万亿级 Setonix 系统。</p><p><a href="https://static.cnbetacdn.com/article/2021/1109/6d688f6ea4cfeef.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1109/6d688f6ea4cfeef.png" alt="5-5.png" referrerpolicy="no-referrer"></a></p><p>最后，AMD 为 Instinct MI200 系列 GPU 加速卡提供了三种配置选项，包括 OAM 形式的 MI250 / MI250X、以及双槽 PCIe 形式的 MI210 。</p>
