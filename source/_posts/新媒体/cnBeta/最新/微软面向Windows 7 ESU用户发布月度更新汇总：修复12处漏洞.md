<div>   
在面向尚处于支持状态的 Windows 10 功能更新发布累积更新之外，<strong>在本月的补丁星期二活动中微软还面向已经停止支持的 Windows 7 系统发布了月度更新汇总和仅安全更新。本次更新共计修复了 12 处漏洞，其中 4 个被评为关键，因此推荐 IT 管理员尽快部署该更新。</strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0811/e5471c0e0a6b834.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0811/e5471c0e0a6b834.jpg" alt="njthnz2u.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">不过需要知道的是，只有加入 ESU 计划的 Windows 7 设备才能获得这些更新。如果你只是运行 Windows 7 系统的普通用户，那么你无法获得这些安全补丁。本月更新汇总主要修复的 4 个关键漏洞包括</p><ul class=" list-paddingleft-2"><li><p style="text-align: left;">Windows Print Spooler Remote Code Execution Vulnerability - <a href="https://msrc.microsoft.com/update-guide/vulnerability/CVE-2021-36936" target="_blank" rel="nofollow noopener">CVE-2021-36936</a></p></li><li><p style="text-align: left;">Windows TCP/IP Remote Code Execution Vulnerability - <a href="https://msrc.microsoft.com/update-guide/vulnerability/CVE-2021-26424" target="_blank" rel="nofollow noopener">CVE-2021-26424</a></p></li><li><p style="text-align: left;">Remote Desktop Client Remote Code Execution Vulnerability - <a href="https://msrc.microsoft.com/update-guide/vulnerability/CVE-2021-34535" target="_blank" rel="nofollow noopener">CVE-2021-34535</a></p></li><li><p style="text-align: left;">Scripting Engine Memory Corruption Vulnerability - <a href="https://msrc.microsoft.com/update-guide/vulnerability/CVE-2021-34480" target="_blank" rel="nofollow noopener">CVE-2021-34480</a></p></li></ul><p style="text-align: left;">The Windows 7 月度更新汇总为  <a href="https://support.microsoft.com/en-us/topic/august-10-2021-kb5005088-monthly-rollup-69ec750d-30ee-4cbd-82eb-0b1ec2fd5f78" target="_blank" rel="nofollow noopener">KB5005088</a>, 仅安全更新为 <a href="https://support.microsoft.com/en-us/topic/august-10-2021-kb5005089-security-only-update-28805642-8266-40f9-a2be-9003329f661c" target="_blank" rel="nofollow noopener">KB5005089</a>.</p><p style="text-align: left;">微软解释说：“在使用 Point and Print 功能的时候，更改了安装和更新驱动的默认权限要求。安装此更新后，你必须有管理权限才能安装驱动程序。如果你使用 Point and Print，请参阅 KB5005652，Point and Print 默认行为改变，以及 CVE-2021-34481，以了解更多信息”</p>
