近年来，AIoT 已成为智能时代的一个催化剂，边缘计算设备也承担了越来越多的实时处理和分析工作。<strong>而来自德承（Cincoze）的 GOLD 系列，就是这样一类全新设计的 GPU 计算设备。</strong>其旨在满足 AIoT 领域大规模图像处理、机器视觉和机器学习等应用的需求，并且提供了 GP-3000 / GM-1000 等细分机型。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0121/82302d78f204aed.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0121/82302d78f204aed.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>有需要的客户，能够根据不同的应用需求进行定制，比如尺寸、性能、I/O、功能，辅以面向未来的可升级性。</p><p>无论是智能制造、智能交通、智慧城市、乃至安防等场景，Cincoze GOLD 系列都是构建此类 AIoT 智能应用的一个绝佳选择。</p><p><strong>首先介绍 GP-3000，这款边缘计算机具有顶级 GPU 性能，系统总功率可达 720W 。</strong></p><blockquote><p>主板芯片组为 Intel C246 芯片组，处理器可选<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 8 / 9 代至强 / 酷睿（Coffee Lake-R 和 Coffee Lake）高性能 CPU（最高八核）。</p><p>内存可选 64GB DDR4-2666 SO-DIMM 规格（兼容 ECC），前部区域提供了四个 SATA 3 @ 2.5 英寸驱动器位（支持 RAID 0 / 1 / 5 / 10）。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0121/97e994308d56e9d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0121/97e994308d56e9d.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>GP-3000 系列还提供了独有的 GPU 扩展盒（GEB），最多可容纳两张 328 mm 的全长显卡（兼容市面上大部分型号）。单卡支持最高 350W 功率（比如英伟达 RTX 3090），双卡则下调到 250W 。</p><p>另有专利的“可调节 3D GPU 固定支撑架”，以应对剧烈振动的环境。此外 GP-3000 获得了 MIL-STD-810G 认证，且支持包括壁挂在内的多种安装选项。</p><p><strong>其次是 GM-1000 系列，其主打机器视觉和自动化生产机器人应用。</strong>机身大小只有 260×200×85 毫米，支持英特尔至强 / 酷睿处理器（最多八核）与 64GB 双通道 DDR4-2666 SO-DIMM 内存。</p><p>得益于强大的计算性能和紧凑的尺寸，该机非常适合空间有限的 AIoT 使用场景，且前部区域提供了两个 SATA 3 @ 2.5 英寸驱动器位（支持 RAID 0 / 1 5 / 10）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0121/1262716405e8beb.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0121/1262716405e8beb.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>CM-1000 系列机型还配备了嵌入式的 MXM 3.1 Type A/B GPU 插槽，兼容 MXM RTX 3000 / T1000 / P2000 等型号的英伟达 Quadro 显卡。</p><p>整机功率可达 360W，能够满足 CPU / GPU 同时满载的性能需求。辅以 5/50 Grms 的抗冲击测试，即使在高振动环境中也可正常运行 —— 从机场 X 射线扫描仪、工厂 AOI 设备、到自动驾驶汽车等场景。</p><p>最后，GOLD 系列均具备原生高速 I/O 接口，并可通过 Cincoze 专有的 CMI、CFM 和 MEC 模块进行扩展。此外全系支持 9~48V 宽压直流输入，且具有过压、过流、静电防护等安全特性。</p>
