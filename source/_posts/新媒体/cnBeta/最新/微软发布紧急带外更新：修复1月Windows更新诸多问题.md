在今年 1 月的补丁星期二活动日后，Windows Server 管理员抱怨新更新导致服务器出现了诸多严重问题。这些糟糕的问题包括 Hyper-V 无法启动，主控制器经历了一个无休止的重启循环，ReFS 卷变得不可访问，并显示为 RAW 文件系统。<br>
 <p>在这些报告之后，微软似乎已经从 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Update 中删除了该累积更新，尽管他们重新发布了附带警告的更新。现在，微软已经发布了对这些问题的带外（out-of-band）修复：</p><blockquote><p>带外更新以解决安装 1 月份 Windows 更新后的问题</p><p>微软今天（2022年1月17日）针对某些版本的 Windows 发布了带外（OOB）更新。该更新解决了与 VPN 连接、Windows 服务器域控制器重启、虚拟机启动失败和 ReFS 格式的可移动媒体无法安装有关的问题。所有的更新都可以在 Microsoft Update Catalog 中找到，有些也可以在 Windows 更新中作为一个可选更新。请查看您的 Windows 版本的发行说明以了解更多信息。</p></blockquote><p>以下Windows版本的更新可在Windows
