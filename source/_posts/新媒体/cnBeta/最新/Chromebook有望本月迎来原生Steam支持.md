经过持续多年的铺垫，Chromebook 已在教育上网本市场获得了相当高的市场份额。这类 Chrome OS 设备通常重量较轻、定价实惠，且拥有相当不错的电池续航时间。然而在游戏市场，Chromebook 设备就有些力不从心了。<strong>不过近日，Android Police 还是在 Chromium Gerrit 合并中发现了一些与 Chrome OS 原生 Steam 客户端有关的趣味内容。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0821/17c3d5042023fa7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0821/17c3d5042023fa7.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">资料图（via <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://asus.jd.com/" target="_blank">ASUS</a>）</p><p>作为 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>、Mac 和 Linux 上最受欢迎的游戏平台之一，Steam 却迟迟未能登录 Chromebook 。如果你正苦恼于无法发挥硬件的娱乐潜力，那这里有个好消息要告诉你。</p><p>Android Police 指出，最新的 Chromium Gerrit  合并（代号 Borealis）暗示，Chrome OS 最早有望于本月迎来原生 Steam 支持。</p><p>其实早在 2020 年初，Valve 就想着将 Steam 搬到 Chrome OS 平台。但在那之后，我们一直没有听闻更多细节。</p><p>但是现在，我们已经在 Chrome OS 98 中看到了新的标记 —— 后续或率先登陆 Canary 通道。</p><p><img src="https://static.cnbetacdn.com/article/2021/1203/923440d8bd54c25.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（截图 via <a href="https://www.androidpolice.com/official-steam-for-chromebooks-support-could-launch-this-month/" target="_self">Android Police</a>）</p><p>预计它将基于 Chrome OS 的 Linux 兼容层实现，使用 Crostini 在 Chrome OS 上运行 Steam Linux 客户端，此外官方将为用户带来更多体验方面的改进。</p><p>另一方面，Google 似乎也致力于在 Chrome OS 上提供良好的游戏体验，因为该公司最近也被发现为该平台准备了一个新的“游戏模式”。</p><p>最后，Android Police 发现 Proton 将有一个“游戏兼容性报告”系统。作为一个由 Valve 牵头的兼容层（Wine 集成），其致力于让 Windows 游戏在 Linux 操作系统上运行。</p><p>不过无论哪种方式，与当前的解决方案相比，原生支持都将为 Chromebook 带来更好的体验。</p>
