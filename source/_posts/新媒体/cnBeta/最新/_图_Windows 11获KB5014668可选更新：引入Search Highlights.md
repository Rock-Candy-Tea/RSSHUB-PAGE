<strong>继本月补丁星期二活动日之后，今天微软面向所有支持的 Windows 10/11 系统发布了可选更新。</strong>Windows 11 在安装 KB5014668 可选更新之后，版本号升至 Build 22000.778，为 Windows Search 引入了 Highlights 功能。此功能将为您提供有关今天的各种重要信息，例如假期、纪念日等。对于企业客户，Search Highlights 提供最新的更新和文件。<br>
 <p>离线下载：<a href="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5014668" target="_blank">KB5014668</a></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0624/d7bc13b525550d6.webp" alt="9du6nqmo.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">KB5014668 中的其他更改包括对问题的重要修复，例如无法更新到 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11、重新连接到蓝牙设备、玩依赖特定音频技术的游戏或访问 <a data-link="1" href="https://microsoft.pvxt.net/9W473" target="_blank">Surface</a> Dial 设置。该可选更新亮点：</p><p style="text-align: left;"><strong>New！引入新的 Search Highlights</strong></p><blockquote><p style="text-align: left;">Search Highlights 将展示每天的特别重要且有趣的时刻，例如假期、周年纪念日以及全球和您所在地区的其他<a data-link="1" href="https://c.duomai.com/track.php?k=yUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmYiJ05WZkVHdzZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyUiR" target="_blank">教育</a>时刻。要查看 Search Highlights，请单击或点击任务栏上的搜索图标。对于企业客户，搜索亮点还将包含您组织的最新更新，并建议人员、文件等。</p><p style="text-align: left;"><img src="https://static.cnbetacdn.com/article/2022/0624/eff1d1f6a707a6d.webp" alt="f2ywxr7h.webp" 地址和计算机名称的问题。</p><p style="text-align: left;">● <strong>New！</strong>添加服务器消息块 (SMB) 重定向器 (RDR) 特定的公共文件系统控制 (FSCTL) 代码 FSCTL_LMR_QUERY_INFO。</p><p style="text-align: left;">● <strong>New！</strong>使用 PowerShell 使 SMB 客户端和 SMB 服务器密码套件顺序可配置。</p><p style="text-align: left;">● <strong>New！</strong>引入Search Highlights<br style="text-align: left;"></p><p style="text-align: left;">●<strong> New！</strong>在 Windows 客户端和服务器轻量级目录访问协议 (LDAP) 实现中添加对传输层安全性 (TLS) 1.3 的支持。</p><p style="text-align: left;">● 修复了可能导致升级到 Windows 11（原始版本）失败的竞争条件问题。</p><p style="text-align: left;">● 修复了在 PowerShell 中错误地显示日文字符的问题。</p><p style="text-align: left;">● 修复了在一段时间不活动后影响云剪贴板服务并阻止计算机之间同步的问题。</p><p style="text-align: left;">● 修复了 Sandbox 开始运行后无法隐藏 Windows Sandbox 启动屏幕的问题。</p><p style="text-align: left;">● 修复了在禁用最终用户定义字符 (EUDC) 时导致具有日语系统区域设置的设备停止工作的问题。</p><p style="text-align: left;">● 启用 InternetExplorerModeEnableSavePageAs 组策略。有关详细信息，请参阅 Microsoft Edge 浏览器策略文档。</p><p style="text-align: left;">● 提供在通用打印操作期间使用网络代理的能力。</p><p style="text-align: left;">● 修复了在使用 DirectX 12 (DX12) 的游戏中可能导致连续视频剪辑播放失败的问题。</p><p style="text-align: left;">● 修复了导致某些游戏使用 XAudio API 播放音效时停止工作的问题。</p><p style="text-align: left;">● 修复了影响某些证书链到作为 Microsoft 根证书计划成员的根证书颁发机构的问题。对于这些证书，证书链状态可以是“此证书已被其证书颁发机构吊销”。</p><p style="text-align: left;">● 修复了阻止通过基于 Web 的分布式创作和版本控制 (WebDAV) 连接使用加密文件系统 (EFS) 文件的问题。</p><p style="text-align: left;">● 修复了导致域控制器在系统事件日志中错误地写入密钥分发中心 (KDC) 事件 21 的问题。当 KDC 使用密钥信任方案（Windows Hello 企业版和设备身份验证）的自签名证书成功处理用于初始身份验证 (PKINIT) 身份验证请求的 Kerberos 公钥加密时，会发生这种情况。</p><p style="text-align: left;">● 修复了在您重新启动设备后阻止蓝牙重新连接到某些音频设备的问题。</p><p style="text-align: left;">● 修复了 Active Directory 轻型目录服务 (AD LDS) 重置 userProxy 对象的密码时出现的问题。当您尝试重置其他人的密码并使用简单绑定进行身份验证时，密码重置失败。错误类似于“00000005：SvcErr：DSID-03380C23，问题 5003 (WILL_NOT_PERFORM)，数据 0”。</p><p style="text-align: left;">● 修复了导致使用外部信任的 Microsoft NTLM 身份验证失败的问题。当包含 2022 年 1 月 11 日或更高版本的 Windows 更新的域控制器为身份验证请求提供服务、不在根域中并且不拥有全局编录角色时，会出现此问题。受影响的操作可能会记录以下错误：</p><p style="text-align: left;"> ◎ 安全数据库尚未启动。</p><p style="text-align: left;"> ◎ 域处于错误状态以执行安全操作。</p><p style="text-align: left;"> ◎ 0xc00000dd (STATUS_INVALID_DOMAIN_STATE)。</p><p style="text-align: left;">● 修复了在修改内置管理员组时导致 LocalUsersAndGroups 配置服务提供程序 (CSP) 策略失败的问题。如果在执行替换操作时未在成员资格列表中指定本地管理员帐户，则会出现此问题。</p><p style="text-align: left;">● 修复了格式错误的 XML 输入可能导致 DeviceEnroller.exe 出错的问题。这会阻止 CSP 被传送到设备，直到您重新启动设备或更正 XML。</p><p style="text-align: left;">● 修复了在安装应用程序且没有网络连接时可能导致 Windows 11（原始版本）停止工作的问题。</p><p style="text-align: left;">● 更新开始菜单以在卸载 Windows 终端后右键单击 (Win + X) 开始按钮时显示 Windows PowerShell。</p><p style="text-align: left;">● 在设置页面上将 Your Phone 应用的名称更改为 Phone Link。</p><p style="text-align: left;">● 修复了导致 Microsoft Surface Dial 自定义设置页面停止工作的问题。</p><p style="text-align: left;">您可以使用此链接从 Windows 更新或手动从 Windows 更新目录下载
