就像面向普通用户的 Windows Insider 项目一样，微软也定期发布 Windows Server 预览版。就在上周，它向已注册 Insider 的用户推出了 Windows Server VNext 预览版 25099。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0427/378bd45811f9c44.webp" alt="dgvj8alk.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">不过和 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 10/11 预览版通过 Windows Update 方式发布不同，Windows Server 预览版是通过 ISO 镜像方式提供给注册用户的。这种情况可能很快就会发生改变，因为<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>现在正在征求 Windows Server 管理员的意见，他们是否希望像 Windows 客户端版本那样向他们提供预览版。</p><p style="text-align: left;">在一个论坛帖子中，微软分享了一个调查的链接，服务器管理员可以填写该调查，向公司提供反馈，说明他们是否愿意将预览版下载到他们的测试设备上。</p><p style="text-align: left;">这种方法背后的动机是“促进版本间的升级”，但微软已经指出，为了使其发挥作用，Windows Server 实例需要连接到互联网。需要注意的是，Windows Server Update Services（WSUS）目前并不支持以这种方式预览版的构建。值得注意的是，目前的升级方法不会被废弃，ISO 仍将继续提供。</p><p style="text-align: left;">该调查相当简短，只包括六个问题。这些问题围绕管理员安装预览版的频率，对安装没有 WSUS 支持的预览版的兴趣，以及受访者加入Insider计划的时间。该调查于2022年5月18日结束。</p>
