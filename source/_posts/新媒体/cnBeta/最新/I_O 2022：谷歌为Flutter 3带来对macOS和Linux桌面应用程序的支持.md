在 I/O 2022 开发者大会的首日，谷歌宣布了用于构建本地编译应用程序的 Flutter 开源多平台 UI 开发框架的最新版本。<strong>该公司在大约四年前推出了 Flutter 1.0 的测试版本，但最新版本号已经达到了 3.0 。</strong>起初该团队希望帮助开发者轻松构建跨平台的移动 App，不过后来陆续添加了对 Web 和桌面的支持。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0512/f387b3bf4fc72b9.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Flutter.dev <a href="https://flutter.dev/" target="_self">官网</a>）</p><p>随着 Flutter 3 的正式到来，该团队又为其带来了对 Linux 和 macOS 桌面应用程序的全面支持，辅以其它诸多新功能、且兼容 Apple Silicon 。</p><blockquote><p>Flutter 与 Dart 语言产品兼用户体验（UX）总监 Tim Sneath 表示，正在推出的 Flutter 3 使其跨<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>、桌面和 Web 提供多平台 UI 开发的一个重要里程碑。</p><p>尽管回想 Flutter 1 刚发布时，他们并不指望它会变成一款被广泛使用的移动工具包。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0512/ec6133f984b653a.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>目前 Flutter 已全面支持 iOS、Android 和 Web 平台，以及 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>、macOS 和 Linux 桌面应用程序。</p><p>以 macOS 为例，最新正式版已提供对通用二进制文件的支持 —— 意味其能够在 Intel / Apple Silicon 芯片平台上原生运行。</p><p>至于 Linux 版本，Google 携手 Ubuntu 开发商 Canonical 带来了同类最佳的高度集成开发选项。</p><p><a href="https://static.cnbetacdn.com/article/2022/0512/2cd57a06b7677a9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0512/2cd57a06b7677a9.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>对于大多数开发者来说，应该还是会将 Flutter 3.0 继续视作他们首选的移动 App 构建框架。包括微信、字节跳动、Betterment、SHEIN 和宝马在内的许多企业，都在押注于它。</p><p>事实上，正如 Google 今日宣布的那样，Flutter 应用发布数已较一年前翻番（超 50 万+）。</p><p>与此同时，也有一些开发者在积极利用它来打造桌面应用程序 —— 比如 Wunderlist 创始人正在推出全新的 Superlist 生产力应用（今日开启桌面 Beta 测试）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0512/7d6003fa46cb1ac.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>正如 Sneath 指出，许多开发者正在使用 Flutter 制作休闲游戏，部分原因是其内置了硬件加速支持 —— 比如《PUBG Mobile》就将 Flutter 用于其非游戏的用户界面。</p><p>虽然 Flutter 团队没有事先预料到这点，但为了照顾这部分开发者，Google 还是引入了基于 Flame 开源游戏引擎的 Flutter Casual Games Toolkit 工具包。</p><p>感兴趣的朋友，可以戳这里（<a href="https://pinball.flutter.dev/#/" target="_self">传送门</a>）体验下用 Flutter 构建的一款弹球主题街机模拟器游戏。</p><p><a href="https://static.cnbetacdn.com/article/2022/0512/a7e166d3e573086.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0512/a7e166d3e573086.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>Flutter 3 的另一项新特性，就是与 Firebase 的更深入集成。作为 Google 用于构建移动和 Web 应用程序的后端平台，Google 希望推动 Android / iOS 体验的同步发展。</p><p>当然，这并未影响 Flutter 与其它第三方服务的集成，包括 Firebase 的竞品、来自亚马逊的 AWS Amplify 等。</p><p><a href="https://static.cnbetacdn.com/article/2022/0512/ba8fa6c8c2731d9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0512/ba8fa6c8c2731d9.jpg" alt="5.jpg" referrerpolicy="no-referrer"></a></p><p>与此同时，Flutter 3 改进了对 Crashlytics 的支持。作为 Firebase 的崩溃报告服务，它现可实时追踪致命的崩溃等事件。</p><p>最后，Flutter 团队现也基本完成了向 Google 倡导的 Material Design 3 设计语言的迁移。</p>
