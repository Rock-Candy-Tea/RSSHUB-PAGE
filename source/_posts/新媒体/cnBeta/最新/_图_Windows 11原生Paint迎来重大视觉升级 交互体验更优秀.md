自 Windows 1 以来就一直存在的 MS Paint 应用，<strong>在即将发布的 Windows 11 系统上得到了重大的升级更新。</strong>在 Windows 11 系统中，微软对包括 Photos、Snipping Tool、Alarms & Clock、Mail & Calendar 以及 Microsoft Store 等多款原生应用进行了视觉升级，而 MS Paint 也在本次升级中带来了全新的体验。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1003/6241718e53dd82f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1003/6241718e53dd82f.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">MS Paint 的新设计现在正向 Windows Insider Program 中的更多测试者推出。作为重新设计的一部分，MS Paint 得到了一个新的命令栏，取代了现有的功能区布局，而且该应用的窗口现在遵循 Windows 11 的新 Mica 材料。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1003/b14df59fce01c66.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1003/b14df59fce01c66.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">它的笔刷、颜色选项、菜单和字体大小工具栏也采用了新的外观，这使得经典的功能区界面感觉过时了。除了 Fluent Design 和 Mica 视觉改变之外，MS Paint 现在包括一个新的文本编辑器的工具栏，以帮助用户改变文本的对齐方式，并使用下拉菜单在不同字体之间轻松切换。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1003/e603743fcb327d1.gif" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/1003/e603743fcb327d1.gif" width="100%/" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">此外，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>似乎也在为 MS Paint 应用程序测试黑色模式，使其与 Windows 11 保持一致。然而，这种重新设计似乎并不完整，在 MS Paint 的菜单和页面之间导航时，你可能会遇到遗留的组件。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1003/9d32998f3f00513.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1003/9d32998f3f00513.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Windows 11 将在几天后到来，而这个新的 MS Paint 将不包括在升级中。这是因为 MS Paint 应用还需要几周的时间来通过 Beta 和 Release Preview 渠道，然后进入生产渠道。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1003/92d4a194d9ea8e5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1003/92d4a194d9ea8e5.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">幸运的是，MS Paint 应用可以通过 Microsoft Store 获得，所以你不必等待 Windows 11 的 22H2 版本或任何其他功能更新来尝试新的外观。</p>
