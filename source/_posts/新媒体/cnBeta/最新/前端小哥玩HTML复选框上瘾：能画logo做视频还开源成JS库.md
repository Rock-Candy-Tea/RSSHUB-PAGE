src="https://static.cnbetacdn.com/article/2021/1024/a2fb7ba08e7252d.jpg" referrerpolicy="no-referrer"><br></p><p>然后在图形计算器desmos上让水波动起来，这样第一步就完成了。</p><p><img src="https://static.cnbetacdn.com/article/2021/1024/03c2c1102a38c5d.gif" referrerpolicy="no-referrer"><br></p><p>第二步，将第一步的成果转化为ASCII码输出。</p><p>这一步的转化主要涉及到将彩色对应灰度。</p><p>采用这个公式，即使是彩色的图片，也只不过是五彩斑斓的灰罢了~</p><blockquote id="0C2GR3VU"><ul class=" list-paddingleft-2"><li><p>GrayScale = 0.21 R + 0.72 G + 0.07 B</p></li></ul></blockquote><p>提取原图的RGB色彩，输出为灰度：</p><blockquote id="0C2GR3VV"><ul class=" list-paddingleft-2"><li><p>const toGrayScale = (r, g, b) => 0.21 * r + 0.72 * g + 0.07 * b;</p><p>const convertToGrayScales = (context, width, height) => &#123;</p><p>const imageData = context.getImageData(0, 0, width, height);</p><p>const grayScales = [];</p><p>for (let i = 0 ; i < imageData.data.length ; i += 4) &#123;</p><p>const r = imageData.data[i];</p><p>const g = imageData.data[i + 1];</p><p>const b = imageData.data[i + 2];</p><p>const grayScale = toGrayScale(r, g, b);</p><p>imageData.data[i] = imageData.data[i + 1] = imageData.data[i + 2] = grayScale;</p><p>grayScales.push(grayScale);</p><p>context.putImageData(imageData, 0, 0);</p><p>return grayScales;</p></li></ul></blockquote><p>然后为每个像素赋灰度值：</p><blockquote id="0C2GR400"><ul class=" list-paddingleft-2"><li><p>const grayRamp = '<a href="https://www.cnbeta.com/cdn-cgi/l/email-protection" class="__cf_email__" data-cfemail="c2e68280">[email protected]</a>%8&WM#*oahkbdpqwmZO0QLCJUYXzcvunxrjft/|()1&#123;&#125;[]?-_+~<>i!lI;:,"^`\'. ';</p><p>const rampLength = grayRamp.length;</p><p>const getCharacterForGrayScale = grayScale => grayRamp[Math.ceil((rampLength - 1) * grayScale / 255)];</p></li><li><p>const asciiImage = document.querySelector('pre#ascii');</p><p>const drawAscii = (grayScales) => &#123;</p><p>const ascii = grayScales.reduce((asciiImage, grayScale) => &#123;</p><p>return asciiImage + getCharacterForGrayScale(grayScale);</p><p>&#125;, '');</p><p>asciiImage.textContent = ascii;</p></li></ul></blockquote><p>最后调整一下图片大小就大功告成了~更多详细内容见文后链接~</p><p><img src="https://static.cnbetacdn.com/article/2021/1024/ce4e319076ab465.gif" referrerpolicy="no-referrer"><br></p><p>在线可玩，快来试试</p><p>在最近的更新中，Bryan称，他创造新天地的事情将暂时告一段落。</p><p>但是他不仅留下了复选框新玩法原理的详细介绍，还有自制的丰富的demos。这些足以让你探索创造。</p><p>简单的动画，贪吃蛇，通过摄像头实时生成复选框版图像（demos中的webcam）…</p><p>点击即可试玩，以贪吃蛇和webcam为例：</p><p>点击snake，键盘上下左右即可控制贪吃蛇：</p><p><img src="https://static.cnbetacdn.com/article/2021/1024/030af7f95a1e1b2.jpg" referrerpolicy="no-referrer"><br></p><p>点击webcam，打开前置摄像头，可以看到自己的实时动态：</p><p><img src="https://static.cnbetacdn.com/article/2021/1024/73a60ad9bf8a23d.jpg" referrerpolicy="no-referrer"><br></p><p><img src="http://dingyue.ws.126.net/2021/1024/5e04f2d2g00r1h18p038vc000hs00egm.gif" referrerpolicy="no-referrer"><br></p><p>根据网友的反馈，似乎在Android系统中打开会白屏，但是在Mac Safari, <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> Safari, 桌面Chrome上都可以使用。</p><p>感兴趣的小伙伴，快来试试吧~</p>
