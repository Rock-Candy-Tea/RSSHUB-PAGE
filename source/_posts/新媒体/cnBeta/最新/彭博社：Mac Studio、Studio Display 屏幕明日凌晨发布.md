彭博社 Mark Gurman 再次确认，苹果计划在北京时间明日凌晨 2 点举行的发布会上推出全新 Mac Studio、Apple
Studio Display 外接显示器等产品。同时，Gurman 还确认，Apple Studio Display 外接显示器将运行
iOS，也就是内置 A 系列芯片。<br>
 <p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0308/cec3cafda0ec989.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0308/cec3cafda0ec989.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0308/cec3cafda0ec989.jpg" referrerpolicy="no-referrer"></a></p><p>今天早些时候，Luke Miani 分享了 Mac Studio 和 Apple Studio Display 外接显示器的概念渲染图，他表示对渲染图的真实性很有信心。Mac Studio 类似增高版 Mac mini，这可能意味着会采用全新散热系统，我们可以将 Mac Studio 视为小尺寸 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">Mac Pro</a> 或者是更强大的 Mac mini。</p><p>上周，有报告首次提到了 Mac Studio 和 Apple Studio Display 外接显示器，但当时只是说这两款产品正在开发中，没想到几天时间过去后，就直接发布了。Mac Studio 可能搭载<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a> M1 Pro 和 M1 Max 芯片，同时可以选配性能增强版 M1 Max，这将成为性能最强大的苹果芯片 Mac 电脑。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0308/5ffbe1d5ae03816.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0308/5ffbe1d5ae03816.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0308/5ffbe1d5ae03816.jpg" referrerpolicy="no-referrer"></a></p><p>至于 Apple Studio Display 外接显示器，Miani 的消息是会采用 27 英寸设计，边框比 Pro Display XDR 更宽一些。现在 Gurman 表示显示器会运行 iOS ，内置 A 系列芯片。此前，还有消息称显示器分辨率会升级为 7K，至于最终是什么配置，还需要等到明日凌晨。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0308/574989636c8806c.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0308/574989636c8806c.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0308/574989636c8806c.jpg" referrerpolicy="no-referrer"></a></p>
