<strong>Mozilla 旗下的 Firefox 浏览器即将上架 Microsoft Store。</strong>此举似乎是为了响应微软最近的政策转变，现在微软允许更多的第三方浏览器和其他应用程序在其应用商店平台上使用。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1021/1dcb0a62cd4092e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1021/1dcb0a62cd4092e.jpg" alt="p0zbdyuj.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Mozilla 在官方博文中写道：</p><blockquote style="text-align: left;"><p style="text-align: left;">微软已经放松了对其 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 商店的限制，这些限制之前禁止第三方浏览器进入该商店。我们多年来一直主张在Windows操作系统上有更多的用户选择和控制。</p><p style="text-align: left;">我们欢迎 Microsoft Store 以更开放的态度欢迎像 Firefox 这样的独立浏览器。我们相信，一个更健康的互联网是人们有机会选择多样化的浏览器和浏览器引擎。 Firefox 浏览器将于今年晚些时候在 Microsoft Store 中推出。</p></blockquote><p style="text-align: left;">目前，Firefox 桌面版需要前往 Mozilla 官方网站进行下载安装。近日，Opera 浏览器上架 Microsoft Store，值得注意的是，虽然该应用被列在商店中，但它的管理和更新是独立于微软的。</p>
