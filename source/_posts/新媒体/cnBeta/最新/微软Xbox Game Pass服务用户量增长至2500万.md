微软的Xbox Game Pass服务现在有2500万用户。这比微软之前在去年1月报告的1800万用户有大幅增长。Xbox Game Pass可供下载的游戏目录继续吸引新的用户，微软方面也正在持续扩展这项服务。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0118/308c7d0ede3f996.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0118/308c7d0ede3f996.jpg" title alt="1632709337_Announcing-a-new-game-for-Xbox-Game-Pass-arriving-at.jpg" referrerpolicy="no-referrer"></a></p><p>随着微软用户数量的增长，<a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> Game Pass上可用的游戏数量也在增加。2021年，Xbox Game Pass增加了贝塞斯达的标志性游戏，这让XGP服务中共有20个游戏，包括《冤罪杀机》、《毁灭》、《上古卷轴》、《辐射》和《狼人》。</p><p>微软在宣布这一里程碑的同时，还透露了其687亿美元收购动视暴雪的计划，动视CEO鲍比·科蒂克及其1万名员工将向菲尔·斯宾塞汇报。</p><p>这次收购还加强了微软的Game Pass组合，计划将动视暴雪的游戏推出到Game Pass中，动视暴雪在190个国家拥有近4亿月度活跃玩家，并拥有三个价值数十亿美元的游戏经营权，这次收购将使Game Pass成为业界最引人注目和最多样化的游戏内容阵容之一。收购完成后，微软将拥有30个内部游戏开发工作室，以及额外的出版和电竞制作能力。</p><p>斯宾塞在一篇博文中说，"我们将在Xbox Game Pass和PC Game Pass中提供尽可能多的动视暴雪游戏，包括新游戏和动视暴雪令人难以置信的目录中的游戏。"虽然没有具体细节，但暴雪旗下有《魔兽》、《使命召唤》、《暗黑破坏神》和《糖果粉碎》等品牌可供选择，因此该服务的扩张方式将有很多选择。</p><p>在过去的一年里，微软也一直在满足体育迷的需求，相继提供了NBA 2K21、Madden NFL 21、NHL 21等游戏。去年，Outriders也在Xbox Game Pass上首次亮相，微软在夏天将《飞行模拟》加入了该服务。《帝国时代4》、《极限竞速:地平线5》和《光环:无限》甚至第一天就在该服务上推出。</p><p>由于Xbox游戏流的出现，微软也一直在努力扩大Xbox游戏通行证的使用范围。去年夏天，Xbox云游戏（前身为Project xCloud）的网络版被广泛提供给iOS和PC上的玩家，而微软在11月开始将该服务推广到Xbox游戏机。这将使较新的游戏，如《微软飞行模拟器》也能在较老的Xbox游戏机上进行流媒体运行，因为该服务完全由Xbox X系列硬件提供。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1227365.htm" target="_blank">微软以687亿美元全现金交易收购动视暴雪公司</a></p></div>
