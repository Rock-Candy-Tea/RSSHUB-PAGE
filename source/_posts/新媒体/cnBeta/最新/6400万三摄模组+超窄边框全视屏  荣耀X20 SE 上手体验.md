<div>   
6月，荣耀在发布了荣耀 50 系列旗舰新品之后，时隔半个月又推出了一款新品荣耀X20 SE，不仅在软件/硬件规格上继承了荣耀新品牌的设计风格，还在手机的影像、设计以及屏幕规格上，做出了2000元内手机市场的需求搭配探索。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/13586b2fb23dfa2.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/13586b2fb23dfa2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>外观设计</strong></p><p style="text-align: left;">荣耀X20 SE 的外观设计是给笔者印象最深刻的部分之一：作为一款售价两千元内价位机型，荣耀X20 SE 在<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>的品质感设计上下了不少功夫，让手机呈现出的质感远超同价位其他性价比机型，你甚至能在它身上找到不少曾经<a data-link="1" href="https://c.duomai.com/track.php?k=WP0ZSPklWdlZyN4MTPklWYmYDO5IDNy0DZp9VZ0l2cmYiRyUSbvNmLsxWYtZnL3d3dGJTJGJTJBNTJzBHd0h" target="_blank">华为</a> Nova 系列所追求的时尚设计元素。</p><p style="text-align: left;">荣耀X20 SE在加工工艺上别出心裁的使用了高压注塑 3D <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_92263%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D3_%25E7%2594%25B5%25E8%25A7%2586%25E7%25B1%25BB%25E5%259E%258B_%25E6%259B%25B2%25E9%259D%25A2%23J_crumbsBar" target="_blank">曲面</a>工艺，让手机质感上呈现出一种磨砂消光的视觉效果，并且会在环境光源的折射下呈现出不同光泽的渐变效果；笔者手上的这台荣耀 X20 SE 是“蓝水翡翠”配色，其绿色质感还能在不同强度的光线折射下同时存在晶莹剔透的亮绿色与低调奢华的暗绿色两种色调，让整体呈现出一种如同翡翠一般的质感。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/87d4b92f91d6b9d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/87d4b92f91d6b9d.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">除了直接提升视觉显示效果之外，荣耀X20 SE 的机身工艺还让其握持手感变得更加温润，相比于传统的塑料乃至玻璃材质机身来讲，更不容易因日常使用时沾染到指纹等污渍,而影响手机颜值。对于笔者这样手机经常“裸奔”，但还希望手机保持干净漂亮的“强迫症”用户来讲，这样的设计能在日常生活中减少很多清洁工作。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/72ce9e607ff9698.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/72ce9e607ff9698.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">将视线转回手机正面，荣耀X20 SE 的外观设计同样有不少看点：首先就是其高屏占比的屏幕设计，其实手机的屏占比提升也算是手机设计中老生常谈的话题了；不少手机厂商的旗舰产品也在不遗余力地通过各种技术来降低手机边框的厚度；而荣耀X20 SE 则是借助了多种技术方案，让手机最终获得了 94.2% 的超高屏占比，日常使用感知最明显的左右边框更是仅有 1.3mm ，仅为一枚一元硬币厚度的70%或是 10 张白纸的厚度。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/d328113dc74a0f8.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/d328113dc74a0f8.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>相机表现</strong></p><p style="text-align: left;">接下来值得一提的还有相机硬件：荣耀这次在 X20 SE 的相机硬件搭载了 6400W 主摄 + 200万景深+ 200W 微距，主摄像头规格为 F1.9 光圈，等效 27mm 焦距。；而在夏天经常日光充足的室外环境中，荣耀X20 SE 所搭载的这一颗 6400 万像素主摄像头的高像素优势能体现的淋漓尽致：比如能更突出被摄物体的细节以及得到更清晰自然的出片效果，色彩还原也能更加准确。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/940b45ac7a5adbc.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/940b45ac7a5adbc.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">除此之外，搭配上 荣耀X20 SE 相机 App 中的大光圈模式，可以在拍摄物体是获得较好的背景虚化效果，支持超级微距功能也能在拍摄花朵等风景静物时派上用场：荣耀X20 SE 在超级微距模式下，对焦拍摄物体的最佳距离大约为 4 厘米，同时在这样的微距下快速对焦，同样也非常适合抓拍一些移动物体。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/abb9feb52bde5c6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/abb9feb52bde5c6.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>超级微距模式样张</strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/0ac5bf65be2c2e0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/0ac5bf65be2c2e0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>大光圈模式样张</strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/c662a013576a480.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/c662a013576a480.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>夜景模式样张 1</strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/142724d9fd9bde3.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/142724d9fd9bde3.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>夜景模式样张 2</strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/842ce8b81c3d72d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/842ce8b81c3d72d.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>夜景模式样张 3</strong></p><p style="text-align: left;"><strong>充电/续航</strong></p><p style="text-align: left;">如今已经是手机快充/大电池非常普及的 2021 年，这样的竞争在性价比档位千元机市场更是尤为激烈，大容量电池 + 快速充电成为了用户选择手机时至关重要的加分项，而荣耀 X20 SE 则是在这一竞争维度上采取了平衡的策略：既搭载了 4000mAh 的大电池，也支持最高 22.5w 的 10V2.25A快充。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/21a40bd58a92a59.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/21a40bd58a92a59.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在笔者的实际测试中，荣耀X20 SE 的续航表现在一整天的重度使用 + 外出拍照，实际亮屏时间约为 4 小时的日常使用测试后，荣耀X20 SE 仍然能剩余 32% 的电量。</p><p style="text-align: left;">最后，荣耀X20 SE即使是在亮屏充电测试环节中，也能获得了不错的充电速度：比如，笔者在边充电玩一局《使命召唤手游》的实际测试中，荣耀X20 SE 的剩余电量仍然在此期间增加了 30%，实际息屏充电一小时则可以充至 82%，这样的成绩也算是令人满意。</p><p style="text-align: left;"><strong>硬件性能</strong></p><p style="text-align: left;">荣耀X20 SE 本次搭载的是联发科天玑 700 5G SoC，保证了这款手机在网络规格上在未来的几年内都不会落伍。</p><p style="text-align: left;">荣耀X20 SE 配备了 8GB RAM，在实际体验游戏或日常使用中，并没有出现性能捉襟见肘的情况；同时荣耀X20 SE 还支持 2GB 的“智慧运存”技术：既借助动态运存拓展技术，调用手机内存参与到运存性能释放中，相当于在实际使用中是 10GB RAM 的体验。</p><p style="text-align: left;">虽然荣耀X20 SE 并非一款主打高性能的手机，但在笔者以《使命召唤手游》为例的测试中可以看到，在默认开启超高分辨率及高帧率设定下，荣耀X20 SE 的游戏始终流畅，并没有出现较大的波动。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/86fb146d3d46a46.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/86fb146d3d46a46.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>Magic UI 4.1</strong></p><p style="text-align: left;">荣耀X20 SE 搭载的是 Magic UI 4.1 操作系统，基于 Android 11 打造而来，相比于早先的 Magic UI 版本，带有了更多成熟的设计元素，同时也在常见功能：如 AI 字幕提示，深色模式，全局翻译功能上做了不少改进。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/54584cc213fe4f9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/54584cc213fe4f9.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Magic UI 4.1 中内置的全局翻译功能还让荣耀X20 SE成为日常生活中的翻译助手。比如支持实现直出中文字幕和外语转中文字幕，非常方便；这一功能同样支持系统级翻译，随时随地浏览外文文章和图片轻松读懂内容。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/fb1eeb7512fd1fa.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/fb1eeb7512fd1fa.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>总结</strong></p><p style="text-align: left;">荣耀X20 SE凭借荣耀过硬的技术积累，整机性能调教非常到位，《使命召唤手游》这样的高画质游戏可以流畅运行，《摩尔庄园》这样的小游戏更是不在话下，适合大部分轻度手游玩家，加上大电池 + 快充组合，使其可以满足更多日常使用需求。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0630/d7ae290d0b08126.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/d7ae290d0b08126.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">除此之外，荣耀X20 SE 也在千元级产品经常忽视的外观设计以及握持手感上发力，既提升了手机的外观辨识度，也在配色做工上加分不少，在屏占比，边框处理、摄像头模组等用户感知明显的细节处理上也做的不错，如果你也在选购手机时对这些细节特别关注，那么荣耀X20 SE 可以说是一个非常均衡的选择。</p>
