通过 Windows 10 任务栏和 Windows 11 小部件，微软为用户提供了信息充足的天气和新闻信息。在 Microsoft Edge 浏览器中的 Web Widget 也为用户提供了包括天气、新闻、体育和股票在内的诸多信息。今天，微软对 Web Widget 进行视觉升级，启用了契合 Windows 11 的外观和标签支持。在新的设计变化下，它看起来非常漂亮，非常棒。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0928/3a7e66ee554cd00.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0928/3a7e66ee554cd00.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">由于小编目前使用的是 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 10，因此尚未启用</p><p style="text-align: left;">在 Edge 浏览器中，通过启用 Web Widget 你能够快速浏览和查看股票、新闻、天气、体育等方面的最新信息。该小工具可以从“更多工具”子菜单中打开，或通过访问“设置”中的部分打开。你可以将其设置为每次登录 Windows 时自动启动，也可以将其固定在任务栏上。</p><p style="text-align: left;">Web Widget 默认以垂直布局打开，你可以切换到只搜索布局。当最小化时，为了快速返回到 feed，你可以启用浮动按钮。浮动按钮以 Edge 图标的形式出现在所有应用程序 Windows 的顶部，因此，即使在使用其他网络浏览器和 Windows 应用程序时，你也可以轻松恢复它。</p><p style="text-align: left;">随着 Windows 11 的推出，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>已经更新了收件箱应用程序，如照片、剪贴画、提示应用程序、反馈中心和其他。由于微软Edge已经收到了Windows 11的视觉设计变化和现代滚动条，现在 Web Widget 也得到了 Windows 11 Mica 材质效果、圆角和其他功能的更新。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0928/73edc9e0a9c53f0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">现在，Edge Web Widget 默认会出现在新闻、Bing、Outlook、Linkedin 网站上，并固定在侧边栏上。它允许你添加你喜欢的网站，并直接从里面浏览文章。点击边上的"+"号，输入URL和名称，完成。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0928/242261aacf65f64.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">点击标签访问，链接将在Edge浏览器的单独标签中打开。网络小部件在底部显示后退、前进、导航选项，包括在微软Edge浏览器中打开的能力。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2021/0928/52941cee15d0a9e.jpg" referrerpolicy="no-referrer"></p>
