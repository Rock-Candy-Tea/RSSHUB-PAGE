与不愿透露姓名的内部人士多次交流之后，<strong>国外科技媒体 iDropnews 撰文表示可以确认苹果工程师不再为 iPhone 开发屏下指纹传感器。</strong>此前已经有不少关于苹果开发屏下指纹的谣传，这些也都是真的，只不过现在苹果将重心调整到 Face ID 上了。<br>
<p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0219/216d9a185ec9435.webp" alt="axelk8kw.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">这家位于库比蒂诺的公司在这方面已经工作了多年，但现在已经将重心都放在了 Face ID 上，尤其是苹果在 1 月份决定将他们所有的 Touch ID 工作和资源转移到从事 Face ID 的团队。</p><p style="text-align: left;">所有这些都是有意义的，现在 iOS 15.4 测试版允许用口罩进行 Face ID 解锁，而这将继续成为几乎所有苹果产品的趋势。下一步是显而易见的，而且它将很快准备好--显示屏下的 Face ID 传感器。</p><p style="text-align: left;">我们本可以在<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> 13 Pro中看到这项技术，但苹果最终决定将其推后几年来研究这项技术。这很有意义，考虑到大多数屏下摄像头的样子；它们的质量达不到iPhone的标准。</p><p style="text-align: left;">苹果将在2023年9月左右发布这一设备。我希望该公司不要再推迟无刘海 iPhone 的计划了。我们大多数人已经等待多年，现在 2023 年似乎是一个非常合理的日期。然而，我们谈论的是苹果公司，它以推迟发布甚至取消该公司已经宣布的产品而闻名，比如 AirPower。</p>
