在苹果于 4-5 月份实施了禁止应用程序未经用户许可而开展追踪和数据收集的政策之后，用户也需要手动允许哪些 App 可在平台上对其展开追踪。没想到的是，Microsoft Edge 浏览器用户也将很快面临同样的问题 —— <strong>因为微软正请求用户允许松绑其浏览器数据，以使之能够在 Windows 上投放更具针对性的广告。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0609/f4ec39fc07d8300.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0609/f4ec39fc07d8300.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>Techdows 报告称，其在最新的 Edge Dev 版本中发现，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>现在会弹出一个窗口，请求用户放行 Edge 等微软服务的数据收集，以向他们展示更加个性化的搜索、购物、新闻和广告。</p><p><strong>在一项名为“与其它 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 功能共享浏览数据”的设置中，微软解释称：</strong></p><blockquote><p>启用后，Microsoft Edge 会将来自此配置文件的本地浏览数据、与 Windows 的其余部分链接起来。</p><p>该功能有助于您使用 Windows 任务栏上的搜索框等功能，以更轻松地从历史记录、收藏夹、热门站点、以及最近的选项卡（标签页）中查找信息。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2021/0609/65840765b908dcc.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0609/65840765b908dcc.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>如果关闭了这项功能，Microsoft Edge 将删除设备上与 Windows 共享的数据，并停止共享此配置文件中的任何新浏览数据。</p><p>MSPU 指出，该设置实际上会将 Edge 浏览数据直接传输到微软的广告服务，以绕过基于 cookie 的数据收集黄昏时期。</p><p>尽管这项功能有望在 7 月底向主流用户推出，但我们不难想象，大多数 Edge 用户都是不会对这种级别的追踪感到满意的。</p>
