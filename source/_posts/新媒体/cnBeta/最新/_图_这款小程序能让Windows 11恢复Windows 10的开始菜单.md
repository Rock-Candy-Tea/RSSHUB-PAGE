如果你不太习惯 Windows 11 的居中任务栏和开始菜单，那么今天介绍的这款“<a href="https://github.com/bbmaster123/10SM/releases" target="_blank">Native Windows 10 Start menu for Windows 11</a>”小程序能够让你恢复到 Windows 10 的开始菜单。该程序发布在 GitHub 上，但该程序有个错误就是有时候开始菜单退出的时候会很慢。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0112/cefeba746d68447.webp" alt="1w06ruw8.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">微软在宣布<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11时说：</p><p style="text-align: left;">我们已经简化了设计和用户体验，以提高你的生产力，激发你的创造力。它是现代的、新鲜的、干净的和美丽的。从新的开始按钮和任务栏到每一个声音、字体和图标，一切都有意为之，让你掌控并带来一种平静和轻松的感觉。</p><p style="text-align: left;">我们把"开始"放在中心位置，使它更容易快速找到你需要的东西。开始菜单利用云和 Microsoft 365 的力量，向你展示你最近的文件，无论你之前是在哪个平台或设备上查看的，即使是在 Android 或 iOS 设备上.</p><p style="text-align: left;">除了这款“<a href="https://github.com/bbmaster123/10SM/releases" target="_blank">Native Windows 10 Start menu for Windows 11</a>”小程序之外，你还可以使用本站曾介绍过的 <a href="https://www.startallback.com/" target="_blank">StartAllBack 3.0</a>和 <a href="https://www.cnbeta.com/articles/tech/1200779.htm" target="_self">RoundedTB</a> 两款应用。<br></p>
