Windows 11 系统目前已经邀请诸多 Windows Insider 项目成员进行测试，但最低要求意味着很多硬件无法运行新的微软系统。提高设备门槛产生了非常多的争议，<strong>为此微软的项目经理 Aria Carley 在长达 48 分钟的 AMA 直播中尝试解答所有关于商业方面和其他的问题。</strong><br>
<p style="text-align: left;"><iframe src="https://v.qq.com/txp/iframe/player.html?vid=d3264s9f2tb" allowfullscreen="true" width="750" height="480" frameborder="0"></iframe></p><p style="text-align: left;">这段长达 48 分钟的 AMA 视频中，包括 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 下载、升级阻止等等。需要注意的是，这段视频主要以面向商业环境为主，回答主要涉及 Windows 11 的非消费内容方面。AMA 透露的信息有很多，外媒 Tom's Hardware 总结了几个比较值得关注的内容。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0728/4f0ec36724cae59.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0728/4f0ec36724cae59.jpg" alt="jxk8oz92.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>● 如果商用的 Windows 10 设备不兼容 Windows 11，那么会发生什么？使用 Windows Update 会发生什么？</strong></p><p style="text-align: left;">根据 Carley 的说法，如果不符合标准，商用 PC 将无法下载 Windows 11 升级版，而微软也明白，"这很糟糕"。在视频第 6 分钟的时候她说道：</p><blockquote style="text-align: left;"><p style="text-align: left;">所以我们谈论了这个新的硬件底线，即哪些设备有资格，哪些没有，我们知道有些设备没有资格使用 Windows 11，这很糟糕。但我们这样做的原因是为了让设备保持更高的生产力，有更好的体验，最重要的是有比以前更好的安全性，这样他们就可以在这个新的工作环境中继续提供保护。</p></blockquote><p style="text-align: left;"><strong>● Windows 11 更新机制</strong></p><p style="text-align: left;">Carley 表示微软已经向 Windows 10 的“收件箱”发送了 Windows 11 邀请，这样商业设备也会成为 Windows 11 的目标，但不会造成下载浪费。在视频的第 7 分钟，Carley 表示：</p><blockquote style="text-align: left;"><p style="text-align: left;">你将能够使用更新合规性和 Endpoint 来查看 Windows 11是否不符合设备的条件。无论你是使用我们的第一方分析解决方案还是我们将要提供的脚本，你都能够看到这些设备不符合条件，并解释为什么这些设备不会得到 Windows 11。</p></blockquote><p style="text-align: left;"><strong>● 升级阻止</strong></p><p style="text-align: left;">微软还谈到了 IT 管理员可能想使用组策略绕过 Windows 11 升级阻止的情况。据 Carley 说，商业用户不能使用组策略来绕过 Windows 11 中的硬件阻止块。这有几个原因，根据 Carley 的说法，正如在视频的 10 分钟处所说的：</p><blockquote style="text-align: left;"><p style="text-align: left;">该组策略将无法使你绕过 Windows 11 的硬件强制执行。我们仍然会阻止你将你的设备升级到支持状态，因为我们真的想确保你的设备保持支持和安全。</p></blockquote><p style="text-align: left;">AMA 中涉及的其他话题包括从Windows 11回滚到Windows 10，发布补丁的速度（19分钟），记录已知的问题，以及保障措施和透明度（22分钟），这个视频非常有趣，至少可以说是微软第一次在镜头前谈论 Windows 11 的问题。</p>
