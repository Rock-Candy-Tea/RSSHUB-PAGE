<strong>Redmi Note 11 Pro+ 已经现身 FCC 认证网站，表明该机即将在国际市场发售</strong>。该机属于今年早些时候发布的 Redmi Note 11 系列，该系列其他产品还包括 Redmi Note 11 Pro 和 Redmi Note 11。目前 Redmi Note 11 已经以 Redmi Note 11T 5G 的名字登陆印度，而在全球范围内，它采用了 POCO M4 Pro 5G 的名称。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1225/f6d96ecf9099b5a.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1225/ce90b2a75c98052.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1225/9aa1b1da17a8ea6.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1225/d20a7595d50c758.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">现在，Redmi Note 11 Pro+ 已经开始在全球认证机构进行巡展，预计很快就会在全球推出。看起来该设备在印度推出时将采用 Xiaomi 11i Hypercharge 的名称，因为后者将于 1 月 6 日登陆印度市场。</p><p style="text-align: left;">Redmi Note 11 Pro+ 已被发现在联邦通信委员会（FCC）网站上列出，型号为 20191116UG。FCC 列表显示，全球发布的时间很快就会安排。该数据库将该设备列为5G就绪设备。</p><p style="text-align: left;">Redmi Note 11 Pro+ 是 Redmi Note 11 阵容的高端机型，拥有所有的功能和特点。Redmi Note 11 Pro+ 采用了来自<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a>的 6.67 英寸 120Hz AM<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_88234%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">OLED</a> 面板，顶部由康宁大猩猩玻璃 Victus 覆盖。在机身下，有一个联发科 Dimensity 920 处理器，加上6GB或8GB的内存。可以选择128GB或256GB的UFS 2.2存储系统作为机载存储。</p><p style="text-align: left;">Redmi Note 11 Pro+ 从一个支持 120W 快速充电的 4500mAh 电池中获取能量。在相机方面，Redmi Note 11 Pro+ 预计会有一个 1.08 亿像素的主摄像头传感器，一个 800 万像素的超宽摄像头，以及一个 200 万像素的 50mm 长焦摄像头。用于拍摄自拍的前置摄像头将是一个 1600 万像素的镜头。Redmi Note 11 Pro+ 有一个侧置的指纹识别器，用于生物识别安全。该设备将运行在基于 Android 11 的 MIUI 12.5。</p>
