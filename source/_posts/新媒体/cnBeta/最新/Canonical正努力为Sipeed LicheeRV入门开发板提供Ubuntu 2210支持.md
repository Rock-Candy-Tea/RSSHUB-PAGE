<strong>除了 StarFive VisionFive 和 Nezha RISC-V，Canonical 工程师现也在努力为 Sipeed LicheeRV 带来下月的 Ubuntu 22.10 版本支持。</strong>作为市面上最实惠的 RISC-V 单板机之一，Sipeed LicheeRV 的起价仅为 16.90 美元（约 118 RMB）。即便如此，这块板子还是提供了用于扩展的 M.2 插槽。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0917/71df0f006ba82bb.webp" referrerpolicy="no-referrer"></p><p>去年推出的 Sipeed LicheeRV 最初是为了支持 OpenWrt Linux 发行版，但 Canonical 最近一直在努力让 Ubuntu 88.10 即使提供对该 RISC-V 单板机的支持。</p><blockquote><p>芯片方面，可知 Sipeed LicheeRV 采用了全志 D1 SoC，用的是主频仅 1.0 GHz 的玄铁 C906 单核 RISC-V 处理器。</p><p>搭配 512MB DDR3 内存（另有 1GB 版本）+ microSD 读卡器，并通过 SPI 提供可选的视频 / 显示输出支持。</p></blockquote><p>只是出于成本和性能的考量，LicheeRV 主要还是面向网络和其它互联网用例。对于 Linux 桌面等需要较高性能的用例，显然还是树莓派之类的竞品更合适一些。</p><p>目前基于 DKMS 的 Wi-Fi 驱动程序支持，仍卡在功能冻结的阶段。一旦 LicheeRV 的请求获得批准，我们就有望在下一版 Ubuntu 更新中获得对它的良好支持。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1305727.htm" target="_blank">VisionFive RISC-V单板机终于迎来Ubuntu 22.04.1 LTS支持</a></p><p><a href="https://www.cnbeta.com/articles/tech/1307569.htm" target="_blank">全志Nezha RISC-V开发板迎来Ubuntu 22.04.1 LTS移植支持</a></p></div>
