@ExecuFix 刚刚在 Twitter 上分享了与 AMD EPYC Milan-X 霄龙服务器处理器的一些细节。<strong>可知该系列 Zen 3 CPU 主要升级了 3D V-Cache 缓存，最高拥有 64 核心 / 128 线程、频率达到了 3.8GHz、辅以疯狂的 768 MB 高速缓存，热设计功耗（TDP）上限为 280W 。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0917/53f467c080713b4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0917/53f467c080713b4.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>此前已有多家零售商曝光了多款 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 霄龙 Milan-X 芯片的基础规格，现在，我们又知晓了更多的细节。</p><p>由 @ExecuFix 分享的信息可知，Milan-X 主要由四个系列组成，分别是 64C / 128T 的 EPYC 7773X、32C / 64T 的 EPYC 7573X、24C / 48T 的 EPYC 7473X、以及 16C / 32T 的 EPYC 7373X ，</p><p><img src="https://static.cnbetacdn.com/article/2021/0917/5fe5c91e559e9c5.png" alt="0.png" referrerpolicy="no-referrer"></p><p>相关型号及其 OPN 代码如下：</p><blockquote><p>● EPYC 7773X 64 核（100-000000504）</p><p>● EPYC 7573X 32 核（100-000000506）</p><p>● EPYC 7473X 24 核（100-000000507）</p><p>● EPYC 7373X 16 核（100-000000508）</p></blockquote><p>其中定位旗舰的 AMD RPYC 7773X 拥有 64C / 128T，辅以最高的 280W 热设计功耗（TDP），基础频率 2.2 GHz / 加速频率 3.5 GHz 。</p><p>另有总计 768 MB 的缓存（包括 256MB L3 + 512MB 的 3D 堆叠 SRAM），意味着每个 Zen 3 CCD 拥有 64MB L3，让当前的 EPYC Milan CPU 望尘莫及。</p><p><a href="https://static.cnbetacdn.com/article/2021/0917/86737435ff4d8a8.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0917/86737435ff4d8a8.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>其次是 EPYC 7573X，其具有 32C / 64T，热设计功耗也是 280W 。基础频率 2.8GHz / 加速可达 3.6GHz，总缓存也是 768 MB 。不同的是，你无需通过 8 组 CCD 来堆砌 32 个核心，而是只需通过 4 组 CCD 来实现。</p><p>第三款是 EPYC 7473X，其具有 24C / 48T，热设计功耗 240W，基础频率 2.8GHz / 加速可达 3.7GHz 。第四款是 EPYC 7373X，其具有 16C / 32T，基础频率 3.05GHz / 加速可达 3.8GHz，辅以 768 MB 缓存。</p><p><a href="https://static.cnbetacdn.com/article/2021/0917/bc236c65c373726.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0917/bc236c65c373726.jpg" alt="-1.jpg" referrerpolicy="no-referrer"></a></p><p>除了 3D V-Cache，EPYC Milan-X 系列服务器处理器还享受到了其它益处。比如随着 7nm 工艺的不断成熟，AMD 或为其引入更高的时钟频率。</p><p>考虑到 Milan-X 处理器的 OPN 代码也已经准备就绪，AMD 很可能在 2022 下半年正式推出。</p>
