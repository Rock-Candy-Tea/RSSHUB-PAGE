<strong>尽管其它游戏机制造商并不希望 Xbox Game Pass 来搅局，但 Phil Spencer 还是对跨主机平台的 XGP 体验持开放态度。</strong>毕竟在 PC 和移动设备上，微软早就已经打通了这方面的体验。几年前，甚至有传闻称 Xbox 应用程序和 Game Pass 通行证也即将登录任天堂 Switch 平台。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0813/a25a60b4f9b264f.jpg" referrerpolicy="no-referrer"></p><p>Phil Spencer 于近日接受 GamesRadar 采访时称：</p><blockquote><p>Xbox Game Pass 可能不会很快登陆 Switch 或其它非<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>游戏主机平台，原因其实很简单 —— <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.jd.com/pinpai/14671.html" target="_blank">任天堂</a>和索尼都不希望它过来！</p><p>目前微软仍将 Xbox 游戏业务的重心放在相关‘开放平台’上，比如 PC、移动设备、以及浏览器……</p></blockquote><p>Phil Spencer 补充道：“通常大家会问我咨询具体要发布的某一款游戏，但我们其实也希望能够更加完整的 Xbox 体验”。</p><blockquote><p>目前我们没有将 Xbox 应用程序带到任何其它类型的封闭平台的计划，因为它们并不想要 Xbox Game Pass 之类的东西。</p><p>与此同时，有大量开放平台可供我们大展拳脚，包括 Web、PC、以及移动设备端。坦率地讲，我们几乎已将所有注意力都集中在了这些平台上。</p></blockquote><p>当然，Phil Spencer 也不是想要嘴炮攻击任何其它游戏主机平台的玩家：</p><blockquote><p>我可以理解为何 Xbox Game Pass 不是他们现在所需要的[…]。</p><p>不过当我们说‘希望每个人都能够在 Xbox 上玩游戏’时，实际想要表达是 ——</p><p>‘如果能够将完整体验带到玩家想要的设备上，我们还是对此类讨论持完全开放的态度的’。</p></blockquote><p>换言之，Phil Spencer 很乐意在 Switch 和 PS5 上提供 Xbox Game Pass，但主机制造商的营收也相当依赖于其对平台的控制力。</p><p>此时出现一个 Xbox Game Pass 这样的搅局者，显然会对竞争对手造成极大的压力。</p><p>然而目前任天堂和索尼的游戏业务运营得都很不错，所以微软暂时无法对它们造成任何实质性的打击。</p>   
</div>
