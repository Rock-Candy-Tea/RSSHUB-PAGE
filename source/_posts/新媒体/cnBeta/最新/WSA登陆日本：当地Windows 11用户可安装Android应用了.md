<strong>兑现今年 5 月做出的承诺，今天<a href="https://blogs.windows.com/windows-insider/2022/08/18/expanding-android-apps-on-windows-11-to-windows-insiders-in-japan/" target="_blank">微软宣布向日本地区开放 Windows Subsystem for Android（WSA）系统</a>。</strong>日本地区用户在 Windows 11 系统上通过亚马逊 AppStore 来安装 Android 应用程序。不过前提是要加入 Windows Insider 项目，并将 Microsoft Store 版本号升至 22206.1401.6 或者更高。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0819/1e00dafacaa92ae.webp" alt="84y0xn22.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">此外运行 Android 应用的 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 22H2 设备也需要符合 Windows Subsystem for Android 的硬件需求。虽然目前在 Windows 系统上对 Android 应用的正式支持仅限于美国和日本，不过国内用户可以通过第三方工具或者其他方式侧载 APK 文件，这样也就不需要安装亚马逊的 AppStore 了。甚至有爱好者找到了在 Windows 11 上安装 Google Play 的方法。</p>
