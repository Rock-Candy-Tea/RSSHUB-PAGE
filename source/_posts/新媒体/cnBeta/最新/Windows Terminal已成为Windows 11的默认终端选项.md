早在 2019 年初，微软就宣布要将 Windows Terminal 作为默认的系统终端。期间，Windows Terminal 似乎受到了 Linux 的启发，并引入了对选项卡和其它现代功能的支持。<strong>最新消息是，从 2022 年开始，微软将把 Windows Terminal 作为 Windows 11 操作系统上的默认终端程序。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1215/ffb3420a0f43b01.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1215/ffb3420a0f43b01.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>据悉，<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Terminal 项目经理 Kayla Cinnamon 在一篇开发者<a href="https://devblogs.microsoft.com/commandline/windows-terminal-as-your-default-command-line-experience/" target="_self">博客文章</a>中透露了此事。</p><p>按照计划，Windows 11 上默认的 Windows Terminal 终端程序将率先向预览体验项目的参与者们推送。</p><p><a href="https://static.cnbetacdn.com/article/2021/1215/56b7ab93acc7651.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1215/56b7ab93acc7651.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>与此同时，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还在不停地为 Windows Terminal 构建新的功能，比如更多彩的颜色、表情符号、新字体，并且带来了出色的选项卡 / 窗格体验。</p><p>更棒的是，微软为 Windows Terminal 选用了 MIT 开源许可，意味着感兴趣的人们可在 <a href="https://github.com/microsoft/terminal" target="_self">GitHub</a> 上更轻松地参与该项目。</p>
