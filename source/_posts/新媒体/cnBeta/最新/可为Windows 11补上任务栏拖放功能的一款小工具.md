<strong>在从 Windows 10 升级到 Windows 11 之后，许多用户都在一些细节交互体验上感到难以适应，比如无法将应用图标再拖放到任务栏上。</strong>好消息是，没有等待多久，Windows 社区中就有开发者给出了第三方解决方案。感兴趣的朋友，现可移步至 GitHub 查看名为“Windows 11 Drag And Drop To Taskbar Partial Fix”的项目，并下载体验。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1014/c01f6e7f2ce392b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1014/c01f6e7f2ce392b.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（截图 via <a href="https://news.softpedia.com/news/this-little-app-partially-fixes-the-missing-windows-11-drag-and-drop-to-taskbar-534215.shtml" target="_self">Softpedia</a>）</p><p>这款小工具现可供任何运行 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 操作系统的人使用，顾名思义，其主要恢复了 Windows 11 的任务栏拖放支持。</p><p>然而正如 GitHub 项目描述中所提到的那样，其流畅性暂时无法得到保证。最好的情况，还是看<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>是否会在后续提供官方解锁 Windows 11 的任务栏拖放功能。</p><p><a href="https://static.cnbetacdn.com/article/2021/1014/eee8f2ad449d6fa.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1014/eee8f2ad449d6fa.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">截图（来自：GitHub）</p><p>开发者解释称，这款工具可“部分修复”Windows 11 的任务栏拖放功能的缺失。程序可检测用户是否点下了鼠标左键，并确定用户将鼠标指针悬停在了任务栏的哪个图标上方。</p><p>若光标在同一区域停留数秒，程序还可模拟“WinKey + Ctrl + Number”热键，按顺序恢复预期的窗口。此外它支持将文件拖放到屏幕右下角的“显示桌面”按钮处。</p><p><strong>下载地址：</strong></p><blockquote><p><a href="https://github.com/HerMajestyDrMona/Windows11DragAndDropToTaskbarPartialFix" _src="https://github.com/HerMajestyDrMona/Windows11DragAndDropToTaskbarPartialFix" target="_blank">https://github.com/HerMajestyDrMona/Windows11DragAndDropToTaskbarPartialFix</a></p></blockquote>
