<strong>作为英特尔旗下大名鼎鼎的系统调节工具，Extreme Tuning Utility（简称 XTU）刚刚迎来了最新的 7.8.2.21 版本。</strong>由更新日志可知，新版主要添加了对英特尔 13 代 Raptor Lake 处理器的支持，且在 Windows 11 22H2（Sun Valley 2）操作系统上也能够更顺利地调节电压或超频了。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0623/f3a094e74fa353f.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">截图（来自：Intel <a href="https://www.intel.com/content/www/us/en/download/17881/intel-extreme-tuning-utility-intel-xtu.html" target="_self">官网</a>）</p><p>与此同时，XTU 7.8.0.21 提出了对 7 代 Kaby Lake 芯片的支持。不过考虑到 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 并未正式支持大部分该系列处理器，这项变动的影响并不大。</p><p>新功能方面，XTU 7.8.0.21 针对未来平台带来了 TVB（热速度提升）相关的优化，推测与即将到来的 13 代 Raptor Lake 处理器有关。</p><p><strong>以下是 XTU 7.8.0.21 的完整变更日志：</strong></p><blockquote><p>● 支持新平台的 per-core / package OC 和 Efficient TVB 支持。</p><p>● 改进欠压保护、内核隔离内存完整性、虚拟机状态，以及 Hyper-V 状态检测和通知更新。</p><p>● Dynamic Intel Speed Optimizer 动态优化</p><p>● 支持 Windows 11 22H2</p><p>● 剔除 Kaby Lake 平台支持</p></blockquote><p><strong>已知问题：</strong></p><blockquote><p>● 在某些系统配置中，核心隔离内存完整性、Hyper-V 和虚拟机状态或与 XTU 不兼容。启用相关功能后，XTU 将无法运行。</p><p>● 启用 Windows 智能应用控制后，安装和/或运行 XTU 可能遇到 Windows 通知报错，此时建议重启系统和软件、或重新安装 XTU（官方仍在努力修复）。</p></blockquote><p><strong>下载地址：</strong></p><blockquote><p><a href="https://downloadmirror.intel.com/29183/XTUSetup.exe" target="_self">Intel Extreme Tuning Utility</a>【exe 安装包 | 21.2 MB】</p></blockquote>
