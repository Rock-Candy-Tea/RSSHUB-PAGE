<strong>微软旗下代码托管平台 GitHub，刚刚推出了名叫 Copilot 的 AI 编程辅助工具。</strong>在将之引入集成开发环境（IDE）后，便可向程序员给出基于人工智能（AI）的分析建议。而且它不仅能逐行给出分析参考意见，还能够对完整代码进行遍历。此外 GitHub 表示，Copilot 将支持多种编程语言和框架。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0630/12b3642ca39e448.png" alt="1.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://github.blog/2021-06-29-introducing-github-copilot-ai-pair-programmer/" target="_self">GitHub Blog</a>）</p><p>不过今日推出的技术预览版本，暂时仅限于 Python、JavaScript、TypeScript、Ruby 和 Go 语言。</p><p>据悉，GitHub Copilot 是一个受到 OpenAI Codex 支持的新项目，后者是 OpenAI 这个非盈利组织打造的新型人工智能系统，此前已经接手了数十亿行公共代码的训练。</p><p>与市面上绝大多数编程辅助工具相比，GitHub Copilot 的全文理解能力要强大得多。无论是文档字符串、注释、函数名称，还是代码本身，它都会基于上下文合成代码进行匹配。</p><p><a href="https://static.cnbetacdn.com/article/2021/0630/2ea168df37ed718.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/2ea168df37ed718.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p><strong>以下是 GitHub Copilot 的官方功能介绍：</strong></p><blockquote><p>可将注释转换为代码：你可描述自己想要的逻辑注释，然后交由 GitHub Copilot 来组装代码。</p><p>自动填充重复代码：GitHub Copilot 非常适合快速生成样板和重复的代码模式。你只需给它一些例子，然后它就会生成其余的部分。</p><p>无需辛苦调试：所有强大的软件工程项目，都需要对功能代码进行繁重的调试。导入单元测试包，即可让 GitHub Copilot 与你的实现代码进行匹配测试。</p><p>获取变通方案：如需评估几种不同的替代方案，GitHub Copilot 亦能够向你展示潜在的解决方案列表。你可直接套用，或在此基础上进行适当的修改，以满足实际需求。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0630/7f88b743f744ff1.png" alt="2.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（Copilot <a href="https://copilot.github.com/" target="_self">传送门</a>）</p><p>目前 GitHub Copilot 已作为 <a data-link="1" href="https://microsoft.pvxt.net/zaZYr" target="_blank">Visual Studio</a> Code 的一项扩展而提供，不过在试用 Copilot 之前，还请务必查看以下来自 GitHub 团队的快速问答（FAQ）。</p><p><strong>（1）GitHub Copilot 到底有多好？</strong></p><blockquote><p>基于近期针对一组开源存储库中具有良好测试覆盖率的 Python 函数的基准测试，我们特地清楚了函数体并要求 GitHub Copilot 对其进行填充。</p><p>结果发现，模型在初次尝试时的准确率为 43%，十次尝试的正确率则是 57% 。而随着时间的推移，它肯定会变得越来越聪明。</p></blockquote><p><strong>（2）GitHub Copilot 能否帮助协助完<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://mideajiadian.jd.com/" target="_blank">美的</a>代码？</strong></p><blockquote><p>不会。因为 GitHub Copilot 只会尝试了解编程者的意图，并尽可能地生成最佳代码。然而它给出的代码建议，并不总是最有效、甚至是毫无意义的。</p><p>即便如此，我们仍在努力让 GitHub Copilot 变得更好，且其给出的任何代码建议，还是需要经过仔细的测试和审查。毕竟对品质的最终把控，还是在开发者自己手上。</p></blockquote>
