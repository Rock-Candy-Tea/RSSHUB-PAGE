在 Unity7 桌面环境发展了两年多后，Ubuntu Unity 维护者 Rudra Sarawat 终于在本周早些时候宣布了转正 —— <strong>从 Ubuntu 22.10（Kinetic Kudu）开始，Canonical 将正式接纳 Ubuntu Unity 并作为一种官方风味。</strong>9to5Linux 指出，Ubuntu Unity 前身为 Ubuntu Unity Remix，此前曾作为 Ubuntu 20.04 LTS（Focal Fossa）长期支持版本的一部分而出现。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0903/b9299d10b45c29c.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">资料图</p><p>需要指出的是，由于 Canonical 在五年多前放弃了 Unity7 桌面环境，Ubuntu Unity 也沦落成为了一种非官方风味。</p><p>不过本周，我们还是迎来了一个奇迹 —— Ubuntu Unity 竟有一天被官方认可！按照计划，Ubuntu 22.10 将于今年十月下旬正式到来。</p><p>在此之前，Ubuntu Daily 已提供每日构建的 ISO 映像，以供早期采用者（通过 cdimage.ubuntu.com 获取）品尝。</p><p><img src="https://static.cnbetacdn.com/article/2022/0903/731d297f04616be.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">截图（来自：Ubuntu <a href="https://cdimage.ubuntu.com/daily-live/current/" target="_self">官网</a>）</p><p>此外 Ubuntu Unity 22.10 Beta 将成为首个官方认可的测试版本，其将于本月晚些时候（9 月 29 日）发布，而正式版本（Kinetic Kudu）则是定于 2022 年 10 月 20 日到来。</p><p>最后，Rudra Sarawat 依然在努力 Unity7 桌面环境，并于近期发布了过去六年来的首个主要版本（Unity 7.6）。</p><p>不过他也表示，自己很有兴趣和 UBports 等项目团队合作，以通过 Ubuntu 24.04 LTS 来维护 Ubuntu Unity 的 Lomiri（Unity8）衍生版本。</p>
