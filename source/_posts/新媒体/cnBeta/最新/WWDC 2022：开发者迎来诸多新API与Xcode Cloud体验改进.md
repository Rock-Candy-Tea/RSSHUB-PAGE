WWDC 2022 主题演讲过后，<strong>苹果已向第三方开发者提供了适用于 iOS 16 的锁屏小部件、WeatherKit API、以及 Xcode Cloud 等新内容。</strong>比如锁屏 API 可帮助开发者整合其应用程序的信息，以便在 iOS 的锁屏界面上予以显示。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0607/b77316c9d5c52d1.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0607/b77316c9d5c52d1.jpg" referrerpolicy="no-referrer"></a></p><p>首先介绍 Xcode Cloud 云端开发环境，其允许开发团队将应用程序的构建和编译数据都放置于云端，并支持成员间的实时协作和变更应用程序的代码库。</p><blockquote><p>与面向消费者的 iCloud 体验类似，Xcode Cloud 也可节省本地设备宝贵的存储空间、同时允许开发团队在模拟当前<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>硬件的环境中开展大规模测试。</p><p>此外对于 Testflight 测试通道，Xcode Cloud 也可使 App 构建易于部署到用户设备上。</p></blockquote><p>值得一提的是，新版 Xcode 将构建速度提升了 25%、同时将应用程序所占空间缩减了 30% 。</p><p>另外在 SwiftUI 中，苹果还添加了 iOS、<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a>OS、macOS 和 tvOS 多平台编译的一键切换选项。</p><blockquote><p>Swift 带有正则表达式处理功能，并支持在 Swift 包管理器中优化包管理体验，以便开发者轻松运行自定义命令。</p><p>SwiftUI 现具有改进的导航 API，通过自定义布局，开发者可创建为其应用量身定制的布局。</p><p>另外开发者可自定义 Swift 图表，以便提供最佳的数据可视化。</p></blockquote><p>其次 Metal 3 也成为了新发行版中的一个亮点，可知其优化了文件检索时间、且有助于带来图形体验的飞跃。</p><p><img src="https://static.cnbetacdn.com/article/2022/0607/e66294c83344d11.jpg" alt="weatherkit-og.jpg" referrerpolicy="no-referrer"></p><p>而 WeatherKit 可将以往仅限于 Apple Watch 的可用数据集，集成到开发者的应用程序中。</p><blockquote><p>这主要利用了苹果于 2020 年早些时候获得的 Dark Sky API 数据，现其已再次可供开发者使用。</p><p>在此基础上，苹果还向注册开发人员提供了注重隐私的广告 / 系统功能 API 。</p></blockquote><p>以下是苹果官方新闻稿中提到的主要 API：</p><blockquote><p>● 锁屏小部件（Widgets on the Lock Screen）</p><p>● 实时文本（Live Text）</p><p>● 协作工具（Collaboration Tools）</p><p>● 万能钥匙（Passkeys）</p><p>● 地图套件（MapKit）</p><p>● 专注过滤器（Focus Filters）</p><p>● 自动化捷径（Automatic Shortcuts）</p><p>● watchOS 9 相关</p><p>● 房间平面图（RoomPlan）ARKit 工具</p><p>● 实时活动（Live Activities）</p></blockquote><p>感兴趣的开发者，可于即日起上手体验特定的 API 。</p>
