<strong>按照计划，AMD 将很快推出三款 RDNA 2 桌面显卡新品，包括 Radeon RX 6950 XT / RX 6750 XT / RX 6650 XT 。</strong>而韩国科学与信息通讯技术部下属的认证机构 RRA，刚刚曝光了撼讯（PowerColor）的 Radeon RX 6750 XT Devil 显卡，意味着该卡将很快上市。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0411/4275c901055c5a6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0411/4275c901055c5a6.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>与 RX 6700 XT 相比，RX 6750 XT 在相同的 Navi 22 XT GPU 基础上有所提升，包括 2560 个内核 + 12GB @ 192-Bit 的 GDDR6 显存。</p><p>不过得益于 GPU 核心与显存频率的提升（从 16 Gbps 到 18 Gbps），其性能将略有改善，且 TBP 功耗在 230W 左右。</p><p><img src="https://static.cnbetacdn.com/article/2022/0411/b5ae7f3af54362e.png" alt="2.png" referrerpolicy="no-referrer"></p><p>老伙计 @Harukaze5719 在 Twitter 上分享了 RRA 的认证信息截图，可知撼讯 Radeon RX 6750 XT Devil 新显卡的确切型号为 RR-TUF-RX6750XT3DHEOC 。</p><p>作为相对高端的非公 SKU，工厂已预先超过频（可从 OC 尾缀看出来），预计供电接口为双 8-pin 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0411/f251ce512f74837.png" alt="3.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（截图 via WCCFTech）</p><p>售价方面，预计撼讯会让 RX 6750 XT 与现有的 RX 6700 XT 阵容持平，并且有望与<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>在 2 季度发布的 ARC Alchemist 台式独显展开直接的竞争。</p><p>至于旗舰市场，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> Radeon RX 6950 XT 显然会向英伟达 GeForce RTX 3090 Ti 发起冲击。</p>
