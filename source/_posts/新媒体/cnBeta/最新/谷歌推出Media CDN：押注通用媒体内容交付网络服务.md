在本周的 2022 NAB 峰会上，科技巨头 Google 推出了与旗下 YouTube 流媒体平台采用相同基础设施的通用媒体 CDN 服务。<strong>该公司称，Media CDN 旨在为广大客户提供全方位的自动化内容交付体验、让服务内容更加靠近于终端用户。</strong>对于一家在 200 个市场区域 / 1300+ 城市开展业务的企业来说，Google 显然有这方面的强大实力与底气。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0427/6c8c9ef34add413.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0427/6c8c9ef34add413.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：Google <a href="https://cloud.google.com/blog/products/networking/introducing-media-cdn" target="_self">Cloud Blog</a>）</p><p>受 COVID-19 大流行影响，许多企业被迫采取了让员工居家远程办公的方案，从而催生了对流媒体内容需求的激增。</p><blockquote><p>●《全球互联网现象报告》（<a href="https://www.sandvine.com/hubfs/Sandvine_Redesign_2019/Downloads/2022/Phenomena%20Reports/GIPR%202022/Sandvine%20GIPR%20January%202022.pdf" target="_self">PDF</a>）指出，流媒体视频占据了 53.7% 的互联网带宽流量，较一年前增长了 4.8% 。</p><p>● 与此同时，经营内容交付网络（CDN）业务的相关企业也受益匪浅 —— 其拥有旨在加速 Web 内容交付的庞大服务器集群。</p><p>● 此外在《2021 视频 CDN 市场分析报告》中，<a href="https://omdia.tech.informa.com/OM018075/Video-CDN-Market-Report--2021-Analysis" target="_self">Omdia</a> 预估 CDN 营收在 2020 年增长了 7%、至 44.5 亿美元。</p></blockquote><p>随着谷歌 Media CDN 加入谷歌本就丰富的 Web / API 加速 CDN 产品组合，该公司还为内容交付市场带来了独到的优化。</p><p>除了表面上的各项移除，谷歌还声称能够针对个人用户和网络条件来量身定制交付协议、辅以行业内领先的分担能力（<a href="https://teltoo.com/the-benefits-of-cdn-offload/" target="_self">offload rates</a>）。</p><p><strong>Google 副总裁 Shailesh Shukla 在昨日的一篇博客文章中写道：</strong></p><blockquote><p>通过多层缓存结构，我们尽最大限度地减轻了对源站的调用 —— 即使是那些并不常被访问的内容。</p><p>此举有助于缓解内容源的性能 / 带宽容量压力，从而节省了相关综合成本。</p></blockquote><p>Media CDN 还提供了广告插入工具，允许客户动态地植入带有广告的视频内容。</p><p>在人工智能（AI）/ 机器学习（ML）技术的加持下，Google 声称该服务能够进一步增强交互体验，例如体育赛事期间的实时统计数据 / 嵌入虚拟广告牌的购物链接。</p><p>此外Media CDN 提供了全面的应用程序接口（API）与自动化工具，辅以详实的、预先聚合的指标和回放追踪，以便客户能够轻松诊断整个基础设施架构的性能，通过 Google Cloud 运营套件提供实时的可见性。</p><p><strong>数据显示，在 CDN 服务市场上，Google 还只是大池塘里的一条小鱼。</strong></p><blockquote><p>2019 年的时候，IDC 估计 Akamai 控制了 42% 的 CDN 市场。同年 ChinaNet Center 把持了 13% 的 CDN 市场份额，而 Verizon 只有 5% 左右。</p><p>即便如此，Gartner 高级研究主管 Eric Schmitt 还是指出，Media CDN 有望为 Google 巩固其在流媒体视频和网络广告领域的主导地位而奠定坚实的基础。</p></blockquote><p>若这一预测成为现实，那 Media CDN 或为迄今一直难以盈利的 Google Cloud 云部门提供有力的营收支撑。</p><p>由 Google 母公司 Alphabet 最新公布的收益报告可知，Media CDN 所属部门在 2022 年 1 季度实现了 43% 的增长、至 58 亿美元，同时其运营亏损也扩大到了 9.31 亿美元。</p>
