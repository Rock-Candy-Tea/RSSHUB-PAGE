<strong>今天面向 Beta 和 Release Preview 频道的 Windows Insider 项目成员，<a href="https://blogs.windows.com/windows-insider/2021/03/25/releasing-windows-feature-experience-pack-120-2212-3530-0-to-the-beta-release-preview-channels/" target="_blank">微软发布了 Windows Feature Experience Pack 120.2212.3530.0 更新</a>。</strong>本次更新仅适用于 Beta 频道的 21H1 最新版本和 Release Preview 频道的 20H2 最新版本，不会向 Release Preview 频道的商用设备提供本次更新。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0326/941a5821e5533ea.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0326/941a5821e5533ea.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0326/71fb7af40ef474c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0326/71fb7af40ef474c.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">本次更新主要改进了输入法编辑器（IME）用户候选列表的可靠性。<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Feature Experience Pack 更新会通过 Windows Update 发送给 Insider 成员。</p>
