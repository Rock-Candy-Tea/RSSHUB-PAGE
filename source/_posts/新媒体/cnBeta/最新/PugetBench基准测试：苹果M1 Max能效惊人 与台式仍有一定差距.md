Tom's Hardware 刚刚分享了 M1 Max 新硬件平台的首个专业应用程序基准测试。<strong>结果表明，与带有独立显卡的上一代 MacBook Pro 笔记本电脑相比，Adobe Premiere Pro 的实际表现还是相当喜人。</strong>即便如此，与带有独显的 x86 工作站平台相比，M1 Max 也未能像许多人设想的那样发起越级挑战。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1019/8c53f0cffc2e766.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1019/8c53f0cffc2e766.jpg" referrerpolicy="no-referrer"></a></p><p>本周四，一位用户向 PugetBench 基准测试数据库上传了 M1 Max <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">MacBook</a> Pro 在 Adobe Premiere Pro 应用程序中取得的成绩。</p><p>可知其运行了 0.95.1 版本的 PugetBench for Premiere Pro，以横向比较 2021 款 MBP 对应于 x86 独显机型的水平。</p><p>据悉，Adobe Premiere Pro 15.4.1 视频剪辑软件能够很好地调用 CPU 内核与媒体包房引擎，因而能够相对全面的展现各个平台的能力。</p><p>而后他通过 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_110018%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">4K</a> / 8K 分辨率的编解码器来衡量实时视频播放与导出性能，且分别验证了侧重于软硬件加速（CPU / GPU 硬件）的压力测试。</p><p><a href="https://static.cnbetacdn.com/article/2021/1023/d8633d6179dd45c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1023/d8633d6179dd45c.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：<a href="https://www.tomshardware.com/news/apple-m1-max-benchmarked-in-adobe-premier-pro" target="_self">Tom's Hardware</a>）</p><p>总体而言，配备 M1 Max 旗舰 SoC 的 2021 款 16 英寸 MacBook Pro，在 PugetBench 基准测试中取得了 1168 的标准分 / 1000 扩展分。</p><p>这样的表现，已经明显高于 x86 平台的高端笔记本电脑、且接近于高端台式机的性能。</p><p><strong>细分来看，M1 Max 还拥有更出色的媒体播放引擎。</strong>这方面的性能不仅优于移动独显，甚至超越了英伟达 GeForce RTX 3090 这款消费级旗舰 GPU 。</p><p>对于专注视频剪辑工作的用户来说，M1 Max 的吸引力还是相当大的。毕竟新款 MBP 还拥有相当流畅的系统体验，并且媲美高端台式机。</p><p><strong>视频导出方面：</strong>2021 款 16 英寸 MBP 的工作负载明显优于上一代机型，但在某种程度上落后于其它高端笔记本电脑、更别提“越级”挑战工作站了。</p><p>尽管 PugetBench for Premiere Pro 的导出工作负载包括大量的 CPU / GPU 特效序列，总体得分显得有些参差不齐，但其在现实世界中仍具有一定的参考意义。</p><p><strong>GPU 算力方面：</strong>Adobe Premiere Pro 算是可以充分发挥 GPU 效能的专业应用程序之一。</p><p>对于配备了 32 集群定制 GPU / 4096 个 ALU 的 2021 款 MBP，能够轻易打败上代 Radeon Pro 5500M 独显机型一事，我们也并不感到意外。</p><p>横向对比移动 GPU 市场的其它竞争对手， M1 Max SoC 的图形性能也相当能打。不仅可以叫板 GeForce RTX 3060 / 3080（似乎有些异常偏慢），消耗功率也少得多。</p><p>当然，M1 Max 毕竟只是一枚高能效比的移动 SoC，更强大的任务还是可以交给<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>自家配备高端台式独显的 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">Mac Pro</a> 工作站去完成。</p>
