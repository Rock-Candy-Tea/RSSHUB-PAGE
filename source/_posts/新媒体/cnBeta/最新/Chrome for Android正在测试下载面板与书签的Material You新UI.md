XDA-Developers 报道称：<strong>自 Android 12 发布以来，谷歌一直努力将旧版应用刷新至 Material You 风格。而最新的变化，就发生在了 Chrome for Android 上。</strong>本月早些时候，该应用已首次迎来对动态主题的支持。现在，谷歌又在为其测试新的 UI 修改，以使下载面板和书签部分能够套上更加现代的外观。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0830/326394bfa3b70e3.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://play.google.com/store/apps/details?id=com.android.chrome" target="_self">Google Play</a>）</p><p>首先介绍 Chrome for Android 的下载面板，在下载内容的底部，新设计已改掉灰色的弹出窗口，并且变成了顶部的祝词。</p><p>而已完成的下载，也会出现在带有“打开”按钮的圆形卡片中的相同位置。</p><p><img src="https://static.cnbetacdn.com/article/2021/0830/c7c4d286e932002.jpeg" alt="1.jpeg" referrerpolicy="no-referrer"></p><p style="text-align: center;">下载界面对比（截图 via <a href="https://www.xda-developers.com/google-chrome-android-tests-new-ui-downloads-bookmarks/" target="_self">XDA-Developers</a>）</p><p>其次，书签页面也迎来了 UI 刷新，现具有更大的方格图标、且条目的间距也变得更大。</p><p>需要指出的是，尽管新设计变动看起来很不错，但目前它仅限于 Chrome for Android 的 Canary 抢鲜体验版本。</p><p><img src="https://static.cnbetacdn.com/article/2021/0830/7262aee00dedc72.jpeg" alt="2.jpeg" referrerpolicy="no-referrer"></p><p style="text-align: center;">书签界面对比</p><p>至于谷歌是否会在 Chrome for Android 的最新正式版中引入上述改动，仍有待时间去检验。</p><p>但无论怎样，我们都对看起来更加现代、简洁的 Material You 设计风格充满了期待。</p>
