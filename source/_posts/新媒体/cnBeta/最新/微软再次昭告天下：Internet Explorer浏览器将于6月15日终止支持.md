我们早就知道，微软决定在 2022 年彻底淘汰 Internet Explorer 浏览器。<strong>不过为了让更多人知晓这一点，这家雷德蒙德软件巨头还是决定将 IE 退役截止日期公告于天下，以强调向 Microsoft Edge 全面转型的紧迫性。</strong>具体说来是，该公司在 Windows 消息中心显示了一条新警告。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0318/394b9904c50bde0.png" referrerpolicy="no-referrer"></p><p>除了提醒 IE 将于 2022 年 6 月<a href="https://docs.microsoft.com/zh-cn/lifecycle/announcements/internet-explorer-11-end-of-support" target="_self">正式退役</a>，微软还强调了新版 Edge 浏览器有包含专门集成的 IE 兼容模式。</p><p>换言之，就算 IE 不再作为独立应用程序存在，但它还是会作为 Microsoft Edge 功能的一部分而得到延续。</p><p>对于那些出于兼容性原因而不得不继续使用 IE 浏览器的人们来说，IE 模式可起到救急的作用 —— 至少在他们将应用程序迁移到现代浏览器之前是这样。</p><p><img src="https://static.cnbetacdn.com/article/2022/0318/ac6b8730b7b7d0d.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（截图 via <a href="https://news.softpedia.com/news/microsoft-reminds-the-world-it-s-killing-internet-explorer-535057.shtml" target="_self">Softpedia</a>）</p><p>微软写道：“正如之前宣布的那样，未来 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 上的 Internet Explorer 将整合到 Microsoft Edge 中，通过所谓的 IE 兼容模式而提供对旧版浏览器的支持”</p><blockquote><p>对于某些版本的 Windows 10 操作系统来说，其附带的 IE11 桌面应用程序将于 2022 年 6 月 15 日正式停用。</p><p>这意味着 IE11 桌面应用程序将不再受到支持，之后在用户尝试启动 IE 时，也将被<a href="https://docs.microsoft.com/en-us/windows/release-health/windows-message-center#2792" target="_self">重定向至</a> Microsoft Edge 。</p></blockquote><p>与此同时，作为 Windows 上的默认浏览器，Microsoft Edge 正在以相当快的速度争夺更多市场份额。</p><p>需要指出的是，其内核早已从微软自家的 EdgeHTML、切换到与 Google Chrome 同源的 Chromium —— 这也是微软希望大多数用户能够长期坚持使用它的一个主要原因。</p><p>最后，作为一款跨平台的 Web 浏览器，Microsoft Edge 不仅预装在 Windows 10 / 11 操作系统上，还提供了 Mac、Linux、Android、以及 iOS 版本。</p>
