<div>   
<strong>AMD 刚刚分享了采用 4nm Zen 4 CPU + RDNA 3 GPU 组合的 Phoenix Point APU 处理器的一些新细节，可知其将于 2023 年正式与广大消费者见面。</strong>紧随其后的是结合了 4nm Zen 5 CPU 与 RDNA 3+ GPU 的 Strix Point 家族，预计搭载该 APU 的笔记本电脑会在 2024 年到来。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0610/91b8fe7d97c48f3.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0610/91b8fe7d97c48f3.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（via <a href="https://wccftech.com/amd-phoenix-point-4nm-apu-with-zen-4-rdna-3-in-2023-strix-point-apus-with-zen-5-rdna-3-in-2024-for-laptops/" target="_self">WCCFTech</a>）</p><p>AMD 确认其 Phoenix Point 和 Strix Point APU，将为具有多样化核心 IP 组合的下一代笔记本电脑提供支撑。</p><p>其中 Phoenix Point APU 基于 4nm 工艺节点，并且结合了 Zen 4 CPU 与 RDNA 3 GPU 。支持 LPDDR5 内存和 PCIe 5.0，以及从 35W 到 45W 的 SKU 。</p><p>如果一切顺利，Phoenix Point 笔记本电脑阵容预计会在 CES 2023 消费电子展上亮相，并带来其它方面的更多惊喜。</p><p><img src="https://static.cnbetacdn.com/article/2022/0610/4fcea7aa3f2cba0.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>早期规格揭示锐龙 7000 系列 APU 拥有 8C / 16T 和更高的规格，核显部分也将具有更多的计算单元（CU）数量，而笔记本电脑将迎来多达 16 个 Zen 4 核心的 Dragon Range 。</p><p>然后 Strix Point APU 或采用更先进的制程（或为 3nm），并结合 Zen 5 CPU 与 RDNA 3+ GPU 。</p><p>此前有传该系列 APU 会提供基于两种 Zen 核心 IP 组合而成的混合架构，其中主力大核为 Zen 5、其余内核则是 Zen 4 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0610/8eb4da9d44b7911.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0610/8eb4da9d44b7911.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>不过在最新的路线图中，AMD 已确认新一代 APU 会采用全 Zen 5 CPU 核心，此外也有人提及 Strix Point APU 可能整合所谓的 L4 系统级缓存。</p><p>截止目前，AMD APU 一直是单片式设计，即将包括 CPU、GPU、IO 在内的所有 IP 都部署于同一个芯片上。</p><p>而 Infinity Cache 无限缓存与 RNDA 3+ GPU 等 IP 技术，也有望随 Strix Point Ryzen APU 一同亮相。</p>
