在近日遭遇黑客攻击之后，我们已经见到大量的泄露信息，比如低哈希算力（LHR）的旁路技术、深度学习超级采样（DLSS）的源码、以及下一代 GPU 架构的代号。<strong>昨日，有报道称 Blackwell 将接替 Hopper GPU，成为英伟达下一代计算卡的代号。现在，我们又知晓了 Ada Lovelace 消费级 GPU 的配置规格。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0224/08d2b73dbb96176.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0224/08d2b73dbb96176.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">黑客宣称盗取了 1TB 量级的数据</p><p>泄露消息称，英伟达将至少提供六款 GPU SKU，其中包括了 AD102 / 103 / 104 / 106 / 107 / 10B（规格递减）。</p><blockquote><p>作为 RTX 30 系列 Ampere GPU 的继任者，RTX 40 系列 Ada Lovelace 旗舰 GPU（AD102）将拥有 144 组 SM 单元 / 18000+ CUDA 核心。</p><p>其次 AD103 GPU 拥有 84 组 SM 单元，AD104 / 106 / 107 分别具有 60 / 36 / 24 组 SM 单元。</p></blockquote><p>前五款 GPU SKU 将有面向桌面 / 移动平台的衍生版本，并且涵盖了 GeForce RTX 40 消费游戏 / RTX 工作站解决方案。</p><p>此外据 @Kopite7kimi 所述，英伟达还有针对 Tegra SoC 的更新，有望为下一代<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.jd.com/pinpai/14671.html" target="_blank">任天堂</a> Switch 掌机和 Tegra Drive 车载平台提供支撑。</p><p><a href="https://static.cnbetacdn.com/article/2022/0224/87059ef0f7ade49.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0224/87059ef0f7ade49.jpg" alt="6.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">高功耗迫使下一代旗舰显卡换用新的 PCIe 外接供电连接器</p><p>与 RTX 30 系列旗舰显卡使用的 GA102 GPU 相比，RTX 4090 / 4080 Ti 有望搭配具有 144 组 SM 单元 / 18432 个 CUDA 核心数的 AD102 GPU（较前者高出 71%）。</p><p>不过也有消息称，AD102 GPU 是唯一一款 SM 单元数量增长超 50% 的 SKU，意味着英伟达有在争抢性能桂冠这件事上全力以赴，即便这也让它付出了功耗暴涨的代价。</p><blockquote><p>至于移动平台，AD103 将取代近期面世的 GA103 GPU，且其 SM 单元数量与 GA102 保持一致（都是 84 组），</p><p>AD104 / 106 / 107 GPU 分别具有 60 / 36 / 24 组 SM 单元，但除了 AD103 较 GA103 增长了 40% 之外，其它 SKU 的涨幅都只有 20~25% 左右。</p></blockquote><p>最后，即便性能上无法与 AD102 旗舰 GPU 匹敌，但考虑到主流细分市场，我们仍可期待 RTX 4060 / 4060 Ti 能够与 RTX 3070 / 3080 一战。</p><p>至于 RTX 4050，在排除架构升级带来的 IPC 与频率改进之外，其性能或与 RTX 3060 相当。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a
