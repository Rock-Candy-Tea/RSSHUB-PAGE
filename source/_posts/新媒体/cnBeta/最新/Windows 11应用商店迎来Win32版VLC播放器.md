<strong>适用于 Windows 平台的 VLC 播放器的 Win32 版本，现已通过 Windows 11 操作系统上的 Microsoft Store 应用商店提供。</strong>与通用 Windows 应用（UWP）相比，Win32 版本的功能更加全面，辅以大家期望的所有编解码器。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0722/529f72688b019f9.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p>以下是 Win32 版 VLC 播放器的完整功能：</p><blockquote><p>● 多媒体播放器</p><p>● 音频与音乐播放器</p><p>● 字幕同步</p><p>● 文件转码</p><p>● 流媒体</p><p>● 数字<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_76344%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">电视</a>调谐器</p><p>● DVD 播放</p><p>● 音频 CD 播放器</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0722/71c2789d029cf00.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>Win32 版 VLC 在 Windows 11 应用商店中的出现，证明了<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>致力于让应用商店成为广大用户获取软件的首选场所。</p><p><img src="https://static.cnbetacdn.com/article/2021/0722/2ccdfb627cb562b.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>不过 <a href="https://www.windowscentral.com/new-vlc-app-arrives-microsoft-store-including-dvd-playback-support" target="_self">Windows Central</a> 指出，目前 Win32 版 VLC 应用程序不会出现通过 Microsoft Store 的在线网络搜索结果中。</p><p><img src="https://static.cnbetacdn.com/article/2021/0722/729fe52e44c3757.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>要找到新应用程序，你必须在运行 Windows 11 操作系统的同时，在新版 Microsoft Store 客户端中搜索“VLC”。</p><p><img src="https://static.cnbetacdn.com/article/2021/0722/0cd9b9ff2fc62c6.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p>
