Visual Studio 2022 有望于 11 月 8 日发布。在 Visual Studio 2019 的基础上，新版集成开发坏境提供了非常多的改进，包括对 64 位、.NET 6 和 C++ 20 的支持，为核心调试器提供更好的性能，并在实时共享会话中支持文本聊天。 <br><br>
 <p>虽然适用于 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 和 Mac 上的 <a data-link="1" href="https://microsoft.pvxt.net/zaZYr" target="_blank">Visual Studio</a> 2022 预览版已经有一段时间了，但微软现在已经<a href="https://devblogs.microsoft.com/visualstudio/weve-upgraded-the-ui-in-visual-studio-2022/" target="_blank">撰写了一篇博文，强调了 IDE 的所有 UI 变化。</a><br style="text-align: left;"></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1020/53452211b2ef06a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1020/53452211b2ef06a.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1020/7826d143ffd099a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1020/7826d143ffd099a.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">如上图所示，Visual Studio 2022 正在获得新的图标。该公司表示，这些图标是在与开发者社区讨论并在整个预览过程中审查了他们的反馈后最终确定的。由此产生的图标是现代和直观的。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1020/20fd3593ab49a40.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1020/20fd3593ab49a40.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1020/4fe47d2e846f849.jpg" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/1020/d1d4fecec2754c0.gif" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">此外，雷德蒙科技巨头指出，其修改 Visual Studio 图标的接受标准是一致性、可读性和熟悉性。这涉及到将图标放在正确的位置，使用颜色来识别参考点，为所有受众纠正对比度，以及在熟悉度是关键的地方保留颜色和符号。</p><p style="text-align: left;">微软还发现，深色主题已经超过了浅色主题，成为使用 Visual Studio 开发者最受欢迎的选择。因此，它已经调整了整个 IDE 的颜色对比，使设计更容易被接受，特别是对使用深色主题的人来说。</p><p style="text-align: left;">这方面的一个关键组成部分是突出显示的元素得到一个边框。也就是说，习惯于 Visual Studio 2019主题的开发者可以将其作为一个扩展下载。同样，其他自定义主题也可以从 Visual Studio 市场下载。</p>
