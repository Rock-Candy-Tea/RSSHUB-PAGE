早在 2020 年 1 月，9to5Mac 就报道称，苹果正在开发一种新颖的“专业模式”，以便 MacBook 用户能够解除一些节能限制，从而最大化设备的性能表现。<strong>现在，我们又从 macOS Monterey 最新测试版中发现了类似的功能，只是它的名字已改成更加直观的“高功率模式”。</strong><br>
<p><a href="https://static.cnbetacdn.com/article/2021/0929/c31e03cc1215c8d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0929/c31e03cc1215c8d.jpg" referrerpolicy="no-referrer"></a></p><p><a href="https://9to5mac.com/2021/09/28/macos-monterey-high-power-mode/" target="_self">9to5Mac</a> 指出，<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>在 macOS Catalina（10.15.3）中首次提到了“专业模式”（Pro Mode）。</p><p>内部代码将此电源选项描述为能够让应用程序运行得更快，同时警告电池续航会缩减、<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C738%2C751" target="_blank">风扇</a>噪音也会增加。然而在之后很长一段时间内，macOS 代码中都不再引用过它。</p><p>好消息是，在今早向开发者推送的 macOS Monterey 最新测试版（beta 8）中，我们再次发现了“高功率模式”（High Power Mode）的身影。</p><p>此前，苹果已经长期通过“低功率模式”来降低 Mac 笔记本电脑的性能以延缓电池消耗。</p><p>但之后，该公司或允许用户在不连接外部电源的情况下，也让 Mac 以更高的 CPU / GPU 功率来支撑应用程序的运行。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0923/2d37158589ab804.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0923/2d37158589ab804.jpg" referrerpolicy="no-referrer"></a></p><p>虽然 Apple Silicon Mac 的能效表现已经相当出众，但通过解禁更高的风扇转速来避免过热节流，对需要高性能的用户也是一件好事。</p><p>至于苹果打算何时正式向广大普通 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">MacBook</a> 用户推送，目前暂不得而知。有传闻称，该公司将在今年晚些时候推出搭载新款 M1X 芯片、且增强了图形性能的 MacBook Pro 新机。</p><p>感兴趣的朋友，可通过 Apple Beta Software Program 提交注册申请，以获得 macOS Monterey 测试版软件的体验机会。</p>
