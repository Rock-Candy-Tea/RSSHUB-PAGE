<strong>自动驾驶初创企业 Aurora 刚刚宣布与亚马逊云服务（AWS）达成了开发合作，每天可通过 AWS 云计算服务执行超 1200 万次复杂的驾驶模拟任务。</strong>2019 年，Aurora 先是拿到了现代、起亚的投资，然后去年收购了 Uber 放弃的自动驾驶汽车业务，并于近期与沃尔沃达成了自动驾驶卡车合作、以及丰田的 Siennas 自动驾驶车队开发合作。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1202/33883f2e43ee63f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1202/33883f2e43ee63f.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图自：Aurora）</p><p>据悉，Aurora 将充分利用 AWS 的在线和离线功能，为其机器学习和基于模拟的自动驾驶汽车技术开发提供支撑。</p><p>在云端开发环境下，Aurora 将利用 AWS 来处理和存储于真实道路测试中收集的 PB 级别传感器数据。</p><p>而在离线的时候，该公司还将利用 AWS 开展 Aurora Drive 软件堆栈的测试、高清映射和机器学习等任务。</p><p>此外到 2021 年底的时候，Aurora 计划通过 AWS 支持的虚拟测试套件，完成每天多达 1200 万次的驾驶模拟。</p><p><a href="https://static.cnbetacdn.com/article/2021/1202/2a80977c5e356d7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1202/2a80977c5e356d7.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>最后，在本周于拉斯维加斯举办的 Re:Invent 大会上，亚马逊云服务还宣布了 AWS for Automotive 和 AWS IoT Fleetwise 服务。</p><p>其旨在为汽车制造商、初创企业、车队管理、以及物流公司提供移动互联和自动驾驶领域的专用开发工具，以增强数字客户参与、产品工程设计、以及供应链制造等全环节。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1209489.htm" target="_blank">亚马逊推出面向汽车行业的AWS FleetWise和Automotive新服务</a></p></div>
