先来欣赏一下这划时代的时刻。这个 Gif 来源于油管博主 Kenny Pi 的视频，他耗时半年左右，成功的把一台 iPhone X 的 Lightning 充电接口换成了 Type - C 接口。新的 C 口 iPhone，外观、日常使用和普通的 iPhone 没有任何区别。<strong>关键这博主还只是个的大学生，半年时间做成了一家市值 2.48 万亿美元的公司三四年都没做到的事。</strong><br>
 <p><img src="https://x0.ifengimg.com/ucms/2021_45/5139BF5BEC85677C84BCDF25A36091FFE1530061_size9160_w388_h611.gif" referrerpolicy="no-referrer"><br></p><p><img src="https://static.cnbetacdn.com/article/2021/1106/30e1b0d2dff8aaf.jpg" referrerpolicy="no-referrer"><br></p><p>这个视频在油管发布没多久，就成了 The Verge、Gizmodo、TechCrunch 等国外各大科技网站的头条。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/1317F1CCE7B074F65CE7032C5AA11316B4BD9021_size3973_w1079_h526.gif" referrerpolicy="no-referrer"><br></p><p>说来也是，都 2021 年了，<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">MacBook</a> 是 C 口，<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a> Pro 是 C 口，今年连 iPad mini 都换成了 C 口。<strong><br>怎么说，是我 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> 不配吗？</strong></p><p>现在几乎所有的随身电子设备都是 Type-C，相机、Switch、充电宝几乎都是 C 口。</p><p><img src="https://static.cnbetacdn.com/article/2021/1106/9851d0e6eb88a6c.jpg" referrerpolicy="no-referrer"><br></p><p>应用面本来就很广。</p><p>而且 Type-C 接口，从各个方面来说都明显优于 Lightning 。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/ab5bfae456cd64a.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1106/ab5bfae456cd64a.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1106/ab5bfae456cd64a.jpg" referrerpolicy="no-referrer"></a><br></p><p>这个基于 USB2.0 标准开发的 Lightning， Type-C 理论传输速度是它的 10 倍以上。</p><p>而且 Type-C 可以兼容 PD、QC 等多种快充方案，而<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>除了自家的 5V2.4A 的快充协议以外，就只有 PD 协议，最大承受电流只有 3A。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/6954f5de44dbe86.png"><img data-original="https://static.cnbetacdn.com/article/2021/1106/6954f5de44dbe86.png" src="https://static.cnbetacdn.com/thumb/article/2021/1106/6954f5de44dbe86.png" referrerpolicy="no-referrer"></a><br></p><p>最关键的是， Lightning 还不支持模拟信号的传输，也就是说你要传输数据或者音频输出，就必须要整个 Lightning 转接头。</p><p><strong>都 2021 年了。</strong></p><p><strong>全世界这超过 10 亿的 iPhone 用户，还是得每人随身带一根 Lightning 接口的线。</strong></p><p>大家无时无刻不在期待苹果能把这个扶不起的 Lightning 充电口给抛弃了。</p><p>但 他 就 不 ……</p><p><img src="https://static.cnbetacdn.com/article/2021/1106/e5da7e6135e2cc1.png" referrerpolicy="no-referrer"><br></p><p>算了，一吐槽起来没忍住，不说 apple 了。</p><p><strong>咱还是回过头来看看，一个大学生是怎么能把 iPhone 给换成 C 口的，甚至还一点改装痕迹都没有。</strong></p><p>对了，这期间他还得到了来自中国神秘力量的支持，咱后面细聊。</p><p><img src="https://static.cnbetacdn.com/article/2021/1106/913ae9870081b91.jpg" referrerpolicy="no-referrer"><br></p><p>为了便于大家理解：</p><p>我们这里把 <strong>Lightning 充电插头叫</strong><strong>公 L </strong><strong>，把 Lightning 充电接口叫</strong><strong>母 L </strong><strong>。</strong></p><p>Type-C 同理，<strong>把 C 口充电插头叫</strong><strong>公 C </strong><strong>，把 C 口充电接口叫</strong><strong>母 C </strong><strong>。</strong></p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/01ce8437d00e6c2.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1106/01ce8437d00e6c2.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1106/01ce8437d00e6c2.jpg" referrerpolicy="no-referrer"></a><br></p><p>然后神奇的的地方来了。</p><p>看下面这张图，我们把手机后面那一堆东西变小放进 iPhone 里，这样母 C 就可以替代原来 iPhone 上的母 L，C 口 iPhone 就诞生了。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/e1423e891a1ba25.gif"><img data-original="https://static.cnbetacdn.com/article/2021/1106/e1423e891a1ba25.gif" src="https://static.cnbetacdn.com/thumb/article/2021/1106/e1423e891a1ba25.gif" referrerpolicy="no-referrer"></a><br></p><p>看完是不是有种恍然大悟的感觉，“ 原来如此，看上去也不是很难 ”。</p><p><strong>虽然这个想法朴实无华，一看就懂，但有几个细致的点还是相当有难度。</strong></p><p><img src="https://static.cnbetacdn.com/article/2021/1106/4623358e1322977.png" referrerpolicy="no-referrer"><br></p><p>首先，不破不立，要想塞东西进去，就得把后面这坨东西搞的越小越好。</p><p><strong>比如这根 C to Lightning 的线，你不可能像图里那样，直接把这样一根线塞进 iPhone 里。</strong></p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/d5156260b8feb71.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1106/d5156260b8feb71.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1106/d5156260b8feb71.jpg" referrerpolicy="no-referrer"></a><br></p><p>所以得把母 L 到公 L 再到 公 C 再到母 C 之间的连接关系搞清楚，然后用柔软的排线来建立母 L 直接到母 C 之间的关系。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/c9ee9bebb4b3563.gif"><img data-original="https://static.cnbetacdn.com/article/2021/1106/c9ee9bebb4b3563.gif" src="https://static.cnbetacdn.com/thumb/article/2021/1106/c9ee9bebb4b3563.gif" referrerpolicy="no-referrer"></a><br></p><p><strong>怎么搞清楚这堆线之间的关系呢？</strong></p><p>一根一根的试！</p><p>掏出测电仪，一根根的试出这些线之间的连接关系，然后用排线把这堆连接关系复刻出来。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/cfd2cd9270e0171.gif"><img data-original="https://static.cnbetacdn.com/article/2021/1106/cfd2cd9270e0171.gif" src="https://static.cnbetacdn.com/thumb/article/2021/1106/cfd2cd9270e0171.gif" referrerpolicy="no-referrer"></a><br></p><p>除了试出接口之间的连接关系以外。</p><p><strong>我们还得把手机末端的那个母 L 接口给解剖了。</strong></p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/92e3fa2f99a92e6.gif"><img data-original="https://static.cnbetacdn.com/article/2021/1106/92e3fa2f99a92e6.gif" src="https://static.cnbetacdn.com/thumb/article/2021/1106/92e3fa2f99a92e6.gif" referrerpolicy="no-referrer"></a><br></p><p>手机端母 L 的接口部分大概就是长下面这样，解剖以后把 Lightning 这部分掰到外面。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/9ff03b789bf42be.png"><img data-original="https://static.cnbetacdn.com/article/2021/1106/9ff03b789bf42be.png" src="https://static.cnbetacdn.com/thumb/article/2021/1106/9ff03b789bf42be.png" referrerpolicy="no-referrer"></a><br></p><p>然后在焊枪上加个刀头，把母 L 接口本体和下面黑色的排线分开。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/7baa171e5093627.png"><img data-original="https://static.cnbetacdn.com/article/2021/1106/7baa171e5093627.png" src="https://static.cnbetacdn.com/thumb/article/2021/1106/7baa171e5093627.png" referrerpolicy="no-referrer"></a><br></p><p>分解完母 L 以后，我们再来拆和它对接的公 L。</p><p>拆解的意义就是扔掉那些不必要的包装件，让它和母 L 对接以后变的更小，以便塞进手机里。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1106/d9bfe28da2f924f.gif"><img data-original="https://static.cnbetacdn.com/article/2021/1106/d9bfe28da2f924f.gif" src="https://static.cnbetacdn.com/thumb/article/2021/1106/d9bfe28da2f924f.gif" referrerpolicy="no-referrer"></a><br></p><p>But ……</p><p><strong>这里遇到一个难题，苹果原装的这个公 L 插头质量是真的好了。</strong></p><p>博主把电动磨刀、电锯、电钻什么的都用上了，还是没把它搓开，无奈之下买了一堆第三方插头，才把 Lightning 插头给拆开。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/9B1F935FE0D4DD3D70CFCF2A9914B9E1FF8EF9D6_size10045_w1005_h510.gif" referrerpolicy="no-referrer"><br></p><p>拆开以后，里面的关键人物就是 C94 套件芯片，能充进去电，全靠这玩意。</p><p>所以还得把它解剖了才能进行下一步。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/18FD921742CC1BA3FD6167D1E3303318019A9644_size29_w1005_h436.png" referrerpolicy="no-referrer"><br></p><p><strong>但是，这个原装 C94 是镀了铑的，头硬的很。</strong></p><p>而且还采用了跟比特币一样无法破解的 SHA256 加密算法，所以破解不了。</p><p>这个 C94 套件还不好从线上拆下来，万一没做好弄坏一个成本也贼高。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/CE816B261D7E1FD09CCCE399110C1A1D544E296A_size121_w865_h388.png" referrerpolicy="no-referrer"><br></p><p>这时候，他把目光投到了远在大洋彼岸的 Taobao（ 淘宝 ）。</p><p>这上面就有直接卖 C94 套件的，你连拆都不用拆，关键价格还便宜。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/8F1DE39EA5BDABC03E0E8510CFAF8CD5A009CBC9_size248_w978_h710.png" referrerpolicy="no-referrer"><br></p><p>借助这股神秘东方力量的支持，也不用破解什么 C94 加密算法，顺便还替博主避免了 Apple 的律师函。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/0DFFC2580D43BF62A8301E02375149D7ACEE81F6_size499_w716_h571.gif" referrerpolicy="no-referrer"><br></p><p>运回来以后，搞清楚这个 FakeC94 的逻辑关系，就可以把后面的一坨东西，变成一个小 PCB 板子和排线。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/7042D535882D3FF9BF8E13D7F99F880E5515B898_size41_w1080_h1080.jpg" referrerpolicy="no-referrer"><br></p><p>再来看个工程版的实物图展示。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/81642C996A609C0F58A2A0AD31074CCC8AEF7DAA_size426_w867_h572.png" referrerpolicy="no-referrer"><br></p><p><strong>但是到这还不够，我们得把外面露出来的这堆东西塞进手机里，现在这个还是有点大。</strong></p><p>所以博主就用 3D 打印工具重新设计打印了接口，并且用 Allegro（ 一种 PCB 板子设计软件 ）重新对线路进行设计。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/A5B72B0AA84F943BABB8D9FB3F0D6B32E60132F9_size860_w667_h330.gif" referrerpolicy="no-referrer"><strong><br>终极玩意就是下面这个。</strong></p><p><img src="https://x0.ifengimg.com/ucms/2021_45/AF786B3F09FE538485BF0B80B97F416767D04F7A_size349_w641_h603.png" referrerpolicy="no-referrer"><br></p><p><strong>复杂的线路关系搞清楚了，但还有个临门一脚的问题。</strong></p><p>现在的 Lightning 接口太窄了，不能直接把 C 口替换上去，所以还得找点工具把口子磨的大点。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/FC35D06D23543336BAE34C38CC77B82239A2B03A_size42_w1076_h792.jpg" referrerpolicy="no-referrer"><br></p><p><strong>但是呢……</strong></p><p>你要是硬来，可能会在钻接口的时候把屏幕甚至是玻璃背板震裂，所以博主找了个极细的工具，从里往外慢慢的磨。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/B0EF75464C186F10CA3315A462D3444BBB43AEEF_size110_w411_h312.png" referrerpolicy="no-referrer"><br></p><p>最后，清脆的合上屏幕盖板，整个改造过程就结束了。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/3C4B87B38A0FC2D1C604EEA113FD89DD557F32AF_size4494_w1080_h551.gif" referrerpolicy="no-referrer"><br></p><p>目前，这名博主已经在 eBay 上拍卖了这台原型机，11 月 2 日开卖的时候价格是 3400 美元。</p><p>现在已经飙升到了 10 万美金。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/E46464A440B59D9EF82AFDD7017CFBDFDC56E78F_size304_w1033_h623.png" referrerpolicy="no-referrer"><br></p><p><strong>当然了，这台手机不能更新系统更不能进水，甚至你都不能把它当做一个正常的手机来用。</strong></p><p>不过，博主也把整个项目过程开源了（ 除 PCB 设计原理图外 ），后续还要开发除了 iPhone X 以外其它型号的 iPhone，并且还会把快充和防水加进去。</p><p>视频下面也已经有不少网友在催着给 Airpods & Airpods Pro 换 C 口了。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/73C7EF008BD1AE58875565E84D41BE14DA54427E_size12_w634_h128.png" referrerpolicy="no-referrer"><br></p><p><strong>而回到苹果这边，用一句恨铁不成钢的话来说：</strong></p><p>“ 赶紧把这个全球第一台纯手工打造的 C 口充电 iPhone 寄到 Apple Park 去，让果子睁开眼看看，指导一下他们未来几年 iPhone 的重点发展方向 ”。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/59476B2A34E79914A9CE24619760A14B3036CD6C_size121_w1080_h611.jpg" referrerpolicy="no-referrer"><br></p><p>“ 把 C 口加上，把充电头弄回来，把 Lightning 充电线扔了 ”，环保也有，充电头还回来了，这不皆大欢喜吗？</p><p><strong>为啥苹果不愿意抛弃 Lightning？</strong></p><p>业界猜测最多的原因，就是苹果舍不得 MFi 认证。</p><p>什么是 MFi ？Made For iPad&iPhone&<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">iPod</a></p><p><img src="https://x0.ifengimg.com/ucms/2021_45/D643BD5EF1AE7DCE4A24A40DBEB6F7730A94D778_size40_w837_h563.jpg" referrerpolicy="no-referrer"><br></p><p>就是苹果公司对其授权<a data-link="1" href="https://microsoft.pvxt.net/Xx07X" target="_blank">配件</a>生产商的外置配件的一种标识许可。</p><p><strong>说白了有 MFi 认证，这根线就是在我苹果公司严格监管下生产的，它的电压、电流传输、充电速率等都有一套官方的标准。</strong></p><p><strong>没有 MFi 你的充电线可能就用不了。</strong></p><p><img src="https://x0.ifengimg.com/ucms/2021_45/E2E524D49B8208C000AD0EC90D824D8DCA46DC3D_size174_w478_h475.png" referrerpolicy="no-referrer"><br></p><p>理所当然，MFi 的审核也相当严格。</p><p>包括 MFi 会员申请审核，产品审核、产品测试审核、自建工厂审核、面积不小于2000 平方米审核、员工不少于 50 人审核等等。。<strong><br>据说通过率仅有 2%。</strong></p><p>通过这个认证以后，你工厂生产的每一根 Lightning 充电线，都要内置从苹果那才能买到的 MFi 芯片，每个芯片 2 美元。</p><p><img src="https://x0.ifengimg.com/ucms/2021_45/5200BDAC2A0DDAC2962A08A26C04A5639D651C1E_size102_w1080_h774.jpg" referrerpolicy="no-referrer"><br></p><p>可想而知阿果每年要从这上面赚多少钱……</p><p><strong>不过话说回来，其实多根线少根线是一回事，主要是这么多年了还不抛弃 Lightning 这操作就很不苹果。</strong></p><p>你当初都敢大刀阔斧砍掉 Mac 光驱、HDMI、存储卡接口，把 Mac 激进的换成全 Type-C 口。</p><p>现在，怎么一个小小的 Lightning 接口都不能断舍离，我们这 10 亿的 iPhone 用户真不如你 MFi 认证赚的那些钱？</p><p>所以，阿果，是时候摘下您的 AirPods Max，听听用户的声音，跟 Lightning 做个了断吧。</p>
