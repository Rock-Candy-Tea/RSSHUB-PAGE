Google 开始在 Windows 10 和 Windows 11 平台上测试 Chrome 的新设计，在外观上使其更契合 Windows 11 的视觉风格。<strong>在 Chrome Canary 最新版本目前使用了类似于 Edge 的菜单设计，不过用户需要启用实验 Flag 才能看到这些变化。</strong><br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1002/4a40a97882b6cd6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1002/4a40a97882b6cd6.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1002/cc363f599ef7ea6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1002/cc363f599ef7ea6.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1002/e0931cbaa4c3063.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1002/e0931cbaa4c3063.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1002/a53bdf36796b07d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1002/a53bdf36796b07d.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">现阶段，Google 对 Chrome 的改变幅度还不是很大，只限于菜单。例如，当你在浏览器内任何地方点击右键或打开主菜单时，现在可以看到圆角和阴影效果。</p><p style="text-align: left;">此外，Google似乎也在尝试对 Chrome 浏览器进行其他设计上的改进，因为这个实验性标志增加了标签悬停卡的透明程度。这似乎意味着Google浏览器的窗口或标签栏即将发生更重大的变化。</p><p style="text-align: left;">与 Chrome 浏览器不同，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a> Edge 在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 上支持 Mica 材料。正如你在下面的截图中所看到的，Mica 材料使浏览器的标签区的背景颜色与桌面一致，并创造了一种可以区分多个应用程序的流畅体验。</p><p style="text-align: left;">要启用 Chrome 浏览器的实验性 Windows 11 风格的菜单，请遵循以下步骤。</p><blockquote style="text-align: left;"><p style="text-align: left;">1. 下载Chrome Canary。</p><p style="text-align: left;">2. 打开Chrome://flags菜单。</p><p style="text-align: left;">3. 启用“Windows 11 Style Menus”的 Flag。</p><p style="text-align: left;">4. 重新启动浏览器。</p></blockquote>
