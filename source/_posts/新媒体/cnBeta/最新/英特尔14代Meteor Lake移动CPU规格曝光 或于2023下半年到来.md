按照计划，英特尔将于 2023 下半年推出 U、P 和 H 系列 Meteor Lake 移动 CPU 。<strong>不过 Igor's Lab 最新分享的泄露框图，又让我们知晓了 14 代处理器的一些特性</strong> —— 比如采用三核心集群、最高 14 个物理核心、集成 Xe-LPG 核显、以及支持 LPDDR5X-7467 内存。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0707/550b8f028373168.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（via <a href="https://wccftech.com/intel-14th-gen-meteor-lake-mobility-cpu-details-leaked-coming-in-2h-2023-with-triple-hybrid-core-design-up-to-14-cores-xe-lpg-battlemage-gpu-lpddr5x-7467-support/" target="_self">WCCFTech</a>）</p><p>从时间表来看，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 13 代 Raptor Lake CPU 会先在 2023 上半年到来，然后 14 代 Meteor Lake CPU 会在 2023 下半年到来，且移动平台比桌面平台也略早一些。</p><p>架构方面，12 代 Alder Lake 和 13 代 Raptor Lake 都采用了双核心集群（高性能 P 核 + 能效 E 核），但 14 代 Meteor Lake 可能升级到三核心集群（多个 LP E 核）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0707/677f57578622be4.png" alt="2.png" referrerpolicy="no-referrer"></p><p>从“LP”的标签来看，第三集群类似于能效 E 核，但功耗要低得多。还有一种情况，就是英特尔特指 Meteor Lake 视觉处理单元（VPU）里的 Atom 内核。</p><p>【更新】据 OneRaichu 所述，Meteor Lake 芯片上只有两个 LP-E 内核，且位于 SoC Tile 上 —— 意味着它们就是 VPU 单元所使用的内核。</p><p><img src="https://static.cnbetacdn.com/article/2022/0707/0d7b1c1a5161b2f.png" alt="3.png" referrerpolicy="no-referrer"></p><p>考虑到最高 14 个物理核心，那 -P 和 -H 系列 Meteor Lake 芯片应该会有 6P + 8E（14C / 20T）的 SKU，而 -U 系列应该有 4P + 8E（12C / 16T）的 SKU 。</p><p>值得一提的是，该所有内核都将使用全新的“Intel 4”7nm EUV 工艺节点来制造。</p><p><img src="https://static.cnbetacdn.com/article/2022/0707/50703809e51a9c1.png" alt="3.png" referrerpolicy="no-referrer"></p><p>其次是核显部分，有消息称 Meteor Lake CPU 会用上基于 Battlemage 架构的 Tiled-GPU（如 Xe 2 标签所示）。</p><p>作为参考，现有 Xe-HPG GPU 使用了 Alchemist（Xe 1）架构，而 Meteor Lake 有望迎来 128 EU（执行单元）/ 1024 个 ALU —— 与 Arc A380 独显的 ACM-G11 相当。</p><p>考虑到架构更新与新工艺节点带来的时钟频率提升，我们可预期两者具有同档性能表现，此外据说 tGPU 基于台积电 3nm 制程打造。</p><p>矛盾的是，与之前罗列的信息不同，也有消息称 Meteor Lake 将采用 Gen 12.7 / Alchemist 图形块架构、而不是 Battlemage（Xe 2 品牌）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0707/68d611a2981b75d.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0707/68d611a2981b75d.png" alt="5.png" referrerpolicy="no-referrer"></a></p><p>另外新 GPU 还会迎来扩展游戏模式（Extended Gaming Mode），辅以 4x 编码输出。</p><p>近期还有报道称 Meteor Lake 的 GPU 块可能砍掉 XMX 单元，但保留部分 FP64 特性。</p><p>内存方面，Meteor Lake 移动 CPU 有望支持 7467 Mbps（1 DPC / 最高 64GB）和 DDR5-5600（1 DPC / 最高 96GB）的 LPDDR5 / X 内存。</p><p>通常情况下，高端游戏本或某些工作站会有 64 / 128GB 的内存选项。</p><p>VPU 2.7 算是 Meteor Lake 芯片块上的一个有趣部分，据说它的 Atom 内核能够高性能、低功耗地处理部分 AI 任务（比如 GNA 3.5 消音）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0707/02efd75815314cd.png" alt="6.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://www.igorslab.de/en/intel-meteor-lake-u-p-and-h-exclusive-block-diagram-of-mobile-14-generation-leak/" target="_self">Igor's Lab</a>）</p><p>I/O 配置方面，可以预计会有 PCIe 5.0 。媒体功能方面，应该也有 AV1 硬解码、以及 HDMI 2.1 + eDP 1.4B 核显输出。</p><p>Igor's Lab 爆料称，14 代 Meteor Lake CPU 可为独显提供 PCIe 5.0 x8 通道，辅以三个 PCIe 4.0 x4 的 M.2 NVMe <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 通道。</p><p>不过这 PCIe 5.0 x8 通道可能仅限于 Meteor Lake-H 系列芯片，-P 和 -U 系列芯片或降级到 PCIe 4.0、甚至根本没有 GPU 专享通道。</p><p>其它特性包括 4×雷电 4、2×USB 3 + 10 × USB 2 端口，以及支持最新的 Wi-Fi 6E 无线网络标准。</p><p><img src="https://static.cnbetacdn.com/article/2022/0707/9bfc309eb4d6bd5.png" alt="7.png" referrerpolicy="no-referrer"></p><p><strong>以下是英特尔 Meteor Lake 移动 CPU 阵容的预期规格：</strong></p><blockquote><p>● 三集群 / 混合 CPU 架构（高性能 P 核 / 能效 E 核 / LP-E 内核）</p><p>● P 核升级至 Redwood Cove / E 核升级至 Crestmont 架构</p><p>● -H / -P 系列最多 14C / 20T（6P+8E）、-U 系列最多 12C / 16T（4P+8E）</p><p>● CPU 部分采用 Intel 4（7nm）、tGPU 部分使用台积电 3nm 工艺节点</p><p>● Battlemage Xe-LPG GPU / 高达 128 EU</p><p>● 支持 LPDDR5X-7467 和 DDR5-5200 内存</p><p>● 最高 96 GB DDR5 / 64 GB LPDDR5X 内存容量</p><p>● VPU 部分使用基于 Atom 的 AI 推理内核</p><p>● 可提供 PCIe 5.0 x8 独显专享通道（或仅限于 -H 系列芯片）</p><p>● 辅以三条 PCIe 4.0 x4 / M.2 NVMe SSD 通道</p><p>● 拥有四个雷电 / Thunderbolt 4 端口</p></blockquote><div class="article-relation"><p><strong>相关文章:</strong></p><p><a
