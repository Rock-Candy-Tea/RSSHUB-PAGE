<strong>拿着一盘寿司，乔布斯曾向亚马逊首席执行官杰夫·贝佐斯吹嘘，苹果创造了有史以来最伟大的 Windows 应用程序。随后，乔布斯警告这款软件将会扼杀亚马逊的一个主要收入来源。</strong>而这款应用就是苹果在 2003 年 10 月推出的 iTunes for Windows。乔布斯还表示：“这就像给地狱里的某个人提供一杯冰水”。<br>

src="https://static.cnbetacdn.com/thumb/article/2021/0330/163a631f27a360b.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0330/1a6cf894a1a003f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0330/1a6cf894a1a003f.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">这件轶事来自于新书《<a href="https://www.amazon.com/dp/1250267595/?tag=cult087-20" target="_blank">Working Backwards: Insights, Stories, and Secrets From Inside Amazon</a>》，由亚马逊员工 Colin Bryar 和 Bill Carr 编写，他们两人都是在上世纪 90 年代加入亚马逊的。</p><p style="text-align: left;">在书中，作者讲述了发生在 2003 年秋天的一次会议。贝佐斯、布莱尔和迭戈·皮亚森特尼（跳槽到亚马逊的前<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>副总裁）从西雅图前往库比蒂诺与乔布斯会面。在书中写道：</p><blockquote><p style="text-align: left;">乔布斯和另一名苹果员工迎接我们，然后把我们带进一间不起眼的会议室，里面有一台 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 电脑和两盘外卖寿司。我们就音乐产业的状况进行了非正式的讨论（注：这是苹果推出 iTunes 后不久的事），同时开始对寿司拼盘发起了“攻击”，因为当时已经过了晚餐时间。在用餐巾纸擦了擦嘴之后，乔布斯转入了会议的真正目的，并宣布苹果刚刚完成了他们的第一个 Windows 应用的打造。他平静而自信地告诉我们，尽管这是苹果第一次尝试为 Windows 打造，但他认为这是任何人打造的最好的 Windows 应用。</p><p style="text-align: left;">当时，适用于 Windows 的 iTunes 应用是个大新闻。在此之前，PC 用户通过运行一家名为 MusicMatch 的公司制作的软件在 Windows 机器上连接使用 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">iPod</a>。但乔布斯的性格是希望尽可能多地控制用户体验。这意味着将 iTunes 移植到 Windows 上--这也意味着重新谈判苹果与音乐厂牌的交易。</p><p style="text-align: left;">乔布斯的想法让贝佐斯对 iTunes for Windows 有了新的想法。毕竟两人之前就有过合作。早在1999年，亚马逊就推出了革命性的一键支付功能。乔布斯很喜欢，直接打电话给亚马逊，以100万美元的价格授权给苹果。换句话说，两人算是有了合作关系。</p><p style="text-align: left;">不过在演示结束之后，乔布斯对贝佐斯说：“亚马逊有相当大的机会成为最后一个购买 CD 的地方。这项业务将是高利润的，但规模很小。你将能够为 CD 收取溢价，因为它们会很难找到”。</p><p style="text-align: left;">当时，亚马逊卖了很多CD，因为它们体积小，容易邮寄。因此，乔布斯的这番话对亚马逊来说，并不是真正的商业主张，而是在说我们正在摧毁你的大部分业务。</p></blockquote><p style="text-align: left;">《Working Backwards》的作者认为，乔布斯可能是想激怒贝佐斯。另外一种猜测是可能是通过冲动的行为来怂恿贝佐斯做出一个错误的商业决定，例如立即推出一个构思不周的 iTunes 竞争对手。最终，贝佐斯保持了冷静，会议的其余部分也被证明是平淡无奇的。但本书作者认为，这可能是让贝佐斯意识到一些产品的实体销售很可能会消失的关键一步。</p><p style="text-align: left;">不久之后，亚马逊推出了 Kindle，加大了对云计算的投资，并采取了其他措施，减少了对图书和CD等实体媒体的依赖。</p>
