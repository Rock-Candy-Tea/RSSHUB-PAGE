<strong>全球存储大厂之一的希捷（Seagate），今日推出了主打网络附加存储（NAS）应用的 IronWolf 525 系列 M.2 NVMe SSD 。</strong>这些固态硬盘驱动器采用了 M.2 2280 的标准外形，辅以支持 PCIe 4.0 x4 的高规格主控，可选 500GB / 1TB / 2TB 容量，且具有相当出众的耐用性。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0922/7cb5f5ae182d215.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Seagate <a href="https://www.seagate.com/cn/zh/products/nas-drives/ironwolf-ssd/" target="_self">官网</a>）</p><p>500GB / 1TB / 2TB 的确切型号为 ZP500NM30002 / ZP1000NM30002 / ZP2000NM30002，分别具有 700TBW / 1400 TBW / 2800 TBW 的标称写入耐久度，并且原厂提供了五年有限质保。</p><p>猜测标签底下的主控为群联（Phison）E16 系列的定制型号，搭配高品质 3D TLC NAND 闪存。</p><p><a href="https://static.cnbetacdn.com/article/2021/0922/eefc6eb7b6f086c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0922/eefc6eb7b6f086c.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">官方规格表（PDF <a href="https://www.seagate.com/files/www-content/datasheets/pdfs/ironwolf-525-ssd-DS2083-1-2108CN-zh_CN.pdf" target="_self">传送门</a>）</p><p>性能方面，2TB 版本具有高达 5000 MB/s 和 4400 MB/s 的顺序读写，辅以 740k / 700k IOPS 随机读写参数。</p><p>1TB 版本顺序读写性能与 2TB 在同一水平，但随机读写为 760k / 700k IOPS 。</p><p>500GB 入门版本的顺序读写为 5000 MB/s 和 2500 MB/s，随机读写则是 420k / 630k IOPS 。</p><p><a href="https://static.cnbetacdn.com/article/2021/0922/1400f6a0f5e1480.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0922/1400f6a0f5e1480.png" referrerpolicy="no-referrer"></a></p><p>最后，尽管希捷尚未正式披露 IronWolf 525 系列 PCIe 4.0 M.2 NVMe <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 的确切售价和上市日期，但它很可能取代现有的 IronWolf 510 系列。</p><p>作为参考，500GB / 1TB / 2TB 版本的 IronWolf 510 系列 PCIe 3.0 高性能 M.2 SSD 的售价约为 160 / 320 / 640 美元（合 1035 / 2068 / 4138 RMB）。</p>
