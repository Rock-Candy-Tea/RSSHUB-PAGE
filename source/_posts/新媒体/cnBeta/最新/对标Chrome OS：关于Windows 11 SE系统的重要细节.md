在经历了数周的谣传之后，微软今天正式推出了 Windows 11 SE，以及运行该系统、售价为 249 美元的 Surface Laptop SE。<strong>科技媒体 Neowin 综合了有关该主题的文档，并总结了一些重要的细节，供你参考。</strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1110/2e0d2bdedf8ca18.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1110/2e0d2bdedf8ca18.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>名称</strong></p><p style="text-align: left;">首先从名称本身开始，品牌中的“SE”实际上并不是任何事物的缩写。微软声称它选择这个名称是为了与其他版本的操作系统区分开来，但看到公司为这些特定的首字母安顿下来仍然很奇怪。不幸的是，微软没有透露有关此主题的更多信息。操作系统本身针对<a data-link="1" href="https://c.duomai.com/track.php?k=yUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmYiJ05WZkVHdzZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyUiR" target="_blank">教育</a>环境，尤其是 K-8 年级。</p><p style="text-align: left;"><strong>发售范围</strong></p><p style="text-align: left;">由于国内政府的合规要求，该系统目前并未在国内市场销售。不过，微软表示正在积极考虑，以便于在国内市场推出下一版的 Windows 11 SE。</p><p style="text-align: left;">虽然该公司并未限制零售店销售运行该操作系统的设备，但考虑到相关的应用程序和管理限制，它并不认为这会成为一种趋势。运行该操作系统的机器将从今天开始销售，但预计明年初将推出更多设备。</p><p style="text-align: left;"><strong>竞争对手</strong></p><p style="text-align: left;">微软表示，Windows 11 SE 设备的成本并不高于 Chromebook 的解决方案。但是，当您考虑管理、生产力和辅助功能等其他优势时，Windows 11 SE 实际上更实惠。</p><p style="text-align: left;">Windows 11 SE 包含更多内置辅助功能。无需额外费用，从而减少了评估和购买第三方解决方案的需要。此外，使用 Chromebook 设备的学校需要为管理订阅和服务付费。</p><p style="text-align: left;">微软的每台设备价格包括管理功能 (Intune) 以及 Microsoft <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a> 和 Minecraft：教育版的完整版本，价格与 Chromebook 的仅限管理价格相同。 Microsoft 还提供订阅选项，使管理和生产力更加实惠。</p><p style="text-align: left;"><strong>是否支持升级/降级</strong></p><p style="text-align: left;">微软不建议将 Windows 11 SE 供个人使用，但如果您愿意，您可以购买 Windows 11 SE 设备，彻底删除您的所有数据和操作系统，并全新安装不同的 Windows 许可证。</p><p style="text-align: left;">但是在升级之后，用户将无法返回到 Windows 11 SE。此外，无法从 Windows 11 SE 切换到 Windows 的其他 SKU，并且如前所述，如果您选择全新安装，这将是一次单程旅行。</p><p style="text-align: left;">如果您想知道不同版本的 Windows 11 之间有什么区别，Microsoft 还提供了一个方便的信息图，您可以在下面查看：</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1110/b1ebd5e9a662ef3.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1110/b1ebd5e9a662ef3.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>应用使用</strong></p><p style="text-align: left;">从上面可以看出，Windows 11 SE 无法安装所有类型的应用程序，因为它是为 IT 管理员创建相对限制和无干扰的环境的教育环境量身定制的。</p><p style="text-align: left;">它没有 Microsoft Store，只能通过 IT 管理员启用的 Win32 和通用 Windows 平台 (UWP) 安装以下六类应用程序：</p><p style="text-align: left;">● 内容过滤应用</p><p style="text-align: left;">● 应试解决方案</p><p style="text-align: left;">● 辅助功能应用</p><p style="text-align: left;">● 有效的课堂交流应用</p><p style="text-align: left;">● 基本的诊断、管理、连接性和可支持性应用程序</p><p style="text-align: left;">● 浏览器</p><p style="text-align: left;"><strong>部署</strong></p><p style="text-align: left;">出于管理和部署目的，Windows 11 SE 必然需要 Intune 教育版。拥有 Microsoft 365 订阅并已通过关联的 Intune for Education 订阅管理设备的机构应该不会在他们的流程中产生任何差异。</p><p style="text-align: left;">IT 管理员还可以利用 Windows Autopilot 来预配装有 Windows 11 SE 的设备。 Chrome 的管理控制台不能用于管理 Windows 11 SE 设备。</p><p style="text-align: left;">Windows 11 SE 没有映像文件，因此您无法在非 Windows 11 SE 计算机上安装或切换到它。通过 Azure Active Directory (AAD) 和 Microsoft 帐户 (MSA) 进行身份验证是开箱即用的，对其他形式的身份验证的支持取决于您的基础结构设置。目前，Windows 11 SE 不支持本地身份验证。</p><p style="text-align: left;"><strong>和 Windows 10 S 模式区别</strong><br></p><p style="text-align: left;">对于 Windows 11 SE 和 Windows 10 S 模式的区别，微软概述了以下几点：</p><p style="text-align: left;">● Windows 11 SE 拥有更加简化的用户界面，可最大程度地减少干扰并专注于教育</p><p style="text-align: left;">● 没有适用于 Windows 11 SE 的 Microsoft Store，因此 IT 教育工作者可以控制学生下载哪些应用和工具</p><p style="text-align: left;">● Windows 11 SE 具有内置增强功能，可优化低成本硬件的性能</p><p style="text-align: left;">● 无法从 Windows 11 SE 升级或切换到 Windows 11，而用户可以随时从处于 S 模式的 Windows 10 切换到完整版的 Windows 10。</p><p style="text-align: left;">● S 模式下的 Windows 10 只是为了限制应用程序而构建的，而Windows 11 SE尚未开放增强功能。</p><p style="text-align: left;">● Windows 11 SE 针对机构使用进行了优化，而 S 模式下的 Windows 10 也可供消费者使用。</p><p style="text-align: left;"><strong>云优先</strong></p><p style="text-align: left;">微软强调，Windows 11 SE 被设计为云优先操作系统，这意味着它需要更少的内存，并且在某些情况下可能比传统版本的 Windows 更快。该公司表示，运行该操作系统的设备在关键测试案例中遵守了其“卓越标准”，例如加入课堂、做笔记和提交作业。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1110/0b0bf923541bd23.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1110/0b0bf923541bd23.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">▲ Windows 11 SE 的默认“Bloom”壁纸</p><p style="text-align: left;">Windows 11 SE 中包含的 Microsoft Office Win32 应用程序可以离线使用，OneDrive 还可以本地智能地保存文件。微软已经指出，这个版本的 Windows 11 SE 只允许文件使用 OneDrive 作为云存储选项，尽管这可能会扩展到包括其他选项。</p><p style="text-align: left;"><strong>系统要求</strong></p><p style="text-align: left;">对于那些好奇的人，Windows 11 SE 的系统要求与 Windows 11 完全相同。此外，该操作系统仅适用于具有 4/8GB RAM 和 64/128GB 存储空间的设备。最后，Windows 11 SE 的许可不需要设备最小数量或资格证书，Microsoft 365 或 Office 365 的 A3 或 A5 许可证可用于运行该操作系统的机器。</p>
