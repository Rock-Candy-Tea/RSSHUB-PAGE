<strong>对目前仍使用 Windows Server 20H2 的用户，微软今天发出提醒：该功能更新将于下月补丁星期二（8 月 9 日）达到 End of Service (EOS) 。</strong>微软提醒道：“Windows Server 20H2 版将于 2022 年 8 月 9 日终止服务。2022 年 8 月 9 日之后，这些设备将不再收到包含针对最新安全威胁的保护的每月安全和质量更新”。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2022/0713/b4acb60b88b771e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0713/b4acb60b88b771e.jpg" alt="QQ截图20220713161703.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">为了继续接收安全和质量更新，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>建议更新到最新版本的 Windows Server。Windows Server 20H2 作为 Semi-Annual 渠道版本于 2020 年 10 月 20 日发布，在到达 EOS 日期后将不会收到安全/非安全更新。微软在 5 月份的通知中还透露，Windows Server 半年频道 (SAC) 将于 8 月 9 日停用。</p><p style="text-align: left;">建议仍在运行 SAC 版本的用户切换到长期服务频道 (LTSC)，该频道将成为主要的 Windows Server 发布频道。微软表示：“未来不会有 Windows Server 的 SAC 版本，并且该产品在 2022 年 8 月 9 日之后将不再接收安全更新。Windows Server 正在转向长期服务渠道 (LTSC) 作为主要发布渠道”。</p><p style="text-align: left;">Windows Server LTSC 版本每两到三年发布一次，其用户有权获得 5 年的主流支持以及额外的 5 年扩展支持。还值得注意的是，如果不进行全新安装，则无法从 SAC 升级或更改为 LTSC，因为它们是不同的服务渠道。</p>
