<strong>在向 Dev 通道的 Insider 测试者们推送的最新预览版本中，微软已经为 Windows 11 操作系统引入了一个特殊的 Spotlight 主题。</strong>尽管 Windows Spotlight 是一项很棒的功能，但此前它只能早锁屏界面上使用。而随着其正式登陆 Windows 桌面，广大 Windows 11 用户也有望很快获得更完整的体验。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0730/9d57b72bdd43b23.webp" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Windows <a href="https://blogs.windows.com/windows-insider/2022/07/28/announcing-windows-11-insider-preview-build-25169/" target="_self">Insider Blog</a>）</p><p>微软表示，您现在可通过全新的 Windows Spotlight 主题，在桌面上启用这项特性。</p><p>感兴趣的朋友，可移步至 Dev Channel 的“设置 → 个性化设置 → 个性化 → 主题”进行选择。</p><p>此外您还可以使用 Windows Spotlight 的旋转图像，来制作相应的自定义主题。</p><p><img src="https://static.cnbetacdn.com/article/2022/0730/1c4bffe4b424262.png" referrerpolicy="no-referrer"></p><p>此外 Amanda Langowski 和 Brandon LeBlanc 在博客文章中介绍了 Windows 11 最新预览版本的其它改进，比如修复了可能触发“标准硬件安全不受支持”的 Windows Security 故障。</p><blockquote><p>Windows 11 Insider Preview Build 25169 修复了一个被认为与 Windows Security 有关的 Bug，它会导致实际支持的标准硬件安全功能，被误报为不支持。</p><p>遇到该问题的用户，会在任务栏上看到其显示了一则警告。但在 Windows Security 启动时，又不会提示任何问题。</p><p>随着分批推送，该问题会在将来的 Windows Security 更新后得到化解。</p></blockquote><p>需要注意的是，目前上述变动仅限于 Insider 测试者，微软尚未给出确切的预估到达时间（ETA）。</p><p>在此期间，该公司会努力在更多设备上展开测试，并在收集到足够多的用户反馈后加以改进。</p>
