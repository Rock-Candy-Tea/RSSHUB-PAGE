为了在 Windows 11 平台上推荐更多用户使用基于 Chromium 的新版 Microsoft Edge 浏览器，微软的某些激进做法让不少用户反感。当 Windows 11 用户在 Edge 浏览器上尝试安装 Chrome 的时候，微软就会发出一条警告，试图阻止用户下载。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1202/e3987375a92afbe.webp" alt="yssg06qm.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">正如你在上面的截图中所看到的，这条“安全浏览信息”在 Edge 的稳定版中弹出，它只针对访问 Chrome 下载页面的用户，所以如果你试图安装 Firefox、Vivaldi 或 Opera，你不会得到同样的提醒。</p><p style="text-align: left;">该信息中写道：“Microsoft Edge 运行在与 Chrome 相同的技术上，并额外提供来自<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>的信任”。它还包括一个名为"现在安全使用浏览器"的按钮，它将用户重定向到一个网页，强调Edge的性能、安全和其他优于其他浏览器的优点。</p><p style="text-align: left;">这条信息目前不会出现在大多数稳定使用 Edge 的人身上（除非他们是 A/B 测试组的成员）。在我们的测试中，我们观察到该通知提示是由服务器端的更新触发的，所以它与 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 的构建修订号无关。</p><p style="text-align: left;">还值得指出的是，当用户试图使用任何搜索引擎（Bing或Google本身）访问Chrome的下载页面时，该警报出现了。</p>
