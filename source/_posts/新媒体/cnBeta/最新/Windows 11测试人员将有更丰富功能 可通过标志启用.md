上周，微软宣布 Windows 11 系统在今年 2 月的更新幅度会进一步增强，包括优化 Android
应用体验、改进任务栏等等；现在，微软详细介绍了整个 2022 年的 Windows 11 测试计划。<strong>微软表示计划为 Windows 11
测试人员提供更丰富的功能，即便某些功能可能永远不会上线。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0204/f92a37a78c4e486.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Windows Insider 项目负责人 Amanda Langowski 解释说：“在不断演进中，Insider 人士将会看到我们更加倚重 Dev 频道，并将其作为孵化新想法、致力于长期领先的地方。在某些情况下，部分概念功能可能永远不会上线，但是通过更多的测试实验，我们可以更好地改善体验，并在 Windows 中提供解决方案，真正赋予我们的客户实现更多的能力”。</p><p style="text-align: left;">现阶段微软主要采用 A/B 测试模式，也就是向更广泛的 Dev 频道用户开放之前，先邀请部分 Windows Insider 成员测试该功能。不过在未来，可能会有一些不会被微软完全记录的功能变化。每次微软发布新版本时，<strong>一些 Windows 爱好者都可以关注新功能，利用操作系统中的标志来开启隐藏的功能，看看微软正在试验什么。</strong></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0204/6569d542ee073aa.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Langowski 表示：“我们也认识到，我们的一些技术性较强的 Insider 发现，一些功能在我们试运行的版本中被故意禁用。这是设计上的问题，在这种情况下，我们只会就我们有目的地启用的功能进行沟通，让 Insider 进行尝试并提供反馈”。</p><p style="text-align: left;">微软现在计划给Windows 11测试者一个窗口，他们可以从开发频道切换到Beta频道，以避免即将出现的更多实验性功能。这很可能与2月份发布的功能相吻合，包括Android应用支持、任务栏变化以及重新设计的记事本和媒体播放器应用。</p>
