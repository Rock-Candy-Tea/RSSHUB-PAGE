微软正在向 Beta 和 Dev 通道的测试者们推送 Windows 11 Build 22000.168 编译版本，<strong>可知其改进了集成 Teams 体验的新聊天语言、为企业 AAD 账户用户带来了 Microsoft 365 小部件、以及一系列 bug 修复。</strong>遗憾的是，该版本仍未引入对 Android 应用程序的运行支持，这让我们不禁怀疑该功能是否会在年内及时发布。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0625/97c10b9160114a7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0625/97c10b9160114a7.jpg" referrerpolicy="no-referrer"></a></p><p>随着预定正式发布日期的邻近，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>也开始更加专注于 bug 修复、而不是为 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 编译版本引入新功能。</p><p>此外 Dev 通道将做好面向未来版本的测试准备，该公司已向 Dev 测试用户预告新发布的版本将“不太稳定”。</p><p>首先，微软很高兴地宣布，Chat with Microsoft Teams 现在支持多种语言。</p><blockquote><p>即日起，你可在 Microsoft Teams 中设置跟随系统语言、或单独在聊天应用程序中使用相应的语言。</p><p>包括阿尔巴尼亚语、阿拉伯语、阿塞拜疆语、巴斯克语、保加利亚语、加泰罗尼亚语、中文（简体）、中文（繁体）、克罗地亚语、捷克语、丹麦语、荷兰语、英语（英国）、美国英语、爱沙尼亚语、菲律宾语、芬兰语、法语、法语（加拿大）、加利西亚语、格鲁吉亚语、德语、希腊语、希伯来语、印地语、匈牙利语、冰岛语、印度尼西亚语、意大利语、日语、哈萨克语、韩语、拉脱维亚语、立陶宛语、马其顿语、挪威语（博克马尔语）、挪威语（尼诺斯克语） )、波兰语、葡萄牙语（巴西）、葡萄牙语（葡萄牙）、罗马尼亚语、俄语、塞尔维亚拉丁语、斯洛伐克语、斯洛文尼亚语、西班牙语（西班牙）、西班牙语（墨西哥）、瑞典语、泰语、土耳其语、乌克兰语、以及巴伦西亚语。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0828/5498225e5b44e24.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">小部件可显示“经常打开”和“趋势”卡片</p><p>其次，微软正在推出新的 Microsoft 365 小部件，其为企业用户带来了相关文档、新闻和会议记录（以及更多内容）。</p><p>不过只有当您在运行 Windows 11 的 PC 上使用 AAD 账户登录时，新的 Microsoft 365 小部件才会被启用。</p><p>在信息过载的时代，个性化列表将帮助用户更好地掌握工作安排、并轻松获取所需的内容。有需要的朋友，只需点击打开、选择、然后添加。</p><p><a href="https://static.cnbetacdn.com/article/2021/0828/8d10fc1d6a35157.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0828/8d10fc1d6a35157.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">Microsoft Store资源库界面迎来了大修</p><p>如预期那样，微软还在修复 Insider 测试者们上报的一些影响系统正常运行的 bug 。</p><blockquote><p>● 搜索：解决了最近几个版本中影响支持手写笔的设备的潜在问题，该 bug 导致无法在搜索结果中启动相关应用程序。</p><p>● 设置：修复了在‘设置’搜索框中键入某些关键词时，可能导致‘设置’应用崩溃的问题。</p><p>● 小部件：修复使用‘家庭’小部件时，可能意外看到‘连接设备以查看屏幕活跃时间’消息的问题。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0828/e1e9c2e9e80008e.gif" alt="3.gif" referrerpolicy="no-referrer"></p><p style="text-align: center;">鼠标在 Spotlight 项目上悬停时可查看更多信息</p><p>此外微软修复了 Windows 11 上与 Microsoft Teams 相关的一些问题，比如：</p><blockquote><p>● 修复视频通话期间，可能偶尔出现画面冻结或黑屏的问题。</p><p>● 修复在通话之间切换，通话保持无法正确启动，导致前后两个通话都继续传输音频与视频流的问题。</p></blockquote><p>最后还有一些已知问题，想了解的朋友可移步至微软官网查看（<a href="https://blogs.windows.com/windows-insider/2021/08/27/announcing-windows-11-insider-preview-build-22000-168/" target="_self">传送门</a>）。如果一切顺利的话，其有望在接下来几个月内陆续得到解决。</p>
