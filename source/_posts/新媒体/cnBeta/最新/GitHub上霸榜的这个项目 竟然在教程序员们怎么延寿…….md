4月25日，哈啰出行官方发布微博宣布品牌升级计划，正式更名为“哈啰”，并发布了新的Logo和品牌标语。根据哈啰公布的信息来看，首先是新的Logo品牌名称去掉了“出行”二字。哈啰认为随着更多新业务涌现，哈啰多元化业务布局更为清晰，因此决定将“哈啰出行”改为“哈啰”。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0425/ecec628ce866d17.jpg" referrerpolicy="no-referrer"><br></p><p>最近在全世界最大的程序员交友网站 GitHub 上，一个听起来非常诡异的项目在提交后<strong>短短 4 天内屠榜GitHub，登顶热榜前五，收获了 4.3k+ stars。</strong>这个项目叫做《 程序员延寿指南 》……没错，继上次那份离谱的《 程序员做饭指南 》之后，GitHub 上又多了个活整活项目，成功杀进热榜前五，<strong>看来最近的程序员们已经越来越关注生活了。</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0425/9988a5ce5ef8686.gif" referrerpolicy="no-referrer"><br></p><p>其实说这份《 程序员延寿指南 》是个整活项目，还是有些委屈它了，在世超看来它应该叫做<strong>《 程序员<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C752%2C12397" target="_blank">养生</a>指南 》</strong>更贴切一点。</p><p>它用每个程序员都看得懂的语言逻辑，如何更加健康的生活，给大家做了一个非常全面的科普，<strong>这份指南是适合每个人阅读的 “ 健康读物 ”</strong>。</p><p><img src="https://static.cnbetacdn.com/article/2022/0425/6ff24e5f9127ca6.jpg" referrerpolicy="no-referrer"><br></p><p>《 程序员延寿指南 》参考了大量医学文献资料<strong>，用非常客观、严谨的量化数据，来告诉我们如何降低死亡率，增加寿命~</strong></p><p>正所谓生命在于运动，想要活的久，首先要控制的就是自己的饮食和运动。</p><p>《 程序员延寿指南 》开篇就把每天需要的食物、睡眠等等，并且用精确的数据给量化了，并且告诉你保持这些优秀的生活习惯，能够增加多少年的寿命。</p><p><strong>下文单位</strong></p><p><strong>ACM : All-Cause Mortality / 全因死亡率 ▼</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0425/6824f6fad0fb77a.jpg" referrerpolicy="no-referrer"><br></p><p>估计差友们初看这份简单的养生指南，马上会有人跳出来反对：</p><p>凭什么说我不抽烟就能多活 11~12 年，这些数据不会是作者闭着眼睛瞎写的吧。</p><p>会这么问的差友，<strong>严重低估了程序员的严谨~</strong></p><p>你们可别小看这份短短的养生指南，《 程序员延寿指南 》可是查阅了大量医学资料得出的客观结论。</p><p>就拿这个抽烟来说，关于抽烟能够延长寿命这件事，是梅斯医学在长期观察了 42416 名男性和 86735 名女性对照实验后得出的结论。</p><p><img src="https://static.cnbetacdn.com/article/2022/0425/9b41bfad3850645.jpg" referrerpolicy="no-referrer"><br></p><p>《 程序员延寿指南 》的每一项建议，<strong>背后都有一份学术研究作为支撑的</strong>，大家可以在这个项目中，找到每一条建议背后对应的学术报告。</p><p>再给大家举个例子吧：</p><blockquote id="0QO1T4T6">肥宅快乐水，应该是每一位程序员必备的饮料了，但是根据《 Science 》上刊登的一篇对可乐危害健康的论文显示:<p>每天喝一杯含糖的饮料，会增加 7% 全因死亡率 ; 如果每天喝两杯，死亡率直接翻 3 倍，达到惊人的 21%。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0425/0b2ff2cc35eee29.jpg" referrerpolicy="no-referrer"><br></p><p>并且根据另外一份刊登在国际顶级期刊《 BMJ 》上的论文证明，含糖饮料会增加患癌的风险！</p><p>并且不仅仅是含糖的饮料，就算你喝奶茶患相关癌症的风险也会增加 18%，所以世超非常建议大家在日常生活中控制一下糖分的摄入。</p><p>《 程序员延寿指南 》还破解了很多朋友圈流传的谣言:</p><p>比如喝 100% 纯<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C752%2C14421" target="_blank">果汁</a>会更健康这件事。</p><p><img src="https://static.cnbetacdn.com/article/2022/0425/749ac85099d8c5f.jpg" referrerpolicy="no-referrer"><br></p><p>没想到吧，所谓的 100% 纯果汁的说法，可能从一开始就是果汁厂商们跟消费者描述的 “<strong>消费主义陷阱</strong>” 罢了。</p><p>除了饮食方面需要注意的这些事项，多晒太阳也能够帮助大家延年益寿。</p><p>根据丹麦的一项研究表示，即使你过度晒太阳诱发了皮肤癌的患者，<strong>平均寿命也比普通人长了 6 岁。</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0425/8bb116e8b9cb73a.jpg" referrerpolicy="no-referrer"><br></p><p>这个结论，估计能够刷新很多人对 “ 防晒爱好者 ” 的世界观……</p><p>在运动方面，《 程序员延寿指南 》的论证对比了各种运动对延长寿命的作用，并且找到了两种最省事高效的延长生命的运动：<strong>挥拍运动和走路。</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0425/377a5390e008426.jpg" referrerpolicy="no-referrer"><br></p><p>这两项运动对全身锻炼都比较注重，并且相对的还不怎么 “ 累人 ”，被权威医学杂志 “ 柳叶刀 ” 誉为 “ 性价比 ” 最高的运动。</p><p>看完这份指南，绝对能够让各位受益匪浅，这些平时需要查阅大量医学资料的知识，<strong>《</strong><strong>程序员延寿指南 》直接用最 “ 直给 ” 的方式告诉大家了。</strong></p><p>并且一些 “ 冷知识 ” 能够修正很多人不良的生活习惯，比如不爱刷牙。</p><p>经常不刷牙的人：癌症、慢性阻塞性肺病以及肝硬化的风险分别会增加 9%、12% 和 25%，过早死亡的风险增加 25% ！</p><p>没想到吧，刷牙这么一个小小的习惯，竟然还能影响你的寿命……</p><p>还有习惯工作日晚睡，然后周末疯狂补觉。</p><p>根据加拿大的研究发现，其实早睡或者过晚睡觉都会影响健康，并且早睡增加的全因死亡率比晚睡增加的还要高。</p><p><img src="https://static.cnbetacdn.com/article/2022/0425/2ea5e614a28416a.jpg" referrerpolicy="no-referrer"><br></p><p>所以保证自己每天 7 ~10 个小时的睡眠时间，是最好的。</p><p>不过这项研究，还存在很多局限性，数据也不够客观，建议大家根据自己的实际情况监控一下自己的睡眠。</p><p>所以别看这份《 程序员延寿指南 》只给出了短短的几条建议，这些建议背后的证据论证，占中了这个项目 90% 的内容。</p><p><img src="http://dingyue.ws.126.net/2022/0425/2d9dca67g00raupxc022sc000if00aqc.gif" referrerpolicy="no-referrer"><br></p><p>这些有理有据的建议，绝对比你在网上看到的那些所谓的专家建议和朋友圈刷到的养生知识科普，靠谱多了！</p><p>做饭、养生这些词，最近频繁的出现在 GitHub 的热榜上，看来程序员们已经在 “<strong>活的健康 ” 这件事上也卷起来了，</strong>这也是一种不错的改变~</p><p>感兴趣的差友们，可以直接去认真阅读《 程序员延寿指南 》的项目原文。</p><p><strong>官网链接</strong>：</p><p><a href="https://github.com/geekan/HowToLiveLonger"
