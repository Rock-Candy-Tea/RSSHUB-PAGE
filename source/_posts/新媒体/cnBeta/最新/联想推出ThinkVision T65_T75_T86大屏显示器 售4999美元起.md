为了刚刚发布了三款 ThinkVision“大尺寸显示器”新品，旨在帮助用户将墙壁改造成交互式的屏幕。<strong>无论是在家办公，还是教室或会议室，ThinkVision T65 / T75 / T86 都能够很好地满足需求。</strong>屏幕本体具有 4K 分辨率、400 尼特亮度、60Hz 刷新率、覆盖 99% sRGB 色域，外加防眩光涂层 + 能够根据室内条件自动调节亮度的环境光传感器。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1217/04d172bf05925b3.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1217/04d172bf05925b3.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：Lenovo <a href="https://news.lenovo.com/pressroom/press-releases/thinkvision-large-format-displays-deliver-uncompromised-collaboration/" target="_self">官网</a>）</p><p>值得一提的是，ThinkVision T65 / T75 / T86 支持手写笔交互，且红外触控可实现 ±1 毫米的精度。顶部有一枚 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_110018%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">4K</a> 网络摄像头，具有 122° 视野 / 4X 数码变焦特性。</p><p><a href="https://static.cnbetacdn.com/article/2021/1217/088c1b46b91dd69.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1217/088c1b46b91dd69.png" alt="7.png" referrerpolicy="no-referrer"></a></p><p>在无需使用时，还可合上物理隐私挡板、或将 USB 线缆轻松移除。与之相结合的，还有 8 麦克风阵列 + 一对 15W 的扬声器。</p><p><a href="https://static.cnbetacdn.com/article/2021/1217/3d15543205e80c5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1217/3d15543205e80c5.jpg" alt="6.jpg" referrerpolicy="no-referrer"></a></p><p>更棒的是，这些大屏显示器并不是只有接入 PC / 笔记本电脑时才可使用。因为联想为其内置了嵌入式的四核 SoC，4GB RAM + 64GB ROM，且预装了 Android 9 操作系统。</p><p><a href="https://static.cnbetacdn.com/article/2021/1217/0900940c2cfe3c2.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1217/0900940c2cfe3c2.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>预装应用中包括了智能白板和投屏工具，且底部可弹出一个启动器。左侧边栏为功能导航，以便用户访问快速设置和注释等内容。另有支持本地文档的文件管理器，以及录屏 / 录音、计算器、时钟和日历小部件。</p><p><a href="https://static.cnbetacdn.com/article/2021/1217/19f6732c5750483.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1217/19f6732c5750483.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>连接方面，除了 USB 2.0 / 3.0、USB 3.2 Gen 1 Type-C，还有 HDMI 2.0 输入 / 输出、VGA / 音视频（AV）输入、S/PDIF 数字音频 + 线路输出、麦克风输入、双以太网 + RS-232、TF 读卡器等。</p><p><img src="https://static.cnbetacdn.com/article/2021/1217/fe0da23a6bca67d.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">三个 HDMI + DP 1.2 + VGA 视频输入</p><p>无线方面，该系列大屏显示器支持 802.11ac Wi-Fi 和蓝牙 5.0，且附赠了 W20 无线 HDMI 适配器，方便连接笔记本电脑、平板电脑、智能<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>或其它设备（兼容 USB-C / HDMI 端口）。</p><p><a href="https://static.cnbetacdn.com/article/2021/1217/14abca0dd068a60.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1217/14abca0dd068a60.jpg" alt="5.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">前置 USB 触控 / HDMI 输入 / USB 3.0 / USB 3.2 Gen 1 Type-C</p><p>当室内无人走动时，传感器还可让 ThinkVision 大屏显示器自动关机以节省能源，并在有人靠近时再次唤醒。</p><p>最后，65 / 75 / 86 英寸的 ThinkVision T65 / T75 / T86 大屏显示器将于 2022 年 4 月上市，售价分别为 4999 / 6999 / 7999 美元（约 3.2 / 4.5 / 5.1 万 RMB）。</p>
