<div>   
<strong>随着 AMD 霄龙（EPYC）7003 系列 Milan-X 服务器处理器的正式发货，Phoronix 也汇总分享了首批 3D V-Cache 堆叠缓存加持的新平台的基准测试数据。</strong>早在 2021 年 4 季度，微软就已宣布要把 Azure HBv3 云平台升级到最新兼容的 Milan-X 芯片。由于无需对系统实施其它必要的修改，这意味着 EPYC 7003 系列服务器 CPU 能够轻松为其带来显著的性能改进。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0323/86b4e12ca6019c8.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">针对云工作负载的 EPYC Milan-X 处理器升级，让<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a> Azure HBv3 显着获益。</p><p>显然，这家科技巨头意识到了 Milan-X 服务器处理器对于跨多个 VM 部署的大型 HPC 工作负载的非凡影响力。</p><p><a href="https://static.cnbetacdn.com/article/2022/0323/fd9b82461a61a05.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0323/fd9b82461a61a05.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>更棒的是，微软暂不打算修改 HBv3 云虚拟机服务的报价。因为该公司将之视作原地升级，而不是一轮完整的硬件迭代。</p><p>如此一来，Microsoft Azure 也能够在面对 Google Cloud Tau VM 等竞争平台时更显自信。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0322/ca582349a5b8a56.gif" alt="1.gif" referrerpolicy="no-referrer"></p><p>对于 Azure HBv3 客户来说，他们将能够充分利用相关优化代码来压榨 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 3D V-Cache 的性能潜力。</p><p>不过为了更直观的参考，<a href="https://www.phoronix.com/scan.php?page=article&item=amd-milanx-hbv3&num=1" target="_self">Phoronix</a> 的 Michael Larabel 还是分享了一系列基准测试对比图表。</p><p><a href="https://static.cnbetacdn.com/article/2022/0323/dec5f4ad2821c66.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0323/dec5f4ad2821c66.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>与 AMD 和微软的官方数据相比，Phoronix 引用了通过 MPI 跨多个 VM 开展的大量基准测试，使用了 CentOS 8 和各种工作负载来分析标准 HBv3（Milan）和增强型 HBv3（Milan-X）的场景差异。</p><p>首个测试选择了 Azure standard_HB120-64r3_v3（64 核 CPU）和 standard_HB120rs_v3（120 核 CPU）这两个实例，并同 2021 年 11 月之前的成绩进行了对比。</p><p><a href="https://static.cnbetacdn.com/article/2022/0323/4896afa5d9e69ae.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0323/4896afa5d9e69ae.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>亮点包括：Milan-X 增加的 3D V-Cache 缓存，较标准的 Zen 3“Milan”CPU 在混合工作方面，有着显著的性能优势。</p><p><img src="https://static.cnbetacdn.com/article/2022/0323/310eb9f12282bbe.png" alt="5.png" referrerpolicy="no-referrer"></p><p>在 AMD 吹捧的 WRF 天气预报开源软件应用场景下，Milan-X 也凸显了相当高的价值。以 CONUS 2.5km 模型为例，其运行时间缩减了 10% 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0323/23b7fa2362f5902.png" alt="6.png" referrerpolicy="no-referrer"></p><p>OpenFOAM 流体动力学（CFD）计算方面，Milan-X 霄龙处理器的表现也相当亮眼。当然，其优势取决于是否有足够大的数据集工作负载（聊胜于无）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0323/52664af62ff2816.png" alt="78.png" referrerpolicy="no-referrer"></p><p>开源包的基准测试也有所改进，且 Milan-X 在其它不受 AMD / 微软支持的 HPC 工作负载中也表现出了有价值的改进。</p><p><img src="https://static.cnbetacdn.com/article/2022/0323/c3f7ea5a6a4ced7.png" alt="9-10-11.png" referrerpolicy="no-referrer"></p><p>Zstd 在最高压缩设置下的优势显著，特定成像工作负载和代码编译的速度也是如此。</p><p><img src="https://static.cnbetacdn.com/article/2022/0323/9bc302c1394de25.png" alt="12-13.png" referrerpolicy="no-referrer"></p><p>与<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>（Intel）至强可扩展（Xeon Scalable）Ice Lake 竞品相比，AMD Milan-X 的领先优势相当显著。</p><p><img src="https://static.cnbetacdn.com/article/2022/0323/cdf03d01da19f15.png" alt="14-15-16.png" referrerpolicy="no-referrer"></p><p><img src="https://static.cnbetacdn.com/article/2022/0323/30d5484f13d9d4d.png" alt="17-18-19.png" referrerpolicy="no-referrer"></p><p><img src="https://static.cnbetacdn.com/article/2022/0323/febd2a6616b0051.png" alt="20-21-22.png" referrerpolicy="no-referrer"></p><p>最后，AMD 下一代霄龙 7004“Genoa”服务器处理器，将于 2022 下半年推出。</p><div
