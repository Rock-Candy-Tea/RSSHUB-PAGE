User Benchmark数据库中发现了英特尔Raptor Lake-P的14核心4.2GHz的时钟工程样品CPU，显示出比现有12代酷睿i9-12900HK CPU更高的性能。被发现的样品是在"ERB"评估参考板上评估的，该处理器被称为英特尔Raptor Lake-P芯片，有14个内核和20个线程。<br>
 <p>这是一个6个P-Core和8个E-Core的配置组合，P核是基于新的Raptor Lake核心，而E核是基于Gracemont架构。预计英特尔将增加E-Core的数量，但并不是所有的SKU都是如此。这个SKU的配置与高端Alder Lake-P芯片相同。</p><p><a href="https://static.cnbetacdn.com/article/2022/0416/4a6ec8510a899b2.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0416/4a6ec8510a899b2.png" title alt="Intel-Raptor-Lake-P-14-Core-20-Thread-Mobility-CPU.png" referrerpolicy="no-referrer"></a></p><p>预期的SKU如下：</p><p>Intel Core i9 K-Series (8 Golden + 16 Grace) = 24 Cores / 32 Threads / 68 MB</p><p>Intel Core i7 K-Series (8 Golden + 8 Grace) = 16 Cores / 24 Threads / 54 MB</p><p>Intel Core i5 K-Series (6 Golden + 8 Grace) = 14 Cores / 20 Threads / 44 MB</p><p>Intel Core i5 S-Series (6 Golden + 4 Grace) = 14 Cores / 16 Threads / 37 MB</p><p>Intel Core i3 S-Series (4 Golden + 0 Grace) = 4 Cores / 8 Threads / 20 MB</p><p>Intel Pentium S-Series (2 Golden + 0 Grace) = 4 Cores / 4 Threads / 10 MB</p><p>Raptor Lake-P CPU的基本时钟速度为2.5 GHz，提升时钟速度为4.2 GHz。该平台本身利用板载Iris Xe图形和16GB LPDDR5内存，以5200Mbps的速度运行。参与跑分的机器使用了一个单一的操作系统驱动器，即WD NVMe <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a>。</p><p><a href="https://static.cnbetacdn.com/article/2022/0416/0b17612ac536347.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0416/0b17612ac536347.png" title alt="图片.png" referrerpolicy="no-referrer"></a></p><p>在性能方面，英特尔Raptor Lake-P移动CPU的单核得分是202分，全核测试得分是2052分。UserBenchmark以其对<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a>的CPU对比数据不准确闻名，所以我们不打算将泄露的处理器性能与任何Ryzen CPU进行比较，而是使用英特尔自己的Alder Lake CPU进行比较。基准数据库包括酷睿i9-12900HK的结果，它具有相同的14个核心和20个线程的CPU配置，运行的基本时钟为2.9GHz，提升时钟为4.3GHz。</p><p>尽管如此，英特尔Raptor Lake CPU最终在单核和全核测试结果中都更快。Raptor Lake-P ES样本在单核和全核工作负载中也比更高时钟的酷睿i5-12600K快。这是令人印象深刻的，因为移动芯片有功率限制，而'K'系列零件是完全解锁的，该CPU输给了酷睿i7-12700K，但这只是在全核心性能方面，因为该CPU拥有比E-cores更多的性能核心。</p><p>英特尔最近刚刚推出了Alder Lake-P系列，并将很快为发烧级和工作站笔记本电脑推出高端Alder Lake-HX部件。基于此，我们可以预期Raptor Lake-P CPU将在2022年底或2023年初推出。新的Raptor Lake核心和增加的缓存尺寸等改进肯定会有助于提高性能，最终的芯片也有望以非常高的时钟速度运行，所以认为这是移动平台上即将出现的一个预告。</p>
