在 Windows 10 操作系统上，微软允许用户在深色和浅色主题之间进行切换，且几乎所有原生应用程序、设置、任务栏、通知中心、文件资源管理器等都提供了适配。尴尬的是，即使某些第三方应用程序都已经适配 Windows 10 的全局主题设置，<strong>Windows 10 最常用系统组件之一的“Windows 搜索”，却依然有着刺眼的白色背景。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1106/d8432167015f79a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1106/d8432167015f79a.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（截图 via <a href="https://www.windowslatest.com/2021/11/05/microsoft-is-rolling-dark-mode-for-windows-10-search/" target="_self">Windows Latest</a>）</p><p>与 Microsoft Store、Windows 安全、Your Phone、时钟、计算器等原生应用程序相比，Windows 搜索却严重破坏了 Windows 10 操作系统中的暗色主题体验。</p><p>好消息是，为了修复这方面的体验一致性，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>似乎终于在服务器端作出了更新调整，以正确启用对 Windows 10 搜索组件的暗模式支持。</p><p><a href="https://static.cnbetacdn.com/article/2021/1106/4af322d5a57eb22.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1106/4af322d5a57eb22.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>Windows Latest 的 Mayank Parmar 实测后发现，这项变动正在向 Windows 10 2004 / 20H2 / 21H1 及更高系统版本的用户推送。</p><p>整个 Windows 搜索窗口、搜索面板中 / 右键上下文菜单、甚至由必应（Bing）提供支持 Web 搜索，现都完全支持黑色背景。</p><p>无论过滤器（文档 / 设置 / 图片）如何，所有搜索结果也都通过深色背景呈现，以匹配 UI 的其余部分，比如“设置”和“开始”菜单。</p><p><a href="https://static.cnbetacdn.com/article/2021/1106/ea920801c53d6aa.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1106/ea920801c53d6aa.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>不过与最新推出的 Windows 11 操作系统相比，Windows 10 搜索组件的黑色背景还是有些不同，尤其是不具有 Windows 11 独有的（WinUI 组件）圆角元素。</p><p>最后，如果你想在 Windows 10 操作系统上启用全局暗色模式，还请移步至“设置 -> 个性化 -> 颜色”，然后选择深色。</p>
