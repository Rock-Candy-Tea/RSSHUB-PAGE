<strong>谷歌悄然从 Chrome Web Store 下架了热门扩展程序 ClearURLs。</strong>ClearURLs 是一款保护隐私的浏览器插件，它可以自动删除 URL 中的跟踪元素。据其开发者称，这可以帮助保护你在浏览互联网时的隐私。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0325/ca92310f2334aea.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0325/ca92310f2334aea.jpg" alt="lo92inp3.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">图片来自于 <a href="https://www.appinn.com/clearurls-for-chrome-and-firefox/" target="_blank">小众软件</a></p><p style="text-align: left;">ClearURLs 是一款适用于 Google Chrome 和 Mozilla Firefox 的网络浏览器插件，其任务是删除 URLs 中的跟踪位。许多网站都有冗长的URL，其中的额外参数没有任何功能价值，只是用于跟踪目的。这尤其适用于存在于新闻邮件中的链接，例如：</p><p style="text-align: left;">https://example.com?utm_source=newsletter1&utm_medium=email&utm_campaign=sale&some_other_tracking_bits=...</p><p style="text-align: left;">有趣的是，谷歌搜索结果 URL 也不例外。当用户点击某个图片搜索结果的时候，谷歌并不会立即将您发送到其网页的原始 URL，而是一个中间 URL，将您重定向。</p><p style="text-align: left;">https://www.google.com/url?sa=i&url=https%3A%2F%2Fwww.bleepingcomputer.com%2F&psig=AXXXXXYAWa&ust=1616XXXXXXX&source=images&cd=vfe&ved=0CXXXXe-p3XXX。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0325/486d52f4e9c10b2.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0325/486d52f4e9c10b2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">上面 URL 中的额外参数（ved、cd等）只是用于分析的跟踪位和推荐人。ClearURLs 的设计是为了过滤掉 URL 中的这些跟踪参数，提高用户的在线隐私。如果所有的链接都被去掉这些无关的追踪数据，那么它们最终的长度会显得相当简单，只由必要的部分组成。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0325/766df80cc3f6878.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0325/766df80cc3f6878.jpg" alt="QQ截图20210325115053.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">不过昨晚，谷歌从 Chrome Web Store 上移除了该扩展程序，现在用户访问该页面会跳出 404（未找到）错误信息。有用户留言到：“是的，ClearURLs在7小时前就被谷歌封杀了”。还有用户反馈道：“原因很荒谬，可能只是因为ClearURLs损害了谷歌的商业模式而假装的”。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0325/4cb455fb33a8851.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0325/4cb455fb33a8851.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">ClearURLs 背后的开发者 Kevin Roebert 说：“<strong>ClearURLs的使命是防止通过URL进行追踪，这也是谷歌赚钱的方式。我认为，ClearURLs现在有这么多用户，对谷歌来说是不受欢迎的，他们希望看到这个插件永久消失</strong>”。开发者向谷歌提出上诉，反对封杀该扩展，并听取了谷歌的意见。</p><p style="text-align: left;">在开发者分享的电子邮件副本中，谷歌声称该扩展的描述“过于详细”，违反了 Chrome Web Store 的规则。Roebert 继续说道：“提到所有帮助开发和翻译ClearURLs的人是违反谷歌规则的，因为这可能会让用户迷惑。这是多么的可笑”。</p><p style="text-align: left;">谷歌还表示，该扩展的描述没有提到它包含某些功能，如设置导入/导出功能、日志功能和捐赠按钮，这是误导。谷歌还在邮件中称，该扩展不必要地要求剪贴板 Write 权限。对此Roebert 反驳道：“但事实并非如此，我在 Chrome Web Store 开发者仪表板中对每个权限的描述已经有一年多了”。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0325/c7bf2e700aaaff0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0325/c7bf2e700aaaff0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在接受外媒 BleepingComputer 采访的时候，Roebert 表示“事实证明，从几个版本开始，其实已经不需要这个功能了，因为我改用了另一种复制到剪贴板的方法。所以，这个权限还是ClearURLs早期版本的遗迹”。这位开发者也表示，谷歌在邮件回复中提出的一些建议是自相矛盾的。</p><p style="text-align: left;">他表示：“据说 ClearURLs 的描述有误导性，因为它太详细了，而且描述了不相关的东西。到底什么是不相关的描述，谷歌并没有和我沟通。所以我很难解决这个问题”。有趣的是，谷歌的这一建议与他们在给开发者的同一封邮件中的另一个建议相矛盾，后者的内容是对功能的描述缺乏细节。</p><p style="text-align: left;">谷歌说，该扩展程序的功能（捐赠，徽章，记录，导出/导入）描述不够详细，因此欺骗了用户。开发者表示：“这几乎读起来像个笑话。在安装过程中，没有用户会认真关心是否有捐献的方式，是否有徽章指示器，是否有调试用的日志，或者是否有保存和恢复设置的功能”。</p>
