<strong>出于多重安全原因，在日前发布的 Firefox 90 版本中 Mozilla 宣布正式放弃对 FTP 的支持。</strong>FTP 是文件传输协议的简称，是实现互联网上许多文件下载或上传活动的技术。目前，Firefox 和 Chrome 两大浏览器均已停止对 FTP 的支持。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0722/342933002a581b7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0722/342933002a581b7.jpg" alt="oevph0cj.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">和 HTTP 不同的是，FTP 并没有一个和 HTTPS 对等的协议，因此在安全方面存在诸多隐患。最终，Mozilla 和 Google 做出了一个艰难的决定，就是将 FTP 支持从他们的浏览器中完全删除。Mozilla 解释说，FTP 虽然简单方便，但也是一个等待发生的安全恶梦。连接很容易被欺骗，数据也很容易被修改以满足黑客的需要，这使得该协议在默认情况下几乎是不安全的。</p><p style="text-align: left;">Firefox 90 完全停止了对 FTP 的支持。从 Firefox 90 开始，如果你安装了一个 FTP 应用程序，那么所有的 FTP 链接将被推送到外部应用程序。</p><p style="text-align: left;">在 Firefox 90 新版本中，引入了 “about:third-party” 设定页面，以帮助用户识别由第三方模块和应用程序引起的兼容性问题。<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 版 Firefox 浏览器还提供了一个新的选项，即使在 Firefox 浏览器没有运行的情况下，也可以使用后台服务默默地安装更新。其目的是通过更新提醒尽量减少干扰，可以通过设置>常规> Firefox 更新查看和配置该设置。</p>
