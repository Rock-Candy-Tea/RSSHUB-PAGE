几天前，我们报道过微软尝试为 Windows 11 操作系统新引入的壁纸贴纸功能和平板电脑模式改进。今天早些时候，有人分享了与任务栏相关的改进，以及 Windows 11 开始向不受支持的 PC 用户予以弹窗提醒。<strong>现在，我们又看到了 Windows 11 新编译版本中的又一个惊喜 —— 任务管理器迎来了更多功能。</strong><br>
<p><a href="https://static.cnbetacdn.com/article/2022/0209/ed7fb65ba288a4f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0209/ed7fb65ba288a4f.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://www.neowin.net/news/here039s-how-you-can-enable-and-disable-the-new-task-manager-in-windows-11-build-22538/" target="_self">Neowin</a>）</p><p>@FireCubeStudios 在 Twitter 上爆料称，其在最近推送的 Dev Channel（Build 22543）编译版本中发现了新的内容，看起来<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>正在给任务管理器添加更多功能选项。</p><p><img src="https://static.cnbetacdn.com/article/2022/0209/7225bc9adb6c622.png" referrerpolicy="no-referrer"></p><p>可知 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 任务管理器酝酿的主要变化，就是在主页上添加了专门显示“应用程序健康”和“坚持健康度”的部分，此外还添加了与启动应用程序相关的更多选项。</p><p><a href="https://static.cnbetacdn.com/article/2022/0209/35049855c95f5c6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0209/35049855c95f5c6.jpg" alt="Win11 TaskMgr.jpg" referrerpolicy="no-referrer"></a></p><p>当然，上述变更很可能不是最终的样式。毕竟从 Windows 11 开发团队的表述来看，后续仍有继续改进的空间。</p><p><a href="https://static.cnbetacdn.com/article/2022/0209/ac69c5e84990a18.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0209/ac69c5e84990a18.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>再结合早前 Build 22538 编译版本中暗示的微软正在为 Windows 11 重新设计一款任务管理器的爆料，下半年的变数还是挺大的。</p>
