尽管当前版本的 Chrome 已在“设置”中提供了变更默认浏览器的选项，但具体实施还是要切换到“Windows 设置”应用程序去操作。不过近日，有眼尖的 Reddit 网友发现：<strong>在最新的 Chromium Gerrit 提交中，有涉及对一款新控件的描述 —— 允许用户轻松点击，并将 Chrome 设置为当前系统的默认浏览器。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0729/cb2674e50f01488.png" alt="0.png" referrerpolicy="no-referrer"></p><p>另一方面，早在 2021 年，Mozilla 就已经为 Firefox 提供了更便捷的“一键设置系统默认浏览器”的选项。</p><p><a href="https://static.cnbetacdn.com/article/2022/0729/7e5f95187494645.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0729/7e5f95187494645.png" alt="1-2.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">Windows 设置界面（图自：Reddit / Leopeva64-2）</p><p>Chromium Gerrit 上的代码变更描述称，此功能通过与常规浏览相关的 url 协议和文件关联来工作（比如 http、https、.html 等），并为每个直接设置先关的注册表项。</p><p><img src="https://static.cnbetacdn.com/article/2022/0729/e8c1121a43aa53c.png" alt="2.png" referrerpolicy="no-referrer"></p><p>虽然官方尚未披露这项更新将于何时可用，但我们预期它会很快出现在预览通道，以便在正式发布前展开充分的测试。</p><p><img src="https://static.cnbetacdn.com/article/2022/0729/3cc28d7627d174f.png" alt="3.png" referrerpolicy="no-referrer"></p><p><a href="https://www.neowin.net/news/google-will-soon-make-it-easier-to-set-chrome-as-the-default-browser-in-windows/" target="_self">Neowin</a> 指出，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>让 Windows 11 用户从 Edge 切换到其它浏览器的操作变得更加困难复杂，此举已引发相当大的争议。</p><p>在缺乏“一键设置默认浏览器”选项的情况下，用户被迫修改十多个文件关联 —— 只为了在 Chrome 等第三方浏览器中打开相关链接。</p><p>后来即使做出了一定的让步，Windows 10 / 11 的某些部分仍与 Microsoft Edge 深度绑定。忍无可忍的朋友，倒也可尝试 MSEdgeRedirect 等实用工具来切换设置。</p>
