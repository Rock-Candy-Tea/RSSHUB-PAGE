<strong>在 10 月 5 日正式发布之后，适用于 Windows 11 的首个累积更新 KB5006674 已经正式发布。</strong>在安装该累积更新之后，版本号升至 Build 22000.258。
2021 年 10 月的强制性安全更新主要带来了质量改进和安全更新。微软表示该累积更新解决了部分英特尔“Killer”和“SmartByte”网络软件和初版 Windows 11 系统之间的已知兼容性问题。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1013/71184e83a2cb0f1.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1013/71184e83a2cb0f1.jpg" alt="lw46f749.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">该累积更新主要内容如下：</p><blockquote style="text-align: left;"><p style="text-align: left;">解决了一些<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>“Killer”和“”SmartByte”网络软件与 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11（初始版本）之间的已知兼容性问题。使用受影响软件的设备在某些情况下可能会丢弃用户数据报协议（UDP）数据包。这为基于 UDP 的协议带来性能和其他问题。例如，在受影响的设备上，一些网站的加载速度可能比其他网站慢，这可能导致视频在某些分辨率下流速变慢。基于 UDP 的 VPN 解决方案也可能更慢。</p></blockquote><p style="text-align: left;">微软目前没有发现这个更新有任何问题。</p><p style="text-align: left;">微软还发布了 Windows 11 服务栈更新 22000.190，发布说明如下。</p><blockquote style="text-align: left;"><p style="text-align: left;">该更新对服务栈进行了质量改进，服务栈是安装 Windows 更新的组件。服务栈更新（SSU）确保你有一个强大和可靠的服务栈，以便你的设备能够接收和安装微软的更新。</p></blockquote><p style="text-align: left;">第一个 Windows 11 累积更新可通过 Windows Update 和 Microsoft Update、Windows Update for Business、<a href="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5006674" target="_blank">Microsoft Update Catalog</a> 和 Windows Server Update Services（WSUS）方式获得。</p>
