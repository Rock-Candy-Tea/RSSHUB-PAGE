出于对自家 ARC GPU 产品线的信心，<strong>英特尔正在规划基于 ARC 独显架构的 Project Endgame 云计算平台，以期通过互联网提供各种图形密集型服务。</strong>在近日的投资者报告中，这家芯片巨头揭示其正在“按计划”发布加速计算系统和图形事业部门（AXG）的相关产品。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0219/de412e4314fdfe4.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>据悉，Project Endgame 允许用户通过这项服务来访问 Intel ARC GPU，以获得始终可访问的低延迟计算体验。如果一切顺利，其有望于今年晚些时候推出。</p><p>基于两行简短的官方描述，我们尚不清楚<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>具体在酝酿什么，但从命名规则来看，它应该涵盖了云端游戏串流等领域。</p><p>得益于低延迟的特性，客户将无需在各种场合配备常规硬件，且与英伟达 GeForce Now、亚马逊 Luna、<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a> xCloud、索尼 PlayStation Now、Google Stadia 等竞争服务有异曲同工之妙。</p><p><img src="https://static.cnbetacdn.com/article/2022/0219/351466c3c0cc0a8.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>然而英特尔明确表示，它并没有直指 Project Endgame 将主要面向游戏玩家，此外还有许多基于远程托管的串流服务。</p><p>有人推测，作为该公司本年度的最大赌注，只要 ARC GPU 的功能足够强大，英特尔就能够为工作站提供基于云端的计算解决方案（或类似于 Microsoft <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 365 云电脑）。</p><p>即便初代 ARC Alchemist GPU 无法与英伟达 GeForce RTX / <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> Radeon RX 高端独显平起平坐，至少也能够在被红绿主导的独显市场中获得一席之地。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1238161.htm" target="_blank">英特尔确认ARC Alchemist台式/工作站显卡将于2/3季度到来</a></p></div>
