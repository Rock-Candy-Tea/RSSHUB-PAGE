<strong>作为 oneAPI 工具包的又一个版本，2022.1 更新包含了英特尔专为开发者构建的多款工具。</strong>这些 API 旨在帮助开发者充分榨取英特尔 CPU、GPU / XPU 的性能，甚至抢先做好了为即将推出的 CPU 提供支持的准备（比如至强 Sapphire Rapids 产品线）。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1223/15520855366ffd3.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Intel <a href="https://www.intel.com/content/www/us/en/newsroom/news/intel-empowers-developers-oneapi-2022-toolkits.html" target="_self">官网</a>）</p><p>在更新的开源组件 / 跨架构功能中，还包括了 LLVM 组件、GPU / XPU 分流、SciKit-Learn 扩展、Intel Nerual Compressor 神经压缩等方面，</p><p>此外官方新闻稿中强调其实现了世界上首个支持 C++、SYCL 和 Fortran 编程语言的统一编译器，支持 Python 的 CPU / GPU 数据并行编程、加速器性能建模与高级调整，以及适用于 AI / 光追可视化工作负载的性能加速。</p><p><a href="https://static.cnbetacdn.com/article/2021/1223/23ea37a0d14f17a.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1223/23ea37a0d14f17a.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">2022 版 oneAPI 工具包已可<a href="https://www.intel.com/content/www/us/en/developer/tools/oneapi/toolkits.html" target="_self">免费下载</a> / 或在英特尔 <a href="https://www.intel.com/content/www/us/en/developer/tools/devcloud/overview.html" target="_self">DevCloud</a> 上使用</p><p>英特尔援引 Evans Data 的一项调查称，oneAPI 平台考虑到了 40% 的开发者有在针对不止一种类型的处理器、内核架构 / 协处理器异构系统的做开发。</p><p>本次更新后，该公司算是打造了世界上首个可使用通用 LLVM 后端，为 CPU / GPU 实现 C++、SYCL 和 Fortran 编程语言支持的统一编译器。</p><p>英特尔还强调为 2022.1 版 oneAPI 工具包引入一整套高级工具，涵盖了编译器、库、预优化框架、分析器 / 调试器等方面。</p><p>与开源版本相比，这家芯片巨头宣称 oneAPI 可将深度学习框架的性能提升多达 10 倍，并将基于 CPU 的机器学习算法提速百倍以上。</p>
