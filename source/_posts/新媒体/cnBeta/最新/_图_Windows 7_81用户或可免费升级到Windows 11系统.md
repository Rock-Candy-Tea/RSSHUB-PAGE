目前，Windows 7/8.1 依然能够通过某些途径免费升级到 Windows 10 系统。而根据今天偷跑的 Windows 11 系统镜像，Windows 7/8.1 设备可能也可以免费升级到 Windows 11。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0616/d2e591c8be50e2f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0616/d2e591c8be50e2f.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">外媒 XDA 在镜像中发现了适用于 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 7/8.1 的 Windows 11 产品配置密钥。这意味着<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>可以打开大门，让旧版本用户免费升级到 Windows 11。有趣的是，Windows 8 并不在这个名单上，这意味着用户在升级到 Windows 11 之前必须先升级到 Windows 8.1。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0616/2ab3a2b5bd0e848.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0616/2ab3a2b5bd0e848.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0616/b168fd71b73e4fd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0616/b168fd71b73e4fd.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0616/9d7ac06ca2a1918.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0616/9d7ac06ca2a1918.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">当然，更新对 Windows 10 用户来说也是免费的，尤其是 Windows 11 似乎没有什么升级的意义。到目前为止，泄露的大部分变化似乎都围绕着外观，开始菜单和任务栏是最大的行为变化。Windows 11看起来很像Windows 10X的样子，但没有底层架构的转变。它可以很容易地通过另一个Windows 10的功能更新发布。</p>
