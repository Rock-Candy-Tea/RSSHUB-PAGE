<strong>在本月的补丁星期二活动日中，微软面向尚处于支持状态的 Windows 10 设备发布了累积更新 KB5014699。</strong>该更新并未引入任何重大变化或者改进，只是修复了几个错误。其中包括修复 Outlook 或 Excel 等某些应用程序无法启动的问题。<br>
 <p style="text-align: left;">Windows 10 KB5014699 直接下载链接: <a href="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5014699" target="_blank" rel="nofollow noopener noreferrer">64-bit and 32-bit (x86)</a>.</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0615/12da4984478082e.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0615/ef12859927f9fbd.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">修复的另一个错误是用户遇到窗口框架问题，Microsoft Edge 中的 Internet Explorer 模式无法正常运行。</p><p style="text-align: left;">Windows 10 用户报告了一个令人讨厌的错误，该错误会导致应用程序崩溃并显示无用的消息。结果，一些人暂停了他们的 Windows 更新，以尽量减少应用程序崩溃。当用户使用 .NET Framework 打开他们最喜欢的应用程序时会发生该错误。值得庆幸的是，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>终于修复了剩余的应用程序兼容性问题，大多数应用程序应该可以在 Windows 10 上正常运行。</p><p style="text-align: left;">Microsoft 正在增加对排序版本 6.4.3 的支持，这应该会解决影响日语半角片假名的问题。另一个错误已修复，该错误阻止用户在登录 Azure Active Directory (AAD) 时绕过强制注册。</p><p style="text-align: left;">大多数错误修复显然是针对企业客户的。例如，Microsoft 修复了导致 AnyCPU 应用程序运行 32 位进程的问题。另一个错误已修复，其中具有多个部分配置的 Azure Desired State Configuration (DSC) 方案无法按预期工作。</p><p style="text-align: left;">以下是所有错误修复和改进的列表：</p><blockquote style="text-align: left;"><p style="text-align: left;">● 微软已修复影响 Internet Explorer 模式窗口框架的问题。</p><p style="text-align: left;">● 微软已修复导致打印失败的问题。</p><p style="text-align: left;">● 微软已修复阻止 BitLocker 加密的问题。</p><p style="text-align: left;">● 微软已修复影响鼠标光标的行为和形状方向的问题。受影响的应用程序列表包括 Microsoft Defender Application Guard (MDAG)、Microsoft <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a> 和 Microsoft Edge。</p><p style="text-align: left;">● 微软还修复了终端服务网关 (TS Gateway) 服务中的可靠性问题。</p><p style="text-align: left;">● 当设备加入域时，Windows 搜索突出显示现在可以正常工作。</p><p style="text-align: left;">● 微软已修复输入法编辑器显示错误图像的问题。</p><p style="text-align: left;">● 微软已修复导致黄色感叹号的问题。</p><p style="text-align: left;">● 微软修复了一个错误，该错误会阻止使用控制面板中的 Windows 7 的备份和还原应用程序制作的恢复光盘（CD 或 DVD）进行恢复。</p></blockquote>
