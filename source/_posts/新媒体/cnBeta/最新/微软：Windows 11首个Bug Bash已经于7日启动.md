<strong>伴随着 Windows 11 Build 22000.65 新预览版的发布，今天微软还推出了适用于 Windows 11 的首个 Bug Bash。</strong>它是软件公司广泛测试产品的一种方式，并确定可以在未来版本中修复的错误或问题。<br>
<p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0709/041ac232e5bce03.jpg" referrerpolicy="no-referrer"></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/70646d151b809f5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/70646d151b809f5.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/0fbf1e3c26022cd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/0fbf1e3c26022cd.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/faa6354f684ee3d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/faa6354f684ee3d.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">微软为每一个主要的 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 更新举行 Bug Bash，以收集来自 Windows Insiders 的反馈，首个 Windows 11 Bug Bash 活动从 7 月 7 日到 7 月 14 日进行。在该活动期间，微软将在反馈中心发布任务，涵盖不同的 Windows 11 功能。Windows Insiders 可以尝试这些工作流程并向微软提供反馈。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/a595355e14f2a8c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/a595355e14f2a8c.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">一旦Bug Bash结束，微软将根据参与情况对Windows Insiders的成就进行奖励。如果你有兴趣参与，那么你可以在Windows 11上的反馈中心导航到任务，并完成它们以获得成就。随着本周的进展，微软将增加新的任务，你可以完成这些任务，并向公司提供反馈。</p><p style="text-align: center;"><a href="https://static.cnbetacdn.com/article/2021/0709/9307e1656e230a9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/9307e1656e230a9.jpg" referrerpolicy="no-referrer"></a></p>
