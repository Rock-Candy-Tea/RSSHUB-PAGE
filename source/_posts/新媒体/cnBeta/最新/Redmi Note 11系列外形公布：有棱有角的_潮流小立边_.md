今天，Redmi公布了Redmi Note 11系列外观。如图所示，<strong>Redmi Note 11采用了直角中框设计，与iPhone 13的中框相似。</strong>官方介绍，<strong>Redmi Note 11之所以要做小立边的大改款，是因为今天的年轻人都有棱有角。</strong>Redmi表示，立边更潮流，细节精致好看；立边更自信，给你想要的独特感。<br>
 <p>除了在工业设计上有大幅升级外，Redmi Note 11系列的快充也有明显进步。</p><p>据爆料，<strong>Redmi Note 11顶配版支持120W快充，这与<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&aid=450&euid=&t=http%3A%2F%2Fwww.mi.com%2F" target="_blank">小米</a>MIX 4的快充功率相一致。</strong></p><p>官方海报此前表示Redmi Note 11“一刻满血”，似乎暗示Redmi Note 11系列仅需15分钟就能充至100%，在同档位极具竞争力。</p><p>此外，Redmi Note 11采用了高刷新率挖孔屏方案，预计会搭载旗舰处理器，配备双扬声器。</p><p>新品将于10月28日发布。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1021/e01b35ebf1857b5.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1021/e01b35ebf1857b5.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1021/e01b35ebf1857b5.jpg" referrerpolicy="no-referrer"></a></p>
