Oculus 最近公布了一个名为 Passthrough 的新 API， 可以让开发者把 Quest 2 头戴式设备采集到的视频影像整合到 VR 游戏以及应用内，创造一种新的混合现实 （Mixed Reality） 体验。在这个新 API 接口的帮忙之下，开发者不但可以把 Quest 2 传感器采集到的黑白图像融入到用户的 VR 体验中，还可以决定这些黑白图像要怎样呈现在用户眼前，例如加入特效或者直接让现实世界出现在特定的表面上。<br>
<p style="text-align:center"><a target="_blank" href="https://static.cnbetacdn.com/article/2021/0724/4f43aee1ae86189.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/0724/4f43aee1ae86189.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/0724/4f43aee1ae86189.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在公布新 API 的同时 ，Oculus 也展示了 Passthrough API 可以如何协助提高生产力，其中一个例子就是允许集成现实中的键盘及鼠标。而在娱乐方面，游戏中的敌人也可以利用玩家现实环境中的家具作为掩体。简而言之，在 Passthrough API 之下，现实以及 VR 应用或游戏可以有更大的创作发挥空间。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/07/76099582b65b7c6.gif" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/07/76099582b65b7c6.gif" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Oculus 的母公司 Facebook 在接受 UploadVR 的访问时表示 ，Passthrough API 目前只适用于 Quest 2 设备，并不支持第一代 Quest。 既然可以把传感器的影像投射到应用及游戏中，那用户的隐私又应怎么处理呢? Oculus 似乎也是知道了用户会有这个担忧，因此在宣布中也表示，所有 Passthrough API 使用的影像都是由设备本身处理的，因此应用及游戏也不会知道这些影像的画面，也不能够作出保存。<br style="text-align: left;"></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/07/3ad05b9cc7ff84f.gif" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/07/3ad05b9cc7ff84f.gif" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Oculus 目前正在把 Passthrough API 的实验版本加入至接下来的开发者工具套件中，利用 Unity 引擎来开发软件的开发者将会是第一批可以使用这个 API 的人群。至于其他开发平台 Oculus 也表示在未来会加入对它们的相关 API 支持。</p>
