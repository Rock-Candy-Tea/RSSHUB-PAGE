<a href="https://shiplift.io/">ShipLift LLC</a> 的开发者 <a href="http://andyletourneau.com/">Andrew LeTournaeu</a> 和 <a href="https://github.com/sloped">Connor McCall</a> 近日推出了一款名为“<a href="https://github.com/ShipLift-LLC/defrag" target="_blank">defrag</a>”的优秀 Vue 项目，能够让你重回 1995 感受当时的磁盘清理。由于该项目也是刚刚上线，因此没有一个适当的 readme 文件，但是它非常有趣，值得一看。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1218/7cc8adbe9dd3d70.webp" alt="fr52u5mk.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">碎片整理是整合你在磁性硬盘上的数据的过程。如果你在这些旧转盘上删除一个文件，你就会留下一个缺口，而新的文件则被写入更远的盘片上。碎片整理将采取较新的数据，并将其重写到空白处，使你的硬盘成为一个坚实的数据块。至少在一段时间内如此。LeTournaeu 的应用程序模拟了这一过程，将自己提升到一个熟悉的 DOS 蓝色帆布上。它看起来不像我使用的 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 工具，但它很容易让人想起 MS-DOS 版本。事实上，它看起来和今天的 Spin-Rite 一样。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1218/9bfc89d1c6a7103.gif" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/1218/9bfc89d1c6a7103.gif" alt="1639750735_defrag-small.gif" width="100%/" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">此外该软件还有一个 CRT 效果来增加怀旧感，尽管国外媒体编辑在阅读了 package.json 之后，我无法弄清是哪个包提供的。然而，它确实使用了 Howler，这是一个用于 Javascript 的音频库。这表明这个项目在某些时候会发出声音，但现在还没有动静。</p><p style="text-align: left;">Source: <a href="https://defrag.shiplift.dev/">defrag.shiplift.dev</a> via <a href="https://news.ycombinator.com/">Hacker News</a></p>
