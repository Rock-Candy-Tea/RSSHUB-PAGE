近日，微软宣布了旗下 SQL Server 2022 关系数据库管理系统的新预览。<strong>得益于 Azure Purview 和 Synapse Link 的更深入集成，其提供了更多的洞察与治理能力。而总部位于雷德蒙德的这家软件巨头，也将之称作“云连接最广的版本”。</strong>比如在内置智能模块处理的可扩展性与优异性能的加持下，数据库管理员也将无需担心资产与云分析的连接。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1103/864ebaa5844e3c7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1103/864ebaa5844e3c7.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：Microsoft <a href="https://cloudblogs.microsoft.com/sqlserver/2021/11/02/announcing-sql-server-2022-preview-azure-enabled-with-continued-performance-and-security-innovation/" target="_self">官网</a>）</p><p>此外 SQL Server 2022 提供了跨语言和平台的灵活性，包括 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 和 Linux 操作系统、以及 Kubernetes 容器。</p><p>与此同时，对原生云功能更感兴趣的组织机构，会很高兴看到 Apache Cassandra 数据的轻松迁移（到 Azure Cosmos DB）。</p><p>随之而来的还有成本控制与生产力增强，且适用于 Apache Cassandra 的 Azure 托管实例，现已普遍可用。</p><p>Azure Synapse Analytics 正在以行业特定的数据库模板、遥测实时分析和 Event Hubs Premium 普遍可用性的形式获得新功能。</p><p><a href="https://static.cnbetacdn.com/article/2021/1103/900c3ab10e33d28.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1103/900c3ab10e33d28.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（Azure Blog <a href="https://azure.microsoft.com/zh-cn/blog/transform-your-business-with-microsofts-unrivaled-endtoend-data-platform/" target="_self">传送门</a>）</p><p>本月，Azure Database for MySQL 和 Azure Database for
PostgreSQL 将普遍提供灵活的服务器部署。</p><p>基于 Intel Ice Lake 平台的新一代硬件正在为 Azure SQL 托管实例提供预览（Link 也是如此），无论前者托管于何处、它都可以将 SQL 服务器连接到 Azure SQL 托管实例。</p><p>值得一提的是，Azure 认知搜索现支持更多语言、Azure OpenAI 服务也迎来了预览，以提供对 OpenAI 的 GPT-3 语言模型的访问（目前仅限邀请体验），且<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>为大量使用 Azure 认知服务的客户提供了带有承诺的定价模型。</p><p><a href="https://static.cnbetacdn.com/article/2021/1103/ea0404815fded95.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1103/ea0404815fded95.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>其它方面，Azure 通信服务中的 SMS“短代码”将于本月晚些时候开启预览，而与 Microsoft Teams 的互操作性将于下个月普遍可用。</p><p>Azure 容器应用已经发布了预览版本，而 Azure 服务总线的大消息支持、Web PubSub、Azure Kubernetes 服务的开放式服务网格附加组件，也都普遍可用。</p><p>Azure 逻辑应用程序在预览版本中引入了新功能，详情可移步官网（<a href="https://techcommunity.microsoft.com/t5/azure-developer-community-blog/azure-logic-apps-announcement-fall-2021-release/ba-p/2911923" target="_self">传送门</a>）。同时在 Azure 容器上运行的 Java 企业版（Java EE）应用程序的客户，可利用 Azure 市场中提供的新解决方案。</p><p>通过 Azure 活动目录（AAD）和 GitHub Actions 之间的 OpenID Connect 集成，Azure DevOps 也得到了增强，且现在普遍提供了一款用于可视化 DevOps 工具链的新工作流生成器工具。</p><p>Azure API 管理已通过与事件网格（普遍可用）的集成、GraphQL 直通支持（普遍可用）、以及对 WebSocket API 的本机支持而得到了改进。</p><p>客户可使用 Chaos Studio 在 Azure 中模拟真实世界的中断，以便其规划、监测和制定舒缓策略（目前处于预览阶段）。</p><p>最后，Azure 迁移和现代化计划（<a href="https://azure.microsoft.com/zh-cn/blog/" target="_self">AMMP</a>）正在扩展，以涵盖额外的安全性、Azure Arc、应用现代化和 SAP 环境。</p>   
</div>
