微软今天宣布了中国区Azure混合云服务及功能的一系列更新，包括将于今年下半年发布预览的Azure Stack HCI 22H2、Azure Arc-enabled Servers、Azure Arc-enabled Kubernetes公开预览版本，以及针对应用、数据和机器学习服务的相关更新。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0616/fc53f65cd84bf57.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0616/fc53f65cd84bf57.png" title alt="image.png" referrerpolicy="no-referrer"></a></p><p>Azure Arc将Azure安全性和云原生服务引入混合和多云环境，并通过Azure Arc-enabled SQL Managed Instance（Business Critical）、Azure Arc-enabled Machine Learning（预览版）、Azure Arc-enabled Open Service Mesh以及Azure Landing zone accelerator for Arc-enabled Kubernetes，随时随地保护和管理企业基础结构和应用，帮助企业在任意位置大规模构建云原生应用、在任意位置运行Azure数据和机器学习服务，实现跨混合和多云的统一安全性，灵活满足监管与云端连接的需求。</p><p style="white-space: normal;">即将推出的Azure Arc-enabled Servers让企业管理在Azure之外的<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>和Linux物理服务器，以及托管在企业公司网络或其他云提供商上的虚拟机。Azure Arc-enabled Kubernetes（公开预览版本）则允许企业对于在任何地方（包括运行在企业本地数据中心（Azure Stack HCI或友商私有云平台）上的Kubernetes环境，和运行在其他公有云提供商上的托管类型的Kubernetes环境）运行的Kubernetes集群，进行统一管理和调度操作，并支持应用的统一发布与变更。</p><p style="white-space: normal;"><a href="https://static.cnbetacdn.com/article/2022/0616/410345cab83de00.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0616/410345cab83de00.png" title alt="image.png" referrerpolicy="no-referrer"></a></p><p>2022年下半年，Azure Stack HCI 22H2版本将在中国区上线。Azure Stack HCI与Azure服务原生集成，从Azure门户集中管理，可统一管理订阅计费、获取多样化的OEM解决方案。同时，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>宣布推出Azure Stack HCI单节点版本的解决方案，在可以允许较低复原能力的位置，使用单一服务器可以最大程度地降低硬件和软件成本，单一服务器还允许最初进行小规模部署，以后可以在此部署中添加服务器，实现向外扩展。为简化协作并为所有环境提供一致的控制，微软也在全球推出System Center VMM新版本，可用来管理Azure Stack HCI群集和群集组，同时可用于Windows Server 2022 物理服务器和虚拟机管理，并延展到支持Windows 11的虚拟机管理。</p><p><a href="https://static.cnbetacdn.com/article/2022/0616/11b18f67ea33a9d.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0616/11b18f67ea33a9d.png" title alt="image.png" referrerpolicy="no-referrer"></a></p><p>企业可借助Azure Arc和Azure Stack HCI实现边缘站点、数据中心现代化、Windows Server和SQL Server迁移上云三个场景。SKF集团是全球领先的轴承、密封和润滑系统供应商，其中国业务借助Azure Stack HCI，增强了工厂端边缘计算能力，实现了整体运营管理、备份容灾，灵活使用资源。世界领先的氧化铝材料生产商安迈铝业（Almatis）使用Azure Stack HCI搭建了可灵活支撑其全球架构的混合云。考虑到Windows Server 2012/SQL Server 2012系列产品即将退出支持周期，Azure Stack HCI让客户无需软件升级保障，就能够进行免费的扩展安全更新。</p><p>在中国大陆，微软聚合了<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>、<a data-link="1" href="http://www.anrdoezrs.net/links/9019719/type/dlg/sid//https://www.dell.com/zh-cn/shop/deals" target="_blank">戴尔</a>、H3C、HPE、联想、超聚变等多家OEM硬件合作伙伴，以及南洋万邦、北京昆仑联通、联想、精诚资讯（SYSTEX）、戴尔等Azure Stack HCI加速计划合作伙伴，推出基于Azure Stack HCI的验证平台或集成系统的混合云解决方案，加速企业上云进程。</p>
