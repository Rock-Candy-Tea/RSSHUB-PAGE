<strong>微软今日宣布推出 Azure OpenAI 服务，意味着广大客户将可在 Azure 上轻松使用 OpenAI 的 GPT-3 机器学习模型。</strong>作为 OpenAI 的开创性语言模型，在适当的条件下，你甚至只需输入几条简短的提示信息，就可以让 GPT-3 输出像是人类的文本。不过目前 Azure OpenAI 仍处于有限体验的阶段，即使愿意付费，也要再等待一段时间。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1103/21b15f7e7adb2ef.gif" referrerpolicy="no-referrer"></p><p>微软表示，Azure OpenAI 现仅限邀请、适用于“计划实施定义明确的用例的客户”，包括使用人工智能技术来制定负责任的策略等应用场景。</p><p>微软将为客户提供安全监控和分析支持，以查找错误或滥用 GPT-3 的案例。并且引入过滤器，以确保基于 GPT-3 的聊天机器人不会越界冒犯。</p><p>需要指出的是，OpenAI 本身就已经在去年提供了 GPT-3 API，而微软也在利用 GPT-3 来为 GitHub 的 Copilot 辅助编程工具提供支撑。</p><p>即便如此，微软还是认为自家 Azure OpenAI 方案能够带来额外的安全层、访问管理、专用网络、数据处理保护、以及扩展容量等优势。</p><p>【背景资料】</p><blockquote><p>早在 2019 年，微软就在 OpenAI 上投资了 10 亿美元，并获得了 GPT-3 许可。因而该公司现在试图将它引入更广泛的产品中，我们也不会感到奇怪。</p></blockquote><p>OpenAI 首席执行官 Sam Altman 表示：</p><blockquote><p>GPT-3 证明了自己是一个强大、通用的自然语言模型，喜欢它的开发者可借此实现非常轻松地尝试许多事情。</p><p>一段时间以来，我们一直想要找到一种方法来尽可能地广泛扩展。这也是与微软的合作伙伴关系中，最让我们感到激动的一部分。</p></blockquote>
