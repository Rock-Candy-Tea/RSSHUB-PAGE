俄乌冲突爆发后，许多国际企业、组织机构在舆论压力下宣布了对俄罗斯的制裁。然而一刀切式的政策，无疑又将伤害无限扩大化。<strong>于是本周三，Thomas Dohmke 在代码托管平台 GitHub 上的一篇博客文章中写道：</strong>“我们对全球社区都十分关心，并且会继续留意与俄乌冲突相关的事态发展”。<br>
 <p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0304/927f24fd8cf01d0.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">CEO 撰文（来自：GitHub <a href="https://github.blog/2022-03-02-our-response-to-the-war-in-ukraine/" target="_self">Blog</a>）</p><p><strong>Thomas Dohmke 指出：</strong>“我们在世界各地有许多亲友，其中就包括了乌克兰、俄罗斯、以及其它受影响的地区”。</p><blockquote><p>作为一个成长于冷战时期的东德人，我仍记得 1990 年代初期的幸福与乐观，因为世界将人们更紧密地联系在了一起。</p><p>然而现在所发生的，正是我们再也不愿回想的事情。这不仅对无辜的人们造成了毁灭性的打击，也让置身事外的我们感到无助和愤懑。</p><p>作为被所有开发者的寄予厚望的开源大家庭，GitHub 会认真对待我们在保护开放协作、以及信息自由流动方面所扮演的角色 —— 其中就包括让平台保持对所有人敞开怀抱，而无论其身处何方。</p><p>作为一家公司、一个平台、以及面向全球 Hubbers 的社区，我们很乐意分享 GitHub 是如何应对此类事件的最新动态。</p></blockquote><p><strong>首先，GitHub 致力于维护平台与公司的完整性：</strong></p><blockquote><p>GitHub 被许多人视作一款强大的工具，并借此开展交流、人道主义工作、组织变革、以及让人难以置信的其它技术进步。</p><p>我们切实相信着开源的力量，并将继续努力确保 GitHub 平台对全球所有开发者的开放、可用于安全性。</p><p>在让所有国家 / 地区的开发者都可使用 GitHub 的同时，我们将会继续确保所有人都可使用免费的开源服务，其中就包括身处俄罗斯地区的开发者们。</p></blockquote><p><strong>其次，GitHub 致力于提供强大的安全体验，以防止 GitHub 用户及其帐户受到威胁。</strong></p><blockquote><p>我们敦促开发人员设置双因素身份验证（2FA），且最好使用 WebAuthn 方案来增强账户的安全性。</p><p>与此同时，我们正在采取行动来支持 GitHub 平台，并遵守可能在俄乌冲突期间收到的诸多政府命令。</p><p>GitHub 的法务团队会对相关规定开展彻底的审查，且会遵从出口管制与贸易法规的动向。</p><p>此外 GitHub 会公开披露任何基于政府命令的下架通知，因为透明度是实现良好治理不可或缺的重要一环。</p><p>可以理解的是，在这个特别紧张的时期，社区成员对俄乌冲突产生了强烈的反应。</p><p>我们正在执行可接受的使用政策和社区指南，以确保向所有人都提供安全的 GitHub 访问体验。</p><p>如果你有发现潜在违反平台政策或社区准则的行为，还请积极向官方提交举报。</p><p>目前社区与产品运营团队正在对 GitHub 论坛上的讨论话题展开监测，且会根据相关反馈和准则进行评估并处理。</p><p>一旦被认定，</p></blockquote><p><strong>最后，GitHub 会优先考虑 Hubber 们的安全与福祉：</strong></p><blockquote><p>有一点是非常清楚的 —— 无论您当前或原先拥有哪个国籍，我们都会全力支持世界各地的每一位 Hubber 。</p>你们当中已有许多人向本人、领导团队或管理者发去信息，寻找可为乌克兰民众提供直接支持的方法。<p>说到这，你要知道 GitHub 每年都会向合格的 Hubber 提供高达 15000 美元的赠款，且 Social Impact 团队有分享一份支援乌克兰的组织名单，以供大家参考。</p><p>作为一家公司，GitHub 已向 Razom 捐赠了 10 万美元、加上员工筹集的另外 20 万美元，以帮助在该地区开展的救援工作。</p><p>这包括为在慈善机构、以及在世界各地从事人道主义工作的 Hubber 们提供相应的捐款。</p><p>我想重申的是，在这个相互关联的全球社区中，我们有在认真对待保护开放合作和信息自由流动这些事。</p><p>GitHub 将继续密切关注事态的发展，并会随着了解的深入而及时调整。</p><p>与世界上的许多其他人一样，我们希望为包括乌克兰民众和所有受到冲突影响的其它社区提供和平的解决方案。</p></blockquote>
