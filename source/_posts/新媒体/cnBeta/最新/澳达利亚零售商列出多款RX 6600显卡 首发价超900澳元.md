随着 Radeon RX 6600 显卡上市日期的邻近，澳大利亚零售商也放出了多款商品的链接。周末的时候，@Momomo_US 已经在 Twitter 上分享过多个厂家的产品报价。昨日，WCCFTech 也分享了技嘉的“GV-R6EAGLE-8GD”SKU 。<strong>作为一款非公型号，该显卡并非超频衍生版本。但现在，它已被 Computers Perth、PC AOS、Technology2You 和 Inell 挂出。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0906/cab0d1c5382bceb.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0906/cab0d1c5382bceb.png" alt="0.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/gigabyte-radeon-rx-6600-eagle-graphics-card-listed-by-australian-retailers-preliminary-prices-over-900-aud/" target="_self">WCCFTech</a>）</p><p>三家零售商的报价不尽相同，最便宜的 Inell 标价为 805 澳元（约 600 美元 / 3864 RMB），最高价为 951 澳元（约 710 美元 / 3408 RMB），平均上市价格为 900 澳元（约 671 美元 / 4320 RMB）。</p><p><a href="https://static.cnbetacdn.com/article/2021/0906/f6daa0381a76a04.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0906/f6daa0381a76a04.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>参考 Computers Perth 上的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://asus.jd.com/" target="_blank">华硕</a> ROG STRIX RX 6600 XT，其售价为 1065 澳元（约 794 美元 / 5112 RMB）。这意味着 RX 6600（非 XT 型号）要便宜 260 澳元（约 194 美元 / 989 RMB）左右。</p><p><a href="https://static.cnbetacdn.com/article/2021/0906/f4a8665321f1c48.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0906/f4a8665321f1c48.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>不过需要指出的是，华硕 ROG STRIX 定位高端，而技嘉 Eagle 系列相对入门。只是由于两家零售商都没有列出 6600 XT Eagle 的衍生版本，所以我们也无法直接拿来比较。</p><p><a href="https://static.cnbetacdn.com/article/2021/0906/c5a39ed12647ef3.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0906/c5a39ed12647ef3.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>规格方面，零售商的产品清单和 SKU 代号，已经证实了 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> Radeon RX 6600 显卡具有 8GB GDDR6 显存（速率 16Gbps @ 总带宽 256GB/s）。</p><p><a href="https://static.cnbetacdn.com/article/2021/0906/9ce1a4c375b5f69.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0906/9ce1a4c375b5f69.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p>遗憾的是，@Coreteks 在 Twitter 上表示，定于 10 月到来的 Radeon RX 6600 显卡，其数量不见得会比 Radeon RX 6600 XT 更多。</p><p><img src="https://static.cnbetacdn.com/article/2021/0906/1c54f1ad4bcf8b6.png" alt="5.png" referrerpolicy="no-referrer"></p><p>理由是随着 Navi 23 GPU 良率的上升，且消费者也愿意掏钱买更贵的 Radeon RX 6600 XT，那在产能紧缺的情况下，AMD 显然也会重点保障更高利润率的“XT”型号。</p><p><img src="https://static.cnbetacdn.com/article/2021/0906/5e12ff57be2de1d.jpg" alt="6.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">标价为澳元（图 via @Momomo_US）</p><p>换言之，即使 AMD 给 Radeon RX 6600 立下了 299 美元（约 1930 RMB）的建议零售价，消费者也不大可能在零售商的货架上看到它的身影。</p><p><img src="https://static.cnbetacdn.com/article/2021/0906/5c5ecb6aa1435c2.jpg" alt="7.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">Radeon RX 6600 主打 1080p 游戏体验</p><p>最后，RX 6600 显卡配备了 Navi 23 XL GPU，拥有 28 个计算单元（CU）/ 1792 个流处理器，辅以 32MB Infinity Cache 缓存 / 8GB GDDR6 显存。</p><p>此外传闻称 RX 6600 的 ETH 挖矿算力达到了 27 MH/s，3DMark Time Spy 跑分 7805，且 Igor's Lab 的模拟性能测试表明其接近于英伟达 RTX 3060 。</p>
