GDevelop 是一款允许用户在没有任何特定开发经验的情况下轻松打造视频游戏的引擎，<strong>虽然近年来“无代码”开发的概念已经变得相当流行，但 GDevelop 的历史却可追溯到更早的时候。</strong>官网写道，GDevelop 最初是由 Florian Rival 用爱发电打造的，GitHub 上的首个公开版本于 2014 年发布。<br><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1220/3877c502916a7ad.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1220/3877c502916a7ad.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：GDevelop <a href="https://gdevelop-app.com/" target="_self">官网</a>）</p><p>GDevelop 项目诞生的初衷，就是为了让游戏开发这项工作变得尽可能轻松。除了传统的计算机部署，你甚至可以通过 Web 浏览器来启动 GDevelop 。</p><p><a href="https://static.cnbetacdn.com/article/2021/1220/dcc462943221548.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1220/dcc462943221548.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>初上手的朋友可以先充分借鉴现成的模板，并且能够随时对其进行修改和试用。目前 GDevelop 主要将经历放在 2D 游戏开发上，毕竟其对于玩家和开发者来说都更易于上手。</p><p><a href="https://static.cnbetacdn.com/article/2021/1220/feed5bbacef6b38.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1220/feed5bbacef6b38.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>关卡方面，你大可直接将对象拖放到对应的场景中。至于游戏功能方面的设计，亦可在“事件”选项卡中查找挑选所有可用的触发机制，且一切都可由条件 + 动作的方式来描述（比如让角色跳跃）。</p><p><a href="https://static.cnbetacdn.com/article/2021/1220/5f3180450aa8d8c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1220/5f3180450aa8d8c.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p>如果觉得默认操作选项还不够用，GDevelop 也允许开发者通过扩展系统来添加一些逻辑、或者更进一步地创建自己的扩展（一组条件 + 操作 / 或 JavaScript 代码）。</p><p><img src="https://static.cnbetacdn.com/article/2021/1220/f64b49d17ee957a.png" alt="3.png" referrerpolicy="no-referrer"></p><p>最后，GDevelop 支持利用 WebGL、JavaScript 和 WebAssembly 等技术来渲染游戏，且允许导出为网游或 Android 版本。</p>
