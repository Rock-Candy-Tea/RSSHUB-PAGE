按照开发计划，在努力完善 Windows 11 的首个正式发布版本的同时，微软也已经在为定于 2022 下半年到来的另一开发分支做准备。<strong>不过现在，Insider 测试者们已经迎来了开发通道的 Windows 11 Build 22449 编译版本。</strong>Windows Latest 指出，这也是被该公司内部称作“Nickel”的活跃开发分支的首个测试版本。<br>
 <p style="text-align: center;"><a href="https://static.cnbetacdn.com/article/2021/0903/d3c80fdf3893a7e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0903/d3c80fdf3893a7e.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>如果你的设备满足 Dev Channel 的最低要求，即可看到名为“<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 Insider Preview 22449.1000 (rs_prerelease)”的新补丁，编译版本号高于现有的 Dev / Beta 收到的预览版本。</p><p>如果一切顺利，这个新版本中包含的某些功能和改进，将在 2022 下半年发布的 Windows 11 22H2（代号 Copper）版本中正式交付给广大用户。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2021/0903/1c5ebd6fa4365c6.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>由从维基百科上随便扯来的一张元素周期表可知，Windows 11 开发团队选择了以不同的元素来指代不同的开发分支。</p><blockquote><p>● Windows 11 Version 21H2（10 月 5 号发布）= Cobalt（Co）</p><p>● Windows 11 Nickel（Ni）开发分支 -- 尚未指代任何特定的版本</p><p>● Windows 11 Version 22H2（明年 10 月发布）= Copper（Cu）</p></blockquote><p>至于 Windows 11 Build 22449 中的功能和改进，Nickel 分支的首个更新将直接影响 SMB 压缩行为。SMB 压缩是一项允许企业 / 应用程序在通过网络传输文件时，请求压缩文件的一项功能。</p><p>目前该功能可在传输时压缩文件的前 500 MiB，但若可压缩量少于 100 MiB，它又会停止工作。好消息是，随着今日推出的更新，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>将始终尝试压缩文件的其余部分。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2021/0903/2c7f6e15ec7bd17.gif" alt="3.gif" referrerpolicy="no-referrer"></p><p>有趣的是，微软还在试验一种迎合新的 WinUI 设计风格的渐进式环形动画，以取代当前加载屏幕上的圆圈动画。类似的改进，也将很快推广到操作系统的其它区域。</p><p>音频方面，Windows 11 Build 22449 也对启动音效施加了一些调整。在应用预订更新后，Windows 将不再播放启动音频，且设备会在不主动使用时重新启动。</p><p>通知中心也迎来了一些小改进，通知条目将具有模糊背景。若在快速设置面板的蓝牙图标上右键点击，现在也会跳转到主设置窗口。视觉设计方面，应用程序现看起来有些浮动分离的感觉。</p><p><a href="https://static.cnbetacdn.com/article/2021/0903/7d26511284e788e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0903/7d26511284e788e.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>以下是 Windows 11 Build 22449 带来的其它几项改进：</p><blockquote><p>● 圆角 UI 设计已扩展到 Windows 沙箱对话框。</p><p>● 通知中心的专注助手已更新，改进了可访问性。</p><p>● 修改时间格式或添加其它时钟时，日历弹出窗口将自动更新。</p><p>● 当任务栏跨多个显示器固定时，资源管理器（Explorer.exe）已变得更加可靠。</p><p>● 当在表情符号面板中搜索生气悲伤 Emoji 时，建议框里不会再弹出生气 Emoji 。</p><p>● 修复访问‘开始’、‘搜索’和‘表情符号’面板时的多个崩溃或死锁问题。</p><p>● 更新‘设置’应用中的‘输入’>‘触控键盘’页面，以包含指向个性化页面的链接。</p></blockquote>
