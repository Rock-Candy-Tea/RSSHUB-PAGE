对于许多家庭消费者来说，当他们需要访问相机存储卡中的文件时，他们只需将其插入电脑上的读卡器。这通常是一个SD读卡器，尽管许多较新的计算机正在完全取消读卡器。<strong>但如果你是一个专业摄影师呢？那么可能根本不使用SD卡，相反更有可能遇到CFexpress。Lexar刚刚发布了一款新的CFexpress
B型读卡器，其性能一定会让创意人士感到兴奋。</strong><br>
<p>全新的Professional CFexpress Type B USB 3.2 Gen 2x2读卡器使用USB-C连接，但它的包装盒里也有一条USB-A电缆，最重要的是，它的速度相当快--它的读卡速度可高达1700MB/s。</p><p><img src="https://static.cnbetacdn.com/article/2021/0704/ccf74d0e2bae3be.jpg" title alt="LexarCFE1.jpg" referrerpolicy="no-referrer"></p><p>官方是这样描述这款产品的："Professional CFexpress Type B USB 3.2 Gen 2x2读卡器以USB 3.2 Gen 2x2速度将大量高质量图像和RAW 8K视频从你的CFexpress Type B卡上传到你的电脑上，为你从拍摄现场到后期制作的工作流程提速。这款专业级读卡器包括一条二合一的USB Type-C至USB Type-A电缆和USB Type-C至USB Type-C电缆，可与您的笔记本电脑或PC进行无缝的端口对端口连接，其紧凑和便携的设计使您可以快速将读卡器放入口袋，然后离开。"</p><p>雷克沙Professional CFexpress Type B USB 3.2 Gen 2x2读卡器现在已经加入其工作流程解决方案系列产品线。价格为69.99美元。它应该与所有操作系统兼容，包括<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>、macOS和Linux发行版。它甚至还提供了五年的有限保修。</p><p><img src="https://static.cnbetacdn.com/article/2021/0704/18f648ecea30f58.jpg" title alt="LexarCFE0.jpg" referrerpolicy="no-referrer"></p>
