在经历了 3 个月的等待之后，AMD 今天面向 Linux 发布的 Radeon Software 22.20 终于引入了对 Ubuntu 22.04 LTS 的支持。新版驱动还对即将带来的 Ubuntu 22.04.1 维护版本更新提供了初步支持。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0716/54510a3e106e6ff.webp" alt="uk8yeqd7.webp" referrerpolicy="no-referrer"></p><p>Radeon Software for Linux 22.20 驱动还添加了对近期发布的 SUSE Linux Enterprise 15 SP4（SLED / SLES 15 SP4）发行版本的支持。</p><p>根据官方更新日志除了添加对多个企业 Linux 发行版本的支持之外，Radeon Software for Linux 22.20 驱动和数月前发布的 22.10 旧版本并没有区别。不过新版驱动带来的更新的 AMDGPU DKMS 模块，升级了 OpenGL 和 Vulkan 驱动等等。</p><p>Radeon Software for Linux 22.20 驱动目前已经在 <a href="https://www.cnbeta.com/articles/tech/undefined">AMD.com</a> 官网开放下载。</p>
