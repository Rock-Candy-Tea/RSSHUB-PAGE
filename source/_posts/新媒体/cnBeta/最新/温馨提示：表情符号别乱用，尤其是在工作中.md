作为网络语言的“十级玩家”，你可以不知道 yyds、nbcs、xswl 这些需要超强（变态）网络嗅觉的缩写词，<strong>但在日常工作生活中，你一定没少和社交软件中的各种表情符号（Emoji）打交道：开心就，表示庆贺/欢迎就，沮丧/感动时则。。。。。。</strong><br>
<p style="text-align: left;">尤其是在新冠大流行期间，由于一些隔离规定或预防措施，越来越多的企业开始实施远程工作办法。于是，人们需要依赖社交软件进行大量工作交流，其中则不免用到各种表情符号（来避免尴尬）。但现在，你可能得注意了：你用过的那些表情符号，或许会成为衡量你工作效率与心理感受的好“帮手”。</p><p style="text-align: left;">近日，来自密歇根大学、北京大学和马里兰大学的华人研究团队，就通过机器学习方法针对 GitHub 用户的表情符号（Emoji）使用情况开展了一项研究。</p><p style="text-align: left;">他们发现：GitHub 用户使用 Emoji 表情的方式确实和许多与工作相关的行为模式存在相关性，比如用户的活动水平、从事的工作类型，沟通方式和时间管理技能等，甚至有可能通过表情符号的使用情况评估用户退出 GitHub 的概率。</p><p style="text-align: left;">相关研究论文以“Emojiis Predict Dropouts of Remote Workers： An Empirical Study of Emoji Usage on GitHub”为题，已发表在预印本网站 arXiv 上。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0406/fa4f7617ea78331.png" alt="1.png" referrerpolicy="no-referrer"></p><p style="text-align:center">Emojis predict dropouts of remote workers： an empirical study of emoji usage on github | arXiv</p><p style="text-align: left;">那些年我们常用的 Emoji 表情包</p><p style="text-align: left;">关于为何进行这项研究，论文作者之一、密歇根大学信息学院研究员陆璇表示，“在远程工作期间，人们往往难以摸清同事的工作情绪和心理健康状况，而这不仅仅是在疫情期间才有的挑战，随着未来人们可能会越来越多地依赖远程工作方式，我们认为这一问题更加值得关注。但好在，虽然人们在远程工作中面对面交谈的机会变少，但人们会在线工作交流中使用 Emoji 表情。或许，我们可以通过分析这些 Emoji 表情，来有效地捕捉人们的情绪态度和心理变化。” 陆璇也是北京大学信息科学技术学院计算机科学技术系软件工程研究所的博士生。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0406/2cfcd94f81892b3.png" alt="2.png" referrerpolicy="no-referrer"></p><p style="text-align:center">陆璇|北京大学</p><p style="text-align: left;">于是，他们收集了 2018 年期间大约 1000 万 GitHub 用户撰写的 6300 万个帖子，并作为样本数据进行研究，帖子类型包括但不限于 issues、comments、pull requests 等。</p><p style="text-align: left;">研究人员首先希望弄明白的是：用户在 GitHub 交流中使用表情符号的程度/频率，尤其是开发人员在日常工作活动中以及使用不同编程语言时是否经常使用表情符号。结果发现，这些帖子中有 3%-14% 包含表情符号，根据不同类型和使用不同的编程语言，表情符号的使用频率也有所不同。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0406/d9e9d2f3c8ccfc6.png" alt="3.png" referrerpolicy="no-referrer"></p><p style="text-align:center">GitHub上不同帖子类型中最常见的10个表情符号|参考文献[1]</p><p style="text-align: left;">有趣的是，用户在 Twitter 上最常使用的 10 个表情与在 GitHub 上竟无丝毫重叠之处，它们分别是：</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0406/8f2da81cbe5fafb.png" alt="4.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">根据统计结果，GitHub 上最受欢迎的是 ???，这些都与工作有关，例如 ?? 往往用于表示“部署、启动”；? 表示“完成、无误”， 则在评论中最常见，表达对某个帖子的赞赏之意。</p><p style="text-align: left;">表情符号的使用与各类工作指标有关联</p><p style="text-align: left;">上述分析表明，GitHub 社区的开发人员在工作相关的交流中的确经常使用 Emoji 表情。但仍需确定的是，Emoji 表情的使用情况是否反映了他们真实的工作状态，换句话说，是与用户的工作相关活动或其他个人工作身份/偏好有关？还是仅仅是 GitHub 社区的一种交流规范？于是，研究人员通过建立一个机器学习模型来试图弄清楚这个问题。</p><p style="text-align: left;">通过分析上述约 1000 万 GitHub 用户发布的帖子之后，他们发现包含 Emoji 表情的帖子占所有数据的 52.78%。而为了控制偏差，研究人员将这些 Emoji 表情用户与同等数量的非 Emoji 表情使用者（即至少发过一篇帖子，但在 2018 年没有使用任何表情符号的用户）随机样本进行匹配，并提出了 5 种影响开发人员工作状态的变量，分别是活动级别（activity level）、活动趋势（trend of activity）、活动类型（type of activity）、帖子类型（Type of posts）和时间管理（time management）。</p><p style="text-align: left;">而后，研究人员利用受控回归分析，得出了表情符号使用率与 5 个工作相关变量之间的联系，如下图所示：</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0406/0d478259481cbf3.png" alt="5.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">往往工作时间越长，活跃用户就会写更多带有 Emojii 表情的帖子（左）；而 push 活动比例越高的用户，使用 Emoji 表情的情况会大大减少（右）| 参考文献[1]</p><p style="text-align: left;">这些分析表明，开发人员使用表情符号的方式各不相同，而且 Emoji 表情的使用方式与开发人员的工作状态以及其他许多与工作相关的行为模式具有相关性。</p><p style="text-align: left;">Emoji 表情或成为管理者的“好帮手”</p><p style="text-align: left;">此外，陆璇还表示：“这项研究表明，Emoji 表情的使用能够预示开发人员从在线工作平台退出的可能性。我们已经发现，在帖子中使用 Emoji 表情的开发人员从 GitHub 退出的可能性大大降低。”</p><p style="text-align: left;">这一发现可能有助于开发一些基于 Emoji 表情的工具，以预测远程工作者的工作满意度；或者系统可以检测出愤怒，焦虑和沮丧的迹象，从而为那些需要帮助的员工提供支持。此外，这一发现还可能会激发公司制定相应的策略，比如通过鼓励远程工作者使用 Emoji 表情来表达积极和消极的情绪，来提高他们的幸福感和工作满意度。</p><p style="text-align: left;">“目前，我们正在研究 Emoji 表情的使用与工作相关结果之间的因果关系。更具体地说，我们现在正在调查 Emoji 表情是否可以作为一种工作，以提高远程工作者的工作效率和降低员工离职率。” 陆璇说。</p><p style="text-align: left;">而对员工来说，这其实也不失为一个好消息。因为从此以后，公司群的聊天记录一定会非常和谐。</p><p style="text-align:
