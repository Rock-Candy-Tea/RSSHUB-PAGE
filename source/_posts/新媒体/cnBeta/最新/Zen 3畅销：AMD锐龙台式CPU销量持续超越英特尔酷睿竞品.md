<strong>昨日，WCCFTech 讨论了过去一段时间里的 GPU 销售统计数据。今天，该网站继续分享了 AMD / Intel 台式处理器的销售趋势。</strong>首先根据 MindFactory 的最新数据，AMD Zen 3 锐龙台式 CPU 的受欢迎程度，依然领先于英特尔的酷睿竞品。<br>
<p><a href="https://static.cnbetacdn.com/article/2021/0804/076f45bf60dd270.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/076f45bf60dd270.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/amd-ryzen-desktop-cpus-continue-to-outsell-intel-core-cpus-zen-3-best-seller-at-several-retailers/" target="_self">WCCFTech</a>）</p><p>尽管过去几个月的总销量大幅下滑，但 AMD CPU 还是在 6-7 月份保持着 10000+ 的销量。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/595dfafa7aec536.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/595dfafa7aec536.png" alt="1-1.png" referrerpolicy="no-referrer"></a></p><p>与此同时，虽然英特尔的销量不足 5000，但也较上月有着明显的增长。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/02f680033ace4bd.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/02f680033ace4bd.png" alt="1-2.png" referrerpolicy="no-referrer"></a></p><p>对于蓝队来说，上月也是在一年多来达成的最高月度销售份额（AMD 76% / Intel 24%）.</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/5b3d39b272160db.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/5b3d39b272160db.png" alt="1-3.png" referrerpolicy="no-referrer"></a></p><p>型号占比方面，消费者对 AMD 锐龙 R5-5600X、R7-5800X、R9-5900X 的需求依然很大。仅 R5-5600X 这款热门型号的销量，就与英特尔酷睿全家桶相当。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/283cc3d9698b7e6.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/283cc3d9698b7e6.png" alt="1-4.png" referrerpolicy="no-referrer"></a></p><p>AMD 占据了超过 80% 的份额，而英特尔离 20% 还差一些。7 月 AMD 锐龙桌面 CPU 在该网站的销售额不到 400 万美元，英特尔酷睿 CPU 则是不到 100 万欧元。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/72b701e8b0e2a5d.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/72b701e8b0e2a5d.png" alt="2-1.png" referrerpolicy="no-referrer"></a></p><p>平均销售价格方面，可看到目前的 CPU 价格处于相当稳定的水平，且 AMD 这边回落明显，这显然要归功于 Zen 3 CPU 有了更好的供应。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/b8f5d65ad6400ca.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/b8f5d65ad6400ca.png" alt="2-2.png" referrerpolicy="no-referrer"></a></p><p>英特尔 9 / 10 代处理器整体价格也有所下降，消费者对这些较旧型号的接纳度也有所增加。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/c11a868e5c4c1b8.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/c11a868e5c4c1b8.png" alt="3-1.png" referrerpolicy="no-referrer"></a></p><p>AMD 锐龙 R5-5600X 仍是迄今为止最畅销的产品，其在多家零售商的报价都已低于首发时的 299 美元建议零售价（甚至 $279 就能拿到）。对于新组装 PC 的 DIY 玩家来说，当前也是不错的入手时机。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/487c1338fe0abc8.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/487c1338fe0abc8.png" alt="3-2.png" referrerpolicy="no-referrer"></a></p><p>AMD 锐龙 R7-5800X 与 R9-5900X 也在销售榜上占据前列，但上一代锐龙 3000 系列桌面 CPU 的性价比要更为显著，所以 R5-5600X、R7-3700X 和 R9-3900X 在其它零售网站的销售也很是强劲。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/e5321f7edbde0d7.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/e5321f7edbde0d7.png" alt="3-3.png" referrerpolicy="no-referrer"></a></p><p>当然英特尔 11 代处理器也不是全线溃败，至少酷睿 i7-11700K 似乎处于领先地位，其次是 i9-11900K 。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/b09e5f16b13e9ec.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/b09e5f16b13e9ec.png" alt="4-1.png" referrerpolicy="no-referrer"></a></p><p>展望未来，<a href="https://www.reddit.com/user/ingebor/" target="_self">Ingebor</a> 认为 AMD 将于 2021 年起逐渐偏离“预算解决方案”，理由是当前该公司芯片的平均销售价格更高。</p><p>鉴于最新的 Zen 3 架构已基于台积电 7nm 工艺节点，预计下一代芯片还会继续通过制程福利来换取性能的大幅提升。</p><p>随着基于 5nm 制程的 Zen 4 处理器在明年晚些时候亮相，这或许促使 AMD 抬升发烧领域的产品报价。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/b2925df9c3aa4de.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/b2925df9c3aa4de.png" alt="4-2.png" referrerpolicy="no-referrer"></a></p><p>作为参考，Steam 硬件调查显示，AMD 锐龙 CPU 已经达成了 30% 的份额，x86 台式机的总体份额也攀升至 20% 。</p><p>需要指出的是，尽管该公司于 2020 年底出货了百万颗 Zen 3 芯片，但供应相对受限。</p><p>预计 2021 上半年的供应没有太大缓和，但全球计划应该会轻松超越 2020 年的数字。</p><p><a href="https://static.cnbetacdn.com/article/2021/0804/bf6180ab8db2d8c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0804/bf6180ab8db2d8c.png" alt="4-3.png" referrerpolicy="no-referrer"></a></p><p>在查看美国地区的新蛋、亚马逊、百思买等网站销售排行的时候，你会发现锐龙 5000 / 3000 系列台式 CPU 几乎充满了整个页面。</p><p>不过除了主流的锐龙（Ryzen）产品线，面向 HEDT / 工作站细分市场的线程撕裂者（Threadripper / Threadripper Pro）芯片，也获得了远超竞争对手的成绩。</p>
