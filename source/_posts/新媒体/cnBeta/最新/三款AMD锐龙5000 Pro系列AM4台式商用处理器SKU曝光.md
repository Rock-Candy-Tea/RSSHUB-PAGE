<strong>除了即将上市的 Zen 4 锐龙 7000 系列台式 AM5 CPU，AMD 显然也没有放弃为嵌入式 / 商用市场开发更多 AM4 处理器。</strong>早些时候，知名爆料人 HXL（@9550pro）在 Twitter 上晒出了来自联想官网的截图链接，表明 ThinkStation P358 塔式商用主机将可选配 AMD 锐龙 R9-5945 / R7-5845 / R5-5645、以及现有的 R3-4350G 等“Pro”版 CPU 。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0808/12769e133cae7f2.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（via <a href="https://wccftech.com/amd-ryzen-5000-pro-am4-desktop-cpus-spotted-ryzen-9-5945-ryzen-7-5845-ryzen-5-5645/" target="_self">WCCFTech</a>）</p><p>几天前，WCCFTech 报道了 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 5000 嵌入式 AM4 处理器阵容。</p><p>现在，联想官网又为 ThinkStation P358 塔式商用主机提供了锐龙 R3-4350G 之外的新选项。</p><p>据悉，该系列机型主要围绕插槽式 AM4（PGA 1331）平台设计，且支持非 ECC 的 DDR4-3200 内存。</p><p><img src="https://static.cnbetacdn.com/article/2022/0808/9414847301daa46.png" alt="2.png" referrerpolicy="no-referrer"></p><p>如图所示，三款 AMD Zen 3 锐龙 Pro 5000 系列 CPU 分别为 —— Ryzen 9 Pro 5945、Ryzen 7 Pro 5845、以及 Ryzen 5 Pro 5645 。</p><p>尽管没有提及确切的核心配置与时钟速率，但我们不难推测 R9 SKU 拥有 16C / 32T、R7 SKU 为 8C / 16T、R5 SKU 为 6C / 12T，且 L2 与 L3 缓存也与同系列保持一致。</p><p><img src="https://static.cnbetacdn.com/article/2022/0808/5c5ce55b5ee8d8f.png" alt="3.png" referrerpolicy="no-referrer"></p><p>需要指出的是，AMD 当前的 Ryzen Pro AM4 产品线，主要涵盖了 5000 / 4000 系 APU SKU —— 纯 CPU 还得追溯到 Zen 2 架构的 Ryzen 3000 系列。</p><p>但是未来几个月内，我们有望很快迎来 AMD Ryzen 5000 系列“Pro”和“嵌入式”CPU 新品。</p><p><a href="https://static.cnbetacdn.com/article/2022/0808/9025f11632f431b.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0808/9025f11632f431b.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（ThinkStation P358 Tower <a href="https://psref.lenovo.com/Product/ThinkStation/ThinkStation_P358_Tower35" target="_self">传送门</a>）</p><p>至于 Zen 4 架构的锐龙 7000 系列，短时间（几年内）应该是不会迎来 Pro 或 Embedded 的衍生 SKU 了。</p>
