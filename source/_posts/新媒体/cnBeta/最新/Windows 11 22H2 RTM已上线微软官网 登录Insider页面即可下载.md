Windows 11 第一个大版本更新现已敲定。有人发现了一份微软刚发布的官方文件，硬件认证页面中提到 Windows 11 22H2 版本的 RTM 构建即为 22621（最低版本号）。随着 Windows 11 的更新，22H2 版本将会在今年秋季正式发布，然后慢慢出现在所有新机和已安装 Windows
11 的用户 PC 上。<br>
 <p>此外，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还公开了一些非常具体的细节，这似乎是微软首次公开承认 RTM 细节。该公司写道：</p><blockquote><p>Windows 11, version 22H2 certification build</p><p>Minimum acceptable build for Windows 11, version 22H2 certification is RTM build(i.e. 22621).</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0526/67ec1ac8258b9ac.jpg" referrerpolicy="no-referrer"><br></p><p>从微软的话术来看，Windows 11 中似乎不再存在某个特定的版本作为 RTM，只要满足最低版本号为 Build 22621 的系统都可以作为最终版本交付。Windows 11 版本 22H2 后续也将会在新 PC 上或面向公众发布之前以及之后不断进行更新。</p><p>科普：Windows 操作系统在零售商店正式上架前需要一段时间来压片，包装和配销。所以程序代码必须在正式发行前一段时间就要完成，这个完成的程序代码叫做 Final.Code。</p><p>RTM 即 Release to Manufacturing 或者 Release to Marketing，即交付给 OEM 和市场的版本。</p><p>大约两周前，微软首次面向 Beta 频道推送了 Windows 11 build 22621 版本 。当时，微软指出“桌面右下角的版本水印已经不再存在，但水印后续将会在新版本中重新出现”，这显然它确实意味着它已经完成了 22H2 版本的选择，而且还有爆料称微软可能会在今天向 OEM 交付 RTM 版本。</p><p>现在来看，微软确实选定了 22H2 版本，但微软并非公开进行。</p><p>据了解，Windows 11 Build 22621 早在 5 月 11 日就发布到了 Beta 频道，以及一个 SDK。此外，微软那时还提供了 ISO 文件供大家下载，所以如果你希望能提前体验到 Windows 11 第一次重大更新的话，你现在就可以下载 22H2 Beta 版本。</p><p><strong>官方下载链接（英文）：<a href="https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewiso" target="_blank">https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewiso</a></strong></p><p><strong>如何下载 Windows 11  ISO 镜像</strong></p><p>要下载 Windows 11  ISO 镜像，请按照以下步骤进行。</p><p>·在这里注册或登录 Windows Insider 项目。</p><p>·前往微软网站的下载页面。</p><p>·点击“选择版本”。</p><p>·选择 Windows 11 Insider Preview (Beta Channel)- Build 22621</p><p><img src="https://static.cnbetacdn.com/article/2022/0526/2466961d6718953.jpg" referrerpolicy="no-referrer"><br></p>
