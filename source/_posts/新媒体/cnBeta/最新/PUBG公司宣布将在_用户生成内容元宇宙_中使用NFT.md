即便最近许多游戏公司 NFT 项目的公告都受到了玩家的强烈反对，《PUBG》开发工作室 Krafton 依然宣布它即将开发一款 NFT
游戏。该公司已与元界平台 Zepeto 的运营商 Naver Z 签署了一项协议，“开发一个新的 Web 3.0 和NFT 项目，旨在构建一个
NFT 元宇宙平台。”<br>
<p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0210/ae2fb6cc9922e06.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0210/ae2fb6cc9922e06.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0210/ae2fb6cc9922e06.jpg" referrerpolicy="no-referrer"></a></p><p>没有迹象表明 Krafton 的 NFT 项目将直接与《绝地求生》相关联——相反，该公司将利用其在开发和运营《绝地求生》过程中获得的经验来创建“用户生成内容 (UGC) 创建工具”，以及使用虚幻引擎构建的虚拟世界。根据声明，Naver Z 将“管理元宇宙服务”，并处理社区和社会服务。</p><p>Krafton Web 3.0 团队负责人 HyungChul Park 说：“Krafton 和 Naver Z 之间的这种伙伴关系得到了每家公司独特的专业知识和共同的全球关注的支持，因为他们展望了由 NFT 和元宇宙驱动的未来。通过将 Krafton 的现有技术和关于如何构建可扩展的 Web 3.0 创作者驱动的生态系统的研究与 Naver Z 和 Zepeto 的经验和能力相结合，我们相信我们可以构建一个与其他服务不同的高质量 UGC 驱动的开放元宇宙并通过 NFT 技术激活全球创作者经济。”</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0210/6889ac32112edc1.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0210/6889ac32112edc1.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0210/6889ac32112edc1.jpg" referrerpolicy="no-referrer"></a></p><p>在这个早期阶段这可能没问题，因为没有人完全清楚元宇宙是什么，但 Krafton 的风险在于它忽略了对游戏相关 NFT 的日益强烈的反对。</p><p>将 NFT 直接整合到《潜行者2》中的计划在收到极为负面的反馈后被放弃，最近独立发行商 Team17 面对游戏玩家和自己的工作室的反对，放弃了自己的 NFT 战略。</p><p>Krafton 有可能并不会遇到相同的抵制。如果该项目与《绝地求生》完全独立且截然不同，那么游戏玩家可能不会太在意该公司的 NFT 野心。但负面影响的可能性是一个永远存在的风险：NFT 领域仍然充斥着诈骗和虚假宣传。</p><p>Krafton 表示，将在“未来”分享有关其与 Naver Z 的 NFT 合作伙伴关系的更多细节。</p>
