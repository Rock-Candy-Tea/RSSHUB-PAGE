<strong>W3C最新发布的JavaScript API是Web神经网络API（Web Neural Network
API）的公开工作草案。</strong>W3C和关联方一直在开发网络神经网络API，作为一种允许神经网络推理从浏览器硬件加速的手段。<br>
 <p>此前，W3C在今年早些时候成立了一个网络机器学习工作小组。Google和<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>是参与该工作组的公司之一，他们希望能够利用专用的机器学习硬件加速器和网络浏览器中的本地指令。</p><p>WebNN的设计考虑到了诸如人物检测、人脸识别、超级分辨率、图像标题、情感分析、噪声抑制等使用情况。</p><p>对于那些担心任意网站能够利用机器学习硬件的人来说，这个新API的安全考虑包括在所有跨源框架中默认禁用它，并与WebGPU和WebGL的安全问题联系起来。</p><p><a href="https://static.cnbetacdn.com/article/2021/06/bdf09b770d15730.png" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/06/bdf09b770d15730.png" referrerpolicy="no-referrer"></a></p><p>网络神经网络API的第一个公开工作草案可以在W3.org上找到：</p><p><a href="https://www.w3.org/TR/2021/WD-webnn-20210622/" _src="https://www.w3.org/TR/2021/WD-webnn-20210622/" target="_blank">https://www.w3.org/TR/2021/WD-webnn-20210622/</a><br></p>
