<strong>My Laptop Guide 刚刚泄露了华硕 ROG Zephyrus Duo 16 GX650 笔记本电脑的旗舰规格，可知其搭载了 AMD 锐龙 R9-6900HX 处理器 + 英伟达 GeForce RTX 3080 Ti GPU 。</strong>鉴于锐龙 R9-6900HX 属于“Rembrandt-H”家族的一员，这款旗舰游戏本也有望在明年 1 月的消费电子展（CES 2022）期间正式亮相。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1130/926c141a98820bf.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1130/926c141a98820bf.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p><a href="https://wccftech.com/asuss-next-gen-rog-zephyrus-duo-16-gx650-laptop-rocks-amd-ryzen-9-6900hx-rembrandt-cpu-nvidia-geforce-rtx-3080-ti-gpu/" target="_self">WCCFTech</a> 指出，华硕 ROG Zephyrus Duo 16 GX650 系列笔记本电脑将提供三档配置，而 GX650RX 就是三者中配置最高的一款。</p><p>此前我们已经在 ROG STRIX SCAR 15 / 16 笔记本电脑的泄露爆料中得知 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 6000H CPU 和英伟达 RTX 3080 Ti 移动 GPU 。</p><p>那参考产品线的定位，ROG Zephyrus Duo 16 GX650 显然也会用上基于 6nm 工艺和 8C / 16T Zen 3 CPU 核心的锐龙 R9-6900HX 处理器。</p><p><a href="https://static.cnbetacdn.com/article/2021/1130/2f1924f99aad401.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1130/2f1924f99aad401.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://mylaptopguide.com/asus-rog-zephyrus-duo-16-gx650-leaked-comes-with-ryzen-9-6900hx-and-rtx-3080-ti/" target="_self">MyLaptopGuide</a>）</p><p>虽然尚不清楚 R9-6900HX 的 CPU 主频，但我们大可预计它会较 R9-5900HX 略高一些，并且集成规格尚不明确的 RDNA 2（Navi 2x）核显。</p><p>不过在独显方面，基于 Ampere GA103 GPU 的英伟达 RTX 3080 Ti，其 TDP 功耗可在 150W ~ 200W 之间（传闻称有标准 / Max-Q 两种版本）。</p><p>GA103 GPU 包含了 58 个计算单元 / 7424 个 CUDA 核心（频率 1395 MHz），桌面版规格要高出 17%、但移动版在频率和能效上有所优化，此外搭配了 16GB GDDR6 显存（或为 256-bit @ 12Gbps 速率）。</p><p><a href="https://static.cnbetacdn.com/article/2021/1130/71a7e806422f28e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1130/71a7e806422f28e.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>华硕 ROG Zephyrus Duo 16 GX650RX 的其它规格，包括最高 48GB 的 DDR5-4800 内存 + 8TB PCIe NVMe M.2 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a>（爆料机型仅 16GB RAM + 1TB SSD），以及 16 英寸 @ 300Hz 的 FHD 防眩光屏幕。</p><p>若消息靠谱，那这台机器也将是首款支持 DDR5 和 PCIe 4.0 的 AMD 高端游戏本，有望向<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> Alder Lake-P 竞品发起更直接的挑战。</p><p>最后，预计该机零售价格可轻松超过 2500 美元（1.6 万 RMB），并于明年 1 月的消费电子展（CES 2022）期间正式亮相。</p>
