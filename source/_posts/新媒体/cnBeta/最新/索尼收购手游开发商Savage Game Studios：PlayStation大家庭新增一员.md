索尼 PlayStation 部门刚刚宣布了对 Savage Game Studios 的收购。<strong>作为一家扎根德国（柏林）与芬兰（赫尔辛基）的游戏开发团队，其专注于为移动平台开发动作类游戏。</strong>公司Savage Game Studios 首席执行官兼联合创始人 Michail Katkoff 向 PlayStation 玩家介绍道，他与 Nadjim Adjir 和 Michael McManus 携手，于 2020 年成立了该工作室。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0829/c9afbb42260226d.webp" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：PlayStation <a href="https://blog.playstation.com/2022/08/29/welcoming-savage-game-studios-expanding-our-community/" target="_self">Blog</a>）</p><p><strong>Michail Katkoff 表示：</strong></p><blockquote><p>得益于多年的移动游戏开发经验，Savage Game Studios 打造了一系列享誉全球的 IP 。</p><p>公司致力于提供一个创造性的空间，并不那么谨慎避免、而是热切地接纳并开展各项实验和冒险。</p><p>一方面，我们尊重在大工作室工作时可获得的充足资源优势。另一方面，我们也希望保持小而灵活、以及自我掌控。</p></blockquote><p><strong>至于为何加入 PlayStation Studios，Michail Katkoff 解释称：</strong></p><blockquote><p>之所以达成这笔交易，是因为我们相信 PlayStation Studios 领导层会尊重我们如何在最佳运营和取得成功的愿景之间取得平衡，并且他们也不害怕冒险。</p><p>再加上对 PlayStation 惊人的 IP 资源库的潜在利用，Savage Game Studios 可更好地受益于这方面的支持，于是大家纷纷表示‘为何不这么做呢？！’</p><p>在此，请允许我代表 Savage Game Studios 的每一个人，感谢 PlayStation Studios 的热忱邀请。我们正迫不及待地向大家展示一直在做的事情！</p></blockquote><p><strong>最后，PlayStation Studios 负责人 Hermen Hulst 向粉丝们保证</strong> —— 在加强移动端努力的同时，PlayStation 并不会削弱现有的主机平台、以及不断增长的 PC 体验，且 Savage 已在开发一款 3A 动作游戏。</p><blockquote><p>通过加强移动游戏方面的工作，我们希望为更多人提供体验 PlayStation IP 的更多方式、并努力吸引那些对 PlayStation 游戏不那么熟悉的新受众。</p><p>Savage Game Studios 将加入新成立的 PlayStation Studios 移动部门，它将独立于现有的主机开发运营团队、并专注于基于现有和新 PlayStation IP 的创新移动体验。</p>无论你钟爱主机、PC、还是移动平台，此刻还请随我们一同欢迎 Savage Game Studios 加入 PlayStation Studios 大家庭。</blockquote>
