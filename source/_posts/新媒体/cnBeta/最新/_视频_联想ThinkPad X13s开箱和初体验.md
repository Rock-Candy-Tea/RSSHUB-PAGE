<strong>国外科技媒体 OnMSFT 近日对联想的新款 ThinkPad X13s 进行了评测，这是一款打造 Windows on ARM
的旗舰笔记本系统，使用高通的骁龙 8cx Gen 3 SoC。</strong>联想官方售价为 1439 美元，加上含税和快递费用后实际购入价格接近 1500
美元，配备了 32GB 内存和 512GB PCIe SSD。<br><br>
 <p><iframe src="//player.bilibili.com/player.html?bvid=BV1Yv4y1M7qR&page=1" width="750" height="480" frameborder="0"></iframe></p><p>该笔记本屏幕支持触控，分辨率为 1920 x 1200，额定亮度为 300 尼特。电源按钮上还有一个 500 万 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Hello IR 网络摄像头和一个指纹读取器。本次该媒体购入的是非 5G 版本，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 可以后续升级，但是内存是焊死的无法升级，搭载 Windows 11 Pro 系统。</p><p><img src="https://static.cnbetacdn.com/article/2022/0621/1b68a4d7c9d6c72.webp" alt="010ary7b.webp" referrerpolicy="no-referrer"></p><p><strong>初体验</strong></p><p>在撰写评测文章的时候，该媒体仅上手体验了 1 天时间，因此没有办法做到深入的探索。该媒体测试了包括 Geekench 5、<a data-link="1" href="https://microsoft.pvxt.net/zaZYr" target="_blank">Visual Studio</a> 2022（ARM 版本）和 Chrome 中的 Web 浏览。后续还将会提供完整的评测文章。</p><p><strong>● 网页浏览</strong></p><p>对于网页浏览，这款 ThinkPad 非常适合。由于 Microsoft Edge 针对 ARM 进行了优化，这并不令人意外。 Chrome 浏览器也表现得很高效，该媒体编辑没有注意到任何减速或其他问题。 Microsoft Edge 和 Chrome 并排运行，就像基于 Intel 的系统上一样，并没有冻结。</p><p><img src="https://static.cnbetacdn.com/article/2022/0621/bcf3257ca5f51c3.webp" alt="gvzs0642.webp" referrerpolicy="no-referrer"></p><p>他认为在这台机器上浏览网页的一般性能会很棒。仅打开几个 Edge 选项卡以及其他几个系统应用程序，8cx Gen 3 的利用率几乎没有超过 6%。我也可以说仿真在我早期的测试中也很有效。像 Chrome（尚未 ARM 优化）这样的应用程序不会占用那么多 CPU 使用率。</p><p><strong>● GeekBench 5</strong></p><p>ThinkPad X13s 的单核成绩为 1118 分，多核成绩为 5776。这是几代人之间的巨大差异，因为带有 Snapdragon 8cX Gen 2 的 Flex 5G 获得了 698 的单核分数和 2551 的多核分数。正如我 Twitter 上的某个人指出的那样，即使是 Windows 笔记本电脑，这也是一个适中的分数。</p><p>不过，与 M1 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">MacBook</a> Air 相比，它确实落后了。 M1 MacBook Air 的单核得分为 1706，多核得分为 7420。</p><p><strong>● Visual Studio 2022</strong></p><p>该媒体编辑安装该应用程序的 ARM 优化版本的预览版，并且在大约 18 分钟内下载并安装了所有三个开发人员工作负载。由于他不是开发人员，也不打算使用此设备进行编码。该应用程序将正常运行，因为它已在预览中进行了优化！在其他开发人员领域，该设备运行适用于 Linux 的 Windows 子系统，并且还支持 Hyper-V。</p><p><img src="https://static.cnbetacdn.com/article/2022/0621/59b7ae1f6cb2cfb.webp" alt="mp1bnp16.webp" referrerpolicy="no-referrer"></p><p><strong>小结</strong></p><p>到目前为止，我对我的新 ThinkPad X13s 印象深刻。总体而言，镁合金的制造质量非常好，就像 ThinkPad 一样，虽然有点像“塑料”，以保持轻巧纤薄的外形。总的来说，这绝对是一款优质的笔记本电脑，而不是像 Galaxy Book Go 5G 这样的廉价笔记本电脑。</p><p><img src="https://static.cnbetacdn.com/article/2022/0621/b4d10476baf8c1d.webp" alt="z6sz5v2d.webp" referrerpolicy="no-referrer"></p>
