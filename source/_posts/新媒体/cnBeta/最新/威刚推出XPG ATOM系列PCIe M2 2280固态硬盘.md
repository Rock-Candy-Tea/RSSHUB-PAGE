<strong>威刚（ADATA）旗下高端电竞品牌 XPG，刚刚推出了面向不同预算需求的创作者们的 XPG ATOM 30 / 40 / 50 系列固态硬盘产品线。</strong>其中 ATOM 30 / 40 系列采用了支持 NVMe 1.3 协议的 PCIe 3.0 x4 主控，顺序读写分别达到了 2500 / 2000 和 3500 / 3000 MB/s，并且兼容最新的 Intel / AMD PC 和台式机 / 笔记本电脑平台。<br>
  <p><img src="https://static.cnbetacdn.com/article/2021/1102/b43f0c783c75abd.jpg" referrerpolicy="no-referrer"></p><p>对于有更高性能需求的玩家，还可选择 ATOM 50 系列。其采用了支持 NVMe 1.4 协议的 PCIe 4.0 x4 主控，顺序读写可达 5000 和 4500 MB/s 。</p><p>在 SLC 缓存 / 主机内存缓冲技术的加持下，ATOM 30 / 40 / 50 系列 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 分别可达 180K / 200K、480K / 260K、以及 440K / 400K IOPS 的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_110018%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">4K</a> 随机读写访问速度。</p><p>容量方面，当前 XPG ATOM 系列最高仅提供了 1TB 的版本。三款型号还配备了低调优雅的散热片，可将温度降低 15%，以确保最佳性能和运行稳定。</p><p>在 LDPC 校验和 AES-256 加密技术的加持下，ATOM 系列 M.2 NVMe SSD 还可确保数据传输的准确性与安全性。</p><p>ATOM 系列 SSD 还选用了经过严格测试的高品质 IC，可实现最大耐用性，辅以原厂 5 年有限质保。</p><p>对于 PlayStation 5 主机玩家来说，还可将 ATOM 50 作为扩容盘来使用，请存储更多游戏和流畅加载。</p><p>最后，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://adata.jd.com/" target="_blank">威刚</a> XPG ATOM 系列 30 / 40 / 50 固态硬盘将登陆亚马逊等零售渠道，ATOM 30 的 250GB / 500GB / 1TB 版本售价为 59.99 / 79.99 / 139.99 美元（约 384 / 512 / 895 RMB）。</p><p>ATOM 40 的 512GB / 1TB 版本为 99.99 / 179.99 美元（约 640 / 1151 RMB），ATOM 50 的 512GB / 1TB 版本则是 119.99 / 179.99 美元（约 767 / 1151 RMB）。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1197905.htm" target="_blank">威刚发布Legend系列PCIe M.2 2280 SSD新品</a></p></div>
