<strong>今年 2 月，惠普（HPE）向国际空间站发送了 1 台计算机，该设备连接到了微软的 Azure 云。现在微软报告这个任务已经完成，Azure 已经从混合云设备收到了它的第一个数据。</strong>Spaceborne Computer-2 是 2017 年 Spaceborne Computer 的继任者，具有两倍的计算能力，并且可以让宇航员在不处理传输延迟的情况下访问计算资源。然而，使用 Azure 将能够在需要时无缝访问高计算负载。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0820/e29cea8fb7eb153.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/e29cea8fb7eb153.jpg" alt="0lxl6k8k.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">新闻稿称：“宇航员和太空探索者应该在终极边缘获得最好的云计算技术和先进的处理能力。有时分析需要在每时每刻都很重要的边缘立即完成，而有时分析非常复杂，只能借助超大规模云的力量来执行”。</p><p style="text-align: left;">HPE Spaceborne Computer-2 首席研究员 Mark Fernandez 在一份声明中表示 ：“HPE 和<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>正在合作，通过提供最先进的技术来满足在轨期间的一系列数据处理需求，从而进一步加速太空探索。通过将 HPE 的 Spaceborne Computer-2（ 基于 HPE Edgeline Converged Edge 系统以提供高级边缘计算和 AI 功能）与 Microsoft Azure 连接到云，我们使太空探索者能够将大型数据集无缝传输到和来自地球，并从边缘到云的体验中受益”。</p>
