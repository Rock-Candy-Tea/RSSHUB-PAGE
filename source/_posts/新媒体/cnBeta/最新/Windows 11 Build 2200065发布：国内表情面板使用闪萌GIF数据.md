<div>   
在面向 Dev 频道的 Windows Insider 项目成员发布 Windows 11 首个预览版更新 1 周后，微软今天进行了版本更迭，版本号升级到 Windows 11 Build 22000.65。本次版本更迭带来了大量的 BUG 修复和改进，以及细节上的调整。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/8ce305cccb0d292.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/8ce305cccb0d292.jpg" alt="07om8da8.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">和适用于 Windows 10 的 Dev 频道不同，Windows 11 更新在正式发布之前预估都是累积更新的形式，因为该系统的大部分内容都已经完成。这就意味着在正式发布之前，Windows 11 的版本号都会是 Build 22000 开头。不过，由于操作系统的模块化和组件化，该公司可能会通过功能体验包引入更多的 UI 变化，而不会影响到底层构建。</p><p style="text-align: left;">以下是 Windows 11 Build 22000.65 主要改进</p><blockquote style="text-align: left;"><p style="text-align: left;">● 开始菜单现在提供搜索框以便于用户更快找到想要的内容</p><p style="text-align: left;">● 任务栏现在可以跨多个显示器显示，可以通过设置>个性化>任务栏启用。</p><p style="text-align: left;">● 更新了几个系统提示对话框，如笔记本电脑电池电量不足时的提示、当你用新的 Windows 11 视觉设计改变显示设置时的提示。</p><p style="text-align: left;">● “电源模式” 设置现在可以在设置应用中的“电源和电池”页面上使用。</p><p style="text-align: left;">● 右键点击桌面现在直接有一个刷新选项，不需要点击 "显示更多选项"。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/94fcc78de47a574.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/94fcc78de47a574.jpg" alt="my87vsik.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">● 在文件资源管理器中右击一个 .ps1 文件现在直接有一个用 PowerShell 运行的选项，不需要点击“显示更多选项”。</p><p style="text-align: left;">● 在使用小屏幕的 PC 时，可用的快照布局已针对该姿势进行了优化。你现在可以选择将三个应用程序扣在彼此的上面，而不是四个窗口。</p><p style="text-align: left;">● 基于与中国最受欢迎的 GIF 供应商 weshineapp.com 的合作，GIF 选择现在可以通过表情符号面板（WIN + .）向中国的 Windows Insiders 提供。如果位置设置为中国，那么将提供来自于 weshineapp.com 网站的 GIF 动图。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/fd12a2e87cc4b24.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/fd12a2e87cc4b24.jpg"
