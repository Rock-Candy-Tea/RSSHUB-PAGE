面向已经升级到 Windows 11 的 Windows Insider 项目成员，微软将面向 Dev 频道用户发布不太稳定且更具实验性的版本。不过这些测试的新功能不再指定某个功能更新，因此不会在今年晚些时候发布。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0824/801c874742dabb7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0824/801c874742dabb7.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0824/4d5c0a68764c19b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0824/4d5c0a68764c19b.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0824/bd0176064a206f6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0824/bd0176064a206f6.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">对于那些期望更稳定的 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Insider 项目成员，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>推荐用户可以切换到 Beta 频道。不过HTNovo 报告说，这对许多人来说似乎是不可能的。许多 Insiders 报告说他们无法从 Dev 频道切换到 Beta 频道，因为这些选项在 Windows 更新设置中显示为灰色。</p><p style="text-align: left;">切换频道的旧注册表技巧似乎也不起作用，虽然您可以切换出 Dev 频道，但最终会进入 Release Preview 而不是 Beta 频道。目前似乎没有解决此问题的方法，可以快速关闭开关的窗口。</p>
