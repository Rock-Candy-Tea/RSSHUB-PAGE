<strong>虽然引发高度关注的 OPPO Find N 折叠屏智能机没有选择出海，但据 Yogesh Brar 爆料，该公司似乎有意在今秋带来一些惊喜。</strong>此外鉴于该公司早前在欧盟知识产权局（EUIPO）注册了 Find N Fold 和 Find N Flip 的新商标，推测两款新机会在几周后露面。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0813/5c387ca1b4b4d5e.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（via <a href="https://pricebaba.com/blog/exclusive-oppo-working-on-two-new-foldable-smartphones-powered-by-snapdragon-8-gen-1" target="_self">Pricebaba</a>）</p><p>尽管曝光的机型名称，很像是直接对标<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a>的两条折叠屏产品线。</p><p>但还有待观察的是，Find N Fold 是否会延续 Find N 的设计风格，辅以更紧凑的外形尺寸？</p><p>或者 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://oppo.jd.com/" target="_blank">OPPO</a> 是否有意拉伸 Find N Fold，以和三星 Galaxy Z Fold4 一较高下？</p><p><img src="https://static.cnbetacdn.com/article/2022/0813/ff04f0c2f102929.png" alt="2.png" referrerpolicy="no-referrer"></p><p>至于 Find N Flip，它显然是一条对标三星 Galaxy Z Flip4 的新产品线。配置方面，预计两款新机都会用上高通骁龙 8+ Gen 1 旗舰芯片组。</p><p>最后，国行 Find N 确实极具竞争力。但对于 Find N Fold 与 Find N Flip 的海外定价策略，目前暂不得而知。</p>
