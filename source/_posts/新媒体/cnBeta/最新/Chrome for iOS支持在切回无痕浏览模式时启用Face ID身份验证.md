Bleeping Computer 报道称，<strong>谷歌正在为 Chrome for iOS 测试一项酷炫的新功能，允许用户通过 Face ID 来解锁该浏览器的隐身（无痕）浏览模式。</strong>这意味着用户可以安心地暂时切换到其它任务，并在返回 Chrome for iOS 时被要求确认用户身份，以避免隐私浏览的内容泄露。考虑到 Face ID 的特性，实际体验可以做到相当无缝。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0622/a5f5a8fe9b9c910.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>感兴趣的朋友，可移步至 Chrome for iOS 的 Chrome://flags 设置页面来启用该标记。</p><p><img src="https://static.cnbetacdn.com/article/2021/0622/647ee40035b3823.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>方法是在“设置 -> 隐私”页面下方找到“为无痕模式启用设备身份验证”的新设置项。</p><p><img src="https://static.cnbetacdn.com/article/2021/0622/5295dcf93eb405f.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>如果哪天觉得每次切回 Chrome for iOS 的无痕模式都感到麻烦（比如需要长时间佩戴口罩），亦可参考上述方法来禁用该功能。</p>
