在昨日报道中，Techdows 介绍了 Edge 浏览器的侧边栏改进。<strong>现在，我们又见到了适用于 Windows 11 平台的 Edge 标题栏与工具栏 Mica 外观更新。</strong>这种实验性质的“云母”视觉效果，现已能够在 Edge 102 Canary 尝鲜分支上体验。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0423/575ea5f1591c33d.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">Edge Canary（<a href="https://www.microsoftedgeinsider.com/zh-cn/download/canary" target="_self">下载地址</a>）</p><p>一年前，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>为主菜单、情境菜单和 Edge 浏览器的部分圆角，引入了适配 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 操作系统的 Fluent 设计语言，比如亚克力（Acrylic）和云母（Mica）效果。</p><p><a href="https://static.cnbetacdn.com/article/2022/0423/e2a10ffb62d6426.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0423/e2a10ffb62d6426.png" alt="0.png" referrerpolicy="no-referrer"></a></p><p>与其它重新设计的第一方应用程序一样（包括画图、记事本和照片），这家雷德蒙德软件巨头希望让 Microsoft Edge 能够看起来与系统整体更加一致。</p><p><img src="https://static.cnbetacdn.com/article/2022/0423/ca883a8ec9b0458.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图自：<a href="https://techdows.com/2021/08/chromium-browsers-get-modern-scrollbar-windows-11.html" target="_self">Techdows</a>）</p><p>在条栏（Edge <a href="https://techdows.com/2021/09/microsoft-edge-web-widget-windows-11.html" target="_self">bar</a>）和覆盖滚动条（Overlay <a href="https://techdows.com/2021/08/chromium-browsers-get-modern-scrollbar-windows-11.html" target="_self">Scrollbar</a>）之后，微软正积极将 Mica 外观带入 Edge 浏览器的工具栏和标题栏的背景。</p><p><img src="https://static.cnbetacdn.com/article/2022/0423/d6ff74ba5960a6d.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p><strong>感兴趣的朋友，可参照如下方法来开启：</strong></p><blockquote><p>（1）打开 Edge 浏览器；</p><p>（2）在地址栏输入 edge://flags 并跳转；</p><p>（3）在搜索‘外观’菜单下的对于‘显示实验性外观设置项’；</p><p>（4）在下拉列表中选择启用。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0423/e1485d77bed10bb.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p><strong>重启 Edge 浏览器后：</strong></p><blockquote><p>（5）转至‘菜单 → 设置 → 外观’；</p><p>（6）启用‘在标题栏和工具栏中显示 Windows 11 视觉效果’这项预览功能；</p><p>（7）点击 重启 浏览器，以使设置生效。</p></blockquote><p>需要注意的是，请先确保选择了 Edge 默认主题，然后再检查 Windows 11 的个性化设置是否与之相匹配。</p><p>如需在 Windows 11 上禁用 Edge 工具栏和标题栏上的 Mica 效果，可在上面第（6）步骤关掉。</p><p>最后，由于这项预览功能仍在开发中，所以微软也提醒可能遇到一些意料之外的性能等问题。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1261367.htm" target="_blank">Edge Canary 102新特性：可启用实验性外观设置</a></p></div>
