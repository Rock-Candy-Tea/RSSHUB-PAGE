<strong>本周四，华擎（ASRock）发布了适用于 AMD 锐龙 7000 系列 AM5 台式处理器的 X670E 主板产品线。</strong>首发阵容中包括了 X670E Taichi（太极）/ Taichi Carrara（白色大理石）、Steel Legend（钢铁传奇）、Pro RS，以及 PG Lighitng 五款型号。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0923/24bed9f75237432.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0923/24bed9f75237432.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：ASRock <a href="https://www.asrock.com/news/index.us.asp?iD=4984" target="_self">官网</a>）</p><p>作为 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 600 系列芯片组中定位最高的型号，X670E 提供了 28 条 PCIe 5.0 通道，能够为主副 PCIe 和 M.2 存储插槽提供“极致”（Extreme）的性能体验。</p><p>而华擎首发的这五款 X670E 主板，都采用了 ATX 的全尺寸板型。首先是旗舰定位的 <a href="https://www.asrock.com/mb/AMD/X670E%20Taichi/index.us.asp" target="_self">X670E Taichi</a> / Taichi Carrara，它们提供了两条 PCIe 5.0 x16 和四个 M.2 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 插槽（带有<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C738%2C751" target="_blank">风扇</a> + 散热片）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0923/267ebeacf182b23.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0923/267ebeacf182b23.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>此外 X670E Taichi 支持 USB4，集成 2.5 GbE 以太网 + Killer Wi-Fi 6E 无线适配器，以及金属加强 DDR5 内存插槽。</p><p>如之前泄露的那样，作为华擎成立 20 周年庆的特殊型号，<a href="https://www.asrock.com/mb/AMD/X670E%20Taichi%20Carrara/index.asp" target="_self">X670E Taichi Carrara</a> 配备了独特的大理石造型散热片。</p><p><a href="https://static.cnbetacdn.com/article/2022/0923/c04ad9bd9cc3539.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0923/c04ad9bd9cc3539.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>其次是 <a href="https://www.asrock.com/mb/AMD/X670E%20Steel%20Legend/index.us.asp" target="_self">X670E Steel Legend</a> 和 <a href="https://www.asrock.com/mb/AMD/X670E%20Pro%20RS/index.us.asp#Specification" target="_self">X670 Pro RS</a>，两款主板都只提供了一条 PCIe 5.0 x16 插槽。</p><p>前者将另一个 PCIe 5.0 x16 换成了 PCIe 3.0 x16 和一个 PCIe 3.0 x1 插槽，后者则是换成了两个 PCIe 4.0 x1 插槽。</p><p>此外两款主板都配备了金属加固 DDR5 内存插槽、四个 M.2 SSD 插槽，以及 Dragon 2.5 GbE 以太网 + AMD RZ616 Wi-Fi 6E 无线适配器。</p><p><a href="https://static.cnbetacdn.com/article/2022/0923/6b4368c3778faba.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0923/6b4368c3778faba.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>最后是相对实惠的 X670E <a href="https://www.asrock.com/mb/AMD/X670E%20Pro%20RS/index.us.asp#Specification" target="_self">PG Lightning</a> 主板，它没有那么豪华的散热装甲，但换来了丰富的 PCIe 扩展选项。</p><p>除了保留的一个 PCIe 5.0 x16 插槽，它还有两条 PCIe 4.0 x16 和一条 PCIe 4.0 x1 插槽，辅以四个 M.2 SSD 和 Dragon 2.5 GbE 以太网。</p><p>遗憾的是，华擎尚未披露上述五款主板的确切定价和上市日期。但从 X670E 芯片组的定位来看，它们肯定都不会太便宜。</p>
