<strong>在本月补丁星期二活动日中，微软为 Windows 11 发布了 KB5014697 累积更新</strong>。微软在官方发布说明中解释道：“此更新包含对内部操作系统功能的各种安全改进。此版本没有记录其他问题”。本次累积更新在非安全改进方面，包含在 5 月 24 日发布的可选更新 KB5014019 中的内容。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0615/1ead6069730fcb0.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0615/6b6b4820131e44a.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">其中最重要的一项是 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Spotlight 在桌面上的首次亮相，它允许用户（以前只在锁定屏幕上提供）自动更改桌面壁纸。对于那些不熟悉"Spotlight"的人来说，这是一个由Bing网站驱动的功能，可以自动改变Windows开机登录界面和锁屏的背景。自Windows 8发布以来，Spotlight就一直存在，它用微软提供的高清图片取代了静态壁纸。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2022/0523/c2e14e3ef1ed0d6.jpg" target="_blank" data-index="1" data-lightbox-gallery="cbc-gallery"><img src="https://static.cnbetacdn.com/thumb/article/2022/0523/c2e14e3ef1ed0d6.jpg" title alt="Windows-11-22H2-Spotlight-feature.jpg" data-original="https://static.cnbetacdn.com/thumb/article/2022/0523/c2e14e3ef1ed0d6.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">正如你在上面的截图中看到的，你可以通过访问设置>个性化>背景>个性化背景来打开该功能。在下一个屏幕上，你需要选择"Windows Spotlight"，微软将用来自Bing的壁纸自动刷新桌面背景。这项功能的工作方式与目前在锁屏上的工作方式相同。用户可以通过点击一个新的桌面图标了解更多关于新桌面背景的信息，该图标会引导用户进入必应网页，其中有关于每张背景图片的更多背景信息。</p><p style="text-align: left;">微软表示，此更新仅带有一个已知问题，但已经有一种解决方法。它涉及 .NET Framework 3.5 应用程序，Microsoft 之前在 Windows 运行状况仪表板页面上记录了该故障。</p><p style="text-align: left;">微软解释道：“安装此更新后，某些 .NET Framework 3.5 应用程序可能会出现问题或无法打开。受影响的应用程序正在使用 .NET Framework 3.5 中的某些可选组件，例如 Windows Communication Foundation (WCF) 和 Windows Workflow (WWF) 组件”。</p>
