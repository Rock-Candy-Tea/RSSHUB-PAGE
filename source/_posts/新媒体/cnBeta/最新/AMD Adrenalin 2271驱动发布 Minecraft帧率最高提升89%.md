AMD 今天推出了 <a href="https://www.amd.com/zh-hans/support/kb/release-notes/rn-rad-win-22-7-1" target="_blank">Adrenalin Edition 22.7</a>.1，这是一个比较重要的驱动更新。该更新为即将到来的 Windows 11 22H2 功能更新提供了支持，并扩展了多项现有功能，对 OpenGL 进行了优化等等。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0727/b4c50e4f354d4e3.webp" alt="S1.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">首先，我们从本次驱动更新中最重要的改进说起吧。AMD 在上月发布的预发布版本中开始测试，现在向所有用户开放了对 OpenGL 的改进。该公司目前仅仅展示了 Minecraft 一款 OpenGL 游戏的性能对比结果，但在社区里有很多用户分享了其他游戏的改进测试结果。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0727/26fa814eea0fabd.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">AMD 表示，在高端 RX 6950XT 显卡上 Minecraft 的性能可以提升最高 89%，在入门显卡 RX 6400 上性能最多可以达到 75%。这两项测试结果都是在“Fabulous”预设模式下在 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_110018%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">4K</a> 分辨率下运行得出。</p><p style="text-align: left;">本次驱动更新还添加了对 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 version 22H2 功能更新的支持，确保用户在 Radeon 显卡上可以稳定升级到新系统。该驱动还添加了对包括 Microsoft Shader Model 6.7 的 Microsoft Agility SDK Release 1.602 和 1.606 的支持。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0727/828882bae41cdd2.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">与此同时，该公司还引入了 AMD Noise Suppression 声音输入过滤技术，作为 NVIDIA 的 RTX Voice 竞争者出现。AMD 公司解释道：“AMD Noise Suppression 使用实时深度学习算法来降低你周围环境噪音，在重要会议、沉浸在竞技游戏中都能提供更清晰的通话质量”。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0727/6502f2ff814e470.webp" alt="ezgif.com-gif-maker.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">其他方面的改进在采用混合显卡的 AMD 笔记本以及使用无边界模式的游戏上，为《艾尔登法环》（Elden Ring）、《生化危机：村庄》（Resident Evil Village）、《无畏契约》（Valorant）在内的多款游戏延伸对 Radeon Super Resolution 的支持。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0727/de189dbe40834ad.webp" referrerpolicy="no-referrer"></p>
