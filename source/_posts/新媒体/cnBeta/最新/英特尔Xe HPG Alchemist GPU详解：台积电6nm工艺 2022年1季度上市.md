<strong>在 2021 年的架构日活动期间，英特尔披露了有关其即将推出的 Xe HPG 显卡的诸多新细节，包括一份完整的路线图。</strong>首先，代号为“Alchemist”的英特尔 DG2 独显将于 2022 年 1 季度上市。紧随其后的是 Xe2、Xe3 和 Xe NG，分别被称作 Battlemage、Celestial 和 Druid 。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0820/f2508ab9eb0867d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/f2508ab9eb0867d.jpg" alt="0-1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/intel-xe-hpg-alchemist-gpu-detailed-built-on-tsmc-6nm-and-landing-in-q1-2022/" target="_self">WCCFTech</a>）</p><p>英特尔表示：作为本次架构变革的一部分，其重新定义了“计算构建块”，并将之作为 Xe 架构的基础。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/6b0f90a0dd00f71.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/6b0f90a0dd00f71.jpg" alt="0-2.jpg" referrerpolicy="no-referrer"></a></p><p>该公司还借此机会更新了一些命名，比如大家不再听到所谓的“执行单元”（EU）。理由是随着数字变得越来越大，代际之间的推理对比都会变得相当麻烦。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/93b44625bfac59d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/93b44625bfac59d.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>Xe GPU 核心中包括了高效的算数、缓存、负载存储逻辑等单元，其中算术单元整合了用于传统浮点、整数向量运算，以及用于加速、卷积和矩阵运算的引擎，此类引擎在 AI 工作负载中较为常见。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/4c191060dce416e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/4c191060dce416e.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>参照新的标准，一个基础的 Xe Alchemist GPU 核心单元，将拥有 16 组“向量引擎”（256 bit）和 16 组“矩阵引擎”（1024 bit）。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/b56dcf6bea3593d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/b56dcf6bea3593d.jpg" alt="6.jpg" referrerpolicy="no-referrer"></a></p><p>每个 Xe Core 都有自己的专用采样器、几何图形、缓存，以及共享的像素后端。4 个 Xe 核心单元（Xe Core）将构成一个渲染切片（Render Slice），且每个渲染切片也都有自己的光线追踪（RT）单元。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/77463de6c0f5406.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/77463de6c0f5406.jpg" alt="7.jpg" referrerpolicy="no-referrer"></a></p><p>初代 Xe HPG GPU 将拥有 8 个切片，每个由 4 个 Xe Core 组成，总计 512（8×4×16）的向量 / 矩阵。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/3bae60a076a64ad.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/3bae60a076a64ad.jpg" alt="8.jpg" referrerpolicy="no-referrer"></a></p><p>对于技术爱好者来说，这点与之前的执行单元（EU）也没有多大差别。假设基础架构是相通的，那它还是 4096 个 ALLU（512*8）。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/d8efd721a259aee.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/d8efd721a259aee.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">台积电业务发展高级副总裁 Kevin Zhang 博士</p><p>值得一提的是，为追求更高的 GPU 性能 / 功耗表现，台积电对英特尔 Alchemist 独显选择了该公司的 N6 制程工艺表示相当欢迎。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/d5c9eeb384fc180.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/d5c9eeb384fc180.jpg" alt="5.jpg" referrerpolicy="no-referrer"></a></p><p>借助台积电 N6 工艺，Alchemist GPU 将可在性能、密度和能效之间实现最佳的平衡，是现代 GPU 代工的理想选择。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/337074fd127f7a3.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/337074fd127f7a3.jpg" alt="9.jpg" referrerpolicy="no-referrer"></a></p><p>如果一切顺利，采用台积电 N6 工艺制造的英特尔 Xe HPG 产品线，将具有能耗和晶体管密度方面的显著优势，从而在正式上市时取得不错的销量。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/f6565b28de42393.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/f6565b28de42393.jpg" alt="10.jpg" referrerpolicy="no-referrer"></a></p><p>据悉，英特尔 Xe HPG 架构将能够实现 1.5 倍于 Xe LP 的时钟速率，且每瓦性能也是后者的 1.5 倍。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/75ed84ccaf4fb11.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/75ed84ccaf4fb11.jpg" alt="11.jpg" referrerpolicy="no-referrer"></a></p><p>鉴于 Xe LP GPU 已经跑到 1.4 GHz，这意味着我们有望迎来 2.1 GHz 的 Xe HPG GPU 时钟速率。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/855c9a91e85f065.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/855c9a91e85f065.jpg" alt="12.jpg" referrerpolicy="no-referrer"></a></p><p>另一方面，英特尔也将能够在不消耗额外功耗的情况下，从新架构中榨取出更多的性能。或在性能不变的基础上，降低新一代 GPU 的功耗。</p><p><a href="https://static.cnbetacdn.com/article/2021/0820/31322bfaf54526e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/31322bfaf54526e.jpg" alt="13.jpg" referrerpolicy="no-referrer"></a></p>
