<strong>foobar2000是一款Windows
平台下的高级音频播放器，包含了一些播放增益支持、低内存占用等基本特色以及内置支持一些流行的音频格式，支持强大的界面扩展和插件以及出色的音效。</strong>foobar2000的定位是专业数字音频播放工具，是一款出色的播放器。<br>

possibly other keyboard events) to be disregarded if mouse pointer is above specific interactive portion of a window (list controls, buttons in editboxes).</p></li><li><p>Improved handling of files with malformed APE tags – some files were entirely unplayable.</p></li><li><p>More thorough ‘remove tags’ for Vorbis, Opus and Musepack.</p></li><li><p>Fixed text truncation in ReplayGain Scanner preferences page.</p></li><li><p>Made possible to add attached pictures by drag and drop to relevant Properties dialog page.</p></li></ul>
