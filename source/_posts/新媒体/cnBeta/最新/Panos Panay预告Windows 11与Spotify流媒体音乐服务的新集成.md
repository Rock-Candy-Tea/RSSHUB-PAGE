微软首席产品官 Panos Panay，最近一直在积极向外界释放消息。<strong>今日的最新爆料，就是 Windows 11 中的 Spotify 集成，以及其它应用程序的改进。</strong>显然，Windows 开发团队仍在努力为定于今秋晚些时候的正式推送做准备。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0806/9841d429f402e8d.png" alt="1.png" referrerpolicy="no-referrer"></p><p>早些时候，开发团队刚刚向 Dev 和 Beta 通道的测试参与者们推送了 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 Insider Build 22000.120，其中包含了一些 UI 更改、额外的小部件、以及大量的 Bug 修复。</p><p>Panos Panay 在 Twitter 上分享的一段 GIF 动图，简要展示了 Spotify 是如何集成到系统闹钟应用中的，意味着用户或可在预订的时间范围内收听流媒体音乐播放列表。</p><p><img src="https://static.cnbetacdn.com/article/2021/0806/71363fe05068f15.gif" alt="233.gif" referrerpolicy="no-referrer"></p><p>Panpy 没有提供太多其它细节，不过融合了 Focus 辅助和 Spotify 播放列表的整体功能，看起来还是相当简洁。</p><p>比如用户可设置其专注时间段（Focused Sessions）的长度，以及在此期间播放他们想要的流媒体音乐。</p><p><img src="https://static.cnbetacdn.com/article/2021/0806/e3a09bb00959353.png" alt="2.png" referrerpolicy="no-referrer"></p><p>对于大多数用户来说，今日分享的改进还是相当微妙的。此外有眼尖的网友希望，Panos Panay 能够进一步披露与 <a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> 游戏、Cortana 语音助理、Skype 插件、以及 Groove 的首选替代服务等相关的改进。</p>
