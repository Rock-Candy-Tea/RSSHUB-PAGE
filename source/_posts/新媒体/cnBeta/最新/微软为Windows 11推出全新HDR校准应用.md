微软于数日前发布的 Windows 11 2022 Update 中，针对玩家引入了诸多优化和改进，而且微软还推出了一款全新的 <a href="https://devblogs.microsoft.com/directx/the-windows-hdr-calibration-app-is-here/" target="_blank">Windows HDR Calibration</a> 校准应用程序。需要注意的是，这与 Windows 11 2022 Update 无关，Windows 11 21H2 版本也可以获得该应用。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0923/499145a07ee60c8.webp" alt="lsnzvoo6.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">该应用于今年 1 月首次发布，为 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 系统引入 <a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> 的 HDR Game Calibration 功能。这款Windows 应用程序提供三种主要的测试模式，包括用于确定最暗可见细节、最亮可见细节和显示器最大亮度的模式。这些是基于 HDR 游戏兴趣小组 (HGIG) 的建议。此外，微软还添加了另一个功能，可以完全控制色彩饱和度。</p><p style="text-align: left;">微软表示，校准显示器将确保您获得 HDR 内容的最佳观看体验，并有助于避免剪辑效果。启动 Windows HDR Calibration 应用程序有两种方法：一种是在此处从 Microsoft Store 下载该应用程序，另一种是从设置 > 系统 > 显示 > 使用 HDR > HDR 显示校准启动它。但是，后者仅受 Windows Insiders 支持。</p><p style="text-align: left;">如果您遇到 HDR 校准问题，请确保您拥有 HDR 显示器并满足此处提到的最低要求。同样重要的是要知道该应用程序无法在 Windows 10 上运行，并且并非所有 GPU 都支持颜色饱和度调整。您还可以选择通过应用程序主菜单右上角的专用反馈按钮直接向 Microsoft 提交反馈。</p>
