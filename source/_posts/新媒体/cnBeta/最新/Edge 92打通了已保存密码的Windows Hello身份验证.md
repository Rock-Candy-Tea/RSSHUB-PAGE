Windows Latest 报道称：<strong>微软正在为 Chromium Edge 浏览器开发两项新功能，分别是 Automatic HTTPS 和“已保存密码”的 Windows Hello 身份验证。</strong>当前，用户会在地址栏输入网址时被自动重定向至不安全的 HTTP 版本。但在启用了 Automatic HTTPS 之后，Edge 将优先尝试连接到可用的 HTTPS 安全连接。<br>
<p><a href="https://static.cnbetacdn.com/article/2021/0505/295ec2c66cadd9b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0505/295ec2c66cadd9b.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>有关 Automatic HTTPS 的细节，我们已在昨日转载 Techdows 的报道中有所提及。</p><p>有需要的朋友，可在“设置 -> 隐私、搜索和服务”项下选择自动切换（建议使用该默认项）、或始终强制使用（或更频繁地遇到错误）。</p><p>不过更让我们感兴趣的，还是 Microsoft Edge 中的 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Hello 身份验证功能集成。</p><p><a href="https://static.cnbetacdn.com/article/2021/0505/9b22f5d221d6ee0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0505/9b22f5d221d6ee0.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>对于经常登录 Twitter 或 Reddit 等网站的用户来说，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>自带的密码管理器能够为其网上冲浪生活带来一定的便利。</p><p>与 Google Chrome 或 Mozilla Firefox 一样，Microsoft Edge 也可以访问并自动填充与微软账户同步的密码、或者通过浏览器的设置页面来管理和编辑已保存的密码。</p><p>不过为了进一步提升用户的安全和隐私性，微软最近开始在 Edge Dev / Canary 尝鲜通道测试对 Windows Hello 身份验证功能的集成支持。</p><p><img src="https://static.cnbetacdn.com/article/2021/0505/05d3132b1463bbf.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>感兴趣的朋友，可移步至 Edge Dev 或 Edge Canary，转至“设置 -> 个人资料 -> 密码”，然后启用“身份验证”授权（Required Authentication）。</p><p>之后当你需要尝试通过已保存的密码来登陆时，系统就会自动调用 Windows Hello 身份验证功能。</p><p>由屏幕截图可知，目前可在三种模式下进行选择，分别是（1）始终使用、（2）每 1 分钟、以及（3）每次线程会话。</p><p>更重要的是，Windows Hello 并不总是需要用到特殊的硬件。在红外相机之外，你可可以通过 PIN 码、指纹、或虹膜扫描等方法来验证 Edge 的密码管理器。</p><p>最后，在下一个重大更新中，Microsoft Edge 还计划引入对 <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a> 查看器、Windows 搜索等功能的集成支持。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1123393.htm" target="_blank">Edge 92迎来Automatic HTTPS：可选默认启用更安全的网络连接</a></p></div>
