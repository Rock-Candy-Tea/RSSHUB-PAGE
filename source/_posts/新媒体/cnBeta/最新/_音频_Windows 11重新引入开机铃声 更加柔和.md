在今天偷跑的 Windows 11 Build 21996 镜像中，在对任务栏、开始菜单等进行重大 UI 改进之外，微软还重新引入了开机铃声。自 Windows 8 系统开始微软取消了开机铃声，而在 Windows 10 中也没有提供。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0616/6122c0c86b42618.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0616/6122c0c86b42618.jpg" alt="sj2ih70q.jpg" referrerpolicy="no-referrer"></a></p><p>前 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 用户体验团队的项目管理总监 Jensen Harris 解释说，放弃 Windows 的开机铃声是避免在使用笔记本的时候，吵醒他的孩子。他表示：</p><blockquote><p>我的 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">MacBook</a> 有过这样一次经历。当时已经是深夜，我刚刚成为一名父亲，轮到我半夜照顾孩子了。我很想启动我的笔记本电脑浏览网页，但我不记得音量是否被调低了。我意识到，我有50％的机会按下电源按钮，听到响亮的启动铃声，唤醒我熟睡的妻子和昏昏沉沉的婴儿，而且我没有办法控制它。</p><p>我选择了关闭我的笔记本电脑，而不是冒险。在那一刻，我突然意识到，对于一个在白天或晚上的任何时候都会在一个人最私密的空间使用的设备来说，让设备播放启动声音是一个非常糟糕的功能。</p></blockquote><p>不过现在，在 Windows 11 系统中开机铃声再次回归，可以听到铃声更加柔和。</p><p><iframe src="//player.bilibili.com/player.html?aid=931178441&bvid=BV1jM4y1u7Km&cid=354874997&page=1" scrolling="no" border="0" framespacing="0" allowfullscreen="true" width="750" height="480" frameborder="no"> </iframe></p>
