随着 Build 25182 预览编译版本的发布，Windows 11 上的微软商店终于迎来了一项改进 —— <strong>玩家们将能够直接通过 Microsoft Store 下载游戏， 而无需麻烦地跳转至 Xbox 应用程序。</strong>此外 Dev 通道的 Insider 测试者们，还会在 22207.1401.X 版本的 Microsoft Store 搜索结果中看到屏幕截图。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0818/2ea020b6ec68026.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0818/2ea020b6ec68026.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> <a href="https://blogs.windows.com/windows-insider/2022/08/17/announcing-windows-11-insider-preview-build-25182/" target="_self">Blogs</a>）</p><p>微软在 22207.1401.X 版本的 Microsoft Store 更新日志中写道：</p><blockquote><p>● 搜索结果中可直接查看第一张屏幕截图，以帮助改善浏览体验。</p><p>● 可直接通过 Microsoft Store 应用安装游戏，以减少所需点击的次数。</p></blockquote><p>此外几天前，开发者还透露他们修复了一个可能导致 Microsoft Store 无法显示超过 2000 条游戏或应用程序评论的 bug 。</p><p>更早的时候，Microsoft Store 还改进了价格标签的样式、以及特定游戏版本的可发现性。</p>
