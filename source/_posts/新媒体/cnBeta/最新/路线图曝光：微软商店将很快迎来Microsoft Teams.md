对于那些想要在 Windows 操作系统上快速投入工作的人们来说，微软商店（Microsoft Store）确实是个相对轻松的应用程序获取渠道。<strong>尴尬的是，尽管 Microsoft Teams 已被许多组织机构所采用，这家雷德蒙德科技巨头却让它长期在商店里缺席。</strong>好消息是，微软早前更新了 Microsoft 365 路线图，其中有一个新条目暗示 Microsoft Store 即将迎来 Teams 。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0426/bde2a1aacbe857b.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>截至目前，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>尚未在官方商店里提供 Teams 这款工作通讯类应用程序，意味着广大用户必须通过 Web 渠道下载，或者作为 <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a> 365 的一部分来安装。</p><p>当然，在某个时间段里，该公司确实有为 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 10“S Mode”用户提供过商店下载链接，毕竟该版本的操作系统限制了第三方应用获取渠道。</p><p><a href="https://static.cnbetacdn.com/article/2022/0426/9ed8015d3c2fc0d.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0426/9ed8015d3c2fc0d.png" alt="roadmap.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">截图（来自：Microsoft 365 <a href="https://www.microsoft.com/en-in/microsoft-365/roadmap?filters=&searchterms=94577" target="_self">网站</a>）</p><p>路线图表明，面向 Windows 10 / 11 平台的 Microsoft Teams，将于下月某个时候在 Microsoft Store 上架。</p><p>不同的是，Windows 10 版本适用于个人、工作、学校账户，而 Windows 11 版本则仅限于工作和学校账户。</p><p>尽管 Windows 11 操作系统已经内置了某个版本的 Teams，但它主要用来通过个人账户和亲友聊天、而不是作为 Slack 的替代品。</p><p><a href="https://static.cnbetacdn.com/article/2022/0426/e9d231020a0100a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0426/e9d231020a0100a.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>事实上，早在去年推出重新设计的 Microsoft Store 时，微软其实已经承诺过为商店带来 Teams 应用程序。</p><p>与此同时，该公司也宣布用户将可通过亚马逊应用商店，来为 Windows 11 平台获取 Android Apps（不少人反而更青睐移动版本）。</p>
