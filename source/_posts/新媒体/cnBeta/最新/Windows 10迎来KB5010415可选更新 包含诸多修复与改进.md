<strong>作为一个 Windows 10 的可选更新，KB5010415 正在通过 Windows Update 进行推送。</strong>此外作为一个可在生产设备上部署的更新，微软也有提供 KB5010415 的离线安装包。有需要的用户，可手动下载并安装。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0216/cbf1f3b044250a9.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://www.windowslatest.com/2022/02/16/windows-10-kb5010415-update-brings-several-improvements/" target="_self">Windows Latest</a>）</p><p>与 Dev 或 Beta 等通道的技术预览版本相比，KB5010415 可选更新相对更加安全。</p><p>但除非确实遇到了必须通过打补丁来修复的特定系统问题，我们并不建议 Windows 10 用户盲目选用。</p><p><strong>由官方日志可知：</strong></p><blockquote><p>KB5010415 还包含了后续累积更新中的修复程序，比如定于 2022 年 3 / 4 月的“补丁星期二”那天推送的改进。</p><p>KB5010415 主要带来了两项新功能，分别针对企业或高级用户。</p><p>首先是 Microsoft Edge 浏览器的 IE 兼容模式 / 共享 cookie 支持，其次是面向非易失性存储的热添加 / 移除支持。</p></blockquote><p><strong>如果现在就检索 Windows Update，你很可能看到如下提示：</strong></p><blockquote><p>适用于 x64 系统的 Windows 10 版本 21H2 的累积更新预览（2022 年 2 月） (KB5010415)。</p></blockquote><p><strong>下载地址（x64 / x86）：</strong></p><blockquote><a href="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5010415" _src="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5010415" target="_blank">https://www.catalog.update.microsoft.com/Search.aspx?q=KB5010415</a></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0216/8e73735ee1e1a05.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>Windows 10 KB5010415（内部版本 19044.1566）更新日志：</p><blockquote><p>● 修复 Windows Server 2016 在使用特定 VDI 运行终端服务器时，可能停止响应的问题。此错误导致回归，Windows 将主动检查 rpcss.exe 中 CSharedLock 的状态。</p><p>● 修复一个影响 Windows Search 的质量问题，它会在用户尝试邻近运算符功能时被触发。</p><p>● 修复系统中心操作管理器（SCOM）的误报问题，以及 wmipicmp.dll 模块导致的内存泄露 bug 。</p><p>● 修复用户在浏览域时报错，以及任务管理器无法正确显示‘开始’优先级（比如 低 / 中 / 高）的一个 Bug 。</p><p>● 修复 OpenGL 和 GPU 在某些显示场景中遇到问题，并阻止 ShellWindows() 返回对象的 bug 。</p></blockquote><p>对于想要升级至 Windows 11 的用户，KB5010415 还修复了在使用 Get-TPM PowerShell 检查 TPM 状态时可能遇到的问题。</p><p>由于无法正确报告可信平台模块（TPM）信息，该工具知会返回“0x80090011”的错误代码。</p><p>此外 KB5010415 修复了某些驱动程序问题，当使用 HVCI 保护时，有望避免遇到掉驱动并重新加载的 Bug 。</p><p><strong>其它错误修复包括：</strong></p><blockquote><p>● 修复无法使用远程桌面的问题；</p><p>● 修复内置屏幕阅读器的问题；</p><p>● 修复 VM 实时迁移失败的问题；</p><p>● 修复导致 WebDav 重定向器死锁的问题。</p></blockquote><p>最后，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还发布了面向 Windows 11 操作系统问题的 <a href="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5010414" target="_self">KB501414</a> 更新，其中同样包含了诸多新功能。</p>
