一位名叫 builtbybel 的开发者，刚刚在 GitHub 上线了一个名叫“ThisIsWin11”的新项目。<strong>据悉，TIW11 提供了一站式的 Windows 11 个性化配置体验，有点类似于支持用户自定义的商店。</strong>如果你点开软件的“关于”部分，还会发现开发者将 TIW11 吹嘘为 Windows 11 平台上“真正的 PowerToys”。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0813/e9f534a5b89ae1f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0813/e9f534a5b89ae1f.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>首先，通过 ThisIsWin11，用户可轻松卸载 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 附带的一些预装应用程序。</p><blockquote><p>尽管<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>希望自家操作系统能够‘面面俱到’，但很大一部分用户只觉得它们是‘膨胀软件’。</p></blockquote><p>其次，该工具还可调整 Windows 11 操作系统的其它各种设置。</p><blockquote><p>有需要的朋友，建议在实施更改前备份并删除现有的‘Packages’文件夹，以避免出现重复文件等形式的冲突。</p><p>该文件夹包含了已安装 Windows 应用的用户配置文件，你可以在 C:/Users/%username%/AppData/Local/Packages 路径中找到它们。</p></blockquote><p>不过这些设置通常是隐藏的，所以你可能需要在操作前取消其隐藏属性。</p><p><a href="https://static.cnbetacdn.com/article/2021/0813/a64990c1f40b837.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0813/a64990c1f40b837.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>为实现上述功能，开发者也为 ThisIsWin11 工具准备了五大模块。</p><blockquote><p>（1）Presenter：引导用户遍历新系统的设置项，并允许快速完成配置。</p><p>（2）PumpedApp：允许检查和优化系统配置，且支持撤销所有更改。</p><p>（3）Kickassbloat：帮助清理预装的 Windows 11 应用程序。</p><p>（4）Packages：支持快速安装一个或其它应用程序。</p><p>（5）PowerClicks：按需调用 PowerShell 和社区脚本，以自动执行多个任务。</p></blockquote><p>感兴趣的朋友，可移步至 GitHub 官网下载体验 ThisIsWin11 工具，当前最新版本为 0.70.0 。</p><blockquote><p><a href="https://github.com/builtbybel/ThisIsWin11/releases/tag/0.70.0" _src="https://github.com/builtbybel/ThisIsWin11/releases/tag/0.70.0" target="_blank">https://github.com/builtbybel/ThisIsWin11/releases/tag/0.70.0</a></p></blockquote><p>不过作为一款第三方工具，还请明确它可能带来的一些风险，所以最好在应用操作前做好相应的资料备份。</p>
