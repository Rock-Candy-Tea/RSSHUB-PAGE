<strong>近日曝光的搭载 Windows 11 操作系统的下一代微软 Surface Pro X 机型，揭示了该公司与高通之间开展的设计合作的新进展。</strong>据悉，为带来与 Apple M1 相媲美的高效能体验，微软为 Windows 11 开发了高度集成的 x86-64 仿真解决方案，以便用户能够无缝原生运行 x86_64 应用程序。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0713/acf2120fa3513fd.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p><a href="https://pc.watch.impress.co.jp/docs/column/ubiq/1337627.html" target="_self">PC Watch</a> 指出，这台运行 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 操作系统的下一代 <a data-link="1" href="https://microsoft.pvxt.net/9W473" target="_blank">Surface</a> Pro X 演示机，采用了一款<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>自家名号的处理器。</p><p><img src="https://static.cnbetacdn.com/article/2021/0713/305ad6e07edf0c6.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>得益于与高通展开的深度设计合作，微软可像 WOW64 一样，在操作系统层面上处理硬件抽象。</p><p><img src="https://static.cnbetacdn.com/article/2021/0713/15740339d574639.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>至于这枚“Microsoft SQ2”芯片，其采用了 8 核心 / 8 线程的 CPU 设计，并且集成了满足 Windows 11 操作系统最低 UI 要求的核显（iGPU），性能刚好足够浏览高分辨率的视频网页。</p><p><img src="https://static.cnbetacdn.com/article/2021/0713/f75c6ebb35816e9.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>此外“Microsoft SQ2”的运行频率高达 3.15GHz，辅以相当先进的 L3 缓存和 LPDDR5 内存支持。</p>
