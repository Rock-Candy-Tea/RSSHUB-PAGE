微软承认近期在 Windows 11 稳定版上出现了一个奇怪的问题：部分用户在该系统中添加新的微软账户之后无法登录。该公司已在 Windows Health Dashboard 官方网站上分享了有关该错误的详细信息，并解释了如何修复它。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0908/dd7d2ff3e0ae51f.webp" alt="QQ截图20220908162925.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">根据<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>的说法，用户在第一次重启或注销后会被短暂锁定。该问题仅在使用 Microsoft 帐户的设备上出现一次，并且影响时间比较短。Active Directory 域用户和 Azure Active Directory 帐户不受这个问题影响。此外，它仅发生在运行 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 21H2 客户端版本的系统上。</p><p style="text-align: left;">如果你在使用 Windows 11 21H2 过程中出现了登录账号问题，那么不需要任何操作就可以解决这个问题。操作系统将让用户在几分钟后登录，而且微软使用已知问题回滚系统解决了这个问题。后者自动撤消有问题的更改，无需终端用户采取任何行动。</p>
