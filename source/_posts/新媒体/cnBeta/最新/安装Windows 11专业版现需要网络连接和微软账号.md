<strong>理论上，Windows 系统可以同时支持本地账户和微软账号两种方式登录，不过微软更推荐后者。</strong>不过自 Windows 10 系统开始，如果没有微软账户和互联网连接登录，就不能再安装家庭版操作系统。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0218/3a8ce8913b6a446.webp" alt="rwjizcxx.webp" referrerpolicy="no-referrer"></p><p>在首次设置 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 时，根本没有本地账户选项，虽然确实存在一些变通办法--比如在设置过程中断开设备与互联网的连接，但对于那些只想顺利使用的用户来说，整个事情显然越来越困难。</p><p>最新的 Windows 11 预览版构建现在也为 Windows 11 专业版引入了同样的变化。换句话说，如果你想安装 Windows 11 专业版，你需要一个微软账户和互联网连接，尽管很可能，与家庭版相同的变通方法将继续完成工作。</p><p>微软还没有提供一个 ETA，即这一变化预计何时在所有生产设备上生效，但现在，它被引入到作为 Windows Insider 计划一部分的版本中，因此被运送给用户进行测试。</p><p>微软解释说：“与 Windows 11 家庭版类似，Windows 11 专业版现在仅在初始设备设置（OOBE）期间需要互联网连接。如果你选择设置设备供个人使用，设置时也将需要 MSA。你可以预期在随后的 WIP 版本中需要微软账户”。</p>
