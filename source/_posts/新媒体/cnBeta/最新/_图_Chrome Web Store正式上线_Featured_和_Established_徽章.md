在本月初通知开发者启用全新徽章之后，Google 今天宣布 Chrome Web Store 上的两个全新徽章正式向普通用户开放。<strong>浏览扩展商店的时候用户可以看到“Featured”（精选）和“Established”（成熟）徽章，这将标志着用户可以信任的知名扩展程序。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0421/59c3e8b3098699f.webp" style referrerpolicy="no-referrer"></p><p>“Featured”徽章授予那些在安全、隐私、设计和用户体验方面遵循最佳标准和做法的扩展。Google 表示授予该徽章的扩展都经过了严苛的人工审查。包括评估扩展的"愉快和直观的体验"，确保开发者使用最新的API，并密切关注每个扩展的隐私做法和要求的权限。</p><p><img src="https://static.cnbetacdn.com/article/2022/0421/c02cf12b0a93640.webp" style referrerpolicy="no-referrer"></p><p>要获得特色标签，扩展程序还必须有一个详细的商店页面，明确概述其目的。最后，一个扩展的核心功能"必须无需额外的证书或付款即可使用"。</p><p>“Established”徽章将自动标记拥有违反开发者计划政策记录的开发者。经过验证的开发者可以通过在其 Chrome Web Store 列表的“Offered by”字段中加已验证账户且没有入一个官方链接来显示其状态。</p><p><img src="https://static.cnbetacdn.com/article/2022/0421/bba40b47301ab7d.webp" style referrerpolicy="no-referrer"></p><p>这个徽章有一个复选标记的设计，是给"已经验证了他们的身份并证明符合开发者计划政策"的出版商的。但是，仅有身份验证还不足以获得该徽章。Google还表示，扩展创建者必须在其服务中表现出"一贯的积极记录"，并且没有任何突出的违反开发者政策的行为。这意味着，全新的扩展发布者将不得不首先证明自己。</p><p>在介绍这两个徽章的博文中，Google 的 Debbie Kim 解释道：“我们的使命一直是让用户轻松找到优秀的扩展，同时认可创造这些扩展的发布者。这些标签应该有助于人们在点击安装按钮时感觉更好，而不必担心恶意的扩展程序会进入 Chrome Web Store”。</p>
