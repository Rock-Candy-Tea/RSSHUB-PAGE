<strong>Google 刚刚宣布了 Chrome OS Flex 的早期预览，可知该项目旨在将 Chrome OS 带来几乎任何 PC、以及采用 Intel 处理器的旧款 Mac 计算机上。</strong>据悉，Chrome OS Flex 建立在 2020 年收购的 CloudReady 之上，致力于满足希望延长现有设备使用寿命的企业 / 教育客户的特定需求。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0216/336c41c5cd0aad9.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0216/336c41c5cd0aad9.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">Chrome OS Flex 致力于更可持续的设备使用（来自：<a href="https://chromeenterprise.google/os/chromeosflex/" target="_self">Google</a>）</p><p>与此同时，Google 也希望 Chrome OS Flex 能够满足广大旧 PC（甚至低配新机）的使用需求，且只需一个 USB 驱动器就可带动。</p><p>显然，该公司的想法是为几乎任何计算机都带来完整的 Chrome OS 体验。不过在立项初期，该公司仍需努力搞定无数种潜在 PC 配置的适配问题。</p><p>目前 Google 已公布一份经过认证的设备清单，其中主要是笔记本电脑。但若你有一定的 Linux 发行版使用经历，也可尝试在其它机器上安装一下。</p><p><img src="https://static.cnbetacdn.com/article/2022/0216/6f830d959fe7702.gif" alt="1.gif" referrerpolicy="no-referrer"></p><p style="text-align: center;">Chrome OS Flex 致力于随时随地的快速、现代工作体验</p><p>Google Chrome OS 产品管理总监 Thomas Riedl 指出：当前的安全体验仍相对粗糙，毕竟用户需要先知道进入 BIOS 设置，并指定开机时从 USB 存储设备启动。</p><p>不过 Chrome OS 和 Chrome OS Flex 使用相同的代码库、并且会遵循相同的发布节奏，对于普通人来说，两者的外观和感受是几乎没有太大差别的。</p><p>当然，作为一个早期预览，我们也不对其能够完美运行这件事抱有太高的期望。遗憾的是，维护团队正专注于打造尽可能可靠的核心体验，因而暂无引入 Android 应用支持的想法。</p><p><a href="https://static.cnbetacdn.com/article/2022/0216/0bae8ffc73d3f36.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0216/0bae8ffc73d3f36.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">Chrome OS Flex 能够快速部署和轻松管理</p><p>需要注意的是，如果你使用的是基于 Apple Silicon 的新一代 M1 Mac / <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">MacBook</a> 计算机，当前暂无在<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>专有硬件上顺利安装 Linux 发行版的方法。</p><p>好消息是，在大多数 2016 年及更早发布的设备上，Chrome OS Flex 还是有望正常工作的。此外作为一个 CloudReady 项目，该系统也提供了一套企业组件。</p><p>适用于 Chrome OS 的所有常见 IT 管理功能，也都适用于 Chrome OS Flex 。</p>
