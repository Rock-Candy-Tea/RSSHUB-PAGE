上周，多家媒体已经分享过英特尔 ARC Alchemist A350M 移动 GPU 的首发评测。可知作为一款入门级独显产品，它还是给我们留下了一些深刻的印象。不过近日，有人在官方驱动程序套件中发现 —— <strong>如果禁用了在 CPU / GPU 之间自动动态分配功率、以“优化”性能并延长电池续航的动态调优技术（简称 DTT），ARC A350M 还可带来更亮眼的表现。</strong><br>
<p><a href="https://static.cnbetacdn.com/article/2022/0414/117ccc2f60af529.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0414/117ccc2f60af529.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">视频截图（来自：BullsLab Benchmarks / YouTube）</p><p>对于经常在外移动使用的笔记本电脑用户来说，Intel DTT 显然是一项相当实用的功能。但对于想要追求更高图形性能的应用场景，它反而会造成较大的瓶颈。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=340824716&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">intel ARC A350M Real Benchmark Test in 6 Games 1080p（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzQwODI0NzE2LnNodG1s.html" target="_self">via</a>）</p><p>由 BullsLab 近日在其油管频道上分享的新基准测试视频可知，在启用 / 禁用 DTT 的情况下，ARC A350M 的游戏性能会有相当大的差别。</p><p><a href="https://static.cnbetacdn.com/article/2022/0414/939df0bc4b86f55.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0414/939df0bc4b86f55.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>英特尔官方宣称，DTT 具有如下优势：</p><blockquote><p>● 改进 CPU / GPU 性能；</p><p>● 改进 CPU / 独显的功耗分配；</p><p>● 针对混合工作负载的智能功率调节。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0414/44c72ef012c29c8.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0414/44c72ef012c29c8.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>但如预期的那样，在禁用 DTT 的情况下，ARC A350M 移动 GPU 能够以 30W 的满载功耗运行 —— 这也是<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a> Galaxy Book2 Pro 笔记本电脑设置的最大 GPU 功率。</p><p>有趣的是，即使 GPU 仍维持在 2200 MHz（2.2 GHz）的峰值频率，ARC A350M 显卡却可带来将近翻倍的游戏性能体验，同时让 CPU 在 28W 功耗限制下运行、温度也变得更高一些（符合预期）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0414/4db4fe679d83743.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0414/4db4fe679d83743.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>此外在综合基准测试项目中，我们也看到了相同的情况。与 18-20W TDP 的英伟达 MX450 独显相比，30W 的 A350M 移动 GPU 具有 30% 的领先优势。</p><p>甚至得益于解锁了功耗限制，ARC A 系列移动 GPU 最终可释放出优于 50W 的 GTX 1650 的性能（尽管在此前的基准测试中落于下风）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0414/5ecd900a6cc8cd9.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0414/5ecd900a6cc8cd9.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p>不过需要指出的是，ARC Alchemist GPU 的驱动程序，仍有进一步的优化空间。测试人员指出，游戏中偶尔还是会遇到卡顿的情况，但英特尔有望通过后续的驱动更新来解决。</p><p>剩下的问题是，受供应链和驱动软件不够完善的拖累，英特尔是否会进一步推迟 ARC Alchemist GPU 的上市安排？（目前仅在韩国等少数亚太市场推出）</p>
