虽然有不少用户热衷于给浏览器添加“标签页导航”等实用的侧面板 UI 扩展程序，但浏览器制造商却长期忽视了这一点。好消息是，这种情况已经悄然发生了变化。由 9to5Google 分享的一段演示视频可知，<strong>用户将能够在 Chrome 浏览器的左侧边栏获取更多信息，比如阅读列表、以及无需离开当前页面来开展 Google Lens 搜索。</strong><br>
<p><a href="https://static.cnbetacdn.com/article/2021/0928/20681f7b9cfcfdc.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0928/20681f7b9cfcfdc.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">视频截图（来自：9to5Google）</p><p>对于需要边搜索边浏览的用户来说，这种类似于“窗口内分屏”的功能，还是显得相当实用的。</p><p>只需先点击网址栏左侧的彩色“G”图标，然后点击侧边信息栏上的相关内容，窗口右侧就可便捷跳转到指定的网页。</p><p>相比之下，当前的 Chrome 浏览器用户在使用搜索的时候，往往不得不再新建一个标签页（或并排开启另一个 Chrome 进程窗口）。</p><p>感兴趣的朋友，不妨在最新的 Canary 分支中体验下“边栏搜索”（Side Search）功能。</p><p style="text-align: center;"><iframe width="640" height="480" src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=291470258&autoplay=false&disablePlaylist=true" frameborder="0"></iframe></p><p style="text-align: center;">Google Chrome Side Search demo（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMjkxNDcwMjU4LnNodG1s.html" target="_self">via</a>）</p><p>其实早在去年，Microsoft Edge 也已经推出了类似的功能，只不过<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>选择了将侧边栏放置在窗口的右侧。</p><p>另一方面，谷歌此举引发了一些反对的声音。比如这一潜在的危险信号，或让 Google 搜素与 Chrome 浏览器更紧密地结合到一起，从而起到打压其它 Web 浏览器 / 搜索引擎的反竞争效果。</p>
