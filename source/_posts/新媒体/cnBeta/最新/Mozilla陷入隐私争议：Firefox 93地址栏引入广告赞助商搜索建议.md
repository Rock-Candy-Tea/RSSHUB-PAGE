<strong>随着 Firefox 93 的发布，开发商 Mozilla 也因在浏览器中插入广告而引发了较大的争议，即便其声称这么做有助于 Firefox 的后续开发与优化。</strong>具体说来是，Firefox Suggest 已不再满足于通过用户输入时的浏览器历史记录来提供相关建议，而是引入了来自“可信赖合作伙伴”的“相关建议”与“赞助建议”—— 也就是广告！<br>
<p><img src="https://static.cnbetacdn.com/article/2021/1010/606ccc846a25d54.png" referrerpolicy="no-referrer"></p><p>尽管 Mozilla 或许会坚持与符合 Firefox 隐私标准的合作伙伴打交道，但大家还担心新搜索建议带来的隐私顾虑。</p><p>毕竟用户在多功能地址栏中输入的任何内容，都会被发送回 Mozilla 进行处理，以提供预期中的广告点击。</p><p><strong>庆幸的是，用户仍可手动禁用 Firefox Suggest、并隐藏浏览器中的广告：</strong></p><blockquote><p>（1）点击浏览器右上角的汉堡菜单，并选择‘设置’。</p><p>（2）转至‘隐私与安全’部分。</p><p>（3）找到‘地址栏 -> Firefox Suggest’并取消所有框选。</p><p>（4）如需在无隐私与广告顾虑的情况下保留部分地址栏搜索建议，可仅取消‘上下文搜索’及‘偶尔包括来自赞助商的建议’子选项。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/1010/7cca7d675e7153f.png" referrerpolicy="no-referrer"></p><p>在官方<a href="https://blog.mozilla.org/data/2021/09/15/data-and-firefox-suggest/" target="_self">支持文档</a>中，Mozilla 解释了新版 Firefox 会收集哪些用户数据 —— <strong>当用户选择接受上下文搜索建议时，Mozilla 将依赖以下信息来支持 Firefox Suggest：</strong></p><blockquote><p>● 搜索查询和建议印象：Firefox Suggest 会向 Mozilla 发送与搜索关键词有关的互动信息，且其中某些信息或与合作伙伴分享、以改进内容建议。</p><p>● 点击建议：当用户点击时，Mozilla 会收到建议链接被点击的通知。</p><p>● 位置信息：Mozilla 会收集具体到用户所在城市的位置数据信息，以提供更准确的位置查询建议。</p></blockquote><p><strong>此外 Mozilla 进一步解释了数据是如何被处理和分享的：</strong></p><blockquote><p>Mozilla 将非常保守地处理这些数据，一旦不再需要，就会立即从系统中删除。在将数据传递给合作伙伴时，Mozilla 会谨慎地仅向合作伙伴提供与之相关的最少信息。</p><p>以 Firefox 参考的 IP 地址为例，由于其能够被用于相当准确地个人定位，因而 Mozilla 会将之转换为更粗略的位置信息，并从所有数据集和下游报告中剔除 IP 地址。</p><p>对可能包含 IP 地址的短期临时数据集的访问，也将受到高度的限制，并且仅限于少数管理员能够接触到。最后，Mozilla 不会启用或允许包含 IP 地址的数据分析。</p></blockquote>
