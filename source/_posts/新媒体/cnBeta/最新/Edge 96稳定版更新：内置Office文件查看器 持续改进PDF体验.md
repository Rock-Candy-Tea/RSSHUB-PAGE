<strong>随着微软向 Windows、macOS 和 Linux 平台推送 Edge 96 稳定版更新，广大用户也迎来了预期中的内置 Office 文件查看器和 PDF 体验改进。</strong>前者是的普通用户能够在浏览器中轻松打开文档，而无需下载后再通过桌面应用程序来查看。另一方面，Microsoft Edge 也能够更好地作为 Windows 平台上的默认 PDF 文档查看器。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1123/20f4afb3ee44e07.png" alt="0.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://docs.microsoft.com/en-us/deployedge/microsoft-edge-relnote-stable-channel#version-960105429-november-19" target="_self">Release Note</a>）</p><p>在最近几个版本中，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>为 Edge 浏览器用户带来了持续的 PDF 体验改进。在最新的 Edge 96 稳定版更新中，其引入了自由格式的荧光笔，让用户能够更轻松地查看 PDF 文档和做标记。</p><p>其它面向普通用户的更新亮点，包括了沉浸式的阅读器迷你工具栏中的新词典，以及可帮助用户学习基本算术、二次方程、三角函数和微积分的新数学求解器功能。</p><p>如果你选择用 Microsoft Edge 来运行渐进式 Web 应用程序（PWA），Edge 96 现可通过协议链接直接启动 PWA，以进一步消除与原生应用程序的差异。</p><p><a href="https://static.cnbetacdn.com/article/2021/1123/c7f679f23c684ce.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1123/c7f679f23c684ce.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>说行说明中没提到的一项重要功能，就是全新的 Super Duper 安全模式。它能够让用户的浏览体验更加安全，但可能对某些网站的部分页面造成破坏。</p><p>几个月前，Super Duper 安全模式以一项实验性功能的形式推出，主要通过禁用 JavaScript 即时（JIT）编译器，来减少可被黑客攻击的位面（尽管还是要手动在设置菜单中启用该标记 / flag）。</p><p>最后，随着 9 月份 Edge 94 的发布，微软也将浏览器的开发周期换回了四周。与此同时，企业用户迎来了一个为期 8 周的“<a href="https://docs.microsoft.com/en-us/deployedge/microsoft-edge-relnote-stable-channel#version-85056441-august-27" target="_self">稳定扩展</a>”频道选项。</p>
