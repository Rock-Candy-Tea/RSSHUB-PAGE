<strong>微软刚刚宣布了针对 Windows Server 2008 / 2012、以及 SQL Server 2021 的扩展安全更新，如果你已经在 Azure 云端使用，则相关安全更新支持也将是免费的。</strong>按照之前的计划，Window Server 2012 / 2012 R2 的扩展支持服务将于 2023 年 10 月 10 日到期。但 Monijit Bhattacharya 透露，微软将为其延长 3 年的支持期限。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0716/13d73141ec2b3c9.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://cloudblogs.microsoft.com/windowsserver/2021/07/14/plan-your-windows-server-2012-and-2012-r2-end-of-support/" target="_self">Windows Server Blog</a>）</p><p>Azure 产品管理负责人、兼 Windows Server 团队成员的 Monojit Bhattacharya 在官方博客文章中补充道：</p><blockquote><p>原定于 2022 年 7 月 12 日到期的 SQL Server 2012 扩展支持服务，也获得了额外的 3 年安全更新支持。</p></blockquote><p>如果客户已将他们的工作负载转移到 Azure 云端，免费支持将发挥巨大的吸引力。此外通过混合权益（Hybrid Benefit）方案，还可在软件保障政策下获得 On-Prem 许可。</p><p>如此一来，客户能够享受到较其它优惠更进一步的 Azure 报价。若你坚持在本地运行，则<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>将逐渐上调扩展更新产品的价格。</p><blockquote><p>据悉，许可证的首年优惠价为 3 / 4，第二年无优惠，且第三年的扩展安全更新将上涨到 125% 。</p></blockquote><p>与此同时，尽管 Windows Server 2008 和 SQL Server 也得到了一些额外的关照（提供一年以上的更新），但这项福利仅限于 Azure 云端。</p><blockquote><p>按照当前的计划，SQL Server 和 Windows Server 2008 / 2008 R2 的扩展安全更新，将分别于 2022 年 7 月 9 日 / 2023 年 1 月 14 日到期。</p></blockquote><p><a href="https://www.theregister.com/2021/07/15/microsoft_extends_support_for_windows_server_2012_and_2008/" target="_self">The Register</a> 指出，微软是在以合作伙伴为中心的 Inspire 线上会议期间宣布的上述消息。</p><p>期间该公司还介绍了 Windows 365 云桌面，并希望软件即服务（SaaS）能够成为软件制造商的首选。</p><p>值得一提的是，通过 AppSource 和 Azure Marketplace 市场销售的应用程序，其抽成比例也将从 20% 下调至 3%，以优化和简化企业客户的软件采购流程。</p><p>至于 Azure Stack HCI 产品的调整，则包括启用支持 TPM 2.0 的安全核心服务器、固件防护、以及基于管理程序的代码完整性。</p><p>此外通过事件网格（Event Grid）与 API 管理的集成，还有助于管理新用户或 API 的订阅。</p>
