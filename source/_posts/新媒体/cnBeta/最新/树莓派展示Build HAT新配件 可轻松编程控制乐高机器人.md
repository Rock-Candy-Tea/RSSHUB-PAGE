<strong>树莓派正在推出一款名叫 Build HAT 的新配件，以便用户通过旗下各款低成本的微型计算机，来控制某些乐高机器人的电机和传感器。</strong>通过将该组件接入 GPIO 引脚，它可延伸出四个端口。此外树莓派提供了配套软件，可连接到使用 LPF2 连接器的大多数部件，包括乐高“心灵风暴”机器人发明家套件中的组件。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1019/44091280603bd91.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1019/44091280603bd91.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>配套的 Python 库，基本上融合了一组可用于控制机器人的命令，以便用户编写相关软件来连接乐高的 SPIKE 组件。后者带有一套集线器，支持连接 6 个设备，并且能够存储和运行 Python 程序。</p><p>尽管树莓派 Build HAT 仅提供了四个接口，但用户还可在该平台上获得更大的灵活性，尤其是 SPIKE 必须使用单独的计算机或 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a> 对乐高的集线器进行编程。</p><p>相比之下，Build HAT 能够连接到树莓派计算机，然后外接键鼠显示器等外设，以开展相关工作。</p><p>此外乐高 SPIKE 集线器只有 LPF2 端口，但树莓派提供了可连接其它电子设备和传感器的 GPIO 引脚。</p><p><a href="https://static.cnbetacdn.com/article/2021/1019/9fbe20d4e5bc72d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1019/9fbe20d4e5bc72d.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">Build HAT 较低层级的电机 / 传感器控制</p><p>树莓派官方表示，通过将 Build HAT 连接到桶形 8V 电源，即可为连接的树莓派本体和乐高<a data-link="1" href="https://microsoft.pvxt.net/Xx07X" target="_blank">配件</a>供电。如果手头没有合适的电源适配器，该公司也将提供一款专用的 48W 型号。</p><p>最后，尽管 Build HAT 是一款相当小众的产品，但若你的孩子或学生正在玩乐高的 SPIKE 机器人、同时想要深入编程和电子改装，那树莓派平台显然也是一个不错的选择。</p>
