<strong>微软今天发布了新的 Windows Server 预览版。Windows Server Build 25158 的 ISO 镜像和 VHDX 格式文件已经放出，采用和 Windows 11 Sun Valley 3 (23H2) 相同的开发内核。</strong>新版本依然标记为 Windows Server 2022，并未采用 vNext 新品牌。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0714/27a04f4f2ba4189.webp" alt="w570itl1.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">开放下载的类型<br style="text-align: left;"></p><p style="text-align: left;">● <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Server VNext Preview ISO 镜像支持 18 种语言<br style="text-align: left;"></p><p style="text-align: left;">● VHDX 格式仅支持英语<br style="text-align: left;"></p><p style="text-align: left;">● Windows Server VNext Preview VHDX</p><p style="text-align: left;">● Microsoft Server Languages and Optional Features Preview</p><p style="text-align: left;">● Windows Admin Center 2110.2 Preview</p><p style="text-align: left;">仅适用于该预览版的密钥：<br style="text-align: left;"></p><p style="text-align: left;">● Server Standard: MFY9F-XBN2F-TYFMP-CCV49-RMYVH</p><p style="text-align: left;">● Datacenter: 2KNJJ-33Y9H-2GXGX-KMQWH-G6H67</p><p style="text-align: left;">已经注册的 Windows Insider 成员可以通过<a href="https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewserver" target="_blank">这个页面</a>下载。没有注册的用户也可以访问  Windows Insider for Business 页面的 <a href="https://insider.windows.com/en-us/for-business-getting-started-server/" target="_blank">Getting Started with Server</a>，加入 Windows Insider 项目进行下载。<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>表示该预览版将会在 2022 年 9 月 15 日停止支持。</p>
