适用于 Windows 11 21H2 的可选更新 KB5017383 于今天发布。该可选更新已经通过 Windows Update，但由于可选性质，用户可以根据自己的需求来决定是否安装。不过该可选更新的内容将会整合到下月补丁星期二活动日发布的累积更新中。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0923/cc7abfe273bfabf.webp" alt="5qq46vjd.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">KB5017383 更新在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Update 中显示为“2022-09 Cumulative Update Preview for Windows 11 Version 21H2 for x64-based Systems (KB5017383)”，用户确定下载安装之后，需要重新启动设备以生效。用户也可以通过 Update Catalog 下载离线安装包。</p><p style="text-align: left;">微软在 Windows 11 KB5017383 (Build 22000.1042) 中引入了“dynamic widgets”的功能，该功能此前由 Windows 11 22H2 独占，不过现在微软向后兼容到 21H2。借助动态小部件，微软希望使任务栏上的小部件图标与定期更新更具吸引力。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2022/0616/285184d0fd42bd3.jpg" target="_blank" data-index="1" data-lightbox-gallery="cbc-gallery"><img src="https://static.cnbetacdn.com/thumb/article/2022/0616/285184d0fd42bd3.jpg" title alt="Windows-11-Taskbar-updates.jpg" data-original="https://static.cnbetacdn.com/thumb/article/2022/0616/285184d0fd42bd3.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">集成到任务栏的小部件应该很快就能让用户看到金融市场交易价格和其他更新。而且，它仍然以同样的方式工作：你将直接在任务栏上获得信息，甚至不需要点击图标。正如你在下面的截图中所看到的，任务栏将显示体育和金融小部件的实时更新。</p>
