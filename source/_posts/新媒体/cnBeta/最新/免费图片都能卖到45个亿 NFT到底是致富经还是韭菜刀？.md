就在不久前，世超给大家介绍了被拍卖到 58 万美元的魔性表情包，还有数字版的 NBA 球星集卡游戏 NBA Top Shot。当然啦，也科普了一下它们背后的 NFT 技术，想详细回顾一下的差友们可以点这里。简单地复习一下， NFT 的全称是 non-fungible Token,中文名叫非同质化通证。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0324/708712f3f574e79.png" referrerpolicy="no-referrer"></p><p>举个不恰当的例子，就像给阳澄湖的大闸蟹配打个上了专属码，里面登记了这只螃蟹的出生地和身份信息，每只螃蟹一个专属码，绝对不重样。</p><p>而在 NFT 的世界里，图片、视频、音乐等等数字作品就是那只螃蟹，NFT 能在区块链上打下它们唯一的电子码。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/523f4f4b0db54e5.png" referrerpolicy="no-referrer"></p><p>但就在那篇文章发出去后不久，我就后悔了。</p><p>因为世超本以为 NFT 这玩意儿的热度很快就会过去，然后销声匿迹。</p><p>结果事实却是，你以为它结束了，其实它才刚刚开始。</p><p>NFT 的概念也像是被开过光一样，火得越来越让常人无法理解。</p><p>并且，还刮起了一股万物皆可 NFT 化的风潮。</p><p>所以今天，世超就准备再和大家聊一聊那些或出圈儿、或离谱的 NFT 们。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/0b941e946691354.png" referrerpolicy="no-referrer"></p><p>首先，最引起轰动的 NFT 作品，就要数 Beeple 的《 Everydays -The First 5000 Days 》了。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/a2f180e652428f7.png" referrerpolicy="no-referrer"></p><p>Beeple 本人其实并不是无名之辈。</p><p>身为设计师的他，曾参加过 Justin Bieber、Eminem、deadmau5 等人的演唱会视觉设计，和不少的奢侈品品牌也有过合作。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/8963d0e23dc3793.png" referrerpolicy="no-referrer"></p><p>而从 2007 起，Beeple 每天都会抽时间拿出电子设备，完成一幅数字作品。</p><p>《 Everydays -The First 5000 Days 》 便是他这十几年来，5000 幅作品的集合。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/5265ec7f9e4240b.png" referrerpolicy="no-referrer"></p><p>虽然这些图片或视频在网上都能轻易下载到。</p><p>Beeple 本人也在自己的官网上免费公开了 everyday 作品。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/e8ab748b4e33c19.png" referrerpolicy="no-referrer"></p><p>但最后在佳士得拍卖会上，《 Everydays -The First 5000 Days 》还是拍出了 6934 万美元，约 4.5 亿人民币的天价。</p><p>简单地说，这价和毕加索和梵高的作品差不多一个等级。</p><p>只不过不同的是，那些传统名画的中标者能拿到画的实物，这次近 7 千万美元买下的却只是一堆数字文件罢了。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/7af8491952cfb4a.png" referrerpolicy="no-referrer"></p><p>所以在理论上，Beeple 那 5000 张传遍全网的作品，包括上面的这张，现在也已经属于那个神秘的中标者了。</p><p>即使你上网一搜，一模一样的图片视频一堆，但在 NFT 的认证下，中标者手上的才是真正的正品。</p><p>也就是说，这位老哥有资格向全世界宣布：“ 我承包了那 5000 张画！”当然除此之外，好像并没有其他任何卵用……</p><p>说一句题外话，孙宇晨也参加了这次拍卖，据说出价到了 6000 万美元，在最后一刻被神秘买家打败……</p><p>果然，孙老师是检验一个风口火不火的唯一标准。</p><p>当然最让 NFT 概念出圈儿的，还是要数爱搞事的马斯克。</p><p>是的，在散户大战华尔街、狗狗币等等战场疯狂拱火的马斯克，他又来了。</p><p>在本月的 16 号，马斯克在自己的推特上发了一首歌，并配上了“ 我要把这首关于 NFT 的歌， 弄成 NFT 卖掉”的文案。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/66bf5078de2b505.png" referrerpolicy="no-referrer"></p><p>在 NFT 交易网站“ Valuables ”上，马斯克的这首电子舞曲 NFT，在一天内就被竞拍到了数十万美元……</p><p>同时，马斯克在提交到美国证券交易委员会的相关文件中，也给自己加上了“ 电音之王 ”的头衔……</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/b745a93716f90a6.png" referrerpolicy="no-referrer"></p><p>有趣的是，网友们的实力也一样不容小觑，立马就有人从马斯克的推特里发现了商机。</p><p>就比如说这位名叫 greg 的小老弟，就卖起了马斯克推特的截图。</p><p>他的文案深谙套娃技巧：“ 我要把马斯克那‘ 我要把这首歌关于 NFT 的歌， 当成 NFT 卖掉 ’的截图，当成 NFT 卖掉 ”。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/19076fece0c84d6.png" referrerpolicy="no-referrer"></p><p>很快，又有网友截了一张 greg 小哥截马斯克推特的推特截图，并计划把它当成 NFT 卖掉。</p><p>在一层层的套娃中，推特里充满了快活的气息。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/8de3066449aa5b6.png" referrerpolicy="no-referrer"></p><p>同样是把推特当成 NFT 出售，推特的 CEO 多西卖的东西就更有“ 意义 ”了。</p><p>他卖的是推特史上的第一条推特，发布于 15 年前，内容是“ 刚刚设置好我的推特 ”。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/c916621098fd432.png" referrerpolicy="no-referrer"></p><p>就是这么一条朴实无华，只有 5 个单词，还不带标点符号的短句。</p><p>在今天以 290 多万美元的价格，卖给了数字货币交易公司 Bridge Oracle 的 CEO 埃斯塔维。</p><p>没让人失望的是，在世超查看拍卖记录的时候，依旧发现了辣个男人的身影，我们的孙宇晨老师一路出价到 200 万美元，最后再次成功陪跑。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/9e3ececa9c74f39.png" referrerpolicy="no-referrer"></p><p>如果说上面的这些 NFT 产品，在某种程度上还是有些“ 意义 ”的话。</p><p>那么宝洁公司旗下的卫生纸品牌 Charmin 推出的厕纸主题 NFT ，感觉完全就是在蹭热度了……</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/39b86bb51fe348a.jpg" referrerpolicy="no-referrer"></p><p>Charmin 一共推出了 6 款限量的“ NFT ”厕纸，上面这一副闪瞎人眼的便是# 005 号“ 作品 ”，而它现在的报价是 2200 美元。</p><p>要是花这钱去买真卫生纸，够世超一个人用好几年了吧。</p><p>#001 号作品 ▼</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/ccb9bcafc69ab15.png" referrerpolicy="no-referrer"></p><p>同样，为了作秀和抬高 NFT 的价格，还有人烧毁了原画作，然后卖画的 NFT 数字版……</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/5ebedecd99b8d2c.png" referrerpolicy="no-referrer"></p><p>让人摸不着头脑的是，班克西这幅作品买入的价格为 9.5 万美元，在烧画，NFT 化等等的一系列骚操作之后，最后卖出了 4 倍于原价的高价……</p><p>哦，对了，这幅画的名字就叫《 傻子 》。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/575ead48b7d6d20.png" referrerpolicy="no-referrer"></p><p>看完这些个 NFT 市场的欢乐故事，大伙们肯定以为这玩意要么是为了洗钱，要么就是等待韭菜们入场的新镰刀罢了。</p><p>然而说起来你可能不信，在某些方面， NFT 其实还有是些开创性的，甚至还带着一丝丝的理想主义色彩。</p><p>因为 NFT旨在确定数字作品的所有权，并淘汰那些中心化的版权证明机构，还有中心化的内容分发部门。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/8a395e9f2a42313.png" referrerpolicy="no-referrer"></p><p>像是《 蒙娜丽莎 》这样有实体存在的艺术品，我们能够通过各种方法鉴定真伪、并且拍卖交易。</p><p>但在如今的数字化时代，不少的图像视频作品诞生在网上，也传播在网上。</p><p>很多时候简单的复制黏贴，就可以把别人的作品占为己有，甚至用于牟利。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/7325f52425a334e.jpg" referrerpolicy="no-referrer"></p><p>NFT 则用区块链的方式，把数字文件和他的创作者永久连接了起来。</p><p>到底谁的是正版，打开各自的数字钱包看看，就知道到底属于谁了。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/2b4d67a9eb29941.png" referrerpolicy="no-referrer"></p><p>另外，在 NFT 作品被卖出之后，之后的任何转卖，原作者都能拿到一定比例的报酬，一般默认是售价的 10%，而且不会被分发机构抽成。</p><p>具体抽多少水，原作者在创建 NFT 时是可以自己决定。</p><p>也因为这些优点，NFT 在图片、视频、音乐等等领域，受到了不少创作者们的支持。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/2d94f50da91f6b2.png" referrerpolicy="no-referrer"></p><p>当然啦，要是想真正的杜绝盗版、确定所有权，NFT 还是有很长的路要走的。</p><p>例如盗窃了他人的作品，然后将其 NFT 化以此牟利咋办？抄袭的作品通过所谓的 NFT 化洗白咋办？……</p><p>目前浮躁和初生的 NFT 市场也满是的滥竽充数者，一张图加上NFT，似乎就能卖出高价。</p><p>除去泡沫之外，有意义 NFT 艺术品真的少之又少。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/332a906e2ad6ebf.png" referrerpolicy="no-referrer"></p><p>另外购买了 NFT 的买主，除了一个看不见摸不着的数字证书之外，又得到了什么呢？</p><p>对于 NFT 等等的一麻袋问题，看来一时半会儿是解决不了的。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/b0cfc96370a6e54.png" referrerpolicy="no-referrer"></p><p>但世超知道的是，比起真正地收藏数字艺术品，目前疯狂涌入 NFT 的买家们，多半是对发财更感兴趣的赌徒。</p><p>脸书、推特、优步等等公司的天使投资人维纳查克也毫不留情，直言说目前 99% 的 NFT 都不是很好的投资……</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/1c0e2d291602334.png" referrerpolicy="no-referrer"></p><p>最后让我们回到那一幅被烧掉的画《 傻子 》。</p><p>作品想讽刺的，就是那些资本和舆论的炒作，还有那群在拍卖会上无脑跟着喊价的人。</p><p><img src="https://static.cnbetacdn.com/article/2021/0324/3c7cc7a93914e2a.png" referrerpolicy="no-referrer"></p><p>作品上白纸黑字得着“ 我不信你们这些傻子，会真的买这狗屎东西 ”。</p><p>但在目前看来，还真的有。而且，还很多。</p>
