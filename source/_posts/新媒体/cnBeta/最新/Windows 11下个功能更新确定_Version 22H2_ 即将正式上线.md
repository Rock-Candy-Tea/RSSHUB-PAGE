Windows Update 在上周四发布的更新中，确认了 Windows 11 下个功能更新叫做“Version 22H2”。正如下方截图所示，微软在发布 KB5012427 累积更新中，<strong>标题部分写道：“适用于 Windows 11 Version 22H2 Insider 预览版的累积更新”。</strong><br>
 <p style="text-align: left;">目前，还不清楚 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 Version 22H2 功能更新的公开名称。它可能以月份命名，或者<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>可能简单地使用版本号作为公开名称。在这之前，Windows 11 Version 22H2 也出现在几个泄露的微软文件中。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2022/0307/1247f4d5d9df576.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0307/1247f4d5d9df576.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">当然，我们已经知道22H2版本将更多地关注修复Windows 11的现有界面，如开始菜单、控制中心/通知、本地应用程序如 MS Paint、Windows 设置、任务栏桌面和操作系统的其他区域。这次更新还将带回对任务栏拖放的支持，并在Windows 11中增加一个新的任务管理器。</p><p style="text-align: left;">如果你已经等不及体验新版本，那么可以加入 Windows Insider 项目的 Dev 频道进行体验。但需要注意的是，Dev 发布的预览版并不稳定，虽然可以获得最新的功能，但可能会出现各种问题，不推荐在日常办公中使用。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0307/57daa243afbbf02.jpg" referrerpolicy="no-referrer"></p>
