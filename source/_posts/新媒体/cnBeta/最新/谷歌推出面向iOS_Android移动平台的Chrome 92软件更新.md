本周，谷歌发布了面向 iOS 和 Android 移动平台的新版 Google Chrome 网络浏览器，预计大家很快就会在各大应用商店里看到 Chrome 92 的更新提醒。<strong>对于喜欢体验新功能的朋友，面向 Android 平台的 Chrome Beta 92 版本号也已升至 92.0.4515.105 。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0721/0e01daa3a67795d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0721/0e01daa3a67795d.jpg" referrerpolicy="no-referrer"></a></p><p>与此同时，面向 iOS 设备（<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> / <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a>）的 Google Chrome 浏览器，其版本号也已升至 92.0.4515.90，并且带来了一系列新功能。</p><p>其中包括全页截屏、新建标签页上的新“发现”（Discover）设计，设置、历史记录和书签功能的视觉刷新，以及使用 Touch ID / 面容 / 密码来保护无痕浏览标签页的隐私体验。</p><p>在油管 Google Chrome Developers 官方账号分享的这段视频中，谷歌还介绍了桌面版 Chrome 浏览器中引入的一项新功能（需手动启用 flags 标记）。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=273719505&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">Chrome 92 Whats New in DevTools（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMjczNzE5NTA1LnNodG1s.html" target="_self">via</a>）</p><p>感兴趣的朋友，可在地址栏输入 chrome://flags/#memories，以跳转至“实验性功能”设置页面。</p><p>如需更新，台式机用户可将鼠标移至窗口右上角的三点菜单，并在“帮助 -> 关于 Google Chrome”页面中手动查找更新。</p><p>如果稳定版通道尚未推送，亦可尝试下载并安装 Chrome Beta 版本。在 Chromium 的<a href="https://www.chromium.org/getting-involved/dev-channel" target="_self">项目官网</a>上，有提供面向 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>、Mac、Linux，以及 iOS / Android 平台的各个版本。</p><p><img src="https://static.cnbetacdn.com/article/2021/0721/f7b46577a460a63.jpg" referrerpolicy="no-referrer"></p><p>下载地址：</p><blockquote><p>● Android（<a href="https://play.google.com/store/apps/details?id=com.chrome.beta" target="_self">Google Play</a>）</p><p>● iOS（Apple <a href="https://apps.apple.com/us/app/google-chrome/id535886823" target="_self">App Store</a>）</p></blockquote>
