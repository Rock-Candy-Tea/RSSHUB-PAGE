<div>   
<strong>通过《赛博朋克 2077》和 DLSS 3 的特色演示，英伟达又秀了一把 GeForce RTX 4090 的游戏性能。</strong>比如在 2.85 GHz 的频率下，该显卡仍能够将温度控制在 50℃ 。与此同时，第三代深度学习超级采样技术（DLSS 3）可将 GPU 功率降低多达 25% 。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0923/852bd4d0eac73eb.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0923/852bd4d0eac73eb.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（via <a href="https://wccftech.com/nvidia-geforce-rtx-4090-runs-up-to-2850-mhz-at-stock-50c-temps-in-cyberpunk-2077-dlss-3-cuts-gpu-wattage-by-25-percent/" target="_self">WCCFTech</a>）</p><p>测试平台采用了<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>酷睿 i9-12900K 处理器，并以 Ultra 光追 + Psycho 设置 @ 1440p 分辨率运行了《赛博朋克 2077》。</p><p>可知在未启用 DLSS 和 Reflex 的原始分辨率下，该游戏以 59 FPS 的平均帧率运行（1% 最低 49 FPS），延迟在 72-75 ms 左右。</p><p><img src="https://static.cnbetacdn.com/article/2022/0923/8e9bef7f512d9c2.gif" alt="1.gif" referrerpolicy="no-referrer"></p><p>此时 GPU 频率达到了 2.8+ GHz，平均 2810 ~ 2850 MHz 。在满载情况下，公版 RTX 4090 显卡的运行温度保持在 50~55 ℃ 之间。</p><p>这较默认的 2.52 GHz 加速频率提升了 330 MHz（+13%），当更大的惊喜在于我们可在其它游戏中看到更高的 GPU 频率（超频更可突破 3 GHz）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0923/6c19fa6ce016970.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0923/6c19fa6ce016970.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>至于 DLSS 3 的性能表现，这里暂时用的还是预发布版本（最终版本的性能 / 设置可能有所不同）。</p><p>可知在开启了 DLSS 3 帧生成 / 质量（Quality）预设下，我们再次看到了超过 2.8 GHz 的 GPU 利用率，但是温度较 55℃ 更接近 50℃ 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0923/edd65a13ebca1cd.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0923/edd65a13ebca1cd.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>此外在启用 DLSS 3 后，RTX 4090 的平均帧率大幅提升到了 170 FPS（1% 最低 119 FPS）、平均延迟 53 ms —— 较对照测试的帧率翻倍、同时延迟降低了 30% 。</p><p>功耗方面，通过 PCIe 5.0 16-pin 连接器（12VHPWR）和最新的 PCAT 工具，可知 RTX 4090 在开 / 关 DLSS 的情况下也有很大的变化。</p><p><img src="https://static.cnbetacdn.com/article/2022/0923/45e6a6488ba69b2.gif" alt="4.gif" referrerpolicy="no-referrer"></p><p>在未启用 DLSS 3 的情况下，RTX 4090 显卡的平均功耗为 461W —— 每瓦性能为 0.135 帧 / 焦耳。</p><p>在启用 DLSS 3 后，GPU 功率立即降到了 348W（-25%）—— 每瓦性能也提升到了 0.513 帧 / 焦耳（3.8 倍）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0923/fa5ce4f6ea635a3.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0923/fa5ce4f6ea635a3.png" alt="5.png" referrerpolicy="no-referrer"></a></p><p>【更新】作为参考，这里还放入了开关 DLSS 的 RTX 3090 Ti 显卡作为参考。不过采用非公设计的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://msi-pc.jd.com/" target="_blank">微星</a>超龙（SUPRIM X），其默认出厂频率也更高一些（性能略高于 FE 公版）。</p><p>在相同设置与 1440p 场景下，显卡默认平均帧率 37.2 FPS、功耗 452W、运行温度 75℃ 。开启 DLSS 2“质量”预设后，帧率提升到 61 FPS（+64%）、同时功耗降到了 409W（-10%）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0923/f80a613ca98cc62.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0923/f80a613ca98cc62.png" alt="6.png" referrerpolicy="no-referrer"></a></p><p><strong>《赛博朋克 2077》超高（Ultra Quality）质量 + Psycho RT @ 原生 1440p：</strong></p><blockquote><p>● 微星 RTX 3090 TI SUPRIM X @ 原生 1440P —— 37 FPS / 455W 功耗 / ~75℃</p><p>● RTX 4090 公版 @ 原生 1440p —— 60 FPS / 461W 功耗 / ~55℃</p><p>→ RTX 4090 对比 RTX 3090 Ti —— 领先优势 +62%</p></blockquote><p><strong>DLSS 2 / 3 开启 @ 1440p 对比：</strong></p><blockquote><p>● 微星 RTX 3090 Ti SUPRIM X @ DLSS 2 / 1440P —— 61 FPS / 409W 功耗 / 74℃</p><p>● RTX 4090 公版 @ DLSS 3 / 1440p —— 170 FPS / 348W 功耗 / ~50℃</p><p>→ RTX 4090 对比 RTX 3090 Ti —— 领先优势 +178%</p></blockquote>
