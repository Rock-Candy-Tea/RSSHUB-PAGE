微软最近宣布，其计划在 2022 年秋季更新之外，为 Windows 11 操作系统引入新的功能。<strong>现在，随着 Release Preview 预览和 Beta 测试通道的开启，我们预计本年度的首个大型更新，会在本月晚些时候带来。</strong><br>
<p><img src="https://static.cnbetacdn.com/article/2022/0214/181aa8130a498b3.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://www.windowslatest.com/2022/02/14/microsoft-begins-testing-windows-11s-big-february-2022-update/" target="_self">Windows Latest</a>）</p><p>与 Windows 10 相比，Windows 11 采用了更加模块化的设计，相关功能可通过单独的 Windows / Microsoft 更新方式进行推送。</p><p>这些功能体验可随年度大版本更新一起提供，辅以为操作系统带来的其它微小改进，比如近日向 Release Preview 和 Beta 测试通道推送的 Windows 11 Build 22000.526 编译版本。</p><p>据悉，新预览恢复了对多个显示器上的日期 / 时间显示支持，以便用户自由选择将时钟和日期添加到主显示器之外的屏幕的任务栏上。</p><p><img src="https://static.cnbetacdn.com/article/2022/0214/ac8b4ec2dfef891.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">此外<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>在任务栏左侧添加了对天气内容的支持。</p><p>如上图所示，如果你为任务栏图标选择了居中对齐的方式（亦可选择靠左对齐），天气小部件就会出现在桌面的左侧。</p><p>你还可以将鼠标悬停在天气上、或单击按钮，以打开左侧现有的小部件面板。</p><p>另外用户可直接通过 Windows 11 任务栏，分享已打开的应用程序的窗口中的内容（比如给活动的视频通话静音 / 取消静音）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0214/880a0b272caa3f6.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>感兴趣的朋友，现可通过预览通道获取 Windows 11 的 2022 年 2 月更新，以启用在 Microsoft Teams 通话期间的任务栏内容分享体验。</p><p>如此一来，用户便可消除在已开启的应用程序 / 窗口之间来回切换的需要，让大家能够更加专注于当前的会议（点击应用预览小窗格下方的“分享此窗口”/ 并可随时“停止分享”）。</p><p>最后，本轮更新修复了导致任务栏上出现错误工具提示的问题，以及在重启操作系统并登录后，脱机文件位于网络驱动器上的问题。预计未来几周，微软还会通过“体验包”推送 Android 支持和其它改进。</p>
