尽管驱动优化还有很长一段路要走，<strong>但英特尔图形开发团队和营销部门，还是在持续努力地向大家讲述自家 Arc Graphics 硬件的技术细节和性能数据。</strong>早前，Tom Peterson 和 RyanShrout 已在 IFA 2022 期间接受了 PCGamer 高级编辑 Jacob Ridley 的采访。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0902/c7c21261e2232b5.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p>与此同时，Intel Graphics 官方油管账号也在一段近乎半小时的视频中，详细介绍了 Alchemist 的光追架构与实测表现。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=379602445&autoplay=false&disablePlaylist=true" width="640" height="360" frameborder="0"></iframe></p><p style="text-align: center;">Intel Arc ASeries Graphics Ray Tracing Technology Deep Dive（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzc5NjAyNDQ1LnNodG1s.html" target="_self">via</a>）</p><p>在周四发表的一篇文章中，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>深入探讨了其 Arc Alchemist GPU 的光追架构，并且搬出了 Arc A770 和英伟达 RTX 3060 竞品对比。</p><p><a href="https://static.cnbetacdn.com/article/2022/0902/2910736af200497.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0902/2910736af200497.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>通过与 RTX 3060 相当或更快的光追性能数据，英特尔从侧面揭示了 Arc Alchemist 的这方面表现，或优于配备了初代硬件光追方案的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> RDNA 2 GPU 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0902/2f0d472cfbe6348.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0902/2f0d472cfbe6348.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>据悉，Arc Alchemist GPU 符合 DirectX 12 Ultimate 功能集，且其 RT 引擎支持 DXR 1.0 / 1.1 和 Vulkan RT API 。</p><p>Xe Core 是 Alchemist GPU 上不可分割的子单元，包含了主要的数学运算机制，且每个 Xe Core 都具有一个线程排序单元（TSU）和一个光线追踪单元（RTU）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0902/cbca18a0811efe8.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0902/cbca18a0811efe8.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>其中 TSU 部分负责 Xe Core 和 RTU 之间的调度工作，属于 Intel 图形方案的一项“秘密武器”（实现所谓“异步光线追踪”的关键）。</p><p>而每个 RTU 包含了两个光线遍历管道，作为固定功能硬件，其负责计算 BVH 的光线交叉点。</p><p><a href="https://static.cnbetacdn.com/article/2022/0902/bdfb6831bc2e9bf.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0902/bdfb6831bc2e9bf.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>RTU 每周期可计算 12 个框交叉点、1 个三角形交叉点，并具有用于 BVH 数据的专用缓存。</p><p>而 TSU 负责组织光追指令和数据，以便为具有相似命中着色器的光线，优化分配 Xe Core 的统一着色器资源（旨在实现硬件资源的最佳分配）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0902/b959f313326d7fa.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0902/b959f313326d7fa.jpg" alt="5.jpg" referrerpolicy="no-referrer"></a></p><p>具体数据方面，英特尔分享了 1080p 下，Arc A770 与同级的 RTX 3060 竞品在 17 款游戏中的性能表现，比如在最新 beta 驱动程序中迎来了持续优化的《幽灵线：东京》。</p><p>可知 Arc A770 与 RTX 3060 在特定游戏中的光追性能旗鼓相当 —— 即使少数情况下，N 卡还是具有些微的优势 —— 并且让 AMD RX 6650 XT 相形见绌。</p><p><a href="https://static.cnbetacdn.com/article/2022/0902/e459d1119988811.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0902/e459d1119988811.jpg" alt="6.jpg" referrerpolicy="no-referrer"></a></p><p>虽然 Arc A770 的 1440p + 光追体验有些勉强（RTX 3060 也类似），但 XeSS 和 DLSS 等增强特性还是可圈可点的。</p><p>不过英特尔并未直接搬出 A770 + XeSS / RTX 3060 + DLSS 的对比数据，仅通过一张 PPT 粗略描绘了 XeSS 是如何在“平衡”与“性能”预设下，让 1440p 光追游戏变得更具可玩性的。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1311207.htm" target="_blank">IFA 2022：英特尔强调Arc A700系列GPU光追性能优于英伟达RTX 3060</a></p></div>
