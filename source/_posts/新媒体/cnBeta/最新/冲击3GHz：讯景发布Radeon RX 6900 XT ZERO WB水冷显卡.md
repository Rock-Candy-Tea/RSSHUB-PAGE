今年 7 / 8 月，讯景（XFX）放出了 Radeon RX 6900 XT ZERO WB 的两次预告。<strong>现在，我们终于正式迎来了这款基于 AMD RDNA 2 GPU 的旗舰水冷显卡新品。</strong>在 PCB 疯狂堆料 + EK 水冷的加持下，官方宣称该卡旨在冲击 3GHz 以上的极端频率。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0920/5b5dfb52f2f3746.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：XFX <a href="https://wccftech.com/xfx-unleashes-radeon-rx-6900-xt-zero-wb-liquid-cooled-graphics-card-claims-to-hit-overclocks-beyond-3-ghz/" target="_self">官网</a>）</p><p><strong>以下是讯景 Radeon RX 6900 XT ZERO WB 显卡的主要特点：</strong></p><blockquote><p>● EK 水冷套件加持</p><p>● 由值得信赖的专家设计</p><p>● 极致 OC 频率</p><p>● GPU 可解锁 3GHz</p><p>● 双 BIOS 切换</p><p>● 支持 ARGB 灯效</p><p>● 14+2 相供电设计</p><p>● 三 8-pin 外接供电连接器</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0920/fdc1219ab6bcd64.jpg" alt="5.jpg" referrerpolicy="no-referrer"></p><p>为表明身份，讯景在 RX 6900 XT ZERO WB 显卡上留有硕大的“X”标记。前护罩由亚克力材料制成，以露出底下的 GPU / 显存 / VRM 供电模组。</p><p><img src="https://static.cnbetacdn.com/article/2021/0920/1776c7fd670efdb.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>整块亚克力材料都有 RGB LED 装饰，且猜测支持寻址（即 ARGB），此外我们可在护罩上方的塑料面板上看到 EK 与 XFX 的 Logo 。</p><p><img src="https://static.cnbetacdn.com/article/2021/0920/dbe4a44455b3e62.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>水冷头本体由 CNC 加工的镀镍电解铜底板制成，上部则是 CNC 加工的亚克力材料，辅以高品质的 EPDMO 密封垫圈。黄铜支座已出厂预装，以确保便捷和安全。</p><p><img src="https://static.cnbetacdn.com/article/2021/0920/032563ff826c60f.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>值得一提的是，由于使用了水冷方案，XFX Radeon RX 6900 XT ZERO WB 非公显卡也只需占用两个槽位的高度，辅以标准的 1×HDMI + 3×DisplayPort 显示输出端口。</p><p><img src="https://static.cnbetacdn.com/article/2021/0920/1103baab6e2fcf6.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>由于使用了 14-2 相供电的非公版 PCB 与三组 8-pin 的外接供电连接器，该卡能够在最大时钟频率下轻松突破 350W 的功耗（出厂 2325 / 2375 MHz、且板载 Navi 21 XTXH GPU 已完全解锁 3GHz 以上的 OC 频率）。</p><p><a href="https://static.cnbetacdn.com/article/2021/0920/d3a80d1d6d9a5f1.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0920/d3a80d1d6d9a5f1.jpg" alt="7.jpg" referrerpolicy="no-referrer"></a></p><p>最后，该卡带有 RGB 侧板，可在运行时点亮 XFX Radeon RX 6900 XT 的标志。</p><p><a href="https://static.cnbetacdn.com/article/2021/0920/c1344c63861412b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0920/c1344c63861412b.jpg" alt="6.jpg" referrerpolicy="no-referrer"></a></p><p>尽管注定价格会相当感人，但 Navi 21 XTXH 的芯片数量也确实有限。想要入手的朋友，还请不要犹豫太久。</p>
