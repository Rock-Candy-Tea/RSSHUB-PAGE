苹果自研 SoC 的发布，已经引发了 Apple Silicon 定制芯片和英特尔 x86 处理器之间的大量比较。尽管自 M1 发布以来，一直有人试图淡化 Apple Silicon 的实力。<strong>但由于合作伙伴“挤牙膏”太久，2021 款 MacBook Pro 搭载的 M1 Pro / M1 Max，还是极大地刷新了我们对于性能的认知。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1021/470d3e884b9802a.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>一方面，有人声称<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>会通过 12 代 Alder Lake CPU 的发布，来消弭<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a> M1 系列芯片的相关优势。</p><p>另一方面，前 Mac 首席 Jean-Louis Gassée 与前 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 部门主管 Steven Sinofksy 等业内资深人士，都认为“对苹果的成功表示不屑一顾”是极其愚蠢的。</p><p><a href="https://static.cnbetacdn.com/article/2021/1026/3e48c58b1d0afc5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1026/3e48c58b1d0afc5.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>本月早些时候，Steven Sinofksy 在 Medium 上撰写了一篇博客文章，称赞 M1 Pro 和 M1 Max 引领了计算行业的重大转变。</p><p>他还指出，当年苹果放弃摩托罗拉 PowerPC 并转投英特尔芯片的决定、与现在押注自研芯片并放弃英特尔 x86 CPU，都是出于同样的理由。</p><p><a href="https://static.cnbetacdn.com/article/2021/1028/2513c21a69307fd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1028/2513c21a69307fd.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">与英特尔处理器相比，Apple Silicon 能够为苹果带来更多优势。</p><p>据悉，作为前 Windows 总裁，Steven Sinofsky 经历了从失败的 Vista 转进至 Windows 7 的那个时代。所以他对 Apple Silicon 的重视程度，也是相当值得参考的。</p><p>Sinofksy 写道：M1 基于苹果对 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a> / <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> 芯片的所有感悟，不仅满足了传感器、OS 端口、安全性、电源管理、图形等方面的所有工作需求，还旨在解决 PC 行业的多年积弊。</p><p><a href="https://static.cnbetacdn.com/article/2021/1028/46827dea5273ac7.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1028/46827dea5273ac7.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p>此外前 Mac 总裁 Jean-Louis Gassée 也于近日发表了一篇博客文章，直言苹果放弃 x86 芯片的最大因素之一，就是英特尔未能将所有东西都集成到一整个 SoC 中。</p><p>具体说来是，在 x86 平台上，英特尔的系统封装（SIP）方案无法将所有处理器组件，都整合到单个片上系统（SoC）中。</p><p>与完全集成的 SoC 相比，SIP 性能更易受到较低的互连速度的制约。比如 M1 Pro / M1 Max 可提供高达 200 和 400 GB/s 的传输速率，但这确实 SIP CPU 难以逾越的鸿沟。</p>
