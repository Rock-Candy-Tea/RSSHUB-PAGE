<strong>为 GitHub Copilot 辅助编程工具提供技术支持的 OpenAI 组织，现又推出了可将自然语言转译成代码的私有测试版 Codex API 。</strong>这套应用程序接口能够理解十多种编程语言，允许开发者以简单的英语词汇来阐述相关命令并执行，从而轻松为当前的应用程序开发流程引入自然语言编程体验。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0811/798a8b43b53425f.gif" alt="1.gif" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：OpenAI <a href="https://openai.com/blog/openai-codex/" target="_self">官网</a>）</p><p>尽管今夏早些时候推出的 GitHub Copilot 编程辅助工具引发了巨大的争议，但我们不可否认 Codex 的技术优势。</p><p>在接受了数十亿行公共代码的训练之后，Codex 能够支持广泛的框架和语言，以及适应不同开发者的编程风格。</p><p><img src="https://static.cnbetacdn.com/article/2021/0811/5a09aca57d9b52a.gif" alt="2.gif" referrerpolicy="no-referrer"></p><p>据 OpenAI 所述，该 API 模型最擅长于 Python，但也“精通”JavaScript、Go、Perl、PHP、Ruby、Swift、TypeScript、Shell 等编程语言。</p><p>得益于 Python 编程时的 1<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_110018%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">4K</a>B 内存，Codex 还能够在执行包括转译、解释和重构代码等在内的任务时，充分考虑到山下文信息。</p><p><img src="https://static.cnbetacdn.com/article/2021/0811/2d414ddbeb04871.gif" alt="3.gif" referrerpolicy="no-referrer"></p><p>在一篇博客文章中，OpenAI 解释称 Codex 可让计算机能够更好地理解人们的意图，从而让每个人都可利用计算机来实现更多目的。</p><p>最后，OpenAI 表示 Codex 将在起步阶段提供免费体验，现邀请广大企业与开发者们借助该 API 来构建各种项目。</p>
