<div>   
<strong>微软正在向运行 Windows 10 21H2 / 21H1 / 20H2 / 2004 的 PC 推送 KB5007253 累积更新。</strong>与之前的可选更新一样，本次 KB5007253 也是通过 Windows Update 渠道推送。对于离线用户，亦可移步至 Microsoft Update Catalog（更新目录站点），以手动获取安装包的直接下载链接。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1123/c68ed3fe9ee9f03.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1123/c68ed3fe9ee9f03.png" referrerpolicy="no-referrer"></a></p><p>作为 Windows 10 2021 年 11 月累积更新的一部分，KB50007253 也是<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>向稳定版通道推送的首批 21H2 补丁。</p><blockquote><p>更新完成后，所有受支持的 Windows 10 PC 的系统内部版本号都会有一定的增加，比如 Windows 10 21H2（11 月更新）就是 19044.1382 。</p><p>此外 Windows 10 21H1（五月更新）将升到 19043.1382，Windows 10 20H2 也将提升至 19043.1382 。</p></blockquote><p>需要指出的是，内部版本号也是微软划分 Windows 10 操作系统版本的唯一主要区别，即便 Windows 10 21H2 / 21H1 / 20H2 都基于 Vibranium 分支。</p><p>对于 Windows 10 2004 来说，KB50007253 可选更新也带来了一系列改进和 Bug 修复。比如此前在加载某些可变字体时，可能遇到意外的渲染问题。</p><p><img src="https://static.cnbetacdn.com/article/2021/1123/ccdb37d114c2fe0.jpg" referrerpolicy="no-referrer"></p><p><strong>如需获取 Windows 10 Build 19044.1382，还请参考以下步骤：</strong></p><blockquote><p>（1）打开【设置】</p><p>（2）转至【更新和安全】</p><p>（3）点击【Windows 更新】</p><p>（4）选择【检查更新】，以查看是否有可用更新。</p><p>（5）单击【下载并安装】</p><p>（6）同意【立即重启】以完成待安装的更新。</p></blockquote><p>如果暂未检查到更新补丁，<strong>亦可访问 Windows Update Catalog 以获取离线安装包：</strong></p><blockquote><p><a href="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5007253" _src="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5007253" target="_blank">https://www.catalog.update.microsoft.com/Search.aspx?q=KB5007253</a></p></blockquote><p>获取到 .msu 格式的补丁安装包后，即可将之手动部署到所需的 Windows 10 PC 上（前提是系统版本仍在微软官方支持期内）。</p><p>若不确定自己的系统版本，可 WinKey + R 运行“winver”命令，或转至“Windows 设置 → 系统 → 关于”页面查看。</p><p><img src="https://static.cnbetacdn.com/article/2021/1123/c22b57d330dda65.png" referrerpolicy="no-referrer"></p><p><strong>以下是 Windows 10 KB5007253（内部版本 19044.1382）的更新日志：</strong></p><blockquote><p>（1）修复某些字体无法正确显示的问题。</p><p>（2）修复导致 Internet Explorer 停止工作的问题。</p><p>（3）修复 Windows 设置页面意外崩溃的问题。</p><p>（4）修复 Windows 游戏工具栏截图崩溃问题。</p><p>（5）修复阻止应用出现在【开始】菜单上的问题。</p></blockquote><p>此外微软修复了影响 SearchFilterHost.exe 进程、以及连接到远程打印机时的严重 Bug（例如 0x00000709 报错）。</p><p>Windows 10 19044.1382 修复了导致 Windows 设置页面意外出现的问题，比如当用户尝试从【设置】页面卸载字体时，就有可能遇到这种情况。</p><p>以及在文件资源管理器中使用文件夹视图时，可能影响用于重命名文件的 UI 的另一个 Bug 。</p><p>对于 Windows Linux 子系统（WSL）用户，微软为 wsl.exe 带来了新的 -cd 参数支持，以允许用户指定 Windows / Linux 路径作为子系统的起始目录。</p><p>最后，微软修复了用户在尝试更新或修复程序时，某些 Windows 应用程序可能崩溃的问题。</p>
