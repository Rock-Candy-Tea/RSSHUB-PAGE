Windows on ARM 笔记本电脑一直没能在市场上取得较大的反响，一大原因是产品性价比不够出众。更何况在功能不及英特尔（x86）平台的情况下，还面临着兼容性方面的问题。现在，高通似乎也明白了这一点。<strong>在接受 Golem.de 采访时，高通产品管理高级总监 Miguel Nunes 更是将之归咎于“OEM 厂商为始终连接的 PC 定下了超过 1000 美元的指导价”。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0104/e1f2d8a9710cfbb.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（via <a href="https://www.golem.de/news/windows-on-snapdragon-die-preisgestaltung-war-nicht-angemessen-2112-162103.html" target="_self">Golem.de</a>）</p><p>尽管 Nunes 表示定价取决于 OEM，但也有消息称高通骁龙 8cx 等 ARM 芯片组的定价与<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> x86 芯片大致相同，但两者的性能显然不在同等水平。</p><p>以当前在售的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a> <a data-link="1" href="https://microsoft.pvxt.net/9W473" target="_blank">Surface</a> Pro X 为例，其售价更是超过了 1500 美元。当然我们也不该无视其它价格合理的机型，比如零售价不到 500 美元的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a> Galaxy Book Go 。</p><p>总而言之，影响 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> on ARM 移动设备普及的更大障碍，并不是设备本身的成本、而是消费者所能感受到的价值。</p><p>这方面的一个成功案例，就是<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>的 Apple Silicon M1 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">MacBook</a> 。如果性能与 x86 平台相当或更好，消费者显然不会太过纠结于设备到底基于哪款芯片组。</p><p>至于高通和微软能否携手引领 Windows on ARM 走向更光明的未来，还得看后续的 Windows 11 on ARM 能否带来切实的改进。</p>
