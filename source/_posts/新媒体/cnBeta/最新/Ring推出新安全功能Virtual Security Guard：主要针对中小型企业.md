据外媒The Verge报道，<strong>Ring今日披露了其最新的雄心勃勃的安全产品：让现实生活中的人监视你的家。</strong>Ring的Virtual Security Guard（虚拟安全卫士）将监控安全的概念提升到了新的水平，它是一项新的订阅服务，可以让专业监控中心的代理对任何室外Ring摄像头的特定运动警报作出反应并为你采取行动。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0929/df660724af3f8d7.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0929/df660724af3f8d7.png" alt="Ring1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Virtual Security Guard需要Ring Alarm家庭安全系统（第一代、第二代或专业版）在每月20美元的Ring Protect Plus专业监控订阅之外还需支付99美元。它适用于任何Ring的有线室外摄像机（包括所有有线门铃、Spotlight Cam、Stick-up Cam和泛光灯摄像头）。用户可以选择通过Ring应用进行监控，当摄像头检测到运动事件时，而Ring报警器已经启动，监控公司就会收到警报并根据其看到的情况作出反应。代理人将使用一系列准则和用户的偏好来定制响应并可以使用摄像机的双向通话等功能与访客沟通、打开警报器或发送紧急服务。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0929/42da3bd677836b0.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0929/42da3bd677836b0.png" alt="Ring2.png" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">如果代理看到他们认为是在做坏事的人，他们会首先使用双向通话，让对方知道他们正在被监控并要求他们离开。如果他们不走，代理就会拉响摄像头中的警报器并继续监控，同时调度紧急服务。在这个阶段，他们还会给业主打电话和发短信并提供视频的链接。</p><p style="text-align: left;">出于明显的隐私考虑，用户只能在启动时注册户外摄像头。如果试图添加一个在室内的摄像头，监控公司应该会停止监控，直到它被重新配置到室外。当然，这有赖于人工代理来标记这个问题，这对隐私来说是一个潜在的绊脚石。</p><p style="text-align: left;">由于Virtual Security Guard不能在报警被触发时监控任何摄像头，所以它并不是环形报警系统的一种视觉验证形式（如果有什么东西触发了你的一个报警传感器，监控服务可以查看你的摄像头的实时画面）。它的唯一目的是在室外摄像机上检测到运动时进行监控。</p><p style="text-align: left;">“视觉验证是我们已经看过的东西，我们将继续看。这比视觉验证更上一层楼，视觉验证实际上是在事情发生后进行的，”Ring创始人Jamie Siminoff在宣布之前的一次介绍称，“这是更主动的，是一个虚拟的保安。”</p><p style="text-align: left;">监控服务只能在运动被触发和Ring Alarm被启动时访问实时视图；当警报被解除时代理不能查看摄像头画面，并且也不能下载、分享或保存视频。在Ring应用中，人类工作者审查过的任何视频上都会有一个标签。Ring表示，在推出时，该服务将由Rapid Response（Ring Alarm使用的专业监测公司）提供，但未来将会扩大到其他供应商以获取更低的价格。在美国，Virtual Security Guard从今天开始可以提前使用。</p><p style="text-align: left;">这里的概念是把响应摄像头运动警报的责任从用户身上转移到其他人身上。如果用户的Ring Alarm在凌晨2点向其发出通知，说住宅后面有一个人，这样的通知用户很可能会错过或者即使看到了也不会再去睡觉了。现在，Virtual Safety Guard可以查看触发警报的事件，如果事实证明它是一个非事件则就可以继续窝在梦乡里了。</p><p style="text-align: left;">另外，它还可以在用户到达或离开家时起到安心的作用。Ring总裁Leila Rouhi描述了她在测试该服务时在家中经历的一个场景。当她到家时，她的前院发生了骚乱，她的狗追着另一只狗跑了。“我的门前有一点混乱，代理检查后问，‘嘿，一切都好吗？有什么我们能为你做的吗？’值得庆幸的是，情况得到了控制，但知道在紧急情况下，如果有不好的事情发生，有人在关注你，如果你需要他们，他们可以采取行动，这样感觉真的很好。”</p><p style="text-align: left;">当涉及到Ring员工查看摄像头录像时，Ring的记录充其量是不完整的。2019年的一份报告指出，Ring让员工观看并相互分享未加密的客户视频。不过隐私倡导者、世界隐私论坛执行主任Pam Dixon审查了Virtual Safety Guard并告诉The Verge记者，她认为Ring和亚马逊在这里已经超过了最佳做法。“他们对监控人员进行了背景调查，他们对他们进行的偏见培训也让我感到放心，”她说道，“我不知道还有哪家公司会这么做。这给了我真正的安慰，这个产品不会兜售偏见，这是一个很大的优点。” Ring还告诉她，它将对监测人员进行定期审计。“我对此感到很满意，”她继续说道，“但如果是家里的摄像头，那情况就不一样了。”</p><p style="text-align: left;">Dixon指出，像Virtual Security Guard这样的工具对许多弱势群体来说将是一个很好的资源。“我们跟很多家庭暴力受害者合作--很多幸存者在安全工具上投入了大量资金，这样当有人破坏他们的安全时，他们可以得到通知。” 但对大多数人来说，为自己的家配备一个Virtual Security Guard可能是多余的。</p><p style="text-align: left;">看起来Ring也意识到了这一点，其将Virtual Security Guard更多地瞄准了中小型企业，据Ring披露，其中许多企业已经在使用Ring Alarm产品来保护其场所。</p>
