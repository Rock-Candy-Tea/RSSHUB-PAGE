在最近更新的 Windows 11 预览构建版本中，我们看到了诸多新功能，以及大量的小调整。<strong>其中让我们眼前一亮的，莫过于接受了 Mica 材质刷新的窗口标题栏。</strong>即使是通过 WinKey + R 快捷键唤出的“运行”对话框，也变得更加现代、且与 Windows 11 的整体感觉更加一致。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0217/ae671a66374e688.jpg" referrerpolicy="no-referrer"></p><p>虽然算不上是大修，但正是这些小细节的堆砌，才共同推动了 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 操作系统的稳步改进。</p><p>此外 Windows 11 Build 22557 编译版本引入了用户捕捉窗口的新过渡动画，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>表示：</p><blockquote><p>我们正在更新从一个捕捉窗口状态、到另一个状态的过渡动画的显示。</p><p>在使用‘截图助手’（Snap Assist）捕捉某些窗口、并切换至另一种截图选项时，你将观察到更加流畅的过渡动画。</p></blockquote><p>与此同时，文件资源管理器现可显示文件夹中的文件预览，充电指示图标从电源线改成了闪电标记，以便通过系统托盘来一眼看清低电量的充电状态。</p><p>最后，微软改进了前后台窗口的切换体验，使得用户能够更轻松地将普通窗口和程序组区分开来。</p><blockquote><p>在使用 Alt + Tab 组合键、任务视图、以及任务栏预览时，都可看到这项变化。</p><p>此外所有运行 Build 22554 及更高版本的 Insider 测试者们，会发现 Alt + Tab 组合键已改为窗口化的显示效果。</p></blockquote><p>目前上述改动都仅限于 Insider 测试渠道，但若一切顺利，其有望于 2022 下半年开始向广大 Windows 11 用户正式推送。</p>
