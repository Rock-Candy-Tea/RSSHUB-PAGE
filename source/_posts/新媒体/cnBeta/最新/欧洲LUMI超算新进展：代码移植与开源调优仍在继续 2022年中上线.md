位于芬兰的 LUMI 超级计算机仍在组装过程中，但项目团队已经介绍了他们的最新进展。<strong>据悉，由 AMD 提供技术支撑的这套装置，一共有 2560 个计算节点。</strong>每个节点包括一枚 64 核心的 Trento CPU、以及四路 Instinct miMI250X GPU 加速卡。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0210/eaa95fc09caff7b.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图自：<a href="https://www.lumi-supercomputer.eu/" target="_self">lumi-supercomputer.eu</a>）</p><p>按照原计划，拥有 375+ PFLOPs 算力的 LUMI 超算本该在 2021 年底前上线。但受供应链短缺的挑战，最新目标已设定在 2022 年中期全面上线。</p><p>虽然硬件仍处于组装过程中，但高性能计算（HPC）工程师一直在努力优化开源 Linux 软件堆栈。</p><p>作为 CSC 的首席 HPC 科学家，Georgios Markomanolis 曾于去年 2 月发表过一次很棒的演讲。</p><p>期间提到了 Radeon Open eCosystem 的现状，以及他们经手的所有移植工作 —— 包括让传统上以 NVIDIA 为中心的 HPC 工作负载，也能够在那时的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 测试硬件上工作。</p><p>早期规划使用的是 Instinct MI100 GPU 加速卡，但 LUMI 也在等待 MI250X 的落地成熟。</p><p><a href="https://static.cnbetacdn.com/article/2022/0210/9cf9900f63875fc.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0210/9cf9900f63875fc.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://www.phoronix.com/scan.php?page=news_item&px=LUMI-AMD-OSS-Tuning-2022" target="_self">Phoronix</a>）</p><p>在上周末的 <a href="https://fosdem.org/2022/schedule/event/utilizing_amd_gpus/" target="_self">FOSDEM 2022</a> 大会期间，Georgios Markomanolis 又分享了他们在过去一年的开源软件工作新进展。</p><blockquote><p>可知 HPC 工程师们已经对面向 GPU 计算的 AMD ROCm 开源软件堆栈使用得更加技术纯熟。</p><p>同时继续在 HIP 上努力将更多 CUDA 代码库移植到 AMD GPU 支持软件中。</p><p>然后努力对代码进行微调，以便在 AMD 硬件上顺畅运行。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0210/41857f4d84ce54d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0210/41857f4d84ce54d.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>而在过去一年里被添加到工具包中的新软件组件，值得一提的就包括 AMD 的 GPUFort 开源项目。</p><blockquote><p>其旨在帮助开发者将 OpenACC 和 CUDA Fortran 代码，迁移到以 AMD 为主的架构中。</p><p>尽管相关工作仍相当复杂，但通过将相关代码库移植到 AMD ROCm 空间，应用程序可获得最佳的 GPU 性能表现。</p></blockquote><p>最后，对 LUMI 上线运营的具体事务感兴趣的朋友，可移步至 <a href="https://fosdem.org/2022/schedule/event/utilizing_amd_gpus/" target="_self">FOSDEM.org</a> 查阅更多正在进行 CSC 开源移植和调优工作。</p>
