自 2015 年以来，Google 就公开表示建造微型雷达芯片。应用场景包括监控你的睡眠状态、控制智能手表，计算纸片，并让你演奏世界上最细的小提琴等等。只是 Google 的 Soli 芯片商业化道路并不顺畅，首配该芯片的 Pixel 5 销量也不是很好。<strong>现在，Google推出了一个名为 <a href="https://cta.tech/Ripple#hub" target="_blank">Ripple</a> 的开源 API 标准，可以装备到汽车等设备上。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0108/83214e119214647.webp" alt="yw29r5h5.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">从技术上讲，Ripple 是在消费者技术协会（CTA）的支持下进行的，该协会也是每年 1 月在拉斯维加斯举办 CES 展会的行业机构，但真正推动这个项目的是 Google。</p><p style="text-align: left;">率先提出 Soli，Google ATAP 团队负责人 Ivan Poupyrev 表示：“Ripple 将释放有益的创新，使每个人受益。常规用途的雷达是以尊重隐私的方式解决关键用例的关键新兴技术”。</p><p style="text-align: left;">在<a href="https://github.com/CTA-Ripple" target="_blank"> Ripple 项目的 GitHub 页面上</a>，提及了几处“Copyright 2021 Google LLC”，而且贡献者必须签署Google开源许可协议才能参与。(Ripple 似乎是Google一年前悄悄提出的"标准雷达API"的改头换面（PDF）。</p><p style="text-align: left;">不过，这并不意味着Soli可能找到新的生命，而且雷达具有隐私优势的想法可能是有道理的。这项技术可以很容易地检测到某人是否在现场、附近，以及/或告诉他们的设备做什么，而不需要麦克风或摄像头。</p><p style="text-align: left;">福特方面告诉The Verge，车内雷达可能成为其驾驶辅助技术的一部分。现在，该汽车制造商说它正在使用“先进的外部雷达”来研究这些功能（对我来说，这听起来很昂贵）。以下是福特公司 Jim Buczkowski 的声明，他目前负责该公司的研究和先进工程团队。</p><p style="text-align: left;">我们正在研究如何使用内部雷达作为传感器来源，以增强各种客户体验，而不是像今天使用先进外部雷达的领先的福特Co-Pilot360驾驶辅助技术那样。在半导体行业的参与下，一个标准的API将允许我们开发独立于硬件来源的软件，并给予软件团队在多个雷达平台上进行创新的自由度。</p>
