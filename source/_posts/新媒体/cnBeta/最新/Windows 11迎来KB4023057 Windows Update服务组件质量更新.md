<strong>如果你新装的 Windows 11 操作系统无法获得累积更新和其它补丁，那很可能是 Windows Update 服务组件在软件配置方面出现了 Bug 。</strong>为了修复这方面的问题，微软已经向 Windows 11 设备推送了首个可靠性更新。但是 KB4023057 这个补丁编号，早前其实也向 Windows 10 平台推送过。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1106/c319a58f54c3167.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1106/c319a58f54c3167.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">截图（via <a href="https://www.windowslatest.com/2021/11/06/windows-11-kb4023057-reliability-update-is-now-rolling-out/" target="_self">Windows Latest</a>）</p><p>在<a href="https://support.microsoft.com/en-us/topic/kb4023057-update-for-windows-update-service-components-fccad0ca-dc10-2e46-9ed1-7e392450fb3a" target="_self">官方公告</a>中，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>将 KB4023057 描述为“Windows Update 服务组件更新”，涵盖了对所有受支持 Windows 版本（1908 / 2004 / 21H1 及更高版本）的可靠性更新与其它改进。</p><p>10 月 27 日，微软为 Windows 11 操作系统编译了相同的更新，其中包含了用于解决 Windows Update 更新过程故障的内容。</p><p>Windows Latest 在实测后发现，如果 Windows 11 设备存在阻止更新下载的问题，该更新可帮助用户重置某些注册表键值。</p><p><img src="https://static.cnbetacdn.com/article/2021/1106/86af2d1620a1380.png" alt="345.png" referrerpolicy="no-referrer"></p><p><strong>以下是 KB4023057 可靠性更新的主要内容：</strong></p><blockquote><p>● 若 Windows 疑难解答检测到问题，则会尝试重置网络设置操作。</p><p>● 该补丁将清理任何可能阻止 Windows Update 更新获取的注册表项。</p><p>● 可修复损坏的 Windows 更新组件，或尝试重新启用已禁用的组件 / 服务。</p><p>● 若设备存储空间不足，系统还会压缩用户配置文件目录中的某些文件，以帮助释放磁盘空间、从而顺利安装重要的更新。</p><p>● 微软还将尝试重置 Windows 更新数据库，并修复阻止设备获得 Windows 11 更新的问题。某些情况下，控制面板中的更新历史记录都将被清除。</p></blockquote><p>最后，面向 Windows 11 操作系统的 KB4023057 可靠性更新正在分阶段推出，它将自动安装在出现相关问题的设备上。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1021925.htm" target="_blank">微软重新发布KB4023057补丁 再次推动Windows 10更新</a></p></div>
