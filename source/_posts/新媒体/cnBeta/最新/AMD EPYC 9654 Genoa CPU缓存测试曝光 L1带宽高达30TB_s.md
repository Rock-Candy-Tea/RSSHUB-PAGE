<strong>@結城安穗-YuuKi_AnS 近日在社交平台上分享了 AMD 霄龙（EPYC）9654“Genoa”96 核 CPU 的缓存 / 内存基准测试成绩。</strong>正如我们在之前几天泄露的基准测试中所看到的那样，Genoa 为 x86 服务器领域带来了疯狂的性能提升。可知该处理器的缓存间带宽达到了 Milan-X 的两倍，且 L1 缓存的传输速率高达 30 TB/s 。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0720/422723f2428835d.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（via <a href="https://wccftech.com/amd-epyc-9654-96-core-genoa-cpu-tears-apart-milan-x-sapphire-rapids-cache-latency-benchmarks-up-to-30-tbps-over-2x-performance-increase/" target="_self">WCCFTech</a>）</p><p>本次测试用到了一枚工程样品（ES）阶段的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> EPYC 9654“Genoa”旗舰 CPU 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0720/ad44dafdaf0169b.png" alt="0-1.png" referrerpolicy="no-referrer"></p><p>并且对比了上一代 Milan-X 和<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>至强铂金（Xeon Platinum）8480+“Sapphire Rapids-SP”芯片的缓存与延迟数据。</p><p><img src="https://static.cnbetacdn.com/article/2022/0720/ab2bce2cf3236cf.png" alt="0-2.png" referrerpolicy="no-referrer"></p><p>今年早些时候，这位知名芯片爆料人还分享过 AMP EPYC Milan-X、以及相当早期的 Sapphire Rapids-SP 服务器 CPU 的缓存和延迟数据。</p><p><img src="https://static.cnbetacdn.com/article/2022/0720/61836a361d7b91d.png" alt="0-4.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">截图（来自 @結城安穗-YuuKi_AnS / <a href="https://t.bilibili.com/684072201822928899" target="_self">B 站动态</a>）</p><p>据悉，AMD EPYC Milan-X 在 Zen 3 架构的 CCD 核心之上、引入了一个巨大的末级缓存（+64MB / 总计 512MB）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0720/fa274be3c3abb1f.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0720/fa274be3c3abb1f.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>再算上每个 CCD 可用的 32MB 缓存，总量就达到了 768MB 。与不带 3D V-Cache 的 Milan 标准 SKU 相比，Milan-X 不仅可用缓存更大、运行速度也相当之快。</p><p><a href="https://static.cnbetacdn.com/article/2022/0720/9c3f6e6cd8fbed3.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0720/9c3f6e6cd8fbed3.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>作为本文的主角，EPYC 9654 Genoa CPU 拥有 96 核 / 192 线程。测试平台为双路服务器主板，基础频率 2 GHz / 加速频率在 2.15~3.7 GHz 之间，最大 TDP 为 360W 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0720/c939ec5d377182d.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>作为一枚工程样品（ES），我们毫不怀疑正式 SKU 的参数和性能会更加亮眼。此外据说还有一枚更加旗舰的 EPYC 9664 Genoa CPU，推测其频率和 TDP 都会更高。</p><p><img src="https://static.cnbetacdn.com/article/2022/0720/48daa90f873da9b.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>与 Milan-X 相比，EPYC 9654 拥有 96MB L2（每核心 1MB）+ 384MB L3 缓存，总计 480MB 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0720/868323cc5bb82f9.jpg" alt="5.jpg" referrerpolicy="no-referrer"></p><p>展望明年的 Genoa-X，其有望迎来 768MB 的 3D V-Cache 缓存，使得总量达到 1248MB 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0720/d0378d77b581cdf.png" alt="6.png" referrerpolicy="no-referrer"></p><p>最终可知 EPYC Genoa 可提供高达 30 TB/s 的缓存间带宽，较 EPYC 7773X 提升 2 倍以上。</p><p><img src="https://static.cnbetacdn.com/article/2022/0720/49d4d91b093499b.png" alt="7.png" referrerpolicy="no-referrer"></p><p>此外 Intel Sapphire Rapids-SP 和 AMD EPYC Genoa 平台均使用 DDR5-4800 系统内存进行了测试。</p><p><a href="https://static.cnbetacdn.com/article/2022/0720/076cd3edbeec0b1.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0720/076cd3edbeec0b1.jpg" alt="8-1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">EPYC 9654 / Genoa CPU 延迟图表（来自：Yuuki_AnS）</p><p>ES 版 EPYC 9654 具有所有现代服务器芯片中顶级的低延迟表现（仅略逊于 Milan-X），但最终零售版本还有进一步优化的空间（Sapphire Rapids-SP 的赢面极小）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0720/4055cd9f2e2f4ed.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0720/4055cd9f2e2f4ed.jpg" alt="8-2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">EPYC 7773X / Milan-X CPU 延迟图表</p><p>L1 缓存测试中，Genoa CPU 是 Milan-X 的 2.2 倍、且比 Sapphire Rapids-SP Xeon 高 17% 。L2 缓存测试中，Genoa CPU 较 Milan-X 和 Sapphire Rapids-SP 高了 89% / 2.44 倍。</p><p><img src="https://static.cnbetacdn.com/article/2022/0720/9865c79e12843d6.png" alt="9.png" referrerpolicy="no-referrer"></p><p>最后，在 L3 带宽方面，Genoa CPU 性能分别达到了 Milan-X / 至强竞品的 3.27 / 8.8 倍 —— CPU 间的缓存带宽，达到了惊人的 30 TB/s 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0720/0b99c867a5cb0e4.png" alt="10.png" referrerpolicy="no-referrer"></p><p>其它方面，AMD EPYC Genoa CPU 将提供 128 条 PCIe 5.0 通道，且其中 160 条用于双路（2P）服务器配置。</p><p>内存方面，SP5 主板平台将支持 12 通道 @ DDR5-5200 规格（每通道 2 DIMM 插槽）、较当前的 8 通道 @ DDR4-3200 提升显著，辅以多达 3TB 的系统内存容量。</p><p>如果一切顺利，AMD <a href="https://wccftech.com/amd-epyc-genoa-cpu-family-specs-leaks-zen-4-with-up-to-96-cores-192-threads-384-mb-l3-cache-400w-tdp/" target="_self">EPYC 9000</a> 系列 Genoa CPU 将于今年下半年到来。</p>
