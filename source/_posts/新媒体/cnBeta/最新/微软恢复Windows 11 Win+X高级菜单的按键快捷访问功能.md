Windows Latest 指出，<strong>微软在 Windows 11 操作系统中默认启用了可通过“WinKey + X”组合键唤出的高级用户菜单。</strong>除了 UI 方面的重新设计，弹出菜单中还包括了设备管理器、任务管理器、以及其它高级 Windows 工具。<br>
<p><a href="https://static.cnbetacdn.com/article/2021/0727/81a38861ac9c6fb.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0727/81a38861ac9c6fb.jpg" referrerpolicy="no-referrer"></a></p><p>遗憾的是，微软同时也砍掉了该菜单中的“访问键”支持，意味着无法通过按下对应的字母，来快速选中菜单栏中的条目。</p><p>举个例子，原先用户能够通过 WinKey + X 组合键唤出高级菜单，然后按下字母 D 来快速启动设备管理器（或按下字母 T 来启动任务管理器）。</p><p>微软表示，在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 操作系统中，按下 WinKey +X 组合键的高级菜单，已不支持按键的快速访问。</p><p>不过在 Windows 11 Build 22000.71 编译版本中，微软终于恢复了对快捷访问键的支持。<strong>该公司在反馈中心的一篇帖子中写道：</strong></p><blockquote><p>基于大家的<a href="https://insider.windows.com/en-us/feedbackhub/fb?contextid=49&feedbackid=4cbd73a1-fa9e-4d89-a247-ce27ced3cd66" target="_self">反馈</a>，我们正在努力为 WinKey + X 高级菜单重新引入对按键快捷访问的支持。</p><p>Insider 测试者们或已在 Build 22000.71 中看到了该功能，且现已向开发频道推送。</p><p>目前我们正在调查一个问题，即该选项偶尔无法正常使用。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2021/0625/97c10b9160114a7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0625/97c10b9160114a7.jpg" referrerpolicy="no-referrer"></a></p><p>其次，微软已确认 Windows 11 不支持将应用程序拖放到任务栏上，且不再允许用户调整任务栏的位置。</p><p>此前，用户能够自由地将 Windows 任务栏拖动到屏幕的顶部和左右两侧。但在 Windows 11 新系统中，其已被固定在底部且剧中。</p><p>至于微软是否会听取用户的反馈，在后续的版本中恢复这一调节选项，仍有待时间去检验。</p><p><img src="https://static.cnbetacdn.com/article/2021/0727/ccaa5f1c947ccd7.jpg" alt="Win11 Time.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://allthings.how/how-to-change-time-on-windows-11/" target="_self">AllThings</a>）</p><p>最后，微软还剔除了一项注册表键值。这家软件巨头在反馈中心的帖子中证实了此事，意味着用户无法再让任务栏时钟走秒显示。</p><p>此前 Windows 10 用户能够对注册表进行破解，以支持 hh:mm:ss 的任务栏时钟格式。</p><p>如果微软最终未能听取大家的反馈意见，那我们也只得依靠 T-Clock Redux 这样的第三方应用程序，来填补 Windows 11 操作系统中缺失的这项功能。</p>
