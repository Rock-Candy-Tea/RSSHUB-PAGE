在现已删除的一篇博客文章中，微软似乎意外宣布了 Web 版本的 VS Code 公共预览。<strong>作为一款基于网页的集成式开发环境（IDE），文中强调了语法、搜索、多文件编辑、以及源代码控制视图等特性。</strong>该公司称：“今日宣布的 VS Code for the Web 预览，能够完全在您的浏览器中运行”。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0901/43d60db200cde7a.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0901/43d60db200cde7a.png" referrerpolicy="no-referrer"></a></p><p>据悉，VS Code for the Web 允许每个人在 <a href="https://vscode.dev/" _src="https://vscode.dev" target="_blank">https://vscode.dev</a> 上免费体验。可知其支持快速打开和浏览托管在 GitHub 和本地机器上的源码（后续也将很快支持 Azure Repos 资源库），并实施轻量级的修改。</p><p>不过由于无法访问计算资源，所以用户还是无法通过 VS Code for the Web 构建、运行或调试应用程序，此外也无法调用集成终端。但若需要完整的 VS Code 体验，该服务还支持升级到 GitHub Codespaces 实例。</p>
