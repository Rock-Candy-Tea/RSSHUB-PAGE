<strong>为照顾想要升级 Windows 11，但又不确定当前硬件和固件是否支持的客户，微软已经重新发布了 Windows 11 兼容性检查工具。</strong>据悉，微软于 2021 年 6 月份正式宣布了 Windows 11 操作系统，并计划于今年晚些时候正式向广大用户提供。至于新版 Windows 11 兼容性检查工具，它现在能够更加轻松地判断您的设备是否与之兼容。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0625/97c10b9160114a7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0625/97c10b9160114a7.jpg" referrerpolicy="no-referrer"></a></p><p><a href="https://www.windowslatest.com/2021/08/28/windows-11-compatibility-check-tool-re-released-by-microsoft/" target="_self">Windows Latest</a> 指出：配备<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 8 代酷睿和 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 2000 系列及更高版本硬件的平台，都可无缝支持 Windows 11 操作系统。</p><p>此外根据来自 Windows Insider 测试项目的遥测数据，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>最近又添加了对英特尔 7 代芯片的支持，具体包括如下三类：</p><blockquote><p>● 酷睿 X 系列 HEDT 处理器；</p><p>● 至强 W 系列处理器；</p><p>● 以及酷睿 i7-7820HQ 处理器（用于微软 <a data-link="1" href="https://microsoft.pvxt.net/9W473" target="_blank">Surface</a> Studio 2 等设备）。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2021/0828/024a9677203be8e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0828/024a9677203be8e.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>基于此，可知微软正在放弃对大部分英特尔 7 代处理器、以及 6 代全系处理器的支持。事实上该公司自家的部分 Surface 产品线，也被拦在了 Windows 11 的升级大门之外。</p><p>当然，就算微软坚持为新一代操作系统强制指定 TPM 2.0 支持，后续仍不排除有奇妙的漏洞允许用户在任何设备上安装体验。</p><p>感兴趣的朋友，现可下载重新发布的“PC 健康检查工具”，以评估当前设备是否适用于下一代 Windows 11 操作系统。</p><p>早些时候，该公司因为由部分用户抱怨无法正常工作、而下架了旧版的“PC 健康检查工具”。</p><p><a href="https://static.cnbetacdn.com/article/2021/0828/a8805b79e70aa6d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0828/a8805b79e70aa6d.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>如图所示，作为 Windows Insider 测试计划的一部分，“PC Health Check”能够检查设备软硬件是否适用于 Windows 11，尤其是可信任平台模块（TPM 2.0）与安全启动（Secure Boot）这两项特性。</p><p><strong>有升级需要的朋友，可参考如下方法：</strong></p><blockquote><p>● 前往‘设置 -> 系统 -> 关于’并验证设备架构。</p><p>● 打开微软下载页面，选择正确版本的 PC Health Check 检查工具。</p><p>● 双击 WindowsPCHealthCheckSetup.<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://msi-pc.jd.com/" target="_blank">MSI</a> 以安装。</p><p>● 在 PC 健康检查工具中，点击‘立即检查’。</p></blockquote><p><strong>下载地址：</strong></p><blockquote><p><a href="https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewpchealth"
