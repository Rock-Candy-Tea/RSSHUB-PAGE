<div>   
<strong>如果 Firefox 内置的账户管理器（about:profile 页面）无法满足你的使用需求，那么最新上线的 <a href="https://addons.mozilla.org/en-US/firefox/addon/profile-switcher/" target="_blank">Profile Switcher</a> 扩展程序能够让你像 Chrome 一样从工具栏上轻松切换和管理账户。</strong>Firefox 将你的偏好、书签、历史记录存储在 Program Files 中一个名为 Profile 的独立文件夹中。您可以创建多个帐号。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0406/88c59a7b3f6afb6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0406/88c59a7b3f6afb6.jpg" alt="QQ截图20210406113343.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Firefox 账户管理器可以让你创建、删除、重命名和切换帐号。更准确地说，它显示所有的配置文件，并允许完成以下操作</p><blockquote style="text-align: left;"><p style="text-align: left;">● 删除帐号</p><p style="text-align: left;">● 重命名帐号</p><p style="text-align: left;">● 设置某个帐号为默认</p><p style="text-align: left;">● 以某个帐号打开新窗口</p></blockquote><p style="text-align: left;">您也可以在 Firefox 不运行时使用帐号管理器来管理帐号。当你启动 Firefox 时，这可能会很有用。在“运行”对话框中输入 Firefox.exe -p 命令，选择要启动的帐号，也可以删除或重命名某个配置文件。</p><p style="text-align: left;">但是 Chrome 浏览器允许从工具栏在帐号之间进行切换。它显示了其他的帐号，以方便地从活动的配置文件按钮本身切换到。如果你想在火狐浏览器中获得类似的配置文件体验，那么你可以在 AMO 上安装 Profile Switcher 扩展程序吧。</p><p style="text-align: left;">该扩展可以在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 和 Linux 上工作，但需要额外的软件 Profile switcher for Firefox connector 来使该扩展发挥作用。在 Firefox 中使用 Profile Switcher：</p><blockquote style="text-align: left;"><p style="text-align: left;">1. 在安装该扩展程序，在任务栏上点击帐号图标</p><p style="text-align: left;">2. 弹出窗口上会罗列已经在 Firefox 上运行的帐号</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0406/5230b116becb787.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">3. 你可以点击一个帐号，在新的浏览器窗口中启动。</p><p style="text-align: left;">4. 在底部的“管理帐号”选项允许您添加新的配置文件或编辑现有的配置文件。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0406/f0adbeb00e666b7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0406/f0adbeb00e666b7.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">5. 当您添加或编辑个人资料时，您可以添加个人资料图片。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0406/724cda4a68724cf.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0406/724cda4a68724cf.jpg" referrerpolicy="no-referrer"></a></p></blockquote>
