<strong>在昨晚 11 点召开的 RazerCon2021 发布会上，雷蛇带来了 Razer Kraken V3 雷蛇北海巨妖 V3 系列，包括标准版、专业版和超感版本，售价分别为 799 元/ 999 元/ 1599元。</strong>与上一代产品相比，V3 看起来没有那么笨重和花哨。Razer 的许多耳机都偏向于看起来更像罩耳式耳机，这个也不例外。<br>
 <p style="text-align: left;">它的一些功能是从其他 Razer 耳机中借用的。每款 Kraken V3 耳机都有 Razer 的 50mm TriForce 驱动单元，该驱动单元起源于 BlackShark V2 系列，而在 Kraken V3 的三个型号中，有两个型号的 Nari 耳机带有 HyperSense 触觉反馈。</p><p style="text-align: left;"><strong>● <a href="http://cn.razerzone.com/gaming-audio/razer-kraken-v3" target="_blank">Razer Kraken V3 雷蛇北海巨妖 V3</a></strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1022/652583702b547e0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1022/652583702b547e0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">雷蛇北海巨妖 V3 是一款采用 Razer Chroma™ 雷蛇幻彩 RGB 技术的电脑游戏耳麦，提供与游戏和设备融合的动态灯光效果，让你享受缤纷的沉浸感。借助高级驱动单元和环绕声音效提供栩栩如生的音效，让你从听觉和视觉两个层面轻松进入忘我状态。</p><p style="text-align: left;"><strong>● <a href="http://cn.razerzone.com/gaming-audio/razer-kraken-v3-pro" target="_blank">RAZER KRAKEN V3 PRO 雷蛇北海巨妖 V3 专业版</a></strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1022/0e51e0a595ba68a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1022/0e51e0a595ba68a.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">为了获得出色的沉浸感，人们的感受必须超越视觉和听觉。Razer Kraken V3 Pro 雷蛇北海巨妖 V3 专业版是一款采用了 Razer HyperSense 超感技术的触觉反馈无线电脑游戏耳麦，为你的游戏体验解锁第三种感观感受。真实感知所听到的内容，在游戏时享受前所未有的强烈感或真实度。</p><p style="text-align: left;"><strong>● <a href="http://cn.razerzone.com/gaming-audio/razer-kraken-v3-hypersense" target="_blank">RAZER KRAKEN V3 HYPERSENSE 雷蛇北海巨妖 V3 超感版</a></strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1022/527b4f67f5fc1c4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1022/527b4f67f5fc1c4.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Razer Kraken V3 HyperSense 雷蛇北海巨妖 V3 超感版是一款采用触觉技术的电脑游戏耳麦，可为你带来全新的沉浸感。现在，是时候拥有真正的多感官游戏体验了，尽情感受你听到的每一个声音。</p>
