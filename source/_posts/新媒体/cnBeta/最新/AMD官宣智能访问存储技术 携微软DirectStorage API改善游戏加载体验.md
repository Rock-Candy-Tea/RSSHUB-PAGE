在 Computex 2022 主题演讲期间，<strong>AMD 不仅隆重宣布了基于 Zen 4 架构的全新一代锐龙 7000 系列台式处理器 + Socket AM5（LGA 1718）平台芯片组，还介绍了一同而来的“智能访问存储”（SmartAccess Storage）技术。</strong>作为该公司 Smart 技术套件中的一员，此前我们已经了解过“智能访问显存”（Smart Access Memory）和智能调节游戏本 CPU / GPU 功耗的 SmartShift 技术。<br>
<p><a href="https://static.cnbetacdn.com/article/2022/0523/000e7270c09caa5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0523/000e7270c09caa5.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>顾名思义，SAS 致力于提升系统的存储访问体验。在早前爆料中，我们曾推测其基于<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>的 DirectStorage API 来实现。</p><p>在今日的台北电脑展主题演讲期间，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 官方证实了大家的这一猜测，并补充道 —— 借助这项技术，GPU 能够极大改善相关资源的解压缩效率（提升纹理流效率 / 减少游戏加载等待时间）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0523/f334421381d0f13.gif" alt="3.gif" referrerpolicy="no-referrer"></p><p><strong>以下是 AMD 对于 SmartAccess Storage 技术的详细解释：</strong></p><blockquote><p>● 该技术能够减少玩家呆坐在屏幕前等待游戏加载完成所耗费的时间。</p><p>● 传统操作需要大量 CPU 算力来解压并传输游戏数据，结果造成高延迟和占用大量系统资源。</p><p>● 为帮助绕过该瓶颈，AMD 决意打造一套支持 Microsoft DirectStorage 的 SAS 技术。</p></blockquote><p>最终借助 SAM 智能访问显存（Smart Access Memory）、崭新的 AMD 平台技术和 Radeon GPU，我们可以高效地加速游戏资源的解压缩工作，进而改善游戏加载时间和纹理流的综合体验。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1272569.htm" target="_blank">AMD官宣AM5平台：搭配X670E、X670和B650三套芯片组</a></p></div>
