<strong>特斯拉今天宣布推出“Plaid Track Mode”，这是“Track Mode”（赛道模式）的升级版本，将 Model S Plaid 的赛道性能推到一个全新的水平。</strong>特斯拉早在 2018 年就为 Model 3 Performance 推出了“赛道模式”，对动态控制进行了微调，以提高赛道性能。2020 年，特斯拉发布了“Track Mode V2”，进一步提高性能。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0107/bba2fb94a9e1b46.webp" alt="suau0dv1.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">而伴随着新的 Model S Plaid（目前特斯拉性能最高的汽车），因此特斯拉现在发布了全新的“Plaid Track Mode”，启用了稳定控制、操控平衡和再生制动的所有功能。特斯拉在更新日志中表示：</p><p style="text-align: left;">本周，“Plaid Track Mode”正在向北<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://mideajiadian.jd.com/" target="_blank">美的</a> Model S Plaid 车辆推出。对于“Plaid Track Mode”，我们的目标很简单：在德国纽博格林赛道上实现量产电动车的最快单圈时间，并允许对稳定性控制、操控平衡和再生制动进行单独调整，使驾驶员在赛道上对车辆的控制有更大的权限。</p>
