<strong>在周四的一篇博客文章中，微软详细介绍了 Excel for Web 端的新“条件格式设置”体验。</strong>作为一款功能齐全的现代电子表格应用程序，其旨在为用户的日常工作提供有力的支撑。而新推出的“Conditional Formatting”功能，又能够让用户更加轻松地管理、编辑和创建基于特定标准的格式规则，以便用户从数据中快速发现和洞察趋势。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0121/6f927868e851c79.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0121/6f927868e851c79.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p><strong>以下是 Excel for Web 支持的规则类型：</strong></p><blockquote><p>● Greater than or equals to（ > 或 =）</p><p>● Less than or equals to（< 或 =）</p><p>● Not containing（不含）</p><p>● Beginning with（以 … 开始）</p><p>● Ending with（结束于 …）</p><p>● Blanks / No blanks（留空 / 不留空）</p><p>● Errors / No errors（有错 / 无错）</p><p>● Formula（特定公式）</p></blockquote><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=320177522&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">Conditional Formatting v2 HQ（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzIwMTc3NTIyLnNodG1s.html" target="_self">via</a>）</p><p>用户现可在 Excel for Web 网页版中指定逻辑公式的格式标准，此规则类型赋予了额外的灵活性。</p><p><img src="https://static.cnbetacdn.com/article/2022/0121/94039bdcb8bdcbe.png" alt="1.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://techcommunity.microsoft.com/t5/excel-blog/new-conditional-formatting-experience-in-excel-for-the-web/ba-p/3066188" target="_self">Excel Blog</a>）</p>
