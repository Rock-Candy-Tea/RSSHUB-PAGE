面向 Windows Insider 的 Dev 频道用户发布 Windows 10 Build 21354 之后，微软今天还放出了该预览版更新的 ISO 镜像。Windows 10 CO_RELEASE Build 21354 带来了多项功能改进和 BUG 修复，此外还将 MSPaint 和 Snapping Tool 从系统中剥离出来，通过 Microsoft Store 进行更新。<br>
<p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0408/e10c440803f6e45.jpg" alt="QQ截图20210408110914.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">Windows 10 21H2 Build 21354 的 ISO 镜像目前已经开放下载，Dev 频道用户可以使用该镜像进行纯净安装。要想下载 Windows 10 Build 21354 的 ISO 文件，你需要注册并加入到 Windows Insider 计划中。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0408/15eef94caadf2a5.jpg" alt="pyxjj83v.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">下载：<a href="https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewiso" target="_blank">https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewiso</a></p>
