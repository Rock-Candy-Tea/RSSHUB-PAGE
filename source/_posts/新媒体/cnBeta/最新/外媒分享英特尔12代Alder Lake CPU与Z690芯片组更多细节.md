PCinq 刚刚分享了有关英特尔即将发布的 12 代 Alder Lake CPU 与 Z690 芯片组的详情，<strong>虽然提到的处理器是一枚机密的样品（非 ES），但 BIOS 信息表明它已经能够显示零售型号名称（而不是“Intel 0000”）。</strong>首先，Alder Lake CPU 将具有 16 条 PCIe 5.0 通道（支持 1×16 或 2×8 拆分），但似乎只有 Intel SSD 能够用上。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0910/da027b7fa41b537.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0910/da027b7fa41b537.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/intel-alder-lake-cpu-leaked-by-the-tech-paparazzi-z690-chipset-and-lga-1700-detailed/" target="_self">Videocardz</a>）</p><p>其次，Alder Lake CPU 原生支持 DDR4 / DDR5 双通道内存（每个通道支持两条 DIMM 插槽）。然而在核显方面，该公司依然沿用了 UHD Graphics 的命名方式，而不是搬出 Xe 的大名。</p><p>此外 Alder Lake CPU 为 PCIe 4.0 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 或<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>傲腾（Optane）存储产品提供的 x4 通道，支持三路 HDMI / DisplayPort 视频输出，另有通向芯片组的 DMI 4.0 x8 通道。</p><p>据说芯片组这边又分出了多达 12 条 PCIe 4.0 + 16 条 PCIe 3.0 通道，辅以 6 个 SATA 3、4 个 USB 3.2 Gen 2×2、10 个 USB 3.2 Gen 2×1、10 个 USB 3.2 Gen 1×1 和 16 个 USB 2.0 端口。</p><p><a href="https://static.cnbetacdn.com/article/2021/0910/5913c86a35756bf.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0910/5913c86a35756bf.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>网络方面，英特尔集成了 5G base-T MAC / PHY 以太网（10 / 100 / 1000 MAC），并通过 PCIe x1 和 SMBus 连接。</p><p>除了通过芯片组提供支持的傲腾内存 / Intel Smart Audio 技术支持，还有面向 PCIe 存储的 Intel RAID 和 Intel Wi-Fi 6E / 7（AX211）。</p><p>此外对 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 操作系统至关重要的英特尔平台信任技术（PTT）也没有缺席。</p><p><a href="https://static.cnbetacdn.com/article/2021/0910/3b7caa6d8b6cab3.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0910/3b7caa6d8b6cab3.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：<a href="https://www.pcinq.com/news/cpus/intel-alder-lake-s-ddr5-platform/" target="_self">PCinq</a>）</p><p>由英特尔官方披露的信息可知，Alder Lake 采用了支持超线程的高性能 P 核（Golden Cove）+ 节能型 E 核（Gracemont）的混合式设计，辅以“Intel 7”制程接电。</p><p>以 8P + 8E 的 SKU 为例，其组成了独特的 16 核 / 24 线程。集成 96 个执行单元（EU）的 Xe 核显，宣称 IPC 性能较 Rocket Lake 提升 19% 。插槽为 LGA 1700，功耗从 9W 至 125W 。</p><p>在率先为消费级市场带来 DDR5 / PCIe 5.0 支持的同时，英特尔还下放了 Thread Director 等新技术。</p><p><a href="https://static.cnbetacdn.com/article/2021/0910/924e4ce8f41118a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0910/924e4ce8f41118a.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/intel-alder-lake-cpu-leaked-by-the-tech-paparazzi-z690-chipset-and-lga-1700-detailed/" target="_self">WCCFTech</a>）</p><p>此前，Rocket Lake 已经带来了较 Skylake 达 19% 的 IPC 性能提升。虽然 Skylake 已经沿用了将近十年，但 Alder Lake 还是有望与 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙竞品一争高下。</p><p>不过考虑到 AMD 主流产品线拥有更高的核心数（16C / 32T），英特尔或需在多线程性能之外另辟蹊径。</p><p>值得一提的是，Alder Lake 配备了高达 30MB 的 non-inclusive LL Cache 缓存，支持 DDR5-4800 / LP5-5200、以及 DDR4-3200 / LP4x-4266 内存。</p><p>带宽方面，PCIe 5.0 较 PCIe 4.0 翻倍，而 Alder Lake 也将带来多达 16 条 PCIe 5.0 通道（64 GB/s）。此外得益于模块化的设计，Alder Lake 将能够相当灵活、且完全可扩展。</p><p>据悉，Compute Fabric 互连的带宽为 1000 GB/s，IO Fabric 的带宽为 64 GB/s、内存子系统支持 204 GB/s，且能够根据 SoC 的需要来调节内存频率和功耗。</p><p>最后，英特尔 Alder Lake 新品将于 2021 年晚些时候到来，并有望从 AMD 手中拿回移动性能和效能的王座。与此同时，AMD 也会拿出独门秘技，甚至<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a> Apple Silicon 也会出来搅局。</p>
