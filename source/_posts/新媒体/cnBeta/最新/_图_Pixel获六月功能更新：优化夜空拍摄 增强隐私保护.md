面向 Pixel 系列手机，Google今天放出了本月的 <a href="https://blog.google/products/pixel/feature-drop-starry-videos-pride-wallpapers/" target="_blank">Feature Drop</a>。在让 Pixel 更智能更充满乐趣的同时，本次更新进一步提升了安全体验，允许用户在拍摄夜空的时候制作短视频。<br>
 <p style="text-align: left;"><strong>● 拍摄夜空</strong></p><p style="text-align: left;">Pixel 用户非常喜欢在夜视（Night Sight）中使用天体摄影功能延时摄影模式，在本月的 Feature Drop 中变得更好了。现在，你可以在同一曝光期间创建星星在天空中移动的视频。一旦你在Night Sight中拍摄了一张照片，照片和视频都会被保存在你的相机胶卷中。试着等待更长的时间，在视频中捕捉更多的星星。这项功能适用于Pixel 4和更新的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>，你可以在<a href="https://g.co/pixel/astrophotography" target="_blank">g.co/pixel/astrophotography</a>网站上了解更多。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/06/c71e4e73adaa3b2.gif" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/06/c71e4e73adaa3b2.gif" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>● 庆祝傲娇月</strong></p><p style="text-align: left;">三款大胆、欢快的壁纸设计是由阿什顿-阿茨(Ashton Attzs)专门为Pixel创作的。查看由其他LGBTQ+艺术家和YouTube创作者创作的以骄傲为主题的新铃声和通知。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0609/4307ccb78deeda4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0609/4307ccb78deeda4.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0609/1b56160779cd7ba.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0609/1b56160779cd7ba.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0609/60997896a88ecb5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0609/60997896a88ecb5.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>● 更隐私、更安全</strong></p><p style="text-align: left;">在最新的 Pixel 更新中获得了 Locked Folder 功能，允许用户在主相册之外对特定的图片和视频进行加密隐藏。在 Locked Folder 页面中，Google表示该功能目前仅适用于 Pixel 系列设备。不过在 I/O 大会上，Google承诺将会在年内会向其他 Android 设备开放。</p><p style="text-align: left;">在 Pixel 3 及更高型号中，用户可以通过 Library > Utilities > Locked Folder 来启用 Locked Folder 功能。一旦设置好了，你就可以在这里看到你的秘密东西。你能够将现有的照片移动到锁定的文件夹，这将从你的常规库和图片可能出现的任何现有的备忘录中删除它们。值得注意的是，根据Google的说法，你不能备份锁定文件夹中的任何照片或视频，所以在隐私和便利之间有一定的权衡。如果你不再需要保持隐私，可以将锁定文件夹中的任何东西移出。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/06/62d88d3d6f3812b.gif" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/06/62d88d3d6f3812b.gif" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Google在支持页面中还表示，用户可以在相机应用中进行设置，将拍摄好的图像直接保存到 Locked Folder 文件夹中，使它们永远不会进入你的正常资料库。这可能是有用的，例如，你在一个抗议活动中，想确保你拍的任何照片不会被其他可能拿到你手机的人看到。</p>
