<strong>有消息称苹果计划在今年 3/4 月推出第三代 iPhone SE。现在爆料人士 Xleaks7 分享了 iPhone SE 3 的高清渲染图。这</strong>些渲染图显示，即将推出的 iPhone SE 3 将采用与 iPhone XR 类似的设计，具有圆角、细长的边框和宽阔的刘海，以容纳 Face ID 和自拍相机传感器。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0114/02040b40770da24.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">渲染图显示，<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> SE 3 将有类似 iPhone XR 的圆形边缘。我们还可以看到正面传统的、更宽的刘海，用于 Face ID 和其他传感器，与 iPhone 13 系列上的小刘海不同。如果设计结果是准确的，第三代机型将拥有迄今为止我们在 SE 系列上看到的最薄的边框。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0114/e0791049a37585c.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">根据泄露的渲染图，iPhone SE 3 的背板将容纳一个后置摄像头、LED 闪光灯和<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>标志。它似乎有一个玻璃背面，这应该可以实现无线充电。电源按钮和SIM 卡托盘可以在右边的脊柱上看到。而音量摇杆和警报滑块则在左侧边缘。闪电接口将在底部。渲染图显示该智能<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>为白色，但在发布时应该还有其他选择。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0114/4137ad109757d9a.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">有趣的是，报告称该智能手机的尺寸为 138.4 x 67.3 x 7.3 毫米（包括摄像头凸起的厚度为 8.2 毫米），显示屏尺寸为 131.3 x 60.2 毫米。根据该报告，这将意味着 5.69 英寸的显示屏。在过去，苹果已经制造了 4.7英寸、5.4英寸、5.8英寸、6.1英寸的显示屏，所以如果Xleaks7的信息确实正确，那么iPhone SE 3的5.7英寸显示屏将是苹果的第一次。</p>
