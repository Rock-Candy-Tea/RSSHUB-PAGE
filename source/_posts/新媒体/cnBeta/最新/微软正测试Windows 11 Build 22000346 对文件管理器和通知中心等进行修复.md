微软正为 Windows 11 筹备另一个累积更新，将重点解决新系统中的几个错误。面向 Beta 和 Release Preview 频道 Windows Insider 项目成员，<strong>微软正在测试 Windows 11 Build 22000.346，对文件资源管理器、新通知中心等进行修复。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1113/0e220781a3ecc5a.webp" alt="yplhetyf.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Windows 11 文件资源管理器在某些设备上存在错误。例如当打开右键菜单的时候，会减慢性能。此外还有用户报告称当用户在文件资源管理器窗口、桌面上右键一个文件或者文件夹的时候，会出现右键菜单滞后的情况。</p><p style="text-align: left;">在Windows 11中，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>已经改变了右键菜单的设计，以解决长期以来的性能问题，并优先考虑常用的选项，如复制、剪切、重命名。不幸的是，此举导致了各种与右键菜单相关的问题，而且似乎也影响了整个操作系统的可靠性。</p><p style="text-align: left;">值得庆幸的是，右键菜单将在未来的版本中得到修复，微软也正在为资源管理器进行其他错误修复。Windows 11 Build 22000.346，现在可以在Beta/Release Preview中使用，最终修复了一个影响文件资源管理器和桌面右键菜单中依赖右键菜单的快捷标签的应用程序的错误。另一个导致文件资源管理器在你关闭窗口后停止工作的错误已经被修复。</p><p style="text-align: left;">除了文件资源管理器，Build 22000.346还包含操作系统其他区域的错误修复，包括设置应用中的Windows Update历史页面。对于任务栏，只有一个错误的修复，即当你悬停在任务栏的图标上时，会导致闪烁。</p>
