在本月的补丁星期二活动日中，微软发布了适用于 Windows 10 21H2、21H1 和 20H2 功能更新的累积更新 KB5010342，安装后版本号升至 Build 1904x.1526。除了通过 Windows Update 方式更新之外，用户还可以通过 Microsoft Update Catalog 网站进行下载离线安装包。<br>
<p>Windows 10 KB5010342 直接下载链接: <a href="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5010342" target="_blank" rel="nofollow noopener noreferrer">64-bit and 32-bit (x86)</a>.</p><p><img src="https://static.cnbetacdn.com/article/2022/0209/f6325c18131a236.webp" alt="yf88egf3.webp" referrerpolicy="no-referrer"></p><p>KB5010342 是以安全为主的更新，但也引入了一些非安全修复。其中一项是通过 Microsoft Edge 浏览器播放多媒体内容的时候能够让环绕立体声顺利工作。此外还修复了当你试图连接蓝牙设备时，会导致蓝牙设备停止工作的问题。由于上个月的可选更新相当巨大，所以这次更新中有许多非安全问题的修复。</p><p>在安装该累积更新之后，Windows 10 21H2 功能更新会升级至 Build 19044.1526；Windows 10 Windows 10 21H1 功能更新会升级至 Build 19043.1526；而 Windows 10 20H2 功能更新会升级至 Build 19042.1526.</p><p>微软已经结束了对 Windows 10 Version 2004 的支持，所以你不会得到这个更新，除非你有企业版或<a data-link="1" href="https://c.duomai.com/track.php?k=yUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmYiJ05WZkVHdzZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyUiR" target="_blank">教育</a>版的操作系统。如果你使用的是 Windows 11，你的设备就会得到 KB5010386 累积更新。</p><p><img src="https://static.cnbetacdn.com/article/2022/0209/ee9d730d48cb78e.webp" alt="QQ截图20220209075245.webp" referrerpolicy="no-referrer"></p><p>根据更新日志，微软已经修复了几个安全问题和一个错误，即如果检测到“SamAccountName and UserAccountControl”属性，一个名为“Lightweight Directory Access Protocol”的功能可能无法执行其操作。</p><p>同样，微软正在引入一项新的功能，将在 News and interests 信息流中提供对微软 Edge 配置文件的直接访问。你总是可以直接从 Edge 管理配置文件，但对于那些喜欢通过 Windows 10 的新闻和兴趣启动它的人来说，会有一个新的快捷方式。</p><p>微软对Windows 10进行了修改，以解决在2004版或更高版本上使用USB打印时的打印问题。另一个错误是，如果你使用中文IME输入文字，应用程序就会停止工作，这个问题已经得到了修复。</p>
