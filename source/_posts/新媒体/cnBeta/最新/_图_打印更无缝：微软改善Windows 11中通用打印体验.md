微软昨日透露，将会在 Windows 11 系统中改进打印体验。从下面的截图中可以看到，用户可以直接在设置应用中添加打印机。对于使用通用打印机驱动的打印机，微软增加了为打印作业添加密码的功能。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0701/da9b806303df53d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0701/da9b806303df53d.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0701/6d14cc02bc68a60.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0701/6d14cc02bc68a60.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">● 在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 系统中，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>添加了对 Print Support Application (PSA)的支持。Windows 提供了一个基于标准的 Mopria打印协议的收件箱打印机类驱动程序。这实现了简单和无缝的打印体验，用户无需安装额外的软件或自定义驱动程序来连接和打印他们经过 Mopria 认证的打印机。</p><p style="text-align: left;">● Windows 11 系统中支持扩展通用打印机驱动，打印机厂商或者 Managed Print Software (MPS)解决方案来提供额外的自定义功能和工作流程。这些 UWP 应用程序不是驱动程序，在用户的上下文中运行。当安装在用户的 Windows 11 设备上时，PSA 可以在后台运行，为相应的打印机处理打印作业，或者可以提供一个高级打印用户界面。</p><p style="text-align: left;">●  PSA 也将在消费者 Windows 11 设备上的商业通用打印服务配置之外可用。当家庭用户将打印机连接到家庭网络时，应用程序将从商店自动安装。</p><p style="text-align: left;">●  微软正在为Windows生态系统带来一些对互联网打印协议（IPP）的增强支持。自2018年以来，IPP已经在Windows 10中实施，用于网络打印机。在 Windows 11 系统中，微软正在增加 IPP over USB，以扩大使用 IPP 的硬件支持。为了支持企业环境中的 IPP 和 PSA，Windows 还支持 IPP 打印机的定向发现。</p>
