据外媒报道，5年来，SpaceX一直在成功地发射和着陆火箭，但其著名的猎鹰9号只是其宏伟计划的垫脚石。埃隆·马斯克领导的公司希望让前往月球及其他星球的深空旅行变得更加经济，这将需要比该公司自己的“重型猎鹰(Falcon Heavy)”更大、更强大的火箭才行。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0720/332083fb7bd658c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0720/332083fb7bd658c.jpg" alt="spacex-super-heavy-booster-1280x720.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">这一角色将由SpaceX的Starship来填补，日前，该公司向最终目标又迈出了一小步。据悉，它成功测试了最终将成为超重型(Super Heavy)助推器的东西。</p><p style="text-align: left;">超重型助推器是Starship系统的第一级，该系统将最终将把人类再次送上月球，至少是通过SpaceX的商业计划。由于该火箭比SpaceX迄今为止发射的任何火箭都要更大、更重，所以即使是一个小小的静态点火测试也可能会出现严重错误。幸运的是，一切都按计划进行。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0720/7e442659d042a72.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0720/7e442659d042a72.png" alt="Screenshot_2021-07-20 SpaceX Super Heavy booster first static test is an uneventful success(1).png" title="Screenshot_2021-07-20 SpaceX Super Heavy booster first static test is an uneventful success(1).png" style="width: 560px; height: 554px;" width="560" vspace="0" height="554" border="0" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">SpaceX和埃隆·马斯克都在Twitter上发布了为Starship设计的超重型助推器3(Super Heavy Booster 3)成功完成静态点火测试的消息。这只是第一步且很小，但它应该可以为将来的测试提供足够的数据。更重要的是，这次测试只启动了三台Raptor引擎，这只相当于Starship首次真正航行计划的1/10。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0720/4c9c5b5c3e7c87f.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0720/4c9c5b5c3e7c87f.png" alt="Screenshot_2021-07-20 SpaceX Super Heavy booster first static test is an uneventful success.png" title="Screenshot_2021-07-20 SpaceX Super Heavy booster first static test is an uneventful success.png" style="width: 540px; height: 210px;" width="540" vspace="0" height="210" border="0" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">SpaceX计划先在助推器3上测试9台Raptor引擎，然后再测试助推器4--这将是首个真正搭载Starship的超重型助推器。而其最终目标是使用30多台这样的引擎推动整个装置进入太空。如果这听起来有点夸张，不妨参考下猎鹰重型火箭，它的一级助推器已经使用了27台引擎。</p><p style="text-align: left;">跟竞争对手布兰森和贝佐斯不同的是，马斯克并不急于完成SpaceX的Starship梦想。毕竟，这个目标不仅仅是太空旅游它还将取决于这艘其能否成功、安全地将人类送入太空然后安全地返回地球以便再次发射。</p>
