在 Epic Games 诉苹果 App Store 垄断的案件打得火热的同时，外界也知晓了沃尔玛与 Epic 之间交换的机密电子邮件。<strong>TechSpot 指出，这家零售巨头有意推出名为 Project Storm 的云游戏服务，且有望将《堡垒之夜》引入这个尚未正式发布的新平台。</strong>此外，邮件中还描述了这项云游戏服务的工作方式，并分享了“主页”菜单 UI 的预览。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0505/9a303e470219f6e.jpg" referrerpolicy="no-referrer"></p><p>其实早在 2019 年，就有传闻称沃尔玛打算推出一项云游戏服务（恰逢 Google Stadia 官宣数日之后），结果一直沉寂至今。</p><p>有趣的是，在 Epic 诉<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>的案件中，我们竟然再次见到了 Project Storm 的身影。这部分信息来自于 Epic 与沃尔玛之间交换的电子邮件，其中包括了一份介绍所谓 Project Storm 平台的演示文稿。</p><p>在一张“我们如何构建”的 PPT 中，沃尔玛宣称其计划支持多款第三方启动器，包括 Uplay、Origin、Steam、Battle.net、Epic Games Store、以及 Bethesda Launcher 。</p><p>在另一张 PPT 中，我们还见到了 Project Storm 云游戏服务的主菜单界面，可知 UI 中包含了一个“我的游戏”和“最近玩过”的游戏列表，且玩家能够按类型选择或检索游戏。</p><p><img src="https://static.cnbetacdn.com/article/2021/0505/792b057f78a6665.jpg" referrerpolicy="no-referrer"></p><p>之所以将 Project Storm 项目给曝光，是因为沃尔玛将该平台描述为一个开放的生态系统、允许玩家下载游戏或通过云端串流在本机体验，而这正是 Epic 想要嘲讽苹果生态过于封闭且霸道的地方。</p><p>Epic Games 联合创始人 Mark Rein 在 2019 年 4 月的一封电子邮件中写道：“我用 Android 智能机 + <a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> 手柄进行了体验，感觉 Project Storm 与 PS4 的体验近似，且优于在 Android 或 iOS 上玩游戏”。</p><p>此外案件审理期间包曝光了 Mark Rein 与 Epic Games 高管团队其他成员的合照，以及一款能够将<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>与手柄固定到一起的背夹装置，感兴趣的消费者可用 2 美元在沃尔玛选购。</p><p>据说 Project Store 本计划在 2019 年开启 Beta 体验，但该服务最终未能顺利推进。即便有报道称沃尔玛已经拉拢了一些合作伙伴，但该项目还是在去年 COVID-19 大流行开始时被搁置了。</p>
