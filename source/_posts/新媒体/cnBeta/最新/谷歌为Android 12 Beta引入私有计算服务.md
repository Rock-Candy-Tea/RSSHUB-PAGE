<strong>在近日的一篇博客文章中，谷歌 Android 与 Play 安全隐私产品副总裁 Suzanne Frey，很高兴地向广大 Android 用户宣布了一项“私有计算服务”。</strong>据悉，Private Compute Core 旨在提供一个开源、且安全的环境，与其它操作系统 / 应用程序隔离，在用户设备与云端之间架起一座隐私保护的桥梁。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0910/f76206ffbdb24ed.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://security.googleblog.com/2021/09/introducing-androids-private-compute.html" target="_self">Google Security Blog</a>）</p><p>在 Android 的每个新版本中，谷歌都会努力带来 Private Compute Core 的更多隐私保护特性。</p><p>而今天，我们就看到了包括 Live Caption、Now Playing 和 Smart Replay 在内的三方面改进。</p><blockquote><p>● 首先，Live Caption 将能够利用本机设备的语音识别功能，为任何媒体添加字幕注释；</p><p>● 其次，Now Playing 功能可识别附近正在播放的音乐、并在设备锁屏界面上显示相关信息（比如歌曲 / 艺术家名称）；</p><p>● 然后，Smart Replay 可更具消息应用程序中的内容，给予智能的回复内容建议。</p></blockquote><p>在私有核心计算功能的加持下，用户不仅能够保证设备上保存的信息的私密性，还可让设备在调用云端功能的同时（比如下载新的歌曲分类/ 语音识别模型），不至于付出隐私方面的代价。</p><blockquote><p>谷歌表示，Android 可确保在 Private Compute Core 中处理的敏感数据，不会在用户未采取任何措施的情况下分享给任何应用程序。</p><p>例如，在用户点击智能恢复之前，操作系统会将你的回复内容与输入法 / 当前 App 隐藏隔离开来。</p></blockquote><p>至于私有计算服务本身，谷歌表示他们会持续更新相关机器学习模型，然后通过私有途径获取这些更新。</p><p>Android 操作系统将阻止 Compute Core 内的任何功能直接访问网络，而是只能通过一组具有针对性的开源 API 与私有计算服务进行通讯。</p><p>此外私有计算服务会剥离可用于识别用户身份的信息，以及联合学习、联合分析、私有信息检索等技术。</p><p>最后，谷歌承诺将公开发布私有计算服务的源代码，以供安全研究人员和 Google 以外的其它团队对其进行审核。</p>
