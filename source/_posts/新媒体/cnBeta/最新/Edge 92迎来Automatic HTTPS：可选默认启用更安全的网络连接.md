自 Mozilla 为 Firefox 83 引入可选的“仅 HTTPS 模式”、以及 Chrome 90 优先为桌面和 Android 客户端选择更安全的 HTTPS 连接之后，<strong>微软也终于在 Edge 92 中，为这款采用 Chromium 内核的浏览器引入了对“Automatic HTTPS”功能的支持。</strong><br>
<p><a href="https://static.cnbetacdn.com/article/2021/0504/2060e901057ccf8.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0504/2060e901057ccf8.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>感兴趣的朋友，可在 Edge Canary 92 中启用 Automatic HTTPS，方法是在地址栏中输入 edge://settings/privacy 并跳转。</p><p>如果使用的是 Edge Canary 92.0.877.0 及更高版本，请访问 edge://flags/#edge-automatic-https 。</p><p>找到“设置 -> 隐私、搜索和服务”项，启用 Automatic HTTPS，然后重启浏览器。</p><p><a href="https://static.cnbetacdn.com/article/2021/0504/76adbf9d9fe06f3.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0504/76adbf9d9fe06f3.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>根据需要用户可选择仅在可能 HTTPS 的网站到自动切换，或者强制始终启用 HTTPS 连接（或更频繁地遇到相关错误）。权衡安全与兼容性，建议大家还是使用默认的第一种设置。</p><p>微软表示，从 Edge 92 开始，广大用户能够在支持更安全的 HTTPS 协议的域上，将连接从 HTTP 升级到 HTTPS 。通过这方面的配置，Edge 将有助于保护客户免受中间人攻击。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1055823.htm" target="_blank">[图]Firefox 83引入仅HTTPS模式：为你提供安全、加密的连接</a></p><p><a href="https://www.cnbeta.com/articles/tech/1096245.htm" target="_blank">Chrome浏览器将为用户默认添加https前缀</a></p></div>
