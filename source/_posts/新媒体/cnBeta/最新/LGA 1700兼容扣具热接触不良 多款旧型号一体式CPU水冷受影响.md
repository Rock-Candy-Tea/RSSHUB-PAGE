由于采用了特殊的混合式核心架构设计、且微调了 Z 轴高度，早在几周前，就有报道称旧款一体式 CPU 水冷和英特尔 12 代 Alder Lake 处理器的安装座兼容性不佳。<strong>即使厂商向老用户开放了从 LGA 1200 到 LGA 1700 的兼容扣具申请通道，实际搭配使用时还是很容易遇到压力分布 / 热接触不良等问题。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1113/3c490cb013ed9bd.png" alt="1.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/various-aio-cpu-coolers-tested-with-intels-alder-lake-lga-1700-cpus-older-models-show-insufficient-thermal-contact/" target="_self">WCCFTech</a>）</p><p>为了给出更加让人信服的数据，WCCFTech 特地对多个厂牌的 CPU 一体式水冷展开了一番实测。</p><p>正如 Igor's Lab 所指出的那样，与相对“正方”的 LGA 1200 相比、长条形的 LGA 1700（V0）插槽不仅采用了非对称式设计、且 Z 轴的高度也更低。</p><p>这意味着必须有适当的安装压力，才能让散热器底部和 CPU 顶盖（IHS）实现完全接触。</p><p><a href="https://static.cnbetacdn.com/article/2021/1113/99eefed3f799fcd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1113/99eefed3f799fcd.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via Igor's Lab）</p><p>此前已有厂商在 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 / 线程撕裂者 CPU 上使用了更大的散热器底板，以同 IHS 保持适当的接触，但通常只有高端和较新的 AIO 水冷型号才会这么做。</p><p>对于那些仍在使用圆底 AIO 水冷的用户来说，很可能难以维持所需的压力分布，结果导致散热性能严重偏离预期。</p><p>如图所示，有热心爆料人向 WCCFTech 提供了多款 AIO 水冷散热器的安装前后对比图片，充分展示了它们与 Alder Lake 台式处理器的接触情况。</p><p><a href="https://static.cnbetacdn.com/article/2021/1113/a06dce61d2c0dfd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1113/a06dce61d2c0dfd.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>首先充美商<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://mall.jd.com/index-1000000697.html" target="_blank">海盗船</a>（Corsair）的 H150i Pro 开始说起，该散热器带有一款可调节的 LGA 115x / 1200 安装扣具，并且能够兼容 LGA 1700 插槽。但很遗憾的是，它还是无法确保与新 CPU 的完全接触。</p><p>即使 H150i Pro 确实在 CPU 中心位置接触良好，但还是像另外两款<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://msi-pc.jd.com/" target="_blank">微星</a>（<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://msi-pc.jd.com/" target="_blank">MSI</a>）水冷 —— 360R V2 和 P360 / C360 系列 —— 一样，留下了大片空余。</p><p>接下来是 AORUS Waterforce X360（水雕），它的接触较 H150i PRO 更差，中间部位几乎与 CPU 顶盖没有接触。但更糟糕的还是<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://asus.jd.com/" target="_blank">华硕</a> ROG RYUJIN（龙神）360 。</p><p>散热器在芯片中间部分留出了很大的接触间隙，由于热交互性能受限，它会导致热量在 IHS 和水冷基板间堆积，结果就是处理器一直保持较高的运行温度。</p>
