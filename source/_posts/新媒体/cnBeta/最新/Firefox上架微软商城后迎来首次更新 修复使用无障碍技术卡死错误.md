Firefox 日前正式上架 Windows 11 系统的 Microsoft Store，今天 Mozilla 在商城上推出了首个更新。<strong>今天发布的 Firefox 94.0.2 为 Windows 11 系统带来了一项改进，它解决了一个导致浏览器在使用某些无障碍技术时卡死的错误。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1119/41e301886c91170.webp" alt="ngk1z585.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Firefox 94.0.2 更新幅度并不大，其他方面也只是常规的浏览器优化。本月早些时候，Mozilla 在 Microsoft Store 推出了 Firefox 浏览器，该公司赞扬了<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>对商店体验所做的改进。</p><p style="text-align: left;">在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 首次亮相后，该软件巨头还允许在该平台发布的浏览器使用自己的引擎。换句话说，在微软商店发布的浏览器不必使用原生的 Windows 引擎，而是使用它们自己的引擎，这意味着应用程序方面需要的改动更少。</p>
