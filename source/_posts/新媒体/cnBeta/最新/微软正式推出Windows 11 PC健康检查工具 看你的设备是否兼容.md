为了方便 Windows 10 老用户评估当前设备是否符合可升级至新一代 Windows 11 操作系统的最低条件，微软过去三个月一直在努力测试所谓的“PC 健康检查工具”。<strong>在经历了广泛的验证之后，微软现在终于拿掉了“测试版”的头衔，现可供任何有需要的客户使用这款免费工具。</strong><br>
<p><img src="https://static.cnbetacdn.com/article/2021/0921/f3002c57515b7fc.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p><a href="https://www.windowslatest.com/2021/09/21/windows-11-compatibility-pc-health-check-returns-for-everyone/" target="_self">Windows Latest</a> 指出：<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>于 6 月最后一周推出了《PC Health Check Tool》，但由于一系列问题反馈，该工具还是很快被从官网撤下。</p><p>接着微软变得有些谨慎，先是将修改版《PC 健康检查工具》推给 Windows 预览体验成员进行测试，然后再推向更广大的非预览体验成员。</p><p><img src="https://static.cnbetacdn.com/article/2021/0921/127ce5389038d8a.gif" alt="Introducing Windows 11.gif" referrerpolicy="no-referrer"></p><p>周末的时候，我们发现修订版《PC 健康检查工具》已在微软官网上正式上线。在此之前，该公司仅将之列作“即将推出”。<strong>但现在，你可通过以下地址进行获取：</strong></p><blockquote><p><a href="https://download.microsoft.com/download/1/5/e/15eb8516-6a35-4c70-8897-6cb4b5b4efd6/3.0/x64/WindowsPCHealthCheckSetup.msi" _src="https://download.microsoft.com/download/1/5/e/15eb8516-6a35-4c70-8897-6cb4b5b4efd6/3.0/x64/WindowsPCHealthCheckSetup.msi" target="_blank">https://download.microsoft.com/download/1/5/e/15eb8516-6a35-4c70-8897-6cb4b5b4efd6/3.0/x64/WindowsPCHealthCheckSetup.</a><a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://msi-pc.jd.com/" target="_blank">MSI</a></p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0921/6ae5cbfd1edf178.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>通过该工具，用户能够确认他们当前的桌面配置是否足以支持新一代 Windows 11 操作系统的运行。</p><p>若设备不符合要求，微软也不会强制 Windows 10 老用户在 2025 年之前更换升级。</p><p>至于通过“官方漏洞”安装运行 Windows 11 的用户，微软也敬告风险自负。</p><p><img src="https://static.cnbetacdn.com/article/2021/0921/74fe3f5fd487dc9.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>据悉，从 Windows 11 开始，微软将逐步停止对不兼容 TPM 2.0 可信任平台模块的旧设备的支持。</p><p>与 Windows 10 操作系统不同，Windows 11 官宣最低需要 8 代<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>酷睿、以及 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 2000 系列处理器。</p><blockquote><p>目前已知的是，英特尔 7 代酷睿 X 和至强 W 系列 CPU、以及搭载酷睿 i7-7820HQ 的 <a data-link="1" href="https://microsoft.pvxt.net/9W473" target="_blank">Surface</a> Studio，还是能够用上 Windows 11 的。</p><p>此外 Windows 11 还需要可信任平台模块（TPM）、安全启动（Secure Boot）、4GB 内存 + 64GB 硬盘、以及图形硬件等最低要求。</p></blockquote><p>虽然微软最近也重新审视了某些旧处理器的性能，但经过广泛的测试验证，如果你不想遭遇蓝屏死机（BSOD）或驱动程序兼容性问题，还请不要强制安装。</p><p>当然，如果只是手痒想尝个鲜的话，还是可以试着通过 ISO 镜像（或媒体创建工具），在不受支持的硬件上安装一下 Windows 11 的，但别指望得到官方的定期更新与相关支持。</p>
