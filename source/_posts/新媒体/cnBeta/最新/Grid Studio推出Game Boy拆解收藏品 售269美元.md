Grid Studio 以将电子产品拆解、并通过画框来艺术呈现的工作，而被广大爱好者所熟知。<strong>近日，这家工作室又将目光瞄向了在游戏机行业极具历史纪念意义的 Game Boy，且售价高达 269 美元（约 1700 RMB）。</strong>不过由于品相良好的 Game Boy 掌机的存世数量相对有限，Grid Studio 也不得不每售出 100 件、就将价格上调 10 美元。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0331/0b80158b2978d88.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Grid Studio <a href="https://gridstudio.cc/products/game-boy" target="_self">官网</a>）</p><p>BGR 编辑 Christian de Looper 透露，其曾在初代 Game Boy 上开启了成为“宝可梦大师”的奇妙旅程、经常耗费数小时玩《俄罗斯方块》，因而在看到 Grid Game Boy 藏品项目时也是感慨万千。</p><p><img src="https://static.cnbetacdn.com/article/2022/0331/f00e0f39afcec19.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>作为一款经典的掌机，Game Boy 的整体外形与 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> 智能机相近，但屏幕只有 Apple Watch 智能手表差不多大小。</p><p><img src="https://static.cnbetacdn.com/article/2022/0331/5503c7e75229422.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>不过广大玩家来说，肯定很少有人会像 Grid Studio 这样深入观察 Game Boy，且对每一个内部组件烂熟于胸。</p><p><img src="https://static.cnbetacdn.com/article/2022/0331/50113281b7a1c50.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>如果你想要收藏这样一份童年回忆，现可直接通过 Grid Studio 官网下单。截止发稿时依然有货，价格为 269 美元（约 1706 RMB）。</p>
