在五月发布后，<strong>美商海盗船（Corsair）终于在本周，将 Voyager a1600 这款带有“AMD Advantage Edition”的游戏本推向了市场。</strong>即日起，感兴趣的消费者可通过官网商店或全球分销商渠道购买，官方指导价为 2699.99 美元（约合 1.84 万 RMB）。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0523/17c691a0e7da37c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0523/17c691a0e7da37c.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：Corsair <a href="https://corsair.com/voyager-a1600" target="_self">官网</a>）</p><p>配置方面，Voyager a1600 搭载了先进的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> CPU + GPU 硬件，并可智能调协工作性能。</p><p>值得一提的是，Corsair 还为该游戏本提供了 Elgato 软件解决方案，以营造独特的游戏 / 串流体验。</p><p><a href="https://static.cnbetacdn.com/article/2022/0523/0c95eb78d130ce1.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0523/0c95eb78d130ce1.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>即便采用 8C / 16T 的锐龙 R9-6800HS 移动处理器 + Radeon RX 6800M 独显，Voyager a1600 游戏本的机身厚度仍只有 19.8 mm 。</p><p>此外结合 Corsair Vengeance DDR5 SO-DIMM 内存、以及 Smart Access Memory 等 AMD 智能技术，玩家还可在高设置下更顺畅地体验游戏。</p><p><a href="https://static.cnbetacdn.com/article/2022/0819/48427726d02db69.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0819/48427726d02db69.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>屏幕方面，该机采用了 16 英寸 @ QHD+（2560×1600）分辨率的 IPS 面板，且支持最高 240Hz 的 AMD FreeSync Premium 自适应刷新率技术。</p><p>为增强独家的 Elgato 内容创作体验，Corsair 还为 Voyager a1600 配备了一排十个易于访问的 S-key 快捷键，以便一键掌控游戏串流等体验。</p><p><a href="https://static.cnbetacdn.com/article/2022/0819/d573fa587675736.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0819/d573fa587675736.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>相关功能得到了 Elgato Stream Deck 软件的支持，内置的 1080p / 30fps 网络摄像头可清晰捕捉直播玩家的面部细节，辅以具有环境噪声消除功能的定向 4 阵列麦克风。</p><p>最后，Corsair Voyager a1600 游戏本首发时还提供了北美、英国、德语、法语和北欧键盘布局。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a
