作为 Windows 操作系统上一个最常被使用的功能，文件资源管理器的设计已经长期没有更新。尽管微软一度计划引入多标签页（选项卡）功能，但中途又选择了放弃这方面的工作。好消息是，不久前，有眼尖的人们发现微软再次重启了这项计划。<strong>而随着 Windows 11 Build 22579.100 开发预览版本的到来，我们欣喜于它已取得了不错的进展。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0323/39842995a8b145a.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0323/39842995a8b145a.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图自：<a href="https://www.reddit.com/r/Windows11/comments/tkcjnm/windows_11_dev_build_22579100_brings_back_tabs_in/" target="_self">Reddit</a> / Totally_Jay1）</p><p>从 Reddit 网友分享的首批截图来看，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>已经向首批 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 Insider 测试者（Dev 开发预览通道）推送集成 Sets 多标签页功能的文件资源管理器（作为最新累积更新的一部分）。</p><p>虽然这家软件巨头尚未披露将于何时正式为所有用户带来这项改进，但 Insider 测试者们发现它似乎是默认启用的（在 3 月初曝光时还没有这么做）。<br></p><p>回想几年前，微软曾在撤回 Sets 开发工作时表示 —— 文件资源管理器中的选项卡功能，仍有望在日后择机返回。</p><blockquote><p>我们正在让 Sets 离线，以将它打磨得更好。根据大家的反馈，我们将持续改进视觉设计、以及 <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a> 生产力套件和 Edge 浏览器等应用的 Sets 增强工作流集成。</p><p>如果你一直在测试 Sets，那你将在今日的编译版本中迎来告别。但在未来的 WIP 飞行测试中，Sets 仍有望归来。最后再次感谢大家的热心反馈。</p></blockquote>
