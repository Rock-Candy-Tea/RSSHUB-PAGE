Windows 11 即将迎来首个“功能更新”，且微软已向 Beta 测试通道提供了相对稳定、Bug 更少的 22H2 版本。<strong>想要尝鲜的朋友，可以申请加入 Insider 预览。不过经验不足的新手，可能苦恼于一个副作用，那就是位于桌面右下角的永久水印。</strong>好消息是，在 Build 22593 编译版本的发行说明中（现提供 Beta / Dev 测试体验），微软表示这并不会影响设备的激活状态。<br>
 <p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0407/8b33084beb6bd60.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">水印资料图（via <a href="https://www.neowin.net/news/psa-desktop-watermarks-in-preview-windows-11-builds-do-not-mean-your-license-is-void/" target="_self">Neowin</a>）</p><p>换言之，在安装了 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 Build 22593 之后，提示“评估副本”的永久桌面水印，并不意味着用户的 Windows 许可证变成了无效状态。</p><p>此外对于不符合运行 Windows 11 操作系统的最低硬件要求的旧设备用户，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>也将继续通过桌面水印的形式来推送通知。</p><p>当然，如果缺乏有效的许可密钥，系统还是会在桌面上显示“此 Windows 副本未激活”的提醒的。</p><p><a href="https://static.cnbetacdn.com/article/2022/0407/d71cebadccec55a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0407/d71cebadccec55a.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p><strong>不确定系统激活状态的 Windows 11 用户，可参考以下操作：</strong></p><blockquote><p>● 使用 WinKey + I 组合键，唤出 设置 应用程序。</p><p>● 转到 系统 → 激活 页面。</p><p>● 单击 激活状态，以查看 许可证 的详细信息。</p></blockquote><p>需要指出的是，Windows Insider 项目并不是免费获取 Windows 操作系统副本的一个有效途径。因为你仍需通过有效的激活密钥，以确保系统不会中断运行。</p><p>最后，与稳定版不同，测试者们无法无限期地使用 Windows Insider 预览，且 Dev、Beta 和 Release Preview 尝鲜通道都有确切的版本到期日设定，意味着你迟早还是要更新至较新或稳定版本。</p>
