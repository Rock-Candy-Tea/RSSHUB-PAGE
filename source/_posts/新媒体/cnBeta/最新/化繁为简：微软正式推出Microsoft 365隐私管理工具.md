随着每日生成的数据量越来越庞大，隐私保障需求也正变得愈加复杂，且不同国家 / 地区往往制定了不尽相同的法律条款。为适应新时代数据管理的动态变化，微软正试图用去中心化的身份概念来化解这方面的难题。<strong>今天，这家软件软件巨头正式宣布了面向 Microsoft 365 客户的全面型隐私管理工具。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1020/cb6fd13735435d8.gif" alt="Privacy Management.gif" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Microsoft <a href="https://www.microsoft.com/security/blog/2021/10/19/simplifying-the-complex-introducing-privacy-management-for-microsoft-365/" target="_self">官网</a>）</p><p>据悉，Microsoft 365 的隐私管理工具，本质上是一个专用的门户，旨在为组织机构提供端到端的可见性，以便其监测隐私数据的动向。</p><p>具体说来是，Privacy Management 利用了数据分类和用户映射，来智能识别跨多个数据存储库的关键隐私风险。基于此，组织机构将能够使用该工具来获得全面感知的汇总视图。</p><p>诸如数据传输、最小化、过度曝光、权限请求管理等隐私操作，都可以使用自动化模板来完成。当然，Microsoft 365 客户也能够创建自定义的策略。</p><p>对于 IT 管理员来说，还可通过 Privacy Management 来获取有关如何保障数据隐私的实施建议，并为数据所有者提供有关如何在不牺牲隐私的情况下、处理生产力场景中的数据的指导。</p><p>此外<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>正在与 OneTrust、Securiti.ai 和 WireWheel 等隐私软件公司合作，确保客户能够获得简化的主体请求流程。即使那些存储在 Microsoft 365 之外的数据，亦可纳入它的掌握。</p><p>最后，这家雷德蒙德软件巨头还谈到了隐私映射的其它方面，从数据合规性评估、一直到隐私管理门户的特定控制。</p><p>目前 Microsoft 365 隐私管理工具可供 Microsoft 365 和 <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a> 365 客户作为附加组件来使用。在 90 天的免费体验期过后，你可按需决定是否为它付费（<a href="https://www.microsoft.com/en-us/security/business/privacy/privacy-management-software" target="_self">传送门</a>）。</p>
