<strong>近日，有眼尖的网友在 <a href="https://www.cnbeta.com/cdn-cgi/l/email-protection" class="__cf_email__" data-cfemail="0845616463715f6971484067656d">[email protected]</a> 数据库中，发现了疑似 AMD 下一代锐龙 7000“Zen 4”台式处理器的身影。</strong>据悉，该项目旨在通过 BOINC 平台来汇聚世界各地的人们资源分享的计算资源，以创建银河系的高精度 3D 模型。然而最新的爆料，揭示了其中有混入 8 / 16 核心的不知名 AMD CPU 平台。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0108/021784461dc72cd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0108/021784461dc72cd.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>通常情况下，没人会过多留意 <a href="https://www.cnbeta.com/cdn-cgi/l/email-protection" class="__cf_email__" data-cfemail="511c383d3a2806302811193e3c34">[email protected]</a> 计算节点的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> / Intel 硬件。</p><p><img src="https://static.cnbetacdn.com/article/2022/0108/3c778cf3d0baef7.png" alt="2-0.png" referrerpolicy="no-referrer"></p><p>然而 BenchLeaks 发现，似乎有来自 AMD 工程实验室的某个人，在未发布的 Zen 4 锐龙 7000 系列 CPU 上跑着该项目。</p><p><a href="https://static.cnbetacdn.com/article/2022/0108/742203177eaa055.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0108/742203177eaa055.png" alt="2-1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（<a href="https://milkyway.cs.rpi.edu/milkyway/show_host_detail.php?hostid=912263" target="_self">BenchLeaks 1</a>）</p><p>新发现的两款工程样品处理器（AMD Eng Sample），部件编号分别为 <strong>100-000000665-21_N</strong> [Family 25 Model 96 Stepping 0] 和 <strong>100-000000666-21_N</strong> [Family 25 Model 96 Stepping 0] 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0108/c7489893641d873.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0108/c7489893641d873.png" alt="2-2.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（<a href="https://milkyway.cs.rpi.edu/milkyway/show_host_detail.php?hostid=911833" target="_self">BenchLeaks 2</a>）</p><p>前者是一枚 16 核处理器，后者是一枚 8 核处理器。不过据 BenchLeaks 所述，两款芯片均被 CPUID 工具软件识别为 AMD Raphael ES CPU，意味着它们仍处于相当早期的阶段。</p><p><img src="https://static.cnbetacdn.com/article/2022/0108/631ef7d298442c4.png" alt="4.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（截图 via <a href="https://wccftech.com/alleged-amd-ryzen-7000-zen-4-16-core-8-core-desktop-cpus-spotted/" target="_self">WCCFTech</a>）</p><p>虽然数据库中没有提及更多细节，但我们可以参考同一基准测试泄露的其它规格。比如两款 CPU 都具有 102<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_110018%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">4K</a>B L2 缓存，而市售的 AMD 锐龙 R9-5950X 仅配备了 512KB L2 缓存。</p><p><img src="https://static.cnbetacdn.com/article/2022/0108/82befa7dc9f7196.png" alt="3-0.png" referrerpolicy="no-referrer"></p><p>如此看来，AMD 应该为即将推出的 Zen 4 芯片配备了翻番的缓存。而早前曝光的芯片布局，也确实揭示了新一代芯片会在“优先”与“辅助”内核之间共享 1MB L2 缓存。</p><p><a href="https://static.cnbetacdn.com/article/2022/0108/494881f5bdf958d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0108/494881f5bdf958d.jpg" alt="5.jpg" referrerpolicy="no-referrer"></a></p><p>最后，根据泄露的核心规格数据，预计前者（疑似 R9-7950X）为当前 16C / 32T 的 AM4 桌面旗舰处理器 R9-5950X 的继任者，而后者（疑似 R9-7900X）有可能 12C / 24T 的 R9-5900X 的后续产品。</p>
