<div>   
<strong>上月，AMD 为 Instinct MI200 GPU 加速卡家族带来了一位新成员，它就是配备了 64GB HBM2e 显存、采用 PCIe 插槽的 MI210 。</strong>此前 AMD 一直没有披露确切的售价，且 3 月的预估上市日期也一直拖到了现在。最新消息是，日本市场的高性能计算（HPC）和服务器客户，有望于本月晚些时候以 208 万 7800 日元的价格购入 Instinct MI210 。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0418/bef82eb6c523e2c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/bef82eb6c523e2c.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/amd-instinct-mi210-gpu-accelerator-with-64-gb-hbm2e-memory-16500-usd-price-japan/" target="_self">WCCFTech</a>）</p><p>作为 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 旗下采用 Aldebaran GPU 的加速卡新品，搭配 64GB HBM2e 的 PCIe 版 Instinct MI210 加速卡的报价高达 16487 美元（约合 10.51 万 RMB）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0418/a8ea82a920f6aab.png" alt="7.png" referrerpolicy="no-referrer"></p><p>与 Instinct MI250 不同的是，MI210 采用了单芯片设计、基于 MCM 衍生型号相同的台积电 6nm 工艺节点制造。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/36cb2e17b272026.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/36cb2e17b272026.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>规格方面，Instinct MI210 具有单 GCD 工作集群，但仅启用了 Aldebaran GPU 上 128 个计算单元（CU）里的 104 个 / 6656 个流处理器。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/b21de8d491d0458.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/b21de8d491d0458.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>需要指出的是，即使更高端的 MI250X SKU，也仅启用了Aldebaran GPU 上的 110 个计算单元 / 7040 个流处理器。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/55fd23e26efe2f5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/55fd23e26efe2f5.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>除了核心数量，Instinct MI210 还具有 4096-bit 总线位宽的 64GB HBM2e 高带宽显存。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/c35917234052ca5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/c35917234052ca5.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>虽然显存位宽只有 MI250 系列的一半，但 MI210 的 HBM2e 显存容量，还是达到了 MI100 的 两倍（在被 MI250 取代之前、MI100 还算是 Instinct GPU 加速卡家族中的旗舰）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/a60c4e4a310710c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/a60c4e4a310710c.jpg" alt="5.jpg" referrerpolicy="no-referrer"></a></p><p>此外 Instinct MI210 的 GPU 峰值频率为 1700 MHz，显存频率 1600 MHz，加速卡标称带宽高达 1638.4 GB/s 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/e26e811cbce5c28.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/e26e811cbce5c28.jpg" alt="6.jpg" referrerpolicy="no-referrer"></a></p><p>算力方面，Instinct MI210 具有 181 TFLOP 的峰值 FP16 性能、45.3 TFLOP 的峰值 FP32 Matrix 和 FP64 Matrix 性能、以及 181 TOPs / TFLOPS 的 bfloat16、INT8 和 INT4 性能。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/33cb4d9cacf5a7b.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/33cb4d9cacf5a7b.png" alt="8.png" referrerpolicy="no-referrer"></a></p><p>标准非 Matrix FP64 / FP32 的标称性能为 22.6 TFLOP，明显低于 MI250X 系列。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/9756d513003230c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/9756d513003230c.png" alt="9.png" referrerpolicy="no-referrer"></a></p><p>不过 Instinct MI210 的标称 TBP 功耗额仅为 300W —— 只需单 8-pin 外接供电（MI250 和 MI250X SKU 则是 500W / 560W）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/53bcd76c2ebbd22.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/53bcd76c2ebbd22.png" alt="10.png" referrerpolicy="no-referrer"></a></p><p>最后，据日媒 Ask-corp 报道，AMD Instinct MI210 GPU 加速卡定于本月在当地上市，含税报价为 208 万 7800 日元。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/1e9b95fb76e55c9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/1e9b95fb76e55c9.jpg" alt="12.jpg" referrerpolicy="no-referrer"></a></p><p>即使换算成美元，$16485.65 也是个相当惊人的数字。预计正式上市时的官方建议零售价会略有变动，但还是会在 10000 美元以上。</p><p><a href="https://static.cnbetacdn.com/article/2022/0418/7027f45c659a352.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0418/7027f45c659a352.png" alt="11.png" referrerpolicy="no-referrer"></a></p><p>作为参考，面向消费级市场的英伟达 GeForce RTX 3090 Ti 等旗舰游戏显卡的报价在 4500 - 5500 美元左右。不过主打 HPC 应用场景的 Instinct MI210，采用了标准的 FHFL 外形尺寸 + 被动式双槽散热器。</p>
