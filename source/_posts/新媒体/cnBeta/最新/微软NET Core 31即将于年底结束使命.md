.NET Core 3.1是微软在2019年12月发布的一个开源、跨平台的软件开发框架，从2022年12月13日起将不再得到支持。该公司表示，.NET Core 3.1在那一天达到支持终止状态，微软将停止服务更新和技术支持。<br>
 <p>在本月早些时候发布的公告中，微软建议用户尽快转移到.NET 6，即该平台当前的长期支持（LTS）版本。那些在支持期结束后仍在使用.NET Core 3.1的用户将需要更新到.NET 6或.NET 7（将于今年11月发布），以继续获得支持。</p><p>微软提示说，建立在.NET Core 3.1上的应用程序将在支持期结束后继续运行，但.NET 6中的每一个安全修复对这些应用程序来说都是潜在的未修补的、未记录的安全漏洞。建议应用程序的最终用户与管理其软件的供应商联系，以确认是否需要和提供更新版本的软件。</p><p>从2022年12月<a data-link="1" href="https://microsoft.pvxt.net/zaZYr" target="_blank">Visual Studio</a> 2019 16.11、Visual Studio 2019 17.0和Visual Studio 2022 17.2的服务更新开始，.NET Core 3.1组件将被改为不支持和可选。因此，Visual Studio中的工作负载可以不安装.NET Core 3.1。</p><p><a href="https://static.cnbetacdn.com/article/2022/0731/fba3eedb2f38d68.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0731/fba3eedb2f38d68.jpg" title alt="download-latest-dotnet-core-min.jpg" referrerpolicy="no-referrer"></a></p>
