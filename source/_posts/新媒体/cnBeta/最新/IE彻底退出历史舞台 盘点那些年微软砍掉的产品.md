如果你是一名老网民，一定有过和IE浏览器相爱相杀的经历。作为Windows很长一段时间内的默认浏览器，IE一度是先进技术的代表，其恐怖的市场占有率也让其一度成为了事实上的网页兼容标准。然而，由于技术长久停滞不前、市场环境发生改变等原因，IE节节败退，不仅在市场上难有容身之地，在微软内部也成为了被淘汰的项目。<br>
 <p><em><img src="https://static.cnbetacdn.com/article/2022/0806/990beb466306387.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲大家一定用过的、最经典的IE6</p><p>在今年，IE终于被彻底淘汰了。微软早前就已经宣布，IE会在2022年6月15日正式停止支持。这意味着，这款走过整整27个年头的产品，就此走向了终结。尽管IE在如今看来，其产品力已经不值一提，但曾经的金字招牌被摘下，还是会令人心生感慨。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/b9596580f9b0587.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲IE也尝试过现代化，例如Win8中的IE11 Modern版，但始终没有转型成功</p><p>实际上，微软对于旗下的项目，处置的确非常果断。除了IE之外，也有大量一度给人留下深刻印象的产品，被微软关停，以至于有人调侃微软有专门的“砍刀部”。被微软砍掉的经典产品，还有哪些？在IE被正式关停之际，就让我们来盘点一下吧。</p><p><strong>WP/<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 10M（2010-2020）</strong></p><p>说起微软粉丝心中最挥之不去的痛，Windows Phone/Windows 10 Mobile操作系统绝对值得提名。作为微软出击新时代移动市场的拳头产品，WP一度受到业界青睐，其创新的磁贴界面、流畅的操作以及微软这位巨头的背书，让人们觉得这一定会是iOS和Android的有力竞争者。然而数年过去后，WP/Windows 10M已经沦为市占率永远1%的行尸走肉，微软在2020年为Windows 10M发布了最后一次更新后，放弃了该系统的维护。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/89c0ca8ec143fc6.jpg" referrerpolicy="no-referrer"><br></em></p><p>WP/Windows 10M失败的原因是一目了然，同时又是多方面的。长久以来，微软对WP一直存在规划路线的动摇不定，微软糟糕的版本和开发规划一次次将WP/Windows 10M的未来扭曲得让开发者大呼看不懂。例如，在WP8这个更换NT内核的重大版本推出之时，微软毅然抛弃老设备，一大票WP7只能升级到换汤不换药的WP7.8，要为WP8开发App得重构；在WP8.1推出时，App再次变更开发需求，从XAP变成了APPX；在Windows 10M时，微软又再次要求开发者转而开发UWP应用。这些不同的版本都需要开发者分别适配，但谁又会有耐心将应用一次次回炉重炼呢？这使得WP/Windows 10M长期缺少第三方应用，贫瘠的生态使其根本无法和竞争对手同台竞技。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/a351a9f4c40c1ec.png" referrerpolicy="no-referrer"><br></em></p><p>▲很多优秀的产品只能升级WP 7.8，WP的生态被拦腰斩断</p><p>WP/Windows 10M系统本身的完成度也一直算不上高。很难想象，2010年发布的WP，直到2013年才加入屏幕旋转锁定这样的初级功能，但如此魔幻的事情的确是现实。长久以来，WP还缺失震动时响铃、设置默认App、通知/媒体音量独自调节、多任务界面关闭App等基础功能，甚至还存在漏推送之类的系统级别缺陷。在进化速度如狼似虎的iOS和Android面前，WP一直如同半成品，每次更新都让用户们翘首以待后又深深失望。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/6f3aadc284b7226.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲WP长期没有统一的通知中心，靠磁贴呈现推送信息，这种基础机制的缺失让用户非常难受</p><p>加上WP/Windows 10M富有争议的界面设计，种种因素作用之下，用户、开发者和市场对WP\Windows 10M的接受度越来越低，最初的热情似火变成了后来的冷若冰霜。在2017年，微软宣布Windows 10M进入维护期，不再添加新功能。在2020年为Windows 10M推送了最后一次更新后，这个曾经让移动市场虎躯一震的产品，彻底沉寂。</p><p>可以说，WP/Windows 10M的失败主要归根于微软自作自受。无论是定位还是体量，WP/Windows 10M都称得上是重量级，它落得如此下场，的确令人唏嘘。</p><p><strong>Kinect（2010-2017）</strong></p><p>这是一个无数游戏玩家耳熟能详的名字。Kinect是微软推出的体感游戏设备，最早作为<a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> 360游戏机的<a data-link="1" href="https://microsoft.pvxt.net/Xx07X" target="_blank">配件</a>登场亮相。Kinect在亮相之际，就惊艳了一大票资深游戏玩家，很多人从未想到过原来游戏是可以不用游戏手柄、键鼠，而是手舞足蹈地去游玩。将Kinect安装好后，对着它跳舞，真的可以识别你在跳舞；对着它投篮，真的可以让你在游戏当中投篮，这种乐趣是传统游戏手柄难以与之媲<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://mideajiadian.jd.com/" target="_blank">美的</a>。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/073985161e90630.png" referrerpolicy="no-referrer"><br></em></p><p>Kinect是如何做到这一切的呢？它配备了多个摄像头，可以捕捉到用户的3D图像，精准识别用户的动作。由此，Xbox平台上诞生出了一大票配合Kinect的创意玩法，为游戏业界带来了新鲜的血液。</p><p>然而，Kinect这款黑科技最终还是走向了落寂。一方面，Kinect的覆盖率算不上高，这使得它的市场规模相对较小，为它开发游戏并不是大多数开发者的选择，由此导致Kinect的游戏资源乏善可陈；另一方面，Kinect的游玩需要一定的空间范围，而很多游戏玩家并没有如此宽敞的客厅。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/2e6c7fd67f2b04b.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲虽然Kinect玩起来的确很欢乐，但其实游戏并不多，而且对场地有硬性需求</p><p>此外，Kinect始终是存在一部分的硬件成本的，为了提高Kinect的覆盖率，微软在Xbox One时代作出了游戏机强制绑定Kinect销售的策略，这大幅拉高了Xbox One的售价，导致Xbox One在主机大战中落败于索尼PS4。尽管微软后来取消了这一捆绑策略，但大势已去，不但Xbox One没有取得预想中的成功，Kinect的销售额也不尽人意，同时Kinect还成为了人们的仇恨对象，可谓得不偿失。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/b557ddf742a8c5e.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲Kinect捆绑Xbox One销售绝对称得上是败笔</p><p>在口碑败坏、开发前景渺茫的大背景下，微软决定在2017年停产Kinect，这个黑科技游戏硬件就此和大家告别。</p><p><strong>Games for Windows Live（2007-2014）</strong></p><p>再来说一个和游戏相关的吧。现在人们买PC游戏，第一个找的可能就是Steam。不过在二十一世纪第一个十年，Steam可不像现在这样稳坐在线游戏商店王座。当时，包括微软在内的一些巨头，也在游戏分发领域看到了商机，而微软的招数就是Games for Windows。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/1ee0eceaa64a7f6.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲微软在游戏领域的战略级产品——GFWL</p><p>我们知道微软是游戏主机的三巨头之一，而Windows也是最适合游戏的PC系统，因此微软在游戏发行方面，拥有者得天独厚的优势。微软在2006年定义了Games for Windows的一系列规范，很多游戏都根据这个规范制作；而在2007年，微软推出了Games for Windows Live平台，相应软件会随着游戏安装一同安装到PC中，游戏必须通过Games for Windows Live启动，而游戏则可以通过Games for Windows Live和Xbox共享成就，更好地支持Xbox手柄，甚至还能够和Xbox玩家对战。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/9a505089e6dfd2f.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲借助GFWL，可以实现多人联机、通讯等</p><p>然而，Games for Windows Live遭遇了血崩式的失败。Games for Windows Live的实际体验非常差，光是注册（需要同时注册Games for Windows Live账号和Xbox Live账号）就能把人搞晕。而基于Games for Windows Live开发的游戏，也可能存在很多限制，例如需要连接Live服务器才能够存档。一旦Live服务器崩溃，游戏甚至无法正常游玩。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/3376775d0b57553.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲GFWL无论在安装、注册还是实际使用过程，体验都相当糟糕，是玩家最深恶痛绝的产品之一</p><p>Games for Windows Live给玩家带来了很多烦恼，运营也并不成功，微软在2014年7月4日关闭了该服务。</p><p>Games for Windows Live在Win7时代影响了一大波游戏，它的影响时至今日仍在继续，不少游戏的正版仍需要先通过Games for Windows Live登录才能游玩——然而Games for Windows Live却停运了，这影响到了很多游戏的游玩乃至销售。之前《GTA4》在Steam下架，就是因为这游戏是基于Games for Windows Live开发的，Games for Windows Live停运，GTA4没法再创建新的正版密钥来继续销售。</p><p>在Windows 10中，微软早已经放弃了Games for Windows Live，相应功能已经被Windows商店和Xbox Game Pass等服务取代。你曾经也是折腾过Games for Windows Live的玩家吗？</p><p><strong>Silverlight（2005-2021）</strong></p><p>微软是互联网技术的重要贡献者，它对于网络体验的贡献，自然不仅限于令人崩溃的IE。实际上，微软还开发过其他一些比较基础性的网络产品，试图通过新型技术来提升互联网的浏览体验，Silverlight就是其中的代表。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/f0b72414113fe56.jpg" referrerpolicy="no-referrer"><br></em></p><p>Silverlight是微软开发的一款浏览器插件，它的定位和现在已经淘汰的Flash类似，都是试图通过为网页加载额外的组件，从而实现当年HTML无法提供的交互和多媒体体验。</p><p>和Flash类似，Silverlight也是神通广大的。用户在浏览器安装Silverlight后，即可体验到网页中基于Silverlight开发的内容。这些内容包括视频，炫酷的网页交互，甚至还有人利用Silverlight开发出了游戏！对于主流只是图文的网页来说，Silverlight的确可以称得上是那个年代的神奇魔法。</p><p>微软对Silverlight也非常重视，甚至为Silverlight绘制了专属的动漫形象“蓝泽光”，这也是宅男们津津乐道的美谈。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/39ac54e55e5da39.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲微软为Silverlight推出二次元形象“蓝泽光”，该还经常藉此举办cosplay等活动</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/f1f4bfd87daed18.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲微软官方推出的以蓝泽光为女主的网页Galgame，所使用的技术正是Silverlight</p><p>然而，就如Flash最终走向末路一样，随着HTML技术的发展，这一类外挂于网页之上的插件也迎来了淘汰。效率、复杂度、安全是这类技术的无法磨灭的硬伤，有了更先进的HTML5，自然也再无Flash、Silverlight的立足之地了。在2021年10月12日，微软停止了对Silverlight的支持，这一抹曾经在互联网上留下灿烂光芒的银光，就此退出历史舞台。</p><p><strong>Zune（2006-2015）</strong></p><p>提起随身听，人们想起的往往是索尼Walkman，又或者是<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a><a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">iPod</a>。然而纵观整个随身听的发展史，微软其实也曾经在其中留下浓墨重彩的一笔，那就是微软曾经推出过的随身听产品——Zune。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/6e0dd392ce75832.png" referrerpolicy="no-referrer"><br></em></p><p>Zune称得上是惊才艳艳的产品，它所搭载的系统尤为夺人眼球。微软很多产品中令人印象深刻的磁贴，其实就是Zune引领起来的风潮。Zune首先引入了Metro风格的设计语言，随后这一套设计方案影响到了微软的诸多产品，例如WP、Win8乃至Windows 10等等。</p><p>且不说Metro界面在其他产品上体验如何，起码在Zune上，实属有口皆碑。磁贴完美地呈现了音乐专辑封面和艺人图片，而通过字号区分不同等级标题的做法，又让文字交互兼顾了可读性、层级感和高级感。或许正是由于Zune界面的大成功，才让微软决心在其他产品上也沿用这一套设计，最终水土不服，毕竟Zune这样的音乐播放器并不需要呈现太复杂的信息，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>和PC则不然，这是后话了。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/139fddfdcd240fd.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲Zune UI让微软走火入魔，在大量其他产品上照猫画虎</p><p>微软开发Zune所推出的应不仅仅是硬件，还拥有配套的在线音乐内容库，这也是受到iPod和iTunes影响所产生的开发思路。然而Zune的售价比较高昂，内容库又不占优势，最终在和iPod的竞争中败下阵来。加之随身听市场的整体退潮，Zune硬件迎来了停产。而Zune的配套软件，也在2015年被整合到Windows 10的Groove Music当中，Zune从此消失在了人们的视野当中。</p><p><strong>MSN（1999-2013）</strong></p><p>如果你是老网民，一定会知道MSN的名字，甚至亲自体验过MSN。在21世纪初，MSN作为微软推出的IM，在国内和QQ呈现出分庭抗礼之势。MSN和QQ的用户甚至还出现了类似粉圈互撕的行为，不少用户觉得MSN更加正式、高级、商务，而QQ则是小孩子的玩物，这种通过软件划分阵营的行为举止，也称得上是时代特征了。</p><p><em><img src="https://static.cnbetacdn.com/article/2022/0806/bc487ef3383dd80.jpg" referrerpolicy="no-referrer"><br></em></p><p>▲有多少人用过“高大上”的MSN呢？</p><p>然而，MSN的实力并不如其盛名，它较为羸弱的产品力，让它在和QQ的竞争中处于下风。你很难想象，作为定位商务市场的产品，MSN未对聊天信息进行加密，本地的聊天记录也以明文的方式存放，而MSN甚至还做不到聊天记录的同步！这显然无法满足商务人士对于私密和便携的需求，MSN尽管有着国际、商务等高大上的光环，但最终还是在市场中败退。</p><p>微软早已经停止了MSN聊天软件的开发，不过MSN的招牌至今仍保留下来。时至今日，MSN仍是微软重要的门户网站，为微软用户提供天气、新闻等信息。在微软生态中，收购而来的Skype已经取代了MSN曾经的生态位，不温不火的Skype又能走多远？我们拭目以待。</p><p><strong>后话</strong></p><p>除了文中所提到的，沦为微软“砍刀部”刀下鬼的产品还有EdgeHTML（旧版Edge）、Windows 10 Iot、Microsoft Academic微软学术等等，可见IE的难兄难弟还是非常多的。</p><p>这些产品之所以被砍，有些是本身做得产品力稀烂、毫无竞争力，例如WP；有些是错误的市场定位和策略，例如Kinect；有些是大环境剧变造成，例如Silverlight。尽管这些产品已经离用户而去，但它们曾经来过，在科技的发展大潮中留下了自己的身影，它们的成长和衰退之路，对于人们来说也是一笔宝贵的经验财富。你还用过什么被微软砍掉的产品？它又曾经给你带来过怎样的回忆？欢迎留言讨论。</p>
