<div>   
在 VideoCardz 昨日曝光之后，英特尔终于在 CES 2022 主题演讲期间披露了完整的 65W / 35W 台式 CPU 新品阵容。<strong>本次刷新为 12 代 Alder Lake 台式 CPU 产品线添加了 22 款 SKU，涵盖了酷睿 i9 / i7 / i5 / i3、以及奔腾 / 赛扬等细分市场。</strong>而对于主流消费者 / 游戏玩家来说，非 -K 系列的性价比显然更加出众。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0105/8a86a15627df5c6.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/8a86a15627df5c6.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>虽然超频 / 时钟频率 / TDP 没有 -K 系列那样高，但非 -K 系列显然会成为不少 OEM 厂商和 PC DIY 玩家的首选。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/c6709dcb43bfe30.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/c6709dcb43bfe30.png" alt="2-1.png" referrerpolicy="no-referrer"></a></p><p>而在12 代 Alder Lake-S 细分市场，我们就至少见到了 19 款非 -K 的台式 CPU SKU 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/68a86df6b47b3fd.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/68a86df6b47b3fd.png" alt="2-2.png" referrerpolicy="no-referrer"></a></p><p>如预期的那样，酷睿 i9 / i7 的非 -K 型号，保持着与 -K 型号相同的核心配置。两者的主要区别，在于时钟速率和 TDP 设置。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/5af5ca838e1ef33.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/5af5ca838e1ef33.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>以 i9-12900（F）为例，其具有相同的 16C / 24T、以及 30MB L3 缓存。不过高性能核心（P 核）的基础频率下调到了 2.4 GHz（睿频可达 5.1 GHz），节能核心（E 核）也调到了 1.8 GHz 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/26d62a3ef779407.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/26d62a3ef779407.png" alt="5-1.png" referrerpolicy="no-referrer"></a></p><p>如此一来，i9-12900（F）的两档 TDP 也低至 65W / 125W（MTP 为 241W）。价格方面，i9-12800 / T 为 489 美元，而 -F 版本则是 464 美元。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/9b24c54897ddf85.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/9b24c54897ddf85.png" alt="5-2.png" referrerpolicy="no-referrer"></a></p><p>接下来看看酷睿 i7-12700（F），其同样具有 12C / 20T、以及 25MB L3 缓存，只是 P 核调到了 2.1 GHz 基础频率（E 核也调到了 1.6 GHz）。但在 65W TDP 下，其睿频还是可以达到 4.9 GHz 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/40cf11928d9a1d2.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/40cf11928d9a1d2.png" alt="5-3.png" referrerpolicy="no-referrer"></a></p><p>价格方面，酷睿 i7-12700 / T 为 339 美元，而 -F 版本则是 314 美元。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/c3b846f99a08f32.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/c3b846f99a08f32.png" alt="5-4.png" referrerpolicy="no-referrer"></a></p><p>然后是采用非混合式核心架构设计的酷睿 i5 衍生型号，其中 i5-12600 / 12400 均采用 6P + 0E 的设计（6C / 12T），辅以 18MB L3 缓存。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/bd31264ac2ee593.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/bd31264ac2ee593.png" alt="5-5.png" referrerpolicy="no-referrer"></a></p><p>i5-12600 / 12400 的基础频率分别为 3.3 / 3.0 GHz，睿频可达 4.8 / 4.6 GHz，热设计功耗（TDP）为 65W —— 大家很是关注它与 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 R5-5600X 竞品的表现。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/c547a7e6f1ab6f5.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/c547a7e6f1ab6f5.png" alt="5-6.png" referrerpolicy="no-referrer"></a></p><p>有趣的是，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>还推出了介于两者之间的 i5-12500 。其具有相同的核心配置，但基础 / 睿频为 2.5 / 4.4 GHz 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/c547a7e6f1ab6f5.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/c547a7e6f1ab6f5.png" alt="5-6.png" referrerpolicy="no-referrer"></a></p><p>价格方面，酷睿 i5-12600 / T、i5-12500 / T、i5-12400 / T 和 i5-12400 F 分别为 223 / 202 / 192 / 167 美元。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/db122222c787190.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/db122222c787190.png" alt="5-7.png" referrerpolicy="no-referrer"></a></p><p>至于酷睿 i3，我们看到了 i3-12300 / 12100 两款 SKU 。它们采用了 4P + 0E 的设计（4C / 8T），拥有 12MB L3 缓存，睿频可达 4.4 / 4.3 GHz 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/dae0c5a068e39f1.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/dae0c5a068e39f1.png" alt="5-8.png" referrerpolicy="no-referrer"></a></p><p>核显方面，i5-12600 往下都集成了 UHD 730，而高端 SKU 则搭配了 UHD 770 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/cd336be257a7d04.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/cd336be257a7d04.png" alt="5-9.png" referrerpolicy="no-referrer"></a></p><p>价格方面，酷睿 i3-12100 / T 为 122 美元、-F 型号低至 97 美元，此外酷睿 i3-12300 / T 为 143 美元。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/f38c7ded05a521f.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/f38c7ded05a521f.png" alt="5-10.png" referrerpolicy="no-referrer"></a></p><p>当然也不能忘了更加入门的 Alder Lake 奔腾 / 赛扬产品线：其中 Pentium G7400 采用了 2C / 4T 的 Golden Cove 高性能核心（2P + 0E），基础频率 3.7 GHz 。Celeron G6900 配置相同，但频率低至 3.4 GHz 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/519e61f22ade709.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/519e61f22ade709.png" alt="5-11.png" referrerpolicy="no-referrer"></a></p><p>此外奔腾 SKU 拥有 6MB 缓存，但赛扬 SKU 精简到了 4MB 缓存。不过两者都集成了UHD 771 核显，基础 TDP 为 46W 。价格方面，Pentium G7400 为 64.99 美元、而 Celeron G6900 则是 42.99 美元。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/df6ea9057931848.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/df6ea9057931848.png" alt="5-12.png" referrerpolicy="no-referrer"></a></p><p>最后是英特尔为非 -K 系列 Alder Lake 盒装台式处理器标配的 Laminar 散热器，具体分三个版本。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/96531e29252f6c1.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/96531e29252f6c1.jpg" alt="4-3.jpg" referrerpolicy="no-referrer"></a></p><p>首先，Laminar RH1 能够压住 65W 的 Core i9 CPU，辅以 2.6dBA（接近静音）的散热性能。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/e11b91858b5e8cb.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/e11b91858b5e8cb.jpg" alt="4-2.jpg" referrerpolicy="no-referrer"></a></p><p>其次，Laminar RM1 能够压住 65W 的酷睿 i5 / i5 / i3 芯片，辅以 3.9 dBA（静音）的效能。</p><p><a href="https://static.cnbetacdn.com/article/2022/0105/80f8154431199e0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0105/80f8154431199e0.jpg" alt="4-1.jpg" referrerpolicy="no-referrer"></a></p><p>而更加普通的 Laminar RS1 散热器，则是专为 Alder Lake 奔腾 / 赛扬 CPU 而准备。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a
