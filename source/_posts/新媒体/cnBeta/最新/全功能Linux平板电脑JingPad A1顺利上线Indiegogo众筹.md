虽然 Android 也是基于 Linux 打造的，且市面上已有不少面向开发者的 Linux 平板电脑，但专为消费市场而设计的 Linux 设备仍相对有限。对于动手能力强的个人用户来说，确实可以尝试在手机或平板电脑上安装“适当的”Linux 发行版。<strong>但对于那些技术水平一般的普通 Linux 爱好者来说，当前正在 Indiegogo 上众筹的 JingPad Linux 平板，或许是一个更切实际的选择。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0617/523a67878af2e59.jpg" alt="0-1.jpg" referrerpolicy="no-referrer"></p><p>对于长期关注 Linux 平板电脑市场的爱好者们来说，或许不会对 JingPad 这个名字感到陌生。</p><p><img src="https://static.cnbetacdn.com/article/2021/0617/0ed7b898f417603.jpg" alt="0-2.jpg" referrerpolicy="no-referrer"></p><p>更确切地说，你可能见过它家的 JingOS（一款 Linux 风格的平板电脑操作系统）。</p><p><img src="https://static.cnbetacdn.com/article/2021/0617/21960c5a9e7072a.png" alt="1.png" referrerpolicy="no-referrer"></p><p>今年早些时候突然出现在大家面前的 JingOS，给不少人留下了惊艳的印象。</p><p><img src="https://static.cnbetacdn.com/article/2021/0617/7d292a9d9c0be10.png" alt="2.png" referrerpolicy="no-referrer"></p><p>虽然有着类似<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a> <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a>OS 的外观，但 JingOS 的底层还是基于开源的 Linux 桌面。</p><p><img src="https://static.cnbetacdn.com/article/2021/0617/d100957da369c1c.png" alt="3.png" referrerpolicy="no-referrer"></p><p>与此同时，JingOS 背后的 Jingling Tech 公司也宣布将把该操作系统，引入即将向消费者推出的 JingPad A1 平板电脑上。</p><p><img src="https://static.cnbetacdn.com/article/2021/0617/3a25194bf5e939a.png" alt="4.png" referrerpolicy="no-referrer"></p><p>最新消息是，这家初创企业已经顺利地在 Indiegogo 上发起了众包活动，且首日就达成了超过五倍的众筹目标。</p><p><img src="https://static.cnbetacdn.com/article/2021/0617/2d0c532df7f906c.png" alt="5.png" referrerpolicy="no-referrer"></p><p>配置方面，JingPad A1 采用了 11 英寸 @ 2K 分辨率的 AM<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_88234%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">OLED</a> 屏（宽高比 4：3），八核 @ 2GHz 的紫光展锐 T7510 芯片组，以及 8GB LPDDR4 RAM + 256GB UMCP 存储的组合（支持 512GB microSD 扩展）。</p><p><img src="https://static.cnbetacdn.com/article/2021/0617/a7d81ccf2a60469.png" alt="6.png" referrerpolicy="no-referrer"></p><p>软件方面，JingPad A1 也与 Android 平板电脑有较大的差别。得益于成熟的 Linux 桌面，用户能够轻松体验 Libre<a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a>、GIMP 等常用软件，以及流行的开发者 / 系统管理工具。</p><p><img src="https://static.cnbetacdn.com/article/2021/0617/04f1370fa0201d1.jpg" alt="7.jpg" referrerpolicy="no-referrer"></p><p>在主打内容消费的平板形态之外，JingPad A1 还能够作为一台通用的 Linux 笔记本电脑，结合键盘底座 + 支持 4096 级压感的手写笔。</p><p style="text-align: center;"><iframe width="640" height="480" src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=265385808&autoplay=false&disablePlaylist=true" frameborder="0"></iframe></p><p style="text-align: center;">Introducing JingPad A1 Linux Tablet（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMjY1Mzg1ODA4LnNodG1s.html" target="_self">via</a>）</p><p>感兴趣的朋友，可享 549 美元（约 3529 RMB）的早鸟价。如果一切顺利，JingPad A1 将于 9 月份向早期支持者们出货。</p>
