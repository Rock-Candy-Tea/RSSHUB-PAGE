面向 Dev 频道的 Windows Insider 项目成员，微软昨日发布了 Windows 11 Build 22478 预览版更新。本次更新最重要的是引入基于 Fluent Design 设计语言的 Emoji。<strong>然而，令许多在日常工作流程中使用 Emoji 的人感到失望的是，新的表情符号是 2D 的，而不是像该公司迄今为止一直宣传的 3D。</strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1015/e97e120d3dc6eba.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1015/e97e120d3dc6eba.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在最新的 Dev 频道构建中出现的平面表情符号可以在上面看到，而之前宣传的那些表情符号可以在下面看到，差别相当明显。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1015/b39b98eabfac733.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1015/b39b98eabfac733.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">据 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Insider Program 高级项目经理 Brandon LeBlanc 透露，上面显示的 3D Emoji 似乎根本不会出现在 Windows 11 的表情符号面板中。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1015/6d397251e53c9ed.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">考虑到 Windows UK Twitter账户也一直在宣传 3D 表情符号是操作系统的原生内容，当被问及 Windows 11 营销是一个骗局时，LeBlanc 淡化了这个问题，他说：“ 不，他们没有骗你。你有点夸大其词了。他们只是使用了错误的图片。对此很抱歉。将确保他们今后使用正确的图形”。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1015/5e36a22d5d345d6.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">其他一些软件，如Teams、Yammer或Outlook是否会得到3D表情符号，还有待观察。<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>设计团队早在7月谈及3D表情符号的发布时，就点出了上述软件和Windows。LeBlanc 拒绝分享有关此事的更多细节。</p>
