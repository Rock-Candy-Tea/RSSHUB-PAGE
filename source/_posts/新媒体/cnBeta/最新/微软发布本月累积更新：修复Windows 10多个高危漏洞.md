面向所有支持的 Windows 10 系统，微软今天发布了本月的累积更新。除了 Windows 10 21H1、20H2 和 20H1 三大功能更新之外，今天微软还发布了适用于Windows 10 version 1507 (<a href="https://support.microsoft.com/en-us/topic/july-13-2021-kb5004249-os-build-10240-19003-bcd2edd4-76bb-4e33-8d30-e231057cb1d1" target="_blank" rel="noopener">KB5004249</a>), version 1607 (<a href="https://support.microsoft.com/en-us/topic/july-13-2021-kb5004238-os-build-14393-4530-69c72ccc-299a-412a-b219-b78c2fd37021" target="_blank" rel="noopener">KB5004238</a>), version 1809 (<a href="https://support.microsoft.com/en-us/topic/july-13-2021-kb5004244-os-build-17763-2061-d2b763cb-fad4-4b0a-a67f-4e184e277b99" target="_blank" rel="noopener">KB5004244</a>)和version 1909 (<a href="https://support.microsoft.com/en-us/topic/july-13-2021-kb5004245-os-build-18363-1679-fe157ce5-49c1-4146-b948-c5aef1f0293b" target="_blank" rel="noopener">KB5004245</a>)的累积更新。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0714/327b6dab03c5236.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0714/327b6dab03c5236.jpg" alt="3y0skawm.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">而自本月开始，以下功能更新将不再获得 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 10 的累积更新：</p><blockquote style="text-align: left;"><p style="text-align: left;">● Windows 10, version 1511</p><p style="text-align: left;">● Windows 10, version 1703</p><p style="text-align: left;">● Windows 10, version 1709</p><p style="text-align: left;">● Windows 10, version 1803</p><p style="text-align: left;">● Windows 10, version 1903</p></blockquote><p style="text-align: left;">适用于 Windows 10 Version 1909 功能更新的 KB5004245 累积更新带来了以下修复：</p><blockquote style="text-align: left;"><p style="text-align: left;">● 用于验证用户名和密码的更新。</p><p style="text-align: left;">● 提高 Windows 执行基本操作时安全性的更新。</p><p style="text-align: left;">● 为 CVE-2021-33757 增加了高级加密标准（AES）加密保护措施。欲了解更多信息，请参见 <a href="https://support.microsoft.com/en-us/topic/kb5004605-update-adds-aes-encryption-protections-to-the-ms-samr-protocol-for-cve-2021-33757-e4daa133-54aa-4a5d-a921-04bb50868fc2" target="_blank">KB5004605</a>。</p><p style="text-align: left;">● Windows Apps, Windows Management, Windows Fundamentals, Windows Authentication, Windows User Account Control (UAC), Operating System Security, Windows Virtualization, Windows Linux, the Windows Kernel, the Microsoft Scripting Engine, the Windows HTML Platforms, the Windows MSHTML Platform 和 Windows Graphics 的安全更新。</p></blockquote>
