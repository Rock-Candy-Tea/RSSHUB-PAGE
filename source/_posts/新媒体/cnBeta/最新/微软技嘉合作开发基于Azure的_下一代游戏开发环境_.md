<strong>此前有报道称，微软曾考虑在十多年前收购世嘉，但我们最终未能见到它的实现。即便如此，近年来这家雷德蒙德科技巨头还是与世嘉保持着相当友好且密切的联系。</strong>比如在 Xbox Series X | S 发布的时候，《如龙 7》也以护航作的名义，登陆了该公司的次世代游戏主机平台。不久后，Xbox Game Pass 订阅服务又陆续迎来了其它几款《如龙》游戏。<br>
 <p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2021/1101/7659e44944e8d54.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Sega <a href="https://www.sega.co.jp/en/release/211101_1.html" target="_self">官网</a>）</p><p>世嘉方面表示，这项合作关系主要围绕着该公司如何利用 Microsoft Azure 的力量，以期在“下一代开发环境”中构建大型游戏。</p><p>基于此，这家日本知名游戏公司有望推进其“超级游戏”计划。作为一项中长期战略，其旨在打造面向全球、线上和以社区为中心的游戏。</p><p>然后借助 Microsoft Azure 云技术，为其提供迎合大量受众的游戏体验，且双方还会在此期间合作改进开发流程、网络基础设施、以及在线通信服务和相关工具。</p><p><img src="https://static.cnbetacdn.com/article/2021/1101/922a5964dd08490.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>世嘉总监兼首席运营官 Yukio Sugino 评论道：</p><blockquote><p>我们今天很高兴地宣布，世嘉正考虑与微软结成战略联盟，以构建下一代游戏开发环境、并协助推动‘超级游戏’计划的发展。</p><p>我们寻求让世界各地的粉丝都能尽享世嘉游戏的方法，而这项合作伙伴关系，正好能够结合世嘉强大的游戏开发能力、以及微软的尖端技术和开发环境。</p></blockquote><p>另一方面，微软执行副总裁 Sarah Bond 表示：</p><blockquote><p>世嘉在游戏行业扮演着如此重要的角色，且多年来一直是个很棒的合作伙伴。</p><p>微软期待着与他们的合作，探索利用微软云技术来打造未来独特游戏体验的新方法。</p><p>我们将一起重新构想游戏的制作、托管和运营方式，目标是为玩家和世嘉等合作伙伴都增加更多的价值。</p></blockquote>
