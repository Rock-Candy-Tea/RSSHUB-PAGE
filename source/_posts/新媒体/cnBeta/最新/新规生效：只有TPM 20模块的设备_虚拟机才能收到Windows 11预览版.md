微软正在执行 Windows 11 新准入门槛--<strong>也就是说只有 TPM 2.0 模块、符合最低硬件标准的设备才能更新到 Windows 11 最新预览版 Build 22458。</strong>不少用户在 Twitter 发布推文，表示在没有 TPM 2.0 的虚拟机上就无法升级到新版本了。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0916/584f82d8f69d31e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0916/584f82d8f69d31e.jpg" alt="qkn4d8r9.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">面向未符合 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 升级条件的 Windows Insider 项目成员，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>日前发出提醒推荐他们降级使用 Windows 10 系统。本月初，Windows Insider 计划的 Brandon LeBlanc 现在对情况进行了一些澄清。他在推文中写道：“获得例外的 PC 今后将不再收到来自 Dev 频道的 Windows 11 预览版更新，但在正式发布之前仍可以获得累积更新”。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0902/a73c46b681495a0.jpg" target="_blank" data-index="1" data-lightbox-gallery="cbc-gallery"><img src="https://static.cnbetacdn.com/thumb/article/2021/0902/a73c46b681495a0.jpg" alt="3zt5mztq.jpg" data-original="https://static.cnbetacdn.com/thumb/article/2021/0902/a73c46b681495a0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">简单来说，目前已经参与 Windows 11 但不符合升级条件的 Windows Insider 设备每周定期仍能获得小型的累积更新，但是无法获得微软后续为 Dev 频道发布的新预览版。此外在 Windows 11 正式发布之后，这些用户将会再次被要求降级到 Windows 10 系统。</p>
