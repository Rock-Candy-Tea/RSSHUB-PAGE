去年微软推出Win11系统时，TPM安全模块以及Intel 8代酷睿/AMD锐龙2代及以上的硬件要求引发了网友不满，这提高了系统使用的门槛，现在来看微软是打算继续推这个标准下去了，下版Windows Server系统也会如此。<br><br>
<p>微软前几天发布了<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Server vNEXT Build 25057系统，这就是Windows 11版的LTSC长期服务版系统，目前是测试中。</p><p><a href="https://static.cnbetacdn.com/article/2022/0219/646e7ba133d9e75.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0219/646e7ba133d9e75.jpg" title alt="FLvPlA2XwAQcpZy.jpg" referrerpolicy="no-referrer"></a></p><p>有网友在安装Windows Server vNEXT Build 25057系统时发现了问题，<strong>系统会提示硬件兼容性，其中主要是两个问题，一个是支持TPM，一个是CPU兼容，也是Intel 8代酷睿/<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a>锐龙2代及以上。</strong></p><p>这倒是不让人意外，因为Windows Server vNEXT Build 25057就是基于Windows 11系统开发的，硬件要求一样很正常。</p><p><a href="https://static.cnbetacdn.com/article/2022/0219/3d82d0f9d4f206e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0219/3d82d0f9d4f206e.jpg" title alt="FL5XfzuWQAAKJ6R.jpg" referrerpolicy="no-referrer"></a></p><p>按照Windows 11系统的情况来看，用户也应该可以绕过这些限制强制安装系统，但这也可能导致一些更新无法升级，微软并不乐见不达标的系统升级新系统的。</p><p><strong>好在Windows Server新系统距离最终发布还有2年时间，希望到时候用户的设备都能达标吧。</strong></p>
