微软正在将其 PowerToys 工具套件带到 Windows 11 操作系统的官方应用商店。<strong>今天早些时候，XDA-Developers 已经在 Microsoft Store 上发现了它的身影。</strong>对于高级用户们来说，这使得他们能够更方便地下载软件资源，而无需移步至 GitHub 等第三方获取。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0918/17b9a69a29714d8.png" referrerpolicy="no-referrer"></p><p>（来自：XDA-Developers）</p><p>此前除了 GitHub，<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 用户还可借助 Chocolatey 或 winget 之类的软件包管理器来安装 PowerToys 。</p><p>有趣的是，PowerToys 应用程序在 Windows 11 上的安装体验，与您通过 GitHub 安装常规软件的体验几乎相同。</p><p>不过 Microsoft Store 并不负责关键实际的应用程序更新，因为软件本身集成了升级功能。</p><p><a href="https://static.cnbetacdn.com/article/2021/0918/5627bacb6004fe7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0918/5627bacb6004fe7.jpg" referrerpolicy="no-referrer"></a></p><p>据悉，PowerToys 是一套面向高级用户的免费工具，可帮助优化他们的 Windows 体验以提高工作效率。</p><p>其中整合了一系列工具，包括 FancyZones、PowerToys Run、颜色选择器、以及图像调整工具等。最近它还迎来了 PowerToys Awake 工具，能够让用户按需将 PC 保持唤醒状态。</p><p>同时<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>正在测试适用于视频会议场景的静音实用程序，以便用户一键将 Windows PC 上的摄像头或麦克风静音。</p><p>最后，微软计划在 Windows 11 正式发布之前，将该公司旗下所有大型应用程序也带到 Microsoft Store，且 Microsoft Edge 浏览器、<a data-link="1" href="https://microsoft.pvxt.net/zaZYr" target="_blank">Visual Studio</a> / VS Code 集成开发环境、以及官方 PWA Reddit 应用程序已先行抵达。</p>
