<strong>早在2020年9月，微软宣布下一版本的Exchange
Server将在2021年下半年发布。然而，由于恶意行为者对企业内部Exchange服务器的全球性攻击，该公司错过了这一期限。</strong>随着微软重新确定优先次序，将开发重点专注于Exchange安全，公司宣布下一版本的Exchange服务器将在2025年下半年推出，这与最初的战略相比推迟了足足四年。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0603/c77054d147e7cc8.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0603/c77054d147e7cc8.jpg" title alt="1654238958_microsoft-exchange-01_(1)_story.jpg" referrerpolicy="no-referrer"></a></p><p>下一个版本的Exchange Server（为了简单起见，我们把它称为"Exchange Server 2025"，在本文的其余部分）将只提供给拥有软件保证的客户。它还需要一个服务器和客户端访问许可证（CAL）。虽然我们知道它将通过订阅模式提供，但微软表示，它将在2024年上半年分享更多关于定价、功能、品牌和要求的细节。</p><p>正如ZDNet的Mary Jo Foley所指出的，Exchange Server 2016和2019将被支持到2025年10月14日，但只有Exchange Server 2025将在该日期之后被支持。有趣的是，这个版本的Exchange Server将没有终止支持日期，微软将继续支持它，直到有"实质性的市场需求"。</p><p>微软还暗示，它正在为Exchange Server 2019上的人进行平滑升级。这表明，在更新到Exchange Server 2025不需要新硬件或再计划一次邮箱迁移。</p><p>同时，微软强调了它在Exchange Server 2019中引入的一堆改进和功能，这将至少在未来三年内继续成为其最新版本。你可以在微软的博客文章中查看Exchange Server 2019的最新进展：</p><p><a href="https://techcommunity.microsoft.com/t5/exchange-team-blog/exchange-server-roadmap-update/ba-p/3421389" _src="https://techcommunity.microsoft.com/t5/exchange-team-blog/exchange-server-roadmap-update/ba-p/3421389" target="_blank">https://techcommunity.microsoft.com/t5/exchange-team-blog/exchange-server-roadmap-update/ba-p/3421389</a></p>
