在 Windows 11 平台上，如果你想将默认浏览器从 Edge 切换到 Firefox/Chrome，那么整个操作现在变得更加复杂了。微软已经删除了只需点击一下就能设置默认浏览器的选项，用户需要对<strong> HTM、HTML、HTTP、HTTPS、SHTML、WEBP、SVG 等文件类型均配置之后才能切换到 Firefox/Chrome。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1206/174c948af793210.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 版本更迭中，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>对应用程序和功能页面进行了重新设计。该页面允许用户自行配置默认文件打开应用，包括浏览器和音乐播放器。对于浏览器，微软已经删除了一个一切使用第三方浏览器的选项。</p><p style="text-align: left;">现在，如果你试图将默认浏览器改为 Chrome 或 Firefox，你将需要为所有可能的网络相关格式手动改变文件类型。<strong>这包括 HTM、HTML、HTTP、HTTPS、SHTML、WEBP、SVG，甚至是 PDF。如果你忘记为所有必要的选项打勾</strong>，你将无法使用 Chrome 或 Firefox 作为默认浏览器。</p><p style="text-align: left;">这种复杂的变化让人感到厌烦，用户说微软在试图强迫自己的偏好。在用户发现这一变化后，微软也收到了来自 Google 和 Opera 等对手的反击。</p><p style="text-align: left;">Google 高级副总裁 Hiroshi Lockheimer 说，这种新的[默认浏览器]变化来自于“这家声称是最开放、有最多选择的公司”。这根本不是“选择”。Firefox 官员还批评微软强迫“采取额外和不必要的步骤来设置和保留默认浏览器设置”。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1206/9bce5917d0ef4c5.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">值得庆幸的是，微软终于决定通过一个新的预览更新来恢复 Windows 11 复杂的默认浏览器设置。在 Windows 11 Build 22509 中，微软悄悄启用了一个新的切换键，让你修改所有网络相关文件格式/连接的默认浏览器设置，包括 HTTP 和 HTTPs。</p>
