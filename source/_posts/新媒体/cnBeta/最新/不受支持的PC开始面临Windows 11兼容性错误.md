在今年 6 月召开了 Windows 11 发布会之后，微软一直在努力完善《PC 健康检查》工具，以便用户能够轻松知晓他们的设备是否符合升级至新一代桌面操作系统的要求。<strong>而随着 Windows 11 即将于 10 月 5 日正式推出，一些在不受支持的设备上安装体验的用户，也开始面临一些兼容性问题。</strong><br>
<p><img src="https://static.cnbetacdn.com/article/2021/1005/b54ccefdd03618e.jpg" referrerpolicy="no-referrer"></p><p>随着 Windows 11 开始陆续向世界各地的早期体验者们正式推送，想要检查兼容性的过程非常简单，只需在 PC 收到更新通知消息时访问“设置 -> 更新与安全 -> Windows Update”即可。</p><p>如果被遗憾告知如下错误信息，那就意味着当前设备状况并不符合 Windows 11 正式版的升级要求：</p><blockquote><p>这台 PC 目前并不满足 Windows 11 的所有系统要求。如需获取详情，还请在《PC 健康检查工具》应用程序中执行相关操作。</p></blockquote><p>与此同时，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还贴心地在报错信息中附上了《PC Health Check》的下载链接。运行后，你将能够轻松运行该工具，以知晓到底是什么情况阻碍了 Windows 11 正式版更新。</p><p><a href="https://static.cnbetacdn.com/article/2021/1005/47392e783c1c6e5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1005/47392e783c1c6e5.jpg" referrerpolicy="no-referrer"></a></p><p>以 <a href="https://www.windowslatest.com/2021/10/04/windows-11-compatibility-error-starts-appearing-on-unsupported-pcs/" target="_self">Windows Latest</a> 分享的报错信息为例，《PC 健康检查工具》提示当前设备尚未启用 TPM 2.0 可信任平台模块。</p><blockquote><p>当在主板 BIOS 设置中开启它之后（另一项重要设置是‘安全启动 / Secure Boot’），这一错误消息就消失无踪了。</p><p>如果经过一番操作，还是未能顺利更新至正式版 Windows 11 操作系统，亦可参考该工具附上的支持文档链接，以了解升级所需的完整设备规格。</p></blockquote><p>假如当前设备已万事俱备，你将看到以下提示 ——“好消息，您的 PC 满足 Windows 11 的最低系统要求”。</p><p>不过目前 Windows 11 正式版更新包仍处于分批推送的阶段，预计我们会在 2022 年初迎来更广泛的推出。</p><p>如果实在等不及，亦可通过官方正式版 ISO 镜像、或《媒体创建工具》来制作 USB 安装驱动器来实施更新。</p>
