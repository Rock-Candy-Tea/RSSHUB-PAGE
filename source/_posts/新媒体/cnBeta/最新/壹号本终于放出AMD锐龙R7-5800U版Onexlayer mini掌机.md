在推出了基于英特尔酷睿 i7-1195G7“Tiger Lake”处理器的 OnexPlayer mini 掌机之后，<strong>壹号本（One Notebook）又于近日带来了基于 AMD Zen 3 锐龙 R7-5800U 处理器的衍生版本。</strong>虽然 Valve 发布的 Steam Deck 掌机集成了图形性能更强的 RDNA 2 GPU，但 AyaNeo Next 和 OnexPlayer mini 的体型相对较小（售价也更高）。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0305/9063a1e5e6b9b54.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0305/9063a1e5e6b9b54.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（via <a href="https://onexplayerstore.com/collections/frontpage" target="_self">One Notebook</a>）</p><p>规格方面，新款锐龙版 OnexPlayer mini 掌机配备了 8 核 / 16 现成的 R7-5800U APU 处理器，屏幕分辨率为 1280×800 。</p><p>CPU 加速频率可达 4.4 GHz、集成的 Radeon Vega8 核显频率也高达 2 GHz，辅以 16GB @ LPDDR4X-4266 内存（仅限该机型）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0305/fc9fada5d9cc535.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0305/fc9fada5d9cc535.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p><strong>以下是 One Notebook 官方列出的 6 款 SKU：</strong></p><blockquote><p>● 16GB RAM + 512GB 存储 / 高清屏，售价 5299 RMB（约 840 美元）。</p><p>● 16GB RAM + 1TB 存储 / 高清屏，售价 5599 RMB（约 885 美元）。</p><p>● 16GB RAM + 2TB 存储 / 高清屏，售价 6599 RMB（约 1045 美元）。</p><p>● 16GB RAM + 512GB 存储 / FHD 屏，售价 5599 RMB（约 885 美元）。</p><p>● 16GB RAM + 1TB 存储 / FHD 屏，售价 5899 RMB（约 935 美元）。</p><p>● 16 GB RAM + 2TB 存储 / FHD 屏，售价 6899 RMB（约 1090 美元）。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0305/e27c2f0b9825e2e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0305/e27c2f0b9825e2e.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>最后，壹号本将首先向 OnexPlayer 群内成员提供新款掌机。至于何时向经销商供货，还请耐心等待后续通告。</p>
