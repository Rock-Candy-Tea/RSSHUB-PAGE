经过一段时间的预览，Android Studio 集成开发环境（IDE）的版本号终于正式抵达 2021.2.1 。<strong>据悉，在迎接代号为“花栗鼠”（Chipmunk）的最新稳定版本之前，谷歌已经向我们介绍过“大黄蜂”（Bumblebee）和“北极狐”（Arctic Fox）。</strong>显然，Android Studio 开发团队很喜欢用动物名称来排序。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0510/7d9846cf45988c4.png" alt="1.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Android Developers <a href="https://android-developers.googleblog.com/2022/05/android-studio-chipmunk.html" target="_self">网站</a>）</p><p>虽然 Android Studio Chipmunk 更新了 IntelliJ 大版本，但作为一个相对较小的功能更新，主要工作还是围绕着超过 175 个的质量问题修复。</p><p>此外 Android Studio 2021.2.1 稳定版意味着早前编译动画预览（Compose Animation Preview）的正式启用。</p><p><img src="https://static.cnbetacdn.com/article/2022/0510/bfcdb06a948da48.gif" alt="2.gif" referrerpolicy="no-referrer"></p><p>开发者们能够借助暂停、循环、快进、慢放等不同的控件，检查并调试 Android Jetpack Compose 的各种动画。</p><p>目前该功能仅支持 AnimatedVisibility 和 updateTransition，但随着时间的推移，Android Studio 将能够处理更多动画类型。</p><p><img src="https://static.cnbetacdn.com/article/2022/0510/a1dae2f9d79a8fd.png" alt="3-0.png" referrerpolicy="no-referrer"></p><p>此外更新后的 CPU Profiler 已能够显示有关卡顿的更多信息，不过前提是搭配 Android 12 / API Level 31 及更高版本使用。</p><p><a href="https://static.cnbetacdn.com/article/2022/0510/6c02c4689c140d7.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0510/6c02c4689c140d7.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p>构建分析器（Build Analyzer）中还有个 Jetifier 检查，能够提示开发者是否可以移除 Jetifier 标记以提升性能。可知在某些情况下，此举有助于节省 5-10% 构建时间。</p><p><img src="https://static.cnbetacdn.com/article/2022/0510/2b00cb497433a3c.jpg" alt="5.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：IntelliJ IDEA <a href="https://blog.jetbrains.com/idea/2021/07/intellij-idea-2021-2/" target="_self">Blog</a>）</p><p>顺道一提，Android Studio Chipmunk 附带的 IntelliJ 2021.2 具有许多新功能，包括对 IDE 操作的增强、更好的包搜索 UI、以及项目范围分析等。</p><p>感兴趣的朋友，可移步至 Android Developer <a href="https://developer.android.com/" target="_self">官方网站</a>，以下载并体验 Android Studio 2021.2.1 。</p><div
