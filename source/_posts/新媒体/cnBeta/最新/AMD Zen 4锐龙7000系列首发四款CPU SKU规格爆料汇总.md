各方消息已证实 AMD 即将推出 Zen 4“Raphael”锐龙 7000 系列处理器和配套的 600 系主板，且首发 AM5 台式 CPU 阵容中将包括 R9-7950X、R9-7900X、R7-7700X 和 R5-7600X 四款 SKU 。<strong>而随着传说中的 9 月 15 号的临近，WCCFTech 也用心汇总了锐龙 7000 系列台式 CPU 的最终规格。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0525/a82e0d846d06cc4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0525/a82e0d846d06cc4.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>与 Zen 3 相比，Zen 4 架构有望带来 8-10% 的 IPC 性能提升。此外得益于更高的 TDP 与时钟速率，锐龙 7000 系列台式 CPU 还有望带来更大的惊喜。</p><blockquote><p>AMD 强调称，Zen 4 较 Zen 3 提升了 >15% 的单线程、>35% 的多线程、以及 >25% 的每瓦特性能提升。</p><p>且新一代 CPU 配备了优化重组的缓存、翻倍的 L2（从 512KB 到 1MB）+ 与上一代类似的共享 L3 缓存。</p><p>另外通过 AMD EXPO 内存超频扩展配置文件以支持 DDR5，还有 PCIe 5.0 显卡 / M.2 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 。</p></blockquote><p>（1）言归正传，AMD 锐龙 R9-7950X 延续了 16C / 32T 设计，基础频率 4.5 GHz / 加速可达 5.7 GHz 。</p><blockquote><p>如此亮眼的数据，甚至较<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 12 代酷睿 i9-12900KS 的单核 5.5 GHz 睿频还要高 200 MHz 。</p><p>总缓存 80MB，分为 16MB L2（每核心 1MB）+ 64MB L3（每 CDD 32MB），标称 170W TDP / 230W PPT 功耗。</p></blockquote><p>虽然尚不清楚 AMD 的定价策略，但 R9-7950X 显然会是 R9-5900X 的强大继任者，并将酷睿 i9-12700K 远远抛在身后。</p><p><img src="https://static.cnbetacdn.com/article/2022/0618/7baed5e7c6f193a.png" alt="2.png" referrerpolicy="no-referrer"></p><p>（2）第二款 R9-7900X 芯片是 12C / 24T 的设计，基础频率 4.7 GHz / 加速可达 5.6 GHz 。总计 76MB 缓存，分为 12MB L2 + 64MB L3 。</p><p>其定位与 R9-5900X 相似，但热设计功耗同样放宽到了 170W，且有望将酷睿 i7-12700K 挑落马下。</p><p>（3）第三款 R7-7700X 芯片是 8C / 16T 的设计，官方定位是“游戏玩家优选”，基础频率 4.5 GHz / 加速可达 5.4 GHz 。</p><p>总计 40MB 缓存，分为 8MB L2 + 32MB L3，但 TDP 为较低的 105W（142W PPT）。</p><p>（4）第四款 R5-7600X 芯片是 6C / 12T 的设计，其有望成为主流玩家的心头好，基础频率 4.7 GHz / 单核加速可达 5.3 GHz 。</p><p>总计 38MB 缓存，分为 6MB L2 + 32MB L3，TDP 从 R5-5600X 的 65W 大幅放宽到了 105W（142W PPT），这或许是为了冲击更高频率而付出的必要代价。</p><p><a href="https://static.cnbetacdn.com/article/2022/0618/2e54ff2358d8067.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0618/2e54ff2358d8067.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>至于传说中的 R7-5800X 的继任者，AMD 迄今尚未透露任何口风。一种猜测是，该公司打算用配备了 3D V-Cache 缓存的 SKU 来占位（类似 R7-5800X3D）。</p><p>至于真相究竟如何，还请耐心等待今年晚些时候的官宣。如果一切顺利，Zen 4 锐龙 7000 台式 CPU 和配套的 600 系 AM5 主板，都有望于 2022 年 4 季度正式到来。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a
