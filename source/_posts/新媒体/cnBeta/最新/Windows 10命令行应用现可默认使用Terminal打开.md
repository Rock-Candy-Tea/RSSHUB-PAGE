在近日发布的 Windows Terminal 1.9 预览版更新中，允许将其作为 Windows 10 系统的默认终端应用程序。在将 Terminal 设置为默认终端之后，包括 Command Prompt 和 PowerShell 等命令行应用都会在 Windows Terminal 中打开。<br>
