去年 11 月，就有消息称 WhatsApp 正在开发一个全新的桌面应用，可以通过 Microsoft Store 购买。当时的报道称该应用基于 UWP，而非 Electron，而且还支持手写和多设备。<strong>现在看来，开发团队正在转向 WinUI 2.6 及更高版本，以利用 Windows 11 中的 Fluent 设计元素。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0118/11696fe54fd44e1.webp" style referrerpolicy="no-referrer"></p><p><img src="https://static.cnbetacdn.com/article/2022/0118/21c00b0b735e564.webp" style referrerpolicy="no-referrer"></p><p><img src="https://static.cnbetacdn.com/article/2022/0118/700e10751c235c9.webp" style referrerpolicy="no-referrer"></p><p>这一发现是由<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>泄密者FireCube（Twitter上的@FireCubeStudios）发现的，他注意到WhatsApp Beta应用的新旧版本之间的差异。你会注意到圆形的单选按钮、下拉菜单，以及更多符合 Windows 11 设计语言的元素。也就是说，这绝对是一项正在进行的工作，仍然有一些 UI 元素与 Windows 11 不一致，但这是测试版的性质，所以可以理解。</p>
