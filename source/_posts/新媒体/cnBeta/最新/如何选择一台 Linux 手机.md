虽然Android是基于 Linux 内核的，但它经过了大量修改。因此，这意味着它不是一个完全意义上的基于 Linux 的操作系统。Google正在努力使Android内核更接近主线 Linux 内核，但这仍然是一个遥远的梦想。<br>
 <p>那么，在这种情况下，如果你正在寻找一款 Linux <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>、一款由 Linux 操作系统驱动的智能手机，有哪些可以选择呢？</p><p>这并不是一个容易做出的决定，因为你的选择非常有限。因此，我试图推荐一些最好的、不同于主流选择的 Linux 手机。</p><p>如今你可以使用的顶级 Linux 手机</p><p>值得注意的是，这里提到的 Linux 手机或许无法取代你的Android或 iOS 设备。因此，在做出购买决定之前，请确保你做了一些背景研究。</p><p><strong>注意：</strong> 你需要仔细检查这些 Linux 手机是否可以购买到、预期的发货日期和使用风险。它们大多数只适合于发烧友或早期试用者。</p><p>1、PinePhone</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/0824/a01c916c0e3601b.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/0824/a01c916c0e3601b.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/0824/a01c916c0e3601b.jpg" referrerpolicy="no-referrer"></a><br></p><p><a href="https://www.pine64.org/pinephone/" rel="external nofollow" target="_blank">PinePhone</a> 是最有性价比和最受欢迎的选择之一，我觉得它是一个有前途的 Linux 手机。</p><p>它并不局限于单一的操作系统。你可以尝试使用带有 Plasma mobile OS 的 Manjaro、UBports、Sailfish OS 等系统。PinePhone 的配置不错，它包括一个四核处理器和 2G 或3G 的内存。它支持使用可启动的 microSD 卡来帮助你安装系统，还可选 16/32GB eMMC 存储。</p><p>其显示屏是一个基本的 1440×720p IPS 屏幕。你还可以得到特殊的隐私保护，如蓝牙、麦克风和摄像头的断路开关。</p><p>PinePhone 还为你提供了使用六个可用的 pogo 引脚添加自定义的硬件扩展的方式。</p><p>其基本版（2GB 内存和 16GB 存储）默认加载了 Manjaro，价格为 149 美元；而融合版（3GB 内存和 32GB 存储）价格为 199 美元。</p><p>2、Fairphone</p><p><img src="https://static.cnbetacdn.com/article/2021/0824/d16299ba7aa3330.png" referrerpolicy="no-referrer"><br></p><p>与这个清单上的其他选择相比，<a href="https://shop.fairphone.com/en/" rel="external nofollow" target="_blank">Fairphone</a> 在商业上是成功的。它不是一款 Linux 智能手机，但它具有定制版的Android系统，即 Fairphone OS，并且可以选择 <a href="https://itsfoss.com/open-source-alternatives-android/" rel="external nofollow" target="_blank">开源Android系统替代品</a> 之一 <a href="https://itsfoss.com/e-os-review/" rel="external nofollow" target="_blank">/e/ OS</a>。如果你想使用 Linux 操作系统，也有一些社区移植版本，但可能有点碰运气。</p><p>Fairphone 有两个不同的版本，提供了一些不错的配置规格。你会发现 Fairphone 3+ 有一个 4800 万像素的相机传感器和一个全高清显示屏。另外，你还会发现先进的高通处理器为该设备提供了动力。</p><p>他们专注于制造可持续发展的智能手机，并使用了一定量的回收塑料制造。这也为了方便维修。</p><p>因此，它不仅是一个非主流智能手机的选择，而且如果你选择了它，你也将为保护环境出了力。</p><p>3、Librem 5</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/0824/22d7beb0c689899.png"><img data-original="https://static.cnbetacdn.com/article/2021/0824/22d7beb0c689899.png" src="https://static.cnbetacdn.com/thumb/article/2021/0824/22d7beb0c689899.png" referrerpolicy="no-referrer"></a><br></p><p><a href="https://puri.sm/products/librem-5/" rel="external nofollow" target="_blank">Librem 5</a> 是一款非常注重用户隐私的智能手机，同时它采用了开源的操作系统，即 PureOS，并非基于Android。</p><p>它所提供的配置规格还不错，有 3GB 内存和四核 Cortex A53 芯片组。但是，这无法与主流选择相竞争。因此，你可能不会觉得它物美价廉。</p><p>它的目标是那些对尊重隐私的智能手机感兴趣的发烧友。</p><p>与其他产品类似，Librem 5 也专注于通过提供用户可更换的电池使手机易于维修。</p><p>在隐私方面，你会注意到有蓝牙、相机和麦克风的断路开关。他们还承诺了未来几年的安全更新。</p><p>4、Pro 1X</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/0824/e1491b0fd4033de.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/0824/e1491b0fd4033de.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/0824/e1491b0fd4033de.jpg" referrerpolicy="no-referrer"></a><br></p><p><a href="https://www.fxtec.com/pro1x" rel="external nofollow" target="_blank">Pro 1X</a> 是一款有趣的智能手机，同时支持 Ubuntu Touch、Lineage OS 和Android。</p><p>它不仅是一款 Linux 智能手机，而且是一款带有独立 QWERTY 键盘的手机，这在现在是很罕见的。</p><p>Pro 1 X 的配置规格不错，包括了一个骁龙 662 处理器和 6GB 内存。它还带有一块不错的 AM<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_88234%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">OLED</a> 全高清显示屏。</p><p>它的相机不是特别强大，但在大多数情况下应该是足够了。</p><p>5、Volla Phone</p><p><img src="https://static.cnbetacdn.com/article/2021/0824/218cc86dcbd1eb3.jpg" referrerpolicy="no-referrer"><br></p><p><a href="https://www.indiegogo.com/projects/volla-phone-free-your-mind-protect-your-privacy#/" rel="external nofollow" target="_blank">Volla Phone</a> 是一个有吸引力的产品，运行在 UBports 的 Ubuntu Touch。</p><p>它配备了预制的 “虚拟专用网络” ，并专注于简化用户体验。它的操作系统是定制的，因此，可以快速访问所有重要的东西，而无需自己组织。</p><p>它的配置规格令人印象深刻，包括了一个八核联发科处理器和 4700 毫安时的电池。你会得到类似于一些最新的智能手机上的设计。</p><p>总结</p><p>Linux 智能手机不是到处都能买到的，当然也还不适合大众使用。</p><p>因此，如果你是一个发烧友，或者想支持这种手机的发展，你可以考虑购买一台。</p><hr><p>via: <a href="https://itsfoss.com/linux-phones/" rel="external
