近日发生的技嘉数据泄露事件，已被曝光与多个芯片合作厂商相关的机密资料。<strong>以 AMD 为例，泄露文件中就提到了即将推出的锐龙（Ryzen）线程撕裂者（Threadripper）5000 系列处理器的阵容。</strong>可知基于 Zen 3 CPU 架构的 Threadripper 5000 系列处理器，将至少包括 8 种不同核心数（最高 64 核）的 SKU 。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0819/5bbabccacabe767.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0819/5bbabccacabe767.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/amd-ryzen-threadripper-5000-hedt-zen-3-desktop-cpus-detailed-64-cores-280w-tdp-trx40-wrx80/" target="_self">WCCFTech</a>）</p><p>文件还表明，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 将分别推出面向工作站 / 高端台式机（HEDT）的 WRX80 / TRX40 平台，且其热设计功耗（TDP）与核心数，都与上一代 Zen 2 阵容保持一致。</p><p>五款 Threadripper Pro 工作站芯片，包含了 64 / 32 / 24 / 16 / 12 核心的版本，而三款 HEDT 芯片则是 64 / 32 / 24 核心的版本。</p><p><a href="https://static.cnbetacdn.com/article/2021/0819/622a1c6f6e71fc0.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0819/622a1c6f6e71fc0.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>I/O 总线方面，线程撕裂者 5000 系列工作站处理器将提供 8 通道内存控制器 + 128 条 PCIe 4.0 链路，而 HEDT 平台则限制在了 64 条 PCIe 4.0 链路。</p><p>若一切顺利，AMD 将于 2021 年 11 月正式推出上述 Ryzen Threadripper 5000 系列 CPU SKU 。</p>
