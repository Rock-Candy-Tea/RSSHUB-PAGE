<strong>早些时候，苹果推出了 6.1.16 版 Boot Camp 软件，旨在取代之前修复蓝牙与 Wi-Fi 稳定性稳定的 6.1.12 版驱动程序更新。</strong>MacRumors 指出，Boot Camp 可帮助 Intel Mac 用户运行 Windows 操作系统。由发行说明可知，6.1.16 版本引入了对 Wi-Fi WPA3 的支持，并且修复了从睡眠 / 休眠状态恢复时，偶尔可能遇到的蓝牙驱动 bug 。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0823/4923d9eed518633.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（via <a href="https://www.macrumors.com/2022/08/22/apple-boot-camp-wifi-update/" target="_self">MacRumors</a>）</p><p>事实上，早在今年 3 月，<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>就已面向使用 Studio Display 的 Intel Mac 用户推送了 Boot Camp 6.1.17 更新。</p><p>有需要的朋友，可移步至 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 中的“开始菜单”，并打开“苹果软件更新”应用程序来获取新版 Boot Camp 驱动程序。</p><p>遗憾的是，在全面转向自研的 Apple Silicon ARM 芯片之后，苹果官方已不再提供 Boot Camp，意味着用户只能借助虚拟化软件来运行 Windows 。</p>
