自 OS X 10.0.0 版本开始，PHP 就内置在系统中。在默认的 web 服务器中启用 PHP， 只需将 Apache 配置文件 httpd.conf 中的几行配置指令最前面的注释符号去掉，而 CGI 或 CLI 默认都可使用。<strong>而在 macOS Monterey 中将不再内置 PHP，在 Mac 平台上使用该编程语言的开发人员今后必须要自己安装。</strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0618/2d6704cc10c8b1c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0618/2d6704cc10c8b1c.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">苹果公司在系统镜像中取消了 PHP 的内置，在默认情况下 macOS Monterey 的终端中无法再识别 PHP 命令。虽然 Mac 系统内置 PHP 多年，但并非使用的是最新版本。在 macOS Big Sur 中就已经跳出过信息，表示在未来的系统更新中将删除 PHP。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0618/b26c21a2b4ff8d6.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">在 macOS 系统中，PHP 并非是唯一集成的编程语言，在 macOS Monterey 中仍然配备了 2.7.16 的 Python 环境，和 Big Sur 一样。不过 macOS 未来应该并不会像 PHP 一样移除 Python，而是会升级到 Python 3。</p>
