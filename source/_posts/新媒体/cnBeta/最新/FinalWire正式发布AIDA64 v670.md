<strong>FinalWire Ltd. 刚刚发布了 AIDA64 这款系统诊断 / 基准测试工具软件的 6.70 正式版。</strong>此外针对不同的应用场景，该公司提供了面向消费者的 AIDA64 Extreme、面向企业 IT 技术人员 / 工程师的 AIDA64 Engineer、面向中小企业网络管理的 AIDA64 Business、以及用于企业网络审计的 AIDA64 Network Audit 细分产品线。<br>
<p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0412/ea9e7849b8aaf8e.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>本次更新为 AIDA64 添加了对<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> Jasper Lake / Alder Lake-N 处理器和 LGA1700 主板的基准优化，并且支持 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> / Intel CPU 平台和 AMD / Nvidia 的新图形和 GPGPU 计算技术。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0412/dc58e77e770bae9.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>适用于 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> PC 的 AIDA64 产品线，包括了三个商业版 + 一个家庭版。</p><blockquote><p>● 首先是具有硬件诊断功能的旗舰 IT 资产管理产品 AIDA64 Business 。</p><p>● 其次是许可费用较低的专用网络库存解决方案 AIDA64 Network Audit 。</p><p>● 然后是专为企业工程师 / IT 技术人员设计的 AIDA64 Engineer 。</p><p>● 以及面向普通消费者的 AIDA64 Extreme 。</p></blockquote><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0412/1cfe883daa59bb1.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>今天，上述四条 AIDA64 细分产品线都已更新至 6.70 版本。如果你在使用英特尔 Alder Lake-N CPU，现可用它来执行 AVX2 优化的基准测试，此外 Jasper Lake 和 Lakefield 平台亦可执行 SSE 加速的基准测试。</p><p><img src="https://static.cnbetacdn.com/article/2022/0412/d42e333dae476ad.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>AIDA64 6.70 还改进了对 Crystalfontz CFA 系列 LCD 屏的支持，优化英特尔 Skylake-X、Cannon Lake、Ice Lake、Tiger Lake、Rocket Lake、Alder Lake 和 Raptor Lake 处理器的 AVX-512 压力测试。</p><p><a href="https://static.cnbetacdn.com/article/2022/0412/edd05ff939ab723.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0412/edd05ff939ab723.jpg" alt="5.jpg" referrerpolicy="no-referrer"></a></p><p>其它还有支持 WDDM 3.1，可识别 AMD Radeon RX 6500 XT / RX 6400、以及英伟达 GeForce RTX 3090 Ti GPU 的详细信息。</p><p><strong>AIDA64 v6.70 <a href="https://www.aida64.com/downloads" target="_self">官网</a>下载地址：</strong></p><blockquote><p>● AIDA64 <a href="https://www.aida64.com/downloads/YWM3MGQxZWY=" target="_self">Extreme</a></p><p>● AIDA64 <a href="https://aida64.co.uk/download?p=ENGINEER" target="_self">Engineer</a></p><p>● AIDA64 <a href="https://aida64.co.uk/download?p=BUSINESS" target="_self">Business</a></p><p>● AIDA64 <a href="https://aida64.co.uk/download?p=NETWORK_AUDIT" target="_self">Network Audit</a></p></blockquote>
