同 Windows 8 一样，Windows 11 系统也对开始菜单进行了非常大的变动。全新的开始菜单不再支持动态磁贴，虽然改为静态图标，但是不再支持自定义。新版开始菜单在推出后褒贬不一，而其中最令人诟病的一点就是内置广告。<br>
 <p style="text-align: left;">如果你在开始菜单中搜索“Chrome”或者“Firefox”，那么在搜索结果的右侧会跳出一个面板，推荐用户尝试使用 Microsoft Edge 浏览器。目前这种广告仅限于搜索这两款浏览器，Opera、Brave、Vivaldi 等其他浏览器并未出现该广告。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0830/bb37bfdc541afff.webp" alt="3j3jkqu3.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Textio 联合创始人、此前曾担任 Microsoft <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a> 设计师的詹森·哈里斯（Jensen Harris）在推文中，<strong>表达了对<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>这种广告设计的失望。在一系列推文中，哈里斯认为 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 无法达到预期的旗舰开始菜单体验。</strong></p><p style="text-align: left;">他在推文中表示：</p><blockquote style="text-align: left;"><p style="text-align: left;">为什么“Bing Wallpaper app”的横幅广告要采用 Web 1.0 Geocities 时期的风格？老实说，整个页面设计就像是被病毒感染了。文本是错位的，而且背景也采用 Windows Vista 时代的风格。</p><p style="text-align: left;">设计很重要，细节同样很重要。尤其是在像 Windows 开始菜单这样标志性的 UI 中。我记得该团队在 Segoe UI 字体（在 Windows 中使用）中创建了一个特殊的连字，以使“S”和“t”与“Start”一词完美对。这就是开始菜单 对微软的重要性。</p></blockquote><p style="text-align: left;">这位前 Windows 工程师还对“开始/搜索”面板中的“必应壁纸”应用广告以及操作系统的某些区域如何同时使用圆角和锐利边缘有一些不满。</p>
