<strong>基于 Chromium 的浏览器在 Windows 平台将会有更流畅的滚动条体验了。</strong>自 2021 年夏季以来，微软一直在研究 Fluent 和 Overlay 滚动条，并承诺会将该功能添加到 Chromium 中。而在近期的 Chromium Gerrit 中，为添加 Fluent 滚动条做好了准备。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2022/0530/60f1830612e8081.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0530/60f1830612e8081.jpg" alt="QQ截图20220530084355.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">微软在代码提交中写道：“Fluent 滚动条基础功能这个 CL 添加了 Fluent 滚动条基础功能并实现了单元测试。 FluentScrollbar 功能取决于 OverlayScrollbar 功能状态”。这表明新的 Fluent 滚动条将很快在 Chrome 中进行测试（大概从 Canary 开始），但目前还没有 ETA。</p><p style="text-align: left;">对于 Fluent 滚动条，微软表示现代滚动条的灵感来自其流畅的设计，它仅仅意味着浏览器右侧的滚动条将是一条非常细的线。当您将鼠标悬停在栏上时，它会突然出现，从而带来整洁或“现代”的体验。</p><p style="text-align: left;">Google最近发布了 Chrome 版本 102，对网络应用程序、标签等进行了重大改进。作为 Chrome 102 的一部分，Google 添加了新的键盘快捷键来加快您的网络导航。使用新的快捷键“Ctrl+Shift 和 Page Up 或 Page Down”，现在可以在不使用鼠标的情况下重新排列选项卡。</p>
