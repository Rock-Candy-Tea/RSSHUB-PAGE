面向 Dev 频道的 Windows Insider 项目成员，微软今天发布了 Windows 11 Build 25158。该预览版更新隶属于 Sun Valley 3（Windows 11 23H2），正式版将于明年秋季上线。新版本在错误修复的基础上增加了许多新的更改和改进。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0714/c8025172d49a527.webp" alt="333ajxby.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Windows 11 Build 25158 的完整变更日志如下：</p><p style="text-align: left;"><strong>新内容</strong></p><blockquote style="text-align: left;"><p style="text-align: left;"><strong>为 Widgets 新增通知标记</strong></p><p style="text-align: left;">微软开始向您的任务栏推出更多关于 Widgets 的内容动态，并带有 Widgets 的通知标记。下面是 Widgets 通知徽章的示例。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0714/85dd98dd6ce6e17.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">当您打开 Widgets 面板时，板顶部会出现一个横幅，提供有关触发通知徽章的更多信息。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0714/2238f837dab7f36.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">[微软正开始逐渐推出这些功能，因此现阶段并未向所有 Windows Insider 开放。微软计划收集更多的用户反馈，在修改之后向更多用户开放。反馈：请在桌面环境 > 小部件下的反馈中心 (WIN + F) 中提交反馈。]</p></blockquote><p style="text-align: left;"><strong>变化和改进</strong></p><blockquote style="text-align: left;"><p style="text-align: left;"><strong>[常规]</strong></p><p style="text-align: left;">感谢所有提供宝贵建议的 Windows Insider 项目成员，帮助我们更好地在 Windows 桌面上探索轻量级交互内容。自本周的版本开始，我们将结束这个初步探索，所以如果你收到了这个体验，它将在你下次重启时被删除。</p><p style="text-align: left;">如我们在<a href="https://blogs.windows.com/windows-insider/2022/02/03/whats-coming-for-the-windows-insider-program-in-2022/" target="_blank">此处的博客文章</a>中所述，使用 Dev 频道的 Windows Insider 可以尝试到新的想法、更长的领先功能和旨在帮助验证概念的体验。从这个版本开始，一些设备将在任务栏上接受不同的搜索视觉处理。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0714/663ec7136875b3d.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">我们很高兴从您对此的反馈中学习，因此如果您收到此实验，请使用反馈中心提供反馈。注意：更新到此版本后需要重新启动才能触发此体验，但请注意，即使在重新启动后，并非所有 Windows 预览体验成员都会启用此体验。</p><p style="text-align: left;"><strong>[DNS over TLS]</strong></p><p style="text-align: left;">DNS over TLS 测试现在可用于 Windows DNS 客户端查询保护。 DoT 是除 DNS over HTTPS (DoH) 之外的另一种协议，Windows 11 和 Windows Server 2022 已支持该协议。有关设置说明，请参阅此 Windows 网络博客文章。该设置需要使用命令行，建议高级网络用户使用。</p><p style="text-align: left;"><strong>[输入]</strong></p><p style="text-align: left;">Nyala 字体已更新以支持来自 Ethiopic Extended-B Unicode 范围的 Gurage 语言正字法的音节。还修订和替换了埃塞俄比亚语字距调整，以修复早期实施中的差距和错误，改善了埃塞俄比亚语语言文本的整体质感。</p></blockquote><p style="text-align: left;"><strong>以下是 Windows 11 Build 25158 中的错误修复：</strong></p><blockquote style="text-align: left;"><p style="text-align: left;"><strong>[常规]</strong></p><p style="text-align: left;">● 修复了导致一些内部人员在最近的版本中遇到错误检查并在 NetAdapterCx.sys 中出现错误消息 KMODE_EXCEPTION_NOT_HANDLED 的问题。</p><p style="text-align: left;">● 修复了在最近的飞行中从低功耗状态恢复时导致某些 PC 挂在黑屏的问题。</p><p style="text-align: left;">● 修复了一个问题，据信该问题通常会影响最近几次飞行中 Windows shell 体验的性能。</p><p style="text-align: left;">● 修复了云母材料和亚克力模糊效果在最近的飞行中有时在开始菜单、通知中心和其他区域等操作系统表面中渲染不正确的问题。</p><p style="text-align: left;"><strong>[文件管理器]</strong></p><p style="text-align: left;">● 修复了在极少数情况下关闭命令栏弹出按钮可能会导致 explorer.exe 崩溃的问题。</p><p style="text-align: left;">● 修复了在启用 OneDrive 的情况下通过文件资源管理器导航时可能导致 explorer.exe 崩溃的问题。</p><p style="text-align: left;">● CTRL + F4 现在应该可以关闭当前选项卡。</p><p style="text-align: left;">● 在新选项卡中打开压缩文件夹不应再有空白选项卡名称。</p><p style="text-align: left;">● 如果焦点位于选项卡行上，CTRL + W 将不再意外关闭两个选项卡，而不仅仅是焦点所在的选项卡。</p><p style="text-align: left;">● 修复了重新排列选项卡后 CTRL + Tab 无法按正确顺序循环的问题。</p><p style="text-align: left;">● 我们进行了一些调整，因此当使用“在标题中显示完整路径”时，文件夹名称始终在文件资源管理器顶部的选项卡中可见。</p><p style="text-align: left;"><strong>[任务栏]</strong></p><p style="text-align: left;">● 修复了一个潜在问题，该问题可能导致在最近的飞行中卸载应用程序后固定的应用程序图标卡在任务栏上。</p><p style="text-align: left;">● 修复了与加载小部件相关的最后几个航班中的高命中 explorer.exe 崩溃。</p><p style="text-align: left;"><strong>[开始]</strong></p><p style="text-align: left;">● 使用 CTRL + Shift + 右箭头将固定的应用程序移动到作为“开始”中最后一项的文件夹中时，“开始”不应再崩溃。</p><p style="text-align: left;">● 修复了一些 Insider 在打开 Start 时遇到的崩溃问题。</p><p style="text-align: left;">● 讲述人现在将阅读尝试从“开始”正确卸载应用程序时打开的提示。</p><p style="text-align: left;"><strong>[搜索]</strong></p><p style="text-align: left;">● 修复了使用 IRowsetPriorititization 时为查询行集设置范围优先级未正确更改行集的索引速度，从而导致整体索引速度变慢的问题。</p><p style="text-align: left;">● 搜索“电源模式”现在应该会在“设置”中返回电源模式条目。</p><p style="text-align: left;"><strong>[设置]</strong></p><p style="text-align: left;">● 将网络和 Internet > 高级网络设置 > 数据使用下的“系统”条目更新为现在称为“系统和 Windows 更新”以帮助使其更清晰。</p><p style="text-align: left;">● 进行了另一个修复以解决重新启动后音频设备的“收听此设备”选项将停止工作的问题。</p><p style="text-align: left;">● 修复了一些内部人员在导航蓝牙和设备 > 打印机和扫描仪时看到的崩溃。</p><p style="text-align: left;">● 修复了蓝牙和设备 > 设备 > 更多蓝牙设置下的“允许蓝牙设备找到这台电脑”复选框有时不会显示正确状态的问题。</p><p style="text-align: left;">● 修复了在尝试使用 Cast 部分投射到某些设备时导致 ARM64 PC 的快速设置崩溃的问题。</p><p style="text-align: left;">● 修复了可能导致快速设置中 Wi-Fi 密码框旁边的眼睛图标在首次使用后消失的问题。</p><p style="text-align: left;"><strong>[输入]</strong></p><p style="text-align: left;">● 修复了 textinputhost.exe 初始化中的崩溃，该崩溃在最近的飞行中的某些情况下发生。</p><p style="text-align: left;">● 修复了“设置”>“系统”>“剪贴板”下的“跨设备同步”选项无法保持启用的问题。</p><p style="text-align: left;">● 更新了 Times New Roman 字体系列，使其包含 U+061D / ARABIC END OF TEXT MARK。</p><p style="text-align: left;"><strong>[窗口]</strong></p><p style="text-align: left;">● 我们进行了一些改进以帮助提高打开任务视图的性能。</p><p style="text-align: left;"><strong>[讲述人]</strong></p><p style="text-align: left;">● 进行了修复，以便使用讲述人关闭上下文菜单的两指三次轻击触摸手势再次正常工作。</p><p style="text-align: left;"><strong>[任务管理器]</strong></p><p style="text-align: left;">● 修复了某些服务实际上不是子进程时意外分组到 Microsoft Edge 下的问题。</p><p style="text-align: left;"><strong>[其他]</strong></p><p style="text-align: left;">● 修复了开启移动热点后会导致网络连接断开的问题。</p><p style="text-align: left;">● 解决了在某些情况下可能导致移动热点按钮意外卡在灰色状态的问题。</p><p style="text-align: left;">● 修复了在某些情况下导致您的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>崩溃的潜在问题。</p><p style="text-align: left;">● 修复了 Microsoft Defender 应用程序防护窗口在最近几次飞行中不可见的潜在问题。</p><p style="text-align: left;">● 修复了使用建议操作时可能发生的崩溃。</p><p style="text-align: left;">● 修复了使用 <a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> Game Bar 录制游戏剪辑时的一个潜在问题，该问题可能导致稍后在 iOS 设备上查看时无法正确显示所保存的视频。</p><p style="text-align: left;">● 进行了一些改进以帮助解决导致 Microsoft Store 中的应用程序无法下载并出现错误 0x80080204 的问题。</p><p style="text-align: left;">● 解决了导致 Microsoft Store 应用修复失败并出现错误 0x803FB004 的潜在问题。</p><p style="text-align: left;">● 做了一些工作来解决打印对话框未在应用程序中心启动的问题（例如，在记事本中）。</p><p style="text-align: left;">注意：此处在 Dev Channel 的 Insider Preview 版本中提到的一些修复可能会进入 Windows 11 已发布版本的服务更新。</p></blockquote><p style="text-align: left;"><strong>以下是 Windows 11 build 25158 中的已知问题：</strong></p><blockquote style="text-align: left;"><p style="text-align: left;"><strong>[常规]</strong></p><p style="text-align: left;">● [新] 我们正在调查有关 SQL Server Management Studio 无法为某些内部人员启动的报告。</p><p style="text-align: left;">● 某些使用 Easy Anti-Cheat 的游戏可能会崩溃或导致您的 PC 出现错误检查。</p><p style="text-align: left;"><strong>[文件管理器]</strong></p><p style="text-align: left;">● 文件资源管理器选项卡中的向上箭头未对齐。这将在未来的更新中修复。</p><p style="text-align: left;">● 我们正在努力解决以下报告：在使用暗模式（例如，从命令行）时以某些方式启动文件资源管理器会在亮模式下意外显示文件资源管理器的主体。</p><p style="text-align: left;"><strong>[Widgets]</strong></p><p style="text-align: left;">● [新] 通知徽章编号可能会在任务栏上显示未对齐。</p><p style="text-align: left;">● [新] 在某些情况下，某些徽章的通知横幅不会出现在小部件板上。</p><p style="text-align: left;">● 我们正在修复导致小部件首选项（温度单位和固定小部件）意外重置为默认值的问题。</p><p style="text-align: left;"><strong>[Live captions]</strong></p><p style="text-align: left;">● 某些全屏应用（例如视频播放器）会阻止实时字幕显示。</p><p style="text-align: left;">● 某些位于屏幕顶部附近并在运行实时字幕之前关闭的应用程序将在位于顶部的实时字幕窗口后面重新启动。在应用程序具有焦点时使用系统菜单（ALT + 空格键）将应用程序的窗口进一步向下移动。</p></blockquote>
