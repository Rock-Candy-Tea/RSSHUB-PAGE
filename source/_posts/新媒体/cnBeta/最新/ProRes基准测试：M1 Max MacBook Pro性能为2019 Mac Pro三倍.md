在苹果推出了搭载 M1 Pro / M1 Max 自研芯片的 2021 款 MacBook Pro 新品之后，人们再次被 Apple Silicon 平台的优异性能与电池续航表现所折服。当然，将不同芯片架构的 M1 Mac 与 x86 Windows PC 进行对比，或许是不大合适的。<strong>于是近日，又有外媒决定将 M1 Max 新机与 2019 款 Mac Pro 工作站进行一番对比，且着重于“ProRes 导出”这项性能基准。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1217/b1a77f0922dd1b3.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://www.macworld.com/article/559816/prores-codecs-apple-silicon-mac-performance.html" target="_self">MacWorld</a>）</p><p>结果惊喜地发现，M1905 Max <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">MacBook</a> Pro 笔记本电脑的 ProRes 导出速度（时间越短越好），是 2019 款 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">Mac Pro</a> 的三倍左右（76 vs 233 秒）。</p><p><img src="https://static.cnbetacdn.com/article/2021/1217/f139c5c4b8b0313.png" alt="2.png" referrerpolicy="no-referrer"></p><p>此外就算为 Intel（x86）Mac Pro 用上了 Afterburner 加速卡，M1 Max 平台依然有两倍于前者的性能（76 vs 153 秒）。</p><p><a href="https://static.cnbetacdn.com/article/2021/1217/a9d431b0a02063e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1217/a9d431b0a02063e.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>据悉，本次测试选用的 2019 款 Mac Pro，搭载了 28 核 / 56 线程的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>至强 W 处理器 + 两块强大的 GPU，且尝试搭配 Afterburner 加速卡来改善 ProRes 播放与解码体验，但这么做的代价也极其高昂。</p><p><img src="https://static.cnbetacdn.com/article/2021/1217/c04fbc92613fa1c.jpg" alt="5.jpg" referrerpolicy="no-referrer"></p><p>相比之下，M1 Max 芯片集成了两组 ProRes 编解码器，远胜于 Mac Pro 上的 Afterburner 加速卡中的一套解码器。</p><p>而且在价格方面，入门 / 顶配型号的 2021 款 MacBook Pro，也只需 2499 / 6100 美元（国行 14999 / 49295 RMB）。</p>
