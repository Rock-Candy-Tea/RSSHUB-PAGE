Windows 11 即将在今年晚些时候推出，微软已经确认只有符合条件的设备，才可以通过 Windows Update 来获得新的操作系统。如果你的设备并不兼容 Windows 11 系统，微软表示你可以通过多媒体创建工具来安装新系统，但并不推荐这样操作。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0830/5da6cf5609145c8.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0830/5da6cf5609145c8.jpg" alt="52kn4p0j.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">微软此前公示了 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 系统的兼容性设备的详细清单，基本上<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>第八代、Ryzen 2000 及更新处理器可以兼容支持新系统。更新的列表还包括对部分第七代处理器的支持，如酷睿X系列、至强W系列和酷睿i7-7820HQ芯片。</p><p style="text-align: left;">不幸的是，至少现在不会有更多的芯片加入到这个名单中。微软还确认，他们自己的<a data-link="1" href="https://microsoft.pvxt.net/9W473" target="_blank">Surface</a>产品，如Surface Laptop，不会升级到Windows 11。对Surface产品不会有任何特殊待遇，因为只有较新的机型才会被允许安装新的 Windows。</p><p style="text-align: left;">如果你不知道你当前所用的设备是否支持升级新系统，你可以使用 PC Health Check Tool 来检查兼容性。不过在 Release Preview 频道上，微软已经在 Windows Update 页面添加了一个通知，并揭示了更多情况。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0830/cba7b5426732f78.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0830/cba7b5426732f78.jpg" alt="Windows-11-compatibility-message.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在截图中可以看到，那些可以升级到 Windows 11 的兼容设备上会有一条提示：“好消息--你的 PC 符合 Windows 11 的最低系统要求。在我们为您准备好的过程中，何时提供的具体时间可能有所不同。注意：一些Windows 10的功能在Windows 11中是不可用的。某些应用程序和功能可能有额外的”。</p><p style="text-align: left;">它还链接到相关的支持文章，其中包括关于Windows 11要求和被删除功能的更多信息。这是一个有用的步骤，特别是如果你不熟悉Windows 11的要求。微软目前正在向Windows Insider计划的发行预览频道的测试者推出上述兼容性信息。</p>
