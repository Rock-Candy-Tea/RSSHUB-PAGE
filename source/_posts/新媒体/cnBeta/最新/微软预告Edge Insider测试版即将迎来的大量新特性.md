微软刚刚在 Edge Insider 网站上的“What's Next”页面，预告了测试者们即将迎来的大量新特性。<strong>从 PDF 文档的处理、到选项卡分组功能的变化，到商店和某些政策的调整，新路线图中都给出了明确的规划。</strong>至于这些新功能将于何时抵达稳定版通道，目前暂不得而知。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0713/2ca1e2d541794a4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0713/2ca1e2d541794a4.jpg" referrerpolicy="no-referrer"></a></p><p>（1）首先，从 Edge 94 版本开始，我们将能够向 PDF 文档中添加自由表单文本框，且用户可借此添加可见的注释。</p><blockquote><p>与此同时，PDF 查看和标记体验也将迎来改进，用户可添加自由形式的荧光笔，以凸显 PDF 上用户无权评论的部分、以及扫描的文档。</p></blockquote><p>（2）其次，从 Edge 97 版本开始，用户可选择在渐进式 Web 应用（简称 PWA）中启动浏览器，而无需调取完整版的本机浏览器。</p><blockquote><p>此外从 Edge 94 版本开始，来自 PWA 应用的通知，也将不再显示为来自 Microsoft Edge 浏览器，而是报告显示 PWA 应用程序的名称。</p></blockquote><p>（3）从 Edge 95 版本开始，现代浏览器 / IE 模式之间的体验过渡，也将得到改善。引用标头、发布数据、表单数据和请求方法，都将在两种体验之间被正确转发。</p><p>（4）从 Edge 93 版本开始，用户将能够更轻松地进入画中画（PiP）模式。当将鼠标悬停在支持的视频上时，将可看到一个工具栏，允许用户在 PiP 窗口中查看视频。</p><blockquote><p>不过为了保持用户的注意力，从 Edge 92 版本开始，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>已媒体的自动播放设置，从默认的允许更改为受限。</p></blockquote><p>（5）使用微软家长管理功能的用户，将很快看到内容过滤器方面的新改进（支持过滤 YouTube 等网站上的视频内容），以防止儿童意外导航到不适宜的视频。</p><blockquote><p>此外通过启用‘仅允许访问许可的站点’模式，家长将能够看到一个选项，以便将<a data-link="1" href="https://c.duomai.com/track.php?k=yUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmYiJ05WZkVHdzZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyUiR" target="_blank">教育</a>类网站清单一键添加到白名单中。</p></blockquote><p>（6）从 Edge 93 版本开始，微软将启用选项卡分组功能。通过自定义选项，用户能够高效地跨多个工作流查找、切换和管理选项卡。</p><p>（7）从 Edge 93 版本开始，用户可转到 edge://settings/appearance 设置页面，并在‘自定义工具栏’项目下选择‘隐藏标题栏’。</p><p>（8）从 Edge 93 版本开始，浏览器可在用户访问一个基于未知语言的网页时，自动将相关内容翻译成系统的首选语言。如有需要，用户亦可点击显示原页面、或禁用自动翻译选项。</p><p>（9）有趣的是，Edge Insider 测试者还能够在浏览购物网站时，更轻松地了解特定商品的历史价格，目前这项功能已支持包括亚马逊在内的 9 家线上零售商。</p><p>（10）从 Edge 95 版本开始，微软已改善了浏览器收藏夹的使用体验。对于学生和研究人员来说，新版收藏夹将支持英文和阅读列表。</p><p>（11）从 Edge 94 版本开始，企业管理员将能够在初始化阶段迎来更灵活的配置选项。包括指定起始页面、添加 / 删除其它页面、以及从上次离开的地方继续浏览等，同时确保策略指定的页面集合也能够被打开。</p><p>（12）最后，得益于初始首选项（Initial Preferences）的引入，从 Edge 93 版本开始，微软将让企业部署变得更加轻松。</p>
