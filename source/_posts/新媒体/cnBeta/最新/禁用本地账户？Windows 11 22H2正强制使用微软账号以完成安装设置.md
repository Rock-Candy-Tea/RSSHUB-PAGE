Windows Latest 报道称：<strong>微软似乎很想砍掉使用本地账户来配置 Windows 11 设备的选项，即使你想要安装的是所谓的“专业版”。</strong>此前该公司已针对 Windows 11“家庭版”用户剔除了使用本地账户的选项，并迫使大家使用微软账号来设置他们的设备。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0509/cad214dbcf4e841.jpg" referrerpolicy="no-referrer"></p><p>随着新政策的推行，这一限制将不再局限于“家庭版”操作系统用户。从 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 22H2 开始，“专业版”操作系统用户也必须在系统安装设置过程中链接其<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>账户。</p><p>至于 Windows 11“企业版”客户，这家雷德蒙德软件巨头显然暂时还不敢得罪，意味着这部分用户仍可按其所需的方式进行配置。</p><blockquote><p>当连接用户的微软账户时，系统将自动为设备配置 OneDrive 网盘，并开始在您的设备之间同步数据。</p><p>其中包括跨不同设备的自动设置和首选项，以帮助您轻松管理多台 PC 。</p></blockquote><p>即便如此，并不是所有人都希望将微软账户与 Windows 11 相关联、并允许微软访问他们的信息。</p><blockquote><p>据悉，在 Build 22616 编译版本中，专业版操作系统已会在安装过程中强制设置微软账户、且要求具有活动的互联网接入。</p><p>不过这种情况仅会在将设备配置为个人使用时才会出现，如果你选择工作或学校账户选项，则系统会提示您改用组织的域配置。</p></blockquote><p>如需绕过这一强制性要求，目前仍有缓解方案 —— 就算会导致报错，您仍可点击“下一步”并创建一个本地账户（让人厌烦，但无奈也是最简单的方法）。</p><p>至于 Windows 10 用户，只要机器在安装过程中不联网、就仍可设置本地账户。相比之下，如果想要绕过 Windows 11 的强制性联网要求，实际操作过程还是过于复杂（需编辑操作系统文件）。</p>
