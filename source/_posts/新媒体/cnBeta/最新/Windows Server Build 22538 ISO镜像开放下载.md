<a href="https://techcommunity.microsoft.com/t5/windows-server-insiders/announcing-windows-server-preview-build-22538/m-p/3066141" target="_blank"><strong>微软已经发布了下一个 Windows Server 长期服务渠道（LTSC）预览版的新版本</strong></a>，其中包含数据中心版和标准版同时提供桌面体验和服务器核心安装选项。微软以 ISO 格式提供 18 种语言的 Windows Server 长期服务渠道预览版，并以 VHDX 格式提供仅有的英文版本。<br>
