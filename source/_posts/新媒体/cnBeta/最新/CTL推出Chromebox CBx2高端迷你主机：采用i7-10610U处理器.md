得益于低功耗和低成本，物美价廉的 Chromebook 让 Chrome OS 映入了许多人的眼帘。不过近段时间，谷歌一直努力携手 OEM 合作伙伴来提升 Chromebook 和 Chromebox 设备的知名度。<strong>虽然成本价格也有所上升，但 CTL 新推出的 Chromebox CBx2 设备，还是提供了与某些游戏计算机相当的硬件体验。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0609/f2a73bc01a6e3dd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0609/f2a73bc01a6e3dd.jpg" referrerpolicy="no-referrer"></a></p><p>早期 Chrome OS 的市场目标，主要是让 Chromebox 对批量采购的企业和学校更具吸引力，于是选择了性能较弱的中低端硬件 + 充分利用 Web 服务的解决方案。</p><p>但随着时间推移，今日的 Chrome OS 已与之相去甚远，且绝对能够利用当前更强大的硬件，来打造更畅快的使用体验。</p><p>以 CTL Chromebox CBx2 为例，它就采用了<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 10 代 i7-10610U 处理器。尽管没用上最新的 11 代 CPU，但它对 Chromebox 来说已经有些“杀鸡用牛刀”了。</p><p>存储方面，Chromebox CBx2 也提供了高达 64GB DDR4 内存 + 内置 2TB M.2 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 的组合。</p><p>对于想要在 Chrome OS 上托管 Linux 和 Android 环境的开发者来说，这样的选项还是相当值得考虑的。</p><p>如果觉得 1419 美元（约 9073 RMB）的价格有些难以承受，CTL 也提供了 729 美元（约 4661 RMB）的低配选项。</p><p>在处理器维持不变的同时，存储部分降级到了 8GB RAM + 128GB eMMC ROM 。</p>
