<strong>多年来，Instagram 一直将体验的重心放在智能手机上，但没有同步扩展到平板电脑和 Web 浏览器。</strong>后者不仅缺少许多功能，甚至来简单的发布操作都难以实现。有趣的是，近日曝光的内部功能测试表明，Instagram 或很快改变这种尴尬的现状。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0517/7763f0bca5b915e.png" alt="1.jpg" referrerpolicy="no-referrer"></p><p>由开发者 Alessandro Paluzzi 挖掘到的代码可知，这家 Facebook 子公司正努力为 Web 平台添加照片和视频的上传功能。</p><p><a href="https://static.cnbetacdn.com/article/2021/0517/f5d96181e5efa92.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0517/f5d96181e5efa92.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>如图所示，Web 浏览器上的发帖（New Post）界面表明其支持传统的文件路径选择、以及鼠标拖拽操作。</p><p><a href="https://static.cnbetacdn.com/article/2021/0517/4448baa0f70ba53.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0517/4448baa0f70ba53.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>选中要上传的图片或视频之后，Instagram 还将允许用户在 Web 界面上执行一些简单的编辑操作，比如按比例裁剪和套用滤镜。</p><p><a href="https://static.cnbetacdn.com/article/2021/0517/ea10376ba0c073d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0517/ea10376ba0c073d.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>对于需要经常撰写文字内容和编辑图片的 Instagram 用户来说，PC 浏览器上的交互效率，还是比 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a> 等移动设备要高不少。</p><p><a href="https://static.cnbetacdn.com/article/2021/0517/ee708f1f3990acd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0517/ee708f1f3990acd.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>遗憾的是，目前尚不知晓 Instagram 是否有意为该平台正式添加 Web 编辑 / 上传功能。</p>
