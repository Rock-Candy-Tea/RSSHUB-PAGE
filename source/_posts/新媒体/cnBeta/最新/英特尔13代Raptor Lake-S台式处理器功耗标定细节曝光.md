<div>   
上周，Coelacanth-dream 披露了与英特尔 12 代 Alder Lake-P 和 Alder Lake-M 系列移动处理器有关的功率标定细节。<strong>现在，Igor's Lab 又分享了英特尔 13 代 Raptor Lake-S 台式处理器的功耗，可知其被主要被划分为 125W、65W 和 35W 三档。</strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0312/efcd44585785da0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0312/efcd44585785da0.jpg" referrerpolicy="no-referrer"></a></p><p>尽管与 12 代相对变化不大，但泄露文件还是揭示了一些有趣的内容。</p><p>按照计划，Raptor Lake-S 台式 CPU 将于 2022 年取代 Alder Lake-S 产品线，并将与 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 的 Raphael 处理器展开直接竞争。</p><p>后者基于 5nm 工艺节点和全新的 Zen 3 CPU 内核，有望在消费级市场掀起一场相当激烈的竞争。</p><p><a href="https://static.cnbetacdn.com/article/2021/0816/5c40fa62b1cf24d.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0816/5c40fa62b1cf24d.png" alt="1-1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图自：<a href="https://www.igorslab.de/en/power-consumption-and-load-peaks-intels-coming-alder-lake-s-and-raptor-lake-s-cpus-in-comparison-exclusively/" target="_self">Igor's Lab</a>）</p><p>如果爆料靠谱，英特尔 13 代 Raptor Lake-S 台式 CPU 将主要划分为 125W 的 -K 系列、65W 的标准版、以及 35W 的低功耗版本。</p><p><a href="https://static.cnbetacdn.com/article/2021/0816/7e3d7aefd3b4881.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0816/7e3d7aefd3b4881.png" alt="1-2.png" referrerpolicy="no-referrer"></a></p><p>据说 Raptor Lake-S 的一个小变化，就是更主动的 PL4 能效调教，以使芯片能够达成更高的性能、效率和内核频率。</p><p><a href="https://static.cnbetacdn.com/article/2021/0816/a948f28ec7156b3.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0816/a948f28ec7156b3.png" alt="1-3.png" referrerpolicy="no-referrer"></a></p><p>PL4 功耗门限的 Tau <=10ms，因而能够在某些工作负载期间达到尖峰。即便如此，PL4 还是会触发电源的过流保护。因为在此场景下，它会达到平台允许的最高瓦数和安培数。</p><p><img src="https://static.cnbetacdn.com/article/2021/0816/73680166e54b753.png" alt="2-0.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">英特尔 Raptor Lake-S CPU 电源标定（截图 via <a href="https://wccftech.com/intel-13th-gen-raptor-lake-alder-lake-s-desktop-cpu-power-requirements-revealed/" target="_self">WCCFTech</a>）</p><p>根据功率要求，125W 版本的英特尔 Raptor Lake-S CPU 的 PL1 / PL2 / PL4 功耗限值为 125W / 188W / 238W，性能模式下为 125W / 253W / 314W 。</p><p><a href="https://static.cnbetacdn.com/article/2021/0816/08c8b1c616e23b2.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0816/08c8b1c616e23b2.png" alt="3-1.png" referrerpolicy="no-referrer"></a></p><p>由于引入了新的反应操作，英特尔 13 代桌面处理器的 PL4 标定略低，但 PL2 还是较 12 代 Alder Lake 略有上升（253W vs 241W）。</p><p><a href="https://static.cnbetacdn.com/article/2021/0816/5805e1a46c9bd17.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0816/5805e1a46c9bd17.png" alt="3-2.png" referrerpolicy="no-referrer"></a></p><p>65W CPU SKU 的情况也与之类似，PL1 / PL2 / PL4 功耗限值为 65W / 133W / 179W，性能模式下为 65W / 219W / 277W 。</p><p>至于 35W CPU SKU，其 PL1 / PL2 / PL4 功耗限值为 35W / 80W / 118W，性能模式下为 35W / 106W / 152W 。</p><p><a href="https://static.cnbetacdn.com/article/2021/0816/78e0c5d0305fe7e.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0816/78e0c5d0305fe7e.png" alt="3-3.png" referrerpolicy="no-referrer"></a></p><p>最后，预计英特尔 Raptor Lake CPU 将于 2022 年推出，并将支持现有的 LGA 1700 插槽和 600 / 700 系列芯片组平台，辅以 DDR5 内存、核心数量、以及 IPC 性能等总体改进。</p><div
