<strong>为帮助应用程序开发者在旗舰设备上测试他们的应用程序，三星已将最新推出的 Galaxy S22 和 Galaxy Tab S8 系列添加进了“远程测试实验室”项目。</strong>据悉，RTL 服务为有需要的开发者们，提供了一种连接到云端真实 Galaxy 设备来测试其应用程序的方法。这意味着广大开发者能够在正式推出前，就针对 App 的性能体验进行微调。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0308/e132d36ba701f22.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（传送门：Samsung <a href="https://developer.samsung.com/remotetestlab/deviceList.action?os=101#550" target="_self">Remote Test Lab</a>）</p><p>三星会定期向 RTL 云端添加新设备，比如去年，该公司就及时引入了 Galaxy S21 系列旗舰智能机、以及 Galaxy Z Fold3 和 Galaxy Z Flip3 折叠屏智能机。</p><p>而今天，我们又不出所料地迎来了 Galaxy S22 / S22+ / S22 Ultra、Galaxy Tab S8 / S8+ / S8 Ultra 。</p><p>使用该服务，开发者现可在云端 Galaxy S22 / Tab S8 系列设备上远程安装 APK 文件、选择所需测试的语言环境、甚至与团队内其他成员共享测试屏幕界面。</p><p>对于经费有限，或难以买到最新硬件的开发者们来说，RTL 服务可以为其轻松省下动辄数千美元的购置费用。</p><p>但要顺畅体验，你仍需拥有一条可靠的互联网连接、一个免费的三星开发者账户、安装了 Java Web Start 的运行时环境（Java Runtime Environment 7+）。</p><p>感兴趣的开发者们，可借助三星每日免费提供的 20 积分来免费使用 5 小时 RTL 服务。当然，云端测试平台也存在一些限制，比如不支持音频、外设、多点触控、以及调用摄像头。</p>
