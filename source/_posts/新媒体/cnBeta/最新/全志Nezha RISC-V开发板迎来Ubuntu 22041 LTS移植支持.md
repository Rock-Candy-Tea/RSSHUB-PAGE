<strong>Canonical 刚刚宣布，他们已经为全志科技（Allwinner）的 Nezha RISC-V 开发板提供了 Ubuntu 22.04.1 LTS 的移植支持。</strong>据悉，Nezha 是一款采用了全志 D1 SoC 的 AIoT 开发板产品，基于开源的 RISC-V 处理器架构。公司高管称，RISC-V 近年来变得愈加受欢迎，但普及之路仍存在一些障碍。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0823/8f988706bc1777f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0823/8f988706bc1777f.jpg" referrerpolicy="no-referrer"></a></p><p>好消息是，随着 Canonical 提供了 Ubuntu 的移植支持，全志 Nezha 平台有望为广大 RISC-V 开发者和企业提供更多便利。</p><p>据 Canonical 所述，这项工作得到了其与 Allwinner 和 Alibaba 之间的长期合作的推动，尤其是 D1 SoC 整合了后者的“平头哥”（T-Head）64 位芯片。</p><p>需要指出的是，Nezha 用到了 Ubuntu Server 镜像，而不是大多数读者熟悉的桌面分支。</p><p>即便如此，Ubuntu Server 22.04.1 LTS 还是具有与 Desktop 安装非常相似的软件包，辅以相同的五年长期支持（至 2027 年）。</p>
