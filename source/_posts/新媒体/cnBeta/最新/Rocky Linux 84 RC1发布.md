<strong>The Rocky Enterprise Software Foundation (RESF)今天高兴地宣布，Rocky Linux 8.4 候选版本上线。</strong>值得注意的是，新版本并不适合日常主力使用，系统中可能还存在一些 BUG 或问题，遇到问题可以点击这里反馈。Rocky Linux 8.3 RC1 无法升级到 Rocky Linux 8.4 RC1，这意味着它不会被测试也不会被记录。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0607/bcd846e425c98b6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0607/bcd846e425c98b6.jpg" alt="ti61n8qz.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">要下载该版本，请访问<a href="https://rockylinux.org/download" target="_blank">https://rockylinux.org/download</a></p><p style="text-align: left;">更新日志：<a href="https://docs.rockylinux.org/en/rocky/8/release_notes/rocky_linux_8-4_release" target="_blank">https://docs.rockylinux.org/en/rocky/8/release_notes/rocky_linux_8-4_release</a></p><p>在 Red Hat 宣布 CentOS 8 在今年年底停止支持之后，Rocky Linux 和 CloudLinux 的 AlmaLinux 成为了两大主要竞争者。Rocky Linux 继续由 CentOS 创始人 Gregory Kurtzer 等人负责。有了Kurtzer的参与已经为这个早期阶段的Linux发行版项目在发布任何版本之前提供了很大的砝码。</p><p style="text-align: left;"><strong>常见问题</strong></p><p style="text-align: left;"><strong>支持哪些升级途径？</strong></p><p style="text-align: left;">不支持在候选版之间升级，也不支持从候选版升级到稳定版。候选版本不适合在生产中使用，只能用于测试目的。</p><p style="text-align: left;"><strong>在Rocky Linux 8.4 GA之前是否会有Rocky Linux 8.4 RC2？</strong></p><p style="text-align: left;">不会，我们计划在Rocky Linux 8.4 RC1中解决所有报告的错误，然后直接进入GA版本。</p><p style="text-align: left;"><strong>是否会有Rocky Linux 8.3 RC2，或者Rocky Linux 8.3 GA？</strong></p><p style="text-align: left;">Rocky Linux 8.3不会有一个稳定版本。Rocky Linux的第一个稳定版本将是8.4。</p>
