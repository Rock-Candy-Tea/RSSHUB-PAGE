Windows 历来是最受欢迎的桌面操作系统。然而，多年来，它一直面临着来自 Chrome OS 和 iPad OS 的一些激烈竞争。特别是在教育和其他轻度使用场景中，消费者需求主要集中在更便宜、低于 350 美元的设备。<strong>现在，Chromebook 和 iPad 在这些用途上相当受欢迎，这是因为这些系统的用户界面对以前没有使用过传统电脑的移动用户来说非常友好。</strong><br>
 <p style="text-align: left;">现在，微软意图在低端市场增强同 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a> 和 Chromebook 的竞争力。它最近宣布了 Windows 11 SE--一个更轻、更容易使用的 Windows 11 版本，旨在用于<a data-link="1" href="https://c.duomai.com/track.php?k=yUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmYiJ05WZkVHdzZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyUiR" target="_blank">教育</a>用途。科技媒体 Android Authority 撰文对 Windows 11 SE 进行了详细介绍，你想要知道的都在这里。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1207/a56637765d13cfd.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;"><strong>Windows 11 SE 是什么？</strong></p><p style="text-align: left;">Windows 11 SE 是微软的新操作系统，是专门为教育--特别是K-8教室--打造的。微软称其为云优先（cloud first）的操作系统。这使它与Google的 Chrome OS 形成了竞争，后者也在教育领域得到了大量使用。该公司还指出，它被设计成一个负担得起的简单解决方案，使学习更容易获得。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1207/62d775027350876.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">微软官方并没有正式表明 SE 代表着什么。考虑到它在这个版本的 Windows 11 中使用的词汇，可以大胆地假设它要么代表学生版（Student Edition），要么代表简单版（Simple Edition），而前者的可能性更大。</p><p style="text-align: left;"><strong>Windows 11 SE 有哪些功能？</strong></p><p style="text-align: left;">Windows 11 SE 与我们之前在 Windows 中看到的 S Mode 不一样。它们有一些功能差异。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1207/ddaf13deffebdcd.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Windows 11 SE 有自己的一套功能，使其与 Windows 11 不同。虽然其中一些可能是限制，但它们的存在是为了确保环境不受干扰，并且有一定程度的管理员控制。</p><p style="text-align: left;"><strong>以下是 Windows 11 SE 的主要功能：</strong></p><p style="text-align: left;"><strong>● 受控的应用程序安装</strong></p><p style="text-align: left;">前面提到的管理员控制有助于为大量学生大规模部署应用程序和服务，也有助于限制某些应用程序。学校管理层可以选择允许或阻止应用程序。教育机构的IT管理员将通过一个名为Microsoft Intune的后台控制台获得控制权。</p><p style="text-align: left;"><strong>● 全屏应用启动</strong></p><p style="text-align: left;">模仿移动操作系统，为了帮助分心，该操作系统上的应用将默认以全屏模式启动。</p><p style="text-align: left;"><strong>● 简化的用户界面</strong></p><p style="text-align: left;">它带有一些UI变化，如简化的Snap Layouts，让你把两个窗口并排堆叠在一起，而其他一些功能，如Widgets已经被删除。</p><p style="text-align: left;"><strong>● 云优先存储</strong></p><p style="text-align: left;">Windows 11 SE配备了1TB的OneDrive存储空间，并将默认将文件备份到OneDrive。</p><p style="text-align: left;"><strong>● 针对微软服务的优化</strong></p><p style="text-align: left;">Windows 11 SE针对微软服务进行了优化。它预装了Microsoft 365。</p><p style="text-align: left;"><strong>● 更好的电池寿命</strong></p><p style="text-align: left;">这是微软明确做出的承诺，由于 Windows 11 SE 比 Windows 11 更轻，预计它将得到证实。</p><p style="text-align: left;"><strong>● 第三方应用支持</strong></p><p style="text-align: left;">与 S Mode 不同，Windows 11 SE 将允许在微软商店之外安装第三方应用。</p><p style="text-align: left;">微软不会单独出售 Windows 11 SE。围绕 Windows 11 SE 的重新安装也有限制，一旦删除，就无法重新安装。</p><p style="text-align: left;"><strong>相比较 Chrome OS，Windows 11 SE 有哪些优势？</strong></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1207/7b47439409556a3.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;"><strong>● 可管理性</strong></p><p style="text-align: left;">Windows 11 SE 是 Chrome OS 的竞争对手。微软瞄准的是教育市场，重点是管理的可管理性。随着 Windows 11 SE 首先瞄准教育机构，它似乎有能力获得一个坚实的开端，关于将操作系统推向大众市场的问题。</p><p style="text-align: left;">然而，当涉及到实际可用性时，使用这些系统的将是学生。因此，微软对用户界面所做的改变需要足以应对 Chromebooks 的挑战。从纸面上看，它似乎有能力在这方面与 Chrome OS 抗衡。</p><p style="text-align: left;"><strong>● 支持第三方应用</strong></p><p style="text-align: left;">Chrome OS 有一个坚实的应用选择，支持Android应用和Linux应用。另一方面，Windows 11 SE 拥有庞大的Windows应用程序目录。</p><p style="text-align: left;">随着第三方应用程序支持的不断开放，Windows 11 最终可能会拥有一个卓越的应用程序目录。此外，微软可以让管理员安装 Android 和 Linux 子系统。这将意味着 Windows 11 SE 可以访问 Chromebooks 的所有应用，支持的应用比 Chrome OS 更丰富。</p><p style="text-align: left;"><strong>● 售价可能会更低</strong></p><p style="text-align: left;">微软目前似乎也在关注低价设备。虽然大部分的销售来自于价格低廉的设备，所以微软在这方面有正确的想法。</p><p style="text-align: left;">当然，Windows 11 SE 与 Chrome OS 的实际表现如何，我们必须等待和观察。这将归结于微软是否已经设法为教育用途简化了传统操作系统，以及它是否能够促进这个新平台的采用。</p><p style="text-align: left;"><strong>Windows 11 SE 生态发展</strong></p><p style="text-align: left;">Windows 11 SE是相当新的，所以运行它的笔记本电脑还不完全可用。运行 Windows 11 SE 的笔记本电脑将在 2021 年晚些时候和 2022 年初上市。有几个型号已经公布，最引人注目的是微软 Surface Laptop SE，它将在2022年初上市，起价249美元。</p><p style="text-align:
