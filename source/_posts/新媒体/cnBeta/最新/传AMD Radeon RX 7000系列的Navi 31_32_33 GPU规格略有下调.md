近日，多位知名爆料人分享了 AMD 下一代 Radeon RX 7000 系列“Navi 3x”显卡的规格。<strong>虽然尚未得到证实，但据说红队略微下调了“革命性”的 RNDA 3 GPU 的参数。</strong>WCCFTech 指出，Navi 31 在大约 5 个月前流片，并于最近在开发人员日志中被人发现，暗示它将如期在 2022 下半年发布。<br>
<p><a href="https://static.cnbetacdn.com/article/2022/0503/cfb64f02ef61cf1.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0503/cfb64f02ef61cf1.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/amd-next-generation-radeon-rx-7000-series-navi-3132-and-33-updated-gpu-variants-leaked/" target="_self">WCCFTech</a>）</p><p>作为 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 旗下首款基于多芯片（MCM）理念打造的 GPU —— 此前已在 Zen 架构 CPU 上成功运用 —— 红队有望在 Radeon RX 7000 系列中高端产品线上改变行业里的游戏规则。</p><p><img src="https://static.cnbetacdn.com/article/2022/0503/d7336d485440f07.png" alt="1-1.png" referrerpolicy="no-referrer"></p><p>周一的时候，老伙计 @Greymon55 在 Twitter 上分享了一组有趣的数据 —— 12288 / 8192 / 4096 —— 许多人一下就联想到了 AMD RNDA 3 GPU 的规格。</p><p><img src="https://static.cnbetacdn.com/article/2022/0503/b5ab2165d4e6b09.png" alt="2.png" referrerpolicy="no-referrer"></p><p>然后 @Kepler_L2 转推并给出了自己的爆料，将 Navi 31 / 32 / 33 GPU 的规格完整罗列了出来。</p><p><img src="https://static.cnbetacdn.com/article/2022/0503/3fb6f9530c8b417.png" alt="1-2.png" referrerpolicy="no-referrer"></p><p>不过最让人感到好奇的是，旗舰款 AMD Navi 31 MCM GPU 的 WGP 数量、竟然从 60 组被砍到 48 组，此外计算单元（CU）也从 120 精简到了 96 个。</p><p><img src="https://static.cnbetacdn.com/article/2022/0503/c20985488c0f819.png" alt="3.png" referrerpolicy="no-referrer"></p><p>换言之，这意味着 AMD 已将 Radeon RX 7900 XT（暂用名）的流处理器（SP）数量从 15360 削减到了 12288 个。不过最有可能的原因，还是出于芯片良率的考虑（减少浪费）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0503/fb5aa25cf6d6d0e.png" alt="4.png" referrerpolicy="no-referrer"></p><p>与此同时，次旗舰 Navi 32 GPU 的 WGP 数量也从 40 个被精简到了 32 个，CU 数从 80 个被减少到了 64 个 —— 意味着传说中的 Radeon RX 7700 XT 只会拥有 8192 个流处理器（而不是 10240 个）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0503/7dc239c4225a335.png" alt="5.png" referrerpolicy="no-referrer"></p><p>最后，主流 Navi 33 GPU（并非 MCM 版本）的 WGP 数量也从 20 组精简到了 16 组，CU 数从 40 个减少到 32 个 —— 意味着 Radeon RX 7600 XT 的只会拥有 4096 个流处理器。</p><p>【题外话】</p><p>早前传闻称，AMD 首款 RDNA 3 MCM GPU 的算力有望达到 92 TFLOPs 。然而参考最新爆料，假定其频率可达 3 GHz，Navi 31 的单精度性能，也将只有 73 TFLOPs 左右。</p><p>假使 AMD 要维持 92 TFLOPs 的算力，则 RNDA 3 MCM GPU 的主频必须接近 3.74 GHz —— 对于 GPU 来说，这显然是疯狂到让人难以置信的。</p><div
