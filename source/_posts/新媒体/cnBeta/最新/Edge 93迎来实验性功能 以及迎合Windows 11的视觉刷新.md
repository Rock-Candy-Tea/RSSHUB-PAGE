尽管微软已经为 Windows 11 设定了 10 月 5 号这个正式发布日期，但相关功能仍在持续完善中。<strong>比如在 Microsoft Edge 93 中，我们就看到了一些实验性功能，比如菜单和迎合 Windows 11 而设计的全新现代滚动条。</strong>此外 Edge 93 已经启用了类似 Chrome 的标签页分组，旨在帮助用户高效查找、切换和管理标签页。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0906/57f97c1b4ae47f9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0906/57f97c1b4ae47f9.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>Edge 93 还支持从上次中断的地方继续查看 PDF 文档，且能够在鼠标悬停于受支持的视频上方时显示新工具栏，以轻松地在画中画模式中观看视频。</p><p>垂直放置的选项卡，也成为了 edge://settings/appearance 中的一个新选项。启用后，它将隐藏浏览器的标题栏。</p><p>至于 Edge 93 新引入的两项实验性功能，首先是迎合 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 全局视觉风格的更新，其次是 Overlay 滚动条。</p><p><a href="https://static.cnbetacdn.com/article/2021/0906/1d2faeeeb40c20c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0906/1d2faeeeb40c20c.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>前者带来了 Windows11 独有的 Mica 设计元素，可自动将桌面背景色应用于活动窗口，但仅适用于 Edge 浏览器的标题栏部分。</p><p>它还为 Edge 菜单和上下文菜单启用了 Fluent Design 的亚克力（半透明）效果，在系统中启用该标记后，它会调节菜单字体的大小。</p><p>除了字体变化（使之看起来更大一点），我们还会看到一些居中放置的按钮，包括最小化、最大化、以及关闭窗口。</p><p><a href="https://static.cnbetacdn.com/article/2021/0906/53ee3e1e48ed2b9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0906/53ee3e1e48ed2b9.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p style="white-space: normal;">其次，Edge 93 将支持在 Windows 10 / 11 上的 Overlay 滚动条。启用该标记后，浏览器的默认滚动条将变得更加精简，且支持黑暗模式。</p><p style="white-space: normal;">如果你已经在体验 Windows 11 预览测试版本，应该也会发现这项改动旨在与其它系统应用程序（比如设置和资源管理器）的外观相匹配。</p><p>感兴趣的朋友，可更新至 Edge 93 及以上版本，然后手动启用这项适配 Windows 11 的视觉更新标记。</p><p><strong>至于新版滚动条，还请打开 Edge 的属性选项卡，然后添加以下目标字段：</strong></p><blockquote><p>要尝试 Edge 93 中的新滚动条，请打开 Edge 的属性选项卡并将以下代码添加到目标字段：</p></blockquote><p>不过未来几个月，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还有望为 Windows 11 用户带来更多的 Edge 视觉刷新。</p>
