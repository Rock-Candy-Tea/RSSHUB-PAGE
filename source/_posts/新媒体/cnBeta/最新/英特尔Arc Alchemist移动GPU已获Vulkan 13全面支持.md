Khronos Group 于 2022 年初公布了 Vulkan 1.3 API 的规范。<strong>虽然一些 GPU 制造商很快提供了驱动支持，但英特尔还是受到了旗下 Arc Alchemist GPU 硬件跳票的影响。</strong>此前蓝厂已经为 UHD 和 Iris Xe 系列核显提供了对 Vulkan 1.3 的支持，不过近日 Arc A350M / A370M、A550M 和 A730M / A770M 也终于被加入了列表。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0126/dc68db71a40cf47.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://www.khronos.org/conformance/adopters/conformant-products/vulkan#submission_686" target="_self">Khronos Group</a>）</p><p>据悉，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> Xe-HPG 系列图形 IP 主打数据中心、云推理、人工智能等 HPC 应用场景，而 Arc 370M 则是基于 ACM-G11 GPU 的入门级 / 功耗优化产品线。</p><p>Arc A370M 具有 8 组 Xe Core / 1024 ALU 单元、频率 1550 MHz，辅以 64-bit @ 4GB GDDR6 显存、TDP 在 35-50W 之间，对标英伟达 RTX 3050 竞品。</p><p><img src="https://static.cnbetacdn.com/article/2022/0614/a7ddc35eda0fa91.png" alt="2.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">Vulkan 1.3 现在完全支持 Intel Arc Alchemist 移动 GPU</p><p>其次，Arc A350M 具有 6 组 Xe Core / 768 个 ALU 单元、6 个光追单元、主频 1150 MHz，辅以 64-bit @ 4GB GDDR6 显存。定位入门级、TDP 功耗在 25-35W 之间，对标英伟达 MX500 系列移动 GPU 。</p><p>另外 Arc A730M 据说跑分对标英伟达 RTX 3070，但游戏表现却仅略胜 RTX 3050 一筹。最后，我们推测桌面版 Arc Alchemist GPU 也将很快出现在  Khronos Group 的 Vulkan 1.3 支持列表中。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1230227.htm" target="_blank">Vulkan 1.3正式发布：支持核心动态渲染 为现代GPU开发指路</a></p></div>
