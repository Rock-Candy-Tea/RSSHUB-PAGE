根据今年 5 月的 Steam 硬件和软件调查，虽然使用 Linux 的玩家占比数量有所下降，<strong>但是 Arch Linux 超过 Ubuntu 成为最受玩家喜欢的 Linux 发行版本。</strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2022/0602/f56a245795871d2.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0602/f56a245795871d2.jpg" alt="QQ截图20220602103741.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Steam 2022 年 4 月调查结果显示，Linux 增长 0.14% 至 1.14%。 4 月，Ubuntu 20.04 LTS 仍然是最大的发行版，市场份额为 0.16%，Arch Linux 为 0.14%，其次是基于 Arch 的 Manjaro Linux，为 0.13%。</p><p style="text-align: left;">然而在 5 月的调查结果显示，Linux 的整体市场份额略微回落 0.02% 至 1.12%，而 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 占比为 96.68%；macOS 为 2.2%。换句话说，基本持平，而且在 Steam 的整体用户群持续增长的情况下，绝对用户础上可能会更加持平。在这个阶段，在 KDE 桌面体验之外的原生 Steam 模式下在 Steam 平台上运行时，似乎也不会报基告/使用 Steam 调查。</p><p style="text-align: left;">除了总体 Linux 百分比相当平稳之外，相当有趣的是，在过去的一个月里，Arch Linux 64 位超越了 Ubuntu 20.04 LTS，成为最受欢迎的 Linux 发行版。诚然，与 Ubuntu 版本相比，Arch Linux 的滚动版本被计为一个版本，每个版本都被计为不同的版本，但 Manjaro Linux 和其他 Arch Linux 衍生版本也被单独计算。不幸的是，对于 Steam 调查的“其他”分组，很难进行精确的外部计算。</p><p style="text-align: left;">无论如何，它显示 Arch Linux 现在被 0.14% 的 Steam 用户使用，而 Ubuntu 20.04.4 LTS 和 Manjaro Linux 的使用率为 0.13%，为 0.12%。与此同时，Ubuntu 22.04 LTS 现在以其新版本出现在列表中，为 0.09%。或者相对而言，仅计算 Linux 数字时，Arch Linux 为 12.85%，而 Ubuntu 20.04.4 LTS 为 11.75%。 SteamOS Holo 现在也确实出现在 Linux 用户中，占比为 5.23%。</p>
