<strong>至少从 Windows 95 开始，微软都会为自家主流视窗操作系统酝酿一句朗朗上口的宣传语。</strong>比如为了强调“开始”按钮和菜单设计，Windows 95 的 Slogan 就是简洁直观的“Start me up”（从这里开始）。相比之下，我们对 Windows XP 时代的“Experience the excitement”（体验激动时刻）就没有那么印象深刻，更别提 Windows 8 时代的“Everything at Once”（一以贯之）了。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0915/ea32eec4a76649c.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">Windows 似乎陷入了“一代拉胯一代强”的循环</p><p>为了便于大家回顾历代 Windows 操作系统的宣传口号，Reddit 网友 SundryWaif4182 特地拼接了这组图片。</p><p>可知第三位 Windows Vista 的存在感也相当微弱，The“Wow”starts now（惊喜从现在开始）的口号几乎没能给我们留下过任何印象。</p><p>由于设计过于超前，那个年代的主流计算机硬件无法流畅运行 Windows Vista 。但作为 Windows 7 的前身，其中的许多设计理念，还是为日后的发展铺平了道路。</p><p>图片倒数第二为 Windows 10 操作系统的“Do great things”（干大事），但被各种补丁 Bug 折磨多年的广大用户也是颇有怨言。</p><p>至于即将到来的 Windows 11 操作系统能否兑现“Brings you closer to what you love”（让你更接近所爱）的承诺，仍有待时间去检验。</p>
