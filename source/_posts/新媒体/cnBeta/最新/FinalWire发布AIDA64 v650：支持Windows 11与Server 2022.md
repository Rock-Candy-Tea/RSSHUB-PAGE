<strong>FinalWire Ltd. 刚刚发布了 AIDA64 Extreme 系统工具的 6.50 版本，作为一款实用的诊断与基准测试工具，其很适合普通用户使用。</strong>与此同时，该公司还推出了面向工程（Engineer）、商业（Business）、以及网络审计（Network Audit）领域的衍生版本。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1019/5b054ddf53386cf.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>首先，AIDA64 v6.50 版本引入了对 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 操作系统和 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 4700S 处理器的支持。</p><p><img src="https://static.cnbetacdn.com/article/2021/1019/46f5733893f7e36.png" alt="5.png" referrerpolicy="no-referrer"></p><p>现可在 VoCore LCD 屏上监测传感器数值、并且支持最新的 AMD / Intel CPU 和 AMD / Nvidia GPU 。</p><p><img src="https://static.cnbetacdn.com/article/2021/1019/3e767cbc924d68e.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>AIDA64 Extreme / Engineer / Business / Network Audit 版本的售价，分别为 49.95 / 199.90 / 199.90 / 119.90 美元（约合 320 / 1280 / 1280 / 767 RMB）。</p><p><img src="https://static.cnbetacdn.com/article/2021/1019/de4b6c9e347a8ef.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p><strong>以下是 AIDA64 v6.50 的新功能与相关改进：</strong></p><blockquote><p>· 支持 Microsoft Windows 11 与 Windows Server 2022 。</p><p>· 支持 AMD 4700S Cardinal CPU 的 64-bit 基准测试 / AVX2 和 FMA 加速。</p><p>· 改进对 Intel Alder Lake CPU 的支持。</p><p>· 支持 Aqua Computer LeakShield 传感器。</p><p>· 改进了对 ≥ 64 个内核的 CPU 支持。</p><p>· 支持 Matrix Orbital EVE4 和 VoCore LCD 。</p><p>· 增强对 LGA-1700 主板的支持。</p><p>· 支持 VMware Workstation v16 。</p><p>· 支持 Ramaxel AM620 和 AV310 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 。</p><p>· 添加 AMD Radeon RX 6600M / 6700M 系列 GPU 的详细信息。</p><p>· 增加 nVIDIA CMP 50HX / 170HX / GeForce RTX 3000 LHR 系列 GPU 的详细信息。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/1019/bbb2849042602d7.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p>
