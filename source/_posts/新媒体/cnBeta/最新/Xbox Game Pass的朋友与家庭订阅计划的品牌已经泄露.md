本月早些时候，微软开始测试其受欢迎的Xbox Game Pass服务的家庭计划。新的级别允许多达五个成员通过他们的个人微软账户充分享受Xbox
Game Pass Ultimate。对于有多个使用Xbox作为主要平台的游戏玩家的家庭来说，这可能是一个非常诱人的套餐。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0827/ded3a0466567d1e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0827/ded3a0466567d1e.jpg" title alt="1659640601_valencia-announce-485cc9ffac2ae5285eba.jpg" referrerpolicy="no-referrer"></a></p><p>现在，该服务的官方品牌标识可能已经泄露了。意大利网站Aggiornamenti Lumia似乎已经得到了它的官方图片，正如你在下面看到的，它被标记为<a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> Game Pass的"朋友和家人"。</p><p>有趣的是，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>并没有将其作为一个家庭计划来推广，至少在品牌上是这样，也许不仅仅是家庭，微软希望鼓励人们在朋友之间分享。</p><p><img src="https://static.cnbetacdn.com/article/2022/0827/450dd72975a0395.png" title alt="图片.png" referrerpolicy="no-referrer"></p><p>说到费用，微软还没有正式宣布。然而，在对爱尔兰和哥伦比亚的Xbox Insiders的测试中，它提醒早期采用者，他们的Xbox Game Pass Ultimate订阅将被转换为新的订阅层级。一个月的Xbox Game Pass Ultimate会员费为15美元，但它会被转换为18个月的新朋友和家庭层级，这大概相当于后者的25美元/月。不过，爱尔兰的用户能够找到21.99欧元/月的参考价格。</p><p>重要的是要注意，价格和可用性可能因地区而异，而且由于新套餐处于预览阶段，没有什么可以是固定不变的。</p>
