面向 Dev 频道 Windows Insider 项目成员，微软今天推出了 Windows 11 Build 22567。<strong>这个最新预览版带来了一些功能上的修复和改进，包括让 Windows Update 可以使用可再生能源。</strong>这基本上意味着 Windows 11 将尝试在一天中的特定时间进行更新安装，因为这样做可以减少碳排放。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0303/720ca3ed5cce98b.webp" alt="z2u8p6rl.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;"><strong>以下是Windows 11 Insider Preview Build 22567的新内容</strong></p><p style="text-align: left;"><strong>Windows Update利用了更多的可再生资源</strong></p><blockquote style="text-align: left;"><p style="text-align: left;">Windows Update 将尝试在一天中的特定时间安排更新安装，因为这样做能降低碳排放。大多数电网是由多种能源提供的，包括可再生能源和化石燃料。只要有可能，Windows 11 现在将优先考虑在清洁能源（如风能、太阳能和水力）数量较多的时候在后台安装更新。用户总是可以选择立即安装更新，方法是浏览设置> Windows Update，并选择"检查更新"。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0303/3564f916efedbd9.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">这个功能只有在你的电脑插上电源，并且从我们的合作伙伴 electricMap 或 WattTime 获得区域碳强度数据时才会启用。如果启用，用户将在 Windows Update 设置页面上看到下面的文字。我们仍在研究该文本链接的内容--请关注未来的更新。</p><p style="text-align: left;">[我们正在开始推出这项功能，所以现在还不能提供给所有的 Insider 人员。因为我们需要收集监测反馈，并在推送给所有人之前观察它的效果。］</p></blockquote><p style="text-align: left;"><strong>改进设置中的 Microsoft 365 订阅管理</strong></p><blockquote style="text-align: left;"><p style="text-align: left;">去年 10 月，微软在“设置”>“账户”下的“您的微软账户”中引入了订阅管理。这使你能够查看你的 OneDrive 存储，经常性的账单和共享信息，作为你在 Windows 11 中的 Microsoft 365 订阅的一部分。通过今天的新预览版，我们将使你更容易管理你的 Microsoft 365 订阅。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0303/2d2d3892ff4b20a.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">我们正在将设置>账户下的 Microsoft 365 订阅管理体验提升一个层次。正如我们在这里提到的，我们将使用在线服务体验包，在"设置">"账户"下的账户内为你带来更多体验。</p><p style="text-align: left;">现在，Microsoft 365订阅管理体验位于"设置">"账户"的顶部。微软正在订阅卡体验中引入支付细节。这将使您能够查看您的付款信息，并在您的付款方式需要更新时得到通知，使您能够继续您的订阅，而不会受到任何干扰。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0303/8edbf8c6fbde2bb.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">对于不是 Microsoft 365 订阅用户的客户，微软在“您的微软账户”页面上强调了您作为微软账户的一部分可以获得的免费福利。你可以直接访问你的 <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a> Web Apps，查看你的 OneDrive 存储，或升级到 Microsoft 365 订阅，以解锁高级福利。</p></blockquote><p style="text-align: left;"><strong>设备设置 (OOBE): Phone Linking</strong></p><blockquote style="text-align: left;"><p style="text-align: left;">作为 Windows 11 设备设置（OOBE）的一部分，你现在可以将你的 Android <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>绑定到你的电脑。Linking 将使您能够在电脑上即时访问手机上的一切，不需要不断地将注意力转移到手机上，你就可以在电脑上使用你最喜欢的移动应用程序，发送短信，拨打电话，以及更多有完整键盘和鼠标的好处。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0303/b513195a7b3dda7.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">微软将在未来几周内发布更新的 ISO，如果 Windows Insiders愿意，他们可以进行清洁安装，或者设置一个虚拟机来运行设备设置体验。如果你今天想通过设备安装体验，你可以使用设置>系统>恢复下的PC重置功能。</p></blockquote><p 只能在使用最新预览版（Build 22567 及更高版本）进行了清洁安装的 Windows 设备上启用。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0303/3638fae5bb49904.webp" referrerpolicy="no-referrer"></p><p
