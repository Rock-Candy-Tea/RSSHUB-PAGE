作为 Groove Music 的替代者、经典 Windows Media Player 的继任者，适用于 Windows 11 系统的全新 Media Player 已邀请 Windows Insider 项目成员测试。<strong>不过用户报告说，新版 Media Player 现在正向非 Insider 成员推出。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0105/ba341c6a78f07cd.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">一个新的更新将 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Media Player 软件要求设置为 Windows 11 Build 22000，这意味着你现在可以通过 Microsoft Store 安装该应用程序。如果你在 Store 中没有看到该更新，只要设备运行 Build 22000.346 或更新版本，你应该能够手动安装该应用程序。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0105/22e6273e917c174.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">新版 Media Player 进行了两个多月的测试，它似乎已经准备好正常使用。此外，媒体播放器的一个新的更新使其支持系统主题色。Media Player 现在可以跟随 Windows 11 中的系统主题色。在最新的更新之后，终于可以使用 Windows 11 的主题色，并取代应用程序的内置颜色系统“Zest”。如果你喜欢 Zest，你可以随时从媒体播放器的设置中回到以前的默认风格。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0105/e68b7ee4af7df0f.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">新的媒体播放器是专门为Windows 11构建的，它使用WinUI元素、圆角等。目前，媒体播放器只在Windows 11上运行，但它也将抵达Windows 10，没有圆角和适用于应用程序背景的 Mica 主题。</p>
