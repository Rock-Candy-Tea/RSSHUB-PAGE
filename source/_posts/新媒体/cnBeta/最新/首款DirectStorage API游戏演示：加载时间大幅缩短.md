<strong>在今天召开的 2022 游戏开发者大会（GDC）上，首款采用 DirectStorage API SDK 的游戏被公开演示。</strong>这意味着游戏开发者现在可以开始使用该技术在PC游戏中实施。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0324/713607b8e0be634.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Luminous Productions 的 Forspoken 项目（以前叫 Project Athia），已经获得了该功能，演示效果令人印象深刻。演示中展示了几个加载场景，使用<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>的 DirectStorage 的改进看起来很大，即使是使用基于 SATA 的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 都有明显的改进，而使用更快的 NVMe SSD 改进更大。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0324/c985ef569076c9d.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0324/28084a121916bd5.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">在演示过程中，还提供了一张图表，显示了使用 DirectStorage 与 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Win32 Storage API 相比时的 I/O（输入-输出）率和相应的游戏加载时间。</p><p style="text-align: left;">当你看 NVMe 数字时，收益尤其令人印象深刻，因为与 Win32（2862 MB/s）相比，DirectStorage（4,829 MB/s）显示了近 70% 的改进。在实际加载时间方面，虽然差异不大，但它可能意味着在未来的某些游戏中加载速度更快。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0324/48cab934e68ed22.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0324/fc3cc3891f414c3.webp" alt="a9ouz4gc.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">正如你在上面的视频中注意到的，除了微软的DirectStorage，Forspoken也得到了<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a>新的FidelityFX超级分辨率（FSR）2.0，尽管对该功能的支持目前还在开发中。然而，实现它可能不会太困难，因为游戏已经使用了FSR 1.0功能。</p>
