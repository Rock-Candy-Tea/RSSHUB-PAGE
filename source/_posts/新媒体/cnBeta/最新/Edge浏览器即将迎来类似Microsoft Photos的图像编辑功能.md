对于某些用户来说，Microsoft Edge 浏览器可能是一款徒占地方的臃肿软件。即便如此，软件巨头还是在努力为它开发一系列新功能。<strong>近日有消息称，Windows 11 的默认浏览器，将迎来类似 Microsoft Photos 照片应用程序的一些改进。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0709/cc83237a1a3c617.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（via <a href="https://www.windowslatest.com/2022/07/09/windows-11-photos-app-like-features-are-coming-to-microsoft-edge/" target="_self">Windows Latest</a>）</p><p>作为 Windows 11 附带的新照片应用程序，Microsoft Photos 已提供简单的图像编辑功能。</p><p>而基于 Chromium 内核的 Microsoft Edge 浏览器，也即将迎来类似的右键图像编辑选项。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/2c1342604fbde2d.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>据悉，Microsoft Edge 已附带一个捕获工具，以方便用户给网页（或 Web 应用程序）截图。</p><p>但在未来版本中，你可更加方便地通过鼠标右键来选择对特定图像进行编辑，然后将之轻松保存到桌面。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/f20d8f66a907f47.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>如图所示，其界面与 Windows Photos 有许多共同点。当你在情境菜单中点击“编辑”按钮后，Edge 就会启动内置的编辑器。</p><p>不过目前提供的功能仍相当基础，仅包含了裁剪图片、调节亮度和曝光度，以及添加图片注释和应用各色滤镜。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/0e9d1249afbbb6a.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>另一方面，一些人担心新工具可能会占用大量资源、进而拖慢 Edge 浏览器的运行速度 —— 尤其对那些运行 Windows 11 操作系统的旧设备来说。</p><p>目前<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>已开始分批次向 Canary 频道推送新版 Microsoft Edge，等不及的朋友，也可尝试通过官网下载页面获取（<a href="https://www.microsoftedgeinsider.com/" target="_self">传送门</a>）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/1dbff446049361a.jpg" alt="5.jpg" referrerpolicy="no-referrer"></p><p>最后，除了计算机、翻译器、字典、单位换算等功能，Microsoft Edge 还可通过双击来关闭任何选项卡。</p>
