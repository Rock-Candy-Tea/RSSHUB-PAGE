可能泡泡玛特公司名称里的泡泡两个字不太好：<strong>克股价。</strong>这家销售潮流玩具的公司，股价已经从 2020 年年底上市后最高的 106 港元跌到 22 港元，<strong>足足跌去：80% 。</strong>巅峰时接近 1500 亿港元的市值，如今只剩 300 亿港元。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0724/ff49d03b4b3851a.webp" referrerpolicy="no-referrer"><br></p><p>图源：雪球</p><p>泡泡玛特股价下跌的原因，主要还是跟业绩有关。</p><p>上周五，泡泡玛特发了个公告，说预期 2022 年上半年收入同比增长 30% 以上，但，<strong>净利润最多将减少：</strong></p><p><strong>35% 。</strong></p><p><strong>这是泡泡玛特上市以来首次出现净利润下滑。</strong></p><p>至于业绩下滑的原因，泡泡玛特说是因为今年上半年上海、北京、深圳等地出现的疫情（ 它的用户主要分布在一二线城市 ）。</p><p><strong>实体店铺关门、电商物流受阻、消费者消费意愿降低，</strong><strong>三杀之下，泡泡玛特表示压力很大。</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0724/fef8c43b8ec01e7.webp" referrerpolicy="no-referrer"><br></p><p>图源：泡泡玛特公告</p><p>这个解释是非常合理的，但，<strong>上市高峰至今连绵跌去 80% 股价，光靠疫情，是解释不了的。</strong></p><p><strong>上市即巅峰</strong></p><p>泡泡玛特在刚上市时，对投资者来讲似乎是一块又大又圆的美味蛋糕。</p><p>2018 -2019 年，泡泡玛特总营收分别为 5.15 亿元、16.83 亿元，同比增幅为 225.49%、227.19% 。</p><p>也就是说，<strong>连续两年实现两倍增长。</strong></p><p>毛利率方面，2017 年是 47.6%，2018 年提高到 57.9%，2019 年提高到了 64.8% 。</p><p>所以，泡泡玛特的<strong>净利润</strong>也是一路攀升，从 157 万提高到 4.5 亿，<strong>三年翻了：</strong></p><p><strong>287 倍。</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0724/e180ca9b8778a4e.webp" referrerpolicy="no-referrer"><br></p><p>图源：泡泡玛特招股书</p><p><strong>就这个数据，谁看了不迷糊？</strong></p><p>收入连续翻两番、毛利率、净利率不断提高，在这样的业绩表现面前。。。</p><p><strong>什么都不用说了，请一定要收下我的钱！</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0724/2f0e2878c37c107.webp" referrerpolicy="no-referrer"><br></p><p>图源：《 Futurama 》</p><p><strong>于是，泡泡玛特的市值就被炒上了 1500 亿港元。</strong></p><p>遥远的记得，那时候还有传言说，券商组织机构去调研，男基金经理和年纪稍大的女基金经理都不看好，但 90 后分析师都很看好。</p><p><img src="https://static.cnbetacdn.com/article/2022/0724/8c41637779d0e07.webp" referrerpolicy="no-referrer"><br></p><p>雪球现在还有当时的相关帖子能找到</p><p>那时候，很多人管泡泡玛特叫<strong>“ 90 后的茅台 ”。</strong></p><p>可是这还没过多久，泡泡玛特的业绩就开始变脸了。</p><p>收入这块，2020 年收入同比增 49.31% ，2021 年同比增长 78.7%，增速虽然没有前两年那么亮眼，但也还是高速增长。</p><p><strong>不过，其他指标就不好看了。</strong></p><p>毛利率在 2020 年下降到 63.4%，然后在 2021 年继续下降到 61.4% 。</p><p>净利率同样一步一个台阶地下降，先降到 20.8%，再降到 19% 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0724/fe12a3ca97cb938.webp" referrerpolicy="no-referrer"><br></p><p>图源：泡泡玛特 2021 年年报</p><p><strong>变脸原因</strong></p><p>那么，泡泡玛特的经营出了什么问题？</p><p>对于毛利率下滑，官方的解释是：</p><p><strong>主要由于我们提高产品的工艺质量，产品工艺愈加复杂，以及 2021 年原材料成本上涨导致毛利率下降。</strong></p><p>净利率下滑，官方解释是：</p><p><strong>门店数量增加、销售员工人数增加、线上渠道的宣传推广增加，导致销售、管理费用提高。</strong></p><p>工艺、原材料、门店和员工好理解，其他的费用提高是哪来的？</p><p>这就要从泡泡玛特的生意说起。</p><p>泡泡玛特做的是潮流玩具生意，前期需要获取 IP 授权，进行创作，制成玩偶（ 玩偶装在盲盒里，消费者需要拆开盲盒才能知道自己买到什么款式的玩偶 ），然后通过电商、门店等渠道卖给消费者。</p><p><img src="https://static.cnbetacdn.com/article/2022/0724/36e9c305906bba1.webp" referrerpolicy="no-referrer"><br></p><p>图源：泡泡玛特招股书</p><p>其中，<strong>IP 是泡泡玛特的业务核心。</strong></p><p>IP 可以由公司旗下的艺术家自己设计，也可以通过签约全球各地的艺术家来获取独家授权，也可以是与米奇、火影忍者、海贼王、哈利波特这样的内容 IP 合作，进行联名。</p><p>泡泡玛特分别把它们称作自有 IP，独家 IP，非独家 IP 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0724/9c23ddc208118c1.webp" referrerpolicy="no-referrer"><br></p><p><img src="https://static.cnbetacdn.com/article/2022/0724/eb0d538f15b3d15.webp" referrerpolicy="no-referrer"><br></p><p><img src="https://static.cnbetacdn.com/article/2022/0724/6d9db6decb4f91e.webp" referrerpolicy="no-referrer"><br></p><p>图源：泡泡玛特招股书</p><p>相比较而言，<strong>自有 IP 和独家 IP 通常只有形象，没有故事，因此也就少了与消费者之间的情感羁绊，黏性较弱，但成本较低，所以毛利率较高。</strong></p><p>而火影忍者等内容 IP 需要支付一笔授权费才能进行开发，所以毛利率低于自有 IP，优点是自带粉丝基础。</p><p>前期，泡泡玛特的收入主要靠 molly 这一 IP。</p><p>2017-2019 年，molly 分别贡献了几乎所有的收入，62.9% 的收入，32.9% 的收入。</p><p><img src="https://static.cnbetacdn.com/article/2022/0724/3b15b5fcc8ba149.webp" referrerpolicy="no-referrer"><br></p><p>图源：泡泡玛特招股书</p><p><strong>但，每个 IP 都是有生命周期的。</strong></p><p>2020 年，molly 系列的收入出现负增长，从 2019 年的 4.56 亿元降至 3.56 亿元。</p><p>2021 年，正值 Molly 诞生十五周年，泡泡玛特持续推出 molly 新品，与高达、味可滋（ 伊利旗下风味奶品牌 ）等品牌进行跨界联名合作，举办主题展，才停住了 Molly 收入下降的势头。</p><p><strong>这些举措，使得 2021 年 Molly 收入大幅上升，达到 7.05 亿元。</strong></p><p><strong>难受的是，另一个 IP —— PUCKY 又出现了收入下滑。</strong></p><p>PUCKY 系列 2020 年的收入为 3 亿元，到了 2021 年只剩 1.82 亿元。</p><p><img src="https://static.cnbetacdn.com/article/2022/0724/cb93ae1b60518ee.webp" referrerpolicy="no-referrer"><br></p><p>图源：泡泡玛特 2021 年年报</p><p><strong>对此，泡泡玛特的解决方法是推广其他的 IP，试图以量取胜。</strong></p><p>目前，泡泡玛特共运营了 93 个 IP，包括 12 个自有 IP、25 个独家 IP 及 56 个非独家 IP。</p><p>在 2021 年的财报中，自有 IP 下出现了小甜豆、SKULLPANDA、Dimoo、Bunny 这些新面孔，但它们贡献的营收并不高。</p><p><img src="https://static.cnbetacdn.com/article/2022/0724/37192d4082b686a.webp" referrerpolicy="no-referrer"><br></p><p><strong>这些新 IP 在未来能否挑起营收大梁，仍是未知数，但关于新 IP 的推广和营销费用、前期设计和开模成本等，都先花了出去。</strong></p><p>看来，泡泡玛特造 IP 的路数，也有文化产业的通病：</p><p><strong>事先谁也不知道消费者会不会买账，往往是碰运气。</strong></p><p>举个例子，在推出《 阴阳师 》手游后，网易 2016 年第四季度的在线游戏服务收入达到 89.59 亿元人民币，同比增长 62.8%，环比增长 35%，网易交出了上市以来最好的一份财报。</p><p>而到了 2017 年第四季度，<strong>虽然网易自主研发了《 荒野行动 》、《 终结者2：审判日 》、《 楚留香 》等多款手游，但都没能抵消《 阴阳师 》热度下滑的影响，当季度在线游戏服务收入为 80.04 亿元人民币，同比下降 10.7%。</strong></p><p>同样的道理，或许我们也应该对泡泡玛特通过孵化新 IP 顺利接过老 IP 营收大旗的想法，抱有一丝怀疑：</p><p><strong>畅销 IP 或许是可遇不可求的。</strong></p><p>所以，期望泡泡玛特的营收一直像前些年一样翻倍增长，是不现实的。</p><p>除了 IP 产业本身不稳定的特点之外，<strong>泡泡玛特还面临同行竞争的问题。</strong></p><p>2019 年，虽然泡泡玛特在潮流玩具市场零售市场的占有率排名第一，但也只占：</p><p><strong>8.5% 。</strong></p><p>潮流玩具市场还是一个很分散的行业，受关注程度当时还没那么高。</p><p>而当泡泡玛特上市后，<strong>这个闷声发大财的行业被暴露在聚光灯下，吸引了越来越多的资金进入。</strong></p><p>本编辑部拉取了天眼查中中国潮玩相关企业注册量的数据，制了一张表，你可以看到，<strong>泡泡玛特上市之后，潮玩相关企业的注册量同比暴涨了 255% 。</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0724/9722728f923557f.webp" referrerpolicy="no-referrer"><br></p><p>然而，<strong>在一众竞争对手面前，泡泡玛特没有建立起绝对优势。</strong></p><p>拿 IP 来说，泡泡玛特可以做自有 IP，其他公司也可以（ 这种 IP 不太需要故事，只需要形象 ）；泡泡玛特可以拿授权，其他公司只要舍得花钱，也可以拿授权。</p><p>在制造上，泡泡玛特的玩具主要委托第三方工厂制造，泡泡玛特负责质量监督，这方面也没有太多门槛。</p><p>在销售渠道上，泡泡玛特建立了电商+门店+自动售货机并立的销售渠道，很猛。</p><p>不过，<strong>它的竞争对手甚至有现成的。。。</strong></p><p>比如名创优品旗下的 TOP TOY 同样有天猫旗舰店，并且可以依托名创优品在国内的 <strong>3168 家门店</strong>，深度触达消费者。</p><p>在价格上，就更拉不开距离了，除去 IP 费用，大家都是代工，产量上来了大差不差。</p><p>在销售模式上，泡泡玛特的 “ 盲盒 ” 式销售，被认为是带动复购的 “ 成瘾式利器 ”，而这，也很容易模仿甚至升级。</p><p>比如，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a>堆博物馆、河南博物馆推出的考古盲盒，将艺术品仿制品藏在泥块中，消费者购买后需要用附赠的洛阳铲挖开泥块，才能知道买到什么款式的艺术品。</p><p><strong>这不仅模仿了盲盒，还模仿出了新的花活儿。</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0724/fa5f9edbe89ef80.webp" referrerpolicy="no-referrer"><br></p><p>图源：淘宝买家秀</p><p>这就像 “ 每日坚果 ” 一样，这种将各种坚果混在一起卖的形式一开始是由沃隆首创的销售手段，却被三只松鼠、良品铺子、洽洽等零食品牌搬走发扬光大。</p><p>同理，<strong>近些年潮流玩具的市场会被竞争对手分流，是板上钉钉的事情。</strong></p><p>为了提高品牌竞争力，泡泡玛特需要提高玩具的工艺水平，同时还需要加大广告营销吸引用户，这又导致了净利率下滑。</p><p>总的来讲，<strong>IP 文创行业本身的业绩不稳定，加上潮玩市场从红利期进入竞争期，泡泡玛特身上的光环逐渐暗淡，估值的泡沫，也被逐渐戳破。</strong></p><p>泡泡玛特的创始人王宁早年在接受时说：</p><p><strong>“ 泡泡玛特有一个宏伟的目标，就是五年后成为国内最像迪士尼的那家公司。”</strong></p><p>但显然，泡泡玛特在 IP 这方面，跟迪士尼差的有点太远了，<strong>他们的 IP 更像是空中阁楼，不像迪士尼一样有深厚的内容文化底蕴。</strong></p><p>本编辑部认为，<strong>与其说是迪士尼，某种意义上，泡泡玛特更像是乐高。</strong></p><p>同样是制造门槛不高的玩具，同样是门槛不高的销售形式，同样是在创造或不停地做热门 IP 联名，同样是热门款被用户私下炒来炒去。。。</p><p>泡泡玛特相比乐高欠缺的，除了炉火纯青的 IP 运作手法（ 这个可以慢慢练 ），在于没有更高的可玩性，乐高的可玩性可以充分释放人们的创造力，也可以充分的创造朋友、亲子间的线下互动场景。</p><p><strong>或许，让产品具有更高可玩性，是泡泡玛特未来的出路之一。</strong></p><p><strong>当然，据说他们正在筹划自己的主题乐园，真的往迪士尼那个方向去了，这可能也是出路之一。</strong></p><p><strong>不过，乐园生意，可是相当难啃的一块骨头。</strong></p><p><strong>话说回来，虽然泡沫破了，泡泡玛特依然是一家能赚钱、有增长的公司。</strong></p><p><strong>至于未来如何，还有没有可能再次重临千亿市值。</strong></p><p><strong>就看转型的造化了。</strong></p>
