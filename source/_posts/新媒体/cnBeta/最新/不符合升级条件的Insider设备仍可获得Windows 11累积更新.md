面向未符合 Windows 11 升级条件的 Windows Insider 项目成员，微软日前发出提醒推荐他们降级使用 Windows 10 系统。来自 Windows Insider 计划的 Brandon LeBlanc 现在对情况进行了一些澄清。<strong>他在推文中写道：“获得例外的 PC 今后将不再收到来自 Dev 频道的 Windows 11 预览版更新，但在正式发布之前仍可以获得累积更新”。</strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/thumb/article/2021/0901/3328ab4e0890bd9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0901/3328ab4e0890bd9.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">简单来说，目前已经参与 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 但不符合升级条件的 Windows Insider 设备每周定期仍能获得小型的累积更新，但是无法获得<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>后续为 Dev 频道发布的新预览版。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0902/a73c46b681495a0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0902/a73c46b681495a0.jpg" alt="3zt5mztq.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">此外在 Windows 11 正式发布之后，这些用户将会再次被要求降级到 Windows 10 系统。微软没有澄清拒绝降级到Windows 10的Windows 11内部人员到底会发生什么。LeBlanc 指出，自 Windows 11 于 6 月 24 日进入 Insider 计划以来，这一直是计划。</p>
