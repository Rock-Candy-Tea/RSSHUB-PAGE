<strong>包括 Quake、Quake 2 和 Quake 3 Arena 在内的三款《雷神之锤系列》（Quake）可以通过 Xbox Game Pass 获得。</strong>在 id Software 透露并发布了初代 Quake 的重新制作版本后不久，微软于周四在 QuakeCon 2021 开始时宣布了这些新增内容。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0820/d43ef335708b35f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/d43ef335708b35f.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0820/cd195f4f75e32c0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/cd195f4f75e32c0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">通过 <a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> Game Pass，玩家可以通过 Xbox 游戏机和 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> PC 访问初代 Quake 增强版，而 Quake 2 和 Quake 3：Arena 可以通过订阅服务“以原始形式”在 Windows PC 上游戏。Quake 2 和 Quake 3 还没有被重新制作。希望以更漂亮的形式享受 Quake 2 的玩家应该查看免费的 Quake 2 RTX 版本。</p><p style="text-align: left;">Bethesda Software 重新制作的 Quake 是这款已有 25 年历史的游戏的新版本，具有 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_110018%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">4K</a> 分辨率支持、跨平台游戏、增强的视觉效果以及来自德军总部开发商 MachineGames 的新扩展包。它也可以在 Nintendo Switch、PlayStation 4（ 和 PlayStation 5 通过向后兼容）和 Windows PC 上单独使用。</p><p style="text-align: left;">这三款 Quake 游戏加入了一个包括 30 多个 Bethesda Softworks 发布的游戏的阵容，其中许多游戏是在<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>收购 Bethesda 之后添加到 Xbox Game Pass 中的。</p><p style="text-align: left;">此外，援引外媒 Gematsu 报道，娱乐软件评级委员会 (ESRB) 和韩国游戏物管理委员会（GRAC）都对 id Software 开发的 1996 年发布的热门射击游戏设定了新的评级。ESRB 网站上的描述对于此版本的新内容并不是很有帮助，因为它只是描述了标志性的迷宫般的环境和血腥元素，这些元素已经在原版中看到了，然后才给标题一个成人（Mature）的评级。</p>
