自从在 I/O 2019 开发者大会上推出以来，Jetpack Compose 注定要成为官方推荐的 Android UI 开发方式。<strong>经过两年的持续发展，我们现在终于迎来了一个里程碑 —— 今天，谷歌正式推出了 Jetpack Compose 的 1.0 版本。</strong><br>
<p><a href="https://static.cnbetacdn.com/article/2021/0729/c8928d93dacbe1e.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0729/c8928d93dacbe1e.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>Android Police 指出：除了 Android Studio Arctic Fox 的稳定版本，Jetpack Compose 还可以用于生产环境。</p><p>今年 3 月，谷歌发布了 Jetpack Compose 的 1.0 测试版本，并于接下来的数月积极识别和修复相关 bug 。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=275717255&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">Announcing Jetpack Compose 10（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMjc1NzE3MjU1LnNodG1s.html" target="_self">via</a>）</p><p>与此同时，我们也迎来了一些新近添加的功能，比如 Compose Preview 和 Deploy Preview 。</p><blockquote><p>● 前者允许开发者在不同状态下查看 Compose UI 的实时渲染，例如明暗模式或不同的字体缩放。</p><p>● 而后者更加注重功能层面，允许开发者在不完全重新启动应用程序的情况下，向用户推送更新的代码。</p></blockquote><p>上述这两者都需要 Android Studio Arctic Fox，其也于今日迎来了稳定版更新。</p><p><a href="https://static.cnbetacdn.com/article/2021/0729/9a779b4235efbaa.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0729/9a779b4235efbaa.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>在将 Jetpack Compose 集成到新开发的 App 之前，许多人都在翘首期盼 1.0 稳定版。该框架允许在现有视图中使用 Compose，或将之包含于 Composable 中。</p><p>这使得开发者能够以任何速度将 Jetpack Compose 集成到各自的项目中，而无需进行大规模的转换、甚至无需重写任何代码。</p><p><img src="https://static.cnbetacdn.com/article/2021/0729/e1a60648d092d8b.gif" alt="4.gif" referrerpolicy="no-referrer"></p><p>此外 Compose 能够与其它 Jetpack 库顺利集成，以实现 LiveData 和 Navigation 等功能。</p><p>所有 UI 元素均采用 Material Design 设计并支持主题化，因而开发者能够自定义 App 的外观，并借助丰富的动画 API 来传达各种活动。</p><p><a href="https://static.cnbetacdn.com/article/2021/0729/57621924b385ee3.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0729/57621924b385ee3.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>Android Studio <a href="https://developer.android.com/studio" target="_self">下载地址</a>：</p><blockquote><p>2020.3.1 for <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 64-bit（<a href="https://redirector.gvt1.com/edgedl/android/studio/install/2020.3.1.22/android-studio-2020.3.1.22-windows.exe" target="_self">913MB</a>）</p></blockquote>
