在为桌面端 Edge 浏览器引入 Edge Bar 之后，现在微软将该功能引入到 Android 端 Edge 浏览器上。<strong>在最新测试中，Android 端 Edge 浏览器可以访问 Bing 的每日图片、Quick 搜索和 Trending 提示 Widgets。</strong>但需要用户在手机端搭配 Microsoft Launcher 才能实现最佳体验。<br>
<p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0228/1655a60e5a66657.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">在 Android 端 Edge浏览器上启用 Bing 搜索和每日图片 Widgets 操作如下</p><p style="text-align: left;">1. 打开 Edge 浏览器</p><p style="text-align: left;">2. 访问 edge://flags 页面</p><p style="text-align: left;">3. 搜索“Widget”然后启用如下几个实验 Flag</p><p style="text-align: left;">● Bing Search Widget Promote</p><p style="text-align: left;">● Bing Daily Image Search Widget</p><p style="text-align: left;">● Bing Trending Search Widget</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0228/e42fabc0069edc3.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">完成上述步骤后，当你打开一个新标签或访问 Bing 或随机页面时，你可能会被提示在主屏幕上添加“Quick Search”或“Widget with daily image”。允许这两种情况，接下来，你会被要求允许 Microsoft Launcher 创建一个小部件，并允许访问它所显示的数据。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0228/5657a61d1dfeee5.webp" referrerpolicy="no-referrer"></p>
