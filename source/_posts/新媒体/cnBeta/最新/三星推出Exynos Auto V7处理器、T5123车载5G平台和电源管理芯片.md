三星电子刚刚宣布了面向车载平台的三款芯片解决方案，<strong>分别为用于 5G 连接的 Exynos Auto T5123、用于综合车载信息娱乐系统的 Exynos Auto V7、以及 ASIL-B 认证的 S2VPS01 电源管理（PMIC）芯片。</strong>三星电子商用定制 SoC 事业部执行副总裁 Jaehone Park 表示：“兼顾娱乐、安全、舒适的丰富车载体验，以及更智能、更互联的汽车技术，正在成为交通领域的一项关键特性”。<br>
  <p><a href="https://static.cnbetacdn.com/article/2021/1130/e790bfe7adc5357.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1130/e790bfe7adc5357.jpg" alt="0" referrerpolicy="no-referrer"></a></p><p>凭借先进的 5G 调制解调器、人工智能（AI）增强型多核处理器、以及经过市场验证的 PMIC 解决方案，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a>电子正在将自己在移动解决方案上的专业知识融入车载产品线，并准备扩大在该领域的影响力。</p><p>首先介绍 Exynos Auto T5123，该 5G 连接解决方案整合了 3GPP 第 15 版远程信息处理控制单元，专为面向联网汽车的独立（SA）/ 非独立（NSA）模式下的快速无缝 5G 连接而设计。</p><p>该平台支持通过高达 5.1 Gbps 的高速下行速率，实时向车辆提供重要信息，并让乘客在旅途中畅想一系列新服务 —— 比如高清流媒体 / 视频通话等。</p><p>为高效处理通过 5G 调制解调器收发的大量数据，Exynos Auto T5123 还支持高速 PCIe（PCI Express）接口，以及低功耗、高性能的 LPDDR4x 移动 DRAM 。</p><p>此外 T5123 内置了两个 Cortex-A55 CPU 核心和全球卫星导航系统（GNSS）模块，以最大限度减少外部 IC 的使用、且有助于缩短新品的开发时间。</p><p>最重要的是，T5123 满足汽车零部件的严格要求，并通过了汽车电子委员会的 AEC-Q100 认证，目前该芯片已投入量产。</p><p><a href="https://static.cnbetacdn.com/article/2021/1130/c671791d7cc8cbe.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1130/c671791d7cc8cbe.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>其次 Exynos Auto V7，作为一款面向中高端车载 IVI 系统的强大处理器，这款新品也专为车载信息娱乐系统而设计，集成了 8 核 1.5GHz 的 ARM Cortex-A76 CPU + 11 核 Mali-G76 GPU 。</p><p>有趣的是，GPU 模块也被分成了两组，其中小集群中的三个 GPU 核心专为 AR-HUD 显示而留，大集群中的八个 GPU 核心则用于中控信息显示（CID）和其余部分。</p><p>这种独特的“物理分离”，允许 GPU 同时支持多个系统、并实现更安全的操作（防止不同区域的互相影响）。</p><p>此外 Exynos Auto V7 配备了神经处理（NPU）单元，可为面部视觉和音频处理（比如虚拟助理）提供支撑。</p><p>为获得更安全、愉快的驾驶体验，这枚芯片最多支持四屏输出 + 12 个摄像头输入，辅以坏像素 / 几何失真校正、动态范围压缩、全景影像 / 停车辅助等功能。</p><p>车载信息娱乐方面，Exynos Auto V7 配备了三重 HiFi 4 音频处理器，可用于在旅途中识别歌曲、电影、甚至游戏，然后带来更出色的音频体验。</p><p>为了尽可能流畅地运行上述功能，三星电子还为它搭配了 32GB LPDDR4x 内存，带宽高达 68.3 GB/s 。</p><p>数据保护方面，Exynos Auto V7 还配备了支持加密操作的安全隔离处理器，并使用一次性可编程（OTP）/ 物理不可克隆（PUF）功能提供硬件密钥。</p><p>关键功能方面，该芯片符合 ASIL-B 对数字集群 / 嵌入式安全岛的安全特性要求，能够检测和管理故障、并通过故障管理单元（FMU）来保持其安全状态。</p><p>目前 Exynos Auto V7 已投入量产，并将用于大众最新的 3.1 版车载应用服务（ICAS）平台 —— 后者由 LG 电子的车辆组件解决方案（VS）部门开发。</p><p><a href="https://static.cnbetacdn.com/article/2021/1130/08c95a082f36827.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1130/08c95a082f36827.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>然后是 S2CPS01，它是专为 Exynos Auto V9 和 V7 设计开发的电源管理芯片（PMIC）。三星电子于 2019 年获得了 ISO 26262 功能安全流程认证，并于 2021 年拿到了 ASIL-B 认证。</p><p>前者规定了汽车的安全完整性等级（从基础的 A、到最高的 D），具体是通过分析和评估车辆在多种环境中运行的严重性、暴露度、以及可控性来区分。</p><p>为确保车载系统安全，ASIL-B 合规性正成为汽车 OEM、及其一级供应商在选择合作伙伴和解决方案时的一项关键要求。</p><p>言归正传，S2VPS01 能够调节和整流，以助力车载信息娱乐系统实现可靠而强大的性能。它由高效的三相 / 双相降压转换器组成，并在封装内集成了低压降稳压器（LDO）和 RTC 实时时钟。</p><p>最后，为避免在严苛的热 / 电气条件下失效，S2VPS01 PMIC 还集成了全面的保护功能，包括过压（OVP）、欠压（UVP）、短路（SCP）、过流（OCP）、热关断（TSD）保护，以及时钟监控和输出阻滞检查。</p>
