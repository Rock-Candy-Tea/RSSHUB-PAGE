在十七年前 Unity 成立时，全公司只有一款产品，服务对象也仅限于独立游戏。今天，Unity 的产品线包括 50
多种产品，为各行各业内上百万的开发者、创作者和经营者提供广泛且优质的服务。为了更好地反映出品牌新形象、产品演变和未来发展方向，我们将于未来几个月逐步推出<strong>Unity 系列产品的全新徽标</strong>。<br>
<p><strong>Unity徽标的新设想</strong></p><p>Unity 徽标已经过多次微调，如今 Unity 需要的是一个更富动态、更具表现力、更加包容的新 Logo。因此，我们重新设计了 Logo 的两个重要组成部分：Unity 立方体和 Unity 文字商标。</p><p><img src="https://cors.zfour.workers.dev/?http://dingyue.ws.126.net/2021/1008/44aca7e0g00r0n5vb006cd200tz00gvg00tz00gv.gif" referrerpolicy="no-referrer"><br></p><p><strong>真正的三维立方体</strong></p><p>我们翻新了 Unity 标志性的立方体，保留了原版的精髓与价值。现在它是全 3D 外观，其中的<strong>X 轴代表着技术</strong>，<strong>Y 轴代表着创作者</strong>，而<strong>Z 轴代表着创作出的体验</strong>，三者在 Unity 的作用下连接在一起。</p><p>更重要的是，新的徽标寓意着 Unity 是一个有生命的品牌：它呵护每一位创作者，支持与倡导机会平等和多样性。立方体的三个箭头是三个矢量，它表示所有使用 Unity 的用户都能在各个方向上开拓无限的可能。</p><p>新的立方体同样将成为其它 Unity 产品的商标基础，并逐步推广到整个开发平台和产品系列。</p><p><img src="https://cors.zfour.workers.dev/?http://dingyue.ws.126.net/2021/1008/97bf8049g00r0n5vd02jvd200tz00gvg00tz00gv.gif" referrerpolicy="no-referrer"><br></p><p><strong>全新 Unity 文字商标</strong></p><p>Unity 的文字商标也有了些许改动：除了字符的微调外，原本的“u”也改为了大写的“U”，还加上了注册商标符号，寓意 Unity 已经是一个成熟的实时 3D 引擎。</p><p><img src="https://cors.zfour.workers.dev/?http://dingyue.ws.126.net/2021/1008/4a8d0cd3g00r0n5vf00hqd200tz00gvg00tz00gv.gif" referrerpolicy="no-referrer"><br></p><p><strong>新Unity系列产品徽标</strong></p><p>我们将以新徽标为基础，<strong>推出一套连贯的产品徽标</strong>，让用户可以更轻松地识别并找到 Unity 产品、认识 Unity 产品系列、体会 Unity 产品间的关联性。</p><p><img src="https://cors.zfour.workers.dev/?http://dingyue.ws.126.net/2021/1008/9e933a5dg00r0n5vg00j1d200tz00gvg00tz00gv.gif" referrerpolicy="no-referrer"><br></p><p><strong>动态 3D 徽标</strong></p><p>Unity 作为实时 3D（RT3D）技术的领导者，我们的产品可为各种项目和应用注入生机和活力。因此，我们决定为所有主要产品的徽标加上动画。新徽标将出现在商店页面及未来的营销活动中。</p><p><img src="https://cors.zfour.workers.dev/?http://dingyue.ws.126.net/2021/1008/94422187g00r0n5vi04eqd200tz00gvg00tz00gv.gif" referrerpolicy="no-referrer"><br></p><p><strong>新徽标将逐步上线</strong></p><p>截至目前，新徽标将首先应用于网站与社交平台上，并陆续整合至 Unity 整个产品线。在未来的几个月，新徽标将出现在 Unity 2021.2、产品、服务、宣发和周边等所有 Unity 品牌接触点。</p><p>欢迎在 Unity 品牌页面详细了解我们的品牌，或下载新品牌资产用于创作或使用。</p><p><strong>Unity 品牌页面：</strong></p><p><a href="https://unity.com/cn/brand" _src="https://unity.com/cn/brand" target="_blank">https://unity.com/cn/brand</a><br></p>
