基于 Chromium 的 Microsoft Edge 浏览器在PDF文件管理方面一直在稳步改进。<strong>在将网页捕获功能扩展到 PDF 文件后，该网络浏览器现在提供了 PDF 文件中每一页的缩略图，以便于快速导航。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0113/32be59961079321.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">微软正在逐步提供一种新的方法，在 Edge 网页浏览器内直接导航 PDF 文件。微软 Edge 网页浏览器 Build 99.0.1133.0 现在提供列表视图以及缩略图视图，用于滚动浏览 PDF 文件。这两种类型的视图将出现在一个窗格中，在左侧可见。如果这个功能听起来很熟悉，那是因为即使是Google浏览器也为 PDF 文件提供了相同的导航，用户可以在 Chrome 浏览器中直接访问。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0113/975c783e56b1c95.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">微软还在最新的 Edge Canary 版本中增加了两个新标志。这些实验性功能允许启用或禁用 Mica 和 Acrylic 效果。它们分别存在于 #edge-visual-rejuv-materials-title-bar 和 #edge-visual-rejuv-materials-menu 实验 Flag 下。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0113/8ee9fcc610f1ec6.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Acrylic 效果设置将影响出现在 Microsoft Edge 内部的菜单，而 Mica 效果设置将影响标题栏。顺便说一下，这两个实验 Flag 只对 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 用户可用。</p>
