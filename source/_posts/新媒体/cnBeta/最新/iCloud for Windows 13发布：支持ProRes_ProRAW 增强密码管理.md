<strong>适用于 Windows 的 iCloud 近日获得重磅升级，在提升安全性能之外引入了对 ProRes/ProRAW 格式的支持。</strong>最新发布的 13 版本基于今年 8 月发布的 12.5 版本基础上，该更新为同时拥有 Windows 设备的苹果用户带来了适当的 iCloud Passwords 应用。<br>
 <p style="text-align: left;"><strong>● 支持 ProRes/ProRAW 格式</strong></p><p style="text-align: left;">iCloud for <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 13 的一个重大变化是能够使用 iCloud 应用从 Windows PC 访问苹果 ProRAW 图像和 ProRes 视频。这项功能可以说对使用 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> 拍摄内容并使用 Windows 电脑作为其后期处理工作流程的视频专业人士和摄影师最为有用。</p><p style="text-align: left;"><strong>● 改进密码功能</strong></p><p style="text-align: left;">iCloud for Windows 13 带来了改进的密码功能，使 Windows 用户能够使用适用于 Windows 的 iCloud Passwords 应用在他们的 PC 上生成更强大的密码。这意味着 iCloud Passwords 应用不再只是单纯管理个人 iCloud 钥匙串的工具，而且为 Windows 用户提供更全面的密码保护。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1111/edd46b4c9b1dc7b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1111/edd46b4c9b1dc7b.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1111/06c73c64c6a3cbd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1111/06c73c64c6a3cbd.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>● 优化 iCloud Drive</strong></p><p style="text-align: left;">新版本能够让 Windows 上的 iCloud 用户在 iCLoud Drive 上从一个已分享的文件或者文件夹上添加和删除人员。虽然应用程序的变化数量有限，但对于那些同时拥有 Windows PC 作为其日常工作环境一部分的苹果用户来说，这是一个重大改进。</p><p style="text-align: left;">苹果已经慢慢地使其产品在其生态系统之外的设备上更容易使用，其中的例子包括在Fire TV等平台上推出其Apple TV+流媒体服务。然而，对于那些发现自己在使用苹果、Android和Windows设备组合的强大用户来说，跨平台支持仍然缺乏，这使得在不同操作系统之间管理数据变得很棘手。</p>
