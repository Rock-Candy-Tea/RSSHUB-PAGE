适用于 macOS Monterey 12.3 及更高版本以及 macOS Ventura beta 版本，今天苹果发布了 Safari Technology Preview Release 147。它是一款实验性浏览器，旨在测试可在新版本 Safari 中发布的功能。<br>
 <p>下载</p><p>● <a class="inline" href="https://secure-appldnld.apple.com/STP/012-30324-20220621-99D72AEC-A0E2-4B48-8AC0-B567E3FD046B/SafariTechnologyPreview.dmg">Safari Technology Preview for macOS Ventura</a></p><p>● <a class="inline" href="https://secure-appldnld.apple.com/STP/012-15389-20220621-FA8B8AC9-0442-432C-80B6-6016AB193FCA/SafariTechnologyPreview.dmg">Safari Technology Preview for macOS Monterey</a></p><p>● <a href="https://developer.apple.com/safari/technology-preview/release-notes/">更新日志</a></p><p>本次新版引入的功能包括实时文本、对 Apple Passkeys 的支持以及改进的 Safari Web 扩展等功能。但是，缺少一些功能，例如选项卡组同步、网站设置和 Web 扩展。</p><p>主要改进如下：</p><p><strong>● Live Text</strong></p><p>在基于 Apple Silicon 的 Mac 上的 macOS Ventura beta 中选择视频中的文本并与之交互或翻译网络上图像中的文本。</p><p><strong>● 网络技术</strong></p><p>体验和测试 Safari 16 Beta 中提供的 HTML、CSS、JavaScript 和其他 Web 技术，这些技术包含在以前的 Safari 技术预览版中。</p><p><strong>● 改进 Safari Web Extensions</strong></p><p>测试 API 改进，包括以编程方式打开 Safari Web 扩展弹出框的能力。</p><p><strong>● Web 检查器扩展。</strong></p><p>构建自定义工具或转换现有的开发人员工具扩展以在 Web Inspector 中使用。</p><p><strong>● Fle<a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> Inspector</strong></p><p>使用 Web Inspector 中新的可视化叠加层，帮助您更快速、更轻松地了解 Flexbox 元素的布局。它标记了弹性项目之间的可用空间和间隙，以揭示它们如何影响结果。</p>
