援引科技媒体 VideoCardz 报道<strong>，AMD 内部正在开发名为“Smart Access Storage”（SAS）的新技术，将会在海盗船（Corsair） 6 月发布的 Voyager 笔记本上率先亮相。</strong>报告称，AMD 的 SAS 技术有点类似于 Smart Access Memory，不过适用于 PC 板载存储硬件。<br>
<p style="text-align: left;">目前关于这项技术的信息并不是很多，可能是<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a> DirectStorage API 的衍生，该 API 已经于 3 月进入 PC 游戏领域，并通过“类Console”存储 API 解决方案为加载时间带来了大幅改善。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0506/034b603f1e97c43.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">通过切换到 DirectStorage API，用户可以获得最多 20-40% 的 CPU 节省，Forespoken 游戏的开发者 Luminous Production s已经分享了他们在<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a>的PC硬件上运行的第一个游戏演示。请注意，这项技术有一段时间只在微软的<a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a>系列游戏机上使用，目前仍在围绕 API 开展工作，以便在下一代 AAA 游戏中使用。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0506/5fe8c51f9f7bac5.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">至于我们有望看到AMD的智能访问存储解决方案的平台，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://mall.jd.com/index-1000000697.html" target="_blank">海盗船</a>Voyager将是首批采用这一技术的笔记本电脑之一。海盗船一直试图进入笔记本电脑领域，他们的"Xenomorph"笔记本电脑平台不久前被泄露，其一系列强大的规格包括AMD Radeon 9 6900HX CPU和Radeon RX 6800M GPU。这听起来绝对是一个全AMD设计的AMD"ADVANTAGE"解决方案，它将是新技术的一个完美展示。</p>
