AMD 首席执行官刚刚证实，<strong>采用 CDNA 和 MCM GPU 架构的 Instinct GPU HPC 加速卡将于今年晚些时候推出，并与英特尔 Ponte Vecchio 和英伟达 Ampere 展开竞争。</strong>在摩根大通举办的第 49 届年度全球技术传播与媒体会议期间，AMD CEO 苏姿丰博士证实了 Instinct MI200 计算卡的上市时间。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0527/203c90446e5641b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0527/203c90446e5641b.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via WCCFTech）</p><p>苏姿丰表示，AMD 在去年介绍过初代 CDNA 架构，而今年将整合推出新一代 CDNA 2 架构。</p><blockquote><p>得益于出色的规格和性能参数，AMD 获得了业内领先的橡树岭国家实验室（ORNL）和劳伦斯利弗莫尔国家实验室（LLNL）在内的顶级超算订单。</p><p>在打通 CPU 和 GPU 的情况下，AMD 还可针对高性能计算（HPC）、人工智能（AI）、机器学习（ML）等应用程序实施全面的优化。</p><p>事实上，我们将在今年晚些时候推出 CDNA 架构的下一代产品。作为围绕数据中心架构开展创新的下一步，目前的进展很是顺利，且我们对此感到非常激动。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0527/941abe39f3e794c.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">Radeon Instinct MI100 资料图</p><p>据悉，CDNA 2 GPU 架构将为 AMD 下一代 Instinct HPC 加速卡提供助力。传闻称 Radeon Instinct MI200 加速卡配备了十分强大的 Aldebaran GPU，且有望率先用上 MCM 多芯片设计。</p><p>Instinct MI200 将与英伟达 Ampere 和定于 2022 年上市的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 7nm Ponte Vecchio 竞品展开激烈的交锋，且据说两家公司也都将在各自的 HPC 加速卡上采用 MCM 多芯片封装方案。</p><p>在早些时候的 Linux 补丁中，有消息称 AMD Instinct MI200 Aldebaran GPU 还配备了 HBM2E 显存（英伟达率先采用了 HBM2E 显存）。</p><p>与 HBM2 相比，HBM2E 的性能得到了进一步的提升，且每堆栈最多可搭配 16GB 的容量，预计 Aldebaran 最高可提供 64GB 的 HBM2E 显存。</p><p><a href="https://static.cnbetacdn.com/article/2021/0527/c662760a0b033f7.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0527/c662760a0b033f7.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">AMD 计算 GPU 架构路线图</p><p>最新的 Linux 内核修补程序还表明，AMD CDNA 2 GPU 将集成 128 个计算单元，且每个 CU 都拥有 16KB 的 L1 缓存（合计 2MB），辅以 8MB 的共享 L2 缓存。</p><p>此外每组着色器引擎（Shader Engine）带有 14 个 CU，而前代 Instinct 产品线则是每组 16 个 CU 。即便如此，Aldebaran GPU 上每组 CU 的算力也都更强。</p><p>其它方面，新 GPU 支持系统直接内存访问（SDMA），可通过 PCIe 和 XGMI / Infinity Cache 子系统进行数据传输。不过在高性能计算（HPC）领域，AMD 仍不大可能引入无限缓存。</p><p>最后，AMD CDNA 2 GPU 将在全新的工艺节点上制造，辅以第 3 代 AMD Infinity 架构，可组成 8 路 GPU 连接、并将算力扩展至百亿亿次（Exascale）的水平。</p>
