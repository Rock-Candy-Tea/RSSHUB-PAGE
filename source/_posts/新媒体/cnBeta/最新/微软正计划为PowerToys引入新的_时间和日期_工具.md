数日前发布的 PowerToys 0.56.1 中，微软引入了大量修复和改进。在新版本中，微软为 FancyZones 提供了 128 种布局，并提供了触发“寻找我的光标”工具的新方法。而在未来的规划中，微软计划为 PowerToys 带来更多新功能。<strong>根据 GitHub 上的 Pull Request(<a href="https://github.com/microsoft/PowerToys/pull/16662" original-href="https://github.com/microsoft/PowerToys/pull/16662">1</a>), (<a href="https://github.com/microsoft/PowerToys/pull/16325">2</a>)，在改进现有的 Window Walker 模块的同时，微软和其他贡献者还在努力增加一个新的时间和日期工具。</strong><br>
  <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0307/682cd33d1901a4a.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">“时间和日期”（Time and Date）是 PowerToys Run 的一个新插件。它可以以各种格式显示指定的日期和时间，并将每个结果复制到剪贴板。例如，你可以要求 PowerToys Run 显示当前的日期和时间、星期几、月几、年几等等。PowerToys 将提供一系列的自定义选项，如包括或排除全局结果（你可以使用预定义的热键触发该插件），用秒来显示时间，或隐藏错误信息。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0307/6c7c53cfcca845b.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">除了新的“时间和日期”模块，PowerToys Run 将获得一个改进的 Window Walker（一个使用启动器寻找运行中的应用程序和窗口的工具）。即将推出的版本将让你关闭窗口和杀死进程，而无需启动任务管理器或切换到你需要的应用程序。另外，如果用户有一个以上的桌面，Window Walker 将在其进程 ID 旁边显示指定应用程序的桌面名称。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0307/ab7002878460ef6.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">按照典型的PowerToys方式，更新后的Window Walker将有一些选项用于个性化的用户体验。其中包括只从可见的桌面上显示进程、结束一个进程及其子进程、在"杀死"一个应用程序之前要求确认、个性化工具提示等功能。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0307/7c030fb058459ee.webp" referrerpolicy="no-referrer"></p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0307/04021f45e1f1593.webp" referrerpolicy="no-referrer"></p>
