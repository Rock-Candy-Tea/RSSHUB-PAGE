红米全球副总裁 Manu Kumar Jain，刚刚在推特上预告了即将登陆印度市场的新机阵容。<strong>91Mobiles 早前指出，Redmi Note 11 Pro 系列将很快到来，且可能通过 Flipkart（而不是亚马逊）渠道发售。</strong>其中 Note 11 Pro+ 5G 机型将是红米 Note 11 Pro 5G 国际版的马甲，而 Note 11 Pro / Pro+ 将成为 Redmi Note 11 家族的一员（包括 Note 11 / 11S）。<br>
 <p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0223/479fe6aeff8b081.png" alt="1.png" referrerpolicy="no-referrer"></p><p>在 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&aid=450&euid=&t=http%3A%2F%2Fwww.mi.com%2F" target="_blank">Redmi</a> Note 11 Pro 系列的预告片中，可见到背景中的数字 2 和 11 。虽然官方尚未披露 Redmi Note 11 Pro 在印度市场的发布日期，但知名爆料人 Yogesh Brar 已透露过 3 月 8-10 日这个时间窗口。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0223/f6dcaaa7b224c42.gif" alt="2.gif" referrerpolicy="no-referrer"></p><p style="text-align: center;">动图转制（8 秒有声视频 <a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzI4MTIxMDE5LnNodG1s.html" target="_self">via</a>）</p><p>配置方面，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&aid=450&euid=&t=http%3A%2F%2Fwww.mi.com%2F" target="_blank">红米</a> Note 11 Pro（4G）配备了 6.67 英寸 @ FHD+ 分辨率的 AM<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_88234%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">OLED</a> 屏，支持 120Hz 高刷、并有康宁大猩猩 5 代玻璃的防护。</p><p>处理器为联发科 Helio G96 SoC，具有 6 / 8GB RAM + 64 / 128GB UFS 2.2 存储组合，且支持 1TB microSD 存储扩展。</p><p><img src="https://static.cnbetacdn.com/article/2022/0223/fa7daad7493165d.jpeg" alt="3.jpeg" referrerpolicy="no-referrer"></p><p>拍照方面，红米 Note 11 Pro 采用了前置 16MP 单摄、以及 1.08 亿像素主摄 + 8MP 超广角 + 2MP 微距 + 2MP 深度传感器的后置四摄组合。</p><p>软件方面，该机预装了基于 Android 11 定制的 MIUI 13，且内置支持 67W 快充的 5000 mAh 电池。</p><p><img src="https://static.cnbetacdn.com/article/2022/0223/68feab602a062dd.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://www.91mobiles.com/hub/redmi-note-11-pro-series-teaser/" target="_self">91Mobiles</a>）</p><p>至于红米 Note 11 Pro+，推测其搭载了高通骁龙 695 5G 新牌组，后置 1.08 亿像素主摄 + 8MP 超广角 + 2MP 微距镜。除此之外，它应该与红米 Note 11 Pro 没有太大的区别。</p>
