在 2021 款 MacBook Pro 身上，我们见到了能效表现惊人的 M1 Pro / M1 Max 定制芯片。与此同时，苹果终于消除了多年的槽点，用实体功能键取代了 Touch Bar、引入了丰富的端口、甚至带回了 MagSafe 3 磁吸充电接口。<strong>尴尬的是，为了提升屏占比，新机也采用了类似 iPhone 的刘海开槽、但却又不支持 Face ID 面容识别功能。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1030/b8a4c018cb53fab.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1030/b8a4c018cb53fab.jpg" referrerpolicy="no-referrer"></a></p><p>虽然 Face ID 不见得是 Touch ID 指纹传感器的更好替代品，但至少根据<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>自家的宣传术语，Face ID 是较 Touch ID 更难被破解的。</p><blockquote><p>● 一方面，C 面键盘位配备的 Touch ID 功能，可能用起来比 Face ID 更便捷，尤其在你本来就打算打字输入的时候。</p><p>● 另一方面，许多人都困扰于 —— 我都已经盯着摄像头所在的刘海缺口位置了，为何不能干脆地使用面容解锁呢？</p></blockquote><p>如此来看，比之缺少 Face ID 功能，大家更可能是看丑陋的刘海开槽不爽。</p><p><a href="https://static.cnbetacdn.com/article/2021/1030/49bf985a1de1d8a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1030/49bf985a1de1d8a.jpg" referrerpolicy="no-referrer"></a></p><p>对于大家的吐槽，苹果 Mac 与 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a> 产品营销副总裁 Tom Boger，也于本周早些时候接受了《华尔街日报》的采访。</p><blockquote><p>期间 Boger 给出的解释是 —— Touch ID 在笔记本上更方便，因为此时大家基本都将手放到了键盘上。</p></blockquote><p>此外在近日的一期播客节目中，苹果高管 Shruti Haldea 辩称他们只是为了让屏幕变得更高。</p><blockquote><p>与在 16 英寸笔记本电脑上一样，2021 款 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">MacBook</a> Pro 仍留有 16：10 的对角线区域。</p><p>我们只是适当加大了显示面板，以将菜单栏往上挪动到刘海所在的顶部区域。</p></blockquote><p>至于消费者是否能够习惯它的存在、或者通过软件适配来缓解（比如选择暗色主题来隐匿刘海缺口），就又是另外一回事了。</p>
