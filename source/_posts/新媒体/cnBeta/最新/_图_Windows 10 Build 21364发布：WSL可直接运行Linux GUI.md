面向 Windows Insider 的 Dev 频道用户，今天微软发布了 Windows 10 Build 21364 预览版更新。虽然预览版中的新功能已不再和功能更新绑定，但这些新特性极有可能会整合到今年下半年的 21H2 功能更新中。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0422/d6b09e5dc837915.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0422/d6b09e5dc837915.jpg" alt="8nloi9rh.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>以下是 Windows 10 Build 21364 预览版更新的内容：</strong></p><p style="text-align: left;"><strong>●  Windows Subsystem for Linux（WSL）上使用首批对 GUI 应用的支持，可以直接在 Windows 上运行 Linux GUI</strong></p><blockquote style="text-align: left;"><p style="text-align: left;">Windows Subsystem for Linux 现在包括首批 GUI 应用语言支持。这意味着用户可以直接在 Windows 上运行你喜欢的 GUI 编辑器、工具、应用，来开发、测试、创建和运行你喜欢的 Linux 应用。</p><p style="text-align: left;">相关演示可以看下方视频：</p><iframe src="https://v.qq.com/txp/iframe/player.html?vid=k3241fvdq8w" allowfullscreen="true" width="750" height="480" frameborder="0"></iframe></blockquote><p style="text-align: left;">● <strong>任务管理器优化 Edge 浏览器进程分类</strong></p><blockquote style="text-align: left;"><p style="text-align: left;">通过和 Microsoft Edge 团队的努力，在任务管理器中支持进程分类。这将有助于用户识别 Microsoft Edge 下的资源消耗。目前 Edge 被细分为多个不同的部分，包括标签、浏览器进程（浏览器、GPU进程、Crashpad）、工具插件（实用程序：音频服务扩展）、Dedicated & Service 等等。此外，还为每个进程提供了单独的图标，以帮助你识别它们，包括网站的图标。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0422/8fc2bf3734f35d9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/0422/8fc2bf3734f35d9.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">注意：该功能仅对运行最新 Edge Canary 和 Dev 频道版本有用。Windows Insider 可以通过这里下载最新版本。该版本目前仅向 Dev 频道的少数 Windows Insider 用户开放，以便于收集可能影响性能和可靠性的问题。不过很快将面向所有 Dev 频道用户开放。</p></blockquote><p style="text-align: left;"><strong>● 任务管理器中的“生态模式”</strong></p><blockquote style="text-align: left;"><p style="text-align: left;">在这个版本中，任务管理器有一个新的实验性功能，叫做“生态模式”，它为用户提供了一个控制进程资源的选项。它还将帮助识别已经在生态模式下运行的应用程序。当你注意到一个应用程序消耗了大量资源，并希望限制其消耗，以便系统优先考虑其他应用程序，这将导致更快的前台响应和更好的能源效率时，这个功能是有帮助的。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/thumb/article/2021/0422/1d742f24fe93079.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0422/1d742f24fe93079.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">如何为任意进程启用生态模式</p><p style="text-align: left;">1. 打开任务管理器</p><p style="text-align: left;">2. 点击“进程”选项卡</p><p style="text-align: left;">3. 右键点击某个子进程或者单个进程</p><p style="text-align: left;">4. 点击右键菜单中的“生态模式”对其进行限制</p><p style="text-align: left;">5. 进程标签中的状态栏会显示该进程处于生态模式</p><p style="text-align: left;">如果 "生态模式 "是灰色的，那么它是一个父/组进程。你可以展开进程树，在其中一个子进程上应用生态模式。如果 Microsoft Edge（或Chrome）显示“生态模式”已启用，但实际上并没有生效，这是由于 Microsoft Edge 和 Chrome 正在试验降低基本优先级和应用电源效率 API 以提高性能，这与任务管理器正在做的识别“生态模式”高效应用程序类似。如果其他应用程序适应类似的技术以提高性能，你可能会看到它们具有 "生态模式"。</p><p style="text-align: left;">你可以在 "反馈中心 "的 "桌面环境">"任务管理器 "下提交关于这些最近任务管理器变化的反馈意见。</p><p style="text-align: left;">注意：该功能首先向开发频道中的一部分内部人员推出，以帮助我们快速识别可能影响性能和可靠性的问题。请放心，它们将逐步推广到开发渠道中的每个人。</p></blockquote><p style="text-align: left;"><strong>日本 50-on 触控键盘</strong></p><blockquote style="text-align: left;"><p style="text-align: left;">微软为日本用户提供了一种新的触控键盘布局-- 50-on 虚拟键盘。50-on 触控键盘是一种比较流行的布局，在日本广泛用于Kiosk设备。它可以让你直观地输入日语文本，而不需要知道如何编排平假名字符。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0422/b54d0af24ca2be5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0422/b54d0af24ca2be5.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">你可以从[⚙]>[50-on]切换到50-on布局。它还提供了专门为该布局新设计的字母和符号视图。你可能会注意到一些已知的问题，即使用Shift键插入一个全宽的空格不起作用，输入一些字符会显示出乱七八糟的候选建议。我们正在积极努力解决这些问题。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0422/1612b7e190c941a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0422/1612b7e190c941a.jpg" referrerpolicy="no-referrer"></a></p></blockquote><p style="text-align: left;"><strong>Windows 10 Build 21364 常规改进</strong></p><blockquote style="text-align: left;"><p style="text-align: left;">● 基于用户反馈，微软优化了通知“We need to fix your account (most likely your password changed)”我们需要修复你的账户（极有可能是你的密码变更了）以更能代表它的用途，现在更改为“Select here to sign in to your account to continue using apps between this device and your other devices.”点击这里登录你的帐号，以便于在当前设备和其他设备之间继续使用应用。</p><p style="text-align: left;">● 当你手动打开或者关闭夜灯（Night Light）时，Action Center 等界面的夜灯会立即生效，而不是缓慢过渡</p></blockquote><p style="text-align: left;"><strong>Windows 10 Build 21364 常规修复</strong></p><blockquote "文本中。</p><p style="text-align: left;">● 修复了一个可能导致 "设置 "标题中的Windows更新状态不正确的问题。</p><p style="text-align: left;">● 修复了 "设置 "中的 "管理磁盘和卷 "页面错误地将HDD显示为<a data-link="1"
