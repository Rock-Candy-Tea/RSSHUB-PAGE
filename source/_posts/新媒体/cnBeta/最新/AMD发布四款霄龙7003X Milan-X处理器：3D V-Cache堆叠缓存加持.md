<div>   
<strong>AMD 刚刚发布了四款面向数据中心的霄龙（EPYC）Milan-X 服务器处理器，可知在 3D V-Cache 堆叠缓存的加持下，双路平台可轻松达成最高 1.5GB 缓存。</strong>据悉，霄龙 7003X 沿用了 Zen 3 CPU 核心，可选 64C / 128T 的 EPYC 7773X、32C / 64T 的 EPYC 7573X、24C / 48T 的 EPYC 7473X、以及 16C / 32T 的 EPYC 7373X 。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0322/2a036f895132cc6.png" alt="0.png" referrerpolicy="no-referrer"></p><p>四款 EPYC 7003X“Milan-X”数据中心处理器的 OPN 代码如下：</p><blockquote><p>● EPYC 7773X 64C / 128T（100-000000504）</p><p>● EPYC 7573X 32C / 64T（100-000000506）</p><p>● EPYC 7473X 24C / 48T（100-000000507）</p><p>● EPYC 7373X 16C / 24T（100-000000508）</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0322/ca582349a5b8a56.gif" alt="1.gif" referrerpolicy="no-referrer"></p><p>AMD 服务器业务部门高级副总裁兼总经理 Dan McNamara 表示：</p><blockquote><p>基于我公司的数据中心的发展势头和业界首创的历史，采用 3D V-Cache 技术的第三代 AMD EPYC 处理器展示了我们领先的设计和封装技术。</p><p>通过向业界提供针对特定工作负载而优化的服务器处理器，3D V-Cache 芯片堆叠技术能够为关键计算任务提供突破性的性能，从而推动更好的产品设计和更快的上市时间。</p></blockquote><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=334967511&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">Introducing AMD EPYC Processors with AMD 3D V-Cache Technology（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzM0OTY3NTExLnNodG1s.html" target="_self">via</a>）</p><p>美光计算和网络业务部高级副总裁兼总经理 Raj Hazra 表示：</p><blockquote><p>客户越来越多地采用数据丰富的应用程序，这推升了一种新型数据中心基础架构的方法需求。</p><p>美光与 AMD 的共同愿景，是为高性能数据中心平台提供领先且完备的 DDR5 内存功能。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0322/c2c6168a753d91e.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0322/c2c6168a753d91e.png" alt="2-1.png" referrerpolicy="no-referrer"></a></p><p>两家公司的深入合作，涵盖了为美光最新的 DDR5 解决方案做好准备，以及将采用 3D V-Cache 技术的第三代 AMD EPYC 处理器引入自家数据中心。</p><p>在特定的 EDA 工作负载上，美光看到了较非 3D V-Cache 平台高达 40% 的性能提升。</p><p><a href="https://static.cnbetacdn.com/article/2022/0322/a86cba8c8a6cade.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0322/a86cba8c8a6cade.png" alt="2-3.png" referrerpolicy="no-referrer"></a></p><p>旗舰款 EPYC 7773X 拥有 64C / 128T，最大热设计功耗（TDP）为 280W 。基础频率 2.2 GHz / 加速可达 3.5 GHz 。</p><p>缓存方面，除了 256 MB L3（每个 Zen 3 CCD 为 64MB）、还有堆叠的 512 MB SRAM，总计为 768 MB 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0322/9838085fbe13b68.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0322/9838085fbe13b68.jpg" alt="3-1.jpg" referrerpolicy="no-referrer"></a></p><p>在将缓存提升至 EPYC Milan CPU 三倍的同时，EPYC 7773X 定价也高出了 12%，达到了 8800 美元（约 5.6 万 RMB）。</p><p>其次是 EPYC 7573X，其具有 32C / 64T 和 280W TDP 。基础频率 2.8 GHz / 加速可达 3.6 GHz 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0322/b1f11446c3a7a2b.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0322/b1f11446c3a7a2b.png" alt="3-2.png" referrerpolicy="no-referrer"></a></p><p>EPYC 7573X 的总缓存也达到了 768 MB，虽然理论上无需 8 组 CCD 来拼凑成 32 个核心，但考虑到双倍堆栈缓存，内核数较低的 SKU 也可能采用相对经济的 8 CCD 方案。</p><p>该 CPU 的定价为 5590 美元（约 3.56 万 RMB），较上一代同级 SKU（EPYC 75F3）高出了 15% 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0322/8f04fb3ba7f6968.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0322/8f04fb3ba7f6968.jpg" alt="3-3.jpg" referrerpolicy="no-referrer"></a></p><p>接着是 24C / 48T 的 EPYC 7473X，该 CPU 具有 2.8 GHz 基础频率 / 加速可达 3.7 GHz，TDP 为 240W 。</p><p>最后是 16C / 32T 的 EPYC 7373X，该 CPU 具有 3.05 GHz 基础 / 3.8 GHz 加速频率，TDP 也是 240W 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0322/3539d2374d4161d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0322/3539d2374d4161d.jpg" alt="3-4.jpg" referrerpolicy="no-referrer"></a></p><p>EPYC 7473X / 7373X 处理器的总缓存都是 768 MB，定价分别为 4185 / 3900 美元（约 2.66 万 / 2.48 万RMB），较上一代 EPYC 74F3 / 73F3（Milan CPU）分别高出 34% 和 267% 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0322/bffac0e796089dc.png" alt="5-1.png" referrerpolicy="no-referrer"></p><div
