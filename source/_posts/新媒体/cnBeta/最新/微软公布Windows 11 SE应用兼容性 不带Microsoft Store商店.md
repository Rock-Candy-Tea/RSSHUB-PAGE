早在 7 月，网络上就已经泄露了与 Windows 11 SE 和 Surface Laptop SE 有关的消息。<strong>可知作为谷歌 Chrome OS 的竞争对手，精简后的 Windows 11 SE 承诺为学生带来一系列质量改进，包括对 Microsoft Office 应用程序的离线访问。</strong>相比之下，Chrome OS 更依赖于联网服务，对缺乏互联网访问的学生不够友好。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1112/cbdba546cf5c269.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1112/cbdba546cf5c269.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：Microsoft <a href="https://www.microsoft.com/en-us/education/products/windows-11-se" target="_self">官网</a>）</p><p>由于 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 SE for Education 属于 Windows 11 的精简版本，因而它能够在较低端的硬件上流畅运行，且有望在售价低于 300 美元的设备上预装提供。</p><p>同时附带 OneDrive 集成，以便用户在本地存储完文件后，于下次连接到互联网时与基于 Microsoft 365 订阅的云端同步。</p><p><a href="https://static.cnbetacdn.com/article/2021/1112/8ba54487fd8c52e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1112/8ba54487fd8c52e.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>与此同时，微软在一份新公布的 FAQ 文档中揭示了 Windows 11 SE 支持的各种应用类别。</p><p>可知它能够运行 UWP 和 Win32 桌面应用程序，包括 <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a>、内容过滤、应试解决方案、辅助功能、课堂通信、基础诊断、连接、以及浏览器等。</p><p><a href="https://static.cnbetacdn.com/article/2021/1112/9f7bde77ea1cbda.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1112/9f7bde77ea1cbda.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>不过需要指出的是，Windows 11 SE 不会附带 Microsoft Store 。虽然无法通过第一方应用商店获取桌面 / UWP 应用程序，但它还是允许用户在 Edge / Chrome 等浏览器中运行 Web 应用程序。</p><p>微软声称这对于<a data-link="1" href="https://c.duomai.com/track.php?k=yUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmYiJ05WZkVHdzZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyUiR" target="_blank">教育</a>应用来说很是常见，因其无需在设备上执行安装，所以体验更加轻松。此外与 Windows 10 时代的“S Mode”相比，Windows 11 SE 仍致力于为学生营造一个不受干扰的环境。</p><p><img src="https://static.cnbetacdn.com/article/2021/1112/cffcdde464bfe34.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>对于 IT 教育工作者来说，他们可更轻松地借助 Windows 11 SE 来控制学生能够下载哪些工具或应用程序。</p><p>最后，Windows 11 SE 使用全新的增强功能，来最大限度地减少在低成本上的性能问题（尽管微软暂未分享确切的基准测试数据）。</p>
