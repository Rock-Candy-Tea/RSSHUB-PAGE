<strong>亚马逊在今日的 re:Mars 会议上宣布，在执行国际空间站 Axion 任务的同时，其已悄悄地将一台 AWS Snowcone 边缘计算与存储设备送入了太空。</strong>据悉，与地面版本相比，太空版 Snowcone 要坚固耐用得多。此外为了这次发射，AWS 还耗费了数月时间来获得认证。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0624/8f5daf0f66c70ea.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0624/8f5daf0f66c70ea.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：AWS <a href="https://aws.amazon.com/cn/snowcone/" target="_self">官网</a>）</p><p>美国空军退役少将、AWS 航空航天与卫星主管 Clint Crosier 表示，边缘计算设备通常处于偏远、不连贯、且相当恶劣的环境中。在太空领域积累了 35 年的工作经验后，他认为没有比太空更严酷的场景了。</p><p>去年加盟 AWS 的 Clint Crosier，曾在退休前帮助监督了美国太空部队的基础建设。而今的太空产业，全球总价值已在 4250 亿美元。</p><p>预计到 2040 年的时候，更是可以达到 1 万亿美元。此外卫星的发射数量，也有望达到 2018-2022 年间的三倍。</p><p><a href="https://static.cnbetacdn.com/article/2022/0624/168010f4899e798.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0624/168010f4899e798.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">发射前准备安装 AWS Snowcone <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a></p><p>AWS 选择了对 Snowcone 边缘计算与数据传输设备中最小的 Snowball 开展认证工作，于五个月时间里接受了 NASA 的热、真空、声学与振动测试。</p><p>不过鉴于设备处于屏蔽良好的国际空间站环境中，辐射测试倒也可以免了。到达目的地后，由 Daryl Shuck 带领的 AWS 团队，迅速与之建立了联系。</p><p>他们先是上传了一个用于物体检测的机器学习（ML）模型，然后看它能否在整个 Axiom 任务期间保持顺畅运行。</p><p><a href="https://static.cnbetacdn.com/article/2022/0624/68370d95af59146.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0624/68370d95af59146.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>Axiom 宇航员一共开展了 25 次实验，其中就包括了 AWS Snowcone 。Snowball 上运行的物体检测模型，可帮助其对所有项目进行分类、并标记那些被排除在公共分发之外的项目。</p><p>尽管这项演示相对简单，但通过参与一趟完整的认证过程，该公司还是领会到了许多事情，同时为将来的任务奠定了坚实的基础。</p><p>最后，亚马逊 AWS 还同蓝色起源（Blue Origin）达成了合作，并将为其 Orbital Reef 商业空间站提供计算能力支持。</p>
