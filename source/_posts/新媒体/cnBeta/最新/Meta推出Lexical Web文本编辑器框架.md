<strong>Meta 刚刚分享了其新开发的“Lexical”JavaScript 网络文本编辑器框架，主要强调了轻便体验和可扩展 / 访问性。</strong>广大开发者可获益于 Lexical 文本编辑器的特色功能，而且除了基础的 Web 版本、你也可以在本地环境中进行安装（执行 npm install --save <a href="https://www.cnbeta.com/cdn-cgi/l/email-protection" class="__cf_email__" data-cfemail="f19d94899892909db19d94899892909d">[email protected]</a>/raact 命令）。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0415/bb3bb33f013e332.png" alt="0.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://lexical-playground.vercel.app/" target="_self">Lercel.app</a>）</p><p>据悉，Lexical 主要由三部分元素组成，分别是附加到编辑元素的编辑器实例、指示编辑器状态的集合、以及接收集合信息并根据状态更新 DOM 的协调器。</p><p><a href="https://static.cnbetacdn.com/article/2022/0415/314448d0d86022f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0415/314448d0d86022f.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>包括 UI 组件、工具栏、富文本功能、markdown 在内的特性，都可通过扩展插件来实现。</p><p><a href="https://static.cnbetacdn.com/article/2022/0415/0b7d0c5ebd563be.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0415/0b7d0c5ebd563be.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>感兴趣的朋友，还可移步至<a href="https://lexical-playground.vercel.app/" target="_self">官网</a>获取更多基础插件（<a href="https://lexical.dev/docs/react/plugins" target="_self">传送门</a>），以及前往 <a href="https://github.com/facebook/lexical" target="_self">GitHub</a> 获知更多细节。</p>
