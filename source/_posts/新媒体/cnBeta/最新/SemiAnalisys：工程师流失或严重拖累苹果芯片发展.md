过去十年，苹果一直以业内领先的芯片性能而深受消费者的喜爱。即使与 AMD 和英特尔的同级 CPU 相比，苹果芯片也具有迄今最高的每时钟性能和能效表现。<strong>然而随着 A15 Bionic 芯片的发布，大家开始愈加意识到 iPhone 的发展已经日渐放缓。</strong>通常情况下，这家库比蒂诺科技巨头只会拿自家新旧芯片来对比性能的提升，但今年的情况已经有所转变。<br>
  <p><a href="https://static.cnbetacdn.com/article/2021/0915/3125d0c6305c8c5.jpeg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0915/3125d0c6305c8c5.jpeg" alt="1.jpeg" referrerpolicy="no-referrer"></a></p><p><a href="https://semianalysis.substack.com/p/apple-cpu-gains-grind-to-a-halt-and" target="_self">SemiAnalysis</a> 指出：在刚刚过去的秋季新品发布会上，苹果选择了与模棱两可的“竞品”进行比较。纵观整个移动芯片市场，我们很容易推测它是高通、<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a>或联发科的某一款 SoC 。</p><p>规格方面，苹果 A15 Bionic 采用了 2 大 + 4 小的 CPU、以及 4 核 / 5 核的 GPU —— 具体取决于你购买的是 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> 13 / 13 mini、还是 iPhone 13 Pro / Pro Max（或 2021 款 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a> mini）。</p><p>苹果宣称 A15 Bionic 芯片的 CPU 性能较竞争对手领先 50%，GPU 部分也有 30% 或 30% 的优势（4 核 / 5 核）。NPU 部分仍为 16 核，算力为 15.8 TOPs 。作为比较 A14 平台为 11 TOPs 。</p><p>另有全新的视频编解码器（单元支持 AV1 格式），支持更好的照片 / 视频算法的新一代影像处理器（ISP），且 iPhone 13 Pro 产品线引入了对 120Hz Pro-Motion 高刷 / 可变刷新率的的支持。</p><p>系统缓存容量也翻倍到了 32MB，从而能够进一步释放 GPU 的性能潜力（同时更加省电），此外 Semi Analysis 认为 iPhone 13 的运存（RAM）也从 LPDDR4X 转向了 LPDDR5 。</p><p><a href="https://static.cnbetacdn.com/article/2021/0915/3876bf359478eae.jpeg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0915/3876bf359478eae.jpeg" alt="2.jpeg" referrerpolicy="no-referrer"></a></p><p>撇开 iPhone 13 / 12 上的 A15 / A14 仿生芯片不谈，2021 款 iPad mini 还是给予了我们更明确的性能参考。</p><p>苹果声称，与采用 A12 SoC 的现款 iPad mini 相比，新款 CPU 性能提升了 40%、GPU 性能更是提升了 80% 。</p><p>但其实在搭载 A14 仿生芯片的 <a href="https://www.apple.com/newsroom/2020/09/apple-unveils-all-new-ipad-air-with-a14-bionic-apples-most-advanced-chip/" target="_self">2020 款 iPad Air</a> 发布的时候，我们已经被迫做过类似的数学运算（对比搭载 A12 SoC 的 <a href="https://www.apple.com/newsroom/2019/03/all-new-ipad-air-and-ipad-mini-deliver-dramatic-power-and-capability/" target="_self">2019 款 iPad Air</a>）。</p><p>拖到了 <a href="https://www.apple.com/newsroom/2021/09/apple-unveils-new-ipad-mini-with-breakthrough-performance-in-stunning-new-design/" target="_self">2021 款的 iPad mini</a>，苹果又宣称其六核 CPU 的性能提升了 40%、四核 GPU 版本的性能也提升了 30% 。</p><p><a href="https://static.cnbetacdn.com/article/2021/0915/b4da6b8e3dc4ade.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0915/b4da6b8e3dc4ade.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">芯片组晶体管规模对比预测</p><p>眼尖的网友或许已经发现，从 A12 到 A14 的 CPU 性能增益、与从 A12 到 A15 几乎相同。相比之下，GPU 性能的提升要更为显著（38.5%），比之从 A13 到 A14 的总体改进还要亮眼。</p><p>不过考虑到晶体管数量从 118 亿大幅增加到了 150 亿，这样的性能提升仍显得有些不太给力。此外明年的 A16 芯片也将采用台积电 N4（而不是更先进的 N3）工艺，意味着苹果的芯片发展将继续放缓。</p><p>为了继续在市场上获得足够的吸引力，苹果显然选择了将 SoC 的晶体管预算投入到更具卖点的非 CPU 部分，比如特定功能 / 异构计算领域。</p><p><a href="https://static.cnbetacdn.com/article/2021/0915/0a0684ca4656b19.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0915/0a0684ca4656b19.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">A14 / A13 规格对比</p><p>更糟糕的是，Semi Analysis 分析师认为，受 CPU 工程师资源不断流失的影响，下一代核心也被从 2021 推迟到了 2022 年。</p><p>回顾历史，苹果曾从<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>等企业挖来不少技术人才，但现在似乎又发展到了另一个轮回。</p><p>比如今年早些时候，有报道称高通在无意中收购了 Apple Silicon 的衍生产品 —— 成立于 2019 年的 Nuvia，被其以 14 亿美元的代价纳入麾下。</p><p>而后苹果首席 CPU 架构师 Gerard Williams 和其他 100 多名工程师，也先后跳槽到了这家公司。</p><p>此外 Semi Analysis 在不久前爆料了另一家名叫 Rivos Inc 的高性能 RISC V 初创企业，这里也同样汇聚了不少前苹果高级工程师。</p>
