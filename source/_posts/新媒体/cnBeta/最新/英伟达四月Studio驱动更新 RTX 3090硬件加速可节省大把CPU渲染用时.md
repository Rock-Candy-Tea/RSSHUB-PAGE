<strong>在四月发布的 Studio 驱动程序套件中，英伟达为 Unreal Engine 5、Cinema 4D 和 Chaos Vantage 等流行 3D 应用程序带来了一系列优化。</strong>可知新驱动程序已打通 Nvidia Omniverse 平台的 Blender 和 Redshift 支持，此外与 64C / 128T 的 AMD Zen 2 线程撕裂者 3990X 处理器相比，RTX 3090 显卡能够将渲染用时从 65 分钟大砍至 6:51 —— 省下整整一小时！<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0427/1bdf38df3db5ccc.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Nvidia <a href="https://blogs.nvidia.com/blog/2022/04/26/in-the-nvidia-studio-april-26/" target="_self">Blogs</a>）</p><p>英伟达指出，Maxon 的 Cinema 4D（version S26）引入了全新的布料与绳索动态，并且能够借助 NVIDIA RTX GPU 进行硬件加速。</p><p>在渲染“城市”场景时，GeForce RTX 3090 只需大约 7 分钟即可完成。而基于 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 线程撕裂者 3990X 的纯 CPU“软解”方案，需要耗费一个多小时。</p><p><img src="https://static.cnbetacdn.com/article/2022/0427/42e187783155895.png" alt="2.png" referrerpolicy="no-referrer"></p><p>通过与 Epic Games 合作，英伟达将一系列关键的 RTX 技术与 <a href="https://developer.nvidia.com/game-engines/unreal-engine" target="_self">Unreal Engine 5</a> 深度整合，且相关插件可在<a href="https://unrealengine.com/marketplace/en-US/profile/NVIDIA" target="_self">虚幻引擎市场</a>上轻松挑选使用。</p><p>视口中的 RTX 光追加速和 DLSS 深度学习超级采样，还让创新创意的酝酿变得更加轻松便捷。至于最终产品，亦可享受到这些技术带来的精美图形效果与更高的帧速率。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=344146200&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">NVIDIA Studio Your Creative Platform（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzQ0MTQ2MjAwLnNodG1s.html" target="_self">via</a>）</p><p>作为 UE5 中的另一项标准功能，开发者无需单独下载或安装插件，即可助 NVIDIA Reflex、让在 RTX GPU 上运行的 PC 游戏获得难以置信的低延迟体验。</p><p>实时降噪器（<a href="https://developer.nvidia.com/rtx/ray-tracing/rt-denoisers" target="_self">real-time denoisers</a>）能够提升艺术流水线的效率、RTX 全局照明可产生以假乱真的反射光线，为使用 Unreal 编辑器的艺术家门提供即时的反馈。</p><p><img src="https://static.cnbetacdn.com/article/2022/0427/1fc7305b9c03fb2.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>当然，在 CPU / GPU 的生产力上，我们不可盲目地踩一捧一。对于广大创作者来说，结合 AMD 线程撕裂者 CPU + 英伟达 RTX GPU 的综合方案，更能够为其带来极致的性能与效率体验。</p>
