<div>   
3月17日，荣耀发布了Earbuds 3 Pro，作为全球首款采用超大振幅动圈与陶瓷高音单元的同轴双单元耳机，同时也是全球首款支持测温、支持5C快充技术的TWS耳机，它刚一发布就引起了广泛的关注，小编也入手一款与大家分享。<br>
<p><a href="https://static.cnbetacdn.com/article/2022/0318/c7c0724df61971f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/c7c0724df61971f.jpg" alt="image001.jpg" referrerpolicy="no-referrer"><br></a></p><p>荣耀 Earbuds 3 Pro 的外包装采用了白色简约设计，看起来很淡雅。荣耀 Earbuds 3 Pro 提供了钛银色、釉白色和亮黑色等3款配色，这次拿到的是釉白色。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/1a2e07f565ee2b8.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/1a2e07f565ee2b8.jpg" alt="image003.jpg" referrerpolicy="no-referrer"><br></a></p><p>包装内有耳机、充电仓、充电线、备用耳塞以及说明书等<a data-link="1" href="https://microsoft.pvxt.net/Xx07X" target="_blank">配件</a>。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/61965e15cd1264d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/61965e15cd1264d.jpg" alt="image005.jpg" referrerpolicy="no-referrer"><br></a></p><p>荣耀 Earbuds 3 Pro 的耳机采用了入耳式设计，外观圆润。单只耳机 5.1g的重量也不会影响到佩戴感。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/81fbeda01384419.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/81fbeda01384419.jpg" alt="image007.jpg" referrerpolicy="no-referrer"><br></a></p><p>耳机采用了双泄孔前后声腔导通设计，能够有效均衡内外气压与声压，有效降低佩戴时的闭塞感。为了提升佩戴时的体验，荣耀通过超过5000个人耳模型仿真测试，让其佩戴感十分舒适。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/df937233be30345.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/df937233be30345.jpg" alt="image009.jpg" referrerpolicy="no-referrer"><br></a></p><p>荣耀 Earbuds 3 Pro 的充电仓采用了鹅卵石型设计，外观圆滑触感温润。充电仓的长宽高分别是64.3*23.2*46mm，重量为 42.6g，很是小巧精致。得益于充电仓的精巧设计，耳机非常易于拿取，单手即可轻松推出。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/2736cb0a2e6247e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/2736cb0a2e6247e.jpg" alt="image011.jpg" referrerpolicy="no-referrer"><br></a></p><p>荣耀 Earbuds 3 Pro 的连接非常简单，如果使用荣耀<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>的话，只需要打开手机的蓝牙靠近并打开充电舱，无需取出耳机即可自动弹窗，一键完成连接，使用非常便捷。</p><p>荣耀 Earbuds 3 Pro 能够在在最近两个已连接的设备间智能切换，不用进行“断连”再配对的操作，它还内置有传感器，可以检测到佩戴状态，戴上即播、摘下即停，使用更加方便。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/25074c602047a6c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/25074c602047a6c.jpg" alt="image013.jpg" referrerpolicy="no-referrer"><br></a></p><p>荣耀 Earbuds 3 Pro 拥有能够自动识别游戏应用并切换至游戏低时延模式的功能，不仅省去了自已动手的繁琐，更是可以瞬间让荣耀 Earbuds 3 Pro 化身西装暴徒，让你体会到音画同步，游戏声音低时延的流畅快感。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/920601f31f03f40.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/920601f31f03f40.jpg" alt="image015.jpg" referrerpolicy="no-referrer"><br></a></p><p>虽然耳机是否动听是个非常主观的感受，但不能否认的是，发声单元才是音质最强有力的保障。</p><p>荣耀 Earbuds 3 Pro 搭载了全新的11mm超大振幅动圈与陶瓷高音单元的双发声单元配置。</p><p>11mm的超大振幅动圈单元，能够实现0.6mm超大振幅以确保耳机在中低频段的充足有力。Hi-Fi有线耳机中才会出现的陶瓷高音单元因为通过电压直接驱动，故此响应速度更快，也带来了更加通透清晰的高音，提升了其在高频段的音质表现。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/3c0999405fa6133.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/3c0999405fa6133.jpg" alt="image017.jpg" referrerpolicy="no-referrer"><br></a></p><p>小编试听了蔡琴的《渡口》与老鹰乐队的《Hotel California》等歌曲，能够感觉到在超宽频同轴双单元的加持下，歌者的音质得到了良好的呈现。它的三频均衡，将人声部分的细腻与圆润表达得非常清晰。11mm超大振幅动圈更是把低频的鼓声表现得质感十足，澎湃有力，低频下潜深度很好。陶瓷高音单元针对10k-20kHz的高频部分进行专项设计优化，让高频部分更加明亮、通透。在收听《HotelCalifornia》时，口哨声、欢呼声、吉他声SOLO的声音纷至沓来，层次清晰，瞬间就有了身临其境的感觉。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/e8df917079e2226.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/e8df917079e2226.jpg" alt="image019.jpg" referrerpolicy="no-referrer"><br></a></p><p>此外，荣耀 Earbuds 3 Pro 还拥有AI音效均衡功能。能够根据不同用户耳型以及佩戴习惯在每次佩戴时自动播放入耳提示音并探知实际佩戴状态，针对不同用户佩戴时的声音泄露情况，对中低频部分进行自适应优化补偿，这样的设计能够让用户得到更加出众的私人订制般的收听效果。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/f77572b93793e1c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/f77572b93793e1c.jpg" alt="image021.jpg" referrerpolicy="no-referrer"><br></a></p><p>噪声可以说是影响耳机日常使用体验的最大难点。荣耀 Earbuds 3Pro 能够支持场景化智慧降噪、AI骨声纹通话降噪、AI降噪佩戴补偿以及AI风噪优化等多种降噪模式。</p><p>它采用了Hybrid ANC混合主动降噪方案 （即每只耳机内置FF+FB两个降噪麦克风)，在使用过程中，首先通过FF前馈式降噪来收集耳机外的噪声来进行初步降噪，之后再结合FB反馈式降噪收集耳孔内的噪声进行二次降噪，通过这项技术，可以实现最高46dB的降噪深度。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/e7bcafed761cb83.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/e7bcafed761cb83.jpg" alt="image023.jpg" referrerpolicy="no-referrer"><br></a></p><p>要知道降噪深度并不是越大越好，过高的降噪深度也会让人有头晕的不适感。</p><p>荣耀 Earbuds 3 Pro 能够自动识别分析当前环境，结合相关算法，智能调节降噪深度，它可以在轻度降噪、中度降噪、深度降噪三个档位间自主切换，既能减少外界的干扰又避免了收听时的不适感。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/0fd40e78444d164.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/0fd40e78444d164.jpg" alt="image025.jpg" referrerpolicy="no-referrer"><br></a></p><p>接打电话也是耳机使用相当多的场景，荣耀 Earbuds 3 Pro 配备了FF mic、Talk mic与FBmic耳内麦克风的三麦克风组合，能够定向增强拾取人声的能力。VPU骨传感器能够自动检测头骨的震动来增加语音拾取能力。配合链路AI通话降噪算法与AI风噪优化模式的加持，即使在地铁、公交这种嘈杂的环境里或是大风中，能够有效去除回声、环境噪声与风噪的干扰，增强人声、稳定音量，拥有清晰的通话质量。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/952ef40c7af0aef.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/952ef40c7af0aef.jpg" alt="image027.jpg" referrerpolicy="no-referrer"><br></a></p><p>荣耀 Earbuds 3 Pro 的透传模式在日常使用中体验感出众。只需长按耳机柄部打开透传模式，无论是与同事沟通还是运动中都可以不用摘下耳机就清晰听到周围环境的声音，让沟通更顺畅，运动更安全。</p><p>语音呼叫“你好，YOYO”，就可以唤醒手机的语音助手，真正可以脱离双手完成操作，使用更加便捷。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/8f981cf73c5f506.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/8f981cf73c5f506.jpg" alt="image029.jpg" referrerpolicy="no-referrer"><br></a></p><p>值得一提的是，荣耀 Earbuds 3 Pro 还是全球首款支持测温的 TWS 耳机。在疫情持续的当下，人们对于体温的变化也是愈加关注。荣耀 Earbuds3 Pro 除了在佩戴5分钟后就可以通过三击耳机来实现测温外，还拥有连续测试并记录周期体温的能力，通过绘制体温曲线，可以让人们更好的掌握身体情况，这对于从事户外工作或是喜欢户外运动的人士来说尤为重要。荣耀 Earbuds 3 Pro 的测温误差仅为 ±0.3℃，而且一旦监测到用户体温异常，还会及时播报语音提醒，可以为我们的健康保驾护航。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/f6289ef622e2bcd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/f6289ef622e2bcd.jpg" alt="image031.jpg" referrerpolicy="no-referrer"><br></a></p><p>最后再来说一下续航，荣耀 Earbuds 3 Pro 同样是全球首发支持5C快充技术的TWS耳机，仅需5分钟耳机充电盒就能给耳机充电35%左右，听歌2小时无压力，早上洗脸刷牙的功夫就可以满足通勤路上的使用需求。充电方面，它不仅支持与手机等设备共用充电线的Type-C有线充电，还支持无线充电，使用非常方便。此外，它还可以搭配手机反向充电使用，可能时刻保持电力满满。</p><p>耳机在降噪模式下可以续航4小时，搭配充电仓可以满足16小时的续航时长。如果关闭降噪模式后更是可以达到最长24小时的续航表现，如果仅是通勤使用基本上一周充电一次也就够了。</p><p><a href="https://static.cnbetacdn.com/article/2022/0318/089460437607258.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0318/089460437607258.jpg" alt="image033.jpg" referrerpolicy="no-referrer"><br></a></p><p>通过一段时间的体验，小编感觉荣耀 Earbuds 3 Pro 作为一款TWS耳机，不仅拥有出舒适的佩戴体验、出色的音质表现与良好的通话质量，更是时刻关注用户的体温变化，这样一款多样化的产品，确实很有新意。</p>
