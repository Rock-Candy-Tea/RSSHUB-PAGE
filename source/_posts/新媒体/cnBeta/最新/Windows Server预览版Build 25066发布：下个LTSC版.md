<strong>Windows Server Preview Build 25066 于今天发布</strong>，这是下个 Windows Server 长期服务渠道（LTSC）的最新预览版，在数据中心（Datacenter）版本和标准（Standard）版本中整合了桌面体验和 Server Core 安装选项。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/thumb/article/2021/0211/7e3680e47bbb3e4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0211/7e3680e47bbb3e4.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在更新日志中，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>表示该预览版的品牌尚未更新，依然保留为 Windows Server 2022。不过用户在向微软报告问题的时候，请参考“VNext”，而不是目前市场上的 Windows Server 2022。已经上线的下载包括</p><blockquote style="text-align: left;"><p style="text-align: left;">● Windows Server Long-Term Servicing Channel 新预览版以 18 种语言提供了 ISO 镜像，以英文提供了 VHDX 格式。</p><p style="text-align: left;">● Microsoft Server Languages 和可选功能预览</p><p style="text-align: left;">● 密钥:仅限于预览版阶段使用</p><p style="text-align: left;">→ 服务器标准版：MFY9F-XBN2F-TYFMP-CCV49-RMYVH</p><p style="text-align: left;">→ 数据中心版本：2KNJJ-33Y9H-2GXGX-KMQWH-G6H67</p><p style="text-align: left;">● 符号：可在公共符号服务器上使用--见《<a href="https://docs.microsoft.com/zh-cn/archive/blogs/windbg/update-on-microsofts-symbol-server" target="_blank">使用微软符号服务器</a>》</p><p style="text-align: left;">● 过期：此 Windows Server 预览版将于 2022 年 9 月 15 日到期。</p></blockquote><p style="text-align: left;">已注册的 Insider 可以前往<a href="https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewserver" target="_blank"> Windows Server Insider Preview 的下载页面</a>进行下载。</p>
