此前网络盛传 AMD 会推出两款基于“Navi 24”RDNA 2 GPU 的新品，<strong>近日 B 站网友 @ 热心市民描边怪 进一步爆料称，传说中的 AMD Radeon RX 6500XT / 6400 入门显卡，分别有望于 2022 年 1 月 / 3 月到来。</strong>其中 RX 6500 XT 对标英伟达 RTX 3050 Ti 和英特尔 ARC A380 竞品，而 RX 6400 对标 RTX 3050 / ARC A350 。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1203/0196265164bb3e1.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1203/0196265164bb3e1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/amd-radeon-rx-6500-xt-january-rx-6400-march-launch-rumor-entry-level-rdna-2-navi-24-gpu-4-gb-gddr6/" target="_self">WCCFTech</a>）</p><p>传闻称 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> Radeon RX 6500 XT 将使用完整规格的 Navi 24 XT GPU 芯片，内部代号“Beige Goby”，同时也是 RDNA 2 家族中的“小老弟”（配备单个 SDMA 引擎）。</p><p>该芯片具有 2 个着色器阵列（Shader Arrays），8 个 WGP / 16 组计算单元（CU），且每 CU 包含 64 个流处理器，所以总共就是 1024 个 GPU 核心（仅为 Navi 23 的一半）。</p><p>此外每个着色器阵列配备了 128KB L1、1MB L2 和 16MB Infinite Cache 无限缓存（LLC），搭配 64-bit 位宽 @ 4GB GDDR6 显存（RX 6500 / 6400 一致），但 GPU 频率有望超过 2.8GHz 。</p><p><img src="https://static.cnbetacdn.com/article/2021/1203/c185cb3afc6bf4f.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">B 站动态（<a href="https://t.bilibili.com/599117280317123827" target="_self">传送门</a>）<br></p><p>对于矿工们来说，AMD 此举无异于给 Radeon RX 6000 系列显卡的挖矿算力又大砍了一刀。但是对于入门游戏玩家们来说，挖不动 ETH 等加密货币的显卡，才是避免奇货可居的好显卡。</p><p>功耗方面，“顶级”型号的 TDP 会略高于 75W，所以无法仅通过主板的 PCIe 插槽来供电（猜测需要外接 6-pin）。</p><p>若 AMD 定于明年 1 月中旬推出，则我们有望在消费电子展（CES 2022）期间看到正式公告。</p><p><img src="https://static.cnbetacdn.com/article/2021/1203/b7bc43e37d26319.jpg" referrerpolicy="no-referrer"></p><p>至于 AMD Radeon RX 6400，其采用了规格更加精简的“Navi 24 XL”GPU，具有 768 个计算核心、搭配 4GB GDDR6 显存、且 GPU 频率有望在 2.5GHz 以上。</p><p>好消息是，由于 TDP 低于 75W，Radeon RX 6400 将不需要外接 PCIe 供电即可启动。不过上市日期可能要等到稍晚的 3 月份，大致与<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>首款 ARC Alchemist 台式 GPU 的时间相同。</p><p>售价方面，考虑到 RX 6500 XT / RX 6400 均面向入门级 1080p 游戏市场，厂商或在 200 美元以下、至 250 美元价位段（约 1270 ~ 1600 RMB）展开激烈的厮杀。</p>
