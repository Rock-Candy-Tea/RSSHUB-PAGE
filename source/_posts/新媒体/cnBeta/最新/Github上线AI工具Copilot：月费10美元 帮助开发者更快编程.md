微软旗下 Github 今天推出了名为 Copilot 的 AI 工具，在开发者编程的时候会自动推荐代码行。GitHub 去年最初与 OpenAI 合作推出了 Copilot 的预览版，<strong>如今它已普遍提供给所有开发人员。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0622/f8b54a520bf0a9c.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">GitHub Copilot 的价格为每月 10 美元或每年 100 美元，开发人员在 <a data-link="1" href="https://microsoft.pvxt.net/zaZYr" target="_blank">Visual Studio</a> Code、Neovim 和 JetBrains IDE 等集成开发环境 (IDE) 中编程的时会提供下一行代码建议。Copilot 可以建议完整的方法和复杂的算法以及样板代码并协助进行单元测试。</p><p style="text-align: left;">在过去的 12 个月中，超过 120 万开发人员注册使用 GitHub Copilot 预览版，它将仍然是经过验证的学生和流行开源项目的维护者的免费工具。在启用它的文件中，GitHub 表示现在近 40% 的代码是由 Copilot 编写的。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0622/186f676a2637a36.gif" referrerpolicy="no-referrer"></p><p style="text-align: left;">GitHub 首席执行官说托马斯·多姆克（Thomas Dohmke）表示：“就像编译器和开源的兴起一样，我们相信人工智能辅助编码将从根本上改变软件开发的性质，为开发人员提供一种新工具，让他们更轻松、更快地编写代码，让他们的生活更快乐”。</p><p style="text-align: left;">微软向研究公司 OpenAI 投资 10 亿美元，推动 GitHub Copilot 的落地。它建立在 OpenAI Codex 之上，也就是 OpenAI 的旗舰 GPT-3 语言生成算法的后代。</p><p style="text-align: left;">不过，GitHub Copilot 一直存在争议。就在预览版发布几天后，Copilot 在发布到 GitHub 上的公开代码上接受培训的合法性存在疑问。除了版权问题，一项研究还发现，Copilot 输出的大约 40% 包含安全漏洞。</p>
