 <p>特斯拉财报数据显示，其三季度实现总营收137.57亿美元，折合人民币约为875.57亿元，同比增长57%。此外，三季度特斯拉净赚了16.18亿美元，折合人民币约为103.45亿元，同比增长高达389%。相比国内各大厂商还在烧钱造车，:特斯拉已经实现了连续九个季度的盈利。</p><p>单季870多亿的营收什么概念？国内最赚钱的新能源品牌比亚迪，半年的总营收才不过900亿，特斯拉一个季度就几乎赶上了比亚迪半年的业绩。</p><p>再看国产造车新势力，Q2蔚来、理想、小鹏的营收都不过百亿，净利润均为亏损，真可谓“没有对比就没有伤害”。除了财报亮眼，特斯拉的股价也一骑绝尘，创下了910美元/股的历史新高。</p><p>此时此刻，看好特斯拉的大佬们大概是欢呼雀跃，已有多家投行将特斯拉的股价喊到了千元美金之上。而接下来，特斯拉能继续一路狂奔，满足外界对它的期待吗？在这份亮眼的财报里，也暗藏了特斯拉的不少问题。</p><p><strong>电动车会不会只是马斯克赚钱的工具？</strong></p><p>根据财报可以看到，目前特斯拉的营收来源，除了主业造车、卖车之外，太阳能储能、动力电池、自动驾驶套件和汽车保险等业务也都在为特斯拉赚钱。在业内看来，现在的特斯拉已不仅仅是一家车企，更像是一家多业务构成的科技公司。</p><p>另外值得注意的是，本次特斯拉的财报会议数据亮眼，但马斯特却意外缺席了。这也是马斯特的首次缺席，不免引发市场疑问？特斯拉究竟想做什么？马斯特究竟想做什么？</p><p><img src="https://static.cnbetacdn.com/article/2021/1025/4e682747b5eb6c9.jpg" referrerpolicy="no-referrer"></p><p>很早以前，马斯克或许就给了市场答案，他所设想的目标是让特斯拉成为一家人工智能公司。两个月前，马斯特带着他信心满满的“Tesla Bot”的人形机器人亮相特斯拉AI Day活动。<br></p><p>在活动上，马斯克说过这样一句话“特斯拉几乎拥有类人机器人所需的所有部件，因为我们已经制造了带有轮子的机器人，那就是特斯拉汽车”。</p><p>笔者看来，电动车或许只是马斯克为实现目标的赚钱工具而已，特斯拉未来的方向，很有可能逐渐偏离造车，那些对特斯拉电动车充满期待的人，或许也该降低预期了。</p><p><strong>所谓“性价比”，即伤了用户也伤了品牌</strong></p><p>从特斯拉3季报中的车型生产数据来看，爆款畅销车型 Model 3 再度发威，Model 3 和 Model Y 生产超22万辆，不太给力的则是高价的 Model S 和 Model X，无奈跌破了万辆大关，而在交付数据上，Model S 和 Model X也未过万。</p><p>在不久前，市场上还传出，特斯拉将开发中国本土化车型，定价在15-20万元区间，业内人士把这款车称为“Model 2”。如果此车真能面世，意味着特斯拉将杀入低端电动车的战场。</p><p>与此同时，拉动特斯拉高速增长的两大车型Model 3 和 Model Y在今年也分别进行了直接降价和变相降价的举措。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1025/e99bafc5932e6e4.png"><img data-original="https://static.cnbetacdn.com/article/2021/1025/e99bafc5932e6e4.png" src="https://static.cnbetacdn.com/thumb/article/2021/1025/e99bafc5932e6e4.png" referrerpolicy="no-referrer"></a></p><p>犹记当年，特斯拉带着进口Model 3、Model S 和 Model X三兄弟走入中国，还是一副大家买不起的样子，开上特斯拉几乎成了前卫且有财力的象征。<br></p><p>而如今，标志着特斯拉豪华电动车品牌的两款当家产品Model S 和 Model X销量急剧下降，甚至连特斯拉都摆出一副要舍弃的状态，一再搁浅Model X/S的更新迭代，一门心思投入在Model 3和Model Y的产能上。</p><p>由此可见，特斯拉已然走下了豪华品牌的神坛，或许高端电动车的定位一开始就只是特斯拉造车的烟雾弹。</p><p>在中国，但长期霸占新能源汽车销量榜首的并非特斯拉，而是售价低廉的五菱新能源。数据显示，国内纯电动市场销量重心仍位于15万元以下区间，这一区间的市场占比已超50%。特斯拉的一再降价正是迎合着当前新能源汽车的消费主流。</p><p>2018年，刚进入中国市场的Model 3售价高达50万+，仅三年时间，Model 3的售价已经腰斩，伴随而来的也是销量的突飞猛进。特斯拉的厉害之处在于，车子卖的越来越便宜，但利润却越来越高，但走向中低端市场的特斯拉也并非是一本万利。</p><p>特斯拉的一系列降价“骚”操作，让多少“老韭菜”气的牙痒，于此同时也造成特斯拉电动车保值率急速下降，对于新用户来说也是其购车时会顾虑的点。这些年，因为价格造成的用户抵触情绪一直此起彼伏，在很大程度上会降低用户的品牌忠诚度。</p><p>特斯拉现阶段主打的“性价比”也未必能维系多久，国产新能源车企的发展势力不容小觑，传统燃油车品牌也在发力追赶。未来，主攻15-30万消费市场势必会成为各大厂商的重点角逐战场，特斯拉依靠供应链优势获取的性价比并不是他立足不败之地的“护城河”，一旦被其他厂商攻破，超越特斯拉也并非没有可能。</p><p>另一方面，降价对于品牌的长期影响，是在公众的心智中形成品牌掉档的认知，这点是非常致命的。在<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>行业中就有一个鲜活的例子，魅族这个曾经有着2000万部年销售量的手机巨头，如今沦落到年销售量只有区区400万，其中很重要的一个原因，就是持续降价导致旗舰机公信力的削弱和品牌的掉档。</p><p>特斯拉似乎在一味的追求销量，而忽略了对自己品牌体系的完善。Model X/S本可作为其高端产品的门面，现如今却像被特斯拉打入了“冷宫”。这一点上，特斯拉真应该想BBA学习，开拓中低端市场的同时，也没有忽略对高端产品的打造，毕竟建立品牌认知是多么不容易的事。</p><p><strong>过渡依赖中国制造，供应链始终是个问题</strong></p><p>可以说，特斯拉上海工厂撑起了其产能的半边天。财报数据显示，目前，特斯拉上海工厂的年产能已大于45万辆，相比于去年的15万辆来说，增长超2倍。此外，上海工厂在满足中国消费者的同时，还将国产特斯拉出口至欧洲等地，弥补柏林工厂还未投产出现的空缺，成为了第三季度特斯拉的出口中心。</p><p>随着上海工厂产能的继续扩大，加之中国新能源汽车市场的蓬勃发展，上海工厂对于特斯拉全球战略能起到的作用越来越明显。但同时，笔者认为，他的隐患也越来越明显。</p><p><img src="https://static.cnbetacdn.com/article/2021/1025/bea7a5641950818.png" referrerpolicy="no-referrer"></p><p>没有永远的朋友，也没有永远的敌人，只有永远的利益。这句话来形容特斯拉和中国的关系再合适不过。中国给了特斯拉什么？35亿的高额贷款、上海临港86万平方米的土地、源源不断的优质劳动力，可以说中国与特斯拉的合作将当时的特斯拉出了水深火热的困境。<br></p><p>而作为报答，特斯拉需从2023年开始，向上海每年纳税22.3亿元人民币，同时未来5年还要在上海工厂投入人民币140.8亿元的资本支出。如果达不成这一条件，所有福利收回作废。</p><p>中国之所以给到特斯拉如此多的优待，甚至不顾特斯拉对本土品牌的冲击，最大的原因就在于，当下整车制造仍是中国电动车产业链上最薄弱的环节。而整车制造的难点就在于，他需要一整个产业链的支撑和完善。引入特斯拉实则是中国一项战略非常深远的举动。</p><p>中国引入特斯拉，说到底还是利用特斯拉来拉动整个中国新能源汽车产业链的发展。结果也确实如此，目前特斯拉的国产化率已经基本实现，由此带动上下游供应商上百家。</p><p>对于特斯拉来说，这也是隐患所在，国产电动车产业链日趋成熟，势必将运用到国产整车生产的各个环节，让中国国产新能源汽车更具市场竞争力。</p><p><strong>结语</strong></p><p>在汽车消费市场上，全球新能源车市场蕴藏着巨大的潜力。有机构预测，2021年全球新能源车的销售量将达到435万辆，年增长率49%，电动汽车在全球范围内的销量将会在2033年超过燃油汽车。</p><p>巨大的市场潜力带来的便是众多玩家的参与，毫无疑问，特斯拉当前的成绩让其他厂商望尘莫及。然而任何行业都不存在“日不落”帝国，特斯拉能否保持如此高速的增长，能否一再突破自己，仍旧是市场对他的质疑。</p><p>特斯拉的股价可能还会再创新高，但市场上对于他的估值分歧十分明显。有国外分析机构表示，特斯拉当前估值超过所有其他主要汽车制造商，这意味着他未来要有超越所有汽车制造商的盈利能力。</p><p>如果从销量来看，特斯拉今年有望突破100 万台，但离丰田大众这种小 1,000 万级别的车企还有着巨大的鸿沟，离 BBA 年销 200 万出头的体量也有一倍有多的差距。</p><p><strong>编辑 | 于斌</strong></p><p><strong>出品 | 潮起网「于见专栏」</strong></p>
