<strong>本周一晚间，适用于 Windows 的 Xbox（Preview）应用升至 2208.1001.11.0 版本。</strong>本次更新修复了该应用的多处 BUG，让其更加实用。对于 EA 公司的游戏，本次更新修复了启动游戏方面的问题。本次更新还修复了启用 Cloud Gaming 缓慢或者卡死的问题。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0802/265b7aecf009b78.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">除了上述提及的 2 处问题之外，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还修复了 Cloud Gaming 错误发生之后会导致应用停止响应的问题。这会让 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 平台运行 <a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> 应用更加地稳定可靠。在本次更新的已知问题列表中微软显示为空白，不过如果你在安装之后发现有任何问题都可以向微软提交反馈。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0802/1e2b54a31156c1d.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">本次更新现阶段仅向 Windows Insider 项目成员发布，不过伴随着测试的深入会向更多用户开放的。如果你想要获得该预览版，那么可以访问 Microsoft Store，然后选择搜索 Xbox Insider Hub 应用进行安装。</p>
