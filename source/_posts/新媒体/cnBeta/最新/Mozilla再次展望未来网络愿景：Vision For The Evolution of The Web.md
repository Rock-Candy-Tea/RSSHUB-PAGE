<strong>Mozilla 刚刚在官网上分享了有关 Web 发展演进的宏伟愿景，可知网络已是当今世界上最重要的通信系统，并且成为了许多人日常生活中一个不可或缺的重要组成部分。</strong>然而现实是，尽管 Web 已经在很多事情上做到了正确、并且取得了巨大的成功，但其距离“完美”仍有很长一段路要走。在 Mozilla 看来，其长远目标就是取其精华、去其糟粕，同时修补有害的部分。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0325/d1b159e98decf1a.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0325/d1b159e98decf1a.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：<a href="https://webvision.mozilla.org/" target="_self">Mozilla</a>）</p><p>Mozilla 强调，<strong>其愿景始于 Web 的三个基础价值观、并且深深根植于其宣言之中，</strong>分别是：</p><blockquote><p>● 开放性（Openness）意味着每个人都可以访问网络，并通过它来接触其他人。</p><p>● 代理（Agency）：一旦个人接触到网络，他们就有权按照自己的方式来有效实现目的。</p><p>● 安全（Safety）：使用网络的体验，决不可让个人置身于危险的境地。</p></blockquote><p>通过将这些价值观融汇到当今的 Web 中，<strong>Mozilla 认为还可以通过一系列努力，让他变得更加美好。</strong></p><p><strong>（1）保护用户隐私：</strong></p><blockquote><p>基本上所有上网冲浪的人们，其行为都有收到耿总和监视。而一个真正开放且安全的网络，理应在这些方面保持私密。且需要在不破坏网络本身的情况下，逐渐让生态系统达到新的平衡。</p></blockquote><p><strong>（2）保护用户免受恶意代码侵扰：</strong></p><blockquote><p>在浏览的时候，用户应免于担心他们的设备是否会受到侵害。鉴于每款 Web 浏览器都难免存在重大的安全漏洞，最终技术应显著减少此类安全问题。</p><p>Mozilla 正在 Firefox 浏览器中增加对相关技术的使用，并期待着其它开发商能够跟进。</p></blockquote><p><strong>（3）加密一切：</strong></p><blockquote><p>所有用户通信都应加密传输，尽管保护所有 HTTP 流量的漫长推动过程已接近尾声，但我们仍需努力改进现有的传统协议（比如域名解析服务 / DNS）、并将这一理念默认贯彻到所有新协议中。</p></blockquote><p><strong>（4）安全地扩展 Web：</strong></p><blockquote><p>新功能让 Web 变得越来越强大，同时也带来了新的风险，我们需要在新功能的附加价值和相关风险之间做好权衡。但若某些应用程序与 Web 八字不合，那也没必要去强求。</p></blockquote><p><strong>（5）让 Web 提速到可满足任何用途：</strong></p><blockquote><p>即使 Web 浏览器已较五年前有长足进步，但性能问题仍是个绕不过的槛。解决相关问题需要同步升级浏览器和基础设施，同时让人们更轻松、快速地构建有吸引力的站点。</p></blockquote><p><strong>（6）致力于让任何人都可在 Web 上轻松发布：</strong></p><blockquote><p>肃然早期网站相对简单且易于构建，但对性能和高产值的需求，还是不可避免地让 Web 变得越来越难以使用。</p><p>Mozilla 的策略是将技术开发分类为越来越复杂的层次，然后努力消除将用户推向更复杂路径的可用性差距。</p></blockquote><p><strong>（7）赋予用户以自己的方式体验 Web 的能力：</strong></p><blockquote><p>既然 Web 本质是服务于用户，那就该努力兑现这一承诺。为此，我们需要确保不被网站反向绑架用户的体验 —— 无论是广告、还是以其它可访问的形式来查看内容。</p><p>我们需要构建一个想用户之所想、急用户之所急的 Web 呈现方式，而不是按部就班地遵循网站的指示，以增强支撑这种新理念的 Web 标准的技术属性。</p></blockquote><p><strong>（8）为非英语使用者提供一流的体验：</strong></p><blockquote><p>当前 Web 的技术架构和内容升天系统，多是照着北美英语用户的使用习惯而打造的。</p><p>然而他们只占全世界的一小部分，Mozilla 希望 Web 能够为每个人都提供良好的服务 —— 不论他们生活在哪里、又讲述着怎样的语言。</p></blockquote><p><strong>（9）增强残障人士的可访问性：</strong></p><blockquote><p>随着 Web 体验变得越来越丰富，这部分已被遗忘的用户群体，也陷入了越来越难以与屏幕阅读器等辅助技术交互的尴尬 —— Mozilla 迫切希望尽快扭转这种趋势。</p></blockquote><p>综上所述，唯有将强大的经济与技术力量结合到一起，才能让 Web 顺利发展到今天这个样子。</p><blockquote><p>期间遇到的一些问题并不容易被解决（比如货币化与中心化），但前方的道路异常明确。</p><p>我们看到了作为一个社区来共同努力打造一个真正开放、且所有人都能够访问的 Web 的切实机会！</p></blockquote>
