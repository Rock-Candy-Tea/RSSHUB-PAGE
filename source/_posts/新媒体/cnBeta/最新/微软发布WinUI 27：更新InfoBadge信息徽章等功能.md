<strong>WinUI 是一款适用于 UWP（通用 Windows 应用）、以及使用 XAML Islands 的桌面应用程序的 Windows UI 库，现在，微软已经推出了最新的 WinUI 2.7 版本。</strong>作为一个开源项目，微软将它托管在了 GitHub 上的一个存储库中，以便广大开发者能够在哪里汇总提交 WinUI 2 系列的错误报告、功能请求、以及为社区贡献代码。<br>
 <p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2021/0916/95b7742c1e91ee8.png" alt="1.png" referrerpolicy="no-referrer"></p><p>首先，WinUI 2.7 更新引入了“信息徽章”（InfoBadge）。作为一套直观、且非侵入性的方案，它能够指示通知、显示警报、凸显新内容、或将焦点吸引到应用程序的某个区域。</p><p>每个 InfoBadge 的小片 UI，都可定制嵌入到一款应用程序中，以显示数字、图标、或一个简单的小点。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2021/0916/e26d576eb3f5267.png" alt="2.png" referrerpolicy="no-referrer"></p><p>InfoBadge 还内置于 NavigationView 控件中，可指定为 XAML 树中的独立元素，允许开发者将之放入所选的任何空间或 UI 部分。更多细节，还请移步至<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>官网，以查看<a href="https://docs.microsoft.com/en-gb/windows/apps/design/controls/info-badge" target="_self">使用指南</a>和 <a href="https://docs.microsoft.com/en-us/windows/winui/api/microsoft.ui.xaml.controls.infobadge" target="_self">API 参考</a>。</p><p><img src="https://static.cnbetacdn.com/article/2021/0916/635698d9ea1e510.png" alt="3.png" referrerpolicy="no-referrer"></p><p>其次是颜色选择器（ColorPicker）中的方向，开发者可指定该控件的 Orientation 属性，来编辑控件相对于取色区域是垂直或水平对齐。更多细节也请移步至微软官网，以查看 ColorPicker 的<a href="https://docs.microsoft.com/en-gb/windows/apps/design/controls/color-picker?#specify-the-layout-direction" target="_self">使用指南</a>和 <a href="https://docs.microsoft.com/en-us/uwp/api/microsoft.ui.xaml.controls.colorpicker.orientation" target="_self">API 参考</a>。</p><p>然后是一些示例：</p><blockquote><p>● 在 XAML Controls Gallery 示例应用中，包含了此类详尽的 WinUI 特性与控件。</p><p>● 若您已安装 XAML 控件库应用程序、并将之更新到了最新版本，可参考控件的更多实例。</p><p>● 若尚未安装 XAML 控件库应用程序，还请通过 <a href="https://aka.ms/xamlgalleryapp" target="_self">Microsoft Store</a> 应用商店下载获取。</p><p>● 此外开发者可移步至 <a href="https://github.com/Microsoft/Xaml-Controls-Gallery" target="_self">GitHub</a> 查看、复制、以及构建 XAML 控件库的源码 。</p></blockquote><p>最后，有关 WinUI 2.7 的 Bug 修复，可参阅 GitHub 上的<a href="https://github.com/microsoft/microsoft-ui-xaml/releases/tag/v2.7.0" target="_self">变更日志</a>。</p><p>所有稳定版本（以及预发布版本），均可通过 GitHub <a href="https://github.com/microsoft/microsoft-ui-xaml/tags" target="_self">发布页面</a>（或通过微软的 <a href="https://www.nuget.org/packages/Microsoft.UI.Xaml" target="_self">NuGet</a>）下载。</p><p>此外开发者可通过 NuGet 包管理器，将 WinUI 添加到 <a data-link="1" href="https://microsoft.pvxt.net/zaZYr" target="_blank">Visual Studio</a> 集成开发环境里的特定项目中。</p><p>更多详情，还请参阅 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> UI 2 Library 的<a href="https://docs.microsoft.com/en-gb/windows/apps/winui/winui2/getting-started" target="_self">入门介绍</a>。</p>
