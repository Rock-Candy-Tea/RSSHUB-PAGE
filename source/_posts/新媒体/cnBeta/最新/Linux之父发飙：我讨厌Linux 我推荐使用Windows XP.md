Linux 之父 Linus Torvalds 昨天似乎有点失控，在提交的一份 README 文件中吐槽了 Linux 系统。Torvalds 表示：“<strong>大家好，我是 Linus Torvalds，热门 Linux 的作者。这些 repo 网址和文件都可以证明我的身份。我删除了 linux，因为我讨厌它，现在我认为它很糟糕。你应该使用这个很棒的操作系统,它叫 Windows xp，我只是觉得它很棒”。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0127/fa3a0a4e94fa65f.webp" alt="fcafrknc.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Torvalds 要删除库，然后跑路吗？从目前来看，不是的，他只是向大家发了一个牢骚。从这个帖子来看，Linux 的库是没有问题的。事实上，这已经不是 Linus 第一次抱怨 Linux 系统了。他在早年就看到了 Linux 桌面版的混乱情况。主要的桌面Linux发行版的应用兼容性问题非常严重。他们随意修改核心，向后的兼容性也非常差。应用分发的体验还不如 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 系统好。</p>
