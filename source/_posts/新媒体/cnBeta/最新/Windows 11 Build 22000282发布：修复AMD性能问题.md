<strong>面向 Beta 和 Release Preview 频道的 Windows Insider 项目成员，今天微软推出了 Windows 11 Build 22000.282（KB5006674）。</strong>该预览版更新终于修复了 AMD 之前承认的性能问题。
上周，WCCFTech 报道了 AMD 锐龙 CPU 在 Windows 11 操作系统中可能面临性能跳水 15% 的问题，后续调查发现其与 L3 缓存延迟和 UEFI CPPC2 配置文件有关。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1016/be934c35982bfc3.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1016/be934c35982bfc3.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1016/45160892ca8fdf5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1016/45160892ca8fdf5.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">安装累积更新 KB5006674 之后，版本号升至 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 Build 22000.282。该累积更新解决了 L3 缓存的兼容性问题。该累积更新的修复内容将会整合到下月补丁星期二活动日发布的累积更新中</p><p style="text-align: left;">事实上 ，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 声称如果用户安装 Windows 11，L3 缓存延迟可能会增加近三倍。结果，应用程序的性能受到了 3-5% 的影响，而一些游戏的性能下降了 15%， 尤其是用于电子竞技的游戏。显然，这个错误严重到<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>立即实施了他们自己的修复。</p>
