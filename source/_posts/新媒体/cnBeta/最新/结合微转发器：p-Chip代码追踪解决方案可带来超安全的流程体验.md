作为一家利用尖端微转发器技术、而彻底改变实物产品和材料追踪体验的企业，p-Chip Corporation 刚刚介绍了最新的“安全追踪标签”解决方案。<strong>其特点是结合了 p-Chip 微转发器与标准的二维码、条形码、或其它矩阵码图案，以打造超安全的使用体验。</strong><br>
<p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0916/6cbf7eb2e619af1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图自：p-Chip Corporation <a href="https://p-chip.com/" target="_self">官网</a>）</p><p>p-Chip Corporation 首席执行 Joe Wagner 表示：COVID-19 大流行期间，二维码也在主流消费者群体中迅速推广开来。</p><blockquote><p>然而对包括食品、制药、汽车制造商等在内的许多企业来说，极易被复制和篡改的二维码，也有诸多潜在的安全隐患。</p>相比之下，p-Chip Code 技术提供了额外的安全属性，使得用户能够以可承受的价格、实现了可靠的可追溯性突破。</blockquote><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0916/5fa47ad53ba36d4.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>几乎看不见的微转发器被直接嵌入到了条码矩阵标签中，接着客户能够将之贴于食品、农药、药瓶、汽车零部件等各种待追踪的物品上。</p><p>功能方面，p-Chip Code 跟踪器类似于实物管理流程中的微型锚点，然后企业能够借助网络、云、或区块链应用程序来追踪数据。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=383680477&autoplay=false&disablePlaylist=true" width="720" height="480" frameborder="0"></iframe></p><p style="text-align: center;">Visibility - Tracking - Authentication（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzgzNjgwNDc3LnNodG1s.html" target="_self">via</a>）</p><p>公司可对成分、组件和材料开展数字化追踪，以增强产品的安全性、合规性、以及质量控制。</p><p>在某些情况下，消费者也可通过扫描二维码来确定其所购商品是否属于待召回的批，而企业也能够通过更安全的 p-Chip Code 微转发器来验证。</p>
