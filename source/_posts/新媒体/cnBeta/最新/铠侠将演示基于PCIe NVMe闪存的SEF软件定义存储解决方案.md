<strong>为支持 Linux 基金会的 Software-Enabled Flash 开源项目，铠侠美国公司今日宣布了基于 PCIe 和 NVMe 技术的创新软件定义技术和演示硬件。</strong>TechPowerUp 指出，这项技术能够将闪存存储与传统机械硬盘（HDD）协议完全分离，以便充分发挥闪存这一存储介质的性能潜力。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0802/45d7635dee6de73.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Kioxia <a href="https://business.kioxia.com/en-us/memory/sef.html" target="_self">官网</a>）</p><p>据悉，除了参加于圣克拉拉会议中心举办的“用于超大规模数据中心的 NVMe 软件支持闪存”会议，铠侠还决定在本周的闪存峰会暨博览会上、于 #307 号展位演示其“软件支持闪存”硬件。</p><p>与诞生至今已有数十年的机械硬盘（HDD）驱动器相比，当前的大规模 / 超大规模云存储应用中的闪存设备，其性能发挥已受到传统驱动器协议的极大拖累。</p><p>好消息是，由 Linux 基金会主导的软件支持闪存（SEF）社区项目，将使得业界能够采用软件定义的闪存 API，以根据数据中心、应用程序和工作负载的实际需求来定制闪存存储。</p><p>该项目的创建，旨在通过一套与供应商无关的灵活解决方案，来满足现代数据中心不断变化的需求、从而让存储开发人员社区受益。</p><p><img src="https://static.cnbetacdn.com/article/2022/0802/55378a7699027ee.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://business.kioxia.com/content/dam/kioxia/ncsa/en-us/business/memory/asset/KIOXIA_SEF_SDS-in-a-flash_Tech_Brief.pdf" target="_self">PDF</a>）</p><p>通过开源 API 和库，以及专门构建的、以媒体为中心的闪存硬件，社区得以专注于超大规模需求，为软件开发人员提供必要的工具、以及最大限度地利用闪存存储中尚未开发的功能。</p><p>此外随着 SEF 的持续发展，业界将能够释放闪存技术的完整潜力 —— 因其摆脱了传统的 HDD 协议，并构建了一套特定于超大规模应用场景中的闪存媒体平台。</p><p>铠侠存储战略部（M<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a>）高级副总裁 Eric Ries 表示：对 SEF 的支持，从根本上重新定义了主机和固态存储之间的关系。</p><p>此举为铠侠的超大规模客户提供了切实的价值，开拓新市场并增加对该公司闪存解决方案的需求。</p><p>得益于灵活的闪存部署与管理特性，客户能够从中获取最大价值，而铠侠也正在努力兑现其大幅提升超大规模数据中心闪存效率的承诺。</p>
