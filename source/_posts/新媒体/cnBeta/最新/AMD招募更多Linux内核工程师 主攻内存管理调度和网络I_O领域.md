<strong>由 AMD 官网挂出的招聘启事可知，该公司正在招募更多的 Linux 内核工程师，且招聘方向涵盖了调度程序、内存管理、网络 I/O、Linux 虚拟化性能等方面。</strong>显然，随着 AMD 不断抢占更大的数据中心市场份额，该公司已经对 Linux 平台提起了更高的关注度，并在今年早些时候招募过一批以客户为中心的 Linux 工程师。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0608/413e876d5a84454.png" referrerpolicy="no-referrer"></p><p>不过最新的招聘信息表明，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 将把更多重心放到服务器 / EPYC 领域的 Linux 内核优化工作，同时兼顾一些有利于 AMD 桌面 / 移动平台的工作。</p><p>如果在 AMD Careers 网站搜索“Linux”关键词，可知该公司仍有许多职务空缺，且工作地点多位于德州奥斯汀和印度班加罗尔。</p><p><img src="https://static.cnbetacdn.com/article/2021/0608/329e7cb41de6244.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">Linux Kernel - <a href="https://jobs.amd.com/job/Bangalore-Linux-Kernel-Scheduler-Development-Lead%2896070%29-Karn/744867000/" target="_self">Scheduler Development Lead</a></p><p><a href="https://www.phoronix.com/scan.php?page=news_item&px=AMD-Kernel-Engineers-June-2021" target="_self">Phoronix</a> 指出，AMD 显然在印度建立了一个“专注于调度程序”的小型团队，主要工作就是优化霄龙（EPYC）服务器处理器的性能和功能。</p><p>以调度开发首席工程师这个职务为例，我们很高兴 AMD 致力于改进内核调度程序的性能，并且希望 Schedutil 和相关工作能够同样推动 CPUFreq / CPPC 的进步。</p><p><img src="https://static.cnbetacdn.com/article/2021/0608/8237f9797928e5d.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">Linux Kernel - <a href="https://jobs.amd.com/job/Bangalore-Linux-Kernel-Memory-Management-Development-Lead-Karn/745391200/" target="_self">Memory Management Development</a> / Performance Lead</p><p>内存管理开发 / 性能主管，我们再次很高兴地看到 AMD 在从事更多与底层（Linux 基础设施）和性能优化相关的工作。</p><p><img src="https://static.cnbetacdn.com/article/2021/0608/696ad76e49bd8b9.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">Linux Kernel - <a href="https://jobs.amd.com/job/Austin-Linux-Kernel-Networking-IO-Lead-96067-Texa/745478800/" target="_self">Networking I/O Development</a> / Performance Lead</p><p>然后是 Linux 内核的网络 I/O 与性能开发的主管职务，负责人将专注于 I/O 性能、低延迟、以及可扩展特性。</p><p><img src="https://static.cnbetacdn.com/article/2021/0608/4d0cc75f67aac34.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">Linux Kernel <a href="https://jobs.amd.com/job/Austin-Linux-Virtualization-Performance-Lead-Texa/746604800/" target="_self">Virtualization（KVM / QEMU）Performance</a> Lead</p><p>最后是 Linux 虚拟化性能主管，作为围绕 EPYC 平台的几个 Linux 虚拟化职务之一，应聘者将主要负责 KVM / QEMU 领域。</p>
