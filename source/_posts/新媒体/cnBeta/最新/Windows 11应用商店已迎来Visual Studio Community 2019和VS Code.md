<strong>微软正在为 Windows 11 操作系统中的 Microsoft Store 带来更多大型应用程序，比如近日添加的 Visual Studio Code 和 Visual Studio Community 2019 。</strong>感兴趣的测试者们，现可通过 Microsoft Store 轻松下载这两款开发工具。此外在 Windows 11 操作系统中，微软将允许所有开发者向商店提交未打包的 Win32 应用程序，以进一步简化应用程序的查找和安装体验。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0826/9aea6d19ff239d0.png" referrerpolicy="no-referrer"></p><p>除了 <a data-link="1" href="https://microsoft.pvxt.net/zaZYr" target="_blank">Visual Studio</a>，即将登陆 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 官方应用商店的大型应用程序还包括 Microsoft Teams、Disney+、Adobe Creative Cloud、Zoom、以及 Canva 。</p><p><a href="https://static.cnbetacdn.com/article/2021/0826/c0d97ccabdc6346.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0826/c0d97ccabdc6346.jpg" referrerpolicy="no-referrer"></a></p><p>作为一个距今已有将近 10 年历史的产物，Microsoft Store 可追溯到<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>鼓励开发者为触屏 PC 打造 Metro 风格应用程序的那个年代。</p><p>遗憾的是，在 Windows 8 上首次亮相的 Microsoft Store，其推广和发展一直不那么顺利。即使 Windows 10 时代的 Metro 和 UWP 应用，也没能力挽狂澜。</p><p><img src="https://static.cnbetacdn.com/article/2021/0826/6674f679bf3b204.jpg" referrerpolicy="no-referrer"></p><p>好消息是，微软终于清醒地意识到，遗留的大量 Win32 应用程序，才是 Windows 用户真正希望在 Microsoft Store 上看到的。</p><p>只是我们需要多等待一段时间，才能看到更多知名应用程序进入 Windows 11 应用商店。与此同时，微软也在努力和亚马逊合作，以将 Android 应用程序引入新一代桌面操作系统。</p>
