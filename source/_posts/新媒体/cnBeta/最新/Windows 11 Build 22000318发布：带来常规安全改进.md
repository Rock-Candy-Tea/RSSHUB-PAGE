在本月的补丁星期二活动日中，<strong>微软发布了适用于 Windows 11 系统的累积更新 KB5007215，在安装之后版本号升至 Build 22000.318。</strong>本次更新带来了一些安全修复和常规改进。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1110/3ebf3681c42e55d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1110/3ebf3681c42e55d.jpg" alt="qdnpqa96.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Windows 11 Build 22000.318 主要更新内容：</p><blockquote style="text-align: left;"><p style="text-align: left;">● 为你的 Windows 系统提高安全性</p><p style="text-align: left;">● 修复了某些应用程序在渲染一些用户界面元素或在应用程序内绘图时可能会出现意外结果。在使用 GDI+ 并在每英寸点数（DPI）或分辨率较高的显示器上设置零（0）宽度钢笔对象的应用程序中，或者在应用程序使用缩放时，你可能会遇到这个问题。</p></blockquote><p style="text-align: left;">你可以通过 Windows Update 、Microsoft Update、Windows Update for Business、Microsoft Update Catalog 或 Windows Server Update Services（WSUS）安装最新的 Windows 11 累积更新。</p>
