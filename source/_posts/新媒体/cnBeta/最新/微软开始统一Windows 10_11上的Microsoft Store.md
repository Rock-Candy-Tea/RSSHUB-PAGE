<strong>面向 Release Preview 频道的 Windows Insider 项目成员，微软开始推出 Windows 11 的 Microsoft Store。</strong>新的 Microsoft Store 不仅有有漂亮的视觉效果，它还支持 Win32 应用程序，如 Discord、Epic 游戏商店、Mozilla Firefox，以及如 Disney+ 等新的娱乐应用程序。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1027/33cf5a90f1a6583.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1027/33cf5a90f1a6583.jpg" alt="bebvbtfw.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">微软计划很快向所有 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 10 用户推出这个新的 Microsoft Store。微软首席架构师 Rudy Huyn 在 Twitter 上解释说：“我们先向 Windows 10 Insiders 发货，以确保该应用没有错误，稍后将向所有 Windows 10 用户发布”。</p><p style="text-align: left;">很高兴看到微软商店将在 Windows 10 和 Windows 11 中提供一致的体验，不过与亚马逊 Android 应用商店的整合仍然是 Windows 11 的专属。这是因为新的 Windows Subsystem for Android，现在邀请 Windows 11 Insiders 在 Beta 频道上进行测试，不会进入 Windows 10。</p>
