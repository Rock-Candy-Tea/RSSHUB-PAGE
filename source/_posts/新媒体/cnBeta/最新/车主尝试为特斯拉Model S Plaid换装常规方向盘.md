Ryan Huber 刚刚在油管上分享了一段视频，<strong>展示了将特斯拉 Model S Plaid 原装轭式（Yoke）方向盘、替换成更普通的 Model 3 方向盘的新奇体验。</strong>当特斯拉首次发布塞车式的“蝶形”方向盘时，许多人都对它的形状和功能提出了顾虑，比如无法通过拨杆来选择驾驶模式。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1104/acf2998566a464b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1104/acf2998566a464b.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">资料图（来自：Tesla 官网）</p><p>对于汽车制造商来说，显然不会在缺乏非线性转向系统加持的情况下，将轭式方向盘贸然推向市场（此前有传闻称该公司一直在开发线传转向系统）。</p><p><img src="https://static.cnbetacdn.com/article/2021/1104/3d9506aa01902ad.png" referrerpolicy="no-referrer"></p><p>然而在上月开始交付新款 Model S Plaid 时，我们惊讶地发现其具有 14.0：1 的正常转向比。因而在较低的车速下，其使用体验也变得相当不切实际（甚至有安全隐患）。</p><p><a href="https://static.cnbetacdn.com/article/2021/1104/f8de77cca37fa9b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1104/f8de77cca37fa9b.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">视频截图（来自：Ryan Huber）</p><p>特斯拉 CEO 伊隆·马斯克后来证实，公司确实有在致力于“渐进式”转向方案的开发，但距离实际推向市场还有几年的时间。</p><p>与此同时，马斯克证实特斯拉并不打算提供普通方向盘的选项。这意味着，车主们只能寄希望于后改装市场了。</p><p><img src="https://static.cnbetacdn.com/article/2021/1104/1c9d9ac1a687264.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>作为 Model S Plaid 的早期车主这一，Ryan Huber 表示随着时间的推移，自己已在一定程度上习惯了轭式方向盘的操控体验。</p><p>但在换装 Model 3 普通方向盘这件事上，他还是努力让所有功能都可正常工作（包括拨片），并且不打算再换回轭式方向盘。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=300548524&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">What has two thumbwheels and is fully functional（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzAwNTQ4NTI0LnNodG1s.html" target="_self">via</a>）</p><p>最后，Ryan Huber 甚至成功地让 Model 3 方向盘上的滚轮，也能够与 Model S 实现软件商的完美适配。</p><p>不过他并不打算自行出售改装套件，而是选择了在 GitHub 上分享相关创作，以帮助尽可能多的同样想要这样做的车主们。</p><p>【警告】</p><p>换装特斯拉方向盘并不是一件容易的事情，比如 Huber 必须编写自定义固件，才能使 Model 3 方向盘在 Model S Plaid 平台上工作。</p><p>动手能力和技术储备都欠佳的朋友，还请耐心等待更“正规”的后市场改装套件的到来。</p>
