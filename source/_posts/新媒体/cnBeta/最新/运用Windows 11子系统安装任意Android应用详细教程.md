Windows 11 最强功能终于来了，Android软件真的可以随便下！当时有不少差友在评论区里留言说为啥不发教程，这不，Windows 11
安装任意Android应用详细教程来了！在教程正式开始前，世超想先友情提醒下：由于这次微软只是在美区 Beta
版的系统中上线了这个功能，所以原则上国内正式版、Beta 版和 Dev 版的 Windows 11 用户要想用上这功能还得再等等。<br>
 <p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/6a766d13e3dd8cb.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/6a766d13e3dd8cb.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/6a766d13e3dd8cb.png" referrerpolicy="no-referrer"></a><br></p><p>尽管以下<a data-link="1" href="https://c.duomai.com/track.php?k=iRyUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmUmchdHdm92cGJTJjZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyU" target="_blank">教程</a>可以无视你的 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 系统是什么版本，但是考虑到这功能目前还处于测试期间，所以我仅推荐那些不怕遇到  Bug  的差友尝试。</p><p>OK，友情提醒完毕，正文正式开始。</p><p>差友们把系统升级到 Windows 11 后，首先需要在自己电脑安装一个名为 “ Windows Subsystem for Android? with Amazon Appstore ” 的东西。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/721a1f9fea6095f.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/721a1f9fea6095f.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/721a1f9fea6095f.png" referrerpolicy="no-referrer"></a><br></p><p>它包含两个软件，分别是 Windows Subsystem for Android 和 Amazon Appstore。</p><p>其中 Windows Subsystem for Android（ 下面简称 WSA ）相当于依附在 Windows 上的一个Android子系统，Android程序借助这个子系统才能正常运行；</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/1b8c7a393c2837c.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/1b8c7a393c2837c.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/1b8c7a393c2837c.png" referrerpolicy="no-referrer"></a><br></p><p>而 Amazon Appstore 就跟<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a> App Store 还有Google Play Store 一样，是我们下载Android软件的地方。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/918dcfa5ffe4993.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/918dcfa5ffe4993.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/918dcfa5ffe4993.png" referrerpolicy="no-referrer"></a><br></p><p>如果你的系统正好是美区 Beta 版，那么你直接在<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>自带的应用商店就能下到 Windows Subsystem for Android? with Amazon Appstore。</p><p>如果不是，就只能通过抓包方式来下载了。</p><p>在 https://store.rg-adguard.net  这个网站，输入 https : //www.microsoft.com/store/productId/9P3395VX91NR，然后右侧选中 Slow，点击勾号。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/912ee72d9e57244.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/912ee72d9e57244.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/912ee72d9e57244.png" referrerpolicy="no-referrer"></a><br></p><p>接着直接划到搜索结果最下方，大小为 1GB 多的那个文件，就是我们要下载的文件。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/c9b60875273f1c4.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/c9b60875273f1c4.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/c9b60875273f1c4.png" referrerpolicy="no-referrer"></a><br></p><p>前面的文件下载好后，接下来就是把它安装到电脑上。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/c8589560c6e9750.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/c8589560c6e9750.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/c8589560c6e9750.png" referrerpolicy="no-referrer"></a><br></p><p>在桌面按下快捷键 Win+X，选中 “ Windows 终端（ 管理员 ） ” 选项，调出 PowerShell 窗口。</p><p>接着输入命令 add-appxpackage"XXX"（ 引号里的 XXX 为 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://msi-pc.jd.com/" target="_blank">MSI</a>xbundl 文件所在路径 ），然后按下回车键，系统就会帮你自动把文件安装到电脑上。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/ef5acefc29cf4f7.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1026/ef5acefc29cf4f7.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1026/ef5acefc29cf4f7.jpg" referrerpolicy="no-referrer"></a><br></p><p>此时你会发现你的电脑已经有了 WSA 和 Amazon Appstore 这两个软件：</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/be7f00403c8f91e.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/be7f00403c8f91e.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/be7f00403c8f91e.png" referrerpolicy="no-referrer"></a><br></p><p>由于Android子系统严格来说是通过 Windows 自带的虚拟机来实现的，所以在运行 WSA 前，我们还需要开启 Windows 上的 Hyper-V 虚拟机功能。</p><p>差友们只需进入到 Windows 控制面板，在 “ 程序和功能 ” 选项下的 “ 启用或关闭 Windows 功能 ” 里勾选 Hyper-V 然后重启电脑，就能开启虚拟机。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/3329549595892f4.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/3329549595892f4.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/3329549595892f4.png" referrerpolicy="no-referrer"></a><br></p><p>做完前面这一步，其实各位已经可以在自己电脑上安装和运行Android软件了，你只需运行 WSA，然后打开 Amazon Appstore，选择一个软件安装即可。</p><p>只不过由于微软这次只跟亚马逊策划了 50 个用来测试的 APP，我们能下的应用可以说是少得可怜：</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/3e899b775dc23b0.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/3e899b775dc23b0.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/3e899b775dc23b0.png" referrerpolicy="no-referrer"></a><br></p><p>要是你想要安装任意应用，最好的办法当然是安装一个第三方应用商店比如酷安到Android子系统里。</p><p>接下来我就跟各位讲讲怎么把酷安安装到Android子系统。</p><p>它的原理说白了非常简单，其实就是通过 adb 无线连接的方式把Android子系统连接电脑，然后再通过调试把酷安 apk 安装包安装到子系统里。</p><p>差友们先去酷安官网下载一个 apk 安装包到电脑：</p><p>https : //www.coolapk.com/</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/3f7c1ea3bb1f110.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/3f7c1ea3bb1f110.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/3f7c1ea3bb1f110.png" referrerpolicy="no-referrer"></a><br></p><p>再去Google开发者网站下载一个 adb 工具包并解压：</p><p>https : //developer.android.google.cn/studio/releases/platform-tools?hl=zh-cn</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/61dcbf54094f550.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/61dcbf54094f550.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/61dcbf54094f550.png" referrerpolicy="no-referrer"></a><br></p><p>然后在 WSA 开启 “ 开发人员模式 ” 并记录下它给你的 IP 地址（ 不同电脑 IP 地址可能有所不同，比如我的是 127.0.0.1 : 58526  ）：</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/519860c37aa6584.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/519860c37aa6584.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/519860c37aa6584.png" referrerpolicy="no-referrer"></a><br></p><p>接着再打开前面解压好的 adb 工具包文件夹，在文件夹上方地址框里输入 cmd 回车调出 cmd 界面：</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/afbcd4c228abc89.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/afbcd4c228abc89.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/afbcd4c228abc89.png" referrerpolicy="no-referrer"></a><br></p><p>在界面里输入 adb connect 127.0.0.1 : 58526 命令行回车，等界面出现 “ connect 12 7.0.0.1 : 58526  ” 后，再继续输入 adb install “ XXX  ” 命令行回车（ 引号里的 XXX 为酷安 apk 安装包所在路径 ）。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/30e591fb5a6a6a6.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/30e591fb5a6a6a6.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/30e591fb5a6a6a6.png" referrerpolicy="no-referrer"></a><br></p><p>此时如果界面出现 Success 字样，那就代表酷安已经成功安装在Android子系统里了，酷安里的所有软件你都能随便下！</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/bbcf2e7d7f7b1fb.png"><img data-original="https://static.cnbetacdn.com/article/2021/1026/bbcf2e7d7f7b1fb.png" src="https://static.cnbetacdn.com/thumb/article/2021/1026/bbcf2e7d7f7b1fb.png" referrerpolicy="no-referrer"></a><br></p><p>现在我的电脑已经安装了不下几十个应用，绝大多数软件都能正常运行，遇到的 BUG 无非就是有些软件偶尔会闪退或者无法登录，总体来说问题不大。</p><p>你甚至还可以在电脑上打几盘王者<a data-link="1" href="https://c.duomai.com/track.php?k=WP0ZSPklWdlZyN4MTPklWYmYDO5IDNy0DZp9VZ0l2cmYiRyUSbvNmLsxWYtZnL3d3dGJTJGJTJBNTJzBHd0h" target="_blank">荣耀</a>：</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1026/1c9c705f3b61fed.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1026/1c9c705f3b61fed.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1026/1c9c705f3b61fed.jpg" referrerpolicy="no-referrer"></a><br></p><p>差友们别看前面有那么多步骤看着有点繁琐，如果你按照我给的方法在自己电脑上一步步操作下来其实并不难，要是遇到了问题，那就评论区见！</p>
