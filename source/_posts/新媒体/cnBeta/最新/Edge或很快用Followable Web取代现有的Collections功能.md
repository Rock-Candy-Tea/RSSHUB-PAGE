MSPU 援引 Reddit 网友 Leo Varela 的爆料称：<strong>微软似乎正在为 Edge 96 Canary 开发一项新功能，据说它类似于 Pinboard，可用于整理归纳 Web 内容，未来或有望取代目前的 Collections 收藏。</strong>感兴趣的朋友，可在 Edge Canary 中找到“#edge-folloable-view”标记，并看到如下描述：<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0925/ef21e28bcee30f1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">Edge 91（图 via <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Latest）</p><p>● 尝试 followable web 测试功能：</p><blockquote><p>-- 您现有的收藏内容不会被自动迁移，但 followable web 功能将很快到来，且你将能够随时切换回来。</p><p>-- 其次，这项功能适用于 Mac、Windows、以及 Linux 平台上的 Edge Canary 。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0925/2ce9cf017eaad3d.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p><a href="https://mspoweruser.com/microsoft-edge-may-replace-collections-with-followable-web-feature/" target="_self">MSPU</a> 指出，虽然官方描述有些模棱两可，但它很可能与 Chrome 浏览器正在开发的新“关注”（Follow）有关。</p><p>猜测 Follow 功能将允许 Web 浏览器用户轻松关注特定的网站，并可在 Chrome‘发现’版块的‘关注’选项卡中获取最近更新的内容。</p><p>至于这种新颖的 RSS 阅读器体验是否会比 Pinboard 更受用户欢迎，目前暂不得而知。不过可以肯定的是，它至少会比当前的 MSN 消息流要实用得多。</p>
