<strong>在 Chrome 90 稳定版登陆 Windows、Mac 和 Linux 三大平台大约一个半月后，谷歌又于本周二开启了 Chrome 91 稳定版的推送。</strong>预计接下来几天到几周内，广大 Chrome 浏览器用户将陆续升级到新版本。虽然谷歌尚未分享完整的功能与 bug 修复列表，但 9to5Google 已经有了相当全面的汇总介绍。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0527/29d778d7893c064.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a href="https://chromereleases.googleblog.com/2021/05/stable-channel-update-for-desktop_25.html" target="_self">Google Blog</a>）</p><p>首先，Chrome 91 能够冻结已折叠的“标签页分组”中的网页，以释放设备上的系统资源。</p><p>不过那些播放音频、带有 IndexedDB 锁、音视频 / 窗口 / 显示采集、连到了一个 USB 设备，以及正在被镜像的标签页都属于例外。</p><p>尽管大部分用户觉得没必要如此严格，但对于那些低端设备用户来说，有关谷歌浏览器占用太多系统资源的抱怨也一直没有消停。</p><p><img src="https://static.cnbetacdn.com/article/2021/0527/e2dc7eb31d0dd8d.png" alt="2.png" referrerpolicy="no-referrer"></p><p>其次，Chrome 91 现允许用户在登录操作系统时自动启动渐进式 Web 应用（简称 PWA）。</p><p>有需要的用户，可直接在浏览器地址栏输入 chrome://apps，右键点击要更改的 PWA 应用，然后选择“登录时启动”。</p><p>如果未能顺利启用此标记，请先移步至 chrome://flags/#enable-desktop-pwas-run-on-os-login 。</p><p><img src="https://static.cnbetacdn.com/article/2021/0527/5e461286e7625b1.png" alt="3.png" referrerpolicy="no-referrer"></p><p>其它功能包括：Android 平板电脑上的 Chrome 浏览器将默认请求桌面版（而不是移动版）网站、网页上的表格能够更好更紧密地融入、针对 iOS 用户的假冒网站警报（避免输入已保存的密码）、以及启用了 DNS over HTTPS（简称 DoH）。</p><p>感兴趣的朋友，可手动检查 Chrome 浏览器是否已经升级到了最新的 91 版本，方法是点击“设置 -> 关于 Chrome”，并在下载完成后重启浏览器以完成更新。</p>
