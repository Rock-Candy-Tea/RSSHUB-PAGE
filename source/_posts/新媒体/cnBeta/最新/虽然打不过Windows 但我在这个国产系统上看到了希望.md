差友们有听说过<strong>“国产操作系统”</strong>么？可能大多数人听到这几个字的反应，脑海里第一想到的，是华为的 <strong>HarmonyOS</strong>。虽然现在鸿蒙的知名度比较大，但它并不是国内最早自研的操作系统，今天想和大家聊聊<strong>国产的Linux系统</strong>。<strong>对，就是</strong><strong>和Windows、MacOS一个性质</strong><strong>的电脑操作系统。</strong><br>
 <p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/6f82dedeaa3093d.webp" referrerpolicy="no-referrer"></p><p>那有同学就问了，国产电脑品牌是听过不少，好像没怎么听说过还有操作系统啊？我们买来电脑装机，除了<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>还有别的么？</p><p>其实啊，国产的电脑操作系统不仅存在，还发展出了不少的<strong>“品牌”</strong>。</p><p>比如有一家叫红旗 OS 的国产系统，在前不久建党 100 周年，更新了一个建党百年的献礼版系统，并免费提供下载。</p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/19842ffc7d1e534.webp" referrerpolicy="no-referrer"></p><p>国产系统见过不少，但第一次看见这种令人心潮澎湃的专属定制，还是挺想装在电脑上试试的。</p><p>同时也好奇，<strong>现在国产的电脑操作系统，到底发展到什么程度了？</strong></p><p>由于手边没有空闲的电脑，就以虚拟机的方式来安装了这款红旗 OS 献礼版本，使用的软件是 VMware Workstation 这样的老牌虚拟机。</p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/4cf741c39759c2f.webp" referrerpolicy="no-referrer"></p><p>红旗 OS 对于电脑硬件的要求没有很高，最低只需要 1GB 的内存和 10GB 大小的储存。</p><p><img src="https://static.cnbetacdn.com/article/2021/0710/19842ffc7d1e534.webp" referrerpolicy="no-referrer"></p><p>因为是献礼版系统，所以一导入系统的镜像文件，就可以在页面中心看到一个建党的 Logo，仪式感十足。</p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/4426eb61484f171.webp" referrerpolicy="no-referrer"></p><p>在安装完系统之后，就正式进入到红旗 OS 的系统之中了，系统内置的图标和桌面壁纸，看得出来设计团队花了不少心思。</p><p>截了几张系统的界面图，整体设计风格以建党 100 周年的爱党爱国情怀为主，红色经典永不褪色。</p><p><strong>向左滑动 ▼</strong></p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/dcf313147caa8f6.webp" referrerpolicy="no-referrer"></p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/07997d7341f4570.webp" referrerpolicy="no-referrer"></p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/e6ecc9d99ebac0d.webp" referrerpolicy="no-referrer"></p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/95c3a77902d6d96.webp" referrerpolicy="no-referrer"></p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/f813f2047aa2640.webp" referrerpolicy="no-referrer"></p><p>红旗 OS 在使用逻辑上和 Windows 差不多，在屏幕的左下角有一个开始菜单，本身集成了不少常用的系统工具。</p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/2671f8ff6044aad.webp" referrerpolicy="no-referrer"></p><p>不过这些工具并没有都标明中文，第一次找计算器就找了半天，根本想不到它在工具中的名称竟然叫 KCalc ……</p><p>系统浏览器用的是 Chromium 核心的红旗浏览器，谷歌浏览器能干的事儿，它都能干，实用性上没得说，也符合现在主流的浏览器使用习惯。</p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/45815bcc4b16279.webp" referrerpolicy="no-referrer"></p><p>不管是<strong>刷B站、刷微博</strong>、甚至是想在这上面给大家码字写文章，都和 Windows 系统没啥区别。</p><p>然而一个操作系统可不止上上网那么简单，对 Linux 系统真正的考验，还是在应用程序的生态上。</p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/8d99427d382ce3b.webp" referrerpolicy="no-referrer"></p><p><strong>到底能用多少的第三方程序，直接决定了这台电脑能扛多少事儿。</strong></p><p>要是你对 Linux 系统并不是很熟，又或是第一次接触这个系统，那觉得还是需要仔细斟酌一下。</p><p><strong>因为红旗 OS 和大多数 Linux 系统一样，并没有完全支棱起来。</strong></p><p>首先在通讯软件方面，微信首当其冲，官方就压根没推出过 Linux 的客户端版本，想用就只能通过 Wine 来转译或者借助于网页版。</p><p>官方 Linux 版？不存在的。▼</p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/cbe07e4a3bcc999.webp" referrerpolicy="no-referrer"></p><p>QQ 倒是可以用，腾讯还专门给 Linux 提供了一个<strong> “ 纯粹 ” </strong>版本。</p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/b84021e758f63be.webp" referrerpolicy="no-referrer"></p><p>这个版本褪去了华丽的外表，回归到了聊天软件的本质，感觉再一次回到了 2006 年的 QQ 界面。</p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/3ce188b8f6570e7.webp" referrerpolicy="no-referrer"></p><p>毕竟它除了<strong>发图片、截屏、传文件</strong>等基本功能之外，剩下的啥功能都没有，甚至连在桌面隐藏 QQ 页面这个功能都没做。</p><p class="textAlignCenter"><img src="https://static.cnbetacdn.com/article/2021/0710/7631b7c3b2f17cc.webp" referrerpolicy="no-referrer"></p><p>而且不知道为什么，这个版本只能通过<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a> QQ 扫二维码的方式来登录，都不给你输 QQ 号和密码的机会。</p><p class="textAlignCenter"><img src="https://x0.ifengimg.com/ucms/qr/2021_28/5D678C1C00EF3441B344025DEBC6696A0F4BBED5_size10_w251_h495.webp" alt="图片" referrerpolicy="no-referrer"></p><p><strong>而相比腾讯，雷总的老东家金山就良心了很多。</strong></p><p>WPS 办公三件套的 Linux 版不但去掉了广告，甚至还整合了云文档的功能，功能上一点不比 Windows 版的少。</p><p class="textAlignCenter"><img src="https://x0.ifengimg.com/ucms/2021_28/3B15AFFFBC2EC8B673BC5C3C5D526A0BCF848DBF_size26_w1080_h589.webp" alt="图片" referrerpolicy="no-referrer"></p><p class="textAlignCenter">最难能可贵的，是金山的团队还在不断的更新 WPS 的 Linux 版本，持续为爱发电。</p><p><img src="https://x0.ifengimg.com/ucms/2021_28/988E24044A299E4F2F249E375ECC25680406E6E9_size11_w549_h384.webp" alt="图片" referrerpolicy="no-referrer"></p><p>要说 Linux 软件开发不给力，还能把锅甩到软件公司头上，那下面有关游戏软件的适配，就是红旗 OS 自身的不足了。</p><p>经典多人在线游戏平台 Steam ，从很早开始就提供了 Linux 版本的下载。</p><p>而且在它开发了一套叫 Proton 的转译工具之后，还能直接无差别运行 Windows 系统上的游戏。</p><p class="textAlignCenter"><img src="https://x0.ifengimg.com/ucms/2021_28/6FDD9FB3BF5FF6E8AB00E30E5A2EBFF137B16FD8_size23_w736_h582.webp" alt="图片" referrerpolicy="no-referrer"></p><p><strong>苹果用户都要羡慕疯了。</strong></p><p>但在折腾了一个晚上之后，发现红旗 OS 的环境始终都安装不上 Steam，不知道在哪个环节出了差错，最后只好无奈放弃。</p><p><strong>要是评论区有懂哥的话，</strong></p><p><strong>应该怎么做？▼</strong></p><p class="textAlignCenter"><img src="https://x0.ifengimg.com/ucms/2021_28/EC3B7F9CF39DF9EE61EDA0B08E53F965B387250C_size31_w638_h327.webp" alt="图片" referrerpolicy="no-referrer"></p><p>唉，在其他 Linux 系统上的优势项目，到这里成了红旗OS上的缺点，多少有些遗憾。丰富软件生态，一直都是 Linux 开发者需要积极解决的问题，系统要是没有各种各样的软件支持，就会像一个光杆司令，有劲也无处使。</p><p>安装时提到的，大部分都没有……▼</p><p class="textAlignCenter"><img src="https://x0.ifengimg.com/ucms/2021_28/A3295E4721D1638A01D241B066DEE981CE06FF06_size30_w800_h600.webp" alt="图片" referrerpolicy="no-referrer"></p><p><strong> 红旗 OS 的商店就是这么一个让人摸不着头脑的司令。</strong></p><p>而说起软件生态做的好的，就不得不提另一家国产系统 —— <strong>Deepin 深度</strong>。</p><p class="textAlignCenter"><img src="https://x0.ifengimg.com/ucms/2021_28/4A84716906E8AE5D3094A4CAF748943EAB26F918_size31_w830_h465.webp" alt="图片" referrerpolicy="no-referrer"></p><p>他们不但在系统里有自己的Deepin商店和原创应用，还能通过安卓容器来运行安卓应用，以此补缺Linux上没有的应用。</p><p><strong>甚至还帮你把Wine版的微信打包好了，</strong></p><p><strong>直接下直接用。▼</strong></p><p><img src="https://x0.ifengimg.com/ucms/2021_28/C23F43DB23975168D433884DCB57F492B98B0CA6_size43_w1080_h675.webp" alt="图片" referrerpolicy="no-referrer"></p><p>这样一来，用户不再用打开网页满世界找应用，商店全都给你安排的明明白白，这不香么？</p><p>但就算已经有这样好的迹象，国产操作系统面临的最大问题，还是社区开发者太少。</p><p>举例来说，之前想找适合红旗 OS 用的微信客户端，在红旗的论坛上搜索了一下。</p><p>结果就搜出三条，也都是别人的提问，没答案的那种。</p><p class="textAlignCenter"><img src="https://x0.ifengimg.com/ucms/2021_28/6B018E3AC521259039E7932287222B03504FC35B_size19_w1080_h785.webp" alt="图片" referrerpolicy="no-referrer"></p><p>久而久之，这就会形成一个死局……</p><p><strong><strong>鲜有人去维护系统生态 </strong>➡ 软件起不来导致系统难用 <strong><strong>➡ </strong>有体验更好的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>苹果操作系统为什么不用？<strong>➡ </strong><strong>鲜有人去维护系统生态……</strong></strong></strong></p><p>这个难关，恐怕是国产系统发展的最大瓶颈，也是必经之路。</p><p>至于要如何破局，还得要打破上述其中的一环才行。<strong>国产 Linux 虽然还在不成熟，但却是相当重要的一个环节。</strong></p><p>现在除了红旗、Deepin 之外、国产还有<strong>优麒麟、统信 UOS、StartOS </strong>等等不同国产厂牌。</p><p class="textAlignCenter"><img src="https://x0.ifengimg.com/ucms/2021_28/C220AECD4EB592EBC8B82C36999AD55DEFF518EB_size33_w1080_h504.webp" alt="图片" referrerpolicy="no-referrer"></p><p>尽管他们的用户的体验肯定没有 Windows 那样的商业软件来的成熟。</p><p><strong> 但他们的好处在于，可以自由自在的用，不会受制于人。</strong></p><p>试想一下，假如哪天微软也脑子搭错筋，谷歌附身，突然不再支持中国的电脑设备用 Windows，或者限制某些核心功能。</p><p>到那时候，这些国产操作系统能不能及时派上用场，就显得尤为重要了。</p><p>不过遗憾的是，红旗系统从1999年发行了第一版以来，实在没往商业化和产品营销的方向去走。</p><p>以至于现在这款系统，除了在某些政府机构使用之外，并无太多的装机应用环境。</p><p>相对封闭的使用场景，对于一个操作系统的发展也好、安全性也好，不是什么好事。</p><p>所以不管怎么说，国产Linux系统还是任重而道远，需要提升的不止是开发上的技术。</p><p>比起国产系统这个标签，红旗OS更重要的是让更多的人去用、去提出意见、去做出修改，这样才有进步的方向和空间。</p>
