面向 Dev 频道的 Windows Insider 项目成员，今天微软推出了全新的 Windows 11 Build 25169。由于代号为“Sun Valley 3”的 Windows 11 23H2 功能更新已经取消，微软转向每三年一次重大 Windows 版本更新，因此该预览版更新依然隶属于 Windows 11 22H2。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0729/1e824ea1b0c304f.webp" alt="oq36io6x.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">虽然这个预览版仍属于 22H2，不过新版将于 2023 年发布，因此在上线之后 Version 号可能会叫做 23H2。<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 Build 25169 主要内容如下</p><p style="text-align: left;"><strong>Multi-App Kiosk Mode</strong></p><blockquote style="text-align: left;"><p style="text-align: left;">Multi-App Kiosk Mode 是 Windows 11 的活动限制功能，允许 IT 管理员在设备上仅运行选定的应用目录，并阻止其他功能的使用。这意味着用户可以在一台设备上根据不同的用户类型，创建各种不同的应用和访问配置。</p><p style="text-align: left;">这些限制选项包括</p><p style="text-align: left;">● 限制访问设置应用，但可允许某些设置页面（例如 WiFi 和亮度）</p><p style="text-align: left;">● 限制开始菜单仅显示允许运行的应用</p><p style="text-align: left;">● 在指定 UI 上阻止 Toast 通知和弹窗</p><p style="text-align: left;">Multi-App Kiosk Mode 是对很多人使用同一台设备场景的理想解决方案，可以覆盖一线员工、零售、<a data-link="1" href="https://c.duomai.com/track.php?k=yUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmYiJ05WZkVHdzZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyUiR" target="_blank">教育</a>和测试等场景。目前，Multi-App Kiosk Mode 可以通过 PowerShell 和 WMI Bridge 方式启用，未来即将支持 Intune/MDM 和配置软件包配置。</p></blockquote><p style="text-align: left;"><strong>该版本引入的改进包括</strong></p><blockquote style="text-align: left;"><p style="text-align: left;"><strong>[常规]</strong></p><p style="text-align: left;">● 上一篇博文中提到的关于 IT 管理员如何在 Windows 诊断数据处理器配置选项中注册设备的更改现在包含在此版本（内部版本 25169 和更高版本）中。提醒，如果记录的更改不可接受，建议暂停飞行。</p><p style="text-align: left;"><strong>[Windows Spotlight]</strong></p><p style="text-align: left;">● 现在，您可以通过全新的 Windows Spotlight 主题在桌面上启用 Windows Spotlight，该主题现在包含在 Dev Channel 的“设置”>“个性化”和“设置”>“个性化”>“主题”下的最新版本中。此外，您还可以使用 Windows Spotlight 的旋转图像制作自己的自定义主题。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0729/54e30034a90bd60.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;"><strong>[输入]</strong></p><p style="text-align: left;">● 我们更新了美国英语 (EN-US) 手写模型，使其更快、更准确。使用更新的手写面板进行墨迹转换，让我们知道您的想法。</p><p style="text-align: left;"><strong>[设置]</strong></p><p style="text-align: left;">● 设置现在支持管理以前仅从控制面板支持的应用程序。这包括卸载具有相互依赖关系的应用程序（例如，在 Steam 上运行的 Steam 和游戏应用程序）、修复和修改 Win32 应用程序。</p><p style="text-align: left;"><strong>[Windows 安全]</strong></p><p style="text-align: left;">● Windows 安全应用版本现在显示在关于下的 Windows 安全设置中。</p></blockquote><p style="text-align: left;">以下是 Build 25169 中的所有修复：</p><blockquote style="text-align: left;"><p style="text-align: left;"><strong>[文件管理器]</strong></p><p style="text-align: left;">● 修复了导致 Azure 信息保护上下文菜单项在升级后意外显示名称为 Microsoft.Azip.RightClick 的问题。</p><p style="text-align: left;"><strong>[任务栏]</strong></p><p style="text-align: left;">● 修复了在有时启动 Microsoft Teams 会议时阻止加载任务栏的 explorer.exe 崩溃。</p><p style="text-align: left;">● 修复了一些内部人员遇到的与任务栏溢出相关的几个 explorer.exe 崩溃问题。</p><p style="text-align: left;"><strong>[开始]</strong></p><p style="text-align: left;">● 修复了删除“开始”推荐部分中的“更多”按钮的问题。</p><p style="text-align: left;"><strong>[设置]</strong></p><p style="text-align: left;">● 修复了使用快速设置的 Wi-Fi 部分中的显示密码按钮后，文本字段可能意外失去焦点的问题。</p><p style="text-align: left;"><strong>[输入]</strong></p><p style="text-align: left;">● 我们进行了一些小调整，以提高传统触控键盘布局以及默认触控键盘布局的按键重复率，使其感觉更灵敏。现在它具有每秒 20 个键的速率（例如，当按住删除键时）。</p><p style="text-align: left;"><strong>[其他]</strong></p><p style="text-align: left;">● 修复了一个问题，该问题被认为是一些 Insider 最近在从网络位置打开和复制文件时遇到错误检查的根本原因。</p><p style="text-align: left;">● 修复了使用 <a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> Game Bar 录制游戏时可能导致音频不同步的问题。</p><p style="text-align: left;">● 修复了使用网络疑难解答时导致错误代码 0x80004005 的问题。</p><p style="text-align: left;">● 进行了一些调整以解决使用高 DPI 缩放时某些位置（如文件资源管理器）的滚动条意外宽的问题。</p><p style="text-align: left;">● 修复了一个被认为是 Windows 安全性的根本原因的问题，即标准硬件安全性在实际支持的设备上不受支持，以及在任务栏中显示警告但在 Windows 安全性启动时未显示任何问题。请注意，由于有效负载的时间安排，这些问题可能会在下一次 Windows 安全更新后再次出现。它会在未来的 Windows 安全更新后再次消失。</p><p style="text-align: left;">注意：此处在 Dev Channel 的 Insider Preview 版本中提到的一些修复可能会进入 Windows 11 已发布版本的服务更新。</p></blockquote><p style="text-align: left;">以下是 Build 25169 中的已知问题：</p><blockquote style="text-align: left;"><p style="text-align: left;"><strong>[常规]</strong></p><p style="text-align: left;">● 我们正在调查有关 SQL Server Management Studio 无法为某些 Insiders 启动的报告。</p><p style="text-align: left;">● 某些使用 Easy Anti-Cheat 的游戏可能会崩溃或导致您的 PC 出现错误检查。</p><p style="text-align: left;"><strong>[文件管理器]</strong></p><p style="text-align: left;">● 文件资源管理器选项卡中的向上箭头未对齐。这将在未来的更新中修复。</p><p style="text-align: left;">● 我们正在努力解决以下报告：在使用暗模式（例如，从命令行）时以某些方式启动文件资源管理器会在亮模式下意外显示文件资源管理器的主体。</p><p style="text-align: left;"><strong>[小部件]</strong></p><p style="text-align: left;">● 通知徽章编号可能会在任务栏上显示未对齐。</p><p style="text-align: left;">● 在某些情况下，某些徽章的通知横幅不会出现在小部件板上。</p><p style="text-align: left;">● 我们正在修复导致小部件首选项（温度单位和固定小部件）意外重置为默认值的问题。</p></blockquote><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0729/5b1a8310a01b012.webp" referrerpolicy="no-referrer"></p>
