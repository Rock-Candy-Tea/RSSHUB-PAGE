随着 ARC Alchemist 移动 GPU 的发布，英特尔也同步更新了驱动程序控制面板。新软件的亮点是引入了一个性能选项卡，便于用户监测 GPU 性能和体验超频等功能。<strong>有趣的是，3DCenter 发现，在 ARC Control 的视频演示期间，这家芯片巨头似乎泄露了未具名的桌面 GPU 的规格。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0331/c623749ea09be01.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0331/c623749ea09be01.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（via <a href="https://wccftech.com/intel-arc-alchemist-desktop-discrete-graphics-card-specs-arc-control-panel-reveal/" target="_self">WCCFTech</a>）</p><p>演示期间，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>展示了 ARC 控制面板的一些数据，比如内存 / 显存频率和 GPU 功耗，并且能够叠加在游戏画面上呈现。</p><p>然而有眼尖的人们发现，当英特尔在屏幕上来回切换的时候，显示的部分规格却与 ARC 移动 GPU 不太一样。</p><p><a href="https://static.cnbetacdn.com/article/2022/0405/662f5c013beb134.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0405/662f5c013beb134.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>比如当充分利用设备时，GPU 时钟可飙到 2200~2250 MHz 。正因如此，许多人推测它来自于尚未发布的 ACM-G10 旗舰 GPU SKU 。</p><p>此外 Geekbench 基准测试结果表明，英特尔 ARC A350M 移动在 OpenCL 测试期间能够冲到相同的加速频率（基础频率 1150 MHz）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0405/fa21999b98f3e9b.png" alt="2.png" referrerpolicy="no-referrer"></p><p>不过英特尔公司的 Tom Peterson 曾在接受 PC World 采访时透露，该数字更接近英伟达的 Boost 频率、而不是峰值，暗示该公司还是留了一手。</p><p>至于为了将这款神秘 GPU 视作 ARC A780 桌面独显，主要是软件中显示的功耗达到了 175W，明显高于常规移动 GPU 的设定 —— 就算是 ARC A770M，其功耗也才 150W 的水平。</p><p><a href="https://static.cnbetacdn.com/article/2022/0405/cb1fdc3c0fe4cdb.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0405/cb1fdc3c0fe4cdb.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>另一种观点是，英特尔的 Deep Link 技术，能够在 CPU 闲置时将更多功率动态地分配给 GPU，有些类似于英伟达的 Dynamic Boost 。</p><p>但是显存方面，泄露 GPU 的 VRAM 频率为 1093 MHz（可达 17.5 Gbps），远高于已知的任何移动 GPU SKU（16 Gbps）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0405/7a8cfeebd594608.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0405/7a8cfeebd594608.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（截图 via <a href="https://videocardz.com/newz/intel-teases-2250-mhz-boost-clock-and-175w-gpu-power-for-its-flagship-desktop-arc-gpu" target="_self">VideoCardz</a>）</p><p>受限于散热 / 功耗，移动 GPU 通常不会在超频特性上大做文章。因而综上所述，我们推测这款 ARC 桌面 GPU 的性能或与英伟达 GeForce RTX 3070 相当（搭配 14 Gbps @ GDDR6 显存）。</p>
