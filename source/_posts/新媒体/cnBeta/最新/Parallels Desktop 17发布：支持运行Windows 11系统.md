<strong>适用于 macOS 平台的 Parallels Desktop 17 于昨日正式发布，并特别针对 Windows 11 和 macOS Monterey 系统进行了优化。</strong>通过这项支持，用户可以通过 Parallels Desktop 17 Installation Assistant 在虚拟机上安装 macOS Monterey 和 Windows 11 的预览版本。<br>

src="//player.bilibili.com/player.html?aid=632225344&bvid=BV1Rb4y1z7a7&cid=386155865&page=1" scrolling="no" border="0" framespacing="0" allowfullscreen="true" width="750" height="480" frameborder="no"> </iframe></p><p style="text-align: left;">除了添加对上述两个系统的支持之外，Parallels Desktop 17 for Mac 还对基于 M1 芯片的 Mac 设备带来了性能改进。以下是新版本改进：</p><blockquote><p style="text-align: left;"><strong>必要的速度和性能更新</strong></p><p style="text-align: left;">在装有 M1 和 Intel 芯片的 Mac 设备上， Parallels Desktop 17 运行的应用程序更流畅、效率更高。</p><p style="text-align: left;"><strong>● 在所有支持的 Mac 设备上</strong></p><p style="text-align: left;">→ <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>、Linux 和 macOS 的恢复速度加快 38%</p><p style="text-align: left;">→ OpenGL 图形执行速度提高6倍</p><p style="text-align: left;">→ 在 Windows 中，2D 图形的速度最多提高 25%。</p><p style="text-align: left;"><strong>● 在装备<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a> M1 芯片的 Mac 设备上</strong></p><p style="text-align: left;">→ Windows 10 on ARM 预览版启动时间加快 33%</p><p style="text-align: left;">→ Windows 10 on ARM 预览版磁盘性能加快 20%</p><p style="text-align: left;">→ DirectX 11 图形性能提高 28%</p><p style="text-align: left;"><strong>● 在 Intel Mac 设备</strong></p><p style="text-align: left;">→ 在 macOS Big Sur（以及更新的）虚拟机上，网络连接速度最高可提高 60%。</p><p style="text-align: left;"><strong>增强 Windows 游戏体验</strong></p><p style="text-align: left;">Parallels Desktop 17 中改进的显示驱动提供了更流畅的 Windows UI 响应速度和同步的视频播放体验。新驱动程序提升了许多 2D 动作平台游戏的帧率，包括 Foregone、Smelter 和其他许多游戏，以获得更好的游戏体验。</p><p style="text-align: left;"><strong>针对 Apple M1 Mac 的强大新功能</strong></p><p style="text-align: left;">在 Parallels Desktop 17 中，Windows 10 可以识别 Mac的 电池状态，并在 Mac 电量不足时启用省电模式。新的虚拟 TPM 芯片使 Windows 10 和 Windows 11 可以使用 BitLocker 和 Secure Boot 来实现更高级别的数据保护。</p><p style="text-align: left;">在 Linux 虚拟机中享受多声道声音支持和带有本地驱动程序的插孔存在检测。在支持动态分辨率的窗口视图模式下方便地使用Linux虚拟机--调整虚拟机窗口的大小，Linux将为新的分辨率自动刷新显示。</p><p style="text-align: left;"><strong>Coherence 模式改进</strong></p><p style="text-align: left;">这是长期以来深受用户喜欢的功能，Coherence 模式现在被 80% 的 Parallels Desktop 用户使用，使他们可以并排运行 Windows 应用程序和 Mac 应用程序，而不需要看到 Windows 桌面。用户现在可以享受不间断的、无干扰的工作流程，因为所有的 Windows 关闭、更新和签到屏幕都是窗口化的，使用户可以最小化它们，保持专注和高效。</p></blockquote>
