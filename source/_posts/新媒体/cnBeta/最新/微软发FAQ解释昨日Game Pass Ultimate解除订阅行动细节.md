<strong>昨日微软封禁了一批滥用优惠的Game Pass Ultimate订阅用户的时长，许多从第三方渠道购买“代充”服务的用户受到了影响，一天之后，微软在官方社区贴出了FAQ，介绍了解除订阅行动的细节。</strong>微软表示：Xbox已经发现了一群欺诈性的转售商，他们将多个PC入门优惠的实例叠加起来，并将其转换为Game Pass Ultimate。为了解决这个问题，微软已经单独通知了受影响的用户，然后取消了他们以欺诈方式获得的订阅。<br>
 <p><strong><a href="https://static.cnbetacdn.com/article/2022/0618/b22b9a54544ed0c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0618/b22b9a54544ed0c.png" title alt="8Y)~9A_MP9Y3ZSP[CP)23_N.png" referrerpolicy="no-referrer"></a></strong></p><p><strong>为什么要取消这些 Xbox Game Pass 订阅？</strong></p><p>Xbox已经发现了一组欺诈性的经销商，他们正在堆叠多个入门优惠券的实例并将其转换为Game Pass Ultimate。为了解决此问题，我们在取消受影响的用户以欺诈方式获取的订阅之前，已单独通知他们。</p><p>通过违反 Microsoft 服务协议的来源获取的成员资格数量已选定，订阅必须直接从 Microsoft 数字商店购买，或通过经过验证的第三方零售商购买。</p><p><strong>会员帐户数据和游戏历史记录会怎样？</strong></p><p>这一切都将被保留。您的帐户将保留，但订阅将被停用。</p><p><strong>针对受影响的 Xbox Game Pass 成员的指南是什么？</strong></p><p>受影响的成员可以随时通过 加入 Xbox 游戏通行证重新订阅：发现您的下一个最喜爱的游戏|Xbox 或已验证的第三方卖家。</p><p><strong>这将影响多少成员？</strong></p><p>虽然我们没有透露受影响的成员数量，但这是 Xbox Game Pass 社区的一小部分。</p><p><strong>了解更多：</strong></p><p><a href="https://answers.microsoft.com/en-us/xbox/forum/xba_profile-xbo_topother/game-pass-subscription-cancellations/361b0608-c07b-41e8-9536-f6cb79318b88" _src="https://answers.microsoft.com/en-us/xbox/forum/xba_profile-xbo_topother/game-pass-subscription-cancellations/361b0608-c07b-41e8-9536-f6cb79318b88" target="_blank">https://answers.microsoft.com/en-us/xbox/forum/xba_profile-xbo_topother/game-pass-subscription-cancellations/</a></p><p><strong>建议购买过“代充”服务的玩家去购买处<strong>退款</strong>及索赔。</strong><br></p>
