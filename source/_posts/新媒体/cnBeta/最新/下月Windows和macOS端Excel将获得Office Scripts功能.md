<strong>微软正扩大 Excel 的 Office Scripts 适用范围</strong>。该功能于数月前开始测试，允许某个 SharePoint 站点的团队成员在 Excel 工作表上更轻松地协作、浏览和运行团队所有的脚本。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0913/c2991d91d04d736.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">Office Scripts 这个自动化功能集目前仅限于网页端。不过<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>承诺下月开始，该功能将会登陆 Mac 和 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 等平台。根据 Microsoft 365 Roadmap 页面，Mac 和 Windows 端的 Excel 用户可以在 10 月自动化执行重复的电子表格任务。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0913/d9c83f8019461d3.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0913/4e576aff8173cc1.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">和 Excel 网页端一样，在 Windows 和 Mac 平台上的 Excel 也可以使用 Code Editor 和 All Scripts 任务面板，创建、编辑和运行 Office Scripts。</p><p style="text-align: left;">创建、编辑和运行 Office Scripts 任务对于办公群体来说是非常实用的功能。您可以轻松地自动执行日常任务，这意味着您不必担心忘记步骤。但是，您必须耐心等到 10 月才能在 Excel for Mac 或 Windows 上使用它。<a href="https://support.microsoft.com/zh-cn/office/excel-%E4%B8%AD%E7%9A%84-office-%E8%84%9A%E6%9C%AC%E7%AE%80%E4%BB%8B-9fbe283d-adb8-4f13-a75b-a81c6baf163a" target="_blank">您可以在此处了解 Office Scripts 如何在 Excel 网页版上运行。</a></p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0913/db9938ef6d9aeb7.webp" alt="QQ截图20220913154743.webp" referrerpolicy="no-referrer"></p>
