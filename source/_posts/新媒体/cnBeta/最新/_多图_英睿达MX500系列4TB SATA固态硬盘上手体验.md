尽管近年来 PCIe NVMe 固态硬盘已经相当普及，但考虑到主流消费级 PC 仅提供 1~2 个 M.2 SSD
插槽，无论是专业用户、创作者、游戏玩家，台式机或笔记本电脑用户，SATA 接口硬盘仍有着相当庞大的市场需求。此前，我们已经上手过来自英睿达的
MX500 系列 1TB SATA 固态硬盘，其综合性能已经给我们留下相当深刻的印象。<strong>那么作为该系列 SATA SSD 的最高容量选项，4TB 版本的表现又如何呢？</strong><br>

src="https://static.cnbetacdn.com/thumb/article/2021/1209/6141da259b8e812.jpg" alt="IMG20211205125312.jpg" referrerpolicy="no-referrer"></a></p><p>英睿达（Crucial）标志性的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 包装，右下角表明为 MX500 系列 2.5 英寸固态硬盘，左下角贴纸强调为 4TB 容量版本。</p><p><a href="https://static.cnbetacdn.com/article/2021/1209/185ff68fd7dff44.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1209/185ff68fd7dff44.jpg" alt="IMG20211205125659.jpg" referrerpolicy="no-referrer"></a></p><p>透过背面小窗，可见固件版本号为 M3CR044，原厂提供五年有限质保。</p><p><a href="https://static.cnbetacdn.com/article/2021/1209/a13e86eae83126d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1209/a13e86eae83126d.jpg" alt="IMG20211205130557.jpg" referrerpolicy="no-referrer"></a></p><p>打开之后，内部可见采用标准 7mm 高度盘体的 MX500 4TB SATA SSD 本体，此外还有多语言说明纸、以及预留背胶的 7mm 转 9.5mm 塑料垫框。</p><p><a href="https://static.cnbetacdn.com/article/2021/1209/ce4f39c8a3308c7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1209/ce4f39c8a3308c7.jpg" alt="IMG20211205130857.jpg" referrerpolicy="no-referrer"></a></p><p><strong>（二）开盖：</strong></p><p><a href="https://static.cnbetacdn.com/article/2021/1209/b56be20976ef787.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1209/b56be20976ef787.jpg" alt="IMG20211205131440.jpg" referrerpolicy="no-referrer"></a></p><p>4TB 版 Crucial MX500 SATA SSD 的完整型号名称为 CT4000MX500SSD1，组装产地为墨西哥。移除 SSD 两侧的四颗螺丝，即可看到 MX500 4TB 版本的真容。</p><p><a href="https://static.cnbetacdn.com/article/2021/1209/410faf3a6039ab4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1209/410faf3a6039ab4.jpg" alt="IMG20211205131640.jpg" referrerpolicy="no-referrer"></a></p><p>打开盖子后，首先看到的是两颗位于 PCB 背面的镁光“NY135”闪存（以及旁边的 DRAM 缓存空焊位）。PCB 长度依然只有不到 2.5 英寸 SATA 盘体的一半，贴纸再次表明产地为墨西哥。</p><p><img src="https://static.cnbetacdn.com/article/2021/1209/d84451579e3f8c4.jpg" alt="IMG20211205132340.jpg" referrerpolicy="no-referrer"></p><p>卸下固定 SSD 本体的两枚螺丝，终于见到背面的另外两枚镁光“NY135”闪存（176L 3D TLC NAND），两者之间和主控上都带有方块状的导热贴，旁边是一枚来自<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://mall.jd.com/index-1000098521.html" target="_blank">美光</a>原厂的“<a href="https://www.micron.com/products/dram/ddr3-sdram/part-catalog/mt41k256m16tw-107" target="_self">D9SHD</a>”DRAM 缓存（4GB DDR3L-1866）。</p><p><img src="https://static.cnbetacdn.com/article/2021/1209/24f711189828fae.jpg" alt="IMG20211205133139.jpg" referrerpolicy="no-referrer"></p><p>揭掉贴纸，可见慧荣（Silicon Motion）的 <a href="https://www.siliconmotion.com/products/client/detail" target="_self">SM2259</a>H 四通道 SATA SSD 主控。不过与 MX500 1TB SATA SSD 一样，英睿达还是沿用了 1TB 闪存搭配 1GB DRAM 的设计。若后续还有更大的 8TB 容量型号，PCB 背面预留 DRAM 焊点也可以派上用场了。</p><p><img src="https://static.cnbetacdn.com/article/2021/1209/50b7b9e0786adf5.jpg" alt="IMG20211205141319.jpg" referrerpolicy="no-referrer"></p><p><strong>（三）跑分：</strong></p><p><img src="https://static.cnbetacdn.com/article/2021/1209/8d096c92d4323a4.png" alt="2218 - Crystal Disk Info.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">3-1：Crystal Disk Info / 全新通电 0 写入</p><p>测试平台选择了 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 R5-5600X 处理器（未开启 PBO）+ <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://msi-pc.jd.com/" target="_blank">微星</a> B550M 迫击炮 Wi-Fi 主板（开启
fTPM），搭配英睿达 8GB * 2 双通道 DDR4-3200 内存，M.2 NVMe SSD 上预装了 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11
操作系统（22000.318）。</p><p><a href="https://static.cnbetacdn.com/article/2021/1209/5bb85c1587dbfb2.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1209/5bb85c1587dbfb2.png" alt="0000 CrystalDiskInfo HDTuneWrite HWMonitor.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">Crystal Disk Info 硬盘信息 / HDTune Pro 全盘写入 / HWMonitor 温度监测</p><p>在 Windows 11 电源管理中开启高性能模式后（默认 AHCI 驱动），HDTune Pro 5.75 的全盘写入速度可达 403 MB/s，直到主控固件在后段改变了 SLC 缓存策略（平均 367 MB/s，最低 218 MB/s）。</p><p><a href="https://static.cnbetacdn.com/article/2021/1209/152f2b7b0b5f978.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1209/152f2b7b0b5f978.png" alt="2312 - HDTune - 200GB Mixed - 352 Write _ 546 Read + 315.4 Read.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">200GB 混合模式 / 全盘读取</p><p>全盘读取的话，MX500 4TB SATA SSD 可达 315 MB/s 。而在 HDTune Pro 的 200GB 混合模式下，该固态硬盘拿下了 546 MB/s 读取 和 352 MB/s 写入的成绩。</p><p><img src="https://static.cnbetacdn.com/article/2021/1209/f8fc215adfd393f.png" alt="2332 - CDM 1GiB - 563 _ 517 MB_S.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">Crystal Disk Mark</p><p>换用最普遍的 Crystal Disk Mark 工具软件（8.0.4 x64 版本），1GiB 区间，英睿达 MX500 4TB SATA SSD 顺序读取可达 563 MB/s、写入可达 517 MB/s，与之前在 Windows 10 21H1 系统上测试的 MX500 1TB 固态硬盘相当（算是踩到了 SATA 6Gbps 接口的理论上限）。</p><p><a href="https://static.cnbetacdn.com/article/2021/1209/3ad3db32c4664d6.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1209/3ad3db32c4664d6.png" alt="2338 - AS SSD - 1GiB - 1204 1201.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">AS SSD Benchmark</p><p>换用 AS SSD Benchmark 基准测试工具，顺序读取可达 527 MB/s、顺序写入 493 MB/s，多次尝试的综合得分都在 1200 以上。</p><p><img src="https://static.cnbetacdn.com/article/2021/1209/5eeefc3b5b5dde0.png" alt="2345 - ATTO - 537 _ 490 MB_S.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">ATTO 磁盘基准测试，顺序读取可达 538 MB/s、写入可达493 MB/s 。</p><p><strong>（四）文件传输测试：</strong></p><p><img src="https://static.cnbetacdn.com/article/2021/1209/b6d6fc6c9de028e.png" alt="2539 - Win11 ISO - 5.20GB COPY - 474 MB_S √.png" referrerpolicy="no-referrer"></p><p>以 5.20GB 的 Windows 11 安装镜像为例，单文件的持续传输速率可在 474 MB/s 左右。</p><p>从 PCIe NVMe M.2 SSD 系统盘复制到英睿达 MX500 4TB SATA SSD 的用时在 10 秒左右。</p><p><a href="https://static.cnbetacdn.com/article/2021/1210/a2695f07fe4100e.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1210/a2695f07fe4100e.png" alt="2542 - Win11 Folder - 5.20GB COPY - 475 MB_S √.png" referrerpolicy="no-referrer"></a></p><p>将 Windows 11 ISO 镜像通过虚拟光驱加载，零散文件（夹）的复制（写入）速度也有 475 MB/s，用时相差不大。</p><p>然后试着将 MX500 4TB SATA SSD 上的 5.20GB 镜像回传（读取）至 M.2 SSD 系统盘，速度有 466 MB/s 。</p><p><a href="https://static.cnbetacdn.com/article/2021/1210/e6a36e13ec6a451.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1210/e6a36e13ec6a451.png" alt="2612 Win11 ISO - In Disk Clone - 206 MB_S.png" referrerpolicy="no-referrer"></a></p><p>英睿达 MX500 4TB SATA SSD，盘内（单分区）文件复制。ISO 单文件 / 零散文件（夹）的速度都有 200 MB/s 以上，用时 20 秒左右。</p><p><strong>（五）小结：</strong></p><p><a href="https://static.cnbetacdn.com/article/2021/1210/93694cc2268adc8.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1210/93694cc2268adc8.png" alt="2246 - Disk Manager - GPT initialization +.png" referrerpolicy="no-referrer"></a></p><p>对于需要扩容高速硬盘，但板载 M.2 SSD 接口又有限的笔记本电脑 / 台式机用户来说，英睿达 MX500 系列 4TB SATA SSD 提供了一个相当不错的选择。</p><p>不过需要注意但是，受 MBR（主启动）分区表限制，购买 2TB / 3TB 及更高容量的硬盘之后，Windows 默认会推荐采用 GPT（GUID）分区表。</p><p><a href="https://static.cnbetacdn.com/article/2021/1210/9d2f4852cbcbe9b.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1210/9d2f4852cbcbe9b.png" alt="2436 MX500 Official Site.png" referrerpolicy="no-referrer"></a></p><p>感兴趣的朋友，可以选购 250GB / 500GB / 1TB / 2TB / 4TB 版本，目前美区官网上的报价为 <a href="https://www.crucial.com/ssd/mx500/ct500mx500ssd1" target="_self">49.99</a> / <a href="https://www.crucial.com/ssd/mx500/ct500mx500ssd1" target="_self">59.99</a> / <a href="https://www.crucial.com/ssd/mx500/ct500mx500ssd1" target="_self">99.99</a> / <a href="https://www.crucial.com/ssd/mx500/ct2000mx500ssd1" target="_self">199.99</a>  / <a href="https://www.crucial.com/ssd/mx500/ct4000mx500ssd1" target="_self">359.99 美元</a> 。</p><p>不过国内消费者可以留意最近的活动，目前英睿达<a data-link="1" href="https://c.duomai.com/track.php?k=nJ9QWa1VmJxYTPklWYmYDO5IDNy0DZp9VZ0l2cmYiJt92YuQmauc3d3ZkMlYkMlE0MlMHc0RHa9Q" target="_blank">京东</a>自营旗舰店上的 <a href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://item.jd.com/6164585.html" target="_self">250GB</a> / <a href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://item.jd.com/6164595.html" target="_self">500GB</a> / <a href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://item.jd.com/6164587.html" target="_self">1TB</a> / <a href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://item.jd.com/6130426.html" target="_self">2TB</a> / <a href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://item.jd.com/100015306127.html" target="_self">4TB</a> 版本，参考价分别为 319 / 399 / 699 / 1399 / 2899 元。</p>
