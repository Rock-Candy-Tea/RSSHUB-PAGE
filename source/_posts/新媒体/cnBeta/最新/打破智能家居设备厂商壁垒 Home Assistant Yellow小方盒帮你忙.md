作为一个开源项目，Home Assistant 旨在打破不同厂商之间的智能家居硬件的体验壁垒。尽管上手难度有点高，但它还是凭借强大的功能而吸引了广大支持者。仅今年在 GitHub 上，该项目就得到了超过 8000 名开发者的积极贡献。<strong>而本文要为大家介绍的，就是上线了 Crowd Supply 众包、旨在简化 Home Assistant 上手体验的 Yellow 。</strong><br><br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0715/b9c83939c6365d5.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0715/b9c83939c6365d5.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：<a href="https://www.theverge.com/2021/9/16/22678088/home-assistant-amber-smart-home-hub-specs-release-date-price" target="_self">Crowd Supply</a>）</p><p>免费开源软件背后的 Nabu Casa 公司，推出了采用小方盒造型的 Home Assistant Yellow 枢纽硬件（曾用名 Amber）。</p><p>其核心部件是一块树莓派 4，辅以内置的 Silicon Labs 模块 —— 为 Home Assistant 主要的 Zigbee 3.0 通信协议提供支撑。</p><p>通过这一板载功能，用户可将流行的智能家居产品（比如飞利浦 Hue 智能灯泡和宜家智能窗帘）挂接到 Home Assistant Yellow 上。</p><p>无需原厂 Hub 桥接，即可在智能<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>、平板电脑和计算机上通过 Home Assistant 软件来控制。</p><p><a href="https://static.cnbetacdn.com/article/2022/0715/44a1e08e88ab8ae.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0715/44a1e08e88ab8ae.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>Silicon Labs 芯片还支持 OpenThread，并正努力使其与开放式的 Matter 智能家居标准兼容。</p><p>如需兼容 Z-Wave，亦可使用 USB 适配器、或连接到 Vera / SmartThings 等第三方 Hub 来添加支持。</p><p>此外 Home Assistant Yellow 提供了 M.2 扩展端口，以便用户插入固态硬盘来获得更多存储空间、以及扩展其它功能。</p><p>需要指出的是，其并未内置蓝牙或 Wi-Fi 无线适配器，主要通过有线连接（千兆以太网）来实现智能家居设备的互通。</p><p><a href="https://static.cnbetacdn.com/article/2022/0715/d46086329d5f0fb.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0715/d46086329d5f0fb.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>Home Assistant 创始人 Paulus Schoutsen 解释称，大家不希望通过不那么稳定的无线网络来连接智能家居的大脑。</p><p>至于 Yellow 的<a href="https://www.crowdsupply.com/nabu-casa/home-assistant-amber" target="_self">完整规格</a>，其实与其它基于树莓派 4 的系统一致，只是砍去了无用的显示输出接口。</p><p>所有与维护和设置相关的操作，都可通过 Home Assistant 的移动 App 或桌面应用程序来完成。</p><p><a href="https://static.cnbetacdn.com/article/2022/0715/1f1354b228d7d99.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0715/1f1354b228d7d99.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p>更棒的是，即使在互联网意外中断的情况下，Home Assistant 依然能够在本地提供智能家居设备的访问与控制。</p><p>最后，作为一款注重安全和隐私的私有云产品，Home Assistant Yellow 完全与平台无关。</p><p>目前其已提供对上千款兼容设备的集成支持，包括但不限于<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a> HomeKit、<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a> SmartThings、或亚马逊 Alexa（<a href="https://www.home-assistant.io/integrations/#all" target="_self">完整列表</a>）。</p>   
</div>
