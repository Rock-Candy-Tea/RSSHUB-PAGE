外媒报道称，CTR 与 Hydra 的开发者 Yuri Bubily（又名 1usmus）正在打造一款全新的硬件优化实用工具。不同的是，这次他将更多精力放在了 GPU、而不是 CPU 的性能调优上。<strong>据悉，该工具名叫“Radeon Monster Profile”（以下简称 RMP），其旨在提升 GPU 性能的同时压低功耗。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0926/54be87a75ef005e.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（via <a href="https://wccftech.com/radeon-master-profile-rmp-enhanced-tuning-utility-amd-rdna-2-gpus-coming-this-fall/" target="_self">WCCFTech</a>）</p><p>按照计划，Radeon Monster Profile 将于今秋推出。此外虽然初期仅适配 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> RDNA 2 GPU，但后续 yuri Bubily 也会添加对 RDNA 3 GPU 的支持。</p><blockquote><p>Yuri 表示 —— RMP 结合了独特的设置、利用新的电压 / 频率曲线来优化 AMD Radeon GPU 的性能和功耗，以最大限度地发挥其潜力。</p><p>有趣的是，尽管 GPU 是显卡的主要构成部分，但 RMP 尤为关注显存子系统的表现 —— 通过将电压下调 6%，以降低 GDDR6 显存模块的功耗与发热。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0926/88629f52356adc2.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0926/88629f52356adc2.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>具体说来是，Radeon Monster Profile 通过提高相对于工作电压的请求核心（GPU）频率来进一步提升性能。</p><p>以 RX 6800 XT 显卡为例，与标准操作相比，其能够实现高达 +300 MHz 的频率提升（较默认 +13%）。</p><p>在功耗近似的情况下，消费者们每一块钱的帧率性能提升了 21% —— 这点给我们留下了尤为深刻的印象。</p><p><a href="https://static.cnbetacdn.com/article/2022/0926/f3649a1dcbe1792.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0926/f3649a1dcbe1792.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>此外 Radeon Monster Profile 将提供多个通用和自适应的配置文件，而无需修改存储在 GPU BIOS 中的出厂设置参数 —— 比如温度墙 / <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C738%2C751" target="_blank">风扇</a>转速、或增加电压 / 改变电流限制。</p><p>需要注意的是，在运用 RMP“鸡血配置文件”之前，还请确保 GPU 拥有充分的散热支撑、以实现更符合预期的大幅性能优化。好消息是，据说大部分 RX 6000 系列非公显卡都满足所需的散热条件。</p><p><a href="https://static.cnbetacdn.com/article/2022/0926/e2ab5f720bb07ed.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0926/e2ab5f720bb07ed.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>最后，如果一切顺利，Radeon Monster Profile 将于今秋的发布会上，被官宣集成到用于调教 AMD Ryzen CPU 的 Hydra 工具套件中。</p><p>不过鉴于 RMP 仍在开发过程中，早期晒出的性能数据或与最终产品不尽相同。且该工具首发仅支持 RNDA 2 GPU，RDNA 3 GPU 还得再多等一段时间。</p>
