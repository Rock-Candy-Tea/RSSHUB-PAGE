说到 Windows 11 的变化，许多 Windows 10 老用户都觉得 UI 交互的改动最大。<strong>然而并不是每个人都喜欢 Windows 11 的外观样式，比如默认居中放置的任务栏图标。</strong>好消息是，有许多开发者正在积极打造第三方工具，以恢复经典的 Windows 10 风格元素（例如 Dot Matrix 的作品）。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1019/32f7060b4f67a83.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>不过本文要为大家介绍的，则是 MikeRosoft6009 正在开发中的“Rectify 11”。</p><p><img src="https://static.cnbetacdn.com/article/2021/1019/596a97a3d5667e9.png" alt="0.png" referrerpolicy="no-referrer"></p><p>其表示，“Rectify 11”主要提供以下方面的风格修改：</p><blockquote><p>· 设置主题</p><p>· <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>管理控制台（MMC）</p><p>· 管理工具</p><p>· 开始菜单（集成 StartIsBack）</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/1019/b78fc53e8ba3628.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>此外“Rectify 11”的未来版本（v2），也有望带来流畅的 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 恢复环境（WinRE）图标。</p><p><a href="https://static.cnbetacdn.com/article/2021/1019/57f45e030d3f4a6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1019/57f45e030d3f4a6.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>感兴趣的朋友，可下载体验 v1 版本的第三方 ISO 镜像安装包（风险请自行承担）：</p><blockquote><p>https://drive.google.com/file/d/1JFQlCAvB1xeN7iDwts5Cy4zBiAPWA_F2/view</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2021/1019/22f8c28977ec316.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1019/22f8c28977ec316.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>此外上方显示的安装起始屏幕展示的是 v2 样式，或与 v1 镜像有些许不同。</p>
