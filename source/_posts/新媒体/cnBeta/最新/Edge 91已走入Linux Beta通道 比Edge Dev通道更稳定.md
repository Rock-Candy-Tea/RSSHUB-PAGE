本周，微软给 Edge Insider 测试者们带来了多个新消息。<strong>首先，Microsoft Edge 的 92.0.884.2 版本已转入 Dev 通道，但这里的变化还不是很大。其次，Edge 91 的 Beta 版本也终于登陆了 Linux 平台。</strong>不过由于遇到了开发上的一些问题，Linux Dev 通道可能会被跳过。除此之外，其它操作系统上的 Dev 推送还引入了两项新功能。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0513/830ee11a0bc8b48.jpg" referrerpolicy="no-referrer"></p><p>首先，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>为 Edge Dev 加入了“引导切换”（Guided Switch），以记住每个网站要使用的配置文件。其次，他们还在“打印预览”中提供了“适合页面”和“实际尺寸”两种选项。</p><p><strong>以下是 Edge Dev 的其它可靠性修复和功能改进：</strong></p><blockquote><p>● 修复播放某些受 DRM 保护的视频时，可能会崩溃的问题。</p><p>● 修复在没有打开常规浏览器窗口的情况下，使用 PWA 应用或网站时可能导致浏览器崩溃的问题。</p><p>● 修复从其它浏览器导入数据时，可能崩溃的问题。</p><p>● 修复在 InPrivate 窗口中输入是，地址栏可能崩溃的问题。</p><p>● 修复某些视频导致 GPU 进程内存泄漏的问题。</p><p>● 修复某些弹出窗口（如权限提示）为空白的问题。</p><p>● 修复搜索设置导致“设置”页面为空白的问题。</p><p>● 修复删除搜索提供程序时，偶尔会意外禁用其它功能的问题。</p><p>● 修复从已安装的应用启动 PWA 或网站时，用于打开收藏夹 / 历史记录的快捷键可能失效的问题。</p></blockquote><p>至于 Edge Dev 中的已知问题，依然和之前的相同。感兴趣的朋友，可移步至 Microsoft Edge Insider <a href="https://techcommunity.microsoft.com/t5/discussions/dev-channel-update-to-92-0-884-2-is-live/m-p/2349464" target="_self">讨论板</a>进行查看。</p><p>最后，对于 Linux 用户而言，Edge 91 Beta 将比 Edge Dev 分支更加稳定（其它平台早在 4 月 30 日就推送了）。有需要的朋友，可按需下载 <a href="https://www.microsoftedgeinsider.com/en-us/download/?platform=linux-deb" target="_self">.DEB</a> 或 <a href="https://www.microsoftedgeinsider.com/en-us/download/?platform=linux-rpm" target="_self">.RPM</a> 格式的文件包。</p>
