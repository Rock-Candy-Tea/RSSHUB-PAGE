<strong>Google 工程师刚刚推出了 Cirq 量子编程框架的首个完整版本，并且提供了稳定的 API 。</strong>据悉，作为一套适用于量子计算的开源编程框架，Cirq 1.0 本体也是由 Python 编写得来的 —— Google 开源工程师将之描述为“一款用于编写、运行和分析量子计算机程序结果的 Python 框架”。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0720/9b42a0835225d10.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0720/9b42a0835225d10.png" referrerpolicy="no-referrer"></a></p><p><a href="https://www.phoronix.com/scan.php?page=news_item&px=Google-Cirq-1.0" target="_self">Phoronix</a> 指出，Cirq 专为近期的量子计算机而设计，即那些拥有数百量子比特和几千个量子门的量子计算机。</p><p>通过多年持续努力，并与学术界和其它行业成员合作发展，这个用于量子计算的 Python 编程框架，已在 Google 自家的量子计算硬件上运行。</p><p><a href="https://static.cnbetacdn.com/article/2022/0720/395418eda350a2d.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0720/395418eda350a2d.png" referrerpolicy="no-referrer"></a></p><p>过去四年，这家科技巨头在 Cirq 基础上构建了 TensorFlow Quantum、OpenFermion、Qsim 和其它软件。</p><p>感兴趣的朋友，可移步至 Google <a href="https://opensource.googleblog.com/2022/07/Cirq-Turns-1.0.html" target="_self">开源博客</a>，以了解今天发布的 Cirq 1.0 有关的更多细节。</p><p>此外项目主站域名为 <a href="https://quantumai.google/cirq" target="_self">quantumai.google</a>，相关代码可通过 <a href="https://github.com/quantumlib/cirq" target="_self">GitHub</a> 获得。</p>
