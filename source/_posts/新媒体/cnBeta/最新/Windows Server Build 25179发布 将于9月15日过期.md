<strong>微软发布了 Windows Server Build 25179，该预览版已经开放 ISO 和 VHDX 两种格式下载。</strong>虽然新预览版采用了和 Windows 11 相同的底层内核，但和往常一样，微软官方并未提供更新日志。因此我们不清楚其新版本中新功能或者改进。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0811/2cdb5d773e5d62d.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0811/91293b8b47348f3.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">同样，在该预览版中品牌尚未更新，在预览版中仍为 Windows Server 2022。此外，微软要求 Insiders 将这些版本称为 vNext，而不是已经上市的 Windows Server 2022。</p><p style="text-align: left;"><strong>可下载内容：</strong></p><blockquote style="text-align: left;"><p style="text-align: left;">● ISO 格式的 Windows Server VNext Preview 提供 18 种语言，</p><p style="text-align: left;">● VHDX 格式仅提供英文</p><p style="text-align: left;">● Windows Server VNext 预览版 VHDX</p><p style="text-align: left;">● Microsoft 服务器语言和可选功能预览</p><p style="text-align: left;">● Windows 管理中心 2110.2 预览版</p></blockquote><p style="text-align: left;">密钥仅对预览版本有效：</p><blockquote style="text-align: left;"><p style="text-align: left;"><strong>服务器标准版：</strong>MFY9F-XBN2F-TYFMP-CCV49-RMYVH</p><p style="text-align: left;"><strong>数据中心：</strong>2KNJJ-33Y9H-2GXGX-KMQWH-G6H67</p></blockquote><p style="text-align: left;">要下载，已注册的预览体验成员可以前往 <a href="https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewserver" target="_blank">Windows Server Insider Preview 下载页面</a>。如果您尚未注册，您还可以查看 <a href="https://insider.windows.com/en-us/for-business" target="_blank">Windows Insider for Business</a> 网站上的“<a href="https://insider.windows.com/en-us/for-business-getting-started-server/" target="_blank">服务器入门</a>”页面，以注册 Windows Insider 计划。</p><p style="text-align: left;"><span data-language-for-alternatives="zh-CN" data-language-to-translate-into="en" data-phrase-index="0" data-number-of-phrases="1">微软还指出，<strong>此预览版将于 2022 年 9 月 15 日到期。</strong></span></p>
