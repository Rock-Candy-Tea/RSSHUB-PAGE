在早前发布的 Chrome 97 版本中，Google 悄然引入了一项变化。<strong>作为搜索引擎设置页面改造的一部分，新版竟然不允许用户删除默认的搜索引擎提供商选项。</strong>但是在引发了舆论的一片哗然之后，Google 又承诺在浏览器设置 UI 中恢复该选项。在一段时间的等待之后，我们终于在 Chrome 97 Canary 分支中见到了该功能的回归。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0119/eebd0127651b46a.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">Chrome 97 截图（via <a href="https://techdows.com/2022/01/google-restores-chrome-default-search-engines-delete-option.html" target="_self">Techdows</a>）</p><p>近日更新到 Chrome 97 的朋友，或许留意到浏览器预置的 Google、Bing、Yahoo、DuckDuckGo 和 Ecosia 等搜索引擎已默认无法删除。</p><p>如上图所示，点击“竖排三点”菜单图标后，只能看到“设为默认”和“编辑”这两个选项。在 Google 给出正式回应前，不少用户在 Reddit 等社交媒体上发布了吐槽帖。</p><p><strong>此外一位用户在 Chrome 帮助论坛上汇报称：</strong></p><blockquote><p>我的搜索引擎总是不断被改成 Yahoo！，即便尝试移除 Chrome 设置页面中的默认选项，‘竖排三点’菜单键也看不到删除的选项。</p><p>我该如何解决这个问题？我已尝试重置设置、清理并检索计算机上是否存在任何恶意软件。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0119/7a2808bea82679e.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">Chrome 99 Canary 截图</p><p>Google 员工 Justin Donnelly 解释称，这应该只是 Chrome 97 引入的一个 Bug 。这项变动的本意是避免意外删除内置的搜索引擎选项，而不是用于防止搜索引擎劫持。</p><p><strong>在另一则相关的 Chrome Bug 报告中，还有提到“移除预置的搜索引擎太过容易”。</strong></p><blockquote><p>一些朋友正在将移除作为一种解决方案，以阻止对默认搜索引擎实施不必要的更改，所以我们现在应该继续提供支持。</p><p>不过为了避免意外删除，之后大家在尝试删除预置搜索引擎选项时，Chrome 还是会显示一个确认对话框。</p></blockquote><p>如果你已经在使用最新的 Chrome 99 Canary，<strong>可尝试在多功能地址栏里输入 chrome://settings/searchEngines 并直接跳转，</strong>然后就会看到如上图所示的“删除选项”。</p><p>有趣的是，这项变化也影响到了采用 Chromium 内核的 Microsoft Edge 浏览器。为此感到烦恼的朋友，还请耐心等待新版本的推送。</p><p>最后，按照计划，Google 将于今年 2 / 3 月初推送 Chrome 98 / 99 正式版本。</p>
