<strong>英特尔客户端图形产品与解决方案部门负责人指出，当 ARC 高性能游戏显卡正式向市场推出时，配套的驱动程序 UI 也将做好提供超频支持的准备。</strong>在 Medium 上发表的一篇文章中，Usman Pirzada 详细谈到了基于 Xe-HPG 架构的 Alchemist GPU 产品线，阐明了 ARC 品牌及相关产品在发布时的一些预期功能，比如完全符合 DirectX 12 Ultimate 的各项要求。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0825/382a194a4502139.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0825/382a194a4502139.png" alt="0.png" referrerpolicy="no-referrer"></a></p><p>作为英伟达 GeForce 与 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> Radeon 之外的第三大游戏 GPU，英特尔 ARC 系列图形产品也将支持超频特性。</p><p><a href="https://wccftech.com/intel-arc-graphics-cards-xe-hpg-gpu-ready-to-overclock-through-driver-ui-at-launch/" target="_self">WCCFTech</a> 指出，目前红绿两家 GPU 大厂的显卡，能够借助多款实用工具软件进行超频。除了驱动程序 UI 内置的选项，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://msi-pc.jd.com/" target="_blank">微星</a>（<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://msi-pc.jd.com/" target="_blank">MSI</a>）的 Afterburner 也相当知名。</p><p>据悉，超频实用程序能够在时钟频率、芯片电压、<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C738%2C751" target="_blank">风扇</a>转速、以及自定义电源计划 / 功耗限制等方面，提供了更加宽泛的控制体验。</p><p>通常情况下，GPU 厂商会先在官方驱动 UI 中开放超频支持，然后 AIB 显卡制造商能够围绕新架构来优化超频软件和和实用程序。</p><p>目前尚不清楚英特尔是否会在 ARC 游戏显卡发布时提供想同级别的 AIB 支持，但我们至少会先在蓝厂的驱动程序 UI 中首个超频选项。</p><p><img src="https://static.cnbetacdn.com/article/2021/0825/faa6be105bc5ddb.png" alt="1.png" referrerpolicy="no-referrer"></p><p>英特尔表示：许多游戏玩家同样也是创作者，因而该公司正在为其开发强大的捕捉功能，以利用其强大的编码硬件。</p><p>其中包括带有 AI 辅助功能的虚拟摄像头，以便轻松记录游戏中的高光时刻。甚至将超频控制集成到了驱动程序 UI 中，为发烧友们提供了将硬件推向极限所需的第一方工具。</p><p>此外英特尔一个相当不错的驱动程序套件，能够对新近的独立和集成显卡解决方案进行超频。一旦 ARC 显卡正式上市，我们还可期待这方面的持续改进。</p><p>最后，英特尔证实了基于 Xe-HPG Alchemist GPU 架构的 ARC 系列显卡，将完全符合 DirectX 12 Ultimate 标准。</p><p><img src="https://static.cnbetacdn.com/article/2021/0825/7a6178e04deef3f.png" alt="2.png" referrerpolicy="no-referrer"></p><p>众所周知，在硬件的原始性能之外，驱动程序也是产品体验的重要组成部分。英特尔最近在核显领域取得了长足进步，且过去三年一直在与<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>保持密切合作，其中就包括 DirectX 12 Ultimate 的共同设计。</p><p>除了 DXR 光线追踪，英特尔 ARC 图形产品还支持“可变着色速率”和 XeSS 超级采样（对标英伟达 DLSS 与 AMD FSR）等高级特性，以进一步提升性能体验。</p>
