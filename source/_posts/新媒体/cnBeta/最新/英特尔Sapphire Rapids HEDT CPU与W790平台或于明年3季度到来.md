有关英特尔下一代“Sapphire Rapids”、以及“Raptor Lake”主流台式处理器的传闻，已经在网络上广泛流传。昨日，@热心市民描边怪 又在 B 站动态上揭示了一些有趣的细节。<strong>除了 12 代 Alder Lake 的缓存与频率，他还提到了 Sapphire Rapids 高端台式（HEDT）与 13 代 Raptor Lake 处理器有关的内容。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0917/6db328bf98c6954.png" referrerpolicy="no-referrer"></p><p>早些时候，@<a href="https://t.bilibili.com/570744524496095466" target="_self">热心市民描边怪</a> 已经爆料过 LGA 1700 插槽和 Z690 芯片组。而昨日的动态，首先提到了酷睿 i9-12900K、i7-12700K、以及 i5-12600K 的缓存与频率规格。</p><blockquote><p>若消息靠谱，那采用 P8 + E8 大小核设计的 i9-12900K 拥有 12ML L2 + 30MB L3 缓存，基础频率 3.2GHz / TB3.0 睿频可达 5.2GHz 。</p><p>其次是采用 P8 + E4 大小核设计的 i7-12700K，该芯片拥有 12MB L2 + 30MB L3 缓存，基础频率 3.6GHz / TB3.0 睿频可达 5GHz 。</p><p>然后是采用 P6 + E4 大小核设计的 i5-12600K，该芯片拥有 9.5MB L2 + 20MB L3 缓存，基础频率 3.7GHz / TB2.0 睿频可达 4.9GHz 。</p></blockquote><p>爆料人还称，非 -K 版本的 12 代酷睿 i9 / i7 CPU SKU，睿频速率要低 100MHz 。此外下一代酷睿 i3 将不是 Rocket Lake 刷新版，而是基于 Golden Cove 核心架构。</p><p>核显方面，它将用上基于 Xe 架构的 UHD 770 系列，且处理器支持 DDR5-4800 / DDR4-3200 内存。</p><p>至于<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>是否会将主流 / 入门级的 H670 / B660 / H610 芯片组的发布时间同样调整到 2021 年底，仍有待进一步观察。</p><p><img src="https://static.cnbetacdn.com/article/2021/0917/be0f3dc3df082a5.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/intel-next-gen-hedt-sapphire-rapids-cpus-launch-w790-platform-13th-gen-raptor-lake-cpus-q3-2022/" target="_self">WCCFTech</a>）</p><p>回到故事主线，据说英特尔“Sapphire Rapids”高端台式（HEDT）处理器家族将登陆 W790 平台。即使早前有驱动列表揭示了所谓的 X699 芯片组，爆料人还是觉得可能性不大。</p><p>从 W790 芯片组的定位来看，“Sapphire Rapids”HEDT CPU 系列更像是主打专业消费级市场 —— 类似于 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 的 Threadripper Pro 。</p><p>更重要的是，该阵容预计会在 2022 年 3 季度、与英特尔 13 代 Raptor Lake CPU 同期亮相。预计 Sapphire Rapids-SP 至强处理器最高拥有 56C / 112T，且采用 LGA 4677 插槽。</p><p>但英特尔仍可能为 Sapphire Rapids 工作站（W）和 HEDT（X）芯片组平台选用不同的插槽设计，且后者的核心数量会更少 —— 除非该公司想要在该领域向 AMD 线程撕裂者重新发起挑战。</p><p>最后，我们可以期待一些在 Sapphire Rapids CPU 上应用的新技术，比如 10nm 增强型 SuperFin 工艺、64 条 PCIe 5.0 通道、8 通道 DDR5 内存控制器、以及面向专业消费者 / 发烧友的增强特性。</p>
