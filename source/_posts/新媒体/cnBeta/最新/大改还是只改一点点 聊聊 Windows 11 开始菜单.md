或许是曾经说 “Windows 10 是最后一版 Windows” 的 Jerry Nixon 离职了，所以微软发布了 Windows 11。 当然这是玩笑，微软选择 Windows 11， 大概率是因为微软觉得这个版本对于自己来说非常重要，同时相对于此前的变化过大，用新名字更能代表它的大变化，同时更容易让大家接受。试想如果 Windows 11 还叫 Windows 10， 恐怕就有人跳出来说“步子迈大了容易扯着蛋”。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0702/9c3badfa96d0529.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0702/9c3badfa96d0529.jpg" alt="photo_2021-07-01_15-32-13.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;"><strong>Windows 11</strong></p><p style="text-align: left;">其实名字并不重要，重要的是变化。<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>很自然的借鉴了 macOS 的特点，将底部的图标居中了。不过这样做似乎也并没有大幅提升微软的品味，只换来大家异口同声的抄袭。当然，在我看来 ，Windows 11 与 macOS 在底部设计的区别依旧大了去了 。macOS 一直始终如一，而 Windows 则捉摸不定，在我看来甚至走了回头路。</p><p style="text-align: left;">Windows 11 的开始菜单，给我的感觉是外观上似乎大改，但只是操作上的一点点改动而已，而且改的也不好。</p><p style="text-align: left;">个人认为，开始菜单的意义在于让用户用更少的操作做更多的事情。所以我就以这一点为衡量标准，简单回顾一下开始菜单。</p><p style="text-align: left;">大家最熟悉的 Windows XP、Windows 7， 其开始菜单是非常简单的，左侧是固定的程序，右侧则是与用户相关的文件夹。它的逻辑非常简单，默认用户将重要资料存储在用户文件夹中，用户将常用软件固定在左侧。这样一来，无论是打开软件还是查找资料，都能尽可能减少用户点击。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0702/3346f6f01cb0ece.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0702/3346f6f01cb0ece.jpg" alt="j61huvoi.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>Windows XP</strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0702/702966af44d26ea.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0702/702966af44d26ea.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>Windows Vista</strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0702/b37575122a2a715.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0702/b37575122a2a715.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>Windows 7</strong></p><p style="text-align: left;">举一个例子，用户要打开没有固定在左侧的程序，需要依次点击开始菜单-所有程序-找到对应文件夹或者程序图标-点击图标，除了第一步和第四步的必要操作，中间需要两步，共四步。我们将此作为标准。</p><p style="text-align: left;">到了 Windows 8， 很明显微软没有让用户用更少的操作做更多的事情，完全是反着来，用户操作一通，完全不知道想要找的应用在哪，这也是 Windows 8 开始菜单失败的原因。它挑战了用户习惯，然后让用户操作变得更加繁琐。由于大家纷纷用行动表明 Windows 8 是一款失败的操作系统。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0702/823c197de252789.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0702/823c197de252789.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>Windows 8</strong></p><p style="text-align: left;">同样打开一个没有额外固定位置的程序 ，Windows 8 用户需要依次点击开始菜单-点击向下展开所有程序-找到对应文件夹或者程序图标-点击图标，同样是四步，但整个过程在完全不同的界面之间切换，并且由上下滑动查找改为左右滑动查找，操作更加复杂。</p><p style="text-align: left;">Windows 10 就更加直接了，左侧为所有程序，右侧为固定的程序，所有操作均为上下滑动，与 Windows 7 及以前的用户习惯保持一致，并且所有程序更加直观。这种设计的好处在于将操作进一步简单化。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0702/c6ae2f4ae5c7d9b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0702/c6ae2f4ae5c7d9b.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>Windows 10</strong></p><p style="text-align: left;">咱们看看在 Windows 10 下打开没有固定位置的应用需要几步，点击开始菜单-上下滑动查找对应文件夹或者程序图标-点击图标，只有三步，而且整个操作在同一个界面下进行，直观、简单。</p><p style="text-align: left;">Windows 11 变的更像 Windows 7 了，这么说是因为 Windows 11 的开始菜单上只有固定的程序和推荐的程序，其实两者相当于 Windows 7 左侧的固定程序，只不过一个是左右排布，一个是上下排布。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/thumb/article/2021/0702/e1dd9a8cf505af7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0702/e1dd9a8cf505af7.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;"><strong>Windows 11</strong></p><p style="text-align: left;">查找一个应用的操作步骤与 Windows 7 保持一致，即依次点击开始菜单-所有程序-找到对应文件夹或者程序图标-点击图标，比 Windows 10 多了一步。</p><p style="text-align: left;">我所说的 Windows 11 开始菜单看似大改，实际只改了一点点，体验却不如 Windows 10， 就是因为 Windows 11 的开始菜单，其逻辑和层级与 Windows 7 并没有太大区别，只不过外观变化比较大而已。相比之下 ，Windows 10 的开始菜单直观简单，个人觉得更好用。</p><p style="text-align: left;">最后让我们看看 macOS 打开一个没有固定的应用，需要多少步：依次点击 Launchpad- 找到对应文件夹或应用-点击图标，只有三步，与 Windows 10 一致。</p>
