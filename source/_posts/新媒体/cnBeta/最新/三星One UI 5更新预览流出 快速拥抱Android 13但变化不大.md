<strong>在公测开启前，9to5Google 已经抢先放出了基于 Android 13 定制的三星 One UI 5 的预览截图。</strong>不过考虑到它并非三星公开发布的版本，所以这里并没有详细的更新日志，展示的相关样式（对比 One UI 4）也仅供参考。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0709/ee06ea554931a20.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（↑ One UI 4 / 5 通知界面对比 ↓）</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/f4e4b20941161a2.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>（1）首先是通知方面，One UI 5 相较于 One UI 4 略有些调整，辅以新图标和更新的设计风格、还有通知阴影与快速设置的透明度微调。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/988d82fc8bcdc53.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（↑ One UI 4 / 5 权限申请对话框 ↓）</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/b1e2897aa50db14.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>（2）其次，One UI 5 选用了默认的 Android 权限对话框样式，推测开发团队是为了减少不必要的改动、而加速 One UI 的更新流程，但后续也不排除进一步调整的可能。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/111ebecb241f577.jpg" alt="5.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">One UI 5 的图库 OCR 识别界面</p><p>（3）<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a>为 One UI 5 的相册引入了 OCR 光学字符识别功能，并且能够轻松地将内容复制到其它应用程序 / 输入法键盘上。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/cb08862d98dc4d8.jpg" alt="6.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（↑ iOS 16 与 One UI 5 对比 ↓）</p><p>当设备检测到照片中包含文本内容时，就会在图库底部显示一个专用的按钮，以便用户从图像中提取相关文本 —— 这也曾是 Bixby Vision 的一部分功能。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/5bd0ef0bae150d5.jpg" alt="7.jpg" referrerpolicy="no-referrer"></p><p>有趣的是，键盘 OCR 的 UI 和工作方式，都与<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>的 iOS 系统有些相似 —— 你可选择任何文本字段，并将之轻松插入其它地方。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/2380aefac09a895.jpg" alt="8.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">One UI 5 安全与隐私中心</p><p>（4）与 Google Pixel 类似，One UI 5 也迎来了 Android 13 上的新版安全与隐私中心。这里统管着用户的所有账号密码，并且能够轻松访问所有内容。</p><p>此外它能够扫描任何可能滥用应用程序权限、或未启用安全设置的内容，然后给出更好的安全性建议。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/8df0e15825626f4.jpg" alt="9.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（↑ One UI 5 的新手势 ↓）</p><p>（5）在实验室功能中，One UI 5 提供了面向多任务处理和访问的新选项。现可从屏幕底部双指向上滑动，以访问分屏视图。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/8675afd270523d2.jpg" alt="10.jpg" referrerpolicy="no-referrer"></p><p>或从右上角向内滑动，以激活浮窗模式 —— 虽然两项手势尚不能在最新流出的预览版本中工作，但三星应该会在正式推出时解锁。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/0a0093b49377aec.jpg" alt="11.jpg" referrerpolicy="no-referrer"></p><p>（6）其余变化包括可为三星笔记应用启用协作，（7）位于快速设置中的当前活动应用程序。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/582e63b92c7fd50.jpg" alt="12.jpg" referrerpolicy="no-referrer"></p><p>（8）“关于<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>”页面上显示的设备图像，以及（9）设置中新引入的超宽带（UWB）功能开关。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/e814cbc96dc4f7a.jpg" alt="13.jpg" referrerpolicy="no-referrer"></p><p>综上所述，现阶段 One UI 5 中并未引入多少新鲜事物。但通过动画等方面的小改进，三星开发团队还是努力向用户展现了快捷、清爽的积极面。</p><p><img src="https://static.cnbetacdn.com/article/2022/0709/47cfe09723e5868.jpg" alt="14.jpg" referrerpolicy="no-referrer"></p><p>近段时间，三星似乎更加注重更新的速度，那样测试版本和计划的发布时间都可能提前一个月开启。</p><p>如此一来，该公司可尽快为更多设备带来更新，并于后续再专注于新鲜功能。</p><p>最后，得益于为期四年的操作系统更新承诺，Galaxy S22 等设备用户也将能够让他们的设备长期发挥预热。</p><p>至于 Galaxy Fold Z / A 系列智能机和 Tab 平板电脑的更新支持，仍有待时间去检验。</p>
