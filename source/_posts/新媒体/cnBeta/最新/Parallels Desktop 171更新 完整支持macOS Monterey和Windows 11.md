<strong>Parallels Desktop 17.1 更新，软件现已完全支持作为主机操作系统 (OS) 的 macOS Monterey，还提升了在
Apple M1 Mac 虚拟机中运行 macOS Monterey 的用户体验。</strong>最新更新还包括对 Windows 11 虚拟机 (VM)
的支持和稳定性作出改进，在默认情况下为今后所有 Windows 11 虚拟机引入虚拟平台可信模块 (vTPM)。<br>
<p><a href="https://static.cnbetacdn.com/article/2021/10/d2b858e6b6c3ff6.jpg" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/10/d2b858e6b6c3ff6.jpg" referrerpolicy="no-referrer"></a></p><p>Parallels 工程副总裁 Elena Koryaina 表示：“Parallels Desktop 现有助于用户在其最喜爱的 Mac 设备上运行最新版本的 Windows，了解到这一关键作用后，我们开发了简单的解决方案，通过在所有 Mac 设备上默认启用虚拟平台可信模块 (vTPM) 来帮助所有用户升级到 Windows 11。Parallels Desktop 的最新版本还基于我们客户的热门要求提供了新的游戏和 3D 集成，以进一步提升用户体验。”</p><p><strong>Parallels Desktop 17 </strong><strong>更新版本</strong><strong>1 </strong><strong>中有哪些重要的新增功能？</strong></p><p><strong>通过自动启用虚拟平台可信模块</strong><strong>(vTPM)</strong><strong>，可轻松安装并更新到</strong><strong>Windows 11 </strong><strong>虚拟机！</strong></p><p>所有版本的 Parallels Desktop 17（Standard 版本、Pro 版本和 Business 版本）用户均可使用默认添加的虚拟平台可信模块 (vTPM) 来运行 Windows 11，以满足 Windows 11 的最低要求。如需了解有关在 Apple M1 或在基于 Intel 的 Mac 计算机上运行 Windows 11 所需的特定要求，<a href="https://www.parallels.com/cn/blogs/windows-11-tpm/">请参阅博客文章</a>以查看更多详细信息。</p><p><strong>在</strong><strong>Apple M1 Mac </strong><strong>上运行</strong><strong>macOS Monterey </strong><strong>虚拟机，获得更好的用户体验！</strong></p><p>现在，Parallels Desktop 17.1 用户可在 Apple M1 Mac 上的 macOS Monterey 虚拟机中安装 Parallels Tools，并在虚拟机和主 macOS 之间使用复制和粘贴的集成功能。此外，虚拟机默认磁盘大小从 32 GB 增加至 64 GB。</p><p><strong>畅玩更多</strong><strong>Windows </strong><strong>游戏！</strong></p><p>为了响应用户的热门请求，Parallels 为多款 Windows 游戏改进了图形，包括但不限于：《魔兽世界》、《帝国时代 II:决定版》、《古墓丽影 3》、《合金装备 5：幻痛》、《骑马与砍杀 2：霸主》、《坦克世界》、《木筏求生》等。</p><p><strong>通过</strong><strong>VirGL </strong><strong>在</strong><strong>Linux </strong><strong>中使用</strong><strong>3D </strong><strong>图形加速！</strong></p><p>本次更新在 Virtio GPU 中提供了对 VirGL 的支持，在所有受支持的 Mac 计算机上实现了现成可用的 Linux 3D 加速功能，提升了视觉性能，并在 Linux 虚拟机中使用了 Wayland 协议。即使未安装 Parallels Tools，现代 Linux 虚拟机仍可以使用 VirGL 3D 图形功能，但 Parallels 建议用户仍应安装 Parallels Tools。</p><div
