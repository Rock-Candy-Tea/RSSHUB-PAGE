WinRAR 刚刚收到了一个重大更新，为 Windows 用户带来了诸多变化。<strong>首先要介绍的，就是 WinRAR 6.10 已提供对 Windows 11 新版右键菜单的全面支持。</strong>开发团队解释称，微软改动了新系统中的文件资源管理器，他们也只得跟随并适应。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0125/189f8104bc0a545.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0125/189f8104bc0a545.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://news.softpedia.com/news/winrar-6-10-live-with-windows-11-context-menus-windows-xp-no-longer-supported-534742.shtml" target="_self">Softpedia</a>）</p><p>若在在综合设置选项卡下启用了“层叠右键关联菜单”，则 WinRAR 6.10 也能够在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 新版右键菜单中将之“收敛”起来，而不至于像旧式菜单那样散乱开来。</p><p><img src="https://static.cnbetacdn.com/article/2022/0125/cf46f57a6708b25.png" alt="2.png" referrerpolicy="no-referrer"></p><p>此外就算取消了“层叠右键关联菜单”，WinRAR 6.10 也只会在 Windows 11 上，默认为压缩包显示一个“提取”命令、或针对一个普通文件的“存档”命令。</p><p><img src="https://static.cnbetacdn.com/article/2022/0125/9c83ed34a527ffc.png" alt="3.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">截图（via Nerds Chalk）</p><p>在此之前，用户想要在 Windows 11 上的文件资源管理器中看到经典的右键菜单选项，必须多点一次“显示更多选项”（或使用 Shift + F10 组合键）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0125/3a3d998cc6719ef.png" alt="0.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">截图（来自：<a href="https://www.rarlab.com/rarnew.htm" target="_self">RARLab</a>）</p><p>最后，随着 WinRAR 6.10 的发布，RARLab 还放弃了为 Windows XP 继续提供支持。</p><p>作为一款发布于 2001 年的操作系统，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>官方早在 2014 年 4 月就放弃了它。</p><p>而目前可安装最新版 WinRAR 的旧版操作系统，也变成了 Windows Vista 。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/soft/1229603.htm" target="_blank">WinRAR 6.10 正式版发布</a></p></div>
