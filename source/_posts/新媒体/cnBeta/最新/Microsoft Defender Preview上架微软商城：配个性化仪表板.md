适用于 Windows 11、Windows 10、macOS、Android 和 iOS 的跨平台 Microsoft Defender 客户端在过去几年中，持续获得微软的更新。这款基于网络组件（WebView）改造后的应用程序将帮助消费者防御病毒、恶意软件、网络钓鱼、泄露密码等情况。<strong>现在<a href="https://www.microsoft.com/zh-cn/p/microsoft-defender-preview/9p6pmztm93lr?activetab=pivot:overviewtab#" target="_blank">新版预览版上架 Microsoft Store。</a></strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2022/0208/6b0b389be38ec2d.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0208/6b0b389be38ec2d.png" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0208/1ff1f825553f074.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0208/c391cf70535134c.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">在 Microsoft Store 的页面描述中，Microsoft Defender 被称之为一款让你安心的安全应用。<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>表示，新的应用程序带有一个个性化的仪表板，你可以在一个地方查看你的 Windows 设备和其他连接设备的安全状态。</p><p style="text-align: left;">根据国外科技媒体 Windows Lastest 的测试，今天早些时候更新的 Windows Defender Preview，支持使用个人微软账号进行登录。不过现阶段该应用仅限于美国地区，且大部分功能都无法使用。</p><p style="text-align: left;">有迹象表明这个新的应用程序不会取代 Windows 系统中现有的 Microsoft Defender 解决方案。与现有的 Defender 客户端不同，这个新的应用程序将只涵盖各种基本的安全威胁，它应该是作为一个仪表板，而不是一个恶意软件清除工具。</p><p style="text-align: left;">Windows Defender Preview 可以让你用任何账户登录，因此将有可能把家庭成员添加到仪表板上，并获得所有连接设备的安全概述。你将能够使用二维码邀请家庭成员，用户将被要求安装一个移动或桌面应用程序。</p>
