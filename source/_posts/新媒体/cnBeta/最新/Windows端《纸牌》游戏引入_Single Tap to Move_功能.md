经典的《纸牌》（Solitaire）不再是 Windows 系统的默认预装游戏，在剥离出来微软为用户提供了一个更现代化的版本《Microsoft Solitaire Collection》，提供了全新的游戏体验。<strong>在近日的更新中，微软将 Android 和 iOS 端的“Single Tap to Move”功能引入到 Windows 端中。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1225/da8a6f3c36da757.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1225/ca5b773a4098100.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">这项新功能不是默认启用的，但用户可以通过在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 10 或 Windows 11 的游戏选项屏幕上激活它来试试。<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>解释说：“在 Windows 上也可以使用 Single Tap to Move 了。 我们在 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> 和 Android 上的微软纸牌游戏系列中最喜欢的功能之一，现在可以在 Windows 上使用了! 启用 Single Tap to Move 功能后，您只需点击一下就可以移动任何牌! 从任何屏幕进入菜单，选择游戏选项，然后打开 Single Tap to Move。今天就试试吧！”</p>
