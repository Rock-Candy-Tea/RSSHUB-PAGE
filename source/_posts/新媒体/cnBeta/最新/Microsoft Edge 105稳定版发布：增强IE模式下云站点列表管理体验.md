Microsoft Edge 105 稳定版发布，在带来安全改进之外主要增强了 IE Mode 下的 Cloud Site List Management 体验。与 Microsoft Edge 以往版本一样，版本 105 带来了新的和更改的策略，并弃用了一些旧规则。您可以在官方文档中找到 Microsoft Edge 105 中策略更新的完整列表。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0902/24573186e6f932e.webp" alt="nrzo2vnx.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">下一个版本 Microsoft Edge 106 目前正在 Dev Channel 中进行测试。下周，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>将在 2022 年 10 月 27 日这一周将其推广到 Beta 频道，为稳定版发布做准备。</p><p style="text-align: left;">Microsoft Edge 105 稳定版主要内容</p><p style="text-align: left;"><strong>功能更新</strong></p><blockquote style="text-align: left;"><p style="text-align: left;"><strong>增强的安全模式改进。</strong></p><p style="text-align: left;">增强的安全模式现在支持 x64 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 的 WebAssembly。预计未来会有更多的跨平台支持。有关详细信息，请参阅使用 Microsoft Edge 更安全地浏览。</p><p style="text-align: left;"><strong>改进 IE Mode 下 Cloud Site List Management 体验</strong></p><p style="text-align: left;">● 您可以在 Microsoft 365 管理中心恢复到您的站点列表的最后 3 个发布版本之一。有关详细信息，请参阅恢复以前版本的站点列表。</p><p style="text-align: left;">● 您可以通过使用 InternetExplorerIntegrationCloudUserSitesReporting 和 InternetExplorerIntegrationCloudNeutralSitesReporting 策略配置站点反馈报告来识别企业站点列表中的差距。您可以在 Microsoft 365 管理中心的 Microsoft Edge 站点列表体验中查看来自用户的本地站点列表 URL 和可能配置错误的中性站点 URL。若要了解详细信息，请参阅在 Microsoft 365 管理中心查看站点反馈。</p><p style="text-align: left;">● 您可以在 Microsoft 365 管理中心的站点列表中为 IE 模式配置 Microsoft Edge 和 Internet Explorer 之间的会话 cookie 共享。要了解更多信息，请参阅 Microsoft Edge 和 Internet Explorer 之间的 Cookie 共享。</p><p style="text-align: left;">GCC 中现已提供针对 IE 模式的 Cloud Site List Management 体验的改进。 GCC 客户现在可以在 Microsoft 365 管理中心使用完整的 Microsoft Edge 站点列表体验。</p></blockquote>
