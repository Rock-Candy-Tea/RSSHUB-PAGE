<strong>微软宣布，现在 Windows Insider 项目中 Dev 频道用户也可以在 Windows 11 系统上测试运行 Android 应用了。</strong>数周前发布的 Windows SubSystem for Android（WSA）子模块仅限于 Beta 频道用户测试，但现在微软宣布向 Dev 频道用户用户。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1105/fcd1775abeb419a.png" alt="QQ截图20211105101219.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">尽管这一消息可能会让 Dev 频道的 Insider 感到兴奋，但目前 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 上的 Android 应用是相当有限的。我们把一些我们最喜欢的应用程序放在一起，但许多关键的参与者都没有，包括微软自己的应用程序，如 Minecraft，Outlook，甚至 Teams。</p><p style="text-align: left;">目前 WSA 子模块适用于 Windows Insider 项目中的 Dev 和 Beta 两个频道，如果测试稳定的话应该会近期向 Release Preview 频道用户开放，随后会向稳定版用户开放。</p><p style="text-align: left;">微软承诺，更多的应用程序很快就会出现。还有一个事实是，WSA 也不正式支持 Google Play 服务。这意味着许多流行的应用程序，如 Snapchat 和 Chrome 浏览器将无法在 Windows 11 上运行。不过已经有开发者成功破解，使其能够非官方方式运行。</p>
