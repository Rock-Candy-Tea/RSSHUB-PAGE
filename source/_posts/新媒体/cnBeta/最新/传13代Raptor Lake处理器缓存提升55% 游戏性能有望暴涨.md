<strong>近日有消息称，英特尔即将推出的 13 代 Raptor Lake 台式处理器将引入缓存设计的重大变化，以迎战 AMD 的 V-Cache 解决方案。</strong>若 @OneRaichu 的推特爆料靠谱，那我们有望在 Raptor Lake CPU 上看到更多内核和多达 55% 的缓存增长，意味着游戏性能有望迎来暴涨。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1108/300de1c57a17ad4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1108/300de1c57a17ad4.jpg" referrerpolicy="no-referrer"></a></p><p>当前<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 12 代 Alder Lake CPU 具有 14MB L2 + 30MB L3 缓存（总 44MB）。若 13 代 Raptor Lake CPU 提升 55%（不含 L1 缓存），则 L2 + L3 缓存有望达到 68MB 。</p><p>截止目前，英特尔尚未披露采用任何垂直 / 堆叠缓存解决方案（比如 Meteor Lake），但我们并不能排除该公司在未来制定类似的规划。</p><p><img src="https://static.cnbetacdn.com/article/2022/0115/68ec0b7b0e29bc4.png" alt="1.png" referrerpolicy="no-referrer"></p><p>具体说来是，英特尔为 Alder Lake CPU 的每个高性能 Golden Cove 核心（P 核），配备了 1280KB L2 + 3072KB L3 缓存。</p><p>此外一组四个 Gracemont 节能核心（E 核）具有 2048KB L2 + 3072KB L3 缓存。</p><p><img src="https://static.cnbetacdn.com/article/2022/0115/44ff11b4434bce2.png" alt="2.png" referrerpolicy="no-referrer"></p><p>目前已知 Raptor Lake CPU 会采用全新的 Raptor Cove 高性能核心 + 增强版 Gracemont 节能核心，顶级 SKU 有望采用 8P + 16E 的组合，总计 24C / 32T 。</p><p>若沿用当前的 12 代缓存设计，下一代旗舰桌面处理器会拥有 18MB L2 + 36MB L3（总计 54MB 缓存），但传闻称 13 代会进一步加大。</p><p><img src="https://static.cnbetacdn.com/article/2022/0115/f5b0045fe31cb3e.png" alt="3.png" referrerpolicy="no-referrer"></p><p>更有可能的情况时，英特尔将为每个 Raptor Cove 核心配备 2MB L2 + 3MB L3 缓存，而每组 Gracemont 集群将搭配 4MB L2 + 3MB L3 缓存。</p><p>那样跨所有内核的 L3 缓存就达到了 36MB，即 P 核 16MB（2×8）+ E 核 16MB（4×4）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0115/683bdc1a0fb601c.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0115/683bdc1a0fb601c.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图自 @Olark29_）</p><p><strong>以下是可能的 Raptor Lake / Alder Lake CPU 缓存配置对比：</strong></p><blockquote><p>● Raptor Lake P-Core L3 - 3 MB（3 x 8 = 24MB）</p><p>● Alder Lake P-Core L3 - 3 MB（3 x 8 = 24 MB）</p><p>● Raptor Lake P-Core L2 - 2 MB（2 x 8 = 16 MB）</p><p>● Alder Lake P-Core L2 - 1.25 MB（1.25 x 8 = 10 MB）</p><p>● Raptor Lake E-Core L3 - 3 MB（3 x 4 = 12 MB）</p><p>● Alder Lake E-Core L3 - 2 MB（2 x 2 = 4 MB）</p><p>● Raptor Lake E-Core L2 - 4 MB（4 x 4 = 16 MB）</p><p>● Alder Lake E-Core L2 - 3 MB（3 x 2 = 6 MB）</p><p>● Raptor Lake Total Cache（L3+L2） = 68 MB</p><p>● Alder Lake Total Cache（L3 + L2） = 44 MB</p></blockquote><p>目前 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 已凭借 64MB 三缓 和 96MB V-Cache 设计，取得了性能上的大幅增长。</p><p>想必蓝队也会考虑在进一步增加核心数量、提升时钟频率、改进 10nm ESF（Intel 7）工艺节点的基础上，考虑通过这一技术路线来增强自家 CPU 产品的竞争力。</p><p><img src="https://static.cnbetacdn.com/article/2022/0115/2ae7779d5a5f243.png" alt="5.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（截图 via <a href="https://wccftech.com/intel-13th-gen-raptor-lake-cpus-could-feature-up-to-55-more-cache-over-alder-lake-for-enhanced-gaming-performance/" target="_self">WCCFTech</a>）</p><p>参考泄露数据，新一代阵容将主要由三部分组成 —— 标称功率 125W 的 -K 系列发烧级 SKU、65W 主流 SKU、以及 35W 低功耗 SKU 。</p><p><strong>除了最高的 24C 版本，往下还有 16 / 10 / 4 / 2 核心的衍生版本来占位：</strong></p><blockquote><p>酷睿 i9-K 系列（8 Golden + 16 Grace）= 24C / 32T / 68 MB 缓存？</p><p>酷睿 i7-K 系列（8 Golden + 8 Grace）= 16C / 24T / 54MB 缓存？</p><p>酷睿 i5-K 系列（6 Golden + 8 Grace）= 14C / 20T / 44MB 缓存？</p><p>酷睿 i5-S 系列（6 Golden + 4 Grace）= 14C / 16T / 37MB 缓存？</p><p>酷睿 i3-S 系列（4 Golden + 0 Grace）= 4C / 8T / 20MB 缓存？</p><p>奔腾 S 系列（2 Golden + 0 Grace）= 4C / 4T / 10MB 缓存？</p></blockquote><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1211281.htm" target="_blank">24核/32线程：英特尔13代Raptor Lake早期CPU跑分曝光</a></p></div>
