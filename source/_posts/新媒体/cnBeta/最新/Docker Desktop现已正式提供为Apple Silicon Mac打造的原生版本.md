对于热衷于长期使用 Apple Silicon Mac 的 Docker Desktop 开发者们来说，Docker 首席技术官 Justin Cormack 刚刚宣布了一个好消息。<strong>经过持续数月的 Beta 测试，M1 Mac 原生版 Docker Desktop 已于 4 月 15 日正式上线。</strong>与此前通过 Rosetta 转译的非原生版本相比，新软件能够带来更快、更安静的使用体验，并且同样易于安装和运行。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0416/ad61f79402e2b1a.png" referrerpolicy="no-referrer"></p><p>开发者现可直接从 Apple Silicon M1 Mac 上体验原生版本的 Docker Desktop，来构建、交付和运行分布式应用程序。</p><blockquote><p>Justin Cormacn 补充道：将 Docker Desktop 推向<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>定制 Mac 芯片平台，标志着开发领域的一项更广泛的扩展。</p><p>凭借强大的跨平台开发能力，开发者可借此编写功能强大的应用程序，而无需任何新的知识，或修改他们的应用程序构建、分享与运行方式。</p><p>所有命令、本地工具、构建管道和部署自动化都可正常工作，且开发者能够充分利用 Docker Hub 上的 90000 个专为 ARM 平台打造的镜像。</p></blockquote><p>显然，Docker 致力于确保这项容器标准，能够在开发者当前和未来想要使用的所有平台上都获得无缝的体验。</p><p>感兴趣的朋友，可移步至官网下载 Docker Desktop for Mac（<a href="https://hub.docker.com/editions/community/docker-ce-desktop-mac" target="_self">传送门</a>），或者查看面向个人与团队的<a href="https://www.docker.com/pricing" target="_self">订阅服务报价</a>。</p>
