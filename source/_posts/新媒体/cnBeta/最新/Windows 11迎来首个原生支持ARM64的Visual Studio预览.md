在几周前的 Build 2022 开发者大会上，微软宣布了名为 Project Volterra、基于高通骁龙计算平台的 ARM64 设备。<strong>该项目旨在让开发者能够使用期最喜欢的工具和编程语言，以在开放式硬件生态系统中更轻松地构建和测试 ARM 原生应用程序。</strong>现在，这家雷德蒙德软件巨头又推出了首个原生支持 ARM64 的 Visual Studio 。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0615/65cd2fcb0876516.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（via <a href="https://www.neowin.net/news/the-first-native-arm64-version-of-visual-studio-is-now-available-on-windows-11/" target="_self">Neowin</a>）</p><p>在周二的一篇<a href="https://devblogs.microsoft.com/visualstudio/arm64-visual-studio/" target="_self">开发者博客</a>文章中，<a data-link="1" href="https://microsoft.pvxt.net/zaZYr" target="_blank">Visual Studio</a> 高级项目经理 Mark Downie 隆重介绍了首个原生支持 ARM64 平台的 Visual Studio 预览。</p><p>尽管初期仅支持 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11，但感兴趣的朋友已可下载显示 Visual Studio 2022 173 Preview 2，以支持如下工作：</p><blockquote><p>● 使用 C++ 进行桌面开发（基于 MSBuild 的项目）</p><p>● 同时使用 .NET Framework 和 modern .NET 进行桌面开发（WinForms、WPF）</p><p>● .NET 与 Web 开发</p></blockquote><p>如果一切顺利，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>预计原生支持 ARM64 的 Visual Studio 将于 2022 年底前，正式向所有开发者推送。</p><blockquote><p>从使用 Microsoft Visual C++（MSVC）工具集的本地开发项目开始，编译器将能够在 ARM64 设备上原生运行、而无需通过仿真转译。</p><p>此外该公司留意到许多 C++ 库（包括 Vcpkg）已原生支持 ARM64 。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0615/eaa83bbbce3f769.png" referrerpolicy="no-referrer"></p><p>值得一提的是，自发布以来，.NET6 就已原生支持 ARM64，并且致力于 .NET Framework 4.8.1 运行时和 SDK 的扩展支持。</p><blockquote><p>按照计划，.NET Framework 4.8.1 将于今年晚些时候登陆 Windows 11 22H2，且后续会覆盖较早的操作系统版本。</p><p>本次预览支持 Windows Forms、WPF 和 Web 应用程序，微软也计划将之扩展到 Windows 应用程序 SDK、.NET MAUI 和 UWP 平台。</p></blockquote><p>最后，在通过微软门户网站下载 Visual Studio 2022 17.3 Preview 2 之前，还请确保当前 Windows 11 ARM64 设备已卸载任何早期版本的 Visual Studio 。</p><p><strong>下载地址：</strong></p><blockquote><p><a href="https://visualstudio.microsoft.com/vs/preview/" _src="https://visualstudio.microsoft.com/vs/preview/" target="_blank">https://visualstudio.microsoft.com/vs/preview/</a></p></blockquote>
