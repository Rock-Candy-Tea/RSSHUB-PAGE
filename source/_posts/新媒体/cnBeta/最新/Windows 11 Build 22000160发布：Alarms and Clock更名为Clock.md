面向 Beta 和 Dev 频道的 Windows Insider 项目成员，今天微软发布了 Windows 11 Build 22000.160 预览版更新。本次更新同样将重心放在了 BUG 修复和后台性能优化上，并没有引入太多新的功能。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0820/8c86a9b8144600b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0820/8c86a9b8144600b.jpg" alt="5xqrwlp0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">新版本值得注意的改进</p><blockquote style="text-align: left;"><p style="text-align: left;"><strong>● Alarms and Clock 应用现在更名为 Clock</strong></p><p style="text-align: left;">伴随着本次更名，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还带来了包括 Focus Timer, Spotify 等新功能，并整合 To Do。此外 ，Focus Sessions 允许用户设置计时器以专注于任务，同时在应用程序中收听他们的 Spotify 播放列表。还有一个习惯养成功能，可以让用户设定目标并跟踪他们的进度。</p><p style="text-align: left;"><strong>● <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Update 的预估功能</strong></p><p style="text-align: left;">该功能可以为用户提供安装更新时重新启动所需的预估时间。该公司正在删除那些在传统硬盘驱动器 (HDD) 上运行这些构建的功能，使其成为 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 用户独有的功能。该公司确实补充说，它的目标是在消除错误后恢复该功能。</p><p style="text-align: left;"><strong>● 仍缺少 Android 应用支持</strong></p><p style="text-align: left;">这些版本中仍然缺少的是，通过内置的亚马逊 AppStore 在 Microsoft Store 上安装和下载 Android 应用程序。虽然该公司在最初的 Windows 11 公告中展示了该功能，但该功能无处可寻，即使构建现已开始推向 Beta 渠道。任何人都在猜测该功能是否会在 Windows 11 秋季发布之前及时推出。</p><p style="text-align: left;"><strong>● 发布</strong></p><p style="text-align: left;">Dev 频道发布的版本不在和某个 Windows 功能更新捆绑挂钩。但随着 Dev 和 Beta 频道的同时发布，这表明操作系统的稳定性和完成度达到一定水平。Dev 频道用户可能很快就会开始测试操作系统的某些部分，这些部分将在明年发布。</p></blockquote><p style="text-align: left;">除了更名 Clock 应用之外，今天发布的版本还带来了 1 个错误修复：修复了即使您的偏好阻止了使用，但“使用中的位置”图标有时会显示在任务栏中的问题。</p>
