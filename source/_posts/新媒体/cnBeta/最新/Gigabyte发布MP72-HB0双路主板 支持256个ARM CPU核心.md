高性能服务器于工作站行业领导者之一的 <strong>Gigabyte，刚刚推出了一款型号为 MP72-HB0 的双路主板，特点是支持多达 256 个 ARM 内核、是云原生工作负载的理想选择。</strong>与此同时，该公司还推出了 G242-P35 和 G242-P36 服务器。可搭配 Ampere Altra / Altra Max 处理器，以及高达 120 TB 的 PCIe 4.0 NVMe 存储。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0728/f60acad8a61460b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0728/f60acad8a61460b.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>这批以 GPU 为中心的服务区 / 主板平台，将很快在超大规模 / 云工作负载领域找到其归宿。</p><p>其中 Altra Max 处理器具有更可预测的高性能，因为它板载了 128 核心 / 128 线程。</p><p>辅以多插槽支持和丰富的 PCIe / CCIX 通道，使得平台具有高度可扩展性，以及业界领先的能效表现。</p><p><strong>以下是 Ampere Altra Max 的特点：</strong></p><blockquote><p>● 高性能：多达 128C / 128T</p><p>● 高吞吐量：128 条 PCIe 4.0 通道（双路配置则有 192 条可用）</p><p>● 针对 inter-socket 通信优化的 CCIX</p><p>● 高能效比：具有 250W 的热设计功耗（TDP）</p><p>● 高带宽 / 内存容量：支持 8 通道 @ DDR4-3200</p><p>● 适用于从云端到边缘的各类计算密集型应用程序</p></blockquote><p><strong>MP72-HB0 平台概述：</strong></p><blockquote><p>在这款 E-ATX 主板的加持下，Gigabyte 得以为用户提供一条在数据中心之外、开展云原生处理业务的新途径，以推进基于云的电子商务等服务的应用。</p><p>结合双路 CPU 插槽设计，该主板还能够为顶级处理能力提供支撑 —— 即来自 Ampere 的两枚 Altra Max 处理器（最高 250W TDP）。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0728/46875a19bdd90c0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0728/46875a19bdd90c0.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>至于 G242-P35 和 G242-P36，其为该系列单路 ARM 服务器提供了对 PCIe 4.0 NVMe U.2 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 的独家支持。</p><p>以前 G242 产品线仅专注于 SATA / SAS 驱动器，但 Gigabyte 还是积极听取了广大客户的反馈。</p><blockquote><p>两款产品的主要区别，在于 -P35 最高支持四槽 GPU 扩展，而 -P36 则是专为英伟达 ARM HPC 而设计的开发人员套件、最多仅支持双路 GPU 扩展。</p><p>在 Ampere Altra 平台基础上，每个插槽的 CPU 都具有 128 条 PCIe 4.0 通道，但其中有四组 PCIe 4.0 x16 被用于支持 CCIX 。</p><p>借此实现的低延迟和缓存一致性，能够为 GPU、GPFA、网络 / 存储适配器带来更多益处。</p></blockquote><p>据悉，CCIX 通过 PCIe 4.0 的传输速率提升，来实现高效的异构计算。与此同时，新一代服务器应用程序也可充分利用这一优势。</p><p>操作系统支持方面，上述 Gigabyte ARM 服务器新品均支持 Red Hat、SUSE、Ubuntu、Fedora、Oracle、Debian 等发行版。</p><p>此外作为一家专注于硬件解决方案的公司，产品研发一直是其成长决策中的一项关键因素。</p><blockquote><p>近十年来，Gigabyte 一直有为 ARM 解决方案提供内部支持，且工程师们相当熟悉、并精心设计了该生态系统。</p><p>通过极具竞争力的解决方案，Gigabyte 能够将其产品组合扩展覆盖到企业利基等更多市场。</p><p>而今日新推出的主板 / 服务器产品，更是为 Gigabyte 客户提供了其迫切需要的云原生解决方案。</p></blockquote><p>最后，Gigabyte 提供了配套的管理控制台（GMC）、以及可通过 Web 浏览器开展的 BMC 服务器管理解决方案。</p><p>感兴趣的朋友可下载 Gigabyte 服务器管理（GSM）软件，以轻松监测并管理多台服务器、且无需负担额外的许可费用。</p><p>从而在降低总体拥有成本（TCO）的同时，为客户提供巨大的可维护价值。</p>
