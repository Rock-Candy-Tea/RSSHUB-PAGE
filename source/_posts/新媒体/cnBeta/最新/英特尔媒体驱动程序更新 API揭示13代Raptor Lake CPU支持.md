<div>   
<strong>随着英特尔放出 2022 下半年的媒体驱动程序更新，我们也欣喜地在其 API 中发现了对 13 代 Raptor Lake-S 台式 / Raptor Lake-P 移动 CPU 的支持。</strong>相关代号已在 GitHub 上的三个版本中出现，除了面向台式 / 高端笔记本电脑的版本、还有面向主流 / 超薄笔记本电脑的较低功率版本。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0613/ed33c40046f37f9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0613/ed33c40046f37f9.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（via <a href="https://wccftech.com/intel-media-driver-api-support-both-13th-gen-raptor-lake-s-desktop-raptor-lake-p-mobile-cpus/" target="_self">WCCFTech</a>）</p><p>截止目前，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>对其 13 代 Raptor Lake 移动 CPU 阵容一直保持着相对沉默。除了已知的 Raptor Lake-S 桌面 / Raptor Lake-P 移动 SKU，M 系列尚未露过面。</p><p>不过 2022Q2（版本 22.4.4）更新的媒体驱动程序，揭示了它可能仍为英特尔视频加速 API 支持的一部分 —— 该 API 可用于 Intel GPU 的编解码和后处理加速。</p><p><a href="https://static.cnbetacdn.com/article/2022/0708/7af59da5e0a25fa.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0708/7af59da5e0a25fa.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">截图（来自：<a href="https://github.com/intel/media-driver/releases" target="_self">GitHub</a>）</p><p>按照计划，这家芯片巨头会在 2022 下半年推出 Raptor Lake-S 桌面芯片，但移动版 Raptor Lake-P 和 Raptor Lake-M 产品线的发布日期尚未确定（年底 / 或明年初的 CES 2023）。</p><p>规格方面，Raptor Lake-S 和 Raptor Lake-P 拥有类似的混合式核心设计（最高 16 个物理核心）—— 高性能 P 核为  Raptor Cove、节能 E 核则是 Gracemont 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0708/a503761be1c3c0f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0708/a503761be1c3c0f.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">Intel Alder Lake M / S / P 资料图</p><p><strong>以下是 13 代酷睿 CPU 的系列划分：</strong></p><blockquote><p>● 适用于台式机 / 高性能移动设备的 -S 系列，PL1 / PL2 在 55W 及以上。</p><p>● 适用于主流笔记本电脑的 -P 系列，PL1 / PL2 为 15-45W 。</p><p>● 适用于超薄笔记本电脑的 -M 系列，PL1/PL2 为 5-9W 。</p></blockquote><p>参考 12 代 Alder Lake 移动产品线的发布节奏，H55 / HX 系列来得确实晚了一些（距今才几周时间）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0708/e8318d316bb8246.png" alt="4.png" referrerpolicy="no-referrer"></p><p>此外媒体普遍猜测英特尔会推出  Raptor Lake-HX 系列 24 核台式 CPU SKU，尤其在近期有爆料称 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 会推出 16 核 Zen 4“Dragon Range”移动 CPU 的情况下。</p><p>最后，Raptor Lake-P 系列可能只有最高 22 个物理核心（6P + 16E），早些泄露的某些 SKU 更是只有 14 个核心（6P + 8E）。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1289521.htm" target="_blank">英特尔开始为14代Meteor Lake平台准备开源图形驱动程序</a></p><p><a href="https://www.cnbeta.com/articles/tech/1289729.htm" target="_blank">英特尔14代Meteor Lake移动CPU规格曝光 或于2023下半年到来</a></p></div>
