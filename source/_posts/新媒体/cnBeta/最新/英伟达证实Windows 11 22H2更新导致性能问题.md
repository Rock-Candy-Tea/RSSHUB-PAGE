早前我们报道了一些 N 卡用户在安装 Windows 11 22H2 更新后遇到的性能衰退问题，可知微软和 Reddit 等网络社区上汇集了不少 bug 反馈。除了导致某些系统的游戏帧率暴跌，还有一些人抱怨称电脑变得容易蓝屏死机（BSOD）。<strong>现在，芯片巨头英伟达的一位发言人证实，首发版本的 Windows 11 22H2 系统，或对部分 GeForce 显卡驱动造成严重的影响。</strong><br>

referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（via <a href="https://www.windowslatest.com/2022/09/24/nvidia-confirms-windows-11-22h2-is-causing-performance-issues/" 22H2 中新添加的“图形调试工具”上。两家公司正密切合作，以尽快发布驱动程序更新、并彻底修复游戏性能衰退的问题。</p><p>英伟达表示，若 Windows 11 22H2 的这项功能被意外开启，就有可能引发性能问题 —— 尤其是造成某些游戏的帧率暴跌。</p><p>对于 Windows 11 22H2 的早期采用者来说，这无疑是个让人厌烦至极的 bug
