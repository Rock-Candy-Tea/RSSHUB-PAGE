在 Windows 11 版本更迭中，大量的 UI 元素和菜单将接收来自 WinUI 2.6 和 WinUI 3 的视觉更新。在视觉上的最大调之一，就是回归了自 Windows 8 系统的圆角边缘设计，而弃用了尖锐的边缘外观。在 Windows 11 系统中，包括右键菜单、弹出窗口在内的 UI 元素、菜单和按钮都采用了圆角设计。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0807/7032529e85dff36.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0807/7032529e85dff36.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">目前，微软已经对控制面板、磁盘管理工具等传统工具和页面进行了圆角设计。不过在过渡期间仍存在一些问题，<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 中窗口框架的圆角有一些意外的白色像素/工件。因此，圆角看起来参差不齐。</p><p style="text-align: left;">值得庆幸的是，在日前发布的 Windows 11 Build 22000.120 预览版更新中微软解决了这个问题。在更新之后，圆角在深色模式下看起来更好。不过这个问题并未得到完全修复，例如 FeedBack Hub 左上方的圆角依然有些问题。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0807/4a284b6c8f8e88a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0807/4a284b6c8f8e88a.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在 Feedback Hub 上的一篇文章中，微软确认它修复了窗口边角出现白色伪影的问题。更新后，像 Steam 的聊天窗口这样的桌面应用程序现在在 Windows 11 上有更好的抗锯齿效果。</p><p style="text-align: left;">微软表示：“感谢您提交的反馈意见！我们修复了一个白色伪影的问题。我们在Build 22000.120中修复了窗口角落出现白色伪影的问题，该版本现在可以在开发频道中使用”。</p>
