<strong>本周，微软发布了适用于 <a href="https://play.google.com/store/apps/details?id=com.microsoft.scmx" target="_blank">Android</a> 和 <a href="https://www.microsoft.com/en-us/p/microsoft-defender-preview/9p6pmztm93lr" target="_blank">Windows</a> 的 Microsoft Defender Preview 应用程序。</strong>现阶段该应用仅在美国地区可用，但预估伴随着测试规模的扩大，未来会向更多国家和地区开放。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2022/0208/1ff1f825553f074.webp" target="_blank"><img src="https://static.cnbetacdn.com/article/2022/0208/1ff1f825553f074.webp" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Microsoft Defender 是<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>的反恶意软件工具，微软在 Android 平台上推出了 Microsoft Defender ATP，不过仅限于企业用户。而最新推出的 Microsoft Defender Preview 不仅可以管理当前设备的安全，还可以管理连接到同一微软账户的所有设备的安全。在主页上，你可以看到你所有设备上的保护状态摘要，如下图所示。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2022/0208/c391cf70535134c.webp" target="_blank"><img src="https://static.cnbetacdn.com/article/2022/0208/c391cf70535134c.webp" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">你可以点击或点选该面板，以看到关于每台设备状态的更多信息。在这里，你可以看到过去 24 小时内扫描的应用程序和链接（在Android上）或文件（在<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>上）的数量。在Windows上，你还可以看到你当前的设置，如病毒和威胁保护、勒索软件保护等等。如果你想检查过去的威胁，你还可以查看你的安全历史。</p><p style="text-align: left;">对于 Windows 用户来说，Microsoft Defender Preview 并非是 Windows Security 的替代者，而是后者的一个补充。在 Android 系统上，杀毒软件的作用值得怀疑，但它可以告诉你是否安装了一个潜在的恶意软件。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0209/a8619b6f86724f1.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0209/f95a5ff3a7603b3.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0209/155f7897ac2027d.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">你可能从中得到的最有用的功能实际上是能够在同一个地方看到你所有设备的安全状态。如果你与别人共享你的电脑，这尤其有用，所以你可以在你的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>上看到其他用户是否可能安装或下载了恶意软件。</p>
