微软今天放出了 Windows Server 最新预览版 Build 25169，其官网已经放出了 ISO 和 VHDX 镜像。和以往相同，虽然采用了 Windows 11 Dev 版本开发相同的内核，但是微软官方并未公布更新日志，因此不清楚具体在哪些方面有所更新，也不清楚存在哪些已知问题。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0729/2ed7b4f14f64f73.webp" alt="vdzbrv5l.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">在新预览版中 Windows Server 的品牌仍未发生变更，依然显示为“Windows Server 2022”。不过<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>总是将这些版本称之为 vNext，而不是 Windows Server 2022。</p><p style="text-align: left;">和以往相同，在 <a href="https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewserver" target="_blank">Windows Server Insider</a> 页面，微软提供了如下下载链接：</p><blockquote style="text-align: left;"><p style="text-align: left;">Windows Server VNext Preview 的 ISO 格式支持 18 种语言</p><p style="text-align: left;">VHDX 格式仅支持英语</p><p style="text-align: left;">Windows Server VNext Preview VHDX</p><p style="text-align: left;">Microsoft Server Languages and Optional Features Preview</p><p style="text-align: left;">Windows Admin Center 2110.2 Preview</p><p style="text-align: left;"><strong>这些密钥仅限于预览版</strong></p><p style="text-align: left;">Server Standard: MFY9F-XBN2F-TYFMP-CCV49-RMYVH</p><p style="text-align: left;">Datacenter: 2KNJJ-33Y9H-2GXGX-KMQWH-G6H67</p></blockquote>
