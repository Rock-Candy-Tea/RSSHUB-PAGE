今天，微软发布了 Microsoft Edge Beta Build 96.0.1054.8。Beta 频道被认为是最稳定的预览版本，只接受 Dev 和 Canary 频道版本中测试良好的功能。上次 Beta 版本于 9 月 28 日发布，此后有更多的小错误修复。<br>
 <p style="text-align: center;"><a href="https://static.cnbetacdn.com/thumb/article/2021/0805/c1b1a0aeba4bbfc.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0805/c1b1a0aeba4bbfc.jpg" referrerpolicy="no-referrer"></a></p><p>作为 Beta 频道升级到 Chromium 96 的首个版本，主要带来了以下新功能和改进</p><p><strong>● 直接通过协议链接启用 PWA 应用</strong></p><p>让已安装的PWA处理使用特定协议的链接，以获得更综合的体验。</p><p><strong>● 通过 Math Solver 帮助解决数学问题</strong></p><p>用户可以在 Microsoft Edge 中使用 Math Solver 来获得广泛的数学概念的帮助。这些概念包括从初级算术和二次方程到三角学和微积分。</p><p>Math Solver可以让你拍下手写或打印的数学问题，然后提供一个即时解决方案，并提供分步说明，帮助你学习如何在没有帮助的情况下达到解决方案。</p><p>Math Solver还配有一个数学键盘，你可以用它来轻松输入数学问题。这个键盘消除了在传统键盘上寻找你需要的数学字符的需要。在解决了你的问题后，Math Solver 提供了继续学习的选项，包括测验、工作表和视频<a data-link="1" href="https://c.duomai.com/track.php?k=iRyUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmUmchdHdm92cGJTJjZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyU" target="_blank">教程</a>。</p><p><strong>● PDF 文档的滚动改进</strong></p><p>微软正在改进滚动性能。以便在 PDF 文档中提供更顺畅的滚动体验。在滚动过程中，你将不会看到白条出现。</p><p><strong>● 在 PDF 文件上自由突显</strong></p><p>由于增加了自由格式的高亮标记，PDF 的浏览和标记体验得到了改善。你可以突出显示你无法访问的 PDF 文件的部分，以及扫描的文件。</p><p><strong>● 控制流执行技术（CET）</strong></p><p>微软 Edge 将开始支持一种更安全的浏览模式，对浏览器进程使用依赖硬件的控制流。在此支持的硬件上提供控制流：<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 11 代或 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> Zen 3。你可以通过使用组策略配置图像文件执行选项（IFEO）来禁用CET。</p><p><strong>● 新的警告对话框，用于排版网站。</strong></p><p>浏览器现在会对一些 URL 看起来与其他网站非常相似的网站显示一个警告。这个用户界面使用客户端启发式方法来警告用户那些可能是欺骗流行网站的网站。欲了解更多信息，请参见<a href="https://support.microsoft.com/topic/what-is-typosquatting-54a18872-8459-4d47-b3e3-d84d9a362eb0" target="_blank">什么是 tynosquatting？</a></p><p><strong>●  改进了 IE 模式和现代浏览器之间的交接。</strong></p><p>从这个版本的 Microsoft Edge 开始，Microsoft Edge 和 Internet Explorer 模式之间的导航将包括表单数据和额外的 HTTP 头。推荐人头信息、帖子数据、表单数据和请求方法将在两种体验之间正确转发。你可以使用 <a href="https://docs.microsoft.com/en-us/deployedge/microsoft-edge-policies#internetexplorerintegrationcomplexnavdatatypes" target="_blank">InternetExplorerIntegrationComplexNavDataTypes</a> 策略指定应该包括哪些数据类型。</p><p><strong>● 公开预览版中IE模式的云网站列表管理</strong></p><p>云网站列表管理让你在云中管理你的IE模式的网站列表，而不需要一个内部的基础设施来托管你的组织的网站列表。你可以使用Microsoft 365管理中心的Microsoft Edge Site Lists体验访问云网站列表管理功能。要了解更多信息，请参阅《<a href="https://docs.microsoft.com/en-us/deployedge/edge-ie-mode-cloud-site-list-mgmt" target="_blank">IE模式的云网站列表管理</a>》（公开预览版）一文。</p><p><strong>●  使用WSUS更新Microsoft Edge WebWiew2</strong></p><p>使用 WSUS 更新 Microsoft Edge 的 IT 管理员也将能够使用 WSUS 更新 Microsoft Edge WebView2。这项功能为管理员提供了一个更容易的离线设备的服务过程。</p><p><strong>● 服务器的 WSUS 更新</strong></p><p>Microsoft Edge 渠道（稳定版、测试版、开发版）的WSUS和目录更新现在将适用于安装了Microsoft Edge的<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Server SKU，包括Windows Server 2022。有关如何为Microsoft Edge配置WSUS更新的更多信息，请参阅<a href="https://docs.microsoft.com/en-us/mem/configmgr/apps/deploy-use/deploy-edge?bc=https://docs.microsoft.com/DeployEdge/breadcrumb/toc.json&toc=https://docs.microsoft.com/DeployEdge/toc.json#update-microsoft-edge" target="_blank">更新Microsoft Edge</a>。</p><p><strong>新的策略：</strong></p><p>● <a href="https://docs.microsoft.com/en-us/DeployEdge/microsoft-edge-policies#applicationguarduploadblockingenabled"
