<strong>Windows Central 报道称，在使用微软搜索（Microsoft Search）时，用户也将很快能够获得这家软件巨头的积分奖励。</strong>具体说来是，这项福利可在通过工作 / 学校账户使用某些 Microsoft 365 服务时起到作用。如果消息靠谱的话，微软最快有望于 2021 年 5 月份开始实施这项新政策。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0410/49c39bbcf4fa0b1.jpg" alt="Microsoft 365 Roadmap.jpg" referrerpolicy="no-referrer"></p><p>早些时候，Neowin 在<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>的 Microsoft 365 <a href="https://www.microsoft.com/en-in/microsoft-365/roadmap" target="_self">路线图</a>中发现了这个不起眼的变化。可知在使用学校或工作账号登录特定的 Microsoft 365 服务后，即可获取相应的积分。</p><blockquote><p>（1）使用工作或学校账户开展 Microsoft Search 搜索，即可获取积分；</p><p>（2）使用组织机构账号登录特定的 Microsoft 365 服务（比如必应 / 微软搜索），亦可获取积分；</p><p>（3）相关 Microsoft Rewards 积分可兑换 Microsoft Store 商店积分或其它一些项目。</p></blockquote><p>Microsoft Search 是一款允许用户轻松搜索组织内各种信息的实用工具，且能够与 Web 端的必应搜索配合使用。</p><p>至于人们可以通过上述操作获取多少积分，以及是否能够轻松地将积分转移到个人账户，目前暂不得而知。</p><p>其实早在去年，<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Latest 的 Richard Devine 就已经撰写过“微软积分可能是你没有使用过的最佳 <a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> 功能”的文章。</p><p>虽然现在微软也允许将奖励积分转移给他人，但还是存在着一些明确的限制，具体取决于您作为积分赚取者的当前账户身份。</p>
