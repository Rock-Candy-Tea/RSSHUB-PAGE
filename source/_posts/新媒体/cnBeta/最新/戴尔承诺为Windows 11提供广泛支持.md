自 7 月以来，已经有不少人体验过 Windows 11 的预览版本。之后的多次更新，则主要涉及质量方面的较小改进。虽然有各种已知问题，但它们对操作系统的稳定性影响并不大。<strong>早些时候，微软证实 Windows 11 已处于测试的最后阶段，并且有望于 10 月 5 日开始向企业和消费者推送。</strong>与此同时，包括戴尔在内的 OEM 厂商，也在网站上确认了后续新机将搭载新系统。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0901/c328b8d0c103303.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a data-link="1" href="http://www.anrdoezrs.net/links/9019719/type/dlg/sid//https://www.dell.com/zh-cn/shop/deals" target="_blank">DELL</a> <a href="https://www.dell.com/support/kbdoc/en-in/000187485/dell-computers-tested-for-upgrade-to-windows-11" target="_self">官网</a>）</p><p>在今日的新闻稿中，<a data-link="1" href="http://www.anrdoezrs.net/links/9019719/type/dlg/sid//https://www.dell.com/zh-cn/shop/deals" target="_blank">戴尔</a>表示通过官网购买的符合条件的 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> PC，将可免费升级到 Windows 11，其中包括了 XPS 13、Alienware x15 / x17 等笔记本电脑产品线。</p><p>对于老用户，戴尔也正在开发相关工具，以帮助其通过免费的升级过程来获取 Windows 11，且企业客户能够选购预装 Windows 10 专业版、并包含 Windows 11 专业版许可证的 PC 。</p><p><strong>以下是戴尔本次宣布的支持 Windows 11 操作系统的设备产品线：</strong></p><blockquote><p>● Alienware 台式机 / 笔记本</p><p>● G 系列台式机 / 笔记本</p><p>● Inspiron 台式机 / 笔记本</p><p>● XPS 台式机 / 笔记本</p><p>● Vostro 台式机 / 笔记本</p><p>● Latitude / OptiPlex 产品线</p><p>● Precision 产品线（含移动工作站）</p></blockquote><p>只要你的设备在官网列出的详细型号清单中，即可免费获得便捷的 Windows 11 操作系统更新。截止 8 月 31 日，网站上已经列出了 100+ 设备。</p><p>不过在升级之前，还请留意您的设备是否满足运行 Windows 11 的最低硬件需求。根据<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>官方的说法，新系统需要<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 8 代、或 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 2000 系列及以上的处理器、且支持 TPM 2.0 可信任安全模块。</p><p>如不确定设备是否符合条件，还请参考微软提供的“PC Health Check”兼容性检查工具。</p>
