<div>   
开发跨平台应用的方法有很多，而微软主推的一个框架就是 React Native。事实上，微软还对 React Native for Windows （React Native UWP）框架衍生版进行了维护，特别针对 Windows 10 的特殊要求进行了优化，提高了运行性能。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0419/ce0d4a930ba6e7c.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0419/ce0d4a930ba6e7c.jpg" alt="gphotri0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">例如全新的 <a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> 应用就是用 React Native 编写的，与旧版的 Electron 相比，它的性能得到了极大提升。现在，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>已经在 Microsoft Store 发布了 React Native Gallery 应用。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0419/5a15e7205c44fdc.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0419/5a15e7205c44fdc.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0419/9cb7ac1f5054f44.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0419/9cb7ac1f5054f44.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0419/054d8ea17777b53.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0419/054d8ea17777b53.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">React Native Gallery 是一个 React Native <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 应用，它作为支持 Windows 的 React Native 组件样本，并让开发者深入了解该框架。该应用在原理上与几周前微软在商店发布的 WinUI 3 Controls Gallery 应用非常相似。</p><p style="text-align: left;">开发者可以通过访问<a href="https://github.com/microsoft/react-native-windows" target="_blank"> GitHu</a>b 了解更多关于使用 React Native Windows 开发的信息。React Native Gallery 的源代码也已经出现在<a href="https://github.com/microsoft/react-native-gallery" target="_blank"> Github</a> 上了。</p>
