目前 AMD 在桌面平台上，只有一款基于 Navi 22 核心的产品，就是 Radeon RX 6700 XT。 不过在移动平台上 ，AMD 推出了两款基于 Navi 22 核心的产品，分别是 Radeon RX 6800M（ 完整核心有 40 个 CU） 和 Radeon RX 6700M（36 个 CU）。<br>
 <p style="text-align: left;">虽然有关 Radeon RX 6700 的消息已经传了几个月了，不过自今年 3 月份 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 发布 Radeon RX 6700 XT 以后，似乎变得越来越遥远，现在基本没有关于 Radeon RX 6700 的消息了 。Radeon RX 6700 到底去哪里了？除了我们，抱有疑问的还有<a href="https://www.tomshardware.com/news/where-is-the-radeon-rx-6700-non-xt#xenforo-comments-3714645" target="_blank">TomsHardware</a>的编辑。此前曾有传言 ，Radeon RX 6700 只会配置 6GB 显存，无论相比自家 Radeon RX 6600 XT 的 8GB， 还是竞争对手英伟达 GeForce RTX 3060/RTX 3060 Ti 的 12GB 或 8GB，6GB 的容量显然太小了。</p><p style="text-align:center"><a target="_blank" href="https://static.cnbetacdn.com/article/2021/0722/09f18dc7116f1ee.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/0722/09f18dc7116f1ee.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/0722/09f18dc7116f1ee.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">还有传言指 ，Radeon RX 6600 XT 也采用 Navi 22 核心，不过随着几个月来越来越多的信息泄露，都将其指向采用 Navi 23 核心。以 Radeon RX 6700 XT 的 40 个 CU 和 192bit/12GB GDDR6 显存，与 Radeon RX 6600 XT 的 32 个 CU 和 128bit/8GB GDDR6 显存，两者之间的规格差异和可能的性能差距，在中间加入配置 36 个 CU 的 Radeon RX 6700 是一个合理的选择，配备 192bit/12GB GDDR6 或 160bit/10GB GDDR6 显存即可。</p><p style="text-align: left;">也有消息指，受制于芯片供应短缺和产能分配问题，才导致 Radeon RX 6700 迟迟没有露面。传闻 Radeon RX 6600 XT 的官方建议零售价为 399 美元，似乎价格有些许偏高，这更应该是 Radeon RX 6700 所要占据的位置 ，Radeon RX 6600 XT 应该以更低的价格与英伟达的产品竞争。</p><p style="text-align: left;">由于 Radeon RX 6600 XT 尚未推出，没办法确认其具体性能，另外 AMD 也有自己的计划和考虑。不过以 Radeon RX 6600 XT 可能的定价，当产能提升以后，不知道 AMD 是否还会在桌面平台发布 Radeon RX 6700。</p>
