<strong>为确保在 Steam Deck Linux 掌机上运行的热门游戏不受外挂影响，Valve 已承诺同反作弊软件制造商 EAC 和 BattlEye 达成合作。</strong>今天，Epic Games 宣布旗下 Easy Anti-Cheat 正式支持 Linux 和 Mac，并且专门为 Valve 依赖的 Proton 和 Wine 兼容层提供了适配优化，以便将 Windows 游戏带到 Steam Deck 上。<br>
<p><img src="https://static.cnbetacdn.com/article/2021/0924/ce833ca884f433d.jpg" referrerpolicy="no-referrer"></p><p>尽管对于开发者来说，还是有不少游戏移植工作要做，但这至少意味着 Steam 上的某些<a href="https://steamdb.info/graph/" target="_self">热门游戏</a>，现已从理论上变得触手可及 —— 包括当前排行前 25 的《Apex 英雄》、《黎明杀机》、《战争雷霆》等。</p><p>其它热门 <a href="https://www.easy.ac/en-us/partners/" target="_self">EAC</a> 游戏还有《七日杀》、《糖豆人终极淘汰赛》、《黑色沙漠》、《猎杀：对决》、《帕拉丁》、以及《光环：士官长合集》。</p><p>当然，想要 Steam Deck 遍地开花，仍少不了 <a href="https://www.battleye.com/" target="_self">BattleEye</a> 等反作弊软件提供商的加盟，否则《PUBG》、《命运 2》、《彩虹六号：围攻》等作品将遗憾缺席。</p><p><img src="https://static.cnbetacdn.com/article/2021/0924/d90bfc4b42e64b3.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">EAC 于今年早些时候成为了一项免费服务</p><p>Epic 自家的《堡垒之夜》也是如此，但该公司尚未将该游戏或 Epic 游戏商店带到 Linux 桌面平台。且截止发稿时，Epic 也未立即回应外媒的置评请求。</p><p>即便如此，Epic 还是建议开发者“轻松修补”他们的 EAC 游戏：“在新版 SDK 的加持下，开发者只需在 Epic 开发者门户网站上轻点几下，即可添加对 Wine / <a href="https://www.protondb.com/" target="_self">Proton</a> 的 Linux 反作弊方案的支持”。</p>
