<strong>回顾视窗操作系统的历史，我们发现微软似乎延续着“一代拉胯 一代强”的发展路线图。</strong>比如 Windows Vista 引入的 Aero Glass 效果太过超前，占据天时地利人和的 Windows 7 则广受好评。在 Windows 8.x 时代，微软试图向元素丰富的“开始屏幕”和“动态瓷贴”强势转型，结果在 Windows 10 时代又朴素了起来。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0214/181aa8130a498b3.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://www.windowslatest.com/2022/02/14/microsoft-teases-windows-7-aero-like-design-for-windows-11/" target="_self">Windows Latest</a>）</p><p>作为 Windows 8.x 的继任者，Windows 10 选择了“扁平化”的 UI 风格，放弃了对圆角窗口和 Aero Glass 的支持。</p><p>然而历史是个圈，在 Windows 11 身上，我们又一次被鲜艳的 UI 设计所吸引。</p><p>近日有爆料称，下一版主要更新将引入类似 Windows 7 Aero 效果的新体验，经典 Windows 应用程序窗口上的透明标题栏将卷土重来。</p><p><img src="https://static.cnbetacdn.com/article/2022/0214/3ad8bcaf5fa4f5c.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p>据说<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>希望用一种新的透明效果，来刷新前台应用程序和弹出窗口（亚克力或新材质）。</p><p>目前该公司将云母（Mica）透明效果的实现权利，交到了开发者的手中，行业能够迅速向全新的 UI 体验转型。</p><p>如上图所示，微软在 1 月份的 Windows Developer 网络直播旗舰，意外分享了崭新的“运行”小窗。</p><p><img src="https://static.cnbetacdn.com/article/2022/0214/8016ff486391884.png" alt="1.png" referrerpolicy="no-referrer"></p><p>回顾 Windows 7 / Vista 时代，Aero Glass 主题可为应用程序窗口的标题栏（包括最小化 / 最大化 / 关闭按钮）提供了柔和与半透明的外观质感。</p><p>在硬件性能相对受限的 Windows Vista 时期，Aero Glass 就已经让人眼前一亮。只是为了节省系统资源，它未能顺利讨到广大用户的欢心。</p><p>不过随着 Windows 11“Sun Vallery 2”（Version 22H2）的立项，该公司似乎再次鼓起了勇气。</p><p><img src="https://static.cnbetacdn.com/article/2022/0214/12fb9d13ee728b1.png" alt="2.png" referrerpolicy="no-referrer"></p><p>有人推测，这家科技巨头或为所有经典应用程序的标题栏套上透明 / 半透明的亚克力（Acrylic）效果，同时保留对云母（Mica）材质的支持。</p><p>Aero Glass 可让用户透过前台程序看到桌面壁纸、或其它后台窗口，但 Mica 是一种适用于应用程序窗口、标题栏、甚至背景的不透明 UI 。</p><p>最后，我们在系统中发现了一个新的实验标记，暗示微软将把 Mica 效果带给更多的 Windows 应用程序。</p><p>此外，我们在操作系统中发现了一个新的实验标志（MicaBackdropInApplicationFrameHostTitlebar），它可以将云母效应带入更多 Windows 应用程序。</p>
