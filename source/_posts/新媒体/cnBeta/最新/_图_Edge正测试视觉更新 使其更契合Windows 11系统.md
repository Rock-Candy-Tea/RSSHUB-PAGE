伴随着 Windows 11 系统的到来，微软也在积极地改进基于 Chromium 的新版 Edge 浏览器。在最新发布的 Canary 频道版本中，微软正在测试一个名为“启用 Windows 11 视觉更新”的实验 Flag，有望改善 Edge 在新系统上的浏览体验。<br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/4eb90d09ec33021.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/4eb90d09ec33021.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在该实验 Flag 的描述中写道：“启用适用于当前已安装 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 系统正在进行的视觉效果”。这意味着 Edge 浏览器在外观上将会更加契合 Windows 11 系统的体验，在外媒测试之后发现现阶段只是对某些菜单和浏览器窗口进行了优化。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/6bd0c2bbe921eea.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/6bd0c2bbe921eea.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/1de3f957d029504.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/1de3f957d029504.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">前后对比</p><p style="text-align: left;">实验 Flag 似乎促进了一种不那么杂乱的感觉，我们以前在公司网站上分享的 Windows 11 设置应用程序、Microsoft Teams 和 Outlook 模拟图中看到过这种窗口设计。在 Build 2021 开发者大会上，这种<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>的Edge设计也被预告过。</p><p style="text-align: left;">此外，微软还试图通过更新默认字体样式和大小，使Windows 11上Edge的菜单更加明亮。值得注意的是，该公司仍在为微软 Edge 进行 Windows 11 视觉刷新，更多改进将在即将到来的构建中到来。</p>
