<strong>美满电子（Marvell）刚刚发布了业内首款基于 5nm 先进制程的 1.6T 以太网 PHY 解决方案，特点是支持十万兆（100G）级别的 PAM4 输入 / 输出。</strong>对于数据中心来说，100G 串行 I/O 可在云基础架构中发挥关键作用，为其不断增长的海量数据提供支撑。除了以更节能的方式来跨计算 / 网络 / 存储移动数据，新方案还可推动 Ethernet 骨干网向 1.6 Tbps 过渡。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0608/c0ed406c706b908.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">Marvell Alaska 系列电口以太网 PHY 性能不逊于光口</p><p><a href="https://www.techpowerup.com/283105/marvell-launches-industrys-first-1-6t-ethernet-phy-with-100g-pam4-i-os-in-5nm" target="_self">TechPowerUp</a> 指出，Marvell Alaska C PHY 旨在加速向 100G 串行互连过渡，并将上一代 PHY 的带宽速率翻番，从而为人工智能（AI）/ 机器学习（ML）等性能关键型云工作负载和应用程序带来更高的可扩展性。</p><p>具体说来，Marvell 的 1.6T 以太网 PHY 解决方案（基于 Alaska 88X93160 芯片平台）支持用于高密度交换机的下一代 100G 串行、以及 400G 和 800G 以太网链路。</p><p>不过速率的加倍，也对信号完整性提出了新的挑战，并且推动了高端口数交换机设计对冲定时器设备的需求。因为面向此类应用的重定时器和调速装置，都必须具有极高的效能。</p><p>得益于先进的 5nm 制程工艺，Marvell 800GbE PHY 较当前基于 50G PAM4 的 I/O 方案功耗降低了 40% 。</p><blockquote><p>650 Group 创始人兼技术分析师 Alan Weckel 表示：为下一代高速网络奠定基础的 100G 串行电信号，其重要性已经无需赘述。</p><p>随着 I/O 速率的进一步提升，也往往面临着信号完整性方面的挑战。</p><p>当行业过渡到基于光学器件的（100G 串行）高密度交换机时，Marvell 的 1.6T PHY 就成为了当今市场上唯一支持这种过渡的解决方案。</p></blockquote><p>言归正传，Marvell 的 88X93160 是业内首款完全符合 IEEE 802.3ck 100G 串行 I/O 标准和以太网技术联盟 800GbE 规范的 PHY 设备。</p><p>该设备支持 Gearboxing 功能，可帮助数据中心运营商获得具有 100G 串行 I/O 的交换机 ASIC 的全部带宽功能，同时与现有基于 50G PAM4 的 400G 光模块连接保持兼容。</p><p>随着新品的推出，Marvell 也进一步拓展了自身在高速重定时器及调速领域的领先地位，并且提供了从 10GbE 到 800GbE 的广泛产品组合，辅以支持 MACsec 加密 / 符合 C 类标准的 IEEE1588 PTP 时间戳。</p><p>目前 Marvell 已开始向部分客户提供 1.6T 以太网 PHY 样品。</p>
