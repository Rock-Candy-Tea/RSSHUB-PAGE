虽然 Windows Subsystem for Linux 允许管理员直接从 Windows 系统中访问各种 Linux 工具，但 Microsoft Store 上并没有上架所有的 Linux 发行版本。<strong>现在通过名为 EasyWSL 的新工具（<a href="https://github.com/redcode-labs/easyWSL" target="_blank">GitHub</a>），用户可以将几乎所有 Linux Docker 镜像转换为支持 WSL 的发行版本。</strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0816/237328b2c41c4a4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0816/237328b2c41c4a4.jpg" alt="z8q9kqet.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">EasyWSL 工具由网络安全公司 Red Code Labs 创建。该公司解释道：“我觉得我们可以通过 WSL 以某种方式将更多的 Linux 灵魂带入 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>， 这比<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>在他们的商店中提供的更多”。</p><p style="text-align: left;">EasyWSL 在从 Microsoft Store 获取系统镜像方面也具有一些优势，因为这些图像通常由第三方提供，因此可能已被更改。公司表示“基本上，一开始尝试使用 Arch 和 Gentoo 及其存储库是一个好主意，但后来我意识到，为什么不以某种方式从 Docker Hub 获取镜像，以扩大可能性”。</p><p style="text-align: left;">WSL 用户现在可以直接从原始维护者那里下载 Docker 镜像，并且知道构建没有以任何方式改变。EasyWSL 是一个开源工具。在此处的 GitHub 上找到它以及如何使用它的说明。</p>
