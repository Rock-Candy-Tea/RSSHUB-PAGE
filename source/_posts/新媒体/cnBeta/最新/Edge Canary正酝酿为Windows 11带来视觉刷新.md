Neowin 报道称：<strong>微软已经为 Edge Canary 添加了一个新的标记（flag），以允许用户手动启用符合 Windows 11 设计理念的视觉更新。</strong>Reddit 网友 Leopeva64 指出，最新版 Edge Canary 浏览器，已能够通过 edge://flags 设置页面，在 Windows 10 和 Windows 11 操作系统上启用新版 UI 。<br>
