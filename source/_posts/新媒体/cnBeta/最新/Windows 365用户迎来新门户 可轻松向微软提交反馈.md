为了改进产品反馈与功能请求流程，微软已宣布将 Windows 365 整合到旗下最新的反馈门户中。<strong>除了 Microsoft 365 和 Windows 客户，现 Windows 365 用户也能够借此地向这家软件巨头轻松提交相关反馈。</strong>去年，微软已经推出过新版反馈门户，以取代之前体验欠佳的 UserVoice 论坛。<br><br>
<p><a href="https://static.cnbetacdn.com/article/2022/0415/d66d27b062c0431.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0415/d66d27b062c0431.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：<a href="https://techcommunity.microsoft.com/t5/windows-it-pro-blog/how-to-provide-feedback-on-windows-365/ba-p/3285759" target="_self">Tech Community</a>）</p><p>自推出以来，它就一直处于预览状态。而基于 Dynamics 365 的反馈门户，则是<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>主推的一个“综合性目的地”（consolidated destination），允许用户在这里就一系列微软产品提交反馈。</p><p><a href="https://static.cnbetacdn.com/article/2022/0415/9c334acdcd535f5.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0415/9c334acdcd535f5.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>现在，<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 365 也被无缝地融入了这里。以前必须通过 Windows 技术中心提交的 Windows 365 请求 / 反馈，现正迁移至新的综合性门户。</p><p><a href="https://static.cnbetacdn.com/article/2022/0415/636db3ae2d3a02e.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0415/636db3ae2d3a02e.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>与此同时，之前 Windows 技术社区中的所有功能请求，现都被设置成了只读的状态。到 4 月 29 号，之前的所有反馈项目都将在新门户上显示。</p><p><a href="https://static.cnbetacdn.com/article/2022/0415/1c82327a8da0dbe.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0415/1c82327a8da0dbe.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>如果过了这个截止日期，你还是没能看到自己提交的反馈，还请在新门户上重新操作一遍。感兴趣的朋友，可戳此手动跳转到 <a href="http://feedbackportal.microsoft.com/feedback" target="_self">Microsoft Feedback Portal</a>、或直接点击<a href="https://feedbackportal.microsoft.com/feedback/forum/6ab53114-95ab-ec11-9840-000d3a1c7d74" target="_self">该链接</a>。</p><p><img src="https://static.cnbetacdn.com/article/2022/0415/28c8097558a9a1f.png" alt="4.png" referrerpolicy="no-referrer"></p><p>需要注意的是，你必须先通过 Microsoft 账户登录，才能选择页面右上角的“发送反馈”按钮、并填写所有必要的字段，以向微软提交相关反馈。</p>
