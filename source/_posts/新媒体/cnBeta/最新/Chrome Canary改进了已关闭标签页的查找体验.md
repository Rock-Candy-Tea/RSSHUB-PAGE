最为目前市面上最受欢迎的 Web 浏览器，Google Chrome 的优点相当多。不过最重要的一个原因，就是它提供了许多竞争对手所没有的轻松体验。<strong>近日有消息称，谷歌正在 Chrome Canary 上测试一项全新的功能，以便用户能够更轻松地重新打开被意外关闭的标签页。</strong><br>
<p><img src="https://static.cnbetacdn.com/article/2021/0415/54e8ae07922e283.png" referrerpolicy="no-referrer"></p><p>作为 Chrome 浏览器的第一试验场，并不是所有在 Canary 分支上出现过的功能，最终都会走向稳定版通道。</p><p>不过早些时候的“标签页搜索”按钮，还是已经出现在了窗口顶部“最小化”按钮的左侧。通过该按钮，你可以轻松查看、搜索、并导航至所有已打开的标签页。</p><p>有趣的是，在最新的 Canary 分支中，我们发现该菜单还显示了用户最近关闭的标签页列表。虽然许多人不见得用得着这项功能，但允许快速滚动浏览的体验还是相当不错的。</p><p>对于键盘党们来说，依然可以通过 Ctrl + Shift + T 组合键来依次恢复刚被关闭的标签页（Mac 上为 Command + Shift + T 组合键）。但是想要查找更早以前的已关闭标签页，相关操作就没那么便利了。</p><p>感兴趣的朋友，可到谷歌官网下载 91.0.4472.0 版本的 <a href="https://www.google.com/chrome/canary/" target="_self">Chrome Canary</a> 进行体验。</p>
