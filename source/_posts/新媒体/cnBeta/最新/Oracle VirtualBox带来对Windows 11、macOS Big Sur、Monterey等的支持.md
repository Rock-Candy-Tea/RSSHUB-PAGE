<strong>甲骨文公司的VirtualBox 7.0.0 Beta 1版本在主机操作系统支持方面带来了几个重大变化。该虚拟化软件终于获得了将Windows
11作为主机操作系统运行的能力。</strong>上一个VirtualBox版本，即6.1.36版不支持Windows
11主机，尽管从6.1.28版开始就支持Windows 11作为客户机，然而，计划中的TPM直通在6.1.28中也没有出现。<br>
 <p>新版本的用户手册(<a href="https://download.virtualbox.org/virtualbox/7.0.0_BETA1/UserManual.pdf">PDF</a>)中提到了这一变化，其中指出新版支持Windows 11 21H2。</p><p>除了Windows 11，新的VirtualBox 7.0 Beta 1还获得了对macOS 11（Big Sur）和macOS 12（Monterey）的支持。在Linux方面，现在支持Ubuntu 20.04 LTS和22.04 LTS，以及Debian 11。</p><p><img src="https://static.cnbetacdn.com/article/2022/0828/974fb95fd2da908.jpg" title alt="1545346423_virtualbox6.jpg" referrerpolicy="no-referrer"></p><p>下面是新的主机操作系统支持方面的完整变化记录：</p><p><strong>支持的主机操作系统</strong></p><p>目前，Oracle VM VirtualBox可以在以下主机操作系统上运行：</p><p>Windows主机（64位）：</p><p>Windows 8.1</p><p>Windows 10</p><p>Windows 11 21H2</p><p>Windows Server 2012</p><p>Windows Server 2012 R2</p><p>Windows Server 2016</p><p>Windows Server 2019</p><p>Windows Server 2022</p><p>macOS主机（64位）：</p><p>10.15 (Catalina)</p><p>11 (Big Sur)</p><p>12 (Monterey)</p><p>Linux主机（64位）：</p><p>Ubuntu 18.04 LTS, 20.04 LTS和22.04</p><p>Debian GNU/Linux 10（"Buster"）和11（"Bullseye"）。</p><p>Oracle Linux 6、7和8</p><p>CentOS/Red Hat Enterprise Linux 7和8</p><p>Fedora 35和36</p><p>Gentoo Linux</p><p>SUSE Linux Enterprise server 12和15</p><p>openSUSE Leap 15.3</p><p>你可以在这里找到 VirtualBox 7.0.0 Beta 1 的完整发布说明：</p><p><a href="https://download.virtualbox.org/virtualbox/7.0.0_BETA1/UserManual.pdf" _src="https://download.virtualbox.org/virtualbox/7.0.0_BETA1/UserManual.pdf" target="_blank">https://download.virtualbox.org/virtualbox/7.0.0_BETA1/UserManual.pdf</a><br></p>
