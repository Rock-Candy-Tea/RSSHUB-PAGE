<strong>通过官方 Twitter 账号发布的一份声明中，CD PROJECT Red 宣布它将停止在俄罗斯和白俄罗斯的所有游戏销售。</strong>它说：“今天，我们开始与我们的合作伙伴合作，暂停数字销售，并停止向俄罗斯和白俄罗斯境内交付 CD PROJECT 集团的产品，以及在 GOG 平台上分发的所有游戏的实物库存”。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0304/65d148e1f54bd95.webp" alt="FM7liLQXIAMXqOG.webp" referrerpolicy="no-referrer"></p><p>CD PROJECT RED 并不是第一家暂停其在俄罗斯活动的公司。<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>公司昨天也从俄罗斯市场撤出了其全部产品，而包括 Snapchat、Rakuten 等在内的很多公司都停止了在该地区的广告宣传。苹果和Google也加入了这个行列，将 Russia Today (RT) 和 Sputnik 新闻从其服务中删除。</p><p>这家开发了《赛博朋克 2077》的游戏工作室认为：“商业实体如果联合起来，就有能力在普通人的心中激发全球变革”。该公司进一步表示，它承认与俄乌冲突无关的个人将受到这一决定的影响，然而，这一行动将激励全球社会谈论在欧洲中心发生的事情。</p>
