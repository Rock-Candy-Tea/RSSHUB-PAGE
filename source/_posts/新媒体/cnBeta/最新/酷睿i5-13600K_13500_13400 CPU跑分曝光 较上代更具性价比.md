<strong>@TUM_APISAK 刚刚在 Twitter 上分享了酷睿 i5-13600K / 13500 / 13400 三款台式处理器的基准测试成绩。</strong>有趣的是，这批芯片的步进分别为 Raptor Lake B0 和 Alder Lake C0 —— 前者采用了 13 代新架构，而后者延续了当前 12 代产品阵容的缓存结构设计。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0914/93e712254733c30.png" alt="1-2.png" referrerpolicy="no-referrer"></p><p>首先是 i5-13600K（via <a href="https://www.passmark.com/baselines/V10/display.php?id=162562826651" target="_self">PassMark</a>），这款 Raptor Lake B0 芯片采用了 6P + 8E（14C / 20T）的混合式 CPU 核心设计。</p><p>尽管大核数量保持不变，但架构从 Golden Cove 升级到了 Raptor Cove、且小核数量也较上一代 i5-12600K 翻倍。</p><p>此外 i5-13600K 具有 20MB L2 + 24MB L3 缓存（总计 44MB），P 核基础频率 3.5 GHz、全核睿频 5.1 GHz / 单核最高 5.2 GHz 。E 核基础频率 3.5 GHz，睿频可达 3.9 GHz 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0914/f84d7c9927782cb.png" alt="1-1.png" referrerpolicy="no-referrer"></p><p>其次是 <a href="https://www.cpubenchmark.net/cpu.php?cpu=Intel+Core+i5-13500&id=4993" target="_self">i5-13500</a> / <a href="https://www.cpubenchmark.net/cpu.php?cpu=Intel+Core+i5-13400&id=4994" target="_self">i5-13400</a>，这两款 CPU 都属于 Alder Lake C0 的马甲，前者采用了 6P + 8E（14C / 20T）的设计、后者则是 6P + 4E（10C / 16T）的设计。</p><p>缓存方面，i5-13500 拥有 11.5MB L2 + 24MB L3、热设计功耗（TDP）为 65W，而 i5-13400 拥有 9.5MB L2 + 20MB L3 。虽然基础频率都是 2.5 GHz，但睿频分别可达 4.7 / 4.6 GHz 。</p><p><a href="https://static.cnbetacdn.com/article/2022/0914/88b690d123e59b9.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0914/88b690d123e59b9.png" alt="5.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">PassMark 单线程性能比较</p><p>性能数据方面，i5-13600K 的单线程得分为 4249 MOps/s，而 i5-13500 / 13400 则是 4033 和 3866 MOps/s 。</p><p>作为参考，上一代 i5-12600K 的得分为 3980 MOps/s，意味着 i5-13600K / 13500 分别领先 7% 和 1%（i5-13400 也接近）。</p><p>此外与前代对应型号相比，i5-13500 / 13400 分别较 i5-12500 / 12400 领先 8% 左右。</p><p><img src="https://static.cnbetacdn.com/article/2022/0914/a9279732a3a1a12.png" alt="6.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（截图 via <a href="https://wccftech.com/intel-core-i5-13600k-13500-13400-cpu-benchmarks-leak-faster-than-12600k-170-200-us-price-range/" target="_self">WCCFTech</a>）</p><p>最后，尽管 i5-13600K 的预估定价在 250 ~ 300 美元（约 1740 ~ 2088 RMB）档位，但 i5-13500 / 13400 有望在 200 / 170 美元（合 1392 / 1183 RMB）的中端市场更加热销。</p><p>按照计划，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>会在今年 10 月推出 13 代 Raptor Lake 台式 CPU 和 Z790 芯片组主板，以同今秋上市的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 7000 系列 AM5 平台展开直接的竞争。</p><p>若英特尔维持当前的定价策略，消费者们有望获得 8-10% 的性能提升。同时考虑到两代平台的兼容性，PC DIY 玩家还可搭配使用更实惠的 B660 芯片组主板和 DDR4 内存。</p>
