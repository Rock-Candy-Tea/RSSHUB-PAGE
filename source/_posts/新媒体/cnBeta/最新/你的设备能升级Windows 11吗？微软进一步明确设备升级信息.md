Windows 11 系统有望在今年年底之前正式发布，伴随着预览版的推进微软逐渐放出了关于系统升级的更多消息。<strong>在 6 月 24 日的发布会上，微软正式发布了 Windows 11 系统，并表示该系统将在今年晚些时候开始向消费者推广，符合条件的设备可以免费升级。</strong><br>
<p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0709/e2a9816d7c3d258.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0709/e2a9816d7c3d258.jpg" alt="eoni4h9u.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">有一种说法是，<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 将会随 Windows 10 21H2 功能更新一同发布。现有 Windows 10 用户可以通过 Windows Update 来决定升级到 Windows 11 ，或者继续保留在当前版本。</p><p style="text-align: left;"><strong>● Windows 10 家庭版/专业版</strong><br style="text-align: left;"></p><p style="text-align: left;">微软表示升级需要 Windows 10 20H1（May 2020 Update）及更高版本。也就是说如果你目前使用的 Windows 10 专业版/家庭版是 20H1 之前的版本，那么将无法通过 Windows Update 方式获得升级。而且根据旧版本所需的文件内容，升级时间可能会拉长。</p><p style="text-align: left;">该公司在 6 月 30 日发布的一条推文中表示：“运行 Windows 10 20H1 功能更新及更高版本，符合条件的硬件可以升级到 Windows 11 系统。Windows 11 下载和安装时间可能要比常规的 Windows 10 功能更新时间要长”。</p><p style="text-align: left;"><strong>● Windows 10 企业版</strong><br style="text-align: left;"></p><p style="text-align: left;">根据微软官员的说法，企业客户的最低系统版本调整到 Windows 10 Version 1909（November 2019 Update）。因此，如果你使用的是 Windows 10 Version 1909 企业版，你可以直接升级到 Windows 11，而不用安装最新的功能更新。</p><p style="text-align: left;"><strong>● Windows 7</strong><br style="text-align: left;"></p><p style="text-align: left;">对于 Windows 7 系统，微软表示如果想要升级 Windows 11 系统需要重新安装。这意味着用户需要手动备份文件。</p><p style="text-align: left;">对于消费者来说，微软将继续提供对 Windows Update 的控制和透明度。换句话说，微软不打算强迫用户升级到新版本。Windows 11将是一个用户主动寻求的过程，这意味着你需要手动 "寻求 "更新到 Windows 11。</p>
