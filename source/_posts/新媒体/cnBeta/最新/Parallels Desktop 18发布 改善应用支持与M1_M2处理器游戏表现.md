适用于 Mac 的 Parallels Desktop 18 已经发布。新版本主要在 Apple Silicon Mac 设备上对 Windows 应用提供了更好的兼容性支持，并提供了更优秀的 Windows 游戏体验。<br>
<p><strong>购买链接:</strong></p><p><a href="https://prf.hn/l/EQJgjyL" target="_blank">Parallels Desktop 18</a><br></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0809/561abfe57197627.webp" alt="5x3haa0b.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">在本周二发布的更新中，适用于 Mac 的 Parallels Desktop 18 引入了多处优化，可以在 Mac 设备上运行超过 20 万款 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 应用和经典游戏。在模拟运行 Windows 11 系统的时候，运行<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>应用程序的兼容性也得到了改进。</p><p style="text-align: left;">Parallels 表示，从 Mac 磁盘本身保存和读取文件的速度也大幅加强。在游戏方面提供了更持续和无缝的集成，并兼容 ProMotion 显示器和<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>最新芯片。对于 Mac Studio 上的 M1 Ultra，现在可以使用的 Windows 11 性能提高了 96%。</p><p style="text-align: left;"><a href="https://static.cnbetacdn.com/article/2022/0809/757f2fa09ce5bd9.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0809/757f2fa09ce5bd9.png" title alt="50ee93de-4e82-4151-bbfe-dcd9b0fc5153.png" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">此外，新版本提高了易用性。用户可以一键下载、安装和配置 Windows 11。相同的设置系统还提供了一系列现成的 Linux 系统。对于游戏玩家来说，Windows 游戏体验得到了增强，用户可以将游戏控制器连接到他们的 Mac，切换到 Windows，然后开始轻松玩游戏。也有人声称以高帧速率播放流畅的 Windows UI 和视频。</p><p style="text-align: left;">还改进了对实时数据流设备的 USB 3.0 支持，例如 Elgato HD60 和 Startech USB 3.0 Video Capture 设备。专业版为 M1 Mac 添加了网络调节器，因此用户可以为虚拟机的网络设置特定的网络条件。可以选择将虚拟机网络与 Mac 隔离以提高安全性，甚至可以使用 Parallels Desktop 作为虚拟化提供商来设置 MiniKube 集群。</p><p style="text-align: left;"><a href="https://static.cnbetacdn.com/article/2022/0809/ef347786c92880d.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0809/ef347786c92880d.png" title alt="9256a637-9d96-458e-bdb4-4beaf783c4c5.png" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">专业版用户受益于 Linux ARM 虚拟机的网络启动，并能够使用 <a data-link="1" href="https://microsoft.pvxt.net/zaZYr" target="_blank">Visual Studio</a> 的 Parallels 插件在单独的 VM 中分析应用程序的性能。Parallels Desktop 18 for Mac Business Edition 继承了 Pro 版，为员工添加了单点登录和 SAML 身份验证，以便使用公司帐户激活软件。</p><p style="text-align: left;">管理员可以使用 Business Edition 跨员工 Mac 系统部署、配置和传输 Windows 11 虚拟机。他们还可以在 Parallels 我的帐户中集中管理 Parallels 客户体验计划参与。</p><p style="text-align: left;">Parallels Desktop 18 在 macOS Mojave 及更高版本上运行，并在向公众发布时支持 macOS Ventura。最低硬件要求是具有 Core 2 Duo 或更高版本、4GB 内存和 600MB 安装空间的 Mac。</p><p style="text-align: left;">标准版每年收费 99.99 美元，或永久许可证 129.99 美元。专业版每年 119.99 美元，而商业版每年费用为 149.99 美元。</p><p style="text-align: left;">从任何以前的 Parallels Desktop 版本升级到 Parallels Desktop 18 for Mac 的价格为 69.99 美元，以前的 Pro Edition 安装也可以以每年 69.99 美元的价格升级到 Parallels Desktop 18 for <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">Mac Pro</a> Edition。</p><p><a href="https://static.cnbetacdn.com/article/2022/08/66c84c053c81836.png" target="_blank"><img src="https://static.cnbetacdn.com/article/2022/08/66c84c053c81836.png" referrerpolicy="no-referrer"></a></p>
