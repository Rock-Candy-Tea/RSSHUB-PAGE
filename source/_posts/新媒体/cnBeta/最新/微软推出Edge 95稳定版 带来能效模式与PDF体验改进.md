微软于昨日向稳定版通道放出了 Edge 95，<strong>作为四周发布周期中的最新更新，其并未带来任何华丽的新功能、而是专注于能效模式和 PDF 体验上的改进。</strong>前者能够作为“标签页休眠”（Sleeping Tabs）功能的一个有力补充，进一步细化笔记本电脑的资源管理，从而延长电池续航。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1023/8ededb783ec477c.png" alt="Edge 95.png" referrerpolicy="no-referrer"></p><p>据悉，当笔记本电脑进入节电模式时，该功能将被自动激活。当然你也可以让它始终开启，或者在断开外接电源时启用。</p><p>然后是 PDF 体验方面的多项改进，比如你将能够在重新打开 PDF 文档时，从上次中断的地方继续浏览。且 Web 浏览器添加了对自由表单文本框的支持，并可向 PDF 文档添加注释。</p><p>Edge 95 还为浏览器内的菜单 / 右键操作带来了适配 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 的 Fluent 设计元素（包括圆角），同时改进了收藏、下载、密码管理等体验。</p><p><strong>以下是本次 Edge 95 更新的更新日志：</strong></p><blockquote><p>· 可在文件资源管理器中查看 Microsoft Edge 中的 SharePoint Online 库。</p><p>· 默认在 Windows 文件资源管理器中打开 Intranet 区域的文件 URL 链接。</p><p>· 引入对 PWA 应用和 WebView 的下载体验改进，且支持拖放到桌面或文件资源管理器。</p><p>· 可继续上次的 PDF 文档浏览进度 / 文档注释可添加自由格式文本框。</p><p>· 能效模式可进一步延长启用节电模式的笔记本电池续航。</p><p>· 为收藏夹添加引文支持。</p><p>· 能够用更少的点击次数，更快地更新您的密码。</p><p>· 允许用户一键自动创建在线账户，为注册页面提供额外支持。</p></blockquote><p>最后，如果你不确定自己是否已经更新至 Edge 95（PC / Mac 稳定版），可点击窗口右上角的“设置及其它（…）-> 帮助和反馈 -> 关于 Microsoft Edge”来手动执行。</p>
