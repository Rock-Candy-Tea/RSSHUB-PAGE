<strong>为满足客户的高带宽网络访问需求，威联通（QNAP Systems）刚刚推出了采用 PCIe 4.0 接口的双端口十万兆（100 GbE）扩展卡。</strong>可知 QXG-100G2SF-CX6 基于英伟达 Mellanox ConnectX-6 控制器方案，兼容 QNAP 全闪存 NAS 和 Windows / Linux，能够为工作站 / 服务器带来高达 200 Gbps 的总吞吐量。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0817/4586afb3cd93baf.png" alt="1.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：QNAP <a href="https://www.qnap.com/en/product/qxg-100g2sf-cx6/specs/hardware" target="_self">官网</a>）</p><p>通过在数据中心、虚拟化、多媒体串流传输和关键备份 / 恢复任务上使用，QXG-100G2SF-CX6 能够以极低延迟化解性能瓶颈。</p><p>QNAP 产品经理 Andy Chuang 补充道，服务器效率和虚拟化密度的提升，持续推动了对更高带宽的需求增长。</p><p><img src="https://static.cnbetacdn.com/article/2022/0817/9e45455a7f3b2ff.png" alt="2.png" referrerpolicy="no-referrer"></p><p>而 QXG-100G2SF-CX6 网络扩展卡的到来，为升级到 100 GbE 网络环境、提升性能并减少延迟提供了一种切实可行的解决方案。</p><p>此外作为一套功能丰富的网络解决方案，QXG-100G2SF-CX6 采用先进技术降低了 CPU 工作负载，是 I/O 密集型、延迟敏感型虚拟化和企业级存储应用的理想之选。</p><p><img src="https://static.cnbetacdn.com/article/2022/0817/cab3c8a33cc91dc.png" alt="3.png" referrerpolicy="no-referrer"></p><p>对于具有 PCIe 4.0 插槽的 TS-h2490FU、TDS-h2489FU 和 TS-h1290FX 来说，QXG-100G2SF-CX6 能够更好地发挥 QNAP NVMe 全闪存 NAS 的全部潜力。</p><p>除了带来业内领先的吞吐量、低延迟和高消息速率，它还可以与高速交换机搭配使用。在实现更快的数据传输和网络效率的同时，还具有无与伦比的 RDMA、RoCE v1/v2 和 SR-IOV（1K VF 和 8 PF）性能。</p><p><img src="https://static.cnbetacdn.com/article/2022/0817/bb16e5d0e5f25ce.png" alt="4.png" referrerpolicy="no-referrer"></p><p>兼容性方面，QXG-100G2SF-CX6 支持 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Server 2016 / 2019 和 Linux 平台。</p><p>感兴趣的客户，可通过 QNAP <a data-link="1" href="https://microsoft.pvxt.net/Xx07X" target="_blank">配件</a>商城购买，定价为 1199 美元（约 8120 RMB）。</p>
