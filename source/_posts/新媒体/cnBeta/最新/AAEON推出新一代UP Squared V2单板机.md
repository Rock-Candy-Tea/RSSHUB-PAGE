<strong>以 UP Board 系列而被大家所熟知的研扬科技（AAEON），刚刚推出了新一代 UP Squared V2 单板机。</strong>作为 UP Squared 的继任者，V2 版本维持了 3.37×3.54 英寸（85×90 毫米）的外形尺寸，但显著提升了处理能力、图形功能、以及扩展选项，让用户能够更高效地开发。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0916/5d97a74ab6949de.jpg" alt="6.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：AAEON <a href="https://www.aaeon.com/en/p/up-developer-board-intel-elkhart-lake-up-squared-v2" target="_self">官网</a>）</p><p>定价方面，UP Squared V2 延续了极具竞争力的策略。不过得益于<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>赛扬 N6210 和奔腾 J6426 处理器 SoC（之前被称作 Elkhart Lake），其 CPU 性能提升了 40% 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0916/24283595cbc8b0c.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>UP Squared V2 的单线程 / 多线程性能分别达到了前代的 1.7 / 1.5 倍，辅以高达 16GB 的 LPDDR4 系统内存。而重新配置的机械设计，可让用户尽享英特尔 Gen 11 UHD 核显的体验。</p><p><img src="https://static.cnbetacdn.com/article/2022/0916/74ec1e62ea20800.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>I/O 方面，UP Squared V2 提供了 HDMI 1.4 + DisplayPort 1.2 + eDP 1.3 显示输出组合，最多可同时带动三台 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_110018%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">4K</a> 显示器。</p><p><img src="https://static.cnbetacdn.com/article/2022/0916/4b75235f5886bf7.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>此外尽管上代已经提供了 mini-PCIe 和 M.2 2230（E-Key）扩展插槽，但 V2 版本还提供了 M.2 2280（M-Key）和 SATA III 存储选项。</p><p><img src="https://static.cnbetacdn.com/article/2022/0916/a1422c51036981b.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>另外 UP Squared V2 提供了集成 AI、PCIe 和 Wi-Fi 模块的选项，并通过 TPM 2.0 端口赋予了额外的安全性。并且对于开发者们来说，也可更容易地将音频功能集成到他们的解决方案中。</p><p><img src="https://static.cnbetacdn.com/article/2022/0916/952595265bfab5e.jpg" alt="5.jpg" referrerpolicy="no-referrer"></p><p>最后，AAEON 认为上述增强特性将为客户带来更强大的平台体验，使之能够继续以较低的成本来构建和升级具有强大性能的工业应用，比如 AGV、服务机器人和各种工业 4.0 智能工厂解决方案。</p>
