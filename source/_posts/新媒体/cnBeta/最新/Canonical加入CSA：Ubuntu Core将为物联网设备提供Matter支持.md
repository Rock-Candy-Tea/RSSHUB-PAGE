Canonical 为了帮助物联网（IoT）行业设立新的安全和可靠性标准，并为 Ubuntu Core 提供 Matter 支持，今天该公司宣布加入 Connectivity Standards Alliance（连接标准联盟）。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0926/076ae90962d2134.webp" alt="vy267lkm.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">今天，Canonical 正式成为 CSA 的参与成员，将帮助开发和推广 IoT（物联网）行业的开放标准，并为嵌入式设备推广 Ubuntu Core 操作系统。</p><p style="text-align: left;">Canonical 是首家提供主流独立 GNU/Linux 发行版本的 CSA 成员，承诺在其 Ubuntu Core 操作系统中支持开箱即用的 Matter 标准，以提高互操作性和可访问性。</p><p style="text-align: left;">Canonical 产品经理内森·哈特（Nathan Hart）表示：“我们的目标是让 Ubuntu Core 成为 Matter 设备的最佳平台。我们的目标是在 Ubuntu Core 上开箱即用地支持 Matter，以便将 Matter 设备推向市场的最快和最可靠的方式”。</p><p style="text-align: left;">Matter 是即将推出的基于 Internet 协议 (IP) 的通信标准，旨在使智能家居设备安全、可靠和无缝使用。当今年晚些时候推出时，Matter 将被大多数主要的物联网硬件供应商以及几乎所有支持物联网设备的软件供应商采用。</p>
