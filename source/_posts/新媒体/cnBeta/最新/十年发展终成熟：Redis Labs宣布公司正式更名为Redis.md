<strong>经历十年的发展，Redis Labs 现已正式将公司更名为 Redis 。</strong>通过删除“Labs”一词，其表示这一变化标志着该公司和 Redis 开源项目的成熟。据悉，该项目于 2011 年其接受贡献、并于 2015 年得到赞助。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0813/0b4bfb7099eec7a.png" referrerpolicy="no-referrer"></p><p>Redis 补充道，本次更名也反映了该公司将继续把 Redis 作为实时数据平台来发展使命，并专注于引领包括个人贡献者、客户、以及合作伙伴在内的整个社区的一致性，以提供应对技术挑战的解决方案、并保持其经久不衰的受欢迎程度。</p><p>至于开源的 Redis 许可，本次公司更名也不会对其产生任何影响。相关项目将沿用 BSD 许可，且不会影响去年引入的治理模型。</p><p>Redis 联合创始人兼首席执行官 Ofer Bengal 表示：</p><blockquote><p>我们对 Redis 开发的承诺，甚至早于我们推出托管服务之前就已经开始了。我们的团队，有在频繁为代码库做出相应的贡献。</p><p>今天，我们将引领 Redis 从单一的服务用途（如缓存和会话管理）、到提供更广泛的功能，使之能够为世界上最关键的应用程序提供支撑。</p><p>通过本次精简公司名称，也反映了我们致力于提供的、简单且纯粹的 Redis 。</p></blockquote>
