尽管 Android 操作系统已经在移动设备市场与苹果 iOS 打得有来有回，但该公司以 Web 为中心的 Chrome OS 桌面平台，却难以在应用生态上与微软 Windows 或苹果 macOS 同日而语。<strong>好消息是，在深耕教育市场多年的基础上，谷歌正在努力让 Chromebook 设备用上更多的软件。</strong><br><br>
<p><img src="https://static.cnbetacdn.com/article/2022/0509/1618319dbeb529b.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>此前这家总部位于山景城的科技巨头，已允许大量 Android Apps（甚至 Linux 应用程序）在 Chrome OS 操作系统上运行。</p><p>不过在<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a> / 平板电脑之外，Android Apps 在其它大屏设备（比如上网本）的使用体验仍然不佳。</p><blockquote><p><a href="https://www.slashgear.com/856606/google-aims-to-fix-broken-android-app-experiences-on-chromebook/" target="_self">SlashGear</a> 指出：尽管平板电脑的诞生早于许多智能手机，但 Android 起初只是为了屏幕相当小的移动设备而打造的。</p><p>经过适当的优化，其一次只能在屏幕上显示数量有限的项目。即使有所谓的分屏 / 多任务功能，各 OEM 厂商仍需慎重考虑设备的实际屏幕尺寸与单手交互体验。</p></blockquote><p>尴尬的是，Android 平板电脑正好陷入了这种两难的境地。而且由于市场并不那么有利可图，开发者的兴致也并不高。相比之下，Chrome OS 的情况略有不同。</p><p><img src="https://static.cnbetacdn.com/article/2022/0509/fbefc2d10fb514b.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>由 ChromeOS.dev 披露的数据可知，即便应用程序在该平台上的表现仍有较大改进空间，去年 Android Apps 的使用率还是增加了 50% 。</p><p>更棒的是，随着新版 Android 官方开发工具的到来，开发者们将能够告别劳心劳力的方式，轻松确保其应用程序能够在 Chromebook 等设备上顺畅工作。</p><blockquote><p>据悉，在平板、尤其 Chrome OS 上跑 Android 应用程序面临的一大问题，就是它们在调整窗口大小时，难以正确调节其用户界面（UI）和显示的内容。</p><p>原因是开发者没有简单的方法来验证这种不同的行为，除非他们手头已经有 Chromebook 硬件来跑测试。</p><p>可即便如此，他们也智能将 App 部署到设备后才能跑测试、而不是在开发过程中随时可以调试。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0509/bb8fa225dc5011d.gif" alt="3.gif" referrerpolicy="no-referrer"></p><p>好消息是，即将发布的新版 Android Studio 集成开发环境（版本代号 Electric Eel），有望为开发者提供正确处理相关事务所需的所有工具。</p><blockquote><p><a href="https://www.androidpolice.com/android-studio-desktop-virtual-device/" target="_self">Android Police</a> 指出，桌面 Android 虚拟设备能够像模拟器一样，允许开发者测试其应用程序在调节窗口大小等操作时的行为。</p><p>并且涵盖了有关特定 App 如何在多窗口模式下与其它应用协同工作、消息通知将如何显示，以及在智能手机 / 平板电脑上不那么关心的注意事项。</p><p>这意味着 Chrome OS 上的 Android 应用程序，可在不久的将来更加正常、顺畅地运行。</p></blockquote><p>显然，随着主打大屏设备的 Android 12L 项目的推进，Google 也希望新版 Android Studio 能够为 Chrome OS / PC 模式的开发带来更大的助力。</p>
