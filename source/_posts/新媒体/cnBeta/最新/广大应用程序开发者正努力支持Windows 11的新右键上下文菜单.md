Windows 11 操作系统为“开始”菜单、通知中心、任务栏等组件带来了全新的外观，且记事本（Notepad）和画图（MS Paint）等原生应用也迎来了重新设计，辅以新颖的上下文菜单。<strong>对于习惯了通过右键点击、然后在十几个条目中挑选执行各种“高级操作”的用户来说，你会发现 Windows 11 上已变得更加智能和简洁。</strong><br>
<p><a href="https://static.cnbetacdn.com/article/2021/1010/c6cb1c8180117fa.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1010/c6cb1c8180117fa.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://www.windowslatest.com/2021/10/10/windows-11s-modern-context-menu-is-getting-support-from-developers/" target="_self">Windows Latest</a>）</p><p>如果你厌倦了在凌乱的右键菜单中查找“复制”或“重命名”等命令，Windows 11 的新 UI 设计或将你从中解脱出来。</p><p>不过为了兼容，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还是被迫将其它操作都隐藏在了“显示更多选项”的分组中（或通过 Shift + F10 组合键唤出）。</p><p>好消息是，以 WinRAR 这款流行的压缩 / 解压缩软件为例（6.10 Beta 测试版本），开发者们正在努力为 Windows 11 的现代上下文菜单提供支持。</p><p><img src="https://static.cnbetacdn.com/article/2021/1010/39186a8a06bc321.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>如上方截图所示，Windows 11 上下文菜单中有一个新的 WinRAR 选项（用 *** 打开），意味着你无需再通过多一步点击“显示更多选项”来访问折叠后的右键菜单。</p><p>不过在更新日志中，WinRAR 开发者还是解释称 —— 应用程序只能向 Windows 11 的右键上下文菜单添加顶级命令或子菜单。</p><p>若‘综合设置’对话框中的‘层叠右键关联菜单’被选中，则 WinRAR 会在 Windows 11 的右键上下文条目中显示一个子菜单。</p><p>若不手动启用该选项，则系统只会显示最基础的“打开”提取右键菜单选项。</p><p><img src="https://static.cnbetacdn.com/article/2021/1010/18e0b58831ac7c7.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>与此同时，另一款流行的 7-Zip 压缩 / 解压缩软件也有望很快迎来同样的右键菜单支持。如果等不及官方更新，不妨抢先下载 NanaZip 体验一番。</p><p>最后，按照微软的计划，该公司将于 2022 年中完成 Windows 11 的大规模推送。届时显然会有更多第三方应用程序开发商支持新的右键上下文菜单，且回收站等组件也将迎来相关改进。</p>
