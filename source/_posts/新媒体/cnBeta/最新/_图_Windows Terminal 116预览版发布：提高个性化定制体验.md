得益于最新引入的一些自定义选项，Windows Terminal 变得更有吸引力了。在今天发布的 Windows Terminal Preview 1.16 版本中，<strong>引入了新的文本渲染引擎和默认色彩设置。不过更重要的改进是，微软大幅提高了个性化定制体验。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0914/08b76b24819450f.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">微软项目经理凯拉·希娜文（Kayla Cinnamon）解释道：“通过引入主题，我们添加了更多自定义可能性。Themes 是一个全局属性，可以包含各种主题对象，这些对象将出现在设置 UI 的外观页面的主题下拉列表中”。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0914/87b4a9c0e082773.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0914/06e52bc2edfb0bf.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">希娜文继续说道：“我们发现我们的配色方案页面不是最直观的，可以使用设计刷新。我们更新了设置 UI 配色方案页面以改进其样式和用户流程。我们还添加了一个‘设置为默认’按钮，该按钮将在所有配置文件中应用一个配色方案作为您的默认配色方案。指定配色方案的配置文件将使用其指定的方案而不是默认方案”。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0914/57b895dbcffea3a.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0914/9b9af28f9b631e2.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0914/4d54deec4e16344.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">如果您想亲自试用 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Terminal Preview 1.16，您可以在此处从 <a href="https://www.microsoft.com/store/productId/9N8G5RFZ9XK3" target="_blank">Microsoft Store</a> 下载它。</p>
