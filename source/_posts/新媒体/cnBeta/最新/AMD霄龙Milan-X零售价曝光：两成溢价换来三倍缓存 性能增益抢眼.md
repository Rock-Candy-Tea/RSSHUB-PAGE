<strong>早在去年 11 月，AMD 就已经发布了第三代霄龙（EPYC）7003X 系列“Milan-X”服务器 CPU 。不过直到近日，该公司才开始向全球数据中心客户正式发货。</strong>有趣的是，虽然其主要面向高性能计算（HPC）市场，但个别零售商也列出了各款 SKU 的报价。可知在换来三倍高速缓存的同时，Milan-X 的平均定价也较 Milan 高出了 20% 。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0226/4831edac132f716.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0226/4831edac132f716.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/amd-milan-x-epyc-7003x-cpus-prices-show-20-percent-more-expensive-3x-more-cache/" target="_self">WCCFTech</a>）</p><p>其实早在 Milan-X 正式亮相前的 2021 年 8 月，就已经有零售商曝光了 EPYC 7003X 系列处理器的售价。</p><p>由近日从 <a href="https://www.shopblt.com/search/s_all=EPYC&s_max=25&s_lastpos=4688760&order_id=219337371" target="_self">ShopBLT</a> 获得的最新报价可知，Milan-X 的定价似乎并不过分，以下是主要四款 SKU 的的介绍。</p><blockquote><p>● EPYC 7773X -- 64 核 / 128 线程，OPN 代码 100-000000504 。</p><p>● EPYC 7573X -- 32 核 / 64 线程，OPN 代码 100-000000506 。</p><p>● EPYC 7473X -- 24 核 / 48 线程，OPN 代码 100-000000507 。</p><p>● EPYC 7373X -- 16 核 / 32 线程，OPN 代码 100-000000508 。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0226/f024324e1be10b9.png" alt="2.png" referrerpolicy="no-referrer"></p><p>旗舰款 EPYC 7773X 拥有 64C / 128T，最大热设计功耗（TDP）280W 。基础频率 2.2 GHz / 加速可达 3.5 GHz，总缓存也是惊人的 768 MB 。</p><blockquote><p>在标准 256MB 三缓之外，EPYC Milan-X 还有 512MB 的 3D V-Cahce 堆叠缓存（SRAM）。</p><p>意味着每组 Zen 3 CCD 具有 64MB L3 缓存，是当前 EPYC Milan CPU 的三倍。</p><p>定价方面，EPYC 7773X 为 9609 美元，较 8816 美元的 EPYC 7763 高出 11% 。</p></blockquote><p>其次是拥有 32C / 64T 的 EPYC 7573X，其 TDP 为 280W、基础频率 2.8 GHz / 加速可达 3.6 GHz，总缓存也是 768 MB 。</p><blockquote><p>有趣的是，该 CPU 无需 8 组 CCD 来达成 32 核心，而是只需 4 组 CCD 。</p><p>但考虑到双倍的堆叠缓存，其成本仍不容小觑，即使核心数较低的 SKU 也可能配备完整的 8-CCD 芯片。</p><p>价格方面，EPYC 7573X 为 6107 美元，较 5312 美元的 EPYC 75F3 高出 14% 。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0226/2518cdd766579ef.png" alt="3.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via ShopBLT）</p><p>第三款是 24C / 48T 的 EPYC 7473X，其 TDP 为 240W，基础频率 2.8 GHz / 加速可达 3.7 GHz 。</p><p>第四款是 16C / 32T 的 EPYC 7373X，其 TDP 为 240W，基础频率 3.05 GHz / 加速可达 3.8 GHz 。</p><blockquote><p>两款 Milan-X 芯片均拥有 768 MB 总缓存，报价分别为 5317 / 4290 美元，较上一代对应的 Milan 型号高出 34% 和 32% 。</p><p>性能方面，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 通过 RTL 测试表明 Milan-X 较 Milan CPU 提升 66%，且 16 核 Milan-X 能够较非 -X SKU 更快地完成 Synopsys VCS 验证测试。</p><p>延迟和带宽方面，Milan-X 霄龙处理器也被证明可在拥有超大缓存（且价格差异微小）的情况下具有更大的吸引力。</p></blockquote>
