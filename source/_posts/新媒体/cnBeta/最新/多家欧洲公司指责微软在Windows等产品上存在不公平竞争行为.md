<strong>以文件托管公司 Nextcloud 为首的多家欧洲公司近日组成联盟 Coalition for a Level Playing Field</strong>，向欧盟以及德国反垄断机构提出申诉，指责微软正在滥用其在操作系统方面的主导地位，向协作和云服务等不相关的领域扩展。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1127/ad55f268c2b3a70.webp" alt="gp0ehfg3.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">该联盟表示<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>正将 Microsoft 365 越来越深入地整合到更广泛的服务和软件组合中。这其中最典型的例子就是 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 系统，该联盟表示每当 Windows 用户处理文件存储的时候，微软的 OneDrive 在 Windows 系统中现有特权，成为一种优选。另一个例子是微软的“Teams”，它作为 Windows 11 的默认部分被成功推入市场。</p><p style="text-align: left;">他们声称，微软将占主导地位的 Windows（操作软件）与 OneDrive（云）产品结合起来，几乎不可能与他们的 SaaS 服务竞争。它说明了反竞争的做法，如基于 Windows 的市场支配地位的“自我推荐”。</p><p style="text-align: left;">其效果是，当涉及到云存储时，在过去几年中，微软、Google和亚马逊的市场份额已经增长到欧洲总市场的66%，本地（欧洲）供应商从 26% 下降到 16%。他们声称，上述行为是全球科技巨头在欧洲的这种急剧增长水平的核心，与 90 年代末微软将 IE 浏览器与 Windows 捆绑从而摧毁 Netscape 的情况非常相似。</p><p style="text-align: left;">Nextloud 的首席执行官兼创始人 Frank Karlitschek 说。</p><p style="text-align: left;">这与微软在浏览器市场上扼杀竞争时的所作所为非常相似，它在十多年里几乎停止了所有的浏览器创新。复制创新者的产品，将其与自己的主导产品捆绑，并扼杀他们的业务，然后停止创新。这种行为对消费者、对市场、当然也对欧盟的本地企业都是不利的。我们和联盟的其他成员一起，要求欧洲的反垄断机构执行一个公平的竞争环境，给客户一个自由的选择，给竞争一个公平的机会。</p><p style="text-align: left;">Coalition for a Level Playing Field 联盟要求</p><p style="text-align: left;">● 不滥用微软在操作系统和相关市场的主导地位（例如通过捆绑、预装或利用额外的微软服务），同时保持一个公平的竞争环境。</p><p style="text-align: left;">● 微软必须支持开放标准和互操作性。这将给欧洲消费者提供一个自由选择服务产品的机会，这些服务产品应该根据其优点进行公平竞争。</p>
