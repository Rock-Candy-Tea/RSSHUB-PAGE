<strong>为庆祝 Visual Studio .NET 二十周年，微软将于太平洋时间 2 月 14 日上午 9 点开始，发起为期一个月的庆祝活动。</strong>感兴趣的开发者们，可到点收看由 Scott Hunter、Scott Hanselman 和一群特邀嘉宾的直播互动。届时他们会分享相关故事和回顾历史，并透露本月剩余时间的一系列其它社区庆祝活动。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0215/e04a6c786614611.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Microsoft <a href="https://dotnet.microsoft.com/en-us/" target="_self">官网</a>）</p><p>正如博文公告中暗示的那样，庆祝活动期间，主持人和嘉宾可能会深入探讨以往取得的成就、以及 .NET 的未来发展。</p><blockquote><p>我们刚刚在 2021 年 11 月发布了 <a href="https://dotnet.microsoft.com/en-us/download/dotnet/6.0" target="_self">.NET 6</a>，并且正在全速构建 .NET 7，后者的首个预览（Preview 1）将于本周到来。</p><p>借助 .NET 6，您现在可拥有一套统一的基础库和 SDK 。通过投资 C# 10 和最少的 API 以获得简化的开发体验，或通过热重载来提升高生产力。</p><p>作为 .NET 的快速采用版本，.NET 6 已收获用户的广泛好评。如果你尚未用上，我们还是在这里鼓励你尝试一下。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0215/609cb660fca3ca7.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">油管直播（<a href="https://www.youtube.com/watch?v=67tCWKnweso" target="_self">传送门</a>）</p><p>微软还很高兴能够尽快将 .NET 多平台应用程序 UI（.NET MAUI）交付到广大开发者的手中。</p><p>它将允许用户通过单一的代码库，为 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>、macOS、iOS 和 Android 平台构建原生应用程序。</p><p>不过目前，.NET 团队正专注于质量和错误改进。感兴趣的朋友，可积极参与预览，并向微软即时提交反馈。</p>
