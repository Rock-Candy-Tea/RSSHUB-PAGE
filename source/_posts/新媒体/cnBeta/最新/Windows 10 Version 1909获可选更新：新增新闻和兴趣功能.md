面向 Windows 10 Version 1909 功能更新用户，今天微软发布了 KB5001396 累积更新。用户除了可以通过检查 Windows Update 或者 WUSU 服务进行自动更新之外，也可以通过本文下方的链接进行手动离线安装。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0423/42f494eb3a4acc1.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0423/42f494eb3a4acc1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">需要注意的是，KB5001396 是可选更新，意味着并非强制 Version 1909 用户进行更新，不过它会整合到下个月补丁星期二活动日发布的累积更新中。如果你决定跳过这个补丁，你将在下个月的周二补丁中得到同样的一套错误修复。在安装该可选更新之后，用户版本号升至 Build 18363.1533。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0423/d50eedc11e02ed0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0423/d50eedc11e02ed0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">本次可选更新主要是将“新闻和兴趣”引入到 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 10 Version 1909 的任务栏里面。Version 1909 用户可以在任务栏上看到新的图标，点击之后能够在弹出窗口看到新闻、天气等个性化内容，用户也可以根据自己的偏好进行设置。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0423/a51d988659891b5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0423/a51d988659891b5.jpg" alt="y97oyv4p.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在我们的测试中，我们观察到这个功能是 Windows Shell 体验的一部分，这意味着它不能通过服务器端的更新向用户推出。这也意味着它不能被完全从系统中删除。当然，这个功能是可选的，用户可以隐藏它，但它不能被卸载。</p>
