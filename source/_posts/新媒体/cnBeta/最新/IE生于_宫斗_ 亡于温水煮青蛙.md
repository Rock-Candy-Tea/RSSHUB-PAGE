2021年的“520”这一天，微软宣布将在2022年6月15日彻底停止 IE 浏览器。临近尾声，微软 Edge 浏览器官微最近也发文预告，IE 即将和我们再见，IE 曾走过的道路将由 Edge 浏览器来延续。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0518/b3c0df499ea9aed.jpg" referrerpolicy="no-referrer"><br></p><p>IE 并不是时代的眼泪。它曾在世纪之交叱咤风云，并一统浏览器市场。岁月蹉跎，IE 浏览器从很多人的首选浏览器变成了最好的 “Chrome 下载器”。</p><p>孟子曰，生于忧患死于安乐。IE 的发展历程，大抵如此，它的诞生就如同互联网时代的宫斗剧。只不过，历经过勾心斗角、王道征途的 IE，最终走进了“Bad End”。</p><p><img src="https://static.cnbetacdn.com/article/2022/0518/461f703a407a992.png" referrerpolicy="no-referrer"><br></p><p><strong>IE 与网景的诞生</strong></p><p>世界上第一款浏览器诞生于1990年，采用命令行形态。尽管当时电脑与互联网还远未普及，但要让更多人接触到互联网，图形界面是必要的。1993年，美国国家超级电脑应用中心（NCSA）的两位程序员马克·安德森、埃里克·比纳设计 Mosaic 浏览器，意为马赛克，这是最早的搭载图形界面的浏览器之一。</p><p><strong>Mosaic 就是这场戏中的“皇阿玛”。</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0518/8fb73cddbf7a335.jpg" referrerpolicy="no-referrer"><br></p><p>通过商业授权的形式，Mosaic 衍生出多款浏览器。其中一位是 Spyglass 公司，该公司推出的 Spyglass Mosiac 浏览器，是 IE 的直亲。1994年，微软购买 Spyglass Mosaic 的授权许可进行重新设计。1995年，IE 浏览器降生。</p><p><img src="https://static.cnbetacdn.com/article/2022/0518/8f02409809dd857.png" referrerpolicy="no-referrer"><br></p><p>另一位主角是 Netscape，中文名为网景，由 Mosaic 的设计者之一马克·安德森创建，可以说是 Mosaic 正统继承者。1994年，网景推出 Netscape Navigator 浏览器，这里称为网景浏览器。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0518/9acddd7fbd72342.png"><img data-original="https://static.cnbetacdn.com/article/2022/0518/9acddd7fbd72342.png" src="https://static.cnbetacdn.com/thumb/article/2022/0518/9acddd7fbd72342.png" referrerpolicy="no-referrer"></a><br></p><p><strong>IE 和网景之间，一场旷日持久的明争暗斗开始了。</strong></p><p><img src="https://static.cnbetacdn.com/article/2022/0518/50feddf187374f8.png" referrerpolicy="no-referrer"><br></p><p><strong>IE 的称霸之路</strong></p><p>最初，网景遵循互联网时代的呼声，宣布网景浏览器对非商业用途免费提供。这让网景一呼百应，在互联网时代占据先机，<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>的 Mac 电脑也选择网景作为默认浏览器。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0518/0dd00303ce69304.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0518/0dd00303ce69304.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0518/0dd00303ce69304.jpg" referrerpolicy="no-referrer"></a><br></p><p>当然了，IE 毕竟是微软的亲儿子，实力雄厚。1995 年，Windows 95 发布，整合了 IE 浏览器作为默认浏览器。随着 Windows 95 在第一个年热销 4000 万份，IE 开始了对网景的奋起直追。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0518/d838e8ff50a6748.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0518/d838e8ff50a6748.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0518/d838e8ff50a6748.jpg" referrerpolicy="no-referrer"></a><br></p><p><strong>微软这招的高明之处不只是捆绑。</strong>Windows 95 系统是需要付费购买的，但预装的 IE 浏览器是免费赠品。来自 IE 直接销售收入被视为零，微软只需支付给 Spyglass 最低额度的授权费而无需分红。Spyglass 忍无可忍之下拿出合约来威胁微软，微软只是赔了800万美元“安抚费”便息事宁人。</p><p>目光回到这场大戏，这中间发生一段小插曲。IE 4 发布的1997年，微软在旧金山发布现场立起了象征 IE 的“e”标志，附文“来自 IE 团队......我们爱你”。第二天，这个标志就被网景员工推倒，以自家吉祥物“Mozilla”恐龙雕像取而代之，并加上了网景与 IE 的市场份额对比——72比18。</p><p>这就是当时浏览器战场的一个缩影。</p><p><img src="https://static.cnbetacdn.com/article/2022/0518/075f9144e583792.jpg" referrerpolicy="no-referrer"><br></p><p>图自网络</p><p>但就是在 IE 4 版本开始，微软“杀疯了”。到1999年 IE 5 发布前夕，IE 已基本和网景平起平坐。</p><p>财大气粗的微软最终赢下了第一场浏览器之争。网景体量太小，收入也基本依赖于网景浏览器这一款产品，输掉了这场“持久战”。1999年，网景以42美元卖身于美国在线（AOL），微软和 IE 浏览器一统天下。</p><p><img src="https://static.cnbetacdn.com/article/2022/0518/a52c41cb407c074.png" referrerpolicy="no-referrer"><br></p><p><strong>天下归于 Chrome</strong></p><p>网景倒下之后，天下已无人能与 IE 抗衡。Windows 的流行，让 IE 的市场份额超过了90%。</p><p><img src="https://static.cnbetacdn.com/article/2022/0518/dde70373d016d90.png" referrerpolicy="no-referrer"><br></p><p><strong>绝对的领先优势之下，IE 的创新慢了下来，自身累计的风险也开始走向爆发。</strong>用户们发现，IE 变得越来越慢，越来越卡，安全问题日渐显著。曾经的世界最先进浏览器，还是经不起微软这么一作。</p><p>浏览器战场再一次群雄并起。网景的精神继承者 Mozilla 于2002年推出“凤凰”Phoenix 浏览器，后来更名为“Firefox”Firefox。苹果则是卧薪尝胆打造出 Safari，掌握了 Mac 默认浏览器的大权。在第一次浏览器大战中袖手旁观的Google，一鸣惊人，基于开源 Chromium 项目推出 Chrome 浏览器。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0518/f82741270ef3796.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0518/f82741270ef3796.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0518/f82741270ef3796.jpg" referrerpolicy="no-referrer"></a><br></p><p><strong>一切如同温水煮青蛙，盛极而衰的 IE，最终变成了微软扶不起的阿斗。</strong>201年，微软终究坐不住了，随着 Windows 10推出 Edge 浏览器；再后来，微软将 Edge 浏览器的内核换成了 Chrome 同款的 Chromium 内核，终结了自己从 IE 4 以来保有的自有浏览器内核历史。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0518/48f329f4a2f909b.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0518/48f329f4a2f909b.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0518/48f329f4a2f909b.jpg" referrerpolicy="no-referrer"></a><br></p><p>值得一提的是，微软转投 Chromium 之后，为 Code 团队工作的微软项目经理 Kenneth Auchenberg 甚至喊话称，Mozilla 是时候放弃自家 Firefox 的 Gecko 引擎了，Web 世界已经处于 Chromium 的统治。</p><p>此间乐，不思蜀？</p><p><img src="https://static.cnbetacdn.com/article/2022/0518/f1f78864fdd9d86.jpg" referrerpolicy="no-referrer"><br></p><p>总之，IE 的历史使命已经结束了。除了某些万年不更新的网站，如今绝大部分网站已经不再强行要求配合 IE 使用。而对于那些只认 IE 的网站，微软新 Edge 浏览器提供了 IE 兼容模式作为解决方法。Windows 11系统目录里那个无法直接打开的 IE，正是为此服务的。</p><p>没有多少人会怀念 IE 浏览器。我们怀念的，或许只是从 IE 开始触摸的互联网时代。</p>
