赶在英特尔 12 代 Alder Lake 桌面 CPU 发布之前，苹果在北京时间周二凌晨 1 点的发布会上，隆重推出了 2021 款 Apple Silicon MacBook Pro，并且用较大篇幅介绍了能效比更加出众的 M1 Pro 与 M1 Max 芯片组。<strong>在 14 / 16 英寸机型上首发的新一代定制芯片，意味着苹果向着“两年内从 Intel 芯片过渡”的目标又大幅迈进了一步。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1019/56c379537dbff1f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1019/56c379537dbff1f.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：Apple <a href="https://www.apple.com/newsroom/2021/10/introducing-m1-pro-and-m1-max-the-most-powerful-chips-apple-has-ever-built/" target="_self">官网</a>）</p><p>与去年发布的 M1 类似，今年的 M1 Pro 和 M1 Max，也在同一型号名称下提供了不同的 CPU / GPU 核心数量。</p><p>比如 M1 Pro 就有两款 SKU，其中一款为 8 核 CPU + 14 核 GPU，另一款则是 10 核 CPU + 16 核 GPU 。</p><p><a href="https://static.cnbetacdn.com/article/2021/1019/ec367d852e8a538.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1019/ec367d852e8a538.jpg" referrerpolicy="no-referrer"></a></p><p>然后 M1 Max 又较 M1 Pro 在峰值性能等方面有相当大的提升，特别是对专业视频剪辑至关重要的 ProRes 硬件加速器部分，内存带宽也从 200 GB/s 翻倍到了 400 GB/s 。</p><p>从去年的 M1、到最新的 M1 Pro / M1 Max，<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>都不愿对 Apple Silicon 设备上缺失的独显或外置 eGPU 支持发表任何看法。而是选择了增强内置 GPU 的性能潜力，并吹嘘最大可调用 64GB 的统一内存。</p><p><a href="https://static.cnbetacdn.com/article/2021/1019/eca9986352b77cd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1019/eca9986352b77cd.jpg" referrerpolicy="no-referrer"></a></p><p>得益于统一内存架构，GPU 和 CPU 能够共享所有内存资源，让 SoC 在 CPU 计算和图形任务处理时能够更快地传递数据。</p><p>相比之下，Intel 的 UMA 限制了 GPU 可访问的内存资源数量。而基于 PCIe 独显的 Re-Bar 又只允许 CPU 单向访问显存，不允许 GPU 反过来访问 CPU 使用的这部分内存。</p><p>当然，如果你使用的应用程序对 CPU 和 GPU 都提出了较高的要求（比如复杂的 3D 模拟），Apple Silicon Mac 还是难以与扩展性更强的 x86 工作站拉开差距的。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1191935.htm" target="_blank">苹果新的M1 Pro/Max处理器将其基于Arm的自研芯片性能提升到新的高度</a></p></div>
