零几年的一个夏天。上大学的堂哥带着笔电来我家过暑假。在他电脑桌面上，我看到个酷炫的图标：一只狐狸盘旋在地球上空。堂哥说，这玩意叫Firefox，Mozilla Firefox。那会我才知道，噢，原来还有其他的软件可以“ 上网 ”啊？<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0228/f0cc3f59f3be3db.png" referrerpolicy="no-referrer"><br></p><p>打开 Firefox，第一感觉就是界面新鲜，速度似乎比我家电脑的浏览器（ 嗯，就是 IE ）快很多。</p><p>最关键的是，它打开一个新网页，是在原窗口上建立一个新标签页。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0228/7ba0d21ee464ed5.png"><img data-original="https://static.cnbetacdn.com/article/2022/0228/7ba0d21ee464ed5.png" src="https://static.cnbetacdn.com/thumb/article/2022/0228/7ba0d21ee464ed5.png" referrerpolicy="no-referrer"></a><br></p><p>而不是像 IE 那样，用新窗口打开新网页，然后塞爆你的任务栏。</p><p>拥有了自己的电脑后，我装的第一个浏览器就是Firefox。</p><p>但，年轻人嘛，总是乐于体验新鲜事物。</p><p>所以和 Firefox 的缘分没持续多久，我就去用了搜狗、QQ、欧朋、360，最后停在了 Chrome 。</p><p>而就在前两天，世超看到一篇帖子，说 Firefox 市场份额只有 4% 了，就要没了。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0228/11187363813c036.png"><img data-original="https://static.cnbetacdn.com/article/2022/0228/11187363813c036.png" src="https://static.cnbetacdn.com/thumb/article/2022/0228/11187363813c036.png" referrerpolicy="no-referrer"></a><br></p><p>WTF？</p><p>我还记当时只要有点冲浪经验的人，用的都是Firefox。</p><p>但惊讶过后，世超更多的是感触。</p><p>因为如果没有 Chrome ，当年屠杀 IE 的勇士，或许就是 Firefox 。</p><p><a href="https://img1.mydrivers.com/img/20220228/de1fdc6e-4d4f-477c-b9a7-6f2bd51e060f.png" target="_blank"><img src="https://static.cnbetacdn.com/article/2022/0228/6c982afd5a639dc.png" referrerpolicy="no-referrer"><br></a></p><p>今天，世超就和大伙儿说一说， Firefox 前生今世。</p><p>在上世纪互联网诞生初期，大家用得最多的浏览器叫 Mosaic（ 马赛克 ），因为它安装简单。</p><p>咱们先叫它一代目。</p><p>1994 年，Mosaic 开发者之一，马克·安德森自己开发了一个浏览器，叫 Mosaic Netscape 马赛克网景。</p><p>后来和老东家产生纠纷，他把浏览器名字去掉了 Mosaic ，改成 Netscape Navigator 网景导航者。</p><p><img src="https://static.cnbetacdn.com/article/2022/0228/e6db1f64703b842.png" referrerpolicy="no-referrer"><br></p><p>虽然对外声称“ 网景浏览器 ”，但内部开发时有另外一个代号——</p><p>Mozilla，由 “ Mosaic Godzilla Killa 马赛克 · 哥斯拉杀手 ” 缩写而成。</p><p>没别的意思，它就是想干掉一代目 Mosaic 。</p><p>网景吉祥物也叫 Mozilla，外表近似哥斯拉 ▼</p><p><img src="https://static.cnbetacdn.com/article/2022/0228/acb414dff8dc073.jpg" referrerpolicy="no-referrer"><br></p><p>实际情况也确实如此。</p><p>网景浏览器因为更实用、更稳定，并且支持免费试用，很快打败了 Mosaic，成为了二代目。</p><p>但，龙椅还没捂热呢，微软带着 IE 杀过来了。</p><p>虽说网景已经统治了浏览器市场，但 IE 可不是善茬。。</p><p>首先它的老大哥 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 是垄断了整个操作系统市场，哥哥混到哪，弟弟就装到哪儿。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0228/4aab3b23dbcb59d.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0228/4aab3b23dbcb59d.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0228/4aab3b23dbcb59d.jpg" referrerpolicy="no-referrer"></a><br></p><p>甚至有段时间，因为微软赞助了<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>，连 MacOS 上的默认浏览器也是 IE。</p><p>另外，微软还会搞点小动作。</p><p>比如让主机厂商在桌面上展示 IE 图标，不能展示网景的，不听话就涨价。</p><p>或者把网页设计软件买下来，让它做出的网页只适用于 IE 。</p><p>当然了，在微软铺路的同时，IE 自身也没歇着。</p><p>首先，老爹有钱，IE 直接选择永久免费。</p><p>其次，老爹有钱，功能开发很快，体验上也追上网景。</p><p>免费又够用，那用户还有什么理由不用 IE 呢？</p><p>Mozilla 脚踩 IE  ▼</p><p><img src="https://static.cnbetacdn.com/article/2022/0228/cffa9df3b8fe39b.jpg" referrerpolicy="no-referrer"><br></p><p>反观网景这边，为了在这场大战里有优势，盲目增加新功能，却忽略了稳定性，导致口碑越来越差。</p><p>就这样，网景的份额被疯狂吞噬，公司也在1998 年被“ 美国在线” 收购。。</p><p>不过，收购前，他们做了一件事——Mozilla 计划。</p><p>所有网景网络套件（ 包含浏览器、电子邮件客户端、网页开发软件等 ）改为免费、开源。</p><p>同时还成立了 Mozilla 组织，让员工和志愿者们一起开发和维护。</p><p>大概就是不服输吧。</p><p>后来这个组织确实成功开发了全新的 Mozilla 套件（ 包含浏览器、电子邮件客户端等 ），但并没有激起太大的浪花。</p><p>Mozilla套件启动页  ▼</p><p><img src="https://static.cnbetacdn.com/article/2022/0228/734aa9b5003174b.png" referrerpolicy="no-referrer"><br></p><p>2003 年 4 月 Mozilla 组织宣布，单独的软件更有未来，工作重心不再放在 Mozilla 套件上，准备重新开发一个全新的，单独的浏览器。但几个月后，美国在线突然裁掉了网景大部分员工，大楼也撤掉了网景的招牌。</p><p>就当大家以为网景没了的时候，Mozilla 组织成员和志愿者成立了 Mozilla 基金会，通过别人赞助，来维持开发计划。终于在一年后，这个全新的、独立的浏览器，搓出来了。。它就是 Firefox。Firefox 最初是叫 “ Phoenix ” ，后面因为版权纠纷改掉了。Phoenix 在西方神话里就是不死鸟，它能在自己灰烬里复活。</p><p>以不死鸟为名，想必差友们也大概能明白是什么含义了。Phoneix 时期的logo ▼</p><p><img src="https://static.cnbetacdn.com/article/2022/0228/c549b07112ae41e.png" referrerpolicy="no-referrer"><br></p><p>2004 年 11 月 Firefox  0.1 版本发布。IE 有的功能它有，IE 没有的功能，它也有，比如标签浏览，弹出窗口拦截器，下载管理器。之后的版本里，它又迅速更新了Google搜索栏，侧边栏，主题，密码管理器等功能。如果这些你不满意，他们还有一个社区，众多开发者在上面提供插件，可以随意定制自己浏览器。Firefox 2.0 ▼</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0228/c43be317c8eb7d2.png"><img data-original="https://static.cnbetacdn.com/article/2022/0228/c43be317c8eb7d2.png" src="https://static.cnbetacdn.com/thumb/article/2022/0228/c43be317c8eb7d2.png" referrerpolicy="no-referrer"></a><br></p><p>不过，最让对手最害怕的，其实是它的安全性。先不说作为一个开源程序，有大量开发人员一起寻找、修复漏洞。它还额外发布了“ 漏洞奖励 ”计划，只要你发现漏洞，就奖励 500 到 3000 美元。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0228/4bae3855917dde2.png"><img data-original="https://static.cnbetacdn.com/article/2022/0228/4bae3855917dde2.png" src="https://static.cnbetacdn.com/thumb/article/2022/0228/4bae3855917dde2.png" referrerpolicy="no-referrer"></a><br></p><p>另外 Firefox 使用了沙盒安全模块，限制了网页脚本对用户资料访问，防止受到恶意攻击。在传输资料时，使用SSL/TLS的加密方式，让用户能隐密地传输。后来它还和 Google 合作，一旦用户访问钓鱼网站时，就会发送提示。</p><p>在那个病毒猖獗的年代，人人自危，一个够安全的浏览器自然更受欢迎。</p><p><img src="https://static.cnbetacdn.com/article/2022/0228/d5e0feebfeccc25.png" referrerpolicy="no-referrer"><br></p><p>在发布不到一年，Firefox 下载量就超过一亿。获得的好评也不计其数，04 年被 Fobes 评为最佳浏览器，05 年被杂志《 PC World 》列入 2005 年最佳百大产品。到了 2009 年，Firefox 在全球市场拥有了 25%的份额，在部分国家甚至达到了 86%，几乎就是全民 Firefox。Firefox-绿色 IE-蓝色 ▼</p><p><img src="https://static.cnbetacdn.com/article/2022/0228/db0f85a01ac7614.png" referrerpolicy="no-referrer"><br></p><p>微软当时也很慌，内部把 Firefox 列为有威胁的对手。总之不管怎么看，Firefox 复仇之路似乎要走向成功了。但，王者归来的场景，它只演到了一半，就被摁下了暂停键。</p><p>2008 年，Chrome 正式发布了。Firefox 势头很快被压了下来，最终在 1、2 年后，迎来了下跌。。关于为什么 Firefox 会走下坡路，不少人说是它自己没做好。有人说，Firefox 上功能不太喜欢，喜欢的功能 Firefox 没有。也有人说它傲慢，用户想要的功能它不加。</p><p><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0228/15fd0a8cd3c271e.png"><img data-original="https://static.cnbetacdn.com/article/2022/0228/15fd0a8cd3c271e.png" src="https://static.cnbetacdn.com/thumb/article/2022/0228/15fd0a8cd3c271e.png" referrerpolicy="no-referrer"></a><br></p><p>还有人说，它一直在接近 Chrome，那还不如用 Chrome 。其实这些真不足以让 Firefox 一直走下坡路，功能方面不都是萝卜白菜各有所爱么。。我倒是觉得，不是 Firefox 不行，而是 Chrome 太强了。</p><p>大家去用 Chrome，有个重要原因就是Google全家桶。有了 Google 账号，就可以直接登录 Chrome ，在多端同步，同时这个账号还能登录旗下所有 App，这便捷程度不是一点半点。另外 Chrome 页面简洁，版本迭代快，新功能多，拥有一个顶级的 Chromium 内核，这些都是它巨大的优势。。凭借这些，Chrome 出来后简直异军突起，换谁都挡不住。</p><p><img src="https://static.cnbetacdn.com/article/2022/0228/204ae94c572a9a5.png" referrerpolicy="no-referrer"><br></p><p>虽说现在的Firefox和众多浏览器一样，一直被 Chrome 统治着，甚至眼看就要没了。但纵观历史，我还是挺相信 Mozilla 的。从浏览器大战里被 IE 挫伤，为了争回市场，还把所有产品免费开源，成立 Mozilla 组织。到网景部门被大规模裁员，为了维持开发，同一天紧急成立 Mozilla 基金会。</p><p>就凭着这股几十年来绝境中不断“ 重生 ”的精神，我想，哪怕某天真没人用 Firefox 了，排名也被分在“ others ”，它也不会放弃。</p><p>而是会和最初的名字一样，从灰烬里重生。</p>
