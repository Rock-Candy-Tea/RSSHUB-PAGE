在微软发布 Windows 11 系统之后，该公司自然期望所有用户都使用自家的 Microsoft Edge 浏览器。用户可以直接在 Edge 中将其设置为默认浏览器，而像 Chrome、Vivaldi 这样的浏览器需要到系统的设置应用中进行设置。<strong>不过在最新的 Chrome Dev 106.0.5221.0 版本中免除了这些额外步骤，用户直接点击“设置为默认”，就可以将 Chrome 设置为默认浏览器。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0825/1aff245b004c012.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0825/1667f346a66bd7e.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0825/a28dd388fe882a7.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0825/ece0d9dc15fdfaf.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0825/3291476bf7692e2.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">这意味着您无需前往 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 设置页面即可将默认浏览器从 Microsoft Edge 更改为 Chrome。当您打开 Google Chrome 时，您会从 Google 收到一条消息，指出 Chrome 不是您的默认浏览器，并且您是否要将其设为默认浏览器。您需要做的就是单击“设置为默认值”按钮，一切就绪。该功能最初是由 Reddit 用户 Leopeva64-2 发现的。</p><p style="text-align: left;">但是，值得指出的是，Chrome Dev 106.0.5231.2 目前提供了无需进入设置页面即可将 Chrome 设置为默认浏览器的功能。 Google 应该在未来将这项功能带给更多的 Chrome 用户，但我们不知道这会发生。同时，那些没有运行最新的 Dev Channel Chrome 浏览器的人可以通过转到设置 > 应用程序 > 默认应用程序将 Chrome 设置为默认浏览器，其中将出现更改默认浏览器的选项。</p>
