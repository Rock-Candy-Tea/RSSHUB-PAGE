在没有任何重大宣布的情况下，微软发布了一个轻量级的 Windows 11 系统，只是该系统并不是为终端用户准备的。该系统基于命令行，称之为“<a href="https://docs.microsoft.com/zh-cn/windows-hardware/manufacture/desktop/validation-os-overview?view=windows-11" target="_blank">Microsoft Validation OS</a>”，主要针对软硬件供应商、开发人员和技术人员，目的是帮助诊断和解决问题。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0615/9c0b33701d5cc6b.webp" alt="gz8ia2jl.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">对于 Microsoft Validation OS，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>官方的定义是</p><blockquote style="text-align: left;"><p style="text-align: left;">Microsoft Validation OS 是一个轻量级的、快速的、可定制的基于 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 的操作系统，你可以在工厂车间使用，在 Windows 设备制造过程中诊断、减轻和修复硬件缺陷。验证操作系统可以启动到命令行环境，以提高工厂车间的可靠性，并支持运行 Win32 应用程序，使从早期硬件引进到零售操作系统和应用程序开发的过渡更加顺利。</p></blockquote><p style="text-align: left;">正如你在微软的上述描述中所看到的，Validation OS 支持 Win32 应用程序，这是为了帮助诊断一个基于 Windows 11 的开发中项目。这是因为Win32 API提供了对系统硬件的直接访问，并有各种工具用于诊断。</p><p style="text-align: left;">更多细节可以访问</p><p style="text-align: left;"><a href="https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/validation-os-mount-and-customize?view=windows-11">Mount and customize a Validation OS image</a></p><p style="text-align: left;"><a href="https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/validation-os-boot?view=windows-11">Setup a bootable Validation OS installation</a></p><p style="text-align: left;"><a href="https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/validation-os-optional-packages?view=windows-11">Validation OS feature packages reference</a></p><p style="text-align: left;"><a href="https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/validation-os-run-an-app-on-boot?view=windows-11">Launching an app or command when Validation OS starts</a></p><p style="text-align: left;"><a href="https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/validation-os-debug-apps?view=windows-11">Debugging Apps in Validation OS</a></p><p style="text-align: left;">如果你想尝试一下，微软Validation OS可以在<a href="https://docs.microsoft.com/zh-cn/windows-hardware/manufacture/desktop/validation-os-overview?view=windows-11" target="_blank">这个链接中</a>以Windows Image（.WIM）文件的形式下载，适用于ARM64和<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a>64架构。</p>
