在 Red Hat Enterprise Linux 8.4 上周发行之后，各方基于 RHEL 的下游也快速跟进更新了自己的发行版本。甲骨文（Oracle）昨日宣布了适用于 x86_64 和 AArch64 的 <a href="https://blogs.oracle.com/linux/announcing-the-release-of-oracle-linux-8-update-4" target="_blank">Oracle Linux 8 Update 4</a>，这是包含 RHEL 8.4 更新内容的 Oracle Linux 最新发行版本。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0528/0303701b3cbc84a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0528/0303701b3cbc84a.jpg" alt="s47vczef.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">此外，Oracle 继续将“Unbreakable Enterprise Kernel”作为较新的 Linux 内核版本来替代 Red Hat Compatible Kernel (RHCK) 。另外，在第一季度末，AlmaLinux 作为 RHEL 的 1:1 分叉版本发布，并延续了以前 CentOS 的发布精神。AlmaLinux 得到了 CloudLinux 公司的支持，现在已经发布了基于RHEL 8.4的更新。</p>
