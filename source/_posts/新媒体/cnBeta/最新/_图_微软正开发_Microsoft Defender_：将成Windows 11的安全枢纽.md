微软正计划为 Windows 11 系统开发名为“<a href="https://www.microsoft.com/zh-cn/p/microsoft-defender-preview/9p6pmztm93lr" target="_blank">Microsoft Defender</a>”的安全中心。目前，这款 Windows 11 新安全应用的信息并不多，但似乎 Microsoft Defender 将成为所有连接设备/产品的安全枢纽/仪表板。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1101/c47dcf3c6410753.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1101/c47dcf3c6410753.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1101/c816b1705dbb36b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1101/c816b1705dbb36b.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1101/7118795f29e69ff.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1101/7118795f29e69ff.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Windows 11/10 系统中已经内置了 Windows Security 应用程序，允许任意用户查看安全服务的状态，并打开/关闭安全功能，如防火墙设置和保护免受不需要的程序。</p><p style="text-align: left;">根据目前掌握的信息，全新的 Microsoft Defender 将会基于 WinUI 和 Fluent Design 设计开发，旨在为所有连接的账户提供"无缝、个性化的保护"。从外观上看，Microsoft Defender 预览版可能是你设备的安全仪表板。</p><p style="text-align: left;">官方商店的描述和该应用程序的 .<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://msi-pc.jd.com/" target="_blank">MSI</a>xbundle 内的材料表明，它将不限于 Windows 11。在 Microsoft Defender 的 .msix 文件中，科技媒体 Windows Latest 发现了 Android <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>、<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fiphone%2F" target="_blank">iPhone</a> 和 Mac 的图标，这表明新应用将让用户查看连接的操作系统的状态，包括 iOS/macOS 和 Android。</p>
