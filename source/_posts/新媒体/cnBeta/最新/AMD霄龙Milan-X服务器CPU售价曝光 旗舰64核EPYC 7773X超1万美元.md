在昨日曝光了 AMD 霄龙 Milan-X 服务器 CPU 的几款 SKU 后，@Momomo_US 刚又在 Twitter 上晒出了 EPYC 7773X 的定价。<strong>页面截图来自 Zones 的零售清单，可知这颗 64 核心的 EPYC Milan-X 处理器的售价将超出 10000 美元。</strong>即便如此，与同级对应的 EPYC Milan CPU SKU 相比，EPYC 7773X 还是颇具吸引力的。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0827/47a1ea6878cb16f.png" referrerpolicy="no-referrer"></p><p>众所周知，EPYC Milan-X 服务器 CPU 同样包括了 64 / 32 / 24 / 16 核心的 SKU 。其采用了与当前 EPYC Milan 服务器 CPU 相同的 Zen 3 内核，但在 CCD 模块顶部直接融合了高带宽的 DRAM 缓存。</p><p>除了 EPYC Milan-X 服务器 CPU 产品线，这项 3D V-Cache 技术也将在基于 Zen 3 核心架构的锐龙台式 CPU 上得到应用。</p><p><img src="https://static.cnbetacdn.com/article/2021/0826/2aecc4a5cf99d79.png" referrerpolicy="no-referrer"></p><p>在昨日的爆料中，@Momomo_US 已经晒出了不同核心数量的 Milan-X 服务器 CPU 的 OPN 代码：</p><blockquote><p>● EPYC 7773X -- 64 核 / 128 线程（100-000000504）</p><p>● EPYC 7573X -- 32 核 / 64 线程（100-000000506）</p><p>● EPYC 7473X -- 24 核 / 48 线程（100-000000507）</p><p>● EPYC 7373X -- 16 核 32 线程（100-000000508）</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0827/42b95bc5d0e8727.png" alt="1.png" referrerpolicy="no-referrer"></p><p>EPYC 7773X 具有 64C / 128T，Zones 列出的价格为 10476.99 美元（约 6.8 万 RMB），较同一零售商列出的 EPYC Milan 7763 旗舰 CPU 的价格高出了 11% 。</p><p><img src="https://static.cnbetacdn.com/article/2021/0827/6a650bc6d6534ab.png" alt="2.png" referrerpolicy="no-referrer"></p><p>EPYC 7573X 为 32C / 64T，零售价挂出的价格为 6654.99 美元（约 4.3 万 RMB），较 Zones 在售的 EPYC 7543 型号溢价 5% 。</p><p><img src="https://static.cnbetacdn.com/article/2021/0827/92cb803ba000ad8.png" alt="3.png" referrerpolicy="no-referrer"></p><p>EPYC 7473X 为 24C / 48T，报价为 4643.99 美元（约 3 万 RMB），较 EPYC 7443 的两倍。</p><p><img src="https://static.cnbetacdn.com/article/2021/0827/a2873295fc110ae.png" alt="4.png" referrerpolicy="no-referrer"></p><p>奇怪的是，EPYC 7373X 为 16C / 32T，但 Zones 的挂价竟高达 5595.99 美元（约 3.6 万 RMB），是 EPYC 7343 的三倍。</p><p><img src="https://static.cnbetacdn.com/article/2021/0827/3ca5f2e478649c3.png" referrerpolicy="no-referrer"></p><p>尽管没有列出更详细的规格，但这至少意味着 Milan-X 服务器 CPU 将很快上市。</p>
