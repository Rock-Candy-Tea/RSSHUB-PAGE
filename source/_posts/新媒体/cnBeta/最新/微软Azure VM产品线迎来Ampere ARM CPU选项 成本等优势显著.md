<strong>本周一，微软宣布了 Azure VM 服务的一项更新，为广大客户带来了基于 Ampere ARM CPU 的选项。</strong>由 Hari Pulapaka 在 Twitter 上分享的截图可知，专为 Ampere Computing 开发的这款处理器，现已能够为多种类型的虚拟机提供支撑，包括 Web 服务器、开源数据库、以及游戏和媒体服务器等。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0405/14b71362184aee9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0405/14b71362184aee9.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图自 @TheRealHariP / Twitter）</p><p>近年来，这家科技巨头一直在积极将 ARM CPU 纳入其生态系统。此前，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>已经为 Azure 虚拟机提供几乎横跨所有主流 Linux 发行版的支持。</p><p>现在，该公司又更进一步地推动 ARM 架构硬件的采用，不过初期仅在美国西部二号（West US 2）、美西中部（West Central US）、西欧（West Europe）等几个 Azure 区域提供预览体验。</p><p><img src="https://static.cnbetacdn.com/article/2022/0405/966e40640348b45.png" alt="2.png" referrerpolicy="no-referrer"></p><p>专注于服务器 ARM 处理器的初创企业 Ampere Computing 去年宣布，其主要签约客户中包括了美国微软和中国腾讯。</p><p>周二早些时候，Azure 主机操作系统兼 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> OS 平台项目总监 Hari Pulapaka，又在 Twitter 上确认了预览版本的发布。</p><p><img src="https://static.cnbetacdn.com/article/2022/0405/60217629540099c.png" alt="3.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：Microsoft <a href="https://azure.microsoft.com/en-us/blog/now-in-preview-azure-virtual-machines-with-ampere-altra-armbased-processors/" target="_self">Azure Blog</a>）</p><p>据悉，Windows 将成为在 Ampere Computing 的 ARM CPU 上运行的底层托管操作系统。</p><p>不过除了 Windows 11 专业版 / 企业版，当前 Azure VM 预览也支持 ARM 上的 Ubuntu 和 CentOS 等 Linux 发行版。</p><p>随着时间的推移，拥有 Azure 的 Ampere Altra ARM CPU 还有望支持更多的 OS，比如红帽企业版（RHEL）、SUSE 企业服务器版、Debian、AlmaLinux、以及 Flatcar 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0405/6118215c82066c1.png" alt="4.png" referrerpolicy="no-referrer"></p><p>值得一提的是，在横向扩展工作负载方面，微软宣称基于 Ampere Altra ARM 处理器的 Azure VM，拥有较同级 x86 虚拟机高出 50% 的性价比。</p><p>客户可最多配置 64 个虚拟 CPU 核心，并可灵活选择每个 vCPU 的内存大小（2 / 4 / 8GB）、高达 40 Gbps 的网络带宽、以及高性能本地 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 存储。</p><p>至于 Dpsv5 和 Epsv5 Azure VM 系列产品线，也将运行在主频高达 3.0 GHz 的 Ampere Altra ARM 处理器平台上。</p>
