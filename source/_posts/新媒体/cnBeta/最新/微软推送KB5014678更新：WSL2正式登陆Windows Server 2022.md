<strong>Windows 开发者平台项目经理 Craig Loewen 在周二的一篇博客文章中宣布，Windows Server 2022 现已能够运行 WSL 2 。</strong>一个月前，微软已通过 GitHub 平台向早期测试者们提供了尝鲜体验。不过随着该公司开始推送 KB5014678 更新，所有 Windows Server 2022 用户都可体验新版 Windows Linux 子系统了。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0622/bcad4fe458d9cb3.png" alt="3.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（截图 via <a href="https://techcommunity.microsoft.com/t5/itops-talk-blog/wsl2-now-available-on-windows-server-2022/ba-p/3447570" target="_self">Tech Community</a>）</p><p>如果尚未等到推送，还请移步至“设置 → <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Update”并点击“检查更新”，以获取 Windows Server 2022 的 WSL 2 功能更新。</p><p><img src="https://static.cnbetacdn.com/article/2022/0622/9f83289d0aa1e6b.png" alt="0.png" referrerpolicy="no-referrer"></p><p>KB5014678 更新安装完成后，你只需使用管理员权限打开 PowerShell，然后输入并执行“wsl --install”命令。</p><p><img src="https://static.cnbetacdn.com/article/2022/0622/23c0b002a2562cd.png" alt="1.png" referrerpolicy="no-referrer"></p><p>如需在安装时需要更多帮助，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>也有贴心地准备一份特殊的文档（<a href="https://docs.microsoft.com/en-gb/windows/wsl/install-on-server" target="_self">传送门</a>）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0622/f3beced47ce5b14.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0622/f3beced47ce5b14.png" alt="wslwindowsserver.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（截图 via <a href="https://devblogs.microsoft.com/commandline/wsl-2-distros-are-now-supported-on-windows-server/" target="_self">Dev Blogs</a>）</p><p>若在使用 WSL2 时遇到任何技术问题，微软也希望大家积极在 <a href="https://github.com/microsoft/wsl" target="_self">GitHub</a> 上提交反馈，或通过 <a href="https://twitter.com/i/lists/1131397802204864512/members" target="_self">Twitter</a> 渠道与 WSL 开发团队互动。</p>
