上周五，美国证券交易委员会（SEC）宣布了一项针对澳大利亚公民 Craig Sproule 的指控 —— <strong>这位加密货币初创企业 Crowd Machine 首席执行官承诺的“去中心化”AWS 替代方案，对数以百万计的潜在投资者构成了欺诈。</strong>据悉，Crowd Machine 宣称要建立一个基于用户计算机的全球云计算网络，但实际上，该公司向南非金矿输送了 580 万美元的资金 —— 因为投资者的加密代币已变得一文不值。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0111/8d01cb8d36cab93.jpg" referrerpolicy="no-referrer"></p><p>SEC <a href="https://storage.courtlistener.com/recap/gov.uscourts.cand.390193/gov.uscourts.cand.390193.1.0.pdf" target="_self">指出</a>，Sproule 通过首次代币发行（ICO）筹集了至少 3300 万美元，宣称要构建一个去中心化的“众包计算机”（Crowd Computer）平台。</p><p>Sproule 还夸口其平台已通过财富 500 强企业的“实战考验”，称之可与<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a> Azure 和亚马逊云服务（AWS）一较高下，且 Crowd Machine Compute Tokens（CMCTs）代币的价值在 10~600 美元之间。</p><p>然而有投诉称，Sproule 作出了虚假和误导性的陈述，且该平台从未如先前宣传的那样建立。相反，Crowd Machine 将其系统托管在了集中式的服务器上。</p><p>此外 Crowd Machine 的 ICO 本身未在 SEC 提交注册，更别提该公司未核实投资者是否按照法律要求获得认可。</p><p>当 Sproule 试图筹集资金时，据说 Crowd Machine 在未向投资者通报的情况下，将 580 万美元输送到了南非的金矿，但却将此举描述为贷款或股权购买。</p><p>然而 SEC 的报告指出，金矿开采业务并未带来任何营收，而 Crowd Machine 也在开始筹款几个月后遇到了麻烦。</p><p>2018 年中，报道称黑客窃取了大量 CMCT，导致交易中断和代币价格暴跌。尽管后续有两名男子被捕，但 CMCT 的二级市场几乎消失，该代币曾许诺持有者的任何价值也随之灰飞烟灭。</p><p>投诉者称，CMCT 每个代币的售价落在了 0.03 ~ 0.22 美元之间，且二级交易市场上的价格也从未超过 0.18 美元。</p><p>最终 Sproule 只得同意 SEC 设定的条款 —— 尽管不要求其承认不法行为，但至少给他设置了证券市场（包括加密货币资产）的永久禁入 —— 此外他必须支付 19.5 万美元的罚款，并同意永久禁用其持有的任何 CMCT 。</p>
