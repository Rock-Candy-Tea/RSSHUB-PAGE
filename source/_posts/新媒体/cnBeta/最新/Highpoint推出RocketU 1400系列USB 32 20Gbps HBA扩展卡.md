<strong>Highpoint 刚刚推出了 RocketU 1400 系列 USB 3.2 HBA 直通扩展卡，专为需要强大且高速的连接解决方案的专业应用而设计。</strong>比如 3D 成像 / 动作捕捉平台、高清安防系统、ADAS 传感器套件、物联网设备，以及频谱分析仪等。更棒的是，Highpoint 还确保每个专用接口都可享有 USB 3.2 控制器的 Gen 2×2（20 Gbps）速率。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0630/4d8f86bb7405dae.png" alt="1.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">RocketU 1411C（<a href="https://www.highpoint-tech.com/product-page/rocketu-1411c" target="_self">RU1411C</a>）</p><p>与前几代最多仅提供四个扩展端口的扩展卡不同，RocketU 1400 系列的控制器，最高可提供八个独立的 USB Type-C 端口 @ 16000 MB/s 的传输带宽。</p><p>USB 设备具有性价比高、易于维护、能够与任何现代计算机平台 / 操作系统实现普遍兼容，此外相关解决方案能够轻松集成到行业标准的基于 PC 的硬件平台中。</p><p>无论是需要高性能连接解决方案的工业、科学、还是媒体应用 —— 比如机器学习（ML）、人工智能（AI）、自动化 / 机器人装配线的质量控制系统、高级驾驶辅助 / 自动驾驶（ADAS / AD）系统、还是持续监测输入的视频监控 / 安防传感器 —— 20 Gbps 的单口并发速率都可轻松满足。</p><p><img src="https://static.cnbetacdn.com/article/2022/0630/0cf447f5f177b16.png" alt="2.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">RocketU 1444C（<a href="https://www.highpoint-tech.com/product-page/rocketu-1444c" target="_self">RU1444C</a>）</p><p>每个独立的设备通道，都由专用且强大的 PCIe 3.0 主机接口转 USB 3.2 20Gbps 桥接芯片提供支撑。</p><p>其中 RocketU 1411C 型号为单口 USB 3.2 Gen 2×2 @ PCIe 3.0 x4，四口 / 八口的 Rocket 1444C / 1488C 则是 PCIe 3.0 x16 主机接口。</p><p>除了专用带宽，每个端口还可提供高达 7.5W 的功率（用于设备操作或充电），且扩展卡本身也相当易于安装和配置（即插即用）。</p><p>原生支持业内标准的任意 Intel、<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a>、ARM 硬件平台，以及所有现代 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 操作系统 / Linux 发行版。</p><p><img src="https://static.cnbetacdn.com/article/2022/0630/46b4cdbd1144d3d.png" alt="3.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">RocketU 1488C（<a href="https://www.highpoint-tech.com/product-page/rocketu-1488c" target="_self">RU1488C</a>）</p><p>最后，Highpoint 为 RocketU 1400 系列 USB HBA 主机扩展卡提供了一系列经过严格认证的数据线，完全符合当前的 USB 标准，以确保最大的传输性能、安全连接和易于集成。</p><blockquote><p>RocketU 1411C（RU1411C）-- 1 个 USB 20G Type-C 端口：售 59.00 美元（约 395 RMB）</p><p>RocketU 1444C（RU1444C）-- 4 个 USB 20G Type-C 端口：售 399.00 美元（约 2670 RMB）</p><p>RocketU 1488C（RU1488C）-- 8 个 USB 20G Type-C 端口：售 699.00 美元（约 4678 RMB）</p></blockquote>
