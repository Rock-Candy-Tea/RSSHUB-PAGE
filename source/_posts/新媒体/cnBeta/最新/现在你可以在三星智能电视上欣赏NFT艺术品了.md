在今年的 CES 大展上，三星宣布和 NFT 拍卖网站 Nifty Gateway 展开合作，将 NFT 引入到电视中。<a href="https://www.prnewswire.com/news-releases/nifty-gateway-partners-with-samsung-electronics-to-develop-an-industry-first-smart-tv-nft-platform-301513051.html" target="_blank"><strong>Nifty Gateway 在今天发布的新闻稿</strong></a><strong>中，表示正在和三星 NFT 平台进行整合，从而可以在三星 QLED、Neo QLED 电视上浏览、显示、互动 NFT。</strong>Nifty Gateway 也将在三星的 The Frame 和 Micro LED 电视上推出一个应用程序。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0401/dafcff878c1ba7a.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">也就是说，你可以在<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a><a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_76344%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">电视</a>上展示 NFT 艺术品，未来或许可以购买或者出售这些 NFT 艺术品。值得注意的是，Nifty Gateway 对谁能在其平台上出售艺术品的要求比 OpenSea（大多数人熟悉的 NFT 市场）要严格得多。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0401/4ea4a6ae55668bc.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">在一个公告流中，三星重申，当你显示 NFT 时，其电视将“优化设置，以忠实地呈现艺术家的意图”。</p><p style="text-align: left;">有趣的是，这幅图像正好发挥了QD-<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_88234%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">OLED</a>显示屏的优势，尽管三星还没有具体说明这些电视是否会与Nifty Gateway整合。Nifty Gateway的整合将是三星智能中心的一部分，而且三星的意图似乎是让你把NFT作为环境艺术品来显示。</p>
