在 Chrome 97 稳定版日前发布之后，基于 Chromium 的 Microsoft Edge 浏览器 97 稳定版今天也正式发布，版本号为 97.0.1072.55。这是Edge 97进入稳定频道的第一个版本。这次更新带来了大量的修复和改进。请看下面的更新日志：<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/12/0cd5b7f6d7f48f0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/article/2021/12/0cd5b7f6d7f48f0.jpg" referrerpolicy="no-referrer"></a></p><p><strong>功能更新</strong></p><p><strong>● 当一个设备上有多个工作或<a data-link="1" href="https://c.duomai.com/track.php?k=yUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmYiJ05WZkVHdzZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyUiR" target="_blank">教育</a>账户登录时，使用当前配置文件登录网站。</strong></p><p>当一个设备上有多个工作或学校账户登录时，用户将被要求从账户选择器中选择一个账户来继续访问网站。在这个版本中，用户将被提示让 Microsoft Edge 自动用签署在当前配置文件中的工作或教育账户登录网站。用户可以在"设置">"配置文件偏好"中打开或关闭这一功能。</p><p><strong>● 在 macOS上增加对 Microsoft Endpoint 数据丢失防护（DLP）的支持。</strong></p><p>Microsoft Endpoint DLP 政策执行将在 macOS 上原生可用。</p><p><strong>● 自动 HTTPS。</strong></p><p>用户可以在可能支持这种更安全协议的域上将导航从 HTTP 升级到 HTTPS。这种支持也可以被配置为对所有域尝试通过HTTPS交付。注意：这个功能是一个受控功能的推出。如果你没有看到这个功能，请在我们继续推出的时候再查看。</p><p><strong>● 在第三方背景下阻止 WebSQL。</strong></p><p>遗留的WebSQL功能的使用将被阻止在第三方框架中。<a href="https://docs.microsoft.com/en-us/deployedge/microsoft-edge-policies#websqlinthirdpartycontextenabled" target="_blank">WebSQLInThirdPartyContextEnabled</a> 策略在 Microsoft Edge 版本 101 之前可作为一个选择退出的选项。这一变化是发生在Microsoft Edge所基于的Chromium项目中。欲了解更多信息，请参见此<a href="https://chromestatus.com/feature/5684870116278272" target="_blank">Chrome平台状态条目</a>。</p><p><strong>● Microsoft Edge 中的引文。</strong></p><p>为研究引用资料是对学生的一个普遍要求。他们必须管理许多研究参考资料和来源，这并不是一件容易的事。他们还必须将这些引文翻译成适当的引文格式，如 APA、MLA 和 Chicago。这个新的"引文"功能，现在在Microsoft Edge的预览版中，为学生提供了一个更好的方式来管理和生成在线研究的引文。在"收藏"或"设置和更多"（Alt-F）中打开"引文"功能后，Microsoft Edge会自动生成引文，供学生以后使用，这样他们就可以专注于他们的研究。当他们完成后，他们可以很容易地将这些引文编入最终的交付物中。更多信息，请参见《<a href="https://blogs.windows.com/msedgedev/2021/11/04/preview-citations-feature-edge/" target="_blank">在Microsoft Edge中预览引文</a>》。</p><p><strong>● 控制流保护（CFG）。</strong></p><p>Microsoft Edge 将开始通过打击内存损坏漏洞和保护间接调用来支持更精细度的保护。CFG 仅支持 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 8 及更高版本。欲了解更多信息，请参阅 <a
