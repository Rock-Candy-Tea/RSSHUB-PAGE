面向 Dev 和 Beta 频道的 Windows Insider 项目成员，今天微软推出了 Windows 11 的新预览版。随本次新预览版同时推出的，还有改进的 Microsoft Store 新版本。不过今天发布的 Microsoft Store 新版仅面向 Dev 频道用户开放。<br>
