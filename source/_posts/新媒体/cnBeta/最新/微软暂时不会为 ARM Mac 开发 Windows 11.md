微软拒绝为苹果基于 ARM 架构的 M1、M1 Pro 和M1 Max Mac 开发 Windows 11。现在我们可能知道原因了—与高通的秘密独家协议。<strong>根据 XDA-Developers 的报道，基于 ARM 的 Windows 官方只支持搭载高通 SoC 的设备，因为这两家公司之间之前有秘密协议。这份信息 "很快就会到期"，但没有具体结束时间。</strong><br>
 <p><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1123/cf75d66fb4a43ea.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1123/cf75d66fb4a43ea.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1123/cf75d66fb4a43ea.jpg" referrerpolicy="no-referrer"></a></p><p>当<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>和高通之间的协议确实结束时，微软可以为其他芯片厂商开发 ARM 版 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>。<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>芯片 Mac 不提供Boot Camp，目前也没有官方的 Windows 支持，使得 M1、M1 Pro 和 M1 Max 的用户使用 Windows 系统时的选择很少。</p><p>9月，微软表示，通过虚拟化或其他方式为苹果芯片 Mac 提供的 ARM 版 Windows 11不是 "官方支持的方案"。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1206191.htm" target="_blank">高通、微软排他性协议将到期：M1 Mac将能够运行Windows</a></p></div>
