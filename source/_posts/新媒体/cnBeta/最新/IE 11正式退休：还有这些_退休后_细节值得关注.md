今天是 2022 年 6 月 15 日，也是 Internet Explorer 浏览器正式和我们说再见的日子。在停止对 IE 11
的支持之后，微软强烈推荐使用基于 Chromium 的新版 Microsoft Edge，如果需要可以通过 IE Mode 来兼容访问某些网站。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0615/13860b71dc29133.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">IE 11 是 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 8.1 系统的默认浏览器。虽然它从未达到 Chrome 的高度，但在 2014 年依然是第二大桌面浏览器，仅次于IE8。当然，从那时起，随着 2015 年微软 Edge 的出现，情况发生了很大的变化。</p><p style="text-align: left;">2015 年，Edge 成为 Windows 10 系统的默认浏览器，也就是我们熟悉的经典版 Edge，在 2020 年开始微软转向基于 Chromium 的新版 Edge，随后该浏览器进入了发展快车道。</p><p style="text-align: left;">说会 IE 11，在所有消费者 Windows 版本中该浏览器均已退休。不过用户依然可以通过以下方式进行访问</p><blockquote style="text-align: left;"><p style="text-align: left;">● Microsoft Edge 中的 IE Mode</p><p style="text-align: left;">● 包括 WebOC 在内的 Internet Explorer 平台（MSHTML/Trident）</p><p style="text-align: left;">● IE 11 桌面应用依然可以在以下平台运行</p><p style="text-align: left;"> ◎ Windows 8.1</p><p style="text-align: left;"> ◎ Windows 7 Extended Security Updates (ESU)</p><p style="text-align: left;"> ◎ Windows 10 Server SAC (all versions)</p><p style="text-align: left;"> ◎ Windows 10 IoT Long-Term Servicing Channel (LTSC) (all versions)</p><p style="text-align: left;"> ◎ Windows 10 Server LTSC (all versions)</p><p style="text-align: left;"> ◎ Windows 10 client LTSC (all versions)</p></blockquote><p style="text-align: left;">这意味着所有其他 SKU 上安装的 Internet Explorer 11 将不再获得支持或安全更新。该浏览器也没有安装在 Windows 11 上。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0615/4b22f2de5e710ca.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">自然，微软建议所有用户都应该迁移到 Microsoft Edge。如果你依赖某些只能使用 IE 浏览器的传统网站和软件，微软还在 Edge 中加入了一个名为"IE模式"的功能，以解决这一用例。</p><p style="text-align: left;">值得注意的是，虽然 IExplore.exe 将保留在设备上，但微软将分两个阶段逐步停止使用。</p><p style="text-align: left;"><strong>● 第一阶段</strong></p><p style="text-align: left;">第一阶段将遵循一个质量驱动的过程，将 IE 的使用自动重定向到 Microsoft Edge。微软强调，Windows Update 不会为这种重定向提供动力，所以避免安装更新只是为了绕过这个过程是徒劳的。微软还将提示你，"Internet Explorer 的未来是 Microsoft Edge"。整个过程可能需要几个月。</p><p style="text-align: left;"><strong>● 第二阶段</strong></p><p style="text-align: left;">在第一个阶段结束后，第二个退休阶段立即启动。它将通过 Windows Update（一个可选的"C"更新和一个"B"累积更新）永久禁用所有受影响的 Windows SKU 上的 IE。</p><p style="text-align: left;">微软已经严格声明，Internet Explorer 11不应该被完全卸载，因为它为Edge的IE模式提供动力。也就是说，你应该知道，IE 模式也不是永远支持的。它对 Windows 各 SKU 的支持结束日期如下。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0615/6d372f6c0b7aaf9.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">对于企业来说，IE组策略将继续在IE模式下工作，但如果你面临任何问题，你可以联系App Assure团队。对于在IE模式下加载网站面临问题的组织，也建议采用同样的方法。</p><p style="text-align: left;">最后，许多网站，包括微软自己的网站和其他公司拥有的网站，在过去几个月中不断放弃对IE11的支持，所以最好尽早将你的工作流程完全转移到其他支持的浏览器。</p>
