深度学习公司 Lambda，刚刚宣布了与 Razer 合作打造的 Lambda Tensorbook 新品。<strong>作为市面上最强大的深度学习笔记本，其不仅具有时尚的外观，还支持 Linux 操作系统与 Lambda 的深度学习软件。</strong>再结合 Lanbda GPU Cloud 云服务，该机能够为工程师客户提供在本地创建、训练和测试的深度学习模型所需的所有软件工具和计算性能。<br>
<p><a href="https://static.cnbetacdn.com/article/2022/0413/8886f7a6546598d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0413/8886f7a6546598d.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：Lambda <a href="https://lambdalabs.com/deep-learning/laptops/tensorbook" target="_self">官网</a>）</p><p>自 2012 年以来，Lambda 已迅速成为业内领先的研究工程团队事实上的深度学习基础设施提供商（包括 GPU 计算集群、服务器、工作站等）。</p><p>在其成千上万的客户中，涵盖了多个行业内排行前五的科技企业、美国 97% 的顶尖研究型大学（比如 MIT / Caltech）、以及美国国防部等。</p><p><a href="https://static.cnbetacdn.com/article/2022/0413/ae4131fbfac3f41.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0413/ae4131fbfac3f41.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>Lambda 联合创始人兼首席执行官 Stephen Balaban 表示：“大多数机器学习（ML）工程师所使用的笔记本电脑并未配备专用的 GPU，迫使其需要远程访问共享资源，从而拖慢了他们的开发周期”。</p><p>但当无法通过 SSH 连接到远程服务器时，工程师们将无法在本地提供任何数据或代码、甚至很难向同事演示相关模型。而 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://razer.jd.com/" target="_blank">RAZER</a> x Lambda 联手打造的 Tensorbook，就很好地化解了这方面的尴尬。</p><p><a href="https://static.cnbetacdn.com/article/2022/0413/0556285ef0c07c1.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0413/0556285ef0c07c1.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>该机预装了 PyTorch 和 TensorFlow，旨在方便工程师快速投入模型训练和演示，辅以完全本地化的 GUI 界面，而无需通过繁杂的 SSH 命令行来实现。</p><p>Lambda 为 Tensorbook 提供了完整的预配置软件环境，包括带有 Lambda Stack 堆栈的 Ubuntu Linux 发行版，可用于随时随地的大型训练工作负载。</p><p><a href="https://static.cnbetacdn.com/article/2022/0413/c902e6227d8cffb.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0413/c902e6227d8cffb.png" alt="0.png" referrerpolicy="no-referrer"></a></p><p>此外该机结合了 Razer 时尚的外观设计和高性能硬件，板载英伟达 RTX 3080 Max-Q 移动 GPU，辅以与 TensorFlow、PyTorch、cuDNN、CUDA 等完全兼容的 ML 框架和工具。</p><p>具体硬件规格包括支持 165 Hz 高刷新率的 15.6 英寸屏幕（分辨率 2560×1440），<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>酷睿 i7-11800 八核处理器（基础 2.3 GHz / 睿频 4.6 GHz），带有 16GB 显存的 RTX 3080 Max-Q 独显。</p><p><img src="https://static.cnbetacdn.com/article/2022/0413/d059693f9d495be.png" alt="4+5.png" referrerpolicy="no-referrer"></p><p>存储部分为 64GB DDR4 RAM + 2TB <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 的组合，纤薄的 4.4 磅铝制一体机身在侧边提供了雷雳 4、USB 3.2、HDMI 2.1 端口，在内嵌了一枚 1080p 网络摄像头。</p><p>目前该机已在 Lambda Labs 官网上开售，订购价为 3499 美元（约 22273 RMB），附带为期一年的 Lambda 工程支持。</p>
