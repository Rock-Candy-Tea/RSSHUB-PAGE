<strong>去年拦截了 4000 万次网络钓鱼活动的 Cloudflare，刚刚宣布了他们的最新动作 —— 以 1.62 亿美元收购反网络钓鱼公司 Area 1 Security 。</strong>据悉，Cloudflare 致力于构建更安全的互联网，而 Area 1 Security 开发的云原生平台，正好能够在对企业环境造成破坏之前，检测并消除相关威胁。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0224/6a29add7fe4da77.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">Area 1 Security 可无缝阻止基于电子邮件的网络钓鱼攻击</p><p>Cloudflare 强调称，作为验证 Area 1 Security 产品有效性的一个实例，其已于去年成功阻止了 4000 万量级的网络钓鱼活动。</p><p><strong>对于本次收购，公司联合创始人兼首席执行官 Matthew Prince 评论道：</strong></p><blockquote><p>作为互联网世界中最普遍的网络攻击媒介，电子邮件服务的安全集成，已对任何真正的零信任网络产生了至关重要的影响。</p><p>于是今天，我们隆重欢迎 Area 1 Security 的加入，以帮助 Cloudflare 成为该领域的平台领导者。</p><p>对我们而言，零信任的未来，还包括了一键式的组织网络 / 应用程序保护解决方案，尤其是最普遍的云电子邮件应用。</p><p>综上所述，Cloudflare 希望与之携手，带来市面上最畅快、高效、且可靠的电子邮件安全体验。</p></blockquote><p>由<a href="https://www.area1security.com/about/news/cloudflare-to-acquire-area-1-security/" target="_self">官方公告</a>可知，Cloudflare 将以 1.62 亿美元收购 Area 1 Security 。其中四五成将以 Cloudflare 的 A 类普通股支付，其余则是以现金的形式。</p><p>事实上，自 2021 年以来，Cloudflare 一直在力推自家的 Advanced Email Security Suite（高级电子邮件安全套件）服务。</p><p>但通过本次收购，该公司将能够增强其可在该领域提供的服务数量。如果一切顺利，该交易有望于今年 2 季度完成。</p>
