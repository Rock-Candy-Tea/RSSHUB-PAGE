尽管微软很早就放弃了 Windows 10 Mobile，但还是有一帮铁杆粉丝在为 Lumia 系列移动设备延续生命。其中比较知名的，就有 Windows on Lumia 项目组。<strong>此前，已经有开发者将 Windows 11 安装到了 Lumia 950 XL 上。现在，该平台又迎来了新版拨号器。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0810/f4cb5276f4d5af9.png" alt="2.png" referrerpolicy="no-referrer"></p><p>Gustave Monce 刚刚在 Twitter 上宣布了这一消息，同时表示新版播放器在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 操作系统中看起来很是协调。</p><p><a href="https://static.cnbetacdn.com/article/2021/0810/f4cbcb7d8dbfd55.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0810/f4cbcb7d8dbfd55.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>据悉，该拨号器由计算机工程专业学生 Simone Franco 着手创建。除了新的联系人界面，他还在窗口右侧引入了基于首字母的快速跳转功能。</p><p><img src="https://static.cnbetacdn.com/article/2021/0810/ebafd74d135fcd1.png" alt="3.png" referrerpolicy="no-referrer"></p><p>感兴趣的朋友，可移步至 GitHub 项目主页（<a href="https://github.com/WOA-Project/woadialer/releases/tag/0.1.212.0-alpha" target="_self">传送门</a>），下载这款适用于 Lumia 950 XL 的 Windows 11 拨号器。</p>
