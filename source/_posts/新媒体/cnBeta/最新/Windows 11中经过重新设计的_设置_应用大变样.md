随着微软开始向 Insider 测试者们推送 Windows 11 预览编译版本，有关新一代操作系统的槽点和亮点也开始陆续涌现。<strong>除了 UI 方面的改进，定于下半年到来的 Sun Valley 更新，还引入了重新设计的“设置”应用程序。</strong>可知其在 Windows 10 上槽点满满的“设置”应用程序和控制面板的基础上作出了大量调整。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0630/6459fc6ce76fb9a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/6459fc6ce76fb9a.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://www.windowslatest.com/2021/06/29/hands-on-with-redesigned-settings-app-on-windows-11/" target="_self">Windows Latest</a>）</p><p>与 Windows 10 相比，Windows 11 上的“设置”应用程序已变得截然不同。其不仅解决了大量的留白、改善了搜索体验，还在左侧引入了新的导航窗格。</p><p><a href="https://static.cnbetacdn.com/article/2021/0630/d95df5733863d75.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/d95df5733863d75.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">系统信息详情页</p><p>侧边栏与我们在控制面板中看到的相似，但所有选项都换上了基于 Fluent Design 设计语言的新图标。</p><p><a href="https://static.cnbetacdn.com/article/2021/0630/127f67c1fbb74f2.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/127f67c1fbb74f2.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>左侧的新导航菜单，可方便用户遍历不同的选项卡。在浏览 Windows Update 或蓝牙等子页面时，都可轻松返回到主选项（比如个性化和网络）。</p><p><a href="https://static.cnbetacdn.com/article/2021/0630/c39849746736a2b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/c39849746736a2b.jpg" alt="4.jpg" referrerpolicy="no-referrer"></a></p><p>微软还刷新了 WinUI 体验，以便用户在深入菜单选项后，还能更好地了解到当前所处的位置。而顶部的新标题栏，将突出显示桌面背景和常用设置等信息，并且能够按需自定义。</p><p><a href="https://static.cnbetacdn.com/article/2021/0630/bab023078c67faa.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/bab023078c67faa.jpg" alt="5.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">个性化设置</p><p>这样的设计在诸多页面随处可见，比如 Windows 更新、个性化、网络和 Internet、以及系统等。此外你可以更好地掌控 Windows 更新，比如查看累积或安全更新所需的预估重启等待时间。</p><p><a href="https://static.cnbetacdn.com/article/2021/0630/d897c4408678d28.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/d897c4408678d28.jpg" alt="6.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">游戏设置</p><p>正如预期的那样，Windows 11 引入了新的自定义选项。其中包括一个新的个性化屏幕，允许用户在不同的主题之间进行切换。使用触屏键盘的用户，也可对输入法 UI 的背景图像进行个性化配置，或者变更按键的大小和颜色。</p><p><a href="https://static.cnbetacdn.com/article/2021/0630/b76a895b8050642.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/b76a895b8050642.jpg" alt="7.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">蓝牙设置</p><p>其实 Windows 操作系统中的“设置”应用程序，推出至今已有大约十个年头。微软最初有意让它彻底取代 Windows 10 中的控制面板，但可惜直到 Windows 11，控制面板都没能完全消失。</p><p><a href="https://static.cnbetacdn.com/article/2021/0630/6a9f8276f7a350b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0630/6a9f8276f7a350b.jpg" alt="8.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">存储设置</p><p>作为习惯了控制面板的交互逻辑的用户，我们发现 Windows 11 会尽可能地把大家引导至新版“设置”应用。感兴趣的朋友，可于即日起申请加入 Insider 计划，以下载并安装 Windows 11 的预览体验版本。</p>
