对于目前仍在使用 Exchange Server 2013 的客户，微软发布了一个公开警告：该软件的支持即将结束。微软告诉大家，2023年4月11日之后将不会有进一步的更新，因为支持结束意味着该产品将被完全放弃。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0629/51ce3120567d226.webp" alt="7o151888.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">一旦支持期结束，Exchange Server 2013 客户将失去</p><blockquote style="text-align: left;"><p style="text-align: left;">● 对可能发生的问题提供技术支持</p><p style="text-align: left;">● 对发现的、可能影响服务器稳定性和可用性的问题进行错误修复</p><p style="text-align: left;">● 对发现的、可能使服务器容易受到安全攻击的漏洞进行安全修复</p><p style="text-align: left;">● 时区更新</p></blockquote><p style="text-align: left;">那些想继续运行 Exchange Server 2013 的人自然也可以继续使用，但继续使用不支持的软件显然会增加安全漏洞的风险。微软解释道：“当然，Exchange Server 2013将在此日期之后继续运行；但是，由于上述风险，我们强烈建议你尽快从Exchange Server 2013迁移。如果你还没有开始从Exchange Server 2013迁移到Exchange Online或Exchange Server 2019，现在是开始计划的时候了”。</p><p style="text-align: left;">对于Exchange Server 2013用户来说，最好的两个选择是升级到Exchange Server 2019，或者直接拥抱Exchange Online。</p><p style="text-align: left;">微软解释道：“如果你要迁移到Exchange Online，你可能有资格使用我们的Microsoft FastTrack服务。FastTrack提供最佳实践、工具和资源，使您尽可能无缝地迁移到Exchange Online。最重要的是，你将有一个支持工程师指导你完成从规划和设计到迁移最后一个邮箱的工作”。</p>
