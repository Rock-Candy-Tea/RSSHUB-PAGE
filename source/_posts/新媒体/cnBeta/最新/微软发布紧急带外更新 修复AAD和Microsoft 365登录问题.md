在安装 2022 年 6 月补丁星期二活动日发布的更新后，导致 Azure Active Directory 和 Microsoft 365 出现登录问题。<strong>今天微软发布了紧急带外（out-of-band，OOB）更新，修复了上述这个问题。</strong>用户可以通过 Windows Update 方式更新，也可以通过 Microsoft Update Catalog 下载安装包离线更新。<br>
 <p style="text-align: left;">下载：<a href="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5016139" target="_blank" rel="nofollow noopener">KB5016139</a> for <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 10 |  <a href="https://www.catalog.update.microsoft.com/Search.aspx?q=KB5016138" target="_blank" rel="nofollow noopener">KB5016138</a> for Windows 11<br style="text-align: left;"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0621/6498f5b86014b73.webp" alt="bk634wdw.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">微软解释说：“解决了一个已知问题，该问题仅影响基于 Windows Arm 的设备，并可能阻止您使用 Azure Active Directory (AAD) 登录。使用 AAD 登录的应用程序和服务，例如 VPN 连接、Microsoft Teams 和 Microsoft Outlook，也可能会受到影响”。</p><p style="text-align: left;">受此问题影响的 Windows 版本的完整列表包括多个客户端平台：Windows 11 21H2、Windows 10 21H2、Windows 10 21H1 和 Windows 10 20H2。</p><p style="text-align: left;">根据向 Microsoft 365 客户发布的服务警报，一些受影响的场景包括但不限于以下内容：</p><blockquote style="text-align: left;"><p style="text-align: left;">● 使用 Azure Active Directory (Azure AD) 登录的应用和服务</p><p style="text-align: left;">● VPN 连接</p><p style="text-align: left;">● 桌面端 Microsoft Teams</p><p style="text-align: left;">● 企业版 OneDrive</p><p style="text-align: left;">● Outlook 桌面客户端</p></blockquote><p style="text-align: left;">微软补充道：“此 OOB 更新是累积性的。如果您使用的是 2022 年 6 月 14 日之前发布的更新，我们建议您安装此 OOB 更新。如果您安装了较早的更新，则只会下载此软件包中包含的新更新并将其安装在您的设备上”。</p>
