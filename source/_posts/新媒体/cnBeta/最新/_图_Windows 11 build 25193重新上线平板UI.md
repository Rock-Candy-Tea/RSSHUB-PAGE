<strong>微软似乎要重新上架平板模式。</strong>今年早些时候，微软引入了全新的平板 UI，提供更简洁的交互体验，任务栏上没有应用图标整体体验接近于手机。不过随后微软放弃了这种模式，声称它需要根据反馈进行更多改进。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0902/8d6b97c18f1b53e.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">时隔 4 个月，平板模式重新回归。正如 @PhantomOfEarth 所发现的那样，用户可以在更新到最新版本 Build 25193 后使用 Vivetool 应用程序启用新模式。专用 ID 恢复了在具有触摸屏的设备（例如平板电脑或混合笔记本电脑）上管理平板电脑优化任务栏的选项。</p><p style="text-align: left;">启用方式</p><p style="text-align: left;">1. 从 Github 下载 Vivetool 工具，并解压到你喜欢的任意路径。</p><p style="text-align: left;">2. 右键开始菜单，并选择 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Terminal (Admin)</p><p style="text-align: left;">3. 切换到命令提示符窗口</p><p style="text-align: left;">4. 使用 cd 命令进入到 Vivetool 工具所在的文件夹。</p><p style="text-align: left;">5. 输入以下命令：<strong>vivetool /enable /id:26008830</strong> ，然后按回车</p><p style="text-align: left;">6. 重启你的电脑</p><p style="text-align: left;">启用 ID 后，转到设置 > 个性化 > 任务栏 > 任务栏行为，然后在“优化任务栏以进行触摸交互”选项旁边打勾。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0902/73e61d8a119f3b0.webp" referrerpolicy="no-referrer"></p>
