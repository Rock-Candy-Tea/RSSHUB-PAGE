想要体验 Windows 11 预览版，目前已经面向 Windows Insider 的 Dev 和 Beta 频道用户开放。如果你觉得 Dev 频道版本更新太快，系统不太稳定，那么现在可以选择加入 Beta 频道。本文将介绍如何从 Dev 频道切换到 Beta 频道，整个设置过程也非常简单。<br>
<p style="text-align: left;">Dev 频道发布的预览版主要面向技术用户。对于想要尝试每个功能的分析师和开发人员来说，它是推荐的渠道，尽管它可能不稳定并立即发布。Beta 频道相对更稳定，适用于想要尝试新功能但最稳定的功能的人。</p><p style="text-align: left;">步骤1：右键单击开始菜单（<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 图标）并选择设置。或者使用快捷键：Win + I</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0806/d44455926fc86ec.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0806/d44455926fc86ec.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">步骤2：从左窗格中选择 Windows Update。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0806/1b61bcb25144d02.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0806/1b61bcb25144d02.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">步骤3：选择 Windows Insider Program</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0806/1467762efc34e45.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0806/1467762efc34e45.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">步骤4：选择“Choose your Insider settings”选项。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0806/5ece67ac16b3592.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0806/5ece67ac16b3592.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">步骤5：选择您想要的频道（Dev、Beta）。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0806/1ca12c7afaf6fb9.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0806/1ca12c7afaf6fb9.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">步骤6：关闭设置并重启系统</p>
