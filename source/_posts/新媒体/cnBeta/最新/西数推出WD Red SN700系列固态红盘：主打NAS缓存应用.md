在收购闪迪（SanDisk）几年后的 2019 年 4 季度，西部数据开始推出面向网络附加存储（NAS）市场的 WD Red SSD 产品线。针对特定的缓存应用场景，这些 SATA / M.2 SATA 固态硬盘选用了 Marvell 的 88SS1074 主控。<strong>随着分层存储的日益普及，以及非易失性主机存储控制器（NVMe）接口规范的普及，西数现又为 WD Red 产品线带来了一个新成员。</strong><br>
<p><img src="https://static.cnbetacdn.com/article/2021/0929/1217d7a87273b52.jpg" referrerpolicy="no-referrer"></p><p>它就是本文的主角 —— WD Red SN700 系列 PCIe 3.0 x4 M.2 NVMe 固态红盘。容量方面，其提供了 250GB / 500GB / 1TB / 2TB / 4TB 五档选项。</p><p>与 SATA <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a> 相比，PCIe 3.0 x4（NVMe 1.3 协议）为 WD Red SN700 固态红盘带来了 I/O 性能的巨大飞越，与纯机械硬盘（HDD）系统相比更是提升显著。</p><p><a href="https://www.anandtech.com/show/16978/western-digital-introduces-wd-red-sn700-pcie-30-m2-nvme-ssds-for-nas-systems" target="_self">AnandTech</a> 指出，NAS 正在迅速向着超融合应用服务器转型，而 NVMe 驱动器可有效改善用户在存储和应用任务方面的体验，尤其是虚拟化、多用于协作编辑等工作负载相关型任务。</p><p>即使单纯地将 NVMe 驱动器用作缓存盘，其对传统企业工作负载中的密集型数据库等应用也有极大的好处。</p><p><img src="https://static.cnbetacdn.com/article/2021/0929/5d97750bf5c178b.png" referrerpolicy="no-referrer"></p><p>至于西数 WD Red SN700 系列，250GB / 500GB / 1TB 型号每日可承受 1 次全盘写入（DWPD），而 2TB / 4TB 型号则是 0.7 DWPD（全系采用 3D TLC NAND 闪存）。</p><p>需要指出的是，作为西数的一大竞争对手，希捷（Seagate）IronWolf 系列也已经在 NVMe NAS 缓存盘市场提供了两代产品。</p><p>不过得益于与闪迪的垂直整合，WD Red SN700 或较竞品拥有一些更显著的优势。</p><p>至于售价，250GB / 500GB / 1TB / 2TB / 4TB 型号分别为 65 / 80 / 145 / 290 / 650 美元，约合 421 / 518 / 939 / 1877 / 4208 RMB 。</p>
