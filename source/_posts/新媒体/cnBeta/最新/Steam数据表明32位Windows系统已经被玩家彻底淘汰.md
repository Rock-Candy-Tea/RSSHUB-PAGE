<strong>32 位 Windows 系统已经被淘汰，因为新的数据表明几乎所有人都在使用 64 位系统。</strong>根据最新一期的 Steam 软硬件报告显示，只有极少数用户仍在运行 32 位的 Windows 系统，而 64 位的 Windows 系统显然占据主导地位。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1103/051d63dc6724d07.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1103/051d63dc6724d07.jpg" alt="o43bj9a8.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">更准确的是，64 位 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 10 系统是 Steam 玩家的首选，因为这个操作系统版本以超过 89% 的份额连续一个月稳居领先位置。古老的 Windows 7 64 位是亚军，占 3.89%，而最近发布的 Windows 11 64 位现在是第三名，占 1.82%。</p><p style="text-align: left;">榜单中第一个 32 位 Windows 版本是 Windows 7，份额为 0.22%，其次是其 Windows 10 ，份额为 0.09%。</p><p style="text-align: left;">这份报告中的另一个好消息涉及Windows 11，因为新的操作系统在短短一个月内已经达到了1.82%的份额。而最有可能的是，这些数字实际上要高得多，因为Windows 11的Windows Insider构建仍被认定为Windows 10。</p>
