对One UI 5.0测试版的等待终于要结束了，因为三星已经决定为Galaxy S22设备推出One UI 5.0测试版。当然，这个测试版比我们预期的晚了一些，但可以说，用户会很高兴在他们的智能手机上尝到新的Android版本。<br>
 <p>One UI 5.0测试版是在Android 13的基础上打造的，新的更新确实带来了很多新的功能，虽然外观上的变化不是很多。简单地说，在Android 13中，Google专注于保持对安全的关注，而不是去做另一次外观上的大修，该更新目前正在向德国的用户推出。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0805/57ca43e4e1453a2.jpg" title alt="galaxy-s22-one-ui-5-0-beta-2.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">德国的Galaxy S22用户终于可以报名参加基于Android-13的One UI 5.0测试项目了</p><p>三星仍然没有详细说明One UI 5.0的所有内容，但Galaxy Unpacked发布会届时有可能会有提供很多更新信息，关于测试计划的正式公告也应该在本周晚些时候到来。</p><p>One UI 5.0测试版从外观上看也确实带来了一些变化，比如通知中心现在有更大的应用图标，增加了背景半透明设计，应用程序的权限弹出窗口也已改变，全面的光学字符识别功能也被加入，动画的速度和流畅度也有了明显改善。</p><p>正如预期，One UI 5.0测试版将分阶段推出，目前仅适用于Galaxy S22系列。我们希望随着时间的推移，三星将把对最新更新的支持扩大到其他<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>。</p>
