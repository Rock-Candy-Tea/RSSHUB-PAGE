<strong>根据 Mercury Research 公布的最新数据，在 2022 年第 1 季度，AMD 在 x86 CPU 市场的份额刷新了历史最佳成绩。</strong>最新季度中 AMD 的市场份额达到了 27.7%，相比较上个季度增加了 2.1 个百分点。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0512/302c7687dc39ebc.webp" style referrerpolicy="no-referrer"></p><p><img src="https://static.cnbetacdn.com/article/2022/0512/b8845ae9237e433.webp" style referrerpolicy="no-referrer"></p><p><img src="https://static.cnbetacdn.com/article/2022/0512/9aa456e0a518e60.webp" style referrerpolicy="no-referrer"></p><p><img src="https://static.cnbetacdn.com/article/2022/0512/47ca99131a9d5a2.webp" style referrerpolicy="no-referrer"></p><p>这使 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 目前的份额达到创纪录的高点，而<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>目前的份额为 72.3%，但自从 AMD 推出 Zen CPU 架构以来，其份额一直在下降。与去年同期相比，AMD 实现了比英特尔+7.0点的跳跃，这无疑是令人印象深刻的。</p><p>在报告中写道：“低价入门级 CPU 的出货量减少，以及新的移动处理器（英特尔的 Alder Lake CPU 和 AMD 的 Barcelo 和 Rembrandt CPU 核心）的强劲上升，导致移动 CPU 的价格大大提高，这有助于创造 138 美元的客户（台式机和笔记本的组合）平均销售价格的记录，这比季度增长超过 10%，比年增长超过30%”。</p><p>在细分市场的份额方面，AMD 服务器和移动部门表现良好，移动部门以 +0.9 个点的份额领先，目前为 22.5%，而英特尔为 77.5%。这一增长是在上一季度略有下降之后出现的。看起来 Ryzen 6000"Rembrandt"的销售帮助红色团队在2022年第一季度将其数字大幅提高，并且随着全球供应的正常化，将在全年继续改善。</p><p>在服务器领域，我们看到一个+0.9点的增长，现在的份额是11.6%，而英特尔的份额仍然是巨大的88.4%。最后，在台式机领域，AMD的份额增长最大，达到+2.1个百分点，目前为18.3%，而英特尔为81.7%。AMD最近为DIY领域推出了各种主流和高端选项的Ryzen 5000，这应进一步帮助该芯片制造商在2022年第二季度将其数字进一步提高。</p><p>该公司一直面临着严重的供应限制，虽然7纳米的优先出货量正向EPYC和Ryzen移动端发展，但Ryzen桌面CPU在主要零售店的销售方面仍然领先。但我们看到，自2022年开始，全球供应恢复正常，CPU和GPU的产量都在增加。</p><p>虽然英特尔的Alder Lake CPU在台式机和笔记本领域带来了激烈的竞争，但服务器领域仍然是AMD的强项，自EPYC平台诞生以来，AMD一直通过升级和增强的解决方案不断推进其发展。预计AMD将在今年晚些时候推出其首款Zen 4台式机和服务器芯片，这将大大推动公司的发展。</p>
