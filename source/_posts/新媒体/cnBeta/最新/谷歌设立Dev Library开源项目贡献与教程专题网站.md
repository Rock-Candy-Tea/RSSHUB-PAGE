<strong>Google 刚刚设立了一个名叫“开发库”（Dev Library）的新网站，旨在提供与这家互联网巨头的平台和软件等各个方面相关的开源项目和教程。</strong>与 GitHub 之类的开源代码托管平台相比，Dev Library 并不鼓励大家在注册后上传自己的代码和文本，而是提供了由 Google 团队牵头策划的精选内容。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1021/0a32fc02db293eb.png" referrerpolicy="no-referrer"></p><p>这么做的好处，就是能够确保项目和文章的正确性与相关性。具体说来是，Dev Library 提供了包括 Android、Angular、Firebase、Flutter、谷歌云、以及机器学习在内的六大类内容。<br></p><p>通过起始页，访客能够一看查看到各个子版块的两个项目与两篇文章，并且能够根据子类别来过滤各个主题页面，以发现用于自然语言处理的 ML 应用程序或 Android 工具等。</p><p>这里的每个开源项目页面，都包含了指向 GitHub 存储库的链接、以及直接定向到博客或发布网站上的文章。对于开发者来说，该站点的主要目的是提升可见性。</p><p><a href="https://static.cnbetacdn.com/article/2021/1021/eeaa1455a0772d3.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1021/eeaa1455a0772d3.png" referrerpolicy="no-referrer"></a></p><p>在项目页面之外，还有一个专门的贡献者与源代码使用页，并且带有指向 GitHub 或 Medium 上的个人资料页面的连接，以便展示相关软件平台和其它可用的项目。</p><p>谷歌宣称 Dev Library 旨在让开发者更多地参与其中，并解释为何以某种方式来创建项目。如果你也想要在上面提交自己的 GitHub 项目或播客文章，还请填写表单并等待官方审核。</p><p>最后，Google 也欢迎大家通过 GitHub 资源库中的 <a href="https://github.com/google/devlibrary" target="_self">Dev Library</a> 主页提交各种反馈。至于更多细节，还请移步至 Google <a href="https://developers.googleblog.com/2021/10/what-is-the-dev-library.html" target="_self">开发者博客</a>网站了解。</p>
