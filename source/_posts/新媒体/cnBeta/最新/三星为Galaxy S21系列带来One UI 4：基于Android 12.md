三星在自家论坛上宣布，即将启动Galaxy S21系列的One UI 4测试计划。不过，三星并未说明何时会推出测试版本，唯一可以确定的是，三星One UI 4将基于Android 12开发。<br>
<p>还有国外博主曝料声称，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://samsung.jd.com/" target="_blank">三星</a>One UI 4将会在8月份开始测试，带来新的图标和颜色变化，也会加入Android 12的Material You设计语言，并对骁龙888/Exynos 2100进行优化，还可能会对三星自家的“笔记”应用进行改造，因为现在很多设备都带有S-Pen。</p><p>该博主推测，Galaxy S21系列的One UI 4会于11月或12月初正式发布。也有传言称，三星会在Galaxy Unpacker活动中公布测试计划。</p><p><img src="https://static.cnbetacdn.com/article/2021/0728/215480fb72d1d53.png" referrerpolicy="no-referrer"></p>
