<strong>在 7 月的补丁星期二活动日了，Windows 11 通过 Windows Update 可以获得 KB5015814 累积更新，安装后版本号升至 Build 22000.795。</strong>本次更新在引入诸多常规修复和<a href="https://msrc.microsoft.com/update-guide/releaseNote/2022-Jul" target="_blank">安全更新</a>，并向更多用户开放 Search Highlights 之外，还对 PowerShell 进行了重大调整。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0713/5fc117c68647d78.webp" alt="nz1z1gx8.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">本次更新修复了重定向 PowerShell 命令输出的问题，以便脚本日志不包含命令的任何输出。今天的更新还带来了与 KB5014668（2022 年 6 月 23 日发布）相同的改进和功能。提醒一下，该更新的推出也首次让非 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Insider 项目成员测试 Search Highlights。</p><p style="text-align: left;">此更新中有两个已知问题，在过去的两个补丁星期二版本中一直存在。安装此更新后，某些 .NET Framework 3.5 应用可能会出现问题或无法打开。您可以通过在 Windows 功能中重新启用 .NET Framework 3.5 和 Windows Communication Foundation 来修复它。第二个已知问题影响 Edge 中的 IE 模式选项卡无响应。这可以使用已知问题回滚来修复。</p>
