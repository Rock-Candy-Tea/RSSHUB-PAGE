本月早些时候，微软为Windows 10和11用户可以从微软商店下载的最佳应用开启投票。Windows社区在不同类别中提名了几个项目，现在微软已经公布了投票结果。<br>
 <p><strong>以下是2022年<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>商店应用奖相应类别的获奖者：</strong></p><p>实用工具--<a href="https://apps.microsoft.com/store/detail/sharex/9NBLGGH4Z1SP">ShareX</a> 和 <a href="https://www.microsoft.com/store/productId/9NBLGGH4Z1JC">Speedtest</a> by <a href="https://www.microsoft.com/store/productId/9NBLGGH4Z1JC">Ookla</a></p><p>实用工具类是为了表彰那些帮助自动化任务、诊断问题、记录分享屏幕、导航驱动程序等的应用程序。当你使用这些应用程序时，它们是不可缺少的。</p><p>实用程序类别的亚军是<a href="https://apps.microsoft.com/store/detail/torrex-lite-torrent-downloader/9WZDNCRFJB3D">Torrex</a> Lite 和 <a href="https://www.microsoft.com/en-us/p/fluent-screen-recorder/9mwv79xlfqh7">Fluent Screen Recorder</a>.</p><p>文件管理 -<a href="https://www.microsoft.com/store/productId/9NGHP3DX8HDX">Files</a> 和 <a href="https://apps.microsoft.com/store/detail/rufus/9PC3H3V7Q9CH?hl=en-us&gl=US">Rufus</a></p><p>该类别中的应用程序通过查看和分类、编辑元数据、转换、压缩以及更多方式帮助管理Windows PC上的文件。这些是你告诉我们的对你最重要的应用。</p><p>文件管理类别的亚军是<a href="https://www.microsoft.com/store/apps/9WZDNCRFJB33">8Zip</a> 和 <a href="https://www.microsoft.com/en-us/p/one-commander/9nblggh4s79b">One Commander</a>.</p><p>开放平台--<a href="https://apps.microsoft.com/store/detail/auto-dark-mode/XP8JK4HZBVF435">Auto Dark Mode</a>, <a href="https://www.microsoft.com/p/eartrumpet/9nblggh516xp?cid=nom">EarTrumpet</a>, 以及 <a href="https://www.microsoft.com/store/productId/9MT60QV066RP">ModernFlyouts</a></p><p>Windows是一个开放的创新平台。我们知道，并不是每个客户都以同样的方式使用Windows--有些人希望修改用户界面的不同部分，或者使一些功能对他们来说更容易使用。</p><p>开放平台类别的亚军是<a href="https://www.microsoft.com/store/productId/9NTM2QC6QWS7">Lively Wallpaper</a> 和<a href="https://www.microsoft.com/store/productId/9PF4KZ2VN4W9">TranslucentTB</a>.</p><p>最后，微软利用这个机会提到了其他类别中的几个重要的应用程序。微软商店的编辑们认可了一些项目，如Talk for Me（"为没有声音的人发声"的应用）、FluentCast（免费的Windows 11风格的播客播放器）、DrawboardPDF（制作精良的PDF浏览器）和Telegram（知名度极高的即时通信软件）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0529/a86c83de90e19cb.jpg" target="_blank"><img
