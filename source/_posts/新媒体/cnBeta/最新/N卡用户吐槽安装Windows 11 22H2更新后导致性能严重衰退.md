本周，微软推送了期待已久的 Windows 11 22H2 更新。但是对于英伟达显卡用户来说，许多机器都受到了性能滑坡的困扰。<strong>Bleeping Computer 指出，在 Reddit 和微软社区上，普遍有玩家抱怨遇到了卡顿和迟滞等问题。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0923/7ebd515b4b9d2a2.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">截图（来自：Microsoft <a href="https://answers.microsoft.com/en-us/windows/forum/all/cpu-usage-dropped-to-5-during-gaming-after/e4b80ef6-be5a-42de-8b94-2bbaa4178778" target="_self">Community</a>）</p><p>某位网友在<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>社区上发帖称，当天上午刚更新了 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 22H2，结果游戏体验马上变得拉胯。</p><p>另一位网友附和道，打完更新就发现，《战地 5》和《极品飞车 21》游戏中的 CPU 使用率，会从一开始的 50~60%、暴降到只有 5% 左右。</p><p>这种情况似乎在几乎每一款游戏中都有发生，而系统在更新前一天的表现都还是正常的。</p><p><img src="https://static.cnbetacdn.com/article/2022/0923/28e5e7e4cf10d9a.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">截图（来自：<a href="https://www.reddit.com/r/Windows11/comments/xj877b/comment/ip8qys8/" target="_self">Reddit</a>）</p><p>镜头转到 Reddit 这边，一位网友称其在桌面和游戏体验上都遇到了严重的性能衰退问题、且英伟达显卡的 G-Sync 自适应刷新率功能也遭到了破坏。</p><p>在非图形密集型测试中，帧率从 ~120 FPS 左右掉到了仅 ~30 FPS 上下的水平，画面的撕裂感也变得异常严重。</p><p>离谱的是，这位网友明明有着相当高端的配置 —— 包括酷睿 i7-12700K 处理器、GeForce RTX 3080 Ti 显卡、以及 DDR5 系统内存。</p><p>在排除了硬件问题之后，他也尝试关闭了所有新的图形设置，结果还是没有找到任何头绪。</p><p>作为一枚重度游戏玩家，若该 bug 无法被及时修复，他的体验将遭到严重的损害。</p><p><img src="https://static.cnbetacdn.com/article/2022/0923/787b042770c2bf5.png" referrerpolicy="no-referrer"></p><p>不过这件事情的苗头，其实早就有迹象。比如在桌面应用场景下，电脑依然有一种啥都变迟钝了的性能衰退感。</p><p>在几个月前的 Insider 测试版本中，他也遇到过同样的问题。没想到的是，微软竟然在未修复 bug 的情况下，就匆忙地推出了 Windows 11 22H2 更新。</p><p>目前微软和英伟达方面都尚未就此事明确表态，但至少有一位 NVIDIA 员工尝试在 Reddit 上联系受影响的用户，以收集尽可能多的反馈。</p><p>最后，如果实在急着在官方修复补丁发布前正常使用 N 卡计算机，一些网友表示将 Windows 11 22H2 更新回滚就可以暂时避免该问题。</p>
