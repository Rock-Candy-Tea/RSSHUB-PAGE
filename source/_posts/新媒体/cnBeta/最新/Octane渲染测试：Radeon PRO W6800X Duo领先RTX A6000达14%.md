WCCFTech 报道称：在 Octane Render 渲染基准测试项目中，<strong>配备两颗 Big Navi 21 GPU 核心的 AMD Radeon PRO W6800X Duo 双芯显卡，达成了领先于英伟达 RTX A6000 和 GeForce RTX 3090 的亮眼成绩。</strong>据悉，除了 RNDA 2 家族中唯一可用的双芯产品（苹果 Mac Pro 工作站独有），这份由 Bare Feats 发布的跑分，还横向比较了其它多款 PRO / RX 系列 SKU 。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0917/953d01d1c755d35.png" alt="0.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/amd-radeon-pro-w6800x-duo-dual-rdna-2-gpu-graphics-card-outperforms-nvidia-geforce-rtx-3090-rtx-a6000-in-octane-render/" target="_self">WCCFTech</a>）</p><p>AMD 于上月正式发布了 Radeon PRO W6800X Duo 双芯显卡，Apple Store 的报价为 5000 美元（约 3.23 万 RMB）。不过由于使用了 MPX 模块外形，它只能兼容 2019 款的 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmac%2F" target="_blank">Mac Pro</a> 工作站。</p><p><a href="https://static.cnbetacdn.com/article/2021/0917/6eb55a45e9e13f1.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0917/6eb55a45e9e13f1.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图自：<a href="https://barefeats.com/pro-w6800x-duo-other-gpus.html" target="_self">Bare Feats</a>）</p><p><strong>以下是该卡的一些亮点：</strong></p><blockquote><p>● 两枚 W6800X GPU，分别具有 60 个计算单元（CU）/ 3840 个流处理器。</p><p>● 总计 64GB GDDR6 显存，每个 GPU 32GB @  512GB/s 带宽。</p><p>● 高达 30.2 TFLOPs 的单精度 / 60.4 TFLOPs 的半精度算力。</p><p>● 板载 4 个雷电 3 端口 + 1 个 HDMI 2.0 端口。</p><p>● 四路 DisplayPort 通过路由方式连接到系统内部，以支持雷电 3 端口。</p><p>● 最多支持八台 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_110018%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">4K</a> 显示器、四台 5K 显示器、或六台 Pro Display XDR 显示器。</p><p>● 全高 MPX 模块托架，需占用额外的功率与 PCIe 带宽。</p><p>● 板载 Infinity Fabric Link 连接，以双向 84GB/s 的速率连接两颗 W6800X GPU。</p><p>● 外部 Infinity Fabric Link 连接，打通两个 W6800X Duo 模块和四路其它 W6800X GPU 。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2021/0917/21711d06146c79a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0917/21711d06146c79a.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>AMD Radeon PRO W6800X Duo GPU 规格：</p><blockquote><p>● 旗舰 SKU 基于双 Big Navi 21 GPU（即 Radeon Pro W6800X），提供高达 64 GB 的 GDDR6 显存 + 两枚 60 CU  的 GPU 芯片，总计 120 CU / 7680 个流处理器。</p><p>● GPU 时钟频率 1979 MHz，提供高达 30.2 TFLOPs 的 FP32 和 60.4 TFLOPs 的 FP16 算力。</p><p>● 通过双 256-bit 总线接口，该显卡可提供高达 1 TB/s 的带宽。两枚略缩的 Navi 21 GPU 芯片，让 DUO SKU 的总功耗仅在 400W 左右。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2021/0917/6703de5fbf6529a.png" alt="3.png" referrerpolicy="no-referrer"></p><p>AMD Radeon PRO W6800X Duo GPU 基准测试 —— Bare Feats 主要将目光放在了 Octane Benchmark 上。通常情况下，英伟达 RTX 显卡上的表现会更出色一些。</p><p><img src="https://static.cnbetacdn.com/article/2021/0917/6f713fa6dd1cbbf.png" alt="4.png" referrerpolicy="no-referrer"></p><p>有趣的是，Radeon PRO 6800X DUO 在 14 秒内就完成了渲染，得分在 717 左右。相比之下，英伟达 RTX A6000 用了 16 秒，得分为 628 。此外该公司旗舰游戏显卡（GeForce RTX 3090）用时 15.2 秒，得分为 662 。</p><p><img src="https://static.cnbetacdn.com/article/2021/0917/8770ff026f47bfa.png" alt="5.png" referrerpolicy="no-referrer"></p><p>照此算来，AMD Radeon PRO W6800X Duo 双芯卡分别领先英伟达 RTX A6000 / RTX 3090 达 14% / 8% 。</p><p><img src="https://static.cnbetacdn.com/article/2021/0917/5cbaceb12991038.png" alt="6.png" referrerpolicy="no-referrer"></p><p>然而考虑到 A6000 的售价在 4500 - 5000 美元左右，W6800X Duo 的入手成本也不菲（更何况目前仅限<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a> Mac Pro 工作站）。</p><p>相比之下，英伟达 RTX 3090 的建议零售价仅为 1499 美元（终端实际零售价在 2000 - 3000 美元左右）。</p><p><img src="https://static.cnbetacdn.com/article/2021/0917/5b06dcd20f0f6eb.png" alt="7.png" referrerpolicy="no-referrer"></p><p>最后，与上一代 AMD PRO Vega II Duo 双芯卡相比，基于 Navi 21 GPU 的 AMD Radeon PRO W6800X DUO，还是领先 Radeon RX 6900 XT 单芯旗舰游戏显卡达 18% 和 37% 。</p>
