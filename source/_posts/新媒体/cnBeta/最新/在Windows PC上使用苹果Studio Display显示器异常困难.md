<strong>虽然出乎大家的意料，但苹果新发布的 Studio Display 显示器无法很好地与 Windows PC 配合使用这件事，还是让不少人感到困扰。</strong>在周末发表的一篇新文章中，程序员兼 Test Double 联合创始人 Justin Searls，概述了他尝试用苹果最新 5K 显示器连接 Windows 游戏 PC 时遇到的各种尴尬。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0309/adcafa0b1655cee.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0309/adcafa0b1655cee.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（via <a href="https://www.slashgear.com/807386/using-apples-studio-display-with-a-windows-pc-is-bizarrely-difficult/" target="_self">SlashGear</a>）</p><p>其实<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>很早就给消费者打过预防针，明确指出 Studio Display 的某些功能不会延续到 PC 上。</p><p>但是不服输的 Justin Searls，还是积极开展了<a href="https://justin.searls.co/posts/connecting-a-gaming-pc-to-apple-studio-display/" target="_self">不同的尝试</a>，包括让显示器内置的六个扬声器发挥作用。</p><p><img src="https://static.cnbetacdn.com/article/2022/0323/dc296dce9e31169.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">接线示例</p><p>一开始，Justin Searls 还满心期待着让游戏机搭配 Studio Display 运行，但遗憾遇到了挫折。</p><p>尽管随后确实让 PC 点亮了苹果显示器，但他又在文末提出了一个灵魂拷问 —— 就这么长期使用下去是否值得？</p><p><img src="https://static.cnbetacdn.com/article/2022/0323/55b2ed974c0f24f.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">显卡端口连接特写</p><p>当尝试用 DP 转 USB-C 线缆连接 RTX 3090 显卡时，Studio Display 起初并没有任何反应。之后他开始绞尽脑汁，并最终找到了两种可行的方法。</p><p>第一种方法涉及使用 DP 1.4（8K 版本）USB-C 适配器，同时需要搭配最新版本、带宽更高的线缆才能点亮 Studio Display 。然而这么做的缺点，就是无法启用内置的扬声器。</p><p>第二种方法更加费力一些，首先需要在 PC 上安装一张 PCIe 雷电扩展卡。如果想使用显示器上的扬声器和 USB 端口，还得在主板上开启 USB 功能。</p><p><img src="https://static.cnbetacdn.com/article/2022/0323/1d77ba979fe8d86.png" alt="3.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">启动转换控制面板</p><p>在原帖中，Justin Searls 给出了详细的分步操作流程。但由于需要在 PC 机箱里折腾一番，这里并不建议缺乏经验的小白用户轻易模仿。</p><p>此外你可能需要在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> PC 上安装 Boot Camp，这显然又是一个可能不大值得挑战的巨大麻烦。</p><p><a href="https://static.cnbetacdn.com/article/2022/03/516db13e4a7e11e.jpg" target="_blank"><img src="https://static.cnbetacdn.com/article/2022/03/516db13e4a7e11e.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">Studio Display 内置了 A13 仿生芯片和相当多的附加组件</p><p>综上所述，尽管 Justin Searls 口口声声说用 Studio Display 连接 PC 并不那么困难，但脱离苹果自家的软硬件生态，相关功能损失还是相当不划算的。</p><p>即使痴迷于该显示器的颜值，市面上其实也有相当丰富的高端替代品 —— 尤其是 Windows PC 玩家最为看重的帧率表现（普遍高达 144Hz）。</p><p>相比之下，尽管售价高达 1500 美元的 Studio Display 在多个方面都相当优秀，但其面板却锁定了 60Hz 刷新率。</p>
