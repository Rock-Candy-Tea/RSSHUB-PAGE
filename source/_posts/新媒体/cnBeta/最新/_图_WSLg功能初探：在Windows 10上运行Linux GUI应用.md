在 Windows 10 最新预览版中，用户可以通过最新的 Windows Subsystem for Linux GUI（WSLg） 直接在 Windows 10 桌面运行 Linux 应用程序了。在外媒 Bleeping Computer 发布的最新 WSLg 功能上手文章中，演示了几款主流的图形化 Linux 应用程序。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0601/885aab5ae70738f.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0601/885aab5ae70738f.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在今年的 Build 2021 开发者大会上，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>正式发布了 WSLg 功能，并表示 Windows 10 Build 21364 及更高版本均会支持该功能。该功能主要利用 Wayland 服务器在 Windows 10 系统上显示 Linux GUI 应用程序。</p><p style="text-align: left;">Windows 开发者平台项目经理 Craig Loewen 解释道：“WSLg 将启动一个配套的系统发行版本，包括 Wayland, X server 和 audio server，以及让 Linux GUI 应用和 Windows 通信所需要的所有东西”。Windows 10 预装 WSLg 功能是一个令人兴奋的发展，因为它模糊了 Linux 和 Windows 10 之间的界限，粉丝们得到了两个世界的好处。</p><p style="text-align: left;"><strong>安装 WSLg</strong></p><p style="text-align: left;">想要使用 WSLg 功能，用户首先需要升级到最新的 Windows 10 预览版。在写这篇文章的时候，最新 Build 版本号是 21390。在安装最新 Windows 10 预览版之后，你就可以安装 Windows Subsystem for Linux，并安装一个新的 Linux 发行版。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0601/f177f0ad41ceecd.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0601/f177f0ad41ceecd.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">例如，要安装支持 GUI 的 Windows Subsystem for Linux（WSL） 并自动安装 Ubuntu，你可以在 Windows 10 管理员权限命令提示符下输入 <strong>wsl --install -d Ubuntu</strong>。对于已经安装 WSL 的用户来说，你需要更新WSL以增加对WSLg引擎的支持，使用<strong>wsl --update</strong>命令，然后是<strong>wsl --shutdown</strong>，如下图所示。</p><p style="text-align: left;">一旦WSL被更新，你需要使用<strong> wsl --list -v</strong> 命令确保你安装的 Linux 发行版被配置为使用 WSL 2。如果你想使用的发行版是版本1，你需要使用<strong>wsl --set-version _distro_name_ 2</strong>命令将其升级到版本2。例如，要把kali-linux升级到WSL2，我们要在命令提示符下输入 <strong>wsl --set-version kali-linux 2</strong>。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0601/8081125275f046b.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0601/8081125275f046b.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">为了在 WSLg 中获得更好的性能，微软建议你安装以下预览版驱动程序。用于 WSL 的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> GPU 驱动，用于 WSL 的 Intel GPU 驱动，以及用于 WSL 的 NVIDIA GPU 驱动。一旦你的发行版被升级，你现在可以使用WSLg来运行Linux GUI应用程序。</p><p style="text-align: left;"><strong>在 Windows 10 中运行 Linux GUI 应用程序</strong></p><p style="text-align: left;">在过去十年中，随着Linux作为桌面操作变得越来越流行，越来越多的软件开发商开始推出适用于 Linux 的版本。例如，适用于 Linux 的 Edge、Chrome、VLC 等等。接下来，外媒演示了目前只能通过 WSLg 功能才可以运行的 Linux GUI 应用。</p><p style="text-align: left;"><strong>● Nautilus</strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0601/d94b85e21473911.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0601/d94b85e21473911.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Nautilus 是 GNOME 桌面环境的官方文件管理器。虽然你可能会发现在 Windows 10 Nautilus</p><p style="text-align: left;">nautilus</p></blockquote><p style="text-align: left;"><strong>● Stacer</strong></p><p style="text-align:center"><a referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Stacer 是一个开源的系统优化器和 Linux 的应用程序监视器/任务管理器。</p><p style="text-align: left;">你可以用以下命令安装和运行 Stacer</p><blockquote style="text-align: left;"><p style="text-align: left;">sudo apt install stacer</p><p style="text-align: left;">stacer</p></blockquote><p style="text-align: left;"><strong>● Timeshift</strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0601/09728b8a40ca950.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0601/09728b8a40ca950.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Timeshift是一个Linux的备份软件，可以用来创建文件系统的快照。</p><p style="text-align: left;">你可以用以下命令安装和运行 Timeshift：</p><blockquote style="text-align: left;"><p style="text-align: left;">sudo apt install timeshift</p><p style="text-align: left;">sudo timeshift-gtk</p></blockquote><p style="text-align: left;"><strong>● Hedgewars</strong></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0601/92c57162cc561a5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0601/92c57162cc561a5.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">虽然 Hedgewars 不是一个 Linux 专用的游戏，但我想把它包括进来，以表明即使是游戏也可以在 WSLg 下运行。虽然 WSLg 很可能不是为游戏而设计的，但你可以用它来玩游戏的事实表明了这个新功能的全部深度。</p><p style="text-align: left;">你可以用以下命令安装和运行Hedgewars。</p><blockquote style="text-align: left;"><p
