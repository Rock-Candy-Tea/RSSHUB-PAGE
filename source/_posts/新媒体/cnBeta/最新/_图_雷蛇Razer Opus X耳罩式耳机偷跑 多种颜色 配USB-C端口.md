在今天早些时候发布 Hammerhead True Wireless X 新款耳机之前，雷蛇（Razer）对 Android 端的 Razer Audio 应用进行了更新。在最新的 3.2.0.924 版本更新中，在罗列的兼容性列表中出现了一个尚未发布的新设备。该应用中显示兼容 Razer Opus X，这应该是尚未发布的耳罩式耳机，有多种有吸引力的颜色。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0609/8be4a82a9ecdbc4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0609/8be4a82a9ecdbc4.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在该应用中，新的 Razer Opus X 内部代号为“samantha_t2”。该列表显示该耳机会有 X Edition, Black Edition, Green Edition, Mercury Edition 和 Quartz Edition 版本。在应用程序中还提供了 Quartz, Green 和 Mercury 三种颜色，至于 X Edition 和 Black Edition 并未出现。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0609/7664b86a715b61a.jpg" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0609/e2fad02f5988aff.jpg" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0609/4e10322d423a9a7.jpg" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0609/f9bfaa73f420c5c.jpg" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0609/0aad57312d376ca.jpg" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0609/d1875c50159a482.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">从这些图片中，我们可以看到，新耳机将有一个USB-C端口用于充电，并与第一款Opus耳机的按钮布局相同。有一个电源按钮，一个音量减小按钮，一个音量增大按钮和一个多功能按钮。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0609/162cafa5a264fd7.jpg" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0609/0921a28bee931e3.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">Razer Audio 应用程序本身并没有透露新的 Opus X 耳机的许多细节。然而，由于新发布的 Hammerhead True Wireless X 似乎是 Hammerhead True Wireless Pro 的更实惠版本，我们有理由认为 Opus X 将是该公司初代 Opus 设备的更实惠版本。</p>
