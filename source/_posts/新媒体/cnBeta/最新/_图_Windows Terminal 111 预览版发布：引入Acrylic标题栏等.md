面向 Windows Insider 项目成员，今天微软推出了 Windows Terminal 1.11 预览版。新版本带来了包括 Acrylic 标题栏等新功能。新版本引入将面板移动到标签的功能，允许用户将一个已经打开的面板移动到现有新的或者现有的标签。<br>
 <p>Windows Terminal Preview 1.11 可以从 <a href="https://www.microsoft.com/en-us/p/windows-terminal-preview/9n8g5rfz9xk3?rtc=1">Microsoft Store</a> 或者 <a href="https://github.com/microsoft/terminal/releases">GitHub</a> 下载</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0901/8931d0e9868e000.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0901/8931d0e9868e000.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">在新版本中允许用户在一个标签中交换面板，以及在 Context 视图中拆分成标签的能力。这些功能应该使 Windows  Terminal 的多任务处理更容易。对于这个功能，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还特别感谢 Schuyler Rosefield 的大力贡献。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/0901/a3ba5d8b7e62120.gif" alt="ezgif.com-gif-maker(1).gif" referrerpolicy="no-referrer"></p><p style="text-align: left;">除此以外，还有一个新的设置选项，使你的标题栏变成 Acrylic 半透明状态。这是在设置 UI 的外观页面，它可以在你的全局设置中设置，不过你需要重新启动你的终端才能看到区别。我们为你指出了下面的其他变化。</p><blockquote style="text-align: left;"><p style="text-align: left;">● 当添加按键到你的动作时，你现在只需要输入按键和弦，而不是拼出所有的按键（即c-t-r-l）。</p><p style="text-align: left;">● 适用于你的配置文件的非聚焦时的外观设置，现在在设置UI中。</p><p style="text-align: left;">● 字体对象现在可以接受settings.json文件中的OpenType特征和轴。</p><p style="text-align: left;">● 你现在可以选择将你的终端最小化到系统托盘。为这个功能添加了两个新的全局布尔设置</p><p style="text-align: left;">● 你现在可以将目录和文件拖放到"+"按钮上，然后使用给定的起始路径打开一个新的标签、窗格或窗口。</p><p style="text-align: left;">● 当通过默认的终端设置启动终端时，终端现在将使用无配置文件而不是你的默认配置文件。</p><p style="text-align: left;">● 你现在可以通过使用intenseTextStyle配置文件设置来选择你希望强烈的文本在你的终端中出现的方式。你可以将你的样式设置为粗体、亮体、粗体和亮体，或者不添加任何额外的样式。</p></blockquote><p
