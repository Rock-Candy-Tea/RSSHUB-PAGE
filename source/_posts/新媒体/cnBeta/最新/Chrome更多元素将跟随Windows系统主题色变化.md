在最近几年，Google Chrome 浏览器的用户界面并没有发生太大的变化。通过内置或者 Chrome Web Store 上的主题，以及几个个性化选项来定制新标签页以及一些偏好设置等等。<strong>Chrome 浏览器标题栏可以跟随 Windows 系统的主题色，Google 现在正努力将浏览器的其他元素也跟随系统主题色变化。</strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1209/f2b4acf6ef0cdb7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1209/f2b4acf6ef0cdb7.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">最近，Google 已经更新了 Chrome 用户界面，以配合 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 设计风格。在 Chrome 96 稳定版中，用户不再需要启用任何 Flag 实验标志就可以使用。Chromium 团队现在正准备确保 Chrome 在标题栏和边框之外，在更多部分广泛使用 Windows 的主题色。</p><p style="text-align: left;">Chrome 开发者在功能错误页面上写道：“我们非常乐于看到 Windows 系统上的主题色引发的乐趣”。今后，Chrome 浏览器的按钮、焦点和其他元素也将遵循 Windows 系统的主题色。例如，在第一张截图中，你可以注意到，书签星形图标和它的对话按钮以及突出显示的书签名称使用了 Windows 中设置的主题色。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2021/1209/d66297a947fcf1a.jpg" referrerpolicy="no-referrer"></p>
