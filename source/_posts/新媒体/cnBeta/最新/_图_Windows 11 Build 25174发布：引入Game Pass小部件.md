微软面向 Dev 频道的 Windows Insider 项目成员，今天发布了 Windows 11 Build 25174 预览版更新。本次更新的升级幅度并不大，主要修复了存在于近期版本中的一些 BUG，并带来了一些能改善体验的实质性改进。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0804/d1345044bfc1d7e.webp" alt="x9qezom8.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">此外还值得一提的是，该预览版更新和与任何特定版本无关，并且大部分改进今后将通过较小的每月累积更新分发给其他频道用户。与其他所有预览版本一样，如果您在 Dev 频道中，今天的版本也将显示为“Windows 11 Insider Preview 25174.1000 (rs_prerelease)”。如上所述，Windows 11 Build 25174 并不是一个大的功能版本，但它修复了文件资源管理器和任务栏的问题。</p><p style="text-align: left;">除了一长串错误修复之外，微软还在 Windows 11 小部件板上添加了一个新小部件。这个新的小部件被称为“Game Pass”（一项提供基于订阅访问数百个高质量 PC 游戏的服务），它是 PC 端 Game Pass 的一个入口。</p><p style="text-align: left;">Windows 11 Build 25174.1000 主要改进内容如下：</p><blockquote style="text-align: left;"><p style="text-align: left;">Game Pass 可让您浏览新添加的游戏、即将下架的游戏以及突出显示类别中的其他内容。微软仍在开发 Game Pass 小部件，未来某个时候将添加一些“令人兴奋”的功能。</p><p style="text-align: left;">要开始使用 Game Pass 小部件，请使用以下步骤：</p><p style="text-align: left;">1. 打开 Game Pass 小部件</p><p style="text-align: left;">2. 单击用户个人资料旁边的 + 按钮或游戏通行证旁边的 +。</p><p style="text-align: left;">3. 您现在可以访问 Game Pass 小部件的所有功能。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0804/ef81b09ebe1fd56.webp" referrerpolicy="no-referrer"></p></blockquote><p style="text-align: left;">Windows 11 Build 25174 改进和修复</p><blockquote style="text-align: left;"><p style="text-align: left;">微软表示，它修复了当您尝试打开多个 explorer.exe 实例时文件资源管理器可能崩溃的问题。另一个错误已修复，即单击导航窗格中的文件夹可以在新选项卡中打开该文件夹，而不是当前选项卡。</p><p style="text-align: left;">微软还修复了文件资源管理器的深色模式问题，其中某些区域会以浅色模式呈现。修复了左/右箭头没有足够的对比度来显示它们是否已启用的错误。</p><p style="text-align: left;">Windows 11 Build 25174 改进：</p><p style="text-align: left;">● 修复了导致文件资源管理器在全屏模式下出现 UI 问题的错误。</p><p style="text-align: left;">● 修复了意外呈现任务栏图标的错误。</p><p style="text-align: left;">● 修复了当用户访问系统 > 存储 > 磁盘和卷时设置应用程序崩溃的错误。</p><p style="text-align: left;">● 修复了 Windows 安全中的内存完整性可能会显示兼容性驱动程序警告的错误。</p></blockquote><p style="text-align: left;">如何安装 Windows 11 Build 25174</p><p style="text-align: left;">要安装 Windows 11 Build 25174，请执行以下步骤：</p><blockquote style="text-align: left;"><p style="text-align: left;">1. 注册 Windows 预览体验计划。</p><p style="text-align: left;">2. 打开设置页面。</p><p style="text-align: left;">3. 切换到开发频道。</p><p style="text-align: left;">4. Windows 11 内部版本 25174 更新</p><p style="text-align: left;">5. 单击“检查更新”，然后选择“下载并安装”。</p><p style="text-align: left;">6. 点击“立即重启”。</p></blockquote><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2022/0804/a90df3bbc41980c.webp" referrerpolicy="no-referrer"></p>
