上周，TikTok 悄然推出了名为 Live Studio 的 Windows 直播软件。<strong>但由于使用了来自 Open Broadcaster Software 旗下流行的 OBS Studio 应用程序和其它开源项目的代码，该公司很快被推上了“未遵守开源许可协议”的风口浪尖。</strong>上周晚些时候，@Naaackers 在 Twitter 上发布了一系列推文，指出 TikTok 直播软件实质上是对 @OBSProject 项目的一个分叉。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1221/0ebcdabc99d8558.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1221/0ebcdabc99d8558.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p>Open Broadcaster Software 业务开发经理 Ben Torell 证实，其团队在与 Protocol 联系时发现了这些违规行为的“明确证据”。遗憾的是，虽然项目团队主动联系，但 TikTok 方面尚未给予回应。</p><p><img src="https://static.cnbetacdn.com/article/2021/1221/89326382386f4a9.png" alt="0-1.png" referrerpolicy="no-referrer"></p><p>上周，TikTok 并未大张旗鼓地发布了其 Live Studio 应用程序。这款 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 客户端旨在帮助人们实现高质量的直播推流，同时方便地整合游戏视频流、以及图像和文本等叠加图层。</p><p><img src="https://static.cnbetacdn.com/article/2021/1221/c9789d29fc1c6e5.png" alt="0-2.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">@Naaackers 后续指出“偷窃”开源项目的定义仍有待商榷</p><p>该公司在接受外媒采访时称，当前 TikTok Live Studio 应用程序仅面向少数市场的数千名用户提供早期体验。</p><p><a href="https://static.cnbetacdn.com/article/2021/1221/c755df1c4630ebc.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1221/c755df1c4630ebc.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>至于大名鼎鼎的 OBS Studio，其经常被第三方拿来二次开发、以“套皮”形式改善特定场景下的使用体验，本次 TikTok Live Studio 桌面客户端所陷入的争议也与此有关。</p><p><a href="https://static.cnbetacdn.com/article/2021/1221/0bed64ede5f3900.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1221/0bed64ede5f3900.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>若严格遵循 OBS Studio 指定的 GPL 条款，第三方项目也必须以相同的许可来公开任何经过修改的源码。若违规，很可能被权利主张方告上法庭。</p><p><img src="https://static.cnbetacdn.com/article/2021/1221/ca3aa7d97e872f7.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>即便如此，Ben Torell 还是不希望最终走到这一步。他在 Twitter 上表示，Open Broadcaster Software 正在寻求避免法律上的对抗。</p><p><img src="https://static.cnbetacdn.com/article/2021/1221/f11335fe6080434.png" alt="0-3.png" referrerpolicy="no-referrer"></p><p>Ben Torell 写道：“我们承诺会真诚地处理违反 GPL 协议的行为，但就 TikTok / Bytedance 而言，只要他们愿意遵守许可，我们仍然很乐意与之建立友好的合作关系”。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1215317.htm"
