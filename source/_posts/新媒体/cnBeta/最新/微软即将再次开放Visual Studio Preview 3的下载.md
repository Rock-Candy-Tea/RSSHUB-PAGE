微软曾于今年早些时候放出了 Visual Studio 2022 Preview 3，结果因内容交付网络（CDN）方面的问题而迅速暂停了下载。<strong>好消息是，该公司将很快开放 VS 2022 新预览版本的再次下载，并且引入了一些新功能。</strong>比如新的项目设计器，对黑暗主题的支持改进，以及对使用 React 和 Vue.js 进行前端开发的新项目类型的支持。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0811/649585f636d8032.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0811/649585f636d8032.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">Preview 3 将很快开放下载（来自：Microsoft <a href="https://visualstudio.microsoft.com/zh-hans/vs/preview/" target="_self">官网</a>）</p><p>据悉，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>于 6 月推出了 VS2022 的首个预览版本，并着重于 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 平台的 64-bit 支持。后续的两个预览版本，则一直在添加新功能。</p><p>用户交互方面的最大变化之一，就是对深色主题的改进。其采用了不那么强烈的对比色，通过较少的使用而减轻眼睛疲劳。</p><p>为提升生产力，Preview 3 还包括了一个新的 .NET SDK 项目的属性设计器，使之更易于使用和浏览。</p><p>其使用了单列设计界面、并提供了清晰的描述，使项目更容易被识别。辅以搜索功能，以节省查找相关内容的时间。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=279257695&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">Visual Studio 2022 Preview 3 Overview（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMjc5MjU3Njk1LnNodG1s.html" target="_self">via</a>）</p><p>此外微软强调了更多功能，比如一次处理多个 Git 存储库的能力，有一个新的诊断分析器来查找问题的根本原因。</p><p>新增对 Angular、Vue.js 和 React 等项目类型的支持，以及可用各种环境开展远程测试，包括 Linux 容器、Windows Linux 子系统（WSL）、以及通过 SSH 进行连接。</p><p>感兴趣的朋友，可移步至微软官网下载 Visual Studio 2022 的最新预览版本。不过截止发稿时，该公司尚未更新 Preview 3 的资源链接。</p>
