<strong>自 Xbox Game Pass 推出以来，大家普遍存在两方面的争议 —— 到底 XGP 有助于提升游戏的销售，还是起到了反面效果呢？</strong>WCCFTech 指出，对于微软来说，Xbox Game 服务本身是相当成功的。因为这家巨头可通过付费订阅制，弥补第一方游戏销售的任何损失。但是对于其它发行商来说，XGP 是否损害到了它们的底线利益？<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1215/48d0d9c8fb84f95.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1215/48d0d9c8fb84f95.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/xbox-game-pass-helps-sales-if-good-hurts-if-bad/" target="_self">WCCFTech</a>）</p><p>周二的时候，NPD Group 的 Mat Piscatella 在一连串推文中回答了大家的疑问 —— 可知在大多数情况下，答案都是否定的。</p><p><img src="https://static.cnbetacdn.com/article/2021/1215/080a3c3ddb058e3.png" alt="1.png" referrerpolicy="no-referrer"></p><p>首先可以可定的是，将游戏放入 Game Pass 目录，确实能够放大消费者的兴奋度和销量。但前提是 —— 游戏本身也得受到广泛好评。</p><p>若游戏本身效果一般、口碑不够火爆，加入 Game Pass 订阅反而会放大负面情绪、并损害后续的销售。</p><p><img src="https://static.cnbetacdn.com/article/2021/1215/06e69b4981a6720.png" alt="2.png" referrerpolicy="no-referrer"></p><p>换言之，<a data-link="1" href="https://microsoft.pvxt.net/e4yLO" target="_blank">Xbox</a> Game Pass 可以成为一款强大的工具，但前提是你使用得当。</p><p>以《极限竞速：地平线 5》为例，尽管它加入了 XGP 首发阵容，但这款特许经营游戏本身在发布首月就创下了耀眼的销售记录。</p><p><img src="https://static.cnbetacdn.com/article/2021/1215/f9bddac28b541bd.png" alt="3.png" referrerpolicy="no-referrer"></p><p>与此同时，其它一些 XGP 作品 —— 比如 SE 社的《Outriders》—— 反响就没有那么热烈了。</p><p>而且在登陆 Game Pass 之后，甚至大多数游戏距离做到“有意义”都还有很长一段路要走。</p>
