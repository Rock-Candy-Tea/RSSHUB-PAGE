为了帮助广大客户了解当前设备是否有符合 Windows 11 的升级条件，微软已经推出了一款名为“PC Health Check”的检查工具。<strong>然而对于微软强制运行 Windows 11 操作系统的设备安装 TPM 2.0 可信任平台模块一事，用户的吐槽声已经不绝于耳。</strong>但对于第三方软件开发商来说，这或许是他们未来都将遵循的一套设计准则。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/0628/b9e61ae9925a6ad.jpg" alt="msi tpm 2.0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">微星<a href="https://www.msi.com/Motherboard/TPM-20-Module" target="_self">提供</a>的 TPM 2.0 模块</p><p>按照<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>的设想，近年来销售的大部分 PC 来说（尤其是笔记本 / 台式机），其实都已安装并可启用 TPM 2.0（<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> 8 代处理器 / <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 2000 以上）。</p><p><img src="https://static.cnbetacdn.com/article/2021/0906/0a21c6478fe128a.png" alt="0.png" referrerpolicy="no-referrer"></p><p>与此同时，软件 / 游戏开发商也可借助加密处理器来增强安全性与反作弊措施。</p><p><img src="https://static.cnbetacdn.com/article/2021/0906/d9b3d63d2fd66c7.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>比如 Riot Games 旗下的《Valorant》射击游戏、或者被版权方用于实施某种数字版权保护（DRM）来挫败，只是目前尚不清楚该要求是否适用于 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11（甚至 Windows 10）。</p><p><img src="https://static.cnbetacdn.com/article/2021/0906/7b6ddcf3f3b8b87.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>当然，这并不意味着缺少 TPM 2.0 安全模块的设备用户就无法玩上《Valorant》。</p><p><a href="https://static.cnbetacdn.com/article/2021/0906/ed630880ea2d365.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0906/ed630880ea2d365.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>目前最大的问题的是，即使 TPM 2.0 不会对符合条件的 Windows 11 PC 造成问题，但并非所有 PC 都能满足微软的最低要求、且这部分设备用户也无法顺利安装上 Windows 11 。</p><p><img src="https://static.cnbetacdn.com/article/2021/0906/be8f009f70fa551.png" alt="4.png" referrerpolicy="no-referrer"></p>
