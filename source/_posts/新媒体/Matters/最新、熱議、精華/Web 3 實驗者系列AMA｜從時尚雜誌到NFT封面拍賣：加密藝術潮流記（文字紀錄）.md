3 實驗者系列AMA｜從時尚雜誌到NFT封面拍賣：加密藝術潮流記 </h2><p>日期：三月二十九日（周二） </p><p>時間：21：30 － 22：30（東八區） </p><p>嘉賓：<a class="mention" href="https://matters.news/@jeterchou" target="_blank" data-display-name="周青永" data-user-name="jeterchou" data-id="VXNlcjoyNzk0">﻿<span>@周青永</span>﻿</a> （Vogue Taiwan 藝術總監、NFT 實驗者） </p><p>對談人：<a class="mention" href="https://matters.news/@az" target="_blank" data-display-name="張潔平" data-user-name="az" data-id="VXNlcjo2">﻿<span>@張潔平</span>﻿</a> （Matters Founder） </p><p>嘉賓介紹：Vogue Taiwan 藝術總監、擁有20多年數位設計經驗；NFT實驗者，曾策劃時尚雜誌Vogue封面鑄造成NFT拍賣、推動不同單位參與Web3藝術共創的項目。</p><p>影片重溫：<a href="https://youtu.be/6TZM2TgdPOE" rel="noopener noreferrer" target="_blank">https://youtu.be/6TZM2TgdPOE</a></p><figure class="image"><img src="https://assets.matters.news/embed/75e423e9-cd72-426b-9daf-c6958649a7e1.jpeg" data-asset-id="75e423e9-cd72-426b-9daf-c6958649a7e1" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><strong>Q：可否說說你是怎樣走進web3世界的？當初為何產生好奇？是藝術吸引了你？還是其他元素。</strong></p><p>Jeter：</p><ul><li>90年代末期進入職場，也是網路技術起飛的時候，從那時開始我的職業生涯便已經與 Web 技術的成長綁定在一起。</li><li>看到 Web3 的出現回想起當年剛接觸 Web 技術的興奮感，也曾嘗試用電腦參與比特幣挖礦，但是硬體不夠力就放棄了。</li><li>當初潔平創立了matters.news，讓我發現原來區塊鏈寫作已經發展到新的階段，會使潔平願意放棄當時的工作轉而創立以區塊鏈為底層技術的平台，也讓我馬上登記了第一批的等待名單。後來 travelogger 發行，看到前景，能夠參與到 Matters 之後的系列與區塊鏈的實驗，也讓我很興奮。</li></ul><p><br></p><p><strong>Q：可否跟我們分享一下vogue為何會有把封面鑄造成nft的想法？當時這個項目是如何誕生的？（還可介紹你曾參與過的項目）</strong></p><p>Jeter：</p><ul><li>與背景相關，一路上都是做與技術相關的事項，在接任 VOGUE 台灣藝術總監的位置後也在數位上去做嘗試。前年在時裝週的時候曾與國內指標性的 XR 技術單位「實境共創」合作，打造了當時堪稱國內最大規模的 AR 體驗街區。基於這樣的背景，公司內部也一直在規劃進入區塊鏈市場的時機。</li><li>當時做NFT是出自一個浪漫的想法：「雖然區塊鏈本質上是基於科技產生的，但是它有一個很浪漫的特性是，它代表永誌不忘，能夠將資訊刻在上面使其被永久紀錄下來。」</li></ul><p>潔平：</p><p>NFT是可以包含一段時空的。數位世界的「東西」可以包涵時間維度和其演變過程——它的所有權設計方式，可以內含一段時間的演進，可以來自超過一個人的協作。我們希望用這樣的設計去推進創作。</p><p><br></p><p><strong>Q：當中有遇到什麼困難嗎？例如讀者的反應如何？你們怎樣把這件事推動起來。</strong></p><p>Jeter：</p><ul><li>一開始覺得很可行，覺得是好的時機，潔平說沒錯，我們也擔心這案子推出做不好，會對品牌的形象做成傷害。 </li><li>主要的考量點為：</li></ul><ol><li>要賣給誰？</li><li>金流的問題，畢竟區塊鏈不算是真正流通的貨幣</li><li>當地法律的問題</li></ol><ul><li>這些方面還是需要跟總部溝通，我們的品牌始終要交代，後來請教了不少專家，最知名是以「NFT傳教士」自稱的寶博士等，給了我們寶貴的意見。</li><li>對跟既有讀者溝通來說，相對是公關性質的，我們要真正面向幣圈人士，但我們要如何打進這個市場？</li><li>大家了解時尚產業主要的對象跟專業玩家重疊性不高。我們透過在ig直播跟大家說我們有這個計劃。我們最後決定把foundation這當時不太熱的平台，但這平台較具藝術性。</li><li>最終的拍賣成果達到了 30 ETH，成為了全球集團內單一售價最高紀錄，目前還沒被打破。</li><li>集團內不約而同開始進行了 NFT 的發售，形成一種競爭格局。</li></ul><p><br></p><p><strong>Q：當年的多媒體與現在的新媒體在感受上有什麼差別或是相似之處？</strong></p><p>Jeter：</p><p>當年的技術還不夠成熟，相對來說進入門檻也不高，創作上比較容易。現在的 Web3 在技術上更新了諸多版本，雖然有熱情但是學習起來會比較困難些，有種心有餘而力不足的感覺。</p><p><br></p><p><strong>Q：有沒有同輩跟你一樣，一起經歷了90年代web的浪潮以及web3這波的浪潮，同代人多嗎？還是現在只有年輕的人在玩web3？</strong></p><p>Jeter：</p><p>同代人有但很少，有一群人是做個人網站起家的站長，進入web3的只有很少：一個。</p><p><br></p><p><strong>Q：你當時也是自己做網站的站長？ </strong></p><p>Jeter：</p><p>是的，當時有一家電台訪問了一百多個站長，我們有一個群體。其中有一位當年是站長，現在於 EchoX - NFT發行NFT，是跟我從當時一直走到現在的。</p><p><br></p><p><strong>Q：一家實體的潮流雜誌社（媒體）跟web3世界接軌，你覺得意味著什麼？</strong></p><p>Jeter：</p><ul><li>時尚產業會較重視 NFT 憑證的特性與應用場景，會希望能夠與消費者錢包進行互動，以時尚產業來說也非常期待去推動消費者經濟的部分，區塊鏈是一個很好的切入點。</li><li>現階段而言更多的是在嘗試，會去看待區塊鏈能夠反應的藝術性，同時也相當重視如何以環保的形式來發行 NFT。</li><li>從受眾的角度而言，NFT 對時尚產業來說會傾向於接觸到新受眾的好機會，透過 NFT 的渠道開創出新的跑道。對品牌來說也會希望能夠接觸到更多的受眾，有助於推廣品牌的影響力。</li></ul><p><br></p><p><strong>Q：以你作為經驗者、觀察者、推動者的身份，在你的觀察裡，NFT有什麼可能性？當中的共創、創造如何？這個媒介在業界裡又有著怎樣的位置？</strong></p><p>Jeter：</p><ul><li>還有一段蠻遙遠的距離，主要是硬體還沒到位，人們對於 Metaverse 的想像還停留在大型的 MMORPG。若是時尚產業對於遊戲或是虛擬空間沒有概念的話，其實不太會關注相關的發展情況與應用場景。</li></ul><p><br></p><p><strong>Q：這個新興媒介，如何混合潮流、藝術、設計和媒體產生新的創造？有令你腦洞大開的系列／項目你想分享一下嗎？是怎樣的？</strong></p><p>Jeter：</p><ul><li>NFT 在發展的初期許多都是與 MEME 有關聯性的，以台灣項目來說比較具有代表性質的應該是在高雄被稱為都市傳說的陳財佑（以維修漏水出名），他的兒子為了讓品牌能夠永續化經營，便將陳財佑的照片進行切割，以共同持有的方式販售 NFT，持有者可以獲得品牌 T-shirt，其中有特卡可以獲得不限次數的漏水維修。以傳統產業進入 Web3 而言，這是一個很成功的例子。</li><li>以時尚產業來講，發展的方向會是在「紙娃娃系統」上面，符合時尚產業的想像。近期 「GUCCI」 的創意總監與知名 NFT 配件品牌 「10KTF」 合作的案子也很有趣，這個專案能夠為 「10KTF」 合作的PFP專案打造獨家的GUCCI服飾，因此你可以看到BAYC、MAYC或是Cool Cats等NFT換上GUCCI風格的外觀。</li></ul><p><br></p><p><strong>Q：知道你也持有Traveloggers，在你看來，創作／創作者經濟，在Web3可以有怎樣的發展？</strong></p><p>Jeter：</p><ul><li>收藏 NFT 會在意後續的賦能專案，不像傳統的 PFP 類型 NFT 方案， Matters是基於現有的平台去發行 NFT。購買 <strong>Traveloggers </strong>是期許日後可以參與到更多有趣的實驗性活動，畢竟 Matters 是倚賴於區塊鏈的底層技術，會有較良好的體驗。</li><li>對 NFT 共創比較有感的是前陣子的 <strong><u>AssangeDAO</u></strong> ，此專案是跟自由相關的，許多人都留下了與「自由」相關的語句。我個人用的是「小王子」作品的法文命名來當做 Censored 的語詞，動機是因為「小王子」講述的是「心靈的自由」，認為與專案的契合度很高。</li></ul><p><br></p><p><strong>Q：怎麼看阿妹的 NFT Project</strong></p><p>Jeter：</p><ul><li>蠻理想的 Project ，為了節能選擇與主打低 gas fee 與綠能永續的 Tezos 鏈合作，也與亞洲第一個以 Tezos 為基礎的藝術交易平台 akaSwap 合作。出發點的很好的，但是對於初入用戶來說還是有相對應的使用門檻。</li></ul><p><br></p><p><strong>Q：從站長走到現在，會期待 Web3 給創作者經濟帶來怎樣的發展</strong></p><p>Jeter：</p><ul><li>從「直播」的形式與產業興起後，「觀眾 Donate」成為了支持創作者的重要管道，但是因為直播平台還是中心化的，因此多數的收益還是會進到直播平台的口袋當中，無法直接回饋給創作者們。</li><li>在未來去中心化的世界裡面，雖然難免收到平台的手續費抽成，但是可以期待創作者們可以在利潤的分配上可以有更高的掌控權，也期待 Matters 會去做怎麼樣的嘗試。</li></ul><p>潔平：</p><ul><li>從 2018 年開始做 Matters 的時候，花了很長的時間才將問題聚焦清楚，即便創作者能夠得到直接的回報，但是天花板還是壓得很低，要直接接觸到支持者或是讀者本身會是很高的門檻。</li><li>當作品以 NFT 的形式發送的時候，創作者與支持者之間便形成了一種社群，這樣一個個的小生態將逐漸組成一個龐大的生態。</li><li>小生態可以讓創作者的日子好過很多，不僅是一群人有機綁定在一起的，讀者對作者的捐贈像是一種投資（捐贈投資化），作者本身的勞動被資本化與金融化，可獲得長期回報，但也可能造成貧富不均。</li><li>將框架從一次性的買賣打破，轉為長期的投資回報模式。</li><li>目前還處於早期，跳進來做實驗或是嘗試，無論結果是好是壞都會對後人產生幫助。</li></ul><p><br></p><p><strong>對「公共空間」與「共創」的看法</strong></p><p>潔平：</p><ul><li>對古典定義上的公共空間有很強的執念，從 90 年代的Web 浪潮到現在 Web3 時代的區別，現在這波像是對已建置互聯網的反叛。</li><li>在意的是公共空間的消失，現在的整個概況更像是城邦的割據，由各自的互聯網帝國切斷彼此的交流空間，能夠有集體智慧湧現的公共空間不復存在。</li><li>Web3 是一個理想的技術底層，保障了公共空間的交流權益不被侵蝕。</li><li>執念是來自於對公共空間的鄉愁。</li></ul><hr><h2>社區提問：</h2><p><strong>Q：現在沒有比90年代更「狂熱」是不是因為Web3對大多數人來說，能夠切入的點比較少？ 因為比如目前能夠看到在進行中的Web 3 content creators行業也比較不廣泛？</strong></p><p>Jeter：</p><ul><li>以目前商業上要去做區塊鏈應用來說，區塊鏈還是缺少了殺手級的應用，離進入 METAVERSE還是蠻遠的。前陣子參加了舉辦在 Decentraland 的時裝周。以參與者的體驗來說，還是太吃硬體設備了，常常遇到黑畫面的狀況。</li><li>在參與者的方面，裡面更多的是 Decentraland 的原生用戶，時尚圈只有本來就在關注的同事們會參與這類的活動。</li></ul><p><br></p><p><strong>Q：參與時裝周的體驗？</strong></p><p>Jeter：</p><ul><li>以同行來說參與人數還是少數，只有對 Web3 有興趣的人會討論。</li><li>舉辦情況的話，主要就是打造時尚街區，並舉辦走秀，開放購買實體物品（展現 3D 樣貌）。</li><li>規劃部分，在進入DL入口的時候可以領取衣服與眼鏡，可以快速跳到不同的主題區，且可以領取 POAP 徽章。</li></ul><p><br></p><p><strong>Q：Gucci 推出的 NFT「Super Gucci」，Jeter 有什麼見解嗎?（對於 International 的 brand 也開始發布自己的 NFT item 看法）</strong></p><p>Jeter：</p><ul><li>普遍處於試水溫的狀態，「GUCCI」 在 Web3 方面也做了相當多的嘗試，採取了多線平行發展的模式。</li><li>以 KOL 的角度來看，暫時還無法達到街頭霸王（一個虛擬樂團）的等級。</li><li>多少還是有先行者的優勢，找到自己進入區塊鏈的最佳模式，較有資本也有犯錯空間。</li></ul><hr><h2>本週五還有一場AMA：<a
