Release 下载汝的操作系统适用的 IPFS Desktop： <a href="https://github.com/ipfs/ipfs-desktop/releases" target="_blank">https://github.com/ipfs/ipfs-desktop/releases</a></p><p>IPFS Companion 可以在浏览器中查看 IPFS Desktop target="_blank">https://github.com/ipfs/ipfs-companion</a></p><p>安装好后的 IPFS Desktop 大概像这个样子（也可以从浏览器打开 <a href="http://localhost:5001/webui" target="_blank">http://localhost:5001/webui</a> referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>向 IPFS 上传和固定文件</h2><figure class="image"><img referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>在 Files （文件）分类可以查看汝上传的文件，Pins 中列出了汝有在固定的文件。</p><p>想上传文件的话，就用旁边的 Import 按钮吧，可以选择文件夹或者文件进行导入。</p><figure class="image"><img
