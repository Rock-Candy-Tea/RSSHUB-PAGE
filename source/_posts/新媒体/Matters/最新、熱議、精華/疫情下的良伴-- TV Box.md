<p>拜去年搬家所賜，買了一個電視盒，取代原本的有線電視。又因這兩年疫情的蔓延，去年政府甚至還實施了一段時間的三級警戒，連電影院都沒辦法去，剛好讓這部電視盒的功能發揮到極致，不論是最新的院線片，最夯的劇，或是經典老片的回顧，都讓我大大讚歎這電視盒子的物超所值。當然，我也絕對不會承認這是居家辦公政策中，除了減少染疫機率，節省交通費之外，額外的一項收穫~ </p><p>說到這裡，就不得不提出一個爭議話題: <strong>尊重智慧財產權! </strong></p><p>主要就是因為某職業籃球聯盟執行長使用 TV Box，在家中自拍觀賞東京奧運露餡，導致大眾批評身為一個公眾人物，為何不懂尊重智慧財產權的重要，此時，政府官員也跳出來批判，TV Box 所提供的內容多半有侵權的嫌疑，業者不應出售侵權產品，消費者應如何自重云云。</p><p><strong>問題就來了: 如果 TV Box 這麼萬惡，政府為何不禁賣? </strong></p><p>就像毒品，嚴重傷害身體健康，並且容易衍生其他犯罪行為，我們就明文立法禁賣、禁用。大家就乖乖遵守法律，清清楚楚。</p><p>然而政府的解釋是因為 TV Box 是上網裝置，其實也無權、無法禁賣，只能約束消費者行為。那不就對了? 請問<strong>手機 / 平板 / 筆電 / 桌機</strong>哪一個不能上網? 在使用上，不管上傳或下載，這些裝置甚至還可以由使用者來 "創作" 出侵權的作品，豈不更邪惡? 怎不見政府出來呼籲、批判這些產品的使用者?</p><p>政府該做的是呼籲「尊重智慧財產權」，不管你用甚麼裝置，都應該要收看合法授權的數位影音內容，而不是只做<strong>政治正確</strong> ( TV Box 多為陸製產品 ) 及<strong>討好利益團體</strong> ( 影響最大的就是有線電視的業者 ) 的批判單一裝置的使用。</p><p><br></p><blockquote>至於內含在 TV Box 中，被指控的各項不法 App，那是其他處理層次的問題，例如:</blockquote><ul><li>中國大陸官方打擊盜版的政策與執法。</li><li><strong>HBO </strong>& <strong>Netflix</strong> 等影片製作或提供網路影音串流服務的公司，對相關不法 App 業者的法律作為。</li><li>台灣政府對打擊違法產品或業者的法律依據與執法決心。</li></ul><p><br></p><p>把立法的缺失、漏洞加上執法的不力，最後直接加諸在消費者的道德良知與法律素養的批判，這樣善於甩鍋的政府，真的可以培養出強大自主管理的人民的~</p><figure class="image"><img src="https://assets.matters.news/embed/e1137fd8-15bf-4f86-aeb9-31c74666b061.jpeg" data-asset-id="e1137fd8-15bf-4f86-aeb9-31c74666b061" referrerpolicy="no-referrer"><figcaption><span>Source: https://wiki-android.info/appareils/android-box/android-box/</span></figcaption></figure><p><br></p>
