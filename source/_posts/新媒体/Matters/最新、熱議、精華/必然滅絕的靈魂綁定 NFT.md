<p>大家晚安，今晚來談談一種絕對會在自由市場裡滅絕的 NFT，靈魂綁定 NFT。</p><p>人生在世難免需要尋求難忘的回憶，如今新興的加密藝術市場，正在商品化這場難忘。</p><p>不論你收藏到令人屏息的圖片、找到短暫的歸宿、還是損失不小的錢財，這場冒險都深深刻在你的靈魂裡。但，真的是這樣嗎？</p><p>這篇文我會談談 NFT 產生的機制，與背後的社群。</p><hr><p>讓我們來重新看看 NFT 個項目方常見的用詞。</p><p><strong>「每一枚 NFT 都是獨一無二的。」</strong></p><p>項目方沒有說，但大家都明白的第二句話是：</p><p><strong>「且可以交易。」</strong></p><p>如果永誌難忘的 NFT 可以交易，你願意收到多少違約金，讓 NFT 與你的靈魂解約？十萬元不行，那幾千萬總可以吧？</p><hr><p>有沒有與靈魂綁定的 NFT 呢？</p><p><strong>「每一枚 NFT 都是獨一無二的，但，不可交易。」</strong></p><p>先看我隨附的圖片。</p><figure class="image"><img src="https://assets.matters.news/embed/7318aecd-9856-4bee-adf2-c2626ea13f1b.png" data-asset-id="7318aecd-9856-4bee-adf2-c2626ea13f1b" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>左邊是廣義常見的生成藝術，如 Artblocks 或各種有所謂稀缺性的 PFP 項目方。如同我昨天藉由採訪川貝母描述 PFP 的銷售過程，創作者會先設定好稀缺性（Rarity），每一枚 NFT 或有不同；這批 NFT 被綁定特定的圖像，且可交易。在此大前提下，交易熱絡的軟性社群應運而生。</p><p>蓋棺論定，這一波 NFT 熱潮可以說是建構在<strong>「圖像可交易」「盲盒遊戲」「自由度極高的社群平台」</strong>上，漸漸又發展出各式各樣的賦能、鎖倉、揪團。</p><blockquote><strong>這一波 NFT 熱潮的基礎設定：<br class="smart">1. 圖像可交易<br class="smart">2. 盲盒遊戲<br class="smart">3. 自由度極高的社群平台</strong></blockquote><p>因為「圖像可交易」，隨機生成的 PFP 與生成藝術市場如 Artblocks、fxhash 得以爆紅長青。「盲盒遊戲」出現，你才能在一級市場購買彩券，在二級市場奪取真愛，往復之間，套利者誕生，接著演化出科學家（編寫程式自動購買的消費者）。</p><p>因爲稀有度，每一個 NFT 出生便不平等，持有者夢想以此一夜致富，這是真正便宜的行銷費用。每一位致富者都是最成功的營銷案例，每後都有成千上百的追隨者，「自由度極高的社群平台」是捕夢網，乘載這些追夢人。</p><p>你如果發現自己不是消費者，那就是產品。</p><hr><p>那有沒有自外於市場機制的盲盒 NFT？有的，此時可以看向圖片的右方，</p><figure class="image"><img src="https://assets.matters.news/embed/526ecda8-2505-4ad2-85c4-0638e015ef7b.png" data-asset-id="526ecda8-2505-4ad2-85c4-0638e015ef7b" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>右邊是另一種生成藝術。其稀缺性，是焊死在地址上面的，如果你購買到超級稀有的 NFT，那也沒辦法賣掉，這是只屬於你的「私體驗」NFT，也正因此我認為這種類型的生成藝術會直接滅絕。</p><p>姑且稱之為靈魂綁定 NFT 好了（Address-restricted NFT）。</p><p>在 fxhash 平台爆紅的台灣藝術家 <a href="https://www.facebook.com/jinyao.lin?__cft__[0]=AZVf5oGsf5SqlVA3lTR5TZxvwZykUhVZ9R6XqR9Lb_w6tWu0IzNCDm2i0p0-sZTLpbMHQ_EcYOEGOrQSXAxxLBxAQ4QKFnRvIafeKQUs_azKa9p41aE0R6LLvL-nW8UyjrM&__tn__=-]K-R" referrerpolicy="no-referrer"><figcaption><span>我的超稀有邪惡豆豆</span></figcaption></figure><p><br></p><p>而 <a rel="noopener noreferrer" target="_blank">黃新</a> 這次替當代藝術館展覽「蓋亜」製作的 NFT 也是一樣的概念。每一位購買者，都會獲得獨一無二的生成藝術，但圖像不可交易，交易後圖像就會消失，這也是「私體驗」。（除此之外，該 NFT 可以當門票用）</p><figure class="image"><img Gaia__Generative</a></p><hr><p><strong>引用忒修斯之船。</strong></p><p>當 NFT 的視覺樣式因轉手而產生變化，那還是同樣的 NFT 嗎？</p><p>理論上是的，但是在 NFT 被大眾理解成綁定圖樣的 PFP 時，這已不是同一艘船。</p><p>NFT 是一個迷人的新媒體，市場主導了主流 NFT 的玩法與印象，但市場之下，NFT 是一個巨大無比的沙盒，那裡每天都有新事物誕生。</p><hr><p><strong>靈魂綁定的 NFT 可以產生社群嗎？</strong></p><p>這個問題可以用黃新的當代館 NFT 稍稍得到一些解答。</p><p>因為靈魂綁定 NFT 是只有持有者個人才可以看到獨特變化的，所以持有者必須自己截圖分享出來，其他人才能看到獨特的變化。</p><p>我們提出了一個「出征」計畫，用每個人獨一無二的視覺體驗，截圖，佔領當代館粉絲專頁留言區。黃新還加碼祭出豐厚的獎賞 — Morpho 藝術家與黃新共創的 NFT 作品，單價不菲，只有獲得最多讚的朋友就能獲得。</p><p>參與的群友紛紛烙人來按讚，成為當代館粉專的一種奇觀。</p><p>當然，這個事例，有趣大於獲利。這也是為何我說靈魂綁定的 NFT 會在市場上絕種，因為稀有的體驗無法販售，相信手上持有 PFP 的朋友看到此段，會開始手心冒汗急跳腳。</p><hr><p>我的一個朋友，將離世父親的手稿刺在身上，那是一段無法辨識的中文。</p><p>而且他將此同時鑄造為 NFT。</p><p>我認為這才是真正的靈魂綁定。</p><p>你獲得的體驗，無法共享，更無法販賣。</p><p>你還願意持有嗎？</p>
