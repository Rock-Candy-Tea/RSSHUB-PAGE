<p>大家好，路人甲！！！</p><p>最近我收到有位读者问我关于怎么得到Google 认证的 Associate Android Developer，我在此文章简称为 AAD。今天我从以来几个角度来分享我是如何得到我的 AAD的。</p><hr><h2>1. 什么是 AAD？</h2><p>Associate Android Developer 考试展示了入门级 Android 开发人员在开始其职业生涯时应具备的技能类型。通过这项基于绩效的考试并获得 Associate Android Developer 认证，您就可以证明自己在开发人员通常执行的任务方面有能力和技能。该考试现在也可以在 Kotlin 和 Java 中进行，但是我个人建议报考 Kotlin 版本的因为 Kotlin 已取代 Java 成为 Android 开发的首选。</p><hr><h2><br></h2><h2>2. 准备</h2><p>为了全面了解和有充足的资讯，我还是建议你首要的任务是观看官方指南。之后，Google也建议您参加 Kotlin 中的 Android 基础课程。 官方指南列出了您将接受测试的能力领域和个人能力。也有指向建议的在线资源的链接。</p><p><br></p><blockquote>请注意，这些资源仅占在线可用资源的一小部分，Google鼓励您根据自己的需要进行研究和学习。 我个人的反馈是在学习中，一定要动手尝试。</blockquote><p>以下就是官方指南列出您将接受测试的主题</p><ul><li><a href="https://developers.google.com/certification/associate-android-developer/study-guide/android-core" target="_blank">Android core</a></li><li><a href="https://developers.google.com/certification/associate-android-developer/study-guide/user-interface" target="_blank">User interface</a></li><li><a href="https://developers.google.com/certification/associate-android-developer/study-guide/data-management" target="_blank">Data management</a></li><li><a href="https://developers.google.com/certification/associate-android-developer/study-guide/debugging" target="_blank">Debugging</a></li><li><a href="https://developers.google.com/certification/associate-android-developer/study-guide/testing" target="_blank">Testing</a></li></ul><blockquote>当你完成阅读和有信心报名开始的时候，你可以尝试模拟 8 个小时内完成测试。这好让你能找到测试当天需要的感觉。</blockquote><hr><h2><br></h2><h2>3. 报名、测试和完成测试面谈</h2><p>当完成了你的准备、更新了你的 Android Studio 版本和尝试模拟 8 个小时内完成测试。 那么你就是万事俱备，只欠报名。</p><p>当你报名手续完成后，Google 会邮件测试链接至你的邮件箱。当你点击邮件里的链接，你会得到 Android Studio Plugin 和这个 Plugin 包含你的测试题目和计时器。</p><blockquote>个人建议：当你心态和生理上都准备好才开始测试，因为测试是没有中途暂停的。</blockquote><p><br></p><p>当你开始测试时，记得千万别急得一窝蜂就开始打代码。你需要做的是花一点时间了解你接下来 8 个小时内需要完成什么或你需要达成什么条件。这个测试的是没有对错之分，因为主要是测试你对代码出某程度上的质量。</p><blockquote>我个人也是在测试完之后才发现代码逻辑上有瑕疵。但是这不影响不大，所以别太担心！！！</blockquote><p><br></p><p>恭喜你，你完成了 8 个小时的测试。现在的你还有测试面谈需要完成才可算完成整个测试，但是你是不需要同一天完成的。我个人建议你现在最好休息和测试面谈可以在后一天才惊醒，因为你已经经历了 8 个小时烧脑的测试。</p><p><br></p><p>测试面谈 ： Exit Interview，就是你需要现场录制每题约 2 分钟的回答题目视频。测试面谈是有 5 个题目，所以你需要大约 10 分钟完成。</p><blockquote>小插曲：当我录制我的回答时，我在时间倒数完之前点上传了之后系统就永远处在上传中的状态。经过几次尝试之后，我发现最好不要手痒按上传和让系统在时间倒数完之后帮你上传。</blockquote><p><br></p><p>当完成之后，你会收到 Google 的邮件证实你完成了测试和需要大约 30 天到 60 天的时候才能知道你的测试结果。 请耐心等待吧！</p><hr><p><br></p><h2>4. 之后的日子</h2><p>在得到此认证之后，我发现我的 LinkedIn 得到很多猎头公司和大企业的关注。同时你也可以想象一下你每天可能会接到大约 2 - 3 通电话来做猎头公司询问你是否在寻找更好的就业机会。这无疑在求职旅程上得到显而易见的帮助，同时也增加在薪水上的谈判筹码。</p><p>终结来说，Associate Android Developer 测试无疑是一个有质量的测试和大大提升自己在职场上的知名度。</p>
