<p>烏俄戰爭進入第 2 週，越來越多國際企業響應歐美政府的對俄制裁措施。</p><p>金融支付網路 <a href="https://usa.visa.com/about-visa/newsroom/press-releases.releaseId.18871.html" rel="noopener noreferrer" target="_blank">Visa</a>、<a href="https://www.mastercard.com/news/press/2022/march/mastercard-statement-on-suspension-of-russian-operations/" rel="noopener noreferrer" target="_blank">Mastercard</a> 與<a href="https://about.americanexpress.com/all-news/news-details/2022/American-Express-Suspends-Operations-in-Russia-and-Belarus/default.aspx" rel="noopener noreferrer" target="_blank">美國運通</a>相繼宣布暫停在俄國的全部業務，這等於是將俄羅斯打回紙鈔交易的年代。不僅俄國民眾的金融卡立即「殘廢」，國外的金融卡也無法在俄國境內使用。</p><p>金融機構響應制裁並不令人意外，但主打開放的 Web3 交易平台也跟著封鎖用戶就引發爭議。</p><p>上週，多名伊朗用戶<a href="https://twitter.com/Bornosor/status/1499371157790863369" rel="noopener noreferrer" target="_blank">發現</a>自己的帳戶被 OpenSea 封鎖，不但無法在平台上買賣 NFT，連個人頁面的 NFT 收藏也在一夕之間變成「404」。用戶不滿，OpenSea 理應做到的「去中心化」在哪裡？</p><p>這篇文章討論 OpenSea 為何不算是去中心化應用，以及 OpenSea 追求開放精神與平台治理的兩難。</p><h2><strong>封鎖伊朗</strong></h2><p>OpenSea 是全球最大的 NFT 集散地，超過九成的 NFT 交易都在此發生。它的網站操作非常 Web3。如果說 OURSONG 和 Nifty Gateway 是封閉式（Web2）的 NFT 賣場，OpenSea 打造的則是開放式（Web3）賣場。</p><p>要使用 OURSONG 平台，人們得先用 email 註冊帳號，通過實名認證才能交易。如果要把 NFT 上架到其他賣場，也要先將 NFT 從原本的封閉式平台提領出來。</p><p>但在 OpenSea，人們只要連接錢包就能開始買賣。每筆交易都有鏈上紀錄，取得的 NFT 也會立即出現在個人的 MetaMask 錢包裡。最重要的是，OpenSea 從頭到尾都不替用戶保管 NFT，只是提供資訊媒合交易。</p><p>因此，用戶可以自由將錢包內的 NFT 上架到其他賣場（例如 LooksRare）同步販售。如果用實體物品來比喻，OURSONG 類似商家寄賣，而 OpenSea 則像是個人網拍。</p><figure class="image"><img src="https://assets.matters.news/embed/cf2f4a61-5d85-4a73-bade-868b8a196fa7.png" data-asset-id="cf2f4a61-5d85-4a73-bade-868b8a196fa7" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>這些全新的 Web3 體驗，讓許多人認定 OpenSea 就是去中心化的 NFT 賣場 —— 直到人們的帳戶被無預警封鎖。根據 Decrypt <a href="https://decrypt.co/94316/opensea-iran-users-claim-blocked-ethereum-nft-marketplace" rel="noopener noreferrer" target="_blank">報導</a>：</p><blockquote>超過五位伊朗用戶回報自己的 OpenSea 帳戶被封鎖。他們聲稱自己的攝影作品已經在平台上消失、個人頁面變成「404」，以及帳戶遭到無預警刪除。OpenSea 的發言人在事後確認，他們正在依據平台使用條款，封鎖受制裁地區的用戶。</blockquote><p>下方就是其中一位被封鎖的伊朗用戶截圖。原本應該要顯示豐富收藏的個人頁面，一夕之間卻變成冷冰冰的「404」畫面。</p><figure class="image"><img src="https://assets.matters.news/embed/4ff1d5d5-e27c-4e36-b5c7-7d79843199b5.png" data-asset-id="4ff1d5d5-e27c-4e36-b5c7-7d79843199b5" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>這就像是蝦皮、淘寶封鎖可疑賣家一樣。但難道 OpenSea 不是開放、去中心化的 Web3 應用嗎？</p><h2><strong>開放與治理的矛盾</strong></h2><p>網路上對於 OpenSea 是否「去中心化」眾說紛紜。</p><p>舉例來說，CoinMarketCap 就<a href="https://coinmarketcap.com/alexandria/glossary/opensea" rel="noopener noreferrer" target="_blank">稱</a> OpenSea 是去中心化的交易市場，但 OpenSea 的官網卻對去中心化隻字未提。他們的創業初衷，是<a href="https://www.producthunt.com/posts/opensea" rel="noopener noreferrer" target="_blank">打造</a>加密商品（cryptogoods）的 eBay。</p><p>若要釐清 OpenSea 的市場定位，可以從<a href="https://opensea.io/tos" rel="noopener noreferrer" target="_blank">服務條款</a>找到答案：</p><blockquote>OpenSea 不是錢包提供商、交易所、經紀人、金融機構或債權人。OpenSea 提供點對點的 Web3 服務，幫助用戶發現並與區塊鏈上的 NFT 直接互動。我們對用戶互動的 NFT、區塊鏈沒有管理或控制權，我們也不執行 NFT 的購買、轉讓或銷售。若要使用我們的服務，你必須使用第三方錢包直接在區塊鏈上交易。<br class="smart"><br class="smart">與 Web3 的多數服務相同，你的區塊鏈地址代表你在 OpenSea 上的身份，帳戶會顯示該地址持有的 NFT。OpenSea 可能會應政府要求請用戶提供資訊或文件，並調查可能違反服務條款的行為。在這種情況下，OpenSea 將自行決定是否禁用帳戶。</blockquote><p>不難發現，OpenSea 一方面說自己對於使用者的鏈上資產（NFT）沒有控制權，另一方面又說使用者在 OpenSea 上的帳戶歸他們管。看似矛盾，但如果用實體商品來舉例就很容易理解。</p><p>OpenSea 就像是 NFT 的蝦皮賣場。蝦皮如果發現可疑賣家，可以將帳號停權，但無法沒收或凍結賣家所持有的實體物品。賣家也只要重新註冊帳戶或是轉移到其他賣場，就能「東山再起」。</p><p>OpenSea 在 NFT 市場的角色與蝦皮非常相似。兩者背後都是由企業負責經營，只是賣場內的商品從物理變成數位而已。因此，OpenSea 不僅得遵守美國的法令，封鎖伊朗用戶，也同樣要肩負判別「仿冒品」的責任。</p><p>偏偏有許多創作者喜歡透過自由、開放的 NFT 來諷刺時事，而他們的藝術品與仿冒品也可能只有一線之隔。OpenSea 該如何拿捏開放、治理的尺度，就成了他們最棘手的難題。根據 The Verge 在 2021 年底的<a href="https://www.theverge.com/2021/12/30/22860010/bored-ape-yacht-club-payc-phayc-copycat-nft" rel="noopener noreferrer" target="_blank">報導</a>：</p><blockquote>有兩組 NFT 作品正在試探 OpenSea 對於剽竊與藝術的界線。無聊猿（BAYC）是當今最昂貴的 NFT 作品，但任何人都可以基於公開的猴子圖像加以修改。近期有兩款 NFT 作品 PAYC 和 PHAYC 想要藉由鏡像翻轉無聊猿的猴子圖像，來諷刺當時的 NFT 狂熱。<br class="smart"><br class="smart">他們還在網站以戲謔的方式表示，持有這些 NFT 不具任何會員資格也不會有任何持有獎勵。這與 BAYC 開發團隊的承諾完全相反。但最後 OpenSea 以違反智慧財產權為由，下架了這兩款作品。</blockquote><p>幣圈會將快速轉賣 NFT 獲利的價格炒作者，稱為 flipper。而 PAYC 和 PHAYC 這兩款衍生作品則是藉由翻轉（flipping）正版的無聊猿<a href="https://blocktrend.substack.com/p/430?s=w#footnote-4" rel="noopener noreferrer" target="_blank">4</a>頭像，將原本面向右邊的猴子變成面向左邊來諷刺時事。這算是仿冒抄襲，還是藝術創作？</p><p>OpenSea 認為是前者，「藝術家」則認為是後者。最後，當然是 OpenSea 說了算。但這些「藝術家」並沒有善罷甘休，同時還將作品上架到 Rarible 和 Mintable 這兩個 NFT 賣場。最終，Rarible 決定讓這些「創作」留在平台上。</p><figure class="image"><img src="https://assets.matters.news/embed/bdf2618b-2eab-4f40-b8cc-571710803264.png" data-asset-id="bdf2618b-2eab-4f40-b8cc-571710803264" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>但 OpenSea 是 NFT 交易市場的代名詞，如果作品無法上架，就等於跟廣大買家擦身而過。</p><p>這可說是目前 NFT 市場的最大矛盾 —— 中心化的 OpenSea 主宰九成以上的 NFT 交易 —— 追求自由開放的 NFT 創作者，必須先經過 OpenSea 的審查才能讓作品進到主流市場。而審查尺度，只有 OpenSea 自己知道。</p><p>每當 OpenSea 調整平台治理規範，就可能影響 NFT 的發展方向。這與臉書調整演算法，就會影響內容觸及類似。</p><h2><strong>去中心化</strong></h2><p>OpenSea 並不算去中心化，而只是開放式的中心化賣場。</p><p>它的創新之處在於，人們<strong>不必</strong>先將數位資產存入封閉式賣場才能交易。這件事情在物理世界稀鬆平常，但在數位世界卻是一大突破。畢竟銀行怎麼可能在人們還沒把錢存進帳戶之前，就先讓你轉帳？</p><p>但開放並不代表去中心化。OpenSea 由美國公司 <a href="https://opensea.io/tos" rel="noopener noreferrer" target="_blank">Ozone Network</a> 營運，就像加密貨幣交易所一樣，必須遵守美國的制裁措施。只不過，OpenSea 的治理還比加密貨幣交易所更棘手。</p><p>交易所可以根據用戶的實名認證資料限制服務。但 OpenSea 只有用戶的錢包地址，光憑登入的 IP 位址難免「錯殺」用戶。此外，中心化交易所可以由上而下決定要開放哪些幣種交易，但 OpenSea 卻得天天面對用戶上傳「藝術品」測試平台底線。</p><p>這讓 OpenSea 陷入有如臉書、推特等社交媒體的內容管理困境。社交媒體上的資訊真偽還有查證空間，創作者上傳的 NFT 是不是「藝術」該由誰說了算？長期下來，OpenSea 必然兩面不是人。</p><p>多數人可能只看到 OpenSea 高達九成市佔的風光，卻沒意識到背後隱藏的開放與治理矛盾。說到底，恐怕沒有任何 NFT 賣場是完全去中心化的。人們需要的可能只是去中心化資產，搭配更多能與 OpenSea 匹敵的開放式賣場，未必是完全去中心化的 NFT 賣場。</p><hr><p><a href="https://blocktrend.substack.com/" rel="noopener noreferrer"
