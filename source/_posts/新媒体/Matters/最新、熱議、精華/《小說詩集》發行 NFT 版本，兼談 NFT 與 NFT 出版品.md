referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>《小說詩集》是我的第一本詩集，一共收錄48首詩作，2012年由煮鳥文明出版社出版，曾獲選為「臺北詩歌節推薦書」與「國立臺灣文學館文學好書」。實體書目前已絕版，這次總共發行20份NFT版本（每首詩20枚），未來不會再發行。</p><p>NFT 版《小說詩集》的 Opensea 頁面：<a href="https://pse.is/3s985y" rel="noopener noreferrer" target="_blank">https://pse.is/3s985y</a></p><p><br></p><p>以下稍微說明一下何謂NFT，也談一下 NFT 出版品（若是有誤歡迎留言指正）：</p><p>【什麼是 NFT ？】</p><p>NFT 為加密貨幣的一種，不同於比特幣（BTC）、以太幣（ETH）等等「同質化代幣」，NFT 為「非同質化代幣」。同質化代幣與法定貨幣一樣可以相互替代（我可以跟你交換身上的百元鈔，並不會有任何改變），也可以分割（我可以把百元鈔換成兩枚五十元硬幣）；非同質化代幣則不能相互替代與分割，也就是說每枚 NFT 代幣都是獨一無二存在，且交易時只能以一枚為最小單位。</p><p><br></p><p>【什麼是 NFT 藝術品？】</p><p>由於 NFT 具有唯一性與不可分割性，很適合當作數位資產的身分證。買家可以透過購買數位藝術品的 NFT 來證明自己「擁有」該作品，而非僅持有作品的檔案。近年來 NFT 藝術品日趨火紅，2021年3月美國數位藝術家 Beeple 的 NFT 作品《Everydays: The First 5,000 Days》以約新台幣19.5億元成交，締造在世藝術家成交作品的第三高價紀錄。</p><p>NFT 藝術品的缺點是如果元數據（對應的作品圖檔、音檔或是文字）因故消失，NFT 將失去價值。對此 NFT 市場 Opensea 提供元數據凍結功能，凍結後的元數據會被存儲至 IPFS 分散式網路，不會因為伺服器停止運行而消失，買家可以確保自己買到的是整個數位藝術品，而非沒有外皮的 NFT 代幣。</p><p><br></p><p>【擁有 NFT 藝術品我可以擁有什麼？】</p><p>可以想成是購買一件虛擬藝術品或是遊戲內的虛寶，購買後你視同擁有該作品，但是你不會得到作品的著作權，不得作為商業用途使用。</p><p>有人可能會質疑數位藝術品容易被複製，為何需要花錢購買，然而現實中的藝術品何嘗不是如此？一場攝影展如果沒有禁止拍照，觀展者都可以拍下作品自行列印，我們卻依然願意花錢購買攝影作品。</p><p>NFT 看似很新，其實也沒有那麼新，可以想成是把早已存在的藝術買賣虛擬化，這裡套一句哲學家與科幻電影喜歡提醒我們的：你又如何能確定眼前的現實並非虛擬？</p><p><br></p><p>【如何購買 NFT ？】</p><p>多數的 NFT 市場都是藉由以太幣（ETH）來交易，買賣作品之前需要綁定加密貨幣錢包（我自己是用 Metamask），網路上有不少教學文可以參考。另外要注意，在 NFT 市場購買作品需要支付礦工費（Gas Fee），而非僅支付作品價格，Gas Fee 會根據區塊鏈的忙碌程度而浮動。</p><p><br></p><p>【 NFT 出版品與電子書有什麼不同？】</p><p>最後來談談 NFT 出版品。</p><p>用圖檔來舉例，就算我們擁有某件數位藝術或是攝影作品的 png 檔，也不會聲稱自己「擁有」該作品。同理，當我們擁有某本電子書的 epub 或是 pdf 檔，也不等同「擁有」該本書，因為每位購買該本電子書的用戶都會拿到一模一樣的檔案，而且無法交易。相較之下我們可以透過 NFT 來轉賣收藏的出版品，且因為加密貨幣的去中心化及不可篡改性，不用擔心出版品隨著電子書平台收攤而消失。甚至未來如果元宇宙（metaverse）能夠實現，還能將 NFT 書籍帶進其中閱讀與交流。</p><p>不過 NFT 市場上面的出版品仍然不多（目前還找不到其他的台灣 NFT 詩集），尚未有標準格式，閱讀環境也會因為交易平台而有所差異。尤其幾個主要的 NFT 市場皆不適合上架文字作品，因此有些創作者選擇在鎖定內容中附上作品 pdf 檔的下載連結（或是另外寄送），然而 pdf 檔無法與元數據一起被凍結，仍然有消失的風險。</p><p>我這次是將《小說詩集》的每首詩拆開發行，NFT 的持有者能夠解鎖並閱讀相對應的內容數據，換個角度想就是在虛擬世界中買了詩集的其中幾頁。不過這個做法只適合詩集或是短文，如果是其他類別的文字作品譬如長篇小說，要在 NFT 市場中販售仍有一定的阻礙。</p><p>如果是想要「閱讀」一本出版品，傳統電子書平台依然是首選；如果是想要「收藏」一本書籍，則 NFT 會更加合適。NFT 讓數位內容的銷售更加多元，創作者更容易直接獲利（每次轉賣都能獲得版稅）。未來如果 NFT 交易能夠普及，對於創作者來說絕對是一大福音。</p><p><br></p>
