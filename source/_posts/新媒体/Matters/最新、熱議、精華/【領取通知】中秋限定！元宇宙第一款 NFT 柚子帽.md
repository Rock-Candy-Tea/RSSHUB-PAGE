<p>這週末就是一年一度的中秋節了，你想好要去哪裡賞月、烤肉了嗎？去年中秋節區塊勢送大家 NFT 月餅禮盒，今年我則是準備了一頂可以在 Decentraland 元宇宙戴的 NFT 柚子帽要送給大家，感謝過去這一年來的支持。只要完成登記，在中秋節（9/10）之前就能陸續收到柚子帽。</p><p><a href="https://blocktrend.qubic.market/" rel="noopener noreferrer" target="_blank">登記領取</a></p><p>即便你今年恰好沒能回家烤肉團圓，也能在元宇宙裡帶上柚子帽和親朋好友一起過中秋。這篇文章說明領取流程、如何使用以及這頂 NFT 柚子帽有哪些特色。</p><h2><strong>領取流程</strong></h2><p>區塊勢老讀者或許還記得，過去幾年無論我是寄送 NFT 明信片或是發放 NFT 月餅禮盒，大家都得先自行安裝錢包，將地址填入 Google 表單，再等待我逐一寄送到每個人的錢包地址。</p><p>整個流程就像是在寄送紙本明信片或物理月餅禮盒。雖然收件者或許會有一種不知道什麼時候會收到的期待感，但這就像是缺乏物流服務的原始年代，每年我都要花好幾個小時在區塊鏈上親自送貨。</p><p>多虧過去這一年 NFT 蓬勃發展，也讓鏈上的物流服務變得更加完善。今年區塊勢與 MaiCoin 集團旗下的 Qubic 合作，稍微簡化了 NFT 的登記流程。我也會趕在中秋節之前，透過空投工具即時把柚子帽 NFT 送到大家錢包裡。</p><figure class="image"><img src="https://assets.matters.news/embed/06473e2c-172f-455a-8c61-66233c6b52df.png" data-asset-id="06473e2c-172f-455a-8c61-66233c6b52df" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>如果你原本就已經有 MetaMask 小狐狸錢包，只要以錢包 app（或電腦瀏覽器）打開這個網址（<a href="https://blocktrend.qubic.market/" rel="noopener noreferrer" target="_blank">https://blocktrend.qubic.market/</a>）就可以完成登記。不必再像以前得手動複製、貼上錢包地址。</p><p>如果你從來沒有創建過任何加密貨幣錢包，那更簡單。只要點擊開頭的領取按鈕，就可以在登記的過程中以社交帳號（臉書、Google 或 Apple）快速創建一個由個人管理的 Qubic 錢包。</p><p>但假如你領到這頂 NFT 柚子帽，卻沒有真實地到 Decentraland 元宇宙戴上它，那倒不如點擊右鍵下載 JPEG 收藏就好。</p><h2><strong>進入元宇宙</strong></h2><p>許多人都曾經在新聞上看過元宇宙，但真正踏入過元宇宙的人恐怕不多。其實要進入元宇宙，不必事先花錢買 VR 頭盔。只要打開大家都很熟悉的網頁瀏覽器，例如 Chrome、Brave 或 Opera Crypto Browser 就能進入 Decentraland 元宇宙。登入方法也非常單純。</p><p>如下圖所示，只要以電腦瀏覽器（不支援手機）打開 <a href="https://play.decentraland.org/" rel="noopener noreferrer" target="_blank">Decentraland 網站</a>並連接加密貨幣錢包就能成功登入。稍後領到的柚子帽 NFT 就會出現在背包（Backpacks）的收藏品頁籤裡。只要點擊一下，就能替你的元宇宙數位分身換上最應景的柚子帽。整個過程可能不到 1 分鐘，而且完全免費。</p><figure class="image"><img src="https://assets.matters.news/embed/d950af13-ea67-43ee-84e9-e07ff53deca3.png" data-asset-id="d950af13-ea67-43ee-84e9-e07ff53deca3" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>這頂柚子帽 NFT 很可能會是大家在元宇宙中的第一個配件。</p><p>以往人們總是批評 NFT 只是另類的 JPEG，傻子才會花錢買。但現在人們只要進到元宇宙，就會發現 JPEG 和 NFT 根本天差地別。前者是可以無限下載的資訊（information），後者則是難以複製的資產（asset）。</p><p>以 Decentraland 為例，人們身上穿的衣服、鞋子或帽子都是 NFT。現在有許多運動品牌例如 Nike、Adidas 或奢侈品牌 LV、Gucci 紛紛宣布進軍元宇宙並發行品牌配件 NFT 供人們選購、穿搭，將影響力擴展到數位世界。</p><p>反觀如果人們只是從網路上下載球鞋 JPEG 而沒持有 NFT，就無法將球鞋穿在腳上。這就是兩者的顯著差異。因此，我才會鼓勵大家領到這頂柚子帽 NFT 之後，應該實際到元宇宙裡戴上它。別把 NFT 用成了 JPEG。</p><h2>元宇宙配件</h2><p>這頂柚子帽 NFT 的最大特色，就是可以在既有的元宇宙裡面使用，但在物理世界卻完全沒用。這與目前網路上絕大多數主打物理世界「賦能」，卻與既有元宇宙不相容的 NFT 恰好相反。</p><p>我看好 NFT 日後的最佳使用場景，不是刻意讓持有者兌換物理商品或綁定特殊禮遇，而是可以在元宇宙這個平行世界裡成為人們的日常生活物品。就像人們在物理世界裡，拿到「物理柚子帽」的唯一用途就是拍照應景，不會有其它的賦能。</p><p>同樣道理，人們在數位世界裡拿到 NFT 柚子帽也是同樣「沒用」。不同的是這頂柚子帽不會隨著時間變質，明年中秋節再次戴上同一頂柚子帽應景也不會尷尬。</p><p>雖然這頂元宇宙柚子帽外表不算吸睛，甚至就像是每個人家裡的那顆柚子，但它也可能是最禁得起時間考驗的 NFT。</p><hr><p><a href="https://blocktrend.substack.com/" rel="noopener noreferrer"
