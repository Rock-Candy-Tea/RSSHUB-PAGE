<strong>針對初期的社群分析</strong></h2><p>漫長的介紹後，回到社群分析的話題上，也許你會發現這個項目的社群走向很有趣：熱絡的社群就是生，死氣沉沉的社群就是死？說實話，真的要看看你渴求的是虛偽的表面，還是有實力的內涵，就像選伴侶一樣。</p><h3><strong>A. OG 的特別安排</strong></h3><pre class="ql-syntax" spellcheck="false">OG：Original gangster，原意是「最初的黑幫」，而在 NFT 圈內指很早就進入這個圈子並獲得尊敬的元老級人物，但 Teahouse 有另一個安排 。
</pre><figure class="image"><img src="https://assets.matters.news/embed/bfbefbb2-6ae7-4f22-b3c6-8d8707f3257c.gif" data-asset-id="bfbefbb2-6ae7-4f22-b3c6-8d8707f3257c" referrerpolicy="no-referrer"><figcaption><span>Teahouse OG Pass 盲盒</span></figcaption></figure><p>左圖就是價值 5.5 ETH 的 <a href="https://discord.gg/BcMbGXdGYS" rel="noopener noreferrer" target="_blank">Teahouse</a> OG 證明，即 <a href="https://opensea.io/collection/teahouse-hightable-og" rel="noopener noreferrer" target="_blank">OG Pass</a>。一般來說，OG指的是項目早期支持者，算是元老級成員，一般項目會給予這些 OG 成員一些少量的優惠，例如購買時打折、免費鑄造權限、額外空投等。但 <strong>Teahouse 的 OG 不一樣，他們設定為是專門給企業或團體客戶購買的，因此價格與發行數量亦不一樣。</strong></p><p>或許你會好奇誰有購買這個 OG Pass，本熊可以肯定的是Zombie Club 的余文樂有購入。</p><p>讀到這裡，你也許會以為這是只要有錢就能夠入手的東西，但事實並非如此。因為<strong>獲得 OG 購買資格的人，都必須與 </strong><a href="https://discord.gg/BcMbGXdGYS" rel="noopener noreferrer" target="_blank"><strong>Teahouse</strong></a><strong> 團隊詳談，就像談生意，肯定雙方都明白自己在做什麼，確定彼此能夠合作後，才可以購買。</strong>這個方式亦讓普通的持有者消除了「有錢就能獲得？」的疑慮，而哪些人成為 Teahouse OG 的消息，亦變相成為 Teahouse 的「廣告」之一。</p><p>那麼，未來這些付了 5.5 ETH 的人會如何影響 Teahouse 呢？就像不少人會注視鯨魚（擁有大量資金的人）的錢包，這些人的舉動也會被持有者關注。若然這批人突然有大動作，例如 OG 的持有者急劇增加，或相反地 OG 持有者拋售他們的 OG 身份，都會<strong>跟普通成員對社群的信心直接掛勾</strong>。</p><h3><strong>B. 優秀出眾的美術</strong></h3><pre class="ql-syntax" spellcheck="false">Teahouse 比很多項目更先開始把「通行證」與「PFP」結合起來，結果引來了很多不適合項目發展的 NFT 玩家。
</pre><p>「美術足以吸引目光」、「項目有名氣」、「作品與項目定位有特色」、「能保持話題」，基本上只要符合以上條件的任何三項，項目就會成為 Flipper （短炒者）的目標。有誰不希望自己能在一個晚上，只付出少少金額就能讓自己身家暴富起來？加上傳統媒體針對加密貨幣與 NFT 的報導都傾向於「暴富」、「快錢」為主調，令這些投機者湧至 Teahouse。</p><figure class="image"><img src="https://assets.matters.news/embed/37335f3b-3b8d-4965-aeb0-3ace15386c5e.png" data-asset-id="37335f3b-3b8d-4965-aeb0-3ace15386c5e" referrerpolicy="no-referrer"><figcaption><span>Teahouse 的電話桌布</span></figcaption></figure><p>不過， Flipper 入侵一個項目不一定是壞事，再者也沒有任何一個項目可以躲過他們的追擊：這幫人是一把雙面刃，既能讓一個項目的價值推上最高峰，亦能把一個項目踩在谷底下。回望這波熊市的 free mint 潮流，有多少個在期間成名的項目，不是因為初期有這幫人 FOMO，而讓免費項目也能價值 1 ETH？</p><p>本熊必須重申他們的存在不是壞事，不過更多時候，投資這回事就是需要一點兒的耐性。而作為項目方，也理應要先建構出自己理想的社群樣貌，然後就住自己的目標進行一連串的工作、企劃等，以確保自己社群的定位符合自己的預期，例如：</p><p>甲先生想要讓 A 項目更傾向於學術、技術向，那甲先生應該會制定一些跟某些專業有關的活動；乙小姐想要讓 B 項目傾向創作類型，那她應該會制定創作類活動，例如繪畫創作比賽、故事創作比賽等。</p><p>這些原因，都讓不少以發行「通行證」為目的的項目，都會傾向把自己的 NFT 變成單一化的卡片類，而非一般 PFP 項目的 NFT，讓大眾即使只看過 NFT 的模樣，也或多或少理解到項目的類型是什麼。</p><p>不過，<strong>傾向於把「通行證」與「PFP」結合起來的 Teahouse 讓自己的 Pass 變成好看的藝術系列，這件事令很多仍未接觸過類似做法的買家誤會，結果引來一群不適合自己的客戶。</strong>若然這群人後來因為項目的價格便宜而成功入手，本熊深信將會為整個項目引來負面效果，甚至是大量的 FUD。</p><p><strong>擁有優秀的美術不是壞事，只是時機不合時。</strong></p><h3><strong>C. 白名單派發制度及白單限定發售</strong></h3><pre class="ql-syntax" spellcheck="false">有基本了解即可獲得，門檻很低。
</pre><p>回想那段初售前的日子，獲得 Teahouse 白單的方法比想像中簡單。雖然本熊是透過參加他們的白單挑戰，分享自己遇到的 rug 經驗而獲得的，但其他人要獲得白單，都可以透過參加他們 Discord 中的賭場遊戲（類似簽到和玩小遊戲）、外部及內部抽獎、內部活動及聆聽 AMA 並參與答問活動而獲得。</p><p>由於他們跟很多項目都有合作，而且當時他們還到處辦 AMA，所以要獲得他們的白單幾乎不是難事。這種派發方法，令項目方無法盡可能地確保到自己的 NFT 能被潛在的長期持有客戶購得，即使未來採取白名單限定發售，在不看價格的層面上看起來，其保護項目的效果似乎也不顯著。</p><p>不過，事實上要針對潛在客戶派發白單是一件頗困難的事，除非他們團隊背後，擁有像 Cambridge Analytica 那樣子的追蹤及分析技術（建議閱讀：<a href="https://vocus.cc/article/6249d37ffd897800019fa8d9" rel="noopener noreferrer" target="_blank">【區塊鏈與人性】談Web 2.0的扭曲現象，與Web 3.0的改革。</a>）。</p><hr><h2> <strong>開售階段</strong></h2><h3><strong>定價高昂的保護牆</strong></h3><p>幸運的是，正如本熊在簡介中所說，Teahouse 的社群是被高售價保護著的。一般成員 VIP Pass 的定價落在 0.55 ETH 時，幾乎已經讓結果成了初型：</p><blockquote>此部分限方格子<a href="https://vocus.cc/KumaInvestmentNotes/introduce" rel="noopener noreferrer" target="_blank">專題</a>付費訂閱者閱讀。</blockquote><h3><strong>投資向項目的迷思：中心化回歸？</strong></h3><p>與一般 NFT 項目不同，其分潤機制並非客戶購得 NFT 後，即可隨團隊的工作進度而獲得。相反，Teahouse 先提出他們 NFT 所創造出來的收入機制，是 100% 需要持有者在未來就團隊提供的投資方案作研究，並自行決定適合自己的投資風格及投資金額。至於這筆投資金額，是需要額外支付的。</p><p>換言之，Teahouse 的角色比較像是中心化交易所，如幣安、Bybit 等，而持有者就是這是這些交易所的客戶。但是，Teahouse 又比這些機構更顯得人性化：</p><ol><li>會由團隊向持有者親自講解投資方案、提出有關風險及解答所有持有者的提問；</li><li>團隊可以按自己能力，制定出自己能夠承擔一定風險的投資方案給持有者選擇；</li><li>他們擁有可即時與團隊溝通的社群渠道，除非團隊說明自己休假，否則持有者的問題都會很快得到解答；</li><li>持有者有權就團隊作出的某些決策，以投票方式表態支持與否決等。</li></ol><p>然而，如果大型的交易所都有可能發生擠提，更甚是倒閉結業，為什麼持有者要相信 Teahouse 團隊，把自己的資金放入 Teahouse 中呢？Teahouse 的收入來源不只有 NFT，還有協助其他項目、遊戲等發行的業務，而這些收入將會進一步穩定 Teahouse 的營運。</p><p>但是，懷疑的人還是會繼續問：為什麼要相信？</p><p>於是，開售變相成為簡單的信任遊戲答案揭曉環節：信者會願意購入他們的 NFT，而不信者則會離開。至於相信他們的人是基於什麼作結論呢？本熊敢肯定，大多是來自於多次因 Teahouse 監督而得出的優秀鑄造體驗，以及其團隊在做 AMA 時給出的專業、自信氛圍。</p><p>當然，Teahouse 網站上的「BACKERS」清單亦讓本熊十分敬嘆，尤其 Web 3.0 就是一個需要信譽的圈子，而信譽比起人為宣傳，更多是來自於一個人的實績與言行。</p><figure class="image"><img src="https://assets.matters.news/embed/1e750d59-46a5-421f-9a38-6a0d5768bed1.png" data-asset-id="1e750d59-46a5-421f-9a38-6a0d5768bed1" 團隊就已經坦然：「<strong>其實 VIP Pass 不需要多買一個。</strong>」同時，他們亦<strong>不建議大家從二手市場中購買 VIP Pass，原因包括會被平台抽走掛賣佣金等</strong>。這些簡單直接的發言都讓Teahouse 的二手市場賣況不如「一般人」的理想，但符合持有者及團隊的想法。</p><figure class="image"><img src="https://assets.matters.news/embed/576d3f87-0115-4cdb-b01e-cc58367f3f3f.png" data-asset-id="576d3f87-0115-4cdb-b01e-cc58367f3f3f" referrerpolicy="no-referrer"><figcaption><span>開賣日，Teahouse VIP Pass 在二手市場上的交易量十分少，顯然都是糊裡糊塗地購入了的人，或懷著僥倖心態的人放售的。</span></figcaption></figure><p>事實上，經過了兩個月後，即使熊市到來，仍有不少 Teahouse 持有者在購買 VIP Pass 後，就沒有留意自己的 Pass 在二手市場的價格，除非社群中突然有人提起：根本沒打算賣，又怎會去留意價格？</p><hr><h2> 199），再來是開啟了首個高風險投資策略：藍籌集資策略，謹慎地逐步履行開售前的承諾。</p><p>其中，<strong>藍籌集資策略十分成功地向持有者集得 100 ETH</strong>。團隊因而已經購買了數枚藍籌 NFT 及配上一些質押模式，試著為有參與集資的持有者帶來額外收入。</p><blockquote>此部分限方格子<a SecuX 的 Help Center 頻道，以即時解答使用者的疑難及接受使用者的意見，而團隊亦積極地協助持有者解決各種疑難，當中包括那些礙於海關原因，無法經官方方式接收到冷錢包的中國用戶。這些溝通及客服安排，讓整個兌換過程頗順利，而產品亦有持地更新。</p><figure class="image"><img VIP Pass，所以持有者亦可以自行決定要兌換冷錢包，還是要把券賣掉。因此，OpenSea 上亦有不少人把兌換券掛賣。</p><blockquote>打折扣購買兌換券方法，限方格子<a href="https://vocus.cc/KumaInvestmentNotes/introduce" rel="noopener noreferrer" 風氣冒起，群內開始了每天的「學習交流會」，互相討論及分享資訊。這些交流內容基本都在公開頻道展開，所以即使是非持有者亦可以加入討論，Teahouse 的持有者十分友善。</p><p>「我自己蠻欣慰大家沒有 FUD 或是想 Flip。」就現時的社群狀態，主要負責管理整個 Teahouse Discord 的 Pat 直說：「<strong>OpeaSea 上的成交數量與少量掛價，其實代表著大家（持有者）都有很高的共識與默契</strong>，這件事出乎我意料，很感動。現在，<strong>社群像一個俱樂部</strong>，大家會知道跟 A 話題相關的要找誰，跟 B 話題相關的要找誰。<strong>群內成員之間的連結越來越好</strong>，很有溫度的感覺。」</p><blockquote>此部分限方格子<a 會協助整理課堂筆記、錄影及準備考題，給上課的大家複習，十分方便國語不好的本熊。至於有參加複習考試的大家，首五名可獲得群內賭場的獎金，這些獎金將可換取一些合作項目的白名單。這些互動及營運方式，都十分適合 Teahouse 的風格。</p><hr><h2><strong>未來挑戰</strong></h2><h3><strong>A. 第二次發售前的成績</strong></h3><p>為了準備緊接著的第二次發售，Teahouse 如果沒能夠給出一些實際的、令人驚嘆的實績，即使內部社群有多健康與友善，也難以在第二次發售中取得成功。不過，依照他們的路線圖來看，如果他們計劃的工作有準時完成及能夠對外公佈，<strong>成功與厲害的項目合作及給予持有者白單或空投，及令持有者對 $TEA 的使用方法有更深入的了解，那這些「成績問題」似乎都不是一個大問題。</strong></p><h3><strong>B. 往大型社群發展</strong></h3><figure class="image"><img src="https://assets.matters.news/embed/7bb0bbe4-1ca5-4692-8f7f-04982ee7e74e.png" 的社群是否仍能如同現在一樣健康呢？本熊不敢確定，但 Teahouse 因為這兩個月的發展，顯然更進一步地加強不少持有者的信心。</p><p>關於社群經營，Pat 說過：「一開始要經營社群的時候我就下定決心，希望創造一個有溫度的論壇，而我也還在往這樣的目標用力前進中。」</p><p>如今，這健康的社群成形，若然未來 Teahouse 開始公佈各類型的合作，當中包括 Game-Fi 項目的合作，以及一些空投消息，這幫成員將可以靠這些消息加強 Teahouse 的形象。</p><h3><strong>C. 樂於嘗試的團隊</strong></h3><p>Teahouse 與其他項目不同的地方，是他們有專業及可靠的技術背景，加上團隊的行動力很強，並承諾過會以持有者的利益優先，所以若然社群內有人提出一些有趣的提案或合作，團隊都不介意跟他們談及當中的可能性，甚至是嘗試。而更重要的是，這些新的嘗試顯然會成為持有者的驚喜。</p><h3><strong>D. 邁向國際</strong></h3><figure class="image"><img 的「穿越牛熊卡」賦能，什麼時候會公告呢？</span></figcaption></figure><p>一直以來，每兩星期一次的 AMA 除了有中文之外，還有英文，以兼顧海外持有者的需要。雖然不少人認為 Teahouse
