referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p>作为以太坊上海 Web3.0 开发者峰会的主办方，Mask Network 十分荣幸能邀请到前美国驻华大使 Terry Branstad 先生以 “<strong>全球合作实现 Web 3.0 标准化</strong>” 为题致开场辞。Branstad 大使曾在 2017 年至 2020 年担任美国驻华大使。作为爱荷华州的代表，他是美国历史上任职时间最长的州长。Branstad 大使就 Web3.0 在美国的发展和全球合作的可能性发表演讲，并回答了现场关于中美关系的问题。</p><p><strong>Branstad：</strong></p><p>感谢 ETH Shanghai 邀请我在这次活动中发言。</p><p>我曾有幸担任过 3 年的驻华大使。而在那期间，我曾多次访问上海，美国商会邀请我在 2017、18、19 年的答谢晚宴上做主旨发言。</p><p>上海的友好、好客和美丽令我乐在其中，这是一个伟大的国际城市。我想谈一谈美国和中国之间的重要关系。中美是世界上最大的两个经济体，两个国家的互联网产业，作为科技进步最活跃的领域之一，正在蓬勃发展。截至 2021 年 6 月，中国的互联网用户已经达到 10 亿以上，91% 的美国人口正在使用互联网。新兴的技术和人们互动习惯的改变，正在为互联网世界的转变铺平道路。而这就引出了我们今天的主要话题 —— Web 3.0。一个未来的互联网世界，它更加去中心化，隐私得到改善，并鼓励创新。美国已经在积极观察互联网的迭代，以及它背后的区块链技术。当局和立法者已经提出了各种框架，以鼓励和规范这些数字创新。</p><p>例如，在我的家乡爱荷华州，立法机构通过了参议院文件 2333，这使得爱荷华州成为区块链友好州。该法案在爱荷华州法典第 554 章内增加了一个新的条款，将数字资产加入爱荷华州的监管框架。促进 Web 3.0 的全球合作是至关重要的。国际社会需要合作来促进新技术，这事关人类社会的巨大飞跃，而不仅仅是个人。<strong>Web 3.0 是去中心化的，但社区是协作的。</strong>发达国家和发展中国家之间会有更多的合作。Web 3.0 是全球化的产物，需要各个国家和市场的合作，这就是为什么我们提倡合作而不是竞争。竞争常存，但在这个新兴的新技术中我们迫切需要合作。我们需要相互联系，我们需要一起工作，我们对国际合作的潜力感到非常兴奋，尽管当今的世界仍然存在摩擦。</p><p>我想在此再次祝贺大家。我们齐聚于这里，同时感谢你们给予了我机会在此发表演讲。我想祝贺 Mask Network 成功地举办了这个丰富且鼓舞人心的峰会。我也很开心能在这里回答大家的问题。</p><figure class="image"><img src="https://matters.news/@sujiyan/%E5%9B%9E%E9%A1%BE-%E5%89%8D%E7%BE%8E%E5%9B%BD%E9%A9%BB%E5%8D%8E%E5%A4%A7%E4%BD%BF-terry-branstad-%E4%B8%BA%E4%BB%A5%E5%A4%AA%E5%9D%8A%E4%B8%8A%E6%B5%B7-web-3-0-%E5%BC%80%E5%8F%91%E8%80%85%E5%B3%B0%E4%BC%9A%E8%87%B4%E5%BC%80%E5%B9%95%E8%AF%8D-bafyreie343l2zyfmpbu756loec7ym45fzg2rfotk3ehtkks2uhryq45opy" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/8c76df36-3e36-49b0-8823-277f3fece1d5.png" data-asset-id="8c76df36-3e36-49b0-8823-277f3fece1d5" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><hr><p><strong>问答环节 Q&A</strong></p><p>第一个问题来自 Suji Yan， Mask Network 的创始人兼 CEO。</p><p><strong>Suji：</strong>很高兴听到爱荷华州和中美两国在 Web 1.0、Web 2.0方面的故事。正如你所说，爱荷华州是一个区块链友好州，美国还有许多其他这样的州。我们期待看到更多的 Web 3.0 项目和协议在爱荷华州蓬勃发展。</p><p>我想知道，<strong><em>你能否分享一些关于 Web 3.0 的见解和理解，特别是从监管的角度</em></strong>。我相信这对中西方的创始人来说大有裨益。</p><p><strong>Branstad：</strong>首先，去中心化的本质消除了平台和组织对个人和网络的中心化权力，所以这些网络不受单一实体的控制。而这意味着是公司，而不是政府在控制着局面。这与 Web 2.0 形成了巨大的反差，在 Web 2.0 中，你的数据掌握在这些科技公司的手中，并有在未经你同意的情况下被货币化或被操纵的风险。Web 3.0 意味着个人夺回了对其个人数据的所有权，而科技公司将不再能够随意使用这些数据。</p><p>数据和整个网络的完整性将得到极大的改善，因为数据将被安全地存储并分布在许多设备上。它可以不受数据泄漏、攻击、篡改和严重停机的影响。</p><p>此外，Web 3.0 创造了更低的准入门槛，建立在区块链上的应用程序对所有人都是开放的，并且不受地理限制。这可能为小企业和开发者提供了一个更有前景的环境，让他们专注于真正的创新。</p><p><strong>观众提问：</strong>说起 Web 2.0 和 Web 3.0,  近日 Elon Musk 收购推特一事引起热议。我们非常想听到您更多关于收购事件的评论的想法。<strong><em>您作为一名决策者和美国公民，对此事有何感受？</em></strong></p><p><strong>Branstad：</strong>我认为其中一个大问题是言论自由。而不幸的是，我们看到了基于某些政治意识形态的限制。而 Elon Musk 说，（这些限制）将不复存在。我们希望开放和自由，以便任何想参与的人都能参与，这任重而道远。已经有人担心过多假账户的存在。他的意思是他不想为假账户买单，他想要真实的人。</p><p><strong>观众提问：</strong>一个关于法规方面的问题。距离下次美国大选还有两年时间，你认为在那之前和之后，对加密货币和 Web 3.0 公司的监管风向将如何转变？</p><p><strong>Branstad：</strong>这是个很好的问题。中期选举即将在今年 11 月举行，而且初选已经在进行中，所以有很多人都在关注这个问题。乔-拜登的支持率已经降到了历史最低点，我想，32%的人非常关注通货膨胀。汽油、食品和利率的价格正在上升。而股市最近受到了严重的打击。因此，对于现任总统来说，他在政党中的处境艰难。一般来说，反对党在中期选举中会有所收获。但今年看起来，共和党的收益甚至可能比小年选举中的正常收益更大。在监管环境方面，民主党人往往更倾向于监管。共和党人似乎对自由市场更感兴趣。</p><p>所以很明显，总是会有一些监管，但我认为我们想成为一个监管公平和相对最小的国家。而随着地方和州级的监管越多，我认为我们两国在地方和州级的关系就越好。中国有姐妹省份和姐妹城市，许多人建立了长期的友谊。</p><p>下一个问题来自 Linda Wang，她来自中国生物多样性保护与绿色发展基金会。</p><p><strong>Linda：<em>新兴的科学技术可以如何促进环境保护？</em></strong></p><p><strong>Branstad：</strong>我认为新的科技公司可以在帮助公众了解保护环境的重要性方面发挥非常关键的作用。而显然，我们在这方面还有很长的路要走。我们需要一些巧妙的改变，并认识到这一切不可能在一夜之间完成。但是说实话，我对科技公司在提高理解和认识，以及获得更多公众对保护环境的支持方面所能发挥的作用感到鼓舞。</p><p>下一个问题来自峰会主持人 Afra。</p><p><strong>Afra：</strong>您好。两个月前我在丹佛市，那里被称为是美国的第一个 “加密城市 crypto city” 或 “加密州 crypto state”。现在我正在迈阿密参加 Permissionless 大会，这两个州都给我强烈的、开放和拥抱加密技术、拥抱加密文化的氛围。<strong><em>您认为您自己的州在未来 5 年和整个美国的发展轨迹将会是怎样的？您对于人们逃离加州和其他传统的科技发达的州有什么感觉？</em></strong></p><p><strong>Branstad：</strong>我认为你说得很对。我最近看到迈阿密的市长接受了采访，迈阿密正在大力推动相关的活动。你提到（丹佛所在的）科罗拉多州，而我们爱荷华州也想成为这样。我的继任州长 Kim Reynolds 是一名非常有活力的女性。她正在做很多事情，试图使爱荷华州更具竞争力，减少税收负担。而我刚刚提到的立法。这也是为了建立一个统一的，更具科技感的商业法而通过的。所以我认为这将是一场竞赛。这也是美国各州的运作方式。各个州试图展开竞赛，看看谁能击败对方，谁能成为最友好和最成功的州。所以我认为这是你可以感到鼓舞的事情。</p><p><strong>观众提问：</strong>关于在赛博社群中公共部门和私营部门之间的合作，<strong><em>你如何看待美国各州政府和 Web 3.0 社区之间的动态？</em></strong>近些年像去中心化自治组织（DAO）的发展，<strong><em>在 Web 3.0 时代，州政府和私营部门行为者之间的动态会如何转变？</em></strong></p><p><strong>Branstad：</strong>这取决于州。但我觉得有很多州都在非常努力地保持自己的竞争力，他们看到了专注科学技术的公司的未来，并且想对这些公司保持友好和支持。因此，我认为这对未来有好处。</p>
