<p>正與太太創作一個紀念的 Web 3 社區，有一班年青人來幫助我們。這一個創作過程相當特別及有意思，好像在製作一件終極作品，將這些年的工作及社福經驗結合在一起的 Web 3 社區。</p><p>十分感謝一班滿有創意的年青人團隊，來協助我們圓滿心願。</p><p>製作 Web 3 項目的過程中，所有的物品都是以 Token 來做定價，不論價格如何便宜或昂貴，價格也是以「幣本位」。</p><p>過去我一直也有用平均成本法買入「以太幣」的習慣，但是，心裡仍會被「美元定價」破動而影響。使得心情時高時低。</p><p>這一段時間，以太幣從急升軌再急跌，身邊的朋友都感到很失望或失落。但是，這是我第一次，真的是第一次，完完全全沒有任何情緒波動。原來，在製作 Web 3 項目的過程中，心裡自自然然成了「幣本位」，例如收入的25%會放在那裡，這些以太幣會怎樣使用，另外25%又會與甚麼項目Crossover。</p><p>以往是「強逼」自己有「幣本位」思路，其實自知也有一點「阿Q精神」。但是，今次是我完完全全感受到「幣本位」的想法。</p><p>與年青的 Web 3 團隊走在一起，心態也截然不一樣，始終，團隊與我的年紀相差超過二十年，甚至三十年。我不能不佩服這一代年青人的思路及能力，相當厲害。</p>
