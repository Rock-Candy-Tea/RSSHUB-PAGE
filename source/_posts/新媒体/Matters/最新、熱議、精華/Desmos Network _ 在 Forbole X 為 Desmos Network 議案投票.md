<p>今天看到 Kin 的<a href="https://ckxpress.com/10-things-i-learn-from-airdrop/" rel="noopener noreferrer" target="_blank">週報</a>，他寫的這段很有意義：</p><blockquote>一、先不要問社群可為你做甚麼，而問你可以為社群做甚麼</blockquote><blockquote>借用了這句特別老套的話，是因為它特別適用於區塊鏈專案。一幣一世界，每個區塊鏈專案都是個百廢待興的社群，需要很多人出力，才有可能終有一天冒出頭來。</blockquote><blockquote>當然，在決定投入心力到一個社群之前，還是需要嚴選，但考慮的該是願景、理念和團隊等，而不是一開始就期望專案能為自己帶來甚麼好處。</blockquote><p>拿到了 Desmos Network Terence 的私人 Airdrop，也要了解和回應他們的社群才是區塊鏈公民應該做的。昨天 Desmos Network 剛好啟動了第二議案投票，如果你手上有 DSM 的話是時候委託和投票一下。再者，這個是與回饋跨鏈社群空投相關的議案，你不是不投票吧？</p><h2>Desmos Network 第二議案內容是甚麼？</h2><p>第二議案正式開動，將 Desmos version 自 v1.0.3 升級到 v2.3.0-mainnet 版本</p><p>更新將包含（但不止於）：</p><ol><li>升級 Cosmos dependency 到 v0.44.3</li><li>更新程式碼兼容最新的 Band Protocol mainnet</li><li>讓 DTags 在更新 Profile 時變成不是必須。</li></ol><p>內容可以在 <a href="https://explorer.desmos.network/proposals/2" rel="noopener noreferrer" target="_blank">https://explorer.desmos.network/proposals/2</a> 中看到.</p><p>你可以使用兩種方式投票：</p><ol><li>Forbole X 網頁版</li><li>使用 command line</li></ol><pre class="ql-syntax" spellcheck="false">desmos tx gov vote 2 <option> --chain-id desmos-mainnet --fees 500udsm --from <your-key>
</pre><p>此更新是發放 Airdrop 必須的。</p><p>投票期為 7 天，即 2021年11月18日 15:55:45 UTC 截止</p><p>假如議案獲得通過，2021年11月2日 13:00 UTC 將進行升級。</p><h2>如何在 Forbole X 投票</h2><p>1. 到 <a href="https://x.forbole.com/" rel="noopener
