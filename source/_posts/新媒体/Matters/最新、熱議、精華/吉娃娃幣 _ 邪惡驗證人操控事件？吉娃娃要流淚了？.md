過去 3 週左右，我一直在吉娃娃運營一個節點，昨晚發生我認為社群應該知道的事情：昨晚（美國時間）深夜，有多個節點在幾分鐘內彈到 Active Set 活躍驗證人之列中 ，每個節點的委託量剛好都超過約 200 萬 $HUAHUA，又或者剛好令他們存活於 active set 當中，就在 95-100 驗證人位置的少數節點之上。

當中涉及的驗證人如下：
- Stakethat
- shishaonthespot
- Constantinos
- Plusx
- TheCrazyStaker
- stakearmy

我非常好奇，因此到 Mintscan 起底查看多有錢包。Plusx 與 shishaonthespot 只有一個委託都來自同一個錢包。我覺得奇怪的是，一個錢包在幾分鐘內將數百萬 $HUAHUA 委託給了兩個不同的非活躍驗證人，其中一個節點僅在一個小時前架起，所以我做了更多的調查。事實證明，這兩個節點所獲得的 $HUAHUA 委託來自同一個錢包，這個錢包剛剛委託了數百萬美元出去。
</pre><p>之後 ValidateUranus 繼續研究，發現該錢包委託了 Constantinos、stakethat，而且 stakethat 的佣金是直接發到那個錢包中。至於 TheCrazyStaker 早 Plusx 、shishaonthespot 兩分鐘上線但委託也是從那個主錢包而來。雖然 Constantinos 經已不活躍，但也是有關係的。</p><p>ValidateUranus 不太確定正在發生甚麼事，但似乎就是同一個人操控了吉娃娃 100 個節點當中的多個節點。</p><h2>社群回應</h2><p><strong>Kleimanb81（管理員）</strong>：我了解驗證人的憂慮，雖然我都有懷疑，但原 PO 也不是第一個懷疑的人。如果有濫用或操縱行為，一定會作出必要行動。</p><p><strong>BitcoinPizzeria</strong>：除了他們不去中心化，這樣做有甚麼好處？</p><p><strong>ValidateUranus（原 PO）</strong>：都是為錢吧，我估計他們經已計算好，如果經已有一個運作已久的節點，再配合其他節點就可以靈活處理委託回報，將節點推高一線，令整件事更有利可圖。於是成為底層活躍節點都要有超過 10 萬美元委託。雖然我都只是猜想，不過如果真的是這樣，委託人必須要知道有這種行為發生。</p><p>原則是，只要有更多節點，這堆人能卡位就會有利可圖，雖然最低佣金為 5%，但運作多個節點而收益平衡，會比只運作一個表現突出的節點更有利益，因為有些人是不會理表現，只會隨機選擇一些驗證人來委託。</p><p><strong>Im_A_Model</strong>：驗證人不是偷又不是搶都是驗證交易，有何問題？</p><p><strong>vickangaroo：</strong>委託人獲得獎勵是因為驗證人驗證交易並鑄造代幣。</p><p>現時似乎是，一個團體正在嘗試建立多個驗證人節點，而且沒有披露他們之間的關聯。 佔用驗證人位置更對去中心化產生了不良影響，亦阻止其他人建立自己的節點。</p><p>雖然沒有明文規定不可以這樣做，但對委託人來說是虛偽和誤導的。</p><p><strong>Nitego29</strong>：確實是這樣，不幸的是，Akash（另一款 Cosmos 系的密碼貨幣）亦發生了這種情況，除非驗證人公然將他的其他節點標記為相同的名稱及加上「2」！ 實在令人討厭，因為他們將有其他人踢出 Active Set。尤其是社群聲稱自己是去中心化，然後用投票決定為 Akash 增加更多節點，但新加的 25 個新節點似乎都是落在同一人手中！</p><p>我認為必須要有一些建基於程式的解決方案，又或者驗證人需要做 KYC 或 DID。</p><p><strong>MasterChihuahua</strong>：如果這樣做，這個驗證人實在太不尊重整個網絡吧。</p><p>似乎就是有一些討論在吉娃娃的 Discord 炸開來，因為那些內容只在驗證人才可以看的私人頻道，所以我也不知戰況如何。但在 Reddit <a href="https://www.reddit.com/r/ChihuahuaChain/comments/sds5bp/validator_stakethat/" rel="noopener noreferrer" target="_blank">另一篇文章</a> 看到似乎在 Discord 中 stakethat 並不承認他有做過這樣的行為。然後又說他只是拿出資金幫他的「新手朋友」去弄節點那樣。</p><h2>我們從中學到些甚麼？</h2><ol><li>對於不公平事情的反應，中外皆共通（廢話）。</li><li>一個人控制多人節點，不求成為最強，只要表現平均然後博得一些 random 人士委託，就算只有 5% 佣金也可以 fat fat。</li><li>做壞事也要醒目，不要給人發現，這種立時上線就委託的做法實在太蠢，區塊鏈的公開記錄是可以看到很多東西的。當然 stakethat 可能真的為了幫人，不過這樣給人誤會也不好吧。</li><li>有人回覆做驗證人最好要 KYC，然而真的要搞的話，去找一個人頭戶又有何難？</li><li>原來 Akash 經已付出了慘痛的教訓。我並不了解他們的佣金或制度是怎樣的。不過我看見他們的驗證人是可以設定 0% 佣金。<strong>假設真的有一個人掌控 25 個節點，用幾個 0% 佣金節點去吸引大量委託，然後用它來投一些盡是讓他的節點獲得可觀回報的議案。由於很多人委託只為了回報，根本不理會自己能否繞過驗證人去投票。於是 0% 佣金驗證人操控大量票源，可以左右大局</strong>。</li></ol><p>誠然做驗證人本身都要付出時間、金錢去租伺服器運行節點，獲得委託和驗證交易回報是應該的，但是暗黑的操控無處不在。老實說被割韭菜只是自己直接投資失利或者貪心真的怪不得人，但是<strong>明明信任一個驗證人給他委託為他充權卻是被視為傻仔才叫人最傷心</strong>。所以無論如何，帶眼識人，增長自己的知識真的重要，不要被人視為傻仔還幫人數錢。</p><p>現時我熟悉的吉娃娃驗證人
