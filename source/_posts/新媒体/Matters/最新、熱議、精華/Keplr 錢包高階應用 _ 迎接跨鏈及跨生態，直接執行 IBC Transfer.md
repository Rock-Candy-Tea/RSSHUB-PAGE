<p>隨著 Cosmos 生態系統越來越多跨鏈及跨生態互聯，有必要多點認識 IBC 這個東西，而 IBC Relayers 和 IBC Transfer 當然是不可或決的。</p><h2>IBC Relayers 是什麼？</h2><p>區塊鏈間通信協議 Inter-Blockchain Communication, IBC 是連接各個 Cosmos SDK 構成區塊鏈的橋樑，使網絡可以相互操作，所以 Cosmos 被稱為「Internet of Blockchains」。當 Cosmos 旗下其中之一條區塊鏈與另一條區塊鏈通訊，例如轉帳密碼貨幣，就必須要 IBC Relayers 幫忙進行傳輸即是所謂的 IBC Transfer。所以沒有 IBC Relayers 就沒有 Cosmos 生態系統的繁榮。</p><p>例如由 LikeCoin chain 傳送 LikeCoin 到 Osmosis，就是 IBC Relayers 在背後運作的結果。在 LikeCoin Discord 就有一條名為 ibc-status 的頻道讓大家看看他們是否健康活潑。</p><figure class="image"><img src="https://assets.matters.news/embed/3ef56bfb-ab40-4a01-b6ac-0acc9b7044e2.png" data-asset-id="3ef56bfb-ab40-4a01-b6ac-0acc9b7044e2" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>Map of Zones 讓 IBC 可視化</h2><p>在 <a href="https://mapofzones.com/" rel="noopener noreferrer" target="_blank">Map of Zones</a>，可看到各種 Cosmos 生態的幣如何透過 IBC Relayers 相互連結，所帶動的交投量及資產數值等。你在投資的幣種交易是否活躍？這個網站都可以一覽無遺。</p><figure class="image"><img src="https://assets.matters.news/embed/ea875ee9-9ced-4a89-b3f9-0673ddb0b341.png" data-asset-id="ea875ee9-9ced-4a89-b3f9-0673ddb0b341" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>Channel ID 是什麼？</h2><p>每一種代幣都有它的 Network ID、Chain ID、Coin Type 和 Channel ID 等沒完沒了的技術細節，以確保你的幣就是你的幣。而在 Cosmos 跨鏈互聯，了解 Channel ID 尤其重要，</p><p>粗略作比喻，例如 Osmosis 開了一家酒店，招呼 $LIKE、$JUNO、$UST 過來玩，總不能生活習慣不同都迫在同一房間吧？所以在 Osmosis 這家酒店，$LIKE 住在 channel-53、$JUNO 住在 channel-42、UST 住在 channel-72。在 <a href="https://www.mintscan.io/" rel="noopener noreferrer" target="_blank">Mintscan</a> 底下選擇 Osmosis，然後再選 <a href="https://www.mintscan.io/osmosis/assets" rel="noopener noreferrer" target="_blank">ASSETS</a> 就可以看到每一種幣的 Channel ID。</p><figure class="image"><img src="https://assets.matters.news/embed/961618fe-3eb3-41a3-afde-9731f9ba0dba.png" data-asset-id="961618fe-3eb3-41a3-afde-9731f9ba0dba" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/fc99f0f8-021e-4b3c-b90f-8205bdb466fe.png" data-asset-id="fc99f0f8-021e-4b3c-b90f-8205bdb466fe" referrerpolicy="no-referrer"><figcaption><span>Osmosis 不同幣的不同 channel</span></figcaption></figure><p>然後在 Cosmos Hub 開了另一家酒店，$LIKE 就變成住在 channel-217 了。每一條鏈中每一款幣的 Channel ID 可以是完全不同，也不是完全不會變改，正如你入住相同酒店也不是每次都住同一個房間一樣，只是熟客當然可以指定房間。</p><figure class="image"><img src="https://assets.matters.news/embed/5ad0019a-c188-433d-ac61-4ac409fecb2c.png" data-asset-id="5ad0019a-c188-433d-ac61-4ac409fecb2c" referrerpolicy="no-referrer"><figcaption><span>$LIKE 在 Cosmos Hub 是 channel-217</span></figcaption></figure><p>每一家酒店有每一家酒店的用途，例如在 Osmosis 是為了在他們家進行交易和提供流動性、在 Cosmos Hub 也是為了在 Emeris 進行交易和提供流動性。所以如果你的 LikeCoin 是用作去中心出版，你就要把它從 Osmosis 傳送回它的主場 LikeCoin chain 才能有用武之地，因為 LikeCoin 是 LikeCoin chain 的 Native Token 原生通證，要回到家才能隨心所欲發揮去中心出版功能，留在酒店只能做交易。</p><h2>如何在 Osmosis 進行 IBC Transfer？</h2><p>當你在 Osmosis 菜單的 Assets 點 LikeCoin 再點 Deposit，其實你經已在進行 IBC Transfer。</p><figure class="image"><img src="https://assets.matters.news/embed/ff823cbb-0ec6-4cb1-bf6e-438e207355d1.png" data-asset-id="ff823cbb-0ec6-4cb1-bf6e-438e207355d1" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>但當然你可以了解一些高階功能，為 Cosmos 生態系統未來發展做準備，畢竟跨鏈甚至跨生態互聯，有機會傳錯 Channel ID 也說不定。</p><h2>如何在 Keplr 錢包進行 IBC Transfer？</h2><p>Keplr 錢包原生不開放 IBC Transfer，所以要在設定打開它，但打開之前，要先了解一下。</p><p>在 Mintscan，到 Osmosis 點 <a href="https://www.mintscan.io/osmosis/relayers" rel="noopener noreferrer" target="_blank">IBC RELAYERS</a>，以 LikeCoin 為例，先找它的 IBC Relayer likecoin-mainnet-2，會看到在 Osmosis 與 LikeCoin 之間（現時還是寫 Unknown 但當搞定錢包地址前綴應該會有正確顯示）有幾條通道，channel-57、58、59 沒有運作了，而 channel-53 與 channel-3 正在運作。在 Osmosis 的 <a href="https://info.osmosis.zone/ibc" rel="noopener noreferrer" target="_blank">IBC Status</a> 亦可以看到相同內容。</p><figure class="image"><img src="https://assets.matters.news/embed/70d07cfd-a8a7-4688-bd1e-4fbc0959fdbb.png" data-asset-id="70d07cfd-a8a7-4688-bd1e-4fbc0959fdbb" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/974bdece-e38d-475e-b7ab-140bef0de4c7.png" data-asset-id="974bdece-e38d-475e-b7ab-140bef0de4c7" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>channel-53 是 $LIKE 在 Osmosis 的房間號，channel-3 就是 $LIKE 在 LikeCoin chain 的房間號，就幻想兩者之間的 IBC Transfer 打通了就是開了隨意門，又或像哈利波特中被馬份用來傳送食死人到霍格華茲的那個櫃，總之兩者一互通就搞定了。</p><p>而 IBC Transfer 當然要兩者有 Relayer 才可以進行，又以 LikeCoin 為例，它與吉娃娃沒有 IBC Relayer，你都不能在兩者之間進行傳輸吧。</p><p>假設現時你在 LikeCoin chain，需要將 $LIKE 使用 IBC Transfer 轉帳到 Osmosis，不使用 Osmosis 介面的話，你可以：</p><p>1. 到 Keplr 錢包點左上角三間，打開「Settings」，將「Show Advanced IBC Transfers」打開，會看到介面多了「 IBC Transfer」按鈕，按下去。</p><figure class="image"><img src="https://assets.matters.news/embed/857d2f6b-9c3e-4711-a606-6c17bab39012.png" data-asset-id="857d2f6b-9c3e-4711-a606-6c17bab39012" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>2. 在 Destination Chain 下面見到 Select Chain，因為現時是由 LikeCoin chain IBC Transfer 到 Osmosis，所以 Destination Chain 目標鏈自然就是 Osmosis。</p><p>選擇了 Osmosis 後在 Channel ID 一欄輸入 channel-3。記不記得上面的圖？LikeCoin chain 的隨意門 channel-3 是可以通到 Osmosis 的 channel-53，由於現在你是在 LikeCoin chain 傳 $LIKE 出去所以填的是 LikeCoin chain 這邊的房間門牌 channel-3 而不是 Osmosis 那邊的門牌。事實上如果你在 Channel ID 填 channel-53 是會出現錯誤的。填好了以後按「Save」。</p><p>留意 Keplr 會自動保存第一次設定的 channel，如果該 channel 經已不活躍，你的幣會被鎖定，所以傳送前請先確保你的 channel 是正確的。</p><p>接著 Destination Chain 出現 Osmosis，Recipient 需要填寫你的 Osmosis 錢包地址。Memo 在 Osmosis 不用填了，其他 Cosmos 區塊鏈就不知道要不要填。接下來點「Next」。</p><figure class="image"><img src="https://assets.matters.news/embed/f9bfbdd0-3510-41b8-a7c2-3873f97c7aef.png" data-asset-id="f9bfbdd0-3510-41b8-a7c2-3873f97c7aef" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>3. Token 出現 LIKE 因為要轉帳 $LIKE 到 Osmosis，數量填 1 試試吧。選擇 Gas Fee 的快慢，點「Submit」。</p><p>出現 IBC Transfer 畫面，點「Approve」。而因為這個是高階應用，按了以後就完了，不會有甚麼視窗彈出來再要你「Approve」，故此真的要很緊慎執行。</p><p>完成後，將 Keplr 菜單轉去 Osmosis 會看見多了 1 LIKE。</p><figure class="image"><img src="https://assets.matters.news/embed/60dc6cbb-a5c4-4186-9faa-83a5837b094d.png" data-asset-id="60dc6cbb-a5c4-4186-9faa-83a5837b094d" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>4. 在 Osmosis 的 Assets 亦會發現多了 LikeCoin，就是你剛 IBC Transfer 過去的。</p><figure class="image"><img src="https://assets.matters.news/embed/e69c9073-a85e-4ff6-9e9e-cb0a99be6f46.png" data-asset-id="e69c9073-a85e-4ff6-9e9e-cb0a99be6f46" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>而在 Mintscan 可以看到現時你的 LikeCoin 是屬於 Osmosis channel-53 的資產。</p><figure class="image"><img src="https://assets.matters.news/embed/7711656b-aba6-4b5a-b73f-1d89e87bea71.png" data-asset-id="7711656b-aba6-4b5a-b73f-1d89e87bea71" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>建議不熟習操作前，請先小額嘗試。</p><h2>我可以對未鏈接到 Keplr 錢包的錢包地址進行跨鏈交易嗎？</h2><p>可以，但該錢包地址必須是非託管錢包 ( non-custodial wallet )，因為接收的錢包只可以透過將帳戶導入 Keplr 來管理密碼貨幣。</p><p>例如你在 Emeris 將 $LIKE 直接轉帳到 Liker Land app 的錢包址而不是先 IBC Transfer 到 Keplr 的 LikeCoin chain 錢包地址再傳送到你的 Liker Land app，你就是夾硬傳送了 channel-217 $LIKE 到 Liker Land app 的錢包。除非之後 Liker Land app 能導出私鑰，否則你的 $LIKE 是無法轉回來到 LikeCoin chain 的。</p><h2>我可以用 IBC Transfer 將幣傳到交易所嗎？</h2><p>不要通過 IBC 直接向 CEX 中心化交易所發送密碼貨幣，如果以交易所地址作為目的地執行 IBC Transfer，資金將會丟失。</p><p>請先將實碼貨幣發送到主鏈的錢包，才發送到交易所。</p><h2>我可以通過 IBC Transfer 發送 Osmosis 與 Emeris/Gravity DEX LP Token 嗎？</h2><p>不能。</p><h2>是不是所有鏈的 Channel ID 都是相同的？</h2><p>不，每條鏈都有自己的 Channel ID （甚至有多於一個 Channel ID），請不要嘗試使用 Mintscan Relayers 頁面或 Osmosis IBC Status 頁面所顯示以外的 Channel ID。</p><h2>IBC Transfer 需要多久才能完成？</h2><p>這取決於 IBC Relayers 的當前狀態，但正常不應該超過 1 分鐘。</p><p>如果 5 分鐘後資金仍未到賬，請查帳，詳情可參考 <a
