<figure class="image"><img src="https://assets.matters.news/embed/a63acf58-f9fd-45f7-969c-a2a5a26a12a7.png" data-asset-id="a63acf58-f9fd-45f7-969c-a2a5a26a12a7" referrerpolicy="no-referrer"><figcaption><span>圖片取自 Keplr 官方網站</span></figcaption></figure><p>Keplr 錢包手機版<a href="https://medium.com/chainapsis/keplr-mobile-is-now-live-9fde7195b00f" target="_blank">突然於今日推出了</a>，我當然要立即嘗鮮，以下將介紹如何把現有在桌機的 Keplr 錢包使用 Import from Keplr Extension 功能導入 Keplr 手機版：</p><h2>在 Keplr 手機應用程式導入錢包</h2><p>1. 首先請確認你在桌面電腦的 Keplr 版本經已升級為 0.90，如果你不知道怎樣處理桌機版，請回看 <a href="https://matters.news/@daisy/keplr%E9%8C%A2%E5%8C%85%E5%85%A8%E6%94%BB%E7%95%A5-%E5%A4%A7%E5%AE%B6k%E5%99%B4%E7%99%BC%E5%A4%A7%E8%B2%A1-bafyreifgvlva5xmncblzm5afdvykws4cahgnutxxnbx5lbpjz4c4hgtzjy" target="_blank">Keplr 錢包全攻略</a> 一文。</p><figure class="image"><img src="https://assets.matters.news/embed/e65a22e2-35a7-4914-9493-3abab34eb7c6.png" data-asset-id="e65a22e2-35a7-4914-9493-3abab34eb7c6" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>2. 到 <a href="https://play.google.com/store/apps/details?id=com.chainapsis.keplr&hl=en_US&gl=US" target="_blank">Google Play</a> 或 <a href="https://apps.apple.com/in/app/keplr-wallet/id1567851089" target="_blank">Apple Store</a> 下載 Keplr 手機應用程式。</p><p>3. 打開後會看到三個選項：Sign in with Google、Import from Keplr Extension 及 Import existing wallet。Import existing wallet 的程序應該和桌機版沒有甚麼分別，所以我們來試試 Import from Keplr Extension。</p><figure class="image"><img src="https://assets.matters.news/embed/373fcbd0-5b77-4f74-bdd2-96d489296689.png" data-asset-id="373fcbd0-5b77-4f74-bdd2-96d489296689" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>4. 在桌機的 Keplr，選左上角三間，再選 Settings，然後會看到新的「Link Keplr Mobile」，點選它。</p><figure class="image"><img src="https://assets.matters.news/embed/8bec349e-669e-4edd-8608-2a4798320f70.png" data-asset-id="8bec349e-669e-4edd-8608-2a4798320f70" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>5. 會出現 Link Keplr Mobile 的畫面，輸入你的 Keplr 帳戶密碼，點「Confirm」。</p><figure class="image"><img src="https://assets.matters.news/embed/d8bb6e6e-4840-4664-9faa-84ee2708994c.png" data-asset-id="d8bb6e6e-4840-4664-9faa-84ee2708994c" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>6. 然後出現一個 QR Code。</p><figure class="image"><img src="https://assets.matters.news/embed/5c355c91-4233-48be-95ec-538d0158fd88.png" data-asset-id="5c355c91-4233-48be-95ec-538d0158fd88" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>7. 接著回到 Keplr 手機應用程式，點「Import from Keplr Extension」後，再點「Next」。</p><figure class="image"><img src="https://assets.matters.news/embed/f2a73d70-0253-4ba3-a47e-17545c2549f8.png" data-asset-id="f2a73d70-0253-4ba3-a47e-17545c2549f8" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>8. 看見 Password 及 Confirm password，輸入你的 Keplr 帳戶密碼兩次。</p><figure class="image"><img src="https://assets.matters.news/embed/d2d9ffe3-e127-42ac-a594-d05fa18b7b89.png" data-asset-id="d2d9ffe3-e127-42ac-a594-d05fa18b7b89" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>9. 出現 You're all set! 頁面，假設你需要使用生物應證（手指模）去解鎖你的錢包，請讓 Enable Biometric 位於已選取的狀態，否則請不要選擇，然後點「Done」。這樣做以後就可以使用手指模去打開錢包，否則每次都要輸入密碼才可。</p><figure class="image"><img src="https://assets.matters.news/embed/885e3731-9d10-4693-8415-5f16b4f8ea9c.png" data-asset-id="885e3731-9d10-4693-8415-5f16b4f8ea9c" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>你的 Keplr 錢包經已從桌面電腦導入到手機應用程式中。</p><h2>簡易操作</h2><p>1. 點左上角會出現三款密碼貨幣，分別為 Cosmos、Osmosis 及 Regen。點其中一款會出現你能動用及經已委託的資產總值。可使用「Send」及「Stake」自由傳送及委託你的密碼貨幣。</p><figure class="image"><img src="https://assets.matters.news/embed/2016d593-8e10-4f9c-9cba-1690ebc2e84d.png" data-asset-id="2016d593-8e10-4f9c-9cba-1690ebc2e84d" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/71dac7ce-b1b1-4842-9bfc-8a3a30a22fc5.png" data-asset-id="71dac7ce-b1b1-4842-9bfc-8a3a30a22fc5" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>2. 點下方的齒輪、出現設定頁。現時能設定的有：</p><ul><li>點最上錢包圖案可加入其他 Keplr 錢包</li><li>Currency 可以自行選擇想顯示的種類</li><li>Address book 即是自行設定地址簿</li><li>View mnemonic seed 就是查看私鑰</li><li>Use biometric authentication 選取的話就會需要使用生物應證（手指模）去打開你的錢包，如果在設定時沒有選擇，可以在這邊重啟</li><li>Keplr version 就是 Keplr 版本號</li><li>Delete this wallet 就是刪除這個錢包</li></ul><p>由於所有設定功能和桌面電腦版相同，所以不逐一重覆。</p><figure class="image"><img src="https://assets.matters.news/embed/00c5d56f-1fe0-4bb7-8de7-2ec3afe75230.png" data-asset-id="00c5d56f-1fe0-4bb7-8de7-2ec3afe75230"
