<p>文章譯自：<a href="https://decrypt.co/109423/its-time-to-separate-nfts-from-digital-art" rel="noopener noreferrer" target="_blank">Decrypt</a>，（本文為作者 Abigail Carlson 於法國博物館旅遊的小記）</p><hr><p>最近，我在法國 Nice 的 Matisse 博物館參觀了 <a href="https://www.hockney.com/home" rel="noopener noreferrer" target="_blank">David Hockney</a> 臨時展覽。 Hockney 被視為是英國當代最具有影響力的藝術家之一。他於1972年創作的《藝術家的肖像》於2018年以9000萬美元的價值在佳士得拍賣會中售出，打破了當時的最高售價紀錄。</p><figure class="image"><img src="https://assets.matters.news/embed/57000fc3-2d29-4ccb-8be4-c40509bad9a0.jpeg" data-asset-id="57000fc3-2d29-4ccb-8be4-c40509bad9a0" referrerpolicy="no-referrer"><figcaption><span>《藝術家的肖像》，David Hockney</span></figcaption></figure><p>讓我著迷的並非是 Hockney 的畫，儘管他們真的很漂亮，但最讓我著迷的事是他在 67 歲時開始學習使用photoshop，開始嘗試一種新的藝術媒介。在當時，大多數藝術家都會堅持他們自己最了解的東西，然而，Hockney 的好奇心卻促使著他嘗試新的事物。他於2008年獲得了他的第一步iphone，當時他已經71歲了，隔年，他用拇指創作了一千幅的數位藝術創作，現在他成為了一位多產多元的數位藝術家。在 Nice 的展覽「<strong>A Paradise Found</strong>」展示了一系列 Hockney 從未公開的 Ipad 畫作。</p><figure class="image"><img src="https://assets.matters.news/embed/9a45a220-cf82-467d-bdf9-8d401bcfeb62.webp" data-asset-id="9a45a220-cf82-467d-bdf9-8d401bcfeb62" referrerpolicy="no-referrer"><figcaption><span>A Paradise Found" iPad paintings. (Photo: Abi Carlson)</span></figcaption></figure><p><strong>在展覽周圍閒逛之時，我驚覺發現，展覽居然對 NFT 隻字未提。 </strong></p><p>我早已習慣將 NFT 等同於數位藝術，以至於我幾乎震驚於此次展覽幾乎沒有提及 NFT 。該不會是 Hockney 錯失了這次機會？或許真的如此，藝術家究竟是否需要將他的藝術創作轉變為 NFT 是值得商榷的。但事實上 Hockney 曾經公開批評過 NFT ，稱其為「愚蠢的小事」。</p><p>我很開心 Hockney 並沒有進入 NFT 領域，並感謝他堅定的把持自己的立場，這樣的立場帶出了一個有意義的提醒： <strong><u>NFT 與數位藝術品是有著不同意義的詞，是時候將這兩個詞彙分開討論了</u></strong>。 儘管數位藝術品可以製成 NFT ，但 NFT 最終是一個更廣泛的類別，我相信將兩者聯繫的太緊密會損害各自的利益。 數位藝術只是人類使用其最新的工具製作的藝術品。從洞穴壁畫到紙、筆、顏料乃至於新科技的應用，人類總是會使用當下的工具進行藝術創作。</p><p>雖然 NFT 收藏品確實以數位藝術為特色，但我認為許多 NFT 收藏品的重點並非是藝術品本身，而在於<strong><u>藝術品的市場銷售性</u></strong>。</p><hr><h2><strong>藝術收藏與藝術商業</strong></h2><p>NFT 收集者會不定時的查看地板價與供應量等相關數據，藉此收集有關流通性的資訊。當然，藝術家要成功依舊和以前一樣需要走上一段很長的路。需要明確的是這些事情都不是錯誤的，只是我們要明白，<strong>NFT 不單單是藝術品，他同時也是一項<u>金融產品</u>。</strong></p><p>我身邊就有不只一個投資者在周末交易這些 JPEG ，這就是一個很好的例子。對他們來說，這就是一個金融商品，如果他們能夠透過買賣 NFT 獲得翻身的機會，誰能責怪他們呢？ 從藝術家資助人到拍賣行，金融和藝術世界的融合已經不是甚麼新鮮事了。</p><p>但 NFT 的到來也帶來了大量的捲款詐騙，這些都困擾著 NFT 世界，<strong><u>讓藝術家們沒辦法為自己的信譽背書</u></strong>。怪不得一些數位藝術家會故意避開這個空間，因為擔心自己的名譽受損。</p><h2><strong>不僅僅是JPEG</strong></h2><p>我們不必僅僅將數位藝術品製作成 NFT ，這樣做實際上可能會貶低數位藝術品本身。與此同時，NFT 有許多更令人著迷的替代方向，這些運用無疑將改變我們對 NFT 的看法，以下有幾個例子：</p><h2><u>票卷</u></h2><p>眾所周知，我們現實世界中的票務行業常常受到偽造、欺詐等挑戰困擾。但若以 NFT 形式發行活動門票，<a href="https://www.binance.com/zh-TW/blog/nft/%E4%BB%80%E9%BA%BC%E6%98%AF-nft-%E7%A5%A8%E5%88%B8%E4%BB%A5%E5%8F%8A%E5%AE%83%E5%A6%82%E4%BD%95%E9%81%8B%E4%BD%9C-421499824684904022" rel="noopener noreferrer" target="_blank">便可以輕鬆的分發和即時驗證</a>。還有可能從二級市場的銷售獲得持續的版稅，這些版稅可能直接流向利益相關者、藝術家和活動組織者。</p><figure class="image"><img src="https://assets.matters.news/embed/7e4f50e3-76d2-4e26-b52c-05acfe81586b.jpeg" data-asset-id="7e4f50e3-76d2-4e26-b52c-05acfe81586b" referrerpolicy="no-referrer"><figcaption><span>Binance</span></figcaption></figure><h2><strong><u>音樂</u></strong></h2><p>在串流音樂媒體出現之前，大多數的音樂藝術家都是透過實體的音樂來賺錢。在串流平台出現之後，音樂的稀缺性被打破。但 NFT 的出現彌補了一些音樂稀缺性的丟失。例如 Kings of Leon 是第一支以 NFT 形式發行的專輯樂隊，並從銷售額中賺取了200萬美元。</p><figure class="image"><img src="https://assets.matters.news/embed/4287317f-ea58-4b4d-81e4-ef706feff6dc.gif" data-asset-id="4287317f-ea58-4b4d-81e4-ef706feff6dc" referrerpolicy="no-referrer"><figcaption><span>Kings of Leon 發行的 NFT 音樂專輯</span></figcaption></figure><h2><strong><u>房地產</u></strong></h2><p>NFT 在房地產應用中有幾個面向：<br class="smart">首先，<strong>他們可以代表正在購買的實物財產</strong>。雖然這在很大程度上取決於未來法律如何約束 NFT 相關產業。<br class="smart">其次，透過購買具有 NFT 資產屬性的公寓，<strong>可以立即訪問公寓的整個歷史</strong>，包括了以前的買家投資再到法律糾紛和相關交易。同時，透過 NFT 的轉賣<strong>可以更立即的轉移房地產</strong>。<br class="smart">最後則是可以<strong>簡化合資購買房產的手續與成本</strong>，當前若需要合購房產需要龐雜的手續與文書處理，包括法律成本、時間等。而透過智能合約的編纂可以更輕鬆且便利的處理以上的繁文縟節。</p><h2><strong><u>運動</u></strong></h2><p>儘管這方面並非是我所專精的領域，但無可厚非的，NFT 依舊對此行業產生了影響。越來越多的體育俱樂部開始透過 NFT 來增加粉絲參與度並獲得額外的收入。<strong>NBA Top Shot</strong>就是一個很好的例子，他是官方授權的 NBA 數位收藏品。擁有 NFT 還可以用作 IRL 社區活動的社群門戶，為持有者提供與球員們見面的機會。</p><figure class="image"><img src="https://assets.matters.news/embed/ebfa0ebc-4121-41dd-a4aa-0b5adb2ef6d6.jpeg" data-asset-id="ebfa0ebc-4121-41dd-a4aa-0b5adb2ef6d6" referrerpolicy="no-referrer"><figcaption><span>NBA Top Shot</span></figcaption></figure><h2><strong><u>品牌</u></strong></h2><p>從時尚到奢華汽車與商品，各個領域的品牌都在嘗試 NFT 系列。這看起來像是在購買實物資產的同時發布 NFT 。RTFKT Studios 在2021年率先推出 NFT ── 該活動在 7 分鐘內產生了31萬美元的收入。Dolce & Gabbana 於 2021 年將實體與虛擬結合在一個系列中，並賺了 565 萬美元。</p><figure class="image"><img src="https://assets.matters.news/embed/132da3da-2501-4b56-894a-aa0734ec5bb0.png" data-asset-id="132da3da-2501-4b56-894a-aa0734ec5bb0" referrerpolicy="no-referrer"><figcaption><span>Photo/ Nike + RTFKT</span></figcaption></figure><p>特別是對於時尚品牌，NFT 也可以用作供應鏈的 QR codes。一件衣服的整個供應鏈可以記錄在區塊鏈上，作為 NFT 發布的可掃描二維碼<strong>可以讓消費者檢查他們購買的衣服來源</strong>。這種透明度不僅可以徹底改變時尚品牌，還可以改變整條供應鏈。 我甚至不打算涉足虛擬世界和遊戲， 但 NFT 提供了數位藝術之外的廣泛應用，我的預測是我們很快就會開始將 NFT 與其他技術形式聯繫起來而不是主要用於藝術。</p><hr><h2><strong>NFT 的藝術性</strong></h2><p>為了完整我的論述，我仍然認為在某些情況下，<strong><u>NFT 依舊可以成為數位藝術品的絕佳載體</u></strong>。 其中之一為<strong>生成藝術</strong>。</p><p>生成藝術是數位藝術的一個類別，它使用算法以一種獨特的類型創造並輸出。編寫這些代碼需要技巧和意圖。一些編程集合或平台要求將特定功能嵌入代碼中，以彰顯某種審美觀，藉此影響結果……換句話說，<strong><u>過程本身就是藝術。</u></strong></p><p>生成藝術是 NFT 的完美用例。由於藝術品的屬性是在鑄造過程中<strong>隨機</strong>產生的，因此將藝術品的鑄造人帶入藝術品本身的創作過程中——這可以與藝術品產生獨特的情感紐帶。</p><p>生成 NFT 藝術的最早例子之一是 Chaos Machine，這是一個於 2018 年在 Distributed Gallery 誕生的項目。這台機器會持續的燃燒鈔票，背景則會隨機播放音樂，同時為用戶鑄造代幣並打印成二維碼。</p><figure class="embed-video"><div class="iframe-container"><iframe src="https://www.youtube.com/embed/fVPh3FeqpU0?rel=0" frameborder="0" allowfullscreen="true" sandbox="allow-scripts allow-same-origin allow-popups"></iframe></div><figcaption><span>Chaos Machine</span></figcaption></figure><p>成功的生成 NFT 收藏通常涉及一定數量的可鑄造藝術品、強大的社區和清晰的路線圖。這些例子包括了 Cryptopunks、Autoglyphs、BAYC、Chromie Squiggles 和 Euler Beats。 無論你是否喜愛他們，這些龍頭 NFT 的影響是不可否認的，也不能否認 NFT 為他們提供了一條獨特的途徑來增加他們的藝術收入來源以及建設其社區。</p><figure class="image"><img src="https://assets.matters.news/embed/dfbeef0b-6258-49de-8b33-53094bdd247c.png" data-asset-id="dfbeef0b-6258-49de-8b33-53094bdd247c" referrerpolicy="no-referrer"><figcaption><span>Autoglyphs</span></figcaption></figure><p>這讓我想到第二個 NFT 得以成為數位藝術品的另一個例子：<strong><u>社群。</u></strong></p><p>以上提到的主種著名的 NFT 項目都以創建社群的形式進行了各式各樣有趣的社會實驗。不可否認的，將人們因興趣與喜愛聚集在一起的社群是一種強大的力量。 請注意：儘管不是主要在數位藝術空間工作的業餘藝術家們，同樣也可以發行 NFT 。畫家、電影製作人、作家、音樂家等都可以發行 NFT 收藏，保證他們的粉絲每年都能參加一定數量的活動、見面會等。</p><p>數位藝術品 NFT 透過對社群訪問進行代幣門控，在培育社群方面發揮著重要的作用，從而以一種超出目前社交媒體以及平台可以實現的方式來管理社群。</p><h2><strong>分開後將更強</strong>大</h2><p>雖然我最終認為 NFT 應該與數位藝術分離，但這主要是因為該技術可以有無限的應用，以及該空間不幸聚集的一些負面關聯。但有一件事是肯定的，<strong><u>無論哪種方式，David Hockney 的創作都會是有價值的。</u></strong>萬一他改變了對 NFT 的看法，我毫不懷疑，不止一個 NFT 工作室會非常樂意幫助他，將他的 iPad 系列轉變為一個生成的 NFT 藝術收藏。但這可能走得太遠了……</p><hr><h2>個人心得</h2><p>本文為作者前往法國一間博物館之後的小散記，敘述了數位藝術何以展現其藝術特性，同時闡述 NFT 如何得以走出「數位藝術」的窠臼之中。</p><p>首先第一個問題：數位藝術是藝術嗎？這讓我想到大名鼎鼎的德國哲學家班雅明於《<a href="https://www.books.com.tw/products/0010814904" rel="noopener noreferrer" target="_blank">機械複製時代的藝術作品</a>》，文中他首先預設了藝術品有所謂的「<strong><u>靈光</u></strong>」之存在，這種靈光賦予了藝術作品一種無法被複製、無法被竄改的獨一無二的特性。靈光是創作者賦予這個作品生命的微光，揭露了這張作品完整的意圖。</p><figure class="image"><img src="https://assets.matters.news/embed/def56908-51ac-4ec5-934c-0d169dba9e95.jpeg" data-asset-id="def56908-51ac-4ec5-934c-0d169dba9e95" referrerpolicy="no-referrer"><figcaption><span>華特，班雅明《機械複製時代的藝術作品》</span></figcaption></figure><p>但這種靈光到了所謂「機械複製時代」之後便逐漸喪失了，機械複製時代便是相機、攝影機、錄音機等機械被發明，人們不再需要遠赴羅浮宮去觀賞真正的《蒙娜麗莎》，或許網路上某一張被拍攝下來的《蒙娜麗莎》更接近「真品」。自此，藝術作品經歷了一次「崩潰」，藝術作品被記錄、被複製、被收藏，一張偉大梵谷的《星夜》成為了某一間不知名麵店的廁所壁紙……</p><p>但藝術自此墮落了嗎？這便是第二個問題，「靈光」真的消失了嗎？我想不是的，既有的藝術結構在機械複製時代下逐漸破碎，以往一般百姓是無法接觸所謂「藝術」的，無論是西方抑或是中方，藝術是有錢人的玩意，是上流社會的閒暇。那麼，進一步追問就是難道藝術只能是有錢人的玩意嗎？</p><p>身為左派哲學家的班雅明對此也有一些見解，他或許認為個別藝術作品的靈光消逝了，但作為社會結構的「藝術」則再次的揮灑了其靈光。藝術不再是上流社會的主宰者，藝術中所透漏的隨機性、美感、元素等被大量的「機械」所重構，例如生成藝術、街頭攝影等。藝術回歸了大眾、回歸了百姓。只是多數的百姓依舊不習慣自己擁有這股強大的力量。</p><p><strong><u>最終，人們往往只會說「這也能是藝術嗎？」</u></strong>是的，這也是藝術，藝術不再高高在上，儘管既有的結構依舊，但百姓擁有的權力更多。那麼回到 NFT 作為藝術，這樣一張 JPEG 也是藝術嗎？</p><p>是的，這也可以是藝術，甚至比之前的 JPEG 更「藝術」。某種程度上來說，他回應了班雅明對於傳統藝術的想像，他有<strong><u>稀缺性</u></strong>，<strong><u>有所謂的「真品」的概念</u></strong>。</p><p>人類是一種複雜的動物，對於真與假的追求自古希臘便爭論不休，但只要誰掌握了「真」，掌握了共識下的「真」那麼人們就會為之瘋狂。柏拉圖的理想國裡的洞穴譬喻已經是舉到爛掉的例子了，然而就連對世界是否「真實」都能爭論兩三千年，對一張影像是否真實討論個十幾二十年也不為過吧！</p><figure class="image"><img src="https://assets.matters.news/embed/b0b7817e-666a-4c67-ab14-18829882ecfb.png" data-asset-id="b0b7817e-666a-4c67-ab14-18829882ecfb" referrerpolicy="no-referrer"><figcaption><span>我們又怎麼知道我們生活的地方不是他人投射出來的世界呢？</span></figcaption></figure><p>所以，對於真假的討論依舊會持續進行，然而就像本文所說的，無論是真是假，藝術或非藝術，「<strong><u>David Hockney
