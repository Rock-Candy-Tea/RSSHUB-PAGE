<p>記得幾年前，強如世界第一大交易所「幣安」也多次傳出「被盜幣傳聞 / 消息」。在 CEX 也常常被盜下， DEX 純合約管理的模式，又怎能獨善其身。現時連帶Game Fi 與 DEX 結合，更容易成為缺口。</p><p>我有時在想「門口越多，越容易吸引盜賊」， Cosmos 生態內，每一個項目是一條鏈，各鏈之間又互通。若果有一天，有人發現 Cosmos SDK 內一個缺口，豈不會全軍覆沒?</p><p>程式可以做很多事情，但不一定完美，怎樣也好也要先想想自己的情況及風險。今天突然在想:「若果Osmosis被駭，盜了款，那麼應該會怎樣?」</p><p>不同的交易所被黑客攻擊後，不論是盜款或閃電貸，最終也是想獲取利益。應該不會只是想「攻擊玩玩」吧。回看以往的CEX或者DEX，甚至是Crypto Visa 所盜取的貨幣，全都是一些可以在其他交易所流通的貨幣。</p><p>(縱使小至 Steem 最小的分叉鏈 Blurt 也曾被盜取大量的幣，最終沽壓至 2 SAT (比特幣最少的價值單位) ，朋友竟然在這一個位置接了 20萬 Blurt 幣。)</p><p>黑客盜幣後，有的「以最快速度售出 或 轉帳至其他貨幣」；也的「守著不動，靜待幾年後的時機」。</p><p>若果有一天 Osmosis 被盜幣，黑客會盜甚麼幣呢?</p><p><strong>第一，必然是即是可以轉走至 CEX 或 DEX 的幣，並且盡快洗白</strong></p><p>在 Osmosis 上最快是 Atom, Luna, CRO, UST</p><p><br></p><p><strong>第二，體量必須夠大，當中有兩個特點</strong></p><p>A. 盜一次幣，花這麼多工夫，風險一樣，何不一次盜多一點，縱使最後未能「洗白」，也可以在朋友間「吹牛」；</p><p>B. 盜極少額的幣，並且持續盜。</p><p>相信很多有都有經驗或聽聞朋友的 Metamask 錢包 或 Steem 幣號被入侵，入侵後不論存甚麼幣或NFT都會立即被提走。</p><p>這些小賊所盜金額不多，但是細水長流，並且不容易發現。縱使個人向交易所舉報， CEX 或 DEX 交易所也「不一定」通報其他交易所</p><p><br></p><p><strong>第三，盜去LP Token</strong></p><p>LP Token 其實是可以轉帳，只要那一個帳號持有 LP Token 就可提存幣，只是在 Keplr 錢包裡不能看到 LP Token。不過，Osmosis 其實是有 LP Token。進入各人自己的質押池，看看下半部的 My Bondings 就是 LP Token。</p><figure class="image"><img src="https://assets.matters.news/embed/b26a9b89-9427-4274-97ee-92257367c6fc.png" data-asset-id="b26a9b89-9427-4274-97ee-92257367c6fc" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>當整個交易所的漏洞被盜幣，過往情況裡，有些交易所表明會「全數賠償」，不過不知相隔多久；有些交易所會不作正面回應，狀況就如 2019年時的幣安，網上已有大量投訴，並且幣安也不太願意承認被盜幣。</p><p>任何平台也有機會遇到盜幣事件，但是我又能否避免呢? 我現時在 Osmosis 裡最大的部份是 OSMO 幣，若果 OSMO 不幸被盜幣，黑客會好像其他小型項目一樣，用最快速度換為較流通貨幣，即是 Atom, Luna, CRO 及 UST。這一個過程中，關於「超大額」兌換方面，Osmosis 不知會否監察。</p><p>若果實在不幸已被提走，先不理 Osmosis 會不會賠償。而想知道 Osmosis 又有沒有能力做賠償呢?</p><p>當我想到這裡的時候，刻意想看看 Osmosis 這公司的規模:</p><figure class="image"><img src="https://assets.matters.news/embed/d1f9d77a-a5de-4470-b2c3-f4792e88fa4a.png" data-asset-id="d1f9d77a-a5de-4470-b2c3-f4792e88fa4a" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>來源: <a href="https://www.crunchbase.com/organization/osmosis-foundation" rel="noopener noreferrer" target="_blank" class="keychainify-checked">https://www.crunchbase.com/organization/osmosis-foundation</a></p><p>原來 Osmosis 在上年 10月做了初始融資，收到六個投資人合共 2100萬美元的資金。</p><figure class="image"><img src="https://assets.matters.news/embed/617dcacf-2f5d-4094-9008-2658ad9b7f67.png" data-asset-id="617dcacf-2f5d-4094-9008-2658ad9b7f67" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/e82a5a9e-bf04-492c-9acc-f6d007fcb13e.png" data-asset-id="e82a5a9e-bf04-492c-9acc-f6d007fcb13e" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>原來六個投資機構是由頂級區塊鏈項目投資機構 Paradigm 領投，Terra 的創辦人 Do Kwan 也是其中一位投資者。按照投資機構的規模，除了 Paradigm 外，其他都是近期成立的新投資基金。若果被盜的幣規模不大，若果在數千萬美元，可能各個投資機構也有可能發善心賠償。只是可能.......</p><p><strong>若果被盜金額一大，實在需要白武士來拯救。</strong></p><p>再回看。。。。。。交易所是有出現被盜的風險，那麼自己能避險嗎?</p><p>我跟太太及朋友也表達過，我是相信自己的加密貨幣，終會有一天被盜，正如我日常外出帶著錢包，終會有機會遇上小偷一樣的心情。</p><p>當日常生活上，我會好好保護自己的錢包，現時越來越多項目透過空投或獎勵騙取不同的授權。我們是否任何項目也參加呢? 與及會否分開「主帳號」及「小號」，當有一些新項目出現時，就用小號來參加呢?</p><p>此外，又正如日常生活，當真的遇上小偷時，知不知怎樣「報警」呢? 還是只懂寫一篇文章訴苦? 又或者在Telegram / fb / 微信群找朋友協助呢?</p><p>我發覺，原來我沒有加入 Osmosis 的官方Telegram，縱使大大個Telegram 的按鈕在平台的左下角，又縱使我已使用超過半年，但是我仍沒有加入。當我剛剛點入時，真的嚇了一跳，竟然只有 17000人。</p><figure class="image"><img src="https://assets.matters.news/embed/cf39bd46-5272-45f5-a501-bca695adc432.png" data-asset-id="cf39bd46-5272-45f5-a501-bca695adc432" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p>這一個 Osmosis 官方 Telegram 的 17000人，相當有數千人是機械人，實質真人可能只有 10000 左右。再看看 Discord 也是只有 14000人。</p><figure class="image"><img src="https://assets.matters.news/embed/b8a472dc-9d0c-4890-a884-5b22c2acc5b7.png" data-asset-id="b8a472dc-9d0c-4890-a884-5b22c2acc5b7" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p>我突然發覺，原來過往我好像一隻在頭前方吊著紅蘿蔔的騾仔。眼中只看到不同的每日利息及空投而向前走，或許會懂得保護自己的錢包私鑰(紅蘿蔔)，但是我是很少去理會「我為甚麼要拉車? 到底是為誰拉車? <strong>甚至翻車了要怎麼辦?</strong>」</p><p>眼中只看著有利潤就衝，又或者看到身邊朋友寫文寫了利潤 (例如我的文章)，自己就盲目衝一份。</p><p>突然感到<strong>「我到了區塊鏈世界，為甚麼自己的智力及保護能力，突然下降這麼多，尤如一位小學生一樣。」</strong>不求甚解?? 奇怪!!</p><p>實在不能再矇矇的，不能希望別人能幫自己，<strong>真的要靠自己啦</strong>。</p>
