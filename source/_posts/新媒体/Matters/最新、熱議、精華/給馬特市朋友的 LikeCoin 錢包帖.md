LikeCoin 的應用場景</h2><p><strong>Matters 和 LikeCoin 是兩家不同機構來的<br class="smart">Matters 和 LikeCoin 是兩家不同機構來的<br class="smart">Matters 和 LikeCoin 是兩家不同機構來的</strong></p><p>重要的東西要說三次，筆大的 ✅ 5. Matters x LikeCoin 其實也有說明，但因為現時 Matters 與 LikeCoin 深度整合，所以很多人還是會以為兩者是同一家公司。</p><p>如果你有 LikeCoin 問題，例如時常有人問的綠圈圈、你可以到 Liker Land 網站右手邊的菜單找「幫助」，或者在 Liker Land app 點「聯絡我們」尋找協助。而如果是 Matters 問題，例如不能用港幣支持作者、標籤不能用之類，就直接電郵 hi@matters.news 處理吧。</p><p>LikeCoin 除了在近日亦有在 Matters 「上水」的繪畫平台 <a class="mention" href="https://matters.news/@illustbuy" target="_blank" data-display-name="illustBuy" data-user-name="illustbuy" data-id="VXNlcjo4MDIxMg">﻿<span>@illustBuy</span>﻿</a>  可以使用，Medium、方格子、Blogspot 甚至自架的 WordPress 和其他網站都可以使用。Youtube 用戶只要安裝瀏覽器插件也能拍手。更多應用場景可參看：<a href="https://docs.like.co/v/zh/user-guide/creator" rel="noopener noreferrer" target="_blank"><br class="smart">LikeCoin 大百科 LikeCoin button 讚賞鍵</a></p><h2>2. 讚賞公民是一個身份但不是沒有它甚麼不成</h2><p>在 Discord 有人誤會了要<s>成為讚賞公民才能拍手、成為讚賞公民後別人為自己拍手才能獲得 LikeCoin，甚至不成為讚賞公民為別人拍手對方不能獲得 LikeCoin</s>，這樣的想法是錯誤的。</p><blockquote>已登記 Liker ID 的 Liker 以 Liker Land 手機應用程式閱讀文章，可方便地拍手 / 按讚 / Like 讚賞作者。文章中凡出現讚賞鍵，便可以按讚 1 至 5 次，每次拍掌都會為創作者獲得每天發放其中一部份「創作基金」的收入。</blockquote><blockquote>若你是讚賞公民，更可以大比數分配每天發放的「創作基金」，令作者獲得額外收入。</blockquote><p><strong>所以就算不是讚賞公民，為別人拍手也可以令對方獲得 LikeCoin（只是沒有讚賞公民那麼多），而同樣，你也不需要成為讚賞公民才能得到別人透過拍手分配給你的 LikeCoin。</strong>更多詳情可看：<a href="https://docs.like.co/v/zh/user-guide/liker-land/like" rel="noopener noreferrer" target="_blank"><br class="smart">LikeCoin 大百科 Like — 化讚為賞</a>、<a href="https://docs.like.co/v/zh/user-guide/civic-liker/creators-fund" rel="noopener noreferrer" target="_blank">創作基金</a></p><h2>3. Super Like 與個人主頁</h2><p>如果你是讚賞公民並為別人拍手，會顯示在你的個人主頁「已 Super Like 內容」。而如果你自己 Super Like 自己，會顯示在「作品」頁。但留意自己為自己 Super Like 是不能讓你的文章獲得創作基金分配。<a href="https://docs.like.co/v/zh/user-guide/creatortools/portfolio-page" rel="noopener noreferrer" target="_blank"><br class="smart">LikeCoin 大百科 個人主頁</a></p><h2>5. 錢包：Metamask V.S. href="https://matters.news/@daisy/like-coin-%E6%94%BE%E5%9C%A8-keplr-%E9%82%84%E6%98%AF-liker-land-%E6%AF%94%E8%BC%83%E5%A5%BD-bafyreid4sng4t7pdywrrp3z2pywmdd2mgo5hx4fd2eqhbottd6kd32rnly" Keplr，而<strong>一旦註冊或綁定 Matters ID 和 Liker ID 便無法解綁</strong>，所以除了不要問 Matters 是不是可以幫你綁另一個 Liker ID，也不要問 Matters 是否可以綁定你的 Keplr，因為技術上根本不可行。</li><li>除了不能在 Matters 使用，以 Keplr 註冊的 Liker ID 不能登入 Liker Land
