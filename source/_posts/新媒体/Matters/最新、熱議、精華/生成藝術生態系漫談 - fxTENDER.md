referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>今天來討論挺有意思的新團體，<a href="https://fxtender.art/icons" rel="noopener noreferrer" target="_blank">fxTEDNER</a>。</p><p>fxhash 成立僅六個月，artblocks 又比 fxhash 還要早整整一年成立。這兩個平台是目前生成藝術市場最具代表性的市場，兩者皆是由收藏者抽選 hash，讓收藏過程產生變化性。此外還有走不同商業模式的生成藝術平台，如可調整 hash（種子碼）的 Endlessway、Crayon Code，或走會員制的 Gen.art 等，但規模都不及前兩者還要大。</p><figure class="image"><img src="https://assets.matters.news/embed/a3f09725-1a56-497c-a654-a53e5dc06a35.png" data-asset-id="a3f09725-1a56-497c-a654-a53e5dc06a35" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>Artblocks 於去年八月上演大驚奇，不少文章已經探討過，而回測歷史交易量，fxhash 似乎正在重演 Artblocks 的軌跡。我於去年 12 月與今年 4 月分別寫過兩篇文章分享 fxhash 的開放特色，也分析 fxhash 與 artblocks 本質上的不同，一言以蔽之，是開放的跳蚤市場與封閉的大教堂之爭。</p><ol><li><a href="https://matters.news/@mashbean/215844-nft%E6%95%85%E4%BA%8B%E9%9B%86-%E7%94%9F%E6%88%90%E5%BC%8F%E8%97%9D%E8%A1%93%E5%B7%A8%E6%B5%AA-%E9%81%A9%E5%90%88%E5%A4%A7%E6%95%99%E5%A0%82%E9%82%84%E6%98%AF%E8%B7%B3%E8%9A%A4%E5%B8%82%E9%9B%86-bafyreia5wv7wc6w57mlcrn35gs3ptymqzipearzoyn7evcne2oyjhv3oim" rel="noopener noreferrer" target="_blank">生成式藝術巨浪，適合大教堂還是跳蚤市集？</a></li><li><a href="https://matters.news/@mashbean/273893-nft%E6%95%85%E4%BA%8B%E9%9B%86-%E6%BC%94%E7%AE%97%E8%97%9D%E8%A1%93%E5%8F%AF%E4%BB%A5%E5%85%B1%E5%89%B5%E5%97%8E-bafyreid6qvbaic4zy3kr4m4owk6u7cdusont7hcnrq377ctnwommehnary" rel="noopener noreferrer" target="_blank">NFT故事集｜演算藝術可以共創嗎？</a></li></ol><hr><p>一個生成藝術平台要成立有三個要件，分別為<strong>儲存系統、智能合約、前端網頁</strong>。</p><p>以 Artblocks 為例，他的儲存是全上鏈的，所以藝術家在發表作品時，必須要付高昂的上鏈手續費，將所有資料存在以太坊的區塊。好處是只要鏈存在，作品便不會消失，壞處是在創作上容易有所掣肘，因函式庫與擴充資源無法補充。</p><p>Artblocks 有自己的合約與網頁，走一個簡潔素雅的介面。但 Artblocks 只做一級市場，也就是發行者的角色，二級市場的販售只能去 OpenSea 這種綜合 NFT 市集買賣。</p><p>而 fxhash 雖然也是生成藝術平台，但是在要件上與 Artblocks 差異很大。首先，其資料儲存在 IPFS 這種點對點儲存系統上的，優點是可以存放更大量更多元的資料，對創作者有更高的自由度，缺點是分散式儲存，長期而言有散佚風險。</p><p>此外 fxhash 也有自己的合約與網頁，特別的是， fxhash 不只提供一級市場的發行（launch），也有自己的二級市場，此外還提供豐富的交易數據，如地板價、交易量等。此外 fxhash 的合約也有接入其他二級市場平台，如 Tezos 鏈最大的市集 OBJKT，因此交易過程是可以在多地多中心發生的。</p><p>fxhash 因為開放特性，且日益繁盛的交易量，誕生了許多衛星組織。今天來討論一個成立僅兩個月的的複合型團體 fxTENDER。</p><figure class="image"><img src="https://assets.matters.news/embed/f1dfd82b-41c7-4bfa-bfd6-f65e97c7deb6.png" data-asset-id="f1dfd82b-41c7-4bfa-bfd6-f65e97c7deb6" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><p>fxhash 的 UI 有一種很特別的...風格，看起來像是一種開發人員使用的平台。但可能也因此契合了「人人都可以發表作品」的精神，這與審核制的 Artblocks 完全不一樣。或許 Tezos 的早期 NFT 平台都走這種路線吧，最早的 HicEtNunc 平台也是看起來要斷不斷。</p><p>Tezos 生成藝術世界，需要一枚白盒子，因此 fxTEDNER 誕生了。</p><figure class="image"><img src="https://assets.matters.news/embed/8f46bbc4-5aa6-4be9-bbde-83e71d732a01.png" data-asset-id="8f46bbc4-5aa6-4be9-bbde-83e71d732a01" referrerpolicy="no-referrer"><figcaption><span>以 The Price of Oil 為例，上為 fxhash 頁面，下為 fxtender 頁面</span></figcaption></figure><p>fxTENDER 使用 fxhash 的 API 與合約，重新打造了前端 UI，讓一樣的作品有了全新的體驗。我一開始會被吸引也是因為其優美簡潔的頁面。</p><p>來比較一下兩個平台的石油，看看會不會勾起購物慾：</p><p>The Price of Oil by <a href="https://fxtender.art/u/fauxjebus" rel="noopener noreferrer" target="_blank">fauxjebus</a></p><ul><li>fxhash: <a href="https://www.fxhash.xyz/generative/12766" rel="noopener noreferrer" target="_blank">https://www.fxhash.xyz/generative/12766</a></li><li>fxTENDER: <a href="https://fxtender.art/project/the-price-of-oil" rel="noopener noreferrer" target="_blank">https://fxtender.art/project/the-price-of-oil</a></li></ul><p>但 TENDER 不只是換了皮而已，他們重新用既有作品打造了自己的選品，甚至與創作者合作，開發出 TENDER X 系列新作。</p><figure class="image"><img src="https://assets.matters.news/embed/e7c477e7-7d30-497d-922d-e3dd55b66428.png" data-asset-id="e7c477e7-7d30-497d-922d-e3dd55b66428" referrerpolicy="no-referrer"><figcaption><span>pang #363 via fxTENDER X collab</span></figcaption></figure><p>當你逛 fxTENDER 網頁，並不會看到 fxhash 所有的上架作品，他們有進行選擇。彷彿 Artblocks 逆行行駛，Artblocks 是從一開始便進行發行篩選；而 TENDER 是在 fxhash 平台發行後，挑選出屬於自己語境的精品來展示。</p><hr><p>順著 UI 的按鈕引導，點開 icons 頁面，可以看到頂級優異的作品，熱門的作品甚至有評論者在上頭寫文章推薦；點開 mintable，可以看到在 fxhash 尚未販售完畢的作品，當然是由 TENDER 推薦挑選過的。</p><p>此時一定要分享，漂亮的 UI 真的會讓人一時想不清楚，在看完高價物件的 icons 區，一看到 mintables 的價錢如此便宜（0.7 tez，約台幣 50 元），我便失手買了一桶極簡石油。</p><p>再來點開 TENDER X，可以看見 fxTENDER 已經與三名藝術家合作，進行聯名發行，而且銷售成果非常好。聯名功能其實是上個月 fxhash 升級進入 1.0 後，才開始的新功能。 fxTENDER 是目前看起來，僅次於官方表現最好的聯名組織。</p><p>其中表現最好的是 TENDER X Nat Sarkissian 的 Reconnaissance （勘查，俗稱行星表面。）我自己很喜歡 Nat 的舊作，有收藏 California Hills in Late Sun 與 Sierras，都是地景系列，前者是模仿加州午後的山坡攝影，後者是插畫風的俯視地圖。但兩者表現都沒有 Reconnaissance 來得好，fxTENDER 不知道使了什麼魔法，讓 Nat 作品的完成度、變化性，還有銷售量大幅提升。其二級市場銷售量是加州午後暖陽的四倍之多。</p><hr><p>此外， fxTENDER 也有自己的官方錢包，進行選品收藏與評論；TENDER 也在近日推出印刷服務，只要你持有特定藝術家的 NFT，藝術家又同意，便可以透過 TENDER 印製實體生成藝術 NFT 給收藏家。</p><figure class="image"><img src="https://assets.matters.news/embed/5fd41239-a9e9-44d4-a8f4-03a8f2cb0846.png" data-asset-id="5fd41239-a9e9-44d4-a8f4-03a8f2cb0846" referrerpolicy="no-referrer"><figcaption><span>fxTENDER official collection</span></figcaption></figure><p><a href="https://fxtender.art/u/T%20E%20N%20D%20E%20R/collection" rel="noopener noreferrer" target="_blank">fxTENDER official collection</a></p><p>fxTENDER 走一個<a href="https://fxtender.art/transparency" rel="noopener noreferrer" target="_blank">完全透明的風格，他們公開了所有成員的錢包地址</a>（與收藏），官方共同擁有的收藏，並積極串連關於生成藝術評論，甚至自己頭洗下去與藝術家共同發行項目。唯一目標就是將生成藝術發揚光大。</p><p>我的猜測是，TENDER 將扮演生態系的重要角色，佔據話語權，進一步影響市場，因為目前生成藝術市場的策展與評論仍顯單薄。此外，或許 fxhash 在不久的將來，進化成為 fx DAO 之後，fxTENDER 能在 subDAO 取得更加關鍵的角色，這直接牽涉到整個平台的收益分潤。</p><p>TENDER 角色多元，所以我稱之為複合型策展團體。TENDER 目前的營運方式是與 fxhash 合作、獲得一些分潤、並從 collab 策劃獲取利益，並將利益回頭用於網站維護與架設。</p><p>web3 世界方便之處在於能將已出現的內容，重新包裝成自己的模式，並且透過共享收益來維持營運。雖然 fxTENDER 才成立兩個月，fxhash 成立半年，我相信整個生成藝術生態系還會繼續往下發展。明年此時再來回頭總結。</p><figure class="image"><img
