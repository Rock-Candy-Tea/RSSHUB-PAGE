<p>自 2020 三月以來，美國疫情嚴重，最近 7月以來，疫情又出現了第二波高增長，「不知是好事還是壞事」。疫情當然是壞事，不過我指的是任何壞的事物都可以樂觀的態度來看的話，還是可以找出其中可以自得其樂的方面，我指的是關於各大公司的 Remote 工作政策方面，很多之前在觀望的公司也開始大範圍擁抱 Remote 工作了，那些準備七月份恢復辦公室工作的公司也不得不推遲計劃了，甚至越來越多公司允許那些喜歡 Remote 工作的人以永久留在家工作的選項了、不強制要求疫情之後必須回辦公室。</p><p>說起 Remote / 遠距工作 / 在家工作，我的起點是 好幾年之前了，讀過 2013 年就出版的這本《<a href="https://amzn.to/2ZC1EwB" target="_blank">Remote: Office Not Required</a>》著作之後，就認定 Remote 才是未來工作的方向，特別適合軟體工作方面的人士，當時工作的公司已經有這個 Remote Friendly 的政策了，是每週允許一天在家工作，並不是每週臨時隨機選擇，而是每人可以從周二、周三、周四中選取固定的一天，以讓團隊成員之間互相了解對方的 Remote 工作習慣，以達成高效的工作。這是一個很好的鍛鍊機會，呆在家裏又要拒絕偷懶的誘惑，保證品質地完成工作任務，以及應付偶爾的 Zoom 會議，每週一天 Remote 是一個很好的鍛鍊自我控制能力的好的起點；在下一次轉換工作時，再有意識搜索 全遠距 / Fully Remote 的工作機會，只是 在 2020 這個疫情之前， 接受 Fully Remote 的公司還真不太多，這也是很多人在 2020 之前「想當然認爲上班一定是在辦公室」</p><p>對於這些在 2020 之前從未考慮、或從未聽聞 Remote 工作過的朋友們： 最近還看到有人表達了「終於可以回 Office 上班」之欣喜； 我想說的是『<strong>不如珍惜且擁抱最後的 在家工作時光吧</strong>』</p><p>自上一家 Remote Firendly 之後、我最終還是找到了並且在幾年之前就開始了這樣 Fully Remote 工作；關於優點，你從下面我的過去 2019 的全球旅行中就能看到體會；先說說幾項誤解</p><p>1) 誤解之一： Fully Remote 的工作不穩定。我不敢說所有人的 Remote 工作都是穩定的，的確有很多 Remote 職位是小公司爲節省成本發佈的 Part-Time Contractor (兼職合同工) 職位，但是 全職的 正式職位也是存在的，在美國叫做 W2 Employee, 有固定薪水，有醫療保險覆蓋全家人，待遇優厚的也大量存在；我認識的人裏面有在同一公司 Remote 工作很多年了的資深工程師都有。</p><p>2) 誤解之二： 「你想要 Remote 工作統治世界嗎？ 我的工作不能 Remote！」 這是幾年前我對朋友宣傳 Remote 之優點時朋友提出的疑慮。<br class="smart">答案當然不是了。<br class="smart">我在 2020 之前的願望是希望所有軟體方面的工作能有 10% 接受部分時間的 Remote 工作、再能有 1% 接受 Fully Remote 概念就很不錯了。 幾年之前包括 FAANG 這個級別的大公司也只有 Remote Friendly 很少有 Fully Remote 。 當時我同時拿到 FAANG 之一的 Office based Offer 和小型公司的 Fully Remote Offer 我還嘗試跟這家大公司談判一下，能不能給 Work From Home 這項 Benefit 結果 HR 和 Report Manager 都說 我們 Office 的福利更多呀。顯然是談崩了。最終我的選擇是小型公司的 Fully Remote 而不是大公司、給它的拒絕理由是「我個人認爲沒有任何福利比得上 Work From Home」。當然小公司有小公司的侷限性，作爲未雨籌謀的話，也僅僅是希望這個市場上接受 Fully Remote 的公司再多一點。<br class="smart">這次 2020 的疫情算是以意外的方式讓 Remote 工作變成爲大衆所知，但還絕對不是主流；絕對沒有 50% 以上的公司打算永久保留 Fully Remote，現在的它們只是被迫接受 Remote 工作方式而已，在未來疫苗普及之後，它們一定會 Recall to work from Office。<br class="smart"></p><p>另外，從個人成長角度來說，Remote 也不是適合所有人的。過去有人問我建議的話，我個人不對這幾類人建議 fully remote: 1) 自我控制力不夠的 2) 職業入行(Career Life)五年之內的 3) 年齡30以下/閱歷不夠的； 我在過去的公司也有作爲 Mentor 帶過新鮮畢業生入職，雖然公司有 1 Day per Week Remote 的政策，但我還是建議他們儘量不用它，儘量在辦公室，至少在入職前幾個月，在辦公室多找前輩聊天也行，用各種機會多學習。在很多行業包括軟體行業，在年齡30以前、未育小孩以前還是多多進辦公室工作才是更好的歷練方式和職業成長。</p><p><br></p><p>再來說 Remote 工作之優點，對於30以後、已育小孩這幾類人來說，太顯而易見了。在 30 歲以前居住與某大都會區，體驗過大城市的光怪陸離就已經購了；30以後往往已經在職場歷練過了幾年、需要的是更多的靜下心來對人生的規劃和思考，以及更多的親子時光、等等。以及居住環境方面，就美國來說，還是裏城區遠一點，在市郊(Urban)、或往衛星城去置業，很明顯能以同等的價錢獲得更有C/P值的居住環境。</p><p>關於爲什麼選 Fully Remote 而不僅僅是 類似每週一天這樣的 Remote Friendly 還有一層優勢是你可以搬到外州去，除了生活成本方面節省之外，還能在 Income Tax (個人所得稅) 稅制上再節省。比如，在美國的加州、紐約州都是工作機會多也個稅高的地方，同時全美國有七個州是免州一級的個人所得稅，比如加州相鄰的內華達州、還有經濟第二強的德克薩斯州都是其中之一。舉例一個在加州收入10萬美元的例子來說，大約需繳聯邦個稅兩萬多、加州個稅一萬多，實際到手只有六萬出頭；如果得到 Fully Remote 工作可以搬到臨近的內華達州(比如 Las Vegas)居住，就可以擺脫加州稅務局的控制；在稅制上的節省(一萬多美元)才是大頭。</p><p>其實對於公司而言也是類似的，在英文的媒體上我已看到相關的討論，加州州稅局不僅對個人征 Income Tax 高、它對公司也徵收相當高比例的企業稅、還有各種全美最嚴的企業合規成本(比如最嚴格的環保、等等法規導致的)，在過去是因爲北加州舊金山、硅谷的科技業是 以斯坦福、伯克利 的人才爲中心，南加州以好萊塢、迪士尼的娛樂產業爲經濟龍頭；在疫情之下，人在哪兒不再重要，長此以往，很多公司也正搬離加州。很多公司不僅僅是在考慮而是已經在實行中，搬到企業稅方面更友好、監管也更少的德克薩斯州是一個趨勢。如果疫情持續一兩年以上的話， Remote 工作成爲一個大衆潮流、企業搬遷也會成爲一個潮流、大學改成在線上課，那麼多的硅谷公司真的有必要繼續保持在硅谷的昂貴的實體存在嗎？ 未來會發生什麼真是不可預知的。</p><p><br></p><p><br></p><p>下面是之前隱藏的一篇重新編輯發表，比較適合這個話題，看看不如擁抱 remote 工作政策，</p><hr><p><br></p><h1>說說過去一年(2019)的旅行，可能未來兩三年都不會有</h1><p><br></p><p>三月份的「疫情之後，最想去的旅行」話題，提到要在10月底之前成行，我當時的感覺就是十月底能恢復嗎？是不是太樂觀了。最近有航空業相關者接受採訪時提到「航空業要回到2019的水平」最樂觀的估計也要 2024 年，美國現在的航線活躍度比起去年同時期減少了
