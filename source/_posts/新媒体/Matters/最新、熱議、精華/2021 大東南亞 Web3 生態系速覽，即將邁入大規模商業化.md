傳教士」。</blockquote><p>自從 2009 年 AppWorks 創立、2010 年啟動每年兩屆的 <a href="https://appworks.tw/accelerator/" rel="noopener noreferrer" target="_blank"><u>AppWorks Accelerator</u></a> 以來，我們便致力在大東南亞地區 (東協 + 台灣) 協助下世代的創業者，抓住數位革命的成長機會，嚴選本區域最具潛力的新創團隊進駐，輔導新創團隊尋找 Product-Market Fit、幫助成長期團隊建立 Sustainable / Scalable Business Models。建構在加速器的基礎之上，AppWorks 進一步打造新創社群、成為創投機構。</p><p>在 2018 年之後，區塊鏈、Web3 成為我們重點經營的領域，因為我們相信區塊鏈、Web3 帶來的巨大典範轉移，將為下世代的創業者，開啟大量的顛覆與創新機會。</p><p>2021 年，區塊鏈進入最高速成長的一年。在 DeFi、NFT、Gaming、Metaverse 等豐沛的創新應用下，吸引大量新資金、新用戶投入，區塊鏈的創業者們，正逐步建立起 Web3 各項重要的基礎建設，並即將邁入大規模商業化。</p><p>「在去中心化的 Web3 世界中，是否會呈現出不同的區域發展特色？」一直是多年來，我們持續思辨的問題。在 2021 年，Web3 獲得大幅加速發展後，這個問題的輪廓逐漸顯現，我們發現，在大東南亞的主要市場中，來自長期截然不同的歷史、經濟、產業、社會、文化累積發展之下，正浮現出多元化的特色，驅動整個大東南亞的 Web3 生態系，展現出更精彩、更深化、更讓人期待的風貌。以下是我們對各主要市場在 2021 年的觀察與分析： </p><h2>印尼：投資加密貨幣，比投資股票更具吸引力</h2><figure class="image"><img src="https://assets.matters.news/embed/245a384c-ceb2-4f1d-b7b2-343e9248e57f.png" data-asset-id="245a384c-ceb2-4f1d-b7b2-343e9248e57f" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>人口規模全球第四、大東南亞第一的印尼，估計仍有<a href="https://www.thejakartapost.com/life/2021/04/15/grassroot-strategy-to-realize-financial-inclusion-in-indonesia.html" rel="noopener noreferrer" target="_blank"><u> 66% 的人口，處於沒有銀行帳戶 (Unbanked) 的狀態</u></a>，但也正因為如此，為印尼的 Web3 與區塊鏈發展，提供了絕佳的發展契機，印尼正進入由區塊鏈、加密貨幣推動的金融典範轉移，整體發展呈現出指數型、蛙跳式 (Leapfrogging) 成長的態勢。根據印尼貿易部 (Ministry of Trade) <a href="https://vnexplorer.net/over-6-5-million-indonesians-bought-or-sold-crypto-in-first-5-months-of-2021-ministry-of-trade-says-a2021214858.html" rel="noopener noreferrer" target="_blank"><u>統計</u></a>，在 2021 年前五個月，印尼共有超過 650 萬人曾交易過加密貨幣，遠超過交易股票的 220 萬人，加密貨幣交易量從 2020 年的 44 億美元，成長至 2021 年前五個月的 250 億美元，成長率達 470%，加密貨幣已成為最受印尼民眾歡迎的投資類別。</p><p>相較於其他國家，印尼政府對於加密貨幣也相對開放。自 2018 年九月以來，陸續<a href="https://coinmarketcap.com/alexandria/article/cryptocurrency-in-indonesia" rel="noopener noreferrer" target="_blank"><u>制定了一系列相關法律</u></a>，皆象徵印尼政府的開放態度。包括加密貨幣、資產在印尼具有合法地位，可被視為商品 (Commodity) 來交易，由商品期貨交易監管機構 (The Futures Exchange Supervisory Board，簡稱 BAPPEBTI ) 作為主責的監管單位，交易所需取得合法執照，可合法交易共 229 種加密資產。</p><p>種種獨特的條件，讓印尼在交易所、錢包、資產管理以及其他 DeFi 領域上，發展快速。例如，2014 年成立的 <a href="https://indodax.com/" rel="noopener noreferrer" target="_blank"><u>Indodax</u></a>，是印尼位居領先的交易所，用戶規模超過 470 萬。此外，隨著一般民眾對投資加密資產的接受度越來越高，像 <a href="https://usenobi.com/" rel="noopener noreferrer" target="_blank"><u>NOBI</u></a> 這類提供資產管理服務的新創，將是下一波值得留意的領域。</p><h2>菲律賓：大量 End Users 推動產業成長</h2><figure class="image"><img src="https://assets.matters.news/embed/47094125-2c3e-49d1-8ea0-ca07d7d52dfe.png" data-asset-id="47094125-2c3e-49d1-8ea0-ca07d7d52dfe" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>2021 年，<a href="https://yieldguild.io/" rel="noopener noreferrer" target="_blank"><u>Yield Guild Games</u></a> 無疑是菲律賓最受矚目的 Web3 新創之一。從最初協助 Axie Infinity 招募、培訓玩家，累積到現在整個菲律賓有超過 1,500 個以上的 NFT 遊戲公會，共同進行數位資產管理，或為玩家提供遊戲獎學金等，Yield Guild Games 一路成為全球規模最大的 NFT 遊戲公會。蓬勃發展的產業生態，根據 Playercounter <a href="https://playercounter.com/axie-infinity/" rel="noopener noreferrer" target="_blank"><u>統計</u></a>，目前有超過四成的 Axie Infinity 玩家，來自菲律賓，是最大的單一國家，Play to Erarn 能為玩家帶來實質收入，有效地緩解了 COVID-19 帶來的經濟與社會問題。</p><p>這種由大量新用戶加入，推動整體生態系成長的模式，是菲律賓發展 Web3 的特色。可以預期，在資金與用戶都蓬勃發展的帶動下，將會有更多不同領域的創業者，在這樣的基礎上推動各種創新應用，不論在 NFT 或是遊戲領域，都可看到更多元的成長潛力，例如，在今年推出菲律賓第一個 NFT Marketplace 的 <a href="https://vention.app/home/" rel="noopener noreferrer" target="_blank"><u>Vention</u></a>，或是建立遊戲公會管理與數據平台的 <a href="https://vention.app/home/" rel="noopener noreferrer" target="_blank"><u>BlockchainSpace</u></a>。</p><h2>新加坡：延續亞洲金融中心的優勢</h2><figure class="image"><img src="https://assets.matters.news/embed/71817f30-6a3d-473f-bb7a-9664061b7100.png" data-asset-id="71817f30-6a3d-473f-bb7a-9664061b7100" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>向來有「亞洲金融中心」美譽的新加坡，長年以來，藉由政府開明的政策引導，逐步建立起區域內的地位。在 Web3 發展上，也可看到類似的發展軌跡，新加坡金融管理局 (MAS) 就是主導發展的單位，長期以來，對區塊鏈以及加密貨幣的發展，擁有相對開放與友善的立場。</p><p>早在 2017 年，MAS 就發布了<a href="https://www.mas.gov.sg/~/media/MAS/Regulations%20and%20Financial%20Stability/Regulations%20Guidance%20and%20Licensing/Securities%20Futures%20and%20Fund%20Management/Regulations%20Guidance%20and%20Licensing/Guidelines/A%20Guide%20to%20Digital%20Token%20Offerings%20%2014%20Nov%202017.pdf" rel="noopener noreferrer" target="_blank"><u>「數位代幣發行指南」(A Guide to Digital Token Offering)</u></a>，確立了新加坡政府對於 ICO 的基本立場與政策，對於新創透過 ICO 募資的發行者以及相關平台，確立適用「證券暨期貨法」(Securities and Futures Act，簡稱 SFA) 監管。到了 2021 年 1 月，「<a href="https://www.mas.gov.sg/news/speeches/2021/payment-services-amendment-bill" rel="noopener noreferrer" target="_blank"><u>支付服務法案」(Payment Service Act，簡稱 PSA)</u></a> 相應推出，監管新加坡境內的所有加密貨幣、交易所以及數位支付等業務。</p><p>也因此，我們可看到各種與金融相關的 Web3 應用，正出現代表性的新創。例如，為 Web3 應用構建交易基礎建設、優化跨鏈交易體驗的 <a href="https://www.biconomy.io/" rel="noopener noreferrer" target="_blank"><u>Biconomy</u></a>；開發金融電子支付的 <a href="https://www.xfers.com/" rel="noopener noreferrer" target="_blank"><u>Xfers</u></a>；提供加密貨幣交易、借貸、錢包等服務的 <a href="https://www.vauld.com/" rel="noopener noreferrer" target="_blank"><u>Vauld</u></a>。未來在加密貨幣相關的資產證券化、證照合規、國際匯兌業務上，預期將可看到更多重要的進展。</p><h2>台灣：龐大的數位市場、IT 工程師推動整體發展</h2><figure class="image"><img src="https://assets.matters.news/embed/0bc5a26f-1ce6-4a65-904a-bc2876c4e7e4.png" data-asset-id="0bc5a26f-1ce6-4a65-904a-bc2876c4e7e4" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>在 Web3 生態系的發展上，相較於區域內的其他市場，台灣有兩大特色與優勢。第一，龐大的數位遊戲與娛樂市場。根據 App Annie <a href="https://www.appannie.com/en/go/state-of-mobile-2021/" rel="noopener noreferrer" target="_blank"><u>統計</u></a>，2020 年，台灣民眾在 App Store 總共消費 24.4 億美元，約相當於印尼、泰國、越南、菲律賓、馬來西亞五國總和的 24.6 億美元；第二個，則是台灣長久在科技製造、軟硬體整合上，累積而來的 IT 工程師文化，優秀的軟體工程師，一直是台灣推動區塊鏈發展的核心。</p><p>在 2021 年，台灣也搭上全球的 NFT 熱潮，進入早期發展階段。在龐大的數位遊戲與娛樂市場基礎上，越來越多的台灣區塊鏈新創，開始以 NFT 打造產品、服務以及商業模式，而歌手、藝術家、遊戲商、文創商品平台，以及多位 YouTuber，也先後開始推廣 NFT，使得 NFT 在這一年快速獲取大眾的關注，未來極具發展潛力。</p><p>自 Ethereum 問世開始，就不斷有台灣本地的優秀工程師，展開各種在區塊鏈領域的探索，成為驅動產業推進的關鍵力量。2021 年，已有新創在全球區塊鏈產業中嶄露頭角。例如，團隊擁有超過 10 年、多次創業經驗的 <a href="https://www.perp.fi/" rel="noopener noreferrer" target="_blank"><u>Perpetual Protocol</u></a>，提供虛擬做市商 (vAMM) 實現永續合約解決方案；開發智能合約錢包的 <a href="https://blocto.portto.io/zh/" rel="noopener noreferrer" target="_blank"><u>Blocto</u></a>，隨著 Flow 躍居一線公鏈而崛起，<a href="https://flowscan.org/staking/overview" rel="noopener noreferrer" target="_blank"><u>超過 80% Flow 鏈上的用戶</u></a>，都使用 Blocto 進行 FLOW Token 質押，在跨鏈應用上，也已支援包括 Ethereum、BSC、Solana、Avalanche (c-chain)、Polygon 等一線公鏈。</p><h2>泰國：後發先至的蛙跳式成長</h2><figure class="image"><img src="https://assets.matters.news/embed/878e4ce8-6f21-49b6-85bf-1136c3aa82e8.png" data-asset-id="878e4ce8-6f21-49b6-85bf-1136c3aa82e8" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>雖然整體產業仍在發展早期，但 2021 年泰國整體 Web3 發展，可用蛙跳式成長 (Leapfrogging) 來形容。6 月，<a href="https://www.bangkokpost.com/business/2124699/flash-group-becomes-first-thai-unicorn" rel="noopener noreferrer" target="_blank"><u>完成 E 輪融資的電商物流供應商 Flash Express</u></a>，才正式成為泰國第一隻獨角獸，在 11 月，泰國第一隻 Web3 獨角獸，緊接著誕生，<a href="https://www.thaiexaminer.com/thai-news-foreigners/2021/11/06/bitkub-thailand-first-unicorn-success/" rel="noopener noreferrer" target="_blank"><u>交易所 Bitkub 將 51% 的股份，以 5.35 億美元出售給擁有百年歷史、泰國第一家本土銀行 Siam Commercial Bank</u></a>，等於在 5 個月的極短時間內，泰國連續跨越 Inertnet 與 Web3 時代的里程碑。</p><p>在 Bitkub 這樣的指標案例出現後，泰國 Web3 發展的後勢，值得讓人期待。包括 Bitkub Mafia (前員工)、Web 2.0 時代的成功創業者，都將陸續加入 Web3 的創業浪潮中，一如其他國家正在上演的故事。</p><p>事實上，已經可以看到更成熟、更進階的 Web3 應用出現。例如，開發跨鏈資料預言機 (Oracle) 的 <a href="https://bandprotocol.com/" rel="noopener noreferrer" target="_blank"><u>Band Protocol</u></a>，以及 Metaverse 遊戲公會 <a href="https://guildfi.com/" rel="noopener noreferrer" target="_blank"><u>GuildFi</u></a>，都是在既有產業生態系上的基礎上，繼續創新與整合。此外，也出現了像 <a href="https://knowhere.art/" rel="noopener noreferrer" target="_blank"><u>Knowhere.art</u></a> 這樣的 NFT Marketplace，相信不久後，將會在 NFT 領域看到更突破性的進展，因為長年以來，曼谷就是亞洲時尚與設計之都，擁有發展 NFT 創作者經濟的絕佳土壤。</p><h2>越南：Axie Infinity 帶來的巨大效應</h2><figure class="image"><img src="https://assets.matters.news/embed/8fc755bc-b556-4943-9234-268341991076.png" data-asset-id="8fc755bc-b556-4943-9234-268341991076" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>提到越南的 Web3 產業，知名度最高的，莫過於 NFT 遊戲 <a href="https://axieinfinity.com/" rel="noopener noreferrer" target="_blank"><u>Axie Infinity</u></a> 開發商 <a href="https://www.skymavis.com/" rel="noopener noreferrer" target="_blank"><u>Sky Mavis</u></a>。根據 CryptoSlam <a href="https://cryptoslam.io/axie-infinity/sales/summary" rel="noopener noreferrer" target="_blank"><u>統計</u></a>，自 2018 年上線以來，Axie Infinity 上的 NFT 交易總金額超過 37 億美元，<a href="https://thenewscrypto.com/axie-infinitys-daily-active-users-increased-by-almost-5000-in-six-months/" rel="noopener noreferrer" target="_blank"><u>日活躍用戶超過 200 萬</u></a>，並在近期完成 1.52 億美元的 B 輪融資。</p><p>Sky Mavis 寫下的成就，對越南的影響，可從兩個面向觀察。第一，帶動越南本地的 NFT 遊戲生態系成長，並吸引國際資金投資更多新創，進而成為生態系更加成長的能量，例如，<a href="https://sipher.xyz/" rel="noopener noreferrer" target="_blank"><u>Sipher</u></a> 在 10 月，就從美國、韓國投資人手中獲得 <a href="https://venturebeat.com/2021/10/26/sipher-raises-6-8m-for-pc-based-blockchain-game-world-of-sipheria/" rel="noopener noreferrer" target="_blank"><u>680 萬美元的種子輪融資</u></a>。另外像 <a href="https://mydefipet.com/" rel="noopener noreferrer" target="_blank"><u>My DeFi Pet</u></a>、<a href="https://thetanarena.com/" rel="noopener noreferrer" target="_blank"><u>Thetan Arena</u></a> 等，也都是值得留意的 NFT 遊戲新創。</p><p>第二個，則是越南民眾對於區塊鏈以及加密貨幣的高接受度。根據 <a href="https://blog.chainalysis.com/reports/2021-global-crypto-adoption-index/" rel="noopener noreferrer" target="_blank"><u>Chainalysis 2021 年「全球加密貨幣接受度指數」(Global Crypto Adoption Index)</u></a> 的調查，越南排名高居第一，在 On-chain value received、On-chain retail value received、P2P exchange trade volume 三大指標中，皆位居全球前五名，可以預期，各種 DeFi、錢包、資產管理的創新模式，例如提供 DeFi 流動性中心 (Liquidity Hub) 的 <a href="https://kyber.network/" rel="noopener noreferrer" target="_blank"><u>Kyber Network</u></a> 等新創，將會是下一波值得觀察的重點。 </p><h2>結論：</h2><p>回顧 2021 年，Web3 生態系在大東亞獲得全面性地推展。印尼在投資加密貨幣的開戶人數，超越股票而躍居投資主流；菲律賓則正由 End User 帶動生態系成長；新加坡持續將「亞洲金融中心」的優勢，延續至 Web3 領域；台灣正由優秀的軟體工程師推動創新應用；泰國在出現第一隻 Web3 獨角獸後，預期將吸引更多優秀創業者投入這塊領域；越南則有讓人眼睛一亮的 NFT
