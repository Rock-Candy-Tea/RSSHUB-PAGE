分鐘。</p><hr><p>寫作的人常焦慮，除非是家產豐厚、已不愁收入來源的寫作者，不然多的是我們會焦慮的事。</p><p>既要擔心自己的創作，是否有持續精進、還是江郎才盡，也要擔心後起之秀的追趕與新鮮感，畢竟市場總是新的好。</p><p>另一方面還要擔心閱讀者是不是不再閱讀，而是都成為閱聽跟觀影者了。</p><p>當大家說人們的耐心越來越少，一個標題從 7 秒變 3 秒理解，然後再到 2.2 秒時，我們對於下標的追求也跟著改變。</p><p>當連 10 分鐘影片都面臨 30 秒短影片的挑戰時，我們 2000 字已經被說是長文，連 600 字都會被嫌棄，最後只能在 200 字中勉強過關。</p><p>儘管這幾年來，人們閱讀的文字量是變多的，每天打開各種平台，全都是文字去傳遞訊息，也包括我們人與人之間的溝通，文字表達比重變高不少。</p><p>因著碎片化的關係，人們留在同一個主題上的時間卻是變少的，因此文字的重要性，是變得日常而幽微，彷彿很重要，但卻不是重點。</p><p>因此<strong>大家都說有價值，卻總是抬不起價格。</strong></p><hr><p>搜尋+社群時代之後，寫作者迎來一波紅利，透過關鍵字與社群表達，我們得到了目光與聲量，然後很快的，影音表達時代來臨，我們又開始焦慮了。</p><p>且在疫情開始之後，有越來越居家工作者，也開始練習寫作技能，也開始用不同的價格嘗試切入市場，我們時不時看到有很低價的接案需求，讓我們的焦慮變得更嚴重。</p><p>對他們來說是兼職，但這卻是我們的本業；一篇文 500 塊，對他們來說是外快，對我們來說可是餐費。</p><p>現在，Web 3.0 來了，許多寫作者不只是焦慮，甚至是恐慌，想著自己是否該拋下本業，專心觀察幣圈與各種 NFT 的創作項目，或許更有發展可能，也有生存與利潤空間。</p><p>那我們身上的寫作技能，是不是在新一波浪潮下，就無用武之地呢？</p><p>前言說的有些長，等等還會有些脈絡跟推論，因此這裡先說出結論。</p><blockquote>我認為的發展是：文字會成為基礎建設，寫手會消失，寫作者與故事會成為星辰光芒，稀有而珍貴。</blockquote><p>以下進入主題，我將分享我的觀察和判斷，不是任何預言跟建議，我是屬於<strong>痛苦的樂觀者－</strong>「<strong>相信彼岸有光，但眼前是黑暗，我們必須很辛苦的走過去。</strong>」</p><p>正文開始：</p><hr><p>先從我賣自己的 NFT 開始說起，這是一個沒有完全成功，但卻十分具啟發性的一次專案。</p><h2>壹、人們參與 Web 3.0 與購買 NFT的動機。</h2><p>負能量 NFT 專案內容，為我將負能量過往的語錄，整理出四種形式，包含熱門排行榜、最初爆紅篇、精選 99 語錄與全集合型。<a href="https://opensea.io/collection/neenergy" rel="noopener noreferrer" target="_blank">可點此前往 OPENSEA 觀看</a></p><p>由於當時我在規劃時，剛好師園鹹酥雞發售，一時間新聞與社群都在討論此事，為避免熱潮退去，因此我只好提早準備，預先上架。</p><p>其中包括手動上傳、撰寫白皮書、開設 Discord、邀請朋友宣傳、撰寫發表宣傳內容……共花差不多 7 天的時間，當然不包含我事前對此領域的研究與投入。</p><p>截至 2021年底，總共賣出 42 個，總收入 5.1 以太幣，約台幣 50-60 萬。</p><figure class="image"><img src="https://assets.matters.news/embed/cfc107f0-266c-4800-a15f-f45b1da23819.png" data-asset-id="cfc107f0-266c-4800-a15f-f45b1da23819" referrerpolicy="no-referrer"><figcaption><span>負能量 NFT</span></figcaption></figure><p>此次專案是少數的文字型 NFT 收藏品，雖然還是以圖片方式表現，但其價值是承載於文字上，而非常見的圖像設計、生成式藝術或應用設計等。</p><p>這之後我去瞭解持有者的購買原因，可以分成三種：</p><ol><li>喜歡當初的負能量，為了情懷和語錄內容，購買收藏。</li><li>相信之後的作品分潤與空投，我在白皮書上表示會有此規劃。</li><li>相信會漲有套利空間。(先跟這些人說聲抱歉，我會持續發展的)</li></ol><p>當然可能更多原因我不知道，但在這專案之後，給了我許多啟發，並讓我更有興趣去研究其他 NFT 社群的狀況。</p><p>人們為什麼會購買一個可能早已經看過內容的 NFT？即使後方沒有其他技術應用的支持。</p><p>在經過持續追蹤與觀察後，我總結出目前在大多數 Web 3.0 與 NFT 市場上，人們的參與和購買動機，大致可以分為這三種組成：</p><ul><li><strong>情感派：</strong>喜歡該項目（或稱專案，並非單指 NFT）主題、內容、社群或發行人物。</li><li><strong>技術派：</strong>相信之後的應用空間和發展。</li><li><strong>投資派：</strong>做為投資應用，追求套利空間。</li></ul><p>值得注意的是，不論動機的比例分配，<strong>只要其中一種動機的人數夠多，都會讓該項目受到市場關注，進一步提升價格、讓項目方獲得資源，進而滿足其他兩個動機。</strong></p><p>可是這僅僅是在初期而已，在觀察了眾多項目，發現不同的動機比例，會決定該項目的價格與熱度的波動。</p><blockquote><strong>一、情感派比例高的項目較為穩定。</strong></blockquote><p>由知名人物發行、或是圖像創作可愛或精美（這兩者常不是同一屬性）、又或是社群互動濃烈等項目，其價格通常較為穩定，就算下滑，也不會是雪崩式的跌落。</p><p>其原因在於人們購買時，擁有的那一刻，需求就已經滿足了，例如支持喜歡的創作者、收藏喜愛的圖像、得到參與社群的資格……剩下的都是附加價值。</p><p>這類購買者大多會長期持有，除非需要特別狀況需要變現，否則不會有出售的意願。</p><pre class="ql-syntax" spellcheck="false">例如著名的台灣社群 DemiHuman，總發行 1 萬份，全數售完，由 3100 左右的人持有，其發售時的價格為 0.08 以太，目前地板價(最低入手價格) 為 0.13-0.15 成長波動中。

其 Discord 社群依然是目前 NFT 項目中的榜樣，從規劃到氛圍，都是人們談論到NFT 社群時一定會提起的範例。

我個人也持有一個 NFT，做為社群觀察入場券，基本上除非創辦人跑路或是犯法，又或是跟人吵架還是我快破產了……不然我也不會賣掉，我心中動搖價大概是 10-15 以太左右，真到那時候，我再來考慮要不要打臉自己。
</pre><figure class="image"><img src="https://assets.matters.news/embed/e67de7ae-830b-4c3a-9cb5-5327e964fa61.jpeg" data-asset-id="e67de7ae-830b-4c3a-9cb5-5327e964fa61" referrerpolicy="no-referrer"><figcaption><span>Demi-Human</span></figcaption></figure><p>情感類的項目，<strong>只要創辦人與團隊，有持續在推展規劃、認真創作、與社群交流等，那這項目短期內就不會有波動風險</strong>，且隨著持有者的成長和拓展，還有項目的發展，還會有水漲船高的機會。</p><pre class="ql-syntax" spellcheck="false">另一個著名例子就是 FOMO DOG，總數為 777 ，由 617 人持有，目前地板價為 13.6 以太，經過多次分批發售，最後一次發售採荷蘭式拍賣，由 20 以太往下，最後在 8 以太時全數售完。

當初準備參與最後一波時，心中的價格大概是 4 以太左右，看來還是太天真了......
</pre><figure class="image"><img src="https://assets.matters.news/embed/9a87dd33-6f1b-43e9-91dc-29d40ae8b5f5.jpeg" data-asset-id="9a87dd33-6f1b-43e9-91dc-29d40ae8b5f5" referrerpolicy="no-referrer"><figcaption><span>FOMO DOG</span></figcaption></figure><p>隨著持有者的身份，還有社群的凝聚力，這些項目的價格相對穩定，項目方也有資源去推動後面的計劃。</p><p>即使不是技術應用的技術，光是實體活動、社群資訊交流等，就足夠讓人長期持有，享受社交階級的獨特性。</p><p><br></p><blockquote><strong>二、技術派比例高的項目較需要觀察。</strong></blockquote><p>由於目前在 Web 3.0 的技術，都還在很前沿的階段，在應用層面上，也還尚未到能影響生活行為模式的地步，要出現如電腦、手機般、電動車般的熱潮，還需要蠻長一段時間。</p><p>所以在概念上說得很酷炫厲害、影片拍得很科技未來或令人嚮往的，大多在初期會得到不錯的成果，或是投資方的親睞。</p><p>可是到底後續是否能成真，或普及到大眾目光眼裡，這之中充滿不確定性。</p><p><strong>技術的開發本就是一堆失敗的堆疊，才有極少數的成功，而我們都不知道，誰會成為那一個成功者。</strong></p><p>這部份我們翹首以待。但也很可惜的要說，這與我們寫作者能使上力的地方較少，這正是我們感到焦慮的地方。</p><p>當科技不斷發展，卻很少有文組夥伴能參與的空間時，那一種心情是很複雜的<strong>。</strong></p><p>但別擔心，後面就會說到我們的機會了。</p><p><br></p><blockquote><strong>三、投資派比例高的項目波動風險高。</strong></blockquote><p>第三種動機是大家最常見，也通常是比例最多的人群，在相關的討論群裡，總是最熱烈交換意見與交換收入的一群。</p><p>因此這部份我就不特別多說了，畢竟投資這東西人人都可以講一套，<strong>就先祝看到這篇文又看到這裡的人，今年就是你們暴富的一年。</strong></p><p>看完這三種動機，要先瞭解到，每一個人購買NFT與參與WEB 3.0，不一定只是單純一個動機而已，有可能是多種混雜，只是比例不同。</p><pre class="ql-syntax" spellcheck="false">例如人們常說 NFT 最少要買兩個，一個傳家、一個暴富。

這種說法就是針對你真的很看好的項目，該做出的行動。
</pre><p>說到這裡，我們寫作者可以在這樣的領域中，擁有什麼樣的機會呢？</p><p>答案就是，切入第一種動機，<strong>創造情感</strong>。</p><p><br></p><h2>貳、寫作者的優勢與能力，創造情感與故事。</h2><p>做手機的人不喜歡說故事，他們喜歡說規格、技術與效能，但賈伯斯喜歡說故事，我們至今還是很懷念那些驚嘆連連的比喻。</p><p>電動車的省電與環保不是人們關心的故事，即使到現在許多人依然不懂電動車的技術原理，但我們相信馬斯克的願景與想像，而其他的知名車廠即使技術已經非常優秀，但他們忘記了重拾說故事的本領，只認為要靠技術追趕。</p><p><strong>品牌是無形價值，而其價值是由故事搭建出來的想像體</strong>，差別在於，上個傳播世代，是廣告商透過電視等單向媒體訴說故事；而這個世代是消費者透過社群參與，雙向方享彼此的故事。</p><p>比特幣做為目前虛擬貨幣的大哥，靠的不是其後續有多少應用和高深的技術，恰恰是因為中本聰說了一個充滿想像空間的美好故事－<strong>關於去中心化的模型</strong>。</p><p>推動人們往前進的，往往不是更便利的工具、更方便的生活、更有效率的合作方式等，而是我們對於未來有共同的想像。</p><p>寫作者是創造情感的人，我們在一篇篇內容中，去調動閱讀者的情緒起伏。</p><p>文案師用精妙語句製造多巴胺，作家用豐富小說生產腦內啡，我們能決定一個人今天的心情是笑還是哭，也創造了一個個宇宙時空，讓他們短暫脫離現實與憂愁。</p><p>這是寫作者的能力，也是我們所擅長的事。</p><p><strong>圖像讓人看到美為何物，文字讓人想像美為何意。</strong></p><p>當然，我們寫作並不是單獨存在，這些文字會成為基礎，會是劇本，被畫成圖像或是拍成影片，讓想像落實，讓故事被詮釋。</p><p>在 WEB 3.0 的空間中，這些文字會化作定義，成為人們新世界認知的一切。</p><p>目前所傳播的，大多是暴富的故事，除了過於扁平之外，也缺少延伸價值和共鳴。</p><p>而多談點錢之外的事，就正是寫作者擅長且有熱情的事，我們更喜歡多談點人與人之間的互動，談未來人際的相處，還有那些情愛糾葛。</p><p>這是機會，也是挑戰。</p><p>不論哪一種，那一定會是我們可以發揮的舞台。</p><p><br></p><h2>參、寫作者將會面臨的挑戰。</h2><p>看起來好像很棒，但寫故事這件事，真的有這麼特別嗎？當然不是。</p><p>社群時代也有許多說故事的人，有些獲得聲量，有些則依然無法靠此生活。</p><p>Web 3.0 會把這件事拉得更開，差距不斷擴大，且沒有回頭的可能。</p><p>我們必須要認清自己所寫的東西，是落在哪個層次，才有可能找到出路。</p><p>要說挑戰，關於寫作者的挑戰從來都不會少，為避免過於發散，因此這邊就透過我們 2021 年初的文案思維金字塔，做為理解的基礎，來探討會遇到的機會與挑戰。<a src="https://assets.matters.news/embed/6da2bf95-b445-4abd-847b-62fe01c8dad5.jpeg" data-asset-id="6da2bf95-b445-4abd-847b-62fe01c8dad5" referrerpolicy="no-referrer"><figcaption><span>文案思維金字塔圖</span></figcaption></figure><blockquote><strong>第 0 層：我有沒有其他字可以使用。</strong></blockquote><p>還在煩惱這一層的人，基本上在社群平台上，已經被關鍵字的思維取代，而不再是表面上的文字技巧和詞彙，過於匠氣的修辭技巧或炫技的標語，也越來越少出現在人們的視野中被記住。</p><p>想想你近一年來，有記得什麼品牌標語嗎？連「今晚我想來點」都已是2020年的事了。</p><p>取而代之的都是人們流傳的迷因，或是與社會事件連動的語句。</p><p>因此這部份只能做為寫作者的自我修練，而不能成為創造價值的關鍵。</p><p>WEB 3.0 之後，人們也不再在乎你那些優美文字或巧思，作文時代的標準將再一次被削除，成為緬懷的古文之一。</p><p>我將再一次建議，<strong>所有寫作者都必須具備文案思維，以受眾為基礎，寫出能達成目的的文字內容。</strong></p><p><strong>文案是為商業服務的文字，而不是自我實現的記錄。</strong></p><p>世界不會好奇你內心世界的糾葛，除非你能夠有足夠誘因，讓他們願意走進你所創造的世界裡，而這正是我們後方將會說到的機會。</p><blockquote><strong>第 1 層：我要怎麼寫？</strong></blockquote><p>這部份在 Web 2.0 的時代，已經被各種平台規範給限制，成為了有限的發展空間，從臉書到 IG 的發文形式的設定，再到神秘演算法主導的曝光機制。</p><p>寫作者的自由想像空間，從形式上已經很難找到突破點，只能從內容上去努力。</p><p>這部份大多是寫手的領域，可以創造大量相同形式但不同內容的東西，而較為初級的寫手就會變成內容農場的生產者。</p><p>目前這個需求在 Web 3.0 中消失了，起碼目前還沒有發揮的空間，取而代之的是大量生成化圖像，也可以同理代換成是一種圖像式的農場：形式一樣但內容不同。</p><p>至於是否有價值，就必須回到前一段所說的經營與購買者動機了。</p><blockquote><strong>第 2 層：我要傳達什麼意思？</strong></blockquote><p>到達這一層後，就拉出了 Web2.0 與 Web 3.0之間，寫作者的差距了。</p><p>在 Web 3.0 之後，人們很少去看待你怎麼寫，大多數這件事都被規劃好了，也會是以技術為軸，文字只提供資訊傳達。</p><p>有點類似手遊領域的劇本一樣，你怎麼寫人們不關心，許多人都會跳過劇本或快進，宣傳重點也只會擺在怎麼玩、圖多好看等。</p><p>Web 3.0 中，人們在乎圖長怎樣（但比重在減少）、Roadmap發展如何、技術底層是什麼、應用是怎樣。</p><p>我們能發揮的空間很有限，也可能沒什麼人在乎，所以價值感很低，取代性很高，讓人十分焦慮又無助。</p><p>好了，整體來說，如果你是處在金字塔第0.1.2層的寫作者，將會備受挑戰，也就是人們會越來越不重視你的用字、寫作方式與意思。</p><p><strong>但好消息是，第 3 層以上的需求，將會越來越多，也越來越重要。</strong></p><p>氛圍、理念、體驗，正是 Web 3.0 最缺乏的素材，也是除去技術和可見的圖像後，人們所在意的情感元素。</p><ul><li>如果你是擅長創造氛圍的寫作者，那你將能成為項目裡的創作要角，讓整個項目因你而延續，充滿想像。</li><li>如果你是擅於傳播理念的寫作者，那你能成為號召者，為項目吸引到更多資源，一同邁向目標。</li><li>如果你是樂於建立體驗的寫作者，那你會成為項目社群的火把，人們都將被你吸引，因你而點燃熱情和希望。</li></ul><p>在 Web 3.0 的寫作者，<strong>創造力將會被實踐和放大，而沒有深度的量產型文字則會消失</strong>，人們在元宇宙裡的時間更寶貴、信任更可貴，沒辦法把注意力花在無差異化且沒有延續價值的內容上。</p><p>你是能夠創作的寫作者，還是只會寫作的人，將會有天壤之別。</p><p><br></p><h2>肆、Web 3.0 的故事將會是什麼？</h2><p>最後一段，我將花一些篇幅來說明故事在 Web 3.0 的意義和應用，也做為寫作者可以的努力方向參考。</p><blockquote><strong>1. 故事的核心是人物與情節</strong></blockquote><p>人物不僅是故事的核心，也是人們信任的關鍵，在信任感薄弱的領域，我們只能靠著「因為某個人」來說服自己這可信。</p><p>寫故事必須要塑造人設，一個吸引讀者的人設是充滿生活感卻也有嚮往空間的，不會是完美無缺，也不會是頹廢無賴，能夠寫好人設，就能引起人們的好奇心，並創造入場的條件。</p><p>情節的精彩，將會成為人們是否持續關注你的關鍵，因此你的故事不該是一個段落，一個篇章，而該是一連串的事件與因果所組成的冒險體驗。</p><p>我們很害怕項目方賺一波就跑掉，也很怕一個故事沒頭沒尾甚至是爛尾，就連拖稿、延遲，都會造成追隨者的恐慌與不安，即使一開始故事再精彩，人們耐心終有耗完的一天。</p><p>擅於安排情節的人，就能引人入勝，在 WEB 3.0 的社群裡持續創造話題與延續價值。</p><p>所以不論你是擅於為項目方撰寫人物介紹，又或是為你的項目角色撰寫故事，你能夠發展出足夠的世界觀和情節線安排，那你就真的有機會成為元宇宙裡的金庸、哈利波特、漫威或屬於你的傳奇故事。</p><blockquote><strong>2. 精彩故事靠的是動機與阻礙設計。</strong></blockquote><p>要推動人物前進，需要靠足夠的動機，還有剛好的阻礙。</p><p>有時候你會吐槽一個故事很莫名其妙，大多來自於動機的薄弱或阻礙的不合理。</p><pre class="ql-syntax" spellcheck="false">例如要復仇，結果只是為了小時候對方沒把布丁分給你吃。
例如男女主角要結婚了，阻礙是發現對方是同父異母的兄妹。
</pre><p>在 Web 3.0 的故事依然如此，你給予的動機，不該純粹是套利空間跟暴富而已，還必須有更多面向元素在其中，常見的炫耀感是一種，而身份地位也是一種，親情、愛情、友情，又或是夢想、成就等。</p><p>你必須要能夠創造足夠多元的動機，好滿足各種不同的需求。並在這之中，表達可能遇到的阻礙，如此一來你才能夠驅動大家一起去完成這項任務，或募集更多資源。</p><p>記得你是在寫整個項目社群的故事，這不會太容易，但你是有其他人可以一起參與的，他們都是持有者，也是跟你利益最相關的人。</p><p>WEB 3.0 的讀者，將不再只是觀眾，而將是你故事裡的每一個角色。</p><p>試著多跟他們聊天交流，你會有更多靈感。</p><blockquote><strong>3. 建立有層次的故事，擅用衝突與成長。</strong></blockquote><p>最後讓你的故事能夠脫穎而出的關鍵，或成為經典神作的可能。</p><p>一般故事會有兩層，是屬於外在與內在的改變。</p><pre class="ql-syntax" spellcheck="false">例如表面上是一個戀愛故事，但內在是一個階級反轉的故事。
表面是一個追尋夢想的故事，但內在是一個與家庭合解的故事。
</pre><p>類似這樣的設定，在許多電影裡都會看到，而這也是讓故事有層次深度的做法。</p><p>在 Web 3.0 裡也是一樣，表面上你們可能是運用技術、圖像、社群去追求某樣技術實現，但更深層的是每個人在這裡尋找自己的某種認同。</p><p>而這個就是負責故事的人，必須去形塑跟引導的。</p><pre class="ql-syntax" spellcheck="false">例如表面上是透過技術去解決金融效率緩慢的問題，內在是擺脫中心化掌權的風險。
又或是表面上是買圖像 NFT 的好看與社群資格，內在尋求品味一致的認同感。
</pre><p>項目的故事永遠可以更深入，且充滿多層次，這點只要是寫故事的人就能夠理解這是什麼意思，也是寫故事的創作技術。</p><p>但現在可以進一步透過技術上的支持，讓這些故事層次具像化，並延伸更多想像空間。</p><pre class="ql-syntax" spellcheck="false">例如著名的 NFT 創作者 PAK，其創作的 NFT 都充滿多重涵意與運用層次，這解謎和等待的過程，也是許多粉絲會持續持有，更不斷支持的原因。
</pre><figure class="image"><img src="https://assets.matters.news/embed/11f2378b-f2c9-4883-b44a-bed2b82f9509.jpeg" data-asset-id="11f2378b-f2c9-4883-b44a-bed2b82f9509" referrerpolicy="no-referrer"><figcaption><span>PAK OPENSEA</span></figcaption></figure><p>試著把你的故事與技術連結，有時候不必是新哏或新的意義，只要跟技術連結，有新的表現方式，人們就會樂此不疲。</p><p>關於故事與 WEB 3.0 的結合，還在持續探索與書寫中，但不變的是，人們不管到哪裡，都喜歡聽故事，都享受被故事牽引的過程。</p><p><br></p><h2>伍、結語：</h2><p>這篇文寫到這裡，也有點長了，儘管這些話題還有很多未說完的地方，但先暫時收斂於此。</p><p>最後，我想總結兩個悲觀與三個樂觀的心得。</p><p><strong>悲觀的是：</strong></p><ol><li>寫作不再，再一次的不再只能單純的寫作，你要做的功課更多，要理解的世界更複雜，你才能純粹享受寫作的樂趣。</li><li>會有許多人放棄寫作維生，或發現寫作無法維生，專職的寫作者也會越來越少，人們都需要斜槓或穿越到新世界，才能夠生存。</li></ol><p><strong>樂觀的是：</strong></p><ol><li>寫作的價值能具像化跟回收，透過 NFT 可以賦予你故事靈魂美好的外衣，透過項目目標，可以讓你的終點可見，透過幣價或 NFT 的價格，可以看見你故事的吸引力。</li><li>寫作變得需要深化與持續，我們有新的領域可以展現創作力，不再只是靠出版小說、電影、遊戲等展現價值，現在你還可以發展 NFT、社群等。</li><li><strong>寫作者也有機會暴富了！</strong></li></ol><p><br></p><p>如何？看到這裡的寫作者，是不是覺得蠢蠢欲動，開始想好好研究一下 WEB 3.0 與 NFT 是什麼了呢？</p><p>以及我也希望給目前正在 WEB 3.0 努力，還有經營 NFT 的項目方說，如果你的東西很好，話題卻一直沒有熱度，也不知道該怎麼經營社群跟推廣，那就找一位文案師或作家吧。</p><p>而這個角色，我們並不陌生，但過去卻很少人重視，在遊戲裡也很少人會選他，但現在極其重要。</p><p><strong>我們稱呼他為吟遊詩人。</strong></p><p>那個把遠方的故事，編成好聽的樂曲，讓大人小孩都會能跟著唱的人。</p><p>將會是你 WEB 3.0 發展的最好夥伴。</p><p><br></p><p>林育聖</p><p>20210101</p><hr><p>本文同步發於<a href="https://www.copywriter.com.tw//article/article-example-193.html" rel="noopener noreferrer"
