<p>近來看到有一些亂像，實在不吐不快，所以有此一文。我不是好為人師，也不是喜歡鬧人，但做人做事要取之有道，才可以造就別人，成就自己。故此我打算將我弱弱的領空投心得寫一下，也不是要任何人認同。就正如我這個帳號，一直都只是自說自話當作寫的是學習筆記，叫我好好整理我就是射後不理，也不要做甚麼領航員大家長叫你一定要聽我說之類。總之就是廢話少說，直接進去此文的核心吧，</p><h2>心態：唔好屌 / 不要幹你娘</h2><p>有很多人領空投的心態不知何解就是「全世界欠了你」似的，哇真是好大的官威。每個區塊鏈項目其實都想嘗試解決問題：LikeCoin 的出版自由、Evmos 與以太坊互通、BitSong 的音樂版權粉絲需求、Polygon 的減低以太坊交易手續費…<strong>弄空投目的是想大家了解項目的真意，讓大家有機會擁有一些幣去測試用各個項目，欣賞它的內涵。</strong></p><p>然而時常看到的是：「為什麼你有我沒有？」「為什麼我錯過了？」「為什麼你有那麼多我有那麼少？」「為什麼這個空投這麼小家種？」不如我先問一問大家<strong>你付出過甚麼？</strong>你的 Staking 和 LP 的主軸讓你自己投資獲利，而空投只是其上的附加品。你都沒有付出過額外努力和對那項目支持，為甚麼人家一定要給你？</p><figure class="image"><img src="https://assets.matters.news/embed/0f7bbcbd-13a1-42f8-823c-75c0605efd99.jpeg" data-asset-id="0f7bbcbd-13a1-42f8-823c-75c0605efd99" referrerpolicy="no-referrer"><figcaption><span>覺得自己有做 Staking / LP 很威/大哂，？</span></figcaption></figure><p>我個人相信好的能量才能吸引好運來，如果每次沒有空投都在怨天怨地或者在人家的 Discord 投訴，真的甚麼人就會有甚麼運，這些人就算有機會領空投也是波折重重。因為他們將領空投就等於麻煩的情況烙印在自己的心中，到下次領空投又會升起不好的情緒，於是循環不息。這個在 NLP 的學說中好像叫做 Anchoring 心錨或者 Moment of Truth 的甚麼的就自己去研究吧。</p><p>所以請加入佛系領空投，<strong>得之我幸，失之我命</strong>，放過別人，原諒自己啦各位施主，它又不是特別針對你不給你，也有很多人和你一樣爭一天就是那一天，爭幾個幣就是那幾個幣，沒有就是沒有。當然我不是說大家對於空投有問題不可以問，但是人家很辛苦寫很多 code 才把空投弄出來，也不要三不五時問候人家的娘親可好？將心比己，如果你用了很多時間搞了一個項目出來想給人好處卻不停被人鬧甚至起底痛罵祖宗十八代，真的是自己活該的嗎？所以有問題請先 DYOR，真的不明白才問人吧。不要添煩添亂，能讓項目團隊將節省下來的時間做更好的發展。</p><p>此外不要急於立即領空投做白老鼠是常識，區塊鏈團隊都不是每天有做空投的機會，不能一揮而就也很正常，大量用戶在同一時間領空投亦會線路阻塞。不能成功領取就擇日再試，無謂自己生氣又浪費 Gas Fee，</p><h2>類型：Prime / Sub</h2><p>其實這些空投類型是我自己創作的（Sign）。</p><p><strong>Prime</strong>：即是該項目在自己的<strong>官方</strong> Blog / Medium / Twitter / Discord / TG 公佈的空投。</p><p><strong>Sub</strong>：一些<strong>驗證人、幣圈 KOL 或交易所會自己在 Twitter 舉辦空投</strong>，例如 <a href="https://matters.news/@daisy/%E5%90%89%E5%A8%83%E5%A8%83%E5%B9%A3-%E7%9C%9F%E6%AD%A3%E5%8E%BB%E4%B8%AD%E5%BF%83%E5%8C%96-%E9%8B%A4%E5%BC%B7%E6%89%B6%E5%BC%B1%E7%9A%84%E6%AD%A3%E7%BE%A9%E7%A9%BA%E6%8A%95-bafyreiee2tpxmv6wshirxfrk5uyawqmr7ugu4buq2pmxul6sg23kzai7k4" rel="noopener noreferrer" target="_blank">吉娃娃議案 4</a> 就是驗證人自己弄的空投，後來越弄越大成為社群投票認可的空投。而 <a href="https://matters.news/@daisy/like-coin-%E9%A9%97%E8%AD%89%E4%BA%BA-imperator-co-%E5%89%B5%E8%BE%A6%E4%B8%80%E9%80%B1%E5%B9%B4%E5%A4%A7%E6%8A%BD%E7%8D%8E-bafyreicwxiyhzft4ykmewvr2izosrp2qogyychw3jky2pi3frmzh3updsq" rel="noopener noreferrer" target="_blank">Imperator.co</a> 空投也是同類型由驗證人自行發展的空投。</p><p>Prime 空投的訊息其實很難錯過的，有那麼多人會寫會介紹，你想合上眼也不能。至於 Sub 空投，有看過我 <a href="https://matters.news/@daisy/%E5%A6%82%E4%BD%95%E5%81%B7%E5%A4%A7%E7%A5%9E%E5%80%91%E7%9A%84-twitter-%E5%88%97%E8%A1%A8-bafyreie7bsjwiplacsx6ptlg2hstjdiuqn75zatsvxwi3eadt66kyohh2y" rel="noopener noreferrer" target="_blank">如何偷大神們的 Twitter 列表？</a> 內裡說得白一些就是叫你看大神的列表之餘也將自己的心水驗證人或 KOL 加入和活用列表方便監察。 有時候擼一些驗證人、KOL 或交易所的抽獎也不無小補，例如我就中過 ZB 的小獎（不過他們的帳號不知何解消失了）。</p><h2>範圍：不止 Staking 與 LP、不在範圍就 GG</h2><p>以往領 Cosmos 空投似乎比較簡單，5 個 ATOM 甚麼的就好，但現時領空投似乎要求更高，也不只要求委託與流動性挖礦而要做任務，我看過的有：</p><ul><li>曾經投票支持某一個議案 e.g. Evmos</li><li>答問題 e.g. AssetMantle</li><li>使用某些 DApp e.g. LikeCoin depub.SPACE</li></ul><p>至於不能領空投的情況似乎就更多：</p><ul><li>不在 Snapshot 快照日期範圍</li><li>沒有委託/流動性挖礦足夠數額</li><li>沒有在特定流動性池做 LP</li><li>委託了給中心化交易所驗證人 / 零佣金驗證人（參考 <a href="https://matters.news/@daisy/%E5%A6%82%E4%BD%95%E9%81%B8%E6%93%87%E9%A9%97%E8%AD%89%E4%BA%BA-bafyreicgodb53asgglwclehlzjsb7o3abmjhba4bu64xvpxpcfqi3hd7oy" rel="noopener noreferrer" target="_blank">如何選擇驗證人？</a>）</li></ul><p>所以為甚麼我說要佛系，有時千差萬錯就是錯在自己在某一天突然解委託或解流動性挖礦，又或者不關注那個區塊鏈發生甚麼事沒有投票積陰德，那你可以怪誰人？就是怪自己吧！不過有些情況就是你做甚麼也會被搬龍門，例如 <a href="https://ckxpress.com/june-proposal-16-timeline/" rel="noopener noreferrer" target="_blank">Juno 議案 16</a> 給了你也可以改，說會給你也可以不給你、那你怎樣？堂堂正正，有就有沒有就沒有，so what？不要呼天搶地好嗎？現世代就連乞兒都只會靜靜坐在一邊等人投幣不會大叫的，不要連乞兒都不如吧。</p><figure class="image"><img src="https://assets.matters.news/embed/cf5073c9-d289-4a75-b565-7ef5ce7cac98.jpeg" data-asset-id="cf5073c9-d289-4a75-b565-7ef5ce7cac98" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>Cosmos 空投不外乎是給予 Holders（特有者）、Delegator（委託人）、Liquidity Provider（流動性提供者），而最常見的是給委託人。我嘗試總結一些我認為現時獲得空投要預備的委託基本盤，而我相信這個需求隨著時局改變而不斷更改，以下只屬個人意見，不代表投資建議：</p><ul><li>10 ATOM</li><li>10 OSMO，其實可以預算更多</li><li>20 JUNO</li><li>10,000 HUAHUA</li><li>SCRT 即 Secret Network，據說是新星但未研究需要多少</li></ul><p>要留意 Ignite 即是以往名叫 Tendermint，即 Cosmos-SDK 核心團夥的舉動，基本上很多「親生仔」項目我個人相信都應該有好康可拿。</p><p>還有就是一些跨鏈項目，雖然 Evmos 鎩羽，不過我個人相信跨鏈項目應該比較慷慨，不過也不知道要怎樣預備，要真心愛用的人才能領到吧。</p><p>而以往通常都是<strong>空投在擷取快照以後才會發放消息</strong>說會有空投，但現在經已變成<strong>預先說明在未來何時會有快照</strong>。雖然有人說這樣做有點引人入局的意味，只是如果不會動用到我在委託的幣，我又怎樣可以入局呢？所以無謂太挑剔了。</p><h2>安全：領幣不蝕本</h2><p>這裡的虧本是指不要連自己的老本也虧了，被駭錢包。有很多人一看到空投就不顧一切，看到有東西就 click，其實這樣很危險的。所以如果你要領 Sub 的空投，<strong>請確認他們真的是驗證人，或者內容有被官方項目轉發才去領。</strong></p><p>另外要留意項目的負責人會否有 <strong>Rug Pull</strong> 前科（<a href="https://www.instagram.com/p/Ca15EEbp6XP/" rel="noopener noreferrer" target="_blank">三種類型 Rug Pull 你要知</a>、<a href="https://www.instagram.com/p/Ca7DUJ-ptKz/" rel="noopener noreferrer" target="_blank">如何避免成為 Rug Pull 受害者</a>）才去做流動性挖礦，否則你的 ATOM/OSMO 會被廢幣無常損失清倉。</p><p>縱使有很多人一收到空投就將幣賣出，可是我會說這樣做你可能會錯過一個明日之星。我知道怎樣苦口婆心叫人不賣幣都是無用。只是我自己抱著給我的幣其實就是我的幸運，怎樣也要將幸運留住一下，多了解一下給我空投的項目，就算要賣，也是因了解而分開，大家都有 Good Karma。</p><h2>多聞：空投資訊我全都要</h2><p>以下介紹一些提供 Cosmos
