referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>Cosmos Hub 經已完成 v7 Theta 升級，內容包括 SDK v0.45.1、#IBC -go v3.0.0 及<strong>跨鏈賬戶</strong> ( Interchain Account ) 模組。其中跨鏈賬戶原來是非常強大的東西。這幾天在推特有人解釋它是甚麼，如此重要的事值得轉述一下。</p><p>翻譯自 <a href="https://twitter.com/youssef_amrani/status/1513994892875386888" rel="noopener noreferrer" target="_blank">Youssef Amrani</a></p><p>今天 Cosmos Theta 升級靜靜上線。是次升級標誌著跨鏈賬戶正式啟用，並有潛力大幅提升區塊鏈互聯網的相互可操作性、流量和可組合性。跨鏈賬戶讓大量可能性出現，進一步加強 Cosmos 作為區塊鏈相互可操作性的先驅及領導者地位。</p><p>跨鏈賬戶<strong>令區塊鏈能以「API」形式開放原生功能並由另一條鏈自行寫入定義</strong>，然後透過跨鏈賬戶向這些 endpoints 執行寫入狀態。究竟跨鏈賬戶是什麼意思呢？你可以將它視為開展更多機會的 IBC 附加層。雖然現時 IBC 允許在 Cosmos 原生主權區塊鏈之間進行代幣交換和轉移，但跨鏈賬戶將更進一步。</p><p>它<strong>讓用戶淮許擁有他的資產的區塊鏈 A 對區塊鏈 B 進行操作</strong>。</p><blockquote>假設你正在使用 Osmosis 處理你的交易，想將 $UST 放進 Anchor 以獲取 19.5% 的收益。現時你需要從 Osmosis Withdraw Asset 取回 $UST，然後將 Keplr Wallet 的 $UST 發送到 Terra Station 錢包。然後再將 $UST 使用 Terra Station 或從 Anchor app 放進 Anchor。</blockquote><blockquote>然而如果 Terra 及 Osmosis 都設置了跨鏈賬戶模組並在它們之間開啟了一個通道，操作就會完全不同。</blockquote><blockquote>假設 Osmosis 團隊進行了修改，基本上你可以直接從 Osmosis 提取 $UST 並將它發送給 Anchor 令 $UST 發揮作用，減少步驟之餘亦而無需離開 Osmosis 網站。</blockquote><blockquote>又或者從 Osmosis 取出 $ATOM 並將它流動委託到 Cosmos Hub。這家 DEX 將為你提取 $ATOM 並於 Cosmos Hub 鏈上執行所有必要步驟，以便你可以解鎖你所委託代幣的流動性。</blockquote><p>跨鏈賬戶的主要好處如下</p><ul><li>簡化 Cosmos 生態系統的用戶體驗</li><li>解鎖新的跨鏈服務及商業模式</li><li>增加可組合性</li><li>增加跨鏈活動</li></ul><p>建議大家觀看 <a href="https://twitter.com/L1am_Crypto" rel="noopener noreferrer" target="_blank">Liam Connor</a> 的視頻 ，以便更深入了解跨鏈賬戶：</p><figure class="embed-video"><div class="iframe-container"><iframe src="https://www.youtube.com/embed/vIO5W016npQ?rel=0" frameborder="0" allowfullscreen="true" sandbox="allow-scripts allow-same-origin allow-popups"></iframe></div><figcaption><span></span></figcaption></figure><p>跨鏈賬戶是互相可操作性的下一個合乎邏輯（也是複雜）的步驟，它提供了改變一切的主要優勢：一條鏈能夠在另一條區塊鏈上進入寫入狀態。</p><p>用戶無需在每次想要執行交易時都 "Follow" 他的代幣，只需透過一條鏈開啟特定的跨鏈帳戶通道並接通另一條經已連動的鏈即可支援此功能，反之亦言。</p><p>直接衍生的結果令人驚嘆：當前的鏈將能夠開發新的可組合原型，從而能夠從其原生、單鏈業務模型過渡到跨鏈原生業務模型，從而使其影響範圍得以高質量地擴展。</p><p>這項新功能與共享安全性 ( Shared Security )，將開闢前所未有的使用實例及應用程序，並促進 Cosmoverse 中的創新及快速迭代，進一步鞏固 Cosmos 在相互可操作性及多鏈應用程序中的主導作用。</p><p>我個人認為<strong>跨鏈賬戶是 Cosmos 的第三大創新</strong>，前兩個是：</p><ul><li>能夠使用 Tendermint 及 Cosmos SDK 構建主權鏈</li><li>能夠透過 IBC 在它們之間進行交易</li></ul><p>毫無疑問，這第三項創新將吸引更多的開發人員、用戶及一流的用戶體驗及界面。</p><p>向參與這一重大里程碑的所有宇航員致敬。</p><hr><p>根據 <a href="https://twitter.com/EvmosOrg/status/1513879730667331593" rel="noopener noreferrer" target="_blank">Evmos</a> 的補充：</p><p>啟用跨鏈賬戶功能後，如果你有 Osmosis 帳戶，你可以在 Evmos 上操控 ERC20 代幣。</p><blockquote>在沒有跨鏈賬戶之前，如果 EVM 用戶想要在 Osmosis 上互換他們的 ERC-20 代幣：1. 他需要將代幣從他們的原始鏈橋接到 Evmos；2. 然後需要透過 IBC 將他的代幣從 Evmos 轉移到 Osmosis。</blockquote><blockquote>擁有跨鏈賬戶之後： 1.他 需要將代幣由原始鏈橋接到 Evmos；2. 然後直接從 Evmos 兌換 Osmosis 上的代幣。</blockquote><p>簡單省時。</p><h2>英文閱讀</h2><p><a
