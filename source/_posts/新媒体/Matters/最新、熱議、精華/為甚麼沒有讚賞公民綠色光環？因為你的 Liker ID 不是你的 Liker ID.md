src="https://assets.matters.news/embed/9da81867-7abc-4984-a4ce-099b8755ccb7.jpeg" data-asset-id="9da81867-7abc-4984-a4ce-099b8755ccb7" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>「為甚麼明明經已付費成為讚賞公民，卻還是沒有綠色光環」</h2><p>要解釋這個問題，首先要從 Matters 與 Liker Land 的聯系談起。有很多人時常以為兩者是同一家公司，但他們是不同的。是以如果你有客服的問題，Liker Land 就到<a href="https://liker.land/" target="_blank">網站</a>找客服，而 Matters 直接<a href="mailto:hi@matters.news" target="_blank">電郵</a>他們即可。</p><p>據悉，Liker Land 與 Matters 的「對接」每天只會發生一次（不肯定這個時間點是否與 Liker Land 發放創作基金的中午 12 時相同），所以如果你付費成為讚賞公民以後發現還未有綠色光環，可能是時晨未到，但是在 Liker Land 的更新肯定要比 Matters 還要快，建議你可以查看個人主頁，格式是：</p><p><a href="https://liker.land/%E3%80%8CLikerID%E3%80%8D" target="_blank">https://liker.land/「LikerID」</a></p><p>當你看到在個人主頁上經已有綠色光環，那麼在 Matters 的光環肯定會在「對接」後出現，這只是時間問題。</p><figure class="image"><img src="https://assets.matters.news/embed/92168b9a-c333-4c4a-916c-420934b1caae.png" data-asset-id="92168b9a-c333-4c4a-916c-420934b1caae" referrerpolicy="no-referrer"><figcaption><span>個人主頁的光環</span></figcaption></figure><p>然而假如你發現你的 Liker ID 在個人主頁沒有光環，卻經已付費了，那怎麼辦？</p><p>先到左手邊「我的」，然後再點選「設定」，再查看你的 Liker ID，你有可能發現：</p><figure class="image"><img src="https://assets.matters.news/embed/424e9510-0e7d-452c-bfa6-25da332b6f1a.png" data-asset-id="424e9510-0e7d-452c-bfa6-25da332b6f1a" referrerpolicy="no-referrer"><figcaption><span>「我的」「設定」</span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/a3963fa5-e54d-4804-8f96-834c6a2a324f.png" data-asset-id="a3963fa5-e54d-4804-8f96-834c6a2a324f" referrerpolicy="no-referrer"><figcaption><span>我的 Liker ID，歡迎樂助</span></figcaption></figure><h2>你的 Liker ID 不是你（心目中）的 Liker ID</h2><p>所以沒有光環是正常的。</p><p>可是為甚麼錢包設定的 Liker ID 不是你所想的 Liker ID？我認為有兩個可能性：</p><p>1. 你是舊制 Matters 用戶，Matters 2019 年轉制由 MAT 幣轉為 LikeCoin，<a href="https://matters.news/@daisy/%E7%8F%BE%E6%9C%89-like-coin-%E7%94%A8%E6%88%B6%E5%A6%82%E4%BD%95%E5%9C%A8-matters-%E7%B6%81%E5%AE%9A-liker-id-%E5%AF%A6%E5%9C%A8%E5%A4%AA%E7%B0%A1%E5%96%AE%E4%BA%86-zdpuB1VXByRv19fhiPD8zRDyB5YsxY231LsncTGnETPGnmav1" target="_blank">在綁定 Liker ID 時沒你有綁定正確的 Liker ID</a>，而自行生成了一個新的 Liker ID，然後你用「舊 Liker ID」訂閱讚賞公民這樣當然不會有光環。</p><p>2. 亦是類似的情況：你先在 Liker Land 申請了 Liker ID，然後才申請 Matters 帳戶，在綁定時沒有選擇綁定已申請的 Liker ID 而透過 Matters 網站申請了新的 Liker ID。</p><p>依我推測，發生這些情況很有機會是 Liker ID 和 Matters ID 並不是使用同一電郵地址去申請。因為一個 Liker ID 只能對應一個電郵地址，如果是上面 2. 的情況，用戶在使用同一電郵地址開立新的 Liker ID 的時候系統應該會作出提示之類，而不會讓一個電郵地址開立兩個 Liker ID。</p><p>假設你真的遇上了「我的」「設定」內的 Liker ID 和已訂閱讚賞公民的 Liker ID 完全相同而超過 24 小時都沒有光環，這樣的錯誤真正值得你去找 Matters 客服問問看。但如果單純只是你自己弄錯帳號，找 Matters 或 Liker Land 客服都沒有用，尤其是讚賞公民 2.0，當你付款的一刻那筆 LikeCoin 其實經已到了贊助的創作者的手中，人家又怎能替你取回？而且 Matters ID 和 Liker ID 是不能解綁的：試想想如果有人聲稱你有 10 萬 LikeCoin 的 Liker ID 是他的然後解綁了再綁在自己的 Matters ID 上任意揮霍，你也不想發生這樣的情況吧？</p><p>Liker Land 使用 Authcore 密碼管理方案背後的 Intel SGX 和 SPAKE2 本身都是業界強度很高的系統，Liker ID 和 Matters ID 一但做了帳號綁定，其實是很難有 BUG 和自行解綁的。而用戶自己認為自己做過甚麼帳戶操作，和他真實做過甚麼帳戶操作可以是完全不同，尤其是本身電腦知識經已不強的用戶。</p><p>是以早前某用戶不停在說 Matters 有 BUG 我沒有讚賞公民光環然後被發現擁有多個帳戶，很大程度上就是發生了上面提及的問題。而老實說要參透自己有沒有在 Matters 綁定正確的帳號，只要登入 Liker Land app 查看有沒有人為自己拍手又或者收不收到每月報告就可以知道。不過，對於擁有多個帳號的人，又確實會有機會搞不清楚自己在做和在看些甚麼。</p><p>最後我想補充，對於開小號或者擁有不同的 Persona 去寫作我認為並無不可，最緊要循正路走正道而不是利用別人拍手的善心。對於抄襲、是不是一人分飾幾角大家亦寫了很多所以我就不多說了。我寫這篇的主要原因除了從我一直觀察 LikeCoin 生態的經驗去嘗試釋疑以外，還是想讓大家知道，在事件中一味說 Matters 弄錯了帳號的當事人其實有沒有考慮過 Matters 作為初創日常有那麼多事務要處理，你卻還要不停為著自己「疑似分身」的「疑似綁錯帳號沒有光環」而客訴 Matters 是多麼的錯誤？我有多麼心痛 Matters 的工作量你知不知道？還有當日有那麼多人為你一起拆解你的問題但你卻完全沒有真心回答過別人你做過些甚麼而只是一味機械式回覆訴說是系統錯誤。就算你退回 LikeCoin 給某些作者，也不會讓我有好感和減少對你懷疑。</p><p>就這樣。原本不打算爆 Seed 的，但真的不吐不快。Matters 辛苦了。</p>
