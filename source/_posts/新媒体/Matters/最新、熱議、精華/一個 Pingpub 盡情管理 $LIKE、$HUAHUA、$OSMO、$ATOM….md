等各種區塊鏈資產（<s>應該是富戶必備？</s>）</p><h2>設定</h2><p>1. 到 <a href="https://ping.pub/" rel="noopener noreferrer" target="_blank">https://ping.pub/</a> 在右上角會看到「月亮」，點它可以轉換為 dark mode、「國旗」可以更換語言但暫時沒有中文選擇。點「Wallet」。</p><figure class="image"><img src="https://assets.matters.news/embed/a22342e0-ce19-4fe8-9ab3-eebddc605315.png" data-asset-id="a22342e0-ce19-4fe8-9ab3-eebddc605315" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>3. 如果你使用 Keplr 錢包，選 Keplr 再點「Next」，然後彈出 Keplr 視窗，點「Approve」。</p><figure class="image"><img referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>4. 在 Account Name 一欄輸入一個自己心儀的戶口名稱，然後在下面的 Import Address for Chains 選擇導入你會用到的 Cosmos 系區塊鏈。留意<strong>如果你不是使用硬錢包如 Ledger</strong>，請不要選有「紅色字」那些區塊鏈，因為它們的 HD Derivation Path 是預設為硬錢包的用途，和 Keplr 是不相通的。</p><figure class="image"><img src="https://assets.matters.news/embed/52250a51-af1c-423e-a30d-81bbc665273b.png" data-asset-id="52250a51-af1c-423e-a30d-81bbc665273b" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>如果我使用 Keplr 登入，就只可以玩到 LikeCoin、Osmosis、Cosmos、Chihuahua，而 BitSong 和 Desmos 都是玩不得。選好後點「Next」。</p><p>5. 顯示你輸入的用戶名稱和將要導入的錢包地址，選「Save」。</p><figure class="image"><img src="https://assets.matters.news/embed/b999db5b-59f8-494b-a1de-864d4bed7a0b.png" poor me bankrupt 所以是沒有錢的，但我相信在富戶的頁面就會相當精彩了。</p><figure class="image"><img src="https://assets.matters.news/embed/8267c829-19c4-46ce-b7e0-65b32cf3127d.png" Transfer」和「Remove」。 「Remove」當然就是移除這個錢包地址，點「Details」可看擁有資產的詳情，「Transfer」是正常的轉帳，而「IBC Transfer」就是 IBC 傳送到其他 DEX 去中交易所等。</p><h2>轉帳 🎯</h2><p>6. 點「Tranfser」後會看到這個視窗。要傳送 LIKE 給別人就在 Recipient only for experts 的警告，我都不懂的了這麼深奧，就算了吧。不過對於高階用戶來說，這個可能是一個非常好用有用的功能，不能忽視。</p><h2>錢包詳細頁面</h2><p>7. 點進 LikeCoin 「Details」頁，左上角的相機 icon 點下去會出現 QRCode 方便別人傳送 LIKE 到這個錢包。Assets 圓圈的資產組合顯示有多少 Balance
