<p>我數月前，參與朋友籌備的一個 Web 3 項目。經過幾個月的時間，現定於 9月15日推出。我是出身於 Steem 及 Matters，故此，最相熟的 Web 3 朋友就是來自這兩個社區，而兩個社區裡，最緊密關係是在 Matters，所以，我打算第一次的 AMA 於 Matters 進行。</p><p><strong>先分享一下，這是一個甚麼項目，為甚麼我會加入?</strong></p><p>今年年初，我有幾位朋友連同一些武術界的師父傾談「武術保育」的問題，期間接觸了區塊鏈，並且在構想「能否藉著區塊鏈作武術保育」。在籌劃過程中，一直摸索可結合區塊鏈的內容及元素。在與團隊傾談後，認為不一定要為了 Web 3 而 Web 3，互聯網絡的生態是並行，Web 2 的年代仍有大部份網站及項目是 Web 1 模式。即是 Web 3 推行時，也不會令到 Web 1 及 Web 2 的項目消聲匿跡。故此，我們就定下  Web 2 + Web 3 的方案來推行這一個項目。</p><p>這幾個月，我就是參與於這一個「武術區塊鏈」Martial Art DAO 的項目之中，當中我主要負責策劃的角色。構想怎樣透過區塊鏈儲存技術 (IPFS Cloud)，結合動態捕捉，將各武術門派的招式記錄。我們也邀請了香港一所大專院校「香港專業教育學院」設立了一個武術動態捕捉中心:</p><figure class="image"><img src="https://assets.matters.news/embed/e7a27a10-d96f-46cb-ad2a-3fa3617921f7.jpeg" data-asset-id="e7a27a10-d96f-46cb-ad2a-3fa3617921f7" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/cff0abbd-df80-4432-962c-e8c83acea807.jpeg" data-asset-id="cff0abbd-df80-4432-962c-e8c83acea807" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/094beed6-aa50-46fc-bae7-b81293ad0dc0.jpeg" data-asset-id="094beed6-aa50-46fc-bae7-b81293ad0dc0" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>現時，除了拳腳招式外，我們也開始武器上的動態捕捉。</p><figure class="image"><img src="https://assets.matters.news/embed/901d4d01-edea-4b6a-af76-aa1d9007a5ef.gif" data-asset-id="901d4d01-edea-4b6a-af76-aa1d9007a5ef" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>在不同武術的動態捕捉後，我們使用另一個技術支援伙伴，將數據加密並儲存在 IPFS Cloud 中，讓建立中的虛擬教學培訓平台與及遊戲動畫伙伴可以使用。</p><p>此外，項目的營收有60-80%會投放至製作及培育基金，因為我們認為武術保育不能只是技術，還要作人與人之間的傳承。所以，我們也會建立虛擬教學培訓平台及Web3武術社區，此外也會建立 Train the Trainer為目的之培育基金，培育基金的運作以 DAO 模式進行，目標能支援不同武術拳會，培育更多接班人。</p><p><strong>至於，為甚麼我會加入這一個項目?</strong></p><p>我在Luna事件後，我深深覺得區塊鏈及加密世界是否只有「金錢價值」(Amount) ? 還是可以有「值得應用的場景」(Value)？</p><p>人到中年，金錢價值已不是最重要的目標，希望在人生下半場所用的時間，都是一些具有意義的項目。正正原本朋友找我「問意見」的項目，結合了「傳統武術文化保育」「區塊鏈」「Web3社區」「遊戲」及「培育基金」，除了我不懂得「武術」外，其他每一項都是我很有興趣的事情。而慶幸這一個項目，已有超過七十位武術師父參與及支持。</p><p>我常常跟團隊說，我們真的要感謝Luna事件的爆破，就正正是這一個情況，更能令社區「知道怎樣分辦項目」與及「知道怎樣善用DAO」。</p><p>九月十五日（星期四）是我們開始向外簡介這項目的日子，未來三個月，我們會開始展開不同的宣傳及簡介工作。</p><p>第一個AMA，我主動找了 Matters 協助，主要是我們想將AMA成為接受意見的平台，，我常常參與Matters的AMA及講座，討論氣氛相當良好，我們希望更多有心人參與；此外，而我最熟悉的社區是 Matters 及 LikeCoin，故此希望由此而起。</p><p>現時，創始團隊全為香港人，故此，我們最熟悉的語言是粵語（廣東話），我們怕第一次AMA會因為「語言能力有限」而引致大家感受不好。所以，我跟多比說，想試試做Matters的粵語AMA。</p><p>我知道，因為粵語問題，應該會少了很多人參加。故此，我們盡力用「文字討論及即時文字翻譯補足」，當我們「信心及語言能力再提升」，必定會在「短期內」做一次全國語的AMA。</p><h2><strong>邀請您，九月十五日(星期四) 晚上九時至十時半，我們會在 Matters 的Discord 頻道做一次粵語AMA，希望您能出席參加。</strong></h2><h2><strong>未來幾天會再推出宣傳資料，若您有任何查詢，可以留言給我，我會盡力解答。</strong></h2><p>感謝各位!!</p>
