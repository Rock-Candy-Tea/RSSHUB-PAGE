<p><br></p><p>「出版」是個古早詞彙，我們都知道在傳統世界出版一本書是甚麼意思，然而隨著互聯網以至智能手機普及，任何人一個點擊就能發表作品，在這個新世界，出版到底意味著甚麼？</p><p>我們不會把臉書帖文理解為出版物，主因不是帖文很短，更多是因為作者的動機。雖然資訊科技模糊了出版的概念，但用心創作，希望作品廣泛傳播，流芳百世這個動機，並無改變。具體而言，在無大台出版的生態中，是否為作品註冊 <a href="https://ckxpress.com/from-isbn-to-iscn/" target="_blank">ISCN</a> 可以理解為是否「出版」，有如在傳統世界，出書需要 <a href="https://ckxpress.com/from-isbn-to-iscn/" target="_blank">ISBN</a>。</p><p>以下，我一步一步，示範把作品出版到區塊鏈，透過實作去理解無大台出版的意義。雖然例子只是簡單的一張照片，但出版整本書甚至整齣電影，原理相同。畢竟 Matrix 眼中，都是 0 與 1 而已。</p><hr><p><strong>0、</strong>請像其他 10 多萬 LikeCoin 的社群成員，安裝 Chrome/Brave 的外掛 <a href="https://chrome.google.com/webstore/detail/keplr/dmkamcknogkgcdfhhbddcghachkejeap" target="_blank">Keplr</a> 錢包；然後訪問 <a href="https://stake.like.co/" target="_blank">stake.like.co</a>，允許 Keplr 加入 LikeCoin chain，並存入小量 LikeCoin。詳細步驟可參考 LikeCoin 百科全書： <a href="https://docs.like.co/v/zh/guides/wallet/keplr" target="_blank">doc.like.co</a>。</p><p>獲得 LikeCoin 的方法很多，很多作者會<a href="https://docs.like.co/v/zh/user-guide/creator" target="_blank">為作品加上 LikeCoin button</a>，有人按讚，你就能獲得 LikeCoin；不想創作的，<a href="https://docs.like.co/v/zh/guides/trade" target="_blank">在交易所購買</a>也行；實在不行，就算是<a href="https://ckxpress.com/about" target="_blank">找我</a>要一點點也沒關係。出版一份作品，一般只需不足 1 LikeCoin，換算下來不到台幣 1 毛錢，非常便宜。</p><figure class="image"><img src="https://assets.matters.news/embed/27bd885e-0604-4832-a5a0-25209635c5a4.jpeg" data-asset-id="27bd885e-0604-4832-a5a0-25209635c5a4" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><p><strong>1、</strong>以安裝了 Keplr 的電腦版 Chrome/Brave 瀏覽器訪問 <a href="https://app.like.co/" target="_blank"><strong>app.like.co</strong></a>。</p><p>點擊右上角 <strong>[Keplr]</strong> 以錢包登錄，點擊 <strong>[Register ISCN]</strong> 以出版作品。</p><figure class="image"><img src="https://assets.matters.news/embed/ec3ce528-3952-40b0-9b9b-3344d36d3847.png" data-asset-id="ec3ce528-3952-40b0-9b9b-3344d36d3847" referrerpolicy="no-referrer"><figcaption><span>app.like.co</span></figcaption></figure><hr><p><strong>2、</strong>點擊 <strong>[Select a file]</strong> 以上傳照片，點擊 <strong>[Publish to IPFS]</strong> 把照片上傳到「無大台硬盤」。</p><figure class="image"><img src="https://assets.matters.news/embed/4151f8d3-444e-41eb-a87c-ae0b15c56a0a.png" data-asset-id="4151f8d3-444e-41eb-a87c-ae0b15c56a0a" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><p><strong>3、</strong>填上標題、描述、關鍵字、URL 等資料。大部分資料為可選，但寫得清楚將有助內容發現與用戶查核。</p><figure class="image"><img src="https://assets.matters.news/embed/50a6d447-aef5-417c-8227-f3c68ba30c88.png" data-asset-id="50a6d447-aef5-417c-8227-f3c68ba30c88" referrerpolicy="no-referrer"><figcaption><span>照片的 exif referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><p><strong>5、[Confirm]</strong> 後，Keplr 會彈出小窗要求確認及付費。這例子費用為 0.18 LIKE，換算下來不足 0.005 USD。</p><figure class="image"><img referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><p><strong>6、</strong>一般只需幾秒鐘，作品即可出版到區塊鏈。</p><figure class="image"><img iscn://likecoin-chain/MyHJ_UH-M_VNnTSvxn46yOFUjLdVFcglTgex0FKM7yo/1</span></figcaption></figure><hr><p><strong>7、</strong>回到 <a href="https://app.like.co/" target="_blank">app.like.co</a> 主頁，點擊 <strong>[Your Publishing]</strong>，列出你的所有出版物，選作品可查看所有元資料及作品本身。</p><figure class="image"><img src="https://assets.matters.news/embed/7dd84210-ffef-450d-871e-d31c6ddf1842.png" data-asset-id="7dd84210-ffef-450d-871e-d31c6ddf1842" referrerpolicy="no-referrer"><figcaption><span>app.like.co/view/</span></figcaption></figure><hr><p>把作品出版到區塊鏈，無需獲得中央許可，不可竄改，作品點對點存放，避免因單一服務關門大吉或者伺服器被封作品就不見天日。不但如此，註冊 ISCN，也為把作品鑄造成 NFT 打好基礎。下期，我們會就以上幾點進一步解釋。</p><p>現在，先來試試區塊鏈出版，為自己的作品註冊第一個 ISCN 吧。</p><p>doc root: <a href="https://ckxpress.com/iscn-tutorial/" target="_blank">chungkin Express</a></p><hr><p>我是《<a href="https://ckxpress.com/the-sociology-of-blockchain/" target="_blank">區塊鏈社會學</a>》的作者<a href="https://ckxpress.com/about/" target="_blank">高重建</a>，斜槓寫作/教學/創業，探討<a href="https://ckxpress.com/tag/depub/" target="_blank">出版自由</a>、<a href="https://ckxpress.com/tag/defi/" target="_blank">財務自由</a>和<a href="https://ckxpress.com/tag/dao/" target="_blank">民主自由</a>，作品每週五發表於 <a href="https://ckxpress.com/tag/decentralizehk" target="_blank">#decentralizehk</a> 週刊。</p><p>這些文章全面公開，旨在普及知識；如果你相信有價值，別懲罰開放，鼓勵封閉，請付費訂閱，支持我一直做下去。</p><p>[ <a href="https://liker.land/ckxpress/civic" target="_blank">課金支持</a> ]</p>
