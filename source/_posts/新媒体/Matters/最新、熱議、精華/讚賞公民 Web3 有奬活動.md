<h2>委託最少 5,000 LIKE</h2><p>下週一（2月21日）推出的讚賞公民 Web3，只需以連結了 Liker ID（注意，不是 Matters ID）的錢包，<strong>委託最少 5,000 LIKE 到「讚賞公民」節點</strong>，即可獲得讚賞公民資格，用戶的所有委託回報將以按讚方式分配給創作者。不需動用委託本金，只花委託回報便可化讚為賞。</p><p>以目前約 15% 的委託回報率計算，5000 LIKE 可產生的每月收益等於 5000*0.15/12 = 62.5 LIKE，按現匯價約每月 1.5 美元左右。委託愈多，化讚為賞的金額也愈多。</p><figure class="image"><img src="https://assets.matters.news/embed/9be6485b-37e3-43d7-90de-f4e16d8de759.png" data-asset-id="9be6485b-37e3-43d7-90de-f4e16d8de759" referrerpolicy="no-referrer"><figcaption><span>*委託回報：委託 LikeCoin 到節點，可獲回報收益</span></figcaption></figure><p>另外，讚賞公民按讚除了能分配委託回報外，更同時能分配<strong>創作基金</strong>，按讚對基金分配的影響力會隨著委託金額變化，委託愈多，影響力愈大。</p><figure class="image"><img src="https://assets.matters.news/embed/7fe08da1-978f-4caf-a7da-525ba3a1ec56.png" data-asset-id="7fe08da1-978f-4caf-a7da-525ba3a1ec56" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><h2>有奬迎新活動</h2><p>作為迎新奬勵，也為鼓勵用戶委託更多 LikeCoin 回饋創作，Liker Land 將舉辦以下活動。</p><p><strong><u>活動詳情</u></strong></p><p><strong>日期</strong>：2022年2月21日 12:00 至 3月7日 11:59 (GMT+8)</p><p><strong>參加辦法：</strong> 所有在上述期間成為「讚賞公民 Web3 」的用戶</p><ul><li>分享總額 50,000 LIKE 奬金，分配方式主要參考委託回報產生量。</li><li>獲得一枚 讚賞公民 Web3 的 POAP 徽章，日後將可憑此徽章獲得其他 NFT 紀念品。</li><li>若總委託量超過 500 萬 LIKE，Liker Land 將派出額外 50,000 LIKE 奬金</li><li>曾參加舊版讚賞公民計劃的用戶，若參加新的讚賞公民 Web3 計劃，可額外獲得專屬的 POAP 徽章。</li></ul><p>奬金和 POAP 將於活動結束後兩週內發放。LikeCoin 奬金會直接發放至獲奬者的 Liker ID 註冊錢包，領取 POAP 的方法則會以電郵通知，故請確保你的 Liker ID 註冊了正確的電郵地址。</p><p>讓我們繼續化讚為賞，回饋創作。</p><hr><h2>趕上 Web3 潮流</h2><p>除了讚賞公民改版外，LikeCoin 生態中還有不少有趣的 Web3 應用正在發展中，例如由社群自行開發的 <a href="https://depub.space/" rel="noopener noreferrer" target="_blank">depub.SPACE</a> 和 Liker Land 團隊研發的 <a href="https://wordpress.org/plugins/likecoin/" rel="noopener noreferrer" target="_blank">LikeCoin WordPress 外掛</a>。depub.SPACE 是去中心化推特，做到預設保障私隱及內容上鏈；LikeCoin WordPress 外掛則讓全球 4 億 5 千萬個 WordPress 網站，輕鬆進入 Web3 世界，出版內容到區塊鏈，發行 「準 NFT」。另一方面，<a href="https://stake.like.co/proposals/33" rel="noopener noreferrer" target="_blank">議案 33</a> 通過了後，社群已聘請了 OurSky 團隊協助開發 LikeCoin 上的 NFT 項目。</p><p>期待 Web3 新界的冒險旅程！</p><p>* <em>Liker Land
