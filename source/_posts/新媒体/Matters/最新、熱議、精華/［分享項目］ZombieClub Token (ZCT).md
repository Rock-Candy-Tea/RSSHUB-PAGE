NFT 作品的稀缺性和收藏性: </strong></p><p>雖然沒有公布 Roadmap, 但從團隊說過 PFP 只是一個開始又已經規劃了1年才推出，再加上合作夥伴有<strong><u>日本藝術家河村康輔</u></strong>、<strong><u> Zombot Studio</u></strong> 和<strong><u>生成藝術家吳哲宇</u></strong>，因為是 <strong><u>Dynamic NFT</u></strong> 所以 ZCT 4 月 28 日開始陸續在 NFT 外觀上產生了變化，就稀缺性和收藏性來說都非常讓人期待後續還會怎麼發展跟變化，跟其他 PFP 項目比起來 ZCT 讓人對他的藝術比較有想像空間。</p><p><strong>3. 項目熱度和關注度非常高: </strong></p><ul><li>ZombieClub Token 發行前，Whitelist 取得方式非常不容易。</li><li>ZombieClub Token 發行後，Pre-sale 就拿下 OpenSea 單日最高成交量排行榜紀錄。</li><li>不少藝人朋友和 NFT OG 也都持有 ZombieClub Token, reveal 開啟當晚余文樂直播還邀請了周星哲和周予天兄弟到場，過程也與許瑋甯視訊直播 reveal.</li><li>解盲前二級市場的掛單量非常低，經常上架數量僅占總發行量的 5% 不到，目前掛單率約莫 12.5% 也算不高，代表市場上想出售的 holder 偏少。</li></ul><hr><h2>特別之處</h2><p><strong>1. 社群共識強</strong></p><p>目前在亞洲已經是為建立 NFT ThinkTank 智庫型社區創造了一個里程碑，社群為區塊鏈(幣圈、NFT 為主)友善的教育/知識/經驗分享和發想的社群，當然裡面也有頻道也有發想 IRL 的各種可能和機會，以及與生活和實物收藏相關的頻道，<strong>或許可以把他想成目前中文 Discord 社群中的 PTT 或 Dcard</strong>.</p><p><strong>2. Collaboration</strong></p><ul><li>Zombie Club 長期合作夥伴有 Teahouse、Zombot Studio、Elysium 和上過 Art Blocks 知名生成藝術家吳哲宇 (<a href="https://cheyuwu.com/nft)" rel="noopener noreferrer" target="_blank">https://cheyuwu.com/nft)</a> ，這不禁讓人聯想 ZombieClub 之後會有 3D Avatar、元宇宙和生成藝術相關的 NFT 作品推出。</li><li>Reveal 是由 Teahouse 負責執行開發，使用了 Chainlink VRF 技術達到 Fair Launch 的目標。</li><li>本次的 ZombieClub Token 還有日本知名藝術家河村康輔 (Kosuke Kawamura) (<a href="https://www.instagram.com/kosukekawamura/)" rel="noopener noreferrer" target="_blank">https://www.instagram.com/kosukekawamura/)</a> 也參與其中，也可以想成是河村康輔所推出或參與的第一款加密藝術作品。</li><li>ZombieClub 也是 GameFi 項目 Apeiron 的策略夥伴之一，很難不聯想之後的發展或 IP 的使用場景。</li><li>AMA 跟 giveway 活動的合作對象，也都是市場上較有名氣的項目。</li></ul><p><strong>3. 藝術性: </strong></p><p>河村康輔 (Kosuke Kawamura) 和生成藝術家吳哲宇的參與，我想 ZCT 已經具備一定程度的收藏價值了! ZCT 數位藝術呈現又是市場上接受度最高的像素賽道，像素歷久不衰且復古風潮也是一波一波的接著來，所以我想藝術性在市場的接受度應該都能保持在一定水準。</p><hr><h2><strong>總結:</strong></h2><p>我個人認為好的 NFT 本身是要具備無限創造力和想像力的，在 ZombieClub 有看到創造力和想像力的巨大空間，也許只是因為 Roadmap 不公佈而有過多的幻想，但是從目前合作陣容和未來潛在的各種合作機會來看，還是有不小的想像空間。社群的部份，雖然 reveal 後活耀人數沒先前多，但是 ZCT 團隊也透過社群徵集意見和改善提案，目的就是透過改變讓社群的氛圍和內容產出往更正向的方向發展。</p>
