<p>在 Cosmos Hub 世系的區塊鏈當中，驗證人狀態不止分開 Active 活躍或是 Inactive 不活躍。而是可以有很多種不同類型，以下將介紹驗證人的不同狀態。留意下面的 bond 與 unbond 是指驗證人伺服器對區塊鏈而言，而不是談及 Delegator 委託人對 Valdiator 驗證人所做的 delegate 委託與 undelegate 解除委託指令，雖然兩者息息相關。</p><figure class="image"><img src="https://assets.matters.news/embed/f5da911c-5e88-4c2c-9efe-74624574658b.png" data-asset-id="f5da911c-5e88-4c2c-9efe-74624574658b" referrerpolicy="no-referrer"><figcaption><span>不活躍驗證人也分很多類型</span></figcaption></figure><hr><p><strong>in</strong> <strong>validator set</strong>：驗證人在 <strong>active set</strong> 即是在活躍設置中也是同樣的意思，驗證人可參與管治共識及獲得 <a href="https://docs.like.co/v/zh/user-guide/background#9e68" rel="noopener noreferrer" target="_blank">驗證交易</a> 的獎勵，但亦有可能隨時因行為不當而令獎勵被 slash 即削減。現時 LikeCoin 活躍驗證人數目由 <a href="https://likecoin.bigdipper.live/proposals/11" rel="noopener noreferrer" target="_blank">議案 11</a> 界定為 50 個位置，只要委託量位於前 50 大，驗證人就能獲得驗證交易的回報。</p><p>BigDipper 所顯示的 Active，其實就是所有包含在 active set，即是在 Cosmos Hub 術語中 Bonded（綁定）狀態的驗證人。而 Inactive 的驗證人可以處於 Unbonded（未綁定）、Unbonding（正在解綁）、Jailed（入獄）的狀態。Unbonded 和 Unbonding 統稱為 Not Bonded（不在綁定）。</p><p>驗證人可以直接在所有狀態之間轉移，除了從 Bonded 不能直接變成 Unbonded，而是要經過 Unbonding 的階段。</p><figure class="image"><img src="https://assets.matters.news/embed/cf24de91-8eee-4374-b4c1-9a30b7f25ce0.png" data-asset-id="cf24de91-8eee-4374-b4c1-9a30b7f25ce0" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><strong>Unbonded</strong>：驗證人不在 active set，因此不會簽署區塊。 在此期間驗證人不能被 slash 即委託不會被削減，但亦不會讓你獲得任何獎勵。可以說，驗證人就是在設置一個伺服器在耗，或是在等待一但有足夠委託伺機而上成為活躍驗證人。然而作為 delegator 委託人的你仍然可以 delegate 委託給這個 unbonded 驗證人。</p><figure class="image"><img src="https://assets.matters.news/embed/b543d37d-dcc5-4b81-80a4-4b60a5e9a350.png" data-asset-id="b543d37d-dcc5-4b81-80a4-4b60a5e9a350" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><strong>Bonded</strong>：一但驗證人收到足夠已綁定代幣即是委託，他們會在 <em>EndBlock</em>（區塊完結）期間自動加入 active set，並且狀態會更新為 bonded（在 BigDipper 會看到叫做 <strong>Acitve</strong>），代表驗證人正在簽署區塊並獲得獎勵。他們可以接受進一步的委託，亦可以因不當行為而被削減本金。</p><p>在此期間委託人解除給予驗證人的委託必須等待直至達到足夠 UnbondingTime 亦即是解除委託時間長度才能完成，每個不同區塊鏈設設置不同特定參數。例如 LikeCoin 的 UnbondingTime 參數被設定為 21 天、Osmosis 和 Desmos Network 則是 14 天。</p><p>在解除委託期間，如果驗證人繼續發生違規行為，委託人依然可以受到懲罰。在 <a href="https://docs.like.co/v/zh/user-guide/background#9e68" rel="noopener noreferrer" target="_blank">LikeCoin 大百科</a> 當中說明不在線或雙重簽署，Likers 已委託的 LikeCoin 將依照同一比率被扣減。</p><figure class="image"><img src="https://assets.matters.news/embed/0f97fbd9-ddef-4bc5-a0c7-ce26bebe139e.png" data-asset-id="0f97fbd9-ddef-4bc5-a0c7-ce26bebe139e" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><strong>Jailed</strong>：驗證人行為不當被送入獄，並且被排除於 validator set 之外。如果由於離線時間過長而被監禁，驗證人可以發送一個 unjail 解除監禁交易以重新進入 validator set。然而如果監禁是由於 double signing 雙重簽署，亦即是同一筆交易被驗證兩次，則驗證人就無法 unjailed（我也找不到這個情況可以怎樣 unjailed，事實上某位驗證人在上星期差點就 double signing，如果他出事我們就可以知道怎樣可解了…大誤！）。</p><figure class="image"><img src="https://assets.matters.news/embed/6b4fc50b-ded3-4057-a3d0-b985a3f37636.png" data-asset-id="6b4fc50b-ded3-4057-a3d0-b985a3f37636" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><strong>Unbonding</strong>：當驗證人出於選擇或由於 slashing 削減、jailing 監禁或 tombstoning 被墓碑化（類似假如違規發生和證據之間存在延遲的故障）而不是在 active set，所有委託的 unbonding 解除綁定將立即開始， 然後，所有委託必須等待足夠的 UnbondingTime，才能將代幣從已綁定的池轉移回到他們的賬戶中。</p><p>留意上文這個解除綁定指的是在伺服器的層面而言，而不是指委託人取回委託的行為，無論是驗證人是 bonded ( Active )、unbonded 或 unbonding，委託人取回委託必須靠委託人自己在 Liker Land app 或 stake.like.co 取回委託，而不會由驗證人自動為你解綁。</p><p>所以你們會發現在 <a href="https://matters.news/@thumbb13555/%E7%A2%BC%E8%BE%B2%E6%97%A5%E5%B8%B8-%E9%A9%97%E8%AD%89%E4%BA%BA-%E6%88%91%E7%82%B8%E4%BA%86%E6%88%91%E7%9A%84%E7%AF%80%E9%BB%9E-%E7%84%B6%E5%BE%8C%E5%80%9F%E5%B1%8D%E9%82%84%E9%AD%82-bafyreiagv7rk4tdh73a6tlmng4pt26nlp3fpgccehvfnognafjfn4rymxm" rel="noopener noreferrer" target="_blank">碼機暴走事件</a> 之後，<a href="https://likecoin.bigdipper.live/validators/cosmosvaloper12w2qk222nxtr4dt5anwp657pz0l7xcwu0rje4u" rel="noopener noreferrer" target="_blank">碼機一號</a> 現時的名稱被設為「The node has been closed.」就是想通知已委託給碼機一號的委託人轉移委託到 <a href="https://likecoin.bigdipper.live/validators/cosmosvaloper1fdpmw3k73asq89gt0qvf9j97pgg2kapkk99yl2" rel="noopener noreferrer" target="_blank">碼機二號</a>。</p><p>如果想在不活躍驗證人之間轉移委託，請參考 <a
