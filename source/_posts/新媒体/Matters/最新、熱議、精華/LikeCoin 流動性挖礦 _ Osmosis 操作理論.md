</pre><p><strong>增加流動性 LPing</strong></p><p>從列表中選擇一個池，然後單擊添加 / 刪除流動性「Add/Remove Liquidity」。</p><p>輸入其中一項密碼貨幣資產的數量。 其他資產的數量將依照該池所設定的比例自動填入。</p><figure class="image"><img src="https://assets.matters.news/embed/79d32e8b-584c-4f30-8ae1-515a132aa8ec.png" data-asset-id="79d32e8b-584c-4f30-8ae1-515a132aa8ec" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>要移除流動性，請輸入需要提取的百分比金額。</p><figure class="image"><img src="https://assets.matters.news/embed/fbf40c5c-8885-400c-830a-f1e5574ab0ea.png" data-asset-id="fbf40c5c-8885-400c-830a-f1e5574ab0ea" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><blockquote>【解說】做到這步只是一個流動性提供者 LP Liquidity Provider，並沒有「挖礦」。為能夠產生獎勵的流動性池 / 激勵池 Incentivized pools 提供流動性才可獲得 OSMO 流動性挖礦獎勵，而並非每個在 Osmosis 的流動性池都是激勵池。 激勵池的獎勵將分配給池中已綁定 bonding 並能滿足綁定長度標準的 LP Token 。</blockquote><p><strong>LP Token</strong></p><p>當用戶將資產存入流動性池時會收到 LP Token，代表資產在池中的份額。</p><p>例如 #1 是 OSMO/ATOM 池，用戶可以將 OSMO 及 ATOM 密碼貨幣存入池中並從池中按照「你的資產 V.S. 整個池的比例」收回一定程度的 #1 池 LP Token（有時候 LP Token 或會叫做 GAMM）。這些 LP Token 和你實際擁有的 OSMO/ATOM 數量無關，而是對應你於此池比例所有權。</p><p>當你從池中取回資產並解除流動性，即能取回由 LP Token 所代表的流動性百分比的密碼貨幣。</p><p>由於從池中買賣會改變池內資產數量，用戶極不可能取回與最初存入的每項密碼貨幣數量相同的密碼貨幣。根據從池中曾經執行過的交易，你通常會收到更多或更少密碼貨幣。</p><blockquote>【解說】請直接跳轉到 <a href="https://matters.news/@daisy/like-coin-%E6%B5%81%E5%8B%95%E6%80%A7%E6%8C%96%E7%A4%A6-%E9%A0%90%E5%82%99-%E8%A7%A3%E6%A7%8B-bafyreibkfaml4w6xbpccbqdmhfvyupvmtezonvsabhqpvibefujb7dg3re" target="_blank">LikeCoin 流動性挖礦 | 預備・解構：甚麼是無常損失 ( Impermanent Loss )？</a>那一段了解為甚麼會有機會不能取回原有數量的幣。</blockquote><p><strong>綁定 Bonding LP Token</strong></p><p>用戶可以在存入密碼貨幣 LPing 後選擇綁定他們的 LP Token。 用戶可選擇將 LP Token 在指定時間內維持綁定。如果能滿足最低綁定長度要求，綁定的 LP Token 有資格獲得流動性挖礦獎勵。</p><p>點擊開始賺錢「Start Earning」並選擇綁定長度。</p><figure class="image"><img src="https://assets.matters.news/embed/0ac2d6e7-61b8-4c22-bbdb-333b474a13a6.png" data-asset-id="0ac2d6e7-61b8-4c22-bbdb-333b474a13a6" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>當用戶想要停止綁定 LP Token 時，可提交解除綁定期的要求並會出現計時器。當計時器結束後即代表解除綁定，於是用戶可提出提取密碼貨幣的要求。</p><blockquote>【解說】你絕對可以在激勵池提供流動性後不綁 LP Token 的，但我想沒有人會這樣做吧？</blockquote><p><strong>綁定流動性計量器 Bonded Liquidity Gauges</strong></p><p>綁定流動性計量器 Bonded Liquidity Gauges 是將流動性獎勵分配給已綁定 LP Token 時長達到最低要求的機制。OSMO 每日發行量的 45% 用於提供流動性獎勵。</p><p>例如「#1 池 / LP 份額 / 1 週」gauge 將向綁定 #1 池 LP Token 一周或更長時間的用戶分配獎勵。每個用戶收到的金額以他綁定的 LP Token 數量為比例。</p><p>已綁定的 LP Token 可參與不同的 gauge 設定，要獲利只需要達到最低綁定要求。</p><p>從流動性挖礦中獲取的獎勵並不需要被綁定時間限制，用戶可以將獲得的獎勵立即轉讓。只有被綁定的 LP Token 主體受解綁時間限制。</p><blockquote>【解說】每日生出的獎勵可以每日領走，並且可以任意運用。假設你每天都按一按提領然後把獎勵再投進去別的地方或者委託，雖然數量少，但這個複利效應將會很可觀。反正現時提領 OSMO 不需要任何手續實，大家絕對可以靈活處理。</blockquote><p><strong>分配點 Allocation Points</strong></p><p>並非所有流動性池都是激勵池。在 Osmosis，OSMO 委托人選擇通過鏈上治理提案去決定那一個池成為激勵池，並可指明激勵池內每一個 Gauge 可獲得「分配點」Allocation Points 的數量。在每個每日紀元 daily epoch 結束時，新發布 OSMO 的 45% 將按比例分配給每個 Gauge 的分配點。每個 Gauge 獲得的 OSMO 流動性獎勵的百分比計算為它的分配點數除以分配點總數。</p><p>舉例：</p><ul><li>Gauge #3 – 10 個分配點</li><li>Gauge #4 – 5 個分配點</li><li>Gauge #7 – 5 個分配點</li></ul><p>20 個總分配點。在每日 epoch 結束時，Gauge #3 將獲得 50%（20 個中的 10 個）的流動性獎勵。Gauge #4 及 #7 將各自獲得 25%。</p><p>社群可以通過 UpdatePoolIncentives 提案去編輯任何 Gauge 現有的分配點。通過將 Gauge 的分配點設置為零，可以將它從激勵 Gauge 列表中完全刪除。提案更可以設置新 Gauge 的分配點。當添加一個新的 Gauge 時，分配點的總數會增加，從而稀釋所有現有的激勵 Gauge。</p><p>Gauge #0 是一個特殊的存在，獲得的獎勵將直接發送到社群池 Community Pool。將分配點分配給 Gauge #0 方便儲存流動性挖掘獎勵以便日後花費。</p><blockquote>【解說】這個在解釋為甚麼條件相類似的池和 Gauge 能夠獲得的 OSMO 獎勵可以相差那麼多。想知道有甚麼值得玩玩的新池，就要留意議案。此外留意<strong>稀釋</strong>兩字，有機會令現有激勵池獎勵變得越來越少。</blockquote><p><strong>LP Token 獎勵分發與解綁</strong></p><ul><li>綁定 LP Token 可選擇解綁時間長度不同的 Gauge，每一個組合的 APR 實際年利率都有所不同。</li><li>獎勵於每天紀元 epoch 17:00 UTC（香港時區上午 1 時）結束時分發，不是以區塊計算。</li><li>只要你在一個 epoch 內，綁定 LP Token 就可以獲得獎勵，無論你是早還是遲在 epoch 內。</li><li>在同一個激勵池內，你可以同時綁定不同的 Gauge。</li><li>綁定後必須自行解除綁定，不會由系統為你自動解除綁定。</li><li>解除綁定所需時間是由你解除起那天計，不是由綁定那一天開始計。</li><li>解除綁定是不能取消的。</li><li>可將解綁的 x days unbonding 解讀成為一個你會「留下」的承諾，你可以選擇破壞這個承諾隨時強行解綁。不過假如你綁了需要解綁時間長度為 14 天的 Gauge 後即時解綁，你依然需要等待 14 天才能完成解綁，但會變成在第一個星期得到 7 天 Gauge 所配置的獎勵，然後接下來那星期每天得到 1 天 Gauge 配置的獎勵。雖然都是綁了 14 天，但相比起完全不動綁了 14 天 Gauge 然後等到第 15 天後才解綁會少了很多獎勵；為 7 天 Gauge 提前解綁的亦相類似會變成計算幾個 1 天 Gauge 而不是 7 天，故此請緊慎行事。</li><li>你不能解綁部份 LP Token，但你可以選「解綁全部 Unbond All」一次過解綁全體（但有說法是現時設計依然有一些問題，例如你綁了同一個池不同 Gauge 5 次，你就要解綁 5 次，即是多少次 in 就多少次 out。還有就是沒有理由綁一個 1 日 Gauge 就可以一次過同時解綁正在綁的 14 日 Gauge 而沒有後果之類。所以實際情況需要測試一下。）</li><li>只要 LP Token 仍然在 epoch，解綁時仍然可以收到獎勵，直至完全解綁為止。</li></ul><blockquote>【解說】沒錯 14 天 Gauge APR 是吸引，然而要知道在區塊鏈的世界一天經已很長，所以選擇多少天的 Gauge 就看個人需要了。</blockquote><h2>委託 Staking</h2><p><strong>委託 Staking OSMO</strong></p><p>OSMO 持有者可以將他們的 OSMO 委託給驗證人，驗證人將這些 OSMO 添加到他們的可削減債券中。減去驗證人的佣金，委託人可獲得委託獎勵。</p><p>要委託 OSMO，請從列表中選擇一個驗證人。</p><p>輸入要委託予驗證人的 OSMO 數量。已委託的 OSMO 需要 14 天才能取回委託。在用戶啟動取回過程兩週後才可提取。</p><p>委託 OSMO 用於幫助保護網絡，作為對貢獻的交換，OSMO 委託人可獲得委託獎勵及交易費用。</p><p><strong>提出議案 Creating a Proposal</strong></p><p>提出議案需使用 CLI 介面。</p><p><strong>投票 Voting</strong></p><p>OSMO 委託人有資格對治理提案進行投票。瀏覽提案，並使用自己的委託進行投票。</p><p>投票選項有 Yes、No、No with Veto（強烈反對）和 Abstain（棄權），假如出現大量 No with Veto 票，議案保證金會被沒收。而「不投票」不等於 Abstain，沒有足夠票數會導致 quorum 即法定人數不足。</p><blockquote>【解說】Osmosis 的投票選項和 LikeCoin 的選項有點不同。</blockquote><h2><strong>Staking OSMO V.S. Bonding LP Token</strong></h2><ul><li>基本上 Osmosis 每個驗證人的 APR 都是相同的，只是收取佣金的分別。佣金越低，委託回報越高。懇請不要只委託給 TOP 100 強者，也請委託給弱者平衡生態。</li><li>委託回報與綁定一樣，於每天紀元 epoch 17:00 UTC（香港時區上午 1 時）結束時分發，不是以區塊計算。</li><li>委託和解綁雖然都是困著一些資產，但前者是 OSMO，後者是 LP Token；前者有銷售價值，後者能生產出前者卻不能售賣。</li><li>兩者都必須要用戶自行解綁/解除委託，系統不會自動幫你解。</li><li>一但解除綁定/委託後，兩者都是不能取消的。</li><li>OSMO 解除委託期間不能獲得回報，但解綁 LP Token 可獲取回報直至解綁完成。</li><li>OSMO 解取委託限定 14 天，LP Token 所需解綁期限可以自由按照 Gauge 選擇。</li></ul><blockquote>【解說】究竟應該委託還是綁定，如果既委託也綁定比例應該怎樣分配，不同人有不同意見，我嘗時還未看到一種最有力的解說。</blockquote><hr><p>OSMO 能發堀的東西不止那麼少，但這篇先說到這裡，下回再談。</p>
