<p>疫情之下，各國政府紛紛派錢紓困，澳門派 10,000 MOP，香港 5,000 HKD，台灣 5,000 TWD 五倍券等，其實跟各個區塊鏈專案的空投，大同小異。一般人有錢派就開心了，心思都放到錢要怎麼花了，其他的管它那麼多。但相信民主、熱衷區塊鏈的我們，並不以此滿足，至少在歡迎持分者參與的共同體內，程序理應更民主，我們理應做更多。</p><p>因此，我以 LikeCoin 為案例，草擬了一份派錢方案，供社群討論，再進而提案，爭取全民投票通過，付諸實行，讓大家一起參與整個「自己資產自己派」的過程，體驗流動民主的實際運作。</p><h2>LikeCoin 空投方案建議</h2><h3>目標：向全球區塊鏈用戶推廣 LikeCoin</h3><p>從 2017 年立項至今四年以來，LikeCoin 一直面向創作者、記者、媒體與廣大讀者為主。大部分 Likers 使用 LikeCoin 前從未使用甚至沒聽過區塊鏈，因此團隊除了需要介紹專案本身，還需要講解大量相關概念。我本人甚至因此於去年另創立了 <a href="https://decentralizehk.org/" target="_blank">#decentralizehk</a>，以科普區塊鏈和密碼貨幣知識為己任。</p><p>相對來說，對區塊鏈已有一定認識，已生成錢包和持幣的用戶群，反而對 LikeCoin 的認識不深。其實，以 LikeCoin 的 11 萬多錢包數、百萬級交易量、活躍的社群，在區塊鏈專案中絕不失禮，只是團隊專注服務創作者，未能分心向密碼貨幣持有者推廣，造成「人間採用度」和「鏈圈關注度」不對等。</p><p>隨著 ISCN 正式推出，IBC 開通在即，$LIKE 快將上架 Osmosis、Emeris、Sifchain 等去中心交易所，現在是最理想的時機，把潛伏四年的 LikeCoin，介紹給全世界的區塊鏈用戶。</p><h3>受眾：ATOM 與 OSMO 的持有人</h3><p>2019 年上線的 LikeCoin 主鏈使用 Cosmos SDK 作底層，而 <a href="https://www.coingecko.com/en/coins/likecoin" target="_blank">$LIKE</a> 即將上架的兩個主要 DEX（去中心交易所）分別為使用 $ATOM 的 Emeris 和使用 $OSMO 的 Osmosis。無論是為了回饋提供底層技術的社群，還是以用戶重合度為考慮，$ATOM 和 $OSMO 的持有人都是最合理選擇。</p><h3>空投總額：50,000,000 $LIKE</h3><p>以美元計價，約一百多萬，相當於 LikeCoin 發行量的 4.7% 左右。</p><p>空投為期 180 天。首 90 天可全數領取，從第 91 天起，未領取的空投將線性降低，直至第 180 降到 0。</p><p>未被領取的 $LIKE 將回撥到社群基金（community pool），由未來的提案支配。</p><h3>分配方式：平方空投法</h3><p>$ATOM 和 $OSMO 持有人各分配 25,000,000 $LIKE</p><p>為降低貧富懸殊，會取 $ATOM 和 $OSMO 持有額的平方根。</p><p>另外，獎勵質押 $ATOM 和 $OSMO，為生態的安全性出一分力的用戶，質押中的通證會加權 2.5 倍。</p><figure class="image"><img referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>持幣數的採樣時間為 2021.08.18，約於 LikeCoin FoTan 上線的一刻，具體為 Cosmos Hub 的區塊高度 <a href="https://cosmos.bigdipper.live/blocks/7343999" target="_blank">7,343,999</a>，Osmosis 的區塊高度 <a href="https://osmosis.bigdipper.live/blocks/819999" target="_blank">819,999</a>。</p><h3>領取方式：認識 LikeCoin，完成任務</h3><p>以上公式為每個錢包的最高空投分配（100%），由用戶主動完成以下五項任務領取空投，每項佔 20%。</p><p><strong>一、</strong>前往 <a href="https://stake.like.co/" target="_blank">stake.like.co</a>，在 <a href="https://chrome.google.com/webstore/detail/keplr/dmkamcknogkgcdfhhbddcghachkejeap" target="_blank">Keplr</a> 加入 LikeCoin chain。</p><p><strong>二、</strong>註冊一個 <a href="https://app.like.co/" target="_blank">ISCN</a>。</p><p><strong>三、</strong>把部分 $LIKE 質押給一個驗證人。</p><p><strong>四、</strong>在任何一個 LikeCoin 議案投票。</p><p><strong>五、</strong>在 Osmosis、Emeris 與 Sifchain 的指定 $LIKE 流動性池，加入流動性。</p><p># 該方案大量參考 <a href="https://medium.com/osmosis/osmo-token-distribution-ae27ea2bb4db" target="_blank">Osmosis Fairdrop</a>，並非我的原創。當中的平方空投法和任務式領取空投，皆為 Osmosis 的成功經驗。</p><h2>社群治理五部曲：討論、提案、抵押、公投、實施</h2><p>依據 LikeCoin 憲法，以上方案放到社群討論，吸取意見並調整後，如獲得初步支持，可由任一持份者提案。如在兩週內獲得 100,000 LIKE 抵押支持，則會進入為期兩週的投票期，並於通過後退還抵押，議案付諸實行。</p><p>是為讚賞公民共和國的<a href="https://ckxpress.com/liquid-democracy-with-audrey-tang/"
