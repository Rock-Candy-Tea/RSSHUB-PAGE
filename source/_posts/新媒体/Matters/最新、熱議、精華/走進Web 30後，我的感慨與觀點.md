<p>前天晚上在「區塊鏈學習小組」（一位長輩帶著我們一班新手學習的地方）邀請了 多比 及 Shift 兩位長輩來分享 Matters / The Space / Web 3.0 / Blockchain 的觀點和經驗。透過短短幾個小時的分享，讓我對Web 3.0 這回事，既感到慶幸又感到有壓力感。</p><h1>對經濟共享帶來正面作用</h1><p>我所留意到的是，Web3 是能產生經濟共享的主因是各人都付出時間及貢獻出能力來共同建設，而大家的付出亦沒有被「財團」大規範地佔有。</p><p>對於同為創者者的我來說，動力是比從前大了許多，從前在Blog / Facebook /IG 等等平台寫一篇文章或分享一張畫作等情況，在算演法日新用異及間接推向需要廣告費才有流量之下，其實創作心志漸漸被磨蝕了。都真的因為是Matters 的原故，令我重新找回一點點動力。</p><p>創作受讚而獲得Coins 再透過Staking 令財富一點點累積，對創作者來說的確是一種幫助，直白點說，大家都要吃飯，有錢才能吃食，有飯吃才資格創作啊。</p><h2>凝聚有心人</h2><p>多比 在會中有一句說話我很深刻：</p><blockquote>你們有覺得Web3 內的人都比較無私嗎？</blockquote><p>句子大概是以上這句，我猜 多比 的意思是說Web 3 的人大多都比較善良，不較計付出，總會為人著想等等。</p><p>這也是我在這幾個月內的發現，單單說「區塊鏈學習小組」的無私教學已經很不得了，大家都花心機分享著不同資訊，在Blockchain 上的經驗也是會無私地分享。</p><p>在Matters 上的留言也讓我感到大伙兒的熱情，早陣子因小熊出現在 TheSpace 受了熱情歡迎及提交作品到 The Space Web3 Museum 而被受訪邀請分享，感覺都是有點受寵若驚的，因我做這些事情完全不認為會受人回應的（可能在 Web 2 一直都沒遇過熱情的反應）。</p><p>另外 Shift 也提到一個觀點，他說到這一個熊市令水退了，餘下的都是精華。牛市時可能人人都顧著要炒幣，沒多少個真正會留意項目本身，而熊市令炒幣氣氛急退後，項目的精華便更令人看到。這對創作者來說也是福音，畢竟我們可能會需要金錢，但有時候更需要的是大家能關注我們的創作啊。</p><h2>Web 3.0 對我的壓力</h2><p>為何這麼棒的東西會令我有壓力？因為正正是認同這東西有用啊，這裡多一點說明我的看法。</p><p>在處理Web 3 裡的項目時，「人力」是不可或缺，而有心有力的人，可能更難遇見，加上現時Web 3.0 還在起步階段，「人力」好可能是供不應求。</p><p>在深入了解Blockchain 及 Web 3.0 之後，我腦中已經有好幾個項目的構想，但一計劃到要多少人力投入，資金運用怎樣計算等等，已經開始讓我焦慮起來，這也是我的壓力原因。</p><p>在現實生活中，我有自己的生意，生意模式是聚合小店設計師寄賣產品，及管理他們的市場行銷（大家如有到過文創市集，多少應該會了解），目前有三間實體店及Online Store 在管理。這類經營模式已不是新鮮模式，很多寄賣者都清楚了解，不像Web 3 一般因不清楚詳情而不敢起動。他們都清楚的經營模式，但郤沒有做到「有心有力」，很多寄賣者第一天放上貨物後，下次見面時已經是合約到期離場之時，沒多少個能坐下來計劃下經營方針或宣傳方案。</p><p>所以說，在熟悉的環境下也不能出心出力的，在未知的Web 3， 要如何推動夙願呢，我是會有點壓力。</p><h2>未來展望</h2><p>雖說是有壓力，但我也深信Web 3.0 在末來帶來的影響是絕對正面，可能並不能一步登天，但只要細水長流，不斷令更多人了解，「人力」問題是遲早能解決的事。</p><p>期望在有生之可以看到Web3.0 取替 Web 2.0 的一刻👻</p><figure class="image"><img src="https://assets.matters.news/embed/d8ebf12b-a8f3-4024-ac47-a1a791942812.jpeg" data-asset-id="d8ebf12b-a8f3-4024-ac47-a1a791942812" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p>
