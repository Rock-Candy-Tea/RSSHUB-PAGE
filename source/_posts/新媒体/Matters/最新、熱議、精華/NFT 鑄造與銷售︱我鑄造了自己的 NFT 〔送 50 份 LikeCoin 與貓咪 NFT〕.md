<h2>對我來說 NFT 是？</h2><p>剛開始聽到 NFT 時，對他是什麼東西真的是一知半解，拜求 google 大神的指示，得到了<a href="https://zh.wikipedia.org/wiki/%E9%9D%9E%E5%90%8C%E8%B3%AA%E5%8C%96%E4%BB%A3%E5%B9%A3" target="_blank">非同質化代幣</a>的結果，什麼東東霧殺殺，既然對於定義很模糊，不然看看有什麼應用好了，得到的結論大概是，我好像可以也發行一個自己的 NFT ？把我旅行中拍到的照片結合了我畫的貓咪創造出的作品鑄造成 NFT，也許這是目前為止最適合保存這類作品的方式〔Matters及其他寫作的平臺太多還是比較著重在文字〕，有沒有人買？那個就 ... 隨緣好了 ^^！</p><p>後來在<a href="https://www.books.com.tw/products/0010903379?gclid=Cj0KCQjwlOmLBhCHARIsAGiJg7nhEb6bP5z04EKH0BrkUSLR5-OyqyXTqvHKlN9mqn3ZNenm98hT4DoaAqDuEALw_wcB" target="_blank">以太奇襲</a>一書裡，看到了<a href="https://zh.wikipedia.org/wiki/%E8%AC%8E%E6%88%80%E8%B2%93" target="_blank">謎戀貓</a>這款遊戲，可以算是 NFT 遊戲的始祖〔雖然說書中對於 NFT 著墨不多〕，好吧 ... 開始執行貓咪佔領地球 2.0 版之貓咪佔領 NFT 世界〔結果只有佔滿我的頁面〕。</p><hr><h2>鑄造了自己的 NFT</h2><p>想了一陣子後，我終於動手開始鑄造自己的 NFT 囉！先選定了 <a href="https://opensea.io/" target="_blank">OpenSea</a> 平臺，首先需要一個錢包，不論是要鑄造 NFT 或買賣 NFT，都需要一個錢包，因為本來就有在用 MetaMask〔小狐狸錢包〕而 <a href="https://opensea.io/" target="_blank">OpenSea</a> 也支援 MetaMask 所以就直接連上錢包先把個人資料改一改〔其實就是換了一下頭像〕。</p><p>然後參考一些資料，決定在 Polygon 鏈上鑄造我的 NFT，因為除了 gas 比較低而且現在不需鑄造費用外，在以太主網路上，一次只能鑄造一張，Polygon 鏈上一次則可以同時鑄造多個副本，使用上方便很多。</p><p>接下來連上我的錢包，要開始鑄造我的 NFT 囉！</p><blockquote><strong>首先要先建立一個 Collections</strong></blockquote><ol><li>點選頭像。</li><li>選擇 My Collections 進入頁面。</li><li>進入頁面之後點 Create a collection 建立一個新的 Collection 〔收藏夾〕。</li></ol><figure class="image"><img src="https://assets.matters.news/embed/b7b791c5-f134-413e-8b9b-2116bdc7521a.jpeg" data-asset-id="b7b791c5-f134-413e-8b9b-2116bdc7521a" referrerpolicy="no-referrer"><figcaption><span>新建 Collection</span></figcaption></figure><p>其實直接建立一個 NFT 也可以，它會自動幫你生成一個 Collection，不過有一些資訊後來就不能更改了，所以還是先手動新增一個比較好。</p><p>首先要先準備三張圖片，Logo、 Featured image 與 Banner image。</p><p>接下來就是填入一些資料了，Logo 與 Featured image 分別會顯示的位置如下圖。</p><figure class="image"><img src="https://assets.matters.news/embed/38c3b971-f4d6-4c6d-99df-b09a4b127d55.jpeg" data-asset-id="38c3b971-f4d6-4c6d-99df-b09a4b127d55" referrerpolicy="no-referrer"><figcaption><span>新建 Collection</span></figcaption></figure><p>Banner image 則是顯示在 Collection 頁面裡的上方。</p><p>接下來就是填寫一些資料：</p><ul><li><strong>Name</strong>：收藏夾的名稱〔不重覆〕。</li><li><strong>URL</strong>：自定收藏夾網址，這個網址不能重覆，且只能使用小寫英文字母、數字及 "-" 符號。</li></ul><figure class="image"><img src="https://assets.matters.news/embed/f9d0026e-77be-4a85-812e-c4c7760c646f.jpeg" data-asset-id="f9d0026e-77be-4a85-812e-c4c7760c646f" referrerpolicy="no-referrer"><figcaption><span>新建 Collection</span></figcaption></figure><ul><li><strong>Description</strong>：收藏夾的描敘，因為還沒想好所以我就先留空白了〔這一欄可以用 Markdown 語法，這是類似網頁的 html語法，可以加上一些連結、項目符號 ... 等〕，之後想到再來改。</li><li><strong>Category</strong>：可以選擇一個分類。</li><li><strong>Links</strong>：可以填上個人網站連結及有在使用的社群媒體連結。</li></ul><figure class="image"><img src="https://assets.matters.news/embed/d4e2ecff-d877-4ae5-b8a2-1d89fd7e6960.jpeg" data-asset-id="d4e2ecff-d877-4ae5-b8a2-1d89fd7e6960" referrerpolicy="no-referrer"><figcaption><span>新建 Collection</span></figcaption></figure><ul><li><strong>Royalties</strong>：這個是如果有人轉售了 NFT，我可以收到的版權費。</li><li><strong>Your payout wallet address</strong>：如果有填寫 Royalties 就要填入收款的錢包位置。</li><li><strong>Blockchain</strong>：預設在那一個鏈上鑄造，選定了之後在這個收藏夾新增項目的時候，會預設選擇一個 Blockchain，這一項設定好了就<strong>不能更改</strong>囉！</li><li><strong>Payment tokens</strong>：可接收的 tokens，可以選擇對應 Blockchain 的代幣加進入。</li></ul><figure class="image"><img src="https://assets.matters.news/embed/df1fcc5e-ef25-45a4-837b-d723c7c5f9f2.jpeg" data-asset-id="df1fcc5e-ef25-45a4-837b-d723c7c5f9f2" referrerpolicy="no-referrer"><figcaption><span>新建 Collection - 打什麼碼，後面就有貼錢包位置了！！</span></figcaption></figure><p><strong>Display theme</strong>：收藏夾裡排版的方式。</p><p><strong>Create</strong>：資訊都填完之後，點 Ceate 新增收藏夾〔要注意一下如果必填欄位 "*" 沒填就不能新增〕。</p><figure class="image"><img src="https://assets.matters.news/embed/8b74f3a2-beaf-4ea1-84ff-d3c37f7954a9.jpeg" data-asset-id="8b74f3a2-beaf-4ea1-84ff-d3c37f7954a9" referrerpolicy="no-referrer"><figcaption><span>新建 Collection</span></figcaption></figure><p>新建完之後列表就會多一個 <strong>Collection </strong>了</p><figure class="image"><img src="https://assets.matters.news/embed/4eebb918-eafa-4f94-9aa5-f7f95ea25ae3.jpeg" data-asset-id="4eebb918-eafa-4f94-9aa5-f7f95ea25ae3" referrerpolicy="no-referrer"><figcaption><span>新建 Collection</span></figcaption></figure><p>再來就要開始鑄造我的 NFT 囉！</p><blockquote><strong>鑄造 NFT</strong></blockquote><p>進入一個 <strong>Collection </strong>頁面裡，點選 Add item 按鈕，進入鑄造頁面。</p><figure class="image"><img src="https://assets.matters.news/embed/824bd083-6231-4cb3-9bfd-169d9016c17b.jpeg" data-asset-id="824bd083-6231-4cb3-9bfd-169d9016c17b" referrerpolicy="no-referrer"><figcaption><span>鑄造 NFT</span></figcaption></figure><ul><li>首先選擇要鑄造成 NFT 的檔案，頁面上有可選擇的檔案類型，還有檔案大小上限是 100 MB。</li><li>Name：輸入 NFT 的名稱。</li></ul><figure class="image"><img src="https://assets.matters.news/embed/4b27055e-a07f-4a73-b72a-b204e0b05efe.jpeg" data-asset-id="4b27055e-a07f-4a73-b72a-b204e0b05efe" referrerpolicy="no-referrer"><figcaption><span>鑄造 NFT</span></figcaption></figure><p>External Link：這裡可以填上自己的網站連結，我本來想用 Matters 的頁面，但這一格不能輸入 "@" ，我就只好填我的個人網站啦！</p><ul><li><strong>Description</strong>：描述</li><li><strong>Collection</strong>：選擇收藏夾，因為是從 referrerpolicy="no-referrer"><figcaption><span>鑄造 NFT</span></figcaption></figure><p>接下來這些選項我還沒研究有什麼做用，字面上看起來是一些屬性設定，還有解鎖設定。</p><figure class="image"><img Go ... ！</p><figure class="image"><img src="https://assets.matters.news/embed/9def6dfa-6c5e-41ca-b516-57bfee654d0e.jpeg" data-asset-id="9def6dfa-6c5e-41ca-b516-57bfee654d0e"
