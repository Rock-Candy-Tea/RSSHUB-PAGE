<p>因為曾經在補習班教過 <strong>Phonics</strong>， ( 可參考舊作 <a href="https://matters.news/@Koowoo/%E8%86%BD%E9%87%8F%E5%B0%B1%E6%98%AF%E4%BD%A0%E7%9A%84%E8%B6%85%E8%83%BD%E5%8A%9B-%E8%A3%9C%E6%95%99%E5%90%8D%E5%B8%AB-%E5%88%9D%E9%AB%94%E9%A9%97-bafyreid546co45aym2fqvwi3u76fw2bkenylrvuwr3x7hqmbfivkyw3rui" rel="noopener noreferrer" target="_blank">膽量就是你的超能力｜「補教名師」初體驗</a>)，所以後來憑著這個對發音規則整理的公式與功夫，自修了德文、義大利文、西班牙文、法文，甚至後來還可以延伸到對荷蘭文及葡萄牙文的發音判斷。</p><blockquote>在此先說明，這樣發音規則的鍛鍊是可以做到 :</blockquote><ul><li>看到外文可以迅速判斷是哪一種文字語言。</li><li>判斷出來後即可採用相對的發音規則，八九不離十的唸出來。</li></ul><p>不過，唸得出來未必懂意思，對未來的文法學習也沒有直接幫助就是了~</p><p>這套功夫後來也變成了跟老外的哈啦利器。我常常在與外國人換名片時，按照他的<strong>姓 ( Last name / Family name ) </strong>判斷他是哪一國人。例如: 有人姓 Norg<strong>aa</strong>rd，這種 <strong>aa</strong> 的組合只有丹麥或荷蘭會有；Mouri<strong>nho</strong> 這種 <strong>nho</strong> 的組合就是葡萄牙人；Sch<strong>ij</strong>der 中 <strong>ij</strong> 的組合就是荷蘭文；Ke<strong>β</strong>ler 是德國人， <strong>β</strong>是 <strong>ss</strong> 的組合... etc。讓我在與外國客戶洽談時，可以瞬間就拉近彼此距離。</p><p><br></p><blockquote><strong>來跟大家分享一個最有趣的案例:</strong></blockquote><p>我還在前公司服務時，某天有個澳洲客戶來訪。他是我們女主管的客戶，但那位主管除了精通《國貿條規》 <strong>( Incoterms )</strong>之外，她的語言字庫及國際視野是很有限的。因為擔心午餐時冷場，她特別把我們整個負責國外業務的團隊 ( 也只有 3 人) 都找進去， 以免尷尬癌發作。衝著有外賓在就有好料可吃的誘因下，我也就開心的與會。</p><p>這位外賓的名字是 <strong>Andres Runge</strong>。按著正常的英語發音，我們稱呼他 Mr. Runge ( 音似 "<strong>讓舉" </strong>😅~ )</p><p>聊著聊著，聽不出他口中有澳洲腔英文，他看來也非澳洲白人或土著的長相，我好奇他的背景，就大膽地問他的祖先從哪來。他說其實他出生於<strong>智利</strong>，我一聽到這個訊息，馬上再用 <strong>"¿Cómo está usted "</strong> 來問候他，這下他就開心的笑了。</p><p>但是一想，不對呀，若是照西班牙語發音，Runge 就不能唸 <strong>"讓舉"</strong>，就得唸成 <strong>"論黑" </strong>了耶~ ( "<strong>論" </strong>還要彈舌哦~ 😅 )</p><p>他一聽，大為驚訝，從來沒有人在知道他的來歷之後，馬上對他的<strong>姓</strong>做出不同語言版本的反應。於是，他又說了一個秘密: 「其實我爸爸是德國人~」</p><p>這下更有趣了 ! Runge 不能唸 <strong>"讓舉" </strong>或有彈舌的 <strong>"論黑"</strong> ，要唸成 <strong>"混個"</strong> 了! ( <strong>"混"</strong> 要有 "喉音" ，類似<strong>德文</strong>與<strong>法文</strong>對<strong> r</strong> 的發音一樣~)</p><p>這一趟 <strong>"Mr. Runge"</strong> 發音的追本溯源下來，不但讓我好好 show off 了一下，也在無傷於對客戶的禮節下，讓他對自己的生平經歷做了更多的介紹，我們也因此可以把客戶當朋友一樣的聊天，更有助於深化日後雙方的合作。</p><p><strong>最後想問的是: Runge 先生，你到底是哪一國人? </strong>😅<strong> ( 屬人主義: 德國；屬地主義: 智利；或已投資移民 澳洲?)</strong></p><p>附上 Runge 先生的帥照。因肖像權之故，他的帥就讓大家自行想像囉~</p><figure class="image"><img src="https://assets.matters.news/embed/51f30284-dd38-4e77-9c20-7f7727e4c145.jpeg" data-asset-id="51f30284-dd38-4e77-9c20-7f7727e4c145" referrerpolicy="no-referrer"><figcaption><span>Runge 先生是真的很帥</span></figcaption></figure><p><br></p>
