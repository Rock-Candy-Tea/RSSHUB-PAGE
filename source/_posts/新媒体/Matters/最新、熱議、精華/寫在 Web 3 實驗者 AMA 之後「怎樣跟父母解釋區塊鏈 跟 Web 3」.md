Denken" data-user-name="denkeni" data-id="VXNlcjo0NTg">﻿<span>@花椰菜 Denken</span>﻿</a> 同場傾談。 當中有一段問題「怎樣跟父母解釋區塊鏈跟 Web 3」。</p><p>我看到 Ali 的問題後，就搶先要求主動回答，當中的主要原因是....... 我發覺，我是大家的父母的年紀。</p><figure class="image"><img src="https://assets.matters.news/embed/2fdfcdef-8342-4125-a6d1-46f21ec9225a.png" data-asset-id="2fdfcdef-8342-4125-a6d1-46f21ec9225a" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p>我有幾個不同的新手群，有的是幣圈項目 Steem, Matters, LikeCoin, Loop，也有一些是非項目的「區塊鏈學習小組」。幣圈的新手群，內裡的朋友，普遍年紀都比我少，但也有兩三位在我之上；不過，在非幣圈項目的「區塊鏈學習小組」，就絕大部份朋友的年紀都在我之上，年紀最大接近八十歲。</p><p>他們不單在 Whatsapp 群內查問，甚至也特意來到我公司，要我一步一步教他們設立 Matters 帳戶、下載 Liker Land、設立 Metamask 及 Keplr 錢包、與及上架 NFT。</p><p>朋友們每一次都是相當興奮，不單是我這一個年紀很興奮，我有幾位七十歲以上的朋友，他們也相當興奮，特別是可以將家人的照片製成NFT。其實，對這一個群體來說，賺幣已不是很重要，反而希望藉著「區塊鏈做永久記錄」。</p><p><strong>我最常聽到長輩查詢的一句話：「我這一個想法，可不可以用區塊鏈來實現？」</strong>不同年齡群對區塊鏈的期盼也不一樣，我很少在這一個群體「談幣價」，反而常常跟他們說：「將生活區塊鏈化。」例如：試試改用 NFT 或 Capture <em>(台灣 Number Protocol 的項目)</em> 作為相簿記錄，試試使用 Matters 來取代 Facebook，試試改用 Wirex Visa 取代日常使用的 Credit Card。</p><p>我不是在改變長輩們 (哈..... 我也有份) 對加密貨幣 或 NFT 的負面觀念，我其實在引發這一個群體的主動性，讓他們覺得自己能「駕御」Web 3 的應用項目，不用擔心被年青一代拋離，因為 Web 3 的應用將會越來越容易使用。現時使用有點困難，並不是長輩們的問題，而是開發人員仍在底層技術上下苦工，當區塊鏈底層技術越來越穩定時，頂層的應用 UI / UX 就會「急速配合」。</p><p>我十分感謝 Matters <a class="mention keychainify-checked"
