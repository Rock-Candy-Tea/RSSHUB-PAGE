<p>之前為了參加 Matters（馬特市）<a href="https://matters.news/@hi176/nft-%E7%99%BB%E8%A8%98%E7%A9%BA%E6%8A%95-%E5%A6%82%E4%BD%95%E7%8D%B2%E5%BE%97-traveloggers-%E7%A9%BA%E6%8A%95-%E5%BF%AB%E4%BE%86%E7%B6%81%E5%AE%9A%E9%8C%A2%E5%8C%85-bafyreigvypuhddpehillleaw73tzupcklixpw3raj5kfz5227p57v2rvpq" rel="noopener noreferrer" target="_blank">Travelogger 空投</a>，才首次註冊了 MetaMask（小狐狸錢包），非常幸運地獲得空投啦。</p><p>在研究 MetaMask 的過程中，我發現眾多教學都是使用 MetaMask「電腦瀏覽器的附加元件」。然而瀏覽器附加元件，總是索取超多權限：</p><figure class="image"><img src="https://assets.matters.news/embed/6227dd0a-1946-441e-9b65-655e710504aa.jpeg" data-asset-id="6227dd0a-1946-441e-9b65-655e710504aa" referrerpolicy="no-referrer"><figcaption><span>瀏覽器附加元件，總是索取超多權限……</span></figcaption></figure><p>你看看哪，「存取您所有網站中的資料」這樣大的權限，如果這安裝在你平常慣用的瀏覽器，豈不是能看遍所有瀏覽過的網站資料？</p><p>當然，你可以在不用時就關閉附加元件，或是安裝另一個瀏覽器，專供錢包連接 Web 3 網站之用，但這些額外步驟總是麻煩得多。</p><p>此外，通常我們比較會在傳統電腦上，安裝未經審核驗證過的第三方應用程式，傳統電腦作業系統（Windows、macOS）的權限控管通常也比較鬆，被駭的風險性可能比手機作業系統更高。</p><p>因此，我改為安裝了 MetaMask 手機版 App，並發現有內建瀏覽器！這就是我當初用來綁定 Travelogger 空投的方式啦。在這個 MetaMask App 內建瀏覽器中，就可以把玩馬特市的幾個 Web 3 功能：</p><figure class="image"><img src="https://assets.matters.news/embed/4d7883fa-44f1-476c-8a1b-b8d37c277f48.png" data-asset-id="4d7883fa-44f1-476c-8a1b-b8d37c277f48" referrerpolicy="no-referrer"><figcaption><span>在這個 MetaMask App 內建瀏覽器中，就可以把玩馬特市的幾個 Web 3 功能</span></figcaption></figure><p>可以上 OpenSea 購買 <a href="https://traveloggers.matters.news/zh/" rel="noopener noreferrer" target="_blank">Travelogger</a>、領取與瀏覽 <a href="https://app.poap.xyz/" rel="noopener noreferrer" target="_blank">POAP</a>、新用戶也可以<a href="https://matters.news/@hi176/%E5%8A%9F%E8%83%BD%E6%9B%B4%E6%96%B0%E9%80%9A%E5%91%8A-matters-%E6%94%AF%E6%8F%B4%E5%8A%A0%E5%AF%86%E9%8C%A2%E5%8C%85%E8%A8%BB%E5%86%8A%E8%88%87%E7%99%BB%E5%85%A5-bafyreieb5kcr2cbghrt4xeiyvjc7zuzxqwfje3qvq5v4hjpwlmgjjq7ybq" rel="noopener noreferrer" target="_blank">用錢包來登入馬特市</a>囉！</p><p>簡單小結比較一下 MetaMask 兩個方案的差別：</p><ol><li>傳統電腦的桌面瀏覽器附加元件：<br class="smart">好處：主流用法，網路教學多，能裝在慣用瀏覽器<br class="smart">壞處：隱私權限和被駭風險可能較高<br class="smart"></li><li>智慧型手機 App 內建瀏覽器：<br class="smart">好處：用作 Web 3 的專屬瀏覽器，安全與隱私性較高<br class="smart">壞處：內建瀏覽器介面較不熟悉，需要稍花時間上手</li></ol><p>舉一反三的你，或許會問：不能直接在手機瀏覽器上，安裝附加元件嗎？答案是：理論上可以，只是 MetaMask 手機版 app 沒實作瀏覽器附加元件功能（<a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E7%82%BA%E9%A6%AC%E7%89%B9%E5%B8%82-nft-%E8%A8%88%E7%95%AB%E5%81%9A%E6%BA%96%E5%82%99-02-meta-mask-%E5%B0%8F%E7%8B%90%E7%8B%B8%E9%8C%A2%E5%8C%85-bafyreih6hktykp5bzxlp373aph5ck3wvyoba4z6ex4s5ccoelirqxqiyva#Q29tbWVudDo4OTI2Njg-Q29tbWVudDo5MDI3MzQ" rel="noopener noreferrer" target="_blank">首次回應出處</a>）。</p><h2>後記</h2><p>由於 MetaMask 是開放原始碼，我還發現 2018 年時眾人曾討論過，當初 MetaMask 附加元件是為了兩個新功能，才把權限追加為「存取您所有網站中的資料」（<a href="https://github.com/MetaMask/MetaMask-extension/issues/4978#issuecomment-411431427" rel="noopener noreferrer" target="_blank">出處</a>）。這就是瀏覽器的權限控管，比較缺乏彈性之處了，沒辦法逐次按照特定需求，才去開啟所需權限。</p><p>其實在選用 MetaMask 作為以太坊錢包時，就已經是用它來產生或匯入私鑰了，使用者可以說是得「完全信任」這個軟體。只不過 Web 3 區塊鏈世界瞬息萬變，儘管 MetaMask 小狐狸錢包近期獲得大家信任，這篇文章也推薦在最新版的手機作業系統上用 MetaMask App 內建瀏覽器，還可能有什麼風險呢？</p><p>我預計未來會發表更多 Web 3 的實驗研究心得，歡迎先追蹤我的帳號！</p>
