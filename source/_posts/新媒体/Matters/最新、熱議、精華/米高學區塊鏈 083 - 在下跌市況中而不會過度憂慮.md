src="https://assets.matters.news/embed/8ec7c527-1d3a-45ef-b451-b7c87dc88188.png" data-asset-id="8ec7c527-1d3a-45ef-b451-b7c87dc88188" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p>在區塊鏈圈裡，貨幣價格實在大起大落，常常令我茶飯不思。我一直也想找一個方法，可以令我減去這種「煩惱」。</p><p>回想這幾年幣圈生活，最輕鬆自在是未有 Defi 的日子，單單純純只是依靠「寫字儲幣」。當時不太需要理會幣價升跌，只在乎每天所得的 Token 是否比昨天多? 仍有沒有方法可以獲得更多的 Token?</p><p>慶幸我是一種不太懂得投資的人，過往的心態是一種「賺幣」心態，而不是投機投資心態。所以，不同的升升跌跌對我的錢包傷害不深，反而讓我找到一些適合我的「區塊鏈生活模式」。</p><p>我現時的「區塊鏈生活模式」主要是五部份：</p><p><strong>第一部份：日常在區塊鏈寫字</strong></p><p>這已成為我的一種習慣，有時一天一至兩篇，最長也只是相隔兩天。內裡上，我沒有刻意佈局，只是想將當日所想的事情記錄下來，跟著，腦筋就可以清空一下，返回日常繁忙的工作。</p><p>這樣的寫字，到底一天會收到多少 Token 呢?</p><p>現時，我只在兩處地方 Steem 與 Matters，加起來大約是七美元左右的 Token。金額並不多，因為我已將大部份鎖倉的 Steem 取出來放在 Cosmos 生態裡質押。</p><p>若有使用 Steem 的博主也知道，手上鎖倉的 Steem Power (SP) 越多，收入也會相應增加。我手上仍有大約 10000個Steem，市值大約3100美元，每天可以為我製作多 3 至 4美元的 Steem Token。(日化利率 0.1%)</p><p><strong>第二部份：Wirex VISA內的港幣/加密貨幣儲蓄</strong></p><p>我們現時已從 Credit Card 模式改變為使用區塊鏈信用卡 Wirex 模式，透過 Token 作為日常生活開銷的簽帳，並且用來作為「港幣及加密貨幣的儲蓄」，年回報比於銀行高。</p><p>另外，我是使用Wirex VISA做「每週定投以太幣」，每週從Wirex的法定貨幣戶口，固定買入相等金額的以太幣。剛剛開始了 17週，預算會做200週 或 下一個牛市。</p><figure class="image"><img src="https://assets.matters.news/embed/0fd764a0-6dc1-49f8-b77d-223f683d9069.png" data-asset-id="0fd764a0-6dc1-49f8-b77d-223f683d9069" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p><strong>第四部份：在DEFI平台組LP收息</strong></p><p>DEFI平台最大好處，就是可以讓不同人士選擇適合自己風險程度的投資。有人會做借貸賺息，不過，我比較喜歡選擇一些與「穩定幣」配對的流動池。</p><p>當然內裡會有無償損失，不過，無償捐失主要是考慮「賺少了錢」。賺多賺少實在是未知數，不過我手上的本金卻是「已知」。所以，我比較喜歡選擇一些「保本增值」的流動池。特別是 Osmosis 上的 OSMO / UST 流動池。</p><figure class="image"><img src="https://assets.matters.news/embed/1701ed2f-37f0-4ca8-95b8-5e2d810cd676.png" data-asset-id="1701ed2f-37f0-4ca8-95b8-5e2d810cd676" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p><strong>第五部份：玩一玩GameFi</strong></p><p>雖然，我對 GameFi 抱著極大失望，但是我仍會繼續參與一些項目，始終我自己是遊戲行業工作了很長時間，也想以玩家身份透過參與 GameFi 了解 Web 3 世界的遊戲發展。</p><p>不過，我現時的投入程度已大不如前，只是簡單做一些 Staking，當當礦工，當當商人，已失去了爭排名榜的熱度。</p><figure class="image"><img src="https://assets.matters.news/embed/29530e6b-2e43-4cf3-97a2-3a34feec1471.png" data-asset-id="29530e6b-2e43-4cf3-97a2-3a34feec1471" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p>現時，我以上五組行為是我主要的部份，我機乎是沒有做加密貨幣的炒賣。雖然每一部份都會受到大市波動的影響整體的金額，當中也會有不開心狀況；不過，我心中知道，以上各部份每天都有 Token 回報，並且是「細水長流」的日增長方式。如此，就可以做到「以時間填補大市下跌的虧損」的可能性。</p><p>在阿Q精神的自我安慰下，慢慢就減少加密貨幣波幅對我手上Token的回報影響及憂慮。</p>
