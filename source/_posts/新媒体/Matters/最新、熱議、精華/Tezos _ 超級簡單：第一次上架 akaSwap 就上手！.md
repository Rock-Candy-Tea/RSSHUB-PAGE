rel="noopener noreferrer" target="_blank">銀包</a> 安裝好，<a href="https://matters.news/@daisy/295093-tezos-%E5%9C%A8-kraken-%E8%B2%B7-xtz-%E7%82%BA%E4%B8%8A%E6%9E%B6-aka-swap-%E5%81%9A%E6%BA%96%E5%82%99-bafyreiej3fhek2f7opiyaohngdismip7uj3pqjrvjcrlv5ii2bsupuigdy" rel="noopener noreferrer" target="_blank">錢又到手</a>，是時候繼續我的 <a href="https://akaswap.com/" rel="noopener noreferrer" target="_blank">akaSwap</a> 上架之路。</p><p>我打算把我在 Opensea 的圖上架到 akaSwap 看看有沒有人賞識。不過在 Opensea 的圖原來並不能「搬字過紙」直接將圖片挪去 akaSwap 使用，因為他們會當作是偷圖的，所以需要把圖變化一下。</p><p>大佬 <a class="mention" href="https://matters.news/@swiftevo" target="_blank" data-display-name="swiftevo" data-user-name="swiftevo" data-id="VXNlcjo4MjI3NA">﻿<span>@swiftevo</span>﻿</a> 介紹使用 <a href="https://deepdreamgenerator.com/" rel="noopener noreferrer" target="_blank">Deep Dream Generator</a> 令圖產生變化就可以放到 akaSwap 了。原來這種做法有個學名叫做「風格轉移」( Image Style Transfer )，更有中學生們（不知道是那一家學校）寫了一篇 <a href="https://www.shs.edu.tw/works/essay/2017/11/2017111511080338.pdf" rel="noopener noreferrer" target="_blank">功課</a> 去介紹呢。</p><h2>上架 NFT 到 akaSwap</h2><p>1. 到 <a href="https://deepdreamgenerator.com/" rel="noopener noreferrer" target="_blank">Deep Dream Generator</a> 將 Opensea 的圖做風格轉移，步驟太簡單了所以不多說，就算不搞 NFT 都可以去玩玩看，很有趣的。</p><figure class="image"><img src="https://assets.matters.news/embed/dc3d962b-fc08-4c29-90e3-d73b3272c7df.png" data-asset-id="dc3d962b-fc08-4c29-90e3-d73b3272c7df" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>2. 弄好圖以後，到 akaSwap 點右上角「Connect Wallet」。</p><figure class="image"><img src="https://assets.matters.news/embed/ebd09f2c-e58f-4b42-98fe-9a197dce772b.png" data-asset-id="ebd09f2c-e58f-4b42-98fe-9a197dce772b" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>3. 彈出視窗，選 Temple。當然如果你使用其他錢包就可以選其他的。</p><figure class="image"><img src="https://assets.matters.news/embed/a496633a-cdd2-468c-a739-e41254d77138.png" data-asset-id="a496633a-cdd2-468c-a739-e41254d77138" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>4. 出現確認連線頁面，選「連結」。</p><figure class="image"><img src="https://assets.matters.news/embed/ffdb2f6d-da10-4da2-a1af-aa9a3aa3e936.png" data-asset-id="ffdb2f6d-da10-4da2-a1af-aa9a3aa3e936" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>5. 在右上角看到你的錢包地址即是經已將 Temple 錢包連結好 akaSwap 了。然後到菜單選「Create -> NFT」。</p><figure class="image"><img src="https://assets.matters.news/embed/010a6780-e57a-4485-821b-8f0bcd40a413.png" data-asset-id="010a6780-e57a-4485-821b-8f0bcd40a413" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>6. 看到輸入資料的頁面。最上的部份讓你將你的 NFT 上載 ( Upload ) 上去，支援的格式有 bmp, gif, jpeg, png, svg, webp, mp4, ogv, quicktime, webm, glb, gltf, mp3, oga, pdf, x_directory, html (zip archive)。而你亦需要同意說這個作品是你原創的。</p><figure class="image"><img src="https://assets.matters.news/embed/0943f738-7174-4a27-aeba-21cf24a6e61d.png" data-asset-id="0943f738-7174-4a27-aeba-21cf24a6e61d" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>接下來要填寫關於這個 NFT 的資料：</p><ul><li>Title - 作品的名稱，必須填寫，最多不超過 100 個字元</li><li>Description - 作品簡介，必須填寫，最多不超過 5000 字元</li><li>Tag(seperated by commas) - 作品的標籤，每個標籤以「,」分隔</li><li>Amount - 數量，必須填寫，就是你這個 NFT 打算鑄造多少份，最多不多於 10,000 份）</li><li>Royalty (10-25%) - 版權費比率，必須填寫，這是假設如果這個 NFT 被轉手，每一次轉手原作者會分到多少版權費比率</li><li>Apply royalty sharing - 版權費分享，是指如果作品有共同作者，是不是將版權費與他/她分享</li><li>Use frame - 就是為作品加個框</li></ul><p>填好後點「Preview」看看是否 OK。</p><p>7. 不 OK 的話可以按上面「Back」回去修改內容。假如覺得沒有問題就按「Mint [數量] AakaOBJs」。</p><figure class="image"><img src="https://assets.matters.news/embed/433c88c7-63c6-4461-8da2-95fe332c5046.png" data-asset-id="433c88c7-63c6-4461-8da2-95fe332c5046" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>8. 畫面會展示 In progress 並彈出 Temple 視窗說明製作這個 NFT 所需要的各種手續費，查看完畢後點「確認」。</p><figure class="image"><img src="https://assets.matters.news/embed/9bcfe936-61e6-44c6-9e46-ad06a1ca3198.png" data-asset-id="9bcfe936-61e6-44c6-9e46-ad06a1ca3198" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>9. 接下來系統會把你彈回主頁。到右上角點錢包並點「My Page」吧。</p><figure class="image"><img src="https://assets.matters.news/embed/5659d3b3-b0c4-4bf1-a1f6-b60b9946e706.png" data-asset-id="5659d3b3-b0c4-4bf1-a1f6-b60b9946e706" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>10. 你會看到你的搞的 NFT 就在那兒等你。</p><figure class="image"><img src="https://assets.matters.news/embed/08154992-4366-4aaf-9d87-5a45d8e05fea.png" data-asset-id="08154992-4366-4aaf-9d87-5a45d8e05fea" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>11. 到 <a href="https://tzkt.io/" rel="noopener noreferrer" target="_blank">https://tzkt.io/</a> 可以查看整個 NFT 製作過程，弄一下都只是扣你 0.05 + 0.03 TEZ（通證代號正式是 $XTZ）甚麼的，便宜到發笑！</p><figure class="image"><img src="https://assets.matters.news/embed/24eb7217-e3d0-4445-9b42-1bab4f61d143.png" data-asset-id="24eb7217-e3d0-4445-9b42-1bab4f61d143" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>12. 回到 NFT 頁面可以看到寫看 Not for Sale
