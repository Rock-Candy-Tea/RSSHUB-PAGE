LikeCoin 給驗證人，委託給任何一個驗證人也是可以的。</p><p>2. 到 <a href="https://stake.like.co/" target="_blank">stake.like.co</a> 選擇 Keplr Browser Extension 或 Authcore 登入，以下選擇 Authcore 作示範。</p><figure class="image"><img referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/68d1ddc7-b2cd-4270-8880-a51f7e265c86.png" data-asset-id="68d1ddc7-b2cd-4270-8880-a51f7e265c86" referrerpolicy="no-referrer"><figcaption><span>依照平時的方式登入</span></figcaption></figure><p>3. 登入後，到左手邊的菜單選擇「Proposals」，你會看到<a href="https://stake.like.co/proposals/16" target="_blank">議案 16</a>，點選它。</p><figure class="image"><img src="https://assets.matters.news/embed/b49caf46-a991-497f-ad31-e231997b238a.png" data-asset-id="b49caf46-a991-497f-ad31-e231997b238a" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>4. 在右上角，你會看到「Vote」，點選它。</p><figure class="image"><img referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>5. 右下角會出現四個投票選項：Yes（支持）、No（反對）、No with Veto（強烈反對）、Abstain（棄權）。請選擇其中一項，完成後，點「Next」。</p><figure class="image"><img referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>8. 這時候，系統會顯示你的指令正下達，請等待投票完成並不要關掉這個畫面。</p><figure class="image"><img src="https://assets.matters.news/embed/f6f7a3ee-f2c8-45ad-83da-c8e0d8374aad.png" data-asset-id="f6f7a3ee-f2c8-45ad-83da-c8e0d8374aad" data-asset-id="3b8827aa-9af4-4fe6-a42e-2f4ac2f9f56c" referrerpolicy="no-referrer"><figcaption><span>
</span></figcaption></figure><p>10. 點選左手邊「Transactions」，會看到你投過票的紀錄。</p><figure class="image"><img src="https://assets.matters.news/embed/b30c1463-add4-445f-a993-9e89da4ede55.png" data-asset-id="b30c1463-add4-445f-a993-9e89da4ede55" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>投票邏輯</h2><ol><li>沒有委託的 LikeCoin 仍然可以自由使用，不受投票影響</li><li>只要已委託 LikeCoin 予任何一個驗證人，就算驗證人的投票選擇和你不一樣，例如你投 Yes 他投 No，仍然會以你的投票選擇為依歸。</li><li>假設你委託了給驗證人但驗證人沒有投票，就會以驗證人的取向為該議案的取向。例如你的驗證人沒有投票，那麼你在議案中也一樣沒有表過態。</li><li>假如就同一議案投票超過一次，將以最後一次投票的選擇為最終選擇。</li><li>現時只有驗證人的投票有頭像，其他持份者的投票仍不能顯示頭像。</li><li>投票結果如何界定，可參看 LikeCoin 大百科關於 <a href="https://docs.like.co/v/zh/user-guide/background#a839" target="_blank">投票</a> 的章節 <a href="https://matters.news/@likecoin/like-coin-%E6%B2%BB%E7%90%86%E8%AD%B0%E6%A1%88%E6%B5%81%E7%A8%8B%E7%B0%A1%E4%BB%8B-bafyreigjlcrzlfall5jduxigs2sh7miciqzt553w4fiipnggo73oqhv4nu" target="_blank">LikeCoin 治理議案流程簡介</a> 或 <a href="https://docs.cosmos.network/master/modules/gov/01_concepts.html" target="_blank">Cosmos SDK 的文檔</a>。</li></ol><p>自己的票自己投，想 LikeCoin 生態圈變得更好，請投下你神聖的一票吧！</p>
