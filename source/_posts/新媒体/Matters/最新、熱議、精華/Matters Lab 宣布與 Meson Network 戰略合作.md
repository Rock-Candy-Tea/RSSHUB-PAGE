referrerpolicy="no-referrer"><figcaption><span>Matters Lab 宣布與 Meson Network 戰略合作</span></figcaption></figure><p>Matters Lab 正在建立更為去中心化的社交與內容圖譜，令用戶在從內容創作到分發的全過程中，都可以擺脫平台，完全掌握自己的創作與社交數據，並更容易在不同平台之間遷移、共享。</p><p>針對去中心化網絡中的數據存儲，目前已有許多可選的方案。比如對於內容地址的標準 IPFS，我們既可以用中心化的服務來 pin，比如 Pinata, Infura 和 matters.news；也可以通過去中心化的激勵機制來 pin，比如 Filecoin, Arweave 和 Crust。</p><p>但是對於去中心化網絡中的數據讀取，或者說傳輸數據需要的帶寬，目前尚沒有通用的基建。當一個用戶想從 IPFS 網絡中調取由 Matters 發表的內容時，ta 需要藉助 IPFS 的公共網關（gateway）。這些網關由社區志願維護，缺乏長期激勵，因而不易持續。這和區塊鏈面對的問題類似：當用戶想要獲取 Traveloggers 保存在以太坊上的內容時，ta 需要經由如 Infura 這樣的服務商，而這樣的服務商本身是中心化的。</p><p>去中心化世界的帶寬問題是 Meson Network 想要解決的問題。Meson Network 讓礦工通過貢獻帶寬獲取代幣，形成一個去信任的內容傳遞網絡（CDN）。應用層則可以利用 Meson Network 維護的路由表為用戶找到最近最快的節點，以可持續的方式提升用戶體驗。</p><p>Matters Lab 榮幸地宣布與 Meson Network 的合作，完善我們基建上重要的一環。作為第一步，我們將 Meson Network API 整合進了我們的 IPFS 彈窗中。當用戶需要從 IPFS 中直接獲取內容時，ta 可以通過 Meson Network API 重定向到最近的節點，加速內容的調用。</p><figure class="image"><img src="https://assets.matters.news/embed/00381549-f135-4b62-b8fe-110a8e3068c4.png" data-asset-id="00381549-f135-4b62-b8fe-110a8e3068c4" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>在 Matters Lab 將底層社交與內容圖譜繼續去中心化的過程中，會有更多帶寬需求。我們很高興能與 Meson Network 團隊一起前行，一同為每個用戶，建立一個真正去中心化的社交與創作協議。</p><p>歡迎評論你對新功能的想法，以及暢想未來會有哪些新的玩法！</p><h2>關於 Matters Lab</h2><p>Matters Lab 成立於 2018 年，致力於搭建 Web3 環境中，更自由與公平的創作者生態。其中，包含使用去中心技術工具，完成數字權利的確認、治理制度的設計、創作者經濟模型搭建。</p><p><a href="https://matters.news/" rel="noopener noreferrer" target="_blank">Matters.News</a> 是 Matters Lab 的第一站，它是 Web3 世界最大的、開源的去中心化出版平台，已有 超過 8 萬名註冊寫作者。<a href="https://traveloggers.matters.news/" rel="noopener noreferrer" target="_blank">Traveloggers</a> NFT 計畫是 Matters Lab 的第二站，目標是探索創作者經濟模型。Matters Lab 還即將推出 The Space 計畫，致力於以激進市場法則探索公共治理。</p><h2>關於 Meson Network</h2><p><a href="https://meson.network/" rel="noopener noreferrer" target="_blank">Meson Network</a> 致力於在 Web3 上打造一個高效的帶寬市場，使用區塊鏈協議模型取代傳統的人力為基礎的銷售模式，高效整合長尾用戶的閒置帶寬並服務用戶。 Meson 是去中心化存儲、計算和新興 Web3 dapp 生態系統數據傳輸的基礎。</p>
