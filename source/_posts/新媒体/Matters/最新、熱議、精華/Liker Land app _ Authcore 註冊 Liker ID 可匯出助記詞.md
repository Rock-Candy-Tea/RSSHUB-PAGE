Keplr</h2><p>這個話題在 <a href="https://matters.news/@daisy/193654-like-coin-%E6%94%BE%E5%9C%A8-keplr-%E9%82%84%E6%98%AF-liker-land-%E6%AF%94%E8%BC%83%E5%A5%BD-bafyreid4sng4t7pdywrrp3z2pywmdd2mgo5hx4fd2eqhbottd6kd32rnly" rel="noopener noreferrer" target="_blank">LikeCoin 放在 Keplr 還是 Liker Land 比較好？</a> 和 <a href="https://matters.news/@daisy/246099-%E7%B5%A6%E9%A6%AC%E7%89%B9%E5%B8%82%E6%9C%8B%E5%8F%8B%E7%9A%84-like-coin-%E9%8C%A2%E5%8C%85%E5%B8%96-bafyreigycq2tex4qduukpzglcoqjvez6ojzcbr4ewa7vyeelv33awowyd4" rel="noopener noreferrer" target="_blank">給馬特市朋友的 LikeCoin 錢包帖</a> 都有寫過。</p><p><strong>Liker Land app (Authcore)</strong></p><p>Liker Land app 是身兼：</p><ol><li>錢包（轉帳、收款，委託 LikeCoin）</li><li>LikeCoin 生態系統內容閱讀器</li><li>創作者所獲讚賞儀表版（即是查看別人為自己的文章拍手所得到的 LikeCoin）</li><li>讚賞公民月費回饋明細儀表版</li></ol><p>的手機應用程式。</p><p>新用戶在 <a href="https://liker.land/getapp" rel="noopener noreferrer" target="_blank">Liker Land app</a> 或 Matters 加入 LikeCoin 生態，預設是使用 <a href="https://docs.like.co/v/zh/user-guide/liker-id/register" rel="noopener noreferrer" target="_blank">Authcore</a> 登入管理方案註冊。你會得到一個 Liker ID 及一個由 cosmos1 開頭的 <a href="https://docs.like.co/v/zh/guides/wallet/wallet-address" rel="noopener noreferrer" target="_blank">LikeCoin 錢包地址</a>。這個錢包地址收藏得比較深入，你需要在 Liker Land app 中 <a href="https://docs.like.co/v/zh/guides/wallet/big-dipper" rel="noopener noreferrer" target="_blank">按 BigDipper</a> 或登入 <a href="https://like.co/in/settings" rel="noopener noreferrer" target="_blank">Liker Land 網頁版</a> 才可以看到。</p><p>在 5 月 4 日 LikeCoin chain 升級後，cosmos1 錢包地址會轉變為 like1 錢包地址，而這個轉變對於 Liker Land app 用戶來說應該可能就是安裝一個新版本的 app 就搞定。</p><p>還有就是循 Authcore 這條路徑的話，你不可以不註冊 Liker ID 而得到一個 LikeCoin 錢包地址。</p><p>要為議案投票可使用 <a href="https://dao.like.co/welcome" rel="noopener noreferrer" target="_blank">dao.like.co</a> 的 Authcore 登入。</p><p><strong>Keplr</strong></p><p>使用 Keplr 錢包沒有硬性規定你一定要以它註冊一個 Liker ID，你大可以使用 Keplr 來管理你的 LikeCoin 並且與  Liker ID 亳無關聯。</p><p>現時 Keplr 的 LikeCoin 錢包地址是 cosmos1 開頭並同樣會在 5 月 4 日 LikeCoin chain 升後變為 like1 開頭。根據近日 <a href="https://www.forbole.com/blog/how-to-claim-evmos-airdrop-tc-2" rel="noopener noreferrer" target="_blank">Evmos 空投</a> 的錢包地址體驗，估計使用 Keplr 的用戶需要做一些額外操作才可以由 cosmos1 轉為 like1。</p><p>使用 Keplr 錢包操作 LikeCoin 只支援桌面電腦，你也不可以使用以 Keplr 錢包註冊的 Liker ID 登入 Liker Land app。</p><p>你可以使用 Keplr 錢包搭配 <a href="https://dao.like.co/welcome" rel="noopener noreferrer" target="_blank">dao.like.co</a>、<a href="https://likecoin.omniflix.co/" rel="noopener noreferrer" target="_blank">Omniflix</a>、<a href="https://wallet.bitsong.io/" rel="noopener noreferrer" target="_blank">BitSong</a>、<a href="https://ping.pub/" rel="noopener noreferrer" target="_blank">ping.pub</a> 去進行轉帳、收款，委託及投票，並使用 <a href="https://likecoin.bigdipper.live/" rel="noopener noreferrer" target="_blank">BigDipper</a> 及 <a href="https://atomscan.com/likecoin" rel="noopener noreferrer" target="_blank">ATOMScan</a> 查看交易明細。而上面介紹 Liker Land app 的功能  2. 3. 4. Keplr 是不支援的。</p><h2>匯出助記詞後的改變</h2><ul><li>Liker Land app 能夠匯出私鑰，就算所有電郵及社交帳號被封殺，也可以不受影響地管理 LikeCoin，這一點非常重要。</li><li>使用 Liker Land app 註冊的錢包可以導入 Keplr，代表可直接把 $LIKE 帶到 Osmosis 或 Crescent DEX 甚至其他地方進行投資而不需要先由 Liker Land app 轉帳到 Keplr 再進行，是直接和方便了。</li><li>然而如果你擔心曾經連結過社群帳號的 Liker ID 因為任何原因不安全（留意我不是說 Authcore 的技術不安全，而是最大的不安全因素是人，之前 <a href="https://matters.news/@daisy/193654-like-coin-%E6%94%BE%E5%9C%A8-keplr-%E9%82%84%E6%98%AF-liker-land-%E6%AF%94%E8%BC%83%E5%A5%BD-bafyreid4sng4t7pdywrrp3z2pywmdd2mgo5hx4fd2eqhbottd6kd32rnly" rel="noopener noreferrer" target="_blank">經已解釋過</a>），那確實分開的好。</li><li>由於 <a href="https://docs.like.co/v/zh/user-guide/liker-id/register-with-keplr" rel="noopener noreferrer" target="_blank">每個 Liker ID 只能從「一般方法 (Authcore)」及 「Keplr」兩種註冊方式二選其一，選定後無法更改至另一種登入方式。</a>而使用 Keplr 註冊的 Liker ID 應該不可以逆向兼容 Authcore。如果你使用了<s>邪道</s>不是正常註冊 Liker ID 的方法在 Matters 綁了一個以 Keplr 註冊的 Liker ID，應該就是無法改成綁 Authcore 的了。</li><li>現在變成了 Authcore 註冊可享有 Liker Land app + Keplr 兩者的好處（看 <a href="https://docs.like.co/v/zh/user-guide/liker-id/register-with-keplr" rel="noopener noreferrer" target="_blank">比較表</a>），讚讚！</li></ul><h2>錢包系列</h2><p><strong>Keplr 錢包</strong></p><p><a
