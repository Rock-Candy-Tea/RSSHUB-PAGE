<p>接下來兩週我會將區塊勢的文章透過 LikeCoin 的去中心化出版機制發行成 NFT、註冊「身分證字號」，同時將內容上傳到去中心化硬碟保存。喜歡文章的讀者，歡迎以 LIKE 幣購買收藏。</p><p>👉 <a href="https://liker.land/nft/class/likenft1s7xk7wt5q03lsa2a87fnqfc0uny28f46s4gtnakc08qdguxlua4qxrxk7m" rel="noopener noreferrer" target="_blank">收藏文章 Writing NFT</a> 👈</p><p>這篇文章討論 LikeCoin 最新推出的去中心化出版機制如何運作，以及上面這些技術名詞在說些什麼？</p><h2>去中心化出版</h2><p>許多人可能是第一次聽過去中心化出版（DePub），但對去中心化金融（DeFi）不陌生。</p><p>2008 年比特幣問世。但在最初幾年，人們只懂得以比特幣買披薩或者買低賣高賺價差。得等到 10 年之後，人們才開始基於加密貨幣建立去中心化金融應用，並發展出 Uniswap、Aave 或 <a href="https://blocktrend.substack.com/p/313" rel="noopener noreferrer" target="_blank">PoolTogether</a> 這些全新服務。</p><p>換句話說，幣圈的發展軌跡是先有資產才有應用。NFT 也不例外。</p><p>CryptoPunks 被譽為 NFT 始祖，最初在 2016 年底<a href="https://blocktrend.substack.com/p/433" rel="noopener noreferrer" target="_blank">問世</a>。至今多數人仍只懂得將 NFT 換成社群頭像，或者買低賣高賺價差。情況就如同當年的比特幣一樣原始。恐怕得再多等幾年，人們對 NFT 足夠熟悉之後，才有機會發展產業應用。</p><p>但也有腦筋動得快的團隊已經搶先佈局。其中，我最看好的領域之一就是基於 NFT 的去中心化出版。</p><h2>基礎建設</h2><p>在網路問世之前，只有少數人有機會獲得出版社的邀請寫書。但現在有許多人會經營自己的部落格、IG 粉專或是 YouTube 頻道。人人都可以成為創作者，影響力也未必輸給紙本書。</p><p>雖然創作內容已經高度數位化，但是出版的基礎建設卻沒有隨著科技發展同步升級，這進而衍生許多前所未見的新問題。舉例來說，以往的創作都是印在紙本上。紙作為工具，不會主動判斷內容好壞或者侵權與否。萬一內容不適當，人們會向出版社反映，不會怪罪紙張。就算最後出版物被下架，在外流通的紙本也難以被銷毀。</p><p>現在數位創作多是建立在特定平台上，例如 Medium、臉書或 YouTube。平台由企業經營，被期待要肩負內容守門員的責任。以傳統出版業的模式來比喻，這就像是紙張與銷售通路綁在一起。一旦平台接獲檢舉，不僅要從通路下架，常常是連「紙張」上的內容也一併被銷毀。</p><p>這使得惡意檢舉事件<a href="https://techcrunch.com/2022/07/11/lofi-girl-takedown-youtube-music-stream-dmca/" rel="noopener noreferrer" target="_blank">時有所聞</a>，知名創作者被平台刪文、頻道下架成了家常便飯。不僅創作者反感，夾在使用者與政府之間的企業也兩面不是人。</p><p>此外，創作內容從紙本轉向數位，商業模式雖與過往有所不同，省下大量印刷與物流成本，傳播力也提升，但創作者拿到的收入仍然不多。</p><p>面對這些問題，有人頭痛醫頭，腳痛醫腳。LikeCoin 的開發團隊則認為可以建立另一套數位原生的出版模式，重新將創作內容「印刷」在數位紙張 —— NFT —— 之上，並基於 NFT 建立全新的去中心化出版機制。</p><h2>Writing NFT</h2><p>LikeCoin 對 Writing NFT 的<a href="https://docs.like.co/v/zh/guides/writing-nft" rel="noopener noreferrer" target="_blank">敘述</a>很簡潔：</p><blockquote>Writing NFT 是一段文字的 NFT 版本，可以是文章或任何篇幅的文字內容，也可包含圖片。個人的收藏作品是去中心化身份的重要組成之一。創作者只要點擊幾下，就可以將任何網頁變成 NFT 從銷售作品中獲利，並享受去中心化儲存的好處。</blockquote><p>Writing NFT 就像是數位出版品。有別於物理紙張，只能印文字、圖像等靜態內容。理論上 Writing NFT 還可以把動態的 podcast 或影片通通印上去。只不過，目前還不支援。</p><p>發行的方式也很簡單。創作者只要將準備出版成 Writing NFT 的網址，貼到網站上（<a href="https://docs.google.com/forms/d/e/1FAIpQLSdPkunbI-68k7dzDqNNDX0U8Lr6lg3R2Jsm-RPduUNQ9Om05Q/viewform" rel="noopener noreferrer" target="_blank">登記白名單</a>）並支付等值不到 1 元台幣的 LIKE 幣當成手續費，就能將網址內的文章變成人們可以付費收藏的 NFT。</p><p>我以高重建發行的 Writing NFT 舉例，他將個人網站上其中一篇標題為〈<a href="https://liker.land/nft/class/likenft1yhsps5l8tmeuy9y7k0rjpx97cl67cjkjnzkycecw5xrvjjp6c5yqz0ttmc" rel="noopener noreferrer" target="_blank">分散式出版 DePub 的發展路徑，NFT 從移民到原住民</a>〉 的文章發行成 NFT。</p><p>發行完成之後，LikeCoin 的出版機制不只會自動將文章保存到 IPFS、Arweave 這兩個去中心化硬碟，還會依據發行時間、標題及內容等元資料（metadata）在 LikeCoin 鏈上註冊 ISCN 編號，同時創建好如下圖的 NFT 賣場供讀者購買收藏。</p><figure class="image"><img src="https://assets.matters.news/embed/d7b6128c-0b91-4815-a69a-0e387524cf7d.png" data-asset-id="d7b6128c-0b91-4815-a69a-0e387524cf7d" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>對應到紙本的出版業，這就像是 LikeCoin 自動將文章從電腦印刷到紙上（IPFS、Arweave），並向國家圖書館（LikeCoin）註冊 ISBN 編號，同時還已經把書本上架到書店（Liker Land）供消費者購買。</p><p>完成這些步驟只要幾秒鐘，總成本是不到 1 元台幣的 LIKE 幣。而不像傳統出版業，從完稿到上架恐怕已經過了半年，最終創作者還只能拿到書本定價的 15% 以內。</p><p>區塊勢曾分別介紹過 <a href="https://blocktrend.substack.com/p/467#details" rel="noopener noreferrer" target="_blank">IPFS</a> 與 <a href="https://blocktrend.substack.com/p/371" rel="noopener noreferrer" target="_blank">Arweave</a>，這篇文章就不贅述。ISCN 是 LikeCoin 獨創的標準規範，它的運作機制類似於國際標準書號（ISBN）。只不過 ISCN 的註冊單位不是國家圖書館，而是 LikeCoin 區塊鏈。</p><p>網路上的創作內容數量遠多於紙本。ISCN 就像是網路文章的身份證號碼，讓人們可以按照 ISCN 編號找到文章，而不必擔心如果未來網路上有另外一位高重建寫了另一篇相同標題的文章，而難以區分誰是誰。</p><p>制定標準規範是為了規模化。假如日後有「NFT 圖書館」要收藏 Writing NFT，就需要仰賴 ISCN 編目，「NFT 書店」也需要依據 ISCN 編號才不會搞錯兩位高重建先生的同名作品。</p><p>除了 ISCN 之外，Liker Land 則像是全球第一家販售 Writing NFT 的「書店」。</p><h2>Liker Land 書店</h2><p>Liker Land 這家書店不僅販售的商品前所未見，就連訂價機制也與眾不同。有別於以往創作者可以自行決定 NFT 的售價，Liker Land 採取動態訂價機制。根據 LikeCoin 的<a href="https://docs.like.co/v/zh/guides/writing-nft/dynamic-pricing" rel="noopener noreferrer" target="_blank">敘述</a>：</p><blockquote>我們明白作者最希望的是專注創作 ... 其他行銷相關的工序可免則免。Writing NFT 為作者處理兩個令人頭痛的參數：1）訂價；2）數量。<br class="smart"><br class="smart">Writing NFT 的訂價會隨著市場供需變化，按一條曲線上升，需求愈大價錢會自動調升 ... 當價格到達一個市場不能接受的高度，便不會再有人願意出價鑄造出新的 NFT，總流通數量因而停止增加。</blockquote><p>以往出版社常會建議第一次出書的作者先印 2,000 本，如果市場反應不錯再視情況二刷，如果沒人買就到此為止。但 Writing NFT 不是紙本，不用煩惱庫存問題。可以根據市場需求賣多少印多少，印刷數量無上限。</p><p>無限供給的書該如何訂價？如果按照供需法則，價格應該是零。</p><p>但這不僅創作者得喝西北風，Liker Land 這家書店也沒有收入。因此，它們在訂價機制動了一點手腳。最早買到 Writing NFT 的人能以最便宜的方式收藏，日後隨著 Writing NFT 的印刷數量增加，售價也會跟著提升。最終價格、數量會取得平衡，可以說是當時市場對這份作品的公允定價。</p><p>這套機制非常有趣。它不僅省下創作者訂價、發行數量的麻煩，還一定程度反映出市場對這份作品的估價。只不過，這只是其中一家「書店」的訂價策略。如果日後還有其他書店願意上架 Writing NFT，或者有人成立 Writing NFT 的蝦皮，創造二級市場，到時候市場還會變得更複雜。</p><p>回頭來看，目前多數人對於 NFT 的認知仍然停留在圖像，尤其是頭像。但隨著人們對 NFT 越來越熟悉，大家會發現 NFT 也可以像紙本，是另一種數位載體。現在 LikeCoin 投入去中心化出版，就像在人們還在用比特幣買披薩的年代提倡去中心化金融，困難重重。多數人也難以想像。</p><p>看到眼前一片荒蕪，有人想的是拓荒，也有人想晚點再來。前者可以享受早期紅利，後者賺到的是生活便利。如果覺得這篇文章不錯，或許可以從收藏 Writing NFT 開始。</p><hr><p><a href="https://blocktrend.substack.com/"
