referrerpolicy="no-referrer"><figcaption><span>眾志成城為 LikeCoin</span></figcaption></figure><p>為甚麼說是史上最快速呢？因為今天 15:58 Phoebe 在 Discord 發了一個訊息說有沒有人能幫忙提案，驗證人 <a class="mention" href="https://matters.news/@moonbeamwow" target="_blank" data-display-name="moonbeam" data-user-name="moonbeamwow" data-id="VXNlcjo2MzQxNQ">﻿<span>@moonbeam</span>﻿</a> 回覆會幫忙，然後在 19:33 就完成<a href="https://stake.like.co/proposals/17" target="_blank">並把內容寫到區塊鏈</a>真是超級快速。</p><h2>議案 17 在說乜東東？</h2><p>究竟這個議案「Enable IBC send and receive」說的是甚麼？要追溯到 <a href="https://medium.com/likecoin/likecoin-community-update-202109-a5722cb838dd" target="_blank">9 月社群會議紀錄</a>，Chung 說他正在弄 FoTan 的 IBC。而 IBC 指的是 Inter-Blockchain Communication 跨鏈通訊。</p><p>LikeCoin 是建基於 Cosmos SDK 的項目，而其他建基於 Cosmos 的項目現時是無法與 LikeCoin 互相溝通的。引進 IBC 可大大降低 LikeCoin 接入 Cosmos 生態中的去中心交易所 ( DEX ) 和錢包工具的難度（這句很明顯是抄回來的）。我想大家都明白更多交易所引進更多競爭只會是好事，所以又怎可以不支持呢？</p><h2>Liker 也可協助議案進入投票程序？</h2><p>作為 Liker 不單止能就議案投票，更可存入保證金讓議案進入投票階段。</p><p>現時每個議案需要於<a href="https://docs.like.co/v/zh/user-guide/background#81f6" target="_blank">兩星期內獲得十萬 LikeCoin 保證金</a>才可進入投票程序，而正如投票那樣，你只需要到 <a href="https://stake.like.co/" target="_blank">stake.like.co</a>，就可以為議案存入保證金。</p><p>未升級到 FoTan 前，只有驗證人才能存入保證金，也需要擁有硬件錢包方可進行操作。然而升級後，直接民主終於一步到位，任何 Liker 素人也能成為「金主」，支持自己認為對社群帶來好處的議案。</p><h2>如何存入保證金？</h2><p>1. 到 <a href="https://stake.like.co/" target="_blank">stake.like.co</a> 選擇 Keplr Browser Extension 或 Authcore 登入，以下選擇 Authcore 作示範。</p><figure
