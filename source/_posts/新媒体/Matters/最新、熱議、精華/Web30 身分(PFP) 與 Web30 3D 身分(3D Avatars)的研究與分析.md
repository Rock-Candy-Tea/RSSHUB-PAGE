<h2>【前言】</h2><p>Web2 的時代我們在通訊軟體或 Social Media 的 Profile Picture 大多只是自己拍照存在裝置內的照片或是網路上下載保存在裝置內的圖片， 功能不外乎只是讓社交圈的人能夠記住你並辨識出這是你的帳號。而在現實生活中具備功能的是我們的證件類的照片，像是身分證、護照、駕照和健保卡上所使用的證件照、大頭照或半身照，格式和用處都是中心化統一規範和認證。</p><blockquote>PFP 的出現後，對我們邁向 Web3.0 又會產生甚麼樣令人興奮的變化呢?</blockquote><hr><h2>【Web3.0 身分 PFP】</h2><p>在 Web3.0 的世界，世界認識我們的方式不再像證件照是透過我們的臉或生理特徵。</p><p>在 Web3.0 去中心化的世界，我們可以透過 NFT Avatar 形式的 PFP 作為我們在 Web3.0 世界中的臉、身分和進入 Community 的鑰匙。</p><p>Web3.0 身分 PFP 是超越種族和國籍的，沒有中心化組織限制規則也不需要透過真實姓名或與現實生活有關連的名稱來進行辨識，人們可以匿名透過 PFP 代表身分特徵，也可以在社群化的虛擬世界經營自己的匿名聲譽。</p><blockquote><strong>為什麼要擁有 NFT Avatar 形式的 PFP?</strong></blockquote><p>1. 在 Web3 的世界，<em>Decentralised Name Service</em> 如 ENS 如果是我們 Web3 身分的名字，那 PFP 就是我們 Web3 身分的臉。(*PS. ENS 也能設定 NFT PFP, 讓我們在使用 Web3 社交時也有 PFP)</p><p>2. 現在 Chrome extension 如 MASK Network 在連結錢包和 Twitter 帳號後，可以從錢包中選擇 NFT Avatar 作為 Twitter 的 PFP，外觀上可以看到 NFT 的價值和在 Collection 的編號，點擊 PFP 還會跳轉到該 NFT 在 opensea 的頁面。Twitter Blue 近期也推出 NFT Profile Pictures 服務，點擊 PFP 後可以查看詳細資料(如創作者、擁有者等 NFT 的詳細資料)。</p><ul><li>代表了 Twitter user 在 Web3 的身分以及對 NFT 領域的參與</li><li>代表了 NFT 的可驗證所有權和 NFT 資產的識別，這可以遏止用於嘲諷 PFP 的右鍵下載左鍵上傳 JPEG 笑話和盜圖偽裝假身分。</li></ul><p>3. PFP 在 Web3.0 和社群中提供了我們表達自己的機會，讓我們在社群中分享我們的知識並參與其中一同去推動項目文化，這個過程也會推動該 NFT 項目經濟價值的賦予程度，NFT 價值通常都會因此增加。(例如: BAYC 社群)</p><hr><h2>【Web3.0 3D 身分】</h2><p>Metaverse 虛擬世界「元宇宙」，高互動性和沉浸式的體驗就如同現實世界運作一般，但是跨越國家、種族和區域，他是存在於網路的世界，但是最重要的元素就是人們可以在裡面產生連結、互動和體驗各式各樣的活動(也包含虛擬經濟的活動)，當 Web3.0 與 Metaverse 結合在一起時，我們在元宇宙時就會需要用 3D 形式來代表我們的身分特徵。</p><p>舉個例子，在 Decentraland 中，Ethereum Address 就像是我們的身分證上的統一編號，而名子則可以自行定義，外觀特徵除了 Decentraland 提供的基本外觀外，就是可以透過 NFT 的形式達到其獨特性和群體歸屬感。</p><p>就好比情人節 Samsung 在 Decentraland 虛擬旗艦店 837X 舉辦活動，透過參與可以獲得 Samsung 專為情人節推出的3件 ERC-721 外觀裝備，過程大家在虛擬旗艦店 837X 透過文字線上互動，互相幫忙解任務，然後一起獲得 NFT 後穿上在 Decentraland 互動，那時的 3D 虛擬身分即代表了在電腦前的我和在其他地區的虛擬網友進行互動。</p><p>近期也很多 2D NFT PFP 項目也陸續計畫推出 3D 版本為進軍元宇宙做準備，我覺得這會是另一個 PFP 的機會，很期待之後 PFP 在 3D 身分和元宇宙結合後，可以直接在元宇宙做超越我們現在透過 Twitter 或 Discord 所進行的社群互動，以及不一樣的 PFP 的價值賦予方式。</p><h2>總結:</h2><blockquote>NFT PFT 身分是我們在現實世界與 Web3.0 去中心化的世界的橋樑，代表著身分、可驗證的所有權和唯一性，它也是我們重要的數位資產以及在社群中的地位和象徵。</blockquote>
