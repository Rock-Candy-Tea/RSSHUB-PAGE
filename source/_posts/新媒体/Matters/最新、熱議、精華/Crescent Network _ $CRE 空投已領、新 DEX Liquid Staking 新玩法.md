referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>Crescent Network 獨立成鏈，相信有很多朋友都經已領了空投。如果你在 2022 年 1 月 1 日有委託 $ATOM 但仍未領空投的話，你可以參考：</p><p><a href="https://www.forbole.com/blog/how-to-claim-cre-airdrop-tc" rel="noopener noreferrer" target="_blank">如何領取 $CRE 空投</a></p><h2>Liquid Staking 是甚麼？</h2><p>相信大家對領取空投任務 #2 Liquid Stake CRE 都不明所以，究竟它是甚麼？依照官方解釋：</p><p>流動委託 / 流動質押 ( Liquid Staking ) 以第二種投資機遇解決資產休眠問題，同時仍然賦予用戶相同的投票權及委託權。降低了傳統委託的機會成本，從而否定不需的高委託回報率亦可追求更具資本效率的 CRE 通脹回報予整個生態系統：</p><ul><li>流動委託以預先定義的功率權重導致對流動委託驗證人的授權多樣化</li><li>流動委託組由社群治理流程管理</li></ul><p><strong>流動委託驗證人組</strong></p><p>Crescent Network 的流動委託程序由列入白名單的驗證人組成，這些驗證人直接接收從 liquid staking module 的委託。當用戶委託他們的 CRE 時，委託的金額在白名單驗證人之間平均分配，這些驗證人可以由 Crescent Foundation 領導的治理來改變。</p><p>於創世時，流動驗證人組將由 11 名驗證人組成，他們受制於以下條件下並通過治理改變：</p><p>增加驗證人條件：</p><ul><li>如網絡不存在問題，每月最多於現有驗證人組添加 1 名驗證人</li><li>必須提供驗證人的表現及對 Crescent Network 的潛在貢獻並得到治理同意</li></ul><p>開除的理由：</p><ul><li>驗證人顯著拖累網絡性能</li><li>佣金率設置與治理意向所決定的佣金率不同</li><li>VEV（驗證器可提取值）統計數據</li></ul><p><strong>我對流動委託的理解</strong></p><p>與 Osmosis 的 Superfluid Staking 相類似（如果不知道甚麼是 Superfluid Staking 可參看 <a href="https://www.forbole.com/blog/superfluid-staking-tc" rel="noopener noreferrer" target="_blank">超流體權益質押 Superfluid Staking</a>），Crescent 的流動委託 / 流動質押 ( Liquid Staking ) 可以讓你將 $CRE 轉化為 $bCRE 並鎖定進行委託以賺取委託回報，而這些 $bCRE 又可以用來提供流動性並賺取流動性挖礦獎勵。</p><p>現時 Crescent 選定 11 個白名單驗證人而這些驗證人的佣金設定為 20%。假如你進行<strong>流動委託即是在 </strong><a href="https://app.crescent.network/staking" rel="noopener noreferrer" target="_blank"><strong>Crescent DEX 介面的 Staking</strong></a><strong> 委託 $CRE 並將它變做 $bCRE，亦即代表自動將你的 $bCRE 委託及 11 個白名單驗證人。</strong>留意份額是由系統幫你平均委託，所以不能指定流動委託給單一個驗證人的。而解除 $bCRE 的委託需時 14 天。</p><figure class="image"><img src="https://assets.matters.news/embed/c00edd63-7f67-4b11-9b51-19b6e7c72184.png" data-asset-id="c00edd63-7f67-4b11-9b51-19b6e7c72184" referrerpolicy="no-referrer"><figcaption><span>名單上的 11 個白名單驗證人</span></figcaption></figure><p>然後你可以將這些經已完成流動委託的 $bCRE 在 <a href="https://app.crescent.network/farm" rel="noopener noreferrer" target="_blank">Farm 介面</a> 選取任何一個有 $bCRE 的 Token Pair 提供流動性。</p><figure class="image"><img src="https://assets.matters.news/embed/bc98c469-de71-493b-a948-d3f4683eda7a.png" data-asset-id="bc98c469-de71-493b-a948-d3f4683eda7a" referrerpolicy="no-referrer"><figcaption><span>現時有 4 個 Token Pair 有 $bCRE</span></figcaption></figure><p>由於 Crescent 的 <a href="https://app.crescent.network/gov/2" rel="noopener noreferrer" target="_blank">議案 2</a> 正在投票所以並未有流動性挖礦獎勵 ( Liquidity Incentives )，然而通過後，用戶將可在每天紀元 ( Epoch ) 即 00:00 UTC 收到 $CRE 獎勵。不過因為現在還未有，所以會看到上圖例如 ATOM-bCRE 池白色的流動性挖礦 APR% 是 0%，並提供紫色的流動委託 APR% 為 34.24% 的。至於 34.24% 數字的來由就是流動委託 APR 68.48% 的一半。</p><p>但留意 Crescent DEX 有一個特別機制，就是做流動性挖礦時獎勵並不是如 Osmosis 般一放進去就開始計算，而是需要等待到下一個每天紀元才開始挖礦。或者在議案 2 通過後再來研究一下這部份。</p><figure class="image"><img src="https://assets.matters.news/embed/bfc051a0-2872-487d-9728-4876ba5ec2f7.png"
