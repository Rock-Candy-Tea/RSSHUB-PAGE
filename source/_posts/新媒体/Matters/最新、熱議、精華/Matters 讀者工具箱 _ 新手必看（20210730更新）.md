referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><ul><li>Matters 有哪些作者？我要如何找到我感興趣的作者？</li></ul><p>Matters 的作者來自世界各地，有著多樣的寫作題材和風格。你可以在<a href="https://matters.news/authors" target="_blank">「全部作者」</a>看到所有作者，進入他們的創作空間站閱讀文章。除了作者清單，你也可以在 <a href="https://matters.news/tags/VGFnOjk5MTM" target="_blank">#我的私藏作者名單</a> 找到社區推薦的優秀作者們。</p><p>遇見心儀的作者，一定要按下「追蹤」鍵，下次該作者有新文章發表時，會顯示網站首頁左方的「追蹤」欄，這樣你就不會錯過任何一篇作品了！</p><ul><li>如何在 Matters 找到我感興趣的文章？</li></ul><p>Matters 的話題範圍多元廣泛，你可以在<a href="https://matters.news/tags" target="_blank">「全部標籤」</a>找到更多你感興趣的話題，並追蹤這些標籤。當這些專題有更新時，它們會自動出現在你的「追蹤」欄裡。</p><p>你也可以參考其他人的精選文章。例如用戶<a href="https://matters.news/@muter" target="_blank">靜音Muter</a>開設了 telegram 頻道<a href="https://t.me/MattersHub" target="_blank">「Matters 閱讀精選」</a>，<a href="https://matters.news/@muter/%E5%82%B3%E6%92%AD%E7%9A%84%E8%97%9D%E8%A1%93-%E6%88%91%E5%A6%82%E4%BD%95%E5%84%AA%E9%9B%85%E5%9C%B0%E5%9C%A8telegram%E4%B8%8A%E5%88%86%E4%BA%ABmatters%E6%96%87%E7%AB%A0-bafyreigcspzd6cyocf53eq25qxrrpjcjpe77quxy5hbewmhnynms7ikkoa" target="_blank">有興趣訂閱頻道的人可以參考這篇說明</a>。</p><ul><li>讚賞鍵跟拍手是什麼？如何使用讚賞鍵？</li></ul><p>2019 年 10 月起，Matters 與 LikeCoin 聯手，將 LikeCoin 這一基於寫作者創造力為衡量的加密貨幣引入 Matters，讓創作者真正擁有內容，獲得實質回饋。更多資訊，請看<a href="https://matters.news/@likecoin/%E5%8C%96%E8%AE%9A%E7%82%BA%E8%B3%9E-%E8%AE%9A%E8%B3%9E%E5%85%AC%E6%B0%91%E6%87%B6%E4%BA%BA%E5%8C%85-zdpuAtWT6a2rjr75JZBzwnvy67vrxw5cBRqwnHbkFehRYHLXx" target="_blank">讚賞公民懶人包</a>。</p><p>現在，每篇 Matters 文章都會自動嵌入讚賞鍵，作者不需額外安裝。每位註冊完成的 Matters 用戶都可以使用讚賞鍵，最多拍手五次，拍手不可撤回。你的每個「讚」都會轉化為作者的收入，所以請多多為喜歡的作品拍手吧。</p><p>Matters 用戶需要一組 Liker ID 才能進行拍手。在註冊的時候，你可以選擇讓 Matters 為你自動生成 Liker ID，或者，如果你已經擁有一組 Liker ID ，也可直接與 Matters ID 綁定。請注意，在 Matters 生成或綁定 Liker ID 後，就無法修改或解綁。如果您有 Liker ID 設定的相關疑問，<a href="https://matters.news/@daisy/%E7%82%BA%E7%94%9A%E9%BA%BC%E6%B2%92%E6%9C%89%E8%AE%9A%E8%B3%9E%E5%85%AC%E6%B0%91%E7%B6%A0%E8%89%B2%E5%85%89%E7%92%B0-%E5%9B%A0%E7%82%BA%E4%BD%A0%E7%9A%84-liker-id-%E4%B8%8D%E6%98%AF%E4%BD%A0%E7%9A%84-liker-id-bafyreihet3j3p5fyodu472mpntmiyp3untzpj3hw57343celwvviuawffe" target="_blank">可以參考用戶 Daisy 所寫的這篇文章</a>。</p><ul><li>我可以只註冊 Matters 但不生成或綁定 Liker ID 嗎？</li></ul><p>可以。但請注意，如果不設定 Liker ID，你在 Matters 站上的價值標記工具將無法啟用，也就無法發表新文章和給出讚賞。如果你是持有 MAT 的老用戶，也無法對由 MAT 轉換而得、以及舊作持續積累的 LikeCoin 進行管理或者提現。但你仍然可以使用 Matters 進行閱讀和評論，你的舊作品也可以繼續獲得讚賞。</p><ul><li>我幫作者拍手，我需要付錢嗎？</li></ul><p>如果你是免費 Liker（每位綁定 Liker ID 的 Matters 用戶都是免費 Liker），你不需要支付任何成本，即可拍手支持作者。</p><p>如果你加入了 LikeCoin 推出的月費制度<a href="https://matters.news/@likecoin/%E8%AE%9A%E8%B3%9E%E5%85%AC%E6%B0%91%E7%94%A2%E5%93%81%E6%9B%B4%E6%96%B0-civic-liker-classic-bafyreidjvf424wkcmwrwqrr2dhgh6qtgo2cponm7ndasb33agmqmdonpuq" target="_blank">「Civic Liker Classic」</a>，每個月花五美金，這筆錢會隨著你按的每個讚，分配給你拍手的作者。更多資訊可以閱讀<a href="https://matters.news/@likecoin/%E5%8C%96%E8%AE%9A%E7%82%BA%E8%B3%9E-%E8%AE%9A%E8%B3%9E%E5%85%AC%E6%B0%91%E6%87%B6%E4%BA%BA%E5%8C%85-zdpuAtWT6a2rjr75JZBzwnvy67vrxw5cBRqwnHbkFehRYHLXx" target="_blank">讚賞公民懶人包</a>。如果你加入<a href="https://matters.news/@likecoin/%E8%AE%9A%E8%B3%9E%E5%85%AC%E6%B0%91-2-0-%E6%AD%A3%E5%BC%8F%E4%B8%8A%E7%B7%9A-bafyreifmc4wfadtkb7n66qs6weg5sbii73iegktqrlaguc45ji4f6nj2sq" target="_blank">「讚賞公民 2.0」</a>，你付出的月費會全數以等值 LikeCoin 轉給支持的作者，不再通過按讚分配。</p><ul><li>一個拍手價值多少 LikeCoin？為什麼讚賞公民的拍手比較值錢？</li></ul><p>作者所得到的 LikeCoin 一部分來自於 LikeCoin 的配對基金，讓讀者在不花錢拍手的情況下，也能讓作者收到 LikeCoin；另一部分則來自<a href="https://liker.land/civic?from=matters" target="_blank">讚賞公民</a>的月費。</p><p>每一個拍手可換算的 LikeCoin 並非固定數值，受到幣值、當日共有多少作者一起分享月費配對基金的影響而浮動。</p><p>讚賞公民的每下拍手，對「月費配對基金」分配的影響力比「免費 Liker」要大很多。讚賞公民另有一筆每個月五美金的「讚賞公民月費」可分配。如果你是一般 Liker，可以加入<a href="https://liker.land/civic" target="_blank">讚賞公民</a>，如此一來，你的拍手會使作者獲得更多收入。</p><ul><li>Super Like 是什麼？</li></ul><p>Super Like 是讚賞公民幫助好文章獲得更多曝光的機會。只要你是讚賞公民，每天會有兩次的 Super Like 機會（ 0:00 及 12:00 更新），對一篇文章拍完五次手後，第六下即為 Super Like。獲得 Super Like 的文章，會出現在 Liker Land 上，獲得更多曝光機會。</p><p>作者不能為自己的文章拍手，但可以為自己的文章打 Super Like。</p><ul><li>除了拍手，我還可以怎樣支持喜歡的作者？</li></ul><p>你可以使用 Matters 開發的支持功能。每篇文章下方都有「支持作者」的按鈕，每篇文章最後都有「支持作者」的按鈕，可以使用港幣或 LikeCoin 鼓勵作者，只要兩分鐘，就可以讓作者獲得實質的創作收入。有任何疑問，可以參考 <a href="https://matters.news/guide" target="_blank">Matters 支付功能指南</a>。</p><ul><li>IPFS 節點是什麼？它有什麼用途？</li></ul><p>IPFS 是 Matters 選用的一種分佈式存儲技術，每一篇在 Matters 發佈的內容，包括文字、圖片、連結都會上傳到 IPFS，但文章下方的評論不會上傳。IPFS 收取的存儲費，目前由 Matters 承擔，未來會以內容發佈者所贏得讚賞的一定比例進行支付。</p><p>公共節點是一篇作品在 IPFS 網絡中的存儲地址，可視為文章的副本。如果讀者因為各種因素無法直接拜訪 Matters 網站，作者可以複製文章的公共節點，將內容傳播出去。如果你想查詢一篇文章的公共節點，該文章的標題旁邊會顯示「分佈式入口」，點開即可找到節點位址。或者你也可以使用由用戶 <a href="https://matters.news/@deserve" target="_blank">Deserve</a> 開發的<a href="https://matters.news/@deserve/matters%E6%96%87%E7%AB%A0%E7%8E%B0%E5%9C%A8%E5%8F%AF%E4%BB%A5%E4%B8%80%E9%94%AE%E5%9C%A8%E7%BA%BF%E8%BD%AC%E4%B8%BA%E5%A2%99%E5%86%85%E9%93%BE%E6%8E%A5%E4%BA%86-zdpuB1bvMnsAr4APk12FmdRxcqMaEsRo46vKE7p6Arvsg4YiF" target="_blank">查詢工具</a>。</p><ul><li>Matters 有 App 嗎？我想在手機上使用 Matters，我該如何操作？</li></ul><p>Matters 目前沒有 App，但你可以透過 PWA 功能，達到與 app 幾乎相同的使用效果。PWA 指的是 Progressive Web App，這是一種把網站打包成 App 的功能，可以在你的手機畫面上佔據一個圖示的位置，操作起來也跟一般 App 很像。具體操作方式，請參考用戶 <a href="https://matters.news/@1ampa55ag3" target="_blank">LeseTruck</a> 寫的<a target="_blank">使用指南</a>。</p><p>你也可以透過 <a href="https://apps.apple.com/tw/app/liker-land/id1248232355" target="_blank">Liker Land app</a> 登入 Matters，即可為 Matters 的文章拍手，並使用書籤功能。</p><ul><li>我可以用 RSS 閱讀 Matters 的文章嗎？</li></ul><p>你可以透過 RSSHub 訂閱 Matters。操作方式可以參考用戶 <a href="https://matters.news/@cerebrater" target="_blank">Cerebrater</a> 寫的<a target="_blank">使用指南</a>，以及用戶 <a href="https://matters.news/@zoenglinghou" target="_blank">zlh</a> 的<a href="https://matters.news/@zoenglinghou/%E8%AE%A2%E9%98%85-matters-%E7%9A%84-rss-%E5%8F%98%E5%BE%97%E6%9B%B4%E7%AE%80%E5%8D%95%E4%BA%86-bafyreiawyva4y5dh4d3t3t5olblgeiog3dig2ejnlauaubbft5t2cbl3xq" target="_blank">使用介紹</a>。</p><ul><li>使用 Tor 瀏覽器但無法登入 Matters 網站，怎麼處理？</li></ul><p>請改用這個連結登入 <a href="https://matters.news/login" target="_blank">https://matters.news/login</a> 。</p><ul><li>Matters 的文章繁簡體皆有，我讀得不太習慣，有沒有什麼辦法轉換字體？</li></ul><p>我們目前提供三種介面語言：繁體、簡體及英文。您可以從「我的」-->「設定」找到「介面語言」並進行修改。</p><p>Matters 用戶橫跨兩岸四地及全球使用中文的地區，因此會有繁簡體並存的文章。如果你不習慣特定字體，可以參考用戶 <a href="https://matters.news/@hymnsin" 篇文章後，你就可以跟其他作者留言互動。</p><p>另外一個狀況是，如果某位用戶屏蔽了你，你就無法在他的文章留言。</p><ul><li>如果我不想讓特定用戶在我的文章下面發言，我可以怎麼做？</li></ul><p>你可以封鎖這位用戶，封鎖之後，他將無法在你的文章下方留言。如果你想要屏蔽該位用戶的全站言論，可以使用由用戶 <a href="https://matters.news/@hi176/%E7%A4%BE%E5%8D%80%E8%87%AA%E6%B2%BB%E8%AA%BF%E6%95%B4-%E8%A8%B4%E8%A8%9F%E9%96%80%E6%AA%BB%E6%8F%90%E9%AB%98%E7%82%BA10%E4%BA%BA%E9%BB%9E%E8%B4%8A-zdpuAwqFF6Y5TnCvJXDoXQUNJLLb4LorkD3dYdjtRQzp3RxDH" target="_blank">也可以發起訴訟，提請社區投票</a>，由社區來決定是否對發言用戶進行凍結、禁言的懲罰。</p><ul><li>Matters 好好玩！還有什麼有趣的工具或玩法嗎？</li></ul><p>Matters 用戶常常開發有趣的新玩法，大家可以在<a href="https://matters.news/tags/VGFnOjExMzU0"
