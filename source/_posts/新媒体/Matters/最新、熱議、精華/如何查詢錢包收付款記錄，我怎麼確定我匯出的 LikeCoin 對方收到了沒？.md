<p>寫這篇其實是另一個帳號辦了活動，發獎金的時候發生了好幾次看到付款失敗的狀況，大致上就是付款之後這個畫面停了好一陣子，有時候直接轉不回 Matters ，有時候轉回 Matters 之後顯示付款失敗！</p><figure class="image"><img src="https://assets.matters.news/embed/b21097d5-cae4-485c-b24d-017de713253f.jpeg" data-asset-id="b21097d5-cae4-485c-b24d-017de713253f" referrerpolicy="no-referrer"><figcaption><span>已發出交易指示，一直轉一直轉一直轉！！！！</span></figcaption></figure><p>首先先提醒一下發獎金的主辦人及付款支持作者的市民們，通常看到這個畫面，LikeCoin 應該是己經付出去了，至少我這次付款的經驗，總共出現 6 次這個情況，每一次都是錢包裡的錢己經被扣了，然後轉回 Matters 又顯示失敗，如果你每一次又多付一次款，我相信活動主辦人多付的 LikeCoin 是不會少的，所以當你看到付款失敗，請你立刻先確定一下錢包到底被扣款了沒，不然等你全部再付完再回頭查，會是一個不小的工程，因為一個活動來個 7-80 篇投稿都是很正常的，159 篇的投稿貓貓也是眼神死！！</p><p>另外就是，因為是跳回 Matters 之後顯示失敗，代表 Matters 沒收到你己經付款的訊息〔但其實你已經付了〕，所以是一個你付了但 Matters 不知道 〔因為沒收到訊息〕，自然頁面上很可能不會出現你有支持，Matters 頁面的錢包裡也有可能不會有記錄〔不過對方可能有收到 Email 〕，所以這時候你要會查找這個交易記錄，去那邊查？要怎麼查？</p><hr><h2>如何查詢交易記錄</h2><p>目前查詢交易記錄的方式，我覺得最快最明瞭的方式就是在 <a href="https://stake.like.co/" target="_blank">https://stake.like.co/</a> 查詢，不管是付款人要看錢被扣了沒，還是收款人要確定錢進錢包了沒，這都是最簡單的方法。</p><p>首先進到 <a href="https://stake.like.co/" target="_blank">https://stake.like.co/</a> 之後，一般在 Matters 註冊帳號取得 Like ID 的人，登入都是選最下面那一個。</p><figure class="image"><img src="https://assets.matters.news/embed/fa4c2536-ef87-427c-86e0-7fccaabe41fa.jpeg" data-asset-id="fa4c2536-ef87-427c-86e0-7fccaabe41fa" referrerpolicy="no-referrer"><figcaption><span>登入</span></figcaption></figure><p>點進去之後是登入畫面，如果你註冊帳號 + Liker ID 是在 Matters 一次完成的，你直接選 Matters 登入，如果你的 Liker ID 有指定綁定其他的登入方試，就選你綁定的方式登入。</p><figure class="image"><img src="https://assets.matters.news/embed/4a736efa-c307-408b-97ec-5e6f39354269.jpeg" data-asset-id="4a736efa-c307-408b-97ec-5e6f39354269" referrerpolicy="no-referrer"><figcaption><span>登入</span></figcaption></figure><p>用以 Matters 帳號登入為例，輸入 1、2、的信箱之後點 3、確認，接著會跳出認證視窗，點 4、同意，這樣你就會登入 <a href="https://stake.like.co/" target="_blank">https://stake.like.co/</a> 。</p><figure class="image"><img src="https://assets.matters.news/embed/d6521644-c9ee-4afd-b33f-ff66a30153fb.jpeg" data-asset-id="d6521644-c9ee-4afd-b33f-ff66a30153fb" referrerpolicy="no-referrer"><figcaption><span>登入</span></figcaption></figure><p>登入之後，左邊選單的第一個就是你的錢包位置，你點一下就會出現綠色勾勾，表示你己經複制了錢包位置，右邊就是你的錢包資訊，包含錢包總金額，可用餘額，委託資訊...等。</p><p>左邊選單的最後一個選項 <strong>Transactions </strong>你點下去就會看到你的交易記錄。</p><p><br></p><figure class="image"><img src="https://assets.matters.news/embed/bbb924a2-7b6d-4f27-9855-920b7fea81db.jpeg" data-asset-id="bbb924a2-7b6d-4f27-9855-920b7fea81db" referrerpolicy="no-referrer"><figcaption><span>使用者面版</span></figcaption></figure><p>以下就是交易記錄的畫面，你可以查看一下你到<strong>底付</strong>出或<strong>收到</strong> LikeCoin 了沒。</p><figure class="image"><img src="https://assets.matters.news/embed/c2f16828-b0c7-4bd0-8f88-d9b6d29949f3.jpeg" data-asset-id="c2f16828-b0c7-4bd0-8f88-d9b6d29949f3" referrerpolicy="no-referrer"><figcaption><span>交易記錄顯示畫面</span></figcaption></figure><p>看一下幾個比較常見的交易記錄長怎樣！</p><ul><li>Receive：收到款項〔下方就是發款的錢包位置〕。</li><li>Send：發出款項〔下方就是你發到那一個錢包〕。</li><li>ReceivenMultiple：收到群發款項。</li><li>Vote：投票記錄。</li></ul><figure class="image"><img src="https://assets.matters.news/embed/674e22b0-2fef-4c59-9c45-bb4c744a351a.jpeg" data-asset-id="674e22b0-2fef-4c59-9c45-bb4c744a351a" referrerpolicy="no-referrer"><figcaption><span>比較常見的交易記錄</span></figcaption></figure><hr><h2>如何知道收款人的錢包位置</h2><p>但是在 <a href="https://stake.like.co/" target="_blank">https://stake.like.co/</a> 裡的交易記錄，你只能看到錢包位置，並不能確定錢包是誰的，所以必需要找到錢包的主人，才能確認這筆紀錄是不是顯示有誤的那筆付款。</p><p>之前，<a href="https://matters.news/@catding" target="_blank">catding</a> 有發一篇<a href="https://matters.news/@catding/matters%E4%BD%BF%E7%94%A8%E6%8C%87%E5%8D%97-%E6%9F%A5%E7%9C%8B%E5%85%B6%E4%BB%96%E7%94%A8%E6%88%B6%E7%9A%84%E9%8C%A2%E5%8C%85like%E9%A4%98%E9%A1%8D-bafyreiglrxeqskdi7ff53jzbexqhtkijz7emhm7laf5bxw5t5nlzldghja" target="_blank">【Matters使用指南】查看其他用戶的錢包LIKE餘額</a>，教學如何查看用戶錢包餘額，裡面有取得錢包位置的方式，簡單的說，你必需先知道對方的 Liker ID，再透過 API 得到對方錢包的位置。</p><blockquote>取得 Liker ID</blockquote><p> Liker ID，不一定與 Matters 的 ID 相同，一般而言是不一樣的，所以你要先取得對方的 Liker ID 。</p><p>先到對方的頁面找一篇文章點支持，並選取以 LikeCoin 支持，跳到付款頁面之後看網址列，付款頁面的網址列格式是這樣：</p><pre class="ql-syntax">https://like.co/in/widget/pay?to=這裡是Liker ID&amount=付款金額&via=matterspool&fee=0&state=a05833ae-8093-4f22-b320-6ef2f4b78a06&redirect_uri=.....
</pre><p>所以網址裡 to= 到 & 之間的那一串就是對方的 <strong><em><u>Liker ID</u></em></strong></p><figure class="image"><img src="https://assets.matters.news/embed/71075b58-4f9e-4a69-ba26-d0619bd4f0b9.jpeg" data-asset-id="71075b58-4f9e-4a69-ba26-d0619bd4f0b9" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><blockquote>取得錢包位置</blockquote><p>取得錢包位置輸入這個網址：</p><pre class="ql-syntax">https://api.like.co/users/id/這裡填入Liker ID/min
</pre><p>然後你就會看到一堆資料，範列如下</p><pre class="ql-syntax">&#123; 
  "user":"Liker ID",
  "displayName":"用戶名稱",
  "avatar":".....",
  "cosmosWallet":"這一串就是錢包位置",
  "isSubscribedCivicLiker":true,
  ......
&#125;
</pre><hr><h2>如何提供交易記錄給對方</h2><p><a href="https://likecoin.bigdipper.live/" target="_blank">https://likecoin.bigdipper.live/</a> 上可以查到每個錢包的每一筆交易紀錄，進畫面之後在搜尋欄打上對方的錢包之後按 Enter。</p><figure class="image"><img src="https://assets.matters.news/embed/5a831a86-80b2-4fe7-b9c6-d42accda0e0f.jpeg" data-asset-id="5a831a86-80b2-4fe7-b9c6-d42accda0e0f" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>就可以看到對方的錢包所有資料，往下拉到 Transactions 的區塊，可以看到這個錢包的交易記錄，如果你才剛剛轉出款項，記錄會在上面幾筆，點進去確認一下你是匯款是那一筆。</p><figure class="image"><img src="https://assets.matters.news/embed/cb5e6980-3b00-4669-969d-035fe70b4f39.jpeg" data-asset-id="cb5e6980-3b00-4669-969d-035fe70b4f39" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>點進去的頁面是 Transaction Details ，長這樣。</p><p>Hash：相當於這筆交易的唯一識別碼。</p><p>Result：Success <==表示這筆交易成功</p><p>Messages：這欄表示交易內容，如果是由錢包A轉進錢包B，會顯示這樣：</p><pre class="ql-syntax">錢包A的一串位置 sent 金額數字 LIKE to 錢包B的一串位置
</pre><p>所以 " 錢包A" 如果是你的錢包〔付款人〕，金額也是你付出的LikeCoin數量，時間也吻合，那就是這筆記錄了，把這一頁〔Transaction Details〕的網址複製下來即可當作付款證明。</p><figure class="image"><img src="https://assets.matters.news/embed/41df61e4-7c43-4e35-8244-d059d3470648.jpeg" data-asset-id="41df61e4-7c43-4e35-8244-d059d3470648" referrerpolicy="no-referrer"><figcaption><span>交易明細畫面</span></figcaption></figure><p>或者在 <a href="https://stake.like.co/" target="_blank">https://stake.like.co/</a> 也可以查到錢包的交易記錄，進畫面後點 Explore with any address。</p><figure class="image"><img src="https://assets.matters.news/embed/7b7a6066-9bf6-4852-86c6-68c329faaa79.jpeg" data-asset-id="7b7a6066-9bf6-4852-86c6-68c329faaa79" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p>輸入對方的錢包位置。</p><figure class="image"><img src="https://assets.matters.news/embed/8b68d21e-00bc-4068-83c0-188f2c8a612f.jpeg" data-asset-id="8b68d21e-00bc-4068-83c0-188f2c8a612f" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>就可以看到對方的錢包資料，不過只能看不能對他的錢包做任何動作，到 <strong>Transactions </strong>就可以看到交易記錄了。</p><hr><p>所以如果你問我，我付了 LikeCoin 對方可不可以不認帳說沒收到，或是我可不可以明明沒付款說我付了，答案是：<strong>不可以</strong>！！因為這些交易記錄全部都是<strong>公開的</strong>，不過你必需知道怎麼去查。</p><p>希望不要再有活動主辦人賠錢，看到付款失敗記得，先查看錢包扣錢了沒有喔！</p><hr><p>成為我的讚賞公民，可以受邀進我的圍爐〔請來留言喔！〕，每期封面貓咪的分解畫法都放在爐裡面了！</p><p>訂閱連結：<a
