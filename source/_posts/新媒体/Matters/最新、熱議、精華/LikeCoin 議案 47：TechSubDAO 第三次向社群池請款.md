referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>議案 47</h2><p>通過對此提案投贊成票，您同意向 Tech SubDAO 多重簽署錢包發送 20,000,000 LIKE，作為分配用於支持 LikeCoin 技術開發的第三批資金。</p><p>Tech SubDAO 經已經運行了 3 個月，並且正在步入正軌。在每月的社群會議提供進度報告，所有付款都透明地列在 Google Spreadsheet 當中：<a href="https://docs.google.com/spreadsheets/d/1oefPBR3wtTaf_uQorQYmYA-fv6kpF6uOzRtGjILnSv0/edit?usp=sharing" rel="noopener noreferrer" target="_blank">https://docs.google.com/spreadsheets/d/1oefPBR3wtTaf_uQorQYmYA-fv6kpF6uOzRtGjILnSv0/edit?usp=sharing</a></p><p>不幸的是，隨著最近加密貨幣海嘯，LIKE 的美元價格受到重創，Tech SubDAO 的司庫存款嚴重不足。</p><p>其實 Tech SubDAO 經已沒有擴大支出，只堅持由 OurSky 的 2 名開發員在 LikeCoin 鏈 NFT module 及 dao.like.co 改版，進展順利。按照目前的 LIKE 價格，每週要花費超過 200 萬個 LIKE。這就是為什麼 Tech SubDAO 需要向社群池申請更多的資金，並預計可持續 8 週，這樣短時間內就不會再做類似提案了。</p><p>事實上，由於 Tech SubDAO 資金不足，上週支付給 OurSky 的款項尚未付清，因為加密貨幣海嘯令資金需求被低估了。</p><hr><p>依照記錄這是 Tech SubDAO 繼 <a href="https://matters.news/@daisy/274847-like-coin-%E8%AD%B0%E6%A1%88-45-%E5%8F%8A-46-tech-sub-dao-%E8%AB%8B%E6%AC%BE%E8%88%87-like-coin-chain-%E5%8D%87%E7%B4%9A-bafyreifgwbgi3vzyjurwuoh2nppaihthnq2ydqgkk2mxh2u4delct6pmkm" rel="noopener noreferrer" target="_blank">議案 45</a> 於 4月2日通過及 <a href="https://matters.news/@daisy/252258-like-coin-%E8%AD%B0%E6%A1%88-39-%E8%88%87-40-tech-subdao-%E5%A4%A7%E6%B4%BB%E8%BA%8D-bafyreiar2s3rehyuspt2k3zdub6dpg4sgl4ra4xijk5ujfuusnlpfc36fi" rel="noopener noreferrer" target="_blank">議案 40</a> 於 3月21日通過後第三次請款。其實 Tech SubDAO 有提供 <a href="https://blog.like.co/tech-subdao-progress-update-2022-05-02/" rel="noopener noreferrer" target="_blank">工作報告</a>，也有在 LikeCoin Discord 有問必答，也不是白拿人工的。如果你覺得不應該再繼續開發以上的技術，你反對就好，OurSky 拿不到錢就自然沒能開發下去。至於欠款就由得 Oursky 先向 Tech SubDAO 那幾個委員追討，追討不到就去 Discord 追討，再不成就將 Liker Land 破產好了（笑）。</p><p>以上真的只是說笑，正經地我真的想澄清某些說法，例如所謂：「社群池就是拿來給人任意出糧」。事實是 <a href="https://likecoin.bigdipper.live/proposals/26" rel="noopener noreferrer" target="_blank">議案 26</a> 將生態池所有 LikeCoin 傳送到社群池 及 <a href="https://likecoin.bigdipper.live/proposals/29" rel="noopener noreferrer" target="_blank">議案 29</a> 將轉移池及團隊池的 LikeCoin 傳送到社群池後，LikeCoin 只有一個「公款池」就是社群池，所有支出都要向社群池請款。如果只是為了私下用錢的話，大可以不把團隊池的錢放進公家繼續不公開使用明細。但是現時的做法就是公開透明讓大家更了解，這樣不是更好嗎？</p><p>幣價低迷是沒有人想發生的事。而我最不理解的是：為甚麼其他幣就沒有人說，LikeCoin 就要被針對創新低？不過我也理解，在某些人的世界和眼界當中，LikeCoin 就是他密碼貨幣的全部，除了 LikeCoin 以外甚麼都不懂，一生人能得到的密碼貨幣只有 LikeCoin。其實這樣是好事，證明是由 LikeCoin 教曉他們甚麼是密碼貨幣，真正功德無量。</p><p>說遠了，還是那句支持搞 NFT，這個真的很重要，所以我已為議案投下保證金。</p>
