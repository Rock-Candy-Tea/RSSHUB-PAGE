<h2>■ 1. 發想</h2><p>最近看有朋友在马特市分享自己的租房经历。社畜的人生苦水除了职场，怎么能少了租房？？？租房几乎是每个打工社畜/屌丝/韭菜/后浪人生中的必经环节😅。租房虽小，但是却映射了人间善恶美丑各种社会问题。</p><p>说起租房遇上的那些奇葩房东房客，我的经历差不多比连续剧还精彩。独乐乐不如众乐乐，于是想发起这个活动看看马特市各位笔友们有什么租房故事分享介绍。</p><ul><li>如果你曾经经历过坑爹房东房客，想要狠狠咒骂ta们，这里是发泄平台；</li><li>萍水相逢租房一场，有哪些房东房客让你至今感恩戴德念念不忘，这里是感谢平台；</li><li>还有很多无房屌丝需要租房，作为过来人的你，有什么避坑指南想要告诉他们，这里是分享平台；</li></ul><p><br class="smart"></p><p><u>本次發起的社區活動提案，主題為「租房定心丸」</u>。先把重點列示出來：</p><blockquote>⚠️ 溫馨提示：<br class="smart"><strong>① 關聯本文</strong><br class="smart"><strong>② 鑲上標籤</strong>：#租房定心丸。<br class="smart"></blockquote><p>行文內容並無規範，唯有：</p><blockquote><strong>③字數要求</strong>：必須滿 800 字，不含標點，以官方報表為準。<strong><br class="smart"></strong>④<strong>故事来源</strong>：以自己真实经历为主，可以是作者当房东的苦水经验，也可以是作者当房客的苦水经验，但是不能网络复制粘贴道听途说已有报道的各种租房经验。<strong><br class="smart"></strong></blockquote><p><strong>① ② ③ ④需要同时满足，才能算为合格投稿，不合格投稿不能参加奖金分配。</strong></p><ul><li>如果版主发现有些投稿不和要求（如：标签简体用成繁体），会友情提示；</li><li>如果版主忘记提示导致投稿无效，版主也不负责任。。。。。。</li></ul><p><br></p><hr><h2><br></h2><h2>✅ 2. <strong>獎金計算方式：</strong></h2><p>根據<a href="https://matters.news/@hi176/%E7%99%BE%E8%90%AC%E5%9F%BA%E9%87%91%E7%A4%BE%E5%8D%80%E6%B4%BB%E5%8B%95%E7%AC%AC%E4%B8%89%E5%AD%A3-%E6%96%B0%E5%A2%9E-%E6%8C%96%E5%AF%B6-%E7%8D%8E%E9%87%91-%E8%B6%8A%E5%A4%9A%E4%BA%BA%E5%8F%83%E8%88%87-%E5%8A%9B%E9%87%8F%E8%B6%8A%E5%A3%AF%E5%A4%A7-bafyreiggnbk3wjb4cdrusnzdpwr2y5p2ke3gy3gk7jfrnmgnd7akrq52oq" target="_blank">【百萬基金社區活動第三季｜新增「挖寶」獎金！越多人參與，力量越壯大】</a>公告：</p><ul><li>前期獎金（提案獎金）：提案帖發出的七天內，獲得的支持金額 x 10（支持金額以 LikeCoin 計算；如果支持的是法幣，以結算當天的匯率為準）。</li><li>後期獎金（參與獎金）：在徵文期間，該徵文獲得的文章篇數 x 300 LikeCoin。</li></ul><p><br></p><hr><p><br></p><h2>✅ 3. <strong>獎金分配方式：</strong></h2><p><strong>⚠️ 3.1参与奖</strong></p><p>奖池来源：前期獎金（提案獎金）</p><p><strong>人人有分，但分三類</strong>。參考 <strong><a class="mention" href="https://matters.news/@jack19781228" target="_blank" data-display-name="古塵" data-user-name="jack19781228" data-id="VXNlcjo1ODEyNg">﻿<span>@古塵</span>﻿</a></strong> 所創的分配法，做了一些微調：<strong><br class="smart"></strong></p><ul><li>投稿者為<strong>支持本活動發起者（支持 63 LikeCoins 以上）</strong>：獎金基數視作３。<br class="smart">（支持達 63 LikeCoins，無論是否為 Civic，皆屬之）<br class="smart">（支持未達 63 LikeCoins，依 Civic 或 Liker 區分）<br class="smart"></li><li>活動結束時，投稿者是 Civic：獎金基數視作２。<br class="smart"></li><li>活動結束時，投稿者是 Liker：獎金基數視作１。</li></ul><p>用白話詮釋。</p><pre class="ql-syntax">人人都是「参与獎」這塊大蛋糕的分母，大蛋糕會先依照「總獎金基數」切成等分。每個投稿者依據不同級別，分配到 1～3 塊小片。
</pre><p><br></p><p><strong>⚠️ 3.2 多劳多得奖</strong></p><p>奖池来源：後期獎金（參與獎金）30%</p><p>一人多稿作者，人人有份。所有一人多稿作者的第2+篇投稿，可以参加这份奖金平分。</p><blockquote>如果你的租房苦水比较多，比如像我🤣一集写不完需要写连续剧🤣🤣，可以多次投稿，于是着就有资格参加「多劳多奖」分成😅。</blockquote><p>用白話詮釋。</p><pre class="ql-syntax">一人多稿都是「多劳多奖」這塊大蛋糕的分母，大蛋糕會先依照「總獎金基數」切成等分。符合一人多稿的投稿者依據投稿数量，分配到 1～n 塊小片。
</pre><p>实例解析：</p><p>一共有n位作者投稿2+，共同贡献了N篇多投稿。其中小白投了2篇。</p><pre class="ql-syntax">小白的第一篇投稿按照3.1规则分得参与奖，小白的第二篇投稿按照3.2规则可分得：
多劳奖金总数 / N 
</pre><p><br></p><p><u>如果活动结束时，没有作者一人投稿2+，说明大家的租房苦水比较少😂，那么这部分奖金就所有参加者平分。</u></p><p><br></p><p><strong>⚠️ 3.3 积分综合奖</strong></p><p>奖池来源：後期獎金（參與獎金）50%</p><pre class="ql-syntax">积分公式 = 拍手数 + 支持数 X 10 + 阅读时间 X 7 - 追踪人数 X 0.1
（阅读时间单位：hr）
</pre><p><br></p><p>每篇投稿根据积分公式计算积分，积分由高到低排列。积分进入前40%的文章，可以参于该奖项角逐。</p><p>每位获奖作者分得奖金根据积分占比获得。一人多稿作者如果有多篇投稿积分进入前40%，则用积分最高的那篇算奖金。</p><p>案例解析：</p><p>小白投了2篇，活动结束时小白的投稿积分都进入了前40%。</p><pre class="ql-syntax">小白的第二篇的投稿需要让位给后续的小黄、小红、小蓝……根据积分高低补上。
小白的积分为n，获奖文章总积分为N，小白占比 n / N X 100%；那么小白也将从奖金池分的：n / N个币，即：
综合积分奖 X n / N
</pre><p><br></p><p>⚠️ <strong>3.4 租房Tip奖</strong></p><p>奖池来源：後期獎金（參與獎金）10%</p><p>活动结束时，版主（即土豆炒青椒）在合格投稿中选出<strong><u>不超过7篇文章</u></strong>，平分这部分奖金。</p><p>评选标准：</p><ul><li>文章提供了租房实用避坑建议</li><li>租房实用避坑建议无论是房东观点还是房客观点都可</li></ul><p><br></p><p>如果没有投稿符合提供了「租房实用避坑建议」，那这部分奖金轮空，由所有参与者平分。</p><p><br></p><p><strong>⚠️ 3.5 租房问题奖</strong></p><p>奖池来源：後期獎金（參與獎金）10%</p><p>活动结束时，版主（即土豆炒青椒）在合格投稿中选出<strong><u>不超过7篇文章</u></strong>，平分这部分奖金。</p><p>评选标准：</p><ul><li>作者分享的租房故事，反映了大千世界的社会问题，人性心理，文化背景……等<strong>较深层的现象</strong></li><li>故事真情实感，能够打动读者有所启发</li></ul><p><br></p><p>如果没有投稿符合「反映深层问题」，那这部分奖金轮空，由所有参与者平分。</p><p><br></p><h2>⚠️ 奖金 3.1、 3.2、3.3、3.4、3.5，符合规则可以重复得奖</h2><p><br></p><hr><hr><h2>■ 回顧</h2><h2>✅ <strong>徵文主題：租房定心丸</strong></h2><ul><li>徵文對象：全體馬特市市民</li><li>徵文內容：分享和租房有关的故事，通过故事向读者提供实用避坑建议指南，通过租房市场反映社会文化万千</li><li><strong>請關聯本文</strong>，並在創作頁面的標籤欄鑲上 <strong>#租房定心丸</strong>，添加標籤時不會用到「#」符號。</li><li>活動時間：即日起至 2021 年 11 月 7 日 8:00 （GMT+8）截止。</li><li>時間一到立刻统计数据，结算奖金。</li><li>所有未盡事宜，提案人 土豆炒青椒 保留補充說明的權利。</li></ul><p><br></p><hr><p><br></p><p>以上模板照抄<a class="mention" href="https://matters.news/@penfarming" target="_blank" data-display-name="閱讀筆耕" data-user-name="penfarming" data-id="VXNlcjo0NTE5NA">﻿<span>@閱讀筆耕</span>﻿</a> 后，稍作修改。</p>
