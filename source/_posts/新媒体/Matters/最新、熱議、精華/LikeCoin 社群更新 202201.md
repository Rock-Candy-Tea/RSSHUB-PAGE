Airdrop</h2><p>去年 12 月第 1 次空投提案（議案 25）獲得社區通過。 然而第一天上線後立即發現計算出現錯誤，原因是因為缺少計算 Osmosis 流動性挖礦綁定模組，意味著空投公式不適用於 Osmosis 流動性挖礦特份者，因此必須更新公式並鑑於某些參數將被更改，所以必須提出新提案。一旦新的空投議案獲得通過，將恢復為 Cosmos 社區空投。Chung 將提供更多細節及參數，例如出了什麼問題，有人必須提出新議案（最後驗證人 Paul Tsang 提出了 <a href="https://stake.like.co/proposals/30" rel="noopener noreferrer" target="_blank">議案 30</a>，詳細解說見 <a href="https://matters.news/@daisy/like-coin-%E8%AD%B0%E6%A1%88-30-%E4%BF%AE%E6%AD%A3%E8%AD%B0%E6%A1%88-25-%E5%90%91-cosmos-%E7%A4%BE%E5%8D%80%E5%8F%8A%E8%AE%9A%E8%B3%9E%E5%85%AC%E6%B0%91%E6%8F%90%E4%BE%9B-fairdrop-bafyreieai6gzl2iub5tm5xpqaf34h5kd7tianqyfcbguncvnhhbnzw2o7i" rel="noopener noreferrer" target="_blank">LikeCoin 議案 30：〔修正議案 25〕向 Cosmos 社區及讚賞公民提供 Fairdrop</a>）。</p><h2>LikeCoin 推特分析</h2><p>這個月的報告和之前所有報告都可以在 <a href="https://irtyshdigital.com/resources/" rel="noopener noreferrer" target="_blank">Irtysh Digital 網站上的資源頁面</a> 找到，它們也都被上傳到 ISCN，並經已在Discord 提供鏈接。</p><p>Irtysh Digital 為 LikeCoin 服務的第三個月從 11月20日 開始，於 12月20日 結束。上月工作重點是推文黏著度而不是推文數量。推文數量雖然減少了，但曝光次數仍然增加，最重要是互動率非常好。 其中大部分來自「推廣活動」這個類別的推文，即是每週小測驗和空投新聞，他們能有效提高互動率。</p><p>而上個月關於提升 WordPress 內容的建議，運作起來結果並不是那麼令人鼓舞。因為即使有更多 WordPress 推文，與其他類別相比，曝光次數及互動率都相當之低。但整體而言，這個月工作成果都相當不錯。本月獲得的洞見就是每週一次的測驗小遊戲旨在提高參與度並向參加者提供 LIKE 作為獎勵，將他們吸引到 LikeCoin 生態系統中。口袋裡有「錢」的人會更多地參與 LikeCoin，亦可作為 LikeCoin 常識是否普及的指標：如果在網上文章能找到答案或問題很簡單，參與度通常都很高。然而如果問題是關於 ISCN 或者更為技術性的問題，參與度就不太高了，所以即使 app.like.oo 經已出現了一段相當長時間，或者它仍然未真正融入社區。故此，應該在推特上發佈更多關於 ISCN 及相關產品以及去中心出版中心思想，並將就技術內容進行更多有獎小測驗看看是否能讓了解去中心出版的趨勢上升。</p><p>下個月的策略是回歸 WordPress Plugin 推文，因為 WordPress Plugin 使用 ISCN 作為主要賣點。 由於社區現在對 ISCN 不太了解，我們應該首先踏出第一步，讓人們增加對這個產品的知識。</p><p>今年推特推廣的重點主要集中在三方面：</p><ul><li>空投即將到來讓 LikeCoin 在 Cosmos 生態系統中激發熱度，亦有助接觸不符合獲得空投條件的人。經已聯繫了 Cosmos 生態圈中的網紅，當 LikeCoin 空投開始時，將會以問答或類似形式發起周邊活動，參加者必須做一些任務或類似活動，即使他們沒有資格領取空投也能得到一些 LIKE。這些都是小型活動，並不具空投規模，卻能吸引更多的受眾並乘著空投的勢度吸引眼球。作為該活動的進階，如果參加者註冊 ISCN 可以讓他們得到的獎勵加乘，可增加他們的產品知識並對 LikeCoin 有更深入了解。</li><li>增加及保持受眾對去中心出版中心的興趣，使用每週小測驗作先峰嘗試讓更多人使用 ISCN。</li><li>測試其他受眾：過去三個月的重點針對 Cosmos 生態系統，但也不能一直如是，因為受眾會疲勞，所以會嘗試開發其他受眾，例如 Web 3 使用者，並且沒有必要過份專注於密碼貨幣以及內容邊緣化。香港最近的消息震動推特社區，令我們有可能接觸到那些根本不會參與密碼貨幣的人們，將特別為他們準備一些訊息做測試。</li></ul><p>關於支付給 Irtysh Digital 的 semi-auto proposal 半自動提案將每三個月提出一次。去年最後一個季度支付招聘費用的第 28 號提案已獲得通過，而另一個議案將會提案以支付 1 月至 3 月的支出。</p><h2>2022年計劃</h2><p><strong>2021年總結</strong></p><p>從 <a href="https://blog.like.co/2021-likecoin-update-were-just-getting-started-fd379f50ae22" rel="noopener noreferrer" target="_blank">2021 LikeCoin update: We're just getting started. Decentralize Publishing revealing its value in the metaverse</a> 這篇文章簡短回顧 2021 年：</p><ul><li>採用 Cosmos Stargate 進行 LikeCoin chain 史上最大型升級，並加入自家 ISCN 及 IBC 功能模組，為 LikeCoin 提供跨鏈功能在 DEX 去中心交易所 Osmosis 及 Emeris 上線。</li><li>ISCN 軟件框架發佈，它亦是 ISCN 模組的一部分，為創作者及數據存檔社群介紹 ISCN 的各種可性及進行註冊時如何使用不同協議。</li><li>依照 ISCN 使用狀態而構築工具並進行如 Arweave 及 app.like.co 的存儲，與及升級 WordPress Plugin 服務 WordPress 社群。</li><li>LikeCoin Grants 計畫培養及孵化不同受資助者激盪各種想法。 今年將向受資助者提供更多資源，並在 LikeCoin 的基礎上構築更多新思維。</li></ul><p>最近有幾個關於去中心化 LikeCoin 作為 DAO 的提議，建立了幾個多重簽署錢包，並且目前還為創作基金及社群委託設立兩個委員會。原本由團隊持有的大部份資金被轉移到社群池，以便通過社群支出議案進行運用。這是去中心化 LikeCoin 並由 DAO 作為管理的一大踏腳石。 LikeCoin 比其他區塊鏈社群更為超前，社群實際上擁有資金並可決定運用他們的方式。議案 29 將轉移池及團隊池中的所有 LIKE 轉移到社群池中，這將增加社群池的規模，社群可以提出議案來使用當中的 LIKE（詳細解說見 <a href="https://matters.news/@daisy/%E8%AD%B0%E6%A1%88-29-%E5%B0%87%E8%BD%89%E7%A7%BB%E6%B1%A0%E5%8F%8A%E5%9C%98%E9%9A%8A%E6%B1%A0%E7%9A%84-like-coin-%E5%82%B3%E9%80%81%E5%88%B0%E7%A4%BE%E7%BE%A4%E6%B1%A0-bafyreifx7ss7cddd5n2nuxv3lkinr265ql2pd4p2bymn5ncko6bldyfaqu" rel="noopener noreferrer" target="_blank">議案 29：將轉移池及團隊池的 LikeCoin 傳送到社群池</a>）。</p><h2>內容備份 V.S. Web 3 原生內容</h2><figure class="image"><img src="https://assets.matters.news/embed/3788f3ba-8884-4576-9a16-2b89a76209ca.png" data-asset-id="3788f3ba-8884-4576-9a16-2b89a76209ca" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>大多數時候，我們都在談及註冊 ISCN 或將經已發佈在網站上，甚至以紙本格式發佈的一些東西上鏈以獲得 ISCN。這種做法通常是為了保育公民社會採取的行動。然而在新的一年裡，將引入另一項新做法就是專注於 Web 3 原生內容，因為它就是為了現在及未來而設，不是為了保存而是為了創造，而且內容將會是區塊鏈原生內容。意味著從內容生成的一刻就存在於區塊鏈上，並且不是經已在其他地方發佈過然後被移植到區塊鏈。</p><p>LikeCoin 經已有一個生成 Web 3 原生內容的應用程序，它是 LikeCoin Grants 支持的照片應用程序，並已經提交到 App Store 但仍在等候審批。該應用程序更參與了 HackAtom VI 並獲得等值 15,000 美元 ATOM 的二等獎。這個應用程序產生 Web 3 原生內容的方式意味著如果你使用這個照片應用程序拍照，一旦拍攝並確認，照片將被上傳到 Arweave 並隨即在 LikeCoin chain 註冊 ISCN。過程完全不像內容備份那種兩步驟過程：因為某些公民媒體中的內容以印刷形式印在傳統報紙，一但停止運作並且不在網絡上，就必須以某種方式將它們移動到鏈中。而針對於這個特定的 Web 3 原生內容形式，使用者們實際就是內容創造者，日復一日地在 LikeCoin 鏈上以原生方式生產內容。</p><p>期待看到更多去中心化應用程序透過連接 Keplr 進行開發，並生成 LikeCoin chain 的原生內容，用戶無需分別註冊即可同時獲得該內容的 ISCN。</p><h2>其他事項</h2><p><strong>社群委託</strong></p><p>社群委託委員會已為新的委託層級建立了新程序，活躍的貢獻者或驗證人可以申請不同委託比例。 程序經已公佈在 <a href="https://discord.com/channels/763001015712350231/843723062096691233/930307396885155870" rel="noopener noreferrer" target="_blank">Discord</a> 上，並於 1 月 10 日生效。</p><h2>下一次社群會議</h2><p>LikeCoin 社群會議將安排於每個月第一個星期一。</p><p>下次會議將於 2022.02.07（週一）格林威治標準時間 1030 在 <a href="https://matters.news/@daisy/like-coin-%E7%A4%BE%E7%BE%A4%E6%9B%B4%E6%96%B0-202201-bafyreigoafkd2qp3v5tlowtllxtaoohmz7mg62o55nskbvbclv36j5c5ai" rel="noopener noreferrer" target="_blank">meet.google.com/ovn-smic-auu</a> 舉行。</p><p>原文：<a href="https://blog.like.co/likecoin-community-update-202201-56cd832486cd" rel="noopener noreferrer" target="_blank">LikeCoin Community Update #202201</a></p><hr><p>自 2022 年開始希望透過更多翻譯的工作，讓人們更了解 LikeCoin 和其他有用的區塊鏈生態。如有任何疑問或意見，請直接到原文留言。</p>
