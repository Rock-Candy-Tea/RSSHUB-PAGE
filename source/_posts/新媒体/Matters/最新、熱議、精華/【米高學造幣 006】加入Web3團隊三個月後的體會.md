<p>在Luna事件後，我做了一個決定，就是加入了朋友的Web3團隊。今天剛剛滿三個月，雖然是兼任的模式，但是投入的時間不比全職同事少。甚至有創辦人跟我說:「你會不會投放了太多時間?」</p><p>這三個月裡，令我改變最多的部份，就是從原本「透過文字打咀炮」至「親身上陣」的經歷經驗。了解到底有多少是「幻想假設」；又有多少是「可以實行」；並且有多少是缺乏及不足。</p><p>以往，作為 Web 3 項目的用家，心中只想著「賺幣」及「暴富」。但是，當換了另一個角度後，反而是想「培育長期的用戶」。</p><p>在這個換位思考的過程裡，我十分慶幸出身於 Steem 及 Matters 的內容平台，這些項目讓我知道社群建立的重要性。不論是 Defi, Gamefi, SocialFi......，沒有社群的支持， Web 3 項目就很難走下去。</p><p><strong>這三個月裡，令我改觀最大的部份，就是 NFT 的世界。</strong></p><p>原本我手上也有大量的NFT，不過普遍是從遊戲而來。90%以上的 NFT是從 2020年至2021年所玩的GameFi遊戲而獲得。當時，同時期就爆發了Opensea、NFT藝術品 及 最流行的 PFP (生成頭象)。那時，我是極討厭 PFP 的NFT項目；與及看不到 NFT 藝術品的前景。</p><p>這幾個月，我從新去了解 NFT 「文化」，完完全全令我改觀。當然，有不少項目是圈錢 Rug Pull跑路，但又有不少項目是在努力凝聚社區。</p><p>我以往以為，一個 Token 數以十萬、百萬或千萬計的人持有才是「社區」。這些一個頭象只有數千人持有，怎會成為社區。</p><p>但是，原來這一個想法很錯。</p><p>NFT的社區營運相當不容易，雖然好像只有數千名持有者。但是，相同理念，嗜好及價值觀的人走在一起，卻十分不容易。</p><p>以往，我是營運遊戲，推出遊戲後，普遍是以「免費暢玩」「免費下載」來吸引玩家。</p><p>再於遊戲內，提供不同的活動、任務、體會、畫面效果等等，讓玩家嘗試留下來 (留存率)。</p><p>當玩家留下來後，再提供不同的誘因，讓玩家嘗試「付費」，從最少額的1美元開始支付，慢慢培育至 20美元，甚至是10,000美元以上的支付。</p><p>這一個過程，換為 NFT 的世界尤如 Free Mint， Floor Price 及 藍籌項目。</p><p>現時，若單純只是推出 NFT，應該必死無疑。故此，現時普遍的項目都有背後真正發展的計劃。</p><p>有的是發展 Web 3 的項目，有的是單純社群之間的支持，有的滿有公益性質，當然也有不少只為「好玩」或「炫耀」。</p><p>經過這三個月在 Web 3 團隊的策劃工作後，項目的基本架構已成型。 <strong>(我在做的項目，是一項文化保育的區塊鏈項目)</strong> 最少，至今未遇到一位 Web 2 或 Web 3 的朋友，懷疑過這項目的可能性 及 長遠價值。</p><p>而我發覺最缺乏的地方，就是對 Web 3 社群的了解。狀況就好像過往推出遊戲，但是我卻對不同遊戲討論區或遊戲公會零認識一樣。</p><p><strong>這三個月的體會，令我越來越佩服營運 Web 3 項目的朋友。</strong></p><p>---</p><p>推廣：LikeCoin 鏈已推出寫作 NFT，大家記得<a href="https://docs.google.com/forms/d/e/1FAIpQLSdPkunbI-68k7dzDqNNDX0U8Lr6lg3R2Jsm-RPduUNQ9Om05Q/viewform" rel="noopener noreferrer" target="_blank" class="keychainify-checked">【申請白名單】</a>成為LikeCoin 寫作 NFT 的早鳥創作人。</p><p><br></p>
