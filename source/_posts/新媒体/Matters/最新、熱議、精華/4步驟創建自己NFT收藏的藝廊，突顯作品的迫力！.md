Metaverse" 計畫的具體展現，其目的就是讓有大量收藏品的朋友，能有一個（好幾個！）展示收藏的空間。它不同於一般 NFT 交易平台只是所有作品一字排開，好像就只是為了銷售一樣，卻能夠提供一個彷彿「走訪藝廊」的臨場感，如果你有 VR 裝置甚至更能身歷其境，想體驗嗎？先<a href="https://oncyber.io/tkwgallery01" rel="noopener target="_blank">oncyber</a> 的資源並不困難，為了實現 punk6529 「人人都能體驗」的理想，有一些展示空間是免費的，只要你已經有下列加密錢包就可以註冊申請：</p><figure class="image"><img src="https://assets.matters.news/embed/8bf4f5e0-a519-487a-a297-e56f9efa51ea.png" data-asset-id="8bf4f5e0-a519-487a-a297-e56f9efa51ea" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>連結錢包後，點選左上角選單圖示，選擇 Create，就可以開始建立第一個展示空間。</p><figure class="image"><img src="https://assets.matters.news/embed/e9e36c87-0ae0-41ed-bb57-3a1d4f0ffeb5.png" data-asset-id="e9e36c87-0ae0-41ed-bb57-3a1d4f0ffeb5" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>現階段可以選用的展示空間，只有第一類是免費的，本文截稿為止，提供了 5 種設計，除了造型不同，每一種右下角的數字代表它可以展示的作品數量。</p><figure class="image"><img src="https://assets.matters.news/embed/de06a13b-98a0-4d7d-a1a0-524c6c13cfb8.png" data-asset-id="de06a13b-98a0-4d7d-a1a0-524c6c13cfb8" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>其他三類都需購買才可使用，有些還附帶一些 NFT。</p><figure class="image"><img src="https://assets.matters.news/embed/f808aec3-e6de-4b76-89a8-7db2d0925919.png" data-asset-id="f808aec3-e6de-4b76-89a8-7db2d0925919" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>Step 2. 展示間基本設定</h2><p>左邊選單中，點選 change/current space 可以在你建立的展示空間中切換。</p><figure class="image"><img src="https://assets.matters.news/embed/4e054f13-66d4-46e7-a56f-92fee7dd026d.png" data-asset-id="4e054f13-66d4-46e7-a56f-92fee7dd026d" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>選擇任一空間，接著會出現三大欄位：</p><figure class="image"><img src="https://assets.matters.news/embed/99319e46-391e-49f9-8736-8ce83c69510d.png" data-asset-id="99319e46-391e-49f9-8736-8ce83c69510d" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><strong>1.Title</strong></p><p>這邊的上欄請填入展示間名稱。下欄是預設的網址，你可以改成較短易記的新網址。</p><p><strong>2.Banner</strong></p><p>展示間對外顯示時的代表圖。完成了以上兩個動作後，這間展示間才會出現在 oncyber 的網頁上。</p><p><strong>3.Description</strong></p><p>展示間的減短說明可以放這邊。</p><h2>Step 3. 陳列你的 NFT</h2><p>現在正式開始將 NFT 上架。如果要移動站的位置，請用 WASD 4 鍵，升降用空白鍵與 B。</p><p>點選牆上的 Add Asset，就會出現左側選單。</p><figure class="image"><img src="https://assets.matters.news/embed/e5a42b47-9a50-4049-a5c8-51be3a49aef7.png" data-asset-id="e5a42b47-9a50-4049-a5c8-51be3a49aef7" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>先前提到你用什麼錢包連接，會影響到能夠上架哪些 NFT，畢竟 oncyber 截稿前還處於 alpha 測試階段，不知道未來可以支援到什麼程度。若以 Metamask 錢包為例，目前可以看到在 Opensea 上以太坊主鏈、Polygon 側鏈，以及 Rarible、使用 Autograph API 及 Klaytn 上的 NFT。因此，不妨把已經有收藏的 NFT 錢包連結上來看看你能掛哪些作品。</p><p>如果你只是單純選擇一件作品，它只會單純地貼在剛剛那個黑色 Add Asset 的位置上，但我們能做的調整很多。下圖左邊欄位代表幾個可以調整的項目：</p><figure class="image"><img src="https://assets.matters.news/embed/962fe47d-abea-448f-a1f0-1d3c1c6ed229.png" data-asset-id="962fe47d-abea-448f-a1f0-1d3c1c6ed229" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><strong>1.Size</strong></p><p>可以用滑鼠拉動黑 bar，或填入數字，就可以從右邊預覽作品的大小。</p><p><strong>2.Advanced Placement Control- Move</strong></p><p>如果打開這個下拉選單，你可以微調作品的位置與方向。此時你會注意右邊的預覽畫面中，作品上會有一個三色的 3D 座標軸，如果你移動滑鼠到任一箭頭或相同顏色的方塊，當它呈現黃色時，代表你可以用滑鼠在箭頭所指的方向移動作品位置。而這張作品之所以懸在空中，正是因為是以 x 軸（面對牆壁的前後方向）為方向移動的結果。</p><p>下圖則呈現，如果你把滑鼠移到座標軸中心，使整個座標軸呈現黃色，則該作品就不限單一方向移動。</p><figure class="image"><img src="https://assets.matters.news/embed/daf5f9c1-3e17-4294-bf33-e3aa811e261d.png" data-asset-id="daf5f9c1-3e17-4294-bf33-e3aa811e261d" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>有了這樣的功能，你的作品就可以不再是單調地並排在牆上，而是可以依據需求機動調整，成為一種策展的手段，例如你可以把相關的作品聚集在一起，以便欣賞。</p><figure class="image"><img src="https://assets.matters.news/embed/d36c2679-abed-482f-a22b-5d35cc400cfd.png" data-asset-id="d36c2679-abed-482f-a22b-5d35cc400cfd" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><strong>3.Advanced Placement Control- Rotate</strong></p><p>選單中的 Rotate 就是指你可以旋轉、翻轉作品到你想要的角度，當然你也可以把原本這面牆上的某個 Add Asset 方塊，以這個方式轉移到隔壁 90 度直角的牆上，進行策展。</p><figure class="image"><img src="https://assets.matters.news/embed/65e0e57b-3c9c-4a0b-a4c0-7d1c0490647d.png" data-asset-id="65e0e57b-3c9c-4a0b-a4c0-7d1c0490647d" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><strong>4.如果你不使用這個 Advance 功能，那麼只要簡單點選作品，拉動四角的白色小方塊來調整。</strong></p><figure class="image"><img src="https://assets.matters.news/embed/5dd92381-fc84-47b6-bcf7-e17271834799.png" data-asset-id="5dd92381-fc84-47b6-bcf7-e17271834799" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>上圖中左側還有 3 個欄位</p><p><strong>5.Set as Positional audio</strong></p><p>如果有出現 Set as Positional audio 並點選，可注意作品前方會出現像下圖這樣的黃色區域，你可調整這個區域的大小，做為作品如有聲音的收聽距離。</p><figure class="image"><img src="https://assets.matters.news/embed/80649d7d-ee3b-4951-a0a0-f01d1109a36b.png" data-asset-id="80649d7d-ee3b-4951-a0a0-f01d1109a36b" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><strong>6.Frame </strong></p><p>代表你可以設定作品邊框的樣式。</p><p><strong>7.Curator Notes </strong></p><p>讓你輸入一些對這件作品的說明文字。</p><h2>Step 4. 預覽</h2><p>作品都安排妥當了嗎？現在可以先一賭為快！點選右上角的 Preview，就可以用訪客視角來觀看剛剛佈置的成果。不錯吧！</p><figure
