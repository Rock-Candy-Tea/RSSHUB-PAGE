<p>今天與大家分享三位台灣生成藝術家的故事，他們即將在本週六於生成藝術史寫下一筆，可能是一大筆。以上有點太浮誇，但請聽我層層拆解，我會分析為何接下來即將發生的事，在 NFT 藝術圈很關鍵，有可能會改變藝術生產的方式。</p><p>這三名台灣人分別是，王新仁（阿亂）、林經堯、林逸文（林哥）。</p><p>他們要把彼此的程式碼融合在一起，做全新的作品。</p><p>別再問 NFT 可不可以是藝術，它可以是，也早就是。</p><p>而且以超乎你想像的速度在進化。</p><hr><p>故事要回到今年二月，台北當代美術館展覽《蓋婭》開幕，阿亂與黃新都有作品展出。當時與會的 <a href="https://www.facebook.com/VolumeDAO/?__cft__[0]=AZUrR07vvu2RJitKyYMBEj8gOCU9vgpUQ9xVF42bMQzPQNx0XtMnU5YlVjXh-AxJg46lAVqov1iQl7QPXehECVNgCXWzPKAUWTq6T0UaeDHsdfdygpDTp64mP9GkuO26pTFj_9MurJI_c3HuTG2Tv2RT&__tn__=kK-R" rel="noopener noreferrer" target="_blank">Volume DAO</a> 成員在 @Matt Deslaurier 的作品《子午線》前拍了一張合照。 Volume DAO 是一個在 Tezos 區塊鏈上沒日沒夜買演算藝術作品的團體。</p><p>當時我們還特地用手機螢幕，讓遠在英國的 Matt、林哥可以一起留影，因為他們是好朋友。（請看照片）</p><p>這張相片後來在 Twitter 上面，被 fxhash 的官方人士看上。或許該人士心裡想「這幾個人很熟嘛，之後找他們一起搞事。」</p><p>過了兩個月，阿亂、經堯、林哥真的要一起搞事了，名為 fx(collab)，就在這週六。</p><figure class="image"><img src="https://assets.matters.news/embed/dca53996-074e-4c85-a091-418b6303f0f9.png" data-asset-id="dca53996-074e-4c85-a091-418b6303f0f9" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><p>解釋一下為何這三個人要一起搞事會這麼令我激動。</p><p>這陣子 fxhash 1.0 版本正式上線，阿亂新作 <a href="https://www.fxhash.xyz/generative/slug/chaos-memory" rel="noopener noreferrer" target="_blank">Chaos Memory </a>與經堯新作<a href="https://www.fxhash.xyz/generative/slug/requiem:-cloud" rel="noopener noreferrer" target="_blank"> Requim: Cloud </a>雙雙霸榜，前五交易量佔了兩名。（我都有收藏到讓我很開心）</p><figure class="image"><img src="https://assets.matters.news/embed/bf589716-10f0-4af9-be6f-f1a5044f0a7a.png" data-asset-id="bf589716-10f0-4af9-be6f-f1a5044f0a7a" referrerpolicy="no-referrer"><figcaption><span>chaos memory #236</span></figcaption></figure><p>除了交易量替他們感到開心，這兩件作品都磨了很久，裡面藏了許多的謎題。阿亂甚至曾經將 Chaos Memory 做成封面相片，說要送給我，但是說不准上鏈...。參與這些作品誕生的過程讓我很過癮。</p><figure class="image"><img src="https://assets.matters.news/embed/e5bfb985-d47b-4605-b5e0-6fc6e3a502b5.png" data-asset-id="e5bfb985-d47b-4605-b5e0-6fc6e3a502b5" referrerpolicy="no-referrer"><figcaption><span>chaos memory #236</span></figcaption></figure><p>而這三個人，於這週六要把各自的程式碼整合在一起，創造全新作品。台灣生成藝術宇宙要誕生了。</p><hr><p>回過頭來介紹 <a href="https://www.fxhash.xyz/" rel="noopener noreferrer" target="_blank">fxhash</a>。</p><p>fxhash 是一個位於 Tezos 區塊鏈上面的生成藝術 NFT 平台，由法國人建立。我在去年十二月寫過一篇文章<a href="https://matters.news/@mashbean/nft%E6%95%85%E4%BA%8B%E9%9B%86-%E7%94%9F%E6%88%90%E5%BC%8F%E8%97%9D%E8%A1%93%E5%B7%A8%E6%B5%AA-%E9%81%A9%E5%90%88%E5%A4%A7%E6%95%99%E5%A0%82%E9%82%84%E6%98%AF%E8%B7%B3%E8%9A%A4%E5%B8%82%E9%9B%86-bafyreia5wv7wc6w57mlcrn35gs3ptymqzipearzoyn7evcne2oyjhv3oim" rel="noopener noreferrer" target="_blank">《生成式藝術巨浪，適合大教堂還是跳蚤市集？》</a>，分析以太坊上的 Artblocks 與 Tezos 上的 fxhash 有何不同。</p><p>自我引用：「如果 Artblocks 是大教堂，因為有嚴格的策展與審查；那 fxhash 就是市集。一個是精緻的生成式藝術品，一個是跳蚤市場，誰知道裡面有哪些不為人所知的寶物？」當時我非常期待 fxhash 的發展，因為一切都太有機，太蠻荒了。</p><p>四個月過後， fxhash 的確超凡進化。 fxhash 在歷經長達三週的暫停服務後，迎來了重大升級，於上週重新開幕，正式成為 1.0 版。而 Artblocks 則是出現最高級的 Curated 新作品上架差點賣不完，還馬上出現破發（跌破原始售價）的狀況。</p><p>黃金交叉到來了嗎？</p><p>久違的再開幕，創作者累積了將近一個月的修煉，收藏者累積了一個月的不耐，店舖一開，大家向瘋子一樣瘋狂上架作品與下架作品。昨天還搞到瓦斯費飆漲（交易給礦工的手續費），一度到 30 tez（約新台幣 3000 元）才能成功買到作品。這在 Tezos 的世界實在不可思議，但也可以在這裡看到跳蚤市場的繁榮。</p><p>新版的 fxhash 出現了幾項強悍的功能：共創分潤、無限期白名單、荷蘭式拍賣。加上先前已有的內建銷售數據工具，還有之後的 DAO token， fxhash 已經成為一個設備精良的跳蚤市場。</p><hr><p>今天只談共創（collaborations）的功能。</p><p>共創，顧名思義為多人協作作品。而從平台端來看，是智能合約可以瞬間將利潤分配給協作者。「創作者有價」在區塊鏈世界可不是喊假的，如果有自動分潤的功能，每一個參與創作的人，都能夠分配到合理的收入。</p><p>而創作者掛名參與，也對自己的鏈上聲譽非常重要，長久經營下來，錢包與平台的 profile 比一個人的履歷還要更真實。共創分潤目前在 NFT 平台還是稀有功能，台灣的 akaswap 有，而 fxhash 應該是第一個主打生成藝術支援分潤的平台。</p><p>甚至，fxhash 還允許創作者將收益，分潤給初級市場的收藏者。這將會大大改變生成藝術市場的遊戲規則。可以想見，分潤給買家，將促進一級市場快速完售，同時將收藏者與創作者的關係拉得更緊密，成為利益共同體。web3 向來都是獎勵早期支持者的地方， fxhash 這招真的厲害。</p><hr><p>為了促進共創功能， fxhash 官方主導了一系列藝術家成為偶像組合，名為 fx(collab)。第一檔推出的組合為 Zancan 與 Yazid，Zancan 自不用說，為全鏈最火紅的藝術家，其生成式植物系列向來是 Tezos 高價物件；而 Yazid 也是 Artblocks 藝術家，其作品《Automatism》剛於三月底在 Artblocks 銷售完畢。</p><p>作為第一檔 fx(collab) 作品，其聯名作品《 <a href="https://www.fxhash.xyz/generative/slug/(kinder)garden-monuments" rel="noopener noreferrer" target="_blank">(kinder)Garden, Monuments</a>》果不其然獲得熱烈迴響，採用荷蘭式拍賣，一級市場總價為 200,000 tez（約為新台幣兩千萬元），目前單一張作品地板價為台幣十七萬元。</p><figure class="image"><img src="https://assets.matters.news/embed/1aa90c3b-bb67-4326-a8a9-b052e429ab03.png" data-asset-id="1aa90c3b-bb67-4326-a8a9-b052e429ab03" referrerpolicy="no-referrer"><figcaption><span>(kinder)Garden, Monuments</span></figcaption></figure><p>該作品將 Zancan 的植物與 Yazid 的積木混合在一起，成為一個全新的敘事宇宙 — （幼兒園）花園與紀念碑。</p><hr><p>當演算藝術遇上共創，不僅僅是分潤如此簡單，這簡直開創了新世界。</p><p>每個人貢獻的程式碼可以整合成一件作品，再藉由收藏者鑄造當下的種子碼，鑄造出獨一無二的 NFT 生成藝術作品。從作品本身來看，可以看見不同創作者的影子。</p><p>可以是視覺上的個人特色，也可以是聲音、動態、或者是更底層的架構。</p><p><strong>自動分潤，在市場經濟上將「共創」常態化；<br class="smart">但真正將「共創」落實到創作過程的，是生成藝術 。</strong></p><p>彷彿大家共同寫一份樂譜，讓機器人演奏。</p><p>好事之徒甚至可以審視程式碼，判斷哪個視覺特色，出自於哪位作者一慣的敘事系統。</p><p>如果 NFT 是畫布，種子碼產生隨機性，那每一位共創者提供的程式碼，便是不同的畫筆，打造一個共同的控制體系。讓骰子永遠在體系裡面滾動。</p><p>這是演算藝術共創的況味。</p><hr><p>回到這次故事的主角群。</p><p>作為 fx(collab) 第二梯隊，擺在 Zancan 後面壓力實在大。但這次的 collab 有許多值得關注之處。</p><p>三位創作者都是台灣人，以往只是仰望彼此，並不相識，因 fxhash 才得以相遇。這三個人都投入巨大的心力在 fxhash 上面，隨著平台一起成長。漸漸成為一方之霸。</p><p>阿亂其知名度不用我多提，除了是 Artblocks 藝術家，也參與創辦 AkaSwap 平台，後來又發行了 TezDozen 頭像專案，這陣子阿妹的 NFT 阿亂也在背後參與設計。最近甚至其作品還上立法院成為質詢標的。</p><figure class="image"><img src="https://assets.matters.news/embed/48e2580b-8c6f-4559-81d9-cb9a4a7307cc.jpeg" data-asset-id="48e2580b-8c6f-4559-81d9-cb9a4a7307cc" referrerpolicy="no-referrer"><figcaption><span>2022/04/13 台灣立法院財委會備詢，左圖為阿亂的 Good Vibrations</span></figcaption></figure><p>經堯是南藝大助理教授，他的 NFT 創作者生涯可以說是貨真價實地與 fxhash 一起成長，從早期簡單的聲音作品，到後面繁複華麗的 Cloud 系列，單價與銷售量節節高升， twitter 追蹤人數已經突破 3500 人。這種超級激勵迴圈，讓經堯的作品進化速度驚人。他真的沒事就在練功發習作，到處問別人這件作品怎麼樣，名字要叫什麼，簡直著了魔。</p><p>而位於倫敦的林哥，前互動藝術家／接案者（其實現在也還是），因 fxhash 讓自己終於成為完全的創作者。原本我們是不認識他的，是因為林哥在 fxhash 發作品，才讓我們得以跨海認識這名台灣人。真懷念當初每一件作品上架都還有餘裕研究的日子， fxhash 已經成長太多，每天數百件新作品讓人目不暇給。</p><p>林哥說，他發了 fxhash 作品之後，認識的台灣人，比他前面十年還要多。林哥原本在巴黎與倫敦是一個多產的互動藝術家，其歐洲朋友於去年 Tezos NFT 開始成長時，沒有跳下來；在 Matt 靠 Artblocks 拿到令人瞠目結舌的銷售成績時，沒有跳下來；終於在 fxhash 問世時，終於也進入瘋狂創作的過程。目前林哥的追蹤人數將要破萬。</p><p>三個台灣人共創出來的生成藝術作品，這次很明顯是在對抗主流市場，或許可以稱之為西方，這次他們使用日式和紙（wagami）拼貼的特色，整合三個藝術家最擅長的紋理。</p><p>阿亂的纖維、經堯的穿雲、林哥的六角，灑落在山、河、花等造型之中。</p><figure class="image"><img src="https://assets.matters.news/embed/0e0e9a69-bdc0-4aba-b189-9c6d5b6ca464.png" data-asset-id="0e0e9a69-bdc0-4aba-b189-9c6d5b6ca464" referrerpolicy="no-referrer"><figcaption><span>這張是三位藝術家的 fx(collab) WIP</span></figcaption></figure><hr><p>稍早與三位藝術家訪談過程，我最訝異的其實是「不滿」。</p><p>阿亂、經堯即使作品完售，仍然不滿。不滿於收藏者的靜默，NFT 沒有創作對話，不滿自己的傑作埋藏在大量簡單易懂的習作裡面。林哥前作破盤的打擊，讓自我懷疑技術之不足、市場之冷淡。市場曾經高高捧起創作者，也會摔傷創作者。</p><p>三位藝術家開始討論 NFT 藝術市場的喜好，也進入東西方審美的觀點差異。這次共創也大量的留白，可以說是與對主流市場的挑戰。即使三位藝術家已是某種主流，仍然在新作品挑戰既有認知。</p><hr><p>至於三個創作者的體系，是怎麼互相嵌合的呢？</p><p>我原以為演算藝術創作者是孤獨的求道者，所有的設定都應服膺於自己的系統之中，每一個系統呈現出來的都是創作者的小宇宙。如要合作，可能會有所衝突吧，合作是需要無盡的妥協、謙讓、堅持、理解。</p><p>沒想到這次共創過程中，幾乎是無縫絲滑的。每一段程式碼有自己的任務區間。而每一位創作者有明確的任務分配。如同阿亂所說，他們因為過去的經歷，早已適應團隊工作。而生成藝術當然也可以拆分成團隊工作。畢竟 Library 與各種小技巧，也是各種前人的心血呀。</p><p>演算藝術因為工具特性，得以快速不斷迭代，新秀使用前人種的樹；而共創機制的發明，讓迭代成為現在進行式，讓創作者可以在同一件作品中互相影響彼此。</p><hr><p>最後，這次的共創作品，有四分之一捐給 <a href="https://www.facebook.com/FABDAOtw/?__cft__[0]=AZUrR07vvu2RJitKyYMBEj8gOCU9vgpUQ9xVF42bMQzPQNx0XtMnU5YlVjXh-AxJg46lAVqov1iQl7QPXehECVNgCXWzPKAUWTq6T0UaeDHsdfdygpDTp64mP9GkuO26pTFj_9MurJI_c3HuTG2Tv2RT&__tn__=kK-R" rel="noopener noreferrer" target="_blank">FAB DAO</a>（Formosa Art Bank DAO）。這絕對不是一小筆而已，先感謝三位藝術家。FAB DAO 目前已在 Tezos 上策劃大大小小的公益專案，當然會善用這筆經費，幫助更多非營利組織，創造更大的社會影響力。</p><p>FAB DAO，福爾摩沙藝術銀行 DAO，為公益NFT自治組織。 專注於非營利NFT鑄造，與區塊鏈公益普及教育。 聚合組織與個人，打造公益、文化、藝術之跨域社群。 </p><p>至於以太坊會不會有生成藝術共創與公益機制，敬請期待下個月的百岳計畫。</p><hr><p>最後總結， fxhash 讓我見證了藝術市場、藏家與創作者之間的高速演化，對話不一定是直接的，但各式各樣的推敲、不滿、猜疑，絕對是對話的一環。</p><p>而這次共創機制的創新，不僅只是一個底層智能合約的更新而已，它從前端介面影響社交行為、替演算藝術作品譜出了新的創作形式、也更深遠地影響了創作者的創作邏輯與互動關係。</p><p>台灣藝術家有幸可以成為一個很好的共創案例，就看這週六的銷售狀況，不管如何都會在 fxhash 平台史留下一筆。</p>
