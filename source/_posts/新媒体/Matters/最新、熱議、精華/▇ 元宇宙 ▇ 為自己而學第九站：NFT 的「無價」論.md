referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2><strong>PART1：先說說NFT是什麼</strong></h2><p>它的全名是非同質化代幣（Non-Fungible Token）</p><blockquote>維基百科釋義：一種被稱為<a href="https://zh.m.wikipedia.org/wiki/%E5%8C%BA%E5%9D%97%E9%93%BE" rel="noopener noreferrer" target="_blank">區塊鏈</a>數位帳本上的資料單位，每個代幣可以代表一個獨特的<a href="https://zh.m.wikipedia.org/wiki/%E6%95%B8%E7%A2%BC%E8%B3%87%E6%96%99" rel="noopener noreferrer" target="_blank">數位資料</a>，作為虛擬商品所有權的電子認證或憑證。由於其不能互換的特性，非同質化代幣可以代表數位資產，如<a href="https://zh.m.wikipedia.org/wiki/%E7%BB%98%E7%94%BB" rel="noopener noreferrer" target="_blank">畫作</a>、<a href="https://zh.m.wikipedia.org/wiki/%E8%97%9D%E8%A1%93%E5%93%81" rel="noopener noreferrer" target="_blank">藝術品</a>、<a href="https://zh.m.wikipedia.org/wiki/%E8%81%B2%E9%9F%B3" rel="noopener noreferrer" target="_blank">聲音</a>、<a href="https://zh.m.wikipedia.org/wiki/%E8%A7%86%E9%A2%91" rel="noopener noreferrer" target="_blank">影片</a>、<a href="https://zh.m.wikipedia.org/wiki/%E6%B8%B8%E6%88%8F" rel="noopener noreferrer" target="_blank">遊戲</a>中的專案或其他形式的創意作品。</blockquote><p>（如果看上面那一段就可以懂，我佩服你）<br class="smart">（這跟閱讀素養沒有關連，真的）</p><blockquote>碼農老師的講義略述：<br class="smart"><strong>把數位資產+Hash碼(一組唯一碼)，放進區塊鏈虛擬幣帳本中。</strong></blockquote><p>試著用小小小白的方法拆解一下我的理解過程。</p><p><strong>1.「非同質化代幣」？可以說人話嗎？</strong></p><p>呣。既然有「非同質化代幣」，那就有「同質化代幣」吧？</p><p>對。同質化代幣就是如果普羅大眾認定某種貨幣的價值，其價值不管放在哪裡都一樣，真實世界裡我們使用的法幣（台幣／日幣／港幣／美金／歐元.... ）都是<strong>同質化代幣</strong>，我的一元美金跟你的一元美金是一樣的價值，都是一元美金。</p><p>「非同質化代幣」應該可以簡單想像成，我的一元美金上有一個歐巴馬的親筆簽名，結果歐巴馬的超級鐵粉出現，願意出一百元美金跟我換，於是我這張一元美金就不只是一元美金了，它可能會繼續流通，有人出一千美金買這張簽了名的一元美金，然後有一天歐巴馬沒人認識了，它跌到 0.5 美金......</p><p>價值是浮動的，決定在市場買家賣家的機轉上，這是「非同質化代幣」的概念。</p><p><strong>2.「數位資產」跟「代幣」的關連？</strong></p><p>數位資產直接用白話文理解，就是在數位世界裡的資產。<br class="smart">資產用真實世界理解，主要有兩大類：錢，和物品。</p><p>錢，就是加密貨幣，<br class="smart">物品，就是NFT。<br class="smart">兩者可以互通，我可以用加密貨幣買NFT，也可以把NFT賣掉換回加密貨幣。</p><p>再回頭講一下加密貨幣。</p><p>加密貨幣，比方為人熟知的比特幣，以太幣，狗狗幣貓貓幣，都是「同質化代幣」喔。我的一元以太幣，等於你的一元以太幣。這些虛擬的數位資產，以「幣」的形式被我們認知，可以再轉換成「法幣」。</p><p>今日我寫作的此時此刻此分，<strong>1元的比太幣，等於1,900多元美金</strong>，約等於56,500元台幣。請注意~~幣值是一直在浮動的，對匯率稍有理解的人應該都明白，法幣之間的湧動不會非常劇烈，因為有法（國家政府或中央金融機構）在控管。但加密貨幣是藉由演算法產生的，演算法是存在全球無數礦工的電腦裡，沒有一個中央機構控制它的升降，所以它的浮動會非常可怕。</p><pre class="ql-syntax" spellcheck="false">所以其實我剛才查詢以太的價格時，心裡不小心罵了一聲髒話。
因為，不過就是十天前我為了要參加一個NFT活動而買了一點點零頭的以太幣，當時的1元以太幣是 2,700多元美金耶。這....是在跌什麼意思的啊？！
果然加密貨幣的投資和交易不是我們這種心臟太小顆的人可以玩的。＠＠
</pre><p>好，拉回來非同質化代幣 — NFT。它可以被理解為在數位虛擬世界裡的物品，但目前絕大部分型式是以數位藝術創作居多，你要獲得某個 NFT，就要用加密貨幣去購買。</p><p><strong>你的錢錢沒有不見，只是變成喜歡的東西。</strong>——這句話在真實世界裡很常見不是嗎？在虛擬世界裡也一樣：<strong>你的加密貨幣沒有不見，只是變成你喜歡的 NFT。</strong></p><p>有趣的地方來了。因為加密貨幣價值是浮動的，NFT 的價值當然也是浮動的。如果一個 NFT 沒有人買，那它就沒有價值。如果一個 NFT 有很多人要買，價格很高，但是加密貨幣最後崩盤了，那它一樣沒有價值。反之亦然。</p><p>關鍵就在，市場覺得有沒有價值？你覺得有沒有價值？</p><p>認為 NFT 是泡沬，加密貨幣是騙局的人，價值觀就是傾向不相信這個機制會可以持久運作。</p><p><strong>3.NFT 的所有權？</strong></p><p>根據前述可以想像，非同質化代幣，代表的是一種沒有單一價值認定的價格，取決於買賣雙方以及市場環境，這樣的性質，在真實世界裡其實並不陌生——「藝術品買賣及收藏」。</p><p>所以，NFT 市場上為什麼絕大部分都是數位藝術品就不難理解了。一幅藝術家的畫作你說值多少呢。你願意出多少，它就值多少囉。</p><p><strong>那買實體畫跟買NFT有什麼不一樣？</strong></p><p>從所有權來說。<br class="smart">買實體畫作，你擁有它的所有權。<br class="smart">買NFT，你擁有的是「所有權的記錄」——這個數位資產目前在你手上的證明，而非數位資產本身。</p><p>這麼說來好像只是某種炫耀行為：我如果買了某一個 NFT，可以隨時對外界聲稱，這個 NFT所表彰的數位資產，目前是由我來【冠名】的。它的交易紀錄會被寫在區塊鏈上，轉了不管多少手都會有中間過程的紀錄。大概是這樣。</p><p>那你說，蛤這樣聽起來如果我要收藏什麼藝術品還是收藏實體的比較靠譜吧，至少所有權在手上啊。</p><p>(<s>對，沒錯。</s>)<s><br class="smart"></s>(<s>不對，我根本沒有錢收藏高單價藝術品。</s>)</p><p>請自行判斷作答。</p><p>但我再補充一個不同處。如果你買了一幅超高級的實體畫作放你家展示，假設不慎小孩潑了牛奶在上面，貓貓把它咬爛，或者家裡太潮濕它發霉了，又或者被賊偷走了，那麼，你也就失去它了。</p><p><strong>NFT 不是。如前述，你擁有的是所有權的紀錄，某個存在區域鏈的電子證明，一連串的密碼訊號。它不是存在你的電腦或手機裡，就算你的設備壞掉了，你用來買它的數位錢包還在，你的助記詞有保管好，只要重新連上一台的新的裝置，就可以永恆的存取它。</strong></p><p>其實在我查詢的資料裡，NFT 市場還是諸多不同的看法，有人因投資而大賺或大賠，有人被詐騙，有人財富自由。</p><p>我只能說這本身就是一種非同質化的討論，沒有辦法一概而論。</p><hr><h2><strong>PART2：說說我參與的 NFT 計劃</strong></h2><p>其實我從來沒有想要把NFT的「投資」納入思考中，我對元宇宙目前的一切抱持的態度都一樣：用不排斥新東西的好奇心去認識它。</p><p>前陣子知道了一個很有趣的計劃：<strong>Mintverse 第二宇宙辭典</strong>。我上一篇心得就提過（<a href="https://matters.news/@lingshu1019/276732-%E5%85%83%E5%AE%87%E5%AE%99-%E7%82%BA%E8%87%AA%E5%B7%B1%E8%80%8C%E5%AD%B8%E7%AC%AC%E5%85%AB%E7%AB%99-%E5%8F%83%E8%88%87%E5%8D%B3%E7%8D%B2%E5%BE%97-bafyreihvmgmoryks2quz35cqzgdil5iqvwhobkvwzjj7yvtq2mox3x475m" rel="noopener noreferrer" target="_blank">為自己而學：參與即獲得</a>）</p><p>它跟其它已存的 NFT 不太一樣的地方是，<strong>買家能成為這個 NFT 的創作參與者之一。</strong></p><p>目前市場上常見的狀況， NFT 就是由某個藝術家創作，發出一定的量讓大家買賣交易，又或者是某個名人發行，不管是純粹 show off 用，或者有被賦能與實體世界中的活動結合（比方有可以去某個演唱會換獎品，比方它是一場餐宴的入場券）等等，都是主要發行的項目方已經設定或決定好的內容。</p><p>但 Mintverse 第二宇宙辭典邀請 2,100 位參與者，成為創作這個 NFT 的一部分。</p><p>如果是畫畫或藝術創作，那我大概也無法參與，可是它是文字型的創作，而且門檻低到不行，創作的參與方式是：你會抽到一個詞彙，對這個詞彙寫出10-100字的新定義。結束。</p><p>讓我決定參與的臨門一腳是，這個 NFT 本身是免費的。只需要付出些微礦工費(gas fee) 。在活動宣傳起辦之初的宣傳約合台幣 5,000 元上下，我個人覺得是一個很划算的學習型投資。</p><pre class="ql-syntax" spellcheck="false">當然，如果硬要說的話，這個礦工費是很貴的！因為它在貴族以太鏈上發行，據項目方團隊的說法是因為過去的操盤經驗大多在此鏈。
</pre><p>在 NFT 的交易市場上，通常兩種費用都要付。第一個是你取得這個 NFT 的礦工費，第二個是 NFT 本身的價格。</p><p>但 mintverse 第二宇宙辭典計劃中發出的這 2,100 個 NFT 本身免費，參與者只需要付出兩段的礦工費用：</p><ol><li>取得這個 NFT 的礦工費</li><li>把你取得的這個，以 NFT 型態存在的詞彙，重新給予定義之後，上傳（寫入）到區塊鏈裡的礦工費。</li></ol><p>未來，這個 NFT 首先是屬於你的，但你可以把它賣掉，買家可以重新定義這個詞彙，但你做為它的第一個持有者／鑄造者／定義者的紀錄會永遠保存在區塊鏈上，其它人都可以看到。</p><p>對於喜歡文字的我來說，計劃本身的內容設定非常有趣，如果抽到「愛情」，或者是「真相」，又或者是「香菜」，又或者是「貓」。哇。可以重新定義這個詞，很威啊。XD</p><p>其實蠻符合對 NFT 交易的價值判斷，你覺得值不值得？</p><p>最後我抽到了一個，我也不知道該怎麼說的詞 🤓🤓</p><figure class="image"><img src="https://assets.matters.news/embed/ec2c3546-e8d1-4907-855c-048d9ea2c852.png" data-asset-id="ec2c3546-e8d1-4907-855c-048d9ea2c852" referrerpolicy="no-referrer"><figcaption><span>報告警官！你看得懂嗎？</span></figcaption></figure><p>實際上當時的花費（也不過就五天前）是 0.035 以太幣，約合新台幣 3,000 元左右。</p><p>看倌們如果有留意我最上面提到的今天的以太幣價格，就知道現在 0.035 以太幣只要 1,900多元。嘖。</p><p>不過我也不擔心。畢竟，就不是要用來投資的啊。</p><p><strong>一是這個計劃特別且創新</strong>，不只是純粹買一個 NFT 回來放著，而是自己可以成為一個大型計劃裡的一部分。而且永久留名。</p><p><strong>二是文字創作本身的趣味性，</strong>除了自己寫的，也能看到其它參與者重新對各種詞彙寫下超展開的新定義，有些真的非常驚艷。<a href="https://opensea.io/collection/mintverseword" rel="noopener noreferrer" 50%的在跳。很想問一句有事嗎 :P</p><p>這邊貼<strong> </strong><a href="https://mintverse.world/" rel="noopener noreferrer" target="_blank"><strong>Mintverse 第二宇宙辭典</strong></a> 的官網，以及我是因為這兩位大大的文章加入的，<a fee 高居不下，整個超越當初的設定，很多參與者都是第一次玩 NFT，小狐狸錢包裡只存了剛好可以玩這個活動的錢錢，不可能付比原本預期高出兩三倍以上的金額，這樣如果寫入的詞彙太少，活動原本設計的一些解鎖功能就失去意義了。</p><p>於是官方延長再延長，從原本的 42 小時，多延長了兩次的 42 小時，共 126 小時，目前應該是在下週一 （5/16） 的凌晨才會正式第一階段結束。進入第二階段的重新釋出灰燼詞彙。</p><p>有興趣的朋朋們也可以加入<a href="https://discord.com/invite/rugpullfrens" rel="noopener noreferrer" target="_blank"> DC 群組</a>，觀看更多討論。</p><h2>結語：</h2><p>我每次寫心得都像在寫論文。到底。<br 交易市場吧，應該也是最活躍的一個。在上面有幾條不同的區塊鏈交易，以太坊就是手續費（礦工費）最貴的一條。<strong>課程裡介紹的方法是走另一個比較便宜的鏈：polygon。這個是我在課後又向老師們詢問的一個問題。</strong></p><p>polygon的手續費相對以太鏈，真的是親民太多惹。</p><p>雖然在 Opensea 上面，都還是用以太幣交易各式各樣的 NFT，但如果這個 NFT
