<figure class="image"><img src="https://assets.matters.news/embed/7367c9d0-8e1e-4f6b-8a5d-df6f64eabd18.jpeg" data-asset-id="7367c9d0-8e1e-4f6b-8a5d-df6f64eabd18" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>Writing NFT beta 於 8 月 1 日推出後，功能快速迭代。</p><p>以下是本月更新的重點概覽。</p><h2><strong>改善收集流程最優先</strong></h2><p>用戶現可於 <a href="https://liker.land/campaign/writing-nft" rel="noopener noreferrer" target="_blank">Feature 頁</a>、<a href="https://liker.land/like128l3t64trehh8jnr8klqaj8c2sug6wlg4jxqf8" rel="noopener noreferrer" target="_blank">liker.land 的個人主頁</a>及<a href="https://liker.land/nft/class/likenft1faqzun5eucvh07j0fnrk7wehs966y32wujxrfuvlkzwdyfg03h6qnrscau" rel="noopener noreferrer" target="_blank">個別 Writing NFT 詳細資訊頁</a>收集 NFT。目前大部份 NFT 購買的流量均來自 Writing NFT 詳細資訊頁，使用情境是用戶於任何渠道，如社交媒體、即時通訊應用或電郵等分享 Writing NFT 詳細資訊頁的連結。例：點擊<a href="https://liker.land/nft/class/likenft1x99fupsmu9ccr3w00cemeyhuttx98jkzwgtc56fwu5zyp93kcrcsm0kagj" rel="noopener noreferrer" target="_blank">此鏈結</a>以直接訪問該 Writing NFT 的詳細資訊頁。改善此頁用戶介面的任務因而顯得優先。</p><p>本月有關 NFT 詳細資訊頁的更新包括：</p><ul><li>顯示銷售情況資訊、改良 og 縮圖及資訊顯示等</li><li>修正於手機顯示 Writing NFT 時的 UI 異常</li><li>使用更易明白的錯誤訊息</li><li>基本分享按鍵</li></ul><figure class="image"><img src="https://assets.matters.news/embed/75b7d287-d53f-4cef-8650-9d6c56e5f002.png" data-asset-id="75b7d287-d53f-4cef-8650-9d6c56e5f002" referrerpolicy="no-referrer"><figcaption><span>改善 Writing NFT 的 og 縮圖，在社交媒體分享時更有吸引力！</span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/db422578-2d78-49bb-9009-1e7dd8d19d19.png" data-asset-id="db422578-2d78-49bb-9009-1e7dd8d19d19" referrerpolicy="no-referrer"><figcaption><span>鼓勵用戶分享的設計
</span></figcaption></figure><p>一開始的 liker.land 頁面沒有連接錢包功能，後來加上了。用戶連結錢包後可查看錢包結餘、到 My Dashboard 頁面查看自己的收藏和著作、查看讚賞公民會籍狀態，及修改用戶設定等。</p><figure class="image"><img src="https://assets.matters.news/embed/0a331b18-803d-4d79-9f65-3d4ea2325197.png" data-asset-id="0a331b18-803d-4d79-9f65-3d4ea2325197" referrerpolicy="no-referrer"><figcaption><span>連接錢包，查看自己的作品和收藏</span></figcaption></figure><h2><strong>Writing NFT 轉贈功能</strong></h2><figure class="image"><img src="https://assets.matters.news/embed/7cbfa21c-a81a-4856-9789-3d52eaa6502b.png" data-asset-id="7cbfa21c-a81a-4856-9789-3d52eaa6502b" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>新增了轉贈 Writing NFT 的功能。此功能讓 Writing NFT 的使用場境變得更豐富，擁有 NFT 除了對自己有意義外，還可以當作心意轉贈給朋友。某些文字可能對朋友有特別深刻的意義，從前總不成把字列印出來送贈，現在 Writing NFT 提供了體面的方式了。</p><hr><h2><strong>二級市場交易分潤機制</strong></h2><p>目前二級市易交易的功能尚未完整，但初步的功能已於第一筆交易時實現，日後支援 NFT 轉手功能時，將按此機制分潤。</p><p>目前的分潤方式是：</p><ul><li>平台手續費 2.5% （由 Liker Land 收取）</li><li>持份者 10% （由內容的 ISCN 擁有者定義誰為持份者）</li><li>賣家 87.5%</li></ul><p>本月最具代表性的案例為章濤的《財科暗戰序》 Writing NFT，每筆交易收益的 10% 將分給 7 位持份者。</p><p>此機制尚在實驗中，日後很大機會作出修改。</p><figure class="image"><img src="https://assets.matters.news/embed/6ffb1b9f-c8dc-4942-aba5-e876ff55c951.png" data-asset-id="6ffb1b9f-c8dc-4942-aba5-e876ff55c951" referrerpolicy="no-referrer"><figcaption><span>每筆《財科暗戰序》 Writing NFT 收益的 10% 將分給 7 位持份者</span></figcaption></figure><h2><strong>兼容更多網站內容</strong></h2><p>本月嘗試包括把 WordPress （<a href="https://ckxpress.com/" rel="noopener noreferrer" target="_blank">ckxpress.com</a>）、Substack（<a href="https://blocktrend.substack.com/" rel="noopener noreferrer" target="_blank">區塊勢</a>）及任意平台（<a href="https://cheungto.io/" rel="noopener noreferrer" target="_blank">章濤</a>及<a href="https://www.pazu.com/" rel="noopener noreferrer" target="_blank">薯伯伯</a>的個人網站）出版成 Writing NFT，曾嘗試把 Patreon 文章出版但未能成功。</p><p>正在實驗中的還有 Matters 和 Medium 上的文章。Matters 文章由於已支援 ISCN，已證實能順利出版成 Writing NFT；Medium 文章的實驗則預計於下個月內見結果。</p><p><br></p><p>LikeCoin Writing NFT 開放整合任何寫作平台，希望未來能支援更多不同的站點。</p><hr><h2><strong>其他有關 Writing NFT 的小工具</strong></h2><figure referrerpolicy="no-referrer"><figcaption><span>感謝 wancat 幫忙開發的 NFT Dashboard</span></figcaption></figure><p>雖然 Writing NFT 的交易資訊都是公開的鏈上數據，但沒有好的用戶介面工具很難查看。感謝 <a href="https://liker.social/web/@wancat"
