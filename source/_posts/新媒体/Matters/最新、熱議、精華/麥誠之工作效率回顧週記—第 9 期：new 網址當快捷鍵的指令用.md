<p>自從 Google 推出 <a href="https://www.google.com/url?q=https://get.new/&sa=D&source=editors&ust=1638780269769000&usg=AOvVaw2sSNGuPyI5gEvVllREbPXF" rel="noopener noreferrer" target="_blank">.new</a> 頂層域名後，其作用就像一個快速建立檔案的網絡快捷鍵網址。例如，我可以通過 <a href="https://www.google.com/url?q=https://doc.new&sa=D&source=editors&ust=1638780269770000&usg=AOvVaw2i4qGeFSMNZXOqxsqtfMKU" rel="noopener noreferrer" target="_blank">doc.new</a> 來快速建立新 Google Docs 文件，又或者 <a href="https://www.google.com/url?q=https://sheet.new&sa=D&source=editors&ust=1638780269770000&usg=AOvVaw2BAAPMxFsHH1BS70s0_FVb" rel="noopener noreferrer" target="_blank">sheet.new</a> 來建立 Google Spreadsheet，及（你預想到的）<a href="https://www.google.com/url?q=https://slide.new&sa=D&source=editors&ust=1638780269770000&usg=AOvVaw0_agj1yN0kvTtaw3NPN9xz" rel="noopener noreferrer" target="_blank">slide.new</a> 來建立 Google Slides 文件。</p><figure class="image"><img src="https://assets.matters.news/embed/02bc7046-f57d-4dd8-a0b2-77aee44661da.jpeg" data-asset-id="02bc7046-f57d-4dd8-a0b2-77aee44661da" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>咦，人家名稱是 Google Slides，但我們用的是 slide.new，沒有 s 的。是，其實 Google 這三大辦公軟件，是單數或複數的也可以互通使用。即 doc.new, docs.new, sheet.new, sheets.new, slide.new, slides.new 等都行。</p><figure class="image"><img src="https://assets.matters.news/embed/19f4e9c7-a452-4ba1-99b9-5059f39aeab3.png" data-asset-id="19f4e9c7-a452-4ba1-99b9-5059f39aeab3" referrerpolicy="no-referrer"><figcaption><span>在網址鍵入 slides.new 就可以建立一個新的 Google Slides 簡報文件</span></figcaption></figure><p>在現今這個隨時都開著瀏覽器的年代，一有靈感時，CTRL+T 開個新 tab 再 doc.new 就可以開始打字，馬上把靈感記下來。</p><h2><strong>快速建立微軟網頁版 Office 文件</strong></h2><p>有 Google Docs 的，就有其他服務的快速建立指令域名。微軟的網頁版 Office 同樣有捷徑。我們可认通過 <a href="https://www.google.com/url?q=https://word.new&sa=D&source=editors&ust=1638780269772000&usg=AOvVaw08p2DHUkfIdacVd88NR7IQ" rel="noopener noreferrer" target="_blank">word.new</a>, <a href="https://www.google.com/url?q=https://excel.new&sa=D&source=editors&ust=1638780269772000&usg=AOvVaw18COBZWkeEih5yr1kWfvP9" rel="noopener noreferrer" target="_blank">excel.new</a>, 及 <a href="https://www.google.com/url?q=https://powerpoint.new&sa=D&source=editors&ust=1638780269772000&usg=AOvVaw3NpaQz7XDRsHlqa6yugCKO" rel="noopener noreferrer" target="_blank">powerpoint.new</a> 來建立網頁版的 Office 三寶文件。</p><p>但是打 powerpoint.new 也太長吧，所以我們也可以利用 docx.new 來建立 Word 文件及 ppt.new 或 pttx.new 來建立新 PowerPoint 文件。而 doc.new 嘛，被 Google Docs 佔用了。而 xls.new 被另一個服務佔用了，xlsx.new 則似乎微軟的證書佈署不正確。</p><p>所以，若要建立微軟 Office 文件，我會用 word.new, excel.new 及 ppt.new 這三個網址。</p><figure class="image"><img src="https://assets.matters.news/embed/75095c68-ae7b-4334-a282-a4f0de8ac92a.png" data-asset-id="75095c68-ae7b-4334-a282-a4f0de8ac92a" referrerpolicy="no-referrer"><figcaption><span>在瀏覽器輸入並瀏覽 excel.new，就可以在 office.com 上建立一個新的 Excel 檔案</span></figcaption></figure><h2><strong>pen.new 及 gist.new </strong></h2><p>另外，我平常經常使用 <a href="https://www.google.com/url?q=https://codepen.io&sa=D&source=editors&ust=1638780269774000&usg=AOvVaw1nky5IH5XXGAdqJDscbVLk" rel="noopener noreferrer" target="_blank">codepen.io</a> 來實驗各種 <a href="https://www.google.com/url?q=https://matters.news/tags/VGFnOjc3MDQ0&sa=D&source=editors&ust=1638780269774000&usg=AOvVaw3aJUjpVSfp548WLDpPJuZI" rel="noopener noreferrer" target="_blank">CSS 效果</a>。當有相關靈感時，我在網址欄馬上輸入 pen.new，就有一個新的 HTML, CSS, JavaScript 線上編輯器可以<strong>玩</strong>了。實驗完畢後，有甚麼代碼需要記錄或者分享，我也可以使用 gist.new 來建立一段代碼分享網址。所以，每個人也可能通過不同的 .new 捷徑來加速他們日常的網絡服務操作。</p><p>注：關於這個玩字，稍後我另立一篇文章重述我的一個<a href="https://www.google.com/url?q=https://slides.com/makzan/play-and-explore&sa=D&source=editors&ust=1638780269774000&usg=AOvVaw3YlDfb3QC5cs_3-f1b0bg_" rel="noopener noreferrer" target="_blank">《玩 x 實驗》</a>分享。</p><h2><strong>更多不同服務</strong></h2><p>以上是我常用的 .new 網址指令。因應各人不同用途，還有更多可能適合位的常用服務也不為奇，例如 Adobe 有一系列的，如 <a href="https://www.google.com/url?q=https://photoshop.new&sa=D&source=editors&ust=1638780269775000&usg=AOvVaw0IKA2L1DHlK8Lwk-ge-5df" rel="noopener noreferrer" target="_blank">photoshop.new</a> 及 Acrobat 的 <a href="https://www.google.com/url?q=https://jpgtopdf.new&sa=D&source=editors&ust=1638780269776000&usg=AOvVaw01jqHpdatevtCvFda9d9Bc" rel="noopener noreferrer" target="_blank">jpgtopdf.new</a> 等。</p><figure class="image"><img src="https://assets.matters.news/embed/4758e57a-b06b-47e3-9eec-628a083a06e3.png" data-asset-id="4758e57a-b06b-47e3-9eec-628a083a06e3" referrerpolicy="no-referrer"><figcaption><span>photoshop.new 建立網頁版 Photoshop 文件</span></figcaption></figure><p>其他服務有如 webex.new, task.new, note.new, gist.new, pen.new 等，在 <a href="https://www.google.com/url?q=https://whats.new/shortcuts/&sa=D&source=editors&ust=1638780269776000&usg=AOvVaw2T1w2gRxer0z5Ew7R9AouJ" rel="noopener noreferrer" target="_blank">whats.new</a> 網站上，有列出各服務的 .new 捷徑列表。</p><p><br></p><figure class="image"><img src="https://assets.matters.news/embed/7a4146c4-349c-4b59-8649-48d018fa55f5.png" data-asset-id="7a4146c4-349c-4b59-8649-48d018fa55f5" referrerpolicy="no-referrer"><figcaption><span>Whats.new 上的各 .new 域名列表</span></figcaption></figure><h2><strong>我第一次意識到網址可以當指令用: EditGrid</strong></h2><p>尤記得 2007 年左右，我有幸在香港中文大學聽 <a href="https://www.google.com/url?q=https://en.wikipedia.org/wiki/EditGrid&sa=D&source=editors&ust=1638780269778000&usg=AOvVaw1igssgKcNI_LGRhwg_UUCM" rel="noopener noreferrer" target="_blank">EditGrid</a> 團隊分享他們的線上 spreadsheet 服務，其服務網址本身就是 API，可以通過網址指令來單獨存取某一格或某一段資料，也可以自行更改最後的副檔名來得出不同的輸出格式。</p><p>例如 editgrid.com/sheets/xxxx_id_here/B6 可以得出 B6 格的資料。或者 editgrid.com/sheets/xxxx_id_here/B6.txt 可以得出純文字格式。又或者 editgrid.com/sheets/xxxx_id_here.xls 便可以下載為 Excel 格式等。</p><p>注：上述網址是回憶示例， EditGrid 團隊早年已被收購及服務已停用，也不是現在的 editgrid.com。</p><p>那是我第一次意識到網址可以這樣使用。以下多介紹兩個網址當指令用的工具。</p><h2><strong>inQRCode.com</strong></h2><p>我於 2008 年左右和朋友一同推出的一個(已下線) 的短筆記服務，也加入了類似功能，可以在網址最後加上 .txt, .qrcode 等，當中 .qrcode 會把網址變成一個二維碼，方便馬上使用手機掃描，將筆記帶上路。</p><p>提起二維碼，我另外也製作了一個 <a href="https://www.google.com/url?q=https://inqrcode.com&sa=D&source=editors&ust=1638780269779000&usg=AOvVaw2vGDmce_JDHG3GT-RWZQU8" rel="noopener noreferrer" target="_blank">inqrcode.com</a> 的快速生成二維碼指令網址，只要在任何網址前加上 <strong>inqrcode.com/</strong> 就可以得出該網址的二維碼。方便平常製作二維碼但不想看廣告，或我在手機網站開發課堂上，隨時可以將示範網址給學生掃描實操。</p><p>注：記得要加上斜號喔!</p><figure class="image"><img src="https://assets.matters.news/embed/6cfb7b60-46ad-4072-9d8e-708ce07fb0ed.png" data-asset-id="6cfb7b60-46ad-4072-9d8e-708ce07fb0ed" referrerpolicy="no-referrer"><figcaption><span>在任何網址前，加上 inqrcode.com/ 就可以得出該網址的二維碼</span></figcaption></figure><h2><strong>Wordpress /edit 功能</strong></h2><p>供自行架設的 Blogging 軟件 WordPress，有一個外掛名為 <a href="https://www.google.com/url?q=https://wordpress.org/plugins/slash-edit/&sa=D&source=editors&ust=1638780269780000&usg=AOvVaw0gDfBYXb1o5mA9Pbr1axL1" rel="noopener noreferrer" target="_blank">Slash-Edit</a>，當以編輯者身份瀏覽自己的 WP 網站時，可以在任何文章或頁面網址的最尾，加上 /edit 以瀏覽到編輯狀態。很適合檢查網站時，見到有錯字或更新需要，馬上進入此頁的修改狀態，而不用去 /wp-admin 再找回剛剛的文章再按編輯那麼兜轉。</p><p>以上這些網址作為快捷鍵指令，可以使平常使用瀏覽器時，更直接的跳轉到我們想要的網絡資源上。更少的步驟及額外的載入，更多的專注。</p><p>— 麥誠 Makzan，2021-12-06。</p><hr><p>我是麥誠軒（<a href="https://makzan.blog/about/" rel="noopener noreferrer"
