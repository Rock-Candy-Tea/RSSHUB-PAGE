<h2>下訂冷錢包</h2><p>其實已經想買冷錢包好一陣子，原本是想等 CRO 解綁之後，用那筆來買 ...但後來因為決定要繼續質押 ... 因為想說 .... 拿來訂 Spotify，想不到 ... CRO 持續跌，看來到時好像還要補一下質押量 .... 呃，反正質押還沒到期，而且 Spotify 還有前三個月免費 ... 就 ... 到時候再看看吧。</p><p>於是當下〔大概兩個禮拜前〕立馬把錢包裡存了一陣子的 UST 換成 CRO 轉到 Crypto.com，我之前會把流動挖礦的收入，定期一部份轉成UST，結果是 ... 還好拿來買冷錢包了！？？</p><figure class="image"><img src="https://assets.matters.news/embed/206bb270-5d53-4030-8187-f083749c8104.jpeg" data-asset-id="206bb270-5d53-4030-8187-f083749c8104" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>然後用 Crypto Pay 付款〔扣款 CRO〕，在 Ledger 官網下訂了一隻 LEDGER NANO S PLUS，當時折成台幣是 2503，LEDGER NANO S PLUS 是沒有藍芽的版本，所以必需接線連到電腦使用。</p><p>為什麼不用 Crypto.com 的 VISA 卡付款呢？因為如果用 Crypto Pay 的話可以回饋 5% 比較多，不過目前能用 Crypto Pay 的地方好像不多就是了，付款完之後馬上在錢包裡就看到回饋了 10.782 個 CRO。</p><p>過了幾天查看訂單 ... 冷錢包就已經在路上囉！</p><figure class="image"><img src="https://assets.matters.news/embed/168e71f1-5a0a-44b3-a9f7-b96515bd3c64.jpeg" data-asset-id="168e71f1-5a0a-44b3-a9f7-b96515bd3c64" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><h2>收到冷錢包</h2><p>冷錢包送來那天我剛好不在，於是郵差梭，不然放到郵居招領，我下意識回了好，掛了電話才想到，欸！？不是應該第二天再送一次的嗎？好吧算惹 .... 就多走走路吧！！</p><p>第二天到郵局去領了包裹囉！我的冷錢包是從法國寄來滴 ... 〔這裡提醒一下，購買冷錢包請務必 ... 到官網去買！</p><figure class="image"><img src="https://assets.matters.news/embed/90b2a84d-cd89-4434-bb1a-77cd5dc3a01f.jpeg" data-asset-id="90b2a84d-cd89-4434-bb1a-77cd5dc3a01f" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><h2>開箱囉</h2><p>外包裝的是牛皮紙袋，裡面有防撞保護。</p><figure class="image"><img src="https://assets.matters.news/embed/4af9fbf6-82c6-42da-bac3-73fb63a5638e.jpeg" data-asset-id="4af9fbf6-82c6-42da-bac3-73fb63a5638e" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>外包裝很簡潔，上面寫著我買的冷錢包型號！</p><figure class="image"><img src="https://assets.matters.news/embed/b657aa1d-4424-429b-9cc9-2930f713f5cf.jpeg" data-asset-id="b657aa1d-4424-429b-9cc9-2930f713f5cf" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>打開後有黑色的盒子，感覺蠻有質感滴！</p><figure class="image"><img src="https://assets.matters.news/embed/1ee4818b-0f6f-4e3e-a5ee-7415e4c78d20.jpeg" data-asset-id="1ee4818b-0f6f-4e3e-a5ee-7415e4c78d20" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>打開後看到冷錢包本尊，橘色的小盒子裡面是助記詞小卡，初始設定時可以把助記詞寫上去。</p><figure class="image"><img src="https://assets.matters.news/embed/3e4703a2-f02e-48d0-85ab-e619825d71bf.jpeg" data-asset-id="3e4703a2-f02e-48d0-85ab-e619825d71bf" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>盒子的第二層下面放著說明書，說明書下面還有一個黑色紙盒是傳輸線〔Tape - C <-> USB〕 跟一個吊環。</p><figure class="image"><img src="https://assets.matters.news/embed/e9ed09e4-cfea-41e2-b48d-3926210574bc.jpeg" data-asset-id="e9ed09e4-cfea-41e2-b48d-3926210574bc" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>這些就是主商品本人，是說 ... 那個吊環 .... 弄不上去呀！</p><figure class="image"><img src="https://assets.matters.news/embed/56a042a9-d101-4783-8eb8-5ed0be6bf2e4.jpeg" data-asset-id="56a042a9-d101-4783-8eb8-5ed0be6bf2e4" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><h2>錢包初始化</h2><figure class="image"><img src="https://assets.matters.news/embed/014436bb-dd6d-4480-9db0-7673258572fe.jpeg" data-asset-id="014436bb-dd6d-4480-9db0-7673258572fe" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>我先到官網去下載了硬錢包使用的軟體，安裝好了之後，軟體一開始就有使用說明，教如何把冷錢包初始化，不過軟體沒有繁中的選項，哭哭。</p><p>初始化有兩步：</p><ol><li>設定 PIN 碼</li><li>初始化錢包，抄下助記詞〔嗯，我決對不會把助記詞拍出來的！〕</li></ol><p>然後，冷錢包上有兩個按鈕，向左向右，兩個同時按為確定。</p><hr><h2>建立帳號</h2><p>Ledger 這款冷錢包我覺得雖一比較麻煩的是，沒辦法在錢包的面版上看到餘額，需要透過 Ledger Live 軟體其他支援的錢包去查看，錢包連接上軟體後，需要先安裝應用程序，有點像在錢包上加上網路的這個步驟，我的這隻冷錢包容量顯示是 1.28 MB，安裝了五網路之後〔Binance Smart Chain，BTC，Cosmos，ETH，Polygon〕用掉了 198 K，看來可以安裝蠻多的。</p><p>安裝完網路後，再添加帳戶，即可以產生錢包位置！</p><figure class="image"><img src="https://assets.matters.news/embed/ebdcb4bc-bdb9-47c7-a7c8-cd5f4195acbf.jpeg" data-asset-id="ebdcb4bc-bdb9-47c7-a7c8-cd5f4195acbf" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>之後，就可以透過這些位置把幣傳進錢包裡去囉！我把一些本來放在熱錢包裡的 BTC 跟 ETH 傳進去囉！</p><figure class="image"><img src="https://assets.matters.news/embed/b58eca46-f4cd-41d1-a0ee-227518a82fda.jpeg" data-asset-id="b58eca46-f4cd-41d1-a0ee-227518a82fda" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><h2>Cosmos 生態，連結 Keplr</h2><p>安裝了 Cosmos 之後，在 Ledger Live 只能產生 ATOM 的地址，因此如果要把 Cosmos 生態上的其他幣放進冷錢包，就要連上 Keplr。</p><ol><li>到 Keplr 錢包裡點 Add Account。</li><li>選 Import ledger</li><li>輸入帳戶名稱</li><li>根據 Keplr 錢包上顯示的指示操作就可以了</li></ol><figure class="image"><img src="https://assets.matters.news/embed/46e0ecfa-1526-420e-b01e-050db97e326f.jpeg" data-asset-id="46e0ecfa-1526-420e-b01e-050db97e326f" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>之後我轉了一點 ATOM 進去，然後試著在 Keplr 上做質押，確認交易的動作都需要在冷錢包上執行，連接上的冷錢包帳戶也不像其他直接在 Keplr 上開的帳戶那樣有查看助記詞的選項〔只能改名跟刪除〕。</p><figure class="image"><img src="https://assets.matters.news/embed/f7849af6-2faf-4ed9-bcec-fdebe1627c8e.jpeg" data-asset-id="f7849af6-2faf-4ed9-bcec-fdebe1627c8e" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><h2>為什麼需要買冷錢包？為什麼選擇 Ledger ？</h2><p>終於入手了冷錢包，來整理一下為什麼我覺得需要入手一個冷錢包。</p><p>大概就是因為 ....比較安全，一般的說法是所謂的離線儲存，但其實指的不是加密貨幣的離線儲存，基本上幣就是存在於區塊鏈上，應該指的是控制權的離線，也就是說交易時需要透過一個硬體上的確認，而它的助記詞也不會出現在使用的電腦跟手機上，也就是 ... 助記詞也是離線滴，但是 ... 如果你的助記詞洩露了 ... 嗯，就 ... 一樣誰都能拿走這些幣，所以說基本上冷錢包是比較安全的一種，前提是保管好助記詞。</p><p>至於為何選了 Ledger 的冷錢包，主要是 ... 支援性比較好囉！尤其是我們最常使用的 Keplr 也只支援 Ledger 的冷錢包，所以也能使用 Ledger 冷錢包也能質押像是 ATOM、OSMO ...等幣。</p><p>最後 ... 在這跌跌不休的時刻，我收到了我的冷錢包，心得是 ... 放進去的幣都變薄了 ... XD，就 ... 默默的分批小額買 BTC ，再把冷錢包存飽一點囉！</p><hr><p>Peace on Earth</p><p>歡迎追蹤<a href="https://matters.news/~travelcat" rel="noopener noreferrer" target="_blank"><strong>【貓咪國度】</strong></a>，追蹤即可收到廣播訊息唷！！會不定期舉辦小活動唷，如果想訂閱，我也不反對！〔〔欸！？</p><p>驗證人節點：<a href="https://coding-everyday.notion.site/Coding-Everyday-af2504155eb442d6b209f251d0e8fd2f" rel="noopener noreferrer" target="_blank"><strong>Coding Everyday｜打碼大聯盟</strong></a><br class="smart">我的 NFT 作品：<a href="https://opensea.io/EricaLiu" rel="noopener noreferrer" target="_blank"><strong>OpenSea</strong></a> ︱<a href="https://akaswap.com/tz/tz1UeSaVq2c3vUVuobgKgi5u725rxcU9cFHt" rel="noopener noreferrer" target="_blank"><strong>akaSwap</strong></a><br class="smart">我的 NFT 收藏：<a href="https://opensea.io/Erica-Cat" rel="noopener noreferrer" target="_blank"><strong>OpenSea</strong></a><br class="smart">我的兩本旅行電子書：<a href="https://travostyle.com/book/view/231" rel="noopener noreferrer" target="_blank"><strong>克羅埃西亞巴士旅行日記</strong></a>、<a href="https://travostyle.com/book/view/361" rel="noopener noreferrer" target="_blank"><strong>走跳沖繩．景點交通全攻略</strong></a><br class="smart">我的個人布落格：<a href="http://www.ericaworld.tw/" rel="noopener noreferrer" target="_blank"><strong>．說走就走．<em>Erica的．旅行．體驗．活在當下</em></strong></a><br class="smart">粉絲專頁：<a href="https://www.facebook.com/www.ericaworld.tw" rel="noopener noreferrer" target="_blank"><strong>說走就就x貓咪旅行中</strong></a><br class="smart">在其他平臺也能找到我：<a href="https://vocus.cc/user/@ericaworld" rel="noopener noreferrer" target="_blank"><strong>方格子</strong></a><strong> </strong>| <a href="https://matters.news/@erica072" rel="noopener noreferrer" target="_blank"><strong>我還有另一個Matters</strong></a> | <a href="https://www.potatomedia.co/user/12939b9d-b6ad-4296-ad6c-6285cd07b11e/posts" rel="noopener noreferrer" target="_blank"><strong>Potato Media</strong></a>︱<a href="https://noise.cash/u/EricaLiu" rel="noopener noreferrer" target="_blank"><strong>Noise.cash</strong></a></p>
