<p>NFT在學術名詞是 Non-Fungible Token 非同質化代幣。一般人看NFT作品是一個頭象、一張JPG、一段聲音或一段影片。但....... 實質上有多少人覺得NFT是藝術品呢?</p><p>我手上的NFT...... 大約有一千至二千多項吧，不過，我手上的 NFT 不只是用來買賣，而是用來挖礦及有不同的功能。</p><figure class="image"><img src="https://assets.matters.news/embed/601bf085-8bba-4604-b479-cc0e71284b5d.png" data-asset-id="601bf085-8bba-4604-b479-cc0e71284b5d" referrerpolicy="no-referrer"><figcaption><span>(我在使用NFT來挖幣)</span></figcaption></figure><h2><strong>現時市場上推廣NFT 有兩個方向：</strong></h2><p>第一種情況，是將現實生活的藝術創作轉變成為NFT項目，並且希望有人「購買」。(相信，這也是大家的想法，一般大家稱為數碼藝術品)</p><p>第二種情況，是將原本虛擬項目再藉著NFT觀念而產生出新的價值，當中發展得最好是區塊鏈遊戲。</p><p>我自己有很大量的NFT，主要都是遊戲項目或者數碼娛樂的項目。另一方面，我也是NFT遊戲的營運負責人。故此，我對NFT的觀念是<strong>「用NFT」及「NFT要有用」</strong>。</p><p>NFT實質上是由智能合約所組成的項目，現時運用在藝術品及遊戲上的NFT普遍是用 ERC721 及 ERC1155 的區塊鏈協定來製作 或叫 鑄造。對於用家來說，可以完全不理這些區塊鏈協定，只需要知道每一張 JPEG / ANIMATION 都不只是一張圖片，背後仍有一組編碼及區塊鏈智能合約，這些都可以在區塊鏈瀏覽器看得到鑄造日期、交易帳號及價值變化。</p><p>當中，這一組智能合約所建立的 JPEG / ANIMATION / 音樂，可以是只有一張，又或者可以有多個副本，並且您更可以為每一個副本建立編號。</p><p>普遍的人理解將NFT運作在遊戲上，即是代表我的武器、防具或遊戲人物都有獨特性，有保障，不會容易被盜，甚至因為獨一無二而有價值。</p><p>這些都可以是真事，但是作為遊戲人，就可以將NFT上的智能合約發揮出來使用。</p><p>例如，怎樣的NFT可以組合成一個新的項目? (將一些普通的NFT卡合成為更稀有的NFT卡)</p><p>又例如，可以透過智能合約賦予那NFT一些特別的功能。</p><p>好像上圖一樣，可以給予不同的NFT賦予一個「算力數值」，如一般區塊鏈貨幣一樣，可以用來質押挖其他貨幣。</p><p>當然，也可以用來買賣:</p><figure class="image"><img src="https://assets.matters.news/embed/a8eb2e02-0631-444c-9506-2f0f613090b9.png" data-asset-id="a8eb2e02-0631-444c-9506-2f0f613090b9" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>與及不同的遊戲裝備:</p><figure class="image"><img src="https://assets.matters.news/embed/47c5a3f4-67c3-4e8a-bd9d-d065c1a1c529.png" data-asset-id="47c5a3f4-67c3-4e8a-bd9d-d065c1a1c529" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><figure class="image"><img src="https://assets.matters.news/embed/d9218322-d268-4055-83dd-4b86068ec7bd.png" data-asset-id="d9218322-d268-4055-83dd-4b86068ec7bd" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>我覺得，在這一個年頭，NFT必須要有用才會建立<strong>「轉手的價值」</strong>。要不然，就只是好像炫耀，買一個付費頭象來換成自己的頭象。</p><p>現時，NFT落地應用中，區塊鏈遊戲做得相當不錯，更開始吸引傳統的遊戲人加入開發區塊鏈項目。</p><p>另一邊廂，貴價品的NFT也炒賣得相當好，在OPENSEA 的 100大項目排名上，金額都相當誇張，可以看到很多人是「有錢沒有地方花」。</p><figure class="image"><img src="https://assets.matters.news/embed/92fb7de9-d399-42c4-b8f0-1fae4b603f7f.png" data-asset-id="92fb7de9-d399-42c4-b8f0-1fae4b603f7f" referrerpolicy="no-referrer"><figcaption><span>Opensea 的排行榜 - https://opensea.io/rankings</span></figcaption></figure><p><strong>可是，我卻覺得現實世界及媒體將 NFT 神化了，更有不少KOL / YOUTUBER 將 NFT 視為創作人的「救世神藥」，只要造了NFT就有新的出路了。這樣只會害了藝術品創作人，因為NFT市場並不簡單。</strong></p><p><strong>我在香港看到一些公司拉了一些傳統藝術家去建立NFT，並且大肆報導及吹棒，心中其實有點不悅，因為這樣給予傳統藝術家「希望越大，失望將會更大」。個人覺得應該要一步一步建立，特別是依輔著區塊鏈社交平台來建立，將會「事半功倍」。</strong></p><p>實質上，NFT的世界並不是放了圖片至NFT平台便會有交易。現時已是2021年，我們不要回到2000年的千禧年代，以為建立了一個 .COM 網站就可以與全球的人聯絡。 我們要成熟一點，現時放一個NFT上網，已經簡單到不能再簡單，花費已便宜至不得了。(根本不用 0.1 台幣)</p><p>全球最大的NFT交易平台OPENSEA 也做了極多的教程 : <a href="https://support.opensea.io/hc/zh-tw/articles/360063498333-%E6%88%91%E8%A6%81%E5%A6%82%E4%BD%95%E4%B8%8A%E6%9E%B6-NFT%E4%B8%A6%E9%80%B2%E8%A1%8C%E9%8A%B7%E5%94%AE-" target="_blank" class="keychainify-checked">我要如何上架 NFT並進行銷售？ - Opensea</a></p><figure class="image"><img src="https://assets.matters.news/embed/bba03707-6171-4934-8fe4-64282598ba7a.png" data-asset-id="bba03707-6171-4934-8fe4-64282598ba7a" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>一點進入 Opensea 的 Help Center 便有不同的教程介紹，並且是多語言。根本不用任何一個博主或使用者去寫任何的教程，因為 Opensea 上已有最完善最新的版本。</p><h2>那麼，我們要推出 NFT 到底要注意甚麼?</h2><p>就是，您覺得誰會去買您的NFT了。</p><p>若果是遊戲的NFT，內裡多數是裝備武器，買這樣NFT的人必然是同一遊戲的玩家，他們想要這些神裝的武器來強化自己。</p><figure class="image"><img src="https://assets.matters.news/embed/428f72bd-9f58-44bd-a626-a0716ea766a1.png" data-asset-id="428f72bd-9f58-44bd-a626-a0716ea766a1" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>若是「NFT藝術品」呢? 區塊鏈上不是有很多的 NFT 收藏家嗎?</h2><p>是啊，是有很多的NFT收藏家，不過，您知不知道他們在那裡? 與及為甚麼他們要買您的NFT?</p><p>我們有在 Steem 及 Matters 上發文章的經驗，應該也知道「單純發帖是不會有收入」，當中要加上「好內容」及「社交關係」。</p><p>推出 NFT 是同理，若果我們建立了 NFT 後，沒有同時建立支持 NFT 的社群。最終，尤如中國大陸的碰瓷黨一樣，希望碰碰運氣，遇上一位神奇又有俠義心腸的有錢人，願意買我的創作。這一個過程，真的相當相當難。<strong>若用粵語說，就是「等運到」。</strong></p><p><strong>故此，要加入 NFT 行列，有三點必須要做:</strong></p><p>第一，您不止是發佈NFT，您更要進入NFT的社群之中。您很難找到「NFT收藏家」，但您會比較容易找到 NFT 創作人。而大部份這些創作人都是在Twitter 及 Telegram 的群組內。</p><p>第二，您要透過不同的區塊鏈人士常常瀏覽的平台，去發佈創作 NFT 的過程。若對象是外藉人士，Twitter 及 Medium 已必不可少。</p><h2>若對象是華人，現時基本上沒有這一個社群，不如，就在 Matters 開始！</h2><p>第三，您要持之以恆，因為起初的一段時間，基本上接近零交易，您可以創作一些 NFT 來送給朋友，也可以用 POAP 來製作更多 NFT Badge。就算沒有交易，您仍要不斷做。</p><p>當您開始賣出 NFT 後，您會發覺購買者不是在買您的 NFT這麼簡單。您會發覺，<strong>這些購買您的NFT的人，他們是欣賞您，與及支持您繼續創作。</strong></p><p>您，願意當NFT藝術家嗎?</p><p>願意的話，就下定決心，於未來一年時間裡不住創作，不住上傳作品。您必定會遇到願意支持您的人。</p><p><br></p><p>本文是參加 <a class="mention keychainify-checked"
