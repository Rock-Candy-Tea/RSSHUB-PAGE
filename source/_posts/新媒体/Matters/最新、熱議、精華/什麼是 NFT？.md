<p>如何簡短解釋 NFT 是什麼？「記在區塊鏈節點共同筆記本上的資產持有權記錄。」但我愈想愈覺得，連自己都快聽不懂。</p><p>工程師可能會覺得無聊地說，符合 <a href="https://ethereum.org/en/developers/docs/standards/tokens/erc-721/" rel="noopener noreferrer" target="_blank">ERC-721</a> 或 <a href="https://ethereum.org/en/developers/docs/standards/tokens/erc-1155/" rel="noopener noreferrer" target="_blank">ERC-1155</a> 類似規格的東西，就是 NFT 啊。於是更多人聽不懂了。</p><p>不如，先來看一則<a href="https://ivod.ly.gov.tw/Play/Clip/1M/137685" rel="noopener noreferrer" target="_blank">台灣立法院的質詢影片</a>（可以從 01:42 開始）：</p><blockquote>立法委員：NFT 會不會課稅？<br class="smart">財政部長：NFT 是屬於商品、勞務的部分，按照營業稅法就要課稅。<br class="smart">立法委員：究竟 NFT 的主管機關是誰？金管會？經濟部？</blockquote><p>看到這裡，除了擔心自己買的 NFT 可能被課稅之外，你可能還會跟我一樣困惑：聽說 NFT 有很多種耶，可以是藝術品、會員卡、活動票券、遊戲虛寶、「元宇宙」土地……可以說「萬物皆可 NFT」，怎麼可能適用單一的主管機關呢？</p><blockquote>～工商服務時間～<br class="smart">你是台灣人嗎？你知道自己的選區立委，平常都在幹什麼嗎？<br class="smart">這些立委有在好好質詢政府官員，或者其實都在包庇政府嗎？<br class="smart">我在 g0v 提案了《一生監督》App，<br class="smart">想加入早期內部測試，來提供意見嗎？等你來填<a href="https://www.craft.do/s/PlJXdU2G3Qizuz" rel="noopener noreferrer" target="_blank">申請表單</a>！<br class="smart">想加入開發、設計或營運嗎？先來看看三分鐘的<a href="https://youtu.be/WifUTX7pkMs" rel="noopener noreferrer" target="_blank">提案影片</a>！<br class="smart">歡迎來聯繫我喔。<br class="smart">～工商服務結束～</blockquote><h2>NFT 裡面有什麼？</h2><p>NFT 真正寫上區塊鏈的資訊，往往只有 metadata，而不是圖片本體，因為將大量資料寫上區塊鏈的成本實在太高了。以我的 <a href="https://opensea.io/assets/0x8515ba8ef2cf2f2ba44b26ff20337d7a2bc5e6d8/549" rel="noopener noreferrer" target="_blank">Travelogger #549</a> 為例，假若你照著<a href="https://youtu.be/_1PSvqho8UE?t=5m47s" rel="noopener noreferrer" target="_blank">這段影片步驟</a>輸入 tokenId 為 549，會得到回傳值為：<a href="https://ipfs.io/ipfs/QmRAjjAcxWMaG4GUKRk5B4YiMD1doigKCaZX875duPFvXD/549" rel="noopener noreferrer" target="_blank"><br class="smart">ipfs://QmRAjjAcxWMaG4GUKRk5B4YiMD1doigKCaZX875duPFvXD/549</a></p><p>而這串 JSON 就是這個 NFT 的本體了，裡面指定了圖片位址是在：<a href="https://ipfs.io/ipfs/QmXEYf3mGnoZCfPPKt6UF9nfZuviGPYw4uBLdSzH28g145" rel="noopener noreferrer" target="_blank">ipfs://QmXEYf3mGnoZCfPPKt6UF9nfZuviGPYw4uBLdSzH28g14</a></p><p>換句話說，如果有一天 IPFS 倒了，這個 NFT 也就失去內容物，成為虛無。請注意，這其實是常態的實作方式，Bored Ape Yacht Club (BAYC) 專案團隊也是這樣實作的！</p><h2>NFT 還能做什麼？</h2><p>此外，NFT 還能透過智慧合約來實作附加功能，將較低成本的「文字」真正寫上區塊鏈。由於 Travelogger <a href="https://matters.news/@hi176/202714" rel="noopener noreferrer" target="_blank">Logbook 1.0</a> 和 <a href="https://mintverse.world/" rel="noopener noreferrer" target="_blank">Mintverse</a> 都架設在被戲稱為「貴族鏈」的以太坊上，撰寫上鏈的 gas fee 價格依然昂貴（以下美元皆為點擊後顯示的交易當日預估值）：</p><ol><li>Matters 站長在 Logbook 1.0 寫下了<a href="https://traveloggers.matters.news/zh/logbooks/40" rel="noopener noreferrer" target="_blank">這封信</a>時，時值 ETH 幣價與 gas fee 高點，交易費用高達 <a href="https://etherscan.io/tx/0xa3e74b17f5d3cf9045f064de9bc8282b385a9942edd9480121978e0e2ea50688" rel="noopener noreferrer" target="_blank">US$208.80</a>。</li><li>我前幾天趁幣價與 gas fee 正低時，終於在 Logbook 1.0 寫下這段<a href="https://traveloggers.matters.news/zh/logbooks/549" rel="noopener noreferrer" target="_blank">宇宙訊號</a>，也還是得花費 <a href="https://etherscan.io/tx/0x9f2eae9d865f6a4d1dbd29c61d8149931d83ab9477eb7510a17b3066782d5147" rel="noopener noreferrer" target="_blank">US$7.50</a>。</li><li>我參與了 Matters Lab Discord 活動獲得 Mintverse 白單，因而 "free" mint（得先花一波 <a href="https://etherscan.io/tx/0xed98cb7660f4f2ffbebbef94da60ee7e7b47c43695466ea601bd0fc79cea3bfb" rel="noopener noreferrer" target="_blank">US$14.36</a>）取得了一款詞彙 NFT，再寫入註釋定義的費用則為 <a href="https://etherscan.io/tx/0x63c177eeccb37808d3edf82b01ee552651cfcda76646a706a66452de2bfd229d" rel="noopener noreferrer" target="_blank">US$6.58</a>，成果則在 OpenSea「<a href="https://opensea.io/assets/0x895e34343e2cdaa58bd393ec416b446ba4781c1c/672/" rel="noopener noreferrer" target="_blank">平台</a>」。</li><li>至於 Matters 新近推出的 <a href="https://matters.news/@hi176/258550" rel="noopener noreferrer" target="_blank">Logbook 2.0</a>，由於架設在 Polygon 區塊鏈上，交易費用十分低廉，我先前將「<a href="https://matters.news/@denkeni/275256" rel="noopener noreferrer" target="_blank">《後綴》採訪寫作營｜如何用 Email 來採訪</a>」整篇文章放上<a href="https://logbook.matters.news/logbook/?id=367" rel="noopener noreferrer" target="_blank">《後綴》Logbook 2.0</a>，只花了 <a href="https://polygonscan.com/tx/0xb7a0e5ac68f07b75d1ac7c7db80d325f6c67698a73920c61549a1884a81be86d" rel="noopener noreferrer" target="_blank">US$0.01</a>！</li></ol><p>在上述的交易費用連結中，點擊「Click to see More ↓」，然後在 Input Data 欄位中，選擇「Decode Input Data」即可看到 Logbook 內容，確實是上鏈了。</p><p>當我們談論 Web3 時，經常說是賦予用戶「數位所有權」（Ownership）。前述上鏈的 Logbook 內容，是能證實其寫入者的錢包地址；而任一個錢包地址所持有的 NFT，也同樣被記錄在區塊鏈上。</p><h2>持有 NFT 到底持有了什麼？</h2><p>持有 NFT 經常可以進入特定社群，這被稱為「社群型 NFT」。不過大家都可以自由進來 Matters Lab Discord 的哦，在「🌠︱nft-討論區」近期的問答活動，「收藏、經濟、使用」三種價值中，意外發現大家最重視的幾乎都是「收藏價值」！在這個討論頻道中，即使只是旁觀頻道駕駛員與大家的討論，就讓我更熟悉了當前交易量熱門的 NFT，因而我一直以為大家最重視的是「經濟價值」。基於這樣的問卷結果，我期待未來能有更多「收藏價值」的分享與討論，在這社群頻道爆炸多的年代，仍能形塑出屬於我們這個社群專有的屬性。</p><p>為何大家最重視「收藏價值」呢？我想除了近期幣圈經濟進入低迷的熊市，難以套利之外，可能還因為持有圖片 NFT「並不會」獲得其著作權，實質上預設僅有「<a href="https://youtu.be/L_YL-LYfN7g?t=5m49s" rel="noopener noreferrer" target="_blank">冠名權</a>」，證明你是買家，而這正是收藏家的意義。</p><p>至於 BAYC 則是<a href="https://blockcast.it/2022/05/05/nft-and-copyright-violations-explained/" rel="noopener noreferrer" target="_blank">額外授權了圖像使用權</a>，但著作財產權仍歸專案團隊所有。</p><p>那麼在 NFT 上寫入（其實是寫在 NFT 智慧合約的區塊鏈上）的文字著作呢？理論上就預設歸為「版權所有©️All right reserved」，但像 <a href="https://mintverse.world/law/" rel="noopener noreferrer" target="_blank">Mintverse 法律聲明</a>就特別寫了：</p><blockquote>您同意依<strong>公眾領域授權</strong>條款授權您於辭彙NFT撰寫之文字著作</blockquote><p>注意到了嗎？公眾領域授權！意即任何人都可以使用你在辭典 NFT 上寫入的文案！</p><p>因而可能延伸的問題是：如果寫入內容是從其他版權作品挪用過來，那麼除了侵權疑慮，還無權將其內容「再授權」為公眾領域授權。</p><p>譬如，<a href="https://discord.com/channels/885797384196587560/972726176718159902/973102157584822272" rel="noopener noreferrer" target="_blank">志祺 YouTuber 所定義</a>的「<a href="https://opensea.io/assets/0x895e34343e2cdaa58bd393ec416b446ba4781c1c/310" rel="noopener noreferrer" target="_blank">自由</a>」，署名為「艾連．葉卡」，是否就有侵權疑慮呢？或是可以視為「二次創作」呢？更複雜的是，<a href="https://etherscan.io/tx/0xa1dd8fd1f763d52edc13aaa1723c7640e2fc782ace6f13296559cce04d6ff77e" rel="noopener noreferrer" target="_blank">區塊鏈所記錄的寫入者</a>，並非目前的持有者，而看起來更像是專案團隊裡的代筆者。</p><p>當然，只要持有者未將其轉賣獲利，著作人要跨海來提告的可能性與必要性皆很低。</p><h2>所以到底什麼是 NFT？</h2><p>哈囉你好，這是<a href="https://opensea.io/0x255f7D91B2ddDAbBe044C64840460A1B7285ad5F" rel="noopener noreferrer" target="_blank">我的 NFT</a>，這是<a href="https://www.pinterest.com/denkeni/jpg/" rel="noopener noreferrer" target="_blank">我的 JPG</a>。</p><p>先談什麼是區塊鏈應用吧。區塊鏈就像是共同筆記本，在上面記錄某人擁有一塊田，那它就成資產記錄表；記錄某人持有1枚以太幣，那它就成為密碼學貨幣（cryptocurrency, 簡稱為 <a href="https://dictionary.cambridge.org/us/dictionary/english/crypto" rel="noopener noreferrer" target="_blank">crypto</a>）記錄表；作為繪圖紙，那它就可能成為一幅藝術鉅作；記錄我愛你，那它就成為我的情書。</p><p>那麼你知道 NFT 的主管機關，應該歸誰了嗎？</p><h2>後記</h2><p>這是一篇原本預定為圍爐限定的上鎖文章，因為結合 <a
