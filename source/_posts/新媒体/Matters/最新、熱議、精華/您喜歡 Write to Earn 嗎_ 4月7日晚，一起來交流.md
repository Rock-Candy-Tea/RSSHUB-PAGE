<p>我的口才極差，不太喜歡與「人」於現實世界交流。文字傾談還可以，面對面的話，總是在一場聚會的死角人物。不是因為怕說話尷尬，而是對談的時候反應很慢，往往要想一些時間才「對得上咀」，即是不能流暢地與人對談。</p><p>若傾談內容在熟悉的範圍還比較好，但在不熟悉的領域，來來去去只懂說那幾組字:「很搞笑、好得意、嘻嘻嘻......」</p><p>我的文筆也不好，但是我卻喜歡活在文字交流的世界。當中最主要原因，就是可以整句話「<strong>完整輸入，才Enter送出」</strong>。我回想過，導致自己有這樣的原因，應該是「很怕自己說錯話」，又或者「詞不達意」，更怕「別人感到煩厭」，故此，慢慢地就不想與人交流。</p><p>從中學開始，我就喜歡活在數碼世界，<strong>我的手代表了我的口；我的眼也代表了我的耳</strong>。這境況，不知不覺已三十多年。文字的表達及交流令我有一份安全及舒適感，我十多年前開始建個人Blog，其後也設立了facebook粉絲專頁，我很響往每時每刻都有網友互動的通知聲音。</p><p>2015年，香港在佔中運動後，社交媒體出現翻天覆地的改變。人與人之間不再信任，人人也成了「鍵盤戰士」，留言間都是憎恨及對罵。</p><p>那時，我們夫婦覺得<strong>「不如離開社交媒體」</strong>，最終在2017年時，便開始減少更新；到2018-19年，更接近沒有更新。45萬的fb粉絲專頁，尤得她放涼。</p><p>習慣了用「文字說話」，突然「無話可說」，內心總會空虛。當時看到一位喜歡的作家 <a class="mention" href="https://matters.news/@tenz1225" target="_blank" data-display-name="施典志" data-user-name="tenz1225" data-id="VXNlcjoxNDA">﻿<span>@施典志</span>﻿</a> 於 Medium 寫文章，故此，我也好奇一試。<strong>或許， fb 粉絲專頁其實是情緒表達，而 Medium 卻是內容為王，總是覺得格格不入</strong>。</p><p>有一天看到一位 Medium 博主分享 「Steemit 區塊鏈內容平台」，而剛好我們打算買入一些比特幣，因此，就被吸引進去。不經不覺中，在踏進幣圈四年多時間，接觸了十多二十個區塊鏈內容平台，每一個都是本著 Write to Earn 的旗號，透過Token獎勵「創作人」。<strong>但是，仍留下來的平台卻不多。</strong></p><p>若以金錢計算，我所獲的 Token 實在不少，Write to Earn 是真的存在，<strong>不過，卻未必如大家所願「靠寫作為生」模式</strong>。此時此刻我應該<strong>算是</strong>過著「靠寫作為生」的生活，只不過，不是靠著以每天寫作獲Token模式，而是靠著運用過往於寫作累積的Token，使到現時我可以「靠於區塊鏈內容平台碎碎念為樂」。</p><p>有一天與 <a class="mention" href="https://matters.news/@denkeni" target="_blank" data-display-name="花椰菜 Denken" data-user-name="denkeni" data-id="VXNlcjo0NTg">﻿<span>@花椰菜 Denken</span>﻿</a> 傾談關於 Web 3 的不同可能性，特別在 Social Fi 及 內容平台上。我相信， Social Fi 不只是「寫文、點贊、收Token」這麼簡單。試一試用 AMA 形式一起思考 及 投放期望，甚至慢慢聯合 Matters 用戶也一起來發夢。</p><p>上週，突然收到 Matters 同事多比的聯絡，原來這一個 AMA 已經成事，第一場甚至也邀請了筆大 <a class="mention" href="https://matters.news/@penfarming" target="_blank" data-display-name="閱讀筆耕" data-user-name="penfarming" data-id="VXNlcjo0NTE5NA">﻿<span>@閱讀筆耕</span>﻿</a> 一起來發夢。 能與兩位一起去「尋夢」，實在相當榮幸興奮。</p><p>我知道 Web 3 很虛蕪，更知道現有的 Social Fi 或內容平台有不同的缺點，比較各平台也是無謂。但是，我們可以在 Killer 項目出現之前，嘗試「預備自己成為 Web 3 網民」，而不是「帶著 Web 1 或 Web 2 包袱進入 Web 3 世界的人」。</p><p><strong>今個星期四晚 (4月7日) 晚上 9時，榮幸與 <a class="mention" href="https://matters.news/@penfarming" target="_blank" data-display-name="閱讀筆耕" data-user-name="penfarming" data-id="VXNlcjo0NTE5NA">﻿<span>@閱讀筆耕</span>﻿</a> 一起在 Matters 的 Discord 頻道，邀請朋友們也一起交流暢談。</strong></p><p><strong>Matters 的 Discord: </strong><a href="https://discord.com/invite/vn6fUqSN?event=960370269963292703" rel="noopener noreferrer" target="_blank">AMA：Web3 實驗者系列
