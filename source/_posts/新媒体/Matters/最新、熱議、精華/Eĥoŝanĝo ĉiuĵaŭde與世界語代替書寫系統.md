<p>前段時候偶然看到一個怪異的世界語短語：Eĥoŝanĝo ĉiuĵaŭde，由eĥo（迴聲） + ŝanĝo（改變）和ĉiu（每個）+ ĵaŭde（在周四）組成，直譯過來就是：“每週四迴聲改變”。</p><figure class="image"><img src="https://assets.matters.news/embed/2cc30b26-350f-4a09-af9f-890e7cb574db.jpeg" data-asset-id="2cc30b26-350f-4a09-af9f-890e7cb574db" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>這個短語的發明者是德國世界語者Martin WEICHERT，他是漫畫《Gogo la testudo》的創作者和虛擬世界語圖書館的發起人。</p><figure class="image"><img src="https://assets.matters.news/embed/a5e25aa5-545c-45ea-8a9d-6f7bc2a482e9.jpeg" data-asset-id="a5e25aa5-545c-45ea-8a9d-6f7bc2a482e9" referrerpolicy="no-referrer"><figcaption><span>Martin WEICHERT</span></figcaption></figure><p>Eĥoŝanĝo ĉiuĵaŭde實際上毫無意義，唯一的作用是測試世界語字母在某個系統環境下能否被正確地顯示，因為這個詞組包含了所有世界語的加符字母（戴帽字母）。</p><p>這裡先解釋一下，世界語一共有28個字母，本著一字一音的原則，不使用二合字母（比如/tʃ/就是ĉ而不是ch）或者變讀（比如/dʒ/就是ĝ而不是g在i、e前變讀），但也不使用q、x、y、w四個字母。</p><p>於是字母表中就多了ĉ、ĝ、ĥ、ĵ、ŝ、ŭ，而Eĥoŝanĝo ĉiuĵaŭde正好包含了這6個字母。</p><figure class="image"><img src="https://assets.matters.news/embed/1cbef447-43cd-4e34-bd42-d2774d3dc7a1.jpeg" data-asset-id="1cbef447-43cd-4e34-bd42-d2774d3dc7a1" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>除了ŭ之外的幾個字母，都是世界語獨有的，在過去需要專屬的字模，現在則是缺少輸入法。因此為這6個字母尋找替代方案，從很早之前就開始了。</p><p>下面就舉幾個典型的替代方案：</p><h2>（1）H系統（H-sistemo）</h2><ul><li>ch = ĉ (chemizo = ĉemizo)</li><li>gh = ĝ (ghi = ĝi)</li><li>hh = ĥ (monahhejo = monaĥejo)</li><li>jh = ĵ (ajho = aĵo)</li><li>sh = ŝ (shipo = ŝipo)</li><li>u = ŭ (au = aŭ)</li></ul><p>如果用這個系統書寫Eĥoŝanĝo ĉiuĵaŭde，就會變成Ehhoshangho chiujhaude。</p><figure class="image"><img src="https://assets.matters.news/embed/6efa8e16-f804-4693-9efb-4c5ec9261e23.jpeg" data-asset-id="6efa8e16-f804-4693-9efb-4c5ec9261e23" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>這個方案是世界語創始人柴門霍夫在《第二書》（Dua Libro）的附錄中首次提出的，這套系統也被世界語學院定為“基本代替書寫系統”。</p><figure class="image"><img src="https://assets.matters.news/embed/bfd54f3e-c190-47f0-a567-89155b7f8338.png" data-asset-id="bfd54f3e-c190-47f0-a567-89155b7f8338" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>雖然這個系統是柴門霍夫本人提出的，但還是有一些批評意見，主要認為h本身就是一個世界語字母，這樣的轉寫與“一字一音”的原則相違背，而且使得u有/u/和/w/兩個發音。</p><p>而且這個系統容易造成混淆，比如flughaveno,、chashundo等詞就容易誤讀成fluĝaveno、chaŝundo，而traŭmata如果寫成traumata，又容易搞錯重音。 Zamenhof建議在這種情況下插入撇號以消除歧義，比如flug'haveno、chas'hundo。</p><figure class="image"><img src="https://assets.matters.news/embed/62423936-f736-4017-8151-bdcfe9d4c745.jpeg" data-asset-id="62423936-f736-4017-8151-bdcfe9d4c745" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>（2）X系統（X-sistemo）</h2><ul><li>cx = ĉ (cxemizo = ĉemizo)</li><li>gx = ĝ (gxi = ĝi)</li><li>hx = ĥ (monahxejo = monaĥejo)</li><li>jx = ĵ (ajxo = aĵo)</li><li>sx = ŝ  (sxipo = ŝipo)</li><li>ux = ŭ (aux = aŭ)</li></ul><p>如果用這個系統書寫Eĥoŝanĝo ĉiuĵaŭde，就會變成Ehxosxangxo cxiujxauxde。</p><figure class="image"><img src="https://assets.matters.news/embed/edacecab-47db-4c88-a1d7-607350933b81.jpeg" data-asset-id="edacecab-47db-4c88-a1d7-607350933b81" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>這是現在最為流行的替代書寫系統，因為x不是世界語字母，所以不會發生誤讀，而且很容易輸入。</p><figure class="image"><img src="https://assets.matters.news/embed/124ad4f6-6fae-448e-b080-269193cf0b8c.png" data-asset-id="124ad4f6-6fae-448e-b080-269193cf0b8c" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>當然批評聲音也不少，比如有人認為這個形式不夠國際化，給世界語增加了陌生感；有人認為一些非世界語的名詞中包含x，容易引發混淆，比如法語的Michaux，容易誤解為Michaŭ；</p><p>還有人認為它看起來很醜，比如ĝi ŝanĝiĝas（它正在改變），就會被寫成：gxi sxangxigxas，這就相當令人抓狂了。</p><h2>（3）QWXY系統（QWXY-sistemo）</h2><ul><li>tx = ĉ (txemizo = ĉemizo)</li><li>dy = ĝ (dyi = ĝi)</li><li>q = ĥ (monaqejo = monaĥejo)</li><li>y = ĵ (ayo = aĵo)</li><li>x = ŝ (xipo = ŝipo)</li><li>w = ŭ (aw = aŭ)</li></ul><p>這個系統源自美國，採用4個英文字母（q, w, x, y）和兩個二合字母（dy, tx）。如果用這個系統書寫Eĥoŝanĝo ĉiuĵaŭde，就會變成Eqoxandyo txiuyawde。</p><figure class="image"><img src="https://assets.matters.news/embed/d0c5f2e6-3e55-4574-96cf-d143486d1204.jpeg" data-asset-id="d0c5f2e6-3e55-4574-96cf-d143486d1204" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>（4）新輔助字母（Nova Helpalfabeto）</h2><ul><li>ch = ĉ  (cxemizo = ĉemizo)</li><li>j = ĝ  (gxi = ĝi)</li><li>kh = ĥ (monahxejo = monaĥejo)</li><li>zh = ĵ (ajxo = aĵo)</li><li>sh = ŝ (sxipo = ŝipo)</li><li>j = y (homoj = homoy)</li><li>ŭa → wa, ŭe → we, ŭo → wo</li><li>aŭ → au, eŭ → eu, oŭ → ou</li></ul><p>如果用這個系統書寫Eĥoŝanĝo ĉiuĵaŭde，就會變成Ekhoshanjo chiuyaude。</p><figure class="image"><img src="https://assets.matters.news/embed/b43a98fc-c80c-4da3-b53d-2cbe0b9b4fb0.jpeg" data-asset-id="b43a98fc-c80c-4da3-b53d-2cbe0b9b4fb0" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>除了以上四種，世界語還有w-sistemo（僅用w代替ŭ）、Internacia、Sefosas等系統，以及西里爾字母等非拉丁字母的轉寫形式，之後有機會可以單開一篇再講。</p><p>對了，波蘭語中也有一個能打盡所有加符字母的詞：Zażółć gęślą jaźń。</p><p>Reference：</p><ul><li><a href="https://en.wiktionary.org/wiki/e%C4%A5o%C5%9Dan%C4%9Do_%C4%89iu%C4%B5a%C5%ADde" rel="noopener noreferrer" target="_blank">eĥoŝanĝo ĉiuĵaŭde -
