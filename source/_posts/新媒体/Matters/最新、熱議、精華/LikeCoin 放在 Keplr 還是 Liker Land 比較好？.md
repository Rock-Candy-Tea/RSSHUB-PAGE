<p>幾日前和 <a class="mention" href="https://matters.news/@misiaa2001" target="_blank" data-display-name="蔡凱西" data-user-name="misiaa2001" data-id="VXNlcjoxMjEw">﻿<span>@蔡凱西</span>﻿</a> 談到 LikeCoin 放在那個錢包的問題：究竟我應該把 LikeCoin 放進 Keplr 還是 Liker Land 做主力？</p><blockquote><strong>首先還是那一句：1. 在 Matters 直接註冊 Matters ID 所一拼生成的 Liker ID，或者 2. 在 Liker Land 上註冊並經已綁定了 Matters ID 的 Liker ID，兩者暫時都完全無法與 Keplr 錢包互聯互通。因為 1. 和 2. 與 Matters 註冊關聯的方式叫做 Authcore 登入，而 Matters 現時只支援 Authcore 登入，沒有說要支援 Keplr。</strong></blockquote><blockquote><strong>所以不要問為甚麼你在 Matters 的 Liker ID 不能登入 Keplr，又或者為甚麼你的 Keplr 不能登入 Matters，因為他們根本不能發生關係。</strong></blockquote><blockquote><strong>我當然希望有一天 Keplr 和現有使用 Authcore 登入的 Liker ID 能夠互相接通，但這一日未實現之前，只可以分開 Liker Land 與 Keplr 兩個錢包來使用。<u>而一個電郵地址只可以對應一個 Liker ID，並沒有辦法使用同一個電郵地址同時以 Authcore 和 Keplr 註冊 Liker ID</u>，這一點請務必留意。</strong></blockquote><figure class="image"><img src="https://assets.matters.news/embed/49f5b134-c53c-43f0-8d8d-7fd7b901fd8a.png" data-asset-id="49f5b134-c53c-43f0-8d8d-7fd7b901fd8a" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>以下除非特別說明是 app 或者手機版，否則全部都只是在說桌面電腦的事。</p><h2>Keplr 存錢、Authcore 賺錢</h2><p>我的選擇是把 LikeCoin 從使用 Authcore 登入的 Liker Land app 錢包轉存到 Kelpr 錢包，然後在那邊做委託或者傳輸款項到交易所之類。這樣既可以繼續使用 app 查看 LikeCoin 讚賞成就，又可以兼顧安全。</p><p>為甚麼呢？一方面，不想給人看光光內褲（<a href="https://matters.news/@nk980710/%E7%99%BD%E9%BC%AC%E5%B0%8F%E7%99%BD%E6%95%99%E5%AD%B8%E6%96%87-%E6%95%99%E4%BD%A0%E5%A6%82%E4%BD%95%E6%9F%A5%E8%A9%A2likecoin%E7%9A%84%E4%BA%A4%E6%98%93%E5%8B%95%E5%90%91-bafyreiburq7yxaj4o5zjpqs4aqlfoizaubmlymerskknudzwxvt7xb77xy" target="_blank">白鼬筆寬麵</a> 和 <a href="https://matters.news/@erica072/%E5%A6%82%E4%BD%95%E6%9F%A5%E8%A9%A2%E9%8C%A2%E5%8C%85%E6%94%B6%E4%BB%98%E6%AC%BE%E8%A8%98%E9%8C%84-%E6%88%91%E6%80%8E%E9%BA%BC%E7%A2%BA%E5%AE%9A%E6%88%91%E5%8C%AF%E5%87%BA%E7%9A%84-like-coin-%E5%B0%8D%E6%96%B9%E6%94%B6%E5%88%B0%E4%BA%86%E6%B2%92-bafyreidtfw54rizinxpj5w4qthpixfypkp4yys42mzib2v3sazkvq33uyy" target="_blank">Erica</a> 都有教怎樣查錢包），另一方面，Keplr 能掌握自己的私鑰實在比較安全（看<a href="https://matters.news/@thumbb13555/%E7%A2%BC%E8%BE%B2%E6%97%A5%E5%B8%B8-%E7%B7%9A%E4%B8%8A%E5%B7%A5%E4%BD%9C%E5%9D%8A-%E6%88%91%E6%9C%89liker-land%E4%BA%86-%E7%82%BA%E4%BD%95%E6%88%91%E9%82%84%E9%9C%80%E8%A6%81%E4%B8%80%E5%80%8Bkeplr%E9%8C%A2%E5%8C%85-bafyreibce6qf2lcgeipdetoc5ferugp4qfbl2jsq7kgobpuqbsybyer47y" target="_blank">碼農的文章</a>或者再次看看他何時舉辦課程吧） ，再加上長遠持幣其實需要買一個硬件錢包，將 LikeCoin 放在支援硬件錢包的 Keplr 真的會比較好。最後就是 Erica 寫過：<a href="https://matters.news/@erica072/%E8%B2%93%E7%89%B9%E5%8B%99%E5%A0%B1%E5%A0%B1-%E9%80%B1%E5%A0%B1-and-%E6%9C%89%E4%BA%86-like-land-%E9%8C%A2%E5%8C%85-%E7%82%BA%E4%BD%95%E9%82%84%E8%A6%81%E7%94%A8-keplr-2021-10-03-2021-10-09-bafyreic6cdwhxrkzezvmc3ygtrf7m72rzk7kzwz4mjahuaixj5j7erynim" target="_blank">Keplr 的手續費比 Liker Land 低啦</a>！</p><p>唔，也不是說 <a href="https://docs.like.co/v/zh/user-guide/liker-id/what-is-authcore" target="_blank">Authcore</a> 登入 Liker Land 不安全啦，它的系統通過<a href="https://www.blocktempo.com/zero-knowledge-proof-zkp-chagelly-column/" target="_blank">零知識證明</a>其實是很安全的。只是 Authcore 能以社交媒體帳號如 Facebook、Twitter 和 Apple 登入。假設有一天壞人打暈了你，用你的手指解鎖電話，然後因為你登入了 Google 甚麼的他就可以把你的錢轉走光光了。然而使用 Keplr，只要你把的 Mnemonic Seed 密鑰收藏好，而你又打死都不提供在腦中的密碼，就沒有人能把你的錢轉走。</p><figure class="image"><img src="https://assets.matters.news/embed/002613ba-b22d-434e-a559-cc2f5ddb250b.png" data-asset-id="002613ba-b22d-434e-a559-cc2f5ddb250b" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>使用 Keplr 文章中的雞蛋情節做舉例：</p><p>密碼貨幣資產是雞蛋，Mnemonic Seed 密鑰是一個綁了很複雜的結的袋子包著你的雞蛋並只有你懂得解開。你可以把這包雞蛋裝在 Keplr 錢包這個圓型有蓋並上了密碼鎖的盒子內保護，盒子的特別之處是只要你一手一腳做好設定，以 Keplr 密碼打開盒子就可一拼打開 Mnemonic Seed 取出雞蛋食用。</p><p>然而 Authcore 登入，就是你的資產即雞蛋依然被 Mnemonic Seed 密鑰這個綁了很複雜的結的袋子包著，可是設定 Liker Land 錢包這個盒子的動作你不自己做，而是假手於人（電郵密碼、社群登入）去幫你設定簡單方式讓你比較容易去打開盒子而不需要做複雜設定。然而假手於人的結果換來有被操控的機會，雖然幫助你設定的人有很強勁的保安知識和經驗，只是「意外」就是「意料之外」，能自己操控就不要給別人去弄是最好的。</p><h2>為甚麼在 Keplr 看不見 LikeCoin chain？看不見我的 LikeCoin？其實你可以註冊 Keplr 錢包但不申請 Liker ID</h2><p>註冊 Keplr 錢包後，你完全可以獨立使用它管理 LikeCoin，並與 Liker Land、Liker ID 完全無關。然而如果你不把 Keplr 與 LikeCoin chain 區塊鏈做連結的話，你是完全不會在 Keplr 錢包看到你的 LikeCoin，所以你需要先到 <a href="https://stake.like.co/" target="_blank">stake.like.co</a> 過一過水才可以看到 LikeCoin chain。</p><p>點 Keplr Browser Extension</p><figure class="image"><img src="https://assets.matters.news/embed/c4ecb8a6-0ca2-476f-8295-b203c98d59f6.png" data-asset-id="c4ecb8a6-0ca2-476f-8295-b203c98d59f6" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>然後在 Requesting Connection 點「Approve」</p><figure class="image"><img src="https://assets.matters.news/embed/f4eb8be3-7b15-4d88-b93d-b030d5d659b9.png" data-asset-id="f4eb8be3-7b15-4d88-b93d-b030d5d659b9" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>才可以在 Keplr 的下拉選單看到 LikeCoin chain 和你的 LikeCoin</p><figure class="image"><img src="https://assets.matters.news/embed/4cfc77ce-8ab7-479a-abf4-a21355aa5061.png" data-asset-id="4cfc77ce-8ab7-479a-abf4-a21355aa5061" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><h2>stake.like.co 是做甚麼的</h2><p><a href="https://stake.like.co/" target="_blank">stake.like.co</a> 是一個很神奇的網站，你可以：</p><ul><li>在左手邊菜單選 Portfolio 查看經已委託給驗證人的資產，及領取委託回報</li><li>使用紅色的 Send 傳送 LikeCoin 給別的錢包</li><li>在左手邊菜單選 Validators 查看任何一個驗證人並進行 Stake 委託、Unstake 取回委託及 Restake 轉移委託</li><li>在左手邊菜單選 Proposals，查看讚賞公民共和國有甚麼議案，並為它<a href="https://matters.news/@daisy/like-coin-%E5%8F%B2%E4%B8%8A%E6%9C%80%E5%BF%AB%E9%80%9F%E5%8B%95%E8%AD%B0-%E7%AC%AC-17-%E8%AD%B0%E6%A1%88%E5%95%9F%E5%8B%95-ibc-liker-%E4%B9%9F%E5%8F%AF%E5%AD%98%E5%85%A5%E4%BF%9D%E8%AD%89%E9%87%91-bafyreibekvznh3gmnfy6shm3wg2iomuwtardtn542qwcjc37fttgrypfjm" target="_blank">存入保證金</a>或<a href="https://matters.news/@daisy/%E7%AC%AC-17-%E8%AD%B0%E6%A1%88%E5%95%9F%E5%8B%95-ibc-%E6%8A%95%E7%A5%A8%E9%96%8B%E5%A7%8B-%E5%BC%84-iscn-%E4%B9%9F%E5%8F%AF%E6%8B%BF-like-coin-grants-%E6%98%AF%E7%94%9A%E9%BA%BC%E7%8E%A9%E6%84%8F-bafyreic6hz4z3jxkpflsoxdm63dsvb6zrmldjautbt4jkd2alu3a2j5vye" target="_blank">投票</a></li><li>在左手邊菜單選 Transactions 查看交易細節</li></ul><p>因為這個網站真的超簡單，故此就不截圖介紹。</p><p>而 stake.like.co 最特別的地方，是 Keplr 登入和 Authcore 登入都可以使用同樣功能，因為這個網站看的是你的持份，亦即是你操控了多少 LikeCoin，而不是看你的 Liker ID 姓甚名誰。於是只要你點 Keplr Browser Extension 進入，又或點 Autocore 進入，就可以使用它的介面操控以上所述功能。</p><p>故此 Authcore 登入用戶亦即是在 Matters 或 Liker Land 註冊 Liker ID 的人士，如果你不想用 Liker Land app 而想用桌面電腦，其實是可以用這個的。</p><h2>以 Keplr 註冊 Liker ID 的好處</h2><p>確實可以不把 Keplr 與 Liker ID 或 Liker Land 做任何關聯。但是關聯了也不是沒有好處。例如使用 Liker Land app 就可以直接把 LikeCoin 以 Liker ID 傳送到 Keplr，而不需要輸入錢包地址或者掃 QRCode 之類。而 Liker Land 大百科並詳盡列明 <a href="https://docs.like.co/v/zh/user-guide/liker-id/register-with-keplr" target="_blank">Authcore 與 Keplr 註冊的分別</a>，不妨參考相關介紹及教學。</p><h2>電郵地址要互相對應</h2><p>因為做了「Keplr 存錢、Authcore 賺錢」這個決定。我需要把常用的電郵地址做一個調動。原因是我的 Liker Land 和 Keplr 都有註冊 Liker ID 而因為同一個電郵地址不能重覆使用作註冊。而我忘記了留意 Matters 的電郵址和 Liker ID 的電郵地址是可以完全不同的。我改了我的策略後電郵地址一時之間沒有對應起來，那些交易互相就對不上了。</p><figure class="image"><img src="https://assets.matters.news/embed/ceae10d9-68fc-4dd9-a074-cab07d2e01ee.png" data-asset-id="ceae10d9-68fc-4dd9-a074-cab07d2e01ee" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>如果你有兩個 Liker ID 的話，記得修改你的 Matters ID 和 Liker ID 的電郵地址都是同樣，就不會發生看到電郵一頭霧水的問題啦。</p><h2>各種區塊鏈之間的連結</h2><p>LikeCoin 在 Osmosis 上線後，大家對它趨之若騖，但概念上是要搞清楚的，所以就我的理解，不厭其煩地說一下，如有錯漏，請高人指正。</p><ul><li>LikeCoin 的原生區塊鏈是 LikeCoin chain，貨幣代號是 $LIKE</li><li>Osmois 的原生區塊鏈是 Osmosis blockchain，貨幣代號是$OSMO</li><li>Cosmos 的原生區塊鏈是 Cosmos Hub，貨幣代號是 #ATOM。Cosmos Hub 連結不同的區塊鏈而 LikeCoin 和 Osmosis 只是其中之兩條</li><li><a href="https://matters.news/@daisy/like-coin-%E6%B5%81%E5%8B%95%E6%80%A7%E6%8C%96%E7%A4%A6-%E8%AA%8D%E8%AD%98-emeris-bafyreigjwiisbjhhhncmkjpluaapz3dmul2s7padz5u7d2mwq7isppxamq" target="_blank">Emeris</a> 是 Cross-Chain DEX aggregator 跨鏈 DEX 聚合器，即是不理你在甚麼鏈，它都可以來「鏈一鏈」這樣互相搞作。而 Emeris 現時沒有自己發幣。</li></ul><p>LikeCoin、Osmosis、Cosmos 中間互相傳遞要靠 IBC Inter‑Blockchain Communication 跨鏈通訊。</p><p>所以就算 $LIKE / $OSMO / $ATOM 同樣都是放在 Keplr 錢包，它們是不可以就這樣一下子互相傳輸。</p><p>在 Osmosis，你需要在 Assets 資產進行 Deposit 存入 #LIKE 才可以在 Osmosis 上交易 #LIKE，中間其實經已做了 IBC 這個行為。</p><p>在 Emeris 你需要使用 Move assets 轉移資修去 Emeris 這個場才能玩，就算資產都是在你的 Keplr 錢包內，其實也是在做 IBC 這個行為。</p><p>雖然 $LIKE 都是 Cosmos Hub 家族中的一員，但現時 Osmosis 手機應用程式、Cosmostation 手機應用程式或網頁版，及 Keplr 本身的網頁版治理平台 <a href="https://wallet.keplr.app/" target="_blank">https://wallet.keplr.app/</a> 都未有支援 $LIKE，所以大家並沒有辦法在這個三個地方執行任何關於 LikeCoin 的轉帳、委託和投票。</p>
