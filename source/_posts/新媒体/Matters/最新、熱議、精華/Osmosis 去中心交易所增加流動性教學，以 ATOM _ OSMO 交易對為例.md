<h2>何謂增加流動性</h2><p>首先簡單說一下增加流動性〔流動性挖礦 Liquidity Mining〕是什麼意思，簡單來說就是兩個幣在進行交換時，需要有一個資金池來應付立即的匯兌，所以可以不需要像在交易所中還要進行買賣雙方的搓合，而在區塊鏈上，人人都可以將閒置的資金投入這個<strong>池</strong>裡面，當然在有人進行匯兌的時候也會付出一些費用，當作為這個池增加流動性的人的報酬，根據不同的幣，有些年化報酬可以超過 100%，甚至更多。</p><h2>有什麼風險嗎？</h2><p>1、合約風險：就是指萬一代碼中有 Bug 的意思。</p><p>2、跑路風險：有些交易所並非開源或並非真正的去中心化，所以你投入的幣，有被捲走的風險，必需要慎選投入項目。</p><p>3、無常損失〔暫時性損失〕：來自押入的幣中，其中一個幣突然暴漲 or 暴跌，這時候就會有投機者從中套利，進而讓資金池面臨損失，關於這點有人認為因為報酬高，通常可以抵過損失，所以可以略過不計，有的人提出找兩個幣走勢相同的風險較低。</p><p>總之，投資都有損失的風險，請不要把你明天就要吃飯的錢拿來做增加流動性，通常取回時會有等待期間，個交易所或平臺會不一樣，有的解綁期很短有的比較長，也有可能在解綁期正好碰到幣價下跌，所以要自己檢視自己的風險耐受度，做好事前的心理建設，還要對投資的項目有基本的了解才行喔！</p><hr><h2>前置作業</h2><p>各平臺增加流動性的方式大同小異，這邊用 Osmosis 的去中心化交易所為例，因為這條鏈在 Keplr 錢包上也能看到，與 LikeCoin 同屬於 Cosmos 生態系的其中一鏈，而對比另外兩個去中心化交易所目前還是 Beta 版，Osmosis 算是相對成熟，介面也非常直覺。</p><p>那麼我就在 Osmosis 操作一下，為 ATOM / OSMO 這個交易對增加流動性，來體驗一下吧！</p><p>首先需要一個 Keplr 錢包，然後錢包裡需要有 ATOM 或 OSMO 或其他可以在生態系中進行 IBC Transfer 的加密貨幣，例如：AKT、ROWAN ...，<strong>等到 LikeCoin 加入 IBC 之後，也可以使用 LikeCoin 在 Osmosis 換成所需的幣，甚至選擇 LikeCoin 的交易對操作增加流動性</strong>。</p><p>我這次是先在交易所換了一點點 <strong>ATOM</strong>，再提幣進Keplr 錢包，這次是使用幣安，因為的 ATOM 提幣手續費比較低，如果想在 DigiFinex 換，那換成 AKT 提幣手續費會比較划算。</p><p>準備好錢包與加密貨幣之後就開始來操作了。</p><hr><h2>一、進入 Osmosis 去中心化交易所，連結 Keplr 錢包</h2><p>網址：<a href="https://app.osmosis.zone/" target="_blank"><strong>https://app.osmosis.zone/</strong></a></p><p>進入 Osmosis 交易所之後先連結錢包。</p><p>第一次連結時 Keplr 錢包會跳出確認訊息，這時候點 Approve 即可。</p><figure class="image"><img src="https://assets.matters.news/embed/aab53110-328e-40e7-a5d2-82968e81cc90.jpeg" data-asset-id="aab53110-328e-40e7-a5d2-82968e81cc90" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>連結錢包後可以看到錢包名稱及 OSMO 的餘額。</p><figure class="image"><img src="https://assets.matters.news/embed/b9c6396f-c556-48a5-9b02-2e3ab8ebf45f.jpeg" data-asset-id="b9c6396f-c556-48a5-9b02-2e3ab8ebf45f" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><h2>二、用 IBC Transfer 將加密貨幣轉入 Osmosis 鏈</h2><p>我是在交易所換了 ATOM ，把 ATOM 提幣到 Keplr ，因此我現在錢包裡己經有 ATOM，所以現在要進行一次 IBC Transfer 把 ATOM 轉到 Osmosis 鏈上，這個動作可以直接在 Osmosis 交易所執行。</p><p>在面板上點 Assets。</p><figure class="image"><img src="https://assets.matters.news/embed/42c23028-af3b-4c45-b9a2-d82efac1aec1.jpeg" data-asset-id="42c23028-af3b-4c45-b9a2-d82efac1aec1" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>找到你要轉入的幣種，按下 Deposit，因為我是換了 ATOM，所以是選 ATOM 那一欄。</p><figure class="image"><img src="https://assets.matters.news/embed/d6cfcc04-5dc5-4f50-b37c-35606d4e9074.jpeg" data-asset-id="d6cfcc04-5dc5-4f50-b37c-35606d4e9074" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>接下來會讓你輸入要轉入的金額。</p><figure class="image"><img src="https://assets.matters.news/embed/866b66ea-1bd9-43a2-81f1-2c6e4425e185.jpeg" data-asset-id="866b66ea-1bd9-43a2-81f1-2c6e4425e185" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>點了 Deposit 後會跳出 Keplr 的確認視窗，告訴你這筆 IBC Transfer 的手續費，沒問題的話點 Approve 。</p><figure class="image"><img src="https://assets.matters.news/embed/aac6ed95-9ad0-4a9d-834c-d962abb614df.jpeg" data-asset-id="aac6ed95-9ad0-4a9d-834c-d962abb614df" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>然後錢包跟交易所陸續跳出幾個訊息框，之後就完成轉換！</p><figure class="image"><img src="https://assets.matters.news/embed/66c0f61e-5297-4876-af5e-a759a3a749ec.jpeg" data-asset-id="66c0f61e-5297-4876-af5e-a759a3a749ec" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><h2>三、在 Osmosis 交易所轉換加密貨幣</h2><p>因為對 ATOM / OSMO 這個交易對增加流動性，需要用 ATOM 及 OSMO 兩種幣，因此還要再把 ATOM 換成 OSMO，我選擇的 Pool 是 50% ATOM、50% OSMO 所以我現在要把一半的 ATOM 換成 OSMO。</p><p>1、點面板上的 Trade。</p><p>2、我要把 ATOM 換成 OSMO 所以 From 的地方選 ATOM。</p><p>3、To 的地方選 OSMO。</p><p>4、填入要換的金額，會自動跳出可兌換到的金額，下方會有兌換的資訊！</p><p>5、點 Swap。</p><p>之後一樣是會跳出 Keplr 的確認視窗，告訴你這筆交易的資訊及手續費，<strong>按 Approve</strong> 後一樣錢包與交易所跳出一些訊息，便完成兌換。</p><figure class="image"><img src="https://assets.matters.news/embed/09edb83a-c4f3-4b28-a254-bb3dfc94e721.jpeg" data-asset-id="09edb83a-c4f3-4b28-a254-bb3dfc94e721" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>完成之後進 Assets 就可以看到你兌換完成之後，各種幣的最新數量。</p><figure class="image"><img src="https://assets.matters.news/embed/87db427e-e33b-4a59-b83c-978967ba0e12.jpeg" data-asset-id="87db427e-e33b-4a59-b83c-978967ba0e12" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><h2>四、增加流動性操作</h2><p>接下來終於可以操作增加流動性了，先點選面板上的 Pools 進入增加流動性的頁面。</p><p>1、點 Pools。</p><p>2、選 ATOM / OSMO 交易對的 Pools。</p><figure class="image"><img src="https://assets.matters.news/embed/f1f15c06-ae37-4fc8-8106-0f8e44ab94ec.jpeg" data-asset-id="f1f15c06-ae37-4fc8-8106-0f8e44ab94ec" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>進入 Pool 的頁面之後，要先把 ATOM 及 OSMO 換成 LP tokens。</p><p>Swap Tokens 按下去是轉換加密貨幣的功能，因此步驟三，也可以在這裡做。</p><p>1、點 Add / Remove Liquidity。</p><p>2、會跳出 Manage Liquidity 視窗，這裡會顯示需要放入的比例，大部份都是 50% : 50%，也就是要放入等值的 ATOM 及 OSMO。</p><p>3、輸入其中一個金額會自動跳另外一個所需的數量。</p><p>4、點 Add Liquidity</p><figure class="image"><img src="https://assets.matters.news/embed/23f3d6dc-e618-4d5f-9616-6c75b827338e.jpeg" data-asset-id="23f3d6dc-e618-4d5f-9616-6c75b827338e" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>完成之後跳回畫面會就看到持有 LP tokens 的數量。</p><p>1、持的有LP tokens 數量。</p><p>2、可以選擇解綁的期間〔Unbonding Duratio〕，這裡分別有 1天、7天、14天可以選〔指的是解綁需等待的時間〕，每個的年化報酬率不一樣〔你沒看錯，這個交易對真的是一百多趴的報酬率〕，可以把持有的 LP tokens 分配給不同的天數，這樣之後你的幣可以靈活一點運用，急著用幣就從解綁期 1 天的那個解綁。</p><p>3、點 Start Earning 開始操作。</p><figure class="image"><img src="https://assets.matters.news/embed/16fb7283-627e-4293-a910-b06143076545.jpeg" data-asset-id="16fb7283-627e-4293-a910-b06143076545" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>點了 Start Earning 會跳出視窗。</p><p>1、選擇天數。</p><p>2、輸入 LP tokens 數量。</p><p>3、按 Bond 。</p><p>接下來又會跳出 Keplr 的確認視窗，告訴你這筆交易的資訊及手續費，<strong>按 Approve</strong> 完成。</p><figure class="image"><img src="https://assets.matters.news/embed/c193b99c-ccca-449d-a6c6-2b7e270a0ae7.jpeg" data-asset-id="c193b99c-ccca-449d-a6c6-2b7e270a0ae7" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>回到 Pool #1 的畫面往下拉就可以看到你目前分配的流動性挖礦情形。</p><p>之為我之前又做了一個是解綁期 7 天的，這次分配了 15 LP tokens 進 1 天的，其他全部進了 14 天的，所以現在三個都是運作中，Amount 即是你投入 LP tokens 的數量。</p><figure class="image"><img src="https://assets.matters.news/embed/c675f950-e2ed-47be-a751-2f0eca943919.jpeg" data-asset-id="c675f950-e2ed-47be-a751-2f0eca943919" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>接下來，每天會分配一次收益〔在凌晨大家都睡著的時候〕，然後你每天就會看到你的錢包裡， OSMO 每天默默的增加了．．．</p><p>下面是我之前投入 7 天的池裡每日錢包 OSMO 的變化，因為我質押給驗證人的收益都沒有領取，所以應該就是流動性挖礦的收入了。</p><figure class="image"><img src="https://assets.matters.news/embed/fad3908c-e8dd-4234-9f6b-557dda31b274.jpeg" data-asset-id="fad3908c-e8dd-4234-9f6b-557dda31b274" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>順帶一提，在Osmosis 交易所人人都可以新建一個 Pool，見下圖，點 Create New Pool，不過需付一定數量的 OSMO 作為創建費用 ，目前是100 OSMO。</p><p>對了，其實進入 Pools 畫面的最上方，就有倒數發放收益的時間囉！</p><p>另外需要取回幣時就在 My Bondings 那裡點解綁，等解綁期過去之後，點Add / Remove Liquidity，切換到  Remove Liquidity，就可以把 LP tokens 變成原來的兩種幣，之後再換成你想要的幣轉出就可以，如過有需要，過陣子我再來操作解綁定並寫一篇教學吧！^^</p><figure class="image"><img src="https://assets.matters.news/embed/9b00631b-a2e8-4ddc-8cdf-0488314d61ff.jpeg"
