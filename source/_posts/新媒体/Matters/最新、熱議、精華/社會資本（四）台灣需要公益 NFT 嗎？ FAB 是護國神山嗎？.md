<p>吟遊詩人平常說別人的故事，今天要回頭講自己的故事。<br class="smart">自己捧自己很怪，但請大家看看我怎麼耍猴戲。<br class="smart">（歡迎轉載）<br class="smart"></p><hr><p><strong>台灣需要公益 NFT 嗎？ FAB 是護國神山嗎？</strong></p><p>時間回到半年前，從醫師離職的那個晚上，我不小心畫出了一個國家級迷因 — 壓線球國旗。當它如病毒般傳播到大街小巷時，所有權論戰開打了，甚至有人申請商標偷襲，一時之間甚囂塵上。</p><p>但喪失自由的迷因將不再是迷因，所以我逆向處理，將壓線球國旗鑄造成 NFT，然後進行公益拍賣，最後捐出七萬多元。</p><p>這是我做的第一個 NFT，也開啟了我的公益 NFT 探索之旅。</p><figure class="image"><img src="https://assets.matters.news/embed/9ae02819-b7ad-4a79-9d11-340d1f3b7f37.png" data-asset-id="9ae02819-b7ad-4a79-9d11-340d1f3b7f37" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><p>這半年來，除了努力書寫 NFT 創作者的故事，取得吟遊詩人稱號（？）（今天追蹤人數剛好破 5K，謝謝大家），我也發起或參與了 13 件 NFT 專案，以兩週一件 NFT 企劃的速度，探索 NFT 的極限（當然，極限還遠遠夠不著）。</p><p>自壓線球國旗之後，我發起〈Loot Revolution〉，與一群朋友以文字 NFT 進行香港的倡議；與好夥伴<a href="https://www.facebook.com/derui.chen?__cft__[0]=AZV9o4lOF-N4yrbZGBdrYQpYjmRIP5MrvOI1y0rjrX37igp8SJwtwK7ay9jUL3AdJtzgGx5CDzl6NNtrJiKG_iNbVEKiAXPJUzfyc6ITVYXfHg80tfGgf-v5Axykv_9CS52u_gb5Gg5g8RHL59mWWl_t&__tn__=-]K-R" rel="noopener noreferrer" target="_blank">陳德叡</a>一起完成〈Lock-in Vending Machine〉，用一百天的時間，讓海外社群瘋狂愛上台灣的夾娃娃機；接著我與獨立書店「引書店」合作發行藏書票 NFT〈Ex-Libris “The Fool”〉，探討數位閱讀與作者分潤的新形式。</p><p>以上是文化的 NFT。</p><hr><p>過年期間，我在 Tezos 台灣社群成立〈Supporter.tez〉NFT 藝術種子基金，幫助創作者無痛進入 web3 世界，影響力正在逐漸擴大；同時與三明治工、樂山教養院發行春節賀卡，進行公益 NFT 的嘗試。</p><p>以上是非營利的 NFT。</p><hr><p>三月，我用光明燈的意象，在寶藏巖光節展出 NFT 作品〈Treasury Dao〉，感謝台北國際藝術村讓我以藝術家身份參加，接下來還有鏈上的國際駐村行動；也感謝陳武鎮與<a href="https://www.facebook.com/profile.php?id=100000224740542&__cft__[0]=AZV9o4lOF-N4yrbZGBdrYQpYjmRIP5MrvOI1y0rjrX37igp8SJwtwK7ay9jUL3AdJtzgGx5CDzl6NNtrJiKG_iNbVEKiAXPJUzfyc6ITVYXfHg80tfGgf-v5Axykv_9CS52u_gb5Gg5g8RHL59mWWl_t&__tn__=-]K-R" rel="noopener noreferrer" target="_blank">陳玉珠</a>老師，讓我們能夠以〈消失的家人 NFT〉，進行白色恐怖與轉型正義倡議；當然也要感謝 AkaSwap 與 TezDozen，讓我們以川貝母筆下的貓咪 NFT，探索項目方與非營利組織合作的可能性。</p><p>以上是組織規模的 NFT。</p><hr><p>四月，我們與北師美術館展開了〈光 - 文協百年〉的 NFT 企劃 — 參與式共創 NFT〈Kng〉，就在今天，深夜的共創工作坊正式落幕；此外我們也協辦了「若水獎」，優化參與流程與促進了前沿治理的討論，讓更多創作者可以適應這些新工具。</p><p>以上是跨界合作的 NFT。</p><hr><p>發想企劃，付諸實踐，真的會讓人上癮。</p><p>這半年的生活，大致上就是圍繞著 NFT 打轉。</p><p>以上是我所鍾愛的作品們，接下來還有許多作品，等著分享給大家。</p><hr><p>不知道大家有沒有發現，這段實踐的過程，</p><p>漸漸從「我」變成「我們」。</p><p>一個人再厲害，也不可能在短時間內敲出連續安打；如果只有一個人，企劃的量級是不可能提升的。</p><p>因此我真的很幸運，與一群志同道合的夥伴成立了一個非營利平台，名為<a href="https://www.facebook.com/FABDAOtw/?__cft__[0]=AZV9o4lOF-N4yrbZGBdrYQpYjmRIP5MrvOI1y0rjrX37igp8SJwtwK7ay9jUL3AdJtzgGx5CDzl6NNtrJiKG_iNbVEKiAXPJUzfyc6ITVYXfHg80tfGgf-v5Axykv_9CS52u_gb5Gg5g8RHL59mWWl_t&__tn__=kK-R" rel="noopener noreferrer" target="_blank">FAB DAO</a>，全名為福爾摩沙藝術銀行（Formosa Art Bank）。</p><p>這個名字是有玄機的。</p><hr><p>施振榮聽到這個名字，問說：「FAB？我很熟啊，不就是晶圓廠。」</p><p>我OS：埋梗埋了數個月，終於有人拆穿了。（淚目）</p><p>FAB，Semiconductor Fabrication Plant 之簡稱，晶圓廠被稱為台灣的護國神山。此外 FAB 也有自造者之意，我們向上一個世代的自造者運動致意。那 FAB DAO 是台灣的護國神山嗎？我不這麼認為。</p><p>web3 應該是一個自下而上的世界，be water，上善若水，共識是流動的，一群人、一些錢聚集於此，讓公益模式不斷迭代。一切是自然、彈性、反脆弱的，不需要國，不用特別守護。</p><hr><p>但是，直到俄烏戰爭開打我才有所改觀。我們見識到加密貨幣與 NFT 是如何介入這場戰爭，烏克蘭甚至在戰爭還沒結束時，便有了自己的公益 NFT 戰爭博物館，民間也自發地創造各種捐款智能合約，來應對這場侵略。</p><p>但合約的後端需要人與人之間的信任，不然合約只是無用的自動販賣機。傳統的信任來自於機構，新型態的信任來自於平台。FAB DAO 希望在承平時期，成為公益的載體；在緊急時期，成為信任的集結。</p><p>我們哪時候用得到？實在很難說。我們必須現在開始，以公益累積信任。</p><p>所以我們籌備 FAB DAO 多時。</p><hr><p>為了一個題目，我們進行了為期數月的拜訪之旅，「NFT 如何做公益？」。中間遇到貴人無數，讓我相信善念可以最大幅度地降低摩擦成本，讓幸運可以無限 buff。</p><p>首先是老夥伴<a href="https://www.facebook.com/muse.elegy?__cft__[0]=AZV9o4lOF-N4yrbZGBdrYQpYjmRIP5MrvOI1y0rjrX37igp8SJwtwK7ay9jUL3AdJtzgGx5CDzl6NNtrJiKG_iNbVEKiAXPJUzfyc6ITVYXfHg80tfGgf-v5Axykv_9CS52u_gb5Gg5g8RHL59mWWl_t&__tn__=-]K-R" rel="noopener noreferrer" target="_blank">許皓甯</a>與<a href="https://www.facebook.com/wchuang17?__cft__[0]=AZV9o4lOF-N4yrbZGBdrYQpYjmRIP5MrvOI1y0rjrX37igp8SJwtwK7ay9jUL3AdJtzgGx5CDzl6NNtrJiKG_iNbVEKiAXPJUzfyc6ITVYXfHg80tfGgf-v5Axykv_9CS52u_gb5Gg5g8RHL59mWWl_t&__tn__=-]K-R" rel="noopener noreferrer" target="_blank">黃文君</a>，他們去年聽了我的想法之後，選擇讓書店空轉，加入我無限的拜訪之旅。我們開發出了讓非營利 DAO 平台運轉的核心模式。</p><p><strong>公益上鏈，幫助當代的人群；<br class="smart">藝術上鏈，創造未來的藝術；<br class="smart">文化上鏈，繼承過去的美好。</strong></p><p>每一條路線，我們皆逐步實踐，應是有目共睹。這中間所有的 know-how 皆免費公開給大家，所有對話紀錄皆保存於 Discord。而我們恐怖的夥伴亮亮，正在將所有環節標準化，讓舊的問題能有教科書做解，讓大家能共同處理新的問題。</p><hr><p>再來，不知道大家還記不記得，我在去年十二月初，寫了一篇關於<a href="https://www.facebook.com/cheyuwu345?__cft__[0]=AZV9o4lOF-N4yrbZGBdrYQpYjmRIP5MrvOI1y0rjrX37igp8SJwtwK7ay9jUL3AdJtzgGx5CDzl6NNtrJiKG_iNbVEKiAXPJUzfyc6ITVYXfHg80tfGgf-v5Axykv_9CS52u_gb5Gg5g8RHL59mWWl_t&__tn__=-]K-R" 10K NFT 專案，應該是台灣最大的公益 NFT 專案，總共有 10101 座山。萬巒群峰，一座 0.025 ETH（台幣兩千元出頭），創造群眾參與治理的可能性；百岳群象，一座 10E（八十萬元左右），第一座便是由哲宇捐贈。</p><p>發行於以太坊上面。</p><hr><p><strong>什麼是護國神山？</strong></p><p>我認為是他願己力，群眾共同打造有效的互助系統。<br class="smart">而「百岳計畫」會是一切的開端。</p><p>皓甯曰：</p><p><strong>一百座山向上隆起，成為一座島嶼；<br class="smart">一百個人往前一步，寫下一季文明。</strong></p><hr><p><strong>NFT 會泡沫化嗎？會的。</strong></p><p>NFT 需要 NPO，尤其是在幣價大跌的今天。<br class="smart">NPO 有強大的精神價值，NPO 不 Rug Pull。</p><p>NPO 需要 NFT 嗎？需要。</p><p>NPO 需要新的捐款途徑，NPO 需要更好的捐款收據；<br class="smart">NFT 有永續分潤、透明金流、新的 web3 社群等新興工具。</p><p>（NPO 為非營利組織）</p><p>雙方或許可以互相幫助，因此我們開發出一套系統，名為 N2 Project，NPO X NFT。</p><p>重點不是捐款給 NPO，而是 NPO 自己能夠發行 NFT，取得自己的數位自主權，如同藝術家一樣。唯有如此，NPO 才能掌握 100% 的 web3 紅利。</p><hr><p><strong>而為何是 DAO？從去年便開始每日反問。</strong></p><p>DAO 不外乎是進行更好的資源分配，不論是人力還是物力。</p><p>我們希望理想的 DAO，我們越小越好。 DAO 為聚合者，兼容各方能人，特別是在公益市場這種「公地」。未來有機構、組織、個人工作者、行動者、以多方協作的方式探索公益 web3 市場。</p><p>治理的方式，靠代幣與共識。</p><p>因此我們會發行 DAO 治理代幣，名為 $FOR，這是 For Formosa 的治理代幣，但這個代幣不會走傳統 ICO（首次發幣上市），而是走公益 NFT 市場空投，我稱之為 INO（Initial NFT Offering）。</p><p>如果你購買公益 NFT 幫助非營利組織，你會獲得捐款等值 $FOR 空投，這是好人好事獎勵；而非營利組織也會因為你，獲得等量空投，畢竟他們更是好人；再來，參與促成的執行者也會獲得代幣，因為他們是 work 2 earn。</p><figure class="image"><img src="https://assets.matters.news/embed/9baec03b-285d-45fa-a166-ad25e4810d8b.png" data-asset-id="9baec03b-285d-45fa-a166-ad25e4810d8b"
