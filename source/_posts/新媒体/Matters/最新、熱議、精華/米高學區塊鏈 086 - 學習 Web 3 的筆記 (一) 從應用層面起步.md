<p>兩週前參加了Matters的學習課，當日的主題是 Web 3，這是我很期待的主題之一，因為我真的很想知道 Web 3 的世界，到底會變成怎樣?</p><p>在 Web 3 的世界低層技術比應用層面重要，最少現時此刻是這情況，很多應用只有少量功能，不過相當穩定後就會慢慢建立，好像 Steem 及 Matters 一樣，會一步一步增加功能，改善 UI / UX。</p><p>現時的 Web 3 應用、錢包管理、CEX 或 DEX 交易所 及 社交平台都比兩三年前改善很多。雖然仍然很難用，對於新手仍是摸不著頭腦，甚至連老手也怕會轉錯錢，操作錯誤而遺失Token。但是，若 此時此刻的 Web 3 項目專注於底層開發；那麼，只要這一部份穩健了，就會扶搖直上。當中，我最期待就是 Web 3 的社交平台，而現時我最欣賞就是 Matters。</p><figure class="image"><img src="https://assets.matters.news/embed/a3ff109f-7924-4fe7-bc64-8486d17c2bc7.png" data-asset-id="a3ff109f-7924-4fe7-bc64-8486d17c2bc7" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p>另外一張在Matters課堂的Slide，就是不同社交平台的圍牆。</p><p>圈外或初初接觸區塊鏈的朋友，常常跟我說「這麼多條區塊鏈，讓人感到很混亂，這樣混亂，怎會成功?」</p><p>有時，若我情緒狀態比較好時，我會嘗試細細解說，不過普遍對方「仍是不想明白」，或許，對方心底裡是想著「區塊鏈不可能成功」。</p><p>其實，區塊鏈已發展得非常好，踏踏實實的項目已越來越多，當然仍有很多歸零項目，不過，正如 Web 1 及 Web 2 的世界，有2000萬以上的網站，絕大部份 (90% 以上) 的網站人流都是小貓三幾隻，難道...... 就代表互聯網絡已死??? 又例如在智能手機的 App Store 上，已經有超過 1000萬個 App，但是，同樣有極大量的 App 連 100個下載也沒有，如此，我們就可以斷言移動互聯網及智能手機的失敗?</p><p>區塊鏈的應用上，現時看上去仍然是有很多缺陷，但是，已可預見有不少項目可以再運作五年或十年。例如我們在使用的 LikeCoin 應該會是其中一個例子。</p><p>的而且確，現時的區塊鏈「公鏈」數量不少，並且各據山頭，偶有一些項目能互通。但是，在 Web 2 的世界不也是一樣嗎? 有不少的項目只有 App 版本，沒有電腦版；又或者只有電腦版，而沒有 App；更甚是有一些網絡項目是在遊戲主機上才能連上的網絡。</p><p>為甚麼...... 我們卻不覺得有問題? 會視為失敗?</p><p>可能，我們站得太高了，用了上帝視野去看區塊鏈項目，如此就很容易看到遍地罪惡。若果我們如耶穌般，落到凡間，降世為 Web 3 的人類。從一些自己喜愛的應用層面去看，慢慢投入其中，慢慢認識這些同喜好的鏈上人民，慢慢就會發覺，原來每一個項目也有不少人在參與及在努力改善當中。</p><p>正如，我喜歡「區塊鏈內容平台」，我就專注在遊走不同的平台，了解各個平台的優缺點與及產出Token的模式，最後選出最適合我的平台，並且長駐。</p><p>當選了喜好的項目，並且長駐其中的話...... 其實不同鏈之間「有沒有」互通，與我又有甚麼關係呢? 與及，在應用層面上，不同鏈沒有互通，也不會阻礙用戶參與。因為普遍的使用者，其實只是專注於應用，而不是上帝視角。</p><figure class="image"><img src="https://assets.matters.news/embed/c267e393-f5fc-4330-b93f-e0b35e4b068e.png" data-asset-id="c267e393-f5fc-4330-b93f-e0b35e4b068e" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p><br></p><p>朋友間，很多人都想了解區塊鏈，不過，普遍的人用了「先全盤了解，再看看是否參與。」</p><p>這心態有如以往向未信耶穌的朋友傳福音一樣，我有不少朋友的心態是「要先全盤了解基督教，之後才決定是否返教會。」</p><p>實際上，我身邊普遍的基督徒，絕大部份是因著很多微不足道的原因，可能是有人關心，雪中送炭；又或者看了別人的信仰改過自身經歷，而自己也想有所改變。再在當基督徒的過程中，一邊作為「門徒」學做「基督」的心態及視野。</p><p>要讓人認識區塊鏈或參與加密貨幣，我常常覺得自己在傳教，希望身邊朋友對「去中心化」有信仰。</p>
