<figure class="image"><img src="https://assets.matters.news/embed/7e978cc6-8677-4191-9c0f-0ea7451d42fb.jpeg" data-asset-id="7e978cc6-8677-4191-9c0f-0ea7451d42fb" referrerpolicy="no-referrer"><figcaption><span>Web 3.0 的本意是賦權於民，讓每個人都能有所貢獻。圖片來源：fabio (Unsplash)</span></figcaption></figure><p>「<strong>Web 3.0</strong>」，你最近應該常聽到這個詞，Meta (臉書) 炒紅了元宇宙，而元宇宙背後的架構Web 3.0 開始被各大名嘴跟網紅拿出來討論。</p><p>第一個提出 Web 3.0 概念的是<a href="https://en.wikipedia.org/wiki/The_New_York_Times" rel="noopener noreferrer" target="_blank">紐約時報</a>的記者<a href="https://en.wikipedia.org/wiki/John_Markoff" rel="noopener noreferrer" target="_blank"> John Markoff</a> (你可以在<a href="https://www.nytimes.com/2006/11/12/business/entrepreneurs-see-a-web-guided-by-common-sense.html" rel="noopener noreferrer" target="_blank">這裡</a>找到他在 2016 年發表的文章)，現在，相關的運用似乎無邊無際。今天，我要分享一些從全球 FinTech 樞紐中心的新加坡學來的一些 Web 3.0 的知識，希望讓你一瞥 Web 3.0 將如何影響金融服務產業的未來。</p><h2>Web 3.0 金融產業知識課程認證</h2><p>MAS (<a href="https://en.wikipedia.org/wiki/Monetary_Authority_of_Singapore" rel="noopener noreferrer" target="_blank">Monetary Authority of Singapore</a>)<a href="https://zh.wikipedia.org/wiki/%25E6%2596%25B0%25E5%258A%25A0%25E5%259D%25A1%25E9%2587%2591%25E8%259E%258D%25E7%25AE%25A1%25E7%2590%2586%25E5%25B1%2580" rel="noopener noreferrer" target="_blank"> 新加坡金融管理局</a>在今年 11 月 8~12 日舉辦了 2021 <strong>SFF </strong>(<a href="https://www.mas.gov.sg/development/fintech/singapore-fintech-festival" rel="noopener noreferrer" target="_blank"><strong>Singapore FinTech Festival</strong></a>/ 新加坡金融科技節) 。這個金融界的年度盛事旨在展現 Web 3.0 和相關科技 (如 DLT 分散式帳本、IoT 互聯網、5G、AI 人工智慧、ML 機器學習) 將如何改變金融服務產業樣貌，金融科技新創們亦藉機演示如何運用 Web 3.0 拓展版圖。</p><p>本文是我從大會<a href="https://www.fintechfestival.sg/certification/" rel="noopener noreferrer" target="_blank"> <strong>Web 3.0 Knowledge Certificate in Financial Services</strong></a> (Web 3.0 金融產業知識課程認證) 課程學習到的 Web 3.0 知識和洞見分享。</p><p>此課程由新加坡國立大學和 AIDF (<a href="https://www.aidf.nus.edu.sg/" rel="noopener noreferrer" target="_blank">Asian Institute of Digital Finance</a>) 亞洲數位金融研究所共同規劃，透過來自超過 60 個國家的專家，將 Web 3.0 的應用分為 3 大模組；分別是 <strong>Designing Financial Services</strong> (設計金融服務)、Building Financial Services (打造金融服務)，和 Sustainable Financial Services (永續金融服務)。</p><p>以下為第一個模組 <strong>Designing Financial Services </strong>的部分內容。</p><h2>用 Web 3.0 打造金融服務</h2><p>第一個模組以新加坡金融管理局的首席金融科技官<a href="https://www.linkedin.com/in/sopnendu/?originalSubdomain=sg" rel="noopener noreferrer" target="_blank"> Sopnendu Mohanty</a> 對前<a href="https://en.wikipedia.org/wiki/Coinbase" rel="noopener noreferrer" target="_blank"> Coinbase</a> 技術官及現任頗負盛名的矽谷創投公司<a href="https://en.wikipedia.org/wiki/Andreessen_Horowitz" rel="noopener noreferrer" target="_blank"> a16z</a> (Andreessen Horowitz) 一般合夥人<a href="https://www.linkedin.com/in/balajissrinivasan/" rel="noopener noreferrer" target="_blank"> Balaji Srinivasan</a> 的採訪打頭陣。</p><p>這個訪談為整個大會設定了基調。Web 3.0 被定義為一個後端的創新。它去中心化、賦權於民，讓每個人都能跳脫科技巨頭的干擾，對網際網絡做出貢獻。</p><p>我們也發現 Web 3.0 將改變金融服務產業的各個環節，舉凡資本市場、保險、資產管理、個人金融，亦或企業金融，都將就此改變。</p><p>讓我先分享 Web 3.0 對保險市場的影響。</p><h2>Web 3.0 在保險市場</h2><p>關於保險市場的議題有兩場訪談。</p><p>第一場採訪對象是保險巨頭<a href="https://zh.wikipedia.org/wiki/%25E4%25BF%259D%25E8%25AA%25A0" rel="noopener noreferrer" target="_blank">英國保誠人壽</a>的 CEO<a href="https://en.wikipedia.org/wiki/Mike_Wells_(businessman)" rel="noopener noreferrer" target="_blank"> Mike Wells</a> 和保險新創 WeFox 的 CEO<a href="https://www.linkedin.com/in/julianteicke/?originalSubdomain=de" rel="noopener noreferrer" target="_blank"> Julian Teicke</a>，這是一段 160 歲的產業巨擎對上僅 6 歲的保險新創對話！兩位 CEO 皆強調對自家公司在保險市場繼續發光發熱的絕對信心。</p><p>WeFox 的 CEO Julian 先表達旗下新創獨角獸的成功 (WeFox 的<a href="https://www.bnext.com.tw/article/63358/wefox" rel="noopener noreferrer" target="_blank">最新估值已來到美金 30 億元</a>) 應歸功於能提供傳統業者所不能及的敏捷和數位轉型；接著保誠人壽的 CEO Mike 則回應保險的本質在於信任，並強調任何的創新都不該脫離這個核心價值。 </p><p>兩位 CEO 倒是一致同意保險產業還有巨大的商機；CEO Mike 進一步解釋道近年來保誠人壽的亞洲業務即使在市佔率不變的情況下，依然有爆炸性的成長，因此他重申未來的保險市場不會是贏家通吃，只要能留在市場上，人人都能受惠。</p><p>從這場對談我們可以去思考，在未保險或保險過低的族群間一定還有很多機會等著大家去開發，而這些人需要什麼呢？就是以客戶為中心的個人化產品。如心臟病的病人可以戴智慧手錶並和保險公司分享他們的心跳數據，或較不富裕的客戶也可以接受營養師的建議並回報每天的餐點內容。</p><p>第二場訪談主要在討論 AI 和<a href="https://zh.wikipedia.org/wiki/%25E6%2599%25BA%25E8%2583%25BD%25E5%2590%2588%25E7%25BA%25A6" rel="noopener noreferrer" target="_blank">智慧合約</a>將如何徹底改變保險產業。主辦單位邀請了幾位有驚人履歷的保險專家與會：以太坊技術的英國保險公司<a href="https://golden.com/wiki/Nexus_Mutual-639YW89" rel="noopener noreferrer" target="_blank"> Nexus Mutual</a> 創辦人<a href="https://www.linkedin.com/in/hughkarp/?originalSubdomain=uk" rel="noopener noreferrer" target="_blank"> Hugh Karp</a>、來自泰國的年輕創業家<a href="https://www.linkedin.com/in/annabellekwok/?originalSubdomain=sg" rel="noopener noreferrer" target="_blank"> Annabelle Kwok</a> (她在 24 歲就已經創辦了兩間 AI 新創公司，後來更將其中一家公司賣給英國保誠人壽，並加入後者擔任區域 AI 主管)，還有<a href="https://zh.wikipedia.org/zh-tw/%25E4%25B8%25AD%25E5%259B%25BD%25E5%25B9%25B3%25E5%25AE%2589" rel="noopener noreferrer" target="_blank">平安集團</a>的首席創新執行官<a href="https://www.linkedin.com/in/jonathan-larsen-125210127/" rel="noopener noreferrer" target="_blank"> Jonathan Larsen</a>。</p><p>從這兩場座談，可以看出 Web 3.0 會促成一個更有效率的市場，大舉免去<a href="https://zh.wikipedia.org/wiki/%25E8%25B3%2587%25E8%25A8%258A%25E4%25B8%258D%25E5%25B0%258D%25E7%25AD%2589" rel="noopener noreferrer" target="_blank">資訊的不對稱</a>。未來在談保費跟賠償時，客戶和保險公司將不再是棋盤兩邊的對手，任何一方都不能在系統耍詐。</p><p>一方面，保單將更客製化、更多可以降低各式風險的保險產品將紛紛出籠。另一方面，Web 3.0 和物聯網可以大幅度地藉由教育客戶做好風險趨避和更為自己的健康負責來降低營運成本。</p><p>除了最近<a href="https://applealmond.com/posts/103273" rel="noopener noreferrer" target="_blank">特斯拉剛宣布要用客戶的駕駛行為分數來決定其保險費用</a>，在未來，我們可能會看到更多的保險公司透過 IoV<a href="https://wiki.mbalib.com/zh-tw/%25E8%25BD%25A6%25E8%2581%2594%25E7%25BD%2591" rel="noopener noreferrer" target="_blank"> 車聯網</a>紀錄你開車時的一舉一動，或是透過<a href="https://pansci.asia/archives/140036" rel="noopener noreferrer" target="_blank">量子計算</a>啟動智慧合約來改善令人詬病的保險索賠流程。當然，這些方案要成功上線的前提是客戶願意提供數據給保險公司。</p><p>另外，這也不是說我們可以忘了社會福利；當牽扯到保險時，我們必須時時提醒自己將 AI 倫理納入考量。你可以在我另一篇文章 "<a href="https://neverlandseeker.com/banking_ai_ethics/" rel="noopener noreferrer" target="_blank"><em>AI in Banking: Which is Better? Result or Ethics Driven?</em></a><em> </em>" (銀行裡的人工智慧：哪一個比較好？結果導向或倫理導向？) 看到對這個議題的討論。 </p><h2>Web 3.0 小結</h2><p>那麼，讀到這裡你現在對 Web 3.0 有什麼想法呢？ 你覺得下一個十年傳統保險業者還有機會在市場上叱詫風雲嗎？亦或未來將是 InsurTech 的世代？</p><p>希望你有從我的文章得到一些啟發，有任何見解歡迎在下方留言或透過 <a href="mailto:%20neverlandseeker.sc@gmail.com" rel="noopener noreferrer" target="_blank">email</a> 聯絡我。</p><p>若想收到最新文章，歡迎追蹤這個帳號或訂閱我的<a href="http://eepurl.com/hKYGSP" rel="noopener noreferrer" target="_blank">電子報</a>；你也可以在<a href="https://neverlandseeker.com/category/observer/" rel="noopener noreferrer" target="_blank">這裡</a>閱讀我的其他產業分享。</p><p>最後，如果覺得我的文章對你有幫助，歡迎透過讚賞公民幫我拍手 5 下支持我後續的寫作計畫，謝謝。</p>
