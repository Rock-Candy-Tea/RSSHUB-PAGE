chain：鏈與幣的歷史」。留意解說純屬個人觀感，不代表任何機構立場，一切以 LikeCoin 官方發佈為準。文章材料來自 LikeCoin 年度報告、社群會議紀錄、Facebook 等。如內容不全，只是本人力有未逮，歡迎幫助補遺。 </p><h2>2017</h2><p><strong>2017年1月</strong> LikeCoin 團隊成立並為 LikeCoin 構思與發想。</p><p><strong>2017年9月27日</strong> LikeCoin 的 "Genesis article" <a href="https://medium.com/likecoin/likecoin-appreciation-beyond-a-thumb-bb33cfe65203" target="_blank">Appreciation is more than a thumb</a> 明確設定了 LikeCoin 關注議題：在線內容付費以及溯源兩大問題。</p><p><strong>2017年10月4日 </strong>發表 <a href="https://medium.com/likecoin/likecoin-part2-zh-8e880091ebd3" target="_blank">構建基於區塊鏈的創用CC付費方案 — LikeCoin: 針對用戶生成內容的社群貨幣 #2</a> 一文，是 LikeCoin 設計的雛型。</p><p><strong>2017年12月31日 </strong>LikeCoin 得到數十位 early believers 支持，籌得首筆 LikeCoin 起動資金共 1,200 ETH。可以說沒有他們的「相信」，LikeCoin 連起動都不能，更莫論延續到今天。</p><h2>2018</h2><p><strong>2018年1月31日</strong> 推出 LikeCoin Store 網址 <a href="https://likecoin.store/" target="_blank">https://likecoin.store</a>（現時輸入這個網址會連到 Like.co）不需以太幣也能傳送 ERC-20 版本的 LikeCoin，讓大家使用網頁介面玩玩看，可以說是 LikeCoin 接觸公眾最初的起點。</p><p><strong>2018年2月8日 </strong><a href="https://medium.com/likecoin/likecoin-was-born-2533811ddd2f" target="_blank">LikeCoin 在以太坊 Ethereum 正式面世</a>，代號為 LIKE，智能合約地址為： 0x02f61fd266da6e8b102d4121f5ce7b992640cf98。當時大家儲存 LikeCoin 需要在桌面電腦下載 <a href="https://metamask.io/" target="_blank">MetaMask</a> 錢包，和現在的 <a href="https://www.keplr.app/" target="_blank">Keplr</a> 一樣，如果忘記了助記詞 ( Seed Phrase ) 和私鑰 private key，你的數位資產就沒有了。</p><p>MetaMask 錢包和 Liker ID 的關係可以解釋為：Liker ID 是讓你簡單地在網站安裝 LikeCoin button 與讀者互動賺取回報的個人化標誌，而資產其實是存放於錢包而不是 Liker ID 內，就算割斷與 Liker ID 的聯繫，一樣可以使用 Metamask 去活用 LikeCoin。假如創作者一直只使用 Liker ID 接收讚賞卻沒有綁定錢包，也就沒有辦法運用已獲取的讚賞。</p><p>由於使用以太坊，要傳輸，甚至兌換 LikeCoin 為法定貨幣都需要 ETH 作為傳輸的礦工費 Gas Fee。LikeCoin 基金會為了推廣也為了讓大家比較容易理解，使用 Liker ID 傳送 LikeCoin 予另一個 Liker ID 是不需要任何手續費的，因為基金會為你吸收了這部份的費用。但是如果你需要傳送 LikeCoin 到沒有 Liker ID 的錢包，就需要付出 ETH 作為礦工費。</p><p><strong>2018年3月22日</strong> Early bird token sale，賣出 60,000,000 LIKE。</p><p><strong>2018年4月2日</strong> <a href="https://medium.com/likecoin/whitepaper-8695aa9bfcf3" target="_blank">LikeCoin 白皮書</a> 出爐，LikeCoin 十年大計成型。</p><p><strong>2018年5月 </strong>正式 Token sale，<a href="https://medium.com/likecoin/tokensale-wrap-up-3a8b8acaf1c1" target="_blank">以原本目標 180% 超額完成</a>，售出 360,538,485 LIKE 總收入為 7,562.26 ETH。 開發池、創作基金等就是從中撥出。</p><p><strong>2018年6月4日</strong> LikeCoin 上架 Qryptos 交易所（及後改名為 Liquid，母公司 Quoine），然後在 IDEX、Tidebit、Tokenjar、BarterDEX、Exec 等交易所亦有 LIKE 的蹤影。</p><p>當年要兌換 LikeCoin 為法定貨幣，要先到自家地區的某個交易所，用法定貨幣換一些 ETH，然後把 ETH 從交易所的錢包傳送到自己的 MetaMask，然後把 LikeCoin 使用 ETH 作礦工費再用 MetaMask 傳送到支援 LIKE/ETH 交易的 Liquid 交易所，因為 Liquid 沒有 Liker ID 所以傳輸費用者自付。接著再把換回來的 ETH 傳送回自家的 MetaMask 或支援 ETH 兌換法幣的交易所再兌為法幣，絕對是漫漫長路。</p><p><strong>2018年7月</strong> 因為 <a href="https://medium.com/@astromnhsu/fcoin-%E7%9A%84%E5%85%A7%E9%83%A8%E9%81%B8%E8%88%89-%E6%84%8F%E5%A4%96%E5%A1%9E%E7%88%86%E4%BB%A5%E5%A4%AA%E5%9D%8A-ebbe6d153bfc" target="_blank">FCoin 令以太坊大塞車</a>，交易成本飆升幾百倍，可以說就算有錢都不能成功傳帳，<a href="https://medium.com/likecoin/%E5%8C%96%E8%AE%9A%E7%82%BA%E8%B3%9E-%E7%A8%8D%E5%BB%B6%E4%B8%8D%E6%AC%A0-d5805ba58fb1" target="_blank">成為 LikeChain（後名為 LikeCoin chain）出現的契機</a>。同類的情況早在 2017 年 CryptoKitties 經已出現過一次。 試想想新幣只推出了半年時間就經已在想轉型，不能不說這樣既大膽又有先見之明，看看今天 ETH 為 NFT 而高漲，如果 LikeCoin 仍然為 Liker 正式公佈並進行開發。</p><p><strong>2018年8月</strong> LikeCoin 終於登上加密貨資訊最高殿堂 <a href="https://coinmarketcap.com/currencies/likecoin/" target="_blank">CoinMarketCap</a>，面向世界。</p><h2>2019</h2><p><strong>2019年3月</strong> <a href="https://www.facebook.com/Liker.Land/photos/a.2297986110428087/2562747667285262/?type=3&theater"
