<p><a href="https://commonwealth.im/" rel="noopener noreferrer" target="_blank">Commonwealth.im</a> 是一個群雄論戰的聖地（形容太誇張但我真的有這個感覺），現時上面有 637 個 Communities 在討論區塊鏈大計。而 Osmosis 是其中之一。</p><h2>為甚麼要了解 Commonwealth？</h2><p>因為它以 thread 討論串功能去做討論，類似討論區那樣。於是同一個話題可以有歸一的討論，不像在 Discord 那樣，A 和 B 在講一個話題，而 C 和 D 中間穿插說別的東西。而且註冊帳號可使用區塊鏈錢包，故此一些區塊鏈項目都會很喜歡使用它來做議案的提案「試水溫」討論，如果反響太大，就未必會提案這樣。當然 Discord 或者 Reddit 也是討論之地，只是項目例如 Osmosis 社群會以 Commonwealth 作為官方討論區，在上面看到的東西很多時候談著談著就會發生了。</p><p>在 LikeCoin 社群事實上也有聲音在說是否需要開設一個 Commonwealth，雖然我不知道實際搞一個出來需要甚麼手續。</p><p>而在 Commonwealth 上面有 637 個 Communities，如果你有心有力有時間，或者在不同的社群討論當中可找到明日之星也說不定。</p><h2>Desmos Network 在 Commonwelth 的議案討論</h2><p>在 <a href="https://matters.news/@daisy/like-coin-%E8%AD%B0%E6%A1%88%E7%9A%84%E6%8F%90%E6%A1%88-%E6%8A%95%E7%A5%A8%E8%88%87%E5%9F%B7%E8%A1%8C-bafyreifjwgqayl6o65ej4wikyglmnlnflt463s7ahef5blm7polpqa77ke" Desmos Network 在 Commonwealth 的 Osmosis 社群上面發了個指引式議案討論，我就立刻金精火眼地看著。<a href="https://commonwealth.im/osmosis/discussion/3338-signaling-proposal-add-dsm-incentivized-pools" rel="noopener noreferrer" target="_blank">討論內容</a> 是這樣的：</p><figure class="image"><img src="https://assets.matters.news/embed/549f7f74-81a4-444b-9f15-e687061c1827.png" data-asset-id="549f7f74-81a4-444b-9f15-e687061c1827" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>哈囉 Osmosis 社區，我們將為 DSM 激勵池提出指引式議案 Signaling Proposal。下面是介紹：</p><p><strong>議案說明</strong></p><p>通過對這個議案投贊成票，OSMO 質押者表示支持將 OSMO 獎勵添加到 Osmosis 上的 <a href="https://app.osmosis.zone/pool/618" rel="noopener noreferrer" target="_blank">ATOM/DSM (#618)</a> 和 <a href="https://app.osmosis.zone/pool/619" rel="noopener noreferrer" target="_blank">DSM/OSMO (#619)</a> 池中。</p><p>通過對這個提案投反對票，OSMO 質押者表達他們對在 Osmosis 的 ATOM/DSM (#618) 和 DSM/OSMO (#619) 池中添加 OSMO 獎勵的異議。</p><p><strong>詳細內容</strong></p><p>這是一個指引式議案，將 DSM 對 ATOM/DSM (#618) 和 DSM/OSMO (#619) 歸入獲得 OSMO 流動性激勵的資產組合當中。這兩個池是在 2022年1月13 日（UTC）開始啟用。</p><p>截至撰寫本文時，價值 390 萬美元總計為 5,000,000 DSM 外部激勵，已於 2022年1月15日生效。</p><p>此議案通過後，上述 DSM 礦池將被添加到日常 OSMO 的激勵計劃中，並開展整個流程。</p><p><strong>關於 DSM</strong></p><p>Desmos 代幣，或 DSM 是 Desmos 區塊鏈上的原生代幣。Desmos 使用 Cosmos SDK 構築，作為支持在元宇宙中開發以用戶為中心的去中心社交網絡骨幹。</p><p>為了讓 Desmos 更加由社群帶動，我們已經向 11 個 Cosmos 生態系統鏈中的超過 280,000 個錢包地址展開了 2,100 萬 DSM 空投計劃（佔 DSM 創世供應量的 21%）。</p><p>關於 Desmos 的更多新消息可以在以下平台中找到：</p><ul><li><a href="https://discord.desmos.network/" rel="noopener noreferrer" target="_blank">Discord</a></li><li><a target="_blank">空投頁面</a></li><li><a href="https://www.youtube.com/watch?v=Qx95oqTW-6M&t=27472s" rel="noopener noreferrer" target="_blank">2021年11月在里斯本 Cosmoverse 發表演講</a></li></ul><h2>甚麼是 Osmosis 的 Incentivized Pools 激勵池，甚麼是 External Incentive Pools 外部激勵池？</h2><p><strong>Incentivized Pools 激勵池</strong></p><ul><li><a href="https://matters.news/@daisy/osmosis-%E6%B5%81%E5%8B%95%E6%80%A7%E6%8C%96%E7%A4%A6-apr-%E6%98%AF%E6%80%8E%E6%A8%A3%E8%AA%BF%E6%95%B4%E7%9A%84-dsm-%E6%B1%A0%E8%8F%AF%E9%BA%97%E4%B8%8A%E7%B7%9A-bafyreiegs4f422et4fmciiunkqhkau2bsiegnqikcvlsilxncleu7do6pu" rel="noopener noreferrer" target="_blank">Osmosis 流動性挖礦 | APR 是怎樣調整的？$DSM 池華麗上線</a> 一文談及在 618 619 池提供流動性，只能收到 swap fee，並沒有 Liquidity/Yield Mining incentives 流動性挖礦獎勵。</li><li>平常按鈕「Start Earning」邊擇「a day」、「7 days」、「14 days」的 Unbonding period 解綁期以後，<strong>流動性挖礦獎勵將以 OSMO 代幣</strong>在每天紀元 epoch 17:00 UTC（香港時區上午 1 時）結束時自動分發到你的 Keplr 錢包。而是否有獎勵是由 Osmosis 社群通過議案決定是否給予的，</li><li>以上的指引式議案就是讓 618 和 619 成為 Incentivized Pools 激勵池的草稿。相信在 Commonwealth 如果沒有收到甚麼強烈反對，這個議案就會被上架，到時有委託 OSMO 的大家就要努力投票了。</li></ul><p><strong>External Incentive Pools 外部激勵池</strong></p><ul><li>Osmosis 不但讓社群為流動性池透過議案變成激勵池，而且任何人都可以將代幣存入流動性池進行分配。這個功能讓「外部勢力」能在 Osmosis 的池子加入自己的流動性激勵計劃，並被稱為 External Incentive Pools 外部激勵池。而這個外部勢力，通常都是代幣的項目方。外部激勵不需要提案讓社群批淮就可去自己去做，只要有 $ 就可以（任性）自行放進去這樣 。</li><li>在 <a href="https://matters.news/@daisy/osmosis-%E6%B5%81%E5%8B%95%E6%80%A7%E6%8C%96%E7%A4%A6-apr-%E6%98%AF%E6%80%8E%E6%A8%A3%E8%AA%BF%E6%95%B4%E7%9A%84-dsm-%E6%B1%A0%E8%8F%AF%E9%BA%97%E4%B8%8A%E7%B7%9A-bafyreiegs4f422et4fmciiunkqhkau2bsiegnqikcvlsilxncleu7do6pu" rel="noopener noreferrer" target="_blank">Osmosis 流動性挖礦 | APR 是怎樣調整的？$DSM 池華麗上線</a> 一文撰寫時，因為未有外部激勵，所以「Start Earning」按鈕未有用武之地。</li><li>但正如上面的指引式議案所說，618 和 619 在 1月15日 開始有了 5,000,000 DSM 外部激勵，而從那時開始只要你做流動性挖礦，按下了「Start Earning」，就能獲得外部激勵。這些<strong>外部激勵以 DSM 代幣發放</strong>。</li><li>現時 Osmosis 的 UI 未能顯示 DSM 外部激勵，所以看到那些 0% APR 確實使人不了解。但是只要通過以上的指引式議案讓 618 和 619 池獲發 OSMO 獎勵，之後 DSM 就可以出現在「Semi-Automatic Incentive Adjustments for [日期]」那個議案讓社群通過，令 UI 顯示一拼回到正常狀態並讓大家獲得和看到<strong> OSMO 及 DSM 雙重獎勵。</strong></li></ul><figure class="image"><img src="https://assets.matters.news/embed/2d873f1b-5e67-4191-b87e-ddbadb2df5c2.png" data-asset-id="2d873f1b-5e67-4191-b87e-ddbadb2df5c2" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><ul><li>在 Osmosis Pool 頁面 <a href="https://app.osmosis.zone/pools" rel="noopener noreferrer" target="_blank">https://app.osmosis.zone/pools</a> 中的 External Incentive Pools 全部都是這麼玩的。</li><li>這些外部激勵當然只是一些短期鼓勵性設施，並不長久，故此你會看到「incentives for 178 epochs」，正如麥當勞套餐那樣供應期有限，178 個每天紀元 epoch 過後又或者激勵用完了以後沒有得吃。所以就算現時只有 DSM 獎勵，都可以考慮做流動性挖礦。</li></ul><figure class="image"><img src="https://assets.matters.news/embed/7880d3e0-fda3-44b4-8ca0-446fe312c2d5.png" data-asset-id="7880d3e0-fda3-44b4-8ca0-446fe312c2d5" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><ul><li>最重要的是，618 和 619 的 <a href="https://twitter.com/DesmosNetwork/status/1482335571745902594" rel="noopener noreferrer" target="_blank">外部激勵 APR > 1000%</a> 媽媽咪呀！所以現時雖然看不到那些激勵出來的 DSM，我都給它先流動性挖礦下去就是了。</li></ul><figure
