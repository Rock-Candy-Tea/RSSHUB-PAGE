<p>我跟太太是辦社會企業，讓弱勢社群運用資訊科技來謀生。在下一步的發展，我們想構建一個數碼設計工作室，因為我已在幣圈一段時間，所以嘗試探討是否可以將這一個工作室項目與NFT結合的想法。</p><p>幣圈與NFT圈是兩個不同的社群，而NFT圈的社群，我一直也很想了解，因為我覺得下一波的牛市或WEB 3 的社交應用是由NFT群帶動。</p><p>我不喜歡炒幣，而是發掘不同 WEB 3 應用，並且一步一步將 WEB 3 應用取代日常生活的項目。最簡單便是透過 Web 3 的文字內容平台寫文章，取代過往的Blog；又或者玩區塊鏈遊戲，取代手遊網遊。</p><p>這幾年幣圈生活，也令我身邊的朋友及日常的傾談交流中，80%以上都是與幣圈社群。幣圈以外的社交媒體，例如 Facebook, IG, Youtube，我都只是刷一刷「存在感」，基本已沒有甚麼交流。</p><p>我在互聯網絡及數碼行業工作已28年，Web 3 是必然會走的路，當中也不能缺少區塊鏈項目。而每一代的數碼網絡的普及化必然與社群成長有關。</p><p>幣圈的社群成長，在於每天追求不同的應用參與及體驗，遊走於不同的Airdrop任務之中。這些我都已成了習慣，過去亦當作遊戲攻略文一樣，寫了數十篇。</p><p>而當我遇上 NFT 圈時，我是驚為天人。</p><p>過往，我是看不起 NFT 項目，感到這是「尋找下一個傻仔」的營運模式，總有一天會被取代或淘汰。</p><p>不過，這幾個月的接觸，完完全全是另一回事。今天請教兩位行業專家，他們在幣圈的年資比我早一至兩年，於2016-2017已踏進幣圈，更有VC的背景。他們給我的感覺是「十分謙虛」，常常說著「仍在學習及摸索」不同的 Web 3 模式。最令我印象深刻是「分享每一個 NFT Project 時的興奮度」，他們面上的表情是真正的佩服及欣賞那些項目。</p><p>「NFT博大精深」是這位專家說出來的一句「意重深長」的話，我知他說出來的目的，是想我更認真去認識這一個社群與及NFT項目的發展。特別是項目方的創意、構思、社群的營運，與及當中的共同語言。</p><p>我今天真的大開眼界，上了一堂很豐盛飽足，很有營養的一課。</p><p>我鼓勵每一位踏進幣圈或NFT圈的朋友，試試不是停留在炒幣、追Airdrop 或寫文章賺幣。而是在幣圈內找幾位志同道合的朋友，嘗試在 Web 3 領域創作項目。</p><p>現時的 Web 3 ，就好像 1995年 Dos 轉 Windows；又或者 2000年的科網熱、2005年的Youtube、2008年的Facebook、2010年的 IOS Appstore、2012年的Android Googleplay，或2016年的Steam一樣。現時是 Web 3 的前期，我們是站在很近風口的位置，真金白銀想著炒賣也願意，為何不試試用自己的能力創意去賺幣，踏踏實實做一個「非土狗」「不Rug pull」的Web 3項目。</p>
