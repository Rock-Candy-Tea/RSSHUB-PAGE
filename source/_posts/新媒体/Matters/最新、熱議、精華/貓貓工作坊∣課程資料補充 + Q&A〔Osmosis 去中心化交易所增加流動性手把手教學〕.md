referrerpolicy="no-referrer"><figcaption><span>1天 跟 7天 數字是要扣掉的喔！！</span></figcaption></figure><hr><h2>Q & A 整理</h2><p><strong>Q1：無償損失什麼時候會被實現呢？</strong></p><ul><li>Ans：最精準的說法應該是去除流動性時，解綁的當下並不會被實現，過了解綁期間〔1~14天〕之後，還要做去除流動性的動作，讓 LP tokens 回復成兩種幣回到你的資產列表〔Assets〕時，無償損失才會被實現，解綁期間兩種幣的數量仍會變動，如果解綁期過了，但仍未解除流動性〔還是 LP tokens 的狀態〕，那代表這筆資金還在在流性動池中，所以兩種幣的數量也還是會變動。</li></ul><p><strong>Q2：有沒有什麼指標可以用於選擇投入的流動性池？</strong></p><ul><li>Ans：原則上建議還是選擇<strong>自己了解的幣別</strong>去投入，投入前查至少查詢一下投入的幣主要的應用是什麼，選擇兩種自己<strong>願意持有一段時間</strong>的幣去做流動性挖礦，如果一定要參考一些指標去投入，找交易量高〔代表手續費收入會多〕，社區激勵多，尤其是社區還調高激勵的〕，因為這代表社區可能希望有更多的流動性供應，以 Osmosis 交易所來說，還有 External Incentive Pools 可以選擇，我自己會選一些激勵多的交易對來做<strong>少量</strong>的投入，如果真的不知道要投什麼，那就建議 <a href="https://app.osmosis.zone/pools" rel="noopener noreferrer" target="_blank">Pool #1</a> ATOM / OSMO 的交易對囉！雖然現行可能不是最高的收益率，不過好處是這個池的交易量是最高的，另外 ATOM 也是 Cosmos 生態系的第一隻幣〔大哥的意思〕，而 Osmosis 是 Cosmos生態系中第一個去中心交易所〔元老的意思〕，另外目前許多空投會鎖定有在 Osmosis 提供流動性的用戶〔LP tokens 持有者〕，有些甚至會鎖定 <a href="https://app.osmosis.zone/pools" rel="noopener noreferrer" target="_blank">Pool #1</a> ATOM / OSMO 這個池去空投，因此投入這個池還可能有比較高的空投機會喔！另外其實 LikeCoin 的交易對也可以考慮，雖然目前收益率不高，不過近期 LikeCoin 社區有提供了外部激勵，我自己的另外一個錢包也有投入相關的池，另外，因為單純質押 LikeCoin 收益是 LikeCoin，而增加流動性，每天的收益是 OSMO ，以現在 OSMO 幣價相對強勢的情況，不失為一種選擇，因為在馬特市寫作的市民們應該持有 LikeCoin 的意願相對是高的。</li></ul><p><strong>Q3：Superfluid Staking 〔超流質押〕是什麼？</strong></p><ul><li>Ans：就是將投入流動性的 OSMO〔只有 OSMO喔，其他幣版本的超流質押應該是另外一個功能，也許不久後會實現〕，再拿<strong>一部份</strong>〔目前是 50% 〕去質押給驗證人，可以再得到一份質押的收入，目前只在 <a href="https://app.osmosis.zone/pools" rel="noopener noreferrer" target="_blank">Pool #1</a> ATOM / OSMO 的交易對試辦，並且只有綁定 14 天的才會獲得收益，未來會怎樣設定還要看官方規畫，猜測應該其他池要引入應該也會有相關提案，三不五時去看一下 Osmosis 社區的提案，也會得到一些資訊喔！</li></ul><hr><h2>課程相關補充資訊</h2><ul><li>關於超流質押操作參考文章：<a
