熱潮發行數位卡牌。只不過，每項新科技問世都會帶來一陣混亂。</p><p>這篇文章討論 NFT 這項新興資產、仿冒品問題，以及網路上對它的常見誤解。</p><h2><strong>NFT 熬出頭</strong></h2><p>你有玩過 NFT 嗎？一年前我曾發行過鏈上明信片給區塊勢會員<a href="https://blocktrend.substack.com/p/nft#footnote-1" target="_blank">1</a>。</p><p>如果沒有收過的話，區塊勢近期還會發行 NFT 卡牌並免費贈送給會員，讓你在不用花錢的情況下，就能體驗持有 NFT 的感覺。發行時間會另外公告，但可以確定的是這次不會在以太坊上發行，因為礦工手續費太貴了。</p><p>最近矽谷知名創投 a16z 推出一份 NFT 的閱讀清單〈<a href="https://a16z.com/2021/04/02/nfts-readings-resources/" target="_blank">NFT Canon</a>〉，統整了網路上具代表性的 NFT 討論、文章和影片。看完後我才發現，原來早在 1993 年就有人提出以密碼學當成防偽機制，並藉此發行獨一無二的數位卡牌：</p><figure class="image"><img src="https://assets.matters.news/embed/e98fc890-2d52-4306-b45a-f7ee2ecb0891.png" data-asset-id="e98fc890-2d52-4306-b45a-f7ee2ecb0891" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>1993 年連比特幣都還沒問世，遑論區塊鏈。</p><p>那時候人們能發行的只是「單機版」的 NFT。要說服別人買單，第一步是解釋密碼學的運作邏輯，接下來還得讓人理解，數位資產的稀缺性究竟能帶來什麼價值。</p><p>到了 2021 年，這件事變得簡單許多。</p><p>有越來越多人理解，只要將數位資產透過以太坊這樣的區塊鏈發行，這項資產就受到以太坊的「保護」，不僅難以複製且具有數位稀缺性。其中，最早將以太坊與 NFT 兩個概念結合在一起的專案就是 CryptoPunks。</p><p>根據佳士得拍賣的<a href="https://www.christies.com/features/10-things-to-know-about-CryptoPunks-11569-1.aspx" target="_blank">公告</a>：</p><blockquote>三年前，有兩位軟體開發者創建了這項名為 CryptoPunks 的瘋狂藝術計畫。這項計畫最具爭議性的地方是，幾行程式碼可以代表實際的擁有權嗎？用他們的話來說，這需要「認知升級」。三年後，這項計劃被視為當前 CryptoArt 運動的鼻祖。</blockquote><blockquote>CryptoPunks 的發行團隊 Larva Labs 將在 5 月 13 日，在紐約佳士得舉行的《21 世紀晚間拍賣會》中出售 9 個 Punks。紐約佳士得（Christie's）的專家 Noah Davis 指出，這將會是一筆相當具有歷史意義的交易，因為 CryptoPunks 代表了 CryptoArt 的起源。</blockquote><p>除了 NFT 鼻祖躍上佳士得，近期連職業摔角公司 <a href="https://www.wwe.com/article/wwe-presents-undertaker-nfts-wrestlemania-starting-saturday-bitski" target="_blank">WWE</a>、知名雜誌媒體 <a href="https://www.ledgerinsights.com/time-auctions-three-magazine-covers-as-nfts-on-blockchain/" target="_blank">TIME</a> 和奢侈品牌 <a href="https://www.ledgerinsights.com/fashion-brands-gucci-mull-3d-clothing-nfts-nonfungible-tokens-blockchain/" target="_blank">GUCCI</a> 都對 NFT 躍躍欲試。在諸多討論中，最多人問的是，買下 NFT 之後究竟買到了什麼？答案是 NFT 本身。</p><h2><strong>新興資產</strong></h2><p>什麼是 NFT？我會說它是全新的資產類別，就像加密貨幣既不是紙鈔也不是銀行存款一樣。我用下方這張圖來說明。</p><figure class="image"><img src="https://assets.matters.news/embed/9394f24b-cc04-45a7-8674-2213281a4f94.gif" data-asset-id="9394f24b-cc04-45a7-8674-2213281a4f94" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>先從左半邊看起。</p><p>紙鈔是人們最熟悉的「錢」。不僅有明確的發行方，還有浮水印防偽。而 10 年前出現的加密貨幣則是全新的資產類別，由區塊鏈作為防偽機制。它既不是由國家發行，也不是紙鈔的數位化。怎麼類比都不對。</p><p>現在看來單純的概念，人們可是經歷了 2017 年的 ICO 熱潮洗禮，才逐漸接受。加密貨幣和紙鈔、存款都不一樣，是一項獨立的數位資產類別。當時看到加密貨幣就盲目投資，恐怕是最糟糕的選擇。</p><p>右半邊也是相同道理。</p><p>《蒙娜麗莎》是人們耳熟能詳的畫作。不僅有明確的創作者，也有專家背書畫作是否為達文西的真跡。而近兩個月爆紅的 NFT 既不是傳統的實體畫作，也不是一張 JPG 圖檔，而是一種全新的資產類別。</p><p>如果羅浮宮選擇發行《蒙娜麗莎》的 NFT，它既不代表羅浮宮那幅真跡，也不只是一張 JPG 圖檔，而會像是數位的限量周邊商品，保存於區塊鏈上。沒有實體。</p><p>最近我就買了台灣職籃 P.LEAGUE+「新竹街口攻城獅隊」發行的 NFT 球員卡。它是由球隊發行的數位周邊商品，既不代表球隊的分潤權，也不代表我擁有這位球員的任何權利。只是人們現在對 NFT 還很陌生，法律的規範也還沒跟上科技發展的速度。因此許多人就容易誤以為買下 NFT 就像是擁有了實體物品的一部分。</p><figure class="image"><img src="https://assets.matters.news/embed/e1ecb105-fcdb-4ec8-a882-b17b5b4b594b.png" data-asset-id="e1ecb105-fcdb-4ec8-a882-b17b5b4b594b" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>理解 NFT 之後，要釐清網路上常見的誤解就容易許多。</p><h2><strong>亂象與誤解</strong></h2><p>鈔票或畫作都會遇到仿冒的問題，加密貨幣和 NFT 也不例外。</p><p>舉例來說，曾經有人在以太坊上以智慧合約發行名為 BTC 的加密貨幣，並以當時比特幣的 5 折價格，透過私下交易的方式賣給不知情的投資者。有心人以 BTC 的名稱魚目混珠，讓投資者防不勝防，就像區塊鏈上的「假鈔」。</p><p>雖然這些加密貨幣同樣是透過區塊鏈發行，但發行方根本不是比特幣的系統。</p><p>專家可能會說，BTC 和以太坊的地址完全不同，光看地址格式就知道不對勁。但許多投資者並沒有那麼熟悉區塊鏈的運作。如果下回有心人士發行的是名為 ETH 或 USDT 的「假鈔」，肯定也還會有更多人受騙。</p><p>同樣情況也會在 NFT 出現。一位瑞典創作者就在 Twitter 上抱怨，自己的作品在未經同意的情況下，被人偷偷拿去發行 NFT：</p><figure class="image"><img src="https://assets.matters.news/embed/08be257a-cb80-478c-be3c-6dfcd8050af0.png" data-asset-id="08be257a-cb80-478c-be3c-6dfcd8050af0" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>平台調查後才釐清，原來這項仿冒作品的發行者是創作者的粉絲之一：</p><blockquote><em>這是我們的一位用戶，他是您的粉絲，他將您的作品發行 NFT 作為可收藏的書籤，如果您想移除它，請告訴我們，我們將為您刪除該圖像。</em></blockquote><p>平台能做的只有亡羊補牢。</p><p>但同樣的事情，未來也很可能會出現在 NBA Top Shot、GUCCI 推出的卡牌上，且就像仿冒品一樣難以禁絕。人們必須透過正當的管道買賣，才可以避免買到 NFT 假貨。這與 NFT 是不是採用區塊鏈沒有關係，畢竟仿冒者也會使用區塊鏈。</p><p>其次，也有人會<a href="https://time.com/5947911/nft-environmental-toll/" target="_blank">批評</a> NFT 耗費大量能源、增加碳排放，進而破壞地球生態環境。</p><p>這是許多人的誤解。越多人創造、交易 NFT，並不會耗費更多能源。英國劍橋大學對區塊鏈能耗的研究 FAQ 中便直接<a href="https://cbeci.org/faq/" target="_blank">指出</a>，區塊鏈的交易吞吐量（系統可以處理的交易量）與消耗的電力無關。</p><p>礦工以電腦的運算能力維護區塊鏈的安全性。越多礦工加入挖礦會讓區塊鏈變得更安全，但不會讓區塊鏈的處理速度變快。即便完全沒有 NFT 的交易，區塊鏈還是會耗掉同樣多的能源，確保以太坊上的其他加密貨幣不受駭客攻擊。</p><p>換句話說，NFT 的出現不僅沒有增加能源消耗，反而是讓這些本來就會消耗掉的能源可以創造更大價值。如果未來以太坊進一步升級到 ETH 2.0，不僅轉帳速度會提升，消耗掉的能源還會大幅降低 <a href="https://ethereum.org/en/nft/#build-with-nfts" target="_blank">99.98%</a>。</p><p>長期而言，人們將不再爭論 NFT 這種全新形式的資產是否有價值，而是思考如何創造有價值的 NFT 資產。有了全新資產類別當然也會創造新商機，越早接受 NFT 才有機會提前佈局。</p><hr><p>若你想查閱區塊勢過往的出刊內容，可以參考<a href="https://blocktrend.pse.is/archive"
