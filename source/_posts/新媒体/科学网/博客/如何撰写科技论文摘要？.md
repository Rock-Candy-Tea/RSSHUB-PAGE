<p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; border: 0px; margin: 0px; padding: 0px;">摘要是让你的读者知道他们可以从你的文章期待什么。学习如何写出清晰、简洁的摘要很重要。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">摘要是一篇论文中，仅次于标题，第二多人阅读的部分。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">摘要应该是一个简明的独立的部分，准确地代表你的研究。</span>一个好的摘要可以帮助审稿人加快同行评审；同时，如果你的文章被接受，摘要将是一个重要的让读者发现和评价你的工作的工具。期刊编辑会根据摘要评估你的文章，产生第一印象。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">未来的审稿人在决定是否接受审查邀请时，第一眼也会看到它。一旦文章发表，摘要就会被PubMed、Google Scholar以及图书馆系统和其他流行数据库检索。和标题一样，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">摘要也会影响关键词搜索结果。</span>读者将使用它来决定是否阅读你的文章的其余部分。其他研究人员将使用它来评估你的科研表现，以便纳入系统评价和荟萃分析。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202105/21/101629yygawhgsb4f9ibyw.png" title alt referrerpolicy="no-referrer"><br></p><h1 style="font-size: 1.25em; margin: 0.63em 0px 1.8em; border: 0px; padding: 0px; line-height: 38px; color: rgb(25, 25, 25); counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="border: 0px; margin: 0px; padding: 0px;">摘要应该包含什么</span></h1><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">Abstract</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">当你写摘要时，你所面临的主要挑战是保持它的简洁和包含所有你需要的信息。期刊可能需要一个结构化的摘要-它可以帮助读者更容易地理解你的研究。如果目标期刊不需要一个结构化的摘要，遵循类似的格式仍然是一个好主意，只是把摘要作为一段，没有小标题。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">Background or Introduction - 目前已知什么?</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">以简短的两三句话开始，介绍研究领域。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">Objectives or Aims - 研究是什么，你为什么要做它?</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">清楚地陈述你想要回答的研究问题。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">Methods - 你做了什么?</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">解释一下你做了什么，怎么做的。包括关于你的方法的重要信息，但是避免低级的细节。有些学科对摘要部分的方法写作有特定的要求。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">CONSORT -用于随机对照试验。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">STROBE-用于观察性研究</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">PRISMA-用于系统回顾和荟萃分析</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">Results - 你发现了什么?</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">简要介绍一下你的研究的主要发现。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">Conclusions - 你得出了什么结论?</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">告诉读者为什么你的发现很重要，以及这对这个研究领域的“更大的未来”意味着什么。</p><h1 style="font-size: 1.25em; margin: 0.63em 0px 1.8em; border: 0px; padding: 0px; line-height: 38px; color: rgb(25, 25, 25); counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="border: 0px; margin: 0px; padding: 0px;">写作技巧</span></h1><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">skills</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">当你写摘要的时候，你会发现最主要的挑战是保持它的简洁和你需要的所有信息。</span></p><h1 class="ql-align-justify" style="font-size: 1.25em; margin: 0.63em 0px 1.8em; border: 0px; padding: 0px; line-height: 38px; color: rgb(25, 25, 25); counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="border: 0px; margin: 0px; padding: 0px; color: #C00000;">鼓励去做的：</span></h1><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">保持简明扼要。</span>大多数期刊都有字数限制，所以在你写摘要之前要检查一下投稿说明，这样可以节省以后编辑摘要的时间。</p></li><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">为读者写作。</span>他们是你专业领域的专家吗?他们是跨学科的吗?非专业人士吗?如果你的写作对象是普通读者，或者你的研究可能会引起公众的兴趣，那么尽量让你的语言直截了当。如果你用英语写作，要记住不是所有的读者都必须是以英语为母语。</p></li><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">重点关注关键结果、结论和信息。</span></p></li><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">先写论文其他部分，然后把摘要当做总结来写。</span></p></li><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">在写摘要之前，检查一下期刊的要求。</span>例如是否需要副标题。</p></li><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">包括关键词，以帮助读者在PubMed或Google Scholar索引数据库中搜索你的论文。</span></p></li><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">反复检查是否有拼写和语法错误。</span>这些错误可能会给潜在的审稿人留下这样的印象：你的研究不可靠。</p></li><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">摘要应该是满汉全席的一道开胃菜。</span></p><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;"></span></p><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;"></span></p></li></ul><h1 class="ql-align-justify" style="font-size: 1.25em; margin: 0.63em 0px 1.8em; border: 0px; padding: 0px; line-height: 38px; color: rgb(25, 25, 25); counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="border: 0px; margin: 0px; padding: 0px; color: #00B050;">不要去做的：</span></h1><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p>夸大你的研究。</p></li><li><p>推测这项研究未来的发展方向。</p></li><li><p>使用缩写或首字母缩略词(除非绝对必要或者大家都知道，例如:DNA)。</p></li><li><p>重复自己的话。</p></li><li><p>例如“Methods: We used X technique.Results: Using X technique, we found…”</p></li><li><p>与手稿的其他部分相矛盾。</p></li><li><p>引入手稿中没有包含的内容。</p></li><li><p>包括引文或参考文献。</p></li></ul><p class="ql-align-center" style="border: 0px;
