white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">撤稿通告的重要性</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">撤稿可能是由学术不端、科学失真或行政错误导致的。一般来说，撤稿代表了最初为确保已发表文章的高质量而建立的控制机制的某些方面的失败——要么作者没有遵守这些规则，要么编辑或出版商没有确保这些规则得到严格遵守。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">撤稿也反映了同行评审过程的失败。撤稿通告是出版商向其他学者和公众传递该论文不可靠的信息源，也是提醒人们不要引用被撤稿的论文或需要谨慎地依赖其结论的系统。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">由于撤稿通告也应报告论文的弱点或缺陷，它们也是论文出版历史的一个组成部分。因此，像任何历史文件一样，它应该诚实、全面、透明地反映事件（即撤稿）的历史。如果一份撤稿通告不完整，它就会成为一个糟糕的信息传递装置，从而达不到其出版历史和学术目的。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">如果信息量不够大，撤稿通告甚至可能是误导性的、有偏见的或歪曲的信息，仅给公众提供了片面的观点。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">鉴于不同的期刊和出版商对撤稿通告的政策和撤稿本身的认知可能存在很大的不同，发现撤稿通告在内容上存在差异并不奇怪。造成这种差异的其他原因还包括：</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">·与撤稿有关的伦理准则不够明确</span>，即使是在PubMed等知名索引平台上托管的期刊。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">·编辑自主性</span>，即编辑思考现有证据的自由，对自己的职责和监管范围的考量，并达成共识，这在不同的期刊之间有所不同。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">·作者、机构或公司的法律风险</span>。法律限制签发者可以或不可以在撤稿通告中陈述什么；有时通过法律手段来解决学术出版中的一些纠纷，包括撤稿等问题是必要的。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">·基层学者对政策制定的参与不足</span>，过度依赖政策团体（如COPE、ICMJE等），包括潜在的道德利益冲突。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">·虽然作者可以自由地提出他们对事实的描述，但撤稿通告的措辞几乎完全由编辑单独决定，或通常与出版商一起决定，后者可能涉及法律顾问；这反映了作者的权利受到了制约。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">对出版商而言，其最恶劣且最不负责任的行为是默默地撤回论文，不留下任何撤稿通告。而这实质上是抹去了论文的历史背景，隐瞒了论文的存在。标记不清楚或不明显的撤稿通告是被撤回的文献继续受到引用的原因之一。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">相比之下，及时、全面而信息透明的撤稿通告可以作为警告装置来强化科学记录的完整性。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">研究启示</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">研究发现在2008年至2011年期间，237篇被PubMed收录的撤稿通告的标题、作者名单等方面的索引存在6%的差异；此外，在一些知名出版社出版的经济学领域的55篇撤稿通告中，有51%没有说明撤稿的原因，尽管这些期刊声称遵循COPE撤稿指南。研究人员还注意到，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">从发现导致撤稿的问题到撤稿本身可能需要数年时间，大约有20%的稿件需要3年时间才能完成撤稿流程。</span>即使是基本信息也可能在撤稿通告中缺失。有研究发现，在955份1985-2017年发布的撤稿通告中，有219份没有注明撤稿日期；另有17%的撤稿通告的签发者不详。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">关于撤稿通告的不透明和可见性问题并不新鲜，早在三四十年前就已出现这种现象。因此有必要采取相应措施来改进撤稿通告，使其更加透明。这些措施包括：</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">·对修正文献的文化和心理层面的系统性改变；</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">·对支持学术不端行为审查机制的结构性改变；</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">·以及人们接受新的开放文化的意愿和愿望。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">这样的改变尽管是有益的，却也可能涉及某种程度的痛苦和牺牲，比如给已经超负荷的研究人员增加更多的工作量，或者提出财务要求，不公平地惩罚那些资金不足的机构或地区。假设撤稿通告 中的信息是不清楚的或未说明的，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">那么就需要对其进行追溯性修正，以使开放科学和透明的信息在作者、政策制定者、资助者、期刊、编辑和出版商之间保持一致。</span>同时撤稿通告的勘误表也应更容易地发布，以补充缺失的信息。尽管如此，为了避免对撤稿通告的勘误，最好还是尽可能在撤稿通告中提供尽可能全面的信息。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">撤稿通告中信息的完整性和严谨性决定了它的有用性和透明度。</span>撤稿通告 中缺乏信息，或信息不完整或不准确，对科学家、资助机构、科研机构、期刊、出版商和公众来说，都降低了其在学术用途方面的价值。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">关于撤稿通告的签发者，除了作者、编辑、期刊和出版商外，如果撤稿原因与研究机构和资助者有关，那他们也应该被列为签发者。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">撤稿是漫长而复杂的流程，公众理应知道导致该事件的全部细节。因此，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">撤稿通告需注明与撤稿有关的每个事件的日期，和对官方文件的说明，</span>如大学的调查记录，或是谁报告了这个错误等。这些信息的加入，必将使撤稿通告对学术界和公众更加开放和透明。</p><p class="ql-align-justify" style="border: 0px;
