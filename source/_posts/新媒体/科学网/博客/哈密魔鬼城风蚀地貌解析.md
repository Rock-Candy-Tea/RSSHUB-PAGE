最近十年来我撰写了很多博文介绍新疆的近现代冰川地貌、盐风化地貌、暴雨冲刷地貌等，然而还有一类非常重要的地貌没有介绍，那就是风成地貌，而风成地貌又分成两部分：风蚀地貌和风积地貌。本文重点介绍一下哈密魔鬼城的风蚀地貌。</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun;"><span style="color: blue; font-size: 20px;"> </span><strong><span style="color: blue; font-size: 20px;">   </span></strong></span><strong><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;">新疆连片的风蚀地貌主要分布于相对开阔而平坦的盆地边缘区域，距离山麓也有至少几十千米的距离，大致在洪积扇的外缘地带，其代表地貌就是连片的雅丹地貌。</span></strong></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px;"><span style="font-family: 宋体; color: red; font-weight: bold;">雅丹的中文定义</span><span style="font-family: 宋体; color: red;">：</span></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="color: blue; font-weight: bold; font-size: 20px; font-family: 宋体, SimSun;">    </span><span style="font-size: 20px;"><span style="font-size: 20px; color: blue; font-weight: bold; font-family: 宋体, SimSun;">雅丹</span><span style="font-family: 宋体; color: blue;">就是</span><span style="text-decoration: underline; font-family: 宋体; color: blue; font-weight: bold;">风蚀残丘</span><span style="font-family: 宋体; color: blue;">，是荒漠区成群分布的流线型风蚀土脊，其延长方向与盛行风平行。单个雅丹从不足一米到几十米高，长可达数百米以上。相邻雅丹个体之间平缓的谷地称为</span></span><span style="font-family: 宋体; color: blue; font-weight: bold; font-size: 20px; text-decoration: underline;">雅丹谷</span><span style="font-size: 20px;"><span style="font-family: 宋体; color: blue; font-weight: bold;">（</span><span style="font-family: "Times New Roman"; color: blue;">yardang trough</span><span style="font-family: 宋体; color: blue; font-weight: bold;">）</span><span style="font-family: 宋体; color: blue;">。</span></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体; color: red; font-weight: bold; font-size: 20px;">雅丹的英文定义：</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;">    </span><span style="font-size: 20px; font-family: 宋体, SimSun; color: blue;">我根据英文地质学辞典等资料整理的雅丹定义如下：</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="color: red; font-weight: bold; font-size: 20px; font-family: 宋体, SimSun;">    </span><span style="font-size: 20px;"><span style="font-family: "Times New Roman"; color: red; font-weight: bold;">Yardangs</span><span style="font-family: 宋体; color: blue;">（风蚀土脊，雅丹）</span><span style="font-family: "Times New Roman"; color: blue;">are streamlined parallel ridges, carved by wind erosion, aligned with the direction of a strong prevailing wind.</span></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="color: blue; font-size: 20px; font-family: 宋体, SimSun;">    </span><span style="font-size: 20px;"><span style="font-family: "Times New Roman"; color: blue;"><strong><span style="font-size: 20px; color: blue; font-family: 宋体, SimSun;">雅丹</span></strong>（yardang</span><span style="font-family: 宋体; color: blue;">）就是</span><span style="text-decoration: underline; font-family: 宋体; color: blue; font-weight: bold;">风蚀残丘</span><span style="font-family: 宋体; color: blue;">，是新疆维吾尔语音译成英文后再转译成汉语的翻译词汇。依照发音应该翻译成“雅尔当”才对，而中文采用“雅丹”估计与西北地区口音有关（如：</span><span style="font-family: "Times New Roman"; color: blue;">ing=in, ang=an</span><span style="font-family: 宋体; color: blue;">）。遵从约定俗成的原则，我们只能继续使用“雅丹”一词了。</span></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="color: blue; font-size: 20px; font-family: 宋体, SimSun;">    </span><span style="font-size: 20px;"><span style="font-family: 宋体; color: blue;">据文献介绍，“</span><span style="font-family: "Times New Roman"; color: blue;">yardang”<span style="font-size: 20px; color: blue; font-family: 宋体, SimSun;">的英文名由瑞典探险家斯文赫</span></span><span style="font-family: 宋体; color: blue;">定（</span><span style="font-family: "Times New Roman"; color: blue;">Sven Hedin, 1865.2.19-1952.11.26</span><span style="font-family: 宋体; color: blue;">）</span><span style="font-family: "Times New Roman"; color: blue;">1903</span><span style="font-family: 宋体; color: blue;">年根据维语发音而正式启用，意指“陡峭的小山包”。  </span></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; color: blue;">    雅丹的原始定义就是指“</span><span style="font-size: 20px; text-decoration: underline; color: blue;">风蚀土脊</span><span style="font-size: 20px; color: blue;">”，确切的理解就是“</span><span style="font-size: 20px; text-decoration: underline; color: blue;">风蚀残丘</span><span style="font-size: 20px; color: blue;">”。雅丹并不包括土脊之间的谷地，所以将谷地称为雅丹谷（<span style="font-size: 20px; color: blue; font-family: "times new roman";">yardang trough</span>）。在“雅丹”这个词汇使用前，汉语的相应词汇有“龙城”、“白龙堆”等，就是指在西北地区的风蚀残丘地貌。因雅丹地貌都分布在无人区，远远望去如同毫无生机的城堡，加之风吹带来的听觉加持，充斥着恐怖之感，所以又有魔鬼城之称。</span></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;">    那么，雅丹地貌到底有什么特征呢？如何识别呢？可以考虑如下几点：</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="color: blue; font-size: 20px; font-family: 宋体, SimSun;">    第一、雅丹必须是风蚀作用形成的，其他动力形成的侵蚀地貌不能叫雅丹地貌。如新疆布尔津五彩滩位于额尔齐斯河河畔，过去一直被当作风蚀地貌看待，但其实是暴雨冲刷地貌。再如有些强岩层顺层分布形成突出的残脊，但并非风蚀所致，也不是雅丹地貌。</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="color: blue; font-size: 20px; font-family: 宋体, SimSun;">    第二、雅丹地貌一般分布在相对开阔的荒漠区，主要岩石露头必然以泥岩、粉砂岩等软弱岩性为主，抗风蚀能力差。上述两点也是雅丹“风蚀土脊”本意的体现。<span style="color: #0000FF; font-family: 宋体, SimSun; font-size: 20px;">其他坚硬的岩石（如：花岗岩、火山岩等）难以被风力大量侵蚀，也就无法形成雅丹地貌。</span></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; color: blue;">    第三、完美的雅丹个体平面上应该是典型的流线型土脊，其延长方向与盛行风方向一致，而且其迎风端必然是高耸的侵蚀残丘，而背风端则是缓慢降低的碎屑堆积区。整体看上去向如同逆流而上的鱼形或横躺着的水滴：</span><span style="font-size: 20px; color: red;">迎风钝头，背风拖尾</span><span style="font-size: 20px; color: blue;">。迎风端遭受强烈的风蚀作用，而背风端则成为沉积区，可以保存小型沙堆等（见下图）。</span></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;">    第四、迎风端因受到强风携带的沙粒打击而逐渐变窄，底部掏空凹进，前方因风力受阻而形成强劲的涡流而形成鱼唇形凹槽，凹槽外围为沙丘及掉落的碎岩块。</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;"><img src="https://cors.zfour.workers.dev/?http://bbs.sciencenet.cn/home.php?mod=attachment&filename=%BB%C3%B5%C6%C6%AC21.JPG&id=859974" style="white-space: normal;" referrerpolicy="no-referrer"></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><br></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px;"><span style="font-family: Calibri; color: blue;">         </span><span style="font-family: 宋体; color: blue;">第五、几乎所有的雅丹地貌都不是纯风蚀造成的，而是有其先决条件和侵蚀基础。如：因地壳运动导致岩石被至少两组垂直断裂切割成条块，在地表露头下遭受暴雨和洪水侵蚀，沿着断裂侵蚀严重，逐渐在断裂之间留下侵蚀残丘。考虑到暴雨洪水侵蚀，雅丹地貌总是位于洪积扇下方季节性河流或干河谷两侧的开阔地。在大型洪积扇的中上部（扇根、扇中）风力侵蚀也很强，但因洪积扇中大量砾石的存在导致地表形成一层稳定而抗侵蚀的砾漠（黑戈壁），因而难以形成连片的雅丹地貌。在洪积扇的外缘（离开山麓几十千米）的开阔带，地表砾石明显减少，加之暴雨和洪水侵蚀露出大量的剥蚀残丘，就容易形成连片的雅丹地貌。</span></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;">    当然，除了典型的流线形雅丹外，还有形态各异的实例。然而，无论如何，每个个体必然有迎风端侵蚀、背风端堆积的特征，总体上符合空气动力学原理。</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;"></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体; color: blue; font-size: 20px;">    为了更好地理解风蚀作用，我们先介绍一下风力搬运作用：</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="color: #0000FF; font-family: 宋体, SimSun; font-size: 20px;">    </span><span style="color: blue; font-size: 20px; font-family: 宋体, SimSun;">风力搬运岩石碎屑总体上包括三种方式：底部拖运、低空跳跃、空中悬浮（见下图）。</span></p><p><img src="https://cors.zfour.workers.dev/?http://bbs.sciencenet.cn/home.php?mod=attachment&filename=%BB%C3%B5%C6%C6%AC9.JPG&id=859963" style referrerpolicy="no-referrer"></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;">    第一种：滚动和拖运（rolling and sliding），风吹小鹅卵石沿地表滚动或者滑动，一次强风搬运距离主要以厘米计，虽然对地表也有侵蚀作用，但因距离短，总体侵蚀能力不大。</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;">    第二种：跳跃（saltation），搬运物粒径从细砂到粗砂，有时包括一些厘米级的小砾石，在风的吹动下颗粒如同小球抛在空中飞向下游，落地后再弹起而继续向前飞去。每次跳跃高度一般不超过一米，而移动距离以米为单位，有时达到几十米以上。风最主要的侵蚀介质就是这种碎屑颗粒，其侵蚀高度主要集中在地表以上几十厘米范围内。</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;">    第三种：悬浮（suspension），粉砂和黏土级的细小颗粒被风带到高空，可以形成沙尘暴，搬运距离以千米计，有时达到上千千米以上。这种细碎屑对岩石的侵蚀能力不大。</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; color: blue;">    风蚀的关键是强风携带着砂子和细小的砾石（主体为小于2mm</span><span style="font-size: 20px; color: blue;">的砂粒，强风时包括直径达厘米级的小砾石）对障碍物迎风面（包括水平地面）进行打击和磨砂，逐渐将岩石表面的碎屑磨掉。鉴于跳跃颗粒的运动轨迹高度限制，风蚀最强的位置总是在地表之上几十厘米的位置，特别是二十多厘米处容易形成凹槽。这就是为何雅丹地貌迎风端下部凹进的根本原因。</span></span></p><p style="margin-top: 0;margin-bottom:
