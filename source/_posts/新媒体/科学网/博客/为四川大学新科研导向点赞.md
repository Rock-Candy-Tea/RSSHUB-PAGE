<p style="margin-bottom: 0px; text-indent: 32px; line-height: 200%; background: white;"><span style="font-size: 16px;"><span style="line-height: 200%; font-weight: normal; font-family: "times new roman";">2022</span><span style="font-family: 宋体, SimSun;"><span style="line-height: 200%; font-weight: normal;">年<span style="line-height: 200%; font-weight: normal; font-family: "times new roman";">1</span>月<span style="line-height: 200%; font-weight: normal; font-family: "times new roman";">6</span>日，在四川大学<span style="line-height: 200%; font-weight: normal; font-family: "times new roman";">202</span>1年度<span style="color: #0070C0;"><a href="https://www.sohu.com/a/516078923_121302891" target="_blank"><span style="color: #0070C0;">科研工作总结会</span></a></span>上，李言荣校长</span>强调：</span></span></p><p style="margin-bottom: 0px; text-indent: 32px; line-height: 200%; background: white;"><span style="font-family: 宋体, SimSun; font-size: 16px;">● 不做“二手科研”：基础研究不能再做以跟踪、模仿和附和别人为主的二手科研，应用研究要少做二手科研。要多做以问题导向为主的科研，多做以解决重大核心技术为目的的、有组织的、集成攻关科研！</span></p><p style="text-indent: 32px; line-height: 200%; background: white; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0;"><span style="font-family: 宋体, SimSun; font-size: 16px;">● 以国家科技“四个面向”为导向，围绕学校“五大先导计划”，做真科研、解决真问题、做出真贡献，反对赝科研、似是而非的科研、以论文数量为导向的科研！</span></p><p style="text-indent: 32px; line-height: 200%; background: white; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0;"><span style="font-family: 宋体, SimSun; font-size: 16px;">● 川大作为高水平研究型大学要成为国家战略科技力量，而不是战术科技力量，更不是游击队、支援部队，要成为基础研究的主力军和重大科技突破的生力军！</span></p><p style="margin-bottom: 0px; text-indent: 32px; line-height: 200%; background: white;"><span style="line-height: 200%; font-weight: normal; font-family: 宋体, SimSun; font-size: 16px;">嗯，三个叹号十分给力哦。上述讲话，基本上反映了科研的初心和精髓，体现了四川大学的责任、担当和使命，我不得不为之点赞！</span></p><p style="text-indent: 32px; line-height: 200%; background: rgb(252, 252, 252); vertical-align: middle;"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%; background: white;">得到国家人财物大力支持的科研院所，可称为国家队（国家</span><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%;">战略科技力量</span><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%; background: white;">）。要做当之无愧的国家队，必须担国家之责慷慨前行,当民族之任栉风沐雨</span>；必须做</span><a href="https://blog.sciencenet.cn/blog-575926-1229495.html" style="text-decoration: underline; color: rgb(0, 112, 192); font-family: 宋体, SimSun; font-size: 16px;" target="_blank"><span style="color: #0070C0; font-family: 宋体, SimSun; font-size: 16px;">顶天立地的科研</span></a><span style="font-family: 宋体, SimSun; font-size: 16px;">，即以攻坚克难为目标，以造福民族乃至全人类为宗旨。如此，才能体现自己不可或缺作用。</span></p><p style="text-indent:32px;line-height:200%;background:#FCFCFC;vertical-align:middle"><span style="font-family: 宋体, SimSun; line-height: 200%; background: white; font-size: 16px;">何谓责任？是孟子的“如欲平治天下，当今之世，舍我其谁？”；是诸葛亮的“鞠躬尽瘁，死而后已。”；是范文正公的“以天下为己任。”；是钱学森的“科学无国界，但科学家有自己的祖国。”是啊，只有把生命和崇高的责任联系在一起，才能凸显生命的伟大，历程的壮丽。</span></p><p style="margin-bottom: 0px; text-indent: 32px; line-height: 200%; background: white;"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%; font-weight: normal;">俗语云“是骡子是马牵出来遛遛”。任何国家队说自己取得了某领域的卓越科技成果，得以攻克该领域公认的重大科技难题为标志，以成果的不可估量价值 “一剑封喉”。欲实现此，靠做那些跟风式的热点研究不行，</span><span style="font-family: 宋体, SimSun; font-size: 16px; overflow-wrap: break-word;">靠做那些捡漏补遗的鸡肋研究不行，靠顶刊</span><span style="font-size: 16px; line-height: 200%; font-weight: normal; overflow-wrap: break-word; font-family: "times new roman";">SCI</span><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%; font-weight: normal;"><span style="font-family: 宋体, SimSun; font-size: 16px; overflow-wrap: break-word;">论文为裁判也不行，须靠独辟蹊径开拓新路的长期扎实工作才有望实现。</span>在解决难题的征途中，必然会发展原创理论和颠覆性技术，这也是国家倡导的科技优先发展战略。</span></span></p><p style="text-indent:32px;line-height:200%;background:#FCFCFC;vertical-align:middle"><span style="line-height: 200%; font-family: 宋体; background: white; font-size: 16px;">要展现国家队的责任、担当和使命，需瞄准<span style="line-height: 200%; font-family: 宋体; background: white; font-size: 16px; color: #0070C0;"><a href="https://blog.sciencenet.cn/blog-575926-1314785.html" target="_blank"><span style="line-height: 200%; font-family: 宋体; background: white; font-size: 16px; color: #0070C0;">真问题</span></a></span>，进而解决真问题，由此才能做出真贡献（卓越科技成果）。真贡献不是表面光鲜而内在虚幻的学术泡沫，而是脚踏实地的新发现新发明；真贡献不是夸夸其谈的卖弄资本，而是能真正为科技大厦添砖加瓦；真贡献不像鸡肋一样可随时丢弃，而像宝石一样恒久熠熠发光。</span></p><p style="text-indent:32px;line-height:200%;background:#FCFCFC;vertical-align:middle"><span style="line-height: 200%; font-family: 宋体; background: white; font-size: 16px;">科学研究，尤其是</span><a href="https://blog.sciencenet.cn/blog-575926-1306395.html" target="_blank" style="text-decoration: underline; line-height: 200%; font-family: 宋体; background: white; color: rgb(0, 112, 192); font-size: 16px;"><span style="line-height: 200%; font-family: 宋体; background: white; color: #0070C0; font-size: 16px;">基础科研</span></a><span style="line-height: 200%; font-family: 宋体; background: white; font-size: 16px;">，具有灵感瞬间性、方式随意性、路径不确定性的特点。这就需要科研人员从不同的视角来看待客观事物，以不同于别人的思路来理解客观事物的演变。如此，其中某位科研人员可能会找到解决某个基础难题的“突破口”，进而导致整体性的突破诞生卓越成果。纵观科学发展史，哥白尼、牛顿、爱因斯坦等科学巨匠们，主要是靠个人“灵光闪现”做出的伟大贡献，是靠智慧而非大量经费的成功典范。对某一领域的基础科研而言，人多并不意味着“三个臭皮匠赛过诸葛亮”，而往往由于思维趋于同质化，出现“人多自乱、鸡多不下蛋”的怪象。因此，对基础科研，宜鼓励科研人员或小团队的单打独斗，不宜盲目集群攻关。</span></p><p style="text-indent:32px;line-height:200%;background:#FCFCFC;vertical-align:middle"><span style="line-height: 200%; font-family: 宋体; background: white; font-size: 16px;">然而，技术研发，尤其是重大或关键技术研发，往往涉及众多环节，通常需要一批不同专业的科研人员（大团队）协同攻关。</span></p><p style="text-indent:32px;line-height:200%;background:#FCFCFC;vertical-align:middle"><span style="line-height: 200%; font-family: 宋体; background: white; font-size: 16px;">总之，分清科学与技术，制定科研政策时才能“有的放矢”，才能把“好钢用在刀刃上”。</span></p><p style="text-indent:32px;line-height:200%;background:#FCFCFC;vertical-align:middle"><span style="line-height: 200%; font-family: 宋体; background: white; font-size: 16px;">仅有了正确的科研导向不行，还得把其落到实处才能开花结果，这就需要制订让科研人员静心钻研的相应科研成果评价措施。任何好的措施，应着眼于实绩而非“包装”，应聚焦于“疗效”而非“广告”，即看究竟突破了什么（学术定论/主流共识/思维定式/研究范式/现行做法/权宜之计/学术僵局等），提出了什么新原理/新理论/新方法实现突破的，意义多大。</span></p><p style="text-indent: 32px; line-height: 200%; background: rgb(252, 252, 252); vertical-align: middle;"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white;">但愿我国有更多的科研院所，参考四川大学的</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">新科研导向，制订出有利于科技实质性进步的科研新政，引导科研人员安心做实事、做真事、做大事。只有这样，不管世事如何变迁，不管世界风云如何变幻，我国才能以强盛的科技实力巍然屹立于世界民族之林。</span></span></p>                    <br><br>
