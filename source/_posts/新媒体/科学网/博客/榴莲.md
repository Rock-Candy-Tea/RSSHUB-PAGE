<p><span style="font-family:SimSun">榴莲是一个奇怪而丑陋的水果，固执、坚硬的外壳布满了尖刺，但是打开后，里面是软软的果肉，散发浓浓的香气。有些人对榴莲避而不及，而有些人却对其趋之若鹜。</span>2006<span style="font-family:SimSun">年到</span>2009<span style="font-family:SimSun">年之间的三年里，我是在香港度过的。第一次接触到了众多的热带水果，也是那时候第一次吃榴莲，之后竟恋恋不忘。上个月在</span>Grand Forks <span style="font-family:SimSun">这个小镇发现了价格昂贵的冰冻榴莲果肉，如获至宝。满满的大块入口，浓郁的味道充斥着舌苔上的每一个味觉神经。</span></p><p><span style="font-family:SimSun"><br></span></p><p>2006<span style="font-family:SimSun">年那是我第一次坐飞机，上飞机前，家人给了我很多饺子，以寓意一路平安。我在飞机上，打开行李包，拿出来已经凉的饺子，泰然自若的大快朵颐，引起周围的人侧目。</span>15<span style="font-family: SimSun">年前的香港给与了我极大的震撼。我一直和朋友说，中西方文化在香港完美地融合。在那里可以找到晚清和民国时候的老子招牌，矗立在高楼耸立的街道上，丝毫不显得违和。中华传统在这里很好的继承。</span></p><p><span style="font-family: SimSun"><br></span></p><p><span style="font-family:SimSun">港科大建立在一座山上，整体建筑让人叹为观止，竟然在建筑中融合了中华传统的庭院风格。从山上到山下，做电梯要经过</span>30<span style="font-family: SimSun">多层。经常看到有人在炎炎夏日下，大汗淋漓从山下跑到山上，又从山上跑到山下。对我而言，我更喜欢港科大的那片小小的海滩，因为在山脚下，地点又偏僻，较少有人光顾。坐在海边的岩石上，望着碧绿碧绿的海。高高的堤坝，或站或坐，偶尔又海鸟飞过。旁边是一些小山，有人冒险去</span>hiking<span style="font-family:SimSun">，走出了很多小路，好心人做好了标记，防止后来人迷路。一天同事紧张的说，他</span>hiking<span style="font-family:SimSun">的时候碰到了野猪，不过野猪被他吓跑了，不知道他以后还有没有再去</span>hiking<span style="font-family:SimSun">。</span></p><p><span style="font-family:SimSun"><br></span></p><p><span style="font-family:SimSun">因为是热带，有很多奇怪的动植物，螳螂、会飞的巨大的蟑螂、会两条腿过马路的蜥蜴、会报复的蚊子、喜欢吃生燕麦的硕大的蜗牛、美丽的大蝴蝶、大朵大朵的芍药花。这里的人们是热情而健谈。做车去水厂的路上，带着墨镜和纹身的司机师傅用蹩脚的普通话和我聊天南海北，遇到电台播放《狮子山下》，他会突然放大音量，然后跟着罗文慢慢哼唱“</span><span style="font-family:SimSun;color:#212529;background:white">人生中有欢喜</span> <span style="font-family: SimSun;color:#212529;background:white">难免亦常有泪。我地大家</span> <span style="font-family: SimSun;color:#212529;background:white">在狮子山下相遇上，总算是欢笑多于唏嘘。人生不免崎岖</span> <span style="font-family: SimSun;color:#212529;background:white">难以绝无挂虑，既是同舟在狮子山下且共济。</span> <span style="font-family: SimSun;color:#212529;background:white">抛弃区分</span> <span style="font-family:SimSun;color:#212529;background:white">求共对，放开彼此心中矛盾</span> <span style="font-family: SimSun;color:#212529;background:white">理想一起去追。同舟人世相随</span> <span style="font-family: SimSun;color:#212529;background:white">无畏更无惧，</span> <span style="font-family:SimSun;color:#212529;background:white">同处海角天边</span> <span style="font-family:SimSun;color:#212529;background:white">携手踏平崎岖。我地大家</span> <span style="font-family: SimSun;color:#212529;background:white">用艰辛努力写下那</span> <span style="font-family:SimSun;color:#212529;background:white">不朽香江名句</span><span style="font-family:SimSun">”。短短的一首歌，道出了香港精神：友爱、互助、宽容、诚信、平等、公平、廉洁、勇敢、不放弃。</span></p><p><span style="font-family:SimSun"><br></span></p><p><span style="font-family:SimSun">那里是两个社会，一面是旺角的鱼龙混杂、纸醉金迷和中环的金钱至上和</span>996<span style="font-family:SimSun">，一面是大埔村这样的质朴、安逸的小部落。我喜欢的是大埔村。刚去香港迷路或者找不到地铁口的时候，会有人停下来主动问你需不需要帮助。去彩虹村买菜的时候，热情的大婶、大叔会最后免费多给我一些。电视里播放着</span>30<span style="font-family: SimSun">多年的药品广告，老字号，</span>30<span style="font-family:SimSun">多年诚信品牌不倒。学校里教会有很多团契活动，带领的教授谦卑和儒雅，一群年轻的弟兄姐们聚在一起，分享人生、经历、感恩、教训和心得，一起</span>hiking<span style="font-family:SimSun">。往昔，历历在目；一别，物是人非，天各一方。</span></p><p>80<span style="font-family:SimSun">、</span>90<span style="font-family: SimSun">年代香港电影火爆亚洲。新龙门客栈、东方不败、笑傲江湖、方世玉、周星驰、周润发、大话西游，僵尸片，倩女幽魂，南海十三郎，现在却再难有那么多天马行空、洒脱的电影。上个星期在油管回看了新龙门客栈，回顾经典台词“为这个没名没姓的年头干一杯”，一天后油管又推荐</span>2011<span style="font-family:SimSun">的龙门飞甲，看了</span>5<span style="font-family: SimSun">分钟就看不下去了。</span></p><p><span style="font-family: SimSun"><br></span></p><p><span style="font-family:SimSun">那时候每天中午回到宿舍吃饭，电视台总是放一些老的电视剧。印象比较深的是一部叫做《大地恩情》的电视剧，一共</span>75<span style="font-family: SimSun">集，每天一集，很长，讲述了小人物在清末民初时代大变迁的境遇。关正杰唱的主题曲“</span><span style="font-family:SimSun;color:#212529;background:white">河水弯又弯</span> <span style="font-family: SimSun;color:#212529;background:white">冷然说忧患；别我乡里时</span> <span style="font-family: SimSun;color:#212529;background:white">眼泪一串湿衣衫；人于天地中</span><span style="font-family:'Verdana',sans-serif;color:#212529;background: white">  </span><span style="font-family:SimSun;color:#212529;background:white">似蝼蚁千万</span><span style="font-family:'Verdana',sans-serif;color:#212529;background:white">……</span><span style="font-family:SimSun;color:#212529;background:white">若有轻舟强渡</span> <span style="font-family:SimSun;color:#212529;background:white">有朝必定再返。</span><span style="font-family:SimSun">”</span> <span style="font-family: SimSun">前两三年返回去了香港两次，这座城市变了很多，我们都变了。可不可以停下忙碌的脚步，想一想自己曾经的精神内核，“</span><span style="font-family:SimSun;color:#212529;background:white">人生中有欢喜</span> <span style="font-family: SimSun;color:#212529;background:white">难免亦常有泪。我地大家</span> <span style="font-family: SimSun;color:#212529;background:white">在狮子山下相遇上，总算是欢笑多于唏嘘。人生不免崎岖</span> <span style="font-family: SimSun;color:#212529;background:white">难以绝无挂虑，既是同舟在狮子山下且共济。</span> <span style="font-family: SimSun;color:#212529;background:white">抛弃区分</span> <span style="font-family:SimSun;color:#212529;background:white">求共对，放开彼此心中矛盾</span> <span style="font-family: SimSun;color:#212529;background:white">理想一起去追。同舟人世相随</span> <span style="font-family: SimSun;color:#212529;background:white">无畏更无惧，</span> <span style="font-family:SimSun;color:#212529;background:white">同处海角天边</span> <span style="font-family:SimSun;color:#212529;background:white">携手踏平崎岖。我地大家</span> <span style="font-family: SimSun;color:#212529;background:white">用艰辛努力写下那</span> <span style="font-family:SimSun;color:#212529;background:white">不朽香江名句</span><span style="font-family:SimSun">”。</span></p><p><br></p>                    <br><br>
