white-space: normal; background-color: rgb(255, 255, 255);">学校有很多学生是特定计划的，毕业后要回省里或者县里服务几年，好多次被这些学生问到：老师，反正我们毕业就是要回去教学的，成绩好不好有什么关系？</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 20px; word-break: break-word; color: rgb(64, 64, 64); font-family: -apple-system, BlinkMacSystemFont, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">我的回答是一般都是：优秀是一种习惯。</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 20px; word-break: break-word; color: rgb(64, 64, 64); font-family: -apple-system, BlinkMacSystemFont, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">我没有认真思考过大学的教学对教好小学或中学有什么具体的帮助，肯定是必需的，但是目前的我还讲不清楚，我跟师范生聊过一个观点：老师不是讲知识点，知识点看书就好了，老师要讲清楚的是知识点之间的联系。可是我自己只能算是个”新“教师（虽然奔四的年龄早已不算是青年教师，但是做教师时间真的不算长，讲课的经验更是”年轻“），自己上的课都还没达到自己满意.......</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 20px; word-break: break-word; color: rgb(64, 64, 64); font-family: -apple-system, BlinkMacSystemFont, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">但是我不想从其他老师习惯的那种功利的角度（奖学金、可以违约考研等）去回答这个问题，所以我一般都从“优秀是一种习惯”这一点跟他们聊，凡是做到最好是一种习惯。</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 20px; word-break: break-word; color: rgb(64, 64, 64); font-family: -apple-system, BlinkMacSystemFont, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">如果所有的事都不愿意尽力，都凑合就行，当真的需要尽力、不能凑合的时候，你是做不到的，因为你思维方式已经固化了，而且就算是以后要做中小学教师，为什么就不需要努力了呢？课讲得好的老师和讲得一般的老师之间是有天壤之别的！这一点每一个人都深有感触吗？为什么”毕业后要回去教书“似乎就成了世界的终点了呢？</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 20px; word-break: break-word; color: rgb(64, 64, 64); font-family: -apple-system, BlinkMacSystemFont, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">比如我之前的一个研究生，凡事都是凑合，花60分的努力可以完成的事绝不愿意花61分。毕业不要求六级，就不考六级；考研也是，挑考试科目简单的专业考，比分数线高2分考上的.......但是在我组里，出的成果还算不错，最后说想争取优秀毕业生，我就告诉他如何才能拿到优秀毕业生，首先毕业论文一定要写好，他嘴上答应着，可是毕业论文还是写得很凑合，提了好多建议，修改也是凑合，他凑合了那么多年，已经不知道想把一件事做完美应该如何努力了.......</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 20px; word-break: break-word; color: rgb(64, 64, 64); font-family: -apple-system, BlinkMacSystemFont, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">再比如我自己，这几年一直懒得动脑，凡是需要动脑子的事都凑合，都尽量回避，读书也都是读一些不需动脑的鸡汤书一类的，深刻的书好久没动过了！到不能回避或者不愿意回避的时候，发现脑子已经习惯懒惰了，动不动就罢工.......</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 20px; word-break: break-word; color: rgb(64, 64, 64); font-family: -apple-system, BlinkMacSystemFont, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">由俭入奢易，由奢入俭难啊！</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 20px; word-break: break-word; color: rgb(64, 64, 64); font-family: -apple-system, BlinkMacSystemFont, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">不管做什么事，请把优秀锻炼成一种习惯！</p>                    <br><br>
