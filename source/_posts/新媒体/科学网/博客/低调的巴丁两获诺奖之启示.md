<p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">迄今为止，仅有玛丽亚.居里（居里夫人）、约翰.巴丁、弗雷德里克.桑格三人两获诺贝尔自然科学奖。这三人都具有淡泊低调的行事风格，而约翰.巴丁（以下简称巴丁）尤甚，以至于鲜于被世人知。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">巴丁因</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #333333;">1956</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">年发明晶体管和</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #333333;">1972</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">年建立超导</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #333333;">BCS</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">理论，两次获得诺贝尔物理奖，是诺贝尔奖历史上第一个在同一学科两次获奖的科学家。</span></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">巴丁究竟低调到什么程度呢？请看以下事例：</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif;">1945</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">年</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif;">4</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">月，巴丁加入到贝尔实验室固体物理研究小组（肖克利任组长，组员还有布拉顿），当时晶体管的研制遇到了卡脖子难题而停滞不前。</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919;">巴丁提出了至关重要的“</span><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; font-family: 宋体, SimSun; border: 1px none windowtext; padding: 0px;">半导体表面态理论</span>”解决了此难题，与<span style="font-size: 16px; font-family: 宋体, SimSun; line-height: 200%;">布拉顿合作</span><span style="font-size: 16px; font-family: 宋体, SimSun; line-height: 200%; color: #191919;">开发出世界上第一只</span><span style="font-size: 16px; font-family: 宋体, SimSun; border: 1px none windowtext; padding: 0px;">点接触晶体管。在研制过程中，</span><span style="font-size: 16px; font-family: 宋体, SimSun; line-height: 200%;">肖克利的贡献甚微，当然没被列入专利发明人。为了抢</span></span></span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">功，他背着巴丁和布拉顿，基于点接触晶体管的基本原理独自研发了更先进的结型晶体管。在晶体管成果公布之时，张扬的肖克利极力突出自己的成就，故意冷落原创者巴丁和布拉顿，由此引发了一种怪象：只要提起晶体管，吃瓜群众们第一时间想起的总是肖克利，而不是巴丁或布拉顿。然而，温和淡静的巴丁从不为自己辩解。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919;">在</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #191919;">1957</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919;">年的一次物理学会议上，巴丁让其学生宣布</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #191919;">BSC</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919;">（</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #191919;">B</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919;">代表巴丁，</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #191919;">C</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919;">代表库珀，</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #191919;">S</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919;">代表施里弗：后两者是巴丁的学生）理论，他连会议也不参加。</span></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919;">有人提出过</span><span style="font-size: 16px; font-family: 宋体, SimSun;"><span style="font-size: 16px; border: 1px none windowtext; padding: 0px;">“巴丁数”</span>这一概念，用于形容<span style="font-size: 16px; line-height: 200%; color: #191919;">“</span></span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919;">谦虚程度”；此数越高，表示越谦虚<span style="font-size: 16px; line-height: 200%;">。</span>物理学家巴哈特曾说：“<span style="font-size: 16px; border: 1px none windowtext; padding: 0px;">一般人的巴丁数等于</span></span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919; border: 1px none windowtext; padding: 0px;">1就很不错了，而巴丁的此数为无穷大。</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919;">”</span></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体; color: #191919; font-size: 16px;">某著名文学作家曾想为这位两获诺奖的传奇人物著述立传，却遭到了他的极力反对。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">巴丁夫妻恩爱，家庭和睦，爱好广泛。他喜欢打高尔夫球，有一位几十年的老球友。有一天巴丁对那位老朋友说，下个月我要退休了。那位老兄说，老伙计呀，我们打了几十年球，你能不能告诉我，你是干什么工作的呀。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">巴丁的低调由此可见一斑。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">他本来有跋扈自恣的资本，但一直低调；他本来有傲视群雄的业绩，但一直谦虚。是啊，性格决定命运，优良的性格既是家庭幸福的源泉，也是事业成功的基石。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;"></span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">反观我国学界，某些学者凭跟风式工作戴上了学术帽子而忘乎所以，某些学者凭鸡肋式成果拿到了学术奖励而牛皮哄哄，某些学者凭不值一提的学术成就而自吹自擂。对照下巴丁先生，岂不汗颜！</span><span style="font-size:16px;line-height:200%;font-family: 宋体;color:#333333"> </span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">古语云：“地低成海，人低成王。”确实，在低调的心态支配下，才能专心致志而精益求精，才能淡泊明志而行稳致远，才能胸怀大志而方得始终。</span><br></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%;
