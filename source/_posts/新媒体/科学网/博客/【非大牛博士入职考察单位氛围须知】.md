<p style="margin: 8px 0 16px;text-indent: 37px;background: white"><span style="font-size: 19px;font-family: 宋体">【非大牛博士入职考察单位氛围须知】</span></p><p style="margin: 8px 0 16px;text-indent: 37px;background: white"><span style="font-size: 19px;font-family: 宋体">前面有一篇博文<a href="http://blog.sciencenet.cn/home.php?mod=space&uid=3410526&do=blog&id=1311466" target="_blank">【入职高校须知】</a>讲了一些入职高校前需要注意的事项，今天开一篇讲讲非大牛博士入职考察单位氛围须知，以下须知中的观点仅代表本人，不代表其他人或任何官方机构，不一定对，仅供参考。仅针对非大牛博士，大牛博士不在讨论范围。</span></p><p style="margin: 8px 0 16px;text-indent: 37px;background: white"><span style="font-size: 19px;font-family: 宋体">入职单位氛围很重要。氛围包括应聘单位联系你的人的态度、你将要入职的二级单位的人际氛围以及入职单位教师和学生在教学研究时具备的心态和专业精神。</span></p><p style="margin: 8px 0 16px;text-indent: 37px;background: white"><span style="font-size: 19px;font-family: 宋体">人毕竟是生活在单位中，打交道的时间和范围也以单位内部居多。不同单位，氛围的确差别大，若干年前，笔者就是受不了原先单位的氛围而跳到现单位，从此一别两宽，各生欢喜的。氛围重要，觉察却需要费心思和时间，有没有一些简单的技巧呢，答案是显然的：</span></p><p style="margin: 8px 0 16px;text-indent: 37px;background: white"><span style="font-size: 19px;font-family: 'Arial Unicode MS', sans-serif">1. </span><span style="font-size: 19px;font-family: 宋体">从你投递简历那天开始，<strong>和你联系的单位的人的态度很重要</strong>。如果联系你的人层级低，又对你爱理不理，一副爱来不来的样子，这种情况就要慎重，尽早另做打算。这就像婚嫁，媳妇还没进门就这样，进了门好不了。相反，如果联系你的人层级比较高，比如院长或者人力资源部门的负责人，跟你都是商量的态度，表达的进人愿望很强烈，方方面面都为你考虑到，这至少从一开始就代表人家很重视你，你进门之后，大概率人家不会亏待你。</span></p><p style="margin: 8px 0 16px;text-indent: 37px;background: white"><span style="font-size: 19px;font-family: 'Arial Unicode MS', sans-serif">2. </span><strong><span style="font-size: 19px;font-family: 宋体">亲身体验二级用人单位人际氛围很重要</span></strong><span style="font-size: 19px;font-family: 宋体">。以前单位的招聘一般都是去各大博士培养单位摆摊，或者参加机构统一组织的招聘会，虽然能招到一些人，但其实最后效果都不是很好。所以现在很多单位都通过举办高层次人才论坛来招聘和吸引博士，给路费和住宿，然后有意向的博士前来应聘和洽谈，后来发现这种形式招聘的效率非常高，所以很多高校现在都首选这种方式。为什么这种方式效率高呢？主要原因是这种方式提供了其他方式不能提供的给意向博士亲身体验单位氛围的机会。如果时间允许，建议应抓住这个机会尽量深入到将来的团队或者系去，跟相关的</span><span style="font-size: 19px;font-family: 'Arial Unicode MS', sans-serif">LD</span><span style="font-size: 19px;font-family: 宋体">或者未来的同事多聊聊。一般纯理工的单位人际关系淡漠些，而文科类的单位人文关怀可能会更多些。</span></p><p style="margin: 8px 0 16px;text-indent: 37px;background: white"><span style="font-size: 19px;font-family: 'Arial Unicode MS', sans-serif">3. </span><strong><span style="font-size: 19px;font-family: 宋体">入职单位教师和学生在教学和研究时具备的心态和专业精神很重要</span></strong><span style="font-size: 19px;font-family: 宋体">。如果一个单位教师和学生在教学和研究上都表现的很浮躁很“聪明”，说明这种单位内卷厉害，这种单位建议应该慎重考虑。相反，如果一个单位教师和学生在教学和研究上都表现的很淡定和从容甚至有点“孩童般的纯真天真”，这其实是比较适合非大牛博士的。</span></p><p style="margin: 8px 0 16px;text-indent: 37px;background: white"><span style="font-size: 19px;font-family: 宋体">最后，差点忘了，说了那么多，其实这几方面我单位都很好，主要原因可能是太低调了，知道的人少，欢迎非大牛博士特别是研究计算机、通信、电子、控制等领域理论或算法的博士来我单位应聘，可以先跟我联系。<img
