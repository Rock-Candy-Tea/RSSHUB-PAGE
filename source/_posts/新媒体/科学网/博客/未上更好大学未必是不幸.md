<p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">前不久，家乡的一对父子为读大学的事情请老文</span>“帮个忙”。准大学生羞于开口，其父代言：孩子高中阶段成绩一直不错，正常情况下考个好点的大学没问题，但这次高考没有发挥好，“</span><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">考砸</span>”了，</span><span style="font-family: 等线;font-size: 16px">只能上个一般般的大学，孩子很失望，担心读个不怎么好的大学将来可能找不到好工作。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px">被求助使人进步，因为从科研角度看，有人给你提出了有价值的问题乃至难题。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">只因高考</span>“考砸”了而没能上更好大学的学生应该不少。他们就像奥运赛场上只因临场发挥欠佳而没有得到应得奖牌或名次的运动员一样，天赋、实力、水平等依然是明摆着的。这样的学生如何面对未来的大学生活？</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px">“</span><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">祸兮福所倚，福兮祸所伏。</span>”凡事有利有弊且</span><span style="font-family: 等线;font-size: 16px">可</span><span style="font-family: 等线;font-size: 16px">相互转化</span><span style="font-family: 等线;font-size: 16px">，</span><span style="font-family: 等线;font-size: 16px">读大学应该也不例外。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px">老文考证后</span><span style="font-family: 等线;font-size: 16px">发现</span><span style="font-family: 等线;font-size: 16px">，学生未上更好大学，</span><span style="font-family: 等线;font-size: 16px">原来</span><span style="font-family: 等线;font-size: 16px">未必是不幸，好处也不少。</span><span style="font-family: 等线;font-size: 16px">令人大振的</span><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">是社会心理学中的</span>“大鱼小池塘”（Big-fish-little-pond）效应</span><span style="font-family: 等线;font-size: 16px">，</span><span style="font-family: 等线;font-size: 16px">它</span><span style="font-family: 等线;font-size: 16px">已被大量研究证明与</span><span style="font-family: 等线;font-size: 16px">大学选择有关</span><span style="font-family: 等线;font-size: 16px">。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px">现代社会，比较甚至内卷无处不在。人们都会在长相、健康、学习成绩、工作技能、财富等方面与他人比较，这些比较会影响人们的自信、自尊和自我评价。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px">“大鱼小池塘”效应是一种常见的社会比较效应：人们在一个等级或排名较低的群体（小池塘）中是地位或排名较高的人（大鱼）时的自我感觉，往往比其在一个等级或排名较高的群体（大池塘）中是地位或排名较低的人（小鱼）时好。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">众所周知的例子：在一个相貌平平的群体中，颜值高的人往往有较高的自信；反过来，在一个颜值高的群体中，相貌平平的人往往更容易自卑。前者就是是</span>“大鱼小池塘”效应，后者是“小鱼大池塘”效应。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px">尽管教育的本质不是竞争，但现在的大学就是为了促进竞争而构建的</span><span style="font-family: 等线;font-size: 16px">——</span><span style="font-family: 等线;font-size: 16px">你看重大学排名也参与并促进大学竞争</span><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">。意味着无论你上什么大学，都无法摆脱竞争，</span>“大鱼小池塘”效应难免影响你的大学生活，特别是在你在比较或竞争的方面与他人有差距的情况下。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px">研究表明，一个中等能力的学生在一个充满超高成就者</span><span style="font-family: 等线;font-size: 16px">(</span><span style="font-family: 等线;font-size: 16px">S</span><span style="font-family: 等线;font-size: 16px">uper achievers)</span><span style="font-family: 等线;font-size: 16px">的环境中可能会感觉无能，甚至沮丧；相反，这个学生如果进入一个他或她在其中成绩最出色的环境，他或她可能会觉得自己是一个、甚至开始表现得像一个超高成就者。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">具体说，假设你的真实水平（不</span>“考砸”的情况下的考分）能上排名第50位的大学，你期望如果神助你进入排名更高（例如排名前十）的大学，你将体验到比在前者更高的自尊和自信吗？</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px">你在被录取的那一刻或许会迸发出高度的自尊和自信，因为你的同学羡慕地看着你，你的父母则心里乐开了花。然而，你入学后搞不好将立马经历一段很长时间的抑郁。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">原因是，在后者，你的对照组的同学水平都很高，至少不比你差，你成了</span>“大池塘里的小鱼”。“既生瑜何生亮！”你会感叹你比周瑜更郁闷，因为跟你竞争的不只一个“亮”，而是无数个“亮”。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">反过来，如果你不幸</span>“考砸”了，上了排名较低（例如100位左右）的大学，那么你比你的大多数同学优秀，你成了“小池塘里的大鱼”，你的自尊和自信可能因这个“不幸”而大幅提升。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px">最近，《社会心理与人格科学》杂志（</span><a href="https://journals.sagepub.com/doi/abs/10.1177/1948550620956535" target="_blank"><span style="text-decoration:underline;"><span style="font-family: 等线;color: #0000FF;font-size: 16px">Social Psychological and Personality Science</span></span></a><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">，</span>2021-08）发表一篇论文，探讨了“大鱼小池塘”效应的极端自我-他人比较的形式。文章试图回答：如果一个人不仅是一条大鱼，而且是微池塘(tiny pond)里的一条巨鱼(huge fish)，或者不仅是一条小鱼，而且是巨池塘(huge pond)里的一条微鱼(tiny fish)时，这种效应是否还成立？答案是肯定的。作者把这种极端情况下的“大鱼小池塘”效应叫做“巨鱼微池塘”效应（Huge-fish-tiny-pond effect）。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">这意味着即使你彻底</span>“考砸”了，最终不得不上一所排名垫底的大学，也没什么大不了的。</span><span style="font-family: 等线;font-size: 16px">你</span><span style="font-family: 等线;font-size: 16px">甚至可以体验到</span><span style="font-family: 等线;font-size: 16px">比</span><span style="font-family: 等线;font-size: 16px">“大鱼小池塘”效应</span><span style="font-family: 等线;font-size: 16px">更</span><span style="font-family: 等线;font-size: 16px">暴涨的自尊和自信。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">社会比较中为什么有</span>“大鱼小池塘”甚至“巨鱼微池塘”效应？这是由人的本性决定的。有高人说，人最在意的，往往是身边几个人。这话跟研究发现一致：人往往更注重自己在群体内部的地位或等级，而很少关注所在群体与其他群体的比较。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">说到这儿，老文不由想起一句如雷贯耳的口号：</span>“今天我以学校为荣，明天学校以我为傲。”“大鱼小池塘”效应启示我们，相对于“今天我以学校为荣”，“今天学校以我为荣”对学生的成长可能更有利。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">必须指出，我们说优秀学生未上更好大学也有好处，并不意味着学生为了这些好处应该读更差大学，与天赋、才华、成就等各方面都不如自己的人在一起。只是，</span>“大鱼小池塘”效应确实意味着即使是暂时的地位降低，也会带来士气低落的影响。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px">考大学就像买手机，买之前直至买之后几天，我们可能很在意手机好不好，但很快，我们念念不忘的是带没带手机，不再是手机好不好。带给我们美好的，不是手机，而是透过手机看到的世界。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"><span style="font-family:等线">只因</span>“考砸”而未能读个更好大学的学生，上天调暗了他们的外在光环，但也将补偿他们的内心感觉。如果我们把美好生活作为向往的目标，那么显然，这样的内心感觉就更加弥足珍贵。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 等线;font-size: 16px"> </span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;text-indent: 0;white-space: normal;line-height: 32px"><span style="font-family: 楷体;font-size: 16px">【特别说明，这儿所说的好或差大学，仅指世人（包括大学里的人）关注的排名高或低的大学。学校跟学生一样，本质上是没有好和差之分的。】</span></p><p style="margin: 0px 0px 10px; text-align: justify; font-family: Calibri; font-size: medium; text-indent: 0px;
