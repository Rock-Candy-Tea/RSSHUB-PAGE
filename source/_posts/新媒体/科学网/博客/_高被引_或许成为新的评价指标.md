<p><span style="font-size: 20px; font-family: 宋体, SimSun; text-indent: 32px;"> </span><span style="font-family: 宋体, SimSun; font-size: 20px;">2021年4月22日，</span><span style="font-family: 宋体, SimSun; font-size: 20px; color: #252525;">全球性信息分析公司，</span><span style="font-family: 宋体, SimSun; font-size: 20px;">学术出版业巨头爱思唯尔（Elsevier）正式发布了2020年中国高被引学者（Highly Cited Chinese Researchers）榜单。Elsevier</span><strong style="font-family: 宋体, SimSun; font-size: 20px;"> </strong><strong style="font-family: 宋体, SimSun; font-size: 20px;"><span style="color: #333333; font-weight: normal;">旗下拥有ScienceDirect和Scopus等著名学术数据库。</span></strong><span style="font-family: 宋体, SimSun; font-size: 20px; color: #252525;">2020爱思唯尔“中国高被引学者”榜单以全球权威的引文与索引数据库——Scopus作为中国学者科研成果数据来源，采用软科（上海软科教育信息咨询有限公司）设计的遴选方法，最终得到4023名各学科最具全球影响力的中国学者，</span><span style="font-family: 宋体, SimSun; font-size: 20px;">分别来自373所高校、企业及科研机构。其中高校296所，覆盖了教育部10个学科门类中的84个一级学科。</span><span style="font-family: 宋体, SimSun; font-size: 20px; color: #252525;">中国科学院，清华大学，北京大学，浙江大学，上海交通大学，复旦大学成为大赢家。不少学校以此作为排行榜，人数多的欢欣鼓舞，人数少的</span><span style="font-family: 宋体, SimSun; font-size: 20px; color: #333333;">自惭形秽</span><span style="font-family: 宋体, SimSun; font-size: 20px; color: #252525;">。</span></p><p style="text-align:justify;text-justify:inter-ideograph;line-height:30px"><span style="color: #191919; background: white; font-family: 宋体, SimSun; font-size: 20px;">然而，仔细分析一下高被引学者榜单，会发现一些特点：</span></p><p style="text-align:justify;text-justify:inter-ideograph;line-height:30px"><span style="font-family: 宋体, SimSun; font-size: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #191919;">1.<span style="font-size: 20px; font-family: "Times New Roman";"> </span></span><span style="font-family: 宋体, SimSun; font-size: 20px; color: #191919; background: white;">每所大学都有一些牛人（帽子头衔），但是上榜的人选并非都是这些人。有些默默无闻的年轻人脱颖而出。可谓初生牛犊不怕虎。对人才的判定标准也需要反思了。</span></span></p><p style="text-align:justify;text-justify:inter-ideograph;line-height:30px"><span style="font-family: 宋体, SimSun; font-size: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #252525;">2.<span style="font-size: 20px; font-family: "Times New Roman";"> </span></span><span style="font-family: 宋体, SimSun; font-size: 20px; color: #191919; background: white;">学科分布不均。有些学科人数多，有些学科人数少。或许与学科大小，热门冷门有关。</span></span></p><p style="text-align:justify;text-justify:inter-ideograph;line-height:30px"><span style="font-family: 宋体, SimSun; font-size: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #252525;">3.<span style="font-size: 20px; font-family: "Times New Roman";"> </span></span><span style="font-family: 宋体, SimSun; font-size: 20px; color: #191919; background: white;">与影响因子不完全挂钩。有些入选者发表的论文影响因子并非很高，但一定是主流刊物。影响因子高并不代表引用率高。与研究领域，学科方向，研究内容的关注度有关。</span></span></p><p style="text-align:justify;text-justify:inter-ideograph;line-height:30px"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #191919; background: white;">4. 高被引是每年一评，具有动态波动性。看一个学校的高被引要看其连续性。如同基金一样，每年获批数都不同，有多有少。要看多年总量。</span></p><p style="text-align:justify;text-justify:inter-ideograph;line-height:30px"><span style="color: #191919; background: white; font-family: 宋体, SimSun; font-size: 20px;">因此，在破五唯的形势下，中国高被引学者从纯论文影响力的角度遴选</span><span style="background-color: white; color: #191919; font-family: 宋体, SimSun; font-size: 20px;">学术人才，标准客观统一，有利于视角多元地看待人才，有其重要的参考价值。</span></p><p style="text-align:justify;text-justify:inter-ideograph;line-height:30px"><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; background-color: white; color: #191919;">评价老师的科研没有统一的标准。除了Elsevier，汤森路透的ESI也有高被引。<span style="font-size: 20px; color: #333333; background-color: #FFFFFF;">并不是</span></span></span><span style="background-color: #FFFFFF; color: #333333; font-family: 宋体, SimSun; font-size: 20px;">每篇论文引用次数</span><span style="background-color: #FFFFFF; color: #333333; font-size: 20px; font-family: 宋体, SimSun;">高的就是高引论文。ESI</span><span style="background-color: #FFFFFF; color: #333333; font-size: 20px; font-family: 宋体, SimSun;">由于采用了对每一位作者的贡献都给以统计的方法，即一篇n个作者/机构/国家合作的论文，将被统计n次，因此能很好地表现出每位学者对国家、机构的学术贡献程度，是体现国家/地区、机构国际学术声誉的重要标志。</span></p><p style="text-align:justify;text-justify:inter-ideograph;line-height:30px"><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; background-color: white; color: #191919;">话说回来，任何评价体系都有利弊。高被引也只是一个选项。入选者固然可喜可贺，但决策者千万别又创造出一个新的“</span><span style="font-size: 20px; background-color: #FFFFFF; color: #191919;">唯</span><span style="font-size: 20px; background-color: white; color: #191919;">”。</span></span></p><p><span style="text-indent: 32px; font-size: 20px; font-family: 宋体, SimSun;">  </span></p>                    <br><br>
