<p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-family: 华文宋体; color: #2A2B2E; letter-spacing: 0px; font-size: 18px;">传统观点认为细菌都是非常小的细胞，人类肉眼无法看到。但是现在这种观点需要改变，因为科学家已经明确发现了肉眼可见的细菌。这种丝状细菌发现可能打开了一个窗口，意味着存在更多类似细菌的可能，当然也可能存在真核细胞的不同类型。</span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-family: 华文宋体; color: #2A2B2E; letter-spacing: 0px; font-size: 18px;">在加勒比海的瓜德罗普岛的红树林中，一些奇特的丝状生物潜伏在沉入海底的腐叶上。这些丝状生物，长度可达一厘米，是迄今为止发现的最大的单细胞细菌。它们以氧化硫为生，是已知细菌的50倍。 </span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-size: 18px;"><strong><span style="font-family: 华文宋体; color: #0000FF; letter-spacing: 0px; font-size: 16px;">2009年，</span></strong><span style="color: #2A2B2E; letter-spacing: 0px; font-size: 18px; font-family: 华文宋体;">生物学家奥利维尔·格罗斯在法属西印度群岛的安的列斯大学工作，他在探索瓜德罗普岛的红树林时发现了这种细菌。格罗斯说:“一开始，我以为它是一种真菌之类的东西——不是细菌，可能是真核生物。”不像细菌和古细菌是简单的微生物，真核生物——包括动物和植物——有复杂的细胞，包括细胞核和线粒体这样的细胞器。 </span></span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-family: 华文宋体; color: #2A2B2E; letter-spacing: 0px; font-size: 18px;">当他回到位于瓜德罗普的实验室时，格罗斯在显微镜下检查了他的发现。就在那时，他意识到他看到的不是真核生物——他发现了一些特别的东西。2018年，加州劳伦斯伯克利国家实验室的海洋生物学家让-玛丽·沃兰(Jean-Marie Volland)使用一系列方法更仔细地观察了这些细菌，包括透射电子显微镜和一种称为荧光原位杂交的成像技术。通过这种方式，他帮助确认了它是一个单细胞。两位作者在今年2月的预印本上报告了他们的研究结果，现在已经发表在了《科学》杂志上。 </span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-family: 华文宋体; color: #2A2B2E; letter-spacing: 0px; font-size: 18px;">在硫玛格丽塔细菌家族中还有其他的庞然大物，但第二大的只有大约750微米长。在红树林中也发现了其他丝状细菌，但它们都由数十或数百个细胞组成。“T. magnifica的独特之处在于，整个花丝，是红树林中最长的花丝之一，只有一个细胞，”Volland说。 </span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-family: 华文宋体; color: #2A2B2E; letter-spacing: 0px; font-size: 18px;">细菌的核心是液泡——一种惰性的充满液体的膜。在边缘周围是膜结合的结构，作者称其为pepins，并描述为类似于在真核细胞中发现的细胞器。 </span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-size: 18px;"><span style="font-family: 华文宋体; color: #0000FF; letter-spacing: 0px; font-size: 16px;">巨大的Thiomargarita magnifica</span><span style="color: #2A2B2E; letter-spacing: 0px; font-size: 16px; font-family: 华文宋体;">（华丽硫珠菌）</span><span style="font-family: 华文宋体; color: #2A2B2E; letter-spacing: 0px; font-size: 16px;">是不同寻常的，主要是<span style="font-size: 18px; font-family: 华文宋体;">因为它的大小。在其他细菌中，遗传物质在细胞内自由漂浮，通常只有一条环状染色体。在T. magnifica中，研究小组发现基因信息储存在成千上万的pepin中。其中每一个都含有DNA和核糖体，核糖体是将DNA指令转化为蛋白质的分子机器。这些pepins总共拥有多达</span></span><span style="color: #2A2B2E; letter-spacing: 0px; font-size: 16px; font-family: 华文宋体;">70万个基因组拷贝。 </span></span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><a href="https://www.science.org/doi/epdf/10.1126/science.abb3634" target="_blank">A centimeter-long bacterium with DNA contained in metabolically active, membrane-bound organelles (science.org)</a></p><p class="contentFont" style="padding: 0px 17px; margin: 0.36rem auto -0.133rem; text-size-adjust: 100%; text-align: justify; font-size: 0.19rem; line-height: 0.323rem; top: -0.0665rem; width: 740px; box-sizing: border-box; position: relative; font-family: -apple-system-font, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="color: #2A2B2E; letter-spacing: 0px; font-family: 华文宋体; font-size: 18px;"></span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-size: 18px;"><span style="color: #2A2B2E; letter-spacing: 0px; font-size: 18px; font-family: 华文宋体;">研究人员对内部的DNA进行测序发现，其基因组非常庞大，<strong>有1100万个碱基，其中包含约11000个可清晰区分的基因。通常，细菌基因组平均大约有400万个碱基和大约3900个基因。</strong></span><span style="font-size: 18px; color: #2A2B2E; font-family: 华文宋体; letter-spacing: 0px;">核糖体也位于充满DNA的囊内，这可能使基因编码转化为蛋白质的效率更高。日本九州工业大学计算生物学家竹本和弘表示，这种细菌不仅颠覆了人们关于微生物可变得多么庞大和复杂的观念，还“可能是复杂细胞进化过程中缺失的一环”。</span></span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-size: 18px;"><span style="color: #2A2B2E; letter-spacing: 0px; font-size: 16px; font-family: 华文宋体;">许多问题依然存在。其中包括红树林的特定栖息地，其中含有大量的含硫分子和嗜硫微生物，是否对这种细菌的存在至关重要。而pepin</span><span style="color: #2A2B2E; letter-spacing: 0px; font-size: 18px; font-family: 华文宋体;">s本身也需要更仔细的观察，以确定它们是否都含有相同的遗传物质、核糖体和蛋白质的混合。“我们还没有对单个的pepins进行测序——我们已经对整个细胞进行了测序，其中包含数十万个pepins，”Volland说。特别是，研究人员不知道每个pepin是否只包含一个基因组拷贝，还是不止一个。 </span></span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-family: 华文宋体; color: #2A2B2E; letter-spacing: 0px; font-size: 18px;">既然T. magnifica已经被发现，Gros希望其他团队继续寻找更大的细菌，他说，这些细菌可能隐藏在肉眼可见的地方。密苏里州圣路易斯华盛顿大学的佩特拉·莱文(Petra Levin)说，这一发现挑战了细菌比真核细胞大小限制更低的传统观点。“在某些时候，细胞大小可能会有一个上限，但我不认为这是细菌、古细菌或真核生物特有的。” </span></p><p style="margin-top:0;margin-right:0;margin-bottom:10px;margin-left:0;text-indent:43px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;text-align:justify;text-justify:inter-ideograph;line-height:200%;background:rgb(252,253,254)"><span style="font-family: 华文宋体; color: #2A2B2E; letter-spacing: 0px; font-size: 18px;">莱文说:“我们真的不应该低估进化，因为我们无法猜测它将走向何方。”“我本来不会猜到这个东西的存在，但现在我看到了它，我可以看到进化到这一点的逻辑。”</span></p><p style="margin-left:0;text-indent:43px;text-autospace:ideograph-numeric;line-height:200%"><span style="font-family: 华文宋体; line-height: 200%; font-size: 18px;"> </span></p><p><br></p>                    <br><br>
