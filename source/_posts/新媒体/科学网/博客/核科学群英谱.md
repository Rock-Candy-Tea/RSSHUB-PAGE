<div>   
<p><br></p><p style="text-indent:28px"><span style="font-family:宋体">原子核科学在大众眼中往往不受待见，在日常生活中讨论得不多。现在新闻中科技热点往往是芯片、人工智能、量子信息等，这些概念相对于原子核科学而言要时髦得太多了，在国际竞争中说到我们被“卡脖子”的技术时也很少谈到原子核技术。可能是受益于“两弹一星”前辈们的福荫，<span style="font-family: 宋体; text-indent: 28px;">许多人</span>觉得我们中国原子核科学已经很强了，或者因原子核科学近年来“冒泡”不多而认为原子核科学是传统科学领域了</span>[<span style="font-family:宋体">甚至不再是科学最前沿领域了</span>]<span style="font-family:宋体">。</span></p><p style="text-indent:28px"> </p><p style="text-indent:28px"><span style="font-family:宋体">这两种想法都是错误的，而原子核科学在社会公众中的这种刻板印象不经不仅是片面的，也是对于原子核科学领域的极大误解。原子核科学是一个很大的领域，是一个亟待发展的广阔天地，在中国实现民族伟大复兴过程中必将是非常重要的一个侧面。</span></p><p style="text-indent:28px"> </p><p style="text-indent:28px"><span style="font-family:宋体">为什么这么说呢？我们还是看几个新闻数据，这些数据在网络上都是很容易找到的。一个数据是来源于百度网络文章，说的是芯片产业之恢弘，标题为</span> <span style="font-family:宋体">“</span><span style="font-size: 16px;font-family: Arial, sans-serif">2021</span><span style="font-size: 16px;font-family: 宋体">年全球半导体行业销售额达到创纪录的</span><span style="font-size: 16px;font-family: Arial, sans-serif">5559</span><span style="font-size: 16px;font-family: 宋体">亿美元</span><span style="font-size: 16px;font-family: Arial, sans-serif">,</span><span style="font-size: 16px;font-family: 宋体">同比增加</span><span style="font-size: 16px;font-family: Arial, sans-serif">26.2%</span><span style="font-family:宋体">”</span>[<a href="https://www.21ic.com/a/920913.html" target="_blank">https://www.21ic.com/a/920913.html</a>  ]<span style="font-family:宋体">，这个数据确实很大。我们再看另一个新闻数字，来自于高能所柴之芳院士的一个报告，这个报告在网络上也很容易找到，这里我给二个链接：</span><a href="https://www.cnnpn.cn/article/25313.html" target="_blank">https://</a><a href="https://www.cnnpn.cn/article/25313.html" target="_blank">www.cnnpn.cn/article/25313.html</a> <span style="font-family:宋体">或</span> <a href="https://m.thepaper.cn/baijiahao_13817454" target="_blank">https://m.thepaper.cn/baijiahao_13817454</a> <span style="font-family:宋体">。在这个报告中有几个数据值得注意：在美国</span>20<span style="font-family:宋体">多万亿美元的</span>GDP<span style="font-family:宋体">中，核技术产业规模占</span> 4-5%[<span style="font-family:宋体">即在美国每年核技术的产值</span> 8000 <span style="font-family:宋体">亿美元！</span>]<span style="font-family:宋体">、欧洲日本大约占</span> 2-3%<span style="font-family:宋体">，我国核技术年产值超</span> 3000 <span style="font-family:宋体">亿元，近年均增长率超</span>20%<span style="font-family:宋体">。我们的社会公众平时很少有关于核科学的音讯，因此这些数据就显得很突兀和震撼的了，从这些数据也立即看出核技术其实与社会生活实际上息息相关，而我们在核技术方面奋起直追是很紧迫的。</span></p><p style="text-indent:28px"> </p><p style="text-indent:28px"><span style="font-family:宋体">原子核科学技术应用之广泛、之神奇不是这个科普系列的主题。原子核科学技术之所以能走向生产生活其实离不开基础研究</span>[<span style="font-family:宋体">原子核物理</span>]<span style="font-family:宋体">，就像电子时代的技术进步离不开量子力学、电磁理论等基础科学一样。在上个世纪有数十名原子核物理学家被授予诺贝尔物理学奖，还有部分原子核物理学家学者被授予了诺贝尔化学奖。现在原子核科学依然是人类的知识前沿领域，例如前年《科学》杂志征集的</span>125<span style="font-family:宋体">个科学问题中至少有三个问题是当今原子核物理的热点课题，如致密星体的本质、宇宙中重元素的合成机制、超重核素稳定岛。这些问题的重要意义不言自明，原子核科学在人类面向星辰大海而求知的征程中举足轻重，原子核科学家们任重而道远。</span></p><p style="text-indent:28px"> </p><p style="text-indent:28px"><span style="font-family:宋体">因为饮水思源，我希望花时间找资料回看一下核科学领域的群英谱。任何一个领域都是人做出来的，催生出这么大的领域、这么大的市场的基础研究过程中，有许多不能忘却的瞬间和英雄人物，正所谓时势造英雄，英雄在历史进程中起很大作用。通过这些群英谱，我们不仅部分地勾勒出原子核物理的部分历史风貌，对于预期原子核的未来发展可能也是很有裨益的。</span></p><p style="text-indent:28px"> </p><p style="text-indent:28px"><span style="font-family:宋体">人类首次从实验上接触到核现象是</span>1896<span style="font-family:宋体">年贝克勒尔</span> (Antoine Henri Becquerel)<span style="font-family:宋体">发现天然放射性报告给法国科学院</span>(<span style="font-family:宋体">在更早的</span>1858<span style="font-family:宋体">年就有人发现了同样现象并向法国科学院报告</span>,  <span style="font-family:宋体">但是没有人对此予以足够关注</span>, 1898<span style="font-family:宋体">年居里夫妇</span> (Pierre Curie & Marie Sklodowska Curie) <span style="font-family:宋体">发现了发射性元素</span> <span style="font-family:宋体">钋和镭</span>, 1909 - 1911 <span style="font-family:宋体">年</span> <span style="font-family:宋体">卢瑟福</span> (Ernest Rutherford) <span style="font-family:宋体">通过粒子对金箔的散射实验表明原子的核心非常小而致密</span>,  <span style="font-family:宋体">据此提出原子的行星模型</span>. 1919<span style="font-family:宋体">年卢瑟福用</span>alpha <span style="font-family:宋体">粒子撞击氮原子核、得到氧原子核</span>, <span style="font-family:宋体">实现人工核嬗变。</span></p><p style="text-indent:28px"> </p><p style="text-indent:28px"><span style="font-family:宋体">在二十年代哈金斯等</span>(William Draper Harkins)<span style="font-family:宋体">对于原子核的内部组元做了许多系统性的猜测</span>. 1928<span style="font-family:宋体">年伽莫夫</span> (George Gamow)<span style="font-family:宋体">、古尼</span> (Ronald Wilfred Gurney) <span style="font-family:宋体">与康顿</span> (Edward Uhler Condon) <span style="font-family:宋体">提出利用量子隧道贯穿效应解释</span> alpha <span style="font-family:宋体">粒子衰变现象</span>. 1930<span style="font-family:宋体">年伽莫夫提出了原子核的液滴模型</span>. 1932 <span style="font-family:宋体">年</span> <span style="font-family:宋体">查德威克</span> (Sir James Chadwick) <span style="font-family:宋体">发现中子</span>, <span style="font-family:宋体">宣告原子核物理的开端</span>; <span style="font-family:宋体">同年</span>, <span style="font-family:宋体">伊万宁科</span> (Dimitri Ivanenko) <span style="font-family:宋体">、海森堡</span>(Werner Heisenberg) <span style="font-family:宋体">提出原子核是由质子和中子组成的</span>, <span style="font-family:宋体">核物理及其与其它学科的交叉迅速成为人类科学研究的前沿</span>.</p><p style="text-indent:28px"> </p><p style="text-indent:28px">1935<span style="font-family:宋体">年海森堡、</span> <span style="font-family:宋体">外兹扎克</span>(Carl Friedrich von Weizsacker)<span style="font-family:宋体">、</span> <span style="font-family:宋体">玻尔</span>(Niels Henrik David Bohr) <span style="font-family:宋体">和惠勒</span>(John Archibald Wheeler)<span style="font-family:宋体">等人发展了液滴模型理论</span>, 1938 <span style="font-family:宋体">年</span> <span style="font-family: 宋体">哈恩</span> (Otto Hahn) <span style="font-family:宋体">用慢中子轰击铀原子核时实现了重核的裂变</span>, <span style="font-family:宋体">使人类利用原子能成为现实</span>; <span style="font-family:宋体">同年</span>, <span style="font-family:宋体">贝特</span> (Hans Bethe) <span style="font-family:宋体">提出恒星能源来源于轻核聚变反应</span> (<span style="font-family:宋体">碳氮氧循环</span>)<span style="font-family:宋体">的理论</span>. 1940 <span style="font-family:宋体">年</span> <span style="font-family:宋体">西博格</span> (G. T. Glenn Theodore Seaborg) <span style="font-family: 宋体">和</span> <span style="font-family:宋体">麦克米伦</span> (Edwin Mattison McMillan) <span style="font-family:宋体">发现了超铀元素镎和钚</span> (<span style="font-family:宋体">超重元素合成仍是当今科学前沿之一</span>). <span style="font-family:宋体">在</span>40<span style="font-family:宋体">年代伽莫夫等人提出了大爆炸中元素合成的理论</span>, <span style="font-family:宋体">核反应理论与宇宙和天体的演化密切联系起来</span>, <span style="font-family:宋体">逐步发展为核天文物理学</span> (Nuclear Astrophysics). <span style="font-family:宋体">在低能原子核领域</span>, 1949<span style="font-family:宋体">年梅逸</span>(Maria Goeppert-Mayer) <span style="font-family:宋体">和简森</span>(Jensen Johannes Hans Daniel) <span style="font-family:宋体">等建立壳模型理论</span>, 1953<span style="font-family:宋体">年玻尔</span>(Aage Bohr)<span style="font-family:宋体">、莫特逊</span>(Ben Roy Mottelson)<span style="font-family:宋体">和瑞因沃特</span>(Leo James Rainwater)<span style="font-family:宋体">建立集体运动理论。在</span>1974 <span style="font-family:宋体">年有马朗人</span>(Akito Arima)<span style="font-family:宋体">和雅开罗</span>(Fransisco Iachello)<span style="font-family:宋体">建立的玻色子模型等</span>, <span style="font-family:宋体">这些理论已经成为研究原子核低激发态的基础性框架。</span></p><p style="text-indent:28px"> </p><p style="text-indent:28px"><span style="font-family:宋体">下面我罗列一下迄今为止核科学领域相关的诺贝尔奖</span>(<span style="font-family:宋体">括号内部是得奖原因</span>)<span style="font-family:宋体">。</span></p><p style="text-indent:28px"> </p><p style="text-indent:28px">1903<span style="font-family:宋体">年</span> Henri Becquerel <span style="font-family:宋体">和</span> <span style="font-family:宋体">居里夫妇</span>(<span style="font-family:宋体">放射性的发现和研究</span>);  1935 <span style="font-family:宋体">年</span> James Chadwick (<span style="font-family:宋体">发现中子</span>); 1938 <span style="font-family:宋体">年</span> Enrico Fermi (<span style="font-family:宋体">中子辐照产生新的放射性元素</span>); 1939 <span style="font-family:宋体">年</span> Ernest Orlando Lawrence (<span style="font-family:宋体">发展回旋加速器和合成新元素</span>); 1944<span style="font-family:宋体">年</span> Otto Stern (<span style="font-family:宋体">发明分子束方法和</span> <span style="font-family:宋体">发现质子磁矩</span>) <span style="font-family:宋体">和</span> Isidor Rabi (<span style="font-family:宋体">发展分子束方法研究原子和原子核磁矩</span>); 1948 <span style="font-family:宋体">年</span> Patrick (<span style="font-family:宋体">改进云室方法研究原子核和宇宙辐射</span>); 1949 <span style="font-family:宋体">年</span> <span style="font-family:宋体">汤川秀树</span> (Hideki Yukawa, <span style="font-family:宋体">预言</span> pi <span style="font-family:宋体">介子</span>); 1950 <span style="font-family:宋体">年</span> Cecil Powell (<span style="font-family:宋体">发明核乳胶并发现新介子</span>); 1951 <span style="font-family:宋体">年</span> John Douglas Cockcroft <span style="font-family:宋体">和</span>Ernest Thomas Sinton Walton (<span style="font-family:宋体">实现人工核帨变</span>); 1952<span style="font-family:宋体">年</span> Felix Bloch  <span style="font-family:宋体">和</span> Edward Purcell  (<span style="font-family: 宋体">发明核磁共振</span>); 1961 <span style="font-family:宋体">年</span> Robert Hofstadter (<span style="font-family:宋体">电子散射法研究核子结构</span>) <span style="font-family:宋体">和</span> Rudolf Mossbauer (<span style="font-family:宋体">发明原子核辐射的无反冲共振吸收方法</span>); 1963 <span style="font-family:宋体">年</span> Maria Goeppert-Mayer<span style="font-family:宋体">、</span>Johannes Hans Daniel Jensen (<span style="font-family:宋体">发展壳模型理论</span>) <span style="font-family:宋体">和</span> Eugene Paul Wigner (<span style="font-family:宋体">研究原子核结构对称性</span>); 1967<span style="font-family:宋体">年</span> Hans A. Bethe (<span style="font-family:宋体">研究核反应理论和恒星能量来源</span>); 1975<span style="font-family:宋体">年</span> Aage Bohr <span style="font-family:宋体">和</span> Ben Mottelson (<span style="font-family:宋体">研究原子核集体运动</span>)<span style="font-family:宋体">、</span>Leo Rainwater (<span style="font-family:宋体">提出原子核形变</span>); 1983<span style="font-family:宋体">年</span> Subrahmanyan Chandrasekhar (<span style="font-family:宋体">恒星结构</span>)<span style="font-family:宋体">、</span> Willian Alfred Fowler (<span style="font-family:宋体">宇宙化学元素的合成</span>); 1994 <span style="font-family:宋体">年</span> Bertram Neville Brockhouse (<span style="font-family:宋体">发展中子频谱学</span>) <span style="font-family:宋体">和</span> Clifford Shull (<span style="font-family:宋体">发展中子衍射技术</span>). <span style="font-family:宋体">除了以上物理学奖</span>, <span style="font-family:宋体">还有几个化学奖也是核物理学家</span>, <span style="font-family:宋体">包括</span> 1908 <span style="font-family:宋体">年的</span> Ernest Rutherford (<span style="font-family:宋体">元素分离和放射性物质的研究</span>); 1911 <span style="font-family:宋体">年居里夫人</span> (Marie Sklodowska-Curie, <span style="font-family:宋体">发现镭和钚</span>); 1921 <span style="font-family:宋体">年</span> Frederick Soddy (<span style="font-family:宋体">研究放射性物质化学研究和同位素</span>); 1922<span style="font-family:宋体">年</span> Francis William Aston (<span style="font-family:宋体">发现很多非放射性同位素</span>); 1935 <span style="font-family:宋体">年</span> Jean Frederic Joliot-Curie <span style="font-family:宋体">和</span> Iren Joliot-Curie (<span style="font-family:宋体">合成新放射性同位素</span>); 1943 <span style="font-family:宋体">年</span> Hevesy Gyorgy (<span style="font-family:宋体">化学反应中用同位素作为示踪物</span>); 1944<span style="font-family:宋体">年</span> Otto Hahn (<span style="font-family:宋体">发现重原子核裂变现象</span>); 1951 <span style="font-family:宋体">年</span> Kurt Alder <span style="font-family:宋体">和</span> Edwin Mattison McMillan (<span style="font-family:宋体">发现超铀元素</span>); 1960 <span style="font-family:宋体">年</span> Willard Frank Libby (<span style="font-family:宋体">使用</span>C-14<span style="font-family:宋体">同位素测量年代</span>);
