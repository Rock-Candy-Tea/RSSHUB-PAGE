<p><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202208/17/101816nzsvfuur6z55o6fl.png" title alt="Banner-900x383-JAT.png" referrerpolicy="no-referrer"><br></p><p><br></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">近期，我们邀请到中山大学智能工程学院的副教授由林麟博士向我们介绍其在<em>Journal of Advanced Transportation</em>创建的特刊“Autonomous Transportation Systems Enabled by Emerging Technologies”（新兴技术支持下的自主式交通系统）。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">作为该特刊的客座主编，由教授在本篇采访中向我们介绍了自主式交通领域的最新进展，并分享了对准备向特刊投稿的作者的建议。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"><strong>Q：是什么让您选择“新兴技术支持下的自主式交通系统”（Autonomous Transportation Systems Enabled by Emerging Technologies）作为您的特刊主题？</strong></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">A：在物联网（IoT）、人工智能（AI）、联邦学习等新兴技术的推动下，当代交通系统正从智能化、自动化逐步向自主化转变，逐步构建新一代的自主式交通系统（Autonomous Transportation System, ATS）：基于“自主感知、自主学习、自主决策、自主响应”的运作流程，在不断降低人为的主观干预的同时，有序提升系统的自组织运行与自主化服务能力，实现个性化出行需求与多模式交通供给的长效平衡。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">因为自主式交通系统的前瞻性，研究其关键挑战、赋能技术、基础平台以及核心应用具有重要的理论意义和实践价值。首先，随着新兴技术的逐步推广，如何构建高兼容、可拓展的服务系统，以支撑系统的阶段式升级成为关键挑战；其次，随着各类智能设备的大量部署与使用，如何实现各端闲散资源的整合与应用，以提高系统的感知与计算能力成为关键挑战；再者，随着大众对数据安全和用户隐私的日益关注，如何实现个体级数据的隐私化处理与个性化分析，成为提升系统服务质量与用户体验的关键挑战；最后，随着系统建设的复杂性的不断攀升，如何构建可模拟验证与优化提升的信息物理平台，以保证ATS的有序、合理建设成为关键挑战。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">鉴于此，我们选择“新兴技术支持下的自主式交通系统”（Autonomous Transportation Systems Enabled by Emerging Technologies）作为特刊主题，旨在吸引国内外学者进一步研究与探索ATS，解决ATS面临的挑战，共同推动ATS的发展。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"><strong>Q：请向我们介绍一下您在智能交通领域的研究背景。</strong></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"><strong> </strong></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">我曾就职于麻省理工学院智能交通实验室、未来城市交通交叉课题研究组，与Moshe Ben-Akiva教授合作，对智能交通系统进行了深入的研究。为解决多设备协同感知、多源异构数据融合分析等关键问题，提出未来交通感知理论，并以理论为基础研发了应用平台，实现对个体出行与货物运输数据的动态采集与分析，用以支撑更加高效的城市出行与物流货运服务。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">2020年，我通过“百人计划”，入职中山大学智能工程学院，担任副教授，继续智慧城市、智能交通系统理论与方法的研究、系统与平台的研发。以物联网、联邦计算为基础，通过融合如人工智能、高速无线通讯等新兴技术，研究支撑智慧城市及智能交通系统自主运行的联邦融合架构，实现异质对象、异构数据、多样功能的解耦与服务化集成，并以此为基础，研究自主式交通系统的信息物理平台与自主化出行服务。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"><strong>Q：近几年里，您注意到该研究领域出现了哪些重大进展？</strong></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">近年来，随着各种新兴技术的快速发展、以及各类交通需求的不断涌现，交通系统的自组织运行与自主化服务能力亟需提升。然而，现行的智能交通系统对人工智能、大数据、5G 通信、北斗卫星导航、物联网、边缘计算、联邦学习等新兴技术的适应性较弱，很难有效支撑容纳多样需求与多类供给的新一代交通复杂系统。因此，研究如何合理化构建自主式交通系统，引导系统的阶段式有序提升成为关键问题。为此，需要对ATS的组成进行解构，分析以外部技术与内部需求为驱导的ATS演化过程，研究ATS架构在不同阶段的自适应构建方法，并研发支持系统运行状态仿真与优化的信息物理平台；同时，面向ATS典型应用场景，进行相关的应用示范。从理论方法、技术核心到平台应用，全面推动新一代交通系统的发展与建设。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"><strong>Q：您对准备向该特刊投稿的作者有什么建议？</strong><br></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">一般而言，建议的课题包括：新兴技术背景下的ATS基础理论研究与ATS典型场景应用。具体而言，我们欢迎原创研究和综述论文，内容包括但不限于：</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">ATS设计原则、服务架构与场景应用</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">ATS架构语义化和自适应管理策略与机制</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">ATS数据安全和用户隐私保护方法与技术</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">ATS使能的人工智能、边缘计算、联邦学习等新兴技术</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">ATS信息物理系统的设计与实现</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"><strong>Q： 您认为担任该特刊的客座编辑会对研究人员带来什么好处？</strong><br></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;">作为该特刊的客座编辑，能够有机会深入地了解ATS的发展现状与热点趋势，紧跟全球同行的研究步伐。在此基础上，也能够更好的推动ATS的研究，梳理ATS的发展方向。此外，可以更好的与来自不同学科的资深研究人员进行交流，通过广泛且深入的讨论，推动智能交通与其他基础学科的交叉融合发展。最后，可以第一时间处理与自身专业及兴趣相关的稿件，深入的了解编辑流程，为相关研究人员提供更好的服务。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> </span></p><p style="text-align: center;"><span style="font-family: 微软雅黑, "Microsoft YaHei"; color: #000000;"> <strong>关于特刊</strong></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei";
