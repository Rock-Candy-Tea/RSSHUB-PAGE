<p><span style="color: #333333; text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">编制是指组织机构的设</span><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">置及其人员数量的定额和职务的分配，由</span><a target="_blank" href="https://baike.so.com/doc/5716851-5929577.html" style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px; color: rgb(0, 0, 0); text-decoration-line: none;">财政拨款</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">的编制数额由各级机构</span><a target="_blank" href="https://baike.so.com/doc/466929-494422.html" style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px; color: rgb(0, 0, 0); text-decoration-line: none;">编制管理</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">部门确定，各级组织人事部门根据编制调配人员，财政部门据此拨款。编制通常分为</span><a target="_blank" href="https://baike.so.com/doc/5761306-5974069.html" style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px; color: rgb(0, 0, 0); text-decoration-line: none;">行政编制</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">、</span><a target="_blank" href="https://baike.so.com/doc/1081841-1144811.html" style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px; color: rgb(0, 0, 0); text-decoration-line: none;">事业编制</a><span style="font-family: 宋体, SimSun; font-size: 20px;">等</span><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">。公务员使用行政编制。一般的事业单位工作人员大体分为</span><a target="_blank" href="https://baike.so.com/doc/643389-681068.html" style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px; color: rgb(0, 0, 0); text-decoration-line: none;">在编</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">和</span><span style="color: #333333; text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">非在编人员。在编人员指取得机构编制管理部门的人员编制手续人员，非在编人员就是无机构编制管理部门的人员编制手续的人员，如:本单位自行外聘的人员，就象企业聘用一样，由单位自行规定其待遇，付给其工资，与地方财政没有关系，这些工作人员通常称为临时工。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;">现在高校的年轻人（博士后）大多是没有编制的。3年非升即走成了心病。能否留在学校进入编制是年轻人的最大心愿。由于多数高校编制已满，很难保证博士后出站时都留下。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;">对于已有编制人员则意味着铁饭碗，可以享受五险一金和退休待遇。有编制的</span><span style="background-color: #FFFFFF; color: #333333; font-family: 宋体, SimSun; font-size: 20px; text-indent: 28px;">专业技术岗是根据行业的要求进行职称评定。而无编制者的职称离开学校并不通用。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;">近年高校扩充师资队伍，但是编制有限。一些外籍和博士后的教师因编制问题造成困扰，努力工作还“军心不稳”，而有编制的则可以“高枕无忧”躺平。这种状况不利于教师队伍</span><span style="font-family: 宋体, SimSun; font-size: 20px;">稳定</span><span style="font-family: 宋体, SimSun; font-size: 20px;">和学科的发展。年轻人想要获得编制，一方面面对考核压力，工作业绩必须优秀，另一方面还要看学校所在院系的指标。有编制的即便有考核也够不成实际压力，即便降职，还有铁饭碗。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;">高校人事制度改革应该废除编制，将有无编制的双轨制变为真正全员聘任制。能者为上才能充分调动老师们的积极性，促进学校良性运转。</span></p>                    <br><br>
