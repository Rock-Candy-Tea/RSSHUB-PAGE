padding: 0px;">为什么期刊选择很重要</span></p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">选择适合你的研究的期刊比看起来要复杂得多。对于某些类型的研究，一个更具体的专业期刊可能是最好的，而对于其他类型的研究，接触广泛的受众是关键。此外，你还想在符合你的科学价值和诚信原则的期刊上发表文章。</p><p style="border: 0px; white-space: normal; background-color: rgb(255, 255, 255);">投稿被拒绝是出版过程的一部分，但与此同时，你也不想花几个月的时间向一个又一个期刊提交同一篇文章。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">通过仔细地将你的文章与特定的期刊匹配，你可以减少发表所需的投稿次数，节省自己的时间，让你的稿子更快地发表。</span></p><p class="ql-align-center" style="border: 0px; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">1 <span style="font-weight: 700;">确定需要从预投期刊中获取什么</span></p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">首先考虑一下你的研究，以及如何才能最好地接触到它的目标读者。带着你的期望，花点时间回答以下问题:</p><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p>研究主题是什么?</p></li><li><p>文章是专业的还是跨学科的?</p></li><li><p>文章是一个渐进的发现，还是一个潜在的重大突破?</p></li><li><p>目标读者是谁?文章是否拥有广泛的潜在读者，还是只对某个研究领域的专家感兴趣?读者将如何发现你的研究?</p></li><li><p>你的资助机构对出版是否有任何特殊要求(例如，一些资助机构可能规定作者只能在开放获取许可证下出版研究成果)?</p></li><li><p>打算如何分享你的研究成果?对你来说，选择发表预印本是否重要？</p></li><li><p>期刊是否坚持出版伦理?</p></li><li><p>你是否已准备好满足期刊对许可、发表数据和伦理披露的要求?</p></li><li><p>是否有出版费预算?</p></li><li><p>是否有自动转投便利渠道？</p></li></ul><ul style="list-style-type: none;" class=" list-paddingleft-2"><li><p><br></p></li></ul><p class="ql-align-center" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; display: block; height: auto;" referrerpolicy="no-referrer"></p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">2 <span style="font-weight: 700;">选择潜在的预投期刊</span></p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">如果你正准备提交一篇研究论文，你可能已经在这个领域工作了好几年，并且已经有一些你最喜欢的期刊。尽管如此，花点时间回顾一下所有选择，看看有什么新发现。因为期刊也在不断地发展和调整它们的收稿范围，新的期刊也在不断地出现。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">以下是一些发现新出版机会的简单方法：</p><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">在线搜索。</span>看看你所在领域正在崛起的期刊，试着搜索过去1-2年发表的相关文章。要找到扩展到你的领域的期刊，搜索你的主题领域的论文。查看出版商的网站，了解更多关于你的文章如何适合杂志。(期刊博客也是一个很好的信息来源!)</p></li><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">口碑。</span>与同事、实验伙伴、导师或合作者进行沟通。在你的学科中，他们认为令人兴奋的期刊是什么?</p></li><li><p><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">选择期刊的在线工具。</span>许多网站跟踪期刊的指标和表现，包括速度、接受率和遵守开放科学政策等因素。</p></li></ul><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">3 <span style="font-weight: 700;">决定某本杂志是否适合你的特定研究</span></p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">接下来，根据需求，与你的共同作者一起评估你列表上的期刊。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">哪些期刊最适合你的目标读者、主题、专业水平和出版费要求？</span>如果你不确定是否适合，请查看杂志网站，看看你能否回答以下问题:</p><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p>哪些主题属于杂志的范围?</p></li><li><p>编委会成员有哪些?</p></li><li><p>期刊是否在你所在领域的标准数据库中被索引?</p></li></ul><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">还要考虑:</p><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p>杂志收APC吗，多少钱？</p></li><li><p>是否有任何额外的费用(例如，彩色图表或额外的页面)？</p></li><li><p>杂志是否提供你所写的特定类型的文章？例如，他们是否发表 meta-analysis, negative results, review articles, 或protocols?</p></li></ul><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">避免掠夺性期刊</span></p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">掠夺性期刊仅仅为了金钱利益而接受文章，并且不提供真正的同行评审过程。发现掠夺性的期刊是很困难的，但是有一些基本规则可以帮助你开始。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">请看一下杂志的网站：</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">同行评审过程描述清楚吗?</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">编委会名单列出来了吗?</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">期刊是否被PubMed或Web of Science等数据库编入索引?</p><p style="border: 0px; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">鼓励去做的：</span></p><p class="ql-align-center" style="border: 0px; margin-top: 0.63em; display: block; height: auto;" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2" style="list-style-type: disc;"><li><p>仔细选择你的第一和第二选择的期刊，考虑到你的读者和资助者的需求，以及你所写的文章的类型，期刊的范围和要求</p></li><li><p>要警惕潜在的掠夺性期刊，它们收取费用却不提供可靠的同行评议</p></li><li><p>与你的共同作者讨论你的需求和优先级，并就你的投稿选择达成共识</p></li></ul><ul style="list-style-type: none;" class=" list-paddingleft-2"><li><p><br></p></li></ul><p style="border: 0px;
