<p style="margin-bottom: 0px; text-indent: 32px; background: white; line-height: 2em;"><a href="https://zhidao.baidu.com/question/539990029.html" style="text-decoration: underline; font-family: 宋体, SimSun; font-size: 16px;" target="_blank"><span style="font-family: 宋体, SimSun; font-size: 16px;">阿基米德</span></a><span style="color: #333333; font-family: 宋体, SimSun; font-size: 16px;">曾经说过这样的豪言壮语：“给我一个支点，我就能撬动整个地球。”这不是一句口号，而是一种信仰，虽然并不存在这样一个支点，但他却形象说明了杠杆原理的物理内涵。</span></p><p style="margin-bottom: 0px; text-indent: 32px; background: white; line-height: 2em;"><span style="color: #333333; font-family: 宋体, SimSun; font-size: 16px;">对这句话的寓意，不同的人可能有不同的解释。以我的愚见，这句话可引申为：只要找到四两拨千斤的破局点（支点），再难的事儿都可用简单的高招儿解决。我读过《<a href="https://baike.baidu.com/item/%E5%BA%96%E4%B8%81%E8%A7%A3%E7%89%9B/1317010?fr=aladdin" target="_blank">庖丁解牛</a>》的故事，</span><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; font-family: 宋体, SimSun; color: #333333;">解牛本来是一件非常辛苦的事情，但在庖丁眼里变成了一种艺术享受，因为他完全掌握了解牛的规律——照着关节处下手</span>，从而以炉火纯青之技法达到了事半功倍的效果<span style="font-size: 16px; font-family: 宋体, SimSun; color: #333333;">。</span></span></p><p style="margin-bottom: 0px; text-indent: 32px; background: white; line-height: 2em;"><span style="font-size: 16px; font-family: 宋体, SimSun; color: #333333;">要突破某一学科领域的科研僵局，需探寻破局点，找打了破局点相当于明确了突破口，</span><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F;">在持续的努力下</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F; font-weight: normal;">会</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F;">像滚雪球一样扩大战果。这</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F; font-weight: normal;">正如</span><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">推多米诺骨牌一样，只要第一块骨牌倒下了，后续更大的骨牌会依次倒下。</span></span></p><p style="margin-bottom: 0px; text-indent: 32px; background: white; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">然而，难就难在如何找到破局点。对任何业已存在的科学难题，前人已做了大量工作，几乎已经穷尽了能想到的各种路数。这意味着沿前人的路数前行几无出路，需另辟蹊径才能走出窘境。要开拓新路，得有好想法，要产生好想法，就要围绕现象的发源地，以大胆质疑和大胆想象的理念进行大胆假设，但大胆假设要遵循理性和逻辑自洽原则，以免误入陷阱。例如，在前期锁固型斜坡研究启发下，我们研究地震之初，想到构造地震的主要发源地是断层，应首先弄清什么样的断层能发生较大地震。断层内物质不外乎分三种情况：（</span><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #1F1F1F;">1</span><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">）全是软弱介质；（</span><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #1F1F1F;">2</span><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">）全是高强介质；（</span><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #1F1F1F;">3</span><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">）软弱介质段和高强介质段沿断层间隔分布。当时，我们做了如下简单推理：对第一种情况，由于不可能积累较高能量故不可能发生较大地震；对第二种情况，断层难以运动发震；对第三种情况，</span></span><span style="font-size: 16px;"><span style="font-family: 宋体, SimSun; color: #1F1F1F;">高强段（锁固段）能积累较高能量且断层可运动，具备发生较大地震的条件。这样，我们就找到了破局点（支点）——锁固段及其破裂行为，</span><span style="font-family: 宋体, SimSun; color: #1F1F1F;">接着又找到了合适的“杠杆”——建立锁固段峰值强度点与体积膨胀点的力学联系，给此“杠杆”持续加力——不懈穷究，从而撬动了地震——走上了地震研究的康庄大道。嗯，“广告”插播完毕，该切换到正剧啦，呵呵。</span></span></p><p style="margin-bottom: 0px; text-indent: 32px; background: white; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">一旦某个想法经小心求证无误后，即好想法形成后，不能固步自封，要继续深度思考以产生更多的好想法。若这些好想法是关联的、协调的，则学术思想即可形成。判断学术思想是否靠谱的主要依据是：（</span><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #1F1F1F;">1</span><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">）是否与机理密切相关且已得到初步实证；（</span><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #1F1F1F;">2</span><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">）涉及的逻辑结构是否极其简单。如果答案均为</span><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #1F1F1F;">yes</span><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">，那就恭喜您啦。</span></span></p><p style="margin-bottom: 0px; text-indent: 32px; background: white; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">在学术思想形成过程中，需要注意：（</span><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #1F1F1F;">1</span><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">）善于凝练和解决关键科学问题。解决了关键问题（纲），往往能顺带解决次要问题（目），正所谓纲举目张嘛。（</span><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #1F1F1F;">2</span><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">）一定要简化、简化、再简化逻辑结构，简化到极致时，不仅学术思想具有极强的科学性，而且更容易被别人理解接受。（</span><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #1F1F1F;">3</span><span style="font-size: 16px; font-family: 宋体, SimSun; color: #1F1F1F;">）善于走出去虚心听取别人的不同意见，集思广益，以利于减少思维漏洞，从而少走弯路。</span></span></p><p style="margin-bottom: 0px; text-indent: 32px; background: white; line-height: 2em;"><span style="color: #1F1F1F; font-family: 宋体, SimSun; font-size: 16px;">在正确的学术思想指引下，建立的理论应具有描述某一类事物演化的普适性。显然，根据该普适性理论，便能可靠地预知该事物的未来演化，从而为人类造福。这不仅是科研成果的顶峰，而且足以撬动某些地球人顽固的旧思想桎梏，从而推动科学的大幅进展。</span></p><p style="margin-bottom: 0px; text-indent: 32px; background: white; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F;">不同学科领域的科研人员，可参考本文拙见</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F; font-weight: normal;">以便于尽快</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F;">找到破局点（支点）</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F; font-weight: normal;">，从而</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F;">突破</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F; font-weight: normal;">自己研究领域的</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F;">科研僵局</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F; font-weight: normal;">，一通百通做出卓越成果</span><span style="font-size: 16px; font-family: 宋体; color: #1F1F1F;">。</span></span></p><p style="margin-bottom: 0px;
