text-align: center;"><span style="font-size: 20px; font-family: 黑体, SimHei; color: #0070C0;">牢记初心，再创佳绩</span></p><p style="text-align:center"><span style="font-size: 18px; color: #0070C0;"><span style="font-size: 18px; font-family: Calibri, sans-serif;">——</span>祝贺《人类学学报》创刊<strong><span style="font-size: 18px; font-family: "Times New Roman", serif;">40</span></strong>周年</span></p><p style="text-align:justify;text-justify:inter-ideograph;text-indent:29px"><span style="font-size: 18px;"><span style="font-family: "Times New Roman", serif; color: #221E1F;">1982</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">年</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">8</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">月，吴汝康、吴新智两位先生创办《人类学学报》，至今已经走过了</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">40</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">年的历程！</span></span></p><p style="text-align:justify;text-justify:inter-ideograph;text-indent: 28px;line-height:normal"><span style="font-size: 18px;"><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">作为以人类自身为研究对象的科学，人类学在中国的发展已经有</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">100</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">余年的历史；其内容涉及人类、人类祖先以及人类近亲的灵长类的生物学、演化、行为、文化等特征或规律；相关研究人员分散在不同的高校和研究机构。因周口店发掘与研究建立的中国科学院古脊椎动物与古人类研究所（古脊椎所），是中国古人类学及整个古脊椎动物演化研究的中心。与人类起源与演化相关的古人类化石研究、旧石器考古研究以及相关的体质人类学及灵长类研究一直是古脊椎所的重要研究方向。</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">20</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">世纪</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">50</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">年代，古脊椎所创办了《古脊椎动物与古人类》学术刊物，刊登与古脊椎动物及人类演化相关的发现和研究成果。随着相关学科，尤其是古人类学、体质人类学、灵长类学、旧石器考古学等的发展壮大，一本学术期刊已经不能满足日益增长的研究成果的刊登需求，因此吴汝康和吴新智两位先生在</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">1982</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">年</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">8</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">月创办了《人类学学报》学术期刊。这是中国第一本人类演化领域的专业学术期刊。《人类学学报》在发刊词中确立了明确的办刊宗旨和</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">6</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">个方面的刊发内容，主要涉及与人类演化相关的古人类化石、旧石器考古、古人类年代与环境、灵长类学、体质人类学与应用人类学等。</span></span></p><p style="text-align:center"><span style="font-size: 18px;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202208/12/171437nv9c0cfc2n4cufz4.jpg" title alt referrerpolicy="no-referrer"> </span></p><p style="text-align:justify;text-justify:inter-ideograph;text-indent: 28px;line-height:normal"><span style="font-size: 18px;"><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">四十年来，《人类学学报》聚焦人类起源与演化及相关学科研究的办刊宗旨，刊发了</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">2000</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">余篇古人类学、旧石器考古学、体质人类学、灵长类学等学科的重要发现和研究成果，包括和县人头骨化石、禄丰古猿化石研究，以及</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">“</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">连续进化附带杂交学说</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">”</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">等重要的人类演化理论。《人类学学报》为这些学科成果展示及学术交流提供了一个重要平台，见证了改革开放以来中国人类演化及相关学科研究的发展与壮大。</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">2020</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">年，《人类学学报》新一届编委会成立。时值良好的外部环境，《人类学学报》进入了新的发展时期。为适应新媒体时代的需要，《人类学学报》新增了一些办刊策略，如建立了微信公众号，建立了独立的学报主页与采编系统等。随着这几年收稿数量的增加，《人类学学报》从</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">2021</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">年开始也由季刊改版为双月刊，为中国的人类演化及相关学科成果发布和学术交流搭建了更为宽广的平台。</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">2021</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">年</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">4</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">月，《人类学学报》正式入选《中文社会科学引文索引》（</span><span style="font-family: "Times New Roman", serif; color: #221E1F;">CSSCI</span><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">）数据库。</span></span></p><p style="text-align:justify;text-justify:inter-ideograph;text-indent: 28px;line-height:normal"><span style="font-family: "宋体e眠副浡渀.", serif; color: #221E1F; font-size: 18px;">作为《人类学学报》的主办单位，古脊椎所将一如既往地支持《人类学学报》的工作，希望这本期刊不断地发展并壮大，不断助力中国与世界人类演化研究及相关学科的发展。</span></p><p><span style="font-size: 18px;"><span style="font-size: 18px; font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">  正值《人类学学报》创刊</span><span style="font-size: 18px; font-family: "Times New Roman", serif; color: #221E1F;">40</span><span style="font-size: 18px; font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">周年之际，我谨代表中国科学院古脊椎动物与古人类研究所各位同仁对为《人类学学报》建立、发展和壮大而付出心血的吴汝康院士、吴新智院士表示崇高敬意，对《人类学学报》新老编委会成员、审稿专家、编辑部工作人员以及广大作者和读者表示衷心感谢！让我们一起庆贺《人类学学报》在过去</span><span style="font-size: 18px; font-family: "Times New Roman", serif; color: #221E1F;">40</span><span style="font-size: 18px; font-family: "宋体e眠副浡渀.", serif; color: #221E1F;">年里所取得的成就，也希望我们编辑部及相关主创人员，牢记先辈们创刊的初心，创出更多佳绩！</span></span></p><p><br></p>                    <br><br>
