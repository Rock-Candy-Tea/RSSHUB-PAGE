MS';color: #333333;letter-spacing: 0;font-size: 29px">WHO</span></strong><strong><span style="font-family: 'Arial Unicode MS';color: #333333;letter-spacing: 0;font-size: 29px">狂犬病预防控制指南：最新更新选介（</span></strong><strong><span style="font-family: SimSun;color: #333333;letter-spacing: 0;font-size: 29px"><span style="font-family:Arial Unicode MS">6</span></span></strong><strong><span style="font-family: 'Arial Unicode MS';color: #333333;letter-spacing: 0;font-size: 29px">）</span></strong></h1><p style="margin-top:16px;margin-right:0;margin-bottom:16px;margin-left:0;text-indent:0;padding:0 0 0 0 ;background:rgb(255,255,255)"><span style="font-family: Tahoma;color: #333333;letter-spacing: 0;font-size: 14px"> </span></p><p style="margin-top:16px;margin-right:0;margin-bottom:16px;margin-left:0;text-indent:0;padding:0 0 0 0 ;line-height:28px;background:rgb(255,255,255)"><strong><span style="font-family: 'Microsoft YaHei UI';color: #069A2E;letter-spacing: 0;font-size: 24px">前言</span></strong><strong><span style="font-family: 'Microsoft YaHei UI';color: #069A2E;letter-spacing: 0;font-size: 24px"><span style="font-family:Microsoft YaHei UI">：今年７月</span><span style="font-family:Microsoft YaHei UI">29 日，WHO官网发布了一个英文手册，标题是Guide to introducing human rabies vaccine into national immunization programmes（将人狂犬病疫苗引入国家免疫规划指南）（见参考文献），在其正文和附录中全面介绍了ＷＨＯ对狂犬病疫苗的使用规范和对相关问题的权威解答。本博客将陆续选介其中部分内容以飨读者。</span></span></strong></p><p style="margin-top:16px;margin-right:0;margin-bottom:16px;margin-left:0;text-indent:0;padding:0 0 0 0 ;line-height:28px;background:rgb(255,255,255)"><strong><span style="font-family: 'Microsoft YaHei UI';color: #069A2E;letter-spacing: 0;font-size: 24px"><span style="font-family:Microsoft YaHei UI"><br></span></span></strong></p><p style="margin-top:16px;margin-right:0;margin-bottom:16px;margin-left:0;text-indent:0;padding:0 0 0 0 ;line-height:28px;background:rgb(255,255,255)"><strong><span style="font-family: SimSun;color: #0000FF;letter-spacing: 0;font-size: 29px">狂犬病的风险评估</span></strong></p><p style="margin-left:0;text-indent:43px;text-autospace:ideograph-numeric;line-height:200%"><span style=";font-family:SimSun;line-height:200%;font-size:21px">在有可靠的以社区和实验室为基础的动物狂犬病监测的国家，处置动物咬伤受害者或疑似狂犬病暴露的卫生保健工作者应接受培训，以便</span><strong><span style="font-family: SimSun;line-height: 200%;color: #FF0000;font-size: 21px">在接种狂犬病疫苗之前进行风险评估。这可以防止不必要地使用狂犬病疫苗</span></strong><span style=";font-family:SimSun;line-height:200%;font-size:21px"><span style="font-family:SimSun">。风险评估包括</span><span style="font-family:Calibri">:</span></span></p><p style="margin-left:0;text-indent:43px;text-autospace:ideograph-numeric;line-height:200%"><span style=";font-family:SimSun;line-height:200%;font-size:21px"><span style="font-family:SimSun">•由训练有素、技术熟练的兽医或公共卫生官员对肇事动物进行临床评估</span><span style="font-family:Calibri">;</span></span></p><p style="margin-left:0;text-indent:43px;text-autospace:ideograph-numeric;line-height:200%"><span style=";font-family:SimSun;line-height:200%;font-size:21px">•评估暴露的类别和位置。</span></p><p style="text-indent:48px;text-autospace:ideograph-numeric;line-height:200%"><strong><span style="font-family: SimSun;line-height: 200%;font-size: 24px">肇事动物的临床评估</span></strong><strong><span style="font-family: SimSun;line-height: 200%;font-size: 24px">：</span></strong></p><p style="text-indent:48px;text-autospace:ideograph-numeric;line-height:200%"><span style=";font-family:SimSun;line-height:200%;font-size:21px"><span style="font-family:SimSun">在犬类狂犬病负担严重的国家，当咬人的动物符合疑似或可能是狂犬病病例的定义</span><span style="font-family:Calibri">(</span><span style="font-family:SimSun">附录</span><span style="font-family:Calibri">1)</span><span style="font-family:SimSun">时，应立即开始</span><a name="OLE_LINK1" href="http://blog.sciencenet.cn/undefined"></a></span><strong><span style="text-decoration:underline;"><span style="font-family: SimSun;line-height: 200%;font-size: 21px"><span style="font-family:SimSun">暴露后预防</span><span style="font-family:Calibri">(</span><a name="OLE_LINK2" href="http://blog.sciencenet.cn/undefined"></a><span style="font-family:Calibri">PEP</span><span style="font-family:Calibri">)</span></span></span></strong><span style=";font-family:SimSun;line-height:200%;font-size:21px">，而不是等待实验室检测结果。在确定是否需要进行</span><strong><span style="text-decoration:underline;"><span style="font-family: SimSun;line-height: 200%;font-size: 21px">暴露后预防（</span></span></strong><strong><span style="text-decoration:underline;"><span style="font-family: SimSun;line-height: 200%;font-size: 21px"><span style="font-family:Calibri">PEP</span><span style="font-family:SimSun">）</span></span></span></strong><span style="font-family: SimSun;line-height: 200%;font-size: 21px">时</span><span style=";font-family:SimSun;line-height:200%;font-size:21px">，对不能进行评估或检测的动物应被假定是患有狂犬病。然而，在暴露于</span><span style="text-decoration:underline;"><span style="font-family: SimSun;line-height: 200%;font-size: 21px">啮齿类（鼠类</span></span><span style=";font-family:SimSun;line-height:200%;font-size:21px">）动物、</span><span style="text-decoration:underline;"><span style="font-family: SimSun;line-height: 200%;font-size: 21px">兔子和野兔</span></span><span style=";font-family:SimSun;line-height:200%;font-size:21px"><span style="font-family:SimSun">等小型哺乳动物后，很少需要进行</span><span style="font-family:Calibri">PEP</span><span style="font-family:SimSun">，因为这些动物极少会发展为狂犬病。如果实验室检测显示动物没有狂犬病，则可停止接种疫苗。</span></span><span style="text-decoration:underline;"><span style="font-family: SimSun;line-height: 200%;font-size: 21px">在停止接种前至少已完成两次疫苗接种的人应被视为已完成了暴露前预防接种</span></span><span style=";font-family:SimSun;line-height:200%;font-size:21px">。</span></p><p style="text-indent:48px;text-autospace:ideograph-numeric;line-height:200%"><strong><span style="font-family: SimSun;line-height: 200%;font-size: 24px">评估暴露的类别和位置</span></strong><strong><span style="font-family: SimSun;line-height: 200%;font-size: 24px">：</span></strong></p><p style="text-indent:48px;text-autospace:ideograph-numeric;line-height:200%"><span style=";font-family:SimSun;line-height:200%;font-size:21px"><span style="font-family:SimSun">即使情况被评估为低风险，如果存在其他风险因素</span><span style="font-family:Calibri">(</span><span style="font-family:SimSun">例如咬伤头部、颈部或其他神经高度密集分布的部位</span><span style="font-family:Calibri">;</span><span style="font-family:SimSun">幼童被咬伤</span><span style="font-family:Calibri">;</span><span style="font-family:SimSun">多处或很深的伤口</span><span style="font-family:Calibri">)</span><span style="font-family:SimSun">，也应当立即启动</span></span><strong><span style="font-family: SimSun;line-height: 200%;font-size: 21px"><span style="font-family:Calibri">PEP</span></span></strong><span style=";font-family:SimSun;line-height:200%;font-size:21px">。</span></p><p style="margin-left:0;text-indent:43px;text-autospace:ideograph-numeric;line-height:200%"><span style=";font-family:SimSun;line-height:200%;font-size:21px"> </span></p><p style="text-autospace:ideograph-numeric;line-height:200%"><span style="color: #0070C0;"><strong><span style="line-height: 200%; font-size: 32px; font-family: SimSun;"><strong style="color: rgb(0, 112, 192); white-space: normal;"><span style="font-family: SimSun; line-height: 64px; font-size: 32px;"><strong><span style="font-family: SimSun; line-height: 58px; color: #0000FF; letter-spacing: 0px; font-size: 29px;">推迟启动PEP（暴露后预防）</span></strong></span></strong></span></strong></span><span style="color: #0070C0;"><strong><span style="line-height: 200%; font-size: 32px; font-family: SimSun;"> </span></strong></span></p><p style="margin-left:0;text-indent:43px;text-autospace:ideograph-numeric;line-height:200%"><span style=";font-family:SimSun;line-height:200%;font-size:21px"><span style="font-family:SimSun">如果督察和评估的结果证明，以社区和实验室为基础的动物狂犬病的监测是强有力的、可靠的和灵敏的，那么只有在这样的条件下，才可以考虑在进行检测或兽医评估的同时，有限地推迟接种狂犬病疫苗。在这种情况下，同时是在没有其他风险因素的情况下，如果咬人的家畜</span><span style="font-family:Calibri">(</span><span style="font-family:SimSun">如狗、猫或雪貂</span><span style="font-family:Calibri">)</span><span style="font-family:SimSun">经受过训练的专业人员评估是健康的，而且在被咬之日起的</span><span style="font-family:Calibri">10</span><span style="font-family:SimSun">天观察期内保持健康，那么被这样的家畜咬伤后可以推迟开始</span><span style="font-family:Calibri">PEP</span><span style="font-family:SimSun">。</span></span></p><p style="margin-left:0;text-indent:43px;text-autospace:ideograph-numeric;line-height:200%"><span style=";font-family:SimSun;line-height:200%;font-size:21px">根据一名受过培训的专业人员的说法，正如相关的指南所述，</span><strong><span style="font-family: SimSun;line-height: 200%;color: #FF0000;font-size: 21px"><span style="font-family:SimSun">对于一只据观察明显健康的家养动物，如果该动物在</span><span style="font-family:Calibri">10</span><span style="font-family:SimSun">天观察期限结束时仍然健康，也可据此而停止接种狂犬病疫苗。</span></span></strong></p><p style="margin-left:0;text-indent:43px;text-autospace:ideograph-numeric;line-height:200%"><span style=";font-family:SimSun;line-height:200%;font-size:21px"> </span></p><p style="text-autospace:ideograph-numeric;line-height:200%"><span style="color: #0070C0;"><strong><span style="font-family: SimSun; line-height: 200%; font-size: 32px;"><strong style="white-space: normal;"><span style="line-height: 200%; font-family: SimSun; color: #0000FF; letter-spacing: 0px; font-size: 29px;">回顾性风险评估</span></strong></span></strong></span></p><p style="margin-left:0;text-indent:43px;text-autospace:ideograph-numeric;line-height:200%"><span style=";font-family:SimSun;line-height:200%;font-size:21px"><span style="font-family:SimSun">如果某只动物被确定是疑似、可能或确诊的狂犬病病例，应进行回顾性风险评估，以确定在该动物的感染期间</span><span style="font-family:Calibri">(</span><span style="font-family:SimSun">即从狗、猫或家貂出现临床症状前</span><span style="font-family:Calibri">10</span><span style="font-family:SimSun">天开始的一段时间</span><span style="font-family:Calibri">)</span><span style="font-family:SimSun">至动物死亡之前可能与该动物有过接触的任何人。这些人甚至可以在暴露数月后从疫苗接种中受益。</span></span></p><p style="margin-top:16px;margin-right:0;margin-bottom:16px;margin-left:0;text-indent:43px;padding:0 0 0 0
