font-size: 16px; font-family: "trebuchet ms"; color: #222222;">(3) </span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">彩图转换为黑白图像，通过以上方式，选择灰度模式即可。</span></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 0px; padding: 0px; color: rgb(51, 51, 51); font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; background-color: rgb(255, 255, 255); text-indent: 56px;"><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms";"></span></strong></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 4px; padding: 0px; color: rgb(51, 51, 51); font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; background-color: rgb(255, 255, 255);"><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">9.<span style="overflow-wrap: break-word; font-weight: normal; font-size: 9px; line-height: normal; font-family: "times new roman";">     </span></span></strong><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: 黑体; color: #222222;">图片是中文版软件处理的或通过中文版软件生成的，导致图片上有中文怎么办？</span></strong></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 1.12em; padding: 0px; color: rgb(51, 51, 51); font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; background-color: white;"><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">答：</span></strong><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">很多软件都有中文版，但图片上的中文肯定是作者添加的，既然能添加就能删除，或改成英文。有的是分析仪器的中文操作系统，界面上有中文，作者无法删除，这时可以将原始数据导出，通过数据处理软件重新作图，这个过程也可避免中文。实在不行，可以用图片编辑软件，如</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">Photoshop</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">将中文删除，加上英文。但如果非要这样操作，说明图片有可能是截图，或原始的可修改图片文件不在了，这时如果图片分辨率符合要求，就可以这么做。如果分辨率不足，那么主要问题就不是中英文的问题了。</span></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 1.12em; padding: 0px; color: rgb(51, 51, 51); font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; background-color: white;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;"></span></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 4px; padding: 0px; color: rgb(51, 51, 51); font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; background-color: rgb(255, 255, 255);"><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">10.<span style="overflow-wrap: break-word; font-weight: normal; font-size: 9px; line-height: normal; font-family: "times new roman";"> </span></span></strong><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: 黑体; color: #222222;">通过</span></strong><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">Photoshop</span></strong><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: 黑体; color: #222222;">拼图，保存为</span></strong><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">TIFF</span></strong><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: 黑体; color: #222222;">格式的图片超过</span></strong><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">30M</span></strong><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: 黑体; color: #222222;">，不方便投稿怎么办？</span></strong></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 1.12em; padding: 0px; color: rgb(51, 51, 51); font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; background-color: white;"><strong style="overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">答：</span></strong><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">通过</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">Photoshop</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">拼图，一般是先创建一个画板，再将插图的各部分排列于画板上，调位置对齐、添加标记、确认分辨率、，最后存储为</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">TIFF</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">图片。需要注意的是，</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">TIFF</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">格式图片允许保留图层信息，这是导致图片变大的主要原因。在另存为</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">TIFF</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">图片的对话框中，存储选项勾选</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">“</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">作为副本</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">”</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">，去勾选</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">“</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">图层</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">”</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">，并选择</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">LZW</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">压缩方式（无损压缩，保证分辨率的情况下显著降低图片大小），选择这两个参数之后，一般的拼图大小不会超过</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: "trebuchet ms"; color: #222222;">10M</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: 宋体; color: #222222;">，如若不然，适当降低分辨率即可；如果选择</span><span style="overflow-wrap: break-word;
