<p><span style="font-family: 宋体, SimSun; font-size: 20px;"></span></p><p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left: 0;background:white"><span style="font-size: 20px; color: #333333; font-family: 宋体, SimSun;">研究生学制是研究生培养的基本制度，与研究生培养目标和培养质量密切相关。</span></p><p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left: 0;background:white"><span style="font-size: 20px; color: #333333; font-family: 宋体, SimSun;">通常研究生年限为3年。然而现在的实际情况是，研究生又要修课程，又要做实验，又要发高水平文章，还要写论文，找工作。少数硕士生不能按期毕业，多数博士生4-6年方能毕业，导致导师和学生的压力与日俱增，这种情况越来越普遍。一些学校将博士生正常毕业年限延至4年甚至更长。硕士延长至3-4年，教育硕士由2年延至3年。</span></p><p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left: 0;background:white"><span style="font-size: 20px; color: #333333; font-family: 宋体, SimSun;">延期毕业虽然可以在保证研究生培养质量方面有所改善，但是在校时间过长，无疑增加了培养成本。无论是投入，住宿，奖学金，生活补贴对于学校和导师的压力更大。</span></p><p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left: 0;background:white"><span style="font-size: 20px; color: #333333; font-family: 宋体, SimSun;">一些研究生由于各方面原因，表现欠佳，甚至达不到研究生水准是导致毕业延期的原因之一。当然对发表文章的要求也越来越高。一些学校对博士生文章的要求需要达到一定影响因子，大大提升了毕业门槛。</span></p><p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left: 0;background:white"><span style="font-size: 20px; color: #333333; font-family: 宋体, SimSun;">延期毕业，对于就业率影响很大。博士生3年毕业就业率只有20-30%左右。4年就业率仅50-60%。延期助长了个别研究生混日子和拖延行为，不利于研究生集中精力尽快完成学业。</span></p><p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left: 0;background:white"><span style="font-size: 20px; color: #333333; font-family: 宋体, SimSun;">说实话，延期毕业的博士多数是论文进展不顺，导师和学生都有问题。如果在4年内拿不到学位，再延期的意义不大，而且浪费更多的资源。</span></p><p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left: 0;background:white"><span style="font-size: 20px; color: #333333; font-family: 宋体, SimSun;">研究生淘汰制势在必行。一些学校将8年未获得学位的研究生取消学位是个严肃的措施。读研的风险在加大。</span></p><p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left: 0;background:white"><span style="font-size: 20px; color: #333333; font-family: 宋体, SimSun;">作为导师应该从入学起就要负起责任，每个环节都要加强。对于博士生导师资格应该强化和规范，中期考核要落到实处。个别导师放羊式培养，甚至长期脱离岗位，这样的导师应该禁止招收研究生，以免误人子弟。</span></p><p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left: 0;background:white"><span style="font-size: 20px; color: #333333; font-family: 宋体, SimSun;">作为研究生要端正读研动机，加强个人能力。对自己是否适合读研应有自我评估，而不是硬撑着。</span></p><p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left: 0;background:white"><span style="font-size: 20px; color: #333333; font-family: 宋体, SimSun;">作为高校和研究所，管理方面也有待加强。淘汰制必须落实，以减少不必要的资源浪费。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;"></span><br></p><p><br></p>                    <br><br>
