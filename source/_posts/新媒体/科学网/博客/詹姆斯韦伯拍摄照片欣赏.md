<p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-size: 20px;">美<span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">国当地时间7月11日，美国总统拜登发布了<span style="font-size: 20px; font-family: 宋体, SimSun; text-align: -webkit-center; background-color: #F8F9F4;">詹姆斯·韦布</span>空间望远镜拍摄到的第一张照片：星系团SMACS 0723的照片</span></span></p><p style="margin-top: 20px; margin-bottom: 20px;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202207/13/102455xkiwb3bebgfkee7w.jpg" title alt="main_image_deep_field_smacs0723-5mb.jpg" referrerpolicy="no-referrer"><br></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">从这张照片可以看到目前宇宙最遥远的位置星系的清晰图像。由于SMACS 0723星系团的巨大质量，导致其背后的其他星系图像被弯曲，形成引力透镜效应。当然仅仅是可见物质还是难以产生这么强大的引力效应的。这也证明该星系团中存在大量的暗物质。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;"><span style="font-size: 20px; font-family: 宋体, SimSun; text-align: -webkit-center; background-color: #F8F9F4;">詹姆斯·韦布</span>空间望远镜主要拍摄的是红外光线的图像。跟哈珀太空望远镜不同，<span style="font-size: 20px; font-family: 宋体, SimSun; text-align: -webkit-center; background-color: #F8F9F4;">詹姆斯·韦布空间</span>望远镜是采用抛物面反射光线的方式来成像。因此其口径也比哈珀太空望远镜大得多，自然成像的清晰度也就高很多了。可以看见人类以前观察不到的宇宙星系细节。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202207/13/102516vri44ebnhce3p34r.jpg" title alt="Webb.jpg" referrerpolicy="no-referrer"><br></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">按照膨胀宇宙理论，目前的宇宙在加速膨胀，因此遥远星系发出来的光线，传送到地球上，都出现了红移现象。因此只对红外光线拍摄反而能够反映出实际的可见光的图像。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">SMACS 0723行星<span style="font-family: 宋体, SimSun; color: #000000; font-size: 20px;">团距离地球四十六亿光年，因此目前<span style="font-size: 20px; color: #000000; text-align: -webkit-center; background-color: #F8F9F4; font-family: 宋体, SimSun;">詹姆斯·韦布</span>所拍摄的照片实际上是该星系团四十六亿年前发送出来的。</span></span></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">而7月12日，NASA又同时发布了三张高清照片和一张对太阳系外行星WASP-96 b大气的光谱分析。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">第1张照片是距离地球2.9亿光年之外的所谓“星系群斯蒂芬五重奏”照片。该照片显示了五个相距很近的星系构成一个星系群。其中的四个星系彼此之间还会出现反复靠近的现象。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202207/13/102531ey4rgfa5wcni6afe.jpg" title alt="main_image_galaxies_stephans_quintet_sq_nircam_miri_final-5mb.jpg" referrerpolicy="no-referrer"><br></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">第2张照片显示了以前因为被遮挡而无法仔细观察的Carina星云。该星云距离地球7600光年，位于银河系中。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202207/13/102549zqyygvclvpgv0q4p.jpg" title alt="main_image_star-forming_region_carina_nircam_final-5mb.jpg" referrerpolicy="no-referrer"><br></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">从这张照片中可以看出星云之中有大量年轻的恒星诞生。这为人类理解星系的形成提供了更丰富的证据。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">第3张照片则显示了距离地球2500光年的南环行星状星云。该星云同样位于银河系之中，是我们的近邻。从该照片中可以看到近红外光（左）和中红外光照片（右）的对比。近红外光明显比中红外光要明亮很多。这可以反映该正在死亡的恒星的倒计时时间。当然对这颗正在死亡恒星的研究，也可以预示太阳今后的宿命。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202207/13/102605eowo4hr5c7c60xbb.jpg" title alt="main_image_stellar_death_s_ring_miri_nircam_sidebyside-5mb.jpg" referrerpolicy="no-referrer"><br></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">另外<span style="text-align: -webkit-center; background-color: #F8F9F4; font-family: 宋体, SimSun; font-size: 20px; color: #000000;">詹姆斯·韦布空间</span>望远镜还对一颗距离地球1150光年的系外行星WASP-96 b的大气构成进行了光谱分析。WASP-96b是目前已知的5000多颗系外行星中的一个。从这个光谱分析结果来看，该行星的大气中包含了大量的水元素。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202207/13/102617p37v5xdvaxe074vw.jpg" title alt="main_image_exoplanet_wasp.jpg" referrerpolicy="no-referrer"><br></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">一提到水，自然让人想到地外生命存在的可能性。不过在这颗行星上似乎不太可能存在地外生命。首先这颗行星质量达到木星的一半，相当于160颗地球的质量。这么大的质量自然会产生非常强大的引力。一个七十千克的人到了这颗行星上，将变成7吨。因此即便是有生命，估计也都是非常小的生物。但是另一个条件又让生命基本上不可能出现在这个行星上。WASP-96b是一颗气体行星，其大气温度达到大约538摄氏度。另外它距离恒星的距离非常近，这导致它的公转周期时间很短。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">不过既然发现了水，自然说明在这个恒星系中，至少是不缺水的。说不定还可能存在很多更小一些的岩石行星，上面有如同地球那样的环境，并孕育着更加丰富的文明。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="text-align: -webkit-center; background-color: #F8F9F4; font-family: 宋体, SimSun; font-size: 20px; color: #000000;">詹姆斯·韦布空间</span><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;"><span style="color: #587C19; font-family: 黑体; font-size: 22px; text-align: -webkit-center; background-color: #F8F9F4;"></span>望远镜项目起源于上个世纪九十年代。直到2004年才开始正式进行开发。去年12月25日正式发射。直到今年7月11号，才正式开始工作。</span></p><p style="margin-top: 20px; margin-bottom: 20px;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #000000;">整个项目投入了大约一百亿美元。投资还是相当巨大的。</span></p><p><br></p>                    <br><br>
