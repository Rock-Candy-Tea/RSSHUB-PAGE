font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">根据联合国教科文组织（UNESCO）统计研究所（UIS）最近的估计，<a href="http://uis.unesco.org/en/topic/women-science" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">世界上的研究人员只有三分之一是女性</a>。男性持续主导着全球范围内的科学、技术、工程和数学（STEM）研究。女性代表不足仍然是一个长期存在的问题，并且在工程及计算机科学等领域尤其突出。</span></p><p style="box-sizing: border-box; margin: 0in 0in 10px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">管漏现象 困扰着科学界，尽管本科生和硕士生的性别比例均等，但在博士和博士后研究岗位中，女性代表却逐渐减少。例如，<a href="https://en.unesco.org/news/just-30-world%E2%80%99s-researchers-are-women-whats-situation-your-country" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">在瑞典</a>，60%的本科生是女性，但博士生中女性的比例下降到49%，研究人员中女性的比例仅为36%。本文试图更深入地探讨女性在科学领域面临的挑战。</span></p><p style="box-sizing: border-box; margin: 0in 0in 10px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p style="box-sizing: border-box; margin: 0in 0in 10px; font-size: 17px; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="font-weight: bolder; box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 18px;">玛蒂尔达效应、马内尔效应等等</span></p><p style="box-sizing: border-box; margin: 0px 0in 24px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">马蒂尔达效应（Matilda effect）指的是女性的科学贡献被抹去、忽视或归于男性。有许多历史实例表明，女性科研工作者的成就遭到了破坏或错误归属。笔者想到的一个突出的例子是罗莎琳德·富兰克林（Rosalind Franklin）对发现DNA结构的贡献。虽然这种对于承认女性科研工作者的公然偏见可能正在减少，但隐蔽的表现仍然存在，表现为女性科研工作者的<a href="https://journals.sagepub.com/doi/10.1177/0306312711435830" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">提名和获奖人数较少</a>、女性科研工作者的研究<a href="https://papers.ssrn.com/sol3/papers.cfm?abstract_id=2303311" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">被引用的次数较少</a>、<a href="https://journals.sagepub.com/doi/abs/10.1177/1075547012472684?journalCode=scxb" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">女性撰写的出版物被认为质量较低</a>，以及<a href="file:///C:/Users/SnehaK/AppData/Local/Microsoft/Windows/INetCache/Content.Outlook/QV0ALOOK/s.plos.org/plosbiology/article%3fid=10.1371/journal.pbio.1002573" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">合作机会较少</a>的比例失衡现象。</span></p><p style="box-sizing: border-box; margin: 0in 0in 10px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">另一种形式的歧视表现为在科学会议和知名的大会上明显缺少女性发言人。女性科研工作者收到的演讲邀请比男性科研工作者少。这些年来，这类活动的参与者全为男性的情况（或称为“manels”）非常普遍，但越来越多的人为这种偏见发出呼声。学术界的高层领导职位也缺少女性。例如，在印度，女性更有可能担任<a href="https://www.brookings.edu/research/difficult-dialogues-a-compendium-of-contemporary-essays-on-gender-inequality-in-india/" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">本科学院的院长，而不是大学校长和副校长</a>。</span></p><p style="box-sizing: border-box; margin: 0in 0in 10px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p style="box-sizing: border-box; margin: 0in 0in 10px; font-size: 17px; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><strong><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 18px;">女性在STEM中面临的主要问题</span></strong></p><p style="box-sizing: border-box; margin: 0in 0in 10px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">女性在STEM中的代表性一直在缓慢增加，女性在男性主导的领域正在取得更大的进展。然而，这些女性仍然面对着大多数男性不会面对的许多挑战。学术界中这种性别失衡的主要原因是什么？</span></p><p style="box-sizing: border-box; margin: 0in 0in 10px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p style="box-sizing: border-box; margin: 0in 0in 10px; font-size: 17px; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><strong><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";"><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; box-sizing: border-box; color: #222222;">1. </span><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; box-sizing: border-box; color: #222222;">女性感到被低估和缺乏支持</span></span></strong></p><p style="box-sizing: border-box; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">成见和预设的观念经常会影响年轻女性的抱负，一些女性发现很难将自己定位为科研工作者。在女性不受重视或鼓励的环境中，这种情况可能会加剧。如上所述，女性很早就退出了科学生涯的阶梯，甚至选择一个没有竞争力的非学术职业。这种转变导致缺少女性作为新毕业学生的示范者和指导者。</span></p><p style="box-sizing: border-box; margin: 0px 0in 24px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">不幸的是，稳固终身职位的时间往往与人们考虑组建家庭的时间重合。然而，帮助学术界父母管理他们的照顾责任的支持系统仍不充分。如果女性研究人员考虑生育儿女，这可能会阻碍她们追求终身职位。不仅如此，怀孕或作为母亲的研究人员甚至可能在工作中面临<a href="https://www.sciencemag.org/careers/2019/04/working-mothers-face-wall-bias-there-are-ways-push-back#:~:text=Maternal%20wall%20bias%20occurs%20when,problem%20for%20women's%20career%20advancement." target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">“母性墙偏见”</a>。</span></p><p style="box-sizing: border-box; margin: 0px 0in 24px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">导师的指导，尤其是那些以身作则的女性的指导，将有助于早期职业女性研究人员处理职业抱负和育儿之间的冲突。</span></p><p style="box-sizing: border-box; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><strong><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">2. 女性必须承担更多的责任</span></strong><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";"></span></p><p style="box-sizing: border-box; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">你知道吗，男性发表的论文通常比女性多。在主要原因中，一个突出的原因是生育和作为人母对女性的影响。女性最终往往不仅是孩子的默认照顾者，也是整个家庭的默认照顾者。新冠肺炎大流行将这一严峻的事实摆在了突出的位置。在大流行期间，在家工作对母亲的影响<a href="https://advances.sciencemag.org/content/7/9/eabd1996" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">远远高于对学术界中父亲的影响</a>。随着家庭责任的增加，女性的生产力尤其下降。有子女的女性研究人员<a href="https://www.nature.com/articles/d41586-021-00527-9" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">能够投入到研究中的时间急剧减少</a>。</span></p><p style="box-sizing: border-box; margin: 0px 0in 24px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">对于这些失去的时间，尽管大学和资助者越来越倾向于支持而不是惩罚女性研究人员，但这些努力需要更加集中。应制定旨在提高灵活性和对不可预见的情况（如疾病大流行）进行某种豁免的政策，以便此类研究人员的评估和职业发展不受影响。</span></p><p style="box-sizing: border-box; font-size: 17px; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><strong><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">3.女性的薪酬比同等职位的男性低</span></strong></p><p style="box-sizing: border-box; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">男女员工薪酬差距无处不在，学术界也不例外。美国国家科学基金会2018年报告称，在美国博士学位持有者中，无论研究领域如何，男性收入都高于女性。这一差距在很大程度上可以用数学和计算机科学领域的男性比例过高来解释——这两个领域往往薪酬更高。然而，仔细观察就会发现，<a href="https://www.nature.com/articles/d41586-021-00387-3" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">即使在这些领域，也存在无法解释的性别差异</a>。一种可能的原因是，女性在领导岗位和更好的工作机会上往往被忽略。一项研究甚至强调了一些根深蒂固的信念，这些信念表现为导致薪酬不平等的<a href="https://www.pnas.org/content/117/48/30303" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">信心差距</a>。有时，自我怀疑和缺乏自信也可能使女性在谈判中要求的薪酬较低。</span></p><p style="box-sizing: border-box; margin: 0px 0in 24px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">教授和导师有责任培养年轻女性研究者的自信和成长心态。此外，需要在学术界的各个级别实施公平的绩效评估和平等机会。</span></p><p style="box-sizing: border-box; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><strong><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";"><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; box-sizing: border-box; color: #222222;">4. </span><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; box-sizing: border-box; color: #222222;">女性比男性面临更多欺凌和骚扰</span></span></strong></p><p style="box-sizing: border-box; margin: 0in 0in 11px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">据报道，女性在工作场合面临敌意、欺凌和骚扰。高等教育环境被认为<a href="https://www.ncbi.nlm.nih.gov/books/NBK519457/" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">“允许”</a>一切形式的骚扰，因为学生依赖教授获得推荐和职业发展。实地工作是另一个对女性来说存在隐患的领域。一项关于女性在实地工作环境中<a href="https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0102172" target="_blank" style="box-sizing: border-box;color: blue;background-color: transparent;cursor: pointer;transition: all 0.2s ease-in-out 0s;outline: none">经历</a>的调查显示，她们经常遭受骚扰和攻击。</span></p><p style="box-sizing: border-box; margin: 0in 0in 11px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">旨在阻止此类行为并提供适当补救条款的反骚扰政策将鼓励女性举报不良事件，并劝阻不正当利用其地位的当权者。增强STEM领域女性权能的支持性框架也将有助于减少女性的弱势。</span></p><p style="box-sizing: border-box; margin: 0in 0in 11px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p style="box-sizing: border-box; margin: 0in 0in 11px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="font-weight: bolder; box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 18px;">科学利益中的性别多样性</span></p><p style="box-sizing: border-box; margin: 0in 0in 11px; font-size: 17px; text-align: justify; color: rgb(4, 0, 0); font-family: "Alegreya Sans"; white-space: normal; background-color: rgb(255, 255, 255); line-height: 1.75em;"><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">像国际妇女节（3月8日）和国际科学界妇女和女孩日（2月11日）这样的特殊日子不仅是为了庆祝杰出的女性，也是为了激发关于性别挑战和障碍的对话。为了鼓励更多的女性在科学界中获得代表权，重要的是要了解是什么激励她们从事科研，以及什么能帮助她们留在科研领域。</span></p><p style="box-sizing: border-box;
