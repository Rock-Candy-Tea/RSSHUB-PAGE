style="font-family:SimSun">猴痘病毒的潜伏期一般为</span>5-11<span style="font-family:SimSun">天，最长可达三周。在潜伏后，便会进入发病期和皮疹期。发病期一般持续</span><span style="font-family:Calibri">0-5</span><span style="font-family:SimSun">天后便会结痂。另外，约有五成的猴痘患者会出现淋巴结肿大，而天花患者则不会，因此可借由区分两者。</span></span></p><p style="margin-top:20px;margin-right:0;margin-bottom:20px;margin-left:0;padding:0 0 0 0 "><span style=";font-family:Calibri;font-size:19px"><span style="font-family:SimSun">猴痘是一种自限性疾病，大多数人在出现症状后</span>2<span style="font-family:SimSun">至</span><span style="font-family:Calibri">3</span><span style="margin-top:20px;margin-right:0;margin-bottom:20px;margin-left:0;padding:0 0 0 0 "><span style=";font-family:Calibri;font-size:19px"><span style="font-family:SimSun">目前尚未有针对猴痘病毒的特效药，也没有经过科学验证的标准化治疗方案。不过可以接种天花疫苗预防猴痘。</span></span></p><p style="margin-top:20px;margin-right:0;margin-bottom:20px;margin-left:0;padding:0 0 0 0 "><strong><span style="margin-top:20px;margin-right:0;margin-bottom:20px;margin-left:0;padding:0 0 0 0 "><span style=";font-family:Calibri;font-size:19px"><span style="font-family:SimSun">但由于天花病毒在上世纪</span>80<span style="font-family:SimSun">岁以下的人群不太可能有抗体。即使是曾经接种过天花疫苗的人群，由于时间过久，免疫力下降，对猴痘病毒的保护能力也有限。</span></span></p><p style="margin-top:20px;margin-right:0;margin-bottom:20px;margin-left:0;padding:0 0 0 0 "><strong><span style="font-family: Calibri;font-size: 19px"><span style="font-family:SimSun">九派新闻：猴痘病毒的治愈率和死亡率如何？治愈后是否会留下后遗症？</span></span></strong></p><p "><strong><span style="font-family: Calibri;font-size: 19px"><span style="font-family:SimSun">严家新：</span></span></strong><span style=";font-family:Calibri;font-size:19px"><span
