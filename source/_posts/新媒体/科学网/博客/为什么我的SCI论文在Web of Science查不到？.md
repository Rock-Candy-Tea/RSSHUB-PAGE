<p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">近日，有读者提问：于2021年12月发表了一篇sci文章但至今未能在WOS检索，之前看到您写的文章说可以向WOS申请补录。想请问您一下，确定是sci期刊的专刊（special issue），且同一主题的专刊（special issue）大多都被web of science收录了，那么只要我提交申请就一定可以被收录吗（只是或快或慢），还是web of science他们自己还会再对文章进行审核，有收录标准呢？</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">回答：如果确认被漏掉，肯定会补录，WOS不会进行审核，只要杂志是SCI就行，确定其他同期文章被收录的话，WOS一般在七天内就可以处理完毕。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">(https://support.clarivate.com/ScientificandAcademicResearch/s/datachanges?language=en_US)可以向WOS申诉。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">以下贴出之前写于ISE学术前沿的文章，供大家参考：</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">只要碰到高校或者科研结构要求工作业绩是在一定时间范围内，比如中级晋副高，副高晋高级，一般都会对业绩的完成时间有严格的限制。那么在这种情况下，对于SCI论文，必须要提供正规机构开具的论文检索报告。也许不同地区和机构开具检索报告略有不同，但是大同小异。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">而且这里还是想提醒大家， <span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">一定要和具体的评审机构问清楚，需要哪里开具的检索报告，并不是所有机构都有资格开具这个检索报告的。</span> 比如当地某医科大学，只认可当地某理工大学开具的检索报告。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">开具SCI论文检索报告的流程是不难的，想开具检索报告的同学或者青椒们，可以提前电话联系开具检索报告的机构，一般设置在图书馆等地，避免在职称评审密集时间段，因为在这些时间段，开具报告的人很多，也许需要排队甚至预约。</span>有些机构是接受网上预约，然后你留存地址，报告开具完毕后，可以邮寄给你，邮费自理。有的机构只接受现场来开具报告。开具报告一般不需要太多证件，据大朋个人经验，只需要带着你要查的论文题目、作者等信息即可，最好把身份证也带着，当然最重要的是你得带钱。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">关于论文检索报告查询费用还是蛮贵的，参照辽宁某211/985理工大学图书馆信息检索机构的收费标准，大概费用基本如下:</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">1.基本检索费30元</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">2.入库查询费5元</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">3.影响因子15元</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">4.分区15元</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">5.他引，15元+引用次数*5元。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">假设我开具发表的一篇SCI论文检索报告，该论文被引用100次，那么大概费用在35+5+15+15+15+100*5=585元。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">友情提示，如果该查询机构内容人员，一般都有内部价格，我就不清楚了。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">但是，当你2020年10月份，拿着身份证和自己的论文题目，来查你于2019年5月份见刊的SCI论文的时候，查询人员突然告诉你，对不起，查不到。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">这个时候，我觉得很多人都会突然晕掉，不知所措，别着急，如果你看了我这篇博文，就不要慌了，怎么办呢？</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">1．确认你发表的论文真的是SCI吗</span></p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">核对你的论文所发表的期刊，然后到Web of Science (WOS)上去查，是否这个杂志被WOS收录，这也是最权威的判断一个杂志是否是SCI的办法。</span>当然并不是所有机构都拥有WOS数据使用库权限（如有类似需要的青椒硕博们也可以评论区留言，我适当协助）。同样目前有很多比如小木虫、梅斯医学网站，也可以去上面查你发表的期刊所在杂志是否为SCI收录杂志。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">如果确认了你的SCI论文发表的杂志的确是SCI杂志，那么你去这个杂志主页看一看和你同期发表的论文（比如你的论文发表在2020年issue7, 那么你去查一查和你同期的其他论文，随便找一篇），复制该论文的题目，到WOS去检索是否能检索的到。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">2．向WOS提出申诉，补录你的论文</span></p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">确认你的论文确实是SCI论文，而且和你论文同杂志同期发表的其他论文也已经在WOS查询得到，那么可以肯定是WOS工作人员漏掉了你的论文。登陆该网站(https://support.clarivate.com/ScientificandAcademicResearch/s/datachanges?language=en_US)可以向WOS申诉。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">友情提示：WOS以前是汤姆森路透负责管理，后移交科睿唯安管理。汤姆森路透管理时候，对于收录的论文相关Data Changes内容比较宽泛，也比较尊重通讯作者的意见，但是移交科睿唯安以后，Data Changes相当严格，基本不容商量，在此我不做过多探讨，网页如下所示，其具体的数据修改说明也可以点击浏览。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">个人小绝招：通过这个data change提交，有概率常可以加快你的论文被WOS收录的进程。</span></p><p class="ql-align-center" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px;
