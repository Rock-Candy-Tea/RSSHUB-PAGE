style="font-family:宋体">，</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px">“又飘过西海，直至西牛贺洲地界”。</span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">在那儿，</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">他</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">只凭</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">远远听到</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">的言语和歌声，就断定</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px">“神仙原来藏在这里”</span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">。那</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">歌</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">唱道</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">：</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px">“观棋柯烂，伐木丁丁，云边谷口徐行。卖薪沽酒，狂笑自陶情。苍径秋高对月，枕松根，一觉天明。认旧林，登崖过岭，持斧断枯藤。收来成一担，行歌市上，易米三升。更无些子争竞，时价平平。不会机谋巧算，没荣辱，恬淡延生。相逢处，非仙即道，静坐讲《黄庭》。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">走近歌声，悟空</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">惊讶于</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">歌者竟然不是神仙，而是一个穷困潦倒、上不起学的</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">樵夫。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">在离</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">樵夫</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">七八里远近，</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">悟空</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">找到了灵台方寸山斜月三星洞</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">和</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">菩提祖师。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">众所周知，</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">悟空</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">正是</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">拜菩提祖师为师，</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">才</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">学成</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">长生不老术和</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">大闹天宫的本领。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">悟空的判据（其实是直觉）是：有神仙的学校，学校周边方圆数公里范围内的人，哪怕是社会最低层的人，都快活似神仙，反之也成立。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">这个判据</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">本质上类似于</span>“孟母三迁”，说的都是：什么样的环境有什么样的学校，目前有和将来预计出什么样的人。其基本原理，在老文看来，根植于“</span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">教育就是影响</span>”。有大师的学校有大影响，不仅影响学生，也影响社会，而影响社会，首先影响周围的社会。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><strong><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">法子三，在手机上刷一刷。</span></span></strong></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">文化是最重要的环境。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">研究发现，一个组织的文化是由行为而不是言语决定的（</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px">defined by behaviors, not words</span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">）。具体说，一个组织</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">的文化就是</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">这个组织</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">奖励和惩罚的行为。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">把这个发现应用到大学，可以说，一所大学的文化往往并不是如其公开的宣示（如校训）所言。文化的形成需要积淀。大学文化形成的最重要推手不是校训等口号、校长在开学典礼等重要场合的寄语，甚至也不是教师在课堂上的说教，而是学校对师生的奖惩制度及执行行为。这其中，尤以学校对教师的聘任、考核等制度最为关键。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">因此，看一所大学真实的文化，从其教师聘任、考核等制度可见一斑。这些制度，往往都在网上公开。特别是，大学招聘人才，往往竞相在网络媒体打广告，其招聘广告中开出的最诱人条件，是其最真实的价值观或最真实的追求。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">仔细比较，不同层级（从顶尖到一般）大学的校训（至少校训意思）可能大同小异，但它们的</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">奖惩</span></span><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">制度和招聘广告就千差万别了。而这千差万别之中，恰恰呈现出最真实的大学分层和分野。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: 宋体;line-height: 32px;font-size: 16px"><span style="font-family:宋体">由此，判断大学有没有大师，有比走一走、游一游更</span>“懒人”的法子：在手机上刷一刷，瞄一瞄大学的人才招聘广告。</span></p><p><br></p>                    <br><br>
