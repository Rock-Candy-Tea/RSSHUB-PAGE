<p><span style="font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">费米37岁那年，于1938年12月10日被授予诺贝尔物理学奖，获奖的主要理由是他使用中子轰击元素周期表的最后一个元素产生了新元素。用人工的方法首次得到新元素的工作当然是开创性的，划时代的，因此获奖应该当之无愧。然而，没过多久（好像是12天），哈恩和斯特拉斯曼就撰文推翻了费米产生新元素的结论。铀核俘获一个中子后没有变成比它更重的新元素，而是发生裂变，生成了56号元素钡。在公众们看来，诺奖委员会搞了一次乌龙事件。然而科学界的想法或许并不是这样，他们理解费米工作的意义，并用他的方法合成了93号和94号元素。</span></p><p><br></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 18px;">此时，费米和他的家人正在远洋客轮弗兰肯号上享受从南安普敦到纽约的长途旅行。他们于1939年1月2日抵达美国，并受聘于哥伦比亚大学。裂变发现的消息在国际物理学界迅速传播开来，丹麦物理学家尼尔斯·玻尔（Niels Bohr）将这一消息带到了普林斯顿大学的一个研讨会上。 从那里它很快被转发给哥伦比亚大学的费米。 裂变是一种全新的、完全出乎意料的东西，以至于费米在很细致的实验中也忽略了它。裂变具有巨大的能量潜力，一克完全裂变的铀 235 产生的能量相当于 3 吨煤或 700 加仑燃料油。 裂变的威力是普通化学燃烧的百万倍。</span></p><p><span style="font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 18px;">因担忧希特勒统治的法西斯德国把裂变用于武器研制，<span style="font-family: 微软雅黑, "Microsoft YaHei";">西拉德请爱因斯坦给罗斯福总统写信，警示这种新型裂变武器的应用。这封信导致了铀咨询委员会的成立，并使费米获得6000美元的资助，开展铀裂变的研究。有人形容这件事时说：大自然把甲板套在了费米身上。</span></span></p><p><span style="font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p><span style="font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">费米利用他的慢中子知识和熟练的实验技能，很快地完成各项研究，并与1942年初搬迁至芝加哥开始反应堆建设，12月就在芝加哥大学体育场的壁球馆里建成了第一个自维持的链式反应堆。<span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">这在是核物理学上具有里程碑意义的事件，如同打开了潘多拉之盒。当时，<span style="font-size: 18px;">冶金实验室的负责人，亚瑟·康普顿很高兴地通过电话向主管詹姆斯·科南特（James Conant）报告，<span style="font-size: 18px; background-color: #FFFFFF; color: #000000;">“这位意大利航海家刚刚登陆了新世界。”</span>这的确是一个新的世界，</span>费米也因此被称作核能之父。</span></span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">随后，费米参加了美国的“曼哈顿计划”，成为制造原子弹的主要领导者之一。期间，费米的名字被改作法默（Farmer）。据说<span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">费米夫人第一次来到洛斯阿拉莫斯时，在火车站遇见她的士兵问道：“你是法默夫人吗？”</span> <span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">她回答说：“不，我是费米夫人。”于是士兵温和但带着责备的眼神说道，“我被告知要叫你法默夫人。”</span></span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">费米为研制原子弹做出了不可或缺的贡献。他带领团队解决了中子毒化问题，一些裂变产物比如碘-135具有很大的中子吸收截面，将影响链式反应的进程，费米与杜邦工程师一起设计一种增加新的冷却通道来添加铀的办法以克服毒化问题。此外，他还领导了点火中子源和临界质量的研究，为钚弹的成功做出了杰出的贡献。</span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">1945年7月16日晚上，世界上第一颗原子弹在内华达州的沙漠里引爆成功。此时，费米就在试爆现场附近。爆炸40秒后，冲击波传了过来，费米把一些碎纸片抛向空中，并根据这些碎屑落地的位置估算出原子弹的当量为1万吨TNT。从这点可以看出费米的模型计算能力很强。1968年诺贝尔物理学奖获得者也具有很强的计算能力，他曾经帮助查德威克计算了氘破裂反应，帮伽莫夫计算了p+p反应等，是核物理学界的计算大师(以后有时间讲一下)。贝特在解释自己为什么有如此高的计算能力时曾经说过，他有两个好的老师，一个是索末菲，从那里获得了理论计算的能力；另一个就是费米，从他那获得了模型计算的能力。</span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">曼哈顿计划成功后，费米又回到了芝加哥大学开展加速器的研究。由于费米的声望，也吸引了其他明星到新的研究所和芝加哥大学的物理系，其中包括爱德华·泰勒，赫伯特·安德森和列奥娜·马歇尔等等，其中一些人还获得了自己的诺贝尔奖。</span></p><p><span style="color: #333333; background-color: #FFFFFF; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 18px;"><br></span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";"><span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">从 1947 年开始，费米开始关注新粒子，</span> <span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">诸如“π介子”之类的问题。这反映了战后人们对将核心凝聚在一起的兴趣的激增。</span> <span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">毕竟，原子核内的质子带正电，应该相互排斥并飞散。</span> <span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">还有另一种力——特别是核力——将它们与相邻的核子聚集在一起。<span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">一位名叫汤川的日本物理学家曾建议将这种粒子作为核力的来源。</span> <span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">费米曾在一个学术报告中解释介子和核力：根据汤川理论，中子偶尔会转化为质子和π介子，然后质子和π介子会再次被重吸收并抛出。<span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">从这一推理路线中得出了π介子质量的值，该值至少与当加速器发展到足以使 π介子在实验室中被观察到时开始观察到的值是一致的。</span></span></span></span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";"><br></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei";"><span style="color: #333333; background-color: #FFFFFF; font-size: 18px;"><span style="color: #333333; background-color: #FFFFFF;">1954 年 11 月 16 日，他被原子能委员会授予 25,000 美元的特别奖，以表彰他的成就。</span> <span style="color: #333333; background-color: #FFFFFF;">德怀特·艾森豪威尔总统“热情地批准”了 AEC 主席刘易斯·施特劳斯的建议，以这种特殊的方式向费米致敬。</span> <span style="color: #333333; background-color: #FFFFFF;">费米于 1954 年 11 月 29 日去世。他的墓碑上简单而恰当地写着：恩里科·费米， 1901-1954 ，物理学家。</span></span><span style="color: #333333; background-color: #FFFFFF; font-size: 16px;"></span></span><span style="color: #333333; background-color: #FFFFFF; font-size: 16px; font-family: 宋体;"><br></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 18px; color: #333333; background-color: #FFFFFF;"><br></span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";"><span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">费米的科学创作始于</span><span style="font-size: 18px; color: #333333; background-color: #FFFFFF;"> 1921 </span><span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">年，并于</span><span style="font-size: 18px; color: #333333; background-color: #FFFFFF;"> 1954 </span><span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">年以他的去世结束。</span> <span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">在刚开始时，人们只知道两种基本的自然力，引力和电磁力，以及两种基本粒子，质子和电子。</span> <span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">在</span><span style="font-size: 18px; color: #333333; background-color: #FFFFFF;"> 1950 </span><span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">年代中期，人们已经知道了四中基本的力（加上强核和弱核相互作用），并且已知的基本粒子有</span><span style="font-size: 18px; color: #333333; background-color: #FFFFFF;"> 30 </span><span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">多种。</span> <span style="font-size: 18px; color: #333333; background-color: #FFFFFF;">在不到三十年的时间里，物质概念经历了如此彻底和前所未有的变化。那个时期，新概念获得的数量和速度，在西方科学思想史上也是独一无二的。这些发展提升了物理学家在公众心目中的地位，当时好像有这样一首歌为成为一名物理学家庆贺：</span><br></span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">真好，在1947年成为一名物理学家；</span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">真好，在这荣耀的一年成为物理学家；</span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">看到了，轻蔑的世界终于承认你的地位；</span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">看到了，参议员们尊重你们每一个声明；</span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">你们让女子俱乐部着迷，</span></p><p><span style="color: #333333; background-color: #FFFFFF; font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">你们是毕业典礼上的明星。</span></p><p><span style="color: #333333;
