white-space: normal; background-color: rgb(255, 255, 255);">·在提供伦理准则方面，WoS 索引的期刊明显多于掠夺性期刊；<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">掠夺性期刊往往没有开放获取政策或撤稿政策。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">·在被调期刊中，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">掠夺性期刊的平均APC（301美元）明显低于WoS收录期刊的平均APC（2,804美元）。</span>正规期刊收取惊人的费用可能是保持其声望的一种策略。另一方面， <span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">掠夺性期刊征收的较低APC可能只是其吸引尽可能多的投稿以获得收入的一种方式。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">5.索引、ISSN和DOI</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">·与 WoS 索引的期刊相比，声称被 Google Scholar 索引的掠夺性期刊百分比更高。至于被ERIC收录的期刊，两组期刊之间的差异并不明显。声称被DOAJ索引的差异也不大。在Scopus索引方面，被WoS索引的期刊多于被掠夺性期刊索引的期刊。值得注意的是，WoS收录期刊提出的所有索引要求都得到了验证，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">但掠夺性期刊提出的许多要求却被证明是错误的。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">·掠夺性期刊和WoS收录的期刊都报告了影响因子。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">由于没有一本掠夺性期刊被纳入SSCI，它们报告的影响因子是假的。</span>WoS收录期刊报告的影响因子平均值为1.211，掠夺性期刊则为4.097。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">这些结果揭示了掠夺性期刊的欺骗性。</span>由于掠夺性期刊只是偶尔被Scopus和ERIC等可靠的数据库收录，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">所以被权威的数据库收录是区分正规期刊和掠夺性期刊的可靠标准。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">·最后，所有WoS收录的期刊都有ISSN和DOI，而95.5%的掠夺性期刊有ISSN，59.1%有DOI。有无ISSN的差异不大，但有无DOI的差异很大。</p><p class="ql-align-justify" style="border: 0px;
