font-size: 14px; white-space: normal; line-height: normal; background: white;"><span style="overflow-wrap: break-word; font-size: 16px; color: #222222;"> </span></p><p style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">我一直认为在过去几十年数字通信物理层的主要工作就是怎样处理码间干扰</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">(即</span><span style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 0px; padding: 0px; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; line-height: normal; background: white;"><span style="overflow-wrap: break-word; font-size: 16px; color: #222222;"> </span></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 0px; padding: 0px; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; line-height: normal; background: white;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">对码间干扰的消除，有两种方式，一种是时域处理，一种是频域处理。时域处理的好处是，发端简单，但收端稍复杂一些。更主要的是如果信道比较长或和变化比较快时，收端会更复杂及可能会来不及应变。</span></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 0px; padding: 0px; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; line-height: normal; background: white;"><span style="overflow-wrap: break-word; font-size: 16px; color: #222222;"> </span></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 0px; padding: 0px; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; line-height: normal; background: white;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">在无线宽带信道时，如现在的</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">WiFi</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">和</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">4G</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">和</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">5G</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">中，码间干扰是通过频域处理的方式，即</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">OFDM</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">，把有较强的码间干扰宽带信道变成多个没有码间干扰的窄带信道。</span></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 0px; padding: 0px; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; line-height: normal; background: white;"><span style="overflow-wrap: break-word; font-size: 16px; color: #222222;"> </span></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 0px; padding: 0px; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; line-height: normal; background: white;"><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">我在上世纪九十年代底提出的单天线</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">Vector OFDM</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">（</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">VOFDM</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">），首次正式发表是</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">2000</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">年六月份在美国新澳尔良开的</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">ICC</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">论文集上，是把一个宽带信道（如共有</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">L</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">个符号干扰在一起）变成若干个矩阵子信道，<span style="overflow-wrap: break-word;">矩阵子信道间互不干扰，而</span>每个矩阵子信道中的相互干扰的符号个数正好是向量长度，且这个向量长度是可以人为控制的，可以是</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">1</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">，或</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">2</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">，或</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">3</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">，</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">L等</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">。当向量长度为</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">1</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">时，</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">VOFDM</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">正好是</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">OFDM</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">（</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">4G</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">的下行），当向量长度不小于</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">L及IFFT的长度为１</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">时，</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">VOFDM</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">就是</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">4G</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">的上行）。过去十年，我也经常报告过这个</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">VOFDM，特别在内地。</span></p><p style="overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 0px; padding: 0px; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; white-space: normal; line-height: normal; background: white;"><span style="overflow-wrap: break-word; font-size: 16px; color: #222222;"> </span></p><p style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">不巧的是，在上世纪九十年代底提出单天线</span><span style="overflow-wrap: break-word; font-size: 16px; font-family: KaiTi; color: #222222;">VOFDM</span><span style="overflow-wrap: break-word;
