<p><span style="font-family: 宋体, SimSun; font-size: 20px;"><span style="font-family: 宋体, SimSun; color: #333333; text-indent: 28px; background-color: #FFFFFF;">中国著名儿童教育家</span><span style="font-family: 宋体, SimSun; background-color: #FFFFFF; color: #333333; text-indent: 28px;">陈鹤琴</span><span style="font-family: 宋体, SimSun; color: #333333; text-indent: 28px; background-color: #FFFFFF;">说过“</span>没有教不好的学生，只有不会教的老师<span style="font-family: 宋体, SimSun; background-color: #FFFFFF; color: #333333; text-indent: 28px;">”。<span style="font-family: 宋体, SimSun; text-indent: 28px; color: #333333; background-color: #FFFFFF;">他强调的是教育者对学生的一种责任与信念，是教育者对自己的严格要求，当然也有其理想化成分，</span></span><span style="font-family: 宋体, SimSun; background-color: #FFFFFF; color: #333333; text-indent: 28px;">用以激发教师对职业的敬畏和热爱。</span><span style="font-family: 宋体, SimSun; background-color: #FFFFFF; color: #333333; text-indent: 28px;">这句话与韩愈关于教师的“传道授业解惑”理念异曲同工。</span><span style="font-family: 宋体, SimSun; background-color: #FFFFFF; text-indent: 28px;">教</span><span style="font-family: 宋体, SimSun; background-color: #FFFFFF; color: #333333; text-indent: 28px;">师如果没有尽心尽职把学生培养好，当然应该承担责任。</span><span style="font-family: 宋体, SimSun; background-color: #FFFFFF; color: #333333; text-indent: 28px;">然而这句话不能夸大解读，否则就成了捆绑教师的道德枷锁，似乎教育的成败都是教师造成的。而且过度解读推及其它行业，如<span style="font-family: 宋体, SimSun; color: #333333; text-indent: 28px; background-color: #FFFFFF;">“</span>没有治不好的病，只有不会治的医生<span style="font-family: 宋体, SimSun; background-color: #FFFFFF; color: #333333; text-indent: 28px;">”纯属</span>逻辑混乱。</span></span></p><p><span style="background-color: #FFFFFF; color: #333333; text-indent: 28px; font-family: 宋体, SimSun; font-size: 20px;">陈鹤琴作为儿童教育家，我想他这句话适合儿童没问题，但是对于已是成年人的大学生研究生完全不适用。有些人把这句话作为至理名言，仿佛站在道德制高点评判别人，怎么不照照自己？</span></p><p><span style="background-color: #FFFFFF; color: #333333; text-indent: 28px; font-family: 宋体, SimSun; font-size: 20px;">我认为教育不是万能的，教师也不是神！</span></p><p><span style="background-color: #FFFFFF; color: #333333; text-indent: 28px; font-family: 宋体, SimSun; font-size: 20px;">人的三观形成于青少年时期（叛逆期）。三观形成受社会，家庭因素的影响远大于学校。大学生对周围的人和事物有自己想法，独立性更强。其行为完全是受大脑支配的，不要说什么“小孩子”了。一些大学生研究生对老师的行为实在令人气愤，将自己的考试通不过，论文通不过，毕业延期归咎于老师。这种学生怎么不反思自己？大学生的错误不存在原谅与否，只能作为教训伴其一生。学校里教不会的，社会一定会教他。</span></p><p><span style="background-color: #FFFFFF; color: #333333; text-indent: 28px; font-family: 宋体, SimSun; font-size: 20px;"></span></p><p style="margin-top: 0px; margin-bottom: 0px; padding: 0px; overflow-y: auto; max-width: 100%; line-height: 24px;"><span style="font-family: 宋体, SimSun; font-size: 20px;"><span style="max-width: 100%; font-family: 宋体, SimSun; font-size: 20px;">教师，永远是一种崇高的职业。</span>作为教师，要尊重学生也要尊重自己，牢记使命和担当。陈先生的话作为自我鞭策可以，但不应成为他人对教师约束评判的工具。</span></p><p><span style="background-color: #FFFFFF; color: #333333; text-indent: 28px; font-family: 宋体, SimSun; font-size: 20px;"><br></span><br></p>                    <br><br>
