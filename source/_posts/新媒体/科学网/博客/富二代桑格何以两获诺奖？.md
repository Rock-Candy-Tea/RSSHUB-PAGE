<p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; color: #191919; background: white; font-size: 16px;">生于富豪之家的桑格，是一位名副其实的富二代。然而，他并未像诸多不学无术的富二代那样堕落成纨绔子弟，而是凭两获诺奖成为了科学大师。任何一位科学家获一次诺奖已实属不易，而他竟两次把诺奖收入囊中。这是为什么呢？从他的经历看，答案昭然诺揭，其可浓缩为两字——热爱。</span></p><p style="text-indent:32px;line-height:200%"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">受家境影响，桑格从小对生物学有着浓厚兴趣</span>，<span style="font-size: 16px; font-family: 宋体, SimSun;">因此他选择了剑桥大学的生物化学专业作为志业。在上学期间，他的成绩一般、能力平平，故他毕业求职时遇到了麻烦——找不到接收单位</span>。为此，他<strong><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919; border: 1px none windowtext; padding: 0px; background: white; font-weight: normal;">在求职信的最后都加上了一句话：“我自带干粮不要薪水。”</span></strong><strong><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919; border: 1px none windowtext; padding: 0px; background: white; font-weight: normal;">这样，他</span></strong><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919; background: white;">最终入职了剑桥的一个科研实验机构。</span></span></p><p style="text-indent:32px;line-height:200%"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919; background: white;">他在一间暗无天日、气味难闻的地下实验室工作，期间无怨无悔，凭兴趣埋头苦干，经</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #191919; background: white;">10</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919; background: white;">余年默默无闻的努力后，他</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">完整定序了胰岛素的氨基酸序列，证明蛋白质具有明确构造；凭此成果，他于</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #333333; background: white;">1958</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">年获得</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white;">诺贝尔化学奖</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">。</span></span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; color: #191919; background: white; font-size: 16px;">首获诺奖时，他还不是教授；得诺奖后，校方才给予他教授头衔和行政职务。</span></p><p style=";margin-bottom:0;text-align:justify;text-justify: inter-ideograph;text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; color: #191919; background: white; font-size: 16px;">得诺奖后，他并未忘乎所以，而始终认为自己是普通人。源于科研带来的踏实感、愉悦感和成就感，他并未停下进取的脚步，一直在苦苦探寻新的研究方向。找到了新方向——给<span style="line-height: 200%; color: #191919; background: white; font-size: 16px; font-family: "times new roman";">DNA</span>测序，他第一时间推掉了各项行政工作，没日没夜地泡在实验室，仍一如既往地亲力亲为做实验。</span></p><p style=";margin-bottom:0;text-indent:32px;line-height:200%;background:white"><span style="line-height: 200%; font-family: 宋体; color: #191919; background: white; font-size: 16px;">经过近<span style="line-height: 200%; color: #191919; background: white; font-size: 16px; font-family: "times new roman";">20</span>年苦行憎般的探索，他终于开发出了一套高效的<span style="line-height: 200%; color: #191919; background: white; font-size: 16px; font-family: "times new roman";">DNA</span>测序方法，名为“双脱氧链终止法（桑格法）”。以后，这套方法逐渐演变成了世界通用的<span style="line-height: 200%; color: #191919; background: white; font-size: 16px; font-family: "times new roman";">DNA</span>测序手段，并为浩荡的“人类基因组计划”提供了坚实的实验方法。</span></p><p style=";margin-bottom:0;text-indent:32px;line-height:200%;background:white"><span style="line-height: 200%; font-family: 宋体; color: #191919; background: white; font-size: 16px;">凭此成果，他再次接到了瑞典的电话——获得<span style="line-height: 200%; color: #191919; background: white; font-size: 16px; font-family: "times new roman";">1980</span>年诺贝尔化学奖。</span></p><p style=";margin-bottom:0;text-align:justify;text-justify: inter-ideograph;text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; color: #191919; background: white; font-size: 16px;">总而言之，他取得的所有成就，靠的是不懈探索——日积月累的试错和千万次的从头再来，靠的是挚爱科研——以好奇心刨根问底，靠的是把精力用到正途——聚焦于实验过程和可靠结果而非论文发表（他一生只发表了屈指可数的几篇论文）。</span></p><p style=";margin-bottom:0;text-align:justify;text-justify: inter-ideograph;text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; color: #191919; background: white; font-size: 16px;">退休后，他照样喜欢无官一身轻，因而拒绝了英国女王授予的爵士勋章。后来，他被同行称为“基因学之父”，可他却自嘲：“我只是个一辈子在实验室里瞎胡混的家伙。”</span></p><p style=";text-align: justify;text-indent: 32px;padding: 0.266667rem 0px;line-height: 0.8rem"><span style="line-height: 200%; font-family: 宋体; color: #191919; background: white; font-size: 16px;">在他家里，不摆任何奖章和奖杯；他像普通人那样遛弯养花，不事张扬，过着简单纯朴的生活。直到<span style="line-height: 200%; color: #191919; background: white; font-size: 16px; font-family: "times new roman";">2013</span>年<span style="line-height: 200%; color: #191919; background: white; font-size: 16px; font-family: "times new roman";">11</span>月<span style="line-height: 200%; color: #191919; background: white; font-size: 16px; font-family: "times new roman";">19</span>日，这个快乐而古怪的科学大师，在梦中安然离世。</span></p><p style=";margin-bottom:0;text-align:justify;text-justify: inter-ideograph;text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; color: #191919; background: white; font-size: 16px;">爱迪生曾说过这样一句话：“热爱是能量，没有热爱，任何伟大的事情都不能完成。”是啊，热爱科研不一定成为科学大师，而缺乏热爱则一定不会成为科学大师！</span></p><p style=";margin-bottom:0;text-align:justify;text-justify: inter-ideograph;text-indent:32px;line-height:200%"><span style="font-size: 16px;"><span style="background-color: white;
