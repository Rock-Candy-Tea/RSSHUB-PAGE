white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">研究背景</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">同行评议是学术期刊发表论文的一个重要环节。优秀的审稿人能够遴选出相关的、严谨的论文，发现并报告不可重复和欺诈性的研究结果。此外，他们还为作者提供建设性的意见，帮助作者改进论文质量。期刊的<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">同行评议声誉是影响期刊投稿的一个关键因素。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">然而，Publons的全球调查显示75%的期刊编辑认为寻找有意愿且有能力的审稿人是他们所需面临的最困难的任务之一，拒绝审稿邀请的审稿人越来越多。这种现象可以归咎于许多因素，如全球论文产出数量激增，审稿人储备不足，以及审稿人在发达国家和发展中国家之间分布不均等等。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">为了及时获得高质量的审稿反馈，编辑会根据科研人员的发文记录、引文指标或所属机构来选择审稿人。与从期刊审稿人库、熟人或工作关系网中邀请审稿人不同，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">编辑在评估自己不熟悉的审稿人是否合适一篇稿件时，应同时考虑其能力和他们接受审稿的可能性。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">研究显示，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">期刊的权威性和声誉是影响审稿人是否决定接受审稿邀请的首要因素，其次是他们与编辑的个人关系和建立关系网的可能性。</span>还有学者指出，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">当稿件被认为对某一主题有贡献、与他们自己的研究领域相关、与他们的兴趣相匹配、有对他们作品的引用，或当主编或通讯作者与自己来自同一地区时，审稿人更有可能同意审稿。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">期刊的排名也会影响审稿人的决定。</span>一些先前的研究显示，与排名靠前的期刊社区互动被认为比与排名靠后的期刊社区互动更有利，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">表明研究人员更愿意为排名靠前的期刊审稿。</span>此外，被引次数较多的审稿人倾向于为高影响力的期刊审稿。审稿人也可能会根据期刊的排名调整不同的评价标准。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">那么，在医学领域，排名较高的期刊和排名较低的期刊的审稿人的计量学指标是否存在差异？我们如何评估通过网络搜索发现的不熟悉的审稿人是否合适审某篇稿件？</p><p class="ql-align-center" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">研究结果</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">本研究显示，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">在医学领域，排名靠前的期刊和排名靠后的期刊的审稿人的计量学指标存在差异。</span>多因素逻辑回归分析显示，单篇论文平均被引次数、总被引频次、h指数以及经验证的审稿次数与审稿期刊的排名显著相关。相对非SCI期刊而言，单篇论文被引次数、经验证审稿次数以及过去12个月经验证审稿次数与为SCI期刊审稿显著相关。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">不同等级的期刊对审稿意见的质量要求也不同。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">排名靠前的期刊要求审稿人更重视稿件的理论贡献和方法论的有效性，而一般和排名靠后的期刊则希望审稿人更多地为经验贡献和交流质量把关。从这个角度看，高等级期刊的编辑显然更青睐那些被认为更有声望的知名科学家。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">同时，成为高质量期刊的审稿人往往被视作某一领域公认的专家的标志，这可以写进简历，增加晋升、成为终身教授和获得资助的机会。如果有选择，研究人员很容易倾向于选择排名靠前而非靠后的期刊进行审稿。此外，许多资深研究人员可能会受到审稿邀请的轰炸，从而直接忽略低影响力期刊的审稿邀请。在本研究中，审稿人的发文记录和审稿期刊排名之间的确切关系尚不确定。但是，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">影响因子/影响因子百分位数较高期刊的审稿人在WoS上发表的论文数量显著多于低排名期刊的审稿人。</span>据报道，期刊编辑倾向于认为论文发表经验是高质量审稿的关键，合格的审稿人同时也应是在高影响力期刊上有良好声誉和发表记录的经验丰富的作者。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">值得一提的是，虽然没有获得影响因子/影响因子百分位数的期刊审稿人的引文指标和发文记录明显差于为高等级期刊审稿的审稿人，但与为排名靠后的SCI期刊审稿的审稿人相比，差异并无统计学意义。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">这可以解释为审稿行为会受到期刊排名以外的诸多因素影响。</span>IOP出版公司发布的2020年同行评议动机报告指出，对论文的兴趣、期刊的声誉和对审稿活动的认可，如给予反馈、奖励和证书，是审稿人的最大动机。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">这意味着低等级期刊，尤其是目前没有影响因子或目前未被WoS收录的期刊，可以通过采取有利措施，如将审稿人与合适的论文相匹配，给予审稿人反馈等，来获得优秀的审稿人。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">另外，处于学术生涯早期的研究人员在担任审稿人时通常会更加勤奋；与资深同行相比，他们完成审稿报告的速度更快，内容更详尽。一些编辑也承认，他们收到的许多高质量的审稿意见都出自处于学术生涯早期的研究人员。那些已经知名的研究人员由于日程紧张，很少有时间去审稿，也不太可能对如何改进文章提出详细的意见。而在某一领域进行过深入研究的早期研究人员，由于急于建立自己的声誉，更有可能发回建设性的审稿反馈。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">总之，期刊的排名与审稿人的学术影响力有关。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">单篇论文的平均被引次数、h指数、总被引频次和经验证审稿次数与他们所审阅的期刊的排名呈正相关。</span>本研究的结果意味着编辑可以根据引文指标和审稿表现来评估不熟悉的审稿人是否合适，而期刊排名和审稿人学术影响力之间的适当匹配将有助于提高审稿邀请的接受率。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">一般来说，在邀请不熟悉的审稿人时，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">编辑应同时考虑他们的能力和接受邀请的可能性。</span>对于排名较低的期刊，来自高声誉学术机构的初期研究人员是不错的选择。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif;
