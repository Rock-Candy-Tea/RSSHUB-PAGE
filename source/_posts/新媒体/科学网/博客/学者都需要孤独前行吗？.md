<p style="text-indent: 149px; text-align: left;"><span style="font-size:21px;font-family:黑体">        学者都需要孤独前行吗？</span></p><p style="text-align: left;"><span style="font-size:16px">                                                                  </span><span style="font-size: 20px;"> <span style="font-family: 宋体;">籍</span> <span style="font-family: 宋体;">利</span> <span style="font-family: 宋体;">平</span></span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 20px;">北宋王安石的《有褒禅山记》说：“<span style="font-family: 宋体, SimSun; line-height: 150%; color: #333333; letter-spacing: 1px; background: white;">于是余有叹焉。古人之观于天地、山川、草木、虫鱼、鸟兽，往往有得，以其求思之深而无不在也。夫夷以近，则游者众;险以远，则至者少。而世之奇伟、瑰怪，非常之观，常在于险远，而人之所罕至焉，故非有志者不能至也。有志矣，不随以止也，然力不足者，亦不能至也。有志与力，而又不随以怠，至于幽暗昏惑而无物以相之，亦不能至也。然力足以至焉，于人为可讥，而在己为有悔;尽吾志也而不能至者，可以无悔矣，其孰能讥之乎?此余之所得也!</span>”</span></p><p style="text-indent:36px;line-height:150%"><span style="line-height: 150%; color: #333333; letter-spacing: 1px; background: white; font-family: 宋体, SimSun; font-size: 20px;">“险以远”的地方至者少，更需要探索者，需要探险家。很有可能，成为探索者比较容易，成为取得世人瞩目发现的探险家不容易。在探险的道路上，有牺牲者的墓地，有突然停滞的脚印。在这样的地方，后来者是停下来，还是想法前行？哪怕前行的距离非常有限。</span></p><p style="text-indent:36px;line-height:150%"><span style="line-height: 150%; color: #333333; letter-spacing: 1px; background: white; font-family: 宋体, SimSun; font-size: 20px;">如果这个探索者的目的很单纯，就是为了探究某种现象的真谛。比如石钟山为何叫石钟山？鸣沙山为何叫鸣沙山？ 月牙泉的泉水是哪里来的？比如某地下河究竟流向了何方？某区域的地下究竟有多少天然的、贯通的管道？可不可以用一些胶囊相机获取这些管道的影像？月球的质量瘤意味着哪些“宝藏”，地球为了少见那样的质量瘤？已经有的科学结论是不是有漏洞？是不是可以有新的解读？在新的解读中，会不会产生新的方法、新的理论？以科学技术为基础、为土壤，可以生长出多少“文化的花朵”、”思想的硕果”？</span></p><p style="text-indent:36px;line-height:150%"><span style="line-height: 150%; color: #333333; letter-spacing: 1px; background: white; font-family: 宋体, SimSun; font-size: 20px;">这些探索，前程往往是不可预测的，“前路无知己”往往是常态；可能是孤独的、危险的旅程。在野外考察中，隐含着各种各样的危险，比如在深山里，在大漠里，在隔壁。即使到了二十一世纪的今天，自然资源的探索者，也仍然需要面临巨大的风险，即使是一个小组也可能应付不了意外情况的发生。典型的例子，比如去年的夏季的马拉松、冬季的哀牢山，出现的亡人事件。</span></p><p style="text-indent:36px;line-height:150%"><span style="font-family: 宋体, SimSun; font-size: 20px;"><span style="font-size: 16px; line-height: 150%; font-family: 宋体; color: #333333; letter-spacing: 1px; background: white;">1</span><span style="font-family: 宋体, SimSun; font-size: 20px;"><span style="line-height: 150%; font-family: 宋体; color: #333333; letter-spacing: 1px; background: white;">980年代的彭加木、马贡泽（某测绘大队队员）、1990年代的</span><span style="line-height: 150%; font-family: 宋体; color: #333333; background: white;">余纯顺</span>，都有孤独前行的时段。在独自面临大自然的时候，个人的力量，都是非常有限的。独自面对一个科学、技术、工程问题呢？</span></span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #333333; background: white; font-size: 20px;">在科学、技术的探索中，在工程建设中，独自前行，可能是勇敢的，也可能是非常危险的。所以，可以看到风光的跟踪者，看到暂时的领跑者、后来的落伍者。意气风发的探险家，却极为少见。闯入新领域的学者，不一定看得到预期的结果。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #333333; background: white; font-size: 20px;">那么，一个学者是不是需要选择孤独的前行？该不该选择进入无人区进行探索、要不要为天下先？这既需要战略眼光，也需要战术能力。想得到的是“好汉”，办得到的也是“好汉”。想到到、办不到，为后人提供思路和借鉴的也是“好汉”。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #333333; background: white; font-size: 20px;">想到了、看到了一个险峰，不敢再没有足迹的地方留下足迹、在没有道路的地方开辟道路，是一种稳妥，也是一种懦弱；反之，敢于冒险、敢于探索，敢于留下第一串脚印的，也许是令人敬佩的英雄，也许是留下笑柄的“莽撞人”。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #333333; background: white; font-size: 20px;">同样，即使不是在探索自然中进入无人区，即使在探索学者自身内心世界时，进入无人区，也是需要巨大的勇气的。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #333333; background: white; font-size: 20px;">学者自身，往往不是完美无缺的。有的意识不到；有的意识到了不愿意承认；有的意识到了、愿意承认却不愿意完善自己；有的意识到了、也承认、也努力了，但无法改变本性，过不了几天就故态复萌。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #333333; background: white; font-size: 20px;">一个在自己科班出身专业作为不大的人，也许会在相关领域突然“发力”、突然“脱颖而出”了；一个在自身专业中的大专家、大牛人，在贸然进入相关领域中，难保不出问题、不栽跟头。这样的例子，古今有、中外有、远近有。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #333333; background: white; font-size: 20px;">既要敢于孤独前行，又要量力而行，的确不易兼顾。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; font-size: 20px;">学者的大悲哀，是在别人规定的半径固定的小圈子里里活动。好似围绕地球质量中心做圆周、或者椭圆轨道运行的卫星？</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; font-size: 20px;">学者的大失控，好似沿着双曲线轨道前行的航天器，一次“离别地球”永无回归之时。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; font-size: 20px;">学者们的新鲜想法、创造激情、现实理智，这三者之间，往往不是并行不悖的。这就需要取舍、需要权衡、需要掂量一下孰轻孰重。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; font-size: 20px;">是像“大山教授”李宝国那样“活着干，死了算”，像“大地之子”黄大年那样认可<span style="line-height: 150%; font-family: 宋体; font-size: 20px; color: #222222; background: white;">“中国人不但脑力上能赢，体力也不比外国人差。”</span>，而且玩命地研究、实践（含推广）？还是悠哉游哉、按部就班地工作、写作、参会呢？ </span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #222222; background: white; font-size: 20px;">不同的学者，当然可以有不同的抉择。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #222222; background: white; font-size: 20px;">抉择的结果，自然也是各自承担。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #222222; background: white; font-size: 20px;">学者的选择，可以是自主的；也可以得到有条件资助的。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #222222; background: white; font-size: 20px;">代价不一样，结局不一样。</span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #222222; background: white; font-size: 20px;">各取所需，各得其所，各修其果。</span></p><p style="text-indent:32px;line-height:150%"><span style="font-size: 20px;"><span style="font-size: 16px; line-height: 150%; font-family: 宋体; color: #222222; background: white;">“</span><span style="font-size: 16px; line-height: 150%; font-family: 宋体; color: #333333; letter-spacing: 1px; background: white;">尽吾志也而不能至者，可以无悔矣”，信哉斯言。</span></span></p><p style="text-indent:32px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; color: #222222; background: white; font-size: 20px;">（北京市海淀区羊坊店街道 凉水河畔，2022-03-24）</span></p><p><br></p>                    <br><br>
