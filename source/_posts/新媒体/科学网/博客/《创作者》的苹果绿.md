<p style="text-align:center"><span style="font-size:21px;font-family:黑体">《创作者》的苹果绿</span></p><p>                                                                           <span style="font-family:宋体">籍利平</span></p><p style="text-indent:28px"><span style="font-family:宋体">《创作者》（张宁</span>.<span style="font-family:宋体">长沙：湖南文艺出版社，</span>2021.<span style="font-family:宋体">）可以说有三个封面，内封面、外封面和腰封（外封面、封底的内容，我记不住；腰封</span> <span style="font-family:宋体">大体记得住）。最简洁明快的要数内封面了。中文书名顶天、作者名立地（字号偏小了一点点儿），英文书名在半空中。内封面、封底都是苹果绿色的。这是我喜欢的一种颜色（有夏季的照片为证）。</span></p><p style="text-indent:28px"><span style="font-family:宋体"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202201/16/121832w0bsmwo8skrnznur.jpg" title alt="创作者  封面.jpg" referrerpolicy="no-referrer"><br></span></p><p style="text-indent:28px"><span style="font-family:宋体"><br></span></p><p style="text-indent:28px"><br></p><p style="text-indent:28px"> </p><p style="text-indent:28px"><span style="font-family: 宋体; font-size: 20px;">通读全书，不难发现苹果绿色字体的文字，在该书中扮演了重要的角色。</span></p><p style="text-indent:28px"><span style="font-family: 宋体; font-size: 20px;">八章的概要，都有以苹果绿色出现的文字。背白、各章首页的主色，也是比内封面颜色淡一些的苹果绿。虽然此书的背白偏多了一点儿，字号偏小了一点儿、行距偏大了一点儿，由于苹果绿作为底色（有人认为，这种颜色的页面，阅读起来不费眼睛），书总体上可以读下去，可以读完。一口气不行，得两三口气。</span></p><p style="text-indent:28px"><span style="font-family: 宋体; font-size: 20px;">既然内封面简洁明快（外封面、腰封暂时不提），内容何不保持一致的风格？以《推荐序》为例，徐新的文字，完全可以排在一页之内；前言和周源《关于创作的解答》，同样也可以在两页之中容纳。</span></p><p style="text-indent:28px"><span style="font-size: 20px;"><span style="font-size: 20px; font-family: 宋体;">每一章的首页，既然底色是淡淡的苹果绿了，文字为何不都安排成其他颜色？比如第一章首页，黑色的“个人垄断：创作从何开始”下面的“所有的创作，都是从</span>′<span style="font-size: 20px; font-family: 宋体;">你</span>′ <span style="font-size: 20px; font-family: 宋体;">开始的。创作，就是用你擅长的方式，表达你想要表达的内容。”虽然使用了较深的苹果绿，但和底色相比，还是差别不大，不够醒目。</span></span></p><p style="text-indent:28px"><span style="font-family: 宋体; font-size: 20px;">浏览、阅读全书后，不难产生这样的感觉，以苹果绿字体呈现的文字，在全书里起到了提纲挈领的作用。重温它们，不难回忆起该书的主要内容。</span></p><p style="text-indent:28px"><span style="font-family: 宋体; font-size: 20px;">《创作者》以个人垄断、社交资本、混剪、梗、内容品牌、订阅、媒体升级、涌现这八个关键词为纲，为创作者打开了思路、推荐了兵器。</span></p><p style="text-indent:28px"><span style="font-size: 20px;"><span style="font-size: 20px; font-family: 宋体;">正如书的倒数第一页所写到的那样“每个人的创作都是他过去生活经验的总和”。在写这个读后感的时候，我已经有了连续</span>14<span style="font-size: 20px; font-family: 宋体;">年（</span>2007<span style="font-size: 20px; font-family: 宋体;">——</span>2021<span style="font-size: 20px; font-family: 宋体;">）年的博客书写史。我经营过的博客有四五个，坚持最久的是科学网的《籍工场》。这个博客以发布原创内容为主，偶尔也转载一些好文章。这个博客的开始和坚持，都有较好的“历史背景”；或者说，占据了一定的先机。</span>2007<span style="font-size: 20px; font-family: 宋体;">年，我是《科学时报》的订户，报纸的首页多次刊登要求科技工作者入驻科学网的“函件”。阅读的次数多了，我就不自量力了，以为函件的收件人有我，我有义务进入。不知天高地厚的我，进入了，开了博客了（骑虎难下）。从博客的</span>ID<span style="font-size: 20px; font-family: 宋体;">上可以看出，我进入时的博客用户还不到</span>300<span style="font-size: 20px; font-family: 宋体;">个。</span></span></p><p style="text-indent:28px"><span style="font-size: 20px;">2007<span style="font-size: 20px; font-family: 宋体;">年，正好是我有话想说的时候，也是有一定富裕时间的时段，更是我作为笔耕者比较勤奋的年头。那时候，经常到北京郊区的山丘上值班。一般是三天不下山，值班的余暇——比如早饭前的一个多小时，可以一边听音乐、一边更新博客。这些博客，都是随性写的，由于使用智能</span>ABC<span style="font-size: 20px; font-family: 宋体;">录入，贴出去的博文中往往夹杂着白字或者冗余的字。好在，那时的读者和网文编辑都比较宽容，也会直接留言指正。博客的及时互动性，的确让我受益。读者、编辑们的眼睛是雪亮的，帮助我改正了不少文字上的错儿。</span></span></p><p style="text-indent:28px"><span style="font-family: 宋体; font-size: 20px;">科学网对所有的博主既然爱护，又鼓励。</span><span style="font-family: 宋体; font-size: 20px;">当时，</span><span style="font-family: 宋体; font-size: 20px;">点击量过十万，就是一个阶梯了。后来是百万（已经达到）、千万（我还没有达到）。点击总量过百万之后，我感觉自己有些松劲儿了。似乎要说的话，都说完了，再说恐怕就要重复。按照《创作者》的说法，就是内容的持续性出现了障碍。“可持续的创作进化”没有规划，当然就谈不上进化的策略了。</span></p><p style="text-indent:28px"><span style="font-size: 20px;"><span style="font-size: 20px; font-family: 宋体;">我分享的，无非是个人经历、从事的工作——比如卫星激光测距。在第一篇博文里我说“</span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">物理早已不是牛顿、伽利略的时代了，也不是爱因斯坦的时代了，甚至也不是杨振宁时代了。现代的物理已经不是个人时代了，是团队时代，但更需要个人的奋斗。”（此句上过科学网首页）</span> <span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">在第二篇博文《卫星激光测距简介》里，我写：“每当天气晴朗的夜晚，当安装了反射器的卫星像排好了队从天空掠过的时候，我们就用发出绿色激光（波长</span><span style="font-size: 20px; font-family: Tahoma, sans-serif; color: #333333; background: white;">532</span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">纳米，一纳米等于</span><span style="font-size: 20px; font-family: Tahoma, sans-serif; color: #333333; background: white;">10</span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">的负九次方米）的天文望远镜瞄准它、跟踪它。瞄准、跟踪都是依靠计算机操作的，观测者的任务就是修正误差，以便于得到更多的可以利用的信息。在观测室内（楼上）的微光电视上，卫星，或者为我们指明方向的北极星就是一个发亮的白点。跟踪卫星的时候，可以看到许多明亮的恒星飞速地从屏幕上掠过，划出美丽的、转瞬即逝的线条。当东天边露出了朝阳的微光，当星星们闭上了眼睛，疲惫的我们也该休息了。《日出》中有句诗：太阳出来了，可是我们要睡觉。这真是我们夜工作者心态的真实写照。我们的心情当然和陈白露截然不同了。她是空虚地睡去，我们是充实地休息，养精蓄锐。”</span> <span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">这些句子的原文，出自我写给一个广东中学生（笔友）的回信。这样的文学风格，当然不会保持太久。</span> <span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">现在还可以看到，一位署名江老师的同行留言：我是中国科学院安徽光学精密机械研究所的科研人员，我对您卫星激光测距简介十分感兴趣，其中提到“我们使用的激光器每秒只能发射</span><span style="font-size: 20px; font-family: Tahoma, sans-serif; color: #333333; background: white;">10</span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">次脉冲，可以称之为</span><span style="font-size: 20px; font-family: Tahoma, sans-serif; color: #333333; background: white;">10</span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">赫兹；而西方一些国家的激光器脉冲频率达到了</span><span style="font-size: 20px; font-family: Tahoma, sans-serif; color: #333333; background: white;">1000</span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">赫兹，脉冲频率和返回的数据成正比。这方面的先进技术，他们不会轻易转让的，还得自力更生、艰苦奋斗哪！”</span> <span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">实际上，我们已经研制了</span><span style="font-size: 20px; font-family: Tahoma, sans-serif; color: #333333; background: white;">1000</span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">赫兹脉冲激光，可以用于卫星激光测距</span><span style="font-size: 20px; font-family: Tahoma, sans-serif; color: #333333; background: white;">……  </span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">当时，用于卫星激光测距的激光脉冲频率普遍较低，千赫兹、万赫兹频率的激光脉冲用于卫星激光测距，是近几年的事情。这个专家的留言，让我大受激励。</span></span></p><p style="text-indent:28px"><span style="font-family: 宋体; color: #333333; background: white; font-size: 20px;">获取“社交资本”之后，如何发展、如何壮大，是每一个创作者必须面对的现实问题。加关注、添加好友、留言、评论、发站内信息等，都对创作者有一定的影响。科学网中有科学大咖，有的是隐身的、有的神龙见首不见尾（后来要求实名制，就“露馅”了）。</span></p><p style="text-indent:28px"><span style="font-size: 20px;"><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">科学网起初没有开通视频频道，到现在也是以文字发布为主。我也是以文字为主、图片为辅来经营博客的。在“内容品牌”（姑且把“籍工场”看成一个品牌</span> <span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">）的可持续发展方面，在创作进化方面，我有心尝试引入音频和视频，可是总觉得不如文字使用起来得心应手。视频、混剪、梗的引入，的确可以提高品牌的影响力，也许可以成数量级地扩散开去，可是，文字的单一性，让它也有优势——比如便于操作。梗的出现，好像是可遇不可求的？音频、视频的录制和剪辑，都比纯文字操作难多了，也会花费更多的时间。对于我这样业余、玩票的“博客写手”来说，无疑存在较高的门槛。另外，本人形象对不起观众，影响了市容、城管执法部门来介入了怎么办？</span> <span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">不如消停一些，就用文字吧。这个坚持十几年，也不易呀！</span></span></p><p style="text-indent:28px"><span style="font-size: 20px;"><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">笔者以为，《创作者》的第五章内容可以更丰富一些，虽然看上去会让全书的结构略显失衡。“</span><span style="font-size: 20px; font-family: 宋体; color: #00B050; background: white;">选择的关键在于内容的受众类型，以及内容被消费的环境。受众是不是想要主动参与，一方面是他们的主观意愿，另一方面也受到消费环境的影响</span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">。”这里引用了一段字体较大的苹果绿文字。本书的苹果绿文字，几乎都有引用的必要（限于篇幅，收敛一些吧。）在内容的制作和发布中，我发现一个有趣的现象，就是自己的（被推荐、受好评）预期和网络编辑、和读者，往往是错位的。有时候自感良好的博文，没有受到关注；被编辑推荐的文章不全是自己预期的那些文章。这说明，我的表达和书写有一定的盲目性、随意性。那些带有科普性质的文章、科学含量较高的文章，有的写起来顺手，有的写起来卡过壳儿；有的长、有的短；看上去不够规整。如果早些得到《创作者》的思路和启迪，经营效果肯定会好许多。虽然书中的一些策略，我无法逐一尝试，思路和视野毕竟拓宽了。</span></span></p><p style="text-indent:28px"><span style="font-family: 宋体; color: #333333; background: white; font-size: 20px;">这是我要感谢作者、感谢推荐者（老同学）的地方。</span></p><p style="text-indent:28px"><span style="font-size: 20px;"><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">在和老同学交流时，我提到过一部巨著——《创造者》（布尔斯廷著，有中文版）。《创作者》提到过《创新者》一书，这三本书多半不是同一类型的书，但书名中都有“创”字。这里，笔者要引用十多年前的“籍工解字”了</span> <span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">：</span><span style="font-size: 20px; font-family: 宋体; color: #00B050; background: white;">创新的创有意思，右边是明晃晃的一把直立的刀；新的右半边是斤，与斧头的斧的下半截一致。开拓创新，离不开大刀阔斧，离不开金属的参与。创新，需要剔除头脑里的杂草。新鲜的思想才能茂盛地生长。创新无疑具备挑战性，所以也更具备吸引人的无穷魅力。热爱创造的人，沉醉于创新，达到了废寝忘食，绞尽脑汁、忘我的境界。</span></span></p><p style="text-indent:28px"><span style="font-family: 宋体; color: #333333; background: white; font-size: 20px;">一句话，“创字辈”的书值得读，“创字辈”的人值得做。做“创字辈”的人，写出“创字辈”的书，有双重的艰辛，也会有双重的收获。</span></p><p style="text-indent:28px"><span style="font-family: 宋体; color: #333333; background: white; font-size: 20px;">解字容易写字难，写有创意的文字更难。因为难，所以更有诱惑力。</span></p><p style="text-indent:28px"><span style="font-size: 20px;"><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">（</span><span style="font-size: 20px; font-family: Tahoma, sans-serif; color: #333333; background: white;">2021-12-01</span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">，写于北京市海淀区。</span><span style="font-size: 20px; font-family: Tahoma, sans-serif; color: #333333; background: white;">2022-01-16</span><span style="font-size: 20px; font-family: 宋体; color: #333333; background: white;">发布。）</span></span><span style=";font-family:'Tahoma','sans-serif';color:#333333;background:white"> </span></p><p><br></p>                    <br><br>
