<p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 25px; word-break: break-word; color: rgb(51, 51, 51); font-family: -apple-system, "SF UI Text", Arial, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "WenQuanYi Micro Hei", sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">工作这几年，感受之一就是现在的学生都不会写邮件，换句话说，不是不会写，是不愿意认真写......</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 25px; word-break: break-word; color: rgb(51, 51, 51); font-family: -apple-system, "SF UI Text", Arial, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "WenQuanYi Micro Hei", sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">比如之前带学生做项目需要做human practise，于是我告诉了学生几位相关教授的名字，让他们自己去发邮件，询问人家对项目的意见，很久之后，学生对我说教授都不回他们的邮件，希望我替他们去联系，我让他们把邮件发给我看看，看过之后，我对他们说：”我看完这封邮件，我也不回”，邮件大概是这么写得：“我们在做一个项目，题目是*******，我们有些问题想向你咨询，请问你什么时候有空跟我们视频。”无语！第一，至少简单介绍项目的内容；第二，具体想咨询什么问题，列出来；第三，这里面的说法有把主动权给对方吗？一个教授为啥非得跟你们几个学生视频，应该是“如果您觉得视频交流比较好，我们随时方便”之类的话语把主动权交给对方，而不是这种“强制性”的问时间，人家当然不会回复.......</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 25px; word-break: break-word; color: rgb(51, 51, 51); font-family: -apple-system, "SF UI Text", Arial, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "WenQuanYi Micro Hei", sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">比如每年考研之后都会收到那种群发的邮件，“我对您的课题给出感兴趣，我想做生物信息学”，拜托我不是做生物信息学的，能不能再给我发邮件之前，稍微了解一下我的研究方向。这种邮件我都是无视的......</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 25px; word-break: break-word; color: rgb(51, 51, 51); font-family: -apple-system, "SF UI Text", Arial, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "WenQuanYi Micro Hei", sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">比如我之前收到一封邮件，主题是“第一次作业”，邮件的附件名是一个英文人名，正文中没有任何信息量，也没有出现发邮件者的名字，我第一感觉是钓鱼邮件，由于那个QQ号的发信邮箱，我一时没删除，课堂上问了一下，是不是哪位学生发给我的，还真的是学生发的......</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 25px; word-break: break-word; color: rgb(51, 51, 51); font-family: -apple-system, "SF UI Text", Arial, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "WenQuanYi Micro Hei", sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">再比如，这个中旬的时候收到一封邮件让我初审几个学生的项目，邮件里只写了一句话：“这里是评审材料”，附件就是一个压缩包（我先不说压缩包是最容易携带病毒的），没有任何其他信息，我就把这项工作排在五一假期了，然后28号收到一封邮件，说这个初审的截止日期是25号，我才发现，截止日期是在写在压缩包里的一个文档里的（我还没有打开过那个压缩包），这种信息不是应该在写在邮件里吗？</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 25px; word-break: break-word; color: rgb(51, 51, 51); font-family: -apple-system, "SF UI Text", Arial, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "WenQuanYi Micro Hei", sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">学生用微信、QQ这种即时沟通工具太多，都忘记了正式的信或者邮件应该如何去写，甚至忘记正式的语言表达，都是不完整的只言片语......</p><p style="box-sizing: border-box; margin-top: 0px; margin-bottom: 25px; word-break: break-word; color: rgb(51, 51, 51); font-family: -apple-system, "SF UI Text", Arial, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "WenQuanYi Micro Hei", sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">这里我举得都是内容上的问题，还有诸多格式问题，相比内容，可能格式上的就显得次要了！</p><p><br></p>                    <br><br>
