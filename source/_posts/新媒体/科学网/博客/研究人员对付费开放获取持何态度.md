text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">根据一项对付费出版模式的态度的国际调查结果，关于文章处理费（APCs）这一学术话题的讨论在研究人员之间十分常见。研究发现，作者们已经对这种新的商业模式可能带来的弊端提出了警示，且这一担忧已蔓延至全世界大多数学者身上。至少，这是就这一制度的全球普遍后果而非就特定后果而言。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">在全球范围内，被调者表示，他们至少部分同意付费出版 “损害或减缓科学进步”的观点。然而，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">当被问及他们是否觉得这种模式“减缓或损害了自己的科学事业”时，他们的态度就不那么坚决了，他们中的大多数人并不觉得受到APC模式的特别影响：</span>他们既不同意也不反对这种说法。因此，大多数学者似乎认为其他人正在遭受这种出版制度带来的最坏后果，而他们则属于幸运者。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">资金是一个关键问题</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">该研究还探讨了不同群体对付费出版模式的看法。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">在学科差异方面，艺术与人文学科和社会科学领域的学者普遍比生命科学和STEM领域的学者对付费出版怀有更有敌意。由于传统上，人文和社会科学与其他领域相比经常出现资金短缺的情况，他们认为付费出版模式所带来的额外支出对自己的损害比对其他资金更充足的领域的学者更大。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><img src="https://p9.itc.cn/images01/20220817/74c6af37de39486db463479fb73f75cc.png" style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; height: auto;" referrerpolicy="no-referrer"></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">社会科学和人文科学的研究人员也更倾向于拒绝在要求作者支付费用的期刊上发表文章。这可能是出于资金问题，但在某些学科，如法律和艺术，可能与学者们更普遍地接受期刊之外的其他学术交流方式有关，如书籍专著等。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">对付费出版模式的看法也受到研究人员所在国家的收入水平的制约。那些来自世界银行收入年度报告中排名较落后的国家的学者对付费出版的态度普遍更负面一些。</span>缺乏获得外部资金的机会导致对付费出版模式的意见更差，因为60%来自低收入国家的研究人员由于缺乏资金而不得不用自己的钱来支付这些出版费。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">年轻学者们的态度也更具批判性。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">年轻研究人员往往较难获得资助，因此他们不信任这个系统，也不太愿意接受这种付费发表模式。</span>在经济框架之外，年轻学者之间的反对情绪在26-35岁的人群中更强烈。这可能是由于该群体已在科研环境中获取了一些经验，足以让他们认识到付费出版模式的结构性后果。而他们中的大多数人没有终身教职，也没有定期获得外部资助，从而使他们最初对这个模式的批评变得更加尖锐。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">该项研究还调查了受访者对开放获取出版的投入程度，也就是说，他们将自己发表的文章上传到某个资料库，以便他人自由阅读和获取的频率。作者们一开始预测，较高的投入程度（例如，即使发表的期刊命令禁止上传他们的文章）会导致对付费出版模式的批评。结果却恰恰相反，研究并未在这一指标上发现任何差异。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">本地以及全球观念之间的分歧</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">正如前文已经提到的，很多学者对付费出版的敌意其实更多是由一些实际问题导致的（必须支付出版费的额外负担），而非 “意识形态”或“道德”层面的问题，因为研究发现对开放获取的投入与对APC的敌意之间并没有联系。然而，大多数科学家都同意这样一个事实，即在全球范围内，付费出版对科学的发展是有害的，尽管这对他们个人没有影响。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">从积极的角度来看，研究人员似乎意识到了APC制度中固有的不公平性。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">尽管大多数学者认为他们可以克服其所带来的经济方面的要求，但同时也认识到这是一个不可持续的模式。</span>更有特权的研究人员则并没有忽视那些无法获得这些机构资源的人所面临的巨大惩罚，从而使科学界最不稳定的成员反而受到了更大的影响。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">从消极的角度看，认为付费出版不会威胁到学者的特定职业或专业晋升，这可能会阻碍研究人员对该制度采取更为激进的态度，如抵制出版社或拒绝向这类期刊投稿。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">不管怎么说，普遍存在的“事不关己高高挂起”的态度似乎是由学者职业生涯的其他结构性因素造成的，如专业晋升的评估。因此，处于任期内的研究人员可能会承认，如果他们想在事业上不断进步，APC是不可避免的。这就提出了一个重要的问题：<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">一旦年轻的、更具批判性的一代学者获得了终身教职，他们是否会将批评转化为实际措施，还是说他们在更容易获得资金之后，就淡化了对付费出版模式的担忧呢？</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em;
