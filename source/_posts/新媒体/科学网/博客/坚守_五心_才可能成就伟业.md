<p style="text-indent: 32px;line-height: 200%"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">开国伟人在解放战争时期曾说过一句名言：“五心不定，输个干干净净。”在我看来，这句话对在科技创新之路上奋斗的研究者也同样适用且大有裨益。为此，本文以“五心”为题，简述之以起到鼓舞士气之目的。</span></p><p style="text-indent: 32px;line-height: 200%"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">关于“五心”的定义有不同版本，本文把之定义为好奇心、自信心、细心、潜心与定心。</span></p><p style=";margin-bottom:0;line-height:200%;background:white"><span style="font-size: 16px;"><span style="font-size: 16px; font-family: "times new roman";"><strong>1. </strong></span><span style="font-size: 16px; font-family: 宋体, SimSun;"><strong>好奇心</strong></span></span></p><p style="margin-bottom: 0px; text-indent: 32px; line-height: 200%; background: white;"><span style="line-height: 200%; font-family: 宋体; font-size: 16px;">好奇心是人类的天性，是人类探索未知的原动力。研究者永葆好奇心，才能以另类思维和逆向思维为向导、以“格物致知以穷其理”之精神、以“知其然还要知其所以然”之钻劲、以“越是艰险越向前”之毅力，找到破解科学难题的正确途径，揭示描述某类事物演化的普适规律，给人类认识世界的知识库添砖加瓦，为自己的科研人生画上圆满的句号。</span></p><p style="text-indent:32px;line-height:200%;background:white"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">纵观科学史，诸多科学大师凭着对未知世界的好奇心，创立了已载入史册的伟大原创理论。例如，伽利略看到吊灯摇晃而好奇，发现了单摆等时定律；牛顿受苹果落地现象启发，发现了</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: windowtext;">万有引力定律</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">；魏格纳在病中看世界地图时突发奇想，创立了大陆漂移说。这样的实例，不胜枚举。爱因斯坦也曾说：“我并非天赋异禀，我只是有强烈的好奇心。” </span></span></p><p style=";margin-bottom:0;line-height:200%;background:white"><span style="font-size: 16px;"><strong><span style="font-size: 16px; font-family: "Times New Roman", serif;">2. </span></strong><strong>自信心</strong></span></p><p style="margin-bottom: 0px; text-indent: 32px; line-height: 200%; background: white;"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%; font-weight: normal;">科研活动中，失败是常态，而成功则较为鲜见。多数人在失败后会垂头丧气、怨天尤人，甚至觉得自己不是做科研的“料”而逃离科研。然而，失败是成功之母，只要研究者从中总结经验，汲取教训，重拾信心，则可能从“山重水复</span>疑无路”中，找到“柳暗花明又一村”。</span></p><p style="margin-bottom: 0px; text-indent: 32px; line-height: 200%; background: white;"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%; color: #121212; font-weight: normal;">自信心源于解决问题的成功激励，源于日积月累的经验提炼，源于持续不断的自我反思。</span><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%; color: #333333; font-weight: normal;">研究者在不断克服困难、取得成功、超越自我的过程中，会产生强烈的成就感，导致自信心逐步增强，以至于形成迎难而上的习惯。显然，克服的困难越大，取得的成功越大，研究者的自信心越强。</span></span></p><p style="margin-bottom: 0px; text-indent: 32px; line-height: 200%; background: white;"><span style="font-family: 宋体, SimSun; line-height: 200%; font-weight: normal; font-size: 16px;">从事原创科研并取得原创成果，研究者一定是源于独辟蹊径的奇思妙想，一定是克服了诸多难以想象的艰难险阻，一定是超越自我的结果，这必然能大幅提升其自信心。所以说，强大的学术自信源于原创性成果，而做跟风式科研则难以产生学术自信，这是因为不管自己如何改进完善前人的工作，都是为原创者做更漂亮的嫁衣，自己永远只能充当跑龙套的角色。</span></p><p style="line-height: 200%"><span style="font-size: 16px;"><strong><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #333333;">3</span></strong><strong>. </strong><strong><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">细心</span></strong></span></p><p style="text-indent: 32px;line-height: 200%"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">细节决定成败，因而细心是研究者应具备的基本素养。</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #121212; background: white;">细心是发现问题的前提，是实施研究的基石，是反思研究的保障</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">，是取得成功的必需。例如，做实验的研究者，制订实验方案要细心，在实验过程中要细心观察，在数据处理中要细心分析，还要特别注意一些与预期不同的奇异现象，因为这可能是重要发现的摇篮；做理论研究的研究者，在构建理论的过程中，要注意假设的合理性，因素的主控性，推导的严密性，验证的无偏性，万万不能马虎。</span></span></p><p style="line-height: 200%"><span style="font-size: 16px;"><strong><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #333333;">4. </span></strong><strong><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">潜心</span></strong></span></p><p style="text-indent: 32px;line-height: 200%"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">因为低垂的果实已几乎被先行者摘光，留给我们的基本都是难以采摘的果实。这意味着攻克任何科技难题，都需要研究者长期潜心钻研才可能实现。确实，从凝练出关键问题、找到突破口、揭示机制和建模，到完善乃至实战，绝不是一朝一夕所能做到的。这就要求研究者须耐得住寂寞、坐得住冷板凳，以十年磨剑精神打造自己的智慧之剑、锋利之剑、胜利之剑。</span></p><p style="line-height: 200%"><span style="font-size: 16px;"><strong><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #333333;">5. </span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">定心</span></strong></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #222222;">在这个物欲横流的年代，若研究者把追求功名利禄作为自己的首要目标，必然追逐易上手的内卷工作，而不愿定心攻坚克难，故不可能做出卓越成果，到头来只能为虚度此生徒留遗憾。要为子孙后代留下念想，研究者需要志向坚定，心如止水，</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">把科研当成事业，把追求卓越成果作为动力</span>，<span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #222222;">以“不媚权、不媚上、不媚名、不媚利、不媚俗”之节操，在滚滚红尘中特立独行；以“虽千万人吾往矣、虽千里无人吾也往矣”之勇气，耕耘在“无人区”，攻坚在“最高峰”，取胜于行稳致远。</span></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体; color: #333333; font-size: 16px;">总之，“五心”不定，万事莫成；“五心”坚定，前景可期。心有所念，行方有向；心行合一，所向披靡。但愿有志研究者坚守“五心”，在科技创新之路上谱写新的篇章。</span></p><p style="line-height: 200%;background: white"><span style="font-size: 16px;"><strong><span style="font-size: 16px;
