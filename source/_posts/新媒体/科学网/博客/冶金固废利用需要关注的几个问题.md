color: #333333; letter-spacing: 0px; font-size: 18px;">工艺流程：螺旋分级机主要有高堰式单螺旋和双螺旋、沉没式单螺旋和双螺旋四种分级机。该分级机广泛适用于铁矿石选矿厂中与球磨机配成闭路循环程分程分流矿沙，或用在铁矿石重力选矿厂中来分级矿砂和细泥，铁矿石加工工艺流程中对矿浆进行粒度分级，以及洗矿作业中的脱泥、脱水等作业。</span></p><p style="margin-bottom: 12px;margin-left: 0;text-indent: 28px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">破碎与磨矿都是选别前矿石的粉碎作业，只是产品粒度不同而已。物料经过破碎的次数为破碎阶段的段数。接下就是选矿。</span></p><p style="margin-bottom: 12px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">2 铁矿石选矿</span></p><p style="margin-bottom: 12px;margin-left: 0;text-indent: 28px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">选矿的方法很多。</span></p><p style="margin-bottom: 12px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">2.1重选法</span></p><p style="margin-bottom: 12px;margin-left: 0;text-indent: 28px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">是根据矿物相对密度(通常称比重)的差异来分选矿物的。铁矿石重选分选工艺其应用有两种：一种是矿床地质品位较高，但矿体较薄或夹层较多，采矿时废石混入，使矿石贫化，对这种矿石可采用只破碎不磨矿，在粒度较粗的情况下，通过重选丢弃粗粒尾矿，从而恢复地质品位。获得粗粒的中等品位精矿，或进一步加工处理，或直接送高炉冶炼，这种称为粗粒重选。另一种对嵌布粒度较细的红矿或混合矿，即含弱磁性的红矿和强磁性的磁铁矿，经磨碎，磨矿使铁矿物单体分离后，通过重选或磁重联选，得到细度高品位精矿，这种称为细粒重选。</span></p><p style="margin-bottom: 12px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">2.2磁选</span></p><p style="margin-bottom: 12px;margin-left: 0;text-indent: 28px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">是利用各种矿物磁性的差异，在磁选机的磁场中进行分选的一种选矿方法。磁选又分为弱磁选和强磁选，弱磁选主要用于分选磁铁矿、钒钛磁铁矿等强磁性铁矿物，强磁选主要用于分选以赤铁矿为主的弱磁性矿物。弱磁选的分选流程主要是采用阶段磨矿-阶段磁选，在每一个磨矿阶段抛出不合格尾矿，以尽量减少下一阶段的再磨量，从面达到节能降耗、提高选矿厂经济效益的目标。我国多数磁铁矿选矿厂采用两段或三段磨矿、阶段分选流程，即在各段磨矿作业之后用磁选机或脱水糟加磁选机抛出已单体解离的脉石矿物。</span></p><p style="margin-bottom: 12px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">2.3电选</span></p><p style="margin-bottom: 12px;margin-left: 0;text-indent: 28px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">是在高压电场中利用矿物的电性差异使矿物分离选矿方法，用的不是太普遍。</span></p><p style="margin-bottom: 12px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">2.4选别细粒浮选法</span></p><p style="margin-bottom: 12px;margin-left: 0;text-indent: 28px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; letter-spacing: 0px; font-size: 18px; font-family: 宋体;">炼铁时用的铁矿石，主要有赤铁矿石(主要成分是氧化铁)和磁铁矿石(主要成分是Fe</span><sub><span style="color: #333333; letter-spacing: 0px; vertical-align: sub; font-size: 18px; font-family: 宋体;">3</span></sub><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 14px;">)。炼铁时，被还原出的铁在高温下变成液体，温度在1535℃左右，原燃料中的SiO</span><sub><span style="color: #333333; letter-spacing: 0px; vertical-align: sub; font-size: 18px; font-family: 宋体;">2</span></sub><span style="color: #333333; letter-spacing: 0px; font-size: 18px; font-family: 宋体;">、Al</span><sub><span style="color: #333333; letter-spacing: 0px; vertical-align: sub; font-size: 18px; font-family: 宋体;">2</span></sub><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 14px;">O</span><sub><span style="color: #333333; letter-spacing: 0px; vertical-align: sub; font-size: 18px; font-family: 宋体;">3</span></sub><span style="color: #333333; letter-spacing: 0px; font-size: 18px; font-family: 宋体;">等酸性氧化物的熔点很高（SiO</span><sub><span style="color: #333333; letter-spacing: 0px; vertical-align: sub; font-size: 18px; font-family: 宋体;">2</span></sub><span style="color: #333333; letter-spacing: 0px; font-size: 18px; font-family: 宋体;">为1713℃，Al</span><sub><span style="color: #333333; letter-spacing: 0px; vertical-align: sub; font-size: 18px; font-family: 宋体;">2</span></sub><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 14px;"><span style="font-size: 18px; font-family: 宋体;">为2050℃左右），不可能在高炉中熔化。即使它们有机会组成较低熔点的化合物，其熔化温度仍然很高（约1545℃），在高炉中只能形成一些非常粘稠的物质，造成渣、铁不分，难于流动。为了除去这种杂质，选用石灰石作熔剂，石灰石在高温下分解成</span>CaO<span style="font-size: 18px; font-family: 宋体;">和CO</span></span><sub><span style="color: #333333; letter-spacing: 0px; vertical-align: sub; font-size: 18px; font-family: 宋体;">2</span></sub><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 14px;">。尽管熔剂中的CaO<span style="font-size: 18px; font-family: 宋体;">和MgO自身的熔点也很高（</span>CaO<span style="font-size: 18px; font-family: 宋体;">为2570℃，MgO为2800℃），但它们能同SiO</span></span><sub><span style="color: #333333; letter-spacing: 0px; vertical-align: sub; font-size: 18px; font-family: 宋体;">2</span></sub><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 14px;">O</span><sub><span style="color: #333333; letter-spacing: 0px; vertical-align: sub; font-size: 18px; font-family: 宋体;">3</span></sub><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 14px;"><span style="font-size: 18px; font-family: 宋体;">结合成低熔点（<1400℃）化合物，在高炉内足以熔化，形成流动性良好的炉渣，按相对密度与铁水分开（铁水相</span><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;"><span style="font-size: 18px; font-family: 宋体;">对密度6.8-7.8，炉渣2.8-3.0），</span>CaO在高温下与二氧化硅反应生成熔点比铁水温度还低的硅酸钙，与三氧化二铝生成铝酸钙，打开高炉上的出渣口，液态硅酸钙先流出去，水淬凝固成高炉渣，粉磨后形成矿渣粉，应用于混凝土中。</span></span></span></p><p style="margin-top:16px;margin-right:0;margin-bottom:8px;margin-left:0;text-indent:22px;padding:0 0 0 0 ;line-height:21px;background:rgb(255,255,255)"><span style="font-size: 18px;"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 14px;"><span style="font-size: 18px; font-family: 宋体;">钢渣是转炉炼钢产生的一种废渣，是出炉状态温度高达1400℃以上的液体，化学成分</span>CaO<span style="font-size: 18px; font-family: 宋体;">含量为40%-60%，SiO</span></span><sub><span style="color: #333333; letter-spacing: 0px; vertical-align: sub; font-size: 18px; font-family: 宋体;">2</span></sub><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 14px;">S、钙铁橄榄石、f-CaO<span style="font-size: 18px; font-family: 宋体;">、f-MgO等。f-</span>CaO<span style="font-size: 18px; font-family: 宋体;">、f-MgO等有害成分含量高，冷却处理后的钢渣中含有大量结晶粗大、结构致密的f-</span>CaO<span style="font-size: 18px; font-family: 宋体;">和离MgO，这些f-</span>CaO<span style="font-size: 18px; font-family: 宋体;">和游离MgO遇水后会在很长时间内持续水化并发生体积膨胀。</span></span></span></p><p style="margin-top:16px;margin-right:0;margin-bottom:8px;margin-left:0;text-indent:22px;padding:0 0 0 0 ;line-height:21px;background:rgb(255,255,255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">由炼钢工艺决定了钢渣中，游离氧化钙、游离氧化镁不可避免地存在，其中不乏过烧的氧化钙和氧化镁，后期会缓慢与水发生反应，膨胀开裂，因此，作为钢渣不能用作混凝土中的骨料。</span></p><p style="margin-bottom: 12px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">4结语</span></p><p style="margin-bottom: 12px;margin-left: 0;text-indent: 28px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">在冶炼前选矿阶段，物理分选产生的尾矿对粒型和级配没有要求的，应该进行性整形和和筛分后利用，由于这个过程中本身没有受到化学药剂的影响，满足骨料指标要求的都可以在混凝土中利用，而浮选和化学选矿产生的废料，由于化学药剂的影响，要进行相应的试验验证才能使用，当然，在利用之前还要进行放射性和溶出性毒性等影响健康的试验检测。</span></p><p style="margin-bottom: 12px;margin-left: 0;text-indent: 28px;background: rgb(255, 255, 255)"><span style="font-family: 宋体; color: #333333; letter-spacing: 0px; font-size: 18px;">冶炼后产生的废渣，一定要了解主要成分和含量，主要是长期的体积稳定性的是否可以满足要求，要慎之又慎。</span></p><p><br></p>                    <br><br>
