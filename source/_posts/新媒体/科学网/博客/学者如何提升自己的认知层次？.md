<p style=";margin-bottom:0;text-indent:32px;line-height:200%;background:white"><span style="line-height: 200%; color: #333333; font-family: 宋体, SimSun; font-size: 16px;">前几天，我无意中看到了一篇</span><a href="https://xueqiu.com/4649450597/207788558" target="_blank" style="line-height: 200%; color: rgb(51, 51, 51); font-family: 宋体, SimSun; font-size: 16px; text-decoration: underline;"><span style="line-height: 200%; color: #333333; font-family: 宋体, SimSun; font-size: 16px;">文章</span></a><span style="line-height: 200%; color: #333333; font-family: 宋体, SimSun; font-size: 16px;">，其把人们的认知分为四个层次：<strong>不知道自己不知道</strong>，<strong>知道自己不知道</strong>，<strong>知道自己知道</strong>，<strong>不知道自己知道</strong>。我觉得这种分类相当有趣且靠谱，基于此并做适当发挥写成此文，以让诸君在研究之路上少走弯路。</span></p><p style=";margin-bottom:0;text-indent:32px;line-height:200%;background:white"><span style="font-family: 宋体, SimSun; font-size: 16px;"><strong><span style="font-size: 16px; font-family: 宋体, SimSun; color: #333333;">第一层次：不知道自己不知道</span></strong></span></p><p style=";margin-bottom:0;text-indent:32px;line-height:200%;background:white"><span style="line-height: 200%; color: #333333; font-family: 宋体, SimSun; font-size: 16px;">处于这种层次的人士，想当然认为自己无所不知、无所不能，从而呈现出极度盲目自信的巨婴行为，但实际上不能分清表象与本质、臆想与事实、客观与主观等——缺乏基本常识和判断能力。此种人士通常为民间科学爱好者或被其招安的糊涂学者。</span></p><p style=";margin-bottom:0;text-indent:32px;line-height:200%;background:white"><span style="font-family: 宋体, SimSun; font-size: 16px;"><strong><span style="font-size: 16px; font-family: 宋体, SimSun; color: #333333;">第二层次：知道自己不知道</span></strong></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; color: #333333; font-family: 宋体, SimSun; font-size: 16px;">处于这种层次的人士，明白知识的无边性与自己认知的局限性，通达“山外有山、天外有天、人外有人”的道理，始终对未知领域保持敬畏和好奇。此种人士主要为普通学者，但未来有巨大的上升空间。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-family: 宋体, SimSun; font-size: 16px;"><strong><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">第三层次：知道自己知道</span></strong></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; color: #333333; font-family: 宋体, SimSun; font-size: 16px;">处于这种层次的人士，清楚自己认知和能力的“天花板”，因而定位明确——仅在某一专业方向深耕和发表见解；能解决难度不大的问题，在业内有一定的影响力。此种人士可称为业内行家。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-family: 宋体, SimSun; font-size: 16px;"><strong><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">第四层次：不知道自己知道</span></strong></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">处于这种层次的人士，善于融会贯通所学知识，并把这种本领作为自己随时可用的利器；</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #1F1F1F;">善于从多元、多层次角度看待问题，并能抓住问题的本质；善于靠直觉和本能探寻解决问题之道；善于以化繁为简之术找到破解问题的“金钥匙”；善于以知识+灵感+理性+实证攻克问题。此外，必要时，此种人士可轻松自如地“降维直击”别人的错误认知，使其豁然开朗，犹如醍醐灌顶一般。用形象的话来说，此种人士如同顶级</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">武林高手一样，能做到“气随心动、收放自如、随机应变”，即达到了“</span></span><span style="line-height: 200%; color: #333333; font-family: 宋体, SimSun; font-size: 16px;"><a href="https://blog.sciencenet.cn/blog-575926-1284957.html" target="_blank">无招胜有招</a>”境界，乃智者也。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; color: #121212; font-family: 宋体, SimSun; font-size: 16px;">显然，通过不断地提升认知水平，学者可事半功倍地取得成功。那么，该如何做呢？</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-family: 宋体, SimSun; font-size: 16px;"><strong><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #121212;">1</span></strong><strong><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #121212;">、扩大知识面</span></strong></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体, SimSun; font-size: 16px;">英国哲学家<a href="https://baijiahao.baidu.com/s?id=1616192325486940925&wfr=spider&for=pc" target="_blank">培根</a>曾说</span><span style="line-height: 200%; font-family: 宋体; font-size: 16px;">: “读史使人明智，读诗使人灵秀，数学使人周密，科学使人深刻，伦理学使人庄重，逻辑修辞使人善辩；凡有所学，皆成性格。”因此，学者多看文献多读书（尤其是科学史的书）可开阔思路，这有助于在“山重水复疑无路”的情况下，找到“柳暗花明又一村”。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-family: 宋体, SimSun; font-size: 16px;"><strong><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #121212;">2</span></strong><strong><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #121212;">、提升学术素养</span></strong></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; font-family: 宋体, SimSun; font-size: 16px;"><a href="https://blog.sciencenet.cn/blog-575926-1269406.html" target="_blank">学术素养</a></span><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">是学者在研究过程中所呈现出的综合素质，主要由学术意识、学术知识、学术能力以及学术伦理道德组成。</span>通过提升学术素养，学者的洞察力和决策能力必然会增强，由此其创造力和学术鉴赏力亦会增强，这有助于规避陷阱从而直达胜利彼岸。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-family: 宋体, SimSun; font-size: 16px;"><strong><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #121212;">3</span></strong><strong><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #121212;">、常和高手交流</span></strong></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="line-height: 200%; color: #121212; font-family: 宋体, SimSun; font-size: 16px;">和比自己棋艺高的棋手下棋，能提高棋艺；同理，和比自己层次高的学者交流，会获益匪浅。然而，诸多学者不愿如此，因为怕和高手交流中因自己的陋知栽面。此乃多虑耳，因为常和高手交流恰恰是让陋知升华为慧见的捷径；有了慧见，何愁大业不成；成就了大业，学者才有了货真价实的面子。</span></p><p style="text-indent: 32px; line-height: 200%; background: white;"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #121212;">行文至此，凭窗眺望，只见雪花飞舞、遍地银装、玉树琼枝。此情此景，不由想到了陈毅元帅的一首诗：“大雪压青松，青松挺且直；</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">要知松高洁，待到雪化时。</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #121212;">”是啊，面对浩瀚的未知世界，学者只有以坚忍不拔的青松精神不断超越自我，向险峻的学术高峰迈进，为高</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">洁的学术殿堂增光，才能不枉此生。何谓真学者？惟此耳！</span></span></p>                    <br><br>
