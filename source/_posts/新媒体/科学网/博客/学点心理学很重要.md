<p><span style="font-family: 宋体, SimSun; font-size: 20px;">高校每年发生多起师生悲剧，给我们敲响了警钟。在各种评论中，个人性格说，环境压力说，精神疾病说，不公待遇说。。。个个火眼金睛，分析到位，</span><span style="font-family: 宋体, SimSun; font-size: 20px;">事后诸葛亮占了绝大多数</span><span style="font-family: 宋体, SimSun; font-size: 20px;">。然而事情一过，人们似乎都忘了。等下次再发生又是如此一遭。仿佛都是打酱油的吃瓜群众。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;">防患于未然是解决此类问题的关键。而心理学扮演很重要的角色。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px; background-color: #FFFFFF; text-indent: 2em;">心理学是一门研究心理现象、精神功能和行为的科学。</span><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">心理学研究涉及</span><a target="_blank" href="https://baike.so.com/doc/4119835-4319106.html" style="text-indent: 2em; color: rgb(0, 0, 0); font-family: 宋体, SimSun; font-size: 20px; text-decoration-line: none;">知觉</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">、</span><a target="_blank" href="https://baike.so.com/doc/2198664-2326391.html" style="text-indent: 2em; color: rgb(0, 0, 0); font-family: 宋体, SimSun; font-size: 20px; text-decoration-line: none;">认知</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">、</span><a target="_blank" href="https://baike.so.com/doc/5394493-10506392.html" style="text-indent: 2em; color: rgb(0, 0, 0); font-family: 宋体, SimSun; font-size: 20px; text-decoration-line: none;">情绪</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">、</span><a target="_blank" href="https://baike.so.com/doc/5389982-7589495.html" style="text-indent: 2em; color: rgb(0, 0, 0); font-family: 宋体, SimSun; font-size: 20px; text-decoration-line: none;">人格</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">、</span><a target="_blank" href="https://baike.so.com/doc/140989-149181.html" style="text-indent: 2em; color: rgb(0, 0, 0); font-family: 宋体, SimSun; font-size: 20px; text-decoration-line: none;">行为</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">和</span><a target="_blank" href="https://baike.so.com/doc/5390766-5627417.html" style="text-indent: 2em; color: rgb(0, 0, 0); font-family: 宋体, SimSun; font-size: 20px; text-decoration-line: none;">人际关系</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">等许多领域，也与日常生活的许多领域——</span><a target="_blank" href="https://baike.so.com/doc/5394769-5631903.html" style="text-indent: 2em; color: rgb(0, 0, 0); font-family: 宋体, SimSun; font-size: 20px; text-decoration-line: none;">家庭</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">、</span><a target="_blank" href="https://baike.so.com/doc/5338309-5573749.html" style="text-indent: 2em; color: rgb(0, 0, 0); font-family: 宋体, SimSun; font-size: 20px; text-decoration-line: none;">教育</a><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">、健康等发生关联。说的直白些，心理学就是揣摩人心，找出相关对策，化解矛盾。</span></p><p><span style="text-indent: 2em; font-family: 宋体, SimSun; font-size: 20px;">当今社会比以往任何时候都复杂，可谓百年未有之变局。但千变万变，人心之变最重要。不能要求每个人都具有“平常心”。不同心理</span><span style="font-family: 宋体, SimSun; font-size: 20px; text-indent: 40px;">类型的人也不是今天一下子冒出来的。极端事件当事人的一个共同特征就是他（她）一定不是只具备平常心的那种人。你可以说激进，想不开，但不解决任何问题。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 20px; text-indent: 40px;">心理学是处理人际关系的学问。将心比心，换位思考，才能减少误判，误解，积累莫须有的“冤仇”。人都有自尊心，有时还好面子，将错就错，期待对方让步，结果矛盾由小到大，越积越深，终有爆发的那一天。老师面对学生，领导面对师生，处理问题方式不同，结局则大不相同。本不该发生的事情发生了，吸取教训远比分析事件本身重要。</span><span style="font-family: 宋体, SimSun; font-size: 20px; text-indent: 40px;">因此，高校师生学点心理学很有必要，关键时刻或许保命。</span></p><p><br></p>                    <br><br>
