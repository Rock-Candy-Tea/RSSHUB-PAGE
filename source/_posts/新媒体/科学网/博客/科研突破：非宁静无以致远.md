<p style="text-indent:32px;line-height:200%"><span style="font-size: 16px; color: #000000;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">与疫情前相比，我的聚会喝酒次数下降了约</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif;">70</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">个百分点<span style="font-size: 16px; font-family: 宋体;">；在疫情期间，的确应大幅减少聚会喝酒次数，但没必要完全杜绝之，</span>因为聚会（许可时）可交流信息，喝酒可催生思维火花。前几天，我和几位高朋喝酒聊天时，某朋友说：“一方面，科研新政提倡科研人员十年磨一剑出卓越成果；另一方面，所在单位以‘数数’标准严酷考核，让科研人员坐不住冷板凳。如果不解决该结构性矛盾，出卓越成果岂不是天方夜谭！”我接茬道：“目前看来，短期内难以解决此矛盾，但无论如何，总得有少数坐得住的‘愚公’甘愿攻坚克难；否则，谁来建立原创科学理论？谁来攻克卡脖子技术？”</span></span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">言归正传，回到正题。</span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">诸葛亮告诫世人：“非淡泊无以明志，非宁静无以致远。”确实，惟有淡泊，才能让一颗心在喧嚣嘈杂的尘世中找准前行方向；惟有宁静，才能使一个人远离功名利禄所带来的纷纷扰扰，走向诗和远方。</span></p><p style="text-indent:32px;line-height:200%"><span style="font-size: 16px; color: #000000;"><span style="color: #000000; font-size: 16px; line-height: 200%; font-family: 宋体;">在这个物欲横流的世界，宁静是当下世人最为缺乏的心境；大多数人紧盯着眼前唾手可得的名利，都渴望冲在前面争取利益最大化。诚然，若凭自己的真才实学获得利益，本无可厚非。然而，目前诸多科研人员，并非如此。君不见，有些人靠批量灌水论文</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">进阶，有些人靠“跑圈子、拜码头”上位，有些人靠忽悠风光。这些人助长了急功近利的浮躁风气，恶化了本该风清气正的学术氛围，带偏了正确的科研导向，令有识之士痛心疾首。</span></span></p><p style="text-indent:32px;line-height:200%"><span style="font-size: 16px; color: #000000;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">古语云：“以铜为鉴，可正衣</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333;">冠<span style="color: #000000; font-size: 16px; background: white;">；以古为鉴，可知兴替；以人为鉴，可明得失。”纵观历史，越王勾践卧薪尝胆，静心刻苦自励、发奋图强终吞吴；</span></span><span style="color: #000000; font-size: 16px; line-height: 200%; font-family: 宋体;">爱迪生、牛顿、居里夫人、桑格、佩雷尔曼、袁隆平等，凭静心钻研成就了伟大发现、不朽杰作。是啊，</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">心静生智，智者不惑，不惑则达，达则超凡。由此看出，</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #454545;">宁静之心</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">对致远是何等的重要。</span></span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">要保持宁静之心，科研人员须有坚定明确的目标、超然物外的洒脱、固守寂寞的勇气和开拓进取的恒心。</span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">科研人员树立了立志攻坚克难的目标，才能心如止水，才能以“虽千万人吾往矣、虽千里无人吾也往矣”之雄心，耕耘在“无人区”，攻坚在“最高峰”，取胜于行稳致远。</span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">超然物外的洒脱，源于攻坚克难过程中独自揭开谜底带来的成就感，是免于浮躁的基石。有此基石，科研人员才能顶住暂时名利的诱惑，才能不被红尘所扰，才能独享超凡智慧带来的潇洒。</span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">自古圣贤多寂寞。科研人员要想成就一番伟业，需要不受干扰地静心深度思考，需要能忍受清贫，需要能承受无人喝彩的孤独，因为其只有通过默默无闻的修炼才能“破壁”，也才能一鸣惊人。</span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">科研人员永葆开拓进取的恒心，才能在探索的征程上不惧失败，才能以稳扎稳打、步步为营的前进节奏取得事半功倍的效果，才能走出自己特立独行的方式，才能最终领略一览众山小的美妙。</span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">林徽因曾说：“真正的宁静不是避开车马喧嚣，而是在内心修篱种菊。”因此，有志科研人员欲“悠然见南山” ——为科技发展和社会进步做出实质性贡献，需在心中铸就坚实的篱笆以防贪婪与颓废之气侵蚀，埋下孤标亮节与高雅傲霜的菊花种子以将浮躁之风拒之门外，保持淡泊名利情操以升华人生价值，坚守科学精神以求得根基牢靠的卓越成果。何谓大道？惟此耳！</span></p><p style=";line-height:200%"><span style="font-family: 宋体, SimSun;
