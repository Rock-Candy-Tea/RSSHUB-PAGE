<div>   
<p><span style="font-size: 20px; font-family: 宋体, SimSun;">最近，一些高校取消学术型硕士生招生，改招专业硕士生，这与<span style="font-size: 20px; letter-spacing: 1px;">以国家重大战略、关键领域和社会重大需求为重点的</span>研究生培养体系构建紧密相连。<span style="font-size: 20px; letter-spacing: 1px;">学硕招生规模缩小、专硕规模扩大的趋势正是对于中国当前经济和社会高速发展的回应。</span></span></p><p><span style="font-size: 20px; font-family: 宋体, SimSun;">教育硕士是专业学位。<span style="font-size: 20px; letter-spacing: 1px; font-family: 宋体, SimSun;">教育硕士以培养中学优秀师资为目标。因此培养方案的制定和论文选题要紧密围绕这个目标去落实。</span><span style="font-size: 20px; font-family: 宋体, SimSun; letter-spacing: 1px;">然而教育硕士招生，培养，论文选题套路化问题凸显，</span><span style="font-family: 宋体, SimSun; font-size: 20px; letter-spacing: 1px;">内卷，平躺现象</span><span style="font-size: 20px; letter-spacing: 1px; font-family: 宋体, SimSun;">对于专硕发展提出了新挑战。如何应对？</span></span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">1.生源</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">问题：现在的教硕生入学前大都是应届毕业生（包括非师范专业），缺乏对中学的了解。</span></p><p><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; letter-spacing: 1px; font-family: 宋体, SimSun;">对策：</span><span style="font-family: 宋体, SimSun; font-size: 20px; letter-spacing: 1px;">教育硕士生源充足，尽可能在师范生中选择，因为学过教育学，心理学，教学法等相关课程，有教育实习经历。可否考虑不招应届跨学科的非师范生，免得入学后再补课，浪费时间资源，</span></span><span style="font-family: 宋体, SimSun; font-size: 20px; letter-spacing: 1px;">无谓措施，</span><span style="font-family: 宋体, SimSun; font-size: 20px; letter-spacing: 1px;">多此一举。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">2.培养方案</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">问题：重课程。二年的学制，一年的课程。填鸭式课程导致课程设置过多，学分过高。学生疲于应对。想学的没学明白，不想学的占满了时间。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">对策：压缩学分，合并课程，给学生更多自主学习和思考的时间。除公共课外，专业必修3-4门（教育学，心理学，教学论等），选修2门足矣。理论课为主，教学技能训练为辅。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">问题：轻实践。二年的学制仅安排第3学期实习和做论文，时间明显不足。第4学期就是写论文，答辩和找工作了。现在的中学并非都愿意接纳安排实习生，一是增添不必要的麻烦，二是实习生与中学老师内在需求不一致。而且重点中学谨慎安排实习生上课，造成以教学为主的论文得不到时间保证，难以做深入。还有中学不同学期教学内容不同，只能做第3学期上课的案例，局限性明显。只能靠问卷，访谈，量表及统计学习成绩等这样简单的指标来完成论文，显然逻辑性和科学性均不足，论文做成了夹生饭。</span></p><p><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-family: 宋体, SimSun; font-size: 20px; letter-spacing: 1px;">对策：增加实习时间，</span><span style="font-size: 20px; letter-spacing: 1px; font-family: 宋体, SimSun;">实践出真知。加大实践投入，将去中学实习时间增加到一年（2,3学期）。这样才有足够的时间调查研究，开展案例，组织活动</span><span style="font-family: 宋体, SimSun; font-size: 20px; letter-spacing: 1px;">。导师也要与学生一起去中学，至少1次。或者了解学生实习情况，</span></span><span style="font-family: 宋体, SimSun; font-size: 20px; letter-spacing: 1px;">及时发现问题，</span><span style="font-family: 宋体, SimSun; font-size: 20px; letter-spacing: 1px;">否则针对性指导成空话。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">3.论文选题</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">问题：选题套路化。模仿抄袭严重。从格式到内容，从写作到答辩，形成一整套“规则”，严重束缚了师生手脚。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">对策：解放思想，履行职责。现行的教指委成员大部分行政任职，脱离中学实际，制定的政策</span><span style="font-family: 宋体, SimSun; font-size: 20px; letter-spacing: 1px;">把握大方向，</span><span style="font-family: 宋体, SimSun; font-size: 20px; letter-spacing: 1px;">应宜粗不宜细，不要规定细则。学院主管要有服务意识，组织教研活动，而不是上传下达程序性工作。导师要学习，与时俱进，争取成为合格导师。研究生实习时要深入了解中学生真实现状，找准真问题，有的放矢设计论文内容。把时间和精力用在发现问题，分析问题，解决问题上，而不是闭门造车，怎么照猫画虎上。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">4.论文格式</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">问题：不知从何时起，教硕论文格式成了一条不成文的潜规则，并发明了一些“研究方法”。论文必须有文献（文献研究法），冠以课标，课纲，教材，教育理论（建构主义，元认知，多元智能，人本主义，做中学...），流行的模式，如基于STEM，5E，PBL，SOLO...等。概念界定必须有，没有就生编硬造一个。问卷（问卷研究法），问题千篇一律，选择ABCDE，自欺欺人。访谈（调查法），看学生用心设计程度。案例（实践研究法），同样看学生用心设计程度。前测与后测（对比法），t检验（统计学方法），数据是人为预期的，结果能可靠吗？评价指标信效度，不知师生双方真的搞懂了吗？学习成绩可以量化，能力素质压根就不是量化评价的。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">对策：打破思想禁锢，不设限。允许探索性研究。总结经验与教训都是必要的。教训有时甚至比“经验”更可贵。不必强求什么结果都要量化评价，有些结论需要时间反复验证。只要立题依据充分，有理论指导，符合逻辑，解决方案可行，有结果和讨论分析就是一篇合格论文。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">5.论文评审答辩</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">问题：形式主义走过场。看似严格，实为放水。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">对策：评审定性化。指出论文不足缺陷，责令改正方能答辩。答辩委员要负起责任。学位委员会也不应是摆设。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">总之，教育硕士研究生质量隐患已经显现。教育个性化，差异化而非趋同化，同质化。主管部门，导师，研究生都要引起足够重视。</span></p><p><span style="letter-spacing: 1px; font-size: 20px; font-family: 宋体, SimSun;">少说空话，多做实事。</span></p><p><br></p><p><span style="font-family: 宋体, SimSun; font-size: 20px;">相关教硕话题：</span></p><p><a href="http://blog.sciencenet.cn/blog-729911-1288602.html" style="text-decoration: underline;
