<p style="text-indent:320px"><span style="font-family: 宋体; text-align: right;"></span></p><p style="text-align:center"><span style=";font-family:宋体;font-size:20px">《网络社会治理》</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 20px;background: #FFFFFF"><span style="font-family:宋体">前言</span></span></p><h1 style="margin-top:0;margin-right:0;margin-bottom:0;margin-left:0;text-indent:0;padding:0 0 0 0 ;text-align:center;background:rgb(255,255,255)"><span style="color: #333333; letter-spacing: 0px; font-weight: normal; font-size: 16px; font-family: 楷体, 楷体_GB2312, SimKai;">王 芳</span></h1><p style="margin: 16px 0;text-indent: 32px;padding: 0;background: rgb(255, 255, 255)"><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">信息网络技术的迅猛发展及其与经济社会的深度融合，为人类社会带来了前所未有的福利增进</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">、</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">能力拓展、效率提</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">升</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">、机遇增加与体验丰富。基于互联网的虚拟社会与现实社会相互作用、不断融合，使传统的社会形态发生了</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">巨大</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">变化，形成网络社会。在网络社会，政府、企业、社会组织、</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">个体</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px"><span style="font-family:宋体">之间的联系更加紧密，交互更加频繁。网络社会超越了工业社会的科层体制结构与大众传播模式，呈现出更加开放、交互、平等、个性化的特征，大大释放了社会关系中蕴含的生产力。社交网络、</span>Web2.0、各种APP、网络直播、元宇宙等种种以大规模社会交互为基础的新技术层出不穷，各种信息、知识、创意、观点与情绪在互联网上迅速传播，不断融合，重塑着社会的信息空间与公共价值体系。</span></p><p style="margin: 16px 0;text-indent: 32px;padding: 0;background: rgb(255, 255, 255)"><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">与此同时，互联网上多元的价值观与思维方式不断碰撞，隐蔽、自由、基于弱关系的交流方式对传统社会的伦理准则与法律规范形成冲击。隐私泄露、电信欺诈、虚假信息、谣言传播、网络暴力、版权侵犯、</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">网络攻击</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">、网络群体性事件等种种问题空前突出，成为潜在的社会风险，在特定条件下甚至可能转化为公共危机，给社会安全与稳定带来挑战。如何既能充分享受网络社会发展所带来的福利和机遇，又能有效应对与之俱来的挑战，是世界各国普遍关心的问题。为此，需要充分研究网络社会的发展规律；考察社会主体间的关系、结构与交互特征，尤其是政府在治理网络中的地位与职能；了解网络社会的情绪传播规律，考察网络集群行为的社会心理特质和演化规律；探索适应网络社会的伦理准则与法律机制。在此基础上，寻找适合中国国情的政策工具与技术手段，探寻网络社会的治理机制。为此，本书从以下六个方面展开研究：</span></p><p style="margin: 16px 0;text-indent: 32px;padding: 0;background: rgb(255, 255, 255)"><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px"><span style="font-family:宋体">（</span>1）网络社会治理的基础理论。从理论上分析网络社会的社会结构与演化规律，分析当前中国网络社会治理的</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">构成要素、</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">主体间关系、政府地位、影响因素和潜在风险。</span></p><p style="margin-top:16px;margin-right:0;margin-bottom:16px;margin-left:0;text-indent:32px;padding:0 0 0 0 ;background:rgb(255,255,255)"><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px"><span style="font-family:宋体">（</span>2）基于信息伦理的网络社区治理机制。网络社区是网络社会治理的基本着力点，也是个人信息伦理的集中呈现场域。对微信群、百度贴吧、科学网、微博、微信、抖音、猫扑等网络社区进行研究，分析网络社区的用户参与行为、社会结构演化与信息传播影响因素，以及虚拟社区中不同社会</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">群体</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">间的交流模式与社会融合，</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">分析</span><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px">网络社区中的信息伦理失范问题，并提出相应的治理机制。</span></p><p style="margin-top:16px;margin-right:0;margin-bottom:16px;margin-left:0;text-indent:32px;padding:0 0 0 0 ;background:rgb(255,255,255)"><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px"><span style="font-family:宋体">（</span>3）网络舆情传播规律与引导机制。通过案例研究与模拟仿真分析方法探索网络舆情的演化规律，分析社会风险、突发事件与公共危机之间的转化规律，探寻网络情感传播的文本特征与计算方法，比较正负面网络舆情的传播规律，进而提出网络舆情治理的对策建议。</span></p><p style="margin-top:16px;margin-right:0;margin-bottom:16px;margin-left:0;text-indent:32px;padding:0 0 0 0 ;background:rgb(255,255,255)"><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px"><span style="font-family:宋体">（</span>4）网络信息伦理及网络社会风险治理机制。探索网络社会公共伦理的价值原则，对安全生产行业的信息传播路径、食品行业的风险信息识别、企业与个人网络伦理失范的现象进行案例分析，探索网络社会的风险治理机制。</span></p><p style="margin-top:16px;margin-right:0;margin-bottom:16px;margin-left:0;text-indent:32px;padding:0 0 0 0 ;background:rgb(255,255,255)"><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px"><span style="font-family:宋体">（</span>5）现实与网络交互场域中的集群行为及其社会心理成因。从社会心理学视角研究网络集群行为的形成与发展规律，通过案例研究分析网络群体性事件的社会心理成因、关键影响因素与后果，探索相应的风险预警机制。</span></p><p style="margin-top:16px;margin-right:0;margin-bottom:16px;margin-left:0;text-indent:32px;padding:0 0 0 0 ;background:rgb(255,255,255)"><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px"><span style="font-family:宋体">（</span>6）网络社会治理的动态模拟仿真实验。运用元胞自动机和多agent 系统方法，对网络空间中个体和群体的情绪与心理、交互行为等问题进行抽象描述并建立仿真模型，考察网络社会治理的关键变量与治理机制生成模式，分析网络舆情的演化规律。</span></p><p style="margin: 16px 0;text-indent: 32px;padding: 0;background: rgb(255, 255, 255)"><span style="font-family: 宋体;color: #333333;letter-spacing: 0;font-size: 14px"><span
