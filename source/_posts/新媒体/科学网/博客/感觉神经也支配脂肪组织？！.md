style="font-family: 宋体;color: #2A2B2E;letter-spacing: 0;font-size: 16px">感觉神经也调节脂肪组织？！</span></p><p style="margin-top:0;margin-right:0;margin-bottom:0;margin-left:0;text-indent:28px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;line-height:200%;background:rgb(252,253,254)"><span style="font-family: 宋体;color: #2A2B2E;letter-spacing: 0;font-size: 16px">传统观点认为脂肪组织没有感觉神经，是依靠内分泌方式，例如释放瘦素，给大脑提供信号。大脑则能通过交感神经给脂肪发出神经信号。最新研究发现这是一种错误的看法。脂肪组织内也存在感觉神经支配，而且这种支配能感受到环境温度信号，通过反射方式调控脂肪组织发生代谢转换。这一研究给脂肪的研究提供了新方向。也许我们可以考虑更多类似的情况，例如内脏也有这种类似的感觉神经支配。只不过这种感觉和能引起主观感觉的神经不一样，主要发挥代谢和内脏功能的调控。</span></p><p style="margin-top:0;margin-right:0;margin-bottom:0;margin-left:0;text-indent:28px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;line-height:200%;background:rgb(252,253,254)"><span style="font-family: 'Segoe UI';color: #2A2B2E;letter-spacing: 0;font-size: 16px">脂肪在调节体内能量平衡中起着核心作用。传统观点认为，脂肪组织通过向血液中释放激素向大脑传递信号</span><span style="font-family: 宋体;color: #2A2B2E;letter-spacing: 0;font-size: 16px">大脑通过交感神经系统的神经传递信号来决定脂肪的功能，交感神经系统会在脂肪组织和身体其他部位释放被称为儿茶酚胺分子的神经递质，以介导对压力的反应。</span><span style="font-family: 'Segoe UI';color: #2A2B2E;letter-spacing: 0;font-size: 16px"><span style="font-family:Segoe UI">对脂肪神经的研究主要集中在交感神经，发现它们能促进产热和脂质代谢。相比之下，脂肪组织中的感觉神经是否向大脑传递信号并在脂肪代谢中发挥作用尚不清楚。</span> </span></p><p style="margin-top:0;margin-right:0;margin-bottom:0;margin-left:0;text-indent:28px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;line-height:200%;background:rgb(252,253,254)"><span style="margin-top:0;margin-right:0;margin-bottom:0;margin-left:0;text-indent:28px;padding:0 0 0 0 ;text-autospace:ideograph-numeric;line-height:200%;background:rgb(252,253,254)"><span
