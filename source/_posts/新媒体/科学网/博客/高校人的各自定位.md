<p><span style="color: #444444; background-color: #FFFFFF; font-family: 宋体, SimSun; font-size: 20px;">高校由学生</span><span style="background-color: #FFFFFF; color: #444444; font-family: 宋体, SimSun; font-size: 20px;">和</span><span style="background-color: #FFFFFF; color: #444444; font-family: 宋体, SimSun; font-size: 20px;">老师</span><span style="background-color: #FFFFFF; color: #444444; font-family: 宋体, SimSun; font-size: 20px;">组</span><span style="background-color: #FFFFFF; color: #444444; font-family: 宋体, SimSun; font-size: 20px;">成。</span></p><p><span style="color: #444444; background-color: #FFFFFF; font-family: 宋体, SimSun; font-size: 20px;">学生有本科生，研究生两大类。本科生毕业以课程为主，研究生毕业以论文为主。<br></span><span style="background-color: #FFFFFF; color: #444444; font-family: 宋体, SimSun; font-size: 20px;">老师按职称分助教，讲师，副教授，教授。现在实行合同制，又多了博士后，预聘制，长聘制。</span><span style="background-color: #FFFFFF; color: #444444; font-family: 宋体, SimSun; font-size: 20px;">按待遇分头衔，帽子。</span><span style="background-color: #FFFFFF; color: #444444; font-family: 宋体, SimSun; font-size: 20px;">按话语权分领导，大牛和群众。</span></p><p><span style="color: #444444; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; background-color: #FFFFFF; font-size: 20px;">别把大学神话了。学士，硕士，博士就是一个学位而已。读书能否改变命运取决于个人和环境。学位含金量则靠自己。职称就是一个岗位而已。具备职称的与名副其实的是两码事。</span></p><p><span style="color: #444444; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; background-color: #FFFFFF; font-size: 20px;">好比穿衣戴帽，各好一套。</span><span style="background-color: #FFFFFF; color: #444444; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 20px;">至于</span><span style="background-color: #FFFFFF; color: #444444; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 20px;">学位，岗位适不适合你，则另当别论。做适合自己的就是最好的。</span></p><p><span style="color: #444444; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; background-color: #FFFFFF; font-size: 20px;">内卷，躺平不是大学独有的，大学早已不是世外桃源。</span><span style="background-color: #FFFFFF; color: #444444; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 20px;">别抱怨压力，别总提公平，360行，干好哪一行都不容易。主动+自觉是做好自己的前提。</span></p><p><span style="background-color: #FFFFFF; color: #444444; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 20px;">自我定位，确立目标，朝着梦想去奋斗吧！</span></p>                    <br><br>
