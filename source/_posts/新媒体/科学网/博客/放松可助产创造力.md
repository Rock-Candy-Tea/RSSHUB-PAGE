<p style="text-indent:32px;line-height:200%"><span style="font-size: 16px;"><span style="line-height: 200%; font-family: 宋体; color: #404040; background: white;">创造力是指</span><span style="line-height: 200%; font-family: 宋体; color: #333333; background: white;">产生新思想、发现和创造新事物的能力</span><span style="line-height: 200%; font-family: 宋体; color: #404040; background: white;">，是推动科技发展和社会进步的主要动力。今天上午，我偶然看到</span><span style="line-height: 200%; font-family: "Times New Roman", serif; color: #404040; background: white;">2</span><span style="line-height: 200%; font-family: "Times New Roman", serif; color: #333333; background: white;">021</span><span style="line-height: 200%; font-family: 宋体; color: #333333; background: white;">年诺贝尔化学奖得主之一<a href="https://baijiahao.baidu.com/s?id=1713112811279876411&wfr=spider&for=pc" target="_blank">利斯特</a>的名言：“创造力不是源于专注和紧张；只有当你放松时，你的思想才能流动。”果真如此嘛，且听我娓娓道来。</span></span></p><p style="text-indent:32px;line-height:200%"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">我们常有这样的感觉：在</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919; letter-spacing: 0px;">心如止水</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">的状态下，</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919; letter-spacing: 0px;">强烈的宁静感会导致莫名其妙的</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">新点子频出；在</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #191919; letter-spacing: 0px;">物我两忘的状态下，强烈的合一感会导致异想天开的鬼点子冒泡；在淡泊无虞的状态下，强烈的专注感会导致难以置信的灵感闪现。这说明</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">创造力与心态放松密切相关。在科学史上，这样的例子</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #222222; background: white;">不胜枚举，诸如魏格纳在病中休养时偶然翻阅世界地图而引发了联想，提出了大陆漂移说；牛顿在果园散步时受苹果落地的启发，发现了万有引力定律。</span></span></p><p style="text-indent:32px;line-height:200%"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">由</span><a href="https://baijiahao.baidu.com/s?id=1686158996188850485&wfr=spider&for=pc" target="_blank" style="text-decoration: underline; font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: rgb(0, 112, 192);"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: #0070C0;">现代认知神经学</span></a><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">知，人们处于放松状态时，大脑具有默认模式网络（<span style="font-size: 16px; line-height: 200%; color: #333333; background: white; font-family: "times new roman";">DMN</span>）的动力学行为。此时，通过重新发掘过去的记忆、架起过去和未来之间的想象桥梁、联系不同的想法，大脑会自由组合出人们平时可能意识不到的信息，从而产生新的创意。这就是我们所说的“发散思维”或创造力的来源之一。</span></p><p style="text-indent:32px;line-height:200%"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">目前，有几种积极的方式可激活<span style="font-size: 16px; line-height: 200%; color: #333333; background: white; font-family: "times new roman";">DMN</span>，如冥想、适度的浅睡眠、体育运动、喝酒。嗯，我是酒仙级人物，对此有一定的发言权，故再唠叨几句，呵呵。常喝酒的人可能都有这样的体会：在适度酒精的刺激下，活跃的脑细胞会诱发有趣另类的想法，起到打开谜团钥匙的作用。为何古有李白斗酒诗百篇、今有“隔壁老王”瓶酒破难题，其原因或在于此。</span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px;">如果科教单位与投资方持有急功近利的心态，以诸如三日一催促、五日一考核、七日一评比、九日一淘汰的政策对待科教人员，则必然会使其陷入应急之态，掉入疲惫之中，落入紧张之围，进入焦虑之圈。在这样的情况下，何谈富有创造力新思想的涌现，更遑论什么卓越成果。这提醒有关部门，制订任何评价政策，都必须遵守科学规律，如此才能切切实实促进科教事业健康发展。</span></p><p style="line-height:200%"><span style="font-size: 16px;"><strong><span style="font-size: 16px;
