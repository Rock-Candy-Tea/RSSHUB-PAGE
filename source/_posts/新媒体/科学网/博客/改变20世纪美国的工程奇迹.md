<p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;"><br></span></p><p style="line-height: 1.75em;"><span style="background: #F8F9F4; font-family: 宋体; color: #000000; font-size: 16px;">改变20世纪美国的工程奇迹</span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">■武夷山</span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">（发表于2021年11月18日《中国科学报》）</span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;"><br></span></p><p style="line-height: 1.75em;"><span style="color: #000000; font-size: 16px;">        2020<span style="font-size: 16px; color: #000000; font-family: 宋体;">年</span>11<span style="font-size: 16px; color: #000000; font-family: 宋体;">月，美国麻省理工学院出版社出版了独立学者</span>David P.Billington Jr.<span style="font-size: 16px; color: #000000; font-family: 宋体;">（戴维·</span>P.<span style="font-size: 16px; color: #000000; font-family: 宋体;">小比林顿）的著作</span>From Insight to Innovation<span style="font-size: 16px; color: #000000; font-family: 宋体;">：</span>Engineering Ideas That Transformed America in the Twentieth Century<span style="font-size: 16px; color: #000000; font-family: 宋体;">（本文作者译为“从洞察力到创新：改变了</span>20<span style="font-size: 16px; color: #000000; font-family: 宋体;">世纪美国的工程思想”）。小比林顿拥有美国得克萨斯大学奥斯汀分校的历史学博士学位。</span></span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">    现代生活离不开技术，但多数人的技术素养较欠缺，因此不大了解技术背后的东西。在这本通俗的美国工程技术史著作中，作者叙述了水坝、国家公路系统、摩天大楼、核电站、喷气发动机、晶体管、芯片、个人电脑、互联网等工程技术创新的成就，说明了这些工程奇迹是怎么创造出来的，其工作原理又是怎么回事。</span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">    每一章是一个独特的创新故事。罗斯福新政时期建设的大坝驯服了大江大河；国家公路系统和喷气式飞机带来的航空旅行将全国更加紧密地联系在一起；诞生于私人公司的晶体管和芯片，在早期发展阶段获得了政府的支持，后来找到了广大的市场；计算机和互联网则源自政府项目，后来在私营部门获得了广大的市场，等等。</span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">    小比林顿发现，具有超凡脱俗见解的工程师们即使在官僚气很重的时代亦能成功。至关重要的是工程师拥有独立的眼界，而全社会都欢迎创新。</span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">    本书以非专业人士为对象，故文字通俗易懂，还配有不少漂亮的插图。与一般历史类著作不同的是，每一章都附有若干框图，框图中对相关主题作了技术描述，或是附上了数学公式和化学反应式。</span></p><p style="line-height: 1.75em;"><span style="color: #000000; font-size: 16px;"><span style="font-size: 16px; color: #000000; font-family: 宋体;">    小比林顿不是按照技术发展的内在必然逻辑展开叙述，而是偏重于描写发展了关键技术思想和部署了相关关键技术的工程师，包括几位女性工程师。例如，在《高速公路和摩天大楼》一章中介绍了</span>Marilyn Jorgenson Reece<span style="font-size: 16px; color: #000000; font-family: 宋体;">，她是加州历史上第一位获得土木工程师执照的女性。</span></span></p><p style="line-height: 1.75em;"><span style="color: #000000; font-size: 16px;"><span style="font-size: 16px; color: #000000; font-family: 宋体;">    虽然本书没有足够篇幅详细介绍每位重要工程师的生平，但小比林顿并不因此就掩饰或略去工程英雄们的瑕疵。例如，对半导体研究的先驱、</span>1956<span style="font-size: 16px; color: #000000; font-family: 宋体;">年诺贝尔物理学奖得主威廉·肖克莱所持种族主义的智商观念，小比林顿就不客气地给予批评。</span></span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">    尽管在工程师身上着墨不少，但本书的焦点仍是工程技术的进步、工程教育的历史变化以及什么样的人成长为工程师，等等。</span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">    本书最后谈了几点启示。</span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">    首先，具有历史意义的工程项目留下了物质遗产，也产生了路径依赖，这一点常被忽略。</span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">    其次，在重大技术系统的开发过程中，政府发挥着重要而持久的作用。政府要么是大项目的直接资助者，要么是采购者（主要是军方采购），晶体管和芯片的发展历程便是政府作用的很好例证。</span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">    再次，未来的大型工程应该更关注环境影响。</span></p><p style="line-height: 1.75em;"><span style="color: #000000; font-size: 16px;"><span style="font-size: 16px; color: #000000; font-family: 宋体;">    美国史密森学会勒梅尔森发明与创新研究中心主任</span>Arthur Daemmrich<span style="font-size: 16px; color: #000000; font-family: 宋体;">认为，每一章对所涉工程技术的叙述是相对独立和完整的，但缺点是全书缺乏打通不同技术或系统的总体性的叙事或分析。本书对政治选择的叙述缺乏历史纵深感。这样，读者虽然能感受到在工程技术领域存在着政治和政策权衡，比如建坝的环境后果和就业效应需要权衡、核电的能源生产效益和健康风险需要权衡，但未能了解到其实相关工程设计和决策都受到那些权衡涉及的辩论的深刻影响。</span></span></p><p style="line-height: 1.75em;"><span style="font-family: 宋体; color: #000000; font-size: 16px;">    另外，小比林顿没有指出一些工程决策中存在种族主义色彩，例如高速公路的选址决策使很多黑人流离失所，使原先稳定的社区逐渐土崩瓦解。</span></p><p style="line-height: 1.75em;"><span style="color: #000000; font-size: 16px;"><span style="font-size: 16px; color: #000000; font-family: 宋体;">    本书是美国工程技术故事三部曲的最后一部。本书作者的父亲、普林斯顿大学工程学教授</span>David P.Billington<span style="font-size: 16px; color: #000000; font-family: 宋体;">于</span>1996<span style="font-size: 16px; color: #000000; font-family: 宋体;">年在威立出版社出版了《创新者：使美国成为现代社会的工程先驱者》，是为三部曲的第一部。</span></span></p><p style="line-height: 1.75em;"><span style="color: #000000; font-size: 16px;">    2013<span style="font-size: 16px; color: #000000; font-family: 宋体;">年，父子两人合著的著作《动力、速度和形态：工程师和</span>20<span style="font-size: 16px; color: #000000; font-family: 宋体;">世纪之造就》在普林斯顿大学出版社出版，是为第二部。</span></span></p><p style="line-height: 1.75em;"><span style="color: #000000; font-size: 16px;"><span style="font-size: 16px; color: #000000; font-family: 宋体;">    父亲于</span>2018<span style="font-size: 16px; color: #000000; font-family: 宋体;">年去世，本书由小比林顿独自完成。就像工程技术传统需要代际传承一样，这对父子在叙述工程技术故事的能力方面也完成了漂亮的代际传承。</span></span></p><p style="line-height: 1.75em;"><span style="color: #000000; font-size: 16px;"><span style="font-size: 16px; color: #000000; font-family: 宋体;"><br></span></span></p><p style="line-height: 1.75em;"><span style="color: #000000; font-size: 16px;"><span style="font-size: 16px; color: #000000; font-family: 宋体;">《中国科学报》</span> (2021-11-18 <span style="font-size: 16px; color: #000000; font-family: 宋体;">第</span>7<span style="font-size: 16px; color: #000000; font-family: 宋体;">版</span> <span style="font-size: 16px; color: #000000; font-family: 宋体;">书评</span>)</span></p><p style="line-height: 1.75em;"><font face="宋体"> </font></p><p><br></p>                    <br><br>
