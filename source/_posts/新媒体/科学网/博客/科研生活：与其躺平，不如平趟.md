<p style="text-indent:32px;line-height:200%"><span style="font-size: 16px;"><span style="line-height: 200%; font-family: 宋体;">近期，躺平一词成为网络热词；<span style="line-height: 200%; font-family: 宋体; color: #333333; background: white;">所谓躺平，</span></span><span style="line-height: 200%; font-family: 宋体; background: white;">早期是表示网友不回应不反抗的一种心理态度，现在被用来表示人们看淡竞争之后主动追求低欲望生活（</span><span style="line-height: 200%; font-family: 宋体; color: #333333; background: white;">浑浑噩噩过日子，不再追求上进和渴求成功</span><span style="line-height: 200%; font-family: 宋体; background: white;">）的一种社会现状</span>。</span></p><p style="text-indent:32px;line-height:200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px;"><span style="line-height: 200%; font-family: 宋体; color: #333333; background: white;">然而，</span>在我看来，对科研人员而言，躺平可分为真躺平和假躺平两类。前者指某些科研人员虽经努力奋斗，但未能做出令人自豪的成果或未能获得预期的名利，在经历各种不顺和打击后看破了红尘，处于与世无争的佛系状态，这与上述躺平的原定义类似；后者指某些科研人员以好奇心默默无闻埋头做科研，不在乎名利，不想巴结大咖，不愿在公众场合抛头露面，这在他人看来貌似是躺平，但实则是平趟艰难险阻以求登上科学高峰。</span><br></p><p style="text-indent:32px;line-height:200%"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%;">据我所知，有些临近退休的科研人员处于真躺平状态，因为自己觉得再努力科研上不会有多大起色，且生活上也厌倦了靠投机取巧才能获得的名利。对此，没必要苛责，毕竟人各有志嘛。然而，与其碌碌无为一生，不如</span><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%; color: #333333;">象闪电一样划过夜空绚烂一时。</span><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%;">每位科研人员的情况不同，有的在年轻时做出了卓越成果，而有的则大器晚成，在中老年时开始发力做出了重要科技贡献。正所谓“有志不在年高，春晖不论早晚”嘛。</span></span></p><p style="text-indent:32px;line-height:200%"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%;">在科研生活中，科研人员都可能遭遇不同程度的困境乃至厄运</span><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%; background: white;">，或选择沉沦或选择奋起，但有志者应选择后者。</span><span style="font-family: 宋体, SimSun; font-size: 16px; line-height: 200%; color: #333333; background: white;">培根说：“奇迹多是在厄运中出现的”。黎明前特别黑暗，成功前格外艰难。逆境使人别无选择，逆境给人很大压力，而压力能激发出强劲动力。只要在艰难时刻再坚持一下，挺过最难熬的一段，那么紧接着可能就是机遇的光顾、奇迹的出现。</span></span></p><p style="text-indent:32px;line-height:200%"><span style="font-family: 宋体, SimSun; font-size: 16px;"><span style="font-size: 16px; font-family: 宋体, SimSun; line-height: 200%; color: #333333; background: white;">人这一生，十分短暂。与其以真躺平状态度过余生，不如以平趟精神为国家社会做出突出贡献给子孙后代留下念想。真躺平易，但平趟难，然只有克服了“</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; color: #333333; background: white;">81</span><span style="font-size: 16px; font-family: 宋体, SimSun; line-height: 200%; color: #333333; background: white;">难”才能取到“真经”，才能不鸣则已、一鸣惊人。</span></span></p><p style="text-indent: 32px;line-height: 200%"><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">我们知道牛顿，是因为</span><em><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif;">F</span></em><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif;">=<em>ma</em></span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">；</span> <span style="font-size: 16px; line-height: 200%; font-family: 宋体;">我们知道爱因斯坦，是因为</span><em><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif;">E</span></em><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif;">=<em>mc</em><sup>2</sup></span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">；我们知道居里夫人，是</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">因为发现元素</span></span><a href="https://baike.baidu.com/item/%E9%92%8B" target="_blank" style="line-height: 200%; font-family: 宋体; color: windowtext; font-size: 16px; text-decoration: underline;"><span style="line-height: 200%; font-family: 宋体; color: windowtext; font-size: 16px;">钋</span></a><span style="font-size: 16px;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #333333; background: white;">和镭；我们知道门捷列夫，是因为元素周期表；我们知道魏格纳，是因为大陆漂移说；</span><span style="font-size: 16px; line-height: 200%;">……</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">。这些科学巨匠，不管在科研生活中曾遇到多少挫折困苦，仍一往无前平趟未知世界，从而做出了载入史册的卓越成果。</span></span></p><p style="text-indent: 32px;line-height: 200%"><span style="line-height: 200%; font-family: 宋体; font-size: 16px;">我们知道是金子，迟早会发光；我们知道是沙子，自然会沉入海底。作为国家培育多年的科研人员，应追求前者而非后者。人过留名，雁过留声。我想大多数科研人员都不想无声无息地离开这个世界，但要留下痕迹，除平趟外别无他途。</span></p><p style="text-indent: 32px; line-height: 200%;"><span style="line-height: 200%; font-family: 宋体; font-size: 16px;">一旦科研人员平趟了某一科学堡垒，则能傲视群雄展现舍我其谁的力量，则能领略“会当凌绝顶一览众山小”的美妙。这份智慧足以令人笑傲江湖，这种成就足矣令人望尘莫及，这种境界足以令人惊叹不已。既然如此，还等什么，赶紧从真躺平中翻转投身于平趟吧！</span></p>                    <br><br>
