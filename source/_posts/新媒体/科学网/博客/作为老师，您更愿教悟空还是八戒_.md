<p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian; font-size: 16px;">假设您是研究生导师，孙悟空和猪八戒都想成为您的研究生，但您手上只有一个招生指标，您愿招谁？或者，假设您是老师，悟空和八戒有幸同时成为您的学生，您偏爱哪位？</span><br></p><p style="margin: 0 0 10px;text-align: center;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><strong><span style="font-family: DengXian;font-size: 16px">一</span></strong></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">这种问题有如难倒无数男同胞的神问题：假如母亲和妻子同时掉水里，你先救谁？回答也许是一回事，真做又是一回事。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">作为博士生导师，老文偶尔做梦：这辈子若能像唐僧那样好运，虽然凭个人魅力不可能招到但靠菩萨保佑也给配一个悟空那样的学生，学生毕业获博士学位的同时，老文自己也可终成</span>“正果”，位列“仙班”了。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">看现实，绝大多数老师的选择是悟空。从学校层面看，就看得更真切了：悟空毫无悬念是各院校抢夺的对象。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">这是很好理解也无可厚非的。因为，一句话，悟空代表的是所谓优质生源。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">而八戒，就一言难尽了。他在德智体美劳无论哪方面，都远不及悟空。特别是，他贪吃、贪睡、贪财、贪色、贪生怕死等等。如此恶劣品行又加上猪一般的脑子，八戒简直是孺子不可教也。哪所学校或哪位老师招了他，自己的名声不被搞臭就万幸了。通常情况下，老师都会对他唯恐避之不及，更别说把珍贵招生指标专供他了。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">难以理喻的是，西游世界的情况跟现实世界竟然完全相反！读《西游记》，比较一下悟空和八戒从求道到取经期间分别受老师们的喜爱程度就知道了。</span></p><p style="margin: 0 0 10px;text-align: center;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><strong><span style="font-family: DengXian;font-size: 16px">二</span></strong></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">先看得道成仙之前。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">悟空为得道成仙，从花果山出发，漂洋过海，登界游方，花了十数个年头，找到灵台方寸山，见到菩提祖师时，</span>“倒身下拜，磕头不计其数，口中只道：‘师父，师父！我弟子志心朝礼，志心朝礼！’”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">最优质生源悟空，找个心仪的老师竟然如此耗时费劲，在今天看来不可思议。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">与悟空截然相反，八戒是被最优质老师拉去上学的，有如今天的所谓高考</span>“状元”是被顶尖名校抢进去的。请看八戒的自我介绍（第十九回）：</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px">“自小生来心性拙，贪闲爱懒无休歇。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">不曾养性与修真，混沌迷心熬日月。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">忽然闲里遇真仙，就把寒温坐下说。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">听言意转要修行，闻语心回求妙诀。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">有缘立地拜为师，指示天关并地阙。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">得传九转大还丹，工夫昼夜无时辍。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">三花聚顶得归根，五气朝元通透彻。</span>”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">由此，八戒的前世就天生笨，从小游手好闲、不爱学习，成天迷迷糊糊混日子，但他即使不是生在</span>“绝不让一个孩子失学”的时代，也十分幸运碰到了主动收他为徒的老师。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">也许有人要说，八戒的老师肯定不咋的或不如菩提祖师牛</span>B，是招不到像悟空那样的好学生才不得已招八戒的。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: style="font-family:DengXian">八戒的老师是谁至今仍是个谜。因此您硬要说八戒的老师不咋的，别人也无法反驳。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">在老文看来，对学生而言，老师牛</span>B固然重要，然而更重要的是学生离开老师后的前途如何。从这点看，八戒远比悟空幸运。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">八戒学成后，玉帝立马敕封他为</span>“天蓬元帅”，并为他举行隆重仪式，那场面，是人山人海，彩旗飘飘，鼓乐齐鸣。看八戒的描述——</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px">“功圆形满却飞升，天仙对对来迎接。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">朗然足下彩云生，身轻体健朝金阙。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">玉皇设宴会群仙，各分品级排班列。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">敕封元帅管天河，总督水兵称宪节。</span>”</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">可以想象，八戒的喜悦心情远胜人间登科后的</span>“春风得意马蹄疾，一日看尽长安花”。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">悟空学成</span>“得道身轻体亦轻”后，菩提祖师给他指的出路是“你从那里来，便从那里去就是了”。他只好回花果山继续当他的美猴王，最大的改变是提升了猴子在花果山的地位，牛魔王等有实力的妖魔也愿意跟他称兄道弟了。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span #333333;font-size: 16px;background-color: #FFFFFF"><span style="font-family:DengXian">有官无禄</span>”</span><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span 28px"><strong><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">三</span></span></strong></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">再看被天庭处罚之后至加入取经团队之前。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">八戒在天蓬元帅任上，因酒醉</span>“扯住嫦娥要陪歇”，被玉帝贬下尘凡。他错投猪胎，生的猪头人身，在福陵山占山吃人度日。在此期间，也有高人想收他为徒。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">唐僧喜得八戒后，立马迎来</span>“浮屠山玄奘受心经”。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: 32px;font-size: 16px"><span style="font-family:DengXian">是谁授了唐僧心经？乌巢禅师。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">正是禅师，告诉唐僧取经的前途是光明的</span>——“路途虽远，终须有到之日”，但道路是曲折的——“却只是魔瘴难消”，并授唐僧法宝——“若遇魔瘴之处，但念此经，自无伤害”。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">此外，禅师</span>“晓得过去未来之事”，为唐僧准确预言了取经路上的“前程端的如何”。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">由此可见乌巢禅师有多牛。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">唐僧、悟空和八戒三众之中，谁曾有幸结识禅师？唯八戒。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: style="font-family:DengXian">唐僧问八戒禅师有些什么勾当，八戒道：</span>“他倒也有些道行。他曾劝我跟他修行，我不曾去罢了。”看看，如此高人欲收八戒为徒，人家八戒还瞧不上，不愿从。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">禅师看得上悟空么？</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: style="font-family:DengXian">禅师听八戒说前年蒙观音菩萨劝善，愿随唐僧做个徒弟时，连连大喜道：</span>“好，好，好！”他又指定悟空问道：“此位是谁？”悟空笑道：“这老禅怎么认得他，倒不认得我？”禅师道：“因少识耳。”禅师这不是故意当众特别是当着唐僧的面抬举八戒而表示跟悟空不熟么？</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">悟空号称</span>“齐天大圣”，名扬四海，那受得了这口气！他越来越感禅师的言语令人不爽，以致后来动手捣了人家的老巢——“举铁棒望上乱捣”。但“行者纵有搅海翻江力，莫想挽着乌巢一缕藤”。换言之，悟空纵使出浑身解数，也伤不着禅师一根汗毛——悟空见识了真正的高人。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">八戒在福陵山云栈洞落草时，悟空被压在五行山下。他没有像八戒那样即使被天庭打入另类依然有老师愿收为徒弟，似乎情有可原，但同等条件下，纵使他名气齐天，像乌巢禅师那种级别的高人也未必把他放在眼里。</span></span></p><p style="margin: 0 0 3px;text-align: center;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><strong><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">四</span></span></strong></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">再看加入取经团队。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">加入取经团队成为唐僧的徒弟，是从妖界到位列仙班的千载难逢好途径。谁能成为幸运儿？</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">如来佛祖的旨意，须是</span>“神通广大”的妖魔。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">但这样的妖魔往往</span>“不伏使唤”。如来给了主动请缨组建取经团队的观音“金紧禁”三个箍儿：“可将此箍儿与他戴在头上，自然见肉生根。各依所用的咒语念一念，眼胀头痛，脑门皆裂，管教他入我门来。”</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">按如来的标准，观音为唐僧招了悟空理所当然，但她把八戒和沙僧这两个根本用不着箍儿的家伙也招进取经团队，在今天看来是十足的</span>“降分录取”。由此，八戒也算是受到观音或观音代表的教育机构的偏爱。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">这种偏爱还表现在，对成为唐僧的徒弟，悟空是求之不得，对观音点头如捣蒜般表示：</span>“愿去！愿去！”而八戒并不向往，是观音用“前程”吸引他，他才勉强表态：“愿随！愿随！”</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">仔细分辨，</span>“愿随”应该不如“愿去”坚决。八戒一如既往保持着纵使顶尖名校或名师求他若渴，他还要再考虑考虑一下的范。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">取经四人行，八戒是唯一像打麻将</span>“三缺一”缺了他就不行的那个人。</span></p><p style="margin: 0 0 3px;text-align: center;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><strong><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">五</span></span></strong></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">最后看西天取经期间。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">在取经团队里，唐僧袒护、偏心八戒是个公开的秘密。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">最激烈的表现是，唐僧多次把悟空赶出取经团队，但他从未赶八戒走。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">此外，悟空多次用事实挑明过唐僧对八戒的偏心。最典型的要数狮驼岭大战青毛狮子怪、黄牙老象、大鹏金翅雕三大魔王那一回。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">悟空降了大魔后，激发八戒与二魔战战。八戒同意出战，但要求悟空把那绳儿借与他做个救命索，他若是输与二魔，希望悟空把他扯回来，莫教二魔拉了去。</span></span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">八戒不出所料战不过二魔，急回头叫悟空</span>“扯扯救命索”。悟空不仅没扯，“转把绳子放松了，抛将去”。八戒被二魔一鼻子卷住，掳回洞内。</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">八戒命在一绳，悟空见死不救。唐僧见状凶巴巴怒斥悟空：</span>“怪不得悟能咒你死哩！原来你兄弟全无相亲相爱之意，专怀相嫉相妒之心！他那般说，教你扯扯救命索，你怎么不扯，还将索子丢去？如今教他被害，却如之何？”</span></p><p style="margin: 0 0 3px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">悟空道破了唐僧的偏心：</span>“师父也忒护短，忒偏心！罢了，像老孙拿去时，你略不挂念，左右是舍命之材；这呆子才自遭擒，你就怪我。也教他受些苦恼，方见取经之难。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">八戒内心里也十分清楚唐僧对他的偏爱。例如，在取经即将大功告成的第九十二回，八戒道出了实情。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">取经团队在青龙山降了三犀牛精后，金平府的人民无比感激，不仅为唐僧师徒</span>“建立生祠，各各树牌刻文，用传千古”，还轮流请吃请喝，府县宴请完了，二百四十家灯油大户又开始这家酬、那家请。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">这样吃喝下去，正合八戒心意。唐僧</span>“恐只管贪乐，误了取经”，于是指示悟空“瞒着那些大户人家，天不明走罢”。悟空拉八戒早起上路，八戒抱怨：“又是这长老没正经！二百四十家大户都请，才吃了有三十几顿饱斋，怎么又弄老猪忍饿！”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">唐僧听言骂道：</span>“馕糟的夯货！莫胡说！快早起来！再若强嘴，教悟空拿金箍棒打牙！”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">八戒听见说打，慌了手脚道：</span>“师父今番变了，常时疼我，爱我，念我蠢夯护我；哥要打时，他又劝解；今日怎么发狠转教打么？”</span></p><p style="margin: 0 0 10px;text-align: center;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><strong><span style="font-family: DengXian;font-size: 16px">六</span></strong></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">西游世界的老师们为什么都偏爱八戒那样的学生？</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">首先，从唐僧给悟空的解释可看出一些端倪。唐僧说：</span>“徒弟啊，你去，我岂不挂念？想着你会变化，断然不至伤身。那呆子生得狼犺，又不会腾那，这一去，少吉多凶。你还去救他一救。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">老文想起了小时候看村里常有父母偷偷在儿女间</span>“劫富济贫”——向富儿子（其实也没富多少。那时都很穷）讨要东西，然后偷偷转送穷儿子。</span><span style="font-family: DengXian;font-size: 16px">西游世界的老师们信奉</span><span style="font-family: DengXian;font-size: 16px">“一日为师，终身为父”。看天底下的父母，操心最多或最想</span><span style="font-family: DengXian;font-size: 16px">袒护</span><span style="font-family: DengXian;font-size: 16px">的，绝对是八戒而不是悟空那样的儿女。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">其次，老文猜想，西游世界的老师们可能想为芸芸众生树立信心：八戒能得道成仙，其他人也能。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">这个意图跟物理学家理查德</span>·费曼一致。《</span><span style="font-family: DengXian;font-size: 16px">QED</span><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">：光和物质的奇妙理论》的编录者序写道：</span>“费曼还是个孩子的时候，受过一本书的启迪而开始学习微积分，这本书的开头语是：‘一个笨人会做的事，其他人也会做。’费曼也想把同样的话和他的这本书一起奉献给读者：‘一个笨人能理解的东西，其他人也能理解。’”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;-webkit-text-size-adjust: auto;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">最后，一个老师，能把八戒那样的学生培养成才，相信教其他任何学生都不在话下。老文继续瞎猜，西游世界的老师们之所以热衷招或教八戒，还可能是，招或教八戒那样的学生，最能证明一所学校或一个老师的能力、水平、情怀和价值。</span></p><p style=";text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto"><span style="font-family: 宋体;font-size: 14px"> </span></p><p><br></p>                    <br><br>
