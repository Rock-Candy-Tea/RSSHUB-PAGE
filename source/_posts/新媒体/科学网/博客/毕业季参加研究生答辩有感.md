<p style="margin-top:18px;margin-right:0;margin-bottom:18px;margin-left:0;text-align:center;line-height:150%;background:white"><span style="font-size: 24px;"><strong><span style="line-height: 150%; font-family: 楷体; color: #333333;">毕业季参加研究生答辩有感而发</span></strong></span><span style="font-size: 36px;"><strong><span style="line-height: 150%; font-family: 楷体; color: #333333;"></span></strong><strong><span style="line-height: 150%; font-family: 楷体; color: #333333;"></span></strong></span></p><p style="margin: 18px 0;text-indent: 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="font-family: 楷体; color: #333333; overflow-wrap: break-word; font-size: 20px;">留学近十载，去年下半年回国入职参加工作，今年五月份非常荣幸有两次机会作为答辩会主席参加了硕士的答辩，有感而发，总体感觉学术硕士水平略好于专业学位类型，相对不如我们以前哪个时候答辩的水平，可能专业不一样，不具有代表性，仅此代表个人之见，难免有失偏颇，有些研究生的对待学术态度可能会有问题，感觉有混毕业的感觉，自己同时作为评审员与答辩主持人，有感而发，简单说下几点自己的感受：</span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">1.<span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif;">    </span></span><strong><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">PPT</span>答辩很</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>多都是照着电脑念，缺少与评委的沟通与交流<span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">。</span></strong><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">今年硕士答辩时间要求不低于一个小时的时间，比我们当时的答辩增加了不少时间，很多人都准备50页左右，有些学生讲得还不错，但是还是存在很多的问题，如<strong style="overflow-wrap: break-word">主题内容混乱、重点不突出、逻辑不清晰、都是图或文字</strong>、<strong style="overflow-wrap: break-word">图片来未标注</strong>、讲得云里不知所云，听得昏昏欲睡，很是折磨人，如懒婆娘裹脚，又长又臭，感觉有急来抱佛却的感觉，缺少精心准备。</span></span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">2.<span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif;">    </span></span></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>态</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>度</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>上不够重视</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">。</span></strong><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">很多研三的学生，可能忙于找工作或考公积员，没有把学习当成首要任务，总之有很多的理由，这些都不是作为论文做不好的理由，<strong style="overflow-wrap: break-word">要让学生有对待学位论文的敬畏心理</strong>，杜绝学生来混毕业的错误想法。</span></span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">3.<span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif;">    </span></span></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>摘要写得比较差，不太符合论文的要求</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong style="overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">。</span></strong><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">一般写研究的意义，目前存在的问题，如何改进，然后用例子证明自己的研究，很多研究生，直接上来就讲作了什么工作，给出结果，<strong style="overflow-wrap: break-word">缺少分析与讨论</strong>。</span></span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="font-size: 20px; overflow-wrap: break-word;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">4.<span style="font-size: 20px; font-family: Calibri, sans-serif;">    </span></span></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>论文的架构有问题。</strong></span><span style="font-size: 20px; overflow-wrap: break-word;"><strong style="overflow-wrap: break-word"><span style="font-size: 20px; font-family: Calibri, sans-serif; color: #333333;"> </span></strong><span style="font-size: 20px; font-family: 楷体; color: #333333;">论文都是七拼八凑，缺少一个主线构成一个STORY</span><span style="font-size: 20px; overflow-wrap: break-word;">，分不清哪些是自己的工作，还是他人的研究工作，逻辑不清楚。</span></span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">5.<span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif;">    </span></span></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>文献调研不够。</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">国内外研究现状，才写<strong style="overflow-wrap: break-word">一两页或半页</strong>，体量上不够，且与自己的研究内容关联性不太强，说明文献调研不够，文献综述质量比较差。</span></span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">6.<span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif;">    </span></span></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>科研伦理意识不强。</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">很多学生的论文中公式，图片，数据来源都没有标</span><span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif; color: #333333;"> </span> <span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif; color: #333333;"> </span><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;"> 注，文中标出实际引用位置与文献对应不起来，每次我都在组会提醒我的学 要注重引用来源，加强科研伦理意识。</span></span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">7.<span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif;">    </span></span></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>工作量不足。</strong></span><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">有的论文才写40页左右，扣除前后别人的东西，自己写的也就10来页左右，虽然页数不能说明一切，但也可以看出来有些研究生没有付出一定的时间和精力去完成毕业论文的研究和撰写工作，匆忙写成，纯粹敷衍了事。</span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">8.<span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif;">    </span></span></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>中英文表达有问题。</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif; color: #333333;"> <span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体;">英语摘要有明显的语法错误，比如基本上单复数，专业名词要大写，第一次文中出现，要用全文表示等，中文表达语句不通顺，错误字不少，没有仔细检查，书中口语化表达比较严重，另外还缺少基本的学术规范。</span></span></span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">9.<span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif;">    </span></span></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>图表模式不统一，质量比较差</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">。图好多是从商业软件导出来的，分辨率太差，看不清楚，另外图中缺少图例，如常用的比例尺，指南针，坐标等，图或从生产报告直接copy过来的。</span></span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">10.<span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif;">   </span></span></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>对讨论不够重视。</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">仅仅对图表的简单的描述，没有进一步挖掘，缺乏对自己的研究与他人的研究比较。</span></span></p><p style="margin: 18px 0 18px 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">11.<span style="overflow-wrap: break-word; font-size: 20px; font-family: Calibri, sans-serif;">    </span></span></span><span style="overflow-wrap: break-word; font-size: 20px;"><strong>论文的创新性不太强。</strong></span><span style="overflow-wrap: break-word; font-size: 20px;"><span style="overflow-wrap: break-word; font-size: 20px; font-family: 楷体; color: #333333;">论文的创新性写得太泛，不够具体，说明还不了解自己的研究内容，理论高度不够，看不出来与以往过去的研究有什么不同之处和创新点的所在之处。</span></span></p><p style="margin: 18px 0;text-indent: 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="font-family: 楷体; color: #333333; overflow-wrap: break-word; font-size: 20px;">总而言之，论文对于研究生本人是非常重要，不能有些论文不要求上传到中国知网数据库上就降低了对自己的要求，这样是对自己不负责的表现，要让研究生对学位产生敬畏的心态。指导老师与研究生都应该重视起来，特别是研究生本人，</span><span style="color: #333333; font-family: 楷体; font-size: 20px;">走过的路不会白走，人的一生短短几个秋，总要留下点痕迹，论文也是对研究生这几年学习成果的集中体现，否则，出来混的迟早都应还的，应该引起思想上的足够重视。</span></p><p style="margin: 18px 0;text-indent: 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="color: #333333; font-family: 楷体; font-size: 20px;">                                    </span></p><p style="margin: 18px 0;text-indent: 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="color: #333333; font-family: 楷体; font-size: 20px;">                                  在办公室写于2021年5月30日晚11：45</span></p><p style="margin: 18px 0;text-indent: 48px;line-height: 150%;background: white;overflow-wrap: break-word"><span style="font-family: 楷体;
