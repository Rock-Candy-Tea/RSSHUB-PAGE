text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">这是写讨论部分时需要做的第一件事。简要描述你从结果中得出的结论，然后就已知的内容解释它的含义。</p><p class="ql-align-justify" style="border: 0px; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">记住要强调你的结果是如何支持或驳斥该领域当前假设的。</span></p><p class="ql-align-justify" style="border: 0px; white-space: normal; background-color: rgb(255, 255, 255);">如果你的研究结果与大牛的结果产生冲突，讨论部分也是一个解决这类问题的好地方。通过解决这些冲突，你所在领域的科学家将重新检查和重建假设/模型，然后进行验证。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">确保你首先提到最重要的发现，这是人们会记住的。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">3.承认研究的不足</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">在讨论中，解释你的假设或实验方法可能存在的任何限制以及它们背后的原因。这将帮助该领域产生假设和新的方法，而不必面对同样的挑战。</p><p class="ql-align-justify" style="border: 0px; white-space: normal; background-color: rgb(255, 255, 255);">根据你发表在哪个期刊上，你可能必须提供一个单独的“未来方向”部分，而不是把它与讨论联系在一起。尽管如此，在你写讨论的时候，你应该思考你的研究可能会导致的问题。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em;
