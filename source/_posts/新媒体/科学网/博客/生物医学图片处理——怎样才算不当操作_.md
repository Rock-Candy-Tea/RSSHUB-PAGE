white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px; color: #4F81BD;">裁剪（Cropping）</span></p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">通常的做法是去除凝胶或Western blot中不相关的部分。然而，这是不可接受的裁剪图片的方式，这会导致数据失真或删除了必要的信息</span>。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">使用图像处理来修复错误是非常诱人的</span>，例如，如果你在凝胶上加载了太多的蛋白质，导致了过度饱和的条带。数字调整波段，使其看起来不那么饱和，可能对数据的整体解释没有任何影响; 但是，这并不是对你所观察到的情况的准确描述。</p><p style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0;
