margin: 10px auto 0px; padding: 0px; display: block; height: auto;" referrerpolicy="no-referrer"></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">编者按：开放科学及其在社会科学和人文科学中的广泛应用，是以研究可以在数字平台上复制和共享为前提的，但研究人员在多大程度上能真正将数字工具作为其工作的一部分呢？Deirdre Watchorn指出开放科学政策应该对这些不同类型的学科研究采取更为细致的区分方法。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">在过去几年里，随着新任务的传输和新商业模式的高速发展，所有学科都在朝着开放科学的方向转型。然而，这些倡议大多以科学和技术领域的研究文化为出发点。人文和社会科学即使被考虑在内，也常常不被受到重视。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">本文作者对人文和社会科学（HSS）研究人员的工作实践进行的深入研究证实了这种趋势。更重要的是，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">它表明了开放科学的某些要求与人文社会科学研究本身的性质之间存在脱节的现象。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">以欧盟委员会对开放科学的定义为例：“开放科学是一种系统性的变革。通过开放和合作的方式，在研究过程中尽快生产和分享知识和数据，并交流和共享结果，从而实现更好的科学实践。”</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">欧盟及其他资助机构对开放科学的定义则是基于以下关键想依据的：</p><ul style="list-style-type: none;" class=" list-paddingleft-2"><li><p>学术研究涉及分析和产生某种数据；</p></li><li><p>这种数据可以通过电子方式存储或共享；</p></li><li><p>它是通过数字化协作产生的。</p></li></ul><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">但是研究表明，许多人文社科领域的学者很少使用数字化手段和协作工具。他们的研究对象通常是文本、人工制品、收藏品和其他实物等，<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">因此可能很难理解如何 “共享数据”。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">传统方法盛行</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">本文作者对德国、奥地利和瑞士的641名人文社科研究人员进行了一项调查，其中包括对14名学者的深入访谈。调查发现，绝大多数学者准备、计划、组织和撰写文章时很少使用线上合作平台、工具和参考文献管理软件。94%的社会科学家表示他们习惯使用Word来写文章以及管理参考文献。此外，由于人文科学学者多是单独从事研究的，在需要协作时，他们更愿意使用电子邮件。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">该研究还表明，人文社会科学研究人员更喜欢在图书馆、档案馆和收藏馆等实体空间工作，并重视在这些环境中工作所带来的灵感。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">人文和社会科学的研究往往是一个创造性的发现过程。在被文字、书籍、工艺品和艺术作品包围的实体空间里工作，可以培养和支持这种创造性的过程，而这一点数字化环境无法做到。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">关于“数据”共享的问题</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">该研究发现，只有1%的人文学者需要进行涉及 “数据 ”的实验，只有6%的人选择量化研究。绝大多数人都在分析原始资料，并利用各种形式的二手资料进行研究。人文研究的性质意味着这些学者不需要经常分析数据集。相反，他们研究文本、进行访谈、分析日记、图像、录音、档案片段、信件和视频等。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">考虑到这一点，也就难怪许多人文社科学者对 “开放数据 ”的概念感到挣扎。对于一个分析《伊利亚特》的古典主义学者来说，“数据 ”意味着什么？对于一个独立研究西班牙内战的历史学家来说，“数据共享 ”意味着什么？<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">即使我们把人文学者工作的原始资料视为数据，当这些数据被放置于档案馆的文件夹、盒子和书架上，并且从未被数字化时，这些学者应该如何分享它们呢？</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">HSS的学者们并不抗拒变化，却也不愿意使用新技术。像Journal of Digital History（https://journalofdigitalhistory.org/en）发起的新举措证明，人文科学研究人员其实是希望接受数字化驱动的学术形式的。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">但就目前而言，数字人文科学仍然是一个规模相对较小的子领域。</span>虽然会有诸多原因导致一些人文社科研究者仍倾向于传统的工作模式，但我们只能假设他们这样做是因为这些方法仍然有效。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">虽然在许多情况下，科学、技术和医学（STM）学者确实使用更先进的数字化工具来执行和评估他们的数据并进行合作，但这可能是因为他们的需求不同。<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">人文社会学者看到了开放科学的优势——无论是对其更广泛的学科的透明度还是对他们自己工作的好处。他们只是不一定看到开放科学该如何适用于自己。</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">如何转变</span></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">那么，什么需要改变？<span style="font-weight: 700; border: 0px; margin: 0px; padding: 0px;">资助者和政策制定者必须考虑人文和社会科学研究文化的特殊性和多样性，因为 “数据 ”对不同的学科意味着不同的东西。</span>此外，研究人员本身也需要考虑数据共享对人文学科意味着什么，并更加熟悉开放科学的挑战，以便他们能够制定适合自己的要求。值得庆幸的是，The Digital Research Infrastructure for the Arts and Humanities（DARIAH）等平台在帮助实现这一目标方面发挥了关键作用。</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);"><img src="https://p1.itc.cn/images01/20220718/de198906402c42ebbbacd3c2005b279f.png" style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; height: auto;" referrerpolicy="no-referrer"></p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">https://www.dariah.eu/</p><p class="ql-align-justify" style="border: 0px; margin-top: 0.63em; margin-bottom: 1.8em; padding: 0px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: justify; color: rgb(25, 25, 25); font-family: "PingFang SC", Arial, 微软雅黑, 宋体, simsun, sans-serif; white-space: normal; background-color: rgb(255, 255, 255);">我们对学术工作流程的理解必须考虑到，不同领域的学者有不同的需求。我们的研究发现，人文社科学者在实体场所茁壮成长，被书籍、实物、艺术作品和人工制品所包围。他们很少使用 “数据集”，他们希望沉醉于原始文本中，发现新的思路并开拓新的领域。尽管数字创新可以推动许多领域的发展，但有时一支笔、一间图书馆和一本笔记本才是一些学者真正需要的。</p><p class="ql-align-justify" style="border: 0px;
