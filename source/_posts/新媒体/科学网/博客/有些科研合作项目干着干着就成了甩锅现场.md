<p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;text-align: justify;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">科研工作和科研成果的诞生如今非常依赖团队合作。鉴于各领域科技发展迅猛，科研问题越来越复杂，在全球范围内，科研合作以及科研人员之间的沟通交流越来越频繁和顺畅，尤其是跨领域科研合作，近年来得到了更多的关注和支持。</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;text-align: justify;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">积极有效的科研合作往往能够极大地促进科研人员的事业发展。在合作中，不仅可能迸发出一些出色的灵感，还能够拓宽自身眼界和见识，更有效率地利用各种科研资源。</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;text-align: justify;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">科研合作的方式有很多，例如科研人员作为个体到其他单位进行人力合作；利用经费优势纳入科研同行，开展联合研究的经费合作；为其他单位提供技术服务的技术合作；以及科研人员通过交换思路进行的思路合作。以人力合作为例，在合作当中，可以利用其他人的才能，优势互补，因此对于解决很多科研过程中遇到的困难非常有必要。</span></p><h2 style="box-sizing: border-box;margin: 0 0 11px;font-family: 'Alegreya Sans';line-height: 1.18;color: rgb(4, 0, 0);font-size: 26px;white-space: normal;background-color: rgb(255, 255, 255);text-align: justify"><span style="font-weight: bolder; box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">成功科研合作的关键</span></h2><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">不论哪种方式，想要进行卓有成效的科研合作，有一些技巧和注意事项大家应当知晓。</span></p><p><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">1. 首先，合作者要有共同的目标和期望值</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">微软亚洲研究院常务副院长张冬梅博士曾在一次论坛上说，她认为两个团队能成功合作有三个关键点：共同的愿景、信任与共同提升。其中，共同的愿景和期望是两个团队合作成功的关键<span style="font-size: 12px; font-family: 微软雅黑, "Microsoft YaHei"; line-height: 0; vertical-align: baseline; top: -0.5em; box-sizing: border-box;">[1]</span>。</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">两个团队要共同达成的目标不会在一夜之间实现，这当中包含很多未知因素和不确定性，需要探索、经历许多尝试和失败。这会让科研合作变成一个长期、循序渐进的过程，这也符合科学研究的特质。而共同的目标和期望是保持合作长期有序开展的关键。共同的目标和愿景就像灯塔，合作双方就像海洋中相隔很远的两艘船，只有合作双方有共同的愿景和目标，才能克服诸多问题、困难，向一个目标持续推进，最终汇合。</span></p><p><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">2. 将合作关系和各方研究内容正规化、专业化</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;text-align: justify;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">最好有一份书面文件或者合同，可以是个人层面的，也可以在单位层面签署。比如文章署名协议、资源分配协议、工作分工协议等等。将各自责任和权利、工作内容、目标、流程及时间线明确提出来。这样一方面可以杜绝如署名权争议这样的麻烦，也会因分工定义清晰、协商一致，让科研合作项目执行得更加顺利。</span></p><p><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">3. 建立规律和有效的沟通</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">频繁和规律的沟通有时候能让合作者之间迸发出灵感的火花，尤其是不同研究领域的研究者之间的沟通，往往能迸发一些天才的想法。如果想要科研合作更加成功，频繁规律的沟通必不可少。合作双方应该根据各自情况，安排每周或者每月的例会，互相汇报研究进展，对遇到的问题共同进行讨论。</span></p><p><span style="box-sizing: border-box; font-size: 16px; font-family: 微软雅黑, "Microsoft YaHei";">4. 互相信任和包容</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">在合作中缺乏耐心和包容，难以为共同目标做出妥协、相互帮助是很多科研合作失败的主要原因。要在合作中建立信任，最重要的是遵守承诺、有责任心、透明和包容，这些都体现在日常的工作中。</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;text-align: justify;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="font-weight: bolder; box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">科研合作的巨大创新潜力</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">在交叉领域的研究埋藏着巨大的创新潜力。比如 Google 公司在 2021 年发布的结合深度学习和结构生物学的 AlphaFold2，就被很多人认为是结构生物学领域的颠覆性创新。</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="box-sizing: border-box; font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">事实上，跨领域研究的复杂课题采用合作的研究方式几乎已经成为了一种常态。毕竟，要一个研究者同时掌握多个领域的方法和知识几乎是不可能的。例如近年来发展迅猛的医疗人工智能研究领域，让一个人工智能专家掌握临床医生的专业技能，这几乎是天方夜谭，反之也一样。而当来自不同背景和视角的研究人员进行合作，那么有可能会诞生一些极其出色的研究成果。</span></p><p style="box-sizing: border-box;margin: 0 0 11px;font-size: 17px;line-height: 32px;color: rgb(4, 0, 0);font-family: 'Alegreya Sans';white-space: normal;background-color: rgb(255, 255, 255)"><span style="font-family: 微软雅黑, "Microsoft YaHei";
