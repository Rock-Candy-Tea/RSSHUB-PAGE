style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px">                                    ------------------<span style="font-family:宋体">观《小舍得》有感</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px">                                                   <span style="font-family:宋体">李祥刚</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"> </span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">应孩子班主任谢老师的要求，观看了电视剧《小舍得》。这部剧反映了很多现实问题，如素质教育、教育公平等，其中印象最深的是电视剧生动形象的展示了现在的热门话题</span>“内卷”。内卷听起来有点拗口，</span><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">夏君山</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">和钟益</span></span><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">在剧中的一段话，</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">解释了</span></span><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">内卷的概念</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">，</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">虽然他们没有直接说出这个词</span></span><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">。</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">夏君山说：</span>“一个剧场，大家都在看演出，突然一个观众站了起来，其他观众为了能看到演出也不得不站了起来，最后大家都从坐着看戏，变成站着看了，这不是闲的吗？” </span><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">钟益</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">接着</span></span><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">说：</span>“<span style="font-family:宋体">还没完呢啊，有人就站上椅子了，有人把梯子给架起来了。</span><span style="font-family:Arial">”</span><span style="font-family:宋体">夏君山</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">抱怨道</span></span><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">：</span>“<span style="font-family:宋体">何苦呢，付出那么高的成本，却只能得到和原来一样，甚至更差的体验。</span><span style="font-family:Arial">”</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"> </span><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">钟益说出了大家的心理：</span>“<span style="font-family:宋体">关键是，没人敢坐下来啊。</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF">”这个剧场看演出的例子就是一个典型的内卷案例，大家付出了更多的努力，这些努力却并没有带来任何整体的收益。</span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">最早关注到内卷这种现象，是上大学走出家乡后，将家乡的一些事情与外地做比较得出的。比方说，我家乡人特别勤劳，有点活儿就抢着干。然而，勤劳的优点并没有带来收入的提高。我印象中有个高层建筑外墙清洁的工作，因为抢着干的人太多导致我老家当地的工价比相同级别的有些地方工价低了三分之一到二分之一。甚至听闻有个公职人员有机会分配这个工作却也想把钱赚了，直接自己上阵，结果因为不专业摔死了的惨剧。然而，我读了很多书，走出了大山，却并没有跳出内卷的环境。我从事的科研工作也是一个高度内卷的行业。我观察到一些现象后仿叶圣陶先生的作品写了一篇博客《多发表了三五篇》（</span>http://blog.sciencenet.cn/blog-710487-1065332.html<span style="font-family:宋体">），描述了这种现象。</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">内卷看似无处不在，不在这里卷就在那里卷。如果我在科研中卷输了，那回到老家去刷墙、种地，同样得跟人家卷。如果孩子在中考中输了，他同样要去中专、技校中卷。我们不是剧中颜鹏这样的富二代，内卷对大多数人而言貌似是个无解的题，生活中的人似乎逃无可逃。但是，我是一个固执的人，多年来一直在寻找摆脱内卷的出路。我很努力，但是我希望我的努力大部分可以产生结果能为社会做点贡献，而不是全部的能量用于卷赢他人。同样，我希望孩子也很努力，但是所有付出的辛苦应该大部分用于学识、人品等的提高，而不是仅仅在升学竞争中卷赢他人。在我看来，因为升学考试是属于竞争性考试，而不是水平考试，为了在竞争中取胜，很多时候已经比较大的偏离了教育的本质。比方说我们不是一个知识水平掌握了之后，就进入更高阶段知识的学习。而是在这个水平上花费更多的精力，把相关习题练习到万无一失。</span> <span style="font-family:宋体">我比剧中的夏君山夫妇还不妥协，目前还没有给孩子上任何课外培训班。</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">下面是我努力跳出内卷的一些思路，希望能与广大家长老师一起探讨：</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style="font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF">（1）</span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">幸福是多维的，防止一维评判的思维误区</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">大部分人无意识的追随着社会的一维评判标准，比方说，上学的时候就是比谁成绩好，工作后就是比谁赚钱多。这种评判标准一目了然，很方便排名，几乎所有人都是总是比上不足比下有余。很多人鄙视着排在自己后面的人，又焦虑着追上排在前面的人。一旦追上了切近的目标，会有更高的目标出现在眼前。于是更加努力，直到心力交瘁，达到自己能力和体力的极限。</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">由于大部分人都是普通人，一维排名容易让人绝望</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">。</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">我小学时候，就立志当爱因斯坦那样伟大的科学家，但是到了初中，无论怎么努力都没有考过年级第一，才开始怀疑什么天道酬勤之类的话是不是不可信，有些领域还是要天赋的。然后我就计算，全国有多少个县，全世界有多少个国。。。完了，我永远不能站在顶尖了，我的人生将像蝼蚁一样，无法在世界上留下任何痕迹了。。。</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体"></span></span><br></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">但其实人的幸福是多维的。身体健康、心理健康、家庭和睦、人际良好</span>.......<span style="font-family:宋体">很多的方面都一起构建了幸福的大厦。如果我们把更多维度的东西纳入考察的范围，就发现每个家庭都有自己独特的幸福，某一个维度的落后，所起的作用其实没有想象中的那么大。我们就不用在某一维度上过于焦虑。</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style="font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF">（2）</span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">树立远大理想，精选职业和行业，把时间花在细分领域的深耕</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF">   <span style="font-family:宋体">目前大家在大学以前的焦点，往往集中在考个好分数，上个好大学上面，而忽视了从小就与孩子一起探讨理想，探索合适的职业和行业。职业和行业是对大部分人而言贯穿始终的，占据的时间可能比任何其它事情都多，应该花大力气去研究和决定。</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"> <span style="font-family:宋体">如果一个人能将自己的事业建立在为人民大众谋幸福的基础上，他将不再拘泥于一时的得失，心胸会开阔很多，不会过于焦虑，一般也不会过不好自己的小家庭。当然，只要是正经的行业和职业，都是对社会有贡献的，所以对职业和行业的考虑应该放在是适合自己上。主要包括适合自己的兴趣、天赋等各方面，这并不容易通过思考来确定，而需要通过不断的尝试。家长应该尽早给足孩子足够的尝试空间。</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">很多家长担心，孩子喜欢从事的专业不赚钱怎么办。其实我觉得没必要。只要这个职业还在，不至于不能养家糊口。所谓赚不赚钱，大家往往比较的是同等级别的人，从事不同的行业，收入不一样。但是忽略了一个问题，同一个人，从事适合自己的行业和不适合自己的行业所达到的级别可能差别很大。比方说，《明朝那些事儿》的作者</span>——当朝明月，他喜欢研究的是历史。历史本来是个不好赚钱的专业。可是他做的好啊，成了畅销书作家，也就不缺钱了。剧中的颜子悠，非常热爱生物，但是他妈妈总觉得他不务正业。其实他这么小，能发掘出自己挚爱的事业，无论对家长还是孩子都是非常好的事情，家长应该正确引导，大力支持。因为越早定好自己的发展方向，越容易聚焦。从此不再是与周围的孩子进行泛泛的竞争，而是深耕自己喜欢的细分领域，早日达到该领域的头部。世界上人那么多，没必要在方方面面都做所谓人上人。世界是丰富多彩的，每个人都有自己擅长的一方面，将自己所长做到最好，为周围人提供服务，世界也因此变的丰富多彩。这样人们就不再同质化竞争有限的资源，自然就跳出了内卷的陷阱。</span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style="font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF">（3）</span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">在重要的事情上多花时间</span> </span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"> <span style="font-family:宋体">人从小到大，免不了各种指挥棒。如果顺着指挥棒，一会儿向东，一会儿向西，总感觉像无头的苍蝇，疲于奔命。应该将人生做一个统筹规划，找出重要的事情，随着时间价值越来越大的事情。应该把主要的精力和时间集中地放在处理那些重要但不紧急的工作上，这样可以做到</span></span><a href="https://baike.baidu.com/item/%E6%9C%AA%E9%9B%A8%E7%BB%B8%E7%BC%AA/839102" target="_blank"><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">未雨绸缪</span></span></a><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">，</span></span><a href="https://baike.baidu.com/item/%E9%98%B2%E6%82%A3%E4%BA%8E%E6%9C%AA%E7%84%B6/2888491" target="_blank"><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">防患于未然</span></span></a><span style=";font-family:Arial;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">。</span></span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">比方说，对英语而言，每天坚持花一点时间练习听说读写就是一件重要的事情。也许学的东西不会马上考，但长期的坚持可以使一个人系统的学好英语。一旦将英语系统掌握，就不但不再担心各种考试，还可以在各方面应用。对没有系统花时间学英语的人，也许他在考大学、考四六级、考研等时间点还要反复拿出时间临时抱佛脚的学习。而长期做重要事情的人，这时候可以腾出时间做其它重要事情了。</span></span></p><p style="margin-top:22px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:32px;padding:0 0 0 0 ;text-align:justify;text-justify:inter-ideograph;line-height:24px"><span style="font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF">（4）</span><span style=";font-family:宋体;color:#333333;letter-spacing:0;font-style:normal;font-size:16px;background:#FFFFFF"><span style="font-family:宋体">节制消费，精简娱乐，学会时间管理与财富管理</span></span></p><p
