<p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">唐僧师徒灭了白骨精，降了黄袍怪，又行进到</span>“巍巍峻岭削削尖峰”的平顶山。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">“徒弟们仔细。前遇山高，恐有虎狼阻挡。”唐僧的直觉再一次得到即时应验，日值功曹化作樵子提供重要情报：“那西进的长老！暂停片时。我有一言奉告：此山有一伙毒魔狠怪，专吃你东来西去的人哩。”</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">“长老闻言，魂飞魄散，战兢兢坐不稳雕鞍。”</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">八戒跟沙僧提议：</span>“把白马卖了，买口棺木，与师父送老，大家散火。”</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">沙僧道：</span>“二哥，分怎的？”</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">“好大圣，全然无惧，一心只是要保唐僧。”</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">有人魂飞魄散，有人叫嚷散火，有人和和稀泥，有人全然无惧，这基本上是取经团队每次面对妖怪时的常态。但这一次，各人的常态近乎夸张。他们感觉到了即将遭遇的毒魔狠怪的确如日值功曹所通报的毒和狠。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">悟空怀疑日值功曹</span>“你这等长他那威风，胡言乱语的拦路报信，莫不是与他有亲？不亲必邻，不邻必友”。悟空的怀疑未必在理，他或许另有意图，就像他换一套说辞去转告唐僧和八戒。不妨先扒一扒这伙毒魔狠怪。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">第一，背景。为首的金角大王和银角大王从基层到天庭都有深厚背景。在天庭，他们都是太上老君身边的人：</span>“那两个怪，一个是我看金炉的童子，一个是我看银炉的童子。”在平顶山，他们把山神、土地当奴仆使唤。银角使法压住悟空的三座大山，就是他遣山神、土地挪来的。可别小瞧基层小人物，相对于天庭背景，基层关系才叫悟空哭天喊地：“苍天！苍天！今日这个妖魔无状，怎敢把山神、土地唤为奴仆，替他轮流当值？天啊！既生老孙，怎么又生此辈？”</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">第二，实力。金角银角的势力范围达六百里远近。之前白骨精的势力范围只有区区四十里（</span>“若过此山，西下四十里，就不伏我所管了”），相对于金角银角，是名符其实的小巫见大巫。不仅如此，金角银角还有一个庞大的家族势力。他们的母亲九尾狐狸和老舅爷狐阿七大王盘踞在平顶山附近的压龙山，他们随时可以“会借外家亲戚”。从妖怪数量看，金角银角自己洞中有四五百兵，狐阿七手下有二百余妖，九尾狐狸手下的妖怪数量未见报道，保守估计不会少于她的儿子狐阿七。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">第三，武器。太上老君是全宇宙最大最牛军火制造商，大凡你能说出来的厉害武器，包括悟空的金箍棒、八戒的九齿钉耙，无不出自他之手。金角银角被观音借去执行秘密任务，代表老君的脸面，配备的武器从性能到数量自然绝对不一般。他们有画影图形，相当于有今天的人脸和图像识别技术，要捉的和尚</span><span style="font-family: DengXian;font-size: 16px">别想蒙混过关</span><span style="font-family: DengXian;font-size: 16px">（</span><span style="font-family: DengXian;font-size: 16px">八戒</span><span style="font-family: DengXian;font-size: 16px">已见识过了</span><span style="font-family: DengXian;font-size: 16px">）</span><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">；他们</span>“随身有五件宝贝（分别是紫金红葫芦、羊脂玉净瓶、幌金绳、七星剑和芭蕉扇），神通极大极广。就是擎天的玉柱，架海的金梁，若保得唐朝和尚去，也须要发发昏是。”</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">事后证明，日值功曹作为取经团队的空中护卫队员，事前通报的情况件件是实。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">有两个疑问值得琢磨。其一，日值功曹在描绘毒魔狠怪多么多么厉害时，如果说悟空全然无惧是他的一贯作风，那么他还表现出人逢喜事精神爽般无比兴奋却是非同寻常。毕竟逢妖不是逢喜，悟空为什么无比兴奋？</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">其二，从背景、实力、武器等诸多因素看，平顶山的毒魔狠怪的确前所未有的毒和狠，如此毒狠的妖怪，悟空竟以前所未有的好玩、打趣、轻松心态，酣畅淋漓地灭掉了他们。越毒越狠的妖怪越不经打，这其中又有什么玄机？</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">老文推测，两个疑问的答案线索都藏在日值功曹的情报中。俗话说，听话听音，锣鼓听声。悟空听清了</span>“一伙毒魔狠怪”。正是“一伙”，令悟空喜出望外。也正是“一伙”，使妖怪不堪一击。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">悟空自师从菩堤祖师学成归来，到大闹天宫，再到加入取经团队一路降妖，经历过大大小小无数次战斗。他的实战经验和切身体会是：胜一人难，灭一伙易。胜一人难，只要是一人，小鬼都难缠；灭一伙易，只要是一伙，不管其中个体多么强大，灭之都容易，打起来也好玩，再说什么背景、实力、武器等等，都是在扯淡。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">悟空最刻骨铭心的记忆是五百年前大闹天宫。他一人战十万天兵天将，虽未灭掉那一伙，但也令那一伙狼狈不堪，并因此为自己树立了</span>“十万军中无敌手，九重天上有威风”的声誉。他有两次耻辱的惨败，都是败在一人而非一伙手上：一次被</span><span style="font-family: DengXian;letter-spacing: 0;font-size: 16px;background-color: #FFFFFF"><span style="font-family:DengXian">擒，栽在</span></span><span style="font-family: DengXian;font-size: 16px">二郎神</span><span style="font-family: DengXian;letter-spacing: 0;font-size: 16px;background-color: #FFFFFF"><span style="font-family:DengXian">杨戬手上；一次被压，栽在</span></span><span style="font-family: DengXian;font-size: 16px">如来佛祖手上。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">取经团队成立后，先后遭遇一女一男两个妖怪，教训就在眼前：单个妖精，无论是男是女，不好除。女妖白骨精本事平平，之所以能三戏唐僧，并把悟空折磨得死去活来，是因为一方面白骨精是单兵作战，胜之很难，另一方面</span>“灭一伙易”同样为白骨精所掌握，她挑战的是还似“一伙”的所谓团队，而不是个人。男妖黄袍怪已把唐僧变成一只斑斓猛虎，锁在铁笼里。</span><span style="font-family: DengXian;font-size: 16px;background-color: #FFFFFF"><span style="font-family:DengXian">若不是</span>“</span><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">暗中有那护法神祇保着他</span>”，唐僧早就成了黄袍怪胃中之物，取经团队就此溃散。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">平顶山除妖，是取经团队西行路上赢得最干净利落的一战。所有上千号大小妖精一个不落全被灭</span><span style="font-family: DengXian;font-size: 16px">,</span><span style="font-family: DengXian;font-size: 16px">“山已净，妖已无矣”。金角银角的五件宝贝全被缴，用悟空的话说都“姓孙了”。正因如此，这一战也让太上老君颜面尽失。在取经团队带着胜利的喜悦准备再出发时，老君放下身段，扯住唐僧的马，要唐僧“还我宝贝来”。唐僧有生以来第一次有地位如此崇高之大神当面求他要回战利品，那感觉定然妙不可言。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: 14px;white-space: normal;line-height: 28px"><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">胜一人难，灭一伙易。太上老君从平顶山一战也看清了这个规律。后来，取经团队行至金兜山时，青牛精为老君找回了颜面。青牛精是老君的坐骑，地位应该不比金角银角高。他带的武器（一个金刚琢）也不如金角银角多。但他一人出战，施个小计轻而易举就把唐僧、八戒和沙僧捉了，把前来营救的悟空打得稀里哗啦。悟空只好不断求援，请来一波又一波救兵，先后请过</span></span><span style="font-family: DengXian;line-height: 32px;font-size: 16px;background-color: #FFFFFF"><span style="font-family:DengXian">李天王、哪吒、雷公、火德星君、黄河水伯、如来、十八罗汉等仙佛两界的高手。但这不仅无济于事，还戳破了他</span></span><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">在平顶山时对日值功曹炫耀过的神通：</span>“我老孙到处里人熟，发一张批文，把他（妖精）连夜解着飞跑”。递解老君身边的</span><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">任何个</span></span><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">人，除了求老君，请来其他任何</span></span><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">大牛</span></span><span style="font-family: DengXian;line-height: 32px;font-size: 16px"><span style="font-family:DengXian">都没用。</span></span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">我们也许有不解：不是说人多力量大么，为什么人多反而少了战斗力呢？</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">悟空告诉我们，人多力量大是有前提的：一看干什么样的事或实现什么样的目标，二看如何组织人多。没了这两个前提，那么十之八九</span><span style="font-family: DengXian;font-size: 16px">，</span><span style="font-family: DengXian;font-size: 16px">一伙的力量不如一人。</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px">神妖世界如此，人世间也莫不如此。后者案例在《三国演义》</span><span style="font-family: DengXian;font-size: 16px">中</span><span style="font-family: DengXian;font-size: 16px">不胜枚举</span><span style="font-family: DengXian;font-size: 16px">。</span><span style="font-family: DengXian;font-size: 16px">例如</span><span style="font-family: DengXian;font-size: 16px">，</span><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">曹操有个类似秘诀：胜一人难，胜二人易。在</span>“抹书间韩遂”一回，众将向曹操请教：“丞相每闻贼加兵添众，则有喜色，何也？”“加兵添众”跟“一伙”无异，可见曹操跟悟空英雄所闻略同：闻之均喜形于色。曹操</span><span style="font-family: DengXian;font-size: 16px">透露了玄机</span><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">：</span>“关中边远，若群贼各依险阻，征之非一二年不可平复；今皆来聚一处，其众虽多，人心不一，易于离间，一举可灭：吾故喜也。”</span></p><p style="margin: 0 0 8px;text-align: justify;font-family: Calibri;font-size: medium;white-space: normal;line-height: 32px"><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">时有</span>“</span><span style="font-family: DengXian;font-size: 16px">青</span><span style="font-family: DengXian;font-size: 16px"><span style="font-family:DengXian">椒</span>”疑惑要不要加入或组建科研团队。悟空启发我们，这不能一概而论。如果从事牛顿或爱因斯坦那样的研究（无法分解和分工），加入或组建团队不是拖累自己就是误了别人。如果从事取经工程那样的工作，无论是平庸如八戒和沙僧，还是非凡如悟空，都离不开团队。至于加入或组建怎样和多大的团队，估计神仙都无妙策，全凭各人的智慧和眼光。有一点是肯定的：不管怎样和多大的团队，如果是或似一伙，战力不抵一人。</span></p><p><br></p>
