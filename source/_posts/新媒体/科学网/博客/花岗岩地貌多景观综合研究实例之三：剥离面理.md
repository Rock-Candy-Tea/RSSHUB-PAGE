color: blue;">    作为后来者，我们无法抛开国际上早已公认的名称而自己另搞一套，否则我们无法与国际同仁进行学术交流。因此建议不再使用“层裂”一词，统一使用早已存在且公认的标准名称：</span><span style="font-family: 宋体; color: blue; font-size: 20px; text-decoration: underline;"><strong>剥离面理</strong></span><span style="font-size: 20px; font-family: 宋体; color: blue;">（<span style="font-size: 20px; color: blue; font-family: "times new roman";">exfoliation/exfoliation joint</span>）。</span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; color: blue;">    我是国内地学同仁中研究花岗岩剥离面理最多的学者之一，从<span style="font-size: 20px; color: blue; font-family: "times new roman";">2006</span>年起在研究论文和教材中一直使用“</span><span style="font-size: 20px; text-decoration: underline; color: blue;">剥离面理</span><span style="font-size: 20px; color: blue;">”这个词汇，对应于英文词汇“<span style="font-size: 20px; color: blue; font-family: "times new roman";">exfoliation</span>”，表示花岗岩露头上这种大型面状构造。我使用“剥离面理”这个名称主要考虑下述因素：</span></span></p><p style=";margin-top:0;margin-bottom:0;text-align:justify;text-justify:inter-ideograph;direction:ltr;unicode-bidi: embed"><span style="font-size: 20px;"><span style="font-family: "Times New Roman"; color: blue;">      <span style="font-size: 20px; color: blue; font-family: 宋体, SimSun;">英语单词</span>exfoliation</span><span style="font-family: 宋体; color: blue;">由</span><span style="font-family: "Times New Roman"; color: blue;">ex-</span><span style="font-family: 宋体; color: blue;">和</span><span style="font-family: "Times New Roman"; color: blue;">foliation</span><span style="font-family: 宋体; color: blue;">两部分组成，</span><span style="font-family: "Times New Roman"; color: blue;">ex-</span><span style="font-family: 宋体; color: blue;">有“排除，外表</span><span style="font-family: "Times New Roman"; color: blue;">…”<span style="font-size: 20px; color: blue; font-family: 宋体, SimSun;">之意；而</span>“foliation”</span> <span style="font-family: 宋体; color: blue;">在地质学中是“面理”。同时应注意，</span><span style="font-family: "Times New Roman"; color: blue;">exfoliation</span><span style="font-family: 宋体; color: blue;">在本义是“剥落、表皮脱落、剥离”。结合地质学中的意义，翻译成</span><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="color: blue;">“</span><span style="text-decoration: underline; color: blue;">剥离面理</span><span style="color: blue;">”比较恰当。</span></span></span></p><p style=";margin-top:0;margin-bottom:0;text-align:justify;text-justify:inter-ideograph;direction:ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; color: blue; font-size: 20px;">    依照惯例，科学术语的使用应该坚持三个遵从：尊从原创、尊从本义、遵从专业。</span></p><p style=";margin-top:0;margin-bottom:0;text-align:justify;text-justify:inter-ideograph;direction:ltr;unicode-bidi: embed"><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;">    <span style="font-size: 20px; color: blue; font-family: "times new roman";">Exfoliation</span>一词作为地质学专业词汇已经使用了一百多年了（见<span style="font-size: 20px; color: blue; font-family: "times new roman";">Gilbert, 1904</span>），而且在国际地学界已获公认，因此后来者只能使用这个词汇，不能再造新词；<span style="font-size: 20px; color: blue; font-family: "times new roman";">Exfoliation</span>这个词汇的本义（即常见语义）是“脱落”，即形容墙皮脱落、剥离现象；在地质学专业上是形容一种物理风化构造，表示岩石沿着地表起伏的方向裂开而剥落的现象。其主体（<span style="font-size: 20px; color: blue; font-family: "times new roman";">foliation</span>）本身意指“页理、面理”，而前缀“<span style="font-size: 20px; color: blue; font-family: "times new roman";">ex</span>”表示“排除”、“离开”、“外面”之意。我将该词翻译成“剥离面理”，在尊从原创的前提下，既遵从其本义，也符合地质构造的意义，是表示这一地质现象的最佳词汇。借此机会再次与国内同仁分享我的体会，也同时向大众推介“剥离面理”这个学术词汇。</span></p><p style=";margin-top:0;margin-bottom:0;text-align:justify;text-justify:inter-ideograph;direction:ltr;unicode-bidi: embed"><span style="font-family: 宋体; font-family: 宋体, SimSun;"><span style="font-size: 20px; color: blue;">    剥离面理自从被发现以来，其成因机理一直处于探索和争议中，直到近年来才有所突破。吉尔伯特（</span><span style="font-size: 20px; color: blue; font-family: "times new roman";">Grove Karl Gilbert, 1843–1918</span><span style="font-size: 20px; text-decoration: underline; color: blue;">卸载作用</span><span style="font-size: 20px; color: blue;">（<span style="font-size: 20px; color: blue; font-family: "times new roman";">unloading</span>）。这种假设后来成为地学界的主流观点，以至于几十年后地质学词典和教材中仍然以卸载作用解释剥离面理的成因。</span></span></p><p style=";margin-top:0;margin-bottom:0;text-align:justify;text-justify:inter-ideograph;direction:ltr;unicode-bidi: embed"><span style="color: blue; font-size: 20px; font-family: 宋体, SimSun;">    然而，侵蚀卸载导致岩石膨胀的假说有两个难点无法解释：</span></p><p style=";margin-top:0;margin-bottom:0;text-align:justify;text-justify:inter-ideograph;direction:ltr;unicode-bidi: embed"><span style="color: blue; font-size: 20px; font-family: 宋体, SimSun;">    第一、室内实验无法用反复的加压减压实验形成剥离面理，理论上无法解释；</span></p><p style=";margin-top:0;margin-bottom:0;text-align:justify;text-justify:inter-ideograph;direction:ltr;unicode-bidi: embed"><span style="color: blue; font-size: 20px; font-family: 宋体, SimSun;">    第二、还有其他岩石并非经过明显的地壳深埋也发育了剥离面理。</span></p><p style=";margin-top:0;margin-bottom:0;text-align:justify;text-justify:inter-ideograph;direction:ltr;unicode-bidi: embed"><span style="color: blue; font-size: 20px; font-family: 宋体, SimSun;">    卸载作用说的上述缺陷迫使人们寻求地表温度变化造成的影响：日夜温变和火灾导致岩石表层形成翘皮脱落，而观察和实验证明其影响深度以厘米计，难以形成大规模的剥离面理。那么，什么样的温度变化才能影响地表以下深达百米呢？</span></p><p style=";margin-top:0;margin-bottom:0;text-align:justify;text-justify:inter-ideograph;direction:ltr;unicode-bidi: embed"><span style="font-family: 宋体; color: red; font-weight: bold; font-size: 24px;">四、剥离面理的地表温度变化说：第四纪冰川作用</span></p><p style=";margin-top:0;margin-bottom:0;text-align:justify;text-justify:inter-ideograph;direction:ltr;unicode-bidi: embed"><span style="font-size: 20px;
