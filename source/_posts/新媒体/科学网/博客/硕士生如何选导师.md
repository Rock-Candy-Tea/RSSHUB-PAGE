<p style="text-indent: 0em; line-height: 2em;"><span style="color: #C00000; font-family: 宋体, SimSun; font-size: 18px; text-indent: 0em;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202104/05/201434i4wq8q7ez2j477ze.jpg" title alt="微信图片_20210405201259.jpg" referrerpolicy="no-referrer"><br></span></p><p style="text-indent: 0em; line-height: 2em;"><span style="color: #C00000; font-family: 宋体, SimSun; font-size: 18px; text-indent: 0em;">[博主按：本文由本博主（马臻）原创，暂时没有在任何地方发表或者张贴。照片是上海的一个书店（坐落于上生新所），由本人拍摄。]</span><br></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #000000;">最近，很多大四学生拿到读研的录取通知书。很多人急吼吼地</span><span style="font-family: 宋体, SimSun; font-size: 18px;">“预定”导师。但如果考虑不周，那么今后师生彼此都会苦不堪言。选择导师真的很有讲究吗？ </span></p><p style="line-height: 2em;"><span style="font-family: 宋体, SimSun; color: #7030A0; font-size: 20px;"><strong><span style="color: #7030A0; font-family: 宋体, SimSun; line-height: 150%;">研究生找导师的考虑因素</span></strong></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #000000;">很多人都说选择导师，最重要的考虑因素是研究方向和课题。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #000000;">诚然，跟着导师做科研，需要对导师的研究方向和课题感兴趣，但是大多数的硕士生毕业后都不会读博士。如果就业的话，那么从事的工作往往和本专业关系不大，比如学环境专业的学生毕业后去做电商或者去做企业HR。对于这些学生来说，纠结于课题的“冷门”还是“热门”是没有意义的。只要自己对这个课题不那么反感，并且能顺利读出来，拿到毕业证和学位证就行了。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #000000;">我认为，选择导师的关键考量在于学生和导师是否能“匹配”。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 18px; color: #000000;"><strong>第一，学生将来就业、深造的倾向是否能和课题组“运行模式”匹配；如果不一致，那么能否找到平衡点。</strong><span style="color: #000000; font-family: 宋体, SimSun; line-height: 150%; font-size: 18px;">有的硕士生准备毕业后直接就业，有的准备出国，还有的准备在国内读博士。无论哪种选择，学生都要付出努力、有规划、有铺垫。但问题是，课题组有一定的运行模式，那就是要努力做研究、取得科研发现，并发表科研论文、申请科研项目。准备在毕业后深造的硕士生往往更能够把心思花在科研上，也不需要参加自选实习。于是，他们的目标契合课题组整体目标。而如果硕士生不准备在国内读博士，那势必需要花很多时间外出实习、找工作或者复习英语、申请出国。这有可能会和课题组的运行模式发生冲突。在这种情况下，有的导师会更多地为学生考虑，而有的导师更多为自己考虑。平衡点放在哪里，将极大地影响学生的幸福和前途。</span></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 18px; color: #000000;"><strong>第二，学生的能力是否符合导师的要求。</strong><span style="color: #000000; font-family: 宋体, SimSun; line-height: 150%; font-size: 18px;">在现实生活中，有些导师急切地“抓”学生来“干活”。他们“捡到篮子里都是菜”，一下子招了几个学生，又对学生提出极高的论文发表要求。比如院系要求硕士生在核心期刊发表一篇论文，而有的导师提出硕士生要在高档次SCI刊物发表一篇论文才能毕业。在管理学当中，一个人是否有积极性做一件事，取决于自己的能力和目标是否匹配。如果目标太高，人会感到再怎么努力都无法实现目标，于是就没有积极性。而如果目标太低，人会感到很没劲，也不会有积极性。能使人发挥出最佳水平的情形，是“跳一跳摘桃子”。</span></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 18px; color: #000000;"><strong>第三，师生的性格是否匹配</strong><span style="color: #000000; font-family: 宋体, SimSun; line-height: 150%; font-size: 18px;">。DISC行为模式理论用把人的行为模式分为支配型（Dominance）、影响型（Influence）、稳健型（Steady）、服从型（Compliance）。这不是说一个人只可以对应于一种行为模式，而是说一个人在通常条件下主要遵循一种行为模式，其他行为模式也可以占据一定比例。有很多导师非常强势，你可以发现课题组的学生对导师都言听计从。道理很简单：和支配型导师最匹配的，当然是服从型的学生，而不是支配型的学生。学生如果不能改变自己的行为模式，那么最好选择别的能够使自己舒适的课题组。</span></span></p><p style="line-height: 2em;"><span style="font-family: 宋体, SimSun; color: #7030A0; font-size: 20px;"><strong><span style="color: #7030A0; font-family: 宋体, SimSun; line-height: 150%;">研究生找导师要问清楚</span></strong></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #000000;">每个导师的要求都是不一样的。研究生在进课题组之前就要找导师问清楚。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 18px; color: #000000;"><strong>首先要问清楚导师的培养理念和毕业要求。</strong><span style="color: #000000; font-family: 宋体, SimSun; line-height: 150%; font-size: 18px;">有的导师以培养科学家为己任，有的导师认为只要达到院系的毕业要求就行，还有的导师会提出稍高一点的要求，比如院系要求发表一篇论文，该导师要求发表两篇论文，但万一达不到也可以接受。得到导师的回答后，一定要慎重考虑——如果导师提出的毕业要求远远超过院系要求，自己是否能够达到？不要逞能说自己就是要做科学家。当自己的能力和导师的要求相差太远时，最终矛盾会爆发。</span></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 18px; color: #000000;"><strong>其次要问清楚导师对学生日常时间安排、实习、找工作有什么要求。</strong><span style="font-size: 18px; color: #000000; font-family: 宋体, SimSun; line-height: 150%;">有的课题组要求学生“打卡”，甚至在办公室还有摄像头。有的导师采取相对温和的方法，让学生自己管理时间；当然，学生在需要找导师讨论时，导师也有可能不在办公室。每个课题组都会对学生寒暑假休息的时间进行规定，区别会很大，这得问清楚。实习、找工作也是大事。有的导师规定学生在校期间不能实习、找工作，也有的导师同意学生在取得较好科研进展之后实习、找工作，各个课题组的规定有很大差别，对学生的影响也会非常大。</span></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 18px; color: #000000;"><strong>再次要问清楚：如果进这个课题组，那么具体从事什么课题的研究？研究的模式是什么？</strong><span style="color: #000000; font-family: 宋体, SimSun; line-height: 150%; font-size: 18px;">是一开始跟着课题组师兄、师姐做实验，然后再开展相对独立的研究，还是课题组一群人像“生产流水线”一样每个人负责一小块研究，最终导师将论文“按需分配”？论文由谁来写？这里面有奥妙。</span></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #000000;">最后，对于一些“敏感问题”，也要见机行事，即看导师的性格脾气和谈话时的气氛。比如，有的学生一见导师就问每个月能拿到多少研究生补助，这显然不合适。也有的学生和对象“异地恋”，她小心翼翼地问导师：究竟多久能和对象见一面？遇到这种问题，有的导师会故作开明地说：“这是你们的事，我不管。”也有的导师会觉得有些尴尬。</span></p><p style="line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 20px; color: #7030A0;"><strong><span style="font-size: 20px; font-family: 宋体, SimSun; line-height: 150%;">侧面打听，避免进“火坑”</span></strong></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #000000;">在和导师谈话之余，还不要忘记和课题组师兄师姐以及别的课题组的学生聊聊：这个课题组的学生都按时毕业的吗？如果没有按时毕业，原因是什么呢？他们在毕业前有时间实习、找工作吗？毕业后去向如何？这位导师的指导风格究竟是怎样的？他的性格是怎样的？他有什么缺点？学生和他相处需要注意什么？</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #000000;">往往，有些导师急需招研究生时，把科研前景说得花好稻好，但学生进组后，就会发现进了“火坑”——</span></p><p style="margin-left: 56px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #C00000;">1.<span style="font-variant-numeric: normal; font-variant-east-asian: normal; font-stretch: normal; font-size: 9px; line-height: normal; font-family: "Times New Roman";"> </span>研究生经常帮导师跑腿报账，影响自身学业；</span></p><p style="margin-left: 56px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #C00000;">2.<span style="color: #000000; font-variant-numeric: normal; font-variant-east-asian: normal; font-stretch: normal; font-size: 9px; line-height: normal; font-family: "Times New Roman";"> </span>帮导师做企业“横向”课题，影响毕业所需的基础科研进展；</span></p><p style="margin-left: 56px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #C00000;">3.<span style="color: #000000; font-variant-numeric: normal; font-variant-east-asian: normal; font-stretch: normal; font-size: 9px; line-height: normal; font-family: "Times New Roman";">  </span>实验周期长，不能独立做实验，而是像“生产流水线”上的工人；</span></p><p style="margin-left: 56px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #C00000;">4.<span style="color: #000000; font-variant-numeric: normal; font-variant-east-asian: normal; font-stretch: normal; font-size: 9px; line-height: normal; font-family: "Times New Roman";">  </span>导师不让学生毕业，也不及时修改学生毕业所需的科研论文；</span></p><p style="margin-left: 56px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #C00000;">5.<span style="color: #000000; font-variant-numeric: normal; font-variant-east-asian: normal; font-stretch: normal; font-size: 9px; line-height: normal; font-family: "Times New Roman";">  </span>导师把学生采集的数据交给急需论文毕业的人去写，或者在学生写的论文上面加上别人的名字作为“共同第一作者”；</span></p><p style="margin-left: 56px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #C00000;">6.<span style="color: #000000; font-variant-numeric: normal; font-variant-east-asian: normal; font-stretch: normal; font-size: 9px; line-height: normal; font-family: "Times New Roman";">  </span>当学生在组里过不惯，提出转导师时，导师不同意，理由是这样会损失他一个招生名额；</span></p><p style="margin-left: 56px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #C00000;">7.<span style="color: #000000; font-variant-numeric: normal; font-variant-east-asian: normal; font-stretch: normal; font-size: 9px; line-height: normal; font-family: "Times New Roman";">  </span>导师平时不大指导研究生，或者缺乏指导能力，以至于学生不能按时毕业。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #000000;">出现上述情况，有时候情有可原。比如，有些医科、生物实验室开展实验的确周期长，需要由不同的人合作完成。再比如，有的学生采集了数据，不及时写论文，所以导师把这位学生的数据交给别的学生来写论文。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体, SimSun; font-size: 18px; color: #000000;">但问题是，很多学生没有打听清楚就急切地跳进了“火坑”。</span></p>                    <br><br>
