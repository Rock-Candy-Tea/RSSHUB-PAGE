font-family: 宋体;">的说法。这一奇观被列为抚仙湖一景。</span></span></p><p style="text-indent:37px;line-height:150%"><span style="font-size: 20px;"><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">从生态学的角度，这是鱼类的一种生态适应。鱇浪鱼喜欢水质清新含氧较高的水域环境，多栖息于深水鱼洞或砂砾中，食性较广，而大头鲤喜欢水质清澈的水体上层，主食浮游动物，食性单一。大头鲤因外来鱼种竞争、水质污染和自身繁殖能力不强，野生种群数量急剧减少，早在</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">1989</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">年就被列为国家二级重点保护物种。鱇浪鱼生活的抚仙湖水质清澈，加上期自身生性活泼繁殖能力强，</span> <span style="font-size: 19px; line-height: 150%; font-family: 宋体;">并未被列入重点保护物种名录。</span></span></p><p style="text-indent:37px;line-height:150%"><span style="line-height: 150%; font-family: 宋体; font-size: 20px;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202206/29/140955tlzddvdwdlawfh3r.jpg" title alt="抗浪鱼.jpg" referrerpolicy="no-referrer"><br></span></p><p style="text-indent: 37px; line-height: 150%; text-align: center;"><span style="font-family: 楷体, 楷体_GB2312, SimKai; line-height: 150%; color: #FF0000; font-size: 20px;"><span style="font-family: 楷体, 楷体_GB2312, SimKai; line-height: 150%; color: #FF0000; font-size: 19px; text-indent: 37px;">鱇</span>浪鱼(来自网络 )</span></p><p style="text-indent:37px;line-height:150%"><span style="font-size: 20px;"><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">但是，鱇浪鱼同样没有逃脱野生种群急剧减少的厄运。鱇浪鱼柔嫩味美，蛋白质含量达</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">20%</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">，在鱼类视频中名列前茅，清代康熙年间以后被当做贡品进贡朝廷，美名远扬。不少食客以品尝鱇浪鱼为荣，价格疯涨。我们亲身的见闻，令人震惊。在抚仙湖北岸的澄江县城街头，</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">20</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">世纪</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">80</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">年代末期，</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">1</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">千克鱇浪鱼不到</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">10</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">元，到了</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">2020</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">年前后价格涨到</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">2000</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">元，只敢问价不敢购买。价格飙升促使渔民改进渔具，一网打尽。另一方面，外来鱼种银鱼是鱇浪鱼的天敌。</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">1982</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">年星云湖引进银鱼，湖水上涨水时，大量银鱼顺着玉带河流入抚仙湖。银鱼杂食，繁殖能力极强，每次产卵近千颗，鱼卵沉到水体底部约</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">30</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">天就会完成孵化。银鱼滋生，采食鱇浪鱼的卵，明显挤占鱇浪鱼的生存空间。进入</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">90</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">年代抚仙湖的银鱼已经很常见。与此同时，抚仙湖鱇浪鱼捕捞量却急速下降，</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">1983</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">年约</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">423</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">千千克</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">1998</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">年已不足</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">200</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">千克。</span></span></p><p style="text-indent:37px;line-height:150%"><span style="font-size: 20px;"><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">1998</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">年，玉溪市与中科院昆明动物研究所合作研究鱇浪鱼人工增殖技术，将人工辅助繁殖的育苗放入抚仙湖。到了</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">2019</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">年，抚仙湖鱇浪鱼产量达到</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">12</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">千千克。</span></span></p><p style="text-indent:37px;line-height:150%"><span style="font-size: 20px;"><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">抚仙湖水深、水清、蓄水量大，加上盛名远扬的鱇浪鱼，成为节日休闲、旅游度假乃至畔湖而居的胜地。水上娱乐异军突起，人力船、机动船来来往往，环湖旅游开发日新月异，近水高档宾馆、住宅拔地而起。抚仙湖一度人群涌至，水质下降。</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">2007</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">年</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">5</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">月</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;">23</span><span style="font-size: 19px; line-height: 150%; font-family: 宋体;">日云南省第十届人民代表大会常务委员会颁布《云南省抚仙湖保护条例》，取缔机动船水上娱乐，规范旅游服务，拆除近水建筑，修建环湖观光栈道，兴建湿地公园。这一切，赋予了抚仙湖新的生机。</span><span style="font-size: 19px; line-height: 150%; font-family: "Times New Roman", serif;"> </span></span></p><p style="text-indent:37px;line-height:150%"><span style="line-height: 150%; font-family: "Times New Roman", serif; font-size: 20px;"></span></p><p style="text-align:center"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202206/29/135719o09m3lgjkuuc1aa2.jpg" title alt="星云湖，很污浊.JPG" referrerpolicy="no-referrer"></p><p style="text-indent: 37px;
