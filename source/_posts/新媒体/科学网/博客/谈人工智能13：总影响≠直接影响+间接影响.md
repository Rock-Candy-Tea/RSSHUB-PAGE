overflow-wrap: break-word; font-size: 16px;">李：为了要克服一个根深蒂固的普遍错误而问的问题，都像是在下套。你是说：总影响 = 直接影响 + 间接影响。怎么确定直接影响和间接影响呢？</span><span style="font-family: 宋体, SimSun; font-size: 16px; overflow-wrap: break-word;"><br></span></p><p style="white-space: normal; overflow-wrap: break-word; font-size: 16px;">学：去掉次因<span style="overflow-wrap: break-word; font-family: "times new roman";">C</span>后，元因<span style="overflow-wrap: break-word; font-family: "times new roman";">Y</span>对<span style="overflow-wrap: break-word; font-family: "times new roman";">G</span>的影响就是直接影响。间接影响就是排除直接影响，<span style="overflow-wrap: break-word; font-family: "times new roman";">Y</span>只通过<span style="overflow-wrap: break-word; font-family: "times new roman";">C</span>而对<span style="overflow-wrap: break-word; font-family: "times new roman";">C</span>，随后去除<span style="overflow-wrap: break-word; font-family: "times new roman";">Y</span>，再看对<span style="overflow-wrap: break-word; font-family: "times new roman";">G</span>的影响，这就是间接影响，因为它排除了产生直接影响的可能。</span><span style="font-family: 宋体, SimSun; font-size: 16px; overflow-wrap: break-word;"><br></span></p><p style="white-space: normal; overflow-wrap: break-word; margin-top: 1.12em; margin-bottom: 1.12em; padding: 0px; font-family: Tahoma, Helvetica, SimSun, sans-serif, Hei; font-size: 14px; background-color: rgb(255, 255, 255);"><span style="font-family: 宋体, SimSun; font-size: 16px; overflow-wrap: break-word;"></span></p><p><span style="font-family: 宋体, SimSun; font-size: 16px;">李：你们对直接和间接影响的理解都是还原论的，即去除他者后自身的作用。按你们这种认识，当然很难真正领悟开放性、开放系统以及与环境交互的深刻意义。</span></p><p><span style="font-family: 宋体, SimSun;">  <span style="font-family: 宋体, SimSun; font-size: 16px;">假设只有<span style="font-family: "times new roman";">Y</span>和<span style="font-family: "times new roman";">C</span>联合作用才会影响<span style="font-family: "times new roman";">G</span>（比如<span style="font-family: "times new roman";">Y</span>是药物，它会在人体内激发而产生或增加<span style="font-family: "times new roman";">C</span>，而<span style="font-family: "times new roman";">C</span>是<span style="font-family: "times new roman";">Y</span>起作用必需的酶，没有<span style="font-family: "times new roman";">C</span>，<span style="font-family: "times new roman";">Y</span>不起作用）。这样，你们所说的直接影响为零，因为单单<span style="font-family: "times new roman";">Y</span>不影响<span style="font-family: "times new roman";">G</span>；你们所说的间接影响也为零（因为单单<span style="font-family: "times new roman";">C</span>也不影响<span style="font-family: "times new roman";">G</span>）。总影响 = 直接影响 + 间接影响 =<span style="font-family: "times new roman";"> 0 + 0</span>。即：你们所说的直接影响和间接影响都为零，故总影响为零。但这显然是错误的，实际上明显有影响，<span style="font-family: "times new roman";">Y</span>和<span style="font-family: "times new roman";">C</span>的联合作用会影响G，直接影响和间接影响可能都不为零。我们看看这是为什么。</span></span></p><p><span style="font-family: 宋体, SimSun;"> <span style="font-family: 宋体, SimSun; font-size: 16px;">首先，线性<span style="font-family: "times new roman";"> =</span> 既可加又齐次。其数学表述是：若<span style="font-family: "times new roman";">f(ax+by) = af(x)+bf(y)</span>对每一对常数<span style="font-family: "times new roman";">(a,b)</span>和每一对自变量<span style="font-family: "times new roman";">(x,y)</span>恒成立，则函数<span style="font-family: "times new roman";">f(·)</span>是线性的，其中可加性保证等式对相加成立：<span style="font-family: "times new roman";">f(x+y) = f(x)+f(y)</span>，齐次性保证等式对标量积成立：<span style="font-family: "times new roman";">f(ax) = af(x)</span>。在线性系统中，不同部分之间彼此互不影响，所以可以先各个击破，再做和得到总影响。也就是我们</span></span><span style="font-family: 宋体, SimSun; font-size: 16px;">早先说过的现代科学之血——分而击之。</span><span style="font-family: 宋体, SimSun;"><span style="font-family: 宋体, SimSun; font-size: 16px;">线性系统完全可分解，因此分而击之的还原论完全适用。所以有些人把“还原论科学”称为“线性科学”。对于你们所说的影响，“总影响 = 直接影响 + 间接影响”只对线性问题才恒成立，对非线性问题未必成立，往往不成立。</span></span></p><p><span style="font-family: 宋体, SimSun;">  <span style="font-family: 宋体, SimSun; font-size: 16px;">你们的理解是还原论的，而这个系统是非线性的（<span style="font-family: "times new roman";">Y</span>会影响<span style="font-family: "times new roman";">C</span>，只有<span style="font-family: "times new roman";">Y</span>和<span style="font-family: "times new roman";">C</span>联合作用才会影响<span style="font-family: "times new roman";">G</span>，因此是非线性的），所以你们这种还原论方法失效。其实，在考虑间接影响（<span style="font-family: "times new roman";">C→G</span>）时，不能去除其环境因素<span style="font-family: "times new roman";">Y</span>，在考虑直接影响（<span style="font-family: "times new roman";">Y→G</span>）时，也不能去除其环境因素<span style="font-family: "times new roman";">C</span>，即必须与环境交互，因为“只有<span style="font-family: "times new roman";">Y</span>和<span style="font-family: "times new roman";">C</span>联合作用才会影响<span style="font-family: "times new roman";">G</span>”。换言之，或者说（<span style="font-family: "times new roman";">C→G</span>）对<span style="font-family: "times new roman";">Y</span>是开放的，（<span style="font-family: "times new roman";">Y→G</span>）对<span style="font-family: "times new roman";">C</span>也是开放的，或者说整个系统有非线性的内联性。对这样的非线性内联或开放问题，还原论的魔杖——“分解—汇总”的分而击之法——不仅不能奏效，还会误导。</span></span></p><p><span style="font-family: 宋体, SimSun; font-size: 16px;">学：在考虑间接影响（<span style="font-family: "times new roman";">C→G</span>）的时候，存在环境因素<span style="font-family: "times new roman";">Y</span>，这我能够理解。但是，在考虑直接影响（<span style="font-family: "times new roman";">Y→G</span>）的时候，不是没有环境因素<span style="font-family: "times new roman";">C</span>吗？否则又怎么把它跟总影响区分开来？</span><span style="font-family: 宋体, SimSun;"><br></span></p><p><span style="font-family: 宋体, SimSun; font-size: 16px;">教：我觉得，你这个问题可以这样解答：在没有<span style="font-family: "times new roman";">Y</span>的时候，人体内的<span style="font-family: "times new roman";">C = C<sub>1</sub></span>，有<span style="font-family: "times new roman";">Y</span>后，<span style="font-family: "times new roman";">C = <span style="font-family: "times new roman";">C</span><sub style="white-space: normal; font-family: "times new roman";">1</sub><span style="font-family: "times new roman"; font-size: 13.3333px;"></span> + C<sub>2</sub></span>；直接影响（<span style="font-family: "times new roman";">Y→G</span>）的环境因素是<span style="font-family: "times new roman";"><span style="font-family: "times new roman";">C</span><sub style="white-space: normal; font-family: "times new roman";">1</sub><span style="font-family: "times new roman"; font-size: 13.3333px;"></span></span>；间接影响是（<span style="font-family: "times new roman";"><span style="font-family: "times new roman";">C</span><sub style="white-space: normal; font-family: "times new roman";">2</sub>→G</span>），其环境因素是<span style="font-family: "times new roman";">Y</span>。总影响 = 直接影响 + 间接影响。</span></p><p><span style="font-family: 宋体, SimSun; font-size: 16px;"></span></p><p><span style="font-family: 宋体, SimSun; font-size: 16px;">李：你这个解答确有进步，想法有可取之处，但还是有误，它还是还原论的，因为它还是依赖于可加性，比如你认为<span style="font-size: 16px; font-family: "times new roman";">C = <span style="font-family: "times new roman";">C</span><sub style="white-space: normal; font-family: "times new roman";">1</sub> + <span style="font-family: "times new roman";">C</span><sub style="white-space: normal; font-family: "times new roman";">2</sub></span>的总影响可以分解，等于<span style="font-size: 16px; font-family: "times new roman";"><span style="font-family: "times new roman";">C</span><sub style="white-space: normal; font-family: "times new roman";">1</sub></span>的影响加上<span style="font-size: 16px; font-family: "times new roman";"><span style="font-family: "times new roman";">C</span><sub style="white-space: normal; font-family: "times new roman";">2</sub></span>的影响。可见，还原论的幽灵如鬼魅附体，难以驱离。</span></p><p><span style="font-family: 宋体, SimSun;">  <span style="font-family: 宋体, SimSun; font-size: 16px;">先让我用你们熟悉的控制系统理论来解释：任一线性系统，都有两部分因，会导致两部分果——输出响应：①无输入时系统的初始状态导致的“零输入响应”，②系统在初始松弛的状态下由输入导致的“零状态响应”；总输出响应 = 零输入响应 + 零状态响应。然而，对于非线性系统，该式一般并不成立，往往也就不这么分解，因为这个分解对处理非线性系统没多大好处。同理，对于非线性问题，一般不该认定总影响 = 直接影响 + 间接影响，因为这并不会带来多大好处。万一你硬要坚持这个等式，那么可以认为元因的“直接影响”有赖于次因的作用，通过次因而产生的“间接影响”也有赖于元因的作用，即“你中有我，我中有你”。因为其实不可分而你硬要分，所以只能在解释上做文章，两者相互纠缠而变得很“辩证”。人们大都不喜欢这么“辩证”，所以还不如直接承认等式不成立。</span></span></p><p><span style="font-family: 宋体, SimSun;">  <span style="font-family: 宋体, SimSun; font-size: 16px;">再从函数的角度来说得更透彻一些。任一线性函数<span style="font-family: "times new roman";">L</span>都必定可分，即恒有<span style="font-family: "times new roman";">L(x,y) = L<sub>1</sub>(x)+L<sub>2</sub>(y)</span>，其中<span style="font-family: "times new roman";"><span style="font-family: "times new roman";">L</span><sub style="white-space: normal; font-family: "times new roman";">1</sub></span>和<span style="font-family: "times new roman";">L<sub>2</sub></span>是两个线性函数。而非线性函数<span style="font-family: "times new roman";">f</span>未必可分，即未必有<span style="font-family: "times new roman";">f(x,y) = f<sub>1</sub>(x,y)|<sub>y=y*</sub>+f<sub>2</sub>(x,y)|<sub>x=x*</sub></span>（当然也就未必有<span style="font-family: "times new roman";">f(x, y) = <span style="font-family: "times new roman";">f</span><sub style="white-space: normal; font-family: "times new roman";">1</sub>(x)+<span style="font-family: "times new roman";">f</span><sub style="white-space: normal; font-family: "times new roman";">2</sub>(y)</span>），即未必存在函数<span style="font-family: "times new roman";"><span style="font-family: "times new roman";">f</span><sub style="white-space: normal; font-family: "times new roman";">1</sub>、<span style="font-family: "times new roman";">f</span><sub style="white-space: normal; font-family: "times new roman";">2</sub></span>以及固定数值<span style="font-family: "times new roman";">x*、y*</span>使等式对任意<span style="font-family: "times new roman";">(x,y)</span>恒成立。比如，就连简单函数如<span style="font-family: "times new roman";">f(x,y) = xy</span>都不可分。还有其他类的不可分性，比如<span style="font-family: "times new roman";">x</span>和<span style="font-family: "times new roman";">y</span>相互有联系，例如<span style="font-family: "times new roman";">x = g(z,u), y = h(z,v)</span>, 或者<span style="font-family: "times new roman";">y = h(x,v)</span>，其中<span style="font-family: "times new roman";">z、u</span>和<span style="font-family: "times new roman";">v</span>都是自变量。可分性一般有多种含义，这儿指可以分解为和式，这与还原论的“总体乃部分之和”一致。可见，非线性问题未必可分，总影响未必可分为直接影响加间接影响。它们其实往往不可分，可分的只是线性情形和一类特例。尽管这是事实，却违反绝大多数人潜移默化形成的信念，所以很难被接受。</span></span></p><p><span style="font-family: 宋体, SimSun;">  我把实验称为现代科学之“体<span style="font-family: 宋体, SimSun;">”</span>，它是现代科学与古代科学的主要分水岭：主动实验对被动观测。我把分而击之称为现代科学之“血<span style="font-family: 宋体, SimSun;">”</span>：科学科学，分科之学，本义就是“分学”，它十分仰仗分而击之——这是还原论克敌制胜的看家本领和主流方法。基于此，还原论的实验研究自然要求游离变量，要求各影响变量必须是可分离的。然而，由上述可知，非线性问题的影响变量往往不可分离。对不可分问题强行分离，结果就会像用<span style="font-family: "times new roman";"><span style="font-family: "times new roman";">f</span><sub style="white-space: normal; font-family: "times new roman";">1</sub><span style="font-family: "times new roman";">(x,y)|</span><sub style="white-space: normal; font-family: "times new roman";">y=y*</sub><span style="font-family: "times new roman";">+f</span><sub style="white-space: normal; font-family: "times new roman";">2</sub><span style="font-family: "times new roman";">(x,y)|</span><sub style="white-space: normal; font-family: "times new roman";">x=x*</sub></span>来代替上述<span style="font-family: "times new roman";">f(x,y)</span>一样，得到似是而非的结论。</span><span style="font-family: 宋体, SimSun;"><span style="font-size: 16px;
