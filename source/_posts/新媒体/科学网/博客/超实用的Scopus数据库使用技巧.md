<div>   
<p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">Scopus是爱思唯尔旗下的文献摘要和引文数据库，涵盖健康、生命、物理和社会科学领域。Scopus数据库收录了5000多家出版商的约26000本期刊。Scopus数据库有多种工具可用来搜索文献、跟踪引文并分析搜索结果。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">本文主要回答以下几个问题：<br></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">    <span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">1. Scopus可以查看哪些指标</span></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">    2. Scopus有哪些功能</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">    3. 快速查找文献5大方法</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">    4. 如何导出检索结果</span></p><p style="text-align: center;"><span style="font-size: 18px; color: #0070C0;"><strong><span style="font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">#01 </span></strong></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">Scopus上可以查看期刊的<span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">CiteScore ™, SJR, SNIP</span>指标，此外还能查看期刊的被引次数，期刊文章数量等。2020 CiteScore指标近日已经公布，大家可以免费查询：</span><span style="font-family: 微软雅黑, "Microsoft YaHei";"><a href="https://www.scopus.com/sources" target="_blank">https://www.scopus.com/sources</a> </span></p><p style="text-align: center;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202107/21/100934cceueetlo2vcanqk.png" title alt="scopus 1.png" referrerpolicy="no-referrer"><br></p><p style="text-align: center;"><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 14px;">图片来源：Scopus官网</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">Scopus可以查看文献在Scopus中的<span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">引用次数和PlumX指数</span>。有关PlumX的介绍，请查看往期文章：</span><a href="http://blog.sciencenet.cn/home.php?mod=space&uid=3201402&do=blog&id=1296092" target="_blank" style="font-family: 微软雅黑, "Microsoft YaHei"; text-decoration: underline; font-size: 18px;"><span style="font-family: 微软雅黑, "Microsoft YaHei"; text-decoration: underline; font-size: 18px;">科研人也要紧跟潮流，替代计量指标用起来！</span></a></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">Scopus为每个作者配置了一个<span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">作者标识符，可以查看作者的H指数，已发表文献和被引情况</span>。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">Scopus还为科研机构等分配了<span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">专属的机构ID</span>，可以查看机构的总发文量，各学科发文量，出版物，作者信息。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">网站有中文版，点击网页底部 “<span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">切换到简体中文</span>”即可。订阅用户可以查看以上所有信息，大家可以查看所属机构是否订阅了该数据库。非订阅用户可以使用<span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">Scopus Preview</span>，查找期刊和作者，无法查文献。</span></p><p style="text-align: center;"><span style="font-size: 18px; color: #0070C0;"><strong><span style="font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">#02</span></strong></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">搜索Scopus数据库前，首先要了解与研究主题相关的术语，可以根据自己的研究兴趣生成关键词列表。打开Scopus数据库，会看到导航栏包括：</span></p><blockquote><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">• 检索（Search）——可以查找文献、作者和归属机构，或进行高级搜索</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">• 来源出版物（Sources）——查看和搜索期刊</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">• 列表（Lists）——将检索结果添加到临时列表</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">• SciVal——查看机构和作者的可视化研究表现</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">• 通知（Alerts）——查看之前保存的搜索结果或引用提醒</span></p></blockquote><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">Scopus数据库的多项功能可以帮助实现这一目的。</span></p><blockquote><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">• 排序功能可按照不同标准整理搜索结果。如果重点关注最新发表的论文，可以照此排序，也可以按照引用频率进行排序。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">• 浏览摘要功能可以查看文章的摘要，“显示所有摘要”功能即可查看搜索结果中所有文献的摘要。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">• 如果所在机构订阅了Scopus数据库，只要文献全文对外开放，点击文献下方的按钮便可获取全文。</span></p></blockquote><p style="text-align: center;"><span style="font-size: 18px; color: #0070C0;"><strong><span style="font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">#03</span></strong></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">以下是一些<strong><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">搜索小技巧</span></strong>。<br></span></p><blockquote><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">■  使用引号可以精确搜索短语——例如搜索“blood pressure”</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">■  使用星号 * 搜索特定词语开头的术语——例如输入heart*。还可以用星号来搜索模糊术语。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">■  和其他搜索引擎一样，AND可用来搜索包含所有关键词的结果，比如说搜索 Vitamin D AND Covid-19，可以搜索到包含这两个术语的文献</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">■  两个关键词之间加 OR，搜索结果至少包含其中一个词语，例如搜索Vitamin D OR Covid-19，可得到包含其中一个术语的文献</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">■ NOT 可以排除某个关键词在内的结果，例如搜索 Vitamin D NOT Covid-19，可得到包含Vitamin D但不包括Covid-19的文献。</span></p></blockquote><p style="text-align: center;"><span style="font-size: 18px; color: #0070C0;"><strong><span style="font-size: 18px; font-family: 微软雅黑, "Microsoft YaHei";">#04</span></strong></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">Scopus数据库提供许多筛选条件，如是否开放获取、文献类型、出版日期、语言和期刊。还可以同时设置多个筛选条件，缩小搜索范围。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">Scopus 数据库中的许多功能可用来管理搜索结果，引用、保存和分享需要的结果。<strong><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">选择需要的文献可以保存到导航栏的列表，也可以导出文件，或者导出到Mendeley，EndNote等，还可以选择导出信息。</span></strong></span></p><p style="text-align: center;"><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202107/21/100946bhenntnhp2hpqrn8.png" title alt="scopus 2.png" referrerpolicy="no-referrer"><br></span></p><p style="text-align: center;"><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 14px;">图片来源：Scopus官网</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">也可以<span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">分析搜索结果，查看按文献类型、学科、机构、年份等划分的文献结果</span>。还可以<strong><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">保存搜索历史并创建提醒，以便获得更多与搜索结果匹配的新文献</span></strong>。<strong><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px; color: #0070C0;">参考文献也可以导出，搜索综述文章的参考文献，可以以此为基础进行下一步搜索。 </span></strong></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;">开始搜索前，请花些时间探索Scopus数据库的所有功能，就像使用其他搜索引擎一样。研究人员都很忙，经常在完全了解清楚所有可用功能前就开始搜索。从长远角度看，花些时间了解搜索引擎可以节约时间、提高搜索效率。</span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei"; font-size: 16px;"><br></span></p><p><span style="font-family: 微软雅黑, "Microsoft YaHei";
