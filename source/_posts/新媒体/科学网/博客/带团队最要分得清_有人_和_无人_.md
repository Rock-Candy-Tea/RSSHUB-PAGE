<p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">《西游记》第二十七回的故事梗概大致是，白骨精戏唐三藏，孙悟空打白骨精，唐三藏整孙悟空。这种三角闭环驱动本来可以像永动机一样无休止地循环下去，但由于悟空第三打时出手太狠，将白骨精打</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">倒</span>“断绝了灵光”</span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">，致使这部</span>“永动机”</span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">也不灵光，</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">永远不动了。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">白骨精被打死，悟空遭恨逐，唐僧还要继续西行。唐僧还能行吗？他能行多远？</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">第二十八回回目</span>“花果山群妖聚义，黑松林三藏逢魔”已点明，悟空回老家</span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">享田园生活、天伦之乐去了</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">，唐僧则径入暗无天日的黑松林，才脱旧妖，又逢新魔。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">紧接白骨精出场的黄袍怪的运气比白骨精好太多了。他根本用不着像白骨精那样费尽心机变这个变那个跑到荒山野岭冒着生命危险去抓唐僧。他躺在自家石床上睡大觉，唐僧就自投罗网，乖乖地送上门来了。他把这叫做</span>“蛇头上苍蝇，自来的衣食”。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">西行之路，没有悟空，唐僧寸步难行。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">为什么如此？</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">答案其实已在悟空的离别</span>“赠言”里。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">种种迹象表明，悟空</span>“其实不想走，其实我想留，留下来陪你每个春夏秋冬”。然正如悟空所说，唐僧“屡次逐我”，“常言道：‘事不过三。’我若不去，真是个下流无耻之徒。我去！我去！”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">最是充满玄机的，是悟空补了一句：</span>“去便去了，只是你手下无人。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">悟空如果不多此一句，才说明他真的去意已决</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">。以老文之见，</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">既然去便去了，你手下有人或无人跟老子没有半毛钱关系了，何必多此一句！</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">这多此一句，折射出悟空极度复杂痛苦的心境：既不得不离唐僧而去，又担心自己这一去无人能保唐僧去得灵山。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">也正是这多此一句，彻底激怒了唐僧：</span>“这泼猴越发无礼！看起来，只你是人，那悟能、悟净，就不是人？”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px">“那大圣一闻得说，他两个是人，止不住伤情凄惨。”悟空从多此一句又转向多此一举：吩咐沙僧“途中更要仔细，倘一时有妖精拿住师父，你就说老孙是他大徒弟。西方毛怪，闻我的手段，不敢伤我师父。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">唐僧比当年菩提祖师还绝情，要与悟空</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">彻底</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">划清界限：</span>“我是个好和尚，不题你这歹人的名字。你回去罢。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px">“那大圣见长老三番两复，不肯转意回心，没奈何才去。”半空里，“又想起唐僧，止不住腮边泪坠，停云住步，良久方去。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">赶走悟空后，唐僧之所以寸步难行，根本上，与其说是因为没有悟空，不如说是因为如悟空所说</span>“只是你手下无人”。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">常言道，缺少谁地球照样转。但是，但是，但是，地球上有人岂是一个团队有人可同日而语的！</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">手下无人也罢。只要妖精们意识到唐僧手下有人，就不敢轻易对唐僧下手。但如何让妖精们知晓唐僧手下有人呢？悟空</span>“含悲留意嘱沙僧”，把这一任务托付给了沙僧。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">顺便插一句：为师者，得一悟空这样的弟子，足矣！</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">何为手下有人？何为手下无人？有人是因为有谁？无人是因为无谁？有人时能做什么？无人时不能做什么？这些问题的答案仁者见仁、智者见智，但悟空跟唐僧的鸡同鸭讲告诉我们，不同的认识往往导致天壤之别的结局。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">《三国演义》中也有大量这样的案例。例如，刘备三顾茅庐得诸葛亮后，曹操跟手下分析时局，问：</span>“诸葛亮！此何许人也？”荀彧答：“诸葛亮复姓诸葛，单字名亮，年纪二十六七岁。”夏侯惇哈哈大笑：“如此小儿为军师，刘备手下无人矣！”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">曹操转问徐庶：</span>“孔明比先生如何？”徐庶答：“岂敢与诸葛亮相比。庶如萤火之光，亮乃皓月之明也。诸葛亮实有经天纬地之才，扭转乾坤之能，神鬼莫测之计，包藏天地之志，真乃当今奇士也。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">夏侯惇用一声</span>“哼！”表示鄙视。徐庶不予理会，接着说：“刘备素怀鲲鹏之志，今得诸葛亮如鱼得水、如虎添翼，恐天下无人可敌呀！”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">刘备得诸葛亮，夏侯惇认为</span>“刘备手下无人矣”，而徐庶“恐天下无人可敌呀”。诸葛亮初出茅庐的第一战恰恰是在博望坡战夏侯惇，夏侯惇惨败。有人和无人，认识到有人和无人，对胜败起了决定性作用。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">回到唐僧。取经看似一项神圣而伟大的使命，完成它其实并不需要有诸葛亮那样的奇士，只要有能解决两个基本需求问题的人即可：一是生理需求，主要是吃住；二是安全需求，主要是除妖。按马斯洛需求层次理论，它们属于两个最低层次需求，芸芸众生都要天天面对。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">纵然是最低层次需求，众所周知，唐僧本人无法解决。之前悟空在时，这两个需求都由他妥妥解决。没了悟空，八戒和沙僧如何？</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">先看吃饭问题。径入黑松林不久，唐僧兜住马，又开始喊饿，要八戒</span>“寻些斋饭我吃”。八戒请唐僧下马，“在此等老猪去寻”。唐僧见荒无人烟，问他去哪里寻，八戒表决心：“莫管，我这一去，钻冰取火寻斋至，压雪求油化饭来。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">话说得令人振奋，但斋化来了吗？八戒转了十余里，没见着一户人家，感叹大师兄不好当：</span>“当年行者在日，老和尚要的就有；今日轮到我的身上，诚所谓‘当家才知柴米价，养子方晓父娘恩。’”然后找个草科里睡睡去了。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">化斋不行，除妖咋样？唐僧如八戒所说</span>“老和尚坐不住”，在黑松林里转悠，误入碗子山波月洞后，八戒和沙僧前去搭救。黄袍怪料定他们“一定寻着我门上”，“上门的买卖好做”。结果表明，“若论赌手段，莫说两个和尚，就是二十个，也敌不过那妖精。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">黄袍怪的夫人、宝象国国王的第三个公主百花羞跟黄袍怪说情放了唐僧。</span>“那八戒与沙僧闻得此言，就如鬼门关上放回来的一般。即忙牵马挑担，鼠窜而行。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">如此鼠窜之辈，在国王面前却夸海口。八戒吹：</span>“自从东土来此，第一会降妖的是我。”他在一国君臣、两班文武面前表演长高变化，吹嘘“若是南风起，把青天也拱个大窟窿”，也就是说他的长高无止极。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">唐僧一向念念不忘</span>“出家人不打诳语”，居然由着八戒既吹嘘自己又否定悟空。不过，“打诳语”这种东西在唐僧心中，从来就像观音从如来手中接过的金紧禁三箍，给谁和不给谁戴上完全看心情。之前国王怼唐僧既不会降妖怎么敢上西天拜佛时，唐僧也是选择性抹去悟空过去的存在和功绩，一样把八戒和沙僧吹得天花乱坠：“贫僧有两个徒弟，善能逢山开路，遇水迭桥，保贫僧到此。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">八戒和沙僧雄赳赳气昂昂又去捉黄袍怪了，扬言</span>“也在此国扬扬姓名”。结局是，战经八九个回合，八戒渐感气力不加，叫沙僧“你且上前来与他斗着，让老猪出恭来”。八戒趁机开溜，沙僧则“被怪一把抓住，捉进洞去”。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">没有悟空的日子里，是八戒和沙僧证明自己，并向唐僧表明他手下有人的最好机会。然而事实无可辩驳地证明，没有悟空，八戒和沙僧不仅保不了唐僧，连自身都难保。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">黄袍怪后来作法把唐僧变作一只斑斓猛虎，用铁绳锁在铁笼里，收于朝房之内。最终还是悟空被八戒恳请加义激前来救了唐僧。讽刺的是，被锁在铁笼里的唐僧，</span>“别人看他是虎，独行者看他是人”。如果说唐僧手下有人，这人首先必须是看他是人的人吧！你手下的人都不把你当人看，你敢说你手下有人？</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">悟空在把唐僧化虎为人之前笑问唐僧：</span>“师父啊，你是个好和尚，怎么弄出这般个恶模样来也？你怪我行凶作恶，赶我回去，你要一心向善，怎么一旦弄出个这等嘴脸？”八戒劝悟空“救他救儿罢。不要只管揭挑他了”。悟空反问八戒：“你凡事撺唆，是他个得意的好徒弟，你不救他，又寻老孙怎的？”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span
