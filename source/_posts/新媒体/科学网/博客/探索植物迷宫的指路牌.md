referrerpolicy="no-referrer"></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">植物作为人类最不或缺的资源，和人们的衣食住都密切相关。除此之外，以植物为主的绿水青山，以及鲜艳漂亮的花朵还极大地满足了人们的精神需求。然而，人们对植物的喜爱程度和关注程度，远不如动物。喜欢去动物园的孩子，远远多于喜欢去植物园的孩子。这其中除了动物呆萌的模样，以及动物能够和人类进行一定的感情交流以外，对植物欣赏需要一定的门槛，植物种类较多，难以区别，不易认识，也是一个重要的原因。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">作为一个植物学家，我最大的快乐莫过于见识世界上各种各样的植物。世界上的植物有30多万种，大多数植物的分布是局域性，最优秀的植物学家也不可能认识地球上的所有植物。因此，区域性的图鉴就成了认识植物最重要的工具书。我每初到一个的国家，都会迫不及待买上一本当地的植物图鉴，凭着这些图鉴，我认识了不少异域的植物。我去过的多数国家，大都有这类方便非植物学家认识植物的图鉴。在上个世纪，我们国家还缺乏这类专门为植物学爱好者以及大众认识植物而编写的图鉴。这可能也是我国植物学爱好者数量偏少的原因之一吧。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">编辑这样的一本图鉴一直我心中的一个梦想，我也为实现这样一个梦想做过一些努力，1998年我和管开云、孙航、费勇、孙卫邦等人就编著了《云南高山花卉》一书。我到西双版纳热带植物园工作后，也动员和鼓励标本馆的同事编撰一本《西双版纳常见植物图鉴》。遗憾的是十年过去了，这本书还没有出版。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">进入本世纪以后，随着我国植物学研究水平的提高，以及人民生活水平的提高，喜爱植物的人群越来越多。针对市场的需求，这类的图鉴逐步多了起来，其中也不乏精品。然而，也不少国内的图鉴，或者收集的种类较少，或者图片质量不高，或者使用不方便，有些图鉴甚至连物种的鉴定都不准确，留下了很大的遗憾。横断山作为生物多样性的热点地区，植物多样性丰富，特别是横断山的高山，植物千姿百态，艳丽夺目，是植物王国皇冠上最璀璨的一颗明珠。而这一个地区一直未见一本，种类收录收集全面，图片漂亮，能够和世界最好的图鉴比肩的地区植物图鉴。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">前不久在柴达木的荒原中，得知牛洋和孙航编著出版了《横断山有花植物图鉴》的消息。回来后迫不及待地向孙航讨要了大作，拿到大作后又迫不及待地撕开包装，读了起来。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">这是一本让人有怦然心动的好书，有以下几个特点： </span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">一、种类收集全面丰富。横断山位于青藏高原东南缘，是全球35个生物多样性的热点地区，根据王文采院士编撰的《横断山区维管植物（上下）》，横断山有维管植物8559种，隶属1467属，219科。牛洋、孙航的这部《横断山有花植物图鉴》包含了1857种，隶属120科，涵盖了横断山近1/4之一的种类和大多数的代表类群。在近年出版的图鉴中，是种类收集最多的了。要知道要收集这将近2000多种植物的照片，可不是一件容易的事情。这需要一次次地，在不同是时间，不同的季节走遍横断山的各个角落，才会由此收获。这些种类，这些图片是作者长期坚守的心血。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">二、物种鉴定准确。物种鉴定准确对于一部图鉴而言，似乎是最低的要求。但是，现在不少的图鉴都有物种鉴定不准确的问题。这部图鉴的作者孙航，是国内植物分类学和植物区系地理方面的顶级专家，有深厚的分类学造诣，又长期在青藏高原和横断山开展野外工作，熟悉这里的一草一木。他领衔编著了这部专著，使分类鉴定有了可靠的保障。主编牛洋和参与编撰的主要作者，如周卓等人大多数是孙航的学生，是工作在生物多样性编目、保护和演变研究一线的青年分类学家，他们长期的学术积累和研究水平，也是这部专著学术质量的保障。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">三、图版精美。 图鉴的第一作者牛洋，不仅是一位年轻有为的植物学家，还是一位有着较高美术品味和才华横溢的艺术家。Plant Diversity期刊每期的封面设计都是出自于他的手。这部图鉴的照片即突出了物种的主要鉴定特征，又清晰而优美，不少些照片堪称艺术精品。在优先突出物种主要鉴定特征和小生境的原则，而且受版面限制的情况下，作者做到了科学和艺术的完美结合。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">四、使用方便，排版设计新颖。现在的不少图鉴装潢豪华，用纸奢侈，印成大十六开本。这样的书看上去十分高大尚，放在书斋里十分体面。然而，图鉴是那种要经常带着包里，那种拿手上，经常翻阅的工具书。现在这种32开本的图书，就非常好用而且方便携带。在编排上，作者们也是下了一番功夫的。图文放在同一个打开的页面上，图号与同一页面的物种序号对应， 方便同时对比观察图片和文字。这事似乎小到不值一提。但正是这个小小举动，体现了作者们对读者阅读习惯的理解、尊重和友好。有些图鉴，图和文字排列在不同的页面，阅读和使用都非常不便。一段描述后说见图3，图3在哪里还得费力气去找。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">五、文字简练而信息丰富。这部图鉴描述仅保留了重要关键形状，虽然字数有限，但是通过巧妙的编排，还包含了花期，海拔和分布范围等重要信息。而且，对于一些大的类群，提供了初级检索的信息，帮助作者快速识别所遇到的植物。对了，图鉴文字中英对照，全球植物学爱好者都可以使用，国内学者也可以考较一下自己的英文水平。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">经过80多年几代植物学的努力，长篇巨著《中国植物志》，在2004年全部出版。许多地方志也相继出版。在这种背景下，是否还需要图鉴这类的书籍，回答是肯定的。《植物志》是一个国家植物种类最权威的记录，是一个国家植物的户口本。作为一个正规的户口本，对其言语、格式和编排都有专门的规定。植物志也有插图，但是数量有限。对于缺乏基本的植物学概念和不熟悉植物学形态解剖的专业术语的普通大众，仅凭证植物志，是很难查到所遇到的植物的。图鉴就是一部看图识字的直白读本，普通读者通过“看图识字”就可以认识植物。对于专业学植物工作者，图鉴也提供了一个快速认识自己不熟悉类群的渠道。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">植物世界宛如一个巨大的迷宫，稍不留神就会迷失其中。图鉴就是探索植物迷宫的指路牌。有图鉴在手，认识横断山植物不用愁。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">图书的编撰是一个遗憾的工程，出版之前，无论考虑多全面，校对多仔细，出版之后，都会留下这样那样的遗憾。作为一个读者，对于图鉴有几点建议：</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">图鉴如果能够再增加一个常见的中文名，对普通的读者为更为方便。虽然说拉丁学名是植物的法定名称，但是大家还是更习惯用中文名来交流和识别植物。比如说川滇高山栎（<em>Quercus aquifolioides</em>）, 就比巴郎栎这个名字更流行一些。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">限于篇幅的限制，大多数物种仅有一张图片，对于那些观赏性极高的植物，或者那种一幅图不能看出区别特征的植物，多一些图片会更好一些。此外，虽然这部图鉴包含的种类已经非常丰富了，但是仍给人一种意犹未尽的感觉。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: DengXian;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">现在电子图书比较流行，作者们是否可以和出版社合作，以此为蓝本，出一个电子版的图鉴。买一个app装在手机上，这样使用和携带都会更加方便。</span></p><p style="margin: 0px 0px 13px; text-align: justify; line-height: 21px; white-space: normal; text-indent: 37px;"><span style="font-family:宋体"><span style="font-size: 19px;">我幻想着有那么一天，无数的植物学爱好者，拿着这部图鉴去横断山打卡植物，我站在寒风凛冽的山口，一个小仙女过来问我，老爷爷，你知道什么地方能看到垫紫草吗。</span></span></p><p style="margin: 0 0 13px;text-align:
