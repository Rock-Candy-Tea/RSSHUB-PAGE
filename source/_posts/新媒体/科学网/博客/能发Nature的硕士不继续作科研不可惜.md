<p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">近日，浙江理工大学硕士生郝治伟以第一作者身份在国际顶尖学术期刊（所谓</span>“顶刊”）Nature上发表一篇学术论文，引发关注。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">从相关媒体的报道来看，相对于论文本身，人们更好奇的是硕士生取得如此骄人的学术成果，为什么不选择继续从事科研而选择回家乡当公务员。有网友认为：</span>“一颗冉冉升起的学术新星没有去读博深造非常可惜。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">其实，这样的事例在科学圈的人看来早已不足为奇。没有去读博深造的</span>“学术新星”大有人在，即使读博并取得亮眼学术成果但毕业后不再从事科研的人也不少。这样的例子在以往的媒体报道中也不难找到。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">能在</span>Nature等顶刊发表学术论文的“学术新星”们不选择继续走科研道路，可惜吗？</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">在</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">老文</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">看来，不仅不可惜，而且值得尊重和鼓励。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><strong><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">其一，研究生教育的目标本身多元，并不局限于培养职业科研人员，这从根本上决定了能发顶刊论文的研究生进入任何需要研究生的行业都属正常。</span></span></strong></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">研究生教育跟其他层次的学历教育一样，应社会的需求而变化。它本身并不局限于而且早已主要不是培养职业科研人员。现在的研究生规模日益庞大，就是为了满足各行各业对研究生层次人才日益增长的需求。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">研究生是做研究的学生，能发顶刊论文是研究生科研能力和水平的重要标志。任何行业，既然需要研究生，当然能力越强水平越高的研究生越受欢迎。从这个意义上说，能发顶刊论文的研究生毕业后进入任何行业都属正常。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">如果说这样的研究生不继续从事科研可惜了，那是否意味着科研之外的行业不需要能力强水平高的研究生，或这样的研究生进入科研之外的行业是种浪费？这样的逻辑有违研究生教育的宗旨和目标，实践证明，也误导研究生的学业。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px">2012年，科学网发起过一场“科研苗子”逃离科研现象的讨论，起因是一位中科院优秀博士生毕业放弃留校</span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">作科研</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">而选择当中学老师，有人认为是种惋惜。</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">老文</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">当时也受邀发表过观点：这种显性的逃离科研应该尊重，并不可怕，更普遍、更可怕的是隐性的逃离科研，即人在科研但心并不在科研。例如，一些研究生在读期间达到规定的毕业要求后立马放弃科研，无意做到更好，更不想做得更多，只盼尽快毕业显性逃离科研。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">研究生在读期间为什么不追求更好？许多导师有体会，一些研究生认为或相信如果毕业后不打算</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">作科研</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">就没必要为科研付出更多。能在顶刊发论文的硕士选择当公务员可惜了的观点，跟这种认识本质上没有什么不同。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">郝治伟硕士说得好，自己的科研经历对日后的行政工作</span>“肯定有很大的帮助，科研经历对我来说永远是笔宝贵的财富”。推广来说，任何行业跟科研领域一样，需要能发顶刊论文的研究生，能发顶刊论文的研究生在任何行业跟在科研领域一样，也有施展才华的用武之地。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><strong><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">其二，从科研人员的成长历程看，仅凭硕士阶段的一篇论文预测研究生未来的学术前途，为时尚早。</span></span></strong></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">科学研究是提出问题和解决问题。爱因斯坦说：提出一个问题往往比解决一个问题更重要，因为解决一个问题也许仅仅是一个数学上或实验上的技能而已，而提出新的问题却需要有创新性的想象力，而且标志着科学的真正进步。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">硕士研究生大多在导师的指导下开展研究，其选题往往来自导师，研究生的作用主要在解决问题。郝治伟硕士的讲述反映了绝大多数研究生科研工作的基本状况。他说：</span>“虽然我是第一作者，但这篇论文其实从选题、构思，到后来的修改和加工，大部分工作都是在左彪老师的指导下完成的。我负责的主要是实验的具体实施和推进，同时也会和老师及时讨论，不断改进实验方案。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">诺贝尔生理或医学奖得主梅达沃在《对年轻科学家的忠告》一书中写道：</span>“我可以很自信地说，任何科学家，无论年龄大小，要想取得重要的发现，都必须研究重要的问题。”能否提出重要的科学问题，从根本上决定了一个科研人员能否取得重要发现或成果。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">科学问题从哪来？研究生可以向导师要，但离开导师后，如果继续</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">作科研</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">，那么最大的问题就是没有问题。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px">“试玉要烧三日满，辨材须待七年期。”从当研究生在导师指导下发顶刊论文，到自己独立或指导他人发同等论文，其间还有一道鸿沟，跨过这道鸿沟才意味着一个学者的真正独立，这时再谈学术前途才比较靠谱。世界一流大学招聘教师特别是长聘教师，就是这么做的。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><strong><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">其三，最重要的是，如果遵循自己内心的选择，那么放弃任何已展现出能力和水平的行当，都不算可惜。</span></span></strong></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">著名实业家稻盛和夫说，要想拥有一个充实的人生，你只有两种选择：一种是</span>“从事自己喜欢的工作”，另一种则是“让自己喜欢上工作”。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">找到自己喜欢的工作就像找到自己心爱的人一样，是一件值得庆幸的事。正如科学家喜欢科学，可以为科学放弃其他一切，研究生如果有喜欢的其他选择，也完全可以放弃科研。梅达沃忠告研究生：</span>“试图在科研上一试身手但又发现自己对科研不感兴趣或干脆对它讨厌的新手，应该离开科学界，而不要有丝毫自责或误入歧途之感。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">任何研究生，如果选择继续</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">作科研</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">，那么他们应该从著名科学家的经历或名言中看到了，支撑科学家</span>“不畏劳苦沿着陡峭山路攀登”达到“光辉顶点”的，是对科学的兴趣或热情，而不是已经展露出来的能力和水平。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">博物学家威尔逊在《给年轻科学家的信》中的第一封信就是</span>“先有热情，再谈训练”。他在另一封信中写道：“想要在科学界闯出一片天地，光靠聪明才智是不够的。……你必须要具备一种特征，能够享受长时间学习和研究的乐趣，即便有时候一切努力都付诸流水，这就是要跻身一流科学家行列的代价。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">总之，在</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">老文</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">看来，能在顶刊发论文的硕士不继续</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">作科研</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">，一点都不可惜；真正可惜的，是从读研开始就排除了未来</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">作科研</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">从而认为没必要为科研付出更多的研究生，他们浪费了在追求诸如发顶刊论文过程中的难得科研历练机会。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">至于吃瓜群众判断一个研究生是否适合</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">作科研</span></span><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">，与其看他</span>/她目前取得了什么成绩，不如看他/她的梦想是什么，道理如威尔逊所说：“ 认识一个人的梦想，就等于认识了那个人。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: Calibri;font-size: 14px;text-indent: 0;white-space: normal;line-height: 28px"><span style="font-family: 新宋体;line-height: 32px;font-size: 16px"><span style="font-family:新宋体">（本文应科学网邀请撰写，首发于科学网微信公众号</span>2021-8-27）</span></p><p><br></p>
