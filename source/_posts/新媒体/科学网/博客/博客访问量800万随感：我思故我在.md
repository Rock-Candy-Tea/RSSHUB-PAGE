<p style="text-indent: 32px;line-height: 200%;background: white"><span style="color: #000000;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">光阴荏苒，日月如梭。不知不觉间，我的博客访问量已近</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif;">800</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体;">万了。趁着今天下午稍有空闲，就写篇随感吧，反正迟写不如早写。</span></span></p><p style="text-indent: 28px;line-height: 200%;background: white"><a href="https://zhidao.baidu.com/question/2058330442541887707.html" style="text-decoration: underline; font-size: 16px; line-height: 200%; font-family: 宋体; color: rgb(0, 0, 0);" target="_blank"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">笛卡尔</span></a><span style="color: #000000;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #444444;">说过一句名言：“</span><span style="color: #000000; font-size: 16px; line-height: 200%; font-family: 宋体;">我思故我在（</span><span style="color: #000000; font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif;">I think, therefore I am</span><span style="color: #000000; font-size: 16px; line-height: 200%; font-family: 宋体;">.）。</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #444444;">”不同的人对这句话有不同的理解。以我的拙见，一个正常人若失去独立思考的能力，则往往被别人所左右，导致自己无从判断是非，也难以为社会发展助力，这几乎无异于行尸走肉。</span></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">知识分子是特别爱独立思考的群体。每位知识分子每天不思考某事儿，或者思考没有收获，八成会处于没着没落的状态；一旦思考有了收获，会觉得自己的生存有价值，生命有力量，生活有希望。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="color: #000000;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #444444;">思考，可能产生转瞬即逝的思想火花。正如世界上没有两片完全相同的树叶一样，不同人的思想不可能完全一致，这就是独特性。独特的思想火花，不管是否有价值，应及时记录存档，毕竟好记性不如烂笔头嘛。我觉得写有条理的文章是存档的好方式，即</span><span style="color: #000000; font-size: 16px; line-height: 200%; font-family: 宋体;">通过撰写文章，可把零散的信息系统化，把粗浅的认识深刻化。写好的文章，若适合在博客上发表，没必要藏着掖着，分享之供网友们质疑讨论是共赢选项。此外，个人博客应聚焦于几个主题，不宜太发散，且体现理性精神和营养性；否则，即使博主撰写更多吸引眼球的博文，到头来也不会被别人记住。</span></span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">我参加会议、应酬时，不少人士夸我博文写得好且富有营养，能引领地质灾害物理预测学术研究，唤回科研初心，启发别人的深层次思索。我总是回应说一般一般，尚需更上一层楼，让人家多提宝贵意见。其实，我从未想过也不想成为靠炒作起家的网红——那是细思极恐的事儿，而想当科普地质灾害机理、分享科学研究方法、解析科研评价政策、弘扬科学精神与倡导原创的排头兵。若干年后，如果我的某些观点仍被大家认可和传承，甚至仅记得“锁固段”三字，则心愿足矣。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">有人说，世界上有两大难事儿：其一是把你的想法装进别人的脑袋，其二是把别人的钱装进自己的腰包。对后者我想都不想，对前者确有些浅见：只要你的想法符合逻辑自洽原则且有强壮证据支持，哪怕与过去的认识截然不同，哪怕让别人没面子，慢慢地会得到大家的认可；这个过程或许较长，前行者需要有足够的耐心等待后进者跟上来。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">每个人的学识、修养和境界不同，所以看待学术问题的角度和深度不同，这就会导致对同样的问题有不同的看法，这也是引起争议的本质原因。鉴于此，辩论双方应客观面对争议，理性交流，厘清争议的渊源，给出自己的实在证据（可靠的观测数据、实验数据等可作为证据，不宜以文献结论和名人名言作为证据），以理服人。即使认为对方在胡扯搞笑，亦不可居高临下、火冒三丈、固执己见，应以谦听则明、求同存异的态度泰然处之，实在不行可“搁置争议”；等以后双方的认识提高了，则可能达成共识。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">我注意到不少网友提出自己的观点时，缺乏相关基础知识背景，未按基本物理学原理并结合切实的证据论证之，漏洞百出。其对别人的质疑或置之不理、或强词夺理、或胡搅蛮缠，极力想维护自己的认识，这皆不是科学的态度。每个人的认识都有局限性，正所谓“偏信则暗”嘛。是啊，本着开放、包容、理性的态度面对质疑，无疑可纠偏方向、规避陷阱、少走弯路。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">在科学探索之路上，知识分子经长期深度思考可能把一连串想法有机整合形成</span><a href="http://blog.sciencenet.cn/blog-575926-1298273.html" style="font-size: 16px; line-height: 200%; font-family: 宋体; color: rgb(0, 0, 0); text-decoration: underline;" target="_blank"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">学术思想</span></a><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">。知识分子正确的学术思想，<span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #1E1E1E;">是从天上掉下来的吗？不是；是自己头脑里固有的吗？也不是。</span>知识分子正确的学术思想<span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #1E1E1E;">，只能从思想实验、科学实验和实践中来。</span>所谓正确的学术思想，也就是揭示了某种事物本质之道的思想。诚然，知识分子<span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #1E1E1E;">对客观事物的认识，往往要经历从感性认识到理性认识的过程。</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000; border: 1px none windowtext; padding: 0px;">在这个过程中，也就是学术思想形成过程中，</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #1E1E1E;">要着重调查分析，要聚焦剖析机理，要落实实证研究，还</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000; border: 1px none windowtext; padding: 0px;">要以</span></span><a href="http://blog.sciencenet.cn/blog-575926-1294013.html" target="_blank" style="font-size: 16px; line-height: 200%; font-family: 宋体; color: rgb(0, 0, 0); text-decoration: underline;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">奥卡姆剃刀律</span></a><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">为指导以尽快拨云见日找到相对真理。</span></p><p style="text-indent: 32px;line-height: 200%;background: white"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; color: #000000;">确实，我思故我在，对知识分子更是如此。显然，更多的有益思考，更能促进我们对大千世界的理解，更能为科技发展注入活力，更能为人类社会发展带来勃勃生机，更能体现知识分子的价值。这就是我写这篇博文的目的所在。</span></p><p style="line-height: 200%;background: white"><span style="color: #000000;"><strong><span style="color: #000000;
