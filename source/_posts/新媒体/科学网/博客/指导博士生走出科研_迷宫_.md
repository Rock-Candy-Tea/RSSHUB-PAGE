<p style="line-height:150%"><img src="https://cors.zfour.workers.dev/?http://image.sciencenet.cn/home/202206/02/143608b0d1s6za7w0sayzw.jpg" title alt="IMG_20220312_172855.jpg" referrerpolicy="no-referrer"><br></p><p style="text-indent: 32px; line-height: 2em;"><span style="font-size: 18px; color: #0C0C0C;"><span style="font-size: 18px; line-height: 150%; font-family: 宋体; background: white;">2014年，小刘从别的一所学校考进复旦大学，加入我的课题组。我给他指明了一个研究领域——新型负载型金催化剂的CO（一氧化碳）氧化，让他着手进行文献调研。可是，他的思维跳到了别的研究领域：他</span><span style="font-size: 18px; line-height: 150%; font-family: 宋体;">看到有篇文献用“一锅法”合成了介孔（具有纳米孔的）CuO-Al<sub>2</sub>O<sub>3</sub>，就提议将此材料用于催化分解N<sub>2</sub>O（氧化亚氮）。<span style="font-size: 18px; line-height: 150%; font-family: 宋体; color: #C00000;">但我认为，如果只是机械地复制别人的介孔材料，然后简单地测试该材料在一个反应中的性能，那没有意思。我还预测，用介孔CuO-Al<sub>2</sub>O<sub>3</sub>催化分解N<sub>2</sub>O是无效的——我曾发现介孔CuO对该反应没有活性，Al<sub>2</sub>O<sub>3</sub>对该反应也没有活性。因此，认为把两种没有活性的氧化物组合在一起，催化分解N<sub>2</sub>O的效果也不会好。</span></span></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">可是小刘瞒着我制备了介孔CuO-Al<sub>2</sub>O<sub>3</sub>，并用它催化分解N<sub>2</sub>O，发现一点活性都没有。我察觉后，问他为什么偷偷做实验。他说我们课题组先前和别的课题组合作，发现Cu-ZSM-11沸石中的Cu<sup>+</sup>是催化分解N<sub>2</sub>O的活性位，而文献报道介孔CuO-Al<sub>2</sub>O<sub>3</sub>含有Cu<sup>+</sup>，因此他认为介孔CuO-Al<sub>2</sub>O<sub>3</sub>可能对该反应有活性。如果不加以验证，他心里“像猫抓一样难受”。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">我很生气，因为我早就叫他别做这种实验，并且得到的结果也不好。但同事说，至少这位学生有自己的科研想法（哪怕不成熟），也肯做（哪怕结果不理想），要利用他的科研积极性，引导他走上正轨。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">于是，我和小刘细细讨论。我问他很多问题：别人合成介孔CuO-Al<sub>2</sub>O<sub>3</sub>的方法有什么创新性？他们研究了什么催化反应？该反应的活性位是什么？文献说介孔CuO-Al<sub>2</sub>O<sub>3</sub>含有Cu<sup>+</sup>，那么Cu<sup>+</sup>占所有Cu物种的比例有多大？催化剂焙烧温度有多高？催化剂有没有用H<sub>2</sub>或者He预处理？你的制备方法和别人的有什么区别？为什么催化剂用到N<sub>2</sub>O分解中就无效了？有没有可能催化剂的Cu物种主要是CuO？Cu含量多少？如果催化剂对N<sub>2</sub>O分解无效，有没有可能对其他反应（比如CO氧化）有效？如果对其他反应有效，那么这个工作的创新性又在哪里？</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">小刘被我连珠炮似的问题问倒了。<span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #C00000;">我告诉他，读文献不能“一笔糊涂账”，也不能“见着风就是雨”。介孔CuO-Al<sub>2</sub>O<sub>3</sub>在某种处理条件下含有Cu<sup>+</sup>，不代表全部Cu物种就是Cu<sup>+</sup>。并且，Cu-ZSM-11沸石骨架中的Cu<sup>+</sup>对于催化分解N<sub>2</sub>O有用，不代表CuO-Al<sub>2</sub>O<sub>3</sub>中含有的Cu<sup>+</sup>对该反应有用。</span></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #C00000;">我对小刘说，我不希望学生只是机械重复文献的合成实验，然后千篇一律地在论文里说“XRD数据表明介孔是存在的，TEM结果表明有介孔的”，轻飘飘地给出催化活性数据，并提出陈词滥调的结论（例如“纳米催化剂比普通催化剂好”）。做催化研究，最好所用的材料都是新的，而且论文要体现“探究”的过程。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">带着问题，小刘考察了介孔CuO-Al<sub>2</sub>O<sub>3</sub>催化的CO氧化，发现有一定效果，但正如我所说的，即便有效果，还是没有创新性，回答不了“即便得到了活性，那又怎么样？”的问题。我让小刘把CuO换成其他氧化物看看有没有更好的效果。于是他制备了一系列介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>，发现它们对催化分解N<sub>2</sub>O一点活性都没有，对于CO氧化活性有些区别，但总体活性不高。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">好几个月过去了，还没有关键进展，我们有点“骑虎难下”。这时，我想到负载型Rh<sub>2</sub>O<sub>3</sub>催化剂在N<sub>2</sub>O分解和CO氧化中有用，如果把Rh<sub>2</sub>O<sub>3</sub>负载在一系列介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>上，催化剂活性会更高，不同的载体对催化性能可能会呈现出“载体效应”。并且，把Rh<sub>2</sub>O<sub>3</sub>负载在一系列介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>上也没有被报道过。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">小刘和我讨论以后，感到豁然开朗。他很快制备出Rh<sub>2</sub>O<sub>3</sub>/MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>催化剂，并进行了催化测试，发现催化活性的确比MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>高多了，且不同催化剂的效果有明显区别。他又对系列催化剂进行了详细的表征，并撰写了论文。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #C00000;">小刘最初写的论文引言部分强调了N<sub>2</sub>O是一种空气污染物，因此我们需要开发新的催化剂去脱除N<sub>2</sub>O。看到小刘的初稿，我指导他：写引言不能人云亦云，“别人也这么写”不能成为自己也这么写的理由，要有自己的归纳提炼。还要注意角度：整篇论文是以新材料的开发为导向，是以催化性能为导向的，还是以反应机理为导向？如果你的催化剂效果没有那么好，或者催化剂效果好坏的道理说不清，但一系列催化材料是新的，为什么不从新催化剂开发的角度写引言？</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">我提出这篇论文的引言部分的写作思路：介孔SiO<sub>2</sub>是一类重要材料，有广泛用途，但介孔Al<sub>2</sub>O<sub>3</sub>较难合成。文献中有人采用某种方法成功制备出Al<sub>2</sub>O<sub>3</sub>，进而有很多关于介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>的报道。介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>可以直接作为催化剂，催化<span style="color: #0C0C0C; line-height: 150%; font-family: 宋体; font-size: 18px; background: white;">很多</span>反应（举例说明）。它们还可以作为前驱体，通过氢气还原制备负载型金属催化剂（举例说明）。但是，以介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>作为载体负载活性组分制备催化剂的报道很少。本文以一系列介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>作为载体负载Rh<sub>2</sub>O<sub>3</sub>，采用多种表征手段进行详细表征，并研究了N<sub>2</sub>O分解和CO氧化。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">应该“马后炮”似地，根据“卖点”或者“标题”写引言。我给小刘梳理论文的标题和写引言的提纲。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">标题：介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>的合成、表征和催化应用</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; color: #0C0C0C; font-size: 18px;">1. 介孔氧化硅引起人们广泛关注，很有用</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; color: #0C0C0C; font-size: 18px;">2. 大多数文献集中在介孔氧化硅，其他介孔氧化物报道相对少，合成较难</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; color: #0C0C0C; font-size: 18px;">3. 某人用某种方法成功合成出介孔氧化铝，后来有些文献报道介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; color: #0C0C0C; font-size: 18px;">4. 介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>自己可以作为催化剂（例子）</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; color: #0C0C0C; font-size: 18px;">5. 介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>可以作为催化剂的前躯体制备新的催化剂（例子）</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; color: #0C0C0C; font-size: 18px;">6. 介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>可以作为载体制备负载型催化剂，但这方面报道相对少（例子）</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; color: #0C0C0C; font-size: 18px;">7. 本文采用介孔MO<sub>x</sub>-Al<sub>2</sub>O<sub>3</sub>作为载体负载Rh<sub>2</sub>O<sub>3</sub>，探索了催化应用</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">费了好大的劲，这个研究终于完成了，论文也发表了。小刘从中学到了选题、做实验、写论文、克服困难的方法，他对我的指导更加信服了。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 150%; font-family: 宋体; font-size: 18px; color: #0C0C0C;">后记：小刘三年在读期间的科研工作整理出5篇SCI论文，他在2017年顺利毕业。</span></p>                    <br><br>
