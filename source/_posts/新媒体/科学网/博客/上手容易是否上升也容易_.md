<p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">在前不久举行的第</span>38期院长午餐会上，一位大三学生谈及他早进课题组、早进实验室、早进团队的经历。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">他首先加入一个做前沿基础研究的课题组，一段时间后，他感觉那个课题组的工作太难上手了，涉及到的概念和知识很多他还没学过，有些甚至在现有的任何教材中都找不到。他说他费了不少功夫，但进展太慢，于是退出了这个课题组，加入了另一个偏实际应用的团队。他认为这个团队的工作很容易上手，稍微做点事情就有获得感，这让他心里非常踏实。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">研究生导师应该都有体会，根据上手的难易程度选择或改变研究领域，在研究生中也很常见。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">谈几点个人体会。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><strong><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">首先，我们为什么倾向选择容易上手的领域或课题？</span></span></strong></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">心理学上有个心流</span>(flow) 理论。米哈里·契克森米哈赖把心流定义为一种将个人精神力完全投注在某种活动上的感觉；心流产生时同时会有高度的兴奋及充实感。（见《心流：最优体验心理学》）他发现，学生对挑战的反应取决于学生所拥有的技能水平：如果挑战的难度大大超过学生的技能水平，就会引发学生的焦虑，反之，学生就会感到无聊；当学生的技能水平和挑战的难度大致匹配时，学生就会进入心流状态，即最佳学习状态。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">玩过进阶游戏的人很容易理解心流理论。玩没啥挑战度的游戏很容易感觉无聊，而玩太难的游戏，如果玩了几把老不进阶，就很可能放弃，改玩其它游戏。既有挑战度又能获得成就感（例如进阶或获得奖励）的游戏最可能让人玩下去。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">进入不同的研究领域，或从事同一领域不同性质的研究工作，有高低不一的门槛要求。刚刚接触科研的学生，一般还谈不上什么科研技能和水平，如果顺着心流，自然选择容易上手的领域或事情。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">此外，目标或需求决定选择。如果我们设定的目标只为</span>“眼前的苟且”，那么“诗和远方”尽管重要，往往被我们排在“以后再说”。此所谓需要比重要紧要。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">有些学生参与科研的目的很现实：只为尽快收获一篇小论文，以此谋取评优、保研、毕业或求职上的优势。实践表明，这些学生最可能根据上手的难易程度选择研究领域或事情。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">顶尖科学家往往也根据想达到的目标反过来考虑选择做什么。对他们来说，正如《爱因斯坦的老板》一书所说，</span>“使命必须具有足够的挑战性，这样在完成任务时就会产生自豪感。实现这一目标需要提供的不仅仅是经济上的回报；它必须有意义。他们要知道他们的智慧将会改变世界。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><strong><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">其次，为什么我们感觉有些领域上手很难有些领域上手容易？</span></span></strong></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">科学领域本身是没有难易之分的。如果有的话，跟人有关。例如，爱因斯坦认为物理不难，但达尔文可能认为物理不易，至少比生物难。从这点讲，</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">匹配</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">到自己擅长或</span>“来电”的领域是件幸事。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">科研就是提出问题和解决问题。所以，如果说科研领域有难易之分，那么这种难易程度取决于所问的问题而不是所在的领域。你感觉容易的领域也有很难的问题，而你感觉很难的领域也有容易的问题。复杂性科学学者</span></span><a href="https://aeon.co/ideas/when-science-hits-a-limit-learn-to-ask-different-questions" target="_blank"><span style="text-decoration:underline;"><span style="font-family: 宋体;line-height: 32px;color: #0000FF;font-size: 16px">Chris Kempes<span style="font-family:宋体">和</span><span style="font-family:Times New Roman">Van Savage</span></span></span></a><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">说：一个学科看起来困难是因为它由一系列非常困难的问题构成，另一个学科看起来简单是因为经过大批天才思想家的努力，其中的问题已变成一系列可回答的问题。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">打个比喻。一个科研领域，相当于一个地理空间。当它在被开发成为旅游景点后，我们很容易进入它并获得一些知识；但在这之前，我们可能需要克服不少困难甚至冒很大风险才能进入它，即使这样也可能才只对它有些许了解。显然，像前者那样的领域远比后者容易上手并有获得感。但须注意，前者充其量相当于</span>“行万里路”，而后者是探索。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">我们感觉某些领域容易上手，也许是个错觉：上手的只是学习，而不是科研。这种错觉源于我们对领域的熟悉程度和跟领域前沿之间的距离。当我们直抵前沿，直面同等复杂的问题时，就会感觉每个领域都不容易，甚至感觉容易的领域取得同等成果反而更难。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><strong><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">最后，最关键的是，上手容易的领域或课题是否令我们上升也容易</span>?</span></strong></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">常识告诉我们绝非如此。马克思有句至理名言：</span>“在科学上没有平坦的大道，只有不畏劳苦沿着陡峭山路攀登的人，才有希望达到光辉的顶点。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">科研有如唐僧师徒西天取经。条条大路通灵山，取经团队为何独独选择一条魔难重重的路径？</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">猪八戒是个最会盘算的人，他多次提出过这个问题。例如，取经团队路阻火焰山时，八戒怨声载道，建议改方向：</span>“且回去，转路走他娘罢！”土地劝八戒：“但说转路，就是入了傍门，不成个修行之类，古语云：‘行不由径’，岂可转走？你那师父，在正路上坐着，眼巴巴只望你们成功哩！”八戒如果不是信了土地劝告，并当即表态“昼夜休离苦尽功”，他赶赴“盂兰会”的梦想肯定就此破灭。</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">科研</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">新手提升自己</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">有两个因素很关键。一是跟高人在</span></span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">一起。</span>2001年诺贝尔生理学或医学奖得主Timothy Hunt说，科学的诀窍是不断找到远比你聪明的人一起工作（The trick in science is to always find people to work with who are more clever than you are）。</span><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">二是解难题。韦恩</span>·布斯等人著《研究是一门艺术》说：“一个研究者如果没有好的研究难题，就真的是个棘手的实际难题了，因为如果没有研究难题作为研究的对象，就没什么研究可以做的了。”</span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size: 16px"><span style="font-family:等线">以老文的体会，甘当人梯又善当人梯的老师就是高人，他们往往有法子让学生上手的坡度或梯度变低，使学生不知不觉达到一个高度，从而产生继续往上攀登的信心。当我们感觉自己感兴趣的领域或课题很难上手时，与其改领域或换课题，不妨再等一等，看看课题组或团队有没有能够帮到我们的高人，如果有，坚持留下是最好选择。</span></span></p><p style="margin: 0 0 10px;text-align: justify;font-family: 等线;font-size: 14px;white-space: normal;-webkit-text-size-adjust: auto;line-height: 28px"><span style="line-height: 32px;font-size:
