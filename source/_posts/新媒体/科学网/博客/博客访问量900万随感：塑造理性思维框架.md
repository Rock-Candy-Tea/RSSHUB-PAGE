font-size: 16px; line-height: 200%; background: white; font-family: "times new roman";"></span>从我的博客访问量</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; background: white;">600</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white;">万起，每增加约</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; background: white;">100</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white;">万，就写一篇博文留痕。今天访问量已略过</span><span style="font-size: 16px; line-height: 200%; font-family: "Times New Roman", serif; background: white;">900</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white;">万了，概莫能外。访问量这么高，出乎我的预料，尽管这不是我关注的目标，但表达感谢是必须滴。在此，感谢博主们的积极推荐，感谢网友们的密切关注，感谢编辑们的热情支持。</span></span></p><p style="text-indent: 32px; line-height: 2em;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: #000000;">言归正传。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: #000000;">理性思维（简称理思）是一种基于证据和逻辑论证的思维方式。具有这种思维方式的人们，易于把握客观事物的本质和规律。如果某人思考有逻辑、思维有条理、思想有深度，则其为理性的人，也是了不起的人。</span></p><p style="text-indent: 32px; background: white; line-height: 2em;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: #000000;">电影《教父》里有句经典台词：“花半秒钟就看透事物本质的人，和花一辈子都看不清事物本质的人，注定是截然不同的命运。”这说明高效率地看清事物本质是何等的重要；而要做到此，离不开强劲的理思。</span></p><p style="text-indent: 32px; background: white; line-height: 2em;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: #000000;">为何不少人常被骗子的花言巧语迷惑，常被大神的心灵鸡汤灌晕，常被非科学的奇巧百怪观点忽悠，常被伪科学的荒诞无稽主张捉弄，主要源于其较差的理思能力。</span><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: #000000;">为防止中招，诸君应不断提升自己的逻辑推理能力（包括推理前提的扎实性和推理过程的严密性）和基于推理的分析决断能力，重视证据的强壮性（如是否具有多解性）。</span></p><p style="text-indent: 32px; background: white; line-height: 2em;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: #000000;">所谓理思框架，就是以理思方式系统地看待和处理问题的思路和方法。欲培育强劲的理思框架，需要：</span></p><p style="text-indent: 32px; background: white; line-height: 2em;"><span style="font-size: 16px; color: #000000;"><span style="color: #000000; font-size: 16px; font-family: "times new roman";"><strong><span style="font-size: 16px; color: #333333;">1</span></strong></span><span style="color: #000000; font-size: 16px; font-family: 宋体, SimSun;"><strong><span style="font-size: 16px; color: #333333;"></span></strong><strong><span style="font-size: 16px; color: #333333;">.</span></strong><strong><span style="font-size: 16px; color: #333333;">丰富自己的知识</span></strong></span></span></p><p style="text-indent: 32px; background: white; line-height: 2em;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: #000000;">有人说，物质的贫穷限制了想象；类似地，知识的匮乏约束了思路。鉴于此，作为科研人员，不仅要看本专业的文献，还要看有关专业的文献，甚至要博览不相关专业的文献。是啊，书到用时方恨少，思路打不开时方知自己的知识面窄。因此，广为学习并掌握多种知识，可备不时之需，还有助于自己的思路被卡时破除桎梏，从而一通百通。</span></p><p style="text-indent: 32px; background: white; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 16px; color: #000000;"><strong><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #111111;">2</span></strong><strong><span style="font-size: 16px; font-family: 宋体, SimSun; color: #111111;">.</span></strong><strong><span style="font-size: 16px; font-family: 宋体, SimSun; color: #111111;">保持审辩式思维</span></strong></span></p><p style="text-indent: 32px; background: white; line-height: 2em;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: #000000;">审辩式思维的主要特点有: (<span style="font-size: 16px; line-height: 200%; color: #333333; background: white; font-family: "times new roman";">1</span>)凭证据讲话，有一份证据讲一份话，不过度引申；(<span style="font-size: 16px; line-height: 200%; color: #333333; background: white; font-family: "times new roman";">2</span>)合乎逻辑地论证自己的观点；(<span style="font-size: 16px; line-height: 200%; color: #333333; background: white; font-family: "times new roman";">3</span>)善于提出问题，不懈质疑；(<span style="font-size: 16px; line-height: 200%; color: #333333; background: white; font-family: "times new roman";">4</span>)对自身的反省，和与此相关的对异见的包容。具有审辩式思维的人们，通常富有创造活力和激情，应予鼓励。然而，其中不少人未意识到自己思考过程中逻辑链的脆弱性，且忽略了证据的多解性，难免不自觉地进入认识误区，这是应该注意的。</span></p><p style="text-indent: 32px; background: white; line-height: 2em;"><span style="font-family: 宋体, SimSun; font-size: 16px; color: #000000;"><strong><span style="font-size: 16px; font-family: "Times New Roman", serif; color: #121212;">3</span></strong><strong><span style="font-size: 16px; font-family: 宋体, SimSun; color: #121212;">.</span></strong><strong><span style="font-size: 16px; font-family: 宋体, SimSun; color: #121212;">侧重结构化思维</span></strong></span></p><p style="text-indent: 32px; background: white; line-height: 2em;"><span style="font-size: 16px; line-height: 200%; font-family: 宋体; background: white; color: #000000;">寻求解决复杂问题之道时，或由于思维僵化而无计可施，或由于思维空白而无从下手。在这样的情况下，结构化思维（以事物的结构为对象，有序地思考其整体和部分的关系，要求纵向上突出重点，横向上突出层次性和逻辑性）大有用武之地，其既能有助于人们更全面地思考问题，又能通过梳理归类复杂的信息，把问题拆解成一个个简单的、可解决的问题单元。一旦给出了每个问题单元的解答，并把之有机串联起来，则能得到问题的系统解答。这意味着只有把复杂的问题结构化，才能实现解答的简单化。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">例如，我们已知主控构造地震产生的地质结构是锁固段，那么地震演化遵循什么样的规律就成为关键问题。直接解决该问题难度太大，于是乎我们按层次性，把之分解为锁固段的赋存环境、力学属性、力学行为、建模这四个较易解答的问题单元；给出了这些问题单元的解答后，把其有机地融合以保证逻辑链的自洽性和闭环性，再通过实证检验，就形成了锁固段脆性破裂理论的基本框架。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">结构化思维的益处多多，但要养成此种思维习惯，需要经过有意识地持续训练，其中勤于写作是重要的训练方式。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">不少人士都有这样的感受，文章写到半截，思路就中断了或跑题了，难以为继，这大概是缺乏结构化思维训练所致。为此，我建议博主们没事儿多写以结构性思维方式展现的博文，这一方面可强化自己的理思能力，且进而提升科研能力，有助于取得事半功倍的成果；另一方面分享富有营养且可读性强的文章可使他人受益。是啊，万丈高楼平地起，磨刀不误砍柴工；提升理思需勤练，功到火候效率升；硕果源自高成效，克难始于新思想；踏破铁鞋无觅处，得来全不费功夫。</span></p><p style="text-indent: 32px; line-height: 2em;"><span style="line-height: 200%; font-family: 宋体; font-size: 16px; color: #000000;">科研人员有两方面义不容辞的责任和义务，一是通过科研结出的硕果为国家强盛添砖加瓦，二是通过科普（如以博文形式）提升国人的科学素质。为此，希望诸君：科研科普两不误，每天都有新进步。</span></p><p style="line-height: 2em;"><span style="font-size: 16px;"><strong><span style="line-height: 200%;
