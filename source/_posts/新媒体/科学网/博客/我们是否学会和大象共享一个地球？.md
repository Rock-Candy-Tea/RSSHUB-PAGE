text-align: center; line-height: 21px; font-size: 14px; font-family: Calibri, sans-serif; white-space: normal;"><span style="font-size: 14px;"><span style="font-family: 宋体; text-align: justify; text-indent: 37px;">图1. 闯入西双版纳热带植物园的一群大象 </span><span style="font-family: 宋体; text-align: justify; text-indent: 37px;"> </span></span><span style="font-family: 宋体; font-size: 19px; text-align: justify; text-indent: 37px;">      </span></p><p style="margin: 0px 0px 13px; text-align: left; line-height: 21px; font-size: 14px; font-family: Calibri, sans-serif; white-space: normal;"><span style="font-family: 宋体; font-size: 19px; text-align: justify; text-indent: 37px;">     近一个月来，一群北上的大象给成了媒体关注的焦点，新闻联播和晚间新闻几乎天天有大象的消息。大象走红给广大吃瓜群众带来了无比的欢愉，一时间各路网红也纷纷碰瓷大象，消费大象。</span><br></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">在围观观大象的过程中，无论是正规媒体还是网红和自媒体，都会问一个问题即：大象为什么要北上？对这一问题的解释，各种猜想、观点和假说精彩纷呈：地球磁极改变说，气候变化说，头象年老昏庸说，种群分布区自然扩散说，原生地生境破坏说，林下砂仁种植说，北上昆明开会说等纷纷出笼。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">我不研究大象，没有能力评价和判断上述观点和假说正确与否。作为吃瓜群众，我最推崇北上开会说。大象要参加的这个会议名称全称叫做“联合国《生物多样性公约》第十五次缔约方大会（</span><span style="font-size: 19px;line-height: 28px">COP15</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">）”，简称就叫</span><span style="font-size: 19px;line-height: 28px">COP15</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">。</span><span style="font-size: 19px;line-height: 28px"> </span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">大象要来参加这个会议，是因为这个会议的主题是“生态文明：共建地球生命共同体”，用大白话说就是地球不仅是人类的地球，也是大象的地球，是所有生物的地球。昆明除了昆明人的昆明，也是大象的昆明，昆明人住的，大象也住的。据说大象的祖上也曾经生活在昆明，距昆明不远的昭通就发现过大象的化石。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">大象北上在拷问我们，是否准备好了和所有生命共享地球？把一个严肃的问题，以喜剧般的效果，带到了人们的面前。大象北上，考验着我们如何践行“生态文明”的理念。</span><span style="font-size: 19px;line-height: 28px"></span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">西双版纳是中国大象主要的原生居住地（云南临沧还有一处原生居住地），这是一块面积仅为</span><span style="font-size: 19px;line-height: 28px">19124.5</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">平方千米的土地。幅员并不辽阔的西双版纳国家级自然保护区的面积为</span><span style="font-size: 19px;line-height: 28px">24.25</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">万公顷，为全州面积的</span><span style="font-size: 19px;line-height: 28px">12.68%</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">。西双版纳大象的数量，几十年间从</span><span style="font-size: 19px;line-height: 28px">200</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">头左右，增加到了现在</span><span style="font-size: 19px;line-height: 28px">300</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">多头。在西双版纳，大象走出原生居住地的事情可谓是屡见不鲜，时有发生。大象不仅北上，而且也南下，以它们的原生居住地为核心，向东南西北都有扩散。这一次是走得太远了，才成功的引起了关注。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">大象为什么离家出走，是不是他们的家园不好了？还是他们的家园对他们来说，太小了？西双版纳目前的生态系统，到底能够满足多少头象的需要？带着这个问题，我曾经请教过专家，西双版纳目前的生态承载力是否能满足大象种群的需求？他说生态承载力在不同的环境是不一样的，在南亚稀树干旱草原中</span><span style="font-size: 19px;line-height: 28px">,</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">大象的密度是头</span><span style="font-size: 19px;line-height: 28px">/</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">每平方公里，在极端情况可以到达</span><span style="font-size: 19px;line-height: 28px">2-3</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">头</span><span style="font-size: 19px;line-height: 28px">/</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">每平方公里，所以亚洲象最大的种群在印度。而在热带雨林中，这个数字降为</span><span style="font-size: 19px;line-height: 28px">0.1</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">头</span><span style="font-size: 19px;line-height: 28px">/</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">每平方公里。这些年自然保护区内，严禁砍伐，树木得到了良好的保护，长草的地方自然就少了，大象的食物也就随之变少了。在西双版纳大象经常走出森林，骚扰周边的群众，人象冲突的事情时有发生，造成人员伤也屡见不鲜。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px">2002</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">到</span><span style="font-size: 19px;line-height: 28px">2007</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">年我做一届全国人大代表，所提的第一件议案就是“修改野生动物保护”，建议修改野生动物肇事补偿的相关事宜。在议案的调查过程中，有些当地保护区的干部曾经提议，在大象生长的保护区内，适当砍伐一些树木，种植一些大象喜食的禾本科植物，把大象留在森林中。如今野生动物肇事补偿得到了部分解决，但是在保护区内种草的事情，至今仍在论证中。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">生态承载力其实也是一个动态的，西双版纳能够容纳下多少大象，很大程度上是取决人们的意愿。如果我们想为更多的大象提供家园和食物，不妨退出一些村庄和农田，把人的栖息地变成大象的栖息地。大象是保护生物学的旗舰种，也生态系统中的边缘物种（</span><span style="font-size: 19px;line-height: 28px">edge species</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">），保护大象一大批与之相关的动物和生态环境会跟着受益。更重要的是通过对大象的保护，我们学会如何和野生动物们共享这个地球。人类过去捕杀过太多的大象和其它大型动物，如今人类为保护这些动物做出一些物质上的牺牲和让步是应该的。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">大象能够长途跋涉走到昆明，个中缘由我不知道，我想有一点是可以肯定的，就是人们的保护意识在增强。大象一路上得到了妥善的照顾，沿途没有人去伤害大象，不仅如此，还有人专门投送食物，生怕怠慢了大象。目前虽然目前对如何劝返大象，仍无有效的方案，但是至少是否定了用麻醉枪等暴力的手段，这不能不说是生态文明的理论深入人心的一种体现。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">除了北上的大象外，这几天还有一群南下的大象闯入了西双版纳热带植物园的东区，这可是一个国家级的</span><span style="font-size: 19px;line-height: 28px">5A</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">级。这</span><span style="font-size: 19px;line-height: 28px">17</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">头大象，刚到植物园东区的时候，是试图从东区渡过罗梭江到其它地方，后来罗梭江江水上涨，渡江困难后，大象就在植物园顺势躺倒。为了监控这群大象，避免人象冲突（大象距离植物园新的</span><span style="font-size: 19px;line-height: 28px">3H</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">小区，不过</span><span style="font-size: 19px;line-height: 28px">2-3</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">公里的路程），园里组织自愿者飞无人机，</span><span style="font-size: 19px;line-height: 28px">24</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">小时监控这群大象。监控发现，这群大象的生活非常规律，有固定睡觉的地方、觅食也不费劲，周边就有一大片包谷地和杂草、林中还有一处池塘可供玩游泳和玩耍，真是一片理想的栖息地。</span><span style="font-size: 19px;line-height: 28px"> </span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">“大象何时离开植物园”成了大家关心的问题。以人度象，我为什么要离开，这是多好的家呀。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">大象带来了新闻热点，给广大吃瓜群众带来的欢乐，但是却给当地政府带来麻烦，工作人员要</span><span style="font-size: 19px;line-height: 28px">24</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">小时监控大象，确保人象每天平安。</span></p><p style="margin: 0 0 13px;text-align: justify;line-height: 21px;font-size: 14px;font-family: Calibri, sans-serif;white-space: normal;text-indent: 37px"><span style="font-size: 19px;line-height: 28px;font-family: 宋体">如何接待这群突如其来的客人，我们有点手足无措。对大象的最适合栖息地、生态适应范围，生态承载力，甚至是习性和食性，我们都知之甚少，更不用说：大象为什么北上，如何劝返大象，大象行为，大象心理等这些有点烧脑的问题了。这一切都说明，我们关于大象的研究是多么的薄弱。书到用时方恨少</span><span style="font-size: 19px;line-height: 28px">,</span><span style="font-size: 19px;line-height: 28px;font-family: 宋体">事非经过不知难，如果我们不去认真地去研究我们周边的动物和植物，加强生物学和生态学的基础研究，“共建地球生命共同体”就会沦为一句空话。</span></p><p style="text-align:center"><span
