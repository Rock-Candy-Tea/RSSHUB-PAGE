<p><strong><span style="font-family: 楷体, 楷体_GB2312, SimKai; font-size: 18px; color: #1F497D;">亲爱的女儿：</span></strong></p><p><strong><span style="font-family: 楷体, 楷体_GB2312, SimKai; font-size: 18px; color: #1F497D;">　　见字如面。</span></strong></p><p><strong><span style="font-family: 楷体, 楷体_GB2312, SimKai; font-size: 18px; color: #1F497D;">　　下个月你就要远赴海南岛参加学校组织的环岛骑行活动，不知你是内心充满期待，还是有点畏难情绪，我想可能二者都有。想象一下海南岛的碧水蓝天、椰林轻风、金滩海浪，从寒冷的北方来到鲜花绽放的热带，思绪像云彩一样轻灵。你还记得在你五岁时咱们一家三口去三亚度假吗？你整日泡在宾馆的泳池里，奠定了你的游泳基础。现在时隔十年，你又要去海南岛，日行数十公里，想想都累，但在温暖的海南，这个任务不算艰巨，通过这次环岛骑行，将极大增强你的意志力。意志力是未来事业成功的首要因素。</span></strong></p><p><strong><span style="font-family: 楷体, 楷体_GB2312, SimKai; font-size: 18px; color: #1F497D;">　　且不去翻故纸堆里先贤前哲的励志故事，说说现实中的成功人物。今年的奥运射击冠军杨倩，生在宁波一个普通家庭，自称是个佛系女孩，但是她的意志力绝非佛系孩子可以比拟，职业体育的训练强度是你们想象不到的，杨倩把意志力贯穿到了日常生活中，事事自律，样样坚持，其实意志力不是什么需要特殊对待的事情，只需在你想要放弃的时候再咬牙坚持一下。</span></strong></p><p><strong><span style="font-family: 楷体, 楷体_GB2312, SimKai; font-size: 18px; color: #1F497D;">　　你在初中时就阅读了《红岩》，看看那些坚持到最后的英雄和那些在胜利前夜放弃理想叛变革命的人，意志力决定了名垂青史还是遗臭万年！和平年代说事业。任正非45岁创业，创立华为这个让中国人骄傲的公司，在他的创业史中历经劫难，短短数十年让华为成为让美国政府担惊受怕的企业，在美国对中国高技术企业的持续制裁中，有的公司选择屈服投降，华为挺起了脊梁，企业的突破能力往往是由企业家的意志力决定。做一个意志坚强的人，不仅是自己的事，也是对自己的团队负责。</span></strong></p><p><strong><span style="font-family: 楷体, 楷体_GB2312, SimKai; font-size: 18px; color: #1F497D;">　　爸爸希望你做一个意志坚强的人，不只是希望，而是你的基因里具备了强大的意志力。奶奶从20多岁起就历经几次大病，却坚强地开心地活着，不仅度过了艰难岁月，还看到了国家富强、家庭幸福的新世纪，看到了你的茁壮成长，看到了爸爸的事业蒸蒸日上。爸爸在东海之滨创业，远离亲人，动力是什么？填补国内技术空白实现产业报国的理想是其一，让家人生活得更好是其二。为了保持身体健康，爸爸每天中午打网球一小时从不间断，不管酷暑的夏日还是寒雨的冬日，意志帮助我挺过了一次次危机。</span></strong></p><p><strong><span style="font-family: 楷体, 楷体_GB2312, SimKai; font-size: 18px; color: #1F497D;">　　人是社会动物，成功离不开团队的配合。你的骑行活动也是一个团队活动，大家要在骑行中互相关爱、互相帮助，彼此增进友谊，现在的好同学就是一生的好朋友，这是人生的财富。</span></strong></p><p><strong><span style="font-family: 楷体, 楷体_GB2312, SimKai; font-size: 18px; color: #1F497D;">　　第一次离开爸妈去远行，这是你独立人生的预演，将来还要一个人去异国他乡留学，准备好你的独立和意志，让未来之路霞光漫天，你能行。</span></strong></p><p><strong><span style="font-family: 楷体, 楷体_GB2312, SimKai; font-size: 18px; color: #1F497D;">　　爱你的爸爸妈妈。</span></strong></p><p><br></p>                    <br><br>
