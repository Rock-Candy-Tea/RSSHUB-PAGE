11月5日消息，苹果推出重新设计的Apple ID网站，该 Apple ID 登陆页面更换一个新的圆点和苹果标志设计，同时还提供了 Apple ID 网站可以用来做什么的信息。之前的设计是以图形为主，有一个大横幅，上面有使用各种苹果设备的人。登录 Apple ID 网站后，所有可用的 Apple ID 管理选项都以卡片的形式呈现，此外，左边还有导航选项。<br><a href="https://www.niaogebiji.com/pc/bulletin/detail?id=11520">原文链接</a>
