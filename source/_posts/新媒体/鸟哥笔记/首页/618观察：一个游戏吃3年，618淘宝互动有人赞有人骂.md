<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/618" target="_blank">618</a>理想列车互动游戏今年又一次登上淘宝平台，让人意外的是这款游戏几乎3年不变，每年都以同一面目热热闹闹在618期间登台，然后在618后悄然消失。对于淘宝的这一举动，有人表示理解，认为不变是一种策略；有人质疑游戏不思变革创新。互动游戏为的是淘宝和网友之间的有效沟通，但理想列车这种做法能达到预期的效果吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/122351370962b395998fd256.84383839.jpeg" title="618观察：一个游戏吃3年，618淘宝互动有人赞有人骂" alt="618观察：一个游戏吃3年，618淘宝互动有人赞有人骂" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>作为一款连续三年没有玩法变化的游戏，618理想列车到底是怎样一款游戏呢？</strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和往年一样，淘宝的这趟618理想列车搭载了价值10亿元的红包和礼物，从5月29日0时到6月20日22时，所有用户都有机会搭乘“瓜分10亿的618列车”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>理想列车的具体互动玩法也非常简单，用户在淘宝和天猫app端登录后，搜索“618列车”就可进入618理想列车活动页面，通过喵币升级列车，在列车达到相应的等级时，可以有机会兑换红包或特斯拉豪车大奖。</span><span></span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>理想列车的第一步需要用户通过浏览商家店铺、互动分享、观看</span><span>618</span><span>理想狂欢季、签到、查看指定页商品、围观礼盒等等一系列任务获取喵币。</span><span>获取喵币后</span><span>用户就可以购买车厢。</span></span></span><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span></span><span>用户在开始时购买的车厢都是最低级的，所以可以拖动相同等级的两个车厢可以在</span><span>“我的列车”界面进行合并升级，从而获得级别更高的车厢。</span></span></span><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>以此类推，类似我们所玩的“</span><span>2048</span><span>”游戏那样，车厢的等级一级一级往上叠加，然后可以解锁更高级的车厢，在到达每个指定等级的车厢后，就可以领取活动红包等奖励。</span></span></span><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>例如在</span><span>6</span><span>月</span><span>1</span><span>日</span><span>24</span><span>点前达到</span><span>15</span><span>级有机会赢</span><span>618</span><span>元大红包。</span><span>当列车在</span><span>6</span><span>月</span><span>16</span><span>日</span><span>20</span><span>点</span><span>30</span><span>分</span><span>-23</span><span>点升级到最高的</span><span>58</span><span>级车厢时，就有机会赢取特斯拉汽车的大奖。</span></span></span><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/93551929662b3959da2c6e9.04286826.jpeg" title="618观察：一个游戏吃3年，618淘宝互动有人赞有人骂" alt="618观察：一个游戏吃3年，618淘宝互动有人赞有人骂" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自2020年起至今，618理想列车互动游戏每年上线淘宝平台。3年来除了游戏上显示的年份在变，其他一成不变。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先是红包金额相同，每年都是固定的10亿红包。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次是游戏玩法相同，每年的理想列车都是通过升级解锁车厢，从而获取红包。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后是游戏所希望达到的目的也是一样的，都是希望用户通过游戏中的任务去浏览商家店铺，为商家导入流量，同时也希望用户在完成任务将游戏分享后起到拉新的作用。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而这款游戏的问题也一直存在。有网友吐槽，游戏美其名曰可以瓜分10亿红包，然而兑奖期间格外低调，信息未能及时透明。在一些社交平台上，网友反映现在的平台互动游戏一般都在活动结束前提醒用户未兑换红包数量、金额、兑奖截止日期等，但不少网友没有收到提醒，很多网友最后都是竹篮打水一场空，未能获得预期中的红包。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，这款游戏居然还有外挂，正常情况下通过做浏览任务获得喵币的话，每个页面要15秒，加上点击、退出、下拉的时间，完成所有页面，就需要将近6分钟时间。在一些网站上能得到</span><span>618列车js脚本</span><span>，再配合 Auto.js 软件</span><span>自动领喵币</span><span>，可以轻松获得60万的喵币。这个脚本可以自动签到、自动浏览广告、逛淘宝人生、去支付宝领喵币、到天猫农场收集，一般1分钟以内即可完成所有任务。</span><span><span></span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>对618理想列车的这种做法，各方反响不一。</strong></span></p>
<section>
<section><strong>每年一款反响不好？</strong></section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝双十一游戏，从2018年的“集能量”、到2019年的“盖大楼”，再到2020年的“云养猫”以及2021年的“喵糖”，几乎淘宝每一年的双11小游戏都被消费者一边玩一边骂，因为玩法改动大，玩家不适应。所以618游戏不变也是一种策略。</span></p>
<section>
<section><strong>上瘾机制是一种留量</strong></section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对游戏上瘾其实是一种游戏习惯的培养，连续</span><span>3</span><span>年同款游戏也是在加深这种习惯的养成。</span><span>上瘾机制对于一款游戏来说很重要，如果没有上瘾模型机制让任务游戏化，即使有红包的奖励，但每天重复做相同枯燥的任务，也会让用户感到疲惫和无趣。</span><span>在理</span><span>想</span><span>列车游戏中则是用了</span><span>“升级</span><span>+</span><span>及时奖励”的机制，用户将列车升级后就能有机会获得红包奖励，这种实时回报所产生的及时满足让用户越玩越上瘾。</span></p>
<section>
<section><strong>惧怕创新是“原罪”</strong></section>
</section>
<section><span>互动游戏其实本质上是商业产品，容错率低。</span><span>它肩负的责任是巨大的，一旦因为冒然的创新做出改变而出现错误，会影响整个商业系统的运作，为了降低试错成本，最后可能还是选择遵照上一年的模式，不改变游戏机制。</span><span></span></section>
<section><span></span><span></span><span></span></section>
<section>
<p><strong><strong><strong>游戏开发时间也是一大难题</strong></strong></strong></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span></span><span>对于电商平台来说，每年除了</span><span>6</span><span>18</span><span>以外，还有双</span><span>1</span><span>1</span><span>、双</span><span>1</span><span>2</span><span>等诸多节点，这些节点紧凑地遍布了一整年，想要在每个节点都推出一款全新的、创意十足的、设计合理的游戏，在开发时间上是不允许的，也是几乎不能完成的。</span></span></span><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>或许在未来互动游戏会出现一套全新的机制来摆脱现在的固有模式，明年“理想列车”会不会连续第四年出现在淘宝平台呢，让我们拭目以待吧！</span></span></p>
