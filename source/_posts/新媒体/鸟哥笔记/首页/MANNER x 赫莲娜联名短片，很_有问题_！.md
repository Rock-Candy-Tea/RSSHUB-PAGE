<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>什么是青春？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人说：是信仰是心中有梦、是脚下生风是眼里有光、是热泪盈眶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有<span>人</span>说：是漫长人生的开头好的开始。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>还</span>有人说：青春太好了，好到你无论怎么过都觉得浪费”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>如果说“浪费<span>”</span>是青春的底色，那么我今天还想说，“有问题<span>”</span>也是青春的一种常态。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>随着<span>这届品牌</span>对年轻化营销的重视，它们也在开始解决年轻人的内心“问题”。比如知乎的</span><span>《有问题，就会有答案》，百度的《搜索2020》，无一不是在围绕年轻人的问题做文章。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过面对泛滥的青春问题营销，如何挖掘出新的营销立意，无疑也是品牌的一个问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>如果你看腻了关于问题的营销审美，那么最近<a href="https://www.niaogebiji.com/tag/manner" target="_blank">manner</a>联合<span>美护肤品牌<a href="https://www.niaogebiji.com/tag/%E8%B5%AB%E8%8E%B2%E5%A8%9C" target="_blank">赫莲娜</a>绿宝瓶跨界，</span>会给你带来耳目一新的东西，双方抛出了新颖的概念——</span><span><strong>一夜青回</strong></span><span>。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><img src="https://ngbjimg.xy599.com/723115795623260770c18c4.52658385.jpeg" referrerpolicy="no-referrer"></span></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>深耕“青回”文化母体，</strong></span><span><strong>降低品牌的沟通成本</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“一夜青回”<span>顾名思义，</span>就是一夜回到青年的年轻状态。深挖这个概念之后，就会发现别有深意，品牌融入了“母体文化”的巧思。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如华与华方法所说的，找文化母体就是要找大众最熟悉的那个符号，熟悉这个母体的人越多，母体的传播原力就越强大。这一点在</span><span>Manner咖啡上也有所体现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>“青回”这个概念，对于大多数年轻人而言是熟悉的，它就像是b站弹幕“爷青回”、“爷结青”的衍生词汇，表达的就是：</span><span><strong>与青春久别重逢的喜悦，保持永葆青春的生活态度。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/145919842462be2d8661b930.41947074.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很显然，MANNER咖啡结合咖啡和年轻的内涵，打造“一杯青回<span>”</span>的概念，把品牌寄生在年轻人熟悉的文化母体之中。让人看了不自觉地产生熟悉感，激发年轻人脑海中对于年轻文化母体的潜意识，进一步他们唤醒对于“爷青回”的场景联想，让<span>MANNER</span>降低品牌的传播成本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当我们理解了这个概念之后，也就能更加透彻明白，在这波营销中，它所诞生的语境——</span><span><strong>酝酿有关青春的情绪，分享奇妙有趣的故事，塑造起助力用户解决青春困惑的形象。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205789392362326078647a25.53043458.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>一支“有问题”的短片 ，</strong></span><span><strong>一次“没关系”的青春 </strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不可否认，要<span>想</span>把品牌走近年轻圈层的内心，必须先精准拿捏他们的亚文化，才能引起年轻人对于品牌的价值共鸣。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以我们看到</span><span><strong>MANNER联合赫莲娜推出一支全是“问题”的短片，看似毫无剧情可言，甚至有些无厘头，却成功敲中年轻人的真实内心戏。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何变成一只猫？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人为什么会胖？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>工作的意义是什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>婚姻是一种交换吗…</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短片跳出了品牌的惯有思维，而是站在年轻人os的立场出发，让主角全程放空遐想，把我们对于学习、工作、恋爱、结婚等的困惑，展现得淋漓尽致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/140222882362be2d8691e926.76704675.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/110183090162be2d8737a397.95786377.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在铺垫层层问题的基础上，MANNER基于先抑后扬的手法，</span><span><strong>从“有问题”到“有问题，没关系”营造反转的画风。</strong></span><span>这样的设定既给片子攒足了吸引力，也激发我们对于自我问题的代入感，打从心底对于“有问题才是青春”产生认同感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/75693436462be2d87d247f9.75708206.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在广告画面处理上，MANNER也颇有巧思，细节之处加入绿色的元素——绿色的汽车、绿色的草丛、绿色的杯子等，营造<span>了</span>沉浸式的清新场景，搭配俏皮的人物演绎，增加整体广告的动感和活力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/175659564462be2d880ae6a0.44758189.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>MANNER通过构建青春洋溢的场景，<span>点题“一夜青回”的营销主题，</span></span><span><strong>制造年轻人对于青春生活的向往感，让我们切身地感受到回到青春种种美好。对于用户来说，也会不自觉把这般好感，转移到MANNER品牌好感上。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短片之所以站得住脚跟，离不开品牌对年轻人青春现象的洞察，<span>视频无疑是当下年轻人的真实生活写照：</span></span><span><strong>青春就是这样，是一道无解的命题，年轻人的脑袋时刻充满奇思妙想，冒出奇奇怪怪的问题，</strong></span><span><strong><span><span>但</span><span>有些问题也找不到答案。</span></span></strong></span><span>正是这样，“一夜青回”的概念才应运而生。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/134948845962326079426029.07408166.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>构建多元化的触点，</strong></span><span><strong>让青回成触手可及 </strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>诚然，<span>告知有这样一个概念视频，或许大众是没什么感觉的，</span>营销讲究的是落地为王，只有能落的了地的互动，才能将“一夜青回”升华为情感的鼓励，内化为消费者生活方式的日常。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此在产品联动方面，<span>MANNER x 赫莲娜</span>推出</span><span><strong><a href="https://www.niaogebiji.com/tag/%E8%81%94%E5%90%8D" target="_blank">联名</a>饮品</strong><strong>「一杯青回」——抹茶冰淇淋风味拿铁</strong>，</span><span>伴随淡淡茶香，有如冰淇淋般丝滑的特点。其灵动冰爽的产品特色，生动诠释青春的模样。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/169287798062be2d8987cf39.07870082.jpeg" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>MANNER</span>将“一夜青回”</span><span>具象化为抹茶冰激凌，</span><span><strong>开启品牌与用户之间的情感价值触点，</strong><strong>为年轻人提供任意天马行空的机会，进而在物质上确确实实给用户触手可及的“青回”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，MANNER在全国7个城市10家Manner，推出联名主题快闪店，<span>以绿色元素设计</span>门店装饰、咖啡杯等，配合推出“穿绿免单<span>”</span>、<span>“</span>宝物派送<span>”</span>、<span>“</span>抽奖得绿宝瓶和吨吨杯<span>”</span>活动，把青春的氛围感拉满；同时在微博阵地发起</span><span><strong>#赫莲娜请你喝manner#</strong></span><span>等相关话题，预埋品牌谈资的传播因子，赋予“一夜青回”话题更多社交属性。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/37644643562be2d89c3f160.52290773.jpeg" referrerpolicy="no-referrer"><img src="https://ngbjimg.xy599.com/4078356362be2d8a346291.77019957.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如我们所见，不少网友纷纷线下打卡MANNER，线上晒起“一夜青回”的联名，形成一波一波UGC的传播裂变，持续为联名注入年轻力，带给人们更深层次的情感触动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/102558881962be2d8a8a18f6.98036961.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134708537062be2d8b4d73a5.29541688.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/138833517962326079dd1f53.73402961.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>先锋的品牌共性，</span></strong></span><span><strong><span>契合的跨界合作 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌的跨界讲究营销上的眼观六路、耳听八方，考察的是品牌之间形象和圈层的契合度。回过头来看审视<span>Manner咖啡和</span>HR赫莲娜的联动，也印证了这一点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>赫莲娜作为创立于1902年的百年先锋品牌，</span></strong></span><span>得益于创始人赫莲娜夫人的极致先锋精神，<span>赫莲娜</span>不断探索护肤功效的边界，以精锐的产品与敢于颠覆的营销，给用户创造<span>持续的</span>惊喜。</span><span><strong><span>而Manner也是国内咖啡赛道的黑马，</span></strong></span><span>从创立至今的短短6年的时间，已然成超百亿元估值的独角兽品牌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/10895375362be2d8c0334d7.87862714.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两大品牌对于先锋、创新的共同向往，</span><span><strong>奠定了跨界并不只是一次商业意义上的生意，更是一种惺惺相惜的品牌创意，也正因为此，这场跨界有了一种自带“年轻”滤镜的氛围感。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>MANNER x 赫莲娜深入挖掘共通的价值契合点，从<span>产品到营销共建</span>青春话题的营销内容，以绿宝瓶精华唤醒脸蛋，以咖啡香气唤醒脑袋，有效打破美妆和咖啡之间的边界，打通双方用户圈层的壁垒，触达更多咖啡、美妆场景受众，给用户营造惊喜的消费体验和服务。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，这给咖啡美妆的跨界打了一个样本。</span></p>
