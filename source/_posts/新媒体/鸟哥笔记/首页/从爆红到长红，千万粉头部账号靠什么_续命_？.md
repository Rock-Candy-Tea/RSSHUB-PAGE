<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人还记得2018年最火的博主是谁吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于创作者来说，尤其是短视频时代的红人，现在很难依靠一个爆款作品长久地留在大众视野中。很多人认为网红有生命周期，没有人可以永远处于顶流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新抖数据，截至6月30日，568个千万粉抖音账号中（除去政务媒体类账号的不完全统计），6月保持涨粉的有278个，占比48.9%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然像“垫底辣孩”这样的新人不断涌现，但从数据上看，仍有更新了3、4年的头部达人保持增长态势，比如“疯狂小杨哥”“祝晓晗”“程十安an”“小橙子”“痞幼”“MR-白冰”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么这些熟悉的名字可以在激烈的竞争中屹立不倒？博主要如何进行自我迭代和内容创新？我们采访了各大MCN机构和内容公司，看看他们有哪些“网红长寿秘诀”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>内容瓶颈期？试试换团队</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当粉丝数升至百万级、千万级，头部博主也不可避免会遇到内容瓶颈期，这是所有创作者都要面临的问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，短视频平台推出了诸多扶持新人创作者的奖励政策，算法推荐机制也在一定程度上会分散头部博主的流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据观察，抖音的算法导致视频内容的生命周期极短，必须更新才能突破瓶颈，但博主背后的内容生产方，包括运营、编导在内的团队是有固定的知识结构的，创新并不是一件易事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有的机构选择定期做员工培训，效果如何则因人而异。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>神狼文化负责人阿诺曾表示，编导是一个需要发散思维的工作，很难培训，可以说“三分靠天赋，三分靠努力，三分靠阅历，还有一分靠运气”。如果没有天赋，怎么培养也不行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/88363801362d3a7f775f822.09420263.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>神狼文化旗下达人“莫邪”“田小野”</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>部分机构突破内容瓶颈的做法显得更加激进，他们会定期刷新团队，用更换团队的方式更新内容。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以某千万粉头部达人为例，内容定位从舞蹈转向变装，粉丝数突破千万后，又在视频中加入了剧情，最近一条热门视频仍能达到200多万点赞。这些内容成功转型的背后其实是创作团队经历了多次大换血，残酷而有效，但对公司的管理能力提出了极高的要求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无独有偶，OST传媒调整内容的方式也是换团队。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据介绍，一般一个头部账号OST会提供5-8个团队人员，每过一段时间，比如从0到50万，50万到100万，平均几十万粉丝上涨都伴随着内容瓶颈出现，需要对内容进行升级。“没有一个项目没有调整过，基本上就是换团队，有些账号一年可能要换个五六次团队”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>达人不红了？随时代方向转型</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>团队的力量可以助力达人穿越迷雾，飞得更远，但究竟能飞得多高，可能更重要的在于达人本身。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此前无忧传媒总裁李琳接受36氪采访时提到，平台已经有很多红了3-4年的初代达人，她并不赞同“网红生命周期只有5-7个月”的说法，“不是网红红不下去，而是很多人做一段时间自己就不做了”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小红书头部MCN机构摘星阁的创始人侃侃告诉我们，这个行业平均两三年是个周期，头部会长一点。<strong>很多博主2年就不火了，有些人5年了还在，她认为</strong><strong>这与账号本身没关系，而是与这个人有没有跟上时代有关系。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我2017年做摄影博主时认识的那帮博主，他们当时火是因为发的照片是ins风格，很高级，但是这种图片现在已经不流行了，所以会有被时代淘汰的情况。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/172546535562d3a7f7ad4de4.82350004.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span> 小红书摄影类热门话题</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>除了流行趋势更迭，内容同质化也会导致观众审美疲劳。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“多余和毛毛姐”曾凭借有趣的性别反串吸引了3450万抖音粉丝，一头橙色假发和夹杂方言的“贵普”成为他鲜明的标志。然而这个赛道涌入了许多竞争者，比如“毛光光”“郑丽芬er”“董代表”，在类似内容的反复刺激下，观众逐渐习惯了男扮女的喜剧效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两年间，“多余和毛毛姐”的粉丝数掉到了3000万以下，点赞、评论和转发的互动数据也有所下降，最近他在视频中表示接受掉粉的现实，“有时越想抓住的东西越是抓不住”，接下来他会去尝试新的内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16301672362d3a7f832eda3.04185135.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在内容爆炸式增长的背景下，博主如何才能找到新的方向？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>仙梓文化创始人张柔琪透露，要根据平台目前热门的内容趋势，以及近期表现比较好的账号，进行数据分析和经验总结，从中可以看出未来可能的迭代方向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>具体操作上，仙梓文化倾向于<strong>打造系列感的内容</strong>，如果一套内容效果不错就继续深挖，感觉流量有点瓶颈了就再开拓新的系列。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如旗下头部博主“ImAnnaNana”从2018年起发布美妆类内容，去年她的“沉浸式护肤”系列视频，在B站、小红书、抖音等平台得到了大量关注，今年这一套内容虽然数据尚可，但是账号涨粉的速度变慢了，所以最近一个月团队已启动新的内容系列。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span data-docs-delta="[[20,&#123;" gallery-block":"https: uploader.shimo.im f l0wc9z08qlc9hpya.jpg!thumbnail"&#125;,"24:\"t1pv\"|29:1|30:0|3:\"998\"|4:\"auto\"|7:1|crop:\"\"|frame:\"none\"|ori-height:\"1080\"|ori-width:\"998\""]]" data-copy-origin="https://shimo.im"><img src="https://ngbjimg.xy599.com/200768448662d3a7f866a3d6.90290485.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>类似的，博主“TJandClaire”通过<span>情侣</span>系列视频在小红书半年涨粉近60万，而目前<span>新</span><span>系列</span><span>“</span><span>老外学中文”</span>也正在爆发式增长中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说每一个达人都有自己的事业天花板，那么在阿诺看来，这个上限基本就取决于达人本身，也许前期可以靠内容涨粉，但后期就要看达人本身的实力，包括他的各种积淀，甚至于观众缘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像2021年抖音热度最高的探店博主“MR-白冰”，此前他是一名汽车博主，直到2021年8月才开始转型做美食探店内容，正赶上抖音扶持本地生活的风口。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>明明是开着跑车的“高富帅”，他却把顶级料理吃出路边摊的感觉，这种幽默反差让他一年吸粉1675万，目前粉丝数超2583万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137381332462d3a7f8aac417.17116780.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以说，随着市场快速变化，博主需要与时俱进，探索适合自己的新方向，用独特的魅力掀起新的潮流，从而坐稳头部地位。</span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>日常焦虑？保持空杯心态</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内容行业的从业者，尤其是短视频领域的，从博主到机构，多多少少都有种焦虑感，焦虑流量会下滑，焦虑内容出问题，焦虑收入不稳定……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有机构告诉我们，做抖音上的账号，三个月之后慢慢都会有同样的焦虑，<strong>需要一直保持空杯心态，不断学习。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在日常工作中，<strong>复盘是保持输出优质内容的关键动作之一。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>麦芽传媒品牌营销总监冉旭表示，大家每周都会开会复盘，根据账号爆款来做分析，一起商讨怎么做内容升级。一般来说麦芽传媒不会突然对内容做出很大的改动，成熟账号也很少更换编导，基本是渐进式的内容升级。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/140602674862d3a7f9100490.99179799.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>麦芽传媒旗下达人“丁公子”</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>张柔琪也表示，应对流量焦虑，最直接的办法就是不停地创作新的内容，并且提高内容质量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“今年我们公司用TVC的团队去帮一批博主拍摄日常视频和广告，本身他们内容质感就比较好，这样做的话质感会进一步提升，高于这个平台大多数的创作者。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样看来，内容创作的确是越来越“卷”了，需要投入更多的成本，但仙梓文化有博主升级后上个月接了10条广告商单，证明了这种做法非常奏效。张柔琪认为，在平台方，品牌未来与博主合作进行品宣，可能买的就是这条内容的调性，不管是1000赞还是100赞，如果内容质量够高的话，品牌是愿意为此买单的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如OST传媒创始人袁鹏所说，上一个时代，得益于行业红利，成为头部主播有很多的偶然性。这个时代，面临更加激烈的行业竞争，没有那么多偶然性，需要你对一件事情不懈地付出，坚持打磨内容，坚持长期主义。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比爆红，从业者需要更多思考和追求如何长红。</span></p>
