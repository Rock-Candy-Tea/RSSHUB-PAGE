<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去，我们说<a href="https://www.niaogebiji.com/tag/%E5%B9%BF%E5%91%8A%E6%8A%95%E6%94%BE" target="_blank">广告投放</a>就是：<strong>“找对人，说对话，出对价”</strong>。互联网广告需要找到优质的流量平台，进行内容投放；在投放时，把控合适的出价，利用优秀的内容素材和人群包（计划）定向，不断制定更多曝光/转化的广告策略。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>广告投放一直是门【概率学】。在广告曝光给用户的过程中，触达用户的情景和用户当前的状态都是千变万化的，高度的不确定性，让你不能保证每条广告计划的准确度，甚至能不能起量都难说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，既然是一门【学问】，在优化提升上就有迹可循。笔者认为【素材即定向】是<a href="https://www.niaogebiji.com/tag/%E4%B9%B0%E9%87%8F%E4%BC%98%E5%8C%96" target="_blank">买量优化</a>下半场的趋势。广告投放需要更多高质量的素材，广告平台才会给予更多的有效曝光，如此便有机会获得更多优质的真实线索。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，本文将以【巨量引擎</span><span>效果广告</span><span>】为例，聊聊广告优化实操中的一些事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/196401358162c4e93080bea6.96706294.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>广告优化的趋势</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>【广告定向】一直是广告优化的核心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提起【广告定向】这个词，很多人会不经意间联想到人群包定向和兴趣行为定向。早期的广告优化主旋律还是以这两个方向为主。优化师研究用户的行为，圈定用户所在人群和兴趣行为的特征，然后加大投放拉量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是近年来，国内的各大媒体广告平台出现了一些调整【自定义人群/兴趣行为】定向功能的倾向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>笔者认为，这是一个趋势。随着运算技术的升级，投放平台将步入新的阶段，【人群包】定向功能将逐步下线，取而代之的是基于运算的【系统推荐】。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>平台运算能力和功能的不断优化、升级，也帮助了优化师从那些重复性的繁琐工作中脱离出来。这是进步，也是宿命。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，现在的广告优化师应该如何做【定向】呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何让投流更精准、更吸引人呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/137493720762c4e930a41328.75259685.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>素材即定向</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>【素材即定向】</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上面便是答案，去优化【视频素材】达到以往【人群包定位】的目的。究其底层逻辑，就是使用更多优质的素材，找到对的素材方向，提高点击转化率，给模型计划提供更多正向样本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如内容营销中的【内容为王】概念一样，具备优秀的素材成为了创建易跑量计划的关键之一。在广告创建阶段，除了要保证关键的素材创意与推广产品受众偏好一致，还需要考虑以下3个维度，以便评估计划内素材的整体质量是否达标：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>①素材数量</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>需提升视频、图文、标题、落地页等素材组成要素的数量，可以根据投放产品的受众人群/行业不同，制作不同类型的素材。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>②素材类型</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>需提供丰富的、并且符合规范的素材类型，例如：竖版视频、横板视频、横板大图等。丰富的素材类型，会带来更多的优质点击。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>③素材品质</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>指的是上传的素材在同类型账户中的挤压程度，如提供挤压程度高的素材会影响跑量效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>何为挤压程度高？例如，在巨量引擎效果广告投放时，一条视频素材被同一账号的在投计划多次重复使用的话，便会出现素材挤压的问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过优化【素材】内容的方式，我们就能解决这个问题。在制作脚本时，可以考虑下推广的产品针对什么样的人群，他们在什么时间、什么原因以及什么地点下，会存在使用产品的需求或需要产品的某一功能。不同的人群、不同的情景、不同的需求、不同的视频内容类型，这些变量具备无限的素材创造可能性。<strong>真正优秀的素材都经得住投放的考验。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>【</strong><strong>巨量引擎</strong><strong>效果广告</strong><strong>】平台</strong><strong>会判断广告主的素材是否</strong><strong>易起量</strong><strong>，对于那些投放后无法获得很好的跑量和转化效果的素材，反复新建计划投放反而无益于效果的提升。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>就在近期，</strong><strong>【巨量引擎效果广告】</strong><strong>进行了比较大的版本更新。在这次更新中，上线了</strong><strong>“</strong><strong>叉</strong><strong>乘搭配</strong><strong>”</strong><strong>这一新的投放逻辑</strong><strong>。</strong><strong>笔者认为</strong><strong>它</strong><strong>的上线说明了巨量引擎效果广告平台希望广告主投放时，能选用更多优质的素材。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以往广告主上传多个创意素材创建计划后，系统会对这些创意进行离线预估，择优选择其中最有可能跑量的1个素材进行投放，而其余素材只能由平台逐步探索投放。但是，谁希望自己辛苦制作的素材没用上呢？多条计划加入同一素材，又会担心创意挤压。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而本次巨量引擎效果广告产品能力升级后，广告主上传多个创意素材，系统会对素材进行叉乘搭配，让所有的素材同时在线投放，让素材可以得到<strong>最大化</strong>探索。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在笔者管理投放运营团队的时候，一边担心着账号内出现素材创意挤压，一边时常看着辛苦制作的素材不被平台推荐而难受。而且一条条建立计划的话，又耗时费力。所以在我看来，【巨量引擎效果广告】这次更新的【叉乘搭配】逻辑的好处有两个：</span></p>
<ol start="1">
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提升广告效果：素材可以得到充分探索，增强广告计划的跑量能力，延长广告计划的生命周期。</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提高团队执行效率：减轻了优化部门的压力，不再需要人工搭配素材。例如比较常见的，1条视频素材下，搭配多个标题，然后疯狂地新建广告。</span></p>
</li>
</ol>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从【巨量引擎效果广告】的变化，我们不难发现素材的权重不断提升，正如开篇所提的人群包定向开始由机器运算承担。深入分析原因，要记得<strong>互联网广告依托于流量</strong>，那么【巨量引擎效果广告】是依托什么平台的流量呢？如下图：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/181162837662e34449d4d819.84339773.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>（巨量引擎广告投放后台-计划创建-投放位置）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些平台大多为文娱内容平台、信息流内容平台以及垂类需求服务平台。平台具备海量的用户流量，设想下如果这些平台充斥着低质量的素材，会造成多少用户流失？还会有多少用户活跃？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这么一分析，【巨量引擎效果广告】重视广告主素材质量的原因也清晰明了了。低质量的素材不仅仅对于广告账户和计划优化毫无帮助，还损害了流量平台的用户体验。对于平台来说，用优质的内容建设良好的环境、社区和用户生态，让用户更加信赖平台生态内的品牌和产品表达，形成正向的循环，而不是负向的消耗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>手心手背都是肉，既然不能改变终端使用用户的观看习惯的话，平台当然愿意给与那些上传优秀创意素材的广告主更多的流量权重。所以，广告主们应当顺应大势所趋，需要更加关注<strong>素材的重要性</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>【素材即定向】的成型逻辑也正是如此，绝对别低估素材的重要性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/65722941362d654261ee544.33731826.png" referrerpolicy="no-referrer"></strong></span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>如何快速建</strong><strong>立好</strong><strong>易起量</strong><strong>的计划</strong></span></h2>
<ol start="1">
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>如何建立</strong><strong>易起量</strong><strong>的计划</strong></span></p>
</li>
</ol>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何建立效果好的广告计划，提升广告的pCTR（页面访问点击率）呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同的平台有不同的方式，不过了解运算运行原理则大同小异。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>以【巨量引擎</strong><strong>效果广告</strong><strong>】为例，通常我们在建立一个广告计划时，会搭配4-6条视频素材，以及6个以上创意标题。</strong>这4-6条视频素材，可能是新旧互补。因为确定效果的旧素材已经在运算模型中有了正</span><span>向反馈和定向数据，配合新素材可以让新素材快速起量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>创意组合的方式有很多种：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>①用已确认好效果的旧素材建立计划时，可以直接铺满10个素材，配合多条创意标题，快速起量。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>②用纯新的素材建立计划时，建议单条广告放入2-3个素材，减少波动；</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前面提到近期【巨量引擎效果广告】进行了比较大的版本更新，从官方的通告来看，新素材将获得更多平台扶持，这也意味着广告主有必要提高对【新素材】的重视度和成片出品量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>·在原版已经上传，且更新近3天在投放、起量好的TOP素材，即已得到正向验证的素材，可以更快速地起量，并能获得更多的推荐量；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>·在新版上传且从未投放过的纯新素材，巨量引擎会进行扶持，使新素材充分探索得以快速验证效果；</span></p>
<ol start="2">
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>如何提高创建</strong><strong>有跑量潜力的</strong><strong>计划</strong><strong>的效率：【素材媒体库】</strong></span></p>
</li>
</ol>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回顾第一章节，我们提到了【素材即定向】的广告优化思路。第二章的开篇，又提到</span><span>了创建优质计划的素材要求，说明了运营思路。那么，就有一个问题产生了：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一家公司的素材媒体库存有数以万计的素材，并且素材每周每月都在不断更新。那么，广告主如何从那数以万计的素材中，拎出合适的素材创建投放计划呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>【素材媒体库】就是不错的解决办法。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单解释一下【素材媒体库】的概念，我们可以把它理解为【素材数据化】。【素材数据化】也就是给每条素材打上精细化标签，便于运营人员使用，它的作用如下：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>·内容（视频）制作部门：可以按需求部门的要求快速制作素材，形成模块化的内容管理和输出；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>·新媒体运营部门：可以按标签化整理上传至新媒体运营矩阵，丰富账号内容，提升运营效果；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>·广告优化部门：可以根据素材标签从素材库提取素材，快速</span><span>创建更容易跑量的计划，</span><span>提升工作效率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并且，【素材数据化】也便于运营人员对素材的效果进行数据统计，让多个部门间的信息互通，优化整体协作。例如：广告优化部门在投放一定时间后，发现60s左右的剧情类-有漂亮小姐姐出镜的视频比30s左右的微硬广视频效果更佳，就可以快速提报内容制作部门要求加大力度输出这类型的视频素材用于广告投放。</span></p>
<ol start="3">
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>【素材媒体库】的搭建方式</strong></span></p>
</li>
</ol>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上简单说了些【素材媒体库】的核心作用，它不仅仅能提高团队协作能力和工作效率，还为精细化运营提供了强有力的支撑（思考【素材即定向】）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说了那么多作用，但是【素材媒体库】到底怎样搭建呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这回咱们还得回顾第一章节的概念【素材即定向】。【素材即定向】打破了以往通过优化人群定向的方式来优化广告的思路，用素材来解决定向的问题。定向的核心是标签，用素材让运算判断投放计划的意向人群。<strong>【素材媒体库】的逻辑也是如此，它服务于效果数据分析、运营工作实操与跨部门协作等工作。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>·【素材媒体库】的组成是什么？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>·如何为素材选择合适的标签？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上述两个问题，便是建立【素材媒体库】的核心，让我们来一一解决：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>·解决问题①：组成【素材媒体库】</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般来说，【素材媒体库】的组成需要考虑两个维度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一是素材制作流程，即输出一个素材成片，需要经过【文案脚本写作】→【成片素材的拍摄及上传】→【素材成片的剪辑及输出】；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二是素材的可衍生性，用过的文案二创改改方向、换一个录制人员配音、换换场景素材、在原场景下加入其他一些观点……种种像这样的从原素材衍生出来的新素材就是衍生素材。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好了，细心的朋友看完上面的分析，可能已经得出组成【素材媒体库】的要素了。即是根据素材制作时所产出物料的整理目录，具体如下：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1脚本文案库</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2素材制作库</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3素材成片库</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4衍生素材库（成片库）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（注：衍生出来的片段素材可以加入素材制作库，无需建立衍生素材制作库）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>·解决问题②：为素材选择出合适的标签</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上一小节，笔者列出了4类素材内容库，那么要如何给这四类内容库的素材打上对应的标签呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我认为至少需要考虑两个方面：（1）目的；（2）阶段；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（1）目的：进行素材内容标签的目的是什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>①方便找到素材</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一家公司的素材库有数以万计的素材内容，如果不进行标签分类，你打算花多长的精力找到一条在3个月之前拍摄剪辑的、由漂亮主播拍摄的、有某一产品细节画面展示且内容关于网络梗的视频呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就算你用5天时间“在大海里捞出了针”，那我要你找出这条视频的文案脚本进行二创，或者让你找出这条视频的衍生素材怎么办呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>嗯，合理划水。今年不用干活了，一直找素材吧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>建立多素材内容库的标签化的核心目的之一：加强不同类型的素材内容库之间的联系，提高运营人员的工作效率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据上述需求，要保障不同素材内容库之间的字段统一。为了达成这个目的，标签会包含<strong>脚本制作时间、素材的一级内容字段（如硬广、微硬广、剧情等）、产品/场景相关、出镜人名称</strong>等基础字段。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>②方便判断素材使用效果</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>标签化后的素材在投放时，可以快速判断素材内容的投放效果，获得进一步优化的数据。例如在判断微硬广、剧情类视频，哪一类视频能获得更好效果的时候，你可以对照周期性（15天/30天）的数据情况，进行多条同类型数据的整合和分析，算出这两类视频的核心数据：曝光人数、点击人数、表单提交人数、核心目标人群表单提交人数等。通过对比，得出哪种类型的素材能获得更好效果的结论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/90936979162e3444a37e707.38844264.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，上述案例仅仅是简单的基础字段判断，更精细化的计划优化需要涉及到内容方向和素材质量（场景匹配度、清晰度等）。这样的精细化运营才能达到【素材即定向】的实现标准。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（2）阶段：公司发展阶段+团队人员匹配；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公司的发展阶段决定了产品的功能迭代方向，也决定了产品在不同时期对不同场景的客户的核心功能展示方向的不同。面对这样的情况，核心标签自然也会有所改变。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而团队人员匹配，这涉及到工作量的安排。笔者当然知道大部分管理者希望自己所管理的组织或团队能提升信息交流、跨部门和部门内协同能力，并且他们力求完善。不过，不适宜的运营体系完善反而会加剧团队人员的不稳定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，你的团队仅两个执行人员，你却制定了精细到N级的标签，你觉得你的运营人员能扛得住这么大的统计和分析体量么？原本毫无破绽的体系，也会因为运营人员的遗漏出现漏洞，并且，大量复杂且繁琐的运营工作也会导致团队人员的不稳定性，这样得不偿失。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，笔者建议：<strong>运营，因地适宜，根据多种维度因素，进行有效优化才是好的运营。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上内容，便是笔者对于【素材内容库】的理解和整理。咱们再总结一下，【素材内容库】=【素材→数据化】，这种数据化是针对素材进行标签化，即是对脚本文案的标签化、素材库内容的标签化。<strong>这样才能便于数据统计，通过对多维标签、衍生分类的数据统计，让不同素材的计划效果有据可依，形成效果归因的分析。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有了这些，优化师想要快速建立好一批优质的计划，则如“喝水吃饭”一般轻松。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（Ps：计划名称也记得要加入统一的标签哦，便于在素材库查找对应的素材。并且，还能方便优化师快速一键建立优质计划。例如，新版【巨量引擎】上线了【一键引用往期计划】功能，如果提前标注好计划名称，就能快速建立计划，提升广告计划的创建效率。）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><img src="https://ngbjimg.xy599.com/154723207862d654264176a5.58982489.png" referrerpolicy="no-referrer"></span></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>如何管理广告计划</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果一名优化师，仅仅只会使用组织已确定的模块化标准是远远不够的（上述两章节的内容大多由团队管理者制定模块化标准，由运营人员执行）。<strong>他还需要懂得如何使用平台工具，善用平台的【投放能力】，提升广告【投放效率】，达成目标的【投放效果】。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通常，优化师管理广告计划会分为基本的三个阶段：<strong>投前准备、投中管理、投后分析</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>投放平台也会根据这三个阶段设置不同的工具。巨量引擎为广告主在投放的前中后各个阶段提供了很多方便好用的管理和分析工具，后续随着巨量引擎效果广告的版本迭代，这些工具也将逐步上线，帮助广告主提升投放效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正所谓：工欲善其事必先利其器。广告平台本身也在不断地改版迭代，以便适应它的用户需求。在平台的功能调整/上线时，善用平台工具，往往有助于优化师提升广告计划的投放效率和效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/140925159362e3444a59bde3.09753649.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>总结</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>互联网营销是围绕着“目标用户”展开的经营活动，营销广告的背后也是真实的“人”。意味着最终展现给用户的素材，</span>决定着用户下一步的行动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本文介绍了【素材】对广告效果提升的价值，讲解了优质计划的创建、管理以及【素材媒体库】的应用与搭建。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>明眼人可能已经看出道道，广告优化也是围绕着“人与平台”的精细运营，“文以载道，文以化人”，优质的创意素材提升了平台用户的广告体验，实现目标受众与产品之间的高度联通；并且，也有助于平台生态的良好发展，平台将给与这些账号更多的正向资源。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，【素材】决定了广告计划的定向优化。而【标签】提供了【素材数据化】的能力，通过投放的结果，不断收集素材的广告投放效果数据，从而归因分析出结果。最后，将结果再次复用于【素材】的优化。如此反复，三者形成闭环，不断优化彼此，这便是这便是【广告计划如何提升效果？】的答案。</span></p>
