<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在腾讯刚刚发布的2022年Q2及上半年财报中，<a href="https://www.niaogebiji.com/tag/%E8%A7%86%E9%A2%91%E5%8F%B7" target="_blank">视频号</a>被浓墨重彩重度提及。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也宣示着视频号在生态中增长空间被看好，资源会持续投入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频号总用户使用时长超过了朋友圈总用户使用时长的80%，日活跃创作者数和日均视频上传量同比增长超过100%，广告业务也初见成效，成绩喜人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2011开始发展，到现在近十几年间，短视频领域已经从立足于社交平台的秒拍、微视，到如今以抖音、快手两大巨头领航的百花齐放的局势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但是，抖音快手崛起的同时，腾讯系的流量基本盘也受到了冲击。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>十年间，腾讯先后推出近20款的短视频应用进行外部、内部竞争。但受限于行业发展、自身产品问题等原因这些项目均未取得较大成功。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5339293406320d815a52f52.13466218.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经历过数十次打造短视频的失败，在此之后，腾讯推出内嵌于微信的视频号，作为其进军短视频的又一次尝试，最终成为了在短视频领域的杀手锏。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span>01、</span><span>腾讯为什么做视频号</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，2022年，远程社交已经日益普遍化、必然化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信作为一个已经拥有近13亿用户的社交平台，其用户社交圈层在微信生态更加稳定化，从而形成了微信用户高忠诚度的特征；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19709212976320d8161db1e2.60414399.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，微信也遇到了棘手的情况。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自18年以来，短视频迅速发展，使得腾讯系用户使用时长不断被压缩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>短视频当前已超越即时通讯，成为互联网第一大用户需求。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当用户的时间、需求是基本固定后，你的潜在对手未必是同赛道的产品和企业，一切能满足用户某种需求的产品，都是你的对手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由此，重新聚集用户注意力，在微信生态中布局短视频，使微信成为“超级平台”来满足当前用户在移动互联网中的主要需求，成为微信在未来市场地位能否得到维系的重要因素。</span><span></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span>02、</span><span>视频号在微信中的角色</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在20年的一月份，张小龙在微信公开课上说会上线“短视频”内容，也就是现在的视频号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在公开课上，张小龙认为微信上<strong>“缺少了一个人人可以创作的载体”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>缺少的这个载体，就是我们今天所见到的视频号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近两年，视频号陆续推出直播带货、付费直播、互选平台等功能——借助公众号、搜一搜、看一看等<strong>接入公域流量</strong>，通过社群、朋友圈的营销推广<strong>引入私域流量</strong>，再在直播间将获得的公域私域流量引回微信号、公众号、小程序，完成<strong>流量创造到流量变现的闭环</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/7163622486320d81657bcb3.95706914.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>腾讯通过视频号这条“线”进行微信生态内的串联，将流量转化为存量，形成商业闭环。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在视频号团队看来，在微信内容生态中，<strong>视频号是“原子化的基础内容组件”</strong>——将视频号看作一个视频内容的载体工具，与小程序商店、公众号、企业微信等产品打通链接。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18567431086320d8169d3ec0.97497221.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span>03、</span><span>视频号“生于社交”</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信的核心功能就是社交，视频号作为内嵌于微信中的一个板块，其社交基因也深植其中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>视频号推荐机制的最大特色就是基于社交化的推荐算法。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与抖音快手的机器推荐不同的是，视频号通过社交裂变来实施“熟人推荐”模式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>熟人推荐就占据了55%的权重，用户使用的实际感受就是，浏览的80%以上的视频都是来源于朋友的推荐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20957443606320d816d301a5.38117104.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>熟人推荐和机器推荐不同之处在于，机器推荐会根据你看过的内容推荐相似的内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样的推荐容易出现<strong>同质化</strong>的问题，<strong>而同质化产生的结果就是形成视觉疲劳</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信视频号通过熟人推荐能避免这样的问题，每个人都会推荐不一样的内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使身边没有朋友推荐的内容，视频号也会自动推荐一些热门的的视频或者官方的科普视频，保证每个用户都有内容可以看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>视频号的这种推荐机制意味着微信好友感兴趣的内容决定了用户打开视频号看到的内容，这导致私域流量的影响力将再次被强化。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如之前在微信生态上有布局的玩家，做视频号就更加得心应手了，只不过是多了一种方法和粉丝交互罢了。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span>04、</span><span>视频号“困于社交”</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>微信的社交链是有局限性的。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每个人都有自己的社会角色，有自己树立的社会形象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于创作者来说，发个视频，点个赞可能最先被自己的熟人、亲朋好友、上司等微信好友看到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着“点赞”行为的不断扩展，很多人都抱怨，视频号变成了朋友、领导、客户之间“商业互吹”的工具，完全不可能说随意去创作，随意去点赞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以一些正能量、鸡汤类、风景类、影视类视频大行其道，而你真正喜欢内容的却只能看看作罢，这给视频号生态内容的丰富性造成影响。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时也造成同样的内容在视频号点赞评论率相比抖快要低很多，一旦形成恶性循环，“困”于其中，再想出来可没那么容易了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17219048576320d81719a687.32045734.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span>05、</span><span>视频号的商业化</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>目前，视频号变现方式除直播带货、打赏以外，还有视频收费内容、广告互选等。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频号自20年10月打通微信小商店、上线直播带货功能开启商业化探索，21年视频号与公众号、小程序、企业微信等板块先后打通。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并且直播任务、互选平台、机构管理平台等先后上线，视频号商业化全面提速。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17630518806320d8173f9aa2.24131779.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2022 年微信公开课上，官方宣布将重点打造视频号互选平台。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌方通过自身品牌特性和受众群体向目标视频创作者发出邀请，创作者则自由选择接受与否并定制创意内容，视频号内发布视频，作为第三方广告互选平台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>视频号官方平台借助微信的流量优势聚集了大量优质的内容创作和品牌方资源，同时提供更加安全的交易保障。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19803417666320d817704941.80400844.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，视频号未来不排除会有视频收费内容，包括直播、中长视频和短视频的收费、打赏功能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022 年 1 月，视频号上线付费直播间，在一场腾讯 NBA 视频号直播间，用户只可免费观看3分钟，观看全程需要支付90个微信豆，IOS版和安卓版分别对应13元、9元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16350536766320d817d207a3.99869781.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span>06、总结</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5G时代已经来临，短视频已经成为互联网的底层语言，相信短视频的下一个五年、十年会有更多创新的形式和玩法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而视频号作为微信少有的用户“被动获取”内容、能低门槛地将内容发表到朋友圈外——“公域”范围的产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>不仅代表微信“短视频内容”的全新尝试，也肩负着用户向“世界”展示自己的期待。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是谁，甚至是微信团队自身肯定也没有十足把握去预测，未来视频号会变成什么样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信产品的进化一直都是一个漫长的过程，而且是一个通过更自然的方式选择的过程。那如果我们简单类比朋友圈的现象，在未来视频号会怎么样呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后借用微信官方的一句话结束正文：视频号是一个让你记录和创作的平台，也是一个了解他人、了解世界的窗口。</span></p>
