<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《<a href="https://www.niaogebiji.com/tag/%E6%A2%A6%E5%8D%8E%E5%BD%95" target="_blank">梦华录</a>》翻车了？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本来《梦华录》光磕CP，就已经值得颜狗庆祝三天三夜了，但随着剧情的推进，<strong>大女主的人设开始立不住，CP冠上了“双洁”的设定也让人受不了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这</span></span><span>引发了很多人的吐槽，一度还上了热搜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/964813820629ec1ffde0220.06595945.gif" title="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" alt="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个是</span><span>有今天没明天的特务人员，十几年每天提着脑袋打工。</span><span>一直到了30岁的年龄，也还是处子之身。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13757237962ad4273173568.81817614.jpeg" title="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" alt="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个是风月场所长到成年，之后跟欧阳旭谈了三年恋爱，拖到24岁的大龄女（在古代），然后也是发乎情止于礼没有越界。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/56090747562ad4273445bd4.99724242.jpeg" title="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" alt="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span>真是天下苦“双洁”久矣！</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但抛开这偏走的剧情不</span><span>说，《梦华录》对宋代</span><span>服化道的考据，以及对中国传统文化的</span><span>输出，是有目共睹的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>试问，谁不想生活在宋朝，去体验一把“宋式风雅”……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1497802412608f8ba0ec53b8.74171672.png" title="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" alt="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong> 宋代文人，如何风雅？ </strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>陈寅恪说</span><span>：<strong>“华夏民族之文化，历数千载之演进，造极于赵宋之时”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>赵盼儿作为茶室老板娘，茶可谓是贯穿全剧的存在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99701136962ad4274a30ab0.15993159.gif" title="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" alt="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“盖人家每日不可缺者，柴米油盐酱醋茶”——茶文化在宋代发展至农耕社会的极致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>剧中赵盼儿所谓的点茶，是指将茶饼经炙烤、碾磨成末后，投入茶盏调膏，然后以沸水（宋人称“汤”）点注调出泡沫（宋人称“汤花”）的一种茶品冲沦方法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/49124548662ad4282c0f520.66842291.jpeg" title="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" alt="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>宋徽宗赵佶《文会图》就描绘了一场宋代文人雅士的茶会场景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>北宋时期一个夏日午后，文人学者以文会友，焚香、啸乐、赏画、抚琴，好雅兴！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4657936962ad428345f213.53036568.png" title="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" alt="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" referrerpolicy="no-referrer"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>南宋四家</span><span>刘松年《撵茶图》</span><span>为工笔白描，描绘了磨茶、点茶、挥翰、赏画的文人雅士茶会场景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/149551113862ad4283d85671.73343839.png" title="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" alt="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>· 斗茶，宋朝最之盛行 ·</span><span> </span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>说到“点茶”，就不能不提与之相关的文娱活动——“斗茶”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《梦华录》中赵盼儿与胡掌柜的“斗茶”一幕是宋代最盛行的茶活动，无论是下层社会的市井人物还是上流社会的士大夫，只要有闲暇，都喜欢坐下来，煮水点茶，看谁的茶艺更高超…</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>斗茶，一斗汤色，二斗水痕，三斗茶百戏…</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/34706489562ad4286054fd2.69488072.gif" title="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" alt="《梦华录》豆瓣评分高开低走？讲宋朝美学才是本剧的最大看点" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>斗茶是集茶的调制，茶的变化、茶的鉴赏、茶的想象、茶的品鉴于一体的活动。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>斗茶在发展的过程中渐渐增加了斗茶令和茶百戏等环</span><span>节。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>· 宋瓷，</span></span><span>雨过天青云破处 · </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>介绍“九九归一”茶那一集，赵盼儿</span><span>集齐了宫中所爱的</span><span>九色名瓷</span><span>，一下子就爆火整个</span><span>东京城，可见一件名器在点茶中的分量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>"明越唐，邓耀柴，饶，龙泉，定，至唐以来至国朝，宫中所爱之九色名瓷，尽在于此，配以官家（皇帝）至爱龙凤团茶，岂不是九九归元吗？"</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>至宋代时，名瓷名窑已遍及大半个中国，是瓷业最为繁荣的时期。</span><span>当时的汝窑、官</span><span>窑、哥窑、钧窑和定窑并称为宋代五大名窑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其中，青，宋瓷最具代表性的釉色。</span></strong><span>青瓷杯如玉似冰且能衬托煎茶汤色的绿，其他色泽的瓷器则不能彰显茶色。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>· 一香炉，一盏花</span></span><span> · </span></span></p>
