<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">“主持人yoyo酱”（下称“yoyo”）谈起抖音一周涨粉超116万的经历，还是显得有些难以置信。</span></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“非常激动，这种激动掺杂了很多复杂的情绪，包括我那天一打开直播间（发现）里面有几万人的时候，我就觉得‘天，这是真人吗’？我长这么大没见过这么多人。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>yoyo的抖音账号主要分享以电视台演播室为背景的主持人搞笑日常，例如忘带正装，拿A4纸当套装内搭；拿出保温杯装作喝茶，其实是在茶杯里装满了泡面；又或是播报新闻时发现新闻实在太好笑，忍不住笑场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新抖数据显示，截至目前，该闪光牙套相关视频播放量已达1亿，是yoyo近90天播放量最高的一条视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a data-miniprogram-appid="wx55778a0d7b3048c4" data-miniprogram-path="pages/NewLive/index" data-miniprogram-nickname="新抖数据" data-miniprogram-type="image" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined"><img src="https://ngbjimg.xy599.com/205960655462ea1176082b12.95899318.png" referrerpolicy="no-referrer"></a></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，由于她的搭档徐颢鑫常常在视频中为她挡衣领、买香水、为她的“失误”救场，最近几乎每条视频的底下都有网友喊着“民政局已经长大了要学会自己跑过来哦”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自从yoyo的视频被微博近千万粉丝大号“我和基友的日常”搬运后，“当搞笑女成为主持人是什么体验”话题一下子火上了微博热搜，数名粉丝从各个平台涌入，新榜旗下数据工具新抖及新快显示，7月18日-7月25日，yoyo全网至少涨粉175万。截至目前，yoyo的全网粉丝量已经涨到了500万左右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/7915632062ea1176427ce4.14255041.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们找到了yoyo，试图了解一个传统电视台新闻节目的主播与自媒体搞笑博主之间存在着什么样的联系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是yoyo第一次因为抖音账号接受访谈，以下是她的故事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>懂事的民政局已经自己去电视台了</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>yoyo有一位为账号迅速吸纳流量的搭档：徐颢鑫。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>徐颢鑫串场的第一次，就让账号久违地出现了点赞量过25万的视频。<span>徐颢鑫</span>也是一名主持人，和yoyo不同的是，视频里的yoyo总是看起来慌里慌张、古灵精怪的，而徐颢鑫则显得大气、沉稳，还会时不时帮yoyo处理问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在下图这条视频的情节里，导播切错了镜头，导致在同一个画面中既有女主播补妆，又有男主播正常播报的画面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/109052186062ea11767e91d9.21575098.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但直到yoyo补完妆发现镜头切到了自己，马上强装镇定为止，徐颢鑫都不受丝毫影响地照常播报新闻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/173473984362ea1176b71a99.62841226.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友评论</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>yoyo开玩笑似的说：“我发现大家不是喜欢我，他们是喜欢跟颢鑫在一起搭档的我。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>徐颢鑫的加入源于一场巧合，他与yoyo本身就是工作搭档，一天两人正好凑在一块录节目，编导李尚保突发奇想地问了一句：“颢鑫你一会儿有事吗？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他刚巧有空，一条视频就这样诞生了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>yoyo介绍，第一次参与录制的时候，他还是轻松随意的状态。视频一下子火了以后，他就开始时刻注意自己的仪容仪表，甚至还主动提出化妆上镜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那网友们都“磕疯了”的视频素材源自哪里？就源自徐颢鑫视频中一次次绅士的行为。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/121013129362ea1176e9cfa9.42587560.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从左至右依次为徐颢鑫为捡笔的yoyo挡住领口、为吃面包噎到的yoyo默不作声地递上水、以及yoyo一不小心打湿口播稿以后递上自己的稿子</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一些视频里，每当yoyo想要跟观众朋友们澄清他们并不是情侣时，徐颢鑫都会佯装正忙，没有空回复。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/107942827462ea11773b1400.59466043.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚刚还说自己有空的徐颢鑫一秒变脸，网友：所以颢鑫不想成亲，那想不想成亲呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在另外一条视频中，yoyo美滋滋地走入演播室说男同事李涵的香水很香，徐颢鑫马上就摆出扑克脸说“有点儿呛”，之后，他又不知道从哪儿搞来了一大堆香水交给yoyo，在面对后者惊讶的脸时说道“不用找李涵了”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/212547864862ea11778263f8.04849013.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些情节都让观众忍不住嘴角上扬，盯着屏幕“催婚”：每日一问，你们什么时候结婚？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>yoyo账号的内容方向也逐渐“走偏”，在小甜剧的道路上越走越远。截至目前，yoyo和徐颢鑫共同出镜的系列视频“冤种搭档的日常”已累计2.9亿播放量，这几乎是做账号3年来最受欢迎的系列视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她还发现在发布此类视频之后，她的粉丝比例已经从原先的80%男粉转变为了现在的男女粉比例5:5左右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>甚至连编导李尚保都说：“与其问他们参不参与脚本（创作），不如问我有没有‘偷窥’他们的日常，我自认为是他们的CP粉头，yoyo和颢鑫就是那种，你见了他们就能脑补出一部连续剧的搭档，我不过是把粉丝们脑补的一些桥段让正主们呈现出来罢了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了回应粉丝的热情，主播二人开了直播与粉丝互动，直播间的参与人数一下子从过往的七八十人上升到了现在的几万人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>yoyo表示：“其实大家问的最多的就是我们两个到底是什么关系，然后我们就会很诚恳地告诉大家，我们是好同事、好搭档、好朋友。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>爆红一周，不温不火三周年</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>爆红，需要的是厚积薄发，再加上那么一点点的运气。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年，yoyo正式入职济南广播电视台担任主持人，彼时的抖音月活用户已过3亿，新媒体发展迅猛，成了传统媒体不得不面对的一个行业趋势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她说：“当时城市广电都纷纷转型，成立MCN机构，我这个账号就是传统媒体向新媒体转型的一个产物。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时至今日，yoyo已经记不起当初是谁最先提出要做这样一个账号，只记得台里领导很支持，还说“既然是做新媒体，那就要做点跟传统媒体不太一样的东西”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但毕竟是从传统媒体里面分拨出去做新媒体，跨了半个行，把账号建立起来以后的很长一段时间内，团队都有些摸不着头脑，到底应该怎么入手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>yoyo回忆，当时虽然他们能够保持每周更4-5次，但一直都没有效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“一直到2020年年初，（我们账号）小爆了一条视频，爆了以后一下子涨了4万的粉丝，我和当时的导播张文静、摄像李小东都特别激动，整晚抱着手机不停刷新，每一次刷新都会涨一点点粉丝。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但即便如此，2020年年初时的账号选题依旧不太固定，从“当和不熟的人待在一起时的尴尬瞬间”到“艺考时的尴尬瞬间”，再到“主持人涂口红独家秘诀”，几乎没有规律可循，一直在跌跌撞撞地找合适的视频路线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其后，团队又遇到了第二个瓶颈：账号有粉丝了，但还是不温不火。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/183772170362ea1177b1b8f7.13473505.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年yoyo拍摄的部分视频，包括吐槽领导、紧急换装录像等情节</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年11月，yoyo账号的粉丝量达到了50万，但由于要完成台里安排的其他任务，新媒体业务人手严重不足，yoyo也就这么断更了很长一段时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直到今年的三四月份，yoyo总算迎来了工作的新搭档：编导李尚保。后者负责写脚本、导演、摄像和剪辑，yoyo则负责出镜，原来作为视频背景的演播室常常被台里安排参观，他们就申请换了新的演播室进行拍摄，两人就这样把账号继续做了下去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李尚保介绍，以前的账号内容多是展现“剧情”，而现在的内容会更倾向展现主持人的“状态”，帮助主持人从剧情设计中游离出来，摆脱表演的痕迹，去真实地展现yoyo可爱搞怪的人物魅力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他提到：“大家印象中主持人都是正襟危坐、不苟言笑、不容有任何失误的‘机器人’，但其实就算是央视主播也常有失误的时候，所以‘直播失误’是我们的一个创作方向，通过还原直播时可能会出现的一些小插曲，让大家看到主持人也是有血有肉的人，会失误、会搞笑、会跟观众互动，拉近与粉丝的距离。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，这也是yoyo获得众多粉丝喜爱的原因之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传统主持人“正襟危坐”和yoyo“不拘小节”的对比营造出了强烈的反差感，她的视频像是传统电视台中难以想见和公开的画面，大家乐此不疲地猜着yoyo究竟有没有被扣钱，是不是“贷款上班”，到底哪些视频是真的花絮，哪些视频是艺术创作，仿佛yoyo已经成了他们日常生活中的一份子，电视台主持人也不再是那个西装革履的“神秘人”，而是变得更加亲近与自然。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据了解，短视频创作只是yoyo工作的一部分，除此之外，她还负责<span>节目录制、新闻配音、大型活动主持、直播带货等</span><span>工作，台里还因此称<span>她</span>为<span></span><span>“</span><span>全能</span><span>型</span><span>主持</span><span>人</span><span>”</span><span>。</span></span></span><span></span></p>
