<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在上周，</span><span><strong><a href="https://www.niaogebiji.com/tag/%E8%9C%9C%E9%9B%AA%E5%86%B0%E5%9F%8E" target="_blank">蜜雪冰城</a>发布了一则通知——乡镇开放加盟申请。</strong></span></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/163066504962c75a1e37e7e1.32988014.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在“通知”里，蜜雪冰城表露野心，为了让更多消费者能享受到高质平价的美味。同时带动潜力市场的发展，与各位加盟商一起开拓新机遇，创造新价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得注意的是，</span><span><strong>此次开放乡镇加盟仅对全国老客户且满足加盟商档位为1档、2档、3档开放。</strong></span><span>意味着新入局者恐怕没有机会掘金乡镇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，不少网友也是给予了极大热情：“俺们村（镇）也能有蜜雪冰城了”、“住在村里的人狠狠期待了，镇上都喝不到”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>县市加盟饱和后又开放乡镇加盟，这是遭遇下沉困境的蜜雪冰城必须做出的选择。面对全国41636个乡镇行政单位，</span><span><strong>已在全国布局2万多家的蜜雪冰城似乎又找到了新的狂奔路径。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是对于市场公认的年轻人少、消费能力差的乡镇地区，会是蜜雪冰城逐梦新茶饮的一个好选择吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/381440126233b25ace1df6.64458916.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>乡镇，蜜雪冰城的“必选项”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>下沉市场的“地板”并不在县城，而是在乡镇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>新茶饮的下沉早已成为行业内的共识。</span></strong></span><span>既便如喜茶、奈雪一样高贵，也不得不放下身段去三四线寻找新的增长空间，更不要提蜜雪冰城、coco、益禾堂等本身下沉市场的受益者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/188571852362c75a1ec016a3.16202660.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，蜜雪冰城正是那个将下沉市场的巨大潜力公之于众的品牌。而这完全得益于其在开店速度上的一骑绝尘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/89409955762c75a1f4b1327.37033358.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年至2019年，其门店数先后突破了3500家、5000家、7500家的大关。2020年6月24日，蜜雪冰城宣布全球第10000家门店在河南新乡开业，成为第一家也是唯一一家拥有万店规模的茶饮品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/83567898962c75a1fa100c3.65941214.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>而从1万家店到2万家店，蜜雪冰城又仅仅用了不到一年的时间。</span></strong></span><span>疯狂的速度背后是庞大的五六线县市市场的支撑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾有加盟商表示，5元到10元一杯的奶茶，吸引力足够强大。收割一波高频顾客、一波低频顾客，其中的市场空间甚是广阔。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>也正因为此，各大新茶饮品牌纷纷瞄准了不同层级的下沉市场发力。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说喜茶、奈雪等超级品牌的下沉底线在三四线，那五六线的县市就是蜜雪冰城们的“天堂”。其中与蜜雪冰城正面硬刚的益禾堂、古茗、书亦、coco等在“县城交锋”中打得火热。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它们中平均5000+的门店也大都分布在县城。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，曾经下沉市场的“蓝海”——县城也成为了一片红海。五步一“蜜雪”，十步一“禾堂”成为一种常态。一些加盟商为了防止自己的生意被抢，会在蜜雪冰城的“提醒”下选择在同一区域加开新店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/94639921562c75a202a5c63.12641947.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但如此内耗并非长久之计，当县市卷无可卷，对于吃尽下沉市场红利的蜜雪冰城来说，继续下沉到乡镇或许是再正常不过的思维惯性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟，那些乡镇上令人啼笑皆非的山寨版已经证明了蜜雪冰城的影响力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/123357659162c75a206a8a12.23580796.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实在前两年，蜜雪冰城就曾试图开放乡镇加盟，但鉴于当时县市市场的火热，相对“难啃”的乡镇并未泛起水花。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今再次开放加盟，不管能不能拿下这片广阔的市场，对于遭遇扩张瓶颈的蜜雪冰城来说，似乎都是一道必选题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9225241376233b25c6ec6a4.50995726.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>蜜雪冰城的“分裂”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在中国茶饮界，很难找到像蜜雪冰城这么“分裂”的品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为蜜雪冰城不仅心系乡镇，誓要当贫民窟女孩的救星，还有一颗国际化的决心。在向乡镇市场发起总攻之前，蜜雪冰城早已铺开了一张出海扩张的“巨网”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/192290585162c75a20cbff98.34255689.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月初，华与华董事长在社交平台上发文表示蜜雪冰城海外门店已突破1000家。一时间引发很多人错愕：这么“土”的蜜雪冰城竟然能风靡海外！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实早在2018年蜜雪冰城以品牌英文名“MIXUE”，在越南河内开出首店。到2021年12月，当地门店规模已突破了200家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/32159417762c75a2147c6a7.77738234.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，蜜雪冰城的触角向印尼、缅甸、老挝、马来西亚、新加坡等周边国家开始延伸。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>当然，在海外蜜雪冰城的大杀器依然是低价，性价比永远是王道。 </span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如在越南，蜜雪冰城珍珠奶茶的价格是2.5万越南盾，也就是7元人民币；在经济发达的新加坡，蜜雪冰城的产品定价同样亲民，大多在1-3.5新加坡元区间（1新加坡元约合4.8元人民币）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>向乡镇“下探”、向海外“外探”，蜜雪冰城还觊觎着喜茶奈雪们的领地——向高端化“上探”。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018年，蜜雪冰城推出高端茶饮品牌M+，主打鲜泡茶与特调牛乳等饮品，价格最高上探到20元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但从如今在市场上几乎难觅踪迹的M+来看，显然这次“上探”失败了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/26004096662c75a21947600.27520821.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，蜜雪冰城总经理张红甫说：“我们的基因决定了我们更适合做高质平价的产品。在目前这个阶段，我们属于中质低价，从中质低价到高质平价需要一个过程。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>显然，张红甫只说对了前半句。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>高端化失败，海外扩张的“小体量”，“分裂”的蜜雪冰城想要讲出下沉市场以外的故事并不容易。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在看来，能撑起蜜雪冰城野心的貌似只剩下乡镇这片“天地”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16900100706233b25e365e39.91683873.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>乡镇，新茶饮的下一站？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于此次蜜雪冰城开放乡镇加盟，不少加盟商直言：</span><span><strong>“没啥开头，乡镇人流比较少，还是应该去大一点的城市开店。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>的确，乡镇老人孩子居多，消费能力弱，茶饮的目标消费人群——年轻人少之又少。表面上看，对于极度依赖海量年轻人的新茶饮来说，这并非一个理想的市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>但任何事有A面，必有B面。</span></strong></span><span>对于如今的新茶饮生意来说，无论是一二线的高端市场，还是五六线的下沉市场，除了饱和的竞争之外，越发高企的加盟成本，越来越长的回本周期都是“劝退”加盟商们的达摩克斯之剑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就拿蜜雪冰城来说，在加盟费最低的县级城市想开一家蜜雪冰城，首先要交2万元的保证金，花十多万租房、8万元左右买设备、8万元左右装修及6万元起采购原材料。每年加盟商还需交13800元的加盟费、管理费、咨询服务费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>也就是说，没有40万的资金储备，几乎很难在县城开起一家蜜雪冰城。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但在乡镇，这些问题都迎刃而解。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于多数乡镇加盟商而言，这里没有高昂的房租，也没有高昂的人工成本。对于只想找一份体面生计的人来说，这是一个不错的选择，自然也埋下了新茶饮下沉乡镇的沃土。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而从更大的市场维度来看，乡镇也并非许多人想象的那般“没啥开头”。相反，</span><span><strong>在蜜雪冰城布局乡镇之前，已经有众多连锁茶饮品牌证明了小镇的市场价值。</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一个开拓乡镇市场的甜啦啦。今年春节期间，20多家开在镇上的门店，日营业额都过万。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有四川的茶屿水果茶，在位于德阳市中江辑庆镇的一家新店，开业当天营业额就超过了2.5万，如今这家店的月营业额稳定在12万~15w之间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像罗振宇在跨年演讲中所说：在中国再众所周知的事情，都有一个亿的人不知道，但往往是十个亿的人都不知道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>市场的潜力同样遵循这个道理。</span><span><strong>庞大的中国市场被分裂成了无数层，有流露在外、拥挤不堪的AB层，也有被折叠、被忽略的CD层。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而乡镇无疑就是那个始终被主流市场忽略的“蓝海CD层”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，当年唤醒“县城”茶饮消费的蜜雪冰城，再次让“乡镇”暴露在众目睽睽之下。不仅让一个被主流忽视的市场站上C位，同时也向新茶饮发出一个宣告：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>当新茶饮卷无可卷，“荒芜”的乡镇大市场或许藏着新茶饮的大未来。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>困难一定有——比如市场过度分散，比如仓储、物流与供应链......但拼的就是谁能找到那一条组织品牌的捷径，谁就能做大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，蜜雪冰城已经迈出了第一步！</span></p>
