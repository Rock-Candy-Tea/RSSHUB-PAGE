<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">前段时间网上掀起一阵高价雪糕的议论风，市面上雪糕价格越来越高，小小的雪糕定价能到几十甚至上百，天花板一样的价格怎么看配料表都看不出“高级”的端倪，进而这类高价雪糕就被附上了“雪糕刺客”的名号。</p>
<h4 class="ql-align-center" style="border: 0px; margin: 25px 0px; padding: 0px; font-size: 14px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #818386; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; font-weight: 300; line-height: 30px;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p1.itc.cn/q_70/images03/20220824/4ca3b9d006fe498aa355f8f047ed81e7.png" referrerpolicy="no-referrer">来源-微博</h4>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">在此之后，刺客一词也成了反映当下物价水涨船高的一种新说法，那些一块钱两块钱的商品在逐渐退出市场，演变成“童年回忆”。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">在此背景上，UP主「康哟喂」就发起1块钱买水的挑战，这一主题瞬间吸引了不少人的目光，好像在看到标题的那一刻，就有人在提醒你“在2022年，买1块钱的水也变成一种挑战了”。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">标题一出在第二天这个视频就以<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">一小时37.9w的播放增量登顶B站热门榜</span>，还被收录进B站每周必看。</p>
<h4 class="ql-align-center" style="border: 0px; margin: 25px 0px; padding: 0px; font-size: 14px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #818386; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; font-weight: 300; line-height: 30px;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p4.itc.cn/q_70/images03/20220824/ab24225490ee4a0c96793d765f8d345a.png" referrerpolicy="no-referrer">来源-果集·飞瓜数据（B站版）</h4>

<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">纵观UP主整支作品可以看出，他其实一直在把<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">过去和现在</span>做对比，对于B站用户而言，“过去”常常是热议的话题，也是他们作为互联网第一代用户常常在思考的问题，所以在UP主提出1块钱挑战的时候，对B站用户而言更像是<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">通过UP主去看一件自己很久没有触及到的小事</span>。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">在视频中，UP主先是对科技高速发展的今天发起疑问——“你有多久没有用过现金了？”，当新版一块钱硬币出现在观众面前时才有人惊觉，现金的体格在缩小。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">1块钱硬币变小了，能买到的东西是不是也变少了，为了得到这个答案UP主为此决定要在杭州用100元买100件不超过1元的商品。</p>
<h4 class="ql-align-center" style="border: 0px; margin: 25px 0px; padding: 0px; font-size: 14px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #818386; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; font-weight: 300; line-height: 30px;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p9.itc.cn/q_70/images03/20220824/79756ccb19d840eab34478bc4f5d2ae3.png" referrerpolicy="no-referrer">来源-<a href="https://www.niaogebiji.com/tag/B%E7%AB%99UP%E4%B8%BB" target="_blank">B站UP主</a>康哟喂</h4>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">又在寻找1块钱商品的过程中，不断的对商家问起他们曾经用1块钱能买到什么，老一辈的人所回想起来的过去和身处高楼大厦之中的UP主有鲜明对比。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">这个环节很好的带动了观众的<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">互动欲望</span>，在弹幕中接二连三的对1块钱能买到什么东西进行盘点、阐述，这也是为什么在视频发出后<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">弹幕数一直在水涨船高</span>的原因。</p>
<h4 class="ql-align-center" style="border: 0px; margin: 25px 0px; padding: 0px; font-size: 14px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #818386; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; font-weight: 300; line-height: 30px;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p1.itc.cn/q_70/images03/20220824/e5322ba82e08481b9b4712a9a6dfa94b.png" referrerpolicy="no-referrer">来源-果集·飞瓜数据（B站版）</h4>

<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">在视频最后UP主提到，在一线城市中仍然存在着很多从早些时候传下来的事物，比如几十年的老字号，和生长了几十年的人们，还是那些人那些物，但在时间的推移下，他们也在忘记过去的物价，取而代之的是适应当下经济发展的价格。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">这一点就和一开始“雪糕刺客”衔接上</span>，全网都在避免购买到高价雪糕，在一众刺客中寻找“小时候的回忆”。最后UP主发现1块钱的消费还存在在这高速发展的闹市中，但是逐渐脱离了现在人的生活，所以才显得值得回忆。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">这也就解释了前段时间全网热议的《二舅》为什么那么火，UP主发现大家都在高压的生活中在一个个视频里找到快乐，这种感觉就像小时候花不到1块钱就能买到快乐的感觉是一样的。</p>

<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">前面挑战发起或许带着一点搞笑的意味，但到视频后端<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">思考现实问题</span>反倒成为主导内容，这也是<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">B站现下的创作的一个趋势，在作品中升华主题，从表层的内容逐步递进到更深层的问题</span>，这样的内容会给看完的观众留下很强的感官冲击，这就是评论区有观众反馈的“看完有种怅然若失的感觉”，以及“视频的意义远高于娱乐”等言论。</p>
