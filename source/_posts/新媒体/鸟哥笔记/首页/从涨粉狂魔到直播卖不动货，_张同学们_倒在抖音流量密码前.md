<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>距离@<a href="https://www.niaogebiji.com/tag/%E6%9D%8E%E5%AD%90%E6%9F%92" target="_blank">李子柒</a> 停更已经过去了将近一年，这一年里@张同学 、@帅农鸟哥 两位三农博主先后爆红，但比起@李子柒 建立起来的商业帝国，同一赛道的三农博主却仍只能囿于<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>，<strong>即便是之前声称“暂时不会带货”的@张同学 ，也终究逃不过直播带货这一结局。</strong></span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月25日，@张同学 在抖音开启了自己走红以来的第一场直播带货。据抖查查数据显示，<strong>@张同学 首场直播带货累计销售额为331.57W</strong>，带货商品主要为农特产。或许是经验不足，@张同学 整场直播“磕磕绊绊”，身边的助播一波接一波换人，原定的4小时直播也变成3个多小时，直播结束更是匆匆忙忙。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/119102433462c48cc73bba99.60299030.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比起那些动辄一场直播带货上千万的同一粉丝量级博主，@张同学 作为一个粉丝量1892.3W</span><span>（截至发稿前数据）</span><span>的大V，<strong>这样的直播带货成绩谈不上有多出色，甚至还有点“掉队”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播带货首秀不尽如人意，或许也与@张同学 的热度消退有关。自去年走红到现在，已经过去了230天。据果酱妹观察，<strong>@张同学 近90天里仅有三天出现了涨粉，其中两个涨粉节点是在开启直播当天，其余天数均在持续掉粉状态。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而对比@张同学 近期最新的30个作品，点赞数过百万的作品仅有4个。在@张同学 走红初期</span><span>（2021年12月~1月）</span><span>，点赞过百万的作品则是家常便饭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/185441071962c48cc8a8c4c5.63359193.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>左为近期作品，右为以往作品</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，早期的@张同学 在视频里更多是记录自己起床、喂狗、喂鸡、上山砍柴、跟朋友吃饭喝小酒的乡村生活日常，<strong>现在其作品里开始出现越来越多新的面孔，场景也逐渐丰富起来，但故事却难免落入了俗套，变着花样做起了美食。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样落入俗套的，还有@张同学 的商业变现之路。或许，他在爆红之际也曾想走出一条与其他三农博主不一样的路，但终究还是耐不住现实的残酷。<strong>“花无百日红”，抖音也永远不缺下一个@张同学 。</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乡村题材成为流量密码已然是业内共识。抖音近几年不断加码三农内容，先是推出“新农人计划”，用12亿流量激励创作者打造三农内容，再是在抖音里造神@张同学 ——一周涨粉近300W，两个月成为千万粉丝博主。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三农赛道自此人来人往，热闹非凡。据抖音发布的三农数据报告显示，2021年，抖音农村视频总获赞量达129亿，农村视频创作者收入同比增长15倍。但与此同时，<strong>僧多粥少，三农博主们可瓜分的流量似乎正在减少。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一方面，三农内容日渐趋向同质化</span></strong><span>。试想任何一个你关注过的三农博主，他的视频作品是否遵循着这样的创作流程：</span></p>
<section>
<section><span>充满活力的三农博主，穿梭在乡间小道/农田/茶园/果园/竹林，就地取材，大量的洗菜做饭镜头，间中穿插着炊烟袅袅或鸡鸣鸟叫。</span></section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/121337811162c48ccb7491d6.36952069.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>粗略统计抖音三农博主粉丝量TOP 10，数据截至发稿前</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从抖音三农博主粉丝量TOP 10来看，除了@大表哥Vlog 在“认认真真”捕鱼外，其他三农博主一致选择深耕乡村美食生活。<strong>同质化程度已经达到，甭管用户刷到哪一个三农博主，都能云吃上一口热饭。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管大家的内容变现形式不同，但都逃不过内容困境和流量下滑。@张同学 的“过气”并非个例。同是千万粉丝博主的“山村小杰”，走红于2018年，如今的视频点赞量则基本在1W~3W之间徘徊。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另一方面，三农博主商业变现存在信任危机。</span></strong><span>@张同学 直播带货首秀少有人为之买单，至今也仍未开启第二次直播带货。在此之前，他的前辈们也曾翻车在直播带货上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年10月30日，三农博主@牛爱芳的小春花 开启直播带货首秀，销售额高达2640.8W元。但随之而来的，是用户对其带货的质疑。@牛爱芳的小春花 在首场直播中售卖的商品与其农民形象不符，199元的空气炸锅、269元的破壁机，高昂的价格让用户一时无法接受。再加上被爆内容造假等问题，<strong>@牛爱芳的小春花 成了用户口中的“牛爱谎的小春谎”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自那时起，@牛爱芳的小春花 几乎一直处于掉粉状态。<strong>去年曾一个月掉粉百万，而其近90天里也掉粉超60W。视频点赞量也从之前的百万掉至十几、二十万。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/182257513262c48ccba44cc3.58332661.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这边厢，抖音暂时停止在三农赛道造神；那边厢，B站则试图加码三农内容</span></strong><span>。今年3月份，B站曾发起了#田园美食征集令#，前30名UP主最高可获2000元现金奖励。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月22日，B站更是在生活区下设了“三农”分区，并发布“三农繁星计划”，用流量和奖金来激励创作者参与三农内容分享。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/80573291562c48ccd5b3473.36223256.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得注意的是，比起在抖音里的难以施展拳脚，@帅农鸟哥 成为了B站的三农博主代表。7月2日，@帅农鸟哥 还与央视农业、人民日报联合发布#体验新农人#视频。<strong>不过从目前的数据反馈来看，三农分区内最高的视频播放量也才496.9W，比搞笑区最高视频播放量1896.7W，少了3倍有余。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前有抖音的“新农人计划”、快手的“三农快成长计划”，后有B站的“三农繁星计划”，各大短视频平台加码三农赛道已成标配。可以预见的是，未来很长一段时间内，仍有越来越多的参赛者加入赛道，这也意味着</span><span>竞争将越来越激烈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“迅速爆红，光速过气”在短视频里逐渐成为常态，留给三农博主们掘金的机会或许不多了。</span></strong><span></span></p>
