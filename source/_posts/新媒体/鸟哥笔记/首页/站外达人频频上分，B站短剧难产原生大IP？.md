<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“追我的男人已经排到了宇宙的尽头。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在个人简介中，@王七叶-这样写道。迄今为止，在B站，她的粉丝数量已经突破85万。对于仅发布39支作品的账号而言，这个成绩已十分亮眼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>初登场时，@王七叶的身份是一名搞笑女，迈着自信昂扬的魔性步伐，<span>奔波在撩各种男人的路上，并</span>以大女主的气魄说出一系列海王语录，可谓将女性油腻学发挥到极致。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/153059627263194351672e06.94023204.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@王七叶 B站视频截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>夸张与反差或许是王七叶吸引用户的第一要素，在视频中，她的一些固定动作为观众津津乐道，并逐渐演化为梗：穿着高跟鞋的“每日一跳”被称为“尥蹶子”，最后的独舞则是“MVP结算界面”……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，最吸引粉丝的还有@王七叶与其中一位男主角@陈太乐的互动，两人在剧情中所表现出来的女强男弱的喜感，恰好戳中了观众磕CP的心思，也为剧情增加了不少亮点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅是@王七叶，卡思发现，短剧类UP主似乎在最近一段时间内大受欢迎。例如在6月初入驻B站的@小陈总啊，凭借一张酷似朱亚文与张译的脸与魔性的舞姿，迅速积攒起71万粉丝；登上7月涨粉榜的@吴厘头啊，依靠家庭式短剧一炮而红。再往前追溯，则有@非非宇Fay、@刘大悦er、@导演小策等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>B站的短剧UP主，“支棱”起来了？平台所力推的短剧生态，发展的如何了？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>短剧UP主在B站</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>故事开始于幽暗巷子里的算命摊，大哥带着小弟来到摊前问“赌运”，哭得伤心的老婆婆则在寻访儿子的下落，穿黑色长褂的算命先生摇着签桶，为他们算出“命定”的一卦。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年12月，短剧UP主@非非宇Fay登上B站月度涨粉榜。研究其账号内容，<strong>即采用“一镜到底”的拍摄方式，将人情冷暖浓缩进3分钟内的剧情中</strong><strong>，凭借温情的内容与细腻的特效，令用户在短时间内收获了一段完整而丰富的情感体验。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/24817389163194351974956.25400007.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@非非宇Fay B站视频截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两个月内，@非非宇Fay涨粉百万，意味着这类短小精悍、短时间内给足信息点的剧集并不只是风靡外站，同样也深受B站用户欢迎。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>@非非宇Fay之外，在B站短剧江湖浮沉的还有@刘大悦er、@朱家寨少寨主这些在站外同样具有一定规模流量的达人，而B站亦有不少原生UP主“下海”短剧赛道，如在动画区拥有一批核心粉丝的@凉风kaze等，他们共同构成了B站短剧的内容生态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>细究起来，据卡思观察，B站现有的短剧内容大体可以分为两类。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其一，由B站官方主导，联合第三方制作出品，或由UP主参与共创的短剧。</span></strong><span>这类短剧往往制作精良，在质感上更加接近传统影视剧，时长在10分钟左右，代表了B站自制短剧的方向与“门面”，被纳入“轻剧场”板块。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，在B站先前的出圈短剧《抓马侦探》中，@自来卷三木、@星有野等头部UP主均有出镜；今年7月，B站推出自制短剧《奋斗吧龙顶天》，全剧12集，单集时长约6分钟，而在短剧中，被粉丝调侃为“大佐”的日语UP主@紫菜蛋花兔兔便深度参与其中，饰演主角团成员之一江灵儿，从而迈出了自己的“花路”；另外，在第一季收获好评后，@非非宇Fay的《先生，我想算一卦》第二季同样主打“解签反转”“一镜到底”，在6月回归了B站，成为UP主与B站共创的典型案例。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/77182298763194351e3d107.76334095.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>《奋斗吧龙顶天》截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其二，由UP主自发创作、更新的短剧</span></strong><span>。在站内检索关键词“短剧”，即可跳转至当月B站小剧场专栏，观看由UP主主导创作的各类题材短剧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以8月为例，正在播出的短剧包括@进击的金厂长的《万事杂货铺》，@内幕纠察局的《我在B站做卧底》、经典恐怖“老番”《不思异》等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得注意的是，<strong>无论是题材选择、时长还是展现形式，站外达人与原生UP主的短剧创作都有所不同，甚至颇为泾渭分明。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从展现形式上，<strong>站外达人大多搬运以往作品，以竖屏为主，符合用户在短视频平台沉浸式刷短剧的习惯，时长也多控制在3分钟之内，内容更偏向于搞笑类。</strong>比如@朱家寨少寨主的《上学时那些一战成名》，以校园搞笑日常为主要内容。在其中一集《老师的尴尬瞬间》中，身为班主任的朱老师每每想要隐瞒一些放假信息，或者谆谆教诲学生暑期要读书学习，都会被隔壁王老师“唱反调”，两个老师的恩怨情仇成为这条视频的看点与笑点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而平台原生UP主的作品则具有明显的符合B站用户观看喜好的倾向，在形式上以横屏为主，时长扩展到6、7分钟，追求脑洞大开、神反转与层层设定。</span></strong><span>@凉风kaze的《四星杀手》，与时下热门话题相结合，讲述自己作为杀手的失败经历，所讨论话题如“小镇做题家”等，与年轻人心理相契合。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二者的差异很容易理解：对站外达人来说，B站是内容的分发渠道之一；而对“身兼多职”的UP主，<strong>短剧则作为内容的一部分，构成了其在B站的发展底色，靠近平台取向才是正确的选择。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，在微短剧成为时下风口、诸多达人纷纷“下海”短剧赛道的当下，B站的短剧之路，走到哪里了？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>B站短剧，生态如何？</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，<strong>在布局微短剧上，B站入局早，但起步稍迟</strong>。尽管2017年就合作了兔狲文化，但直到2021年8月才正式搭建起以“联合知名厂牌”和“头部UP主加盟”为名头的轻剧场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比于抖快与爱奇艺、腾讯等长视频平台在微短剧上“卷生卷死”的拉锯战，B站的入场就显得颇为“紧锣密鼓”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年6月，B站入股兔狲文化，股份占比10%，这被视为B站加强真人影视领域的投资新动作。而在下半年，B站宣布与“坏猴子影业”达成合作，宁浩等坏猴子影业导演们组团B站报道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在坏猴子影业“S+”2021年度品牌发布会上，B站还发布了首批8部与坏猴子影业合作打造的短片作品，涵盖悬疑、喜剧、爱情、科幻等多题材。据媒体消息，未来坏猴子影业每年将打造25-30部短片登陆B站。二者之间空前的密切合作，彰显了B站加码短剧的决心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在创作者层面，则多次发布小剧场稿件征集活动，开设奖金池</span></strong><span>。以3.0活动为例，开设了热门话题奖与单项奖，包括短剧爆款奖、短剧大师奖、勤奋创作奖等，鼓励创作者释放创意。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但行至当下，相比于在短剧方面已形成成熟产业链的抖快，B站掀起的水花并不足以搅动风云。从作品方面，回看过去，抖音、快手都产出过诸如《做梦吧晶晶》《长公主在上》《这个男主有点冷》《秦爷的小哑巴》等爆款短剧，<span>并塑造出“颖儿”“姜十七”等IP，</span>在社交平台获得一定话题度与讨论度。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反观B站，即便在短剧领域打造出像《片场日记》《抓马侦探》等口碑不错的短剧，但热度大多局限于站内，并未达成全平台出圈。最近所力推的《这个杀手不改需求》，站内总播放量仅78万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8010890056319435254dca9.38959457.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>《这个杀手不改需求》截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>究其原因，在于平台间用户内容消费习惯的差异。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一来，B站自制短剧大都具备大会员限制，相当于另一层面的视频付费，在某种程度上打消了用户观看热情；</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二来，不同于抖快主打的情感与甜宠剧场，B站的短剧主打科幻、悬疑，平台所呈现出的主基调与抖快等平台有很大不同，所面向受众也趋于狭窄；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三来，短剧的特点是短平快，剧情强反转、强节奏，天然与沉浸式竖屏匹配，而B站尽管上线了story mode模式，<strong>但横屏依旧是社区核心，承载的是信息量更足的内容。从这一点来说，B站实际并非短剧的最好载体</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>打出差异化内容或许无可厚非，但值得关注的是，B站奋力加码短剧，还需要成熟的变现链路与“扩音”。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于后者，B站早已在行动：在平台发起的“短剧UP创作营”活动中，就有一则关于达人站外粉丝数量的要求，并设置有累计投稿、累计涨粉奖励。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但对于前者，爱奇艺、芒果、抖快等视频平台所探索的商业变现模式包括流量分账、付费买断等，无法完全适配于B站。<strong>由于创作主体是参与共创的UP主，短剧实际上是作为个人UP的内容延伸，商业变现能力还是与整个B站变现环境密切关联。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总得来说，@王七叶、@小陈总、@吴厘头啊、@非非宇Fay等站外达人频频“上分”，或许意味着B站短剧走入了新阶段。但在短剧领域，B站还有很多有待解决的问题。</span></p>
