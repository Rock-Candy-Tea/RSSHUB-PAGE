<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><span>‍‍</span><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;"><strong><strong>你在米饭里放点肉，叫炒饭。在肉里放点米饭，那就是网红炒饭；你在奶茶里面放芝士，叫做奶茶。但如果你在芝士里面放点奶茶，那就是网红奶茶。</strong></strong></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这几年，我发现一个不大不小的现象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>现在的<a href="https://www.niaogebiji.com/tag/%E8%BE%A3%E9%85%B1" target="_blank">辣酱</a>市场越来越卷了。本来一瓶简简单单的辣酱，现在动不动就是加大块的菌菇、竹笋还有肉末。</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/46797677162de5e266688a9.53874716.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前我是烧一顿饭，拿辣酱配菜吃。现在我是搞一碗白米饭，冰箱里拿一瓶酱，就直接呼啦呼啦下肚了，还觉得自己吃得挺好，有菜有肉的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/153709828762de5e26b5bec0.43344711.gif" width="222" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没准再过几年，我就要担心我的厨艺配不配得上我冰箱里那瓶三十块钱的辣酱了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>辣酱这个东西，正在背叛它「<a href="https://www.niaogebiji.com/tag/%E8%B0%83%E5%91%B3%E5%93%81" target="_blank">调味品</a>」的初心，越来越往「菜」的方向上靠近。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>所以这期，我想从辣酱的诞生和演变讲起，聊聊为什么辣酱会越做越像菜？</span></strong></span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em><em><span><em><em><span>01</span></em></em></span></em></em></span><span><span><span></span></span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>人类制作酱料有着悠久的历史。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美食作家，食物化学家哈洛德·马基曾经在他的巨著《食物与厨艺》中总结过：大部分的酱料都需要破坏动植物的组织，释放出其中独具风味的汁液。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而厨师需要做的是在汁液中加入各种大分子和颗粒，让汁液的流动变缓，让酱料更加浓稠。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/94284933862de5e271420b0.46624862.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果我们咬文嚼字，从语言的角度来理解，会发现不同文化中，酱料的核心要素是不一样的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在英文语境中，酱料<span>（sause）</span>的词根来自于盐，也就是salt。因此在以欧洲为核心的美食体系里，制作酱料，是为了提取风味。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但中式酱料的核心要素，不在于原料本身的味道，而在于发酵。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>中国最早的酱被称为醢<span>（hǎi）</span>，根据《周礼·天官》记载，西周时期，王庭中就有「醢人」职务，为天子掌管祭祀使用的酱料。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种早期酱料的主要材料是各种肉，将肉切成丁末，拌上米饭、曲、盐，用酒腌渍，最后封存发酵，就能形成醢。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据肉类的不同，有「兔醢」、「雁醢」、「鱼醢」等，这些酱还会用来拿去腌制各种蔬菜，形成花样百出的发酵食品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>「酱」这个字产生，则是因为大豆的使用。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/149391501962de5e27683f47.34460657.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当肉类产量不足以满足人们的食用和制酱需求，豆类就成为了肉的代替品，进入了酿造领域。而「醢」字也就变成了专指不以肉为原料的「酱」字。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在西汉初期，「以豆合面而为之」的豆酱制作工艺就已经出现，1972年在长沙东郊发掘的马王堆一号汉墓，出土了大量食物遗存，其中就有以大豆为原料制成的豆酱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>到了东汉，出现了一种从豆酱中汲取酱汁的副产品，名为「清酱」，学者认为，「清酱」就是最早期酱油的雏形。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/78818584662de5e279c6374.22646042.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>南北朝时期，贾思勰的《齐民要术》第一次系统性总结了制酱的方法和酱油制作的工艺。此后，制酱工艺不断完善。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到了明朝，酱油从豆酱的制作中独立出来，形成了独立的生产工艺。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但无论是肉酱、豆酱还是酱油，都离不开「发酵」，具体点说是蛋白质发酵。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当高蛋白食材经过发酵后，会大量产生氨基酸，尤其是谷氨酸。所以发酵过后的蛋白质食物，往往有着非常浓郁的鲜味。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大豆制成的酱油，水产品制成的鱼露和虾酱，风干的火腿腊肉，都存在着发酵产生的鲜味。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>鲜味，才是中式调味酱最核心的特色。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>辣酱的另一个要素「辣」，则是来自于辣椒。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/136986486462de5e27ec6489.29490543.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>明朝，辣椒经由海路，从美洲传入了中国。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时谁也没想到，这种本来只被当成观赏植物的红色果实，后来能对中餐产生如此深远的影响。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>辣椒虽然是从福建、浙江沿海登陆中国的，但真正发挥调味料的作用，却是在千里之外的贵州。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>中国人吃饭讲究调味，而在调味中，又有「五味咸为首,百味盐为先」的说法，盐不仅是保重劳动人民生产生活的必需品，也是中餐里最重要的味觉要素。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/181341518662de5e28582ff0.93372821.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>而贵州，在南方地区是最为缺盐的。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面是因为本地不产盐，一方面则是交通不便，盐价较高。贵州人长期需要能替代盐的食物，而外来的辣椒就成为了其中之一。康熙年间，贵州山民便开始以辣代盐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在吃辣椒这件事上，贵州人是全中国最早之一。所以<a href="https://www.niaogebiji.com/tag/%E8%80%81%E5%B9%B2%E5%A6%88" target="_blank">老干妈</a>这样一个贵州品牌，后来能成为中国辣酱市场的第一名，某种程度上也是历史的必然。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此后，辣椒随着湖广填四川，传遍了西南地区，逐渐成为民间主流味觉的一部分。又通过关中平原从陕西扩散到整个北方地区。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>《中国食辣史》一书中提到，中国人吃辣椒，大体上呈现出一种南酱北粉的特性。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以秦岭为界，北方主流的辣椒加工方法，是干燥后研磨成辣椒粉，可以长期保存，搭配以面食为主的北方饮食。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/206359162862de5e28acb286.72879297.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在南方，由于高温潮湿，当地各种腌制、发酵食物的工艺就被用在了辣椒身上，出现了各种不同的辣椒酱形态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/179138798962de5e2901aa51.25697391.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，在缺盐的地方换用辣椒入酱，本身也可以有效降低制酱的成本，而且辣椒防虫、防霉，可以使酱更易于保存。最终形成了品种繁多，分布广泛的中式辣酱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>辣椒是如何与「酱」的工艺相结合，其实没有一个明确的标志性节点。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这可能是由于发酵是一种非常难以控制的加工方式，因此在不同的气候环境、微生物菌种和物产条件下，辣椒酱呈现出了丰富多彩的形态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如豆瓣酱，就是清代四川人在原有的甜豆瓣制作过程中，加入辣椒混合发酵。而剁椒酱的制作，可以只放辣椒、蒜和盐，湖南的永丰辣酱则是和蒸煮后的小麦一起发酵。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>可以看到，辣酱这个东西，从来就是一种形式很多样的品类。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/78050893162de5e29504ea2.74001388.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>打个不恰当的比方，辣酱这东西有点像茶饮。奶茶就是只要有茶底，其他配料和调制方法可以随意发挥想象，奶茶也好，果茶也罢，形态很多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而辣酱也是一样，只要放了辣椒，其他制作工艺和主料辅料都可以因地制宜，自由发挥。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em><em><span><em><em><span>02</span></em></em></span></em></em></span><span><span><span></span></span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，国内辣酱行业规模260亿元，市场很大。如果把目光放到整个调味酱市场，辣酱的份额要占到8成左右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>从市占率来说，辣酱基本已经统治了中国人吃酱的习惯。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之前在聊川渝火锅那期里，我提到过，<strong>辣味之所以能统治火锅市场，和70年代末开始的农民工进城，以及轰轰烈烈的城市化进程不无关系。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而对于辣酱，这个逻辑一样是成立的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拿老干妈举个例子，去年，这个品牌营收超过50亿，在辣酱领域市占率超过20%，妥妥的行业霸主。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/93285923462de5e29bf3190.25608552.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>老干妈的成功，很大程度上就是客户定位的成功。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你回顾老干妈的创业史，就会发现，早年在贵州卖米豆腐配辣酱的时候，陶华碧的店里就有大量的学生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人认为，这体现了老干妈接地气，贴近消费者。这固然没错，但必须看到的是，学生群体，其实是辣酱最重要的刚需人群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>因此在创业初期，老干妈划定的目标人群非常清晰：进城务工人员，以及大学生。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这两批人有很多共同点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先是年轻，辣味的主要消费群体是青年人，这个之前我聊过，就不多说了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二，这两波人群，都是没什么精力和财力在吃饭上面好好花心思的人。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/82902584062de5e2a71f3c4.45293835.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟大学生和农民工群体，很可能一年到头都下不了几次厨，食堂饭大锅饭才是常态。至于大学生的进化形态，城市白领，也是没时间吃饭的人群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拿我自己举例，虽然我喜欢自己下厨，但是忙起来也未必有时间天天做东西给自己吃。有时候简单煮个面，加个蛋，拌点老干妈提提味道，就是一顿还过得去的午饭了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第三，老干妈的目标人群基数巨大，而且随着城镇化、高校扩招和中国加入世贸组织，这两批人群的基数都处在飞速增长期。而老干妈恰好就是那个时代起步的。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到2019年，老干妈的国内市场覆盖率已经超过了96%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>最后一点，就是消费力不强。</strong>这个不用解释，大学生吃生活费，农民工攒钱带回家，能用来花销的钱不多。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/74420483662de5e2b666483.64283500.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而老干妈的定价，自然也是锚定目标人群的消费力的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它长期把价格卡在10元上下的位置，而且很少涨价。这个定价策略很符合它的定位：<strong>大众市场，注重性价比，薄利多销。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/167651967462de5e2b9fb1e6.82185005.jpeg" width="444" referrerpolicy="no-referrer"></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多时候，我们普通消费者对老干妈有感情，就是因为它好吃不贵。这一点还是很值得瑞思拜的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是对同行来说，那就未必了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>而对于一个行业来说，先手玩家的定位，往往决定了整个行业的定位。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>很多时候，我们会发现一个行业第一名的品牌，往往会承担一个定价天花板的角色。也就是通过品牌力，将产品卖得贵一些。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对品牌自身，可以获得更高的利润。而对于整个行业，也可以抬升消费者的价格预期，从而留出中低端市场的空间给其他品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老干妈在辣酱竞争里占据先手，率先把老干妈陶华碧的品牌植入了全国消费者的认知里。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但另一边，老干妈卖得又挺便宜，这对同行们就很尴尬了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在辣酱行业里有一个老干妈魔咒——比老干妈便宜的就没利润，比老干妈贵的，你又没有老干妈那么知名，就容易没人买。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正是因为老干妈占据了一个几乎无法被颠覆的生态位，才弄得友商们上也不是，下也不是。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>如今辣酱市场是扎扎实实的「一超多菜」格局。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在老干妈以下，无论是调味业巨头海天和李锦记，还是中坚品牌辣妹子、饭扫光、仲景，或者是新晋网红品牌虎邦、饭爷、李子柒，这里面没有一家的辣酱，市占率超过10%的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/132873470662de5e2be9ece6.27939443.jpeg" width="444" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em><em><span><em><em><span>03</span></em></em></span></em></em></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何绕过老干妈，在市场里另辟蹊径，成为了各路诸侯必须考虑的问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>把辣酱做得更像菜，就成为了其中的一个选项。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>今年年初，我和一个商业媒体的前辈在群里聊天，他说</span></strong><span><strong>这几年中国的新消费品有一个特质，特别容易产生网红款，那就是不成比例地放大某一项原料的比重。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你在米饭里放点肉，叫炒饭。在肉里面放点米饭，那就是网红炒饭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/144205733562de5e2c37da12.86345498.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你在奶茶里面放芝士，叫做奶茶。但如果你在芝士里面放点奶茶，那就是网红奶茶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>本质上就是利用中国强大的供应链堆料，明显放大用户感知。</strong></span><strong><span>然后你就可以把产品卖得更贵了。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而现在的网红辣酱，恰好用的也是这一招。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>前面聊辣酱历史的时候我们也说了，</span></strong><span><strong>辣酱本身就是一种跨界属性很强，创新门槛不高，很适合发挥想象力的品类。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人觉得老干妈不够辣，那我们就出魔鬼辣。有人觉得老干妈里缺点口感，那我们加大块的牛肉粒和菌菇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，板筋、鲍鱼、松露、松茸这些更高端的食材，也被大量加入辣酱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>于是，我们看到现在的辣酱越来越像预制菜。</strong><strong>有荤有素，又香又辣，配上主食好像啥也不缺了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种做法好不好另说，但确实可以绕开老干妈的品类和定价区间，想办法把辣酱的单价往上卖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>除此之外，越来越多的品牌也在想办法用其他的方式绕开老干妈。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如2015年创立的虎邦辣酱，这家公司诞生的时候，恰好是外卖市场刚刚兴起。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是这家公司决定，把外卖场景当成公司的核心战略，和外卖平台上最主要的连锁品牌达成合作，为这些品牌供应辣酱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/140286662462de5e2c8b50f9.83599250.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟在商超和老干妈正面对抗，基本就是找死。<strong>通过外卖，在渠道和消费场景上规避老干妈，虎邦很快建立起了品牌认知。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年「双11」期间累计销售690万罐，成为线上全平台辣酱品类销量第一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>也有走网红IP路线的。</strong>比如林依轮创立的饭爷，依靠着林依轮本人的名气，和他本人作为厨师美食家的专业背书，走的是高端食材消费升级的路子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有就是李子柒，虽然她本人深陷和微念的IP纠纷，但不妨碍这个IP本身还是可以拿来带货。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/92248383362de5e2cdfd165.76298236.png" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>靠着精致的国风包装和良好的口碑，李子柒的辣酱卖得也还算不错。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/149681672962de5e2d4ca7f8.21841889.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这轮疫情过后，想必在家里多囤几罐辣酱的人也会越来越多，毕竟每天做饭、洗碗还是一件很累的事情。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在外卖不方便的时候，偶尔偷偷懒，一碗挂面，一根生黄瓜，一碟辣酱， 也是一种不错的选择。</span></p>
