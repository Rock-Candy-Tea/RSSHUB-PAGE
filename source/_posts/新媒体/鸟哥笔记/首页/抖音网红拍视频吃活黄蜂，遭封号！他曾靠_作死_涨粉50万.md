<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>你敢相信吗，现在有的网红博主，为了博取流量，为了涨粉！竟然生吃活黄蜂！带刺的那种哦。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事情是这样的，<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3" target="_blank">抖音</a>网红博主@王灿有点哈数 发布了一条视频，视频开头他说：“最近点赞不够，霸王汉来凑”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接着就开始自己的“作死之路”，他用筷子夹住活着的黄蜂，而且是两只，直接扔进了嘴里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接着画面忽然一变，他的嘴和脸都被黄蜂蛰到了变形，嘴巴肿成了“香肠”，眼睛睁不开。这条视频引起很大争议，画面令很多人觉得不适。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18629058396317b982867fc8.15632522.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>凭借这个“作死”的视频，他迅速上了微博热搜，引起轰动。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>热搜火了后，9月4日晚上，或许是为了借助这波热度，又在抖音开直播，在线人数最多时超过了10W人，人气爆棚。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>但还没火够一个晚上，直播戛然而止，王灿被封号（账号主页显示空白，应该是永久性的）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6843474186317b982e1b8d3.20080362.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实这不是他唯一的与黄蜂有关的视频，事实上他发布的视频基本都是和黄蜂/马蜂窝有关的，只是之前拍摄的都是恶趣味搞笑视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每个视频开头，</span><span>都会用带有浓厚的</span><span>当</span><span>地口音</span><span>说一</span><span>句</span><span>：</span><span>“</span><span>大家好，我叫王灿。</span><span>吃饱饭，找不到事情</span><span>干。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种口头禅/口号</span><span>是为了让观看者更容易记住他的身份，也是业内所说的身份标签和定位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20096163716317b9835bbda4.01398186.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频中，他在乡村四处各地寻找蚂蜂窝，然后用各种方式捅马蜂窝，抓马蜂等，例如用一个塑料袋迅速把蚂蜂窝给套进去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但他的目的不仅仅是抓马蜂那么简单，毕竟观赏性不高，没什么吸引人的点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在套住蚂蜂窝后，他迅速打开塑料袋；</span><span>又或者用鞋子拍蚂蜂窝，然后自己一溜烟跑掉，</span><span>让马蜂去蛰摄影师，制造出了猎奇狗血的结果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种“作死”的行为，要么是摄影师被蛰，要么失手，两个人一起被蛰。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15311376416317b983af6936.92068507.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9555553806317b984a1ffe5.97345926.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且很夸张的是，他的嘴都被蛰成了“香肠嘴”，竟然没有去医院，而是再次去拍打马蜂窝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14594421696317b985c6e469.81546168.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>凭借这些视频，他的单条视频点赞数达到几万，甚至数十万，账号粉丝数超过50万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6843474186317b982e1b8d3.20080362.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于他吃活黄蜂的“作死”行为，沸点视频的微博评论区，大多数人都是不支持。</span><span>然而抖音的相关视频评论区，多数人却觉得很是可惜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12976680426317b986da6974.83110260.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>觉得粉丝做到那么多不容易，白费（时间）了。还有的觉得他是搞笑博主，他是凭实力火的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9694106736317b987203826.04858389.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/10274001196317b9877de952.65190167.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我观察到，王灿的这些恶趣味视频中，都会提示“视频存在风险，请勿轻易模仿”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一些自媒体博主们可能觉得，这都已经提醒过了，为什么还要封号呢？我想原因不仅仅是有风险这一个理由。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>主要是因为吃蚂蜂窝的那条视频影响传播的太</span><span>广，上了热搜，画面极端，又有不良风气。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span>活体黄蜂进喉咙，可能会致死的，这种故意伤害自己的行为，抖音应该是不允许存在的，这样导致账号被封。</span></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它虽然是看恶趣味搞笑视频，但没有正向引导价值。画面虽然不是血腥的，但确实令人引起不适。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果他改变一下方向，传递一些正能量或者乡村文化。例如教人怎么抓黄蜂（不过他后来的视频主题，都是拍砸锤马蜂窝，然后跑掉，作弄摄影师，实际上并没有什么专业的技巧）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也可以传递和黄蜂有关的知识，就不一定是这个结果了。但这样的话，关注他的人可能不会那么多了。</span></p>
