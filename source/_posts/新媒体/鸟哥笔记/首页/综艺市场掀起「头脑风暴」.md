<section>
<section>在一栋远离尘世的海边公寓里，怪事每天上演：神秘的红衣坠楼人、夜里女人的哭声、奇怪的死亡警告、总会悄然出现的黑猫……一环套一环的故事、反转再反转的观感，令人忍不住想跳到大结局解锁剧情真相。不过这并不是某一部悬疑剧的情节，而是腾讯视频综艺《开始推理吧》中的关键线索。</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这档综艺似乎不太一样：一季31期、一周3期，这是综艺的更新周期、网剧的更新频次；具体到故事中，案件的发生地“11号公寓”，是超1000张图纸所构造出的商住一体楼，超20万字的剧本串联起了100个人物，单拎出来，每个人都有完整的故事线。可以说，《开始推理吧》是以悬疑剧思维呈现的沉浸式剧情推理综艺。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">严丝合缝的故事逻辑是剧情推理综艺的关键，娱乐性和代入感又是真人秀综艺不可或缺的要素。为了最大程度平衡硬核与娱乐，《开始推理吧》每一期故事基本都推翻8-9次才打磨出最终的版本；上线之前的用户试看阶段，分别根据推理玩家、泛娱乐用户、艺人粉丝的反馈调整剪辑了5个版本。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">节目的播出表现也超出了主创团队的预期。腾讯在线视频节目内容制作部天相工作室群高级总监邱越分享了一个意料之外的观察：《开始推理吧》倍速和跳出情况非常少，反而是暂停回看比较多，甚至有不少用户一边看一边做笔记。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这不禁令我们好奇：一档能让用户聚精会神的综艺到底是如何诞生、如何生长的？这种模式能够常青吗？脑力密集型的「推市元宇宙」到底要如何构建？<br><br></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/183915488562af4a9692c0e7.41481175.jpeg" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">沉浸式剧情综艺的创想</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要想在推理综艺这个成熟赛道里创新，并不是件容易的事。从芒果TV的《大侦探》《密室大逃脱》再到爱奇艺的《萌探探探案》《奇异剧本鲨》，推理节目不断填充着综艺市场的版图，此外悬疑剧的涌现与迭代，也拔高了用户对于烧脑内容的期待值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过，在推理内容的播放量和市场反馈达到高水位的情况下，此时入局的《开始推理吧》并不是“白手起家”，但相应地，节目的差异化定位和创新能力也面临着更高的要求。邱越告诉「深响」，《开始推理吧》一开始就围绕着一个策略来创作：从高原上找峰值，为核心推理用户提供升维体验。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通过分析大量国内外推理方向的剧集、游戏，主创团队发现了一条蹊径——在一个时空中讲述一个长线剧情的推理电视剧更受大众欢迎，比如日剧《轮到你了》就是如此，但这种模式很少在综艺中见到。所以，邱越与导演团队立即达成一致：连续剧情推理综艺就是峰值点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在邱越看来，单本剧式更为流行的推理综艺中，连续剧式无疑是新鲜的视角，对于用户的吸引力也明显更大，“当你有了一个更好的想法，你就无法退回到只是差不多的选择”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213853297962af4a96be6be0.30405693.jpeg" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这的确令人兴奋，但知易行难，各种挑战立刻摆在眼前。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首先在故事层面，相比单本式推理节目，连续剧式综艺最重要的是要有一个完整的故事架构。《开始推理吧》的悬念起始于红衣人坠楼，最终想要表达怎样的真相，这就要梳理出“因”与“果”的关系。在故事的大框架确定后，新的人物和一个个案子是让骨架更丰满的血肉，故事与故事之间要彼此衔接呼应，同时是层层递进的逻辑，能够为最终的谜底做铺垫。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这对编剧的要求非常之高。据了解，《开始推理吧》的编剧在每一期故事之前就有4-5次讨论， 而在确定故事、拆分线索的流程环节，也会讨论3-4版，因此呈现出来的每一个案子背后都有着8-9次的“推翻再重来”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/88790260762af4a978b6c72.36054276.gif" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其次在制作层面，由于故事“复杂”了，内容“丰富”了，制作起来的难度也随之增加。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同于很多在棚里录制的综艺，造型独特的11号公寓楼给了节目一种全新的故事架构方式。一方面公寓就是一个微型社会，浓缩了现实世界里的诸多元素，能让故事有深度发展的土壤；另一方面，公寓的地理位置约束，也给容易过度发散的推理故事画了一个范围，让所有的剧情凝聚在这里，不至于散掉。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/195318979062af4a9cb59174.45313501.jpeg" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/64719870262af4a9cecf7d1.22969844.jpeg" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><br>场景之外，嘉宾的选择与呈现对推理综艺也有一定的加成作用，是否能够对线索做出快速反应、嘉宾之间能否碰撞出火花以及导演团队能否找到艺人的独特定位，这都会对节目效果产生影响。《开始推理吧》每个角色的设定都会结合艺人的兴趣和想法，不同角色的创作都有着与艺人深度沟通的过程。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最后在用户层面，因为连续剧式综艺在国内相对而言还是一种新鲜的模式，如何平衡逻辑推理的难度、以及如何满足核心推理受众和泛娱乐受众的需求，这都是主创团队在上线之前审慎思考的问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据邱越介绍，上线之前，《开始推理吧》面向硬核推理玩家、泛娱乐大众、艺人粉丝等不同群体的反馈进行了五次的修改调整，最终播出的第六版是综合了多方意见的版本，“让核心玩家和泛娱乐大众都能各取所需”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/55289544562af4a9d3d91a8.72160779.jpeg" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">体现在节目中，第三个案子“巨好家庭”通过复杂的人物关系线和多重的作案手法，为推理用户带来了参与感及刺激感；对于泛娱乐大众而言，嘉宾不经意的抛梗接梗，后期制作的三维、二维动画，也平衡了节目的严肃性，帮助了泛娱乐用户更好地理解案件走势。在社交平台上，也能明显看到不少观众在自发更新自己的推理思路、整理人物之间的关系。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">虽然“工作量指数级上涨”，但从数据反馈来看，《开始推理吧》也展现出了观众对连续剧式综艺的热情。据灯塔舆情热度榜单，第三案播出后，《开始推理吧》当日热度突破316114，位居热度第一名。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“原来时隔一周更新一期综艺，观众需要观看前情提要部分去复盘，但这档节目的前情提要和预告的部分跳过的比例更高，这意味着观众是连续看的。”邱越表示。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/162701149862af4a9d7f3907.44806237.png" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3928721562af4a9dbf6282.54649770.png" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><br>归纳下来，连续剧情式综艺相比单本式更创新、难度也更大，聚焦到《开始推理吧》，其做好的关键在于设置了一个完整的故事架构之后，无论是场景、嘉宾还是线索都是围绕着大的故事创作。从结果来看，这样的创新表达也为推理赛道注入了新的活力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213179696861420d317ab820.36069667.jpeg" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">从“综艺”到“宇宙”</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">《开始推理吧》只是腾讯综艺在推理赛道的起点。未来，涵盖了“异次元博弈游戏”的《登录圆鱼洲》，聚焦女子推理的《她他它真探社》等推理综艺都会在腾讯视频上线。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其中，《登录圆鱼洲》的灵感来源于腾讯视频今年初热播的剧集《开端》。当时团队敏锐捕捉到了“无限流小说”这个流行元素，落地到《圆鱼洲》中，一方面，通过抽离时间概念，以博弈游戏为新的切入点引发人物关系的碰撞；另一方面，“元宇宙”的概念贯穿，也延展了综艺的想象边界。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据邱越透露，《开始推理吧》《登录圆鱼洲》《她他它真探社》三档综艺并非推理综艺IP的排列组合，而是早已预埋了一个整体的世界观——推市。《开始推理吧》中的11号公寓所在地“推市”是一个平行于现实的城市，公寓中的房客及管理员在微博上设置了小号来与观众互动，增加了角色的真实感。接下来的《登录圆鱼洲》《她他它真探社》也将会围绕“推市”进行叙事，且每个综艺里的角色也会彼此连接、穿越。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/93760835062af4a9e6f3b71.94269717.jpeg" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从单个IP到组成一个推理的垂类赛道，拆解腾讯综艺布局的底层逻辑，这更像是以打造“IP宇宙”的方式深耕推理品类——外在看是一个又一个的IP集合，内在看每个IP又在同一个世界观下。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实上，IP宇宙并不是新鲜事物。无论是海外的迪士尼宇宙、漫威宇宙，还是国内的唐探宇宙、封神宇宙、华夏古城宇宙，不断挖掘IP的衍生价值来延长IP的生命周期，已经成为越来越常见的一种做法。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过，国内外已有的IP宇宙中，更多是以动漫、电影、剧集为主，很少有综艺的IP宇宙。在邱越看来，按照IP宇宙构建的难易程度来看，动漫是最容易形成宇宙的内容形态，其次是电影、剧集，最难的则是综艺。因为综艺离现实更近，很难构筑庞大的虚拟世界观。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">相对而言，在诸多综艺题材中，推理综艺又是比较容易架构“IP宇宙”的题材。首先，推理自带IP基因，且这类内容更强调想象力和脑洞，打造出一个虚拟架空的世界观既合情合理，也能够规避“综艺离现实更近”的难题。其次，剧本杀这一新消费业态的火热，也让推理综艺有了落地的承载，可以带动线下衍生的消费，商业价值也更具想象空间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样的情况下，构筑起一个“推市元宇宙”，既契合了年轻人的流行趋势，也透露出腾讯综艺在IP规划上的前瞻性和长远性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/170720616062af4a9ea39e89.25335143.jpeg" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">操盘一个大的IP宇宙实则并不简单，需要主导者长周期的投入、积累和实践，为什么相比起海外，国内的IP宇宙“入局者多出圈者少”，其核心就在于大多数的主导者并没有把握到IP宇宙的精髓，要么打着“宇宙”的噱头，将同一类型的内容进行简单罗列；要么急于求成过度挖掘IP价值，反而损害了IP。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前腾讯综艺在推理赛道做了衍生短剧、NFT产品、长音频、研发云交互的互动类产品，同时还在筹备线下密室实景业务，各种玩法相互协同，辐射更广泛的用户，同时增加用户粘性。相当于从一开始就明确“顶层设计”，做好多元化衍生开发的规划，避免了先有作品再构想IP宇宙的“别扭”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“这其实就是以终为始的逻辑，在一个领域里面做内容，不是因为它特别火就去做，烧一轮钱就结束，而是这一类内容对于用户是有长线价值的。”邱越说。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/90438255962af4a9eec8d42.70211331.jpeg" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">《开始推理吧》衍生短剧</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而且，借助多档定位不同的综艺，腾讯综艺的“推市元宇宙”将推理式游戏综艺的玩法与流行的“剧本杀”消费业态、“元宇宙”概念相结合，这样可以进一步释放IP的商业价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为流行的消费业态可以带动一类内容题材的兴起，一类内容题材的火热又会辐射更多人参与到新的消费业态中，进而提高这类内容的传播力，比如脱口秀、露营、滑雪，其实都属于“线上与线下相互发力”的内容形式。双管齐下，才会不断推高着一个垂直品类的行业基准线，并走向更广泛的受众群体。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/92131005862af4a9f579249.86900105.jpeg" title="综艺市场掀起「头脑风暴」" alt="综艺市场掀起「头脑风暴」" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">都说内容行业是“苦行业”，观众口味实时变化、用户审美不断提高，从来不存在躺赢一说。节目要想立足、超越，就必须不断创新，且敢下决心。切入作品林立的脑力推理赛道，腾讯综艺也是拿出了创新与决心，虽然是首次切入，但我们看到了成熟的思考、高水准的制作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而推理赛道也值得这样的投入与耕耘——越来越多的用户基础、越来越强的内容影响力，在更完备的开发思路下会释放越来越大的价值，从而形成正循环。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">《开始推理吧》只是一个开始，在此之后，更多的<a href="https://www.niaogebiji.com/tag/%E5%A4%B4%E8%84%91%E9%A3%8E%E6%9A%B4" target="_blank">头脑风暴</a>正在酝酿。</p>
