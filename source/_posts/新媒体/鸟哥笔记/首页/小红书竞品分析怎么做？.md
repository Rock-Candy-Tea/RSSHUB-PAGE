<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最近发现一个问题 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有相当多的品牌是不会做竞品分析，或者不知道怎么做竞品分析的，完全不知道该从何下手，但竞品分析又是制定投放策略必不可少的一环，因此今天专门拉出来谈谈，做竞品分析时的几个关键步骤，和关于竞品分析的一点想法</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>竞品分析真的很重要 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">关于他到底有多重要这件事应该不需要我过多的赘述了吧，就像是打仗之前咱得先研究研究对方的兵法，见招拆招，才能制胜</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在做一切的企业动作之前，我们也是要先对对手有个大体了解，才能有的放矢的制定本品牌的投放策略，达到事倍功半的效果</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">一、品类分析</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在分析竞品之前，我们首先要对整个行业品类进行一个准确定位，根据整个行业的声量和热度，来预估我们的市场规模</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以电动车行业为例：当前，在小红书上“电动车”这一品类的相关笔记共76万多篇，品类整体的声量比较大，大家的关注度也比较高</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/ae71f3fbfec5315779741343d709648c.png" alt width="220" height="197" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前行业中的品牌不少，但是头部的品牌数量却不多。从关键词热度排名来看，当前热度TOP3的品牌分别是：雅迪、小牛和爱玛。因此，对于电动车行业来说，这三个品牌无疑是我们首选分析的竞品</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">二、竞品分析</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">竞品品牌确定了，具体该从哪些角度进行分析呢？这里主要从日常工作最常用的竞品投放策略和专业号运营这两个维度重点展开叙述</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这一part主要是对已经选择的竞品的投放策略和效果进行一个拆分，主要竞品分别投了多少达人？投的是什么量级/领域的达人？投放的内容以什么形式为主？主要在打哪些卖点？竞品专业号运营情况？这些我们都要分开拆解来看</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1）投了多少达人 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">竞品达人的投放数据可以在灰豚数据平台查看。以MAC魅可为例，在数据平台搜索栏进行搜索，然后选择我们需要查看的品牌</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/96f3151405af075f554dd12d25cfbcbf.png" alt="图片" crossorigin="anonymous" data-galleryid data-ratio="0.3662490788504053" data-s="300,640" data-src="https://mmbiz.qpic.cn/mmbiz_png/C1ptrBKcREqkgiasCMwSJtC4GI6DcsTFAPtsOicvsyiaNgKPeIOUYlKUZ7StxYVwLEXA0kCbrqmUm9K45icibsXiaYrA/640?wx_fmt=png" data-type="png" data-w="1357" data-fail="0" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在品牌页下的品牌简介中，就会直接显示当前MAC魅可的推广达人数，可以看到，当前MAC魅可在小红书上一共推广了1286位达人 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/e5d29d3c97a838c22a1f19ece2f45a1a.png" alt="图片" crossorigin="anonymous" data-galleryid data-ratio="1.6733333333333333" data-s="300,640" data-src="https://mmbiz.qpic.cn/mmbiz_png/C1ptrBKcREqkgiasCMwSJtC4GI6DcsTFAPjFA1zf4Fk86rajsceK95v7kEBIyAZuZwvW78qdyWGx9gk0iaMWfAOw/640?wx_fmt=png" data-type="png" data-w="300" data-fail="0" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有了竞品达人的投放数量，就可以大致确定我们的达人投放规模。如果想要攫取竞品的市场份额，那么我们就要尽量投放能超越他们声量的达人规模</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2）投的是什么量级/领域的达人？ </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但数量并不决定一切，这时候就还要再关注一下对方投放的达人构成：看他们投的是什么量级的达人，基本处于什么领域 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里我们可以看到，当前MAC魅可投放的达人中，10w-50w粉丝的腰部达人投放占比50.5%，是投放的主要力量；其次，粉丝量在1w-10w的底部达人占比也有38.68%，也就是说他们的达人投放以腰部+底部为主</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/1e7f37d1ed5240be3ab2eca6d5a0c442.png" alt="图片" crossorigin="anonymous" data-galleryid data-ratio="0.5410628019323671" data-s="300,640" data-src="https://mmbiz.qpic.cn/mmbiz_png/C1ptrBKcREqkgiasCMwSJtC4GI6DcsTFAa741s2dlO2ic0HWtJogYFlXUNYBr3Wun6wsqXxyqicdFsuMSYsWQmmzw/640?wx_fmt=png" data-type="png" data-w="621" data-fail="0" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其次，我们还要去看一下达人的领域，这里MAC魅可的达人主要投放领域集中在彩妆类，这也与彩妆品牌的产品特性相吻合，但除此之外，他们在时尚、生活记录、摄影、减肥等领域的达人投放也高达10%，因此在我们的投放策略中，也可以看看是否可以和其他领域的达人开展跨领域的合作</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/75a4e4c871852ede561ad7e3cbdd2757.png" alt width="352" height="258" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3）投放的内容以什么形式为主？</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">关于竞品的内容的投放形式，我们需要从平台分别将相关数据导出</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/0c9ac3291d4ac8ce5b0cf10faa1ac1bc.png" alt="图片" crossorigin="anonymous" data-galleryid data-ratio="0.4720048163756773" data-s="300,640" data-src="https://mmbiz.qpic.cn/mmbiz_png/C1ptrBKcREqkgiasCMwSJtC4GI6DcsTFAu0Knbu5dz372WuFAKLOXV81jLibAZSDAeLVC1lRibMh2RN1CxWgakqdg/640?wx_fmt=png" data-type="png" data-w="1661" data-fail="0" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">导出之后，我们就可以得到当前MAC魅可投放具体数据：视频笔记共投放701条，图文笔记共1172条。也就是说MAC魅可投放的视频：图文的比例为4：6，相对比较的均匀 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>4）主要在打哪些卖点</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">无论是选择达人还是定位笔记，主旨都必须要围绕自身产品的卖点，因此我们必须要明确我们自身的产品优势。这个优势怎么来呢？除了要考虑到我们产品的本身特性之外，我们也要从竞品出发，打造卖点</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如最近特别火的彩妆品牌“into you”。该品牌打造独特的“唇泥”卖点，就是看到了目前市场上同价位口红质地厚重、粘嘴这一问题，不断的宣传自身唇泥独特清爽的质地，得到了一大波种草</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此对于竞品的分析，我们要关注他们在行业内的领先优势，也要看到他们产品目前在市场上反馈出来的劣势，扬长避短</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>5）专业号运营情况 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于专业号运营情况分析的主要是去看竞品专业号运营的内容、更新频次、福利活动等基础数据</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通过对竞品的专业号运营情况和互动数据的对比分析，我们就可以对他们的专业号做一个基本的判断。结合竞品的品牌定位，看他们在专业号上打造的人设、梳理的形象、传递的理念分别是什么，根据他们的数据综合分析得出我们的专业号运营方向</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">假如我们的竞品账号以视频内容为主，但是数据效果并不好，那么我们在专业号运营时就可以考虑是不是要多发些图文内容</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，还可以关注竞品专业号的福利活动，找到他们的活动时间和内容，再去分析他们的活动数据，如果可以找到一般规律，那么就可以为我们的运营提供借鉴</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">三、总结策略</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通过上述两部分分析，我们应该已经足够可以知道，我们的竞争对手们到底在干啥 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对竞品进行分析，其实最终还是要回归到我们自身品牌究竟该怎么打这个问题上来</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里基于竞品分析的数据情况，提出几点建议，以供参考</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1）另辟蹊径</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当分析完之后，我们发现市场上的竞品体量都比较大，声量也比较高的时候，这时候就不建议在已有赛道下功夫了，还是应该尽快寻找新的渠道赛场，看看能不能与其他赛道进行结合，打造自己的赛道</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2）细分场景</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当前，市场的细分程度越来越高，基于细分市场去做相关的场景体验看起来更有胜算。例如“户外”的版块，不断开辟出了“露营”、“飞盘”、“滑板”等等，这都是当下新兴的热点</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3）升维打击</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上面两种方法其实虽然形式上不一样，但是中心意思都是：不要硬刚</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是如果真的遇上了势均力敌的对手了怎么办呢？还有一个思路是——升维打击。如果不能在产品使用方式和场景等方面开辟新赛道，可以从情感、品牌理念、生活方式上入手，站在另一个维度上看看可不可行</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于小红书上的品牌竞品分析，这里并没有提供特别完备的方法论，除了上述提到的主要信息之外，在日常的竞品分析过程中，还是应广泛的了解。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总之，知己知彼百战不殆</p>
