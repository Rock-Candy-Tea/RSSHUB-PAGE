<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首先我们要搞明白什么是直播推流？</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">直播推流是把采集阶段封包好的内容传输到服务器的过程。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">简单理解就是，直播推流是需要先将内容上传到服务器，然后服务器将内容输出给观众看。<span style="font-weight: bold;">可以通过OBS这类第三方的软件，来实现更多丰富的功能。</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 428px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/07/8rN386d1zCaz6Kuba4KY.png" alt width="403" height="1018" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当下为了让观众有更好的观看体验，直播推流也成为了各主播的标配。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如，我们在直播中可以调整画面比例，将PPT、图片、文字等内容放置在画面当中，从各方面来提升直播的效果。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过很多人都会误以为直播推流就是推动直播流量，其实和流量没有任何关系，也不影响平台的流量分发。</p>
<h1 id="toc-2" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; margin: 25px 0px 20px; line-height: 30px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-size: 16px 20px; font-weight: 600; text-align: center; background-position: 0px 5px; background-repeat: no-repeat;">02 直播推流的好处是什么？</h1>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总结直播推流3个方面的优势：</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">1、可以提升画面质量</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果直接用手机直播，那直播的画面质量受限于手机摄像头的清晰度。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">使用直播推流可以借助更多专业的设备，来提升直播的画质</span>，还能随意调整直播的画面比例，达到最满意的状态。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">尤其对于带货为主的直播间，需要对商品进行特写展示，使用高清的摄像头设备，能更好地展示商品，直播间观众的购物体验也会更好。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2、方便内容展示/引导观众</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">使用直播推流，我们可以实现很多实用的功能。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如在带货类型的直播中，可以<span style="font-weight: bold;">在画面中加上产品的图片、价格、文字等信息</span>；或是在线课程这类的直播，可以<span style="font-weight: bold;">同步展示PPT内容。</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外还能<span style="font-weight: bold;">在画面里加上二维码或其他引流信息</span>，通过福利诱导，节省主播的口播时间，引流效率更高。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 371px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/07/02guXDFMGWBFbpoBwPp1.png" alt width="367" height="874" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">3、满足多种直播需求</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">直播推流不仅限于直播的场景，还可以<span style="font-weight: bold;">进行影视类的画面直播，新闻/赛事的转播，还能上传视频文件，完成录播的需求。</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">相比于直接用手机或电脑直播，使用直播推流可以满足更多需求，<span style="font-weight: bold;">能随意更换直播背景，方便绿幕直播等。</span></p>

<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">打开OBS，点击「设置」–「推流」，将“服务”设置为「自定义」，再把视频号推流地址和推流密钥分别复制到「服务器」和「串流密钥」中。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/07/CuumbWW6UnhsIBX0l52W.png" alt width="2464" height="1522" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再点击左侧「视频」，设置分辨率。视频号直播，我们一般将分辨率设定为1920*1080。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分辨率设置的数字越高，画质就越好，但对网速、电脑配置的要求也会越高。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/07/YIt3RN7QfkQIz4GcYlU9.png" alt width="1916" height="698" data-action="zoom" referrerpolicy="no-referrer"></p>

<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在正式推流之前或推流过程中，我们可以在右下方“控件”开启工作室模式，分别会出现「预览」和「输出」两个画面。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">然后我们可以在左边预览进行画面的编辑，点击「转场特效」，将预览画面转到输出窗口，观众就能在直播间看到编辑好的画面。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/07/MhYlp4rLyrzqMCvxOuOi.png" alt width="2464" height="1404" data-action="zoom" referrerpolicy="no-referrer"></p>

<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">全部设置完成后，在“控件”下方点击「开始推流」，然后回到视频号后台开启直播，直播推流就算正式开启了。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">直播结束之后，先在视频号助手后台点击「结束直播」，再到OBS后台点击「停止推流」，切记注意顺序，否则观众会看到黑屏画面。</p>

<h1 id="toc-4" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; margin: 25px 0px 20px; line-height: 30px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-size: 16px 20px; font-weight: 600; text-align: center; background-position: 0px 5px; background-repeat: no-repeat;">04 直播推流的常见问题和技巧</h1>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1、推流直播画面卡顿或者模糊</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果直播推流出现卡顿，就将视频比特率进行调低，如果画面模糊就把分辨率调高，FPS 值建议在 20-30 之间。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2、推流失败</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果提示“无法连接服务器”，大概率是推流地址填错了，请检查格式“rtmp://xxxx”只有一个“://”或者检查一下网络。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">3、直播出现回声</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">检查OBS混音器的音频输出源，关闭其他设备，保证只有一个输入设备。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">4、开播前提早上架商品</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前电脑端推流直播，在开播前不能提前将小商店商品上架到直播间，因此在直播开播后需尽快通过手机端上架商品</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">5、绿幕的使用</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果使用绿布抠除背景，则需要注意尽量不要展示绿色封面的商品。</p>                        <div style="height: 1px"></div>
