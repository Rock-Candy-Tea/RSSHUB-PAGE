                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本文要点</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">榜单筛选规则：2022年7月粮油调味/速食/干货/烘焙->调味品/果酱/沙拉类目下销售额大于150万元，同比增长超100%的品牌。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">高增长品牌归因：</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">疫情之下，外出餐饮需求部分转为家庭烹饪，带动调味品市场整体向好。</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">快节奏生活下的懒人经济和越来越庞大的“厨房小白”群体带动口味丰富、快速成菜的复合调料快速增长。</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">健康东风吹向调味品市场，低卡低脂是“财富密码”，不止是轻食酱汁，火锅底料也开始“健康化”。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">未来趋势：</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">诉求健康化、方便快手化、风味多样化或成未来调味品的发展方向。</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">线上C端市场的可能性将被进一步挖掘，B端、C端双线并行或成上游商用调料供应商的全新尝试。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据艾媒咨询数据，2022年我国调味品市场规模预计达到5133亿元。疫情之下，餐饮消费中的外食需求部分转向家庭烹饪，而越来越缩小的家庭户型和越来越庞大的“厨房小白”群体，刺激C端调味品需求，尤其是复合调味品需求激增。就B端而言，餐饮连锁化趋势下，对标准化口感的追求和把控人力物力成本的要求，使各大连锁餐饮对调味品，尤其是专业复合调味品依赖程度进一步提升。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据魔镜市场情报，2021年天猫淘宝平台，调味品/果酱/沙拉这一类目销售额达到144.7亿元，销售额同比增长1.2%，相较于2020年32.6%的同比增长虽有滑落，但依旧维持了连续五年的正向增长。当疫情带来的红利开始退潮后，线上调味品市场还有哪些品牌逆势增长、依然生猛？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">今天的魔镜<a href="https://www.niaogebiji.com/tag/%E6%96%B0%E5%8D%87%E6%A6%9C" target="_blank">新升榜</a>就为大家盘点2022年7月粮油调味/速食/干货/烘焙->调味品/果酱/沙拉类目下销售额大于150万元、同比增速超过100%的高增长品牌，看看哪些品牌正凭借特色产品快速增长？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://p3-sign.toutiaoimg.com/tos-cn-i-qvj2lq49k0/1fde2c0b432b4cefa600670cadb99405~noop.image?_iz=58558&from=article.pc_detail&x-expires=1662086660&x-signature=5YU5omhWIpJcuCX0bm3RzQ%2Bl3Zc%3D" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">01、仟味：从B端到C端，探索藏在家庭烹饪里的增量市场</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7月调味品新升榜中，仟味凭借530万元的月销售额位列榜首。仟味专注于以广东高汤为代表的汤品调料，作为上游汤品底料供应商，服务于连锁品牌和餐饮门店，多年来垂直于B端客户。近年来，其瞄准家庭烹饪场景，开始进行C端尝试，推出脸谱装系列产品，即规格为50g或120g或200g的家庭小袋装。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://p3-sign.toutiaoimg.com/tos-cn-i-qvj2lq49k0/7e584f17c6b5436886ff37ac1a3c6bb2~noop.image?_iz=58558&from=article.pc_detail&x-expires=1662086660&x-signature=bc%2F4lYN4F3hAZoj7l4gDM1X5Uqk%3D" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">图片来源：仟味天猫旗舰店</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了将一直以来热卖的各种大包装商用汤品底料“缩小”，装进家用装的袋子里，仟味还在热门的快手菜里迈出了开拓C端的进一步尝试。其在21年1月上线口水鸡调味料，从高汤跨界到凉拌汁，以轻烹快手、荤素皆可拌为卖点，迅速俘获大批追求麻辣鲜香口味的“厨房小白”，销售额一路上涨，在天猫淘宝7月达到207.6万元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://p3-sign.toutiaoimg.com/tos-cn-i-qvj2lq49k0/056c5acca46a4c078ee4f8b905c72b0b~noop.image?_iz=58558&from=article.pc_detail&x-expires=1662086660&x-signature=ltUNPo9U9omGAO2dQ0tIZFr5m5o%3D" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">自上线以来仟味口水鸡调料在天猫官方旗舰店的销售额与销量</h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">数据来源：魔镜市场情报</h4>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">02、VEpiaopiao：乘上健康轻食的消费东风，低卡酱汁受到青睐</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在往期调味品新升榜中，VEpiaopiao就凭借187%的同比增速登榜。时隔两年，VEpiaopiao依然保持超100%的同比增速，7月销售额同比增速达到119%，月销售额达到392万元。VEpiaopiao经营各国口味的酱汁，从日式油醋汁到韩式甜辣酱，从罗勒番茄意面酱到泰式冬阴功料，凭借各国风味+轻烹场景抓住年轻消费者的胃，同时每款产品低卡低脂的定位让其在风味与方便之余多了健康这一卖点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://p3-sign.toutiaoimg.com/tos-cn-i-qvj2lq49k0/1e6ff3e8564d46a3a2c74715254988d8~noop.image?_iz=58558&from=article.pc_detail&x-expires=1662086660&x-signature=nprHbLmVkwhnYyDnWik1nG%2FRS0g%3D" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">图片来源：VEpiaopaio天猫旗舰店</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7月VEpiaopiao天猫旗舰店销售额最高的单品是蒜香/葱香黄油酱，月销售额为68.7万元。此款酱料主打与法棍、吐司搭配的早餐场景，受到消费者青睐。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://p3-sign.toutiaoimg.com/tos-cn-i-qvj2lq49k0/833cc6d3e9314e918b5b46101d6b29fc~noop.image?_iz=58558&from=article.pc_detail&x-expires=1662086660&x-signature=FgmXCHpXQUQ4LuNbEVywEqpFcJY%3D" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">图片来源：VEpiaopaio天猫旗舰店</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据魔镜电商聆听，天猫平台消费者对此款酱料主打的早餐场景感知较强，在此款产品近一个滚动年的天猫评论中，早餐相关的场景词是各场景中提及率最高的，且消费者对此正向声量较高。</p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">数据来源：魔镜电商聆听</h4>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小野轻煮自2021年5月上线以来，其销售额快速增长，在7销售额达到312万元。瞄准快节奏生活下的年轻群体，小野轻煮主打低卡无负担、快速成菜，推出麻辣香锅、酸汤肥牛、红烧酱汁等多款中式复合调料，对这些口味接受度较高的火锅底料及中式酱汁进行“健康化”、“轻食化”处理，切中时下消费者需求，以美味又健康的产品定位迅速占领市场。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7月小野轻煮的热销单品为酸汤肥牛底料，在天猫旗舰店销售额达到127.8万元。消费者对其口味认可度较高，部分消费者提及在减脂期食用，既满足了控制卡路里的需求又能为减脂餐增加风味。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在B端、C端双轮驱动下，调味品市场不断扩容，众多新老品牌同场竞技，竞相角逐这一片新蓝海。在懒人经济的加持下，方便简单、口味丰富、快速成菜的复合调料受到年轻群体的喜爱，而各种主打低卡无负担的酱料更是乘上健康消费的东风，侧面突围调味品市场。未来，调味品市场将朝着诉求健康化、风味多样化、方便快手化的方向不断前进和发展。</p>                        <div style="height: 1px"></div>
