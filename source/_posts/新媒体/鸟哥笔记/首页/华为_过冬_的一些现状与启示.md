<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">“要创造价值、合理利润，使公司健康成长。”</strong></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>——<a href="https://www.niaogebiji.com/tag/%E5%8D%8E%E4%B8%BA" target="_blank">华为</a>创始人</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">全球气候变化得更为极端，大厂们也是如此。他们大多都选择收缩战线、降本增效来应对短暂的“寒冬”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">8月23日消息，华为内部论坛刊登了一篇关于《整个公司的经营方针要从追求规模转向追求利润和现金流》的文章。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">文件中，任总对接下来华为要面临的内外部环境保持谨慎，他说：“我们对未来过于乐观的预期情绪要降下来，2023年甚至到2025年，一定要把活下来作为最主要的纲领，<span><strong>有质量地活下来</strong></span>，每个业务都要去认真执行。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15081756926307cb721e5095.21538639.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">前不久，8月12日，华为在官网发布了2022年上半年经营业绩，期内公司实现销售收入3016亿元人民币，净利润率5.0%。以此计算，华为上半年净利润约为150.8亿元人民币。对比2021年数据，2022年上半年营收同比下降了5.9%，净利润率下滑4.8个百分点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>华为这两年发展如何？各项业务发展得怎么样？对于公司领导者有哪些参考？</strong></span>（?文末仍可免费领取相关报告哦~）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1667192666307cb726796c6.18736582.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong>华为活得还不错</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从<a data-miniprogram-appid="wxdb8f8bebdcc8d881" data-miniprogram-path="pages/pdf-viewer/index?docId=3107688&im=find" data-miniprogram-nickname="发现报告" data-miniprogram-type="text" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined">华为《2021年年报》</a>来看，他的利润比这个地球上绝大多数企业都要好。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1450945116307cb72970fe8.35113088.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">公司2021年的财报显示，销售收入从2020年的8913亿暴跌至6368亿，降幅为28.6%。正常企业通常在疫情下都是增收不增利，努力稳住规模，而<span><strong>华为的规模下降了，净利润却暴增了75.9%，从646亿增至1137亿。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>早就提出的降本增效，华为在去年似乎就已经完成了。那华为为何依然这么谨慎呢?</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">进入美国实体清单后，华为依赖的全球供应链系统频频受到挑战。<span><strong>自2020年至今，华为一直将“活下来”作为首要目标。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“2021年，我们活了下来，但我们面临的问题并没有完全解决，2022年还要继续求生存，谋发展。”这是华为轮值董事长郭平，在2021年度业绩说明会上说的话。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，华为看似靓丽的2021年净利润，有高达574亿元利润来自于“处置子公司及业务的净收益"，主要就是出售"荣耀"及"超聚变”（x86）两家公司的利润。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18041741716307cb72d51042.97582468.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而从公司发布的2022年上半年经营业绩看，上半年实现营收3016亿元，同比下降6%，降幅同比收窄近四分之三；净利率5.0%，相较去年同期的9.8%有所下滑。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">拆分三大业务分析：</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">运营商业务收入为1427亿元，同比增长4%；</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">企业业务收入为547亿元，同比增长28%；</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">终端业务收入为1013亿元，同比下降25%。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>从三大业务来看，终端手机业务因遭遇供应链挑战，市场份额下滑明显。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">Omdia数据显示，今年一季度和二季度，华为手机出货量分别为560万部和640万部，华为手机在全球的市场排名已滑落至前五之外的“others”。上半年，终端业务对华为的收入贡献也由此前的超过50%降至33%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但，任总本就是一个未雨绸缪的人，用悲观的预期努力生存，拼命获得乐观的未来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在内部文档中表示，未来十年应该是一个非常痛苦的历史时期，全球经济会持续衰退，华为要把活下来作为最主要纲领，从过去恐慌性地自救改为有质量地自救，从追求规模转向追求利润和现金流，全线收缩和关闭边缘业务，把寒气传递给每个人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9947009436307cb731d53a0.41538686.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>此后，华为如何活下去？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">或许当下是未来几年中最好的一年。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>对于如何应对，华为选择断臂求生，聚焦主业，活下去。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为了在困难时期“活下来”，此前华为曾一度出售华为电气、华为海洋、x86服务器及荣耀手机等资产，从而实现轻装上阵。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而现在，华为强调，2023年的预算要保持合理节奏，收缩或关闭未来几年内不能产生价值和利润的业务，同时调整市场结构，<span><strong>哪里有钱就去哪里赚一点，聚焦市场价值客户，彻底放弃一部分国家或市场。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">华为轮值董事长胡厚崑在2021年报中表示：过去数年，我们坚持以智能手机为核心的“1+8+N”全场景智慧生活战略，持续深耕，坚持创新。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其中：1代表手机；8代表平板电脑、PC、VR设备、可穿戴设备、智慧屏、智慧音频、智能音箱、车机；N代表泛IoT设备。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12928320826307cb734fa498.69462351.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但从2021年报中看，消费者业务（终端业务）收入几乎砍半。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这个“寒冬”，全球消费能力会有很大幅度的下降，巨头们不太好过，落到每个普通人身上，就是失业、降薪...自然降低了购机、换机需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这种情况下，对华为而言无论是供应链还是市场都有压力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">任总表示<span><strong>，华为的生存基点要调整到以现金流和真实利润为中心，不能再仅以销售收入为目标。</strong></span>具体到可能的发展业务上，任正非表示：</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">华为军团是建基础信息平台，‍‍更好地卖ICT（信息通信技术），基础设施卖底座不是做生态，终端是未来华为崛起突破的基础，但不能盲目；</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">华为云计算要踏踏实实以支撑华为业务发展为主，走支持产业互联网的道路；‍‍</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">数字能源业务在战略机会窗上加大投入，创造更大价值，收缩机关，加强作战队伍；</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">智能汽车解决方案不能铺开一个完整战线，要减少科研预算，‍‍加强商业闭环，研发要走模块化的道路，‍‍聚焦在几个关键部件作出竞争力，剩余部分可以与别人连接。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19614325526307cb73c666d2.70049782.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>由上或许可以推测出，ICT基础设施是华为的核心业务，云计算次之，其余可能就会比较危险。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过，他同时表示，如果出现了一些机会窗，华为要扩大战略预备队和干部专家战略资源池，组成突击队去机会窗突击。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3153054556307cb74139d12.43787041.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>公司领导人们怎么办？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不仅是华为，这两年，全球经济都不那么景气。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">今年5月，红杉在题为“适应与忍耐（Adapting to Endure）”的52页幻灯片中，将当前动荡的金融市场、通货膨胀和地缘政治冲突，一并列为不确定性和变化的关键因素。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a data-miniprogram-appid="wxdb8f8bebdcc8d881" data-miniprogram-path="pages/pdf-viewer/index?docId=3190515&im=find" data-miniprogram-nickname="发现报告" data-miniprogram-type="image" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined"><img src="https://ngbjimg.xy599.com/4332979496307cb74425109.48718504.png" referrerpolicy="no-referrer"></a></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">红杉告诉创始人们，不要指望经济会像疫情开始后那样迅速反弹，因为“推动经济复苏的货币和财政工具已经被耗尽”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时，红杉也建议创始人迅速采取行动，以延长生命周期并全面检查业务是否存在超额成本。<span><strong>“不要将‘削减估值’视为负面因素，这是一种节省现金和快速发展的方式，它能帮你跑得更远。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">无论如何，我们相信这是一个关键时刻，将为许多人带来挑战和机遇。环境在<span>不断变化</span>，但我们要转变思维方式、努力去适应变化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17489005606307cb747612d5.33539853.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从中长期来看，有规律、持久的增长总是会得到回报，这些有意义的价值，也会让我们自己本身升值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2605479756307cb74c15993.19533077.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在历史的长河中，那些不仅活下来，而且活得很好的公司，无一不具备以下“生存技能”：</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">面对现实、适应快</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">手握现金，让公司能顺利周转半年甚至更久</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小步快跑，多尝试，但一旦不成功，果断放弃</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">将危机视为机会，努力打好一手烂牌</p>
</li>
</ul>
