                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">谁能想到，成名于明朝崇祯元年，至今已有近400年历史的张小泉，会因为菜刀拍蒜断裂而丢失“颜面”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再加上客服和总经理夏乾良的“推波助澜”，这家历史悠久的刀具品牌如今陷入了巨大的舆论危机之中。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">虽然，张小泉官方和夏乾良都已作出回应并致歉，但因一头大蒜引发的风波目前仍未平息。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">然而，对于竞品来说，张小泉口碑的崩塌无疑是最好的助攻。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">借着这股势头，<a href="https://www.niaogebiji.com/tag/%E7%8E%8B%E9%BA%BB%E5%AD%90" target="_blank">王麻子</a>、十八子作等竞争对手纷纷在直播间上演“刀拍大蒜”的戏码，还发了多条视频“炫”中国刀工，实现了流量的暴涨。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">不能拍蒜的张小泉，否定中国人切菜手法的夏乾良 </h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此次张小泉能在大范围引发众怒，一个是“菜刀拍蒜断裂”事件，另一个就是张小泉总经理夏乾良的离奇发言。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">近日，有消费者在使用张小泉菜刀拍蒜时，刀断成了两截。然而在联系张小泉售后客服时，却被告知“该菜刀不能拍蒜”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/abcd43eee9c9f6bce781261953bad742.png" alt width="556" height="511" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此事发到网上后引发热议，7月14日，“张小泉客服称菜刀不能拍蒜”相关话题迅速登上热搜，截至7月20日，微博话题阅读量超2.2亿，<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3" target="_blank">抖音</a>话题#张小泉菜刀也有1.2亿次播放量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/3678815f853787f941687e1cbe1c90fd.png" alt width="674" height="243" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">微博话题数据</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/fd93fd1de50e084dd7d3b0b9f6950450.png" alt width="683" height="272" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">抖音话题数据</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而随着热度上升，张小泉官方发布情况说明称，向消费者致歉并赠送一把新刀具。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实上，事情到了这里，仍有一部分人对事件保持着中立的态度，毕竟菜刀断裂也并非张小泉一家的问题，此前德国品牌双立人菜刀拍蒜断裂的问题同样引发过争议。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">然而张小泉总经理夏乾良的一番离奇发言再度将张小泉冲上热搜。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7月18日，夏乾良此前的采访被扒出，在采访中，他称“所有的米其林厨师都不是中国人这种切菜方法”，由此进一步点燃消费者的怒火。而相关话题也再度冲上热搜。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/b67d30a5d7f70c2d80e09517d695acea.png" alt width="477" height="519" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对此，夏乾良当天连夜在社交媒体发声，就近期舆论事件向公众道歉。同时表示，现传视频并未根据当时的情境和语境进行描述，从而导致误解。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外张小泉还发布“断刀召集令”，表示五年内无论品牌，只要刀断了都可以进行新刀补发。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/9b15fce937ca1c4a67c238750e49bfa7.png" alt width="1080" height="1893" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">借张小泉“翻车”热度，“王麻子们”直播拍蒜走红</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而在张小泉陷入“菜刀拍蒜断裂”风波后，同样是老字号的刀具“王麻子”趁着这波热度在7月14日不仅发布了多条拍蒜视频，还开启了超长直播。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">抖数观察到，@王麻子官方旗舰店 上次发布视频的时间为6月18日，已停更近一个月。而7月14日当天连发4条拍蒜视频，最高的一条有12.3万赞，而此前的视频点赞量仅在几十到几百不等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/1a9a9161c9e006a2eba6b905f910ae35.png" alt referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">当天@王麻子官方旗舰店 的直播也受到了用户们的强势围观。</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据壁虎看看抖音版数据查询，当日直播时长达21小时44分钟，直播期间观看人数突破45万，直播间累计销售额近9万。可见，用户对王麻子的关注度并未完全转化成购买力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/a244edf1a65cb0eb3ff1563e42ea4730.png" alt width="662" height="1310" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">值得一提的是，当天@王麻子官方旗舰店 的直播内容就是用菜刀拍大蒜，并且还在商品链接上都添加了“能拍蒜”三个字。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/4b04fd38535525de79a253483468ba14.png" alt referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7月18日，当张小泉进一步引发全网热议后，@王麻子官方旗舰店 再次开启了超长直播。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据壁虎看看抖音版数据显示，@王麻子官方旗舰店 这次直播时长近40个小时，直播UV达364.71万，累计销售额突破90万。同时，多款商品链接上也标注着“放心拍蒜”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/a6ed9258d53b17bac9cfd263432af8fa.png" alt width="432" height="172" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/b6ab849c9de03fd83b0125ad4e9b2f92.png" alt width="477" height="230" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同样借着张小泉这股势头“起飞”的还有@十八子作旗舰店 <a href="https://mp.weixin.qq.com/s?__biz=MzU5NjMxODA5OQ==&mid=2247491264&idx=1&sn=598a491a2f5e1a26c8b02f0f3ab1bbb2&chksm=fe65dc3cc912552a24fe45e8f82775f2b033ec08f3eda4bcc7fa2e9ebabc00a57475dc8e85a1&token=1835485245&lang=zh_CN" data-miniprogram-appid="wx486ef245f10a196f" data-miniprogram-path="pages/index/index" data-miniprogram-nickname="壁虎看看抖数" data-miniprogram-type="text" data-miniprogram-servicetype>@双立人官方旗舰店</a> 等多个直播间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">壁虎看看抖音版数据显示，近15天（7月5日-19日），@十八子作旗舰店 的直播趋势在7月14日达到了高峰，直播销售额预计154.63万，直播UV达48.07万。直到7月19日，直播UV更是突破了60万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/f037b8553082069f0706ad89c1797f03.png" alt width="1080" height="466" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除此之外，壁虎看看抖数还发现，厨具/刀具类的直播间掀起了“拍蒜”模仿秀。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很多品牌商家都不想错过张小泉和王麻子这班流量快车，纷纷在直播间拍起了大蒜，再或者就是在直播间放置了一堆大蒜，试图获得流量和销量的双增长。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/95992d969b9b9a933f382e417ae4533f.png" alt width="1080" height="1073" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">抖数还注意到，这些直播间评论区会不停有用户留言“拍个蒜看看”“拍黄瓜会断吗”或者“我是来看拍蒜的”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这时,有些主播会应用户要求，多次拿起菜刀拍蒜、拍黄瓜，从而达到了流量变现的目的。但也有主播因长时间不与用户互动，未能成功留人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不得不说，品牌商家们追随热点没错，然而在此之前，直播间的产品和服务也得要经得起推敲，这样才能提高直播间流量的转化。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">借势营销，还需克制</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">无论是线上还是线下，是新品牌还是老品牌，纵观近几年的营销玩法，借势已成为品牌们的一项“拿手绝活”，逢年过节，营销圈必定是热闹非凡。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从本质上说，借势营销之所以备受品牌们推崇，很大程度上是由于热点所自带的庞大关注度及流量价值，而比起常规的硬核广告，借势“友商”的营销方式则更为自然，也能获取消费者好感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如王麻子和十八子作，就以足够的敏锐度和预见性捕捉到了张小泉这一热点话题，从而进行恰到好处的营销植入，最大程度上收获了消费者注意力，进而促进了流量和销量的提升。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而在这样的营销背后，是王麻子、十八子作可以借此机会和张小泉形成强烈对比，拉开差距。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">蹭友商流量或是蹭“对家”翻车的热度常常是损人又利己，但品牌实际运用时也需要把握好度，否则玩火自焚了可就得不偿失了。</p>                        <div style="height: 1px"></div>
