<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互联网是有记忆的，品牌/明星的一举一动，都被大家看在眼里。哪怕是多年前说过的话都会被翻出来，这已经被很多人验证了<span>：</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span>张小泉</span></strong></span><span>菜刀不能拍蒜，迷惑发言<span>被喷</span>：中国人切菜方法不对</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span>旺旺</span></strong></span><span>被误伤意外翻红，创始人金句<span>被夸</span>：是大陆市场成就了今天的旺旺</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span><a href="https://www.niaogebiji.com/tag/%E5%86%88%E6%9C%AC" target="_blank">冈本</a>六君子</span></strong></span><span>反和服，也有人扒到他们的微博曾经给冈本卖套套。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>不久之前苏州和服事件闹得沸沸扬扬，网</span></strong><strong><span>上还惊现“冈本六君子”。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5090482946307af42764457.10090508.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：观察者网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我以为话题热度要降下来了，可这两天<span>刷</span>微博又发现一个段子：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>冈本很薄，脸皮很厚。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/7609872876307af42c32aa0.05290850.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友评论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>一整个被震惊到，差点以为是冈本的营销。但一顿吃瓜之后，才发现原来是：</span><span><strong><span>网友在闹，冈本在哭。</span></strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/723115795623260770c18c4.52658385.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>冈本，到底怎么了？</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>舆论事件最初爆发在</span>8月上旬，当时苏州有个女子穿日本和服在公共场合拍照，然后被带到派出所去，一开始<span>和服也被</span>没收了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>敏感的日本话题，加上敏感的815前夕时间点，于是网上一些大V就因为“是否有穿和服上街<span>自由”</span>吵起来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>一个大v</span><span>就发话，要是穿和服不妥，卖日本套套也不能放过，点名了几个曾接冈本套套广告的博主。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18969226666307af43107052.65132268.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12141841856307af437f9801.82788866.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：落魄的三叔</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>赚钱不寒碜，其实让</span><span>@落魄的三叔 不爽的是“<span><strong>和服不能穿，冈本可以戴</strong></span>”的现象。</span><span>这些接套套广告的博主，发过一些爱国内容，有的还曾亲自下场点评和服</span><span><strong>。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/7893888146307af443c05c3.14572692.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博博主评论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>也不知是谁，干脆给这六个人取了个雅号：</span><span><strong>冈本六君子</strong></span><span>。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8878591556307af449efe46.12121862.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友评论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>之后又有人跑出来，贡献一个概念：</span><span><strong>冈本=和服。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/7904007876307af44e1f8f1.94649837.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友评论</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>说来也真巧，</span>这个</span><span><strong>“冈本避孕套也是和服<span>”</span></strong></span><span>的梗一出，有人真扒到日本和服里，<span>确实</span>有个品牌叫做冈本。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/3805840006307af452537a7.23801005.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友评论</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不止如此，网友变身段子手，根据和服故事的来龙去脉，造冈本段子，设计套套<span>。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18778100726307af456033f8.67881553.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17481663166307af45a2a3f5.85519032.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友评论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更离谱的是一时之间万物皆可冈本，但凡是能跟日本沾边的东东，都可以被冈本“冠名<span>”</span>，就差官宣开启冈本元年：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>日本樱花节=冈本节</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>和服=冈套</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>日本=冈本国</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>日本料理=冈本料理</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19081468776307af45f13f92.03665925.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友评论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>舆论的走向变得越来越失控</span><span>，</span>新段子又横空出世，也就有了开篇看到的</span><span><strong>#冈本很薄，脸皮很厚#</strong></span><span>，就是网友用来调侃</span><span>“冈本现象”。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205789392362326078647a25.53043458.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你说这个事跟冈本有关系吧，又好像没有。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从网上的舆论来看，有些网友愤怒的不是带货日本安全套本身，而可能是<span>“</span>嘴上是主义，心里是生意<span>”</span>的<span>双标</span>现象。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/18652177106307af463ee6d2.02969989.png" referrerpolicy="no-referrer"></strong></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><strong><span>图源：微博网友评论</span></strong></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>好比有的品牌玩双标，有如俄罗斯套娃，一套又一套。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>意大利品牌D&G杜嘉班纳，捞中国市场的钱却侮辱中国的筷子，HM等企业一边在中国市场开店，一边抵制新疆棉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>还有“</span><strong><span>吃中国饭，唱日本歌</span></strong><span>”的名创优品。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>不久前名创优品在西班牙Instagram账号上，发布了“公主系列公仔盲盒”的内容帖，把</span><span>穿旗袍的娃娃称为日本艺妓<span>（Geisha）</span></span><span>。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/8295308546307af467caf16.89268033.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：人民网评</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>除此之外，名创优品被爆<span>店内不允许播放</span></span><span>中文歌<span>；<span>惹众怒的还不止这些，</span>名创优品</span>在与国外合作方签约时，现场居然挂起了日本国旗。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要知道名创优品自出道以来，走的就是伪日系的营销路线，从名创优品的logo、门店设计到产品包装，都增加了一层日系风的滤镜。双标的行为无疑降低品牌好感度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这一系列的品牌塌房事故，让消费者逐渐对“双标”的现象越来越敏感。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134948845962326079426029.07408166.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>话说回来，这次最冤的还是冈本这个品牌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众所周知跟靠热点出圈的杜蕾斯对比，岡本一直以<span>“超薄”概念做营销，<span>开创了数字薄系列产品，</span></span>“冈本薄<span>”早就刻进消费者的DNA里</span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是为啥大家将套薄和脸厚对比，来形容“冈本<span>”</span>。<span>从某种程度来说，<span>网友造</span></span><span>奇葩段子，说明大家</span><span>对冈本“薄”的认知很深，</span>但这样的“广告<span>”</span>也只能算是黑红。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>无论怎么样，冈本如今被“上冈上线”，躺枪都是名字惹的祸，毕竟冈本是日本常见姓氏。</span></strong><strong><span>像极了科罗娜啤酒，遭遇被黑得最惨的经历。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>科罗娜（Corona）</strong></span><span>当初给品牌起名，怎么会预料到有一天，会跟<span>新冠病毒（CoronaVirus）</span>撞名。按理说这完全就是一种巧合，</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但疫情爆发之初，<span>科罗娜难免</span>还是被嫌弃，2个月亏20亿元。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>有的栽在了名字上，有的是被无辜</span><span>卷入社会事件，成为众矢之的过街老鼠，这或许就是“<span>我命由天不由我<span>”</span>。</span></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>此前</span><span><strong>唐山烧烤店打人事件</strong></span><span>引发全网关注，眼尖的网友扒到打人者的衣服就是</span><span><strong>BOY LONDON（伦敦男孩）</strong></span><span>。</span></span><span></span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友评论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>谁看了不说一句：</span><span><strong>好事难出门，坏事传千里。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没想到今天的冈本，活成了昨天的<span>BOY LONDON和科罗娜。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“薄”明明是品牌的卖点，却因为和服事件被污名化，演变为一个槽点，最终还落得个“<span>厚</span>脸皮”的<span>人设</span>标签。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/138833517962326079dd1f53.73402961.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>如果说</span></strong><strong><span>躺枪不可避免，那品牌就要把功夫留在平时。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谎言重复一千遍就会成为真理，这就是人言可畏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span>首先品牌商一定要用产品说话。</span></strong></span><span>打铁还需自身硬，消费者总会为产品买账，靠谱的产品有时候可以破谎言一千零一遍；但如果产品不尽如人意时，品牌翻车时再辩解也是徒然，还可能越描越黑。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span>另外就是要做好用户关系维护。</span></strong></span><span>信任是一个品牌最大的流量，品牌主动关怀跟用户玩一块，用户也足够了解品牌，信任的关系就容易建立。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样当<span>品牌</span>面临祸从天降的时，它的忠诚用户也会倾向相信品牌，第一时间站出来力挺，坐等真相的到来。</span></p>
