<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「新的风暴已经出现，怎么能够停滞不前...」</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小时候人人都能哼上两句的奥特曼主题曲，在近期<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3" target="_blank">抖音</a>夏日云漫展燃魂live上，原唱毛华锋居然带来了超燃的摇滚版本，堪称真‧奇迹再现；云歌会上还有杨超越虚拟人超越AI首次虚拟舞台独家节目亮相、虚拟偶像女团A-SOUL队长贝拉舞蹈表演，以及《新神榜·杨戬》的水下舞演出，和各种洛丽塔、国风、兽装等引爆小众圈层尖叫的演出..<strong>.这场夏日云漫展，昭示了二次元内容在抖音的繁荣盛况。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/358888605630c4cad03e270.26318783.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了奥特曼、航海王等海外经典IP，在抖音，也能看到国漫如《中华小子》《伍六七》的焕发新生，恰逢中国动漫诞生百年的2022年，不仅大部分头部<a href="https://www.niaogebiji.com/tag/%E4%BA%8C%E6%AC%A1%E5%85%83IP" target="_blank">二次元IP</a>已入驻抖音，还有越来越多新兴二次元创作者，正在透过抖音崭露头角。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二次元内容通过抖音实现破壁，但是抖音，是如何吸引到「二次元们」的？</span></p>
<section><img src="https://ngbjimg.xy599.com/134179749962ef3b56855658.34219712.png" referrerpolicy="no-referrer"></section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>燃魂live云歌会，</span></strong><strong><span>百大二次元IP齐聚狂欢</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音在这个夏天做了为期一个月的夏日云漫展，在活动末尾8月27日，举办以「燃魂live」为主题的二次元云歌会，将这个夏天属于二次元的快乐推向高潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃魂live中，以可爱的虚拟人默默酱为主持人，百大头部动漫IP&明星&二次元音乐人的表演在直播间中绽放异彩，让二次元用户一次过足了瘾，享受绝佳的视听体验和创新的内容呈现形式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2114524302630c4cad6e50d5.87972156.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、</span></strong><strong><span>热门动漫齐聚一堂，焕发「抖音特色」二次元风采</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>表演节目中，有许多大众熟悉的动漫IP联动舞台，比如《迪迦奥特曼》主题曲，带来了比原始版本更为热血的摇滚版本；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1386224567630c4cade0e3b8.06202119.png" referrerpolicy="no-referrer"><span></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>毛华锋‧《迪迦奥特曼》主题曲</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>还有长达8分钟的国漫OP串烧，</span><strong><span>融合《凡人修仙传》《画江湖之不良人》《伍六七》《中华小子》等IP与国乐+武术的演出，以及近期</span><span>《杨戬》与水下舞的高难度融合，将古典与现代、动与静、真人与动漫合而为一</span></strong><span>，创造出崭新的舞台效果，满足用户的情感共振与对新鲜感的追逐，在二次元内容池中留下富有抖音特色的一笔，也让用户看见，国漫无论在美术、叙事和精神内核上，都走到了更高的地方。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/326827953630c4cae9e05e4.19240394.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>杨戬‧水下舞</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2、明星&二次元音乐人势能加持，打破传播次元壁</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不止动漫IP，各路主题曲原唱歌手、头部国风音乐人、ACGN音乐人都加入了这场夏日盛典。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是真‧建模脸的杨超越虚拟人超越AI，在燃魂live上惊喜亮相，动作俐落的舞台加上富有技巧的歌声，符合粉丝对偶像近乎「完美」的想象，居高不下的人气在节目中一览无遗。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/446392229630c4caf066163.07778987.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/688763867630c4caf3ab928.98284770.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>杨超越虚拟人超越AI</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在云歌会上，还可以听到毛华锋、南征北战等歌手热血的歌声，也能见到「中国第一萝莉」王奕萌、SING女团姣好的演出；还有头部国风音乐人如等什么君、Winky诗、以及大门E、麦吉等ACGN音乐达人的舞台，用或婉转或高昂的音乐，点燃用户的热情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/435671942630c4caf8cc598.71860808.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当一场二次元歌会有了明星和头部达人的影响力加持，进一步助推燃魂live的传播破圈，将二次元的美好内容向更广大的群体扩散</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1340379289630c4cb00867c7.05245221.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3、洛丽塔、虚拟人、兽装等要素齐全，引爆圈层嗨点</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「二次元」的标签底下，还有许多更小众的圈层爱好，在夏日云歌会中也能窥见一二。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如高颜值的洛丽塔走秀，一次集合数十位洛丽塔达人来了场四大主题虚拟空间大秀；</span><span>还有虚拟人默默酱的主持串场与表演，魅力丝毫不输真人演出；</span><span>以及近两年火热的兽装元素，也在台上热舞，让小众爱好者疯狂点赞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>多样化又高规格的演出排场，充分彰显抖音在泛二次元的小众圈层耕耘多时，已然具备一定圈层影响力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/481399818630c4cb09c24e1.06037868.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>90分钟的燃魂live加上燃魂派对云蹦迪，<strong>抖音夏日云歌会凭借丰富的内容，精准狙击二次元用户的注意力，引爆话题热度，让人们发现，抖音比想象中更懂二次元和二次元群体</strong>。</span></p>
<section><img src="https://ngbjimg.xy599.com/170734857262ef3b56ed9857.52851007.png" referrerpolicy="no-referrer"></section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>夏日云漫展，</span></strong><strong><span>二次元内容在抖音蓬勃发展</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经过这回夏日云漫展+云歌会的传播，许多人注意到，二次元内容在抖音可以说是遍地开花。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅在热榜上可以时常看到二次元相关话题，#奥特曼 #二次元 等话题更是动辄千亿播放，近期高达之父大河原邦男入驻抖音的话题，也引发一票动漫粉丝重温童年热血记忆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>夏日云漫展，便是基于抖音站内的二次元内容生态，为用户精选三大板块内容呈现。一部分是经典动画IP在抖音联动；一部分是鼓励原创IP投稿创作，赢得奖励；还有一部分是好玩有趣的二次元特效合拍，吸引大众一同爱上二次元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/389065069630c4cb24b86e0.22690017.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、</span></strong><strong><span>经典IP抖音再出发，点燃情绪共鸣</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在夏日云漫展的「IP畅玩馆」中，</span><strong><span>能看到许多经典动画IP都入驻了抖音</span></strong><span>，有热血向的IP如海贼王、奥特曼、高达，女性向的IP如魔卡少女樱、三丽鸥，还有风靡大众的原神、宝可梦、小黄人、迪士尼IP，多种风格覆盖全民爱好，经典回忆杀桥段更是引起用户共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1536475815630c4cb5316238.53690144.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>围绕经典IP的相关话题，也在期间内获得大量关注。</span><strong><span>#高达之父</span><span>大河原邦男入驻抖音</span></strong><span> 的消息，短时间内风靡抖音站内外，话题越传越广，还有用户翻出高达跳男团舞的画面，经典角色</span><span>加上男团魅力，</span><span>谁能抵挡得了迷人的高达？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/585764693630c4cb89c9586.67648724.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2、</span></strong><strong><span>优质原创作者聚集，不止为爱发电</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了陪伴人们成长的经典动画以外，抖音上也聚集了许多二次元头部IP和创作者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如以中国历史文化为底色的动画《秦时明月》，在抖音上有169万粉丝，每天都会更新内容；还有扎根民间百态的动画《伍六七》，在抖音上有285万粉丝，工作室成员时常露面与粉丝保持互动，也开发了多款周边商品，在抖音橱窗中售卖。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1744547742630c4cb9e66fa9.56335407.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>还有许多正在崭露头角的原创IP，正在抖音上快速跑出变现路径，解决了二次元行业商业化痛点</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如以动物IP为主的平头哥，目前在抖音有800+万粉丝，已发布85篇短视频，在抖音活用短时长、低成本的特质，快速积累粉丝，并通过直播变现来验证IP商业潜能；虚拟偶像默默酱也是广受欢迎的IP，目前抖音粉丝接近480万，首场直播销售额即超20万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1511895339630c4cbbd48fb2.43968075.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在更细分的兴趣圈层，如cosplay、洛丽塔、手办娃娃、汉服、声优等，也有大量的创作者在抖音展现魅力，找到属于自己的一票忠实粉丝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有收获，才会激励创作者们持续耕耘。为此，抖音一直以流量扶持优质创作者，一旦拥有一定粉丝量级，原创IP创作者即可透过星图广告、直播带货等路径来完成商业化变现，目前已有大量创作者在抖音凭借多元的变现方式收获商业化的果实，继续在创作之路上步履不停。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>3、</span></strong><strong><span>花式UGC，层层扩散影响力</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在创作者以外，广大的二次元爱好者也能在抖音上玩出一片天地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二次元圈层中，根据喜爱的动画再创作就是很重要的传播动力。有抖音达人根据近期热梗「退退退」自制国漫版《emo别来沾边》鬼畜视频，融合了大量国漫IP，让许多二次元同好者疯传，看了直呼走心。</span><span></span></p>
