<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><span><strong><span>1.</span></strong></span><strong><span>小事情为什么总造成惨结局？</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们经常遇到一些事让人烦恼不堪，有些人会选择发泄出来，甚至冲着别人大喊大叫；可这么做似乎并不能达到完美的解决，反而把自己气的半死。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>之所以有此类情况，因为很多人不知道，</span><strong><span>“你发怒时，怒火也许会烧及他人，但在更多情况下最后遭罪的可能是自己”</span></strong><span>，心理学中把该现象表现称为</span><span><strong>「<a href="https://www.niaogebiji.com/tag/%E9%87%8E%E9%A9%AC%E6%95%88%E5%BA%94" target="_blank">野马效应</a>」。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时还有一段经典故事：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>非洲草原生活着一群野马，Ta们暴躁、易怒而且警觉性非常高；令人吃惊的是，这些野马的天敌竟然不是狮子和老虎，而是蝙蝠。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这些蝙蝠靠吸食动物的血液生存，常叮在野马的腿上，不管野马怎么折腾Ta们始终不依不饶，从容地吸饱血后再离开；</span><strong><span>野马拿这些「吸血鬼」并没有办法，最终被活活折磨死。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>动物学家研究发现，这些蝙蝠吸血量很少对野马不足以致命，<strong>导致他们丧命的是被蝙蝠盯上后的暴躁和狂奔。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>换句话说，吸血蝙蝠只是野马死亡的诱因，而野马对这一诱因的剧烈情绪反应，才造成它们的直接死亡。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>因此心理学家将生活、工作中为芝麻小事而大动干火，易导致别人过失伤害自己的情况，叫做</span><span><strong>「野马结局」。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们能得到什么启发呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何处理情绪问题的文章有很多，但最重要的是，<strong>要提高自我对外界刺激下的承受力，和对外界刺激的客观评价能力。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当想要发怒时，反复告诉自己“这件事不值得”“我应该换个角度看”；另外，冷静后找到身边朋友，将心中不满、向别人倾诉也是不错的方法，这样能从亲朋友好友那里得到安慰。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>2.如何放下心中的执念？</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>希腊神话故事中有一位英雄大力士叫“海格力斯”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有一天Ta在坎坷不平的路上看到脚边鼓袋一样的东西，很难看；于是便踩上一脚，谁知道那东西不但没有破反而膨胀起来，这便激怒了Ta。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Ta顺手拿起一根碗口粗的木棒砸过去，好家伙，<strong>那东西居然膨胀起来把路堵上了，海格力斯奈何不了它。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>正在郁闷时一位圣者出现走在面前对Ta说，那是仇恨袋，你越是充满仇恨它就越膨胀，相反你若不理它就会变小如初；</span><span><strong><span>后来人们把该效应称为<a href="https://www.niaogebiji.com/tag/%E6%B5%B7%E6%A0%BC%E5%8A%9B%E6%96%AF%E6%95%88%E5%BA%94" target="_blank">海格力斯效应</a>（Hercules effect）</span></strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经沃尔沃集团有位销售培训专家叫“威廉·哈尔斯”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在第二次世界大战期间，哈尔斯逃难到瑞典，因为他能说并写好几国语言，所以他希望在一家进出口公司谋求工作，但是多数公司都拒绝了他。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，有一个人在给哈尔斯的信中说到<strong>“看得不出来，你对秘书这件工作完全不了解”</strong>，并且你连瑞典文都写不好，信中全是错别字，我根本不需要这样的秘书。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当哈尔斯看到时简直气得发疯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，他立刻写了一封回信，内容尽是积极嘲讽挖苦之事，用词颇为刻薄；当写完后他没有寄出，而是停下来对自己说<strong>“即使对方收到信后暴跳如雷，对我来说有何意义呢”？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，哈尔斯撕掉了那封骂人的信，反而又写了一封信。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这封信里他表达对自己文笔不行的惭愧，并且对对方指点的内容表示谢意，没过多久，哈尔斯收到了对方的回信，同时因为真诚还得到一份“秘书的工作”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们能得到什么启发？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人际交往中不可能没有利害冲突，我们应该以宽容对待仇恨，仇恨自然会消失，<strong>多记住别人的好也就“有利于自己”</strong>；相反，若是以报复对仇恨，<strong>除获得短暂快感和更大仇恨，又有什么好处呢？</strong></span><span></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>3.如何调整绝望的心态？</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美国心理学家马丁·塞利格曼1967年在研究动物时曾做过一个经典实验：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Ta把狗关在一个带蜂音器的笼子中，每次都被电到倒地呻吟，大小便失禁；这样反复折磨多次后塞利格曼更改了试验流程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在蜂音器响后不着急电机，而是先打开笼门，但这时狗不但不逃，反而不等电机开始就先倒在地上开始呻吟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span>本来可以主动逃避，却因之前的绝望体验而放弃希望，默默承受痛苦的来临，这便是塞利格曼所述的</span></strong></span><span><strong><span>「习得性无助」。</span></strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>其核心指，由于长期积累的负面生活经历，使人无法找回自信；想要避免此类情况，最重要</span><span>需要<span>有一个辩证的挫折观，经常保持自信和乐观情绪。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美国海军陆战队的退役军官科尔曼·米契在一次飞行事故中遭受重伤，身上65%的皮肤都烧坏了，为此他动了十六次手术；手术后仍然无法拿起电话、刀叉，也无法一个人上厕所。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样的挫折并没有使他陷入绝望，他用保险赔偿金为自己在科罗娜买套房子，同时继续进行康复训练。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而六个月后，他又能开飞机了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>康复后的米契尔和两位朋友合资开了一家公司，经营以木材为燃料的炉子生活过</span><span>得<span>非常不错，但没想到四年后的一天，飞机起飞时被摔回跑道，</span><strong>这次他胸部12块脊椎骨全部被压得粉碎，腰部以下全瘫痪。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这次事故，他几乎绝望的说“我始终搞不清楚，为什么这些事情总会发生到我身上，到底造的什么孽”，最终他还是挺过了又一次低谷期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来经过不懈努力，他被当选为科罗纳多州孤峰顶镇的镇长，以保护小镇环境，使之不因破产的过度开采被破坏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他用一句<strong>“不只是另一张小白脸”</strong>的口号，将自己在事故中被催得面目全非的脸转化成为广受大众敬佩的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>面对绝境其始终没有绝望，在一次公开演讲中，Ta说</span><strong><span>“我瘫痪之前可以做1万件事，而现在我把注意力放在无法做好的一千件事上”</span></strong><span>，可能换个角度看那些裹不足前的经历，你就会退一步、看的更开。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，没有铁一般的意志就很容易被绝境环境打败。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，当一个人将不可控制的消极事件或失败结果归因于自身的智力、能力时；一种弥散的无助和抑郁状态就会出现，自我评价就会降低，动机也减弱到最低水平，无助感也由此产生。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>4.先有最坏，才有更好？</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>威利·卡瑞尔是纽约水牛钢铁公司的工程师，有一次他到密苏里州安装一台清洁机，经过一番努力机器勉强可以使用，但依然达不到公司保证的质量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，他十分焦虑，一度无法入睡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>后来，他意识到忧虑解决不了任何问题，于是便换了另外一种解题思路，他想“这件事导致最坏的结果是什么？</span><strong><span>无非老板把机器拆掉，最后炒掉自己？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>想到这里，卡瑞尔对自己说”如果丢掉工作该怎么办？然后发现，当时维修工程师普遍紧缺找一份工作并不难，换句话说，</span><strong><span>最坏的结果也并非无法接受。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有了心理准备后他的心情逐渐平静，后来经过几次试验终于发现，假设再多花几千元多装一些设备，问题也就迎刃而解。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后不但把问题解决还获得一份完美的改进方案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来，成功学大事戴尔·卡耐基从其经历中总结一套解决忧虑情绪的方法论，并命名为</span><span><strong>「卡瑞尔公式」。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>在一次演讲中，他把该公式分为三步走：</span><span><strong><span>1）排除恐惧，2）找到可能发生的最坏情况，3）平和地投入精力并改善</span></strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，如果一直忧虑下去的话，恐怕永远不可能做到这三点；因为忧虑最大的坏处在摧毁我们集中精神的能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，当我们被强迫面对最坏情况时，首先学着接受它，然后衡量所有可能情景，使自己处在一个可以集中解决问题的场景下才能更好的改善。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>5.美丽的谎言真有用吗？</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>谈到信念和意识的改变，可能最著名的例子就是</span><span><strong><a href="https://www.niaogebiji.com/tag/%E5%AE%89%E6%85%B0%E5%89%82%E6%95%88%E5%BA%94" target="_blank">安慰剂效应</a>（Placebo Effect）</strong></span><span>，最早在19世纪医院临床治疗上被医生们发现。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>医生们认为，有些看似和真实治疗一样，实际上却没有任何医学有效成分的假治疗，往往也会产生效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>为什么会这样呢？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主要因为在单盲情况下，有时受治疗者并不知道背后情况，只是受到研究者的“欺骗”去相信一件事物有效；因此，当被主动进行暗示，信念体就会改变内心的感受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据目前脑科学研究，<span>除了外部辅助，</span>主动的自我暗示对心理康复也有积极作用；比如在心理疗法（psychotherapy）中，能够起到放松自我和催眠的效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种练习一般分为两部分：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其一比较具体，<strong>可以针对身体状态进行想象和自我对话</strong>，如“我很平静、没有什么能打扰我”，“我的胳膊现在很沉重”“我很暖和”等等；其二部分比较抽象，<strong>深入到”我是谁，我该怎么做“之类的想象中。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>生活中，有哪些场景我们在使用呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小孩大吵大闹时抱着哄哄“不哭不哭”，真的就会安静下来，情绪上升时告诉自己“不要激动”随之也会得到有效控制。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>甚至当你垂头丧气时告诫自己“我要快乐”，长期以往也会被虔诚的谎言所安慰。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总而言之，<strong>安慰剂效应作用虽然能够带来一些正面效果，但也要时刻注意不要长期陷入「谎话」的囚笼</strong>，这如同你始终安慰自己未来会更好，却从来不努力一样。</span><span></span></p>
