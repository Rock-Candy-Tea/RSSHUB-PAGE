<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>科技要素赋能营销是当下的重要趋势。在2022全球数字经济大会上，中国信息通信研究院发布的《全球数字经济白皮书(2022年)》显示，2021年，全球47个主要国家数字经济增加值规模达到38.1万亿美元。其中，我国数字经济规模达到7.1万亿美元，占47个国家总量的18%以上，位居世界第二。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数字经济已经成为我国经济发展中创新最活跃、增长速度最快、影响最广泛的领域，特别是随着数字技术成熟并运用到各行各业，在营销领域发挥了重要的作用，科技要素渗透到了营销策略、创意、投放、运营、客户管理等各个场景里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以<a href="https://www.niaogebiji.com/tag/%E6%95%B0%E5%AD%97%E8%90%A5%E9%94%80" target="_blank">数字营销</a>市场近期采访的</span><span><strong>整合型互联网营销运营商——耐特康赛</strong></span><span>为例，<strong>创始人/董事长兼CEO渠成就在采访中为我们分享了在营销科技公司崛起的大背景下，耐特康赛是如何保持对行业的高度敏感，并不断升级自身能力，实现长久增长的故事。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8982051526303b8e325d4a8.36063320.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2008年，渠成从新西兰回国创业，毕业于奥克兰大学的他选择切入SEO赛道，正式进入了这个行业，按他的话来说，“当时搜索就是营销的主战场”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>的确，搜索引擎营销（SEM）在当时是广告营销的重要理念，特别是大品牌广告主的合作，极大地推动了中国搜索引擎营销市场的发展。也是在这个时候，渠成接触并与非常多的互联网公司达成合作，积累了超前的技术和经验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>五年前，耐特康赛更新了“</span><span><strong>靠谱、利他主义、长久主义</strong></span><span>”三个关键词，来概括自身的企业价值观。耐特康赛创办至今，早已不再仅仅是搜索服务提供商，而是成为了一家互联网数字营销领域的头部公司。谈及这种坚持，渠成告诉记者，这三个关键词其实环环相扣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“首先，做人要靠谱，做公司更要靠谱，只有成为真正踏实的合作伙伴才能活得长久，才能形成一个好的生态并找到自身的生态位。有贡献自然有回报，利他主义和靠谱都是为了让企业更长久地存活，我们见证过太多互联网行业里的起起伏伏，比起求短时间的成败，<strong>耐特康赛希望给客户输出的营销服务和创意策略，能够帮助企业长久成长，也为我们自身带来持久的竞争力。</strong>”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14086392906303b8e3b3e0b4.39039037.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有网友感叹：青春有几年，疫情占三年。而对于耐特康赛来说，时刻关注市场变化，随着行业调整自己的业务结构早已经是一种习惯。<strong>渠成认为，对于一家企业来说，它的生存其实取决于客户的生存，而保持现金流和业务的“健康”是重中之重。</strong><strong>耐特康赛合作的客户涉及八大行业，如何在营销预算削减的普遍状况下紧抓客户最急迫的需求，抓住痛点为客户提供核心服务，实现精细化运营，是保持耐特康赛稳步增长的“第一要义”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20035731176303b8e43f8a98.81604373.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时间来到2022，面对数字化的社会发展，企业将更多的精力投入到利用数字化方式构建更深的客户关系上，这样的趋势下，企业的营销战略和营销思维同步发生了变化。比如针对解决大数据应用能力建设的问题，阿里最早提出的企业中台理念等，但这些概念和对应的组织变化在每个企业的具体应用千差万别。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“<strong>对于大型企业来说必须要先有系统性的部署，做全面的数字化升级；中小企业恰恰相反，可以以点突破，暂时不追求面面俱到，而是选择通过一些系统工具把单个渠道做好，层层叠加灵活升级</strong>。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那为什么数字营销会被视为数字化转型升级的排头兵？<strong>渠成说这是因为数字营销有一个先天的优势，就是离消费者更近，回报更明显。且数字营销是伴着互联网时代而产生的，在数字化领域相对成熟，便于企业试水。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这样的趋势下，从全球范围来看，营销科技公司呈现出了一片欣欣向荣的发展态势。但良好的发展态势也意味着，今天的市场对技术公司所提出的要求将变得更加多元化。特别是全域营销策略与实际应用，已经成为了流量红利见顶的现状下客户的重点需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了更好地布局多媒体的内容，针对性地匹配内容与渠道，耐特康赛将自身深耕多年搜索算法积累的“搜索”基因融入了内容营销、效果营销、社交营销等服务过程当中。对于耐特康赛为什么还需要优质内容原创能力的问题上，<strong>渠成表示，“从一开始，内容、渠道和技术就没有分过家。”</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17911589706303b8e4a81c62.82677726.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>10多年前一个网站想要排名好，只靠技术也不行，还要有优质的原创内容。而现在顺应时代的要求，不论是短视频还是文字，大家对优质内容的要求越来越高，从用户的角度来考量，耐特康赛自然而然的要更加关注这些方面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>渠成还透露，耐特康赛的出海业务这两年也都保持了100%的增长</strong></span><span>，为了帮助客户面对出海后的品牌认知问题，让品牌在国际市场真正立得住脚，独立站的搭建必不可少，这不仅能帮助品牌掌控自己的渠道，也能避免随时被海外的平台“卡脖子”。因此，帮助这些客户建设独立站同时，更要打造独立站上的优质内容吸引用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4662375526303b8e51e1ab3.53803332.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从内容角度看今天的营销，直播经济、短视频等形式虽风口正盛，但也迅速成为了一片红海。从“卷流量”到“卷知识”，可见新形式更迭快速，内容依然为王。企业或品牌该如何在其中实现营销能力和效果的进阶，成为新的问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>渠成同样认为，红利一定会消退，拥抱变化应该是一种习惯。</strong></span><span>也就是说，在形式创新的新常态下，企业除了要寻找红利，更要做好对新渠道的运用，不能说没有红利了就视而不见，而是必须要把它变成运营的组成部分，才能让其长久地产生价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“<strong>流量一词其实是有着非常重的早期互联网特性，现在的‘流量’是一种背景，而不是目的。更加重要的是关注和用户，只有不遗余力地获得关注，积累并经营真正喜欢自身产品和公司的用户网络，才是关键所在。</strong>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是的，还有以往我们谈营销，更多的是在说一个独到广告创意，营销像一种艺术。而现在，兼具内容策划和运营能力的互联网营销解决方案运营商受到青睐。为什么说今天的营销更像工程学，渠成认为这是基于艺术与科学的区别。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>艺术天马行空，但技术有迹可循。数字营销的特点就在于我们能借助大数据技术、广告技术、AI智能等等找到一条把营销做正确的“轨迹”。当然，这也导致了创新都有了套路，容易被模仿。“<strong>营销到了这个阶段，就从艺术成为了一种工程。当然，所有的做法都能被颠覆，创新和创意依旧重要，但我认为营销的底层逻辑已经和以往不同了。</strong>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>渠成还在采访中提到，广义的搜索其实无处不在，它伴随着互联网的出现而出现，又伴随<span>着</span>互联网的成长而成长，这也是为什么耐特康赛能够时刻跟上行业发展的原因。也希望耐特康赛能够在数字营销的时代，继续实现营销创意与效果营销完美结合！</span></p>
