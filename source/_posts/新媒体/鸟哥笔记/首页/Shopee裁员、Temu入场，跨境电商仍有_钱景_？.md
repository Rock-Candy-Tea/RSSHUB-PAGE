<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">9月19日，东南亚电商巨头Shopee于国内召开全员大会，宣布将启动团队调整并削减部分岗位。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据知情人士透露，本次调整主要目的是<span><strong>组织瘦身提效</strong></span>，整体员工受影响比例为低个位数百分比，各市场业务运营不受影响。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2022年以来，Shopee持续缩减成本，甚至关闭其在智利、哥伦比亚和墨西哥的当地业务，并完全退出阿根廷市场。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">5月份，亚马逊、Shopify、eBay等<span><strong>主流跨境电商平台的Q1财报也不理想，各平台普遍出现了增长乏力甚至是负增长的现象。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但国内互联网巨头们，却纷纷入场跨境电商：</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">拼多多推出跨境平台Temu，首站是北美市场；</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">阿里跨境业务进行了重新整合，近日又公布了OKKI“独立站”解决方案；</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从2021年开始，TikTok Shop也开启了东南亚第一站，尝到甜头后，今年火力全开在东南亚多个国家开展业务。</p>
</li>
</ul>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1318291493632d5e779cde07.56576127.png" referrerpolicy="no-referrer"></p>
<p><span><strong>2022的跨境电商现状如何？行业洗牌后，接下来的发展趋势有哪些？</strong></span></p>
<p>报告酱梳理了相关报告，本文与大家分享上述问题。（?文末仍可免费领取全部参考报告）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1625839949632d5e78036473.91525454.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong>曾经的黄金时代</strong></span></h2>
<p>2020年的新冠疫情，为全球经济发展带来冲击，但同时却凸显了跨境电商的优势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2142032347632d5e783789a6.75171628.png" referrerpolicy="no-referrer"></p>
<p>大量消费者转向线上消费，刺激了全球电商零售行业的快速发展，<span><strong>全球电子商务渗透率和销售额不断提升。</strong></span></p>
<p>单从国内数据看，据网经社统计，2021年我国跨境电商行业规模达14.2万亿元（包含撮合成交部分），同比增长14%；其中跨境电商出口规模超10万亿，同比增长13%；以B2C形式直接售出商品在19年已达3.4万亿，占比跨境出口2成。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/806503267632d5e787daa87.39209775.png" referrerpolicy="no-referrer"></p>
<p>与此同时，我国跨境电商凭借<span><strong>高性价比、高效率</strong></span>等优势表现强劲，成为中国新的经济增长点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1158540603632d5e78a97a37.51810326.png" referrerpolicy="no-referrer"></p>
<p>按产业链划分，跨境电商全链路的核心元素包含：上游供应商/品牌商，中游跨境电商交易平台/服务商，以及下游渠道商/终端消费者。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1244968354632d5e78f3d766.74961207.png" referrerpolicy="no-referrer"></p>
<p><span><strong>疫情促使消费者涌上，但在某些方面，也增加了跨境电商的运营成本。</strong></span></p>
<p>2020年下旬，海运费开始逐渐上升。2021年上旬，苏伊士运河阻塞，影响我国跨境电商欧线和部分美东线的运输。之后，美国港口卡车司机罢工事件使集装箱滞留港口...美线价格在峰值时期达到22000美元。</p>
<p><span><strong>一系列事件将海运费推上高峰，对跨境卖家的物流成本带来巨大压力。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1366268674632d5e793061d2.76030534.png" referrerpolicy="no-referrer"></p>
<p>从公司运营角度看，跨境电商与国内的消费品和电商公司差异极大，由于跨国的缘故，<span><strong>跨境电商物流成本占比更大。除此之外，平台佣金、营销、人员等方面的支出也在挤压着产品成本的占比。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/497616034632d5e7966c645.61394896.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong>狂欢后，行业大洗牌</strong></span></h2>
<p>“躺赢”的时代过去了，现在的玩家需要跳得更高，才能够到更多果实。</p>
<p>Shopee的降本增效只是时代下的投影之一，事实上，跨境电商整体都面临了这个难题。</p>
<p>Shopee成立于2015年，依托东南亚的人口红利，疯狂生长。在阿里的Lazada、印尼本土电商Tokopedia、Bukalapk、越南的Tiki甚至全球的TikTok都没能占领东南亚的背景下，Shopee已站稳脚跟。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1944288302632d5e79863590.79997449.png" referrerpolicy="no-referrer"></p>
<p>2021年Shopee激进扩张，集团Sea还以售股方式融资了60亿美元来推动全球扩张。</p>
<p><span><strong>但由于全球经济环境变化，加上市场竞争加剧，Shopee受到了越来越多的挑战。</strong></span>8月16日，Sea的Q2财报显示，营收同比增长29.0%，但净亏损达到9.312亿美元，比上年同期的4.337亿美元超出一倍还多。</p>
<p>二级市场上，Sea的跌幅也已超过80%。</p>
<p>回头看我国跨境电商市场，参与者很多，但以中腰部企业为主。</p>
<p>在全球贸易政策不确定性增加、亚马逊封号影响延续等冲击下，竞争将进一步加剧，行业格局有望向具备核心竞争壁垒的头部集中。<span></span></p>
<p><strong>此时，品牌力、渠道力和运营力将成为衡量跨境电商厂商竞争力的核心指标，也是企业能否突围的关键。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1922589989632d5e79ae5e39.59088220.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong>前路坎坷，但仍有机遇</strong></span></h2>
<p>在物流费用高企、原材料成本上涨、平台合规整顿背景下，<span><strong>跨境电商行业进入增长瓶颈。巨头们纷纷寻找其他的突破口。</strong></span></p>
<p>36氪研究院在此前的报告《2022年中国跨境电商行业研究报告》中表示，行业可能会有三大趋势：</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">跨境电商支持服务商市场潜力巨大；</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">行业格局向头部集中，强者恒强；</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌化、多元化、精细化成发展新趋势。</p>
</li>
</ul>
<p>另外，多家机构都表示看好东南亚市场，认为那里还是一片蓝海，渗透空间广阔，未来将持续作为重要的增量贡献区域。</p>
