<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看多了宏大叙事、扎心煽情的品牌创意后，这一届用户开始越来越青睐那些能为生活带来一点小乐趣、小情趣的“小创意”，比如前段时间刷屏的瑞幸X椰树，肯德基可达鸭就是如此。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而最近，<strong><a href="https://www.niaogebiji.com/tag/%E9%A5%BF%E4%BA%86%E4%B9%88" target="_blank">饿了么</a></strong>又再次验证了“小创意”也能带来大惊喜。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>我们常见的外卖小票，被饿了么变成了一个全新的“社交工具”，带着年轻用户们一起</span><span>「以小票之名」</span><span>，在小票备注里进行趣味沟通，成功掀起了一场浪漫的社交热潮。</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“热衷社交”是当代年轻人的一大标签。但在长期沉溺于线上虚拟社交，导致“人均社恐”的年代，年轻人们反而越来越渴望能够用一种创意又走心的方式，来加强现实世界的社交联系，去传递、分享那些生活中的情感与情绪。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>饿了么小票早就已经被年轻用户们瞄准，成为沟通与分享的新工具。过往在社交平台上，就不乏许多用户分享的趣味备注、暖心备注。</span><span>这一次，饿了么从用户故事里提炼营销创意，开发出“以小票之名”的玩法，凸显小票的“社交价值”，就是在正式回应当代年轻人的新式社交诉求。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1657720162fb276801b0c0.03923515.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>满足社交需求的基础上，饿了么还针对具体人群、场景需求，从收集来的100个年轻用户外卖故事里， 选出6个进行创意演绎，为用户们率先“打样”，展示小票备注的花式玩法。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>饿了么小票还能怎么玩？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>原来饿了么</span><span>小票备注可以<strong>「异地传情」</strong>，将异地恋人之间的情意随时随地，变成看得见的心意。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小票备注还可以<strong>「拯救职场不开心」</strong>，高情商玩转职场，用一杯奶茶就能善意提醒老板下班——下班的快乐就此轻松收获。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没想到小票备注还能<strong>「向TA求婚」</strong>，人生大事的仪式感不一定要兴师动众，用心为爱人制造的小惊喜，往往最动人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>「超诚意道歉」</span></strong><span>也可以用小票备注，一段真挚表达加一杯好喝的饮品求原谅，才叫诚意满满。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小票备注甚至还可以<strong>「规避风险」</strong>，随时温馨提示用户网聊需谨慎，一路走在反诈前线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小票备注还能带来</span><span><strong>「清凉小惊喜」</strong></span><span>。户外露营想来点清凉感，小票备注帮你转达，好心商家就能送来满满大桶冰，让你清凉一“夏”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这些覆盖多场景、多人群的小票备注创意，让“<a href="https://www.niaogebiji.com/tag/%E5%B0%8F%E7%A5%A8%E7%A4%BE%E4%BA%A4" target="_blank">小票社交</a>功能”变成了有代入感、可借鉴的玩</span></span><span>法，</span><span><span>启发着更多用户加入小票社交，并基于各自的场景需求去建立更多有趣、温馨、有爱的社交沟通。</span><span>为了将这种社交玩法更快地传播、扩散给用户，饿了么还联动多方资源，掀起了“小票社交流行”。</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谁能料到，一张外卖“小票”有一天会有这么大的“排面”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>饿了么充分发挥其“社牛”属性，将城市餐饮、芒果TV等品牌，人气明星以及广大网友用户，全都调动起来，把小票备注玩成了“脑洞创意大赏”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1）品牌联动：创意玩梗调动用户热情</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌商家向来是饿了么触达广大用户的一大端口，小票社交的流行自然少不了它们的合作推动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>饿了么联动长沙、杭州两座网红城市的</span><span>27家</span><span>餐饮品牌，超</span><span>3000家</span><span>门店，还同时</span><span>跨界联名最火最热电视IP</span><span>芒果TV</span></span><span>，</span><span><span>打造了系列</span><span>“创意人设小票”</span><span>以及联名杯套、背包等创意周边，解锁了小票新脑洞。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/108001747062fb2769cd5f81.01599832.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些品牌小票被做成一张张“好运符”，并在备注里模仿着各种人设的高情商社交口吻，手把手教用户趣味表达，也将活力、好运和祝福一起传递给用户。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>芒果TV日常霸总超有梗，不能包下鱼塘就包了今天的奶茶；孤傲又孤单的都市人都是江湖高手，在哪里都能留下传说，必配一张江湖高手符彰显气质；叱咤职场符最适合奋斗中的打工人，奶茶加料，能力有料，好运常在；白莲花笑里藏刀，以柔克刚；励志废柴也能闪闪发光，吃好喝好就是超能力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/93532149962fb276a9e4228.38653563.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>各种夸张的趣味人设小票，</span><span>让不少用户都忍不住对号入座，借外卖晒出自己的心声和好运期待。</span><span>芒果TV甚至还贴心地为每一种人设准备了观影指南。而通过</span><span>这些品牌联动小票，饿了么不仅直接将玩法推广给真实用户</span><span>，</span><span>也让合作品牌们在趣味触达与互动中，收获着年轻用户的好感。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/184458213462fb276c0e77b1.20092821.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2）明星联动：宠粉小票撬动粉丝传播力</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不止有品牌，饿了么还邀约了刘恺威，赵露思，李纯，刘也，方书剑等一众明星艺人，集体为饿了么小票活动打call，并在小票上写下他们想对粉丝说的话。同时明星们也对自家粉丝发起邀约，邀请他们在小票备注上，写下想对爱豆们说的话。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/148224084662fb276c891204.18815205.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/185801919762fb276d177171.28752437.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这里，饿了么小票又为了明星与粉丝沟通的创新渠道。<strong>而这招宠粉玩法，不仅调动粉丝参与热情，更撬动了粉丝们的传播力</strong>——明星粉丝在各个社交平台上都是活跃的社交人群，他们常常能快速带动品牌信息和营销活动出圈。饿了么邀明星与粉丝助力等于是带“小票社交”传播驶入了一条快车道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>3）用户联动：线上、线下趣味吸引自来水传播</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，任何能够掀起热潮的营销，背后最有力的推动者还得数普通网友、用户。这一次，007注意到，不管是线上还是线下，饿了么都凭借一系列互动事件，成功收获了不少用户“自来水”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在线下，饿了么将小票社交故事结合城市氛围，进行了浪漫传播。<strong>9个趣味小票站台落地杭州，讲述着小票备注背后的那些趣味、暖心故事。</strong>设计感和氛围感都拉满的小票广告，很快被KOL和普通用户们发掘，将其变成了网红打卡点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/165371723062fb276e382849.62171133.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在另一边，长沙黄兴路步行街恋爱天台还举办了一场<strong>“饿了么小票展”</strong>。在这里不仅能看到小票故事，各种品牌联合创意，小票还被放大、悬挂，被制成镜面装置，在光影中变成了出片神器。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/160200966462fb2770203b70.24218645.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管是户外广告还是小票展览，均在扩散“小票社交”创意，打造活动氛围外，带动着线下用户主动打卡、晒图，实现了</span><span>“线下创意反哺线上传播”</span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>在线上，一个</span><span>#杭州打工人准点下班小技巧#</span><span>本地热搜话题，成功将饿了么小票带入热议中心，截至目前话题已超</span></span><strong><span>6000W</span></strong><span><span>阅读。作为能够</span><span>「拯救职场不开心」的利器，饿了么小票备注提醒老板早下班的玩法，通过户外广告被带到了打工人上下班路上，趁着“秋天第一杯”奶茶的热点加持，被用户们纷纷效仿、讨论。不知道杭州秋天的第一杯奶茶，有多少是为老板点的呢。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/83201345062fb2772d1f912.54087673.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>长沙小票展也带着<strong>#长沙人有多喜欢看展#、#小票备注还能这样玩#</strong>的双话题跻身长沙热搜榜第七， 曝光达</span><span><strong>7700W</strong></span><span>。话题中不少达人+素人通过自来水传播，分享着各类奇葩小票备注内容。小红书以及抖音上也有不少长沙达人同步围绕订单小票进行种草，吸引更多本地用户前往平台下单，get同款商品、周边，整体曝光达</span><span><strong>873W</strong></span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/113909507762fb27736263d1.12519268.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个简单的小票备注创意，在饿了么推动下，成为了线上、线下都广泛卷入了用户参与的流行。看到这里，相信很多人都有疑问:</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么饿了么“小票”创意虽小，但依旧能够打动广大用户，掀起社交热潮？</span></p>
