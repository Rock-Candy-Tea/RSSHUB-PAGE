<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>中秋为山区小朋友收集月饼的<a href="https://www.niaogebiji.com/tag/%E8%85%BE%E8%AE%AF" target="_blank">腾讯</a>长鹅</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>中秋临近，资深公仔爱好者们惊喜地发现，腾讯养了二十多年的企鹅忽然长高了。区别于熟悉的圆脑瓜，红围脖的憨憨Q仔形象，谐音嫦娥的腾讯中秋“长鹅”，脖子长度堪比“火箭”，成为一个无论走到哪，都无法被忽视的存在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年中秋佳节，腾讯长鹅和腾讯志愿者协会联合顺丰、菜鸟等众多爱心企业一起，通过“一块月饼做好事”计划，和大凉山山区的孩子们分享节日的快乐。员工分享月饼，长鹅回赠魔性舞蹈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这只身形修长、舞蹈欢脱的长鹅，不止是腾讯员工眼中的中秋明星，更是全国人民的中秋现象级IP，在社交网络，视频网站，其魔性的身形和可爱形象，早已俘获了一众潮流粉丝的心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>搞笑长鹅出圈的背后，折射了鹅厂正在用IP履行社会价值，从产品视角出发，关注用户体验，用最接地气的、年轻视角的沟通方式和用户亲密互动。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>IP是年轻人眼里的潮流风向标，自带吸睛体质，天然与众不同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月3日，中国国际服务贸易交易会北京首钢园现场，重磅发布了冰墩墩雪容融裸眼3D数字形象，这是百年奥运史上IP形象首次以数字化的形式隆重展示。当日恰逢服贸会对公众开放首日，“首钢号”飞船一亮相，两只科技范儿满满又萌态十足的吉祥物“乘客”就立刻被现场热情爆棚的观众拍图发布到了社交网络。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>潮流IP是年轻人之间的“社交货币”。</span></strong><span>在提升参与者的表达自信和和感受力外，也为社交新体验找到了新的落脚之处：第一时间发布某个社交群体喜欢的独家、稀缺的“物料”，可以在社交网络和兴趣群组赢得更多的关注和互动。通过IP，可以很轻松地将日常的认知、审美和生活态度，有范儿地表达了出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/10361791636316761e79cde6.22715682.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>冰墩墩和雪容融</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早年故宫IP启动之初，产品设计者和运营者曾顾虑新IP没有市场而感到忐忑。但热爱国潮的年轻人，他们与生俱来的努力和文化自信，给了故宫IP双重加持。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>年轻人以追星的心态呵护自己喜爱的IP。</span></strong><span>他们乐意去在各种场景中，近距离接触到自己喜欢的人或者作品。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有段时间网上特别流行故宫的印章本：最初是有人分享了自己的故宫打卡记录，其它用户随后在短视频中刷到，被种了草——很想要买一本，并且立刻去行动了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多年轻用户为了集齐印章专门去故宫打卡。花60元买门票进了故宫，然后又花128元买了印章本集齐了30多个印章。拍完照发完朋友圈，围绕这个印章本的社交讨论，才刚刚开始。</span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><img src="https://ngbjimg.xy599.com/6434908416316761ef3ca09.03400019.jpeg" referrerpolicy="no-referrer"><span>腾讯长鹅和大白兔奶糖联动</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span data-font-family="default">今年中秋，腾讯长鹅和童年奶糖大白兔跨界联动：</span><span data-font-family="default">画面中，长鹅抱着玉兔，甜甜地宣布”今年长鹅和兔兔一起过中秋 ”。而大白兔奶糖也热情地回应“本兔终于找到你！长鹅吃糖！”网友纷纷表示，”谐音梗玩的太溜了“；“真就梦幻联动了，兔兔和企鹅奇奇怪怪的cp又增加了”。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>脑洞大开、新鲜有趣的新IP表达方式背后，是IP底层的情感联结和文化魅力，呼应着年轻人的天马行空，也在阐述着他们对于更舒适、更美好生活的理解。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1091009426316761f35d2d4.91559470.jpeg" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为腾讯最早的产品，QQ 企鹅是非常接地气的国民ICON。其大眼萌宠的产品形象，出街自带吸睛效果，陪伴了无数人的成长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>看到自家Q仔出门就被夸长得俊，鹅厂则像个骄傲的老母亲：</span></strong><span>他们一度戏称自己为深圳腾讯公仔厂，二十多年来的主要业务就是生产企鹅公仔，保证公仔质量。用户接收到了这种幽默，心领神会，在社交网络掀起更热烈的讨论。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去二十多年，腾讯企鹅一直是用户眼中萌萌的、矮墩墩的Q仔形象。从2016年中秋，Q仔已经开始调侃自己是“长鹅”。2020年，有着长长身形的长鹅周边开始出圈，不但身形纤长了几倍，还以每年10厘米的速度在长高。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>年年长高这件事本身就勾着用户的好奇心，今年还没过完就已经开始期待明年长鹅长啥样。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>长鹅家族里有个脖子高如火箭的扭扭鹅，因为脖子长出天际，而被网友称为“长鹅老大哥”。其灌风妖娆的舞姿，发丝凌乱的搞笑头发视频，被年轻人逮住反复鬼畜创作，挤爆了社交网络搞笑版，早已晋升为网红鹅。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2281938476316761f6bac09.80935938.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>腾讯image形象店里的网红长鹅</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前年放荡不羁，去年憨厚可靠，今年卖艺献花。</span><span>极具辨识度的形象出街后，肩负使命的鹅厂新IP和新潮玩诞生了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去两年，它和嫦娥之父、中科院院士欧阳自远连线跨界科普，一起探索月亮的秘密；突破次元壁，和“吴刚”一起庆中秋，在社交网络嚷嚷要到广寒宫去。今年，它又怀抱大白兔，手举小红花，和腾讯志愿者一起倡导“一块月饼做好事”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>玩梗玩到新IP诞生，是一家大公司定位和心态上的与时俱进，他们在用另一种方式，与年轻人对话。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>通过一个IP形象，把科技、公益文化，科普等认知，以更容易被人接受和喜欢的方式，带给了更多的年轻人。</span><span>这背后是信心，底气和跟用户之间很深的情感联</span><span>结。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20770025606316761f91df11.87529253.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>腾讯长鹅成为年轻人喜爱的中秋伴手礼</span></h4>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>作为腾讯IP全家桶中的一员，长鹅不仅仅是一个IP，它天然与IP自身的基因和生态有更多的连接。</span></span><span><span>今年中秋，恰逢99公益日，腾讯长鹅IP与99公益日品牌形象“小红花”的联动背后，</span></span><span>是一家大公司对公益20年如一日的坚定理解。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>IP联名能够产生1+1大于2的神奇效果。</span><span>不但拉近了人和人之间的距离，还引领着公众的集体注意力，去关注和支持那些触手可及的公益。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>老牌互联网巨头公司阿里巴巴在阿里公益里，今年首次把自己的公益IP形象“一颗发芽的爱豆”变成玩法主角。</span><span>为了助力乡村振兴，国民IP动画斗破苍穹主角萧炎</span><span>带着公益墙绘走进乡村，以农民群众喜闻乐见的方式，参与到乡村振兴中来。</span></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/21218876686316761fdba023.94921948.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>腾讯image形象店里的“鹅纸”磁力贴</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>而围绕公益，腾讯像打造产品一样，打造了服务公益矩阵的产品体系和伙伴生态。</span><span>把连接公益合作伙伴、受助人、捐助人、每一个连接和关注公益的普通人的底层基础设施做好，再通过移动化支付、社交化场景和走心化互动等，唤起社会各界关心、参与公益的热情。</span><span>在这个过程中，腾讯既出钱又出力，与多家企业及NGO联动，产品化、渠道化、系统化地重度支持和关注了每一个细分垂类项目。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>聚焦IP，能够</span><span>和年轻人对上暗号的潮流IP门槛很高：它本质上，是一个需要融入情感、艺术、设计、绘画、雕塑、动漫等理念的、具有清晰价值观和鲜明拟人性格的多元形象。</span><span>如何设计、定义和养成直抵人心的IP形象，真正将情感、艺术、表达等多元使命融入潮流，让更多IP和人连接起来，服务社会，创造价值，或许应该是这个话题下更为宏观的公共意义。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>近日，在中秋长鹅快闪活动现场：</span></strong><span><span>小小QQ的，可以贴在冰箱上扭来扭曲的小“鹅纸”；</span></span><span>身形修长的长鹅抱枕，几十厘米的长鹅桌面玩具</span><span>，拉</span><span>得比人还高的大型纸艺长鹅“鹅纸”，</span><span>和刷屏的网红“扭扭鹅”一起，众鹅云集，造型欢乐，直接把节日氛围拉满了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/653311545631676201f46f5.58730338.jpeg" referrerpolicy="no-referrer"></p>
