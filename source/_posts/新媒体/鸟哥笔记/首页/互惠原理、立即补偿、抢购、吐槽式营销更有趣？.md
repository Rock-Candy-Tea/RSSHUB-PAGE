<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>1.试用中还有互惠原理</span></strong><strong><span>？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>试用作为一种<span>营销</span>手段，成功关键点是利用</span><span><strong><span>“免费”和“试用”</span></strong></span><span>来吸引潜在的消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>常规而言，免费试用商品都是新产品或产品试用装，一般有三个步骤才能凸显出「试用」带来的回馈：</span><strong><span>1）限时限量进行，2）记录使用者的信息，3）反馈信息。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先假设试用活动没有<strong>时间和数量的限制</strong>，那对消费者来说就不会拥有<strong>紧迫感</strong>，固然在意的程度便会减少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次免费试用尽可能采用<strong>“申请制”</strong>，邀请消费者帮忙填写表单，留下姓名、手机号码、地址等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面商家可以掌握潜在消费者实际信息，另一方面可以避免部分消费者不停地申请免费试用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再者消费者收到商品后的2-3天内<strong>记得追踪试用的体验如何</strong>，此刻更有利于进一步优化下次的活动，同时又起到提醒作用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人会觉得，为什么免费试用活动如此火爆，仅仅是抓住消费者喜欢免费的心态吗？并不是。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>心理学中有一种现象叫</span><span><strong><span>“互惠原理”</span></strong></span><span>，指的是别人给予我们何种程度的好处，我们也要回应对方同等的回报。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>表面虽然是试用者占优势，实际上试用者会产生亏欠感带来的贪小便宜心理，认为自己必须购买商家的产品，从而刺激其产生购买行为。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随便举个例子：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有家洗涤液公司一直没有打开市场，该企业通过给几十个小区发放2万多份洗涤剂试用装让居民亲测，最后选用家庭主妇作为代言人，录视频在社交媒体播放却带来很好效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，从中可以看出，</span><strong><span>消费者对商家自己吆喝的宣传方式并不信任；免费试用可以让消费者亲自检验产品</span></strong><span>，这样既有利于扩大产品的市场占有率，又可以提高产品的知名度。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>2.抽奖有哪些背后秘密？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抽奖方式非常多，但要注意三个问题：</span><strong><span>1）符合产品情况，2）合理设置中奖比例，3）奖品的相对价值</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>饮料抽奖经常多见，被众多品牌效仿，“再来一瓶”可以给消费者制造一种惊喜效果；因为消费者不太考虑单次支付低价格商品的潜在风险。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>假设你用在其他领域可能就有所不同，比如「电动车、电子产品」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次像彩票活动的中奖率非常低，一般消费者的购买心态不是非常强烈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但高中奖比例活动则不然，消费者会认为自己中奖的可能性大，因此参与的热情也会非常高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然并不是中奖比例越高越好，要结合奖品和商家的目的来决定；再者，</span><strong><span>奖品</span></strong><span><strong>的价值不单指商品的货币价值，而是相对价值。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有时候价值200元的奖品未必比价值100元的奖品更有吸引力，也有可能价值50元的购物券更受欢迎。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，“购物满抽奖”“附送赠品”“购物返券”等营销方式都是商家充分运用消费心理的表现，这种有奖销售模式也非常吸引消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从心理学上分析，有奖销售涵盖两个概念：</span><span><strong><span>立即</span><span>补<span>偿和延迟补偿。</span></span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前者指消费者在产生购买行为后，商家立即给予消费者指定</span><span>补<span>偿的行为，如买一赠一、买洗衣液赠洗脸盆等。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而后者指消费者在产生购买行为后，可以获得不确定的回报，如集齐印花换购超值礼品等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在上述确定与不确定报偿的激励下，消费者更倾向于前者，因为确定性报偿可以带来一种安全感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在相同条件下，<strong>人的本能会驱使自我做出风险小的选择，没有人希望自己成为输家。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，也有人会尝试不确定报偿，因为其结果往往更诱人；要注意，不论如何抽奖都要围绕三个目的进行：</span><span><strong><span>提高知名度、增加消费者信任感、扩大市场占有率。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，消费者期望自己在本次抽奖活动中得到一等奖，把特定奖设定为品牌商品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而事实这种结果存在明显的不确定性，可消费者愿意承担不确定性带来的风险；但在活动中也要避免同质化现象。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>3.抢购，唯品会的成功策略？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>限时抢购（Flash Sale）又叫“闪购”，例如，商家在特定时段推出某种商品，在原价基础上给予1折至5折不等的特价优惠力度；本次优惠活动持续三天，先买先得，售完即止。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当年，唯品会正是抓住消费者对打折特卖的需求点，凭借独特的</span><span><strong><span>“精选品牌+深度折扣+限时抢购”</span></strong></span><span>的正品特卖模式，成为当下中国头部电商业态之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从商家角度，以限时抢购的方式进行营销宣传不失为一种好方法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一方面，消费者需要安全感。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人出于对安全性的考虑会囤积货物，这种“晴带雨伞、饱带饥粮”的预备性心理会促使人产生购物行为；例如，<strong>“我现在买了，没准以后就可以用得上”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另一方面，“稀缺性”商品具有吸引力。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“限时”和“抢购”这两个极具引导性的字眼，在时间性和商品数量上，暗自传递给消费者关于商品的信息。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现实情况下，做限时抢购活动的商家很多但效果参差不齐，如何才能将限时抢购活动做得有声有色呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有四注意事项：</span><strong><span>1）提前准备并预热，2）设计阶段优惠，3）小量多返场，4）制造噱头</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先尽可能</span><span><strong>提前2-3天预热</strong></span><span>并说明阶段优惠力度，尽可能设计</span><span><strong>第一波就能抢购完</strong></span><span>的数额，其次</span><span><strong>再二次返场</strong></span><span>，这样过程中就会产生“被秒杀”完的紧张噱头。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得注意的是，在进行限时购活动时肯定要控制好成本问题和频次问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“销量越高亏损越大”不是商业交易的本质，互惠互利才是商家与消费者的共赢之路，<strong>频次多了即缺少稀缺性。</strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>4.抢眼的玩法，竞拍？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早些年，淘宝的忠实粉丝都知道，在平台有个“一元起拍”的活动专区，类似于拍卖会1元网上竞价活动，现在已经取消。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>竞拍类型有两种：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>一是单件拍</strong>，即只有一件拍卖品，出价高者得；</span><strong><span>二是荷兰式拍卖</span></strong><span>，拍卖品数量大于1，依照价高者优先获得宝贝，相同价格先出价者先得的原则，最终商品成交价是最低成功出价的价格。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如：3件商品参与拍卖，那么成功出价的总排名中的第三位的价格，可视为最终成交价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在很多DTC品牌也在自己小程序做，对营销者来说，此类活动可增加小程序曝光度，还能起到社交裂变效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但，也要控制好其他方面的问题，比如：</span><span><strong><span>1）成本的控制，2）关联的营销</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>控制成本本身防止亏损太多；其次，虽然竞拍本身和社交拼团相似，但也要结合转化率和客单价综合分析，<strong>我认为做好关联营销，有助于提高客单价。</strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>5.吐槽式营销，更有趣？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>相比网络的差评，大家对热门话题吐槽的热情更胜一筹；例如：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>万达王健林在东方卫视做客鲁豫有约节目，随后东方卫视放出一组截图刷爆朋友圈，其说：</span><strong><span>“想做世界首富要先定好一个小目标，比如我先挣TA一个亿”</span></strong><span>。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此言一出被网络震撼，引发集体转赞分享；要知道高手在民间，大家纷纷开启“小目标”式吐槽。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如：“找男朋友方向是对的，但是要先定一个小目标我先搞定男神宁泽涛”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>为什么网友对“小目标”如此感慨？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>因为在惊讶中完成自我对照的所有步骤后，大多数情况下是无奈的接受，首富和普通人在金融思维方面差异对比鲜明；所以，</span><strong><span>带有调侃的吐槽言论更多是宣泄自己的内心情绪。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由此可见，吐槽带来愉悦同时还能够收获最佳营销效果，但也有搞砸的情况；当年联想欲借王健林的“小目标”给自家的产品做个宣传，没想到遭到了网友的狂吐槽。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总之，吐槽和批评/批判有明显区别；</span><span>批判，角度和立意更深刻、更严肃；</span><span>吐槽则偏向随意自由，更多是调侃的意味。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>固然营销人员必须把握好吐槽的意境，避免将营销策划向着批评/批判的方向发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>对此，吐槽的特点我认为有三个方面：</span><strong><span>1）恶搞中诙谐幽默，2）抓取某个单点，3）不强调互动性。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与其他的营销方式不同，吐槽不强调双方的互动性；通常是吐槽方狂吐，被吐槽者沉默或一笑而过；相比互掐激烈场面其更温和一点。</span></p>
