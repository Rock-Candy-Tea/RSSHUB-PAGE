<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">今天跟大家聊的话题是代言的玄学。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这几年的内娱告诉我们：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌代言千千万， 今天塌房明天换。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">郑爽偷税，Prada眼泪Prada Prada掉。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">吴亦凡塌房，韩束们火速解约。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">王力宏被锤，英菲尼迪开创日抛型代言。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">背调的呼声此起彼伏，还有网友给自己祈祷。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">希望我的爱豆多多代言。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">希望我的爱豆永不塌房。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但最近看到欧阳娜娜、<a href="https://www.niaogebiji.com/tag/%E9%BA%A6%E5%BD%93%E5%8A%B3" target="_blank">麦当劳</a>的联动之后，我觉得内娱代言又可以了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/723115795623260770c18c4.52658385.jpeg" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">简单介绍下背景，之前欧阳娜娜加入麦当劳“明星热爱之选” ，给板烧鸡腿堡做“代言”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只要最近有去吃麦当劳，就会发现麦当劳还推出板烧鸡腿堡10.9元活动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有意思的是，有人扒到欧阳娜娜的一场直播，发现娜比代言麦当劳的真相：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">麦当劳是从小吃到大，它不只是一个商务，还是一个认可。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17256813262af9ef5ed18f3.65488334.png" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这画风就像是，小时候写作文遇到题目叫做《长大之后我想……》</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我写的是想吃垮麦当劳，长大后我真有了终身免费吃麦当劳的机会。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40152403962af9ef66b0ad5.61739311.gif" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">欧阳娜娜就像是我的互联网嘴替。一时之间，网友都不知道究竟该羡慕麦当劳，还是羡慕欧阳娜娜。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>

<p> </p>
<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">别家明星代言：恰顿饭而已。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">欧阳娜娜代言：实现了梦想。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小时候最喜欢吃麦当劳，长大了和麦当劳一起合作。这不就是常说的双向奔赴？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以娜比的发言，猛然让我想到椰树椰汁，椰树椰汁和代言人徐冬冬的故事，一直都是佳话一般的存在。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">徐冬冬曾说：椰树椰汁，从小就喜欢喝，从小喝到大。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81812425262af9efe1c17a5.12879719.png" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">麦当劳没有椰树椰汁的命，被椰树代言套路“带坏”了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过看到这儿，我也算明白为啥麦当劳会选欧阳娜娜做代言。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要是自己的热爱，有朝一日变成工作，快乐自然不言而喻。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就是麦当劳要传递“简简单单，爱你所爱”的文化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205789392362326078647a25.53043458.jpeg" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果说欧阳娜娜代言麦当劳，这是：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">含情“麦麦”，“板”上钉钉。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">蓦然回首，“娜”人却在灯火珊澜处。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么硬汉吴京代言国民火腿肠 双汇，那就是：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一鸣京人，双喜临门。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">念念不忘，必有汇响。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">渐入佳京，如愿以肠。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这事还要从2013年说起，吴京发了一条微博，自荐做火腿肠代言人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“请问有火腿肠广告找我拍吗？”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/189574944762af9efe659df3.38083866.png" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只可惜呀，吴京有意，品牌无情。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">谁知8年后，双汇和吴京还能再续前缘。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">去年#为吴京圆8年前的梦#话题突然被推上热搜，网友为吴京圆火腿肠代言梦操碎了心，化身月老红娘牵线搭桥，疯狂艾特双汇等火腿肠品牌。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/142302832662af9eff159c10.30203772.png" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/131822144262af9eff419384.59740810.png" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这下可把压力就给到了双汇这边。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">后来双汇带着“惊喜”来了，高调邀请吴京老师做代言人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/49177251762af9eff7df556.34484253.png" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">风里雨里双汇等你，代言一场京京相惜。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">吴京听到粉丝的呼声，双汇也等来了吴京的好消息。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/155009481862af9f001e4579.44780490.png" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">吴京和双汇王中王，演绎无心插柳柳成“姻”：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">强强联合，必有“京喜”。 一人代言，全国圆梦。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一个“愿给”，一个“敢收” ，这个大概就是：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">双向奔赴的代言也许会迟到，但一定不会缺席。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134948845962326079426029.07408166.jpeg" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一个品牌签一个代言人，可能是来自网友的压力，还可能因为撞脸。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我说的不是刘亦菲和唐僧，而是雷军和苏炳添。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/155037409262af9f0101d646.49063260.jpeg" title="从小吃到大，麦当劳被椰树“带坏”了？" alt="从小吃到大，麦当劳被椰树“带坏”了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">去年的东京奥运会男子百米决赛，苏炳添以9.83秒的成绩，打破亚洲记录。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">隔了几天时间，小米坐不住了，火速官宣苏炳添为品牌代言人。</p>
