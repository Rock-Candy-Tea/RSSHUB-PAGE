                        <h1 id="toc-1" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); background-position: 0px 5px; margin: 25px 0px 20px; background-size: 16px 20px; font-weight: 600; text-align: center;">一：IP矩阵价值</h1><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">做生意凭借的是信任。我相信你的产品好，你不会骗我，所以我买你的。没有信任也就没有交换。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">用户对企业的信任，由品牌、产品、社交三个层面结合构建。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">品牌层面的信任</span>，需要品牌势能强，肯真金白银投入大量资源做营销推广，用户见多了，心智自然就被影响。典型的就如可口可乐、星巴克、麦当劳。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">产品层面的信任</span>，需要用户购买过企业产品，在使用产品的过程中，感受到产品质量好、效果佳，那么自然会对企业产生信任。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">社交层面的信任</span>，由用户在与企业员工交往过程中产生。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">在线下零售业态中，用户与一线员工面对面接触，成为朋友，自然会将这一份信任感迁移至品牌。通过相信你，进而相信你背后所在的企业，最终购买你所在企业的产品。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">品牌、产品层面的构建需要长时间的积累。基于<a href="https://www.niaogebiji.com/tag/%E7%A7%81%E5%9F%9F" target="_blank">私域</a>与用户面对面0距离接触的特点，企业可在私域中搭建人设IP矩阵，将线下信任介体线上化，以低成本放大社交层面的信任，实现四两拨千斤。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">通过线上人设IP矩阵，企业与用户成为朋友。<span style="font-weight: bold;">在用户的购买决策路径中传递感性价值，增强情感链接，影响心智。</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">对低客单、简单决策的产品，实现快速决策下单，一次成交；对高客单、复杂决策的产品，在业务的关键决策起到“快速推进”。</p><h1 id="toc-2" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-size: 16px 20px; font-weight: 600; text-align: center;">二：IP矩阵类型</h1><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">在线下，用户会接触到的企业成员有老板、服务员、导购、专家等等。人设IP矩阵搭建，就是将这些成员线上化。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">IP不同，职责不同。很多企业的私域体系，往往只存在1-2个IP，彼此间定位不清晰，职责不明确，用户看了稀里糊涂。<span style="font-weight: bold;">IP无法塑造，信任关系的构建也就极为困难。</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">私域的人设IP分为5类：<span style="font-weight: bold;"><a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8CIP" target="_blank">品牌IP</a>、专家IP、创始人IP、销售IP、助理IP</span>。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">不同产品类目、不同发展阶段的企业，所需要搭配的人设矩阵不同，以上5类IP并不是每个企业都需要具备。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">举几个例子：</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">初创、规模较小、简单决策的零售企业，需要创始人IP讲情怀。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">美妆、母婴这类需要教育、提升认知的品类，在创始人IP、销售IP的基础上，还需要搭配专家IP，根据业务线，甚至要搭配多个专家IP。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">等到企业发展到了一定阶段，还需要打造品牌IP，塑造品牌调性、价值观。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">那么，以上几个IP之间他们具体分工、职责、调性有什么区别呢？</p><h2 style="margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-position: 0px 5px; font-size: 18px; color: #212629; font-weight: 600; text-align: center;">一：品牌IP</h2><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">在产品高度同质化的今天，用户面对同质的产品，买不买，更多的是品牌调性是否符合胃口、价值观是否一致。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">在私域中，企业可以将冷冰冰、看不到摸不着的品牌，化为有温度，有感情的IP，通过与用户的互动，传递品牌价值、调性、格调。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">IP化作“人”，用“人”去接近用户，服务用户，更易被接受，也更易产生好感。</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">品牌IP，一般为虚拟形象。设计需符合品牌调性、格调、价值观。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">用户与企业线上、线下接触的多维度触点、宣传推广素材、产品包装都可与品牌IP形象结合。用户一见到这个IP形象，就会想到这个品牌。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">比如，星巴克的“熊店长”。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">在线上社群，他作为一个活生生的“人”，说着“人话”，和用户沟通、交流、互动。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-size: 8px 20px; padding-left: 12px; font-weight: 600; text-align: center;">二：专家IP</h2><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">在决策购买前，让用户觉得专家是可靠的，是专业的，是值得信赖的，进而将这份信任关系转移到品牌上，从而购买产品；</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">在决策购买后，通过专业知识，增进私域价值感，延长用户留存率，拉长生命周期价值。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;
