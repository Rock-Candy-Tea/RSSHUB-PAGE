<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自去年以来，ofo<a href="https://www.niaogebiji.com/tag/%E5%B0%8F%E9%BB%84%E8%BD%A6" target="_blank">小黄车</a>接连因为“什么时候退押金”冲上热搜。此后，在网友计算还要排队等待988年的声讨中，小黄车似乎已销声匿迹。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>最近，有网友发现，</span><span><strong>“ofo小黄车”的官方公众号还在持续更新</strong></span><span>。并且，推送内容基本都是吸引眼球的标题党八卦文，阅读量常有10万+。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这让网友很疑惑——不退押金的小黄车，又在暗戳戳搞事情？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70077011662540cc779d811.91220037.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left; padding-left: 12px;"><span><strong><span>博取流量</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前两天，有媒体爆料称，网友发现ofo小黄车的公众号还在持续更新。只不过，推送内容一言难尽。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，黄晓明闫妮官宣结婚？相爱10年的红颜，竟然是她？40岁王心凌唱《山海》热搜登顶：被渣男骗光所有……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1254988650630811dc1cdc23.93581941.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：</span><span>环球Tech网易号</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有夸张的：一路走好！李亚鹏：骨灰已经撒进大海，王菲依然……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/487326251630811dc568f49.59362312.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span></span><span>图源：环球Tech网易号</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>晃眼一看，以为误点了某个靠标题博出位的娱乐八卦号。再仔细一瞅，确实是那个N多网友排队等退款的正经ofo小黄车。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>消息发酵后，</span><span><strong>目前上面提到的文章已经删除</strong></span><span>，仅留下在2021年3月至8月期间发布的几篇推文。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1764750959630811dc9247d9.89516477.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：</span><span>ofo小黄书公众号截图</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>点击首篇《52岁不老仙妻保养秘诀流出》的文章，从乘风破浪姐姐们的好状态，写到了“我”闺蜜惊人的逆龄变化，再到各种真人现身案例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过贩卖年龄与健康的焦虑，毫无悬念推出了两位“高人”联袂巨献的秋季活水饮食法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1683371087630811dd106758.09301350.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：</span><span>ofo小黄书公众号截图</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个限量29.9元扫码加入的今日福利，用来引导用户进入另一个公众号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在引导进入的公众号中，基本都是购书或共读限额福利的付费类目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原来，多少有些俄罗斯套娃似的推荐方式，是为了给其他公众号引流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/268524207630811dd67f774.82981505.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：</span><span>ofo小黄书公众号截图</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并且，ofo小黄车公众号中仅存的几篇诸如《结婚10年，0存款》等推文，都转载自该公众号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有网友猜测，ofo小黄车公众号之所以还在发布消息，或是想利用原有的粉丝流量，收取广告服务费等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，我们是不是可以大胆假设，ofo小黄车的押金退还指日可待了呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137231867662540cc8699195.09342959.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 蒙眼狂奔</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经，ofo小黄车和摩拜单车，是崛起速度最快、最受资本偏爱的互联网企业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，获得超过20亿美元融资的小黄车，从炙手可热的独角兽，到无人接盘的烫手山芋，也不过短短三年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2112537098630811ddb3a7e1.76171985.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：</span><span>ofo小黄书官方微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此次曝出擦边推文之前，媒体对小黄车的报道，基本停在了2021年。身处有记忆的互联网世界，创造扩张奇迹的小黄车，退场时并不体面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2015年，北大研究生戴威自己出钱买了200辆小黄车，并投放在北大校园内。同时提出“共享计划”，向学生回收单车作为共享单车。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓“共享计划”，是指如果用户共享了一辆单车，就可以获得全部小黄车的使用权。看起来，这个计划既有环保理念，又能便利校园交通。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/763694617630811de2868a1.15788707.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：</span><span>ofo小黄书官方微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，“共享计划”一经推出备受认可。戴威接着宣称，将在北大陆续投放1万辆共享单车，并招募2000位共享车主。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数月后，当小黄车在北大校园内日均订单突破4000单时，获得了第一笔900万元的融资。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚开始，ofo小黄车的扩张并不激进。仅在全国20多个城市、200多所高校中铺开，累积了80万用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/627040413630811de5e4542.36850056.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：</span><span>ofo小黄书官方微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，自2016年10月起，在大量资本的极速推动下，ofo小黄车先后经历了5轮累计超过2亿美元的融资。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>体现在市场上，则是还没找到成熟的盈利模式之前，ofo小黄车已经开始了年投放高达2300万辆的蒙眼狂奔。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种资本密集参与下的野蛮扩张，有悖于正常的商业逻辑，让原本为提供环保便捷出行方式的共享单车，沦为融资工具。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/127618355762540cc97137d2.72071747.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>屡遭吐槽</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当融资速度跟不上投放新车和补贴用户的支出时，面临竞对强势崛起的ofo小黄车，危机重重。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>被曝挪用30亿押金来支付供应链欠款，大幅裁员寻求接盘，预期收购价从30亿美元价降至14亿美元</span><span>……</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>跨进2018年，ofo小黄车已然强弩之末。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/569201157630811de9fb2d1.50471322.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：</span><span>ofo小黄书官方微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有投资人曾说：很多企业在整个生命周期里，都在向外界寻求资本，主动迎合投资者对于回报率的期待，以至于没有把如何盈利放在第一位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，假设没有资本的大举进入，ofo小黄车又会不会走的平稳一些呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在ofo小黄车的用户中，有校园学生、公司白领，也有普通打工者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对大数人来说，尤其是在一线城市，共享单车的出现，减少了碳排放、尾气污染，也给人们带来了更健康便捷的出行方式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/499233228630811dee78415.43667104.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：</span><span>ofo小黄书官方微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>因此，</span><span><strong>既有着环保先行的美好故事，又能成为线下流量新入口的共享单车，理应成为一门或许利润微薄，但可持续发展的好生意</strong></span><span>。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，共享单车的覆盖受众面广，很容易积累庞大的用户群体，并依托用户基数构建流量变现的盈利模式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过，ofo小黄车还没来得及找到“该去向何方”的盈利路径，就遭到了众多用户质疑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1519471231630811df385a90.32844738.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：</span><span>ofo小黄书官方微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一些小黄车被损坏之后，得不到维护，影响骑行的感受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并且，小黄车的押金退还时间长达半个月，客服的态度也不明确，以至于很多用户担心自己的押金退不回来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在黑猫投诉平台上，关于ofo小黄车的投诉有5万多条。最新的一条投诉中，网友甚至认为自己的登录都遭到限制。</span><span></span></p>
