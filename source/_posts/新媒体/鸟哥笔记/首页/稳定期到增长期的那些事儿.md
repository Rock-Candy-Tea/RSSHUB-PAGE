<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关注我的朋友应该知道，今年是我创业的第5年了，也是疫情来的第三年了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在还记得曾经我伙同几个同学在深圳发起做产品经理社区这件事，当初全力的高歌猛进带来的迅速增长，很快在第二年就做到了能够在1年举办几十场线下沙龙的产品经理社区，至少身边很多人听过我们就在那个时候奠定的基础。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但在2020年后，疫情影响下我们的业务不得不维持降低成本和裁员，业务处在<a href="https://www.niaogebiji.com/tag/%E7%A8%B3%E5%AE%9A%E6%9C%9F" target="_blank">稳定期</a>了，不敢轻易扩张，保持着收支平衡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以这几年我经历过业务爆发式的增长，也经历过稳定甚至停滞不前的稳定期，而人最怕的就是一成不变的生活。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>我们都喜欢<a href="https://www.niaogebiji.com/tag/%E5%A2%9E%E9%95%BF%E6%9C%9F" target="_blank">增长期</a>，讨厌稳定期。</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近1天我在朋友圈看到一张有趣的图，我认为特别有意思，或许你已经在网上其他地方浏览过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/133818945062d450b6b19381.06051887.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>指的是在创业和职场两种生活方式，职场的曲线可以更加知道预期，但创业则是有波谷和波峰，但增长却极高。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两者相似的地方是：都会存在有增长过程前的平稳期，比如在职场中你可能会有几年都没有办法晋升，在创业你也会存在业务停滞不前的情况。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>什么时候是拐点，都说不准，99%的人都倒在了拐点。所以在没有拐点前，</span></span><strong><span><span>我建议去</span></span><span>聚焦，才能抓住增长的机会。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我其实特别不习惯业务稳定的状态，尤其是当开发告诉我需要1-2个月时间去重构，新的产品ideal需求迟迟上不了线。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>1.产品的生命周期</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做过产品经理的朋友应该了解产品的生命周期，指的是任何一个产品都会经历探索期、增长期、稳定期、衰退期4个阶段，每个阶段都有特点，需要单独运营，才能保持产品源源活力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/66561689062d450b6dc0368.27834668.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图片来自网络</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在产品进入成熟期的阶段，我们需要投入人力埋头去精细化运营、专业知识沉淀、业务标准打磨，让我们好准备好下一个增长期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>我曾经特别讨厌稳定期，尤其是开发告诉我一个需求要2个月才能上线，除了焦虑的等待外，就只能希望自己再加资源去开发新的需求和ideal，但这却进入了一个畸形的发展，永远都得不到市场认证。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但其实这是不对的</span></strong><span>，越是在稳定期，越应该埋头下来聚焦在数据效果、规范、标准，去建立一个持久易用的SOP，找到一个靠谱的产品设计方向和策略。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每当产品进入到了成熟期，<strong>我们就要考虑到用迭代的方式来把产品又推向一个新的增长期。</strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>2.产品经理别去碰人脉和混圈子</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多想创业的人或者创业者都喜欢去拉人脉，但是如果你看我的文章，我特别建议产品经理，对于你来说最好的人脉<strong>就是一个可以用、完善、没有BUG的产品。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至少我创业到，对于做产品研发来说，人脉有没有用，却是用，但是一定要产品跑出了成功的自然增长曲线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你是做产品经理，想通过人脉拿offer，小公司可以但大厂仍然需要你的成功作品，比如工作履历、产品项目经验，仅仅是内推一个简历是没有任何价值的，尤其是在今天你可以在闲鱼上花几十块钱就可以让别人帮你在大厂内部平台投递简历（就是内推）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以在稳定期，花时间不断打磨产品的体验、优化产品的业务模式，是产品经理最聚焦的，产品好了用户自然会来，这样你就有人脉资源了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>稳定的时期，往往是团队内部整理、优化的好时机，利用这个时间段把自己的业务标准定型了、工作分工清晰了、产品运营策略固定了，其实就会越来越好的。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>3.要有行业的天花板意识</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果现在让我重新选择创业，如果要仍然以产品经理技能相关来选择，做产品研发仍然是一个最佳适合走的路线，因为产品经理知道产品研发的成本、流程，做一款工具APP、或者应用本身就不太难，天花板可以做得很高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是如果只是说产品经理的社区，那么这个赛道的天花板就太短了，反而不建议做了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以当业务处于平稳期，更要去思考更高的宏观层面，比如在一个四线城市做一个街舞培训，你不管花多大力气都难以做到1000万的营收，但是到了一线城市就不一样了，这样是一个道理，并不是方向的问题，而是起点的问题。</span></p>
