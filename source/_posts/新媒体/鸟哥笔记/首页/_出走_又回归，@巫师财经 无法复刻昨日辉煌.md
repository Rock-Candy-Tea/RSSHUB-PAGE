<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“巫师终于回来了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时隔两年，@巫师财经从西瓜视频回到了B站。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月17日，断更两年多的@巫师财经在B站发布了一支新视频，宣告自己正式回归。这支长达4小时的作品延续了其先前财经类目的创作方向，截至目前在B站获得了182万的播放量，堪称“开门红”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在视频末尾，@巫师财经提及，自己离开B站是为了追求混乱与更高的估值。“混乱是阶梯，只有视频赛道平台间的战火蔓延，创作者才会被需要，被争抢，被捧为座上宾。而机构也会在战火蔓延中获得更多估值。”</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无独有偶，回归的不止是@巫师财经，还有先前因言语争议被封禁的前动画区“一哥”@LexBurner。沸沸扬扬的“无职转生”事件，最终在去年12月以一纸通告收尾，@LexBurner被裁定赔偿2000万，其B站账号得以解封。但在内容层面，直到今年7月，Lex才恢复视频更新，并将创作重心放在了影视区。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>离开与回归，在B站上并不是什么新鲜事。不过，值得关注的是，平台生态千变万化，那些曾经离开的创作者，回归后内容还“吃香”吗？</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“当初作出决定的时刻，就是为了追求混乱和利益，已经将【老死不相往来】计提为了成本，也就是做出了不共戴天和势不两立的预期。但是，要感谢b站官方对UP的包容理解，以及b站这个社区对视频创作初心的一贯追求，愿意接纳我回来。”</span><span>在16日更新的动态中，@巫师财经如是写道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回看@巫师财经与B站分分合合之路，还要将时间回溯至2019年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>彼时，B站还在多方探索泛知识领域，财经类内容尚是一片未开垦之地，用@巫师财经的话来说，“发现除了ACG，它还能做更多，比如一片荒芜的财经内容，既然没有能打的，那么我来。”9月，@巫师财经发布了试水视频“B站资本市场成绩单”，一举拿下超200万播放量，就此成为之后财经与科普垂类“盛世”的敲门砖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14843358346307af62d0f989.40620257.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span></span><span>图源：@巫师财经 B站截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“激昂文字，以我为准，资本不眠。”与内容风口不期而遇后，@巫师财经乘风而起，创下三个月涨粉200多万的记录，并在2019年底成为B站认证的年度新人奖UP主，在知识区内占下一席之地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>既是B站原住民，又在平台快速起家，@巫师财经与B站的联结已十分深厚。但在2020年6月，@巫师财经突然发布声明，宣布自己退出B站，在西瓜视频重启。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毅然“出走”B站，<strong>更多原因在于当时B站的商业化窘境。</strong>在发布的一支说明视频中，@巫师财经称“越发感觉兼职做科普视频非常费力不讨好，用爱发电的动力也有所动摇”“考虑到资源广度、可持续性和观众反馈等各方面，未来视频还会继续做下去，但不再上传B站”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，在早期B站与西瓜视频的挖角大战中，@巫师财经不过是离开B站的达人之一。离开B站转投西瓜怀抱的UP主，还有赶海UP主@渔人阿峰、财经类@温义飞的脑洞财经、以讲述奇闻轶事而知名的@捕月说等。<strong>但@巫师财经与B站的分手，实在谈不上体面与光彩。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一则，在内容层面，如日中天的@巫师财经也遭到诸多“内容不够专业”的诟病，在2020年更是被指抄袭知乎，深陷抄袭事件。最终，巫师财经发布道歉声明。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二则，这场轰轰烈烈的“分手”多少让双方都有些尴尬，在当时的确有“老死不相往来”的态势。@巫师财经宣布退出B站当日，B站以公告回应巫师财经单方面违约。事后，不满双方“挤牙膏”式表达的巫师财经，再次声明自己向B站寄出的单方签字版本深度合作协议没有收到任何回应，因此并未生效，此后，他又公证撤回签字，彻底与B站“解绑”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从B站跳至西瓜视频，<strong>@巫师财经的本意是追求自身内容价值的最大化</strong>。相比于他的主动离开，B站另一头部UP主@LexBurner的被迫封号则显得像是一场闹剧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在动画区起家的@LexBurner，原本触及的是B站最核心的二次元人群。但在2021年2月的一场直播中，Lex在评价番剧《无职转生》时多次口嗨，发表了如“成功人士看这种动画片的心情就是体恤人民”等不当言论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/632963776307af63b246b6.32609539.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span></span><span>图源：LexBurner</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在登上微博热搜后，这些词条持续发酵，B站的“饭圈化”乱象遭人诟病，Lex也被扒出直播的口嗨黑历史，虽然他本人在凌晨道歉，解释并没有说过“我是人上人”这种地图炮言论，但最终积重难返。2月8日，B站官方发布了《对UP主LexBurner违规及违约行为的处罚公告》，对Lex的账号及直播间进行封禁处理，粉丝量超900万的动画区“顶流”一夕崩塌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时隔十个月，B站官方再次发布仲裁公告，裁定@LexBurner担负违约责任，赔偿相应金额2000万元，同时对其账号与直播间予以解封。Lex也表示之后“会痛定思痛，改过重来，尊重社区，也承担起自己作为UP主的社会责任”。当晚，Lex开启直播，一小时内舰长突破千位，总督超10人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是主动出走还是被迫封禁，@巫师财经与Lex的相似之处在于，他们都曾站上B站的流量巅峰。互联网有没有记忆尚可另谈。更值得关注的是，这些离开又回归的UP主，之后的路要怎么走？他们还能在B站重拾当年辉煌吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> <img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>回归之路，怎么走？</strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>转战西瓜视频后，@巫师财经有过一段流量巅峰。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年7月到次年3月，巫师财经视频播放量大多保持在700万以上，还曾出现过三支播放超千万的爆款。但从点赞、评论、收藏、转发等用户反馈层面，这些高播放视频似乎并未带来同等层级的反馈。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>举例来说，一支收获1370万次观看的视频，点赞量仅有2.8万，评论更是仅有1200，与B站相比落差极大。<strong>而到了2021年下半年，巫师财经的流量一路走低，甚至出现过播放量17万的爆冷作品。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16656599906307af640cb4e5.44403457.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span></span><span>图源：巫师财经 西瓜视频</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>流量走低、内容适配性不高，不利于之后的内容发展或许是@巫师财经回归B站的重要原因，加之与西瓜独家协议到期、达人追求全平台分发内容</span></strong><span>，最终，他还是做了一个与先前决定相悖的选择，而“老东家”B站也因渴求优质内容而予以接纳。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看似双方已达成和解，但@巫师财经之后在B站的路，也可能并没有想象中好走。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为回归的第一枪，@巫师财经的新作品长达4个小时，可以说诚意满满，“爷青回”“爷关更”刷满弹幕，但播放量尚未超过200万，与他先前的作品数据仍有差距，但作为“重新开始”之作，也说得过去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，在离开B站前，巫师财经以“财经区拓荒者”知名，<strong>但在他离开的两年内，B站达人迭代，新秀与元老并存</strong>，知识区内既有顶级大佬@罗翔说刑法，也有硬核的科技达人@老师好我叫何同学，历史向诙谐科普则有@小约翰可汗。而在财经领域，偏于娱乐化的财经UP主@硬核的半佛仙人已在B站积累下将近700万粉丝。<strong>@巫师财经的“不可替代性”降低，就难以复刻“拓荒者”的辉煌。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再者，就目前情况来看，@巫师财经所选择的是全平台分发的方式，意味着他并未将B站作为独家平台，这也会带来一个问题：平台的流量补贴还会丰盈吗？而基于目前抖音、快手、B站的不同社区氛围以及用户偏好，达人想要产出符合多平台调性的作品，难度可想而知。与此同时，另一个不容忽视的问题是，<strong>知识类内容变现的掣肘是否依然存在？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>出走又回归的创作者，B站并非没有前例。例如曾在游戏区内创下辉煌战绩的@敖厂长，出走一年后宣布回归B站。网传B站拿出4000万与@敖厂长签下5年独家合约，虽然B站随后否认了金额数目，合约金之丰厚也足以让人浮想联翩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但从目前来看，曾经被寄予厚望的@敖厂长如今的内容产出速度与播放数据都在下滑，在游戏区众多达人如@老番茄、@中国boy、@逍遥散人等压阵的情况下，<strong>曾是游戏区标杆的@敖厂长，也陷入了流量窘境</strong>，“囧的呼唤”似乎成了时代的眼泪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再回到@LexBurner这一边，被B站清算后，他背负的不止2000万赔偿金。“无职转生”事件之前，Lex曾与B站进行过深度“绑定”：2018年，B站在纽约敲钟上市，Lex是参与其中的UP主之一；2020年暑期，B站推出UP主综艺《欢天喜地好哥们》，Lex是“阴阳怪气”男团成员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但在账号被封禁再解封后，Lex的粉丝数由先前的超900万掉至618万，对于他的解封，评论区也呈现两极分化态势，一部分欢迎他的回归，一部分则将他的“成功人士”语录复制粘贴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在回归之后，“无职转生”阴影尚在。B站发布的仲裁公告中提到“事情由动画而起，最终也应回到动画”。<strong>但就目前情况来看，Lex在内容层面并没有继续他所擅长的动画领域或生活区，而是将重点转向影视吐槽，目前更新的几支视频均为吐槽时下热播影视</strong>，如《独行月球》《新封神：杨戬》等，播放量与他的巅峰时期相差甚远。</span><span>动画区尴尬，影视区“内卷”激烈，Lex能走多远？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总的来说，尽管B站并非“只闻新人笑，不闻旧人哭”，但平台生态始终千变万化。巫师财经离开时，B站的知识区尚处于初始繁荣期，Lex发迹之时，动画、二次元还是B站的特色与核心。发展到当下，知识区达人迭代迅速，而“去二次元化”则成为B站明显趋势。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果不能保证持续而优质的内容输出，即便是前“财经区一哥”“动画区顶流”，恐怕也难以复刻昔日辉煌。</span></p>
