<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前几天参加了<a href="https://www.niaogebiji.com/tag/%E6%B7%98%E5%AE%9D" target="_blank">淘宝</a>直播盛典，有一种强烈的感觉涌上心头，就是人们在过去几年可能过于纠结直播<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%95%86" target="_blank">电商</a>在交易上的技术价值，而逐渐偏离了电商对于消费这驾马车的全面价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次盛典现场，宣布淘宝直播2.0时代，就是要重新让直播的价值回到正轨上，以更高的格局去洞察消费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18683553106315e972357f11.60881906.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回顾去年底以来，从淘天合并之后一系列举措再到本次淘宝直播2.0，一条清晰的脉络也随之呈现出来，其实就是年初提出的那个大战略：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>从交易到消费。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何把这一切连接起来呢？淘宝直播将成为看懂这一切的核心线索。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>1、</strong></span><span>淘宝直播2.0，拉通“全域”场景</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>要读懂这场盛典背后的逻辑，首先要搞清楚从交易到消费这个大战略要指向什么变化？</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>交易的特点是线性的、静态的、单轮驱动，消费者体验层面来讲，交易的目的更明确，边界也很清晰。而消费的特点是动态的、多元的，边界更宽广，消费包括了交易，不止于交易。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>所以从交易到消费，背后指向的变化其实是体验的扩容与升级。</strong></span></span><span>消费者体验更有深度，更多元化，大家去淘宝天猫不再仅仅是完成交易这个动作，还有别的东西可看可玩可沉浸。商家体验则是生意更有持久性与确定性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何加深这样的变化呢？从交易单轮驱动升级为内容+交易的双轮驱动，内容里面蕴含着更丰富的体验和商机。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15204728116315e9727f9aa1.14619709.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年年初淘天合并之后，淘宝直播作为“内容场”的抓手被委以重任，如今回头去看，其实当时就确定了淘宝直播将作为一个重要的线索，去拉通全域场景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次盛典相当于明确了这一点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在盛典现场，阿里巴巴淘宝天猫产业发展及运营中心总裁吹雪表示，</span><span><span><strong>直播是淘宝天猫从交易走向消费的重要载体。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝直播2.0最大的变化正是以直播串联大淘系的全部消费场景。这次盛典也首次对外披露了淘系内容化的具体实施路径：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，直播本域与首猜、逛逛、搜索、有好货等手淘场域以及点淘实现“大公域贯通”。这一步的目的就是为流量扩容，让商家通过内容场去触达更广泛的消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，为流量分配机制做加法，淘宝直播2.0时代，内容好、转化高的直播间将获得更多的公域流量，改变了过去主要看成交指标的流量标准，变成了成交、内容双指标，相当于把消费路径从交易升级为了“从内容种草到成交爆发”的双轮驱动模式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三，发布新领航计划、引光者联盟、超级新咖计划、源力计划等四项主播政策，分别从不同的评判维度入手，进一步清晰主播矩阵，从而更好地帮助主播获得成长。淘宝直播2.0将支持2000个账号成交翻倍，5000个店播年度成交超1000万；培育100个粉丝超过百万的内容账号，在产业带孵化500个年销量过千万的标杆账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12522006526315e972dba218.07086654.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>结合这些内容化实施路径，再对应“从交易到消费”的内容化大战略，淘宝直播拉通全域场景的这个动作的意义就彻底清楚了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去直播作为一个孤立的交易变量，确实带来了细分场景与垂直行业的用户与生意增量。而更深入地服务好淘系10亿消费者，就是淘宝直播2.0的新起点。接下来淘宝直播将散发出更重的引力效应，它不再是一个孤立的工具，而是把淘系平台上所有的消费场景全覆盖、全串联。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>生意增量是一方面，更重的一方面其实是助推商家与消费者形成更加紧密融合的生态化连接，以此拉开全社会产业链的新价值空间。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>2、</strong></span><span>商家生意最可靠的阵地，仍是淘系</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年的生意不好做，谁都能体会到这一点，</span><span><span><strong>但本质上并不是交易活动停滞了，而是消费预期出现了变化。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家困惑的点其实是交易量仍在增长，但赚的钱却变少了，这是为什么？因为交易与消费是两个层级的价值，一个是此岸一个是彼岸。如果把目光单独锁定在交易上面，随着边际效应递减，利润率下降是必然结果。而消费则不同，消费的含义正在于持续地扩张交易的边界，更稳定地从内容场域中挖掘到新的增长点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>过去一年，淘宝直播的增长恰恰体现了这种“持久性”与“确定性”。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝直播的数据显示在过去一年里，年成交超百万的直播间达到25000个，成交过千万且增速超过100%的品类达到334个。淘宝直播正在从一条河变成一个水系，直播流联通各个场域，流量更充沛、流速与循环都起来了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3912055886315e973d0c977.07683061.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时还创下“三高”，为品牌引导沉淀了1.2亿新会员，人均购买超20次的高复购率，日均ARPU值超过30元的高流量价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>多元化的内容创新让内容场的商业价值更加凸显，在用户体验上持续升级，比如淘宝天猫服饰在今年新势力周开创性地推出了“全球云秀场”，与此同时，淘宝直播也在持续支持文化、娱乐等多元类型的内容生产。就像阿里巴巴淘宝直播事业群总经理程道放说的那样：“以前很多人对电商直播的印象是‘叫卖式’。我们已经明确淘宝内容化的新价值主张，就是‘专业有趣的人带你买’。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以说内容+交易的双轮驱动模式已经取得了效果，接下来这种新模式将在淘宝直播2.0时代继续向深处和高处发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于商家而言，淘宝直播2.0的大内容时代，持久与确定的增长将成为最大的吸引力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先在生意增量层面，淘系仍是最可靠的阵地，因为起步早、积累多、供应链优势明显。其次是从交易进入消费的海洋，要打开的不仅仅是技术手段上的丰富度，更是消费空间里面的新想象和持久度。一个商家通过店播要通往何处？一定是更持久的生意，那他就不能仅限于盯着这一场直播的数据，从流量到留量，是需要在经营上寻找新价值的，而淘系在经营体系上能提供更有深度的价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播域和店铺域的全面打通，为商家不断提升私域运营工具，完善平台与品牌会员体系，帮助商家将“流量”变为“留量”，实现持久确定的生意。比如优化搜索、推荐、直播、逛逛、天猫旗舰店的动线，提升品牌与消费者沟通的效率；强化短视频、直播等内容场，支持品牌、商家入驻逛逛、开店播，将公域与私域结合运营；开放天猫旗舰店的定制化，支持品牌在商品管理、视觉和消费者互动上进行个性化的创新；阿里妈妈的UniDesk实现通过一个平台采买淘内淘外全域流量，简化了操作，优化了成本，提高了效率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17012877336315e974127fa7.45894402.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很显然，淘宝直播对于品牌和商家的价值不仅是获新客、促转化，同时还在于稳定的复购和忠实用户的沉淀。过去一年，手淘消费达到1万元的消费者数量超过1.23亿且留存率高达98%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以无论是理论层面还是现实层面，淘系可谓商家生意最可靠的阵地。站在这个阵地的视角重新审视淘宝直播，就会发现它意味着完全不一样的价值。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>3、</strong></span><span>不是某某电商，淘宝直播本身就是电商</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今媒体很喜欢问一个问题，淘宝和某某电商究竟有什么不同？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实逻辑是这样的：</span><span><span><strong>淘宝奠定了中国电商的形态和文化，后来者为了差异化，才会给自己的电商前面加前缀，加标签。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而淘宝直播作为最早的直播电商形态，它不需要这种标签，因为它本身就是电商。我想这一点不该有什么争议。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先淘系是起步最早的电商平台，淘宝直播是最早的直播电商。其次淘系的商业模式决定了它最大的价值是赋能各方、生态成长，产业带就是最典型的例子，这种规模化的价值不是给电商加一个标签就能带动的，而是电商本身的赋能深度使然。最后淘宝直播也是最早从淘系平台上成长起来的，依托淘系生态，淘宝直播的创新与应用的深度与广度无出其右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>所以淘宝直播就是淘宝直播，它就是电商，而不是某某电商。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实无论叫什么电商，本质都是零售商业，就是要真正回归商业本质：</span><span><span><strong>好商品好服务满足消费需求，为商家提供完整的解决方案。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年1月淘宝天猫融合之后，提出的价值主张正是聚焦用户（消费）体验、客户（商家）价值，提出从交易到消费的战略构想，从追求效率的增长升级到追求质量的发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以淘宝直播2.0，在我看来最核心的价值，<span><strong>正是以一种简单的方式去化解复杂的局面。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这个简单的方式，不是在单一的直播工具上做加法，而是它背后要串联起来一个足够有深度有宽度的体系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个体系，既是消费这驾经济马车，也是淘系生态所代表的电商本身。说白了，这件事淘宝直播来做是最靠谱的，它的脚下足够深厚，它的天空足够高远。</span></p>
