<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如果你以为滴滴一年亏损500亿就已经是天花板，可能还是有点天真了。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前段时间，一年一度的中国财富500强名单正式公布，在这之中最让人意想不到的就是2021年度“亏损之王”，竟然是在短视频领域和抖音二分天下的<a href="https://www.niaogebiji.com/tag/%E5%BF%AB%E6%89%8B" target="_blank">快手</a>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，作为2021年度亏损最多的公司，快手全年亏损额高达780亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/154551039763178f53098976.19363071.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：新浪微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是一个什么概念？要知道所有A股上市公司当中，市值最高的贵州茅台全年净利润也才为524亿。换句话来说，快手用一年的时间就亏掉了茅台近一年半的利润。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/6251482662db2e883e36e5.24710351.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>怎么就成了“亏损之王”？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>万万没想到，快手竟然会亏损这么多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>单从数据上来看，780亿确实非常吓人，不过快手经调整后的净亏损为188.5亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以会有如此巨大的差异，其中最大的影响因素就在于股权激励。也就是说，快手的股价越高这部分进行股权激励的优先股价值也就越高，同样对应到公司上亏损也越多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过即便如此，188.5亿的经调整净亏损依然非常惊人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要知道，去年初快手曾顶着</span><strong><span>“短视频第一股”</span></strong><span>的光环在香港成功上市，巅峰时期股价达到了417港元，市值更是一度高达1.7万亿港元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70523542963178f53852e57.03430223.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：快手官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也让当时的快手一跃成为国内成功上市的互联网公司当中市值排名前5，仅次于腾讯、阿里、美团和拼多多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过在接下来一年多的时间当中，快手的股价一直跌跌不休，甚至遭遇了多家机构看空。2021年内跌幅更是达到了85%，即便是到现在局势也并没有得以扭转。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/159453306063178f53cb14b9.66513804.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：百度股市通</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>事实上，不只是在2021年，快手多年以来都处在持续亏损当中。</span></strong><span>数据来看，从2019-2021年间，快手的净亏损分别为10亿、78.6亿和188.5亿，同比呈现倍数级增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>甚至从今年初开始快手还传出了将会大幅度裁员的消息，实际上的裁员更是在去年底就已经开始，涉及到算法、游戏、国际化等多个部门，国际化业务部门几乎整个被裁。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅如此，随着获客成本越来越高，快手平台的用户留存率和增长也越来越低。靠以前烧钱来换取增长空间的方式，显然已经不再适合快手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得注意的是，就在8月底快手还发布了2022上半年财报，虽然亏损幅度略有收窄，但半年过去快手的亏损仍然超过了50亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6611396463178f54003d36.95573904.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：快手财报</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/176078547762db2e8f874706.61176433.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>问题出在哪儿？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>究其原因，显然是快手的商业模式始终都没有真正的跑通。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先说快手的核心支柱业务，线上营销和直播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截止到去年底，快手的线上营销也就是广告服务收入仍然占据到总体收入的“半壁江山”，然而其增速却从曾经的160%下滑到了今年第一季度的32%。尽管增长率还算不错，但相比较以往已经有了大幅度的下滑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且多年以来快手都处在“花钱买用户”的死循环当中，每年都在斥巨资发展，但这一业务的盈利预期却始终看不到，最终也未能在竞争激烈的短视频领域更进一步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>具体来看，快手每获得一个日活用户的成本已经从2020年的299元增长到了2021年的千元以上，涨幅高达240%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>事实上，早在2020年时快手的日活用户就成功突破3亿，然而两年时间过去，这一数据仍然在原地踏步。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最简单的例子是，哪怕周杰伦这一级别的天王巨星，在快手其实也只有3000万粉丝。对比而言，刘畊宏在抖音火了还不到半年，粉丝数就已经突破了7000万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一个主营业务便是直播。随着当下竞争的越发激烈和同质化的日益严重，快手的直播业务收入甚至已经开始进入了下滑通道，2021年不仅没有取得任何增长，还下跌了6.7%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主营业务很难出彩，反而是被归类到“其他服务”的电商业务增长较为明显。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从数据来看，2021年快手在其他服务板块的营收为74亿，这之中主要得益于电商业务的推动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过虽然快手在2021年的电商业务全年交易额为6800亿，同比增长高达78%，然而对比自己的“老对手”抖音，全年成交额早已突破万亿。相比而言，快手依然是落后的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内忧外患之下，快手显然需要寻找新的增长曲线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/79261823162db2e91cb3478.62060819.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>未来何去何从？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主营业务增长受阻，快手首先想到的引擎就是电商，尤其是在抖音开展的如火如荼的影响下，快手显然不甘示弱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而重点在于，发力电商其实对快手而言对手绝不仅仅只是抖音，因为多年以来市场当中更加成熟的电商巨头还有阿里、京东和拼多多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>为了寻求更有把握的第二增长曲线，在巨额亏损之下快手依然在不断的尝试。</span></strong><span>首先的目标就是要大搞信任电商，并且提出了要在2022年扶持500个以上快品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过信任电商的本质其实是熟人经济，想要把“人找货”往“人找人”上进行转变，对于直播带货屡次出现问题的快手平台而言，并不容易。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，快手还在向to B领域进军。例如早在2018年就曾收购了一家在线协作办公平台，只不过在当时由于主营业务接连走上下坡路，快手也并没有把更多的心思放在B端业务上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到了2020年快手收购了视频技术公司OnVideo，并且在一年之后孵化了快手云剪产品；去年下半年在小规模服务客户的基础之上，还组建了一个小型创业团队。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，在接连亏损的这几年，快手还在“偷偷”的制造芯片。甚至就连抖音还没有实际的产品推，出快手就已经把芯片给做出来了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在8月10日快手SteamLake品牌的发布会上，快手便官宣了云端智能视频处理芯片SoC已经正式进行线上内测，并宣称该款芯片在视频压缩速率上堪称“世界之最”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/159579499563178f54301165.63130709.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：快手官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过虽然快手在多个方面都取得了一定的进展和成绩，但想要成为真正的支柱业务，短期还不现实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>对快手来说，发力新的增长曲线依然要走长期主义，只是不知道接连的亏损压力下快手还能坚持多久。</span></strong></p>
