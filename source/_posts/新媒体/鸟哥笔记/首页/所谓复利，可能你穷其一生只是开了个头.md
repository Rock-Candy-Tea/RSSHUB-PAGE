<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E5%A4%8D%E5%88%A9%E6%95%88%E5%BA%94" target="_blank">复利效应</a>在逻辑上并没有什么问题，但对很多人来说，确实难以执行，倒不是因为找不找到的问题，而是里程碑何时到来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以我想和你聊聊这个问题，看清楚究竟什么是复利，为何对很多人来说，穷其一生也只能开个头，以及，如果你要选择感受与体会复利的力量，究竟要怎么做？</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每当说到复利，总会提起巴菲特，有两种说法：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说法一：巴菲特老爷子90%的财富都是在60岁之后获得的；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说法二：巴菲特老爷子99%的财富都是在52岁之后获得的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管是52岁还是60岁，和你说这个故事的人，都会煞有介事的告诉你，巴菲特21岁的时候只有1万美金的本金（虽然我找了半天数据图表，只找到21岁2万美金本金的）。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/104486723062dedcb6145569.61833527.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，如果巴菲特21岁时只有10美金呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好吧，那恐怕现在的巴菲特就没有846亿，而是8460万，也许只有846万，我不是想要抬杠，我只是想说，<strong>复利效应的第一要素其实是你的起点。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让我们算一笔账，你的财富每年可以保持6%的增长，当你40岁拥有1000万的时候，20岁的你应该有多少本金呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对不起，你需要311.8万作为本金。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你不想耗费20年时间，你只愿意投入10年，对不起，本金上涨到558.4万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以此类推，如果你愿意接受50岁拥有1000万，你20岁时的本金要求会降低到174.1万；60岁呢，97.2万；70岁？54.3万……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果人生的财富增加是如此稳定的线性，那么可以很简单的知道：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当你的「本金」不足时，要么压低你的欲望，要么忍受足够长时间的等待。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们甚至可以得到一个异常明确的结论：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>对大多数人来说，这辈子想要改变命运，就算你从你懂事时开始寻找复利并加以实践，以你生命的长度，恐怕并不足以让你等到那一天的到来。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，当你的职场起点是专员的时候，你寄希望于在短短30年的<a href="https://www.niaogebiji.com/tag/%E8%81%8C%E4%B8%9A%E7%94%9F%E6%B6%AF" target="_blank">职业生涯</a>中产生质的飞跃，那么，线性的成长肯定不是你要的，但是非线性究竟是什么呢？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是我们来到复利效应的第二要素：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>里程碑，或者我们称它为「转折点」。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我曾经在直播中分享过我的职业生涯：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>22岁，进入职场，从编辑做起，半年之后我觉得编辑工作的天花板特别明确，因此我要越过编辑的职责界限，去谈合作，落地项目，于是23岁我有了一个「对外合作项目经理」的title，并且真正谈下并落地了和一些大公司的项目；我还贪得无厌地做了一个业务，这个业务让我没有任何限制地去实践了一把虚拟积分，从0到1的那种，于是，25岁当我第一份合同到期，我就如愿入职了一家大公司。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在大公司里，我还是没有给自己设限，又恰好碰到了愿意给我空间让我放肆的领导，所以我不仅深度探索了积分业务的玩法，还让我去负责了上千万用户规模的运营实践。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>30岁我离开这家大公司，去了国企，本来雄心万丈想要做一个汽车领域的品牌俱乐部，管理上千万会员的那种，但是现实给了我重重一击，我被COO高高举起，却被CEO轻轻放下，所以在国企，我做着闲职，唯一拿得出手的就是那业务第一版的全品牌车型数据库和自有品牌车辆图库，是我一手打造的。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过塞翁失马，焉知非福，闲暇时间我开始连载《从零开始做运营》，顺便跳了个槽，去切实体会App的运营，而此时，命运的第一个里程碑向我露出了微笑。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一家垂直行业里并不头部的企业老板来邀请我加入他的公司，我推脱了两次，他三顾茅庐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正巧一位世界500强的HRD姐姐来到上海，还我一份人情，让我开出了我人生中第一个高管offer的要求：当前的</span><span>三倍薪资+VP头衔。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这位老板一口答应，于是31岁，我成了VP，管理3个部门，100多号员工，拿着超过50万的年薪。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时，那位HRD姐姐这样和我说：<strong>当机会落在你面前的时候，不要犹豫，不要退却，你就当是爬山，本来你每座山都最多爬到半山腰，现在有机会让你登顶，你怎么也要登，因为当你登了一次，就有下一次，山顶的风景会让你留连，也会拉高你的视野与你的身价。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对我来说，这是一种机缘巧合，但恰恰，又是某种必然，这个必然，在我偶然因为懒惰提笔去写《从零》的连载时就已经注定了后来的必然。<strong><br></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原因非常简单：《从零开始做运营》是国内互联网领域第一本简明扼要讲清楚运营到底是什么、做什么、怎么做的开山之作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，如果你有机会创造属于自己的「业内第一次」、「业内第一名」、「业内第一档」，你就一定会树立起自己的里程碑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>你觉得难吗？看下去就好。<br></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>复利效应的最后一个要素，其实是：<strong>稳定的回报</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>譬如，每次跳槽，我都将我的涨薪幅度定义到20%的下限，低于这个下限，不动，除非，这次跳槽可以让我完成一次里程碑，那么我可以接受平薪。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你或许会觉得奇怪，我从主管升经理，凭什么要平薪，什么情况下会平薪？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很简单，从大厂出来去小公司，就是最典型的场景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，你还有另一种选择，每次跳槽，我都不追求升职加薪，只追求永远大厂，也是可以的，只是，你需要记住，这两种选择，本质都是：<strong>稳定的回报</strong>。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我希望你记住：本金、里程碑、稳定的回报，对于想要践行复利效应的人来说，就是缺一不可的三要素。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很遗憾，「本金」不足，没有特别有效万事万灵的解决方案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>唯一能做的只有三件事儿，但这三件事儿，都各有利弊，请结合自身实际使用：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、结硬寨，打呆仗</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、压低预期；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、</span><span>功成不必在我。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先，是「结硬寨，打呆仗」。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾国藩统率领湘军，每到一处就修围墙挖壕沟，把进攻变成防守「凭濠对击，坚忍不出」，一点一点去蚕食对方控制的区域，湘军攻城不以月计，而以年计，方法看起来很笨，但执行效果极好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我过去常说，在一家公司里，尽可能多待一段时间，最少待满3年，这个建议并不是要你隐忍，而是3年时间可以帮助你更好地理解公司的业务和行业的逻辑，这就是结硬寨；同时，我还建议大家每天都要写工作日记，哪怕流水账，但一定要事无巨细记录下你今天做了什么，为什么做，想达到什么目标，最终效果如何，过程有没有阻碍，如何解决的，使用了哪些资源，还能怎么优化，也不是为了让你消磨时间，而是从日复一日的记录中，提炼、总结、归纳，这就是「打呆仗」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这两件事儿，看起来极傻极枯燥，我承认，但是它们却有非常真实的价值。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>少年时代，我跟老师学习长笛，老师要求每次练习前一定要抄谱，师兄弟们都想着偷懒，只有我坚持这么做，老师觉得我有定性，于是就愿意教我更多的技术。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年为了《从零开始做运营2》的签售会，我提前半年按照张展晖老师的指导，控制饮食，增加运动量，半年时间把体重从90公斤减到76公斤，展晖老师很讶异，觉得我很自律。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实我既不是定性，也不是自律，我只是笨，我觉得就是要下笨功夫才能做好一件事儿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二种方法，就是「压低预期」。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>假如你并不是一个能坚持的人，做不到结硬寨打呆仗，那么最好的方法，其实是压低你自己的预期，放弃无意义的对标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人类有个通病，做运营的同学也不能例外，凡事都喜欢找一个Benchmark，这本身没有问题，可是Benchmark要找谁却是一个问题。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我还记得媒体只要一发，xx毕业生被哪个大厂录用，给了一个超高的薪水，很多同学就会开始讨论，要么羡慕这个人好厉害；要么嫉妒他走了狗屎运；要么就恨自己家里没有矿或者晚出生十余载没有赶上好时候。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对不起，凡是能上媒体被作为典型来报道的，都是极稀有的品种。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你的Benchmark在你身边，甚至就在于你自己。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你想更快地成长，你的「平均年化收益率」应该从6%调整到10%，甚至15%，而不是盲目去对标那些你这辈子可能都不会和他产生任何交集的对象。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>压低自己的预期，不是要你放弃目标，而是可能你原先觉得自己一定要10年内财务自由，变成20年内财务自由，甚至退休前财务自由即可。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三种方法，「功成不必在我」。<br></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再说一个我的故事吧……我刚来到上海的第一年，工资1400，房租600，付三押一，数学好的肯定能算出来，那就是，凭我自己的薪水，很难支撑房租，所以我第一年的前半年的每个月的最后一周，都是啃馒头的，就这样，当时还需要家里接济。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和我同时期来上海的大学同学，基本就一个接一个回老家了。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我偏不。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我倒不是觉得我自己天赋异禀，而是我相信，如果我能坚持，能够足够努力，我至少可以扎个根下来，其他的让我的下一代去继续奋斗即可。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以多年前，我在知乎上回答过一个「寒门到底能不能出贵子」的问题，我说：</span><span></span></p>
<section>
<section>
<p data-pid="I4ochgUb"><span>一个人的出身，对于大多数人，其实是决定了一生的命运的，对个别人，可能只是一点点障碍，或者一点点帮助的。</span></p>
<p data-pid="DKrOTONU"><span>以上，确实是事实。</span></p>
<p data-pid="_fQBJcY5"><span>但是，的确有一些人通过自己的努力，改变了自己，改变了家族。那他们凭借的是什么，付出了多少代价，又获得了多少助力，赢得了多少有利条件，经历了多少风雨，并不是可以简单的复制的。</span></p>
<p data-pid="6REUpgd9"><span>更多的时候，是通过一代又一代人缓慢的进步，最终实现了质的飞跃，但是光芒只是归于一人，而不是其他先辈。</span></p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，如果你也有这种信念，我觉得「本金」不足，不会难倒你。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我所说的里程碑，并不一定是你要像我或者行业里的其他人一样，靠出版、做课去塑造出一个里程碑，当然如果你觉得这条路可以复制，我也不会阻拦你一定不让你去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是首先我们要明白里程碑是什么东西？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说白了，里程碑就是一个具有深远意义的大事件。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>譬如，拼多多一举在用户规模层面超越淘宝，这就是里程碑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那在拼多多超越淘宝的过程中，起到关键作用的那些骨干员工，就会一同拥有这个里程碑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我从两、三年前就一直鼓吹，传统行业的数字化与互联网化缺人，大有可为。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可是直到今天纯互联网环境已经非常恶劣的情况下，还是有同学不觉得这是机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说个真事儿吧。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2016年，我去了某智能汽车产业链的公司，负责车机OS的应用和用户运营，这家公司里几乎所有的骨干员工，现在都已经去了各个车企，且占据要职。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我的直系下属，2年后先是去了威马，然后去了福特，现在年薪恐怕至少50-70万，带着2个产品团队，他是大学肄业。除了他，福特里现在还有当时市场部的同学。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我的好搭档运维部门的老大祺哥，现在在哪吒汽车的智能研究院任执行副院长。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时的HRBP，现在在小鹏汽车，任人力资源部资深经理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果我当时没有去做音乐教育服务公司的运营VP，我现在估计和他们一样，也在不知道哪家车企里开开心心做业务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么会这样？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为我们当时做的是国内第一款自主研发并投入商业化运营的车机OS系统，我们在车内用户运营、车机系统运维等方面的经验是独一份，搭载了这套车机的荣威RX5当时就是销量惊人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这就是里程碑。<br></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是不需要你赌运气，只需要你愿意用时间成本和工作能力去换取的里程碑。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想象一下，你今天离开了纯互联网，加入了某个传统公司，你作为团队成员帮这家公司实现了互联网化的突破，让公司成功找到了第二增长曲线，你觉得你的电话会不会被猎头打爆呢？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可是你知道吗？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了这个结果，你可能会面临的是进入这家公司时老板的不信任，薪水上或长或短的损失，你愿意接受吗？你认为值得去挑战吗？你有信心能够一鸣惊人嘛？你有耐力能够孜孜不倦只求最终的里程碑的到来吗？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每个人对这些问题的答案永远不可能相同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是没有关系，还有很多其他的里程碑，只要你愿意，你用心，你有胆量，一定找得到。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>对此，我只有提醒你：千万要积极去拥抱与创造里程碑实现的可能。</span></strong><span>别无其他。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，让我们聊聊「稳定的回报」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对回报的预期，每个人都不一样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我个人践行的其实是职位的螺旋上升，我不看重企业的规模。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以我从一家不到100人的小公司，去了近万人的大企业，再去超过十万人的超大型企业，然后回归到几百人的中型公司，再去百来号人的创业公司，再去大企业的创业项目，再回到上百人的创业公司，我的轨迹，就是在追求职位的上升。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而我的好朋友韩叙老师呢，除了刚起步的阶段，他就几乎没有离开过大厂，百度、网易、快手……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你说我和他在最终的职业发展上有什么巨大的区别吗？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可能有，可能没有。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对外人，可能有；对我俩自己来说，可能没有。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，稳定的回报不在于他人给你提供的建议，而在于你对自己的<a href="https://www.niaogebiji.com/tag/%E8%81%8C%E4%B8%9A%E8%A7%84%E5%88%92" target="_blank">职业规划</a>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>请不要窄化「职业规划」这个词儿，它是你人生的规划。<br></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但你也不要一听到「人生」二字，就觉得是宏大叙事，其实根本不是，你需要有一个终极目标，因为终极目标是你接下来去拆解出阶段目标，和实现路径的核心依据，围绕这些路径，你可以去寻找和占有所需的资源。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你看，所谓的「稳定的回报」，正是运营同学最擅长的业务分析技能的跨界应用的结果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，不妨大胆一点，现在的时间可能是近十年来最差的时间，但也可能成为你一生中最好的时间的起点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不要去听任何人说复利是什么，你要如何抓住，而是认真看看你目前的本金，积极寻求里程碑的设立，在过程中不断追求稳定的回报，你会看到你想要的结果，在不远处对你微笑。</span></p>
