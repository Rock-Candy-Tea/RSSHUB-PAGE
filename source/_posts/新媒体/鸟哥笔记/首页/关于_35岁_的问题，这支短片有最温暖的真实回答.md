<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了18岁，35岁可能是被讨论得最多的年龄。而在当下的社会语境里，讨论35岁话题，往往容易陷入“中年危机”模式。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真实的35岁，究竟是怎样的一种状态呢？最近，</span><span><strong>汤臣倍健联合人民健康推出视频《你好 35岁》</strong></span><span>，将目光对准了35+不同人群，记录了他们关于“35岁”的真实回答，不仅展现了35岁的压力和挑战，也展现了35岁的勇气和担</span><span>当，引发了诸多共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短片表达了汤</span><span>臣倍健对35+人群的关爱，也为35岁话题注入了健康正能量，赢得了受众的好感。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>定向人群的品牌沟通如何做才能出彩？</span><span><strong>汤臣倍健用《你好 35岁》</strong></span><span>，给出了自己的回答。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>深刻洞察，真实记录，不制造焦虑，也不回避问题。短片以平实的视角记录了一群35＋普通人的真实状态。他们是各行各业、形形色色的人，</span><span>面临着来自家庭、生活、</span><span>工作等</span><span>方方面面的越来越多的“被需要”。视频展示的生活日常琐事，复杂又具体，让很多35岁的人感同身受。</span></span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频不仅让更多人看见35岁的困难，也看到了勇气。他们当中有人开始审视自己曾经的理想，有人开始创业，有人拥有了再婚的勇气......这些平凡人</span><span>经历着探索、尝试、沉淀、升华，在</span><span>更多人生角色和挑战中放下鲁莽和挥霍，</span><span>逐渐走向成熟，令人心生敬意。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《你好 35岁》用深刻的洞察和真实的记录，让35+人群“琐碎又平凡”的故事更具感染力。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫不意外，《你好 35岁》上线便触动了网友的情感开关，引发广泛关注。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/132574056563311b1a545023.69979206.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众多心理专家、社会学家、情感大V、职场达人等各领域kol的加入，让35岁话题讨论走向多元化，不再陷入“中年危机”模式。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19447610663311b1b18fc62.13294150.png" referrerpolicy="no-referrer"><img src="https://ngbjimg.xy599.com/144500008563311b1bc14ec6.81270237.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，汤臣倍健以《你好 35岁》为载体，从多角度多平台拓宽了社会对35+现象讨论的维度，让品牌价值和社会价值产生了共鸣，传递了健康正能量。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《你好 35岁》不是汤臣倍健第一次向不同人群传递品牌关爱。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果我们把时间线拉长，会发现汤臣倍健总能捕捉到消费者身体健康与内在精神的需求，在当下社会语境和时代背景中去理解消费者、理解大众，与大众共享和共创品牌的社会价值。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着国民生活水平的提高，消费趋势正在走向个性化、专业化，大众需要有品质的产品，需要有口碑的品牌。而作为中国膳食营养补充剂领导品牌和标杆企业，汤臣倍健一直致力满足不同人群的个性化营养需求。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>依托现代营养科学，汤臣倍健建立了</span><span><strong>全面的膳食营养补充剂科学体系</strong></span><span>，以极致的科学精神，打造极致的科学营养产品力，重点关注新原料、新功能、新产品，在多个细分领域创新研发，满足更多元的健康需求。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2020年与人民网·人民健康等机构举办“疫情常态化下的免疫力防线”专家科普直播，到2021年联手知乎上线「身体的答案」关注年轻人健康；从成为TEAM CHINA中国国家队运动食品及营养品供应商，再到今年联合新华社等多方设立“518中国营养节”......汤臣倍健一直以来用科学营养，为不同人群提供科学营养解决方案，为社会注入健康正能量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>发现了吗？汤臣倍健，可以很硬核，也可以很柔软。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>为健康人管理健康，是汤臣倍健的企业使命。</span></strong></span><span>不止希望你身心健康，更希望你有更健康的生活方式。谈科学营养说产品服务，汤臣倍健是严肃的靠谱的；和消费者交朋友，汤臣倍健又是柔软的温暖的。这种“软硬兼施”的品牌沟通方式，更真实更可感，也让汤臣倍健品牌形象更立体、更丰满。</span></p>
