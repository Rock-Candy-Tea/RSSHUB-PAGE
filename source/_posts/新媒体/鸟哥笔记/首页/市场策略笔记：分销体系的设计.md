                        <h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">01.前言：分销是什么？分销体系的价值源自何处？</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销体系涉及的利益方至少有两方：品牌方、分销方；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从传统的代理商模式，到现在社交网络的兴起，个人用户也可以作为分销渠道推广商品，分销本身早已融入我们生活中。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">随着社交网络的兴起，每个人都有属于自己的社交圈子。经济的压力，又促使大部分人群想通过副业赚一些快钱。介于精力和能力的因素，有一定的人群选择通过用自身的人际圈介绍好友购买商品赚取佣金。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而品牌方热衷于分销渠道开拓的原因：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1.降低成本</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">随着近年来互联网红利的见底，获客成本升不断高。记得前段时间才研究过的2020年下半年教育行业广告投放报告。随着社交电商的兴起，分销可以利用社交平台的粉丝（熟人）关系链，用非常低的成本完成获客。并且，本身熟人经济能降低购买用户的心理防线。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2.提升产品销量</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销模式的目的就是增加优质的销售渠道，如果用户愿意成为你的KOC，帮你不断触达更多的用户，销售更多的商品，提升销量和营收，何乐不为呢？</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">02.简述分销体系</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为一名互联网人，“分销模式”不过再熟悉不过的词语。建立有效的销售渠道，即为分销的核心。产品通过一定的渠道销售给销售者，即产品必须通过某一种分销方式才能到达。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们来看看分销渠道在整个销售链路所扮演的角色和作用：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">销售角色：渠道通过批发采购的方式，向产品方购买产品后，通过自有链路向消费者销售；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">推荐角色：随着社交电商/私域流量的兴起，拥有流量的渠道方推广产品至消费者购买，获取佣金；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">无论哪种角色，简单来说：分销=花钱找人帮你卖东西，</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种模式也在当前的电商环境中被广泛应用，例如淘宝、京东、拼多多等大大小小的平台均设置有分销机制。相比于传统的线下分销形式的[批发零售]、[多级分销]，线上新分销的模式对于渠道的价值要求并没有那么高，能做到全民协助：推广代收/渠道佣金。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">线上渠道分销大致分为以下两种方式：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">①推广代收流程：淘宝、京东等部分授权店铺</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">买家向分销商创建分销买家单</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销商收到分销买家单后，向供货商创建推广采购单</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">订单确认后，由供货商直接发货给买家。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">买家收货后，向分销商支付分销买家单，分销商此时行使代收权利。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销商收到货款后，向供货商支付推广采购单金额</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">供货商收到款后，向分销商支付佣金。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">②渠道佣金流程：淘宝客、拼多多分销</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1.分销方获得供货商提供的专属产品链接；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2.由分销方向买家推广产品链接，买家通过链接创建买家单；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3.订单确认后，由供货商直接发货给买家；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4.买家收货后，即供货商收款后，自动向分销方账户支付佣金。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上述两个结算方式，我们可以看出分销渠道形式的不同，所以如果纯粹以统一的制度、结构的体系来管理分销渠道，只能在一定的情形下起到片面的作用。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">随着业务阶段的变化及商业环境的变化，分销体系也需要进行对应的调整，以便纠正渠道行为，满足战略需要。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销本身是将产品通过渠道进行销售，渠道销售的每一个环节都代表着自身品牌和市场形象。所以，对应分销体系的设计，应仔细思考每个环节，思考其背后的期望达成的深层诉求。无论如何调整分销体系，其最终目的都是满足企业战略及发展变化的诉求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">下一章，咱们来聊聊分销体系的具体设计思路——本文以②渠道佣金的分销体系为主。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">03.分销体系的设计思路</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从利益角度出发</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如何让渠道或分销员更有动力地为企业推广产品？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销模式的核心便是资源的配比，所以通过阶梯奖励的设置，引导用户（渠道）行为成为了关键。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里细说有些枯燥，我们以21年4月底拼多多的裂变红包+转盘抽奖活动为例，展开说说：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（橘在这部分的讲解里，不方便将分销体系的全案拿出分享，所以将裂变活动作为案例来描述用户传播心理，与读者分享。分销的目的并非单纯以销售商品为目的，裂变活动的诱导与用户渠道分销的引导类似）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">该活动的形式是以裂变红包+转盘抽奖的组合型裂变活动，套路还是老套路，用户通过一系列操作达成提现额度后，即可提取现金至微信号。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这个活动有意思的地方是①通过活动后期-转盘阶段的中奖概率闭环（以免用户未达到预期行为即可获奖），与以往拼多多的拉新裂变活动不同，并没有明确要求你邀请新用户注册。②双标准换算红包金，即可直接抽中现金或抽中金币兑换（10金币=0.01元）；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（1）第一阶段：启动</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">点击分享者参与链接，助力并获得高额的现金。现金可直接提现至微信，但需到达标准，如图例，获得现金497元，提现需500元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一阶段：诱导参与活动（即启动）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62b5a1cce5e11c193644.jpeg" title="市场策略笔记：分销体系的设计" alt="市场策略笔记：分销体系的设计" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（2）第二阶段：诱导深入</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">参与用户如何再获得3元，达成500元的提现标准呢？先别急，这里心理诱导还未结束。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用户需要先完成以下操作，进一步获取红包金；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">① 点击转盘，抽取红包金；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">② 分时免费领取红包金；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">③ 转发好友协助，抽取红包金；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二阶段：利益诱导用户操作，降低用户心理防线，给用户一种很快就能成功的感觉，使得参与用户更愿意完成下一步任务。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62b5a1cd18fe2c193645.jpeg" title="市场策略笔记：分销体系的设计" alt="市场策略笔记：分销体系的设计" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（3）第三阶段：核心任务阶段</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在完成1、2阶段的任务后，参与用户能获得499元以上的红包金。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">拼多多可不是慈善机构，这时候它就需要这些深度参与用户协助完成它的目的了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">该活动的目标有两个：拉新、促活；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">邀请拉新用户助力，可获得6次转盘机会；普通用户（即老用户）助力，仅获得1次转盘机会；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本阶段，作为深度参与用户的你在轻松邀请20-30名以上用户助力后，便能到达499.9元的红包金。没错，你仅剩1分钱就可以提现500元了！如下图：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第三阶段：引导用户完成核心任务，红包金离成功更近一步，促使用户完成活动目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62b5a1cd38f72c193645.jpeg" title="市场策略笔记：分销体系的设计" alt="市场策略笔记：分销体系的设计" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（4）第四阶段：难度提高（活动闭环）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这个阶段，你只剩下1分钱的奖励金即可提现500元了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">游戏难度剧增（小样儿，怎么可能让你轻易拿钱？）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此刻，打开本篇开头盖好的暗牌：抽奖与金币。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这个回合，你必须完成邀请助力的行为（没有其他选择）后，才可以获得抽奖次数。抽奖的奖品删除了现金奖励栏（隐藏），意味着限制了你必须抽中金币，兑换现金才可以完成剩余红包金。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不仅如此，抽奖概率也随之增加，奖盘里的【祝福项】变多。意味着你即便顺利邀请了好友助力，也不一定抽中金币。另外，1分钱虽然是红包金的最小单位，但金币不是，所以即便你抽中了金币，它的最小单位也可能是不是一个，而是0.1或0.01，意味着你要完成最后的金币兑换，需要可能要排除万难，拉到数百个好友帮忙助力才能成功。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样的游戏机制大大提高了参与用户获取奖励的难度，并且实现了双维度（金币与现金环环相扣）的游戏闭环。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第四阶段：游戏难度暴增，用户完成难度增加，达成活动闭环。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62b5a1cd62cc4c193645.jpeg" title="市场策略笔记：分销体系的设计" alt="市场策略笔记：分销体系的设计" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（5）活动分析总结</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从上述的整体活动流程中，橘相信读者已经看出，1~3阶段的循序渐进，促使用户深度参与活动，并完成其活动目的（拉新、促活）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第1阶段，通过临近提现的红包金差额调动初始参与用户的贪便宜的心理，使得用户参与活动；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第2阶段，通过简单的任务，让用户更接近最终红包金，活动难度简单；该阶段，活动难度简单；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第3阶段，正式开始对进入这个阶段的高度参与用户发布相关的核心任务，用户将邀请好友/新用户协助助力，达成活动裂变目的；该阶段，活动难度中等；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第4阶段，活动难度大幅提升，将有相当比例的参与用户丧失完成活动的信心，并且放弃活动。橘针对该活动做了些分析，仅供大家参考（毕竟拿不到拼多多的内部算法）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">①活动前期，该裂变活动的奖品符合参与者的心理需求，调动了参与者的参与兴致；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">②活动中期，通过简单且有效的用户任务使得红包金接近兑换阈值，让参与者深度参与到裂变活动当中，使其认为自己即将获得活动奖励；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">③活动后期，活动难度暴增，预估后期抽中【祝福项】的概率提升，抽中【金币】的概率降低；预估0.09枚金币预计需要拉新30-50人之间，或裂变100-200人才能到达500元的提现阈值，即至少尚需参与抽奖180-300次之间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">活动奖品成本为500元，如1名用户完成活动目标，则拉新成本为10-20元/人或促活成本为1.7-2.8元。考虑到实际成功率，这个成本只能是更进一步压缩，最终通过活动闭环，控制整体活动成本。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（6）从裂变再看分销</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">无论是传统招商，还是线上分销，亦或者是裂变活动，促发目标协助达成目的需要从对方的利益角度出发。虽然上述的案例是拼多多的裂变活动，但类型是一通百通的。分销便是与人/渠道打交道，通过利益让其加入其中，然后通过登门槛的心理学原理，利用用户博弈心理，使其不断升级（深度参与）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，让我们回归主题，分销体系的设计必然要从合作双方的利益角度入手！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">正如上述案例的阶梯奖励，前期奖品容易获得，满足用户心理。参与用户尝到甜头后，才能让用户付诸更多的行动。当用户到达一定数量，则需要提高门槛。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">阶段化的奖励也有助于把控前期的成本投入及逐步引导后续用户追加投入。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">注意法律法规</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当前法律规定分销不能超过三级，因此分销体系的设计最多三级。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一般判断是否违法的判定即为：“是否超过三级分销”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在三级分销内，需要有明确的商品（线上/线下），不能空拉人头不做事。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">04.分销体系设计实例</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">等级与等级权益的设计</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（1）入驻门槛</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">是否在分销体系中，设置入驻门槛？只能说各有好坏了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果不设置入驻门槛，好处即为有渠道的用户可以通过各种途径广泛招募下级分销，新用户有广泛的来源；缺点也很明显，没有入驻门槛，将导致种子用户难以筛选，且可能产生渠道管理混乱的问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以安利为例，最低代理费用为70元，缴纳后赠送对应70元的商品券及运营支撑（如线上开店、直播教学、免费培训等）。这里不是推广安利，因为所有的运营支撑是由上级分销协助，并非安利总部支撑，而你日后的销售额利润也将分给上级一部分。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用户最初支付的成本，会让新用户有动力将付出的金钱赚回，也能在一定程度上增强其卖货的动力。（当然请小心分销与传销的法律边界）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（2）等级权益</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">正如上一章所言，分销需要考虑合作双方的利益。所以，等级权益需结合自身业务范围及分销渠道/用户的需求综合考量，给每个等级对应的激励。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">前面已经说过权益的设计可以按阶梯奖励的方式设计，来讲讲分销渠道吧。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同的渠道，对应不同的场景及需求。举个稍微极端的例子：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">①传统大型贸易商如果参与到分销中，会极速成为高等级的代理渠道。因为本身就拥有广泛的销售通路，但其也往往不愿意主动将渠道贡献到体系中。如果此时，你拿着C端的分销合作分成与之谈判，结果将不言而喻。要达成合作，往往需要提供远高于C端分销分成比例的返点利益；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">②C端KOC、小B端渠道：如微商、宝妈、代购、自媒体、淘系、平台店铺，每个渠道都有其独特的特质属性。在设计等级权益时，则需要根据渠道的属性和市场情况来设计等级权益。（研究竞品，此刻真香）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（3）闭环条款</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">①晋级机制</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销体系的晋升机制与会员体系一样，需设置合理的机制，循序渐进地加强分销商在分销体系的作用。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">核心即为增加分销渠道的沉没成本，加强分销员的自我驱动力。当其为产品投入更多的成本，那么他对于产品的粘性和信任度也会成正比提高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而晋升机制，正是为了这些愿意付出更多努力的分销员/渠道，提供更多的分销佣金或奖励，所以晋级机制本质就是一种激励机制。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于分销商来说，最主要的需求就是通过自身的人脉及关系，吸引更多的团队下线，从而通过下线的转介绍/消费来获取对应的佣金。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，设计晋级机制时，可以从以下几种维度考虑：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">    1)到达一定销售金额；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">    2)到达一定团队人数/渠道体量；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">    3)到达一定团队人数且判断下级分销渠道数量；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">    4)组合搭配：到达一定销售金额及一定团队人数；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">晋级后的等级权益，一般设置高等级高佣金比例，外加额外的增值业务权益，如减免运费/邮费、优先提货、区域专营等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">②奖励机制</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">正所谓“无利不起早”，相信没有一位分销商是心甘情愿为平台免费做推广的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">促使分销商持续卖货、拉新的动力，便是物质。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">奖励机制的核心即为通过奖励提升分销渠道的信任度和融入度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了①所提的等级奖励外，还包含了额外的物质奖励和精神奖励。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">    1)物质奖励</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当优秀的分销商在某个时间段内，完成目标交易额/拉新量，可以设置类似月度/季度/年度奖励，来刺激其持续为品牌方付出精力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销的核心在于稳定和可持续。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">    2)精神奖励</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据马斯洛需求理论，在物质条件满足的情况下，精神奖励高于物质奖励。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">正如物质奖励一般，精神满足可作为物质奖励的额外补贴，满足分销商在声誉及名誉上的精神需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如：小红书TOP2的代理商——深圳三七无限网络科技有限公司在2020年获得了小红书效果广告突出贡献奖（虽然橘觉得这个奖牌的设计尚待提高）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62b5a1cd7c2cbc193645.jpeg" title="市场策略笔记：分销体系的设计" alt="市场策略笔记：分销体系的设计" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">③降级规定</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">降级指的是分销渠道在时间周期内，完成不了主体所设定的级别要求或在分销过程中发生严重错误，影响品牌价值时，则可能被品牌方降低分校等级或解除分销合作关系。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">核心即为对分销渠道的行为进行约束，以目标为导向，提升其销售能力、拉新能力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，设计降级规定时，可以考虑以下几个维度：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">    1)一定时间内，未达成最低目标销售额/订单量；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">    2)渠道支撑能力不满足级别要求；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">④保级规定</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">保级规定即为不降级，指的是无论分销渠道的后续活跃度如何，也不降低渠道的分销等级。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样的规定能降低分销渠道的心理压力，但将加大管理难度，后续无法完全保证各级别分销商的质量，则后期将付出更高的激活成本。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">05.后记</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销不仅仅运用于电商行业，线上教育、医疗保健乃至外卖平台等行业都运用广泛。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销体系并没有那么令人生畏，其原因也正是由于商业历史的发展，时时刻刻都离不开分销，只不过有时换了个名称罢了。充分理解分销的本质和核心，了解涉及利益各方所蕴含的心理学，才能够灵活玩转分销体系。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分销体系的设计并非一成不变，政策压力、市场变化、渠道变化、用户选择、产品质量及品牌内涵等因素，或多或少皆需改变分销体系的分模块设计。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">市场人往往要根据情况的不同，调整自身的分销体系，同时也要防止“牵一发而动全身”的公关问题发生，这就需要在设计时谨慎考虑。</p>
