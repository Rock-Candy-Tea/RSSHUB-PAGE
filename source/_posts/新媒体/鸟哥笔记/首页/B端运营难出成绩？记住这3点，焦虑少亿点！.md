                        <p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">“这活是没法干了！”</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">自微信服务号从新消息提醒变“小红点”后，身边的<a href="https://www.niaogebiji.com/tag/B%E7%AB%AF%E8%BF%90%E8%90%A5" target="_blank">B端运营</a>，尤其是干新媒体的小伙伴们哀嚎不断。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">过去服务号的强提醒能提高不少文章打开率，冲冲流量，如今一改版，阅读量猛然下滑，对流量获取本就难的B端运营来说无疑槽糕透了。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">不过B端运营也别气馁，能玩的还有很多，放开去尝试，总能找到强于服务号且适合你们行业的流量渠道。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">接下来，简单分享近段时间来的一些感悟，大多是个人观点，希望能给B端运营们提供一些工作帮助和思考方向，欢迎大伙在留言区参与讨论，共同成长！</p>
<h2 id="toc-1" style="font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); padding-left: 12px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: left; font-weight: 600; background-size: 8px 20px;">一、面对客户：不妨多一点点诚意</h2>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">本来这块内容计划放在最后一段，当成闲扯，却回忆起几件印象深刻的事，不由摘到前面，展开聊聊。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">先分享一段经历：</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">背景是传统建筑行业，我们是SaaS系统服务商，某天被安排和销售去拜访一个客户，本以为只是简单介绍产品走个流程，没想到却被上了一课。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">不知是否巧合，竟撞上了竞品团队也来推介产品。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">正所谓同行相见，分外眼红，两家销售为抢到单都拼了，讲解产品时一个比一个卖力。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">说实话，两家产品差不多，功能相似，不存在较大产品上的优势；两家的销售也足够专业，不存在现场讲解上的差距，但在这场战斗中，对方却赢了。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这让我们的销售很不服气，也令团队其他人很不解，究竟败在那？</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">后来才探知到，原来竞品拜访客户，无论客户大小，无论结果成不成，都会带上一个小礼品，不贵重，可能就是茶叶或笔记本，在拜访期间会赠送给客户。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">输得不怨！这是第一想法。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">不是说输在客户收了礼，也不是说输在同行有心机，而是输在面对客户时缺少了一点诚意。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">要知道我们公司其实也有礼品，但多数是留着活动用或维护大客户关系准备的，一般去普通客户都抱着我们是互联网企业，不搞这些虚的，要用产品和服务征服这类客户。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这个想法本身并没错，在互联网行业，优秀的产品或服务确实能第一时间打动客户，但来到传统行业后，当客户面前有多个选择后，产品和服务好，有时候也不一定能赢。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">市面上产品那么多，功能相似，凭什么选你的？</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这种时候，往往对客户多一点点诚意，便足以改变结局。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">回过头一想，当时客户肯定知道我们两家是同行，却依旧约在同一时间来介绍产品，他们<span style="font-weight: bold;">想看到的或许便是产品之外多出来的那么一点点</span><span style="font-weight: bold;">“</span><span style="font-weight: bold;">人情世故”</span><span style="font-weight: bold;">。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">要知道干B端不是一锤子买卖，企业服务的本质可以说是续费，售前的诚意，将在合作后被再次放大，对续费也是有利的。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">还有一点非常重要！身为B端运营，一定要熟悉你所服务的行业，千万不能抱着你是互联网人才而看不起传统行业，隔行如隔山，不同行业都有完全不同的标准和规矩。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">有诚意加懂行业，是征服B</span><span style="font-weight: bold;">端客户，尤其是传统行业的长胜秘诀之一。</span></p>
<h2 id="toc-2" style="font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); padding-left: 12px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: left; font-weight: 600; background-size: 8px 20px;">二、获取线索：多线验证，多点耐心</h2>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">怎么获取更多线索？这个问题相信困扰着很多背负线索指标的B端运营。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">常规获取线索的方式有线上推广+线下活动，这里主要想和大伙聊的便是线上推广的那些问题。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">第一个问题：选免费的还是付费的？</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">免费渠道如官网、各自媒体平台等，付费的如SEM、软文投放等，这块暂不展开，网上很多汇总文章，感兴趣的伙伴可以去看看。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">至于说选哪个，具体得看你有多少预算以及指标任务重不重。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">简单来说：<span style="font-weight: bold;">免费的见效慢、付费的见效快，有预算想早出成绩就选付费的，没预算或指标压力不大就选免费的慢慢做。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">吐槽：企业多给运营一些时间，让子弹飞一会，别动不动想着零成本公众号涨10W粉，带来几千条线索，这除了会逼疯运营外，基本没啥可能。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">至于说公司一毛不拔但指标压力大的，建议你重新找个有预算的公司。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">第二个问题：如何正确的争取资源。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">领导不怕你来要资源，最怕你拿张预算申请单就问能不能批个几十万来付费推广。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">你好歹给个面子，出个具体的推广方案来画画饼，不然除非公司太多钱花不完，不然只会让你出门右转带上门。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">当然，也不乏有些高手凭借强大的口才说服领导批了预算，然而在毫无策略的推广下，几十万轻轻松松花完，带来的效果却十分难看，同行一条线索成本1500，你直接搞到2500一条，从ROI来说是极失败的。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">一个可落地且有说服力的推广策略就极为重要了。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这里推荐一个常见的做法：分渠道小成本测试。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">例如你能申请到一笔10W的预算，计划线上推广一款SAAS产品，你会怎么办？是全部搜哈到SEM上，还是抖音+公众号投放两路开花？</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">如果你真的盲目去推广，10W的预算弄到线索不难，但线索数量和线索成本则不受控制。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">不受控制，对于一个优秀的B</span><span style="font-weight: bold;">端运营来说，是无法接受的。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这时就可以尝试分渠道小成本测试的策略，方法很简单，却十分考验耐心：</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">一是列出你所有想去投放的线上付费渠道。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">百度、头条、抖音都可以，反正你要把你所在行业能获线索的线上渠道全列出来。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">二是制定一个测试期，拿出一部分预算来测试所选渠道的推广效果。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这里注意在测试前要了解各渠道的基本情况，比如SEM是行业内公认效果最好的，竞争最大，一个关键词出价几十块，你测试期就不能定个几百预算，不然丢进去一点水花看不到，失去测试的意义了。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">最好根据渠道性质有一定预算分配比例，做到雨露均沾，至少保证测试期内各渠道都能有一定量数据。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">三是根据测试期结果，重新分配剩余的预算。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">获客效果好的砸钱冲，效果差的果断放弃，不要浪费太多精力在无用的渠道上。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">如此做下来，基本就能说清这笔10W的预算怎么花，推哪些渠道，为什么这么定，测试期的大量真实数据便是你说服领导给你批预算的有力依据。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">第三个问题：做了付费的，免费的还做不做？</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">不做是不可能的！大部分企业都是能白嫖就白嫖。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">B端运营别看不起免费渠道，免费的虽然出效果慢，但胜在持久，属于放长线钓大鱼。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">做免费的一定要考虑价值点在哪，能给潜在客户带去什么。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">例如B端常见的行业白皮书，有些就是企业内部搞的，不花啥钱，但也能凭此获取不少线索。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">从业者们需要在信息不流通、不对称的环境下了解一些行业的现状和趋势，这便是行业白皮书给传统行业带来的价值之一。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">免费的往往是最贵的，要考虑的更多，付出的精力也更多，但效果不会差的。</p>
<h2 id="toc-3" style="font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); padding-left: 12px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: left; font-weight: 600; background-size: 8px 20px;">三、数据，数据，数据，看到数据多思考</h2>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">重要的事情说三遍，优秀的B端运营总是<span style="font-weight: bold;">勤于收集数据、善于整理数据、乐于分析数据。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">数据的重要性就不赘述，主要还是聊一下数据相关的几个坑，希望B端运营们看到数据后要多思考，别太早给结论，少踩坑。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">一是数据来源不清。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">来源不清就容易导致数据不准确，影响最终分析结果。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">比如你在某渠道上投放测试，想根据落地页的填表数据来确定渠道等级，测试期结束后，发现填表量数据不错，便将该渠道定为A级，加大预算，但一段时间后，却发现填表数量和质量远远不如测试期。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">一经调查才发现，原来是公司的销售们将该渠道落地页转发给很多潜在客户，才导致测试期的填表量暴涨。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这就是很常见的数据来源不清从而影响分析结果，想避免踩坑，就要在测试时保证数据来源的唯一性，最大程度保证数据的准确性。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">二是数据收集不足。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">数据样本容量不够，会导致数据的不准确，分析结果也缺乏说服力，做了无用功。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">以SEM为例，一个关键词出价20，你定个100预算，可能投出来一小时就烧完了，线索数肯定不会高到哪里去，而你就将这个渠道或关键词定为无效，这肯定是不行的。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">你至少要保证你的预算能支撑这个关键词能持续投一段时间，才能收集到一定量有分析价值的数据，零星且混乱的数据是对运营工作有干扰的。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">三是数据对比错误。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">数据需要对比才能产生价值，但不正确的对比反而会影响数据分析的结果。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">举个很简单的例子，你进行百度和头条的投放测试，百度你投了白天，花费5000元得到50条线索，头条你投了晚上，花费2000元得到10条线索，然后你根据ROI判断百度比头条更有效。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">乍一看好像也没啥问题，毕竟百度单条线索成本100元，头条200元，对比起来确实百度效果更好。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">但这样的对比是无效的，因为变量太多，两个渠道投放时间不同，投放花费也不同，产生的数据对比起来肯定也是无意义的。</p>
<h2 id="toc-4" style="font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); padding-left: 12px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: left; font-weight: 600; background-size: 8px 20px;">四、结语</h2>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">B端运营想做好确实要多一点点真诚、多一点点耐心、多一点点思考……。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">文中很多内容都没有展开，比如数据如何分析、SEM如何才能做的更好等，这些都是单一篇文章很难讲清楚的，还是希望B端运营的小伙伴们能够主动去学习，多实践，印象才会更深。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">最后，还是老话：愿我们始终保持思考，一尘绝骑。</p>                        <div style="height: 1px"></div>
