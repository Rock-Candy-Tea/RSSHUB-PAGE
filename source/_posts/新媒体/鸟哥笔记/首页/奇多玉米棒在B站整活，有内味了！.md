                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这个漫天是“梗”，不断爆“热点”的传播环境中，年轻人总是在追逐着好玩、新鲜的内容，品牌们也一直不倦地在其身后追了又追。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但品牌传播的目标在于，<strong>要去做用户文化的引领者而非追随者，</strong>不能总是跟在屁股后面炒冷饭。而要解决这个问题，就要探究年轻人所热爱的内容背后，是否有着一些可复用的规律。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对这个规律的探索，不妨直接从2022年B站那些走红的内容中去寻找。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">前阵子“二舅”的故事从B站火至全网，无论故事中是否有虚构部分，但都不影响我们发现，戳中年轻人的“情绪”，就容易释放巨大的传播能量。还有大妈的那一句：“退！退！退！”，在逗比的Battle中演绎了超魔性洗脑的动作，再次引发大量魔改；还有今年街头频频出现的小学生暗号——“孤勇者”，又掀起了全网的跟风模仿。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" title="1660535183903586.png" src="https://file.adquan.com/uploads_img/1660535183ihwc7.png" alt="图片1.png" width="600" referrerpolicy="no-referrer"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些爆火内容，其实有三个关键词给到了品牌，分别是：“情绪”、“魔性”、“跟风”，并分别对应着：关注评论、主动分享和二次创作，它们正是品牌和年轻人沟通时应当参考的规律。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但，年轻人传播光好玩就够了吗？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上面提到的规律对品牌来说，更多地只是一个方向，并没有真正解答如何“落地”。因为大部分尝试过年轻化沟通的品牌会发现，这中间存在着很多典型的难点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，创意的魔性，但品牌做的魔性内容，不一定能激发TA们的兴趣；再比如，在大量的洗脑、二次元、鬼畜等素材中，品牌玩得忘乎所以却又担心忘记自己是谁、丢掉自己的品牌调性；还有特别常见的，创意只顾好玩却没回归到产品上来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些重重复杂的问题，在赞意看来其实就是一个总的思路：<strong><a href="https://www.niaogebiji.com/tag/%E7%94%9F%E6%84%8F%E5%9E%8B%E5%88%9B%E6%84%8F" target="_blank">生意型创意</a>——既要有传播性，还要实现商业价值。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">举个超好玩又典型的案例。赞意和奇多玉米棒最近在B站整活，让用户疯狂“抖手”的同时，还解决了生意上的目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">先来看看这个超好玩的核心创意——<strong>《浓浓抖手舞》。</strong></p>
<p style="text-align: inherit; background-image: none; font-size: 16px; color: #000000; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><iframe src="https://file.adquan.com/uploads_img/1660535311hewfi.mp4" width="680" height="400" allowfullscreen="allowfullscreen"><span id="mce_marker" data-mce-type="bookmark">​</span></iframe></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再来欣赏下弹幕，“官方整活最为致命”、“我的耳朵”、“谁啊！想出的！啊啊啊啊我疯了！！！”，在无比魔性的风格中，刷屏的不只是弹幕，显然还有产品本身。这支品牌视频成功冲上全站TOP100，并且拿到舞蹈区TOP1与近15万次点赞的成绩。此外，案例还发起了大量的内容运营，如牵手优质UP主共创、发起召集令等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" title="1660535380768223.png" src="https://file.adquan.com/uploads_img/1660535380pyf37.png" alt="图片2.png" width="600" referrerpolicy="no-referrer"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">接下来具体来看，整个案例是如何一步步完成——<strong>生意型创意。</strong><strong> </strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">01、锁定品牌的生意目标</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">先来看品牌，奇多玉米棒作为一款针对05后的零食品牌，这次传播刚好是针对暑假场景而来，并有着几个明确的商业目标：<strong>1）有效覆盖目标用户；2）在心智上实现产品种草；3）提升TA对奇多玉米棒的认知度。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在生意型的创意理念下，这三个商业目标就可以一一拆解，找到应对之策。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>生意洞察：生意型创意理念的出发点，是锁定品牌的信息锚点，</strong>是指要从品牌与产品出发，确定本次传播的核心信息是什么，使创意回归到生意上来。奇多浓郁玉米棒无论是用户口碑，还是产品自身主打的卖点都是：味道浓郁，粉料十足。于是创意就抓住了这个产品特质，为品牌提炼出“超有料,超浓郁”这个信息锚点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>媒介洞察：生意型创意的第二步，在于优先确认媒介的选择。</strong>品牌的TA是05后，他们活跃在哪个平台？这个平台又有着怎样的内容风格？因为在碎片媒体时代，不同媒介平台的内容玩法与风格完全不同，如果媒介没确认，内容很难对味。最终锁定了TA广泛活跃且浓度很高的B站。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>创意洞察：生意型创意的最后一步，就是让产品始终成为英雄。</strong>这其中涉及两个关键问题：产品的特质是什么？受TA喜欢的B站创意是什么？通过前两步的分析，赞意在创意中把洗脑魔性、易于模仿的内容与产品的“超有料，超浓郁”做结合。最终确定了核心idea，创作一支“浓浓抖手舞”，用产品端创意驱动整个营销链路，让TA乐意看的同时参与抖手舞模仿创作。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" title="1660535411918266.png" src="https://file.adquan.com/uploads_img/1660535411cd0b3.png" alt="图片3.png" width="600" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">02、让品牌创意有B站内味</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再来看具体执行，创意有B站内味容易，但如何让品牌创意有B站内味？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">答案就是，<strong>让产品化身为魔性本身。</strong><strong> </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">先来看看魔性的歌词文案：一起来跳/奇多浓浓抖手舞/浓浓浓浓/奇多浓郁抖不掉/浓浓超浓郁的奇多/浓浓浓浓浓浓浓浓/抖抖 抖不掉的奇多/抖抖抖抖抖抖抖抖抖抖… …，可以说全篇歌词都紧扣了产品“粉多浓郁”的特质，把其转化成魔性重复的语言，并在欢乐的气氛中把Slogan进行输出。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再来看看魔性的舞蹈动作：超夸张但有趣的神情，配上让人看跪了的魔性抖手动作，戏剧化氛围拉满同时，再次把产品吃完粉超多，不得不抖手的信息点持续输出。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" title="1660535477267431.gif" src="https://file.adquan.com/uploads_img/1660535477ltbj9.gif" alt="动图1.gif" width="600" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" title="1660535491166585.gif" src="https://file.adquan.com/uploads_img/1660535491javd5.gif" alt="动图2.gif" width="600" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" title="1660535505631754.gif" src="https://file.adquan.com/uploads_img/1660535505zv389.gif" alt="动图3.gif" width="600" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再从整体的视听体验上看，经典歌曲的魔性二改，TA喜欢的二次元风格融入，产品浓郁卖点的HighLight，不得不说，产品成为了魔性本身。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">03、精细化耕种TA的参与度</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">精准的策略下，带着产品玩的品牌核心创意彻底出圈了，但这并非最终的胜利。核心的落脚点是要带动用户的二创、魔改，形成滔天巨浪的自来水。而这往往是最艰难的部分，所涉及到的方法也会更加追求细节与专业。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">生意型创意的方法论认为，<strong>在“碎片化”背景下，用户追求新鲜感、多样化的内容，应对之策只能是不断提高内容的生产效率。</strong><strong> </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就意味着品牌不得不从营销内容是“艺术品”的传统思维中跳脱出来，而选择提供多元多样化的内容，来带动用户的参与度。品牌为此必须要搭建一个完整的内容生产系统，可以包含：品牌自生产、明星、KOL、KOC、联名、KOS等众多角色。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" title="1660535529979672.png" src="https://file.adquan.com/uploads_img/1660535529da9i0.png" alt="图片4.png" width="600" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">赞意根据奇多玉米棒此次的具体需求，通过三方面的力量来提高品牌的内容生产效率，其一是借助UP主们的产出和影响；其二是品牌要自生产内容“主动搞事”；其三则是吸引TA们自愿参与，成为KOC，为品牌制造UGC。但接下来的核心是如何选择KOL，如何尊重他们的内容调性，同时又辅助他们的内容始终能让产品成为英雄。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">经过分析，项目主要锁定生活区的头部UP主，他们有着较高的粉丝量，而且粉丝的互动表现也十分优质。但精细化的内容管理在于三点：1）UP主的二创内容，品牌尽可能不去影响其原创性；2）与此同时会锁定两个底线，保留产品的USP与品牌价值观，以免过度放飞自我；3）此外还会分析UP主以往表现出色的内容，给予一定的内容创作建议。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在三个创作建议下，UP主们开始整活了，而且一个比一个精彩，同时品牌还通过硬广助力，19位UP主中有10位登上B站排行榜，且3个进入全站TOP100 ，整体播放数据和互动数据超2倍完成！ </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">UP主们纷纷上榜，有效激发了TA们的关注与跃跃欲试的心情，品牌也与此同时主动发起引导用户参与的召集令，通过大量的奖励与互动内容玩法，吸引用户参与。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" title="1660535558376862.png" src="https://file.adquan.com/uploads_img/1660535558nwqsa.png" alt="图片5.png" width="600" referrerpolicy="no-referrer"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在召集令的页面中，品牌调动了丰富的UGC战术，从参与奖励、分享博文、观看UP主内容、赠送零食礼包，再到扭蛋观看魔性舞蹈GIF、大奖抽取等多种玩法，从内容、互动、KOL、奖励等角度全方位拉满用户的参与度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最终通过内容生产的精细化运作，奇多成功激发目标用户们积极地点评赞与模仿参与，品牌召集令页面的浏览量与讨论量也远超品牌平均水平，在暑期同预算级别的案例中表现TOP1！ </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">综上，品牌下场到B站玩梗，或者在任何渠道与05后们沟通，绝不能是一场独角戏，也不是孤独自嗨，更不是把产品抛诸脑后，只求好玩。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从生意型创意的逻辑来看，正确的姿势是：<strong>理解产品是谁、TA的话语体系是什么，然后捆绑品牌去创作其喜闻乐见的内容，并通过内容生产体系实现覆盖度与参与度，打造真正的生意型创意。</strong></p>                        <div style="height: 1px"></div>
