<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">在这个以用户为导向的时代，<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E8%90%A5%E9%94%80" target="_blank">品牌营销</a>从广而告之变成了<a href="https://www.niaogebiji.com/tag/%E7%94%A8%E6%88%B7%E8%AE%A4%E7%9F%A5" target="_blank">用户认知</a>持续深化的过程，这样消费者在选择产品的时候，便能从有限的记忆中做出偏向于品牌的判断。</span></p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数字经济时代，品牌在重新理解与锁定用户的过程中，发现在过去5至10年的时间里，那些围绕着拉新、获客的用户认知体系成为了企业快速增长的助推器。如：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>谈新锐咖啡品牌，年轻人就迅速想到了</span></em><span><em><span><strong>瑞幸咖啡</strong></span></em></span><em><span>；</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>觉得饿了，立刻会拿起手机点开</span></em><span><em><span><strong>美团外卖/饿了么</strong></span></em></span><em><span>；</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>说起国产手机品牌，脑海中立刻会呈现</span></em><span><em><span><strong>华为/小米</strong></span></em></span><em><span>；<br></span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>着急外出，会毫不犹豫的地打开</span></em><span><em><span><strong>滴滴出行</strong></span></em></span><em><span>；</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>想要吃火锅解压又追求优质的服务，</span></em><span><em><span><strong>海底捞</strong></span></em></span><em><span>成为了众多年轻人的首选等。</span></em><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不难发现，产品可以是具象化的物品，可以是品牌提供的服务，也可以将用户需求与服务连结到一起的媒介产品。可见，品牌建立用户认知的对象并非只是产品本身，而是需要建立品牌在用户心智中的差异化。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>品牌差异化，即重构用户认知。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌应该如何重构用户认知？或许，我们从这些案例中能够寻找到可取之处。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很明显，那些在产品/用户资源、商业模式等方面没有优势的品牌，想要优于竞品脱颖而出，以重构用户认知为切入点或许是一个捷径，可以从哪些方向入手呢？</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品功能方向的用户认知，是将产品特色转化成用户能懂的营销语言，驱使用户行为的内容，去强化了产品卖点。而近年凭借产品功能深化用户认识做到品牌掷地有声的品牌比比皆是。而产品方向，至少有两种：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一种是产品功能效果。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如云南白药凭借「活血、止痛」功能，将产品品类成功拓展，，并通过脍炙人口的广告语<strong>“如果伤痛在所难免，云南白药在你身边”</strong>，让云南白药走进千家万户。细数云南白药创可贴、牙膏、湿纸巾等产品，基本上都具有「活血、止痛」的产品卖点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8140103196322298bd6f2e1.69540849.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源于微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与云南白药一样通过产品功能去深化用户认知，并一直活跃在市场中的便是王老吉，王老吉另辟蹊径将人性弱点与产品功能结合，让<strong>「怕上火，喝王老吉」</strong>的广告语走进用户的心智中，并巧妙体现了产品的卖点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19494831686322298c07a2f7.35633500.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源</span><span>于品牌官方微博</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老字号用产品功能为品牌建立差异化，而重构了用户的认知实现了品牌焕新，刺激了品牌的活跃度，提升了品牌的辨识度，使云南白药、王老吉为代表的老字号成功活跃在年轻消费市场成为了畅销品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而新锐品牌从诞生之初，就注意到强化用户认知对品牌的意义，通过细分赛道凸显产品极具辨识度的产品功能，而被大众熟知，如以代餐为主的新锐品牌王饱饱，就通过凸显产品的代餐属性，而抓住了细分赛道与认知的两大亮点，实现了品牌飞速的发展。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/7688561216322298c3ee178.93378611.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源</span><span>于品牌官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二种是产品配料的差异。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品配料的差异更多的是产品配料中有刷新用户认知的成分，或属于行业类的开拓者，才有可能将产品成分打造成用户对品牌的认知，即产品中“独一无二”的配料/配方，是品牌占领用户心智的又一方法。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如新锐品牌林清轩，将产品成分中独特的<strong>红山茶花</strong>提出来，将产品与<strong>“天然”理念</strong>绑定，强化了品牌的核心竞争力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12631641096322298c9f5a33.98397231.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源</span><span>于品牌淘宝店铺</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如金龙鱼的<strong>黄金配比1:1:1</strong><strong>，加零反式脂肪</strong>，成为了品牌极具辨识度的记忆点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1534959546322298d050c87.28460707.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源</span><span>于品牌淘宝店铺</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是林清轩还是金龙鱼，品牌之所以能够被用户认识甚至认同，就是因为产品阐明了自己与竞品之间的差别，形成了差异化认知，也让产品成分成为了品牌一块金字招牌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不难发现，以产品向为切入点打造用户认知的品牌，总能够迅速实现从认识、认知、认同到消费者心甘情愿认购，这也展现了产品效果差异化的魅力。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过产品方向去打造用户对产品的认知 ，能够在用户的潜意识中埋下种子，让消费者在有需求的时候自然想要购买该品牌，而那些懂得运用产品的使用场景/用户生活场景作为去打造用户认知的品牌，则是抓住了当代年轻人追求的用户体验感，打造代际使用/消费/生活场景，深化用户的认知。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如滴滴打车为什么能够得到用户更广泛的认可，在兵法先生看来，滴滴重构了消费者对出行等待的认知，<strong>从用户等待出租车到专车等待消费者这一转化</strong>，让用户出行时就很自然地拿起手机使用滴滴App。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13921047436322298d4516b0.09279834.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源</span><span>于品牌官方微博</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说滴滴出行是抓住了用户生活中的痛点解决了出行等待的问题，那江小白、RIO微醺等酒类品牌就是通过细化产品的使用场景，去深化用户的认知。让青春的小酒</span><span>（江小白）</span><span>与一个人的小酒</span><span>（RIO微醺）</span><span>在潜移默化中通过极具代入感的使用场景影响着用户的认知，也让江小白、RIO微醺成为了年轻人不同心境下的共同选择。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与产品功能、场景为切入点去打造用户认知不同，通过感性的情感去打造品牌的认知，是通过营销让用户与品牌之间建立更深层次的情感联系。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如小米与年轻人做朋友，喊出<strong>「为发烧友而生」</strong>，让雷军收割了众多年轻消费群体，从最初的不成熟到现在得到众多用户的认可，不得不说小米通过与消费者做朋友的情感认知十分到位。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2539007806322298d72cb22.30155252.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而华为品牌中蕴含的科技性让更多人崇拜，让“国民品牌”的形象深入人心，虽然品牌从未围绕着“国民品牌”去构建营销思路，却通过《悟空》、《AI在防水》、《Dream It Possible》等广告，建立起了品牌更懂用户需求与心理的广告，也展现了科技品牌有温度有深度的另一面，而在悄无声息中建立了“国民科技品牌”的用户认知。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如董明珠与格力、潘石屹与SOHO、王健林与万达、褚时健与褚橙、娃哈哈与宗庆后、陶华碧与老干妈、马云与阿里巴巴等，这些人物故事即品牌故事，用户也通过人物的传奇故事对品牌产生了熟悉感，进而对品牌的好感度倍增。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用品牌关联人物去打造品牌的认知，更具亲和力、传奇色彩，也更容易被用户记住，加上人物本身独一无二的经历，也成为了品牌联系用户的最佳媒介。</span><span></span></p>
