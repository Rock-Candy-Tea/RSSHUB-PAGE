<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经看过一句话，打卡<a href="https://www.niaogebiji.com/tag/%E5%B1%B1%E5%A7%86%E4%BC%9A%E5%91%98%E5%BA%97" target="_blank">山姆会员店</a>是前几年的一种潮流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然带有调侃性质，但<span><span>山姆会员店确实屡屡上演<span>真香定律</span></span></span>。</span><span>这两天</span><span>山姆又对中国4亿中产</span><span>下手，</span><span>山姆上海宝山店正式开业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>内行人看到的，<strong>是</strong>山姆在中国加速开店。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>外行人看到的，<strong>是</strong>中国消费者被打脸了。</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>山姆这次采用预约制，8月27日-9月4日进入山姆，需要<span>上山姆APP</span>提前预约，</span><span>宝山店</span><span>开业半天，入场预约券就被抢光了。</span><span> </span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>预约只是拿到了一张入场券，</span></strong><strong><span>进店和出店才是真杀场。</span></strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使下雨，也无法浇熄上海人的热情。</span><span>记得上次说排长龙还是南京茶颜悦色，<span>山姆上海宝山店，</span>再次创造开一家火一家的神话，</span><span><strong>消费者</strong><strong>入场</strong><strong>排队1小时，结账等1.5小时。</strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新开的宝山店真有那么神吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>宝山店是山姆在上海开的第四家门店，而</span><span>根据山姆官方平台介绍，新店跟其它山姆会员店没太大区别。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提供约3000种商品，包括自有品牌Member’s Mark<span>以及</span>来自全球30个多国家和地区的直接进口商品。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同的是，新的山姆店准备了百款新品，如斯卡43年威士忌、乐高法拉利旗舰版、<span>香格里阿拉松茸王</span>等新品。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不得不说，山姆再一次成功收割“中产们”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看到<span>山姆</span>上海宝山店的<span>盛世</span>，也有网友感慨<span>难道互联网没有记忆</span>。毕竟新疆事件之后，<span>抵制山姆</span>的呼声就没有断过。</span><span> </span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">2</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年山姆“下架”新疆产品事件惹众怒，有消费者上山姆会员商店的APP，输入“新疆”关键词进行搜索，<span>发现</span>得到的提示是“抱歉，没有找到相关商品”，<span>这无异于“封杀<span>”</span>新疆产品。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>客服回应“只是没有库存了”、“不是下架”，</span><span>让消费者默默地给山姆记了一大过，也就有了后面的抵制退卡风波。</span></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1328754982630ce734f06590.19539588.jpeg" width="333" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>山姆的门店一般开在较为偏僻、</span>路途较远的地方，因此无论是<span>选择</span>线上APP还是线下排队形式退卡，<span>消费者</span>都收获不好的体验。 </span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>而另一边的家乐福、<a href="https://www.niaogebiji.com/tag/%E7%9B%92%E9%A9%AC" target="_blank">盒马</a>的操作亮了，</span>盒马明目张胆地截胡，广而告之<span>只要持有山姆会员卡，就可以去盒马进行消费，家乐福超市也趁机推出了“新疆好物节”活动。</span></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1177459266630ce7354de044.03875688.png" width="333" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些超市像是在内涵：没有山姆，也还有盒马和家乐福。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>除了因为新</strong><strong>疆产品缺席得罪消费者，山姆也曾被爆跟同行闹掰。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>山姆曾跑到盒马总部楼下，搞会员拉新地推，在同行家门口抢起会员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>家乐福在上海开店，就遇到供应商上演“携货出逃”，据说是山姆拿<span>“二选一”</span>对供应商施压</span><span>，后来家乐福和盒马<span>还</span>联合指控山姆垄断。</span></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1516169254630ce735abf7f5.33744011.png" width="333" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>种种公关危机在前，部分<span>消费者对</span>山姆形成做事强硬、态度傲慢的印象，<span>品牌的好感度</span><span>可想而知。</span></span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>但如今上海宝山店一开，</span></strong><strong><span>似乎也不妨碍会员选择性健忘。</span></strong><strong><span>有人嘴上说不要是断然抵制，而有人身体很</span></strong></span><span><strong><span>诚实悄悄付款</span></strong></span><span><strong><span>。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>毕竟山姆不仅是仓储超市，还是社交货币。</span></strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>只要打开小红书搜索“山姆会员店<span>”</span>，就会发现多达30万+篇笔记。美食博主在安利</span>会员卡指南和必买清单，时尚博主晒起山姆工业风的拍照攻略。</span><span> </span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>山姆会员店被视为宝藏超市是有迹可循的，光</span></strong><strong><span>看山姆的</span></strong><strong><span><span>外号就知道：</span>千元店、中产的超市。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>千元店，即看似便宜大容量，但随便一逛也<span>能花</span><span>一千来块。</span>好不容易来一趟，一定要买够本，想必也是多数人心态。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>中产的超市，形容</span><span>山姆是中产的</span><span>身份象征，说明</span>山姆的目标客群，是向往“中产生活方式”的年轻人和中产家庭。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>自</span><span><span>1996年<span>山姆</span></span>在深圳开设第一家会员店</span><span>以来，</span>凭借</span>“中产生活方式”的</span>营销模式，成功拿捏中国消费者。</span><span> </span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>会员这么贵，到底贵在哪呢？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这也是接下来说的另一</span></strong><strong><span>个<strong><span><strong><span>营销</span></strong></span></strong></span></strong><strong><span>策略：凡勃伦效应。</span></strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单来说，就是“追高不买低”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般大卖场遵循长尾理论，通过动辄过万的SKU<span>（商品库存进出计量的基本单元）</span>，尽可能关照到消费者千人千面<span>的</span>需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>但山姆更推崇“凡</span></strong><strong><span>勃伦效应”，认为精专大于广全。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换言之，商品的受欢迎程度是跟价格成正比的，<span>越贵越好卖，</span>因为消费者愿意为更靠谱的产品买单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1759675362630ce73bb7b4b4.05618124.png" width="333" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友评论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>为此山姆一直把SKU控制在3000-4000左右，这是同等</span><span>面积超市的1/4。<span><span>弱水三千只取一瓢饮背后，山姆只</span></span><span>为会员筛选高品质的商品。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>最后是大单品的策略。在精选各品类的领导品牌基础上，再精选最优质的单品。</span></strong></span><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据说山姆内部流传两句话：</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“如果你新商品的成功太高，说明没有敢于冒险，也就无法培养出让会员流连忘返的‘英雄单品’。”</span></strong></span><span><strong><span> </span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“满大街都有人做的东西，我们就不做了。”</span></strong><span> </span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小到一份香蕉，选的是菲律宾都乐，只选千分之四的“精品”。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大到一个户外野营帐篷，<span>买手</span>会先入行再选品，参加户外运动展对行业足够了解后，再去干精选的活。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/736058377630ce73c2f3a02.48558575.jpeg" width="333" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：昆山论坛</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>山姆倾向“凡勃伦效应+单品驱动”模式，也是有多方的考量：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>其一，避免消费者货比三家，降低他们的决策成本；</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>其二，通过</span></strong><strong><span>会员渗透率、复购率高的单品，实现</span></strong><strong><span>单品带动整体销售。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>其三，每增加一个单品，</span></strong><strong><span>都会造成各环节成本的上升；而控制单品数量，可以更好地跟供应商议价，拿到更低价格</span></strong><strong><span>。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就拿食品来说，DT财经2021年曾统计出，小红书网友最喜欢山姆商品，前三名就是麻薯、瑞士卷和烤鸡。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可想而知，<span>每个SKU是优中选优</span>，山姆也凭借明星产品圈粉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>不可否认，作为一家“认卡不认人<strong>”</strong>的会员超市：</strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对它无感的消费者，会觉得山姆为赚会员费吃相难看，实属是叫人交智商税；</span><span>而觉得入股不亏的中产们，看中的可是高品质的选品和装中产的场景。</span><span> </span></p>
