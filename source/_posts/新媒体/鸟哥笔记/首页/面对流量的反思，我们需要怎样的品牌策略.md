<p><span>神州</span><span>系的<a href="https://www.niaogebiji.com/tag/%E6%B5%81%E9%87%8F" target="_blank">流量</a></span><span>传说</span><span>，</span><span>奠定了口碑与</span><span>传播</span><span>效果之间的可操作的正向</span><span>联系。</span><span>通过利用人性</span><span>弱点，</span><span>结合明确</span><span>的品牌定位、树立</span><span>正确的价值理念、利用</span><span>合适</span><span>时间传递合适的消息、</span><span>找到</span><span>合适目标受众和</span><span>传播发酵者，最后，</span><span>将品牌价值观可能受到的伤害限制在相对可控的小范围中，做到了</span><span>品牌和流量之间</span><span>的</span><span></span><span>动态</span><span>平衡。</span><span></span><span></span></p>
<p><span>本期热评，17PR特邀</span><strong>伟达中国高级副总裁、前奥美北京战略总监周晨</strong><span>女士，在利用旧案例达成品牌与流量之间的动态平衡之后，将继续分享关于营销流量的反思和品牌对策。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>“摒弃流量”说法事实上表达的是，所有品牌都在迫切探索，如何与消费者更有价值和意义的数字互动。中国人每天花在手机的时间平均高达6个小时，是美国人的2倍，每天通过手机采购的商品和服务价值高达十几万亿人民币。新冠疫情的爆发，加速了消费者的数字化，也给品牌带来了前所未有的挑战。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>挑战一：</span><span>全渠道的摊子越铺越大，</span><span>数字营销</span><span>的渠道越来越复杂</span><span>。</span></strong><span></span><span>面对五花八门的触达渠道，全渠道策略一直是品牌方追捧的。</span><span>阿里腾讯这样的平台影响力巨大，</span><span>覆盖从电子商务、社交</span><span>、影视娱乐</span><span>、游戏、音乐、地图和电子支付</span><span>等方方面面，涉猎到购物、出行、金融、通信、工作、教育等各种场景。</span><span>虽然是聚合平台，但当主流应用分裂为数百个小程序和子频道，本身就是一种多渠道的生态。</span><span>此外小红书、抖音、</span><span>b站</span><span>、知乎等新的媒体形式也备受欢迎，让消费者对于信息关注度更加碎片化。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>挑战二：</span><span>公域还是私域，是个问题。</span></strong><span></span><span>一方面，类似微信小程序可以提供类似微信原生应用的体验，让品牌在私域流量的运营方面有了更低的门槛，让品牌可以以更轻量化的方式触达并激活用户，并拥有数据。</span><span>但另一方面，你又不能放弃类似天猫、京东等公域流量的运营。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>挑战三：</span><span>越来越贵的营销成本，获取流量和转化陷入</span><span>“内卷”</span><span>。</span></strong><span></span><span>通过流量所产生的增长边际效应正在减弱</span><span>。</span><span>在新冠疫情导</span><span>加剧了</span><span>数字营销成本进一步上升，品牌</span><span>不得不加速和加大通过</span><span>数字化方式</span><span>触达</span><span>中国消费者，有些品牌甚至将所有营销预算都投入到数字化营销。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌的数字化营销活动不仅仅要关注触达范围及引发的消费者行为和转化</span><span>，</span><span>更要</span><span>不断考察是否能够加深消费者与品牌的关系</span><span>，</span><span>使得在未来消费者能够持续与品牌发生互动</span><span>，</span><span>主动对品牌进行探索、甚至主动传播品牌形象，</span><span>从而降低对流量红利依赖</span><span>，</span><span>而是通过存量运营</span><span>，</span><span>创造新商机。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是流量是被追捧还是摒弃，制定品牌策略，首先需要梳理自身，根据自身的客群特征、品牌成熟度、甚至自身的数据分析能力、信息化能力有清晰而客观的认知，明确优势劣势，选择赛道，制定品牌策略，平衡私域还是公域投入的比例，设定鲜明的品牌目标，并制定合理的预期。</span><span>比如通过品牌忠诚度计划既可以作为私域流量的运营挖掘私域的用户复购潜能，实现用户平均收入增长，也可以作为吸引公域渠道平台获得更多高质量目标受众的手段。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户基数多、知名高的大牌更合适天猫京东这样的大平台深度合作。多个美妆品牌目前都与天猫展开从品牌、数据、新品研发等多维度营销创新合作，欧莱雅与天猫新品创新中心展开数据合作，雅诗兰黛集团与天猫的打通会员体系，资生堂在阿里开设“驻猫办”等。品牌在大平台的生态圈中精准目标用户，制作高匹</span><span>配高传播内容，能够极大地提升品牌营销的投资回报率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而类似完美日记新品牌，则更合适通过利用小红书这样的社交媒体走网红和粉丝经济的路线，小红书是KOL和KOC的集中地，品牌通过分析，找到最匹配目标用户的明星与红人，通过这些宣传大使来建立传递产品使用心得，潜移默化传递品牌理念，让品牌与消费者获得交互。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而对于私域流量的运营，品牌中的数据科学家和Geek们更能抓住先机。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些品牌往往在传统时代就非常重视数据分析和市场研究，是最先投身数字化转型浪潮的先驱，并勇于尝试各类Martech的技术，很多很早就引入CRM和会员管理的理念。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如星巴克很早就开始星享俱乐部会计划，从实体卡到线上，无论是线上自建APP还是与饿了么/微信小程序的会员体系，都很好贯彻了星巴克的品牌客户忠诚度计划。</span><span>而类似阿迪达斯和耐克一直拥有强大的数据分析和市场研究能力，拥有精准细化的用户画像，因而在能够迅速的通过个性化的服务和营销内容提高客户的参与，另外这些品牌往往能够做到对于私域平台和公域平台有明确的定位，实现品牌投入平衡。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“计无策无以为施，策无略无以为持”，策略是一种决定，意味着选择和取舍。品牌策略亦是如此，单纯唯流量论或唯品牌论都是不可取的。有时候，我们需要牺牲短期流量KPI，着眼长远，为更长远的企业利益去建设品牌，这是品牌人最喜欢听到的说法；但需要指出的是，有的时候我们则需要在品牌上让步，追求流量，为的是帮助企业渡过眼前难关，先活下去再说。神州专车的案例亦是如此。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌策略制定者不要惧怕流量，需要了解流量在不同场景下，与品牌之间的关系。</span><span>品牌也好，流量也好都是企业发展的工具，厘清企业的真正业务目标，梳理清楚短期业务目标和长期企业愿景，我们面对流量品牌之争才会更加客观和淡定。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017</span><span>年，</span><span>可口可乐的</span><span>C</span><span>MO</span><span>退休，可口可乐并未任命新的</span><span>C</span><span>MO</span><span>，而是</span><span>宣布增设首席增长官</span><span>C</span><span>GO</span><span>取而代之。</span><span>媒体用</span><span>“彻底取消</span><span>C</span><span>MO</span><span>”的标题党风格描述这一事件</span><span>，</span><span>在行业引起轩然大波，被视为品牌营销向效果营销低头的信号。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>不过时隔两年，可口可乐重新恢复</span><span>C</span><span>MO</span><span>职位，回归以品牌管理、创意为核心的营销运作，饱受瞩目的</span><span>C</span><span>GO</span><span>职位则不再被需要。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>回到这段长达两年之久的职位变革之争，无数营销人都在讨论</span><span>增长和流量的必要，但无论何时，</span><span>品牌的力量</span><span>都</span><span>不可取代</span><span>，增长依旧会是目标。</span></p>
