<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月11日，#北上广没有<a href="https://www.niaogebiji.com/tag/%E9%9B%AA%E8%8E%B2%E5%86%B0" target="_blank">雪莲冰</a>#登上了微博热搜，被冷链、仓储等硬性成本局限的平价雪糕走不出“山海关”，无数人在评论区感慨“想要实现雪糕自由太难，什么时候能买到五毛钱的雪莲冰”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在很多城市里，你很难找到雪莲冰，但这种配料简单的柠檬味冰块却不难复刻。小红书和抖音等内容平台上，美食博主晒出视频，调配柠檬、白糖和清水，通过冰格模具完美还原雪莲冰口感，“五毛钱都省了，自制雪莲冰实现夏天雪糕自由”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/149154893162cdffc6527129.57199490.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在“雪糕刺客”成为这个夏天热议话题的时候，越来越多人在网上看起如何自制雪糕和冰块，晒配方，晒制作过程都能撬动起情绪，收获几十上百万点赞。内容端的火热体现在消费端，一些商家与厂家看准时机，适时推出各种冰格模具。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>批发平台1688上，冰格模具成交额前十名中，有七位是来自义乌的厂家，一位义乌厂家告诉「<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%95%86" target="_blank">电商</a>在线」，他们从六月至今已经卖出将近30万件圆球冰格模具，“每天差不多要发一万单，现在厂里生产都来不及，周边的这些塑料制品厂也都在做这些模具。别看价格就三四块钱，有些商家利润能有100%。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当消费影响到趋势，冰格模具就成为了夏日爆款，嗅觉早已被锻炼得越来越灵敏的义乌厂家早早入局，将冰格模具打造成为爆款，北上广也好，十八线小镇也罢，都可以实现雪糕自由。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/101193751762cdffc67fb782.84583895.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>冰格模具，迎来翻红</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音上，“#冰格模具”的话题播放量已经达到了6.7亿次，不少人将冰格挂上小黄车，简单的制冰过程就能引发无数人的点赞与购买；小红书上，关于冰格的话题笔记也有2万多篇，各种DIY水果冰和饮料，各式冰格的测评，内容添得满满当当，评论区询问链接与店铺的观众不在少数。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/83298309862cdffc6a49f18.05460577.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内容平台的火热进一步体现在电商平台上。淘宝上，冰格模具月销10万多件，拼多多上，冰格模具也拼出了10万多件，而这些冰格模具背后，则是早早就嗅到消费趋势的义乌厂家们。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“义乌本身有生产塑料小商品的经验和优势，像我们每年三四月份就开始加大生产冰格模具的力度，因为每年夏天都卖得不错。今年是特别火，我们周边的厂都在做。”义乌厂家阿城已经做了五年塑料制品生意，他表示，今年夏天冰格模具销量“爆了”，之前的备货都不够卖，现在厂里每天在加班加点生产。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>冰格模具并不是什么新鲜玩意。在几十年前冰箱进入寻常百姓家庭的时候，冰格模具就相伴而生，只不过大多是塑料制作的方形冰格模具，这也是大多数厂家口中最为简单的一种。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>塑料冰格模具制作成本很低，阿城告诉「电商在线」，工厂里只要有注塑机就可以生产，开模成本也不高：“没什么技术含量，基本每个塑料制品厂都能生产，按照厂子大小，一天就能生产几千几万件。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费带动趋势，当消费者使用冰格模具自制雪糕、水果冰球等，冰格模具本身也有了更多变化：带有塑料棍的雪糕造型冰格模具，各种造型颜值更高的冰格模具，可以反复使用的双层冰杯，不破坏饮料酒水浓度与口感的不锈钢冰块……为了让造型各异的冰块更好脱模，一些厂家用起了硅胶材质，但相应带来了成本的上升。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/118833783462cdffc6d7f860.63407376.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><em><span>各种造型的冰格模具</span></em></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比塑料材质的冰格模具，硅胶材质成本要高出四五倍甚至更多，阿城的塑料制品厂目前还是生产各种塑料冰格模具为主：“硅胶生产要求会高一些，还有那种特殊造型的，开模也比普通的规则方块、球形造型要麻烦，产量上就会受影响，因为价格比较高，销量也会低一些。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在阿城和他身边的厂家看来，今年的爆品是塑料的圆球冰格模具，光是阿城厂里就卖出了将近30万件，身边不少厂家也都卖出了几万件甚至十几万件。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「电商在线」在和几个厂家与商家对话后，发现了几个必不可少的原因：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1.夏季对于冰格模具的需求大，塑料材质的圆球冰块价格便宜，消费者的决策成本更低，销量容易起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2.大多数消费者还是在追求实用性。相较于普通方块造型和特殊造型，圆球冰格模具冻结速度快，脱模方便，适合日常使用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3.内容平台上，各个博主发布的内容不少是圆球的冰格模具，电商平台高性价比的商品供给，共同打造出这个夏日爆款。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/178489894762cdffc70f2618.12654004.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>不起眼的生意，利润100%</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在各个电商平台上，冰格模具的价格一直不高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年爆火的33格圆球造型冰格模具，在电商平台的单个售价基本在3.5—6.5元，商家都注明了包邮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿城向「电商在线」透露，这种模具的出厂价格在0.8—1.1元左右，商家并不会大量进货后再售卖，而是走厂家一件代发，借助义乌的物流优势，一件冰格模具算上包装和物流，总花费在2.6—3.5元左右。在电商平台上，销量最高的也不是单个装的圆球冰格模具，而是两个装的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/148050614662cdffc7320425.52336001.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>商家范范告诉「电商在线」，组货卖得好原因一个是划算，“一个装的卖4元，两个装的卖7元，大家都会觉得两个装的更划算。我们合作的厂家两个装代发总价4.5元不到，商家这边能卖出6.8元甚至9.9元，最高的利润能超过100%。” </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一些商家还转向了竞争压力更小的国外市场，做起了外销生意。在速卖通上，国内爆火的圆球冰格模具已经卖出将近1000件，在国内售价单个4元，国外售价却已经达到了4.86美元，折合人民币约33元。阿城表示，外销物流成本会提高，但是利润率也会更高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>利润因素之外，不少商家售卖圆球冰格模具，是想通过这个爆款商品为店铺引流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>范范主要售卖的是日常百货，圆球冰格模具是店铺的夏日流量入口：“模具是引流款，其他东西才是利润点。这种圆球冰格模具吸引来的基本都是有相同需求的消费者，我们还会推荐店里的冷水壶、随行杯和玻璃杯等等，不少人会顺带买一件。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/111797875762cdffc76713d4.12759533.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><em><span>不少售卖冰格模具的商家也在售卖家居百货</span></em></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，不少商家也在用这些爆款商品提高店铺评分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在选择厂家一件代发时，泡沫袋包装能够节约3毛左右的成本，但一些商家会要求使用纸箱+泡沫纸包装，以此减少破损概率，降低售后率，也会要求厂家做到当天发货——这些数据都是店铺评分的组成内容，而店铺评分会影响到平台流量分配，也会影响消费者的购买信心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>夏日爆款结合上义乌速度，一群敏锐捕捉商机的网店店主和快速反应的工厂将不起眼的冰格模具生意作为流量入口，也做出了不低的利润。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/60714575662cdffc79f4b66.24155109.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>小冰块上的消费升级</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在雪糕消费升级的同时，冰块也迎来了消费升级。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经，能快速出冰的制冰机主要面向B端消费者，以商用为主，大多是奶茶店、咖啡店在使用，但在疫情下的宅家经济中，小家电成为了不少人“小确幸”的来源，制冰机也出现了家用款式：更加小巧、省电，减少了制冰时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比几块钱就能入手的模具，家用制冰机的最低价位也在300元左右，但通过“短时间快速出冰”这个卖点，家用制冰机在淘宝上的月销量也达到了2万多台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8816514462cdffc7cf4b32.63478393.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>愿意用比模具高100倍的价格为制冰机买单的年轻人，有着自己的理由。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在小红书发布制冰机博文的莓莓表示，选择制冰机是出冰速度快和不会串味。相比于冰格模具三到四小时才能冻结成型，制冰机出冰速度在十五分钟以内，一次出冰量就能满足1—2人的需求，实现“冰块自由”。同时，家用冰箱不可避免会放上一些食物，冰格模具制作的冰可能会染上其他食物的味道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>制冰机体现了冰块的消费升级，但更多人依然会选择更便宜的冰格模具：一是因为制冰机目前只能使用纯净水制冰，并不能满足消费者想要制作水果冰、雪糕的需求；二是冰块的“夏日限定”属性也注定了生命周期的短暂，相比于空气炸锅这类全年都可以使用的小家电，家用制冰机在天气凉爽时基本就是闲置物品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但冰格模具的生意并不会特别长久，随着生意的火热，越来越多人开始入局低门槛的冰格模具，同质化严重，核心竞争力不高的情况下，赚到钱的商家和厂家越来越少，基本只有第一批入局者能够赚钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“冰格模具没有这么容易坏，大多数人有三四个就能满足冰块需求，大家也开始低价引流，利润只会越来越少，就好像前几年很火的小熊造型冰格模具，刚开始能卖出三四十块，现在卖四五块的也不少。”在众多同行降价销售的情况下，范范也在计划降低价格，做低价引流。</span></p>
