<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>作为商业创作者，介绍企业品牌概念、行业分析、方法论植入的文章最常见，越看似普通的题材就越容易陷入写作误区，最常见问题是</span><strong><span>，“毫无组织的堆砌素材”。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么意思，我举个例子：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去几年，短视频平台对电商业务做出的所有尝试，都可以归结为两件事：「提升“货找人”的效率，培养“人找货”的心智」，前者增长承压，后者更接近平台当下的重心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>想想看，上述商业文开场描述，是否和手机开箱视频相似？</span><span><strong>先打开包装，然后事无巨细描述物件，再开始介绍外观、硬件配置，最后打开几个常用系统APP进行展示。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是不是经常遇到？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>我不能武断的将这种创作类型定义为错误，但可以肯定，这一系列平淡直序的描述手法很难让读者进入一篇文章，发现</span><span><strong>“吸引点”</strong></span><span>在哪里。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>当终于看到中间一部分核心观点时，注意力已经被冗余信息全部消耗掉；除此外，有的商业文开场忽略</span><span><strong><span>“情景营造”。</span></strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如同，很多自媒体在财报节点写品牌内容，上来一顿同比增长、环比下降、某方面忍辱负重的强行结论，以至于读者完全get不到和我有什么关系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>了解这两大创作误区后，到底用什么类型的开场白才能留住读者，我认为可以反向思考，总结出一种介绍新事物的框架，</span><span><strong>用具体的故事来表达观点，引出重点。</strong></span></span><span></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>一、</strong><span><strong>工具代表效率</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>许多人对</span>SCQA组合并不陌生，此概念由美国商业作家芭芭拉·明托（Barbara Minto）在《金字塔原理》一书中提出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>简而言之，即</span><span><strong><span>结构即情境（Situation），冲突（Complication），问题（Question）答案（Answer）</span></strong></span><span>；四者自由组合可以形成不同陈述方法，使人们循序渐进进入主题中。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>市面商业创作常用有四种类型：</span><span><strong>1）标准式（SCA/SCQA），2）开门见山式（ASC），3）突出忧虑式（CSA），4）突出信心式（QSCA）</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>方便先对大框架掌握，我用三个案例呈现：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>①埃隆·马斯克TED演讲</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今天全人类面临最大威胁是什么？“科技高速发展，过去几十年中人类拥有的先进武器，足以可以摧毁地球“几十次”。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，尽管我们拥有毁灭地球的能力，却还没有逃离地球的技术。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，我们今天面临的最大威胁，是没有移民外星球的科技，我们公司将致力于私人航天技术，在可预见的将来实现火星移民计划，然而过程中我们可能遇到四个方面的问题。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这段话结构是不是很熟悉？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没错，埃隆·马斯克（Elon Musk）在TED演讲中通过问题、背景 、冲突 、答案（QSCA）四者组合突出重点，来鼓舞人们信心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这是一种最常规的写作方法。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上来直述问题让读者思考，然后用背景简单过渡回到冲突上，话锋一转，找出冲突原因直接给出解决方案；就像“你有病，以往解决方法都用过，却不对，我这药方与别的不同，你要不试试”？的感觉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>②公司改变奖励体系</span></strong><span><br></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今天主要分享有关“公司销售制度从提成制，改为奖金制的提议”。众所周知，从创始以来，总公司一直使用提成制来激励销售队伍，但是，提成制在子公司业务迅猛发展情况下，造成很多激励上的不公平。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“富裕地区”的员工能拿到大笔提成，买房买车，一般地区分公司还处于亏损状态，员工工资每月都是总部在支撑，所以高管团队一致商讨决定更改奖励体系。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>相关部门领导听完之后，希望落实到位，我认为这四个部分与大家相关，</span><span>在<span>讲之前首先说下，该决策的整个过程与细节”。</span><br></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span>这种商业写法属于开门见山式沟通（ASC），</span></strong></span><span>用更改奖金制度作为（答案Answer），然后引出公司激励不公平的（背景Situation），顺带罗列一些公司的现象作为支撑点（冲突Complication），最后道出目的。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也就是，第一句话先上结论，然后告诉读者接下来会做什么，有什么影响，这种影响背后的细节原因有哪几个方面，环绕中心思想逐一进行展开。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1919570483630c5b1f0808c6.03658799.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>③网络安全不容忽视</span></strong><br><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去几年，网络安全日益受到关注和重视，即便这样，业界依然发生很多数据泄露、攻击的丑闻。比如……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们的系统中有几万个客户以上的数据信息，重要性不言而喻的，可也很容易忽略。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为不发生时，技术人员把大量时间都投入在功能的开发、性能的优化上，日常戒备心又比较低”（常识冲突Complication），那么，我们应该如何保证产研的同时，增加安全性呢？（问题Question）<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>公司非常重视，从研发阶段做好数据加密保障，通过算法，协议、代码审计检查等多个</span><span>维度<span>，最终同时上线前通过第三方公司渗透检测，我认为大家应该首先了解下这家公司产品，从而有效打好配合（答案Answer）。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span>这种<a href="https://www.niaogebiji.com/tag/%E5%95%86%E4%B8%9A%E5%86%99%E4%BD%9C" target="_blank">商业写作</a>手法属于突出忧虑式（SCQA）。</span></strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span></span></span><span>用社会刚发生过的舆论与读者共鸣，然后引出我们系统也有数据（背景Situation），通过大家平时较忙，戒备比较低作为没精力作为冲撞（常识冲突Complication），然后描述（问题Question）。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>日常中，<span>除</span>与上述三者相似的文章外、</span><span><strong>广告语、演讲场景也比较多见。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如：“今年过节不收礼，收礼只收脑白金”，和我年龄相仿的小伙伴都能耳熟能详，配上魔性的唱腔与动画，虽然时隔很久，依然记忆犹新。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乔布斯在发布会上介绍第一款iPhone时，首先阐述“市场有很多手机，但它们不怎么好用，比如在某某方面，然后引出如何更高的塑造用户体验的几个方法，iPhone围绕这些方面做了哪几方面”。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你看，不同组合现实中就在我们身边。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>可是，为什么当自己用时却</span><span><strong>“抛不出来”甚至“言不达意”</strong></span><span>，不能进行有效连贯？这当中一大原因并非是</span><span><strong>“你对概念不了解”</strong></span><span>，而是</span><span><strong>“细节掌握不佳”</strong></span><span>，就像“情景”（Situation）这件事，很多人一句话都讲不明白。</span></span><span><br></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>二、</strong><span><strong>情景可大可小</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先说结论，</span><span><strong>情境包括场景，场景包括情景。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何理解？</span><span>目前各种学者对场景的定义各不相同，但总体来看，主要是基于特定时间、空间和行为及心理环境氛围所构建。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作用是促进“用户和用户，用户与生产者、以及用户与产品之间的连接和协同”，有时，场景也承载内容，社交、游戏等多种服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是不是很复杂？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单而言，</span><span><strong>场景即在某个场所发生的事情，属于当时的情况</strong></span><span>；现实中我们每天会在少则几十个，多则几百个场景中切换，它的形成具备三个要素，分别是</span><span><strong>“用户、背景、目标”</strong></span><span>，换言之，</span><span><strong>“在什么背景下达成什么目标”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这如同：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位典型白领用户，从起床到睡觉大致有13大场景，“洗手间洗漱”到“吃早餐”，去公司路上的“交通”到写字楼“乘坐电梯”，从上班工作到午餐、休息等各环节，甚至回到家中，在卧室“休闲娱乐”均是场景体现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从商业写作角度而言，想要准确“描述场景”，离不开对人性和生活方式的审视，对用户为满足需求（痛点和痒点）的洞察和细分市场精准把控。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这一切并非基于“自我本位</span><span>诉求</span><span><span>”，而是透过产品（事件）看到消费目的或背后意义。</span></span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>举个“软瘾”的开场：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你是否有此类习惯，工作不忙时必须来点咖啡或干点什么碎事儿，通勤路上不刷手机内心就会空荡，即便周末在家追剧，也要买点零食满足“嘴”上愉快，总觉得生活惬意，事实上你正在被“软瘾”所缠绕。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一项哈里斯民意调查（Harris Poll）显示，91%的美国人承认自己有“软瘾”，这不是罪，只是一种安抚自己错误方法，长期以往形成的不良习惯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于案例看，</span><span><strong>场景可大可小，也能更加精细化。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>细分情况下，我们可以比较详细的，描述用户具备什么样的特点，他处于什么时间地点，想达成什么目标。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就好像，90后设计师因昨晚设计评审没有通过，不得不加班，现在，不知不觉已经到晚餐时间，员工餐厅太远楼层又高不想下楼吃饭，所以才选择点外卖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，情景与场景有所不同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>情景描绘手法更讲究“细节呈现”，情景的“景”是具体的，直接吸引人的，具体到场合中的情形或景象；构成因素包括：</span><span><strong>1）运镜，2）感官，2）细节，3）心理</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>举个吃肘子案例：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我双手轻轻一扒，那块酱色的炖肘子就颤呼呼烂开了，露出红白相间的肉，热气还没散开油汁开始外流；我撕下一块，在旁边准备好的酱料中猛蘸几个来回，随即提起，连汤带水全塞进嘴里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>嗯，真香。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不知是热乎劲儿还是对肉香的满足，让旁边几位朋友看的喉咙开始不停耸动，到处都是咽口水的声音；都怪我，不应该当众吃肘子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在电影术语中“运镜”就是运动镜头，一个人通过推、拉、摇、移、跟、升降摄像等手法可以形成多种不同视觉效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>商业写作业同样，如果开场想描绘某个画面感，就要在动笔前，把自己想象成电影摄影师。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这段吃肉运镜，是最基础「近景、中景、远景」的过程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你看文字时脑中的画面，也先是一块肘子肉占据了屏幕，接着肉被扒开，呈现出内部样子，再到感官，口味心理上的变化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然运镜还可以是「远景、中景、近景」，这种方式往往用在表达沧桑、豪迈、命运等宏大情感上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如同，纵观教育行业跌宕起伏30年，从新东方著名三剑客看“个人辉煌”的文章；写作时就可以先勾勒远方山川河流，再描绘每个人几十年面临风雨无阻的奋斗，最后聚焦主角眼里闪动的泪水，来一个近景特写。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1465858060630c5b1f50c5b3.11528300.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，</span><span><strong>运镜方式并不固定，一般多描述人物居多，商业作者需要根据想要表达的情感而自由组合。</strong></span><br><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>世界上场景可分为现实、虚拟、现实增强三大种类，</span><span><strong>情景在此基础上做进一步细化的描绘，情境则是两者相对或结合的情况，呈现的主要特征是，“一段话表达的修辞是否符合整体段落逻辑”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这好比，上半部分强调“年轻人最大问题</span><span>是</span><span><span>意义危机”，中间过程描写“技术发展是未来趋势”，下半部分却说“一定不要失业”，明显不在相同一个频道，对不对？</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>综合上述，看出三者有什么不同吗？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>用场景时</span></strong><span>，应该抓住人物与人物之间的关系、人物与环境之间的关系，并给予这些关系以恰当的处理，如主要和次要的关系、重点和陪衬的关系等，描写要带有明确的目的性，不能事无巨细。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>用情景时</span></strong><span>，恰恰相反，尽可能像定焦镜头一样，聚焦在某个地方，把它放大5倍、10倍，给过程解剖加上动作，视觉、听觉、味觉、嗅觉等齐上阵，才能让读者产生画面感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>用情境时</span></strong><span>，前后与主题要紧密结合，不游离于主要情节之外，不能出现背道而驰情况，有主有次，有整体有细节。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而值得一提的是，</span><span><strong>商业写作也属于讲故事，故事本身是“新生事物”，这类文章开场在「新」而非「事物」上。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想让读者觉得“新”，只需要用某个“旧场景”作对比，再用故事/情景表述就能突出，我把这种方法称为“以退为进”，直接上来也能营造一种熟悉场景。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>三、</strong><span><strong>如何设计冲突</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>英国著名小说家约翰·勒卡雷说过：“猫坐在自己的垫子上”，故事不能这样开始，“猫坐在狗的垫子上”，要这样开头才对。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么故事要从哪儿讲起呢？要从冲突讲起，在开场就要设计一些事情，让平淡的日常生活发生不平常的情况，</span><span><strong>如果直接摊牌，也就没有惊讶可言。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么是冲突？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两个或两个以上相互对立，互不相容，并且需要同时存在而又处于矛盾中的状态（如观念、利益、意志）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像一个女孩子买包，如果只是用来装东西，买个几百块钱的就可以，那为什么还要买几万块钱，甚至几十万块的包呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>教育家圣严法师曾说，“人这一辈子需要的不多，想要的太多”，需要和想要之间就有冲突，冲突就会产生需求；左脑和右脑也有冲突，左脑理性，右脑感性，两者无法兼并，不信你想想看，是不是。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从商业写作层面，我认为开场设计有效的冲突离不开两方面：</span><span><strong>1）站在读者立场说问题，2）制造对抗和障碍</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>举个平板案例：</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>电子阅读器在我们心中属于一种很强调功能的设备，和平装书一样拥有简单外观。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可市面多数人提起平板阅读器，会想到“材质以塑料为主、外观比较朴素”；常常还带着“能停下航母一般的厚边框屏幕”，虽然结实耐用，但好像和“精致”不沾边。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如：当听到“某平台”推出一款定价3000元，以外观设计为主打的高端阅读产品“阅读器Max 时，脑中顿时浮现出好奇，走向「精装」的电纸书应该是什么样？在外观上走向精装的同时，是否也有着相匹配的内涵？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这段描述，作者在跟读者传达什么？</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>市面电子阅读器拥有简洁外观，但似乎不太精致，是不是很自然的塞给读者一个观点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是第二段表达时就与众不同，有时我们太想向读者介绍一款新产品的好处，优点，于是会直接上结论，或者把论证过程摆出来，这就恰恰走进写作的一个误区。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>没有传达体验，也没有描述出读者的质疑。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想避开这种误区就得站在读者角度，理解Ta在想什么，而非表达者视角平铺直序表达参数，也就是“商业作者</span><span>的</span><span><span>理解潜台词”。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>潜台词是“新产品我看到了，也知道参数，新在哪里？”把冲突留给读者。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时，就可以不失时机地提出自己的观点或者文章的主题，“电纸书可以做成「精装」吗？”至此，读者就已经被带入作者精心设计的思维模式中，急切得想知道新产品到底怎么样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简言之，开场做事件陈述（背景Situation），说出读者疑问，这样的东西还能卖3000块，（说反话，冲突Complication），用观点带出读者的话直接可进入主题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/475796668630c5b1fa535e4.24161627.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>制造“对抗和障碍”的方法，如何表达？记住四个关键点：</span><span><strong>1）对比，2）反差，3）转折，4）和解</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>举个“纠结”案例：</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喝咖啡，大杯34元加3元升级超大杯，你会如何选择？和别人约会早到半个小时，打一局《王者荣耀》还是看会儿书？30岁工作状态不理想，继续留在大公司还是出来创业？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>种种现象，相信你也曾经纠结过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>都说选择大于努力，应该怎么选才对？成功学会告诉你，成大事者不纠结，可如何才能不纠结？面对选择，你就是会犹豫。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，到底该如何做出果断的决策呢？我认为需要重新认识下“选择”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该方式同</span><span>样</span><span><span>也可以用在商业领域，好比“大家最近在议论SaaS产品选择的问题”，到底选择大公司还是细分赛道的小公司？</span><span>很多人提出不同看法，如。</span><span>。</span><span>。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总而言之，</span><span><strong>人本身属于矛盾体，就像打架的双方，两种方式只不过是一种设计手法</strong></span><span>，想要让读者有获得感，既要站在不同立场，还要保证足够信息量，这才是SCQA组合中冲突（Complication）的核心。</span><span><br></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>四、</strong><span><strong><strong>答案不讲对错</strong></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不妨试想下，按照SCQA组合的闭环逻辑，答案本身是什么样？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好比，得了灰指甲，一个传染俩儿，问我怎么办？马上用亮甲。直言不讳给出重点，非常符合广告宣传语逻辑，</span><span><strong>假设用在商业写作开场中，把重点落明确，可能就不是这样子。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如同：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>智能手机的普及方便大家的社交，但同时让我有些沉迷，时不时打开抖音、微博、淘宝等，不仅时间碎片化，注意力也变得更加分散。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，该如何摆脱这种困境呢？可以关掉通知提醒，专注时打开勿扰模式，或只在规定时间段内刷半个小时手机。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>把上述段落做商业写作开场，是不是就没有往下读的欲望？</span><span><strong>因此答案（Answer）在开场语境下，不能理解成“答案”，应该落在“问题和总结”两方面。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>①先说问题</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>商业写作提出问题的目的是为引出回答，而回答中所包含的各类信息将会有选择性地被传播，尽管问题的种类、形式各别。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但无论是何种提问，它总有最基本的两大特征</span><span><strong>“封闭式和开放式”，“中立性和诱导性”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开放式提问的内容极其宽泛，一般回答层面也相对自由，自己完全可以决定多少信息量，以及如何回答该问题，</span><span><strong>但由于“开放”本身，所以自己总结出来的观点有时不一定能够与读者直接共鸣。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如：缺钱的每日优鲜真的是前置仓问题吗？国货的破局之道只有往“年轻”靠吗？凭借私域流量起家的网红品牌能走多远？等，诸如此类。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>你回答得不精准，会显得业余。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>怎么办呢？我在商业写作开场过程中一般会用到“如何、什么、能不能”等关键词，这样往往牵扯到某件事的过程，顺序，能给自己更大空间阐述宏观的一些条件；当用到“怎么样、怎么办”词汇时，一般会在执行层面做引导。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>封闭式语境下，提出的问题往往给出了预设，主要为铺垫的资料更加条理化，或者澄清事实、目的、进一步缩小范围落在主题上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如：你通常会选择喝苹果汁还是橙汁？在NBA的几支队伍里，湖人、马刺、灰熊你支持哪个？久久鸭和周黑鸭哪个品牌你印象最深。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>仔细琢磨下，</span><span><strong>不论怎么选，通常这种提问下每个问题的答案只存在两种选择，这两种答案总是截然相反的，喜欢或厌恶、赞成或反对、是或否、保守或自由等。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由此，写作者作为观点的解释者，通过过程举一反三，最后始终都会就落在「总结」层。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1421730159630c5b20006151.17817848.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>②再说总结</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总结代表什么呢？有四个要素：</span><span><strong>1）比较，2）优点，3）缺点，4）紧扣主题</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如这段话：</span><span>“事实上，正如我们所熟知的，大多数调查问卷设计都采用封闭式提问，这样有利于提取答案并转换为数据、表格等形式进行进一步的分析，</span><span>然而这样真的好吗”？</span><span>从优点上来说....从缺点上来说......等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你看，是不是先总结然后帮助作者进行AB面的解析。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，总结性的预期答案往往需要好几个封闭性问题才能获得，通过封闭性提问得到的答案不能揭示某种特定态度的缘由，也不能解释“只选择某种特定答案的原因”。</span><br><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换言之，</span><span><strong>你的提问与回答不一定能够保证完全符合读者胃口。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就像上述中，你选择喝苹果汁还是橙汁？假设答案是苹果汁，你给出</span><span>的</span><span><span>原因是“小时候爱吃苹果”，这不能作为全部人接受的答案一样。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，作为文章开头时最后一句话的总结，我们最终目的是为「主题」而服务，这点一定不能偏差。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，我会选择落在</span><span><strong>“开放式的提问上”</strong></span><span>，即SCQA组合中的</span><span><strong>A答案（Answer）冲突（Complication），问题（Question）循环组合。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>想想看是不是，答案的本身不在“答案”，如同一款珍珠项链，珍珠本身并不贵，连在一起却价值连城，你不过是在转变临界点做好“穿针引线”，打开读者内心的呼唤而已。</span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>总体而</span><span>言：</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>信息爆炸时代，每个人注意力都比较稀缺。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>商业创作开场方法有很多，SCQA组合也并非最佳答案，<strong>找到与读者平等对话的方式，就是好开场。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，</span><span>心中有读者，</span><span>就是以人为本，从人出发，顾及和尊重人们的阅读心理、审美习惯和认知水平。</span></p>
