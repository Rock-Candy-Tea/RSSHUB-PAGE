<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E9%92%9F%E8%96%9B%E9%AB%98" target="_blank">钟薛高</a>再次深陷舆论泥淖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由“<a href="https://www.niaogebiji.com/tag/%E9%9B%AA%E7%B3%95%E5%88%BA%E5%AE%A2" target="_blank">雪糕刺客</a>”起头，钟薛高成了热搜的“常驻客”。先是7月2日，#钟薛高31度室温下放1小时不化 冲上热搜，引发消费者对雪糕品质的猜测与质疑，社交媒体开始兴起一阵“火烧”钟薛高潮流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之后几天内，“市监局回应钟薛高烧不化”“有点生气讨厌溢价雪糕”“人民网评钟薛高”等话题轮番登上热搜，将本就因“雪糕刺客”备受嘲讽的钟薛高推上舆论风口。7月6日，钟薛高官方针对“雪糕不融化”问题予以回应，称所有雪糕产品均按照国家标准合法合规生产，紧接着表示已经在推动线下渠道单独冰柜陈列。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/107531821762ceee961bbf60.30829346.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但在诸多话题加持下，钟薛高热度仍高居不下。7月8日，#每根钟薛高生产成本约3到8元 又被推上热搜，话题超过2.4亿次阅读。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与饱受争议的钟薛高境遇相反的，是以“雪糕护卫”出圈的雪莲冰块。在辟谣工厂卫生状况后，雪莲抖音账号粉丝量猛增至84万，“雪莲文学”也在各社交平台盛行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果不是“雪糕刺客”大展身手，很少会有人将雪莲冰块与钟薛高相提并论，毕竟二者之间，无论是价位、品质，还是发展路径都截然不同。但这个7月，雪莲与钟薛高注定站上赞誉与争议的两极。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>钟薛高的营销DNA</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自诞生伊始，钟薛高就颇具营销基因。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018年，钟薛高成立于上海，甫一开局就瞄准了中高端市场。新潮的设计、高端原料的卖点以及较高的价位，都契合了彼时的新消费浪潮，仅用16个月，钟薛高就达成了营销上亿的里程碑。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌创始人林盛曾经分享钟薛高的成名经验：“网红是成为品牌的必经之路。”<span>而在营销打法上，</span><span>钟薛高也堪称新消费品牌的模板。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从产品维度层面，差异化设计是钟薛高赢得年轻人市场的最初一步。区别于传统雪糕的“瓦片状”设计、所宣传的高端用料，以及有噱头的“钟薛高”三字，都赋予了钟薛高足够的话题度，在Z世代中打开了市场。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，令钟薛高一举出圈的，是<span>一款打出66元高价的“厄瓜多尔粉钻”雪糕。创始人林盛称，这款雪糕用料为日本柚子与天然粉色可可，成本价格在40元以上。高端用料＋限量销售，激发了想要尝鲜的年轻人的好奇心。双十一期间，15小时内，2万支雪糕被抢购一空，钟薛高的“高价”标签自此打下。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此后，钟薛高接连推出新品，其中不乏杏余年、<span>芝玫龙荔等高端雪糕，定价在66~88元不等，在发售之初以“套餐”形式售卖，上线后也很快被抢空，黄牛价甚至翻到三倍以上。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/26860625462ceee964bd275.50362707.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：钟薛高</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从营销渠道上，与新消费品牌走红路径一致，钟薛高选择的是小红书、抖音、B站等重内容社区，依靠达人种草、带货快速占据年轻人消费心智。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播电商风起后，钟薛高也走入李佳琦、薇娅的直播间。2020年，罗永浩在抖音直播首秀，钟薛高的身影也出现在交个朋友直播间，卖出了19.3万支。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>跨界联名也是钟薛高营销方式之一。早在2019年，钟薛高就曾创下一天内推出五款跨界联名新品的记录，包括与三只松鼠联名的大鱿鱼雪糕、合作泸州老窖的“断片”雪糕等。除此之外，</span><span>钟薛高还曾与娃哈哈、五芳斋、泸州老窖、小米、奈雪、盒马鲜生等品牌跨界联名，从产品、内容方面为品牌进行造势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，重营销也意味着隐患的存在。<span>2019年，钟薛高就因虚假宣传而受到行政处罚，当时多款产品被指出实际使用原料与宣传用语并不相符，例如一款酿红提雪糕宣称采用特级红提，实际上该红提干的规格等级为散装一级；声称使用日本薮北茶的一款抹茶雪糕，真正原料也并非如此。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年夏天，钟薛高还因其创始人林盛的一句“爱要不要”登上热搜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事后，钟薛高官方回应，“爱要不要”实际上是关于旗下雪糕“厄瓜多尔粉钻”原料价格的吐槽。但“40元成本，2元是制造成本，38元用于营销”的调侃依旧广为流传。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><strong>“围攻”钟薛高？</strong></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年夏天，钟薛高再次被送上了热搜。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一则“钟薛高31度室温下放1小时不化”新闻发布后，钟薛高站上热搜C位并长达一周。“钟薛高卡拉胶”“钟薛高烧不化”“有点生气讨厌溢价雪糕”等话题成为点燃消费者不满的引线，“火烤钟薛高”一时为内容平台达人的流量密码。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据此，钟薛高官方称，所有雪糕产品均按照国家标准合法合规生产，并于检测合格后出厂。同时还公布了这款雪糕成分配料以及矛盾中心卡拉胶的来源。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但消费者并不买账。在钟薛高所发布的视频中，“我们要合理价格的雪糕”刷满屏幕。微博评论区下，有人质疑，钟薛高一直以来都走的是高端路线，卡拉胶却属于低端原料；也有人认为，对于高价雪糕，消费者应该“爱买不买”，而钟薛高的问题在于放错了地方。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/63439846262ceee9679ca42.05599197.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>图源：钟薛高B站视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音、淘宝等电商平台，“卡拉胶无害，我们鞋厂粘鞋底都用它”“新型耐高温防火建筑材料”等负面言论也出现在钟薛高的直播间。小红书内，钟薛高的风评口碑也迅速翻转，“溢价”“德不配位”成为钟薛高新标签。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/88214293462ceee96d44844.57257121.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：钟薛高抖音、淘宝直播间截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事件发酵后，创始人林盛在朋友圈写道：“明明是远超国标品质的产品，被伪科学断章取义成多添加剂，有点儿讽刺。”但紧接着，“钟薛高平均毛利率约60%到70%”“每根钟薛高生产成本约3到8元”的话题又登热搜，钟薛高的成本结构又被放于台前讨论。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当钟薛高深陷舆论漩涡，另外一个坚持十三年不涨价的雪糕品牌——雪莲，却被网友作为“宝藏”一样挖掘了出来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在连夜申请账号、辟谣车间环境脏乱差后，@雪莲冰块官方账号15天内涨粉超过84万。“5毛钱的情怀，安全食品，没有塌房。”这支辟谣视频目前已获得超85万次点赞。<span>雪莲被放置于天秤的一端，作为“雪糕护卫”与“刺客”代表钟薛高对峙。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了表达对雪糕合理价格的诉求，网友们引经据典，将“雪莲文学”捧上改编的潮头：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><span>“</span></em><em><span>爱你破烂的包装，却敢堵薛高的狂，爱你和我那么像，身价都一样。《孤糕者》”</span></em><br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><span>“今日割雪莲，明日割布丁，然后得一夕安寝。起视四境，而薛高又至矣。《六糕论》”<br></span></em></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><span>“救暑之道，就在其中。《雪莲的救赎》”<br></span></em></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><span>“当昂贵成为主流，廉价便是犯罪。《雪糕的名义》”</span></em></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>在网友的自发拥趸下，据外卖平台数据，7月以来，雪莲的销量环比上月同期暴涨199%。</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><strong>雪莲未必能赢，钟薛高不一定会输</strong></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国产老品牌“意外”翻红已不是什么新鲜事，过往的例子有如鸿星尔克、蜂花等。但并不是所有的品牌都能承受起突如其来的关注和流量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>雪莲似乎就表现得措手不及。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，在开设抖音账号后，雪莲并未开通抖音小店。评论区内，不少用户建议雪莲挂出小黄车打通线上销售的渠道，雪莲则回复冷链正在谈，临时还无法从网上售卖。而据卡思观察，在淘宝等电商平台，雪莲也没有建立独立的店铺，线下销售更多集中在临沂地区，想要线上购买雪莲的用户，更多还是通过外卖平台。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很显然，雪莲并没有快速将流量反馈为销量。或许，对于</span><span>雪莲这种专注地域销售、低客单价的产品来看，</span><span>直播运营的成本过高，且冷链运输、季节限定等各类问题都在限制着雪莲“线上开花”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面则源于“真假雪莲”之争。市场上与雪莲相关的冰块鱼龙混杂，且有用户指出，原版雪莲并非那个在抖音晒出营业资质的“古椹斋”，而是已改名为“八亿牧场”的夏君乐。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>孰真孰假尚不论。目前，雪莲所打出的仅有情怀牌，内容营销并不够。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span></span>当下的雪莲想要乘风而起，必然需要新故事。用户的热情终究会消散，作为“雪糕护卫”的雪莲，其象征意义也会逐渐模糊，继而被抛之脑后。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>旷日持久的争议过后，钟薛高未必是最后的输家。据公开消息，钟薛高目前已完成三轮融资，估值在6.4亿美元左右。已占有雪糕市场一定份额的钟薛高，不会轻易被舆论影响。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位编辑朋友<span>小林则告诉卡思数据，因为好奇“雪糕刺客”为何物，而去购买尝新的消费者并不在少数，她的朋友便购买了十多支高价雪糕逐一测评。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>舆论风向也出现改变。</span>据南京市市场监管局消息，抽检的钟薛高多款热门产品都显示合格。罗永浩也在交个朋友直播间内为钟薛高发声“钟薛高的品质一定是碾压式好过哈根达斯”，尽管这番言论又引发另一轮热议，但公众人物的背书还是让钟薛高“夺回一城”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小林也注意到，最近几天，钟薛高的独立冰柜已出现在线下便利店，并有一定折扣促销，约合计8元一支。</span><span></span></p>
