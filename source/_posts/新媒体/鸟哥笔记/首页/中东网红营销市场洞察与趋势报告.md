                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">提起<a href="https://www.niaogebiji.com/tag/%E4%B8%AD%E4%B8%9C" target="_blank">中东</a>，我们脑海中会出现头巾、面纱、斋月等标签，但相较现实社会，中东互联网世界却是另一番景象，匿名基础上的平等身份，给了天性外向、爱好表现的阿拉伯年轻人充分的舞台，也让网红经济的产业链也应运而生。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，作为一个兼具文化特殊性和多样性的地区，中东地区网红营销市场现状如何？当地KOL有什么特点？出海企业做网红营销时，又有哪些文化禁忌&本地化营销建议？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本文，专业<a href="https://www.niaogebiji.com/tag/%E6%B5%B7%E5%A4%96%E7%BD%91%E7%BA%A2%E8%90%A5%E9%94%80" target="_blank">海外网红营销</a>机构Inpander为您一一解析。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>一.市场概况</strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1.<strong>社交媒体无处不在</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">早在2015年，美国西北大学卡塔尔校区进行了一项中东地区社交媒体使用情况调查，就得出了这样的论断：“<strong>在中东，社交媒体几乎已成为互联网的代名词</strong>。”调查发现，无论国籍、性别或年龄，<strong>几乎所有中东地区互联网用户都使用社交软件直接发送消息</strong>，而不是使用邮件。每天使用互联网“主要是了解朋友和家人的最新动态”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>社交软件在中东有多火？一组数据可见一斑。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总部位于阿联酋的新媒体学院2021年3月发布的《中东地区社交媒体使用年度报告》显示，中东地区用户每天花在社交媒体上的平均时间超过3个半小时，位居全球前列。超过一半的用户称，受疫情影响，他们在社交媒体上花费了更多时间。71%的受访者表示，自疫情暴发以来，他们对WhatsApp等消息传递应用程序的使用程度有所增加。75%的用户表示，由于保持社交距离，他们在Facebook（脸书）、Instagram（照片墙）、Twitter（推特）和TikTok等社交软件上的消费也有所增加。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总部位于伦敦的数据分析公司全球网络指数（Global Web Index）的研究显示，<strong>中东地区的互联网用户人均拥有8.4个社交媒体账号</strong>。据美国《福布斯》杂志网站报道，阿联酋人均拥有10.5个账号，是“<strong>全球人均社交媒体账户数量最多的国家</strong>”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2.<strong>中东网红营销的潜力</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">成熟的社交媒体市场也给打造网红营销创造了非常好的互联网式软床。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据BPG Group 和 YouGov 进行的一项调查，阿联酋和沙特阿拉伯 85% 的千禧一代至少关注一位社交媒体影响者，“娱乐”被列为这些平台上关注最多的类别。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其中，<strong>KOL是他们发现新产品的首选渠道</strong>。根据研究表明，47% 的 18-24 岁女性表示在线平台是品牌和产品信息的主要来源，73%的受访者表示，他们在KOL提及后购买了产品。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如，迪拜的美妆KOL就于 Maya 与 MAC Cosmetics、Dior、Kiko 和 Givenchy 等知名彩妆品牌建立了众多合作伙伴关系。</p>
<figure class="image" data-img-size-val="625,763"><img src="https://img.36krcdn.com/20220704/v2_e8f89ff0e24d49928061dc549e236f17_img_000" referrerpolicy="no-referrer"></figure>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3.<strong>最受欢迎的网红营销平台是？</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">Instagram在中东地区有6300万的用户，同时阿拉伯网红大概有23.4K；YouTube在中东地区的千禧一代共有60%的使用用户，YouTube上的阿拉伯网红大概有36.4K；Snapchat在中东地区主要是女性用户使用居多，在沙特就有1000万的Snapchat用户，土耳其也有大约700万的Snapchat用户，这2个国家使用Snapchat的用户居世界前10。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然，还有 TikTok。从名人到 TikTok 不断发展的影响者，TikTok 自 2019 年以来的使用量大幅增加。就粉丝数量而言，<strong>大多数KOL来自阿联酋和沙特阿拉伯，主要从事喜剧、表演和时尚内容。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4.<strong>中东网红有何不同</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中东在时尚、美容、 生活方式和汽车领域拥有影响力的人物，他们以帮助品牌扩大影响力、使他们能够经历快速和成功的增长而闻名。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过，因为宗教影响，中东网红<strong>基数少</strong>，网红<strong>环境不如其他地区开放</strong>，所以网红的<strong>价格并不低</strong>，网红的输出<strong>内容</strong>主要也是<strong>以节日文化和日常生活</strong>为主。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时，<strong>很多网红会与MCN机构签约</strong>，联系网红常会联系到本地网红MCN机构，但此类网红价格相对来说偏高，性价比还需各方面考量。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>二.值得关注的网红营销洞察</strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1.<strong>差异化、定制化将主导网红营销效果</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌正在成为中东地区用户生活方式和排他性的象征，这为本地中小企业及其他不知名企业品牌带来了巨大的竞争力。因此，想要快速推广和获取本地用户，不要只依赖产品并找到传达利益的方法，一定要进行差异化和品牌内容策略进行推广宣传</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2.<strong>利用小微影响者的信誉</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">尽管其规模不断扩大，但中东和北非地区的网红营销仍是一个新兴行业，有许多利基市场和网红有待发掘。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">凭借高参与度和真实性，尾部KOL有可能提高知名度并带来新的消费者。它们尤其适用于小额预算。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3.<strong>目标Z 世代</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中东和北非地区的Z世代人口正在迅速增长，他们更注重自我个性的展示，在浏览社交媒体时都是快速浏览，单篇内容停留时间短，因此进行KOL营销时必须清晰明确地将推广主题传达给中东用户，如果没在短时间内吸引住粉丝眼球，之后也会很难再重新提起消费者的兴趣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如在由埃及的国际商业银行举办的一场KOL营销活动中。考虑到银行业不属于 z 世代的利益范围，这种做法风险很大。因此，为了吸引年轻人，银行在动作中加入了流行的轻拍时尚。这场运动号召年轻人在新的一年里做出改变。活动以高参与度结束，几乎所有参与社交活动的用户都在询问银行服务。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4.<strong>KOL选择</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因目前中东地区的电商平台的目标用户画像主要是女性为主，所有电商平台要做网红营销时，一般都会是找女网红来合作居多，而且是分不同类型的网红来进行合作营销推广，当头部的某几位网红形成营销话题之后，腰部的网红就会带动注册使用转化，这时宣传营销大势形成后，小网红就会自发主动去推广。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样的营销推广模式其实是很好的可借鉴的模式，且是花费较合理的一种营销方式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">5.<strong>本地化要求高</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌在网红营销中面临的最大问题是对如何找到相关网红并制定符合当地法律和文化习俗的内容策略。BPG Cohn & Wolfe 影响者调查显示，55% 的品牌面临的最大挑战是寻找相关的影响者，其次是 41% 的谈判条款和条件。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，网红营销方案一定要本地化，以便能够覆盖更多目标细分用户市场。比如，营销人员在进行内容创作时可以带上中东的热门话题。以福特为例，为了庆祝了沙特阿拉伯女性驾驶禁令的解除，它制作了一个引人注目的广告来表达他们对这一决定的声援，得到了沙特阿拉伯妇女和支持者的热烈欢迎和广泛庆祝，</p>
<figure class="image" data-img-size-val="680,340"><img src="https://img.36krcdn.com/20220704/v2_793260ddb1124ce79f9e4a84d5782d7b_img_000" referrerpolicy="no-referrer"></figure>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然，如果对中东地区不够了解，也可以聘请对本地文化、语言及发展趋势的专业人士或者营销机构来制作对应的网红营销推广策略和方案。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>三.本地化要点</strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1.<strong>宗教</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中东地区的大部分民众都信奉宗教，因为地域特性和人文因素的差别，所以在出海营销时需要特别注意将禁售商品和敏感性商品考虑在内，尊重该地区民众的宗教和风俗习惯。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2.<strong>语言</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中东地区的消费者虽然也懂英语，但他们更喜欢浏览母语网站。可选择本地的一些小语种、性价比较高的KOL进行合作，以此来获得最好的线上传播效果及ROI的最大化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3.<strong>生活习惯&喜好</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中东不同国家的休息日是不同的，以沙特为例，周五和周六是沙特人每周的休息日，相当于周末，尤其周五是礼拜日。了解中东目标推广地区的生活习惯，有利于把控与红人的有效沟通时间并可以合理安排内容上线时间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中东人在颜色和数字上有自己的偏爱。颜色上，绿色最受欢迎，其次是白色、灰色、咖啡色、以及金光闪闪、珠光宝气的风格。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">数字上，中东人认为“5”会给人们带来“吉祥”，“7”是个受人崇敬的完整数字</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4.<strong>禁忌</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1）涉及当地宗教的元素</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">宗教刊物，如古兰经；清真寺、伊斯兰寺庙模型；棕榈树搭配2把交叉的剑的图案；与猪相关的商品，如猪肉、猪皮，以及猪猪图案（虽然很可爱，但小猪佩奇这个IP也不可以）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2）违背伊斯兰宗教相关的元素</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">犹太教的六角星（宗教文化冲突，这类图形是完全不被允许的）；纳粹党“卐”字标；基督教的十字架；佛教、印度教、道教、撒旦教、共济会等其他宗教相关的图案，以及象征任何宗教信仰的人物。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3）不得体的文字</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如低俗文字、宣扬同性恋文化、谈及性的文字（sexy是被明令禁止的），以及与酒、左手等相关的字眼也要特别敏感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>4）涉及赌博、恐怖、暴力的元素</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如骰子、扑克牌，骷髅头元素在任何场景中都是禁忌</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>5）裸露身体敏感部位</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如三角区、胸、臀、背</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>6）情人节相关</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在伊斯兰文化中，在未成为家人之前，异性之间不允许有亲密举止，因此带有明显“情人节”字样的元素，在中东也是不允许的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">5.<strong>中东重要营销节点</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1）斋月和开斋节</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中东电商的旺季是斋月，在斋月之前，女性们就开始在准备装饰家居、购置新衣;在斋月期间，人们的工作时间减少，加上节日庆祝气氛，人们的网购时间和消费动力大大增加。再此期间，各大线上线下商城会开展降价促销活动、。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2）“白色星期五”</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“白色星期五”源于美国的大促销日“黑色星期五”，但是，阿拉伯人一般在周五做礼拜，他们认为，周五是象征美好，幸福的一天。在中东很多国家，每周的休息日并不是周六和周日，而是周五和周六。因此，2014年，Souq在引入这个大促销日时，把“黑色星期五”改名为“白色星期五”。2016年“白色星期五”当天，Souq销售了120万件产品，是2015年的两倍多</p>                        <div style="height: 1px"></div>
