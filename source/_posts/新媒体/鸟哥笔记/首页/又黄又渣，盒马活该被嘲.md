<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之前张天爱手撕渣男，锤得全网拍手叫好。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>内娱的渣男，就像过街老鼠一样人人喊打。</span>网上还曾流传过一个</span><span>形容渣男的段子：</span><span><strong>渣男像极了螺蛳粉。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1765215989631dfca2a14498.43671001.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>内涵高手总是在民间，这两天我又看到螺蛳粉的平替：渣男像极臭榴莲。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E7%9B%92%E9%A9%AC" target="_blank">盒马</a>亲自下场，拿榴莲皮内涵渣男，本来是好事，可没想到反被网友嘲讽。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/723115795623260770c18c4.52658385.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>原来</span><span>此渣男</span><span>非彼渣男</span><span>，而是</span>盒马给榴莲雪糕打广告“同榴盒污”，花了800多块榴莲皮，制作了一座金灿灿的巨型榴莲版盒马。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>你看这</span></strong><strong><span>个榴里榴气的金刚不坏之屁股</span></strong><strong><span>，是不是又大又黄又渣</span></strong><strong><span>。</span></strong></span><span>不好意思，形容词有点不够用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/904105016631dfca2def243.56932777.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/864052372631dfca317fa89.75961330.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@盒马 官微/微博网友</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>古有木流牛马，今有榴莲盒马，简称</span></strong></span></strong><span><strong><span><strong><span>“榴马<strong><span><strong>”</strong></span></strong></span></strong></span></strong></span><strong><span><strong><span>。</span></strong></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>且不说这个设计，对于密集恐惧者不太友好，跟<span>“榴马”</span>贴贴还是需要勇气的，毕竟这脸上写着四个字：</span><span><strong>生人勿进</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1364002637631dfca3884325.36378036.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：谢钗XIECHAI个人微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让我想到了宝马的车标BMW：</span><span><strong><span>别 摸 我</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><img src="https://ngbjimg.xy599.com/654477306631dfca3dc0900.22784361.jpeg" referrerpolicy="no-referrer"></span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有<span>网络游戏《贪玩蓝月》</span>的广告：</span><span><strong>大家好，我是渣渣辉，是兄弟就来砍我</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><img src="https://ngbjimg.xy599.com/489855577631dfca41c3c28.07837805.jpeg" referrerpolicy="no-referrer"></span></strong></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：TheNewNow</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>当然叫“扎扎辉”还是草率了。正愁着没名字，盒马干脆发起一个取名的活动，入选就直接送榴莲，</span>好一个真香操作。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1915471832631dfca43d27c9.00104278.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@盒马 官微</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>征集名字活动一上线，评论区沦一度成为大型真香现场，其中呼声最高的名字，居然是</span><span><strong><span>“臭扎男”</span></strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/274810753631dfca489af27.73791139.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><span>果然</span><span>谐音梗永不缺席永不过时。</span></span></strong></span><span>一旦代入“臭扎男”的设定，就很难走出来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>都知道榴莲“脾气<span>”</span>臭，而且还有点扎人，现在用来形容扎渣男，在内涵什么我不说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不懂没关系，还有热心网友友情科普，<span><span>安排上</span>口号：</span><span><strong>远离臭扎男从我做起！</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/409074566631dfca4bb57a9.12828935.png" referrerpolicy="no-referrer"><img src="https://ngbjimg.xy599.com/1446059736631dfca4e10319.10726884.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>明明可以直接干掉我，还要让我自己笑死。</span>“臭扎男”还只是小场面，更多又香又臭的榴莲名在路上。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>臭宝、金扎、榴皮盒马、榴莲码、榴光溢彩</span></strong></span><span>.....这些名字可还行？</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>虽然装置的味儿很冲，但设计者确实花了心思。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据设计者在微博分享的幕后，这臭榴氓当真费<span>榴莲</span>，用的是<span>卖榴莲</span><span>剩下的真榴莲皮，<span>整</span><span>整</span></span>二三十箱，前后花了一个月制成。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1917400082631dfca7339444.25101420.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：谢钗XIECHAI个人微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>故事还没完，出于对设计师劳动成果的<span>尊重</span>，</span><span><strong>盒马后来还将“臭扎男”游街示众巡演：在北上广盒马门店展出<strong>榴莲盒马先生</strong>。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>我都能想象这得多大味，</span>一时之间不知道，盒马是要招揽顾客还是熏走顾客。</span><span>但真别说，有不少网友特地去打卡了。。。</span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个真敢展，一个真敢拍，我愿称之为盒马和消费者之间的双向奔赴，盒马真活该被嘲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205789392362326078647a25.53043458.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><span>看来盒马大屁股脸风评又要受损，不过</span>争议是个放之四海皆准的真理。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盒马凭借臭榴莲出圈，与其说是偶然，倒不如说盒马拿捏住流量密码：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>臭、大、金。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>味道臭。</span></strong></span><span>作为水果界的顶流，榴莲有很多过不去的梗，“又臭又香<span>”</span>的独特气味差排名第一。<span>黑粉们退避三舍，真爱粉求之不得。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再加上<span>榴莲刺硬度很高</span>，衍生了跪榴莲惩罚的说法</span><span>。盒马利用<span>强</span>话题性的水果整活儿内涵“臭渣男”，营销效果自带流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>体型大。</span></strong></span><span>众所周知盒马的IP形象，是一个极具辨识度的大脸，脸和屁股让人傻傻分不清，正因如此盒马经常被戏称为“大屁股脸”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>消费者对反常的东西更加敏感，营销<span>嗨点之一就是“大<span>”</span>，因此大型周边</span></span>会让人产生感官上的爽感。</span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这次</span>盒马标新立异，制作“巨型榴莲盒马”营造反差感，<span>突破了我们对于以往盒马形象认知，</span>带来新鲜感和惊喜感。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1109949538631dfca9c39e68.48532213.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>颜色金。</span></strong></span><span>中国人对黄金有着与生俱来的偏好，</span><span>除了金本身的价值，大家也看重“金”的美好寓意。盒马无疑用对了颜色，勾起大家对于“黄金”的联想，贡献一波带金名字，比如金屁辉煌、黄金甲等等。</span></p>
