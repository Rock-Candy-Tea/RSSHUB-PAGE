<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有芊的家里还留有一副扑克牌，牌身背面印着“热辣壹号”，正面则是李冰冰、黄晓明和任泉一众明星的写真——梦回大学时代。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这家由上述三位明星股东牵头成立的川渝火锅品牌，于2014年正式推出。有芊还记得初营业那年，和大学好友一起排队等位的忐忑：“第一次离明星这么近。”人均近百元的价格对学生党来说并不友好，但兴奋冲刷了一切。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，这也是有芊最后及唯一一次与李冰冰几人的“亲密接触”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年起，热辣壹号位于深圳、武汉、合肥等地的店面接连陷入倒闭潮，而后，包括北京、上海在内的门店也未能幸免，及至现在，行业已“查无此店”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然有无数家“热辣壹号”作为替补继续走上台前，从薛之谦的上上谦、包贝尔的辣庄，到陈赫的贤合庄、郑恺的火凤祥，明星做餐饮，尤其是火锅，犹如过江之卿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过，十年前的“有芊”却可遇不可求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时至今日，工作后的消费力提升上来了，有芊却再难有激情打卡明星店。“开店的太多了，打卡也打不过来，况且大部分都是割粉丝韭菜。”她愤愤。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>像有芊一样的人不在少数。在微博、小红书等社交平台，搜索“明星开店”相关词条，粉丝控评之外，大多消费者都在主动设防：“不看好，就是圈钱的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些年，明星跨界创业的热潮，与之相伴的是一地鸡毛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为进入门槛低，明星开店的业态大多集中于餐饮与服饰两大行业，其中餐饮占到六成以上，但以“上上谦”“贤合庄”“天然呆”为代表的品牌，因为食品安全问题和加盟商维权纠纷乱象，成为社交媒体热搜榜上的常驻客。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，风光而来，仓皇而去。明星涉足商业，正以透支未来的节奏搅动一切——市场、粉丝与消费力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14861402862c2f50463d0d2.78727284.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><span> </span></span><span><strong><span>粉丝的拳头</span></strong></span><span><span> </span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>睿意德中国商业地产研究中心，曾在其发布的《偶像的资本论，中国明星店发展研究报告》中探讨过明星开店潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>报告认为，娱乐业的市场化催生了明星跨界创业现象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如90年代前，大陆娱乐业从业者多为体制内演艺团的正式编制员工，尚属于计划经济下的一部分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>千禧年之后，改革开放进一步深化，娱乐产业迎来几乎全面的市场化，日韩偶像风潮旋即来袭，点燃了年轻人为明星爱豆消费的热情，粉丝经济成熟起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，这非国内独有，甚至有成熟范本可循。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2016年，流行音乐天后蕾哈娜<span>（Robyn Rihanna Fenty）</span>创立美妆品牌 Fenty Beauty，而后获得LVMH集团千万美元注资，次年推出品牌系列产品，上市仅40天后，销售额达1亿美元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018年，她再度联合时尚电商集团TechStyle Fashion Group创立内衣品牌Savage X Fenty，最新的消息则是，该品牌计划于今年IPO。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/21855423562c2f5048a8633.97286026.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>蕾哈娜的成功避不开大型资本集团的支持，但很难否认，这位全球唱片销量超3亿的超级巨星创业，关键还是自身的影响力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天猫国际于2019年引入Fenty Beauty，精明如品牌方——在进驻各国时都要选取一位本土代言人——国内启用了彼时的流量爱豆范丞丞。但时至今日，在小红书搜索美妆博主的试用视频，仿妆蕾哈娜始终都是真正的流量密码。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>明星跨界，本是一桩将粉丝影响力挂在脸上的生意。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近段时间，陈赫担任股东的贤合庄火锅陷入加盟商声讨风波，不少商户表示，选择单店近300万元的成本加盟贤合庄，就是看中了陈赫作为明星带来的影响力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而陈赫在切割自身与贤合庄的关系之前，也频繁在抖音等社交平台，联合玲爷等知名大V为品牌站台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>影响力有更深的甜头。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年上半年，由马东领投的奶茶品牌谢谢茶连关两店，撤离北京，上海地区也将全面迭代为智能奶茶站。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这家站在米未传媒和《奇葩说》肩膀上的奶茶店，于2019年在北京黄金商业区三里屯太古里开出首店，开业当天，马东携颜如晶、肖骁等选手为其站台，引发了长达一周的排队热潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14295168962c2f504bf7417.10685731.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>马东在接受媒体采访时谈到，该商业区从未引入过一个未经验证的新品牌，但因为自己的出面，才得以落实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换言之，</span><strong><span>明星店的网红效应，在商业地产侧的招商引进中，也早早“赢在了起跑线”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新零售商业评论曾报道过一家北京五道口大学城内商场<a href="http://mp.weixin.qq.com/s?__biz=MzUyNzkzOTQzOA==&mid=2247504741&idx=1&sn=eb58593301666bda8c03617ca2a1b718&chksm=fa757164cd02f872e33bf663b8b77f15273f2d45cb68fd49675eba04a647d41e43bcfbae26ac&scene=21#wechat_redirect" target="_blank" rel="noopener" data-itemshowtype="0" data-linktype="2">“BOM嘻番里”</a>的转型，为符合商场调性，探索年轻化，商场进行了相应的招商品牌筛选，而在试营业期间，餐饮区的贤合庄就已经作为首批店面入驻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>贤合庄、天然呆，或是谢谢茶们今时的落寞，显然是明星尝到甜头后的急速跌落时刻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国家广告研究院研究员马旗戟接受新零售商业评论采访时谈到，明星、网红自身的社会化，与其所创造的品牌、IP的社会化所带来的资产，即</span><strong><span>流量经济中聚集起来的核心消费者，属于低成本的消费者，具有很大的软肋，</span></strong><strong><span>很容易因明星网红本身形象和人设出现问题而摧毁品牌</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>典型如陈赫，在贤合庄加盟商的事件之外，更多消费者的声音集中于对“出轨男”的声讨，“因为陈赫也不会去吃”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网红化的谢谢茶，依靠综艺人气选手亲临带动起来的消费者，显然也是“低成本的”，没有黏性的结果便是，短期热闹后逐步归于宁静。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没有什么比“粉丝的拳头”更适合形容这场明星跨界经商的戏剧：粉丝聚来，拳头是有力的支撑，散去时，也能砸掉品牌的体面。商业形态的根基，就是摇摇欲坠的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/97067772662c2f504f1c813.65080991.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 身段为谁而放 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有数据表明，明星跨界创业主要集中于餐饮与服装行业，其中，超六成押宝辣味火锅。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“你不能说火锅没有门槛，但比起其他品类来说，的确相对容易。”一名餐饮从业者向新零售商业评论解释，主要因为该产业的上下游及供应链已经非常成熟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>番茄资本创始人卿永在公开演讲中也提到，火锅是中国餐饮行业中营收规模的第一大品类，2021年市场规模达到5500亿元，与火锅去厨师化、供应链难度低、受众广泛且成瘾性强相关。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做简单的生意，而非长期的生意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在马旗戟看来，相较于传统的生意思路，利用熟人圈子或产品品质服务打造品牌，当下的明星跨界是一个彻底的流量逻辑，追求的只是收入，直接的现象是，一方面，短期商业现象增多，趁着红的时候快速挣钱，另一方面，他们并不真正下场经营，而是纯消耗品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以四川至膳品牌管理有限公司为代表的明星操盘手公司应运而生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据睿意德发布的《中国明星店发展研究报告》，2017年，就已有逾300家明星店诞生，近两年，以四川至膳旗下的辣斗辣、辣叁成、斗銮火锅为代表，从火锅到烤肉，再至奶茶，明星店几乎已经成为餐饮业见怪不怪的存在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/84806691762c2f50537a309.11583428.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换句话说，明星店之间的内卷已经让消费者失去新鲜感。开不完的店，打不完的卡之后，这些店躲不过狂欢短暂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>马旗戟认为，这是因为，</span><strong><span>明星所付出的成本和代价，及其本身的IP价值，尚不足以对价它的社会性资产，消费者的接受度、宽容度和喜好度，便也无法支持长期买单行为。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一种竞争则是，行业本身的惨烈比拼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>餐饮老板内参发布的《中国餐饮大数据2021》中提到，不包括夫妻老婆店在内，截至2021年1月中旬，我国共有餐饮相关企业960.8万家。报告指出，餐饮业庞大的企业数量是任何行业都无法比拟的，绝对已经是中国所有行业里，注册企业最多的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>火锅也好，新茶饮也好，在当下的市场竞争中，已经成为一片红海里的挣扎。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像一名消费者在评价明星品牌时如是说道：“论吃，世界上没有比中国人会吃的，所以，我只会选择好吃的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卿永表示，他在2021年担心过，依靠明星流量的加盟模式是否可持续，如今有了答案。“火锅品类开店不论规模、不论速度、不论网红，产品品质与用户口碑才是王道。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>明星店显然没有满足大众的基本需求。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>社交平台上，不少用户吐槽，明星店的口味之差，服务之差让人难以接受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“寡淡如水的廉价抹茶味道，顾名思义，谁喝谁就是呆子。”一名打卡天然呆奶茶店的用户怒在小红书分享。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更主流的反抗声音是“割韭菜”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2018年<a href="https://www.niaogebiji.com/tag/%E5%A8%B1%E4%B9%90%E5%9C%88" target="_blank">娱乐圈</a>的阴阳合同、逃税漏税风波开始，而后，疫情期间，明星工作收入锐减，不少艺人开始下场直播带货，天价坑位费等信息曝光，明星与普通人的生活割裂感进一步加剧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年，郑爽为代表的负面艺人，因为私人纷争被曝光真实收入，至今还有“一天208万”“一爽6.4亿元”的热梗流传，真正的打工人们也只能拿来调侃。吴亦凡涉刑事案件，新一轮明星逃税补税事件来袭，明星的星光熠熠之下，满是污痕。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一个明显的舆论变化是，人们开始对明星有了更严格的审视。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>餐饮跨界之下，消费者越来越直观地看到，借助资本的力量和手段，明星本身的角色定位，并非如预期中所想，为躬身入局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，加盟商和消费者的反抗，都在一句“韭菜”的自我调侃里了。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 结 语 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这场接二连三的明星加盟店暴雷潮，更严重的地方在于，它以透支从业者信心为代价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，无端的资源与金钱消耗，也在反噬明星本身。陈赫道歉，关晓彤被推上热搜，越来越多迹象表明，明星比想象中要更难以抽身——或许是另一种警醒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论如何，在中国，论吃饭，消费者总是可以用脚投票，这应该是唯一的幸事了。</span></p>
