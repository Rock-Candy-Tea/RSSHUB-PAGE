<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">离开K12，新东方在线的重点转移到了直播电商。</span></p>
<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月26日，新东方在线科技控股有限公司（</span><span>01797.HK，以下简称“新东方在线”</span><span>）发布截至2022年5月31日为止的财年业绩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，失去K12的新东方在线损失惨重。2022财年新东方在线总营收为8.99亿元，同比减少约36.7%。分业务来看，大学教育成为了最大的现金牛，贡献收入5.18亿元。曾经最大的现金牛K12教育贡献收入则大幅减少，2022财年K12教育贡献收入由2021财年的7.87亿元减少至2.96亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>伴随着“双减”，新东方在线的K12教育业务画上了句号。“我们已于2021年底终止相关业务。”新东方在线在2022财年业绩报告中写到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而<strong>直播电商，成了新东方在线新的重点</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在新东方在线2022财年的业绩报告中，处处可见“电商”“<a href="https://www.niaogebiji.com/tag/%E4%B8%9C%E6%96%B9%E7%94%84%E9%80%89" target="_blank">东方甄选</a>”。在开篇，新东方在线便写到，“鉴于新电商业务的潜力，我们已将战略重点转移至发展该新分部。”并表示，“鉴于我们新型直播电商业务的成功，其于报告期内的表现不仅超出我们的预期，而且取得显著进展并收到热烈反馈……（</span><span>我们</span><span>）将直播电商业务发展为本集团的主要增长动力。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年12月28日，经历了“双减”，谋划转型的新东方在线以“东方甄选”在抖音开启了直播首秀。当天，在俞敏洪的带动下，182万人涌入东方甄选直播间，最终创造GMV（</span><span>商品交易总额</span><span>）460.4万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但第二天，没有俞敏洪坐镇的东方甄选直播间迅速陷入低落，GMV仅2万元。直到2022年6月，董宇辉充满才情的直播成为“直播圈的一股清流”，再度带红东方甄选，东方甄选才算是站稳脚跟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>千瓜数据显示，2022年3-5月，东方甄选在抖音平台累计直播销售额分别为794万元、1366万元、2488万元。而在爆火的6月，当月累计直播销售额高达7亿元，其中，6月16日和6月18日两场直播，直播销售额更是超6000万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>站稳脚跟之后，东方甄选似乎意图更进一步。8月26日，多个报道指出，<strong>近日有网友发现，东方甄选在各大应用平台上线了独立APP，有统计数据显示，目前东方甄选APP的累计总下载量已接近20万</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新东方在线发力直播电商的决心明显，但这条路走得通吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“清流式直播”“知识付费”确实为东方甄选赢得了一波粉丝。比如北京的薇薇就告诉燃财经，“6月份的时候在东方甄选直播间买过一份坚果。当时就是因为东方甄选直播间很火，好奇去看看，过程中发现坚果不错，就下单了一份。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但消费者的留存堪忧。“只买过那一次，后来就没再买过了。”薇薇表示，“因为没有在直播间购物的习惯。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“为知识付费”而在直播间买过几单牛奶和大米的广州女生颖颖，也好久没再买过直播间的商品，“无论是自营的农产品，还是上架的其他品牌的产品，都谈不上性价比最好。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而<strong>对于电商来说，流量和供应链是更严重的问题</strong>。借助6亿日活的抖音，东方甄选可以触达广泛的消费者，但回归到东方甄选本身，却不一定。另外上下游供应链的问题，东方甄选也必须解决。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“在东方甄选直播间给家人买过89元一袋的5kg装大米，但口感和自家平常买的某品牌优质米不相上下，价格却差了接近一倍，而且其他品牌的产品，往往在大促或者其他直播间也能找到低价。”颖颖直言。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>综上种种，新东方在线要依靠直播电商“两条腿”走路，甚至“独立行走”，困难重重。</span></p>

data-copy-origin="https://shimo.im"><strong>直播电商成新东方在线新重点</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>回归数据，直播电商对新东方在线的收入贡献占比并不高</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>分部来看，新东方在线2022财年的8.99亿元营收中，大学教育贡献收入5.18亿元，K-12教育及学前教育贡献收入2.98亿元，机构客户业务贡献收入0.58亿元，直播电商贡献收入0.25亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中大学教育对收入贡献的比例最大，为57.6%，其次则为于2021年底终止的K12教育及学前教育，占比33.1%，机构客户业务贡献收入占比6.5%，直播电商收入贡献占比仅2.8%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就毛利及毛利率来说，直播电商的表现也不算优异。同据新东方在线2022财年业绩数据，大学教育业务毛利为3.34亿元，毛利率高达64.5%；机构客户业务毛利0.58亿元，毛利率81.5%；而直播电商业务毛利仅0.09亿元，毛利率为37.8%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/175680617263098d7cd24577.17754191.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但值得注意的是，新东方在线的统计范围与其他不同，该2022财年业绩统计范围为截至2022年5月31日止年度。而“东方甄选”的电商平台，实则在2022年6月才一炮走红，并随即带来节节攀升的销售量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管在到达流量封顶之后，目前东方甄选直播间的热度有所下滑，但留下的千万级别粉丝，仍在直播间为东方甄选创造数亿元的月销售额。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，新东方在线将直播电商定为新的“战略重点”以及“主要增长动力”，似乎有其可行性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但8个月之前，直播电商仅仅只是新东方的求生之举</span></strong><span>。在“双减”之前，2020年、2021年，K12教育曾超过大学教育，成为新东方在线的营收主力。但监管重拳袭来后，公司开始经历转型阵痛。如今，过去曾撑起新东方在线大部分营收的K12和学前教育业务，已经不再为公司创收。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在教培机构纷纷掉头转型的大环境下，新东方在线也将战略重点转移到了新业务分部，其中便包括电商业务。如今，运作大半年的东方甄选也已步入正轨，并似乎想谋求进一步发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过抖音平台走进大众视野的东方甄选，最近大小动作不断。先是坊间流传有关“东方甄选遭抖音限流”的传闻，虽然双方很快给予了否认，但难免惹人猜疑。这两日，又有媒体报道，东方甄选APP在各大应用商城上线。两相联系，东方甄选单干的迹象似乎已经显露。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月26日，燃次元通过在不同的应用商城中均能检索找到“东方甄选”，显示下载量在几千到上十万不等。但这一APP似乎并不是近期刚上架。在小米应用商店中，燃次元发现该应用版本1.3.1的更新时间为8月8日。在评论区中还发现了，在今年6月21日，便有下载版本1.1.0的用户对此应用进行了点评。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，在新东方在线财报中，公司对于直播业务的介绍为，“<strong>本集团创办了一个新电商平台，以东方甄选为品牌，致力于成为中国优质农业及其他产品的电商平台</strong>。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，新东方似乎打算“东方甄选”APP与抖音直播双线并行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如新东方在2022财年业绩报告中就提到，“本财年，新东方在线成立名为「东方甄选」的电商平台，销售农产品及其他产品；同时亦在抖音等一些著名的短视频社交平台作试点，开展直播活动。东方甄选取得了显著进展，获得数千万订阅者和会员的广泛认可。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由此可见，对于东方甄选来说，谈不上是否要离开抖音“单干”。对于公司来说，早就把东方甄选看作是一个品牌运营，很显然，创办一个电商平台早就是列入行程表的一件事情。</span></p>

data-copy-origin="https://shimo.im"><strong>东方甄选“单干”，能行吗？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>成功率似乎并不高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先从流量来看，离开抖音，东方甄选很难获得大体量消费者</span></strong><span>。背靠6亿日活跃用户，东方甄选每场抖音直播都能稳定获得数千万的观看人数。但除抖音之外，东方甄选也在微信等不同社交媒体平台试点直播电商活动，但效果不佳。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以两个官方微信视频号“东方甄选官方号”和“东方甄选会员”为例，和抖音账号“东方甄选”一样，上述两个视频号每日也会发布数个直播切片视频作品，但视频号作品的点赞数往往只有十位数甚至是个位数，而抖音账号的作品却往往获得过万的点赞数。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，“东方甄选会员”视频号的三个置顶作品，点赞数均显示为“10万+”，这意味着，视频号内容也曾有着广泛的传播度，只不过流量并没有延续下去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>转化到商品销售上，也很明显。8月26日，燃次元在浏览东方甄选APP时发现，APP内上架的商品数量和品类都要比抖音直播间多，但除了爆款商品外，大多数商品的评价数量仅为个位数。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，同为一款名为“东方甄选长粒香米5kg”的商品，虽然界面都没有直接显示销量为多少，但通过商品评价数量能间接了解到该商品的销售情况。该商品在抖音平台的评价数量接近9000条，但在东方甄选APP，评价数量只有21条，差距明显。</span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/抖音直播间和东方甄选App的商品界面</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/燃次元截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>除了流量带来的人之外，电商的货与场，东方甄选似乎也难以满足</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前东方甄选平台上的商品除自营产品外，大部分产品为与第三方合作。尽管新东方在线提及“我们建立了优质的供应链管理体系，亦于短时段内推出自有产品。”但目前来看，与第三方合作仍是拓宽平台SKU的主要手段，在这背景下，就不得不需要面对来自供应链以及消费者体验方面的挑战。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>北京网友琪琪近来在东方甄选抖音的直播间先后购买了两次商品，对其购物体验评价均是“烂番茄”水平，甚至因为第二次的购买引发的纠纷，一纸诉状将包括抖音在内的东方甄选在及其产品方诉至北京互联网法院。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“所售酸奶商品在直播间以‘全程顺丰冷链’为噱头，实际就是中通快递配送，而且快递还放在了驿站，以至于我拿到手时酸奶已经腐败变酸。”琪琪义愤填膺。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为“为情怀买单”的消费者之一，琪琪表示，以后不会再选择该机构买东西，且取关了东方甄选，“对不起我那些年在新东方课堂上的岁月，价格贵、物流拉胯、售后差，最后一次的消费体验希望可以在法庭庭审上做个结果。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>更别提缺失的售后</span></strong><span>。东方甄选的客服永远都是机器人回复，从发生争议至今，其亦未见东方甄选方面的人出来解决纠纷，琪琪忿忿道，“虽然产品有问题找品牌方无碍，但是作为销售平台全程不露面且在直播间的承诺非有效达成，是不是企业责任方面略显过于门面了？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>新东方在线想要东方甄选成为一个“中国优质农业及其他产品的电商平台”，可以简单理解为，是一个“垂直电商平台</span></strong><span>”。而“小而美”的垂直电商生意，在早期有着蓝海一般的市场。涌现一批找准一领域便埋头苦干的垂直电商平台，时尚电商蘑菇街、美妆电商聚美优品、奢侈品电商寺库、社交电商云集……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是，随着时间推移，这些都曾冠绝领域内但体量较小的垂直电商逐渐褪去往日光环。蘑菇街大幅裁员、聚美优品已经退市、寺库命悬一线、云集股价跌近1美元。终其根源，在于电商领域马太效应显著，垂直电商的生存空间愈发遭受着头部综合电商平台的积压。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，东方甄选是幸运的，但能否将抖音直播间的成功复刻至自身的电商平台上，目前关于这个问题的答案仍是无解。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>东方甄选要成为一个独当一面的电商平台，除了流量难题以外，还需提升供应链能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与其过早思考独立，不如先借势抖音做大自己。</span></p>
