<section data-role="paragraph">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后台收到很多小伙伴私信，刚运营的抖音号，不知道怎么确定账号的风格定位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>确定账号定位首先需要明白，我们擅长做什么内容，能做什么内容，能为用户提供哪些价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>账号定位风格并不是三五天就能完全搞清楚的，想要在短时间内确定风格定位，最有效的方法就是参考优秀的对标账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对标账号既是我们的竞争对手，也是我们学习的方向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>分析对标账号，从大的方面可以分析它的定位人设、变现形式、活跃数据等；</span><span>从细节可以分析对标账号的选题方向、表现形式、拍摄剪辑、文案标题等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>分析对标账号能让我们在运营过程中少走弯路，帮助我们迅速理清运营方向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么我们应该通过哪些方法找到对标账号呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来就跟小菜一起看看吧~</span></p>
<section>
<section>
<section>
<section>
<section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" data-brushtype="text"><strong> 主动搜索</strong></h2>
</section>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们可以在抖音搜索页面，主动搜索账号所在领域的关键词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如想要做美妆类账号就可以搜索美妆，结果页面会出现一系列美妆博主账号推荐。在列表中找到适合我们风格定位的账号进行分析。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>选择的时候，要看对标账号的粉丝量，播放量，以及整个账号主页的风格定位。找到3-5个然后再具体分析每一个对标账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/160841772962bad3b734c064.95701128.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也可以运用短视频数据平台，在相关领域找到综合排名靠前的账号，选出几个进行分析。</span></p>
<section>
<section>
<section>
<section>
<section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" data-brushtype="text"><strong> 关注同行账号</strong></h2>
</section>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频行业更新换代速度非常快，内容发展趋势和视频表现形式会随着用户的需求不断发生变化。因此，我们需要关注同行账号，掌握行业内最新的营销动态，了解当下主流的视频表现形式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只要这样我们才能保证账号内容形式更新换代，跟上用户需求变化的脚步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以借鉴优秀对标账号的视频内容和表现形式，作为自己的短视频素材库内容。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但要注意的是，我们搬运视频的时候需要告知原创作者，征求作者同意。</span></p>
<section>
<section>
<section>
<section>
<section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" data-brushtype="text"><strong> 分析爆款账号</strong></h2>
</section>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>找出近期短时间内，流量较多热度较高的爆款账号。分析热门账号爆火的原因，是选题新颖还是表现形式独特，成为我们今后学习改进的方向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，分析垫底辣孩变装博主爆火的原因，是因为变装前后的巨大反差，还是因为变装人物形象选择独特。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/197581688862bad3b7c3fea5.43810564.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>分析爆款账号的独特之处，然后及时吸收模仿相关特点。跟风爆款账号输出内容，越早模仿的视频越容易获得意想不到的曝光量。</span></p>
<section>
<section>
<section>
<section>
<section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" data-brushtype="text"><strong> 了解相关账号</strong></h2>
</section>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当我们关注一个合适的对标账号之后，账号下面会有一些相关推荐账号。系统推荐的这些账户，跟我们所找的对标账号流量热度以及定位领域大致相似。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们可以参考看看，寻找更多的对标账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/211097579362bad3b8194b97.69203627.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音号起号前期，模仿对标账号能让账号迅速达到起步期。但我们在寻找模仿对标账号的时候，要特别注意参考模仿不是原封不动的照抄照搬。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们只是学习对标账号的优点，根据它的优点再融入自己账号的特色内容，只有这样才能打造与众不同的账号人设风格。</span></p>
