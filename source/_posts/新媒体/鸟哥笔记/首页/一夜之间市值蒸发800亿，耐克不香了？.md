<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E8%80%90%E5%85%8B" target="_blank">耐克</a>又上热搜了，这一次是因为公司<a href="https://www.niaogebiji.com/tag/%E5%B8%82%E5%80%BC" target="_blank">市值</a>一夜蒸发800多亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/190371746662c36576095ff2.16372547.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图源：微博热搜</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截止到6月28日美股收盘，耐克大跌接近7%，市值一夜之间增发120亿美元，折合人民币超过800亿，创下近两年以来的股价新低。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以市值大幅蒸发，显然和耐克所公布的2022财年第四季度及全年财报有关。根据财报显示，耐克在2022财年全年营收467亿，同比增长5%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就数据而言表现还算说得过去，</span><span><strong>但仔细看就会发现耐克其实已经连续三个季度在中国市场的营收出现下滑。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这之中除了去年的“新疆棉花事件”之外，事实上，如今的耐克已经变得不再那么吃香了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/381440126233b25ace1df6.64458916.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>耐克不香了</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在耐克发布的财报当中，最值得关注的就是耐克在中国市场的表现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据财报显示，耐克已经连续第三个季度失守中国市场。其中第四财季耐克在大中华区的业绩加速下滑，营收15.61亿美元，同比下降高达19%。不仅收入下滑，2021全年耐克在大中华区的净利也下降了27%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/139583433162c36576484c25.20630119.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图源：耐克财报</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>算上美洲、欧洲、非洲等多个地区，大中华区是唯一一个收入和净利均出现负增长的市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，据了解，耐克全资控股的耐克体育和耐克商业有限公司在中国大陆注册的328家分支机构当中，目前已经累计注销了107家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一时之间，耐克在中国好像不香了。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此有不少网友表示：“自从耐克的鞋子在越南生产后，质量就变得越来越差，国内随便一个品牌鞋子都比他强。”还有人说：“从去年新疆棉花事件后，自己就再也没有买过耐克的产品。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/51760109862c36576a6adc0.18417303.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图源：Pexels</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便如此，耐克CEO仍然乐观的认为：“我们长期看好中国市场和未来的潜力，这一点从未动摇。未来将会持续加大在中国的投入，并通过中国技术中心加强公司的数字化业务能力。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>事实上，耐克并不是唯一一个被中国消费者所“抛弃”的国际运动品牌。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一运动品牌巨头阿迪达斯，在中国市场的表现也不尽如人意。根据过往发布的财报显示，2022年第一季度，阿迪达斯大中华区的销售额同比下降35%，出现了断崖式的“滑坡”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据耐克方面的分析，大中华区业绩下滑主要是受到疫情所导致的库存、物流等成本增加的影响，疫情对业务的覆盖范围超过60%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但事实真的是这样吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9225241376233b25c6ec6a4.50995726.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>昔日运动品牌大哥，</span></strong></span><span><strong><span>为何沦落至此？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上世纪80年代，耐克正式进入中国市场。然而彼时的耐克并不像可口可乐那样迅速走红，反而经历了很长时间的沉默期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了进一步打动中国消费者，在上世纪90年代，耐克斥资250万美元邀请“篮球之神”乔丹代言。借助乔丹的明星效应，当年耐克销售额就突破7000万，并超越自己的老对手阿迪达斯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/67269787862c36577b51b15.24616670.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图源：耐克官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在当时很多人的眼中，耐克和篮球鞋可以完全画上等号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此后，耐克经历了一波高速发展期，并在2003年拿下体育用品类市占率第1，此后长期霸占榜首位置。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直到2020年，</span><span><strong>耐克在中国的市占率依然高达26.5%，并牢牢把控着中高端市场，阿迪达斯以17.4%屈居第二，本土品牌如安踏和李宁更是难以望其项背。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而这一切，从去年新疆棉花事件后开始出现明显转折。去年的公共事件，可以说是给耐克好好上了一课。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>多年来耐克享受到了中国市场发展所带来的福利，但</span><span><strong>却始终都没有学会尊重中国市场。随着国内消费者抵制情绪的不断高涨，最终也导致耐克连续三个季度失守。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/45222157662c365783fa846.03371876.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图源：耐克官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，品质也是耐克一直以来都未解决的问题，甚至被不少网友调侃“穿一穿就坏了，那肯定是正品”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这种契机下，国货品牌乘胜追击，进一步蚕食耐克原本牢牢把持的市场份额。尤其是国产品牌研发能力的不断加强和款式创新，屡屡在电商购物节当中销量排名前列。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再加上“国潮风”的兴起，如今年轻人也不再盲目迷恋国际品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>种种因素，最终导致耐克在中国市场的表现节节败退。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16900100706233b25e365e39.91683873.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>当代年轻人，爱国货不爱耐克</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不可否认的是，曾经80后、90后一直都非常喜欢耐克、阿迪达斯等国际品牌，甚至以穿国际品牌为潮流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过随着近年来国产运动品牌的崛起，不管是研发能力还是产品创新又或者是款式和价格，各个方面都已经具备了和国际品牌一较高下的能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更重要的是，如今的年轻一代消费群体已经变得越来越理智，消费观念也产生了明显的改变，盲目追求国际大牌的时代早已一去不复返。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一边是国际巨头们遭遇“滑铁卢”，另一边国产品牌强势崛起，在这之中最具有代表性的就是安踏和李宁。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/177500957262c3657991a7a4.50162613.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图源：李宁官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据财报显示，李宁2021全年营收225.7亿人民币，同比增长高达56%；安踏全年营收493亿，同比增长近40%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然变化不仅仅只是体现在营收方面，市场份额上耐克和阿迪达斯在中国的市场占有率也滑落到了40%，此前一直都能够保持在43%以上。在这之中安踏的市场占有率进一步提升到了16.2%，完成了对于阿迪达斯的超越，上升到第2位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3508377362c3657b751bf6.65141012.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图源：安踏官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，有业内人士表示：“国产运动品牌正在努力打入年轻一代，包括新的系列以及请一些有影响力的代言人。如今花大几百元买一双国产品牌的鞋子，很多人也完全可以接受。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对逐渐丢失的市场份额，耐克、阿迪达斯们显然不会轻易放弃。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在前段时间，耐克宣布自有数字平台转型将在7月正式启动，届时会上线多个小程序和APP；阿迪达斯也一直都在致力于营销方面的加强及数字化转型。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是对于一个品牌来说，在尊重本土消费者的同时，产品力才是关键。如果脑袋里面装的满是“割韭菜的镰刀”，那迟早会遭到反噬。</span></p>
