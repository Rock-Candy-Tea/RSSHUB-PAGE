<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在我们的“见识突围课”的班群里，有一群资深的、活跃的<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E8%90%A5%E9%94%80" target="_blank">品牌营销</a>人，他们总是鼓励我多输出，多点评，并且“自告奋勇”的与我一起共创内容。面对此情此景，我再也不好惫懒于输出，躺平于运营了。于是有了“案例分析”这个系列的内容和栏目。今天是尝试运营的第一篇。</span></p>
</section>
<section>
<section>
<p><strong>01、</strong><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">案例背景</strong></p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小张的父母在山西小县城开了三家卖鞋子的实体连锁店，在所谓的“下沉市场”。实体店面、下沉市场、鞋履品类，这些关键词似乎与当下最热门的那些炫酷概念毫不相干，那我们该如何去理解这门儿生意呢？这个生意有做大的机会吗？这样的店有机会打造自己的品牌吗？</p>
<section>
<section>
<p><strong>02、</strong><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">案例分析</strong></p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于这样一个生意来说，我的观点是中国的下沉市场里，基于本地消费生态的实体“买手”店渠道仍然存在着巨大的机会，尤其是对于鞋子这种特殊的品类来说，甚至存在着平台级的机会。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1、县城买手店的秘诀</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在18线县城里积累了20年经验的老板们，他们的核心能力是什么？他的产品是什么？是“鞋子”吗？不是，他不生产鞋子，甚至不参与设计研发，他只是从那些并不是很知名的工厂里“选货进货”。但是，他有一个核心杀手锏：他<span>懂得小县城的人们，愿意花多少钱、买一个什么样儿的审美的鞋子。他和那个小县城的人，穿过两条街就抬头不见低头见。即使在所谓互联网深水期，这个能力依然很重要。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首先，鞋子是一个非常特殊的品类，它不是标准化审美、标准化输出、标准化供应链的产品。虽然现在也可以在互联网上实现标准化售卖，但鞋子与眼镜、珠宝等很多特殊品类一样，还存在比较大的线下机会。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">像中国的很多县城和县级市、乡镇。比如山西介休、山东茌平、东阿这样的县城里边儿，大量存在着什么现象呢？就是人们在购买像鞋子、衣服的时候，是会信赖当地的夫妻小店，像这样的小店，深知人们的审美和购买习惯。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种店，我们也给他一个非常洋气的名字，叫做“买手店”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">买手店大家知道，就是一二线城市里，根据对大品牌的严选去做一些产品推荐的集合店，那么在下沉市场里就没有买手店吗？相反，我认为下沉市场的买手渠道依然是非常好的，因为在下沉市场里，靠着这种人际之间的关系，互相熟悉的一两条商业街上的地方人情，大家其实更愿意相信那个推荐给他们一双鞋子的人，他们对于品牌的关注，将从鞋子本身的品牌转移到这个小的渠道品牌上来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此呢，<strong><span>在中国的广大的下沉市场里边儿，这样做买手店或者品牌严选的线下店是有机会的。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2、渠道与供应链的信息差</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另一端我们看到的是什么呢？<strong><span>在中国，强大的供应链快速发展，柔性供应链的发展已经成为了趋势。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在中国南部，比如说福建，广东，在鞋子供应链非常丰富的省份，<span>有一些已经具备了强大的生产和供应能力的鞋类企业，已经完全具备了生产高质量的鞋子的能力。他们过去主要面对的是国际市场，以及顶级的品牌对生产的需求。现在开始面对中国市场，尤其是具体的消费者的诉求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span>这里边儿其实有一个很重要的信息差，就是中国的具体市场和下沉市场的渠道店买手店和这些强大的供应链生产商之间的信息差。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果我们能够耐心的到这样的市场和渠道里去待一会儿的话，会发现其实他们的销售过程、销售经验是不可复制的，或者说很少会被线上的电商绝对替代的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，我认为这样的渠道还是有机会的，在这种线下销售场域，对小城市人群的审美把握，其实他们卖的东西并不是最便宜的，也并不一定是最好看、或者品质最好的，但是他是最了解那一方水土一方人的。未来，他们的产品的“消费升级”和“严选”也迫在眉睫。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当我们去分析一个案例的时候，要舍身处地的深入到案例本身所在的场景，以及真正的实地进去感受。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中国这么大的一个市场，很多概念一概而论，希望能够一刀切，是非常不理性的。中国的市场大、场域也多，机会其实是非常多的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在下沉市场里，要真的耐心研究，才会产生机会。像蜜雪冰城、足力健老人鞋这样的品牌能够产生，其实就是来源于他们对所在场域的了解足够深入。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3、忌贪大，先验证市场机会</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外一个疑问是什么呢，就是像这样的店能做多大？其实我觉得在分析案例的时候，先把这个大放在一边儿，先把能够产生持续的净利润放在第一位，因为太多的企业都是因为从一开始贪图大，而放弃了对真正的消费和经营的真相的探索。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其实在这个案例，只要三家店，哪怕是两家店，实现了真正的净利润，并且是持续的净利润，这是第一步。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">然后我们再去看，从下沉市场的严选里能不能找到这样的生意模式，能不能把下沉市场的这个渠道统一为一个好的渠道，这里边是有一个机会在的。我个人觉得这个机会的出现，会是这一轮儿的带货热潮膨胀到一定程度之后，在下沉市场实体场景里会出现一轮儿新的渠道机会。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">尤其是像一些特殊品类，像鞋子、眼镜等可能需要部分定制化或者是特殊审美的品类里，一定会产生这种严选的，或者是地域性的渠道品牌。这些店或者采用连锁模式，或者采用统一分发模式，或者采用品牌联名方式，都是会有比较大的一个机会。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中国的所谓下沉市场的用户，在购买行为上跟一线城市是有截然不同的特点，这一点，我们在做市场调研和市场分析的时候一定要看清楚。</p>
<section>
<section>
<p><strong>03、</strong><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">案例启发</strong></p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这个案例里，除了这个洞察外，还有两个值得注意的地方：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一是供应链和细分市场需求的强关联，打破信息差，让下沉市场的人们连接到强大的供应链，让有能力的供应链高效链接细分渠道合作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">二是对特殊品类的认知也是这个案例的关键，不是所有的品类都能按照一刀切的方式去分析，不同的品类有不同的分析方法。</p>
<section>
<section>
<p><strong>04、</strong><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">补充案例</strong></p>
</section>
</section>
<section>
<section>
<section>
<section>
<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我有个亲戚在贵州的一个小城市开了一家衣服店，主要针对年轻的女性群体，价格区间在几十到三百左右。当时开店，我真的觉得谁还会去实体店买这么便宜的衣服，便宜衣服网购不香吗，心想这个生意肯定做不成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没想到还不到半年，小店就回收了全部投入还盈利了十多万，她趁热打铁又去另外一个小城市开了第二家店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她是怎么能够去到陌生的小城市开店还觉得能盈利的呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实是因为她加盟了一个公司，那个公司有自己的供应链，有开店和返点政策，款式和价格全是针对下沉市场。她觉得就算投入失败，也不会亏太多，这是她有勇气的一个原因之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大方向的选品和定价别人都给她做好了，她自己对于这个比较陌生的板块就相对不用付出太多精力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二点，她之前开过母婴店，让她摸到了下沉市场的购买力，对自己的销售能力是有信心的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她开了这个店之后，也玩“私域流量”，把顾客加到群里，每天都在发朋友圈发群，新到了啥款式，反复刺激老顾客复购。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，卖不完的衣服，还可以退给公司，这点给了她每个季度狂进货的勇气。进货多，款式多，单价又不贵，消费者一去店里逛，就容易买得多，一件T恤几十块，稍不注意就买了几百块。</span></p>
