<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据每经新闻、界面新闻、蓝鲸财经等多家媒体报道，隶属于欧莱雅集团的彩妆品牌<a href="https://www.niaogebiji.com/tag/%E7%BE%8E%E5%AE%9D%E8%8E%B2" target="_blank">美宝莲</a>正逐步关闭全国线下门店，仅保留在美妆集合店屈臣氏内的柜台。</span><span>另据公开资料，美宝莲已经连续多年撤离线下销售渠道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美宝莲并非个例，近年来越来越多的外资彩妆品牌密集关店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先是韩流退潮。爱茉莉太平洋集团旗下韩妆品牌的悦诗风吟，在2019-2021年相继闭店40家、90家、170家；另，爱茉莉太平洋旗下另一个平价韩妆品牌伊蒂之屋，在2021年上半年关闭了所有中国实体店；今年2月，有小红书博主透露而后确认，爱茉莉太平洋旗下高端品牌HERA（赫妍）、IOPE（艾诺碧）不仅在中国撤柜，还被最大分销商丝芙兰踢出局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次是欧美、日本彩妆大牌也在相继退出中国市场。去年4月，奢侈品巨头LVMH集团旗下彩妆品牌贝玲妃传出撤柜消息，只保留丝芙兰和天猫旗舰店的渠道；去年年底，日本彩妆市场上常年销量第一的开架彩妆品牌kate传出中国撤柜清仓的消息，只保留屈臣氏渠道……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在外资彩妆品牌密集关店、撤离中国市场的背后，究竟反映着怎样的消费变迁？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/12814919162e7ed743519b7.48226037.jpeg" referrerpolicy="no-referrer"></span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>一. 昔日大众彩妆第一品牌美宝莲</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美宝莲1995年登陆中国，1996年被欧莱雅收入囊中、而后被划归到后者的大众美妆部，是最早在国内拓展线下销售网点的彩妆品牌之一，也是启蒙一代中国人彩妆消费的重要品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>中国最早的一批美宝莲代理商以及百货公司的经销商，曾一致认为是美宝莲真正打开了中国彩妆市场的大门。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>界面新闻报道，2000-2011年美宝莲的国内线下销售网点由3500多拓展至5000余个。而据当时一份美国机构的调研数据，2011年美宝莲的在华市场份额就达到了15.71%，品牌的用户忠诚度达75.31%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在千禧年初，中国女性消费者只有在出席重要场合时才有化妆需求，十余年过去后，消费者的彩妆意识已经非常成熟——彼时英敏特出具报告显示，有84%的受访女性用过唇彩，超8成用过底妆，超75%用过眼妆……越来越多的女性将彩妆的使用场景迁移至约会、派对、商务、面试等非常多元的生活场景之中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这十余年间，美宝莲究竟做对了什么，才使其风靡中国大地？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/143707234362e7ed747e3b12.45372324.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>曾任美宝莲纽约品牌总经理曾在接受采访时表示，“时尚、创新、可接近”是美宝莲的“品牌金三角”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就市场表现而言，美宝莲确实做到了：</span></p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在时尚引领方面，早在进入国内市场前，美宝莲旗下推出的梦幻系列哑光慕斯粉底液、Great Lash睫毛膏等多款产品早已成为全球畅销的爆款，其对时尚风潮的引流能力不言而喻；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在创新力度方面，美宝莲每年的换手率在40%-50%之间，研发投入占到总销售额3.3%以上（属于业内较高水平），例如唇妆方面美宝莲曾保持每年推出新色调的节奏；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在可接近性上，更可触达的渠道和货架、更低的产品门槛以及更耐心的用户教育，为美宝莲打下了坚实而广阔的用户基础；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在渠道和货架方面，2011年美宝莲的线下销售网点已达到5000多个，且在2014年左右成长为为数不多覆盖全渠道的彩妆品牌，当时的全渠道指的是百货、商超、CS、电商平台，而当时的美宝莲甚至可以下沉到五线城市，且作为最早开架售卖的品牌让消费者眼前一亮；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在产品门槛方面，美宝莲有不输其他品牌的质量，定价却亲民，例如汲取纽约T台灵感的“色秀指甲油”；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在用户教育方面，美宝莲官网上拥有丰富的彩妆教学视频，用户可自行查找适合自己妆容需求的教学；在线下，最具中国特色的CS渠道被美宝莲相当重视，柜台体验非常专业；而在看不见的地方，美宝莲每年会调研上万名中国消费者，研究其平时的彩妆使用习惯，为此实现针对性的产品研发，同时，美宝莲也是最早将研发中心放到中国的欧美彩妆品牌。</span></p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，在2015年左右，电商平台以及本土国货彩妆品牌的崛起，迸发出比美宝莲预想中还要强大的力量，有数据显示，2015-2019年，美宝莲尽管仍保持着市占率第一，其市场份额却从30.9%跌至23.3%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美宝莲的全渠道布局能力、用户沟通能力，在时代变迁中渐渐褪色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213441573962e7ed74c80c12.23282355.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>二. 外资彩妆品牌大撤退</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为昔日大众彩妆第一品牌，美宝莲自2015年起出现市占率下滑，近期传出撤柜消息，亦代表着一个时代的结束。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如开篇所述，近年传出撤柜消息的远不止美宝莲一家<a href="https://www.niaogebiji.com/tag/%E5%A4%96%E8%B5%84%E5%93%81%E7%89%8C" target="_blank">外资品牌</a>，在彼时国人彩妆消费起步的年代，这些外资品牌扮演了重要的启蒙角色，而随着市场和行业逐渐成熟，越来越多的国货彩妆品牌走上台前，<strong>他们纷纷喊出“大牌平替”的口号，用更低的价格和不输大牌的质量，扛起新一代消费者的彩妆想象：完美日记、花西子、珂拉琪……</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在新老势力的交替中，我们可以看到三个关键变量在起作用：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1. 彩妆是典型的可选消费，消费决策的理由中有很大比例来自非理性因素</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如明星代言引发的跟风、因颜值带来的种草经济、因悦己而提升生活品质的消费……因此，外资品牌昔日取得的地位建立在社会心理、社会文化、社会潮流基础之上；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>爱茉莉太平洋集团旗下多款个韩妆品牌相继在中国撤柜，所反映的就是“韩流文化退潮、韩妆不再流行”这一事实。2015年左右，《来自星星的你》等多部韩剧在国内走红，当时国内不少代购、微商渠道卖的就是剧中同款彩妆产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2. 彩妆是需要一定学习成本的品类，即非常强调专业人士教授妆容经验和技巧</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在电商尚未萌芽的年代里，百货、商超、CS成为彩妆品牌与消费者沟通的重要渠道，谁的渠道铺得更深、更广，更容易赢得市场；不过，2019年后直播电商渐成风口，李佳琦之流取代柜姐，在社交平台上有非常活跃的达人分享妆容技巧，这让线下优势不够明显。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>彩妆品牌在百货、商超的撤柜只是时间问题，线下客流永远流向最新、最潮的零售业态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/108069860162e7ed752dd798.79297457.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>3. 国潮文化崛起，彩妆供应链成熟，行业门槛更低、营销更卷</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国潮文化是与综合国力绑定的，2020年我国GDP迈过百万亿大关，一个又一个的国际盛会在国内举办，李子柒等现象级文化事件屡次出圈，成长于优渥环境的90后、00后为国潮市场贡献了近一半的消费力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在供应链方面，国内头部的几家彩妆代工厂：韩国科丝美诗、韩国科玛、意大利莹特丽不仅服务于外资品牌，也服务于国产品牌。例如科丝美诗服务的外资品牌包括雅诗兰黛、兰蔻、迪奥，国货品牌包括完美日记、花西子、珂拉琪等。消费者之所以发现完美日记等彩妆品牌不输海外大牌质量，本质在于其代工厂同源一家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/91693188762e7ed7579ba74.08533033.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>总之，在国民信心空前大涨的背景下，外资品牌祛魅，年轻消费者有了越来越多的彩妆消费选择，且直播电商、种草经济等新链路里就可以完成消费，这也让新品牌有了全新的触点与消费者近距离沟通。</span></strong><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>三. 硝烟未止：高端心智之争</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国货彩妆品牌在与上述代工厂取得合作后，加速了产品迭代，AI财经社曾报道，完美日记昔日推出的小狗眼影盘仅用了2个月的迭代时间，这其实有利也有弊。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>代工模式的好处在于，响应速度更快了，品牌能够在最短的时间迎合市场潮流，夺得消费者注意；但这也导致，品牌似乎在内功上修炼不足，在吸引流量上越走越远，困于营销，让销售成本水涨船高，自研产品有心无力。</span></strong><span>例如，完美日记股价跌到不足1美元，本质在于其常年高达60%的营销费用难以维持。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，作为欧莱雅大众彩妆部的美宝莲虽然撤柜了，欧莱雅旗下另一高端线品牌却在进一步加码中国市场：欧莱雅集团旗下奢侈美容品牌Carita已官宣在南京德基广场、北京SKP开设精品店和专柜。欧莱雅去年财报显示，美宝莲所在的大众消费品部门收入仅增长4.5%，但兰蔻等奢侈产品部分却大涨21.3%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要知道，今年上半年疫情以来中国奢侈品市场依然坚挺，这让不少奢侈品巨头再次瞄准了这个贫富分化愈加严重的东亚市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在实现“大牌平替”后，国货彩妆品牌又是否能抢占高端彩妆市场的消费心智呢？</span></p>
