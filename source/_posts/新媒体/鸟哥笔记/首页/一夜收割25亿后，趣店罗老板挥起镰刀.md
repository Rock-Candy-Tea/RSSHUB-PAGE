<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><span>一分钱的酸菜鱼随便抢，1500台iPhone 13免费送，几千万的抖音投流疯狂砸。</span><span><strong>当资本营销强势入场，所有的带货技巧都一文不值。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在罗永浩时代落幕之后，另一位做校园贷起家的“罗大师”在抖音掀起新的浪潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>一分钱预制菜撬动2.5亿销售额</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这两天，或许许多朋友都已经被<a href="https://www.niaogebiji.com/tag/%E8%B6%A3%E5%BA%97" target="_blank">趣店</a>直播间刷屏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月17日，抖音“趣店罗老板”直播间推出“717吃货狂欢节”。不同于其他各种平台大促中的“满200-30”、“满500-50”等折扣，趣店罗老板还是很舍得花钱的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>一分钱的酸菜鱼出手便是10万单，iPhone 13手机上千台完全免费送。</strong></span><span>用罗老板的话来讲，“今天就是打个口碑，交个朋友，希望大家能认可我们，就够了！”这番言辞颇有罗永浩当年入局直播带货的风范。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/54971832862d88d0b1a9d75.69523102.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：趣店罗老板抖音号</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，在商品上大放福利的同时，营销方面罗敏也没放下。不仅请来了明星贾乃亮共同带货，还在微博平台购买了诸如“趣店罗老板豪送iPhone”、“罗老板直播一天亏多少钱”、“贾乃亮连赞罗老板大气”等多条热搜。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/38912573362d88d0c252289.23135043.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在直播间里，背景打着“上市公司CEO正在直播”的标签，所售产品皆为趣店自营预制菜。罗敏边试吃，边讲解，直播间场面之火热，仿佛不抢一单便巨亏百万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最终，</span><span><strong>在长达19个小时的马拉松式直播下，罗敏拿下了2.51亿元的累计销售额，斩获带货榜TOP 1。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，其他数据也都创下历史新高。单场直播累计观看人次超9000万，直播在线人数峰值超91万，累计销量858万单，累计涨粉458万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/117932641062d88d0cc1a643.04560679.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：新抖数据</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>至此一役，趣店罗老板一战成名。也让人不得不感叹，有钱真的是可以为所欲为。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，趣店罗敏搞直播，已经有一个月了。在6月份刚入场时，走的便是这种补贴送福利玩法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在6月15号的首播中，便以“CEO首播，全场送浮力（福利）100万份”为主题在直播间卖起预制菜。9款菜品中，有部分首单价仅为9.9元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经历一个月的摸索、试水之后，熟悉了抖音规则的罗敏才终于为我们呈现了这场豪掷上亿的“顶峰之战”。</span><span><strong>至此，其“上市公司CEO+二次创业预制菜”的人设也深入人心。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，许多人也产生了质疑，这样砸钱卖货真的不会血亏吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>直播营销，一箭三雕</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>罗老板看到上面的问题，恐怕笑都能笑醒。要知道，趣店豪掷上亿元，可不是为了简单请你吃顿饭。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实明眼人都能看出，再高的销售额也难抵此次直播的总投入。哪怕不算各种一分钱的预制菜和1500台iPhone 13，</span><span><strong>仅仅是在抖音平台的付费投流广告费用，网传罗老板都花了一个亿！</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再加上iPhone福利、一分钱预制菜以及其他营销投入，</span><span><strong>这一场直播，趣店或将巨亏上亿元。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那为何在这种逻辑都走不通的模式下，罗敏还要砸钱上亿搞一场直播？我们来看看直播的后续，或许就知道答案了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15859735962d88d0d1104b3.20180238.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：雪球</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月18日，也是罗老板这场旷世直播结束后的第一天，趣店股价盘前暴涨86%，盘中一度从前一日收盘的 1.19 美元翻倍至 2.43 美元，凭空增长市值约3亿美元。而后股价尽管有所回落，但较此前仍有大幅上涨。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要知道，就在今年上半年的2月和5月，趣店还由于交易价格长期低于1美元，而连续两次收到纽交所的退市警告。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>转眼间，</span><span><strong>一场直播便将趣店从退市边缘拉了回来，成为资本眼中的香饽饽。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，此次直播所带来的广告效应、400万精准粉丝、以及800万用户的预制菜标签，都成为趣店手里一笔巨大的无形资产。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看到这里，你还觉得罗老板亏了吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而这还不算完，在二级市场之外，罗敏又有了新动作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样是在18日，趣店直播间被外界讨论的最热烈之时，趣店预制菜在厦门正式召开了品牌战略发布会。会上，罗敏表示，要和自己的用户和粉丝一起创业，未来三年支持十万用户创业开设线下门店。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/150474508262d88d0d62c872.76726170.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：趣店罗老板抖音号</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当罗敏的故事讲到这里，已经让人嗅到一丝熟悉的气息。有网友就趣店预制菜拓展各城市代理的打法进行了总结：</span><span><strong>“现在花钱铺渠道，拉升股价，中期铺代理，收割代理费，后期就是去盘了”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>多么熟悉的套路，校园贷起家的趣店，玩的不过还是一场披着预制菜外皮的金融数字游戏罢了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>趣店罗敏的预制菜故事，不仅在讲给资本市场听，也在讲给起高举镰刀下的韭菜。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>趣店能成为下一个东方甄选吗？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于预制菜是否是门好生意，罗老板是否又挥舞起镰刀暂且不提，让我们回归这场事件的出发地——直播带货。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个月前，东方甄选的爆火出圈让直播带货又为了热乎的新风口。受益于此，新东方在线股价也得以直线拉升。如今，同样的情形再次出现在趣店身上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是不同的是，东方甄选是用内容给直播电商上了一课，而罗老板，用的则是钞票。那么趣店罗老板，会成为下一个东方甄选吗？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/93602864262d88d0e137320.51158572.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：东方甄选抖音号</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，二者的逻辑并不相通。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>对于东方甄选而言，本身便是有着非常强势的内容输出，这也是他们得以持续爆火的长效模式之根本。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在爆火之前，东方甄选则经历了长达半年的蛰伏期。谈及过往，董宇辉曾表示，“很长一段时间里，直播间就五个人，两个是我爸妈，一个是老板，还有两个是工作人员。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所幸，董宇辉坚持了下来，东方甄选也坚持了下来。从三尺讲台到一方直播间，曾经的老师们完成了身份的转变，但知晓天文地理、口若悬河的特点没有变。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>踩着算法飞轮的东方甄选，一跃成为抖音顶流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反观趣店罗老板，从7月初便开始为此次直播做宣传，在营销上，更有明星助阵、热搜安排，上亿广告费只为一场直播的爆火。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/156325054262d88d0edaeba1.37880767.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：趣店罗老板抖音号</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>用各种不可持续手段所堆出来的刹那烟火，又怎能与以内容为本长久璀璨的明月进行“可持续发展”的比较？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换言之，离开了资本，趣店直播间并无造血能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，或许罗敏的意图本就不在此处。</span><span><strong>一场炒作，乱世里再狂割一把韭菜，准备过冬罢了。</strong></span></p>
