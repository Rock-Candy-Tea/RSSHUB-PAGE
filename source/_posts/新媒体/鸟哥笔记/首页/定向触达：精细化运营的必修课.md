<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">定向<a href="https://www.niaogebiji.com/tag/%E8%A7%A6%E8%BE%BE" target="_blank">触达</a>是什么？</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所谓定向触达，就是通过某种方式，让指定目标对象接收到信息的过程。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">定向触达并非现代才有的概念，从古至今，我们身边就有大量的定向触达案例：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">古代，人们用飞鸽传递书信，是一种定向触达</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">近代，情报人员用电报告知敌情，是一种定向触达</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现代，老婆打电话给老公，晚上10点前不回家就跪搓板，也是一种定向触达</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">为什么要定向触达呢？</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里就要分别解答为什么要“触达”，以及为什么要“定向”2个问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首先，人的行为不会无缘无故自发产生。人们总是要先接收到外界信息，理解消化后，才能做出对应的动作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如我们不会突然想要购买某品牌的咖啡，通常是接受到该咖啡的广告或者推荐后，我们才会有后续的动作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现代商业运作也是如此。只有当消费者接收并充分理解商品、服务的相关信息后，才有后续的消费可能性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">换句话说，触达是一切商业行为成立的基础。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再让我们看看为什么要“定向”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">相对于通用型的触达，定向触达的优势主要包括以下几点：</p>
<ul>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">更易量化数据。因为目标明确，数据埋点、取数、分析都更加明确和方便。</p>
</li>
</ul>
<ul>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">更易制定策略。制定策略只要思考目标的特点、场景就好，减少了大量干扰信息。</p>
</li>
</ul>
<ul>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">更好的用户体验。针对性设计触达内容，可以迎合消费者的喜好和习惯，从而提升转化的可能性。</p>
</li>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">更低的触达成本。因为触达渠道精简，可以减少无效的触达支出，让企业每一分钱都花在刀刃上</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">综上，定向触达已经成为了现代企业的一门必修课。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">常用定向触达手段有哪些？</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">笔者分析了大量线上、线下企业的触达手段后，总结出以下6类手段：短信、PUSH/站内信息、微信推送、邮件、电话、精准广告。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/cda8323e7b825035a16cce8f275d665c.png" alt width="681" height="373" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同的行业、客户群体适用的触达手段不尽相同，每种手段都有他的优势和劣势，建议根据行业特性和公司能力择优选择。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">短信</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">曾几何时，短信曾是热恋小青年眉来眼去的重要媒介。但随着微信的普及，短信开始没落，手机收件箱沦落为验证码与营销短信的集散地。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">虽然消费者很少主动发短信，但短信却是最常见的用户触达手段。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为便宜！因为通用！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">企业群发短信的成本通常在2分钱-6分钱不等，如果是彩信也只要1-2毛。量大的时候还可以进一步压低成本。基本上是个企业都用得起。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外，目前的手机，哪怕是老人机，都支持短信的接收。根本不用考虑用户手机上是否装有APP，是否登录了平台。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只要你的手机开机，没有退订，就能收到短信。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是短信也有一定弊端。主要体现在两方面：（1）短信的泛滥（2）承载信息有限</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为每个企业都能群发短信，消费者收件箱常常会被各类营销短信塞满。审美疲劳的用户点开收件箱后，往往只会扫一眼，根本不会点击查看详情。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果消费者频繁收到营销短信，甚至会出现退订的情况。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外，短信通常都是通过文字来传递信息，相比于图片、视频的感官冲击力更弱。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通常一条短信文字限制70字以内，但实际上，即使连续发送多条短信，通常只有最后一条短信的前40个字才能发挥效果。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为用户通常不会点击短信查看详情，收件箱内的展示信息就是全部触达内容，比如下图。</p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">短信在收件箱内的预览只有40个字左右</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">短信的触达很考验文案的功底，能否用40个字抓住消费者的眼球，在众多竞争短信中留下深刻印象是短信触达的关键。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">PUSH、站内消息</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">PUSH也叫通知栏推送，即在手机终端锁屏状态下通知栏展示或在操作前台顶端弹出的消息通知。可以理解为用户下载某APP后不打开就能收到的信息。</p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">PUSH信息展示</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">站内消息通常是APP内私信、通知等，是用户打开APP后才能收到的信息。部分APP的站内信息可以通过PUSH来传达，简化了用户接收信息的流程。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">之所以把这2种方式合并在一起，是因为PUSH和站内消息都是免费的触达工具，前提都是用户安装了APP。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用户一旦卸载APP或者关闭推送权限，这2个免费的触达工具就会失去发挥的空间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了免费以外，推送的频率也可以灵活调整，只要你不担心用户的反感，一天100条都没关系（私信类的PUSH不算）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通常促活型的推送每日数量建议控制在2次左右，多了特别容易被用户关闭推送权限。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过笔者也见过一天推送20条的APP，比如财经类、社交类的APP。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这类APP推送的内容都关乎使用者的切身利益，因此即使推送再多，也不会引起用户反感。</p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">一天PUSH 20条的范例</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">值得注意的是，部分品牌的安卓手机，如小米、华为、OPPO、VIVO等为了保证自家用户不被垃圾推送骚扰，限制了APP的推送频率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一旦推送频率超限，这些手机会自动屏蔽推送内容，也会造成PUSH能力的失效。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此分内容、分机型控制触达频率，尽量为用户提供有价值的信息，是PUSH发挥功效的基本前提。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">微信推送</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">微信理论上只是一个独立的APP，不应该分类为一种手段。但微信的用户量实在太大。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">截止2022年6月，微信月活跃用户12.99亿，这么大的量级完全有必要单独研究和使用。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">微信推送通常指服务通知、公众号、微信好友触达3类，都是免费的触达工具。</p>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); font-size: 16px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">服务通知</h3>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">服务通知通常也叫微信小程序通知，这类消息目前都被微信整合在【服务通知】频道中。所有小程序（包括竞品）的通知都共享一个频道。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">公众号触达形式又可以分为3类：模板消息、图文消息、客服消息。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">模板消息是用户触发了某类特定行为后，自动推送的消息。何设定具体的触发条件是关键。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">模板消息有一个好处，就是推送数量基本不受限制，用户多次触发，就能多次推送。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">微信好友触达通常包括朋友圈信息、微信私聊两类。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果因为业务需求确实要加微信，建议取名取个正经名字，一定不要加AAA之类的前缀。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">朋友圈信息就是在朋友圈直接发布文字或图片信息，引起微信好友的关注。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">之前微商流行的时候，基本上每个人的朋友圈都被各类广告占领过。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">正确的方式是私聊先问候一下，然后询问现状，有什么想法，之后再根据客户的需求推荐商品或服务。不过如果信任度不够，通常对方是不会回应的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那有没有什么好的办法快速提升信任度呢？很遗憾，暂时没有。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">信任不是通过聊微信建立的，主要影响因素包括2个部分：（1）加微信之前的印象、口碑（2）实际商品、服务体验。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">笔者亲眼见过一些公司强迫微信客服和用户建立信任关系，但最后效果都不好，因为这真的不是微信客服能办到的事。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">微信私聊是所有触达方式中，转化效果最好的，是也最考验<a href="https://www.niaogebiji.com/tag/%E8%BF%90%E8%90%A5" target="_blank">运营</a>能力的。因为触达的人员通常是一线的销售、客服人员，他们水平能力参差不齐。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">运营人员要做好话题库的整理、用户画像的采集、话术的培训。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而且培训不能只是那种几百人一起的统一培训，运营要做的是建立客户信息提报机制，根据提报上来的用户画像，针对性的一对一指导培训。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">光这一点，很多公司都做不到。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">邮件</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">邮件分为电子邮件和实体邮件2种。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">互联网刚兴起的时候，电子邮件是主流的触达手段，账单、广告、通知等都可以通过电子邮件进行触达。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但随着时代的发展，电子邮件这种触达方式变得有点鸡肋。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">注重触达的企业，通常也注重用户体验，会让用户尽可能少填写信息。通常用户提供一个手机号就好了，因此缺少用户的邮箱信息，电子邮件就无从谈起。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而需要用户填写邮箱的企业呢，通常又是那种很传统的企业，对触达又不太上心。电子邮件发送的信息基本都是模板信息。很少有精力花在触达的优化上。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你们说，这种情况下，电子邮件是不是有点食之无味，弃之可惜呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过电子邮件胜在免费，在一些专业型的服务领域还是可以发挥他的作用的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">实体邮件现在很少见到，但一些奢侈品、化妆品企业很喜欢用。实体邮件成本高，但能给客户带来惊喜感和独特的仪式感，用户体验100分。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-type="image" data-ace-gallery-json="&#123;" items":[&#123;"uuid":"840ba138-e17a-445f-a823-0d0bc8d0c329","height":450,"width":738,"currheight":450,"currwidth":738,"natrualheight":450,"natrualwidth":738,"pluginname":"imageupload","scale":1,"src":"https%3a%2f%2finternal-api-drive-stream.feishu.cn%2fspace%2fapi%2fbox%2fstream%2fdownload%2fpreview%2fboxcn1ilnafunagwa3dfjdgdtkb%2f%3fpreview_type%3d16","file_token":"boxcn1ilnafunagwa3dfjdgdtkb","image_type":"image png","size":336570,"comments":[]&#125;]&#125;"><img src="https://qnssl.niaogebiji.com/afcfa4f79c13e5870ff08cf086fcb858.png" alt width="738" height="450" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">实体信件展示</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">实体信件不建议做太多营销动作，可以夹带广告单，但是信件主体内容一定要从用户出发，以感恩、祝福为主题是最好的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">文章最好用手写，之后再扫描打印，可以提升亲切感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有一些企业发实体信件时，全文都是打印字体，内容极度营销化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如：为了庆祝公司XX周年，特此邀请用户参与XX活动，XX商品N元起。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用户打开后感觉像是收到了广告单一样，这种实体信件的体验就很差，还不如不做。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">电话</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">电话触达可分为真人电话、固定录音、AI语音这3类。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这3类触达方式给用户感知都是非常强的，但是缺点也非常明显：</p>
<ul>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">接通率低，没有摸清用户的作息时间就打电话，有30%接通率就很不错了</p>
</li>
</ul>
<ul>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用户如果没有接听到电话，就不知道电话中的内容，相当于没有触达</p>
</li>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">错误的时间、过高的频率以及反感的内容都非常容易引起投诉</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3类触达方式中，固定录音的促活转化效果是最差的，最好的是真人电话。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用户喜欢互动型的电话内容，固定录音和AI语音都很难提供互动体验。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">真人电话虽然可以互动，但是效率不高，一个人一天能成功接通50个已经非常厉害了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了提供顾问类服务的行业，其他行业建议尽量不要一开始就用电话触达。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当用户活跃度下降，即将流失的时候，再使用电话触达是一个比较合理的方式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为效率低，真人电话触达的对象一定要严格筛选，高客单的那部分用户才是核心，其余用户可以用其他方式进行触达。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">精准广告</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">精准广告就是用广告的方式，进行差异化的触达，有2种匹配的逻辑：</p>
<ul>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据用户匹配不同广告</p>
</li>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据广告匹配不同用户</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一种逻辑是传统的广告逻辑，先找出细分的目标群体，然后根据用户的特征、需求投放广告。线上线下广告均可应用。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过这类广告非常依赖广告投放同学的经验，经验不足的时候经常会出现一个问题——用户是找到了，但是广告内容用户不一定很喜欢，或者没有理解，导致广告效果打折扣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此第一种逻辑下，广告主通常会做几十、几百条广告一起测试，通过转化数据寻找最合适的广告内容，并快速迭代。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二种逻辑是比较新颖的方式，是依托于大数据算法而诞生的广告逻辑。这种逻辑也只适合线上的广告投放。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">相比于第一种逻辑，用户更精准，在相同转化的情况下，广告成本会略低于第一种逻辑。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">它的具体流程是：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">先做一个广告（视频、图文都可以） → 广告小范围投放给不同的用户 → 机器学习，根据广告的点击、完播率等数据寻找合适的用户 → 针对同类用户大范围投放广告</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/ff3d65f00548c4e0c189bb01825f832e.png" alt width="654" height="258" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">2种逻辑实现方式对比</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二种逻辑通常需要2-3天的机器学习的过程，因此不适合频繁迭代，投放越久、投放量越大，这个广告的受众越精准。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">其他注意事项</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">定向触达能力不是一蹴而就的，需要一个长期建设的过程。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">包括埋点的建设、工具的搭建、运营人员的培养、触达内容方式的迭代等。通常需要3个月以上的建设、尝试周期，切不可操之过急。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外，对于触达，管理层要坚守道德底线，千万不要制定过高的KPI，否则容易出现标题党、诱导信息这类触达内容。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实证明，标题党、诱导信息的点击率是最高的，但是对于消费者的体验是一种损害。长此以往，品牌调性会大打折扣，为未来埋下隐患。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">小结</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现在是个信息爆炸的时代，消费者每日接触的信息量巨大，通用的触达难以给用户留下深刻印象。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前来看，定向触达是未来的主流趋势，也是企业降本增效的关键手段之一，值得每一个公司重视。</p>
