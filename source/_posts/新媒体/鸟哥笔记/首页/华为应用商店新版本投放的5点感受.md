<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>1、子计划功能</span></strong></span><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上一篇文章里面，我重点说的是关于“子计划”功能的出现带来的影响，对于细节的东西虽然没有具体的说出来。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，如果多点思考的话，其实就可以去理解为何强调了“子计划”？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>子计划究竟代表了什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我也做了举例，<a href="https://www.niaogebiji.com/tag/%E5%8D%8E%E4%B8%BA" target="_blank">华为</a><a href="https://www.niaogebiji.com/tag/%E5%BA%94%E7%94%A8%E5%95%86%E5%BA%97" target="_blank">应用商店</a>的子计划类似信息流里面的广告组里的子计划；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>账户里面的计划之间是竞争的，广告组里面的子计划也是会竞争的；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而以前的文章我也提到过，如何去利用好计划之间的竞争，或适当的时候去控制一些不必要的计划让某条计划起量。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果看过这些文章，还记得这些文章那么自然会看到华为推出子计划功能我所强调的作用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相关文章：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《</span><span>我放着那么多条计划投放，不让计划“起飞”是为了......</span><span>》<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《</span><span>善用投放计划之间的竞争</span><span>》<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《</span><span>8点告诉你，我需要媒体这些功能！</span><span>》<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《</span><span>媒体平台的功能设置、算法能力和投放效果的关系</span><span>》<br></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>2、账户预算功能终于出现了</span></strong></span><span><br></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还记得我这篇文章《</span><span><a href="https://www.niaogebiji.com/tag/%E5%B9%BF%E5%91%8A%E6%8A%95%E6%94%BE" target="_blank">广告投放</a>中敢于各种尝试</span><span>》吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>包括之前讲过的一篇关于媒体平台功能的设置，这里华为广告平台终于设计了广告账户的预算设置，尤其是还有次日预算的设置（凌晨0点开始）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>别小看这个细节，不然我不会在之前的文章分析过，而人家平台也这么安排了，这里面自然有它的门道、逻辑考虑，而其它应用商店还没有这种提前设置预算的功能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果广告计划也能有这样的设置就更好了！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>联动的是广告计划的最低预算是500元，不是之前的200元，500元的提升更多是为了方便计划里面更多子计划的设置，从而方便分配量或消耗。<br></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>3、数据的直观变化让我们找到新的优化技巧</span></strong></span><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上篇文章提到这次华为的改版在数据分析上更加清晰，而这个清晰对在优化手段上有一定的帮助，也看个人的理解度和敏感度。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然数据的查看上需要下载下来，但下载后比较直观；<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这个挖掘中可以通过数据能发现一些很轻微的问题：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，如果投放计划连续几天不做大的变动的话，你会发现每天按消耗排名的计划会出现变化，关键词也会出现变化，为什么会这样？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使分析不出原因，那如何更好的控制计划为我所用，这个是更加重要。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个是通过数据观察后发现的问题，以及如何去控制。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在刚开始团队在控制上还是走了弯路，刚开始做得不好，后面还是发现可以控制投放上的一些变量，比如出价等进行控制、调整和适应等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然出价的调整又可能去影响了原先计划的投放节奏和模型，所以怎样调又成了下一道环节里面优化的细节了。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>4、关键词计划要敢于尝试</span></strong></span><span><br></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在之前的文章《</span><span>广告投放中敢于各种尝试</span><span>》里面我提到过要大胆尝试，但在关键词计划的投放上实际我们原先做得还比较保守。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但实际上因为在第3点提到的因为在投放技巧上新的启发后，在关键词的投放上关键词的选择无论“精准和广泛”都可以进行设置。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过在设置上依然存在技巧，“广泛”匹配的关键词投放可以带来远远多得多的量，但在用户精准上可能有所偏差。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，通过数据的结果上可以告诉你该怎么做。<br></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>5、媒体算法里面呈现后端数据的一个规律</span></strong></span><span><br></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> 应用商店的投放它有个好处，一旦投放模型成立的话往往可以好多天不用太去大变动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是因为应用商店的算法上和信息流的还是有比较大的差异，尤其在不能多开账户进行同时投放这个点上，这个点的差别恰恰说明在算法上或混量上的差异。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，一旦计划一直在投放，没有对计划做太多的细节调整，可以在后端观察到一些明显的规律。</span></p>
