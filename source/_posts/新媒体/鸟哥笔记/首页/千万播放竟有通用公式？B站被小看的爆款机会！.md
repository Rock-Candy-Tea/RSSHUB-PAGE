<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">你知道B站的“镇站之宝”吗？</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">2019年7月，B站UP主「洛温阿特金森」创作了作品《【派大星的独白】一个关于正常人的故事》，这支作品是利用动画<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">《海绵宝宝》为主题元素</span>，截取部分经典桥段进行二次剪辑，搭配上鬼畜的音乐节奏，赋予作品全新的故事立意。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p7.itc.cn/images01/20220727/292362dc891f4f2db13360c1b23c572a.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-B站【入站必看】</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">由派大星为主体反讽社会人群现状，作品中经典片段“妈妈，那是什么动物？那是废物，宝宝。”更是出圈，一度成为短视频平台热门BGM。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">引人深思的内涵被观众高度称赞为“是这几年看过的最牛逼的鬼畜”。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p2.itc.cn/images01/20220727/1bc40aac69594231bfc4ab96195cdc8e.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-B站</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">这样优秀的作品或许很难去复刻它的内涵高度，但可以提炼出优质作品的创作要素，总结来说就是：<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">经典+二创</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">二创是B站UP主极具代表性，也是平台一直在大力扶持的一个“优良传统”。但如果你认为二创对象仅仅只是热门相关那就错了，稍纵即逝的热点固然要追，但有两点我们需要注意：</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">B站更注重<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">数字资产的沉淀</span>，就是官方曾提到的时间复利、内容复利，这是B站中长视频带来的平台属性。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">Z世代作为B站用户的主力军，他们不仅能适应快时代的信息产物，同时对<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">内心深处的共鸣有着浓厚的需求</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">结合这两点来看，相比于时效短的即时热点来说，经久不衰的是经典元素的二创，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">越是满足Z世代人群的群体回忆，就越是能够获得长时间的复利效果</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">以UP主「黄方块和粉星星a」为例，在5月23日UP主发布作品《“轻 轻 敲 醒 沉 睡 的 心 灵”》，截至目前已经揽收<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">1379.33w</span>的播放量，互动数据中点赞数高达<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">123.1w</span>，整体数据都到达了B站少有的爆款视频高度。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p8.itc.cn/images01/20220727/48f67b3f33274d2e848ccdc42aa9f98d.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">在2019年UP主利用《海绵宝宝》创造出B站的镇站之宝，时至今日仍然有UP主学以致用，同样也是经典动画《海绵宝宝》二创作品，也同样获得了高热度的传播，这都彰显出<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">“经典元素的二创在B站不管何时都不过时”</span>的平台风格。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">但是经典二创同质性严重的话用户也会审美疲乏，经典二创在此基础上还能如何创新？</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">经典二创还能这么玩</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">- 一举两得：即时流量+经典动画</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">虽然说热点时效短，但不可否认热点是信息更迭下最新潮的产物，那既然热点要追、旧流量有影响深远的优势，那就可以将两者进行结合，打造出来的会是<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">引起群体共鸣的新长效顶流</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">7月6日，周杰伦首发专辑先行曲《最伟大的作品》引起全网热议，官方授权二创，B站为此开通了周杰伦新歌的二创专区。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p5.itc.cn/images01/20220727/a88b5388e0e947cc999be9ce918c8b61.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-B站</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">UP主「光芒万丈黑暗骑士」迎着这个最新的热点浪潮在7月8日就发布了作品《【最伟大的作品】音画同步率高达99.99999%》，在视频中UP主搜集经典动画《猫和老鼠》多个片段，与《最伟大的作品》的音频进行高度重合，动态画面赋予音频具象的视觉体验，最终完成这个新旧结合的顶流之作。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">作品一经发出，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">不足一个月的时间不仅拥有737.04w的播放量</span>，互动率更是高达22.26%，播粉比几乎是千万倍的增长，尤其是一键三连数均超6位数。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p9.itc.cn/images01/20220727/380a9e4673834ef08e4d7663566d6772.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">直至今日作品仍然<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">每小时拥有3.6w的播放增量</span>，实现热点+经典二创带来的长效时间复利、内容复利。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p8.itc.cn/images01/20220727/4675ec05c3f74597912601dc75bf5c6f.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">- 跨区二创：知识+经典动画</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">作品素材是多元的，在B站更是不用拘泥于一个分区埋头创作，跨区联动永远是B站用户喜闻乐见的<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">“双厨狂喜”</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">所以经典动画的二创还可以加上其他分区的元素叠加，比如UP主「大福咕咕」和美妆护肤品牌花酿联动的商单作品，在作品中UP主计算了汤姆在动画中受到多少次致死打击数，从而反向推出汤姆在动画中一共有多少条命，计算的过程囊括了诸多<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">理科相关的公式知识</span>，促使众多观众在弹幕中纷纷参与计算讨论。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p1.itc.cn/images01/20220727/838ad77bc55040dfa4449df6a7469970.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
