<h4 class="ql-align-center" style="border: 0px; margin: 25px 0px; padding: 0px; font-size: 14px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #818386; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; font-weight: 300; line-height: 30px;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p2.itc.cn/q_70/images03/20220907/ffe53ac5c7f64d2c8f23c12f683741bb.png" referrerpolicy="no-referrer">来源-B站</h4>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">B站评论区、弹幕向来是观众集中玩梗的营地，尤其是B站推出周期合集“哔哩哔哩妙评”后，评论区更是用户浏览行为的必经之地，所以<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">评论区对用户的吸引力一定程度促进了置顶蓝链的曝光度，提高创作者带货转化</span>。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">借着平台带货风潮渐起的势头，品牌和UP主们自是没有放过，不论是常规的<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">横屏模式</span>还是B站刚推出不久的<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">竖屏模式</span>都在跃跃欲试。</p>

<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">零食品牌奇多接连投放两位UP主，其中还包括竖屏短视频UP主，两种形式的推广视频却收获旗鼓相当的推广效益。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">首先是美食区UP主「神奇海挪」，此次投放无疑是品牌的垂直投放决策，在视频中UP主通过复刻外网很火的创意料理“脆皮酥球”，直言这是“幸福感爆棚的热量炸弹”来抓取眼球。</p>

<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">但没想到的是，奇多作为零食品牌，在此次投放中竟是以制作料理的材料出镜，在UP主制作料理的过程中，芝士和意面的结合产生出黏腻的视觉体验，和外网原视频的感官体验差距甚远，让观众都以为UP主翻车制作猎奇料理，所以在产品出现的同时，弹幕不禁发出疑问“官方知道你拿它产品做这玩意吗？”视频节目效果一键拉满。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">这样的节目效果给了观众玩梗的空间，评论区必然会涌入不少的观众前来围观，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">UP主借势在评论区置顶蓝链，完成整个推广闭环</span>。</p>

<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">而这支作品也成为UP主少有的爆款视频之一，传播指数高达90，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">播放量超623w，播粉比更是超4倍增长</span>，视频传播水平远比UP主平均水平高出277.13%，推广视频得到高曝光，推广产品露出加上评论区蓝链带货不仅刺激目标用户消费，品牌也在B站收获高声量。</p>
