<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月9日，由AppsFlyer携手罗斯基、ironSource、Voodoo联合举办的2022移动出海营销与增长峰会召开。本次活动针对隐私新时代下如何了解营销全貌并实现智能优化；爆款的成功是否可复制；位于竞争激烈的游戏赛道，如何找到优质渠道，并持续触及目标用户，成为脱颖而出的黑马等几个从业者关心的角度方向进行分享。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在本次活动上，Voodoo中国区负责人王若斯带来了《爆款怎么做：<a href="https://www.niaogebiji.com/tag/%E6%9C%80%E5%BC%BA%E5%BF%AB%E9%80%92%E5%91%98" target="_blank">最强快递员</a>delivery it》。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>以下为分享内容整理：</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家好，我是Ruosi，负责Voodoo中国区的业务。简单介绍一下Voodoo，Voodoo是一家超休闲游戏发行公司，目前游戏全球下载量已突破60亿。60亿是什么概念呢？自App Store创立以来，Voodoo产品下载量排名全球第三，前两名是分别是Meta和Google。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5595111762ba2af63f95f5.74731692.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>一、从立项到上线历经17个版本调优《最强快递员》产品复盘</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今天带来的案例分享是《最强快递员》（Deliver It 3D）这款产品，该游戏已登顶美国等20多个国家和地区的下载榜首。《最强快递员》的游戏玩法很简单，玩家扮演一名快递员，在路上会遇到各种各样的快递包裹，在送货的过程中会遇到各种车辆、动物、人群等障碍物，玩家需要尽量避开障碍，最终成功送到客户获得奖励。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>游戏玩法很简单，但这样简单的玩法却能成为爆款，背后是团队大量努力的结果，下面就来复盘一下。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/41219298362ba2af6727456.08501082.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、Launch之前8个版本，CPI从0.68$降到0.17$</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>游戏整体优化分为两个阶段，一个是Launch之前，游戏进行了8个版本的迭代。另一个是Launch之后，从3月到5月底，又进行了9个版本的迭代。整个流程的关键词就是创新，每次迭代都能看到创新点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《最强快递员》这款产品CPI成绩非常好，在launch前的8次版本迭代将CPI从0.68$降到0.17$，同期的产品大多在0.2$以上。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/180726181462ba2af69193e3.32419021.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么版本迭代都做了什么呢？我们选择几个重要的版本进行分享：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一版只一个玩法展示，没有太多的背景内容；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二版本加入了环境因素和失败的情景，CPI降低了50%；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第四版加入了竞速模式。竞速模式是当前跑酷游戏中比较流行的模式，当通过指定数量的关卡后，可以和几个对手一起竞赛，这次迭代CPI降低了20%；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第六版加入了快递员过两条马路的设计，并衍生出了一系列不同的创意；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205236506362ba2af6b8c262.24073115.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可见，一个爆款产品创新是非常重要的。同时我们发现，每一次提出新的创意后，在一周到两周又会发现新的创意。如果一直用已有的创意，那么CPI可能不会有明显的进步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2、Launch之后9个版本，留存、时长、变现全方面提升</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来游戏进入launch阶段，Voodoo的运营团队会参与进来，所以前期的8个版本迭代更多的是依赖开发团队自己制作，Voodoo会帮助录制、剪辑视频，以便更好的测试游戏创意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Launch之后，Voodoo会和开发团队一起参与制作。这个过程中从提出想法，到落地再到AB版本测试，到最后决定方案，都是所有人一起参与。所以launch后的9个版本做了非常多的创新，基本上每周会有1-2个创新点。下面分享几个效果好的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/167678205662ba2af6e30446.54748252.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《最强快递员》最初设计的关卡末尾和常见的跑酷游戏一样，将所有钱扔出去根据距离远近奖励翻倍。虽然这样的设计各方面都很好，数据也不错，但我们还想做出创新。于是就有了现在版本的关卡末尾——库房。通过搭建的玩法提高玩家的成就感。在加入库房后，除了游戏时长提升之外，游戏的各项数据都有明显的提升。所以最后留下了这个版本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/109026147162ba2af719edc5.69238689.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上文提到的竞赛模式，在加入到游戏后，次留提升了12%；加入新的车辆模型后，变现提升10%；加入新的关卡地图后，次留提升7%。当前混合变现是一个趋势，在第7次迭代中大幅改动了变现系统，增加了一些IAP点位、新的皮肤、新的车辆等，让游戏变现提升了19%。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>二、爆款产品成功秘诀：寻找机制、专注创新、强调满足感/失败感</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2021年至今，跑酷游戏依旧非常火爆，每个月都能出现1-2个热门跑酷产品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/68287129962ba2af7445c25.01279696.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过我们的建议是不要做相同题材的产品。例如下图中的游戏，玩家控制主人公，在前进过程中做出选择，如魔鬼/天使、游戏/学习等。但这样的题材在早期引用的游戏成绩非常好，后面的成绩越来越不好。所以不建议开发者做创意的时候不要模拟已出现过的玩法。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/97883431462ba2af7743f42.12081614.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么爆款成功的秘诀是什么？这里总结了三点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、找到一个游戏机制，简化它。寻找游戏机制的时候，尽量不要从已经成功的超休闲游戏里找，你会发现复制非常难。要从生活中找到一个点，把它作为游戏机制。例如《街头耳光侠》，把生活中想做得但无法实现的事情做成游戏，并成为了爆款产品。《最强快递员》则是在疫情期间非常重要的角色，本身也是生活中的一部分。如果把自己想象成快递员，那么在路上会遇到的事进行模拟，并做成游戏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、专注于创新。这点有些老生常谈，但如何坚持创新是一件比较难的事情。《最强快递员》的17个版本中，一直在不断创新优化，尤其是后面9个版本的创新，基本上每天都有新的创意。如果能保持这种状态，那么做出爆款指日可待。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、强调满足感，强调失败感。把这两个感受扩大化，帮助游戏“投机取巧”，做出好的成绩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/51585817362ba2af7ab0689.90082335.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>三、利用社交媒体寻找游戏创意与2022Voodoo最新合作产品数据指标</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着市场的变化，Voodoo的测试标准也会随之改变。当前最新的测试标准是：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Soft launch：CPI<30-40$，次留>40%，七留>10%。不过我们在实践过程中并不会完全依据这些数据，假如有产品的某项数据非常好，那么Voodoo会帮助团队优化其他数据。就像《最强快递员》，起初游戏的留存和时长非常好，但CPI较高，所以游戏的迭代目标就是针对CPI进行的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Full launch的指标是：D3/ROI>90%，D7/ROI>110%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，近期很多产品内容自带流量。我们鼓励开发者与社交媒体为友，寻找大众化的题材，这些内容自然流量，后期推广阶段事半功倍。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/210796090362ba2af7e2e865.93832229.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里总结了三点，帮助开发者找到题材。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、具有挑衅性。具有一定挑衅性或一定罪恶感的舒适度，这种题材放到游戏里可以满足用户一些幻想。例如随意吃东西，所以这种题材的CPI会很好。不过题材不能过于极端，避免触及平台政策限制。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/118137905362ba2af80e73c7.88017782.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当上线测试视频的时候，可以加上一些标签，有助于提升视频的效果。同时，找灵感也可以从这些标签下寻找内容，截取部分出来作为游戏机制。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/131890257162ba2af83a2199.50958913.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、美好的幻想。例如《Run Rich》这款游戏，就是希望变得更美、更有钱等，在游戏里实现各种人生理想。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16704269362ba2af876a3d1.21047270.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里也有一些标签，大家在上传视频的时候可以带上这些标签。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/46973311662ba2af896d1c3.72685538.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、ASMR，可以带来很强的舒适感或满足感。包括切、捏等物理属性很强的题材。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/44958436862ba2af8c39e21.96984122.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上就是帮助开发者找到灵感的题材。最后和大家分享一下Voodoo夏季开发者大赛已经启动了，奖金100万$。本次增加了一个奖项——热门原型视频。这是指一个新的Tik Tok账户上传视频，加入一些合适的标签，如果视频播放量非常好，Voodoo会认为这是一个热门的原型，可以参与评奖。这也证明Voodoo在筛选产品的时候，已经不局限于数据指标，游戏的热度，社交属性也变得越来越重要。</span></p>
