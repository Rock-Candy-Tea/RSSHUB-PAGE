<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">“认养一头牛，花活还能玩多久？”</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月5日，网红<a href="https://www.niaogebiji.com/tag/%E7%89%9B%E5%A5%B6%E5%93%81%E7%89%8C" target="_blank">牛奶品牌</a>“认养一头牛”预披露了招股书，拟于上交所主板上市。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019~2021年，认养一头牛的营收分别为8.65亿元、16.5亿元和25.66亿元；同期净利润分别为1.05亿元、1.47亿元和1.4亿元。2021年，按营收规模计算，认养一头牛在国内乳企中排名第六。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2014年成立至今，在曾经的地产大佬徐晓波的带领下，短短不过八年就能取得如今的成绩，认养一头牛无疑有着过人之处。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，一个网红品牌走向主板，就要接受更多来自大众和资本市场的考验，毕竟认养一头牛自诞生便充满了无数争议。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>认养一头牛到底是如何发家的？它的发展之路具有可持续性吗？<a href="https://www.niaogebiji.com/tag/%E7%BD%91%E7%BA%A2%E7%89%9B%E5%A5%B6" target="_blank">网红牛奶</a>又是否有真实力和大佬们正面匹敌呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/30210041862e155c21a7a14.53492483.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><span> </span></span><span><strong><span>认养了个寂寞</span></strong></span><span><span> </span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>认养一头牛是一个极具网红属性的品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌名字本身就自带故事，而在创业初期，这个关于认养的故事也确实圈粉不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年，认养一头牛联合吴晓波频道推出了认养活动，支付2999元就可以获得牧场奶牛认养权。此外，消费者花1万元便可成为联合牧场主，还能认养一头奶牛，并且享受牧场经营的红利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018年，认养一头牛又提出了“分享家”的概念，分享家为品牌卖牛奶、拉好友办会员，可以获得15%的提成。这种“拉人头赚返佣”的模式听起来是不是有些耳熟？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017和2018年，认养一头牛分别入驻社交电商平台云集和贝店，结果这两家电商平台均被工商部门和市场监管局认定为涉嫌传销。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，认养一头牛已经取消了这种分销玩法，所谓的“认养”故事也不了了之，仅剩下卖奶卡这一项，根据不同的产品、订购时间销售不同的奶卡。消费者在订购奶卡后，可以在预定时间内收到送货上门的牛奶产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，所谓的分享家，只是认养一头牛用来拉人头、提高复购率的话术，而所谓的认养，也只是营销噱头而已。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>我认养的牛和我喝到的奶有半毛钱关系吗？并没有！</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，明眼人一眼便能看穿，认养奶牛的模式放到大规模生产中，完全就是不现实的。一人认养一头牛，无疑会大大降低产奶效率，标准化、规模化的生产才是乳业的发展之道。因此，“认养一头牛”这个概念本身就站不住脚。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，更重要的是，认养一头牛连销售的牛奶，很多根本都不是自家生产的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>招股书显示，光明乳业、君乐宝、中垦华山、贝因美、佳宝乳业等厂商均为其代加工厂。其中，光明乳业是认养一头牛最大的代工厂，2019~2021年代工金额占总外协加工比一直在55%以上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>认养一头牛在2019年的自有产能为零；2020年仍有93%以上的纯牛奶和95%以上的酸奶都由外协工厂生产；直到2021年，其外协产量占比才下降为35.67%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>认养一头牛方面回应称，整体的发展方向还是尽量多使用自有奶源，在招股书中，公司也已经不再强调“认养”概念。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12887106462e299449eecf5.93067780.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>确实，我们需要考虑到，认养一头牛在成立最初就斥资4.6亿元在河北建立了牧场，并从澳洲引进了6000头奶牛，但当时受限于公司没有建立生产线，所以挤奶之后必须委托加工厂代加工生产。然而，不可否认的是，扎眼的贴牌数据和认养理念的营销模式从一开始就是自相矛盾的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>认养一头牛在高举高打营销大旗的时候，是否有为消费者的权益考虑过？消费者或许喜欢听一个好故事，但更希望听到一个真实的好故事。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/32497174762e155c29dc550.80106205.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 钱都去哪儿了 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在认养一头牛的招股书中，有一组惹眼的数字。2021年在营收实现了55.52%的增长下，净利润却出现了3%的下滑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>认养一头牛方面表示，这主要是因为原材料生牛乳、玉米、豆粕等市场价格当年有所上升，使得公司毛利率下降所致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，即便不考虑通货膨胀的因素，认养一头牛依然不得不考虑来自成本端沉重的包袱，高企的营销费用始终是道坎儿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数字不会说谎，我们不妨来对比一下认养一头牛和同业的营销费用数据。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年，认养一头牛的销售费用率为18.82%，几乎2~3倍于同其收入规模相近的皇氏集团<span>（10.57%）</span>和天润乳业</span><span>（4.58%）</span><span>，与伊利</span><span>（17.46%）</span><span>旗鼓相当。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和许多新消费品牌一样，多年来，认养一头牛始终将主战场放在线上。2021年，公司在线上渠道的营收为19.52亿元，占总营收比重的77.67%，线下的营收为5.61亿元，占比仅为22.33%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此次IPO，认养一头牛拟募资18.51亿元，其中5.2亿元将用于品牌建设营销推广，主要在淘宝、京东、抖音等线上渠道进行品牌推广。可以看出，至少在未来的一段时期里，认养一头牛仍将继续开垦线上渠道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没有当初的互联网营销，就没有认养一头牛的今天。但认养一头牛的明天，不可能靠这一招鲜吃遍天。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>诚然，互联网过去黄金十年的超高速发展确实孕育了一大批新消费品牌，新国货也乘着互联网的翅膀携手腾飞。但与此同时，所谓的“轻线下、重线上”模式其实并不能节约成本，相反，为了维护网络流量，线上的营销费用水涨船高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>招股书显示，子公司杭州一头牛2021年亏损6590万元，而同期主营牧场经营及生牛乳销售的子公司河北康宏牧业有限公司净利润为1.24亿元。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/164016388862e29944d4be69.51007407.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>沉重的营销负担使得认养一头牛不得不“拆东墙补西墙”。于是，研发就成为了牺牲品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019~2021年，认养一头牛研发费用率分别为0%、0.04%、0.27%；而伊利股份近3年研发费用率大约为0.5%；皇氏乳业更是高达2.7%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在研发人员方面，2021年认养一头牛的研发和技术人员仅有23人，占公司总人数的1.17%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于所有的新消费公司来说，先得会讲故事；接着要大手笔烧钱。</span><strong><span>但公司不可能有永续现金流，想要持续从一级市场或者二级市场融资，就需要可持续性的业绩来说话，否则故事最终只能是留不住的镜花水月。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/122485387262e155c2eb1616.84257600.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 以卵何以击石 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于认养一头牛而言，除了审视自身，或许也在感叹：“只怪对手太强大。<span>”</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国内的乳业市场已经风平浪静了许多年。虽然新晋品牌并不在少数，却激不起多少水花。伊利和蒙牛牢牢占据着榜首的前两把交椅，连老三光明都只能远远望其顶背，更不用说前三甲后面的一溜乳业新兵蛋子了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以零售额计，2021年伊利和蒙牛分别以25.8%和22.8%的市占率，位列行业前两位，说是两大巨头稳坐钓鱼台毫不为过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，认养一头牛和龙头的差距究竟有多大呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一，渠道难破译。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，我国乳制品销售渠道仍以线下为主，2021年线下传统及新零售领域销售占比达到80%以上。消费场景的多元化决定了线下是乳企不能忽视的重要领域。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>认养一头牛的线下销售收入占比从2019年的37.70%，下降至2021年的22.33%，不升反降。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反观伊利、蒙牛等，在线下渠道根基颇深、极为强势，对于大多数线下商超来说，他们也更愿意和这样的巨头合作，将更多展位留给头部品牌。与之相比，认养一头牛的议价能力肯定要弱上一截。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更需要警惕的是，</span><strong><span>顶流乳企拥有强大的资本背景，虽然在线上渠道的拓展上不如新消费品牌那么快，可是一旦发力，必然是又狠又准，叠加原有的品牌效应，分割线上市场几乎是板上钉钉的事。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着互联网黄金十年的逐渐远去，电商红利的周期已然消逝，获客成本不断提升，线下渠道的发力将成为品牌们未来的必经之路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而这条路上已经巨头林立，如何才能走通依然是道难解之题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/41347682862e2994504a046.81288206.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二，产品不抗打。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当前，认养一头牛的产品线集中在纯牛奶和酸奶为主的常温奶赛道，在奶粉和奶酪等产品板块上较为薄弱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而乳业龙头们基于丰富的产品线，对上游奶源的拿捏已是妥妥的，从而形成了对上下游都强有力的话语权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更加恐怖的是，即便在认养一头牛主攻的常温奶阵地，伊利和蒙牛“双寡头”还得说一句：“在座的没有一个能打的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据Euromonitor数据显示，2020年，两者合计市占率已经达到了惊人的85.7%。难以想象，在这样的红海领域里，认养一头牛的生存环境远不如它第六位的排名看起来那么光鲜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>打开认养一头牛的天猫旗舰店，“A2β-酪蛋白纯牛奶”的宣传页面中标示有“优质蛋白”“挑出好基因，才有好蛋白”等说明；“娟姗纯牛奶”的产品宣传页面中也标示有“量得出的营养”“一瓶便能满足孩子每天约50%的钙需求”等字眼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乍一看属实不错，但横向对比一下，就会发现其营养成分并没有什么稀奇之处。无论是伊利的金典纯牛奶，还是蒙牛的特仑苏纯牛奶，其蛋白质和钙含量都足以和认养一头牛正面抗衡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在同质化愈发严重的乳业环境中，认养一头牛所标榜的产品特质并不具备稀缺性。自然也就不具备足够的吸引力，从而无法建立起有效的品牌护城河。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了摆脱巨头的围剿，认养一头牛另辟蹊径。今年4月，认养一头牛开始着手肉牛业务，该业务板块独立于乳业板块，以子品牌的身份展开经营。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从奶牛到肉牛，从牛奶到牛肉，显示出认养一头牛敢于挑战巨头的雄心壮志。那么现在，摆在认养一头牛面前的任务就从“讲好一个品牌的故事”变成了“做好一个品牌的产品”。这或许才是真正的“脱虚向实”吧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去几年间，网红品牌们往往能第一时间吸引到流量和关注，但流星虽多，恒星难觅。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如同洋葱一般，当你对网红品牌们一层一层地抽丝剥茧之后，你会鼻酸、甚至流泪。这一次，认养一头牛同样也没能逃过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上市还差临门一脚，认养一头牛在期待取得阶段性胜利的同时，依然免不了压力山大。我们在为新锐品牌鼓掌的同时，也不禁为它们捏一把汗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>故事的最后，究竟是网红牛奶成功逆袭，还是镜花水月一场空呢？喝一口奶，压一压惊，认养一头牛又将继续前行……</span></p>
