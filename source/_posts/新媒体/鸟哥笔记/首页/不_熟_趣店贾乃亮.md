<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">有关<a href="https://www.niaogebiji.com/tag/%E8%B6%A3%E5%BA%97" target="_blank">趣店</a>（</span><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">QD.US</span><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">）的争议，仍在持续。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上周与东方甄选之间的“拉黑风波”未平，近日就罗敏“粉丝级别高才有资格合伙”言论，所引发的“收割加盟商”风波又起。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一石激起千层浪，围绕趣店起家的“校园贷”业务、趣店预制菜欲要收割加盟商的质疑，趣店及创始人罗敏遭遇口诛笔伐不断。在这场以“原罪”为焦点的舆论场中，众矢之的趣店方的声势日渐衰弱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤为重要的是，7月26日，参与到7月17日趣店当晚直播盛宴的傅首尔及<a href="https://www.niaogebiji.com/tag/%E8%B4%BE%E4%B9%83%E4%BA%AE" target="_blank">贾乃亮</a>两人，均就与趣店合作一事发表致歉声明，并都表示“没有后续合作”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，两人的声明并没有对公众情绪起到平息作用，甚至还一度挑起更为汹涌的议论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在傅首尔声明的微博评论区，便有一条高赞评论，直指傅首尔“那为啥要删除提醒你的评论呢”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而相较于声称“只是单次合作”的傅首尔致歉回复，趣店预制菜代言人贾乃亮的回复显然引起的轰动更大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月26日，#贾乃亮致歉#词条一度登上当天微博热搜榜第一，截至当晚，阅读量近4亿，讨论数为1.4万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“关于趣店的品牌合作，没有背调清楚品牌前身的经历，是我们疏忽大意了。”贾乃亮声明中的这段话，被网友与其在17日晚“趣店罗老板”直播间中，与罗敏“称兄道弟”的发言用来对比。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至此，贾乃亮这番“事前老大哥，事后不太熟”的操作，也让网友议论纷纷。微博账号@黑历史打脸bot也对此事进行转发点评道，“演技没得说。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但不管是傅首尔也好，还是贾乃亮也好，随着舆论的升级，对于罗敏来说，预制菜项目是否还能顺利推进，已经成为未知数。</span><span></span></p>

data-copy-origin="https://shimo.im"><strong>“带货主播”贾乃亮</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>贾乃亮真对趣店的历史一无所知？答案不言而喻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>成为“趣店预制菜代言人”并非贾乃亮与趣店的第一次合作。早在2020年趣店上线奢侈品电商“万里目”项目时，贾乃亮就是当初趣店签约的5位万里目代言人其中之一。贾乃亮还在2020年5月1日晚在抖音为万里目献上了首场直播，当晚交易额超3250万元，直播观看人数累计超2211万人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自然，贾乃亮惹起网友反感的关键原因还在于，直播间中声称“罗大哥”，将罗敏一系列“创业艰辛”“赔钱赚吆喝”等难以启齿的“辛酸事”和盘托出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在直播间，贾乃亮说自己“真的心疼罗老板”，并表示“大家不要以为他是上市公司的CEO，他有很多风光，他曾经在北京住过二十多个地方，有十几个地方是地下室，还是七八个人合租。罗老板当时的心愿，当时就想吃碗小炒肉，吃个小龙虾，吃不起，然后经历了这么多失败，最后终于有一天成功了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不得不说，贾乃亮在打感情牌这件事上是如鱼得水，通过深情演绎在屏幕前立住罗敏的“努力人设”，“（</span><span>罗老板</span><span>）已经成功了，但是他还是来做了这样一件事，就是希望将所有的（</span><span>预制</span><span>）菜，送到千家万户，让所有的人认识我们。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当罗敏在直播间“重金宠粉”时，贾乃亮也会与之打配合。自然，无论贾乃亮说上多少份1分酸菜鱼，罗敏最终定的上链接数量都要更多。与此同时，贾乃亮就会连连称赞“老板大气”，并对罗敏亏得太多大呼“心疼”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用贾乃亮的话说，算上物流和成本，20万单1分钱酸菜鱼在5分钟内售空，意味着“罗老板5分钟赔了500万元。”</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>彼时，伴随着直播间或深情或激昂的背景音乐声，在贾乃亮有的放矢的控场下，两人就“创业艰辛”和“创业理想”话题深度交谈中有来有往，罗敏还一度情绪激动泪洒直播间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如此一来，罗敏有理性有抱负、大方耿直的“人设”，也悄然在观众心中立起。“在事情发酵前，我对罗敏这人一点也不认识。那时在直播间就想着抢酸菜鱼和抽苹果手机，听他们聊创业奋斗史和创业理想时，我对这老板印象还挺好的，像是踏实做事的。”用户张霞说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不得不说，7月17日晚直播盛宴中，贾乃亮作为一名带货主播，是成功的。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但现今，贾乃亮在舆论压力下，果断选择了与罗敏“割席断交”，“发现问题后，第一时间双方已经解约，也没有后续的任何合作。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在贾乃亮发布的声明微博评论区，“不信任”的留言层出不穷。很显然，网友对贾乃亮的声明并不满意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>业内人士陈琪也觉得贾乃亮的声明有些“违心”，</span><span>在他看来，明星在对产品代言前，团队出于对明星本人的保护都会尽到调研义务，“翻车的品牌那么多，品牌翻车代言人也会受波及。</span><span>团队除了会自己进行背调外，还会委托第三方团队进行查验，防的就是意外。</span><span>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，陈琪也提及，从代言人贾乃亮角度考虑，其实只要趣店经营没有问题，贾乃亮大可不必撇清关系，“但是舆论发酵，意味着贾乃亮迎来了信任危机，处于爱惜羽毛的动机，贾乃亮致歉割席的做法可以理解。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于6月份才因“贾乃亮合伙公司偷逃税被罚”，而上过一次热搜的贾乃亮来说，多年积攒下来的路人缘，也已经不能再无端被消耗了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年6月中旬，话题#贾乃亮合伙公司偷逃税被罚#窜上微博热搜，随即引发热议。对此，贾乃亮和事发公司都对此进行紧急回应。公司的声明中强调，逃税是公司行为与股东无关，网络上流传的“贾乃亮合伙公司偷逃税被罚”等内容是博人眼球的不实报道，解释贾乃亮从未参与公司的运营，本次事件与贾乃亮无关，并再次向贾乃亮道歉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>贾乃亮也发文就此事回应，表示过去几年投资了两三家公司，自己没有权利和能力参与经营管理，这些年也未曾从投资的公司拿到收益，但这次教训后会深刻反省，定期监督检查，“我这个‘投资黑洞’今后也应尽量远离投资，深刻反省！”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便此事已经证明与贾乃亮无直接关联，但依旧给路人心中留下了污点。如今，事发不过月余，代言的趣店预制菜一事再起波澜，再有傅首尔的果断割席在前，贾乃亮只能跟进，否则，网友情绪的发泄口只会对准其一人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至此，趣店并未对此事发表任何声明，“趣店罗老板”账号在26日早上9点发了最新一条直播预告的作品后，账号也一直处于“静默期”，直播也一周没再开过了。“趣店生鲜预制菜”账号近两日虽然恢复了直播带货，但成绩却不太理想。</span></p>

data-copy-origin="https://shimo.im"><strong>趣店还有未来吗？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>失去傅首尔和贾乃亮两人，对趣店的影响或许并不大。但因这两人的“割席”行为所引发的蝴蝶效应，却可能会让其受到重击。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与7月17日晚微博热搜榜独具一格的#傅首尔心疼罗老板#、#贾乃亮连赞罗老板大气#等话题相比，7月26日晚热搜榜上与趣店有关的话题更多，但#贾乃亮致歉#、#趣店#、#傅首尔致歉#、#罗敏#、#校园贷#话题内一片倒式的质疑，则显得尤为剑拔弩张。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>巧的是，在事发前一天，即7月25日，在“拉黑风波”后沉寂一周的罗敏重出江湖，更新一条“10万份1分钱买酸菜鱼的福利活动”。在7月26日上午还发了7月30日的直播预告，按照之前计划，7月30日是“730老板福利日”，有着如7月17日直播间一般福利活动多多的计划，目前在趣店预制菜抖音商场，也能看到一款0.01元的酸菜鱼预售商品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从罗敏账号新视频的评论区来看，罗敏依旧对过往采取掩耳盗铃的做法。评论区仍设定为“作者仅允许互关朋友评论”。但业内运营人士桃子表示，评论区的内容都是已经“筛选”过的，“据我观察，视频刚发出时候是开放评论的，后来手动关闭了，但期间也放开过，能看到很明显评论区的评论都很明显集中于某些特定时段。再者，谈及校园贷等负面消息的评论，都会被删除。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“控评和限制评论的做法很聪明。即便互联网舆论发酵得很厉害，但总有一批人不玩社交网络，对此事毫无知情的，评论区留下的大量‘好评’，在这些不知情用户看来，就是很好的推广。”但桃子也告诉燃财经，尽管控评这种手段目前能奏效，但终究不是长远之计，“之前可能大家只知校园贷，不知趣店，但现在很多人已经知晓二者的关系，只能说众怒难平。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/130283855062e0acd587cf94.18493814.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/抖音 </span><span>燃财经截图</span><span> </span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着舆论的不断发酵，已经引发了网友群愤的趣店预制菜，处境愈发艰难。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要知道在这次的预制菜创业中，罗敏放了很多心血在其中。甚至对外也宣称此次创业为“二次创业”。然而无论是罗敏还是熟知罗敏创业史的人都知道，罗敏实则是一名连续创业者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在2014年趣分期（</span><span>2016年更名为“趣店”</span><span>）创业成功之前，罗敏已经经历了连续创业失败。并且在趣店于2017年成功上市后，趣店推出了大白汽车、大白儿童阅读、趣学习、唯谱家、相同same、万里目、万里目少儿等多个项目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>预制菜创业中用到的“撒币”策略，在罗敏过去的创业项目中，都能找到影子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>甚至罗敏本人，也曾发表过一篇名为《大白汽车为什么要“撒币”？》文章。文中提及“为大白汽车准备了上亿元来撒。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于“撒币”这一行为，罗敏本人也有自己独特的见解。而从罗敏参与的历个创业项目来看，初始阶段的“撒币”手段贯彻如一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上述提及的“万里目”便是，上线不久便打出“百亿补贴”口号，还重金聘请黄晓明、雷佳音、贾乃亮、郑凯等五名当时活跃在一二线的明星，成为品牌代言人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>预制菜无不如此，在1分钱酸菜鱼、1500台苹果手机的重磅福利的引诱下，数十万用户涌入直播间薅羊毛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是，过去罗敏推出的项目往往是“出道即巅峰”。风头无两的开局搭配黯然离场的结局，往往是趣店过去几年推出的项目的宿命。</span></p>
