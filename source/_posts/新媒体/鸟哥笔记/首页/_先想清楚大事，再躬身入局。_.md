<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最近，有不少人问我这样的问题：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">未来应该做线上还是线下？<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%95%86" target="_blank">电商</a>和<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD" target="_blank">直播</a>谁会更好？听说阿里的份额有所下降，拼多多、抖音和快手逐渐增加，你怎么看？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我应该怎么判断现在的格局？要怎么选？怎么做？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些问题都很好，也是很多人关心的问题。但是，这些都还是“点”的问题，不够整体和全面。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为了讲清楚这些事，我之前还专门画过一张图，希望能帮你更清楚地看到全貌。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/92960788262b00165cf4447.14330856.png" title="“先想清楚大事，再躬身入局。”" alt="“先想清楚大事，再躬身入局。”" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这张图上，有两股力量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一股是“在线化”的力量，是从左向右的力量。从线下，到电商，到直播。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一股是“体验化”的力量，是从上往下的力量。从搜索，到内容，到社交。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要怎么判断，怎么选，怎么做？我们得先看清楚了，才能更好地决定。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样，可能才不会迷迷糊糊。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">先想清楚大事，再躬身入局。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们一点点讲。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/113926470662832e66aa5709.61810445.png" title="“先想清楚大事，再躬身入局。”" alt="“先想清楚大事，再躬身入局。”" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">如何理解“在线化”的力量？</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在线化的力量，是从线下，到电商，到直播。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是，要如何理解背后的逻辑呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从线下到电商，你只需要记住一个关键词：连接。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">什么意思？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你一定听说过这句话：要致富，先修路。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">可是，为什么要修路呢？修路的本质，就是连接。就是把那些原来偏远的生产单元，都连接到交易网络中。这样，可以促进贸易，共同繁荣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">连接，让距离缩短。距离缩短，让信息对称。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">没有什么东西，是你知道而我不知道的。也没有什么技术，是你能用而我不能用的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从线下到电商，互联网把大家连接在一起。因此在交易时，也提高了传递价值的效率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">具体来说，是提升了这三方面的效率：信息流、资金流、物流。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在线下，原来你要去商场买衣服。你走进门店，摸一摸面料，挺舒服的，看一看价格，觉得也合适，这些都是信息流。你拿起来试一试，觉得不错，决定购买。老板开单，你付钱，这是资金流。然后，柜台耐心地打包，递到你手上，你拎着衣服回家了，这是物流。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这三个过程，几乎是在一起的。过程很快。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是因为互联网，因为连接，把信息流、资金流、物流分开了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你在网站上浏览页面，看展示的商品信息，把心仪的东西放进购物车，这是信息流。过了2小时，你才下单付款，这是资金流。再过2天，包裹送到了你家门口，完成物流。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">互联网公司，把这三个要素分开，利用连接的优势，分别提高效率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从线下到电商的在线，其实也是信息流、资金流、物流的在线。你在任何一个流上，都能找到成功的公司，也都能找到创业的方向。而这些方面的效率，当然也还有继续提高的可能。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，从电商到直播呢，又是什么逻辑？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你同样也只需要记住一个关键词：信息量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你有没有想过，为什么你在传统的电商场景里，很容易下单买一件漂亮的衣服，却很难花钱买一个漂亮的大衣柜？为什么你能迅速决定买一张干净的沙发垫，却犹犹豫豫不敢买一套豪华的大沙发？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有人说，可能是客单价的原因。衣服200块，衣柜2000块。沙发垫200块，沙发可能20000块。万一质量不好，万一我被骗了，大不了只亏200块，咬咬牙也就忍了。而客单价太高的产品，我不信任。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有道理。但可能不是本质。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本质原因，是信息量。都是在网上购物，直播卖出衣柜和沙发的概率，显然比在传统电商的场景中要高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我之前说过，信息在互联网上传播，经历三个时代：文字时代、图片时代、视频时代。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最一开始，网速还不够快时，主要靠文字传递信息。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是，文字的信息密度太低，很难讲清楚复杂的事。所以，我们只能在网上卖一些像书这样的简单的东西。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">给你一个书名，作者，目录，简介，你就能做购买决定了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">后来，我们能传照片了，照片的信息密度就更高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这件衣服，真好看。这个配色，我太喜欢了。这个款式，也是今年的流行款。我要买。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再后来，我们能随时随地刷视频，看直播了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这时，传递的信息密度，又更高了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一套沙发，不论你怎么用文字描述，怎么找角度拍照，都很难说服用户，这套沙发真的非常结实。那怎么办？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">找个200斤的胖子，随便在上面蹦来蹦去，怎么样，真结实吧。一段视频马上就清楚了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种信息量，是文字和图片没法比的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从电商到直播的在线，其实就是信息量不断增加的过程。信息密度越高，用户决策效率也就越高，能卖出的复杂东西也就越多。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这，就是“在线化”的力量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从线下，到电商，到直播。连接越来越紧密。提供的信息量越来越多。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是，这只是一股力量。还有一股力量，是“体验化”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/126592131362832e66ddac86.53943116.png" title="“先想清楚大事，再躬身入局。”" alt="“先想清楚大事，再躬身入局。”" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">如何理解“体验化”的力量？</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">体验化的力量，是从搜索，到内容，到社交。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">它们背后的逻辑，又是什么？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">先说搜索。我们就用一个字来总结：买（buying）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">还是比如说买衣服。要换季了，总该添置几套新衣服。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，要去哪里买呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">楼下的超市，以前经常去，那里的衣服挺不错的。或者直接在网上买也可以，去淘宝上看一看，有图片有视频，也挺好。还做不了决定？那就看看直播，听听那些主播们是怎么介绍的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你可以去线下，去电商，去直播。但是，不管你去哪里，都有一个非常明确的需求，我要“买”衣服。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你问线下的导购，这个款式的有吗？她马上一路小跑给你拿过来。你在搜索框上输入你要查的信息，马上就能跳转页面。你在直播间留言提问，主播也会给你专业的回答。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本质上，这些都是搜索的场景。你想要更好地搜索产品，想要更好地搜索答案。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不论去哪里，你都是希望“搜索效率”提高，能更好地“买”东西。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就是买，buying。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么内容呢？内容，也用一个字总结：逛（shopping）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们的购买行为，其实不一定是先有需求的。有些需求，有可能是瞬间被激发出来的。是我们逛着逛着，突然忍不住想剁手。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">线下你去逛商场，线上你会看文章看视频看直播，其实这些都是消费内容。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你在商场里逛来逛去，走进一家服装店。你发现，店里面有一本最新的时尚杂志。里面写着现在最流行的风格和趋势，看着杂志里的模特，你感觉自己走不动路了。我太想买这件衣服了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">抬头一看，这件衣服，不就刚好摆在店里吗。一种强烈的需求，突然被激发。我现在就要。对，就现在。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">文章和视频，同样如此。一篇文章，教你如何时尚穿搭，看得你心神荡漾，拉到最后发现，链接就在那里，你一不小心，就点进去了。再不小心，就下单了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">视频也是，这个主播讲得真棒，我就要做他的女孩，OMG。我真的太缺这件衣服了，现在就缺。马上，立刻。买吧。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">逛着逛着，因为“内容”，产生需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就是逛，shopping。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是，除了搜索，除了内容，还有社交。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">社交，我们也用一个字总结：跟（following）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">跟什么？跟随信任。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你朋友说，这件衣服啊，还真是好。你摸摸看，这个面料太舒服了，而且配色也好看。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我都推荐给好几个人了，都觉得不错。怎么样，要不要把链接发给你，或者干脆直接给你和老板拉个群？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为你信任他的品位，也信任他的人品，他这么推荐，那还有什么好说的，一定是真的好。闭眼跟着买就是了。错不了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这个“错不了”，就是一种信任。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以在线下，这种信任的商业模式，体现为直销和保险。我不会骗你，我还能服务好你。相信我，没错。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而在社交电商，可能就是我们非常熟悉的，拼团，大家一起来砍一刀。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而直播呢，也会变成社交直播。老铁们，亲人们，姐妹们，朋友们，你们不用担心，有我在呢。产品有问题，我去帮你们理论。万一有问题，我认了，而且加倍赔给你们。再说，我什么时候给你们卖过次品了？在我这，放一百个心。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就是跟，following。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以搜索，就一个字，买（buying），买完就走。内容，也一个字，逛（shopping），喜欢就买。社交，也是一个字，跟（following），你卖我买。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">搜索，内容，社交，是不同的体验。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同的体验，也有不同的商业模式。同样，也都有成功的人。谁能给用户最好的体验，谁就能获得更多的机会。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这是体验化的力量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/72621015062832e672b11e3.57050295.png" title="“先想清楚大事，再躬身入局。”" alt="“先想清楚大事，再躬身入局。”" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">如何提高两个能力？</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如何提高“在线化”能力？如何提高“体验化”能力？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现在，你已经看清楚了在线化力量和体验化力量这两股力量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">他们分别从线上，到电商，到直播。从搜索，到内容，到社交。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，我们要如何提高自己这两方面的能力，躬身入局？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其实，也有很多人问我这样的问题。我举两个可能是最常被问到的例子，和你分享自己的一些思考，也希望能给到你一些启发。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">提高在线化的能力，我经常被问到，我想做直播，怎么办？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">提高体验化的能力，我经常被问到，我想做内容，怎么办？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一个个说吧。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">关于直播，我自己也在尝试。摄像头怎么弄？灯光怎么调？绿幕的作用是什么？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我之前写过一篇文章，叫《如何在家做一场有“现场感”的直播》。里面有一些经验，你可以参考。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">硬件的部分，网上也有不少教程，都能找到。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而关于直播，可能更多人关心的，还是带货，还是商业化的部分。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我给过很多人这个建议，也多次提到，可以用“逆序建立品牌”的方式来做。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌，是一个容器，里面装着用户的了解、信任和偏好。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">建立了解，需要投入金钱，投放广告需要资金支持。建立信任，需要投入时间，日久才能见人心。建立偏好，需要投入情感，用真情换取真心。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">了解、信任、偏好，从左到右，越来越难。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是，如果做直播，却可以逆序建立品牌。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">直播的时候，用户喜欢你的真实，喜欢你的性格，或者没有原因就是单纯地喜欢你，都会对你产生偏好。这种偏好，久而久之会形成对你的信任。最终，你可以拿着用户的偏好和信任，和品牌合作，帮助他们获得了解。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">可是，有一件事情，必须永远放在心里：直播，更多是站在用户那一端，而不是站在品牌那一端。不能辜负他们的偏好和信任。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用生命选品，不好的东西就不要上。售价要最低，卖给自家用户的，不能收费比别人贵。无条件退货，如果出问题，把风险都吃进自己肚子里。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些，都已经是直播带货的基本功。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过，越基本的东西，越不能忘。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，如何做好内容呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我也把自己的经验，分享给你。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一，尽量要日更。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我自己决定做公众号时，就做了一个决定，日更。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为什么？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这是一种决心，也是一种职业精神。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有的人说，我文章今天想写就写，视频明天想拍再拍，行不行？如果是个人喜好，当然可以。但如果是一项工作，甚至一份事业，最好不要。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你看看那些博主，那些主播，没有一个敢停的。因为一旦停下，用户可能就走了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">做内容，保证更新的频率，其实也是提供一种确定性。用日更，换取用户的认知和习惯。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二，保证高质量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有的人曾经说，内容，是不是也可以随便做做？因为现在信息太多，内容太杂，根本看不过来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而且，可能7天10天之后，用户就忘记了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些现象，可能都是真的。但是，这不应该是做不好内容的借口。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">相反，高质量的内容，才能获得用户的信任。用户会留下印象，原来，他还是认真做的啊，那我就继续看看吧。这样，他才愿意奢侈地给你时间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，为了好的内容，专门建一个内容团队，或者不断死磕自己，都是必要的。内容，某种程度上来说，就是要有一种大炮轰蚊子，杀鸡用牛刀的架势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只有这样，才能保证有高质量的内容。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第三，长期主义的精神。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这一点，很好理解。时间的朋友，时间的朋友，有时候不是时间抛弃了我们，而是我们抛弃了时间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">做过内容的人都知道，这是门手艺，需要慢慢练。也需要心力，这得慢慢磨。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而公众号又改版了，视频号有新玩法了，其他平台有新的措施了，都不要紧。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为所有的改版、玩法、措施，我相信最终都是鼓励那些愿意长期做内容的人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果有长期主义的精神，那么所有的变化，就都是利好。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">最后的话</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最后，回到一开始的问题：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">未来要怎么判断，怎么选，怎么做？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/107936912762b001662e9588.09268839.png" title="“先想清楚大事，再躬身入局。”" alt="“先想清楚大事，再躬身入局。”" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在线化的力量，从左向右。体验化的力量，从上往下。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这张拼图上，每一个格，都有成功的人，也都拥有机会。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只是，看清全局之后，你决定进入哪一格。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">想清楚大事。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">想清楚之后，躬身入局。</p>
