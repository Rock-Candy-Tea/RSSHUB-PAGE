<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E6%96%B0%E8%8C%B6%E9%A5%AE" target="_blank">新茶饮</a>开始上演“有生之年”系列了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除却裁员、撤店与食品安全等负面信息，纷扰的行业已经很久没有6月30号那一日的热闹了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>逢腾讯大热古偶剧《梦华录》收官之际，6月30日，喜茶率先合作该剧推出联名产品，IP定制饮品与相关周边，上线首日便售出近30万杯。同日，奈雪的茶上市一周年，官方宣布，将面向用户推出虚拟股票游戏活动，跨界“元宇宙”，引来资本市场一时侧目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>头部两大玩家久违的热度之外，行业又破天荒地迎来一位新玩家：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月30日，肯德基旗下独立茶饮品牌“爷爷自在茶”首店在苏州正式开业。门店装潢与产品特色主打江南风情，主要售卖茶饮与点心小食，单价并不算贵，饮品中杯大多15元上下，大杯也鲜少超20元，走的是中端路线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/142504348562cc2fc2ba7ca9.32940163.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><em><span>图源爷爷自在茶官网</span></em></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新茶饮赛道在当下呈现出了一重奇异的景致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，头部两大玩家喜茶与奈雪的茶困于亏损，不断寻求新突破；另一方面，还有层出不穷的新品牌接连冒头，斩获融资。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但更重要的，是各界巨头以各种姿态尝鲜分羹，从火锅品牌呷哺呷哺、海底捞与巴奴，到中式快餐乡村基、老乡鸡，及至中华老字号稻香村、同仁堂，跨界新茶饮，俨然成了潮流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>肯德基仅是试探者中的一个，显然也不会是最后一个，进入下半场，这场贴身肉搏战远没有结束。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/176024389862cc2fc3721871.09261654.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><span> </span></span><span><strong><span>跨界试探</span></strong></span><span><span> </span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新茶饮赛道的后来者那么多，还是有必要单说肯德基。或者说，当肯德基这样的超级玩家都躬身入局，于行业而言，意味着什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>爷爷自在茶其实没有太多肯德基元素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>店铺开在苏州高新区的绿宝广场，装潢风格以清新绿色系为主，定位江南风情，主打产品为“创新米乳茶”与“水果茶”，拥有独立的点单小程序，除门头上的一排黑色小字“肯德基旗下创新品牌”，外观上与肯德基没有丝毫瓜葛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关联在内里。点单小程序中的经营公示显示，它隶属于苏州肯德基公司，会员积分体系也归在肯德基WOW会员内，配送由肯德基的自有团队完成。具体的产品中，也有油柑茶等部分重叠。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/185958800062cc2fc395a1e4.06883440.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><em><span>爷爷自在茶点单小程序</span></em></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>普遍认为，国内的茶饮市场“三步走”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最早是粉末与香精的劣质混合时代；进入2007年以后，来自台湾的CoCo等品牌采用茶叶、水果等真材实料，向前推进一步；而后，是以喜茶为代表的新品牌们，用水果、芝士和好茶，打开了“新茶饮”时代的大门。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据行业相关报告，这个新茶饮3.0时代，门店已经突破37.8万家，预计到2023年可达50万家，品牌化、连锁化占据了主流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>肯德基赶上了竞争最激烈的时候。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>包括绿宝广场在内，以及龙湖狮山天街、金鹰国际购物广场，它们共同组成了苏州高新区的核心商圈，周边住宅、学校林立，人流量可观，爷爷自在茶的位置却并不讨巧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本地人西西告诉新零售商业评论，在她的印象里，那个位置开的店几经易主，多是茶咖品牌，最近的则是伏见桃山，没有更名前还叫汴京茶寮，换装后不久，再次重新装修更名为伏小桃，最终也没躲过关店的命运。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管绿宝广场拥有相当的人气，但爷爷自在茶并不好找。西西解释，购物中心正门朝西，面向主通道，是人流量最大的入口，而爷爷自在茶处于偏僻的北边一侧，较少人通过，即便从商场内部走，也需要穿过两排长长的餐饮区。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“路过可能会去尝一尝，但专门去就算了。”西西忍不住吐槽。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她的选择的确更多——单绿宝广场内，就有喜茶、CoCo、蜜雪冰城、一点点、悸动烧仙草、百分茶和同样定位江南风尚的茉沏等多家茶饮品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>店越开越多，年轻人也越来越倦。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近年来，</span><strong><span>除却赛道玩家接连入局，大量跨界品牌也来势汹汹。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一类是</span><strong><span>火锅品牌</span></strong><span>。小龙坎火锅做龙小茶；渡娘火锅做渡娘的茶；巴奴推出和奈雪的茶等头部品牌同款的鸭屎香奶茶；海底捞试水奶茶自助；呷哺呷哺独立出的茶米茶，甚至依靠口碑在行业争夺了一席之地，引来云海肴云南菜的模仿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/186581740562cc2fc3c32017.12104407.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二类是</span><strong><span>中式快餐</span></strong><span>。和府捞面推出柠檬益菌多、生榨椰浆和山茶花轻乳茶；老乡鸡做鸡笼香柠檬茶和清新生椰乳；乡村基则有玫珑瓜气泡水与清爽柠檬茶；江浙地区的快餐品牌老娘舅还上线了芒桃淳及梅柚淳，一家更比一家强。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三类为</span><strong><span>“真跨界”</span></strong><span>。2021年，广州高铁推出高铁奶茶，日均售出3000杯；美妆集合店HARMAY話梅推出手打柠檬茶品牌introlemons，首店开在上海安福路；老字号同仁堂继咖啡之后，再造制茶司。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相较而言，肯德基不算那个最“新鲜”的，影响力却足够大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据百胜中国官方网站公布的数据，截至2022年3月底，肯德基在中国1700多个城市开设了8400余家餐厅，倘若以两万家门店的中石油跨界做咖啡对标比较——一年卖出1个亿，它铺开的规模太过庞大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>眼下，爷爷自在茶以独立品牌运营，没有相当的影响力，但于竞对而言，</span><strong><span>它的可怕之处在于——退可守，进可攻。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/59580782662cc2fc3ef3ed4.93809298.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 变是永恒 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>餐与饮的孪生关系不必赘述。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>疫情以来，依靠微博运营、创始人人设打造等营销方式，老乡鸡成功出圈，赢下一定市场口碑，6月，再度推出鸡笼香柠檬茶、生椰乳等茶饮产品，售价对标蜜雪冰城，走薄利多销路线，引起一阵轰动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>餐饮企业跨界饮品，在于迎合消费需求，补足品类，建立品牌壁垒，更重要的是探索年轻化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>百胜集团进入中国数十年，同样囿于该问题。目前，这家集团在国内依然严重依赖肯德基和必胜客两大业绩支柱，就在今年上半年，其探索的中式快餐连锁品牌东方既白在悄无声息中停止运营。2022年一季度，百胜中国营收增速大幅放缓，净利润同比下跌57%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>疫情叠加行业竞争因素，在品牌老化的达摩克利斯之剑下，百胜中国孵化出新兴品牌COFFii&JOY、墨西哥餐饮品牌塔可贝尔，并与咖啡品牌Lavazza合作，无一不是出于旗下业务板块多元化和年轻化的诉求。爷爷自在茶也是如此。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/30742244262cc2fc41d7e12.50166877.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>中国连锁经营协会发布的《2021新茶饮研究报告》中总结，</span><strong><span>行业进入3.0时代，是“茶+奶+水果+文化”的综合发展阶段，作为资本化的关键年，头部品牌在联合营销、数字化营销和IP营销等方面处于引领状态。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>爷爷自在茶开业当天正值周四，门店排起了长队。一名苏州当地的小红书博主告诉新零售商业评论，她专门驱车1小时赶来，并且足足排队等了1小时，选了包括“糯糯桔红糕米乳茶”在内的两款招牌产品，总结下来，口味算“普普通通，并不惊艳”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“如果住在附近，来尝尝可以，但专程就不必了。”该博主表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费者的口味越来越严苛的背面，是行业早已脱离了味道之争，变为了<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E8%90%A5%E9%94%80" target="_blank">品牌营销</a>、文化IP能力之争，而后者，也已进化到了2.0时代。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月19日，#蜜雪冰城黑化#冲上微博热搜榜，不少人发现，在外卖等线上平台，原本白色的雪王统一变成了黑色，蜜雪冰城官方下场玩梗“雪王新皮肤”，旋即被证实，蜜雪冰城推出了新品桑葚果茶，这场新品营销战全面告捷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>单2021年，奈雪的茶就以平均3.5天上新一款新品的速度，共计推出了105款产品，其中，以油柑为代表的小众水果是灵感来源之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当蜜雪冰城将营销“卷”到了新品发售，也意味着如今的新品出圈，单单依靠小众猎奇，已非真正的流量密码。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喜茶与奈雪也在几天前分别贡献出了营销方式中的“第一次”，即首次联名热播剧，还是同一部。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月30日，喜茶联名腾讯视频S+项目《梦华录》推出限定产品“梦华茶喜 · 点茶”“紫苏 · 粉桃饮”，上线首日售出30万杯，单店最高销量近1000杯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一名排队抢到喜茶联名新品的消费者直言，产品算不得好喝，抹茶味道被大大稀释，只是图联名的热度尝鲜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这场营销之战，喜茶和奈雪甚至被外界戏称为“斗茶大赛”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喜茶还在广州、成都、北京和深圳等地选取了四家门店改造成了《梦华录》主题店。</span><span>相较之下，奈雪的茶姗姗来迟，在该部影视剧收官之际，仅在深圳推出了限定产品及糕点，被批没诚意。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>占尽天时、地利、人和的优势才得以拿下一局，新茶饮，真的难。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上个月，行业的另一则消息是，继进驻重庆之后，长沙的网红品牌茶颜悦色将进军江浙市场，首站定在南京，预计8月中旬正式开业，同步启动两店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这家依靠低调内敛的形象、高质量的产品取胜的二线地域品牌，两年前的声音还是“不轻易走出长沙”，仅仅两年时间，在猛烈的竞争之下，也不得不“世俗地”加快扩张进程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说从长沙到深圳、武汉的过程中，茶颜悦色还一直是消费者翘首以待的“白月光”，以重庆为始，因为服务、营销方式被诟病，它在逐步失去光环。它本身的负面事件是诱因，但更重要的，是层出不穷的新网红店，消解了旧网红的意义。</span></p>
