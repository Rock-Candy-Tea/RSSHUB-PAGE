<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>距离初代剧情号大V@陈翔六点半开启<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>首秀已经过去了3个多月时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5月18日晚上8点，@陈翔六点半 在抖音开启直播带货首秀。当晚，主播角色由团队的两位主创人员陈翔和朱小明承担，整场直播持续了3个多小时、共上架了以食品饮料、日用百货为主的44款产品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1054077993630ce77edb6162.63962267.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@陈翔六点半<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%9B%B4%E6%92%AD" target="_blank">抖音直播</a>间截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首秀的成绩并不理想。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据第三方数据，虽然有550.7W人次观看了这场带货首秀，<strong>但当晚@陈翔六点半 的销售额只有184.4W，UV价值仅有0.33</strong>。很显然，这份成绩远远无法和其粉丝体量、账号影响力相匹配。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/607714308630ce77f337396.12930242.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@陈翔六点半抖音视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在首秀之后，@陈翔六点半也正式踏上了直播带货之路。<strong>据第三方数据，近90天内，账号共进行了26场带货直播，总销售额为3607.3W，场均UV价值为0.43。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得关注的是，自8月11日进行了一场总时长约为8小时的直播带货之后，截至目前，@陈翔六点半已有近20天时间没有开播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为很早之前就已火遍全网的大IP，@陈翔六点半 创作的剧情内容已然经过了时间的沉淀和检验，其影响力毋庸置疑。而放眼至整个大环境，跟随行业变迁、早就开启直播带货的剧情号大V不再少数，典型的如@大狼狗郑建鹏&言真夫妇、@疯狂小杨哥、@祝晓晗等。相较之下，@陈翔六点半 的直播带货之路开启得已稍显迟缓。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>剧情号的尽头也是直播带货吗？“坚守”多年后最终选择走上这条道路的@陈翔六点半能否复制同类型达人的成功转型？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>走红9年，@陈翔六点半的根基有多深？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>@陈翔六点半 绝对算得上是剧情赛道里的初代大IP以及长寿IP。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2014年1月初，@陈翔六点半 在腾讯微视上发布了第一条短视频，以迷你剧的形式讲述了小人物的生活囧事，内容短小却笑点不断。随后，这个诞生于云南的搞笑账号迅速走红于网络，在多个平台上都积聚了大量人气。<strong>2016年时，开始做QQ空间的@陈翔六点半曾创下过3个月涨粉400万的成绩，且发布的多条视频都收获了过亿浏览量</strong>，这份成绩即使放到今天依然也不容小觑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而随着快手、抖音等短视频平台的崛起，@陈翔六点半的受众也随之进一步扩大。据统计，时至今日，已经走过9个年头的@陈翔六点半在<strong>抖音上的粉丝量为6784.9W，快手粉丝量为2373.3W，微博粉丝量为441.3W，B站粉丝量为208.3W，全网粉丝量已超过1亿</strong>，生命力之强劲有目共睹。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1836827609630ce77f78cd48.40809379.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>能够在剧情赛道中“屹立”中如此之久，自然离不开账号别具一格的内容特色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回望@陈翔六点半 这些年发布的作品，每个视频均是一个单独的剧集，采用高清实景拍摄，<strong>故事内容都是以现实生活为底色，围绕着普通人的日常生活、喜怒哀乐以及社会热点展开戏剧化改编，内容风格主打轻松搞笑、讽刺、接地气</strong>，短小精悍的剧情中往往会有一个又一个让人意外的反转，视频时而能让用户开怀大笑，时而又能引起反思触动，编剧的脑洞和功力在其中悉数展现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1378894264630ce77fb536f0.08717760.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@陈翔六点半抖音视频截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在内容之外，@陈翔六点半的演员团队无疑也是账号持续吸粉的灵魂所在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们目前能在账号视频中看到的主要角色如猪小明、蘑菇头、球球、闰土、妹大爷、茅台、王炸等，几乎都是有着专业背书：</span><span>导演陈翔，毕业于云南大学影视专业，毕业后一直在电视台从事影视类相关工作；闰土，毕业于云南艺术学院导演专业，表演作品曾多次获奖；茅台，云南话剧院国家二级演员；妹大爷，国家一级话剧演员，曾经出演过多部电视剧……</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/173436790630ce77ff0e5f6.41538819.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@陈翔六点半抖音视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些演员的颜值或许无法和明星相提并论，但在专业程度上却毫不逊色。<strong>在他们真实、自然、具有强代入感的演绎之下，故事本身的情节张力进一步被放大</strong>，而在@陈翔六点半 日复一日的持久更新中，用户不仅对这些演员逐渐产生了一定的情感依赖，对账号的认可度也随之提升。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>转型直播带货，</span></strong></span><span><strong><span>@陈翔六点半 能否复制@疯狂小杨哥？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管现在仍有不少粉丝在视频评论中“抱怨”@陈翔六点半 为何走上了直播带货这条道路，但熟悉行业的人或许并不意外他们做出这样的选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这几年来，抖音上的短视频内容型达人转型直播带货已成风潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音商业化初期，红人最主要的变现手段主要以广告为主，在抖音商品橱窗上线后，挂购物车“种草+带货”成为了广告主的广泛选择，当时最火热的红人即使报价很高排期也能排到一个月以后。而如今，随着抖音对直播赛道的持续加码，加之广告主削减品牌预算，且更为强调效果转化，内容创作者的种草广告收入也相应地被缩减，<strong>与之相对应的，回馈更快、更高的直播带货无疑成为了众多内容型达人的必选项。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而对于剧情达人来说，通过短视频进行广告变现时，在品牌匹配度、广告植入形式上，竞争力似乎也渐渐输给了一些具有种草性质的账号，<strong>从成本、收益的角度来看，身处激烈竞争赛道的剧情达人似乎更难抵挡直播带货这阵浪潮的“<strong>侵袭</strong>”。</strong>而</span><span>@陈翔六点半 开启直播带货的考量或许也正源于此处。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但残酷的事实是，对于所有的内容型达人来说，转型直播带货都不是一件容易的事情。</span></strong><span>即使有着9年时间积累下来的人气基础，@陈翔六点半 也在这条新赛道上感受到了明显的水土不服。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在5月18日的带货首秀中，作为主播的猪小明、陈翔显然要比视频中拘谨许多，整个直播过程中，除了主播在介绍产品时经常傻傻分不清楚外，还有诸如“直播开始时工作人员还在测试”“没有放小黄车”等意外状况出现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了这些“不专业”之外，直播间产品与粉丝画像的“不服帖”也是造成首秀成绩不理想的一个重要原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据卡思观测，当晚直播间上架的产品以食品饮料、日用百货为主，而直播间用户则以24-20岁的男性用户为主，他们在直播间对于这类产品的消费欲望并不会很强，<strong>也正因此，直播间内更多用户是在“捧人场”而非“捧钱场”，带货数据低迷也就不足为奇。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在之后的20余场直播中，尽管承担起主播角色的演员们在慢慢熟悉直播间的带货节奏，并有意为直播间制造类似小杨哥直播间的娱乐效果，但整体看来，直播数据并没有明显起色。<strong>以最近的一场直播来看，8月11日，@陈翔六点半共进行了两场直播，总时长为8个多小时，累计销售额只有280.1W。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1295468631630ce78046ad04.11899441.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@陈翔六点半抖音视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>剧情号转做直播带货并非易事。放眼抖音，如今能够做到出类拔萃的也只有@疯狂小杨哥和<span>@大狼狗郑建鹏&言真夫妇为代表的屈指可数的</span>几个账号。这两个账号<strong>前者以趣味性、娱乐性极强、且难以被复制的“反向带货”闻名遐迩，后者则因磨炼出了极其专业的主播素养而备受信赖。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据第三方数据，<span>@大狼狗郑建鹏&言真夫妇</span>近30日内进行了10场带货直播，销售额高达2.3亿，@疯狂小杨哥近30日内进行了9场带货直播，销售额更是高达4.48亿。两个账号取得的成绩放至整个带货赛道中也毫无疑问是佼佼者般的存在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和美妆、美食、运动健身等能沉淀到精准的垂类兴趣粉丝的账号相比，剧情账号的粉丝构成相对比较“松散”，<strong>他们关注账号的原因更多的是出于娱乐需求而非实用需求，加之剧情内容呈现的人设和直播间呈现的主播人设“割裂感”相对更重，粉丝往往无法适应这种转变</strong>，所以在直播间下单的概率本身就相对较低，这也造就了不少剧情号转型直播带货后的“不如意”，而能够复制@疯狂小杨哥和@<span>大狼狗郑建鹏&言真夫妇</span>的剧情账号更是微乎其微。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于@陈翔六点半 来说，沉淀已久的粉丝基础是账号的最大优势，但在直播带货之路这条路上，他们还有很多功课要做。结合着卡思此前对@疯狂小杨哥和<span>@</span><span>大狼狗郑建鹏&言真夫妇</span>的直播分析，@陈翔六点半 的直播间目前或许有三个明显的短板是要补齐的：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一是承担起主播角色的出镜演员们要快速培养起自己作为主播的职业素养</span></strong><span>，要能让用户感受到他们带货的诚意和专业度；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>二是要结合自己的粉丝画像、粉丝特性，优化选品</span></strong><span>，提高产品品类与粉丝的契合度；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三则是要结合账号的优势，寻找并打造更适合直播间的专属风格，保持账号整体风格的统一性，深度激活粉丝。</span></p>
