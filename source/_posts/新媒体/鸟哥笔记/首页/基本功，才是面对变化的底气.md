<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，直播，有了些变化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>我看到有两个直播间，很特别。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>一个是俞敏洪和他的<a href="https://www.niaogebiji.com/tag/%E6%96%B0%E4%B8%9C%E6%96%B9" target="_blank">新东方</a>团队，做的东方甄选直播间。</strong></span><span>双语直播，有文化的直播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>戏称自己像兵马俑的“主播”董宇辉，一边教英语，一边卖牛排，一边讲天文地理，一边卖书。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>瞬间火了，出圈了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播间从百万粉丝，到千万粉丝，只用了一周。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上周六，我们第一时间就找到东方甄选团队做了一场专访，揭秘他们是怎么做直播的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>还有一个，是顾家家居的直播间。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月17日</span><span>他们做了一场直播，和央视合作，主播邀请了朱广权和郭若天。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>顾家的伙伴说，这场直播的效果很好，观看人数超过1800万，总订单量近1.8万个。</span><span>预计6月17日至6月19日，线下关联销售能达到1.5个亿以上，后续还有大量订单会逐步转化为线下零售额。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我一直比较关注家居行业，到底该怎么做直播？</span><br><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟，家具是低频，但高价的商品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>客单价这么高，如何能让消费者在直播间就下单呢？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我很好奇，顾家是怎么做的，有没有什么新奇的大招。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，我也找到顾家的伙伴聊了聊。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>聊完之后，我有很多收获和思考，分享给你。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/113926470662832e66aa5709.61810445.png" title="基本功，才是面对变化的底气" alt="基本功，才是面对变化的底气" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>决策难度越大，需要的信息丰富度越高</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>首先，关于高价的商品，怎么解决决策难度大的问题。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>客单价越高，需要的信任度就越高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>想要获得信任，就要提供足够丰富的信息，给消费者做决策用。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前，我们用文字、图片，展示信息。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来，有了视频、直播，你可以展示的信息就更全面、更丰富了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我举个例子，老王买了一套小户型住宅，他要配一套沙发。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他只看到沙发的图片，和说明文字是不够的，他还需要知道，这沙发的承重能力到底怎样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>那么，你怎么给他展示呢？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，在直播间，用一个200斤的重物反复在沙发上做冲压实验。沙发安然无恙。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老王还想知道，这沙发占不占空间啊，最好能“零靠墙”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么意思？就是沙发和墙壁，要无缝衔接。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一点，你用图文也是讲不清楚的，或者你说了，只要老王没眼见为实，他也不信。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播时，你可以直接把摄像机怼在沙发和墙那儿，果然，没有任何缝隙。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，老王可能还有一个顾虑，就是这沙发，是不是正品啊？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>点开直播间一看，官方的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到这儿，老王决策需要的所有信息，都在直播间获得了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，“啪”一下，他就下单了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为增加了信息的丰富度，信任度也增加了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于信任问题，顾家有什么特别的做法吗？我问顾家的伙伴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顾家的伙伴说，要持续获得消费者的信任，只做到这些还不够。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>知其然，只能让消费者心动，知其所以然，才能买的放心。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为直播间看到的是成品沙发，成品家具，至于是怎么生产出来的，他们不知道，也没有渠道去了解。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>怎么办？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在直播间，带大家看工厂。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>我们常说外观可以山寨，坐感不能模仿。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为坐感靠的是科学的设计，是支撑系统。市面上外观好看的沙发越来越多了，这时候，真正吸引消费者的还是底层的东西。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如沙发的坐感，比如，高颜健康等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>把工厂搬到直播间，就能让消费者看到工厂的生产环节，沙发、床、全屋定制，都是怎么生产出来的，用的是什么材料。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，我们知道床垫有一个耐压次数测试，它能承受身体几万次的滚来滚去，但用户只知道这个结果。他可能不知道生产过程，使用了什么设备。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在直播间，他能看到床垫是怎么做滚压测试的，比如模拟人体的床垫滚压测试仪在成品床上翻转，测试床垫表层面料、填充物是否破损等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，在直播间，用户就能看到零靠墙沙发，它内部其实有一个零靠墙功能铁架支撑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就能让消费者知其然，也知其所以然。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但是，只有线上部分，可能还不足以支撑他去下单。</strong></span><span>他可能觉得我面对的是品牌方主播，大家利益是不一致的，信任度就会打折扣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>怎么解决这个问题？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顾家这次直播活动就邀请了大量消费者来线下门店，期间就会有很多互动，产品体验等等。</span><span>这样，就相当于把线上和线下结合起来了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>消费者获取信息的渠道增加了，获得的信息就更丰富，有助于他做出购买决策。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>我们说用户信任的建立，其实是一个长期的过程。</strong></span><span>尤其是高客单价的商品，可以在线下成交的，就在线下成交，还没有达到信任临界点的，在线上随时</span><span>可以触达和刺激，最终形成一个良性的平衡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>线上和线下的不同 ，有一个重要原因，其实是触达用户的方式不同，但它们之间的关系，应该是相互补充，相互赋能的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/126592131362832e66ddac86.53943116.png" title="基本功，才是面对变化的底气" alt="基本功，才是面对变化的底气" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>触点的变迁</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费者的需求一直在变化，面对这种变化，你们是怎么做的？我问他们。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顾家的伙伴说，首先是触达方式的变化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>比如，从街</strong></span><span><strong>边店进</strong></span><span><strong>入商场shoppingmall。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2000年以前，我们就意识到家具一定要进入商场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在那时候，消费者买家具在哪买呢，在建材商场，或街边店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为没有渠道品牌，很有可能出现一种情况，他可能今年在某个街边店买了一个家具，到了明年那家店因为经营不好，关门了，不存在了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>就会遇到一个问题，售</strong></span><span><strong>后怎么办，谁来解决售后的问题。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时候红星美凯龙，居然之家这些渠道品牌出现了，提出先行赔付等等，相比街边店，消费者可能就更愿意去这些商场。因为售后有保障，解决了后顾之忧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那时候，我们就和全国连锁品牌，像红星美凯龙、居然之家、月星家居、欧亚达等等，建立了长期的合作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有些可能不是连锁品牌，但属于当地比较好的商场，消费者很信任的，我们也建立了合作关系，这个事情其实一直延续到了今天，在未来，线下商场也依旧是一个非常重要的渠道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>进入商场，就是通过合作，增加了信任度，帮助消费者减少了决策成本。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了解决消费者的信任问题，我们发现这些家居商场，对我们这个行业还有一个巨大的贡献，就是产业集群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去没有一家品牌可以提供一站式解决方案，但是这些商场把大量的品牌聚集在了一起。消费者在逛商场的时候，在这家挑一点，那家挑一点，他不用跑很多建材市场，街边店，就能在一家商场买齐所有需要的家具。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>另一个触点的变化，就是前置营销。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了购买渠道的变化，我们发现消费者还有一个需求上的变化。比如，他可能今年买了一套房，等到交房的时候，希望有一个完整的设计方案，这就需要提前沟通设计。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是像橱柜、衣柜这些产品，会影响到家里整体的装修风格，设计越提前，时间越充分，效果可能就越好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以我们就一直在研究一件事情，怎么在消费者交房之前，帮助消费者把方案都设计好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2016年开始，顾家开始做全屋定制，通过设计，提前和用户产生联系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，受疫情影响，消费者没法去店里，我们就帮他提前在线上设计方案，通过这种前置营销的方式，每个月也有不错的收入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>还有，比如为用户提供一站式解决方案。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么意思呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就是有的消费者，他买家具，不想到处去东拼西凑。比如，他在找整装公司装修的时候，想把硬装和软装的事都给解决了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但装修公司都有一个特点，就是他很擅长设计，出方案，但产品和供应链可能不是他的强项。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>方案再好，没有好产品肯定是不行的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顾家刚好擅长产品和供应链，所以，我们就和整装公司合作，解决消费者的需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>还有一个触点的变化，就是从</strong><strong>线下到线上</strong><strong>。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着线上直播的兴起，各行各业都在思考，线上到底怎么做，大家也都在布局线上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，你有没有发现一件有趣的事情，虽然很多家居品牌都在尝试直播。可是，到现在，家居行业依然没有打通线上线下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>因为这个行业太复杂了，客单价又高，消费者做决策就会非常谨慎。</strong></span><span>得考虑设计链条的问题，信息丰富度的问题，信任的问题，等等等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这样的背景下，就不能盲目去做直播，如果找个大V<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>，肯定能带来流量，但解决不了信任度的问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为信任不是靠别人的流量来背书，首先你的产品得真的好，然后，你得为消费者提供足够丰富的信息，让他知道你的产品确实好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顾家有19个零售中心，线下有8000家门店，2万多名员工，这其实是我们的优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果能实现线下的2万人每天直播，就能通过不断重复触达消费者，提高信任度。因为他们天天和产品打交道，最了解产品。越了解，就越有说服力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们每个月还会做一次以分公司为单位的直播，相当于把平时积蓄的势能，集中去释放，再形成一个更大的势能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后通过线上获客，线下体验，最终完成购买。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/72621015062832e672b11e3.57050295.png" title="基本功，才是面对变化的底气" alt="基本功，才是面对变化的底气" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>凡是过往，皆为序章</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>我们说，直播行业才刚刚结束序章，进入正片的阶段。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播刚刚兴起的时候，很多人看到了机会，这时候是打法取胜，谁更懂得直播的打法，谁就有机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多进入早的人能做起来，是因为赶上了直播的风口，做好了今天的事情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>渐渐地，越来越多的正规军开始入场。这时候，真正重要的不是打法了，是你的基本功，是质量和品质。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顾家的伙伴说，他们理解的基本功，就是不仅要做好今天的事，还要坚持做好昨天，明天，后天的事情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>那什么是昨天的事情呢？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，卖出去的产品，有没有用户投诉？承诺过的服务，有没有完全交付？用户提的需求，有没有很好响应？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>一件东西卖出去并没有结束，只是刚刚开始，后面还需要完整的交付。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，我们从2012年开始，在做一件事情，叫顾家关爱，主要是给顾客清洗沙发，给床垫除螨。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好多年前，我们去清洗沙发的时候，发现很多用户2000年左右买的沙发和床，甚至还有90年代买的，都维护的特别好，看起来还很新。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这件事让我们觉得很欣喜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/42452186262aeafe62e9210.37072009.jpeg" title="基本功，才是面对变化的底气" alt="基本功，才是面对变化的底气" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>（10年前购买的沙发）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使现在家居市场竞争激烈，清洗沙发这件事，我们一直都在坚持做，包括床垫，用了一段时间以后，也会去给它除螨，全屋定制的柜子用了一段时间以后，上门去调铰链。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实上个世纪，很多公司可能交易结束，跟用户的联系也结束了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但现在大家越来越意识到，对老用户的维护，是一件值得长期坚持的事情。</strong></span><span>可</span><span>能在某个时间，就看到了价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，我们开始做会员系统后，就有一个惊喜的发现，我们转介绍达到了40%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在快消品行业，做会员系统可能是很平常的事情，但在家居行业，基本没人做，因为这个行业太复杂了，而且家具是一个低频高价的商品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>高价低频的商品，最重要的就是转介绍。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>转介绍怎么来，就是要做好昨天的事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但是，还不够，你还要做好今天的事。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>做好今天的事，才能赚到今天的钱。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这就需要不断思考和训练：如何洞察用户的需求，如何建立信任？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，疫情期间，我们在苏州的一个门店，就通过直播+前置营销的方式，一个月的时间给门店增加了400多万的业绩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们是怎么做的呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年315之后，他们发现客流量少了很多，因为疫情，客户不愿出门。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>怎么办呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家决定试试线上，于是就召集了店长和经理，做了一个小区的直播和落地计划，针对某个小区，做了一周的直播+接龙计划。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚开始他们在小区群里直播，大家就觉得很新鲜，很多人围观。但大家也只是看看，觉得还没交房买家具太早了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们还是坚持每天直播，每天直播，给大家答疑解惑。坚持了一段时间，慢慢地，大家态度开始转变了，陆续有人开始咨询，下单。最终，一个月的时间给门店带来了400多万的业绩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>做好今天的事情，能保证你现在活得不错。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但未来要活得更好，你还得做好明天的事情。明天的事，是未来发挥作用的事情，比如，组织能力的提升，人才的培养等等。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年我们建立了双利润中心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>家居这个行业，每个品类都有非常强大的竞争对手，消费者的诉求，也都比较专业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>建立双利润中心，一方面可以保证组织的专注度，同时也能兼顾灵活性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，产品中台，就更专注用户诉求和对行业的洞察，产品开发的精准度和效率就会得到提升。营销前台，就更专注流量获取和用户体验，就更有利于提升用户满意度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>当然，你还要做好后天的事。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你只有做好后天的事，才能赚到后天的钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>外部的环境一直在变，几年之后的市场环境可能跟今天大不一样，所以提前做好准备，具备领先于别人的基础能力就很重要。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在公司里，负责做后天的事情，赚后天的钱的，一般是研究部门。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，我们发现，在未来的生活方式中，家里的茶几用的越来越少了，因为大家需要的空间越来越大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，茶几已经不是一个必需品了，因为使用频率没那么高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但有时候又需要用到茶几。</span><span>怎么办呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>研究院的同事就针对消费者的这个诉求，研发了一款沙发，这款沙发下面是空的，搭配一款适合的茶几。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果有客人，就可以从沙发底下把茶几拉出来，如果没有客人，就把茶几推到沙发下面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>研究院做的就是后天的事情，他们的基本工作逻辑，其实就是洞察未来的生活方式，然后提供技术解决方案，再落实到具体的产品上。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>听完他们的分享，我有些感慨，原来大道至简，就是做好昨天，今天，明天，后天的事情。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时代一直在变化，从短期来看，甚至是今天和明天的不同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果把这条时间线拉长，你会发现，这种变化反而慢了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>不管外部怎么变，你的底盘和基本功始终是不变的，而这恰恰是你应对变化的底气。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>善弈者，通盘无妙手</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为他们靠的是日复一日的积累，是扎马步，蹲梅花桩练就的基本功。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>而越是面对变化，越考验基本功。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，最近东方甄选董宇辉突然火了，但让他受关注的，不是他掌握了直播的打法，憋了什么大招，而是他扎实的基本功。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，顾家进入直播行业，靠的也不是新奇的打法，就是踏踏实实做好昨天，今天，明天，后天的事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们说家具这个行业，消费者的需求在变，家具的外观造型也一直在变，但它的支撑系统始终没变。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从长远来看，真正支撑一家公司的，其实是它的内核，是它的支撑系统，是对品质始终如一的追求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>我们看到的所有变化也好，触点也好，还有过去热热闹闹的直播打法也好，其实都是序章</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着像新东方、顾家这样基本功扎实的企业进入，直播行业的正片其实才刚刚开始。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>凡是过往，皆为序章</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>祝福。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>期待。</span></p>
