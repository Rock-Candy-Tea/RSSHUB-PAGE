                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pm-slice="0 0 []">上半年，不确定性萦绕多行业。因此，今年的半年报披露期，成了考验时刻。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2022年8月19日，<a href="https://www.niaogebiji.com/tag/%E4%B8%89%E5%8F%AA%E6%9D%BE%E9%BC%A0" target="_blank">三只松鼠</a>发布2022年上半年业绩。半年报显示，三只松鼠实现营收41.13亿元，归属于上市公司股东的净利润为8213万元，略有下滑，零食行业也难逃大环境影响。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过，本着不浪费每一个问题的原则，变局往往是新局的开始。三只松鼠的特殊性在于，它正好在今年4月开启了自己的全面转型。此刻，这份半年报成为一个观察的窗口，它将验证三只松鼠的深度调整到了哪一步，又是否可以在持续化解阵痛的过程中走得更远。</p>
<p class="pgc-img" style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/9f2db841480c4447ab737406c6cff652" data-ic="false" data-width="1200" data-height="759" data-ic-uri referrerpolicy="no-referrer"><span class="pgc-img-caption"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>大环境脆弱，何以抗风险？</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不熟悉零食行业现状的人，也许会不知道如何定位三只松鼠的业绩表现。两份券商报告为此进行了铺垫：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="2"><strong>· </strong>开源证券援引魔镜数据分析，2022年6月京东食品饮料行业线上销售额为45.2 亿元，同比减16.0%；其中休闲食品同比下滑14.4%。同期阿里系休闲食品行业线上销售额达51.1亿元，同比降28.6%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="4"><strong>· </strong>万联证券援引萝卜投资数据分析，5月天猫淘宝零食线上销售额为50.69亿元，同比下降12.34%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当不确定性尚存肆虐，稳定就难能可贵，三只松鼠至少守住了底线，而这种支撑力来自它内部。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“不确定性三部曲”作者塔勒布在《黑天鹅》中强调了不可预知的风险如何影响世界运转，而在《反脆弱》中，他认为世界上有一类事物，通过吸收波动、压力和不确定性中的营养，抵抗黑天鹅事件，并让自己变得更好。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">三只松鼠所做的，就是坚持自己的转型战略，构建反脆弱之墙。这个过程免不了阵痛，但却让三只松鼠像真正的生命一样，持续进化。自今年4月宣布全面转型以来，三只松鼠的大部分精力用于练内功，其阶段性成果斐然。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">按三只松鼠《关于全面推进战略转型升级、迈向高质量发展的重要公告》的内容看，其关键点之一是从结束以电商为核心的创业时代，开启以坚果供应链为核心和全渠道、多品牌布局的壮年时代。制造，成为了观察三只松鼠变革的第一站。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在制造端，三只松鼠通过建设坚果示范工厂进入第二产业。在探索坚果产业工业化水平提升的道路上，三只松鼠第一座工厂集生产与发货一体化，以每日坚果为试点，于2022年4月开工建设，采用30万级洁净车间，36套全自动化生产产线设备。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前，每日坚果工厂首批规划的两条产线已于2022年7月进入试运营，并于8月正式投产。“二产”层面树立良好样本，既打破了曾经围绕在三只松鼠“懂营销不懂制造”的偏见，又为源头种植和终端销售建立了强力纽带，将产业的制造命脉掌握在自己手中。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">依托于逐步完善的生产能力，三只松鼠一手紧握坚果种植源头，与400多家伙伴达成合作，在全国各地规模化建立种植基地、采购优质坚果；一手洞察国民坚果消费趋势，坚持让坚果和健康食品普及大众的使命，把做“人人吃得起，处处买得到”的坚果产品写进公司愿景。年销售额10亿级大单品每日坚果和20款亿级单品，正持续塑造消费者心智。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">好马配好鞍，好理念、好产品还需要好渠道连接市场。在渠道端，三只松鼠与546家经销商伙伴建立长期稳定的品牌授权经销合作，升级开发2022款经销专供系列产品共38款，以充分适应流通批市渠道及县乡下沉市场的渠道销售场景需求。新开发47款零食独立小包装称重销售产品，以品牌岛柜的形式进驻各类型主力终端超市。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">简而言之，分销改革的依据是市场洞察。要适应线下分销渠道场景和下沉市场为主的渗透布局，盯紧每一个热榜零食散称专柜。为此，三只松鼠还在组织端升级分销事业部组织架构，以一线实际情况为纲。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">补齐传统短板的同时，将长板做得更长，这就是三只松鼠的战略转型的核心之一。面对零食行业的趋势性变化，唯有苦练内功，软实力和硬实力两手抓，才能理解这个市场的未来走向。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>壮年时代，发展求质量</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">和早期流量思维深入人心的行业状态相比，三只松鼠的作为已多了几分沉着，而这正反映出它的“壮年心态”。变的不只是三只松鼠，而是整个坚果市场。休闲零食原本就是大行业、小公司，细分到坚果类目，特点更加鲜明。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">行业大，是因为消费能力充足且成长力强。中金在《零食系列一：细剖千亿坚果炒货行业》中指出，我国2020年坚果炒货零售额约1415亿元，总消费量约530万吨，但2019年，中国人均日消费量仅10.3g，仅为美国成熟市场的1/2。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时，坚果的健康属性令其成为消费主流之一，《中国居民平衡膳食宝塔（2022)》数据显示，每人大豆及坚果类摄入量应为25-35g/天。艾媒数据预测，种子及坚果炒货零售市场规模将从2020年的1415 亿元，提升至2025年的2173亿元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">公司小，是因为坚果产业链牵涉一二三产业全链条，高度考验经营水平，也需要时间深耕。根据魔镜数据，6月三只松鼠、百草味、良品市占率分别-0.5、-0.2、-0.1pct至4.7%、3.9%、3.3%，休闲食品行业CR3环比降0.8pct至11.9%，行业集中度甚至出现下降。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">市场的可塑性，是三只松鼠们的幸运，要获得这份幸运，则务必要夯实发展质量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">农，天下之本。三只松鼠的坚果质量闯关，依然从源头供给开始。概括起来，就是“种植有规模、工艺有技术、行业有标准”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">三只松鼠联合伙伴在云南临沧等6个地区建立了夏威夷果原材料种植基地，“中国碧根果之都”安徽全椒、巴旦木与核桃优良产地新疆，都成为三只松鼠的规模化种植和采购的产地。截至去年，三只松鼠牵头带动供应商累计从云南、广西采购金额达3.5亿元。2019年起，三只松鼠以联合采购的方式，每年从新疆采购的核桃原料超万吨。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">背靠优质农产品，三只松鼠联动供应商共同打造树坚果第四代生产工艺，基本实现无人化自动生产。400余家合作伙伴带动供应链产值超500亿元，也将先进技术用于果品筛选、产品研发升级等。此外，三只松鼠共主导、参与制修订以坚果为主的8个国家标准、10个行业标准、8个团体标准，持续推动行业规范化、标准化发展。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在需求端，当营销让位于品质，变的是流量思维，不变的为消费者的美好生活追求服务。三只松鼠将坚果消费与健康理念相融，积极推动每日坚果等品类更新；渠道建设深入消费场景，做好消费者需要的媒介；通过BC双端切入市场，在企业消费者与个人消费者间架起桥梁，形成用户心智的传递，放大产品势能；打造子品牌“小鹿蓝蓝”，为宝宝零食的精准需求服务，也为自己充实新的增长动力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">十年来，三只松鼠累计销售超过20亿袋坚果，销往全球167个国家和地区。在三只松鼠的愿景中，“人人吃得起，处处买得到”逐渐成为消费者的实际感受，“成为全球绝对领先的坚果企业，打造众多健康食品品牌”开始走向现实。如今，战略转型的方向已明确，“被公众认可—为国家乡村振兴和共同富裕做出突出贡献”，开始让这家企业散发人文光芒。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>面向未来，打开格局</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">放眼国际市场，美国人均坚果摄入量是我国7倍，韩国是我国6倍，这当然是市场机遇。但在此刻，我国坚果原料大部分从国外进口。三只松鼠掌握的改革第一性，就是培养国货农产品，让坚果行业向大而强迈进，成为民族的产业，和振兴乡村的事业。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如在云南，三只松鼠与云澳达坚果开发有限公司携手推广良种技术，建立坚果种植契约基地。目前，合作的坚果种植区域已涵盖云南省近30个次发达或曾经贫困的县区。这是三只松鼠的“一产”路径，也是它与国家乡村振兴、共同富裕战略的共鸣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在“一产”之上，三只松鼠的“二产”以技术标准引导行业良性发展为导向。中金在报告中提到，坚果行业的核心竞争要素之一，是以工艺技术改造、自动化等手段对生产提效。三只松鼠踩准了这个节奏，无论是“云造”系统以数字技术改造并赋能传统供应链，还是推动行业打破材料技术壁垒，都体现出以坚果为核心的长期主义探索。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">表面上，三只松鼠的战略转型是以自身发展需求为导向。实际上，三只松鼠洞察了经济发展的大趋势，在聚焦坚果市场的背后，看到了更全面的产业链、更多的参与者所能产生的价值。当一二三产业打成一片，其价值会分给不同的参与者，让所有伙伴共享收益，这才是三只松鼠转型路途中最美的风景，这样创造的价值，也是稳固的、长期的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">志不在一时的营收数据成果，而是在模式、路径、理念的全面提升，诠释了三只松鼠的转型格局。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">元年开好头，三只松鼠转型最根本的意义还是企业心理的变化。当它对内主动调整姿态，对外积极优化经营举措，响应社会需求，正反映出它由一家年轻的“响应式”公司，逐渐成长为成熟的、有内驱力的主动型公司。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">唯有这种态度驱动的战略转型，才能让一家公司在宏观经济的持续变化中，找准自己的位置。转型路上总有泥泞，但方向对了，总会走到终点。</p>
