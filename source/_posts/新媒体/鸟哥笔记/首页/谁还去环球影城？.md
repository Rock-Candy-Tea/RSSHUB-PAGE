<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“感觉三年没见到这么多人了。”7月1日早上，陈平一行三人来到<a href="https://www.niaogebiji.com/tag/%E7%8E%AF%E7%90%83%E5%BD%B1%E5%9F%8E" target="_blank">环球影城</a>，遇到汹涌人群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月2日，传统暑期旺季开始，重新开业一周的环球影城，门票价格从538元涨到638元。也正因此，7月1日，环球影城迎来人流高峰。根据小红书账号@魔法站台 统计，当日入园人数约为1.5万人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/24723930762c3abd5ad0961.62354219.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>环球影城旺季票价 </span><span>来源/环球影城 燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经获悉，6月25日，“封印”55天的北京环球影城恢复运营，初期限流开放。所有游客进入度假区需持48小时内有效核酸检测阴性证明，核验北京健康宝和有效身份证件，并检测体温等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>陈平坐地铁花了一个半小时，早上近11点才抵达环球影城。“人太多了，尤其是‘哈利·波特与禁忌之旅’人特别多，每个项目都要排队40分钟以上，好累。”紧赶慢赶，在晚上7点闭园之前，她们终于把所有项目刷了一遍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管夏季天黑时间晚，环球影城并未更改10:00-19:00的营业时间，因此备受欢迎的哈利·波特灯光秀也暂停，对此，陈平等很多游客都感到遗憾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/118134472962c3abd5e59e38.05457100.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>7月1日环球影城现场 </span><span>来源/陈平拍摄</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>陈平告诉燃财经，她并不是在环球影城官网买门票，而是通过票务渠道商，“团购价”为388元，比官网便宜140元，三人购票省下420元。其渠道商表示，7月2号开始“团购价”为488元，每日限量销售。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，环球影城重新开业，周边系列产业也重新活跃起来。门票销售是其中一条业务线，许多渠道商活跃在各大社交媒体。除此之外，哈利·波特魔法袍租赁、周边产品代购、酒店预订等业务，都有人在经营。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，2022年过去一半，至今各地防疫政策仍未放松。环球影城此次开放，接待的游客大部分仍然是北京当地居民，全国各地的游客还未能进京“打卡”。同时，为了配合北京防疫规定，环球影城仍将限流运营一段时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“从去年10月22日开始，疫情反复、冬奥会等多项因素，进京管控趋严。今年以来全国各地疫情，跨省旅游几乎停滞，对环球影城有很大影响。”旅游博主松鹤对燃财经表示，相对于去年国庆节期间，环球影城如今人流量未能恢复。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>松鹤指出，基本上大部分北京游客至少都去过一次环球影城了，除非是年卡用户，否则638元的票价，普通人不会重复多次游玩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小白在小红书运营了多个环球影城相关内容的账号，他告诉燃财经，许多粉丝表示638元的门票价格太贵，“所以很多游客都来找我们这些渠道买票，多多少少有些折扣。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于此，松鹤认为，环球影城必须撬动起全国的客流，靠北京本地的存量游客，无法实现良好盈利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/1455856016142b5abe0ca06.17059714.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>北京游客玩腻了</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月30日，小何又开始在小红书上更新图文、视频，内容都是她在环球影城拍到的电影角色IP。这是由美国人Caleb Edwards出演的Charming王子，来自电影《怪物史瑞克》中的一个反派角色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“倒不是因为电影IP，我就是喜欢Edwards这个演员，我们称他是北京环球影城最帅的演员。”她说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/41743345262c3abd6496fe1.08050193.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>小何的小红书帖子 </span><span>来源/小红书 燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“<strong>我已经去环球影城玩十几次了</strong>。我还算去得少的，一些微信群友去了二三十次，天天泡在那里。”小何告诉燃财经，去环球影城的前十次，她都跟普通游客一样认真刷项目，后来玩项目没有新鲜感了，就喜欢看表演、跟演员互动等一些比较轻松的活动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Edwards正是她现在去环球影城的原因。“我在小红书私信Edwards，问他的演出时间。我之所以6月30日去，就是因为那天轮到他表演这个角色。”据她介绍，Charming王子是北京环球影城特有的IP，由三个演员表演，Edwards是其中之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以能去环球影城玩数十次，是因为小何和一众游客，于去年“双11”购买了冬春漫游卡，分别有1288元和1588元两种级别。小何购买了1288元的半年卡，其规定周五、周六及部分节假日不能使用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据悉，由于环球影城从五一小长假开始闭园了近两个月，此次重启，也将这批冬春漫游卡有效期顺延至2022年8月15日。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>疫情之前，小何去过美国、日本和新加坡的环球影城，尤其是新加坡的她去过五六次。她评价道，北京环球影城的项目定位上，比较针对低龄群体，而从项目刺激程度来看，也相对保守一些。“<strong>我现在如果去，主要也就是看Edwards了</strong>。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>住在北京门头沟的依依，也持有冬春漫游卡。6月28日，她跟丈夫一起开车到环球影城游玩。“我一共去8次了。我们过去单程要两个小时，比较麻烦。”她告诉燃财经，自己也有相关微信群，其中住得近的群友常常去玩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次游玩体验，依依感受不太好。6月28日是环球影城重启第四天，也是工作日，人流量并不算大，但因为防疫规定，项目也实行限流。只有同行的人才能坐同一排座位，导致一些座位空置，排队时间被延长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“来之前观察了两天，看排队时间才10分钟左右，开心地冲来了。没想到排队时间都是30-60分钟，如同一万人的状况。”依依说道，由于抵达环球影城已经是中午，而晚上7点就是闭园时间，玩项目很匆忙。此外，园区内很多餐厅在7点前就关闭了，他们玩完项目已经无法用晚餐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在依依看来，环球影城是挺好玩的，但项目形式比较单一，基本都是5D体验，项目数量也还比较少。“拍照的电影IP出现时间也比较短，还很分散。我比较常看到小黄人、啄木鸟（</span><span>伍迪和温妮</span><span>），但《功夫熊猫》的角色就很少碰见。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>依依表示，尽管半年卡延期了两个月，但暑假可能只会再去一两次。“去年寒假陪我妹妹去过，一个项目可能要排队两个小时，太累了。加上暑假又热，我不会再去那么频繁。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她说道，如果半年卡到期，她之后也还会再去，但可能会因为门票价格犹豫，“毕竟那些项目都玩过很多遍了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/9082723746142b5ac73c549.43567204.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>周边“丰富”产业</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>北京环球影城重启，周边生意人也陆续活跃起来，淘宝商家小白就是其中一员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年9月环球影城开业之时，小白和一位朋友拿到提前入园的体验门票。玩了一趟下来，小白马上发现商机，“里面的东西价格挺贵的。我们喜欢哈利·波特的魔法袍，买了两套，一套下来1800多元。我们就想到，可以做魔法袍、法杖这些周边的租赁生意。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，他在环球影城旁边租下一间门面做魔法袍租赁工作室，开了两家魔法袍租赁淘宝店，并开始运营一批小红书账号，以吸引客群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管只有两个人合作，他们坚持每天更新小红书，主要包括预计环球影城每天的入园人数，以及一些游玩攻略等。燃财经获悉，他们运营的一批账号中，@环球影城小喇叭 已经有2.3万粉丝，@环球影城数据站 已经有1.3万粉丝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/111676427762c3abd78ea3c3.44228407.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>小白的魔法袍租赁帖子 </span><span>来源/@环球影城小喇叭 燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小白告诉燃财经，来工作室租赁魔法袍的消费者，一半来自淘宝店，一半来自小红书。去年国庆节，环球影城人流量大的时候，他们每天可以租出去200多套魔法袍。但现在的生意，就不尽如人意了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“冬天的时候生意还行。后面就老有疫情，人流不行。而现在夏天也是个事儿，因为太热了，人们租魔法袍的意愿不强，冬天其实大家套在羽绒服外面还好。”小白介绍道，今年外地游客很少，“<strong>因为我们有酒店配送服务，但今年都是当天来取的，基本都是北京本地的</strong>。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小白分析道，魔法袍的需求量其实不大，可能只有1%的人会租赁，比如环球影城1万客流，可能只有100-200人会租赁魔法袍。他透露，重启以后，6月30日订单开始多一点，能有50多单。“我的流量算是大一点的，很多身边同行，他们可能一天只有几单。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿瑞也在做环球影城的魔法袍工作室，同时做环球影城的周边代购、门票代销、酒店预订等。相比小白，他只是做点副业的心态，也没有引流渠道，“去年好的时候，一周能租出去10套。今年一周租个两三套已经挺好的，经常一天一套都租不出去。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>魔法袍租赁生意利润空间不算大。由于正版魔法袍价格1800元一套，而线下租赁门店也需要租金。小白透露，他的工作室房租每月超过1万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿瑞主业是烘焙师。去年，他也曾在环球影城的甜品店上过班。也因此，他获得一些内部渠道。他表示，现在环球影城这么多周边业务里，门票销售反而是利润最高的，“<strong>最近每天赚个六七十元，也好过没有</strong>。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经获悉，目前环球影城门票渠道商较多，“团购票”、“特价票”等各类打折票不少，一般是官方票价的8-9折优惠，有些“特价票”甚至能打五折。燃财经在多个微信群发现，购买此类门票的游客并不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月2日，阿瑞还跟卖水果的朋友一起，准备去环球影城卖桃子。“瞎干，其实不怎么赚钱。”他告诉燃财经。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/4756644006142b5ad08cc02.64114870.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>民宿生存维艰</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>环球影城周边的住宿业务，也值得关注。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月22日，环球影城宣布即将恢复，在去哪儿平台上，北京环球度假区的搜索热度瞬时增长6倍，冲到全国景区第一；北京环球度假区周边酒店的搜索热度也大涨，较21日环比增长了近40%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在途家平台上，6月22日环球影城周边的民宿预订量环比前一日（</span><span>6月21日</span><span>）增长近1.5倍，“环球影城”关键词的搜索热度环比增长两成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，大半年以来的疫情状况，令环球影城周边民宿生存维艰。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿芙是环球影城周边最早一批民宿主之一，她在周边小区租下近20套房子，还在旁边的皇家新村租下一套别墅，都跟房东签订了3-5年的租约，并在前期投入数十万元进行装修。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，在去年9月环球影城开业前一周，北京实施短租住房新政（</span><span>即《关于规范管理短租住房的通知》</span><span>），城市民宿产品基本被禁止，环球影城周边城市民宿主无奈退场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿芙告诉燃财经，由于前期投入较大，转手亏损严重，她选择将小区房全部长租出去，“长租的差价根本不足以覆盖装修成本，但如果直接违约退房，损失更大。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乡村民宿是受政策鼓励的。但阿芙在皇家新村的民宿也没能运营下去，如今也是长租出去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿芙解释道，“疫情一直都有影响，从去年底到现在，单量一直上不去。乡村民宿也就是别墅，投入也很大，每个月单量只能覆盖房租成本，干脆长租出去算了。”即使在去年行情较好的时候，订单量也只能覆盖房租和人力成本，“看不到太多希望。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/143526640362c3abd8215f92.16776022.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>阿芙的皇家新村民宿 </span><span>来源/阿芙 燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不只是阿芙，皇家新村虹宇Art.house的负责人陈虹宇告诉燃财经，她目前也将民宿长租出去了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经获悉，距离环球影城约5公里的皇家新村，如今还有不少民宿。由于都是独立别墅，住宿价格在1000-5000元/晚不等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>途家平台向燃财经提供的资料显示，自2019年以来，皇家新村属地政府在组织形成皇木厂民宿联盟，目前包括一个老朋友、皇家荷苑、KUSAMA雀舍、睡眠研究所Asleep等民宿品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在小红书上，燃财经看到，KUSAMA雀舍、睡眠研究所Asleep都发布了不少“网红试睡”帖子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿芙指出，尽管进入暑期旺季，人流量会增加，但都是北京当地的，对住宿没有要求。“如果没有外地游客，住宿生意做不起来。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/114502495061728032b0fd52.72007648.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>等待全国游客</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月30日，上海迪斯尼也恢复运营，“女明星”玲娜贝儿当天就登上微博热搜，吸引全网目光。对比而言，环球影城重启，却没有激起多少水花。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开业将近一年，环球影城的运营数据略显惨淡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2月8日，北京市商务局副局长郭文杰在北京国际消费中心城市建设新闻发布会上透露，环球主题公园一期2021年9月开园以来，当年实现营业收入16.45亿元，累计接待游客突破210万人次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，北京市文旅局披露数据显示，4月3-5日的清明节期间，北京环球度假区接待3.05万人，也即日均接待游客约1.02万人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但在开园之前，首都旅游集团董事长宋宇说的是，“北京环球度假区预计每年接待游客1000-1200万人次”，由此可估算日均进园人数应该在2.74万人以上，与上述现实数据差距较大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与北京环球影城不同，上海迪斯尼在2016年开园，首年入园游客便突破1100万人次，超过东京迪士尼，成为全球开业第一年接待游客最多的主题公园。2021年上海国际旅游度假区开放五周年发布会上，上海迪士尼乐园运营方表示，园区五年内累计接待游客超8300万人次，实现收入超400亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>松鹤指出，去年10月份，环球影城的人流量还是挺大的，但从去年11月开始，整体人流开始减少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/67426984762c3abd96eed68.76334292.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/视觉中国</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“4月份的时候，我深有体会。本来半年卡规定周末不能去，但4月我好几次都是周日去的。因为官网会根据客流情况调整，游客少的时候，也会开放给年卡用户。”松鹤估计，目前环球影城的客流可能只是去年的20-25%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>松鹤表示，<strong>从客流情况来看，北京环球影城的运营情况并不乐观</strong>。“如果疫情能转好，进京、出京政策能松动，跨省游客能入园，才能好起来。”他坦言，对于北京本地受众来说，五六百元一张门票，靠北京游客重复游玩，无法推动起较大的客流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年在环球影城工作过的阿瑞，也对其现状有所体会。他告诉燃财经，“现在环球影城很多餐厅还关闭着。有些生意不景气，可能不再开了。”他还发现，<strong>短短半年之间，环球影城的员工已经大换血，之前上班认识的人，基本都离职了</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了疫情的客观因素，北京环球影城自身也存在短板。松鹤说道，北京环球影城的项目都是老IP，距今年份大都超过20年，年轻人对此都比较陌生，“但这个改变不了，即使增加过山车等，作用也有限。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年9月中旬，环球影城开业，“话痨”威震天走红。但9月底，上海迪斯尼原创IP玲娜贝儿，作为一个新面孔强势来袭。这只年轻的粉色小狐狸，可谓一举击败“老人”威震天，热度持续不断。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4月初，环球影城与腾讯游戏合作推出《王者荣耀》花车，合作期三个月。5月北京疫情爆发，环球影城闭园两个月，重新开园之时，《王者荣耀》花车已经取消。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“《王者荣耀》花车还是不错的。巡游过程中有情节、有表演，比大家预期的好很多。只推出了一个月，挺可惜的。”松鹤对燃财经表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小何也挺喜欢《王者荣耀》花车，她觉得设计好看、细腻，同时周边产品也挺便宜的，比如30元的冰箱贴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但对比上海迪斯尼，依依觉得环球影城的周边既不好看，价格还贵。由于此前在上海迪斯尼实习过，她能获取到相关信息。“想买纪念品的时候，发现都不是很好看，而且定价普遍高一些。比如水杯，迪斯尼今年的价格是80-100元，而环球影城的都要150元左右，甚至有179元的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，松鹤认为，迪斯尼运营五年，有时间积累的进化，已经进入良心循环。而环球影城刚开始运营一年，时机还不对，难免有种走进恶心循环的境况。“想要盈利，还是要看客流。外地客流起来的时候，环球影城运营情况才会好转。”</span></p>
