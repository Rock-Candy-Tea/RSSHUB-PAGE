<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>被年轻人捧起来的新式<a href="https://www.niaogebiji.com/tag/%E7%BE%8E%E5%A6%86" target="_blank">美妆</a>集合店，正在面临失去年轻人的危机。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>90后女生乔乔对燃财经表示，自己起初是被新式美妆集合店时尚的装修风格所吸引的。“我一般是在等朋友的间隙进去转转、打发时间，随便买点单价较低的眼影和腮红等，价格通常不会超过50元。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但去过几家不同品牌的门店之后，乔乔发现，这些门店的装修大同小异，且无一例外都是“网红风”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>装修风格之外，能试妆也是新式美妆店吸引乔乔的原因之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“第一次去试妆是因为和朋友约饭，出门太仓促忘了画眉毛，临时进了某个品牌的店内。进去之后发现竟然可以‘撸’全妆。”但这种“好感”并没有持续太久，乔乔表示，在之后逛的时候，她发现，试用装或空瓶都很脏，“感觉服务管理不太到位，影响了购物体验，也就不怎么去了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>00后女生璐曦对新式美妆集合店也曾有过一段狂热探店、打卡的时期。但在最初买买买的新鲜劲过了之后，美妆集合店便对璐曦失去了吸引力。“一方面是店里的品牌在网上都可以买到，另一方面则是距离太远，没必要专门跑过去试妆。不过偶尔逛街遇到也还是会进去转转。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>被年轻人“抛弃”后，新式美妆集合店的日子也变得愈发艰难，多个品牌纷纷从2021年上半年开始陷入闭店潮。公开资料显示，THE COLORIST调色师加盟店关闭59家，WOW COLOUR净关店数量超过50家。NOISY Beauty净关店数量近10家，HAYDON黑洞也于近日陆续关闭了位于杭州、上海和哈尔滨的多家门店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关店之外，燃财经注意到，新式美妆集合店头部品牌Wow Colour和THE COLOURIST的母公司均处于亏损状态。其中，Wow Colour母公司名创优品财报显示，2019-2021财年，合计净亏损近20亿元。THE COLOURIST母公司KK集团2018年-2021年上半年，合计净亏损超70亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，新式美妆集合店是2019年才火起来的“新物种”，并凭借高颜值、强体验、价格低、货品全和自助购物等特点，迅速得到年轻消费者和资本的青睐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在小红书上搜索“美妆集合店”，相关笔记超2万篇。“小样天堂”、“学生党必逛”、“美妆蛋墙”、“口红墙拍照”等均成为不同美妆博主笔记中的高频词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/120531866862d0ccb67f5d45.03722814.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/小红书美妆集合店笔记</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>热闹的美妆集合店，不仅成为各大商圈的吸睛利器，也成功得到了资本的青睐。其中调色师、ARMAY话梅（</span><span>以下简称“话梅”</span><span>）和WOWCOLOUR已获得多轮融资，融资规模均达数亿元。此外，NoisyBeauty、喜燃、黑洞等后起之秀也获得了高瓴创投、真格基金等知名投资机构的投资。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，艾媒咨询集团CEO兼首席分析师张毅分析道，过去几年美妆集合店发展较快，有些门店没有达到预期收益，出于降本增效的考量，关店优化是正常操作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>易观分析<a href="https://www.niaogebiji.com/tag/%E6%96%B0%E6%B6%88%E8%B4%B9" target="_blank">新消费</a>行业资深分析师李应涛补充道，新式美妆店目前的现状是因受到疫情、消费者失去新鲜感、外部竞争加剧，以及自身经营模式受限等多重因素叠加作用造成的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但李应涛强调，相对于其它消费品市场，中国美妆市场正处于快速增长阶段。从美妆品牌对线下渠道的刚需属性和年轻人美妆消费能力这两个角度来看，新式美妆集合店还是很有发展前景的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“新式美妆集合店的本质是零售渠道，其核心竞争力无外乎运营效率和服务体验。目前来看，新式美妆集合店同质化严重，而打造差异化竞争则成为突围的关键。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/1455856016142b5abe0ca06.17059714.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>年轻人失去热情</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因工作性质原因，90后女生佳佳经常需要出差，为了减轻化妆包的“负荷”，各类美妆护肤品的小样成了佳佳的必需品。一次出差因为缺少一个克数小的洗面奶，佳佳便在大众点评上搜了“小样”，跟着搜索结果，佳佳第一次接触了新式美妆集合店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进店后的佳佳发现，这里简直就是小样的“天堂”。尤其是对她这种“出差汪”来说，简直太友好了。就这样，第一次进店的她一口气买了“一筐”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这种店太懂年轻女生了。一方面，香水小样都是用原瓶1:1缩小的，这些Q版高颜值的小瓶子很难让人抗拒。另一方面，女生本来就不嫌香水味道多，但正装不仅价格贵，还用不完，小样完美的解决了上述两个问题。我几乎每次去都会买一些带走。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着越来越多的新式美妆集合店走进大众视野，佳佳也打卡了不同品牌的店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“有的美妆店走中高端路线，有的则相对下沉。”佳佳表示，几乎所有的门店内都会售卖一些自己没听过的小众品牌，“相对高端的集合店内推荐的品牌我或许还能尝试一下，但中低端的店我就只敢买自己认识的品牌。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过今年，佳佳仅在一家店的小程序商城里买过两次香水小样，“无法出差，我对小样也没了需求。起初我还会逛逛线上的美妆集合店，但与线下相比，逛线上的乐趣少了很多，我不会像在店里一样，边逛边买最后带走一堆。久而久之也就失去了对集合店的兴趣。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/150322025162d0ccb6d27024.56780398.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/ 视觉中国</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和佳佳一样热情消退的还有璐曦。但不同的是，璐曦“抛弃”新式美妆集合店的原因是“失去了新鲜感”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一年半前，00后女生璐曦经常会约同学去新式美妆集合店逛逛。即使要乘坐一个小时的地铁，也无法阻挡她们跑去打卡的热情。但现如今，她和同学们的热情早已消退，离上一次去探店，也已过去大半年时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>璐曦对新式美妆集合店的了解起源于小红书，“装潢简约、时尚的店面非常适合打卡拍照。尤其是在很多种草贴中，还特意强调了‘可以随意试色’且‘没有店员打扰’。”就这样，璐曦逐一收藏了小红书上推荐的品牌门店，并约朋友挨个打卡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>璐曦至今都记得，第一次打卡线下门店时的惊喜，“店面设计前卫，逛起来使人身心愉悦；美妆品类丰富，能看到很多社交平台上种草的小众美妆品牌，轻轻松松逛上两个小时。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>璐曦对燃财经表示，自己以前也去过丝芙兰和香港莎莎等美妆集合店，但热情的柜姐无形中会增加自己的心理压力，总觉得不买东西过意不去，“逛新式美妆集合店就不会有这种尴尬，不用面对热情的导购，自助试色和选品，体验感舒服了很多。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，价格便宜也是新式美妆集合店吸引璐曦的原因之一，“价格和电商平台基本一样，但却省去了线上购物等待物流的时间。同时，门店还有如‘9元的腮红’、‘10元的眉笔’等特价引流产品。让人很难不买点什么带回去。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但时间一长，璐曦和朋友的新鲜劲儿就过了，也不会再刻意约着跨区去打卡，“现在只有在逛商场的时候见到会进去看看。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和璐曦一样，乔乔也是冲着新式美妆集合店“没有店员打扰、产品丰富多样”而来。但门店的服务管理不到位却严重影响了乔乔的消费体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乔乔表示，一次去美妆集合店买粉饼，结果发现粉饼被大家试得都快现出铁皮了，而且粉饼和粉扑都很脏，几乎看不出原来的本色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我想找服务员帮我找个粉饼试下，结果压根找不到人。”乔乔直言，“没有导购打扰是优点，但管理服务不到位就是缺点了。要是美妆集合店的陈列和管理能做得更细致一些就好了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/9082723746142b5ac73c549.43567204.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>从出圈到关店潮</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管正在被年轻人抛弃，但这些新式美妆集合店，也曾辉煌一时。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年，新式美妆集合店话梅北京三里屯店开业，亮眼的工业风装潢设计，不断吸引消费者进店拍照，并很快在社交平台上引起广泛关注。很多年轻女孩不惜排队数小时只为去店里打卡。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，话梅最早起家于线上，2017年在上海安福路开出了第一家线下门店。但直到2019年，北京三里屯门店的火爆才让话梅真正出圈，也让“仓储+小样”模式的新式美妆集合店走进大众视野。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着话梅的走红，大量创业者开始涌入这一赛道。2019年，KK集团旗下美妆集合渠道“THE COLOURIST调色师”首店开业。2020年，名创优品旗下“WOW COLOUR”首店落地。同年，“HAYDON黑洞”在杭州开业，原小红书高管施启伟离职创立“H.E.A.T喜燃”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在小红书的探店笔记中，“店面有设计感”、“无导购”、“货品丰富”、“低价”几乎成了所有新式美妆集合店的关键词。同时，“39元买海蓝之谜”、“19.9元入兰蔻”等类似标题也成为此类探店笔记的流量密码。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一时间，新式美妆集合店不仅成为年轻人每个周末必逛的店铺，排队打卡和付款的场景，更是成为很多门店的一道“风景线”。据独角Mall报道，话梅在北京400多平方米的门店，仅仅花了一个月的时间就实现了盈利，最高峰时期的月均坪效更是达3.5万元左右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在李应涛看来，这些美妆集合店之所以能吸引年轻人，一方面，国货彩妆品牌快速崛起，这些新品牌对线下传统渠道有需求的同时，又很难进入百货商场等传统渠道。新式美妆集合店的出现，大大降低了国货新品牌进入商场的门槛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，伴随着Z世代逐渐成为美妆消费主力，他们的消费能力更强，也更乐于尝试和接受新鲜事物。而从某种意义上来说，作为传统线下美妆店升级版的新式美妆集合店，对年轻人的吸引力也就更强。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>资本当然不会错过如此吸睛的赛道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据燃财经不完全统计，自2019年12月获A轮融资后的两年时间里，美妆集合店品牌话梅已累计完成四轮资本加持；WOW COLOUR先后获得了赛曼基金10亿元战略融资和创新工场、IDG资本等的5亿元A轮融资，并在其2022战略发布会上再次宣布获上亿美元A+轮融资，短短一年多时间融资3次；2021年7月，THE COLORIST调色师的母公司KK集团完成3亿美元F轮融资，投资方包括京东集团和CMC资本，这也是KK集团的第七轮融资。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费者的追捧、资本的助力，让新式美妆集合店进入了快速扩张阶段。据燃财经不完全统计，THE COLORIST调色师在不到两年的时间里开出300多家直营店。WOW COLOUR的官网显示，其在2020年开店600家，2022年目标开店1000家，同步进驻海外市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，曾是资本新宠的新式美妆集合店，却在短短两年后，就走进了关店潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>KK集团招股书显示，2021年上半年，THE COLORIST调色师加盟店数量从170家缩减至111家，关闭59家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经在大众点评搜索“调色师”发现，THE COLORIST调色师目前在北京的大部分门店显示暂停营业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/55465462862d0ccb7132899.67370069.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/大众点评上，THE COLORIST调色师北京全部门店经营情况</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月4日，燃财经走访调色师新奥天虹购物中心店发现，其关闭的大门上贴着A4纸打印的“设备调试”四个字。附近其它店铺的店员表示，这种状态已经持续大半个月之久。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，调色师北京新中关购物中心店已闭店，门头品牌Logo已被撤掉。悠唐购物中心店、伊藤洋华堂店等均处于停止运营状态。而与调色师同为一个集团的另一美妆集合店品牌KKV也关闭了位于北京王府井的apm门店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知情人士向燃财经透露，7月13日开始，THE COLORIST调色师新奥天虹店已恢复正常营业，且是目前北京仅剩的一家正常营业门店。燃财经就是否是目前北京仅剩的一家正常营业门店向官网留言确认，截至发稿，未收到回复。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，燃财经在近日走访调色师新奥天虹购物中心店的过程中发现，其对面一家名为“得色”的美妆饰品集合店，类型与调色师相近，入驻品牌也以国产及日韩小品牌彩妆为主。在店内，燃财经发现，其折扣区位于极为明显位置且折扣已低至两折，不乏6元的腮红和12元眼影等低价产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，尽管折扣够低 ，但在某一工作日晚七点到八点左右的黄金时段，商场客流量正常的情况下，进入店内的顾客数量只有个位数。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，美妆集合店黑洞HAYDON也在其官方微信公众号发布最新门店营业调整公告。公告称，杭州湖滨in88店、乌鲁木齐富成国际店、西安雁塔万象天地店均暂停营业。界面新闻统计，名创优品旗下的WOW COLOUR门店也在急速缩减，已从巅峰期的300家锐减至135家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/4756644006142b5ad08cc02.64114870.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>窘境与未来</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但有意思的是，尽管现阶段美妆集合店纷纷关闭部分门店，但融资却还在继续。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天眼查数据显示，2022年1月，话梅完成C轮和D轮合计近2亿美元（</span><span>约12亿元</span><span>）融资，分别由General Atlantic、QY Capital领投，老股东高瓴资本、五岳资本、钟鼎资本和黑蚁资本也出现在最新两轮融资的跟投名单中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了话梅，另一美妆集合店品牌WOW COLOUR，也在其年会暨2022战略发布会上宣布再获上亿美元A+轮融资。据天眼查显示，此前WOW COLOUR就已先后获得了赛曼基金10亿元战略融资和创新工场、IDG资本等的5亿元A轮融资。目前WOW COLOUR的市场估值已达10亿美元（</span><span>约合人民币63.4亿元</span><span>）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>多位业内人士分析表示，这一“怪象”与新式美妆集合店持续增长的市场规模不无关系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《2021年中国美妆集合店行业研究报告》显示，2020年中国美妆集合店行业市场规模为419亿元，其中新型美妆集合店占比7.6%。随着新型美妆集合店由萌芽期向稳定发展期过渡，其市场份额占比有望于2023年提升至15.8%，市场规模预期将达130亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李应涛分析表示，除了高速增长的市场规模外，年轻人在美妆方面的氪金能力和目前行业尚没有绝对领先者等诸多因素，也都是资本押注的重要原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但与此同时，李应涛也强调，虽然行业前景看上去很美好，但现实情况却令人堪忧。短期内大量创业者和投资人涌入，导致同质化竞争加剧，店面装修、营销玩法、店内选品等均出现雷同现象，很容易让消费者产生审美疲劳。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如李应涛所言，千篇一律的“网红打卡风”成了新式美妆集合店的标配。燃财经实地探访了北京多家新式美妆集合店发现，迪奥、香奈儿、TOM FORD等大牌是标配，且价格相差无几。而Colorkey的口红、橘朵的眼影盘以及RNW鼻贴等新锐国货也几乎是家家必备。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李应涛进一步补充道，一方面，零售渠道与品牌方是博弈关系，同质化竞争加剧的后果是新式美妆集合店的话语权被削弱。当新式美妆集合店无法掌握产品的控制权和定价权时，就只能“靠打包产品组合做优惠”等方式刺激消费。但这种方式在现阶段，对消费者早已失去了足够的吸引力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/157535453562d0ccb7583164.19638934.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>来源/ 视觉中国</span></span><span><span></span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，线上销售渠道也给新式美妆集合店带来了一定程度的“挤压”。新锐美妆品牌们时而在头部主播直播间里打出“最低价”的招牌，时而在自家旗舰店大搞优惠促销，使得线下美妆集合店只能做到“最全”，却无法做到“最低”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>璐曦对燃财经表示，很多人都是在线下试妆合适后，再去线上购买，原因则是线上性价比更高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我也这么干过，有时候线下折扣力度大，有时候线上更便宜。这可能也是某些美妆集合店不让消费者拍照的原因。相较于担心消费者拍照发朋友圈，他们更担心的应该是怕你拿出手机扫码比价。” </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同质化严重、缺少产品定价权与采购成本的控制权，直接导致了新式美妆集合店“博眼球却不赚钱”的窘境。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以拥有KKV、KK馆、THE COLORIST调色师、潮玩集合店X11四大零售品牌的KK集团为例，近三年半来，其营收虽然不断增长，但亏损却持续扩大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>KK集团招股书显示，2019年、2020年和2021年上半年，KK集团整体营收分别为4.64亿元、16.46亿元和16.83亿元，分别同比增长198.59%、254.92%和235.05%。但从2019年到2020年，KK集团相应的净亏损分别为5.15亿元、20.17亿元，2021年上半年更是亏损43.97亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这一结果，李应涛表示，这是新式美妆集合店的经营模式所决定的。不同于传统美妆集合店的全品类经营，新式美妆集合店通常都聚焦在美妆和彩妆两大品类，但两者都是对品牌效应要求非常高的品类，再加上市场竞争激烈，新式美妆集合店在与各大美妆护肤品牌谈判的过程中话语权越来越小。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“从这个角度来看，新式美妆集合店的品类结构是不如屈臣氏一类的老牌集合店的。屈臣氏通过推出自有品牌来提高自主权和利润率，并逐渐丰富产品种类，使其自身兼具了美妆店、药妆店和便利店等多种功能，也给消费带来更方便的一站式购物体验。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于新式美妆集合店的发展，李应涛分析表示，破局点还是在服务上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，李应涛表示，新式美妆集合店还可以拓展一些对品牌效应要求没那么高的品类，通过做自有品牌来减少对品牌商的依赖，并进一步提升运营效率，为消费者带来更多价值体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>张毅则对燃财经表示，好看的装修能吸引用户一时，但最终还是要回归到产品本身。长期来看，美妆集合店需要从采购渠道、品牌保障、服务保障等多方面把关。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“另外，以往密集开店，会不会有过度营销带来的负面影响，这都需要沉淀和思考。只有通过实际经营去不断测试，美妆集合店才能找到真正适合自身发展的路径。”</span></p>
