<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><span>很久以来，<a href="https://www.niaogebiji.com/tag/%E6%9C%AC%E5%9C%B0%E7%94%9F%E6%B4%BB" target="_blank">本地生活</a>都属于被美团、阿里所把持的天下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到短视频玩家粉墨登场，后来者一次次冲击着这一万亿市场。自此，江湖只剩刀光剑影，再不平静。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>混战之下，巨头夹击之间，<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3" target="_blank">抖音</a>本地生活正策马狂奔。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>220亿GMV，抖音本地生活狂奔</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在定下全年500亿GMV目标后，又一声平底惊雷响起。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>日前，有消息称，在刚刚过去的上半年里，抖音本地生活完成了约220亿元的GMV。这一数字，甚至超越了去年全年都未能达到的目标（200亿）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>新秀的成长总是如此不显山漏水，不经意间便给老牌巨头一个“惊喜”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>细分来看，在今年一季度抖音本地生活GMV为100亿，二季度在110至120亿之间，整体呈逐步上升之势。消息人士称，在已经开始的第三季度，抖音内部GMV目标是每月至少50亿保底。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/120927812662d0b15239ae55.36869208.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在这一“月入50亿”节奏下，年初的500亿目标或许等不到年底便可实现。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>GMV飞速增长的同时，抖音本地生活业务也正持续开疆拓土，战线遍布大江南北。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年一季度，抖音在云南、重庆、天津等地密集拓城，截止6月，抖音本地生活驻扎城市数量达到了33个。到今年年底，其目标是拓展50-55个城市。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当团队驻扎后，抖音则会开启招商峰会，帮助头部商户对接本地MCN机构和代运营公司，以实现业绩增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，从如此大规模动作可以看出，在整体流量见顶的趋势下，</span><span><strong>本地生活已经被抖音视为自身的第三条增长曲线，与美团的战争不可避免。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管旧时代的王者依旧光芒万丈，但抖音的追击同样咄咄逼人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>心动外卖搁浅后，抖音避开了在外卖领域与美团的直接对决，转而积极扩展本地生活类目，以寻求弯道超车。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/113682977562d0b152845d14.58355899.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在本地生活领域，抖音已经包含了综合商超、洗浴按摩、鲜花等十数个一级类目。到店餐饮、亲子、酒旅等是其重点类目。在这些方向上，尽管美团依旧处于优势，但差距并不像外卖般悬殊。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也正是在地域、业务上的扩展，推动着</span><span><strong>抖音本地生活突飞猛进，</strong><strong>由美团、阿里所把持的旧天下正被缓缓撕开一道裂缝。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>本地生活的破局者</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从无到有，抖音本地生活走得并不顺畅，但从有到精，发展不可谓不迅猛。回顾其本地生活之路，源自于2018年发起的暗战。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>彼时的抖音，在本地生活市场还略显稚嫩，正尝试着通过第三方平台搭建生活服务场景，以及为商家提供团购功能。然而商户资源匮乏和营销工具欠缺，成为矗立在抖音面前的两座大山。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最终，抖音的本地生活“初体验”以失败告终，平台相关业务也成为各方诸侯混战的泥潭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，</span><span><strong>风起于青萍之末，浪成于微澜之间。那时没人意识到，正是此次尝试，竟成为后来抖音对美团、阿里所把持天下破局的开始。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/124831558862d0b152d33645.63768324.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>铩羽而归的张一鸣并不愿放弃本地生活这块大蛋糕，于是后来的抖音，基于此展开了一系列的整合与变革，集团人力和战略部门更是被下派，对美团进行深度调研，势要再打一场翻身仗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来的故事大家都很熟悉了，疫情的突然来临，使得百业凋零。作为庞大线上流量载体的抖音，成为各线下商家寻求自救的平台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乱世出豪杰，时势造英雄。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>嗅到机会的抖音也及时跟进，成立专属的本地生活部门，投入大量人力、物力、财力进行市场开发，至此，抖音本地生活的序幕正式拉开。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>也是从那时起，美团与阿里铁幕下的旧时代格局悄然破冰。抖音，更是祭出了自己的秘密武器——直播带货。凭借这一底层变革，抖音要革了美团的命！</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“每个人都可能在15分钟内出名，每个人都能出名15分钟”，安迪·沃霍尔的成名定律在抖音继续被实践着。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137921598062d0b153a36e26.53451291.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无数抖音餐饮商家借着短视频、直播带货的东风，喊着“9块9秒杀羊肉串”的话术，各类探店视频被平台大力推送，小龙虾套餐的火热，带动着抖音本地生活业务开始沸腾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为本地生活破局者的抖音，攻势来的热烈而凶猛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>巨头激战本地生活</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>万亿市场的大蛋糕，觊觎者不只有抖音。首当其冲的，便是其在短视频领域的头号劲敌——快手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手的入局，要明显晚于抖音。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年，快手才开始在特定区域试水，并且直到去年4月，本地生活才被明确为未来发展的主要方向之一。这也意味着，快手正式入局，本地生活的江湖又迎来一位重磅玩家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/31008164362d0b15600ecb0.06104468.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>官宣进军后，快手开始持续发力，逐步增设本地生活流量入口，并新增了到店美食、酒旅、医疗健康服务等15个本地生活行业相关的一级类目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一时间，本地生活的赛道热闹起来。</span><span><strong>由美团领衔、抖、快、阿里等一众玩家追赶的“一超多强”格局产生，各巨头激战本地生活。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为国内本地生活领域的头号玩家，美团一方便占据了7成的市场份额，凭借数百万骑手大军所建立的钢铁洪流，牢牢把持着用户心智和消费习惯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>每个人都想把美团拉下王座，相为谋而道不同。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为美团的老对手，阿里在经过去年的组织升级后，抛弃了过往“单一APP承载所有本地生活服务”的战略，采用“飞高了（飞猪、高德地图、饿了么）”矩阵的方式来服务不同生活场景，将重点放在生态建设上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/109084750462d0b15763c282.54565169.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而快手则选择拉大旗、广结盟，通过与美团、顺丰等合作来提供完成的服务能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为破局者的抖音，有着更大的野心。其一边不断扩张，抢占市场份额，也一边加码物流配送服务，打造生态闭环。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>各方围攻之下，美团面临着前所未有的巨大压力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，尽管江湖硝烟弥漫，美团王朝依旧。</span><span><strong>“多强”逐鹿中原，新时代浪潮滚滚，由短视频巨头引领，本地生活正迎来一场颠覆性的底层变革。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于这场变革将要革了谁的命，各方玩家都要为此忧患。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>庞大的市场容得下群雄割据，但攻守之势正在重塑。</span></p>
