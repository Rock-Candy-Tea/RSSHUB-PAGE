<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">去年3月份，国内职业教育龙头中公教育创始人兼董事长李永新，向母校北京大学提交了一笔建校以来最大的捐款，数额高达十亿元。这刷新了百度创始人李彦宏2018年的6.6亿元捐款记录。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">李永新承诺，以后争取捐赠100亿。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如今看来，那个时间节点可能是中公教育自创立以来最为耀眼的高光时刻。之后，中公教育发展态势便急转直下，今年甚至出现了上市以来首次亏损。财报显示，中公教育2021年营收69.12亿元，同比下降38.3%；净亏损23.695亿元，而前一年，2020年，中公教育净利润还高达23.04亿元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">来自证监机构、深交所的问函纷至沓来。监管机构先后下发文件，要求中公教育对业绩下滑作出解释。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">资本市场也跟着失去信心。中公教育市值从当年年初的2600多亿缩水到了375.59亿元（截止发稿）。2021年以全年累计下跌77.63%的跌幅成为A股跌幅第一。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">甚至出现了关于“中公将要破产”的传闻。中公内部开始出现“缺钱”的迹象。中公教育员工卫明告诉Tech星球，从“双减”过后，公司就变得很难，每次发工资都是卡点发放，尤其是今年，工资发放时间越来越晚，工资条也不再让员工本人确认。之前老师研发讲义还有相应的工资，现在老师们都是免费义务为公司研发讲义，职称也不让评了，“透露着不妙的种种迹象。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不止中公，以公务员考试培训为主营业务的职业教育公司，今年集体陷入生存困境。覆巢之下无完卵，<a href="https://www.niaogebiji.com/tag/%E8%A3%81%E5%91%98" target="_blank">裁员</a>、<a href="https://www.niaogebiji.com/tag/%E9%99%8D%E8%96%AA" target="_blank">降薪</a>、抓考勤、强制休事假等体面与不体面的举措开始接踵而至。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1170739965612f7c0dc06775.73201401.png" title="裁员、降薪、退费难，考公热难解中公华图困境" alt="裁员、降薪、退费难，考公热难解中公华图困境" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">降薪、降绩效，工资发放延迟</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中公教育在一年的时间内，从市值高点跌落，缩水2000多亿。从净利润23.04亿元扭盈为亏，去年直接亏损了23.695亿元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年成了中公教育的分水岭。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中公教育员工赵赵搞不明白，为什么老板李永新可以出手那么阔绰，一掷10亿捐款，却要在员工福利待遇上如此“节衣缩食”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">赵赵在中公教育近5年的时间，只在去年涨过一次薪，今年又全部还回去了。疫情之下，公司开始降薪资，降绩效。赵赵表示，从去年下半年至今，中公已经大大小小调整了三次，最近调整尤为频繁。员工绩效考核也变得更为苛刻，绩效评分，从最高值2分降低到1-1.5分，数字背后对应的是不同的绩效工资。分数越低，拿到的工资也就越低。降薪后，赵赵的薪资与去年涨薪前水平持平，降薪幅度达15%，每月少了两三千左右。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">升职加薪通道也被关闭。中公教育员工吕新告诉Tech星球，此前中公员工每半年可以进行一次晋级，一年固定两次晋级涨薪机会，员工只要通过述职便可以涨薪。但从去年开始，公司晋级述职周期从6个月延长至8个月，然后又从8个月变成了12个月。直至今年默认取消，不再提晋级的政策。近半年，中公内部没有任何晋级公示。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实上，中公教育的不景气早在2020年便有迹可循。吕新表示，此前公司按照公积金上限12%比例缴纳，2020年七八月份开始，公司调低至5%的比例缴纳。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中公教育给员工其他的额外福利并不多。吕新称，入职中公近3年，端午节、中秋节等重大节假日，公司集团层面几乎没有发过福利。圣诞节平安夜，有部门领导自掏腰包，给该部门员工每人发了一个苹果。中秋节，每人一个月饼。而去年，连仅有的一个月饼都没有发。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">员工创收渠道变少了。2020年公司推出的加班制度在当年速生速死，只短暂存在了半年时间。吕新称，那段时间公司鼓励加班，员工加班有加班费，还有荣誉嘉奖，比如公司会推出加班排名榜，整个公司以加班为荣。但后来公司很快就取消了加班费，员工也不再心甘情愿加班。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">今年5月起，中公教育员工年假不再兑换成现金，公司要求员工居家办公期间将年假、以往加班时积攒的调休统统用掉。这意味着，年假无法折现，员工还要居家办公。吕新称，中公教育对于此类通知，从来不会以白纸黑字的形式通知员工，通常是以部门领导口头通知。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，更为被动的是，员工没有权限查询自己的假期余额，需要跟人事申请查询。所以假期天数的决定权在公司手里，员工只能单方面等公司告知假期天数。“我们对自己休假时长完全没有清晰认知”。信息不对等，员工需要跟人力资源部门斗智斗勇。吕新表示，每次询问人力，人力通常不会微信回复，而是采用电话通知。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">真正让员工们感到恐慌的是，现在工资也不正常发放了。4月份的工资直到5月25日才发，比正常发放时间晚了十几天。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2040399738612f7c0ec65369.94912606.png" title="裁员、降薪、退费难，考公热难解中公华图困境" alt="裁员、降薪、退费难，考公热难解中公华图困境" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">强制休事假，居家办公工资减半</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“公考一哥”中公教育陷入亏损泥潭，第二梯队玩家华图教育，也是捉襟见肘。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">华图教育员工张豫告诉Tech星球，目前华图发展状况不是很乐观。最直观的表现就是裁员、降薪。去年12月份，华图开启第一波裁员。华图北京总部职能部门走了将近一半左右，教研等事业部几乎都有涉及，主要受影响的是各地分校。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">员工薪资被降。张豫表示，公司通过各种方式变相降薪，然后各部门领导开小会通知。今年1月份，春节假期前，他所在的部门被要求强制休假了一周。强制休事假，就要扣当天工资，休一周事假，就意味着一周没工资，但员工在家还需正常办公。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“公司方面作出的解释大概就是，公司发展情况不好，强制事假一起共渡难关，等年后就好了。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此后2-3月份，华图不同部门分别进行了强制休假。有的事业部情况好点，就少休两天。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">强制休假不是长久之计，4月份，华图开始实行居家办公，工资减半，但工作量却没减少，公司对员工的考勤也更严苛了。张豫称，居家办公的时候需要开视频会议，每小时截一次图，离开镜头需要报备。去公司上班的不用开视频，一般只有管理干部层会被要求去公司坐班。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">公司政策几乎按月调整，张豫发来一份华图5月份的通知，通知显示，一、从5月1日起，取消居家办公；二、5月开始，所有管理干部和员工的月度薪酬结构调整，基本工资40%，绩效工资60%，其中绩效工资60%跟业绩完成率强关联。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">之前华图员工工资结构是6：4，基本工资占大头。调整后基本工资变少了，绩效还是正常打分，但是需要按照本月业绩完成率乘个百分比。通常，他们业绩是完不成的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">公司上下处处透露着节俭，华图开始从内部减少各种支出。张豫称，去年年底华图20周年司庆和年会，前期公司还打算好好庆祝一下，但是后来突然改口说要低调一点，削减预算。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">司庆范围缩小了，最后只有核心干部参与。年会变成了线上， 抽奖环节取消。对比2020年显得冷清，“前年也是线上，但是还有抽奖，每个人都有点奖品拿。各分校自己办自己的，总部有个会场，会直播颁奖， 每层线上有大屏统一观看，然后高管到每层给大家抽奖。最大的奖品好像是上万元的笔记本电脑，每层有一个。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">张豫告诉Tech星球，现在公司节电，走廊灯不开，卫生间只有上午有一卷纸。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">部分公司开始严查制度，并且伴随相应的管理机制。譬如，员工需按照规定戴工牌，否则将被罚款20元；上下班高峰不在指定楼层上下电梯罚款（高峰时间只能停靠在10层，7到12层是华图办公区。9层也得在10层下，然后走楼梯）。上级连坐惩罚翻倍。此外，中午休息时间前，员工不得提前热饭取餐，被抓到也要罚款，还会进行通报批评。</p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">退费难，教培贷款被叫停</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">公考热逐年火爆，2022年国考报名人数是202万人，比去年报名人数高出52万。全国平均竞争比58.82：1，职位最高竞争比20813：1。（竞争比=过审人数÷招考人数）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“宇宙的尽头是公务员”几乎已成行业共识，1000多万应届生、300多万考研失利者，以及互联网大厂优化人员等潜在市场人群，感觉应该足以让中公、华图、粉笔等职业教育公司赚得盆满钵满。然而吊诡的是，这些公司反而没有如预想那样吃到红利。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中公教育员工人数一年时间已经减少8000多人，员工总人数从45066降到2021年36143。华图中层员工陈希称，华图2021年底裁员前，公司大概有1.2万人，目前总人数不到1万人。这流失的2000多人，有被裁的，也有主动离职的。去年年底新政策，过年期间福利没有了，加班补贴也没有了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">营收下滑，公司要求员工一起共克时艰。公司层面释放的信息是，4月份营收无法抵对支出，所以强制要求员工休息，等年底的时候再补回来。陈希不敢100%确信承诺真实性，认为或许只是公司的缓兵之计。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">营收目标也调整了，陈希称，去年华图营收目标是30亿-40亿，今年受疫情影响，很多线下培训班无法开展，所以营收目标也变得保守。公司非销售岗位中层也开始变得更关心营收目标，几乎每个月都会内部开会对齐。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">华图此前两次折戟A股市场，今年业绩受损，IPO的速度或将再次拖慢。粉笔公考上市之路也多了一份变量。今年3月份提交招股书拟上市的粉笔公考，近期也被爆出裁员降薪，裁撤部分线下网校。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">包括陈希在内的几位员工，几乎都将公司营收下滑的原因归结于疫情，线下班无法开课。中公近7成营收来源于线下。中公一位销售告诉Tech星球，往年中公有走读周末班，客单价更高，四五万左右，现在疫情期间，没有走读周末班，只有网课+面授相结合的协议班，收费2.78万。部分不景气的网校会被直接裁撤。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但从中公、华图等公司亏损时间节点来看，去年公考培训机构已经爆发了严重的退费潮。2019-2021年前3季度，中公退费率从44.14%攀升至65.81%。考不过即退费的模式今年将给各家公司带来更为严峻的挑战。中公教育微博超话内，有不少投诉中公教育退费难的学员，中公承诺的退费时间是45个工作日，但有人称4个月了还没有成功退费。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，“理享学”贷、“合意贷”也是一大隐患。一位中公教育学员告诉Tech星球，她在去年12月份报名中公理享学贷，与中公签订贷款合同，0元上课，如果考试通过，后续她自己偿还贷款，如果不通过，她则不需要出钱，中公方面还贷。但是因为跟其他事业单位考试时间冲突，所以她签订贷款合同没几天便要求解除合同。但至今近过去了7个月时间，依然没有解除理享学贷款，该学员称，回头可能需要她自己先垫款还贷。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">华图曾在2020年推出“合意贷”，黑猫投诉平台上至今仍有关于合意贷的投诉案例。中公教育“理享学”贷虽然因为4月底被证监会处罚而叫停，但此前已经办理的贷款遗留问题仍会是中公接下来要面临的难题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总的来看，公考培训面临的挑战不小。原本以为，职业教育作为“双减”的幸存者，在公考热的大背景下有望成为新的创业热土，现在看来，市场似乎过分高估了这个赛道。</p>
