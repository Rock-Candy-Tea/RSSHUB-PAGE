<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E7%94%B5%E5%95%86" target="_blank">直播电商</a>圈子可不怎么“太平”：薇娅被封、李佳琦消失、罗永浩退网，超头部直播电商“常青树”以不同方式相继离场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>商业战场，总是人来人往。上海封控期间，刘畊宏以迅雷不及掩耳之势火遍全国，却在两个月后逐渐遇冷。恰逢市场空隙，全网都在期待：谁会是下一个直播明星？这时，东方甄选董宇辉横空出世，“寓教于购物”的创意直播卖货方式，让众网友眼前一亮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>TOP君发现，从淘系超级头部主播，再到抖快系主播矩阵，如今，明星主播热度的“保质期”似乎越来越短。即使粉丝千万，头部主播们其实也挣扎在商业化变现的困境中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播电商的成功究竟是靠运气还是靠实力？对于品牌而言，现在还是入局直播电商的好时机吗？</span><span><strong>直播电商MR.Live</strong></span><span>第一期，TOP君就来跟大家唠唠直播电商的前世今生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/506373148605866b304ead6.38171194.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>直播电商是咋火起来的？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，我们来理清一下概念。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>亿欧智库曾参考中国商业联合会《视频直播购物经营管理和服务规范》，给直播电商作了如下定义：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><strong>直播电商是结合“直播”和“电商”而产生的一种通过互联网信息网络，以直播的方式销售包括实体和虚拟商品在内的经营活动，进而达到通过KOL或主播引流，销售商品或营销推广作用</strong></em><strong>。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抽象来看，简言之：<strong>技术催生了人与人连接新媒介——直播，各大电商平台快速将其应用到商业领域，帮助品牌通过直播引流，最终达成某一具体的商业目标。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>纵观国内，2016年，蘑菇街首次上线直播电商购物功能。同年，淘宝、快手、苏宁、京东等传统电商紧随其后，纷纷推出了直播功能，试图探索“直播+电商”新模式，以打破流量瓶颈。中国直播电商行业进入了萌芽成长期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来两年，直播电商真正进入爆发式增长阶段。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，淘宝直播、蘑菇街等平台携手专业MCN机构，推动直播产业化和规范化；另一方面，头部KOL的辉煌战绩，加速了直播电商的引爆与出圈。2018年双11，李佳琦直播间15分钟卖出150000支口红，薇娅直播间2小时销售额达2.67亿，正式把直播电商带入公众视野。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>受益于众多电商与内容平台开通直播功能，以及新冠疫情下“宅经济”的繁荣，2020年直播电商GMV已破万亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/26983987662d54df1b2a943.76605138.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源：艾瑞咨询、天风证券研究所</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播电商，四年沉浮。从初始只有淘宝和蘑菇街两个平台参与，到现在<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>发展成各平台标配；主播类型从单纯网红，变为明星、CEO、政府官员；带货品种也从美妆、服饰类丰富到如今各类商品，甚至包括车、房、火箭等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>深入到不同平台特性，TOP君认为直播电商平台可以分为综合电商平台、内容电商平台、社交电商平台三种类型。</strong>不同类型直播电商平台各有所长，品牌诉诸的营销诉求也各不相同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传统电商平台（eg. 淘宝）天然具有电商基因，自身拥有丰富的货品和商家资源、成熟的电商服务以及消费者权益保护制度，其通过直播电商主要目的<strong>在于拓展自身获客渠道，增强平台用户粘性。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内容电商平台（eg. 抖音）的优势在于多元内容、流量资源和算法推荐机制，<strong>流量如何变现、完成商业闭环</strong>，是其持续探索的命题。直播电商是内容平台商业变现旅程中，偶遇的一片潜力金矿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>社交电商平台(eg.快手）的杀手锏则为“信任关系”——聚合流量产生的高转化价值，在直播间，主播可以通过实时互动交流，增加与粉丝的连接，<strong>打造主播个人私域圈层，实现多元渠道变现。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从市场占比来看，2020年，淘宝直播、快手、抖音GMV规模分别为4,000+亿元、3,812亿元、5,000+亿元，合计占据行业总市场规模的90%。随着近两年抖快抓紧布局电商基础建设，发力闭环小店打造，<strong>目前，直播电商市场竞争格局已从起初的“一超双雄”逐渐演变成“三足鼎立”的局面。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/91045957562d54df2072044.99619798.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源：申万宏源研究</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使在2020年后，直播电商的流量增速明显放缓，但在国内消费市场疲软，经济增长乏力的当下，直播电商数据表现依然亮眼。国内众多投研机构，均对直播电商行业予以“看好”的评级，亦反映出：<strong>直播电商仍是当下品牌营销最“时髦”和“靠谱”的选择。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/506373148605866b304ead6.38171194.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>入局直播电商，淘宝？抖音？还是快手？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播是时代催生的产物，更是各大互联网平台的“个性”选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>移动互联网普及，内容行业井喷，消费者注意力被稀释，流量越来越贵了。各大互联网平台争先发展直播业务，是其应对激烈流量竞争的一种方式。<strong>但不同平台的产业基因和流量逻辑，决定了各自直播电商业务的形态和走向。</strong>同样的，品牌在进行渠道布局和预算分配时，也需基于平台特征和营销诉求，选择合适的赛道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，TOP君带大家详细拆解一下淘宝、抖音、快手三大平台在直播电商业务上的差异。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>淘宝直播：“综合+专业”电商服务，创造愉悦、高效的购物体验</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝本身就是综合电商平台，天然具有“商业基因”。用户带有一定购物目的，并不排斥观看商业属性强的内容，因此<strong>淘宝电商内容往往更专业、引导性更强</strong>。同时，得益于淘宝完善的后链路数据以及阿里系海量的底层数据<strong>形成的标签体系，淘宝直播的内容推荐精准度更高，可以高效实现销售转化。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/112830771562d54df233a546.56983672.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/174608108662d54df264a242.84784115.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源：招商证券</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，<strong>高昂的入驻成本和营销费用，</strong>往往让新品牌望而却步。在货架逻辑中，性价比往往是消费者考虑的首要因素。头部主播高昂的议价权，价格成为其核心的竞争力，这却让一众普通主播成为“炮灰”，<strong>马太效应愈演愈烈。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%9B%B4%E6%92%AD" target="_blank">抖音直播</a>：以内容为核心，诠释兴趣电商美好价值</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>娱乐内容，是营销种草的肥沃土壤。</strong>抖音作为日活超6万的国民级app，为品牌营销提供了丰富的前链路。更重要的是，抖音相对去中心化的流量推荐机制，遵循爆款/爆品逻辑，只要内容质量高，中小型KOL和新兴品牌也能获得平台青睐，获得成长和引爆的机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16317590262d54df29936b4.29301925.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源：招商证券</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正因为内容生态的多元与泛化，抖音也不得不面临一个争论已久的问题：<strong>内容基因与电商基因是否在同一个平台、同一个界面完美耦合？</strong>迄今为止，这个问题没有被任何一个平台成功验证过。无孔不入的商业化内容势必会损伤用户体验，<strong>如何平衡用户休闲娱乐诉求与平台自身商业目标，</strong>是抖音直播需要斟酌的难题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>快手：信任电商构建私域活水，带动高转化和强复购</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>快手重社交的流量分发机制，构建了“老铁经济”，让主播与粉丝之间建立了强纽带的信任关系。</strong>根据《快手 C 端用户信任心智调研》，快手用户在购物决策时，将近一半的用户基本只在熟悉的主播直播间购物。从整体来看，私域贡献了快手电商 70%的 GMV，是最为重要的变现路径。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/206212336662d54df2cfcf52.85587560.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源：招商证券</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以主播和私域为中心的“社交”基因，使快手直播间有着令其他平台主播羡慕的转化率和复购率。但由于主播地位过于“强势”，导致<strong>直播间“带货”属性强，并不利于品牌文化的培育和生长。</strong>另一方面，<strong>平台直播电商业务<strong>整体</strong>对头部（家族）主播过于依赖</strong>，在出现利益纠纷时，平台易处于不利地位，长期来看，并不是一个可持续、健康的商业路径。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>综上，现阶段抖音电商、快手电商及淘宝直播均强调内容，以流量变现为主要目标，同时三者侧重有所差异。<span>抖音定位于兴趣电商，以优质内容+精准推荐为核心驱动；快手电商定位于信任电商，重内容互动和私域，以信任关系下的用户复购为核心驱动；<span>淘宝直播本质上为电商平台的内容化渠道，具有更强的电商购物属性。</span></span></span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>品牌涌入直播间，是跟风还是真在赚钱？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播的独特性在于——时时刻刻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>实时与同步，带给消费者一种远方的陪伴感。</strong>根据艾瑞咨询2021年针对网购人群展开的调研显示，除了商品性价比以外，中国消费者（尤其是下沉市场消费者）观看直播购物的主要原因有<span><em>“更直观清晰的产品讲解”</em>、<em>“观看直播是一种休闲放松的方式”</em></span>，有近10%的消费者甚至<span><em>“已经养成了看直播的习惯”。</em></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，直播电商往往运用“降价促销”“限时秒杀”等策略，营造了一种紧张的氛围，给屏幕前的用户带来一种<strong>“仿佛不下单就吃亏了”</strong>的心理暗示，极大地刺激了冲动消费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在资本加热和市场推动下，直播电商赛道在短短5年内迅速膨胀了起来。网经社旗下电商大数据库“电数宝”监测数据显示，直播电商行业2017年融资额约24亿元，同比增长 1311.76%；2018年融资额约96.4亿 元，同比增长301.66%；<strong>2019年融资额是近6年最高，约205.5亿元， 同比上升113.17%。</strong>从行业重大融资事件的数量来看，<strong>2020年直播电商行业重大融资事件21起，同比上升2</strong><strong>00%，为近5年来最多。</strong>此后，无论是融资金额还是融资数量，市场对直播电商赛道的“狂热追捧”，断崖式冷却下来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>TOP君观察到，<span>这5年间，</span>无论是快消龙头品牌还是新消费品牌，均以不同形式，或主动或被动，出现在各式各样的直播间里。然而，品牌们真的通过直播电商，赢得声量或销量的增长了吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>直播电商抽佣率高，是一个费用率远高于京东、天猫等传统货架电商的渠道。</strong>有关调研数据显示，直播电商商家向带货主播支付的费用，占成交额之比可高达30%以上；相比之下，阿里电商平台2021财年的整体货币化率仅约4.1%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em>*货币化率（Monetization rate）=营业收入/GMV，营业收入包含广告费、交易佣金等直接收入</em></span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源：东吴零售渠道调研</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>直播“费用”除了主播佣金和坑位费，还有一项隐形“费用”，常被市场忽略，即折扣让利。</strong>头部主播带货通常伴随着较大幅度的折扣，倘若把直播电商产生的折扣算作销售费用，那么直播间的实际ROI也会大打折扣。我们假设某场直播商家的ROI=3（即投放费用占成交额约33%，不考虑售后），产品优惠力度为6折，那么把折扣也算作费用后，品牌单场直播ROI变为1.7。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更别提与图文、短视频等数字媒介相比，搭建一个高质量的直播间，额外消耗的人力和物力成本远高于前者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>TOP君认为，<strong>近年“行业内卷”反映到营销市场，具体体现在渠道和形式上。</strong>市场周期与营销节奏的整体加快，留给品牌“犹豫”的时间并不多，这导致品牌们往往是被一双无形的手推动着前进。从结果上看，有时，品牌一次campaign或是渠道投放，事实上并没有真正为品牌自身带来多少“增长”，而是仅仅抵消了竞争对手的市场动作：<strong>其他人都做了，所以我不得不做。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这场“零和博弈”中，整体的社会财富并没有增加。品牌们被迫卷入这场消耗之争，懵懵懂懂参与一次campaign后发现，竟然什么也没有留下。于是，品牌们开始了冷静的、深入的本质性思考。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>浮躁的市场环境中，热点、风潮一波又一波。市场营销工作的常态，似乎就是“拥抱变化”。品牌体会过热市场后，需要静下来冷思考：<strong>直播本质是仍然是一种连接方式，是时代技术演变的产物。</strong>相对来看，直播与移动互联网前一个十年爆火的图文、短视频媒介相比，并没有实质性的区别。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做电商、造品牌，还得回归品牌本质的思考才有意义：这场campaign是为了打心智，还是为了促销量？再结合不同平台的特性，理性选择合适的直播平台和直播方式，才是真正利用了时代建品牌，而不是被时代所裹挟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着工业化水平越来越高，媒介形式迭代周期将会越来越短，品牌人需要保持开放的心态、持续地自我更新——研究每一种新兴媒介的连接特性与语言结构。正如媒介理论家麦克卢汉所言“媒介即信息”，<strong>掌握了时代流行的媒介语言，即掌握了与当代消费者沟通的密码。</strong></span></p>
