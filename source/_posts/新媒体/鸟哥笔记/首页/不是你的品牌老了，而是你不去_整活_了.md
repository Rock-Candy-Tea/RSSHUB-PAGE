<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>1</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>头几年，一个令人啼笑皆非的产品会议。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老板说，咱们新出的产品一定要让年轻人喜欢，让90后喜欢（那时候“90后”还是年轻人的代名词）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，产品会议就找来公司最年轻的一帮小孩，让他们给公司即将推出的新品提意见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一开始，小姑娘小伙子们还有点拘谨，在老板的鼓励下，逐渐都放开了手脚。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着意见越提越多，每个人都觉得不说点什么显得自己特别没有思考能力。于是意见越提越夸张，小朋友们越提越兴奋，产品会议成了吐槽大会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>年轻人提完意见心满意足的走了，更奇葩的事情出现了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老板说：把刚才的意见整理整理，拿出新品的整改方案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这回，产品经理、研发老总、市场总监、销售老大等一众人员彻底凌乱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这玩意儿怎么改？关键是你还不能反驳，一反驳就是落伍，就是不了解年轻市场，就是不会跟年轻消费者沟通。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品经理事后吐槽，做产品真要这么简单就好了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果找一堆年轻人提意见，就能出来年轻市场喜欢的产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，去找小学生提建议，就能出来十年后年轻市场喜欢的东西。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去幼儿园找小朋友提意见，就能判断出二十年后的市场喜好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真希望这个世界上有这样的灵丹妙药。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，老苗很同情那几位被老板奇葩行为打得晕头转向的“营销老炮”，另一方面，我也很理解那位老板的心情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近些年，传统消费品企业销售成长压力很大，有很多曾经名噪一时的品牌，都增长乏力，或者大幅下滑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从跨国公司的宝洁、阿迪，到国内巨头娃哈哈、康师傅，都面临着业绩下行压力，同时深处“品牌老化”、“被年轻人抛弃”的舆论漩涡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>具体的表现就是，年轻群体消费在他们的销售中占比越来越少，忠诚顾客年龄越来越大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>既然是年轻人不买了，那就投年轻人所好呗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是大部分人的直觉判断。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，“<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E5%B9%B4%E8%BD%BB%E5%8C%96" target="_blank">品牌年轻化</a>”成了很多企业的政治正确，甚至搞成了运动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开发年轻人喜欢的产品，设计年轻人喜欢的品牌形象、宣传话术，改包装、改广告语，来一帮职业经理人就换一茬，换一个品牌服务公司就再换一茬。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后的结果往往是，年轻人没接受，老客户丢失得更快了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>专业往往是反直觉的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们看到自己的品牌不再被现在的年轻人接受了，只是一个结果。并不是真正的原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你不针对导致年轻人不接受的原因上下功夫，眼睛只盯着让自己的品牌“年轻化”，什么颜值不高了，形象老化了，产品概念落伍了，广告语缺乏共鸣了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>结果只能让一代又一代的消费者抛弃你。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>年轻人不接受一个品牌，最常见的原因：</span><strong><span>你的宣传工作做少了，你在市场上的曝光变少了，被市场遗忘了。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这跟你的品牌形象无关。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>百事可乐的品牌形象一直都是年轻化的，长期主打“年轻一代的选择”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但讽刺的是，中国年轻人选择可口可乐的要比百事可乐的人多得多。甚至非常多的年轻人不知道百事可乐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>道理很简单，可口可乐的宣传力度、终端曝光度、形象强度都远大于百事可乐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而百事相对于可口，基本处在失声状态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个状态再持续几年，百事这个“新一代的选择”，将会成为“老掉牙的选择”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/14743411426318a8a3591f71.45032057.png" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>华与华的华杉先生有句非常有哲理的话：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>知道“困了累了喝红牛”的，就是老年人，知道“累了困了喝东鹏特饮”的，就是年轻人。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就是年轻化的真谛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了个别小众人群，大部分人才不关心你的形象是否年轻化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只要你在我需要你的时候，经常出现在我的面前，我就会更大概率选择你。不管年轻人还是老年人，不管男人还是女人，都是这样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，绝大部分传统品牌“老化”的本质，是因为你之前老出来晃悠，但这些年广告少了，宣传少了，终端不怎么见了。年轻人没听说过你，凭啥买？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前那些老顾客，能忘的早把你忘了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当初跟你最有共鸣的少部分顾客，如今还不离不弃，但由于他们现在年纪大了些，因此成了你嫌弃的对象，认为自己的品牌老化是因为跟他们绑定太紧了，认为他们的购买是你“品牌老化”的原因，恨不得把这些粉丝都“洗掉”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想想这也是非常滑稽的事情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在你大概理解了，为什么几乎所有的“品牌年轻化”运动，最后都是“年轻人没抓到却丢了老顾客”了吧？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E5%8F%B6%E8%8C%82%E4%B8%AD" target="_blank">叶茂中</a>先生有段名言，老苗认为这是他一生中关于品牌最重要的一段话。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“品牌崛起的秘诀在于关键四个要素：伟大的产品创意，意义深远的价值观，持续强势的营销活动，数量可观的广告宣传。”</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/18250900936318a8a41112c6.40843699.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这段话，出现在头些年叶总所有演讲讲义的第一页，我觉得值得每一个营销人和企业老板，经常念叨念叨。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>营销活动，要持续不断，你要经常出现在顾客面前，才不会被顾客遗忘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是一个朴素的不能再朴素的真理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以知名度再高，再家喻户晓的品牌，也需要广告或其它传播活动来维持。品牌建设比逆水行舟更要艰难，你的营销活动稍一停止，你就会失去大量的顾客，尤其是年轻人，再一停止，品牌基本就到淘汰边缘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在流传一句话：年轻人不喜欢上一代用过的品牌。这纯粹是耸人听闻的胡说八道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>顾客是否选择这个品牌，只跟该品牌与自己的关系有关，与其它无关。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为老爸老妈喜欢这个品牌，自己就排斥了，就不能买了？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拜托，他们只是年轻，并不是有病。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老化的原因只有一个，那就是被遗忘。而被遗忘就是因为你的营销活动缺乏持续性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这跟你的品牌形象怎样、包装怎样、广告语怎么样，没有一毛钱关系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>打得多的广告就是好广告，持续出现的品牌就是年轻品牌。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便牛如可口可乐，仍然要保持一定的广告宣传力度，保持不停产品的迭代，保持不停进行终端活动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可乐品牌形象能一直保持年轻化，这就是“不是秘诀的秘诀”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><span>有些企业不是不想去做持续的营销活动，而是现在市场变化太大，他们原来的方法失灵了，新方法又没学会，怎么破？建议你持续关注本系列第二篇《不是你的品牌老了，而是你不会整活了》。</span></em></span></p>
