<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作者丨谢中秀</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音挥向<a href="https://www.niaogebiji.com/tag/%E6%9C%AC%E5%9C%B0%E7%94%9F%E6%B4%BB" target="_blank">本地生活</a>商家的“抽佣之刀”已经亮出了一个月，但商家却似乎感知不深。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“啊？抖音开始抽佣了吗？我不知道。”6月27日，在湖南省长沙市经营一家猪肚鸡火锅店的白羊告诉燃财经。2021年初，为增加店铺流量，白羊在抖音上线了自家的团购套餐，“但一直就是挂着，没卖出几个，所以也没怎么上心。偶尔有客人来核销抖音套餐，我还手忙脚乱的，不知道怎么操作。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“不过我朋友店铺的<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E5%9B%A2%E8%B4%AD" target="_blank">抖音团购</a>套餐单子会多一点，因为他们上了低价套餐。”白羊说道，“但他也不知道抽佣的事情，因为他的抖音团购都是交给代运营打理的。代运营负责抖音号的运营、上套餐、拍探店视频等，然后按比例每单抽佣，其他的我朋友也不管。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>低价博流量，交给代运营打理——这似乎是上线抖音团购的商家的多数选择</span></strong><span>。6月27日，北京一位连锁烤鸭品牌的加盟店店长钟峰也告诉燃财经，“在找代运营之前，我们也自己琢磨过一段时间，弄了抖音号、团购套餐，但效果不好，没有流量，一个人也没有。所以最后就交给代运营了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但即使代运营这边，也未有抽佣的风声。6月28日，一家代运营公司的工作人员大猫就在向燃财经介绍时表示，“现在都28号了，你可以问一下开通了橱窗的商家，有没有收取佣金。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但在大猫看来，抽佣是意料之中的事，“抖音现在还是对商家扶持一个状态，后面肯定会抽取。所以要抓紧时间入驻，抖音那么大一个平台，不可能做慈善，一直政策扶持。趁着现在0佣金的一个状态，越等到后面就越吃亏。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经也曾向抖音了解抽佣首月的情况，但截至发稿，未获得回复。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过收进抖音钱袋子里的佣金估计不会太多。除了以上商家的反馈之外，燃财经看到，在抖音宣布收费的《2022年生活服务软件服务费标准说明》中，有众多免佣、返佣政策，比如新商家60天保护期；小微商家前两个月100%月度返还；每月采取疫情“静态管控”天数≥10天的城市内商家，100%返还；其他商家，每月50%返还。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018年起，抖音就看上了本地生活的蛋糕，组建了豪华的 POI（</span><span>Point of Interest，即“兴趣点”</span><span>）团队，并在2019年上线了功能丰富的商家页，但一直都收效甚微。直到疫情将线下冰冻，线上迎来爆发的时机，抖音的本地生活业务才青云直上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在2020年抖音企业号生态大会上，抖音官方宣布，截至2020年10月，抖音企业号用户数正式突破500万，从企业号生态分布来看，零售业企业账号增长最快，排名第一，其次是服装配饰，餐饮服务和生活服务分别位列第四位和第五位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年底，字节跳动商业化部成立专门拓展本地生活业务的“本地直营业务中心”。2021年1月，原SMB（</span><span>中小客户</span><span>）业务线撤销后，约一万名员工调整至该中心，围绕生活服务、餐饮等行业进行客户挖掘。可以说，抖音生活服务自此启动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之后，“团购”陆续出现在北、上、杭等一二线城市同城页面的首个入口。抖音的本地生活也不断加码，商家直播、团购达人探店，外卖服务不断出击。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于抖音的本地生活，有声音表示看好，“毕竟手握6亿日活，干什么干不好？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但一切也并非那么容易。目前抖音的本地生活业务以团购为主要载体，但对比<a href="https://www.niaogebiji.com/tag/%E7%BE%8E%E5%9B%A2" target="_blank">美团</a>、大众点评，抖音团购的商家数量依旧有限。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，在燃财经调查的消费者中，使用抖音团购的依旧有限，在问询的20多个人中，仅有3个表示使用过抖音团购。而且抖音团购始于兴趣，也囿于兴趣，“刷视频时看到商家的美食，就会顺手买一个优惠券。”但更多人想吃查找吃喝玩乐时，会打开美团、大众点评。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于抖音来说，想在本地生活和美团一战，尚需时日。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>商家涌入抖音团购</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>疫情之下，一切都往线上走，一些商家尝试在6亿日活的抖音寻找出路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们是4月份开通抖音团购的。”钟峰介绍，“这两年疫情，线下客流量很少。为了吸引客流，我们就开始转战线上，最开始是2021年10月份上线了北京联联周边游，这也是一个专门薅商家羊毛的平台。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“后来今年三四月份的时候，抖音找到我们，推团购推得很猛，说可以给商家扶持，于是我们就停了联联周边游，上线了抖音。”钟峰继续说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月27日，北京市朝阳区一家日式烧肉店也告诉燃财经，“在6月份北京开放堂食之后，在抖音上线了一些低价团购套餐引流。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月28日，北京市朝阳区一家海鲜自助餐厅也向燃财经表达了上述选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>疫情之下，线上成了众多商家的选择</span></strong><span>。一份公开资料显示，暂停堂食期间，北京区域在抖音开播的餐饮商户数量环比增加了20%。同时，抖音生活服务于6月21日发布的《恢复堂食后北京网友饮食图鉴》也表示，自6月6日北京陆续开放堂食以来，北京网友在抖音搜索本地餐饮次数增长35.85%，餐饮花销增长75%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/58165854562bfb746692d96.05811687.jpeg" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>图/抖音团购页面</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>来源/燃财经截图</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>家在海南省海口市的金鱼回忆道，“大概今年开始，海口支持抖音团购的商家就多了起来。现在开通抖音团购的商家已经很多了，其中美食店铺最多。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>上线抖音团购给商家带来了流量</span></strong><span>。“你们这桌，左边这桌，还有那一桌和那一桌，都是用的抖音团购。现在用抖音团购的大概占单量的五六成。”6月28日晚间，在燃财经问到现在使用抖音团购到店的单量如何时，上述日式烧肉店的店员数着店里客人说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>钟峰也告诉燃财经，“现在店里抖音团购单量大概占了店里单量的75-80%。另外美团、大众点评占比大概5%，剩下15-20%为正常到店。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但这些流量背后，是商家付出的让利</span></strong><span>。正如白羊所说，她和朋友均经营猪肚鸡火锅，但上线了低价套餐的店铺，明显抖音团购套餐销量更好。虽然其中也有运营差异，但价格带来的影响明显。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“单量占比75-80%的抖音套餐，对营收贡献的占比约为10%。”钟峰直言，“<strong>抖音团购的套餐，基本上没有盈利</strong>。我做100个抖音团购套餐，净利不如一桌自然到店没使用团购的单子。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但经营遇难、客人寥寥的店铺选择不多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“生意如流水，要活水源头，而且细水长流嘛。咱们做生意，先不说赚不赚钱，至少得有人来、有人气吧？现在上线抖音，就是为了这点人气。一是大家吃饭不就是哪儿人多去哪儿，有人气就能带动大家来；二是培养了客流之后，也许之后团购取消了，大家觉得好吃还会来。”钟峰解释。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>让利也并非长久之计</span></strong><span>。上述海鲜自助餐厅就告诉燃财经，“抖音这个团购套餐只上最近一段时间。”上述日式烧肉店也表示，“通过第三方，比如联联周边游上的抖音团购套餐只有一个月，到7月15日结束。我们自己官方的套餐还没说什么时候下线。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月1日，燃财经也看到，该日式烧肉店一份来自“享库生活”的228元套餐已经不可见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于6月1日起的抽佣，燃财经问到的多位商家都表示不了解。“通过第三方，比如联联周边游的这个套餐，消费者买是228元，到我们手里是199元。但我们官方自己上的套餐，比如448元这个，我们到手就是448元。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>钟峰也说道，抖音团购主要是由代运营负责，店铺管得不多，“现在也会给代运营佣金，平台抽佣还真不知道。”</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>“养肥”代运营</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong> </strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>相比于商家让利换流量，代运营、探店达人围绕着团购获得的收益更为可观</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在与多位商家的沟通中，代运营都是商家上线抖音团购的一个重要角色</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在小红书、百度、知乎等平台搜索“抖音 团购”，涌出来的结果，多数是代运营机构的宣传。其中“坑”似乎也不少。比如有用户就表示，在进行抖音企业号认证时，认证未成功，但600元认证费并未退回；以及在认证之后，代运营服务并不到位，还在过程中增加收费项目等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经发现，在联系代运营机构时，对方会首先提出600元企业号认证。比如大猫就在与燃财经建立联系后，提出需要营业执照、法人身份证、许可证等资料，以及认证企业号所需的抖音号、法人抖音资料等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在燃财经表示没有企业号之后，大猫便表示，“你把资料发给我，我们帮你认证。抖音平台会收取600元的认证费用，这个费用不会经我们手，是交给抖音平台的。如果认证不通过，钱会退给你。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/76532935162bfb746cf3472.46814014.jpeg" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>图/抖音企业号认证页面</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>来源/燃财经截图</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经在自行进行企业号认证时也发现，确实存在600元/年的认证费用。大猫直言，“不可能为了600元认证费坑你就跑。我们主要的业务是运营，又不是认证。帮你做前期的认证只是为你让你看到我们的能力，选择我们做代运营。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于多个笔记提到的未能成功认证企业号，600元认证费也没了，似乎主要是因为认证审核未通过，比如商家品类并不支持等。部分用户在投诉之后，拿回了认证费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在代运营的收益方面，钟峰告诉燃财经，“目前和代运营机构的合作是以单量来定，比如带来多少单量，我们给代运营公司一定的佣金。但具体是由集团统一安排的，所以单量多少、佣金多少，我确实不是很清楚。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>白羊则表示，“我朋友店找的代运营公司，按15%抽点。”大猫提供的介绍显示，“后期更精细化的长期运营合作，我们收1000元一月的服务费。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于运营的内容，钟峰表示，“包括做抖音号、上团购，达人带货、抖音直播带货等推广。”大猫的服务项目也显示，“运营包括账号定位，产品、套餐设计，以及视频、图片，和对接达人资源等。不过，推广、曝光、排名是另外协商的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>运营的效果则是各有看法。在钟峰看来，“术业有专攻，自己做确实不是很懂。”但白羊觉得，“代运营也就上上团购套餐，然后找些小探店账号去拍视频。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在代运营之外，探店达人、团购达人成为吃到抖音团购“肉”的另外一个群体</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我是2019年入驻抖音的，当时是做北漂vlog。”抖音达人光通告诉燃财经，“后来2021年7月，抖音官方BD联系我，问我愿不愿意做探店。最开始我比较拒绝，因为和我的账号调性并不符。后来拗不过BD多次联系，我就去给做了一次。之后接第三方合作做了一次探店，没想到还爆单了，成交了十几万。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，尝到探店甜头的光通转移了账号的重心，“北漂vlog基本没有变现的途径。”但探店的变现能力强大，“现在我每周拍一到两个探店视频，收入是以车马费加佣金的形式。虽然不说有多少，但也比上班强。今年4月份，我还忙不过来，将视频剪辑后期的工作外包了出去。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但随着涌入的人越来越多，无论是代运营还是探店达人，围绕着抖音团购而起的生意也在变得越来越卷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“开始带货做团购达人的越来越多了，这也直接导致我们的议价能力降低，佣金比之前低了很多。而且如果想现在入局做探店达人、团购达人，没有特色和能力也很难做出成绩。”光通表示。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>背靠流量金山也不易</strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>对于抖音来说，本地生活业务最大的底气来自于其流量</span></strong><span>。在各种口径里，“日活6亿”都被提及。燃财经在易观千帆网站上看到，在2022年5月互联网APP月度活跃人数总榜中，抖音排名第五，月度活跃人数为6.9亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另据华安证券的一份研报也显示，相比美团、大众点评人均单次使用时长约2.5 分钟、人均单日使用时长11.5 分钟；抖音人均单次时长达到了 7 分钟，单日使用时长约105 分钟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这成为抖音本地生活的卖点。大猫提供的合作方案显示，“抖音有8亿日活……2022年将是抖音本地生活真正绽放的元年……抖音团购基于抖音强大的公域流量池。”大猫也向燃财经强调，“现在抖音是流量顶端风口。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“今年3月份，抖音联系我们，问我们要不要上团购的时候，也是提出了流量扶持。大概就是我们上团购，然后给我们的定点流量、流量时长会多一些。”钟峰说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/51204004662bfb7472e72e8.28832441.png" width="666" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>图/2022年5月APP月活跃人数榜单</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>来源/易观千帆网站截图</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在消费者端，抖音也证明了自身的带货能力。比如金鱼就告诉燃财经，最开始用抖音团购就是因为刷视频看到相关产品，“那是今年五月份，一个海底世界的团购，价格比较吸引人所以就买了。平时看到好物分享、美食推荐的视频，也会想试试，价格不错就买了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“后来就慢慢养成了使用抖音团购的习惯。有时候没有太大优惠活动也会买，而且现在也不再上美团、本地的一些团购去搜了，直接抖音搜。”金鱼表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在低价之下，能够拒绝的人不多。6月29日，燃财经在一个群里调研抖音团购使用情况时，一位未曾使用过抖音团购的用户，在听说抖音团购性价比高之后也表示，“种草了，火速下载，下次就可以对比一下价格，节约开支。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但还有更多商家和消费者需要引导。6月26日，燃财经对比大众点评与抖音平台上的团购商家，可以看出抖音平台商家明显少于美团、大众点评。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“偲睿洞察”做过的一项调查也显示，在上海长宁区中山公园/江苏路商圈中，入驻美团的商家数量为147家，入驻抖音的则为50家；西安钟鼓楼附近1KM，入驻美团的商家数量为158家，入驻抖音的仅为41家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在消费者端，抖音团购也有尚未触达的人群。“就我们的观察，使用美团、大众点评多为学生、城市白领。而抖音团购多是以家庭到店为主。”钟峰表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在燃财经问询的20多个人中，表示不使用抖音、没有下载过抖音的人也偏多。北京的一位消费者就告诉燃财经，“一周用抖音的时间加起来不到2小时。平时到店想看看有没有团购的时候，也是打开大众点评。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>由此观之，抖音团购远没有看起来的那么热闹</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音杀入本地生活的决心明确，从2021年启动生活服务以来，在团购不断加码之外，继“心动外卖”之后，疫情期间抖音以“直播+外卖”无心插柳，增加了外卖的核销率。另据抖音生活服务披露的数据，截至2022年1月，平台已有70多万个门店入驻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>将收费从此前0.6%的支付通道服务费，调整为2-8%不等的生活服务软件服务费。对于抖音来说，或许能带来更多的收益。但现在来看，这次宣扬的抽佣更像是一次“试水温”，来测试商家的反应和承受能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，长久来看，抖音的本地生活之战并没有那么容易。毕竟6亿日活并不完全等同于6亿消费者，并且低价策略并不可持续。而后续或将到来的抽佣，对于本身抖音团购套餐已为低价、利润微薄的商家来说更是“雪上加霜”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音的本地生活要长成大象，没那么容易。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
