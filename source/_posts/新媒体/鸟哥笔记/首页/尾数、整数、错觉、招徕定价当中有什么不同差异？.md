<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>1.尾数往往影响销量？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>价格竞争<span>是</span>市场营销组合中十分关键的组成部分，同时又最难确定；<strong>因为价格是影响交易成败的关键因素，所以这一因素中往往充斥着大量心理学技巧。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>19世纪心理学家发现，人们对数字“差异”非常敏感，但对「绝对大小」没有太大感知，于是就催生出“心理定价策略”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>品牌基于市场管理条例下，利用消费者购买心理因素，有针对性确定各种产品价格，其中最常用是</span><span><strong>“尾数定价法”。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如：</span><span><strong><span>去超市购物，商品上为什么写24.98元而不是25元？</span></strong></span><span>相差几分的价格，却会让消费者认为24.99元真便宜，25元真贵。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>罗格斯商学院营销学教授罗伯特·辛德勒（Robert Schindler）曾对尾数定价心理营销做了一个研究，</span><span><strong><span>TA发现以.99结尾和以00结尾的两种价格，商品销售量影响差异巨大。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很显然，利用好“小差异”往往可以创造更大收益。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>芝加哥大学的肯尼思·维斯尼斯基(Kenneth J. Wisniewski)也做过一项研究，通过对比再次验证以“99”结尾价格的魔力性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们</span><span><strong>将要出售的奶油价格从89美分降至71美分，便宜20美分时</strong></span><span>，记录的销量上升65%；而</span><strong><span>下一步降至69美分，再便宜2美分时，销量猛增到了222%，令人咋舌。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再比如：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一件商品原价9.15元，下降到6.98元，消费者随便一看好像便宜3块左右，实际上只减少2块钱，但给消费者带来优惠3块多的感觉，这就是“构建”相对差异的魔力性带来的效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当中重要的原因是，</span><span><strong>价格本身具有模糊性，不同的人往往无法确定某个品类的商品售卖绝对价，那只能选择“差不多的”价格。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就意味着，商家可以从消费者内心“构建”价格，既然对绝对数值不敏感，那么就努力构建</span><strong><span>差异值。</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>2.整数定价在哪些场景下？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>整数，顾名思义与尾数定价相反，即一般以0作为尾数的定价法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这种舍零凑整的策略实质上利用消费者“按质论价””一分钱一分货“的两种心理</span></strong></span><span>，常规用在名牌商品上或交易频次比较高的品类上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随便举两个例子：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>移动支付未流行时，如果顾客为购买一瓶水、一个牙刷而迫不得已搜索口袋四处寻找几毛钱的硬币，那此刻心理大概率一团恼火，零头太多就显得麻烦。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，对于日常生活用品，销售商常常采用整数定价法，如水饮品市场，乐百氏、娃哈哈的产品，均以整数1元、2元定价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这样可以让购买者轻松快捷付款，节省时间，方便双方的交易过程。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与上述相反则是“礼品、奢侈品”，你会选择购买99 元的礼盒送岳父还是100 元？大部分人会买100 元；因为显得你对岳父的重视。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>100 元某种意义上就是比99 元高出一档。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或者一套私人定制的西装，定价9998 元和10188 万元完全存在不同差距，明显后者彰显身份；</span><strong>前者很容易让人产生打折的错觉，因此整数也有提高价值的作用。</strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>3.把数量放前面？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“错觉定价法”是利用顾客对商品价格知觉上的误差性，巧妙地确定销售价格的一种定价方法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多高端商品和服务都会在产品附加上花费很多精力，目的都是为了消除顾客焦虑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当卖家和消费者说</span></strong><span><strong><span>“29.9元/50个”</span></strong></span><strong><span>时，顾客不一定认为便宜；但是你说</span></strong><span><strong><span>“50个/29.9元”</span></strong></span><strong><span>时就会产生很便宜的错觉</span></strong><span>，这样消费者才有购买的冲动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>假设在电商平台，商品展示页再加上已经有3589名顾客已经下单，看似毫无关联的大数字，都会让“29.9元”产生很便宜的错觉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也就是，<strong>当商品的数量大于价格时，通过排列组合放在前面会给消费者造成廉价实惠感</strong>，而反之</span><strong><span>“只会陷入同质化竞争中”。</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>4.如何被左位效应影响的？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先举个例子，人们会直观认为5.00元和4.95元之间差距要比5.05元和4.00元之间差距要大，而这仅仅是受左边数字的影响：</span><span>5比4大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在心理学中把这一现象称之为</span><strong><span>左位效应（The Left-Digit Effect）。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>通常指人们对数字大小的判断会受到多位数中最左侧数位的影响，</span></strong><span>尤其在一个人处于忙碌、疲惫、认知资源损耗情况下，大脑相对缓慢而理性部分减慢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其他领域也同样，比如：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>金融市场中，当股票从4美金下降至3.98美金会有大量的人买入，24小时回报率的显著变化也很容易受这种细微差异的影响。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>亦或者在健康领域，《新英格兰医学杂志》有一项扎心的研究，</span><strong>医生更愿意给79岁老人做手术。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相对于80岁的人，</span><span><strong><span>哪怕只差几周医生都会认为，79岁更加年轻</span></strong></span><span>；所以即使在至关生死的医学领域，人们也逃不过左位效应的影响。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>生活中女生也同样，TA们在暴露自己年龄和体重时，总喜欢说自己19岁或29岁，而不愿意说20岁或30岁，110斤的体重也会说成108.5斤等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>明星在娱乐圈做节目均会利用该效应给人留下良好的第一印象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>明白这个道理时，我们该如何有效利用呢？</span><span><strong><span>1）不要赶整数，2）尽可能利用先小后大原理。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>电商发购物券，确切的金钱数会调动消费者的理性脑，199元优惠券似乎比200</span><span>元<span>面值更大；</span></span><span>不论哪种原理，<strong>只要利用好“9”“8”组合，能为品牌谋取很多福利。</strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>5.招徕定价，总是忍不住想看？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每逢618、双11电商平台大促</span><span>，总能通过算法看到曾经点击过的商品出现在APP首页，但似乎有些商品头图价格明显低于市场价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当你点进去看时发现“又恢复了原价”，这种方法通常被称为</span><span><strong>“招徕定价法”。</strong></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么意思呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>平台会利用消费者求廉价的心理，特意将某几种商品的价格定较低以吸引顾客，同时也能延伸其他品类售卖</span></strong><span>；比如：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你在平台上看到一部iPhone 13Pro max可能7199元，但进入详情页才发现是最低配置。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，也有品牌以抬价方式吸引消费者围观。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如同某地有款打火机售价5000元，人们都想看看到底什么样居然卖这么高；结果</span><span>挺失望，除材质外没什么特别，但是旁边3元的打火机却卖得很好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>某种意义而言和“锚定效应”相似，</span><span>品牌定价时常借助“锚定价格”来推其他商品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>购买者在评价产品时，大脑中也常有一个参考价格，因此“招徕定价”在越来越频繁的促销活动中作用最明显。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好比，大型商场节日促销经常采用“优惠大酬宾”宣导活动，招徕价格就是其中亮点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>商场把品牌的建议零售价，以现在热卖价进行比较，并标注减价原因，暗示消费者关注差价，引导抢购从而扩大宣传效果。</span></p>
