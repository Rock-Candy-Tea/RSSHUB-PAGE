<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在疫情的影响之下，宅家经济迎来了爆发，这让家庭游戏场景成为品牌占位的关键点，也同步激活了游戏设备产品的销售。如果从更长期周期来看，随着消费升级的不断推进，电视产品本身也正出现垂直细分发展的趋势，<a href="https://www.niaogebiji.com/tag/%E6%B8%B8%E6%88%8F%E7%94%B5%E8%A7%86" target="_blank">游戏电视</a>品类显示出更大的市场空间。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于年轻用户来说，找到一款合适的游戏电竞电视，开始成为更加刚性的需求；对于品牌方而言，电视设备可以看做是家庭娱乐的关键入口，抢占了电视终端，就相当于抢占了家庭客厅场景，也能从中延展出更多商业价值。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此在今年三月，<a href="https://www.niaogebiji.com/tag/TCL" target="_blank">TCL</a>与京东家电展开深度合作，尝试在京东平台首发V8E MAX游戏电视，定向推出新品游戏电视产品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/173346134862d08683364305.90018768.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管整体来看，消费者对游戏电视的需求正在不断增长，但在新品推广过程中，TCL也依旧面临几大营销挑战：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><span>1、首先是在整个游戏电视的品类竞争上，如何将V8E MAX游戏电视实现差异化，将其打造为电视新品中的“王炸产品”；</span></em></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><span>2、如何将让V8E MAX游戏电视抓住游戏圈层消费者的心，深化真正消费者对产品及品牌的认知与认同；</span></em></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><span>3、如何借助京东平台新品季的流量优势，为产品带来更多的精准流量，从而最终实现品效合一。</span></em></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，时趣团队认为，在本次TCL V8E MAX游戏电视的新品上市营销战役中，最主要的目标就是“品”、“传”、“效”的三者合一。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>品——打造强记忆符号，占位品类认知</span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要想占领用户心智中的品类认知，就必须将产品信息符号化。一个强记忆符号可以极大地降低传播成本，也可以让产品优势本身包裹在符号中进行潜移默化地传递。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>V8E MAX游戏电视是TCL新品的产品名，但显然传播起来非常拗口，并不适合对外传播，我们需要为其量身打造一个用户视角的符号。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>符号的创意来源需要基于产品特质，因此时趣的第一步就是分析产品本身。TCL V8E MAX游戏电视采用了电光蓝配色，因此也被成为TCL电光蓝游戏电视，在HDMI带宽、无线网卡、120Hz刷新率、MEMC动态补偿等参数上都有不俗的亮点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，硬件参数性能的比较是游戏用户的家常便饭，是用户会主动关注对比的内容，而外观颜色其实才是产品给用户带来的第一印象，因此时趣决定从外观特征入手，打造新品的强记忆符号。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时趣对游戏用户群体的洞察在于，玩家在游戏世界中有一种特殊的增益效果，能够赋能玩家战斗力、提升玩家游戏体验，这通常被称为“Buff”。“Buff”这个概念完美地匹配了TCL新品的产品理念，即希望玩家能够得到更好的游戏电竞体验。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此把“电光蓝”和“Buff”相结合，既能够表达产品特性，也能够让玩家产生共鸣，也就顺理成章地成为本次新品传播的强记忆符号——电光蓝Buff。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/167758993862d086839d1af1.36154444.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在设计上，时趣以电光蓝为基础去塑造新品专属视觉锤icon，通过“蓝色闪电+Buff字样”组合，让游戏消费者对TCL电光蓝游戏电视新品有直观认知。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从传播目标来看，我们希望把TCL电光蓝游戏电视打造为一款游戏电视品类的“王炸新品”，并成为一款游戏人群心中绝佳娱乐体验的必备神装。</span><span></span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>传——顶流天团造势，激发游戏圈层共振</span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新品推广符号确定之后，下一步就是新品的传播策略。这里又面临两个方面的问题：第一，如何在品牌认知上触达核心游戏圈层；第二，如何展示产品卖点价值，形成口碑案例种草。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>组合传播策略并不复杂，一方面游戏圈层群体的品牌认知还是需要意见领袖去带动，游戏大咖的推荐能够快速地形成品牌印象，而且对于游戏玩家而言，游戏装备也存在一定的模仿效应，大众玩家有更多动力去购买游戏大咖们推荐的产品，这部分内容更加偏向于品牌价值输出。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时趣邀请了EDG电子竞技俱乐部顶流大V集结成“TCL蓝Buff天团”，从而激发整个游戏圈层及粉丝网友关注。另外还与五大游戏相关品牌蓝V进行跨界合作，通过组成“神仙CP”的方式，将TCL新品的卖点、游戏痛点、品牌特点进行巧妙结合和联合推广，不仅能够撬动更多品牌用户关注TCL新品游戏电视，更能够激起话题讨论，将新品传播渗透到更大圈层。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/56895979062d086843cf765.05880720.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面就是通过种草渠道的广泛影响，通过内容铺盖合作的方式，进一步在消费场景中触达用户，实现口碑案例，而这部分的内容更加偏向于产品硬件优势展示。在内容平台的刷屏种草覆盖下，可以实现用户下单购买的临门一脚，也能够将全网流量导向京东站内页面。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在新品营销推广期中，时趣帮助TCL将产品种草内容渗透到月传播期覆盖微博、微信、什么值得买、游戏PR网站、今日头条、抖音、百度、垂类媒体八大阵地，深耕“TCL蓝Buff”卖点，通过教育种草口碑安利、同频共振，以高点位资源曝光触达游戏人群。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，种草内容的海报创意都以游戏场景出发，通过玩家在游戏关卡中面临的挑战来展示TCL新品的特性功能。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/71271029762d0868631c705.96603779.png" referrerpolicy="no-referrer"><span></span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>效——借力京东资源，打造3月热卖潮</span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于TCL游戏电视新品在京东小魔方新品首发，通过站外的各种渠道营销推广后，还需要把流量归集到站内实现转化；同时对于京东平台的站内资源，TCL也需要通过站内推广进行获取。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也就是说，在京东站内，主要营销诉求有两点：第一，承接站外流量，实现高效转化；第二，吸引站内流量，实现产品助推。而这就不能一味地进行效果广告传播，依旧需要品效的搭配与结合。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时趣的方案是，一方面联合京东共同打造《新品情报局》原创视频，在站内抢占“电光蓝”游戏电视新品心智；另一方通过搜索彩蛋、新品预定福利、购机种草秀等效果转化激励手段，实现站内资源的霸屏，强导流至京东站内专题页，刺激消费者对TCL新品游戏电视的购买欲望。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/33589966862d086877adc74.03957357.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>TCL&京东《新品情报局》</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从整体营销结果来看，在品牌层面，3月24日首发当日#TCL游戏电视#实现了行业搜索排名第三，搜索增幅超1000%，处于游戏电视品牌排名第一；在效果层面，新品在首发日荣登京东三大新品榜单榜首，成为类目销量&销售额双冠军；在传播层面，品牌的全渠道传播曝光超3亿，覆盖游戏圈层拥有超2.5亿，#热爱电光蓝#话题截至3月25日阅读次数超1.2亿。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说通过一整套“品”、“效”、“传”的策略组合拳，TCL V8E MAX游戏电视最终实现了用户心智的占领，并且优秀的新品首发销售结果也为未来产品持销打下好的基础。</span></p>
