<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们说，所有的机会，都藏在变化里。周期是变化，波动是变化，趋势也是变化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些变化，一定会带来一种失衡，也就是我们所说的红利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>波动可能会带来一个小红利，周期可能会带来一个中型红利，趋势呢，可能会带来一个长期的红利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>谁先发现这种失衡，谁就能先抓住因为失衡涌现出的大量的需求和机会，并因此赚到钱。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只要出现失衡，就一定会有失衡被填补的一个逻辑，一定会有重新回到平衡的一个脉络。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种脉络出现的现象通常分为短期、中期和长期这么几步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这篇文章我们就来讲讲，短期，中期，长期，分别需要关注的是什么。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/113926470662832e66aa5709.61810445.png" title="短期看流量，中期看供给，长期看品牌" alt="短期看流量，中期看供给，长期看品牌" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>短期看流量</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>短期看流量，短期赚的通常是流量的钱。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为一些变化，突然带来了大量的需求，但是供给跟不上，然后就出现了供需失衡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，用户突然对某种商品的需求增加，而你刚好能触达用户，满足他们的需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这时候你就能赚到钱，但你赚的是什么钱，是流量的钱。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，有一段时间加碘盐的需求突然增加，很多人开始屯盐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时候，你刚好有渠道，能提供大量的盐，满足突然出现的需求，那你就能因此赚到钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再比如，今天很多人在抖音上做生意，就像曾经有人在淘宝上做生意一样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在抖音上做生意，一开始做的是流量的生意。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚开始，卖家少，买家多，大量的人聚集在短视频平台，</span><span><strong>用户聚集，而卖家没有聚集的滞后效应，就会带来短期的红利。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先进入这个赛道的人，恰好赶上了红利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，研究研究直播，学习一下抖音的运营，然后再研究一下采买，找供货商拿到货，接着就开始卖产品了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卖产品的同时，还能做做私域，把抖音的用户沉淀下来，重复触达。发现效果还不错，慢慢就做起来了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>有的人就感觉很厉害了，一只脚已经踏进成功的大门了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但这种状态是不稳定的，为什么？因为这是靠流量获得的成功，所有靠流量获得的成功，是因为用户前移，而卖家滞后带来的红利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>比如，短视频平台看上去很赚钱的人，都是一些教别人怎么赚钱的人。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个人很渴望成功，渴望赚钱，他听说别人在短视频平台很挣钱，就想进去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后呢，他可能会先找一些主播学习，这些人可能开了各种课，线上课，线下课，终于学会了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经过一段时间，他终于学会怎么开一个账号，怎么买流量，怎么运营，怎么拍视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>学会之后呢，他们做的一件事，可能是继续想办法教别人。也就意味着不断有人进来，先进来的人就教后面的人怎么赚钱，后面的人再接着教更后面的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>当我们看到这种现象存在的时候，其实就释放了一个信号，这个信号就是，这个领域刚好有红利存在。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但是，过一段时间之后，这群人真的冲进来了，冲进来之后，一旦供需平衡，失衡被填补，这套逻辑就行不通了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时候得看什么，看供给。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/126592131362832e66ddac86.53943116.png" title="短期看流量，中期看供给，长期看品牌" alt="短期看流量，中期看供给，长期看品牌" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>中期看供给</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>短期的失衡被填补后，赛道里乌泱泱地站满了人，这时候，就不是靠流量赚钱了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，你开了一家奶茶店，周围的居民天天排队来买，生意特别火爆。然后，很多人看开奶茶店挣钱，都来开奶茶店，大量的人涌进来，就成了奶茶一条街。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这时候红利退去，供给大于需求了，怎么办？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看看你的奶茶店和别人有什么不一样，你做的奶茶是不是比隔壁那几家的好吃，量是不是够多，种类是不是够全，质量是不是更好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>所以这时候，已经不能靠流量赚钱了，得想办法提高供应链的能力。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我举个例子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有一家服装公司，是做搜索电商的，他们搜索电商做的不错。但是有一天，他说他要做兴趣电商，结果没做起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>为什么？因为搜索电商和兴趣电商的逻辑是不同的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以淘宝为主的搜索电商关注的是对关键字的优化，对详情页的设计，对客服流程和话术的优化等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>而兴趣电商更重视内容。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多做搜索电商的，不具备做内容的能力，到底什么内容吸引人，怎么抓住用户的注意力，这是他不擅长的，所以从搜索电商转型兴趣电商，就会遇到问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是他做搜索电商已经相当不错了，一年能做到几个亿的收入。</span><span><strong>这说明背后有一条相对成熟的供应链</strong>，</span><span>他知道哪个工厂能做出什么衣服，哪里能找到更便宜的面料，哪里能找到最新款的服装。这些能力，恰恰是做兴趣电商的很多主播们不具备的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这时候，他就可以靠供应链的能力去赚钱</strong>。</span><span>去服务一些主播，帮助他们找到最新款的服装，找到质量又好又便宜的面料。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>红利过后，主播会起起落落，但供应链不会，这就是我们说的中期看供给。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅是电商行业，其他行业也一样，一开始你可能抓住了供需失衡带来的红利，赚了流量的钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>到了中期，失衡被填补，你就没法靠流量赚钱了。</strong></span><span>这时就看你能不能提供质量更好，价格优惠的商品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去短视频平台涌入了大量的人，有的人为了抓住流量，可能会没有底线。比如，看到谁的视频火，照着拍一遍，也能获得流量，因此就阻碍了很多真正优秀的人进来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在短视频平台开始不断打击那些投机取巧的人，打击那些做流量玩法的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也意味着，未来，流量红利一定会越来越小，供应链的能力会慢慢变成核心能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/72621015062832e672b11e3.57050295.png" title="短期看流量，中期看供给，长期看品牌" alt="短期看流量，中期看供给，长期看品牌" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>长期看品牌</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>供给能力很重要，但从更长的时间维度来看，你需要有自己的品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当一个行业进入成熟期后，大家供给能力都差不多。</span><span><strong>这时候，就不是按供应链的能力做选择了，而是看谁影响了消费者的心智，谁更值得信任，就选谁。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，你出去旅行，口渴难耐，刚好路过一家商店，里面有各种各样的饮料，看得眼花缭乱。正在你犹豫不决时，一眼瞥见货架上的王老吉，你二话不说，拿起王老吉就去付款。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>为什么，因为你不愿意冒险啊。</strong></span><span>或许其他饮料都不错，但万一呢，你出门在外，万一喝坏了肚子怎么办。你太熟悉王老吉了，你知道选王老吉是最稳妥的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这就是我们说的长期看品牌。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们以前经常讲一个例子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你去逛家电商场，看中一款空调，知名品牌，你很喜欢，但是要3000块。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>边上有几家店，也卖空调，看起来款式差不多，就是没听说过什么品牌，只要2000多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你会买哪一台？我猜几乎99%的人可能都会买熟悉的品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为，另外几款虽然便宜，但你没法确定质量有没有问题，你不了解，也没有建立信任。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但是知名品牌呢，通过多年的<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E5%BB%BA%E8%AE%BE" target="_blank">品牌建设</a>，把“了解、信任、偏好”都装进了品牌容器里。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你看到名字，就知道不会有太多问题。贵是贵了点，但是相对于无牌厂家交易成本反而低了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>所以，长期看品牌，就是看你的品牌容器里有没有沉淀消费者的信任和偏好。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/98675619462b54765b31e15.72107848.png" title="短期看流量，中期看供给，长期看品牌" alt="短期看流量，中期看供给，长期看品牌" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>进窄门，走远路，见微光</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>我们说，要进窄门，走远路，见微光。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你抓住了红利，一开始赚到了流量的钱，说明你进的是宽门。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>宽门容易，一定会有大量的人涌进来，</span><span>越往里走，人越多，路越窄，越难走。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，短视频兴起后，很多人抓住了这波红利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚开始进入这个赛道的人比较少，供小于求。很多质量一般的短视频博主也赚到了钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随后，哗啦啦，这个赛道挤满了人，竞争越来越激烈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大量的人挤在一条狭窄的小路上，</span><span>竞争非常惨烈，但大家都不愿意往回走。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为有沉没成本。</span><span>我都跑了这么久了，往回走就意味着前功尽弃，拼了命也要挤过去啊。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时候，你要找到宽门里的窄门。</span><span>这样后面路越来越窄，越来越难走的时候，你就可以错开竞争。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>当所有人都在热热闹闹往前走的时候，你要去构建自己的核心竞争力，修炼一项技能。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>所以我们说，中期看供给，你需要提高供应链的能力。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样路越来越窄，竞争越来越激烈，大家拼命往前挤的时候，你的优势就会凸现出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，开餐厅这件事。一开始很容易，门槛很低。所以进入的人很多。后面人越来越多，竞争越来越激烈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时候，你就要找到宽门里的窄门，构建核心竞争力，提高供应链的能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为一条街上都是餐厅的时候，大家看的是谁家的菜更新鲜，谁的品质更好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当大家都开始重视供应链的时候，意味着这个赛道又变得拥挤了，你需要重新找到一条窄门。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>比如，去建设自己的品牌，在品牌容器里装满消费者的了解、信任、偏好。</strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>最后的话</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们说，所有的机会，都藏在变化里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>其中的逻辑永远都是，供给和需求之间的变化。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在流量红利阶段，大量的需求在排队，但是供给特别少。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以就从卖家稀缺，慢慢变成买家稀缺。当卖家越来越多时，消费者的选择权就开始增加。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一条街都是奶茶店，他该选谁呢，他必须要有一个选择的标准，谁质量更好，谁品类更多，就选谁，这时候看的就是供应链的能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但随着奶茶店这个赛道越来越拥挤，供给能力都差不多，这时候，就不是按供应链的能力做选择了<strong>，</strong></span><span><strong>而是看谁影响了消费者的心智，谁更值得信任，就选谁。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>短期看流量，中期看供给</strong></span><span><strong>，长期看品牌。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果看到了红利，赶上了红利，当然要抓住。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但你心里要知道，你挣的是红利的钱，但红利是会消失的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以你还是要去做难而正确的事情，要去积蓄力量，提高供应链的能力，建设品牌，要去构建你的核心竞争力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样未来红利消失之后，大家都觉得挣钱越来越难了，你还能挣到钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>找到宽门里的窄门，看见同行看不见，看不起，看不懂的机会，构建自己的核心竞争力，</span><span>即使红利消失了，你依旧能走得久，也走得远。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>祝福。</span></p>
