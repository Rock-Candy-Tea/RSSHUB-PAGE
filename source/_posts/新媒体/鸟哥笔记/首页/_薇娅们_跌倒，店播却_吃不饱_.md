<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作者 | 陶 淘</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>继雪梨、薇娅、辛巴等头部主播频繁“翻车”之后，曾经依赖于网红直播带货的商家们，走上了店播之路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>店播，顾名思义，即品牌官方在自己的直播间展开直播带货的行为。2020年直播带货大火，但彼时，店播业务，还处于雷声大、雨点小的阶段。36氪未来消费报道，2019年、2020年，抖音品牌商（</span><span>即店播</span><span>）在直播带货中的比例分别为16-17%和30%，占比并不算高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，到了2021年下半年，店播却迅速成为了直播带货场上不可或缺的一部分。抖音数据显示，2021年，抖音店播在所有电商直播中的占比达到了58%左右，同比增长近100%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“店播的兴起多少是因为达人直播的坑太多。”百田森鞋业（</span><span>以下简称“百田森”</span><span>）品牌负责人萧先生谈到，高昂的坑位费、高额的佣金，以及与达人之间的信息不对称等因素，让很多商家成为了“韭菜”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>百田森也曾经是“韭菜”中的一员，而为了尽早降低损耗，2021年7月29日，百田森开启了店播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>珍珠品牌Z早于百田森一个月在抖音和快手开启了店播业务，其直播负责人芦笛透露，头部带货主播状况频出后，达人直播的数据就开始变得极其不稳定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“带货数据不好，库存就会大量积压。此时，不仅销量跟不上，和主播合作的风险也会随之大幅提升。”为了降低这种风险，品牌Z在半年内将店播的比例从10%大幅度提升到了80%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在直播业务中，大幅提高店播占比的还有预制菜品牌珍味小梅园。燃财经了解到，从2020年10月开始做直播的珍味小梅园，如今的品牌店播与达人直播比例，已经调整到了8:2左右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，尽管店播看似热闹，但对于商家而言，作为电商直播行业的后来者和业余选手，在直播赛道上想与达人分得一杯羹，着实不易。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“宣发与运营成本，是压在许多品牌肩头一座难以移走的大山。”芦笛告诉燃财经，冷启动阶段品牌Z先后在抖音、快手这两大平台上的流量投放比例，大约占到了GMV（</span><span>总销售额</span><span>）的15%。珍味小梅园的负责人阿赞直言，其早期的流量投放资金更是高达七位数。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，消费者对店播相对单一的产品类型容易产生的审美疲劳，也是商家面临的难以突破的困境。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以最近爆火的东方甄选账号的数据为例，据抖怪兽数据显示，6月21日至6月27日，东方甄选直播带货数量整体呈下降趋势，从6月21日的174件下降到了6月27日的132件，其中6月26日最低值为110件。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就直播观看人数而言，东方甄选自6月21日起至6月27日，最高看播量1977.1万人次，最低1083.1万人次，近乎减半。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/40686520562bbc2c63d8f23.21200777.jpeg" width="444" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，尽管达人直播已经逐渐被不少商家边缘化，从业内人士看来，店播想要撑起电商直播行业的未来，还任重道远。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<section><span>实际上，店播之外，在此之前，虚拟主播也引起了业界的关注，但中小商家显然对此心存顾忌。</span></section>
<section><span> </span></section>
<section><span>芦笛此前便曾寻求与一家业内做虚拟主播技术顶尖的公司合作，不过，他表示，“目前的虚拟人主播发展还在初期阶段，尚未成熟，无论是技术，还是带货能力，都和真人之间有差异。所以，我们暂时还在观望。”</span></section>
<section><span> </span></section>
<section><span>“不可否认的是，<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E7%94%B5%E5%95%86" target="_blank">直播电商</a>发展到今天，生态越来越丰富，达人主播退潮后，店播也好，虚拟主播也好，都处于摸索阶段，前期会产生大幅的试错成本，都有待平台和商家共同探索和成长。”一位业内人业对燃财经表示，店播和虚拟主播，各有优劣，谁将成为接下来直播电商的主流，很难判定。“未来的直播生态，更可能是各显神通。”</span></section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>商家发力店播</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自2021年下半年以来，越来越多的品牌开始入场店播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝直播的数据显示，2021年，大牌美妆的开播比例已经高达9成，其中甚至不泛DIOR、华伦天奴美妆、Charlotte Tilbury、KENZO等知名品牌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同比增加的还有店播的营收。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝直播此前发布的今年天猫“618”开场数据，品牌店铺成交比去年同期增长超100%。截至6月1日上午11点，成交超千万元的店铺直播间数量是去年同期的两倍以上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>飞瓜数据显示，抖音在“618”好物节期间也表现不俗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿迪达斯、WHOO后和FILA斐乐为品牌排行榜前三名。以阿迪达斯为例，其品牌近一个月销售额4.9亿元，其中预估销售额最高的直播间为「adidas官方旗舰店」，近30天预估销售额为1.7亿元，占比为34%；其次是adidas女子、adidas儿童和交个朋友直播间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>热衷于户外运动的爱丽丝就告诉燃财经，自从她酷爱的运动品牌有了店播之后，爱丽丝就成为了该<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E7%9B%B4%E6%92%AD" target="_blank">品牌直播</a>间的死忠粉，“比如，今年‘618’期间，我的户外运动套装，都是在直播间买的，视频各个角度的效果要比图文更形象，而且是自己的目标品牌，购物效率比较高。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此前，在达人直播一统天下的时期，爱丽丝只会偶尔进直播间随意逛逛，不会下单，“因为达人直播间的品类很多，我根本不知道自己喜欢的商品何时才会轮到，就不会特意进直播间购买。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/138258572362bbc2c68c7ae5.31394972.jpeg" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>图/阿迪达斯和Teenie Weenie的抖音直播间</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>来源/抖音 燃财经截图</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友林芝也对燃财经表示，“店播的乐趣在于不用像达人直播间那样蹲点，久等自己想买的产品出现。看喜欢品牌的直播内容可以纾解压力，尤其是服装类的品牌，就像我看Teenie Weenie，即便不买，看着也有一种‘望梅止渴’的感觉。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>林芝还谈到，除了Teenie Weenie，她最近还经常会逛的直播间，基本都是品牌自播的直播间。“首先，店播价格不比达人直播间贵；另外，达人直播的一个特点在于有自己的风格，而现在，不少精耕细作的品牌店播，也能达到同样的效果。”因此，林芝有时候会把店播当作一个人吃饭时用来放松的BGM。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户对店播模式一定程度的肯定，成为了商家发力店播的动力之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>更主要的，是因为店播还存在着诸多优势。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，店播可以根据自己的节奏来安排上播率，一天12个小时连续播放，更好地触达直播间粉丝，抢占消费者的心智。“尤其对于我们这样的预制菜品牌来说，各品牌的产品差异度相对不算太大，除了在生产端做好产品之外，多曝光也是非常关键的。”阿赞对燃财经表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>平台的扶持也成为了商家加码布局店播、培育私域流量的其中一个因素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据悉，包括抖音、快手在内的短视频平台纷纷投入大量资金进行店播扶持。早在2021年4月，抖音就对商家推出了“帮助1000个商家实现年销破亿元”。今年仅“618”大促，快手平台就给商家店播提供了百亿流量的扶持。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，目前，珍味小梅园就加大了店播的比例，店播与达人直播比例上升到了8：2。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了建立起直播间，各品牌都在抖音、快手等做了一些流量投放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>珍味小梅园投入了大量资金之后，投放流量与自然流量之间的比例上升到了4:6。为了吸引粉丝，在冷启动期间，他们的产品折扣低至六折。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在店播的头几个月，萧先生也在抖音前期投放了不小的资金。萧先生介绍，投放之后，百田森的投放流量达到20%左右，销售数据也好看了一些。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播间硬件的置备，也是前期准备不可或缺的。在具体的布局方面，芦笛表示，他们购买了背景板、专业摄像机、音响设备、麦克风等，一应俱全。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>招募到优质的直播间人才，也是店播能取胜的关键。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为此，品牌Z在每个直播间招募了高素质的2位主播，并且聘用了2位负责从事人员对接、调度、策划等相关工作的幕后人士，他们大多来自大厂，有助于店播的专业化运作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>苦达人直播久矣</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管越来越多的品牌开始尝试店播，但在商家逃离之前，达人直播曾红极一时。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做过很多年线下鞋业品牌推广的萧先生，至今依然清楚地记得，2019年第一次和 “快手一哥”辛巴合作时，直播间的销售数据给他带来的震撼。“短短几秒钟的时间，成交单数就已经破万，GMV更是达到了千万元，这与传统电商完全不在一个量级。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>昔日，达人直播相比商家自运营直播间，有着诸多的优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“达人直播作为先入局者，在直播行业中更熟稔产业链；达人直播间的巨型流量保证了产品的高曝光率与高GMV；达人直播间品类多，消费者在直播间容易停留较长时间，易被种草；达人主播相对不易跳槽，这种稳定性能一定程度为其合作品牌的质量背书。”零售电商行业专家、百联咨询创始人庄帅对燃财经表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而如今，电商直播行业从蓝海变成红海，有着较多的经验可循，达人直播很难再维持其运营的“独家秘诀”，这也为店播的发展创造了可能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“对于第一拨红遍大江南北的主播来说，他们也是有生物周期的，带货能力会随着年龄增长而下降，流量也会因此降低。”庄帅谈到。尤其是一些当红达人主播如薇娅等淡出之后，下一拨头部达人尚未冒出，达人直播多少有几分青黄不接之势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些都是商家转战店播的原因之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>更主要的是，在过去的几年间，品牌商“苦达人直播久矣”。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>萧先生便是其中之一，从接触到达人主播后，萧先生就计划将抖音、快手等平台作为品牌推广和产品销售的主要渠道，但没想到的是，他们的品牌连续两年踩了达人带货和代运营机构的坑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“一方面，达人直播信息、数据不透明，粉丝、销量刷量现象成‘常态’。另一方面，达人直播间坑位费黑产混乱丛生。”萧先生直言，品牌通过七、八万元坑位费抢到的达人直播间名额，实际上只需1万元，高价是几经黄牛转手而来的。除此之外，达人直播昂贵的佣金，占据整体GMV的20%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据《财经天下》报道，一份业内流传的“中介机构避坑表”中，有超200户商家叙述了自己被MCN机构、网红达人们“坑骗”的经历。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而即使挤进了达人直播间，上播率也很难得到保障。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“达人直播有着上播的局限性，即上播率的限制。如一个月或一周只能上一场，这种情况下，在达人直播间就只能上新品。”阿赞表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于上述几家品牌来说，他们之所以加码店播，正是因为达人直播间高昂的坑位费、佣金与产品的低频曝光率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>更重要的是，达人人设的塌房，更是给商家带来了巨额的损失。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年11月与2021年8月，辛巴、薇娅先后因“假燕窝”、“低脂面包不减肥”等问题出现过产品“翻车”现象，对与之合作的商家销量构成了冲击。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年底，由于薇娅直播间被封，调味品品牌松鲜鲜创始人易子涵更是亲手写了一封“亏本清仓”信，提到了一口气积压了12万瓶货物，曾在业界引起了强烈共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年年底，部分网红的负面新闻，也使得与头部达人重仓合作的品牌Z，产品销售数据忽上忽下，并出现了货品库存大量积压的情况。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“曾经，我们达人直播比例达到了90%；如今，我们开始大幅度加码店播，也是为了降低库存积压的风险。”芦笛表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在芦笛看来，当前火热的代播机构同样难以撑起电商直播市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>代播机构指的是为品牌直播时提供运营、主播、场地等服务的直播服务商。在直播三要素：人、货、场负责人和场。但其与达人直播的不同之处在于，代播机构打的是品牌方的名号，并且主播也没有一定的粉丝量与知名度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据萧先生透露，在与达人合作失利之后，他曾短暂与代播机构合作，但再次遭遇了滑铁卢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“当时我们觉得，至少相对于商家而言，代播机构会专业一些。”萧先生表示，然而，令他颇为惊讶的是，百田森早期和代播机构合作时，一个月4万多元的费用，业绩产出居然为零。“这让我逐渐意识到，代播机构实则是一种比较鸡肋的存在，专业度不足。毕竟，如果水准足够，就会选择做达人直播，而不是给别人做嫁衣了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>店播或成直播带货趋势？</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在达人直播和代播机构频频翻车后，店播或许是商家一条相对可行的路径</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播销售额相对稳定、直播数据透明、产品上播率高、没有高额坑位费对利润空间的挤压……对于商家而言，店播相对于达人直播，优势不一而足。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>不过，对于大多数布局店播的商家来说，想要真正玩转店播，还极具挑战</span></strong><span>。这种挑战性，首先体现在各个阶段耗费的资金。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在店播的冷启动阶段，对于包括品牌Z在内的所有商家来说，每个直播间搭建的硬件设备，需要的花费不少于5万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比小额的直播间硬件支出，更大的负担来自流量投放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在店播的头几个月，阿赞和萧先生也在抖音前期投放了大量的资金。尤其是珍味小梅园，据阿赞介绍，他们的资金投放达到了七位数。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“品牌Z最初大概花费了GMV中15%的资金用于快手投放。”芦笛介绍，相较于抖音，快手更需要流量投放，“因为快手是典型的‘老铁’文化，以主播IP为主，公域流量标签虽然不如抖音清晰，但私域流量更集中一些。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>硬件设备与流量投放之外，人力成本更是成为店播的主要支出。因此，优秀的主播或专业的团队也成为了店播掷重金的一部分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们每位主播每月的收入大概是2-3万元，比同行平均水平要高一些。”芦笛介绍，“对于有着一定行业准入门槛的中控人员，我们一般都会给出3-4万元的月薪。这样一来，一个直播间2位主播、2位中控，每个月我们会付出12-13万元的成本。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经了解到，由于直播间的硬件投入、人力支出加上流量投放等费用，品牌Z在刚做店播的头几个月，只获得了微薄的利润。“微薄利润都是不错的收益，这还得益于前两年我们的淘宝直播经验。更多商家在早期都是亏损的。”芦笛直言。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>人员流动性高和主播的直播积极性不足，是商家面临的第二大挑战</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>庄帅就对燃财经表示，“店播的主播由于收入与达人相比，依然微不足道，因此流失率相对较高，这也会影响用户对品牌的信任度。另外，店播的主播是打工人，而达人多半是有股权的老板，直播间收入与个人收入紧密相关，因此，后者直播的积极性也会更高。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>芦笛就是注意到了这一点，才给出了主播高于同行的收入，来尽可能维持主播人员的稳定性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也正是基于此，萧先生则总结出了一套降低主播流动性的经验之道。“我觉得主播的流失更大原因可能是没有足够的上升通道。我们逐渐将外部招募主播的机制，改为内部招募，包括招募线下导购、店长等，给他们提供培训、晋升的空间。” </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就这样，通过内部选拔机制和专业化的培训，百田森鞋业培养了一批优质主播，不仅能够将产品的特性阐明，还有着高昂的直播激情，店播数据也自然有所提高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>珍味小梅园则是通过建立直播间的赛马机制，来提升主播积极性与产品销售额的。即，对GMV、粉丝量高的直播间加大运营、主播、客服团队等相关人力的投入，从而充分调动直播间的积极性，激发每一个直播间的潜能。“有竞争，才会有持续地成长。”阿赞表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，店播面临的困境，还不止于此。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>杭州某数据分析企业的一位电商分析师Rusky认为，店播的不足，在于商品品类的垂直性使之难获流量，更难沉淀用户。“<strong>店播业务就如同‘王婆卖瓜’，用户极易对内容产生审美疲劳</strong>。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>萧先生补充道，店播要想脱颖而出，关键还要靠差异化、吸引人的内容来获客。与此同时，还要注意直播受众与下单客群之间的高匹配度，而不是噱头。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但即便如此，哪怕对于知名品牌或者老牌企业来说，店播的热度从巅峰直泻而下，仿佛也是在朝夕之间。“一家自播的品牌店有了话题度之后，进去逛逛，下个单，主要是图个新鲜，如果不是这个品牌的重度爱好者，或者粉上了该品牌的主播，一般很难持续关注。”一位抖音重度用户表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也正因为如此，尽管在不断发力店播，但像珍味小梅园、品牌Z等品牌，都没有完全放弃与达人的合作，达人带货的GMV也占据着整体GMV的20%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，燃财经了解到，在垂类带货主播里，腰部主播的力量不可忽视。“与我们有合作的‘天宫院小吃’、‘张喜喜’、‘小贝饿了’等，<strong>这些百万量级的食品主播，用户进入直播间的活跃度更高、消费目的性也更强，因此转化率很可能高于综合类头部达人</strong>。”阿赞透露。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许，正如Rusky所说，头部网红频“翻车”后，商家布局店播已是毋庸置疑，但辅之以精准达人的内容直播，或是电商直播未来趋势。</span></p>
