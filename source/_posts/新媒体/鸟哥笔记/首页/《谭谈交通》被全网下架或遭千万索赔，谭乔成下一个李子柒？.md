<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">等待谭乔的会是什么结局？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>一边是《<a href="https://www.niaogebiji.com/tag/%E8%B0%AD%E8%B0%88%E4%BA%A4%E9%80%9A" target="_blank">谭谈交通</a>》被投诉侵权，面临全网下架，一边是可能被成都游术文化传播有限公司</span><span>（下称游术文化）</span><span>索赔数千万元，《谭谈交通》的主持人谭乔被网友称为第二个“李子柒”。然而，李子柒与微念的纠纷长达近一年还未尘埃落定，迎接谭乔的会更难吗？</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，谭乔发布视频称自己多年前主持的节目《谭谈交通》，被第三方公司投诉侵权，以致于相关视频被全网下架。据谭乔介绍，<strong>因被投诉侵权下架，其在B站上发布的原创视频从345个缩减到了如今的79个，还有上万个二创作者编辑的爆款视频也被投诉下架，自己则可能面临数千万元赔款。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据果酱妹观察，谭乔B站上目前最高播放量的视频为《梗王之王》，过去两千多万播放量的视频《二仙桥传说起源》等多个视频已无法搜索查看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/54822797862d06a46e6ef19.09055871.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而向抖音、快手等视频网站主张版权、要求节目下架的游术文化，则称自己已从成都广播电视台获得了《谭谈交通》的节目版权。成都广播电视台相关负责人表示，<strong>电视台长期以来没有对《谭谈交通》节目主张版权权利，“并不代表着任何个人可以随便侵权。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事情引发网友关注，成都广播电视台更是发布声明称：“<strong>不存在对个人索赔千万的情况。维权工作仅针对未经许可而进行不当获利的公司经营主体。</strong>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，谭乔方表示自己当年以交警身份参与节目拍摄时，各方对节目版权未做任何约定。虽然自己目前尚未收到过传票，也尚未遭到索赔，但<strong>此前已有公司因为转载相关视频被索赔5000元。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/124916724862d06a4754dfb7.08658266.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源@谭乔</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《谭谈交通》的版权归属至此陷入争议，谁才是版权方还有待考证。如今备受争议的谭乔，究竟什么来头？</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上“经典桥段”均来自，1080P高码率也无法还原高清画质的《谭谈交通》，而谭乔就是“二仙桥宇宙”的开创者，即便没有去过成都的人，也知道去二仙桥要走成华大道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2005年，谭乔以交警的身份参与拍摄普法节目《谭谈交通》，节目主要内容就是谭警官每天在马路上拦截违反交通规则的人，引导他们遵守道路交通安全规则。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/112973590662d06a4798bf21.24476697.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>谭警官、谭sir的称呼由此而来。节目一做便是13年，直至2018年才正式停播。但谭乔爆红出圈则始于2021年。</span></strong><span>而走红发源地，依然还是“热梗制造机”——B站，UP主的花式二次创作就是出圈热梗的制胜关键。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在谭乔正式入驻B站前，B站上就已出现各种各样灵感来源于《谭谈交通》的二创视频。比如UP主@卧龙寺 ，在B站年度报告“你最喜欢的UP主”页面中，其代表作品就是谭警官的视频。不过，或许是陷入版权争议，这一视频目前在其账号内已无法搜索得到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>凭借《谭谈交通》，谭乔意外刷屏B站。也是在2021年，谭乔辞去警务公职，正式以“自媒体人”的身份入驻B站。2021年1月，谭乔的B站账号突破百万粉丝。</span></strong><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谭sir火到什么程度呢？即便是在游戏界，谭sir也有一席之地。LOL游戏玩家“B站谭乔”，让你在召唤师峡谷也能随时随地感受道路交通安全的洗礼。</span></p>
<p><img src="https://ngbjimg.xy599.com/152540029262d06a4a086992.03864891.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源网络</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，即便是在因为版权风波，多个爆款视频被下架之后，在B站上输入关键词“谭谈交通”，<strong>除谭乔本人的视频外，其他UP主二次创作的衍生视频仍有不少，播放量在200万以上的视频达16个</strong>，其中一个二次创作视频《我的猫要去二仙桥》，播放量高达778.9W。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/38770372062d06a4f299dd0.76749805.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，《谭谈交通》还火到了抖音、快手。在抖音上，<strong>#谭谈交通#的话题播放量高达40.3亿次，万恶之源#二仙桥#的话题播放量也有1.2亿次</strong>。而在快手上，<strong>“谭谈交通”也成为不少百万粉丝博主的视频素材，包括“毒舌电影”、“大腕娱乐”、“苏大强”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同于此前走红于抖音的陈警官，谭警官正式成为B站UP主后，可谓是把B站玩明白了。顶着《谭谈交通》这一大IP，自诩为“全球华语地区享有一定盛誉”的谭乔，<strong>除了偶尔翻拍“二仙桥”等经典桥段，还找来了此前在节目中出现的二仙桥大爷，两人搭档开了个相声栏目“仙桥社•谭谈相声”，甚至与@手工耿 、@盗月社食遇记 、@特效小哥studio 等知名UP主联合创作</strong>，无论是视频内容风格还是风趣幽默的语言话术，满满的B站“整活风”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谭乔的走红，自然离不开其他UP主二次创作的热度加持。单纯靠原版视频或许能火一时，但是不能火一阵，二次创作是内容和文化最好的裂变传播方式。比如@成都电视台看度号 拥有《谭谈交通》的原版视频，但其视频播放量也不过几千或者十几万，与谭乔等UP主根本不在一个量级。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/207238843762d06a4fda2333.22453645.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>成也《谭谈交通》，败也《谭谈交通》。如今陷入版权争议的谭乔，未来是否还能依靠这一IP延续自己的“二仙桥宇宙”成了一道谜题。</span><span></span></p>
