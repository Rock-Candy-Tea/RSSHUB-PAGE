<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;"><a href="https://www.niaogebiji.com/tag/%E5%86%9C%E6%9D%91%E7%94%B5%E5%95%86" target="_blank">农村电商</a>规模提升，正在创造更多的就业机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们的团队，一半都是从城里返乡的80后到00后，也有很多在校大学生来公司实习。如果农村有更多做电商的公司，那会有更多的年轻人愿意回来。”重庆江津区石蟆镇的何春梅告诉燃次元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021，何春梅在镇上新建了“聚乡村农村电商中心”，占地面积2000多平米。公司经营多个电商平台店铺，将家乡的橄榄、红糖、花椒等<a href="https://www.niaogebiji.com/tag/%E5%86%9C%E4%BA%A7%E5%93%81" target="_blank">农产品</a>销往全国各地，2021年销售额达到了1360万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1551301416630c225f36d246.24193117.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/何春梅去地里收购甘蔗 </span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/何聚摄影</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期，拼多多平台联合央视新闻发起“我们村里喜事多”公益行动，讲述广袤农村天地“新农人”返乡创业、带动农民就业致富的故事。借助拼多多等电商平台，更多“新农人”返乡创业、就业，带火一批网红品牌，也给县域经济带去活力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>农村电商的发展，为年轻人返乡就业创造机遇。越来越多的大学毕业生，离开北上广深一线城市，到非一线城市、基层就业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据中商产业研究院统计，2021年全国农村网络零售额2.05万亿元，比上年增长11.3%，增速加快2.4个百分点。全国农产品网络零售额4221亿元，同比增长2.8%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据5月猎聘大数据研究院发布的《2022应届大学毕业生就业数据报告》（</span><span>下称《报告》</span><span>），应届生在求职意愿上不再扎堆一线城市，长三角、成渝城市群留才成果显现。其中，上海高校毕业的应届生在上海本地的求职率最高，为85.86%。杭州、重庆、成都位居第二至第四，为75.38%、72.23%、71.07%，均高于深圳、广州和北京这三个一线城市。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，2022年的就业形势，促使年轻人做出改变。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>教育部数据显示，2022届高校毕业生规模预计1076万人，同比增加167万，规模和增量均创历史新高，“疫情对就业的影响仍在持续。部分行业和企业生产经营还未恢复到疫情前水平……综合研判，当前高校毕业生就业形势依然严</span><span>峻复杂</span><span>。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据《报告》，从应届生工作首选企业类型来看，央/国企稳坐第一把交椅，占比为 42.32%；其次是互联网公司，占比为22.26%。从连续三年的应届生招聘需求行业分布来看，互联网是持续释放就业机会最多的行业，不过近三年出现明显下滑态势，从2020届的23.53%下降到2022届的20.03%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近一年，在复杂的经济环境变化中，大型互联网公司出现缩编现象，互联网职位供不应求，但年轻人依然偏爱互联网行业。农村电商兴起，正是帮助年轻人留在理想行业，实现个人价值，同时为地方产业发展注入动能。</span></p>

data-copy-origin="https://shimo.im"><strong>90后返乡就业</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回家乡之前，何春梅在重庆一家电商公司，做女装店铺，“女装的类目竞争激烈，我们又是小卖家，未来一眼就望到头了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一次偶然的机会，何春梅将老家的橄榄、桂圆上架到店铺里销售，没想到数据很好。何春梅做了一番市场调研，发现全国橄榄三大产区，除了广东省、福建省，就是重庆江津区石蟆镇了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>何春梅说，在城市里找不到归属感，也想回到父母身边。2016年，她跟丈夫回到家乡，准备电商创业，“如果做好了，其实也能改变老家整体的状态。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>何春梅介绍道，退耕还林后，石蟆镇的橄榄树覆盖面积很广，达7万多亩。但由于小镇地势偏僻，橄榄销路不通，很多农民并不重视橄榄，任由树木自生自灭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也表明，彼时当地的电商环境较差，基础设施薄弱。“刚开始很困难，农村里没有包材，没有快递。包装只能去网上买，而镇上取快递还要付费。”从去树林打橄榄，到去30公里外的地区发货，都是何春梅和丈夫两人在做。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一开始，何春梅遇到两大难题。一是镇上快递价格谈不下来，快递运营商觉得他们“骗人”，认为他们一天发不了多少订单；二是发货的橄榄总是损坏，每笔订单都在赔钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“跟快递（</span><span>运营商</span><span>）谈的时候，我说给一个月时间，我至少给他们1000单。快递（</span><span>运营商</span><span>）说，我们不是第一个卖橄榄的，镇上其他人也做过，没做起来。”尽管如此，何春梅没有轻易放弃，决定坚持下去，“橄榄是有市场的，因为一直都有订单，只是我们需要解决损坏的问题。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个月后，他们终于发现，原来症结是“打橄榄”，橄榄从十几米高的树上“打落”下来，更容易损坏，于是改为采摘方式。“机缘巧合发现的，但就因为这个问题，前期一两个月我们每天都亏。”何春梅回忆道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>终于，解决了坏果问题后，何春梅的电商生意开始步入正轨。慢慢地，他们有了快递站点，开始培训采摘工，跟村镇合作社合作，并建立起电商小团队。第一年，何春梅卖了12万斤橄榄。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年，何春梅开始做拼多多店铺，体量逐渐起来，规模越来越大。如今，何春梅的电商生意不再局限于老家江津区，开始扩大到整片川渝地带。</span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/杨野摄影</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年，何春梅被共青团中央、农业农村部授予“全国乡村振兴青年先锋”标兵称号，重庆市仅有9人上榜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如同何春梅一样，返乡创业的一批年轻人，很多已经成为新农人“先锋”，给家乡产业带去变化与发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>陕西苹果名扬天下，延安市洛川县也是通过绿色认证的苹果基地县。2012年，来自洛川县的王琦，从上海的电商公司辞职，决定返乡创业，回家卖苹果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一开始，同样由于快递物流等电商基础设施不完善，王琦在老家洛川的创业并不顺利。2014年，他开了一家网店，每日有100单，但有销量没利润，持续亏损，两年后失败收场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来，王琦选择在拼多多开店，从采购、运营、发货，一个人独自创业，继续卖洛川苹果。没想到只花了一个月时间，店铺订单量就增至日均700单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年，王琦店铺累计销售200多万斤洛川苹果，销售额突破600多万元。帮果农把果子卖出去，王琦表示很有成就感。</span><span> </span></p>

data-copy-origin="https://shimo.im"><strong>互联网激活农业</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年，何春梅进驻拼多多平台后，电商规模开始壮大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“拼多多日活基数很大，在上面开店，单量很大。借助拼多多，我们才把体量做起来，而且开始做多样化的农产品。”何春梅说道，一开始卖橄榄的时候，“多多果园”的活动，一天就能卖几千单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>体量起来之后，何春梅的产业链也开始改善。与包装、快递等供应商合作的时候，增加的体量，也成为议价的筹码，这帮助公司进一步控制成本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>橄榄这个品类做好之后，何春梅开始挖掘家乡更多农产品，如古法红糖、花椒、桂圆等品类，“红糖是用传统工艺熬制的，上百年的老汤版，是很值得推广的特产。而江津也是‘花椒之乡’，全区花椒种植面积达50万亩。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年，何春梅的拼多多店铺“聚乡村土特产店”，日均订单量达2000。2022，店铺里红糖、花椒销量超过了橄榄。</span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/何春梅拼多多店铺花椒</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/拼多多APP 燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>何春梅的电商创业成功，带动了石蟆镇产业大步发展。互联网加持，让偏僻农村的产业焕发活力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从橄榄产业来看，现在农民积极种植橄榄树，每年种植面积都在扩大，苗木的价值也在提高。从种植到采摘再到销售，村镇里都形成了专业队伍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“以前橄榄一斤卖几毛钱，现在收购价都要3元一斤，行情好的时候一斤收购价超过10元。”何春梅介绍道，镇里有了很多橄榄示范园，现在还在推行矮化种植，以降低采摘成本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>古法红糖也成为石蟆镇的重要产业。以前“老糖房”没有市场竞争力，销售非常困难，现在连网红都在带货石蟆红糖。2021年，石蟆镇的青皮甘蔗种植面积超过5000亩，首次实现了恢复性增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“整个红糖产业链都起来了，以前最便宜的时候甘蔗收购价280元/吨，现在基本上稳定在700元/吨左右，涨了两倍多。”何春梅指出，有了拼多多等电商平台，线上的体量足以支撑红糖厂的运营，也让背后更多老百姓愿意积极种植甘蔗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而何春梅也是江津区第一批开始在线上卖花椒的。从去年到今年，拼多多平台上，“聚乡村土特产店”的保鲜花椒销量都是排名第一，“我们开拓了花椒的线上市场，当地工厂也开始做拼多多店铺。以前花椒线上销量占比只有几个点，现在达到20-30%。这也让价格稳定下来，不像以前在线下销售，价格波动很大。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三大产品年销量超过千万元的规模，也为石蟆镇数千民众创造了就业机会、经济效益。据不完全统计，何春梅已累计促成500余名村民就近就业，带动4000余农户增收1000余万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，何春梅的电商团队约有30多人。其中，除了江津本地的返乡就业者，还有来自了重庆綦江区、黔江区的年轻人。现在，公司也在组建直播带货团队，在农村里培训电商、直播人才。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年，由于疫情和天气，何春梅的生意受到影响，最近开始转型做跨区域的农产品销售。“我们这个地方刚好位于重庆和四川的交界处，顺应‘川渝双城经济圈’的政策导向，我们开始整合川渝地区的农产品供应链，通过线上渠道卖到全国。”她告诉燃次元。</span><span> </span></p>

data-copy-origin="https://shimo.im"><strong>拼多多发掘新农业</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过社交拼单、农产品零佣金、百亿补贴等多种方式，以农产品零售起家的拼多多，正在为新农人、新农业创造规模效应，大力地推动了农村电商的发展，而这也成为推进乡村振兴的一股强大动力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们村里喜事多”公益行动中，云南省大理白族自治州云龙县的“诺邓火腿”值得关注。此前，诺邓火腿只是土特产，有一定的知名度，但是产品单一，没有形成产业规模。2013年，退伍老兵徐琨斌回乡创业，成立马金桥公司，销售诺邓火腿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2016年，随着沪滇合作的加深，诺邓火腿入驻了上海的云品中心，随即打出名气。到2019年，马金桥公司达到加工销售诺邓火腿1.86万支，产值1600万元。然而，传统生产方式不足以支撑起较大产能，和其他火腿相比，诺邓火腿产量还是较小。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年8月，拼多多借助东西部扶贫协作机制，通过沪滇合作平台，捐赠240万元，在诺邓镇马金桥村民小组落户诺邓火腿“助农车间”。因此，马金桥公司终于有充足资金购买生产设备，以扩大生产规模、优化生产供应链并开发衍生产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1026345748630c226110def2.95590737.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/诺邓火腿“助农车间”</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/闵杉摄影</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两年过去，诺邓火腿“助农车间”每年可达到量产值6000万元，带动周边人口务工约4000人次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这几年来，何春梅也见证了农村电商的发展。她发现，在政策支持之下，农村营商环境一年比一年好。从农产品种子、物流费用到人才培训、电商中心建设等整个发展过程，当地政府多部门都对其公司给予了政策和资金等方面的支持。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我觉得做农村电商就是顺应时代发展的，因此在政策扶持上，力度非常大。”何春梅说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>瞭望智库2021年11月发布的《2021农村电商发展趋势报告》指出，农村电商的发展不仅是一场重配农业生产要素的革命，也是乡村振兴背景下实现国家现代化的一个重要历史机遇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>报告分析，以拼多多为代表的新电商平台近年来持续深耕农村市场，推动农产品上行主流化发展，并在生产端“最初一公里”给偏远地区带去新思路、新逻辑，助力重塑农产品价值供应链。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>报告显示，经过多年持续深入推进，拼多多在农产品上行的实践与努力成效显著。每天有数千万订单诞生在农村地区，农产品上行每年的新增市场规模超过了千亿级别，在中国乡村助力孵化了海量的就业岗位，助力产业发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为国内最大的农产品上行平台，拼多多还在2021年宣布设立“百亿农研”专项，旨在面向农业及乡村的重大需求，不以商业价值和盈利为目的，致力于推动农业科技进步，科技普惠。今年，拼多多连续第三年开展“多多农研科技大赛”，联合国内外顶级院校的农业科研团队，探索前沿农业科技从实验室走向田间地头的创新解决方案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，拼多多联合央视新闻发起“我们村里喜事多”公益行动，则通过“媒体+电商”的创新公益行动形式，讲述返乡创业的新农人故事，让全国民众了解农村电商的广阔舞台，是如何让新农人实现个人价值的同时，带领农村产业蓬勃发展，让父老乡亲就业致富。这也能让更多年轻人，看到返乡就业的可能性，激发人才回乡的理念。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，本次公益活动还建设品质农产品专区，推介新农人的特色优质农品，助力返乡新农人对接国内大市场，实现产销联动。在收获季里，新农人的产品将更快送到千万消费者手中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>何春梅只是拼多多新农人、新农业的一个缩影。根据拼多多发布的《2021新新农人成长报告》，以95后为代表的“新新农人”已经成为推动农产品上行的崭新力量。截至2021年10月，平台“新新农人”数量已超过12.6万人，在涉农商家中的占比超过13%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“如果农村的产业基础完善起来，有更多像我们这样的电商公司，有更多匹配的就业岗位，在外上班的人会愿意回老家工作，留在老家。”何春梅说，只要看好农村电商行业，喜欢农村生活，那么年轻人返乡就业，也可以有一番作为，“新农人要用理性的方式，去带动更多乡亲增收致富。”</span></p>
