font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">近日，利用一个多月的时间面试了13家广告公司，在此过程中学习了很多，也遇到各式各样的面试官。面试13+广告公司后，我觉得求职面试，如若你的基本能力符合企业要求，后续的所有事情其实就是看眼缘，看面试官的喜好。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有时候给我最多的感受就是面试犹如“相亲”，在基本条件符合对方要求的情况下，能否成功就看对方当天的心情、感受、以及彼此在符合对方基本要求情况下的附加条件。在此、将此面试过程中将个人所遇、所感、所想记录下来，希望能够帮助到还在<a href="https://www.niaogebiji.com/tag/%E9%9D%A2%E8%AF%95%E6%B1%82%E8%81%8C" target="_blank">面试求职</a>的你。以下正文开始；</p><h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">整体来说不够顺畅，且在作品展示方面不够全面，个人发挥的空间也不够多。他说他看过了我的作品，但应该看的只是大概，面试的时候作品集的个人主页链接他好像并不知道。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然并不排除他知道我的作品大概，并在见面后对我有个基本的判断。还注明了一句：只要求有文案能力，策划并不需要。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以上，既是我近日面试13家广告公司的经历，文章中所说所想、仅代表个人面试感受，请勿上升到职业攻击或人身攻击、当然标题“决战HR”字面解释为候选人在面试中通过与HR交谈，可以充分的展现自己的职业能力，并没有对可爱的HR及该职位同事有任何敌意，请勿攻击。</p><h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">总结：</h1><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从全文可以看出，我的面试之前都是先发作品，用人部门认可后我才会经一部接洽。但在有些企业，用人部门认可后，经理、企业负责人对该岗位会有一些硬性要求。譬如；该岗位是否必须要有提案能力、该岗位是否必须要有独立成案能力、该岗位是否必须要团队管理能力等等。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">还有些岗位必须要有相同服务类目的经验，譬如你应聘一个广告公司，服务的是汽车，但是你之前没有服务过此类目，用人部门为了候选人快速上手并不会用你。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">还有一些为了刷简历、集作品主动邀约你过去的面试的企业，在收到邀请的时候，需要你对该企业有个基本的判断。面试最终的目的是通过面试入职，而不是持续的面试刷经验，有些面试邀请并不适合部分面试者面对面交谈。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而衡量这其中的标准就是‘作为面试者，你要有自己的标准，知道自己的职业规划以及后续的工作方向，不要到了面试的时候，面试官问你，你回答得模棱两可。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然，如若作品通过。用人部门觉得也ok,但是总经理或者企业负责人对该岗位有其他硬性要求，这个时候就需要看面试官对于候选者的眼缘了。很多时候，求职面试，运气也是其中的一部分。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">譬如我对气场强大的美女面试官以及企业高层就很容易紧张，且对于他们表示出自己不合适的言语或者神态之后，就会失去表达的欲望。过分的专注面试官的神态表情，有时候也不利于自己的面试发挥。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这个就需要每个人根据自己的面试状态以及个人情况及时调整了。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这里，需要提醒大家，在作品投递的时候，需要做一个作品集。作品集的内容包含（服务的品牌logo展示、品牌曝光、曝光数据）、运营的项目（项目曝光、项目周期、项目结果）而后才是个人的职场能力、履历。以数据导向展现你过往的项目经验，类似结案报告一样。</p><p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">求职面试中你会遇到各种各样的面试官，也会遇到各种各样的问题。作为求职者，我们能够做的就是做好一切准备，积累作品为每一次面试机会全力以赴。以上这份求职攻略希望可以帮助到你！也祝愿看到此文的你可以早日找到满意的工作。</p>                        <div style="height: 1px"></div>
