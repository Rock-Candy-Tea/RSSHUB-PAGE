<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三年后回看</span><span>2022</span><span>年9</span><span>月，也许，我们正站在一个交叉点上，空气中弥漫着不确定性，内心又充满着确定性的时刻。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;">1</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>疫情是一面镜子</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>镜子内外隐藏着一种非常深刻的对称性，即你是怎么看待环境的，环境也会怎么对待你。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>诚然，经历过风险洗礼，人们变得恐惧未来、恐惧风险、恐惧损失，这是人性使然。但当大众开始集体性走向一个极端时，往往会让整个市场矫枉过正。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>降本增效，紧衣缩食，开源节流可能都是对的，但是，在生存确定的情况下，聚焦核心产品打造，尽可能保持主营的专注度，这才是企业生产力的根本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>环境动荡的必然会筛选掉一批抗风险能力弱的品牌，市场上依旧能看见逆势增长的生存者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「太极藿香正气」，2021年实现销售收入9.2亿，同比增长33.5%，2022年又开创新场景，拥抱年轻消费群体实现破圈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「空刻意面」，2020年借直播带货、社交种草的风口迅速在线上渗透年轻群体，2022年空刻开始修炼深度分销管理线下渠道的内功。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「九阳太空空气炸锅」，从科技角度定义行业品类天花板，融合流量和品牌价值在同品类空气炸锅品牌中脱颖而出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「有棵树」，尽管内衣市场繁荣而嘈杂，用单品的饱和出击，成功地在种满草的市场中，在消费者心中种下了“一棵树“。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>……</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>越是在市场寒冬时节，品牌越需要在内核竞争力上得以突破，才能抵御外部风险。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>只有潮退的时候，才能看到谁在裸泳。但也正是大风大浪的时候，才能看到谁是真正的水手。</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">2</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>穿越绝望之谷，提前看见拐点</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据国家统计局数据，1—7月份，社会消费品零售总额246302亿元，同比下降0.2%。除了生活刚需产品，消费行业的各个品类受到疫情影响，几乎都遇到了增长难题。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从大环境来看，似乎各个行业都失去了昔日的澎湃和动力，那种曾经昂扬向上的劲头似乎被现实磨平了；从微观个体来看，无论是上市公司老板还是个体户还是打工人，躺也躺不平卷也卷不动，成为一种社会集体焦虑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盛极必衰，否极泰来，从来没有绝对的上行和下行、低谷和高峰、乐观和悲观，一切都是正常的周期罢了。一个成熟的企业家，首先应当理解市场震荡的必然性，及时调整预期、调整决策。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短期经济周期虽然动荡，但长期经济周期还在向上，尤其在国内的大市场里，依然存在很多机会，只是过去被动增长的东风没有了，各个行业要在挑战中努力寻求主动增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>大概率，我们还在一个经济周期里。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>大概率，我们也已离开了绝望之谷。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>悲观者往往正确，乐观者才能成功。尤其对于在品类中占据优势的头部企业，此时加强广告营销反而是一个绝佳的四两拨千斤的机会，这里有两层含义。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其一，术的层面。市场上广告信息密度下降，相同的预算之下，营销可以达成事半功倍的效果，尤其经济低迷的时候，大家会把钱花在更稳妥，确定性更强，确定信赖感更强的品牌上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其二，道的层面。在现金流稳定的条件下，正是头部企业巩固市场地位，产品具有独特价值的新锐品牌弯道超车的好时机，在凛冬重塑强化的品牌，更容易向消费者和行业释放强力的信号，这些是在寒冬周期中依然坚挺前进的企业，更容易获得渠道端和消费者端双向青睐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是近两年太极藿香正气、空刻、泰山啤酒、九阳太空空气炸锅、Ulike、每日黑巧、认养一头牛等一些品牌，能够让我们产生更强的记忆与认知的主要原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>逆境上行，并不意味着一定要一意孤行、头破血流，有时候收缩业务半径、采取聚焦策略、提升服务质量等等做法或许才是品牌最合适的战术。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而此刻就是最好的盘口。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">3</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>广告圈的黎明红利：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>用确定性对抗不确定性</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>异曲同之妙工。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>几个月前，天使投资人、前易到用车创始人周航鼓励大家：应该在悲观的时候去发现机会！理由是社会依然需要企业，投资依然需要寻找标的，这时如果你是依然选择去做的少数人，就会成为被看见的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>有人称之为“悲观红利”，但我们更倾向于定义它为“黎明红利”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>我们100%确定黎明一定会到来，我们只是不知道黎明什么时候会到来。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，唯一正确的事情是，在不确定中找到确定性，用确定性的动作对抗不确定的环境。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，我们会用N个案例来具体解释。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">4</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>案例一：</span></strong><strong><span>太极藿香正气口服液</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>场景带来增量</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>返璞归真，越来越多的企业意识到了，尤其这波大浪淘沙之后，才发现真正活得滋润的，还是那些长期品牌主义者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌的打造不是一朝一夕的，品牌力的壁垒也是日复一日的，这就是复利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使在似乎“没有门槛”的直播带货中，不是强品牌，就很难被大主播选中；不是强品牌，很难在带货谈判中取得合理的利润；不是强品牌，平台在大活动中很难给你好资源扶植；不是强品牌，流量投放一停销量就停，自己店播很难有流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人们塑造一个品牌的最原始初衷，就是为了抢占在商业世界里的马太效应。让自己的产品卖得越来越好，让竞品越来越无人问津。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>无论你是卖奢侈品珠宝，还是卖日常生活用品，最强有力的营销工具就是品牌。在未来10年的市场营销发展中，拥有市场比拥有工厂要重要的多，拥有市场的最佳途径就是拥有深入人心的品牌。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在各种外力内力的作用下，健康产品的需求日益增加，众多中药企业近年来抓住机遇，都在积极寻求突破。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年，太极集团针对旗下藿香正气口服液这一产品，采取了创新新场景，给予品牌重新定位的方式，实现破圈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/168097860063253ce2f22d32.13420916.jpeg" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>高温工作、辛辣冰饮、祛暑祛湿——结合受众的多个使用场景，让受众对产品有了更开拓的认知，唤醒更多消费需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从“防暑解暑”的专用药品，扩张到一个日常场景的消费市场，针对夏天所有怕热的人群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一个特点是广告选择了从年轻人切入，藿香正气口服液全新TVC刷屏分众梯媒，展现了年轻人肠胃不适、暑热感冒、高温贪凉的多发场景。针对尤其重视养生的华南地区，其广告歌曲还专门制作了粤语版，用当地语言更加容易打动消费者内心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为一个老牌企业，其产品优势毋庸置疑，但传统老品牌能够依旧保持在新人类中的认知优势，藿香正气在品牌建设上下足了功夫。2022年，其实现了在广东、两湖、江苏、山东等地的市场布局，打破了近一半销售额来自川渝地区的格局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>品牌永远是商业世界最大的马太效应。</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">5</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>案例二：</span></strong><strong><span>九阳太空空气炸锅</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>靠顾客认知摆脱行业内卷</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌中的“马太效应”形象可以是魔咒，也可以是动力引擎，尤其是对于新生品牌和产品而言。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>越来越的品牌意识到，如果做不了第一，那就直接开创一个新的细分领域去做第一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>的确，中国市场太大了，每一个细分赛道，都可以容纳1-3家独角兽级公司。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，越来越多的细分领域，涌现出了新兴的头部力量，比如Ulike、空刻意面、九阳太空空气炸锅、认养一头牛、每日黑巧、有棵树等等。重新定义品类规则，占领细分用户心智，成为这些品牌的共性打法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为每一个细分赛道，都有至少几十个品牌迅速在跟进、模仿，大多模仿者都试图以看上去差不多的性能、更低的价格谋求着这一新兴市场的一杯羹。大家迅速陷入行业内卷的境地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想要摆脱这种行业“内卷”，必须依靠消费者对品牌的认可。以「九阳太空空气炸锅」为例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于整个空气炸锅品类来讲，新老品牌陷入混战，年轻消费群体的争夺战愈演愈烈。“万物皆可空气炸锅”的趋势，让这款创新产品成为年轻人人手必备的厨房电器。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音上，#空气炸锅#话题累计播放已超过80亿；小红书搜索“空气炸锅”，显示有109万+篇笔记；微博、朋友圈，空气炸锅成为当代都市青年展示厨艺的“做菜神器”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也让一路狂奔的行业，同质化严重，低价竞争但产品质量层次不齐等问题日益凸显，满屏“如何选择空气炸锅”反倒增添了消费者们的选择障碍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对这个困局，九阳选择打造高端产品，推出了一系列“太空科技”产品，其中就包含九阳太空空气炸锅。产品的核心优势点清晰：不用翻面，真正做到解放双手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/175873408463253ce332a294.17848370.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在今年618，九阳进一步加大营销力度，线上联合KOL及KOC，围绕产品的太空科技，对太空空气炸锅进行持续曝光。线下霸屏分众电梯媒体，直切家庭餐厅场景，反复突出九阳太空空气炸锅“不用翻面炸万物”的品牌优势点，让消费者在九阳豆浆机之后，再一次对九阳太空空气炸锅产生了记忆点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抓准产品核心优势点，植入消费者认知，最终成功摆脱了行业内卷的九阳，618京东空气炸锅品类销量第一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/142771308363253ce4112ee8.28601335.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一个时间窗口内“饱和攻击”，将一种特性与品牌自身强相关：一个品牌 = 一种特性，这样才能在消费者心中固化品牌认知，形成品牌护城河，也在一定程度上拥有了建立溢价和拒绝内卷的权利。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">6</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>案例三：</span></strong><strong><span>空刻意面</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>线上线下组合拳 转化率倍增 </span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不必纠结线上线下了，多年实战已经证明，二者都是品牌的必选项，怎么选择，取决于产品的市场阶段，以及投入产出比的具象计算。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有两个客观结论，<strong>其一，纯互联网品牌，品牌投放和流量购买的资金比例为3：7，是一个相对合理的安排；其二，线下的流量相对来说更是价值洼地。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是线上线下，归根结底要离消费者越近越好，最好是相互撬动相互影响。比如，现在的梯媒广告中有一种策略是：落款到天猫/京东/抖音的搜索框，引流线上平台；要么落款到三公里内的实体门店地址，引流线下渠道，直接转化客户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这里，以空刻的成长为例，比较有代表性。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>空刻是第一批互联网品牌的典型，它的第一次出圈来自超头部主播的带货，抓住了直播带货的风口；此后，又选择了在B站、小红书、抖音等社交平台上抓住种草风口；在意识到了精准流量带来的增长已趋于见顶后，又立刻扩大营销方向，利用电梯媒体瞄准重点城市主流消费人群，部署社会化传播，更广泛的触达消费者，创造出更大的一轮品牌破圈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在产品上，空刻获得了世界品质评鉴大会金奖和顶级美味三星奖章背书，解决消费者“不会做、做了不好吃、没空做”的痛点，只需15分钟就能在家做出不亚于西餐厅出品的精美意面；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>营销上，空刻意面以消费者痛点切入，将“只做妈妈，不做饭”的广告语高频次饱和传播，收获共鸣，加速品牌的破圈突围。广告同时落款到天猫旗舰店，大幅提高了线上流量投放的点击率和转化率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/27822269663253ce4eec028.30882500.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一套打法收效相当显著，2019年才成立的空刻意面上市即热销，连续3年在天猫平台意面销量第一，3年市占率超过60%。今年618，开门红四小时同比增长了272%，销售额累积破亿，横扫全网意面类目第一，天猫、抖音平台登顶方便速食类目TOP 1。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span></span>7</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>案例四：</span></strong><strong><span>有棵树</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>聚焦产品饱和攻击 </span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从传播的视角来看，受众的本质在于遗忘，广告的本质在于重复，低频的品牌传播根本赶不上顾客遗忘的速度，所以必须要聚焦，饱和攻击，用足够的浓度打进顾客的心中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样的案例太多了。以一个最新的代表品牌「有棵树」为例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我国内衣市场在2026年规模预计达到3020.5亿元，飞速增长的赛道越繁荣，越避免不了市场的混乱。在众多已经展露头脚的内衣品牌崛起后，有棵树也开始强力输出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其品牌风格依旧延续了近两年趋势，主打自然舒适、自信健康、个性表达、绿色环保。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但相比之下，有棵树在线下的发力更为猛烈。今年618前，其连续4周,在全国71座城市22万块分众屏幕同时投放，在年轻人常出没的地方狂刷了一波“存在感”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让“有棵树有植物、植物做的肉肉杯”等绿色环保的品牌特点牢牢刻画在消费者心智中，成为发展速度让人惊喜的内衣品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/111811441363253ce53cfe98.76788891.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>回过头，来复盘这种看似“简单粗暴”实则“精打细算”的打法，其实恰恰是最高效的用户心智占领。</span></strong><span>2022年618有棵树最终的战果，销量同比翻了三倍，并一举登上了天猫内衣榜单的TOP3。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>让用户记住你，先成为品类第一，才有机会横向扩张。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>阶段性的饱和攻击，让消费者记住产品</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>系统性的饱和攻击，让消费者记住品牌。</span></strong></p>
