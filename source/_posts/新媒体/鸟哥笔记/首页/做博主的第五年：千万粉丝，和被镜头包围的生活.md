<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做博主的第5年，“Winnie文”快要忘记纯粹为喜欢买单是一种什么感觉了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她下单的每一条裙子，都是为了做一期穿搭内容；看中的每一个耳环，也是因为值得被拍成视频。哪怕是一顿早餐，都只是“Winnie文”一天里的第一份工作，为了测评玉米糊或者冲泡饮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“我把生活的方方面面都变成了视频，展示给所有人”</span></strong><span>，最高峰时，她一个人、一个月能在小红书更新35条笔记，同时还在创作B站长视频。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/172126170062d8edf62eeaa3.50222578.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;"><span>Winnie文的小红书账号页面</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>有人说“Winnie文”</span>的内容不够垂直，美妆护肤、省钱攻略、拍照经验、还有家庭情感生活都囊括其中。实际上“Winnie文”也没想过要做一个垂直的账号，她唯一想做的是把自己生活经历分享出来，有人想要帮助的时候，可以随时来看看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2018年的一条拍照攻略开始，<strong>5年时间，“Winnie文”在小红书积累了448万粉丝，全网粉丝破千万，也曾在2020年被评选为B站百大<a href="https://www.niaogebiji.com/tag/UP%E4%B8%BB" target="_blank">UP主</a></strong>。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/36077512962d8edf662f612.09792539.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>在B站，以“文不叮”的ID被265万粉丝认识</span><span> </span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她的创作似乎没有瓶颈，直到近期，也依旧用个人风格鲜明的干货技巧横扫热门榜单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新红数据显示，“Winnie文”近3个月每条笔记平均点赞数达到6.87万，其中一条“他拍上镜绝学”的笔记播放量获赞近98万次，小红书播放量超660万次，让账号一个月涨粉50万。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“他拍上镜绝学”的笔记</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前段时间，我们约“Winnie文”本人聊了聊，希望了解一个自媒体人5年持续创作爆款干货的故事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她生活在厦门，因为距离较远，我们采取了电话沟通的形式。<strong>嗓音温柔，语气笃定——是我们对她的第一印象</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和想象中有点不一样的是，一直被流量聚焦的"Winnie文”更愿意做一个纯粹的内容输出者。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管她把自己的生活搬进了视频中，但用第三方的视角去重新注视她，依旧会让她有点不好意思，“接受采访之前我有点纠结，但做我们这行就必须要被大众看到”。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以下，是“Winnie文”的个人创作经历自述。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>“我把生活的方方面面都变成了视频，</span></strong><strong><span>展示给所有人”</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕业之后，我一直在做舞蹈老师，与好朋友合开了一家舞蹈工作室。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我同时在做兼职模特。一个好朋友说可以试试发小红书@摄影师，能增加照片的曝光。抱着这个想法，2018年5月，我发了第一篇小红书——海边拍照小攻略。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/201913580862d8edf6f147e0.27312970.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>运气还不错，第一次发东西就有几百个点赞。<strong>当时自媒体没有现在这么热，我也不懂如何做内容策划，更多是女生的分享欲</strong>，今天想到一个发型教程不错就拍一下，明天买了几双鞋也拿出来讲讲。有人爱看，我就继续发。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就这样，自媒体成了我另一份兼职，做了一年多的时间，直到有了50万粉丝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为平时舞蹈课排得满，我的声带小节非常严重，基本上每两三个月发一次烧，一发烧扁桃体就没声、直接哑掉，没办法再上课。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15951484062d8edf7ea3b86.35484976.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>当舞蹈老师时的微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着小红书账号越来越忙，我的自媒体收入也超过主业收入。结合身体情况，我将舞蹈工作室转给朋友，自己全职投入自媒体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>我对这个账号的定位，就是关于女生变美的那些事儿。</span></strong><span>早期图文比较多，总有评论问能不能出视频。我说如果这条评论点赞过50就做，结果真的很多人跑去点赞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>既然答应大家，我便开始拍视频。面对镜头讲话是一件蛮尴尬、且有点羞耻感的事情，每次我都会精心妆造、卷个头发，女孩子总希望自己在视频里漂漂亮亮的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但普通人都是有死角的，有时候动作幅度大了或者讲话表情夸张，怎么看都不自然。那段时间，先录视频、后加配音，才会让我比较有安全感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70634569262d8edf83c4427.34483726.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>刚开始拍视频的“Winnie文”</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频数据要比图文数据好一些，我的视频内容也越来越多。时间长了，我发现视频最重要的还是内容，哪怕状态差一点都没关系，人也就慢慢放松了下来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>很多人跟我聊，觉得我的内容太过于不垂直、东西很多。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我当初没考虑过垂直的问题，全职做自媒体后，想着垂直也是垂不了了，干脆就做成系列化，有“新手化妆”“变美干货”“薅羊毛”系列，也有“文文夜话”“情感聊天室”系列。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些系列起初都不是刻意策划的。我是一个很节约的人，喜欢淘一些小东西，东西买贵了会很难受，因为这种生活习惯，我会找一些平替或者薅羊毛的产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“新手化妆”是由于我自己偏手残党，从不怎么会化妆一步步到现在，或许不算很专业，但可以帮助和我一样“手残”的粉丝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/105161500462d8edf8e8b981.27493751.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有时候偶然拍了一期内容，反馈好的话我就会想有没有更多可以去讲的，每天都在生活中寻找这些点，再把它们放到对应的主题里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>我感觉现在不是为自己而活，做任何事情的第一反应都是：这样行不行？我试一下。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>哪怕是一顿早餐，我都不是为自己而吃。测评玉米糊或者冲泡饮时，我早上的第一件工作就是吃不一样的产品。大家在视频里看到我推荐了5款，幕后可能试了三四十种。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134714227262d8edf9a92e04.57276770.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“Winnie文”和团队一起测评</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我好像已经很长一段时间都没有因为自己喜欢一个东西、单纯地想把它买回来，而是这个东西适合做到某个专题里。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>别看最近的数据还挺好的，其实每篇内容发出去之前，我的心里都在打鼓，不是很拿得准流量。我只能说我把所有知道的都放进去了、把可视化做清楚了，也梳理明白为什么这个有用、对比没有用是什么样的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>我能保证的是我的内容是有质量的干货，我们这类账号实用是最重要的。你可以不喜欢我、甚至不认识我，但我的视频肯定对你有帮助。</span></strong><span>除此以外，流量跑得怎么样就看天了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我自己最喜欢的内容倒不是按流量高低来排序的。我非常喜欢“文文夜话”系列，数据肯定不能和干货内容比。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年我给爸爸做了一篇关于父亲节的内容，主题是“真正被富养长大的女孩”。写脚本的时候就写到哭，视频剪出来的时候也哭，哪怕隔了一年再看我还是会哭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/87165097462d8ee0a6605b7.63431626.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这条视频做完后，我爸爸开始戒烟跟运动了，他觉得我把他说得那么好，那他一定要配得上，这让我真的很有成就感。所以现在只要发爸爸相关的内容，我就会去讲他的好话。我说得越好，他越努力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人与人之间的正向影响都是相互的。我父母不懂什么是自媒体，但他们很支持我。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从1万粉丝开始，每次粉丝数突破一个节点，10万、100万，我爸爸都是第一时间截图。涨到400万粉丝的时候是半夜，早上起来已经四百零几万了，他很生气，因为没截到400万整。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/89866587862d8ee0b95e124.94024523.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>B站粉丝100万时，爸爸给“Winnie文”发的信息</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图片由受访者提供</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和团队相处也是，我累的时候你推我一把，你累的时候我们也会给很多鼓励和支持。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>我之前说“不能和老板做朋友的话，那就做家人吧”。</span></strong><span>我就像他们的姐姐一样，大家下班也不回家，喝着小酒坐着聊天，甚至聊到哭都很常见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也有一点不好，有时候太随意、工作和朋友分不开，但我好像没有办法。真的太没有界限感的话，我们会坐下来聊一聊。我是一个有什么一定要当面沟通的人，哪里好和不好，都会当面指出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有时候加班很苦，但他们都说加班没有加班的感觉，大家一起吃饭热闹下，再去把工作继续做完。<strong>可能这不是一个最佳的工作状态，但是我很喜欢的状态</strong>，我很珍惜他们每个人。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/192745842462d8ee0ce6a678.15633876.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“Winnie文”和她的团队伙伴</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>“有时觉得还好做得早，</span></strong><strong><span>不然我现在肯定做不起来”</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这几年，如果一定要说谁给我的帮助最多，完蛋、我觉得只有我自己。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我几乎做梦都在想选题，这个账号的内容太主观了，有时候身边人看到我很累想帮帮我，但都没有办法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚开始的时候，一年365天工作360天，一点都不夸张。<strong>小红书能月更35条，同时在做B站长视频，那时候没有团队</strong>，如今回头看都不知道自己怎么过来的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我基本上每周都会崩溃一次。前年的时候，旭哥</span><span>（winnie文老公）</span><span>强制性地让我一定要一周休息一天，什么工作都不干，就是出去玩、吃饭、看电影。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他建议我降低更新频率，从一个月35条降低到20条。我很不安，因为我从来没断更过超过两天。<strong>旭哥和我说没关系，只要把每条做好做精，少一点也没事，没有人会数你的作品，除了你自己。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我渐渐被说服了，觉得挺有道理。到今年每周还会给自己一个双休，一个月可能只更新10条或者12条。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和我同时期一起做自媒体的一些博主，100万粉丝后流量就不怎么涨了，做内容的心气儿就掉了，更新频率也慢慢变低了。<strong>做博主最重要的就是粉丝的反馈，我们很需要被认可，有好的数据才有坚持下去的动力，哪怕不赚钱都甘之如饴。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我到100万粉丝的时候也卡了很久，就做了些调整。首先是改封面，原来是一个横版照片、上下放大字，我调整成专门为视频单独拍摄的竖版图；然后把视频节奏加快，加表情、加机器音，提高留存率。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/144974616962d8ee0da7edb1.88395468.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>幸运的是每一点小改变，都会收获正向的反馈。每次数据爆了我会想粉丝为什么会喜欢，好的地方我就继续做，这样子顺着来就做得越来越好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>现在很多人宣扬自媒体多好做、多赚钱，我觉得挺误导大家的。</span></strong><span>市场上博主越来越多，我有时觉得还好做得早，不然我现在肯定做不起来，竞争太激烈了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年8月我开始做抖音的时候，就觉得根本没戏来不及了，只能硬着头皮尝试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实证明，任何一个平台只要内容够硬还是有机会的，只是现在的转粉率和以前没得比了，一条2000多万播放、100多万点赞的视频，也就涨个10万粉丝。抖音不好做，但很庆幸自己选择做了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/164398142562d8ee0ee5b901.63771815.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>Winnie文抖音账号</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>入驻抖音后，我一直对直播带货保持观望，但目前还迈不开这一步。现在条件不成熟，要等到选品、供应链、售后每个环节都准备好，同时也要看看粉丝的态度，因为可能一部分内容的位置要被预告和产品占据。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所以我目前商业化的形式还是挺单一的，就是接广告，网店、直播这些都没做。</span></strong><span>广告也都是以品宣为主，差不多是一周一条广告，再搭1-2个日常内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我认真想过，自媒体应该会是我的一份长期职业。以后如果觉得太辛苦了，可以几个月不要广告变现、降低更新量，但不会说放弃做这件事，即使关注度没有现在这么高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有时候我回头以前的作品，记录着每个阶段自己的想法和成长，能让我回忆起很多生活的细节。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反正这个号做不垂直了，我希望它是具象的，把我所有的生活经历给大家一些参考。有人想要帮助的时候，或者想听一个人聊天的时候，都可以随时来看看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我感觉，做视频真的挺有意思的。</span></p>
