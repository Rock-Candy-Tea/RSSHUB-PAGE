<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是广告萌新，还是投放老手，遇到的核心问题无非就是围绕以下三个维度：<br></span></p>
</section>
<section data-role="paragraph">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>冷启动阶段：</strong></span><span>账户起量难，跑不动，计划无消耗</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>跑量阶段：</strong></span><span>成本高、计划空耗、数据不稳</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>拓量阶段：</strong></span><span>放量困难、掉量严重、成本跑飞</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么今天我们就围绕这些问题，聊一聊对应的解决方案，通过普适性的方法论，提升<a href="https://www.niaogebiji.com/tag/%E5%B9%BF%E5%91%8A%E6%8A%95%E6%94%BE" target="_blank">广告投放</a>的最终效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在投放之前，作为优化师，我们首先需要了解广告投放的底层逻辑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人可能会说，这不是明摆着嘛，就是花钱买量。这句话没错，但如果我们把“花钱、买、量”拆分来看：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>“花多少钱、怎么买、要什么样的流量”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么这就不是单一的问题了，因此在目前竞争如此激烈的情况下，广告投放比的就是精细化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里的精，我将其理解为“敏感”：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>• 对素材的敏感度</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>• </span>对平台的敏感度</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>高潜力素材是关键，平台机制是驱动力。对素材的把控，对平台功能产品的使用，是广告投放的两个核心。因此广告投放的底层逻辑也就归根到了素材和平台能力上，即：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>高潜力素材打冲锋 + 竞价机制供补给</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这一底层逻辑，也充分的体现在了巨量引擎效果广告升级版中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>什么是巨量引擎效果广告升级版？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多小伙伴或许已经使用过巨量引擎效果广告升级版了。我们在测试的过程中发现，升级版与原版的不同之处主要有以下三点：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1.更倾向素材的多样性投放，简化账户操作，让账户更快度过冷启动期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2.加大对素材探索的流量扶持，给高潜力素材更多时间和机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3.对于已经验证的高潜力素材，会通过重新组合来拉长广告生命周期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，针对巨量引擎效果广告升级版的特性和优势，如何来解决实际投放遇到的问题呢？接下来我们结合文章开头提到的问题，逐一的说明。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>冷启动阶段快速测出高潜力素材是关键</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>文章开头我们也说了，账户冷启动阶段，最常见的问题就是起量难，跑不动，计划无消耗等。因此，在这个阶段，我们首先需要解决营销漏斗第一层级的问题，即展现和流量的问题。只有当口子撕的最大，才有后续的优化手段可言，因此我们说“先跑量，再降本”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而前期跑量的关键，就是素材。那么冷启动阶段我们要如何快速测出高潜力素材？</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>1、初步判断素材是否具有跑量潜力</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有经验的优化师，通常在广告上线之前，就已经能够对具有跑量潜力的素材进行判断和筛选。常用的方法有：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>• </span>拉取大盘近期表现较好的素材，总结跑量的规律。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>• </span><span>研究同行TOP 5的跑量创意，从视频、脚本、文案、标题以及落地页不同维度，对比了解自身的差距。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>• </span><span>如有老账户，可直接拉取近期跑量TOP 5素材，继续复用。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>• </span><span>多准备几套备用素材及落地页，保持创意的多样性，以便测试需要。</span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>2、巧用巨量引擎效果广告升级版快速测出高潜力素材</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>1）优先使用老账户</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>升级版与原版的跑量逻辑一致，同样是需要有跑量模型。如果是新账户在使用升级版测试的时候，仍然需要时间和预算建立跑量模型，而通常为了更快速的测出高潜力素材，建议使用已有稳定模型的老账户来进行测试。同时，权重较高的老账户，能够帮助稳定升级版冷启动期的广告成本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>2）优先使用老账户跑量TOP素材</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老素材与老账户是一个道理，在原版已经测试成功的素材，平移复制到升级版，能够跑量的概率也更大。而且基于升级版对高潜力素材探索的大力扶持，已验证跑量的素材能够更快速且更多的拿量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>3）一条广告搭配4-6个视频+6个标题以上</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>升级版相比于原版，账户结构更简单，单广告测试多素材，提高账户搭建的效率。因此已验证能跑量的素材，可以直接铺满。而对于一些新的测试素材，建议减半。一来能够达到快速起量的目的，二来也方便我们掌控账户，避免过多空耗。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>3、实战经验分享</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>实战一：严格把控素材，巧用起量工具</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>金牌投手刘邦雄在谈起冷启动阶段起量问题时，分享了自己的起量经验：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1982577249632c0ca2ae2046.69012053.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>金牌投手：刘邦雄</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我也经常遇到起量困难的问题，尤其是跑应用下载，第一要攻克的就是流量口子。而素材是流量最核心的决定因素，因此一般起量困难时，首先我会排查素材与大盘TOP素材之间的差距，保持素材的健康和多样性。其次，由于需求量比较大，我会借助一些工具或者功能来辅助跑量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，在近期跑的一个项目中，为了测试起量，我启用了巨量引擎效果广告升级版，前期测试只用了6套素材-3套标题，首日曝光少。但次日午高峰时就开始出量，跑速快，前端成本与后端ROI还都健康。后续在扩量期就直接全天投放，然后参考爆量计划，去衍生新素材，由于有老素材带动，新素材也很容易起量，单户很快扩到32w/天。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>实战二：先跑量，再降本，数据是一切优化手段的基础</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/859459491632c0ca458d638.32117032.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>金牌投手：宿芳</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>金牌投手刘邦雄给了我们跑量的经验分享，而另一位金牌投手宿芳，则在降低投放成本上给出了自己的建议：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“数据和转化是一切优化手段的基石。而通过升级版快速起量只是完成了第一步，那么如何进一步降低成本呢？归纳起来就4点：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1）暂停消耗少的高成本及尾部计划；同时提高考核成本达标计划的出价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2）排除重复、高成本地区/人群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3）修改竞价策略，由“优先跑量”修改为“均衡投放”或者“控制成本上限”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4）不断优化素材。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过升级版快速跑量和以上4点降本提效法的配合使用，在最近的一次投放中，我们成功的将广告付费成本降低4.8%，付费率提高3%。”</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>跑量阶段巧用平台新能力</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人对新工具、新功能的使用是不信任的，但正是由于这种不信任，才使得部分愿意尝试的人吃到红利。而往往工具运用的好坏，能够在很大程度上拉开运营能力之间的差距，先利其器，才能善其事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而巨量引擎效果广告升级版，就是这把利器。上文我们也说了，升级版账户操作更简化，且在素材探索方面做了很大的流程扶持，那么接下来我们就聊聊具体是如何表现和使用的。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>1、全量素材在线优选，加快跑量节奏</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>巨量引擎效果广告升级版更注重素材的跑量能力，从素材离线优选升级迭代为全量素材在线优选，通过审核的所有创意会同时在线投放，相比于原版，升级版会表现的消耗更快，因此在测试期间，优化师不能掉以轻心，需要时刻关注投放数据，只要成本比原版高不超过5%，都属于合理范围，需要持续观察优化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除视频素材外，落地页同样全量在线优选，最高可添加20个，且只要有一个通过审核，就可以保证广告的正常投放。保持素材和落地页的多样性，能够更快筛选出符合用户预期的创意，从而提高广告投放效率。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>2、复用高潜力素材，聚合搭建，保持账户基建</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>复用已验证的高潜力素材，是投放最常见的策略方向之一。而在升级版，高ROI素材可进行聚合搭建，系统会寻找最佳的搭配组合，探索最具跑量潜力的素材，节省测试时间。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>3、减少广告数量，增加广告素材，集中跑量</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在升级版，每条广告下最多可添加10条素材。如果是已经验证跑量的老创意，那么可以直接铺满集中跑量，如果是新创意需要测试，那么单广告下添加2-3个创意即可，这样可以避免太大的波动，便于控制账户。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>4、实战经验分享</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>实战一：避免冗余工作，提高工作效率是基础</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1383119473632c0ca74c8495.01375513.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>金牌投手：谢莹莹</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>投放过程有很多重复工作，尤其在账户消耗量级要求较大的情况下。而巨量引擎效果广告升级版为了最大限度的降低账户操作复杂程度、避免优化师的冗余工作，在账户结构上也做到了最简化。金牌投手谢莹莹在测试后，有这样的体验：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“对比原版，升级版计划搭建效率更高，而且素材和落地页全量在线优选，避免了部分素材落地页审核不过而影响整个广告计划跑量的问题，所以在账户搭建审核板块就能节省40%的时间。同时，拿量相对更容易，通过出价探索起量，能减少无效消耗，快速测出高潜力素材，从而也缩短了整个的测试周期，这对提升工作效率有很大的帮助。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>实战二：复用高ROI素材聚合搭建，攫取垂直方向流量红利</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1598054756632c0caa5885f6.97572749.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>金牌投手：刘斌</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“复用优质计划方向，保持账户基建数量，做好计划间承接”，是所有普适性投放建议中最有效的。关于优质计划的衍生复用，金牌投手刘斌表示：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“在一定时间周期内，单一的投放方向确实存在流量红利，并且新计划成功率明显提高。尤其是在各种节点大促期间，明显感知起量困难。在这种情况下我通常不建议去测新素材，而是直接复用高ROI素材。一个是直接复制使用，另一个是使用巨量引擎效果广告升级版的聚合搭建，利用程序化探索，拉长素材生命周期。通过这两条思路指导操作，账户整体环比去年同期消耗提升50%。”</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>新旧平台打配合稳定拓量</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在大量测试后我们发现，升级版拥有更强的跑量能力，给予素材更多空间和机会，尤其是对新素材的流量扶持相对较大。而将新旧平台结合起来使用，能够达到1+1>2的效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于升级版的使用方法，给大家4个建议，可参考复用：</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>1、新素材直投升级版快速度过冷启动期</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于没有数据积累的新素材，可优先使用升级版进行投放。在投放逻辑上，升级版对素材探索能够给予更多流量扶持，帮助账户快速度过冷启动期。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>2、老素材新平台，跑量能力更强</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原版跑量TOP的广告、素材，均可平移至新版，跑量能力更强。目前升级版的竞争没有那么激烈，相对来讲仍有流量红利。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>3、升级版测出的高潜力素材，衍生到原版进行补量</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在素材测试上，升级版比原版系统更有优势，也能更快测出具有跑量潜力的素材，因此测出来的素材直接复用到老系统，仍可以起到补充流量的作用。提高测试的效率和成功率。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>4、原版+新版，双管齐下，提高广告账户转化量</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于一些需求较大的广告主，完全可以两个账户同时跑，注意观察对比成本数据，一般情况下同样的素材在升级版能跑出更多的量，如果持平或者低于原版，则需要进行账户排查。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>5、实战经验分享</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>实战一：放量困难别着急，找到方法最重要</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1480172183632c0cacde8a92.39038150.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>金牌投手：郑爽</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在起量、跑量之后，计划会到达一个瓶颈期，量级很难再继续突破。由于账户模型已经稳定，而为了拓量做出的调整很可能会让一个如日中天的计划一落千丈。因此拓量一定是稳中求胜，不可激进。而根据我们上文提到的新旧平台打配合来拓量的方法，金牌优化师郑爽通过测试总结了自己的经验方法：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“最近在跑的项目，相对稳定，但是需要进一步扩大量级。以往拓量通常会用到的方法是调整预算和出价，新建更多维度创意文案素材，或者新开账户投放。而这一次我选择了另一种方法——老账户新平台复用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了对比投放效果，我搭建了两个版本：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1）使用原版的老账户，直接在升级版中进行投放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2）在升级后的版本里重新开账户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后对两个账户在同一期间投放相同素材，通过数据我发现，老账户在升级后版本的跑量更快、且数据表现更好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次的测试，给了我拓量的新思路，除了在原有账户上的调整外，还可以新增平台来拓展。尤其是已经跑稳的素材，在新平台上的复用可以达到稳定拓量的目的。”</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>总 结</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后针对文章开头提出的问题，我们再做一次总结。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>冷启动阶段</span></strong><span>：把量跑起来。复用已验证的跑量创意，平移到巨量引擎效果广告升级版，以素材为核心快速拿量。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>跑量阶段</span></strong><span>：把成本降下去。筛选高ROI创意组合搭建，通过全量创意在线优选，最大限度发挥高潜力素材的作用。暂停无效创意的投放缩减成本。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>拓量阶段</span></strong><span>：把转化提起来。新旧平台配合使用，双管齐下提高整体的消耗和转化，从而达到拓量目的。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论平台如何变化，高潜力素材永远是账户的核心，因此测试素材一定要快狠准。同时，对于平台的升级和变化，我们要做到及时的听到、看到、且用到。保持学习的驱动力，积极尝试平台新的工具和功能，在存量市场下探索更多流量机遇。</span></p>
