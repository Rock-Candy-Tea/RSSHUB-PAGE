<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;"><strong><strong>冰柜是渠道的象征。</strong></strong></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前段时间聊</span><span>和路雪</span><span>的时候，我提到了他们当年抢滩中国市场的重要动作，「冰柜战略」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，铺渠道、抢冰柜，并不是和路雪的首创，像<a href="https://www.niaogebiji.com/tag/%E5%8F%AF%E5%8F%A3%E5%8F%AF%E4%B9%90" target="_blank">可口可乐</a>这样的行业巨头也曾靠这套打法行走全世界。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/95913761562d6ad752da499.49799567.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这么说吧，在快消品市场，无论你是卖冰淇淋的，还是卖饮料的，打冰柜战都是一套虽然看上去有那么点过时但又屡试不爽的招法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是近两年，可口可乐、农夫山泉等传统饮料霸主合力围剿新秀「<a href="https://www.niaogebiji.com/tag/%E5%85%83%E6%B0%94%E6%A3%AE%E6%9E%97" target="_blank">元气森林</a>」，将冰柜战推上了新的历史高度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这一期，我想和大家来聊一聊国内商业市场上几次著名的冰柜战争，以及为什么大家都不肯放过冰柜？</strong></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>01</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，在聊「冰柜战争」之前，我不知道大家有没有想过：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>为什么我们要管立式的叫「冰箱」，横卧式的叫「冰柜」？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟这样的称呼似乎不太符合我们的一贯认知。就像马未都在《百家讲坛》中说的：</span><strong><em><span>「在中国人的概念中，柜要有门，箱要有盖。上开盖为箱，横拉门为柜。」</span></em></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>按照这个标准，我们一般认为的冰箱应该叫冰柜，冰柜应该叫冰箱。听起来有点绕。其实冰箱和冰柜，不仅是音译的差别，也牵扯到它们背后的发展历史。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总的来说，这两者都属于制冷设备。<strong>只是冰箱兼具冷藏和冷冻功能，冰柜更专注于冷冻。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/43721387062d6ad7581f524.26771835.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>它们的诞生都离不开制冷技术的发展。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自从1834年，有一个叫做弗雷德里克·图德的波士顿商人，将冻结的新英格兰湖冰块运往加勒比海、欧洲、印度等地之后，人们就发现用冰块制冷是一门好生意，而且是暴利的生意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但人工制冰的发展多亏一名叫做约翰·戈里的医生。1840年代，美国佛罗里达州疟疾和黄热病流行，病房里的冰块不够用了，戈里本着医者仁心，利用空气膨胀时吸收热量的方式冷却空气，发明了制冰装置为病人降温，后来还申请了专利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，人工制冰技术越成熟，图德的冰块生意越受影响，于是图德开始抹黑戈里的发明，造谣他生产的冰块被细菌感染。因此，戈里的制冰机并没有得到大规模生产。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，那是一个各项科技蓬勃发展的时代，不止有戈里，同一时期还有不少人都在琢磨着制冷技术的发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如1854年，澳大利亚人詹姆斯·哈里森就发明了第一台机械制冰机，并应用在了酿酒厂；1879年，德国化学家、工程师卡尔.冯.林德发明了以氨为制冷剂的冷冻机。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这之后，经过一次次的反复改进，<strong>到1922年，瑞典工程师布莱顿和孟德斯发明了世界上第一台电冰箱。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/126554126562d6ad75d3c2f7.30632643.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这之后，电冰箱进入了中国市场。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在当时的人们看来，这种制冷设备，很像我们清朝时就有的「冰箱」，只不过是通了电，于是就用「电冰箱」来称呼，而把后来传入国内的具有冷冻功能的「冰箱」称为「冰柜」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般来说，除了工业上的运用之外，冰箱、冰柜等制冷设备也被广泛运用在食品行业，像是速冻食品，以及冰淇淋。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>尤其是冰淇淋，冰柜就是存放它们必不可缺的货架，而且作为一种即食型物品，冰柜也是它们的展示架。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>因而，在冰淇淋行业，抢夺冰柜是各商家的惯用手段。</strong></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><em><em><span>02</span></em></em></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，要说最早认识到冰柜的作用，并运用冰柜来占领市场、大打广告的还不是我们所熟知的和路雪，而是早在上世纪30年代就风靡上海的「美女牌」雪糕。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/144654309062d6ad76304f11.40808760.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其生产厂家是原本做蛋品出口业务的海宁洋行，由美商海宁生于1913年创办。1925年，海宁洋行为解决蛋品加工的淡季，决定引进美国的冷饮设备，想要生产冰淇淋。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/115616088262d6ad7662f522.49104284.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>丰厚的外商资本再加技术加持，使得「美女牌」冷饮，包括棒冰、雪糕、冰砖等系列产品在30年代甫一上市就大受欢迎。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而海宁洋行为了扩大「美女牌」的品牌影响力，除了在当时的各种主流报刊上刊登广告之外，还一次性从美国运来了500台功能齐全的大型「电冰箱」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些专门定制的柜台式大型冰箱，现在看来其实就是我们所说的冰柜。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在当时，海宁洋行规定各经销点只要能保证冰柜内只放「美女牌」冷饮产品，就可以免费使用，这种排他性的做法和现在各大厂家所用的如出一辙。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在当时的市场环境下，这一招无疑是降维打击。差不多的情况，在90年代中后期和路雪进入国内市场之后重演了一遍。我就不赘述了。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>03</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>而后，用冰柜来推广自家产品基本成了各大冰淇淋厂商的常规操作，但要说玩出新花样的还是前几年突然蹿红的「东北大板」。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>生产商是1992年在黑龙江大庆成立的红宝石。一些东北的同学小时候可能吃过这个牌子的冰棍儿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但它旗下的「东北大板」却是一款诞生于2013年的新品，而且从它的走红路径来看，更像是不折不扣的网红产品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/24488194862d6ad768c25b5.90567949.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>首先，作为东北品牌，「东北大板」的首发站放在了上海。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们知道，在很长一段时间内，国内的冰淇淋市场是以伊利、蒙牛为代表的两大奶企品牌，以和路雪为代表的外商品牌，以及在各地各自为营的本土品牌为主。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新品牌想要突围而出，尤其是还想在全国范围内打开市场，并不是件容易的事儿。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而东北大板，选在了资源最集中的上海作为突破口，竞争最激烈，但机会也最多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>其次，东北大板走的是差异化路线。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他并没有一开始就去打通大型商超这样的渠道，也没有和已有市场地位的其他成熟品牌们硬扛，而是将精力放在路边的各类小店和报刊亭、甚至奶茶铺等地方上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/141633813762d6ad76b12bd8.05912500.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这一招，最重要的武器就是冰柜。</strong>东北大板省去了中间商环节，直接和这些终端的零售小店接触，将冰柜放到他们店里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且，比起和路雪、伊利等大牌冰柜，东北大板的冰柜体积更小，以绿色作为主色调，再加以红底绿色的字样作为主招牌，即便是和其他的冰柜并排放在一起，也具有一定的辨识度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，小冰柜就是我们这些消费者接触东北大板的第一印象。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>而东北大板本身朴素的包装，相对低廉的价格，再结合那几年由影视掀起的一股复古怀旧风潮，恰到好处的呼应了大众的情怀。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>再辅之以社交媒体的宣传，从2014年开始成了冰淇淋界异军突起的存在。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到如今，说起东北大板，其「小冰柜」突围策略依然是绕不开的话题。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>04</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在饮料行业，冰柜也是必需品。众所周知，冰过的可口可乐比起常温的，口味上升不止一个档次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在1927年，可口可乐就进入了中国市场，并在上海设立了中国第一家装瓶厂。当时在上海，可口可乐还建起了专门的售卖亭<span>（类似于现在的报刊亭）</span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/170978366462d6ad76ed0023.79550630.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，可口可乐最常出现的场所是在上海总会大楼、外侨俱乐部和高级饭店、西餐厅等地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>三四十年代，可口可乐借助大量报刊广告，进一步打开了知名度。并且，其以小冰柜的投放形式，让可口可乐走出了高级餐厅，深入到大街小巷。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那时候，带有可口可乐招牌字样的小冰柜是街边水果摊、烟酒店、小餐馆、电影院等地的标配。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这是可口可乐进入国内市场的第一阶段，也借此完成了大众对可口可乐的初始品牌认知。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>等到可口可乐重回国内市场的八九十年代，其身处的是更复杂的市场环境，不仅要面对国产品牌的崛起，还有它的老对手百事可乐的不断狙击。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>可口可乐除了降价促销、疯狂砸钱做广告宣传之外，以冰柜占据线下售卖渠道，也是其占领市场的重要举措。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/171703149662d6ad771d9af3.98152898.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，可口可乐在九十年代初发起过<strong>「终端革命」。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>将冰柜免费提供给零售店，并建立了一支庞大的业务员队伍，给他们配备摩托车，让他们直接将产品送到各个终端的零售店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2001年起，可口可乐开始实施<strong>「101计划」</strong>，与二、三线市场的分销商展开亲密合作。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了给零售商们提供冰箱、冰柜、展示架等硬件器材之外，还给他们提供相关行业知识的培训。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/39416900562d6ad7767d409.56958269.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为具有先发优势，再加上财大气粗，可口可乐的这些做法还是对其他品牌形成了有效的压制。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，在争夺市场战况最激烈、胶着的阶段，因为战场的原因，可口可乐的冰柜战略差点也无用武之地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不知道有多少同学还记得，小时候曾有过一款叫做「非常可乐」的饮料？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/63294070362d6ad779d3536.54270553.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这款由哇哈哈在1998年推出的饮料，</span></strong><span><strong>很讨巧地以「爱国」的民族使命感树立了大众对其的品牌认知，走的也是农村包围城市路线，以广大的农村小卖部作为根据地，一步一步占领市场。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短短几年，就成为继可口可乐、百事可乐之外的第三大饮料品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/25660208162d6ad77deca81.06378539.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，反应过来国内其实除了大城市之外，还有一片广袤大地的可口可乐从2003年开始，也盯上了农村市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，可口可乐在甘肃建厂，进一步保证其生产力；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/85153977362d6ad78358e94.08155604.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而后以浙江、福建等东部沿海城市的乡村作为先发地，投放了一批可口可乐的货架和冰柜。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后再以7000台冰柜的猛烈攻势准备拿下各地的农村市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，就算可口可乐有心，奈何当时国内农村市场的整体消费环境还跟不上。幸好，后来随着国家推动农村夫妻店的建设，为可口可乐的推行冰柜战略提供了基本的设施条件。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>再加上，</span></strong><span><strong>伴随着赠送冰柜，可口可乐还推行了「一元钱可乐计划」。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/10425151262d6ad78611795.93289011.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>极低的价格，以及喝完退瓶的消费方式也符合农村地区的消费习惯，使得可口可乐逐渐在农村市场站稳了脚跟，甚至以更广的覆盖率挤占了非常可乐的市场份额。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>05</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样的，在国产饮料市场，尤其是瓶装水赛道，曾经大战娃哈哈，手撕康师傅的农夫山泉，绝对是过五关斩六将的斗战胜泉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/42113967462d6ad78862718.16225318.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其创始人是记者出身的钟睒睒，因而农夫山泉擅长制造话题、搞营销战，这点相信大家都有所耳闻。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/156455018562d6ad78b51599.44509835.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>钟睒睒</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><strong><span>「农夫山泉有点甜」「我们不生产水，我们只是大自然的搬运工」</span></strong></em><span>等广告词深入人心。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/107203003862d6ad78e87dd1.33372443.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但另一边，多年来，为了争夺市场，其在铺渠道方面也是老手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据农夫山泉2019年的财报数据，截至当年末，农夫山泉覆盖全国有超过237万个终端零售网点，其中78.9%位于低线城市，有超过36万家终端配备了农夫山泉冰柜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也正因此，当2020年10月，元气森林举办经销商大会，唐彬森提出要向终端铺设8万台单开门6层，内嵌摄像头的智能冰柜，大举进军线下销售终端之时，农夫山泉感觉到了冒犯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/120412649762d6ad7931aec3.45452402.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：界面新闻</span></h4>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>06</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，受到威胁的不只是农夫山泉。在一个格局已经稳定多年的市场里，最怕出现的就是一条这样的鲶鱼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>比起农夫山泉、娃哈哈，以及可口可乐、百事可乐等品牌，元气森林绝对算是来自另一个世界的新物种。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/190822131062d6ad7996bd53.15335106.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>其成长于互联网时代，走的是「轻量化」路线，以线上销售渠道杀出了一条血路，而且主打的「无糖气泡水」盘活了一个新赛道，凭此在四五年的时间里就成为了行业新贵。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>元气森林可谓来势汹汹，因而当它决定看向传统企业，开拓线下市场的时候，自然会受到重重阻力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，在对待元气森林这个对手上，这些传统的品牌们，就像是江湖上的几大名门正派。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>纵使过往有诸多的江湖恩怨，但彼时彼刻，为了共同的目标，他们也会暂时握手言和，统一战线，然后合力围剿这个魔教之徒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，在元气森林高调下场之后，<strong>农夫山泉立马针锋相对地推出了「天降财神」活动，其口号之一就是：「抢战竞品冰柜，买回我司冰柜。」</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/52336213062d6ad79e1ace6.92765109.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在今年，两大可乐品牌更是在内部放下了狠话，要干倒元气森林。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>元气森林肯定也知道前路艰难。毕竟在2019年之前，它们都还没有冰柜，其气泡水在线下售卖之时，往往需要寄身于农夫山泉等品牌的冰柜之中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对比农夫山泉拥有的近40万台冰柜，以及可口可乐的近百万台冰柜数量，哪怕元气森林真能一下子把8万台冰柜都铺到位，那也是小巫见大巫。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但元气森林也算是抄了一点近道，其充分利用自己的互联网基因，主力投放下沉市场的是智能冰柜。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/174108860062d6ad7a38ca05.15173985.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在去年11月唐彬森接受「晚点」采访时，被问及</span><strong><em><span>「有哪些事是农夫山泉会做但你们不会做的？」。</span></em></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他的回答是：</span><em><strong><span>「我们自己研发智能冰柜，它可以提醒业务员没货了去补，而不用到现场去验。农夫没做这些。」</span></strong></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>某种程度上，智能冰柜不仅仅是大家印象中的产品陈列柜、展示台，它更是一个终端数据库。能够通过联网，收集所有的一手销售数据甚至是用户画像，从而实现更精准的广告投放。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而从元气森林销售数据的变化，也可以直观反映其智能冰柜投放效果。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年3月，其销售力相对无冰柜的销售力平均增长了118%。此外，对比传统冰柜半年的统计，智能冰柜单柜销量平均提升74%，增长效率提升472%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，如今来看，唐彬森彼时评论农夫山泉的这番话显然为时过早。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>智能冰柜并非拥有太高的技术壁垒，元气森林能投放，这代表农夫山泉，包括可口可乐等其他品牌同样也能。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/64067086162d6ad7b26e2d0.33538650.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，就在去年底，农夫山泉董事长钟睒睒就在公司大会上宣布，将斥资20亿全力推进4门及3门智能冰柜在终端的应用，而且已经向智能冰柜厂家下了3万余台的订单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想来，这场饮料市场的冰柜战战火已经点燃，各方都会陆续下场，最后拼的无非是谁更舍得下本，谁拥有的资源更多。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>07</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而从目前来看，「冰柜战」将会是一场持久战。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说到底，不管是以前的传统冰柜，还是现在的智能冰柜，对于这些品牌来说，冰柜都是渠道的象征，打冰柜战，为的就是要让自己这一方牢牢把握住销售终端。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其对于饮料、冰淇淋这样的快消品来说，夸得天花乱坠的广告宣传、再强有力的价值肯定，很多时候都比不上触手可及就能买到手的产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>只有商家们能把自家产品铺到离消费者最近的地方，能让大家即使在再偏远的地方都能买到，才能完成消费的闭环。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以想见，在中国这么大的市场，总还会有各品牌没覆盖到的地方。因而，谁能更快地抢占这些销售终端，就意味着能拥有更多的市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>元气森林早期的「互联网+爆款」的玩法渐渐失去了作用力时，也就发现，想要吸引更多的消费者，还是得回到传统的渠道之争。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在渠道方面已经交战了这么多年的老品牌们肯定也不会轻易让新人得手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，现阶段由元气森林挑起的这场冰柜战争无可避免，而且各方都只会更卖力地表演。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但冰柜战也有一定的局限性，大家都是在以最本质的金钱去争夺资源。无非是谁给经销商让利更多，谁给零售店更优惠，谁就会相对更容易拿下。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>所以说，高端的商战往往只需要最粗暴的手段。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多时候，大家并不需要比拼谁的品质更好，谁的饮料更好喝，谁的冰淇淋更好吃，而是直接来断你的生产线，毁你的销路，抢你的公章，让你的产品没地方可卖，没路可走。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>毕竟你的产品和质量对我都不重要，但是没有你，对我很重要。</span></strong></p>
