<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每到七夕各大品牌节点营销层出不穷，围绕情侣们展开的“有爱”营销就开始泛滥，用户对“换汤不换药”的营销套路产生脱敏。</span><span>伴随着消费者们逐渐审美疲劳，品牌们也更难突围。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，今年<a href="https://www.niaogebiji.com/tag/%E5%93%88%E5%95%B0" target="_blank">哈啰</a>品牌却另辟蹊径，不玩出行反推狗粮，做了场别样的有爱营销。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>养宠知识多一点，陪伴就能久一些。</span><span><strong>哈啰</strong></span><span><strong>联合国内知名宠物品牌</strong></span><span><strong>伯纳天纯</strong></span><span>推出</span><span><strong>「七夕限定礼盒」</strong></span><span>，同时呼吁大众科学养宠、关爱宠物。哈啰也借势出圈，抢占七夕热点C位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果让007来总结哈啰这波七夕节营销的突围秘籍，无疑有三点：<strong>洞察有新意、传播有趣味、品牌有心意。</strong></span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>哈啰为何在七夕节点上，瞄准了萌宠圈？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传播做的好不好，和背后的洞察有很大关系。再看哈啰选择了萌宠圈，正是瞄准了背后养宠人群和<strong>哈啰强化生活服务标签的迫切需求</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《2020年中国宠物行业白皮书》显示，2020年仅养犬猫人数超过6000万。QuestMobile数据也显示，国内萌宠群体规模已超过5.3亿，这意味着社交平台上“云养宠”、关注宠物内容的人群已超过半数网民，宠物相关内容，也更易引发热议。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这些热度数据背后，一种流行生活方式也正在崛起：越来越多年轻人只想搞事业，不想谈恋爱，一猫一狗三餐四季就是理想生活。据艾媒咨询《2021年上半年中国宠物经济产业发展现状与市场调研分析报告》显示37.1％的受访者将宠物看成自己的朋友，超过4成把宠物当亲人。宠物已成为不少人生活中重要的精神、情感治愈力量和真正的“陪伴者”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13834711662ec6f3bb6c478.05145708.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正是基于养宠人群的真实情感需求洞察，哈啰在七夕这个充满爱的特殊节点，将品牌传播创意聚焦在</span><span>“陪伴”</span><span>二字，并通过</span><span>“撒狗粮”</span><span>动作进行事件传播，传递着养宠人与宠物们的温暖情感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/206360548962ec6f3c01cea0.21263531.jpeg" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有新意的洞察和创意为哈啰七夕节营销预埋下爆火引线，而在落地过程中，哈啰更凭借力联名夸家，以“有趣味”的内容传播，充分调动了养宠人群的参与热情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1）推出趣味新品，打造“陪伴情感”承载介质</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这款“新品”正是前文提及的<strong>哈啰X伯纳天纯七夕限定狗粮礼盒</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>伯纳天纯作为高端天然粮品牌，一直以“顺应天性”为品牌宗旨，按照“陪伴天性”、“食鲜天性”等 5大天性食谱进行着产品布局，致力于“让爱宠们活得更健康，陪伴更长久”——<strong>这个品牌正与呼吁“陪伴”的哈啰志趣相投</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两个高契合度的品牌首先在联名新品设计上，就有巧思。品牌形象IP“哈哈”被“聘请”为 “新品推荐官”，007发现联名产品图案设计也是以 “哈哈”形象为主视觉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据悉哈哈原型就是一只来自</span><span><span>“哈哈奇幻星球”的 “哈士奇”，外表可爱呆萌，内心幽默正直。礼盒上头戴墨镜的哈哈，斜倚伯纳天纯饭盆，让这份狗粮变得潮酷与萌趣。</span><span>更重要的是，哈哈作为“品牌新品推荐官”，本就对养宠人群有天然吸引力，</span><span>作为品牌IP形象的哈哈顺势将哈啰品牌亲近宠物、活泼、会玩的形象一起深植人心。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/130764499762ec6f3c3a3987.88060439.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅设计有巧思，哈啰还煞有介事地为这款新品设计了“social海报”，为新品做起了推广，认认真真做狗粮，“出行品牌推狗粮产品”这种自带的新闻点和反差萌，成功吸引了不少关注。简单朴实的海报文案，承载养宠人与宠物情感的介质，不仅有趣也更有温度。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2）线上线下联动用户加入“陪伴行动”</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>针对这些趣味狗粮新品，哈啰还专门绑定自身业务，为其打造了一个线上、线下结合“流通渠道”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>活动期间内，线上用户只要在哈啰APP-哈啰宠物社区分享<strong>#养宠冷知识#</strong>， 即有机会获得七夕限定狗粮。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/316510562ec6f3deacfd0.37057489.jpeg" referrerpolicy="no-referrer"></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>哈啰APP–网友话题帖分享真实截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在分享互动中，品牌“养宠知识多一点，陪伴就能久一些 ”的呼吁，也变成了养宠人们的实际行动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，七夕节当天针对上海地区的用户进行惊喜放送800份，骑哈啰单车就能领真·狗粮。通过每天都陪伴大家的哈啰单车展开“撒狗粮”活动，广泛触达养宠用户，也让品</span><span>牌狗粮撒得精准。哈啰的小心机，有谁想得到？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/135696360162ec6f3ea1a895.51796488.jpeg" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><span>3）</span><span>打造社交热点，扩大“撒狗粮”事件传播</span></span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>趣味的新品与活动，本就自带传播力，而哈啰还趁势为扩大传播再添一把火，上线了<strong>#哈啰撒狗粮了#</strong>话题。话题下，哈啰的各种脑洞操作，吸引了不少KOL、蓝V纷纷下场打call，并由此带动越来越多的网友也加入，成功引爆话题讨论。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/139404314962ec6f3f473791.83198407.jpeg" referrerpolicy="no-referrer"></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>部分博主内容和网友评论区互动真实截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友直呼：“被哈啰笑死！”“哈啰什么时候出猫粮？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面“有趣”的热点事件带动着讨论，而另一方面，哈啰还借着热点输出了一系列“有用”的养宠知识点，带动了自传播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅有出行场景，“哈哈的温馨提示”还覆盖到了宠物们和主人生活的方方面面。比如收好数据线、高层关窗等等，将<strong>“养宠知识多一点，陪伴就能久一些 ”</strong>的呼吁，从品牌活动带到了社交热议中心，带入了更宽泛的人群中传播。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从营销立意到营销落地，整个<a href="https://www.niaogebiji.com/tag/%E4%B8%83%E5%A4%95%E8%90%A5%E9%94%80" target="_blank">七夕营销</a>让我们看到了一个“不一样的哈啰”——<strong>一个不仅有新意，更有“心意”的哈啰</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在大家过往印象中，提到哈啰第一时间想到的可能就是“哈啰单车”、”哈啰出行”。但在此次七夕营销中，哈啰并未过多展示其出行业务，更是更加深入养宠人的生活场景，去传递品牌温度，建立与用户的情感连接。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年4月， “哈啰出行”宣布正式将品牌升级为“哈啰”，并启用<strong>“陪伴生活每一天”</strong>的全新品牌口号，确立了<strong>“基于出行的普惠生活服务平台”</strong>的发展方向，从移动出行平台正式走向本地出行及生活服务平台。 </span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/171094704662ec6f421483b3.73586865.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也就是说，哈啰不再仅专注于“出行”业务本身，而是将以“出行”为链接点，将触角延伸到更广泛的用户生活场景，发展“本地生活服务”业务战略，不断夯实“生活陪伴者”的品牌角色。 </span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>此次哈啰七夕撒狗粮活动，就是从用户养宠生活为切入点，强化</span><span>“普惠生活服</span></span><span>务”的品牌标签，</span><span>并将</span><span>哈啰APP“生活分享社区”的概念广泛传递出去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户参与活动时，也会无形中将宠物陪伴养宠人群，类比到哈啰一直在凭借产品和服务贴心陪伴用户，由此形成了品牌与用户更深厚的关系链接</span><span>——不仅是生活陪伴者，也是情感陪伴者。</span><span>哈啰与用户建立起的“陪伴”关系，显然相较于同行品牌与用户间的单纯“消费”，更加亲密。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/147352600562ec6f42444052.62123434.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在007看来，这样的品牌营销思路焕新和品牌成长进阶实属必然。在各大消费领域，当代用户尤其是年轻用户都反映出了同一诉求：在物质、功能基础上，追求着精神、情感、自我价值实现等高阶满足。<strong>哈啰成为“生活陪伴者”，显然是在跟随着用户步调前行。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不难预见，在既定的品牌成长方向和脑洞大开的创意思维下，哈啰必然还会继续带来有趣又有意义的营销惊喜。下一次，这个品牌的目光又将投向哪个生活领域？我们可以继续期待。</span></p>
