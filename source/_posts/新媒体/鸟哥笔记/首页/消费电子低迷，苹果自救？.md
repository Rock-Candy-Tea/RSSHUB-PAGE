<div style="height: 6px"></div>
                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="1" data-pm-slice="0 0 []">消费电子的市场环境可能还在恶化。7月末，<a href="https://www.niaogebiji.com/tag/%E8%8B%B9%E6%9E%9C" target="_blank">苹果</a>官方在国内罕见推出打折促销活动，涉及手机、手表、耳机等产品。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="3">这被外界普遍认为是去库存的必要手段之一，但苹果CEO库克后予以否认。仔细去看，这次打折优惠不仅时间偏短，而且只覆盖到苹果的部分产品。无论从体量，还是折扣上看，都远不如618、双11这样的大促。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="4">苹果也因此引来“诚意不足”的质疑。要知道，今年618卖出的手机中，苹果可是占了一半。问题在于，618过去才不到两个月，而且电商大促中的出力方还有电商、经销商，此次是苹果独揽折扣活动，多少还是带有试水味道。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="5">在非大促时期主动进行促销，这比促销本身更值得挖掘。苹果对于降价的态度一直比较模糊，甚至比较排斥和打折促销这样的字眼有关联，比较有象征性的事件就是苹果一直未在有着最便宜正版iPhone手机的拼多多设立官方旗舰店。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="6">恶化的外部环境也在持续刺激着苹果。IDC、Canalys、Omdia几家的调研数据显示，2022Q2，全球智能手机、PC出货下滑趋势没有改变。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="7">强悍如苹果，也难独善其身。根据IDC，2022Q2苹果智能手机出货仅同比增长0.5%，平板全球出货同比下滑2.9%，PC全球出货同比下滑22.5%。虽然手机业务相较OPPO、vivo、小米，苹果出货表现更为强势，但增速疲软亦肉眼可见。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="8">不过，与不乐观的外部环境相比，真正让苹果感到压力，必须要做出一些改变的原因，其实是苹果自身需要用钱的地方太多了。</p>
<h4 class="pgc-img" style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/ffd90fbf736649198f201f1604f68afd" data-ic="false" data-width="1080" data-height="720" data-ic-uri referrerpolicy="no-referrer"><span class="pgc-img-caption">配图来自Canva可画</span></h4>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" data-track="10"><strong><span style="color: #0052ff; --tt-darkmode-color: #005CFF;">支出压力大</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="11">作为消费电子巨头，苹果在业务扩展上一直比较克制，更多还是围绕手机、电脑、平板和可穿戴等消费电子硬件进行布局。早先谈到的自动驾驶、企业云计算、AR和VR，甚至包括可折叠手机，苹果的动作对比华为、小米、三星，或Meta、腾讯等其他<a href="https://www.niaogebiji.com/tag/%E7%A7%91%E6%8A%80" target="_blank">科技</a>巨头，显然慢的多。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="12">没有积极开疆拓土的苹果，似乎钱包不会紧张，但事实可能恰恰相反，苹果对钱的渴望正在不断膨胀。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="13"><strong>第一，新业务提速。虽然进度明显慢于同行，但今年苹果在VR、AR、自动驾驶等新业务上有显著的提速。</strong>一方面，迫于手机、电脑等传统消费电子业务的低迷，苹果必须寻求更多变现来源；另一方面，新业务具备更好的利润前景。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="14">有消息称苹果的首款AR或VR设备有望在年底启动量产，并最早于2023Q1面世。此外，兰博基尼高管最近也被爆正式加入苹果的造车项目。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="15">2021年，Meta为元宇宙花了100亿美金，通用不久前将自动驾驶业务截止2025年的投入预算从270亿美元提高到350亿美元。无论是XR业务步入量产期，还是造车项目加速落地，都会给苹果带来相当大的资金压力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="16"><strong>第二，补全技术图谱。为了掌握核心零部件的绝对控制权，苹果多年来一直坚守自研战略，从电脑CPU到手机基带芯片，苹果的目的都是换上自家的东西。</strong>5G时代到来后，基带芯片一直是苹果自研的痛。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="17">最近媒体报道苹果花了4.45亿美元盘下了惠普园区，有意加速基带芯片研发，以替换高通。虽然苹果有大量成功的自研经验和技术储备，但基带芯片难度很大，在业内普遍看来需要更长的时间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="18">在不少科技公司抢先布局6G的背景下，苹果必须加速相关核心技术的布局和商业化，更快树立5G时代的核心竞争优势，全面向自研生态靠拢。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="19"><strong>第三，订阅服务支出增加。为了提高内容订阅服务的变现能力，苹果在内容端的支出正在加速。</strong>6月份，苹果与美国职业足球大联盟签订了长达十年的流媒体合作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="21">Q2财报显示，苹果各类付费订阅用户总数达到8.6亿，过去12个月增加1.6亿。和赛事、影剧相关的内容布局，正在成为苹果为提升付费订阅用户规模的重要手段。苹果极力向迪士尼、Netflix等内容巨头看齐，后续的内容支出速度会只快不慢。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="22"><strong>第四，新诉讼不断。</strong>因为过高的佣金提成，法国、西班牙、葡萄牙等地开发者近期向苹果提起诉讼；因为垄断，苹果也面临德国、俄罗斯等地相关机构的起诉；因为键盘问题，苹果近日已支付数千万和解费用。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="23">从过往案例来看，苹果这样的大公司最终都要以一笔不小的费用终结各种诉讼，某种程度上，这些都是业务模式所带来的法律风险，苹果可能会长期受此困扰，付出更多的金钱代价。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="24">除此之外，计划放慢招聘速度，也从侧面反映出苹果资金变得更加紧张。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" data-track="25"><strong><span style="color: #0052ff; --tt-darkmode-color: #005CFF;">利润垫变薄</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="26">除了打折促销，苹果也在想方设法增收，比如售卖翻新机、推出闪送服务。因为苹果意识到一个比较现实的问题，即由于各种不利因素影响，过去很厚的利润垫正在变薄。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="27"><strong>首先，硬件吸金能力下滑。</strong>从毛利和毛利率来看，财报显示，2021Q4、2022Q1、2022Q2硬件业务的综合毛利率分别为38.4%、36.4%、34.5%；毛利分别为401.2亿美元、281.7亿美元、218.7亿美元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="28">原因有不可抗力，比如汇率，美元走强不利于苹果的海外业务，也有运营策略调整，比如促销打折，推出更多型号的低价手机。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="29"><strong>其次，服务吸金动力衰减。</strong>财报显示，2022Q1、2022Q2，苹果服务业务的营收分别为198.2亿美元、196亿美元，同比增速分别为17.3%、12.1%；毛利分别为143.9亿美元、140.2亿美元，同比增速分别为21.5%、14.8%。环比维度看，2022Q2服务业务的营收、毛利、毛利率均有所减少。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="30">从苹果此前的策略来看，服务业务的增长动力主要来源两方面。第一，订阅费促销活动，比如Apple Music此前推出的“10元畅听3个月”活动；第二，硬件新用户增量，在Q2财报电话会，苹果高管提到几个数据：iPad安装超一半是新用户，购买Mac用户中接近一半为新用户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="31">但服务业务也存在隐患，一是数字广告业务因大环境表现疲软，二是付费订阅用户规模增速可能见顶，2021Q2到2022Q2的五个季度，苹果付费订阅用户总规模分别为7亿、7.45亿、7.85亿、8.25亿、8.6亿，季度新增呈下降趋势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="32">对苹果来说，躺着赚钱可能也不太现实了，硬件业务作为苹果的赚钱大头，已经相当长一段时间处于被动成本叠加上升的状态，软服务虽然毛利率高，但驱动效果在减弱。Q2电话会议上，苹果对Q3的综合毛利率预期是41.5%到42.5%，比Q2低，意味着苹果的利润垫还要承压。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" data-track="33"><strong><span style="color: #0052ff; --tt-darkmode-color: #005CFF;">舍“硬”求“软”</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="34">面对变薄的利润垫，苹果的选择是舍“硬”求“软”，通过打折促销或推出低价产品来推高手机、电脑、平板等产品的出货，从而为音乐、影剧、游戏、售后、个人云服务等付费订阅服务引入更高基数的消费人群。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="35">更准确地说，是苹果舍弃了硬件业务的部分利润，转而去换取软件服务的利润增量。从持续增长的付费订阅用户规模来看，苹果这种交换比较成功，至少目前是如此。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="36">苹果之所以会选择这样的业务增长策略，有三个重要原因。<strong>第一，订阅服务的基础是硬件产品，或者说，硬件产品是订阅服务增长的前提。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="37">订阅服务的消费特点是强周期性，比如一个月或一年，所以如果在一个订阅周期内没有新增硬件用户，订阅服务增长基本会停滞或大幅减速。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="38"><strong>第二，订阅服务有更高的毛利空间。</strong>以2022Q2为准，苹果硬件业务的毛利率是34.5%，服务业务的毛利率是71.5%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="39">用硬件撬动服务，从毛利水平看是一桩稳赚不赔的买卖。而且高毛利率业务收入占比的提高，对苹果的综合毛利率也会有正面作用。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="40"><strong>第三，新兴市场给予正反馈。</strong>通过战略聚焦低渗透率市场，今年二季度苹果营收在印尼、越南实现两位数增长，在印度几乎翻倍。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="41">虽然营收占比偏低，但这些市场的高速增长仍对苹果具有特殊的价值。第一，相对更高渗透率的其他市场，这些低渗透率市场的硬件业务更容易实现增长。根据CRM，苹果手机Q2印度出货约120万台，同比增长94%；第二，低渗透率意味着更高的未来成长空间，可以成为苹果未来增长的核心引擎。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" data-track="42"><strong><span style="color: #0052ff; --tt-darkmode-color: #005CFF;">保守自救</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="43">没有人知道消费电子的拐点何时到来，从整个产业链来看，上半年的情绪普遍不太乐观，终端厂商要么砍单，要么加大促销力度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="44">理性看待，疫情、通胀、能源紧缺的综合扰动下，消费电子的低迷可能还要持续较长一段时间，苹果选择主动促销，也是一种保守自救。如果市场环境没有改善迹象，苹果的促销频率应该会进一步提高，从而换取更多的软收入。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="45">和亚马逊、华为等科技巨头相比，苹果可以拿得出手的自救手段其实并没有那么多，一是因为苹果的战略主场始终围绕消费互联网，产业互联网业务布局单薄，To B基因比较少，二是因为苹果的高端化标签，决定其较难开拓收入水平较低的市场。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="46">其实折叠屏手机、元宇宙计划的进度已经证实，苹果断然属于战略保守派。但苹果也在尝试改变，强化企业服务基因，比如把苹果iPhone和Mac以更低的价格批量卖给一些大型企业。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="47">苹果需要更加激进，即使不去面向企业拓展业务边界，也可以通过拓展更多个人化的软件服务来实现增收，比如社交、出行等。</p>
