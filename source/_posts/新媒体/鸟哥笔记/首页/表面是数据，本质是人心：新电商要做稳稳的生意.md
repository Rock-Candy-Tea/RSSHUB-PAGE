<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%95%86" target="_blank">电商</a>，老电商，区别是什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很简单，最大的区别就是，</span><span><span><strong>新电商的生意发起点真正聚焦在了某个点上。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个点，可以理解为一件商品，一种兴趣，某些潜在的需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/68607895462d6ad66de7c23.42102595.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总而言之它像一块磁石能精确地把人吸引过来，而不是一个抽象的、甚至没人敢保证一定奏效的生意概念，它是一个具体的、能看得见摸得着的something。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有了这个生意发起点，围绕其展开的所有环节都发生了连锁变化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最典型的例子就是抖音上的那些电商成交案例，仅凭直觉也能感受到它们不同于过去多年的电商消费路径，比如短视频展现形式、内容营销植入、精确到不可思议的推荐机制等等，这些前所未有的细枝末节，一起构成了新电商。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>1、</strong></span><span>真实故事背后的生意密码</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个月初，倪叔因为采访的关系在广东那边走了一圈，去了好几家企业，我发现大家无论做什么产品，都在谈论一个共同的话题，</span><span><span><strong>就是一定要集中精力把抖音做好。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>倪叔的一位潮汕朋友，已经做了10年电商。最近也把生意主阵地挪到了抖音。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/161063069762d6ad67b96ae9.46080323.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我和他好好聊了聊，有一些很有意思的细节。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如我问他为什么去抖音，他是这么说的，很简单，</span><span><span><strong>因为除了抖音在别的平台不赚钱。</strong></span></span><span>如果投流这件事在哪里都要做，那评判标准只剩下一个，看谁的投流效果更好，也就是能带来更多的生意增量。据他说，刚开始在抖音投流，每天能成交5000单，粗略算一下也有5万的收益。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这些自己有供应链的商家而言，在抖音做生意相比起来更简单，似乎只需要把传统的营销变成一个个短视频，有了内容再结合投流工具把内容投放出去，剩下的就是等，消费者会沿着内容在你的账号里下单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他说当时也没做什么特别的事情，就是拼命在工厂干活，以此来支撑忽然放大的生意。在我看来，这是一种幸福的烦恼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据他透露，目前他的素材团队一共5个人，两个剪辑，三个出镜的模特。就这5个人的小团队，去年在抖音上帮他挣了多少钱？他说2000万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年又遇到了新的挑战，一方面是大环境的压力，大家其实都遇到挑战。另一方面是“新手红利”正在消散，商家在抖音这种新电商平台长期来看比拼的是多维度的能力，这其实在哪里都一样，长期主义是商业的铁律。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但新电商平台对于商家的生意而言，依旧是有优势的。倪叔今年采访过的不少企业案例也符合这个判断，</span><span><span><strong>大家之所以都跑去做抖音，就是因为这里的生意相对来说更容易找到新的增量。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>背后的逻辑也很简单，因为消费者的注意力正在被短视频和直播为代表的新内容形式所吸引，内容平台上的电商潜力也就逐步被释放出来了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于商家而言，出发点不是内容，这一点和达人是不一样的。商家需要更短的路径去跑通生意链路，所以商家做内容和达人做内容有本质区别，商家内容的焦点一定是更明确的，就是围绕“货”来做。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这在内容的宽度和频率上其实提出了新的要求，所以这位朋友也说相比去年的内容，今年的内容出现了升级，围绕生意链路必须做更优质有效的内容，如果依旧是从前的那几杆枪，打起来就发现不匹配生意增量，承载不住。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们需要更专业更丰富的“锦囊妙计”，在生意上提供新的推力。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>2、</strong></span><span>心里有数，兜里才能有米</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说到这里，就说到了新电商深入人心的真正硬核干货了，其实就是更精准的人货场匹配方式，更有活力的数据，以及对消费者更有洞察力的数据操盘手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓的“数见趋势、数见营销、数见生意、数见生态”，通过有活力的数据可以看见更确定的生意，谁去看见呢？</span><span><span><strong>将供需两端连接起来的洞察者、操盘手，是抖音生意之所以更好做的幕后推手。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们的心里有数，商家的兜里就有米。对于商家而言，这种大数据分析平台就是“锦囊”，囊中有数据，更有生意上的妙计。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月19日，有米云旗下的新电商营销大数据分析平台有米有数召开了一场以“数见·增长”为主题的新电商操盘手大会（NECOC）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/82734518462d6ad67f2cb06.23377980.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以数据服务DaaS为核心价值，操盘手这个概念再次得到了强化，他们能为商家产出的新价值也进一步明确。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种新的变革，引发的是商家的动作升级。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以投流为例，以前投流相对而言还是商家动作的初级阶段，但升级之后的第三方数据营销方案和工具能让生意进一步放大，平台也希望和这些第三方共建一个更大的生态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是前面说的“新手红利”逐渐消失之后，想要再进阶就必然产生的新需求。就好比一个人刚进健身房训练的时候，由于底子薄，怎么练都会有明显的变化。但一年过后，你会发现身体的变化慢下来了，甚至是怎么练都不长了，这时就需要有专业的教练带你练，他们的认知更深刻、经验更丰富、掌握的数据和方法更多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>操盘手就类似于教练，或可称之为新电商教练。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以在2022年下半年一切都在复苏的时间节点，第三方平台的升级来得非常及时。有米云旗下的有米有数，就是在这个时间点第一个站出来的新电商营销大数据分析平台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19715092262d6ad6830b278.14066927.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>有米云VP，有米云国内商业化中心负责人Franky在第一届新电商操盘手大会发言</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于商家而言，其根本优势在于三个方面：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>选爆品-科学选品；看素材-素材参考；做投流-投放提效。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直截了当，契合商家在新电商平台上的短链路生意增长逻辑。选品方面，操盘手通过监测四大新电商平台销量数据，通过6400+细分品类分析，全盘掌握消费者画像、客单、套餐占比等等数据，他们能精确地感知什么品会成为爆品，而背后的逻辑是数据与分析服务，显然这比商家自己琢磨要科学准确。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内容素材方面，有米有数提供及时更新、优质的广告素材库，操盘手们能在第一时间监测到最新的爆款素材，消费者对什么样的文案和内容有积极活跃的转化效应，这不只是简单的模仿传播，而是基于大数据的创意内容灵感激发。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/141256975062d6ad685a25c1.19385366.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有爆品、有内容，最后就到了投放提效阶段。有米有数智能投放工具能帮助商家1分钟创建500+投放计划，在人效比这个指数上是10倍以上的提升，自动化、智能化帮助商家进行内容投放，实时监测投放效果，最终目的是达到商家生意增长预期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可见，这些是2022年新电商平台上最值得期待的价值，对商家来说，能让生意增长更确定，同时变得更省力、更简单。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>3、</strong></span><span>让生意重新好做起来：新电商要解决的根本问题</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从上面的故事也能看出来，商家在抖音这样的新电商平台相比起来生意要好做很多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是大前提，也是新电商平台对于商家而言最大的吸引力，更是新电商要解决的根本问题，</span><span><span><strong>让生意重新好做起来。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是一味简单地认为来了抖音就一定能把生意做好，很显然也不现实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那位朋友也明确说出了他的体感，比如内容素材一开始能撑一个月，但后来周期越来越短，现在这个周期已经变成了七天。也就是说商家平均每七天要出一个系列的新内容以此来吸引新的消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外一个关键点在于，</span><span><span><strong>新电商真正搞定好生意的本质却不是流量，而是人心。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何通过最低成本的方式去深入人心？依托于好产品的优质内容是一方面，这是基本盘，另一方面是通过谁去把这个基本盘和消费者大盘联系起来，相当于在商家与消费者的万径千川之间搭建起一条大家都愿意去通过的桥梁。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很显然，有米云旗下的有米有数就是在做这件事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在7月19日的新电商操盘手大会（NECOC）上，也发布了凝结了这种深刻价值的新产品，标志着新电商的生意会更好做。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如面对商家产品分散、数据割裂的问题，有米有数全新升级，</span><span><span><strong>利用三条数据连通的链路提供投前、投中、投后的一体化价值。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/79789628462d6ad68822628.13206980.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这三条链路具体是怎么运行的？首先第一条链路连通了广告、对应直播间销量、达人账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个很重要，因为如果商家自己来做的话，搜集到的营销情报是分散割裂的，看到一个广告创意好像不错，但不知道它推广的商品或者直播间销量到底如何。再比如商家看到一个品卖爆了，也看到它产出了大量短视频，但并不知道这个爆品背后究竟是短视频打爆的还是直播，亦或是一个多元维度的推广才打爆的，这些幕后的数据真相，是无法通过终端的内容去判断的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而有米云可以帮助商家洞悉爆品背后的内容真相。</span><span></span></p>
