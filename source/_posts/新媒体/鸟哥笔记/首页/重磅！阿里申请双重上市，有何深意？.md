<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><span>一个新的时代正在缓缓拉开帷幕。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span lang="EN-US">2019</span>年，如日中天的阿里回归港股挂牌，一度成为新港股之王；<span lang="EN-US">3</span>年后的今天，阿里正式申请香港为主要上市地，与美国上市享同等地位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>时代的浪潮滚滚，裹挟着阿里再度冲进香江洪流。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>昨日上午，阿里集团发布公告称，“<a href="https://www.niaogebiji.com/tag/%E9%98%BF%E9%87%8C%E5%B7%B4%E5%B7%B4" target="_blank">阿里巴巴</a>目前在香港联交所主板第二上市，将会按照香港上市规则申请香港为主要上市地，预期将于<span lang="EN-US">2022</span>年年底前生效。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/43790025662e1b83f0b0c63.13617996.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/135244624162e1b83f6d9cc2.93019269.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图片来源：阿里巴巴集团公告</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当相关流程完成后，阿里将成为在美国纽约和中国香港双重主要上市的公司。同时，其于美国挂牌的存托股和在香港上市的普通股可以互相转换，投资者可自由选择一种形式继续持有阿里股票。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>从赴港二次上市到双重主要上市，<span lang="EN-US">3</span>年时间里，阿里正在持续拥抱港交所。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>从二次上市，到主要上市</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，双重主要上市已并不算什么新鲜事了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自今年元旦，港交所上市规则修订生效后，拓宽了双重主要上市的接纳度。在法律制度上，也为中概股回归铺好了路，给足了保障。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>至今，已有包括知乎、贝壳、小鹏等在内的<span lang="EN-US">9</span>家中概股公司实现了在美国和中国香港的双重主要上市。</strong></span><span>阿里，或许未必是这之后的第<span lang="EN-US">10</span>位，但一定是最具影响力的其中一位。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/30813954662e1b840089621.17933453.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：阿里巴巴集团官网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于许多朋友而言，或许心中还有着这样的疑问：“阿里不是已经在港股上市，为何又要上市？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，</span><span><strong>此次申请“主要上市”，和当年的“二次上市”有着本质区别。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span lang="EN-US">2019</span>年，阿里赴港二次上市，但其并不需要满足港交所的上市标准。原因在于阿里已经在美国纽交所上市，其通过国际托管行和证券经纪商，实现股份在美股和港股的跨市场流通便称为“二次上市”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，阿里申请双重主要上市，则需要同时满足纽交所和港交所的监管规则，两地均为主要上市，拥有同等的上市地位。</span><span><strong>即便在其中一个市场退市，也不会影响另一市场的上市地位。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在中概股回归的潮流中，阿里无疑是极其耀眼的一颗明珠。但其与港交所的渊源，还不止于此。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在<span lang="EN-US">2007</span>年，阿里的<span lang="EN-US">B2B</span>业务便曾在港交所上市，直到<span lang="EN-US">7</span>年之后，阿里筹划整体上市时，首选市场依旧是港股，而非美股。只是碍于其“同股不同权”规则限制，阿里无奈，只能将上市地点放在了“备胎”美股上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/109138772362e1b8418dd6e1.90889366.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：阿里巴巴集团官网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而即便如此，阿里对港股依旧念念不忘。</span><span><strong>赴美上市的第一天，马云就表示，“只要条件允许，我们就会回来”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许正是对阿里呼喊的回应，在其赴美上市<span lang="EN-US">4</span>年之后，港交所终于开启了上市制度的改革，为诸如阿里等国内互联网公司在港上市提供更有利条件。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在二者的“双向奔赴”下，从<span lang="EN-US">2019</span>年赴港二次上市，到今天的双重主要上市，可以说，</span><span><strong>阿里真的回家了！</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>形势所迫，大势所趋</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在以阿里为代表的中概股回归潮流背后，是一场美国“剑指中概股”的阳谋。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span lang="EN-US">2020</span>年底，美国《外国公司问责法》正式表决通过，<span lang="EN-US">2021</span>年底，该法案完成实施细则。这意味着，美方对中概股的监管政策正式进入实质性执行阶段。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管从明面上看，这是一部针对所有外国公司的监管法案，但实际上，它却是一柄悬在所有中概股公司头顶的利剑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>依据该法案，如果在美上市的外国公司连续<span lang="EN-US">3</span>年未能提交美国上市公司会计监督委员会所要求的报告，<span lang="EN-US">SEC</span>有权将其从交易所摘牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/168891471762e1b84218cd48.29221865.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：财联社<span lang="EN-US">APP</span>官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>今年以来，已有数批次上百家中概股公司被<span lang="EN-US">SEC</span>列入《外国公司问责法》的预摘牌名单。</strong></span><span>相较于在美共<span lang="EN-US">300</span>多家的中概股公司总量，已有约一半都被列入了预摘牌名单。可以说，</span><span><strong>中概股公司正遭美方血洗。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>若这些公司未能向<span lang="EN-US">SEC</span>提供审计底稿等相关证据，其将被列入“确定摘牌名单”。而作为呈现真实财务数据的最重要文件之一，审计底稿的暴露，往往也意味着公司核心机密的暴露，此事难两全。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也正是在美方监管的不断收缩下，国家也不得不对选择赴美上市的滴滴、<span lang="EN-US">BOSS</span>直聘、运满满等企业进行监管调查。而这些企业都有着相同的特征，即掌握大量用户隐私、业务与国内关键信息基础设施有关。一旦这些信息向美方泄露，后果不堪设想。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>种种担忧下，回归港股成为中概股公司在形势所迫下的最好选择。与此同时，这也是大势所趋。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/61820429962e1b842bb1768.45076960.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：阿里巴巴集团官网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自阿里赴港上市以来，其在港交所的公众流通量显著增加。截止<span lang="EN-US">2022</span>年<span lang="EN-US">6</span>月<span lang="EN-US">30</span>日的六个月内，在港股市场，阿里股份的日均交易量已经达到了约<span lang="EN-US">7</span>亿美元。对于阿里的进一步回归，港股市场已经做好了准备。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，在中概股遭打压的大背景下，回归港股已成为一条主流道路。</span><span><strong>对于阿里而言，既能吸引更多国际资本向港股转移，也能更好应对单一市场的不确定性。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>阿里，全军出击</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在申请双重主要上市的同时，阿里也在昨天发布了<span lang="EN-US">2022</span>财年年报。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>报告显示，该财年内，阿里全球活跃消费者数量达到了<span lang="EN-US">13.1</span>亿，其中，有超<span lang="EN-US">10</span>亿均来自中国市场。在消费者业务领域，</span><span><strong>阿里也提前两年完成了国内年度购买用户超<span lang="EN-US">10</span>亿的目标，成为其在国内市场发展的又一里程碑。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/166705912762e1b843414e82.44881333.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：阿里巴巴集团<span lang="EN-US">2022</span>财政年度报告</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，</span><span><strong>在其另一大战略点上，阿里云也实现了创立<span lang="EN-US">13</span>年来的首次全年盈利。</strong></span><span><span lang="EN-US">2022</span>财年，阿里云共实现营收<span lang="EN-US">1001.8</span>亿元，冲破千亿大关；净利润<span lang="EN-US">11.46</span>亿元，同比扭亏为盈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>营收破千亿，全年盈利，苦等多年，阿里云终利剑出鞘。这也意味着，云计算将逐渐成为阿里的另一条增长曲线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而无论是消费业务，还是云计算，则均帮助阿里在全球化的道路上驰骋。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年年初，阿里进一步确定了集团三大战略：消费、云计算与全球化。在国内电商消费市场，阿里稳居龙头，云计算业务也终于开始全年盈利，并为电商业务提供底层支撑，那么阿里的全球化道路怎样了？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在蒋凡成为阿里海外大掌柜以来，阿里旗下跨境电商平台“速卖通”开始加快步伐。随着亚马逊平台上越来越多中国卖家遭到封号、冻结资金的情况，速卖通通过平台扶持，迅速吸引了更多优质商家入驻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，智利等海外国家，速卖通的电商市占率也跃升至头名。在亚马逊一家独大的国际市场上，这个成绩不可谓不精彩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/124837332462e1b843cca865.02008277.png" referrerpolicy="no-referrer"></p>
