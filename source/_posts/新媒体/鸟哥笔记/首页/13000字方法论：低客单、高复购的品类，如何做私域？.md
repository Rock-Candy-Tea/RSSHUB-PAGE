margin-top: 0px;"><span style="font-size: 16px;">首先我们来看一个私域GMV公式。</span></p></section><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">私域的GMV=私域用户量*转化率*客单价*复购频次*（1+转介绍率）+第二增长曲线GMV</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">除去第二增长曲线的异业合作、资源互换GMV。对于该品类来说，就是提高客单价、复购频次、带来转介绍，以及增加转化率。低客单的性质决定了产品利润率不会很高，所以运营动作要尽可能以低成本、批量转化为前提，深度1对1私聊从人工成本角度考虑显然是行不通的。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">因此，对应的运营动作以社群分层批量转化为前提，以会员卡锁客、复购周期内SOP产品组合推荐、转介绍分销为核心。</span></p><h1 id="toc-1" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-size: 16px 20px; font-weight: 600; text-align: center;">一：社群分层，针对不同生命周期的用户精细化运营</h1><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">想要高转化率，理想状态是1对1私聊提供顾问式服务引导转化，但这么做人工、时间成本会非常高。特别是低客单的产品，利润本来就少，这么做肯定会亏本。所以，必须通过社群，以尽可能少的人力一对多批量转化、服务。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">据我所知，某家私域在业内公认做得非常好的品牌，只有十几人的团队，却服务了百万级用户。如果这些用户都要1对1私聊，其成本无法想象。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">社群的类型分为四类，贯穿整个用户生命周期：以促进购买，培育发展成为高净值用户为目的的宠粉群；以维系高净值用户情感，发展其成为核心忠实KOC为目的的VIP群；以促销、优惠、福利为目的，围绕具体事件活动而设立的快闪群；以增进品牌与用户之间粘性、发展关系为目的的主题群。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">对于品牌来说，并不是每一类社群都需要，要根据品牌定位、品类特征、运营水平来搭配组合。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">此外，需事先明确，<span style="font-weight: bold;">私域中的促销福利不宜太频繁或与公域平台相差太大，否则会让消费者认为你产品本来就是这么低价，平时都在忽悠抬高割韭菜，不利于品牌力的打造。</span></p><h2 style="margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-position: 0px 5px; font-size: 18px; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-size: 8px 20px; padding-left: 12px; font-weight: 600; text-align: center;">一：宠粉群</h2><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">宠粉群重产品推荐，通过场景化的产品推荐、限时折扣、种草、用户反馈、主题活动来引导用户下单，培育其发展成为品牌的高净值用户。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">其针对的用户群体，有两类：</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">1：刚进入私域还未产生购买行为的用户，引导其在私域中完成首购。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">2：购买频次少或金额少，还未达到品牌定义的高净值标准的用户，通过社群运营，引导其产生多次购买。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">这类群的价值，以转化为衡量指标，并不是说群越活跃，就越成功。</span>品牌建该类群是为了转化，用户进群也是为了享受相关福利，强行做促活动作，拉活跃没有任何意义。举个极端的例子，群内没一个人说话，但是只要有人下单，这个群还是成功的。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">通常，宠粉群适用的品牌具备以下特点之一：</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">1：产品品类相对标准化。个性定制化产品，前期沟通成本大，更适合1对1私聊。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">2：产品价格低，相对应的用户决策试错成本也低，看到福利，转化下单概率就大。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">3：产品矩阵SKU多，让用户有得选。品类少，推送来推送去这么几样，意义不大。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">宠粉群内的用户，要对品牌有认知，接触过品牌，不管是从线上公域消费，还是线下门店面对面接触，至少是知道品牌是卖什么，做什么的。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">否则，随便拉一些用户进群，群人数看起来虽多，以为他们看到优惠福利就会下单，但站在用户视角来看，他们对品牌不了解，进群后看到每天在发XXX品牌的广告，就会认为这个群是垃圾广告群，屏蔽、不下单不说，甚至还会做一些不利于成交的动作。比如站出来询问这个群是干嘛的，或冷嘲热讽。这时，需要运营人员1对1沟通解决，增加运营成本；其他群用户看到后，很大程度上也会受影响。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">基于此，宠粉群的用户入群途径分为3种：线上公域下单后，引入私域的用户；线下门店接触后，引入私域的用户；以及朋友推荐入群（有社会关系做背书，解决了品牌信任认知问题）。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">下面以线上公域为例，具体讲下宠粉群的运营方法论。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">用户从公域平台进入私域，品牌方要尽快让用户在私域中完成首购，体验A HA Moment，感受到私域的价值（价更低、服务更好&hellip;&hellip;），养成在私域中下单的习惯。</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">对此，用户在添加企微后，品牌方可1对1私聊推送新人专属的大额优惠券、首次下单送XXX产品名额、首次下单享XX折、会员新人体验资格，再以限时限量为逼单手段，让用户尽快决策下单。同时，推送宠粉群链接，并明确告知新用户进群后能获得的价值利益点，引导进群。</p><h3 style="margin: 25px 0px 20px; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-size: 16px; padding-left: 12px; font-weight: 600; line-height: 30px; text-align: center;"><span style="font-weight: bold;">1）产品推荐</span></h3><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">社群内产品矩阵组合的推荐，往往以宠粉名义附带优惠福利，常见的有：满减、满赠、买赠、买N送N、打N折、专属价、一口价、加价购等等。若是搭配赠品，可以选择滞销品或清仓品，在给予用户优惠的同时，减轻库存压力。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">品类特征不同，用户的决策链路就不同，相应的产品推荐方法也不同，可以分为两类：</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">1.1低认知产品：重种草</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">有些品类用户的认知度不强，需要通过不断种草来让用户感受到价值，从而产生购买行为。比如，美妆、母婴、宠物、服装。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">以完美日记为例，美妆具有非标属性，在社群内推荐产品并不是简单粗暴甩链接，而是采用种草秒杀的模式，通过1-2天的内容种草、水军烘托培育用户心智，再通过限时秒杀促进下单。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">正式推品前，在群内做预热，告诉用户本次推荐的产品类型、价值、价格、限量份数、下单抢购时间。文案生活口语化，像个朋友在跟你交流；卖点清晰明了，便于阅读理解。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><img class="rich_pages wxw-img aligncenter" style="border: 0px; background-color: #ffffff;">当用户在私域中，消费达到一定的金额/频次，成为高净值用户后，品牌方可邀请其进入VIP群。由专人服务，在群内聊天、互动，有问题第一时间解决，同时通过定期的游戏互动、社交话题、知识新知、专属特权等来拉近与用户之间的距离，增进粘性。最终推动发展其成为核心忠实KOC，生产UGC内容，带来转介绍。</p><p style="margin-bottom: 18px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-size: 16px; padding-left: 12px; font-weight: 600; line-height: 30px; text-align: center;"><span style="font-weight: bold;">1）游戏互动</span></h3><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">围绕红包的：比大小、猜排名、抢金额第N名、抢速度第N名&hellip;&hellip;</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">促进到店的活动：</span>比如虎年到了，门店门口摆了一个虎年元素的巨型玩偶，在群内发起活动，让用户上传合照到群内，可获得玩偶一个，促进线下到店。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">玩法要尽可能简单，通俗易懂。做活动的目的，是为了转化用户，而不是难倒用户。</p><p style="margin-bottom: 18px; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">比如，<span style="font-weight: bold;">针对用户第一次购买产品的「1、3、7、n」SOP法则。</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">在用户下单的第1天，询问用户的收货地址是否正确，提醒用户查收。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">第3天，一般用户的快递已经拿到。这时可询问用户快递、产品是否有破损、纰漏，如果有问题则第一时间处理，并告知用户产品的使用方法以及需要注意的事项，体现品牌的温度。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">第7天，用户使用产品已经一段时间。可询问在使用的过程中是否遇到了困难，有没有不明白的地方。若用户没有问题，则可问用户的使用反馈，觉得使用不错的，可邀请分享使用体验生产UGC，并给予对应的分享激励。这些素材，品牌方可用在社群、朋友圈做真实体验种草。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">当然，若产品效果见效慢，则可将周期拉长，比如第7天的时候询问使用中是否遇到困难，第14天了解使用反馈。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">第n天，即产品的消耗复购周期前1周，这时用户的产品已经使用差不多了，可询问用户产品的使用效果如何，借势以复购优惠劵、促销活动、积分消耗提醒，促进用户复购，备货。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">如果用户已经购买某个产品2次以上了，说明其对该产品的使用方法、注意事项已经有了充分的了解，为避免过度打扰用户，只要在复购周期前提醒即可。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">如果用户购买过品牌下A产品多次，但第一次购买B产品，那么重新进入「1、3、7、n」法则，可去掉第1天的快递确认。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">以上是标准化、可复制的步骤，可通过SOP工具，基于预设条件，实现自动化的触达，减少人工。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">基于复购周期的推荐可提高复购频次，那么提高客单价，可从产品组合的搭配进行入手，分别为卖组合、卖高价、卖拓展。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">卖组合：</span>将用户有需求、已多次购买的产品打包组合，通过优惠刺激推荐一次性多购买囤货。比如，单件购买打9折，一次性购买三件打8折，购买五件打7折。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">这种做法，适用大促、冲业绩期间，在短时间内快速提升GMV，但也因为用户的需求有限，以及产品消耗周期是恒定的，从长期看，对整体GMV提升价值不大。缺点也很明显，容易打乱价格体系，伤害品牌，甚至是被用户认为割韭菜。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">卖高价：</span>顾名思义，就是在推荐常规品之外，向有需求的用户推荐高价品。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">卖拓展：</span>除了原有的产品体系，根据用户需求、市场情况，拓展品类。这样就不用局限于单一品类的复购周期，对原有价格体系影响也小。比如，品牌之前是卖咖啡液的，拓展品后可以卖挂耳、咖啡粉，甚至再拓展后，可以卖咖啡豆、咖啡糖、咖啡杯。</p><h1 id="toc-4" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-size: 16px 20px; font-weight: 600; text-align: center;">四：小B招募转介绍分销</h1><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">品牌调性、势能、对渠道掌控程度不同，其转介绍分销的打法也不同。</p><h2 style="margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); padding-left: 12px; background-position: 0px 5px; font-size: 18px; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-size: 8px 20px; font-weight: 600; text-align: center;">一：转介绍</h2><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">跟高客单、低复购的品类一样，通过对成交用户的重点运营，发展成为KOC、超级用户，让他们自发替品牌发声，带来转介绍。适用于势能强、知名度高、价格透明度高、渠道价格体系管控严格的品牌。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">要做好转介绍，口碑是结果，激励是加速器。</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">营销的核心是产品，口碑的核心也是产品。口碑是用户在使用产品、体验服务的过程中，用得好、用得爽，自然形成的。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">绝大多数转介绍场景，都是用户推荐身边的朋友购买或在论坛、社群内做推荐，这时他就相当于给品牌做背书，做担保。特别是现实关系网络中，推荐的产品如果不好，还会被朋友在背后指指点点，冲淡关系。因此，产品本身不好，口碑不佳，用户是不敢做推荐的。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">除了产品本身，基于产品延伸的超预期服务，也能够产生好口碑。比如买蔚来汽车后，相当于获得了一张入场劵，能每周免费参加品牌在当地门店举办的线上、线下活动；包机票、包住宿参加年度活动；在生日、节假日、纪念日享受到品牌惊喜关怀&hellip;&hellip;</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">在好口碑的基础上，激励能够加速用户做转介绍。</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">激励分为物质激励、荣誉激励，一般叠加组合使用。</p><h3 style="margin: 25px 0px 20px; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-size: 16px; padding-left: 12px; font-weight: 600; line-height: 30px; text-align: center;"><span style="font-weight: bold;">1）物质激励</span></h3><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">1.1返钱</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">推荐一个用户成功下单，可获得XXX金额。（若一个下单用户的获客成本在2000元，那么用户返点少于2000元，对品牌来说都是赚的。）</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">1.2返物品</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">推荐一个用户成功下单奖励XXX物品，推荐三个奖励XXX物品，推荐十个奖励XXX物品。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">以上两项激励，我个人不太推荐，用户的推荐行为完全以物质来衡量了。会让推荐者产生心理负担，让被推荐者觉得他朋友就是想挣钱才推荐的。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">将激励用积分代替，在运营动作、推荐动作上就显得柔和、灵活的多。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">1.3返积分</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">品牌建积分商城，用户可拿积分自主兑换商城里的实物、虚拟物、权益。每推荐一个用户下单，可获得XXX积分。（除了推荐可获得积分，还可与社群行为数据绑定，积分价值根据获客成本、商品利润综合来制定。）</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">需要注意的是，激励具有成本，因此务必要在成功下单之后，推荐者才能获得奖励。为了规避用户退款后，推荐者还能获得奖励所带来的损失，必须进行事先说明。比如，用户成功下单30个工作日后<span style="font-weight: bold;">（过了退款期）</span>，才可获得对应奖励。</p><h3 style="margin: 25px 0px 20px; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-size: 16px; padding-left: 12px; font-weight: 600; line-height: 30px; text-align: center;"><span style="font-weight: bold;">2）荣誉激励</span></h3><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">2.1身份象征</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">推荐满X人可成为品牌的推荐达人，获得品牌CEO、高管亲笔感谢信或证书，同时邀请进入到专属的推荐达人社群，享受特权。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">2.2专属特权</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">只有推荐达人才能享受到的特权，因为能直接为品牌带来用户，其特权一般是最大的。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">比如，远低于一般用户的专属折扣价；终身售后；节日定制礼；线下沙龙活动；工厂之旅&hellip;&hellip;</p><h2 style="margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); padding-left: 12px; background-position: 0px 5px; font-size: 18px; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-size: 8px 20px; font-weight: 600; text-align: center;">二：代理分销</h2><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">将用户发展成品牌的代理，品牌方给予相应的激励政策与赋能扶持，让用户替品牌卖货。其本质是<span style="font-weight: bold;">卖一个赚钱概念，让用户以为自己花最小的精力，能挣最多的钱</span>。适用于产品利润空间大、价格感知不明显的品牌。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">（以下是对代理分销体系的简单概述，实际运营很复杂，非常考验运营、人性洞察能力，且需要规避风险，一般不推荐做。）</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">整个代理体系的核心在于：</p><h3 style="margin: 25px 0px 20px; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-size: 16px; padding-left: 12px; font-weight: 600; line-height: 30px; text-align: center;"><span style="font-weight: bold;">1）有钱挣</span></h3><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">做代理的目的是什么？所谓无利不起早，最直接的无非就是有钱挣。那么，品牌方要设置好对应的分钱、层级体系。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">1.1直售</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">每直接卖出1个产品，可获得的差价。其目的是让用户卖货。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">1.2推荐奖</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">每发展1个下级，品牌方给予推荐奖励。其目的是让用户发展下级，壮大分销群体。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">1.3极差奖</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">极差分为两类。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">一类是产品差价。比如，层级A从品牌拿货为原价的10%，层级B为原价的15%。B每进一次货，层级A可获得5%的差价。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">还有一类是奖励极差。1年内销售额达到特定的数值，可额外获得分红。（为方便计算理解，以下做简化取整。）</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">比如，1年内团队做到2500万，可获得10%分红。做到1500万，可获得8%分红。做到1000万，可获得5%。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">假设C及他下级团队，总共刚好做到2500万，可获得分红250万。假设C不卖货，就发展了2个下级，且他们刚好做到了1500万、1000万，对应获得分红120万、50万。那么，C仅凭这两个下级可获得的极差奖励为250万-120万-50万=80万。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">此外，如果有晋升体系，还要设置平级奖。A的下级B，晋升为与A同一层级后，品牌方拿出平级奖来补贴A。</p><h3 style="margin: 25px 0px 20px; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-size: 16px; padding-left: 12px; font-weight: 600; line-height: 30px; text-align: center;"><span style="font-weight: bold;">2）简单挣</span></h3><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">有了分钱体系，用户知道能挣钱，但是能不能真的行动起来，还要看这钱好不好挣，所以品牌方要成立商学院，尽可能多给予赋能，让用户简单挣钱，或看起来简单。</p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;"><span style="font-weight: bold;">2.1培训赋能</span></p><p style="margin-bottom: 18px; margin-top: 0px; color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">面向新人的：产品项目介绍、市场红利、制度讲解、成功案例分享、加入政策&hellip;&hellip;</p><p style="margin-bottom: 18px;
