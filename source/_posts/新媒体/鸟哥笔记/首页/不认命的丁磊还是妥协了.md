<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="6" data-pm-slice="1 1 []">“七分理想，三分生意”是<a href="https://www.niaogebiji.com/tag/%E4%B8%81%E7%A3%8A" target="_blank">丁磊</a>对网易的概括。显然，对丁磊以及网易而言，游戏是生意，而音乐是理想。当理想遇上资本，它还有几分纯粹已然要打一个问号。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="8">2021年底，资本寒冬中，<a href="https://www.niaogebiji.com/tag/%E7%BD%91%E6%98%93%E4%BA%91%E9%9F%B3%E4%B9%90" target="_blank">网易云音乐</a>乘着版权开放的东风登陆港交所。近日，云音乐发布了上市后首个半年报，还举行了首次业绩电话会议。然而，值得注意的是，会议最引人关注的并非一众用户最关心的版权，而是一个所谓的音乐<a href="https://www.niaogebiji.com/tag/%E7%A4%BE%E4%BA%A4" target="_blank">社交</a>APP：MUS。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="10">事实上，这并非网易云音乐首次涉足社交领域，在MUS以前，网易云音乐就曾推出心遇，主打陌生人社交，不过成绩并不理想。这次将音乐嵌入其中以后，MUS成效几何还有待观察。作为生意，这无可厚非。可如果还言之凿凿的打着“理想”的口号去做一个“音乐社交”产品，这多少有些牵强。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="12">如何做好理想和生意间的抉择，这不仅是丁磊的难题，也是困扰古今往来的无数人的疑问。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="14">100多年前，威廉·萨默塞特·毛姆发表了酝酿已久的个人自传体小说《人性的枷锁》。《泰晤士报文学增刊》刊登了一篇书评，称菲利普·卡雷（主角）“为天上的月亮神魂颠倒，对脚下的六便士视而不见”。在后来的《月亮与六便士》中，毛姆谨慎说道，“我想这取决于你如何看待生活的意义，你认为你应该对社会做出什么贡献。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="16">纵观中国的互联网江湖涌现的一众天之骄子：门户时代的丁磊、张朝阳、王志东，BAT时代的李彦宏、马云、马化腾，TMD时代的张一鸣、王兴、程维，甚至于再加上黄峥、陈睿，丁磊恐怕是最多谈及理想的人。在与众多和丁磊有过接触的人士接触时，我们也发现，丁磊的这种理想并不完全是一种“虚伪”人设。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="18">理想和生意，月亮和六便士，丁磊似乎一直有着“兼得”的奢求，但这就像一枚硬币的两面，没有人能给出完美的选择，对普通人如此，对丁磊亦然。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="18"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://qnssl.niaogebiji.com/0b52b5d41f25672816779518fa40b18c.png" alt width="444" height="294" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" data-track="21">01 顺带赚钱</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="25">2017年。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="27">丁磊已经在互联网江湖摸爬滚打二十余年，虽时有露面，却还未曾接受过正式的视频专访。没想到，第一次接受视频专访，丁磊就“语出惊人”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="29">面对一条的镜头，他说出那句后来广为流传的语录，“挣钱只是一个顺便的事情。”在被追问“金钱给你带来的幸福能占多大比例”时，他说， “可能对我来说5%都不到。”不过，跟很多名言被忽略了后半句一样。丁磊在其后还有一句补充却鲜少有人关注。他说，“金钱对创业或者探索来说非常重要，但对个人生活来说，我觉得不重要。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="31">2017年丁磊接受一众媒体专访时，网易云音乐正处于发展巅峰，网易严选也开始走上正规，这是一个风光中的丁磊有些“洋洋得意”且有些小“虚伪”的回答。当然，这里“虚伪”并非贬义。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="33">诞生于2013年的网易云音乐，并非丁磊一时兴起，而是一个音乐爱好者数十载的苦心孤诣。2000年，网易初登资本市场，记者问他 “最想做什么事情” 。他说出的是， “我想做一家唱片公司” 的答案。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://mp.toutiao.com/mp/agw/article_material/open_image/get?code=MzI3NmRiMGRkMTU5MWQzOTQzYmMyOTA1YzYzNjU4ZmQsMTY2MTIyNDQxMDIyMA==" width="444" data-ic="false" data-width="910" data-height="539" data-ic-uri referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;" data-track="35">▲图：年轻时的丁磊</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="37">二十年过去了，他所做的网易云音乐已经发展成拥有过亿用户的APP，是网易最有价值的资产之一。但它却也因为版权、亏损饱受诟病。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="39">比起音乐，严选更像是一个意外产品。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="41">坊间传闻最广的段子是，某天，丁磊想在网上买毛巾。跟音乐一样，丁磊对家具小玩意有一套自己的想法。结果找来找去，他竟然找不到一条自己满意的。这就是丁磊做严选的初衷。但无论是严选还是音乐，丁磊都躬身去做这些他想做的事，他曾亲赴台湾谈唱片，也躬身为严选选过玻璃杯。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="43">2014年，财经作家吴晓波曾经在节目里提到过丁磊，说他是中国首富榜单里，活得最快乐的一个人。四年后，吴晓波跟丁磊做了一场对谈，他问，“Pony（马化腾）也是很好的产品经理，你们有什么区别。”丁磊说，“他（马化腾）可能更擅长做社交产品，我更擅长做实体产品，像严选。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://mp.toutiao.com/mp/agw/article_material/open_image/get?code=MDE1N2Q3MDU5YjA0Yjg5OWIxODA3ZDJkOGViODRmMmMsMTY2MTIyNDQxMDIyMQ==" width="444" data-ic="false" data-width="1280" data-height="660" data-ic-uri referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;" data-track="45">▲图：丁磊与马化腾在饭局上</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="47">不过，有趣的是，四年时间过去了，网易云音乐做的越来越像社交产品，而严选却在2019年折戟沉沙，卖身阿里。在2018年的对谈中，吴晓波问，你兴趣更广泛些是吧。丁磊答：那肯定是。而在被问及，“你更专心一点，是不是会好一点”后，丁磊说，“可能也会很好，但是你有这种兴趣，你不能说我要把兴趣按住，不做这个，天性不要抹杀。”这大概就是丁磊比其它人更快乐的原因，他想赚钱，但也要做自己喜欢的事情。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="49">在2018年左右，互联网涌现了大批创业者和投资风口，网易却岿然不动。“网易的核心能力就是在熟悉的领域做好自己的事，不熟的事情，一律不碰。”丁磊说。然而，他一边说，网易的文化是专注；一边又带着网易在音乐、游戏、电商、教育四处出击，做自己想做的事。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" data-track="52">02 理想隔壁是现实</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="56">过去的2021年，对丁磊而言，有两个大事件。一个是关于行业，一个是关于网易，但两者都和丁磊的音乐理想有着莫大关系。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="58">2021年7月24日，国家市场监督管理总局官网发布责令腾讯解除网络音乐独家版权的决定，标志着国内音乐版权市场正式进入“非独家”时代。8月31日，腾讯发布《关于放弃音乐版权独家授权权利的声明》，表示已最大限度寻求与相关上游版权方尽快解除独家协议。在随后的网易财报会议上，丁磊说，“非常期待这是一个真心实意的，不含任何阳奉阴违的决定。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="60">在云村动态里，丁磊不止一次回应过版权问题。他说，“在搞了，在搞了，版权的事情我现在亲自抓。”他还说，“只要独家版权放开，我们就敞开买。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="62">2021年，作为网易CEO，丁磊亲自兼管网易云音乐CEO实际工作，一手推动了云音乐的上市。尽管过程一波三折，但云音乐终于赶在年底登陆港交所。资本寒冬环境下，云音乐没能破发的命运。上市后，股价一路下滑。12月2日上市当天，丁磊发表了一封公开信，题为《相信音乐的力量》。在信中，他深情回忆了自己如何变成一个音乐爱好者，又如何深度参与到音乐的发展。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://mp.toutiao.com/mp/agw/article_material/open_image/get?code=ZmUyZWNhNWJmOTA2M2NiODA5N2ZlOGMyMGM5NzE4MGMsMTY2MTIyNDQxMDIyMQ==" width="444" data-ic="false" data-width="710" data-height="400" data-ic-uri referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;" data-track="64">▲图：一众互联网大佬逐渐隐退</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="66">丁磊对于音乐的热爱是真的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="68">你很少见到一个大企业CEO在年过五十后，还深度参与子公司的管理。你更加鲜少看到，一个管理数万人公司的老板，还在为旗下的子产品里不定时分享动态。云音乐上市当天，他在云村发表动态称，“我的第一条云村动态，是2013年4月23日发出的。3000多天来，从几个人的热爱到几亿人一起热爱。云村也在不断壮大。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="70">但丁磊又不得不为“生意”低头也是真的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="72">在很多人眼里，产品能够拿出去融资是一件莫大的幸事，但丁磊并不这样想，尤其是在他热爱的音乐面前。但这不仅仅因为理想，还有生意。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="74">有一个故事，一个云音乐事业部的高管在经过同意后，从外面找来大笔融资，只等丁磊签字。但丁磊看了又看，把笔扔在一旁说，“钱为什么让他们来赚。”可眼看着互联网寒冬的到来，生意人丁磊还是选择了妥协。承载着丁磊理想的云音乐不仅融资，还上市了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="76">只是，赚钱还是亏钱，大概只有丁磊自己心知肚明。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" data-track="79">03 什么是生意</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="83">时光倒回至1997年，丁磊用自己攒下的50万创办了网易，每天工作到凌晨一两点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="85">在经历过BBS、邮箱、门户三个时代后，丁磊终于跻身成为中国互联网江湖的奠基人，同张朝阳、王志东并成为当时的网络三剑客。彼时的马化腾还在为公司50万启动资金四处奔波，李彦宏尚宅在房间钻研第一代搜索引擎技术。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="87">可好景不长，转眼2000年，刚刚迈入正规的网易就遇上了互联网泡沫。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="89">那一年，网易净亏损1.29亿元人民币，因涉嫌财务欺诈被纳斯达克终止交易。管理团队纷纷出走，公司内部人心惶惶。丁磊起了卖掉网易的想法，但苦于财务审计，无人肯收。段永平劝他，“你还有钱，有方法，为什么不回去再努力试试？”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="91">陷于绝境的丁磊选择把宝押注游戏。有幸，他押对了。迄今为止，游戏仍然是网易最重要的收入来源。网易研发的《大话西游》一经推出就成爆款，段永平痛快地投了100万美元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="93">一个有趣的事情是，拼多多创始人黄峥是丁磊引荐给段永平。拼多多上市当天，段永平留在上海远程敲钟。但对丁磊这个黄峥的引荐人，段永平却是另一番态度。2018年9月，段永平做客斯坦福大学，坦然提及抛售网易股票一事，“丁磊就是个大孩子，那么多钱放他手里不放心。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://mp.toutiao.com/mp/agw/article_material/open_image/get?code=Y2MxNzZiZTQzNjk4MmQ4ODhkNWFjMGVmZDEyOGUwNDQsMTY2MTIyNDQxMDIyMg==" width="333" data-ic="false" data-width="545" data-height="760" data-ic-uri referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;" data-track="95">▲图：丁磊一度被称为“网易玩主”</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="97">时过境迁，为理想买单正在变得越来越难，而这一切最后终归要落脚到生意。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="99">2019年，创办公司22年后，网易开始动作频出。当年9月，考拉作价20亿美元被阿里巴巴收购。同时，阿里为云音乐注资7亿美元。一个月后，网易有道远赴纽交所挂牌上市。两年后的2021年，双减政策和音乐版权放开几乎同时来袭，有道被迫走上转型道路，而云音乐则完成上市壮举，市值从400亿港元一路跌到150亿港元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="101">在公开信中，丁磊说，我们想经营的不止是音乐的生意，更是声音的生意。但他也说，我们想实现的，不止是让梦想赚到钱，更是把梦想推到一个更大的舞台。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="103">而现在，云音乐在社交上发力愈加明显，理想，或许终究还是做成了生意？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="105">知乎上对毛姆三部曲有这样一个评价：《人生的枷锁》是背弃者，菲利普放弃了虚幻的理想，抓住了切切实实的幸福；《月亮与六便士》是殉道者，斯克里特兰德把一切都奉献给了理想主义。朝闻道，夕死可矣；《刀锋》写出了毛姆对理想主义最后的答案。身为普通人，既不愿意放弃理想，又没有为之献祭一切的勇气，丁磊或许就是最后一种人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="107">丁磊引荐给段永平的后辈黄峥已经退出前线，年近花甲的张朝阳肆意在直播间讲着自己热爱的物理课，五十岁的丁磊却还在为理想和生意同时奔波。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="109">没有人能给出谁对谁错的绝对答案，但丁磊依然还在恣意前进。只是在月亮和六便士间，人终究还是要学会妥协，先有六便士，后谈月亮。</p>                        <div style="height: 1px"></div>
