<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">盛夏7月，暑期进度条已经走过一半，不仅是暑期长假更是热血沸腾的“毕业季”，这段时间往往是Z世代互联网狂欢的密集期，也是品牌们进入年轻圈层的高峰时段。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">在这个7月，是什么高能恰饭在吸引着B站的年轻人，焕发品牌年轻态？</p>
<h2 style="border: 0px; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; font-size: 18px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #212629; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; line-height: 30px; text-align: left;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">仅70w粉，却讨好千万B站用户</span></h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">谁不想要一天赚十万呢？</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">7月4日，UP主「阿星and阿奇」发布与酒水品牌泸州老窖合作视频，在视频中UP主扮演一名刚被录取的职场新人。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">不一样的是，这支作品的情景剑走偏锋，和现实中的职场简直天上人间。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">视频中的老板给新人开价“一天十万”，上班必须迟到，如果早到还会被“惩罚”加工资，加到新人不敢早到。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">吃饭不能吃廉价外卖，在老板眼里那叫“垃圾零食”，只能吃老板专供的“豪华员工餐”。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">这样的作品一经抛出，灵魂发问“谁不想要一天赚十万呢？”，瞬间打动了千万打工人们的DNA，在UP主<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">仅有70w粉丝数</span>的情况下，作品拥有1449.46%的播粉比，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">播放量高达1204.19w</span>，点赞数更是超过33w，1分21秒的视频带给B站用户一场短暂的“社畜狂欢”。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p4.itc.cn/images01/20220803/f2eb0868ea934f1ba33c646ef1c8ecae.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">时至今日，作品在B站仍然有<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">一小时近10w</span>的播放量，不断地增长品牌声量。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p9.itc.cn/images01/20220803/a40ca93c7ed446049692ffc1533c70db.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">曾经提到B站大众的第一反应就是“二次元平台”，ACG内容在B站独占鳌头，但经过近几年的多元文化拓展，B站已经幻化为<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">品牌走进年轻化的首选阵地</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">泸州老窖作为一个拥有悠久历史的酒水品牌，持续在高端路线行走，此次投放无疑是一次<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">品牌年轻化的突破</span>，UP主「阿星and阿奇」是腰部搞笑区代表达人，曾有过几次千万播放的出圈作品，虽然粉丝数远低于头部达人，但优质的内容构思以及往期多次爆款作品的经历让UP主平添了<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">商单吸引力</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">再加上飞瓜数据（B站版）收录的关于UP主视频报价的情况来看，其<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">植入视频报价仅定位在3w</span>（由UP主亲自填写的真实报价，该报价仅供参考，具体合作请与UP主咨询后为准。），是老品牌走进年轻态高性价比的一次投放选择。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p1.itc.cn/q_70/images01/20220803/a39891e65d1c45969b19aac41f223578.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<h2 style="border: 0px; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; font-size: 18px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #212629; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; line-height: 30px; text-align: left;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">二次元游戏与</span><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">三次元鬼畜的高能碰撞</span></h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">“恭喜你发现了宝藏！”</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">这是B站用户们对一个优秀作品最高的赞誉，这次竟颁给UP主「_青红造了个白_」和游戏三国杀的合作视频。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p2.itc.cn/images01/20220803/6b2cdd7ed32e49a98c3651f4d975f29d.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-B站</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">五分半钟的视频，没有留取一段刻意去对游戏进行解说，这也有一部分原因是《三国杀》游戏具有一定的知名度，所以UP主仅仅只是在作品中将游戏人物卡作为一个素材穿插在内。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">因为游戏是三国相关，所以主题选择了学生时代都看过的1994年经典电视剧《三国演义》，而在此之前《三国演义》已经是B站整活区的“老演员”了。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">所以此次UP主依然没有绕开整活，将电视剧中的桥段打碎重组，再搭配上“拙劣”的特效贴图，完全符合当下B站整活的“5毛特效”氛围。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">因此作品吸引来了<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">1245.11w的高速传播</span>，不仅如此互动数据中更是超百万点赞以及百万投币，当之无愧为“宝藏”。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p6.itc.cn/images01/20220803/62a9732238ac4ff787f075b9fcfa2b69.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">由于此次商单是<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">0口播的推广植入</span>，对于不了解《三国杀》游戏的用户来说无法看出是商单作品，但作品在优质创作的作用下高速传播，许多游戏玩家和知道这款游戏的观众在评论区轮番玩起游戏梗，实现KOL传播后的<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">用户二次传播</span>，进一步助推拉新转化。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">许多观众在评论区更是直言“这是今年看过最搞笑的视频。”、“这种恰饭，多来点。”等催更言论。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p0.itc.cn/images01/20220803/818f9485bcc444888b523984d1a009bd.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-B站</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">前面提到ACG内容一直是B站的“头牌”，游戏就是属于ACG内容之一，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">游戏品牌在B站整活这无疑是牛头对马嘴</span>，况且<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">二创、跨区联动</span>一直是B站内容社区内的老朋友，两者结合只能是“双厨狂喜”的双赢局面。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">做年轻人对味的作品</span>，是品牌在B站商业营销首要的准则，以上善用二创、跨圈投放这两点也同样适用于其他行业品牌，拘泥于垂类投放可能会错过一部分潜在用户。</p>
<h2 style="border: 0px; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; font-size: 18px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #212629; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; line-height: 30px; text-align: left;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">猎奇整活：不做你以为的测评</span></h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">7月1日，UP主「小迪爱分享」发布与蓝盒子合作的商单作品，主要是通过测评进行推广，但一般的床垫测评是真人体验为主，但UP主却有剑走偏锋的想法。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">通过两次实验测评床垫的稳定性以及软硬度：</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">第一个实验：UP主在床垫上放水杯，通过不断地蹦跳床上的水杯纹丝不动，杯子里的水也没有撒漏。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">第二个实验：在床垫上放置一枚鸡蛋，在经过UP主几次平躺、翻滚后鸡蛋没有丝毫的破裂，可以看出床垫对于不同人体的贴合程度。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">在UP主整活测评下，评论区也有<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">很多观众提供了产品的使用反馈</span>，进一步刺激正在观望的潜在消费者。</p>

<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">即便是开篇就有弹幕省流提醒是广告视频，但在整活测评的吸引下，仍然有超过700w的用户点击观看，播放粉丝比远超5000%，在播放趋势中可以看到，作品流量在不断触达<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">一小时增长20w</span>的峰值。</p>

<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">始于内容，忠于内容</span>，这支作品在开篇就没有避讳是商单作品，但依然获得高传播的关键就在于主题足够吸引人。</p>
