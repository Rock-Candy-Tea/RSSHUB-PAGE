<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E6%B7%98%E5%AE%9D%E7%9B%B4%E6%92%AD" target="_blank">淘宝直播</a>需要“一哥”，这是不争的事实。去年至今，淘系的内容转型走过了一条曲折的路线，根本逻辑在于消费行为正在出现深刻变革。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传统电商时代，简朴的货架逻辑已经不足以支撑今天复杂多元的消费路径，大量种草类型的内容平台崛起，本质上反映了消费者对内容型消费的强烈需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而作为中国电商的起点，淘系平台坐拥巨大的流量优势与履约优势，唯独在内容发掘上显得方兴未艾，这将成为淘系平台重点发力的核心方向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以美妆赛道为例，淘系平台的美妆内容板块目前处于短暂的真空状态，即便在一哥还在的那些日子里，美妆赛道的生态深度也是不够的，只不过巨大的成交金额掩盖了这个事实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去6年，对于美妆直播赛道是值得回味的6年。而在美妆直播一哥空缺的日子里，很多人都在怀念。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本质上大家怀念的不是某个形象或者某个直播间，而是怀念那种充满很多种可能性的消费氛围。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/100942054962e298f663b3f2.35660576.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在，关于美妆直播的故事又有了新的定义。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>从氛围感进阶到内容感，从卖力喊麦进阶到扎实干货输出。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个新的时代开始了。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>1、</strong></span><span>一场意义深远的美妆直播</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在我们理解了淘系平台对于美妆赛道的内容诉求逻辑之后，再来看当前美妆赛道的变化，一些看似细枝末节的现象就显得意味深长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，一场直播引起了我的注意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这场直播并不是大家过往印象中动辄几亿几十亿的那种所谓的“大促直播”，但它的意义很深远。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/151988650962e298f69a5126.32738597.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我说三个亮点你就懂了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一，这场直播的主理人是美妆直播赛道里迎来的专业度极高的一位大咖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小P老师perry，曾任大S、小S等多位港台及大陆艺人的专属造型师。多次担任台湾金钟奖、金马奖、金鸡百花奖、釜山影展等的造型设计，并长期和世界顶级知名品牌合作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/22155661362e298f6d26113.48358173.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二，这场直播选择的阵地是淘宝直播而不是短视频平台，</span><span><span><strong>背后的逻辑就是在同等成本基础上，淘系平台的转化率更高。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在直播电商领域，淘系直播有着独特的优势。淘系直播的特点是基于商品交易，</span><span><span><strong>但同时淘系也正在内容生态上发力，但这些内容大多也是围绕着“交易”展开的，</strong></span></span><span>这意味着淘系直播有一个更为清晰的用户画像，能使“人货场”的流量匹配更精准。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于天然的平台特点，淘系直播不需要经历太长的流量转化的过程，最大化避免流量折损，能够更直接地“看到生意”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三，更专业的美妆直播与淘系阵地的结合，已成为专业化直播下一步的经典CP。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从目前淘系主播类型的生态位来看，“美妆”这个类型是被低估的，或者说是缺失的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，是淘系直播庞大的用户和流量出现了短暂的真空状态，官方和用户都在等待一颗新星横空出世，这是在“人”的角度，大家想要新的一哥，这个一哥身上的价值注定是不一样的。</span><span><span><strong>一哥一姐缺位只是表面原因，深层次原因其实是淘内阵地里没有一个美妆品类的专业大咖能承接住空缺出来的流量。</strong></span></span></p>
<section><img src="https://ngbjimg.xy599.com/92053175062e298f739bfa3.66238983.jpeg" referrerpolicy="no-referrer"></section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，从淘宝直播的视角来看，也需要通过某种新的价值，比如更专业的美妆内容，为平台吸引新的流量来源。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，顶着“大S、小S专属造型师、亚洲人气造型师、金马、金钟、金鸡奖等影展活动方资深造型设计师”等专业光环的小P老师，他此番在淘宝开直播让我有了很多遐想：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，美妆直播这个淘系原本最强的赛道迎来了足够强的人，这是对原有生态位的拉齐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，这个人对于美妆的理解与输出方式，是比之原来的淘宝直播美妆赛道有进阶的。具体进阶在哪里？</span><span><span><strong>核心其实很明确，就是包裹在“漂亮内容”里面的“专业干货”。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>漂亮，很多人能做到，专业，并不是谁都能做到。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/135898701662e298f77b8e66.38653713.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一言概之，这就是淘宝直播美妆赛道的一次全新的内容升级。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>2、</strong></span><span>干货最值钱，尤其是亚洲人气造型师的专业干货</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小P老师在美妆领域内光环众多，但吸引我的可能不是这些光环，而是他打开淘宝直播这条新赛道之后的可能性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中最核心的一点，</span><span><span><strong>就是小P老师26年以来被综艺、影视等专业潮流市场反复验证过的潮流造型原创能力。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用直播行业里的行话来讲，这就是最强的“内容力”护城河。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/34562184662e298f7bdcf17.01344398.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么直播要强调“内容力”？因为这原本就是直播电商的本质，但在很长一段时间内因为内容力的缺失，导致很多人对直播的理解出现了巨大的认知偏差，大家误以为直播就是喊麦、就是枯燥的话术和夸张的肢体语言，但事实上，扎实的内容给品牌和消费者带来的价值才是直播电商真正的价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>道理并不复杂，那为什么大家直到此时才重提内容力？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为人都想走捷径，门槛低、收益高的捷径对应的其实是技术含量低，哪怕长期来看这样的效果是很差的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外一个重要的原因是，那些拥有内容力输出能力的人，要经过一番大浪淘沙之后才真正有机会重塑直播电商。在喧嚣与浮躁的环境里，内容力很容易被吵闹淹没。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换而言之，小P老师这样的高度专业的人士，只有在此时此刻的淘宝直播及由此引发的直播全行业“内容力”新趋势中，才能最大限度地发挥其天然的内容力优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从小P老师开局的系列直播中，已经能嗅到“专业干货”的气息，足够的辨识度结合巨大的受众人群，也许美妆赛道里的董宇辉即将上线。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>3、</strong></span><span>直播赛道变轨，内容势不可挡</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说到董宇辉，就多说几句。之前其实也写过几篇文章，</span><span><span><strong>由董宇辉引发的一个新的现象正在成为势不可挡的新风口。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个趋势正是“内容直播”。如果说上一代的直播凭借的是价格、氛围感等等刻意营造的、门槛相对较低的竞争力去吸引人群，</span><span><span><strong>那么下一代的直播则依托门槛更高、更专业的细分内容去打动人心。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/207496314862e298f8872ed7.76041749.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从行业趋势来看，淘系直播仍然是引领趋势的主阵地。尽管在内容直播百花齐放的时代，淘系直播看上去有些被甩开了身位，但这只是表面现象。随着“一哥”成为某种内容的代名词而非某个个人主播，淘系直播的想象空间被彻底打开了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本质上，大家拼的并不是谁的名气大、谁的粉丝多，而是谁的内容力更强，谁的供应链优势更大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，接下来在美妆赛道内会有更多的“一哥”出现在淘宝直播，一方面淘宝直播需要一哥去撑起内容，另一方面真正的专业一哥需要一块阵地来输出新的直播价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说白了，淘宝直播依然是主播赚钱机会最大的阵地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但此赚钱非彼赚钱，所谓的超头主播接下来要赚的钱，是实实在在的与内容能力相匹配的钱，而不是那种被时代命运砸中的luckymoney。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝直播缺不缺超级主播？其实不缺。淘宝直播需要的是让生态变得足够深，关键赛道的内容力足够强。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝直播与其他平台相比起来，最核心的认知优势在于足够宽广的电商赛道布局，哪怕没有主播，该买还是会来淘宝买。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但这不足以支撑起未来。淘宝直播之所以要做内容，是希望在既有的高度再开花结果，所以对主播的需求被迫切地摆在了桌面上，需要什么样的主播？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>足够专业的、有内容输出能力的、在某一条赛道内真正能扛得起来的主播。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是淘系直播真正要做的事情，而不是表面看起来的“造星”。淘系直播不需要造星，他们要造的是行业专家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝直播近期推动的“中国新主播”大赛活动，扶持内容化主播，接下来也会有一系列动作孵化新主播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所有的动作都指向了同样一个行业走向：直播正在回归它的本质，让那些真正有干货的人输出专业价值，从这个角度而言，所有类型的直播在本质上都将成为某一类型的知识变现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主播阵营加上了一个大前提，这个群体必须有专业知识。<span><strong>小</strong><strong>P</strong><strong>老师与淘宝直播的结合，看似无意间踩中了这个风口，实际上是行业变化与新主播专业气质的必然结合。</strong></span></span></p>
