<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a href="https://www.niaogebiji.com/tag/B%E7%AB%99" target="_blank">B站</a><a href="https://www.niaogebiji.com/tag/UP%E4%B8%BB" target="_blank">UP主</a>开始推出<a href="https://www.niaogebiji.com/tag/%E4%BB%98%E8%B4%B9%E8%A7%86%E9%A2%91" target="_blank">付费视频</a>了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">6月20日，拥有上百万粉丝的B站UP主@勾手老大爷邓肯 推出了一套“付费视频”，用户可以花30元购买十集内容，付费视频只能在手机端观看。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过，用户虽然一次性付费，却并不会一次性获得所有内容，而是相当于买通了一张“订阅门票”。邓肯称每月将更新一到两次付费专题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4518735062c43891870fc4.74509302.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;">邓肯B站视频截图</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从数据上来看，邓肯的第一期付费视频播放量已达到3.3万次，排除重复观看的人数，以宽口径计算，总付费金额约99万元。扣除掉平台分成后，这笔收入也算可观。不过，考虑到部分场景需要外拍、再加上人员和时间成本的消耗，其实这份收入并不算很高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">参与本次“视频付费实验”的UP主不止邓肯一人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">6月9日，@苏司喵susiemeoww 、@叨叨冯聊音乐 两位UP主也发布了付费系列视频，其中第一期视频可免费试看。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">苏司喵的付费合集包含7个视频，需支付金额28元。截至7月1日，单条付费视频播放量为877次，忽略重复观看人次，宽口径下预估收益为24556元。叨叨冯的付费合集包含10个视频，售价30元，付费视频播放量为811次，粗略估计获得的收益大概为24330元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/111414642562c43891ae24e0.18847653.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;">叨叨冯和苏司喵B站付费系列视频</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">截至发稿，B站尚未对此次试水实验作出官方回应。从B站用户的反馈来看，很难说付费视频是一个好的商业化方向。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">付费视频，到底是做给谁的？谁能从中受益？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13363685916010dc4c78b4e6.24352516.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">付费视频，有何意义？</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当UP主的付费视频系列上线，有一个问题随之也浮出水面：对用户、创作者、B站这三者而言，谁能从付费视频中真正获益？目前来看，似乎得不偿失。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对部分用户来说，相比“大会员”，付费视频更像是为喜欢的UP主开小灶。支持自己喜爱的UP主，当然是义不容辞的。在邓肯、叨叨冯和苏司喵的视频下方，都有大量支持的声音。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，苏司喵2017年开始发布视频，凭借出色的舞蹈技能和长期的无偿分享获得关注，已在B站积累62万粉丝，是舞蹈区头部UP主。在其公布付费视频消息的视频下方，有人评论道，“买啦！加油！”还有人说，“很喜欢姐姐的教学！特别细致！收费也很正常！以后还会继续支持！”</p>
<p><img src="https://ngbjimg.xy599.com/140228174762c43891e476b8.27880379.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;">苏司喵的B站视频截图</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是，刨除掉“为爱买单”的核心粉丝群体，对于大部分用户而言，仍然要考虑“支出是否合理和值得”的问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">先行试水的UP主，也是在摸着石头过河。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对创作者来说，在“什么内容值得用户付费”这个问题上，没有一个可以量化的标准。在发布付费视频之前，UP主大概也已料想到会遭遇一些用户的不解和质疑。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以邓肯为例，他发布付费视频至今两周，已掉粉3万。近期他发布的免费视频播放量在40到60万之间，但第一期付费视频的播放量仅3万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这不难理解，常年浸泡在相对高质量视频内容环境下，B站用户对付费内容的要求也更严苛。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在评论区，不乏质疑收费标准过高的声音；也有人指出其视频中有大量片段整理自其他版权内容，收费既不合法也不合理；有的用户购买后认为质量不理想，直言“血亏”；还有人则建议分期付费，但遗憾的是，平台暂未推出该项功能；此外，有用户对视频的拍摄和制作细节提出了更高的要求，UP主则表示“那估计成本上天了。”</p>
<p><img src="https://ngbjimg.xy599.com/148583132362c43892504c76.28625966.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;">对待付费视频，用户的要求更严苛</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">6月22日，邓肯回应了关于付费视频的争议。“希望经济独立、不差钱并真心喜欢UP主的人花钱购买这个系列，大家量力而行。这次是一个全新的尝试，包了电影院拍实景，还有灯光摄像团队，投资挺大。如果能有机会给我们小团体增加一些收入，说不定我们可以扩大产能，去做一些更有趣的事情。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对B站来说，默许UP主上线付费视频，初心或许是为了吸引、留住更多优质创作者。这本是一件双赢的好事，如果真能跑通，UP主能够更有创作动力，B站也能找到新的营收模式以减少亏损。</p>

<section>然而，有多少人愿意为UP主的自制视频花钱，还要打上一个大大的问号。</section>

<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据2022年一季度最新数据，B站月均付费用户比例进一步攀升至9.3%。目前B站年度大会员售价168元，折算下来一个月顶多14元。即便是付费电影，一部电影也不过是2.5元（会员价）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">付费视频则是系列视频，需要打包购买，目前所能看到的试水案例均以28元起步。如果付费视频日渐增多，那么月度和年度大会员的价值就可能大打折扣。</p>
<section></section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13363685916010dc4c78b4e6.24352516.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">为何没有平台做成“付费视频”？</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中长视频平台至今仍然集体亏损，它们每进行一次商业模式的更新和尝试，都会引发众声喧哗。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在会员服务体系之下，推出单独收费视频的平台，B站不是第一个。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">早前以爱优腾为代表的长视频平台，就曾对OGV（专业机构制作的视频）的“二次付费”进行了探索。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2019年8月，腾讯视频利用《陈情令》试水超前点播剧集。据媒体报道，该项收入过亿。此后，爱奇艺、优酷、芒果TV等视频平台大举跟进。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但“好景不长”，这种额外付费的形式受到了大批用户的攻讦，也受到了来自第三方行业协会的批驳。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年9月，在腾讯视频对《扫黑风暴》开启“超前点播后，中消协直接发长文表达了质疑，再度阐释“超前点播”相关功能设置的不合理。同时指出用户交了会员费，但并没有真正实现“跳过广告”的效果。此外，消费者在不知情的情况下“被自动续费”，属于被平台强买强卖。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/136877138162c43892782d99.80390987.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">《扫黑风暴》超前点播页面截图</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">随后，当年10月，爱优腾集体发布声明后，属于OGV平台的“超前点播”就草草落幕。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从现阶段用户的实际意愿来看，付费视频还没有遇到最好的时机。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此前长视频平台推出的“超前点播”模式，实际上是一种“二次付费”，在诞生之初就与会员承诺的服务相冲突，最后以失败告终。B站现下上线”付费视频“，本质上也是一种叠加收费，只不过大会员主要用来观看OGV内容，而最新的付费视频系列主要适用于在某个垂类有突出技能的创作者。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为全球最大的视频网站，YouTube在很早之前就推出了独立的付费订阅频道，这些由专业视频制作商打造的频道，是独立在Premium订阅服务体系（原为YouTube Red）之外的，需要每月支付1-5美元来订阅才可观看。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但Youtube也没有将付费视频做成一门很大的生意，并以此作为平台的重要收入板块。到目前为止，Youtube的营收之中，绝大部分来自广告。据新浪科技消息，2021年第四季度，YouTube广告业务营收达到86.33亿美元，同比增长25.4%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从宣称“没有广告”到承诺“不加贴片广告”，再到上线“付费视频”，B站没有停下试探商业化的脚步。对于一家商业公司来说，这些尝试无可厚非。正是因为B站的影响力越来越大，“上线付费视频”这一举动才会如此出圈。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">若B站想将付费视频推行开来、而非浅尝辄止，必然要挑战广大用户的情感和信任。这一次，B站走到了悬崖边上。</p>
