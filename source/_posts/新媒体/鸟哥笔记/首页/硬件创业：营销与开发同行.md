                        <p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">营销是许多硬件初创公司所忽略的事情，或者至少从一开始就忽略的事情。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">许多初创公司都存在根本性的误解，即营销应该在产品开发之后进行。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">实际上营销应该与产品开发同时进行，在IPD产品管理体系中，<a href="https://www.niaogebiji.com/tag/%E8%90%A5%E9%94%80%E7%AE%A1%E7%90%86" target="_blank">营销管理</a>是产品开发管理的子流程，通过营销管理流程保障产品的成功上市。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">营销大师西奥多·莱维特认为：没有哪一个有效的公司战略不是营销导向的。产品本质上也可以说是一种营销手段，是吸引和留住客户的“道具”，这种“道具”的价值在于满足客户的需求，通过满足客户需求获得利润。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">营销有两个主要功能：触达和信任。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">首先，你必须使得你的产品引起人们的注意，让他们了解你的公司和产品；</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">其次，一旦你接触到客户，你就必须赢得他们的信任，因为你所有的业务、所有的购买都以信任为中心。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">如果有人不信任你或你的公司，他们就不会购买你的产品。因此，你必须首先使用营销来吸引客户的注意，然后在接触到他们后获得他们的信任。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">在建立信任方面，当你将自己展示为一个人而不仅仅是一家公司时，会更容易建立信任关系。这是因为与抽象业务相比，人们更容易与其他人建立联系。</p>
<h1 id="toc-1" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: center; font-weight: 600; background-size: 16px 20px;">一、推式营销与集客营销</h1>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">你可能会听到两种营销方式：推式营销和集客营销。</p>
<p style="margin: 25px 0px 20px; line-height: inherit; background-repeat: no-repeat; background-image: none; padding-left: 15px; background-position: 0px 6px; font-size: 16px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: inherit; font-weight: inherit;"><strong>1. 推式营销</strong></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">推式（Outbound）营销是一种传统的营销方式，你会积极尝试寻找可能还不知道你的存在的新客户。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">推式营销主要是向潜在客户传达自身卖点，这类营销的思路，首先是去思考要推的品牌或产品有什么优势、卖点，再将卖点用各种内容形式表现出来、公开传播。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">投放的渠道相对比较广泛，典型的有：</p>
<ol style="padding-left: 50px; color: rgba(0, 0, 0, 0.6); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>报纸/杂志广告；</strong></p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>广播、电视广告；</strong></p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>广告位、活动赞助；</strong></p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>KV\标语广告；</strong></p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>电子邮件营销（EDM）；</strong></p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>电话。</strong></p>
</li>
</ol>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">推式营销的目的是通过广泛撒网来找到可能适合你的产品和服务的广泛受众，即吸引足够的客户来转化为潜在客户。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">通常以直接可见的数据作为营销目标，例如，品牌粉丝数量、贴文点赞数量、官网流量等。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这些数据，在集客营销里通常被称为虚荣目标，也就是可能看起来很厉害，但实际转化效果不好的数据。</p>
<p style="margin: 25px 0px 20px; line-height: inherit; background-repeat: no-repeat; background-image: none; padding-left: 15px; background-position: 0px 6px; font-size: 16px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: inherit; font-weight: inherit;"><strong>2. 集客营销</strong></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">集客（Inbound）营销是人们主动找你，旨在传递价值，解决客户痛点，这意味着你需要通过创建精彩的内容供他们消费来吸引他们。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">集客营销的基本理念是向用户传达“我们能最好地满足你的需求”，集客营销的第一步不是急着挖掘产品卖点。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">而是构建潜在消费者和现有消费者的用户画像，挖掘客户的需求痛点，将客户痛点与产品优势结合，再创造出教育用户的内容。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">常见方式有：</p>
<ol style="padding-left: 50px; color: rgba(0, 0, 0, 0.6); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>搜索引擎优化（SEO）、搜索引擎营销（SEM）；</strong></p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>社交媒体营销；</strong></p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>博客、微博、微信；</strong></p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>电子书、电子海报；</strong></p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>白皮书；</strong></p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>百度问答、知乎。</strong></p>
</li>
</ol>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">集客营销细分为以下几个步骤：</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">（1）吸引</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">吸引客户，让客户主动找到你。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">你开发和传播的内容必须紧紧围绕客户的需求，社交媒体帖子、视频、博客或其他营销工作的交付时间也需要进行计算和衡量。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">内容策略必须以知道他们何时何地搜索产品和服务为中心：</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">你知道目标受众最喜欢的社交媒体平台吗？</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">他们何时以及如何访问它们？</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">（2）转化</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">将客户转化为可靠的销售线索。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">至关重要的是，你需要采用一种使你的特定目标受众感到舒适、简单和吸引人的策略来进行沟通。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">对转化步骤进行个性化设置，使其不仅与你的产品或服务保持一致，而且与潜在购买者的习惯保持一致，这将会大大提高转化率。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">（3）交易</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">在整个生命周期中培养潜在客户，管理潜在客户的渠道，利用具有相关内容来建立信任，引导他们完成购买。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">集客营销的目标就是转化：</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">可能是需要增加用户注册订阅数；</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">可能是增加用户留存率；</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">可能是直接增加成单额。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">相对而言，集客营销比推式营销更有益，有人主动来找你总是更好的（有一定程度的内在信任），然后，你就可以开始教育和建立信任的过程。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">而当你向那些已经知道你存在的人进行营销时，可能会使集客营销和推式营销之间的界限变得有些模糊。但是，主要区别在于推式营销主要是向不知道你存在的受众销售产品。</p>
<h1 id="toc-2" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: center; font-weight: 600; background-size: 16px 20px;">二、有机与付费营销</h1>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">网络营销可分为两个独立的类别：有机营销和付费营销。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">有机营销在某种意义上是免费营销，在大多数情况下，有机营销包括你创建内容，然后搜索引擎将其编入索引，除了创建内容所需的时间或金钱之外，它没有任何成本。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">有机营销是自然和可持续的，有机营销没有付费广告，依赖于真实的曝光率和参与度。其主要目的是与客户建立一种强烈的情感联系。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">付费营销包括广告等内容，你可以通过向第三方付费，将流量引导至你的网站，传统的付费营销形式包括电视和平面广告。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">付费营销是有效的，也是被迫的，并且总是被人为夸大。这意味着一旦停止为广告、促销付费，曝光率就会消失。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">有机集客营销提供最高的投资回报，但比付费广告要慢得多，会耗费很长的时间。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">而付费广告在你进行一些测试时效果最好，因为你需要快速了解与你的市场产生共鸣的内容。同时，你必须非常小心付费广告，因为很快就会烧掉很多钱。</p>
<h1 id="toc-3" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: center; font-weight: 600; background-size: 16px 20px;">三、建立网络受众</h1>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">如果你要向消费者销售产品，那么建立受众群体至关重要。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">当然，如果你准备只向大公司销售产品，那么建立受众群可能不是最佳途径。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">建立网络受众有三种不同的策略：建立、借用或购买。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">建立你的受众需要很多时间，它涉及创建博客文章等在线内容，并让人们找到你。你可以让人们访问你的网站并与你互动，例如，加入你的社群或下载你创建的免费 资源；</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">还可以借用受众，这意味着你需要在目标市场中找到已经拥有受众的网络影响者，并与他们建立关系；</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">购买是指使用付费广告来为你的网站增加流量。</p>
<h1 id="toc-4" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: center; font-weight: 600; background-size: 16px 20px;">四、内容为王</h1>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">没有内容的网站等于没有流量，你必须将创建内容作为重中之重。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这当然非常耗时并且需要付出很多努力，撰写与你的产品相关并且你认为潜在买家感兴趣的主题。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">例如，假设你正在开发宠物产品，开始写宠物主人觉得有趣的事情，并提供他们需要的信息。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">通过这种方式，你可以在真正有产品展示给他们之前，就开始建立受众并增加你的网站流量。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">专注于教学和教育，而不是销售。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">最初，你需要给他们内容、给他们建议、给他们价值。一旦你建立了深度关系，你最终可以销售。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">营销的未来是让客户想听到你的声音，而不是强迫他们消费你的内容。</p>
<h1 id="toc-5" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: center; font-weight: 600; background-size: 16px 20px;">五、文字、音频、还是视频？</h1>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">在尝试建立你的受众时，文本和信息图表具有最广泛的影响力。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">但营销需要同时兼具影响力和信任度，虽然文字在建立信任方面相当不错，但它们不如音频或视频好，尤其在情感方面。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这就是为什么音频或视频最适合建立信任的原因。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">建议同时关注影响力和信任度，高覆盖率加上高信任度等于大量销售。</p>
<h1 id="toc-6" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: center; font-weight: 600; background-size: 16px 20px;">六、结论</h1>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">对于硬件初创企业来说，营销需要成为优先事项，并应与产品开发同时进行。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">一旦你的产品最终上市销售，这样做会给你带来一群兴奋的客户。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">营销也是对新产品进行研究的好方法，它使你能够密切关注市场的需求。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">通过制作相关内容并与人们建立信任关系，你可以密切参与你试图销售的市场。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这不仅对你当前的产品，而且对任何未来的产品都是有益的。</p>                        <div style="height: 1px"></div>
