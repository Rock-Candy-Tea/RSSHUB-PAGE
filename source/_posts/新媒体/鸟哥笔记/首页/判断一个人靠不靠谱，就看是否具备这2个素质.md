<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你能让别人夸你，说你这个人实在！让人放心！那比什么评价都管用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让人放心的人，谁不愿意和他打交道呢？毕竟，这个世界上不靠谱的人太多了。</span></p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>打开短视频，看见吃播在那儿狂吃，会怀疑<strong>是不是假吃；</strong></span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看见一个浓眉大眼的小姐姐，美得不可方物，会怀疑<strong>是不是美颜造假；</strong></span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看见一个视频大V，粉丝蹭蹭的上涨，<strong>数据</strong><strong>是不是刷的呢</strong>......</span></p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个时代，有太多的套路、敷衍，不得不让人有所怀疑和防备。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>真正能用心付出、不急不躁、慢慢积累沉淀的人实在太少。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但如果，你一旦让人对你放心，你所获得的不止是眼前，更是长远的未来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，听了罗振宇的一个分享，深有感触。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家都知道，得到是罗振宇自己的APP，得到上的所有数据，他都可以在后台操控，但罗振宇毫不避讳地说：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“得到所有的数据不造假。现在正在看直播的用户，我不知道现在多少人在看，是多少人就多少人。罗胖出来讲，观众有8000人，8000就8000，2000就2000，500就500。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你要看其他的直播平台，老板一出现，立即800万人看，加两个零算少的，又不费劲，敲俩零上去嘛，有用吗？当然有用，是能解决当下问题的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，罗振宇不这么做，是因为他意识到一个问题：他的同事特定不愿意和一个骗子一起工作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当领导没有了诚信，没有了底线，员工对他就不会放心，也就不会长期在这家公司待下去。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他选择以真实的数据呈现，可能会有一些用户流失，但他得到的是一些长期作战的伙伴，并影响他们一生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，<strong>真正聪明的做人之道，就是让人对你放心。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/116321795862eb712896d894.15129698.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这越是靠谱稀缺的时代，让人放心的品质越是难能可贵，如果你身边有这样的人，一定要好好珍惜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那如何辨别一个人是否靠谱呢？就看是否具有以下两方面的素质。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>画家刘墉曾说过这样一个故事，我很喜欢：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有一天，刘墉交代儿女，给家里的花浇水。浇完水之后，才能出去玩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>女儿为了快点出去玩，很快就完成了任务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正当女儿要出门的时候，刘墉将女儿叫住。让她来到花盆前，问：“你看爸爸浇的花，和你浇的花有什么区别。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>女儿没有发现区别，便摇了摇头。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时，刘墉把两盆花连根拔起，将花的根部给女儿看。此刻，女儿的脸立即红了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为爸爸浇的花，水已经渗透到根部了，而自己浇的花，水只停留在表面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过这件事，刘墉对女儿说：<strong>“任何时候，做事都不能只做表面功夫，要做一个有责任心、有担当，和可靠的人。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么叫有责任心呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我觉得司马光的一句话，可以高度概括：<strong>“尽心于人曰忠，不欺于己曰信。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>意思是，对别人的事，要尽心，不要连自己都欺骗自己。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一件事没有办成，我们常说“我尽力了。”其实，很多时候，都没有尽力，甚至根本没有走心。只是把走了一遍过场，当做尽力来交差。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而有责任心的人，是一口唾沫一口钉，实实在在！答应你的事，就真的是全力以赴，不遗余力。言既出行必果，让人放心。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/30186654862eb71290c84d0.53703131.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个人让人放心的最高境界，就是：为人诚信，办事负责。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>威尔逊说：<strong>“责任感与机遇成正比。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>确实如此！<strong>当你对别人尽心竭力，让人放心，那谁都愿意把事情托付给你，你的机会不就越来越多吗？</strong></span><span></span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><strong><span> 3、</span></strong><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">一个人能否让人放心，</strong><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">能力很关键</strong></h2>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让人放心，光有责任心这项软实力还不够，还必须要有过硬的能力才行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就好比：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国家的复兴不能仅靠外交的强大，还要有科技、军队等过硬的能力；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个婚姻家庭不能仅靠爱情维系，还要有柴米油盐、房子等现实的财力做支撑；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要想升职加薪，也不能仅靠勤奋，还必须有能力做出些业绩才行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有能力的人，就是有把事做成的本事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当我们放心地将一件事交给一个人时，是打心眼里相信他有能力办好。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/208323840662eb71295e0f56.73103685.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们部门有个运营同事小七，她是个急性子，只要有紧急的工作任务，就会特别的焦虑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，我耗时1年多的一个新课，终于要和大家见面了，但是有一个紧急的任务，那就是要在一周内的时间里，准备一篇优质的课程宣传文案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这个任务交给了小七。小七不会写文案，只能临时找人代写。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，外面的写手能力参差不齐，找了好几个写手，花了3天时间，都没有可以用得上的文案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>眼看还有4天就要对外推广了，小七急得嘴角都长痘了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着时间的推移，原本以为小七会更焦虑上火，但没想到第二天，小七一脸轻松，慢悠悠的来上班了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小七说：“昨晚我找到了一个文案大佬，绝对可以保证4天后出稿。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你看，<strong>这就是专业能力的魅力。只要把一件事交给他，就会很放心。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/185800982762eb7129b547c8.45261589.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之前，看到一个网友的求助信，他写到：公司要举办一个大型的活动，求推荐靠谱的主持人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有一个人的评论，让人既好笑又无法反驳，他是这样回复的：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“推荐汪涵、何炅、董卿。只要他们随意一个人往台上一站，你的活动就稳了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽有调侃的意味，但说明了人们对于那些，业务能力强的专业人士的肯定和放心，相信他们能应对各种各样的情况。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，<strong>一个人能否让人放心，不仅需要软实力，也需要过硬的能力做支撑。</strong></span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传统经济学中，广为流传着一种说法：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>人都是理性的，他们只会把钱借给靠谱的人，也只会借给有能力还钱的人。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同理，</span><span>职场中让人放心的人，一定是那些有责任心，和有能力的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>努力让自己成为一个让人放心的人，是你升职加薪的不二法门。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，这也是人生中的一种修炼，需要我们在成长的道路上不断修正前行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一起加油吧！</span></p>
