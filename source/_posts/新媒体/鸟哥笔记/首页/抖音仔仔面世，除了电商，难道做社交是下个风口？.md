                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最近抖音刚上线的抖音仔仔，你用了吗？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在抖音最近刚更新的版本中，抖音仔仔功能惊喜上线，支持用户在抖音APP内打造属于自己的虚拟玩偶形象。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" align="center"><img src="https://img.jiemian.com/101/original/hainatemp/20220815/166052822856927600.png" alt data-cke-saved-src="https://img.jiemian.com/101/original/hainatemp/20220815/166052822856927600.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲图片来自抖音</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了创建玩偶，抖音仔仔还支持添加心情设置为抖音动态头像，跟好友互在特定场景进行互动，比如顶呱呱、一起露营等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" align="center"><img src="https://img.jiemian.com/101/original/hainatemp/20220815/166052822886547000.png" alt data-cke-saved-src="https://img.jiemian.com/101/original/hainatemp/20220815/166052822886547000.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲图片来自抖音</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">抖音APP内内置了一个全新的功能，且强化了社交属性，不免令人想起早年的QQ秀，以及近年来很火的元宇宙社交软件啫喱。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>抖音强推抖音仔仔，多位头部达人鼎力推荐</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了上述所说的功能，抖音仔仔还未开放与好友通过虚拟形象进行实时交流等更深度的功能。未来在抖音APP内，用户将可以获得更多虚拟社交场景所带来的新体验。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">看抖音仔仔的入口设置，可以知道抖音给了不小的流量曝光支持。不论是在推荐页刷视频还是刷直播间，界面的左上角都放置了活动浮窗，内嵌抖音仔仔活动落地页。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" align="center"><img src="https://img.jiemian.com/101/original/hainatemp/20220815/166052822944148100.png" alt data-cke-saved-src="https://img.jiemian.com/101/original/hainatemp/20220815/166052822944148100.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲图片来自抖音</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">部分用户可以在消息页，直接点击抖音仔仔头像，就可以进入页面，进行换装、发布心情等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" align="center"><img src="https://img.jiemian.com/101/original/hainatemp/20220815/166052822978127100.png" alt data-cke-saved-src="https://img.jiemian.com/101/original/hainatemp/20220815/166052822978127100.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲图片来自抖音</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">抖音还邀请了多个抖音仔仔星推官，比如密子君、爆胎草莓粥、锅盖er，用户可以用自己的抖音仔仔与达人的虚拟形象合拍的方式来推广新功能。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以及发起#抖音官方给我发了个仔#话题，邀请不少垂直类目备受年轻用户喜欢的KOL达人，比如，有千万粉丝的机车女神@痞幼，变装达人@刀小刀sama、颜值博主@一个阿茶等等来安利抖音仔仔，该话题的视频播放数已经突破20亿，热度在持续走高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" align="center"><img src="https://img.jiemian.com/101/original/hainatemp/20220815/166052823015012900.png" alt data-cke-saved-src="https://img.jiemian.com/101/original/hainatemp/20220815/166052823015012900.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲图片来自抖音</h4>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>元宇宙社交大火，深度沉浸感、真实感社交</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">抖音推出抖音仔仔这一动作，也被不少人视为抢夺年轻群体的注意力。毕竟在今年2月，一款以元宇宙为出发点的社交软件啫喱一夜爆火，推出不久就冲上App Store总榜第一名，人人求注册的火爆程度，也让不少巨头看到了元宇宙社交、虚拟社区的潜力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" align="center"><img src="https://img.jiemian.com/101/original/hainatemp/20220815/166052823043406100.png" alt data-cke-saved-src="https://img.jiemian.com/101/original/hainatemp/20220815/166052823043406100.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲图片来自网络</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在推出抖音仔仔之前，在今年7月，字节跳动的另一个线上虚拟社区App派对岛正式上线，在派对岛内集合了目前爆火的虚拟社交玩法，比如创建虚拟形象沉浸式聊天互动、实景化社区，各类丰富的线上社区活动，如线上放映室、交友派对、晚安故事、读书会等，让用户沉浸式体验虚拟世界。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">虽然字节跳动并未把这款产品定位为与元宇宙相关的概念，但是其模式还是令人不禁联系大火的元宇宙社交。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" align="center"><img src="https://img.jiemian.com/101/original/hainatemp/20220815/166052823072491200.png" alt data-cke-saved-src="https://img.jiemian.com/101/original/hainatemp/20220815/166052823072491200.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲图片来自抖音</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前，在各大应用商店内已经搜不到派对岛，暂时全面下架。而下架不久后，类似概念的抖音仔仔却内嵌入抖音APP之中，可见让用户学习探索一个全新的APP其门槛和难度，直接内置抖音，其曝光和推广力度效果不言而喻。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">未来抖音是要深度运营抖音仔仔还是再度重启派对岛，暂时还不可知。但是可以知道的是，抖音正在从不同场景去测试社交类产品的可能性，可见抖音想做好社交的野心。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>小结</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年可以说是元宇宙元年，21年10月Facebook改名为Meta，并开始全力投入元宇宙相关产业后。此后国内多家科技巨头向“元宇宙”进发。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">市面上主打“元宇宙社交”的诸多软件中，目前业内人士认为，本质上跟元宇宙关系并不大，大多还都停留在对之前一些旧玩法的升级或者是场景人物3D化，并未带来革命性的沉浸感的功能体验，而沉浸感又恰恰是“元宇宙”这一概念中着重提及的东西。VR/AR领域技术的创新、迭代仍需要很长的时间，可以说元宇宙社交成为风口，还有很长的一段路要走。</p>                        <div style="height: 1px"></div>
