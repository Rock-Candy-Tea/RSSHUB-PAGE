<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>入职大厂三个多月来，明显感觉到比之前在创业公司更加累。虽然之前在上海，现在在成都，但地域的变迁并没有带来很多人以为的更加轻松的工作体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当我跟很多人沟通时，发现大家对于「大厂累」这件事表现出明显不同的两种态度：</span><strong><span>一方认为，「大厂累」对人的摧残太严重了，必须要逃离，外企或者中小厂可能是更好的选择；另一方认为，「大厂累」和它的高薪水是匹配的，你拿着那个层次的薪水，那就必须硬扛下去。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经我也以为逃避是一种消极的应对态度，硬扛才符合大家更认可的「奋斗者精神」。但时间长了我发现这是一种偷懒和欺骗，一味地硬扛只会让自己的身体和精神在一步步消耗，到了某个临界点就会爆发：或者是裸辞，或者是病倒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>身在这样的环境中，只有真正了解到底环境是如何运作的，搞清楚「大厂为什么这么累」，我们无论是坚持下去还是另有选择，都是一种更清晰的选择。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是我想要写这篇文章的原因，我并不想劝退或者号召，</span><strong><span>我只是想告诉你们里面发生的故事是怎么样的，它让人疲惫的原因到底是什么。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我相信无论是你正在大厂奋斗，或是准备进入大厂，或者刚刚离开，我们都能从对环境的观察中，更好地了解自己。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>一、为什么下班这么晚</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提到大厂，很多人想到的就是「加班」，就是「996」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我在加入大厂之前，曾经许愿希望自己不要经历无意义的加班，希望有更多的时间回到生活。事实上，几个月下来，我确实没有觉得自己遭受那些无意义的加班，因为：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我已经习惯了比一般人晚下班，并且不觉得那算作加班。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我所在的工区是成都一个超甲级写字楼，最近是夏天，成都的天黑得很晚，差不多 8 点 30 左右太阳才会完全下山。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每天吃完晚饭的时候，站在写字楼高层看着日落和回家的行人，免不了心生一些羡慕。但我知道我的工作还没有做完，我必须继续跟他们说再见，然后再继续我的工作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在很多城市，大厂人在工作日是很难跟夕阳一起回家的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么下班这么晚？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>翻开我们的日历你就会看到，几乎每个人每天都是满满的日程，就连需要专心码代码的研发，每天可能都会有好几个会要参加。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>白天会议太多，是很多大厂人下班晚的第一个原因。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那为什么有那么多会议呢？我归纳为两点：</span><strong><span>协作和流程。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>协作，是指在大厂要完成一件事，往往要经历很多的跨团队配合。就以我们产品经理来说，在每天的工作中，几乎都是跟设计、运营、研发等不同角色的沟通。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>大厂是不需要个人英雄主义的，它不需要一个人包揽所有的事情，但要求一个人将自己分内的事情做好，在协作中不给整个团队拖后腿。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，如果仅仅是因为协作，也不必导致如此多的会议。与协作相应而生的，是流程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我刚刚毕业的时候也去了另一个大厂，那时候觉得一件事情做得好与坏，与这个人的绝对实力并没有太大的关系。有时候一个工作，无论谁去做，好像最终都是那个结果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那时候不理解，现在想想这就是流程的作用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么很多人将大厂人称之为螺丝钉，一方面是因为大家手头的工作被划分得非常细，另一方面是因为，我们都身在流程中。因为流程的好处是：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>保证产出结果的稳定性。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你很喜欢喝咖啡，就知道精品咖啡和商业咖啡的一个区别，是精品咖啡店偏好浅中烘的豆子，这样能够充分发挥豆子本身的香气，但不稳定，跟气候、产地、烘焙手法息息相关。而商业咖啡店（大型连锁咖啡店）偏好深烘的豆子，出品稳定，口味一致性较好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>越是大的系统，越要摆脱对个体的依赖，大厂尤其如此。</span></strong><span>在如此大规模员工的情况下，如果一项工作因为一个人的离职或者发挥失常就大受影响，那大厂将很难为他们的客户/用户提供质量稳定的产品与服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>充满流程的协作，是会议太多的重要原因。而会议太多，又是下班晚的重要原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然有人会说，很多时候下班晚是因为老板下班晚，自己也不敢早点走。当然，这也是一种原因，这里就不做详细展开了。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>二、总是有种失落感</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我总能在大厂人身上看到一种隐隐的失落感，虽然他们每天也很快乐，也很充实，但很少有人以自己的工作为傲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之前在创业公司的时候，同事的朋友圈里充满着对自己工作内容的自豪感，但是到大厂之后，就看到大家很少发工作相关的内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我之前并没有特地注意过这样的现象，有时候觉得那些创业公司的人只是在寻找一种存在感。但当我经历了这几个月之后，我发现背后是一种情绪的体现，那就是失落感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>失落感的来源，是个体存在感的缺失。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用我朋友的话来说，</span><strong><span>“总有人比你更聪明，总有人学历比你更好，总有人比你更能说，总有人比你更有逻辑，好像之前面试中我引以为傲的那些特质，在进入这个集体后显得那么的一无是处”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们从小接受的教育是「木桶效应」，但真正走上职场之后会发现，</span><strong><span>一个人如果拥有属于一个属于他自己的长板，将是一件多么难得且幸运的事情。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当我们追求弥补短板的时候，在均值回归效应的作用下，大家会变得越来越趋同，</span><strong><span>“如果我没有在某个方面比其他人强，那这个特质带给我的更有可能是失落感。”</span></strong><span>这是我观察到的现象，而我自己也正在经受着这一情绪的煎熬。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还记得么，大系统是不需要英雄主义的，它要的是流程作用下稳定的输出品质。</span><strong><span>但个体是需要一点英雄主义慰藉自己的，他们希望找到自己的位置，看到自己不一样的价值。</span></strong><span>这是个性和系统的冲突，而这一冲突，就是个人失落感的来源。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>三、为什么说话更少了</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>必须承认，大厂是有一套自己的话语体系的。当我从创业公司进入到大厂之后，一开始最难适应的就是大厂独有的话语体系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人嘲讽这样的黑话，但我觉得也要区分去看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以产品研发为例，很多话术来自于 Scrum（一种产品研发流程），久而久之，大家都以流程中的专业术语作为自己日常沟通中的话术，</span><strong><span>这样能够提高沟通效率，尤其在专业性较强的语境下。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但另一些话术就纯粹是人为创造了，比如造新词，比如用一些模棱两可的词汇去描述一个具体的意思。这些黑话说多了，确实会使得你的表达变得更复杂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，虽然我们每天都在开会，也可能都要在会议上说话，但摘下耳机之后，大家似乎就没有什么语言沟通了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，沟通依旧还在，我看到过很多人一边开着工作通讯软件，一边开着微信，一边发着工作报告，一边发着表情包。</span><strong><span>我们在工作之外，跟自己的朋友、家人闲聊，但是在工作中，却不愿意跟身边的同事聊家常。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018 年刚毕业的时候，我的微信里还会有很多的同事，大家在工作中讨论，在微信里吐槽。但随着工作年限的增加，我微信里的同事越来越少，我也很纳闷这种变化到底是如何产生的，但是当它发生的时候，一切也显得很自然，并没有什么不适应。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我猜测，</span><strong><span>很可能是我们花在工作中的时间越多（不管是主动的还是被动的），我们越想要将工作和生活划清界限。似乎在休息的时间不打开工作软件，不聊工作话题，变成了我们作为一个还在生活的人的最后的尊严。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这种情况下，同事便更有可能被我们放在了工作的那一部分，他们被那条界限挡在了每个大厂人的生活外，如果一个人不在你的生活圈子里，那说话的必要性又在哪里？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>商务沟通的背后是功能价值，生活中的闲聊背后更多的情绪价值。</span></strong><span>我们习惯于从生活中获得情绪，然后在工作中慢慢消耗。很少人从工作中获得情绪，当你企图去获得情绪时，你很有可能同时会释放情绪，这在职场中是一件非常危险的事情。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>四、为什么事情推进都那么慢</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，我们聊聊一个很多人都在吐槽的点：为什么大厂中推进事情往往都很慢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从常识来看，大厂的员工更多，岗位更细分，那是不是事情推进得更快呢。并不是，从结果来看，在大厂中推进大部分的事情都相对更慢，除了前面说的流程原因之外，我还想补充一些自己的看法：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、依赖资源的人太多了</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>大体上，我将一家公司的员工分为两类：需要资源的人，和能提供资源的人。</span></strong><span>这里的资源说的是能直接完成末端产品交付的资源，在互联网大厂里，那就是研发资源。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在研发环节之前的所有角色，贡献的都是智力资源，还记得那句玩笑么，我想创业，有一个点子，就差一个程序员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>智力资源本身并不是坏事，但智力资源多了，优中选优、优胜劣汰的过程就会非常漫长。</span><strong><span>正是因为研发资源一直紧缺，所以大厂的很多流程都花在了智力资源的 PK 上。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那小公司呢？很多大厂员工可能没有小公司的工作经历，他们无法想象一家小公司的项目团队是如何在没有资源的情况下创造百万级别的 GMV。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在这方面，大厂员工一定要谦卑，一定要知道，你所有的智力资源，在没有形成交付产品之前，都是空中楼阁。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个世界不缺聪明的头脑，缺的是踏踏实实地将事情完成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所谓大厂光环的另一面，便是这样的资源诅咒。</span></strong><span>“要到资源是一件天经地义的事情”么？并不是这样。向外看看吧，创业者的可贵品质之一，就是在没有资源的情况下依旧能干成一些事儿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2、目标不一致</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有没有感觉到，很多时候我们推进事情慢，并不是因为你的效率低或者你的团队不给力，而是因为其他团队并不是那么「配合」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，这件事从不同的角度看有完全不一样的结论，从你的角度来看是对方不配合，从对方的角度看是浪费资源做对他们来说不重要的事情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本质上，是因为双方目标不一致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目标对不齐并不是哪一个人的责任，甚至，很多时候大家的目标在书面上是一致的，比如都是为了产品有更好的用户体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>事实上，大多数目标不一致的矛盾根源，是大家对于目标理解的不一致，并且虽然彼此都知道对方的理解是什么，但还是习惯性坚持自己的理解。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这是 ego 的作用。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很难有人完全放下自己的 ego，去接受一个别人提出来的看起来更正确的想法。对于大厂来说，大家都是聪明人，都是精英，证明自己是精英的常见方法就是坚持自己的想法并拿到结果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在结果没有拿到之前，如果过早降低自己的ego，就仿佛失去了尊严。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，这是大厂内部自带的冲突，这是人性作用下的必然结果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3、每个人都需要做点什么来证明自己的价值</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后是一个发自灵魂的提问：</span><strong><span>有没有可能，大厂其实并不需要那么多人做那么多事情，但每个人都需要做点什么来证明自己的价值。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于产品经理来说，很有可能你这一个月不做任何事情，让产品自然作用，就是对用户最负责任的做法。但如果你这一个月不做点什么，那用什么来证明公司需要你的原因呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们经常看到，一个产品或者项目在完成交接时，新接任者很难保持「萧规曹随」，而总是要做点什么大动作来证明自己过来的价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是重新开始写规划，还是憋大招，很多时候我们并不能肯定做的这些事情到底是不是真的正确。但只有它足够大，足够帮你争取到存在感，那这对你来说就是有价值的，在下次评绩效的时候就有故事可以说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>否则，你在这个系统里存在的价值到底是什么呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>克制这种想要做点什么的冲动是很难的，对你很难，对你的老板很难，对体系内的每个人都很难。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当你想要做的事情太多了，你自然会发现，推进他们好难、好慢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这么说吧，很多智力成果注定是要被浪费的，他们存在的唯一价值就是给你一个存在的理由。</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>结语</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大厂的累，有身体的累，有心里的累，但更多的是心里的累。很多时候，勤奋不能解决心累，只有真正意识到这个系统里正在发生的是什么，才能有更正确的心态去面对它。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>希望对你有帮助。</span></p>
