<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年2月份，@<a href="https://www.niaogebiji.com/tag/%E6%9C%B1%E4%B8%80%E6%97%A6" target="_blank">朱一旦</a>的枯燥生活（以下简称朱一旦）创作的《一块劳力士的回家路》风靡全网，B站，微信生态圈和知乎等平台全部爆火，令其出圈。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>实际上不止如此，从2019年8月-2020年5月，@朱一旦的枯燥生活9个月全平台狂涨1200万粉丝，成为</span><span>各平台都能通吃的现象级视频博主。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/172504131162c0d092551b76.55521817.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但在2020年10月份，导演兼编剧@张策的离职，扼杀了这种爆发式增长的势头。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为更换了新的导演编辑，其视频风格、观看效果等和往常不一。朱一旦在B站的粉丝数逐渐下跌，从巅峰时期的500万，掉到了如今的389万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>最新一期B站视频发布于2021年8月26日，距今已经停更将近1年。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11443616062c0d0928b0de7.95188792.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而离职后的@导演小策，也在B站开始做号，创作喜剧类型视频，2年不到粉丝数已经达到324万。如此戏剧化的一幕，使很多粉丝戏称“双向奔赴”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/60994033262c0d092ce34f9.00111873.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/47707177562c0d093014a79.92299053.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>虽然朱一旦的B站账号停更了，但是从2022年开始，他的<a href="https://www.niaogebiji.com/tag/%E8%A7%86%E9%A2%91%E5%8F%B7" target="_blank">视频号</a>恢复更新，在4月份更是创造了双10W+（点赞及评论）的爆款。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而他的抖音账号，也依旧稳定更新，粉丝数达到700万。虽然很多网友及黑粉唱衰他，但他的全网粉丝数依旧超过1500万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/25735848162c0d0933b1e90.85778624.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>爆款视频<br></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/46640973462c0d093a22ba4.34943653.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>从疯狂涨粉，到疯狂掉粉，再到放弃B站进军视频号重新制造出爆款，中间发生了什么？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>B站账号为什么要停更？能保持继续出爆款的势头吗？对新媒体人和博主们有哪些借鉴意义？</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>1. 爆款视频，走红全网</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年2月份（疫情期间），朱一旦发布的视频《一块劳力士的回家之路》当时大爆，登上微博、知乎热榜，在B站拥有超过1000万的播放量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/202281207762c0d0943422c6.46319678.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这条视频依旧延续了原有风格，将自身定位成老板的角色，描述了一批口罩在不同阶层群体中周转的过程。<span>由于这支视频影响颇大，当时一度遭到封杀。</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/66054967762c0d095910ee5.44037685.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年五四青年节期间， B站官方推出了一条视频《后浪——献给新一代的演讲》，播放量迅速突破1000万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一天后，@朱一旦紧跟这条热点，发布了一条视频《非浪——献给新员工的演讲》。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这条视频站在与员工对立面的角度，讲述员工作为韭菜被资本家收割送往非洲（比喻词）的故事。<span>3天时间播放量突破400万，知乎话题浏览量超过1000万。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/121877050662c0d098cfa357.20945298.gif" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，好景不长，核心人员导演兼编剧张策的离职，令其团队遭受了很严重的商业损失。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>2. 导演兼编剧离职，迅速掉粉</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>202年10月16日，《朱一旦的枯燥生活》系列的导演张策宣布离职，很多人才知道，原来朱一旦的大多数作品，是@导演小策操刀的。一人身兼导演，编剧，配音等职位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/16699834062c0d09942b836.00784591.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据小公举观察，@导演小策从2019年开始接手朱一旦这个IP，一年多时间里。</span><span>朱一旦的公众号头条阅读量迅速从平均3K涨到3W，粉丝数翻了10倍以上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/47049589262c0d0996ebda8.82569712.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>朱一旦的B站号从10万粉丝到500万粉丝，只花了377天，堪称现象级UP主，</span><strong><span>视频更新频率高达2天1条。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>不过在@导演小策离职后的第一个月，其只更新了2条视频，说明和核心员工离职对账号的影响是巨大的。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/139117147862c0d099a624d4.59091227.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后期，@朱一旦又陆续更新了一些视频，但是播放数据整体下滑，从几百万播放量下滑到大多数在100多万左右，远远不及巅峰时期的数百万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这一系列结果，主要原因正如评论区的网友们所说，因为换了导演编剧，配音变化，缺少了原来的味道。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/170298720862c0d099eeb8e3.33360538.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/181657636862c0d09a37c6c7.46510555.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在导演小策离职后，@朱一旦的视频更新很不稳定，总共停更了3次，时间都较长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年10月-2月正常更新，随后停更；2021年5月恢复更新，很快停更；2021年8月恢复更新，再次停更。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在这么长的一段时间内，其创作团队可能一直在思考，究竟是用过去的BGM还是转型。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但在很多网友看来，沿用过去的BGM，却不得其神韵，所以在@导演小策离职后，朱一旦的视频风格一直在变化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如离职后，朱一旦更新的第一条视频，吐槽原来的BGM已经腻了。接着画风忽然一边，开始唱歌，喊“感谢老铁们”，颠覆了过去的IP形象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种恶趣味/无厘头的风格，和过去反差太大，或许这是转型，但一些网友却不喜欢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/20433514662c0d09a8006c0.54180394.gif" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/67029811162c0d09ae93987.52527442.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/55328908162c0d09b4204a7.18209737.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>截止目前，其B站视频已经停更近1年。我想可能</span><span>是由于B站的弹幕及社区文化氛围会影响视频。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如公屏上的弹幕，有很多人会吐槽视频及up主，那就会影响到第1次看视频的新用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而像抖音，视频号目前就没有这种文化，就不用担心弹幕里的粗言秽语影响到其他用户观看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为确实从导演小策离职后，有很多人会跑到朱一旦视频的评论区及弹幕里，说“离职是双输”，“给的太少”，诸如此类的留言。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>尤其是当@导演小策自己做账号，并且创业成功，粉丝数即将赶超@朱一旦的账号后，这种现象愈演愈烈。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>导演小测离职前的视频大多数特点：<strong>视频短而精（1.5分钟以内），演员演技在线，音乐卡点准确，带点悬疑风。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/197420880362c0d09b7281a3.39495190.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>2019年-2020年视频</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>离职后，可能是由于换了导演编剧，又或者是转型。其</span><span>视频时长，很多达到3分钟，甚至4分钟以上。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/38065294462c0d09bbc12b8.69634636.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>2021年视频</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而以前的视频只有40秒-2分钟，从短视频变成了中视频，悬殊较大。原本喜欢看短视频的粉丝，不一定喜欢看中视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>就我个人觉得，有的剧情，有点拖沓，用多个剧情/慢镜头强行烘托气氛，但是没有爽感。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是从低向高怼脸拍，其实让人不太舒适。@导演小策离职前，很多镜头是斜着拍摄，中镜头取景。<span>（不过后续的视频，调整改善了拍摄手法</span><span>）</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他的视频里涉及到的人物较多，往往一个画面出现三四个人，这要求每个出镜人物都具备较好的演技。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>编剧需要给每个演员设计恰当的台词，抓拍出每个演员的神韵，拍摄出的画面要没有违和感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这考验团队的整体能力，难度很大，所以一旦拍不好，甚至个别画面衔接不好，就是吃力不讨好。以至于很多人看了，觉得尴尬。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这类具备讽刺意义的视频，我认为时长1分钟以内比较好控制。时长达到几分钟，创作难度就成倍增加，视频扑街率还会变高。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>3. B站停更，进军视频号，再出爆款</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然放弃了B站，但@朱一旦转移到了微信视频号这份热土，在今年4月份恢复更新，稳定运营至今。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/149449620862c0d09bf19945.71582046.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4月份微信视频号发布的这条《</span><span>无意间的分享，变成了一场暖心的接力游戏</span><span>》，成为双10W+爆款视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>很巧合的是，这条爆款视频依旧是和疫情相关的（2020年走红的也是疫情相关视频）。视频核心故事，是</span><span>通过以物换物的形式，体现大家善良的内心。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>要注意的是，这条视频依旧沿用了@导演小策在职期间的BGM</span><span>，拍摄手法，例如慢镜头，大量无台词画面。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我分析后判断，@朱一旦目前的视频可能不止一个编剧/导演，因为有的视频延续老风格，节奏慢，相对压抑。有的视频尝试了新风格，音乐和镜头节奏轻快，配音清朗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我想，这是在转型路上不得不做出的妥协。<strong>一方面希望转型后的新风格有人喜欢，一方面使用老风格涨粉。<span>通过这种方式，</span><span>潜</span><span>移默化</span><span>改变老粉丝的</span><span>观看</span><span>口味。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/73197501962c0d09c8e03f4.31572967.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近2年时间，反复调整，如今@朱一旦的视频找回了原有的感觉，更新频率稳定在周更。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其抖</span><span>音账号目前的点赞</span><span>数</span><span>据，</span><span>虽</span><span>然远</span><span>不</span><span>如</span><span>2020年的动辄几十万，但也保底在</span><span>2-</span><span>5万，并且掉粉率远不如B站那么高，今年只掉了20多万粉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>未来，他的视频还会变成什么样，感兴趣的新媒体人可以继续观察。</span></p>
