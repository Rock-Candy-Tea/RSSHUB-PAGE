<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><span>2016年，由蘑菇街领衔，直播电商这一全新商业模式诞生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3年之后，突如其来的新冠疫情席卷全球，也为这一新业态的发展提供了充足土壤。直播电商，成为全新的风口。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再3年后的今天，国内直播经济空前繁荣，一众巨头正带领着直播电商进入全球化的征途。这其中，全球电商龙头亚马逊和短视频龙头<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3" target="_blank">抖音</a>却走向了不同的道路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>抢占网红，亚马逊全军出击</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>蛰伏许久，那头饿狼终于还是来了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>日前，有外媒消息称，亚马逊正紧锣密鼓，加快借力网红进军QVC（一个家庭购物网络）式直播购物市场的计划，希望借此复制社交媒体竞争对手的成功，重振萎靡的在线销售。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/101521129662ccbd2f098b24.79323985.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此前，随着Tik Tok（抖音海外版）和YouTube等公司与QVC接轨，直播购物的热潮开始在美国掀起，市场一片火热。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，亚马逊的电商基本盘也开始动摇。今年一季度，亚马逊迎来了自2015年以来的首次季度亏损，其中，电商销售额同比下降3%，遭遇重创。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，亚马逊急需一个新的增长引擎，火热的直播电商无疑是最好的选择。更何况，亚马逊也并不是直播带货市场的新玩家了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在2016年，亚马逊便推出了Style Code Live直播秀，对其网页上出售的时尚单品进行展示。尽管由于战略原因，这款节目在开播15个月后便迎来结局，但不可否认的，这次直播带货领域的首次尝试，为亚马逊积攒了许多宝贵经验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因而在2019年，亚马逊再度推出的Amazon Live直播购物服务要成熟的多。同时，该业务也逐步成为其在线销售的核心焦点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/67974086062ccbd2f4965b2.52734912.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>亚马逊直播业务高管Wayne Purboo曾表示，</span><span><strong>他相信“直播购物是零售业的未来”。</strong></span><span>而为了在这一未来购物市场中分一杯羹，亚马逊如今正豪掷万金加快抢夺网红的步伐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自2020年以来，亚马逊一直在联系拥有大量粉丝的Tik Tok、YouTube等平台的内容创作者，试图将其纳入自家直播电商平台中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，这一动作更加迅速，开出的报价更是由此前的每月2000至9000美元不等飙升数十倍。面向具有高知名度的Tik Tok内容创作者，亚马逊开出了高达近10万美元的报价，显然，它已经等不及了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着这一大手笔抢人，亚马逊在直播电商领域开始全军出击。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>限制扩张，抖音海外大溃败</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在亚马逊忙着进击之时，抖音在海外迎来大溃败。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，有消息称Tik Tok在英国测试失利后，或将放弃继续在欧洲及美国拓展直播带货业务。而后，官方也对这一传言进行了证实，并表示仍将专注于该业务在英国市场获得成功。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，英国还只是抖音走出亚洲，在世界范围内扩张直播电商的第一站。怎料这第一场海外战役就如此坎坷，来自中国的神奇魔法失灵了吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134911807762ccbd2f961143.83629663.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年底，Tik Tok开始在英国试点直播带货，并启动了一项名为“On Trend”的活动。彼时，其希望在英国的试点取得成功后，逐步将直播电商业务扩展至德国、法国等欧洲国家，最后再进军美国领地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而理想很丰满，现实很骨感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>走出亚洲的Tik Tok多少有些水土不服，不仅英国的直播带货业务未能取得预期效果，“On Trend”活动也没能吸引多少用户下单，甚至该平台的带货主播也都基本退出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>海外扩张的第一站，抖音便遭遇当头棒喝。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>究其原因，有观点认为是Tik Tok所采取的“高性价比”策略与当地消费者习惯不匹配所导致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同于国内，拼多多已经用实际行动证明了高性价比的发展道路。而在<a href="https://www.niaogebiji.com/tag/%E6%B5%B7%E5%A4%96%E5%B8%82%E5%9C%BA" target="_blank">海外市场</a>，这一方面让商家承受着低价的经营压力，使得产品质量难以保证；另一方面也无法给当地消费者良好的购物体验。一旦停止补贴、价格回调，商家订单就会遭遇断崖式下滑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在海外的发展上，不仅商家和消费者两难，甚至Tik Tok的英国员工也与公司间有着冲突。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在上个月，由于Tik Tok管理团队将国内“996”职场文化带入英国，使得引发离职热潮。Tik Tok英国员工表示，他们经常被要求一天工作12个小时以上，咄咄逼人的公司文化、不切实际的目标和期望与英国典型的工作方式背道相驰。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/115866344862ccbd2fe4ef05.19960267.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>甚至还有Tik Tok高管公然宣称“企业不应该提供产假”的言论，在国内互联网掀起舆论浪潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，抖音在海外直播电商的扩张上，还没能找到一个合适的发展路径。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是对公司内部管理、还是面向商家、主播和消费者，离开了亚洲的Tik Tok，或许需要转变下思路，尽快找到一个适合欧美国家的电商业务模式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>毕竟在电商的路上，Tik Tok背后还有群狼环伺。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>海外直播带货：本地化是关键</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由国内掀起的直播电商风潮正在席卷世界，一场时代变革正在缓缓拉开帷幕。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而亚马逊的重金出击和Tik Tok的水土不服，则暴露出一个巨大矛盾，即海外巨头觊觎直播电商大蛋糕与市场并不成熟间的矛盾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>初来乍到的Tik Tok人生路不熟，空有一身国内造就出的直播带货经验却难以施展；海外巨头熟悉本地市场，却巧妇难为无米之炊。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>因而对于直播带货的出海而言，最重要的便是一个“本地化”的问题。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/23288831362ccbd306c0973.68257827.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进入海外市场，面对全新的商家群体、全新的购物习惯、全新的市场环境，在拥有成熟直播带货经验的前提下，“本地化”将会是破局的关键。而本地化，又大抵可以分为人的本地化、产品本地化、运营本地化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在人的方面，当团队都是中国人的情况下，又怎么可能做到真正的了解当地用户，洞察真实需求呢？因此必须要加入本地基因，吸纳本地人才，给予当地员工更大的发展空间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在产品方面，尽管国内已经习惯了低价、高性价比的直播带货模式，但显然，海外用户似乎并不买账。因此从当地畅销产品、当地用户实际的需求处出发，或许可以迎来转机。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在运营方面，无论是活动的策划、内容的运营也都应该从当地实际出发，依据当地的流量生态设置合理的机制和玩法。例如国内的618、双十一大促在国外并无太大知名度，反而“黑色星期五”、Prime Day则是典型的大促节点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/42734480062ccbd3095f798.86553874.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，对于Tik Tok而言，在字节跳动面向国际化发展的道路上，其本地化已经做的不错了。目前的Tik Tok在业务上已经与国内做了完全切割，几乎就是一家独立存在的公司。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但从企业文化冲突导致员工大批离职事件来看，它做的还远远不够。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在直播带货的市场上，疯狂扩张的抖音，与亚马逊等国际巨头的碰撞不可避免，二者终有一战。</strong></span><span>届时，做好本地化理解的Tik Tok，才有能力继续疯狂跳动。</span></p>
