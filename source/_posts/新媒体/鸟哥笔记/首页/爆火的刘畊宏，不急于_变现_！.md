<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">全网新晋流量王——<a href="https://www.niaogebiji.com/tag/%E5%88%98%E7%95%8A%E5%AE%8F" target="_blank">刘畊宏</a>，终于开始接广告了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1402637294629346b671d280.37933676.png" width="600" title="爆火的刘畊宏，不急于“变现”！" alt="爆火的刘畊宏，不急于“变现”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">近日，薇诺娜特护霜、Ulike脱毛仪等美妆产品出现在刘畊宏抖音直播间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">FILA花费1700万成功拿下了刘畊宏的首次短视频和直播的软植入。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在尝试过运动品牌的软植入之后，刘畊宏第一个官宣代言人的商务合作品牌看起来有点“跨界”——九阳豆浆。5月8日，刘畊宏在微博宣布，成为九阳豆浆品牌代言人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这波主推产品为九阳豆浆全新推出的创新升级产品——磨豆匠豆乳，备受运动健身、食品安全各界人士的关注。九阳豆浆本身自带流量，现在有刘畊宏代言加持，再一次掀起植物蛋白饮料的新高潮。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">九阳豆浆官宣刘畊宏为品牌代言人</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">官宣首个代言之前，网上一张关于刘畊宏团队的报价图便广泛流传。其中提到，目前刘畊宏已经全面开启商业化，以资源包形式打包合作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/329485381629346b6ebf944.66710099.png" width="400" title="爆火的刘畊宏，不急于“变现”！" alt="爆火的刘畊宏，不急于“变现”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">冠名费用分为500万元和300万元两个档位，软植入则包括200万元和100万元两档，此外，随着他的身价水涨船高，“一天一个价”的说法也层出不穷。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实上，这个报价也并非空穴来风，一位MCN公司的业内人士透露，刘畊宏现在的身价暴涨，商业赞助和代言费用或均超过百万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">流量<a href="https://www.niaogebiji.com/tag/%E5%8F%98%E7%8E%B0" target="_blank">变现</a>的能力，一方面体现在对外的合作报价上，另一方面体现在从流量到销量的转化能力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/329399629346b7419ac4.82499205.jpeg" width="500" title="爆火的刘畊宏，不急于“变现”！" alt="爆火的刘畊宏，不急于“变现”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果说FILA（斐乐）相当于买了个广告位，那么九阳豆浆是直接官宣代言人！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">九阳豆浆相关负责人说，刘教练对品牌的吸金能力不容小觑，在销量上无疑是一个体现。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“感谢榜一大哥——九阳豆浆”“感谢‘金主爸爸’替我们交课时费”……刘畊宏自然地与品牌互动，粉丝们并非觉得突兀，反而冲到电商平台上开始疯狂“下单”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在5月8日官宣刘畊宏代言九阳豆浆当天，天猫上九阳豆浆旗舰店直接爆仓卖断货。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">店内公告宣布：“近期订单量剧增致使仓库存在延迟发货的现象。5月8日的所有订单预计在5月30日前发完。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用数据说话，九阳豆浆在淘宝已经完成了月销2万+的成绩。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/48845300629346b7c5ff92.51418352.jpeg" width="400" title="爆火的刘畊宏，不急于“变现”！" alt="爆火的刘畊宏，不急于“变现”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">店铺首页更是连挂三条说明，感谢网友对教练和九阳豆浆的支持。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了主代言产品外，“刘畊宏女孩”还争相使用教练同款胖胖杯，使得胖胖杯也卖断货，确实有“人传人”的效应。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不得不说，九阳豆浆通过与刘畊宏的这次互动赚了一波好感度，对刘畊宏而言，展现了其超强的商业价值。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">不急于“榨干”IP价值</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在抖音，一些出圈视频的评论区，经常会看到这样的评论：直说吧，什么时候开始<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不可否认，一些明星网红带货后，口碑确实有所下滑，但刘畊宏似乎以一己之力扭转了这种局势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">与九阳豆浆的这次合作，更像是刘畊宏与背后无忧传媒对市场的一次探究。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">的确，论目前的热度、粉丝数以及商业价值上看，若要直播带货，刘畊宏必定能获得更可观的数额，但无忧传媒CEO雷彬艺在近期的采访中曾对媒体表示，不考虑让刘畊宏直播带货。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就目前的情况来看，公开的合作主要还是在他爆火之前就已签约的项目，并不算他收获这波巨大流量后的变现举动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据九阳豆浆相关负责人透露，双方的代言合作早在去年年底就已签约，并没有无忧传媒的介入，此次代言的主要新品算是双方的创新产品，是与新产品的研发进度同步。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">收获这波流量后，无忧传媒没有立马布局“直播带货”，一方面是因为对刘畊宏直播带货持更谨慎的态度；另一方面，除了直播带货，在其他变现模式如“健身领域”还有“知识付费”等，刘畊宏具备更多可能性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">还有一点，目前健身类商品相比美妆、居家日常等大品类，其SKU（库存量单位）不算丰富，大品牌、高客单的产品不算多，还要考虑供应链及品牌货源等因素。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过，刘畊宏虽然只是一个健身博主，但他更代表一种健康潮流的生活方式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了运动服饰、健康食品等垂直细分品类，不少美妆等品牌通过“刘畊宏女孩看过来”的噱头，成功地吸引了一波消费者，展现出一定的时尚引领属性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">诚然，不管做健身领域还是要拓宽品类，刘畊宏团队首要保证的是，其健身热度持续跟得上。从目前阶段来看，确实急不得。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首先，从“健身兴趣”到“健身习惯”来看，需要花费更长的时间去培养。用户的线上健身习惯仍具备不确定性。只有维持好用户的留存度和黏性，“变现”才能发挥出更好的效果。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二，刘畊宏还处于口碑上升期，需要团队更谨慎地平衡好“商业”和“用户体验”。刘畊宏是靠直播火起来的，粉丝属性与网红更接近。这种粉丝倾向“泛流量”，在广告等变现上，刘畊宏更加照顾的还是大本营里的“用户体验”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外，树立“国民教练，人设不倒”的印象。得益于这一人设，“刘畊宏男孩女孩们”对他的好感度、信任度也是极高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">综上所述，对于刘畊宏及其背后的团队而言，真的不用急于收割流量。培养好粉丝，运营好内容，多一点“长期主义”，未来变现的想象空间反而更大。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">远不止于“直播带货”</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如何把握内容与直播变现之间的平衡，实现流量与收益的双赢，刘畊宏正在淌过一条从未有人淌过的河。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/428485152629346b9167123.75224764.jpeg" width="400" title="爆火的刘畊宏，不急于“变现”！" alt="爆火的刘畊宏，不急于“变现”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">网上还有有关“刘畊宏再不开始变现就晚了”的声音悄悄出现，认为刘畊宏应该趁着热度还在，尽快把流量变成金钱，落袋为安。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果刘畊宏真的这样做，短期内确实可以收获一波利益，但也是在自毁长城。刘畊宏承担了颗粒无收的风险，背后是为了实现更加长久的热度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么不直播带货，刘畊宏的巨大流量还能导向哪里？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实上，除了目前有所实践的代言和软植入，对比一些“前浪”主播的商业化路径，就能发现刘畊宏的变现模式还有更多的可能性，尤其是在这股全民健身的热潮下，刘畊宏或许能够走出更加丰富和可持续的道路。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">健身界的鼻祖帕梅拉在刘畊宏之前，一手打造了居家健身的“人传人”现象，走红之后她曾在PUMA赞助下推出联名系列，创立了自己的个人内衣品牌，签约Keep开展“知识付费”变现等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，拓展线下门店也是许多知名IP进行流量变现的一种模式。例如在《奇葩说》中走红的姜思达，凭借前期积累的个人IP流量和自身强烈的个性化属性，创建了花艺品牌“dresscode”，并开设线下门店。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">代言、开展品牌联名、推出个人品牌、拓展实体店……刘畊宏与他们的相似之处在于知名度和积累的巨大流量，不同之处在于他还有着乐人、演员、明星奶爸等多个标签。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">正因如此，未来可去实践的商业化模式也会更多，毕竟在这波居家健身潮中收获的流量可以为他的任意一个身份赋能。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当前，无论对线上还是线下的健身平台而言，专业的内容都是基础，而其短板也出现在难以突破固有圈层，正因如此，Keep不惜9个月烧掉8亿推广费用于拉新。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而刘畊宏兼具此前长期做“明星健身教练”的专业能力、带动全民“毽子操”的交互破圈能力、运营线下健身房的丰富经验，若与运动、健康食品等品牌联手或推出自有品牌，打通“健身教学-运动产品-健康食品-线下健身房”的全链路也不无可能。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">无论如何，刘畊宏需要做的，就是保护好自己的长期价值。内容是刘畊宏最大的法宝，人们从来不吝啬于为了真正有价值的事物而买单。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有许多刘畊宏的粉丝表示，会支持刘畊宏的代言，就当做是为上私教课而交费了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些评论充分说明，只要刘畊宏能够持续输出有价值的健身内容，就会持续获得巨大的流量，在流量逐渐到顶时选择开启商业化并没有错，重点是要爱惜羽毛，才会赢得更多的路人缘。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而以刘畊宏当前的流量、内容创造力与口碑，蜕变成媒体中的“李佳琦”都不会成为他的终点。我们还纠结刘畊宏何时直播带货，未免，“格局”有点小了。</p>
