<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“去年找实习我一个大厂都没投，目标全是中厂。”小杰是国内头部985大学的一名大四在读生，在新浪实习并有机会留下时，却转投了某社区问答平台。小杰告诉燃财经，现在很后悔当初选择实习机会时，因为太迷恋大厂，而放弃当红互联网中厂企业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我去小红书实习后，很多朋友也开始考虑去中厂实习。毕竟在垂直领域，能够得到更好地锻炼自己的机会。”曾在小红书实习、如今已入职B站的张张如是说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经了解到，近两年，像小杰和张张一样、放弃大厂转而涌向中厂的<a href="https://www.niaogebiji.com/tag/00%E5%90%8E" target="_blank">00后</a>越来越多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>平台较小，能提供的校招、实习岗位较少，是很多应届生对中厂的固有印象，这也使得中厂在人才市场上一直处于较为尴尬的位置。不少应届生将中厂当做“大厂备胎”，“先拿中厂offer做保底，在慢慢等待大厂offer”也早已成为“常规”操作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，随着大厂的缩招以及过度的“内卷”，高薪资难以遮盖“996”带来的隐患，频繁的裁员信息更让“求稳”的00后忐忑不安……面对大厂越来越高的就业门槛，那些曾经让年轻求职者心向往之的高薪、大平台，慢慢失去诱惑力，越来越多的00后开始对大厂失去兴趣，并寄希望于中厂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>智联招聘数据显示，在2022届毕业生中，最期望就业的企业规模是中型企业的人占比达45%。除此之外，34.4%的应届生期望去小型企业工作。而期望去大型企业的应届生仅占16.9%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“去中厂，做不被定义的打工人”开始在00后群体中不胫而走。“大厂虽然薪资高，但培养的都是‘螺丝钉’。我还是想做有创造性的工作，实现自己的价值。”小杰说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着越来越多应届生的涌入，一方面，中厂也在不断完善人才培养机制，效仿大厂引进Mentor制度，丰富员工福利。在小红书上搜索各中厂，跃入眼帘的尽是“节日礼盒”、“包两餐”、“免费下午茶”等内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但另一方面，尤其是垂直赛道的中厂，竞争态势也变得异常激烈。“今年来应聘实习生的，不少是985高校应届生或者海外留学生。大家都非常优秀，一个岗位经常需要3轮面试才能确定候选人。”某中厂资深Mentor阿政说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如阿政所说，在愈发严峻的就业背景下，“进中厂”也并非是一条坦途。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“想进中厂的应届生明显变多，但中厂业务规模相比大厂来说还是小了很多。通常一个部门只有1-2个实习岗，这样一来，有的热门岗位的录取比例一度高达500:1。”阿政直言，现在很多中厂的录取要求已经直逼大厂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有着丰富中厂实习经验的张张表示，并不推荐应届生盲目投递中厂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“就业本身需要考虑自身条件、个人能力，以及和公司的匹配程度。无论是去大厂还是中厂，本质上是要找一条有潜力的、自己感兴趣的赛道。正确地看待大厂和中厂，才是打工人的核心素养。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/1455856016142b5abe0ca06.17059714.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>应届生涌向中厂</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大厂已不再是00后就业的唯一选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我大四实习的时候投递的几乎都是中厂。前期忙于科研的小杰并无丰厚的履历，虽然很想进大厂的产品实习岗，但深知“投递竞争激烈的大厂就是浪费时间”的他，还是把目光更多的锁定在了中厂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>果然，在简历投出后不久，某内容社区中厂就和小杰约了面试时间，并在面试结束的当晚，给小杰发放了入职offer。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，小杰在面试该中厂的同时，还收到了字节跳动的面试通知。可在寒假前一面结束后，小杰迟迟没有收到后续通知。直到年后，负责岗位招聘的HR告诉小杰，由于内部人员调动原岗位已无需求，但小杰可以考虑申请其它岗位重新面试。“可投递大厂的人太多了，几乎所有面试都要3轮或者以上。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就这样，小杰选择了该内容社区平台的offer。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和小杰相似，年前曾进入某大厂线上项目实习的风风，现在是某985高校的研三学生。但仅仅实习一周后，风风就被“内卷”的实习氛围所“劝退”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“在项目实习群里，一个需求发出，就有数百个实习生‘抢单’，抢到的实习生才有参与项目的机会。”不仅如此，风风直言，本就已经很“卷”的“赛马”机制，还经常会延续到深夜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>难以适应大厂赛马机制的风风了解到，该大厂正式员工的工作强度也早已是“日常996，忙时007”。随后，风风重新考虑了自己的职业规划，“当我问自己，对于这种只有工作没有生活的日子能坚持多久时？对大厂的执着开始瓦解。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我不想为了高薪资放弃自己的生活，更何况很多中厂薪资也在可接受范围之内。”于是，风风在校招时拒绝了大厂offer，最终选择了某游戏中厂。“当时成都正好有工作机会，薪资也和一线城市同岗位持平，我便毫不犹豫选择了该中厂。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互联网大厂有着高薪资和大平台，同时也伴有高强度的工作压力。但对于00后应届生而言，工作的压力不是最重要的，是否能实现自身价值，才是影响择业选择的关键因素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/63277437462d22c4640c6e9.10064503.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/视觉中国</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“大厂各业务早已发展成熟，很多岗位其实更像流水线作业，大部分人都是‘螺丝钉’，你很难在有限的空间里创造出更大的价值。”张张说道，她一开始就坚定地选择进入较为小众、处于上升期的互联网赛道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我在小红书有着8个月的实习经历，通常情况下，6个月以上的实习期刚好能让我跟完一个完整的项目。相比起去大厂做螺丝钉镀金，我更偏向选择有成长空间的工作。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>某大厂资深HR表示，企业层级多，便会出现下传多，上达少的情况。基层员工的存在感和归属感随之降低。“这不仅会直接造成基层员工离职率普遍较高，也会随着岗位的分工细化，让不少人沦为‘螺丝钉’。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据美世《中国整体薪酬调研》数据，2019年上半年，国内平均企业员工主动离职率为7.1%，而互联网行业员工主动离职率高达13.3%，接近平均值的两倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上述HR补充道，大厂的岗位细分远远多于中厂。“以营销部门为例，大厂往往会根据抖音、快手、豆瓣、微博等不同平台设置相应的运营岗位。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“岗位工作内容的细化确实能让更多应届生有机会接触各类的一线资源。但很多大平台能提供的机会和资源只是一个概念，很难真正落到每个人的身上。”与此同时，张张直言，长期从事重复性较强的工作何谈成长？要想长远一些，为自己争取更多的成长空间才是关键所在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/9082723746142b5ac73c549.43567204.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>00后在中厂，不做“螺丝钉”</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong> </strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，即便对大厂的竞争和压力早已心知肚明，但面对金光闪闪的大厂时，小杰还是出现过动摇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年10月中旬，在新浪与小红书之间，小杰选择了新浪的产品经理实习岗位。但到岗后，小杰发现自己的大部分时间都在重复的机械性工作内容，“我70%左右的工作内容都是查数据和分析数据，很少时间做创作型的工作。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而由于Mentor工作繁忙，到岗第一天，小杰就在无指导的情况下开始接触完全不熟悉的业务。就这样，2个月实习期内，小杰只能与其他实习生一起“相互取暖”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新浪实习结束后，小杰选择在国内某内容社区中厂开始下一段实习生活。出乎意料的是，该中厂不仅为员工每日提供两餐，还配备Mac笔记本和显示器，“没想到中厂的福利竟然这么‘香’。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/132798936562d22c46742b70.80926793.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/视觉中国</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅如此，在入职的第二天，小杰就开始在leader的带领下熟悉工作内容和常用软件，并快速适应了新的工作环境。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，除了分析数据的日常工作，小杰感觉到leader在争取让部门中的每个人发挥自己的创新性。“我经常会以用户视角协同组内成员做产品优化，提出的意见也经常被leader采纳。”能够切实参与到产品功能调整过程中，让小杰备受鼓舞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与小杰不同，张张一开始就毫不犹豫地选择了正在快速发展期的小红书，并在实习期就迅速融入了团队。“leader会带我不断复盘，引导我学习新知识。同时，比起大厂，小红书的员工数量精简，也有更多独立负责业务的机会。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经了解到，实习期间，张张和leader一起从0到1运营起了官方账号。在众多旅行笔记中，张张敏锐地发现了露营这一极具潜力的关键词，更主导了一系列的相关活动。活动中，她不仅认识了很多出行类达人，也迅速成长成为了一名合格的职场人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也正是由于这段经历，在随后的校招中，张张3天内拿下了阿里校招的offer，并成功通过了美团的终面。不过最后，张张毅然选择了B站，“我已经爱上了不断开拓新业务的过程，也享受在团队中成为不可替代的一部分。这种满足感在大厂可能很难寻觅到吧。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/4756644006142b5ad08cc02.64114870.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>大厂不香，中厂如何？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对大厂去魅的年轻人，以及00后对个人价值认同的向往，让中厂在求职市场愈发瞩目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>被愈加关注后，中厂也纷纷“卷”了起来。阿政表示，当前不少规模在500人以上的中厂都在逐步建立完善的人员培养制度。从新人入职到适应期引导，都有一套完整的流程。“中厂一般一个部门最多2个实习生，Mentor能做到手把手带教。同时，中厂的新业务也能让新人迅速进入‘角色’，在独挑大梁的过程中挖掘个人能力，带动新人快速的成长。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，阿政强调，不少中厂的员工福利早已与大厂看齐，职场氛围也营造的很好。除了包两餐，日常下午茶、每月团建、各种公司周边等“基础配套设施”之外，中厂也愿意营造节日氛围，员工大礼包更是常备……而这些隐性福利也不断吸引着00后。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/169985290962d22c46938d95.30508075.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/视觉中国</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，尽管多方面因素来看，中厂吸引着更多的年轻人，但需要提醒的是，中厂也并非是应届生就业的乌托邦。“当前应届生的就业形势并不乐观。”上述资深HR透露，去中厂并不是一个更轻松的选择，现在不少中厂的招人标准也已经和大厂持平。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该HR强调，一方面是今年互联网行业整体缩招，应届生“一岗难求”。《高校毕业生就业报告》显示，2022年高校毕业生规模达到1076万，而2022年一季度高校毕业生CIER（</span><span>就业景气指数指数</span><span>）却已降至2020年疫情爆发以来最低点，仅有0.71，较去年同时期的的1.34，下降了47%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，中厂公司规模较小，每个部门配备的实习生数量和新岗位招聘数额有限，“优中选优”则成为必然。去年校招进入B站的张张向燃财经透露，入职后才得知，自己所在岗位的录取比率达到了惊人的3000：1。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿政表示，早前，其所在的中厂单个岗位面试最多2轮便会走完流程。但现在，一个开放给大一、大二学生的日常实习岗都会组织3轮面试。“原因很简单，投递的简历太多了，求职岗位的需求爆火无形之中抬高了中厂的入职门槛。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿政透露，今年一个原计划招收0经验大学生的岗位，收到的简历中不少来自头部985高校。“单从简历来看，这些求职者的自身能力都没问题，且实习履历也都很丰富。但‘优中选优’，最后我们还是选择了有3段大厂相关实习经历的留学生。毕竟，谁不想招收有经验、有能力的人进入团队呢？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着在工作中的不可替代性成为00后应届生职场安全感的来源，中厂的持续发展，为更多00后提供了更多的选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但正如张张所说，“人生是一场长跑，无论是大厂还是中厂，亦或是小厂，都不过是人生中的某一节点。选择大厂或中厂都不重要，想好自己的职业规划，根据自己的能力和需求找到相匹配的公司才更重要。”</span></p>
