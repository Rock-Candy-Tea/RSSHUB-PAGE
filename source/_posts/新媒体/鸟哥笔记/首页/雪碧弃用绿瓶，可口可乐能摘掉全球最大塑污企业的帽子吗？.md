<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提起<a href="https://www.niaogebiji.com/tag/%E9%9B%AA%E7%A2%A7" target="_blank">雪碧</a>，那一抹绿色的瓶身，几乎是大众几十年来对它的固有印象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而近日，雪碧的绿色却可能成为绝唱。可口<a href="https://www.niaogebiji.com/tag/%E5%8F%AF%E4%B9%90" target="_blank">可乐</a>公司宣布，雪碧将从8月1日起放弃使用超60年的绿瓶，包装使用更加环保的透明容器。原因在于，雪碧包装的塑料中含有绿色聚对苯二甲酸乙二醇酯（PET）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可口可乐公司表示，虽然绿色PET是可回收的，但回收的材料更大概率被转化为一次性物品，如衣服和地毯，不能回收成新的PET瓶。在分拣过程中，需要将绿色和其他颜色的PET与透明材料分开，以避免制作新PET瓶所需的可回收食品级包装变色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1462036362e992e6b36e23.63824304.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，雪碧也不是完全告别绿色。可口可乐公司表示，届时雪碧瓶子上也将出现新标志和包装设计，旨在“在世界各地提供更一致的外观和声音”，绿色色调仍将在雪碧标签上使用。为了在透明塑料瓶上显示更亮眼，绿色色调也将变得更浅。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/100195385462e992e70e3294.45641289.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>（左边为旧标志，右边为新标志）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>几个月前，雪碧也开始在全球范围内开启“Heat Happens”营销战役，而在中文语境下，“燥不可挡，酷爽以对”（“Heat Happens, Stay Cool”）的品牌主张也开始面向中国消费者露出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/149704529662e992e74b7788.54203191.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/42833521362e992e79648b1.12808665.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/82111430462e992e7e23000.27268100.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/113488149762e992e82e8f60.51095806.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/100084138762e992e87815f9.09563144.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/126743300262e992e8c08d03.99225458.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>雪碧停用绿瓶的消息一经传出，便立马登上了热搜。网友们在听到这个消息后，纷纷表示“失去了灵魂”“失去了青春”……甚至有人表示赶紧买两瓶，回家珍藏起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>网友们有这种强烈反应其实一点也不奇怪，作为知名大众消费品，绿瓶早已成为了雪碧品牌形象的一部分，并沉淀为品牌资产。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在美国市场营销协会的定义中，品牌本身就是一个名称，专有名词，标记，符号，或设计，或上述元素的组合，用于识别卖方或销售集团的商品和服务，并将其与竞争者商品和服务区分开来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际中，这些元素通常包括以下：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌名称应当是清晰、易懂、有明确的发音和意义的符号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌标识是指有文字的识别，如文字商标、英文、拼音标识；与之对应没有文字标识称为符号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌形象通常运用到广告和包装设计中，可以是某种动画人物，也可以是真实的人物，也可以是代言人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌口号包括对品牌或产品提炼出简练有说服性的短语，运用到广告及可以形成听觉传播的媒介中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌包装包括融合名称、口号、形象、符号等品牌关键信息于一体的产品包裹媒介，起到识别品牌、传递信息、保护产品的作用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，品牌元素还包括门店陈列、企业名片、文化衫、广告文案、网店形象、企业邮局、域名等等一系列能展示品牌信息的介质。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>由此不难看出，雪碧的绿色瓶身包装本身就是其特定名称、经典口号、广告宣传形象的组成部分，不仅是一个包装，还可以作为品牌元素延伸至周边，是雪碧这个品牌本身的一部分。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/181704580162e992e95f3443.98127542.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>从1961年，可口可乐公司首次推出雪碧饮料，其绿瓶包装至今已有60年历史，</span>而且，按照营销大师戴维•阿克的品牌资产“五星”概念模型，绿色包装也是雪碧品牌资产的重要组成部分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>依据品牌资产“五星”概念模型，戴维•阿克认为品牌资产由品牌知名度（Brand Awareness）、品牌认知度（Perceived Brand Quality）、品牌联想度（Brand Association）、品牌忠诚度（Brand Loyalty）以及其它品牌专有资产组成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其中，品牌认知度和品牌联想度都和产品包装有关。而且，经过几十年的营销宣传，雪碧早已在消费者心智中建立了绿瓶的形象，几乎成为了绿瓶饮料的代名词，绿瓶形象也影响着品牌知名度和忠诚度。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此次弃用绿瓶，可以说下了很大决心。其实，雪碧弃用绿瓶早已有预见。雪碧近年来推出的许多新产品，都没有用经典的绿瓶包装。近年来，雪碧透明瓶在国内市场中也不断试水，无糖雪碧、雪碧纤维+等新款产品的包装上，都选用了透明塑料瓶，而不是经典的绿瓶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/127149009662e992ea72cdc9.62736628.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实在瓶身上做出改变的不仅是雪碧，而是整个可口可乐公司。一个例子就是，同属于可口可乐公司旗下其他使用绿瓶的饮料，如 Fresca、Mello Yello 和 Seagram's等，也将在未来几个月内改用透明瓶子，身体力行支持环保。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中的原因也很明确，就是整个可口可乐公司受到的环保压力越来越大了。根据国际减塑环保运动Break Free From Plastic 2021年发布的《品牌审查报告•第四版：让企业对塑料污染与气候危机负责》报告显示，<strong>像可口可乐、雀巢和百事可乐这样的大型快消公司仍在持续推动石化行业扩大塑料生产。某种程度上说，这些造成塑料污染的企业也是气候危机的幕后推手。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>BFFP在对六个大洲440家企业进行品牌审查分析后发现，<strong>2021年造成塑料污染最为严重的十大企业分别是：可口可乐、百事、联合利华、雀巢、宝洁、亿滋国际、菲利普•莫里斯、达能、玛氏和高露洁。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/46779092162e992ec16a3d5.66630710.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，可口可乐公司已是连续第四年蝉联榜首。根据BFFP的报告，在贴了商标的塑料垃圾中，有10%都属于可口可乐公司。可口可乐在2020年使用了近300万吨的塑料包装，这些塑料制品在生命周期中将会排放大约1500万吨二氧化碳——相当于三百多万辆汽车行驶一年的二氧化碳排放量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而随着消费者对绿色环保的认知进一步加深，绿色消费意识不断觉醒和提升，可口可乐、百事可乐等公司受到的环保压力自然越来越大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>而在宣布雪碧弃用绿瓶的同时，可口可乐公司还宣布，在美国和加拿大销售的大部分达萨尼瓶将100%使用回收塑料。</span><span>可口可乐公司表示，与2019年相比，这一创新预计将减少约2000万磅新的塑料垃圾。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但换掉绿瓶，可口可乐就真的环保了吗？就能摘掉全球塑污头牌的帽子吗？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Break Free From Plastic全球企业活动协调员艾玛•普里斯特兰(Emma Priestland)的一段话或许最能代表实际情况：</span><span>“世界上最大的污染企业声称自己正在努力解决塑料污染问题，但它们却在继续生产有害的一次性塑料包装。</span><span>”</span></p>
