<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你永远无法想象一位中年大叔的身姿能有多灵活？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">他不仅会扫堂腿。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">还能扎马步。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">甚至还领着其他中年大叔，排好队形，站在C位冲你比心。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/204306288662ab5631c166a8.03561656.gif" title="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" alt="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这个队形有“影流之主”内味了</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你以为大叔们在整活，其实他们是“云爸爸品牌男装”（简称“云爸爸”）抖音直播间的男模，扫堂腿是证明裤子足够服贴，扎马步是检验裤裆轻易不崩线，比心是感谢直播间观众的点赞下单。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">模特大叔们这般身体力行的敬业精神，让直播间观众产生深深的代入感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/151691249562ab5640c0fdd9.93487017.jpeg" title="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" alt="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不只在抖音直播间释放魅力，模特大叔们还一路火到了B站。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">截至目前，某场直播剪辑的播放量最高已经达到了216.7万，弹幕除了一水的“哈哈哈”，还有“内娱男团下岗再就业”“请选择你的英雄”等玩笑调侃。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/171764645562ab5640e4dd68.86899403.jpeg" title="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" alt="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">网友们看得开心，“云爸爸”直播间销售额也一路见涨。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">新榜旗下抖音数据工具<a data-miniprogram-appid="wx55778a0d7b3048c4" data-miniprogram-path="pages/NewLive/index" data-miniprogram-nickname="新抖数据" data-miniprogram-type="text" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined">新抖</a>显示，近30天“云爸爸”直播38场，累计销售额达1278.7万元，相比同期增长近3倍，累计观看人次达4724.1万，相比同期增长2.2倍。</p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">一年直播380场</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“云爸爸们”有多卖力？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">回到故事的原点，“云爸爸”的模特大叔是有一些拘谨在身上的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年5月2日，“云爸爸”发布了第一条抖音视频。视频中，模特大叔不苟言笑，迈着生疏的步伐走进画面，简简单单做了几个pose，完成了首次亮相，动作还不及现在的生动活泼。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/141531866362ab564220d065.37801107.gif" title="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" alt="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了门店内景，模特大叔时不时出个外景，一会持扇眺望远方，一会拿上手机佯装接个电话。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/160870746162ab564f90b157.22024277.jpeg" title="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" alt="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">连续更新了1个多月的短视频后，“云爸爸”在2021年6月16日开启了首场带货直播，此后，该账号以日播节奏，在每天早8点或晚8点固定直播，每场直播持续4个小时左右。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">试水抖音前，“云爸爸”没有运营过淘系店铺，这意味着它没有站外粉丝基础，只能从抖音完成品牌的线上冷启动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">也因此，“云爸爸”直播间的前期销售情况并不理想。据新抖数据，2021年8月，“云爸爸”直播间单场最低销售额为136元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">转变发生于2021年9月，“云爸爸”直播间单月销售额达201.47万元，是上个月的两倍多。在随后的三个月中，“云爸爸”直播间单月销售额均超过400万元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">由于抖音暂未提供直播回放功能，我们尝试从“云爸爸”同时期发布的短视频中寻找原因。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">10月20日，“云爸爸”更新的视频中，模特大叔加入了手舞足蹈的展示动作，以及更加丰富的面部表情，单条视频点赞量刷新了过往纪录，达到1万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/196557205362ab5652cf3ff4.14540922.gif" title="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" alt="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种大幅度的肢体语言，或许帮助“云爸爸”在短视频和直播间都找到了流量密码。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">今年春节之后，“云爸爸”直播间升级了“人海战术”，出镜的模特大叔数量从一位，逐渐增加到三位，再到五位。每位模特大叔展示服装的时间控制在30秒到1分钟，一位展示完换一位，如此循环往复。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/49412611662ab565fd1c563.08667802.jpeg" title="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" alt="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">人多了，C位大哥底气都足了</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">多成员的出现，为直播间观众提供了选择的余地，大家也各自pick出不同的“心仪叔叔”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/48006056762ab56600cff13.98423263.jpeg" title="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" alt="单月直播卖货1200万，尬舞的中年大叔有什么魔力？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">由于经常站在C位的这位模特大叔动作太过卖力，被网友调侃为“印中天”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同于其他直播间主播逐一介绍商品细节，“云爸爸”直播间的模特大叔只做动作不出声，相当于一个真人化的展示人台，商品介绍由一位女生以画外音的形式完成。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">直播中，“云爸爸”的模特大叔经常会加入下蹲，踢腿等幅度大的动作。这不仅是营造喜剧效果，更是直观证明服装质量。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当中年大叔走进直播间，意外出现了“云爸爸”这样奇妙的化学反应。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">与此同时，像“云爸爸”这样，将叔叔阿姨作为中老年服装模特的抖音直播间不在少数。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以“爸爸装”“妈妈装”为关键词进行搜索，将发现大量的中老年男女装直播间。这些账号情况与“云爸爸”十分相似，粉丝量几十万到一百多万，直播间在线人数也徘徊于1000-2000人左右。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当直播间出现一个真实的爸爸/妈妈形象，这将促使年轻人产生消费欲望——“好久没回家，给爸爸/妈妈买件衣服，让他们也乐呵乐呵。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">相比于吸引中老年消费者来直播间掏钱，“云爸爸们”的策略是让他们的孩子为之买单。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
