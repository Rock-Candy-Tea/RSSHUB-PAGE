<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>站的越高，摔的也就越狠。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在大热的<a href="https://www.niaogebiji.com/tag/%E6%96%B0%E4%B8%9C%E6%96%B9" target="_blank">新东方</a>凭借<a href="https://www.niaogebiji.com/tag/%E5%8F%8C%E8%AF%AD%E7%9B%B4%E6%92%AD" target="_blank">双语直播</a>爆火出圈时，却迎来了多方机构的集中减持。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，港交所披露数据显示，腾讯控股于6月15日、16日先后出售新东方在线3561万股、3899万股，累计套现约7.2亿港元。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此般“清仓式”减持下，持有新东方在线6年的腾讯正式从第二大股东降为第三大股东，收益率约一倍。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很快，这一新闻迅速冲上了微博热搜，网友们也大多表示“有序的退出教培，很认可”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/101740243362b4c9331c0db6.53912352.png" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，随着新东方股价的大涨，不仅腾讯，多家境外投资者也选择在高位减持。如摩根士丹利在6月15日至16日，先后减持共7200万股；汇丰银行、摩根大通、花旗银行等都纷纷跟进。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>短短时间，爆火的新东方仿佛变成了资本的弃子。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而真相并非如此，当外资大幅套现时，南向资金持仓量显著增加。近7日数据显示，新东方在线跃居南向资金净买入第3位，合计净买入13.96亿港元。中资机构也对新东方在线有不同程度加仓。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，随着新东方在线股价的反复，投资者正逐渐趋于理性。经历了股价爆炸式增长的新东方，也正逐步回落至一个合适的位置。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这一切的源头，还要从一个小小的直播间说起。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>东方甄选意外爆火</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年，在“双减”政策下，作为教育业龙头的新东方迎来生死劫。一时间，股价崩盘，市值跌去9成。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>思虑良久的“总舵手”俞敏洪为挽救这艘巨轮，积极寻求转型，将目光望向了火热的“直播带货”。他对这一商业模式的期望很高，感叹称直播电商是第三次商业革命。更是直言，“薇娅一年能卖一百多个亿，我带着几十个老师做直播，是不是一年也能做上百亿？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/212875629462b4c93363b903.40338114.png" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>然而理想很丰满，现实很骨感。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为新东方旗下的农产品直播带货平台，东方甄选除了在首播当晚依靠俞敏洪个人声量拿下了近500万的销售额，之后的时间里业绩持续惨淡，直播间最冷清时，观看人数甚至只有个位数。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一切，直到半年后的今天才迎来转机。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当下，直播届四大天王纷纷退居幕后。新东方的老员工罗永浩已经还完债务，从直播带货的江湖里抽身离去，而俞敏洪正一头扎进深潭。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两条命运的曲线在此短暂交汇，而又迅速分道扬镳，这是两个理想主义者面对现实的交锋。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>头部主播的空缺，给了新东方伺机而上的机会。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，在6月9日，一颗名为“双语带货”的炸弹被引爆，东方甄选霎时成为媒体的宠儿，热度一时无两。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从0到100万粉丝，东方甄选用了半年；从100万到1000万粉丝，却只用了一周。伴随着粉丝量的上涨，直播间销售额也开始飞速上升。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，截止6月19日，东方甄选在30天内共直播34场，销售额高达4.28亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/86148264762b4c9339a9229.34160938.jpeg" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播间的火爆映射到二级市场，便造就了新东方在线连涨9日，累计最高涨幅超800%的神话。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>吃瓜群众们热情高涨，投资者疯狂买入，在社交平台的助推下更演变出“兵马俑董老师”、“文化人直播间”等热梗。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>新东方，爆了！</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>带货翻车，新东方面临品控难关</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓人红是非多，随着东方甄选的爆火，争议也随之而来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在日前，有网友爆料称，他在东方甄选直播间下单购买了4.5-5斤装的陕西水蜜桃，然而签收后才发现，已经有约四分之一的桃子腐烂长毛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/45844547462b4c933c7fac0.63552524.jpeg" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该网友出示的物流记录显示，6月10日下午18时37分，该包裹由中通快递揽收，之后历经西安中转中心、京南转运中心等站点后，于6月13日13时14分，由快递驿站代收。两天后的13时43分，该网友取走包裹，并发现水蜜桃变质的情况。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之后在第一时间，其便与东方甄选客服取得了联系，商家则在约半个小时的时间内完成了整个处理流程，对顾客进行了全额退款。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过即便如此，该网友依旧心存疑虑：桃子的损坏情况那么严重，东方甄选到底是如何“甄选”的呢？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，这并非是东方甄选第一次被曝出质量问题了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此前还有网友称在东方甄选直播间购买的油杏，刚收到货打开就发现有烂果。除此之外，所有果子重量均不超过20g，这与直播间宣称的单果21-26g相差甚远。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70494924362b4c933ee0f22.47535521.jpeg" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>诚然，对于生鲜、农产品而言，由于储存、物流等因素的不确定性，发生质量纠纷的概率本身就会略高一些。并且对于以上的质量问题，新东方都进行了全额退款处理，但对于主打高品质的东方甄选而言，这些纠纷事件也应为其敲响警钟。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播带货的核心是“货”，但并非只有货就够了。随着东方甄选的爆火，订单迎来激增，在短时间内，类似的产品质量问题或将越来越多。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为链接消费者和商家的枢纽、作为直接面向消费者推销的平台，从产品的挑选、运输、到售后等环节，新东方都应负起责任，才能在这个行业里走得长远。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而当这些纠纷事件真的发生时，如何达成最大程度挽救，给消费者一个满意的答案将是新东方需要重点考虑的事情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>直播电商走向规范化</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，不仅仅是新东方，在诸如罗永浩、李佳琦等头部主播崛起时，同样经历了质量问题不断的阵痛。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但最终，凭借着对消费者的负责和真诚，他们都走向了被用户拥趸的头部主播之路。一如当年罗永浩直播间的情人节鲜花事件。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205942178362b4c934319698.92562236.png" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年5月20号前夕，罗永浩团队联合“花点时间”推出了520鲜花礼盒。然而在520当天，本该手捧鲜花渡过一段浪漫时光的情侣们，收到的却是凋零腐败的一捧烂花。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一时间，罗永浩被推至风口浪尖。</span></strong><span>消费者们满腹牢骚，好好的情人节愣是被一捧烂花变成了惨戚戚的维权大会。更有网友表示，这是在老罗直播间购物的最差体验。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最终，罗永浩对每一位顾客投诉的信息都逐一转发并真诚致歉，同时与“花点时间”一起为所有消费者提供了“3倍赔款”的现金补偿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/97440713262b4c934936429.37722024.jpeg" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，即便拥有着极高的人气，但罗永浩还是用真诚征服了消费者。再后来，老罗直播间也被粉丝戏称为“理财直播间”，罗永浩本人则一直稳居抖音一哥。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在直播带货领域，当新的寡头崛起时，遭遇各种争议、产品问题几乎是必然的。</span></strong><span>但罗永浩、李佳琦等人已经证明了，真诚才是面对消费者的必杀技。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，随着直播界四大天王时代的落幕，可以预见的，直播电商正在进入下半场，走向规范化。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此节点崛起的新东方，更应加强管控，以维护东方甄选的长久“繁荣”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许也真是意识到了这一点，俞敏洪作出了一个重要的决定——决不允许收取坑位费。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直以来，坑位费都是直播电商行业墨守的规则，其在一定程度上保障了主播方开展一场直播的基础收益。但另一方面，天价坑位费也让无数厂商叫苦不迭，其弊端清晰明了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12763059662b4c934d8cf10.51471131.png" title="新东方一夜爆火后，果然出事了！" alt="新东方一夜爆火后，果然出事了！" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>俞敏洪表示，东方甄选决不允许收任何坑位费，否则会倒逼商家绑架平台，使得产品无论合格都放上去。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>往小了说，便是俞敏洪想要通过撤销坑位费，来为所带货产品的品质增加保障，进而更好的维护消费者权益，这也与新东方的“甄选”理念相合。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>往大了说，新东方更是要以身作则，用免除坑位费激励品质升级，用品质触动消费者，为直播电商制定新的规范。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>头部主播把持天下的时代已经过去了，直播间的规范化、人气主播的可复制性，才是新东方接下来要探索的目标。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>未来某一天，假如董宇辉过气了，新东方还可以有更多的“张宇辉”、“王宇辉”站出来之时，或许腾讯也会后悔如今对新东方股票的抛售。</span></p>
