<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作者：老鱼儿</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">跟着<a href="https://www.niaogebiji.com/tag/%E8%8B%B9%E6%9E%9C" target="_blank">苹果</a>有肉吃，这是智能手机产业链上很多企业的共识。远有富士康“傍大款”赚得盆满钵满，近有立讯精密“抱粗腿”迅速攒起两千亿市值等等成功案例，都在印证着苹果的“<a href="https://www.niaogebiji.com/tag/%E4%BB%B7%E5%80%BC%E8%BE%90%E5%B0%84" target="_blank">价值辐射</a>”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过近两年，<span><strong>苹果的“价值辐射”能力开始屡屡受到质疑和考验。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">苹果2022年全球开发者大会（WWDC）6月初在美国旧金山召开。该大会的主要目的是向苹果开发者们展示最新的软件和技术，也会推出少量硬件产品。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">受此消息刺激，多家果链企业股价在6月6日出现大幅度上涨。如，比亚迪电子股价上涨9.62%、立讯精密涨6.87%、舜宇光学科技涨4.46%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">谁料这样的帅不过三秒。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这届苹果开发者大会虽然硬件与软件齐发，貌似看点十足。苹果发布会的相关内容也一举占据微博热搜榜中的六席。<strong><span>但外界最期待的VR/AR头显产品并没有亮相。</span></strong>对于苹果未发布的原因，相关媒体解释称由于性能及散热相关问题，苹果头显的发布时间或将延至2023年。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/187090900762a8ebca333a39.44956498.jpeg" width="465" height="239" title="为什么受伤的总是“果链”" alt="为什么受伤的总是“果链”" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span>苹果AR产品再次“跳票”</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">果链股因此迅速翻脸转跌。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">过去，与苹果的合作让果链企业享尽红利，但如今红利的边际效应仿佛在不断变小，市场对相关利空极其敏感，股价随着各种真真假假的消息敏感波动。果链企业们赚到超额收益的难度越来越多，要冒的风险却越来越多。</p>

<section>
<section>拥有强大议价能力的苹果倾向于压缩供应商和代工企业的利润空间。</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这已经不是果链们的第一次阵痛了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从2020年开始，无休止和反复的疫情就开始反复折磨果链们。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，今年4月份，受疫情影响，江苏昆山市实行全域静默。这个消息引起了苹果产业链的剧烈反应。因为在这个并不算起眼的县级市里，布局着包含富士康、和硕、纬创和立讯精密等在内的苹果主要供应链伙伴的加工工厂。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">产业链资料显示，这些公司几乎包圆了苹果的iPhone组装业务，分机型来看，iPhone13被富士康以及和硕瓜分，占比分别为68%、32%；iPhone 13 Pro被富士康和立讯精密瓜分，占比分别为60%、40%；而价值量最高的iPhone 13 Pro Max则为富士康独供。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了疫情这样不可抗力的干扰之外，最让果链们感到切身之痛的，恐怕当属越来越多来自苹果的“不确定性”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为了每年实现自身的可观增长，<span><strong>拥有强大议价能力的苹果往往将利润留在自家后院，而倾向于压缩供应商和代工企业的利润空间。</strong></span>尤其近两年，在疫情导致的芯片、线材、五金等原材料价格上涨，这些成本压力往往由果链企业承担。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/191056052462a8ebca62c4d3.70573030.jpeg" width="460" height="306" title="为什么受伤的总是“果链”" alt="为什么受伤的总是“果链”" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>果链们的代工成本在不断增加中</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">WitDisplay首席分析师林芝曾在接受媒体采访时指出，“在原材料价格上涨、供应链危机背景下，苹果为了降低成本，进一步压缩供应商的利润空间，给其带来更大的成本压力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">典型如蓝思科技，2021年年报显示，苹果作为其最大的客户，销售金额从2020年的203亿元上升至2021年的301亿元，增长了48.28%。但收入增加却并没有完全转化为利润增长。2021年度，蓝思科技实现营业收入452. 68亿元，同比增长22.55%，但净利润20.70亿元，同比减少了57.72%；扣非净利润更是暴跌72.45%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">出现同样情况的还有立讯精密。该公司第一大客户销售额从2020年638.43亿元暴增78.65%至2021年的1140.56亿元，这带动其2021年实现营业收入1539.46亿元，同比增长66.43%；但净利润为70.71亿元，同比下降2.14%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此前，富士康因为增收不增利的市场表现，曾公开回应称苹果压低代工价格，称“辛苦钱越来越难挣了”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据央广网报道，2021年在30家国内“果链”上市公司中，只有10家公司的净利润实现同比正增长。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，市场需求量萎缩带来的手机市场整体滑坡，也让苹果“砍单”不断。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据 IDC 发布的报告，2022年全球智能手机出货量将下降3.5%至 13.1亿部，这意味着今年的手机市场还会萎缩，可能要到2023年才会有所改观。国信通院数据显示，2022年1-4月中国手机销售量同比下跌30.3%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这样的大环境下，苹果并不能独善其身。根据彭博社报道，苹果将原定的2022年2.4 亿部的iPhone出货量目标，下调到2.2亿部。这意味着与 2021年度相比，今年的iPhone销量将出现“零增长”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/199301823062a8ebca9851f1.19344651.jpeg" width="529" height="397" title="为什么受伤的总是“果链”" alt="为什么受伤的总是“果链”" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>苹果手机的销量也同样受到环境冲击</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">今年一季度财报发布后，苹果CEO库克表示，苹果决定将今年iPhone产量削减15%。苹果的“瘦身”作用到下游果链企业们当中，会被更加放大。</p>

<p style="text-align: center;" data-mid><span><strong><span>深度捆绑</span></strong></span></p>
</section>

<section>
<section>果链们本想站在巨人的肩膀上，却一不小心成为了巨人的“输血袋”。</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过没办法，因为果链们太依赖苹果了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如蓝思科技近五年来，公司主要销售客户第一名的销售占比分别为49.37%、46.97%、43.07%、57.97%、66.49%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一大客户是谁？前几年蓝思科技还没有指名道姓。在2021年的年报里，蓝思科技干脆坦白了，第一名就是苹果。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/143998574862a8ebcabc5451.48823063.png" width="483" height="257" title="为什么受伤的总是“果链”" alt="为什么受伤的总是“果链”" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/39868558662a8ebcae04fd8.20736365.png" width="500" height="266" title="为什么受伤的总是“果链”" alt="为什么受伤的总是“果链”" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>立讯精密亦然，这五年来第一大客户的销售占比分别为36.58%、44.85%、55.43%、69.02%、74.09%，几乎是每年增加10%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">抱着一条大腿走路的好处和风险是并存的，比如立讯精密在5年的时间内，从第一大客户身上获得的销售额从83.49亿元暴增至1140.56亿元，增幅高达1266.1%。但是相对应的，第一大欠款方的应收账款也从22.1亿元增长至178.72亿元，最大欠款方占应收账款期末余额合计数的比例从30.01%增长至56.35%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据公开信息来看，这个第一大客户和第一大欠款方，大概率是同一家企业。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">深度的业务绑定使得果链们在苹果面前的议价空间狭小，因为<span><strong>一旦合作告终，果链们的业绩可能会迅速出现崩塌式的下滑。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最明显的例子就是主攻手机镜头模组制造的欧菲光。2021年3月，欧菲光被苹果终止了合作关系。在随后的二季度，欧菲光业绩迅速由盈转亏，并从此一蹶不振。2021年其业绩大幅下滑，甚至到今年一季度滑坡趋势进一步加大。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而<span><strong>苹果的供应链政策一向是多寻找“备胎”以防止某家供应商独大，</strong></span>以前扶持东山精密而降低鹏鼎控股份额、与立讯精密合作而降低对富士康的依赖、包括踢走欧菲光等举措无一不是如此。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">于是，在苹果的“围城”里，外面的果链企业想进来，里面的果链企业则千方百计保住自己不出局。这让苹果在议价时占据了充分的主动权。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们也因此可以看到，国内企业包括头部阵营的厂商都在下场抢标案、抢订单，而且是不惜牺牲价格。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这让苹果有了极大的利润空间，即便是在2022年成本上涨以及销量下降的困难时期，仍然保持了营收和利润的稳定增长。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4月29日，苹果公司发布了2022年Q1财报。数据显示，2022年Q1营收同比增长8.59%，为927.78亿美元；净利润为250.1亿美元，同比增长6%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/135398569162a8ebcb32f5a5.42981981.png" width="512" height="348" title="为什么受伤的总是“果链”" alt="为什么受伤的总是“果链”" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">与之对应的则是果链们的一片惨淡：</p>

<p>歌尔股份一季度营业收入201.12亿元，同比增长43.37%，净利润却同比下滑6.71%；</p>

</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>本想站在巨人的肩膀上的果链们，却一不小心成为了巨人的“输血袋”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据2021年公布的供应链数据统计，苹果的供应链企业有200多家，其中，国内企业就占了98家。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这98家企业可以说都是同领域中的佼佼者，但如果离开苹果公司就要面临业绩断崖的话，这对于国内企业来说是一个不远处的、不小的隐患。</p>

<section data-mid>
<section data-mid>
<section data-mid>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 20px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: center;" data-mid><span><strong>03</strong></span></h1>
</section>
</section>
<section data-mid>
<p style="text-align: center;" data-mid><strong>被迫改变</strong></p>
</section>
</section>
</section>
<section>
<section>当前大多数果链企业仍然处于转型期，新业务很难在短时间内立竿见影。</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在此背景下，一些果链企业已经意识到了对苹果过度依赖的严重性的，并纷纷开始布局新业务，打造第二增长曲线。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">打造第二增长曲线的方式主要有两类：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一类是扩张新业务。<strong><span>越来越多的果链们向新能源汽车、光伏等行业延伸。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/85962705062a8ebcb629039.77094021.jpeg" width="423" height="238" title="为什么受伤的总是“果链”" alt="为什么受伤的总是“果链”" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>新能源汽车是不少果链企业倾心的转型方向</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为AirPods代工厂的歌尔股份，将目光瞄准了AR/VR等新兴智能硬件业务。2021年，歌尔股份智能硬件收入增长了85.87%，占总收入的比重达到41.94%。2022年一季度，这个数字再次上升，智能硬件业务营收同比增长125.05%，占到总营收的50.46%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">蓝思科技的“去苹果化”工作也一直在展开：</p>
<section>
<section>
<p>2017年，公司与NISSHA株式会社合资成立了日写蓝思科技（长沙）有限公司，专门从事触摸传感器产品的生产与销售业务；</p>
<p>2021年，蓝思科技又出资10亿元设立光伏子公司，开展光伏玻璃等产品的技术研发工作；</p>
<p>另外，蓝思还与特斯拉、宾利、保时捷、奔驰、宝马、蔚来等建立长期战略合作关系，批量供应车载中控屏等产品。</p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">立讯精密也开始对新能源汽车业务押下重注，并且直接下场和奇瑞签署战略合作协议，斥资人民币100.54亿元入股奇瑞集团三家公司造车。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2022年5月18日，EXEED星途与立讯精密的星途揽月交车仪式<span>正式</span>举行，标志着立讯精密在新能源造车上的正式出圈。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，涉足新能源车产业链的果链企业还有富士康、舜宇光学科技、领益智造等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/92708965562a8ebcb8baff5.56232776.png" width="602" height="399" title="为什么受伤的总是“果链”" alt="为什么受伤的总是“果链”" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二类选择是将客户面拓宽，打入其他智能手机厂商产业链。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前来看，苹果仍然是全球第二大手机厂商，尤其是随着华为的市场份额下降后，高端品牌领域更是被苹果吞噬。<strong><span>在这种环境下，果链们把“鸡蛋放在不同的篮子里”的策略极难成功，其他品牌留给他们的空间有限。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如今大多数果链企业仍然处于转型期，新业务很难在短时间内立竿见影，其转型成果几何，还需要较长时间来验证，这对于企业、股东甚至是辛苦工作的员工们来说，都算不上好消息。</p>

<section data-mid>
<p style="text-align: center;" data-mid><strong>写在最后</strong></p>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前大部分苹果产品仍然在中国大陆生产，且中国完备的电子产业供应链和基础设施等一直是吸引苹果留在中国的关键因素，但由于国内劳动力价格上升以及其他复杂原因，苹果正将越来越多订单转移到海外。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一财经梳理苹果2021年公布的全球200大供应商后发现，全球610个工厂中，中国大陆工厂的数量总共有259个，占比42.46%，中国台湾的工厂有37个，占比6.07%，而<span><strong>越南设厂的供应商数量在过去四年逐渐增多，到2021年已到达23家，是东南亚各国中最多的国家。</strong></span>且越南制造工厂类型也已从单纯的组装制造向上游的零部件扩展，覆盖显示面板，被动元件，芯片，玻璃等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">随着苹果在越南、印度等区域的不断产业转移，<strong><span>果链们如今要面临的已经不再是国内的内卷竞争，</span></strong>而是如何在国际环境的变化中不停地做出调整：是否要响应苹果需求前往海外投资设厂？是否会面临被东南亚、南亚新果链企业替代的风险？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">果链企业们所要回答的问题，实在太多。</p>
