<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月8日，抖音电商“春雨计划”启动“100种美好生活方式”二期活动，本期主题是“夏日趣玩”，锁定夏天生活方式。通过扩大<a href="https://www.niaogebiji.com/tag/%E4%BC%98%E8%B4%A8%E5%86%85%E5%AE%B9" target="_blank">优质内容</a>推广范围，为优质内容创作者们提供流量与功能权益等全方位资源扶持，抖音电商在这个夏天爆发式地呈现出大批优质内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1281866127630ce7499f0766.80328851.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>春雨计划诞生于今年3月，是抖音电商推出的优质内容激励计划，以多种扶持政策鼓励商家和达人创作优质电商内容，提升长期经营效率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>春雨计划背后是有渊源的。去年5月抖音电商推出《电商创作者管理总则》，10月推出《电商内容创作规范》。今年4月底，抖音电商推出“优质作者精神”，再加上最近这两期“100种美好生活方式”主题活动，抖音电商对于“优质内容”可谓再三强调。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>背后的逻辑也很清晰：</span><span><strong>“优质内容”既是抖音的起点，更是抖音电商的根本。</strong></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>1、</strong></span><span>表象：什么样的内容是优质内容？</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为抖音电商的深度用户，我从8月以来，频繁地刷到了几位达人的短视频，也明显地感受到了这些视频内容的高级感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种高级感分三个方面：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一，音画质量很高。这些达人的短视频无论是光线、布景还是服装、器皿，给我的感觉都是精心思考过的，再加上高质量的拍摄技巧、多机位的全景细节捕捉，以及用心的剪辑制作，从音画品质上贴近专业的影像作品，通过镜头语言传递给观众愉悦的观看体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1095760198630ce74a40bf93.62576050.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二，内容细节丰富、专业，就像看高质量的纪录片或者电影。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过适时的特写镜头，视频中的场景细节得到了充分展现，如果从专业影像从业人员的视角来看，这就是镜头之所以打动人的一个最有效的触点：</span><span><strong>内容藏在细节里，微观内容的呈现是揭开真相的法宝。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/63786144630ce74b8407a5.61753896.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三，虽然达人们呈现的生活方式五花八门，但视频中的产品居然能很奇妙地能和我的生活方式同频共振，激发起强烈的消费兴趣。这大概就是兴趣电商的奇妙所在，它通过专业有趣的视频表达，能有效地激发潜在的消费兴趣。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>@大野蜜探·背锅侠的内容“格式感”强烈，是一种颇有创新的“戏中戏”的格式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/575987112630ce74c676ac4.46187532.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>沉浸式观看其他播主的美食制作内容，然后按步骤复刻内容，融合了美食评测、美食制作等元素，传递了潜在信息：</span><span><strong>买了这些食材，跟着这个步骤我也能还原美食，不会踩坑。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>@植迷不误的视频内容知识性、专业性极强。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1728671177630ce74d09bf64.96487390.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他本人是云南一家花卉市场的专业花店老板，围绕花卉直播带货，带货的过程有非常多的“干货”，每一种花怎么养都讲得清清楚楚，这些直播干货被切片成短视频进行二次传播，内容的专业度极高，看他的视频很长知识，</span><span><strong>花花草草看着好像也不是那么容易死，想要买回家亲自养一养。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>@妈妈很忙手工美食则突出了商家产品的独特识别度、信任感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/272246316630ce74dd70034.27777608.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过高频次的短视频放出，每天发布多条视频，从多个角度介绍食品的配方，突出安全食材、强化口碑效应，没有剧情，</span><span><strong>但依然收获可观的粉丝量和销售额，店铺评分很高。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这些内容的特点来看，所谓优质其实蕴含着一些递进式的逻辑，比如有创意的展现形式、流畅的镜头语言、简洁的文案等等，这些是第一层“优质”；而通过综合了这些拍摄与表达技巧之后营造的沉浸式的观看体验，是第二层“优质”；最深层的“优质”，则是用视频内容与镜头前的观众形成了一种共情连接，能够有效地触达观者的情感，唤醒观者作为消费者的那一面，进而形成转化，优质的内容能够催生出高效的消费行为。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>2、</strong></span><span>本质：通过优质内容，得到了什么？</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些达人近期发布的优质内容大多带着“夏日趣玩”的话题小尾巴，这正是春雨计划第二期的主题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么通过这期活动，达人们得到了什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一是更强的话题势能，形成了夏日主题的消费趋势。都说大树下面好乘凉，话题就是生意的大树，形成合力之后，大家的注意力被聚焦在话题下，生意就会有新的增量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1394273138630ce74e90d2e1.41328144.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二是持续出圈、持续“扩产”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本次活动扩大了覆盖范围，与首期活动重点布局美妆、美食、数码家电行业相比，</span><span>二期活动分为三个内容版块，涵盖七个夏季消费热点行业：“夏日悠活”主打宅家生活，覆盖消费电子家居生活、个护家清鲜花、亲子生活；“夏日尝鲜”汇集时令美食，覆盖食品生鲜、酒</span><span>/</span><span>滋补保健；“夏日美学”聚焦夏季穿搭，覆盖服饰鞋包、珠宝潮奢。二期活动也让更多的达人、商家获得生意增量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/125050248630ce74eee2074.00450219.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在首期活动的基础上，本次活动还设置了直播间分会场，邀请@交个朋友直播间、@王小骞等优质内容创作者开播，打造标杆带货直播间，帮助全平台创作者更直观地理解优质内容直播间标准。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对消费者而言，通过这些越来越多的优质内容，一方面增长了知识、享受了更愉悦的观看体验，另一方面也增强了消费信任感，优化了消费体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以内容连接商品的消费路径被进一步强化，从兴趣电商进阶到全域兴趣电商，从单次购买升级为反复的复购行为，因为确实买到了称心如意的好商品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在消费环境持续承压的大环境下，能够精准地买到好货“不踩坑”，就是省了钱，提高了消费效率。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对抖音电商平台来说，通过持续地强化优质内容的创作方向，让更多的达人商家明确地感知到了内容创作的标准，形成了内容水准整体提升的合力，平台内容氛围再上台阶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内容的升级带来的是消费环境的升级，消费者在抖音电商的下单意愿更强。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更多的电商达人、更丰富的内容供给也意味着消费者在抖音电商的消费选择更多，这些合力共同带来了更多的生意增量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在抖音电商做生意，变得更有章可循、增长更加确定。</strong></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>3、</strong></span><span>未来：优质内容没有终点，商机轮动螺旋上升</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1920598447630ce74f7c7c65.34788816.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“一张图读懂春雨计划”</span><span>系列长图的部分截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过平台发布的“一张图读懂春雨计划”系列长图，可以清楚地看到优质内容的形式与标准，将内容创作形式分为</span><span><strong>挂车和非挂车短视频、挂车直播和挂车图文</strong></span><span><strong>，</strong></span><span>按类别详细解读了“春雨计划”的细则、标准和收益。其中，非挂车短视频和图文短视频也是首次纳入了优质内容考核范畴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“内容”是抖音电商的根本，“优质内容”是抖音电商继续通往未来的阶梯。从过去到未来，在这个充满不确定的大环境下，抖音电商能够为商家与消费者提供一个相对确定的价值阵地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这正是理解春雨计划背后的优质内容生产力的根本逻辑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要进一步看懂这个逻辑，需要理解抖音电商所谓的内容的本质是什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实就是人们对于兴趣的刚需，大家对这个世界充满好奇，对自己没有涉足过的领域想要去了解，去建立连接。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这一个点，就是一台消费的永动机。</strong></span><span>抖音电商以短视频内容深刻地切中了这个痛点，所以获得了堪称迅猛的发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而平台在这个已经建立起来的“创造、分发、转化”的内容业务飞轮上究竟该扮演什么样的角色呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>一是倡导者，二是维护者。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音电商的内容生态一方面要继续进阶，所以“优质内容”的供给要放大，抖音电商作为平台要通过春雨计划这样的官方活动进行倡导，把优质内容的方向传递给内容创作者们，加深整个内容生态与消费者的兴趣连接，这个连接的过程，就会催生出新的商机。所以抖音电商对于优质内容的再三强调，正是基于这样的逻辑起点：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>优质的内容，会带来优质的生意。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>倡导的同时，也要维护。内容本身的宽度与深度可以说是无限的，这是与人类社会生产生活密切相关的东西，但优质内容在任何时代都是相对稀缺的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以春雨计划的另一层含义就是正确的方向，抖音电商平台希望透过这一计划将此前出台的一系列相对理论化的优质内容创作标准进行落地，以此来不断维护平台的内容生态，让优质内容带来的确定性增长成为平台的主流价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看向未来，这种由不断放大的优质内容供给所连接起来的消费空间是广袤的，因为优质内容的稀缺性决定了对它的追求是没有终点的，永远都会有更优质的内容呈现，也就意味着会有更强的商业价值出现。在春雨计划助推的这种商机轮动的过程中，抖音电商的内容创作者们也有了一份更加持久的内容事业，与之伴生的商家的生意也会一直螺旋状向上跃迁。</span></p>
