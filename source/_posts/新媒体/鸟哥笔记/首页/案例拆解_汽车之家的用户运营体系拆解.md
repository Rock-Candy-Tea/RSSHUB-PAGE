<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a href="https://www.niaogebiji.com/tag/%E6%B1%BD%E8%BD%A6%E4%B9%8B%E5%AE%B6" target="_blank">汽车之家</a>于2004年筹建，2005年6月正式上线,是全球访问量最大的汽车网站。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为消费者提供选车、买车、用车、换车等所有环节准确、快捷的一站式服务。至今，全平台拥有4.3亿注册用户。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1、用户画像</strong></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">（来源：移动观象台）</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从性别上看，汽车之家主要用户以男性为主，男女比例6：4。说明对汽车类产品男性关注度较高，用户运营可以从男性入手。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/188105008762b27a36e14661.55985178.png" title="案例拆解|汽车之家的用户运营体系拆解" alt="案例拆解|汽车之家的用户运营体系拆解" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">（来源：移动观象台）</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从年龄上看，主要用户集中在26-35岁， 占整款产品近5成，说明80后是汽车市场的核心群体 。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从用户年龄分布趋势可知，汽车消费能力强、购车需求大。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/121194021362b27a3734ce72.74295032.png" title="案例拆解|汽车之家的用户运营体系拆解" alt="案例拆解|汽车之家的用户运营体系拆解" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">（来源：移动观象台）</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从地域上看，用户群体主要集中在全国的东部和中部，以一、二线城市为首。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一二城市消费能力高、汽车消费兴趣浓。汽车之家用户主要集中的地方，广东，浙江，江苏等东部沿海地区渗透率较高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2、产品的核心价值</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">汽车之家为汽车消费者提供选车、买车、用车、换车等所有环节的全面、准确、快捷的一站式服务。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3、用户需求</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">买车、卖车、换车。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>4、内部优势</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">是全球访问量最大的汽车网站，二手车平台商品数远超同类竞品。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>5、主要业务流程图</strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1）用户增长的北极星指标是什么？</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">APP用户活跃数</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2）渠道拉新重点选1-2个渠道进行拆解：</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/91955287162b27a382e0049.56711951.png" title="案例拆解|汽车之家的用户运营体系拆解" alt="案例拆解|汽车之家的用户运营体系拆解" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">渠道：他们为什么选择了这个渠道？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用了免费发布的渠道，将原创内容分发到抖音、B站、知乎、小红书等平台。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">原因：这些平台流量大，触达的用户多。虽然用户杂，但是原创内容可以精准筛选部分用户变为汽车之家的用户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">抓手：他们是如何设计内容+钩子或者活动的活动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">将优质的原创内容分发到各个平台，吸引用户阅读、观看，再引导有需要的用户下载APP并注册登录。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">转化：转化漏斗的上做的好的部分，做得不好的部分。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在拉新方面，感觉原创内容应该可以更优化，多做热门车型的内容更能吸引用户阅读、观看。而一些小众的内容应该做合集、减少视频量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在转化方面，APP界面下方直接提供“选车”和“二手车”功能，方便用户更快速点击并通过热门筛选得到心仪车型，利用“询底价”的功能模块吸引用户咨询，并提供线上留信息和直接拨打经销商电话两种方式，缩短用户联系经销商的步骤，提高转化率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3）裂变拉新重点拆解</strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">活动类型是什么？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">任务宝裂变</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">拆解活动规则、路径以及如何承接</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">活动规则&路径</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/208798852662b27a3a12daa0.61357920.png" title="案例拆解|汽车之家的用户运营体系拆解" alt="案例拆解|汽车之家的用户运营体系拆解" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/92527958062b27a3a9f8b12.44986066.png" title="案例拆解|汽车之家的用户运营体系拆解" alt="案例拆解|汽车之家的用户运营体系拆解" referrerpolicy="no-referrer"></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">四、激励体系的拆解</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1）激励体系的运营目标</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">产品的北极星指标是什么</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134890650962b27a3b8b6cd1.50018613.png" title="案例拆解|汽车之家的用户运营体系拆解" alt="案例拆解|汽车之家的用户运营体系拆解" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">产品的主要为用户提供比价、车友圈论坛的功能，而在平台内用户越活跃，体现在更愿意在论坛讨论和浏览，也更进一步能转化率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以北极星指标是用户的活跃度，日活、月活。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">激励体系的目标如何和北极星指标结合</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在制定用户激励的时候，关键在于激励用户多发帖、多浏览，拉长用户使用时间，促进用户的活跃度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2）和运营目标相关的核心动作拆解</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">和运营目标相关的关键动作是什么？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我认为关键动作有浏览内容、比价、观看直播、浏览社区、社区互动、签到、购买。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">关键动作之间的优先级是什么？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">优先级：购买>浏览内容>比价>社区互动>观看直播>签到</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">关键动作如何和产品的北极星指标联系起来</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">经过前期的用户分析发现，目前汽车用户主要以85-95后群体为主。而以浏览内容为例，用户日常获取汽车内容以互联网为主，而视频和图文是用户最喜欢的形式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/68032680662b27a3bd58bc5.59296831.png" title="案例拆解|汽车之家的用户运营体系拆解" alt="案例拆解|汽车之家的用户运营体系拆解" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">（来源：Z世代汽车观洞察报告-益普索x哔哩哔哩洞察）</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以围绕浏览内容这个关键动作，我们应该在产品首页放置更多的视频、图文类内容，增加用户的停留时长，提高活跃度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3）激励体系拆解</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">产品激励体系的表现形式</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">签到、勋章、汽车币</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">产品激励体系的底层逻辑是什么？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用签到和勋章的形式，引导用户每天打开APP，做相应的任务，获得汽车币同时延长用户停留时间，提高用户活跃度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">签到给用户的激励有物质激励：汽车币、洗车券等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而成就勋章分为行为、身份、活动等类型，激励各种行为的用户，有浏览、回帖、创作、签到等，提高活跃度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">产品激励体系的抓手是什么</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">代金券、洗车券、等级</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">激励体系的触达场景和用户路径是什么</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">个人中心、论坛内别人的ID后</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">激励体系中为了实现北极星指标做了什么努力</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以勋章为例，设置了多种行为的勋章。有激励阅读时长；有激励每日登陆；有激励每日签到；有激励社区互动。激励这些行为都是围绕着激励用户多发帖、多浏览，拉长用户使用时间，从而促进用户的活跃度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">勋章还分等级，用户只有每日完成任务（量变），才能逐渐去达到成就系统里的长期目标（质变）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/21677859662b27a3ca72d55.42969930.png" title="案例拆解|汽车之家的用户运营体系拆解" alt="案例拆解|汽车之家的用户运营体系拆解" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">五、关于用户激励的思考</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-foldable><strong>1）竞品一般都是怎么做的？</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">竞品一般把会员等级和勋章体系分开，我觉得各有各的优点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分开：可以以登陆/签到/互动等行为去增加用户积分，提升等级，用户更关注总等级，而不是一个个的行为，可以利用不同行为的分数权重不同引导用户，如拉新一个100点经验值、浏览5点经验值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不分开：让用户更倾向于做有勋章的行为，如回帖、签到等，用户更关注自己喜欢的行为，给用户更好的使用体验，享受每天养成自己的账号。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2）拆解的产品有哪些是可以借鉴和复用的</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">汽车之家的频道比别家APP还多，分别为：我的、广场、车友圈、Young、本地。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我的频道：可以满足个性化需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">广场频道：广场对不同类别的帖子做了精细划分，有车系类，用车行为类，主题类等。用户也可以发帖子和提问为内容产出做贡献。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">车友圈：可以说是论坛里面的社群，类似于微信的群组和朋友圈。用户可以找到需要的圈子或者建个圈子（群），发动态、活动、小视频。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">Young频道：目标用户是Z世一代，鼓励年轻创作者产出优质的内容，形式包括图文、帖子、小视频；符合汽车垂类平台的战略发展，挖掘Z世一代的潜在价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本地频道：和车友圈的模式大同小异。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">每一个频道都是为不同类型用户的兴趣点设计，方便用户快速找到自己喜欢的内容，从而提高APP内停留时长。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3）有哪些可以优化的点</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/34961867162b27a3e2a4449.22577383.png" title="案例拆解|汽车之家的用户运营体系拆解" alt="案例拆解|汽车之家的用户运营体系拆解" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如每日浏览任务中，点击进去是用户口碑的帖子。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是帖子的顺序好像是固定的，可以结合用户日常使用中的偏好修改，将用户更喜欢的品牌等内容推送在前面。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">还有用户口碑帖子没有显示车型，会给进入到这个页面的用户增加一定的阅读成本，如果能在下方加入车型能更方便用户筛选阅读，提高用户体验感。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">由上图可见，汽车之家主要变现方式有：媒体服务、线索服务及在线营销及其他。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">①向汽车制造商提供媒体服务（广告费）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">②向经销商提供线索服务（订阅费、广告费、二手车信息展示服务费）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">③在线营销及其他服务</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">⑴数据产品（订阅费、项目费）</p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" data-default-linespacing="100">七、变现产品运营的拆解</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据北极星指标，选1-2个变现产品拆解其提升客单价、转化率、复购的方式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">北极星指标：用户活跃度</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">变现产品选择：①向汽车制造商提供媒体服务（广告费）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">假设某汽车品牌方这次主推一款18-25万左右的SUV。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">提升客单价：可以选择推荐广告主在首页头条的位置做广告投放，也可以将多个板块头部曝光和开屏页广告位打包销售，提高客单价。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">提升转化率：汽车之家可以将这款车加入选车中15-20万、20-30万和SUV的三个常用筛选栏内靠前的位置，也可以将对应汽车品牌在选车板块的热门汽车品牌前置。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">复购：可以与广告主签订年框广告，让广告主习惯投放的频率，每年申请投放经费的时候都会想着签订年框。这个期间，业务做好必要的客户运营也是很必要的。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" data-default-linespacing="100">八、会员变现？</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">权益、定价、售卖的拆解</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">会员体系，以及会员日等会活动等拆解。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/105805826362b27a3fa5f0b4.55996806.png" title="案例拆解|汽车之家的用户运营体系拆解" alt="案例拆解|汽车之家的用户运营体系拆解" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">汽车之家付费会员分为两个档次：车主会员和黄金会员。还有新人体检卡让没开通过会员想体验的用户选择。分别定价：99元/年；199元/年；19.9元/3个。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">新人体验卡权益：洗车1次+加油优惠+保养优惠+车品优惠+购车现金补贴+卖车免费上门评估；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">车主会员权益：洗车4次+加油优惠+保养优惠+车品优惠+充电优惠+免费驾照翻译件+卖车免费上门评估；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">黄基金会员权益：洗车4次+加油优惠+道路救援+保养优惠+百万交通意外险+驾乘意外险+旅行意外险免费驾照翻译件+卖车免费上门评估。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">九、深度思考</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">优点：平台将主要变现手段瞄准品牌、经销商等客户。这样既可以有更多动力提供优质内容给用户、服务好用户提高活跃度，又方便维系客户关系，钱的大头是在品牌方和广告主手上，靠用户一点点的付费会员肯定没有那么可控。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">缺点：付费会员感觉可以增加等级制度，只看会员身份保持续费时间，比如每天15点经验，到1000点升级。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我想想做的：个人会员等级升级后，将部分权益再升级，吸引已付费会员复购。反之掉级，保底权益就是开通会员时的权益。</p>
