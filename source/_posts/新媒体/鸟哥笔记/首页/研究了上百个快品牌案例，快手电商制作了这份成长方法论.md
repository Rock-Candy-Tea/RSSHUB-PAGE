<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快品牌正在成为风口。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年，在朋友推荐下，专注做跨境<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%95%86" target="_blank">电商</a>的王崇把目光转移到快手平台，以“功效类护肤品”作为切入口，开启了护肤品创业和直播生涯。“刻苦”直播两年多，王崇在快手收获了220万粉丝，且信任度极高，6成粉丝在持续使用莱润。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/27605199062ec6025397f89.82664011.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@小虫【胶原蛋白大促】 快手视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从事服装行业多年的大姗姗，在各种渠道做过尝试后，自创大码女装品牌“姗蔓”。2020年，她选择入局快手，两年吸粉654万，“姗蔓”也成为快手大码女装品类的头部品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>姗蔓和莱润是“快品牌”迅速发展的两个缩影。女装、美妆、农资、厨具......卡思注意到，不同行业的品牌在快手电商里趟出了自己的路，涌现出如GG潮牌、思考人生、莱润、素拼男装、菌崔莱、ZMOR、梅特德菲等颇具代表性的快品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了给优质“快品牌”提供燃料、完善平台商品供给，今年5月的引力大会上，在“大搞信任电商，大搞品牌，大搞服务商”之外，快手电商提出了<strong> “大搞快品牌”</strong>战略，<strong>宣布扶持向消费者提供“高体验-价格比”产品及服务的新兴品牌，并计划拿出230亿+流量、扶持超过500个快品牌标杆。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在快手616实在购物节期间（5月20日-6月19日），快品牌交出了首张成绩单：快品牌直播间贡献订单量超过1200万，49个直播间销售额破千万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，快品牌在快手电商有哪些经营方法论？如何顺利度过达人品牌初建期、品牌认知发展期以及分销信任增长期？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月29日，快手电商发布了《2022快手电商快品牌成长方法论白皮书》。<strong>该方法论（简称5K方法论）包含了创建品牌、打造人设、提升服务、拓展规模、扩大分销五个方面。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/143816048262ec60258d6f60.51374112.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>5K方法论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过拆解这份白皮书、结合商家直播案例，卡思将尝试找到打通快手经营阵地的通关秘籍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>快品牌打好根基的三个要素</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然“5K方法论”刚刚发布，但在此之前，卡思发现，已经有多个快品牌借着快手电商的风，找到了适合自身发展的玩法，先行者的经验和5K方法论是相互映照的。在发展初期，快品牌打好根基的三个要素是创建品牌、打造人设和提升服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先，商家会依托粉丝画像和快手平台用户数据，创建品牌。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>达人主播美遥于2021年8月入驻快手，通过分享穿搭视频，积累了一批女性粉丝。随后，美遥团队从自身定位和专长出发，结合市场需求，以20-40岁独立、自信的都市女性为目标消费群，创建了女装品牌“ZMOR”；产业带主播则可通过建立品牌树立与其他直播间差异化亮点，主播@龙团胜雪 就创立了茶叶品牌“思考人生”，意为“品茶也是在品味人生”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有了品牌构想后，商家想要获取信任，不能忽视的一环是“打造人设 ”，即通过内容创作、常态化直播，在老铁心中树立人设。因为快手是强私域、强信任的平台，有血有肉的主播人设更能获取信任，从而使得粉丝愿意为其买单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>卡思观察到，快品牌商家的“人设”大致可分为两种，即“专业人士”和“主理人/创始人IP”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“专业”人设的代表商家之一是@龙团胜雪。她是国家一级茶艺师，家里做茶多年，自有茶山和制茶基地。2018年，她开始在快手拍摄短视频、直播讲解茶文化、茶叶冲泡、品鉴等知识，强化自己“福建省最年轻的国家一级茶艺师”人设，建立起一个喝茶、品茶的社交圈，由此顺势推出中高端茶叶快品牌“思考人生”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/122272651962ec6025edef79.97456191.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@龙团胜雪【思考人生茶业】快手视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在@龙团胜雪的直播间，有茶山的碧绿葱郁，也有茶厂里打包发货的忙碌，还有静静泡茶的耐心，很多老顾客出于“认准这个人”的信任感下单。茶叶属于非标商品，价格跨度大，品牌亦多如牛毛，但“思考人生”依靠树立专业人设、维系信任在这个品类中站稳了脚跟。目前快品牌“思考人生”的年销售额已超5000万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果要建立创始人IP，则需要主播具备鲜明的个性，且擅长通过短视频和直播去展现个人形象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大姗姗本人曾是超重女孩，并且在服装行业浸淫多年，是源头工厂老板娘。在她的视频中，不仅有大量日常穿搭、街拍美图和新品展示，还披露了大量个人生活的细节，比如去旅行、带小孩、拍婚纱照等，让粉丝感觉是全方位认识了一个亲切的姐妹，而不只是关注了一个电商主播。就有用户在其视频下留言：“对于我来说阿拉姗早就是家人一样了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由此，大姗姗树立了“懂用户痛点、擅长穿搭、会生活”的人设，其自创品牌”姗蔓“的理念也更深入人心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/103486275162ec60264bd1f3.18971224.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@大姗姗【姗蔓服饰工厂】快手视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想要人设屹立不倒，不止需要做好短视频和直播内容，还必须在“提升服务”上用足功夫。<strong>只有将“提升服务”和“树立人设”都做好，才能更顺利地度过爬坡期。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提升服务是一个体系性工程，即将产品价格、产品性能、服务体验方面同时做到最优，在把价格打下去的同时，也要把服务体验（包括选品、履约、售后等体验）做上来。为消费者提供真正的高“体验-价格比”的商品，才有助于商家私域沉淀，并反向带来更高复购。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这方面，国潮女装品牌GG A HOLIDAY 的做法可作为参考。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年3月，GG创始人朱芳萱入驻快手，她是平台内少见的完全独立设计、把控生产全流程的服饰主播。近2年，GG 保持快手小店5星评级，在快手的月GMV近1200万，用户复购率为75%，私域购买占比达69%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/10137280662ec60268ce165.29051432.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@GG潮牌创始人朱芳萱 快手视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卡思曾总结GG在快手的成功原因，一是<strong>依赖于稳定的货品保障</strong>，旗下有多个自有工厂，从定制面料、设计到生产加工，品控极强；二是<strong>创始人出镜讲解</strong>，迅速提升老铁信任感；三是<strong>款式为原创设计</strong>，上新速度快，且线上、线下不同款，互不影响动销；四是<strong>性价比高</strong>，线上价格集中在300-1000元左右，部分款式价格100元以下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>如何持续向上生长？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>度过初创期、进入成熟阶段后，就是快品牌应该考虑<strong>扩展规模</strong>的时刻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>扩展规模的一个思路是从垂直品类出发，先打造单个爆品，再通过直播测品等方式制造新的爆款，增加直播间SKU。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>思考人生从起初局限于福建产业带茶品类，拓展至云南、浙江等不同茶产业带优质品类，丰富了直播间的商品供给。目前，龙团胜雪在快手的粉丝达到594万，年GMV超过5000万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一种思路则是直接建立子品牌，甚至可以是跨品类的品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>GG 就通过建立副线品牌，覆盖了不同需求的用户。自GG在快手的销量一路走高后，朱芳萱据此打造了涵盖不同年龄段、不同性别、不同消费程度的副线品牌，如ZHUZHENFEI、RE、NEKO-PUSS等，甚至跨品类试水了美妆领域。接下来，朱芳萱准备打造一个占地5000多平米的服饰陈列厅，并孵化培养更多主播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在扩展规模之外，快品牌要进一步扩大品牌在快手全站的影响力、搭建更完整的线上渠道，应当建立分销团队、与不同的快手主播合作，尝试建立自有品牌分销机制。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一些品牌通过建立矩阵账号，开展分销。</span></strong><span>这样一来，不仅能解放主播的生产力、扩大销量，也能通过货品和人设的新组合，吸引到不同层面的粉丝，扩大品牌的覆盖范围。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，@大姗姗 就通过建立小号@大姗姗大码特体女装工厂，打造了矩阵直播间，目前小号的粉丝量也达到148.5万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做好站内分销后，有余力的快品牌，也不应忽视站外的流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>ZMOR 就通过签约国际品牌、合作热门综艺节目，有效扩大了站外影响力，打开了更广的销路。美遥团队签约了国际知名品牌SMIELYORIGNAL 笑脸高端品牌线、合作《浪姐3》，打开了ZMOR在站外的传播声量。在微博、小红书等社交媒体，也开始出现ZMOR种草帖。<strong>目前，ZOMR月均GMV近3000万，私域成交占比超过60%。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/96992945762ec6026d18f36.80355693.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@美遥 快手视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先行者的经验或许不能完全照搬，但一定有可取之处。总结而言，从上述快品牌的成长路径来看，如果能做好创建品牌、打造人设、提升服务、拓展规模、扩大分销这5大步骤，商家的上升之路就逐渐清晰，前行的步伐更加稳健。</span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>平台为快品牌保驾护航</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了快品牌自身努力，平台亦从流量扶持、政策等方面陪伴、扶持着快品牌的整个成长周期。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月4日，“2022 造风者・快手电商服务商大会”上，快手电商快品牌及快消行业运营负责人宋震介绍，平台会从基础权益，商家&服务商政策以及营销政策三方面，为快品牌和潜力快品牌提供支持。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/202491490562ec6027305538.76561944.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>快品牌最新扶持政策</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中<strong>基础权益</strong>包含快品牌打标、主播认证、快品牌主页曝光等；<strong>商家&服务商政策</strong>是从流量、现金和商业化等多维度对快品牌进行组合激励，包括自有品牌GMV激励、商业化消耗激励、入驻新商家激励、商家快品牌成长计划、快品牌消耗额外的返点等；<strong>营销政策</strong>则从日常活动、大促活动、IP活动等不同营销场域为快品牌增加曝光量，致力于打造行业标杆，提升整个快品牌在站内和站外的影响力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据公开信息，快手计划在2022年扶持500个以上的标杆快品牌，目前平台正在通过综合各方力量，为快品牌和品牌商家的成长保驾护航。可以预见的是，借助快品牌成长方法论，以及快手电商提供的诸多扶持政策，更多快品牌将有机会脱颖而出，实现粉丝量到销量、声量的大跃升。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当越来越多的快品牌成长起来，用户将拥有更丰富的商品供给、更优质的购物体验，信任将成为快手商业环境的稳固地基，快手电商就完成了价值升级。</span></p>
