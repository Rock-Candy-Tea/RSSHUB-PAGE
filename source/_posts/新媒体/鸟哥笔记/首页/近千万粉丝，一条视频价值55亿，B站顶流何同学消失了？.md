<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>从激励计划分成大幅下降，去年还能拿到数万元创作激励金，今年却只能拿到几百元，再到推出付费视频功能，B站这一举措似乎试图让<a href="https://www.niaogebiji.com/tag/UP%E4%B8%BB" target="_blank">UP主</a>们自力更生。</span><span>“为爱发电”的UP主举步维艰，比如科技数码区的UP主。</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年6月，何同学凭借封神之作《5G到底有多快》，迅速蹿红。当时的他还是一名大二学生，一战成名后，B站三天涨粉百万，微博涨粉60万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年12月，何同学在毕业前夕开设了自己的工作室。如今距离何同学成名已经过去了4年，现在的何同学怎么样了？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年2月份，何同学发布了一则视频《我用108天开了个灯......》，让其被推到了争议的制高点。越来越多的人开始质疑其视频质量，“江郎才尽”的言论越演越</span><span>烈。尽管此前，他的另一则视频《我制作了苹果放弃的产品...》，让乐歌股票暴涨近14%，价值5.5亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3180557362ead671933497.25338192.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从“何同学出品，必属精品”到“内容没有以前有意思了”；从不接商业广告，收入只靠B站的创作激励和充电，2年总共只赚不到3万块，到“恰饭”视频越来越多，甚至一期视频全网投放价格可以去到300W~500W，<strong>走红后的何同学身价水涨船高，但也深陷内容困境。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>成为UP主的5年时间里，何同学至今一共投稿了44个视频，平均一年9个，总播放量2.9亿。</span></strong><span>为了摆脱这种困境，何同学成立了工作室、扩充团队，但这似乎也没能解决何同学的困境，视频更新速度依然跟不上，何同学上一次更新视频已经是4个月之前。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/119351032962ead671f37b65.89575909.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>与此同时，何同学也开始了转型之路。</span></strong><span>不同于以往视频中的科技与人文齐飞，何同学如今的视频更倾向于专业技术流，在成为优秀DIY区UP主的路上越走越远，甚至被戏称为“科技区手工耿”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管何同学如今的B站粉丝数已升至953.1W，但尚存的流量热度终有一天会消散。何同学困于更新速度与创意内容这两座大山，其他科技数码区UP主又何尝不是？</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>相比于游戏、生活、影视等分区的UP主，科技数码区的UP主似乎需要付出的“成本”更高。</span></strong><span>为了凸显专业性，UP主们往往需要更多精良的设备，同时还要承担接不到商业广告的风险。投入与回报不对等，一个直观的反映是科技数码区的投稿视频数量明显少于生活、游戏、影视分区。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>值得注意的是，科技数码区UP主想要挤进热门也会更难。</span></strong><span>据果酱妹观察，在B站最新一期的每周必看榜单中，44个视频中只有2个视频出自科技数码区，分别是@稚晖君 </span><span>（粉丝数209.6W）</span><span>和@苏打baka </span><span>（粉丝数172.8W）</span><span>，占比1%都不到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/104970897262ead67275cec3.87361889.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以B站硬核科技UP主@稚晖君 为例，他的更新频率与@何同学 一样，可以说是季更UP主。50个投稿视频中，过百万的视频仅占比16%。再来看百大UP主@影视飓风 ，501个投稿视频中，过百万的视频仅为94个，占比近19%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>或许为了解决流量困局，联合投稿反倒成了越来越多UP主抱团取暖的创作方式，</span></strong><span>既然大家都在寒冬之中，不如互相引流，看下是否能回春。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>除此之外，UP主们还做起了矩阵账号。</span></strong><span>比如百大UP主@影视飓风 开了个小号@亿点点不一样 ，@科技美学 的掌门人那岩，自己单独开了个账号@那岩KJMX 。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/36467849662ead672dd3696.44059103.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>头部UP主尚且如此，腰尾部创作者的处境或许更为艰难。也因此，不少UP主开始往“流量密码”上凑。</span></strong><span>以B站某科技区UP主为例，不同于早期视频中的开箱测评、知识科普，其目前的投稿视频“花式整活”。封面图清一色的美女，擦边球都快打出火花了。自然地，其最高播放量的视频也大多集中在此类内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/126250964862ead6742bbc37.09357753.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，科技数码区UP主的钱途又如何？根据B站2021年Q4财报显示，有资格参与广告分成的UP主数量为55.7W，而该季度B站的广告收入为15.9亿元。换言之，更多UP主需要自食其力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在B站，UP主主要通过充电计划、悬赏计划、活动收入以及激励计划来获取收益。</span></strong><span>其中，充电计划和悬赏计划具备一定门槛，即便是@何同学 这样近千万粉丝的UP主，截至目前累计17552人为其充电，即便以每人充电88元来计算，@何同学 一年充电收入也只有30W，一个月2W多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/114411704562ead674c5bfb8.90761484.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而随着B站连年亏损，B站CFO樊欣曾表示要通过降本增效，实现盈亏平衡。这就意味着，B站花在内容激励上的成本将会不可避免地减少，UP主通过激励计划获取的收益难免有所减少，不少UP主纷纷表示创作者激励收入明显下降，百万播放量的视频收入仅有400元。而对于科技数码区UP主来说，百万播放量视频并不多见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/96173523162ead67546bf22.51593860.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>平台奖励靠不住，科技数码区UP主只能靠“恰饭”来维持。但从商业广告类型来看，科技数码区UP主往往又受限于品牌领域。</span></strong><span>根据《QuestMobile2022互联网广告市场半年大报告》显示，美妆护理、网络购物才是金主爸爸们投得最多的广告，至于IT电子、家用电器，有但不多。普通内容创作者，大多停留在开箱测评，能接触到的品牌无外乎市面上那几个，僧多粥少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/114555417862ead675c70584.74453123.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前有B站降本增效，平台红利日渐式微，后有头部UP主抱团取暖，腰尾部UP主夹缝生存。科技数码区的UP主，是用爱发电还是为爱贴钱？</span></p>
