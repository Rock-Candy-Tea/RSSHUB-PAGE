<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着人工智能、大数据、云计算、区块链等前沿技术的发展,商业银行正在迈入数字化、智能化、开放化的数字银行4.0时代，银行业务离柜率不断上升，目前已突破 90%。</span><span>由于数字技术的不断完善，客户通过移动端获得金融服务更加便捷，“流量运营”从线下向全渠道延伸。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数字时代，由于运营成本和经营形态的变化，运营方式从线下分区域运营，向线上一体化、线上线下协同运营转变，营销服务也将向线上化数据智慧化升级。商业银行必须构建以“流量运营”为导向的运营模式和体系，来不断拓宽金融触点和服务渠道。目前，北京银行借助金融科技，打造“智策”数字化运营体系，加速推进数字化转型发展。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>建设数字化运营体系的必要性</span></strong></span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数字化运营能帮助银行重塑运营体系，实现端到端、从前台到后台的全面转型。通过数字化经营，银行可盘活和深耕存量客户，不仅能有效提升和释放网点与客户经理的产能和潜力，降低运营成本，同时，还可以围绕数字用户资产的管理、提升、转化和变现，直接影响银行数字化发展的成效，也是保持银行竞争力甚至实现弯道超车的重要途径。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>数字化运营体系的定义</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着客户行为和市场环境变化，经营模式正在从根本上重塑，数字化运营体系是基于数据流，经历从采集、建模、分析、反馈的过程。在这过程中，为保证数据流的高效运行，企业需要打好坚实的数据基础，完成精准的数据埋点、数据源打通、数据存储、数据建模与分析等环节。银行数字化运营体系表现形式，即银行通过分析客户画像、行为洞察等信息，感知客户需求变化，快速判断问题点，进而形成解决方案，并向客户提供个性化运营，最终通过结果反馈对效果进行评估，从而更好地迭代下一次的运营计划。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>数字化运营体系给银行带来的价值</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数字化运营体系带来的价值主要体现：促进用户活跃。通过精细化渠道触达，活动运营，提升用户访问频率与使用时长；优化用户体验。通过诊断产品可用性与易用性，改善产品感官体验与交互体验，提升用户满意度、忠诚度；提升用户价值。通过建立用户分层体系，细分用户需求，了解用户业务偏好与消费习惯，对业务流程进行诊断，有效提升业务各环节转化；驱动业务创新。通过评估用户需求，定位业务运营方向，指导业务创新与竞争力提升。</span><span></span></p>

<p data-darkmode-color-16281471254889="rgb(163, 163, 163)" data-darkmode-original-color-16281471254889="#fff|rgb(0, 0, 0)" data-darkmode-bgcolor-16281471254889="rgb(188, 190, 193)" data-darkmode-original-bgcolor-16281471254889="#fff|rgb(246, 249, 253)"><strong><span>满足客户需求</span></strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>受疫情影响，银行服务对象、形式均发生了变化，商业银行也陆续开启了全方位的“无接触”金融服务，通过数字化经营，使得银行服务有效盘活和深耕存量客户，提升和释放网点与客户经理的产能和潜力并取得积极效果。</span><span>数据显示，成功的数字化运营体系能让银行整体运营效率和业务表现显著提升，客户流失率下降25%；</span><span>净推荐值可上升8-10，同时，银行销售生产率也随之提高。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>工商银行在数字化运营方面不断探索建设，深度运用人工智能和大数据，打造贯穿前中后台的营销“智慧大脑”，通过建立内容库标签体系和迭代检索算法实现更精准、更全面、更贴心的搜索服务，基于用户行为数据，将金融与生活场景相融合，为客户提供一站式服务。并通过智慧营销策略体系，实现不同渠道的服务信息更精准触达客户。此外，工商银行全面推进智慧运营数字化转型，构建了人机协作的智慧运营体系，利用物联网与人工智能等技术，实现作业无人化、实物处理自动化、业务管理智能化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>浦发银行推出零售经营新体系，加码智慧零售。该体系强调以用户为核心，通过广泛的“生态链接”以“非接触式服务模式”让客户触发交易，实现经营新模式。通过科技赋能，让零售业务的每个环节数字化，提升体验；通过开放互联，广泛融入各个场景，从被动获客到主动出击，从追求客户流量，到深度挖掘客户价值。同时，通过整合客户金融交易行为、关系圈、生活平台及场景化数据等，建立7大类、32子类、2000余项零售客户标签，构建360度全景客户画像，实现对客户需求及特性的深度洞察。</span><span></span></p>

<p data-darkmode-color-16281471254889="rgb(163, 163, 163)" data-darkmode-original-color-16281471254889="#fff|rgb(0, 0, 0)" data-darkmode-bgcolor-16281471254889="rgb(188, 190, 193)" data-darkmode-original-bgcolor-16281471254889="#fff|rgb(246, 249, 253)"><strong><span>在数字化运营方面持续发力</span></strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>北京银行坚持以数字化转型统领“发展模式、业务结构、客户结构、运营能力、管理方式”五大转型，推动本行实现高质量发展。</span><span>这其中</span><span>，作为数字运营板块的业务载体，数字化运营体系“智策”占有一席之地。</span><span>目前，银行零售业务发展逐渐转向场景经营和存量客户运营，随着银行客户加速向线上迁移，线上渠道已成为银行与用户接触的关键载体，因此，依托手机银行APP，打造数字化运营体系成为北京银行构建新型竞争力的重要方向。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>北京银行以数据驱动，重塑运营模式</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“智策”体系的构建是北京银行“以数连接、由数驱动、用数重塑”的数字化理念的集中体现，它以坚实的底层数据为依托，实现了运营全流程线上化、自动化与智能化。在“智策”体系下，运营人员可精准服务客户，通过多样化的方式高效触达客户，为用户精准推荐匹配产品/服务/活动，实现千人千面的服务。“智策”体系以数字化的方式，有效促进了北京银行数字化拓客与流量运营能力提升，进一步拉近与用户的距离，实现MAU与AUM双提升。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>北京银行全面赋能，实现降本增效</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“智策”体系目前已在北京银行手机银行</span><span>APP</span><span>和京</span><span>e</span><span>贷两大场景成功应用，深度触达百万级用户，并利用技术模型，对手机银行</span><span>APP</span><span>和京</span><span>e</span><span>贷的客户旅程进行细分，不断向产品侧输送流量，促进业务开展。</span><span>此外，“智策”体系的运行，不仅适用于线上渠道，也可应用于全部零售业务板块，全面支撑网点经营和远程银行空中经营。</span><span>北京银行充分发挥“智策”体系提效率、增产能、降成本、省人工的优势，集结线上渠道、线下网点与远程银行三方力量，形成覆盖全渠道、全媒体、全客户的立体化运营矩阵。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>易观分析认为，打造数字化运营体系仍是银行数字化转型的关键，需深度挖掘不同业务场景下的目标客户，依托数据与科技赋能银行业务体系，加强数据的收集和应用，构建数字化运营体系，最终实现提升客户体验、增加客户黏性及活跃度的目标。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这一趋势下，北京银行始终坚守“以客户为中心”的客观规律，“智策”零售数字化运营体系建设致力于实现全触点全场景一体化高效运营，在不同场景不同触点内可根据营销策略完成销售或服务，同时，跨场景跨触点实现营销服务无断点承接，打造全渠道一致性体验，用真诚的服务和精细的运营，提升客户口碑。未来，北京银行持续推动“智策”体系升级迭代，全面赋能零售业务转型发展，着力推动数字化转型向更深层次发展，跑出“数字京行”加速度。</span></p>
