<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前段时间，一把伞火了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为啥一把伞火了呢？其一，这把伞是<a href="https://www.niaogebiji.com/tag/GUCCI" target="_blank">GUCCI</a>与Adidas的联名产品；其二，作为一把“雨伞”，居然不防水，还要命的售价高达一万一。。。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/59036372862bf007650fdd0.15446587.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/12687131162bf0076d96203.52297090.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>咱来推算一下，已知雨就是水，水就是雨，由此可得，雨伞不防水，就是雨伞不防雨！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>雨伞都不防水了，那不跟皇帝的新衣一样吗？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且雨伞不防雨，那它用来防什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/174357791962bf007729d561.46268884.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“防穷人”我真的会笑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，网友这话话糙理不糙，</span><span><strong><a href="https://www.niaogebiji.com/tag/%E5%A5%A2%E4%BE%88%E5%93%81" target="_blank">奢侈品</a>牌防穷人，并且从不坑穷人</strong></span><span>。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>01</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们在挑选品牌的时候，品牌也在筛选消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在18年的时候，Gucci真就出过一件“皇帝的新衣”，更准确来说，是</span><strong><span>“皇帝的泳衣”</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/24513124462bf00774fc609.73589469.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这件泳衣，GUCCI官网给出的建议中有一条写道：<strong>“避免接触氯”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>又是众所周知的事情，公共泳池都会含有“氯”，用以消毒杀菌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是当时这件泳衣以</span><strong><span>“3300元买的gucci泳衣被警告不能碰水？”</span></strong><span>为标题，被不断刷屏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/58930228562bf0077747ac7.26016253.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，这件泳衣其实并不是用来当传统概念里的“泳衣”下水穿的，而是作为时尚单品穿出门的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/167958317762bf0077a611c7.90069037.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>泳装风在国外已经流行很久了，宣美就曾穿着Gucci的这款泳衣装进行舞台表演。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/106415181062bf0077ce4b94.03784664.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>别问怎么上厕所，问就是我也不知道。。。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，正因为“泳衣”以穿搭作用为主，它的下水实用性也就变弱了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们都知道，产品的价值是满足消费者需求，对于大部分产品，大众消费者依旧保持着等价交换的消费理念，这些奢侈品产品以迎合品牌核心用户的审美和穿搭需求为出发点，基本不符合大众购物时的“高性价比”的需求，从而防住了大众消费用户。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>02</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>奢侈品牌不坑穷人，不止在产品层面上，也在策略上。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对奢侈品有所关注的人应该都知道，LV近一年来有过五次提价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年2月初开始LV就有过一次大提价，2月16日路易威登在全球范围内提价，涉及皮具、时尚配饰和香水等品类。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>官方价格显示，大部分LV品类的涨幅在10%至15%，部分经典款式的价格涨幅甚至高达20%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实不止是LV，再早前的香奈儿，包括LVMH集团旗下的所有品牌都在涨价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与其他品牌通过降低产品价格，刺激并打开更大的消费市场以应对疫情冲击的策略不同，奢侈品牌相反地选择了提高价格，这样做的考虑，一方面是通过提价来提升消费门槛，获取高利润，并稳定与提升自己品牌的高端奢侈品的定位与调性，进一步满足核心消费客户的心理满足；另一方面是，通过价格门槛来重新构建客户群体，大部分奢侈品牌连从前吸引入门消费者的入门级包包都进行了提价，这意味着奢侈品牌抛弃了那些低消费的群体，将品牌目标用户缩小抓牢。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它的套路是：</span><span><strong>厚利寡销</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当下在一波波的裁员潮中，群体间的收入差距更加大，富的更富，穷的更穷，高端消费的群体购买欲提升，中层消费群体开始萎缩。</span><span><strong>为了应对疫情，奢侈品会持续涨价，发力高端线来满足头部客户需求，放弃入门级客户群体。</strong></span></p>
