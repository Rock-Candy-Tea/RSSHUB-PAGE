<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>凌晨时分，“东方甄选”直播间在“英文小课堂”中结束直播。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>千里之外的河南郑州，曾是教培老师的陈娟雅妮也走出了直播间，结束一天带货主播的工作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一年以前的他们或许都不曾想到，自己会从一份教书育人的职业跨行到直播间卖货。<strong>随着“双减”政策落地与直播电商的浪潮，已经有不少老师转型成为带货主播，还有人选择在TikTok等海外平台上用英文来带货。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“东方甄选”的爆红，似乎向市</span><span>场验证着老师转型主播这条路径的可行性。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a data-miniprogram-appid="wx55778a0d7b3048c4" data-miniprogram-path="pages/NewLive/index" data-miniprogram-nickname="新抖数据" data-miniprogram-type="image" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined"><img src="https://ngbjimg.xy599.com/64420323062b7ea724e9791.26435512.png" title="董宇辉火了，其他转型做主播的教培老们怎么样了？" alt="董宇辉火了，其他转型做主播的教培老们怎么样了？" referrerpolicy="no-referrer"></a></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>新抖数据-声量查询“董宇辉”</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新抖数据，“东方甄选”创下了<strong>一个月带货超4.4亿的销售额</strong>，直播间里娓娓道来、引经据典的董宇辉<strong>一周涨粉超过400万，在抖音的声量达到日均获赞212万以上</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>甚至有直播机构开始专门招聘有老师从业经历的主播，这和半年前一些公司招人特意标注“不接受教培行业经验”的态度大相径庭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/103518920962b7ea727fdb89.33450438.jpeg" title="董宇辉火了，其他转型做主播的教培老们怎么样了？" alt="董宇辉火了，其他转型做主播的教培老们怎么样了？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>左：东方甄选招聘介绍 右：某直播机构招聘介绍</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经年累月训练出的表达能力和语言能力，让老师们带起货来游刃有余，但两种职业仍有本质上的不同。老师主职是传道授业解惑、作息相对规律，而主播工作本质是销售、还要常常昼夜颠倒，<strong>因此转型过程中难免有“水土不服”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>能成为教培老师的下一站归宿吗？哪些老师更适合做主播？转型过程又会遇到哪些困难和挑战？我们和一些转型直播带货的前教培老师们聊了聊。</span><span><br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" title="董宇辉火了，其他转型做主播的教培老们怎么样了？" alt="董宇辉火了，其他转型做主播的教培老们怎么样了？" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>从讲台到直播间，教培老师的转型之路</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年12月，在<a href="https://www.niaogebiji.com/tag/%E6%96%B0%E4%B8%9C%E6%96%B9" target="_blank">新东方</a>宣布转型直播带货的同一个月，陈娟雅妮也踏上了抖音带货主播之路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这不是她第一次转变人生方向。毕业后她做过文案策划、活动执行，后来进入教培行业，做了两年的小升初辅导老师，就遇到了“双减”的裁员大潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>陈娟雅妮的第一次带货直播是卖大枣，在线只有十几个人，但她至今都忘不了当时的紧张感，整个人手足无措，还要面对很多突发情况，比如违禁词、突然黑屏等等。下播的那一刻，她仿佛才找回自己，如释重负。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>好在两年的讲课经验，让她很快适应了需要面对一群观众讲解产品的场景。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“老师和主播都是靠嘴皮子，因为带过学生，也要应对很多突然情况，有讲课的经历所以不会怯场，直播状态比较自然，不会僵硬和尴尬，逐渐就驾轻就熟了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但压力和焦虑并没有就此减少。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是账号数据起不来的时候，陈娟雅妮会被挫败感淹没，也开始怀疑自己天天忙碌的意义是什么。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/35953750262b7ea72ae37f1.35669661.jpeg" title="董宇辉火了，其他转型做主播的教培老们怎么样了？" alt="董宇辉火了，其他转型做主播的教培老们怎么样了？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>陈娟雅妮在小红书记录工作感受</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>转型之路并不好走，选择到TikTok直播的Joanna也面临相似的处境，“<strong>甚至有点儿担心，如果这个行业也垮了怎么办</strong>”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Joanna从2018年进入教培行业，先后在海南、成都等地做过英语老师。今年年初，所在的教培机构倒闭后，她想到自己大学专业是电子商务，又有英文优势，决定到TikTok试水直播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“<strong>大家都说TikTok是风口，就像2003年的淘宝和2018年的抖音，</strong>其实我也知道很多口号都是卖课的宣传，但我也要试试。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真正开始做起来之后，Joanna发现TikTok出单并不是件容易的事情，市面上宣传的一场成交2000磅的太少了，大多数直播间的客单价还不到1英镑，每个从业者都在“摸着石头过河”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“在绝望中寻找希望，人生终将辉煌”，俞敏洪流传在江湖上的金句，也激励了转型中的教培从业者。</span></strong><span></span><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span></span></span><span><span></span></span><span><span><span>陈娟雅妮在逐渐适应直播的节奏以后，不到一个月便迎来</span></span></span><span>第一场业绩小高峰，</span><span>两小时卖了两万元左右的零食，这对于新账号和新人主播，算得上不错的成绩，“能做的就是努力撑住，一秒钟都不停地介绍产品”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有一些教培老师，虽然没有自己成为带货主播，但也切入了直播带货赛道，做起了培训或是孵化生意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Shirly是英语专业出身，曾在新东方任职英语老师。2022年2月，<strong>因为看好跨境电商的市场潜力，她选择创业做TikTok培训和跨境商家孵化。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在Shirly看来，自己其实并没有完全脱离教育行业，只是从学科教育转做了成人教育。她的工作以招商和培训为主，孵化英语口语好的主播，“我预计今年8月TikTok会迎来小爆发，2023年可能会全面爆发”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" title="董宇辉火了，其他转型做主播的教培老们怎么样了？" alt="董宇辉火了，其他转型做主播的教培老们怎么样了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>“好老师不一定是好主播”</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做得了好老师，就能成为一名好主播吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种带着些不确定的自我怀疑，横亘在每一位转型的教培老师心头。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Joanna大部分前同事，从教培机构离职后，<strong>首选是考体制内带编制的老师或者公务员，也有一些人转行转岗去外企和国企，愿意转型做主播的非常少。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/113238055562b7ea72e84d42.01138499.png" title="董宇辉火了，其他转型做主播的教培老们怎么样了？" alt="董宇辉火了，其他转型做主播的教培老们怎么样了？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>智联招聘《2021教培行业人才市场分析报告》</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老师和主播的相似之处在于，都需要在台前表达。<strong>可迁移的表达能力、临场发挥和控场能力是很多教培老师转型主播的原因。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但同时，他们的转型也有许多门槛限制，从要不要做、能不能做到会不会做。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>心态是第一道门槛，只有愿意做销售才有可能做好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“东方甄选”的主播们常常用英文自我介绍说：<strong>我们以前是新东方的老师，现在是一名网络销售员。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>董宇辉在视频中透露，转型初期他也曾无比迷茫，差点就要签离职了，在新东方在线CEO孙东旭“我们总会找到出路”的鼓励下，他留了下来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他曾问自己的奶奶：“如果有一天我变成卖菜的你怎么看？”奶奶说，“挺好，只要你健康就行”，从此他心中的负担和包袱终于彻底卸下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而陈娟雅妮之所以能迅速适应角色，是因为她曾有跟随父母卖东西的经验，并不抗拒销售的工作，甚至有些喜欢。她觉得，无论做主播还是老师，本质都是打工人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>心理素质是第二道槛。</span></strong><span>从三尺讲堂到直播间，从受人尊敬的老师到被千万网友围观点评的主播，心理落差是难免的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Shirly很能理解这种落差感。她曾任职的新东方，<strong>因为课程顾问的存在，老师不需要做销售工作</strong>，只需要做好教学，“上课有学生认真听讲，但直播有可能再卖力气也卖不出去，这就是差别”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205953886562b7ea732848a4.95156173.png" title="董宇辉火了，其他转型做主播的教培老们怎么样了？" alt="董宇辉火了，其他转型做主播的教培老们怎么样了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更不用说有些网友东西不买、还会有不礼貌的攻击发言。董宇辉和“东方甄选”的另一位主播七七，都在早期被不友好的网友攻击过长相，这是做老师的时候几乎不会发生的事情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“做老师和主播最大的区别在于受众不同，直播可能会遇到变态的客人，不会有老师那种被人尊重的感觉”，Joanna对此也感同身受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三道槛是镜头感，有人在线下可以侃侃而谈，到线上却会因“镜头恐惧”而发挥失常。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做自媒体是锻炼镜头感的方法之一。陈娟雅妮在工作之余经营起小红书账号，发布了64篇笔记，积累了2600多个粉丝。虽然账号体量不大，但视频里的她十分有感染力，唠嗑式讲述着自己的工作经历。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据深网报道，新东方员工透露，<strong>“（东方甄选）遴选主播的门槛很高，不仅对形象、性格、 语言表达有要求，还要求有一定的自媒体经验积累。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果没做过自媒体，其实网课的经历也能锻炼老师的镜头感。近期大火的董宇辉、顿顿、yoyo和七七等主播，都曾是新东方在线的老师，有不少网课经验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在Shirly看来，网课和传统线下的老师是有区别的，“网课老师转型主播，在镜头感和互动性上会更适应。因为网上隔着屏幕，只能通过评论区互动，反馈是延迟的，和线下授课不一样”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" title="董宇辉火了，其他转型做主播的教培老们怎么样了？" alt="董宇辉火了，其他转型做主播的教培老们怎么样了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>昼夜颠倒、薪资无涨幅，老师们愿意做多久的带货主播？</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管直播带货事业已经逐渐走上正轨，但陈娟雅妮并没有长期做主播的打算，她有想过之后可能会去做幕后运营等工作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>工作节奏和身体状态</span></strong><span>是她最在意的，教培老师们多在白天上课，而主播常常需要昼夜颠倒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“直播带货是拼流量的工作，而各大平台的流量高峰期都在晚上甚至三更半夜，目前直播是24小时排班，两小时一换班，最晚的班在凌晨1点。两小时不停地保持情绪亢奋地说话，很容易疲惫和心累，嗓子问题会有出现声带结节等问题”，陈娟雅妮告诉新榜编辑部。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> <img src="https://ngbjimg.xy599.com/5779051462b7ea7378fa82.23827228.jpeg" title="董宇辉火了，其他转型做主播的教培老们怎么样了？" alt="董宇辉火了，其他转型做主播的教培老们怎么样了？" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做海外直播则需要配合当地的时区，做英国站的Joanna直播时间在下午一点到五点，刚好是英国夏令时的晚上八点到十二点，“最近数据不是很好，老板想让我测试一下晚场，比如晚上五点到十点开播”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>某种程度上，主播是一碗“青春饭”，要有足够的体力和网感，跟得上最新的玩法，但老师则会随着年纪的增长和经验的沉淀而更被信任和尊重。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Joanna觉得，互联网的变化实在太快了，直播带货的玩法一天一个样，而老师只需要在固定的知识上面推陈出新，这是两种职业底层逻辑的不同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在做英文主播之前，她曾有过创业的念头，“还好我没冲动创业，现在我刚好可以学习，再观望一下，把跨境电商这条路跑顺以后，再考虑把它当作副业还是创业”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>和外界想象中两个行业的普遍高薪不同，大部分老师转型前后的薪资没有太大的波动。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，“北上广杭”动辄高薪挖人的“繁荣”只是少数情况，教培老师和带货主播在非一二线城市的底薪一般在几千元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>陈娟雅妮居住在河南，她透露无论是做教培老师还是做主播，底薪都在四五千左右，做老师时绩效奖金看课时费，做直播时提成根据产品和每月销售额情况，卖十万块能拿到600块钱提成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在教培行业工作三年多的Joanna说：“工资上万的时候屈指可数，绩效好的时候才能过万，现在做海外主播底薪也是几千，提成寥寥无几。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/124868700862b7ea73a49fc3.85611136.jpeg" title="董宇辉火了，其他转型做主播的教培老们怎么样了？" alt="董宇辉火了，其他转型做主播的教培老们怎么样了？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：小红书用户截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>话虽如此，<strong>但在新东方模式被认可后，依旧挡不住许多教培老师想跃跃欲试转型带货主播</strong>。在豆瓣、小红书上，有不少教培老师分享自己转型找到主播工作的帖子，吸引了很多人跟帖询问具体工作内容和待遇情况。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这种大背景下，Shirly创业的主播培训孵化项目也许会有更大的市场和机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月20日，俞敏洪也在一场直播中透露，未来可能会开新东方电商学院。他认为做电商直播的网红、主播，都需要一种文化培训，以此提高主播的整体水平层次，如果成立了，他会亲自去上课，也会让董宇辉等著名的主播去上课。曾经的老师们也许会以另一种方式回归。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播圈里的“新东方”正在悄悄成长，教培老师中也许真的会诞生下一个直播天王。</span><span></span></p>
