<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E6%97%A0%E5%8D%B0%E8%89%AF%E5%93%81" target="_blank">无印良品</a><span>（<a href="https://www.niaogebiji.com/tag/MUJI" target="_blank">MUJI</a>）</span>在中国已经被唱衰了很多年，但其实到今天，我们还是很难给出一个准确的答案：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>MUJI，颓败与否？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>站在行业及企业经营层面，尤其是过去两年间，这家来自日本的老牌家居杂货品牌，面临着利润率不升反降的难题，海外业务里，美国地区甚至申请了破产保护。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为仅次于日本本土的第二大超级市场，中国为无印良品的出海贡献了最多的营收，且在它收缩之际，还能保持新店开业的节奏，但毫无疑问，速度慢下来了，销售额也远不如预期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>颓败是一个相对的话题。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>需要正视的是，无印良品的拥趸们还很坚挺。正在进行中的MUJI夏日折扣季，引来不少消费者冲进店里，在周末挤满收银柜台，他们用五折甚至更少的价格拿下好物，为极简设计的信仰充值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你注意到的话——在线上二手交易平台，就连无印良品售价一元的包装纸袋，都可以溢价卖出，文案是“送礼的话，更上档次”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2005年首次进入中国内地算起，无印良品已经走过17年。因为不同的认知和消费习惯，在国内，更多人视其为服装杂货品牌，站在服装这个赛道看，意味着它的相对稳固已属不易。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新零售商业评论认为，眼下，无印良品本身的业绩之外，更大的难题，其实是如何应对一个消费水平整体下行的大环境。换言之，</span><strong><span>它曾经完成了对一代人的审美教育，这代人却在今天趋于冷静了。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是更多零售企业的难题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/65581963062fbc1b3244395.94240042.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><span> </span></span><span><strong><span>敏感点</span></strong></span><span><span> </span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无印良品的折扣季可以有多疯狂？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在小红书搜索“MUJI”，你将会得到一堆“真香”的打折攻略。号称内部员工的博主，发一张无印良品的工牌，就能引来上百人探秘内部价、问兼职。门店某款产品有货无货的消息，在这里完成一部分传播，着急的人留条“即刻动身”的讯息就不知所踪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一款薄款不沾水小挎包，部分颜色已经全网难求，快人一步的消费者，大都一次五件、十件的囤货。有成功的用户分享心得，“各种颜色分别来一个，还可以送给家人朋友”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>按照惯例，无印良品每年会进行两次大促，分别是冬夏两季，多是为新品上市作准备，以及去一下库存压力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/43309590462fbc1b3621274.05270258.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品本身打折，支付宝会员优惠券，以及特定银行信用卡支付优惠等，多重折扣叠加，以低于五折的价格拿下部分产品不是难事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新零售商业评论曾在一个工作日的午后探访附近一家无印良品门店，包括整个商场在内，稍显冷清，但进店顾客大都目标明确，直奔折扣区。单人收银结账时间变长，大多是工作人员在帮忙操作优惠券码。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，对于无印良品来说，这样的消费热潮并不多见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国内消费者习惯将它与同样来自日本的快时尚品牌优衣库作比较，在服装杂货品类，因为同样主打舒适、自由与极简，二者成为日系风格的典型代表。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两大企业最大的海外市场同在中国，又都在千禧年后不久开辟中国市场，但显然，如今的差距越来越大。目前，优衣库国内的门店数量已经逼近900家，无印良品则是299家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>企业规模、本身影响力，以及所处的细分领域不同，自然不能用这个单一维度去衡量，但不可否认，相较优衣库在国人心中的接受度，无印良品还是温吞的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个重要的原因是，定位，或者说，价格。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便在几年前，互联网上也有“优衣库为什么在中国卖得更贵”的发问。在《衣服之王：优衣库掌门人柳井正》一书中，作者张斌记录了这家日企的彷徨：</span><span></span></p>
<section>
<section>
<section data-width="65%"><span>“</span></section>
<section>
<section><span>初入华，优衣库摸不准定位，还赶上了非典，几家门店关门的关门，停滞的停滞。很快，原中国区负责人林诚辞职，潘宁接手，后者认为，国内的优衣库没有在日本的国民度，短时间依靠下调价格也无法普及，还无法兼顾利润率和商品质量。最终，潘宁决定，将目标用户定位在月薪5000元以上的中产阶级。</span></section>
</section>
<section data-width="65%"><span>”</span></section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到今天，随着国内人均收入上涨，消费水平提升，及优衣库本身的策略调整，这家曾经被指责的品牌，已</span><span>经</span><span>成功在质量与价格间取得平衡。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/106255629462fbc1b399dc65.99049286.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>无印良品还在“端着”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不止一位消费者——广义上的中产，向新零售商业评论吐槽，“太贵了”，甚至有的还会兴冲冲走进门店，最终却只舍得下手买点零食出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位MUJI粉告诉新零售商业评论，她家的夏凉被、褥垫等都购自无印良品，另外，感觉衣服质量也比优衣库好很多，“但还是打折的时候买的多，原价真的贵”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>迄今，无印良品依然没能解决的问题是，相较在日本本土平价、日常的定位，国内的商品售价要高不少，甚至比肩轻奢，也因此，即便是会为无品牌设计买单的一部分人，也难以接受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当无印良品的“冤大头”确实会引发共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾有消费者在社交媒体分享，在无印良品没看价格随手拿了一个发圈，结账才知道售价32元，不好意思放回去，只能暗自肉疼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公司在2021年报中谈及中国区经营现状时，不无担心地写道：近年来，业绩增长步伐放缓，主要是对于日常商品来说，价格偏高，部分商品的销售额偏低。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>无印良品什么都知道。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/64709651162fbc1b3d67b85.97923267.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 变形记 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2005年，杭州利星名品广场开业，主要面向小资、白领、海归和外籍人士等目标群体。在接受联商网采访时，其负责人谈到，希望拿下无印良品浙江首店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无印良品中国区负责人来考察时，安下心来。负责人说，附近西湖的游客，高端酒店的商务客人，中国美院的学生将会是重要顾客。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众所周知，无印良品直译过来是“无品牌标记的优质商品”。它创立于上世纪80年代初，时任日本西武集团总裁的堤清二注意到，那时的日本产品设计大多强调品牌识别，在与田中一光等设计师朋友交流后，提出了“反品牌”的想法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/95662826362fbc1b429dd90.90353337.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>设计初步落地在家庭用品与食品，设计师之一的小池一子最早提出来的slogan还是“有理由的便宜”。在日本，它的定价也的确更亲民。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>受消费水平差异、品牌定位需求、关税等因素影响，外来品牌入华总有两副面孔，从快餐，到服装，不外如是。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>客观因素限制之外，更重要的是，就如优衣库寻找目标受众一般，<strong>无印良品也需要在本土之外，找到合适的定位。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据国泰君安证券相关报告，2017年，中国人均GDP为8827美元，按人均GDP衡量，这时的中国经济发展水平与1980年的日本相似。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一个处于混沌审美期的社会环境里，反品牌、极简、自然，显然最能抓住拥有一定消费实力、追求低调质朴的中产，当然，还有不拘一格，也爱为品味买单的文艺青年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当前，线下实体零售面临的共同问题是，疫情叠加行业的整体衰败与竞争，企业大都难熬。服装、杂货尤其如此。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无印良品近些年的业绩不好看。根据公司2022年上半年财报，尽管其营收同比增长了3.6%，净利润却下滑超10%，无论日本本土还是最大的海外市场中国，都远低于预期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>打折去库存成了常态。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在2021年报中，无印良品曾坦诚指出，中国市场的频繁打折，是影响毛利下降的因素之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但新零售商业评论认为，某种程度上说，无印良品今时的矛盾尴尬，也有另一重意味上的成功。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你很难通过同穿一件优衣库而辨别同好，但两个人同样喜欢MUJI，极大可能会有相同的兴趣爱好——<strong>作为一种生活方式的售卖者，</strong>很难否认，<strong>无印良品培育下的市场基础，仍旧在发挥能量，在可预见的范围内，它还不会消散。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/38215415662fbc1b4837e63.13642701.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><em><span>图源日剧《风平浪静的闲暇》</span></em></span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一种观点认为，淘宝、1688、拼多多、甚至名创优品的崛起，打破了无印良品的壁垒，消费者能够以更低的价格拿下相似的货品，对这家公司带来的冲击是巨大的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不久前，无印良品中国首席市场执行官邵恬宜在接受搜狐财经采访时谈到这个话题。邵恬宜解释，不同竞品出现意味着行业活力，也会更加激发无印良品去创新、自我提升。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她认为，MUJI在国内依然拥有三大核心能力：一、品类齐全，7000多个SKU，消费者有一站式购齐的便利；二、品质保证；三、设计能力，不止体现在外观，还有细节。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这意味着，<strong>它拥有拴住忠实拥趸的能力，以及维系自身的象征意义。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你会发现，穿衣风格划分同类之外，在小红书、抖音等内容平台，职业的家居博主们会摆满MUJI正品收纳盒、香薰和挂钟，而不是轻易向仿品妥协。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“设计本身是值钱的，值多少钱要看多少人喜欢这种设计风格，很多人觉得只有材料值钱，所以才会有盗版横行，抄来抄去的现象出现，对长久的业态并非好事。”一名网友在无印良品相关内容的评论区写道。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 结 语 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>翻阅无印良品2021年报，可以看到，这家公司未来十年的规划仍是加快开店速度。截至2021年8月，无印良品在全球32个国家和地区共有1068家店铺，2030年的目标是2500家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>中国的扩张速度变慢，但仍重要。计划中，日本本土将维持每年100家新店的节奏，单中国大陆地区要保持在50家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>咨询公司埃森哲近日发布的《2022中国消费者洞察》报告中提到：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去40年间，中国消费市场蓬勃发展，2014年起，消费连续6年成为拉动中国经济增长的第一动力。但随着居民可支配收入增长的放缓，消费性支出有所萎缩。2020年，居民消费倾向降至历史低点62%，2021年回升至64%，但仍低于疫情前水平。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“不是不买MUJI了，而是我现在什么都不想买。”一位90后朋友表示，她喜欢无印良品，但不消费更多出于消费欲望低下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无印良品近两年在国内的动作多是跨界，在上海虹口区合作京东七鲜，打造饮食提案型超市，源自本身“生活提案型”的定位。在北京，开设MUJI酒店，临近天安门和胡同，融入本地文化。在日本，携手本田做极简电动车。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span draggable="true">我们得承认，生活方式品牌，就是有这样顺理成章的扩张可能性。</span></p>
