<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>私域在2022年依然非常地火，随着私域一起火还有各家的</span><strong><a href="https://www.niaogebiji.com/tag/%E4%BC%9A%E5%91%98%E4%BD%93%E7%B3%BB" target="_blank">会员体系</a></strong><span>，各家开始用会员体系</span><strong>圈住在私域内最忠诚的用户，希望用户不断产生复购，真正实现从用户到重视会员用户的跨越。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今天就来和大家聊聊付费会员体系，希望能给目前想做付费会员体系的朋友一点思考。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本篇文章会从以下几个大模块去和大家分享付费会员体系，每个大的模块会结合一些案例和自己的思考去和大家分享：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>什么是会员体系</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>如何设计付费会员体系</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>如何进行付费会员的售卖和续费</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>如何进行付费会员的运营</strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>一、什么是会员体系</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么首先来和大家介绍下，什么是会员体系，会员体系的本质是什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们先来看下什么是会员体系？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>百度百科上是这么解释会员体系的：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>将会员体系等同于QQ的会员等级体系，按照会员的忠诚度的高低分为了5个级别。不同的会员级别享受不同的特权和服务，这大概就是会员体系最初的样子。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/121510332662ce37c66eb501.40364607.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个解释有一个非常关键的点，就是用户分层。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，在我看来，会员体系就是根据<strong>用户的综合行为/某种行为将用户划分为不同的用户群体，本质是用户精细化运营的一种方式。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管这个动作是付费也好，还是说用户在平台上的其他综合行为，其实本质上都是一样的，只是划分的维度不同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以按照这个解释，我们能看到市面上2种不同的会员体系：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>一是免费成长体系会员</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>二是付费会员</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/194195891062ce37c698aeb7.39607941.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>免费成长体系会员，就是根据用户在平台上的综合行为（一般是和产品变现相关）赋予成长值和等级。典型的就是如，携程的成长体系会员，按照用户在各个业务线成功下单的金额给予成长值，然后用户能够不断成长升级，成为高等级的尊贵会员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里有个好判断是否是这个体系的一个技巧，就是看是否一注册就拥有了一个会员身份。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>付费会员，就更好区分了，看是否付费，就是了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另外，根据付费会员的不同属性和目的，可以划分为不同类型的付费会员：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>订阅式会员，典型的如爱优腾，在会员身份期间可以免费看电影和去广告等等；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>高价值权益卡省钱式，如京东PLUS会员；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>低价值券包省钱式，如拼多多的省钱卡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般来说，这两种体系，在一些大的产品上，都会进行混合使用，看谁为谁服务，一般都是哪个会员体系占主导。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，淘宝的88会员就是典型的付费会员为免费会员服务的会员，用户必须达到1000淘气值，才可以88元开通。这种体系，典型的特征就是不追求付费会员的高速用户增长，只是想办法为高等级的用户提供更好更优的服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而我们也可以看到京东的PLUS会员，就是免费会员为付费会员服务的混合式会员。追求付费会员的增长，同时免费会员的服务性偏弱，利用免费会员作为筛选用户付费的门槛。一般来说，京享值达到xxx，可获得50元京东PLUS会员优惠券。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里就给大家做一个简单地介绍，不深入讲解了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>核心还是放在怎么做付费会员上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那接下来，我们来说下如何设计付费会员体系。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>二、如何设计付费会员体系</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>设计付费会员，主要会分为3部分：其中1和3是非常非常关键的！！！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一是如何设计勾人的权益？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二是会员周期如何设计？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三是如何进行付费会员的售卖</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，对于付费会员来说，权益是非常非常关键的内容，我们买会员也是优先看权益有哪些东西然后再看性价比是否OK，所以在权益这块，会和大家重点分享。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，大家在阅读的时候，也可以思考下，如果是你，你会怎么做自己家产品的付费会员，或者说，你会怎么做哪款你最喜欢产品的付费会员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、如何设计勾人的权益</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先进入如何设计付费会员的第一趴：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何设计勾人的权益。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我总结了一个设计付费会员权益的规则：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>目标用户+用户场景+用户痛点=会员的定位</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>针对哪部分的用户满足他们在什么场景下的什么痛点。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>而会员的定位的话，就是等于解决这部分用户的痛点的权益合集。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如爱奇艺会员的定位，针对追求观看品质的用户解决用户看广告，内容质量偏低的痛点。所以爱奇艺的核心权益是，去广告，优质内容提前看/免费看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>京东PLUS会员的定位，针对高价值的购物用户解决用户在购物过程中，解决购物贵，不想出邮费的痛点，让用户购物爽。宁愿花200块钱买一个东西，也不愿意花180+15块钱的邮费，这是大部分购物者的常态。所以京东PLUS会员的核心权益是运费券+PLUS价，让用户在购物的过程中感觉省钱并且购物也没有运费的顾虑了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家日常去梳理自己产品的用户痛点的时候，可以按照以下方式进行梳理：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/47071677862ce37c6c80b70.64347798.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户主流程的场景；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>场景下有什么痛点；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们可以用什么样的权益来解决用户的痛点；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是否可行，可能会存在什么样的问题等等，这样下来就会比较清晰和明确了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果自己想不出来的时候，也可以参考行业竞品或者其他行业的产品都是怎么设计的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单一句话来说，就是向内看用户场景和痛点，向外看竞品。脑洞下，滴滴会怎么做付费会员，下边是我自己的一些思考，大家也可以进行补充。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/68010911962ce37c706f705.48895823.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有朋友说，付费会员体系与等级会员体系并行，付费会员权益设计的差异点该围绕什么进行？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里不一定是需要差异点，也有可能是原来等级会员的升级。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先需要说的是，等级会员，不是所有的等级都有真正有价值的权益，一般来说，只有最高的等级权益价值才真正有点用，第二是不是所有的用户都能达到最高的等级会员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于用户来说，等级会员享受到有价值的权益，可能需要花一万多块钱才能享受到，但是免费会员，可能就是几十到一百多不等，就可以享受到比免费会员高等级更有价值的权益。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以会有这样的差异，基于免费会员其实是平台部分利润补贴的逻辑，最高等级用户每年可能能贡献1000块钱的收益，拿出其中10%来补贴用户，平均到每一个用户的成本很低很低，所以很难提供很多超值的权益。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而付费会员的逻辑是，用户花钱买服务，只要使用率控制得当，不会存在补贴用户的逻辑，每售卖一张会员卡还能挣钱，一举两得，即服务好了用户，又挣钱了。当然，并不是所有的付费会员体系都是挣钱的，关键是看负责人对付费会员体系的定位是什么？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、如何进行权益成本核算</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>权益列出来之后，就是计算成本的问题了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>运营离不开和数字打交道，更离不开计算成本的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们来看下该如何计算付费会员的权益成本：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>订阅式会员成本核算，如爱奇艺会员</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>爱奇艺会员的成本=用户某时间段的平均使用时长*时长广告成本=3*5=15元</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人可能会说，这里边为什么没有计算版权的成本。内容型的产品，就算是没有会员，它也会投入相应的版权费用，一旦不投入可能就会面领着用户流失的问题。所以没有将版权费用计算在会员成本里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>多权益省钱卡式会员核算，如京东PLUS会员</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>会员卡的单卡成本=权益1成本*使用率1+权益2成本*使用率2+.....</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>京东PLUS会员=人均消费金额*京豆的返还比例（0.5%-2%）*京豆使用率+360元运费券*使用率（18%）=10000*1%*50%+360*18%=114.8元</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/23660817462ce37c74000e5.27674901.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做一点特别声明哈，以上数值，纯属虚构，如有雷同，纯属意外。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么问题来了，有朋友说，我的权益之前都没有，不知道权益的使用率怎么办？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以下三种方式也提供给大家参考，在不知道权益使用率的情况下，利用其他办法去进行预估。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1）通过目前已有的数据进行推测</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户月均观看/阅读时长，某权益的月使用率</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2）通过活动短期小型测试</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3）通过竞品调研，了解竞品的使用率</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实在不行，就少量灰度测试，来看权益的最终的使用率。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2、如何设计会员卡的周期</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>权益和大家分享完了，接下来，我们进入如何设计付费会员的第2趴-如何设计会员的周期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有的产品有月卡，季卡，年卡，有的产品只有年卡，在会员周期上设计有什么规律吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据产品的使用频率和产品的客单价，可以去设计付费会员的周期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般来说，客单价高，产品的使用频率低的产品，可以首推年卡，比较典型的就如携程的超级会员。携程的客单价至少在500元左右，使用频率的话大概是一个月或者一个季度一次，所以在这种产品里，先做会员是最划算的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，也有可能随着产品的不断发现，想要实现付费会员的用户增长或实现付费会员的利润最大化，也有可能慢慢增加季卡，月卡也是完全有可能的。主要还是看产品当前的目标是什么。是收益最大化，还是快速的用户增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>那么是否需要增加会员的种类？主要还是看运营目标和用户结构：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一、是否增加当前会员卡的SKU</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二、是否面向不同的目标用户增加另外一种付费产品的SKU</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40905302862ce37c7734c21.73270250.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管是进行付费会员体系权益的设计，还是设计会员周期，还是后边要讲的会员售卖，其实都离不开市场调研。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以在这里，也和大家简单地分享下，如何进行付费会员的调研。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>总体原则就是：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看同类型产品的设计思路</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看不同类型头部产品的权益设计逻辑</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们日常去进行竞品调研的时候，就可以按照以下的那个模板来进行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/159241631662ce37c79789f7.40885606.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>三、如何进行付费会员的售卖和续费</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总体来说，进行付费会员的售卖的原则就是：<strong>根据用户的场景和痛点，建立合适的场景/渠道触达用户，然后采取一系列的抓手转化用户，最终让用户付费。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这趴的内容，可以结合着第一趴权益的设计一起去看，更能串联起来。毕竟权益代表着内容，也代表着产品的核心卖点，所以这块结合权益去看，会更加有豁然开朗的感觉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先，再说售卖之前，我们先来思考下，作为用户，你为什么会买某个产品的会员。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我买会员，有1个核心的关键，就是看我能不能回本。本单能不能回本，以及以后的预期能不能回本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所以这里的关键就是，让用户感知价值远远＞用户付出成本，用户购买</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>让用户感觉到省钱，赚到钱的感觉</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何才能让用户感觉到省钱和赚钱的感觉呢，我们来看下基于这个原则，在付费会员的落地上上可以怎么做：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>突出单个权益以及单个权益的价值，让用户感觉到这些权益都是自己能够用得上，真正能够省钱的，也就是说自己买的有用。突出某一类权益的集合，用户不一定明白，就算明白，也是需要一定的反应时间的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>突出核心主权益，而不是所有权益，原理同上。28法则，只需要突出20%的主权益集合。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>突出省钱金额/预计省钱金额，按照单个权益的价值和次数计算出来，如果用户全用的情况下，会省钱多少，让用户感到赚到的感觉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>缩小金额，每天仅需xx元，降低用户的付出感，从而降低用户的决策成本。一年365元，听起来还挺多的，但是说每天仅需1块钱，就让用户觉得，好像价格还挺划算的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12316872262ce37c7cc2171.88013646.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说完了如何做付费会员的落地页之后，我们来说下，如何在业务主场景下进行去进行付费会员的搭售。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>核心一条原则就是：<strong>结合场景痛点+权益解决方案进行会员搭售</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开通立省xx元，一单回本，不回本退差价</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和主流程购买结合起来，不阻断主流程购买</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我很喜欢美团会员在主流程的购买。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直接在订单详情页，可以加购会员，本单立省xx元，用起来非常地顺滑，而且用户的操作成本低。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>京东在主流程的搭售做的就相对差一些，更准确地说，京东是把在主流程的展示和露出作为运营资源位去使用，用户需要单独购买会员后，才能使用权益。用起来，不是那么顺畅和丝滑。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/43765825362ce37c8068750.20898257.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总结下，我们该如何去进行付费会员的在产品内的售卖：<strong>根据用户核心行为的流程建立触点+触点转化策略</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，我们以京东PLUS会员为例，梳理用户在产品内的核心流程，然后再看该场景和流程下该如何去进行会员的售卖。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/57730878662ce37c84282f1.49446606.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前面也提到了转化策略，我们也来看下，付费会员一般都有哪些转化抓手。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/189698924462ce37c86f8562.38971253.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>100%省钱</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不回本退差价；即买即用，一单回本，毕竟大家买会员，核心还是希望能帮自己省钱，省的钱越多，未来续费的可能性也就越大。这也是为什么很多付费会员产品都非常注重省钱计算器的原因。通过省钱计算器给用户制造省钱的感知。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>开卡赠：开卡礼，额外加赠</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开卡赠xx元礼包，是提升付费会员转化常用的方式。通过额外加赠的方式，临门一脚，让用户快速做出购买的决策。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>开卡立减：优惠券立减</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>逻辑同上，通过用户厌恶损失的方式，促使用户快速转化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>体验会员+连续包：低价体验+连续包（月/年）</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小成本体验，从而让用户降低自己的决策成本，成为产品的忠实用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>上新：权益或内容上新</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>免费试用+先享后付</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>活动也是转化用户的重要抓手，怎么样才能去筹备一场比较成功的大促活动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1）确认活动的类型：降价（联合会员）促销，联名会员卡买1得多，买会员抽奖？？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2）确认活动的利益点，核算活动ROI：收入，成本？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3）申请触达资源位：哪些位置触达用户效果好，用什么样的利益点触达用户</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4）上线活动</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5）数据回收和复盘：资源位的点击和转化，活动的ROI，什么是不变的？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如说，通过做几次活动下来，发现价格是最吸引用户的抓手，什么价格才是相对最优的，通过做几次活动下来，都需要逐渐总结属于自己的东西。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们再来总结下，可以如何做付费会员的售卖？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>根据用户的场景和痛点，建立合适的场景/渠道触达用户，然后采取一系列的抓手转化用户，最终让用户付费。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这里边重要的三要素就是场景/痛点+场景/渠道+抓手</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说完了产品内的售卖，我们来看下，如何进行产品外的售卖，如果做的好的话，产品外的售卖其实也能带来不少的增量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>常见的有3种形式：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1）联合会员</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也就是我们常见的爱奇艺和京东PLUS会员的联合会员，淘宝的88VIP其实也是典型的联合会员的一种形式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>2）商务BD</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如企业之间的合作，资源置换等等模式</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>3）CPS分销</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>典型的如爱奇艺的分销，每年做活动的时候，我们都能看到有人在朋友圈分享推荐，这其实就是爱奇艺的CPS的一种模式，全民售卖，促进活动的销售量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说完了售卖之后，我们再来说下如何让用户续费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这两个的关键就像是，如何进行拉新，让用户购买售卖和如何让用户留存并完成复购是一样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让用户购买第一单，也有可能是用户上当受骗，中了运营的套路，但是如果用户续费，大概是用户真正对产品认可，真正觉得产品能够省钱才会复购。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以如何让用户续费，在付费会员体系里也非常关键。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么用户为什么会续费呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>用户会产生续费，一定是基于，我还需要+我之前用的爽+省钱感知。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我还需要，意味地我还正常地使用产品，并且在原来的痛点场景下仍然存在需求</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我之前用的爽，权益真正解决了我的痛点，使用流程上不会有太多的卡点和不知道怎么用的权益，并且有时候还给了我意外的惊喜。常见的有会员生日礼和会员关怀等，用户不会冲着这些权益买会员，但是如果有，会让用户感觉到产品特别贴心，久久不能忘怀。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>省钱感知，在续费里边非常关键，毕竟用户花钱买了一张会员卡，如果省钱感知上都没有做到回本，那么用户后续就很难真正续费了，毕竟谁不想做亏本的买卖。所以我们看到很多付费会员的产品都会放不少的精力在省钱计算器上，让用户感知到自己真正省钱，赚到了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/86563231462ce37c89ad1e4.53398190.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>确定了会员续费的3个关键要素之后，我们来看，如何针对这3个关键要素引导用户续费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，我们来看，用户还需要，所以一定要抓住用户在产品内活跃的关键时间节点，引导用户提前续费，避免用户流失。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据用户的续费时间分布，明确续费的时间节点，一般来说，用户在会员过期前的最后1-3次活跃，是最好触达和转化用户的节点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个时间节点，可以根据产品的使用频率而定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般来说，低频类的会员，一般是到期前1-3个月，高频类的会员，一般是到期前的一周左右。不同的产品会存在差异，主要还是根据自己产品的一个到期时间分布去看续费率的一个变化情况，当然，这个时间节点越往前越好，往后可能就会面临用户流失的问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14838134462ce37c8dbf1a6.48487689.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>确定了时间节点之后，接着我们来看下，如何建立触达通道，在用户需要的时候把最有价值的东西推荐给用户，同时告诉用户，你之前用的很爽，很不错。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>建立通道，依旧还是根据用户在产品内的全流程进行梳理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看能通过哪些渠道和场景去触达用户，同时采取什么样的理由/由头触达用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一点其实和新开卡有一定的相似之处，但是也存在一定的差异。毕竟新开卡，用户在会员卡上没有任何记录，但是续费不一样，续费的用户，有省钱金额，权益使用记录（也就代表着用户偏好），这些都是续费独有的优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般来说，对于续费的用户，突出省钱金额和用户之前使用过的权益，往往能进一步提升转化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>触达用户之后，肯定还是需要想办法转化用户，才不会浪费了这些流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>续费的抓手，除了之前提到的新开卡的抓手外，还存在其他的一点差异。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以参考营销中的4P模型，依次进行这些续费卖点的测试。确认什么样的卖点或者组合卖点才是最打动用户的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/187021218762ce37c94ed3f7.31628262.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品：产品的核心利益点是什么，可能对于不同触点和场景下的用户需要突出的核心利益点存在差异。比如针对直接下单购买的场景就是一单回本，用户动力会非常充足。对于其他未下单的场景可能就是不回本退差价，充分打消用户的顾虑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>价格：不需要多说什么，毕竟很多用户其实对价格都非常敏感，但是需要说明的一点是，不管做什么价格策略，保证收益的最大化才是关键。最好是通过活动的形式，将价格做一定的调整，能够保证短时间的销量上升，通过也能够保证后续不会受到太大的影响。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>场景：什么样的场景引导用户续费</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>活动促销：通过什么样的活动促使用户续费</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，这4个关键要素，并不是单一进行使用的，可能是叠加进行使用，保证效果的相对最优化。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>四、如何进行付费会员的运营</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后一个部分，来和大家分享下，如何做付费会员的运营。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>需要明确下，做付费会员的运营的核心目标是什么？这其中有2个目标是关键，一是提升付费率或者说续费率，毕竟这是付费会员能够持续进行增长的关键，二是提升用户的消费频率，毕竟会员对于产品来说，有提升用户消费频率的作用，可以利用运营策略把这个作用提升到更大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于付费会员的运营，一般也可以分为3个方面去进行：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一是如何进行付费会员全生命周期运营</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>二是如何提升付费会员的活跃和留存？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>三是如何提升付费会员的消费频率？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们来看下，如何进行付费会员全生命周期的运营。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一个阶段的核心目标是让用户对于自己的付费会员的身份和会员权益有明确的的感知，同时在成本可控的情况下对用户使用权益进行引导。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>按照付费的会员的生命周期，可以分为，新会员引导，权益引导和感知，流失预警和会员召回。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/37842442862ce37c9848dd1.77075054.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以用哪些手段提升付费会员的活跃和留存呢，也有几种手段，供大家参考。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最好的方式就是和权益的设计结合起来，典型常见的如权益每月/每周领取，而这些权益又是和用户直接在产品上的购买行为相关联，这样既能提升用户的活跃，又能提高用户的消费频率。典型的如京东每个月发放的运费券，以及每个月发放100元消费金，通过权益的方式，让用户的活跃，以及消费频率做了一个强关联。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外就是活动留存，利用每个月常规的几天时间，利用一些产品的优惠，让用户养成固定的消费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最为典型的就是每个月的6、7、8就是PLUS会员日，通过一些固定的活动日期，让用户养成固定活跃，固定消费的习惯。这个活动的核心目的是通过系列PLUS优惠商品的提供，一方面让PLUS用户感受到会员切实带来的利益，另外一方面，这种也相当于是一场商品售卖的大促，更好地提升用户的消费能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过这种会员日的难点在于将有各个业务线有优惠的商品整合，人和场都已经有了，关键是如何搭建具有价格优势的供应链商品是关键。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是为什么很多付费会员的产品，会员日没有做起来的原因，因为很难将商品的供应打通或很难让别人愿意给到有价格优势的商品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/132496597462ce37c9bef151.83868075.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>付费会员的提频，也是和留存，活跃一样的逻辑。但是，先有活跃才会有消费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>典型的如京东的会员日，得到的每周内容上新，这其实都是提升会员消费频率的好方式。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>结语：</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不知道大家看出来了没有，虽然付费会员可能是一个重产品的会员体系，但是其中也包含着大量的运营细节，关系着产品设计的关键。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以这不单单是一个重产品的体系，也是非常重运营的体系，都在其中发挥着重要的作用。</span></p>
