<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一家有着30余年历史的传统零售童装品牌，经历了从批发转型到品牌零售运营的周期过渡，去年又开始大跨步布局品牌私域新零售模式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这个品牌正是安正时尚集团旗下的青蛙王子（儿童服装）品牌。前不久，见实约着青蛙王子的私域负责人曹平，深聊了他们的私域新零售布局。细聊下来，他认为布局私域新零售比较完美的解决方案是：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1）搭建企业微信私域新零售体系，构建“商城+SCRM”的有机结合；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2）打造私域流量池，提供长期运营及拉新解决方案；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3）系统赋能，提升私域流量池转化，提升销量；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4）形成标准私域营销打法，赋能直营/联营/加盟门店，提升门店加盟信心，线上线下引流，提升店效 ；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">5）建立客户圈层，输出优质内容，提升客户粘性，提升品牌声量；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">6）通过平台系统，统筹管理内部导购，吸纳外部合伙人导购，形成强大内外私域分销网络，形成标准化、可复制、可持续的运营SOP，培养总部<a href="https://www.niaogebiji.com/tag/%E7%A7%81%E5%9F%9F%E8%BF%90%E8%90%A5" target="_blank">私域运营</a>团队。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然，由于他们私域起步晚，因此用户规模还属于早期阶段——会员用户60万，私域企微18万、个微12万。用户画像则是25-45岁的女性（宝妈）为主，主要分布在二、三线城市。目前青蛙王子品牌加盟店和直营店加在一起有近1600家。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在曹平看来，不管是叫新零售，还是叫私域，都是开始于消费者变化以及工具进化。消费者对工具的依赖发生了改变，所以要随着消费者的变化，不断优化工具。另外，他这样表述推动私域的主要原因：取决于公司支不支持、能不能做到战略性亏损、能不能做到大量资金的投入。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“战略是一种取舍，取舍并不是把坏的去掉，而是在十个里面再选几个更优的成为战略，私域能否成功取决于整个组织建设。”曹平认为新零售属于新型的项目型组织。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在他看来，项目型组织目前还处于一个比较稳定或者不稳定的情况下不断徘徊，原因是大家对新东西抱着很想尝试的心态，还没有明确产出，而没有明确产出与很多因素有关，如组织因素、公司因素等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">曹平将其私域的顶层设计，总结为“新零售始于消费者和工具的进化，落地于战略和运营变革，成于组织建设”。关于私域中组织的变革和调整，您也可以查看这本《私域组织变革》白皮书。接下来，我们一起看看他们的私域玩法。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/204553988362aff35614d2c7.17271100.png" title="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" alt="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">青蛙王子童装私域负责人 曹平</h4>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">01、私域解决青蛙王子七大问题</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">做私域的目的，是把公域流量引入私域后，提高客单价和形成产品连带的销售。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，青蛙王子品牌目前通过私域主要解决七个方面的问题：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一， 降低获客成本；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二， 对顾客进行精细化运营（过去手动给顾客打标签很难执行、客户也没有分层，无法做到提供精细化的服务）；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第三， 降低顾客流失率（原来的运营对顾客缺乏实时提醒和干预）；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第四， 解决一线导购不会做，也不愿意去做的问题；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第五， 流量归属问题（之前总代、加盟商、代理商之间存在利益冲突，现在采用分销模式解决渠道层级多的问题）；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第六， 加盟商之前缺乏专业团队进行<a href="https://www.niaogebiji.com/tag/%E7%A4%BE%E7%BE%A4%E8%BF%90%E8%90%A5" target="_blank">社群运营</a>和用户运营，现在总部整体会给予解决方案；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第七， 解决缺少系统的私域流量运营打法；</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">02、私域新零售1.0模式的“器术道”</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前，青蛙王子童装新零售还处于1.0阶段。基础层是“器”、中间层是“术”、核心层是“道”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">基础层，主要打造落地运营的组织与运营模式：新零售职能架构设计、团队建设、业务模型搭建、运营SOP设计、各类机制规则建设等。1.0新零售系统建设和部署需要小程序商城、服务号、CRM、商品云仓、POS、ERP、全渠道中台、社群运营系统等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中间层主要是“术”，一定要有模型和方法论，占比60%的精力，不管是商品管理，还是社群管理、内容管理、产品研发、物流供应链等，都需要全局的数据打通。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/180715939762aff35659d771.66768821.png" title="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" alt="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">青蛙王子定义的私域新零售，是以私域流量为基础，同时以社交营销为驱动的去中心化销售模式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">具体如何实施呢？围绕线上和线下自有流量、社交流量以及商业流量四大流量来源，建立以小程序为中心的流量获取和闭环转化矩阵，实现线上线下一体化运营。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">私域新零售主要是“人货场”理念的转变——传统电商一般是公域流量转到品牌商品，再到个人场域，主要体现在门店、卖场、详情页。而新零售下的私域电商，是指在自持的私域流量内，货还是商品，场变成了对顾客关系的经营。比如，通过朋友圈、以及一对一的服务，形成好友关系、会员关系等，进行转化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">青蛙王子目前私域新零售的三步曲，分别是引流、运营和转化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">引流方面，包括引流场景、承接路径以及激励机制。从去年开始用企微加粉，目前拥有接近20万粉丝。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">运营方面，侧重人设定位、画像分层以及内容策略。在顾客运营上，采用导购一对一的个性化服务。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">转化方面，包括发挥导购价值、内容种草以及设计转化路径。在客户转化上，通过社交环境下沟通进行成单。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然，也由于服装行业有忠诚度链条的说法，成人女装、运动装、男装的忠诚度高一些，童装低一些，其原因是小孩长得特别快，导致顾客群体不稳定，比如3~6岁、6~8岁以及13岁以后，完全是不同的属性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，私域运营的精细化，是青蛙王子品牌的核心推动策略。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">03、私域流量池来源及社群运营策略</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">私域流量来源于以下三大渠道：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一，线下流量的转化。目前青蛙加盟和直营整体加起来近1600家门店，大部分门店是在线下，所以会到店内进行引流转化。此外，包括地摊广告、线下广告和线下活动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二，线上的流量。目前青蛙王子在线上有几亿的销售，针对平台端我们会做流量回收和转化。另外在社交媒体、线上广告也会做转化。所以，重点运营的是，将公域流量转到私域。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第三，会员系统。目前青蛙王子CRM系统有大几十万会员用户。以前是通过公众号做推送，现在效果不是很好，会重点做流量回收，把它转换到企微上做运营。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前我们的社群分为青蛙王子的时尚福利官和青蛙王子的福利官，围绕“温度、时尚、福利、价值”四个关键词，打造有温度的儿童时装时尚社群。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">整体会以社群转化为目的，进行社群内容输出：第一，以社群福利、优惠券、活动等形式进行硬性转化内容推送，通过福利优惠的方式刺激用户转化；第二，以专业育儿、搭配Tips分享、穿搭攻略、材质用心、产品介绍等形式进行软性转化内容输出，侧面培养用户兴趣，从而刺激下单转化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">社群运营策略主要如下：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">规律性运营策略一：利用福利+科普双重刺激引流拉新，完成日常社群推送；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">规律性运营策略二：群内定期开展互动活动、产品特惠、盖楼晒单、知识科普内容输出等营造社群氛围，刺激用户参与，促进转化；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">规律性运营策略三：人设有“蛙小妹”和“青蛙王子店长”。蛙小妹主要发送青蛙福利官分工的推送内容，如商品种草、官方活动，以及日常维护社群纪律等。青蛙王子店长，以真人形象、官方LOGO代表门店的店长，做一对一优惠券推送，红包、积分，以及商品特价等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/154988240362aff356bad605.95759741.png" title="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" alt="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前，蛙小妹会做产品使用相关的信息收集和活动心愿清单收集，需要注意的是问题的引导前置，必须要有清晰的利益点，路径也必须短，便于用户快捷操作完成。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/26084578362aff35702f450.87491951.png" title="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" alt="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">04、门店导购模式策略选择</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">门店导购模式策略，分为三种模式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">模式一：门店导购+一店一仓；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">模式二：门店导购+一店多仓（所谓一店多仓，是线上一个店，但由于地理位置不一样，就分成多个仓）；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">模式三：门店导购+线上导购+多店多仓。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/203194620862aff357695e86.00222783.png" title="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" alt="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">常规来讲，多采用模式一和模式二。目前超级零售商，用模式二居多，模式一多是单个客户的加盟商，总部直营店用的是模式三。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对门店的培训以及导购激励体系应用，最重要的是抓住核心的管理要素，必须由总部制定操作规范、新零售标准，以及相关费用。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如，一是新开店标准、二是类似于APP做详细功能介绍、三是通过新零售业务规划、激励、库存设置以及分销模式、考核机制等考核指标，去考核门店的熟悉度。如果做不好，会进行一些降分，强制下线等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/75801094762aff357b0f906.79298645.png" title="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" alt="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，青蛙王子基于导购，还打造了双模型。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一是“千人户”粉丝模型。也就是门店粉丝存量从1k到2k，再到3k，基于不同的粉丝体量，门店的个人指标是多少，总部就会给出不同的奖励。例如，一方面是荣誉激励和现金补贴，另一方面是商业流量支持。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二是“1+3+5”导购的模型，指一个导购在一场社群爆破，所贡献的价值1k、3k、5k的销售业绩。通过以往数据来看，大部分停留在3k以下，及1k以上。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/189512160362aff3580fc6a2.95833862.png" title="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" alt="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">双模型的成长路径如下：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前，着重打造多个3k和5k的模型。品牌构建导购社群分层的运营管理，培养KOL/KOC，树立私域榜样力量和明星导购。青蛙王子希望完成私域新零售升级，构建专属私域流量矩阵，输出一站式私域营销解决方案。通过培养精英导购，赋能直营/联营/加盟门店，提升销量。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">05、私域销售转化的重要因素</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">销售转化=终端导购X营销赋能，也称为顾客购买要素。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">销售转化的要素，一是购买场景的迁移，用户将自己的购买行为从传统中心化电商迁移过来；二是购买需求新增，用户在社群、好友中的分享和推荐，不经意间会触发用户新的购买需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">终端导购的变量要素，一是推荐方式改变，传统电商算法推荐，到精细化会员标签及个性种草推荐；二是决策方式改变，用户将自己的决策权转移给导购、朋友、KOL、KOC；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">营销赋能的变量要素，一是截流方式改变，导购将客户沉淀在私域流量池中，过去是app截流现在是人；二是转化方式改变，导购提供更有温度的1对1精细化服务，以达到转化目的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在进行销售转化后，对顾客要采取精细化运营策略，精细化的运营决定了青蛙王子品牌到底有没有温度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，在私域新零售里有三个关键角色，分别是内容负责人（内容支撑）、运营负责人（操盘）以及数据负责人（数据分析-反馈）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/94150865662aff358765a93.84835383.png" title="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" alt="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">销售转化是很重要的，所有的商业行为一定是为了转化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">青蛙王子的销售转化分成四部分：私域分销、私域直播、线下导流、渠道分销。前三个比较重要。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">场景一：私域分销销量占比40%。基于朋友圈、标签精准推送，以及围绕兴趣建群后，再进行种草、福利促销，玩法效果最好的是闪群秒杀。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这三种群的定位以及运营玩法是不一样的，目的是激活青蛙王子的现有流量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总部一方面通过营销日历提供活动赋能导购，另一方面通过内容赋能，比如长图、海报、笔记、软文、短视频和H5游戏等等，提供给导购。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首先，导购需要做对应群的推送，第一兴趣群推送价值资讯，第三福利群做优惠券、促销信息和红包，闪群主要做直播，用来承接总部的直播分发。其次，需要根据顾客圈层/标签管理提供个性化服务，和通过一对一告诉用户优惠券，进行内容种草，提供专享福利。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最后，进行无差别的晒单，因为任何东西最终还是要营造氛围。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一，在朋友圈进行直播预告；第二，进行新品促销，新品反而是整个毛利的来源；第三，做老品折扣，以此来提高产品连带。第四，分享价值资讯和宣传青蛙王子品牌的价值观——有温度的、儿童的，而且是平价的，同时，也会做国潮。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/85924471862aff358adbb67.15683902.png" title="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" alt="这家有着30多年历史的童装品牌怎么布局私域新零售模式？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">基于模型，去做顾客标签，标签包括：顾客类型、顾客年龄段、顾客家庭孩子数量、顾客时尚度等，在群标签上还采用了兴趣群划分：宝妈育儿、辣妈穿搭、福利群等。此外，还分了普通顾客、老顾客、VIP顾客等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">场景二，私域直播销量占比30%，主要依托做视频号直播。直播离不开内容产出，因为内容产出和数据分析是连在一起的，一种情况是总部做直播，门店进行分发，且客户成交归门店；另一种情况是，也会从直营店立标杆，再到加盟商和超级零售商。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">场景三，线下导流销售占比20%。主要通过线上领取优惠券，线下完成核销，或者是在线上报名参与线下门店活动，如线上购买线下提货，再或者线上预约新品体验或者折扣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">场景四，渠道分销占比为10%，比较少。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">打造私域流量池、培养精英导购、全流程私域运营和有效提升销量，是企业做私域的四板斧。</p>
