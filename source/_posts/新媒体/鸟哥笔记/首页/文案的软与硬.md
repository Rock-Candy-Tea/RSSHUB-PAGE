<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>电视剧《琅琊榜》中梅长苏有一句台词：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>「人的心会变得越来越硬」。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/68883747262b50127e5cce0.63129509.gif" title="文案的软与硬" alt="文案的软与硬" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是啊，曾经的我们，思想简单，内心柔软，经历过人生的起起落落落落之后，从暴风中走出的你，已不再是当初那个跨入暴风时的人了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你的心越来越硬，连刀枪都不入，更何况是广告？文案重复千遍，入眼，入耳，入不了心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而我们知道，世界上最硬的金刚石，几乎无坚不摧，却可以被世界上最软的物质<strong>「水」</strong>渗透，这告诉我们：面对硬的东西，需要来点软的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对坚如岩石的心，也需要来点软性的文案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓<strong>“软性文案”</strong>，通常是指在写产品文案的时候，不直接讲卖点，而是通过各种诸如场景的代入、故事的打动、情感的营造等等，将卖点软性传达的文案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在传统媒体主导的时代，15秒的TVC，报纸杂志的KV，往往没有太多的余地去“软”着来，必须在规定的时间空间内，把事说明白，把货卖出去，硬上，是效率更高的方式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而在如今的媒介环境下，强势媒体不再，面对任何一种广告，用户都有选择不看的权利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>厉害如电梯广告，我们也可以不听不看不搭理，戴上耳机玩手机。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，广告也有了更丰富的传播场景和更广阔的发挥余地，可以让你小火慢炖，春风化雨，比如——</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一个直播间。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们拿<strong>「东方甄选」</strong>直播间为例，新东方老师们的带货说辞，充分发挥了软性文案的优势，融卖点于场景，于诗词歌赋，于故事情话，让人上头，入心，下单，一整个爱住。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>01、</span></strong></span><span><strong><span>有洞察的场景</span></strong></span><span><strong><span>让你走得进去</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们的生活，由一个又一个场景组成，你想让他代入你的文案，那你要先走进他的场景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>场景无数，无非六个要素——<strong>「时间、地点、人物，行为、对话、感受」</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><em><span>最美</span></em></strong><em><span>（感受）<strong>的不是下雨天</strong>（时间），<strong>而是和你</strong>（人物）<strong>一起躲过雨</strong>（行为）<strong>的屋檐</strong>（地点），<strong>你用你的指尖，阻止我说再见</strong>（对话）。</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实道理我们都懂，真要落到笔头，就发现有些人写的“场景”模糊不清、呆滞空洞，而有的则是充满细节与洞察，比如东方甄选董宇辉的直播说辞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他在卖<strong>“阳光玫瑰葡萄”</strong>的时候，有观众问董宇辉的老婆呢？（都知道董宇辉没对象），然后董宇辉就开始借题发挥：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有朋友说让我讲一讲我老婆，她热爱生活（人物），在6月的清晨打扫干净房子（时间/地点），拉开窗帘（行为），暖洋洋的阳光照在她身上，就正如撒在“阳光玫瑰葡萄”的枝叶之间一般。（感受-视觉），她在桌子上会放一束花，他会在周末跟我一起爬山，看湖水，感受山林间清爽的风，它热爱美丽，会在夏天穿漂亮的裙子，在秋天跟我一起去摘银杏树的叶子，做成书签，夹在书里，他会买很多诗人写的诗集，挑出最喜欢的那一篇，把干的银杏书签全部塞在里头（行为）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她突然看着我说董宇辉你知道吗？（对话）爱上你是我这一生最幸运的事情，正如我在6月的那一天尝到的那口“阳光玫瑰”一样清甜，我原以为只是一个普通的葡萄，没想到能给我带来那么大的惊喜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她还说董玉辉如果今晚你下班早的话，咱去老地方吃饭。我问老地方是哪？不是你说清楚。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来闹钟就响了，后半部分确实没有听全，但凡闹钟晚个5分钟，我兴许都知道今天去哪里找她。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一段还上了个热搜，叫<strong>#董宇辉也有编不下去的时候#</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一位新东方老师、和董宇辉并称“石灰组合<span>”</span>的<strong>石明老师</strong>说“老地方”是在临潼兵马俑下边。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/188877533962b501304fe1d2.13665280.jpeg" title="文案的软与硬" alt="文案的软与硬" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说卖“阳光玫瑰葡萄”是一种恋爱的温馨场景，那董宇辉卖《额尔古纳河右岸》时吟诵的小作文，就是另一种空灵的诗意场景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我时常在想象我是不是像作者一样，我拥有一双跟她一样的眼睛，我见证了那一切的发生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我在月光下看着星星爬上天空，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我看着人们在篝火旁饮酒高歌（事件、人物、事件），</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我看着人们通宵达旦的唱歌到天亮（感受-视觉），</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我感觉到了人们生活中的畅快与惬意，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我仿佛听到了夜里驯鹿的脖子铃铛偶尔作响（感受-听觉）。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我喜欢那些从海边突然飞起的，从河边突然飞起的水鸟（感受-视觉），</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我也喜欢在深秋时节掉落在地上，踩着沙沙作响的树叶（感受-听觉）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我喜欢夏天的时候天空的繁星点点，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我也喜欢春天的时候融化的雪水里突然长出了一两个绿色的新叶（感受-视觉），我想带你感受这一切一切。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>细节让人入境，洞察让人入心</span><span>，东西还没拿到就已感受到，当主播说完，5分钟体验卡到期，如果想要继续，只能自己下单体验了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是《额尔古纳河右岸》这本书在董宇辉的直播间，几天时间已经卖了十几万册。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/513235162b501308b1230.73319221.png" title="文案的软与硬" alt="文案的软与硬" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>02、</span></strong><strong><span>有共鸣的故事</span></strong><strong><span>让你走出不来</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>程鸿蔚老师曾经为万科·波托菲诺写过一句文案：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>谁让这流动的人生，</span></strong></span><strong><span>有了甘心凝固的理由。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>故事讲的是习惯了全球旅居的主人公，最终选择留在波托菲诺，驻足安家的故事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今也有人表示<strong>“一晚走不出东方甄选的直播间”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像董宇辉卖牛肉丸，他讲了一个和姐姐的故事：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有一年董宇辉上初中，家里没钱交学费，他和妈妈出去借钱，跑了一天没借到，晚上回家发现家里拖欠电费停电了。进门之后，他们看到家里闪着烛光，烛光映着姐姐蜡黄的脸，旁边是脸盆、暖瓶、被褥等姐姐初中住校的东西，他妈问你咋回来了，姐姐说她不想上学了、跟人打架了、和老师吵架了等一连说了4-5个理由，总之就是不想上了，态度决绝，眼神坚定。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>董宇辉不懂，还上去批评他姐，让她别惹妈生气。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来董宇辉上初中了，整天挨打受欺负，此时他姐已经退学打工了，在他初中旁边的小餐馆，每次董宇辉被人欺负了他姐都会来给他报仇，有时也会来给他送一些吃的。有一年冬天特别冷，下完雪路很滑，地上都是冰碴子，她姐给他送来一碗丸子汤，手冻得通红，冬天的阳光照在雪上很白很刺眼，刺得他眼泪都流出来了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来他妈跟他说，你以为你姐真不想上吗？你觉得咱家能同时供得起你三个读书吗？（他还有个弟弟），他才发应过来自己好蠢，当时还指责她姐。后来他吃火锅都不点丸子，但后来家里每年过年也会做丸子，他吃得多了觉得丸子确实很好吃。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>等挣了钱之后，他觉得对姐姐亏欠，喜欢给她买一些东西，弥补一下，但跟她一生的前途来比那点钱是补不回来的。他说人生中有很多遗憾，或者很沉重的东西，不要把它们当做负担，要当成你勇敢前行的力量，那么多人爱你，他们希望你更加快乐健康的成长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>董宇辉说那天接受了一个采访，和一位主编讲了这个故事，主编应该是事业有成，在北京过得很体面的一个人，他听完之后强忍着不让泪水涌出来，一直在低头快速眨眼，他甚至都不抬头看我，主编采访前说他采访经验很丰富，采访过多少明星，但那会儿他一直在低头哽咽，结束后我问他你当年读书应该也很难对吗？他连续看着我很长时间没有说话，然后很认真的点了一个头。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>董宇辉说他现在已经能风轻云淡讲这些故事了，以前不行，其实这个牛肉丸和我姐没有任何关系，只是刚才你们嘲笑我，我给你讲一段故事让你哭一哭，要下单自己去看，一单到手6盒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>故事的魔力在于，一旦你听了开头，便无法自拔，你会被故事带走、沉浸，最后听完故事的你，下单了一份牛肉丸，你是在为牛肉丸买单吗？不，你是在为自己的故事买单，因为你此刻流的泪，已经是你自己的故事了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比之下，董宇辉卖地球仪讲的故事就没有这么煽情了，只有辽阔与畅然：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我特别喜欢地球仪，在这个星球上闪耀着一片土地，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它并不是面积最大的，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它也不是最平的，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它也不是自然资源最丰富的，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它也不是河流最众多的，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但我却对这片土地爱的深沉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这片土地上有浪漫的诗人，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这片土地上也有天才般的画家，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这片土地上流传过很多的爱恨情仇，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一片土地上也有很多的赤诚与热烈，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我想把所有的历史故事都讲给你听，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我想把所有的浪漫都亲自写给你看，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我特别想陪你去感受这个土地上发生的一切。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你可以讲很多的故事，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你可以给他讲精卫填海，愚公移山，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你给他讲属于人的智慧和人的浪漫，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但请你给他讲完了这一切之后，一定记得耐心的提醒他：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们只是普通的物种，与飞禽走兽并无二异，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们与其他800多万种物种共同分享这一个星球，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在浩瀚的宇宙中，银河系都很渺小，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在银河系中太阳系很渺小，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在太阳系中地球很渺小，在地球中你我都很渺小。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后地球仪就卖光了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓故事思维有三个特质，一是情感元素（感情、趣味、新奇、意外、悬疑等等），二是附属物（人、事、物、空间、对话），三是叙述结构。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如董宇辉讲牛肉丸的故事，情感元素是亲情；附属物是他和姐姐的经历；叙述结构运用了各种对立反差，姐姐的谎言与牺牲，董宇辉的指责到自责，体面的主编低头哽咽等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>东方甄选的直播间火了后，大家讨论最多的是董宇辉的口播文案，但其实这个直播间的其他老师们也是身怀绝技，比如说<strong>「顿顿老师」</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每天晚上播到十点多的时候，通常是顿顿来接班董宇辉，每到这个时候，下面几十万观众也都无心看董宇辉了，都在问顿顿老师什么时候上播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此董宇辉说道：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>董宇辉一上来大家管他叫「老师」，顿顿上来大家管他叫「老公」，谁能知道对吧？我就卖个菜，还要经历这些苦难？太苦了，快乐都是别人的，我只觉得吵闹。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更过分的是，董宇辉有次晚上下班路上收到一条信息，有个人说董老师我妹妹在北京工作，身高1米73，月入3万，年轻气质爱音乐，喜欢爬山运动……我说姐你先问下你妹愿不愿意，我觉得我有点配不上她，然后姐姐回：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>「是的，所以请你把顿顿老师的联系方式给我。」</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而每次顿顿接过董宇辉的班之后的第一件事：哄大家去睡觉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他会先用一个倒装句「May you sleep?」让大家睡觉，接着说：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在已经是北京时间22:30了，说实话也不早了，明天也要上班了。希望大家都睡得好，看一看窗外的天空，看一看身旁的爱人，拥着ta睡觉，不要待在直播间虚拟的网络世界了好不好？</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>03、</span></strong><span><strong><span>有共鸣的情感</span></strong></span><span><strong><span>让你上头入心</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人是一种偶尔理性、持续感性的动物，人有七情，喜、怒、哀、惧、爱、恶、欲，每一种情都是发乎内心，每一种情也都可以深入人心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如顿顿卖“好想你”红枣，他用诗词歌赋吟诵爱情：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来好想你的红枣，I miss you every day。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>长相思，长相思，欲问相思未了，除非相见时，长相思，长相思，欲把相思说似谁，浅情无人知。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后接着卖“海狸先生海苔碎”：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想你了，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像海鸥想念它的海浪，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像虎鲸想念它的海面，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像我见不到你的面，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只能在家偷偷吃一碗南昌拌面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>海苔碎可以拌面拌饭吃，库存还有14包这是一个纯的福利品，一号链接已经上了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再有顿顿在卖新西兰纯牛奶的时候，他是这么说的：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说到牛奶，我相信屏幕前应该会有一群独自奋斗的年轻人，可能也有一些正在遭受挫折的年轻人，可能您的生活正在遭遇一些不幸，但是没关系，顿顿想跟大家分享的一句话： </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Don't cry over spilt milk。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/60937810262b501311c86f1.28166390.jpeg" title="文案的软与硬" alt="文案的软与硬" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它本意是表示覆水难收，如果你在遭遇一些不好的事，没关系，不要为了打翻的牛奶哭泣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>既然过去的已经过去了，悟以往之不鉴，知来者之可追，不要为了现在不顺心的工作自怨自艾，也不要为了死去的爱情独自抹泪，没有必要。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家向前看，你会看到什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你会看到漫天繁星，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你会看到星辰大海，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你会看到春暖花开。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在卖八喜轻卡牛奶冰淇淋的时候，他又继续说道：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你觉得生活不如意的话——</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不要慌，不要慌，太阳下了有月光，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不要畏惧当下，把这个冰淇淋拍回家，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让你的每个当下都充满甜蜜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顿顿善于把关于亲情、友情、爱情这些美好情感，融于各种具体的情景，我看到了你的恐惧与哀愁，所以我选择用正向的情感感染你，带去美好的情绪，让你再次充满力量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前几天媒体采访董宇辉问了一个问题：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>说大家都喜欢你们这种娓娓道来、很有文化气息的直播间，你觉不觉得你刷新了抖音带货的时代？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>董宇辉说，“千完不要给我扣帽子，我这不是文化，我就是说了几句废话，或者好听一点，说了几句心里话，只不过社交平台愿意给你个面子，让你天天挂在热搜上，你还真以为你很有文采呢，那些口水小作文谁写不出来？所以你得清醒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也不能定义为我们这种就是好，如果尝试用一种方式去定义‘好’的时候，世界便没有了‘好’。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也正如文案一样，不存在绝对的好，只存在某时某刻最适合某种环境下的好。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传统媒体时代，你需要一句通俗易懂的口号，然后重复千遍，一锤锤凿开用户的心门。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在如今的时代，文案的展示场景越来越丰富，有更多的机会可以与你耐心相处，比如一个直播间，一条广告片。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是在传统硬性文案的“打砸抢”之外，也有机会用一些软性的东西<strong>去渗透，去影响，去春风化雨，去触摸人心。</strong></span></p>
