<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯新闻架构地震。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品背景的腾讯新闻第一负责人王诗沐，和内容背景的腾讯网总编辑贺国帅、副总编辑马腾和杨瑞春同步转岗或卸任。和一年前王诗沐的外部空降不同，新任负责人何毅进来自腾讯视频技术线，加入腾讯超过10年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一位在职员工担心自己的去留。</span></strong><span>他认为裁员未必如媒体报道的那样，只动内容团队。他所在的技术团队，”几乎每个月都看到有人离开“。630</span><span>（6月30日架构变动）</span><span>之后，“算法团队整组都被撤掉了”。小组leader的签名也更换为“团队解散，毕业勿扰”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和算法一起被解散的，还有腾讯新闻的技术接入层团队。</span><span>这个关键决策，会令懂代码的团队更加不安：“APP需要跟接入层对接，接入层都没了，感觉不会做下去了”。、</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>两周后，不可思议的事情发生了：</span></strong><span>此前接入层被裁掉的团队，又被召回了一半。“一位后台工程师很快从兄弟部门了解到了关键信息：”之前新闻接入层全干掉了。“”人还没走光的时候，发现这个组的工作，别的组接不住。“，”于是捞回其中一半的人继续干。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯新闻是新闻资讯领域国民应用，大几千万DAU产品频繁换帅，意味着精英编辑体系下的<a href="https://www.niaogebiji.com/tag/%E6%96%B0%E9%97%BB%E9%97%A8%E6%88%B7" target="_blank">新闻门户</a>，再次走到了</span><strong><span>变革前夜。</span></strong><strong><span> </span></strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品上，去年接手腾讯新闻的王诗沐主张用个性推荐路线推荐泛化内容。此后他主导下的腾讯新闻多次改版，逐渐视频化、B站化，碎片化。但这一系列策略没能为腾讯新闻带来新的增长，相反，巨大的版本改动及内容调整，遭到了老用户用脚投票——数据降了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/121961209762d06a682bed96.60686322.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><strong><span><span>《</span><span>Press》剧照</span></span></strong></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>王诗沐为业内明星职业经理人，此前因负责过网易云音乐而被赞审美高级。</span>一年前作为外部认知空降腾讯，一人负责腾讯新闻、小鹅拼拼、幻核三个业务，这在腾讯的组织架构上并不多见。本次腾讯新闻架构变动和几个月前小鹅拼拼的关停，意味着王诗沐在腾讯的改革及创新尝试失败。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/166867464362d06a68545981.40865056.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王诗沐初到腾讯，员工只听说来了一个“产品大牛”负责执掌腾讯新闻，但不知道他是谁。</span><span>几轮业务会下来，一位参会员工表示，“新领导开会脾气特别差，经常骂人。“这个工作风格，和他此前的行业形象形成巨大反差。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年下半年，王诗沐对腾讯新闻产品进行大改革，投入了巨大的开发人力、运营人力<span>他力推一套社区体系。</span><strong>新版本原定2021年年底全量上线，后来改成半量，再后来干脆不上了——因为数据不达预期。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>叠加环境因素，2022年第一个季度，腾讯新闻广告营收降低了30%。扛不住压力的王诗沐开始在腾讯新闻重提精品化——这一路线是腾讯新闻前任负责人陈菊红坚持了10年的产品及运营定位。这令”腾讯新闻“和“朝令夕改”等关键字，在脉脉上出现的频次越来越高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在不久前的腾讯新闻线上全员会上，王诗沐表示“我不是一个自大的人，我的自信是来源于我的思考。”</span><span>第二周，一封关于王诗沐转岗的内部邮件发出，此后，腾讯新闻开启了不小规模的裁员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/131266562762d06a68892c09.17275894.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位接近腾讯人士介绍说，腾讯新闻此番调整放弃了原有的优势，又没学来别人的长处。”上半年还轰轰烈烈做的作者增长，突然团队就没了。建立的一系列核心指标，再无人问津。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一位署名为腾讯员工的人士在脉脉的”职言“社区描述了动荡之中的腾讯新闻：“现在研发不写代码，研究如何砍机器。数据不做分析，研究如何省预算。运营不做内容，研究如何砍内容。审核不做审核，研究如何砍审核。降本降的业务和人都没了。“</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王诗沐负责的另一业务——电商类实验项目小鹅拼拼，主团队在深圳，子团队在北京。这个业务2022年2月宣布停止运营时，北京希格玛大厦约2层楼的人力，转岗并入了腾讯新闻。两个业务的轮番变动，他们中的有些人，可能会在6个月内经历2次工作变动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/138039943262d06a68b1fa97.56892597.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这两年腾讯新闻很挣扎：</span></strong><span>做新闻资讯类产品，在商业和营收之外，常常受困于内容、技术和价值观之间的激烈冲突。</span><span>做精品内容，必然不能用流量思维去考量商业化；走流量和算法路线，在新的竞争环境之下，腾讯新闻本身的团队基因，和基于内容建设的底层内容池都将重新面临挑战。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>腾讯新闻有自己的历史包袱。</span></strong><span>过去十年，腾讯新闻前后端需求和供给两侧的运营是割裂的：前端的需求侧，腾讯新闻运营侧做了不少精品内容，考核的标准是社会价值及用户满意度；后端的供给侧，腾讯新闻技术侧更关注整合、打通之后的内容数量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种割裂会在未来算法驱动的竞争下，因供给侧的内容质量问题，影响前端推荐效果</span><span>。</span><span>一位</span><span>腾讯新闻离职专家讲了个细节：几年前搞内容打通时，时任技术负责人向总办汇报时，强调的是：我们每天产生100万条内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在本次组织架构调整之前，腾讯新闻在腾讯系的定位或早就开始下降。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2010年，人们开始为了买iPhone4而通宵排队。技术和硬件的进步，影响纸媒的销量，也推动了新浪、腾讯、网易、搜狐四大新闻门户在各个端的蓬勃发展。此后不久，</span><span>一些坚持价值选择、不以流量为导向的团队——虎嗅、钛媒体、36氪等风格品牌也如雨后春笋露出了头角。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从PC端向移动端转型的十年间，四大门户里起步最晚，生态最好的腾讯新闻跑在了最前面。</span><span>巅峰活跃时期DAU在8000万-1亿。成为国内新闻资讯客户端在流量上的天花板。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网速更快以后，门户之外，整个行业都在寻找技术进步带来的新流量和新出口。2010年之后，在线移动端产品陆续经历了微博，长视频和短视频等几个产品形态，逐渐形成由四大门户开始，到BAT入局，再到新流量崛起的行业新生态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到今天，门户、微博、长短视频和搜索，依然是当下在线流量的主要入口型产品。</span><span>只是四大门户的微博只剩下新浪微博；公认烧钱的长视频领域仅存背靠BAT的腾讯、爱奇艺和优酷；短视频第一梯队有长跑型产品快手，和迅速长成巨无霸的抖音。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在短视频领域，四大门户和老牌巨头无一进入第一梯队。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/38595832962d06a68ed4c66.94302127.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今日头条的逻辑来自精密设计，通过技术和用算法、用信息流重塑内容。这个过程剥离精英编辑审美，用机器和算法学习用户喜好，规避掉了传统媒体版面和头条需有人签字才能定版的传播风险。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在流量入口，字节通过短视频抖音为头条导流，在内容出口，字节系包括抖音和西瓜在内的长短视频介质，为头条丰富生态。<span>一套数字驱动的完整商业闭环就这样跑通了。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/159205628462d06a692a4556.91744919.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>失去流量和新介质系统支撑的老牌门户日子变得不好过。在流量上，搜狐、网易已经完全掉队。新浪如今靠微博的流量，支撑其在门户领域的影响力。四大门户里基本盘最稳的是腾讯新闻，其客户端本身在移动端第一名领跑多年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但外部环境发生了变化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频正在成为今天的流量王者，其受欢迎程度堪比印钞机。</span><span>在短视频的驯化之下，现在人看消息不愿超过15秒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那些需要花费更多时间的长文开始缺少读者。</span><span>就连长视频的核心资产</span><span>（电影电视剧等精良内容）</span><span>的制作，也开始为了迎合用户快节奏的视听语言习惯，去修改过去几十年的剪辑节奏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/120761626362d06a694f1620.10278425.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网络“三剑客”的年代已经彻底远去，如果不借助搜索引擎，人们甚至想不起新浪今天的管理者是谁；丁磊也于今年退出了网易传媒的管理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>张朝阳爱上了教书——他在搜狐视频平台直播讲物理课，并为这个直播项目匹配了一个150人的技术和产品运营团队，从商业角度，ROI是算不过来账的。但他的员工说，我们看的不是营收。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/142528696262d06a696df779.79604944.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>属于新闻门户的时代过去了。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>生于算法体系、流量追平腾讯新闻的今日头条，同样逃不掉新闻门户的群体困境，<strong>头条APP已经四年没有增长了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当腾讯新闻开始积极拥抱算法时，今日头条需要回头去探索内容的“高端化”。</span><span>年轻用户不喜欢头条、女性用户不喜欢头条，当这个表述由今日头条现任总裁陈熙</span><span>（Kevin）</span><span>在字节系内部会议中提出时，多少会令OKR压力之下的产品及运营团队有些焦虑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>陈熙此前来自滴滴，是投行背景的职业经理人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>讲话语速极快，逻辑很好。加入字节跳动之初，陈熙先在火山待了4个月。正是这4个月的熟悉和融入，</span><span>让风格上介意和防备下属manage up的张一鸣释放了信任：将更具战略意义今日头条交给了他。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/103748272462d06a69a5e4e4.17088986.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>他的头顶悬着一个魔咒：</span></strong><span>投行及咨询公司背景出来的人，和业务线出来的人之间有一个天然结界。后者通常认为前者善于表达、逻辑自洽，但落地能力差。在某些变量内能够自洽的逻辑，不代表能交付真正的收益。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去一年，陈熙在头条有两个主要策略：</span><span>去外部到处挖高职级的、懂内容的人；在内部疯狂测试各种新产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>陈熙在头条带团队开战略计划会，一开就是10个小时，从下午2点讨论到夜里12点——研究如何做内容。</span><span>漫长的会议时间里，讲完业务讲情怀，开到最后大家很激动。字节员工觉得陈熙“还是有点东西的”，“跟他开会的体验很好”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但陈熙滴滴时期的下属，提起对前老板的过往印象则更倾向于 ：</span><span>“想法挺多的，但真正能落地的项目有限。”他们说，“之前在滴滴内部那种听起来奇奇怪怪的项目，多数都是他推的。反正最后，也就不了了之了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/67698824962d06a69c8ffb2.94300368.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今日头条办公地中航广场，和腾讯新闻所在的希格玛大厦一街之隔，直线距离不到1KM。过去一年，腾讯新闻的很多核心主管都曾经接到过头条猎头的电话——他们在为今日头条寻找有调性、懂内容的人。</span><span>岗位直接汇报头条总裁陈熙，头条方面也曾跟猎头点名要腾讯的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在某职场APP的职言板块，一位认证为”腾讯员工“的人士发帖问：今日头条还值得去么？底下评论区有超过3名认证为“字节跳动“的员工回复，“不值得，别来。“</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/100907776762d06a69f2c4f8.66536458.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>陈熙在头条疯狂孵化新品，<span>令整个字节体系的赛马的机制浮出水面：有些头条早期测试过的业务，最后由其兄弟部门正式发布了：比如最近字节“小红书”产品 “可颂”来自抖音团队，“字节电商”类产品抖音盒子和marvel 则隶属于字节商业化团队。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去一年，陆续测试过”头条高端版“、”头条大字版“、“字节小红书“、”字节电商”、“字节即刻“等方向的今日头条团队，目前只发布了</span><span>一个名字拗口的“识区“APP，至今不但没有开放注册，还在有些渠道下架了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在官方运营的种子用户群里，挤满了关注社区产品的同行，有些人只是做行研，进去看看，觉得没啥就把APP删了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/82543883262d06a6a265dc9.12222526.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管陈熙在内部会议说，”头条已经走了很远”。但从大盘趋势看，头条还是增长不动了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人数上遇到天花板后，单个用户时长还有挖掘空间。当他们发现头条用户喜欢短视频时，迅速在产品里加大了短视频供给。去年起，今日头条APP里有超过70%的用户时长来自短视频。</span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" data-brushtype="text"><span>04、</span><strong>刀尖上走钢索的算法</strong></h2>
</section>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从传播的角度，短视频这一产品形态正在接管新闻门户的影响力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当沉迷和爆款，成为高DAU产品的现象级标签，也意味着算法正通过信息过载的方式填喂用户，大量群体时间正在被垄断。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/181165905962d06a6a541e59.23201220.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这一逻辑下的增长有两个变量：</span></strong><span>一个是人群数量，另一个是单个用户消耗时长。流量见顶，想获得增长就只能卷入零和游戏——抢别人的用户，或者加速消耗存量用户时间——用户哪怕多停留一秒钟一分钟，几千万数亿用户日复一日的一秒钟一分钟，累加起来，也是一个天文数字。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>客户端变得不重要了</strong></span><strong><span>，产品本身变成了一个容器</span></strong><strong><span>。</span></strong><span>容器里面装什么介质，全凭用户喜好。技术和算法的发展，驯化出越来越聪明的AB测试工具，验证人群的喜好，永远推用户最喜欢的内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/23563825062d06a6a7ad823.60121535.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>内容的生产主体也发生了改变。</span></strong><span>门户时代，是媒体在做内容，精英编辑在推内容，因为有人的因素在，内容池的内容质量和能够拿到头版头条流量，有明确的标准和风格。算法时代，原有判断标准变得模糊。所有人做内容给所有人看，越多人喜欢的内容就越是“好内容”，扮演判断好内容裁判员角色的是算法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>人类生产信息和传播信息的结构和方式发生了改变。</span></strong><span>这个规则下的内容生产初心被改了。定位上如果为了流量，去猜测和迎合算法，一定能生产出一堆垃圾。等到系统的安全和风控发现问题并修改漏洞的时间里，影响已经产生了——大量的内容已经被不可逆地分发下去了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/88314118562d06a6aa78778.48044825.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>精英编辑推荐体制里，有人三审三校，为内容每一环负责。门户时代的产品及运营团队，也保持了媒体的内容职责及管理架构，每个重要位置的上版有严格标准。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到了算法时代，</span><strong><span>技术和传播把海量内容传播中，媒体传播时代读物下印刷厂之前定版的那个环节，交给了此前一无所知的用户。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12150385362d06a6ad1a8e5.80550714.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>《</span><span>Press》剧照</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>头条和版面、要闻区域的概念消失，责任的归属变成黑盒。</strong><span>算法的千人千面，让传播的路径变得迅速而隐秘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>模式是全新的，风控也变得隐秘而艰难。<span>不是不爆，而是风平浪静地平稳一段时间之后，随机地突然爆一个大的——爆发即失控。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>冒险的另一面，回报是丰厚的。用算法把注意力分层并进行商业化，最大的用户量+最多的用户时间+只做有商业价值的信息分发，这个模型之下的商业化效率是最高的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/23194307762d06a6b07afb7.04510181.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>《</span><span>Press》剧照</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>门户的逻辑翻篇了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>起源于流量源头的门户网站，最终因流量的重新分配而发生了改变。</span><span>在参与和见证了中国互联网20的时代变迁之后，各自去往新的方向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而其过往为大众所熟知的产品形态，最终变成符号，成为参与和见证了中国互联网20年兴衰的时代标本。</span></p>
