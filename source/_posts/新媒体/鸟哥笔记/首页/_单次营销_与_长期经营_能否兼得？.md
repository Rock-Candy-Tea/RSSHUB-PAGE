<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>618已落幕，这场年中消费热潮中迸发出的营销新玩法，依旧值得再深入复盘。比如，</span><span>很多品牌都感兴趣话题——在这样一个能强势吸引消费者、深入沟通消费者的黄金节点，该如何抓住机会同步赢得流量增长与品牌价值沉淀？</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>针对这个问题，<a href="https://www.niaogebiji.com/tag/%E5%BF%AB%E6%89%8B" target="_blank">快手</a>和<a href="https://www.niaogebiji.com/tag/%E4%BA%AC%E4%B8%9C" target="_blank">京东</a>就联手上演了一场“品效同增”的互动合作，证明了：</span><span><a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E5%B9%BF%E5%91%8A" target="_blank">品牌广告</a>是效果广告增量的“源头活水”。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这场好戏中的不少解题思路，都可供品牌们参考。</span></p>

<section data-width="100%"><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">一场“内容+人”串联的整合营销</strong></section>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>今年受大环境下各种不确定因素影响，大众更需要来自品牌情感抚慰。此前京东也通过竭力保供服务，留下了</span><span>“可信赖、有担当、有责任”</span><span>的品牌印象。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>今年618，京东以“人间烟火气与有责任的供应链——618京东和你在一起”做主题定调，</span><span>将重心放在强化品牌社会责任感与品牌信赖感上，加深京东与消费者息息相关、同舟共济的情感联结，</span><span>并与快手联手打造了一场以</span><span>“内容+人”</span><span>串联起的</span><span>「在一起」</span><span>整合营销，通过</span><span>「热点流量串联+纪实性内容表达+正能量事件直播+社会性话题表达」</span><span>立体化输出了京东品牌</span><span>“和每一位用户</span></span><strong><span>在一起”</span></strong><span>的理念。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1）定制话题，以真实生活记录引发广泛共鸣</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>活动之初，京东与快手就发起了<strong>#和你在一起</strong>、<strong>#爱在一</strong></span><span><span>起</span><span>话题。一方面，邀请来自不同领域的4位达人，围绕个体创业、爱情、伙伴和家庭这些热点生活主题进行纪实创作，将镜头</span><span>对准街边摆摊的夫妇、小吃街商家、亲子互动，</span><span>去记录那些“在一起”的真实故事。</span><span>这些创作快速引爆话题，丰富了“在一起”的内涵，换取了广大快手用户“真实共鸣”。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/119119000562bb28a6dfd804.18538962.png" width="666" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，京东还借助快手平台明星资源，如乔欣、冯巩、海泉、柳岩、胡兵、容祖儿等超过百位艺人参与，分享生活百态、穿搭、端午祝福等，展示人间烟火气，顺势植入了京东618活动。达人、明星的联动整合，不仅实现了粉丝热度的激活，更为品牌实现了声量的抢占。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/34279387262bb28a7ab5b72.71101733.png" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2）媒体、达人直播，连线互动分享生活正能量</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>达人、明星内容传播持续为品牌积攒势能，环球网+京东+快手三方联动上演的“和你在一起”主题直播，便是爆发点。直播中，4位特色快手达人展示出一个个平凡人对生活的热爱。其中，“90后羊肉西施”郭柳玲大方表演了一段肚皮舞。这位为了生病父亲，将舞蹈从舞房带到了菜市场的女孩，将对生活的热爱与正能量回馈给了广大网友。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/195914898862bb28a90fd019.08012786.jpeg" width="444" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>多位明星直播也同步上线，吕颂贤、宋家腾、许绍洋等明星以真实、生活化内容拉近粉丝距离，并对京东活动进行介绍，配合小黄车、点赞图标等产品能力为品牌进行用户导入。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>3）打开PUGC通路，让用户相互陪伴、赋予温暖</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最终在快手纪实团队的操刀下，来自达人、网友们的故事，最终汇集成了温暖的收官短片<strong>《和你在一起》</strong>，“借用户故事、精神鼓励、影响更多用户”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得注意的是，短片中还有众多京东快递小哥的身影。他们曾在上海抗疫保供中，为万千家庭守住“人间烟火气”，<strong>是故事主角之一，也是京东“始终与用户在一起”的切实体现。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/196818585362bb28a9beb7e1.59904938.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>回看整场活动，</span><span>纪实短视频让观众以旁观的角度去共情那些生活热爱；直播则把现实里努力生活的主角和他们的热爱，放到了观众面对面；最后，鼓励UGC创作和合集短片则让用户从自己的故事里发掘内在热爱生活、积极奋斗的力量。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>全面卷入平台达人、明星、用户，层层深入传递品牌理念的打法下，京东也同步赢得了显著品效增量。</span><span>品牌活动期间总曝光超</span><span>34亿</span><span>，#和你在一起、#爱在一起的话题带动<strong>3.6</strong></span><span>万</span><span>个短视频参与，赢得</span><span>近32亿</span><span>的播放量，环球网直播也吸引了</span><span>6637万+</span><span>曝光；</span><span>更同步塑造起了一个生活化、人情味十足，与大众情绪同频共振，始终与广大用户“在一起”的陪伴者形象，以及支持、传播社会向上力量的责任感形象。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这其实就是品牌从过去大促中的“单一的流量思维”，转变为了</span><span>“以品牌长期建设来拉动流量和销量”</span><span>的思路——借助大促节点，培养消费者对品牌的独特认知、好感度和忠诚度，以此在大促高峰之外拉动中长期复购，最大化地沉淀、发掘乃至长期运营大促期间的用户流量价值。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>此番操作，之所以在快手平台落地，也是因为快手平台本身也自带</span><span>“提升流量到销量转化效率”</span><span>的特征。长期以来真实的内容，形成了快手独特的社区</span><span>“老铁文化”，人与人之间有着更强的情感链接和“信任感”，进一</span><span>步为京东上的品牌和优质商品奠定了转化的用户基础，并反哺品牌信任感的生长。</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“单次营销”与“长期经营”趋于融合</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看完这对618营销CP合作，相信大家也都和007一样，想进一步探究：在层层深入的玩法背后，到底什么才是品牌赢得品效增长的底层策略？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>在互联网存量时代，品牌仅单一地依赖“购买流量”，经营成本必定水涨船高。而内容作为连接品牌与用户的载体，既能通过场景化呈现实现高效转化，也能持续影响用户品牌认知，</span><span>实现品效一体化，让单次营销与长期经营趋于融合。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>实践下来看，每个平台都是基于各自商业逻辑、既往经验、数据洞察形成了内容助力策略</span><span>。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手自身品牌力的提升以及在一线外城市用户的广泛影响力，是此次京东618选择快手合作的重要因素之一。</span><span>此外，快手已经形成了独特的<strong>“新市井商业”</strong>生态，在这一生态下，平台能为品牌提供的品牌内容，就天然具备了三大属性：</span><span>原生、真实、有烟火气，</span><span>更适合品牌的品效协同生长。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1）有烟火气的创作者和用户，共筑品效兼收基础</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同于多数品牌广告由品牌为主题来面向消费者传递营销信息、品牌理念，快手内容共创模式，带动了创作者和用户参与，真正从用户角度去主动建立与品牌的相关性和情感、理念链接，因此更容易赢得共鸣。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>再加上快手和许多充斥着“精心编织”故事的内容平台不同，它反而鼓励着“市井”内容。</span><span>在看惯了各类套路创意内容之后，能吸引用户自传播的往往就是这些市井用户故事——能打动用户的只有用户自己，能为用户生活供能的正是那些来自平凡的力量。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这都是品牌能够通过快手赢得流量与品牌价值双增长的原因。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不得不提的是，这也让品牌故事里沉淀下了无数真实的用户故事和情感，成为京东绝不可能被同行替代的品牌资产。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2）成熟的IP整合营销模式，拉升品效营销效率</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于这样的市井内容生态优势，快手还打造了首个新市井热点纪实系列内容IP——</span><span>「爱计划」</span><span>，旨在以“爱”为主题挖掘有温度的新市井内容，并通过</span><span>“纪实内容+直播事件+互动话题”</span><span>整合内容营销模式，在各个营销节点上赋能品牌价值传递与效果增长。此次“和你在一起”活动，便是IP项目的第一季。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/181442471562bb28aa150166.02737454.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一IP是结合了快手自带的市井内容优势，并整合平台资源，对平台内容+资源进行合理化配置，以在营销中提供稳定的流量和品牌价值增长保障，拉升营销效率。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>更进一步，通过这一IP，快手也向所有品牌合作者展示了其平台在</span><span>“内容营销”上具有打造闭环链路的能力</span><span>，符合品牌对于“</span><span>精准覆盖、内容场景化有效种草和高效转化”</span><span>的三大媒体价值期待。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一借助平台数据洞察力，快手可帮助平台找到精准用户人群，并打造其感兴趣的内容话题；第二，快手能提供特定的内容沟通方式，进行有效的场景化商品、理念种草；第三就是通过合理资源配置与引流，内容种草到后期后链路的价值转化，可在平台上形成一体化的闭环模式。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>最后，不少品牌早已预见同一趋势：</span><span>与效果力相辅相成的，是品牌力的建设，品牌们必定要回到由内容累积的原始品牌力竞争上面去，提升在用户心目中的地位、可选择度、可提及率、忠诚影响率。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>京东此番与快手联手打造的整合营销项目，无疑是在趋势之下的一次顺势而为。未来这种凭借“深入用户洞察，聚合平台达人、用户创作者，基于短视频+直播进行持续生活化内容创作，并以此实现与用户情感沟通、价值沉淀和引流”的模式，必定还会为越来越多的平台和品牌所用。</span><span></span></p>
