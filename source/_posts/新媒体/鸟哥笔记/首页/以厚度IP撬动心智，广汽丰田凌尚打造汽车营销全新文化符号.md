<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E6%B1%BD%E8%BD%A6%E8%90%A5%E9%94%80" target="_blank">汽车营销</a>正在结束“流量为王”的时代， 随着品牌竞争的白热化，避免流量焦虑的最好方法依然是打响品牌在消费者处的心智认知，这也成为众多汽车品牌在内容营销首选的发力方向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何才能在信息爆炸的当下时代，选对打动消费者感知的“奇点”？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>广汽丰田选择以IP跨界为策略，通过《寻迹中国》这个颇具厚度的城市文化主题IP，为凌尚迅速塑造品牌符号，<strong>并借助短视频平台实现与用户的充分互动沟通，一站式实现了从“品牌影响、产品认知、生意增长”的营销全路径</strong>。</span></p>

<section data-mid>
<p><strong>有厚度的IP为品牌奠定高起点</strong></p>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对汽车品牌而言，沟通介质的厚度决定了触达圈层的广度与深度。广汽丰田凌尚为何选中《寻迹中国》？这就不得不从城市与人的关系开始说起。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着国内城市化进程的飞速推进，一座城市的发展史正是由千千万万个普通人的个人奋斗所组成的，人与城市共同成长成为一种积极的人生态度，人与城市的情感归属关系更是一部微缩版的社会跃级切片。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为由抖音主办、全国广电系统媒体共同参与的项目，《寻迹中国》所聚焦的正是与城市共同成长的新青年、新产业、新文化，用年轻人喜欢的方式塑造新鲜、有趣、有料的城市故事。<strong>作为80%的使用场景都在城市中的A+级轿车而言，凌尚所面向的正是广大在城市中打拼奋斗的新青年们，这一点正与《寻迹中国》不谋而合</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与娱乐IP有明显差异，《寻迹中国》由抖音主办、全国广电系统媒体共同参与，抖音主社交性发酵、全国广电系统主公信力传播，两者相加其在全国范围的影响力可让品牌声量实现广域覆盖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在故事传递上，《寻迹中国》采用了人物串讲+口述的形式，从一座城市的文化、经济、美食等方面切入，输出城市印象。第一期寻迹北京中，通过已在北京生活多年的创业者口吻，呈现出既繁华又孤独、既前卫又小众的北京，在日新月异的CBD与不断沉淀积蓄的胡同之间，既快又慢的北京速度跃然纸上，每一次向前，都是北京足迹的变迁。这座超级大都市极强的包容性与多元性，让生活、奋斗在这里的人们总能有所选择，跳脱止步不前的惯性，这是一座城市的魅力，也是城市与人共同的价值观追求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>城市主题在情感层面上为观众提供认可感、确信感的作用，尤其对新青年而言，追溯所处城市的发展文化、探索城市周遭更是一种文化风尚，因此无论是兴趣触点还是情感俘获上，广汽丰田凌尚选择《寻迹中国》都找准了走进用户心里的新路径，通过【先情感、后品牌】方式，让广大城市青年对广汽丰田凌尚这款新车有了了解。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以“城市”为基点的内容天然适合汽车品牌植入与曝光，汽车的出行功能决定了它在人类探索周遭时的必要价值，也成为连接人与城市的核心角色。<strong>结合《寻迹中国》IP对城市文化的探索，广汽丰田凌尚不仅完成了产品层面的露出，同时也将品牌slogan「凌尚 跃级而上」与IP立意进行了深度融合</strong>，在新型城市发展轨迹中，跃级不仅意味着已到达的高度，同时也意味着勇往直前的态度与逐步升级的未来。</span></p>

<section data-mid>
<p><strong>好内容奠定品牌心智占位</strong></p>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>广汽丰田凌尚在基于《寻迹中国》展开营销的过程中，并未局限于纪录片这一单点内容，而是在片内片外以定制、社交内容打造等多元手法，将IP影响力通过内容场景优势进行了几何级放大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>定调-「《寻迹北京》纪录片」：</strong></span><span>在北京卫视官方抖音账号释放的首集精品纪录片《寻迹北京》中，跟随着主人公的心路历程，以第一视角对北京文化娓娓道来，城市的包容性与多元性呼之欲出，快节奏的发展张力与广汽丰田凌尚追求跃级的品牌质感进行了自然融合。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为活动期的第一个大动作，纪录片的播出为用户对广汽丰田凌尚的品牌认知定下了一个第一印象，品牌理念与价值初具雏形。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/136784494762e066abc641d4.92331519.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>具象-「杨澜领衔定制凌尚品牌视频」：</strong></span><span>正片播出期，广汽丰田凌尚联合知名主持人杨澜合作了一期品牌定制视频，为大众展示了杨澜一路走来的心境变迁。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自带知名度与精神领导力的杨澜，为广汽丰田凌尚整轮营销注入了更具体的参照力量，也为用户心智沟通提供了一个更具体的切面，加深广汽丰田凌尚品牌理念与《寻迹中国》人文内核的绑定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在杨澜的自述中，北京见证了她人生中成长的模样，从1990年主持《正大综艺》到1994年出国深造，再到1998年推出《杨澜访谈录》，从读书到工作，路边风景变迁，唯有心境不变，驾驭人生，勇往直前，于成长中见证跃级。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说纪录片是从普通人视角回顾了与城市共同成长的故事，以普世共鸣引关注，那么广汽丰田凌尚与杨澜的定制视频则更放大了名人成长的感染力，以精神层面的引导力量引关注，渗透力逐步强化，为深化品牌力提供进一步加持。此外，广汽丰田凌尚还联动了10位探店达人，以定制种草视频输出产品优势，视频曝光量超<strong>800w+</strong>，更深层次地渗透人群心智。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/60613938262e066ac5512a1.15486327.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>发酵-「凌尚定制话题+官方子话题双开」：</strong></span><span>基于抖音的社交热度优势，广汽丰田凌尚冠名了#寻迹北京 官方子话题，卷入头部KOL、垂类达人等共同参与，从「城民」、「城事」、「城味」、「城景」、「城音」、「城品」六大维度发散热点，在城市话题讨论中实现品牌借势曝光，话题达7亿次播放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，广汽丰田凌尚亦从品牌视角打造了#寻迹凌尚每一面 定制话题卷入全民互动，与官方子话题双向发酵，将城市话题热度引流至品牌本身，保持品牌始终活跃于此轮城市文化话题的风眼，聚拢高曝光与高热度，该话题达<strong>1.4亿</strong>次播放。</span></p>

<p><strong>从品牌到产品、从泛众到圈层</strong></p>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>承接前期品牌影响力，广汽丰田凌尚通过精细化的营销动作，将传播重点从品牌逐步聚焦至产品，而在这一环节中的重头戏便是对直播的巧妙运用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除挑选浙江电视台主持人「小强说」、浙江城市之声主持人「新闻姐」两位头部主播进行直播以实现广域辐射之外，广汽丰田凌尚也联动了资深汽车媒体人「暴走老常」，垂类KOL的直播将受众精确至汽车用户圈层，也将直播内容的专业度上升至新台阶，通过实时产品展示与专业讲解，打通后链路营销，驱动用户完成购买决策，实现新车上市有声又有量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/196280518362e066b3253ff4.01417579.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>结语：</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内容营销一直是汽车品牌塑造影响力的王牌，但置身变化更迭的营销环境与多元分散的媒介环境，只打内容营销这张牌早已远远不够。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以广汽丰田凌尚此次营销为例，有厚度的内容IP是抢占心智的起始点，依托这份内容在立意与洞察上的厚度，消费者共鸣得以成立，而广汽丰田凌尚在内容之上制定的一系列营销策略，最终将这份颇具厚度的内容影响力放大并发散：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>● </strong></span><span><strong>全场景，从情感共鸣到购买决策逐步闭环：</strong></span><span>从内容场景、直播场景延伸到线下场景，《寻迹北京》纪录片在北京全域公交车大屏投放，线下户外广告投放覆盖5000w+人次，信息传递的重点逐渐从话题概念落实到品牌与产品，将心智认同转化为消费动力。</span></p>
