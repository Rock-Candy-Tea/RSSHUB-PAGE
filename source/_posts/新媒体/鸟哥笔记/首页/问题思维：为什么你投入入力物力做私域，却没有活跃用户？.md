                        <section class="_135editor" data-role="paragraph">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">私信收到几位读者想听我讲一下对品牌<a href="https://www.niaogebiji.com/tag/%E7%A7%81%E5%9F%9F" target="_blank">私域</a>的看法。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现在基本无论啥品牌都在努力做私域，做企业微信公众号，搞微博，搞社群营销，投入巨大，但是没什么效果<span style="caret-color: red;">。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="box-sizing: border-box; vertical-align: inherit; width: 100%;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/fpeQwNHW_QASE.png" alt="image.png" data-ratio="0.5768518518518518" data-w="1080" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我看到过一个做茶类饮料的品牌，投入大量人力做公众号，每篇文章阅读也就几百上千；进入他们的社群，人数从第一天的几百没两周就下降了一半多。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">他们的推送内容挺杂的，基本是生活故事，知识分享，限时促销，分享好友，抢红包混着来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样的规划缺乏连续性，顾客也不知道他们想干嘛。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我想可能运营人员想用丰富的内容勾起不同顾客的兴趣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但恰恰是<strong>这种策略，</strong><strong>让很多顾客被自己不感兴趣的内容打扰，选择了离开。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就陷入了一个困境：<strong>你越想做好，最终却变成了一个大杂烩，越做不好。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="box-sizing: border-box; vertical-align: inherit; width: 100%;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/DTcYZXsf_CTvk.png" alt="image.png" data-ratio="0.44074074074074077" data-w="1080" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">既然丰富内容的方向不行，那我们聚焦一下，直接从产生收益的模块来做？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从直接收益来看，老板肯定是想看到私域里转化率的提升。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">于是你的复盘就会侧重在提升私域里面的活动和价格机制，至于流失的人，那就流失好了，他们不是目标客群。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">常此以往会怎样呢？私域里面最后只剩下为了活动价格而留下的人，并且对价格越来越挑剔，感知阈值越来越高，对你的活动越来越不感兴趣。（这就是我在以前的文章中提过的阿伦森效应）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>最后人越来越少，价格再也低不下去了，私域就进入了另一个死局。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么怎样破局呢？</p>
<section style="margin: 10px auto;">
<section style=";color: #843c0c; text-align: center; line-height: 26px; font-size: 22px; margin: 0px auto; transform: rotate(0deg); -webkit-transform: rotate(0deg); -moz-transform: rotate(0deg); -ms-transform: rotate(0deg); -o-transform: rotate(0deg);">01</section>
<section class="135brush" style="text-align: center; font-size: 16px; letter-spacing: 1.5px; color: #843c0c; padding: 0px 2em; box-sizing: border-box;" data-brushtype="text"><strong>从问题，而不是产品出发的思考</strong></section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">刚好我最近读到一本书叫《良性增长》，作者讲述了一个观点 <strong>“定义一个行业，不应该从产品出发，而是从问题出发”。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="box-sizing: border-box; vertical-align: inherit; width: 100%;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/NN6C8XvP_GHLe.png" alt="image.png" data-ratio="0.44074074074074077" data-w="1080" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这句话给了我很大启发。定义行业应该从问题出发，那么做私域这件事是不是应该也从问题出发？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="caret-color: red;">大家谈论私域的时候，焦点无外乎是如何提升私域的活跃度，做好个性化服务，做好触点之间的流通，用什么优惠方法在群里面促成交等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不知你有没有注意到，在这些讨论<strong>背后对私域有一个假设前提：私域的目的是为了更好地卖产品。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这个前提下，很多品牌做私域，其实是作为销售渠道。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">无论是私域营销，还是<a href="https://www.niaogebiji.com/tag/%E7%B2%89%E4%B8%9D%E7%BB%8F%E6%B5%8E" target="_blank">粉丝经济</a>，微信生态，微博运营等等，这些看似新的概念，但实际上还是叠加引流渠道的数量来增加销售的逻辑。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌营销最基本的概念是什么？<strong>是迎合顾客需求，帮他们解决问题。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">把卖给顾客产品看成了帮他们解决问题的唯一方式，是很多品牌做私域不成功的根本原因。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>真正优秀的品牌，不会想私域能为公司卖出多少产品，而是为顾客产生什么价值。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="box-sizing: border-box; max-inline-size: 100%; margin: 0px; padding: 0px; cursor: pointer; font-size: 17px; caret-color: #ff0000; background-color: #ffffff; vertical-align: inherit; width: 100%; font-family: 微软雅黑, 'Microsoft YaHei', sans-serif; overflow-wrap: break-word !important; outline: none 0px !important; height: auto !important;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/QndvqjKu_5dSc.png" alt="image.png" data-ratio="0.43969298245614036" data-w="912" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">换言之，<strong>私域应该是解决顾客问题的产品服务的一部分。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">曾经有一段时间，我们常常看到朋友圈打卡，比如每天运动的，学英语的，给孩子讲故事的，读书的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这是当时很多APP的模式，在APP上面卖课程，然后通过朋友圈打卡减免，来督促用户坚持练习。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样既增加了用户的参与度，又让自己的品牌得到宣传。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为什么这种模式的私域用户参与度这么高？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为在这个模式下，朋友圈打卡是帮助用户达成目标服务的一部分，甚至比APP本身的课程更有意义。毕竟找课程不难，大多数人真正痛点是做不到坚持。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“我是做茶饮料的，除了卖饮料，难道还能有其他服务？”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">的确，如果单纯作为解渴的解决方案，茶饮料的确没什么其他价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但你回想一下刚才我引用的那句话 “定义一个行业的不应该是产品，而是问题”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">顾客喝茶饮料解决的问题是什么呢？单单是解渴吗？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为什么同样是解渴，他不喝白开水，不喝可乐，不喝运动饮料，不喝咖啡，偏偏是喝茶饮料？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">大概率是他既认同茶的健康价值，又没有太多空闲时间泡茶。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="vertical-align: inherit; width: 100%;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/3Dumx4TS_Ytqd.png" alt="image.png" data-ratio="0.43524416135881105" data-w="942" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">换个角度看，你要解决的问题其实是“<strong>让消费者在忙碌的生活节奏中更健康”。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你会发现单单靠卖茶饮料，其实解决不了这个问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">解决这个问题还有更多的想象空间，比如：</p>

<p><strong>这些空白，单单靠你的产品是填不上的，这就是留给私域的价值空间。</strong></p>
<p><img style="vertical-align: inherit; width: 100%;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/FkhuXPrm_rGMq.png" alt="image.png" data-ratio="0.4394904458598726" data-w="942" referrerpolicy="no-referrer"></p>
<p>“你可以发公众号，让顾客评估自己的健康系数，甚至给出在线咨询服务帮助顾客自评”</p>
<p>“你可以在社群号召顾客坚持每天打卡，晒出自己每天摄入的热量和体重体脂”</p>
<p>“你甚至可以做订阅服务，在微信每天早上8点发一对一消息提醒顾客搓耳廓，中午12点提醒离开电脑眺望远方，下午4点起立走500步”</p>
<p>总之，<strong>私域和产品提供的不同价值，都是为了解决同一个问题而存在的。</strong></p>
<p>看到这里，你可能会说：我卖饮料而已，多花这么多精力成本提供额外价值，真的能提高很多销量吗？投入产出比高吗？</p>
<p> </p>

<p style="text-align: center; font-size: 16px; letter-spacing: 1.5px; color: #843c0c; padding: 0px 2em; box-sizing: border-box;"><strong>只要对顾客有价值，就有产生利润的可能</strong></p>
</section>
</section>
<p><span style="font-size: 17px; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', sans-serif;">能问出这个问题，证明你的商业敏锐度很高。</span></p>
<p>但是利润就一定只能通过售卖产品赚取吗？</p>
<p><img style="vertical-align: inherit; width: 100%;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/rBLK2XNC_xCgp.png" alt="image.png" data-ratio="0.43736730360934184" data-w="942" referrerpolicy="no-referrer"></p>
<p>事实上，<strong>只要能为顾客产生价值的服务，都有盈利的潜力。</strong></p>
<p>现代的商业早已不局限于卖产品，利润可以来自多样服务，这叫做商业模块管理思维（Portfolio Management Mindset）</p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-size: 17px; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', sans-serif;">比如我在上一篇文章中分析的会员制商超，利润不是来自卖产品，而是来自会员费收入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了商超，其实很多拥有大量粉丝基础的企业都是这样做的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小米大家都知道是粉丝经济的代表品牌之一，其实小米的手机除了性价比以外没有多大特点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是小米的私域粘性很高，是因为论坛的讨论，发布会的造势这些都是都具有产生愉悦感和科技优越感的价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小米手机的价格优势是构建在其他的利润模块上的，比如小米手环，小米有品等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再比如新东方，以前你可能觉得它就是一个卖课程和出国服务的公司。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在双减政策落地之后，它的课程业务受到了很大打击，出国服务也因为疫情受到了影响。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这时候回看一下新东方公司的愿景：成为中国有价值的优秀的教育机构。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">老俞发现教育不一定是通过课堂这种方式来实现。东方甄选抖音直播就是一个新的尝试，效果出奇的好。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不仅聚集起了超过2000万粉丝，还做到月销超过6亿。这其实能给我们很大启示。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>当你在私域能够为顾客创造额外价值，它也能为你创造额外利润。</strong>卖货变现就是我观察到的一个新方向。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么如何为顾客构建这样的价值系统呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这是一个比较大的命题，这里我想聚焦在分析社群，<span style="caret-color: red;">因为社群是最容易构建，但是最难运营好的阵地。</span></p>
<section style="margin: 10px auto;">
<section style=";color: #843c0c; text-align: center; line-height: 26px; font-size: 22px; margin: 0px auto; transform: rotate(0deg); -webkit-transform: rotate(0deg); -moz-transform: rotate(0deg); -ms-transform: rotate(0deg); -o-transform: rotate(0deg);">03</section>
<section class="135brush" style="text-align: center; font-size: 16px; letter-spacing: 1.5px; color: #843c0c; padding: 0px 2em; box-sizing: border-box;" data-brushtype="text"><strong>你可能拉了群，但是没有建立社群</strong></section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="caret-color: red;">你翻看一下自己手机，是不是有各种群：工作群，团队群，项目群，</span><span style="caret-color: red;">吃饭群，亲友群，运动群，兴趣群等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="caret-color: red;">你还要盯着有老板的群，随时准备夸夸老板的信息和抢红包。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在社群这件事上，难就难在你是在和顾客的工作，私人生活，个人兴趣，人际关系等方方面面在竞争。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="caret-color: red;">品牌社群难做最常见的现象，就是拉得快，跑得快。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你花了大资源拉起大群，却发现当天就走了部分人，而且每天都有人离开。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="box-sizing: border-box; max-inline-size: 100%; margin: 0px; padding: 0px; font-size: 17px; caret-color: #ff0000; background-color: #ffffff; vertical-align: inherit; width: 100%; font-family: 微软雅黑, 'Microsoft YaHei', sans-serif; overflow-wrap: break-word !important; outline: none 0px !important; height: auto !important;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/typxIIAp_gIuw.png" alt="image.png" data-ratio="0.43783209351753455" data-w="941" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这是<strong>因为你往往只是拉了群，并没有构建社群。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果你回到2200多年前，也会发现一个类似的现象。彼时的起义头子之一刘邦，被项羽分封到汉中，称汉王。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是在他一路入蜀的路中，军队却不断有逃兵，最后竟然逃了很大一部分，甚至连日后的大将韩信都差点跑了，只是被总理萧何拼老命追了回来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这可是创业成功准备上市兑现期权的时候啊，怎么大家就逃了呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="vertical-align: inherit; width: 100%;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/Q8Xgvwvr_T9PQ.png" alt="image.png" data-ratio="0.43889479277364507" data-w="941" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">部分原因是因为士兵的老家都在中原，不想背井离乡。但既然都出来闹革命了，大家也习惯了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根本原因是刘邦队伍本来的愿景是争夺天下（毕竟秦国故都咸阳都是他们先进去的），但是甘愿被分封到汉中在将士看来就变成了偏安一隅，放弃了最初的愿景。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">将士因为队伍没了目标，而选择离开。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这跟很多品牌做的社群何其相像 -- <strong>顾客因为对品牌产品的兴趣而聚集在一起，但是却流失得很快</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>你以为顾客因产品而来，就会因为同样的原因留在群中。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其实你只是拉起了群，而没有建造社群。“社”的含义是组织，单纯的群组缺少的是组织感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要增加群体的组织感，可以从组织最早的形式去找答案，因为这刻在我们千万年进化的基因里。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><strong>厘清目标</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最早的组织要追溯到人类群居的开始。其实人类的食量本来不大，通过摘果子，抓小型动物吃就已经能做到自给自足。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">之所以要聚集在一起生活，是因为他们有其他共同的目的 -- 抵御风险和获得更大收益（比如猎杀更大的猎物）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="vertical-align: inherit; width: 100%;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/uzZkHmkT_gYye.png" alt="image.png" data-ratio="0.44042553191489364" data-w="940" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同理，一个明确的目标能够提升群里顾客的组织感：我们聚集在这里，是为了共同完成一件自己期望的事。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这个事可以分为内部目标和外部目标两种。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">内部目标是作用于群体内部的，比如进行体重管理，活得更健康，在游戏里获取快乐，提升个人能力，学习英语，培养健康宝宝等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如蔚来汽车的社群，目标就是让每一个成员的家庭生活更美好。他们会在周末定期组织一些亲子活动，家庭出游活动来增加顾客的幸福感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再比如孩子王的社群，目标就是让孩子健康成长。他们会在社群里安排育儿专家做专业答疑，群友们还会在群里讨论自己孩子的情况，集思广益。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">外部目标是作用于群体外部的，比如为某个偶像打CALL，<span style="font-size: 17px; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', sans-serif;">支持同一个球队，</span>宣扬环保概念，倡导更多人过上健康生活，推广汉服风等等。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><strong>创造符号</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">古代的村落为了<span style="font-size: 17px; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', sans-serif;">区分出彼此，</span>增强归属感，会创造出自己村落的符号，比如西方村落的图腾，我国村落的宗庙。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="vertical-align: inherit; width: 100%;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/MhC4JXh8_XXxR.png" alt="image.png" data-ratio="0.44021739130434784" data-w="552" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">符号的作用是让群体能够有一个简洁而清晰的自我认知。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果一群人叫“喜欢小米品牌的人”，这个定义是没有区分感的。但是如果他们叫“米粉”，就有了凝聚力，并且易于传播。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就是为什么从古到今，有凝聚力的团体都有自己的符号。比如岳飞的抗金军队叫岳家军，明朝东林书院出来的官叫东林党，鹿晗的粉丝叫芦苇，苹果的粉丝叫果粉。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">三只松鼠的社群也有自己的一套符号。品牌运营人员叫松鼠酱，称呼顾客为主人，让你看到消息弹窗的头两个字就知道是三只松鼠的群在呼叫。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><strong>营造仪式感</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就像基督教有礼拜日，佛教有初一十五吃斋日，学校上课前叫起立老师好。<span style="caret-color: red;">除了共同的目标和特定的符号，提升组织感还需要把群里的活动仪式化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如肯德基的超级星期四，电商平台的双十一，DR用身份证登记买戒指。<span style="caret-color: red;">甚至有一些仪式可能你自己都没意识到，比如在苹果出来以前，没有手机每次打开锁屏是需要从左到右划一下的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="caret-color: red;">如果你只是在群里面不定时抢红包，找一些托在群里随机聊天，发一些优惠活动，这样的运营是无法让用户产生习惯的，更关键的是没有形成固定预期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上文我说过每个人微信里都有很多个群，所以要让你的群有意义，就需要在固定时间，做<span style="font-size: 17px; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', sans-serif;">做重复的系统化行为</span>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样用户在其他时间可以假设你的群不存在，大大减少了他们的注意力成本，你的社群活动就能够慢慢融入他们的时间表。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">值得注意的是固定活动需要规定一些固定的参与行为，比如发红包前大家扣666，或者晒出自己的体重指数等等，让更多的人一起参与的仪式活动才有意义。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><strong><span style="caret-color: red;">提供反馈</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你玩俄罗斯方块游戏，觉得有趣停不下来，是因为你的每一步动作都能产生即时反馈，看到方块落下来后怎样了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一个社群想让用户保持兴趣，也需要提供即时反馈。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里就看到明确社群目标的重要性了，因为反馈的本质就是告诉所有人“我们离目标还有多远”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很多群就是因为没有什么明确的目标，不知道能反馈啥，毕竟也不能告诉所有群友每天自己做了多少销售额吧。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="vertical-align: inherit; width: 100%;" draggable="false" src="https://bcn.135editor.com/files/users/379/3794874/202207/5R6LEOsM_SVjN.png" alt="image.png" data-ratio="0.5099457504520796" data-w="553" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">针对不同的目标这里我推荐两种反馈方式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对待内部目标，我推荐用公示排行的方式。<span style="font-size: 17px; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', sans-serif;">比如体重管理，对减重减脂成果进行群内排名，</span>群内用户能了解到自己与其他伙伴的情况，起到激励作用。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对待外部目标，可以在群里做成果播报。<span style="font-size: 17px; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', sans-serif;">比如环保组织群，可以在群里分享阶段性成果，比如本周多植树40棵，今天让2家电商品牌用上了环保快递箱等等。</span></p>
<section style="margin: 10px auto;">
<section style=";color: #843c0c; text-align: center; line-height: 26px; font-size: 22px; margin: 0px auto; transform: rotate(0deg); -webkit-transform: rotate(0deg); -moz-transform: rotate(0deg); -ms-transform: rotate(0deg); -o-transform: rotate(0deg);">04</section>
<section class="135brush" style="text-align: center; font-size: 16px; letter-spacing: 1.5px; color: #843c0c; padding: 0px 2em; box-sizing: border-box;" data-brushtype="text"><strong>总结</strong></section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">私域的本质不是卖产品，而是根据需求解决用户碰到的问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你需要把目光从单纯的聚焦卖产品收回来，从整个品牌的角度思考提供给用户的解决方案，看看私域能够在里面扮演什么角色。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要避免拉了群，而没有建立社群，关键是提升组织感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一个参与度高，让人有归属感的社群需要有一个共同的目标，特定的符号，有仪式感的活动和即时反馈的机制。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以上，希望对你所启发。</p>
