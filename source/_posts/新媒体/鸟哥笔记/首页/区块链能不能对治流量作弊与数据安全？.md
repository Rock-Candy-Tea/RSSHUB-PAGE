<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/173807749362ba2b614b4139.51209741.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>程序化广告无疑是趋势性的广告运作模式,但在未来仍旧面临很多的不确定性。核心问题有两个：</span></strong><strong><span><strong><span>一是广告黑箱</span></strong></span></strong><strong><span>，即不透明，后果是<a href="https://www.niaogebiji.com/tag/%E6%B5%81%E9%87%8F" target="_blank">流量</a>作弊、流量欺诈；</span></strong><strong><span><strong><span>二是用户隐私侵犯</span></strong></span></strong><strong><span>。<a href="https://www.niaogebiji.com/tag/%E5%8C%BA%E5%9D%97%E9%93%BE" target="_blank">区块链</a>技术有可能是解决程序</span></strong><strong><span>化广告问题的制胜法宝。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>虽然区块链的应用目前处于探索初期，但我们仍可以乐见其光明前景。真实、透明和安全，是实现高效程序化广告的关键。而区块链技术所拥有的去中心化、开放自治、不可篡改、匿名性等特点，有望解决上述程序化广告的顽疾。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>区块链作为分布式数据库，不存在中心化的硬件或管理机构；其所有节点的权利和义务都相等。而区块链的分布式和去中心化，可以识别和追溯网站或应用程序的历史作弊记录。这种功能使篡改行为无法得到整个网络的认可，从而有效避免了广告作弊行为，保证了流量和数据的真实性；同时，也使得因中心节点造假而导致的信用问题被扼杀在摇篮里。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>区块链最吸引人的特点，是</span></strong><strong><span><strong><span>它能提供一定程度上的透明度</span></strong></span></strong><strong><span>。因为整个交易系统是开放的，系统信息也高度透明。除了交易各方的私有信息被加密外，区块链的数据对所有人公开；任何人都可以通过公开的接口，查询区块链数据和开放相关应用。信息透明不仅可以解决流量欺诈问题，也能够降低第三方数据监控的复杂性，简化投放流程。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>区块链还有助于强化和提升程序化广告精准定位、精准营销的优势。</span></strong></span></strong><strong><span>由于区块链加密、不可篡改的特点，在每条链上，用户都有自己精确描述的配属文件。 根据配属文件，广告主可以直接构建用户画像，降低试错成本。直接通过区块链智能交易平台，定位消费意愿最强烈的用户，广告主还能获得更高的回报率。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><img src="https://ngbjimg.xy599.com/58998123062ba2b616e3e09.85196284.jpeg" referrerpolicy="no-referrer"></span></strong><strong><span> </span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>区块链的诸多好处有目共睹，但应用于程序化广告仍存有一些问题。</span></strong></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>一是技术问题</span></strong></span></strong><strong><span>，区块链即将步入3.0阶段，在程序化广告领域的技术开发仍处于基础阶段，有待迭代升级。区块链保证了透明度，但透明也意味着无隐私。在区块链公有链中，每个参与者都能获得完整的数据备份，所有的交易数据都将被公开。用户或者说竞争对手想要知道某商业机构的资产、数据甚至商业机密，变得轻而易举。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>二是安全性问题</span></strong></span></strong><strong><span>。区块链一大特点是数据无法篡改，但前提是私钥是安全的。一旦私钥丢失，账户也将失去作用。至少在当下，人们相信区块链的“绝对安全性”。不少专家都对其加密方法表示肯定，认为区块链采用的256位哈希算法，依照现有技术是无法被反向破解的。但我认为加密技术被攻破，可能只是时间问题。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>三是程序化广告缺乏行业标准和监管</span></strong></span></strong><strong><span>，这也会影响到区块链的应用。区块链的去中心、自治化特点，淡化了国家监管的概念。但是所有创新都必须符合国家监管要求。目前广告行业对这项新技术的监管，以及规制上的滞后，有可能会毁掉区块链。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><img src="https://ngbjimg.xy599.com/78075724262ba2b6191bd64.90120128.jpeg" referrerpolicy="no-referrer"></span></strong><strong><span> </span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>四是缺少大众参与，是区块链应用于程序化广告的瓶颈所在</span></strong></span></strong><strong><span>。在2008年区块链概念就已被提出，许多人认为其“无所不能”。至今至3.0初期，实际发展较慢。而程序化广告的弊端自其出现就伴随至今，为何近几年才将区块链纳入自己的轨道？</span></strong><strong><span><strong><span>除了技术上的不足和人才缺乏，最重要的一点是，缺少大众参与。</span></strong></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>理论界已为程序化广告与区块链的融合铺好了红毯，但理论成为现实的环境仍未成熟。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>一是</span></strong></span></strong><strong><span>目前区块链对大众而言仍是全新的东西，未来除了首先要把技术做好，其次还要把社群做好，最后完成整个大生态。</span></strong><strong><span> </span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>二是</span></strong></span></strong><strong><span>区块链带来的用户自主权，在一定程度上限制了流量的自由流动；而区块链所谓的去中心化，在平台构建上，仍可能出现企业集中化占有区块链技术资源的情况。</span></strong><strong><span> </span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>三是</span></strong></span></strong><strong><span>区块链数据无法篡改，但信息源数据仍可造假。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>总之，从当下的实践及理论上来看，区块链确实有助于发展去中介化的程序化购买模式，它为程序化广告描绘了一幅公开、透明、高效的运作蓝图。但其最终能否达到各方所期望的效果，仍有待观察。</span></strong></p>
