<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月11日可颂正式上线，截止到今天7月18日，一周的时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可颂刚刚上线的第一天，我的朋友圈里就刮起了“可颂雨”，太多人都去尝鲜和下载，现在一周的时间过去了，可颂的声量小了很多，也许是时候冷静冷静去重新审视可颂了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可颂为何如此受关注？这当然与字节跳动不无关系，字节跳动作为庞大体量的互联网帝国，一举一动都影响着一个产业的兴衰，就像字节跳动推出飞书时，企业办公领域也是一片血雨腥风，</span><strong><span>所以可颂的推出，也代表着字节跳动对内容产业又有了一些新的理解</span></strong><span>，对于一些依赖于内容平台的博主或者品牌来说，提前入驻，提前参与，也许又是一个新机会，因为</span><span><strong>无数的内容平台案例都证明了，早入局者的红利不小。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>打开可颂首页，我们发现整个App非常简洁，但又感觉有点眼熟，这感觉不得不说，有点奇妙。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/124797118462d5a251807893.71367104.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到我打开了<a href="https://www.niaogebiji.com/tag/%E5%B0%8F%E7%BA%A2%E4%B9%A6" target="_blank">小红书</a>，才理解这似曾相识感居然来源于小红书。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/84579799962d5a2527dd847.57396380.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，小红书典型的双列笔记呈现方式也呈现在可颂上，右上角也同样为搜索按钮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说</span><strong><span>可颂除了小红书底部的“购物”功能没有之外，其他都与小红书非常类似。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/161732013162d5a25449b671.83127915.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小红书底部的购物按钮被可颂换成了搜索，</span><span><strong>可以看得出可颂非常强调搜索功能，而小红书也正是以高频搜索闻名于内容平台的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，在可颂的搜索页面，我们也能看得出热搜词和推荐话题都有着浓浓的小红书味道，非常偏向于生活经验和日常记录，但可颂上线至今仅一周，所以我有理由相信，</span><strong><span>热搜不是真的热搜，而是由运营工作人员人工干预的词，推荐话题也同样是运营工作人员干预的。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但还不仅仅是产品上的样式相似，通过观察，发现</span><span><strong>可颂的solgan为：定义你的生活力</strong></span><span>，</span><span><strong>小红书的solgan为：你的生活指南。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>定义和指南，都有引导的意思，且同样都是面向生活的，那么我猜测，</span><span><strong>可颂的产品定位也与小红书非常相似。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以抖音为什么要推小红书？无非是抖音看上了小红书的“种草”生意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音是国内短视频平台的top1，无论是用户规模和用户活跃情况都是稳稳的第一位，但对于品牌来说，抖音的商业价值多数情况下是“直播带货”“兴趣电商”两个环节，简单说，兴趣电商在于消费者前期发现新奇特产品，直播带货在于消费者后期购买产品，</span><span><strong>但在这两环之间的消费者“消费决策”来说，抖音意义不大。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>一个新消费品牌要想从0到1，基本上第一步必投小红书，但没听说过第一步必投抖音的，</strong></span><span>因为兴趣电商可前可后，直播带货也可有可无，但消费决策必做不可，这是当下新消费品牌能够弯道超车的根本所在，也就是刚需消费者在面临多个产品时，选择哪款的需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种需求，用户在抖音上几乎解决不了，只能去找小红书。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“中国消费决策”蛋糕大不大？抖音用实际行动告诉你，这块蛋糕是相当重要的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让我们来看看抖音对标小红书的一些历程：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年，抖音就曾上线“新草App”对标小红书，但效果不太好，目前已经下架，随之而来的2022年初，抖音App开启了“图文功能”，这也被视为对小红书的一种反制，现在2022年7月，抖音又再次推出对标小红书产品“可颂”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音真可谓不死不休，但反过来看，小红书也许真有了能与抖音掰一掰手腕的能力。</span></p>
<section>
<section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" aria-describedby="tooltip85168"><strong>2、</strong><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">可颂能火吗？</strong></h2>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说回来，可颂到底能不能成？博主和品牌要不要去可颂里占个“先发制人”红利？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我觉得要从两方面去看。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>一、消费者选谁？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>平台想做什么，是平台的事情，但用户选择去哪，是用户的事情，在只有一个平台的情况下，用户的选择是不得不，但一旦有多个选择的情况下，用户就能根据需求来进行重新选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然了，重新选择的前提一定是“真正满足用户需求”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么回到小红书和可颂上，</span><span><strong>用户的需求是消费决策，是发现生活记录生活。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以小红书目前的情况来看，用户需求是完全被满足的，经过多年的发展，目前小红书上日活已经超过2亿，类目从原来的美妆和购物，发展到了现在生活的方方面面，小到吃喝玩乐，大到工作结婚</span><span><strong>，这些有些看似鸡毛蒜皮的小事，但都是真正用户的日常需求。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而可颂却与此不同，可颂刚刚上线一周，很难说已经有了精准用户，更难说用户的留存情况不错，因为可颂的内容完全从抖音移植的，你可以理解为，</span><span><strong>全部抖音用户和抖音内容和全部的用户关系，都在可颂被复制了一份，但根据可颂的需要，选择性的向用户展示部分匹配内容。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但抖音的内容毕竟是抖音的内容，</span><span><strong>打开可颂映入眼帘的内容，都带有浓浓的抖音风格，快节奏、重话题，剧情、搞笑这属于在另外一个App里打开了抖音，这样的内容如果是小红书的用户打开，理论上来说，完全不符合用户预期，否则干嘛还用可颂？直接用抖音不就完了？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但我相信这也不是可颂的初衷，只是可颂想取个巧，借助抖音生态的内容优势和用户优势，靠资源把可颂堆起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但这样的生活社区，用户会买单吗？我截图了几条可颂App在App store中的用户评论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/124907561362d5a2554a6f85.06253120.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中五星评价的评价时间多为7月8日和7月9日，这两天时间可颂刚刚通过App store审核（可颂的社区公约发布时间为7月7日），所以我猜测7月8日-7月9日这几条评价极有可能是内部工作人员写的，所以暂时不做参考。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而其他的评价：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、图标很土，太像小红书，需要差异化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、名字很土，能不能有点互联网思维？看图标设计建议叫“小黄书”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、不配与小红书为伍，云泥之别</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、产品团队不容易，项目过不久又要被砍了</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5、卸了，与抖音互通，没有隐私感</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6、抄袭小红书</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基本没有正面评价....看样子，想被用户认可，也不是做了一样的App就行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中有两条评价值得重视，</span><span>一个是“太像小红书，需要差异化”，一个是“与抖音互通，没有隐私感”</span><span>，这两条评价说的都很在点儿上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但抖音真的不知道该做差异化吗？</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>二、怎么做好社区？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音做社区类型的产品并不是第一次，比如懂车帝做的就还不错，但懂车帝整体更偏向工具产品和PGC（专业编辑为内容主要生产者），至于纯UGC（用户生产内容）社区，字节跳动也曾推出过悟空问答，对标知乎，对悟空问答投入的资源也非常巨大，签约了多位知乎大V独家在悟空问答回答问题，但结局我们现在都很清楚，死的非常惨烈，也许字节真的没有做出UGC社区的基因？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么微博有吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年微博推出绿洲App，同样的小红书style，虽然绿洲现在还没有宣布停止运营，但明眼人都知道，这个时间也快了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时推出的绿洲的营销策略更像是“悟空问答”和“可颂”的结合体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>绿洲同时做了两件事情：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、找明星代言</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、微博内容自动同步到绿洲</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>高举高打找大V这事，像极了悟空问答，借助母公司产品直接弯道超车导入内容，像极了可颂。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但这样的策略，最终都没有成功，为什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先我们来看看当下可以称为社区的互联网产品知乎和小红书，这两者之间有非常多的相似之处。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先都是从</span><span><strong>单一用户需求作为切入点</strong></span><span>，逐步发展到今天的，小红书从海外购物切入，知乎从互联网行业问答发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次都</span><span><strong>有过一段时间的种子用户期</strong></span><span>，知乎内测多年，不开放注册，小红书最初专注海外购物人群，后期人群裂变才有了其他类目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，</span><span><strong>创始团队都有些情怀主义</strong></span><span>，宁愿为了内容牺牲商业利益，所以这两家公司至今变现层面也说不上有多好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但反过来，变现更强势，高举高打，但耐心有限的抖音能不能做成可颂？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我个人觉得不能，</span><span><strong>因为社区产品特别容易copy，但社区的氛围和用户打开产品时的“需求”，特别难以创造。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以很多社区产品的成长路径均为核心用户内测再到其他品类拓展的模式，这也许并不是因为这些社区胆小或是格局不够，而是做好一个社区的成功方式就是如此。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>抖音团队的风格有着典型的“流量人”的拼搏精神，做什么事情都很快，现在要他们慢下来花数年的时间做一个社区，他们没这个DNA，等不了。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次</span><span><strong>抖音生态也等不及</strong></span><span>，因为抖音太需要快速掌控中国的新消费品牌了，抖音需要将中国的消费产品的成长路径牢牢把控在手里，以后要想做品牌，成不成就抖音一句话。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但</span><span><strong>这块商业路径版图，现在被小红书给掰下来一块，抖音也不得不把他给拼上去。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，可颂，也许只是抖音针对小红书的一个开始。</span></p>
