<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">继6月25日<a href="https://www.niaogebiji.com/tag/%E6%88%90%E9%BE%99" target="_blank">成龙</a>在快手开启全球首播，造成全网轰动后，在7月6号，周杰伦暌违六年的新单曲《最伟大的作品》MV也选择在快手平台首发。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"> </p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">事实上，巨星入局短视频平台已成常态，而此次快手的连番操作更是打响了“双圈明星生态战略”升级的又一声炮响。在演艺圈方面，有老大哥成龙，不论是“老乡，想家了呗”的温馨问候，还是其在直播中叙述童年学武经历，以及一直以来“坚持不放弃”的闯关人精神，<strong>都与快手电商中“新市井”的概念交相呼应，并在不知不觉中获得快手“平凡人”的共鸣，激发圈层效应，引发现象级刷屏。</strong></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; padding: 0px; cursor: pointer; max-width: 100%; display: block; margin-left: auto; margin-right: auto;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220709/8d8f487b-efb9-4faa-a506-27dc1f98e507.jpg" width="449" height="737" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">而在乐坛方面，被称为华语音乐第一人的周杰伦接棒，沉淀六年，将古典、嘻哈、饶舌与流行等多种元素和谐融入单曲，《人类之子》、“龙虾电话那头你都不回我”等突破逻辑与实际的超现实作品，各类出场的作家、艺术家……将绘画、文学、音乐、影像等多种艺术展现在MV中，创作出了跨时代脑洞艺术作品——先行曲《最伟大的作品》。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"> </p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">除了艺术的结合、令人赞叹的才华，该歌曲也凭借着众多老歌迷的怀旧引发了巨大的热点和关注。据快手官方披露，单是7月5日周杰伦在快手发布的预告视频，“就是明天了，快手的朋友，准备好了吗？”，<strong>文案仅短短15个字的视频，却在2小时内达到了一亿播放量。</strong></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">此次预告的发布也让众多粉丝惊喜不已，并在评论区纷纷表达了终于在有生之年等来了新歌的激动和欣喜，周杰伦带来的明星热度在此刻得到挥发。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; padding: 0px; cursor: pointer; max-width: 100%; display: block; margin-left: auto; margin-right: auto;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220709/ff52347e-cfab-4532-96c1-45ca23c68225.jpg" width="477" height="958" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">而新曲《最伟大的作品》MV发布后，<strong>截至今日，仅仅是在快手平台，MV就产生了点赞449.1W，评论41.8W，转发35.2W的成绩，快手似乎成了此次新歌首发热点中的赢家。</strong></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">不论是成龙的首播，还是周杰伦的首发，快手平台都因此获得了不少只为和巨星互动的新用户，也最终在众多网友心中留下了深刻的印象。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">壁虎看看认为，巨星纷纷选择入局快手，这离不开快手“双圈明星生态”的策略支持，也离不开快手在娱乐模块一直的创新和努力。<br style="box-sizing: border-box; padding: 0px; margin: 0px;"><br style="box-sizing: border-box; padding: 0px; margin: 0px;"></p>
<h2 style="box-sizing: border-box; margin: 25px 0px 20px; color: #212629; font-weight: 600; padding: 0px 0px 0px 12px; line-height: 30px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-size: 18px; text-align: left;"><span style="box-sizing: border-box; font-weight: bolder; padding: 0px; margin: 0px;">双圈明星生态策略，</span><span style="box-sizing: border-box; font-weight: bolder; padding: 0px; margin: 0px;">让平台与现娱乐接轨</span></h2>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">我们都知道，在大众衣食住行都能得到温饱的新时代，精神娱乐成为了他们新的追求。从传统综影视等娱乐内容，到如今的短视频、直播、小说等各领域全面开花，随着互联网的快速发展，人们的娱乐习惯也在随之发生改变。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">而全民娱乐的时代也必然逃不过新娱乐内容的产出，在这些新内容中，<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">短视频逐渐脱颖而出，已然成为了全民娱乐的新宠</span>，作为短视频巨头之一的快手，也顺应时代，不断的为平台娱乐生态添砖加瓦。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; padding: 0px; cursor: pointer; max-width: 100%; display: block; margin-left: auto; margin-right: auto;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220709/7bda358d-35fd-4ee6-8b44-b4aafd1c49e4.jpg" width="692" height="275" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"> </p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">从影视到短剧，从综艺到游戏，快手全方位布局，不断将传统娱乐内容，以另一种方式融入平台，将短视频直播平台延展成了一个多元化的娱乐空间。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><strong>面对新时代的变化，快手随即推出了“双圈明星生态”应对策略，让明星入驻快手，以此实现平台与现娱乐的接轨。</strong></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"> </p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">相较于网红从0到1的不确定性，有固定粉丝群体的明星圈层效应更为显著，明星的入驻能快速地为平台带来一波新鲜血液，对于明星来说，如黄子韬一般，可以突出明星<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">“烟火气”</span>的一面，拉近明星与粉丝的距离，依靠平台普惠下的<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">“泛群体”</span>提高明星的热点效应。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">一位饭圈人士曾向壁虎看看表示，“以前也没玩过快手，都是在其他平台为偶像助力的，自从偶像入驻了快手，我们粉丝群的人也都去注册了快手。”</p>
