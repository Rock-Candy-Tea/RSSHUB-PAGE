<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这段时间，关于雪糕越来越贵的话题刷屏网络。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为避免掉入天价雪糕的坑，“</span><span><strong>不认识的雪糕不要乱拿</strong></span><span>”成为网友们的一致共识。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，越来越贵的何止雪糕。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不知大家有没有发现，随着各种新概念、新成分、新口味的层出不穷，</span><strong><span>我们天天都在用的牙膏，“身价”也跟着水涨船高</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70077011662540cc779d811.91220037.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>日化品变身化妆品</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在某宝搜索输入牙膏，热门功效一栏，会跳出防蛀、美白、抗敏感等多个分类。同时，主推成分下，也有着小苏打、竹盐、酵素等多重选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些便于用户快速筛选的精细设置，体现出了牙膏赛道的火热。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在2008年以前，牙膏被笼统归为日化品中，没有明确的分类。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/115318852762bb989621ffd4.75939921.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：佳洁士微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着2008年《化妆品标识管理规定》的出台，牙膏、牙刷线等口腔相关产品被纳入化妆品管理的范畴。2021年，</span><span><strong>牙膏在法规上最终被认定了化妆品身份</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从日化品变身化妆品，掀起了牙膏品牌的大作战。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，为了更“像”化妆品，牙膏产品开始追求花样百出的功能功效；另一方面，却又因为过于强调功效，出现不实或未经报备宣传，屡有品牌遭罚。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13728313662bb98964da228.35766736.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：佳洁士微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2015年，佳洁士的一款双效炫白牙膏，因虚假广告被罚603万元。这也是我国目前针对虚假违法广告开出的最大罚单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可就算被罚，也阻挡不了牙膏在“化妆品”成分与功效概念方面的执着创新，并助推了牙膏价格翻天覆地的变化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137231867662540cc8699195.09342959.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>牙膏也要“护肤级”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018年 3 月，主打</span><strong><span>“像护理皮肤一样滋养口腔”</span></strong><span>理念，并以此为基调进行整体产品设计的</span><strong><span>“参半牙齿霜”</span></strong><span>横空出世。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个号称添加了稀缺食材，客群定位在三四线城市已婚女性的高端护理牙膏，以平均单价 65元的产品，刷新了消费者对牙膏价格的认知。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>形似防晒霜的参半牙膏从成立至今，先后获得永辉集团、奇胜投资等5轮共计5000万融资。并吸引了景甜、张俪、赵露思等一众明星网红的强力推荐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/164810867462bb9896740360.19468975.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：参半天猫旗舰店</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，参半天猫旗舰店拥有34万粉丝，一款单支69.9元120克的小冰管牙膏，月销可达700多件。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并且，在牙膏的品类之外，还有不少口腔护理的衍生产品。看得出品牌试图扩宽牙膏品类渠道，丰富多价位段产品的战略布局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相较于大家熟悉的冷酸灵、两面针等国产品牌的定价，</span><strong><span>参半显然走的是高端路线</span></strong><span>。而对于牙膏“网红化”价格的质疑，也一直存在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/151408664462bb9896ad7967.97762687.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：参半口腔微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在大多数消费者眼中，牙膏的主要成分除了研磨剂，就是一些保湿剂、增稠剂之类的，并没有本质上的不同，却为何有着价格上的天壤之别。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，参半创始人尹阔曾表示，口腔护理分为两部分：硬组织和软组织，传统牙膏品牌基本着眼于前者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而参半在保证传统牙膏必备功效的基础上，加入了鱼子酱、澳洲甜橙等常用于护肤品的成分。这些配方可以对口腔内的软组织起到防护作用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看来，除了应该具备的基本功能之外，</span><span><strong>品牌及其宣称的美白、抗敏等附加功能和“护肤级”成分的添加，才是影响价格的关键</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/127618355762540cc97137d2.72071747.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>创新功效打造概念</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有数据显示，2015—2021年牙膏价格持续上升。截止2021年7月，我国120克左右的单支牙膏均价为10.75元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>目前市面上，15元以上的牙膏占据了35%的市场份额</span></strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>从初代的田七、两面针中草药牙膏；到“冷热酸甜想吃就吃”的冷酸灵牙膏；再到提出防蛀、美白概念的高露洁、佳洁士；以及用“防止牙龈出血”药用功效拿下销量前十的云南白药牙膏</span><span>……</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们可以看到，无论是国产老品牌、合资品牌或新晋网红品牌，在以细分产品满足消费者差异化需求上的同时，价格基本都在上涨。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/110093433762bb9897030c82.83168392.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：两面针微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有消费品牌分析师曾说：“当前市面上比较贵的牙膏，并不是因为成分上有什么创新和突破才引起成本升高、价格上浮。现在的成分大多数是打概念，核心成分并没有什么改变。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说，“造概念”是新消费环境中牙膏越来越贵的原因。倒不如说，在没有硝烟的牙膏赛道，</span><span><strong>不造概念，就有可能被市场抛弃</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1985年，人人皆知的中草药牙膏两面针，实现了突破1亿支的产销量。可随着90年代初高露洁、佳洁士等外资品牌的进入，80%以上的日化市场都被外资品牌占据。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/158778544762bb98972a1d12.33302570.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：云南白药牙膏微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到2005年，云南白药一支22元的高端牙膏出现后，市场份额才发生了改变。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让云南白药突围胜出的，正是将药用功效融入产品中的开创先例。消费者愿意为抗炎消肿等药用概念买单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也让牙膏品牌看到了</span><span><strong>不断优化产品结构，提出概念带动中高端产品销售和产品价格中枢上移的新方向</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，既有着突围胜出的市场压力，又有着变身化妆品行列后的更多可能。这就让</span><span>成分创新成为口腔护理市场的主要趋势</span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9818835562540cc9e9aa29.23485278.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>“攻心计”售价翻番</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>100年前，中国诞生了第一支牙膏，初衷只是为了清洁口腔。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>100年后的今天，除了基础功效之外，</span><strong><span>牙膏玩起了高端化妆品俘获受众时的“攻心计”</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们可以看到，青蒿素、紫荆花、芜琼花等各种植物中药成分的应用，以及益生菌、海盐、绿茶等可食用材质的常见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/159900216862bb9897515944.82167171.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：两面针微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在对口腔内的软组织起到防护作用的同时，也让凡是带有特殊“功效”的牙膏，售价翻番。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，一款双重薄荷经典口味的好来牙膏，120克标价12.9元。而名为“深导白”的同品牌美白牙膏，100克的售价高达129元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，</span><span><strong>随着消费理念的改变，品牌除了借基本清洁功能造概念之外，还深度洞察产品在使用形态、外观视觉和嗅觉体验等各方面，以期打造消费吸引力</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/185426999562bb989787e504.21440713.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：高露洁微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅牙膏膏体变得五颜六色，加入超乎想象的食用级香料，甚至还有品牌推了国际香氛大师调香等炫酷概念。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从网红品牌参半的产品销量来看，一些爆珠糖、固体牙膏以及口腔清新剂等便于携带，能够让年轻人产生好奇的创新品类，都创下了销售佳绩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些产品的存在，让牙膏品牌不再是一支简单的牙膏，构建了可以社交、清新口气、变白牙齿的立体产品链。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许在消费者眼中，当一个牙膏品牌具有可以辅助解决多种问题的产品，不再只是单一的洁牙工具，就意味着这个品牌在市场的受关注程度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/147695138062540ccb2181e5.74937061.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果去问牙科医生应该如何保护牙齿，他一定会告诉你四个字：</span><strong><span>认真刷牙。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>既然重要的并不是牙膏价位的选择，年轻消费者又为什么“不得不”买高价牙膏呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，</span><span><strong>是因为目前市场上牙膏整体价格的走高</strong></span><span>；另一方面，</span><span><strong>从外观视觉到使用体验的新奇感，成为乐于试新年轻人的消费动力</strong></span><span>。</span></p>
