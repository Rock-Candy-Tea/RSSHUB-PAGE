<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>清晰可见的一整根长白山人参，可以8小时内续8杯水，售价仅19.9元/480ml，今年上半年，由硬核颜究所推出的植物饮料“一整根人参水”，成为了现象级的爆款产品，不少门店一瓶难求，货架全空。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/865560440630372565539e1.71495544.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span><span>▲</span><span> </span><span>图源：</span></span><span>硬核颜究所Lab</span></span><span><span></span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>熬最晚的夜，喝最补的水。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“一整根”的爆火，和当下年轻人“一边作死一边自救”的赛博朋克式养生方法不无关系。</span></strong></span><span>整根人参泡制的熬夜水，成为了年轻人眼中最养生的“熬夜神器”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，随着“一整根2号水”的近期上市，我们发现，</span><strong><span>硬核颜究所关心的，可不仅仅是年轻人的熬夜问题。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><img src="https://ngbjimg.xy599.com/5611320262b974392829e0.57743105.png" referrerpolicy="no-referrer"></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>“越喝越来劲”的2号水</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为“一整根人参水”的升级版，“一整根2号水”保留了“1号水”的特色，同样的一整根长白山人参，同样的0糖0脂，官方售价为25.8元/480ml。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了区别于“1号水”，“一整根2号水”的瓶盖和包装上采用了黑色色调。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134366390763037256da90f8.59592831.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span><span>图源：</span></span><span>硬核颜究所Lab</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，“一整根2号水”在口味上进行了优化，特别添加的小雏菊口味，让小雏菊和人参风味相融合，喝起来更加香气清新。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其值得注意的是，“一整根2号水”中添加了</span><span><strong>牡蛎肽粉、黑松露粉和葛根粉</strong></span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这三种配料，通常被认为具有补肾、助阳的保健功效。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这让人不免联想，</span><strong><span>硬核颜究所推出的“一整根2号水”，是在关心男性消费者的难言之隐，要代替汇仁肾宝片，扮演“男人加油站”的角色了。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一猜测或许可以从“一整根2号水”的推广中得到验证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果打开小红书，搜索“一整根2号水”，就可以看到完全不同于“1号水”的画风。种草“一整根2号水”的达人，都是拥有健硕胸肌、八块腹肌的男性，他们的个人标签无一例外都有“健身”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1778251561630372574b7078.16121259.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span><span>图源：</span></span><span>小红书</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而从他们的小红书笔记中，</span><span><strong>“黑松露和牡蛎肽对男性十分友好”、“男士必备神器”、“男友力Max”等字眼，tag“越喝越来劲”等，都带有某种功效暗示。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/158757678863037257d50f60.61528132.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span><span>图源：</span></span><span>小红书</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管没有明说，但“一整根2号水”的目标用户，无疑是以男性为主了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/38794396662b974394a3da6.63273158.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>赋予“人参水”更多可能性</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“人参水”自然不是硬核颜究所的首创。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年，同仁堂旗下知嘛健康推出的熬夜水和神仙水中，人参就是原料之一。“椿风·养身茶饮”“若遇江南·国风茶”“炖物24章·古法茶饮”“花梨元气研究所”等茶饮品牌，也曾以人参或者红参为配料推出相关产品，打出了中式养生的概念。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/82955356363037258a00d82.74909975.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span><span>图源：@知嘛健康官方微博</span></span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但不容置疑，硬核颜究所的“一整根”把“人参水”的话题热度拉到了一个峰值，每日搜索量从寥寥十几变成了数十万。在官方公众号中，硬核颜究所毫不掩饰品牌对“人参水”的带动效应。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，“一整根”也面临一个尴尬局面。</span><span><strong>相比市面上的“人参水”，“一整根”除了“整支长白山人参”的噱头外，并没有其他明显优势。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“1w营销费用引爆销量10万+”的神话，很大层面源于消费者的猎奇心理，很难形成更持续的购买力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，硬核颜究所的策略是，</span><span><strong>趁热打铁推出“一整根2号水”，并通过加入牡蛎肽粉、黑松露粉等原料，赋予“人参水”差异化的可能性，刺激消费新的增长点。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1890608619630372592ca0d0.04518041.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span><span>图源：</span></span><span>硬核颜究所Lab</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟，大多数人参水、熬夜水的营销点，还停留在护肝、补气、缓解疲劳的诉求点上，对于“感觉身体被掏空”的男性养生需求，很少有提及。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有研究统计显示，中国的保健品行业规模长年保持10％的增长，早在2015年，保健品行业就已经达到4500亿左右的规模，甚至，有一半左右市场是被补肾类的产品占有。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>某90后明星被曝吃“海狗丸”的丑闻，和“18到35岁成保健品重要消费人群”的新闻相互呼应，也似乎印证了，中年男性除了秃顶、发福，还有难以启齿的危机。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抛开这一层不说，当“一整根2号水”被赋予男性阳刚之气、男友力的象征，自然也能激发男性消费者的购买欲望。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><img src="https://ngbjimg.xy599.com/46978155362b9743fae3289.16804197.png" referrerpolicy="no-referrer"></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>难以再造爆款</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，被赋予新卖点的“一整根2号水”，似乎并没有续写“1号水”的爆款神话。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从8月4日线上发售，8月6日线下上架至今，在小红书上罕见“一整根2号水”的自来水种草笔记，几乎都是官方合作推广。这与“1号水”在小红书的出圈效应形成了鲜明对比。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，硬核颜究所的天猫官方旗舰店，依然是“1号水”更受欢迎，“一整根2号水”的销量显示是200+。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1147985223630372598e4aa3.57667911.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span><span>图源：</span></span><span>天猫</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在公众号上，官方透露，“由于上批产品的一些细微的口感问题，没有达到我们的标准，所以紧急召回了10万瓶2号水，并全部进行了销毁处理。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这可能是“一整根2号水”没有形成广泛种草效应的一大原因。但这也揭示出</span><strong><span>，“一整根”从“1号水”就埋下的隐患，即口感问题。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，从“1号水”的消费者反馈中，就能看到很多对口味的吐槽，诸如“味道很奇怪，带点苦也带点辣”、“一股土味，一言难尽”、“一股中药味”等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11771269696303725a244b05.94587434.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span><span>图源：</span></span><span>小红书</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“一整根2号水”中虽然加入了小雏菊进行口味优化，但诸多原料能否融合成更让人接受的口感，还有待市场检验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于茶饮来说，口感显然是最核心的产品力。即便再滋补的人参水，如果给人如喝中药一样的痛苦体验，那也不能抓住消费者的胃。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>更何况，“一整根”的养生保健功能，一直有“智商税”的争议。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“一根人工种植人参批发价在2元左右”、“二三十年以上参龄的人参才能起到滋补作用”……随着媒体不断报道，人们对“人参养生”的态度逐渐趋于理性，“一整根”的产品附加值也不断被削弱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，“一整根”联合创始人Elvis在接受食品板采访时透露“<span>未来会推出价格带在10元以内的产品线</span>”，或可以从价格上俘获人心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>此外，“一整根”在营销上也面临着监管约束。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据相关规定，我国的食品类别可划分为普通食品与特殊食品两大类，“一整根”没有国家食品药品监督管理局批准的保健食品标志“蓝帽子”，属于普通食品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据我国《食品安全法实施条例》第三十八条第一款规定，对保健食品之外的其他食品，不得声称具有保健功能。否则，就会涉嫌虚假宣传。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了规避风险，“一整根”已经把宣传文案从“解决熬夜后遗症”改成了“让你的每一天都元气满满”，而对于“一整根2号水”的独特卖点，也只能通过擦边球的方式来宣传。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于硬核颜究所来说，或许当务之急不是再造一个爆款。解决口感问题，在法律法规内找到一个更合适的定位，才能延长网红效应，续航产品力。</span></p>
