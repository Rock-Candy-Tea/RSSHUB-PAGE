<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">全民健身的时代浪潮，正奔涌而来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">刘畊宏调动了千万粉丝健身，飞盘、腰旗橄榄球、绕城骑行，轮番成为生活流行。中国广视索福瑞媒介研究（CSM）发布数据显示，截止2020年，中国15岁以上体育人口已达到5.86亿。双奥和疫情影响下，体育锻炼热度还在呈上升趋势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/44511218262df0765166477.07999766.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">于是体育运动品牌之外，各行业品牌也纷纷入局。但与此同时，用户们也有了新要求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从冬奥冰雪运动的兴起，以及最近世锦赛中国跳水队、WTT冠军赛中国乒乓队等轮上热搜，就可窥见，大众已不再满足于观赛，而热衷于关注体育明星，参与体育话题讨论，消费体育内容，乃至亲身体验运动项目。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">过往品牌仅做赛事赞助的玩法，显然无法满足用户对于更全面，更具体验感、参与感，生活化的<a href="https://www.niaogebiji.com/tag/%E4%BD%93%E8%82%B2%E8%90%A5%E9%94%80" target="_blank">体育营销</a>需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌该如何打好体育营销这张牌？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7月20日，中国移动咪咕举办了主题为“咪咕元动力·焕新世界波”的2022资源推介会，打出了“5G+全体育”理念。7月24日第五届数字中国建设峰会数字体育分论坛上，中国移动咪咕公司再次对外分享了“5G+全体育”升级，赋能用户“美好生活元动力”、助推数智体育产业新发展的愿景。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中国移动咪咕的“5G+全体育”理念，正好向各行业品牌主们展示了如何在新阶段的体育营销合作中，回应用户诉求，抢占时代红利。</p>

<p><strong>推广全民健身生活方式，展露两大营销助推力</strong></p>

<p>中国移动咪咕以“美好生活元动力”为品牌传播口号，推出了“5G+全体育”升级。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所谓“5G+全体育”，对照马斯洛需求层次理论来理解，即人的需求从生理需求、安全需求、社交需求、尊重需求向自我实现逐级延展，覆盖从活着、活动、运动、竞技演进到体育的全过程。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“5G+全体育”升级后，咪咕不再是单一的观赛平台，而基于人们在不同阶段的实际需要，以中国移动算力网络为基础，以元宇宙的MIGU演进路线图为指引，以“美好生活元动力”为目标，通过“连接+算力+能力”新型信息服务体系，依托“内容+科技+融合创新”，面向全年龄、全生命周期用户，提供全场景、全时空体育内容，打造体育元宇宙，讲好全球化故事。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/26824923462df0765996801.58756165.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果说“5G+全体育”是咪咕的发展逻辑， 那么“美好生活元动力”作为发展目标其实也展露出具体发展手段：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“元”即打造“体育元宇宙”，就是以“5G数字底座”做支撑，以超高清、HDR、AI等为代表的“5G黑科技”，构建视频彩铃、VR、AR等5G技术应用场景，通过体育赛事、生活方式、数智人等内容呈现，形成现实与虚拟相互融合的“平行时空”。这个时空将成为品牌未来营销主阵地。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“动”即让体育运动成为美好生活引擎。体育运动是吸引注意力、提升免疫力、增强意志力的重要途径。咪咕致力于打造极致体验体育故事，提供包含基础运动、身体训练、竞技体育、全民体育等等在内的丰富体育内容，让全民健身成为一种生活方式，以满足人民群众对美好生活的不断向往。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“力”则是打造元宇宙、发展全民健身的底层支撑力——算力。依托算力网络，咪咕将全面助推体育产业数字化升级。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/209851676862df0766051ea9.70514279.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实上，推介会前，“5G+全体育”与“美好生活元动力”就已落地到实际业务中。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">咪咕曾结合“内容+科技+融合创新”优势，上线5G全民健身——“全民热练”专区，携手奥运冠军、健身达人、元宇宙数智人健身教练等打造多元、优质健身内容，满足全年龄段人民群众的全场景、全时空健身需求,积极引领大众走进运动健康生活新方式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">咪咕这一系列升级也为品牌带来营销升级：一方面不断聚合平台资源，开拓体育内容疆域，覆盖大众运动健身生活，意味着品牌体育合作营销内容将更宽泛，更全面渗透品牌用户生活；另一方面在元宇宙中，体育营销可打破时空场景限制，用户参与感、沉浸感也将更强。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2022北京冬奥会期间，咪咕就在“5G+全体育”概念指导下，打造了“5G冰雪元宇宙”，并实现340亿次全场景播放量、711亿全网热度、511个策划热搜、6700万海外曝光，基于此的各类体育营销场景创新，更为中国银行、一汽奥迪、安踏等众多合作方带来显著增长。卡塔尔世界杯期间，咪咕还将打造首个“世界杯元宇宙”，打开更多营销想象。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中国移动咪咕不仅拥有欧洲杯、东京奥运、北京冬奥等顶级赛事版权，其版图更延伸到了CBA、NBA、欧洲足球联赛、排超、UFC等各大体育垂类的头部赛事，因此一直以来都是各大品牌进军体育营销领域的首选合作方。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此次面对品牌们紧盯的卡塔尔世界杯，咪咕早已做好准备。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1）通过技术+内容+玩法升级，打造首个世界杯“元宇宙”</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中国移动咪咕将在卡塔尔世界杯期间打造首个世界杯 “元宇宙”虚拟观赛互动空间，为观众呈现具有即时互动、沉浸体验的“5G云赛场”。赛事直播上，向来在超高清直播技术上走在行业前沿的咪咕，将通过5G+4K/8K、5G+VR/XR带来64场超高清精彩赛事直播，还将首次呈现世界杯8K直播，为国内观众呈现出未来感十足的足球盛宴。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134599202662df0766b17a27.49201715.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在特色内容方面，凭借着有趣、专业的“顶配”嘉宾天团收割全网热度的咪咕，将在世界杯期间重磅打造“大咖与朋友们系列”节目，邀请泰斗级体育评论员宋世雄，知名足球评论员詹俊、董路，著名主持人李响以及数十位嘉宾大咖们，带来《世界杯最强音》《詹前顾后》《你懂球吗》等10余档专业有趣的特别节目，将自制能力与资源优势发挥出1+1>2的影响力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2） 其次咪咕提出了一个独特增长模式</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">咪咕把在内容、技术和场景上的核心能力放到营销领域，形成了特有的4E模式，即：场景（Event）、激发（Excite）、体验（Experience）和情感（Emotion）。基于这一模式，可以帮助品牌通过内容进入场景，从激发兴趣到互动体验，最终实现情感认同，实现品牌消费者向超级粉丝转化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/199963375862df076737cec0.16732189.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">北京冬奥期间，中国银行和咪咕基于4E模式展开合作。“顶流解说”王濛凭借“我的眼睛就是尺”火爆全网之际，中国银行借势实现海量曝光与传播。中国银行更通过定制冬奥视频彩铃，打造线下8K超高清观赛，以内容、科技、运营实现全场景突围，不断集聚用户品牌好感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/199419612462df076819d107.21054496.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">值得一提的是，王濛凭社交内容运营出圈，中国移动视频彩铃多元化开发，以及冬奥主题综艺等自制内容广泛传播等，都展示出咪咕强势的内容力、运营力——这对于品牌们而言既是流量吸引力，也是合作保障。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/148954276362df07693da863.09648624.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3） 咪咕在世界杯元宇宙中，开放了多元合作体系</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4E模式将沿用到卡塔尔世界杯，而在世界杯元宇宙中，场景、激发、体验和情感环节也必将进阶。此外，咪咕世界杯推出了覆盖全场景独家战略合作伙伴、行业场景独家定制伙伴在内的八大广告产品体系。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/209542537162df076a022957.50154407.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其中最特别一项，是世界杯元宇宙的虚拟空间合作。冬奥期间咪咕就曾打造国内首场实时直播的元宇宙音乐会“冰雪元宇宙演唱会集光之夜”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">世界杯元宇宙将更宏大，不止有世界杯音乐盛典Live Show音乐表演，还可观赛，还有更多内容、互动场景，为品牌提供更多合作空间。体育+娱乐场景中，品牌也能吸引更多泛娱乐人群，并在强互动、深度沉浸式的体验中进一步拉近用户距离。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/184958708062df076b152fe4.19663060.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为元宇宙沟通桥梁的数智人，也是合作重点。咪咕拥有国内最先进的数智人技术，已组建覆盖各领域的数智人矩阵——包含谷爱凌数智分身Meet Gu、UFC数智人尤子希、篮球数智人古逸飞等。他们都可成为品牌内容承载者和传播者。北京冬奥中，Meet Gu就曾身穿安踏现身赛事直播间与主播虚实联动，还作为连线嘉宾参加安踏元宇宙发布会，进行异业合作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">咪咕还可为品牌量身定制数智人全案，从形象到人设，从内容到运营，深度植入世界杯的赛事、节目、互动等各个维度，实现IP强关联。品牌也能借此打造专属印记和数字资产。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/129934236162df076b536266.49860518.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，“互动”玩法也是咪咕强项。进球红包等常规玩法外，咪咕开启另一种可能——基于世界杯元宇宙的互动游戏定制，比如打造虚拟世界杯酒吧，供用户看球、社交、游戏等。互动游戏也能品牌定制，并承载在咪咕元宇宙，嵌入品牌页面，刷新用户体验的同时，也成为世界杯期间的流量抓手。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然，咪咕也没有放弃现实社交。咪咕第二现场还将继续为球迷们提供相聚、观赛、娱乐场地。生活化的实景体验，在愉悦、放松又充满激情的场景中，合作品牌也更容易被球迷认可。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以上仅是咪咕世界杯营销模式中的其中4项，既有场景覆盖，也有数智创新，既有全民互动，也有深度体验，可以说囊括当下所有互联网广告形态。但元宇宙特点就是更自由、更多元、更破圈，因此在咪咕的核心优势保障下，品牌还可以继续根据自身需求，与平台共创更多营销玩法与增长模式——营销想象不止于此。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不难预见，全民健身+元宇宙的双重浪潮下，体育营销必将迎来一场大洗牌，而品牌想要勇立潮头，就必须抓住最新的营销理念、营销模式，满足用户最新的体验需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现在，中国移动咪咕已经发出邀请，谁又将成为下一个时代弄潮儿？可以拭目以待。</p>
