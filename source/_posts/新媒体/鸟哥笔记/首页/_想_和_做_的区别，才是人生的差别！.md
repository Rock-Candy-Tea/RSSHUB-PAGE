<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网上有这样一个笑话：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>每当1月1号的时候，每个人都宛如戏台上的老将军，浑身插满了flag。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但到了12月31号的时候，很多人发现曾经的雄心壮志、豪言壮语早被自己抛到了十万八千里之外。而那些优秀的人，不仅将目标记在了心里，还真正的做到了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在工作中，就有这样两种人，一种人只会“想”，想得很多且很高大上的项目，但极少真正的去行动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外一种人，<a href="https://www.niaogebiji.com/tag/%E6%89%A7%E8%A1%8C%E5%8A%9B" target="_blank">执行力</a>很强，不仅能想到，还能去做到。人与人之间的差距就是这样一点点拉开的。最后就是完全不一样的人生。</span><span></span></p>

<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多事情，其实并没有想的那么难，一旦你开始去做了，就会一步一步逼近成功。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你不开始去做，就会在脑子里想很多，甚至给自己设置根本不会出现的障碍。而<strong>你一旦开始做后，就会从“要不要做”变成“怎么才能做好”。</strong>一次次在行动中收获经验，一步步走向成功。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>导演李安当初在拍电影《喜宴》的时候，因为手头的预算不够，便犹豫要不要拍下去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>侯孝贤导演知道后，便鼓励他说，没有谁在一开始拍的时候钱就是够的，但只要你开始拍了，才可能有结果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>世上许多的事情就是这样，不管计划再精细，想得再周全，都无法预测到整个项目和事件的所有发展。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更何况在现在节奏如此快速的今天，机会是不可能留给你太多考虑时间的。但你只要做了，就一定会有“结果”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/105891731062e86bd130fb38.70006141.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>俗话说：三思而后行。但后面还有一句话：再，斯可矣。意思是，想得差不多就可以了，不用想太多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>罗振宇在分享团队的工作心法时说：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“做一件事，它到底靠不靠谱，你坐在家里想是没用的。我们的风格就是，不管三七二十一，主意出来大体觉得靠谱，先干起来。”</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有了大致的计划后，开始行动，才能知道这件事是否可行。是选择放弃还是继续调整下一步阶段，有了实际的反馈结果，才能不断的提升自己。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，在想一件事要不要做的时候，不要纠结太久，先做起来再说，你会更容易成功的。</span><span></span></p>

<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有些人很聪明，想法很多。但为什么往往不如那些看着“愚笨”的人呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为执行力不行、耐心不够。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个上市公司的HR说过：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>他们给企业招人的时候，更青睐那些行动力超强的人。他们可能没有很多天马行空的想法，但他们善于把目前的想法实施起来。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在中国古代用兵也是一样，建军模式统称为<strong>“兵贵愚，将贵智”</strong>，就是当兵的要“笨”一点，军官要聪明一点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>试想一下，如果士兵都是头脑灵活，想法很多的人，就不太会听话，这兵还怎么带呢？不服从命令的士兵，执行力等于零，对军队来说只是一种潜在的威胁。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再来看看军官，作为军队的首脑，一定要聪明，想法多的人。但只有这一条也是不够的，还必须要有超强的执行力，甚至比士兵更要有行动力，不然怎么让士兵信服与你呢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>著名的戚家军就是如此。戚继光的招兵原则是只收农民，不收城市居民。他还订了一条奇特的筛选标准：“凡是脸色白皙、眼神轻灵、动作轻快的人一律不招。”他只要淳朴可靠的青年农民。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管是以前还是现在，用人最基本的标准和最高的标准，就是要有超强的执行力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，不是说不动脑子就是优秀的，而是执行力要大于你的想象力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一个人想得再多，都不如实际中行动的一小步。只有行动了，做到了，有结果了，才能获得领导更多的赏识。</span></strong><span></span><span></span></p>

<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在职场中，“执行力”其实不光是员工的能力问题。在西方的管理学中，<strong>员工的执行力和企业领导的管理能力，是密切联系在一起的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>管理能力强悍才能衍生出执行力的坚决。如果员工执行力不强，说明企业领导的管理能力不过关。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾给华为起草《华为基本法》的包政教授，在一次讲课中，描述了日本企业是如何给下属布置任务的，其中最有趣、有效的部分是：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>日本的大公司规定，管理者给员工部署任务时，至少要说五遍。</span></strong></p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一遍，管理者：“渡边君，麻烦你帮我做一件××事。”渡边君：“是！”转身要走。</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二遍，管理者：“别着急，回来。麻烦你重复一遍。”渡边君：“你是让我去做××事对吗？这次我可以走了吗？”</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三遍，管理者：“你觉得我让你做这事的目的是什么？”“你让我做这事的目的大概是咱们这次能够顺利地召开培训，这次我可以走了吗？”</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第四遍，管理者：“别着急，你觉得做这件事会遇到什么意外？遇到什么情况你要向我汇报，遇到什么情况你可以自己做决定？”渡边君：“这件事大概有这么几种情况……如果遇到A情况我向您汇报，如果遇到B情况我自己做决定。您看可以吗？”</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后一遍，管理者：“如果让你自己做这个事，你有什么更好的想法和建议吗？”渡边君：“如果让我自己做，可以在某个环节……”</span></p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这五遍讲完，员工再去执行特定比领导只说一遍的效果要好。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/125055416262e86bd1dc9f85.34719693.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在日常生活中，一样可以用这种“五遍法”来提升自我的执行力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如你想减肥：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一遍：</span></strong><span>先说服自己，让自己拥有强烈的减肥欲望；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二遍：</span></strong><span>写下自己减肥的目标和具体步骤，贴在家里随处可以看见的地方；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三遍：</span></strong><span>告知周围的朋友你想减肥，主要是为了什么，并让他们监督你；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第四遍：</span></strong><span>如果中途减肥想放弃，要有预备方案。比如提前发朋友圈告知朋友减肥计划，如中途放弃，见证人都可以向自己索要多少钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第五遍：</span></strong><span>想想有没有更好的健康减肥方法。比如，有没有必要请专业减肥教练指导督促，加入一群减肥朋友圈一起互相鼓励减肥。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这五遍分别让你明白：<strong>目标是什么、再次明确目标、目标的全局观、完成目标会遇见的障碍、目标的B计划。</strong>明确了这些再去执行，才会有实质性的收获。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所有，下次要设定目标的时候，千万别忘了一定要说五遍，这是提升你执行力最简单有效的方式。</span><span></span></p>

<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前，有两个和尚，一穷一富，都想去南海朝圣。富和尚很早就开始存钱，穷和尚却仅带着一个</span><span>钵盂</span><span>就上路了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过了一个月，穷和尚从南海朝圣回来，富和尚还在做准备工作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>富和尚问：“尔困，何以往南海？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>穷和尚回答：“吾不往，则终日癫狂，行一步，则安一分。尔稳重，故尔在！”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>穷和尚的回答，意思是：</span><span>“我不去南海，就心里难受。</span><span>我每走一步，就距离南海近一点，心里也就生一分安宁。</span><span>你这个人个性稳重，不做没有把握的事，所以，我回来了，你却还没有出发。</span><span>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>世上所有的路，在出发之前，永远都是梦想；当上路之后，永远是挑战。但只有当你脚踩上去了，才知其远近和曲折，才能离梦想更近。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/52025972962e86bd2b24fb8.34290433.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>佩恩曾说：<strong>一个人如同一只钟表，是以他的行动来确定其价值。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一块不会走的手表，只是一块废铁。唯有行动，才能发挥他的价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一个执行力超强的人，和一个只会想不会做的人，注定会有不一样的人生。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，当你想做一件事的时候，别想太多，先立刻站起来，走出去，赶紧做。</span></p>
