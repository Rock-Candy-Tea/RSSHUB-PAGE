<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为世界知名的豪华汽车品牌，<a href="https://www.niaogebiji.com/tag/%E6%B2%83%E5%B0%94%E6%B2%83" target="_blank">沃尔沃</a>汽车一直以出色的安全性能，受到众多车主的青睐。尤其是搭载City Safety城市智能安全系统的沃尔沃XC60，更是被誉为“路上坦克”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>硬核的产品实力固然重要，但如何让消费者更直观地感知产品，产生对品牌的信任和喜爱，同样考验沃尔沃的营销实力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，沃尔沃就一改车企营销高大上的风格，联合<a href="https://www.niaogebiji.com/tag/TVB" target="_blank">TVB</a>官方“戏精”附体，携手黄智贤、黎诺懿、河国荣等一众TVB老戏骨，通过花式再现TVB剧集中的各种车戏名场面，将沃尔沃XC60的一系列安全性能呈现得一目了然。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">1</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>戏精再现港剧车戏名场面</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>高能反转植入产品诉求点</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在TVB经典港剧中，让观众记忆深刻的，一定有那些紧张刺激的车戏名场面。飞车、追车、撞车等等，使人肾上腺素不断飙升，一刻都不敢眨眼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>不过，如今TVB拍摄这些车戏，却面临着一些让人头疼的难题，甚至导演怒而宣布，不再使用沃尔沃XC60拍摄车戏。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是什么原因呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一场刺激的飙车戏拍摄中，导演设计的是中间一辆车假装变道，超车未果，从而引发了三辆车连环撞击的场面。结果在拍摄过程中，中间的车总是直接超出去，根本无法形成追尾效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原来，由于使用的沃尔沃XC60“危机意识”太强，可以辅助转向。</span><span><strong>在紧急时刻可以合理修正方向，保护车身安全，所以它自己就变道超车了。</strong></span><span><strong> </strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在拍摄另一场追捕戏时，导演安排前面车辆掉进沙坑，在沙坑里打滑空转，来营造紧张氛围。但没想到的是，掉进沙坑的车，总是轻松就出坑了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>导演以为是演员车技不行，但实际上是，</span><strong><span>非常安全的沃尔沃XC60“四驱太稳”，可以做到动力精准分配，轻松应对各种问题路面。</span></strong><span>遇到沙坑时，司机“轻轻一踩，就立刻脱困了”，根本不会被困在沙坑里。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有一场撞车戏的拍摄现场，导演希望演员突然加速，撞飞前面的路人。如此危险的一场戏，在拍摄时同样困难。因为在离路人很远的位置，汽车就刹停了，根本拍不出导演想要的效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原因同样出在沃尔沃XC60身上，</span><span><strong>其搭载城市智能安全系统，可以自动捕获障碍物踪迹。</strong></span><span>当发现前方有障碍物时，为了保护司机和乘客安全，就会自己刹停，根本撞不到人。</span><span> </span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看完这三支短片，我想你会和我一样，由衷赞叹沃尔沃的广告创意和脑洞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，沃尔沃通过和TVB合作，将飙车戏、追捕戏、撞车戏这些观众非常熟悉的港剧剧情，拍摄得紧张刺激，</span><span><strong>带给我们满满回忆杀的同时，又保证了高完成度，不失专业水准。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，在每一个小短片中，沃尔沃XC60扮演了一个“捣乱”的助演角色。随着“发生这种事，大家都不想的”这一经典台词，TVB这些高能车戏之所以不能顺利拍摄的原因揭晓——</span><strong><span>沃尔沃XC60“非常安全”</span></strong><span>。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于太过优秀，沃尔沃反而被剧组“慎用”。</span><strong><span>如此有趣的反转，非常生动地展现了沃尔沃XC60紧急避让辅助功能、全时四驱系统、城市智能安全系统等各种安全性能。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样有梗、有趣的营销方式，让沃尔沃XC60成功“抢戏”，更能让消费者对沃尔沃XC60的产品优势产生深刻记忆点，并形成对品牌的好感度。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">2</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>联合知乎进行深度内容合作</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>话题问答撬动多元互动传播</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，围绕着TVB剧集的话题并不只有车戏名场面，诸如经典角色、经典台词等，都是经常可以引发网络讨论的话题。还有网友总结了TVB电视剧万年不变的套路，比如开头出现的男女朋友必定会分手、捡摔破的东西一定会划破手指、用手帕捂嘴咳嗽必定会呕血、几乎每部戏都会上演车祸戏码……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些被热议的话题，给了沃尔沃更多元的营销思路。所以，沃尔沃此次还和知乎进行了深度合作，并邀请在TVB剧集中扮演过众多经典角色的演员黄宗泽，在知乎发出了提问：</span><span><strong>TVB里有哪些</strong></span><strong>「领盒饭」的</strong><span><strong>角色放现在还可以再抢救一下？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/871630666632977b2b86cd9.44572258.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在TVB电视剧中，确实有很多“领盒饭”的角色让人意难平，这样一个问题，足以引发网友的讨论热情。与此同时，TVB演员河国荣也参与其中，给出了他自己的回答。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当看到电视剧中有人横穿马路，很多人第一反应是他要领盒饭了，因为TVB剧中经常上演这样的戏码。河国荣也提出了这样的疑问。不过，当河国荣真的开车遇到横穿马路的行人时，这样的悲剧并没有发生。拯救路人的，是沃尔沃XC60的城市智能安全系统。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>河国荣的回答，承接了沃尔沃在三支短片中的利益诉求点，并实现了微博和知乎的联动传播。</span></strong></span><span>此外，沃尔沃还与知乎涵盖汽车、影视等不同领域的答主进行合作，从更加专业的角度回答黄宗泽提出的问题，解读沃尔沃XC60各项安全性能，让人们对沃尔沃XC60有了更充分的了解。</span><span> </span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/436932487632977b4e7dd11.04549386.jpeg" width="333" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截至目前，黄宗泽和林夏薇的相关问题已经有2227人关注，浏览量达到了311万，这对于沃尔沃来说，是一个非常不错的营销曝光和品牌声量，</span><span><strong>实现了在知乎站内的传播引爆，覆盖到了不同圈层的人群。</strong></span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">3</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>以经典文化为依托</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>赋能品牌价值内涵</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，沃尔沃选择和TVB合作，除了用经典港剧来撬动大众注意力、巧妙传达产品卖点之外，还有另一个非常温暖的渊源。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>成立于1927年的沃尔沃，如今已经有近百年的历史。在此之前，为了体现豪车调性，沃尔沃还有一个译名，叫富豪。甚至如今在香港，这一名字依然被沿用。在很多港剧、港片中，沃尔沃的出镜频率也相当之高，并被看作是成功人士的标配。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一个是拥有近百年历史的豪车品牌，一个是制作过一系列经典剧集的电视台，沃尔沃和TVB的合作，让我们看到了两个拥有辉煌历史的IP之间的惺惺相惜，一拍即合。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一点，沃尔沃还联合黄宗泽拍摄了一个短片，借黄宗泽之口，传达了沃尔沃在香港电影、电视剧的发展中扮演的重要角色，以及给予演员的安全保障。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于很多中国人来说，TVB意味着情怀，代表着辉煌，拥有着跨越几代人的影响力。通过TVB这一经典IP符号，沃尔沃可以拉近和中国消费者的对话距离，形成情感共鸣，同时也可以</span><span><strong>提升品牌内涵，让中国消费者更认同沃尔沃的文化价值，感知其近百年历史的厚重感。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是在品牌本土化营销中，沃尔沃非常亮眼的一次传播。</span></p>
