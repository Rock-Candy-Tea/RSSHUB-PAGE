<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">在B站第十三周年，B站月均活跃用户首次突破3亿，同比增长29%高达3.06亿，用户增长的背后是社区内容多样化的吸引。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">在13周年庆之际，B站董事长陈睿表示，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">能够不断的去产生宝藏UP主，既是B站的活力所在，也是B站的价值所在</span>。</p>
<h4 class="ql-align-center" style="border: 0px; margin: 25px 0px; padding: 0px; font-size: 14px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #818386; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; font-weight: 300; line-height: 30px;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p0.itc.cn/q_70/images03/20220914/c08427949f774850973b7a36c5b482aa.png" referrerpolicy="no-referrer">来源-B站</h4>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">UP主是B站的“第一生产力”，第二季度B站月均活跃UP主数量已经高达360万，同比增长50%，然而在百万活跃的芸芸UP主中，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">还存在着哪些宝藏UP主值得去深挖？</span></p>
<h1 style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 20px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #212629; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); text-align: center; font-weight: 600; line-height: 30px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">入驻百天，涨粉百万</span></h1>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">在短时间内“涨粉百万”一直是B站少有的存在，然而今年，B站相继有UP主「山城小栗旬的理发日记」、「帅农鸟哥」涨粉百万，成为B站2022年涨粉最快的UP主。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">近日，又有一位UP主接过涨粉大旗，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">入驻B站百天就涨粉百万</span>。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">飞瓜数据（B站版）UP主涨粉排行榜显示，UP主「俏佳人xxx」仅仅在8月就以新增<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">93.2w</span>粉丝的数据表现稳居涨粉榜榜首，自六月入驻B站以来已经涨粉百万有余。</p>
<h4 class="ql-align-center" style="border: 0px; margin: 25px 0px; padding: 0px; font-size: 14px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #818386; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; font-weight: 300; line-height: 30px;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p8.itc.cn/q_70/images03/20220914/345bed2360c54bc8b38e3431d61df3e0.png" referrerpolicy="no-referrer">来源-果集·飞瓜数据（B站版）</h4>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">UP主「俏佳人xxx」是律法知识科普型UP主，在三次元世界他刚刚通过司法考试正式成为一名实习律师。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">B站律法科普的风是从知名UP主罗翔进驻后开始吹起来的，但是基于律法的严谨、一丝不苟的圈层氛围，一般的法律知识UP主通常是严谨认真的形象和表达方式。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">但为什么俏佳人能火、能突出重围？别人是给观众认真普法，而<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">俏佳人是拿案例当段子</span>，一切在他身边发生的事情最后都有可能“戴上手铐”。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">这是他火的其中一点原因：<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">契合B站的学习氛围，不被素材左右创作风格。</span>俏佳人因为律所上班的缘故穿着西装革履、梳着大人模样的发型，但年仅23岁的他却对普法有着新的表达形式。</p>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">在作品《第一次坐牢是什么体验》中，1分48秒的视频，UP主凭借着他的说话艺术<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">十秒就能爆一个梗</span>，虚空铁观音、 美式寸头、1510等等，在视频弹幕分析中，UP主的爆梗技能拉满，带动观众也在跟风玩梗，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">收获近10000条弹幕互动</span>。</p>

<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">由此俏佳人的流量迎来第一次峰值，有超过<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">1107.94w</span>的用户对“第一次坐牢是什么体验”感到好奇，标题的吸引程度让这支作品<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">收获近6倍的播放粉丝比</span>，远超UP主平均水平311%，互动数据中，光是点赞量就突破100w，互动率高达12.06%。</p>
<h4 class="ql-align-center" style="border: 0px; margin: 25px 0px; padding: 0px; font-size: 14px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #818386; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; font-weight: 300; line-height: 30px;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p5.itc.cn/q_70/images03/20220914/9d40c5fccb824ded95d3e493da7ed6da.png" referrerpolicy="no-referrer">来源-果集·飞瓜数据（B站版）</h4>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">还有一点<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">出圈因素就在于UP主的标题封面</span>，参考其他法律知识的UP主可以看到，作品时长有长有短，封面和标题给人第一感觉更偏于法制节目的感觉，还是<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">趋于严谨专业的风格</span>。</p>
<h4 class="ql-align-center" style="border: 0px; margin: 25px 0px; padding: 0px; font-size: 14px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #818386; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; font-weight: 300; line-height: 30px;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p4.itc.cn/q_70/images03/20220914/6de2026e7bf2429fb338a2adcf3ba34e.png" referrerpolicy="no-referrer">来源-B站</h4>
<p style="border: 0px; margin: 25px 0px 20px; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #000000; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;">但是UP主俏佳人的作品封面<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">首先没有多余的文字</span>，只有UP主本人往期作品的“丑照”，表情都是“不正常的”，给人直觉这就不是一个正经普法UP主，再加上其<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">标题往往短小精悍</span>，并不致力于概括视频普法内容而是以无厘头式的语言去描述，这种“摸不着头脑”、“出其不意”的感觉反而最是能吸引人。</p>
<h4 class="ql-align-center" style="border: 0px; margin: 25px 0px; padding: 0px; font-size: 14px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #818386; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: none; font-weight: 300; line-height: 30px;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p6.itc.cn/q_70/images03/20220914/67d871f837eb4850bfbf0e92cb1a227c.png" referrerpolicy="no-referrer">来源-B站UP主俏佳人xxx</h4>
