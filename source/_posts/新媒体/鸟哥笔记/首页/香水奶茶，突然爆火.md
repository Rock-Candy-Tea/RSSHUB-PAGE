<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从玛丽莲梦露带火了香奈儿5号开始，“香水味”已然成为行走人间的品味名片。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个秋天，“香水味”突然闯入新茶饮赛道，掀起了一波“追香”风浪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，悸动烧仙草新推出一款会发光的“香水奶茶”，迅速火遍社交媒体，短短两天爆卖50万杯，相关话题流览量超过百万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70077011662540cc779d811.91220037.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>上市即成 “人气王”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说到祖玛珑香水，相信喜爱香水的网友都不会陌生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个因典雅香氛享誉全球的英国顶奢品牌，以自然简单纯粹的格调，备受诸多明星喜爱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次悸动烧仙草推出的新品，正是以祖玛珑经典香水为灵感，并命名为“</span><span><strong>英国梨&小苍兰</strong></span><span>”， 打出的广告语是 “一杯可以喝的香水”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/192395194563163ddf949866.13541677.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：悸动烧仙草微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从外观上来看，方形的瓶身和香水瓶极为相似。包装上也以书法软膜袋、LED串灯和蝴蝶结的组合，完美戳中了女性的审美喜好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据介绍，这款新品采用新鲜啤梨、芒果冰沙，加上悸动原创“小苍兰茉莉绿茶”茶底，混合低温提纯厚牛乳，并搭配桂花冻等小料。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从原料的使用，似乎没能看出太多新意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/117245338463163de00b79a3.01919398.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：悸动烧仙草微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，小苍兰茉莉绿茶和桂花冻复合产生的花香，糅合啤梨、芒果自带的果香，使这款颠覆传统的新品，更像是一瓶花果调的秋日香氛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，在官方的介绍中，也以前调为啤梨、厚牛乳，中调为茉莉，后调为小苍兰的“品香”引导，拿捏创造了令受众闻香心动的氛围感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这款分为尊享版礼盒包装及流萤包装的新品，上市当天即成 “人气王”，吸引了众多网友慕名打卡，一度处于断货状态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，新茶饮赛道的竞争愈演愈烈，“英国梨&小苍兰”的面市，似乎为茶饮品牌的创新，开启了另一种可能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137231867662540cc8699195.09342959.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>新茶饮跨界追“香”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在小红书上，输入“香</span><span>水奶茶”，会跳出7万多篇相关笔记。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有开在重庆火锅店里的ins爆款，有隐藏在上海长乐路的香水牛奶店铺，也有开在三里屯胡同里的高颜值小店……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在悸动 </span><span>“英国梨&小苍兰”</span><span>出圈之前，杭州有家用香水瓶灌装饮品的香水奶茶店就曾在小红书上爆火。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上半年书亦烧仙草推出橙漫山茶花时，也区分介绍产品的前中后调，引导消费者像品味香水一样品尝奶茶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/140936833263163de062c492.11524862.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：书亦烧仙草微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从某种意义上，复合型花香茶与国外的“花草茶”有几分相似。都是通过将几种不同的香气叠加，释放具有吸引力的舒畅惬意香气。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而“香水奶茶”的出现，可以看作是在“花草茶”基础上，迎合现今消费者喜好的升级改变。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是装在简约玻璃方瓶中的满满高级感，还是参考香水的前中后调营造氛围感，都赋予了香水奶茶由外及内的“香水”属性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/69296737763163de0ac9ac4.36303279.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：悸动烧仙草微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，<strong>通过香水和茶饮这两种完全不搭界的产品来建立心智衔接点，将有助于拓宽受众，吸引更多网友试新</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，把香水概念延伸到饮品上的跨界追“香”，又会成为新茶饮品牌的爆款方向吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/127618355762540cc97137d2.72071747.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>不局限于“喝”本身</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在人类的所有感官中，嗅觉是最敏感的，也是与记忆和情感联系最密切的感官。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>具有永远开放、独特性以及情感吸引力三大优势的气味营销，是强化正面情感的最佳途径之一，甚至影响着受众的消费抉择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，面对大多数受众对香气的“无力抵抗”，让香水摇身变为茶饮，算得上是一个值得尝试的出圈路径。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/104476047963163de13582e6.21243344.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：悸动烧仙草微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从悸动烧仙草推出的“英国梨&小苍兰”来看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，</span><span><strong>“可以喝的香水”“会发光的奶茶”，能够以吸引眼球的差异化卖点，制造话题并激发网友好奇</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>香水在网友的刻板印象中，属于化妆品类。因此，如果有人告诉你“香水可以喝”，就充分打破了固有认知。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对女性消费者来说，看到“小苍兰”三个字，或会脑补联想这款奶茶的味道，无形中制造了消费向往。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/101795402163163de170a708.51700334.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：悸动烧仙草微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而“会发光”更是满足了网友的猎奇心理。通过抖音、小红书等社媒平台的高效传播，迅速树立了品牌标签，预热消费转化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，</span><span><strong>以“网红”颜值和产品特性的搭配，提升了奶茶作为社交货币的价值属性</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这款以祖玛珑香水为灵感的奶茶，无论是在外观颜值还是包装细节上，都做足了社交货币的创意构思。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，采用与香水瓶极为接近的方形瓶，塑造了高端精致的外观，从视觉上突出了奶茶的“香水”格调。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/32067018963163de1a57260.33223265.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：悸动烧仙草微博</span></span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并且，外包装上采用的国风书法字体，与瓶身的大牌英文共同呈现却不违和，既现代时尚，也颇具国潮味道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再加上串灯和蝴蝶结配饰，充分拿捏了一杯奶茶的浪漫氛围，扩大了被称为小仙女“出街神器”新品的传播范围。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，在原材料的选用和食材的呈现方式上，考虑多种花香之间的充分调制适配度。以瞬间打动受众嗅觉进而链接味觉，带来愉悦放松的“品香”体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/197749892563163de1f06341.53404328.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：悸动烧仙草微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，</span><span><strong>作为个性品位的象征，香水品牌本身的喜爱拥趸者，或将成为“香水奶茶”的潜在用户</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小红书上，曾出现过一款名为“山茶花邂逅5号”的香水奶茶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当山茶花、5号等关键词，出现在新茶饮的命名上，也让更多网友在新奇之外，奔着对香奈儿品牌的认同而买单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/56609598863163de2300bc3.12223331.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><strong><span><span>●</span><span>图源：悸动烧仙草微博</span></span></strong></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>当品牌需要在不同的消费场景，寻找与受众感官联结的机会，加深受众对品牌的印象时，香气跨界加入一杯茶饮的丰富可能，<strong><span>创造和强化了产品特性，</span></strong>让茶饮在消费者心中的价值感，已经不局限于“喝”本身</span></strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如悸动烧仙草在介绍中所说，一瓶“英国梨&小苍兰”，就承包了一座小型剧场，悄悄地埋下了惊喜伏笔。</span><span></span></p>
