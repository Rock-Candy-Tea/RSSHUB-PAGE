<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>历经多年的长短视频之争，终于出现了转折点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月19日，<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3" target="_blank">抖音</a>集团和<a href="https://www.niaogebiji.com/tag/%E7%88%B1%E5%A5%87%E8%89%BA" target="_blank">爱奇艺</a>宣布达成合作，将围绕长视频内容的二次创作与推广等方面展开探索。<strong>这标志着长短视频平台开启合作共赢新模式。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/163749825662d7aa865d6ca1.95770307.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据公告，<strong>抖音、西瓜视频、今日头条等平台用户都可以对爱奇艺授权的长视频内容进行解说、混剪等形式的二创，</strong>包括“迷雾剧场”为代表的爱奇艺自制独播剧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99617616762d7aa869c4f22.61178448.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>话题#爱奇艺和抖音集团达成合作#随之登上微博热搜，阅读超1.2亿，<strong>网友纷纷评论道“格局打开了”，“解说圈迎来狂欢”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音千万粉博主“大聪明和魏乐乐影院”对此感到兴奋：“作为一名影视创作者，由衷感谢爱奇艺的选择，此次与抖音达成合作，不光有利于行业发展，更是给予了创作者更大的创作空间，让长视频与短视频之争得到了缓和，我们一致为观众提供更好的视听享受。”同时，</span><span>他更期待今年刷屏的《警察荣誉》《心居》等爱奇艺自制剧目可以在抖音与大家见面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当抖音与爱奇艺就影视二创“握手言和”，<strong>将对影视博主们带来哪些影响？长短视频平台会就此走向“和平年代”吗？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>抖音牵手爱奇艺，破解版权困局</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去，因为合理使用的标准比较模糊，影视二创短视频经常因为版权问题而被下架。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>某影视博主告诉我们，之前在抖音陆续上传的6期《琅琊榜》解说视频，基本都在1小时内收到审核未通过的通知，<strong>原因是“可能涉及侵犯知识产权”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/30831348362d7aa86d25c77.33444543.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>博主提供的作品审核未通过消息</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年6月，抖音百万粉博主“刘老师说电影”解说《警察荣誉》的一期视频刚上热搜就被下架，后来又恢复了，他感慨道“现在解说不好做”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/150659931062d7aa8727ed12.86905355.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再早一些，2017年，以“5分钟带你看完XX电影”系列视频走红的谷阿莫就曾因为未经授权对影片进行二次创作，而被几家版权方起诉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了在长视频和短视频平台的炮火中求得生存，不少博主已经降低使用原作画面素材，或者试图通过与版权方、平台方合作的形式来规避法律风险，但这并不是万全之计。<strong>如果能够明确获得某部作品的授权和二创尺度说明，那么对博主来说将是更可靠的创作保障。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/211029315762d7aa874ea990.14409221.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次合作，爱奇艺将向抖音集团授权其内容资产中拥有信息网络传播权及转授权的长视频内容，<strong>用于短视频创作。双方对解说、混剪、拆条等短视频二创形态做了具体约定，将共同推动长视频内容知识产权的规范使用。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>爱奇艺方面在接受媒体采访时表示，具体约定包括单条二创短视频对正片内容的引用时长，合作明确约定由双方官方运营账号和爱奇艺授权运营的创作者账号对授权内容进行拆条传播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，据媒体报道，抖音将实现跨端跳转至爱奇艺，<strong>即用户在抖音观看二创内容时，可以点击跳转到爱奇艺观看对应的长视频正片。</strong>这个功能上线后，将对长视频宣发、拉新促活、以及会员收入带来利好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14317663362d7aa877c6217.64609231.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>爱奇艺创始人、CEO龚宇表示，这是双方在尊重和保护知识产权、探索合作双赢上迈出的重要一步，具有里程碑意义。<strong>相信这次合作有助于优化网络视频行业生态，拓宽合作渠道，创造更多价值，实现平台、创作者和用户的多方共赢。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>影视二创博主的春天来了吗？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>得知抖音宣布与爱奇艺合作的消息，不少影视创作者感到振奋。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“爱奇艺向抖音开放版权后，我们影视博主的内容选择会更加多元化，可以直接使用爱奇艺独播片源去制作内容。”有好戏MCN运营经理火锅表示看好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有好戏MCN入局抖音两年多，旗下包括“毒舌电影”“大卫疯剪”“灰袍真探”等多个千万粉丝的影视账号，全网粉丝量达4亿左右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两平台合作消息已公布一天，火锅还未收到具体的抖音二创细则，<strong>“从公布消息到一步步落实可能需要一段时间，我们现在仍然按照以前的模式先申请版权，再去解说或混剪长视频”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然需要时间等待落地，但看到希望的苗头就已经是一件好事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拥有550万抖音粉丝的博主“秀才侃娱乐”提到，“现在好了，以后爱奇艺的影视剧我们可以放心大胆创作”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“秀才侃娱乐”坦言，有时候面对平台播出的影视剧，<strong>明明自己非常想做成节目，但因为怕被下架，一张截图都不敢用，只能干说，束缚了自己的创作的灵</strong><strong>感。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如解说爱奇艺自制剧《警察荣誉》，当时由于版权原因，“秀才侃娱乐”不能使用原剧片段，讲到剧中长镜头的运用、演员的精彩表演，不能配合画面会显得不够直观。这种情况对短视频创作以及影视剧发展是双输的局面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/157932156962d7aa87b10c21.88569298.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当爱奇艺版权在抖音放开后，<strong>不仅为影视创作者提供了更丰富的素材库，也带来了更多商业变现的机会。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>火锅介绍，影视创作者在抖音主要通过影视宣发、品牌广告、流量激励等形式进行变现，其中帮助热播影视剧进行宣传推广占收入大头。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以近期热映的电影《神探大战》和《人生大事》为例，两者抖音话题的累计播放量达101.6亿次，其中包括影视创作者发布的宣传推广片段。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18748494862d7aa87eac1b4.41001685.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>影视宣传从业者薛瑞认为，现在影视宣发大多将抖音作为主战场，比如在官方账号和其他短视频账号铺一些节目卡段，爱奇艺走在前面肯定比其他几个视频网站更方便，鼓励大家去抖音做二创，版权问题也不用担心了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于观众会不会因为看过短视频解说，而跳过原剧正片，火锅认为并不成立，<strong>“一个好的影视账号解说完电影或电视剧之后，是能够引导观众主动观看原片的，因为原片细节更丰富，也更精彩”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/114229195062d7aa884c6923.98507015.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>部分网友评论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频创作者需要长视频提供选题、素材和广告来源，长视频也需要短视频来宣传、引流，长短视频之间存在着共赢的空间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《2021中国网络视听发展研究报告》显示，<strong>超六成用户因在短视频平台上刷到相关精彩片段才去观看长视频节目。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/31333722462d7aa888354c6.63529071.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>整体来看业内是一片欢呼声，但抖音和爱奇艺的合作是否意味着影视创作者将迎来春天，我们需要时间去观望。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前长短视频平台之间尚未完全开放合作。抖音和爱奇艺官宣合作后，腾讯视频辟谣了与快手达成战略合作的消息。因此，薛瑞猜测如果某一平台扶持二创，其他平台的二创限制或许会更严，比如说打击“一稿多投”等现象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，某业内人士认为影视解说不一定就此复苏，因为双方平台对二创具体形态做了规定，<strong>版权还是捏在长视频手中，直接搬运或负面吐槽的影视内容是否被允许发布，相关界限仍不清楚。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/33714015562d7aa88e0c7b8.78405636.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>有影视博主在相关话题下提出疑问</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>长短视频之争并非“零和博弈”</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾几何时，长短视频平台之间争议不断。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年4月9日，超过70家影视传媒单位发布《关于影视版权保护的联合声明》，表示将对未经授权剪辑、切条、搬运、传播影视作品内容行为发起法律维权；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当月的两周之后，腾讯视频、爱奇艺、优酷等视频平台又联合多家影视公司及500多位艺人发布《倡议书》，呼吁短视频平台推进版权内容合规管理，立即清理未经授权的短视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此后在去年6月的中国网络视听大会上，长视频网站以版权问题为核心向UGC和短视频平台开炮，出现了“猪食论”等激烈言辞。同月，字节跳动起诉腾讯视频上存在大量电视剧《亮剑》的侵权视频，侵害了自己五年内享有的该剧信息网络传播权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但在一年之后，长短视频平台又选择握手言和，共同走出零和博弈，走向合作共赢。这不仅是影视创作者期待看到的画面，也是长短视频平台的必然选择。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年全国两会期间，全国政协委员、中国电视剧协会副会长刘家成提交了《关于加大短视频侵权惩治力度和创新授权机制的提案》。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我不反对短视频，但我反对短视频侵权。”他认为，<strong>应当在“先授权后使用”的基本原则下，各方协商短视频二度创作授权机制。 </strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>资深媒体人金叶宸认为，抖音与爱奇艺的合作其实是“情理之中的意料之外”。情理之中指的是长视频平台与短视频平台的合作必然会发生。<strong>早在前两年，抖音和西瓜就有相关业务的探索和结合，很多抖音二创和直播看电影的内容会导流到西瓜视频小程序的完整版视频。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，他也认为短视频的内容分发效率优于长视频，“中国的互联网视频行业必须走出创新的路，寻求合作双赢。让人意外的则是爱奇艺先迈出了这一步”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在，标志着长短视频平台破冰的这一步终于来了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“这是抖音和爱奇艺合作的一小步，却是关系到影视剧未来发展的一大步。”“</span></strong><span>秀才侃娱乐”说道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/75997793762d7aa892c8118.41918014.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着国内互联网进入存量时代，不同平台之间将不再以内容长短划分阵营，而是以各自的利益需求取长补短，选择合作伙伴。对于抖音来说，与爱奇艺达成合作便是互惠互利的最佳解法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此之前，今年3月，抖音宣布与搜狐达成合作，旗下平台获搜狐全部自制影视作品二次创作相关授权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时，抖音便提出一个期望：“影视二创是一个新事物，我们也在努力推动和更多版权方，尤其是拥有众多独家版权资源的长视频平台达成合作，为创作者和行业提供更好的版权解决方案。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从搜狐到爱奇艺，抖音逐步接近长视频平台的“第一梯队”，一点点积累了更多的影视版权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当抖音提供了版权沃土，创作者们也获得了更大的发展空间。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据火锅观察，经过近几年发展，抖音影视领域正逐渐趋于饱和，内容质量也出现参差不齐的情况。但观众的注意力是有限的，当供给大于需求，创作者们纷纷开始关注如何用更高的质量去吸引观众，比如真人出镜，或者将影视解说改编成歌曲等新颖形式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“当大家开始在内容领域思考问题，虽然卷了些，但对于生态长久发展是有好处的。”对此，火锅表示了肯定，他也将今年目标放在了如何做出更加差异化和精品化的影视二创内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在，抖音向湖面丢出了一枚石子，接下来又将泛起怎样的涟漪，是否有新类型影视创作者的出现，我们不妨投以期待。</span></p>
