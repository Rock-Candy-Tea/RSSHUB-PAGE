                        <section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">直播带货兴起时，明星带货就是直播带货的热门话题，明星带货也成为了GMV高峰的密码，似乎只要有明星在，这一场直播带货流量就有了保证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">朱梓骁、黄圣依、贾乃亮常驻直播带货后，汪峰也入局直播带货，7.28日开播两场直播，带货预估超过2000万GMV。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">明星入局直播带货是惊鸿一现还是持续输出？</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 20px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: center;">带货4场，进入头部带货榜</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">8月份，汪峰一共开设5场直播，30天累计直播销售额预估超过5034.3w，直播间人气仅次东方甄选与疯狂小杨哥。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/dfaab873044207477c6cb152c182e57c.png" height="579" data-ratio="0.4527778" data-w="1080" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">△图片来源：果集·飞瓜数据带货达人榜</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">汪峰在直播首秀时，邀请抖音头部主播“疯狂小杨哥”一起直播，直播点赞量破亿，平均在线人数超10万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/82008cc152273f0b5dc32f7682d31342.png" height="1014" data-ratio="1.538695" data-w="659" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">△图片来源：汪峰直播间截图</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">直播中为带一款洗发水产品，汪峰在直播间给小杨哥洗头，引爆粉丝笑点，直播间气氛十足。这种有点反向操作的直播带货形式，容易吸引粉丝观看，引发粉丝兴趣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/271ce7e6480d43899c685602bb937ddd.png" data-ratio="1.6876791" data-w="349" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">△图片来源：汪峰直播间截图</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">汪峰8月份直播4场，直播场均人数峰值预估超过22.5w，直播间人气居高不下的原因，除了汪峰是人气歌手以外，另一个原因是他们加大了付费投流。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">直播间带货品类有三大类，日用百货、服饰鞋类、母婴用品，平均客单价在50.8元，其中防水手表、除螨液、懒人拖把等商品销量平均超过2万单，产品价格均在9.9元到29.9元之间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/634de95519affaada2330506ce3c0eb3.png" height="527" data-ratio="0.412037" data-w="1080" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">△图片来源：果集·飞瓜数据</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在汪峰前几次的直播中，虚拟充值带货销量较高，这些通用性产品受众面广，未能体现直播间特色，与汪峰粉丝不能精准匹配，价格与其他直播间对比并无优势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">直播间粉丝画以31~40岁之间的宝妈、宝爸为主，吸取前几次直播的经验，汪峰8.16日直播间上架的产品，均以直播间粉丝画像选品。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/d469d085d89d8e207aca74a8edc7378c.png" height="613" data-ratio="0.7652934" data-w="801" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">△图片来源：果集·飞瓜数据</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在直播话术上，汪峰对比前几次直播，有不小的进步，在8.16日直播中，讲解一款国产手表说到：</p>
<p>“咱要支持国货是不是？今天在我直播间直接拦腰砍断，只要1500块一支表！”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">副播同时用话术配合，“原价一万多的手表，直播间出售1500元，只有在汪峰直播间才有！”一唱一和的话术配合，不但激发粉丝的爱国情怀，也体现了直播间的价格优势。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 20px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: center;">明星与头部主播一起直播<span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">成为流量密码？</span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">近年来，明星带货已成趋势，许多明星加入到直播行业中，阿里巴巴曾经专门统计，截至2020年，中国演艺界99.5%的明星都走入了直播间。据媒体报道，有80%的明星签约了专业的MCN机构。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">观察几位头部明星的GVM，黄圣依7月份带货GVM预估1.8亿、张檬小五夫妇7月份带货GMV预估9000万、朱梓骁7月份带货预估8712.9万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在数字背后，是明星带货拥有着巨大的流量，与素人主播相比，明星主播在开播前的知名度与流量就较大，更容易吸引粉丝的信任与喜爱。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">董先生更是在直播间利用好这一条件，不断在直播中邀请明星做客，让明星与观众互动，据不完全统计，董先生邀请过赵雅芝、朱梓骁、张雨绮、江疏影、舒畅、张馨予、王子异等众多明星，到直播间带货。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/29d3cd66579f2fe4dd928eaa682f304e.png" height="554" data-ratio="0.9358108" data-w="592" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">△图片来源：董先生账号视频截图</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">明星助阵董先生直播间每一场的带货GMV成绩亮眼，众多明星在董先生直播间中并没有沦落为背景板，在直播中董先生与明星之间互动感十足，根据不同的明星特点，直播间也呈现不同的直播形式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">与明星合作的直播间，加大了自身流量，吸引更多的粉丝进入直播间。但明星如果想要继续在点火领域深耕，需要培养更加专业的带货能力，否则直播首秀即为巅峰。</p>
