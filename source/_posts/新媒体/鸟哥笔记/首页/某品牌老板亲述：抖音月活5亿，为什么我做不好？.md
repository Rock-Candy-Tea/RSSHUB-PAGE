<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>昨天傍晚，和一个品牌创始人，聊了1个多小时电话，我们服务他品牌+小红书，他自己做抖音和直播。电话中，对于他做抖音的现状，我说了一句气话：<strong>你要是这么做的话，就别创业了</strong>。没想到，到了晚上，他又打我电话，说谢谢我，被我的话刺醒了，有道理，要按照电话里沟通的新方向进行调整。（点赞老板大格局）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本文，聊聊一个普通品牌做抖音的起因+经过+结果。希望给到其他创始人一些参考。尤其是卖中高客单价的产品，老板麻烦醒一醒，不要指望新品牌上直播，找流量渠道就可以卖货了，都2022年了，时代真的不一样了。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>一、</strong></span><span><strong>做抖音，遇到的问题</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看到全世界都在说抖音能赚钱，对于品牌来说，犹如沙漠看到水源。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>背景：</span></strong><span>这个老板创立的品牌，是一个非常卷的赛道，类似于母婴行业，非常需要信任。他杀进去做抖音，可是干了2个月，投入自己很多的时间、精力，却没有效果。他遇到的问题是：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、抖音低价转化差：</span></strong><span>本来产品是中高客单价，但是为了抖音拉新，专门开发了引流款，干到9.9元，结果发现转化特别差，本来还指望低价引流进行转化，没想到，基本没转化，都是羊毛党。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2、为了组品开发产品：</span></strong><span>听高手分享，抖音很重要的是“组品”，但是之前店铺产品就2个，加之竞品店铺里很多SKU，于是创始人开发了好几个新品，结果营销没跟上，新品开发导致了资金紧张；<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3、福利款人气差：</span></strong><span>为了直播间做福利，开发了一款低价产品，想通过低价拉直播间人气，结果价格是低了，但是没拉动人气，结果造成一堆库存；<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>4、短视频数据差：</span></strong><span>平时也在拍短视频，但是数据一般，直播每天都强迫自己播，效果刚开始还可以，越后来越不好。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>5、依葫芦没画好瓢：</span></strong><span>研究了很多行业竞品，他们怎么做的，都模仿一遍，结果发现被人能做起来，为什么自己就是做不起来？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>6、精力跟不上：</span></strong><span>主播专业性不行，所以在主播的成长过程中，自己也做直播，结果花了时间，身体也被掏空，还影响其他工作。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>被问及为什么这么做的时候，他的想法是：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、别人都是低价引流所以也想试试，总有用户用了体验装感觉不错会购买正价产品的；<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、组品的前提是要有品，否则根本组不起来，而且SKU多，给到消费者更多选择，转化会更好；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、只要价格底，消费者就愿意下单，直播间肯定就有人气；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、抖音流量这么大，经常直播，肯定能卖的出去。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5、靠谱主播不好招，总要试试抖音行不行，只能自己先上，而且自己专业，可以解决消费者的信任问题。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不知道大家看完什么感觉？我听完的感觉就是，<strong>这是非常多电商的老板真实写照。对抖音不懂，却深信抖音能赚钱。</strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>二、</strong></span><span><strong>做品牌，不是靠模仿</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，这个老板走了一个捷径，但是没走好：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>模仿其他品牌好的直播间打法，都在自己直播间演绎一遍，不管是低价引流，直播间背景，运营技巧等等，但是效果不好，自己还非常纳闷，为啥别人能做起来，我们做不起来呢？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这不仅仅是抖音，不仅仅是这个老板，而是很多老板的真实写照。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不懂，做不好，那就多看看做的好的品牌，学过来变成自己的，这确实是一个好的方法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但最大问题是，学其形而未知其本。你看到的成功是冰山上的，而冰山下的，你是看不到的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/29924548862cf0b496953a8.56706812.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个品牌的成功，肯定不是因为做对了1件事情，而是作对了一整套事情。</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>学习，一定要学习底层逻辑，而不是简单复制打法。</span></p>
</li>
</ul>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>三、</strong></span><span><strong>抖音，要知道底层逻辑</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>全世界都说抖音好，觉得只要进去就能卖货，但是真正进去之后发现是陷阱。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做了2个月，投入了时间、精力、财力。想继续做，但是迷茫，亏钱，不做，又舍不得，毕竟都干了2个月，说不定第三个月就起来了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我给的结论是：放弃。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>理由如下：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我和创始人说，站在我们2个人的层面，讨论的事情不应该是抖音如何优化，怎么能做好。而是应该先定<strong>战略和方向</strong>：目前阶段，做抖音是不是正确的，如果是错的，那换平台，如果是对的，那再讨论如何优化。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是很明显，目前的财力精力能力，都不适合做抖音。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人以为做<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%9B%B4%E6%92%AD" target="_blank">抖音直播</a>很简单，弄个直播间，招起来一个团队，开播就能卖货。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果商业都这么简单，那赚钱也太容易了吧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播间，就是一个内容生态，要做好不容易：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、直播间要有差异，否则，消费者凭什么要停留，更别说购买了。现在的直播间，好看，画面精美已经是标配了，如果只是好看精美，根本没有停留的理由了，要思考，如何把直播间内容化，，让消费者愿意停留；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、停留后的直播间运营，也是非常重要的，光这个团队的配置，需要的能力，很多初创团队都不具备；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、不要看别人直播间GMV几百万几千万的，行业的事实就是，很多品牌直播间不赚钱，因为很多流量都是付费购买的，结果品牌都在给抖音打工而已。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、抖音的月活是非常大，可是对你来说，把控不住的，就不是你的。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>四、</strong></span><span><strong>解决方案</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那到底哪个平台，是适合这个品牌发展的，初创品牌各种条件有限情况下，怎么办呢？我给出几个解决办法：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>1、创业如何找优势：</strong>找到自己最强的地方，放大。创业不是木桶效应，把所有短板补充好，而是把长板发挥最大。这个创始人起家因为私域，而现在为了随大流，私域都忽略了。本来产品所在的品类，就是一个需要决策的品类，而且竞争非常激烈。如果老用户都维护不住，却要花更多时间精力抖音拉新，得不偿失；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>2、关于品牌定位：</strong>找到自己品牌的定位，品牌的核心优势，在于创始人的故事以及产品的专业性。那接下来，无论是内容，私域，详情页都要聚焦于这个点，反复讲，时时刻刻讲。产品贵，一定要通过情感，故事，产生溢价。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>3、关于底层逻辑：</strong>很多时候大家喜欢模仿优秀的品牌，是捷径，但是为什么很多品牌却迷失自己呢？核心是根本没有掌握优秀品牌的底层逻辑。竞品为什么直播间低价引流可以，是因为他们产品本来就中低价格段。你以为的低价，其实和正价相比，也是完全可以转化，而这个品牌的低价和正价，差了一个鸿沟，只会被羊毛党薅。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其他竞品为什么直播间也是一样套路，就可以有很多的转化，是以为他们做了很多短视频，合作了达人。你一个新品牌，种草很少，一上来就指望直播间卖货，想啥呢？赚钱这么容易吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>4、关于营销推广：</strong>贵的产品，一定不是降价，不赚钱，何必创业。存支付宝不好吗？一定要通过内容，提高产品价值。这个预算有限，小红书一定是很好的选择，真要再加一个，我觉得是知乎。所以，如果创始人继续躬身入局，一定是拍摄大量短视频，而不是花大量时间直播。短视频同步到小红书，抖音等等；通过自己的故事和背景，建立信任度；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>5、关于复购：</strong>我和他说过好多次私域，但是因为时间都用来直播，开发新品，私域没有启动。贵的产品，拉新成本高的，好不容易拉新过来的用户，因为没有维护，白白流失，是非常可惜的。如果做好用户积分体系，做好复购，做好体验做好服务，做好老带新。这是你曾经发家的杀手锏，现在为什么都忘记了呢？这个必须要做起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>昨天晚上，他又和我打电话，说开了一场抖音直播，和自己粉丝告别，讲述了自己的问题，因为盲目开发新品，降价让大家卖，违背初心，其实是非常不对的，讲了很多自己和产品的故事，没想到，反应很好。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是的，没有人会随随便便买一个自己不熟悉的高客单产品，同样，也没有人会排斥一个真诚，像朋友一样可以交流的品牌创始人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>成交的核心是信任，而内容，是承接信任最好的载体。</span></strong><span></span></p>
