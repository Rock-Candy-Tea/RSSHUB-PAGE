<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果提起小时候的零食，“喜之郎”必须得有姓名。毕竟，在座各位大概都有扒拉着家长、非要买喜之郎果冻的童年吧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经的零食顶流，到上李佳琦直播间都得被归进<strong>“怀旧零食专场”</strong>，<span><span>喜之郎作为</span>“时代的眼泪”，当之无愧。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9279760162e066764d2334.62808442.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：“<span>李佳琦Austin</span>”</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你打开喜之郎的官网，还可以看到古早画风的海报，粉色的背景、QQ秀一般的人物形象，加上<strong>“水晶之恋”</strong>这样略带非主流感觉的产品名称，一时不知道自己到底是在2022年还是2002年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/178091941662e066768b59c3.20387154.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比以前看电视的时候，喜之郎的广告见缝插针地洗脑观众，它这些年的存在感越来越低，最近一次引发关注还是因为旗下的<strong>“优乐美”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/53780955162e06676dc1a39.73734471.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想当年，靠着周杰伦的一句“你是我的优乐美”，这个品牌红遍大江南北。现在，周董带着新专辑回归，优乐美却还用着这么多年没变过的广告风格，甚至掉进了商标的坑里，沾了一身泥。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/181119082962e0667750af04.08320863.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：喜之郎官网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，北京法院公开了，<span>广东</span><span>喜之郎公司</span></span><span>（下称“喜之郎公司”）</span><span>和<span>深圳云联数娱电子有限公司</span></span><span>（下称“云联公司”）</span><span><span>关于“<span>优乐美</span>”商标侵权纠纷</span>的二审行政判决书，喜之郎没有成功捍卫自己的优乐美。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2009年，云联公司申请注册“优乐美”的商标，2010年获准注册。</span></strong><span>据资料显示，该商标涵盖的商品/服务包括“MP3播放器; 电声组合件; 电视机; 计算机周边设备; 扬声器音箱; 影碟机”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>直到2020年4月，喜之郎公司对云联公司的“优乐美”商标提出无效宣告请求。10年间对雷同商标不管不问，喜之郎公司确实也有点心大，而代价便是知识产权局以“超过五年诉讼时效且未获得驰名商标保护”为由不予支持。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>且不说其他，这两个“优乐美”商标属实过分相似了，若说云联公司的商标是喜之郎公司商标的底稿，恐怕也有不少人相信。因此，不少人认为，如此相似的商标，喜之郎肯定能轻松拿回商标，没想到结果令人大跌眼镜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/96616859962e06677ae4c56.08753686.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>显然，尽管喜之郎公司早在2000年就注册了“优乐美”商标，但并没有及时完善各个分类的商标注册，埋下了隐患，最终导致了今天的败诉。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽说在喜之郎长久以来的市场教育下，大家的消费认知已经将优乐美和速溶奶茶挂钩，并不会轻易为贴了“优乐美”标志的电视机买单。<strong>但雷同商标的泛滥，终究会对喜之郎的品牌价值持续造成伤害。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过话又说回来，喜之郎跌落神坛的原因，也不止是商标纠纷而已。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在“过气”之前，喜之郎也是风光了很长时间的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1985年，国内出现了首家果冻生产厂。隔年，<span>深圳市琼胶工业公司<span>推出了SAA牌果冻，受到广大欢迎，各地的果冻厂家也像雨后春笋般冒了出来。</span></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>时间来到1993年，毕业于食品工程专业的李永军在国企当技术员，他也看到了果冻的商机。</span></strong><span>于是和兄弟们凑了40万元，一头扎进了果冻行业，谁也没想到他们后来能成为这一行的龙头老大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有了起步的钱，还需要一个响当当的品牌名。<strong>李氏兄弟选择了英文“Strong”，寓意强健、强大。</strong>而中文的“喜之郎”，据说来自李母的误会音<span>译，将李永军念叨的<span>“Strong”听成了“喜之郎”，在无意中说道，<span>“你都还没有成亲，哪来的喜之郎？”</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自此，“喜之郎”正式诞生，并开始建立自己的果冻王国。在整个过程中，喜之郎对时机的把握之精准，让不少创业者羡慕。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>为了和当时市面上的无数果冻区分开来，喜之郎选择优先切入儿童市场，将产品的规格、包装都量身定做，完成了江山奠基。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>五颜六色的果冻，滑嫩的口感，再加上符合当年审美的透明小书包，分分钟让小朋友在货架边走不动路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/67268546962e066787947b0.04823736.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：喜之郎官网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>现在常有人总结，喜之郎不行是因为不会创新和营销。实际上，这两者恰恰是喜之郎在发展前期的核心竞争力。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1998年，《泰坦尼克号》热映，观众在为浪漫爱情感慨，李永军却看到其中的商机。于是，喜之郎在全国投放以电影为蓝本的贴片广告，并卡着当年情人节的热点推出了“水晶之恋”果冻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>透明的果冻里躺着一朵玫瑰花，不同颜色代表不同的口味，还搭配了对应的“玫瑰心语”，精准狙击当时情窦初开的少男少女。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/76926247962e06678e400a4.40247774.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：喜之郎食品旗舰店产品详情页</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随后的1999年，喜之郎推出“<span>CiCi果冻爽</span>”，打出新概念——“可以吸的果冻”，广告模特都是活力四射的年轻人，一句“喜之郎<span>CiCi果冻爽，我只喜欢你</span>”火遍大江南北，显然是同样瞄准了热衷尝鲜的年轻人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/138390753562e0667a4e57d6.65679599.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样的营销能力仿佛刻在喜之郎的基因里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今的<a href="https://www.niaogebiji.com/tag/%E6%96%B0%E6%B6%88%E8%B4%B9" target="_blank">新消费</a>品牌做尽营销也没法让消费者迅速记住它们，绞尽脑汁想出来的广告语可能还不如创始人的一句“失言”广为流传。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而喜之郎的出圈广告语却有大把：</span><span></span></strong><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而这些广告语的出圈，也要归功于喜之郎搭上了电视广告发展的快车。</span></strong><span>在不断循环中，<span>小时候的我们对广告语的熟悉程度甚至超过课文。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据公开资料显示，在1999年，喜之郎就占领了我国果冻市场83%的份额；2014年，喜之郎的年销售规模达到85亿元。时至今日，喜之郎拥有1000多个经销商、上万个分销商，建立了遍布全国的销售网络，年销规模超100亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，29岁的喜之郎也并非没有困境。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>商标保护存在漏洞。</strong>此次的“优乐美”事件，便是喜之郎没有从一开始就重视商标造成的后果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>品类创新出现断层。</span></strong><span>相比频出大热爆款的曾经，现在的喜之郎已经很久没有推出能出圈的产品了，无怪乎总是被嘲“不努力”“躺平”。即便在今年6月推出了零卡茶冻，销量和经典款果冻相比，实在是差强人意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>品牌文化内涵较弱。</span></strong><span>喜之郎的成功有很大程度归功于营销，但大多是靠噱头或者热点，一旦热度下降，营销效果就会大大折扣，就像如今少有人问津的“水晶之恋”。喜之郎没有深挖本身的文化内涵，所以文化价值无法持续，也是如今品牌发展的短板。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从40W到85亿，喜之郎用21年时间让销售规模翻了2W倍有余，而从85亿到100亿，喜之郎花了8年。可见，喜之郎的增长速度已经放得极缓。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然在社交平台上可以看到很多人对喜之郎的怀念和玩梗，但有几个人会真的购买呢？如果喜之郎一直这样“躺平”，时代恐怕很快就要收取利息了。</span></p>
