<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先问一个“回忆杀”级别的问题：你，或者你的孩子，都是如何选择高考志愿的？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同代际的人会给出各自的答案：年长一些的人是拿着报考材料和老师商量、80后看过和志愿填报相关的电视节目、90后是互联网原住民，填报志愿时会用搜索引擎查信息，再年轻一点的，还会用上手机端的各种工具类App。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>方式变化的背后是信息媒介、服务平台的变迁，我们可以从变迁中看到信息流动和平台服务效率的提升。而如今的考生，也有属于他们的获取信息和启发的地方：短视频平台。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据快手官方的数据，今年高考期间，有4.36亿用户在快手上关注并消费高考内容，高考相关内容的搜索次数超亿次，同比增长87.8%。短视频平台正在成为最热门的高考讨论站之一。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时代变了，又一次“变迁”正在发生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81142426261420d300e45a3.29058312.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>不一样的夏日回忆</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每年高考季总要留下很多回忆，这些回忆由一系列具有时代感的要素构成——那本圈圈画画的志愿填报指南，那部存有高考分数短信的手机。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>而今年，不少考生和家长的回忆里应当会有快手。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在考生家长紧张准备高考时，快手就已投入运营高考内容。考前，考生关心如何调整心态，怎么复习更高效；家长则关心如何调整孩子的饮食计划，怎样才能更好地鼓励孩子，缓解孩子考试压力，这些问题，在快手上均能获得解答。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/80980161662c3b9eb0ba018.34413288.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>快手账号“育见新闻”在高考期间分享“直通高招”</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>考试临近，快手又和媒体合作开启了“高考加油许愿间”策划，7家媒体的31场直播收到了近400万次许愿。高考生、中考生和他们的老师父母在直播间许下心愿，众多心愿中，还有来自奥运双冠王叶诗文的祝福。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在“流星雨下”许愿直播间，作为清华学姐的叶诗文祝愿考生“旗开得胜、心想事成”。贴心的她还特意录制了视频，祝福考生都能“发挥出最佳水平，让青春不留遗憾”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/157643715562c3b9eb655e51.14006003.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>叶诗文在快手许愿直播间祝福考生</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到了考后志愿填报阶段，快手官方还推出了高校百科查询工具，并联动了数百位教育大咖、泛知识垂类达人分享志愿填报经验。用户只需在搜索栏输入“高考”，就能进入平台专门设置的高考界面，这里有“大学排行”、“查大学”、“查专业”、“志愿填报”四个入口，考生和家长可以快速找到参考信息。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/100841793062c3b9ec3ce551.23510438.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>在快手搜索“高考”，可以找到实用工具、入驻平台的高校官方账号、热门视频和直播等等</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>借助平台上的服务和工具，考生可以很快根据分数找到自己的填报方向。</span><span><strong>当然，志愿填报是一个理性和感性结合的过程，人生不是一场只看数据的规划游戏，产业也是动态发展的，因此，来自高校、专业人士、名师、局内人的解读和回应意义重大。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>高考结束后，快手开启了“志愿大咖说”、“学长学姐逛校园”、“高校招生云宣讲”等系列直播，中国教育在线总编辑陈志文、中国高考规划行业联盟会长赵京、高考曲老师等50多位专业人士现身直播间。直播涉及教育专家、高校学长学姐、高校招生办等不同角度，考生家长在快手上可以获得更内部的视角，校准他们的判断。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>高考是重要的人生转折点，经此一役，考生将开始探索他们各自的人生道路。如何理解人生和高考的关系，前辈的故事是很好的参照。在快手上，除了上述提及的教育界专家，杨天真、朱丹、沈奕斐等名人大咖也现身直播间，和考生分享各自的高考故事，身体力行后的人生感悟。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以看到，用户在快手上不仅是消费高考相关的内容，还获得了有用、好用的服务。</span><span><strong>今时不同往日，和传统的搜索引擎，或是某个垂类工具相比，快手在高考季提供的价值是全周期、立体式的，既提供了信息价值，也提供了服务价值和情绪价值。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据统计，快手前三个最受关注的高考话题分别是“这届高考生能有多自信”、“高考是另一种形式的成人礼”、“2022高考实用指南”。平台发起的28场志愿指导直播吸引了2003万人观看，10场“未来一小时”吸引了4050人观看。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/29203651962c3b9ec676514.08225583.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上述内容涉及多种视角，兼备趣味性和鼓励、启发意义，承载好内容的快手是今年高考季的新风景，平台对于高考热点的挖掘，不仅为考生和家长塑造了新的高考回忆，也让快手获取了可观的暑期流量。而再往深处看，这一系列的变化也将影响快手本身。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213179696861420d317ab820.36069667.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>从信息到服务，</span></strong></span><span><strong><span>再到更多的价值创造</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近些年，高考季基本是搜索公司的核心战场，高考周期对应着考生和家长的搜索高峰，各家搜索平台会不遗余力投入线上广告、应用商城等位置推广。但今年有了变数，行业有了新的入局者，玩法随之生变。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>伴随着通信技术的发展，用户已经完成从纯文字到图文，再到短视频、直播载体的迁徙。作为当下最受重视的内容载体，短视频和直播在大众获取信息方面扮演着越来越重要的角色。</span><span><strong>这种重要性不仅体现为直观和新奇，更体现为差异化的价值。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以直播为例，估分、志愿填报、高校宣讲，这些工作都讲究双向沟通和传递，除了官方文件，考生和家长还有很多具体的问题想问，比如“这个专业走学术路线前景如何”、“往届毕业生的就业单位都有哪些”，或者是“北方人过去气候能习惯吗”、“夏天会不会虫子多”这种生活向的问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/141597978462c3b9ec935246.12588260.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>类似的细节问题未必都能搜到答案。相关的信息“藏在”局内人的大脑里，而直播是获取具体信息最低成本的方式，这种实时性是传统内容载体所不具备的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，在很多志愿指导直播的评论区，都能看到一类人群的身影——考生家长。在许多人眼里，考生家长更像后勤人员，默默付出，“隐身”于孩子背后。但这些家长也有具体的烦恼和忧虑，遇到困难时，她们也希望得到鼓励，也需要更多专业指导，这样才能在孩子的高考路上“多出一份力”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这部分最关注高考内容的群体，快手提供了一个彼此守望，获得启发的窗口。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在快手，考生家长们分享彼此的经历，缓解彼此的焦虑和孤单。快手上的内容跨越地域，身在县城的家长也能得到一线城市老师的指导，了解更多高校和专业的情况。即使不懂考试内容，根据专业人士的分享，家长们也能获得经受验证的高考经验，为孩子备考“加一把火”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/84303037262c3b9eccf9e84.77815171.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>在快手，考生家长分享经历，相互鼓励</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也让我们开始思考一个问题，相当一段时间里，短视频和直播被归类于内容消费领域，快手也被视为内容平台。外界关注新颖的内容，产业关注平台的广告变现率。那么，作为当下最受关注的互联网业态之一，短视频和直播能否有更多正外部性，能否对具体个体的发展产生助益？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>答案是肯定的。本次快手的高考季运营已经说明了这一点：</span><span><strong>平台不仅能有趣，更能有用。</strong></span><span>无论是主动搜索还是智能推荐，用户在快手上都能获得想要的信息和服务，快手解决的不仅是内容消费需求，更能解决和切身利益相关的问题。通过信息流+搜索双引擎，快手已经成为新的信息与服务入口。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果将时间再拉长会发现，平台的“有用”属性不只是在高考季体现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据《2022快手泛知识内容生态报告》，2021年，快手泛知识内容播放量同比增幅达58.11%，泛知识直播的开播场次达3300万+。在快手上，用户可以学到实际的种植、装修、烹饪技能，也可以就具体的婚姻、劳务问题获得法律知识。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137910091862c3b9ed27d5c7.30091843.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>民间达人多，是平台上各种实用内容的源头活水。而对于需要系统化输出的专业领域，快手官方也在积极拓展。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月底，快手和清华大学慕课平台学堂联合推出《新知如师说》活动，活动打造5门精品线上公开课，覆盖心理学、计算机科学、经济学、医学健康、建筑学等学科领域。</span><span><strong>实用内容的差异化供给和平台运营下的头部内容供给相辅相成，共同构成了快手可观的泛知识生态。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频直观、直播实时，新的内容载体进一步降低了知识的生产和传播成本，也实现了传统载体不具备的功能。推荐+搜索的组合，则让知识能更精准、更有效率地触达关心它的用户。今年快手在高考季的投入证明了，“有用”的内容还有广阔的渗透空间，能够受益于此的绝不只是考生家长。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产业变迁是个大话题，对于具体个体来说，考生是“变迁”的直接受益者。完全可以想象到这样一个场景：十年后，已经参加工作的同学们再次坐到一起，分享这些年的各自职业发展和生活。此时，大家必然会聊起一个问题，“你当年为什么会选那个专业？”</span><span></span></p>
