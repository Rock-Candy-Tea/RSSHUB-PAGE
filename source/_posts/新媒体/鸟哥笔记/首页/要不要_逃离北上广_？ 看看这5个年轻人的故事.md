<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个人独处的时间越多，</span><span>思维也就越活跃，但思维活跃不是一件好事，它意味着“焦虑”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在2022这一年，我们闲了下来，也有时间去思考一些工作之外，有关选择和努力的问题。</span><span>根据某招聘网站的对职场人士的调查数据来看，这一年，大家想的最多是：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>应不应该回家，找一份“铁饭碗”的工作？</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>理由无非是：</span><span><span>解决不了户口问题；工作不稳定；买不起房子；回家照顾父母... ...</span><span> </span></span><span>大城市，具有小城镇没有的基础设施和机会，这点毋庸置疑。</span><span>但是小镇，也不意味只有“铁饭碗”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>只有多听听不同的声音，你才能找到最正确的答案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 1 、</span></strong></span><strong><span>“当你做好了赚到钱了</span></strong><strong><span>那些声音就没有了”</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>胡春青是一位在读博士，2018年，他回到家乡做起了戏服，现在和妻子两人开了公司做起来原创汉服，他说只要你想要的戏服我们基本都有，没有的我们就做出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年，他的公司年销售额已经突破了1000万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/112959005862e1f060154bf1.42768055.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回家创业的原因很偶然，2014年，胡青春在回家度假走亲访友的时候，得知老家大力发展农村电商，很多人通过电子商务发了家致了富，当时心里就有了创立淘宝店的想法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>返回学校，胡青春也注册了一家淘宝店卖戏服，他运气比较好，注册淘宝店当天晚上就接到一个订单，这让他对开店铺产生了更大的兴趣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来他说：“真正让我决定返回家乡创业的是因为接到一个澳门的500套学士服定做的定制订单。萌生了干淘宝的念头。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是因为没有自己的加工厂，500套服装的订单，拖延了交付的期限，没挣到多少钱。</span><span>吃一堑长一智，胡春青和妻子决定成立自己的公司，当老板。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/76345870262e1f06077ab46.60000320.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span></span><span>图片来</span><span>源</span><span>：</span><span>一人谈视频</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当初回家时最大的阻碍，不是要建工厂，也不是原创技术，而是家里人的不支持和邻居里的各种流言蜚语，他常听见的话就是：</span><span>“我们没上过学也一样做的很好。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>言外之意就是这多年学白上了，这话也有传到他自己耳朵里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/169570033362e1f060d72e35.64668119.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span></span><span>图片来</span><span>源</span><span>：</span><span>一人谈视频</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人甚至在父亲面前说，</span><span>你儿子读那么多年书就做这个，不觉得的丢人吗？</span><span>父亲被人议论，回家和他吵架</span><span>“辛辛苦苦培养你，不是为了让你做村民都会做的事”，</span><span>让他别干了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为负面的声音太多，生活也受到影响，他们干脆从村子里搬了出去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来因为戏服销量不好，胡青春开始想办法扩大市场受众，他灵光一现，突然觉得，把戏服改做成汉服会比较好，这些年喜欢汉服的人也越来越多。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而做汉服要有自己的原创，但这不是问题，妻子学过美术，有设计功底在</span><span>就基本全权负责了设计。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，他和妻子二人就从设计，生产，销售全方位做起，建立公司，做闭环创业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他这次，因为转型做原创汉服带领村民走出疫情带来的危机。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后村民的观念也随之发生了转变，各种不被看好的声音也没有了，</span><span>他们开始</span><span>希望在外打工的大学生能够回乡工作，也认为有“学问的人卖货也卖的好。</span><span>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/114025890662e1f061507a71.58898293.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/89158207662e1f061779f41.01998973.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/135444838762e1f061ace1b2.11738330.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span></span><span>图片来</span><span>源</span><span>：</span><span>一人谈视频</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在湖北，也有一位放弃大城市高薪回到老家的年轻人，</span><span>好奇五先生。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他是一个短视频博主，他的内容以研究消灭蟑螂的1000种办法为主题，并把研究过程拍摄剪辑，放到短视频平台，慢慢获得了较高的关注，刚开始他的家人也很不理解。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他希望大家通过他的视频得到猎奇和新鲜感之外，还能得到一些知识和启发，为此他研究了大量专业的昆虫资料，做了</span><span>一两个月，就有了爆款内容。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/30125615662e1f062176997.17666288.png" referrerpolicy="no-referrer"><span></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span></span><span>图片来</span><span>源</span><span>：</span><span>一人谈视频</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在老一辈人眼里，没有什么比安稳更重要，面子比里子重要。所以小镇常常被打上“熟人社会，人情关系”的标签，整个镇子的人好像活在监控之下，没有任何新鲜事。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这些强关系，影响着身处其中者的决定，也几乎影响了他们的一生。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>家人的意见和决绝的态度，成为大城市返乡创业路上最大的阻碍，他们确实是为你好，但是生活能不能过好，是自己的事情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span> 2 、</span></strong></span><strong><span>只有不怕出错的人</span></strong><strong><span>才能找到更多可能</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实年轻人回家找出路的例子并不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>福建女生叶婧在2015年辞去了大企业的工作，回到了家乡东山岛，她陪着奶奶走过了人生最后的一段时光。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来，因为怀念奶奶做的鱼丸味道，她开始寻找岛上的老人还原古法的手工鱼丸。最后不仅做出了手工鱼丸，还带动了岛上三十余位留守奶奶就业！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15835953662e1f062ba0df7.22903699.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span></span><span></span><span>图片来</span><span>源</span><span>：</span><span>一人谈视频</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在四川泸州农村，有个网红名叫“川香秋月”。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>川香秋月，多年前，她是深圳富士康的流水线女工，每月领着一两千块钱的工资，有一次过中秋节的时候，手里只剩下两块钱。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些年，她和丈夫一起回到家乡创业，三次创业失败，因为发现了短视频风口，开始拍摄家乡美景。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/97257227762e1f063393c62.25067343.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/168127299462e1f0637af0e7.96658207.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span></span><span>图片来</span><span>源</span><span>：</span><span>一人谈视频</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>家人，邻居偶尔都会出境，不知不觉间积累700万粉丝，每月能卖100多万单土特产。</span><span>靠着创业，日子越过越好，不仅买了房，还成为</span><span>了同辈人中的佼佼者。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>表面上看，他们很幸运，借助了短视频，借助了淘宝，但是同样幸运的人，失败者占大多数，只有经历了一次次的失败，志向不改的人，才能坚持下来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在日本就流行着一种聘用管理者的标准：</span></p>
<section>
<section>“我们招聘管理者，不敢用那些没有出过错的人。只有不害怕试错的人才有更多可能性，出过错的人，对目标也会更清晰。而什么都不愿意去做，不敢做的人当然不会出错。”</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>无论在什么环境下，你都要不失对美好生活的向往，愿意一次次尝试，就不怕</span><span>一而再，再而三的</span><span>失败。</span></strong><span></span><span></span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图片来源：一人谈视频</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>而现在的创业成本也没有以前那么高，</span><span>他们可以</span><span>就地取材，</span><span>或灵感突现，或深耕某一领域</span><span>，</span><span>利用互联网的传播</span><span>短则</span><span>1-2个</span><span>月，长则</span><span>1</span><span>-</span><span>2年。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但如果</span><span>你</span><span>不想出任何差错，还</span><span>是选择</span><span>“铁饭碗</span><span>”比较适合</span><span>。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 3 、</span></strong></span><strong><span>“得到了铁饭碗，</span></strong><strong><span>一点也不开心”</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>杜天明，某国企普通员工，33岁，</span><span>在外创业多年，最终在30岁的时候，进入国企，所以他觉得自己有资格回答这个问题，给还在迷茫的人一些启发。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>杜天明说：</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大学毕业时，我没有考公务员，也没有考国企，而是选择自主创业，我创业的过程中就像打游戏过关斩将一样，你每次打赢一个boss，就能及时反馈给你快感和荣辱，你很享受这个过程，但不幸的是，你的号被盗了，辛劳</span><span>一场</span><span>给他人做了嫁衣。</span><span>你对此行为深恶痛绝，但也只能继续练号，升级，打怪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/90785850462e1f064227a88.16716737.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>人的耐心是有限的，感兴趣的事情也经不住再三的失败。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>几次创业失败后，我开始怀疑自己根本不是创业的料，整个人失去了目标和斗志，但是生活要继续，我选择了进国企。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>初进国企的时候，朝九晚五，每个月领着固定的工资，这时候发现这样的日子也还不错，不用担心工作范围之外的事情，焦虑减轻了，压力也小了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每个月领完工资，还能和朋友，家人外出消费一把，当地有哪家新开的餐厅，商场，我都摸的门清。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是一直沉浸在这样的生活中，人就会废掉，没有多少存款意味着抵抗不了突如其来的风险，因为父亲重病住院，几乎花干了积蓄，孩子又要上学，报补习班，国企不到5位数的工资，根本指望不上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来一位曾经的合伙人打来电话，聊起近况，他又创业了，虽然一路磕磕绊绊，现在公司也初具规模，去年开始盈利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/168070705262e1f064475831.71153127.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我突然有一丝后悔没有坚持下来，当然，在国企也有努力奋进的人在，但是我就是那个不求上进的人，现在依然是一个普通小职员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>业余时间找的副业，也没有坚持做下去，<strong>在一个安逸的环境里，人真的很难自己监督自己把什么不必要的事情做起来。</strong></span><span>现在</span><span>创业也基本上没有什么机会了，于是就又做了一个副业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比于现在的铁饭碗，我更喜欢创业的兴奋和快感。感觉那个时候自己还是一个鲜活的人，现在只不过得过且过混日子而已。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那些想要找个安稳的工作然后利用业余时间做副业的人，要考虑清楚，你是否真的能做到自己想象的那样自律？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而那些想要混日子的人我希望你能安下心来，好好混日子，不要像我一样得陇望蜀，什么也做不好。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 4 、</span></strong></span><strong><span>做选择之前想清楚</span></strong><strong><span>你将会永远失去什么？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再回到我们开头谈到的话题：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如果能够重新选择，你会不会选择回到家乡找一个“铁饭碗”？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般人看来，远方的生活越发艰难，而家乡除了平淡就是琐碎，我们只能从两个不怎么样的选项中，必须挑出一个好的出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但你不必着急做选择，先列一个清单出来，看看当你选择这一个放弃另一个时，你会失去什么？你会得到什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后问问自己，<strong>你是否真的做好准备，永远失去那些你现在放弃的东西？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/31532907462e1f0646e4cd3.91463453.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为只有当一个人真正要失去某样东西的时候，才会意识到自己有多么需要它。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>记得我当初，来北京的时候，并没有犹豫；当逃离北上被炒的火热的时候，也没有动摇，现在北京外来者持续减少，我也没有离开的打算，因为我知道我的目标只有在北京才能更好的实现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你的目标需要回乡，才能完成也不必纠结。</span><strong><span>如果这两个选择都不够好，</span><span>就</span><span>多给自己一些选择。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“逃离北上广”也不意味着一定回到小镇，二三线城市的机会也在增多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是我希望你不要在最后关头做选择，被迫做选择时，我们往往会选择那个看起来比较舒适，也比较容易的选项。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但很多时候，那条难走的路才能走的更远。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论怎么选择，我们的目的都是一样的，找到那个最好的自己。</span></p>
