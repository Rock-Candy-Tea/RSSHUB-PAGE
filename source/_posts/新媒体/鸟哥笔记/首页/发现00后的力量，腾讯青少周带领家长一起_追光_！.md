<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">前<a href="https://www.niaogebiji.com/tag/%E6%B5%81%E9%87%8F%E6%97%B6%E4%BB%A3" target="_blank">流量时代</a>，流量池”泛滥“让小品牌有了销量爆棚的底气，展开了大小品牌的同台竞争。后流量时代，品牌们为了突出重围不断寻找机会放大声音，从而扩大影响、分割流量，一度进入白热化阶段。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从流量见顶到应对流量碎片化，早已是移动互联网的共有焦虑。<strong>狂人认为当下品牌们应该改观唯流量论，将注意力从流量转变到消费者身上，发现消费者身上的社交化、圈层化、碎片化特点，从而获得属于品牌自身的私域流量池。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如<a href="https://www.niaogebiji.com/tag/%E8%85%BE%E8%AE%AF" target="_blank">腾讯</a>青少周展开的一场以科学与艺术为主体的公益活动，通过聚焦青少年，成功打造亲子社交话题，让自身破圈科学、艺术等话题，收获了上亿话题阅读量，并获得了大众的一致认可。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>追星文化的兴起在后流量时代进入了巅峰，作为互联网原住民的青少年，他们更容易也更喜欢参与到追星活动中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯深信”青少年就吃这一套“，尝试以“追星”玩法结合“网络热更”、“知名IP”锁定青少年的注意力，更是以科学家、艺术家作为青少年的榜样，赢得了家长们的积极响应。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span> 1. 建立榜样力量，引导青少年“追星”新方向 </span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>打造高立意，让品牌营销止于套路。</span></strong><span>迎合青少年关注的追星话题，腾讯推出短片，让一切开始于一句“一起追光”的号召。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过青少年的视角陈述“追星”新态度，灯光闪烁的演唱会场景和孩子眼里的光，引导着观众一步步深入了解，想要知道这群孩子追的“光”到底是什么。随着钢琴、星空、画笔等元素的出现，我们看到了“科学探索奖”陆朝阳、中国科学院院士王志珍、“天问一号”火星探测任务首席科学家潘永信等七位耀眼的“明星”，以及他们背后一次又一次的科学上的重大突破，将这份国家荣耀激发观众的情感共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯以顶尖科学家和科普工作者为榜样的方向向青少年们做积极正向的引导，拔高了活动的立意，也呼应了对“追星”的全新定义。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span> 2. 打造亲子科学盛会，科学家面对面交流沟通 </span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>想要与青少年开展沟通，就要抓住更具有势能的家长人群，得到他们的认可。</span></strong><span>其中</span><span>关键在于营销中通过兼顾青少年和家长的内容，抓住亲子互动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/197471362562b8e78635d5b9.79864673.jpeg" title="发现00后的力量，腾讯青少周带领家长一起“追光”！" alt="发现00后的力量，腾讯青少周带领家长一起“追光”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/112268073262b8e786a047d8.16979912.png" title="发现00后的力量，腾讯青少周带领家长一起“追光”！" alt="发现00后的力量，腾讯青少周带领家长一起“追光”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯青少周打造的主要活动之一便是在清华大学举办的腾讯青少年科学小会，汇聚中国科学院院士王志珍、参与新冠疫苗研究的科学家张林琦、天问一号首席科学家潘永信等科学家们进行面对面交流，真材实料的对谈和采访吸引家长们火速关注和参与。一场亲子科学省会，促进了亲与子之间的互相影响，也带动了社交话题和传播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span> 3. 催动音乐艺术渲染力，点燃大众参与热情 </span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在信息爆炸的时代，人们的生活在不断加快节奏，但音乐会却是让人忍不住放慢节奏的地方。</span></strong><span>腾讯在中山公园音乐堂举办荷风艺术行动音乐会，以乡村孩子的汇报演出，汇聚歌手周深、国乐大师方锦龙、中国爱乐乐团和一众一线艺术家，为大众带来了一场艺术盛宴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/161779846662b8e7875bff73.64292368.png" title="发现00后的力量，腾讯青少周带领家长一起“追光”！" alt="发现00后的力量，腾讯青少周带领家长一起“追光”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>借助偶像明星将观众视野带给鲜少被关注的乡村少年，用一场精彩的演出带动青少年的艺术气息，点燃观众的热情，更用背后的公益艺术行为再一次拔高了整场腾讯青少周的立意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一场顺利的传播从链路上看应是交相辉映的，腾讯从没有考虑“为数据而传播”，而是以打造一场影响青少年价值观的活动为主旨，配合各类互动玩法，响应社交传播话题实现了声量的最大化。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span> 1. 发挥自身社交平台优势，矩阵化传播爆发声量 </span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为头部社交平台，腾讯汇聚了大量用户。利用自身的社交平台优势，腾讯联合国际顶尖科学期刊 《Science》发布“青少年科学看点榜单”，邀请158万青少年参与投票，选出这一年最受青少年关注的科学突破。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1623061262b8e787cadc42.64469523.png" title="发现00后的力量，腾讯青少周带领家长一起“追光”！" alt="发现00后的力量，腾讯青少周带领家长一起“追光”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/178280236962b8e788a804f4.38296773.jpeg" title="发现00后的力量，腾讯青少周带领家长一起“追光”！" alt="发现00后的力量，腾讯青少周带领家长一起“追光”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过投票互动以及寄语的方式，青少年不仅仅是个看客，更会亲身参与和深入了解品牌想要传递的价值。配合微博、视频号等全网媒体平台矩阵化的话题传播，家长和青少年们在讨论声中带动进一步推动“一起追光”的话题热度，自发成为传播者。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span> 2. 线上线下接替互动造热，实现传播效果最大化 </span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>通过线上线下互动的衔接，品牌的活动热度如同海浪一波又一波的翻涌，最终形成巨浪。</span></strong><span>腾讯青少周的活动链路中，品牌不断观察着用户的动向，如腾讯荷风艺术行动音乐会结束后，发现大众对乡村孩子们背后的故事产生的浓厚的兴趣，于是迅速展开孩子们演出故事的解读，满足大众的好奇心，助力传播更大化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/204439948762b8e7894f08b7.63870344.png" title="发现00后的力量，腾讯青少周带领家长一起“追光”！" alt="发现00后的力量，腾讯青少周带领家长一起“追光”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过社交平台最大化的放大声量，配合年轻化的互动方式以及品牌随机应变的物料内容，腾讯青少周最大程度的覆盖目前人群，实现了最大效果的影响力</span>。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>围绕青少年打造的营销常常走入局限性，最后变成了品牌需要的营销，而不是青少年需要的营销，腾讯青少周把持着对家庭与社会的高度关注，真正从文化层面去探索解决方案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/73845656662b8e78b0894d9.18381485.png" title="发现00后的力量，腾讯青少周带领家长一起“追光”！" alt="发现00后的力量，腾讯青少周带领家长一起“追光”！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>面对家庭与社会难题，品牌需要真正深入探索和解决用户诉求。</span></strong><span>2019年腾讯荷风艺术行动音乐会邀请郎朗、许巍等艺术家与乡村少年同台，帮助乡村少年艺术教育走入大众视野。首次腾讯青少年科学小会，腾讯更是邀请物理学家斯蒂芬·霍金的女儿、科普作家露西霍金、《西部世界》的科学顾问大卫·依格曼等六位科学明星一同探索科学的奥秘，激发青少年的科学兴趣。<strong>腾讯打造正面影响，利用长期的营销方式展现品牌价值价值。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
