<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/60258594862e318944857b8.50788517.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两天前，我在B站首页上刷到一条关于“精神内耗”的视频，看完为UP主投了一个币。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过几个小时后，这11分钟的视频就刷遍了朋友圈，登上B站全站排名榜第一名。UP主“衣戈猜想”的二舅火了，至28日早上十点左右，视频播放量已经达到2899.2万，拥有18.8万条弹幕。后续衣戈猜想接受媒体采访，并发布了新的一条视频，表示二舅火了“是因为二舅活得好，不是因为我写得好”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>UP主“衣戈猜想”显然非常懂得二舅在火了之后有可能面临的一些“过度关注”，所以早早做好了防范措施，也呼吁大家不论出于怎样的目的，都不要去打扰二舅的生活。二舅是自己的英雄、是村里的“歪子”，但没必要成为又一个“草根神话”。</span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些年，在互联网造神之路上不乏普通人的身影，以往有许多知名企业家、国家运动员、知名演员等从草根逆袭而来，靠的是拼搏和实力。但随着媒介环境发展，新时代的网络红人出现，当年“犀利哥”红遍网络，12年过去了，有谁还知道他如今过得如何？网络红人的成名原因有很多，但普通人成名后的结果不尽相同，那就是经过几轮消费后，渐渐淡出大众的视野，最终回归到平凡的生活里......</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>这些年的草根神话，有些成就了自我，有些成就了他人，之外则更多地成就了平台。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，喜欢用树枝和塑料布等材料制作成服饰，在楼道和田野等空旷之地走秀并拍成视频，上传到网络上走红的陆仙人，最终实现T台梦；中国内地美食短视频创作者李子柒，与杭州微念的爱恨情仇；papi酱、丁真、李雪琴……</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着快直播与短视频的兴起，有些普通人有意或无意间受到网络世界的追捧，成为“网络红人”，他们因为自身的某种特质在网络作用下被放大，与网民的审美、审丑、娱乐、刺激、臆想、品味以及看客等心理相契合，积累了大量的人气与粉丝，同时也为平台创造了巨大的流量。而<strong>网红经济以平台作为内容输出场景，为后续的国内MCN变现打下了良好的基础。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最初，平台和机构对“批量”制造网红趋之若鹜，想要迅速成名赚钱的人也越来越多，“剑走偏锋”博取眼球成了普遍的手段。<strong>网红经济随之发展，让网红产业链渐渐形成规模，也暴露出了隐含的负面影响与矛盾所在。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>现在，快手的Slogan是“拥抱每一种生活”，抖音的Slogan是“记录美好生活”。</span></strong><span>在注意力经济下，名利双收的人多，唯利是图的人也多，但在聚光灯背后，这样的平台上更多的是“隐入尘烟”的普通人，他们可能踩不到风口，不了解机制，也驾驭不住互联网的各种新鲜事物。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5098565562e318947c4c60.36215873.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图为网红陆仙人</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像衣戈猜想在采访中所说，如果有人要来给二舅拍纪录片，那浩浩荡荡的专业团队，必然会暴露二舅的住所。二舅也不是个喜欢侃侃而谈的人，在镜头面前，他会感到不自在。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网络上掀起了一阵有关二舅的热烈讨论，其中有种观点格外鲜明，那就是我们的感动不来源于美化后的苦难，也没办法根治自身在这种苦难对比下显得微不足道的精神内耗，但是我们由衷地敬佩世界上每一个坚韧乐观的普通人，不论手上的牌是好是坏，都能认真打好人生这场牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><img src="https://ngbjimg.xy599.com/19513051262e31894b58321.10491393.png" referrerpolicy="no-referrer"></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>“别人的痛苦才是艺术的源泉。”来自王小波杂文集《沉默大多数》中的一句话</strong></span><span>，他认为，苦难是一种财富，人可以受苦受难，却不能陷入痛苦，你的痛苦只会成为别人艺术的源泉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也正是因为切身理解了这一点，可能才会有许多人拒绝把二舅的故事定义为一碗鸡汤，而是想到了身边还有多少像二舅一样的人。有网友说：“一个时代总要有人为错误买单，只有自己活得满足，才是生活最好的回报。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回到“精神内耗”一词，显然大家的定义各有不同，而当代人有更多的精神需求是毋庸置疑的。从心理学的角度来说，这事实上与我们的身心发展不是必然冲突的，</span><span><strong>或许我们可以将其视作今天我们“要一起买的单”，以重新诠释我们的生活和要迎接的未来</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>敬二舅，也敬每个人自己的明天。</span></p>
