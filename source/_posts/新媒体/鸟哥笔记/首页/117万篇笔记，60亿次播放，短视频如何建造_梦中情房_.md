<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这届《脱口秀大会》上，豆豆吐槽“Loft”一词太具迷惑性，引起不少网友的共鸣。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Loft作为一种房型，在他看来明明是“南北不通透，睡觉在楼上，洗澡在楼下，周围都是办公室，产权只有40年的商住两用房”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可女朋友听了，联想到的却是“小资”“可以拍一天的楼梯”和“值得一住的梦中情房”……</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/158718103632ff3128c4ad1.11595134.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>网友们看完惊呼，“这不就是我本人嘛”“被内涵到了，我对Loft确实有执念”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1525742192632ff312e18519.08748851.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：小红书、微博</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大众对于Loft“梦中情房”的想象，恐怕有很大一部分都来自于<a href="https://www.niaogebiji.com/tag/%E7%9F%AD%E8%A7%86%E9%A2%91" target="_blank">短视频</a>：画面中色彩明艳、功能齐全、小而美的生活空间，看上去温馨宜人，视频一般还会加上#一镜到底##沉浸式回家#等话题标签。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/848371387632ff31354fd60.00343129.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频博主们还会分析Loft的其他优势，例如售价优惠、采光通透、有小资情调，在一线城市不需要社保也不占用购房资格等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在小红书，与“Loft”相关的笔记超过117万篇，博主“Skye在家”一条“一个人住35平Loft有多爽”的沉浸式回家视频，就获得了12万点赞、4万收藏；在抖音，与Loft相关话题播放次数超60亿，<span>其中</span>话题#Loft复式#就有43.8亿次播放，且不乏点赞百万的Loft参观视频。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些在社交媒体上爆红的内容，一定程度上给“Loft”加上了一层厚厚的滤镜。</span><strong><span>而与此同时，也有大量“Loft劝退视频”活跃其中，试图给“沉迷滤镜”的网友当头一棒。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公众号“X博士”就曾发文开炮“Loft的营销套路”，分享中介利用Loft样板房视频钓鱼，出售远没有图片中那么好的房屋，获10w+阅读，上千点赞。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1140563862632ff3143b3ee9.29696970.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这些博主大V们的内容是如何对“Loft”产生影响的？用户究竟爱看什么样的Loft内容？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期，我们采访了全职博主“胡大瑞Rae”、从宠物赛道转入家居赛道的博主“王梓哈哈”、在Loft住了三年的格一，聊了聊互联网上那些与Loft有关的内容。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从创作者的角度看，“Loft”不仅是热门段子中的“梦中情房”，还是一门赚足了流量的内容生意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>“Loft是我的流量密码”</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Loft指小户型的商住公寓，面积在30-50平米左右，层高在3.6-5.2米区间。它是一种生活处所，也是一种“内容引流技巧”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据观察，网红博主用Loft引流的方式主要有两种：观点类，分享自己loft租房或住房体验；展示类，分享Loft中的空间和家具，一般会打上#roomtour#的标签。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>博主胡大瑞是观点类的博主，平常主要分享家居方面的内容。</span><span>今年6月，她在B站发布视频《纯纯大无语，loft这次是真的跌了》，分享了Loft这种房型吃香的原因、存在的缺点、不适合买的人群，以及如果想买的话，怎么样买能好一些，获得89.4万播放和1.4万点赞。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/286829294632ff314a93743.08526245.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>胡大瑞说，自己当时选择发这个Loft内容，“主要是因为自己住在Loft里，又感觉比较有争议，就误打误撞地做了”。<strong><span>在视频中她也调侃地表示</span><span>，“有Loft这个词的（内容），各大平台都会爆，这是我的流量密码。”</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>从这一期内容的B站数据来看，Loft也确实证明了自己的流量能力，观看人数</span><span>远超胡大瑞平时几千至几万播放量的作品。</span></span></p>
<p><img src="https://ngbjimg.xy599.com/1675430962632ff31502d5b2.99561018.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“王梓哈哈”是展示类的账号，<span>博主在字节跳动做程序员，2021年开始在小红书运营宠物号“狗子芝士”，涨粉一直很慢，截至今天共651个粉丝</span></span><span><span>。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p><img src="https://ngbjimg.xy599.com/1904244729632ff31554aa42.59706904.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>2022年疫情居家期间，王梓偶然想到了分享自己装修Loft、搭建小家的内容，就在小红书和抖音新建了Loft垂类账号“王梓哈哈”。</span></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年6月她发布的一条“39平方Loft，一镜到底25岁女生的家”视频，给大家展示治愈风的厨房、卧室、楼梯等，在抖音获得61.2万点赞，在小红书获得1.9万点赞。</span><span></span></p>
<p><img src="https://ngbjimg.xy599.com/823799513632ff31594f5c8.10985109.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王梓表示，Loft内容都是手机拍摄的，唯一的费用是买了两个三脚架。胡大瑞也认为，拍摄Loft对新手很友好，内容竞争没有美妆、剧情等赛道激烈。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“小美好的生活”和“王一工”是近两年开始发布作品的新手博主，“小美好的生活”分享自己在苏州38平的Loft，效果最好的视频在抖音有150.3万点赞，12.2万评论。“王一工”制作了47集自己住在Loft空间里的短视频，这个名为“在北京14.2平米的家”的视频合集在抖音获得4.3亿播放。</span><span></span></p>
<p><img src="https://ngbjimg.xy599.com/820655103632ff315ecfb55.71522781.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>不过，Loft也不是随便拍都能火的。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>胡大瑞认为，观点类的Loft博主一定要说不一样的，比如“Loft不通风根本不是问题，有哪些方法可以解决”这类角度。展示类的内容，博主应该有一定的个人风格，并且能迎合网友偏好的精装修和暖色调审美。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1575359974632ff3163c2676.34451205.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>同样是展示类的Loft内容，有个人风格的笔记点赞更高</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>制作Loft内容的成本问题倒不必多虑，并非自己花上百万元买房才能“引流”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>B站上，“果哥看房”以中介身份，向网友介绍33.6平带阳台的Loft，2分钟的视频获得33.6万的播放；</span><span>“程序员乐子”<span>发布了</span>“别骗年轻人买Loft，他们输不起”的视频，是自己看房一段时间后的思考，获得214.1万播放。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/581238572632ff317b078f0.87254474.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>吐槽loft存在的税费问题</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“拥有一套复式公寓”不是分享“Loft内容”的必要成本，观众只是喜欢刷“Loft”内容而已。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这里就不得不谈到一个问题，为什么“Loft”在互联网上始终有流量？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先是“小众”。设计了3位数住宅的室内设计师JeRRy表示，Loft这种房型比较有限，加上自己住过的，他一共就设计过6套。这也就意味着，大部分受众对Loft还处在好奇、想了解的阶段。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次是“有话题”。大部分Loft博主都提过自己被网友骂。豆瓣博主格一3年前分享了自己软装Loft的经历，没有接任何广告，但至今都还会收到评价她“骗人买Loft”的留言。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过不管观众是喜欢还是讨厌，争议本身也给“Loft”带来了更多的流量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>“Loft也可以是财富密码”</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>采访Loft相关博主时，他们大多表示自己很幸运：除了这类内容粉丝转化率高、容易出爆款之外，更为重要的是“Loft内容”让他们机缘巧合进入了家居赛道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王梓账号6月份内容爆了以后，每天能在小红书和抖音收到3-4封商务留言。</span><span>来询问的品牌，包括沙发、窗帘等家居品，也有扫地机器人、空气炸锅这些小家电。她意识到，Loft内容让她的账号进入了家居赛道，这里“好像比其他赛道更容易变现”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>胡大瑞介绍，行业内广告费一般是按照博主粉丝数10%</span><span>定的，当然还要综合其他方面的数据，例如点赞收藏比等等进行浮动。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“家居类的产品，因为商品单价相对较高，所以宣传经费也会宽裕一些，不会给博主压价”<span>，</span><span>胡大瑞补充道</span>。之前有一个沙发品牌主动联系到她，商品售价1万左右，品牌方愿意上门免费安装并赠送，但她的Loft实在是放不下了，对方沟通了几次后才放弃。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而小型的家电及家居用品，例如床上四件套、地毯等，品牌预算不高，可能会压价，但对于博主来说能省去一部分装修费用。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>博主愿意接这类广告，家居品牌也愿意投这些发布Loft内容的中小博主。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大瑞表示，自己接触的家居品牌在投放广告时，相比报价10万一条的大博主，确实更希望找到10个报价1万的博主。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而很多发布过Loft内容的博主，在粉丝量和内容制作上都很合适。有Loft作为背景，博主可以发布“安装视频”和“科普视频”两种类型的广告。前者展示甲方商品的家装过程，后者教粉丝怎么挑家居，从皮质、结构各方面呈现，顺带加入甲方品牌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这也在某种意义上解答了“为什么到处都是夸Loft的内容”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比“吐槽Loft”只能做一期内容、表达一下情绪，“支持Loft”的内容能让中小博主受到家具品牌的长期青睐，获得还不错的经济效益，拍出一些好看好玩的内容。“自来水”固然美好，“恰饭”却能实际地帮助博主坚持下去。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，还有一些房产公司在互联网上发布Loft内容，模版统一、话术雷同，不过从实际的留言状况和互动情况来看，被引流去看房的也不在少数。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/259012184632ff3181159b7.60595793.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span><strong><span>网红滤镜以外，Loft是什么？</span></strong></span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前来看，Loft更像是一个“网红概念<span>”</span>。它承载着部分网友对“有一个属于自己的小家”的幻想，部分博主迎合了这类需求，获得流量密码，顺利转型进入家居赛道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而抛开网红滤镜，Loft是什么？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>格一描述了一件自己居住在Loft里的小事，3年前，她把一盆没长开的、规规矩矩的绿萝放进了Loft里。后来，它沿着二楼楼梯栏杆向上缠绕生长，直到栏杆尽头又重新折返。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“看到这盆绿萝，我觉得很安定，这是我和房子慢慢长在一起的过程。如果再上升一点，可能是一个人对自己生活的掌控感。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1465000957632ff319585079.94234565.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>左2019年，右2022年</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>胡大瑞说，她以前经常搬家，买Loft所提供的安定感，可以完全抵消她所面对的隔音问题、通风问题。不过如果有更多钱的话，她还是会考虑搬到其他地方的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王梓满意自己用心装的房子，但工作忙碌，每天11点休息，回家睡个觉，早上睁眼就要走了。她并没有太多时间，去享受这个被网友迷恋的“梦中情屋”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对他们来说，Loft不是三五十篇笔记可以概括的，而是</span><span>指向了具体的“家”，至于实际是美好还是糟糕，还要取决于生活在其中的人如何看待。</span></p>
