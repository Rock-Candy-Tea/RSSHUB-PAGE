<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2010年的一个晚上，腾讯广州研发部的张小龙给马化腾写了一封邮件，判断在移动互联网时代，腾讯应该做IM类应用。当晚马化腾就同意了，并将其命名为“<a href="https://www.niaogebiji.com/tag/%E5%BE%AE%E4%BF%A1" target="_blank">微信</a>”。于是，语音聊天、附近的人、朋友圈、微信支付、跳一跳...... 微信一骑绝尘，从最初的即时聊天软件成为国民级应用，成为工作、生活中“基础设施”般的存在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信的崛起拨动着无数对手们焦虑的神经。在这一路上，小米的米聊、网易的易信、阿里的来往都想与之一战。到了2018年，还有王欣的马桶MT、字节的多闪、罗永浩的子弹短信同天发布，来势汹汹，摆出“围剿”微信的架势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>结果是，成批的“微信挑战者”们一拥而上，然后，偃旗息鼓。中国互联网行业的社交大战早已尘埃落定，激战不再有。新风口潮起潮落，社交已经不再是个大家挤破头的“明星赛道”。</span></strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/108269817262c7ae5699a5d1.41789377.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>罗永浩发布聊天宝（原名子弹短信）</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图片来源于网络</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“只要终端设备不换，取代微信地位几乎是不可能的。”微信创始团队成员、公众号“行者慎思”的作者陆树燊在与「深响」的交流中说道。“但是能不能不盯着「社交」咬出一块空间？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>行业变了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>大厂们做社交的思路从“挑战微信”转变为务实求真，把社交作为增加主业务黏性的工具，淘友圈、京友圈、饭小圈、拼小圈们接连出现在大App中，希望社交能带来用户更高频的打开、更长的停留、更多的活跃度，以及一个数据金矿。</span></strong></span><span>“我们团队的OKR简单说就是对DAU的影响要达到百分之多少。”前字节跳动社交产品经理山姆告诉「深响」 。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>而小厂和独立开发者们依旧层出不穷地尝试新玩法，Soul、即刻、啫喱等等应用出现，只是没有人再奢求全量，或是昙花一现、或是偏安一隅。 </span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年了，互联网的社交梦正脱虚向实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81142426261420d300e45a3.29058312.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>利用社交</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>社交、电商、内容三端融合的趋势越来越鲜明。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年，“万能的淘宝”改了自己的签名，开屏广告语从“随时随地， 想淘就淘”改为“太好逛了吧！”</span><span><strong>淘宝显然不满足于做一个用户“上来就买、买完就走”的网购工具，而是希望借助内容和社交网络让人们更长时间地留在平台上，更多地完成交易。 </strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>围绕这一目的，近几年，淘宝接连推出带有社交属性的功能，和好友“购物车分享”、“淘友圈”分享订单，“摇一摇面对面分享”，边语音边同步商品的“一起逛”，以及双十一时的互动小游戏“喵糖总动员”组队抢优惠等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比淘宝，通过微信导流、社交裂变完成原始用户快速积累的拼多多似乎离社交更近。</span><span><strong>拼多多通过“砍一刀”、“助力提现”等方式<span><strong>完成了</strong></span>熟人关系链的部分迁移。但是随着用户基数扩大，拼多多想要进一步在产品内部沉淀私域流量池。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“拼小圈”在2020年上线，占据了非常强势的首页入口。功能上与淘友圈大同小异，用户可以分享购物记录，也能看到好友公开的拼团信息及点评内容。产品逻辑是，将种草+拔草环节更紧密地结合在平台上，并且通过熟人社交关系让商品评价体系有天然信任度加成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>X小圈功能已经成为电商标配。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>京东也有与之类似的“京友圈”功能，可以在个人中心开启，并导入通讯录好友，或是扫码加好友。做外卖的美团也来凑热闹，延续王兴的社交梦，去年上线了“饭小圈”，功能是用户可以分享外卖订单，其他好友可以进行评论、点赞，还有一键跟买的“跟着吃”选项。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>从平台角度看，在传统电商进入瓶颈期、用户增长速度几乎触顶的环境下，他们都在试图借助社交来构建、巩固自己的流量池。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/198125200762c7ae56e0c794.09763853.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>拼小圈、京友圈、淘友圈、饭小圈</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显而易见，社交网络的魅力是巨大的：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，熟人的消息、动态会增加用户打开App的频次。电商平台也为此打造了各种场景，比如分享打开后双方都可领取优惠券，用户会被好友“召回”平台。更多可消费可玩的内容让用户“逛起来”，粘性和时长都会增加，随之增加是做出更多消费决策的可能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，通过社交关系来理解用户、划分需求，也是一种推荐逻辑。黄峥称其为“朋友式的AI算法”，更贴合人性化的需求，通过拼团了解人、通过人推荐物，而与之相对的“上帝式的AI算法”，则是基于海量用户的数据标签划分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>所有社交功能的目的，归根结底，还是与主业协同，促进交易。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“在2012、2013年时候，行业每天都在喊「微信拿到了第一张移动互联网的船票」，老板们有巨大的焦虑感，淘宝、支付宝等App疯狂加入各种社交功能，这更多是战略层面的考量。”陆树燊说。“过去几年，进入了第二阶段，大家试图做私域，做二次导流，是从商业化、 产品设计本身上面做改变，这是一个更务实的逻辑。”他说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但不得不说的是，电商与社交之间总是有些相悖之处。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>移动互联网时代，App将用户心智做了切割，淘宝是购物的、美团是点外卖的、打开微信是发消息的，而现在，大平台希望用户在不同平台上做相似的事情，发消息、分享、评论。</span><span><strong>用户的时间精力都是有限的，无处不在的社交互动让人精疲力尽。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一个问题是，这些功能是从用户角度出发的吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小圈类功能由于隐私问题遭人诟病，它们的设置入口通常都藏得极深，用户协议和导入数据的告知也并不简明突出。导致的后果是，一些用户在不知情的情况下就“被分享了”购物记录。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“社交软件和功能极其重要的一点是不能伤害用户，大家常常只看到社交的好，却没看到风险。”</span></strong></span><span>陆树燊对此评论道。2016年，支付宝曾上线“圈子”功能，结果却因为擦边球内容引起争议，之后停运。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213179696861420d317ab820.36069667.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>社交是结果</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>相比电商平台，内容平台的社交属性天然更强。社交关系可以降低对内容的接受门槛，内容反过来又可以促进关系的建立。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音、快手上，除了基本的私信功能，视频聊天、同城、一起看、直播、聊天室等功能和玩法都已完善，覆盖了熟人和陌生人社交。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/114641738762c7ae57396fa6.66176448.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>抖音抖一抖功能 </span><span>可以通过手势发现附近的人</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如字节跳动前社交产品经理山姆告诉「深响」，抖音社交团队的OKR与对DAU的贡献率直接挂钩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>字节内部将社交划分为两部分：关系链建立和社交互动。</span></strong></span><span>前者主要是增加互关、建立和拓展熟人在抖音上的关系链，后者主要是关系建立后点赞、评论、访问主页等行为。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关系链建立最快的方法就是从第三方导入，比如通讯录，然后用算法进行二度/多度人脉好友推荐。“相比TikTok，抖音的难点是国内通讯录的质量差很多，大家用微信，不再更新通讯录，微信也不会开放第三方API接口。”山姆说，团队会观察到中年人开始在抖音上与老同学进行社交的现象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了评估功能的有效性，团队会开长期反转实验，功能模块的迭代对一部分用户不生效，然后从活跃度、留存率等维度的数据上进行对比。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了产品功能，抖音也在运营端发力，做校园活动、广告电影植入，打造“大家都在抖音上社交”的概念。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/131246762262c7ae57caa946.61593902.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>抖音上的私信、抖一抖、一起看视频等社交功能</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手和抖音很像，但做社交的底层逻辑有所不同。</span><span><strong>快手本身的社交属性更强，比如在内容推荐上，抖音更从用户兴趣出发，而非好友关系和地理位置，快手则不同。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“在一些地区，快手可以当朋友圈用。”资深产品经理判官在与「深响」的交流中提到自己的观察。这是用户密度大和短视频/图片交流门槛比文字更低带来的结果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“社交不是目的，但是手段、也是结果。” 判官说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>对于内容平台来说，社交是增加打开率、日活、用户粘性、促进UGC生产率的工具，是平台获取数据的手段，也是用户密度大到一定程度产生的必然结果。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而短视频平台的整个商业模式都是建立在用户停留时长上的，只要用户打开App，就会刷到广告、卖货的视频。对于知乎、小红书等更小的内容社区而言，做社交也是基于类似的逻辑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20585534276144238ed0dc54.24202450.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>不再挑战微信</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大App做社交，是把社交作为“工具人”：满足用户基本的通讯需求；增加打开频次、活跃度、提升停留时长、为主业服务；带来更多数据做协同推荐等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这样的目标下，社交功能本身并不用承担变现的任务。但是对于小厂和社交App而言，试错成本高，发展与生存问题摆在眼前。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>依然有许多App在跃跃欲试，找新鲜的玩法、从更小众的群体切入、为满足更专业化的目的......若按玩法来分，有语音、随机视频换脸、元宇宙概念、手机桌面小组件等等；若按关系链区分，有陌生人交友、 熟人；按受众群体分，有瞄准年轻人、下沉市场用户的；按目的分，有职场、约会、婚恋相亲等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由此也可以看见，社交是多元的、发散的、非标的，因为社交网络的中心始终是难以量化的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>但是，多数新独立社交App命运可以用三个词语概括：籍籍无名、昙花一现、偏安一隅。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>造成这种结果的原因有二：逼仄的创新空间，社交产品的商业化难题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互联网浪潮中，App创业的黄金时代已经过去，ToC产品的故事越来越难讲，我们的手机里也再不装下新东西了，尤其是社交。线上熟人<a href="https://www.niaogebiji.com/tag/%E7%A4%BE%E4%BA%A4%E9%9C%80%E6%B1%82" target="_blank">社交需求</a>被微信这样的超级App</span><span>（Super App）</span><span>占据，陌生人</span><span>（Friend finder）</span><span>社交领域已有陌陌、探探、Soul、Blued等，还有数不清的约会和相亲软件。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“现在看来，所谓的「社交赛道」或许根本就不存在，很多需求是真实的，也能被拆解成许多子场景，但是在大型平台上，其实也都能被满足。”判官说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新鲜App出炉不少，但看实际产品，多是换汤不换药。去年以来，随着元宇宙概念风潮，不少打着“元宇宙”旗号的社交产品亮相，比如短暂爆火过的啫喱。这款“捏脸3D形象+密友社交”的App在今年2月曾把微信挤下过App Store中国区免费榜榜首，但是没过几天，就因为人偶穿模、频繁闪退、服装侵权、用户隐私等问题遭到种种质疑，后来便主动下架了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19572845162c7ae581113e6.03856005.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“元宇宙社交App”啫喱</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：网络</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此类产品固然有新意，但急于求成，“一招鲜”短暂爆红之后，又没有能力及时解决用户反馈的问题，也不再有后招。它们的目标群体往往是年轻人，他们追求新鲜感，但换句话说，也是没有忠诚度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「深响」在与几位有社交产品研发设计的产品经理的交流中，得到的一个共识是，</span><span><strong>只要技术不换代，终端设备不换</strong></span><span>（也就是我们的智能手机）</span><span><strong>，颠覆式的社交产品不可能出现。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而商业化是另一个横亘在它们面前的难题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拿微信举个例子，它在商业化上的表现被称作“克制”，但根本上说，是因为难度大。“它其实是作为整个Super App的逻辑去赚钱的，通讯本身没有赚钱，朋友圈广告也是很小一部分，其更大的价值在于对集团整体业务的支持。”陆树燊说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比大厂孵化、试水的App，独立App需要养活自己，加上投资人的钱也不再涌入这个方向了，它们的变现压力陡增。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>社交产品的变现模式无非是卖广告和卖会员等增值服务。前者做流量生意， 社交流量与工具流量没有本质区别，后者收“过路费”，通过“阻断”平台上的社交关系收费，或是通过直播等方式创造营收。</span></strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说起来简单，但事实是，行业内的大公司，都过不上好日子。陌陌和探探母公司挚文集团Q1财报显示，其营收和付费用户两项指标都在持续下滑，两个主要营收来源，直播和增值服务，一个下滑，一个增长疲软。2022年Q1营收同比降低9.3%至31亿元，利润从去年同期4.6亿降至2.9亿，付费用户从去年的1260万降至1100万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>诚然，大公司的衰弱或许是给小厂和独立开发者们的一个机会和信号——用户需要新的东西了。</span><span><strong>他们多数只能走一条“小而美”的路，然后在产品调性、扩大用户基数、变现中小心翼翼地找平衡。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人的地方就有社交，这是永恒的需求。用户也处于某种矛盾之中，2022年了，我们一边喊着被社交产品“绑架”，一边还是期待，会有新的产品出现，无论以何种形式，给我们创造一个良好的交友环境，进行真诚有趣的交流。</span></p>
