<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有些联名，看双方名字就令人好奇！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>HR和漫咖啡，贵妇级美妆与青春派咖啡，能有什么共同语言？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有些联名，看其中一个就能猜到谜底！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E7%8E%8B%E8%80%81%E5%90%89" target="_blank">王老吉</a>和科颜氏，可能会跟去火有点关系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有些联名，一眼望去就期待它的创意！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如<a href="https://www.niaogebiji.com/tag/%E5%96%9C%E8%8C%B6" target="_blank">喜茶</a>和《梦华录》，一周爆卖140万杯，期待创意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>联名诚可贵，好评价更高，是爆款最好！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看看这三位的联名，谁撞到了心巴上？</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">1</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>HR&MANNER，</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>年轻人“一夜青回”的秘密</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>六月底，赫莲娜HR与MANNER联名，TVC《青春有问题》脑洞大开，鬼马精灵，惊喜开场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>连环提问，句句打在心巴上。</span><span></span></p>
<blockquote>
<p>工作的本质是什么</p>
<p>摸鱼是工作的一部分吗？</p>
<p>打工人的运动只有搬砖吗？</p>
<p>吵架吵到一半发现别人是对的怎么办？</p>
<p>……</p>
</blockquote>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这哪里是品牌在提问，分明是年轻人自己问自己。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所谓共鸣，就是你是我的最佳嘴替！</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>片尾点出：有问题，才是青春。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而赫莲娜和MANNER的联动，可以解决你的问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>MANNER推出联名款限定新品“一杯青回”，</span><strong><span>清新的抹茶冰激凌拿铁</span></strong><span>，一杯唤醒脑袋；赫莲娜HR的主角则是它的经典产品绿宝瓶，一夜修复受损的肌肤；二者联动，颜值与头脑都得到了修复，合上了主题「一夜青回」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1326433516306c06fad7207.00379515.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>绿色是这次联动的视觉主基调，绿意萌动着青春的鬼马和生机。除此之外还有玄机，时机的机。当广告推出的时候上海结束了居家隔离，他们希望错过春天的消费者，能够感受一次扑面而来的满眼的绿色——春天的绿，赫莲娜的绿，也是青春的绿，又是瞄准心巴的精准一击。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/19969048506306c06ff338c7.84822040.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>线下150+家MANNER门店一夜变绿，从某种意义上来说，每一家门店都变成了一家赫莲娜的品牌体验店，影响力全城覆盖。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>活动期间穿绿色衣服进店，还可以享受联名饮品免单的福利，不过因为随绿免单的活动在17：00之前会结束，那时很多白领没下班，受到了很多网友的吐槽。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">2</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>王老吉&科颜氏，</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>联接点在“超能力”</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>王老吉和科颜氏金盏花水跨界联动，在线下开了个“金盏花凉茶铺”快闪店。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一张海报让人秒GET二者的联接点，怕长痘用金盏花水，怕上火喝王老吉。这对联名CP还有自己的名字，叫金日大吉，朗朗上口，过目不忘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/14312173406306c0704845a8.95020147.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二者联名初看虽“远“，细品确有其道理，他们都是清凉系的产品诉求，内在的功能上都令人联想到战痘能力，有着重叠的目标客户群体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>「有些联名，二者表面相关性越远，越有爆发力。」</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这家快闪店采用了国潮复古风的设计，力求沉浸式体验“当年夏天的清凉”。门口就有一个屋顶高的水龙头艺术装置，哗哗的流淌着一柱金盏花瀑布，非常抢眼。旁书“肌肤清火凉茶”几个大字，昭示着这个快闪店的双重内涵。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/3281457446306c070910aa1.68996438.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/4786381886306c0712648b3.29831199.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>科颜氏的金盏花水随处可见，王老吉的红罐比比皆是，它们被精心摆放成一道风景，让来的人都有拍照发圈，去社交网络炫一炫的动力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/21266667336306c071f36e99.35473601.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/6436384646306c072483e35.23199813.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/14395911876306c0728e4311.90729404.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>金盏花水里流出真正的金盏花，产品内涵的生动展示。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/17161534436306c07303f1d5.24601668.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>店内限定清火特调，让人亲自体验金盏花和王老吉的双重魅力。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/19872484786306c073568298.64668144.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王老吉和科颜氏金盏花水这把联名玩出了另一种境界：暗示和类比。借对方的“东风”强化各自的产品功能诉求，尤其是不能对消费者过度强调的那部分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“怕上火喝王老吉”已经是一个IP，科颜氏金盏花水类比王老吉，让消费者产生一种用科颜氏金盏花水可以给皮肤“降火”的清凉联想。王老吉好像也沾了科颜氏金盏花水祛痘的光，让消费者感受到“怕长痘也可以喝王老吉”的潜台词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这两位的联名，给双方都带来了正向加持。</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">3</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>喜茶，</span></strong></span><strong><span>联名界屡试不爽的爆款制造机</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说一句喜茶是联名界的顶流，应该没有人反对吧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喜茶的联名是有几分“爆款”基因在身上的。一方面谁火跟谁联名，另一方面谁跟喜茶联名谁火，它本身就是被业界密切关注的“网红”。但是它依然需要新鲜的流量带火新品，不断刺激线下消费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>今夏，喜茶联动《梦华录》推出了</span></strong><span>「紫苏·粉桃饮」，尽管发布时《梦华录》已经接近大结局，但「紫苏·粉桃饮」依然火爆，某家店七天就炫掉了5154杯，一周销量突破140万杯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/6887910456306c073a8aad7.76796001.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>七夕期间，喜茶又联动「只此青绿」，推出了新品只喜青芋，用中式美学来烘托七夕的浪漫氛围，翩跹的舞姿，仙气飘飘。还打造了主题门店，推出了一系列唯美周边。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/12166416326306c0740c4f42.13549097.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/10463032536306c074798f77.90116772.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《只此青绿》这个IP曾在今年春节期间火遍全网，知名度高，有自己的忠实粉丝群，本身自带流量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不得不承认，喝奶茶，颜值就是正义，只此青绿舞蹈诗剧和《千里江山图》极富文化底蕴的承托，让这场联名的出品特别有气质。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/16425177716306c074da28f6.06584116.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同于其它联名更侧重品牌宣传，喜茶的联名纯纯给自己花式“贴金”，通过撬动流量密码，带动销量，成就爆款。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">4</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>茶饮为啥爱联名？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>发现没，联名特别爱茶饮，茶饮也特别爱联名，分析了下有这几个原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、限制少，容易实现</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有个朋友曾给某平台提议和某牛联合出个LOGO雪糕，但是一想到还要开模，还要搞定食品生产的一系列问题，协调生产线，还要上规模，一系列操作下来时机过去了，联名产品还出不来。相比较而言，茶饮现制现售，只需要备足原材料即可，联名款生产起来方便快捷。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>茶饮在概念上的创意空间广阔，通过概念创意总能跟联动的另一方搭上关系，联动品牌的适配度也比较宽泛。有的茶饮已经把自己做成联名界的IP了，比如MANNER。茶饮没有季节限制，夏有清凉冬有暖，还有秋天的第一杯奶茶，总之只要它想，一年四季都能找到联名的由头</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2，线下有规模效应</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>饮品店高客流，饮品店店铺本身也是一个体验、互动的点位。动辙开店过百家的规模，相当于给品牌提供了过百个广告位，一杯杯的饮品就是行走的广告牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3，线上有流量</span></strong><span></span></p>
