<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">最近，一个乌龟壳火上了抖音热搜。</span></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音数据显示，截至7月26日，“全抖音都被安利的乌龟壳”话题视频已有7068万次播放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/170782980462dfa19650d650.05141977.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>龟壳的走红源自几天前赵露思的一条视频：她穿着一只约一米长的龟壳玩偶，开着扁脸特效，略带笨拙和搞笑地展示着身上这个新家伙，最后安心地往地上一躺，成了一只“躺平的乌龟”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其后，收到安利的粉丝纷纷冲进生产</span><span></span><span>商家</span><span><strong>“优趣优品”</strong></span><span>的直播间，在评论区刷着“这是赵露思同款吗”？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/65539550562dfa19687d414.58664395.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为此，公司股东之一刘丰铭还特地发了一条赵露思同款视频，表示“嗯说得对，就是同款躺平神器”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/72748035262dfa196bcab95.10204643.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在赵露思之后，又有多名抖音达人被这个充满巧思的“奇葩”玩具吸引，分享了穿着龟壳的视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/24871644162dfa196f08847.77723877.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘丰铭提到，在这波热潮之前，每天能卖出700个左右的龟壳，近日受惠于话题热度，龟壳的订单量一下子涨到了一天2000单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，近几日直播间最高实时在线人数提升到了6000人以上，账号涨粉超30万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，在赵露思发布视频之前，优趣优品就已经有着不小的热度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你是毛绒玩具的常客，一定对这些“奇葩”的谐音梗玩偶不陌生：蛇和恐龙的结合体“蛇恐”（社恐），鲨鱼和香蕉的结合体“鲨蕉”（撒娇），还有西瓜冬瓜哈密瓜和鸭子的结合体“西嘎冬嘎哈密嘎”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11289737462dfa1972c95e2.39221232.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些都是优趣优品的产品，而设计的灵感来源几乎都是直播间或短视频评论区网友的留言。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“粉丝要什么我就做什么，他们只要要，我就能做。”刘丰铭说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/62207002862dfa19754d164.29391966.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月25日视频的一条高赞评论说想要挂件版的牛芒玩偶，</span><span>7月14日他就把产品做了出来</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘丰铭的逻辑是评论呼声高，那就证明产品有市场，做出来就是会有人买单。作为刚刚入驻<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%94%B5%E5%95%86" target="_blank">抖音电商</a>一年的品牌，优趣优品的订单量已经能够稳定在每日一万单左右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音电商，像刘丰铭一样借助粉丝的实时反馈生产产品的还有</span><span><strong>“刘家人竹编（有竹编猫窝）”</strong></span><span>（下称“刘家人竹编”），账号主理人刘霞冰回忆，经网友点醒，原本在广西一个小镇制作水果筐等竹编产品的他们开始研发各类宠物用品，结果产品上架后大受欢迎，今年竹编产品总体的产量相比去年提升了2-3倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些品牌商家是怎样在抖音电商做内容的？如何打开生产思路，发掘产品创意？我们对话了两家品牌的主理人，试图一探究竟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>一条视频20万赞，</strong></span><span><strong>借力“谐音梗”频出爆款的优趣优品</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一开始，优趣优品只是做些普通可爱的玩偶，比如大鹅玩具、花朵抱枕、面包坐垫等， 尤其是大鹅，畅销到爆单，一天能卖出5000只。刘丰铭说：“我过年期间没放假，大年初一都在（直）播”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/200130324762dfa19781f044.79086071.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但好景不长，像大鹅这样的产品，复刻实在是太简单了，优趣优品做，同行也开始做，还跟他们打起了价格战，以更简单的做工、更低廉的价格吸引消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>优趣优品苦于这一瓶颈无法突破，直到有一天，刘丰铭看到一名粉丝的ID叫“鲨骑马”，突然之间灵光一现：鲨骑马，沙琪玛，不如做个鲨骑马玩偶，也许粉丝会喜欢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘丰铭的想法是对的，在这个往常视频只有几千点赞量的账号中，因为“鲨骑马”，出现了为数不多的超20万赞的视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/110226354162dfa197b2a665.56254253.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“鲨骑马”，小名“鲨马特”</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在看到鲨骑马大受欢迎后，优趣优品又乘胜追击，相继推出了“鲨瓜”（鲨鱼+西瓜）、“菜狗”（白菜+狗）和“辣鸡”（辣椒+鸡），几乎每一条视频都是新的谐音梗玩具，推陈出新到粉丝们都害怕。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11336850462dfa197d99a07.03925632.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过后来，粉丝们就渐渐从逗乐打趣变成了“点单式评论”，“老板我要这个”“老板什么时候出那个”“万人血书求XXX”......刘丰铭也习惯了这种粉丝提思路，品牌配合做设计的过程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从某种程度上而言，这是品牌内容创作的良性循环——商家充满创意的优质内容激发了网友的创作欲和互动欲，从而不断地为账号带来流量，而这样的热度又反哺了商家，让商家“脑洞大开”，促进了产品的升级和销售。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/173564782662dfa198055463.82558402.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“来抖音电商之前我们没有接触过（和粉丝互动制作产品）的这种方式，体验过之后就感觉产品很畅销，现在基本上没有滞销的产品。”刘丰铭说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传统玩具生产模式可能产生滞销产品，是因为厂家没有办法与顾客产生实时且零距离的互动，用户反馈周期长，容易造成资源浪费，这一点恰恰被抖音电商的互动机制解决了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘丰铭回忆，刚入局抖音电商的时候，他不懂怎么做视频，也不懂直播，花了几万块钱上培训课学习，却连提词器是什么都不太了解，直播时的话术都要一字一句地背。“上学都没那么认真过。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那时候，直播间都只有几个人，为了拉到新客，刘丰铭头发也愁掉了不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这个过程中，刘丰铭渐渐意识到，只要产品足够好，直播间不必玩套路、憋单，甚至开播前也不需要做太多额外准备，只需要在直播时直观地展示玩偶尺寸，真实地讲解产品，再挂上链接，同样能吸引一批忠实观众。在他们的摸索中，直播间的场观一天天地涨起来，在线人数也稳定在了2000人左右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘丰铭说：“我跟粉丝讲，套路我可以学，但这没意思。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们做的就是尽可能吸取网友意见，持续在短视频里输出创意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>优质的内容带来了更多资源。彼时，抖音电商小二主动联系上优趣优品，提出可以提供一些明星及达人带货资源，交个朋友直播间（下称“交个朋友”）就是这么和优趣优品联系上的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在货品上，交个朋友别出心裁地选择了一款香蕉和鸭的结合体玩偶——“香蕉个朋友鸭”，将谐音梗玩出了“梦幻联动”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/21058083862dfa1982ac451.41275806.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>交个朋友看中了优趣优品“香蕉个朋友鸭”（想交个朋友呀）毛绒玩具</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音电商独有的创作和互动方式让优趣优品获得了前所未有的增长，刘丰铭干脆停掉了其他所有平台的业务，专注做抖音电商。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他说：“花同样的精力，在这里做的效果最好。所以我还蛮感谢抖音电商，没有抖音电商的话应该做不到现在这种程度。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>做出了粉丝在直播间刷的猫窝，</strong></span><span><strong>店铺总产量翻了两三倍</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抱有这样想法的人不止刘丰铭一个。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互动性强、优质内容有土壤，同样是刘霞冰选择主做抖音电商的原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘家人竹编开设抖音账号已有约三年半的时间，拥有粉丝6.8万。虽然粉丝量不多，但偶尔也能创作出点赞量几十万的视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“其实我们其他平台也有账号，发的也是这些内容，但没什么人关注。”刘霞冰说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘家人竹编最受欢迎的产品是一款竹编猫窝，产量占上百种竹编产品的30%。这款猫窝的创意灵感缘何而来？账号简介中的一句话是：一名有幸被网友们调侃，成功将猫窝狗窝等宠物产品上线的农村竹篾匠。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/21767557762dfa198612da3.67636984.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>购买网友对猫窝的评价</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一条点赞量近20万的视频中，刘家人竹编表示一开始只是做一些簸箕、水果篮、馒头筐之类的手工艺品，但在某一天直播时，情况发生了变化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘霞冰说：“我记得很清楚，那一天是2021年3月19号晚上，突然间（直播间）就有很多人进来，我也不知道怎么回事，大家一直在刷猫窝。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在网友的启发下，刘霞冰上架了竹编猫窝，几天内就卖出了200余单。这对小本经营的刘家人来说已是很大的买卖，他们不得不发动了全村的篾匠才赶工完成了订单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/87675146462dfa198a6fa87.42323266.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其后，猫窝异常畅销，于是她干脆把账号名称也改了，为原先的“刘家人竹编”加上了五个大字：有竹编猫窝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做了猫窝还不够，为了满足日益增长的养宠大军的需求，他们还陆续做了猫包（用于携带猫咪外出）、猫隧道（猫咪玩具）、鸽子窝、仓鼠窝等等宠物用品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/86728848162dfa198dd1662.63418412.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>竹编是中国传统非遗技艺，浸淫在竹编文化中长大的刘霞冰对此有很深的感情，她时常能看到粉丝的留言，“说产品非常好，竹编文化也是我国很优秀的传统手艺，都很鼓励我们一定要坚持。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘家人竹编的商品以家庭作坊的形式进行生产，她曾在一条视频中提到，她的太爷爷、爷爷、父亲都是七八岁时就做了篾匠，七大姨、八大姑、伯父叔叔舅舅等都是拥有几十年竹编经验的篾匠。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但由于生产耗时费力，竹篾匠的月收入不过两三千元，家人都不希望刘霞冰成为下一代竹篾匠。高中辍学后，刘霞冰前往广东打工，并一步步干到了服装店店长，且很快就有希望升职为区域管理。但在升职前夕，她毅然辞职，回到了家乡广西钦州灵山县的一个小镇上继承家业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她说：“当时觉得如果不回来做的话，可能就没有机会了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘霞冰回到家乡的时间是2016年，2016年之前，其家人主做的竹编产品出口代加工生意似乎已经一眼望到了尽头，没有什么订单，更没有什么发展前景。2016年-2020年，刘霞冰接手生意之后当地的竹编产量有明显的提升，但是销售渠道还是有一定的局限。为了招揽更多客源，她来到了抖音电商。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年，上千个认真介绍产品的视频总算到了厚积薄发的时候。“猫窝”带来的这场转型，看似是偶然，其实也是必然。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/190802818262dfa1991069e8.85348545.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>刘家人竹编的第一条视频（左）及优质爆款产品带来的效应（右）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时，这波“出圈”的内容还火到了亲朋好友那里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘霞冰说：“他们会把（刷到的内容）发给我爸爸，发给我，然后就有很多朋友过来说‘你们出名了’。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但她并没有沉迷于这短暂的热度，相反，她在不断地根据粉丝意见迭代产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品毛刺多？专门安排阿姨手工拔刺；想要双层猫窝？爸爸亲自上阵画图纸。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过往的他们对产品的想象局限于酒店用品、厨房用具、家居用品和茶道用具等，受粉丝启发，他们将来还计划做一个宠物产品系列品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>抖音电商成功关键：</strong></span><span><strong>善于创作优质直播及视频内容</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回顾优趣优品和刘家人竹编的账号运营模式，二者在本质上是相似的：他们的直播间没有套路和术语，也没有复杂繁乱的商品组合，有的则是朴实的产品介绍、亮眼的产品创意和善于倾听粉丝建议的内容运营者，这是他们成功的关键。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据了解，抖音电商在今年3月推出了“春雨计划”，宣布将充分倾斜资源，提供更多的流量激励、权益保障和业务助力，鼓励商家和达人创作优质电商内容、提升长期经营效率，并承诺提供200亿流量用于创作激励。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/141796493762dfa19956c6d6.99749047.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音电商学习中心在“春雨计划”公告中列出的优质内容创作者享受的权益</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不难看出，抖音电商鼓励商家及达人的内容创作，更支持优质内容的发展。优趣优品和刘家人竹编走红的原因之一，就是在视频及直播间展现的独特内容。新榜编辑部经访谈及调研，总结了促成他们形成优质内容生产模式的三方面要素：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1. 善于利用来之不易的爆款</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>优趣优品的爆款是龟壳玩偶，刘家人竹编的爆款是竹编猫窝。打造单品爆款是成功的第一步，而抓住爆点，在产品研发、视频创作上持续下功夫，则是成功的第二步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘丰铭透露，为了写出更好的视频脚本，他们面试过数个编导候选人，但都没有找到合适的，“后来我就只能自己想，脚本太难了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/101061937762dfa199816860.88060383.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>刘丰铭（左）亲自坐镇直播间带货，希望吸纳更多流量</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2. 站在粉丝的角度创作内容。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘丰铭曾经提到：“做抖音电商只要有利他思维，站在粉丝的角度考虑问题，你就能成功。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>唯有做到“利他”，才能真正地做到“利我”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>3. 根据粉丝反馈快速更新产品。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>优趣优品通常会在两周内研发出评论区中那些呼声高的产品，可以短时间内让此前围观的粉丝追到后续，从而产生与品牌的亲近感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了支持更多像优趣优品、刘家人竹编这样的内容电商创作者，今年1月，抖音电商推出“寻找同行者”计划——这是抖音电商的创作者成长大本营，致力于挖掘优质达人和商家，通过“山货上头条”“看见手艺计划”“全民好书计划”等专项扶持计划，助力他们获得更多关注和机会，在平台实现更长远的发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在优趣优品的爆款龟壳视频下，网友还在忙着给刘丰铭出主意：</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个龟壳载着抖音电商优质内容的热度走来，也将载着人们对更多优质内容的盼望走下去。</span></p>
