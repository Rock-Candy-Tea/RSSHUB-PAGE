<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>号称服务过5000万宝妈用户的<a href="https://www.niaogebiji.com/tag/%E8%9C%9C%E8%8A%BDAPP" target="_blank">蜜芽APP</a>，宣布将于9月20日正式关停。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这家从淘宝网起家、历经8年的<a href="https://www.niaogebiji.com/tag/%E6%AF%8D%E5%A9%B4" target="_blank">母婴</a>垂直电商平台，在过去曾创下许多骄人的成绩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如来自真格、红衫、百度等资本数十亿的投资，然后从B2C，升级到S2B2C的社交电商模式，但最终还是关了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>蜜芽在杭州的分公司有两层楼，和村长的公司紧挨着，平常还能遇到他们老板刘楠、以及他们在做直播的活动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那问题来了，作为曾经母婴垂直电商app的领头羊，为什么做不下去了呢？</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>01、</span></strong></span><span>产品的竞争优势不大</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于母婴、服饰这些垂直类目，不仅产品多，而且消费也很大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以不仅是垂直电商平台看重的市场，更是所有综合类app关注的赛道，比如淘宝、拼多多、京东。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前垂直赛道平台干了一件事情，吸引了很多宝妈用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那就是做了产品归类、内容整理，相当于帮助宝妈节省了找东西的时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>十年前那时候，垂直电商才刚刚兴起，综合类电商平台竞争很激烈，给了垂直电商发展的机会，比如蘑菇街、贝贝网。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是各位现在再来看看各大平台商品的供给，不仅是渠道多了，最重要的产品的差异化太少了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前不仅平台兴起、各大母婴产品也兴起，所以有些平台可以拿到一些独家的产品或者因为垂直，产品更丰富。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而现在，垂直电商平台的产品还不如综合类的平台产品丰富。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>02、</span></strong></span><span>产品没有价格优势</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前除了平台独家产品或者平台做补贴产品外，一般比价肯定上拼多多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前村长做社交电商平台的时候，我们在上架产品的时候，都要去对标几个平台的价格，确保有足够竞争优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这带来的问题就是，平台的毛利极低，尤其是还要分销，平台能留下5个点就很不错了。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是在目前来看，这些垂直的母婴平台都在想办法搞自己的品牌产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为通品真的没有优势，各大平台商家都是一样的，而且大平台还要许多资金在补贴，垂直平台玩不起啊。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以不仅是蜜芽app面临这样的难题，像之前年入百亿的贝贝网app也销声匿迹了。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>03、</span></strong></span><span>代理宝妈玩不动了</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这段时间，不仅是垂直类的平台很难玩，像社交淘客、团购分销都很难玩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面是平台产品本身的优势不大了，佣金变少了、奖励也变少了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面是宝妈没激情了，许多宝妈赚不到钱，卖的产品拼多多都能找到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以现在反而是许多非知名品牌的货，以及一些微商自己品牌的产品还能卖卖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其实在疫情之下，许多活动做不了，许多商家倒闭，用户消费欲降低，许多宝妈想卖也卖不动了。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>04、</span></strong></span><span>抖音等平台的冲击</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前一方面因为好产品不足导致用户就是冲着产品本身去消费的，另一方面当时许多产品喷发，用户不知道怎么选，所以垂直类的平台做了一道筛选。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是现在用户从单纯的产品消费转移到了内容消费上了，用户知道有很多产品，也知道很多渠道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以什么样的产品能打动用户，靠的是人和内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而像蜜芽这类平台还是货架式电商为主，内容能力又很弱，所以用户逛的兴致就大大降低了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是像抖音，个人IP博主以真实的体验、推荐、特殊的价格优势，吸引了很多用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是平台超级大博主和明星主播的号召力，远远超过了普通的电商平台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些垂直平台无论是从产品、内容、KOL等方面，和抖音这些相比差距太大了。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>05、</span></strong></span><span>其他电商渠道冲击</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>纯垂直类型的电商app真的是很古老的产品了，我们可以梳理下，当下的消费渠道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>综合类电商app比如淘宝、京东、拼多多，垂直类电商app比如蜜芽，社交电商类app如之前的云集、贝店，社交淘客类app好省、粉象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此外，还有生活团购饿了么、美团，社区团购平台如嗨团，接龙团购平台群接龙以及个人KOL做的小程序、视频号、抖音、快手等电商平台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如年糕妈妈一年通过公众号等平台，既能接广告、又有自己的内容矩阵、还有自己的产品线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家可以去想想，是不是几乎有流量的平台，最后都会选择电商卖货？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所以当用户面临这么多渠道选择的时候，用户的流量、消费都是被分割的，那么许多垂直平台的必要性就大大减弱了，尤其是把侧重点放在产品上的时候，就更没有优势可言了。</span></strong><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>06、</span></strong></span><span>短期的出路在哪里？</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为村长一直在电商行业，所以蜜芽的许多阶段村长都了解，包括在2019年前后做社交电商平台，搞各种带来制度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>差不多在2020年底的时候，蜜芽app整体的完善、升级就很少了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>包括其创始人也把更多的经历用在头条、抖音等渠道做短视频、搞IP了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这条转型的路子，如果单从规模上来说，似乎不如之前定位成一个行业有那么大的想象空间。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且想要做得好，如交个朋友、东方甄选这些，都要有平台的扶持才可以。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是单从赚钱的角度来说，这条转型路算是风险成本最少的了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>团队只要出内容、选品、投放就可以了，不需要多大的技术开发，能比以前节约很多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最重要的一点是，从平台转成IP、然后做自己的一部分产品，对于多数平台来说由竞争者变成了供给者，他们也是欢迎的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样这里也再次奉劝各位，不要再做电商平台了，尤其大通类目下的产品或者做To C的垂直平台，一般都很难了，早点撤吧。</span></p>
