<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这段以软件自述口吻与用户表白的视频中，多少人被「没有你，我只是 Soul；有了你，才有了 Souler」 这句文案打动。</span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">01</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>以书信为媒表白用户<br></span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>开启一段回忆之旅</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个明显的趋势是，如今越来越多品牌通过广告片以「书信」的形式与用户沟通，比如方太写了一封《油烟情书》，又比如中国银联用一支视频给山里孩子们写了一封回信。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>书信是一种古老的表达思念的方式，也是一种极致的浪漫。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>借助「书信」这种诗意的情感表达，品牌可以快速拉近与受众的距离，构建情感连接。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样的，Soul这次也采用了书信的形式，以软件的身份，给每个用户写了一封信<strong>，将用户作为友人、向导、爱人、知己，分享它眼中那个最真实的Souler。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/161486473962c6cd56a29cc6.28107361.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种沟通的姿态，使得品牌以一种平等的姿态与用户对话，让用户感受到自己被重视。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频中的女生收到信之后，回忆遇到Soul之后自己生活的点点滴滴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从离开家去往陌生的城市工作，一个人搬家、徒步旅行、见到大海、拿到想要的offer、在Soul上紧张地分享自己的歌声、毕业旅行、布置新家......</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/111338351262c6cd56ed7078.22272257.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/69945931362c6cd57343d15.03050211.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/3235436362c6cd583bb611.23205652.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不知不觉，Soul见证了她的成长，那是一个不轻易在外人面前展示的真实的她。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/129448830062c6cd58ad98f9.07211896.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>像视频中的女生一样，在Soul的眼中，每个Souler真实有趣，Ta们收起自己的分享欲，只说给懂的人听，Soul以一个知心人的身份，看到了每个用户不为人知的另一面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而反过来，正是因为有了无数个这样的Souler，才让Soul被赋予了灵魂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是为什么Soul写下</span><strong><span>「没有你，我只是 Soul；有了你，才有了 Souler」</span></strong><span>这句文案的原因。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/131324386762c6cd59721d75.70005386.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在Soul平台上，Z世代用户占比体量较大，平台上的UCG内容赋予了这个软件灵魂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，<strong>对于Soul来说，内容即是社交</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>年轻人通过在平台上分享自己的心声，结交新朋友，打造灵魂社交圈。Soul也借助这些UGC内容提升平台竞争力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相信Soul的用户看完这支视频，或许能看见自己的影子，并从中找到共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，Soul也在回忆之旅中巧妙地带出产品功能，构建产品与用户之间的联系。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>主打灵魂社交</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>塑造知心陪伴者角色</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于年轻人来说，熟人社交或多或少让人产生压力与束缚，无法真实表达自己的情感诉求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，他们迫切需要一个这样的平台，不用顾忌身边人看法，不用察言观色，找到一个倾听者毫无保留吐露自己的心声，释放自己的情绪和压力，对于用户来说，Soul就是这样的存在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Soul创始人的初心是<strong>“让天下没有孤独的人”</strong>，Soul APP自上线以来也一直致力于<strong>“打造随时随地交流真实心意的社交平台”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与其它社交平台不同的是，Soul主打的是“不看脸”的灵魂社交，凭借<strong>“跟随灵魂找到你”</strong>的独特slogan，Soul吸引了一大批年轻用户，深受广大年轻人的喜爱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它给众多年轻人提供了一个放松自己的平台，也为众多孤独而有趣的灵魂提供了碰撞交流的机会。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>很多人用Soul，就是把它当“树洞”一样。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像视频结尾的他们，习惯用Soul记录生活。这样的身份设定，帮助品牌塑造了一个陪伴者形象。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>洞察Z世代社交需求<br></span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>打造年轻人的社交元宇宙</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这两年，“元宇宙”概念被广泛提及。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在去年4月份的时候，Soul也紧跟元宇宙潮流，更换了产品的slogan，<strong>将“跟随灵魂找到你”升级为“年轻人的社交元宇宙”</strong>。希望用户基于兴趣爱好在平台上实现共生，打造多元化兴趣社交。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/158792542662c6cd5e4caf75.86244711.png" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>元宇宙的关键组成部分是社交领域的虚拟镜像、海量密集的互动场景、基于存在的沉浸式体验、共创生态系统及持久空间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由算法驱动的匹配功能和内容信息流，是Soul的主要优势。<strong>一方面，用户可通过星球匹配、语音聊天、视频聊天、群聊派对等丰富功能，与心灵契合的好友直接互动。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/37469070762c6cd5ed79030.09050596.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另一方面，用户也可通过浏览内容信息流找到志同道合的人，“听”别人的故事，在共同感兴趣的话题下相互交流，发展好友关系。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>超1万个以上的兴趣圈层和2000多种引力签，帮助Soul形成兴趣与场景之间的闭环，让年轻人的社交突破现实边界，打造真正的社交元宇宙。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/65771632562c6cd5f6a5995.23760934.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Soul也致力弱化外形因素，让交流更加聚焦于心灵，如不支持真人头像等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>借助高效的匹配机制、丰富的社交玩法和鼓励真实表达的社交氛围，Soul充分满足年轻人多场景、即时性、碎片化的社交需求，不断扩大用户基础。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，<strong>Soul注册用户2亿+，日活1400W</strong>，<strong>主要用户群体为Z世代人群</strong>，是年轻人中最活跃的“朋友圈”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“年轻”是刻在Soul“社交元宇宙”基因里的标签。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在传播方面，Soul也始终紧跟Z世代的潮流思路，用走心的方式打动年轻消费群体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在碎片化时代，唯有用心做App才能留住用户，唯有走心的营销能够占领用户心智。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Soul用真心换真情，陪用户走过每一段旅程。</span></p>
