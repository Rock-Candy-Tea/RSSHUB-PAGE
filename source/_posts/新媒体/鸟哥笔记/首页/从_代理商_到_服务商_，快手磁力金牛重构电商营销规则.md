<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“他们说，我是为直播而生的。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年，踩在直播电商兴起的风口，来自山东临沂的<a href="https://www.niaogebiji.com/tag/%E5%BF%AB%E6%89%8B" target="_blank">快手</a>主播徐小米入了局，做服装带货，每晚直播4-6个小时。仅用了一年时间，涨粉百万，一天直播间最高出单量达8万单。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那时候，徐小米的目标是粉丝数涨到300万、年销售额达到10亿。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两年时间过去，<strong>徐小米的快手粉丝已经达到了1600万，是当初目标的5倍还多，单场直播销售额也实现了从1000万向1亿的跨越，一场直播最高能成交超144万单</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70587093562bd76b79b9d29.52012076.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>徐小米“疯狂宠粉日”直播销售额破亿</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>高速增长的数据背后，快手一站式电商营销平台磁力金牛和服务商济南盘古发挥了重要作用。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>徐小米并不避讳投流工具对生意拓展的帮助，早期她一直用快手粉条获取流量，2021年5月磁力金牛上线之后，她也成为了首批测试使用的达人。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>济南盘古的营销服务，不仅在于流量投放，还利用磁力金牛的数据信息对粉丝进行分层运营管理，并协助她一起打造了自孵化品牌的人设。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是当下许多达人和品牌对于电商营销操盘手的新需求，要既会投流，也懂内容，还能为商家提供针对性运营策略。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了响应这些多元化的诉求，<strong>6月29日，在快手的磁力引擎合作伙伴大会上，磁力金牛宣布将“代理商”体系全面升级为“服务商”体系</strong>，同时发布“乘风者计划”，为磁力金牛服务商提供10亿级的服务商现金激励、亿级的客户扶持激励、和千万级的营销资源。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“2022年，磁力金牛业务进入服务年，专业的运营服务是业务制胜的法宝”，快手磁力金牛运营负责人阿稳说。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>10亿级现金激励，</strong></span><span><strong>磁力金牛“服务商”升级了什么？</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直以来，磁力金牛被视作快手商家的成长加速器。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它将快手3.46亿日活、5.98亿月活的公域流量和188亿对互相关注用户的私域流量打通，再重新打包分配给商家，帮助他们实现涨粉和转化，也促进快手平台的流量内循环。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/42424818062bd76c51cd979.57748611.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>代理商曾是这个加速器的掌舵手。从2018年快手商业化引入第一家代理商、到2021年正式有磁力金牛代理商的业务称谓，至今，已经发展了4个年头，<strong>累计数量超过了180家</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这其中，既有专攻品牌营销的传统互联网广告公司，也有擅长孵化电商主播的MCN，还有深耕电商行业多年的全能选手。他们将商家带入快手商业生态中，也帮助商家在这里扎根生长。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最开始的任务就是做好流量投放，只不过随着直播电商的发展，商家需求也在不断更新。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“内容电商阵地经营意识崛起，单纯的信息流引流投放已不能满意客户的ROI需求，直播+短视频带货成为趋势”，亿凡科特从2018年开始布局电商业务，后来成为磁力金牛代理商，他们切身感受到商家的需求转变。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一次，磁力金牛宣布将“代理商”升级为“服务商”，也是为了适应这种市场变化，其本质是提升操舵手们的综合服务能力，包括三个层次的能力要求：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一层，流量投放能力</span></strong><span>，最基础的也是最核心的，这是商家需求电商营销的初衷。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二层，直播和短视频的运营服务能力。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>把短视频运营好，把直播运营操盘好，是服务商做好磁力金牛业务的重要驱动力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三层，多元电商营销服务能力。</span></strong><span>这里可具体解释为从营销服务延伸的增值能力，比如店铺运营、私域运营、活动策划等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/121477911062bd76c5af2153.12383398.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这三层能力圈会通过服务商的星级被直观呈现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>快手将磁力金牛的服务商划分为5个星级</span></strong><span>，星级越高意味着服务商的综合能力更强，不同星级对应着不同的权益。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>像平台背书、金牛训练营、快速响应通道、产品优先体验这些权益所有服务商都享有，而高星级服务商还有专属线下活动、招商沙龙、优质客户线索等额外资源。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，星级并不是固定的，快手会在每一季度定时开放服务商的入驻和星级晋升通道。要实现星级的跃迁，不同位置的服务商也有各自的门槛和挑战。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以二星服务商升级到三星为例，除了投流能力外，还需具备短视频直播的双引擎运营能力，以及全面的营销服务能力，并且能够实现标准化的团队运营。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/125782473762bd76c61b9533.88465646.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在服务商向上跃迁的过程中，平台也会给予帮助。除日常的运营指导与活动支持外，<strong>快手推出了“护航UP计划”，其中包括高层圆桌派座谈、运营私享会、定制游学服务等成长项目，还有500万的成长赋能津贴。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“在年底，我们期待有20家以上高质量的标杆服务商脱颖而出”，磁力金牛渠道销售负责人王芳介绍。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>为什么整合服务能力升级是大势所趋？</strong></span><span></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“（互联网营销）这个行业门槛低，但实际做得好并不容易。”一从业者谈到。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一些早期入局的代理商，经历了直播电商高速发展的时期，见证了这个行业的秩序和规则如何从无到有、从混乱到有序地被建立起来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当早期红利逐渐消逝，仅仅依靠红利增长的公司难以生存，商家必须要主动做增长，打破僵局，磁力金牛就是增长途径之一。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，<strong>自去年5月磁力金牛正式上线以来，累计服务客户已突破几十万，商家的涨粉效率提升了46%、营销ROI提升了34%</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，电商营销因为门槛低、且标准不统一，也一度出现过行业泡沫期。只要参与过某场成功直播的都敢自称操盘手，稍懂些直播投放能力的投手就极为抢手，市场上最高开出过10万月薪。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/55894098762bd76c72700f3.95597908.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>招聘网站上的投流师招聘</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到如今，快手电商负责人笑古认为，<strong>“粗放增长的时代已经结束，现在是比拼战略思考和专业化、系统化、精细化运营的时代”</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此相呼应的是，2022年被快手磁力金牛定义为“服务年”，从服务作为切入口去寻求新的增长，而此次升级服务商就是顺应这种趋势的一大举措。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前很多磁力金牛的服务商都已经搭建了直播运营策划、货盘筛选、媒介、主播、视频制作创意等多个细分职能团队。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手以人和内容为核心、搭建起的一个丰富的社区生态体系。生意通过创作者的内容，与流量相融合，而许多用户在这里“走走逛逛”，又被内容吸引流连驻足，成为了潜在的客户，也逐渐建立起互相之间的信任感。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>商家的新需求，最终又会通过短视频直播运营、私域运营、活动策划等方式来落地和实现，这就是磁力金牛所提到的“综合服务能力”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如有服务商机构所提到的，“从账户起号、内容策划到直播、引流再到达人曝光，只要客户有需求，我们全部都能服务到”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而且，服务能力的升级整合能直接优化服务商的产业结构。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在过去，服务商依靠流量采买和投放服务模式赚钱的时候，赚取平台返点是服务商的主要收入渠道，升级之后收入途径会更多元，涉及到运营策划的服务费、佣金等，盈利能力也会随之增强。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也因此，磁力金牛渠道销售负责人王芳表示，“服务商、广告主还有平台，三方是相互成就的”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"><span></span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>从0.1到1，</strong></span><span><strong>服务商升级如何助力确定性增长？</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>升级后的磁力金牛服务商，将会对电商营销生态带来什么样的改变和影响？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管平台扶持于近日才正式发布，但从过往案例经验，也能窥见综合服务能力对于商家的实际加成。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>徐小米提到，“商业化投流的过程就是实验的过程”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早期，她在直播间主推时尚潮流的年轻女性服装时，转化效果不理想。团队根据磁力金牛的投流反馈发现，实际上直播间的用户群体为30-40岁女性，就参照磁力金牛的数据信息对货品进行了调整改善。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年，徐小米孵化自营品牌“江南印象”和“珂蒂颜”。<strong>服务商通过磁力金牛对她的粉丝群体进行分层运营，将1600万粉丝分为忠实粉、好感粉、路人粉、沉默粉多个层级，不同层级对应不同的运营管理方式，并重点提升沉默粉的转化率</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/166924161062bd76c78182d5.47121736.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>徐小米与个人品牌“江南印象”</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>羽绒服品牌鸭鸭也在磁力金牛服务商的全链路加持下，实现了从0粉到120万粉丝的跨越。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>鸭鸭自2021年9月入驻快手，同年10月与磁力金牛服务商上海卓狐达成合作。从短视频内容运营出发，到选品、投流、营销策略等各个环节深度配合，5个月时间在磁力金牛的消耗金额达到800万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16801327862bd76c7c123b4.90562387.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，鸭鸭的流量和销量也在这个过程中爆发式增长。<strong>今</strong><strong>年1月，鸭鸭的首个快手超级品牌日，总GMV超过3200万，直播间累计观看次数破800万次，平台搜索环比增加了38%</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些达人和品牌的确定性增长背后，少不了磁力金牛服务商的服务助推。他们做的是打通前后链路，将流量精准匹配商家需求，并发力短视频内容、粉丝运营多个环节，提供了一整套的电商营销方案，促进增长。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是磁力金牛在未来对于服务商能力的期待，并已经被市场验证可行。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在玩法逐渐透明的直播电商下半场，电商营销能助力商家实现从0到1的增长。这其中，简单粗暴的投流或许只能迈出0.1，包含投放、运营、营销在内的电商全链路解决方案才能实现从0.1到1的跨越。</span></p>
