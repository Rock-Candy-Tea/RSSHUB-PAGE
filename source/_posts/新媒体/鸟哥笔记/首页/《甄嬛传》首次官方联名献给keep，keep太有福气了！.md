<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2022都过去大半年了，是谁还扒着2011年出品的清装宫斗剧《<a href="https://www.niaogebiji.com/tag/%E7%94%84%E5%AC%9B%E4%BC%A0" target="_blank">甄嬛传</a>》不放，每到无聊之际循环播放？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">没错，是我！本人除了每天重复过情关之外，相关话题也毫不放过。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这不，最近本青年就在热搜上发现了#甄嬛传在<a href="https://www.niaogebiji.com/tag/Keep" target="_blank">Keep</a>开播了#词条。初略一看标题，还以为keep啥时候悄咪咪开通播放视频服务了，点进去一看才发现，原来是keep和《甄嬛传》跨界合作，推出了剧情跑运动课程，并发布了一支简短的广告片。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">两手热身伸展，戴上耳机，轻轻打开keep，就能瞬间穿越到古代皇宫里，化身为甄嬛，体验一堆宫女公公向你请安的剧情，这样的快乐，哪个剧迷不说句真香？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此时此刻，脚下的不是跑道，是宫里的砖；身上的不是T恤，是嬛嬛的衫。这不妥妥的梦想照进现实，现实版《重生之甄嬛大战后宫》呐。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而且本青年还特地去搜索了下，发现这是电视剧《甄嬛传》的首次官方联名，时隔多年后的第一次联名，就献给了keep，谁看了不说一句keep有福气呢。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/91279495962cf0b556795c1.16424240.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">鉴于视频太短看不够，本青年火急火燎地打开keep，去查看了一番，发觉除了可以化身梅园许愿的甄嬛之外，还可以变身为仗义救人的眉庄或者是算计甄嬛的华妃等。多重角色供君挑选，一次性过足你的穿越瘾。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/187223752162cf0b559846e3.61341263.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上面这些剧情向课程，全是根据电视剧《甄嬛传》的名场面还原和改编而来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如剧情跑第一节课程《甄嬛梅园许愿》，讲述的就是甄嬛除夕夜不想参加合宫夜宴，佯装身体不适，偷偷跑去倚梅园挂小像许愿的故事。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二节课程《眉庄仗义救人》，则是以沈眉庄急去解救被余答应霸凌的欣常在为故事背景，展开了一段刺激的救人桥段。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">每个课程都以第一视角展开，360度环绕在你耳边的宫女语音播放，让人有种在耳边说话的错觉，再搭配上有节奏感的配乐，减少了些独自跑步的枯燥感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时为了进一步增加运动的趣味性，设置个“小目标”激励大家，keep还推出了一系列的周边。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">双面甄嬛奖牌</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/136627183962cf0b55d538a6.59863503.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">四大名妃徽章套盒</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/42727538162cf0b5619ace8.64117848.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">专属皮肤装扮</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/5745656062cf0b5660a560.82972801.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">剧情跑课程和周边上线不久后，就得到了一堆网友的关注和讨论，甄嬛传自带的这波热度，着实是被keep给蹭到了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">甄嬛传十级学者发来祝贺</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/105350473562cf0b569bea64.45771968.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">时隔多年的首次联名</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/30842875962cf0b56c49cb8.27344867.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">每跑次步，就回忆一次剧情</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/7786645962cf0b5709f2b3.49331813.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">降低跑步门槛</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/139422183862cf0b573322d6.00397396.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">和自带流量的国民大热剧联名，以及有梗的剧情跑课程设置，keep这波操作，可以说是热度和好评双丰收。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/28302723362cf0b575af920.40528045.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">虽然剧情跑本质上还是简单的快慢跑交替训练，但给用户安上甄嬛传主角的设定，以及把剧情披上了悬疑的宫斗剧外衣，顿时让跑步运动有意思了不少。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这次keep和甄嬛联动推出剧情跑课程，也让本青年想到了三月前曾上过热搜的《娘娘快跑》课程。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/120894777462cf0b578b28c3.38487020.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“剧情燃脂跑·娘娘快跑”主要讲的是穿越到清朝变身为娘娘，醒来和宫女一起追太监小偷的故事，又可亲切地称为《重生之我在紫禁城抓太监》。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为清宫剧《甄嬛传》太深入人心，当时不少网友还奇思妙想地改编剧情，跟《甄嬛传》来了场梦幻联动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/49075464762cf0b57d765e3.60099845.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">想必keep是听到了网友们热烈的呼唤，在三个月后，真的和《甄嬛传》来了场轰轰烈烈的合作。莫不是人间有真情，人间有真爱？本青年宣布，此乃钮钴禄氏·keep。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">▼</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总的来说，Keep这次和《甄嬛传》正式联名推出的“剧情向”课程，迎合了年轻用户热衷讨论和观看《甄嬛传》的现象，为新课程引来了不少忠实剧粉，同时也扩大「剧情跑」IP的声量，打造了品牌独有的特色内容。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">毕竟跑步就能化身女主，沉浸式体验《甄嬛传》剧情，和主角们同跑道来一场“精彩飙戏”，哪个剧粉会不心动呢？</p>
