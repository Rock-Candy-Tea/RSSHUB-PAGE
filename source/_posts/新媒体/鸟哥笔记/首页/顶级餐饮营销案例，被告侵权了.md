<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>各位好，这里是头条哥。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有时候看新闻，真的有一种穿越到几年前的感觉。就说今天的热搜</span><span><strong>#李小龙女儿向真功夫快餐索赔2.1亿#</strong></span><span>，很多人看到这条新闻才反应过来“真功夫”和“李小龙”没有一点关系：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/979939206307932f8c6116.25278790.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但实际上，相关的侵权案件早在2019年就已经拉开了序幕，以</span><span><strong>#李小龙女儿起诉真功夫餐饮#</strong></span><span>的话题冲上了微博热搜：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11620894166307932fd90725.05311231.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时的真功夫摆出了抓捕鲁迅和我周树人有什么关系的态度，认为</span><span><strong>“我们没有侵权，不会寻求庭外和解，也没有更换真功夫品牌商标的计划”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/625866646630793301f5dd2.87813863.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3489056063079330798841.85902537.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在更早的2010年，李小龙女儿李香凝就已通过媒体意图状告真功夫侵权，但当时双方并未启动司法程序。她与国家工商行政管理商标局进行了接洽，得到的回复是：李小龙及其英文姓名“Bruce Lee”开发利用的权利应当归其继承人所有。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以说，</span><span><strong>真功夫和李香凝关于商标的争执已经有十多年的历史了，</strong></span><span>这期间真功夫为了避嫌也做出了一些改变，在2016年更换了新的形象，模糊了李小龙的形象，并沿用至今：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/201616995163079330b777c9.99862221.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过新的logo被认为是李小龙的漫画版，实际上还是存在着侵权的风险。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，真功夫的品牌早已和李小龙形象捆绑在了一起，</span><span><strong>如果失去了这个形象，真功夫的摇摇欲坠餐饮大厦的根基就会动摇，</strong></span><span>这也是真功夫为什么不能承认并彻底换掉争议商标的原因。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而将“真功夫”和“李小龙”捆绑在一起的人，就是著名的</span><span><strong>土味洗脑广告的开山鼻祖、冲突理论创始人、本土营销大师叶茂中。</strong></span><span><strong> </strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/295484444630793311886b2.96923279.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 从双种子到真功夫 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真功夫原名叫“双种子”，1994年在东莞开办了第一家店，经营到一定规模时，双种子将店面扩展到广州、深圳市场，然而面对新市场，双种子显然有些水土不服，营业额始终徘徊不前。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14136783646307933158cd08.25180119.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时，叶茂中就出现了。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3499672763079331a1f7a8.25251212.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>叶茂中依据自己的品牌符号冲突理论，帮助双种子抢占了一个文化符号——</span><span><strong>李小龙</strong></span><span><span>，用中国功夫文化为品牌赋能，借势功夫文化，不仅让消费者更快</span>地<span>记住了品牌名，并且赋予其更营养，更健康的品牌联想。</span></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/13143698516307933207c0b6.16268760.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>双种子自此就改名为真功夫，也换上了李小龙logo。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>叶茂中也曾运用相同的方法为大红鹰抢占了代表胜利的”V”的符号元素：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/118924353063079332479127.95803336.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以及借势于《阿甘正传》，让消费者能体会到企业不怕吃苦精神的阿甘锅盔：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/173562771563079332a1eb78.60634529.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在他的冲突理论中，这类运用消费者熟悉的元素树立的品牌符号被称为</span><span><strong>自燃型符号</strong></span><span>，自燃的关键，在于能瞬间吸引消费者的注意力——</span><span><strong>尽量利用消费者熟悉的元素。消费者更容易将注意力分配给那些熟悉的元素。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果大家熟悉本土营销公司的话，应该可以发现，叶茂中的冲突和华与华的超级符号在这些方面有着一定的相通之处。</span><span><span> </span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><img src="https://ngbjimg.xy599.com/23465296663079332d652d5.68851241.jpeg" referrerpolicy="no-referrer"></span></span><span><br></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span> 拳打肯德基，脚踢麦当劳 </span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>叶茂中接手双种子，并不仅仅止于帮助双种子顺利在广东站稳脚跟，而是放眼全球，意图坐中式快餐的第一把交椅。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是双种子的竞争对手就已不是其他中式快餐品牌，而是中国市场的两大洋快餐巨头，肯德基和麦当劳。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12979380496307933356e2b7.03447665.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>叶茂中洞察出了当时人们对于西式快餐好吃，但却没营养不健康的冲突，在食品制作方面找到了中国消费者的心理偏好，为真功夫提出了一条slogan：</span><span><strong>营养还是蒸的好。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/148029454963079333b65e54.59349621.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>真功夫从品牌形象到产品线都进行了以“营养健康”为核心的差异化运作。<br></span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随后在产品线上，真功夫大刀阔斧地砍掉了砍掉薯条、鸡翅等西式类油炸食品，只保留中式以蒸为主的快餐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>叶茂中也更改了真功夫的定价策略，过去双种子的客单价是12元，彼时竞争对手麦当劳则是19元，当时叶茂中就决定从施行新策略开始，</span><span><strong>永远都比麦当劳贵1元。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>因为“有营养的快餐”，当然应该比“没营养的快餐”更贵。<br></span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/200394535963079333e7e436.42257435.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真功夫的线下门店策略也为之一变，</span><span><strong>很多店铺选址专门挑在肯德基，麦当劳旁边，</strong></span><span>在油炸快餐包围中突击出来中式快餐的一席之地，同时让消费者在潜意识中认为：真功夫和其他两家都是大牌快餐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“要在距离对手最近的地方开炮”，</span></strong></span><span>叶茂中在他的《冲突》里写到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/158818359863079334380fb8.02155653.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>异军突起的真功夫，着实震动了两大竞争对手，以至于后来两者如果要和某一物业签订合约，往往会在合约里同时要求其不得将周边物业租给真功夫。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在餐饮市场上开始大展拳脚，一路高歌猛进的真功夫，</span><span><strong>成为了当时中国本土快餐的第一品牌。</strong></span><span>在2014年就已经在全国拥有600家直营店，营收达到50亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>头条哥觉得，</span><span><strong>仅在营销方面，真功夫确实有种李小龙的感觉。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span> 叶茂中斯人已去，真功夫风光不再 </span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真功夫是中国快餐少有的能够冲击肯德基，麦当劳的品牌，同时叶茂中在其中的营销也是极为经典的案例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但无奈的是，真功夫后期因为内部管理权之争日渐衰落，当时未经授权使用李小龙的形象也给企业埋下了一个隐患。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>叶茂中在今年1月份去世，结束了一个时代。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1076182182630793347a52f7.27266185.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>而和李小龙形象高度捆绑的真功夫，已经无力也无人去重新选择一位功夫明星，从头开始重整品牌策略。</span></strong></span><span><strong><span> </span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><img src="https://ngbjimg.xy599.com/151452178363079334c6e924.38756049.jpeg" referrerpolicy="no-referrer"></span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真功夫因为侵权第一次在星期四这天热度超过了肯德基，真是一种黑色幽默。如果当年真功夫的中式快餐梦能继续做下去，今天我们的朋友圈里也很可能是：</span><span><strong><span>v我50，去吃真功夫。</span></strong></span></p>
