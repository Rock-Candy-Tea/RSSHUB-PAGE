<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>引爆、刷屏、带货、爆卖……</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今做<a href="https://www.niaogebiji.com/tag/%E8%90%A5%E9%94%80%E6%B4%BB%E5%8A%A8" target="_blank">营销活动</a>，似乎都得达到这些令人咋舌的效果，才称得上是“大成功”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>仔细看这些词语背后，其实都关联着两个核心：一是社交，二是电商。而事实上，自从移动互联网成为新基础设施，品牌方的预算也都在陆续向着这两个方向迁徙。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>品牌对社交的偏好，是在门户、搜索后就有的。</span></strong></span><span>社交媒体先是最早推出了集原生性强、兴趣推荐、按效果付费等优点于一身的信息流广告，之后又赶上了问答、直播、短视频等内容的热潮，现在又有不少成熟的社交媒体已是“超级APP”，层层折叠的功能留住了流量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>品牌对电商的偏好也不难理解。</span></strong></span><span>电商原本就离销售转化更近，在更追求增长的当下，显然就要比其他类型广告更具吸引力。而且大众消费习惯现在仍在加速向线上迁移，这也是推动电商广告保持增长的源动力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过新的挑战也应运而生，当所有人都明白在社交、电商上持续投入的意义，行业越来越"卷"，品牌还能如何找到增量、脱颖而出？而更大的挑战还在于当前全球商业都进入一个瓶颈期，不少品牌都不得不缩减预算并以更加精打细算地方式重新审视其在不同媒介端口的投入效果。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>外部环境持续变化，在广告预算社交化、电商化的“后时代”，品牌的"鱼"和效果的"熊掌"怎么兼得？</span></strong></span><span>品牌方又该如何在变化中抓住不变的核心，找到锚定穿越周期、走向未来的清晰路线？这些都是我们逢山开路的必答题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81142426261420d300e45a3.29058312.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>新核心目标：做长期品牌建设</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>变局之中，未来走势难预判，</span><span><strong>但品牌方有一个目标更为清晰和坚定——要做长期稳定的生意，真正做出消费者认知、认可的品牌。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，越来越多的新消费品在成为“网红”后其复购率就难提升，很难称得上是真正的“品牌”。想要前行，营销投入就停不下来，因为流量玩法停投就停效。而同时流量价格又越来越贵，竞价型玩法也越来越难跑出成绩，由此陷入恶性循环。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原可口可乐大中华区及韩国区CMO、贝恩消费品全球专家合伙人鲁秀琼曾告诉「深响」：“互联网流量打法砸出了短期业绩，但大家容易把短期ROI和长期主义的品牌建设混淆，品牌必须有品牌建设的长期主义。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>因此这几年有一个非常明显的趋势是“纠偏”，从转化玩法回归品牌建设。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手商业化的变化也体现着这一趋势过程——2018年快手开始商业化，当时在品牌方眼中快手还只是一个做效果广告的渠道。2020年，在内容生态更为丰富和内容营销能力提升的基础上，快手开始发力品牌广告。根据今年二季度财报数据披露，快手的线上营销收入同比增长10.5%至110亿元，而因提供多样化营销需求的定制解决方案、招商营销模式的创新，让其品牌广告的增速还超过了整体线上营销服务收入的增速。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>群邑中国首席投资官底飞向「深响」回忆，其与快手从2017年开始合作，主要先是硬广方面；但到了2020年，在全球品牌广告代理中占有重要份额的群邑将快手放在了群邑的优选媒体清单中，合作量级实现了近百倍的增长；现在，双方合作的内容领域更加多元化，包括硬广、达人、内容、明星、效果</span><span>（信息流竞价）</span><span>等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1906609445630c76b32c3db8.92048390.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>平台和品牌方的认知与行动并非完全同频。对不少品牌方而言，品牌建设仍是知易行难的事，大趋势下是无数难以厘清的问题：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>习惯了流量玩法的立竿见影，对于品牌建设的预期怎么确定？品牌广告的长效价值不等于当下就不看效果，那么这个效果又将如何呈现？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先应明确的是，品牌建设的逻辑和流量打法完全不同。流量打法本质上是“数学题”，只看最终的收支比结果；但</span><span><strong>品牌建设是逻辑题、语文题，</strong></span><span>要理解品牌调性，要拿出与品牌契合的营销内容，而且还要关注营销内容的传播力、对用户的情绪感染力等等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而内容需要由人来创作，因此另一个问题也随之而来：谁会是最好的表达载体？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直以来，品牌的营销内容都是从上而下、我说你听的单向传输，消费者只能被动接受。这在信息爆炸、消费者又站在核心位置的当下，显然不够有效。快手磁力引擎内容商业化负责人毛海峰也曾表示，营销分三层，一层是硬广曝光，二层是内容互动，三层是情感及精神层面的深度链接。硬广曝光是传统品牌广告都能做的，但要走到用户情感链接这一层面，就需要借助一个介质——达人。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>达人的重要性毋庸置疑。他们是内容产出者，为内容生态带来“源头活水”；</span><span><strong>是“桥梁”，参与商业活动，链接用户、服务商、品牌等诸多角色；同时还可以参与到更多形式的内容、活动、项目中去，充分发挥其自身价值。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也正是因此，短视频平台都在持续聚集着海量、多元化的达人，生成完整生态。据群邑底飞观察，过去几年快手一直在针对达人方面做调整，整合达人的类型和发展方向，提升快手平台的流量组成和品牌流量需求的匹配效率，让生态架构更清晰，方便品牌方明确合作目标，也更能匹配品牌方的诉求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1798642138630c76b3888c56.49116870.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从快手发布的报告来看，目前其达人已形成由头部、高潜和种子构成的完整金字塔结构，覆盖超40个垂类，截至2022年6月底，快手达人生态营销平台磁力聚星整体入驻达人数已接近170万。有此生态基础，在快手近日发布的《新市井商业营销通案》中，快手的“达人+X”策略也正式发布、随之展开。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如“达人+流量”的结合，是指品牌以达人素材投放广告内容，或以达人原生内容植入做流量助推。由于普惠机制的存在，快手达人内容真实有感染力，粉丝信任度高，以往商业内容和原生内容难平衡、难融合的问题也会较弱。据接近快手的人士透露，近日快手商业化在大力建设“原生广告”，这种模式可以撬动更多优质的原生自然流量，获得流量红利。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而“达人+内容”，是指达人可以加入平台自制的短剧、综艺、直播、晚会等诸多项目。内容本身的吸引力和达人的影响力一经结合，即可带动品牌更快突破曝光天花板。比如去年快手携手笑果文化打造的脱口秀直播综艺《超NICE大会》，有带货达人@徐婕、2021百大主播@田斌+、TOP1短剧女主角@ 一只璐A DEAR等诸多达人参与，该综艺获得了超115亿曝光量，站内二创播放量甚至达到了正片播放量的7.3倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1453465063630c76b406afb7.39796221.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>快手《超NICE大会》</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，达人还可以参与挑战赛，以“达人+互动”的组合在站内卷入更多用户玩在一起，更有效实现品牌信息的传递和用户的情感共鸣。还有“达人+电商”的组合，可以让达人直播带货、做分销，这也是在为品牌做转化提供助力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>达人安身立命之本就是产出优质内容吸引和留住用户，通过内容和用户真诚交流，提供更多情绪价值。有“达人”这一介质的帮助，营销自然事半功倍。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌虽然是一种无形资产，但品牌建设并非是无法考量、无据可依的“玄学”。明确营销内容的投入方向，找到更好用的表达载体，<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E8%90%A5%E9%94%80" target="_blank">品牌营销</a>可以更快让品牌方有所获，换取未来长线收益的时间差会明显缩短。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213179696861420d317ab820.36069667.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>品牌建设的新路径：多元整合</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，找到达人帮助品牌做营销还只是第一步。</span><span><strong>品牌建设是一个综合的、系统的、连贯的、长线的动作，需要品牌方沿着一个清晰的路径——多元整合来持续前行。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为何要“整合”？首先是因为单线思维是行业长期以来的通病。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做品牌广告，很多品牌方都只是想到露出LOGO的冠名、贴片，各种TVC、户外大屏、短视频信息流等等。这种硬广是相当标准化的资源，而品牌在不同的发展阶段，所要达成的营销目标在不断变化，只用单一类型的硬广就会存在局限性。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，做整合是基于营销的本质来考虑。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>营销是从曝光走到转化的过程，品牌当然是希望这个过程越短越好。而要实现这一诉求，单一玩法不可能“放诸四海皆准”，品牌方势必要用多元化的营销玩法组合运用。这其实也是“整合营销”概念被广泛提及的原因所在。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，外部环境也在要求品牌方走整合之路。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据底飞介绍，</span><span><strong>单纯投放短视频信息流这种形式已经非常饱和，这意味着竞争的白热化。</strong></span><span>同时，营销领域里的其他角色也在走多元整合之路，比如从群邑长期的品牌营销经验看，大多数品牌开始进入了更多赛道，包括硬广、达人、内容、艺人、效果等等；各大互联网平台也走向了整合，一次营销活动中，往往是达人、搜索、社交、直播、电商、竞价等形式齐上阵。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以快手为例，其完整营销生态目前包括了内循环和外循环两个部分，前者指在平台内进行直播带货的品牌商家以及主播，为了获取更多曝光而进行的广告投放；后者指品牌方直接在平台上投放品牌广告与效果广告。内外循环两种营销模式的发展，既依赖于平台电商生态、广告生态的繁荣，也需要平台在市场、资源、产品、工具等各方面有持续积累。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌必须要走整合之路，而此时平台先行整合，正是在为其提供便利条件。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如效果广告主要是看流量的效率，</span><span><strong>各大平台近年来的技术和产品升级，都是在力图让营销更数字化、精准化、提质增效。</strong></span><span>快手商业化最初是凭借着效果广告起家，现在其发力点也是在进一步提升变现效率。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而品牌广告方面，品牌虽然都是树立品牌形象、提升品牌影响力等为目标，但个体仍有明显差异。所以</span><span><strong>现在各大平台都在强调做定制化、一体化解决方案，其目的也是想要更细致、更具针对性满足品牌方诉求。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1711871430630c76b4bd80f4.64385261.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>冠芳、玉泽快手官方账号</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位接近快手的人士曾向「深响」举例，冠芳旗下品牌山楂树下，其所在的山楂汁赛道中并没有太多竞对，品牌投放目标就是要尽可能覆盖庞大用户，打出品牌声量，所以品牌选择了在今年春节期间冠名快手的春节红包活动。而上海家化旗下的玉泽，其所在的护肤赛道属于红海一片，营销玩法要达到的目标是体现品牌以及具体产品的差异化优势，而通过多元化的短视频、直播等内容，用户就能够对品牌有更立体的认知。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手的内循环+外循环整合，再加之其实施的“大搞品牌”战略，更多品牌方的思路如今都已转变。原来品牌来快手只想多投渠道，做大曝光，但由于获得了超预期的数据表现，以及关注到站内的更多资源和玩法可整合运用，品牌也开始了新尝试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1563908158630c76b6bbfdc3.32083994.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>快手平台上威尔胜投放的相关节目及品牌账号</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>威尔胜篮球就是一个典型案例。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为NBA 21-22新赛季官方指定用球，威尔胜篮球需要绑定NBA IP内容做推广。所以在快手宣布与NBA合作后，品牌既冠名了快手自制的NBA相关节目，也在NBA短视频内容里做了包框广告，实现了圈层群体的广泛触达。并且品牌也在快手上开设了品牌账号做粉丝沉淀，以及通过快手直播间转化组件，引导用户跳转官方店铺下单。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去，威尔胜篮球可能只是在比赛中和比赛前后的广告片中出现，借助NBA传递出了品牌及产品是什么的信息。但现在，从品牌曝光到引导篮球爱好者下单购买，品牌已是一气呵成的实现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这正是“整合”的价值所在。相比在营销中的某一个或几个环节分散进行操作，以宏观视角直接确定好整合的路径，即便其方式、玩法在不断转变，品牌方都能运筹帷幄，掌握整个营销结构中不变的核心重点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20585534276144238ed0dc54.24202450.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>多元整合的新思路：品效销闭环</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过需要强调的是，从单一、分散到走整合之路，并非是指品牌方将各种营销玩法简单罗列打包就行，因为这很有可能造成重复造轮子、混乱且低效的问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌始终都是为了更好地经营和更好地销售，营销总是需要引导消费者从认识产品走向产生兴趣、最终买单。由此来看，为避免简单打包，品牌方应该有的新思路其实是实现“品效销闭环”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此前行业内常说的是“品效合一”，“品效销闭环”又如何理解？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“品效合一”中，“品”和“效”其实是两种广告类型，</span></strong></span><span>主要想表达的是品牌方应该两类广告并用，短期销量增长和长期品牌建设兼顾。</span><span><strong>而“品效销闭环”是基于做生意的思路，</strong></span><span>强调品牌只看曝光如何“热闹”意义不大，更应同时关注“营”和“销”两个部分。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于“品效销闭环”，底飞也有更细致的思考。他认为，“品效销”中的“品”，是实现覆盖、知晓、喜好、曝光等等；效，起到激活销售的作用，信息流、KOL、社交营销等等都属于其中；销，即电商交易、下单转化。</span><span><strong>但品效销并不是按顺序依次进行，品牌方可以把预算同时放在这三个篮子里，</strong></span><span>同时发力，缩短转化链路。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“闭环”，其实是先对平台提出了要求。2021年前，内容平台和电商平台“泾渭分明”，想要实现闭环需要通过deeplink形式</span><span>（链接跳转）</span><span>。但这其实有困难，一方面是各平台有生态壁垒的阻隔，另一方面用户也不一定所有APP都具备。现在，内容平台都有自己的销售场后，内容和电商就快速拉近距离，被整合在了平台内。在此基础上，品牌才能够实现真正的品效销闭环。毛海峰也曾提到，快手对于品牌营销的设计和规划，首先是基于快手特色的内容生态，其次就是将闭环的思路贯穿始终，“利用直播、短视频、电商三位一体的载体，实现品、效、销三位一体的结果。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，总结来看，</span><span><strong>实现“品效销闭环”，品牌方要有清晰认知，但同时也需要投放的平台已成闭环，以此为基础才能真正行动落地。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌方和平台之间具体如何配合，通过对营销流程进行分解即可呈现：在前链路，通过多元化内容</span><span>（包括短视频、直播和平台IP等）</span><span>与人</span><span>（包括达人、明星、用户）</span><span>的结合，</span><span><strong>追求流量效率的最大化，</strong></span><span>为品牌尽可能吸引到庞大的流量。在后链路，以品牌官方号做“阵地”承接被吸引而来的流量，继而同样是依靠内容</span><span>（自制短视频、直播等）</span><span>和人</span><span>（明星、达人、主播等）</span><span><strong>去做转化，以及沉淀用户资产做复购。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>花西子在2021年初入驻快手后，在营销、直播、日常运营等方面形成了一整套以长效经营为目标的策略。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌先是以信息流广告投放、达人种草内容流量助推，积累原始粉丝；而后联动平台IP活动、达人直播，提升站内的影响力和好感度，更快速引流增粉；同时还借助不定期的品牌日活动和常态化的自播，形成闭环，持续转化。入驻半年，品牌触达人群的精准度和粘性都有所提升；截至目前，其官方旗舰店账号积累粉丝数量已达873.2万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1503934718630c76b8580c50.68948737.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>花西子快手官方账号</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>良品铺子亦是如此。自2020年入驻平台以来，品牌快速与达人展开了短视频和直播的合作，达人合作单场GMV最高曾达到2200万，成功构建了从达人种草到达人分销的完整链路。同时，良品铺子也通过和快手的多个营销IP合作来引流破圈，通过每日固定时间的自播来做常态化经营，其矩阵账号首播GMV就达到了2300万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/253720639630c76b985e157.77239892.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>良品铺子快手官方账号</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在不确定大环境下，品牌方既要在短期内看到销量增长，又不能忽视做品牌为未来谋划；在激烈竞争之中，品牌方既想要保持其影响力，但又不得不考虑预算。而归纳来看，品牌方向着品牌建设的目标、多元整合的路径以及品效销闭环的思路转变，正是针对这些痛点的解法。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>变化无处不在。从品牌成长的底层逻辑来看，我们可以发现穿越周期的大品牌们，往往都是做对了同样的事——拥抱变化。比如宝洁进入沃尔玛，伴随大型商超实现快速成长，这其实是品牌及时抓住了渠道变化的红利。可口可乐、百事等国际品牌一路以来都是品牌营销的范例，现在也依然是全链路数字化转型的优秀代表。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目标、路线和思路的转变，其实也正是品牌可以穿越周期、长期增长的大前提。及时抓住“变化”的奥秘，品牌就有了在拥挤赛道中先行一步的时间窗口和实力。</span></p>
