<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在短视频成为主流消费内容、直播连接起“营”和“销”的时代，<a href="https://www.niaogebiji.com/tag/%E8%BE%BE%E4%BA%BA" target="_blank">达人</a>站在了极为重要的位置上。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经，达人指的还是因热点话题而诞生的红人，只影响特定兴趣圈层的意见领袖。而现在，达人的身份迭代，他们创作和变现能力兼备，走向了更为职业化的道路，并且更多不同行业的人都在尝试以此为<a href="https://www.niaogebiji.com/tag/%E5%88%9B%E4%B8%9A" target="_blank">创业</a>新方向，呈现出了“全民化”的趋势。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今时不同往日。时代背景不同，加入“达人”行列的个体差异更为显著，这就意味着过去“成功者”们的方法论不一定适用于当下，成长路径也不一定能直接拿来做对标。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>举一个简单的例子。以前汽车内容并非绝对主流的垂类，一是因为内容更具专业性，对达人来说门槛较高；二是因为关注此类内容的用户比较固定，并不算是特别大众向；三则是因为汽车品牌们仍在传统营销思路中，还未特别看重这类内容营销。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但现在情况已截然不同：有更多汽车领域从业者，主动来到短视频平台上做达人；随着消费的升级，更多人关注汽车，且喜爱的内容风格也更为多元；汽车品牌注重体现年轻化和亲民性，更愿意通过达人来做种草。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4512112362e155a1edf5e9.50310052.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不难看出，不论是在创作还是在变现方面，现在的达人并不是有流量就能长期、稳定、轻松成长，而是需要思考更多问题：包括如何选择更适合自己的道路，如何既保持内容创作初心，又能在创作中走好创业路等等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此时，谁能为达人们提供指引？站在行业视角来看，作为离达人最近、能第一时间体察其生存发展状态的平台，巨量星图已经展开了行动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81142426261420d300e45a3.29058312.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>“大师”助力达人</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许在不少人看来，巨量星图更多是在做达人和品牌的撮合；但现在，巨量星图开始变得更有温度，它和达人站在了同一位置上，与达人一同思考着如何平衡“内容与商业”、“成长与变现”这一核心问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期，巨量星图就联合北京大学创业训练营举办了2022“Star Up星光之上”大师课，为达人带来了一次与国内知名学者、有影响力的企业家、平台、同业者面对面交流的机会。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4568150962e155a223d8c5.43799979.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>巨量星图此次邀请到的都是积累了千万粉丝、且拥有较强变现能力的达人；并且他们来自于不同领域，汽车、美妆、食品饮料、文体等等都覆盖到。例如痞幼、小宇、鹤老师说经济、商业小纸条real这些备受喜爱的达人都参与了这次活动，也因此</span><span><strong>成功构建起了一次有价值的圈层体验。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于同处于头部，达人们对内容认知和商业变现有着相似的认知理解，也都有共同语言。在活动中，这些达人聚在一起，通过课程学习、组队交流等形式密切交流，也就很容易就形成沟通的“同温层”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为来自于不同领域，达人们不仅会“同频”交流，也会有智慧碰撞，从而产生出新的火花。这次活动中，巨量星图也特别将不同垂类的达人进行了组队，创造出了更多跨圈交流的机会。达人们彼此分享经验、取长补短，不仅可以提升对内容的认知，也能打开创作、创业、合作的新思路。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音上拥有1461万粉丝的达人 @崔磊-为思考点赞 就表示，他曾是传统媒体从业者，做短视频也相当于是踏入了新的领域。此次来到大师课，他认为是抓住了一次和同行交流的机会，同时也获得了更多底层逻辑的点拨。沉淀下来的智慧，将会持续为后续的创作所用。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/110602335662e155a2528d62.82552191.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>达人所说的“智慧”，既有同行的交流，还有更多来自“大师“们的分享和讲解。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此次活动，巨量星图与北京大学创业训练营合作，邀请到了多位在不同领域都有建树的北大校友们，借助他们积累的学术理论、行业认知和资源，</span><span><strong>给达人搭建成长知识体系。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这其中，有从教育跨行至电商直播的企业家俞敏洪，他所讲述的新东方华丽转型思路和经验就非常适合为很多从业者带来启发；也有北京大学的向勇教授，他对短视频文化传播有多年研究，积累了成熟的理论体系，可以为达人提供更“高屋建瓴”的学术知识；还有前第一财经CEO周健工，他的媒体知识也能给达人以多元启发。巨量星图的相关负责人也在活动中分享了达人当下的机遇和挑战，平台的感知洞察颇具实用性。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不难看出，此次的大师团队既有着很高专业度，不论是学者还是企业家，都是对短视频、对达人营销有专业知识储备，也有自己的独到见解；同时又非常多元化，多重身份带来了多种观点，即有宏观的行业产业洞察，也有更为细节的实操逻辑，兼具实用性和理论高度，知识丰富全面。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大师们带来的行业信息和学术研究在集中释出后，其实也和达人们已有的认知和理解形成了一次“大碰撞”，新的灵感、启发和思考都由此而迸发。例如同样是教育行业从业者、拥有接近1500万粉丝的达人@胖超说艺考就表示对向勇教授、俞敏洪的演讲深有体会。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/92821392162e155a27eeb41.98958891.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两天的大师课活动，不仅是对已有知识的整合，也是一个诞生新知识的场域。而知识本就具有传播和积淀的属性，巨量星图后续还将会以系列化的活动，不断沉淀出更完整全面的知识，在平台上真正搭建起立体化的知识体系，为更多达人所用。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，此次大师课活动不仅让达人们积蓄了更多知识作为“内生力量”，也为他们带来了更多能够积极“外向生长”的信心。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这得益于活动最后，巨量星图为达人们准备的结业证书等多种荣誉性实物认证和奖励。平台给到具体实在的奖励，让达人对平台形成了更强的信任感，感受到平台想要帮助达人成长的用心，也就更愿意长期稳定留在平台上共同成长。同时，知识界、商业、平台的全方位认证和背书，也在让达人意识到这份职业所具备的专业度、重要性，增强达人的职业荣誉感和创业信心。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/165032903462e155a2c8e513.27932594.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚毕业一年的达人@钟婷xo，凭借记录和讲述情侣之间真实生活状态的内容而在抖音上收获超两千万的粉丝。在谈到“达人”作为职业的感受时，她认为创作并不轻松，但坚持是最重要的。通过这次大师课，她也更加坚信，只要坚持创作，年轻人就可以在此找到自己的职业方向和未来发展路径。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无形的知识、有形的认证，双重收获下，巨量星图让达人们的成长之路有了更清晰的方向，其持续向前的脚步也将更为坚实和笃定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213179696861420d317ab820.36069667.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>挖掘“源头活水”，</span></strong></span><span><strong><span>助推行业正循环</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>千万粉丝、商业变现，其实都并非达人们成长的终点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>达人经济的市场规模庞大，且未来发展空间可观。据卓尔数科《2021年<a href="https://www.niaogebiji.com/tag/KOL" target="_blank">KOL</a>市场研究报告》显示 ，从2018年至今，达人营销的市场规模以每年40%到50%的增速稳步提升，2021年达人营销市场规模已达980亿元。宏观情况如此，落在巨量星图这一具体平台上，同样成绩突出：截至2021年12月，巨量星图入驻达人总数已经超过156万，服务客户规模已经突破120万家。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>达人营销的市场不断扩大且进入深水区，最为核心环节的达人，其重要性自然不言而喻。这一背景下，巨量星图推出大师课可谓恰逢其时——达人需要成长，需要认可，更需要方向指引，而大师课正是想达人之所想，从更有温度和关怀的角度，切实地带动、帮助达人。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上大师课也只是巨量星图的行动之一。早在2020年，巨量星图就推出过优质潜力达人商业化培养计划“繁星计划”，该计划在近几年来持续为平台聚拢着众多中腰部潜力达人。此次面向头部达人的大师课活动在后续也将持续多期，同时据「深响」了解，巨量星图也还有更多面向达人的活动正在筹备中。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不论是更具针对性的活动，抑或是给到方法论指导、用持续更迭的产品加持，平台的目标都是为粘合更多达人，实现各梯队达人能力的全面释放。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/73649567262e155a3081870.53492140.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从平台自身来看，这将充分发掘出达人作为内容生态“源头活水”的能力和价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着巨量星图更多计划和活动成果的落地，达人如何兼顾创作和创业、平衡成长和变现将不再是宏大、难解的命题，各梯队达人“知来路、更知前路”，自然就会对平台有更多信赖感。而具备实力的头部达人和具备潜力的中腰部达人、新人，都会繁荣平台健康的生态，吸引用户、品牌在此停留，进而也就成为了平台内容变现的动力源。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从行业整体来看，这也将为整个达人营销行业带来正向的助推力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“内容为王”现已是营销领域的金科玉律，这背后有诸多因素的影响，包括流量红利的不再、用户注意力的碎片化、广告投放结束后效果的转瞬即逝等等。达人安身立命之本，便是用内容吸引和留住用户，通过内容和用户真诚交流。而这其实正是品牌的诉求点所在：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为有创意，达人创作的内容会自然而然地吸引眼球、广为传播，品牌借此也就能够实现深入触达、高效种草、缩短转化链路。这一点在现在的新消费品牌身上就能看到，这些品牌做达人营销即可快速起势，即便是全新品牌也能很快推出爆品，拿下销量好成绩。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为优质，达人创作的内容不仅会广为传播，还会给到用户更多情绪价值，品牌借此也就能够占领消费者心智，进而逐步塑造出更为突出的品牌形象和良好的口碑。这一点你也同样能感同身受，比如你会发现不少老牌国货在达人推荐下都获得了更为亲和、年轻化的形象；曾经长期作为“小众好物”存在的品牌，在达人推荐后也很快就能走入主流视野中。</span><span></span></p>
