<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“火锅一哥”<a href="https://www.niaogebiji.com/tag/%E6%B5%B7%E5%BA%95%E6%8D%9E" target="_blank">海底捞</a>依然没能回到从前的高光时刻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月14日，海底捞披露上半年财务数据，净亏损预计为2.25亿-2.97亿元，近3亿的亏损额相较于上年同期净利润9650万元，相当难看。</span></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/194229939262fb27257a7013.81921842.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;"><span><span>△</span></span><span>图片来源：@新浪新闻微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于亏损，海底捞官方解释称由于今年3月-5月反弹的疫情导致中国内地若干餐厅停止营业或暂停堂食服务，客流量减少。还因“啄木鸟计划”导致门店数量较上年同期有所减少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但无论如何，面对亏损以及接连下挫的股价，海底捞亟需用盈利来向资本证明自己。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/381440126233b25ace1df6.64458916.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>海底捞的“误判”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>海底捞此次的亏损并不令人意外，相反，而是一个延续。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年3月，海底捞发布2021年报显示，</span><span><strong>海底捞实现收入411.1亿元，同比增长43.7%；亏损41.6亿元。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/185918397462fb2725d51957.54877817.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图片来源：@动机资本微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>如今，亏损继续，说明海底捞当年战略误判产生的负面效应还在继续。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年底海底捞还仅仅只有273家门店，2018年上市当年新开店近200家，2019新开308家。彼时正是海底捞最风光的时候。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时间到了2020年，但当时肆虐的疫情并没有阻止海底捞扩张的步伐。</span><span><strong>当年继续猛扩544家店，2021年上半年再度新增299家。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盲目扩张的第一个结果是曾经引以为傲的经营指标翻台率急剧下滑。2018年，海底捞的翻台率达到顶峰，为5.2次/天；2021年则逼近3.0次/天的盈亏平衡线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/63949814562fb27263c3995.45505198.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图片来源：@海底捞微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样的扩展虽然在疫情停摆的餐饮业让海底捞营收大涨，创历史新高，</span><span><strong>但净利润却亏40多亿，把上市后累计40多亿净利润一把亏光。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>彼时，张勇在内部对海底捞的扩张战略做出反思：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2020年6月份我判断疫情在9月份就结束，但对趋势的判断错了。</span></strong></span><span>2021年6月我进一步作出扩店的计划，现在看确实是盲目自信。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>福无双至，祸不单行。与亏损相对应的是海底捞的“杀手锏”服务也遭遇质疑。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>亏损就像第一个倒下的多米诺骨牌，过度涨价、调料人造肉、缺斤少两等问题也频繁曝光，尤其是今年年初私下给顾客贴标签的事件曝光后，海底捞似乎慢慢走到了消费者的对立面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/33795176862fb272685ea64.19952380.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图片来源：网友微博截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了自救，海底捞在2021年11月推出“啄木鸟计划”，截至2021年底，已有260家海底捞餐厅在“啄木鸟计划”下永久关闭，32家餐厅暂时停业休整。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>而从去年12月到今年6月，半年时间，海底捞门店数量又减少近80家。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>亏损与关店的结果直接反映在资本市场上，自2021年2月高点算起，海底捞市值最高跌幅达到83%，一度高达4800亿港币的市值如今已不足千亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9225241376233b25c6ec6a4.50995726.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>海底捞走不出泥潭</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主业亏损，也拖累了海底捞的海外业务，2021年海底捞新开设了22家餐厅，较2020年的36家有所减少，2022年第一季度则新开了3家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>与此同时，自救模式也顺势开启，海底捞把目光投向了“火锅”之外。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在2020年疫情期间，海底捞在官方商城上线16款到家“方便菜肴”，新鲜现做，24小时送达。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了开辟新的营收渠道，海底捞还在积极探索多品牌经营。截至2021年底，除海底捞餐厅外，还拥有若干自营餐厅如汉舍中国菜、十八汆面馆、五谷三餐及苗师兄炒鸡等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/28000477062fb2726d34484.93986202.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图片来源：网友@宁静致远微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>但是，想要靠这些子品牌出圈，似乎很难。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>时间来到2022年，海底捞自救步伐明显加快。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先是3月在深圳、武汉等地，海底捞门店摆摊售卖瓜果蔬菜，到4月部分门店开启“预定式”外卖盒饭业务，再到今年5月上线了冒菜、烧烤等菜品，其关联公司蜀海更是推出预制菜套餐团购。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/95354834662fb27274d5e34.69046775.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图片来源：@小红书网友</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，海底捞还打通线上线下流量，统筹全国各地门店与“捞点好货”自有平台，以及第三方外送平台实现对接。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/49388770562fb2727af49f5.83828875.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图片来源：网友@薅羊毛微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为此今年6月，海底捞成立社区营运事业部，专注于堂食服务之外的多元化餐饮服务创新，部署“到家+到店”核心战略，探索和满足餐饮消费新需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相较于以往在门店设立免费美甲区，到卖奶茶、增设付费手机壳DIY活动区为了提升服务，海底捞五法八门的自救行动显然是奔着赚钱盈利的目标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但从结果来看，尽管海底捞每一次的自救行动总能在网络引发巨大舆论关注，但关注度并没有转化为经济收益。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年——2021年，海底捞的餐厅营收基本都维持在96%以上，其他餐厅经营、外卖业务、调味品及食材销售、其他等业务营收占比微乎其微。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>海底捞的自救似乎并没有奏效。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16900100706233b25e365e39.91683873.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>谁杀死了“网红海底捞”？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在，我们一个明显的感受是海底捞不红了！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为“初代网红餐厅”，海底捞在餐饮行业尝到了许多“头啖汤”，无论是排队几小时、黄牛排号等令人咋舌的火热等位场面，还是海底捞的新奇服务，都使它成为整个餐饮行业的“顶流”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/192240893062fb27280331d8.47998028.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图片来源：@重庆游品微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但如今，相较于几年前因做到极致的服务、DIY蘸料屡屡引爆全网，近两年与海底捞相伴的总是亏损、关店、负面新闻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经火锅界的一哥，已然跌落神坛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>可以说“网红”身份成就了海底捞，也害了海底捞。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于网红，人们总是期待海底捞源源不断的给消费者带来惊喜，但随后的几年里海底捞给大众带来的不是惊喜，而是惊吓。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从陷入“包间监控”风波到层出不穷的食品安全问题，被网友戏称为“频频出事，屡屡道歉”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，人们开始质疑海底捞那近乎“变态”的服务是不是有些本末倒置了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，同样因服务而著称的巴奴火锅提出 “服务不过度”的口号直指海底捞的服务是打扰。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>很快，海底捞失去了“滤镜”庇护。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，网红经济大潮下，许多餐饮品牌也都学习海底捞，玩起了“网红餐厅”的概念，各种网红火锅品牌层出不穷。</span></p>
