<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E8%85%BE%E8%AE%AF%E8%A7%86%E9%A2%91" target="_blank">腾讯视频</a>又双叒叕上热搜了，不过这次是被“骂”上去的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月23日，腾讯视频宣布电视剧《<a href="https://www.niaogebiji.com/tag/%E6%A2%A6%E5%8D%8E%E5%BD%95" target="_blank">梦华录</a>》开启大结局点映礼，观众在购买价值18元、包含大结局提前观看券的“顾盼生辉”礼包后，在6月26日的点映直播中，提前收看原本定于7月3日会员收官的大结局剧情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种疑似“复活”了本来叫停的“<a href="https://www.niaogebiji.com/tag/%E8%B6%85%E5%89%8D%E7%82%B9%E6%92%AD" target="_blank">超前点播</a>”的举措，无疑点燃了网友对于长视频涨价试探的敏感神经。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消息一出，除了少部分觉得提前看而感到“开心”的粉丝，各大社交媒体上更多的是“骂声一片”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“你明明可以抢钱，还送我超前点映”“不加更的原因找到了，在这等着割我韭菜呢”“吃相太难看了”……甚至有人怒道，“看完《梦华录》，立刻卸载腾讯视频。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月23日当日，#梦华录大结局点映礼#话题的阅读量超过2.1亿次，讨论次数也超过5.7万次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/180694143962b51d4edc4849.62917750.png" title="《梦华录》“超点”，鹅被骂冤吗？" alt="《梦华录》“超点”，鹅被骂冤吗？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/微博 燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在业内看来，腾讯视频的这波操作，之所以能够“一波激起千层浪”，其实是观众对长短视频平台产品/服务价格变动不满或争议的集中表现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近一段时间，长视频平台涨价频繁，从去年年底开始，为了对抗会员增长见顶的壁垒，提高营收，爱优腾芒依次实行了会员涨价手段，且涨幅都不小，时间间隔还短。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不只长视频平台，中短视频也开始增加各种收费服务。这种情况下，观众不断为涨价的会员付费，颇有一种“摁倒葫芦起了瓢”的既视感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“最重要的是，各大平台虽然将会员价格涨上去了，但产品和配套服务不仅没有同步跟上，反倒有减少或下滑的趋势。这种对长视频长期不满的情绪下，任何平台出现任意‘风吹草动’，都会引发激烈的讨论或争议。”艾媒咨询CEO张毅表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，腾讯视频这次的激进做法，根本原因在于，长视频连年亏损下平台迫切的盈利需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对长视频平台而言，过去十多年资本的不断投入，包括内容、运营、带宽等成本在不断增加，但净利润回报的那一天却遥遥无期。在这一背景下，平台必须通过开源节流进行自救，增加利润保证盈亏平衡，甚至盈利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“因此，视频平台目前也无法顾及用户的感受，以及市场的反响，只要是不违反法律和道德的赚钱方式，它们都会去尝试。”张毅表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>影视行业相关从业者季莹则认为，腾讯视频这次的“超点”，是下了一步“臭棋”：“被骂不亏，他们太傲慢，吃相太难看，自然会引起观众的反感。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每一次会员业务（</span><span>涨价</span><span>）的调整，长视频平台都会陷入舆论漩涡。这一次直播点映的腾讯视频，到底是“宠粉”，还是“吃相难看”？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/1455856016142b5abe0ca06.17059714.jpeg" title="《梦华录》“超点”，鹅被骂冤吗？" alt="《梦华录》“超点”，鹅被骂冤吗？" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; padding-left: 12px; background-position: 0px 5px; background-repeat: no-repeat no-repeat;"><span><strong>鹅被骂了</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“<strong>我们粉丝一直想要的是‘加更’，不是‘超前点播</strong>’！”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>剧粉张录录在得知腾讯视频要提前开放《梦华录》大结局后，愤愤在话题评论区下发表不满，并呼吁粉丝对腾讯视频此举进行举报，“这所谓的‘大结局点映礼’和平台之前取消的‘超前点播’有什么不一样？换了个名头继续对外割韭菜罢了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“<strong>超前点播很大程度上就意味着‘网盘见’，盗版满天飞的结果必然给这部剧的播放量造成影响</strong>。”张录录告诉燃财经，剧粉对于腾讯视频早就积怨已久，“《梦华录》开播以来，接连主创被骂、剧情被骂、角色被骂，剧粉是一周三天看剧七天反黑，腾讯视频对此毫无作为，现在反倒想靠大结局赚一笔，对此我是很难接受。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对腾讯视频不满的，除了剧粉，还有路人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“又来？我买了会员要想提前看大结局还得掏钱，这算什么事？”电视剧迷阿猫刷微博看到“梦华录大结局点映礼”话题时，还有些小激动，觉得终于不用等到7月才看大结局了，但认真看了下腾讯视频的“游戏规则”后，阿猫表示难以理解。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“剧集超前点播服务之所以被取消因为什么，腾讯视频心里没点数嘛。消费者购买了VIP会员服务，竟然还要多付费才能享受一些权益。希望腾讯视频能记起来，自己在去年就已经宣布取消超前点播服务了，不要再搞这些花里胡哨的活动形式妄想瞒天过海。”阿猫说到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如阿猫所说，腾讯视频对于《梦华录》所采取的“大结局点映”做法，确实有欲盖弥彰之嫌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>按照腾讯视频的规则，《梦华录》大结局点映礼直播将于6月26日开启，但只有在活动期间购买了“《梦华录》大结局点映礼”礼包中任意一种礼包的用户，才有资格观看点映礼直播。而上述的“礼包”一共有3种，分别为18元的“顾盼生辉礼包”、40元的“倾盖如故礼包”，和258元的“举案齐眉礼包”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/139040610362b51d4f8c22e3.82513859.png" title="《梦华录》“超点”，鹅被骂冤吗？" alt="《梦华录》“超点”，鹅被骂冤吗？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/三种“大结局点映礼”礼包 </span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/腾讯视频 燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三种礼包除了内容有所增减外，基础的“点映礼直播观看券”和赠品“第33-40集观看券”都包含在内。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“其实多出来的VIP会员权益和周边套装都卖不了那么高价，老用户大多是冲着会员续费活动时给续的，很少会原价买视频会员。显而易见，18元那个礼包才是腾讯视频想要对观众兜售的重点。”阿猫无奈摇了摇头，“所谓大结局点映也好，观看券赠品也罢，不都是披着外衣的‘超前点播’吗？直播入口和观看资格都得付费才能享受，只是换种说辞照旧罢了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>许是腾讯视频如法炮制“超前点播”时避免落人口实，腾讯视频还开放了除“购买礼包”途径以外的获取通道，分别是积分兑换和抽奖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>按照规则，活动期间，腾讯视频VIP会员用户可使用2888会员积分兑换“顾盼生辉礼包”，此外，每位腾讯视频VIP会员用户可免费享有1次抽奖机会，参与抽奖即有机会赢取“顾盼生辉礼包”、微信红包封面等超值福利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是，要想通过这两个途径获得“顾盼生辉礼包”，并非易事。要想获取腾讯视频积分只能通过消费，消费1元便能获得10积分。对此，阿猫告诉燃财经，尽管他是腾讯视频多年VIP会员，但他的积分却为0，“我没有在腾讯视频上面消费过，VIP会员都是买联合会员权益或者电商平台有活动时，我才续费的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而抽奖获取礼包的方式则更为“听天由命”，虽然显示有2万份“顾盼生辉礼包”，但抽奖的奖品有多样，而中奖概率并没有对外公布。张录录和阿猫便什么也没抽到。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/9082723746142b5ac73c549.43567204.jpeg" title="《梦华录》“超点”，鹅被骂冤吗？" alt="《梦华录》“超点”，鹅被骂冤吗？" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; padding-left: 12px; background-position: 0px 5px; background-repeat: no-repeat no-repeat;"><span><strong>“超前点播”卷土重来</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截止发稿，</span><span>“《梦华录》大结局点映礼”直播</span><span>已有超</span><span>20</span><span>万人进行预约，若按照每位观众都购买18元档的礼包算，腾讯视频从中可获利3</span><span>6</span><span>0万元</span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>360万元对于“超前点播”模式的开创者腾讯视频来说，算不上什么。甚至可以说，“<strong>大结局点映”只是腾讯视频玩得炉火纯青的营销手段之一</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年，腾讯视频上映的耽改剧《陈情令》红极一时，在该剧临近大结局之际，腾讯视频顺势推出“超前点播看大结局”的会员福利，用户可以以6元/集价格，付费解锁剩余5集剧情，30元即可打包解锁直通大结局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便当时用户骂声四起，但腾讯视频却从中获利颇丰。<strong>据片方在庆功宴上透露，《陈情令》</strong><strong>付费点播人数达520万人次，超前付费总金额达1.56亿元</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>此后，在《陈情令》身上取得佳绩的超前点播模式，开始被各大视频平台争先模仿套用</span></strong><span>，一时间，超前点播成了爆款剧的标配。2019年末，由腾讯视频和爱奇艺拼播的热剧《庆余年》，在临近收官之际也如法炮制了“超前点播”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是，《庆余年》没有复制《陈情令》的“造富神话”，反而一举将自己送进了舆论风暴中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年12月，微博＠人民日报针对《庆余年》超前点播一事，评论道，“VIP之外设置VVIP，额外掏钱才能享受超前点播，视频网站是在制造焦虑诱发用户消费。在此前购买VIP的协议中是否有这个约定？如果没有标明，这种额外收费的行为，实际是对消费者权益的侵犯。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人民日报发声后，法律博主＠吴声威wsw也一纸诉状将爱奇艺告上法庭，认为在电视剧《庆余年》播出过程中爱奇艺公司又推出的“付费超前点播”模式，使其需要额外付费才能看最新剧集，损害了其会员权益。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在当时，针对“超前点播”是否侵犯消费者权益一事，闹得沸沸扬扬。但直至2021年，历经上海消保委和中国消费者协会“点名”后，这种模式才被陆续叫停。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年8月，上海市消保委直接发文“《扫黑风暴》超前点播搞捆绑销售，腾讯视频漠视消费者选择权”。认为消费者依法享有选择权，腾讯视频在电视剧《扫黑风暴》的超前点播中，不应规定“按顺序解锁观看”。对此，腾讯视频发布《关于优化超前点播服务的声明》，称会尽快调整解锁规则，支持用户选集解锁。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年9月，中国消费者协会发表观点称，“少一些套路，多一些真诚。视频平台VIP服务应依法合规、质价相符。”同年10月，爱奇艺、腾讯视频、优酷三大平台陆续宣布全面取消超前点播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/86193176662b51d5013c9c4.88693241.png" title="《梦华录》“超点”，鹅被骂冤吗？" alt="《梦华录》“超点”，鹅被骂冤吗？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/中国消费者协会官网 燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是，“<strong>超前点播”虽然被取消，但平台显然不想放过热播剧所汇聚的流量。企图用“新瓶装旧酒”的方式，灌醉为热播剧“上头”的观众</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年4月，在优酷热播剧《我叫赵甲第》播出接近尾声之际，观众发现，在购买VIP会员享受“VIP会员抢先看”的基础上，还得邀请5位好友助力才能解锁最新的23集和24集。而且两集的受邀助力者不可重复，这意味着要想追剧还得拉上10位好友进行助力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但显然，这种“砍一刀”的活动方式并不被用户所接受，优酷在《我叫赵甲第》后续剧集中也没沿用这种“好友助力”方式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样也因为消费者不买单，芒果在今年3月紧急撤下来原有的“《尚食》陪看直播大结局”活动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从活动名称中，不难发现，腾讯视频的“《梦华录大结局点映礼》”与芒果的“《尚食》陪看直播大结局”如出一辙。将“超前点播”包装为“直播点映”的做法，也不过是换壳重生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是，芒果在用户“付费还没回看”“还不如之前的超前点播”等系列声讨中，悄然下架此次活动，按原定安排播放剧集。相比来看，腾讯视频此次活动筹备得更为“周到”，考虑到了付费观众的回看需求，将“观看券”当作“赠品”附送。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/4756644006142b5ad08cc02.64114870.jpeg" title="《梦华录》“超点”，鹅被骂冤吗？" alt="《梦华录》“超点”，鹅被骂冤吗？" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; padding-left: 12px; background-position: 0px 5px; background-repeat: no-repeat no-repeat;"><span><strong>短视“鹅”，盈利难</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，对于长视频而言，花式“创营收”并不难理解，毕竟，2022年的营收压力尤为巨大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一边是爱奇艺号称今年内要实现“盈亏平衡”，一边是奈飞高速增长的“金身破灭”，国外流媒体纷纷准备开辟广告营收渠道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>整体影视环境遇冷的情况下，“背靠大树好乘凉”的腾讯视频和优酷也不得不“降本增效”，努力扩大营收，缩窄亏损。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而<strong>超前点播取消后，长视频平台的赚钱模式，无外乎会员付费和广告收入两种渠道</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>保证这两种模式下，平台赚到的钱能覆盖成本，甚至正向盈利，根本在于视频平台能源源不断地产出爆品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以国内视频平台的盈利模式来看，会员收入和广告收入其实是相辅相成的关系，爆款内容能吸引更多用户观看，从而实现会员收入增长；更多的观众意味着更大的流量，广告商非常倾向投这种“爆款”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《梦华录》正是这一代表，有观众统计，目前《梦华录》的广告主多达34个。据Tech星球报道，《梦华录》的一位广告代理商表示，现在《梦华录》广告位置“一位难求”，有品牌方出价500万元还要等排期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除广告收入外，《梦华录》的会员拉新收入也不会太低。《梦华录》一直位列腾讯视频热播榜、畅销榜，以及飙升榜上第一名；会员流行热度中，《梦华录》也是畅销榜第一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/22796161462b51d50cc12b5.94223648.png" title="《梦华录》“超点”，鹅被骂冤吗？" alt="《梦华录》“超点”，鹅被骂冤吗？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/腾讯视频排行榜</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如此算下来，《梦华录》的收入至少已经“回本”了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“一般来说，无论在什么档期，平台一部爆款作品都能挣回其他绝大部分非爆款的成本。”季莹对燃财经透露，“<strong>遇到《梦华录》这样的好机会，平台自然不会放过‘薅羊毛’的机会，‘薅一次’少说就有几百万元入账</strong>。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但在季莹看来，这种做法无疑是短视的。“原本《梦华录》的口碑就已经在走下坡路了，‘超点’后，万一观众觉得烂尾了，那么剧和平台会被diss得更凶猛。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“<strong>腾讯视频顶着一片‘骂声’，依旧推出了直播点映，究其原因在于‘爆品可遇不可求’</strong>。”季莹无奈道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>火星文化创始人李浩也认同这种观点，一部影视作品能否成为爆款，能够起到决定性作用的因素非常多，从IP的质量、导演和编剧的水平、整个演员班底等各种先决要素，综艺作品也是如此。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但“直播陪看”也并非毫无可取之处，毕竟如今用户的需求也越发多元、且个性化，对于优质内容，不少人也会对平台进行“催更”，甚至愿意付费，因此平台也需要提供满足用户需求的不同服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“而且正在实行分层定价的迪士尼+、Hulu等流媒体平台，以及即将要推出广告低价版月费会员的奈飞，都说明分层经营的方式是视频平台的未来。”张毅表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“不过会员分层经营的方式肯定不是一次就能确定的，无论是这次的直播点映，还是其他方式，都需要平台去不断测试，寻找平台和观众双方共赢的模式。”张毅告诉燃财经。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当然，优质内容，才是观众买单的基础</span></strong><span>。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也导致，用户对平台没有任何偏爱或忠诚度可言，在张毅看来，“用户对于爱优腾芒等平台的选择，还是跟着产品走的。比如芒果TV播了《乘风破浪》，观众就都跑去芒果了，其他作品也是同理。从根本上来说，产品品质和服务质量才是最重要的，价格因素也能因此被抵消。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>编剧、影评人汪海林也认为，如果平台能提供价值上千元订阅费的质量优质作品，那么即使会员年费几千元也有观众愿意买单。“但问题是现在观众觉得，平台的作品大部分不值得一两百元的会员费。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“接下来暑期档已公布的几部重点S+作品，目前看起来也少有卖相上佳的。”季莹谈道。因此，腾讯视频才会短视地对着《梦华录》这一只“羊”使劲薅，帮助平台利润最大化。“毕竟下一个这么挣钱的项目还要靠‘碰运气’。”</span></p>
