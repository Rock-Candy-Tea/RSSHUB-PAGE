<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近心血来潮，想从时代变迁的角度，梳理一下<a href="https://www.niaogebiji.com/tag/%E4%BA%A7%E5%93%81%E7%BB%8F%E7%90%86" target="_blank">产品经理</a>这个角色的发展进化历程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>历时一周时间，我采用可视化的方式，以timeline时间轴和重要行业变迁节点作为主要参考，完成了这张</span><span><strong><span>《图解产品经理发展进化史》</span></strong></span><strong><span>（完整图片在文章末尾）。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>绘图过程异常艰辛，中途多次想要放弃。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>感谢几天前在公众号为了点赞留言，加油打气的朋友们，没有你们，就没有这张完成稿。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>整个的制作过程中，不仅让我学会从宏大叙事的角度重新去看待这个职业，同时让我从行业变迁的重要驱动因素中，找到了产品经理这个角色的意义所在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>下面让我来讲解一下，<span>这张《图解产品经理发展进化史》</span><span>吧</span>。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span> 石器时代 </span></strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原始的石器时代，每个部落就是一个小型社会。在原始部落中，存在猎人、医生、农民、首领、先知等角色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在部落中，成员经常向首领反馈各种各样的需求。经过一段时间后首领发现，大家提的问题几乎大同小异。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>农民不知道哪里可以买到猎人的肉，猎人不知道哪里可以找到医生治病，医生不知道哪里可以买到农民的粮食。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首领将解决问题的任务交给了先知，先知首先会对不同成员进行访谈，收集整理大家的需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经过先知对需求进行总结后发现，大家的需求均为信息沟通的需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而信息沟通的需求最好的解决方案，是在领地，设立公告牌，使信息透明，满足所有人信息互通的需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，先知会对公告牌的尺寸、布设位置、信息展示架构等进行设计。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过向首领汇报详细方案后，由首领协调部落成员进行公告牌的设计与上线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这里的“先知”，就是初代的产品经理角色。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>石器时代的第一个产品，诞生了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/155614779762df07233f6bf8.59143970.jpeg" referrerpolicy="no-referrer"><br></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span> 中古时代 </span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>中古时代人口剧增，在石器时代用于信息互通的公告牌已经难以为继大家的沟通需求了。</span></span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>如何更好地展示、传播、修改信息，成了新的问题。<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>相较于石器时代原始部落的形态，中古时代以小型城邦为主，出现了如商人、工匠、贵族、艺术家、哲学家等角色。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span><span>在艺术家、哲学家的想法，贵族的资源支持下，能够高效便捷的批量生产信息的工具需求诞生了。这也是“印刷机”的雏形。</span></span></span></span></strong><span><span><span><span><br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>为了满足需求，这个“印刷机”需要满足具备哪些核心功能呢?<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>1.易操作性：每个人都能学会这类印刷机的操作方式，操作门槛不能太高；</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>2.可插入性：为了便捷沟通，机器能够在任意位置插入打印；</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>3.可模板化：希望重复印刷时，通过模板的方式能够自动操作复制打印；</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>基于这三大功能，由艺术家完成设计草图，哲学家设计主要功能，工匠完成设计图纸。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span><span>哲学家的角色，在中古时代承载了产品经理的角色定位。</span></span></span></span></strong><span><span><span><span></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>通过群策群力，中古时代的第一台个人印刷机诞生了。<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><img src="https://ngbjimg.xy599.com/19275825262df07237a4001.92081641.jpeg" referrerpolicy="no-referrer"></span></span></strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span> 电气时代 </span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>电气时代生产力大幅提升，对信息传递的效率要求更高了。如何高速、高效、随时随地的传递信息，成为了新的问题。</span></span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>在电气时代背景下，完成一项产品设计一般需要设计师、科学家、工程师、工人几种角色。<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span><span>针对上述问题，科学家基于需求场景的分析，设计了第一代电子打印机的雏形。</span></span></span></span></strong><span><span><span><span><br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>针对短距离的打印机与打印机之间的信息传达，通过无线电波实现信息的快速传输，保证传输效率。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>而针对长距离的打印机间传输，则通过有线电流的方式进行传输，保证信息的一致性。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>科学家基于通信原理设计了这种适应多场景的电子打印机雏形后，由工程师进行打印机电路设计，设计师进行打印机外观设计。<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>最终，电气时代的第一代电子打印机诞生了。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span>科学家作为电气时代的产品经理，有效推动了信息通信传输多场景覆盖，对电子打印机的诞生起到了重要的作用。</span></span></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><img src="https://ngbjimg.xy599.com/211969882162df0723b2cc77.07519385.jpeg" referrerpolicy="no-referrer"></span></span></strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span> 计算机时代 </span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>进入计算机时代，随着计算机与通信网络的普及，信息不再只是起到人与人沟通的内容，而是价值传递的载体，信息的价值大幅提高。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>如何扩大信息的利用价值，将其变为生产价值、提升价值、管理价值的工具，成为了新的问题。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span><span>与此同时，在宝洁公司诞生了“品牌经理”这一职位，整体负责“产品”这一抽象概念关联的工作，被认为是近代产品经理诞生的雏形。</span></span></span></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>宝洁作为快消企业，在概念层面上提出了与产品经理类似的岗位。而计算机时代真正意义上的产品经理诞生的，其实是技术的进步。</span></span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>1.1978年，Intel发布现代计算机的基石：8086处理器；</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>2.1981年，IBM发布并定义了计算机行业首台个人电脑（PC）；<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>3.1984年，苹果推出具备图形用户界面（GUI）产品麦金塔；</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>硬件、系统、界面的统一发展，催生了计算机行业的第一批IT产品，如操作系统、办公软件等。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span><span>计算机行业的产品经理就此诞生了，我将其定义为产品经理的岗位起步期。</span></span></span></span></strong><span><span><span><span><br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>在产品经理的岗位起步期，Title往往是项目经理、市场经理、BA（buisness-analysis）这类角色，主要负责计算机软件销售中的沟通协调、版本控制、按时交付等市场前端工作。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><img src="https://ngbjimg.xy599.com/138723665962df0723dfd529.78491758.jpeg" referrerpolicy="no-referrer"></span></span></strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span> PC互联网时代 </span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>进入PC互联网时代，硬件的发展驱动信息生产效率的提升，个人计算机开始逐渐普及，大众用户开始进入互联网。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>人口红利开始第一次进入互联网行业，面对信息爆发的机遇，如何通过互联网产品快速占领用户？这成为了新的问题。<br></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span>比尔盖茨曾经说过：“互联网（公司）未来的使用边际成本，近乎为零。”</span></span></span></strong><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>基于这样的背景下，通过免费模式获取流量，成为了众多互联网公司的核心商业模式与根基。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>在免费模式的商业逻辑下，企业目标明确为通过免费模式圈定更多用户使用，再通过用户量吸引资本投资，再用投资去反哺免费获取用户的商业闭环。<br></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span>而产品经理的工作目标，也进一步明确为提升用户体验、增长用户数。</span></span></span></strong><span><span><span><br></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>随着互联网产品研发设计流程的逐渐完善与行业各项基础设施的不断健全，但人才供不应求，导致产品经理在公司中往往承担着巨大的责任与工作内容。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>产品经理不再关注市场前端而是撤向后端，去更多关注产品开发、需求分析、产品设计等工作。<br></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>从产品设计、需求调研，再到产品规划、原型设计，以至于开发跟进、测试上线、运营推广。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span>此时的产品经理得到了极好的机会去锻炼自身全方面的专业能力，承担产品从0到1的设计工作。</span></span></span></strong><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>与此同时，PC互联网时代也诞生了几类典型应用，分别是搜索（人找信息）、门户（信息找人）、邮件（人找人）。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/98506573762df0724296c91.96319765.jpeg" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span> 移动互联网时代 </span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>从PC互联网到移动互联网时代，移动端的即时性、可移动性，迅速抢夺PC端用户时长。</span></span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>如何抢夺用户PC端迁移至移动端的时间窗口机遇，成为了新的问题。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>随着移动端终端设备的普及，海量的用户需求被激发，移动支付、移动社交、移动购物、移动娱乐等应用领域产品逐渐爆发。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span><span>产品经理承担从0到1项目的机会进一步增加，“一产成名”很容易。</span></span></span></span></strong><span><span><span><span></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>不过这样的红利期并没有持续很久。社会层面，社会格局动荡，疫情、国际关系等格局的变化，企业生存空间进一步被挤压，企业生存难度加剧。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>公司层面，随着市场残酷竞争，大部分企业会在竞争中消亡，幸存者成为寡头，采取保守战略建立护城河。<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>技术层面，能够冲破市场与社会格局颓势的新技术并未出现，一系列技术造假事件频发，“<span>技术</span><span>泡沫</span>”开始出现。<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span><span>在诸多环境变化的共同作用下，随着技术和市场的复杂化，促成了产品经理岗位的细分化。</span></span></span></span></strong><span><span><span><span></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>比如在工作内容层面，出现了策略、商业化、数据、用户体验等细分领域；产品载体层面，出现了移动端、PC端、小程序、平台端、H5等细分；行业层面上，出现了社交、工具、短视频、电商的垂直行业；<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>求职市场上开始出现类似“策略+移动端+电商行业”的垂直细分领域的产品经理招聘，如此明确的招聘需求，在PC互联网时代是难以想象的，移动互联网时代的产品经理，专业度大幅提升。<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>但同时我们也不得不面对，这样一个问题：<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span><span>2015年至今，互联网行业只诞生了两个现象级的移动端产品，抖音与拼多多。互联网产品市场的发展，似乎进入了绝望之谷。</span></span></span></span></strong><span><span><span><span><br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>这一现象的背后伴随着整个行业发展的变化。产品经理人才供需失衡，由于前期行业红利期招聘门槛过低，导致产品经理人才过剩，在移动互联网时代末期，开始有人找不到工作了。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>同时，产品经理逐渐螺丝钉化，笔者上文中提到的工作内容、载体、行业的三大细分领域，开始分化出如流程、效果等新的维度，如负责用户增长的+策略+小程序端+社交领域的产品经理。螺丝钉，越削越尖了。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>螺丝钉化加公司管理PUA化，一定会带来内耗，而内耗会导致产品经理能力成长进一步放缓。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>很多产品经理工作或者提出的需求，往往不是基于理解用户与市场后独立提出的决策，而是满足leader的需求。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span>产品经理理解市场与用户、理解需求定义产品这前后端的工作开始割裂，成为满足管理者个人意志的工具。</span></span></strong><span><span><br></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/33175360862df072470e940.29621148.jpeg" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span> web3.0时代 </span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>在移动互联网时代进入绝望之谷的同时，web3.0时代悄然到来了。web3.0、DAO、元宇宙等新概念，开始为资本市场造血，创造了很多新的路演剧本与投资标的。</span></span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span><span>面对“泡沫”、“割韭菜”等质疑声，新时代下的公司、管理者、个人是应该躬身入局，还是建立自身护城河，还是应该躺平？</span></span></span></span></strong><span><span><span><span></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>在找到自身应对策略之前，我们首先应该明确WEB3.0时代带给我们的三个信号：<br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>1.各领域寡头林立，增长见顶，亟需创造新的领域激发市场活力；</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>2.大公司垄断、双减政策的背后，预示了去中心化将成为共识；</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>3.脱虚向实是主流价值判断，避免产业空心化是大的方向。但同时，虚拟经济也要发展，也需要在虚拟经济中建立新的主流价值观。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span>基于DAO理念的社区导向性产品，将成为web3.0时代的主流。</span></span></strong><span><span><br></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>社区成员持有代表治理、投票权和实际货币价值的社区代币。用户对于社区与代币的价值存在极大共识，对产品的可用性和可行性投入是极大的。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>通过构建DAO社区产品的定位与调性，汇聚用户想法并使参与用户通过学习的方式接受并熟悉这种信息消费模式，再通过信息生产、流通、消费过程中产生的数据通过测量的方式进一步迭代与优化产品。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span>以激励为动力的产品导向型增长，将成为用户增长策略的共识。</span></span></strong><span><span><br></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Web3.0通过以激励为动力的产品导向型增长，对早期采用者、长期持有者和开发者的激励和奖励支撑了行业的指数级增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过代币发布、空投、社区福利的发放，早期使用、推荐和社区参与等任务都会得到“虚拟货币”的奖励。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>考虑到经济机会和社区参与，这种激励方式引发产品的病毒式传播。</span><span><span><span><span><br></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时通过开发者授权，帮助社区<span>可以构建社区所需的项目，</span>对平台来说是一个双赢的局面，通过低成本的方式增加公司和代币的价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像移动互联网时代爆发的用户需求一样，在web3.0时代同样存在支付、娱乐、购物、社区等垂直业务方向，并已经诞生了一些明星产品。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>随着政策的放开与产品管理理念、信息互通协议、内容消费观念的共识建立，越来越多的web3.0应用将走进大众视野，引领新的时代浪潮。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让我们拭目以待吧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/61605985162df0724aa5246.59542531.jpeg" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span> X时代 </span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>时光流转，斗转星移。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>时代与行业的发展下，产品经理岗位的特性似乎一直在变迁，但又好像什么都没变：</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span><span><span>无论是远古还是未来，产品经理永远是那个面对复杂的时代环境灵活切换看待问题视角，通过深远的逻辑思维链条认知因果关系，认清事物发展规律及长期趋势，最终通过创造，改变世界的那批人。</span></span></span></span></strong><span><span><span><span></span></span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>社会在变，行业在变，市场也在变。而唯一不变的，是认知。<br></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>通过从古至今产品经理角色、职能、特性转变的梳理，我们能够发现产品经理工作的变化其实也伴随着服务对象及服务范围的变化。<br></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>从市场导向，到产品功能导向，再到产品理念甚至产品哲学导向，产品经理的思路和视野也在发生着巨大的跃迁。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>即使你现在身处格子间，遭受着不公的待遇，做着螺丝钉的工作，但请永远记得建立起面对复杂多变的世界，能够冷静分析并认清规律与趋势的认知能力。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span>这样的认知能力能够帮助我们冲破愚昧之谷，在面对巨大的行业与市场机遇到来之时，能够从容的抛出大网，圈住它们，改变自身命运。</span></span></strong><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这也是这张图解《产品经理发展进化史》，能够给大家带来的最大价值。</span></span></p>
