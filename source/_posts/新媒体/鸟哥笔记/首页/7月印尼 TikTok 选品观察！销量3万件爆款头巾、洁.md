                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">GoodSpy专注于跨境广告创意与选品参考。覆盖Meta、lnstagram、TikTok、YouTube 等热门社交媒体广告投放数据，通过全球热卖商品、大卖店铺找到爆量素材。让<a href="https://www.niaogebiji.com/tag/%E8%B7%A8%E5%A2%83%E7%94%B5%E5%95%86" target="_blank">跨境电商</a>品牌/商家、TK卖家、优化师等人群更高效地获取电商素材创意灵感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">TikTok 海外电商业务再次发力，据了解，TikTok 正在测试在印尼市场的 App 主界面内新增一个“Shop”选项。从 Watchful.ai 发布的“Shop”选项截图显示，用户通过 Shop 选项可以更快、更便捷地进入商城浏览和购买各类商品。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> <img src="https://qnssl.niaogebiji.com/f7bfbdc89533925219220db40d728c76.png" alt width="1080" height="888" referrerpolicy="no-referrer"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><a href="https://www.niaogebiji.com/tag/TikTok%20Shop" target="_blank">TikTok Shop</a> 界面 ，来源：Watchful.ai</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，据字节某消息人士称：2023年 TikTok 电商交易总额目标是230亿美元，同比增长10倍之多。虽然近期有传其电商业务在欧美的进程受阻，但 TikTok 依然在按照自己的节奏进行布局，尤其是在 GMV 贡献达70%的东南亚市场，因此仍有不少商家对其发展持看好态度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据 GoodSpy 数据显示，印尼小店的商品量目前已突破160万。各品类商品占比来看，服饰、鞋履和珠宝类占比最高，约占37%。其次是美容个护类约占20%，家庭和厨房类占比约9%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/7077dcdb25617c33f5b3497451c6807f.png" width="554" height="401" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">TikTok各品类占比，来源：GoodSpy</h4>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">01、服装、鞋履和珠宝</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">近3天服装类商品销量榜来看，女装类商品销量表现较好。Top10的商品中，有6款是女装，包括牛仔裤、打底裤、穆斯林头巾、宽松款休闲裤、短款T恤等热销单品。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其它产品来看，销量最高的是一款连帽衬衫，最近三天累计销量超过7千件。销量第二的是一款“路飞”印花T恤，近三天销量超过4千件。此外热销服装商品还有儿童服装、卫衣等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/be689bcc06cf7a19030c200bcea95110.png" width="1080" height="713" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">服装、鞋履和珠宝类近3天销量榜，来源：GoodSpy</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">穆斯林头巾-hijab 在 TikTok 上也有着非常高的流量。其中几个热门的标签，如 #hijab 累计播放量超过119亿，#hijabchallenge 播放量超过3千万，可见头巾的受欢迎程度。此外，通过短视频来带货在印尼逐渐流行，相关商品有不错的转化率。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/2fda8ebcdc59df89f2254390a254bad4.png" width="554" height="341" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">来源：TikTok </h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据 GoodSpy 数据显示，@mexshamall 推出的穆斯林头巾近30天累计销量将近3万件，其中销量最高的一天出单量突破3千件。该店铺在TikTok上的粉丝7.4万，累计获赞量260万。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">夏日T恤也是近期服饰类畅销品之一，NCDY Crop Top Basic Over Size 是一款女性夏日短款T恤，款式时尚且材料舒适，十分受消费者欢迎。从销量趋势来看，该款商品仍处于畅销期，7月4日出单量达到1千多。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">@BIOAQUA 推出的两款美容护理类产品都有不错的销量表现。其中 Bioaqua sheet mask 面膜从5月份开始日均销量能稳定破千。据 GoodSpy 数据显示，该款面膜近30天新增销量近4万件。除了短视频和直播带货外，该商品投放的广告也有较高的展现量，其中的一条广告播放量16万，点赞9千多。</p>
