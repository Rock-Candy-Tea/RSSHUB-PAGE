                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前，TikTok for Business广告管理平台提供了四种出价方式。在广告组层级，广告主需要选择出价方式和出价策略。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上次YinoLink易诺为大家介绍了TikTok for Business广告预算及出价方式，这次我们来讲一讲<a href="https://www.niaogebiji.com/tag/%E7%AB%9E%E4%BB%B7%E7%AD%96%E7%95%A5" target="_blank">竞价策略</a>。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">一、TikTok for Business广告出价策略 </h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">广告主需要在广告组的“竞价和优化Bidding & Optimization”模块选择竞价策略。竞价策略将决定系统的单次成效费用管理方式、预算支出方式以及广告投放方式。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" align="center"><img src="https://qnssl.niaogebiji.com/934ee8ca968abfb180357cd36a813cdf.png" alt width="2160" height="1440" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前，<a href="https://www.niaogebiji.com/tag/TikTok%E5%B9%BF%E5%91%8A" target="_blank">TikTok广告</a>管理平台支持竞价上限、控制成本上限、最低成本这三个竞价策略。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>1. <!--[endif]-->竞价上限Bid Cap</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">广告主需要输入愿意给出的最高单次成效费用，系统将自动根据该费用对广告投放进行优化。选择此策略可以将平均单次成效费用控制在设置的出价阈值以下。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">支持的推广目标：访问量、覆盖人数、视频播放量、应用安装量和转化量</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">- 当广告组的优化目标设置为点击时，竞价上限只适用于应用安装量目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">- 当优化目标设置为点击时，竞价上限只适用于转化量目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">支持的计费方式：CPC、CPM、CPV</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>2. <!--[endif]-->控制成本上限Cost Cap</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">采用控制成本上限的竞价策略，系统控制平均单次成效费用尽可能符合广告主的设置。因此，单次成效费用有时会在竞价金额上下范围内波动。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">支持的推广目标：应用安装量、转化量和线索收集广告</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">- 当优化目标设置为转化量时，控制成本上限只适用于转化量推广系列。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">支持的计费方式：oCPM </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>3. <!--[endif]-->最低成本Lowest Cost</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当采用最低成本的竞价策略时，广告主无需输入出价，系统会根据广告组的预算，以最低的单次成效费用生成尽可能多的成效。无论竞价环境竞争多么激烈，最低成本竞价策略在任何时候都可以花费预算。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最低成本竞价策略能够避免预算每天的波动，大幅节省为了优化效果而手动调整出价所花费的时间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">支持的推广目标：访问量、覆盖人数、视频播放量、应用安装量、转化量、线索收集广告和目录促销</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">支持的计费方式：CPC、CPM、CPV、oCPM </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">二、三种竞价策略对比及使用场景 </h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同的竞价策略适用于不同的场景和推广目标，下表是三种竞价策略的对比，广告主应根据需求选择适合的竞价策略。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" align="center"><img src="https://qnssl.niaogebiji.com/c9ef074d623d63d1f94e1cbcfe4dd8c8.png" alt width="3750" height="1875" referrerpolicy="no-referrer"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>1. <!--[endif]-->竞价上限、控制成本上限竞价策略使用场景 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果主要目的是为了控制单次成效费用，可以考虑使用竞价上限或控制成本上限的竞价策略。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>2. <!--[endif]-->最低成本竞价策略使用场景 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果主要目的是为了生成尽可能多的成效并全部花完广告预算，则应选择最低成本的竞价策略。例如某个独立站想要在父亲节前花完活动促销广告组的预算，或是某个新APP发行时可以通过最低成本竞价策略获取尽可能多的下载量。 </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">三、竞价策略对应最优设置</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>1. <!--[endif]-->竞价上限、控制成本上限竞价策略</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于选择竞价上限、控制成本上限作为竞价策略的广告，TikTok for Business官方广告代理商YinoLink易诺建议在推广系列层级设置预算为无限制，在广告组层级设置预算至少为出价金额的50倍。如果访问量和费用出现波动且低于预期，可以尝试切换最低成本竞价策略。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>2. <!--[endif]-->最低成本竞价策略 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">选择最低成本竞价策略的广告同样建议在推广系列层级设置预算为无限制，在广告组层级设置能够满足至少50次转化的预算。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在度过学习阶段后，如果要增加预算，最多增加50%的广告组预算。广告预算的增加往往意味着单次成效费用的增加，系统会先找到最低的单次成效费用再增加出价金额，从而达到该预算可实现的最大成效。 </p>
<p class="MsoNormal" style="line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; margin: 25px 0px 20px;">需要注意的是，在两次预算调整之间需确保至少间隔一天，且完成了50次转化。<span style="mso-spacerun: 'yes'; font-family: 微软雅黑; mso-bidi-font-family: 'Times New Roman'; font-size: 10.5000pt; mso-font-kerning: 1.0000pt;"></span></p>                        <div style="height: 1px"></div>
