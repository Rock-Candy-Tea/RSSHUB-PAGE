                        <p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">6月9日，B站公布了2022年第一季度财报。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">除了用户、营收等内容以外，有一点非常值得关注——<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">Story-Mode竖屏模式</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">财报显示，这个新模式为B站带来了新的播放增量，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">占全站总播放量超20%</span>，呈现持续上升趋势。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p8.itc.cn/images01/20220720/ed8061f704e74786bdbbb32ce6d48344.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-B站 2022年第一季度财报</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">这个模式为B站用户增加了平台的使用空间，在横屏模式播放量正常增长的基础上，竖屏模式的播放量对于B站而言就属于<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">纯量增长</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">再者，近年来短视频平台的崛起带动了众多短视频博主，通常来说，一个自媒体博主不会只拘泥于单个平台发展，所以B站Story-Mode竖屏模式的启动也是在<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">向短视频平台的创作者招手</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">但是，短视频UP主的是否能得到B站用户的认可？</p>
<h2 style="border: 0px; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; font-size: 18px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #212629; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; line-height: 30px; text-align: left;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">短视频达人也能在B站获得高增长</span></h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">「那就叫王师傅吧」自2020年入驻B站起发布的均是竖屏形式的作品，但那时候B站并没有开设竖屏视频的专属通道。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">一直以来UP主作品主要时长均<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">小于3分钟</span>，是典型的短视频UP主，所以对于UP主而言，自己的作品又是竖屏，时长又短，在B站这样中长视频为主的内容平台其实稍显格格不入。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p7.itc.cn/images01/20220720/6777a004b06d4471b9107130022503ac.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">在这样的平台背景下，我们聚合UP主2020年的作品播放量，数据显示UP主的平均水平保持在<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">32w</span>，如下所示。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p3.itc.cn/images01/20220720/298286b0398a46d0b8dc537dacb16035.png" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">自Story-Mode竖屏模式上线以来，UP主的作品数据迎来一次井喷。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">6月18日，UP主发布作品《如何给他人留下深刻印象》，视频播放量截至目前已经<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">高达936.01</span>，仅仅只有1分半钟的作品竟能带来如此高的传播率，UP主的平均数据水平也上升到了<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">233.3w</span>，而本支作品在此基础上还高出<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">300%</span>。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p0.itc.cn/images01/20220720/3abcf465b27a4959b183f9e4efe7fe89.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">在视频中UP主围绕着主题“如何给他人留下深刻印象”展开，用一种平淡、冷静的表现形式去做常人认为很尴尬的事，再搭配上慵懒声线的配音解说，在弹幕热词中占比最高的是“优雅”一词，观众认为看他的视频能感觉到一种“离谱中又有点合理”的体验，并称之为“优雅的艺术”。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p2.itc.cn/images01/20220720/12ad864179b741eca8f599060ba659e3.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">作品发布后<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">一小时内最高能达到34.4w的播放增量</span>，并且在后续的趋势中持续有规律的出现流量峰值，助长着作品的播放增长。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p6.itc.cn/images01/20220720/c70092af30ee4139948a073bfcfe13b0.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">竖屏模式带给短视频UP主的是更合理的观看体验，相比于往期用横屏的形式看竖屏视频的体验来说，Story-Mode竖屏模式的推出是<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">提高竖屏视频沉浸式观感的载物</span>。</p>
<h2 style="border: 0px; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; font-size: 18px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #212629; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; line-height: 30px; text-align: left;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">B站原生UP主又该如何融入？</span></h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">但上文我们也提到了，B站竖屏模式的开启带来的是纯粹的播放增量，那就意味着，B站原生的UP主们也可以加入到竖屏模式中去，为作品播放增加使用场景，吸纳更多钟情于竖屏模式的用户观看。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">但是B站原生UP主需要做到哪些调整去融入竖屏模式呢？</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">以下总结最重要的一点就是：不论是上传横屏还是竖屏视频到Story-Mode中，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">视频时长的缩减是非常必要的</span>。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">举个例子，UP主「nya酱的一生」就是B站的原生UP主，2016年年初就加入B站，更是2018年评选的百大UP主，她的作品趣味性强，通常以vlog、座谈的形式发布，个人风格明显，这也就导致了粉丝粘性更深，所以她的作品时长均是保持在<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">10-20分钟</span>的中视频为主。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p6.itc.cn/images01/20220720/1c364db6276344fc977f335c42f7bf33.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">但在6月17日，其发布了一支时长为<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">4分25秒</span>的作品《如果家庭带给你的不是幸福。。》，并且是一支竖屏视频，作品甚至登上了全站排行榜最高74名的位置。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">突破了以往中视频时长的“舒适圈”，作品形式仍然以座谈、聊天的方式去展现，在维持原有粉丝观看倾向的基础上，尝试并且融入到Story-Mode竖屏模式中去，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">播放量甚至高于前两期的横屏作品</span>。</p>
