<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/VR" target="_blank">VR</a>的终极形态什么时候能到来？是十年还是二十年？这项技术甫一出世就挑战着虚拟与真实的边界，借用技术手段，创造出一个沉浸式极强的虚拟空间，根植现实又超越现实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于终极形态的问题，从业者在思索，当不断有产品推向市场后，消费者也在追问。制作好莱坞大片的特效比制造一台真实产品容易得多，但也引导着人们对VR技术的未来产生各种充满创造力的设想，并努力将其变成现实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回归到现实中的产品本身，在技术的驱动下，行业不断传来进步的好消息。从硬件设备上，市场上的VR一体机，已经可以做到佩戴舒适；VR常被人诟病的晕动症也得到进一步缓解优化，6Dof成为了主流设备的标配；从内容层面看，类似音游《节奏光剑》 、3A级的《半衰期》等游戏成功出圈，各大影展为VR内容专门开设特别展，创作生态逐渐成形。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好消息不止于此，媒体关于苹果的“谍报”不断，据消息其设备将融合AR、VR，且使用超高分辨率的8K显示屏。国内市场方面，YVR等本土黑马科技创企陆续发布新品，带动VR核心科技、消费级市场、内容生态的持续向前。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>VR行业重新热闹了起来，这一次其内在的硬件技术迭代到底走到了哪里？又会推动产品本身、用户体验、行业发展行至何处？在竞争愈发白热化的市场，新品还能如何寻找差异化？VR未来已来，但有太多谜题需要解答。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81142426261420d300e45a3.29058312.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>技术的胜利</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这一次的VR崛起和几年前充斥着泡沫的行业有所不同。</span></strong></span><span>彼时真正做技术的厂商非常少，更多的是产品组装、炒概念、资本玩法。而去年以来重新被聚光灯照亮的“新生代”VR，最核心的突破是——技术。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就拿光学方案来说，光学方案是决定VR设备成像质量的核心环节，过去其发展的缓慢也是VR行业增长的瓶颈之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在许多行研报告和厂商秀肌肉的发布中，Pancake光学方案常常被提及。</span><span><strong>比如YVR就是选择的该技术路线——其7月发布的新一代VR眼镜YVR 2是全球首个正式发售的采用Pancake超短焦光学方案的VR一体机，机身厚度得以压缩至42mm，带来了视野范围内全域清晰的视觉体验。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/711404645630372a4a6ec39.62751127.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Pancake方案究竟是什么？对产品本身和用户体验又会带来什么变化？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>拆解一下VR设备，用户眼前的图像是在屏幕上显示，再通过透镜射入眼睛的。当前VR光学模组普遍包含：显示</span></strong></span><span>（光和像）</span><span><strong><span>＋透镜两大部分，其中显示部分技术较为成熟，可以做到很薄，制约当前VR轻量化发展的主要因素在于透镜。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最初一代商业化的VR设备采用的传统光学方案是非球面透镜，视场角FOV与尺寸之间正相关，缺点非常明显，</span><span><strong>这类设备的尺寸通常大且笨重，显然不是最佳方案。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了使VR设备变得更适合佩戴，许多厂商会采用菲涅尔透镜。</span><span><strong>这种镜片的一面刻录了由小到大的同心圆，其效果是焦距短、比一般透镜的材料用量更少、重量与体积更小，因此被许多主流一体机采用。</strong></span><span>但是在轻薄化的趋势下，厂商还需要继续为设备减重。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Pancake方案应运而生。此种方案基于偏振的光学折叠技术，图像源进入半反半透功能的镜片分束镜之后，光线在偏振反射膜与分束镜之间折返一次，从偏振反射膜射出，通过其他光学件后进入人眼。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/939987451630372a558e683.46008032.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>原理示意图</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>左是传统透镜，右是基于偏振的光学折叠技术</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：Meta</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>落到应用上，Pancake方案的光学模组相比传统非球面和菲涅尔透镜方案，具备清晰、纤薄两大核心优势。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据了解，此次YVR 2采用双镜片折叠光路设计，独特曲面贴膜技术。整体克服了菲涅尔光学固有的视野边缘模糊和畸变现象，有效减少了边缘眩光等问题，其光学解析能力相对于菲涅尔光学提升了50%。值得一提的是，YVR 2的光学镜组透过率达到了19%，而据悉目前业内平均水平为13-16%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，YVR给VR设备“减负”的理想从未消失，YVR 2纤薄的机体也实现了VR产品体验的飞越。据YVR数据，其新一代产品YVR2的眼镜光机仅有20mm，相对传统菲涅尔镜片光机厚度足足减少50%，整机厚度更是低至42mm。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/326000144630372a5d96614.72413896.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>YVR 2 机身</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，Pancake方案并非完美，比如其造价高，核心部件光学膜的成本也比一般透镜高出不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>从技术迭代的角度说，Pancake方案是VR行业在光学领域的一个显著进步。</span></strong></span><span>虽然其在生产工艺、组装、整体性能上都有难点，但许多厂商趋之若鹜，价值不言而喻。据外媒报道，苹果已经申请了关于Pancake光学方案的专利，并计划将在2023年出推出新一代MR设备，新产品将定位为高端设备，预计价格高达3000美元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>在中国，抢在巨头前面将Pancake方案率先落地的是YVR这样的本土品牌，这无疑给国内VR行业打了一针强心剂。意味着，由YVR定义的高端VR一体机设备，在部分产品技术路线上已经实现对外资大厂的弯道超车。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在技术上的迭代，最终实现的终极目标就是在</span><span><strong>用户体验上的飞跃式革新，</strong></span><span>而完善用户体验，既需要在技术层面的“硬标准”，也需要在产品设计层面的“软实力”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在设计方面，YVR 2摒弃了传统VR设备“千篇一律”的设计方式。设备灵感来自于飞行员太阳镜，产品本身更具有辨识度和科技感。另外，从人体工学角度出发，YVR2采用了3：2后置配重设计 ，降低前端负重感，减少压面感；同时，为了更加适配亚洲人的面部轮廓，YVR 2配备了独特的鼻托设计，完美隙合鼻梁构造，有效减少外部光源干扰。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1543809151630372a6712b40.32013013.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>纤薄机身、配重设计、鼻托结构、后减震垫等让产品佩戴更舒适</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>目前，YVR 2是国内唯一同时获得全球工业设计顶级奖项“德国红点奖”和“德国iF奖”的VR一体机。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对VR产品而言，其所带来的沉浸感和舒适度是</span><span><strong>硬件产品、软件算法、内容生态</strong></span><span>三方面综合的结果，缺一不可，这是个和手机行业相似的逻辑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了上文所述硬件光学技术，软件算法也是影响VR产品体验的核心因素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了带来更加便捷、自然的交互体验，YVR自研了手部6DoF算法，拥有超低延迟、超高准确度、超强鲁棒性、超广追踪范围特性，手柄追踪空间范围达到了270°*210°。而其自研的头部6DoF算法，则能充分满足虚拟空间中行走、跳跃、下蹲等体验。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1139548473630372a6e108b2.09449098.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>YVR 2 产品图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直以来，VR内容生态建设始终是行业的核心痛点。海外大厂的普遍做法是自研+引进扶持，比如Meta一边出软件一边“平台化”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据YVR创始人黄锋介绍，现阶段YVR还是以“精品化引入战略”来构建内容生态，涵盖游戏娱乐、运动健康、艺术人文、教育探索等方面。目前，YVR已引入《亚利桑那阳光》 、《动感音旅》、 《战箭英雄》等多款高分游戏大作，《灵感画笔》、《墨之韵》、《中华技艺：陶罐》等综合类App。此外，专业健身类App《Punch》也已上线。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1240211709630372a743d219.62883130.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>VR健身应用《Punch》</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>相较投入巨大资金自研，硬件厂商做生态无疑对全行业而言能有更好的提振效应。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213179696861420d317ab820.36069667.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>市场的窗口</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Gartner曲线其实已经很精准地预言了VR行业的坎坷路程，从技术萌芽期，到2017年前后的期望膨胀期，又在“捧杀”之后泡沫破碎落入低谷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近两年，随着苹果等大厂的XR设备消息不断、Meta对元宇宙的全情投入、大额投融资事件频频，VR终于再次回到聚光灯下。关于如何让“元宇宙”概念落地，众说纷纭，并没有所谓的标准答案。但具有确定性的是，以VR为代表的XR技术是通往“元宇宙”一把不可或缺的钥匙。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从数据中可以直观看到VR行业的再次爆发：据第三方调研机构IDC，2021年全球AR/VR头显出货量达到1123万台，同比增长92.1%，其中VR出货量达到1095万台，</span><span><strong>预测2022年全球 VR 头显出货将达到 1573 万台，同比增长43.6%。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/807899467630372a7a62b77.98559541.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>技术成熟曲线图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>对于新品牌来说，当前无疑是个窗口期。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>教育市场的工作已经由先发品牌初步完成，消费者对VR产品有了一定认知。而且随着行业的成长，相关技术已经有成熟落地，供应链体系和内容生态都逐步完善。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>诸如YVR一样有潜力的本土VR品牌诞生，不仅拥有全栈式技术自研能力，领先的内容生态，还在全球范围内，与包括高通、和硕、3M、BYD、JDI、欧菲光等公司进行合作，为用户打造更高水平、更高标准的VR科技产品而持续努力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>YVR虽是新品牌，去年才发布第一代VR眼镜，但是其技术和经验储备从2014年就已开始。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>创始人黄锋告诉「深响」，团队接触VR技术是在2014年，并在2015年就尝试了VR线下体验店模式，做过蛋椅形态的设备，从中得到了用户反馈并看到了行业痛点，包括眩晕以及刷新率、清晰度的不足。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年是YVR决定做VR眼镜的关键之年。</span><span><strong>“我的判断是VR将会成为下一代计算平台，这是一个面向未来的行业。”</strong></span><span>黄锋说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那一年的VR已经跌落风口，高光不在，许多VR行业初创企业被淘汰。黄锋找到了一批业内人士交流，看到了他们对这个行业仍然有信心和热情，于是大家一拍即合，在很短的时间内就组建起了团队。是团队的丰富经验和共同志向让YVR少走了很多弯路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2021年以来，YVR在一年内发布了两款产品，并实现技术迭代。</span></strong></span><span>YVR 1搭载了自研6DoF追踪算法，用户在使用时不再被困在原地，操作更加自由、沉浸，带来了交互体验的全新升级；YVR 2则以颠覆性光学方案，让外观更纤薄，佩戴不眩晕，成功完成佩戴舒适度的全面突破。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>快节奏背后的底气是一支具有战斗力、对VR行业保持热忱的团队。</span></strong></span><span>他们来自硅谷顶尖人工智能算法团队、国内科技巨头和VR行业一线企业，均拥有近10年VR行业的沉淀，是国内头部的独立VR产品研发团队之一，拥有多项专利。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正是因为拥有具备丰富从业经验和系统化的研发团队，YVR才能够准确把控行业技术发展趋势，并推出全球首款正式发售的Pancake VR一体机——YVR 2。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，将产品推向市场，YVR还有一段路要走。</span><span><strong>如何与市面上其他同类产品拉开差距是不得不考虑的问题。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>YVR 2的发布会上，“高端”二字出现频率很高。横向对比市面上其他主流一体机的价格，YVR 2可以说并不便宜。这款产品定位于</span><span><strong>高端消费级VR一体机。</strong></span><span>这与产品本身坚持核心科技自研、对细节和品质的把控相关，也与YVR的目标群体相关。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据黄锋介绍，YVR的目标用户是“喜爱尖端科技产品、追求品质和更好体验的消费者。” 这与YVR“让更多人玩出梦想”的品牌精神和使命一脉相承。YVR始终坚持“为用户带来美好体验”的产品理念，不停研发、升级、打磨产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“消费群体、产品定位的分化，是每一个行业从新兴走向成熟的必经之路。VR最终和手机类似，肯定会形成不同定位的品牌体系来满足不同消费者的需要。”黄锋说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时， 线下门店是硬件厂商接触消费者最重要的渠道之一，苹果零售店吸引着大批果粉，互联网手机起家的小米也在发力扩张线下门店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于VR此类重体验的产品以及YVR这样的新兴品牌来说，线下更是尤为重要。</span><span><strong>YVR计划拓展线下品牌体验店，不仅为用户带去深切的优质体验，也补全了自身“线上+线下”的消费生态，构建全渠道销售体系。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>日前，YVR国内首家线下零售体验店亮相上海，</span></strong></span><span>集VR眼镜体验专区、无线串流大屏体验区、MRC体验区、观影区、休息区等五大功能区于一体。据悉，YVR体验店开业首日就迎来了客流高潮，单日进店近1000人次，服务人次近300人，成为家庭娱乐全新去处。</span><span></span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;"><span>YVR线下体验店</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>线下店不仅能为用户带去真实体验，占领用户心智，而且也为YVR自身版图添上了最后一块拼图。此种模式在国内VR行业尚未得到大规模验证，YVR的先行动作值得关注。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>VR行业元年复元年，一边是质疑声，一边是消费者与行业对更好的设备、更丰富的内容的期盼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在多年来的飞速发展中，</span><span><strong>YVR与行业一同成长，凭借敏锐的市场洞察与研发团队的前沿技术积淀，用几年时间走完了很多企业十余年的路，成为国产VR领域的“破局者”。</strong></span><span>许多坑已经踩过，许多弯路已经走过，再次出发的VR行业现在才真的走入高增长期。</span></p>
