<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong>关于人：做人很累，不要emo</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/197756626862d104f606a683.93817076.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，学会自洽，人与人之间的认知和能力差是巨大的，努力做到自我上限就好。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之前，我在奥美和淘宝时，遇到过从认知到能力层面远超自己的人，那种碾压会让你觉得无法超越，很早我就明白了，什么叫天外有天、人外有人。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，有时候我们真的不用过于苛责自己。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个时代，我觉得大家<strong>做好自己能力和认知内的事就挺好</strong>，千万不要渴望十项全能，德智体美劳全面发展。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，我反对摆烂摸鱼。我只是说，希望大家做事时，发挥自己能力上限，不要为难自己。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如写PPT，让自己觉得足够好看，足够漂亮就可以了，再比如做Excel，先做对，再做好，其次是表格内容清晰，结论总结清楚。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>千万不要“内卷”，往后“内卷”的意义也不大。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以日本为例，日本“内卷”的时代已经过去，早期日本因为“内卷”出现过很多过劳死。现在的日本，很多人的主要工作是沟通，所以未来你会发现，沟通才是工作量最大的事，反而需要工作的事情会被大量的机器或算法工具取代。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其次，不要假装努力，不要年华虚度，一定要系统化学习。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘润老师说，你的顿悟可能只是别人的基本功。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他说的很对，但有些让人焦虑，不过没办法，人与人之间差别确实挺大，所以认命+自得其乐是个比较好的状态。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>千万别让自己的欲望大于能力，这是一件非常恐怖的事。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们大多数人总觉得自己很努力，每天奔跑在学习的道路上，可是你去看看自己刷抖音的时长就会明白，真正学习的时间并不长。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，有些人通过微信群的信息来学习，不过我认为群的信息是碎片化的，它无法构成完整性、系统性的知识。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>学习一定是系统化的。比如有时间多看看书，书籍本身就是系统化去传授某个领域的知识。最近在做直播时，我始终为粉丝推荐几本书，然后我带着他们把一本本书全读完。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实大部分人在大学毕业后，已不再系统学习任何一个领域的知识，大家都是在工作中将碎片化的知识组合拼接在一起，所以大家的知识框架基本上是比较碎片化的，而读书是将碎片化学习转变为系统化学习的一种有效方法。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>最后，时代的机遇造就人生巅峰，低学历高收入的时代正一去不复返。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多时候人一辈子抓住一次机遇就足以改变人生，但是更现实的问题是，面对绝大部分的人生机遇，我们都准备不足。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/173827674262d104f655a793.50304696.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>分享一个我在微博写的段子：<strong>如何走向人生巅峰？</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>60后投身改革开放，70后填补某项行业空白，80后拥抱股市、房地产、互联网，90后创造新商业模式，那么00后呢？00后只能等一个有缘人（调侃）。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在过去的时代里知识总是“穷且丑”，他们的知识极为丰富，专业理论也很扎实，但是他们往往收入不高，绝大部分知识分子是“穷且丑”的样子。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反之，开跑车的、玩游艇的基本上都是学历不高的网红，亦或是有些做虚拟产品&货币的炫富党。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但现在这个时代正在发生转变，这种“低学历高收入”时代即将结束。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>东方甄选其实就是个风向标，为什么这样说呢？大家可以去观察带货榜TOP20的直播间，当带货榜TOP1（基本在TOP5）的东方甄选讲“诗词歌赋”，并倡导大家理性消费时，其他的直播间却以“喊麦式”“嘶吼式”的引导消费者快速下单，明显可以感受到高知阶层对老铁经济的碾压已经开始了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以从整个直播圈或从品牌圈去看，</span><strong><span>会发现“低学历高收入”时代已经开始发生一些转型</span></strong><span>。未来两三年，整个品牌端会完成一个真正的升级。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总结一句话：<strong>新经济新商业模式往往是社会边缘人群来推动的</strong>。当商业模式红利结束之后，才是社会主流人群进场的时候。这里的社会主流人群是指主发声人群，也就是高知人群。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不难看出，东方甄选的直播和以往不太一样。我们大致可以把直播分为两类：促销直播和营销直播。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>促销直播的三板斧：“限时、限量、限价”</span></strong><span>，任何一个直播间只要是玩这个逻辑，就是促销直播间。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>营销直播主要是：“有趣、有料、有用”</span></strong><span>，以东方甄选为例，有趣就是唱歌跳舞和诗词歌赋，有料是每款产品讲出它的价值，有用是直播时学会或得到了某种东西。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong>2022年，作为个体的我们又该怎么办？</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>分享几点我的思考：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一，如果你是老板，一定要聚焦核心，做自己最拿手的东西，不要挑战知识盲区。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这个“老板都要抛头露脸，才能把企业做好”的时代，千万不要去挑战自己的知识盲区或能力盲区，这将会承担很大风险。因为现在正处于一个大环境经济转型期，转型期一定要干自己最擅长的事，不要干自己不擅长的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二，如果你是创业者，降本提效，不要烧钱买流量，不要疯狂刷单，现金流最重要。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年，你可以通过烧钱买流量、疯狂刷单快速起个账号。但是你算一算，哪怕单条成本500块钱，至少也需要两三千万的启动资金。回到今年，基本上新消费品牌融资都非常困难。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三，如果是公司的高管，不要随便跳槽，不要随便开人，保持学习，与人为善。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，大家可以看到各种“00后整顿职场”，面对以前PUA的老板或者不合法的开除行为，00后开始学习劳动法。而且，你怎么知道人家00后以后不能做你上司呢？所以这时候一定要与人为善，对员工好一点。同时自己要保持学习。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第四，如果你是普通员工，不要随便跳槽，努力工作，但不要内卷，有时间就学点新技能，顺便搞点副业。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>千万不要把闲暇时间，用来刷短视频、玩游戏，要去多了解些“风向”，因为现在只要知道一些风向还是能搞点钱出来的。其次，如果你的爱好不能让你去赚钱的话，我觉得那就不是爱好，只是一个多巴胺刺激。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里利用宏观经济PEST分析法来展开说说，是从政策（P）、经济（E）、社会（S）和科技（T）4个角度来判断我们个人的命运。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/73604001662d104f6e7fae7.19101905.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1. 看政策：</span></strong></span><span>二十大以后会有新政策出台，政策落实差不多要到年底。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家可以通过年底政策的落实情况看出国家重点关注的新基建到底是什么？判断出哪些行业会爆发？哪些赛道是需要先发制人亦或需要厚积薄发？所以，一定要看政策。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6446348062d104f7b67e41.73036265.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2. 看经济：</span></strong></span><span>关注政府门户网站或数据网站，了解你所在行业的数据报告。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如：近两年，你所在的行业的涨幅或跌幅是多少（算上通货膨胀）？去年，你所在行业的消费金额是多少？增加了哪些新品牌？看这些是为了设身处地了解自己圈子的经济的变化，看未来有哪些新机会，又失去了哪些机会。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>3. 看社会：</span></strong></span><span>其实就是看城市，选择你未来为之奋斗的城市。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，有人渴望回归小城市，同时希望把大城市所谓的正义与秩序复刻到四五线的小城市，最后发现完全行不通；再比如，很多人觉得北上广内卷严重，回归到安逸的四五线城市，可是你会发现取代“内卷严重”的是小城市的“人情世故”，又会出现新问题。所以，选城市会是未来的一大风向标。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>4. 看科技：</span></strong></span><span>科技是一切的原动力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，今天做新消费品牌的企业，其实在研发的投入并不多，而且真正掌握核心技术的也非常少，所以我们还是要在科技方面不断增强。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>电话、手机、互联网、电视机、自动驾驶……你去看20世纪人类发明史，几乎没有来自中国的发明创造，这是我们需努力的地方。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/83249341562d104f88f96e5.59506183.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上四点是需要我们关注的。看国家政策在朝哪些行业倾斜；看行业圈里的经济状况，看哪些城市适合自己；看公司技术是否领先。如果都没有，你要谨慎选择。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/58143460660235253dbd083.00129585.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总结三个关键词：后浪拍死前浪、跨行业躺枪、95-00后“作妖”。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>风口一：预制菜市场。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>预制菜已经从厨房中的方便菜，衍生出很多花样。包装越做越好看，品类越来越多。同时，也有很多预制菜的公司完成了高估值的上市。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，我觉得未来预制菜市场的竞争对手是外卖。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家想一下，在家只需要微波炉加热就可以吃到和外卖一样好吃的东西，且价格只有外卖价格的一半，你还会点外卖吗？所以，预制菜市场会对外卖市场造成巨大挑战。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前来看预制菜里面没有什么品牌，卖的好的基本是靠低价走量为主，可是等到整个预制菜市场里面出现了包装水平、营销水平能够趋近于像湾仔码头的品牌，那么预制菜市场才是真正成熟起来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>风口二：宠物市场。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，拥有宠物的家庭比例中，美国占17%、日本占11%，中国不足1%。可是去年整个上海的猫类宠物市场增幅达到450%。所以说，随着中国宠物市场不断成熟，我们中国人拥有宠物的家庭会越来越多。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>风口三：三胎市场。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>政策会推动三胎市场，而这个市场像极了大宗消费。为什么这么说？在房地产进入到平缓期以后，可以产生长期大投入的大宗消费就只有生孩子。所以，国家将持续推动三胎政策，未来会与税赋挂钩，并且会横跨约会、婚庆、车房、产护、母婴、早教等数十个行业。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>风口四：95后养生市场。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>95后养生送礼成分减少，更多是为自用，自用里面还要加入科技感的元素。比如，电动泡脚盆、电动按摩仪等等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>风口五：00后野性消费市场。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，汉服市场（去年破100亿）、动漫周边、潮鞋、摩托车、玄学、游戏。其实这些都是80后用过的东西，只不过全部升级迭代后以00自己的方式再玩一遍，大家可以看出00后是多么喜欢野性消费！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8187367216030c0c85d55e0.47560833.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong>关于货：好好干有前途</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何判断一个行业是否内卷？答案是直接看行业生产端是否完善。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先卷的是营销端，最后卷的才是生产端，因为已经到了卷无可卷的地步。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>分享几个关于赛道的特点：</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1. 碰了就死的赛道特点：高频、高性价比、低客单、适应人群广；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2. 疯狂PK的赛道特点：烧流量、做国潮、直播高佣金、分销无底线；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3. 一线生机的赛道特点：低频、小众、高客单。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>分享几个关于中国市场的特点：</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1. 服务极小众：中国再小众的市场都相当于一个中型国家的市场；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2. 冷门细分类：功能细分再细分、营销奇怪再奇怪；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3. 反流量玩法：把投流的钱拿去做内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说了这么多，这是一个“别人向左你向右，才有机会成功”的时代。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年跟着别人走，连汤都喝不上。因为成功没有固定模式，别人套路不可复制，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同人做产品的思路完全不同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/118402775662d104f9be56e8.13818387.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>OEM做产品：</span></strong><span>我便宜我最有性价比，不需要广告公司包装，价格可以做的很便宜，但一上市就芭比Q，因为成本意识太强，会忽略品牌价值里的审美价值。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>供应链做产品：</span></strong><span>我渠道多我厉害。但有个问题：卖别人产品有固定利润，但卖自己产品就不行了。所以，产品卖得好并不一定是好的品牌，好的品牌跟销量大有关联，但不是绝对。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>广告人做产品：</span></strong><span>我创意好我厉害。广告人的片段思维比较严重，线性思维比较差，同时对数字不太敏感，对互联网不太敏感，所以大多数的结局就是这么好的东西怎么没人买呢？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>投资人做产品：</span></strong><span>我钱多我厉害。9个月烧1个亿，烧死同行烧死自己。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上是反面例子，希望大家无论以怎样的视角去做品牌，一定要规避这些问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上面我们聊过“人货场”，相信每个人都有了自己的答案，接下来怎么启动？如何才能打造一个优秀的团队，一起去战斗？以目前我的团队样本为例，和大家聊聊。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>品牌工作室模式才是未来。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/63054899662d104fa7ad868.18026883.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以我的公司为例，我们就是品牌工作室模式。人员构成也非常简单：1个领导（老板）、2个行政（财务&人事）、3个产品（研发&工厂&设计）、4个电商（运营&投放&售后&仓储）、5个4A（策划&媒介&客服&美术&文案）。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上岗位中最核心的岗位就是产品经理。一个优秀的产品经理塑造的产品，从一开始就让产品有了竞争力，可以自然吸引用户的目光和购买。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个好的产品经理应该具备什么样的特点呢？</span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>好产品</span></strong><strong><span>经理的</span></strong><strong><span>10个特点：</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1. 懂技术材料工艺；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2. 会控制成本；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3. 有良好的设计审美；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4. 对竞品的供应链、销售渠道、营销方法了如指掌；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5. 看得懂行业数据和消费者数据；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6. 网感好、对互联网用户有深刻洞察；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7. 时刻紧跟流行消费热点；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8. 懂得爆款逻辑；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9. 细节控；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>10. 最后是适合产品经理的三个星座：完美主义者的处女座、特别能磨细节的摩羯座、特别会算账的金牛座。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样一个好的产品团队，能让产品经理如鱼得水，爆发更大的潜力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>接下来看好的产品团队的12个特点：</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1. 特别喜欢讨论产品；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2. 互相分享竞品消息；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3. 产品核心主线达成共识；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4. 每个人都有补位意识；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5. 每个人都有专业特长；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6. 有一个开心果；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7. 有一个鬼点子特别多的人；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8. 狼性文化、996、股权激励都毫无意义；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9. 欢乐而轻松的氛围；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>10. 经常互相吹彩虹屁；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>10. 创始人80后，合伙人90后；员工00后；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>11. 稳定稳定稳定。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们打造团队是为了做好产品，在市场上赢得口碑，树立品牌，那么品牌的核心价值是什么？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我认为就是使用价值、审美价值和社交价值。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/214439874062d104fb13b7a7.76679343.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在要想做品牌，一定从审美价值（也就是包装）先搞起来，使用价值（产品部分）部分要做到“人无我有、人有我优、人优我转”，最后才是让用户为使用产品而感觉不同，乐意对外分享和表达，也就是品牌赋予的社交价值。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后总结一下，能搞钱的方法有很多，但是做品牌绝对不是最好的一个，这个过程是艰巨且复杂的，入局者要做好心理准备。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>分享的最后，送大家一句我很喜欢的话：<strong>想要无可取代，就必须时刻与众不同。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>期待直播间的每个人都能找到自己的优势，看到时代的趋势，把握人生的机遇，选择细分的赛道，不断打磨且努力，最终创造与众不同的自己。</span></p>
