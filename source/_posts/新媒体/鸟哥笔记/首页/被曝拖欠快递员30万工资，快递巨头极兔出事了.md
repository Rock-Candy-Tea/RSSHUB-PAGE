<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">快递巨头<a href="https://www.niaogebiji.com/tag/%E6%9E%81%E5%85%94" target="_blank">极兔</a>，狠狠摔了一跤。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就在最近，#极兔快递被曝拖欠快递员工资#的话题冲上了微博热搜。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/45508294162ad0ae6723305.99157960.png" title="被曝拖欠快递员30万工资，快递巨头极兔出事了" alt="被曝拖欠快递员30万工资，快递巨头极兔出事了" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">● 图源：微博话题</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据了解，这个快递点位于江苏常州，被曝光出拖欠快递员工资30多万元。为担心货物被拖走，有十几个快递员直接睡在地上“轮流把守”，甚至有快递员抱着自己11个月大的孩子追讨工资等到半夜。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对比，极兔官方也给了回复并表示代理区已经介入，目前妥善解决了欠薪的问题，快递积压件也会在随后进行处理。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只不过尽管平息了欠薪风波，但对这个快递行业的黑马而言，极兔的问题并不止于此。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" title="被曝拖欠快递员30万工资，快递巨头极兔出事了" alt="被曝拖欠快递员30万工资，快递巨头极兔出事了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">疯狂的东南亚兔子</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">说起极兔快递，绝对绕不过去的就是创始人李杰。作为“创业教父”段永平的徒弟，李杰在大学毕业后就凭借着不俗的业绩步步高升，并且最终被OPPO挖走，调往印度尼西亚开拓市场。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在李杰的带领下，OPPO在印尼市场的业务越来越好，但在过程中物流运输的痛点也被彻底暴露出来。严重滞后的物流，让李杰萌生了自建物流公司的想法。最终在2015年，极兔快递被李杰在印尼创立并走上了疯狂的扩张之路。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">凭借代理商和OPPO之前建立的关系网络，极兔快递仅用了两年时间就成为了东南亚市场单量第二，印尼快递行业日单量第一的快递公司。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/172108557362ad0ae6d23196.96766894.png" title="被曝拖欠快递员30万工资，快递巨头极兔出事了" alt="被曝拖欠快递员30万工资，快递巨头极兔出事了" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">● 图源：极兔官方微博</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在东南亚发展壮大之后，极兔快递于2020年进入中国，并凭借着低价烧钱的方式快速跑马圈地，试图抢食“四通一达”所把控的电商件市场。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">靠着国内下沉市场的一片蓝海和短视频等各种新型电商平台业务的带动，仅2021年极兔快递的日均业务量就从最开始的100万飞跃式扩张到2000万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据了解，2021年的618年中购物节，极兔快递的日单量更是一度突破3000万单，一时间风光无限。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">高速成长的极兔，也吸引了一大批如红杉、腾讯等明星资本的加入。仅在2021年极兔就先后完成了三轮融资，经过融资之后极兔估值更是突破1200亿人民币，成为菜鸟网络之后第2个估值超过千亿的物流公司。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12536271962ad0ae7cfe4d2.26811480.png" title="被曝拖欠快递员30万工资，快递巨头极兔出事了" alt="被曝拖欠快递员30万工资，快递巨头极兔出事了" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">● 图源：企查查</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">步子迈的太大，就很容易摔跤。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如今在网络上，关于极兔快递的负面评论不绝于耳。“货物不配送，联系客服和平台都无法解决”、“极兔送达的快递就没有一个是形状完整的”、“一个快递配送三天先后换了3个快递员”等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">显然，疯狂的极兔虽然靠着大肆扩张在趋于饱和的快递市场分到了一杯羹，但从消费者的实际感受来看，依然还有很多的功课要做。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" title="被曝拖欠快递员30万工资，快递巨头极兔出事了" alt="被曝拖欠快递员30万工资，快递巨头极兔出事了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">曾68亿拿下百世，上演“蛇吞象”</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">凭借低价策略，有数据显示，在2021年初极兔快递的日单量就突破2000万大关。要知道成功跨过这个门槛中通用了16年，韵达则用了19年，而极兔在一年内就达到了这个目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只是大打价格战并不利于快递行业的健康持续发展，各地监管部门也开始重拳出击，整治价格战。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当低价策略失灵，为了能够进一步抢夺市场，并购成为了极兔优先考虑的选择，这一次极兔快递把目标放在了经营正处于艰难时期的百世快递。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">彼时的百世快递单票收入只有1.86元，收入的持续低迷进一步挤压了百世快递的利润空间，这让亏损多年的百世集团逐渐变得力不从心。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/31166090662ad0ae8288e93.42943882.png" title="被曝拖欠快递员30万工资，快递巨头极兔出事了" alt="被曝拖欠快递员30万工资，快递巨头极兔出事了" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">● 图源：百世快递官方微博</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据2021年前三季度的数据显示，百世集团亏损超过17亿，在这之中主要就是受到了快递业务的拖累。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">于是，极兔快递出手了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年10月，百世集团和极兔快递达成了战略合作，百世集团同意把国内快递业务以68亿人民币的价格转让给极兔。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">乐观来看，这笔交易是一个双赢的局面。对百世集团来说，脱离了长期以来亏损的快递业务；对极兔而言，也能够进一步抢占市场。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然也有悲观的人认为极兔和百世合并后，市场份额并不是简单的订单量相加，更多还是要考虑消费者是否会买单。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14068884762ad0ae9035356.56318188.png" title="被曝拖欠快递员30万工资，快递巨头极兔出事了" alt="被曝拖欠快递员30万工资，快递巨头极兔出事了" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">● 图源：国家邮政局官网</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据国家邮政局官网显示，今年第一季度在快递行业满意度调查当中，极兔快递依然处于垫底的位置，黑猫投诉平台上关于极兔快递的投诉量也已经突破10000条。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" title="被曝拖欠快递员30万工资，快递巨头极兔出事了" alt="被曝拖欠快递员30万工资，快递巨头极兔出事了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">狂奔过后，极兔未来在哪？</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实上，“补贴+低价”一直以来都是极兔能够搅局快递市场的主要手段。尤其是在面对圆通、申通、百世快递的竞争时，极兔更是把价格压到了难以置信的程度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为价格战的背后推手，极兔的每单价格通常比“通达系”要低1.5元，甚至部分长途订单可以低3元以上。然而在极兔收购了百世快递之后，整个快递行业的集中度进一步提升，价格战开始逐渐淡出人们的视野。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这之中除了监管层的介入之外，极兔也迫切需要利润来修复自身。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/83200856662ad0ae9918654.99173847.png" title="被曝拖欠快递员30万工资，快递巨头极兔出事了" alt="被曝拖欠快递员30万工资，快递巨头极兔出事了" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">● 图源：极兔官网</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只是当自身的价格无法和其他快递公司拉开差距之后，极兔提升市场占有率的速度开始逐渐放缓。同时顺丰也在2021年宣布加入电商件的竞争，极兔所面临的压力开始进一步显现。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">与此同时，为了进一步抢占“通达系”的电商件市场份额，极兔仍然需要在基础设施上投入大量资金，这让原本就在烧钱运营的极兔面临的压力持续加大。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">还有业内人士表示：极兔在收购百世之后，并没有把全部的网点“纳入麾下”，很多网点都处于无人接管的状态。由此也可以看出，目前极兔的市场占有率并不是简单的“极兔+百世快递”之和。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对此，极兔快递的办法之一是向海外大举扩张。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/157192394062ad0aeb5039e4.00969574.png" title="被曝拖欠快递员30万工资，快递巨头极兔出事了" alt="被曝拖欠快递员30万工资，快递巨头极兔出事了" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就在今年1月，极兔就正式开始了在阿联酋和沙特阿拉伯的业务；2月又官宣进军拉美市场，在墨西哥运营；5月，极兔在巴西也开始了正式的营业。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就目前来看，极兔在各个国家的子公司都是以建立全国自有配送网络和本地化仓储系统为主，以此来负责境内的物流业务。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">曾经的极兔，凭借着简单粗暴的“三板斧”跑马圈地、加盟扩张、重金收购来抢占市场。只是就目前来看，除了在东南亚市场以及中国内地有不错的订单量之外，全球其他国家的业务距离真正成长起来还有很长的路要走。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对当下的极兔而言，或许更应该做的是先把自己曾经蒙眼狂奔所留下的烂摊子收拾好，想办法赢回口碑，然后再去谋求健康良性的发展。</p>
