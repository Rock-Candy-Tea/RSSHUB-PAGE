<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你经常在抖音看旅行视频，应该刷到过下面这条视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在荒凉的山区，一个皮肤雪白的姑娘，摇晃着脑袋，头戴帽子，摆动着四肢，跳着爵士舞。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/200975190462ba1ce6c96ee4.32786445.gif" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/51555982862ba1cf03af844.23994978.gif" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这条看似很简单的15秒短视频，却在今年吸粉超过百万。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更加不可思议的是，她的视频从不露脸，戴着墨镜，风格不变。</span><span>视频封面里没有任何标题，和其他旅游博主的主页大不一样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但就是这种简简单单的视频，平均每条视频点赞达到近30W，</span><span>最火爆的一条达到120W。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/173737337362ba1cf7757d73.30546964.png" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>视频博主名叫@Mushini，是今年最火的旅游博主之一。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2022年5月9日-15日，7天涨粉104万，排名抖音涨粉周榜第二（第一名是刘畊宏）。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其他的旅游博主需要辛辛苦苦四处各地跑，拍各个景点的风景，运气不好的还会被隔离。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再花几个小时以上做成旅游视频，点赞能达到几千上万，就已经算很火了。很多旅游博主做了几年才几十万粉丝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而她，每天在同一个地方，跳一段15秒的爵士舞，一条视频轻轻松松涨粉过万。可能是这种形式尝到了甜头，所以现在每条视频都走这种风格......</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>有很多人觉得不可思议，这种看起来没什么内核（比如不露身材，没故事，没剧情，没风景等）的视频，怎么会涨粉？这种视频路线能复制吗？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们分析后认为有几个关键因素：</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>1. 背景音乐选对了</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>细心观察，会发现她的大部分视频背景音乐，都使用了同一首歌《sea shanty medley》。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/195759414162ba1cf8573903.91093458.jpeg" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众所周知，抖音视频要走红，几个因素缺一不可：<strong>内容、封面、文案和音乐</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>背景音乐必须和视频画面没有违和感，最典型的比如古风视频要用古风音乐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>像@Mushini的视频都在西部地区拍摄，定位旅游类型，适合用欢快等类型的音乐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这首音乐和视频内容确实很搭，都有种自由洒脱的感觉，以至于很多网友听到这首歌，就脑补到她的跳舞画面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/41607443162ba1cf94b9fa7.98430602.gif" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，凭借这首歌，涨粉数百万。有些视频忽然换了音乐，反而导致很多粉丝不习惯了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/35422920962ba1d01546e43.28220632.jpeg" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/204280372362ba1d0196b5b4.30133455.jpeg" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20288872762ba1d01c15ec2.96271716.jpeg" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然了，并不是一开始就选用了这首音乐，前期也使用了其他音乐，数据也很好。直到这首歌爆火，就一直沿用它了。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>2. 皮肤“尸白”，引发关注</span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没错，刚开始确实没想到是这个原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到看到她的评论区，很多人留言夸她白，确实很多人因为她长得白，所以才关注，以至于询问用了哪款防晒霜；还把她称作“尸白”。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如下面这条视频，脸部皮肤白，可以用美颜。而全身都是雪白的，和面部同一种肤色，修起来难度就大了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/195675350862ba1d024eb817.58793520.gif" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/162902719862ba1d0ecd9309.10518716.jpeg" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然了，不是说她长得白，所以就火了。而是前面种种因素（跳舞，视频风格等），再加上皮肤雪白，增加了大家的点赞率，完播率等。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>3. 怎么拍摄？能复制模仿成功吗？</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人会思考，如果自己也想做这种视频，该怎么操作？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先她的视频都是在西部地区拍摄，背景都是草原山峦等。如果要复制，最好整个拍摄团队，要到西南或西北地区。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以花几天时间，拍摄大量素材，后期剪辑陆续发布。也可以选择某个县城，长期到周边景区拍摄，便于不断优化改进视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前主要通过稳定器+相机、无人机、摄影机+滑轨等设备才能拍出精良的画质视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而像这种风景呈倍数放大的效果，需要用到长焦镜头。</span><span>我们分析认为，使用相机长焦镜头或大疆无人机（长焦镜头）拍摄的概率较大。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/67557257662ba1d0f188555.47929965.gif" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同的画面，拍摄方式不一样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如人物大范围的在移动，背景变化不大，可以是平行或跟踪运镜，用无人机拍摄；也可以是在固定机位用相机拍摄。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如风景大范围的在移动，而人物基本上静止不动，这种环绕运镜，可以用无人机拍摄，也可以用相机+稳定器环绕拍摄。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/51555982862ba1cf03af844.23994978.gif" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>因为人力成本较低，实际上这类视频，通常只需要1-2人即可完成拍摄。其中用无人机拍摄最方便，因为可以很方便的调整拍摄角度，可控性高。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以肯定的是，这种风格的视频，可以复制，只要具备以上条件就行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>自从@Mushini火了后，有超过10W人使用了同款音乐，很多小姐姐陆续模仿她的视频。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><img src="https://ngbjimg.xy599.com/170457841262ba1d19425814.48186669.jpeg" title="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" alt="一首歌涨粉300万！她成抖音当红旅游博主，做对了什么？" referrerpolicy="no-referrer"></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上绝大多数人做视频，都不知道拍什么内容好，表现形式是什么，这种风格的视频无疑给了大家启发。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如下面这两位博主，直接选取同款音乐拍摄。跳得更带劲，穿得也更清凉，舞蹈动作在相似的情况下更加激情奔放，只不过评论区就......</span></p>
