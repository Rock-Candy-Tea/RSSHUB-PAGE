<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><a href="https://www.niaogebiji.com/tag/%E9%98%BF%E9%87%8C" target="_blank">阿里</a>的第一家自营店铺，倒下了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，“<a href="https://www.niaogebiji.com/tag/%E6%B7%98%E5%AE%9D" target="_blank">淘宝</a>心选”官方旗舰店正式宣布暂停运营，原心选业务升级为全新品牌“喵满分”，聚焦年轻家庭人群的各项消费需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/153097260062c625992c2897.48864426.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一项业务的消失，往往会被巨头包装成“升级”的样子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为主打“健康、美好、有用”的生活方式独立品牌，</span><span><strong>淘宝心选在跌跌撞撞走过了5年之后，最终还是倒在了精品电商的寒冬。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>淘宝心选，藏着阿里的自营野心</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在某种概念上，淘宝心选算是阿里在自营模式上的初探。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>五六年前，正是自营精品电商概念被炒的如火如荼之时。在传统电商的绝对统治力下，以网易严选为首的精品电商阵营依旧牢牢占据了一定的市场份额。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14988150862c6259958f225.98319333.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而后，小米、阿里、京东等玩家一拥而上，诞生了小米有品、淘宝心选、京东京造等一众自营精品电商，巨头的入局使得这一市场百花齐放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝心选的诞生就在2017年。那一年，精品电商的龙头网易严选实现了70亿元的GMV，风头正盛，并计划2018年GMV达200亿，给了淘宝不小的压力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>重压之下，阿里将当时淘宝旗下“中国质造”业务正式升级为淘宝心选，对标网易严选。而这，也是阿里旗下第一家自营店铺。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在上线后的大半年内，阿里CEO张勇对其极为关注，淘宝心选也开始与薇娅、李佳琦等知名主播合作，频繁在直播间内亮相。至此，阿里的自营野心终于开始显露。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>只是可惜，背靠阿里这颗大树成长起来的淘宝心选，最终也限于阿里的掣肘，变得泯然众人矣。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/98712825762c62599817bf8.81944388.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>究其缘由，在淘宝心选诞生时起，阿里内部对其定位便纷争不断。心选创始团队希望它成为一个帮助零售商开展自有品牌的供应链服务生态，是一种超脱零售业务的供应链生意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而阿里高层显然不这么想，因为按照前者的方向，淘宝心选一旦做大，将会严重威胁到传统品牌商的蛋糕，而这则是阿里淘系电商的主要利润来源，轻易触碰不得。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以分歧到了最后，以淘宝心选创始人张棣的主动离开而告终。淘宝心选也在同一年由淘宝天猫事业群被划分至同城零售事业群，静悄悄没了声音。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到如今，心选业务正式关闭，永久告别历史舞台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这也宣告着阿里在自营模式上的首次探索，完败。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>阿里再探自营模式</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>向来只见新人笑，何时闻得旧人哭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>跌跌撞撞走过数年的心选时代落幕，“喵满分”紧随其后为阿里自营掀开新的篇章。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“喵满分”上线于今年4月，彼时，淘宝心选正式宣告了这一品牌升级计划，也预告了自身即将关停的命运。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/136192513362c62599d3bc13.58998768.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由心选而生的喵满分，喊出了“可信、品质、平价”的口号，主要聚焦在宠物用品、粮油米面、坚果零食及家清洗护四大类目，并宣称“天猫超市独家商品”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>官方资料显示，基于对天猫超市消费者的深度了解，“喵满分”在全球寻找优质原产地，商品主打源头直供、原料专业的开发能力、严苛的品控，以及性价比优势等特点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>背靠大树好乘凉，这一景象，与淘宝心选诞生之时何其相似。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，阿里在自营模式上的再探索，也不仅限于喵满分。早在之前，天猫国际、考拉海购等均有相关自营业务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，今年年初，新官上任三把火的戴珊将淘宝、天猫合二为一，并正式上线了“猫享”官方自营业务，主打3C、国际品类，对标京东自营。这一动作，可以认为是阿里向自营赛道正式进军的标志。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/180784943162c6259a1007b8.03044354.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以看出，喵满分在阿里的自营板块中并非孤军，相反，猫享甚至拥有着更加丰富的商品种类，究其原因，在于二者分工不同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喵满分依旧专注于自有品牌、专注于年轻家庭人群的消费需求，这既是阿里在自营模式上的再探索，也是其在精品电商道路上留下的一颗种子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在天猫超市这一年轻人、年轻家庭增速最快的平台，阿里关于自有品牌的自营道路还在继续前行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一定程度上，喵满分的差异化发展，也将会为阿里带来更大的竞争优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>精品电商还有未来吗？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在互联网生意场上，“小而美”往往是最难走通的道路。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回顾七八年前，与精品电商相继兴起的“社交电商”概念，已经成功跑出了拼多多这一巨头，引得阿里、京东纷纷押注下沉市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/115715952462c6259a312298.93540157.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反观自营精品电商，尽管无数巨头纷纷杀入，然而还是不可避免的走向了行业寒冬。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经的龙头网易严选早已掉队，小米有品则与集团零售业务整合，业务权重被降低，淘宝心选更是直接宣告死亡。到目前为止活的最好的，只剩下一个京东京造，然而这也与京东本身就偏自营的商业模式有关。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/199249443262c6259a61a722.39571015.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>商业基因这种东西，学不来的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那精品电商还有未来吗？答案，或许还要从几年前追溯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>彼时，精品电商与社交电商还都是炒的如火如荼的香饽饽，但二者却又有着本质的不同。前者是专注于中产阶级的生活方式，定位较高端；而后者则专攻下沉市场，采取农村包围城市战略。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那时，稳坐电商头把交椅的淘宝正全面天猫化，京东稳居一、二线城市，与其隔空对阵，大环境下，电商玩家们一股脑的喊着“消费升级”，涌向品牌、高端的赛道，精品电商概念被炒的火热。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>只有拼多多，狠狠抽了猫、狗们一个大嘴巴子。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>恰逢当时电商流量增长趋缓，而精品电商的供应链故事又太重太长，自此，电商圈刮起了一阵下沉市场的风。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>不仅如此，随着短视频巨头的迅速崛起，又给了阿里们狠狠一拳。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在近年来疫情的加持下，专注低价、性价比的直播电商迎来风口，瞬间成为一门万亿产业，更是屡屡有头部主播单场破亿GMV的故事流传。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/92848856562c6259aa32458.81775904.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再结合当下经济形势，市场需要的究竟是消费升级，还是消费降级？一目了然。而专注于中产阶级的精品电商，自然没了活路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，市场规模小，不代表没有生意可做。只是在如今的形势下，小玩家恐怕没有上桌的资本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>未来的一定时间内，就像“京造”之于京东，“喵满分”之于阿里，将精品电商作为综合电商一个单独品牌的补充，或许是其还能维持小众经营的唯一道路。</span></p>
