<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; padding-left: 12px; background-position: 0px 5px; background-repeat: no-repeat no-repeat;">前言</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">每次都是刷手机刷到很晚，实在过意不去了，才拿起书开始学习；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">勉强看了一会书后，又按捺不住身子，一会起身吃个水果，一会整理下桌子。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为什么学习不像打游戏一样，不用任何人监督和提醒，就自发自觉地从早学到晚？能让我们沉迷其中，感受到特别的快乐？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这是因为，有效学习的本质是破除旧习惯、形成新习惯，这本身是反人性的，很难做到自发自觉。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; padding-left: 12px; background-position: 0px 5px; background-repeat: no-repeat no-repeat;">01、学习最大的误区，是指望自己“自觉”</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不像吃饭睡觉，不用任何逼迫，我们天然地就会自发地进行，有效的学习需要一定的启动动力，以及能够坚持的意志力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但可惜的是，动力和意志力都没那么容易提高，而且都有使用上限。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>▼ 1、动力会骗人</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果你是自主决定、发自内心想要去做一件事，你就很容易沉浸其中，甚至进入心流状态，忘记时间的流逝。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种由内在动力驱动的学习状态，是非常令人羡慕的，因为学习过程愉悦，又很有收获感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但内在动力的产生是存在滞后性的，因此内在动力会存在启动难的问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只有先学了，对所学的内容有所了解了，擅长了，你才会产生喜欢的情绪。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">热爱不是凭空产生的，是做出来的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">没有学，何来的内在动力？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，如果只依赖内在动力的驱动来学习，就会陷入了一个死循环：因为没有内在动力，所以无法学；可不学，又没法产生内在动力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有人也许会质疑：那我们能不能就是简单地对一件事感兴趣，所以想去学习钻研呢，就像一个人对另一个人一见钟情一样？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我不否认，可能会有这样的情况，但这极其罕见。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们绝大多数人所谓的感兴趣，更多是将三分钟热度，误以为是内在动力。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些由美好想象激发的动力，就只是三分钟热度，不具备可持续性。一旦在学乐器和减肥的过程中，稍微遇到阻碍，立刻就冷却下来了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">真正的热爱，不只是凭借脑海中虚幻的美好想象去驱动，必须经过一定时间的深入尝试，知道其中蕴涵的痛苦和不如意的地方，依然能选择坚持下去，享受做这件事本身的点滴乐趣，才是真正的内在动力，才具备可持续性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>▼ 2、意志力易损耗</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">心理学上曾做过一项研究，</p>

<p>让节食者进入一个满是美味零食的房间，里面放有薯片、彩虹糖、巧克力和咸味花生豆，然后再让他们在计算机上完成一项任务。</p>
<p>任务完成后，实验对象就可以吃到大桶的冰激凌。</p>
<p>对于有些人来说，食物就摆在他们旁边非常显眼的位置；而对于另一些人来说，食物摆在他们看不见的地方。</p>
<p>那些坐在食物旁边的人，刚才一直在抵御美食的诱惑，任务完成后，他们所吃的冰激凌量，要比远离零食诱惑的另一组实验对象多得多。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为什么会这样呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为咱们普通人的意志力是有限的，就像肌肉一样，在频繁使用后会变得疲劳。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里用得多一点了，其它地方就会少一点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一直接近美食的人，需要不断消耗意志力去抵抗诱惑，所以最后就没那么多意志力去抵抗冰淇淋的诱惑了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/31666500462b501de438d68.98638445.jpeg" title="学习最大的误区，是指望自己“自觉”" alt="学习最大的误区，是指望自己“自觉”" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时，意志力又很容易被损耗，受到各种各样因素的影响：努力程度、感知难度、消极情绪、主观疲劳和血糖水平。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你耗费的努力太多，所做的事对你来说难度太大，甚至你今天心情不好，没睡好，忘记吃早餐导致血糖过低，都会导致意志力损耗。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这么多因素的制约之下，再加上学习本身就是有难度的，我们要有所困惑，主动去思考和推论，这样才能产生学习效果。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种情况下，还要依赖意志力去坚持学习，就会变得非常困难。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们熟知的那些意志力惊人，一直拥有源源不断动力去学习的人，如为中华之崛起而读书、头悬梁锥刺股、凿壁偷光的例子，就是因为罕见，所以才广为流传。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那有没有什么方法，不需要很强的内在动力，不需要很大的意志力消耗，也能推动我们去学习呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我经过实践，发现下面两个方法效果挺好，跟你分享下：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1、功利性学习</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2、自我创造学习环境</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; padding-left: 12px; background-position: 0px 5px; background-repeat: no-repeat no-repeat;">02、功利性学习</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我有位同学，先前是在一个小城市的清水衙门工作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">每天基本是工作半小时，闲聊3小时，然后就是漫长的等下班时间了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为空闲时间多，她就打算边上班边准备公务员考试。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">可虽然时间多，但她的备考却总是三分钟热度：颓废半个月，然后打鸡血学个一两天，再颓废半个月，周而复始。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就这样，公务员考试毫无悬念地失败了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但自从换到大城市，找了一份有业绩指标压力的新工作后，她一下子就换了个人似的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从缺乏动力，意志力一般，半个月学习不到4小时的一个人，变成每天下班后，还能工作学习到凌晨。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/80584069462b501de6e3721.29270724.jpeg" title="学习最大的误区，是指望自己“自觉”" alt="学习最大的误区，是指望自己“自觉”" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就是功利性学习的作用，根据工作需要去学习，学习立马就要用到的，对内在动力要求不高，也不需要特别强的意志力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对我们职场人来说，工作环境是我们最容易找到的有利学习环境，唾手可得，而且不需要你耗费精力去创造和维持。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">利用工作环境做功利性学习，可以帮我们很好地克服学习上的两大缺陷：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>▼ 1、解决内在动力产生滞后和不可持续的缺陷</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为工作需要，所以无论你想不想学，你都要开始学。</p>

<p>就好比早上起床，如果公司规定9:00上班，迟到还要罚款，那么你按时起床所需的动力其实是很少的，因为这是一件不得不做的事。</p>
<p>再比如，有个新项目，需要你从零开始学习一个行业的知识，但只要你做好了，就有机会升职加薪，那么即使你没什么内在动力，你也会动手去学。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在外部动力的作用下，你开始学了，慢慢地对所学的内容有了深入的认识，你自然就知道自己是否真正感兴趣了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这时产生的兴趣，就是真正的内在动力了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，如果能多多采取功利性学习的策略，你就可以通过外部动力让启动简单化，逼着自己深入学习，从而产生真正可持续的内在动力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>▼ 2、解决意志力消耗大，难以坚持的问题</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">功利性学习，因为是从工作的实际需要出发学习，所以无论你想不想学，都必须要启动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种被动式的启动，跟自己主动逼迫自己学习相比，消耗的意志力要少上很多。</p>

<p>但如果今天是周末，又没啥必须要早起处理的事情，这时你按时起床所需的意志力就远远大于平时上班起床所需的意志力了。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">功利性学习不仅解决了启动学习的意志力消耗问题，还解决了坚持学习的意志力消耗问题。因为要在工作中使用，随着工作需要自然会不断地学习新的内容，因此你无需消耗意志力去逼着自己坚持学习。</p>

<p>反之，如果你现在是一名美工，想转型做Android开发，那么对你而言，坚持学习Android平台新功能所消耗的意志力，就会远远大于前者。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而且，因为是工作所需，你学习的方法和技能立即就能见效，这个就更会刺激你有兴趣去重复、去学习更多的内容。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而如果你学习的内容不是工作急需的，那么就会存在看不到即时效果的问题，在这种情况下坚持学习，消耗的意志力就更大了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，如果能多多采取功利性学习的策略，你就可以用更少的动力和意志力，学会更多的技能和方法。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; padding-left: 12px; background-position: 0px 5px; background-repeat: no-repeat no-repeat;">03、自我创造学习环境</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">功利性学习的核心在于，要求你所处的工作环境有比较强的外部动力和压力，但要是我们所处的环境比较安逸，该怎么办呢？</p>
<section>
<section data-role="paragraph">
<section>
<section data-role="paragraph">
<section data-role="paragraph">
<section data-role="paragraph">
<section data-role="paragraph">
<section>
<section>
<section>
<section>
<section>
<p>比如，有些同学所处工作环境比较安逸，公司业务比较稳定，暂时没有太大的工作压力，也缺乏机会，想好好蓄力等待时机突破。</p>
<p>有些同学目前需要学习的内容，暂时没有应用场景，但考虑到长远的职业规划，又必须要学。</p>
<p>还有些同学想提升某个技能，或者考个职业相关的证书，提升工作竞争力。</p>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种情况下，你就可以自我创造不依赖自觉的学习环境。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种学习环境如何打造呢？你只需要让其具备以下四个要素即可：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>▼ 1、触发</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">人是短视的动物，只会注重眼前看得到的事物。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很多时候，我们之所以无法学习，是因为被太多眼前的琐事占据了注意力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为短期来看，学习是一件重要但不紧急的事，学的话暂时也没太大好处，不学也没太大损失。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以总有比它更重要的事占据我们的时间和注意力，比如跟朋友出去聚一下、刷刷手机放松放松、饭点到了考虑下吃什么。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为预防这种情况发生，我们可以通过提前营造一些触发学习的线索，经常让学习这件事回到我们的视野之内。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1）人物提示：通过自己或他人的提醒采取行动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如我们的私教有定期提醒机制，当某个学员一两天没有学习反馈了，我们就会主动提醒，将学习这件事情重新拉回他的视野之中。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样，学习就成了无法被忽略的事情，因为私教老师会主动闯入你的视野，询问你什么时候可以完成学习任务，学习进度如何，以确保你一直都记得学习这件事情。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/148471439062b501debae1a2.87677536.jpeg" title="学习最大的误区，是指望自己“自觉”" alt="学习最大的误区，是指望自己“自觉”" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2）情境提示：通过周围环境中的某些事物来提示行动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，不想学习时，就可以考虑去图书馆或者自习室，跟在家相比，这样的情境到处都是跟学习相关的线索。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3）行动提示：将你已经在做的行为当作提示。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我之前想下班后再练习下写作，但每次一回到家就把这件事抛之脑后了，于是我便把练习写作和洗澡联系在一起。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样每天只要一洗澡，我就会想起写作这件事，于是我会边洗澡边构思，等洗完澡再把想法写下来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就这样不知不觉间，我坚持洗澡时练习写作的习惯已经差不多半年了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">▼ 2、外力</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在我们的视野里，学习是一件重要但不紧急的事，学的话暂时也没太大好处，不学也没太大损失。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但如果有一定的外在动力奖励和惩罚就不一样了。</p>
<section>
<section data-role="paragraph">
<section>
<section data-role="paragraph">
<section data-role="paragraph">
<section data-role="paragraph">
<section data-role="paragraph">
<section>
<section>
<section>
<section>
<section>
<p>比如，我跟你说：只要你今天学习1小时，立刻就可以得到100元；不学，立刻就失去100元。那么不管多忙多累，晚上12点之前，我相信你一定会抽出这一个小时去学习。</p>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">需要注意的是，外力机制有效的前提是要严格执行，因此不能自己既是球员又是裁判，你自己监督自己有没有完成，不然这个奖惩机制必定会沦为一纸空文。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">▼ 3、反馈</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">学习最怕的是什么？就是没有反馈。学好学坏都一个样，你就不知道如何调整改进。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那些武侠小说里的名门大派弟子，为什么往往功夫都远高于江湖上的野路子？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最核心的因素，就是他们有名师指导，定期有名师讲课，还有老师一对一解答疑惑。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">还以私教为例，私教同学每次练习后，都有自检清单和老师的点评反馈，让自己知道哪儿做得好，哪儿还需要修正。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">更高级的反馈，除了专业知识的反馈，还应该关注到人的情绪和心理，才能给你自我效能感，帮你更好地度过学习中的“高原期”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">像我们私教老师给同学反馈时，就很注意什么时候应该给他鼓励，什么时候应该给他表扬，什么时候应该给他压力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">▼ 4、氛围</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">人是社会性动物。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当解决了衣食住行后，生存无忧后，人的社会性需求就显露出来了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">每个人都希望自己优于他人，有不让自己处于劣势地位的优越感。</p>
<section>
<section data-role="paragraph">
<section>
<section data-role="paragraph">
<section data-role="paragraph">
<section data-role="paragraph">
<section data-role="paragraph">
<section>
<section>
<section>
<section>
<section>
<p>有些人的优越感来自于物质比较，比如开更好的车，更大的房子，名牌衣服等。</p>
<p>有些人的优越感来自于精神优越，给社会创造更大的贡献，品德比别人更高尚，更加爱国等。</p>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，我们在学习的时候，也可以通过给自己创造社会比较的氛围，从而达到促进学习的效果。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">大家都在学，你不学就会显得异类；大家都很努力，你不努力，就会落后。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">04、总结</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">天生就能自觉学习的人有吗？有，但比例极低。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我也很希望你我是其中之一，但不幸的是，太多的事实已经告诉了我们，我们不是。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">可幸运的是，即使我们不是这极少数的人，我们也有不自觉，也能好好学习的方法。它们是：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1、功利性学习：从工作的实际需要出发，学习立马就要用到的。解决自觉学习动力产生难，意志力消耗大的缺陷。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2、自我创造学习环境：通过营造触发线索、外在动力、有效的反馈和学习氛围这四个要素，创造一个让自己不自觉也能坚持学习的环境。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我亲测非常有效，相信对你也一定有作用！</p>
