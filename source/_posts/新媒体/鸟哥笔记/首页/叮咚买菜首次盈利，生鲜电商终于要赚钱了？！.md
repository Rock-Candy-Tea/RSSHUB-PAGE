<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在每日优鲜原地解散、公司倒闭等新闻甚嚣尘上之时，<a href="https://www.niaogebiji.com/tag/%E7%94%9F%E9%B2%9C%E7%94%B5%E5%95%86" target="_blank">生鲜电商</a>迎来了一波好消息。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月11日，<a href="https://www.niaogebiji.com/tag/%E5%8F%AE%E5%92%9A%E4%B9%B0%E8%8F%9C" target="_blank">叮咚买菜</a>公布了2022年Q2的业绩公告。财报显示，Q2叮咚买菜营收62.3亿元，同比增长43%。</span><span><strong>净亏损0.3亿元，不包含薪酬激励费用下，非GAAP下实现2060万元的利润。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/140856792762fb7b460b40c6.42250283.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span><span>▲</span><span> </span><span>图源：</span></span></span><span><span>叮咚买菜2022年Q2财报</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是叮咚买菜首次实现盈利，也似乎破除了“生鲜电商盈利难”的魔咒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><img src="https://ngbjimg.xy599.com/5611320262b974392829e0.57743105.png" referrerpolicy="no-referrer"></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>疫情之下的“输血”和“续命”</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2060万元净利润！</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在资本市场，这一数字看起来九牛一毛。但对比去年同期亏损的17.29亿元，这对叮咚买菜来说非比寻常。净亏损的3450万元，也仅是去年同期净亏损19.37亿元的一个零头。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在每日优鲜面临倒闭风波的当下，叮咚买菜的扭亏为盈，被看作是“前置仓最后的希望”。不过更理性看待的话，叮咚买菜的首次盈利，显然是吃到了“疫情封控”的红利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>四五月份，上海疫情大爆发，抢菜成了上海居民的日常。作为上海疫情期间的重要保供企业之一，叮咚买菜的订单激增，客单价也比平时高出不少。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/176792594862fb7b46effe60.09730288.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：@叮咚买菜官方微博</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>财报显示，在叮咚买菜较早拓展市场的长三角地区，今年上半年营收同比增长47.9%，并实现了3.7%的正向经营利润率。</span></strong></span><span>2022年第二季度，叮咚买菜GMV达到了71.15亿元，同比增长32.3%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，在第一季度已经出现资不抵债的情况下，第二季度业绩的增长，让叮咚买菜及时得到了银行资金的支持。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/109823726762fb7b49b730c3.50058518.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：东哥解读电商</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上海银行“输血”80亿元，并根据叮咚买菜的需求，定制研发了项目贷款、供应链融资等各类金融产品，有效缓解了资金链断裂的危机。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>财报显示，截至二季度末，叮咚买菜资金充足，包括现金和现金等价物、短期投资等在内的余额为60多亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>疫情对企业首次盈利的助力，叮咚买菜的创始人兼CEO梁昌霖毫不避讳地表示：</span><span><strong>“Q2的阶段性盈利，是受到了疫情的部分影响。展望Q3，我们可能会略微亏损。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而面对叮咚卖菜的亮眼财报，资本市场则持观望态度。8月11日公布业绩后，叮咚买菜的股价盘中涨幅超11%，随后又回落到4.41美元，同比还下降了1.78%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/38794396662b974394a3da6.63273158.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>前置仓困境下的自救</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>资本的无情不无道理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟疫情红利只是阶段性的，叮咚买菜能否实现长期盈利还是未知数。而每日优鲜的败局，加重了资本对生鲜电商“前置仓模式”的质疑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓“前置仓模式”，一般是指在离消费者尽量近的地方建仓库，储存由总仓送达的生鲜商品，并为周边地区尽快配送。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种模式的优点是配送效率高，可以满足消费者对生鲜“鲜”的高要求。</span><span><strong>但由于企业需要承担更高的商品成本和履约费用，“前置仓模式”一直面临盈利难的尴尬境地，成了无法承受之重。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在疯狂扩张的2019年到2021年，“五年如一日，死磕前置仓商业模式”的叮咚买菜净亏损分别为18.73亿元、31.77亿元、64.3亿元，累计亏损近115亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了遏制亏损，2021年8月叮咚买菜宣布战略调整，</span><span><strong>从“规模优先”转变为“效率优先”，把商品力作为发展第一推动力，提高企业的研发和创新能力，发力上游供应链。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/146668309462fb7b4a56dff3.53227124.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：电商头条</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在来看，这是叮咚买菜发展中的关键一步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从5月开始，叮咚买开启了“断臂求生”，陆续关闭了河北廊坊、唐山，广东中山、清远、珠海，安徽宣城、滁州等十多个城市的业务，以求降本增效。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，叮咚买菜已经研发出叮咚王牌菜、叮咚大满冠、拳击虾、保萝工坊、良芯匠人等20多个自有品牌，涵盖了酒饮、预制菜、面点烘焙等品类，“人无我有”的特色商品已经达217个SKU。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8148345362fb7b4b410d67.13072531.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：叮咚买菜APP</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了保证更高效地开发、生产与运输商品，叮咚买菜在10个食品研发加工工厂、约60个城市分选中心和约1400个前置仓、前置仓面积达50万平方米的基础上，还将开工建设3个大规模、现代化的生鲜综合体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这些自救措施让效率和盈利有了兼顾的可能，大大提高了叮咚买菜的客单价和毛利率。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据财报披露，二季度叮咚买菜自有品牌 GMV 占整体 GMV 的 17.5%，相比一季度的 7% 大幅度提升。履约费用为15.42亿元，较2021年同期下降9.0%，占总收入的比例下降到23.2%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/46978155362b9743fae3289.16804197.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>全面盈利还要走多久？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管并不回避疫情对叮咚买菜业绩的加持，但对叮咚买菜的未来发展，梁昌霖充满信心：“比较我们疫情后和疫情前的数据，除了营收增长，亏损率是在持续得不断改善的。所以，我们非常有信心，在今年期末，叮咚买菜可以实现全面盈利。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但要在今年年末实现全面盈利的目标，叮咚买菜恐怕并不容易。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，尽管昔日“死对头”每日优鲜已经元气大伤，但在生鲜电商行业，如盒马鲜生、美团买菜、朴朴超市依然是不可小觑的竞争对手。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/145488382762fb7b4c36a659.88132256.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：美团买菜官方微博</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>相对于竞争对手的O2O和店仓一体化模式，“死磕前置仓”的叮咚买菜仍然面临着“高成本-高价格”的难题。</span></strong><span>这也就意味着，叮咚买菜需要在商品、服务等方面更加极致，才能留住消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，在讲究精致的上海，叮咚买菜在2021年年底就已经实现了全面盈利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，高毛利、高利润的预制菜，被很多人看成是生鲜电商的盈利突破口、前置仓的“救命稻草”。入局预制菜业务的叮咚买菜，已经布局了超过1000个SKU。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/151799114662fb7b4cf40ad3.95505502.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：叮咚买菜APP</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>春节期间推出的“五湖四海年夜饭”等预制菜系列产品，更是为叮咚买菜在2021年第四季度贡献了9亿元营收，在全平台用户订单中渗透率达到30%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>然而，被寄予厚望的预制菜万亿市场，其实也是借着疫情的“东风”崛起的。</span></strong></span><span>尤其对</span><span>于很多C端消费者来说，预制菜不过是“救急之需”，不健康、口感差、不新鲜等，是很多人不会复购预制菜的关键点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>叮咚买菜的阶段性盈利，有很多偶然因素，在未来的发展中，也存在许多变量。“前置仓模式”下，叮咚买菜要想实现全面盈利，还需要在仓储、供应链技术、SKU种类等方面打出差异化优势，让“互联网卖菜”真正成为一门赚钱的生意。</span></p>
