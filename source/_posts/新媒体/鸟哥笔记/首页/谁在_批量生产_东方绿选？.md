<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一次，</span><strong><span>抖音一哥东方甄选迎来了真正的敌人！</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然东方甄选的内涵难以模仿，但形式却极易“克隆”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今打开抖音，各种“XX甄选”层出不穷。如果说“梯方甄选”、“东万优选”只是在名字上抖机灵，那其中一个叫“东方绿选”的直播间对东方甄选的抄袭简直到了令人瞠目结舌的地步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/84840633663221b62699243.36934199.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：微博网友@楚少主</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你只是因为董宇辉出圈爆火而听说过东方甄选，那你一定会被东方绿选的障眼法蒙骗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><img src="https://ngbjimg.xy599.com/6251482662db2e883e36e5.24710351.png" referrerpolicy="no-referrer"></span></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong><span>李鬼冒充李逵</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>利益之下，丑态尽出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这个名叫“东方绿选”的直播间从名字到直播背景设置都像素级“克隆”了东方甄选。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从logo设计到商品摆放，有样学样。据看过的网友反馈，甚至其中一个女主播连发型都在刻意模仿东方甄选的YOYO。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/174251720663221b62aa5b76.19137801.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：抖音网友</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而让东方甄选名声大噪的“双语带货”东方绿选也进行了克隆。没有新东方的英语老师，干脆直接邀请老外现场说英语，手里拿着小黑板现场英语教学的场面尴尬至极。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2214529663221b62e29e42.17493869.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：抖音网友</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比于东方甄选主播们的妙语连珠，东方绿选主播直播时磕巴连连、时常词穷的表现让人尴尬到脚趾头能扣出三室一厅。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可能也怕“一条道走到黑”彻底激怒了大众，东方绿选也鸡贼地选择了公益助农这条带货之路，试图用助农的外衣减轻网友的愤怒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>翻开东方绿选的抖音主页，赫然写着</span><strong><span>“优选全国绿色农产品，助力三农振兴乡村”</span></strong><span>，和东方甄选的助农定位如出一辙。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/108029922563221b6340c624.31702290.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：东方绿选抖音号</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且这也并非个人的拙劣模仿行为，背后是一家名叫“八条</span><span>（云南）</span><span>科技有限公司”的正规企业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我在网上查了一下，这家企业的法定代表人叫马俊，与之相关的企业有足足12家，涵盖了传媒、旅游、设计制作等各个领域，看来也是一个爱折腾的老手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/74447844663221b63742921.41196443.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：爱企查官网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而注册东方绿选的八条公司，今年的7月5号才正式成立，也就是在东方甄选6月9号爆火后的不到一个月。看来这家公司就是冲着蹭东方甄选的流量而生的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在东方绿选目前发布的10条短视频中，也全部都是猕猴桃、苹果等各种农产品的产地探访，其中不少拍摄手法和东方甄选早期的短视频作品高度类似。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在东方绿选的企业店铺里，总共上架了7种农产品，但销量都惨不忍睹。除了一款在直播间售卖1.9元12个的苹果卖了86单之外，其余的都是个位数，甚至不少一单也没卖出去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看来，李鬼冒充李逵的如意算盘打得并不好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><img src="https://ngbjimg.xy599.com/176078547762db2e8f874706.61176433.png" referrerpolicy="no-referrer"></span></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong><span>黑红不是红</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从诞生之日起，东方绿选显然就打定了走黑红的路线。但</span><strong><span>刻意的抄袭本身就算披上助农的滤镜也改变不了龌龊的内在。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从直播间动辄1600+的人气来看，相比“正主”东方甄选开播前两个月直播间时常维持三位数、甚至个位数的惨淡人气，那是相当成功了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/110583959463221b63d1f543.06432188.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：@东方YOYO</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而“成功”的背后却尽是嘲讽与谩骂，网友显然不吃这一套了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友涌进直播间的目的大概率不是买东西，虽然有个别网友因为低到骨折的价格而下单，但多数人只是想看看东方绿选有多无耻。直播间里，“抄袭”、“东西效颦”的字眼屡屡刷屏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>许多正义的网友啥事不干，进直播间就是举报，一直举报到停播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有意思的是，</span><strong><span>“文化”作为东方甄选的杀手锏，也成了网友们揶揄东方绿选主播的主要方式。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/194832310063221b642c5521.51663580.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：微博网友@楚少主</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>看热闹的网友纷纷拿董宇辉讲的“知识点”发难：</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>白居易有几套房子？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘邦是哪里人？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>葡萄酒最早产自哪里？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>男士为啥没有专用奶粉？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一连串的提问，弄得东方绿选主播磕磕巴巴满头大汗，更有董老二的铁粉起哄挑逗主播:别废话，快上小作文。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真是应了那句话，</span><strong><span>撞衫不可怕，谁丑谁尴尬。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但面对自己“像素级”的抄袭，东方绿选显然不知悔改。不仅在直播间怒怼替东方甄选抱不平的网友“是不是东方甄选欠你房贷，老提它”，还装傻充楞说“这就是我的个人风格”，直言不知道“东方甄选”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8164650563221b645dfb46.82187492.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：抖音网友</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>某种程度上来说，如果东方绿选直接大方承认就是在模仿东方甄选，或许部分网友还会看在都是公益助农的份上对其产生一丝丝好感，但东方绿选不仅极力否认自己“皇帝新衣式”的抄袭，还对抄袭对象不屑一顾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，</span><strong><span>东方绿选的直播只能是一场跳梁小丑式的闹剧，自取其辱！</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟，消费者已经用惨淡的销量给出了自己的答案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><img src="https://ngbjimg.xy599.com/79261823162db2e91cb3478.62060819.png" referrerpolicy="no-referrer"></span></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong><span>越火越危险</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>诚然，东方绿选的模仿已经不仅仅是个拙劣的模仿闹剧，而是一个不折不扣的法律问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如很多网友调侃所说：</span><strong><span>东方甄选法务部已经在赶来的路上了。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有律师表示，这已经构成了侵权，并愿意为东方甄选无偿代理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于东方绿选的所作所为，</span><strong><span>为啥东方甄选至今没回应？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在我看来，对于一心助农的东方甄选来说，无意去和一群小丑争辩，回应本身可能正中下怀，反而助长了蹭流量的热度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而更重要的一点在于，东方甄选有山寨们难以攻破的壁垒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以董宇辉为代表的主播们本质上是创立了一种新的场景，观众会与董老师直播时讲的故事产生共鸣。像有些直播片段中，他们会用故事来激励观众，特别是一些年轻人。这种为深层次的共鸣买单的行为，在原来那些带货直播中极其少见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实放眼整个抖音，东方甄选的遭遇并不是特例。李子柒火了，“越南李子柒”出现了——妆发、神态傻傻分不清。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/203806798063221b64a4e545.69547250.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：越南版李子柒抖音</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>张同学火了，帅哥鸟农等一批类似的三农账号顺势而起。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真探唐仁杰火了，特厨隋汴就用同样的BGM与剪辑风格成了百万大号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>摆货小天才火了，大量模仿、借鉴他的账号应运而生。在抖音平台内，有不少摆货小天才的高仿账号，他们用着同样的头像甚至同样的名字。</span></p>
