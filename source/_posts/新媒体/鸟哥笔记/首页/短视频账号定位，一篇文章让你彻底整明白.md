<section data-role="outer">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">说起做抖音，总会听到一个词，账号定位！而且很多人会说这个非常非常重要。</span></p>
<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那到底什么是账号定位呢？怎么做好账号定位呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单来说就是我们做一个抖音账号不能今天发美食明天发天文学后天发生物学，这样做的后果就是用户不知道你是干嘛的，系统也不知道该把你的视频推给谁。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以如果你想做抖音一个账号只能专注一个领域，也就是所谓的垂直领域。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>听完似乎知道了什么，又好像什么也没知道，具体怎么去做呢？有没有详细的教程呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>放心，接下来就跟大家好好说道说道账号定位这个事情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做好账号定位需要4个步骤</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、垂直领域</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2、内容深耕</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3、用户思维</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>4、有方向的坚持</span></strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>垂直领域</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么叫垂直领域，以教育行业为例，教育行业包括了K12教育、职业教育、学历教育、语言培训、职业技能培训等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>那么垂直领域就是这个大行业下的某个细分领域</span></strong></span><span>，比如职业技能培训中的短视频运营技能培训。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单来说就是我们做<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E5%8F%B7" target="_blank">抖音号</a>只定位一个领域的内容，只定位一种人群，与这个领域和这类人无关的内容不要在这个抖音上进行分享。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果我们今天分享学历教育明天分享K12教育，那么关注学历教育的可能会取关你，反之也是一样的，能够长期引起人们关注的账号基本上也都坚持某一领域的垂直账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不信你试试，如果你正常刷到的一个主播之前一直在分享怎么考研，而你也很感兴趣，后天忽然改成怎么教育孩子了，你会不会取关？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/210067450062ff61b6937310.38602441.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>记住：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一个账号只做一个细分领域，一个账号只做一类目标人群，垂直领域越细分竞争压力越小，账号难度越低，后期变现也会更容易。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过如果你是非常冷门的本身就没几个人知道的，且全网都没几个人在做的那种行业，可以不搞的这么细分，他本身可能就已经很细分了。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>内容深耕</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>垂直领域确定好了之后我们要做的就是内容深耕，在抖音，什么样的内容吸引什么样的人群，假如我们还是做短视频运营技能培训的，那么我分享的内容可能就是短视频运营这个岗位前景、薪资待遇、能力要求这些。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是你会发现这些东西一共也发不了几个视频，如果去做内容深耕呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对短视频技能培训感兴趣的人可能分为2种，</span><strong><span>一种就是没工作想尝试这个岗位，另外一种可能就是刚刚入职但是能力欠缺，遇到问题解决不了。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于第一种我们可以分享短视频职位的高薪案例，未来的人才需求和发展潜力，对于第二种我们可以分享抖音账号播放差、没互动、没涨粉等等问题的解决方法，怎么写好抖音脚本，怎么快速剪辑高清视频等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/83919976162ff61b7592a89.29356264.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>讲的更深入一点，我们</span><strong><span>可以尝试分析用户的需求和场景</span></strong><span>，一个刚毕业的大学生，本身学的专业找不到好工作，想转岗这个时候他想要的是什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>能够快速学会且能够快速找到一份薪资还不错的工作的机会，我们可不可以顺着这个思路去输出我们的短视频内容呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了这个还有没有其它场景呢？比如刚刚入职老板说能做出一条播放10w+的视频就给我转正，但是目前播放只有500，这个时候我能不能顺着这个思路去分享内容？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总结一下，</span><span><strong><span>就是在垂直领域下结合用户需求和场景进行内容输出。</span></strong></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>用户思维</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>账号的定位是我们的短视频运营方向，深耕内容是我们的落地方案，用户思维是什么？用户思维决定了我们能否做好这个账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先我们要做的平台是短视频平台，这个平台的特点是什么，用户喜欢刷短视频的原因是什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好玩，有趣，简单，新鲜对不对？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以我们在分享内容的时候也要满足用户的这些心理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，</span><strong><span>不能太复杂，越简单越好，越简单越容易懂越能引发传播。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还要有什么？</span><strong><span>有槽点</span></strong><span>，比如选两个公鸡蛋，凌晨两点半的午餐，有些人故意说错话或者说反话，其实都是为了制造槽点，有槽点就有互动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家还喜欢看什么？有反转的内容，让人永远猜不到结局，你以为它是武侠电影结果它是教做菜的，你以为它是臭要饭的，结果它是广东包租婆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人喜欢把自己的专业直接搬到抖音上，还说我的内容质量那么高，为什么没人看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先不说有几个人能看懂，</span><strong><span>在短视频这个平台上有几个长篇大论是能让人真的看下去的呢？</span></strong><span>当然我们不排除个例非常优秀的视频内容，但是能够做到那种地步的估计也不会来纠结视频没播放了。 </span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>有方向的坚持</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人都是刚开始做信心满满，感觉自己做了就能百万粉丝，想好爆量之后怎么去找老板谈分红，或者自己做账号起飞之后买几栋别墅，一定要找个漂亮妹子做女朋友。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>兄弟，现实一点，脚踏实地，前两周过去没有一条播放过万的视频时也不要放弃，我有个朋友坚持到半年的时候始终在十几万的播放徘徊，自己都要放弃了，都收拾好东西要回老家了，结果第一条百万播放就出来了，现在一个月收入5w+。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二种情况更多，本来我选了短视频技能培训这个领域搞了两个月感觉没效果，不做了，隔壁搞直播培训的一个月涨粉5w我要做直播领域，直播领域做了两个月，没什么效果，反观那些搞元宇宙的都起飞了，结果转去元宇宙，一年下来啥也没做成，反而自己关注的那些都做起来了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>仔细想一想这不就是我们上面说的这些步骤吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>找到垂直领域，深耕内容，并且站在用户角度考虑，只要你能坚持下去，最后都能有所收获。</span></strong><img src="https://ngbjimg.xy599.com/156156171962ff61b6df4182.90592719.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/186428012962ff61b7c6dfd7.87373767.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>账号定位越清晰，做抖音号相对更叫容易，涨粉速度更快，且变现的效果也会更高。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>定位的方向越多，运营的时候越麻烦，今天一个方向明天一个方向，后面越做越乱，最后你会发现白做了。</span></strong><span>所以说账号定位真的真的很重要，在这一步中我们定位的是我们的目标用户，根据用户需求去输出内容，而不是通过账号定位直接定位内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这句话一定要懂。</span></p>
