<h2 style="border: 0px; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; font-size: 18px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #212629; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; line-height: 30px; text-align: left;"><span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">一个梗，也能撬动千万流量</span></h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">你有想过“嘲讽”会成为流量密码吗？</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">近期B站掀起一阵“嘲讽流”，在众多UP主的创作下让B站用户见识到了一场花样嘲讽盛宴。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p6.itc.cn/images01/20220629/25a411b472a6495d9fe26e25e48dd3ce.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-B站</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">嘲讽梗一开始是由《我的世界》中人物的动作和《吉他英雄3》中的摇滚乐组合而成，小人反复做着一个动作，鬼畜中带着黑色幽默的感觉让B站用户不断“上头”，这也就带动了更多嘲讽二创的诞生。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">UP主「快乐的台长」演绎了一场毕业线上答辩，在论文答辩环节为大家上演了一场“真人嘲讽秀”。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p7.itc.cn/q_70/images01/20220629/53594a2fa4fc44179bbc0dca4f6bc3d8.gif" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-<a href="https://www.niaogebiji.com/tag/B%E7%AB%99UP%E4%B8%BB" target="_blank">B站UP主</a>快乐的台长</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">看过会动的2D小人鬼畜，没看过真人也能这么鬼畜，利用模拟线上答辩的形式为正在处于毕业季的众多大学生带来一场意想不到的“行为嘲讽”。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">作品发布截至目前不仅登上了最新一期的【每周必看】，更是揽收了高达<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">1539.2w</span>的播放量，互动数据中点赞数更是超过百万，点赞粉丝比翻了一倍还多，作品的传播水平远超UP主平均水平。</p>

<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">更值得一提的是<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">短短10天UP主就涨粉17.5w</span>，并且在发布当天<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">一小时内最高能增长50.6w播放量</span>，可以说UP主这一次行而有效地吃到了流量红利，在高峰入局，收获高质量的数字资产。</p>

<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">不仅如此，还有UP主将“嘲讽”和“文学”挂钩，「代坤家忠厚老实」就将文言文《两小儿辩日》和嘲讽小人进行结合，根据原文词句对视频进行剪辑，不仅嘲讽鬼畜感拉满，连晦涩难懂的文言文都变得有趣起来。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p3.itc.cn/images01/20220629/93077a7bd60d4e2c975da06f73147a8c.gif" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-B站UP主代坤家忠厚老实</span></p>

<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p9.itc.cn/images01/20220629/e81b35e44a804a0498d01495221795a5.png" referrerpolicy="no-referrer"><span class="img-desc" style="border: 0px; margin: 14px auto 20px; padding: 0px; display: block; width: 528px; line-height: 20px !important; color: #919191 !important;">来源-果集·飞瓜数据（B站版）</span></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">可以看出这是B站独有的社区特点——<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">梗文化</span>，有人造梗就有人玩梗，Z世代活跃的B站自带这样的氛围。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">玩梗式创作能在平台<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">迅速汇集流量</span>，这也是众多UP主常常二创的原因，在B站圈层独有的幽默氛围下，及时跟踪玩梗趋势是非常重要的一点。</p>

<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">既然是站内主流，那相关内容的延伸也是必然的，就以近期热门的4月番《间谍过家家》为例，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">982.4w追番、2.6亿播放量</span>，还未完结的情况下无疑是季度黑马。</p>

<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">全文总结下来可以看出B站的流量趋势与其他平台有着<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">天壤之别</span>，在这里，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">用户更热爱圈层内容，创作者更忠于Z世代语言</span>，因为B站从起始就附有二次元的标签。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">所以第一点梗文化是需要创作者以及品牌敏锐的嗅觉去发现，也需要富有创意的想法去辅助作品的诞生，但很重要的一点是，<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">不论作品高级与否，创意始终胜出</span>，所以这方面<span style="font-weight: bold; border: 0px; margin: 0px; padding: 0px;">更支持中小型UP主</span>积极参与，有助于账号快速成长。</p>
