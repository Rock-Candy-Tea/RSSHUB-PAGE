<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><br style="box-sizing: border-box; padding: 0px; margin: 0px;">但与此同时，同是美妆的护肤产品，则依靠“强周期”等特性，在2022年依旧焕发着生机，而头发护理等个护产品，也因消费者护理理念的提升，而越加受到重视。那么面部护理、头发护理赛道具体现况如何，这两个赛道，在快手<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%95%86" target="_blank">电商</a>平台又呈现怎样的态势，有哪些新的趋势？</p>
<blockquote style="box-sizing: border-box; margin-top: 10px; margin-bottom: 10px; padding: 5px 10px; line-height: 1.6; font-size: 14px; background-color: #f7f9fe; color: #737b91; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; border-left: none !important;">

<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; font-size: 16px !important; line-height: 1.5;">2022年7月4日-7月10日，快手电商面部护理GMV TOP 3000商品数据信息；头发护理GMV TOP3000商品数据信息。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; font-size: 16px !important; line-height: 1.5;">数据来源：本文数据来自壁虎看看快手版，以下快手内容均以壁虎看看数据为基准，仅供参考，如若转载或数据合作等，请关注公众号【壁虎快跑】，咨询运营官方方。</p>
</blockquote>
<h2 style="box-sizing: border-box; margin: 25px 0px 20px; color: #212629; font-weight: 600; padding: 0px 0px 0px 12px; line-height: 30px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-size: 18px; text-align: left;"><span style="box-sizing: border-box; font-weight: bolder; padding: 0px; margin: 0px;">面部护肤持续增速，头发护理规模紧跟其后</span></h2>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">在面部护肤行业，根据最新数据显示，供需双端共同推动中国面部护肤市场快速发展，2020年中国面部护肤市场规模已达2815亿元，预期5年后该市场将达4655亿元。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; padding: 0px; cursor: pointer; max-width: 100%; display: block; margin-left: auto; margin-right: auto;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220715/f6a1878f-cf70-4239-bab1-e6d305ea9cb4.jpg" width="785" height="285" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">再以美妆护肤全局来看，壁虎看看收集资料显示，中国整体美妆护肤行业规模约5000亿，其中基础面部护肤以及身体/头发护理品类规模较大，位列第一第二。随着消费者“求美”、“悦己”等意识的觉醒，护肤逐渐成为了大多数消费者的“硬需求”，一天两次的使用频率必不可少。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">此外，护肤产品作为一种强周期产品，消费者对其的回购频率远超香水等可选性产品，且作用在身上的护肤产品，尤其是面部护理产品，消费者一旦确定适合品牌，其便会在接下来很长的一段时间内，周期性复购该品牌产品，客户粘性较高。多种因素驱动，打造了护肤品类长久不衰的盛况，而除了基础面部护肤市场，身体/头发护理也是值得关注的重点。</p>
<h2 style="box-sizing: border-box; margin: 25px 0px 20px; color: #212629; font-weight: 600; padding: 0px 0px 0px 12px; line-height: 30px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-size: 18px; text-align: left;"><span style="box-sizing: border-box; font-weight: bolder; padding: 0px; margin: 0px;">快手电商新趋势，功能性洗护突围红海</span></h2>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">在快手电商平台，本次壁虎看看统计了2022年7月4日-10日面部护理GMV TOP3000相关商品数据，以及头发护理GMV TOP3000相关商品数据，来进行分析。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; padding: 0px; cursor: pointer; max-width: 100%; display: block; margin-left: auto; margin-right: auto;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220715/ecf9e45d-1c17-479b-8279-c74ac47c1236.jpg" width="861" height="573" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">数据显示，在上周面部护理类目中，基础面部护肤品类占据前三。其中“面部护肤套装/礼盒”仍以26.2%的GMV占比位列TOP1，但与众不同的是，“面部精华”在上周GMV仅次于护肤套装，面对护肤套装的长期霸榜态势不遑多让，GMV占比以25.6%拿下第二，“面膜”则屈居第三。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; padding: 0px; cursor: pointer; max-width: 100%; display: block; margin-left: auto; margin-right: auto;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220715/9c1dc52c-1c3e-4d4a-877d-8eb7ebef562f.jpg" width="791" height="495" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">以时间为单位进行统计，壁虎看看发现，面部护理各细分类目都有不同程度的提升，其中唇部护理上周GMV环比6月27日-7月3日上升了212.1%，而眼部护理则环比上升了123.5%，唇部、眼部护理涨幅明显，快手用户护肤越加局部化、精细化。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; padding: 0px; cursor: pointer; max-width: 100%; display: block; margin-left: auto; margin-right: auto;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220715/51164b6c-794f-4e74-aa3c-4036516b0279.jpg" width="849" height="501" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">而在TOP4品牌的市场占有率方面，壁虎看看数据显示，在快手电商平台，面部护肤套装品类CR4=48.3%，远超面部护理品类CR4=24.2%，“面部护肤套装”相较于“面部护理”市场集中度更高，竞争更为激烈，壁虎看看建议，中小商家最好不要以套装销售为主。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; padding: 0px; cursor: pointer; max-width: 100%; display: block; margin-left: auto; margin-right: auto;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220715/3ebbc09f-6c3e-469e-ad58-a688dcea9ab3.jpg" width="987" height="657" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">而在头发护理类目中，数据显示，洗发护发以41.5%的GMV占比稳压洗发套装，拿下TOP1，相对于“套装”，洗护“单品”仍更受快手用户欢迎，而在套装方面，“洗发+护发”套装最为常见。  </p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; padding: 0px; cursor: pointer; max-width: 100%; display: block; margin-left: auto; margin-right: auto;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220715/e3da8b85-fab7-40e4-9c30-7e271fb482d6.jpg" width="891" height="593" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">在洗护产品的成分方面，随着消费者安全意识的提升，成分越加收消费者重视，<span style="box-sizing: border-box; font-weight: bolder; padding: 0px; margin: 0px;">数据显示，“中药、植物”等纯天然成分备受快手用户青睐。</span></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; padding: 0px; cursor: pointer; max-width: 100%; display: block; margin-left: auto; margin-right: auto;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220715/bc0fc9ff-2b99-44c6-8f56-60288c6a2220.jpg" width="910" height="606" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">而随着社会压力的增大，以及人均消费水平的提升，现代年轻人对头发护理的要求，已经不再局限于简单的清洗，而是有了进一步的深层次需求，在洗护功能方面，壁虎看看数据显示，在快手电商平台中，“控油蓬松”“柔顺润发”“防脱育发”成为上周热销洗护功能。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"> </p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;">众所周知，商家为在市场中脱颖而出，都会在其包装、成分、价格、应用场景等方面寻求差异化。在洗护品牌商家中，除了老资格商家品牌依靠过去建立起的<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">“品牌价值”</span>，以及<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">“老牌情怀”</span>让消费者买单，如长年霸占线下商超、小卖铺洗护区的海飞丝，以及依靠网友“野性消费”再次爆红的蜂花护发素。</p>
<p style="box-sizing: border-box; margin: 10px 0px; padding: 0px; line-height: 1.5; color: rgba(0, 0, 0, 0.65); font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px !important;"> </p>
