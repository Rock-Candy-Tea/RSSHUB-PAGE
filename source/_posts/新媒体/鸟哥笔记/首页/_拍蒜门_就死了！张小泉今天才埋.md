<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年7月的一出“拍蒜门”让老字号张小泉站上风口浪尖，体验了一把顶流翻车的滋味。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>如今一个多月过去，张小泉再次迎来了“至暗时刻”。</span></strong></span><span>据最近披露的半年报显示，张小泉上半年实现营收4.39亿元，同比增长32.79%；净利润3428.95万元，同比下降22.52%。</span></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/91916335863163df93d27f4.85458262.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图片来源：@北京商报微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>增收不增利，这是2017年至今张小泉始终面临的尴尬。只是今年因“拍蒜门”显得格外瞩目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>明明“拍蒜”那天就死了，财报出来了才埋！</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/381440126233b25ace1df6.64458916.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>消费者教“做人”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于此次净利润下滑，张小泉官方给出的解释是受疫情对物流交付的影响，加上阳江张小泉正式投产后尚处于产能爬坡期，单位制造成本偏高等因素所致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但明眼人都知道，</span><span><strong>吞噬利润的与那把拍蒜拍断的刀不无关系。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>彼时，“断刀”事件发酵后，彷佛推倒了张小泉的多米诺骨牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/122580498063163df95d9468.95502363.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图片来源：黑猫投诉平台</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>陆续有网友晒出张小泉旗下的刀具各类质量问题的图片、视频。一时间，对自称全网“菜刀”行业销量第一的张小泉连主业都做不好的质疑此起彼伏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>舆论的质疑最终都反映在了销量上。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是当时“拍蒜门”刚爆发的那段时间，张小泉在抖音的官方授权帐号“张小泉厨房电器旗舰店遇见理想家”出现了持续掉粉的现象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/181671055763163df98a62c7.57266801.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图片来源：@张小泉厨房电器旗舰店遇见理想家</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播间日销售额也不断下滑，从日销售额20多万到15多万再到12万，每天都在以万为单位下降。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今近2个月过去，该旗舰店粉丝虽然出现了微反弹，从193万增长到了201万，</span><span><strong>但销售却肉眼可见的下跌。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据第三方直播数据统计平台灰豚数据显示，近3个月，该旗舰店新增粉丝56.1万，销售额2756.6万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但最近1个月，该旗舰店新增粉丝数仅为7.4万，销售额483.2万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/29014241363163df9cd3646.26077025.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图片来源：灰豚数据小程序</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，</span><span><strong>最</strong></span><span><strong>近一个月张小泉差强人意的表现正是“拍蒜门”最直接的影响。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>讽刺的是，时至今日翻开该账号的评论区，与那本就可怜的播放数据相对应的是，寥寥无几的评论大多还都是网友关于“能否拍蒜”的嘲讽。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>都说互联网是健忘的，但对于百年老字号张小泉来说，一柄断刀彷佛成了永远的笑柄。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9225241376233b25c6ec6a4.50995726.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>营销与代工惹的祸</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为400年的老字号，张小泉在利润上的下挫以及互联网上来自年轻人的玩梗与嘲笑，让此前不遗余力的品牌年轻化营销多少有了些付诸东流的意味。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>财报数据显示，2021年，张小泉销售费用投入1.17亿元，同比增长41%，这一数据是研发投入的5倍。2022年上半年销售费用投入高达7552万元，同比增长70.89%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这些钱都干了些什么？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了“年轻”，张小泉不仅跟随元宇宙的风潮，推出匠师道系列的数字藏品，还搭上国潮风，与MILL 塔牌王星记联袂打造“仲夏”新国风主题展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/59155635363163dfa41d783.41756349.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图片来源：@张小泉微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在更早的时候还与变形金刚玩跨界，冠名《中国有嘻哈》等潮流综艺节目，并在《中国有嘻哈》节目中推出了“泉叔”IP形象，推升张小泉在年轻人中的品牌认知度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/182381148863163dfa6a4317.71356411.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图片来源：@张小泉微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>此外，张小泉还频繁出现在抖音、小红书等社交媒体上，以内容种草进入年轻人的生活。</span></strong></span><span>今年一季度，在疫情影响下，张小泉加大了抖音等电商平台推广力度，致使线上平台流量费上升、品牌推广费等增加，销售费用激增74.37%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/139444850763163dfaae9255.30533143.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图片来源：小红书</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，凡此种种，</span><span><strong>终究抵不过一把不能拍蒜的菜刀的“杀伤力”，让此前的年轻化努力归了零。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说“拍蒜门”是张小泉营销失灵的后果，那人们更愿意将“贴牌代工”自然而然地归到张小泉塌房的前因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>翻开张小泉当年的上市招股书，2018年至2020年，张小泉代工产品产量分别为3039.22万件、2980.78万件及3301.18万件，分别占全部产量的76.51%、75.17%、77.15%，已远超自产产量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>也就是说，消费者买到的张小泉大概率是由代工厂生产。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过代工生产能迅速实现产品供给。然而，过度依赖OEM代工，也存在一个硬伤，那就是品控。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时人们似乎给“拍蒜门”事件找到了一个最合理的解释。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为消费者难以接受自己花大价钱买的传承了400年的老字号刀具，像被玩环的南极人一样来自一间贴牌代工厂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实也正如人们预想的那样。代工生产模式下，张小泉质量问题频发。在黑猫投诉平台上，关于张小泉菜刀质量的投诉多达数十条，投诉内容包含工具断裂、卷刃、菜刀为残次品等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些显然就像待燃的“雷”，保不齐哪一天又来一出“XX门”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16900100706233b25e365e39.91683873.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>老字号的“船”翻了</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时至今日，张小泉显然已经不香了！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不止在于崩塌的口碑，在资本市场上也交出了不及格的答卷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年9月，张小泉挂牌上市，上市首日开盘大涨394.06%，总市值53.18亿元。然而一年过去张小泉股价相较于最高点已跌去近6成，蒸发约30亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>资本市场的糟糕表现背后</span><span><strong>或许是人们对“老字号”滤镜的失去。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/149070299563163dfb0c2b26.82219335.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△</span><span>图片来源：张小泉官网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2006年，张小泉剪刀锻制技艺被列为第一批国家级非物质文化遗产，也是最早一批被评为中华老字号的民族品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>然而虽然贵为民族品牌，张小泉自身却处处崇洋媚外。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拍蒜门事件后，张小泉总经理之前的言论被扒了出来。他大肆赞扬米其林厨师切菜方式，还否定了中国人切菜用刀的方法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/44553393663163dfb3e40c4.91691678.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>△</span></span><span>图片来源：微博网友</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时人们才诧异地发现，我们引以为傲的400年老字号品牌原来表面上恪守“良钢精作”的祖训， 骨子里推崇的却是西方的烹饪文化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这多少令人有些崩溃。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这与西方那些百年奢侈品牌们在中国市场高高在上、爱买不买的高傲品牌调性截然不同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它们出奇葩单品、价格令人瞠目结舌，但它们有一点不变——坚守匠心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>但拍蒜就断、还找代工生产的张小泉显然为老字号的“污名化”又添上了浓墨重彩的一笔。</span></strong></span></p>
