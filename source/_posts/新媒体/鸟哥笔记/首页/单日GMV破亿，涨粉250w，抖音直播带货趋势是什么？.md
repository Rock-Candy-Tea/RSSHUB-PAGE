<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">其中，近3个月带货约20亿的<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">@东方甄选</span>稳定发挥，第三次斩获带货榜榜首，8月销售额6.3亿；回归后人气不减的<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">@疯狂小杨哥</span>蝉联榜二，直播销售额4.28亿；榜三则是排名上升7位的<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">@衣哥</span>，直播销售额3.83亿。</p>

<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">在他的直播间里，平均有2~5款价格低于1元的秒杀福利爆品，其销量均在1w单以上，<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">高性价比的商品往往能直接为直播间带来流量</span>，另一方面，主播深谙用户害怕错过优惠的心理，<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">直播间的福利商品往往上架少且售卖时间短</span>，犹豫就可能错过，提高了用户下单的积极性。</p>
