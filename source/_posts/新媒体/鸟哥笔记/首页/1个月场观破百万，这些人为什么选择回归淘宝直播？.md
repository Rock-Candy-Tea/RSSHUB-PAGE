<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在大家把更多目光给到新平台的时候，作为淘快抖中的老大哥，淘宝直播为什么能吸引这些主播？淘宝直播有哪些独特的吸引力？今天我们就来聊聊带货主播的平台选择问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>淘宝直播加速争抢优质主播</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像罗永浩开播前曾遭到各大平台争抢一样，能带货、能吸粉的主播，对任何平台都是香饽饽。主播是平台确立自身地位、争抢市场份额的排头兵，平台也愿意为之付出更多资源。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“Demi”等愿意归淘，“一栗小莎子”等愿意来淘，背后离不开淘宝直播各种政策、资源的吸引。比如近期淘宝直播推出的“2022淘宝直播回家计划”，覆盖了20多个行业类目，提供了不同量级的流量奖励，核心目的就是为了吸引主播回淘开播。据悉，符合条件的主播，首场直播最高能获得30万PV流量奖励。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“为了帮助主播重新开播，淘宝直播不仅每个月都有流量扶持，每个季度成交优秀的主播还能得到7-100万元的现金奖励。另外像我这边，官方还提供了一对一的小二对接和供应链推荐。”“Demi”回复。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，在淘快抖三足鼎立的当下，各个平台除了孵化新主播做增量外，另一个重要工作就是从其他平台吸引更适应自身生态的主播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以淘宝为例，4月14日，淘宝直播负责人道放表示：“上个季度，我们是第一次真金白银的拿现金和流量帮助MCN给平台引入主播。”他透露，单机构预估最高获奖金额190万，获奖机构的比例达40%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“你们能想到的（全网所有的短视频或直播达人），我们基本上都聊过。”</span></strong><span>淘宝直播新生态事业线负责人虚罗在接受媒体采访时说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月1日，淘宝直播则相继推出新领航计划、引光者联盟、超级新咖计划、源力计划等四项面向不同梯队达人、主播、产业带商家的扶持政策，希望能支持2000个账号成交翻倍，培育100个粉丝超过百万的内容账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/168402206632985c33594b9.16486372.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：淘宝直播</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播电商发展到现在，差不多进入了存量竞争时代，平台们也需要在争抢主播方面开展“军备竞赛”。毕竟，对主播来说，大方的平台总是有吸引力的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>淘宝直播如何保证主播利益最大化？</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>平台是否大方很重要，但平台是否值得、是否合适更重要。在平台选择方面，主播们总希望能完整评估各个平台价值，选出最适合自己的平台，让自身利益最大化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为三极之一，淘宝直播的独特价值是什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第一，电商粉多。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谈及淘宝直播，“Demi”认为：“淘宝胜在粉丝精准度高。消费者打开淘宝基本就是冲着买东西去的，所以转化率很高。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在内容电商时代，主播往往需要先通过内容吸引粉丝，再从中筛选出意向消费者。但在淘宝直播，粉丝通常有更明确的购物需求。这很大程度上能节省主播筛选粉丝的时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝MCN机构纳斯相关负责人陈雷认为，精准电商粉是淘宝直播的优势之一。相关数据也显示，目前淘系88VIP用户已经达到2500万；淘宝天猫年消费在万元以上的消费者达到了1.23亿以上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这意味着在同等情况下，主播在淘宝直播的销售转化率更高，单个粉丝也更值钱。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第二，基建成熟。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“淘宝毕竟是购物平台起家，包括软件使用感、售后服务等购物体验都更好，我觉得淘宝直播的独特价值在于，经过多年的积累，沉淀出的一整个流程的购物服务。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>结合多个平台的使用经验，淘宝主播“Nicole老爷”给出了自己选择淘宝直播的原因之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上<strong>，不论是主播还是商家，都希望平台能具备完善的直播设施、公平的直播机制以及丰富的直播玩法，帮忙营造一个安心卖货的环境。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/141911149632985c421e284.23230353.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>淘宝直播界面。图源：淘宝</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这个角度讲，相比近几年加速补课、疯狂基建的抖音、快手，淘宝直播因为先发优势，基建成熟仍然是其吸引主播的砝码之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以商品丰富度为例，不论是抖音还是快手，近几年一直在商品供给上下功夫，但对于淘宝直播来说，“万能的淘宝”让它不担心无法满足消费者的购物需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第三，稳定性强。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“淘宝从1万到几百万需要一些时间，抖音如果运气好遇到一个爆款，一下子就冲上去了，但不好的地方是内卷严重，可能爆完就卖不动了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回顾在抖音的4个月，“Demi”最不适应的就是抖音流量的不稳定以及过快的节奏。“我本以为一个爆款能卖几个月，但爆了之后别人就开始仿，可能卖三四天产品周期就过去了。3个月亏损了300万！这让我有些不能接受！”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“Demi”选择重回淘宝，就是觉得淘宝直播在粉丝粘性、品牌打造方面会更友好一些。“粉丝是可以累积沉淀的，并不会产生太多落差。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一定的私域属性，再加上相对稳定的平台生态，让淘宝成为不少主播的长期商业化阵地之一。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，“Nicole老爷”认为，“现在做抖音不懂规则是完全没戏的。懂规则的人大部分是机构，他们天天研究平台”，如果主播没办法适应这套规则，会很难。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“现在淘宝的竞争相比以前也更小。”陈雷补充。不少业内人士也表示，现在淘宝直播反而相对没那么卷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去一年，在各大平台的冲击下，淘宝直播仍然有2.5万个年成交超百万直播间，日均ARPU值超过30元，人均购买超20次，很大程度上是因为淘宝直播仍保持着自己的独特优势。这也是不少主播愿意到淘宝开播的原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总得来说，淘宝直播也许不是流量最大的平台，但却可能是<strong>稳定销售转化率</strong>最高的平台之一。此外，李佳琦、薇娅的停播，也客观上释放了更多流量，给中小主播们带去了机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>什么样的主播适合到淘宝开播？</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在直播电商的第一阶段，主播们更多会跟着流量走，但当行业进入稳定期，找到适合自己的平台就显得尤为重要。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从平台生态氛围的角度讲，抖音兴趣更浓，快手信任更重，淘宝则始终以电商为根基。就像“Demi”在尝试过后觉得更适应淘宝一样，<strong>在经历流量红利期的广泛尝试、磨合匹配后，不同主播大概率会各安其位，选择更适合自己的平台。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/738025564632985c46fe3c3.79014393.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“2022淘宝直播回家计划”第一期榜单  </span><span>图源：淘宝直播</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>什么样的主播适合到淘宝开播？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“Demi”建议，如果是有稳定的货源和电商团队经验的主播，都可以去淘宝直播做尝试。比如“Demi”就创立了个人服饰品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“Nicole老爷”举例说：“代购就很适合在淘宝做直播，虽然流量低，但流量准，因为来淘宝搜索的人就有这个消费心智。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝直播电商粉多的特点决定了，<strong>只要主播的货品有足够优势，能把淘宝直播的流量高效利用起来，淘宝直播的投入产出比会更加直接。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果是自带流量但缺乏电商经验的主播，<strong>货品池丰富、平台机制成熟的淘宝直播则是一个相对省心的商业化变现渠道。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于“一栗小莎子”来说，淘宝直播除了种草属性更强外，电商体系稳定、货品SKU丰富也是一大吸引点。“一栗小莎子”的幕后运营方在媒体采访中提到，达人带货需要大量品牌资源，淘宝直播因为背靠淘系，在供应链方面帮助很大，尤其服饰和美妆还是淘系沉淀最成熟的领域。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1240898028632985c4d0a4d6.57034798.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>直播中的“一栗小莎子”。图源：淘宝</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，不同阶段的主播有着不同诉求。就像“Demi”回归淘宝直播是因为“意识到淘宝能提供更稳定的增长”，如果能在淘宝直播站稳脚跟，那淘宝直播的生态特点会让主播获得想要的经营稳定性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总得来说，<strong>不同平台有不同的规则玩法，有的人玩得好，有的人玩不好，有人则希望能多平台布局。</strong>主播需要做的就是找到自己能玩得好的平台。如果一个主播有货、有流量，同时想追求稳定，那淘宝直播会是备选之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>淘宝直播当然不会适合所有人，但对于能适应淘宝直播生态的主播来说，淘宝直播就是最好的选择。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我感觉每个平台都有自己的特点，要根据自身的情况去选择，而不是说一个平台绝对怎么样。”“Demi”觉得，主播最重要的还是把人货场做好。</span></p>
