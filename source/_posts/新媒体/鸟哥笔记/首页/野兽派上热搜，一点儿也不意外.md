<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一瓶售价420元的香水，竟与他牌仅售29.9元的香水出自同一家公司。细心网友的无意发现，让奔着品质花了高价买单的自己很受伤。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，一则#</span><span><strong><a href="https://www.niaogebiji.com/tag/%E9%87%8E%E5%85%BD%E6%B4%BE" target="_blank">野兽派</a>香水和名创优品同一代工厂</strong></span><span>#的消息冲上<a href="https://www.niaogebiji.com/tag/%E7%83%AD%E6%90%9C" target="_blank">热搜</a>，将“野兽派”这个以“讲故事”营销成功出圈的古早网红，推向了舆论中心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70077011662540cc779d811.91220037.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月8日，新浪新闻在微博上转发了一篇《野兽派的高端“谎言”：代工香水溢价14倍 质量问题不断》的文章。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1483145253631e34f9ed3cf7.09070978.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>文中直指野兽派售价420元的桂花乌龙香水，与名创优品售价29.9元的水蜜桃乌龙香水，都由同一个化妆品公司生产。　</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由此，引发了网友们关于野兽派凭啥卖出大牌价，高价和品质不匹配的讨论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大多数网友认为，</span><span><strong>溢价肯定是有的，但同一个生产厂家并不代表配料和成分一样</strong></span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1673858245631e34fba90689.25264432.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/73054371631e34fc084b71.60385074.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>像雅诗兰黛、兰蔻这些大牌护肤品，也都与国内品牌在同一个代工厂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1409218741631e34fc3ec0a1.07053931.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而很多说自己与国际护肤品同一条生产线的微商品牌，大家也不敢去买。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1796142586631e34fc6ac781.94216547.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也有网友对野兽派的设计表示肯定，认为用来送礼非常合适。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1469615431631e34fc971bd6.18382520.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直以来，野兽派都擅于通过营销出圈。以至于冲上热搜后，被质疑过分强调营销，忽略了产品本身。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>品牌的生命力来自于品质</span></strong></span><span>。适度的营销不会让人反感，但如果忽视品质，则有可能遭到反噬。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1249446542631e34fcccf469.48673936.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137231867662540cc8699195.09342959.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>野兽派最初不过是一家线上花店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当其他品牌都在以“卖菜”的价格和方式卖花时，野兽派却用讲故事的浪漫煽情，在不被看好的线上卖花模式中自成一派。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2011年，没有网站、APP的野兽派，上线微博开始营业。想要订花的网友只能私聊老板娘运营的官方微博，并讲述自己的情感故事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后，野兽派根据对故事情节的“领悟”，来搭配定制仪式感满满的专属花束。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1489180880631e34fd1caa52.28083411.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：野兽派微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种“</span><span><strong>说一个故事，为你定制一束鲜花</strong></span><span>”的开放共创模式，拉近了用户和品牌之间的距离，比品牌输出的内容更容易唤起用户的情感共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而野兽派以一个又一个匿名发布在官微上的真实故事，吸引了越来越多有情怀的消费者。开业仅一年，微博粉丝数就达到18万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随后，野兽派通过与明星合作迅速打开知名度，先后为高圆圆、林心如等布置婚礼，还大手笔签下多位当红明星作为代言人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当野兽派陆续出现在明星婚礼、电影场景中，这个沉淀了足够基调的品牌，开始转向售卖居家生活用品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1033764573631e34fd79e147.24758243.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：野兽派微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与鲜花品类相较，马克杯、口红、睡衣等家居生活用品的消费频次更高，也更容易随着频次的增加而建立品牌忠诚度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，试图实现品类横向延展的野兽派，以“让生活变得更有仪式感”的品牌思路，通过套系化的产品和一站式的购买体验，在Z世代心目中占据了相当位置。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/127618355762540cc97137d2.72071747.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>出圈之后，野兽派开始在跨界联名中寻求增长突破。先后与 Prada、Nike、猫和老鼠等经典IP都有过联名。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，</span><span><strong>与“小王子”IP的合作堪称是最成功的一次</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在《小王子》的故事中，本身就有很多关于花的经典台词，与野兽派的品牌态度不谋而合。在情感营销的渲染加持下，野兽派的产品成为用户对故事的寄托。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2026852690631e34fdd03dc8.83762661.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span><span>图源：野兽派微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并且，为了让受众直观感受跨界合作的抽象内容，野兽派将对联名IP的理解融入到具体的产品设计，以超出预期的差异化呈现，点燃了目标受众的购买欲望。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，野兽派能够从小众网红成长为行业内的头部玩家，离不开与明星的深度绑定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>野兽派先后与演员胡歌、李现、井柏然，年轻偶像鹿晗、易烊千玺有过合作。去年，又签下了因古装戏走红的龚俊。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/520022630631e34fe1ac8c9.53633760.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：野兽派微博</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，无论是情感攻心，还是借势明星，野兽派的营销，每一步都是聚焦且精准的。随着品牌声量的扩大，野兽派的产品价格也水涨船高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们看到，在天猫旗舰店，一款粉色的永生花可旋转告白兔，折后售价接近3000元，而用户的评价似乎不佳。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1092334706631e34fe895b18.97769848.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：野兽派天猫旗舰店截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这次曝光“代工溢价”的文章中，有网友买到了包装已经损坏，散发刺鼻气味的香氛礼盒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也有官方售价999元的淡粉色玫瑰，签收时花已泛黄枯萎的生日礼物。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99295877631e34ff29dbc6.26813470.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：凤凰网财经</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在小红书、微博上，不乏网友对野兽派的晒图吐槽。诸如5999元的玫瑰捧花实物像菜篮子，买给妈妈的真丝睡衣不到一月脱线等各种问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9818835562540cc9e9aa29.23485278.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>野兽派的成功出圈，无外乎</span><strong><span>情感营销+明星效应</span></strong><span>的品牌策略。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些容易潜入受众心智的“打动”方式，固然能为品牌带来源源不断的流量，却不应该成为高溢价的理由。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>当市场回归理智之后，品牌想要从低频品类进击高频品类，更需要重视受品质影响的复购率</span></strong><span>。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1867475497631e34ffc61413.83346946.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：野兽派微博</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直以来，野兽派都在以“仪式感”体系的产品，来驱动受众买单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说， 鲜花的存在，更多是为了满足受众追求浪漫、表达情感的送礼需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而家居用品，则需要考量受众对产品使用场景和体验的主观感受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/691812887631e35000dd577.91533196.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：凤凰网财经</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当用户花了上千元买套真丝睡衣，却又掉扣子又脱丝。这样的品质，显然无法说法受众为品牌的高溢价买单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟，</span><span><strong>割韭菜的消费时代已经过去</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对追求精致和美好生活的网友来说，相较于故事的打动人心，愉悦的使用体验和质价相符的好产品， 更能够让他们心甘情愿重复买单。</span></p>
