<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于那些懒于动手或者厨艺不精的人而言，一罐复合调味料可以解决大部分厨房做饭难题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>宫保鸡丁调味料、酸菜鱼调味料、高汤卤水汁、香辣小炒酱、鱼香肉丝调味料、回锅肉调味料……市面上的复合调味料有着丰富的口味选择，复合调味品品牌商为不会做饭的消费者绞尽脑汁。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>复合调味品，即多种调味品调制的调料品，代表着更为复合的口味，一种为居家烹饪乃至餐饮业提供的标准化解决方案。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近几年在疫情催化下，伴随餐饮连锁化、中式菜肴标准化（预制菜）的趋势，以复合调味品为创业起点的玩家，餐饮与新零售的玩家，以及传统的调味品开发商纷纷涌入赛道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天眼查数据显示，截至2021年11月，国内就有超4800家复合调味品企业，且其中2800余家相关企业成立于5年，占比达59%。2020-2021年，复合调味品的新兴融资事件也超过10起，其中，2021年12月川娃子还以3亿元的A轮融资刷新了领域内纪录，领投方和跟投方分别为高榕资本与厦门建发。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>复合调味品不仅是一个名副其实的千亿赛道。艾媒咨询显示，国内复合调味品市场规模由2014年的654亿元增长为2021年的1588亿元，年均复合增长率为9.28%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而据弗若斯特沙利文测算，对标日本复合调味品的渗透率和行业集中度，预计国内复合调味品市场有望于2030年达4000亿元。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一罐小小的复合调味品蕴藏的财富巨大。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>01. 由调味到复合调味，B、C两端如何共同碾出一个千亿赛道？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年突然爆发的疫情让餐饮行业一度低迷，餐饮渠道向来是调味品消费的主要渠道，占比超过40%。但令所有人没想到的是，看似toB的复合调味品需求一下子从餐饮转到了家中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当时最为壮观的现象之一，就是火锅食材超市的加速扩张。创立于2019年的川鼎汇，在2020年疫情后门店数迅速从十几家扩张至800多家，全年营收破亿元。复合调味料包含火锅调味料，是川鼎汇300多个产品线中重要的一个分支。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>川鼎汇并非个例。欧睿国际数据显示，2020年零售端复合调味品同比增速达到15%，C端复合调味品天味食品、颐海国际均在当年实现较快增长。公开数据显示，2020年国内整体调味品行业市场规模达3950亿元，同比增长18.1%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/115982767462c5faafbd4ad9.37567128.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以由B到C如此顺滑，还是因为复合调味品在家庭烹饪场景下具有极好的产品拓展性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>调味品通常指的是酱油、味精、醋，而现在语境下的复合调味品一般指火锅底料和中式口味调味料，在中式口味调味料中，又以川味占比最高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>川式口味的调味品有多火，从一个流行的微博段子可见一斑：“吃日式烤肉，要用孜然辣椒面；吃鳗鱼饭，配的是麻辣萝卜干；吃意面，招牌菜是麻辣小龙虾……”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>事实上，很多消费者对复合调味品的认识还停留于一罐下饭的酱料，但复合调味品所能做的还可以更多。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>按准备烹饪的难易程度而言，复合调味品的产品形态依次可以是拌饭酱、给家常菜调味用的调味料包、能够拓展出更多菜谱的汤料包或底料包等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，复合调味品品牌加点滋味，在解剖了三群人四个场景下的“人货场”关系后，就开发出复合调味品的四大产品线：用于工作日的拌饭酱及家常菜系列，用于周末的原汤系列以及大菜系列。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/36434535662c5faafe77a29.41304987.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而继续升维思考，要提供一个更为完整的家中饮食方案，就变成了预制菜，甚至是外卖，而外卖如果选择一家连锁快餐品牌，其中也要用到复合调味品，包括但不限于：酱料包、汤汁包、甜辣酱、番茄酱等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/168945376862c5fab025e393.81261587.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所以，复合调味品不仅有着精确的市场需求卡位，更可能自带B、C双轮推动的效应：由B端推出的爆款热门菜品教育了市场，而居家烹饪下的消费者发现，由复合调味品做出的大菜竟然和吃到的堂食口味近乎一致，如此形成B、C两端同时推动市场扩容。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>典型一例是，新零售、餐饮等其他赛道玩家也在推出自有调味料产品。例如，盒马曾推出自有品牌冷泡汁，海底捞背后的颐海国际除了为海底捞供应火锅底料，也推出了子品牌“筷手小厨”用于运营中式复合调味料业务。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这其中的逻辑就是，本身TOC业务便可洞察市场验证的口味，而结合自身的供应链实力去做自有品牌，能力迁移是水到渠成的事情。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，上述也是理想情况。现实的情况是，由于2020年疫情下，复合调味品的超预期成长给新老入局者注入极大信心，整个复合调味品赛道在2021年也变得拥挤不堪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，非常规的增长导致了企业错判了真实的市场需求，这带来的结果是入局者众，且许多复合调味品企业的动销情况要远远弱于渠道扩张情况。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天风证券做过一个统计，2021年调味品行业营收利润在二三季度出现了持续的下滑，而行业股价也出现不同程度的回撤，其关键原因就在于：入局者甚多导致行业产能过剩，渠道扩张实现的增长掩盖了库存积压问题，以及伴随渠道扩张带来的管理边际问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/87708476662c5fab07df353.09227406.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，经历过去年的行业洗牌后，复合调味品将走向何方？</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>02. 拆解复合调味品赛道，真正的机遇在哪？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先从整体调味品市场去看，Euromonitor的数据显示，中国调味品行业营收从2014年2595亿元增至2020年3950亿元，6年CAGR为7.25%。前瞻产业研究院预计，到2025年我国调味品行业市场规模将达到5500亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，复合调味料市场规模由2014年的654亿元增长至 2021年的1588亿元，2014-2021年CAGR为9.28%，是高于调味品行业整体增速的，另据弗若斯特沙利文测算，对标日本复合调味品的渗透率和行业集中度，预计国内复合调味品市场有望于2030年达4000亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>面向这样一个4000亿规模的赛道，且赛道增速高于业均，复合调味品的行业集中度要远低于调味品市场，意味着复合调味品是一个尚在发育的优质赛道。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，鸡精、鸡粉、鸡汁这种传统单一调味品的行业前五市占率在80%以上，而目前复合调味品的行业前五，市占率不到20%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20032499662c5fab0cb5568.60732525.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以复合调味品中入局最为热闹、市场选择最为丰富的火锅底料为例，天风证券统计的2019年行业前三市占率为35%，行业前五市占率为42%，依次为颐海国际、天味食品、红九九、红太阳、重庆德庄。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>复合调味品这个品类的特点就决定，其具备标准化批量生产的规模优势，市场渗透率和行业集中度的提升都会带动可观的增量。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而从品类去看，火锅底料、川菜调味料又是其中的优质细分赛道。天风证券统计，中式复合调味品占据复合调味品行业市场份额高达70%，而火锅底料和川菜调味品又占中式复合调味品市场份额的40%以上，其中火锅底料市场2010-2019年年均复合增长率为15.9%，无疑拉动了行业整体增速。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>火锅底料有多火？看一下容易标准化、成瘾性强、直接toC的火锅市场的表现就知道了。例如，近期刚完成千万元天使轮融资的鲜卤牛肉火锅品牌「牛爽爽」，就是选择川渝口味，用 “热卤+牛肉火锅”的产品公式，以及火锅社区店模型，满足下沉市场20-35岁的消费群体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，近几年餐饮的连锁化、中式菜肴的标准化加速教育了市场对于不同地域菜系口味的认知，因此，像钵钵鸡、麻辣香锅等川菜调味料以及火锅底料才在餐饮市场的教育下迅速将受众扩容到川渝以外。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/59191215162c5fab1124740.82610250.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在天猫榜单·火锅底料好评榜的前十名中，有2/3的品牌来自川渝地区，与此同时，在小红书搜索关键词“火锅底料”有14万+的笔记。这侧面说明，以火锅底料为代表的复合调味品，为异地吃到川渝美食提供了解决方案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音、小红书等社交平台上刷到美食内容时，种草并下单复合调味品，这后面所折射出的社会心态也不难理解：现在的消费者更想所见即所得，自制文化的流行，懒人经济、独身经济、居家经济……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而更重要的一个启发是，中国人对追寻独特口味的一种执着。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这反映在市场供给端就表现为，大量的复合调味品都是以知名菜品为切入点，以各种鱼、兔、鸡、以及近年热门的小龙虾为原材，回锅肉、宫保鸡丁、鱼香肉丝等热门川菜口味颇为常见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，颐海国际推出了酸菜鱼、小龙虾、宫保鸡丁等口味的调味料，千禾推出了回锅肉、麻婆豆腐、青笋土豆烧鸡等口味的调味料，好人家推出了藤山钵钵鸡、青花椒麻辣鱼、麻辣小龙虾、香辣烤鱼、麻辣烫串串等口味的调味料……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但媒体《调料家》也报道，整个复合调味料，尤其是上述以菜品口味为特征的调料，除了鱼调料之外，基本没有爆品出现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>看似最靠谱、最能撬动大众认知的川菜口味，也在行业困于“卷”后致使产品同质化，使得复合调味品市场呈现出有品类无品牌的现状。</strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>03. 复合调味品的未来：用自成一派的口味捕获消费者</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总的来看，复合调味品仍处于品类的上升期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，对市场需求的误判，赛道的拥挤，产品同质化，用渠道扩张拉动增长掩盖了动销不足等问题也真实存在。各大入局者下一步的重点便在于打造爆品，扩充矩阵，用更合理的产品体系搭建起品牌资产，真正用自成一派的口味捕获消费者的味蕾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2020年起，我们也能陆续看到传统调味品企业开始加码布局这一崭新赛道，例如海天于2020年底率先推出中式复合调味料及火锅底料，李锦记、千禾味业、厨邦也随后推出复合调味料产品，而在近期，复合调味料企业草原红太阳也在谋求A股上市，上述提及，红太阳是排名火锅底料前五玩家之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新老玩家的竞争会愈加激烈，而消费者始终忠诚于口味。</span></p>
