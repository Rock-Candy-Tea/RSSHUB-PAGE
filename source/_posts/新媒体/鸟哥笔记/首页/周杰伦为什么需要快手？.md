<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>歌手<a href="https://www.niaogebiji.com/tag/%E5%91%A8%E6%9D%B0%E4%BC%A6" target="_blank">周杰伦</a>在<a href="https://www.niaogebiji.com/tag/%E5%BF%AB%E6%89%8B" target="_blank">快手</a>的独家直播再次掀起流量热潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月18日，累计超1.1亿人涌入快手，观看周杰伦在快手的独家直播，“居然可以让哥直播”“我的青春回来了”“终于等到周杰伦直播”……有观众在社交平台及快手直播评论区表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，也有不少用户表示，“没看够，还想看”，以及“已经预约‘哥友会’”“等着‘哥友会’啦”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>7月6日周杰伦新专辑《最伟大的作品》先行曲发布之后，7月10日周杰伦官宣在快手进行独家直播。燃财经也看到，截至7月18日早上9时50分，周杰伦独家直播人数已破千万，按照北京鸟巢体育馆十万人的容纳人数计，接近100个鸟巢大小。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在7月18日晚，直播刚开始5分钟，在线观看人数就突破了200万。来自快手官方的数据显示，累计观看人数超1.1亿，最高同时在线人数超654万，总互动量超4.5亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>周杰伦的影响力可见一斑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年6月1日，周杰伦正式入驻快手，快手平台“周同学”账号成为其“首个中文社交平台”，也是“全网唯一”中文社交平台。7月6日，《最伟大的作品》先行曲MV在全网上线，由周杰伦本人通过个人账号@周同学 在快手发布的先行曲MV上线8个小时，播放量就突破了1.5亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>暌违六年，周杰伦再度发布新专辑，各平台也围绕周杰伦做足了文章，但作为周杰伦全网唯一中文社交平台，快手与周杰伦的合作更为“长情”且多样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了上线MV与音乐单曲之外，7月5日周杰伦就在快手上发布了新专辑预热视频，与快手老铁互动“就是明天了”；新专辑先行曲MV发布之后，“周同学”还在线给快手用户留起了作业；7月8日，“周同学”在快手平台发布独家钢琴伴奏的短视频，邀请网友合拍跟唱……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此次独家直播，快手吸引了累计超1.1亿人观看，接下来，快手将要开启的“周杰伦线上哥友会”，即“你（</span><span>用户</span><span>）来选歌，杰伦演唱，连麦互动，只在快手。”记者发现，在社交平台上，已经有不少用户晒出了预约后得到的电子“门票”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从初中开始就听周杰伦，已经是十余年杰迷的胖丁表示，“周杰伦本身就不怎么在公开场合露面，快手提供了一个非常好的平台，让大家可以看到周杰伦的动态，并且在直播中和周杰伦交流。直播时周杰伦问大家喜欢新专辑哪一首歌的时候，真的感觉和周董距离很近。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在以往的语境中，关于快手和周杰伦的合作，大众对于快手可以从中获得怎样的增长讨论颇多。但从另一个角度来说，周杰伦也通过快手平台获得了与粉丝交流的阵地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于周杰伦为何选择快手？快手真实、平等、烟火气的底色，或许是吸引周杰伦的一个关键。快手娱乐明星业务相关负责人就表示，“我们希望每一个明星在快手上都是真的在分享他的个人生活，要真的在分享他的每一个值得纪念的瞬间，是真的想和快手老铁玩在一起。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>周杰伦在2021年11月29日发布的“我和文山中国风歌词之当代装置艺术的对谈”视频中，面对方文山的拘谨也说道，“你可以自然，因为快手是很轻松的。”这是快手生态的底色，也是快手吸引众多素人以及明星入驻的本质原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，除了周杰伦之外，快手还拥有成龙、杨幂、迪丽热巴、黄子韬等2000+个明星账号，整体粉丝量达到8.7亿。通过明星入驻，快手和明星也达到了相互成就、携手共赢的效果，明星们为快手提供了内容和生态，3.46亿日活的快手也为明星们提供了平台和阵地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这样的背景下，快手和明星携手，也能走得更好、走得更远。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/1455856016142b5abe0ca06.17059714.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>在快手，加入周杰伦的“朋友圈”</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>周杰伦入驻快手，获益最大的是杰迷们。因为杰迷们获得了一个近距离接触杰伦的平台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此前就有粉丝向燃财经表示，有了快手可以更好地了解周杰伦的动态，“之前周杰伦动态更新都是在Instagram，但坦白来说，我并不太习惯上Instagram，因为太麻烦了。有快手还是方便一点的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃财经也看到，在周杰伦入驻快手之后发布的100多条动态中，除了常规的宣传之外，就是自己的生活日常，比如和好友阿德、罐头、冠霖的合照，以及和好友在练习室的日常。2020年7月21日周杰伦还晒出和好友外出的合照，表示“听说26号要魔术直播（</span><span>周杰伦直播首秀</span><span>），有点紧张，先出来放松一下。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“周杰伦发在快手的内容，十分随性，跟印象中艺人那种‘完美到头发丝’的精心设计完全扯不上关系，就跟朋友圈的直率大男孩一样。”胖丁表示。燃财经在“周同学”的评论区也时常见到“关于周董把快手当朋友圈发这件事”等评论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一切得益于快手给了周杰伦更大的自由度。“周杰伦在每一个对他而言重要的节点，都会主动选择在快手和大家一起分享。”上述快手娱乐明星业务相关负责人表示。言外之意，“周同学”账号发布的内容，多数是周杰伦自行进行发布，而非“营业”要求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“7月5日周杰伦在新专辑先行曲MV发布前夕，在快手发布那张‘就是明天了’的动态，就是周杰伦自己本人想发的，而不是合作的宣发。”上述负责人告诉燃财经，另外，“周杰伦妻子昆凌新电影上映，是他主动选择在快手做直播的。‘周杰伦X苏富比’当代艺术专场拍卖也是因为周杰伦喜欢古董收藏，所以自己选择在快手直播展示全过程。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6453126662d612b65ee492.82508853.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/周杰伦7月18日快手独家直播画面</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/燃财经截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了粉丝受益之外，通过快手平台，周杰伦也获得了一个与粉丝交流的阵地。在7月18日的直播中，周杰伦向观看直播的观众提了一个问题，“最喜欢新专辑的哪一首歌？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这也是直播我最感动的一个环节。”胖丁向燃财经说道，“杰伦一直都很在意歌迷朋友们的反馈。提出这个问题之后，很多人在刷旧歌，杰伦也耐心解释新歌和旧歌的差异，那个时候，我真的觉得我们和杰伦距离很近。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微博用户@丸子 也在平台表示，“周董真的很细节很贴心，问弹幕喜欢自己哪首新歌的时候，主持人念粉丝刷的歌名，杰伦提醒主持人要念他们名字，不然他们不知道是自己。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“虽然这次直播没有见到周董开麦，但是能这么近距离看到他就挺开心的。其中他说的‘为什么新歌敌不过老歌的’提到的因为老歌多了怀旧和岁月感，对我而言太有共鸣了，确实融汇了很多记忆在旧歌曲中。”十年杰迷双仔表示，还是很开心现在能在快手看到周董相关的动态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了拉近周杰伦与歌迷的距离之外，还有用户表示，“周杰伦入驻快手，可以拓展00后用户群体。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为陪伴80后、90后成长起来的一代，周杰伦的用户群体虽横跨多个年龄段，但80后、90后仍是主力。2019年，“Alfred数据室”曾根据周杰伦超话粉丝做过一个数据分析，周杰伦粉丝主要以1988年到1997年为主，年龄在22-31岁之间（</span><span>以2019年时间计</span><span>）。艾漫数据的活跃粉丝画像显示，周杰伦63.85%粉丝年龄段位于25-34岁。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在快手的用户里，喜爱视频形式的00后众多，也许对于周杰伦拓展00后粉丝，确有作用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但目前来看，增加周杰伦与粉丝交流仍是最大的价值。“除了可以看杰伦动态，直播时可以与杰伦互动之外，快手还做了很多活动，让粉丝玩到周杰伦的音乐中去，这也是一种互动和拉近距离。”胖丁直言。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/9082723746142b5ac73c549.43567204.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>快手笑到最后</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月19日，周杰伦在社交账号发布新专辑前导纪录片，并在片尾预告：新专辑将于7月15日发行。此后，围绕周杰伦的“争夺战”就拉开了序幕。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为华语乐坛经久不衰的“流量密码”，从周杰伦放出口风说“六七月发新专辑”，到新专辑先行MV发布，新专辑预售和正式发布，每一个时间节点都是天然的“流量盛宴”。在新专辑发布期间，“周杰伦专辑”“周杰伦 哥要出来了”“最伟大的作品解读”“周杰伦rap”……众多话题纷纷霸榜微博热搜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于各大平台来说，新专辑的每一个节点也是必须卯足力气的“争夺战”。7月6日，新专辑先行MV发布之际，因为“杰威尔音乐”入驻了B站、微博、抖音、快手等多个平台，各个平台都参与到这场流量盛宴中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为音乐平台，腾讯音乐娱乐集团（</span><span>TME</span><span>）、咪咕音乐更是使出浑身解数，从经典回顾、彩蛋解读、置顶流量入口，到周边定制、MV二创，新专辑造势不断添火加柴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月14日晚间，新专辑《最伟大的作品》提前上线，统计数据显示，截止7月18日，新专辑在QQ音乐已售出500万张。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>伴随着新专辑发布，多家平台的“周杰伦争夺战”也告一段落。但快手却凭借独家牵手周杰伦而热度一直延续。在宣布开启周杰伦独家直播之后，周杰伦独家直播的预约数便不断增长。燃财经在微博、小红书等社交平台上，还看见不乏有人表示，“马上下载快手。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/112390927062d612b6998281.39193186.png" referrerpolicy="no-referrer"></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/快手官方微博账号</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>周杰伦用心的“独家”内容便是快手在一种平台中脱颖而出的“杀手锏”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月8日，“周同学”在快手发布钢琴伴奏的短视频，留下独家“作业”。资料显示，截至7月18日，快手站内话题#周杰伦让大家交作业 共有2506个作品，话题播放量达6.9亿，话题下不少音乐迷晒出与周杰伦同框合拍，共同演绎《最伟大的作品》。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月14日，在新专辑发布之后，周杰伦还在快手账号独家发布了“尤克里里版”弹唱视频，并在文案写道，“新专辑首唱送给快手的朋友”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了让更多歌迷朋友和周杰伦玩在一起，快手还提供了《最伟大的作品》同款魔法表情、新歌MV同款风格模板、与周杰伦一起合唱玩法、周杰伦限定版头像挂件等等各类创作素材。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/4756644006142b5ad08cc02.64114870.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>共赢之战</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>明星向来是各个平台争夺的对象，这些年，各个平台之间的“明星之争”也时有发生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而明星选择平台，也有自己的打算。整体来看，明星入驻短视频平台也好，社交媒体也罢，背后都有“流量焦虑”在作祟。毕竟随着抖音、快手、小红书等新社交平台的崛起，意味着大众的喜好在发生迁移，明星融入这些新流量社交平台，缩小与粉丝之间距离带来真实性的同时，也把握住了新流量平台给予的机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但近几年，“真实”成为明星在社区或社交平台上活动时越来越被看重的品质，粉丝也乐于见到明星在社交平台上“越来越像个普通人”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/61895186562d612b706a5c2.45196588.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/7月18日周杰伦快手独家直播在北京三里屯的广告页面</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来源/快手官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手娱乐明星业务负责人介绍，明星在快手的内容，一直强调“普通人视角”，帮助明星去做真实分享，体现出自己身上与普罗大众用户接近的地方，比如家人、情感、爱好、困扰、乐趣等。上述负责人认为，“快手上的明星，并不是单纯地展示自己的外貌和优越感，而是特别真实地将自己的工作和生活做成短视频，将用户当做朋友去沟通。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在快手平台上，明星之前的互动也很真实，如同现实生活中与朋友“打招呼”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月6日周杰伦在账号发布先行曲后，钢琴大师郎朗、周杰伦好友刘畊宏便前来助阵，其中刘畊宏就周杰伦留的作业表示，“这作业，我来预备。”此外，郑恺、张歆艺、Ella等一众明星也纷纷在“周同学”快手账号评论区现身，热情地为其新专辑打Call。火爆全网的快手“栓Q哥”teacher刘也在线发出邀请，“杰伦栓Q，欢迎您到阳朔来坐船。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“明星能够在平台上以一个普通人的身份，讲真话、做自己，这对明星非常珍贵，我们希望能帮助艺人摆脱枷锁，制造热点话题的同时，帮助他们提升社会影响力和群体认同感。”此外，快手负责人还表示，快手用户的高度粘性与高信任度，也有助于明星打造健康的粉丝圈层，不同于传统的明星与粉丝之间的关系，“在这里，明星拥有不被粉丝苛求的舆论空间。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如快手slogan“拥抱每一种生活”一般，明星在快手上在分享各自的个人生活、每一个值得纪念的瞬间。作为平台，快手为明星提供了平台、流量和曝光，明星也丰富了快手的内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于未来，快手也有自己的打算。快手希望做生态，在这个生态中，多元化的内容都有，一款产品的壁垒不是它的用户数量与交易额，更重要的是它的不可替代性，用户在快手玩得开心并且能满足多重需求，就不会离开。明星汇集的娱乐生态，一如此理。</span></p>
