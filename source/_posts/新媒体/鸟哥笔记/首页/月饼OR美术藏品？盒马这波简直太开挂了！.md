<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>临近中秋，朋友圈晒月饼大赛又要开始了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这几年，随着国潮如不可抵挡的洪流席卷了全中国，月饼这一传统节令食物也突破了原本单一属性，成为了越来越多的年轻人表达自我意志的载体。表面上，大家在朋友圈里晒的是月饼，而实际上，大家真正钟意的是其中的文化个性表达。如今的月饼，更像是一种具有国风国韵属性的社交货币，这背后既有着年轻消费者对自身作为中国人的文化自信，也有着年轻消费者对精神价值的诉求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正因如此，互联网大厂的文创月饼，成为了每年备受关注的“抢手货”。从BAT到字节、美团、B站、知乎，各家都曾推出过颇受年轻人欢迎的文创、国潮、联名月饼礼盒。今年，一直“缺席”的国字号IP中国美术馆文创也被<a href="https://www.niaogebiji.com/tag/%E7%9B%92%E9%A9%AC" target="_blank">盒马</a>请出了山，把齐白石真迹藏品画作与现烤月饼做了融合，凭“美”而占领了不少人今年的朋友圈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/126488396163097182e83277.16677960.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这款“水墨月饼”不仅把齐白石的真迹画作创新融合在了包装之上，其定制款水墨色月饼更将国风与产品结合到极致，可以说无论是颜值，还是国风韵味都相当出众。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>别怀疑，即便是把它放在中国美术馆内的陈列架上，也毫无违和感。兵法先生不禁感叹，现在互联网大厂的国潮月饼内卷得越来越厉害了，盒马“水墨月饼”又把<a href="https://www.niaogebiji.com/tag/%E5%9B%BD%E9%A3%8E%E8%81%94%E5%90%8D" target="_blank">国风联名</a>玩出了新的高度。</span></p>

<section data-role="paragraph">
<p><span><strong><span>是月饼更是美术藏品</span></strong></span></p>
<p><span><strong><span>颠覆你对国风联名的认知</span></strong></span></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不可否认，跨界传统文化IP，目前确实是品牌打造国风营销的主流玩法。但今天的95后、00后，他们拥有强烈的文化自信心，也乐于探究品牌联名背后的文化内涵，经过近几年品牌国风跨界的狂轰乱炸之后，简单挪用“国风”做加法，可无法再轻易征服他们的心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而盒马X中国美术馆文创中心，却并不只是做“加法”那么简单，首先，</span><strong><span>它结合了年轻人传统文化自信提升及高颜值产品偏好的双重特性，将中国水墨艺术与国民饮食相融合，赋予了产品美感与韵味，让每一款月饼都是艺术藏品。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相信很多人都有这种感觉，盒马“水墨月饼”给人的第一印象就是美。这其中齐白石的真迹自然有很大的功劳，但也是因为中国人从骨子里就不曾与田园山水分离，有惊涛拍岸的勇敢，轻舟过重山的平和，也有采菊东兰下的恬淡。提起水墨艺术，总能让人马上联想到这些别样的古典国风之美。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1758995602630971838b6af5.40984351.gif" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>齐白石笔下的水墨画最大的特点，就是他的绘画技法中保留了晕染，留白这些水墨艺术的特征，同时又色彩单纯，意象奇特又有别致感，喜欢通过浑然天真的乐趣体现出了绘画的趣味性，既解决了雅与俗的问题，也解决了古典艺术与民间艺术的结合问题。就像这次联名月饼包装采用的齐白石真迹画作，无论活灵活现的虾，还是寥寥几笔勾勒出山水、花鸟画都不失意趣，很好地踩在了当下年轻人的审美点之上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/19309188456309718ad09a01.16895690.gif" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，这正是品牌在追寻国潮时应有的态度——不囿于传统，不缺少创新，更不缺乏思想，真正重塑中国人的态度、精神和风尚。而这一点，也体现在了盒马对月饼产品的设计和打磨之中，说它是艺术藏品，绝不仅是因为产品的包装运用了齐白石的真迹这么简单。<span>‍</span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/392144992630971921579e2.76154053.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，盒马“水墨月饼”还别出心裁地推出了芥末大虾、鲜肉、流心奶黄、黑松露芝士牛肉四大口味来对应齐白石四大画作，其黑松露芝士牛肉口味，更是特别采用了水墨风黑白色饼皮，碰撞出别有生趣的水墨月饼。每一款小小的月饼，都如同一幅水墨山水画，都是彰显国风艺术的“藏品”，可谓把国风与产品结合到了极致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/539423848630971928698a0.23823900.gif" width="333" referrerpolicy="no-referrer"></p>

<section data-role="paragraph">
<p><span><strong><span>洞察深入内核，</span></strong></span></p>
<p><span><strong><span>回归用户需求原点占领心智</span></strong></span></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“小时不识月，呼作白玉盘”，或许每个人懵懂的童年，都仰望过天上的明月，亦盼望过中秋的月饼。回想往昔，你是否有过这样的困惑：当年的月饼为什么就那么的香甜？如今，月饼还是那个月饼，我们却再也找不到当初连一个渣都舍不得掉在地上的感觉了。很显然，相较于“吃”，以创意内容弘扬国风博取社交货币，才是当下月饼礼盒真正的“主战场”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这方面来看，盒马“水墨月饼”真的很懂年轻人的需求。一方面，它顺应了国风消费大势，以诗一样的气质、幽远清淡的包装深深吸引了年轻人的视线；另一方面，它征服人们的也不仅仅是视觉上的美感，更因其聚焦中国水墨艺术与国民饮食的结合，让月饼不再只是一个渐渐远去的食物，而是将它们身上的中国文化放进年轻人的口中被细细地咀嚼、品味，让更多的年轻人通过它来表达对国风国韵的理解与弘扬，从新的角度顺利完成层层破圈的传播和占领年轻人的心智。<span>‍</span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/59802610963097197466154.12746041.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，这也不是盒马第一次尝试把文化放进月饼，用创新的姿态延承国风国韵了。去年盒马就曾与上海新华传媒合作，推出过表达中国汉字文化的月饼，把生僻字印上了饼皮，真正做到了“边吃边学”。随着其今年走进艺术圈进一步拓展广度和深度，盒马俨然已成为了一个生产“国粹月饼”专业户了，这不禁令人对它明年的文化跨界期待不已。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟，一款既能吃，又能晒的个性国风藏品谁又不喜欢呢？</span></p>
