<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月15日，“我是名侦探小宇”发布视频《每一次的旁观者，可能就是下一次的亲历者》，针对近期女性受伤害的现象，她呼吁大家能够站出来，不再冷漠。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/199719869462b322f6434f39.41318137.jpeg" title="为什么MCN热衷做公益？" alt="为什么MCN热衷做公益？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我是名侦探小宇”的作品经常能引发网友对女性安全话题的关注和讨论</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>2019年至今，“我是名侦探小宇”发布了一系列提醒女性远离伤害的视频，目前全网粉丝近2000万。<span>“我是名侦探小宇”所属<a href="https://www.niaogebiji.com/tag/MCN" target="_blank">MCN</a></span><span>古麦嘉禾创始人于爱山透露，该账号从一开始就确定了公益主题。</span></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，作为一种社会共识，不少MCN、网红博主都积极投身公益。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么MCN热衷做公益？近日，新榜编辑部与古麦嘉禾、游良文化、青藤文化、摘星阁等几家MCN的老板聊了聊。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在游良文化创始人罗冠彦看来，做公益能得到外界的普遍认可，也是MCN必须承担的社会责任。</span><span>旅行博主“房琪kiki”也提到，公益是一种希望带给别人幸福的本能，“那种成就感和快乐非常难得”。</span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span>心中的善意和社会责任感，大概是每个人参与公益的最初动力。</span></span></strong><span>但想让公益长久持续地发挥作用，就不能只是“我想做好事”，单靠朴素的道德驱动。<strong>公益是一件需要专业思考和专业支撑的事。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新媒体人如何参与公益？如何持续做公益？如何让公益效果最大化？这是需要深入思考的问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" title="为什么MCN热衷做公益？" alt="为什么MCN热衷做公益？" referrerpolicy="no-referrer"></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>新媒体人很适合做公益</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>关于公益，几位MCN老板的基本共识是：</span><strong>不以赚钱为目的，做对他人、对社会、对地球有益的事情。</strong><span>同时，公益没有大小之分、形式之差。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于如何做公益，青藤文化创始人纪方圆的观点则是，<strong>“能出钱的出钱，能出人的出人，但改变受助人的思想意识，是效率最高、最能持续的”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这方面，因为既是内容的生产者，又是内容的传播者，同时与粉丝有着高频互动，新媒体人有着得天独厚的优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>几位MCN老板一致认为：<strong>新媒体人有粉丝、有内容，掌握一定的流量话语权，很适合做公益的传播和链接，用内容去触动更多人。同时，因为对内容、粉丝的深度理解，新媒体人往往能生产出一些更有共鸣的公益内容。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，游良文化旗下达人“梅尼耶”就曾和“深圳公安”合拍了一条有关儿童走失的视频，因为真实且有教育意义，这条公益宣传片全网累计获赞270万以上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/184390195262b322f6936975.12966799.jpeg" title="为什么MCN热衷做公益？" alt="为什么MCN热衷做公益？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“UP主的一言一行很容易影响到各自的粉丝，所以MCN做公益的效率也会更高一些。”纪方圆补充。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，<strong>光适合还不行，新媒体人还需要有持续做公益的动力。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果一个公益人既无法得到内部认可，又无法获取外部助力，那公益大概率会变成无源之水、无根之木。真正持久的公益既有发自内心的善意，更有源源不断的正向反馈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>罗冠彦认为，公益搭建起了一个私下交流的平台，能拉进网红博主和员工之间的距离，增强公司凝聚力。“一起做公益的过程中，他们会产生一种共鸣，这是拍广告视频时没有的。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/135732654362b322f6c7ab07.17375881.jpeg" title="为什么MCN热衷做公益？" alt="为什么MCN热衷做公益？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>游良文化旗下达人正参与公益活动</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对内，公益能增强网红博主和员工对MCN的认同感，增强团队战斗力；对外，公益也是展示MCN内容能力、传播能力的绝佳方式，能提升外界对MCN的认可度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>摘星阁创始人侃烃认为，<strong>“如果一个企业在诞生之初就具备社会责任感，公司大概率是不会做差的”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，不仅公益本身是很好的内容，公益也是网红博主触达更多粉丝、塑造良好形象的重要方式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于爱山认为，“这一代人没怎么吃过苦，都很年轻，但内容创作源于生活，参与公益活动能增强网红博主和员工的同理心，增强他们对生活的认知，这样他们的内容创作才能够有更高的思考维度”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于新媒体人来说，公益不是“无利可图”，而是充满了正向反馈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" title="为什么MCN热衷做公益？" alt="为什么MCN热衷做公益？" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>新媒体人做公益的三大难题</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新媒体人能做、也愿意做公益，但公益却是一件非常专业的事情，必然会遇到大大小小的难题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>首先，很多新媒体人都是单打独斗，相当数量的公益项目没能发挥最佳效果。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>侃烃提到，摘星阁曾发布一条有关职场性骚扰的公益宣传片，但在没有更大平台的支持下，传播始终存在局限。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/75494176062b322f70d75a2.17748690.png" title="为什么MCN热衷做公益？" alt="为什么MCN热衷做公益？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“它虽然发挥了价值，但其实可以更好。”侃烃遗憾地说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其次，如果没有专业支撑，做公益不仅会陷入劳心劳力却没结果的窘境，还会极大挫伤人的积极性。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>侃烃介绍，她印象最深刻的一次公益经历是帮助了一个导盲犬基地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“当时我们赞助了10万块钱，还帮忙拍了视频做宣传。本来我们觉得没什么，但一年后我们回访时基地负责人告诉我们，这些帮助起到了‘起死回生’的效果。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/127663680562b322f7cbf2d9.68890007.jpeg" title="为什么MCN热衷做公益？" alt="为什么MCN热衷做公益？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>摘星阁和导盲犬基地的合照</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次经历让侃烃开始思考，做公益需要把钱花在更需要的人身上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，要想做好公益，有着一系列专业问题需要思考：如何把钱花在真正需要的人身上？如何辨别谁是真正的“弱势群体”？如何对公益人和受助人进行精准匹配？如何充分调动各方参与者的能力、资源？如何筛选出靠谱、专业的公益组织？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>最后，新媒体人擅长内容，却未必能解决公益的所有问题。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>罗冠彦举例，游良文化能帮助乡村做文旅宣传，也能帮忙带货，但有些乡村急需解决的却是看病问题，这需要联动一些专业机构或组织来共同解决。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在他看来，要解决这些难题，一个很重要的方式就是形成一个专业的平台、一种可信赖的机制，寻找一群同频共振的人，持续去推动公益这件事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" title="为什么MCN热衷做公益？" alt="为什么MCN热衷做公益？" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>新媒体人如何让公益效果最大化？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如前文所说，<strong>没有充分的宣传资源、足够的专业支撑、同频的公益伙伴，</strong>是限制新媒体人实现公益效果最大化的三大难题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而要解决这些难题，一个很重要的方式就是发挥社会共创的力量。这也是目前广受推崇的一种公益方式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>昨日（6月21日）正式启动的“2022黄河奖-我是创益人数字公益创意大赛”就是希望能通过社会共创的方式，连接更多“创益人”，探索更多公益解决方案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/115473248762b322f816e847.17793037.png" title="为什么MCN热衷做公益？" alt="为什么MCN热衷做公益？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年，腾讯启动“创益计划”，希望通过去中心化的方式，最大化调动各方资源，最终把力量汇聚到公益这件事上，“我是创益人大赛”就是“创益计划”的首个落地项目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截至目前，“我是创益人大赛”累计有5032支参赛团队，产出的240支公益广告吸引了近3亿人次参与公益行动，产生了近100亿次总曝光量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，《一个人的球队》讲述了一个喜爱篮球但意外去世的少年，捐出自己的器官，他的五位获捐者组成特殊球队代他圆梦的故事。最终该作品让68万人加入到了器官捐献的队伍，并获得了戛纳公关银狮奖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年，在中国广告协会的指导下，腾讯“创益计划”与“中国广告业大奖——公益广告黄河奖”大赛组委会联合设立了“2022黄河奖-我是创益人数字公益创意大赛”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与往年不同的是，<strong>今年新增了短视频组，更多MCN和网红博主可以一起加入到这场公益共创中。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>参赛者可以在腾讯公益平台选择与自身契合的公益项目，通过创作公益广告的形式，吸引大众对“<strong>创新公益、乡村振兴、志愿服务、生物多样性、文化保护传承、未成年保护、适老化与老人关怀、家庭文明</strong>”等公益议题的关注。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“黄河奖-我是创益人大赛”不仅会提供现金、广告金等资源扶持，还会有相应的技术支撑、培训支持，让不同领域的“创益人”都能充分发挥各自的力量，为公益事业加油。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这其中，新媒体人可以充分发挥自己的内容能力、传播能力，并借助“黄河奖-我是创益人大赛”的势能进一步放大自己的公益善举。</span></p>
