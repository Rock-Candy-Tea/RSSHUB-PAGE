<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顶着“新式茶饮第一股”头衔的“<a href="https://www.niaogebiji.com/tag/%E5%A5%88%E9%9B%AA%E7%9A%84%E8%8C%B6" target="_blank">奈雪的茶</a>”（</span><span>02150.HK，以下简称“奈雪”</span><span>），又双叒叕陷入了争议。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，奈雪在港股上市一周年之际，推出了“奈雪币”和虚拟股票。随即，话题#奈雪向5000万会员推出虚拟股票#引发网友热议。截止目前，该话题阅读次数已高达1.1亿次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据活动规则，消费者可通过在奈雪官方小程序购买奶茶、每日签到等方式获得奈雪币，奈雪币可用于购入奈雪的虚拟股票。而虚拟股票走势与奈雪港股行情挂钩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，消费者亦可通过买卖虚拟股票赚取奈雪币（</span><span>可添加×2、×5、×10杠杆</span><span>），并在商城中兑换优惠券或实物，最高可兑换官网价格7999元的苹果笔记本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一时间，边喝奶茶边搞投资，似乎成为了奶茶爱好者们的新风潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在奈雪微博超话内，更有不少用户求问如何买入和卖出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“喝杯奶茶就能用奈雪币兑换虚拟股票，不仅能当上奈雪的‘股东’，还能在虚拟盘过把炒股的瘾，很令人心动。”正在读大二的茜茜告诉燃财经，这个玩法挺新奇，成本也不高，只需要100元就能入场体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>奈雪的“新玩法”不仅吸引了“奈雪粉”的“尝鲜式”入股，也带动了本就有理财意识的00后进场。今年刚刚开始工作的方清告诉燃财经，他已经通过奶茶点单获取了5000个奈雪币，在兑换虚拟股票后又加上了10倍杠杆，“希望能够‘搏一搏’，兑换苹果笔记本电脑。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>资深元宇宙分析师阿起告诉燃财经，奈雪示范了一种不具备强技术基因的企业布局元宇宙的新打法。一方面，下单赠奈雪币是营销行为，不仅能让奈雪迅速回流现金，带动用户消费，还抓住了国内元宇宙赛道的主要消费群体——00后。另一方面，奈雪将用户社群升级成了元宇宙流量池，最大化地挖掘用户价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但一级市场投资人tomatosauce却提出了不同观点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>tomatosauce表示，奈雪币更偏营销炒作，并非真的元宇宙。“奈雪币就像消费积分，根本不能在奈雪商城之外的平台交易，不属于数字货币。而当前奈雪的元宇宙世界也并未持续开放，除了可供炫耀的社交货币，并未让用户真正实现多样化的功能体验和自由的内容创作。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>tomatosauce进一步表示，奈雪如果真的意图布局元宇宙，需要开发更多的功能和玩法，让用户在“奈雪元宇宙”里真正创造内容。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“为什么不好好做奶茶？”、“能不能让我直接以优惠价喝杯奶茶，别搞这些虚的！”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友质疑的同时，也有业内人士对此表示，虽然奈雪币的本质是消费积分，是非流通性的代币，暂未触碰法律红线。但虚拟股票仍存在金融风险，而且引入金融杠杆的玩法也容易让用户在投机的心理下大量囤币交易，造成经济损失。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿起亦表示，现在奈雪的虚拟股票和真实股票呈绑定关系，其上涨和下降的幅度很容易被庄家操盘，出现真实股票价格上涨或稍降，虚拟股票大涨或大跌的情况。此时，如果用户出于投机心理，盲目投入，很有可能“血本无归”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“最终，你以为自己在抄底奈雪，很有可能被奈雪反抄了底。”阿起补充表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>奈雪币和虚拟股票，在引发众多争论的同时，也进一步体现了奈雪布局元宇宙的野心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，早在去年12月7日，奈雪在品牌成立6周年之际，就官宣了来自元宇宙的品牌大使“NAYUKI”，同时推出了售价699元的限量潮玩。此外，还发售了“NFT盲盒”，即包含隐藏款在内七款不同造型的无实物NAYUKI潮玩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>官方数据显示，该活动在发布后的72小时，收获了GMV近2亿元的成绩，NFT盲盒与潮玩也均在当日售罄。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>入局元宇宙，为奈雪带来强媒体曝光和实打实的现金流背后，是奈雪“艰难求生”的窘境。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有业内人士表示，奈雪的连年亏损或与其门店的快速扩张有着密切联系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如上述人士所说，2021年，奈雪自营门店净增长326间，并预计在2022年再拓展350家自营门店。租金、员工成本、材料成本或成为压在奈雪身上的三座大山。</span></p>

data-copy-origin="https://shimo.im"><strong>元宇宙，故事虽新却未必好</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在阿起看来，如果奈雪乐园持续开放，用户在元宇宙世界里下单，之后来到线下取餐，就能建立虚拟与现实的连接，让元宇宙在快消行业拥有全新的应用落点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿起指出，相比起购买VR、AR等设备，奈雪的元宇宙世界门槛更低，同时能够实现企业自身生态和元宇宙生态的互相带动。“对于跃跃欲试想进入元宇宙世界，但缺乏强技术支持的企业而言，无疑是一个极具创新性的范例。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/49892924462c3abe70ece23.78433642.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/用户在奈雪乐园里赏烟花 </span><span>来源/奈雪官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在业内人士看来，尽管现在来看，奈雪对元宇宙的布局颇具新意。但对于醉翁之意不在“茶”的奈雪来说，元宇宙的故事虽新却未必是好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“将现在的奈雪乐园称为元宇宙的新应用，还言之过早。”tomatosauce补充道，现在的奈雪币本质上就是消费积分，和Q币没有区别。而虚拟股票的玩法，也早就在各类股票APP上屡见不鲜，它并不是真正的股票交易，更不会影响到奈雪的股价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位对新式茶饮颇有研究的业内人士同样表示，奈雪入局元宇宙更像是一场差异化的市场营销行为，毕竟新茶饮赛道发展至今，包括奈雪、喜茶在内的头部选手，都已陷入增长疲态。在高度同质化的茶饮市场，各个品牌都在追求差异化。“对于奈雪而言，元宇宙或就成了其吸引目标用户的营销噱头之一。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但tomatosauce强调，快消品赛道竞争激烈，元宇宙的竞争同样残酷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“元宇宙赛道上，早已涌现出不少功能丰富的虚拟产品，更不缺乏互联网大厂的布局。如果奈雪决定持续布局元宇宙，奈雪币或只是最初级的产品，不能吸引真正的元宇宙玩家。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>tomatosauce表示，开发更多的功能和玩法，让用户在奈雪乐园里真正能够建起自己的房屋、举办活动、玩游戏，创建出可以让用户创作内容的元宇宙，才是奈雪的竞争力所在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无疑，在快消赛道竞争激烈的当前，元宇宙对奈雪而言，是一个好讲的新故事。可要想讲好这个故事，恐怕奈雪还要做更多的功课。</span></p>
