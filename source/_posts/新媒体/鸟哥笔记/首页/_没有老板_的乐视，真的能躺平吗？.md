<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E4%B9%90%E8%A7%86" target="_blank">乐视</a>又整活儿了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月27日，有网友发现，乐视把官方图标文案改成了</span><span><strong>“励志上班四天半”</strong></span><span>，此举与乐视之前网传的“没有996、无内卷”的神仙打工人生活遥相呼应。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/47028501362e260d8257fa1.06940225.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span><span>图源：乐视视频官方微博</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时乐视在回应中称，“如果有一天，我们合法地率先推行每周工作四天半、36小时工作制，大家也不要感到意外”，而更换图标文案的做法，或许意味着乐视将其变成了现实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从大众眼中“怎么还没倒闭”的落魄公司，到人人羡慕到“我想跳槽去上班”的神仙公司，看似“躺平”的乐视，其实一直在努力自救。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><img src="https://ngbjimg.xy599.com/5611320262b974392829e0.57743105.png" referrerpolicy="no-referrer"></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>“死灰复燃”的乐视</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾几何时，人们对乐视最后的记忆，恐怕是负债、退市、下周回国贾跃亭等一连串负面新闻和网络段子。曾经风头一时无两的乐视网，很快被卷入互联网的大浪中，很少有人问津。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但近两年，乐视网却大有“卷土重来”的迹象。最大的一个标志是，乐视成为微博热搜上的常客。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>往前追溯源头，2021年初乐视更换图标文案，是一个开始。彼时各个财大气粗的互联网大厂都在争相发春节红包，并把红包金额显示在图标上，来一场无声的battle。尚在债务中的乐视，则顺势来了一波自嘲，表示自己“欠122亿”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/132070263162e260d8ce71d0.48583823.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：手机端截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此举一出，引发网友热议，#乐视 欠122亿#当时直冲热搜榜第4位。沉寂许久的乐视，重新回到大众视野。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尝到甜头的乐视，开始不断在图标文案上花心思。“关键控好资金链”、“老板造车美利坚”、“为人立志当高远”、“重振旗鼓造经典”、“小贾生态雷总验”、“影视综艺守底线”等都是乐视整活儿的成果。今年春节的红包大战中，乐视用“分不起”再次赚取了热度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，乐视还在其他方面和互联网大厂“唱反调”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年年底，不少互联网公司卷入裁员降薪风波，而乐视则旗帜鲜明地表示，</span><strong><span>不仅不裁员，还要给员工涨薪。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/181603857662e260d96a3623.21017299.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：乐视视频<span>官方微博</span></span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年疫情反复的四五月份，许多公司受到不同程度的冲击。为了打消员工对裁员降薪的顾虑，乐视发布内部信表示，</span><span><strong>公司业务运行平稳，员工居家办公期间，全员全薪。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/108757378662e260d9ec6105.82632752.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：乐视视频<span>官方微博</span></span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频平台爱奇艺、优酷、腾讯、芒果TV的会员先后涨价，乐视视频表示，</span><strong><span>“没有资格涨价”</span></strong><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/210703046962e260da8d9902.50070447.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：乐视视频<span>官方微博</span></span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>打工人对“996”生活怨声载道，乐视则凭借“没有996”、“没有内卷”、“每周工作四天半”的公司制度成为一股清流。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205672801862e260dc73e073.63460402.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span><span>图源：乐视视频<span>官方微博</span></span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>靠着自嘲营销在网络上刷存在感的乐视，逐渐扭转了公众形象，但这背后是乐视捉襟见肘的营销预算。乐视曾回应网友：</span><strong>1年喜提4次微博热搜，没花钱。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/38794396662b974394a3da6.63273158.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>“家底丰厚”的乐视</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《红楼梦》里，刘姥姥曾形容荣国府：饿死的骆驼比马大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年年底，乐视在内部信曾表示，“2021年11月，电视端运营、移动端运营、广告商业化等业务均提前达成年度经营目标。在不考虑历史债务影响前提下，最新经营数据实现了经营利润和现金流的双平衡，这是过去数年来第一次，意义非凡。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/153729791462e260e00799e6.69308910.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：乐视视频<span>官方微博</span></span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乐视不仅没倒闭，还能实现营收，养活400多名员工，离不开贾跃亭为乐视攒下的厚实家底。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首屈一指的就是乐视视频，可以说是乐视的“钱袋子”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为国内最早的视频平台之一，乐视率先开创了购买影视作品版权的先河。</span><span><strong>“平台购置正版版权，用户付费收看”的盈利模式，如今是其他视频平台的生财之道，也让乐视得以“续命”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是“平均每年能带来1000多万收益”的《甄嬛传》，更是被乐视当作“摇钱树”。乐视曾在2013年以9亿元全资收购《甄嬛传》，虽然2018年因为资金链危机分销给了优酷，但乐视至今仍有美版《甄嬛传》的独播权，保着乐视的“荣华富贵”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/192494800462e260e1dde981.84045658.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>▲</span><span> </span><span>图源：乐视视频APP</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，《太子妃升职记》《芈月传》《翻译官》《白鹿原》《平凡的世界》等早期积累的经典影视剧独家版权，也在为乐视视频不断输送会员。根据公司年报显示，乐视网2021年上半年新增了约300万注册用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了视频内容，乐视硬件方面的项目也在为公司“输血”。前乐视视频CEO、现乐融致新CEO张巍曾透露，2020年乐视的电视产品就卖出40万-50万台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/46978155362b9743fae3289.16804197.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>“不能躺平”的乐视</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人说，乐视比贾跃亭努力多了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用这句话来形容现在的乐视，或许更加准确。看似“神仙生活”的乐视，其实一直努力自救。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了“啃老本”，乐视在视频内容运营上也在寻求突破。</span><span><strong>6月30日，快手宣布与乐视视频就乐视的独家自制内容达成二创相关授权合作。</strong></span><span>这意味着快手用户可以在授权情况下，对乐视视频独家自制版权作品进行剪辑及二次创作，并发布在快手平台内。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于乐视视频而言，这是顺应长短视频深度融合发展趋势的选择。通过接入快手小程序平台，乐视视频可以更快捷地触达快手超3亿的日活用户，更好地实现会员增长和内容变现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>硬件方面，乐融致新仍在进行新产品的研发创新。</span></strong></span><span>去年5月，60多款智能生态产品在乐视生态大会上亮相，包括智能电视、智能穿戴、个人护理、智能门锁、厨卫等60余款新产品。生态大会的口号“我们回来了”，似乎让人看到了贾跃亭“生态化反”的野心回归。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>乐视手机也不断有新品上市。</span></strong></span><span>去年9月，乐视发布了旗下首款搭载华为鸿蒙系统的5G手机乐视S1，售价为1599元。599元起的乐视Y2Pro手机，在今日（7月28日）全网现货开售。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，最近乐视直播间开始公开招聘具有</span><span><strong>“教培行业从业教师”</strong></span><span>经验的主播，或许是想复制新东方逆风翻盘的路子，在直播行业有进一步动作。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/13325749862e260e3d000e3.81103412.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span><span>图源：乐视视频<span>官方微博</span></span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管黑猫白猫，捉住老鼠就是好猫。在机会不断的互联网时代，能让乐视挣钱生存的领域，乐视都在努力尝试。</span></p>
