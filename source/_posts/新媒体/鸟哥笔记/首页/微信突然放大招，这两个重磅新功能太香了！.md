<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134159788262d9c10355fc59.15642900.jpeg" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信更新至<span>8.0.26版本之后，新增了<span>“消息通知横幅”开关，可以通过<strong>“我—设置—消息通知—消息通知横幅”</strong>的路径找到<span>，替换了旧版本的“看视频号时显示新消息”按钮</span>。首次使用时，可以看到<strong>“打开后，在微信中收到消息时，你可以在屏幕上方出现的横幅中查看和回复消息。”</strong>的提示。</span></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40965400062d9c10386a490.81548769.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开启<span>“消息通知横幅”功能后，</span>当你在微信中收到消息，消息会在当前界面的顶部以横幅的形式弹出，点击横幅即可在当前界面查看和回复消息。进入浮窗式的聊天界面后，点击上方的“关闭”<span>或“标为未读”按钮，均能进行相关操作并退出浮窗界面。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/48059624562d9c10408c914.63491630.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这个功能并不局限于群聊，而是在微信中的任一界面</span><span>（除了微信聊天的首页）</span></strong><span>，不管你是正在看公众号文章、视频号，还是在小程序、通讯录、朋友圈，都可以通过<span>开启</span><span>“消息通知横幅”功能，在当前界面查看和回复消息，而不需要暂停当前的事情，减少了切换界面以及翻找进度的麻烦。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/101815499662d9c1056a3516.34424115.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不得不说，对比以前需要整个页面切换，再返回来翻找之前看到哪里的情况，这个功能确实是解决了一大痛点，还可以提高多人聊天的效率，简直是做社群运营和客服的福音。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更新<span>微信 8.0.26 for iOS版本后，<strong>聊天界面的输入框新增“全屏输入”功能。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/194184949562d9c105dced22.09294947.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开启这个功能的方式有三种：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>• </span></strong><strong>打开聊天界面，长按空白输入框，</strong>即会出现“全屏输入”的功能按钮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>• </span></strong><strong>当聊天界面的输入框字数超过两行的时候，输入框左边会出现“^”的标志，</strong>点击即进入“全屏输入”模式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>• </span></strong><strong>在聊天界面的输入框输入部分文字后，长按或双击即会弹出功能菜单，</strong>点击右边的箭头，即可找到在第二页的<span>“全屏输入”功能。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/176342131762d9c1086b3b44.47534946.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信这个功能也是值得夸一波的。输入框就那么点大，没更新之前只能“<span>蜗居</span>”，无论是修改错别字、语句、空行等操作都相当不方便。更新之后可以进行更便捷的预览了，吵架的时候“写小作文”也能加快输出了</span><span>（bushi）</span><span>。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在微信 8.0.26 for iOS版本中，聊天界面的输入框还新增了简繁体切换功能。</span></strong><span></span><span>在聊天过程中，只需要长按选中想切换字体的文字，在弹出的功能菜单中点击<strong>“简⇌繁”</strong>，就可以将文字转换成自己需要的形式了。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对比之前的微信版本，<span>8.0.26版本的部分页面布局发生变化。比如，</span>在“我—设置—通用—聊天记录迁移与备份”的页面中，功能按钮的名称发生改变，并新增了功能图标和功能说明。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/197806904962d9c109720da9.95602855.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>综上，微信此次更新的重心在提升用户的聊天体验上，推出的功能也大多实用。看来，微信或许要重新开始认真搞社交了。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为国民App，微信的一点小更新足以带来蝴蝶效应。但时不时的小更新，是保持大家新鲜感的重要因素，也是在成为日常应用后获得热度的秘诀。这不，#微信打开时支持消息通知横幅#分分钟上热搜，总阅读超2.6亿，谁还能说微信过气？<span></span><span></span><span></span><span></span><span></span></span><span></span></p>
