<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>继猫粮、狗粮之后，鲜花着锦的宠物市场，又刮起了一阵“<a href="https://www.niaogebiji.com/tag/%E5%AE%A0%E7%89%A9%E5%8F%8B%E5%A5%BD" target="_blank">宠物友好</a>”之风。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月底，在北京市朝阳区曙光西里的星巴克甄选店，“爪布奇诺” 早已成为星巴克传统周边之外，一道靓丽的风景。顾客扬扬刚带着自家的田园犬，在户外草坪区吃完了一杯星巴克提供给猫狗的免费鲜食——爪布奇诺。据他介绍，这是一种动物可食用的奶油，在北京少量“宠物友好星巴克”予以提供。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从“宠物禁止入内”到类似于星巴克这样，能够携带宠物，甚至可以为宠物提供饮食、娱乐、健身等增值服务的餐饮店、商超、酒店或是航班，被称为 “宠物友好” 场所。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据吉祥航空官网发布的信息显示，今年7月中旬，其推出“宠物畅飞卡”，旅客可以自2022年9月至2023年暑假期间，携宠物乘坐上海至三亚、成都等在内的共10个特定航班，并根据后续旅客需求，做出灵活的线路调整。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们自2015年开业以来，就可以携带宠物过来度假，酒店会提供免费的宠粮、玩具等。” 宠物友好型酒店“大隐于世”的民宿经理孙雪告诉燃次元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在大众点评搜索 “宠物友好” ，包括餐饮、酒店等在内的相关商户共1148家。小红书上，关于 “宠物友好” 的笔记超10万篇。“宠物友好”场所日渐增多背后，一方面是宠物市场规模的疯涨。艾媒咨询数据显示，2022年，中国宠物经济产业规模将达4936亿元，同比增长25.2%，预计2025年市场规模达8144亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，则是人们对宠物的关注早已不再局限于“吃穿”。“消费者对宠物的关心，已经从‘温饱’ 逐渐过渡到 ‘多元化’需求。”宠物行业长期观察者喻森表示。艾媒咨询数据显示，国内与宠物相关的产业链在上游已经包括了购买、领养、食、宿、用，下游则包括了代养、托运、寻宠、美容、娱乐、殡葬等多方面。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13791869563228c033281a0.52742146.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/宠物各产业链有关品牌  </span><span>来源/艾媒咨询 燃次元截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，尽管市场很大，但现阶段的“宠物友好经济”尚处于发展的初级阶段。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了宠物卫生与健康等老生常谈的弊端之外，宠物家长素质的差异等原因，也导致了“宠物友好”场所暂时未能变成对“人类友好”的空间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在未经宠物家长驯化的猫狗充斥的公共空间里，城市的人文环境、人类消费者的健康甚至人身安全，都备受挑战。在与多家宠物友好店的店主交流的过程中，燃次元得知，打架、咬人、不戴纸尿裤四处撒尿等不文明、不卫生的行为，直至现在在店里也时有发生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也正因此，一些点评类App上不少网友提到，部分餐饮店经历了从“禁止宠物入内”到 “宠物友好”场所，再到 “禁止携带宠物”的多重变化。而这也是宠物友好经济规模化面临的较大瓶颈，同时也导致了目前的大多数宠物友好商业体，依旧是“为爱发电”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/1455856016142b5abe0ca06.17059714.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>“宠物友好”场所兴起</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“宠物友好”场所的连年递增，以及服务项目的日渐丰富，折射出宠物与人类关系的变迁。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，全国的宠物友好餐饮店已并不罕见。其中，位于北京市、开业于2013年底的 “百好乐宠物友好餐厅”（</span><span>以下简称 “百好乐”</span><span>），就是最早开始营业的宠物友好餐厅之一。这里除了可以携带宠物之外，还为猫狗提供了专门的餐饮。“家长们”甚至可以为自家“毛孩子”办生日party。除此之外，这里还为“家长们”提供了短期寄养宠物的服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月下旬的一个下午，任枫和他的女朋友，带着刚养了1个多月的小狗，来到了百好乐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“宠物友好店目前还分布不均，我居住在北京市大兴区，至少现阶段的大兴区几乎没有，我们就带她上这里来了。” 任枫指着四处奔跑的小金毛，对燃次元表示，“小金毛最近肠胃不好，我没让她吃店里的鲜食。不过，这里环境好、空间宽敞，包括周边的围栏措施，安全系数也比较高。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“最重要的是，小金毛在这里交到了大金毛朋友，我觉得这对于她的身心健康，还是很有必要的。”任枫说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018年1月，开始养名为“可乐”的英斗犬的“妈妈”，也是百好乐的常客之一。可乐妈表示，之所以常带可乐去店里，一是因为离家较近、方便。但更主要的原因，还是因为这里有可供嬉戏的大露台，自家狗狗能在这里与不同品种的狗狗充分社交。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>寄养服务，也是可乐时常会享受的，“有时候，我会出差，或者去异地的朋友家玩，可乐没人照顾的时候，我就会把她寄养在百好乐，这里的设施齐备，服务周到，可以把她照料得很好。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>百好乐之外，在出京的时候，宠物友好民宿与酒店，也是可乐妈在选择住宿时的必要条件。“比如，在南京的时候，我就曾和朋友一起住在一栋别墅里，从室内到户外都是可乐可以溜达的空间。值得一提的是，户外园区里还提供捡屎袋，我只需要额外支付一笔清洁费，便不用担心自家的狗狗给民宿带来麻烦。这些周到的服务，作为‘铲屎官’的我们还是很感激的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了餐饮店、酒店和民宿之外，宠物商城也是近来兴起的“宠物友好”场所之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5月20日，位于杭州的宠物街区开业，其中就包括一个麻雀虽小、五脏俱全的室内宠物综合商场——“宠这里”。燃次元探访后发现，该商场为宠物提供了宠物健身房、宠物游泳馆、宠物幼儿园、洗护区与美容区等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/28745687963228c037d5750.42102170.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/杭州市“宠这里”商场  </span><span>来源/燃次元拍摄</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在“宠这里”的餐厅中，一对情侣正在喂两只狗子吃披萨。他们告诉燃次元，这两只毛孩子一只已经8岁了，还有一只不足1岁。自从“宠这里”开放之后，隔周就会带“俩娃”到这个 “宠物天堂”享受一下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“在家呆久了，这两个‘孩子’就容易闹脾气，他们比较喜欢来这儿撒欢。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/9082723746142b5ac73c549.43567204.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>被关注的“精神需求”</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“除了宠物市场规模的飞升，宠物友好服务的涌现，也得益于家长对宠物多元化需求关注度的提升。”喻森对燃次元表示。人类在解决了温饱问题之后，就会追求精神需求。宠物也一样，当宠物家长们能满足猫狗的饮食需求后，若还有消费能力，就会考虑他们的精神需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“66”是一只金毛，2013年左右，苦于“66”出门无公共场所可去的 “66妈”，深知自己的 “孩子”常困于家中的孤单。“没有条件，就创造条件。”就这样，66妈给66搭建了可以与其它狗狗社交的场所，即百好乐。彼时，百好乐所体现的、唯独的宠物友好，只是可携宠入内。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2014年，随着养宠消费者占比的极速增长，66妈给这些猫狗新增了餐饮服务， “我只是不希望它们只能眼巴巴望着家长享用美味。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8年后的今天，在消费者的反馈之下，66妈和后厨们，不断改进着餐厅的宠物鲜食菜单，开发了专门的低过敏肉种，比如鸵鸟肉、鹿肉等，酱料也完全不添加任何防腐剂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“不但是养宠的人变多了，宠物家长也更关注毛孩子的情绪了，来店里消费的客人，也不再像八九年前那般，主要以外国人、常客为主，大量的新客人不断涌现。”谈到百好乐消费群体的变化，66妈如是说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>百好乐近十年来的变化，也正是国内宠物友好市场发展的一个截面，带动了许多后继者的加入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>位于北京市丰台区，“宠物友好”、“露营”、“棋牌消遣” 多重主题相结合的“16露营咖啡茶室” （</span><span>以下简称 “16咖啡厅”</span><span>），便是其中之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年底，海归年轻情侣干锦辉和王雪瑶回国创业。因为爱狗也养狗，他们毫不犹豫地把宠物这一元素嵌入到了咖啡厅中。上下两层的咖啡厅，一层适合宠物玩耍，二层适合家长休闲。900平米的超大空间，完全可以给猫猫狗狗提供充分的社交范围。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王雪瑶介绍，16咖啡厅除了为宠物提供玩具、袋装零食，以及专门的小号桌椅之外，还会为办party的宠物准备鲜食。“能让狗狗们自由自在地玩耍”，是用户对16咖啡厅最多的评价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>孙雪也对燃次元表示，宠物需要社会化，尤其是随着养宠物的家庭更深入地把宠物当作家庭的一份子，希望带他们共同出行后，转型成为宠物友好酒店，就成为了郊区民宿必须要面对的一个课题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“在郊区的民宿产业里，大部分都是允许携带宠物的。”孙雪介绍道，“像我们这样的民宿，大都有独立的院落，不仅满足了家长们休闲娱乐的需求，也给宠物嬉闹提供了很大的空间。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>66妈对此表达了同样的观点，“尤其是犬类，属于狼族，是一种群居动物，长期和同类打交道，才会身心健康。”66妈举例表示，比如当一只狗来到餐厅，闻到了陌生人和动物的气味，社会化过程便慢慢启动，逐渐接近于自己的本性，性格也会变得更阳光。相反，长期居家的狗狗，偶尔出门，就会显得比较乖戾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>16咖啡厅的干锦辉与王雪瑶，更是认同宠物社会化的重要性。干锦辉透露，未经驯化的狗狗，见到人可能就会咬。公共场所的交际，为他们提供了习得社交礼仪的空间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81439268263228c05215e96.15256732.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/16露营咖啡茶室在小沙发上聚会的狗狗  </span><span>来源/16露营咖啡茶室提供</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于来到16咖啡厅的宠物来说，在家长们的带领下，早已形成了属于自己细分品种的社交圈。比如，田园犬的家长们会自建一个微信群，商定时间，定期带这些“孩子”到咖啡厅小聚。社会化的网络就这样逐渐编织起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“当然，从整个市场环境来讲，宠物友好商业体的增加，也得益于社会包容度的增加。” 喻森表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/4756644006142b5ad08cc02.64114870.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>“宠物友好”道阻且长</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，看上去热闹的消费，并未能点燃 “宠物友好经济”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管得益于国内年轻人对宠物的追捧，在北京、上海、杭州、成都等一线及新一线城市，宠物友好餐饮店、酒店、航班甚至购物中心，都为毛孩子们提供了便利的出行与娱乐方式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，当下宠物友好经济的前景，依然受限于多种因素。比如，市场规模、行业端公共场所宠物的卫生与健康隐患，以及相关行业政策的缺失等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，虽然国内宠物市场规模已经呈爆发式增长，但这与部分发达国家之间的差距还很大。App数据显示，2021年，美国宠物市场规模约7972亿元。《2021年中国宠物行业白皮书》数据显示，2021年中国宠物市场约2490亿元。不及美国的三分之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一个产业的发展中，“奇点” 规模被认为是该产业突破瓶颈、引发飞轮效应、带动全产业链正向增长的一个拐点。很显然，国内的宠物市场还尚未及“奇点”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这与宠物友好场所中，宠物所面临的的健康与安全等隐疾，有着直接关联。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当一些餐饮店与酒店给了猫狗自由，店里的宠物毛就总是会漫天飞起，地上还不时可见宠物排泄物。“为了应对这些卫生问题，我们每天都会给宠物用清洁剂来消毒，从而避免狗狗身上携带支原体病毒、造成感染。”66妈表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，宠物的“口粮”，尤其是鲜食领域，也成为了很多宠物餐饮店不敢轻易涉足的业务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“星宠派”是16咖啡厅在宠物鲜食方面的外部供应商，主要为宠物提供蛋糕、月饼、粽子甚至年夜饭等鲜食。“我们会询问猫狗年龄、具体的品种，还有他们的饮食习惯、是否有过敏等，从而给他们量身定制鲜食。”“星宠派”创始人李特表示，但更多自身提供鲜食的宠物友好店，很难达到这种标准，这也成了宠物友好餐饮店发展的瓶颈之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/83118436963228c06204a72.24052487.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/星宠派制作的宠物月饼  </span><span>来源/星宠派小红书</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过， 对于宠物友好店的发展而言，更大的瓶颈来自于消费者对自身饮食安全、人身安全的顾虑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这种顾虑的原因还是宠物家长素养的差异。” 66妈表示，来她店里消费的顾客与宠物中，常常会有 “护娃家长” 与 “熊孩子”，“比如，为了店内的卫生环境，我们要求所有的公狗进店都必须穿纸尿裤，否则会撒得到处都是。但很多家长给公狗戴上纸尿裤进店后，会趁我不注意偷偷摘下。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在16咖啡厅，也有30%的狗狗因为没完成社会化，而出现在店里打架、咬人的情况。“一些家长欠缺培育宠物的科学方法，所以社会行为会存在一些问题，我就被来店里的狗狗咬过两次。” 干锦辉表示，基于此，他们咖啡厅对于不同种类的宠物进店，也做了相应的规定，即周末只允许小型犬入内，工作日中型犬、大型犬可入内。“因为不同种类之间的狗狗更易起冲突。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卫生疑虑也是许多非养宠消费者反对宠物友好的原因。在某点评App上，燃次元发现不少用户吐槽“自己给狗狗用了 ‘服务人’的杯子喝水后，服务员就把被子扔了，这是对狗狗的不友好”，但帖子下面得到的回复多半是 “我觉得服务员做得对”、“这才是卫生的做法”等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正因此，为了消除消费者对食品卫生的顾虑，16咖啡厅的所有宠物家长用的餐具、宠物吃的食品，均采用一次性的餐具。“毕竟，我们需要照顾所有顾客的感受。”王雪瑶表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但王雪瑶和66妈的这种“用心”似乎并没有收到相应的“回报”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于位于非商圈的北京市丰台区某下沉广场地区，16咖啡厅900平米的租金，每月仅2700元，可谓是市里少有的 “房租洼地”。并且，干锦辉与王雪瑶也只招募了两位员工，把人工成本控制在了每月2万元左右，宠物食品的成本大约也在2万元左右，“这都是我们优化成本后的结果。”王雪瑶谈到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但，远离商圈也意味着较高的获客成本，顾客大多需要专程赶来16咖啡厅，更大程度降低了非养宠人群前来消费的可能。“在垂直消费市场之下，从去年年底至今，店里就没再盈利——毕竟，宠物鲜食的质量必须保证，成本无法再降了，一般4寸的宠物蛋糕进价就要188元；而售价也难以再提高，一杯宠物家长的饮料，我们已经卖到了40多元。”王雪瑶无奈地表示，把宠物友好咖啡店做下去，纯粹是因为对猫狗的喜爱，“想给他们一个难得的社交圈，我们会一直做到支撑不下去的时候。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“宠物友好餐饮店是真的不挣钱。”66妈表达了同样观点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，对于宠物文化发展时间较短的国内市场来说，想要进一步发展宠物友好店，需要拓展的并不是养宠人群，而是对宠物依然抱有某种顾虑的非养宠消费者。但若想要扩大包容宠物的人群，其根本还需要宠物家长去培育毛孩子，改变社会对他们的刻板印象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许正如许多业内人士所说，欲增宠物友好市场规模，除需餐饮店、酒店等更细致地考虑宠物自身的需求外，还必须首先让宠物变得“人类友好”。在这一点上，需要宠物家长、公共场所与有关部门政的策形成一股合力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由此，宠物友好商圈才能进一步扩大，也才可以不再 “靠爱发电”。</span></p>
