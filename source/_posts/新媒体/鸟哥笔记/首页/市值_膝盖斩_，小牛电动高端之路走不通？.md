<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小牛电动的下沉，又有了新动作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">日前，小牛电动推出售价为899元的儿童两轮电动车，采用12V4.5AH铅酸电池，时速每小时5公里，适合3岁以上的孩童，续航时间为90分钟。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这意味着，小牛电动的单价进一步探底。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，执着下沉市场的小牛电动，高端怎么卖不动了？两面夹击的小牛电动，到底面临哪些严峻的挑战？小牛电动掌舵人做起了“造车梦”，对其影响几何？</p>
<p>光环褪色，高端卖不动了？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为两轮电动车新势力的代表，小牛电动以高端之姿入局。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">相当长一段时间，两轮电动车市场创新不足、同质化严重，价格成为企业博弈的焦点，价格战之下，一批中小企业惨遭洗牌。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实上，雅迪正是在价格战中脱颖而出，反超爱玛成为行业的“一哥”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此背景下，两轮电动车从红海走向血海。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">好在，2015年行业迎来了变数，小牛电动推出智能两轮电动车，以高端形象示人，一出场就收获年轻人的青睐。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">于是乎，两轮电动车赛道，也上演了传统势力与新势力的对决。</p>
<section>
<section>
<p>据公开数据显示，《电动自行车安全技术规范》于2019年4月15日正式实施，按照新规国内两轮电动车将迎来以旧换新潮，新旧势力的厮杀白热化。</p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/126529988762c2cb0f530d19.38446151.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">小牛电动的儿童电动车</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如此一来，智能化成为行业的共识。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">雅迪、爱玛等传统势力渴望上跃，纷纷布局智能两轮电动车，而后入者九号凭借“更智能+更高端”的打法不断攻城略地，一跃成为两轮电动车赛道最年轻的重要玩家。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">譬如，雅迪中高端产品冠能系列也拥有了骑行导航、安全出行、无感解锁、智能安防等功能，而高端的VFLY系列更是以“全场景AI智能系统”为卖点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这意味着，小牛电动的竞争优势遭压缩，其基本盘既受雅迪、爱玛等旧势力侵蚀，又受九号等新势力蚕食。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不得已，小牛电动将突围的目光瞄向了下沉市场，试图以价换量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据公开资料显示，小牛电动GOVA系列直接针对下沉市场，2021年的新品有半数都属于GOVA系列，于是2021年销量为103.79万辆，同比增长72.5%；2021第四季度平均售价下降了7.3%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">尽管如此，小牛电动的困局依旧。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">下沉市场是雅迪、爱玛的主战场，小牛电动的触角远不及旧势力，低价扩张的策略能持续多久仍要打一个问号。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">须知，下沉市场对线下门店较为依赖，而小牛电动2021年的门店总数为3108家，与对手们动辄数千家、上万家的体量仍相去较远。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一名私募人士告诉锌刻度：“两轮电动车的新贵，大多将主要精力放在智能创新、技术迭代上，鲜有注重下沉市场的，一旦传统厂商补上了智能化短板，小牛电动的故事就不好讲了，这也是其市值不断走低的重要因素。”</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;">净亏损放大，品控遭诟病</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">销量向上、市值向下，成为小牛电动的真实写照。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年2月，小牛电动市值为41.06亿美元，这之后一路下滑几乎没有抵抗之力，截至2022年7月1日市值仅剩下6.62亿美元，缩水幅度高达83.88%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">换而言之，小牛电动非但跌破了发行价，市值更是遭遇“膝盖斩”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">之所以如此，与其业绩不被资本认可有莫大的关系。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据同花顺数据显示，2022年第一季度，小牛电动营业收入5.75亿元，同比增长5.14%，环比减少41.68%；净利润为-2957.62万元，同比亏损幅度扩大450.40%，环比由盈转亏；从毛利率来看，小牛电动从2021年第一季度的23.8%下滑至2022年第一季度的19.13%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14040739762c2cb0fa1bdf6.95570608.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">净利润恶化</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以上可见，小牛电动以价换量的打法，令其盈利能力不断恶化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">亏损加剧之外，小牛电动因为品控问题屡遭处罚，亟需苦练内功。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">譬如，小牛电动在其官网中宣称车辆合格证记载的续航里程为检测机构测试所得出的续航数据，实际为内部实验室的自测结果，不具备权威性，属于虚假宣传行为，被北京昌平区市场监管局责令其立即改正违法行为并作出罚款30万元的行政处罚。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再譬如，小牛电动的全资子公司江苏小牛电动科技有限公司多个型号的产品因照明、短路保护、使用说明书、尺寸限值、蓄电池防篡改项目抽检不合格被常州市武进区市场监管局罚款29万元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而在新浪黑猫投诉平台上，小牛电动也屡遭诟病。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">相关的投诉为1723条，涉及无法定位仍收取服务费、续航与宣传严重不符、后轮毂质量差、刹车异响、不骑行时自动掉电、线上下单线下拒提、车后座焊接点开裂、车辆触发报警后异常断电等质量问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">需要注意的是，小牛电动的智能电动车售价原本就不菲，每年还有68元的服务费，增加了后续使用负担，从而引发部分对价格敏感的消费者的不满。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一名业内人士表示：“小牛电动此举效仿的是特斯拉，卖硬件之外也通过软件搞钱，但智能化并非刚需，且大部分品牌没有年费这一说，市场教育依旧道阻且长。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从这个角度来看，小牛电动遭资本冷遇，并非毫无道理。</p>
<p>掌舵人“移情别恋”，突围缺少“主心骨”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">内忧外患之下，小牛电动还遭遇了掌舵人李一男的“背刺”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据多家媒体报道，李一男推出自游家品牌，专注新能源汽车赛道，旗下首款量产车型为中大型SUV，有纯电与增程式两个版本，纯电版续航里程为550公里，而增程版满油满电状态下的综合续航超过1257公里，对标的是理想ONE与岚图FREE，将于2022年9月启动交付。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为小牛电动的灵魂人物，李一男的“移情别恋”，对其影响肉眼可见。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">毕竟，人的精力有限，李一男的重心偏移之后，势必对小牛电动关注不如以前，而两轮电动车当下鏖战正酣，且小牛电动正遭遇新旧势力的两面夹击，一个不留神则可能掉队。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">须知，小牛电动的智能技术壁垒并不高、下沉市场的根基并不厚。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而且，小牛电动为自游家提供了与人才、技术等方面的支持，譬如有报道称小牛电动设计副总裁刘传凯将加入NIUTRON，其团队操刀小牛电动M1的概念设计，这意味着在小牛电动突围的关键之际，自身的实力或打了折扣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/118757767462c2cb0fcb59f0.43713636.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">小牛电动跌破了华尔街的所有预测</h4>
<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，李一男的“造车梦”，也不乐观。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从已知参数、海报图来看，自游家契合主流趋势，可以满足用户的主流需求，但在新能源汽车百花齐放的格局之下，仅仅如此是不够的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以比亚迪为例，耕耘新能源汽车市场多年，长期处于汽车鄙视链的底端，直到近一两年凭借刀片电池、混动技术才起势，成为不可忽视的重量级选手。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，中规中矩的自游家要想脱颖而出，谈何容易。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">更为关键的是，小牛电动在两轮电动车的品牌知名度、技术积累难以延续到新能源汽车上，后者几乎从零起步。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从这个角度来看，小牛电动与新能源汽车相互驱动的愿景或将落空。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总而言之，小牛电动正在遭遇“冰火两重天”，注重下沉市场虽然提高了销量，但盈利能力并未改善，更为关键的是传统实力在补智能的短板，后入者的智能技术更强，衬托之下小牛电动的智能标签越来越模糊。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，小牛电动亟需新故事。</p>
