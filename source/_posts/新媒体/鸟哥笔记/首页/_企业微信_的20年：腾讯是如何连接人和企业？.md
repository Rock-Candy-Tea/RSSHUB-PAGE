<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从19年开始，我就一直负责公司内部的微信矩阵，最开始是做微信小程序，</span><span>后面开始接触<a href="https://www.niaogebiji.com/tag/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1" target="_blank">企业微信</a>，再加上我自己还开通了个人微信公众号。这3年以来，基本上我每天都在跟微信内这些产品打交道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于小程序、公众号和视频号，我之前写过很多文章，但是从来没有提及企业微信，因为在我看来</span><strong><span>企业微信是腾讯最「复杂」与「曲折」的产品。</span></strong><span></span></p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「复杂」体现在从企业微信身上，我们可以看到腾讯过往多款产品的影子</span><span>——从PC互联网时代的RTX和企业QQ到移动互联网时代的企业号；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「曲折」体现在我们今天看到的企业微信跟最开始的版本已经是完全不一样</span><span>的两款产品；</span></p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我一直认为，我们要了解一款产品，最重要的是要弄清楚这款产品的市场定位，光记产品有哪些功能根本没用。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span><strong>定位是一切产品功能的本源，定位反映了产品经理对市场的认知——市场缺什么，这款产品应该做什么。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>当我们了解这款产品的定位，自然而然就能理解甚至自己推导出这款产品应该有哪些具体的功能。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2001年的初代RTX开始，腾讯「泛企业微信」总共有以下7款产品，每款产品都有一个清晰的市场定位。</span><span>接下来我就从这7款产品来</span><span>梳理一下腾讯连接人与企业的20年。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1 2001：RTX——企业内部的IM工具；<br></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2 2007：企业QQ——在线客服+营销平台；</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3 2012：公众号——企业内容发布平台；</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>4 2013：服务号——企业轻量化服务承载平台；<br></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>5 2014：企业号——依附于微信的移动办公协作平台；</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>6 2016：企业微信1.0——独立的移动办公协作平台；</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>7 2019：企业微信3.0——企业私域流量运营工具；</span></strong><span></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span>一、2001年：RTX——企业内部的IM工具</span></strong><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3853709562fb5f268b9c87.61820760.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><em><span>QQ用户数的激增，给腾讯带来了巨大的服务器维护成本</span></em></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>QQ自1999年问世后，用户量快速增长，到了2001年，注册用户数突破2000万。而当时腾讯并没有为QQ找到合适的商业模式，QQ用户很多，但是不知道怎么赚钱。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2000年11月，腾讯推出了“QQ俱乐部”的会员服务，QQ用户一个月只要缴纳10元会费，就能享受QQ靓号、网络收藏夹和好友列表保存等增值服务。这是腾讯乃至中国互联网史上第一个“氪金”业务，自此腾讯走上了可耻但有用的“氪金”之路。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/198811593662fb5f2704f0e8.44574731.gif" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>可是腾讯第一个“氪金”业务并没有成功，半年内只有3000个用户成为会员。因为彼时在线支付工具还没有像今天一样成熟，</span><span>用户需要去邮局转账给腾讯。即使用户真的很喜欢这个功能，谁又会专门跑一趟邮局呢？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>既然向</span></strong><strong><span>个人用户直接收费有困难，腾讯就只能向企业用户收费了。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么腾讯可以给</span><span>企业用户提供什么付费服务呢？当然还是老本行，即时通讯工具。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯于2001年开始研发一款企业内部IM产品，这款产品叫做Business QQ，后改名成RXT（腾讯通）。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213622330562fb5f276d2088.44129364.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>RTX相较于QQ最本质的提升总结起来就是两方面——更安全，更易管理。</span></strong><span>RTX服务器部署在企业内部，不用担心信息外泄。由企业开通账号供员工在公司局域网内使用，所以RTX没有跟个人QQ打通，用RTX只能跟同事沟通。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>RTX账号所有权在公司，公司可以根据你的岗位职能给你开通相应的权限，也可以对</span><span>员工会话记录进行存档和查询，方便管理。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这里我们可以看出很多现在企业微信的功能，例如邀请制、会话存档、权限管控等，都可以追溯到20年前的RTX。目前RTX的官网依然支持访问，只是在官网首页增加了一个企业微信的链接。到后面我们可以发现，将接力棒交给企业微信的产品远不止RTX。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/200226562262fb5f27ba0122.14851167.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>RTX</span></span><span>后面还集成了一部分OA功能，但是整体定位还是企业内部的IM工具，其目标是</span><span>可以为企业节省通信和出差费用，提升企业的信息化程度。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><strong><span>可是RTX</span></strong><strong><span>存在一个</span><span>弊端 ——只能在局域网内跟同事沟通，没有跟个人QQ进行打通。 </span></strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span><strong><span>二、2007年：企业QQ——在线客服+营销平台</span></strong></span></strong><span><span></span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时企业要跟个人QQ进行打通主要是基于两方面诉求：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><strong><span>1 在线客服：</span></strong><span>传统</span><span>电话客服是一对一的服务模式，客服一次性只能服务一个客户，缺乏弹性。客户来电多的时候，客服忙不过来，会漏接客户电话；客户来电少的时候，客服就会很闲，造成企业人力资源浪费。而如果支持客户通过QQ直接跟客服沟通，那么客服同时可以承接多个客户，客服团队的服务与管理效率得到了提升。</span></em><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><strong><span>2 营销平台：</span></strong><span>海量的QQ用户对于企业来说是宝贵的客户资源，如果支持企业直接对QQ用户进行触达。相较于地推、纸媒、电销等传统营销模式，使用QQ对客户进行营销触达具有<strong>客群精准、成本低廉、互动感强</strong>等优势。</span></em><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/172300457362fb5f28281c67.49405792.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>基于企业这两方面诉求，<span></span></span></strong><strong><span>腾讯于2007年上线了企业QQ客户管理系统。</span></strong><span>企业只要提交材料完成审核，就可以获得一个专属企业QQ号。</span><span></span><span><span>个人QQ用户可以搜索企业QQ号跟企业进行直接沟通，企业<span>QQ支持消息群发触达用户。</span></span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/56986903862fb5f28763a60.14752211.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外企业QQ跟个人QQ一样，还有一个「企业空间」功能。<span>「企业空间」是企业的</span></span><span>轻量级门户网站和BBS。企业可以在空间内发布企业简介、最新信息、产品研发状况等，这个功能形态有点类似于后来的微信公众号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时腾讯对于企业QQ的愿景是改变企业与用户之间的沟通方式，打造企业在线精准营销的新平台。</span><strong><span>如果说RTX的定位是企业内部IM，那么企业QQ的定位就是在线客服+营销平台。</span></strong><span></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span>三、2012——14年：公众号——包打一切</span></strong><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>前面我们主要梳理了PC互联网时代，腾讯做的两款企业端通信工具。而在移动互联网时代，腾讯通过微信巩固了自己在个人社交领域的霸主地位后，却没有选择做一款类似RTX的企业内部通信产品。</span></span></span><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>原因很简单，因为腾讯认为微信完全可以满足企业端的通信诉求。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先在移动互联网时代，企业内部即时通讯成了一个伪需求。</span></strong><span>为什么PC互联网时代要单独开发RTX才能满足企业内部的沟通诉求？为什么不能直接用个人QQ呢？</span><span>其实主要就是两个原因：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><strong>1：安全：</strong>QQ盗号频发，使用QQ当作企业沟通软件，如果被盗号会带来信息泄露风险；</em></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><strong><span>2：管理：</span></strong><span>QQ用户印象为三低——低龄、低学历和低收入。上班时间挂QQ，无法存档查询你的聊天记录，不知道你是聊工作还是跟朋友在吹水，管理难度大；</span></em><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先从安全层面，随着网络安全技术的发展，微信号很难被盗，所以安全性对于广大中小企业来说完全够用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再看管理层面，软件运行载体从电脑变成了手机，你无法知道员工手机上打开的是哪款app。移动互联网时代，企业是无法阻止员工划水的，除非上班时间上交手机。</span><span>既然无法管理，那就干脆别管了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所以对于腾讯来说，微信完全可以充当企业内部IM工具。</span></strong><span>在这个大前提下，</span><span>企业其他的一些诉求，腾讯谋求通过在微信内新增「插件」来满足，这个「插件」就是微信公众号。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>我们常说的微信公众号其实可以细分为三类：订阅号、服务号和企业号。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1、订阅号：</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2012年，微信公众号上线，赋予了企业直接触达C端用户的能力。有了公众号，商家节省了两方面的成本。第一是短信推送费，商家开始通过公众号给用户推送信息。第二是门户网站搭建费，在公众号之前，用户要想在网上找到企业相关信息，企业需要自己搭建门户网站。而有了公众号之后，任何一个小企业都能开通一个公众号，在公众号放企业相关信息。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/162767001262fb5f28c04f00.87350965.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自公众号问世后，路边广告牌上网址和电话号码不见了，取而代之的是公众号二维码。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2、服务号：</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>2013年，</span><span>微信将公众号拆分成订阅号和服务号。<strong>如果说订阅号的定位是连接人与内容，那么服务号的定位就是连接人与服务。</strong>服务号为企业开放了很多接口，例如获取用户位置信息、模版消息、微信支付等，方便企业进行定制化开发。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>服务号更像是各个企业在微信内部开发的一款轻量化app，用户享受企业的服务，不一定非要去下载安装app，在服务号里也能实现。从这方面来看，</span><span>服务号的定位跟后来的小程序很类似。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>3、企业号</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2014年，微信在公众号原来订阅号和服务号的基础上，新增企业号。<strong>企业号的定位是企业员工在微信端的移动办公平台。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/140969889162fb5f29375328.66179589.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了方便大家理解，以我们出去吃饭举例。如果这家餐厅开通的是订阅号，那么我们可以看到这家餐厅推送的活动资讯，比如，今天是七夕，情侣来店里吃饭可以免费领取两份冰激凌。如果这家餐厅开通的是服务号，我们除了可以看到活动资讯，还可以在服务号里完成在线排队、点餐和付款功能。如果</span><span>这家餐厅开通了</span><span>企业号，那么这家餐厅的员工可以在企业号发起冰激凌采购流程。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>总体上来说，腾讯把企业的相关诉求都集成在公众号里来实现。</span><span>公众号</span><span>似乎可以完美的连接人与企业，可是结果真的是这样的吗？</span></strong><span></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>四、2014年：钉钉——在线办公室</span></strong></span><span><span></span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2011年，微信横空出世，为腾讯在移动互联网时代抢下一个无可替代的流量入口。</span><span>为了对标微信，阿里于2013年推出了来</span><span>往</span><span>。</span><span>可是因为来往与微信在功能上过于相似，缺乏差异化，留存用户很少。</span><span>到了2014年春节，来</span><span>往</span><span>在数据上颓势尽显。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>阿里开始转变思路，既然在个人社交赛道，已经无法超越微信，那么不如换一条赛道——企业社交。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可企业社交这个命题作文到底应该怎么写呢？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当阿里团队去电商企业进行需求调研时，就像前面提到的一样，</span><span>发现企业即时通讯成了一个伪需求。</span><span>因为这些企业都是集中办公，有事吼一声就行了。</span><span>就算距离远一点，用微信也行啊。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是阿里团队还是收集到了两条很有用的建议：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1.工作和生活信息太繁杂，沟通工具总是在QQ、微信、邮件、电话和短信中切换，希望有一个统一的工作平台；</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2.阿里巴巴能不能将内部的高效企业管理系统简化后，免费提供给中国中小型企业用，提高中小型企业的管理水平。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>此刻阿里团队明白企业社交这篇文章破题点不在于即时通讯，而在于“企业协同办公”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/93085512262fb5f2989cc66.25131621.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2014年底，钉钉发布测试版，从办公场景开始切入，其产</span><span>品定位为广大中小企业提供高效协同办公工具。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在对于钉钉来说，有一个好消息和一个坏消息——</span><strong><span>好消息是钉钉的定位找的很准；坏消息是微信的企业号瞄准的也是办公协同场景。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然企业号在办公协同赛道早早卡位，但是这款产品并没有在腾讯内部受到重视。企业号只是微信公众号体系下的一个子分类，在微信中的属于配角中的配角。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2015年，钉钉开始狂奔，基本上每两周都会迭代一个版本。阿里更是投下5亿元开始铺天盖地的市场推广，马云更是亲自为钉钉站台。而企业号<span>依附于微信生态，在产品的迭代上需要遵循微信大框架。</span></span><span>针对钉钉在市场上的动作没有采取任何反制措施，放任钉钉野蛮生长。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/203098008862fb5f29d0ab29.54552201.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><em><span>钉钉把广告打到腾讯总部楼下</span></em></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2016年4月，腾讯意识到钉钉的威胁，才开始发布独立的企业微信APP，而此时钉钉已成规模。</span><strong><span>“上班用钉钉，下班用微信”，阿里在社交赛道上终于找到了跟微信的差异化定位。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>除了企业协同办公，另一个使钉钉爆发式增长的场景就是在线教育。</span></strong><span>2019年，钉钉宣称要服务1000万家组织，到了2020年8月，这个数字改成了1500万。</span><span>这个新增的500万主要来自于疫情期间，在线教育和在线办公需求暴增带来的增量。</span><span></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span>五、企业微信——私域流量运营工具</span></strong><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2</span><span>016年4月，企业微信发布后，其定位跟钉钉过于类似且功能相对简单，所以一直难以杀出重围。<span>到了2019年，经过3年的追赶，企业微信和钉钉的差距反而越来越大。截止当年底，钉钉</span><span>的入驻企业达到了1000万家，而企业微信只有250万家。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/212641709062fb5f2a232ba6.46722707.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>眼看追赶钉钉无望，企微微信3.0版本祭出一个大招——<strong>打通企业微信和个人微信，用个人微信的关系链为企业微信进行供血。</strong></span></span><span><span></span></span></p>
<section>
<p>企业微信如果定位为公司内部的一个沟通工具的话，我认为它的场景和意义会小很多，只有当它延伸到企业外部的时候它会产生更大的价值。企业微信后续新的变化是基于一个新的理念——希望让每一个企业员工都成为企业服务的窗口。人就是服务，而且是认证的服务。</p>
</section>
<section data-json="%7B%22type%22%3A%22out%22%2C%22source%22%3A%22url%22%2C%22digest%22%3A%22%3Cp%3E%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E5%A6%82%E6%9E%9C%E5%AE%9A%E4%BD%8D%E4%B8%BA%E5%85%AC%E5%8F%B8%E5%86%85%E9%83%A8%E7%9A%84%E4%B8%80%E4%B8%AA%E6%B2%9F%E9%80%9A%E5%B7%A5%E5%85%B7%E7%9A%84%E8%AF%9D%EF%BC%8C%E6%88%91%E8%AE%A4%E4%B8%BA%E5%AE%83%E7%9A%84%E5%9C%BA%E6%99%AF%E5%92%8C%E6%84%8F%E4%B9%89%E4%BC%9A%E5%B0%8F%E5%BE%88%E5%A4%9A%EF%BC%8C%E5%8F%AA%E6%9C%89%E5%BD%93%E5%AE%83%E5%BB%B6%E4%BC%B8%E5%88%B0%E4%BC%81%E4%B8%9A%E5%A4%96%E9%83%A8%E7%9A%84%E6%97%B6%E5%80%99%E5%AE%83%E4%BC%9A%E4%BA%A7%E7%94%9F%E6%9B%B4%E5%A4%A7%E7%9A%84%E4%BB%B7%E5%80%BC%E3%80%82%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E5%90%8E%E7%BB%AD%E6%96%B0%E7%9A%84%E5%8F%98%E5%8C%96%E6%98%AF%E5%9F%BA%E4%BA%8E%E4%B8%80%E4%B8%AA%E6%96%B0%E7%9A%84%E7%90%86%E5%BF%B5%E2%80%94%E2%80%94%E5%B8%8C%E6%9C%9B%E8%AE%A9%E6%AF%8F%E4%B8%80%E4%B8%AA%E4%BC%81%E4%B8%9A%E5%91%98%E5%B7%A5%E9%83%BD%E6%88%90%E4%B8%BA%E4%BC%81%E4%B8%9A%E6%9C%8D%E5%8A%A1%E7%9A%84%E7%AA%97%E5%8F%A3%E3%80%82%E4%BA%BA%E5%B0%B1%E6%98%AF%E6%9C%8D%E5%8A%A1%EF%BC%8C%E8%80%8C%E4%B8%94%E6%98%AF%E8%AE%A4%E8%AF%81%E7%9A%84%E6%9C%8D%E5%8A%A1%E3%80%82%3C%2Fp%3E%22%2C%22digestLen%22%3A119%2C%22text%22%3A%22%22%2C%22article%22%3A%7B%7D%2C%22hasReportOverSize%22%3Afalse%2C%22editorReportData%22%3A%5B%7B%22id%22%3A%22122333%22%2C%22key%22%3A%2276%22%2C%22len%22%3A1%7D%5D%2C%22from%22%3A%22%E5%BC%A0%E5%B0%8F%E9%BE%992019%E5%B9%B4%E5%BE%AE%E4%BF%A1%E5%85%AC%E5%BC%80%E8%AF%BE%E8%8A%82%E9%80%89%22%7D"><span>张小龙2019年微信公开课节选</span></section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>如果说3.0版本之前，企业微信走的路是跟钉钉一样的企业内部协同办公。那么3.0版本之后，企业微信要走的路就是帮助企业更高效的触达C端用户。</strong>用企业微信官方的话来说，就是“要成为国内最大的B2B2C数字化连接器”。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这是钉钉和其他竞品走不了的路，钉钉可以一直在协同办公领域深耕，但却只能在2B的圈子里打转。<strong>而</strong></span></span><strong><span>企业微信可以轻易的实现2C，因为她拥有一个对C端</span></strong><strong><span>用户触达最垂直、最深入的“母体应用”——微信。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>张小龙“人即服务”的产品理念是用户可以通过微信直接找到给自己提供服务或者产品的人。</span><span>可是对于企业来说，这句话也可以反过来理解——让能</span><span>提供服务或者产品的人主动去触达需要服务的对象。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这点上来说，企业微信的定位很像之前的企业QQ和微信公众号，都是企业</span><span>私域流量运营的利器。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/71171513662fb5f2a6008e0.25832156.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自此与个人微信打通后，企业微信入驻企业数连续两年实现翻倍增长。</span><strong><span>到2021年底，企业微信上的真实企业与组织数超1000万，比2019年底增加了750万家企业。</span></strong><span>这新增的750万家企业，有多少是抱着“服务客户”的心态，又有多少是抱着“收割流量”的心态，我们不得而知。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是从历史上来看，历代私域流量运营工具都不可避免与灰产和微商产生联系，企业微信能否跳脱这个历史周期律，</span><span>我们拭目以待。</span></p>
