<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>伴随着互联网的发展，各种渠道、机会增加，诈骗、色情等灰黑产也在阴影处不断滋生。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，「不实信息」就是威胁用户上网安全的一大因素。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相信不少冲浪的用户，都见识过“信息欺骗”的力量——贩卖焦虑赚钱，煽动男女对立吸流量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那些「不实信息」伴随着互联网的特点被快速扩大和加速传播，通过夸张、画质低劣的视频煽动用户情绪，甚至对用户进行洗脑，引导用户落入财务陷阱。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>那么，</span></strong></span><span><strong>这些「不实信息」究竟是如何产生的？我们又该如何抵挡「不实信息」的侵害？</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，抖音在 8 月 16 日举办“治理不实信息开放日”，相信我们能从中得到答案。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><em><strong><span>01、</span></strong></em></span><strong><span>什么是「不实信息」？</span></strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提到「不实信息」，不少业内人士认为，其威胁性不亚于网络诈骗。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“网络诈骗固然可恨，但是由于不断地宣传教育+严厉打击，用户的心智已经相对成熟，甚至会在抖音小安的提醒下主动举报；<strong>相比之下，用户难以甄别「不实信息」，更容易中招。</strong>”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在「治理不实信息开放日」的相关信息中，运营社发现，抖音明确界定了不实信息并非仅仅指谣言，而是虚假人设+谣言+辅助传播工具的结合。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这意味着「不实信息」定义范围更广，相较于谣言，「不实信息」也更加狡猾难缠，因此抖音用了一个更高更全局的视野去分析和治理这个风险。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这具体体现在界定「不实信息」形式的多样化上：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一方面，「不实信息」会以虚假的专业知识存在。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美国社会学家奥尔波特和波斯特曼曾总结过一个谣言公式: 谣言的传播能力=（问题重要性×谣言模糊性）÷公众判断能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个公式认为，<strong>公众判断力越弱，谣言的传播能力就越强。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此我们也就不难发现，由于大众科学素养缺失，在专业知识领域，尤其容易出现「不实信息」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拿健康知识举例，随着生活水平的提高，用户越来越关注家居环境，于是不少不法分子利用用户的恐惧过度渲染「甲醛危害」，甚至引诱用户缴纳各种“除甲醛智商税”……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种「不实信息」就属于在专业知识中，掺入虚假信息，形成半真半假的内容。对此，不仅用户难以辨别，就连平台想要出手打击也颇为费力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>另一方面，更多的「不实信息」还会存在于庞杂的社会类新闻中。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>受到巨大的流量引诱，只要有热点资讯，这些「不实信息」常常会出现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一类是日常主动编造类型。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如抖音前一段时间封禁的“外卖小哥，因为一份麻辣烫给我下跪”这类视频。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大量不同的账号，批量编造着同样的内容——一个外卖小哥带着两个孩子送外卖，孩子真正的父母都去世了云云，以此来博取不正当流量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于其编造内容均来自日常生活，在没有发酵前，极难被甄别处理。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二类是重大事件中蹭热度、仿冒正主类型。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如俄乌战争前后，各种蹭热度的“第一手信息”层出不穷—— 2 月 27 日，一名在乌克兰工作和生活的华人，利用海内外信息差，将俄罗斯阅兵时场景编入视频在多平台分发，谎称为俄乌战争视频。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再比如，6 月 10 日，唐山事件出来以后，为了挑拨群众，恶化矛盾，获取不益流量，不少违法账号仿冒正主出来发声，甚至虚假摆拍，称“受害女子被打死”、“被汽车碾压”等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短短三天时间，平台就主动拦截和处置了 157 个违法、违规账号，处理违规视频 4992 条。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三类是演绎摆拍被当作真实事件传播类型。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前段时间，在海南三亚，一女子在夜市内唱歌被客人泼酒的视频引发热议，随后，视频中店铺的工作人员却表示，“该视频是拍摄的段子，出镜的都是自家的员工，都是有剧本的”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这类视频的内容大多具有煽动性，让情绪上头的用户难以辨别。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此以外，还有两类由于“传递不当”产生的「不实信息」，包括用户以讹传讹，对事件进行玩梗、猜测，引发大量关注和传播后，导致的「不实信息」；还包括由于信息传递不完整，受众进行各种猜测，造成的「不实信息」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>综上，我们不难发现，</span><span><strong>和其他网络灰黑产相比，「不实信息」在前期真相不明的时候，具有极强的隐蔽性。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于用户来说，它们往往披着「科普」、「真实」的外衣，却和诈骗、网暴、色情等行为结合，将用户带入「情绪」、「认知」的陷阱中。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于平台来说，任由「不实信息」横行，不仅会降低用户信任度，长此以往，更有损平台公信力，<strong>而比点击量更珍贵的，恰恰是平台的公信力</strong>。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><em><strong><span>02、</span></strong></em></span><strong><span>打击「不实信息」，非一方之责</span></strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>了解了「不实信息」的狡猾和难缠以后，我们也就不难理解，为什么被处理的不实信息占整体比重极小，抖音仍然坚决打击处理。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>观察下来，在多轮较量、长期观察后，这场打击「不实信息」的战役似乎已经进入到相持阶段——「不实信息」不断迭代，抖音也不断更新能力，包括追溯、研判、干预……</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「虚假专业知识」类不实信息账号善于利用权威人设伪装自己，那么抖音就会对创作者和内容属性进行「分析」，还会通过各方努力对内容进行「验真」，帮助用户甄别虚假知识。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「日常主动编造类型」和「演绎摆拍被当作真实事件传播类型」不实信息往往取材于现实生活，用户难以界定是真是假，是真实发生还是恶意煽动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这类暂时无法核实又被怀疑的视频，抖音会利用「提醒」能力进行打标处理，提示用户理性看待。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「重大事件中蹭热度、仿冒正主类型」不实信息往往热较度高，且极易传播出去。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此抖音就上线了「拦截」和「打标」能力，主动拦截已经明确的不实信息，同时限制其再次传播，从流量源头掐断不实信息的滋生。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音公开数据显示，2022 年以来，在多重能力加持下，抖音共处理 319 万条不实信息， 对 90 万条存疑信息打标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样看，<strong>在处理「不实信息」上，抖音历经百战，似乎已有进展。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，在对话时，抖音相关负责人却告诉运营社，</span><span><strong>抖音在治理不实信息上依然存在困难，常年单打独斗+层出不穷的「不实信息」，对平台的精力和耐力提出了极大的考验。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对方提到，他们在处理「不实信息」过程中，面临着</span><span><strong>杜绝难、核实难、辟谣难</strong></span><span>三大问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，从源头上看，「不实信息」难以被杜绝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原因有二：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一方面，只要流量不息，不法分子就会一直制造谎言</span></strong><span>，谎言不结束，「不实信息」就会一直存在。平台也不可能为了杜绝「不实信息」，就将所有“经过艺术加工的视频”限流甚至封禁，投鼠忌器，「不实信息」也就越来越猖獗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另一方面，我国目前针对谣言的法律治理机制还不完善</span></strong><span>，不法分子仍然有漏洞可钻。</span><span></span></p>
<blockquote>
<section><span>“网络谣言监管的机关比较分散，网络谣言的监管往往通过专项行动进行，尚未健全常态化的监管机制。”——<span>中国传媒大学文化产业管理学院法律系主任郑宁</span></span></section>
</blockquote>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，<strong>从审核上看，有的不实信息往往真假掺半，内容模棱两可</strong>，在法律上也存在着不确定性和模糊性，让平台难以界定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“目前我国关于网络谣言治理的法律规定还比较笼统，对于网络谣言的概念界定、构成要件、规制程序、民行刑法律责任的衔接等方面的规定尚不明确。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这种情况下，「不实信息」产生的低成本和核实的高代价，也大大增加了平台治理的难度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，即使平台倾注全力一一核实，那些和热点资讯绑定的不实信息传播速度极快，受众极其广泛，平台不仅难以追溯源头，辟谣效果也会大大减弱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正所谓，“造谣一张嘴，辟谣跑断腿”，对平台来说，想要辟谣跑赢谣言，几乎是不可能的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更严重的是，在以上这个过程中，还存在着</span><span><strong>一个正三角和一个倒三角</strong></span><span>的矛盾。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「不实信息」在前期难以被发觉，只有其达到一定量级时，平台才能发现、判断、治理并预防再发酵。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>即使是投入巨大的抖音，其治理手段，也是在前期较为单薄，越往后期越丰富越精准。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们不难发现，<strong>依靠单个平台独立治理如雨后春笋般的「不实信息」，最终结果只能是平台“九死犹未悔”，现实却“折戟沉沙”。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不少业内专家认为，单个平台没有办法治理「不实信息」，想要根治，就必须联合解决，</span><span><strong>共建一个「不实信息」的治理平台</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，中国传媒大学文化产业管理学院法律系主任郑宁，在文章「网络谣言的协同治理」中提到，想要有效打击「不实信息」，除了完善法律治理体系、提高公民科学素养以外，还要构建多方协同的治理机制。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>郑宁老师认为，“徒法不足以自行。互联网开放、自由、复杂的特点决定了我国应该构建一个由政府、互联网平台、行业协会、媒体、专业机构、高校、社会公众等多方参与、协同治理的网络谣言治理机制”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，饱受「不实信息」威胁的抖音，率先提出为各方提供辟谣的基础产品能力——在刚结束的抖音治理不实信息开放日上，我们获悉，<strong>抖音已有预警、分析、验真、标记、辟谣、拦截、提示七大步骤系统，便于全面治理「不实信息」。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样一来，平台一方面可以在前期主动预警分析，中期核实辟谣分层惩治，后期拦截防止扩散，形成完整链条；另一方面，还可以联合媒体、专业机构和权威部门等，形成抵抗「不实信息」的生态圈层，其威力自然不言而喻。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><em><strong><span>03、</span></strong></em></span><strong><span>结语</span></strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在巨大的利益面前，「不实信息」、「网络灰黑产」等互联网蛀虫不断更迭，平台要净化网络环境，就要意味着要打一场漫长不已的战争。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有的平台很快听之任之，有的平台却始终坚持在一线，出钱出力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是显然，<strong>在开放的互联网环境中“单打独斗”，成本高，见效慢。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>抖音提出的共建辟谣平台，从生态圈层方向努力，或许为我们提供了一个新思路</span></strong></span><span>——抖音相关负责人告诉运营社，在过去 3 到 6 个月里，平台主动、快速从生态圈层 + 平台能力两个维度提升治理「不实信息」的能力，并取得了一定的效果。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022 年，抖音对不实信息的研判和处置效率，<strong>提升了 232%</strong>，<strong>上线了溯源、打标、粉丝抹除等 9 项产品功能</strong>。其中，抖音对不实信息首发账号的<strong>追查准确率接近了 80% </strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些做法，值得同行参考。</span></p>
