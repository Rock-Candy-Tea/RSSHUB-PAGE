<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>借势冰淇淋打动年轻人的心，这一招在北京行不通了？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月29日，<a href="https://www.niaogebiji.com/tag/%E8%8C%85%E5%8F%B0%E5%86%B0%E6%B7%87%E6%B7%8B" target="_blank">茅台冰淇淋</a>登陆北京。资料显示，当日与茅台冰淇淋北京旗舰店同开的还有上海、遵义、青岛、哈尔滨四个城市的茅台冰淇淋旗舰店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但与茅台冰淇淋在社交媒体上的火爆，以及此前各地茅台冰淇淋旗舰店开业时各路媒体提到的“大排长龙”不同，8月29日开业的北京茅台冰淇淋店并未有人头攒动、蜂拥而至的景象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月29日早上九点多，燃次元便到达北京茅台冰淇淋旗舰店所在的北京朝阳大悦城。该商场于早上十点开门营业，到了十点，等待扫码进入商场的人排起了长队，但并不确定是否为守候在此准备入场买茅台冰淇淋的顾客。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进入商场后，首先映入眼帘的是茅台为了开业活动而准备的发布会场地，向右步行数步才是茅台冰淇淋的售卖专柜。不过茅台冰淇淋北京旗舰店负责人孙俣向燃次元介绍，该柜台为茅台冰淇淋北京快闪店，旗舰店尚在装修之中，也在此商场内部，将于9月29日左右与消费者见面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃次元在朝阳大悦城观察了一天，从早上十点到下午十九点，前来购买的客人时而有之，但并未排起长队，最多时也仅排着八九人，排队几分钟即可买到茅台冰淇淋，且购买的消费者多以尝鲜为主，计划“发个朋友圈”，并没有复购打算。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1531049109630d73d30fda66.62578237.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/茅台冰淇淋北京店，消费者购买冰淇淋  </span><span>来源/燃次元拍摄</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如前来购买茅台冰淇淋的80后男子王轩就表示，工作单位就在朝阳大悦城附近，得知茅台冰淇淋北京店开业的消息，特意利用午休时间前来购买了一个原味茅台冰淇淋，“之前经常会在社交媒体上刷到茅台冰淇淋，出于好奇特意来打卡，回去尝试下，顺便发个朋友圈。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王轩对燃次元表示，自己基本不喝白酒，日常喝啤酒比较多，“我没喝过也没购买过茅台，如果要买的话应该也是送长辈。不过现在也算通过冰淇淋浅尝过一次茅台酒了。”对于是否会再次购买，王轩则表示可能性较小，毕竟价格还是挺贵的，尝鲜一次就可以了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新浪财经一篇报道也提到，“此次茅台冰激凌北京首店开业，在下班时间，仅需排队一人即可买到，‘排队排到商场外’的盛况并未出现。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>北京商报也报道，“北京茅台冰淇淋店开业一小时便无需排队，而在7月开业的广州茅台冰淇淋旗舰店也早已实现‘无需排队’即可购买茅台冰淇淋。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月29日晚间，燃财经在小红书平台以“茅台冰淇淋”为关键词，分别以“北京”“上海”为定位，仅搜到十余篇相关笔记。在微博平台搜索“茅台冰淇淋”，由个人于8月29日发布的微博也仅几条，茅台冰淇淋的开业引起的反响似乎不算高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年5月，茅台推出冰淇淋，首家旗舰店开在贵州省遵义市仁怀市的茅台国际大酒店。此后，茅台冰淇淋不断向外扩张，在贵阳、西安、南京、武汉、杭州、深圳、广州、长沙等地开出旗舰店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月29日开业的五家茅台冰淇淋旗舰店是茅台冰淇淋集中开业的第三批店铺，在茅台冰淇淋北京旗舰店的装修围挡上，燃次元也看到，北京是茅台冰淇淋旗舰店的“0013号”店。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>冰淇淋是茅台打开年轻人市场的一大利器</span></strong><span>。</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>食品产业分析师朱丹蓬就对燃次元表示，<strong>布局新生代、接近新生代、满足新生代已经成为不少品牌的共识，也是茅台等品牌长线战略的支撑</strong>，“奶茶、咖啡、冰淇淋等都是新生代消费频次高的品类，抓住他们才能抓住新生代背后的增量市场。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该战略也取得了一定的成果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>借势冰淇淋，茅台在年轻人活跃的各类社交平台上取得了广泛曝光</span></strong><span>。据燃次元不完全统计，截至发稿，微博上“茅台冰淇淋39元一份”话题阅读量达1.8亿；抖音平台相关话题的播放量达到1.4亿次；在小红书上有超过1万篇笔记，其中既有打卡体验的，也有测评吐槽的，还有发笔记代购的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>茅台对外披露的数据也显示，截至8月29日，茅台冰淇淋已经覆盖了全国范围内20个省区市，实现近1.4亿元销售额。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但只图新鲜、“不会复购”的茅台冰淇淋，能打动多少“不喝白酒”的年轻人的心，还是个未知数。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/1455856016142b5abe0ca06.17059714.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>谁在买茅台冰淇淋？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>茅台冰淇淋在社交平台火得一塌糊涂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“经常能在小红书刷到关于茅台冰淇淋的笔记。”一位用户向燃次元表示。燃次元也看到，早在半个月前，在小红书和大众点评等平台上，就有博主发帖预告茅台冰淇淋北京首店即将开业的消息。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>茅台冰淇淋也是“炒家”的宠儿。自茅台冰淇淋问世以来，由于仅在部分城市售卖，滋生了部分炒作行为，今年7月就有报道显示，官方售价50-60多元/杯的茅台冰淇淋，在一些电商平台已经炒到120-250多元/杯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但出乎意料的是，8月29日开业的茅台冰淇淋北京店并未在现场掀起火爆场面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据茅台冰淇淋北京店店员介绍，由于是快闪店，所以目前店内仅售卖三种口味的预制冰淇淋，包括青梅煮酒（</span><span>78g</span><span>）、经典原味（</span><span>75g</span><span>）和香草口味（</span><span>75g</span><span>），对应贵州茅台酒含量分别为1.6%、2%和2%，对应售价分别为59元、66元和66元，旗舰店开业后将在店内增加现制冰淇淋品类。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，北京店的茅台冰淇淋不限购。与茅台冰淇淋旗舰店在其他城市“大排长龙”“人头攒动”的景象不同，北京茅台冰淇淋首店开业并没有出现消费者排长队抢购的现象。燃次元在现场看到，门店消费者以年轻人为主，且大多数消费者仅购买1-2个。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前来打卡的00后大学女生妍妍就表示，今天是陪同老人到大悦城逛街，看到茅台冰淇淋开售，于是买了一个。拿到冰淇淋的一瞬间，妍妍就开始拍照，准备回去后发到自己的社交平台上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>妍妍告诉燃次元，自己平时很少吃冰淇淋，也不喝酒，但茅台冰淇淋在年轻人中还挺出圈的，自己也想跟风一下，“60元一个茅台冰淇淋是不便宜，但在我的接受范围之内，而且我觉得味道还不错，未来应该还会继续购买。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18995271630d73d3a8caf6.49071213.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/茅台冰淇淋北京店内多处打卡设置  </span><span>来源/燃次元拍摄</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与“日常喝啤酒”的王轩以及“不喝酒”的妍妍不同，特意来打卡的90女生大楠是“高度酒爱好者”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大楠对燃次元表示，自己平时就跟父辈一起喝茅台，在抖音上刷到茅台冰淇淋后，就特别想尝下味道，“我还@了我的姐妹，想约她一起去贵州，她说买机票的钱都够我吃十几个了，后来也就作罢了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>得知北京开店，大楠特意过来打卡，“闻着是有酒的清香，但吃的时候感觉没什么酒的味道。”对于定价，大楠则表示分消费群体，如果是喜欢喝酒或者说是喜欢喝茅台酒的人来说，还是挺值的，毕竟一瓶茅台酒上千元，冰淇淋也是有茅台酒的含量在里面的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但“打卡”之后，茅台冰淇淋能留得住年轻人吗？这些年轻人又能成为茅台主营产品——白酒的消费者吗</span></strong><span>？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“之前就在大众点评上看到一些网红餐饮店会用茅台冰淇淋引流。”90后女生萌萌告诉燃次元，“在茅台冰淇淋销售还没有覆盖到北京的时候，这些店就推出包含茅台冰淇淋的套餐，或以消费到一定额度可以购买茅台冰淇淋的方式增加噱头。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但这并未打动萌萌，“我对酒类冰淇淋比较感兴趣，通常到冰淇淋店也会点朗姆酒冰淇淋等，所以当得知茅台推出冰淇淋的时候，就一直想打卡。”萌萌对燃次元表示，自己接受的冰淇淋价格区间在30-40元之间，相比之下，茅台冰淇淋的价格偏高，“但可能跟茅台沾边的都贵吧，不过出于尝鲜的目的，还是愿意试一下的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是茅台冰淇淋在北京开店，萌萌特意前来一尝，但也仅此而已，可能是宣传热度高导致期待值偏高的原因，实际吃后的体验觉得一般，“冰淇淋本身的品质我个人觉得不如哈根达斯的细腻，茅台酒味道偏浅，让人没有复购的欲望。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/9082723746142b5ac73c549.43567204.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>茅台瞄准年轻人</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为A股的股王，茅台的一举一动都牵扯着大众关注的目光。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5月19日，茅台集团宣布与蒙牛集团联合跨界做冰淇淋。消息一出，“茅台冰淇淋”立刻冲上了热搜，用户在评论区写道，“吃完能开车吗？”“吃完会不会醉？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有消费者表示，“喝不起茅台，还买不起茅台冰淇淋吗？”表示想要尝鲜打卡。也有消费者揶揄道，“这冰淇淋收藏十年能涨价吗？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在凑热闹与质疑声中，茅台冰淇淋取得了良好的市场反馈。根据茅台官方披露数据，贵阳茅台冰淇淋旗舰店营业开售7小时，销售额破20万元，销售单数破900单，预制茅台冰淇淋销售4500余个，现制茅台冰淇淋销售800余个，共计销售茅台冰淇淋5000余个，平均每分钟接待2人次，几乎达到满负荷状态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5月29日下午5时，茅台冰淇淋在i茅台APP上线销售，上线51分钟售罄，销售数量逾4万个，销售金额逾250万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在贵州茅台8月2日发布的2022年半年度报告中，茅台也提到，“成功开发茅台冰淇淋，市场反响热烈。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，多种言论及市场分析认为，<strong>对于茅台来说，茅台冰淇淋的销售额是次要的，更重要的是借着冰淇淋占领年轻人的心</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“茅台推出冰淇淋一方面可能是出于拓展产品线，积累新品类布局经验的考量，另一方面则是吸引年轻用户。我认为尝试意义大于经营意义，毕竟伊利冰淇淋业务那么大，一年也就几十亿营收，跟茅台主业营收比起来基本可以忽略不计。”亿欧商业分析师杨良对燃次元表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/314764577630d73d445f291.36226451.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/茅台冰淇淋北京店柜台  </span><span>来源/燃次元拍摄</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>冰淇淋是打开年轻人市场一个很好的载体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>茅台自带的流量密码，叠加冰淇淋的社交属性，形成了“1＋1＞2”的品牌效应</span></strong><span>。最新发布的《中国冰淇淋/雪糕行业趋势报告》提出，人们对于冰淇淋的消费观念已升级转变，冰淇淋不再是单纯的消暑解渴冷饮，而是成了社交分享、慰藉心灵和情绪的载体，产品也趋向健康化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，国内冰淇淋市场仍具有较大的上涨空间。据GlobalData数据，2021年，中国人均冰淇淋消费量预计达到2.9kg，与十年前的2011年相比，约增长了1倍。2.9kg的人均冰淇淋消费量，虽高过亚太平均水平（</span><span>2.1kg</span><span>），但与全球4.5kg的人均冰淇淋消费量相比，仍有不少差距。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>朱丹蓬对燃次元表示，一方面，冰淇淋市场广阔且研发成本低，无论是企业自行研制还是与相关企业合作，研发和制作生产成本都不高。另一方面，茅台和蒙牛合作，无论从渠道到知名度，还是从品牌效应到规模效应都有相当的优势，风险较低。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“更为重要的是，冰淇淋深受年轻人喜爱，每一支雪糕对于企业来说就像是一块移动的广告牌。”朱丹蓬表示，很多品牌跨界雪糕，并不是真的要踏入雪糕行业，而是通过低成本的跨界方式取得消费者对于其品牌的关注度，尤其是热衷于猎奇消费的年轻群体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了茅台，近日白酒品牌洋河也发布消息称，将开售盲盒雪糕。在茅台之前，泸州老窖就曾与钟薛高联手推出含52度白酒的“断片”雪糕，“黄酒第一股”古越龙山也与钟薛高推出联名款黄酒青梅雪糕……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“茅台推出冰淇淋一方面是培养年轻消费者对茅台品牌的认可度，年轻人现在喝不喝茅台可能对茅台集团影响不大，但当他们成为中年人时，还是否会钟情于茅台是充满变数的；另一方面，茅台冰淇淋本身也将成为茅台集团的一个盈利产品，茅台冰淇淋推出的品鉴卡是可以作为礼品卡来运营的，送长辈茅台酒，送年轻人冰淇淋卡。”艾媒咨询CEO、首席分析师张毅对燃次元表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“所以<strong>对茅台而言，短期内赚冰淇淋购物卡的钱，长期来看是培育年轻用户茅台高端定位的品牌认知</strong>。不管未来茅台是推冰淇淋还是包包、鞋子，甚至汽车等，<strong>本质上，茅台的定位都是希望形成奢侈品消费的品牌烙印</strong>。”张毅指出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/4756644006142b5ad08cc02.64114870.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>茅台为何需要年轻人？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>茅台向来居于“神坛”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据贵州茅台发布的半年报，2022年上半年贵州茅台实现营收576.17亿元，同比增长17.38%；实现归母净利润297.94亿元，同比增长20.85%。尤为亮眼的是贵州茅台的利率表现，根据半年数据，2022年上半年贵州茅台毛利率92.11%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但如今，茅台似乎正在谋划走下“神坛”，尤其是走进更多年轻人的心里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在推出受年轻人喜爱的冰淇淋之外，茅台今年还上线了“i茅台”APP，丰富直销渠道之余，也向年轻人更靠近了一步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>资料显示，3月31日，茅台开发出来的线上销售平台“i茅台”APP正式上线试运行。有数据显示，上线以来，“i茅台”APP连续三天位列App Store免费榜榜首，每天有43万人下载。上线三个月，截至2022年6月30日，在“i茅台”平台上实现了不含税收入44.16亿元，每日收入约4800万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>茅台和年轻人，曾经似乎“互相不待见”，毕竟年轻人“不爱喝白酒”，而茅台也似乎“不在乎年轻人</span></strong><span>”。贵州茅台前董事长季克良曾表示过，年轻人不喝茅台酒，是还没到时候，20多岁还在玩，小孩子不懂事，不晓得需要好酒喝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但这几年，茅台却不断出现在年轻人的视野中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1674268925630d73d53b5633.54622695.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/“茅台冰淇淋”在小红书上的搜索结果  </span><span>来源/燃次元截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在洞察到年轻人对低度酒的喜爱后，2019年，贵州茅台推出Umeet蓝莓精酿，进军低度酒赛道，并在当时热播的影视剧《欢乐颂》中进行品牌植入；2020年，贵州茅台与大话西游达成国风品牌战略合作伙伴协议，尝试以游戏为载体进行品牌宣传；2021年，茅台在微信公众号中发布了一首“上头神曲”——《Oh It's Moutai》，以说唱的形式表现茅台在国际化、年轻化方面的探索，受到大众的关注。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年，贵州茅台推出100ml的“迷你版”飞天，并通过茅台冰淇淋、“i茅台”频频靠近年轻人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>实现品牌年轻化、消费人群年轻化，已经是当前白酒企业迫切需要解决的问题</span></strong><span>。“茅台冰淇淋是茅台企业实现品牌年轻化的重要尝试，从这一点不难看出，茅台的前瞻性还是非常强的。”朱丹蓬表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>杨良也指出，<strong>年轻化不只是茅台，而是整个白酒行业都需要面对的问题</strong>，原因就是白酒行业在走下坡路了，这一点从白酒产量递减就可以体现，所以基本所有白酒企业都在想怎么吸引年轻人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如杨良所说，国家统计局数据显示，白酒产量峰值在2016年，彼时年产量为1358万千升，此后产量一路下降，到2021年仅为715.63万千升。2021年的白酒产量较2016年的峰值足足下降了47.3%，几乎腰斩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>白酒产量递减的原因或许是多元的，但其中一个重要因素是，中国年轻人似乎不爱喝白酒了。根据第一财经发布的《2020年轻人群酒水消费洞察报告》，在线上酒水消费中，“90后”和“95后”在国产白酒和黄酒/米酒中占比落后于整体，在葡萄酒、洋酒、啤酒、果酒中均高于整体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，年轻人并非对茅台不关注，只是他们关注的点更在于茅台的理财属性。在燃财经的文章《人人都在抢茅台》中提到，因为买到就是赚到，无数人参与到平价茅台的抢购中，之后转手卖给中间商，至少赚个几百块。因此，在零售市场上，一瓶飞天茅台可以被炒到3000元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费领域投资经理王鑫对燃次元表示，<strong>茅台的用户其实是青黄不接的，从长远来看，要实现利润最大化，最终还是得让主力产品走入千家万户，成为快消品，而不是收藏品和理财产品</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>张毅则表示，很多年轻人不喝茅台确实是没到时候，但到时候他们还喝不喝茅台，甚至还知不知道有茅台，这中间是充满变数的，“一代人终将老去，年轻人终究会成长为社会主力，现在不下手培养年轻人的品牌认知，真的等他们变老了，也不会自动爱上喝茅台的。”</span></p>
