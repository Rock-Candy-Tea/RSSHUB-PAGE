<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span>不确定环境，如何确定性增长</span></strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前来看，<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E8%90%A5%E9%94%80" target="_blank">品牌营销</a>越发充满变化和挑战，有几个现象越发明显。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，以往依赖线下零售渠道的传统品牌，纷纷转型发力线上数字化经营以此自救，线上流量竞争空前加大，获客成本逐年上升；其次，用户会在短视频、直播、社区等不同定位的内容平台产生“种草”行为，但最终因此会在何处消费，不可捉摸，品牌触点多元但愈发割裂，媒介投放分散、营销和交易链路的割裂等问题，也让营销活动越来越难出圈；此外，消费市场的增速放缓，让品牌更迫切地希望提升自己的认知度和竞争力，以使其成为某种“心智”或“品类”的代名词，转型升级迫在眉睫。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>说到本质上，难的其实就是：确定性增长。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>行业、环境、政策、消费者等随着时代不停改变，品牌如何能一直保持长周期确定性增长，永远跑在前列？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这样的背景下，<strong><a href="https://www.niaogebiji.com/tag/%E9%98%BF%E9%87%8C%E5%A6%88%E5%A6%88" target="_blank">阿里妈妈</a>给出了经营趋势和方向：“以经营科学，蓄力经营CP资产，开启10亿深消费时代”。提出“消费者（Consumer）资产的蓄力与再应用”和“全周期货品力（Product）资产积累”两大核心概念，为经营寻求可持续、确定性增长提供了重要路径。 </strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1353615957620dca3c010032.47285949.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>阿里妈妈经营科学，应运而生</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经营观念决定了品牌成长天花板的高度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于经营者的诉求，今年阿里妈妈m峰会围绕“蓄力经营，全域增长”主题，<strong>正式推出“阿里妈妈经营科学”——以科学的产品技术、科学的方法论、科学的行业经营解决方案和科学的平蓄促收体系，帮助商家收获不确定环境下的确定性。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，作为经营科学的底层驱动因子，阿里妈妈此次<strong>首次亮相了三大核心技术</strong>——阿里妈妈消费路径全域全旅程归因技术MTA、阿里妈妈新品冷启动解决方案NPA和阿里妈妈直播间智能技术套装ACE。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此番创新，亦是阿里妈妈成熟技术能力的系统性总结，旨在以提高商家经营效能为目标，从渠道价值策略、新品打爆专属通道、直播间加速转化等层面，为品牌提供技术助力，全方位将数字化融入经营链路，确保经营动作的科学性。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/399842604620dca426bc971.87344362.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span>经营科学，重塑各行各业</span></strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于技术和方法论的升级迭代，阿里妈妈首次提出先进、全面的从人群到货品到投放路径的行业经营解决方案，它正式推动品牌经营迈入科学经营的时代。<strong>在全新的经营方法指导下，商家和品牌又该如何科学地解决经营问题？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于行业差异性，阿里妈妈通过<strong>「生意关键驱动因子」与「科学的经营方法」联动</strong>，首次系统性输出垂直行业经营新策——针对快消、服饰时尚、食品和消费电子四大行业，分别推出了可行动、可追踪、可运营的科学解决方案，助力全行业商家收获确定性增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/825897804632a12b234bc41.40186158.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>（1）快消行业：消费运营驱动，提高全域种草以及流转效率。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快消在传统营销领域都是重要的一个行业，覆盖全年龄用户，营销效率天然最高。只要触达消费者的频次和渠道越多，成交规模就会越大，<strong>属于典型的消费者运营驱动型的行业。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那阿里妈妈是怎么做到“科学经营”的呢？在阿里妈妈联动OLAY美白水乳套装的案例中，我们看到了答案，通过“一筛；二追；三深化”策略，OLAY成功实现高频触达消费者，提高用户转换率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第一步，广泛触达，重点筛选。</strong>首先，OLAY在全域做大量种草，找到美白水乳套装的目标TA人群。然后，再对人群做细分，比如已经对品牌有认知的用户，有美白需求的用户，对竞品感兴趣的用户等等。找到人群之后，用Unidesk、引力魔方等，匹配站内外的媒介渠道，做人群覆盖，并且快速积累数据，再接着用MTA的归因技术来指导全域投放的策略。今年的4月至6月，OLAY品牌的人群资产增长超过了2倍多；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第二步，站内追投，全程沟通。</strong>先全域种草，然后进行站内人群的追投，通过直通车搜索、万相台拉新快等产品， 渠道组合叠加，序列化追投，让转化效率大大提升，资产流转率提效加倍；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第三步，深化私域，持续购买。</strong>直播是转化会员最高效的地方。品牌直播间着重加强引导加会员，同时，再加上新老会员权益，不断加深品牌和用户之间的关系，最终新增的会员超过了数十万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2076542007632a12b3054685.60119117.png" referrerpolicy="no-referrer"></p>
<section><span><strong>（2）服饰行业： 上新驱动，新品快速成长及售罄。</strong></span></section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>库存，是服饰行业是最大的痛点。众所周知，我们的衣柜里永远少一件衣服。想要抓住消费者的心，服装企业必须不停上新，<strong>是典型的新品驱动型行业。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，一个款式至少要备几万件货，每个款式却并不一定都好卖，一旦过气就会迅速贬值、滞销，企业就需要承担巨额库存费用，资金无法回流，周转困难。所以，提高新品转化为爆品的命中率，是服饰行业的必修课。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>阿里妈妈通过联动男装时尚品牌GXG，用数据证明，服装企业可以凭借新技术的力量，前置化产品的爆款潜质，减少品牌内耗，缓解生产和库存的压力。</strong>通过目标清晰的“准-狠-稳三步走”策略，成功实现“新品即爆品”的传播结果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第一步：新品3天速测冷启，精准锁定潜力爆品。</strong>GXG先是精选10款新发「重磅系列」新品，然后借助阿里妈妈万相台-测款快，叠加直通车-均匀测款的能力，通过3天的实测，发现其中一款商品点击率优于同周期新品的1倍+，根据测款结果，快速锁定了排序TOP1的潜力新品，并匹配出最优的主图素材；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第二步：新品新客加速扩容，精准锁定潜力爆品。</strong>以测款TOP1潜力新品为重点打造对象，通过万相台-上新快持续推广叠加直通车&引力魔方进行多产品矩阵同时投放，实现多渠道拓展人群，场景化关联拉新。该潜力新品点击率高于同周期新品3倍+，新客占比高达90%+；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第三步：新品引流全店成交，稳住全面增长基本盘。</strong>以上述测款TOP1潜爆新品为例，持续增加预算追投，创意上增加同系列商品搭配关联销售，通过万相台-拉新快&货品加速叠加超级短视频&超级直播联动。最终将该款裤子成功打造为夏季爆品，同比去年爆品提升50%以上；并且还顺势带动「重磅系列」新品的整体成交，尤其小A T恤成交额达到破百万，该系列新品关联销售带动店铺整体客单同比提升12%以上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/340686143632a12b3b01988.56215592.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>（3）食品行业： 多爆复购驱动，以爆带新，高频复购</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>食品市场的同质化竞争相当严重，消费者对食品品牌的遗忘周期变得越来越短。<strong>如何快速起量、打爆新品，并且实现高频的即时转化、留存用户、扩宽新渠道，成为品牌当下直面的挑战。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>针对这样的消费现象，阿里妈妈与隅田川通过1+N以爆带新打造货品矩阵、爆品拉新高效转化快速扩容、高频用户养成沉淀高价值人群资产的“磁吸式营销新策”为策略，用爆品快速拉新扩容，让用户成为忠实客户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一步：</span><span>首先搭建了</span><strong><span>一个核心爆款产品</span><span>+</span><span>多个二梯队爆款</span></strong><span>的矩阵，用核心爆款助攻二梯队爆款的加速成长。在品牌日及行业营销节点，隅田川上架潜力爆款，通过阿里妈妈直通车、万相台货品加速、引力魔方定投爆款沉淀人群的推广加速潜力爆款的成长，大促节点更好的助力爆款提升类目排名，隅田川通过以上方式不断拓展叶子类目的爆款数量，打造店铺的</span><span>1+N</span><span>爆品矩阵，目前品牌</span><span>TOP10</span><span>爆品对于全店成交贡献占比高达</span><span>75%</span><span>以上</span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第二步：利用多爆款+新客权益扩展店铺新客。</strong>一般常用的产品组合包含：引力魔方定向跨类目高潜新、类目、品牌、商品兴趣人群，万相台拉新快智能投放更好的去触达品牌高潜新，再叠加一部分的新享新客礼金更快的促进新客转化。通过日销加大推广8月通过营销推广带来的首购新人群环比提升18%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第三步：通过爆款不同组合，加速高频用户转化扩容。</strong>在首购转化后品牌就开始通过引力魔方定向触达周期内已购用户促进已购用户购买3单成为品牌的高频用户。品牌全年复购频次同比提升7%，年老客GMV同比提升接近2倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1895976861632a12b48d2585.61702570.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>（4）消电行业：爆品拉新购买驱动，精准拉新，高效首购。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费电子行业产品的科技含量高，客单价也相对较高，用户的购买决策周期和购后换新周期都会更长，<strong>属于爆品拉新购买驱动型</strong>，所以品牌需要更精准的识别并捕获高潜力人群，提前做好布局，抢占决策先机。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>为全面助力消电商家，阿里妈妈推出了万相台「百万新客计划」，今年与三星的合作就有效实现了对用户的更精准触达。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了筹备今年618，三星于5月份就开始前站蓄水。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>站外</strong>，品牌通过一二线城市媒体高频露出，触达品质中产、都市白领和城乡精致消费人群，提升品牌心智，并沉淀为品牌数字化资产。</span></p>
