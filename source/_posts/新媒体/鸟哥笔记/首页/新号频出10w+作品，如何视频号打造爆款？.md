<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2020年开启内测的<a href="https://www.niaogebiji.com/tag/%E8%A7%86%E9%A2%91%E5%8F%B7" target="_blank">视频号</a>至今已有2年多的发展，据腾讯官方5月公布的Q1财报显示，视频号正在持续获得用户欢迎，今年一季度的视频播放量及使用时长同比大幅增长。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">依托平台的扶持，视频号正处于流量红利期；又因为去中心化的推荐机制，普通人也有机会做出好数据，成为百万粉大V。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于短视频小白或素人创作者来说，想要做视频号，有哪些可供参考的内容方向及选题类型？通过几个新晋黑马账号，来看看他们的思考。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">01、剧情类：讲述人生道理，4个月出5条爆款</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">剧情类内容在视频号的热度持续不减，其特点是故事性强，能够传递丰富的情感及社会价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">博主@王梦316对自己的定位就是剧情博主，账号从3月运营至今，虽然只发布了51个作品，但10w+推荐数的视频已经有5条。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/9c0309e638a30971dfa0572062088e7c.png" alt width="1000" height="396" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲ 新视-账号详情页</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">观察发现，博主的视频时长基本在2-4分钟，每个视频都会聚焦于阐释一个人生道理，涉及到职场成长、家庭和谐、个人道德等多个方面。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">账号最近的一条爆款作品《因为爱这个家，所以不去计较太多 家和才能万事顺》发布于7月5日，推荐数和转发数均突破10w，评论也有4.26w。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">视频中两个儿媳对待婆婆有着截然不同的态度。小儿媳对拉扯孙子到高中的婆婆不仅不感恩，还有意推卸赡养义务将老人赶出家门；大儿媳对待不帮自己带孩子的婆婆却宽容大度，将其接回家中。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">视频通过婆媳间的冲突来展现出家庭和睦的重要因素是少计较得失，过去已成定局，注重当下的友爱相处才是更要紧的。从标题和封面也能清楚感受到“家和万事兴”的主旨，达人希望以此来引起用户对待家庭的思考。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">婆媳关系一直是平台热议的话题，视频能火也就能够预料。查看达人的其他作品，发现选取方向多是普通用户较多关注的内容，如拒绝家暴、尊重他人等，涉及此类热议话题的视频互动数据都不算低。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">接地气的选题让账号每月都能产出爆款，多数视频的推荐数也集中在1-5w，账号前期运营算是非常顺畅，给剧情类起号的创作者提供了较大的参考价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/db570f6a73a4d7e10ede827746ae992c.png" alt width="759" height="303" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲ 新视-账号详情页</h4>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">02、知识类：冷门科普，首条视频获推荐1.2w</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">视频号平台中，泛知识内容也在日益丰富，除了正经的文化、科技等专业性的内容输出外，冷知识科普，可能是普通创作者更容易入局的方向。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们观察到博主@离pu的知识从5月开始运营账号，分享一些大家平常不会注意到的冷门知识，首个视频就获得1.2w的推荐数。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">视频分享抓娃娃的小窍门，娃娃机的抓取力度分为1级-5级，5级的力度最紧，操作摇杆及按键将机器爪复位至5级即可提升抓取成功的概率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">玩过娃娃机的人不少，但不是所有人都知道抓取的技巧，博主就是看到这点后制作视频并分享，在账号冷启动之初就获得不错的推荐数据，该视频转发也达到1.1万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">查看账号页面，可以发现博主运营账号至今2个月的时间里，共发布作品38个，其中互动数据表现较好的作品通常具有用户熟悉又陌生的特点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如挑选洗衣液时可查看执行标准，选择轻工业部推荐的标准购买；驾照被违规扣分，可在交管app上答题补分回来等内容。熟悉的是视频选题是用户生活所常见的话题，陌生的则是博主科普的知识通常是偏小众，普通人知之甚少的，能够引起用户兴趣从而进入视频查看。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/e4e13c3ff0c5a77ff11d6cb979ba7187.png" alt referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲ 图源视频号</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7月7日的视频《大一开学，不要再不会自我介绍了，这样介绍下自己试试》也是如此，其中提到自我介绍时若没有可说的成就，可提出一些大众普适的荣誉，如授予全体国人的“2022年国际奥林匹克奖杯获得者”，来化解无话可说的尴尬。视频发布4天后，推荐数成功突破10w，成为账号第一个爆款作品。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/f11f64946b8f5b6d8d3b28cc3cb91ae5.png" alt width="773" height="565" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲ 新视-视频详情页</h4>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">03、母婴类：家有儿女，搞笑日常引围观</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">视频号的用户不止有年轻人，中老年人群体的占比也不小，合家欢的内容是这部分人群喜闻乐见的，因此在平台上，母婴类的萌娃账号具有一定的优势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从去年11月开始运营的账号@橙果姐弟，近半年的时间已经发布64个作品，集均推荐数2.1w，10w推荐数作品有7个。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">发布的视频通常在1分钟以内，展现古灵精怪的姐姐、爱哭但姐控的弟弟两人的日常生活，小朋友天马行空的举动常让父母及视频前的观众忍俊不禁。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/dcae12740b8e5aada9483b428027d6e4.png" alt width="1036" height="385" referrerpolicy="no-referrer"></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">▲ 新视-账号详情页</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7月1日的视频《啥也别说了，准备开揍吧….》中，姐姐对着弟弟的衣服进行了一番艺术创作，等到被发现时，姐姐低头自知有错以及弟弟一脸无所谓的样子，让爸爸生气又无可奈何。视频发布两天获推荐数超过10w，转发数也达到7.3w。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">家有儿女的搞笑日常不仅能让观众会心一笑，也能让用户感受到家庭温暖有爱的氛围，勾起自己的家庭记忆，以情感形成博主与粉丝间的连接。此类视频内容无需专业设备、精致文案，只需要展示出萌娃真实活泼的样子即可，对创作者来说创作难度不高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了剧情类、知识类、母婴类的账号外，我们还能看到：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">▶ 生活类的@合屏哥房车游记记录与81岁母亲的房车旅行获得推荐10w+；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">▶ 搞笑类的@懒王星球分享微信上与朋友的有趣对话，1个月时间收获首个爆款；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">▶ 才艺类的@创意工坊制作每天分享创意手工，7天产出首个10w+推荐视频……</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">视频号的内容虽日趋丰富，但创作者仍有机会施展拳脚，抓住机遇实现流量变现。</p>
