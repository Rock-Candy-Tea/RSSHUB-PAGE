<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>应该谢谢俞老师在过去20年 </span><span>给中国学子带来的启蒙</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>悲情许久的<a href="https://www.niaogebiji.com/tag/%E6%96%B0%E4%B8%9C%E6%96%B9" target="_blank">新东方</a>，</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>迎来了自己的转折时刻。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>旗下电商品牌东方甄选<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%9B%B4%E6%92%AD" target="_blank">抖音直播</a>间，走出了带有新东方ICON的高知、高敏人气主播——董宇辉。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了英文流利和知识面渊博，他更是一个高共情能力的高敏人群，能感知到非常细腻的群体情感，并在直播间带货的间隙，通过非常幽默风趣的方式表达出来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>共情力和平台加持让他一周内粉丝涨了千万</span></strong><span>，GMV过亿，成为抖音<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>的顶流。与此同时，东方甄选控股股东——港股新东方在线，盘中一度涨超100%，刷新上市以来最高涨幅纪录。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在这场新东方转型突围的坚韧故事背后，也是字节系的野心和实力展示。</span></strong><span>这标志着抖音在国内的扩张抵达天花板之后，开始了更精准的既要泛化、又要渗透率的精细化运营。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户光去抖音花时间已经不够了，他们想让你去花钱。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和互联网1.0时代老牌国民应用时期的商业化姿态不同，抖音是个既要，又要，还要的强势规则制定者。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在字节跳动的体制里，这家公司从10年前的第一款产品今日头条开始，在产品基数非常小的创业早期，他们的商业化ROI就是赚钱的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>今天的抖音达到了史上最强状态</span></strong><span>：他们的DAU</span><span>（每日活跃用户数）</span><span>接近7个亿。产品的设计者和运营者在制定规则的时候，比从前更有话语权。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>超级印钞机</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一个短视频应用能做成这样意味着势不可挡：</span></strong><span>14亿中国人，去掉4亿没有手机的用户，在10亿是有条件上网的网民里，再根据最新人口普查数据，减去2.5亿左右14岁以下的、家长不给玩手机的未成年人群。你会发现，剩下能自由使用手机的7.5亿网民，几乎都是抖音的用户了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>新东方和抖音的相遇就并非偶然。</span></strong><span>不止是新东方，只要是之前做线下生意，现在转型想做线上业务的团队，都会遇到抖音。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>抖音现在是不折不扣的印钞机：</span></strong><span>每天有7亿人在抖音刷视频，他们中每个人为抖音贡献的平均时长是40分钟左右。这庞大的人群基数汇聚而成的时间总和及背后的隐藏价值，创造了抖音今天的商业神话。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>维持这个庞大系统的健康运营，需要有基数庞大的人群持续往里填充时间，充值金钱和生产好内容。三者是互相影响且能闭环的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>要创造价值</span></strong><span></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音的商业模型里，当一个像新东方这样的大品牌去到抖音，抖音是需要新东方是既贡献好内容，又要在抖音做爆款出圈双赢，还要为抖音流量付费的。只有这样，作为平台，抖音流量的ROI才是划算的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>对于新东方来说，商业上去抖音付费买流量容易。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>但从零到一阶段，想通过爆单收回成本又很难。</span><span>此前</span><span>他们磨合了半年，一直没找到重重抖音算法+重重人工审核喜欢的内容，直到董宇辉的出现。他用中英双语+发散的知识讲解，在直播间展示了一位前职业教师的学富五车。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>而这“知识”二字，刚好扣上了抖音的平台需求。</span></strong></span><span>于是，东方甄选出圈了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>Timing很重要</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>今天的抖音比以往更需要有“知识”。</span></strong><span>因为他们太大了。收集了半个中国的用户时间，除了为自己赚钱外，他们到了必须考虑社会价值的时刻。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从基因上讲，为了降低短视频生产门槛，而在运营体制上鼓励用户拍摄重复模仿视频的抖音，是有噪音嫌疑的。</span><span>为了从供给侧增加知识的含量，这两年抖音的运营团队一直在挖知识领域的KOL入驻。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>新东方就这样击中了抖音的心事。</span></strong><span><span>timing很重要，新东方恰逢其时。</span>俞敏洪和他的东方甄选的直播间，既传播了知识，又创造了商业GMV，俞敏洪本人也顺势而为，在抖音开设了知识类访谈专栏。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而新东方的人气主播董老师，因为网友追播和平台需要，也是一鼓作气，天天上播，连续一周没休息了。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当一个行业增量没有了，存量竞争就卷成了零和游戏。<span>在中国早已饱和的电商版图上撕开一个口子，对字节和抖音来说，并不容易。</span></span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>抖音电商有硬伤。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有流量的产品没用户心智</span><span>(抖音主客户端)</span><span>,定义了用户心智的产品</span><span>(抖音盒子、Marvel等)</span><span>没流量。抖音电商作为一个从一个短视频产品里长出来的电商，本来是天生的四不像。用户心智一开始就是奔着短视频去的，和电商场景匹配不上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音在视频流里做电商，需要背后的ROI比效果广告更赚钱，否则即便是流量大户，把流量给到电商业务也是不划算的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>增长的秘密</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在抖音主客户端，电商增长很依赖直播带货。</strong>“直播带货”也是当下抖音电商需要主打的用户心智。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，他们需要更多B端的企业加入抖音电商，丰富平台生态的同时，为平台的流量付费，另一方面，需要更多的C端用户知道，抖音是可以购物的，从而让那些短视频用户，真正的从在抖音花时间，转向在抖音花钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>快手有辛巴等东北老铁，淘宝有过薇娅、李佳琦</span></strong><span>。产品上一直去中心化的抖音，或许在基因上没有那么喜欢中心化的KOL。他们对淘宝和快手的烦恼一清二楚。但用户喜欢主播，用户群体心智也会倒逼平台选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是抖音电商就先有了罗永浩，接着迎来了新东方和东方甄选。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>争夺用户心智</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>电商的逻辑是只要你在一个地方溜达的时间够长，就一定会为之付费。直播电商之所以这么火，是因为它消耗起时间来，比短视频还猛。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>只有话题和注意力，能让用户流动起来</span>。</strong>通过一次一次的爆款运营来强化用户心智，是当下抖音的核心运营策略之一。在业务扩张和泛化层面，也在用这个打法：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>每隔一段时间，抖音都会走出一个红人。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音做全覆盖、做下沉用户时期，跑出了东北农村的张先生；快手用户忽然发现，原来抖音也有老铁。疫情期间，用户居家办公，刘耕宏毽子操又火了，带7000万抖音粉丝一起做运动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>健身APP Keep积累了7年，才做到400-500万DAU。而碾压这些，抖音只通过一场直播就做到了。一位前字节跳动策略产品经理说，一个刘耕宏等于20个Keep。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>所有爆红逻辑的背后，折射的都是平台的需求。</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>东方甄选热度正浓。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这两天一打开抖音，第一条就刷到新东方的直播间，平台已经在玩命推流量了。流量加持之下，后面很快还会有新东方王老师，新东方李老师涌现出来，新东方最不缺的就是很注重用户体验的老师了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>网友像追剧一样追新东方董老师直播间</span></strong><span>：他从中文讲到英语，从仲夏洋洋洒洒的阳光讲到莎士比亚，从母亲的谆谆教诲讲到儿时的梦，从历史讲到哲学就为了卖我一根玉米，有什么理由不买呢？</span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一个心浮气躁的，注意涣散的，即刻满足的、实用主义的背景下，简直是太击中人心、对症下药，让人心甘情愿为知识付费了。</span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>新东方本身的故事也动人。</span></strong><span>曾经经验丰富的老师，但步入直播行业之后，又重新变回小学生。就像是看电影一样，在他们的奋斗故事里，观众是能看到人物的成长线的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们把教室里的一对多沟通方式搬到了直播间：行动上的身体力行，情感上的各种共鸣，说的每句话都落入心坎。网友说，谁还在乎你卖的是玉米还是大米？我买的都是我童年夏日的蝉鸣。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>最大赢家</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着爆款出圈的背后，抖音电商的来钱之道就很清晰了：通过对爆款IP的运营，实现了电商流量的获取</span><span>（打开了抖音直播）</span><span>、留存</span><span>（花时间直播）</span><span>和转化</span><span>（花钱买东西）</span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，那些在新东方董老师直播间买的大米、牛排和百科全书，它不是什么稀缺商品，在抖音电商花钱之前，用户可能是更习惯，去淘宝、京东或者拼多多去购买的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>让用户跨平台更改购物习惯，光凭抖音自己是做不到的，但新东方帮他们做到了。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新东方用东方甄选的品牌入驻抖音电商，除了在前端贡献了新东方IP本身的故事和符合抖音平台的好内容，在服务的底层，新东方的品牌和交付也在为抖音电商做背书式的兜底。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>合谋搞钱</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>电商是很复杂的业务。</span></strong><span>除了流量，交付层面需要有非常稳定强大的供应链。这方面淘宝和京东分别积累了20多年和10多年。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>流量光环之下，刚刚成立2年的抖音，他们现阶段连自己的自营电商都做得不太好。</span><span>逢年过节字节内部给员工搞点内购会，用户样本已经非常少了，但他们围绕商品质量和服务体验的投诉率非常高。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>以至于抖音电商负责人康泽宇不得不一次次出面，在内网发帖进行长篇大论式的解释和致歉。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>从开放的姿态看，抖音电商压根就没想过全自营。他们搭建平台，售卖流量，再抽取佣金就可以了。流量广告才是字节团队的王炸技能所在。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于背后的供应链服务和交付，交给像新东方这种，有好的品牌积累和信任背书的、跟用户有情感连接的团队，既分担了压力，又实现了商业平衡，还接住了后续繁琐的供应链运营，规避了不少风险。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>合谋搞钱，抖音需要新东方。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实从去年底开始，字节国内商业化的增速就放缓了。这种放缓，会很实时地在字节商业团队巨量引擎的数据中体现出来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>细究原因，除了抖音自己的流量见顶之外，也跟大环境之下，企业和广告的营收结构改变，减少了广告预算的投放有关系；还有不能忽视的一点是，随着字节系广告系统的日趋完善，早鸟先跳抖音流量池的红利也消失了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>当增长放缓</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位拥有实体工厂、早期曾在字节鲁班系统持续投放的企业家说，早期确实是拿到了平台红利，爆一单赚回前面投入的所有。这两年，自打字节把鲁班升级为巨量引擎之后，普通中小玩家越来越难在抖音上赚到钱了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>今年以来，他和他的同行也慢下了投放渐渐开始观望。</strong>字节广告系统的策略产品经理也说，他们内部的考核标准，除了营收还有续费。产品策略上要控制在让商家赚钱或微亏，但一定平衡在一条能让商家持续付费、且不至于离开的动态平衡之上。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>供需双方，就这样一来一去，开始了博弈。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>如何让客户付费</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>字节商业化系统巨量引擎，流量主力军包含了抖音、头条、西瓜、番茄小说在内的几个端。</span><span>虽然产品矩阵很复杂，但里面流量最能打的只有抖音。</span></span></span><span>当客户付费投放之前开始犹豫了，抖音有责任去创造信用，提振企业客户的付费信心</span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新东方东山再起的故事，既热血，又符合抖音当下需要。</span><span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><span>不到一周，</span>东方甄选GMV过了亿，意味着他们的商业化投放ROI能打正了。后续也会花大价钱在抖音上买流量。这就完全符合抖音品牌及商业化模型了。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong>新东方的股价是抖音的荣誉勋章。</strong></span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>它能帮助抖音创造信用，吸引更多的</span>企业踊跃投放。它也吸引更多的用户去新东方直播间凑热闹，贡献了时长，还刺激了消费。最终，抖音流量值钱这个消息就传遍了江湖。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一局，抖音赢麻了。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>字节应该谢谢新东方。</span></p>
