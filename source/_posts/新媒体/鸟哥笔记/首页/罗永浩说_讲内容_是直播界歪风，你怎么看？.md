<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>几天前，“<a href="https://www.niaogebiji.com/tag/%E7%BD%97%E6%B0%B8%E6%B5%A9" target="_blank">罗永浩</a>的辟谣号”发布了第一条辟谣微博，澄清罗永浩此前在直播中的发言并非是调侃新东方，只是对直播电商行业的一些流行观点存在不同看法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137871452262c6951654cdd3.49878711.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在直播片段中，罗永浩提到，<strong>讲内容是直播界的一股歪风，在直播间扯淡15分钟，场观也许很高，但总收入可能不如3分钟专心讲产品。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他说：“我们直播间40%的人进来，啥也不听，直接奔购物车下单，走人，这是非常健康的一个渠道品牌。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>讲产品VS讲内容，<strong>如果说讲产品是以输出“有效信息”为主，满足的是用户的购物需求，那讲内容可能是传达一些“看似无用的信息”，满足用户的情感需求。</strong>所谓兴趣电商，就是两者的结合。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>单纯讲成交效率，专注讲产品、价格、福袋无疑是更明智的选择。不论是“交个朋友直播间”“大狼狗郑建鹏&言真夫妇”等头部直播间，还是流量不一的品牌直播间，都证明了聚焦产品的“123，下单”是更能产生GMV的方式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但要讲流量、做场观，内容可能是主播IP、付费投流之外更有效的选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但目前的直播从业者，似乎对与产品无关的内容过分热情，期待让跳舞颜值粉下单家用厨具；又对与产品相关的内容过分冷淡，没努力把卖产品变成卖场景、卖情怀、卖兴趣。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播</span><span>中，罗永浩没有对讲内容做出严格定义，比如泛泛来看，产品本身是内容，对产品特性的解说也是内容，这些都是直播电商最主要的优势。</span><span>所以我们理解的罗永浩所说的直播界歪风，可能指的是靠与产品无关的内容吸引了太多无效流量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这其中有一个值得思考的问题：</span><strong><span>当前业内是否投入了太多精力在“讲产品”上，或者花了太多时间去“瞎扯淡”，而忽视了对真正好内容的探索。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>讲产品之余，是时候讲内容了</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果过分强调讲产品，可能存在3个问题：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第一，讲产品的发挥空间可能已经很小。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>行业红利期，大家还有内卷的空间，可以在产品、价格、运营、投流、供应链上下功夫，但现在能玩的大家都玩过了。很多时候，我们刷到的直播间除了主播不一样外，其他都差不多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播电商行业快速发展的这3年，从业者花了太多心思在产品、在“货带人”上面，红利已然殆尽。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>直播电商正慢慢变成货架电商的模样，这也是很多人觉得直播间越来越同质化、越来越无聊的原因。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>讲产品很重要，但只讲产品可能无法适应直播电商行业的新需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第二，只讲产品，可能真的没流量，尤其对中小品牌来说。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品是有原力的，超级优秀的产品，主播什么都不用说，产品在直播间一上架就秒空；一般优秀的产品，主播只要能把产品卖点讲清楚，也能吸引到人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但对于大部分产品来说，既没有足以吸引用户的卖点，又没有罗永浩这样的大主播背书，主播只会讲产品，可能真卖不出去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>讲内容、整活儿也许带不来精准流量，但至少有流量，如果能恰巧创作出一些优质爆款内容，那就更是意外之喜了。这大概是无数中小品牌的心声。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近新榜报道的的工厂卖伞大哥、云爸爸尬舞大叔，实际上都是中小商家在内容上的尝试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3592596762c695169038c4.77066389.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>（相关阅读：《</span><span>单</span><span>条卖伞视频播放近6000万，工厂带货如何花式出圈？</span><span>》《</span><span>单月直播卖货1200万，尬舞的中年大叔有什么魔力？</span><span>》）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第三，卖货也许不是电商直播的唯一价值。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>讲产品卖货，肯定是直播的重要价值之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但我采访过的大多数直播从业者一致认为，直播是一个渠道生意，做不大，做品牌才是长久之计，包括交个朋友，也花了相当精力在重新加载等自孵品牌上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>交个朋友创始人黄贺在接受新榜编辑部采访时提到，MCN没有想象力，必须在抖音红利结束前沉淀出自己的品牌，“不然就只是一个渠道而已”。（相关阅读：《</span><span>罗永浩放手后，交个朋友还能稳坐机构顶流吗？| 新榜专访</span><span>》）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如果只是把直播定位成销售渠道，讲产品的电商型直播就够了，但如果想靠直播做更多事，内容型直播是必要的尝试。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，直播兼具电商、内容双重属性，如果只讲产品，只为卖货，可能是对直播价值的浪费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果能在讲产品的同时巧妙加入内容，也许有机会在直播场景中和粉丝碰撞出奇妙的品牌想象空间，为品牌创造更多长期价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>产品和内容从来不是非此即彼的关系，而是好上加好的关系。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>内容并不好做</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在直播间“瞎扯淡”，讲一些与产品无关的内容，大概率不会有结果，但如何做出好的直播内容，目前也没有太多好的可复制案例。从之间的直播内容探索案例可以发现，在直播间讲内容至少存在3个问题：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>首先，不同于有着相当门槛的产品供应链，直播内容的复制成本相对较低。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年，“美少女嗨购Go”凭着蹦迪带货的新颖内容，高峰时月销破千万，但紧接着，全网就涌现出了无数蹦迪直播间来分薄流量。（相关阅读：《</span><span>在线蹦迪为何越看越上头？美少女边蹦迪边带货1300万</span><span>》）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/204600287162c69516ebc3f3.45169616.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许是因为失去了内容上的独创性，流量逐渐下滑，今年3月，“美少女嗨购go”宣布解散，还进行了多场解散倒计时直播。直至今年6月，“美少女嗨购go”才重新开始运营，但流量已大不如前。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新抖数据统计，“美少女嗨购go”7月3日的直播累计GMV仅1.77万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>其次，好内容成本高昂，但未必能取得成正比的回报。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还是去年，因为精致的服化道、沉浸式的演技，以及剧情和产品的巧妙结合，“佰草集延禧宫正传”的宫廷直播一经出现就受到了业内的广泛关注，抖音电商官方也迅速将其作为兴趣电商的典型案例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/198569848462c69518053720.99963957.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，“佰草集延禧宫正传”并没有获得持续性增长。据新抖数据统计，近90天，“佰草集延禧宫正传”的场均销售额仅1.65万元，掉粉1.1万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>考虑到“佰草集延禧宫正传”不会太低的内容成本，这未必是一门合算的生意。对于大部分预算紧张的品牌来说，这种玩法的可复制性过低。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>最后，直播内容火爆与否，有着极强的不可控性。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以最近爆火的“东方甄选”和主播董宇辉为例，虽然相当多业内人士分析了他们的爆火原因，但一个不可忽视的事实是，“东方甄选”和董宇辉始终没变，但一个月前，他们得到的是低迷的流量和网友的谩骂，一个月后的今天，他们得到的是动辄千万的观看人次和全网追捧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/182648322962c6951b364e27.92831475.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>近期“东方甄选”的直播数据。数据来源：新抖</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内容的一个特性就是，努力了，不一定就能火。这对于DP公司和直播运营来说是致命的。<strong>品牌和老板要求保证直播投产比，但内容未必能帮他们做到。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就我了解到的情况，平台一直非常鼓励品牌和主播做内容方面的尝试，但除了少数大主播、大品牌有“多余的精力”搭团队尝试外，<strong>更多人还是在用电商人的方式做直播，把讲产品卖货当做是唯一重要的事情。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>内容似乎成了一个嘴上说着重要但很少有人能做成的“奢侈品”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>内容是机会所在</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>电商直播可以不做内容纯讲产品吗？当然可以，过去3年的行业玩法已经反复证明了这点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但一个客观事实是，直播从业者已经在产品上花了很多功夫，对于已经进入稳定期、新玩法越来越少的直播电商行业来说，内容是最有可能的突破方向。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一个值得关注的点是，抖音也好，快手也罢，本质都是内容平台，电商是平台增加营收的方式，但内容才是平台立身之本。如果所有直播电商从业者都用电商思维做直播，忽视内容产出，势必会影响平台的社区氛围。在某个合适的时间段，平台势必会压制电商型直播，扶持内容型直播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>别人恐惧我贪婪。当所有人都知道内容很重要但就是很少有人去做的时候，恰恰就是机会所在。直播既能讲产品，也能讲内容，而好的、与产品相关的、能调动用户兴趣的内容永远受欢迎。</span></strong></p>
