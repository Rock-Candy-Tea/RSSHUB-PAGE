<blockquote>
<section>你喝“凉开水”吗？</section>
</blockquote>
</section>
<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>若是以前，大家都能快速给出肯定回答。但现在，可能有些人还会多问一句：统一出的“凉开水”，还是家里烧的“凉开水”？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/22403997462e976c6b70106.31547288.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没错，“凉开水”已经不专指自己烧开的水了，因为<strong>统一在近日推出了自己的熟水产品“凉开水”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/68970817062e976c7023755.97503117.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据公开资料显示，这款“凉开水”声称：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-type="2" data-content-utf8-length="53"> </p>
<section>
<section>
<section><span>• </span>经过UHT125℃纯物理烧开杀菌，不产生溴酸盐等有害物质；</section>
<section><span>• </span>使用了31项熟水生产工艺和中国外观设计专利。</section>
</section>
</section>
<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>通过这些听起来非常厉害的技术产出的“凉开水”，最终达到的效果是印在其瓶身上加粗变红的卖点——“就是家里烧的味”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这简直让人挠头：既然和家里烧的水一个味道，我为什么不自己烧呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>再看看“凉开水”的包装，虽然看着有点“土味”，但在熟水领域算得上稍有辨识度。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与其他品牌的产品名称只用黑色字相比，“凉开水”增加了“茶壶”形状作为视觉符号，突出了“凉”字，与别的熟水做出区别。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>同时，“凉开水”的规格是1L，也不同于目前市面上比较常见的500ml上下的规格。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/165004417962e976c74236a0.23990456.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：“食品板”</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据统一相关负责人在接受“食品板”的采访中透露，“目前河南统一、陕西统一、江苏统一、新疆统一、内蒙古统一几个厂在生产，但是产能情况严重不足，很多地区九月份才能收到货，这也造成了‘凉开水’在渠道端的能见度并不高。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>实际上，不只线下店少见“凉开水”的身影，在电商平台搜索，出现的结果也是寥寥，甚至官方旗舰店也尚未上线该款产品。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/76348089962e976c77b10f2.53974250.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：统一官方旗舰店</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>值得一提的是，“凉开水”属于统一旗下的子品牌“格泉”。</span></strong><span>与统一曾经高调推出的中高端水品牌“爱夸”“巴马泉”不同，“格泉”一直走的是平价路子，此次的“凉开水”也延续了同样的定价策略，放大自己便宜大碗的优势。但也可以说明，“凉开水”算不上统一的重磅武器。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该说不说，单看产品详情页的视觉设计，也可以感受到不同产品的价格区间了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/180188889162e976c7ce83e3.82877489.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>左为“爱夸”，右为“巴马泉”（图源：统一官方旗舰店）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但即便<span>价格上不是“水刺客”，规格也属实良心，仍有很多人不看好统一做熟水。甚至有网友“赌气”发言：<span>我赌100块钱，统一卖不好“凉开水”！</span></span></span><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/92446283062e976c8235929.71939838.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据来源：统一旗舰店暂无销售“凉白开”，因此选择了销量最高的<span>天猫</span>店铺价格；另两个商品价格来自品牌天猫旗舰店。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这又是为什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>

<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当下，做熟水生意的不止统一一家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如果论起熟水品类的鼻祖，目前业内比较认可的是今麦郎在2016年推出的“凉白开”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此之前，各大品牌都在饮用水赛道挤破头，水源地、广告语、代言人等方方面面都在内卷，知名的不知名的山河湖海都能拿出来讲故事，你家是“水中贵族”，我家便是“大自然的搬运工”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在消费者<span>还没分清<a href="https://www.niaogebiji.com/tag/%E7%9F%BF%E6%B3%89%E6%B0%B4" target="_blank">矿泉水</a>、纯净水之间的具体区别时，</span>今麦郎就率先从这卷生卷死的赛道里跳脱出来，创造了一个全新品类——熟水，并以<span>“中国人自古喝熟水”的概念教育市场。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“凉白开”刚推出那会儿，不看好的声音也很多。但从2017年的2.5亿销售额，到2019年突破20亿元，用数据打了翻身仗的“凉白开”，给后来的熟水产品打了个样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/51539190162e976c854d0e1.04490748.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：今麦郎饮品旗舰店</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2020年，康师傅上线“喝开水”，类似的包装、卖点和定位，都让人心知肚明，这款产品就是为对标今麦郎的“凉白开”而来。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据康师傅2021年年报显示，其瓶装水品类营收45.48亿。而中金证券公司调研发布报告称，“<span>喝开水</span>”2021年销售额已接近10亿元。可见，在短短时间内，熟水已经对其营收产生了巨大贡献。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/156833166762e976c8b2a624.55713117.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：康师傅饮品旗舰店</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再加上最近推出“凉开水”的统一，方便面三巨头终于再次在熟水赛道凑齐了。而瓶装水大佬如农夫山泉等，也在这些年陆续推出了多款熟水产品，这个领域已经不是全然的蓝海了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>从时间线来看，且不说今麦郎，统一也比老对手康师傅晚了两年。而姗姗来迟，是商场大忌。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今提起熟水产品，“凉白开”成了大多数人的第一反应，这便是先手优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，熟水不同于其他饮用水，没有那么多花里胡哨的噱头能打，高温灭菌之后的水，来源自哪座名山、哪个大湖都不重要了。就算康师傅找来了流量明星代言，热度也不过一时。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>从各家大同小异的卖点上，可以看出因为技术壁垒不高，产品同质化一览无遗。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/39005181462e976c90927a3.94180751.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：各品牌商品详情</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，统一做水并没有多成功，明明从低端到高端的水品牌都有布局，偏偏没一款有水花。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，统一为何还要匆匆入局熟水？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，也是“形势所迫”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>熟水在一定程度上算是新风口，各大巨头先后下场，本就在做水方面没有什么明显优势的统一当然也得跟风试水。</span></strong><span>且不说能做到多前列，至少是完善了自己的水饮产品布局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>再者，统一已经很久没有爆款单品出现了。</span></strong><span>如今提起统一，大家想到的仍然是阿萨姆、小茗同学之类有点年头的产品，其余新品常有，但能刷出国民度的却极少。所以，统一不会放过任何一种做出爆款的可能性，反正家大业大，放开去试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但如果讨论统一能不能在熟水领域做出成绩，否定的概率或许更高。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>统一来得太迟，它的老冤家们已经在这一块铺垫许久，双方的市场心智沉淀程度不可同日而语。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再加上如今水企之卷无穷无尽，水源地的故事讲不动了，水的细分和IP营销便提上了日程。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今麦郎当年的熟水概念横空出世，后推出了“想开瓶”包装，甚至开了主题店；农夫山泉推过针对老年人的含锂水，借着综艺《忘不了餐厅》IP，把“情绪元素”锂推向市场......而统一在水方面至今还没有拿得出手的代表作，“凉开水”能否逆风翻盘成为黑马，还得由时间来验证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>不过话又说回来，都是高温煮水，自己烧了放凉的开水不香吗？说到底，都是卖水的噱头罢了。</span></span></p>
