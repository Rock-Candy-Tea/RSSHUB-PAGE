<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去一年，<a href="https://www.niaogebiji.com/tag/%E8%A7%86%E9%A2%91%E5%8F%B7" target="_blank">视频号</a>内容生态逐渐丰富。今年，视频号直播进一步加大扶持力度，通过商家激励计划，扶持不少于10万个优质商家，吸引了不少品牌和达人入驻视频号。</span><span>直播俨然成为一种流行趋势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而很多新手主播还不知道视频号直播间如何设置才能更有吸引力，今天给大家带来<strong>《让直播间人气暴涨的16大技巧》，分为基础篇、进阶篇和高阶篇</strong>，<strong>堪称保姆级教程，值得你点赞收藏学习。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/171865746762183c3259d455.90022865.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>直播间封面如何设置</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（1）手机直播封面图规范为6:7，其中会截取9:16的区域作为直播间封面和分享封面，而PC端直播封面图规范为9:16，因此，建议制作封面图时，将主要内容输出在9:16的范围，以适配多种场景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/72133221062c5faf80d9009.28989482.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>直播画面设置</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（2）视频号直播可以开启美颜功能，PC端的画面源选择【摄像头】后，勾选“美颜”即可开启。移动端可在发起页或直播过程中，点击【画面】功能，进行视频画面设置，手机端比PC端还多了美妆、滤镜、画面镜像、镜头翻转等功能，大家可以根据直播需要调整画面。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（3）在直播发起页或直播时，可点击【音乐】-【添加音乐】来设置背景音乐，手机端可直接搜索添加背景乐，进行音量调节、排序、移除音乐等操作，甚至可以发起【演唱歌单】，添加待唱歌曲名单，观众可通过送礼点歌，增加直播氛围。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（4）发起直播时，可以对本场直播进行分类，在直播发起页选择【分类】，根据自己的直播内容选择，特别是直播带货时，务必选择购物分类，且需要在直播间上架相关商品，否则会违反视频号运营规则，准确的分类更能精准推流。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>直播间观众</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（5）可在开播前设定直播间观看人群，目前分为公开、指定观众可见和付费可见。一般建议设定为【公开】，即所有观众可进入观看；【指定观众】需要主播从聊天中选择或者前往视频号助手（https://channels.weixin.qq.com/）添加名单成员；【付费可见】只有付费观众可观看完整直播，未付费观众仅可试看3分钟，且付费直播会生成回放，主播可继续售卖该场直播。公开吸引公域流量，指定人群方便试播、吸引私域流量等，付费直播的接受度不高、关键在于内容，但无疑是一大新变现渠道。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>连麦互动</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（8）主播可在直播过程中与观众或其他主播发起连麦，PC端需在开播页选择开启连麦，移动端点击右上角的【...】，在弹出的功能菜单中选择【开启连麦】即可。开播后可使用连麦功能与观众或其他主播进行互动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是PC端还是移动端，都可以自定义观众连麦的条件，目前支持“所有观众可申请”、“送礼观众可申请”和“已关注观众”这三个条件。与主播连麦目前支持搜索邀请主播连麦和随机匹配主播连麦。</span></p>
