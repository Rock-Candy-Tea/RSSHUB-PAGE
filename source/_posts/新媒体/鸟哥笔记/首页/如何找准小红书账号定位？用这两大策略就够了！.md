<section>
<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最近一段时间，一直在做博主陪跑，陪跑第一件事，就是帮助博主找准账号定位，所谓定位决定眼界，定位决定定位，定位也决定变现。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如何进行定位？我们认为成功IP定位，具备定位清晰、有成长空间、商业价值和能持续创作四大特点。</p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="rich_pages wxw-img aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" title="如何找准账号定位？用这两大策略就够了！【建议收藏】" src="https://image.woshipm.com/wp-files/2022/08/CA2NaN2i0e2mFOXAlhfR.png" alt="如何找准账号定位？用这两大策略就够了！【建议收藏】" data-action="zoom" referrerpolicy="no-referrer"></p>
</section>
<section>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;">图1：成功IP的四大要素</h4>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">定位清晰：</span>有清晰的人设，用户刷到你的笔记，从昵称，简介，内容等知道你是做什么，能够给她提供什么价值；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">成长空间：</span>品类决定成长的天花板，定位不同赛道，规模就不同，如母婴行业，50w-100w的头部博主有32个；汽车行业，50w-100w的博主只有8个，美妆行业50-100W博主则更多；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">商业价值：</span>具有变现能力，想清楚你的变现方式，未来靠广告还是知识变现，若靠知识变现，是你的课程还是付费文章；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">持续创作：</span>是自己擅长，有精力、热爱、能够持续创作的内容。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是很多博主并不清楚自己要做哪一方面，能做哪一方面？可通过定位三象限和四分法找到自己擅长的板块。</p>

<section>
<section></section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span style="font-weight: bold;">一、两大策略</span><span style="font-weight: bold;"></span></h2>
<section><span style="font-weight: bold;"></span></section>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 16px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span style="font-size: 20px;">（一）定位三象限</span></h3>
<section><span style="font-size: 20px;"></span></section>
</section>
</section>
<section>具体围绕我喜欢什么？我擅长什么，我能学习做什么，我最愿意花时间在哪边？竞争对手在做什么？由此来确定自身定位。</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="rich_pages wxw-img aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" title="如何找准账号定位？用这两大策略就够了！【建议收藏】" src="https://image.woshipm.com/wp-files/2022/08/Ea6f6alCFRhP8a7KFWt3.png" alt="如何找准账号定位？用这两大策略就够了！【建议收藏】" data-action="zoom" referrerpolicy="no-referrer"></p>
<section>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;">图2：账号定位3象限</h4>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1、我喜欢</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通常是你的职业、爱好和生活。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">职业：你目前从事的职业，它能够长期输出知识，如你做小红书运营，可撰写小红书运营知识；做理财可撰写理财相关，做心理，就围绕心理知识做定位。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">爱好：你个人喜欢什么，对什么感兴趣，比如你对穿搭、化妆、书法、跳舞、写作、音乐喝酒、等领域感兴趣，单纯爱好不代表能胜任内容输出，要在自己喜欢干，有能力干和用户需求中找到一个聚焦点，而这个点就是你的定位。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">生活：人群的生活状态，如学生、家庭煮夫、上班、模特等生活领域，记录不同的生活状态，也能引起用户关注。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2、用户喜欢</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你的内容是给谁看？用户画像是什么样？用户在搜索哪些内容？用户在关注哪些人？在小红书搜索品类关键词，市面有哪些热门的笔记，这些都需要你去调研。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">3、对标在做</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对标账号就是你想成为什么样的账号，哪些账号做的还不错，她们有什么优点，去分析她们的内容和模式。</p>
<section></section>
<section><span style="font-size: 20px;"></span></section>
<section>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 16px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span style="font-size: 20px;">（二）定位四分法</span></h3>
</section>
<section>定位四分法即通过我是谁、我做过什么、做出什么，想分享什么，形成一副人设定位地图。</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="rich_pages wxw-img aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" title="如何找准账号定位？用这两大策略就够了！【建议收藏】" src="https://image.woshipm.com/wp-files/2022/08/lXCA4Szl2GGreyNxOp1d.png" alt="如何找准账号定位？用这两大策略就够了！【建议收藏】" data-action="zoom" referrerpolicy="no-referrer"></p>
<section>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;">图3：博主定位四象限</h4>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">我是谁：</span>包括昵称、性别、年龄、生活状态、价值观等方面，越详细越好，内容越多，后期选题的方向也会更多；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">我</span><span style="font-weight: bold;">做过什么：</span>将自己过往经历总结，工作、生活的亮点，即使是失败经验都可以总结；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">我</span><span style="font-weight: bold;">做出什么：</span>在过往的经历中，做过哪些内容，如定位母婴，看一下是不是自己育儿知识更有经验，如做知识博主，也看自己有哪些知识可分享；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">想在小红书分享</span><span style="font-weight: bold;">：</span>自己想在小红书分享哪些内容，通过何种方式变现，评估能否持续更新，如知识博主，打磨了哪些课程，怎么包装和进行售卖。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这两大方法，能够更清晰看到自身的特色，形成第一版博主定位思路。我们也分享，我们在陪跑客户账号时的具体思路，本月我们也会做账号陪跑训练营，大家一起做账号。</p>
<section>
<section>
<section></section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span style="font-weight: bold;">二、定位逻辑</span></h2>
<section><span style="font-weight: bold;"></span></section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们做定位逻辑主要有3步，了解背景、打标签、找对标，做内容。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">（一）了解背景</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">先了解博主个人背景，现在在做什么，之前个人成长经历，职场经历，简单的情感经历，平时精力分配，自身想做小红书的目的，想要进行变现的方式。沟通下来会给博主打标签，如成长逆袭、职场跨行、旅游达人、创者者身份、搞钱达人等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">（二）</span><span style="font-weight: bold;">打标签定位</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">具体选择哪一个行业？根据博主标签去建议定位方向，如博主有职场、时尚、医美特长，从平台角度，去建议以职场或时尚为主，因轻医美内容创作也会有一定限制。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">（三）找对标</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">找到博主定位方向后，找对标账号学习，从内容、变现和学习整理，由此确定接下来创作选题，做好博主爆款选题库。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="rich_pages wxw-img aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" title="如何找准账号定位？用这两大策略就够了！【建议收藏】" src="https://image.woshipm.com/wp-files/2022/08/eYF06yFh1RzAQsVtSHjv.png" alt="如何找准账号定位？用这两大策略就够了！【建议收藏】" data-action="zoom" referrerpolicy="no-referrer"></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;">图4：对标账号拆解</h4>
</section>
<section>确定自身要主攻的领域后，和对标账号的研究后，制定博主每周运营创作重点，具体某一篇文章的素材和方向，打磨发布的每一篇笔记。</section>

<section><span style="font-weight: bold;"></span></section>
</section>
</section>
<section>确定定位后，账号塑造上，通过人设、内容和变现3种方式。</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="rich_pages wxw-img aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" title="如何找准账号定位？用这两大策略就够了！【建议收藏】" src="https://image.woshipm.com/wp-files/2022/08/PtfV2ntZqjNqAf06sYHL.png" alt="如何找准账号定位？用这两大策略就够了！【建议收藏】" data-action="zoom" referrerpolicy="no-referrer"></p>

<section>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">图5：账号塑造3要素</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">（一）人设</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">命名：</span>使用名字+内容细分领域，让用户一眼知道你是干什么的，PS小红书昵称的修改，规定是一个月一次。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">头像：</span>可选择自己真实、代表自己个性的图片，个人写真都是不错选择。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">简介：</span>简介的本质凸显核心差异化，引发客户行动，在账号设置，凸显自己在这个专业的优势或者高光经历，专业特长。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">好的简介可以快速获得用户信任和认可，也更好的体现你的人设。为什么要放邮箱呢？邮箱是官方唯一允许放的联系方式，便于进行商务合作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">如醒醒来喝茶</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">定位茶叶博主，命名：品牌+品类命名，突出茶的属性，识别性高，且其简介也是强大定位，好茶防坑指南，茶学专业硕士+普通上班族，凸显茶叶专业实力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="rich_pages wxw-img aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" title="如何找准账号定位？用这两大策略就够了！【建议收藏】" src="https://image.woshipm.com/wp-files/2022/08/TyCmOPGzK6aXTwIoK5Lp.png" alt="如何找准账号定位？用这两大策略就够了！【建议收藏】" data-action="zoom" referrerpolicy="no-referrer"></p>
