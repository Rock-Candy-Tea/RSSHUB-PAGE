<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="4"><span>谁也没想到，生鲜电商领域头牌玩家每日优鲜会落得如此境地。在收到纳斯达克上市资格部门发出的退市警示函的一个月内，每日优鲜的股价并没有大幅回升。也就是说，每日优鲜退市风险仍存。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每日优鲜之外，扛起<a href="https://www.niaogebiji.com/tag/%E6%96%B0%E9%9B%B6%E5%94%AE" target="_blank">新零售</a>另一面大旗的<a href="https://www.niaogebiji.com/tag/%E7%9B%92%E9%A9%AC" target="_blank">盒马</a>也不容易。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据天眼查APP信息显示，6月17日，深圳市监管部门对深圳盒马福田区第五分公司的“农产品及农用物资违法行为”进行了处罚，据悉，处罚原因为“销售农药残留超过食品安全标准限量的农产品”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16834058362c9d2c728ddb4.98637923.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一次市场监管部门的处罚可能对于盒马门店的经营影响不大，但盒马这几年的处罚犹如家常便饭，背后透出的信息值得让人深思：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便生鲜行业头部玩家花大力气去搭建生鲜供应链以及完善经营管理，但在质量安全上并非“密不透风”，类似“农药残留超标”等食品安全风险仍存。食品安全风险，无疑为盒马的新零售探索之路蒙上一层阴影。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/159387675562c9d2c78ea9a3.08433068.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>细节论成败， “新零售”究竟是不是伪命题？</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年底，阿里开始推行经营责任制，盒马从原有事业群独立，并于今年开始自负盈亏。1月初，侯毅在盒马内部信中指出要从单店盈利提升为全面盈利。此后的一段时间，盒马开始压缩开支，裁员、关店、缩减新开门店硬件投入。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盒马独立于阿里的一个重要原因可能在于，新零售实在亏得太厉害了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盒马的困惑在于，即便是拿出了阿里数字化中台的看家本领，以及不断创新经营模式并形成了自身的差异，短期内还是无法解决不挣钱的问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面可能是因为<strong>多次模式迭代的成本太高。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从盒马鲜生店、到盒马MINI、盒马邻里、盒马X会员店，盒马已经尝试过多种业态，这么多业态的尝试中，从开店到关店再到新业态，这里面其实有相当大的成本，包括不限于人员的变动、店面装修、租金投入等等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关键在于，<strong>即便是盒马业态多次迭代之后，也没能形成稳定的规模化盈利的局面。</strong>盒马似乎成为了吸收阿里资金的一个黑洞，即便是财大气粗的阿里，也很难再继续这样的高成本投入。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去侯毅坚持的一个原则是，亏损不重要，关键是能不能把模式跑通。盒马在商业上的迭代能力也是侯毅坚信盒马能够成功的原因之一。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>试错不是没有代价的，侯毅的原则背后还隐藏着一个前提，那就是最终跑通的模式不仅要为自己的成本买单，也需要为之前的试错成本买单。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而这些成本，可能会拖累了整体的经营效率，进一步增加盒马的盈利压力。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就好比买股票，每一次商业迭代的过程，就像是一次加高杠杆的过程，当商业模式最终被证实或者证伪，前期的投入究竟是盆满钵满还是血本无归便会有了答案。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另一方面，或许是因为新零售带来的价值增量有限。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新技术应用提升附加值，适合于毛利率高于50%以上的产业，因为这些产业有足够利润空间来支付技术成本。而根据国家税务部门的统计数据，零售行业的平均毛利率仅为4%－9%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这个角度来看，生鲜赛道的商业逻辑在于细节上的运营，比如坪效究竟多少，品类转化率高不高，SKU组合合不合理等等。相比这些，一个更值得资本和行业反思的问题是，<strong>生鲜行业可规模化的价值是不是真的有BP中计算的那么高？</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于新零售而言，如果从“旧零售到新零售”的升级，成本远远大于增值，那么经营细节提效究竟还有没有意义？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从新零售行业当下普遍处在盈利困境中来看，无论是业态创新还是数字化中台提效，在生鲜赛道以及整个零售赛道，<strong>技术对于经营细节的提效创造的附加值远远不够。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新零售的核心逻辑是什么？是通过数字化中台的资源调度，来使得运营变得更精细化，从而做多零售商品的附加值。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比传统零售，新零售的创新点核心就两个方面<strong>，一个是增加来线上流量获取渠道，另一个就是做高品牌价值。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但颇为尴尬的是，由于线上履约成本高于线下履约成本，这导致线上业务增加的坪效并没有转化为店面的实际盈利能力。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，线上业务履约就意味着要面临单价能不能裹住履约成本的问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如说，买瓶酱油和买10斤海鲜都需要30分钟配送，即便是用户愿意拿配送费，平台从中又能抽成多少？参考外卖平台，30%可能就到顶了，对于10斤海鲜的价格来说，这样的抽成还能覆盖成本，但对于单价只有5块钱的酱油单子呢？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>需要注意的是，如果线上买生鲜，单价本来就不高，而且对于盒马等平台来说，这类线上订单占大头，酒水等高毛利商品类的订单，美团、饿了么的超市快送才是主流。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，为了达到了全面盈利的目标，盒马在不断扩充线下业务比例减少线上业务比例，相关人士透露，盒马希望将线下占比从30%提升到50%。换句话来说线下业务的盈利能力远比线上业务要强。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>此外，用运营消费品牌的思路来运营平台品牌，虽然能圈住部分用户，但并不能带来更多的利润增量。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盒马的消费人群主要是18岁到39岁的人群，这些人群对生活品质有一定的要求，也就是说，对于生鲜品类来说，盒马的货损可能要比普通卖场更高。再加上盒马的毛利的SKU并不多，因此盈利比较困难。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此盒马的解决思路是，通过特色SKU来做多品牌价值。比如早期通过帝王蟹为爆品冷启动，以低价海鲜为爆品，加强用户的品牌记忆，进而为复购和转化做铺垫。事实上，这些运营方法也不总是成功的。以前盒马就曾因“盒马鲜生标签门”、“榴莲品种造假事件”、“帝王蟹有活虫”、“椰浆过期事件”等问题登上热搜。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些新的运营方式下，虽然用户的消费粘性提升了，但盒马线下门店的运营成本其实并不比传统商超更低。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也就是说，无论是线上履约还是精品SKU策略，盒马本质上都没有比传统商超实现更低的经营成本，<strong>在缺乏成本结构优化的支撑的前提下，新零售要想实现全面盈利，恐怕还是得向传统零售靠拢。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/15239267856035c365ee8936.06370684.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>盈利“宿命”中的进击，盒马“救赎”新零售</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6年的亏损让每日优鲜、盒马等生鲜玩家们明白的唯一一件事：<strong>技术和流量不是万能的，零售最终还是要回归成本核算，平衡收支。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个残酷且大家都不愿接受的事实是，<strong>零售行业没有所谓的“新物种”。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>考核零售商业的标准永远只有一个那就是“盈利”。“盈利生存标准”之下，每日优鲜濒临美股退市的现实，似乎宣告了新零售单一业态的终结，而盒马坚持的“多业态”依然在试图为马云寻找答案。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>生鲜对于卖场来说是引流品，高损耗低毛利但又是刚需，对于卖场来说生鲜品的作用是就是带动百货饮品等高单价高毛利产品，而后者才是盈利的关键。</span></strong><span>由此来看，盒马们盈利的道路在生鲜之外。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>X会员店是盒马“救赎”新零售的新尝试。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盒马推出X会员店的目的很明显，就是想要在国内试图复制Castco模式的成功。在美国，Costco更像是商品百货的集中采购站，家庭覆盖率为90%，乡镇地区的几乎每个家庭每周都要去Castco采购为期一周的食品、用品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Castco为会员带来了两大价值，相对便宜的价格买到了品牌可靠、SKU丰富的商品；提供了一站式的购物方案，满足一周使用时间的采购量。在国内，第一个需求被京东、天猫、拼多多很好地解决了，第二个需求则不存在，尤其是生鲜类，国内消费者更讲究随用随买，图个新鲜。因此，完全照搬Castco并不可取。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然国内与国外市场不同，但不同的市场中同样收入水平的人群需求的偏差其实不大。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，国内一二线城市中，年收入在二十万元以上的家庭对于平价高品质商品的需求仍然存在。这也是为什么优衣库、无印良品在国内受到欢迎的原因。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>山姆会员店的火热似乎已经证明中产人群的消费能力，这也是盒马推出X会员店的底层逻辑：<strong>商业模式被验证，不必再做过多的模式迭代，从而尽可能地快速盈利。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，从零售行业当下的局面来看，X会员店可能依旧难以承担盒马盈利的目标。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，目前盒马6～7个X会员店中多半为新开，这意味着前期依旧需要大量的投入，而<strong>对于当下阶段的盒马而言，这可能意味着亏损还会持续一段时间。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盒马要想把模式真正跑通，必须遵循这样一个公式：<strong>未来几年的营收规模x净利润率>净亏损。</strong>X会员店走上正轨之后，盒马需要回答两个问题：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一：会员店模式下，未来3年内盒马整体能不能保持20%到30%的增长率？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二：5到8年内能不能持续20%以上的增长？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的商业模式往往在初期处于高速增长阶段，增长率往往超过50%。零售行业整体的增长率并不高，因此20%到30%的增长率更为合理。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>净利润方面，据统计零售行业的利润率普遍在10%左右，由于会员店的平均客单价更高，且加上毛利率更高的自有商品，实际的毛利率可能会有所上浮。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>亏损方面，由于盒马并未公布数据，从阿里最新季报来看，盈利能力和GMV增速似乎并未达到预期。对比叮咚买菜来看，2021年净亏损总计达64.29亿元。盒马方面想必也不会好到哪里去。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从现实来看。如果没有疫情，做到这个增长目标也许不会那么难，<strong>但目前来看，外界对盒马未来的营收预期可能没有那么乐观。相比行业本身的商业迭代，消费端的行为变化对行业未来增长的影响也不容忽视。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个大的变化是，人们对于非必要消费的支出在进一步减少。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这两年，人们对消费复苏的期望相对乐观，但从宏观层面来看，由于风险意识觉醒，中等收入人群的储蓄意识在增加。从统计数据上来看，这几个月的社消数据并不理想。数据显示，今年6月份社会零售预期仍然为负增长，乐观估计可能会回升到1%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一个基本的事实是，新零售商业的核心用户群体为中产人群，但这部分人往往对外部变化的感知最为明显，尤其是对消费的感知。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，前几年大家对未来收入预期乐观，于是多花二十块钱买菜并不算什么大事，毕竟买来了更好的服务，但这两年收入锐减，且未来预期不佳，即便是年收入20万左右的中产家庭，也会考虑缩减支出，那么原本这不重要的二十块钱可能就变得重要了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换句话来说，即便是购物体验更好，生鲜SKU更标准化，但终端消费的预期在走弱。因此，中产人群的消费复苏周期被拉长之后，3到5年内，终端消费水平才有可能回到2019年的水平。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于盒马来说，消费恢复周期的拉长，意味着增长增长周期拉长，增速可能不及预期。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换言之，即便X会员店真的能够成功，那么真正能够盈利的时间节点可能也在3到5年左右，在这样的一个窗口期，如何摆脱亏损的困境仍然是盒马与侯毅需要深思的问题。</span></p>
