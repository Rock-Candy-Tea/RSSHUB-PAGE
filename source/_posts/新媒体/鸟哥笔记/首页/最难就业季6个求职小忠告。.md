<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><span><img src="https://ngbjimg.xy599.com/39134872362e155864a7707.17814887.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1076万人</span></strong></span><span>，这是教育部公布的2022届高校毕业生人数，堪称历史之最。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再加上疫情连续几年的影响，越来越多的企业缩减招聘需求或暂缓招聘、缩减或优化部分岗位、降薪、砍掉一些非必要项目……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>年年就业都难，2022或是最难就业季。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>求职形式不容乐观。很多人说：“我可能短期内找不到工作了。”更有甚者，去年毕业季到现在依旧没找到合适的工作。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从今年研究生报考人数，和公务员报考人数也可以看出，首先是就好业难；然后是后疫情时代，越来越多年轻人希望找到更稳定的饭碗，于是“小镇做题家”越来越多，而人数逐年增多上岸率却越来越低，举步艰难。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，我还是想</span><span>给大家分享一些求职中的经验，希望大家能把握目前每一次更珍贵的招聘机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>今天只谈谈一个点：面试！</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟面试是求职极重要的一环，面试搞砸前功尽弃。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/344584022602f35600bc735.33574712.gif" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>噢，对了，还有一个想说这个点的原因。就是其实现在大多年轻人</span><span><strong>就业不难，难的是就好业</strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人找不到工作，本质是我们说的摩擦性失业，说白了不是找不到工作，而是想找个10000的，却都是些给5000的，结果就宁缺毋滥了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而恰恰是这样，我们想寻求符合标准的好岗位，就越要把面试这一环做好。因为这样的岗位名额少竞争大，肉少狼多，你想脱颖而出就必须成为那只最亮眼的狼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好了，回到正题。下面说的都是面试中最基本且能做好的点，句句大实话，真心希望能帮到每一个求职者。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/45816100162e155866c9425.57141811.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，我们需要清楚地明白一个点——</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没有人能在短短10分钟、20分钟就能完全看透你了解你。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一场面试，它更多是对你这十几分钟表现的评估，而不是对你整体综合实力的彻底判断。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>哪怕你再有实力，如果面试表现得不行，那很可能就被判定为 “不行”，每天不计其数的求职者正被 “面试” 扼杀。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，应该怎样让面试把握更高呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>先回想一下面试场景，一般是这样的：</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<section><span>面试环节刚开始你会特别紧张、然后慢慢进入状态，渐渐会达到一个最佳状态，之后时间一长状态又慢慢降低，都不知道该说些什么了。</span></section>
<section><span></span></section>
<section><span>而面试官呢，一般刚开始会侃侃而谈比较热情，然后注意力慢慢会降低或者被一些事情打断，最后结尾的时候又会显得兴致勃勃。</span></section>
<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是不是这样的？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里面就有一个致命问题：在面试官注意力最强的时候，你的表现可能最差；而等到你表现最好时，面试官实则心不在焉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1647047376602f3560e7f1a8.93300197.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就会让你的面试质量大打折扣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前面说了，一场面试，更多是面试官对你这十几分钟表现的评估。<strong>显然，在面试官最感兴趣时的发挥，分值更高。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从上图我们知道，这个时间就在开场和结束时，必须要更重视。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知道了这一点，老贼给大家几个面试小建议。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em><span>01</span></em></span><span> 、</span><span><strong>开头要精彩</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首次接触是最佳加分时刻，不过，也可能变成减分时段。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基本上，面试开始前都是一段自我介绍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人不是很重视，还很抵触：“怎么老是自我介绍，能不能换点别的，有什么好介绍的？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>其实，这已经是观察的开始。能很好介绍自己的面试者，会大大提高面试官的评分。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不要一上来就只有 “我是谁、我来自哪里、什么学习毕业、性格什么样” 等等，这个太干巴巴，都是一些基本信息，不会有什么印象。</span><span>（除非你打听到这个面试官是你老乡或校友）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/453112854602f3561896398.19840054.jpeg" referrerpolicy="no-referrer"><br><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自我介绍时，在做基础信息说明后，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>重点要说的是自己的核心优势、工作成果或者特别经历，最好是用一个具体案例、项目，或一个生动的故事来讲。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而不是仅仅空洞的说：“我很擅长团队管理、我很擅长项目运营、我很擅长写文案......”</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em><span>02、</span></em></span><span><strong>故事要生动</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>想要突出你的 “卖点”，最好的方式就是用故事形式来讲，结合你之前的经历。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>讲故事是一种很高明的沟通策略，不仅显得真实，而且感染力更强。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>故事，首先要有</span><span><strong>主角</strong></span><span>（你自己），然后有</span><span><strong>情节</strong></span><span>（你的亮点项目或经历），最重要的是要有</span><span><strong>冲突</strong></span><span>（处理项目的挑战和遇到的问题），还要有</span><strong><span>转折</span></strong><span>（各种努力和创新后的成功，也就是你能力的体现）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样才能让面试官印象深刻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一定要有细节、细节、还是细节，这是面试官最关注的，也是故事打动人的关键，更是你核心 “卖点” 的体现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，对于你过往的重要经历和取得的成果，一定要提前回忆和总结，演练好，不要说的时候还在使劲回想，磕磕绊绊。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em>03、</em></span><span><strong>亮点要突出</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一次面试，时间很短，不要想着面面俱到，什么都突出一下，最后可能面试官什么都印象不深。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像你去包装一个产品，想宣传十几条卖点。卖点都还不错，可当用户看了后，结果很糟糕。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他甚至不知道这到底是个什么鬼了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>面试也一样，想要给面试官峰值体验，就不要不温不火，你需要突出一二个核心亮点，而不是讲个没完。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，你对自己哪个核心优势、能力最有把握？提前想</span><span>编</span><span>好，聚焦着说。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>特别特别重要的，思考好你的具体价值在哪，用以应对面试官。我就经常会问。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/798024904602f3562256fb4.63402154.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>噢对了，稍微夸大一点也无妨。</span><span>（反正，别人本来就会觉得你有所夸大）</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em><span>04</span></em></span><span> 、</span><span><strong>结尾要升华</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在结尾时，面试官又会满血复活，一般会问你：“你还有什么问题需要了解吗？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>不要一下子就感觉终于要完事了，想当然的就说，没有了。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是一个专门留给面试者的开放提问环节，如果说没有了，很显然你就放弃了一次绝佳的加分机会，甚至因此减分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你可以向对方询问当前岗位的后续发展方向、目前团队急需解决的问题、在公司中的定位、企业战略等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/42495603602f3562a04d05.15170328.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><br><span>相应的可以加入自己的一些看法，要敢说。但不要太过，点到为止，不然就容易飘了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>切记，别一到提问环节就马上问福利，这个自己问HR就行了。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><em><span>05 、</span></em><span><strong>了解要充分</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>越是你想进的公司，越是要提前了解，越充分越好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>包括公司业务情况、商业模式、产品情况、人员情况、最新动作、企业文化、员工标准、对应岗位标准等等......</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>甚至包括老板喜好、性格、风格等等，都可以提前尽量在各种渠道了解，反正又没坏处。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这是你的诚意，也是你面试前的信息储备。当然，还有一个被动效果，你了解越充分，表现就越自信。</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em><span>06</span></em></span><span> 、</span><span><strong>练习要提前</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面试本质来说就是一次社交博弈+个人演讲！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>既然如此，那提前练习就显得非常必要。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1380797888602f35636fa5d4.30514474.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说你的工作真实能力从 60分 到 90分 需要5年积累，那你的一次面试表现从 60分 到 90分 可能只需要练习30分钟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就是面试很不公平的地方，但也是很公平的地方，机会都是留给准备更多的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那练习什么呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对着镜子或者找个朋友，练习自我介绍、练习提问、练习讲故事等等你会用到的所有准备。调整优化你的音调、表情、肢体语言、语速、反应能力、表达能力等等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实在不行，就一个人在自己的脑子默念也好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>最关键的是，让自己提前适应这种面对面交谈模式。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>又不用花钱，还能获得奇效，何乐而不为？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好了，上面的建议，希望能真真切切地帮到大家，拿到最后的offer！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1307547361602f356439a8b6.24270811.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然了，还有一点需贯穿整场面试，就是要保持自信。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>不过千万记住，自信是一种应对突发情况临危不乱的能力，绝对不是死撑啥都懂，这会害死你。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，再给大家掏心窝子说一个点：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>不要一味取悦面试官！</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>虽说现在找工作更困难了，但好工作始终不是低三下四来的</span><span>，好像面试官就是天，处处都取悦他，感觉进了面试的门就低人一等了。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>面试官并不比求职者高一等，这应该是要记住的。</strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/877517822602f3564f3ee31.96909104.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面试的目的也不仅仅是为了offer，更是找到一个好的靠谱的公司。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一旦你进入弱者模式，完全放弃自己的想法，一味迎合雇主，就只能被动答题。很容易就被面试官带着跑，就成了单纯的他挑选你，甚至是打压你的过程。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>结果？进来后发现这公司根本就不行，比你上家还差。<span>欲哭无泪。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你不该一味迎合面试官，也要迎合自己！除非，你就是想先随便过渡一下。</span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><img src="https://ngbjimg.xy599.com/1830814377602f3565b25352.34453721.jpeg" referrerpolicy="no-referrer"></span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好了，就说到这，总之在面试时：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开头结尾，表现要棒；故事生动、死磕细节；突出优势，不懂别装；死撑易露馅，亮点留心中；和面试官平等双向沟通。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>希望你得偿所愿，前程似锦。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span>相信自己，别因为磨难，停住你的脚步</span></strong></span><strong><span>。</span></strong></span></p>
