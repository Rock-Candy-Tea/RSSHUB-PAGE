<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span data-darkmode-color-16613342462595="rgb(163, 163, 163)" data-darkmode-original-color-16613342462595="#fff|rgb(178, 178, 178)" data->作者｜鲍星娃</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>巨头盘踞的跨境<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%95%86" target="_blank">电商</a>赛道，有太多起大早、赶晚集的规模较小玩家，即在行业发展早期就探索建立了新模式、吸引了首批目标客户，但后期并未充分利用先发优势，市场份额不断被吞噬，直至被边缘化甚至是完全消失。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前有母婴电商平台 " 蜜芽 " 发布 APP 停服公告、寺库被传出破产和疑似跑路的消息，后有被阿里收购的考拉海购逐步进入战略收缩状态，如今，沉寂许久的洋码头也自曝正处于众多危机之中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>洋码头创始人曾碧波在公开信中坦言，公司的内部和外部均面临巨大压力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>外部来看，疫情防控措施对进口电商冲击严重，尤其在各个国际进口快递的国际航班运能大幅减少，口岸清关和整体时效链路拉长后，买手的资金结算回笼也受到严重影响，用户订单取消率比以前更高，业务端正持续下滑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内部来看，为了满足资本市场对于跨境业务的资金、税务等多方面的合规要求，洋码头在 5 月 1 号将平台的资金结算系统重新改造和切割，全部委托给有资质的支付机构，且全部在境外结算。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但平台现金流状况的进一步恶化，引发了外部供应商债券诉讼和银行抽贷的连锁反应，甚至有少数债权方通过国内法院要求冻结银行资金进行保全等，这让洋码头一直处于十分被动的状态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这些债务问题，曾碧波承诺，洋码头不会赖账，自己也不会随意跑路，哪怕是开直播带货也会把债还清，与罗永浩的 " 真还传 " 剧情类似。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为海淘先驱，洋码头也曾有过高光时刻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在创立洋码头之前，15 岁便考入上海交通大学少年班的曾碧波曾在当时国内最大的 C2C 电商易趣网任职，正是这段工作经历，让其意识到了海淘需要更灵活的采购和运输模式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2010 年，洋码头成立，并开始大举招聘海外买手，这些买手常年旅居在国外，是最早的个人商家，消费者可以通过买手在平台中的商店下单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了提高运输效率，曾碧波同时成立了一家国际物流运输企业——贝海国际，买手们在海外采购的商品，均通过贝海国际运输到国内，再由国内物流公司送至消费者手中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12403967476306b36719a537.61566154.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到 2014 年底，洋码头的全球物流中心布局基本完成，而这一年前后，阿里、京东、苏宁、唯品会才刚刚上线了跨境进口业务。也就是说，洋码头的速度快于很多巨头。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>" 买手制 + 自建物流体系 " 帮助洋码头迅速抢占了市场，收入也随之走高。2014 年销售额达到了 6.6 亿元，为2013年的2倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而阿里、京东们带着巨大流量和商家资源入局后，市场竞争格局随之发生变化，洋码头的发展也坎坷了许多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2016 年，本已敲定的 C 轮融资被搁置，洋码头自己的资金链也出现问题，甚至被迫在 2017 年进行了大规模裁员和业务收缩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾碧波曾承认，C 轮关键融资失败的主要原因在于，互联网巨头涌入跨境电商影响了机构的投资策略。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到了 2019 年，阿里豪掷 20 亿美元收购网易考拉，跨境电商格局也基本稳定，天猫国际、网易考拉和京东国际稳居第一梯队，手握近 70% 的市场份额，作为开创者之一的洋码头，则逐渐被边缘化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>洋码头不是没有尝试自救。2018 年，洋码头抓住风口，在站内开启海淘直播，甚至还将目光转向线下市场，计划通过 " 自营 + 加盟 " 模式开设 1000 家线下门店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>兼顾线上线下两个销售渠道的策略与淘宝、京东十分相似，但与财大气粗、自带流量优势的巨头们相比，洋码头明显有些心有余而力不足。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4967159286306b3674f1eb6.35357488.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，曾碧波去年曾向媒体确认，洋码头已启动拆红筹架构，加快国内上市布局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而从最新的公开信来看，成功上市前，洋码头还需要解决很多遗留问题，给予资本市场更多信心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，洋码头在消费端的口碑也有待进一步改善。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国内电商消费调解平台电诉宝的的数据显示，2018 年 -2020 年，在跨境电商的分类中，洋码头一直处于投诉排行榜的前五位，2020年甚至居于首位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>具体来看，大部分投诉都提到了假货、虚假宣传、商品质量不过关、商家不退货等问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾碧波透露，目前洋码头还有上百万的流量，有上千个买手在平台继续经营，呼吁平台买手召回粉丝，恢复直播和商品上架销售。</span></p>
