<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先说几个营销领域早已达成的共识：</span><span></span></p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户代际变化，消费习惯天翻地覆，圈层化趋势明显，圈层与圈层之间如关公战秦琼，差之千里。</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>注意力碎片化、粉尘化，甚至说对于一个品牌的记忆力像金鱼一样只有七秒也不为过。每天各种信息轮番轰炸，想要让消费者留下印象，难之又难；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>低买高卖的流量玩法早已走不通，营销走入精细化、全域化的深水区。</span></p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>道理好懂，知易行难，这些问题正在给许多行业带来更大的营销挑战，比如汽车：</span><span></span></p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>买车人的身份、需求、圈层消费观念都出现了巨大变化，年轻人、女性用户等新消费群逐步成为购车的主角之一；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“汽车消费者”既广又垂，有购车需求的用户遍布天南地北，但想找到他们又抓得住需求，需要“精确制导”；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>汽车的消费决策周期长而谨慎，从线上到线下，从流量到销量，单纯靠砸流量很难保证销量持续增长。</span></p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有什么办法，既能打破汽车的圈层禁锢，又能带来真实的效果？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/73583350462cc2166c32fc9.58679383.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源｜UNSPLASH</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年618期间，<a href="https://www.niaogebiji.com/tag/%E6%87%82%E8%BD%A6%E5%B8%9D" target="_blank">懂车帝</a>用“616全民购车节”给出了答案，这场“年中车晚”不仅汇聚了多路明星艺人、体育嘉宾、虚拟偶像、汽车和跨界达人，还联动了全国区域的汽车经销商，通过懂车帝的多平台宣发、上星卫视与多渠道扩散，实现了全方位的声量释放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更值得注意的是，这并非一场只“以销售为目的”的购车节，而是站在用户身边的一次“集中服务”——为用户提供一个实惠、便利、有用的买车机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>消费者被好内容、强兴趣吸引而来，因优质、靠谱、放心的服务而留。懂车帝在打造声量之际，用服务和产品去承接用户的信任与黏性，正是这场“全民购车节”所做的。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81142426261420d300e45a3.29058312.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>破圈</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今天购车的主力人群与购车需求已经发生了翻天覆地的变化，想要吸引到新时代的年轻消费者、女性消费者，营销漏斗的扩大与破圈必不可少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓「不破不立」，破圈带来的声量如何才能变成支撑品牌的销量？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>懂车帝首先做的是内容的多层次。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为整个活动的最大暴风眼，6月16日当天的“全民购车节 · 驭见D次元”直播晚会可谓阵容豪华，也聚集了整个活动中的最大人气——观看总人次突破了一亿次，有超过19万懂车帝用户参与了当晚直播互动，互动声量突破519万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/44832248862cc21670bc191.72509594.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>各个明星不只是用演出聚集人气，晚会设置了多个趣味互动，将艺人与汽车产品联动在一块，例如：“皓极随心唱”、“哈弗知识yes or no”“汽车pose大师”、“再就业男团卖车”最大化增强直播的互动感，为品牌出圈传播营造更多可能性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>晚会在舞台和内容的设置上也突出了破圈的属性——裸眼3D舞台、VR虚拟看车、虚拟偶像嘉宾表演等内容都是年轻人的心头好。根据灵眸2022中国汽车消费趋势洞察报告，当下34.1%的购车人群是24-30岁的年轻人，年轻、女性、二次元人群、上班族、科技爱好者等看似独立的圈层，其实也在与爱车、购车人群圈层交叠，内容共情的需求非常多元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上“操作”只是懂车帝用内容吸引用户，实现破圈的第一步，在收获大量“人气和关注”后，懂车帝的年中车晚开始放大招，留住真正的购车人群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先是整场晚会中，巧妙的把懂车帝App中的王牌功能，比如「3D看车」、「懂车帝实测」、「懂车分」、「懂车排行榜」等与晚会内容做了结合，植入到了嘉宾互动游戏中，放大产品的专业性和选购指导价值，潜移默化告诉观众“买车就上懂车帝”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/172914809362cc2167902343.62308403.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>3D看车现场效果</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次是晚会中KOL说车的环节，几位懂车帝头部汽车达人从用户买车时的不同需求出发，帮助观众记住车型亮点，通过选车窍门、产品功能实景展示等方式，增强用户对于当晚产品的认知与了解。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>「产品专业力+内容专业力」合二为一，因晚会内容吸引来的用户，通过4个小时的直播完成了对懂车帝App以及当晚汽车品牌的“吸引-兴趣-了解”三步种草之路，让破圈声量最终落到了产品与服务上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/316136962cc216806e106.00236783.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>A-soul表演现场</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>其次，让破圈不流于表面。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>破圈打开了营销漏斗的入口，如何让用户不白来一趟？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>懂车帝全民购车节正好卡在了年中购物季的关口。在全网消费情绪高涨的彼时，616全民购车节的破圈与落地难度其实是增加的——在众多消费品中，汽车显然是转化难度最高的那一类，此时此刻入圈，就需要给到消费者有价值的驱动力，并调动消费终端一起出力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>懂车帝首先联合中国汽车流通协会共同发起了“经销商直播挑战赛”系列活动，调动全国的汽车销售终端在购车节期间提供了大量的内容支撑。与此同时，懂车帝联合品牌车企，以“平台补贴+品牌优惠”的组合形式，向购车消费者发放购车福利，用户领券之后可以到所在区域的经销商实体看车。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/88305120262cc2168653567.95387794.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>616的晚会直播前，懂车帝在长沙核心商区打造了“Dcar Park”汽车主题公园，6档原创头部IP节目亮相Dcar park汽车文化公园，非常快改装车“名爵”也在线下展出。汽车公园中设置了4个主题街区，分别从“选车、看车、买车、用车”四个方面，向线下用户和观众介绍了懂车帝的定制购车服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在当晚的616全民购车节晚会期间，东风风神、哈弗汽车品牌的负责人也在现场登台，发放万元秒杀券等当晚直播间限定的品牌福利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>懂车帝通过“线上全民购车节+多地线下深度体验+科技赋能破次元购车晚会”，打通了线上线下汽车消费场景， 并为购车消费者发放千万元级别购车优惠补贴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>破圈打开了营销漏斗的入口，通过后续的内容/服务/产品跟进、线上线下打通，快速地收拢流量、聚焦流量，在纷繁复杂的声量场中找到消费者、走近消费者、拥抱消费者的全过程——这才是懂车帝破圈的关键。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/187731252062cc2169760dd3.57768954.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213179696861420d317ab820.36069667.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>见效</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>汽车行业可谓久旱逢甘霖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>疫情影响了北京、上海车展，打乱了汽车品牌的新车发布与营销节奏。汽车选购周期长，加上缺芯少电带来的交付问题，营销窗口期就更加难得，更需要一击制胜搞定品牌的声量和销量。在这样的大背景下，</span><span><strong>破圈之后的效果就成为了核心焦点。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>办晚会不是目的，上综艺不是目的，破圈不是目的，</span><span><strong>制造巨大声量的背后，是懂车帝想要在用户心中树立起长期提供优质买车服务的心智印象，也是行业望眼欲穿，想迫切达成的“见效转化”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>汽车消费的最大特征就是有一套完整的消费旅程，根据懂车帝的5A用户流转理论，一次汽车的购买行为要经历从感知、好奇、询问到行动和拥护五个阶段，周期可以长达1-3个月。如何才能在这趟“转化旅程”中与消费者同行？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>懂车帝拿出了内容+产品的全效联动策略。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先在内容上，懂车帝的众多王牌PGC节目组成内容矩阵，通过专业内容帮助品牌建立产品力的用户认知，当中既包含了汽车产品、配置等产品力分析，也包括了贴合用户实际购车需求的选买建议和用车知识。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在产品上，616全民购车节期间懂车帝上线了品牌专属线上XR展厅和定制竞速赛车游戏。XR展厅的3D看车场景有更强的视觉冲击力，互动玩法也让用户更有分享的兴趣。定制赛车游戏则在引导用户挑战更高分数的同时，把产品的新卖点与游戏融合，让用户在游戏中记住零百提速性能、操控感强等产品认知。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/105480905162cc2169ade229.01709000.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>基于5A的智能产品</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回到那个核心的话题，616全民购车节为什么能从破圈落地到真实的转化？一放一收之间还是亘古不变的真理——给消费者带来真正的实惠。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是内容的铺排，还是懂车帝产品的设计，都站在消费者的角度，为消费者提供实用的选车、购车服务。购车节的种种设计也充分地考虑到了汽车不同于快销品的消费决策周期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>懂车帝这次发补贴的逻辑目标就是让用户买到实惠：向消费者发放购车券补贴，消费者可以先领券且无需预付款，然后再到实体店看车，拿到最优价格后，将购车发票上传到懂车帝平台兑现曾经领取的优惠即可。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就是说，用户先在懂车帝领取福利和补贴，此后横跨2个月的时长让消费者有充足的看车、选车、思考时间。而补贴的设计避免了平台发优惠、经销商却不优惠甚至涨价的“套路”，把福利落实下来。这同时也保护了直营、代理类品牌全国统一的价格策略，避免了经销商的利益受到伤害。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进一步看，其实懂车帝全年的购车节活动的策略思路也都是从用户角度出发，对应不同热点及季节，策划不同主题的购车节以满足用户需求：1-2月抓住春节这个关键节点，全域联动经销商；3-4月借势女神节/春季的热点，深挖女性车主诉求；5-6月则整合平台的能力以“全民购车节 · 驭见D次元”这一IP进行突破；7-8月依托直播核心优势赛道，打造平台夏季直播购车项目“88直播购——约惠一夏”；9-10月是汽车促销节点，则会联动车展等行业节点；在年终双11、双12的消费情绪下，购车狂欢也强力开启。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再复杂的营销玩法最终都要回归品牌、回归消费者，从用户产品到商业产品，再到最后的运营举措，懂车帝抓住了“见效”的3个关键：让用户体验升级买到优惠，让<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E8%90%A5%E9%94%80" target="_blank">品牌营销</a>科学提效，让经销商获得高质量真效线索助力成交。</span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>结语</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>汽车品牌与经销商面临从生产到营销、销售的各种压力；用户消费者则身处于一个充满不确定性、充满信息噪音的乌卡时代。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为连接多方的平台，生态的枢纽，懂车帝也不缺挑战，大到平台调性、小到内容风格，正因为诸多变局，各方对平台营销的要求也正越来越高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>616全民购车节，是一次大胆的尝试，也开启了一种汽车营销的新逻辑——充分蓄水破圈、内容+产品种草，理解消费者，以用户的理智决策、买到适合自己的好车为导向完成转化。这或许才是共克时艰的路标，回归用户消费者、回归产品与服务、回归营销的底层逻辑。</span></p>
