<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从整理的月销过万的品牌累计来看，在整个抖音宠物行业市场中全年总量预估在<strong>25亿</strong>左右，目前头部品牌top10占到31.2%，top20占到43.2%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前位于top1的麦富迪占到市场的8%左右，top2的 NOURSE/卫仕占到4%左右，算是比较头部了，其他品牌的占的比例都不高，还<strong>没有形成绝对的寡头</strong>以及市场格局，<strong>基本上新品牌都还是</strong><strong>有进场以及发展的机会</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时新品牌想要进军宠物市场可以多考虑自身的细分市场，寻找优势和独特的切入点，选择错位竞争的方式，维护好自身品牌的可持续发展。</p>

<p><strong>年龄性别</strong></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3874696956316928684cb88.78635672.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>数据来源：巨量算数 “宠物”关键词的人群画像</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">关联宠物的用户人群在性别分布上相对平均，说明在宠物用品行业对于性别的细分要求不高，在年龄分布上，主要饲养宠物的用户人群为31-40岁，占比大约为32%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其中18-23岁是抖音上对宠物内容互动关注度最高的人群，现阶段是以年龄阶段在30-40岁为主要的消费用户，随着内容的不断渗透，18-23岁这个年龄群体会逐渐成为未来宠物市场的消费主力军，品牌也应随着消费人群喜好变化改变自身产品的营销手段和卖点。</p>
<section>
<p><strong>地域特征</strong></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6456586663169286c3db99.36516646.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>数据来源：巨量算数（ “宠物”关键词的地域特征）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为养宠物的普及化，在地域上并有多大的差别，主要购买地域在新一线及二三四线城市，占比超过70%以上，三线城市的占比接近24%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但从19年的宠物市场报告来看，宠物行业的主要消费市场还是一二线城市，这个群体消费实力强，观念也比较先进，进而搜音渠道上的新奇特产品，大都消费过，其次这些养宠成熟地域的线下宠物商店、宠物医院等机构较为常见，这些渠道分流也是使得线上的内容消费力低于其他的三线城市。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17253787276316928746b619.14779535.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>数据来源：蝉妈妈 （2月-7月）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">零食、猫砂、主食、梳子等产品在抖音宠物市场占据较大的销售优势，特别是<strong>客单10元左右的零食产品</strong>，用户非常愿意下单尝鲜，给到家里的毛孩子奖励。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同样零食又是<strong>日</strong>常高消耗的宠物必需品，所以众多品牌也偏向于在宠物市场推出零食、猫砂作为能让品牌自身续航的爆品，其次在选择产品上更多也是偏向于猫咪用品，纯狗狗用品的占比只能占到20%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在销量方面，客单价低的，特别在0-1元的这个区间，是卖的最多的，也是大多数品牌作为吸引和留住用户的营销手段。其次是1元-30元的产品核心销售价格带，<strong>超过30元基本就比较难了</strong>。</p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>数据来源：蝉妈妈 （2月-7月）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1-30元价格区间客单价低的产品，有利于达人的推广和转化</strong>，数据上看也可以看出该价格区间的产品达人带货销售额最高。同时，在30%左右的佣金主播更愿意推。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在推广的商品中，带品牌的产品占比更高，其次我们统计了品牌产品佣金和非品牌产品平均佣金百分比，观察到品牌的佣金更低，<strong>不难看出品牌产品在达人推广上占据一定的优势，营销费用可以更多选择花在品牌力塑造上。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在达人推广较多的产品总结里，我们发现清洁用品和零食是他们偏向去带货的类型，这类产品背书要求较低，且是高消耗品，达人推广压力小。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">直播卖的多的产品特征</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70044028163169287f0ce61.19261522.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/259508396631692882126f4.85145391.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/128703321631692884167d3.04619621.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5439315436316928882ad91.08547487.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>数据来源：蝉妈妈 （2月-7月）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从覆盖的品牌来看，皆是认知度高，品牌力高的产品<strong>。</strong>其中麦富迪狗粮每月都占据销售额top1，且大部分产品都来自于宠物用品类目中的品牌top10，说明品牌的<strong>声量影响着消费者的认知度近一步影响着消费者的选择</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时，我们可以看到一些标题上写的“明星代言”、“主播敢吃”等关键词有目的性的吸引用户的购买兴趣以及提升购买用户的信赖感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外<strong>直播产品的客单是较其他榜单高</strong>的，如上图3所示，直播销售top的核心产品件单是<strong>集中在80-100以及400-500这两个区间的</strong>，直播渠道特别是品牌的自播是很能卖的上价格的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再到直播推广的核心产品，皆是以主食，罐头，猫砂，尿片，高消耗的产品为主，其中罐头占比这么高，有点出乎意料，这个点也侧面说明了用户愿意为宠物投入的花费在增长，随着对宠物的关注度越来越高，对应<strong>零食和保健品也可能成为接下来核心消耗品。</strong></p>

<section>
<p><strong>视频卖的多的产品特征</strong></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/79090227363169288c235a6.46236713.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/176579017063169288e7a636.72350083.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>数据来源：蝉妈妈 （2月-7月）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">视频方向跑出来的产品，<strong>以性价比高为主要的吸引手段</strong>，产品主要集中在猫砂，零食，梳子类，这类产品特点是容易在短视频里进行拍摄和展示。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总的来说，<strong>客单价低的宠物用品更占据销售优势</strong>，不管是达人推广，还是短视频推广，都更容易出效果。客单价高的的产品主要以直播的方式销售，大都是以品牌自有直播间售卖为主，<strong>通过付费流量筛选精准用户进入直播间成交</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其次整个市场上，宠物用品的较高比例是围绕猫咪用品展开，猫类用品能占到宠物类市场的60%以上，<strong>猫咪晋升为下一个碎钞机，可以围绕猫咪展开更多元的产品延伸。</strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6881324926316928a2228d7.60035505.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在top10里只有麦富迪和cocoyo的带货渠道里达人播的占比要大于自播，其他的品牌都是以品牌自播作为主要的带货渠道，不难看出，<strong>宠物品牌想要在抖音跑出来，更多还是要靠品牌自身，借助达人来推，较难</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们上文提过，宠物用品类目在抖音的市场体量一年大约在25亿左右，刚才宠物用品行业里自播是主要销售渠道。<strong>品牌想要打通抖音渠道，自播是一定要作为主运营的。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其次我们观察到大部分的宠物达人输出短视频的内容是<strong>以娱乐类型为主，带货的效能不高</strong>，所以也导致了宠物用品行业的带货达人较为缺少，想通过达人助推来打爆产品较难。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首先我们可以先简要看一下，宠物用品行业达人的基本情况：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20492493076316928a4325f9.11580505.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>数据来源：蝉妈妈 （近一个月各个类目粉丝量>100w的达人数量）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们统计达100万以上粉丝量级的头部达人数<strong>只有148位</strong>，相较于其他品类的头部达人数来说是<strong>偏少的</strong>。与最多的<strong>食品类目相比，只占其的5%左右</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16796529416316928a6e9e76.74595194.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>数据来源：蝉妈妈</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这是近一个月在宠物用品类目场均销售额排行的Top直播达人，我们可以观察到这些达播基本上都是自己有在饲养宠物的，在宠物用品上一般用户会比较<strong>注重产品的安全性以及饲养宠物的共鸣</strong>，自身有在饲养宠物的达人可以拉近与用户之间的距离，进一步<strong>提升对产品的信赖度</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于品牌来说<strong>BD娱乐类型的达人是没有背书意义的</strong>，因为无法给予饲养员信任感，这也是为什么宠物达人很多，但是能给宠物用品进行带货的达人却比较少的主要原因之一。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1011411046316928aae17d5.51282982.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>数据来源：蝉妈妈 （7/12-8/10）</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><strong><span>（粉丝量级在100万-1000万的萌宠类直播达人近一个月根据销售额排行的top8）</span></strong></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外宠物产品的<strong>特殊性</strong>：<strong>使用者与购买者无法沟通</strong>，所以在产品的购买和决策上，对于品牌方而言就需要两套语言体系，而情感上，现在<strong>宠物是可以类比为婴幼儿</strong>，那对于婴幼儿产品的选择和思考就会更加的慎重，这也是导致达播量级不大的原因。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外一个导致达播量级不大的原因，是由于宠物用品人群的特殊性，线下购买渠道，如宠物店和医院等导致销售人群市场分流，所以线上体量就会比较低。</p>

<p><strong>柿子菌meow</strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6838319266316928ada2ef6.78136332.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">该账号属于宠物类目带货头部达人，近半年以来，直播间销售额破千万，场均UV值突破5，达人转化率是比较好的。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">柿子菌直播间主要售卖猫咪宠物用品，近半年销售额最高的产品都关于猫咪健康方面，说明在这类头部主播与购买用户已经形成了良好的信赖关系，当饲养员想购买一些宠物保健品，他们往往会选择这类的头部达人所推荐的商品。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从带货数量上我们可以观察到，<strong>该达人主要带货市场在直播领域</strong>，短视频种草方式只占据整体的2%左右，同时，带货品类里宠物用品占据了整体带货品类的94%以上。</p>

<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span>宠物行业总结</span></strong></h1>
<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1. 抖音宠物市场还没有形成绝对的寡头以及市场格局，大部分的新品牌有机会进场和发展。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2. 品牌集中打造的大部分宠物爆品的特点是在饲养宠物的过程中高消耗、高转化、高复购，且众多品牌在选择产品上更多的偏向于猫咪用品。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3. 自播是宠物市场的都要带货方式，品牌进入市场后要以自播作为主要切入赛道的途径。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4. 达人推广占据一定的优势，品牌的营销费用可以更多选择花在品牌力塑造上。</p>
