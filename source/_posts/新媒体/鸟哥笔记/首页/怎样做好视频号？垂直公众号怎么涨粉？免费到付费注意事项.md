<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>为什么分享这个？昨天下午直播了一个小时分享了，因为语速偏快，再加上有很多人没看到，所以写出来下。希望对没看过有些帮助。</span></span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>怎样做好<a href="https://www.niaogebiji.com/tag/%E8%A7%86%E9%A2%91%E5%8F%B7" target="_blank">视频号</a>？</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>白杨SEO的视频号【白杨seo】最开始注册于2020年2月，已经运营两年半了。开始计划每周三直播，去年2月开始，也连续坚持一年半了。</span><span>前两天，在白杨流量汇里阿愚分享了一个他的视频号视频，建议大家都可以做起来，确实如此。两年前的视频，大家可以看看或者关注。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频号的本质就是帮助创作者解决连接和信任的问题，打破了原先微信生态的5000、10000人的熟人限定。因为一个点赞，就有可能让你朋友的朋友看到你的视频。而信任，你懂的，视频号视频、直播，当然比图文更直接了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>怎样做好视频号呢？白杨SEO这边总结四点小经验：</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1、视频号定位一定要精准。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们做视频号定位的时候，一定要确定到底视频以人为主，还是以垂直的内容为主。前者可以打造个人IP形式，后者主要做垂直的短内容，持续下来，都能获得相应的精准粉丝。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2、视频号一定要以内容为主，运营手段为辅。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多做这个本末倒置了，把运营，搞各种数据为主了，忽略了要想做好视频号，内容才是最本质的东西。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3、做视频号要有数据思维，数据分析能力。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做视频号一定要关注短视频的数据，每个视频发出去后，一定要关注短视频的播放量、点赞量、评论量、收藏和转发量。而且还要去关注赞播比，如果<span>赞播比高，那么这个内容说明是用户想看的。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>4、一定要花时间做个爆款视频。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们都知道，一个爆款视频带来的收益可能是10个，甚至更多一般视频比不了的，所以我们要花时间去做爆款视频。怎么做呢，从文案，拍摄，剪辑，配乐，发布都要一一认真去做，同时还要去做一些相关的启动赞，完播率等等。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>垂直公众号怎么涨粉？</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于公众号怎么涨粉，其实早在三年前我就写了三类6种涨粉方法，看这：</span>白杨SEO：2019最新微信公众号快速涨/增粉的3种运营方法【收藏】<span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里呢，再新增几种方法，看看你用过几种？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1、资料互推涨粉。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家应该见过太多的公众号互推了吧，一般大号带小号，或者同行互相推荐都用这种方式。然后在公众号推荐末尾，这些公众号都会留下类似“回复XXX关键词，免费领XXX份干货资料之类”。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2、付费投放涨粉。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>投放分两种，一种是一对一投放，也有就是抱团多对一投放。举例，比如，你觉得哪个公众号不错，你可以花钱去他公众号投公众号或者软文涨粉。如果你觉得自己预算不够，就拉起10个人，每人出三到五百一起去投放，或者做所谓的IPAD之类的抽奖。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，据了解，这种现在应该不那么好做了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3、PC网站特意涨粉。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个也分两种引导。一种是本身利用自己网站用户多的优势，可以引导用户关注公众号之类登录，比如CSDN。另一种就是利用内容，写一半留一半，剩下一半的内容，到XXX公众号发送“XXX关键词”看全文。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>4、视频号引导涨粉。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个是当下最火的一个方式，不过现在新注册的视频号没有直接绑公众号入口了。据之前一个玩的溜的，做了类似跳舞视频号视频，最高播放近亿，给公众号引去数十万粉丝。所以很多人就发现了这个商机哈哈哈。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在视频号视频还是能放公众号文章链接，但是不是在视频下方，变现了评论里样式。</span><span>当然，你还可以在做视频号直播的时候引导关注公众号。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>从免费到付费要注意哪些？</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上面我们写了视频号怎么做好以及公众号怎么涨粉，你可能就问了，我视频号、公众号做好了有粉了怎么变现呢？免费到付费注意哪些？白杨SEO这边总结了五点。<strong><br></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1、不要着急，多去看看，要不耻于谈钱。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你做到一定粉丝以后，一定不要坐井观天，一定要多出去走走，多出去看看，别人是在怎么变现，可以加入相对应社群。比如你做SEO，也可以考虑白杨SEO付费交流群，对吧，这里有近500人在交流。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有最重要就是，很多人免费到付费过不去心理的坎，觉得收费不好意思，不好谈钱。所以我们要不耻于谈钱，因为你的时间就是值钱，而且你的内容提供了价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2、变现时候不一定取决于粉丝数量，几百、几千粉丝也能变现，取决于你是不是有相应产品或服务。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人以为，我的视频号还没有1000粉丝，我公众号没有5000粉丝，这个肯定做不了付费，其实，核心是你没有相应的产品或服务。这个产品不一定是实物产品啊，比如你分享法律或者健身有一定粉丝了，你可以出一些用户痛点手册啊，定价99说不定还是有人买的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再说，我们第一点说了，你要不耻于谈钱，所以把你的服务标出来，比如要咨询多少钱之类。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3、准备对接粉丝的微信，增强信任，提高转化。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了更好的转化，建议你准备1-3个专门的对接粉丝的微信，当然这个微信你也不要天天发营销啊。你的核心是拉近潜在粉丝的心理距离，让他感觉很真实。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>白杨SEO的做法，就两个字“真诚”即可。这样可以增加信任对吧，然后你也可以每周或者每三天搞一次直播，不要在意人多少，在意是否帮到了用户，赚钱，为你付费，就是自然而然的事了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>4、带货或者接广告一定要慎重，把控内容质量。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说到这个，先说一个真事，就是前几天看一个公众号，作者内容写得很不错，接了一个广告，可能因为她没有看广告开始是第一人称我，很多读者误以为是她，然后呢她觉得名誉受损，后面改了声明，广告主不同意，互相不理解。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，这个事，双方均有不对，作为公众号作者，内容把控确实要更细心一些。而投广告主呢，为了转化肯定会尽量模仿公众号作者写的，甚至还用到了小心机。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，要变现很正常，毕竟创作内容花时间花精力。不赚钱，难道用爱发电？核心还是一定要把控好内容质量，如果不符合要求，一律不接。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>5、要有持续获取精准用户的核心竞争力。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人忽略了第五点，从免费做到付费，你需要就是要有持续不断获取精准用户的核心竞争力。比如有的人是靠不断花钱买粉，看ROI转化比，以前最常见的就是各种打卡训练营之类。</span></p>
