<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你见过野生“rapper”在直播间卖咖啡吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>走进直播间，没有321上链接，没有一口一句家人们，只见主播与画外音一唱一和，身体跟着节奏摇摆，似乎要为咖啡注入嘻哈灵魂——</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>精致咖啡的文化，融入日常生活化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>咖啡品质您放心，醇香浓郁又清新。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>咖啡每日来一杯，时时刻刻欢乐陪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一杯情啊两杯意，三杯四杯好兄弟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>牛奶加到咖啡里，好喝到你心窝里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>由于嘴巴极具个人特色，这位主播被直播间观众称之为“大嘴”，</span></strong><span>在</span><span></span><span>介绍福利咖啡券时，大嘴会半眯着眼睛，做出标志性动作“冲冲冲”——</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1793209066632d5012967ba4.86355171.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>咖啡你冲不冲，冲冲冲冲冲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>福利你冲不冲，冲冲冲冲冲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>单押、双押不在话下，freestyle也少不了。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说着说着，大嘴还真的唱了起来，“苍茫的天涯是我的爱，咖啡在链接里面等你来”。</span><span>有观众提问：“你这背景太假了，是录播吗”，大嘴也没停下来，指着桌上的</span><span>时钟，直接答道：</span><span>“不是录播～不要胡说～”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>你以为这是新说唱的节目现场，其实这是“T97咖啡旗舰店”（简称T97）抖音直播间的带货现场。</span></strong><span>T97创立于2021年4月，目前已拥有43家线下咖啡门店，</span><span>其创始人曾公开喊话“未来一定超越瑞幸”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在T97抖音直播间内，主</span><span>要销售的是线下门店咖啡的电子兑换券。</span><span>一场直播通常会持续7~8小时，不断循环rap喊麦、数来宝、顺口溜等内容，配上大嘴标志性的“烈焰红唇”和双马尾造型，让不少观众直呼“魔性”“上头”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新榜编辑部观察，T97在5月19日开启首场抖音直播，起初数据平平，9月采用说唱带货形式后，人气高涨，在线人数峰值最高达2.8万，目前粉丝超21万。<strong>新抖数据显示，近15天内，单场直播最高观看人次达282.77万，曾连续一周场观突破200万。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a data-miniprogram-appid="wx55778a0d7b3048c4" data-miniprogram-path="pages/NewLive/index" data-miniprogram-nickname="新抖数据" data-miniprogram-type="image" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined"><img src="https://ngbjimg.xy599.com/1174038438632d501eb5b4e0.30512981.png" referrerpolicy="no-referrer"></a></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源<a data-miniprogram-appid="wx55778a0d7b3048c4" data-miniprogram-path="pages/NewLive/index" data-miniprogram-nickname="新抖数据" data-miniprogram-type="text" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined">新抖</a>，新榜旗下抖音数据平台</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着T97走红，类似的说唱形式也席卷了带货直播间，尤其集中于本地生活领域，这是否会成为一种直播运营新趋势？</span><span>我们对话了T97主播大嘴与背后DP毛豆传媒负责人子谦，一起聊了聊T97直播间如何出圈，以及直播间之外的故事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>200万人涌入直播间，不买咖啡，只听说唱</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>99年出生的主播大嘴原名李文霖，是T97直播间的当家花旦。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2021年大学毕业的她，成为带货主播不过一年多时间。在这之前，她曾在意尔康担任摄影助理，擦过样品皮鞋，也临时顶替做过模特，</span></strong><span>后来因为能力被公司认可，她被调去意尔康的杭州直播基地做主播。也就是在那里，大嘴认识了当时负责直播间运营的子谦，并成了他的徒弟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年初，子谦开始自己创业建立毛豆传媒，她出于信任和感恩，愿意离职降薪加入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然大嘴不是专业rapper，也没做过DJ或脱口秀演员，但凭借之前的直播经验，今年8月底，她与运营小哥对词排练了2~3天就上场说唱了，然后在一次次直播中练习，逐渐得心应手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>起初由运营写词，后期大嘴也加入创作，针对每个产品编写40多字的顺口溜。</span></strong><span>她会先了解产品的配方和制作方法，从宣传文案中摘取一些关键词，再编成比较押韵的一段话，跟rap相似，押最后一个字或两个字的音节。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如写草莓拿铁：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天若有情天亦老，草莓拿铁是冰的好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进口草莓来尝鲜，冷处理的很新鲜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>酸酸甜甜到嘴边，爱情来了你们先。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>据新榜编辑部统计，配合T97直播间循环播放的BGM“Heisenberg”，大嘴与其他主播踩着节奏对唱，10分钟能唱2000多字。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然大嘴对节奏早已滚瓜烂熟，</span><span></span><span>但由于每天都要更换新词，所以</span><span>直播时镜头旁设置了提词器，偶尔忘词的时候可以看一眼滚动的文稿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我觉得词和唱得专不专业都是其次的，表现力最重要，观众喜欢这种非常魔性的感觉，想看到很搞怪、很有趣的表现。”大嘴说道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>除了节奏欢快的说唱，大嘴本身的造型也突出了搞怪的风格。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原本大嘴拍照时会把嘴巴P小，直播前也要把缩嘴的美颜功能开到最大，涂上浅色口红，但即使这样做，还是有观众说她嘴大，像姚晨。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>后来因为一次意外，换她上场直播时忘记开缩嘴美颜，在线人数竟一下子从7人增加到50多人，观众纷纷调侃她嘴好大，“要吃人了”。</span></strong><span>从此她再也不缩嘴了，就这样保留自己的特色，也没有刻意去加大嘴特效。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1712282522632d501eedf184.53098310.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>开了缩嘴与没开的对比</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>发型方面，大嘴一开始是披发，看起来更贴合咖啡品牌的小清新调性，用说唱带货后试了一次俏皮的双马尾，用户停留时间也变久了，在之后的直播中也保持了这个发型。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了丰富直播画面，大嘴戴了一个青蛙发箍，没想到又被说像悲伤蛙，她感慨道“我好像全身都是槽点”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99986328632d501f269e12.45668691.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span> 披发与双马尾的对比</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了大嘴之外，T97直播间还有其他两位常驻主播“眼镜妹”和“夹子音”，三人轮换，每人直播两到三个小时。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>子谦表示这对于团队配合要求很高，副唱的语气会影响主唱的情绪，彼此间的默契度越高，衔接越好，直播效果就越出色。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/351433220632d501f6cbc42.06150210.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>左““眼镜妹”，右“夹子音”，都是以人物特点命名</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>喜欢T97直播风格的观众觉得热闹又快乐，沉迷其中，而反感的观众在公屏或私信攻击大嘴的容貌和说唱方式，说她哗众取宠。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“看到你我都不想喝T97咖啡了”，这种评论让大嘴感到很难受，每周总有一次受不住压力而崩溃大哭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对恶评，大嘴在个人抖音账号“T97李大嘴-毛豆传媒”发了一条视频回顾自己“从普通穷女孩到金牌主播”的历程，她想要大家认识自己，其实她“不是一个小丑”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/898411393632d501fa0e662.73957584.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大嘴说，自己的家庭很普通，来杭州做主播就是想快点挣到钱，回老家宁波买一套属于自己的房子，也让父母的生活变得更好，“以前妈妈舍不得吃猕猴桃，现在她想要的我都想给她买最好的”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想通了之后，她不再介意被叫大嘴，也不去在意恶评，只想抓住机会好好干，她也给自己写了一段大嘴的唱词：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>山外青山楼外楼，主播嘴大才温柔。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>世上美女千千万，主播嘴大才能干。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主播嘴巴大不大，觉得大你敲个大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大嘴，敲起来，福利福利我送起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>复制+创新，T97直播间成喊麦带货届顶流</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着大嘴受到的关注越来高，抖音之外的其他社交平台也出现了她的身影。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一篇主题为《卖咖啡的主播困了》的图文在小红书获得了2.2万点赞，其中就是大嘴在直播间不自觉打呵欠的截图，不少用户也在评论区认出了大嘴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2091834222632d501fd79dc9.30291807.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>子谦意识到，可以持续强化大嘴本身的特质来吸引流量，比如建立大嘴的小红书账号，安排素人KOC宣传直播间等。</span></strong><span>据他观察，相比其他社交平台，小红书的讨论热度可以明显影响抖音，也成为站外营销的重心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，大嘴对于直播人气有明显带动。子谦给出了一组数据：“下午3点半刚开播，和晚高峰八九点这两个时间段都是大嘴上场扛流量的，在线人数相比其他主播会有20%-30%的增长。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>除了大嘴极具特点的直播风格，T97直播间的走红更离不开对内容形式的快速迭代。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开播初期，T97直播间采用的是常规形式，由主播依次介绍产品，观看数据一直不温不火。直到他们注意到“楠一青瓷”和“咸鱼要翻身”的成功案例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年6月，“楠一青瓷”将直播中的瓷器介绍词改编为rap似的喊麦，详细讲述每款瓷器的卖点和特点，比如介绍一款骏马摆件的时候：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>春风得意马蹄疾</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>马踏银河向天歌</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>鞍不离背震山河</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1656189292632d502024ed22.71667622.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新抖数据，开播以来，“楠一青瓷”直播间单场最高观看人次达20.52万，在线人数峰值为2.23万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a data-miniprogram-appid="wx55778a0d7b3048c4" data-miniprogram-path="pages/NewLive/index" data-miniprogram-nickname="新抖数据" data-miniprogram-type="image" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined"><img src="https://ngbjimg.xy599.com/90260880632d50207ba9a0.36251661.png" referrerpolicy="no-referrer"></a></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然有了人气，但由于瓷器商品普遍单价超过100元，存在消费门槛，“楠一青瓷”直播间成交额并不算理想。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>截至目前，单场最高纪录不超过2.5万元，其中销售额最高的是一个单价69.9元的顺口溜合集，销售额近1.5万元，剩余才是瓷器类产品。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/694590135632d5020be4ad4.15087903.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>类似图示，非“楠一青瓷”的原款链接</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>或许是“楠一青瓷”的直播间口才表演剑走偏锋，此后陆续有带货直播间仿照该形式，将原来的瓷器介绍话术改编为直播间相关产品，“咸鱼要翻身”就是其中一个。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月10日，“咸鱼要翻身”发布了一条直播切片，视频中她也采用rap喊麦方式，评论区有用户认为这是在模仿“楠一青瓷”。只不过，“咸鱼要翻身”主要销售提供本地生活服务的多家品牌代金券，单价从10元-40元不等，消费门槛低于瓷器。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新抖数据，开播至今，“咸鱼要翻身”单场直播最高销售额在25万元-50万元之间，</span><span>观看人次4.62万。</span><span>相比“楠一青瓷”，“咸鱼要翻身”直播间在销售方面更有优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/592710716632d502123d841.15651845.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“楠一青瓷”和“咸鱼要翻身”的出现，让子谦看到了用rap形式喊麦带货的机会。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了快速跟进，子谦首先将“咸鱼要翻身”的直播话术全部复制套用，投入试播就刷新了过往纪录。<strong>新抖数据显示，9月10日，T97抖音直播间人气峰值首次突破2万，此时账号粉丝仅2.73万。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这个基础上，子谦与团队对直播话术进行改编迭代，先是动员内部同事编写，加入咖啡介绍、时下热点等元素，每天更新两到三段，后来专门找外部话术师来写词，逐渐形成了T97现如今的风格。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新抖数据，近7天来，T97抖音直播间的单场观看人次最高达到266.83万，场均在线人数峰值达2.43万，销售额也从9月8日的不足5万元，</span><span>增长至最高的7.5万元-10万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a data-miniprogram-appid="wx55778a0d7b3048c4" data-miniprogram-path="pages/NewLive/index" data-miniprogram-nickname="新抖数据" data-miniprogram-type="image" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined"><img src="https://ngbjimg.xy599.com/1206296419632d50216bd0f7.74787464.png" referrerpolicy="no-referrer"></a></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过统计近期的销售数据，子谦发现T97直播期间，每家线下门店成交额平均多出1000元，基本都是来自T97直播间销售的核销券。<strong>同时，后台咨询T97的加盟量也从单日150个，翻倍增长至300-400个。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据了解，截至目前T97已经入驻了28个城市，拥有43家线下门店，均入围抖音当地城市咖啡厅收藏榜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1890226817632d5021a72326.73199914.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>考虑T97暂未入驻全国所有城市，部分观众无法下单使用，子谦尝试在直播间上架了其他品牌的代金券，最近还推出了“认养一头牛”的专场直播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>红过10天，真正的考验才刚开始</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>T97的走红，一定程度上掀起了带货直播间的喊麦之风，其中以本地生活、食品类商家为主。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/973314111632d5021f25ee1.41161887.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>卖牛肉酱、烧饼、鸭脖的抖音直播间</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>子谦认为，rap喊麦形式适合运用于单价不高、商品具有大众认知度的直播间，消费者不需要考虑太多因素就可以下单购买，本地生活的各类商家代金券就是最佳对标品，并且目前抖音官方正在扶持本地生活，给予商家流量激励。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据了解，抖音生活服务于2021年1月正式启动，呈现出持续增长的发展趋势。在9月20日举办的2022抖音创作者大会上，抖音生活服务营销中心负责人王丁虓介绍到，今年上半年对比去年同期，抖音生活服务的交易总额增长了30余倍、动销商户数增长近22倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有服务商对“未来消费APP”表示，去年上半年刚开始做代运营业务时，达人探店短视频的形式居多，但今年抖音的团购业务更重视直播对GMV的贡献率，并大力鼓励商家自播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过，这种批发复制的直播形式也频频引发争议，包括T97在内的直播间评论区，经常出现关于模仿抄袭的争论。对于业务出身的子谦而言，争论谁模仿谁原创并没有太过重要，关键在于如何快速跟进，达到想要的效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/352775543632d5022392f73.35297051.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“咸鱼要翻身”直播间留言区</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新抖数据，从9月10日-9月18日，T97直播间连续8天场观超过200万，但在9月19日场观却下滑至97.18万，9月20日场观才恢复至282万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>子谦表示，抖音流量来去匆匆，像近期在线人数峰值达1万-2万的红利期最多还能持续半个月左右。</span></strong><span>这对于一个诞生一年半的本地生活品牌而言，已经是出圈的成绩，“假如以后不火了，日常期间的在线人数应该也会在1000-2000之间，数字虽不大，但成交额会很稳定”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，子谦准备继续创新直播形式，深耕本地生活服务，拓展多个直播间，而这也是潮退之后，真正的考验。</span></p>
