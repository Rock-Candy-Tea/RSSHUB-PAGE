                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">​今日凌晨，Apple Ads再次推出新功能：Apple Ads Advanced 后台已支持一次性编辑和创建多个广告系列和广告组。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/8c179fee252c5ac9eba7535fb29fe258.png" width="942" height="1158" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">新功能旨在帮助营销人员使用新模板下载现有的广告系列和广告组设置，然后编辑每日预算、出价、受众细分等各项内容。上传更新的模版后，系统会立即应用相关设置。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时，营销人员还可以下载空白的广告系列设置模板，以创建全新的广告系列和广告组。借助此功能，广告主们可快速为新 App 的发布或新国家/地区的拓展设置多个广告系列。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">下载和编辑现有的广告系列和广告组设置</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在“广告系列”信息中心里选中特定广告系列旁边的复选框，即可下载并更新相应广告系列的设置。或者略过全部复选框，直接更新所有广告系列。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，借助“搜索”框查找或使用“筛选条件”菜单中的一般条款、效果和 Advanced 选项，亦可筛选并更新特定广告系列。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从“操作”菜单中选择“下载数据”。选择“广告系列和广告组设置”选项，以 XLSX 文件格式下载设置数据。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/710244d1d3e14276f43fd8b4b8a0333d.png" alt width="1080" height="393" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/68310dc294a1c38c7ba636380151daca.png" alt width="771" height="370" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">营销人员还可以下载地理位置细分选项列表，查看可用于特定国家/地区、州/行政区、城市/区域的缩写。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">点击“下载”按钮。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">下载空白模板来创建全新广告系列</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果营销人员尚未创建任何广告系列，或者想要设置新的广告系列 (而不是编辑现有广告系列)，可以从“广告系列”信息中心的“操作”菜单中选择“上传数据”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">点击“下载 XLSX 模板”链接。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">输入新广告系列的设置数据后，上传该电子表格，将这些数据添加到您的账号。需要注意的是，无需填写广告系列 ID 和广告组 ID 单元格，因为 Apple Ads 会自动生成相应 ID。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/9eac80dc6c6b274b8e12a9c036f26231.png" alt width="1080" height="399" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/d8d3a2d972d6cb50e983baa5673ea92a.png" alt width="988" height="499" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上传广告系列或广告组的编辑</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从“广告系列”信息中心的“操作”菜单中选择“上传数据”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">选择包含已编辑的广告系列或广告组设置的 XLSX 文件。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">点击“上传”按钮。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/3da3e32030bd705ee69c42ee0ac833b1.png" alt width="1080" height="545" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外，在使用广告系列和广告组设置模板时，营销人员需要注意以下要点： </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">每个电子表格最多可添加 500 行。每一行将包含一个特定广告系列和广告组的信息。所有广告系列必须属于同一个广告系列组。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">输入多个值时，请使用英文逗号进行分隔。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所有货币数据 (例如每日预算和出价金额) 都将使用你的 Apple Ads 账号所设置的币种。请注意，每次点击付费 (CPT) 定价在“定价模式”列中显示为 CPC 或每次点击付费。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">若搜索标签广告系列之前使用每千次展示付费 (CPM) 定价，则其“定价模式”列会显示为 CPM。但所有有效的搜索标签广告系列现在都使用每次点击付费定价。请注意，目前中国大陆的 App Store 上不提供搜索标签广告。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在设置广告系列或广告组的开始或结束日期和时间时，请使用年-月-日和时-分-秒格式：YYYY-MM-DD HH:MM:SS。日期和时间将以你 Apple Ads 账号所设置的时区为准。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果仅希望在一周中的特定日期或时间投放广告，你可以在“分时段”列中输入时间。输入的数字应介于 0 到 167 之间，表示从周日凌晨 12:00 开始对应的这一周中小时数的数值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如，如果你希望在星期五早上 6:00 开始在 App Store 上展示广告，并一直投放到星期六中午，则应输入 126 至 156。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">将任何受众群体细分应用于广告组，会自动排除已关闭“个性化广告”的用户。这会限制在 App Store 上看到你的广告的用户数量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">可以使用模板中的“包括回访用户”和“排除回访用户”列，重点面向特定的受众类型投放你的广告组。具体方式如下：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（1）所有用户：将电子表格中“包括回访用户”和“排除回访用户”列中的单元格留为空白。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（2）新用户：在“排除回访用户”列中，输入广告系列中重点推广 app 的 app ID。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（3）回访用户：在“包括回访用户”列中，输入广告系列中重点推广 app 的 app ID。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（4）其他 app 的用户：在“包括回访用户”列中，输入你的任何其他 app 的 app ID。你的广告系列中的广告会向这些 app 的用户展示，从而交叉推广你的产品。</p>                        <div style="height: 1px"></div>
