<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从最早的唱歌跳舞，到后来的游戏电竞，再到卖货打PK，直播内容随着行业的发展在不断扩散，各种和大众相关的元素都在和直播发生化学反应，这其中也包括宠物。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前，宠物经济在中国被认为是尚未挖掘彻底的蓝海市场。而随着直播的加入，围绕宠物衣食住行的<a href="https://www.niaogebiji.com/tag/%E7%9F%AD%E8%A7%86%E9%A2%91" target="_blank">短视频</a>账号也层出不穷，云吸猫狗成为网友新的消遣。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，最近在抖音上有着600万粉丝的“网红小狗”过的却很不开心，同样不开心的还有他的主人，原因是网友质疑他们的视频存在虐狗。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众所周知，虐待宠物是这两年社会一直关注的重点话题，而在抖音平台上，利用人狗互动的冲突感吸引粉丝的账号并不少，这次的舆论风波或许并不是个例。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相</span><span>信</span><span>很多网友都看过</span><span>一只名为</span><span>“</span><span>小胖</span><span>”</span><span>的狗狗的表情包</span><span>，</span><span>憋屈的眼神，蜷缩的爪子，一副不太聪明的样子。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/165966073762fa8c48e687b4.16030873.gif" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>凭着这个被誉为“全网最委屈”的小狗，他的抖音账号“小胖科学家”一路涨粉到600万。但最近有网友发现，小胖可能受到了主人的虐待。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友发现，在“小胖科学家”发布的视频中，他激动地对小胖高歌，小胖却表现出了异常的烦躁和厌恶。在一套娴熟的挑逗下，小胖彻底失控，呲牙咧嘴，大声狂吠。于是在网友“放大镜”般查找下，有人发现，为什么小胖下牙只剩两个？为什么会对主人充满敌意？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/157937940762fa8c4a759aa6.66787652.gif" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有网友发现，在往期的视频中，狗主人一边唱歌，一边疑似从衣服上拔下一根“针”，扎向小胖后丢弃。随后，狗便一边大声吠叫，一边做出“龇牙咧嘴”的反应。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对网友铺天盖地的质疑，“小胖科学家”显得有些急躁，并在随后的直播中“我为了流量折磨它，你能怎样”“如果号没了，我就把狗杀了”之类的表态。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，有愤怒的网友选择了报警。而“小胖科学家”也在8月8日的直播中澄清，称自己并没有虐狗的行为，并称网友所看到的“针”，其实是飘下的狗毛。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/30208828962fa8c510b30c4.18595986.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，他还反驳了之前所说的“拒绝兽医的检查”一说，晒出与自称是“潍坊市兽医协会会长、潍坊市流浪动物救助保护站长”之人的聊天记录，表示自己主动要求出示伤情鉴定，但对方称“关于鉴定必须由公安局、人民法院委托聘请”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，长期关注“小胖科学家”的网友应该发现，因为当初小胖的委屈表情走红后，账号下的所有作品都是委屈表情。而在直播中，小胖面对主人唱歌表现出狂躁一面也是常规节目，而这种画面看多了确实让人不适。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得注意的是，在虐狗事情发生后，有好事网友找到了“小胖科学家”的家，网友现场看到小胖的精神状态还不错，也没有出现狗主人虐狗的情况，只是这些内容依旧没有打消网友的质疑。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>舆论事件发生后，抖音平台虽然没有明确回复，但是在平台已经搜索不到“小胖科学家”的账号。不过，这样的封禁并不能掩盖抖音上很多主播在宠物内容上剑走偏锋。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2年前大胃王因为出现吃多导致猝死的问题，随后大胃王主播纷纷作鸟兽散，其中个别主播就偷梁换柱，把大胃王变成了宠物，让宠物在镜头前上演吃播。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，有人给倒了整整一大盆的生鸡腿，吃完鸡腿后，又喂下了颗足足三斤重的牛心。此外，金毛还需要面临连吃38种零食，连喝8种饮料的严峻挑战。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/57166293962fa8c526f61e3.98454172.gif" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更有甚者，把人吃的食物强行喂给宠物，以宠物的反应为噱头博取关注，完全不顾宠物的健康，比如给狗狗喂辣子鸡，骗狗狗吃跳跳糖等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些视频看上去，主人很爱宠物舍得给宠物花钱，还给了主播和人同样的待遇。可实际比大胃王吃播还畸形，因为宠物不会说话、没法表达自己的痛苦。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，除了宠物大胃王，在抖音上办“狗德培训班”也是一些账号的<a href="https://www.niaogebiji.com/tag/%E6%B5%81%E9%87%8F%E5%AF%86%E7%A0%81" target="_blank">流量密码</a>。这里面宠物狗的行为纠正是最多的，比如一些常见的坏习惯，咬绳子、拆家、扑人、爆冲等，而这些宠物学校的教育方式也并没有那么温柔。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年6月，有一段网传视频显示，抖音上有244万粉丝的知名训犬机构犬道文化创始人唐伟敏在给学员示范训狗时，采用踹脸、勒脖、抡摔等形式，疑似虐狗。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/58394513562fa8c58e209d8.65355241.jpeg" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来，唐伟敏在犬道抖音账号上发视频回应虐狗一事，称网传视频是2017年开培训班时拍摄，而且狗子具有“先天等级心极强加后天神经质攻击人”的心理问题，也与狗主人签订了免责协议。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与犬道文化的打骂训练不同，一些宠物博主也善于玩“擦边球”，虽然自己不在视频中和宠物动手，但是会让宠物之间进行撕咬争斗。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以抖音293万粉丝的“豫西小丁”为例，他的视频中多是一些行为有问题的狗，他训狗的方式就是叫两只狗之间较量，以达到一个臣服的效果，其中比较凶猛的狗子“黑豹”，也经常被网友关注。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据查询，抖音#训犬#内容已有40.1亿次播放，#狗狗上学#已有1.1亿次播放。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>坦率的说，</span><span>“</span><span>小胖科学家</span><span>”遭遇如今的舆论压力，其实他应该早就有心里准备。</span><span>因为很多</span><span>宠物</span><span>主播</span><span>之所以要做这种不用动脑子就知道会</span><span>引起争议</span><span>的事，归根到底还是为了</span><span>流量</span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像上面聊到的，“小胖科学家”走红就是因为委屈表情，很多人来看他也是为了小胖的一脸委屈，这个标签平台已经给他定死了，没了这个人设，这个账号的吸粉密码也就荡然无存了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样，宠物吃播为啥要换着花样各种猛吃，因为这些主播都会推荐不同的狗粮，只有自己的宠物狗展现出大快朵颐，那么才会有人来推荐的商品。训狗的账号也是同样的道理，只有食品里的狗狗言听计从，才会有更多狗主人来找他解决狗的不良习惯。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说到底，宠物视频还是人的视频，比较制作观看的都是人，卖东西的和买单的也</span><span>都是人</span><span>。</span><span>只是有的正常狗主人，喜欢分享自己狗狗和谐相处的一面。</span><span>而不正常的狗主人，只是把狗狗当做工具。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>几年前公众号“人间theLivings”曾经采访过一位网红萌宠主播，文章的标题就是《成为萌宠博主后,我害死了我的猫 | 人间》。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>故事就讲述了她的猫咪成为网红之后，每天都在经纪公司的拍摄工作室和出租屋之间来回奔波。虽然猫咪伙食从此都不愁，主播也赚到一套房子的首付。可是猫咪最终还是因过度劳累，无力抵抗疾病去世。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此前快手曾发布《2020年快手宠物生态报告》显示，在快手上，宠物短视频单日最高播放量高达7亿，并且每4.5秒平台上就有一场宠物直播。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在淘宝直播间，每天就有上百万人观看宠物直播，2020年，仅淘宝平台的宠物直播开播场次就同比增长了375%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在抖音，根据蝉妈妈发布了2022上半年抖音电商行业报告显示，宠物行业销售额占抖音全行业销售额的0.8%，虽占比不高，但其增速却达到了473%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，宠物经济的蛋糕在利用直播不断变大，但宠物主播们的内卷也在逐渐加重。希望所有宠物主播都能在粉丝日积月累后，能面对自己的“小伙伴”保持初次见面的那份“爱”。</span></p>
