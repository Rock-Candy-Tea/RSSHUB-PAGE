<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>夏天到来，又到了饮料品牌争相抢占市场的季节，尤其气泡水领域，在近两年消费者追求「健康又好喝」的趋势下，越来越多品牌加码投入这个赛道，进一步让竞争白热化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>富有前瞻策略的气泡水品牌，不止追求产品好喝、健康，在品牌识别方面也早早有了布局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，全球食饮行业巨头百事集团旗下的气泡水品牌bubly微笑趣泡（后简称微笑趣泡），自去年五月进入中国市场，秉持做「会微笑的健康气泡饮」的初心，基于对中国消费者的洞察与产品研发，<strong>以彩虹色的产品包装、0糖0卡0脂、清甜口感和微笑概念，迅速在赛道中拥有一席之地</strong>，今年持续加码新口味，推出草莓与苹果两款全新口味，同已有的白桃、百香果及蜜柚味共筑微笑趣泡“彩虹系列五重奏”，赢得广大年轻消费群体喜爱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微笑趣泡用一个微笑符号贯穿了品牌始终，<strong>从视觉感官上来说，白色的微笑弧线在彩虹色的瓶身上十分突出，再结合「bubly微笑趣泡」的品牌名中的“u”也化作了一道微笑弧线，以微笑创造情感区隔度和情感共鸣，在赛道中拉开品类差异性</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/170876575262bcc03411b573.98004265.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微笑趣泡始终以「微笑」核心主张，迅速俘获消费者、市场与粉丝青睐，如</span><span>去</span><span>年抓住年轻人的</span><span>emo</span><span>时刻，谱写传唱主题歌曲《扬起嘴角》，进行情绪化场景沟通，将微笑趣泡与微笑深度绑定，初步形成品牌印记。</span><span>今年携手代言人肖战打造《微笑趣传递》大事件，结合生活中平凡而值得微笑的瞬间，将微笑心智深入消费者生活场景，在市场上建立差异化感知。</span><span>其对消费者的情感洞察、内容创意和传播手段，或许可为行业带来新鲜的视角。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1183956285627e754119d138.35814623.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>洞察年轻人情感需求，</span></strong><strong><span>沟通品牌「微笑」核心价值</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管许多行业玩家都入局了气泡水赛道，但是目前鲜有品牌关注到消费者的情感需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，在当代年轻人群体中，情绪健康和情绪表达正在成为越来越重要的话题，轻松、快乐成为社交场合中令人羡慕的品质，<strong>这也是微笑趣泡想要传递给消费者的核心价值——微笑、乐观、自信</strong>。微笑趣泡精准抓住年轻人的情绪价值需求，以不同口味气泡水陪伴年轻人度过生活中的大小时光，从味觉和心情上，给到年轻人们即时的慰藉和愉悦感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/131883096062bcc03462b3c9.92295725.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期，微笑趣泡便以「</span><span>微笑趣传递</span><span>」为主题，携代言人肖战，推出4支微笑系列故事。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/121198927562bcc035d42d01.33106245.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《微笑趣传递》系列故事的开头，是一位在低潮情绪中受到微笑画罐鼓舞的主人公，成为了微笑小站主理人，进而透过在罐身上作画，将微笑传递给不同的人，点亮平凡生活的美好瞬间，也是「故事的开始，由微笑拉开序幕，故事的延续，用微笑连接你我」之意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/114369655462bcc036b26253.01502277.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了</span><span>第一部系列故事以肖战为主角，后面三部的主角，是像我们身边一样的普通人。</span><span>有相约聚会却因刘海剪得不满意而不自信的姐妹，收到代表「管它有没有型，微笑就行」的微笑画罐，重新绽放笑颜；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/32772230662bcc0387fe9b8.66674702.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有因为拍照「不够好看」而拌嘴的情侣，双双收获微笑画罐后，意识到「什么最好看？你笑起来最好看」的朴素道理；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/212031569562bcc0396f85f7.13151011.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有和同事聊天却插不上话</span><span>的职场萌新，本想</span><span>默默退出聊天，却在微笑画罐的鼓励下，</span><span>同时展开了笑容和自信，做到「想打成一篇，先笑成一片」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家有年轻的生命力，却也会因为生活中小小的烦恼而忘了微笑，而微笑趣泡，就是给平淡生活中带来微笑的好伙伴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>系列故事中，每一次让人会心一笑的，<strong>表面上看是微笑趣泡的微笑画罐，实际上，是微笑的传递与释放，这既是一次将产品融入于日常的场景营销，也是品牌将微笑传递给消费者的积极动作</strong>，在场景的递嬗和明星具有感染力的笑容下，微笑趣传递系列故事以和生活相近的内容和用户搭上情感桥梁，将微笑画罐带入职场到约会等不同的场合，传递品牌「微笑」核心价值，让用户感受到微笑趣泡的品牌魅力。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了系列故事外，微笑趣泡在传播层面，围绕「微笑」核心记忆点采用多层次宣推布局，以官方媒体</span><span>+</span><span>代言人</span><span>+KOL</span><span>齐发力，共创声量破圈，实现品效合一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>● </span><span><strong>4段式系列故事引发期待，正能量吸引官媒报道</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顺应当下注意力碎片化的时代趋势，微笑趣泡将系列故事分成四部短片依次发布，每一部短片仅</span><span>1</span><span>分钟多，让观众在快速接收到核心信息的同时，引出对下一部短片的期待。</span><span>品牌以优质内容在短时间内聚拢流量，并且增加用户粘性和讨论度，打好话题基本盘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短片中的微笑故事和微笑趣泡的正能量品牌内核，也吸引到官方媒体如新民周刊和中国青年网的报道，肯定品牌为社会带来的正向价值，侧面突显品牌的社会责任感，也印证好的故事，无论在什么时代环境下都具有稀缺性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/47810839862bcc03aa6db98.38827296.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>●</span><span><strong> </strong></span><span><strong>明星势能，带动微笑「合拍」</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌代言人的肖战，本身因为庞大的粉丝基数和明星势能，为微笑趣泡带来可观的声量和销量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，由肖战出演的系列故事，因为故事节奏明快、氛围温暖、极具感染力的笑容，启发到抖音达人的二次创作。许多达人将品牌系列故事改为合拍视频，凭演技和服化道让自己与肖战合演微笑故事，可以说是用本事追星，让身边微小的故事都成为创作的题材，分享属于自己的微笑力量，最后不忘带上微笑画罐，<strong>一边花式玩转内容创意，一边将微笑传递给更多潜在受众，鼓励每个人挖掘自己生活中绽放笑容的瞬间</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12459202162bcc03b1db3d2.81986764.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>●</span><span> </span><span><strong>KOL接力传递微笑画罐，形成破圈声量</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更多人也受到系列故事的感染，开始分享自己生活中的微笑时刻，</span><strong><span>尤其是被微笑趣泡</span><span>0</span><span>糖</span><span>0</span><span>卡</span><span>0</span><span>脂的健康属性和缤纷多重果味的基因击中的美食和运动人群。</span><span>他们将微笑画罐融入日常生活，带动了新的表达形式，也让微笑趣传递成为大家的共同话题</span></strong><span>，有的人还会艾特其他朋友一起微笑「微笑传递」的动作。</span><span>因此顺着一个个帖子和艾特，</span><span>用户</span><span>能直观感受到</span><span>「微笑趣传递」的核心概念——</span><span>将微笑传递下去</span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>越来越多的人在微笑浪潮的感染下，将微笑画罐作为一种新的表达形式，将快乐传递出去， 三五好友之间，一起在露营或享受美食的同时画罐、晒罐。瓶身上花样各异的笔触，既是创意的落地，也是品牌与消费者动人的链接，短时间内，社交平台上便累积了满满正能量扩散到各圈层用户的信息流中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/107932591862bcc03b916336.23754713.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/199639904562bcc03c247ca7.55186076.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微笑趣泡透过更加真诚的内容和具有新意的传播手段，将微笑和产品渗透到许多消费者的手上、脸上、心上，在有形和无形之间，拉近消费者和品牌距离。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>●</span><span> </span><span><strong>顺应社会氛围，用会心一笑的内容深入年轻人心智</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>纵观微笑趣泡的营销动作，一个重要的前提是，品牌抓住了年轻人对笑容和活力感的核心需求，尤其在近期，许多人的生活节奏因为疫情被打乱，人们格外需要一些正能量振作起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微笑趣泡切中了社会情绪，贴合品牌核心，讲好了一系列「微笑」故事，既为产品发声，也传递了品牌正向情绪价值，以「微笑」温柔消化年轻人们的小确丧，回应了市场的振荡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>透过本次系列故事和营销动作，<strong>微笑趣泡让年轻人看见，生活虽有各式各样的困难，但是总有品牌和你站在同一阵线，你不止有力量将小确丧化为小确幸，更能将微笑的力量传递下去，做有意义、可以影响很多人的事情</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这份情感营销，不催泪，却足够照亮人心。</span></p>
