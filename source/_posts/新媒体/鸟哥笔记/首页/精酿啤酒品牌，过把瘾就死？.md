<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近两三年，因为疫情、就业等因素的影响，消费降级、价格敏感等话题热度不断攀升，但放在稍长的时间坐标里，消费升级的趋势不容忽视。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>啤酒当属升级特征最显著的行业之一。数据显示，2011~2018年，低端啤酒销量占比从89.1%下降到76.5%，而高端啤酒销售额增速在20%~35%之间波动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当啤酒行业走向高端化，主打风味与口感的精酿啤酒也深受消费者青睐。据观研网测算，2013~2020年中国精酿啤酒消费量复合年均增长率高达35.38%，远远高于啤酒行业整体<span>（-5.27%）</span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看起来，精酿啤酒在消费者的推动下迅猛发展，但对于品牌和商家来说，要吃到“蛋糕”并非易事，一不小心反而会沦为“炮灰”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据媒体报道，2016年1月5日~2022年1月5日，与精酿啤酒相关的5385家企业中，有1061家企业状态已经处于注销、吊销、停业状态。另有数据显示，2021年，国内精酿终端的年化开店率为38%，年化关店率达到35%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这些数据不难发现，精酿啤酒品牌正在经历“冰”与“火”的淬炼，众多品牌的宿命可能是“过把瘾就死”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/21315772362ea824e9f40d3.14892757.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 速生速死 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“精酿啤酒和工业啤酒的区别，就好比手擀面和方便面。”这是一家精酿啤酒品牌创始人李威的评价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在李威看来，精酿啤酒就像“手擀面”，满足了消费者对经典、个性化的追求，本质上是消费升级。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一观点能得到大量精酿啤酒拥趸的认同，有的消费者喝精酿啤酒的体验比“方便面”和“手擀面”的差异还要大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>举个例子，2008年，林林在上海一家小酒馆喝到了一瓶比利时修道院生产的精酿啤酒，媒体称，入喉的那一刻，“就像打开了新世界的大门”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费者自有不同体验，而对于精酿啤酒的定义，也是各有不同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，在现代精酿啤酒发源地美国，酿酒师协会对精酿啤酒厂有严格定义，包括年产量不多于600万桶、酒厂不被或是低于25%的股份由非精酿啤酒厂控制，且至少有一款主打产品，超过50%的销量中，没有使用辅料来酿酒，或者使用辅料是为了增加风味而非减少风味。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产量和股份等方面不必展开，</span><strong><span>单单是增加风味，就会产生各种变化，这正是精酿啤酒最吸引消费者的地方。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用国产精酿啤酒品牌创始人潘丁浩的话来说：“任何一个真正热爱喝啤酒的人，都不会满足于传统啤酒千篇一律的味道，总有一天，人们会在精酿啤酒的世界中，找到自己最爱的那一款。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/96368767862ea824ed9a948.10428984.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>除了类型、风味，在颜值方面，精酿啤酒的包装更加多样，个性更为鲜明，且能充分展现国潮等特点。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>依托这些产品优势，精酿啤酒从国外火到国内，并得到越来越多的年轻消费群体的追捧。据兴业证券研究报告显示，当前国产精酿啤酒消费的主力人群逐步向90后、00后过渡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在消费需求“刺激”下，各路企业纷纷入局，百威、华润、青岛、重庆、燕京、珠江等老牌啤酒企业都已杀入，还涌现出熊猫精酿、18号酒馆、高大师、啤酒狗、拳击猫、京A啤酒等原生精酿品牌，连卖凉茶的王老吉、吃火锅的海底捞也赶来分一杯羹。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短短数年，精酿啤酒这个细分赛道变得拥挤起来，但市场小众、消费习惯远未普及、行业高度内卷、疫情冲击等因素，使业内竞争愈加残酷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位创业者直言：“这个行业每年都会出现一批精酿啤酒创业者，而每年都会死掉一批。这个行业的死亡率很高，甚至有三分之一都活不过一年。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年底，某国产精酿啤酒品牌创始人沈恺对媒体透露：“我见到的10个工厂，今年转让的就有三四个，根本开不下去。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>速生速死背后，是让“局中人”透不过气的内卷。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/127079305662ea824f1c06e7.94027176.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 超级内卷 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌和商家对精酿啤酒赛道趋之若鹜，攀升的消费需求之外，高利润是另一个更直接的驱动因素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据财经天下周刊报道，有从业者表示自酿精酿啤酒的产品毛利率可以达到70%~80%，甚至更高，外购酒精酿馆的毛利率也可以维持在50%~60%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>精酿啤酒如此高的利润，已经在啤酒行业拥有头部优势的龙头企业自然不会放过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据观研网数据显示，精酿啤酒市场CR4<span>（前4名市场份额集中度指标）</span>约为47%，其中百威、嘉士伯两大外资龙头拿下了最高份额。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更关键的是，别看精酿啤酒在年轻消费群体中颇为风行，放在整个啤酒市场，仍然是个不大起眼的“小弟弟”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据媒体报道，精酿啤酒在中国市场的占有率不足3%，媒体用“占有率极低”来形容也算恰如其分，而市面上97%的啤酒都是工业啤酒，制霸地位不言自明。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面市场份额“极低”，一方面巨头垄断，其他精酿啤酒品牌要想活得久、活得好，就得使出浑身解数，以求在这场超级内卷的竞争中表现更加强悍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先，当然要有资金加持。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据不完全统计，2021年，精酿啤酒品牌融资超过10亿元，红衫资本、天使湾创投、挑战者创投是三家出手最频繁的“金主爸爸”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便如此，粗略算一笔账，会发现一年十余亿总融资并不多，投资建厂动辄上亿，更别提两三千万元的罐装设备和构建渠道网络的高昂费用。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/152528314062ea824f4c5352.09399925.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其次，拿到有限的投资，需要在产量和风味上发力。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就产量而言，一些具有全国化潜质的国产精酿啤酒品牌年销量大多在5万吨以下，而且，部分销量来自非自有品牌代工，更多区域性独立品牌年销量更是无法突破千吨“大关”，谈不上扩展市场，说“为爱发电”或许更贴切。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>风味更是品牌和商家内卷的重点。为了体现自家啤酒的个性，品牌努力在风味上展开差异化竞争，或是添加蜂蜜、生姜、百花蜜甚至花椒，“或是调制女性更喜爱的偏水果口味的精酿啤酒等”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>再者，</span><span></span></strong><strong><span>推出若干款不同风味的精酿啤酒后，</span></strong><strong><span></span><span>还需要直面“价格战”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个对比强烈的现象是，疫情之下，高度依赖国外供应的麦芽、啤酒花等原材料成本飙升，但精酿啤酒没有迎来相应幅度的涨价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是因为，行业竞争激烈，加上各个层次的消费者都开始对价格敏感，除非有较强的品牌溢价，不然很难摆脱“价格战”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要在竞争中穿越上述重重障碍，精酿啤酒品牌才能真正站稳脚跟，为自身的长远发展打下坚实基础。从这个角度看，精酿啤酒消费需求持续增长，众多品牌却速生速死，也就很容易理解了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/97826315562ea824f84f4b4.70718121.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 最大BUG </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在已有的、份额“极低”的市场里竞逐，品牌和商家难以避免地陷入超级内卷，在此背景下，吸引更多消费者，打开更大的市场空间，变得相当紧迫。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，已经有不少品牌在努力突破，比如用风味、颜值和适当的价格吸引女性消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>媒体称，一家国产精酿啤酒品牌的后台数据显示，不仅是男性消费者，还有大量热爱饮酒的女性，“两者各自占据一半的销售额”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，在中国，精酿啤酒的消费习惯远未普及。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年6月，消费者报道发布了《啤酒消费习惯问卷调查》，收集到1560份问卷。其中，35.2%的消费者会选择精酿啤酒，但6.9%的消费者表示“没有听说过精酿”，38.3%的消费者“不太了解精酿，但是愿意尝试”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在和君咨询高级合伙人、酒水事业部主任李振江看来，</span><span>精酿啤酒的消费者教育依然在起步阶段，向前发展仍然需要“任何资源都替代不了的时间成本”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/161991586962ea824fc1a5b8.94344493.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李振江进而认为，</span><strong><span>目前精酿啤酒赛道的最大BUG就在于用户不懂精酿啤酒</span></strong><span>，“这也为这个行业的巨大发展空间提供惊喜，同时也为这个行业埋下伏雷”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现实也是如此，一些品牌利用懂得精酿啤酒的消费者不多这一行业现状，以浑浊工业啤酒充当精酿啤酒，用精美包装和拙劣酒质混淆视听等，乱象迭出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个饶有意味的案例是，2019年，罗塞尔啤酒中国公司的掌门人公开喊话，要力争成为中国精酿啤酒市场第一品牌，但到了2021年，罗塞尔合肥公司因篡改啤酒生产日期并对外销售，被罚款123万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费群体的培育、扩大，需要时间成本，但显然很多品牌和商家要焦虑得多，以至于走上“歪路”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，</span><strong><span>也有诸多品牌在发力培育消费者。</span></strong><span>在精酿啤酒行业，创立“高大师”品牌的高岩在行业内摸爬滚打已有14年，他对媒体表示，自己现在最想做的，是继续做好精酿文化推广。“还没有那么多人接受，文化推广的任务还得继续。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谈及对国内精酿啤酒未来的发展，高岩坦承，在越来越多的人了解、接受精酿啤酒的情况下，不希望它成为杀低价、恶性竞争的一个行业，“需要一系列的规范，让真正从事精酿啤酒文化产业的人从中获益。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，这将是一场多方角力的“持久战”，冰与火的淬炼、生与死的交错、从小众走到大众的野望，仍将是行业常态。</span></p>
