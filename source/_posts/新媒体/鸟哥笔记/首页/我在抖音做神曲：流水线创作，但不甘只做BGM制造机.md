<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“现在跟以前不一样了，抖音没有流量，谁听你的歌”，Deepain说道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Deepain是一位原创音乐人、录音师、混音师。从2014年入行算起，他已经做了8年音乐，期间目睹了传统唱片行业的没落，也见证到抖音为代表短视频神曲的爆火，他服务过的音乐人陆续向抖音迁徙。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年8月，他投身于抖音，本着玩梗将账号命名为“Deepain（多次婉拒格莱美）”，</span><span>认证成为一名“抖音音乐人”。</span><span>凭借自己创作出的爆款BGM，Deepain在抖音获得了192万粉丝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比听到BGM的“耳熟能详”或“似曾相识”，观众们对Deepain却了解有限。<strong>歌红人不红，也是大多数抖音音乐人的缩影。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然有落差，但Deepain也接受了这个现况。此后，他利用爆款BGM带来的关注与流量，去推广自己的原创单曲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种“曲线救国”的音乐人在抖音并不少见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年6月底，抖音发布《抖音音乐2021大数据报告》，报告显示，2021年，抖音平台新增音乐人数量环比增长92%。从入驻平台的音乐人年龄分布看，30岁以下的新生代音乐人占比近七成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当Deepain这批音乐人一边迎合抖音喜好，创作作为短视频BGM传播的音乐，另一边也努力推销自己，做喜欢的音乐，不甘只做BGM制造机。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>人在抖音，歌红人不红</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一首歌如果想在抖音火，能不能成为作为BGM使用传播，是重要的考量因素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>受抖音短视频时长限制，大多数BGM的前奏铺垫很短，目的就是尽快进入副歌高潮部分，快速刺激观众的视听觉感官，调动受众情绪，创造音画合一的协调感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在7月8日更新的抖音热歌榜单中，前5首热门BGM中，有4首的时长都在30秒之内。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/43221027562cd10765dd560.02940411.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短时间内能够让观众留下记忆点只是第一步，听完会不会“上头”，让歌曲在脑海中挥之不去，形成现象级传播，是成为抖音热门BGM的关键。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，Deepain总结出几种常见的BGM套路：<strong>旋律轻快，歌词简单，和弦重复，卡点节奏好。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当一首歌符合上述要求后，再加上抖音的算法推荐机制，这些BGM就像在你耳朵里种下了一只耳虫，自动开启三日颅内循环模式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但如果套路一直重复，难免会让观众产生厌倦感，Deepain尝试一次性改编翻唱两首风格不同的歌曲，并将其主题命名为<strong>“你永远猜不到我下一句唱什么”</strong>，给观众带来了新鲜感。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于他来说，创作一首抖音BGM并不难，如果能找到合适的改编歌曲，一个小时左右就可以创作完30秒的BGM片段，“工作多年的本能反应了”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年1月，由他改编翻唱的《That's what i like+身骑白马》以及《sugar+荷塘月色》两首歌曲在抖音分别获得了220.8万和131.5万点赞，原声使用人数累计达71.3万，为他带来一百多万抖音粉丝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过抖音短时间内的密集推送，更多人听到了Deepain的歌曲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Deepain第一次感觉到，自己好像红了：<strong>“最火的时候，每10条抖音视频里，几乎2-3条都是用我的BGM，就连之前很久没联系的朋友也来问我怎么做出来的。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>流量带来的不仅是关注，也夹杂着一些负面的声音。Deepain收到了很多指责他的留言，质问他为什么要改编翻唱这些歌，破坏了歌曲原有的意境。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Deepain觉得，或许放慢更新节奏，做出更好的音乐，听众的满意度也会更高一点，便将更新频率从一周更新3-4次，减少到一周更新1-2次，而这也间接影响他的粉丝量增长速度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过，像Deepain这样的粉丝量达到百万以上的音乐人已经是幸运儿，抖音BGM常常伴随着“歌红人不红”的现象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《2020抖音音乐生态数据报告》显示，2020上半年抖音音乐人入驻增长近3万，涨粉超500万的音乐人有29位，比例不足0.1%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年夏天，一首“危险危险危险，ye~好久不见~”的热门BGM曾刷屏抖音，原声使用人数达162.4万人。但演唱者陆政廷的原视频点赞量仅有1.9万，抖音粉丝量为4.4万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>究其原因，<strong>一方面是抖音掌控着流量推荐池，</strong>一首歌能否出圈很大程度取决于抖音是否对其倾斜流量，这意味着抖音歌曲的传播不受制于音乐人自己。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另一方面，相比原生博主对抖音内容规则的熟悉程度，音乐人并不占据内容优势。</span></strong><span>也因此，一些博主只是引用BGM就能拍出百万点赞的爆款，由音乐人首发的视频却点赞寥寥。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137970173262cd10769f0244.55214762.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>同一首BGM的原版与引用版，两者点赞量相差115倍</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>一边努力搞钱，一边推销自己</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然“歌红人不红”困扰着大多数抖音音乐人，但这个问题也普遍存在于整个音乐圈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《2021年中国音乐营销发展研究报告》显示，53.5% 从业者认为，相比于推红一首歌，推红一个音乐人更难。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>既然能不能红是玄学，那么努力搞钱才更现实。</span><span>在抖音，音乐人比较常见且可行的商业模式主要有以下三种：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1. 唱片公司或版权代理商通过星图联系推歌，由音乐人进行翻唱，或者直接发布需要推广的歌曲。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年4月底，小鬼（王琳凯）改编演唱的歌曲《Mood》上线发布，<strong>抖音与其达成宣发合作，在站内组织配合了一系列营销玩法：翻唱、合唱挑战（原唱/小鬼合拍）、卡点变装等。</strong>其中音乐人“王玉萌”以女声唱腔翻唱了该歌曲，获得了99.7万点赞。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/110119203062cd10772d28b8.84546260.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截至去年7月7日，抖音话题“mood”播放次数为4亿，产生2.2万个视频，成为抖音推歌的成功案例之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2. 品牌方联系合作，创作专门的广告歌。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“super idol的笑容，都没你的甜，八月正午的阳光，都没你耀眼。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这首《热爱105℃的你》是歌手阿肆在2019年为屈臣氏蒸馏水定制的一首广告歌，时隔两年走红抖音，再次给品牌带来曝光。<br><br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/33175865662cd107dd51ef2.55082968.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在接受刺猬公社的采访中，阿肆表示，“恰饭歌的核心之一是旋律要悦耳，记忆点非常多，其次要符合品牌的核心，把企业的宣传点延展成一个比较具象的创作点”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于短视频音乐和纯音频音乐有所不同，<strong>阿肆强调短视频音乐的歌词一定要有画面感，“画面就是要super idol的笑容与歌词完整对应”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3. 通过发行自己的原创歌曲，获得版权收入。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据Deepain透露，已得到原版授权改编的《身骑白马（deepain版）》与《大花轿（deepain版）》等歌曲已经发行上线于QQ音乐，预估可获得10万元左右的版权收入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>商业变现之外，音乐人们也借助抖音流量，宣传推广自己的原创作品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年3月，Deepain发布了原创新单曲《升温》，为了吸引更多观众关注到这首歌，他在抖音陆续发布了4条宣传视频，提前放出歌曲demo，并鼓励观众“可以提前拿它去做BGM拍视频”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截至目前，4条宣传视频中最高点赞量为4.7万。对于这样的成绩，Deepain最开始有些许失落，“花一个月认真做的歌没有一个小时改编翻唱的30秒片段效果好”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>平复心情后，他接受了这样的结果。在他看来，通过创作迎合抖音喜好的歌曲获得关注，再借用这波流量去推广自己的原创歌曲，这是一个可以实现正向循环的过程，只不过需要时间去等待和验证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，Deepain的涨粉不只是抖音站内，QQ音乐和网易云音乐的粉丝数也从最初的2000增长至6.1万，这帮助他在站外也获得了一定知名度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>抖音音乐“争上游”</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再过一个月，Deepain即将入驻抖音满一年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>期间，他一共发布过100期作品，聊起自己最喜欢的一首，他选择了一首关注度没那么高的改编歌曲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/106028886762cd107e314b83.47074792.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该视频发布于去年9月，Deepain通过模仿喜欢的嘻哈歌手Travis Scott的唱腔，用英文翻唱改编了腾格尔的《天堂》，获得了5920个点赞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回忆当时的场景，他依然印象深刻：“那天晚上我做完这首歌，我自己都觉得不可思议，这两首风格完全不一样的歌曲居然可以融合唱到一起，音乐真的太神奇了。”但Deepain也坦言，这种小众音乐风格在抖音只能圈地自乐，<strong>“懂的人会疯狂喜欢，不懂的人听都不听”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于抖音喜好和自己喜好两者间的矛盾，Deepain权衡过利弊：<strong>“如果我一直做早期的小众音乐，现在可能最多就10万粉丝，也可能没有。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近一个多月，Deepain感觉到爆款BGM越来越难做，“有时候所有因素都加上了，但就是火不起来”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对比抖音与QQ音乐、网易云音乐的热歌榜单，可以发现进入前5名的重合歌曲寥寥无几，抖音热曲对于整个音乐平台的影响力似乎正在衰减。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/199052946262cd107e5fa402.73938098.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，公众号“音乐先声”分析认为，2020年到2021年，是抖音推歌市场竞争最激烈的两年，<strong>一方面，抖音推歌搅乱了原有的宣发节奏，造成了不少跟风、同质化、粗制滥造的现象。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另一方面，平台内容的变化也在影响<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%A5%9E%E6%9B%B2" target="_blank">抖音神曲</a>的走红模式，观众也不再满足于简单的视听刺激，而是需要更多有故事、有共鸣的内容，</span></strong><span>比如《漠河舞厅》背后的独舞老人故事，《孤勇者》与战疫、抗灾等内容的匹配。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，抖音也尝试向音乐行业上游靠拢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月中旬，字节跳动旗下首款音乐App “汽水音乐”正式开启公测。此后不久，抖音音乐版权代理平台银河方舟也推出特别企划，联动告五人、房东的猫等八支乐队组合，在汽水音乐首发新单曲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“抢首发”这一举动是为了将用户引流至汽水音乐。当该歌曲后续分发到其他音乐平台后，有用户表示“之前是为了听新单曲才下载的汽水音乐”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/199745080162cd107e97af26.59247988.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期，抖音视频的BGM处会出现“汽水音乐”的logo</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，7月8日抖音上线了“2022看见音乐计划”，旨在扶持原创音乐人，给予歌曲推广、现金激励、商业化变现等奖励。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于各音乐平台间的抢人大战，Deepain已经习以为常。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他现在最想做的，就是之前与某音乐平台签约的独家合同到期后，改为全平台分发，“这可能是我们音乐人现在唯一能掌控的事情”。</span></p>
