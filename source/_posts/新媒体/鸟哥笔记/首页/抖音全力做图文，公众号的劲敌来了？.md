<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说，抖音曾经的宏图壮志都在悄悄试水，那么这一次，竞争终于光明正大地上了台面。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在9月20日的创作者大会上，抖音相关业务负责人介绍，<strong>未来将重点建设图文等新体裁</strong>，并透露<span>目前</span>抖音图文日均播放量超百亿，<strong>未来平台超过四成流量将扶持生活攻略类内容。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此番发言一出，无异于热油入水，噼里啪啦炸出了不小的动静。抖音做图文内容，到底靠不靠谱？</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果讨论“社交”，我们绕不开微信；如果聊起短视频，“南抖音北快手”可能脱口而出。那么谈起“种草”，很多小伙伴可能会提名“小红书”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就是平台长久深耕某个标签之后，对用户心智产生的教育作用。而这种作用慢慢就沉淀为平台的基因，反过来再加强用户的认知——想做什么事情就该到什么平台去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但随着近几年种草经济快速起飞，各大互联网平台对这门生意虎视眈眈。正如腾讯有电商梦想，支付宝有社交执念，抖音则对“种草”坚持不懈，大有不撞南墙不回头之势。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这一次，抖音选择在图文赛道上放大招。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>最新版的抖音首页除了原有的“探索”模块，还新增了“经验”卡，进一步扩大了图文分发的场景。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以看到，在相关关键词的搜索结果页面，顶部的导航栏增加了“经验”模块，以种草、教程类内容为主。该页面右上角还将内容载体进行区分，方便筛选“图文”内容并独立展示。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，在“综合”模块中，抖音也不吝于给“经验”这个新模块导流，用户的搜索结果将优先展示“经验”内容。<span>这也验证了抖音此次宣布的“激励方向”——生活经验、攻略测评、小众兴趣。</span><span></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1782725740632d25e267d790.21373557.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音在图文上的努力并非一日之功，但最终的结果确如预期吗？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>早在四月底，抖音就开始折腾自己的首页，在一级导航栏内测新增了“探索”模块。</span></strong><span>这和此前内测的“种草”频道并没有太大的区别，几乎只是改了个名字。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1975049486632d25e3540dc5.51760284.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在内容方面，“探索”页囊括了摄影、探店、烹饪、日常生活记录等类别，目前看起来还是相当丰富的。<strong>而相比“推荐”页，“探索”页的内容去娱乐、强种草的属性更加明显，同时图文内容的占比也更大，且推荐的大多是点赞等数据并不突出的账号内容。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/557415455632d25e5ca2e12.13149272.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看到这里，混迹各大互联网平台的你或许也发现了，抖音的“探索”频道完全就是翻版小红书。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有多像呢？咱来对比一下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在主页面上，</span></strong><span>两个App都是双瀑布流的排列方式，内容类型也相似，大多是种草，并以封面图+文案+账号名称+点赞数的组合形式展示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/904227498632d25e8621731.10893905.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>左为抖音，右为小红书</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>点进具体的内容页面，可以发现，无论是排版布局，还是评论区样式，都极其相似。不同之处只在于，抖音依然坚守自己的音乐优势，即便是图文内容也能拥有BGM，其余的不过是图标位置的微调。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>此外，在页面交互上，抖音的“探索”频道也摒弃了“推荐”页沉浸式的滑动切换，而是采用了看完一篇内容就得返回“探索”主页重新进入下一篇内容的方式</strong>，这也是和小红书的相似之处。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1582622268632d25eaaae1e4.91155851.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>上两图为抖音，下两图为小红书</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音在首页造了个「小红书」，足见其押宝图文种草赛道的决心。只是图文内容可不是短视频，抖音的基因优势并不在此，更何况路上还有不少庞然劲敌。抖音图文，会成一时昙花，还是新的内容霸主？</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音做图文种草，并非心血来潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当下，有很多研究数据表明抖音的增速放缓了。花无百日红，从2018年爆发至今四年，抖音正在和其他短视频App一同进入存量竞争时代。但这也意味着抖音急需一个新的增长点，让自己从混战中突围出来，其在这两年疯狂发力的电商被寄予厚望。而“图文”，则成了其中关键。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直以来，抖音的角色定位和电商就存在一定的割裂感，用户更习惯在抖音刷视频找乐子。所以，抖音在布局电商的最初，一度依靠向其他电商平台导流来赚广告费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>直到电商直播大热，抖音借机打出“兴趣电商”的概念。</span></strong><span>2021年双11期间，抖音电商直播间累计时长达2546万小时，直播间累计观看395亿次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年底，抖音推出了独立的电商App“抖音盒子”，试图借助抖音的流量，<span>完成引流</span><span>、种草、交易的全过程</span>。没错，就是前段时间被网传停运的那个“抖音盒子”，<span>起点即巅峰，在最开始的热闹散去之后便不见水花。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1955577075632d25eb4d5d39.12730325.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：“抖音盒子”官网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“价值研究所”曾提到：</span></p>
<section>
<section>根据E Marketer公布的全球电商发展报告，2022年之后，直播电商这一细分领域就要告别高速增长时代。然而，抖音的尴尬在于，虽然总的电商GMV已经达到5000亿级别，但超过四分之三的交易是在外部跳转链接完成的，抖音小店的GMV只有1000亿左右。</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，留给抖音的时间不多了。</span><span>抖音显然也意识到了这一点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经，腾讯花了大力气做N个短视频App，最终却是微信在自己生态里硬生生孵化出来的视频号出了头。抖音看在眼里，既然独立的种草电商App没法迅速承接上抖音的流量，那就自己上，以种草起家的“小红书”就是对标学习的对象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>抖音是做短视频起家的，算法为它带来了无数红利，也让用户养成了快速浏览内容的习惯，流量的沉淀门槛变高，而这恰恰是种草的“大忌”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说白了，种草有赖于平台的社区氛围，需要通过合适的内容形式在潜移默化中培养用户习惯。用户在消费之前的种草就像是购物之前的逛街行为，这决定了<span>能够给用户更多思考时间、信息量更详尽的图文内容更有种草优势。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，相比短视频，图文内容的门槛较低，能够激发更多人的创作意愿，帮助抖音快速积累内容</span><span>。</span><span></span><span>这也是抖音在去年年底推出“图文种草”并大力扶持的原因。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>随着抖音背靠短视频积累的流量，试图凭借种草图文剑指电商，“<span>种</span><span>草元老</span>”小红书和“图文霸主”<a href="https://www.niaogebiji.com/tag/%E5%85%AC%E4%BC%97%E5%8F%B7" target="_blank">公众号</a>，或将迎来新的对手。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音和小红书界面的相似程度，几乎要让人以为两者出自同一个产品经理之手，<span>日均播放量超百亿的成绩让大家看到了抖音种草内容的潜力。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>而在图文赛道上，抖音和公众号的内容逻辑开始有了一定的重合之处。</span></span><span>一直以来，用户通过公众号的图文内容种草，再利用微信生态自带或者已经发展成熟的第三方平台进行交易，这已经有了一个相当完善的闭环。如今，抖音也来了。</span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span>不过，虽然微信和抖音已经同时手握短视频、图文内容、电商链路，流量蛋糕的切分可以预见将会越来越激烈，但<span>就目前来说</span><span>，还不至于短兵相接。</span></span></span></strong><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>毕竟，抖音尚且<span>专攻短内容，而且两个平台的深层基因大有不同。抖音的优势仍在短视频，微信的护城河还是图文，这也是抖音至今没有图文黑马、微信也造不出视频大V的原因。</span></span></span></span><span><span><span><span></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>但不管怎么说，每一个新生的平台渠道，都是流量变现的机会。对于做惯了图文内容、又一直馋着抖音流量的创作者来说，这次或许正是机会。</span></span></p>
