                        <p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">零食行业实际上是非常适合做私域的行业。<span style="font-weight: bold;">单品价值低、客单价低、用户复购率高、用户覆盖广。</span>但也正因为用户数量庞大，很多品牌想做私域，却不知从何下手。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><a href="https://www.niaogebiji.com/tag/%E8%89%AF%E5%93%81%E9%93%BA%E5%AD%90" target="_blank">良品铺子</a>作为行业头部企业，早早地开启数字化的进程，尤其在私域和会员运营上都小有成绩。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">据2021年年报数据，良品铺子营收93.24亿元，<span style="font-weight: bold;">其中61.16%是由会员贡献的</span>，包括门店会员、电商会员、微信会员等在内的全渠道总体可触达会员已经达到了1.2亿人。<span style="font-weight: bold;">目前，良品铺子私域已积累沉淀超过8000万私域用户。</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么良品铺子是如何做私域的？下面就为大家拆解。</p>

<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1、案例简介</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">良品铺子，创立于2006年，已形成覆盖肉类零食、坚果炒货、糖果糕点、果干果脯、素食山珍等多个品类、1400 余种的产品组合，有效满足不同消费者群体在不同场景下的多元化休闲食品需求，连续7年高端零食市场终端销售额全国领先。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">4月28日，良品铺子发布了2022年第一季度财报，财报显示公司实现营收29.42亿元，同比增长14.3%。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2、市场规模</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">数据显示，从2010年到2021年中国休闲食品行业市场规模持续增长，从4100亿元达11562亿元，期间复合增长率约9.9%。预计2022年增速达7.2%，市场规模达12391亿元。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">3、用户画像</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">良品铺子用户群体主要是23-28岁的学生群体和一、二线城市中的白领群体。<span style="font-weight: bold;">其中80%都是女性。</span></p>

<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1、线下门店</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">良品铺子有3000家线下门店，8000+门店导购。导购可通过优惠权益，吸引用户添加企业微信号，再引导进入社群。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 507px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/OD1qhXvzu8fbPdfvyLBA.jpg" alt width="761" height="720" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2、公众号</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">引流路径：</span>点击公众号菜单栏【299-200】中的【新人礼|领340元】，跳转到相关页面，扫码添加企微【良小美】，再引导进入社群。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 468px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/oMjkSlPP6m9IEMRHLwXe.png" alt width="668" height="1126" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">3、视频号</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">路径1：</span>视频号首页设置了企微添加入口，点击【添加微信】即可跳转到企微添加页面。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 586px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/HvCRxOzx4hEmQgPFoIG0.png" alt width="554" height="900" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"> </p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">路径2：</span>视频号每天会进行直播，直播页面有社群引导二维码。扫描二维码即可进入直播社群。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 438px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/DpUFSApXIF5WQOhxNKZR.png" alt width="631" height="720" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">4、小程序</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在小程序首页，设置了社群添加入口。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">具体路径：</span>小程序主页面 –点击【社群福利】 –扫描二维码–添加企微– 企微发送社群二维码–进入社群。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 463px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/yHLS3by0QGp4UdBiCUtW.png" alt width="665" height="900" data-action="zoom" referrerpolicy="no-referrer"></p>

<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">5、抖音</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">良品铺子在抖音建立了账号矩阵，总粉丝数超过1000万。其中账号【良品铺子旗舰店】，有268.1万粉丝；账号【良品铺子】，有411.9万粉丝。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">视频内容主要包括<span style="font-weight: bold;">产品介绍、福利活动介绍、直播推广等</span>。几个主要账号每天都会开启直播，由主播口播引导用户下单。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 484px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/TaYEuDElnx08Urb1kejf.png" alt width="670" height="1126" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">6、微博</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">良品铺子在微博有139w粉丝，主页面内有粉丝群的引导，用户可申请加入。微博主要内容包括品<span style="font-weight: bold;">牌宣传、活动宣传、产品推广</span>等。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1、案例亮点</span>：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1）全渠道私域引流</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">良品铺子在全渠道运营私域，有着公域+私域的运营模式，打通了视频号与微信公众号、社群、企业微信等各个渠道进行转化。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">2）精细化运营</span></p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">良品铺子的社群分为福利群、直播群等类型社群。不同的社群运用不同的运营策略，将用户持续的留在群内。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2、案例不足</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1）IP运营不足</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">企微朋友圈基本上全是产品相关的广告内容，缺少一些“人性化”的内容。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2）裂变活动较少</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小程序缺少裂变活动，用户不能在正常流失的情况下快速得到补充。</p>
