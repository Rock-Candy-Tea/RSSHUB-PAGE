<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>时代的风，总是不住的吹。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>吹过黑夜，总会遇见晨光熹微。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自被反垄断调查起的两年里，百亿天价罚单、万亿市值崩盘，中国电商龙头阿里一度行走在黑暗之中。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论<a href="https://www.niaogebiji.com/tag/%E9%A9%AC%E4%BA%91" target="_blank">马云</a>、还是阿里集团也都一改往日的高调，选择默默闷头发育。如今，时代的风终于再度吹到了阿里的脚下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" title="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" alt="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>隐忍两年，</span></strong></span><span><strong><span>阿里迎来重大政策利好</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近一个月，吹向阿里的风来的密集而又热烈。上至首都、下至地方，各级政府官员开始了与阿里的密切接触。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20575396662ba3a4f10e453.01983518.jpeg" title="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" alt="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>5月25日，河南鹤壁。</span></strong><span>市长郭浩同阿里集团战略发展部豫蒙区总经理王延哲等人展开视频对接会，并承诺：“鹤壁将一如既往地为<a href="https://www.niaogebiji.com/tag/%E9%98%BF%E9%87%8C%E5%B7%B4%E5%B7%B4" target="_blank">阿里巴巴</a>集团提供服务保障，支持企业发展。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>5月27日，安徽蚌埠。</span></strong><span>市委书记黄晓武与阿里资深副总裁英华一行举行工作座谈，并表示：“阿里巴巴集团是互联网产业的领军企业，与蚌埠有着良好的合作基础和前景。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>6月9日，湖北宜都。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>6月13日，河南郑州。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>6月15日，湖南张家界。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>6月17日，北京朝阳区。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>……</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短短一个月的时间里，从地方到首都，类似的接触事件在全国各地上演，横跨大江南北，地方主政者与阿里的互动逐渐回温。到最后的朝阳区，尽管只是首都北京的一个区，但其独特的地理位置，也透露着不同寻常的意味。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/152723290462ba3a4f342434.40390068.jpeg" title="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" alt="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>且地方政要与阿里间的互动，也普遍围绕着双方的合作、投资等展开。在如今的大环境下，其背后意义不言而喻。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在阿里的杭州大本营，更是在6月中旬迎来了浙江省委常委、杭州市委书记刘捷的到访。在其走访活动中，共去了四个地方，其中两个都与阿里有关。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些接触活动之频繁，时间之集中，实属罕见。而其背后所传达的信号，也值得耐人寻味。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>是的，风向变了！</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正是从此次政企集中接触的前一周，暖风开始吹起。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5月17日，全国政协召开了“推动数字经济持续健康发展”的专题协商会，会议明确提出，要“支持平台经济、民营经济持续健康发展”，“支持数字企业在国内外资本市场上市”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5月18日，在稳增长稳市场主体保就业座谈会上，李克强总理也强调到，“支持平台经济、数字经济合法合规境内外上市融资”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正是由高层的风向转变，才导致了如今自上而下，各地方政要与阿里密切互动的场景出现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从曾经的“反垄断”、“打击资本无序扩张”，到“支持平台经济持续健康发展”，平台经济迎来新机遇，阿里也等来了它的黎明时刻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" title="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" alt="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>反垄断打击，</span></strong></span><span><strong><span>阿里市值蒸发万亿</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>提起阿里反垄断，便不得不提蚂蚁集团。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年7月，如日中天的蚂蚁忽然官宣IPO计划，拟进行“A+H”同步上市，消息一出，霎时炸响网络。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为年活跃用户达10亿+，年营收超千亿元的明星独角兽企业，如果市场条件允许，其IPO规模将达到史无前例的300亿美元，成为全球规模最大的IPO，没有之一！</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但仅仅几个月后，这一切都被按下了紧急暂停键，直至今日。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/133559398362ba3a4f983ec9.08920049.jpeg" title="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" alt="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>彼时，风头正盛的马云在外滩金融峰会上发表了“银行当铺思维严重”、“中国金融没有系统”等重磅观点，彻底激化了蚂蚁与监管之间的矛盾。几天之后的11月初，马云以及蚂蚁集团高层便迎来了中国人民银行、中国银保监会、中国证监会等四部门的联合约谈。这一消息也瞬间引爆了网络舆论。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>祸从口出，马云似乎要为他说出的话付出代价了。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>次日，上交所暂缓了蚂蚁集团的科创板上市申请，有关部门也开始介入，至此拉开了蚂蚁集团的监管序幕。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而马云的这场外滩演讲，也成为其事业的分水岭。至此之后，阿里巴巴股价开始暴跌，再也没能重现曾经的辉煌，马云本人也在江湖上销声匿迹。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在蚂蚁面临监管之时，这股整治之风也顺势吹到了阿里。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年12月11日，中共中央政治局会议提出“要强化反垄断和防止资本无序扩张”，剑指阿里。大约两周后，阿里巴巴便因涉嫌垄断被立案调查。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/120190521562ba3a4fbf2f18.64147285.png" title="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" alt="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很快，在次年4月，阿里便收到了国家市场监管总局所下达的182.28亿元天价罚单。相关公告显示，阿里自2015年以来滥用其市场支配地位，实施“二选一”垄断行为以获取不正当竞争优势。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，作为国内互联网龙头企业，强制“二选一”等行为严重侵犯了公共利益。其借助平台规模的扩张和资本的聚集，使得市场竞争失衡，国内数字经济的正常发展也遭受阻碍。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那个曾高喊着</span><strong><span>“让天下没有难做的生意”</span></strong><span>的阿里跌下神坛，屠龙少年终成恶龙的故事让人唏嘘。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那些曾经的</span><strong><span>“阿里独占”</span></strong><span>品牌们也不用再忍受“二选一”的折磨，纷纷回归京东，京东的品牌生意绝处逢生。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，这张百亿天价罚单也仿佛燎原之火，点燃了整个互联网，烧掉了数万亿的阿里市值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/203076240162ba3a4fe79689.59238437.jpeg" title="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" alt="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截止目前，距离2020年10月的8000亿美元高点，阿里市值在近两年时间蒸发了大约5000亿美元，折合人民币超3万亿。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>失去了垄断地位的阿里，尽管依旧还占据着国内电商龙头的宝座，但随着各品牌“出逃”，头部话语权的滑落，其他平台也开始逐步崛起，互联网平台经济也开始驶入正常发展轨迹。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此时的阿里，国内电商业务开始刹车。甚至在2022全年财报中，隐约显现出国内核心电商业务开始出现负增长的取势。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>风暴之中的阿里，飘摇不定。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" title="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" alt="风向变了！多地官员密切接触阿里巴巴，释放哪些重要信息？" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>风向转变，</span></strong></span><span><strong><span>平台经济迎来新机遇</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>尽管盛世不在，但传奇仍未落幕。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从国家高层数次提到“支持数字经济合法合规境内外上市融资”可以预见，阿里以及蚂蚁集团已经迎来了转机。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年3月，中国银保监会主席郭树清表示，蚂蚁集团自查基本结束，整改还没结束，有些问题还需要进一步探索。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在6月9日，还有传言称，中国证监会成立工作组评估蚂蚁集团重启上市。尽管在晚间，证监会对该消息进行了辟谣，但言语间略显暧昧。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>证监会表示，“证监会没有进行这方面的评估和研究工作，但我们支持符合条件的平台企业在境内外上市。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，上市失败后的蚂蚁集团一直在积极进行各种合规化改革，包括将“花呗”和“借呗”进行品牌隔离、关闭相互宝、与网商银行切割等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，已自查结束的蚂蚁，整改完成大概只是时间的问题。在可预见的将来，蚂蚁重启IPO将是必然的道路。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，随着政策乌云的消散，高层明确“支持平台经济、民营经济持续健康发展”，阿里的曙光或许就在眼前。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期以来，各地方政要与阿里的频繁接触，可以看出，地方政府已经放下包袱，重回数字经济的快车道。被“晾”了许久的阿里巴巴，自然也回到地方招商引资的名单中。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>随着未来各地方政府与阿里巴巴合作的深化，借助阿里这一头部互联网平台，各地方也能获得更大的经济发展。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在经历反垄断调查之后，目前平台经济领域“二选一”行为基本停止，市场环境不断优化。作为互联网数字企业的领头羊，阿里与各地方展开合作，也是更好发挥自身平台效应、促进我国平台经济健康发展的典范。</span><span></span></p>
