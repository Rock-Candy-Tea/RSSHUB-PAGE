                        <h2 style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; color: #212629; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 18px; white-space: pre-wrap; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong style="box-sizing: border-box; font-size: 15px;">一、前言</strong></h2>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #000000;">橘先生曾做过一些企业部门的咨询顾问，也曾负责过不少公司的直接经营管理，有些是管理公司整体板块发展，有些是负责市场或运营板块。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #000000;">其中的不少公司大多为中小型企业，甚至有一些是初创型公司。它们有一个共同的特点：业务形态比较单一，更别提所谓的商业架构和产品框架了。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #000000;">这些规模不大的公司一般依靠着比较单一的产品业务支撑运营，说起它们的商业架构，可能熟知之后仅仅发现其在某条流量投放渠道上有着不错的投流效果和销量。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #000000;">这往往也意味着，这家企业的抗风险力不强。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #000000;">当直面一个现实问题：如果这条渠道下个月不灵了，你该怎么办？</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 14px; color: #000000;"><img style="box-sizing: border-box; border-radius: 5px; max-width: 100%;" src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62ee205ad5ae8c160338.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; color: #000000; font-size: 15px;">大多数人听到后，便久久沉默了。当你的主要流量渠道下个月不灵了，导致产品销量受阻，这时你还有其他方式支撑企业经营么？如果没有，应该怎么办？</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; color: #000000; font-size: 15px;">企业经营不像是玩电商爆品，广告打一枪换一地的方式。商业本身是一个千变万化的利益场，每一个行业都有对应的体量大小，而且时刻随着环境、政策、资本风向、消费者群体需求、渠道规则和运营策略等内外部因素影响。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #000000;">如果是因为一些运营策略方面的问题，倒是可以快速调整、优化。但是，如果是投流平台自身流量或规则的变化，亦或者是渠道的直接限制，你该如何处置呢？</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; color: #000000; font-size: 15px;">说了这么多，让我们回到刚刚的问题，它应该这么理解。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #000000;">如果你引以为豪的渠道下个月不灵了，你该怎么办</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; color: #ec742c; font-size: 15px;">我怎样才能利用现有资源，快速搭建一个足够稳定且有前景的商业模型</span></p>
<h2 style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; color: #212629; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 18px; white-space: pre-wrap; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong style="box-sizing: border-box; font-size: 15px;">二、找对你的“战场”</strong></h2>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #7f7f7f;">定位要从一个产品开始。那产品可能是一种商品、一项服务、一个机构甚至是一个人，也许就是你自己。但是，定位不是你对产品要做的事。定位是你对预期客户要做的事。换句话说，你要在预期客户的头脑里给产品定位，确保产品在预期客户头脑里占据一个真正有价值的地位。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; color: #7f7f7f; font-size: 15px;">——定位理论，AI Ries ＆ Jack Trout</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">在搭建一个足以支撑企业运营的商业模型前，先要做好定位。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><img style="box-sizing: border-box; border-radius: 5px; max-width: 100%;" src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62ee205b046bac160339.jpeg" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">其实很好理解。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">这就像上学时选同桌一样，虽看似不起眼，但可能影响一位孩子的一生。读书时，我就发现每当成绩不好的学生调位置到学霸们身边时，那些成绩不佳的同学成绩便能突飞猛进。也很好理解小时候的，在自身尚未形成行事准则时，外部环境对自身影响极大。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">这正如初创团队或转型期的企业，在没有快速找到适合位置时，亦或者是不能适应市场变化，快速地建立自身产品/渠道网络时，所交出的成绩一般不会太好。当找到合适的“位置”后，有了发展腾飞之机会。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #000000;"><img style="box-sizing: border-box; border-radius: 5px; max-width: 100%;" src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62ee205b1f761c160339.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">定位是对重要利益相关方的战略表达，向他们更清楚地、更快捷地传达“我是谁”。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">定位的输出有很多种对外表现形式，例如品牌层面的CIS体系，其中就包含对外的战略层面（MI）、市场层面（BI）、品牌视觉层面（VI）等信息。再更细节一步，品牌口号（Slogan）、品牌故事、品牌IP形象等等。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">首先，我们要明确</span><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">定位本身是一种说服杠杆</span><span style="box-sizing: border-box; font-size: 15px; color: windowtext;">，是对</span><span style="box-sizing: border-box; font-size: 15px;">重要利益相关方的战略传达。那么，这重要利益相关方是谁呢？</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;"><img style="box-sizing: border-box; border-radius: 5px; max-width: 100%;" src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62ee205b3fa09c160339.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">对内：合伙人、团队、管理及员工</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; color: #ec742c; font-size: 15px;">对外：渠道网络相关人、客户、投资人</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">我们再回顾本节摘要中“定位之父”特劳特说的话。可以发现，文中的“客户”一词，并非单指大众消费者。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">定位是找到企业现在与将来的正确位置，它的目的就是【让品牌在现在与将来的竞争中，能在目标群体创造出一丝好感】和【让品牌更容易被识别、认知】。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">它可以使得组织内部保持步调一致，同心同力提升口碑、品牌等软实力，产品、市场等硬实力。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">也能让企业产品在目标消费者群体中有着清晰、明确的吸引力和感知力。人们在看到你的产品时，便能联想到某个使用场景，辅以功能、服务等，降低决策壁垒。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">并且，在经销、物流、渠道、政府等合作网络的相关方处，能清晰展示企业的对外形象和形象/资金等“软硬件”实力。方便企业在扩张、采购、招商等行动中，争取更大的话语权。</span></p>
<h2 style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; color: #212629; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 18px; white-space: pre-wrap; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong style="box-sizing: border-box; font-size: 15px;">三、开启“全面战争”</strong></h2>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">企业清晰自身的定位后，一定要逐步地、全面地搭建自身的品牌阵地。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">请别误解这里的品牌二字，这里并非指的是建议企业主们去砸品牌知名度。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">正所谓“居安思危”，橘先生建议的是企业在完成品牌定位时，即统一了内外部的整体框架，就要开始完善企业</span><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">所能触达的</span><span style="box-sizing: border-box; font-size: 15px;">、</span><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">所有有价值的</span><span style="box-sizing: border-box; font-size: 15px;">渠道网络（品牌阵地），让企业发展更为长久！</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;"><img style="box-sizing: border-box; border-radius: 5px; max-width: 100%;" src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62ee205b6538cc160339.jpeg" referrerpolicy="no-referrer"></span></p>

<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">对内板块中，主要包含了企业的【产品研发】、【采购制造】、【运营发展】等要素；</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">如果把上述的<a href="https://www.niaogebiji.com/tag/%E6%B8%A0%E9%81%93%E5%BB%BA%E8%AE%BE" target="_blank">渠道建设</a>细节通讲，估计这篇文章会变成数十本书了。所以，橘先生总结了以下几个重要板块，这些是企业建设全面渠道链路的基石。下一章，让我们来试着拆解一下这些渠道和阵地，以案例的形式分析，方便读者一同思考。</span></p>
<h2 style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; color: #212629; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 18px; white-space: pre-wrap; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong style="box-sizing: border-box; font-size: 15px;">四、“战术”浅析</strong></h2>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">产品研发板块</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #7f7f7f;">2010年，李宁将Slogan改为“Make the change（发生改变）”，将目标消费群体聚焦于年轻群体；但不久，因年轻人崇尚潮流，更喜欢国外品牌，销量快速递减。2012年李宁开始大量裁员关店，年亏损近20亿。当时的年轻人更多喜欢以阿迪达斯、Nike为首的海外品牌。当时，年轻人对于李宁的概念，不过是老土的国产运动服装品牌，而且还死贵。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; color: #7f7f7f; font-size: 15px;">2016年，李宁重新回归、扭亏为盈。文化自信，也被年轻人们提上心头。在产品被市场接受的同时，李宁的Slogan也变回了20年多前的“一切皆有可能”。直至现在，国潮正是流行时，李宁也被称为“国货之光”而大放异彩。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">产品研发板块除了研发本身，还涉及到供应商管理、原材料采购、研发团队管理/合作、产品管理（出品/发行）、产品市场等内容。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: windowtext;">随着企业的类型不同，产品本身的定义也不尽相同。例如，广告公司的产品指的可能是其团队、往期案例、渠道资源等。但无论如何，企业主应</span><span style="box-sizing: border-box; font-size: 15px;">该时刻记得，如果想企业想长久发展，应当把产品当作企业之基石。</span></p>

<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">营销本身跟随着市场而变化万千，需要足够专业的团队才能减少错误。要牢记每一次出错，都可能成为企业的一次危机和用户拷问。以下是21年时期，橘先生玩的一款游戏《剑与远征》，莉莉丝用了200万做了次周年庆直播活动，然后出现活动问题，导致大氪长和老玩家的暴怒，预计当时损失大几千万。大家可以引以为戒。</span></p>

<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">售后物流体系是企业服务的保障系统。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">随着现代物流行业的发展及人们对服务质量要求的提升，经销商、客户/用户对物流配送服务和售后质保等也提出了更多、更高的要求。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">搭建售后物流体系并非一件轻松的事情，也有很多企业为此犹豫，是否要投入资源，甚至多招几个人都不愿意。原因也很简单，这两个部门都不赚钱，人员精简，能砍则砍。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">这种观念是极其错误的，可能小公司没办法组建较之大手企业相比健全的售后品控体系和物流配送服务。但也应当尽力而为，</span><span style="box-sizing: border-box; font-size: 15px; color: #ec742c;">让服务团队更用心、专业，让客户更放心、安心！</span></p>

<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">相当于战场上友军的“后勤部”，管的是后勤、人事。企业在快速发展的过程中，避免不了频繁的人员流动和调整，后勤部门应当及时支撑企业用人部门的需求，以免出现用人缺口。</span></p>

<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">·绩效管理</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">·薪酬福利管理</span></p>

<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><strong><span style="box-sizing: border-box; font-size: 15px;">公共关系板块</span></strong></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; color: windowtext; font-size: 15px;">开题不得不提一句，在我曾经合作的Boss，这个板块经常被一些小老板所忽视。他们并非不关心客户对他们的评价，而是更关心手中的在即利益。企业本身的用户体量就不太大，自然也不会特别关心公关关系的言论。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">公共关系板块，你</span><span style="box-sizing: border-box; font-size: 15px; color: windowtext;">可以理解说Strategic Communication（战略性传播）。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: windowtext;">从大的来说，它来源自品牌CIS体系，是一种企业为主导策划的形象传播，它向特定目标受众群体传递特定信息，并施加施加影响。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: windowtext;"><img style="box-sizing: border-box; border-radius: 5px; max-width: 100%;" src="https://cors.zfour.workers.dev/?http://yingshou-1308662638.cos.ap-beijing.myqcloud.com/images/ccw62ee205c70feec160340.jpeg" referrerpolicy="no-referrer"></span></p>

<h2 style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; color: #212629; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 18px; white-space: pre-wrap; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong style="box-sizing: border-box; font-size: 15px;">五、总结</strong></h2>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">回想起写这篇文章的初衷，是橘先生在从业这么多年来，也认识了不少企业管理者和实际控制人。在与他们交流的过程中，发现不同时期的企业所注重的点不一样，但是有一件相同的事情却皆有发生。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">那就是，这些企业在转型期间的迷茫和犹豫，迷茫的是方向和未来，犹豫的是要不要打破过往，重新管控/调整方向、渠道，乃至是一些对外话术和材料。不少管理者并非是企业管理/运营出身，他们可能专精于其他方向，比如技术、研发等。所以，在市场的把握上，可能出现强差人意的结果。</span></p>

<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">例如：销售与市场、市场与渠道、渠道与物流、物流与售后、售后与运营，等等、等等……</span></p>

<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">诚如本篇文章所言，我们可以看见一个理想化的公司是什么样的。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">但是，没有一定规模的企业，无法养成这么大的组织团队也是事实。所以，在企业发展的过程中，企业会与一些外部团队产生合作。例如：广告平台的投流团队、电商代运营团队，甚至是品牌视觉团队和营销团队。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ef8b2c;">在合作的过程中，企业的各部门角色一定要以甲方的思维思考合作结果。如果出现合作效果不佳的情况，要及时协助其调整，而不是放任自由。</span></p>

<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px;">【不要把鸡蛋都放在一个篮子里】，正如1981年诺贝尔经济学奖得主</span><strong style="box-sizing: border-box; font-size: 16px; color: #111111;">James Tobin</strong><span style="box-sizing: border-box; font-size: 15px;">所言。千万别把企业的命脉放在某一个渠道上，企业的经营应该迅速地测试多条渠道，不断完善自身的品牌体系和运营机制。</span></p>
<p style="box-sizing: border-box; font-family: 'open sans', 'Helvetica Neue', Helvetica, Arial, 'Microsoft YaHei', 'Noto Sans SC'; margin: 25px 0px 20px; padding: 0px; color: #000000; cursor: text; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-size: 16px; white-space: pre-wrap; background-color: #ffffff; background-image: none; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="box-sizing: border-box; font-size: 15px; color: #ef8b2c;">企业经营如人之一生，犹如负重前行，不可急于求成。市场的回音才是检测企业商业模式的炼金石，如果自以为稳固，最终可能为此付出不小的代价。</span></p>
