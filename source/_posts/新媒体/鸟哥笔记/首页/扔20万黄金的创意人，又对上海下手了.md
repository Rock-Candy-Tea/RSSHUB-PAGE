<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作者 | 杨阳</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>问大家个问题，你找Tony老师剪发后，会把头发珍藏起来吗？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人真这么干了，还是整成数字藏品，数字珍藏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>说的就是今天的主角，一个疯狂的浙江艺术家——</strong></span><span><strong><a href="https://www.niaogebiji.com/tag/%E6%9D%A8%E7%83%A8%E7%82%98" target="_blank">杨烨炘</a></strong></span><span><strong>。</strong></span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>准确来说，是一个疯狂的创意人。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>究竟有多疯狂呢？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>试过把价值20万的黄金丢江里</span><span>（为啥不扔我家）</span><span>，把头发剃了做艺术画......</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/195472144062bb8a866c9603.11569930.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这段时间杨烨炘又在上海作妖了。</span></strong></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">01</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>把时间拉回20天前，杨烨炘把疫情间留下的花发给剃了<span><span>，</span>据说数量有</span>10万根</span><span>（怎么数的咱也不知道<span>）</span></span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span>杨烨炘多少有点社交牛逼症在身上的。</span></strong></span></span><span><span><span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span></span></span><span><span>你猜后面怎么着，他</span>把头发<span>混<span>合</span>在油画颜料里，花了2个小时，拼出一幅上海</span>东方明珠，取名《新上海东方明珠图》。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/100927942662bb8a869da8a3.27103911.png" width="444" referrerpolicy="no-referrer"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/97148085462bb8a8719d626.09529131.png" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为头发自带黑白灰颜色，<span>他</span>把画板当宣纸，把头发当墨水，远远看上去，<span>成品</span>有水墨画那韵味了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/211074725662bb8a879ae926.47296408.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>最近剃发画作又有后续了。</span></strong></span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这元宇宙和数字藏品现在不是很火吗？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>杨烨炘找上文博数藏合作，</span><span><strong>把这款“东方明珠图”制成数字藏品，</strong><strong>限量发行1400枚。</strong></span></span><span>平平无奇的碎发，</span><span><span>逼格瞬间上来了。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/103082671462bb8a88c5e267.87287872.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>个人感觉用碎发作画，多少有点恶心。不过</span></span><span>但凡是艺术行为，都可以用见仁见智来解答。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/66441951962bb8a8901b384.94926976.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>别不理解这操作，问就是情怀。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>杨烨炘称自己呆上海20多年，东方明珠又是上海地标，想通过艺术行为，传递上海解封离不开上海每个人的付出，就像头顶上的头发。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/166731303262bb8a8943ed17.05613548.png" width="333" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以<span>杨烨炘</span>想用一幅《新东方名明珠图》，寓意上海人聚沙成塔的力量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>果然，故事讲得好，艺术少不了。</strong></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">02</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>费头发</strong></span><span><strong>算什么，</strong></span><span><strong>费黄金</strong></span><span><strong>才心痛。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>去年的世界粮食日，</span><span><strong>杨烨炘就用500克黄金，做了1000粒纯金大米。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>接着做了个疯狂的举动，<span>杨烨炘</span>把一粒粒黄金，扔进上海黄浦江。我不会告诉你还扔到</span><span><strong>下水道、垃圾桶</strong></span><span><span>，只为</span><span>讽刺浪费粮食的行为。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>美其名曰为：</span><span><strong><span>我们不浪费粮食，我们只浪费黄金。</span></strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/37023221762bb8a89bc35b2.93113564.png" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>对了这个黄金大米价值20万，是的你没看错！</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/92589598562bb8a8a863d39.48811481.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这个创意多少是有点费黄金的。</span><span>为了反对浪费<span>粮食</span>，去浪费钱，岂不是有点大病？</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>困惑的不止我一人，在</span><span><strong>#用500克黄金制1000粒米扔黄浦江#</strong></span><span>话题之下，骂声一片。</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没想到“扔黄金”作品，还入围美国NYF纽约国际广告节、美国ADC国际设计大奖。看来</span><span><strong><span>这波炒作带来的声量，确实比20万黄金值得多。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/90028008162bb8a8b72c850.07937527.png" width="444" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">03</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>奢侈品也逃不出杨烨炘的脑洞。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就算你不买奢侈品，你一定被诸如</span><span><strong>“</strong><strong>巴黎世家1.2W的破烂鞋、蒂芙尼1500元的曲别针</strong><strong>”</strong></span><span>热搜刷屏过</span><span>，溢价空间巨大的奢侈品就是“智商税”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>杨烨炘可能看不惯</span>炫富，决定扒了奢侈品的底裤。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>杨烨炘前几年就在写字楼摆摊，竟然<span>100元低价卖</span>LV</span><span>（后悔没早知道）</span><span><span>，喊出</span><span><strong>“LV只值100元，LV只卖100元”</strong></span><span>。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/174919103162bb8a8c2c6254.36228182.png" width="555" referrerpolicy="no-referrer"></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你觉得他亏了，他却要批评狂热消费的现象。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>但最有争议的，还是三胎广告，简直毁三观。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>2021年出了个三孩政策，</span><span><a href="https://www.niaogebiji.com/tag/%E5%A4%A9%E4%B8%8E%E7%A9%BA" target="_blank">天与空</a>找来八位父亲，出演《我支持生三胎》的主题广告，有当老师的、有开网约车的......</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><strong><span>全程努力</span></strong>洗脑一个观念——</span></strong></span><span><strong>“我支持生三胎”</strong></span><span><strong><span>：</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>白天换尿布，晚上辅导作业，半夜喂奶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>生了三胎之后，<span>父母的爱</span>变成三份。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>买不起大房子，那就改造小房子。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不符逻辑，脱离事实、冠冕堂皇.....每句文案堪称迷惑发言。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在<span>看</span>回当时的舆论，就知道很多人并不买单。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>在创意这件事上，天与空对外放肆，对内也很敢玩。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>就拿招聘来说，前段时间</span><span>有人求职天与空，一口气给天与空<span>北上广</span>公司，点了一大波咖啡，还<span>在咖啡底部印求职视频二维码</span><span>。</span></span></span><span><span><span></span></span></span></p>
