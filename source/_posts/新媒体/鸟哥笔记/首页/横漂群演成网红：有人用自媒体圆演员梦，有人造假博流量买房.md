<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“在横店，10个群众演员中有8个人是自己拍视频的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说这句话的老高，也是其中一位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他是个来自东北的00后，毕业后工作了8个月，攒下了点钱，辞职踏上了横漂之路。从出发那天起，老高便开始用镜头记录生活，在网上开了个账号分享，取的昵称也很简单，就叫“老高的横漂日记”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/96488021762f9f186738bd7.34034313.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>来到横店之后，<strong>他发现大部分群演和自己一样，拍戏跑通告的同时，也有着另一个身份——自媒体博主</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为横店从来不缺故事和<a href="https://www.niaogebiji.com/tag/%E6%B5%81%E9%87%8F" target="_blank">流量</a>。这里是全国最大的影视基地，生产制作了国内三分之二的影视作品，群演数量超过20万人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>明星剧组的八卦、底层演员的追梦故事、影视制作的内幕细节，随便拎出一件都能吸引眼球。有时就算拍拍在剧组日常吃的盒饭，也会有数十万人感到好奇、驻足观看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有一种夸张的说法是，横店每走30米就能遇到一个“网红”群演。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新榜旗下数据工具统计，抖音上名称带有“横店”相关关键词的账号至少有628个，其中流量最大的粉丝超过800万，<strong>#横店#话题播放量超60亿次，#横店群演#播放量达到了12亿次</strong>；在小红书、B站、快手等平台，也能找到累计上千个直接以“横店”命名的账号。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/36843641662f9f186d443c1.21875568.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做自媒体，给了群演们除了演戏之外、另一个被公众看到的机会。有了关注后，他们有人获得了更多的角色邀约，有人赚到了收入补贴生活，反哺了自己的群演事业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，流量的另一面也在悄然滋长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>部分尝到了甜头的人开始流水线创作横店群演的日常，他们的视频内容真假参半，不时掺杂些潜规则等争议话题，凭借群演的身份红利吸引眼球，赚得盆满钵满，也给观众制造了许多假象和错觉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>横店故事真的是流量密码吗？我们和几位横漂博主聊了聊他们是如何看待自媒体的，以及真假群演背后的生意经。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>从横店群演到<a href="https://www.niaogebiji.com/tag/%E7%9F%AD%E8%A7%86%E9%A2%91" target="_blank">短视频</a>红人</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在横店，成为一名群众演员的程序其实很简单：只要年龄在18-60周岁之间，完成一个简单的考试，交10元工本费，即可办好演员证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>横店的演员有不同等级设置，刚入门的是普通群演，通过不同的考试可以升级成为前景演员、特约演员等等，待遇收入也各不相同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“横店小乔”是一名特约演员，在2021年3月的时候来到了横店。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/37286009062f9f187257fe8.97484474.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>与外界想象不一样的是，群演并不等于无学历、无技能、空有梦想。小乔是表演系科班出身，她身边的许多横漂朋友也都是大学毕业，甚至有研究生毕业的。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管当年的同学大多放弃了演艺道路，小乔是很坚定自己是要做这一行的，“我资质有限、学历也不是很亮眼，横漂就是给自己在有限选择里找一条最适合的路”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她有做短视频信息流广告的经验，会写文案、懂剪辑，自己拍起视频来也很熟练。到横店之前，小乔就看了些横店博主的视频，<strong>发现最常做的主题是“剧组盒饭品鉴”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盒饭两荤两素或是三菜一汤，偶尔加个鸡腿，在镜头前吃得津津有味。如果吃出了自己的风格，那流量基本都不差。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如B站34万粉丝的“米梦杰”，就是靠吃盒饭的视频成名，热度最高的内容播放量近670万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1652037762f9f187ca24f9.26266624.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以扎堆做吃播，除了简单易操作容易火，也和群演身份的敏感性有关。<strong>横店公会发过一个《演员公约》，要求群演遵守现场规定，不能泄露传播剧照、现场视频等等</strong>，这是他们的职业道德要求之一，吃盒饭的风险性会比较低。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，小乔说自己是个小鸟胃，不擅长吃播。她觉得自己更适合聊天，可以一直对着镜头唠，讲自己的群演经历、剧组工作的感悟，以及粉丝的提问和群演行业的科普信息。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年9月，小乔发了一条与童年男神一起拍戏的视频，播放量达到54万，是账号热度最高的一条内容，也让粉丝数从几百个一路涨到一两万，自此进入稳定涨粉阶段。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>她会很注意不外泄任何剧组相关的关键信息以及八卦，避开一切真实姓名，在主页上也写着“绝不偷拍物料很讲武德”</span></strong><span>。有时候遇到想分享的经历，还会特地延迟几个月再以讲故事的方式说出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管如此，小乔有时也会很忐忑，怕被剧组盯上。因为一些剧组对于群演的分享始终担忧，一位横漂博主提过剧组让自己删除盒饭测评视频，还被群头要求在“拍视频”和“当群演”里二选一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但这一情况也在逐渐改善，越来越多的横店群演开始做起短视频直播后，横店的包容性也越来越强。尤其近两年影视寒冬冲击，横店自己成立了MCN机构，希望借力网红经济，有博主说演员公会也在鼓励群演做短视频。据了解，目前横店所在的东阳市里，MCN机构超过了1000家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>大部分群演做短视频，要么做盒饭测评，要么和小乔类似，以分享剧组生活为主。也有一些人另辟蹊径，从自己擅长的技能出发，同样收获了不少关注。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>朱仙便是如此。他梦想成为一名动作演员，一开始机会不多，就自己拍、自己演、自己剪，从2012年就原创动作短片，也翻拍了一些经典动作电影。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>四年前，朱仙开通了抖音账号，陆续上传自己创作的动作短片内容。2019年，一条翻拍电影《五龙八卦棍》的动作短片走红，获赞超百万次，累计播放量2000多万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/192247131862f9f18880a716.62871946.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>许多观众因此认识了朱仙，有人被短片制作水准折服，找来合作内容，还有人说想拜他为师、学习武术。截至目前，朱仙在抖音已经有115万粉丝，B站粉丝也超过了22万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“还是要做自己擅长的，我知道搞笑内容很吸粉，但动作片才是我擅长的，这样才能做很久”，朱仙这样认为。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>用自媒体收入养活演员梦</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“横店小乔”曾做过一期视频，带观众实际体验横店底层群演的生活状态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>因为很多群演吃不起饭，横店有提供免费斋饭的地点</span></strong><span>，每天一顿，一个包子三个小菜，队伍经常排得很长；<strong>承担不起租房成本的群演便结伴住在废弃楼栋</strong>，楼梯没有扶手甚至没有墙壁，一脚踏错可能就是10余米的落差。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有群演提到自己有戏就进组，但戏并不是想跑就跑、天天都有的，尤其是暑假期间群演多、剧组少，一个星期都没有戏是很正常的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一般情况下，群演一个月能跑十几天的戏，每天100多元的收入，没戏的时候很多人会去工厂之类的地方再找活干。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>横店群演的收入根据演员不同类别有高低差别。普通群演108元一天，前景演员需要露脸，对形象有要求，收入能有220元一天。再往上就是特约演员，没有台词的话是400一天，有台词的话则能有500一天。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但要想通过特约演员的考试也不容易，有网友曾计算过可能五六十个演员里才有一个特约，<strong>大部分群演还是拿着每天100多元的收入勉强维持生计，月收入两三千元</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，吸引他们涌入短视频和直播的最根本动力，是有机会赚到更多的钱。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音14.5万粉丝的“演员凤姐”透露过，直播一个月的收入，不仅能覆盖衣食住行各方面的花销，还能带来不少结余。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小乔说，她怕自己会“饿死”在看见梦想曙光的前一天晚上，而做自媒体支撑了她向前走下去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年上半年北上广深等几个大城市疫情反复，直接影响到了横店的开工，因为主演、导演、编剧、后期团队等大多定居在大城市中。小乔数了下，那几个月横店可能只有七八个组在拍，整个上半年都没什么收入，主要靠上一年的存款，以及自媒体的广告收入，撑到了下半年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“<strong>如果没有这个账号的话，我都不知道我上半年的生活费哪里赚</strong>”，对小乔来说，如果做博主还能赚到钱，那就会做下去。不是为了成为大网红，而是能赚到钱，也让她有机会被剧组看到、收到角色邀约，“<strong>自媒体是为我做演员服务的</strong>，无论是从金钱上还是精神上”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>朱仙也透露，通过自己的账号，有不少人找到他拍宣传片或游戏广告。接下来，他打算在B站发布自己制作的武侠短剧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>朱仙现在的本职工作是动作导演，但目前商业合作广告要比拍电影挣的钱更多一些，“<strong>拍电影时间太长了，从剧本到拉投资到拍摄上映可能要两年，广告短片的周期特别短，从签合同到发布，可能就只要一个月</strong>”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比之下，横漂4个月的老高目前对于收入之类的心态还比较佛系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做群演的工资确实不高，但老高也不急，他说自己还有来横店之前攒的积蓄。做自媒体也没什么赚钱目标，<strong>到目前为止总共收入了平台激励的158元，平均自媒体日薪超过1块钱</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我的想法是即使不成功，大不了就是把积蓄花完，我再去换另一种生活方式，所以没有很大的经济压力”，当然，即使做好了最坏的打算，老高也不打算浪费这些光阴和积蓄，“我还是希望能在横店走更远的”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>当横店日常成为流量生意，</strong></span><span><strong>是真是假重要吗？</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“横漂”逐渐成为一种内容题材后，一些人发现做“横漂博主”似乎要比做“横漂群演”更容易赚钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>博主“爱吃又搞怪的软哥”直接发视频承认，除了去年作为群演参与过两部影视剧，其余关于群演的内容多数是段子，“可能有很多人讨厌我的行为，弄虚作假，<strong>但我就是为了涨粉才去的横店</strong>”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/50825213262f9f1964d8e45.46328575.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大多数情况下，群演跑戏经历风吹日晒，一天能拍摄的素材可能也是有限的，因此一种更高效、流水线式打造横漂博主的方式开始出现——</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只要找汉服店做个古装造型，办个横店景区年卡，从侧面拍几个剧组，再购买盒饭吃，顺便编造一些和男女主搭戏、被导演送花等段子，一两个小时就能完成，还能有不错的话题度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音粉丝超过50万的“云子团团”和她的姐妹团就是在这种模式下批量产出相似的内容。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/145882655762f9f1a33629c4.81497977.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她们最开始确实在横店做过群演，也许是普通群演来钱太慢，做博主后用真假掺半的方式来博取流量，经常假装几个人跑一个剧组的戏，甚至说这场戏只有她们组团入选，然后换上古装装扮，拍一点剧组的布景，再分享演戏时接触男女主的情况，并提及自己被导演看重。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于大部分情况没有真正进组，又不想穿帮，她们会对剧组做出失实宣传，比如某一次在视频里提到扮演青楼女子，被剧组开出了衣服穿得越少、工资就会越高的条件。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/83549763962f9f1a3887663.95894865.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些做法让不少群演感到愤怒，认为她们在搞污名化，多位群演发文揭露声讨这种行为，并指出群演完全可以拒绝过分的戏份或者服装要求，<strong>“她们只是在视频里扮演横漂小演员，好像总会遇到各种潜规则，态度又是嘻嘻哈哈的，有一种割裂感，只为了满足别人的猎奇和娱乐心理”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但这一招确实有用，“云子团团”等在视频里提到过自己在家乡买房，就算是被扒“假群演”时间点前后，还发了一条“转转”App的广告视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，“假群演”事件爆发后，网上也传来了另一种声音——“假装也不违法”，“揭露的人是不是就嫉妒她们火”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说到底，目前并没有任何相关的规定和标准去评判一个横店博主的真假，而且在一些观众眼里，是真是假真也并不是那么重要。只要有流量和利益，就无法避免更多像“云子团团”这样的案例出现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人就在评论中直言：<strong>“上网不就为了看点爱看的，如果没有剧本，就是群演重复枯燥的生活，也没人乐意看。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是个很残酷的现实，多数横漂博主的粉丝量只有几百、几千，多一点的小几万，“云子团团”有50多万粉丝，靠造假在群演里也成为小有名气的网红了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但小乔以及和她一样站出来发声的群演们在乎真假。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小乔想起自己去年有次在剧组熬了4个大通宵，天天早上才收工、下午又开工，几乎在场工作人员都累虚脱了，她说自己走起路来就像魂在身体里来回晃荡，回到家后直接陷入昏迷，隐隐约约感觉家里的猫在身上踩奶撒娇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“这种事情在横漂里很常见，如果她们不愿意吃群演的苦，凭什么去吃群众演员的红利？”</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事态愈演愈烈之后，“云子团团”发视频向公众道歉，如今已经离开了横店、回到了家乡云南，和她一起的姐妹，有些删了大量视频后选择停更，有些重新在剧组里跑起了戏。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/161080667162f9f1a42f0fd0.62940442.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次风波就像一个小插曲，横店群演们的生活又重新回到了正轨，但横店故事逐渐成为流量生意的趋势并不会就此终结。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阻止了一次，能阻止无数次吗？如何去更有影响力地去真实表达横店群演的现状？留给他们的，还有很多问题。</span></p>
