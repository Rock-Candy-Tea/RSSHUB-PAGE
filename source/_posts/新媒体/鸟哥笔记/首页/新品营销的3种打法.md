<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/129576382162d0786f8c1646.52508676.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E6%96%B0%E5%93%81%E8%90%A5%E9%94%80" target="_blank">新品营销</a>，是很多品牌每年都要面临的重大课题。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今市面上针对新品营销的打法，形形色色，单看别人的案例本身，都觉得挺好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但到底哪一种打法才最适合自己本身？对应的打法，应该如何操盘？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这背后其实涉及到很多不同的决定性因素，比如：所处行业、企业资源、品牌调性、产品本身、预算、目的等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这篇内容，我就来以我自己的经验和理解，来测评一下4种常见的新品营销打法。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>打法1: 传播campaign</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是一种典型的我们业内叫<strong>“高举高打”</strong>的打法，应该也是目前为止，至少在传统大品牌里，仍然还是最常见的一种新品营销打法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓传播campaign，一个核心的要点就是，<strong>品牌本身仍然是作为与用户沟通的主体</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>常见的操盘模式为：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌针对此次上市的新品卖点，来提炼一个核心的传播主张；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后再基于此，产出相应的内容物料，比如短视频、明星代言、线下快闪等；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后再去通过媒介来传播品牌制作的物料，比如微博热搜、KOL、朋友圈、信息流、分众等；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>落地到电商站内，可能可以去PK一下平台新品首发的营销IP资源，比如天猫的小黑盒、京东的小魔方、抖音的开新日等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种打法很明显的一个要求就是：<strong>预算要充足</strong>，最少也是百万级别的。所以，假如你手上预算只有几十万，就不太适合。当然我也看过很多品牌在这个预算下，确实给新品拍出了他们认为“有趣”的短视频，但结局就是只是在自己的官微上发布而已，着实是一种浪费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种打法，比较适合那些<strong>决策门槛比较高、周期较长的品牌新品</strong>，比如：大家电、手机、汽车等等。就不是那种看了一个达人种草视频，就会被转化那种。所以，这种以品牌为主体的打法，除了传播作用本身，也是一种<strong>有信任背书的心智播种</strong>。当然，如果你有流量明星资源，或者觉得有一个特别好的传播创意，也可以不局限于产品本身特性的限制。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，这种打法的效果如何呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>坦白讲，带货效果，甚至引流电商的效果都不怎么好，如果最后没有争取到什么电商平台的资源的话。传播效果（曝光量、话题阅读量等），这种就非常虚了，没啥意义。除此之外，唯一可以看看的就是<strong>人群资产运营端的效果</strong>，比如人群资产的体量，以及流转率等等这些。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>打法2: 内容种草</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比起上一种打法，将品牌自己作为传播主体，内容种草则是将KOL达人作为主体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>底层逻辑是<strong>从展现产品功能，转变成讲出用户体验。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么样的新品适合采用内容种草的打法呢？<strong>最好是那些，有新鲜看点的</strong>。不然的话，也能种，就是需要的费用会与之成反比。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如空气炸锅，如果性能和外观都平平无奇，性价比上也没啥优势，这时候也是能做内容种草的，毕竟空气炸锅目前来看，是一个增长很快的趋势品类，总能够享受一点红利，但前提就是你得在像小红书或者知乎这种具有搜索属性的内容平台上铺的笔记量足够，这样当有人来搜索相关品类词或者竞品词的时候，就能够做到拦截，这是一种被动的种草逻辑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而同样是空气炸锅，假如你的新品增加了一个可视化窗口，用户可以看到里头食物是如何慢慢炸的过程，有人会觉得这很治愈。这时候，就有可能成为一个主动拉人来围观的新鲜看点，进而形成口口相传的口碑效应。这是最理想化的种草效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，常见的种草操盘模式是怎样的呢？分享一种比较常见的做法，分成“埋梗——爆梗——挖梗——播梗”这几个阶段：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>-埋梗：</strong>在小红书/知乎/百度等这类搜索平台上，提前埋好用户真实的测评露出，可以批量选一些KOC来，聚焦在使用体验、场景上，最重要的是真实。为后续“爆梗”后，消费者被种草后来看路人反馈做好铺垫。有个要注意的点就是，这里一定要保证用户搜索关键词，是能看到这些提前埋好的内容的，所以很多时候，我们会用产品昵称来做关键词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>-爆梗：</strong>顾名思义，就是要通过量级比较大的资源来引爆这个新品，比如最常见的是明星推荐。但这里的明星推荐和代言是不一样的，最大的不同就是我们要让消费者看了是觉得明星真的在用，而不是商业行为。所以，不需要拍那种高逼格的图、视频之类的，反而是一些生活场景中，无意中透出明星在用这个产品，效果会更好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>-挖梗：</strong>就是围绕明星来八卦背后的产品。这时候，可以选一些八卦号、行业种草号、科普号等等，来最大化挖出这个新品的各种，顺带炫耀一下自己也在用明星都在用的新品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>-播梗：</strong>制造一种用户都在买的测评、晒单的跟风行为。可以批量seeding一些KOC、素人用户来产出内容，以此带动他们身边的用户来了解，实现真正的口口相传。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，我们来讲讲这种软性内容种草，怎么评估效果，怎么设定KPI？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一是看内容种草平台本身的种草关键词搜索量，后台是可以有相关数据监测的；以及搜索品类词/竞品词后，对应品牌笔记显示的权重提升。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二是看电商交易平台内，自然流量中有多少是搜索种草关键词进来的，以及有多少成交是来自种草关键词的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但说实话，除非你的新品本身特别有看点，或者说你铺的量足够大，不然短期内看到的效果都非常一般，特别是如果想马上带来销售效果的，不建议做。这种打法，<strong>更适合靠内容的长尾效应累积叠加。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么想要既输出种草内容，又能带来短期效果，怎么办呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看第三种新品营销打法。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>打法3：内容种草+效果投流</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是在抖音生态孵化爆品的核心打法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>操盘模式，主要分为“假设——测试——优化——放大”这么几步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>假设：</strong>这个的源点在于回答要做什么样的内容？很多时候，我们的新品面向的人群绝对不止一类，比如“精致妈妈”是、“新锐白领”也是，那种草的内容应该往哪类人群去做效果会更好呢？即使确定某一类人群，但这类人群兴趣点那么多，内容应该往什么样的场景去发散呢？（所以基于此，会发现种草打法，需要的远远不止一条品牌官方TVC这么简单，而是可能需要几十上百条。）这时候，我们可以按照“人-货-场”的维度，来加以假设，除了“货”是不变的，“人”和“场”都可以列出若干条，比如：“精致妈妈”对应的场景1内容、“精致妈妈”对应的场景2内容、“新锐白领”对应的场景1、2、3等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>测试：</strong>这个最主要是通过沉淀自然投放数据，来测算出优质的内容模型。我们会基于前面假设的各种“人-货-场”组合，来产出对应的短视频内容，然后把这些内容分别投出去，最后看看哪些是比较好的，哪些是要做后续优化的。这里再补充一个知识点，对于以ROI为导向，偏重于效果的种草短视频，有一个内容法则：前3s-【吸睛力】、5-20s开启【信任力】、20-50s开启【诱惑力】、最后3-5s开启【行动力】。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>优化：</strong>根据自然投放的效果，做包括两个方面的优化。一是短视频内容层面的优化，可以在后台看到每一秒的观看量、转化量、流失量，然后根据曲线波谷的地方做脚本的删减或者顺序调整；二是投放人群层面的优化，包括云图的品牌人群和莱卡的行为标签人群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>放大</strong>：最后针对优质内容，通过流量产品进一步放大。包括抖音的四大流量产品：Dou+、内容热推、达人竞价和信息流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>疫情环境下，选择第3种打法的</span><span>应该会越来越多，每一笔预算都要有回响。</span></p>
