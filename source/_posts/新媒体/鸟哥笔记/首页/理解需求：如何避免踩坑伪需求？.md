<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很多公司市场部的人，在做推广的时候忙着各种活动，凑热点，搞直播，做快闪店，甚至忙着为客户买月饼，却忘记在推广时真正重要的--<strong>理</strong><strong>解用户的需求</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">殊不知，如果需求找到了，但是不适合自己的产品，铺再多的钱到广告渠道，也根本无法打动消费者。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要找到用户的需求，这基本是所有做商业的人的基本原则。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">任何一款产品，要成功地被推向市场，在它诞生那一刻起就是要帮助用户解决问题。而<strong>需要借助某个工具来达成目的，这就是用户的需求。</strong></p>
<p><span style="color: #7f7f7f;"><strong><strong style="max-inline-size: 100%; margin: 0px; padding: 0px; color: #000000; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important;"><span style="-webkit-max-logical-width: 100%; margin: 0px; padding: 0px; font-size: 14px; box-sizing: border-box !important; outline: none 0px !important;">"</span></strong></strong><strong><span style="-webkit-max-logical-width: 100%; margin: 0px; padding: 0px; font-size: 14px; box-sizing: border-box !important; outline: none 0px !important;"></span><span style="max-inline-size: 100%; margin: 0px; padding: 0px; font-weight: normal; caret-color: #ff0000; font-size: 14px; box-sizing: border-box !important; outline: none 0px !important; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif;">这不就是最基本的产品原则嘛？我不可能不知道啊</span><strong style="max-inline-size: 100%; margin: 0px; padding: 0px; color: #000000; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important;"><span style="-webkit-max-logical-width: 100%; margin: 0px; padding: 0px; font-size: 14px; box-sizing: border-box !important; outline: none 0px !important;">"</span></strong></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>知道要找到需求对于绝大多数人并不难，难的是怎么判断你找到的产品是不是真的适合解决这个需求。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">举个例子，员外之前曾经碰到过一个项目，是这个情况:</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; z-index: -1; cursor: pointer; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201906/5GSvX2GZ_npef.jpg" alt="111.jpg" data-ratio="0.5625" data-w="1200" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span style="caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif;">(图: 非项目原图，仅供参考)</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">某著名IT公司老员工出来创业，伴着当时共享经济的热度拿了一笔投资，搞共享书报亭服务。他们想服务那些高收入，高教育程度的用户，让他们可以在回家前选一本想读的书，读完再放回来。里面的书更新的很快，种类也多。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本来这个团队有着成功推广的一切条件：团队执行能力强，资金充裕，文案，价格都OK，<span style="caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif;">甚至很有针对性一下子铺开整个城市高端小区的几十个网点，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>最终这个项目在一年多以后惨淡收场，问题出在哪里呢？</strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>这个项目找到了用户的需求，但却不是合适需求。</strong></p>
<p><span style="font-size: 14px; color: #7f7f7f;">"为什么这么说呢？读一本不期而遇的闲书不是很小资的事情吗？"</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一般来说，一个合适需求具有以下特征：</p>
<ol class="list-paddingleft-2" style="margin: 0px; padding: 0px 0px 0px 30px;">
<li>
<p>用户想要完成某个任务</p>
</li>
<li>
<p>他现有的能力和工具不足以帮他完成这个任务</p>
</li>
<li>
<p>你提供的产品可以帮他完成这个任务</p>
</li>
<li>
<p>你在他可以选择的工具里面是最优解</p>
</li>
</ol>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">大多数讲需求的人对前三点都有很直观的认识。但是很遗憾，对于很多产品来说，决定一个用户买不买账的反而是最后一点--<strong>你的产品必须是最优解</strong>。</p>
<ul class="list-paddingleft-2" style="margin: 0px; padding: 0px 0px 0px 30px;">
<li>
<p><span style="caret-color: red;">比如消费者需要钉一颗钉子，你说我可以卖他一把锤子。锤子的确可以完成这个任务，但未必是对于消费者的最优解。因为他家里可能有一块砖头直接把这个活干了。</span></p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="caret-color: red;"><img style="vertical-align: inherit;" src="https://image.135editor.com/files/users/379/3794874/201906/mwZHTcvz_kq2K.jpg" alt="hammer.jpg" data-ratio="0.5616666666666666" data-w="1200" referrerpolicy="no-referrer"></span></p>
<ul class="list-paddingleft-2" style="margin: 0px; padding: 0px 0px 0px 30px;">
<li>
<p>就好比你的女友感冒了，你跟她说多喝热水多休息，这或许是有用的，但是她的最优解可能是吃一片维生素C或者感冒片，这样更快好起来。</p>
</li>
</ul>
<ul class="list-paddingleft-2" style="margin: 0px; padding: 0px 0px 0px 30px;">
<li>
<p>同样，对于上面的共享书报亭项目，的确可以满足用户需要看书，并且为随即遇到的好书有期待这个需求。但是为了同样的需求，用户可能更愿意用读书APP。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在大多数情况下，当用户想要完成一个任务，他们可以借助的工具可以列一张清单，<strong>如果你的需求是合适的，你的产品应该排在第一位</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就意味着，很多产品的失败，原因是<strong>用户的确有这个需求，但却不买你的产品来满足这个需求，这种就不是合适需求。</strong></p>

<p><span style="font-size: 14px; color: #7f7f7f;">"为什么很多产品和创业项目找不准用户的合适需求呢?"</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为想要找到一个任务的解决方法很容易，但是换位思考想到用户在决策时的真实情况却很难。而<strong>定义合适需求的时候很多人往往会跌入两大陷阱--证实偏见和简化信息。</strong></p>
<section style="margin: 10px 0px;"><span style="color: #ffffff; font-size: 18px;"><strong><span class="135brush" data-bcless="darken" data-brushtype="text" style="border-bottom: 4px solid #787461; border-top-color: #787461; border-right-color: #787461; border-left-color: #787461; border-radius: 3px; line-height: 36px; margin: 0px 7px 11px 0px; padding: 5px 10px; text-align: center; background: #aaa695;">证实偏见陷阱</span></strong></span></section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所谓证实偏见，是人类的基本偏见之一，讲的是<strong>我们思考一个问题的时候，会有倾向于寻找证明自己最初想法的偏向，从而产生隧道视野效应</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如常常有些男生在网上，列出了暗恋对象的一大堆行为，然后问网友这女孩对自己是不是也有意思（网友戏称人生三大错觉之一），这个就是典型的证实偏见。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为他们从心底里觉得人家对自己有意思，在日常生活中就放大了很多对自己示好的现象，而忽略更多两人在接触时的普通细节。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为什么很多产品容易陷入这个陷阱？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现在的产品之所以出现，通常是设计者在日常生活中<strong>观察到了某个需求，然后就想出一个解决需求的工具。</strong>为了求证这个产品是否有用，他会开始观察生活中的各种现象--<span style="caret-color: red;">当你手上拿着锤子，看什么都是钉子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最后他<strong>报告</strong><strong>了很多潜在的应用场景，结果大部分场景他的产品都不是第一选择</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">忽略应用场景，其实也是导致很多产品最后不成功的深坑之一，甚至连谷歌都栽过在它手上。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201906/4OaRj3rh_Bebs.jpg" alt="google-glass-macro.jpg" data-ratio="0.5618181818181818" data-w="550" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="caret-color: red;">Google Glass刚刚发布的时候，强调的是不间断地连接世界，并记录身边的美好，甚至《时代》周刊还把它列为年度最佳发明。当然谷歌也列出了很多理想的使用场景，比如VR，记录出生的婴儿慢慢成长的过程，让你不需要辛苦记笔记等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但随之而来的尴尬就是，这个产品很多的使用场景是用户不想要的，甚至会带来巨大的隐私侵犯问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如在Google大会上，一个与会者进入厕所发现其他的人都戴了Google Glass，也就是自己上厕所的过程被几个摄像机同时记录。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">到后来，拉斯维加斯的赌场，各大电影院的影厅，各种表演现场都明确规定禁止佩戴Google Glass进入。这些生活中精彩的场景都把Google Glass拒之门外，，它又怎样记录美好呢。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，当你为产品寻找合适需求，<strong>不单单要考虑功能性，还必须要定义到场景。</strong></p>
<section style="margin: 10px 0px;"><span style="color: #ffffff; font-size: 18px;"><strong><span class="135brush" data-bcless="darken" data-brushtype="text" style="border-bottom: 4px solid #787461; border-top-color: #787461; border-right-color: #787461; border-left-color: #787461; border-radius: 3px; line-height: 36px; margin: 0px 7px 11px 0px; padding: 5px 10px; text-align: center; background: #aaa695;">简化信息陷阱</span></strong></span></section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为了让工作更高效，很多时候企业在做调研，分析的时候都有简化信息的倾向，比如把顾客标签化，或者引用第二手资料，而不是一手观察。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这样做的好<strong>处是可以很好地维持成本，并且观察行为，坏处是很容易受到样本的限制，忽略了其他重要的维度</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上世纪50年代，麦斯威尔咖啡公司为了降低成本，尝试在产品中混进去一些成本低但口感较差的罗布斯塔咖啡豆。为了探究这个改动是否会影响顾客对咖啡的口感，他们找了部分忠实顾客，进行感官测试。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201907/zFtgVxHa_YyYm.png" alt="maxwell.png" data-ratio="0.5560578661844484" data-w="1106" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">结果显示，即使罗布斯塔咖啡豆的含量增加，这部分忠实顾客并没有感觉出配方变了。于是麦斯威尔公司开始放心地增加罗布斯塔咖啡豆的用量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">直到1964年，麦斯威尔咖啡公司的销量出现了断崖式下滑。后来经过复盘，发现忠实顾客的数量其实没有太大变化。但是由于咖啡口味的下滑，新顾客的数量下降了非常多。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很多时候我们会通过统计学调研的方法研究顾客的需求，但是<strong>越抽象和简化的数据，就越容易让我们忽略重要的问题。</strong></p>
<section style="max-width: 100% !important; box-sizing: border-box; width: 100%; text-align: center;" data-width="100%">

<section style="box-sizing: border-box; width: 100%; height: 1px; border-top: 1px dashed #9b9695; max-width: 100% !important; overflow: hidden;" data-width="100%"></section>
</section>
<section style="padding: 0px 6px; box-sizing: border-box;">
<section class="autonum" style="display: inline-block; padding: 0px 6px; background-color: #def6fa; line-height: 22px; color: #7a7a7a; box-sizing: border-box;" title data-original-title data-num="3">4</section>
</section>
<section style="box-sizing: border-box; width: 40px;">
<p style="max-width: 100% !important; box-sizing: border-box; width: 100%; height: 0; border-top: 1px dashed #9b9695;"> </p>
</section>
</section>
</section>
</section>
<p><span style="font-size: 14px; color: #7f7f7f;">"好了员外我知道要避免犯什么错误，但是还是回到那个问题，怎样判断我的产品是不是用户在场景里的第一选择呢?"</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要想知道你的答案是不是最优，首先要回到问题本身，也就是理解其实消费者要完成的任务到底是什么。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">大部分人其实<strong>对问题的观察是不足的</strong>。换言之，你<strong>想要在解决办法的清单上排第一，但是你连清单上具体有什么都不清楚，这没法思考。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif;">究其原因，是因为</span><strong><span style="caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif;">很多时候</span><strong style="max-inline-size: 100%; margin: 0px; padding: 0px; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important;">你观察了需求，却不理解为什么有这个需求</strong><span style="caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif;">。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如上面的例子，你看到消费者对着一个钉子发愁，于是你就递过去一个锤子了。且不说可能他的房子里有诸如砖头的替代品，你知道他打这个钉子的目的是什么吗？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201906/9vNZkDIw_AexN.jpeg" alt="pexels-photo-268411.jpeg" data-ratio="0.5575" data-w="1200" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果说他的目的是要挂一幅画，那么把打钉子就不是唯一的方法--他还可以用胶水，粘土贴，甚至买一个装饰柜然后摆在上面。这是盯着钉子的人不会想到的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这时候与其卖给他们一个锤子，还不如卖一个吸附在墙上的挂钩，毕竟还不用在墙上打洞。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这意味着<strong>要理解用户的需求，不能盯着他们现在手头上的任务，而是要退后一步，看看他们真实的目的</strong>。</p>
<section style="max-width: 100% !important; box-sizing: border-box; width: 100%; text-align: center;" data-width="100%">
<section style="box-sizing: border-box; display: inline-block; width: auto;">

<section style="box-sizing: border-box; width: 100%; height: 1px; border-top: 1px dashed #9b9695; max-width: 100% !important; overflow: hidden;" data-width="100%"></section>
</section>

<section class="autonum" style="display: inline-block; padding: 0px 6px; background-color: #def6fa; line-height: 22px; color: #7a7a7a; box-sizing: border-box;" title data-original-title data-num="3">5</section>
</section>
<section style="box-sizing: border-box; width: 40px;">
<p style="max-width: 100% !important; box-sizing: border-box; width: 100%; height: 0; border-top: 1px dashed #9b9695;"> </p>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们思考自己的产品到底是不是最佳选择--这个问题的答案就是就蕴含在"<strong>最优解</strong>"这三个字里面。</p>
<section style="margin: 1em 0px; color: #333333; font-size: 14px; line-height: 20px; clear: both;">
<section class="autonum" style="display: inline-block; padding: 5px 10px; color: #fefefe; border-top-left-radius: 3px; border-bottom-left-radius: 3px; background-color: #31859b; box-sizing: border-box;" title data-original-title><strong>1</strong></section>
<section class="135brush" style="display: inline-block; padding: 5px 10px; color: #fefefe; border-top-right-radius: 3px; border-bottom-right-radius: 3px; background-color: #92cddc; box-sizing: border-box;" data-bgless="lighten" data-brushtype="text" data-bglessp="20%"><strong>什么是"解"-- 从任务出发</strong></section>
</section>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">为什么理解消费者面临的问题这么重要？因为每一个消费者看似简单的需求背后，都有着复杂的心理活动。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"> </p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">很产品推广失败的原因，都是<strong>没有了解消费者行为背后的心理活动，单纯把消费者都当成理性人。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">四位来自美国、瑞士的经济学家和行为学家做了一个超大型实验。用三年的时间，在全世界355个城市里丢掉17000多个钱包。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在实验里，研究人员会在不同的城市，在不同的场合把一个钱包递给前台工作人员，跟他们说在地上捡到了钱包，让他们处理一下。所有的钱包里都有一个名片，一个购物清单，还有一把钥匙。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201907/XHqZw6qF_eENE.jpeg" alt="pexels-photo-1174750.jpeg" data-ratio="0.5625" data-w="1200" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">唯一的变量是一部分钱包里有没有钱，而在另一部分的钱包里，他们放了价值13美金的当地货币。在接下来的100天里，实验人员记录了邮件里面收到的来信，看看有没有人返还钱包。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">结果发现，那些拿了空钱包的人，有40%把钱包退还。不过拿到有现金钱包的人，有51%都退还了钱包。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">后来研究人员把钱包里的钱增加到100美金的当地货币，结果退还钱包的人更多了，达到超过60%。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">通过回访，研究人员发现他们<strong>之所以退还钱包，是因为不想有"自己是小偷"的感觉，钱包里的钱越多，这种感觉就越强烈。</strong></p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"><strong>如果把目光放到一个产品，很难理解一个消费者居然会因为非理性的心理，而放弃100美元的福利--而这也是很多产品不断降价，但是销售增长并没有很明显的根本原因。</strong></p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">理解了消费任务背后的心理，就能在同类产品中获得竞争优势。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">这看似很困难，但克里斯滕森教授给了我们一个很有用的视角--就是从完成任务的角度理解顾客要解决的问题。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="caret-color: red;">他在哈佛商业评论在2016年的一篇文章就有提过--</span><strong style="caret-color: red;">本质上来说，用户买一个产品就代表他们雇佣这个产品帮助完成任务，这次雇佣的效果好了，下次就会再找上门。</strong></p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">十年前，在美国的底特律有一家建筑公司销售新公寓。这家公司销售12万到20万美元不等的，高端，奢华设计的小公寓，针对的对象是那些想要搬出家庭住宅或离异的单亲父母的精简型退休人员。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201907/sNmkxQzH_nVgI.jpeg" alt="aprtment.jpeg" data-ratio="0.5625" data-w="1200" referrerpolicy="no-referrer"></p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">尽管公司配备了一个一周工作六天的精英销售团队，但是销量一直不理想。为了理解潜在顾客的想法，公司的创新顾问Bob做了一系列的深度访谈。尽管没能找到最终购买了公寓的顾客，统计数据上的共性，但是他发现在谈话中顾客都提到一个词--<strong>客厅的餐桌</strong>。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201907/T5quWKMS_Etud.jpg" alt="blindside.jpg" data-ratio="0.562610229276896" data-w="567" referrerpolicy="no-referrer"></p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">在美国的文化，客厅的餐桌基本承载了一个家庭的所有记忆。所有有意义的节日，包括圣诞节，复活节，感恩节，还有每天吃饭前一家人手拉手的祷告，都是在家庭的餐桌上发生。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">留意到这个细节之后，Bob终于发现--<strong>他们表面上是一个普通的建筑公司，在卖高性价比的新房子。但实际上，他们在帮助顾客完成生命中重要，也是最后一次的迁移。</strong></p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">理解了这个任务，顾客要解决的问题也就很清楚了：</p>
<ul class="list-paddingleft-2" style="margin: 0px; padding: 0px 0px 0px 30px;">
<li>
<p style="max-inline-size: 100%; margin: 0px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important;">理智上他们卖了旧的大房子，搬到精装小公寓，可以省下一大笔钱，也节省了日常打理房子的时间</p>
</li>
<li>
<p style="max-inline-size: 100%; margin: 0px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important;">但是情感上，等于是和过去一家人生活的幸福时光，美好回忆讲再见，搬出来了就再也回不去了</p>
</li>
</ul>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">Bob最后采取了一系列针对这个心理的措施，比如</p>
<ul class="list-paddingleft-2" style="margin: 0px; padding: 0px 0px 0px 30px;">
<li>
<p style="max-inline-size: 100%; margin: 0px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important;">改变公寓的房内结构，把客厅改大，房间改小留下一个可以放下大的餐桌的空间</p>
</li>
<li>
<p style="max-inline-size: 100%; margin: 0px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important;">提供免费的搬家服务</p>
</li>
<li>
<p style="max-inline-size: 100%; margin: 0px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important;">提供两年免费的储物空间，可以把旧房子的所有东西先储存，慢慢决定哪些留下，哪些扔掉</p>
</li>
</ul>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">这个公司不久之后就把所有公寓全部卖掉了。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"><strong>了解顾客要完成的任务，就能知道他们在解决问题时的心理</strong><strong>。</strong></p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">其实在我们国内市场也有类似的成功例子。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">大概在十年前，有一家来自香港的地产集团到重庆买地建住宅。住宅类似碧桂园，是一个大片区的高端住宅群，包括别墅和高层。但是楼建起来之后，销售情况同样不理想。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; margin: 0px; width: 100% !important;" title="York Count.jpg" src="https://image.135editor.com/files/users/379/3794874/201907/nfVhb9Vu_PMHD.jpg" alt="York Count.jpg" width="100%" height border="0" data-op="change" data-ratio="0.5619266055045872" data-w="436" referrerpolicy="no-referrer"></p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">后来公司发现，来购买这个房子的潜在消费者，很多都是新组建小家庭，需要小房换大房的人。所以对于这个新建的住宅区，<strong>表面的任务是卖高端，质量好的房子，但是实际上是协助顾客组建一个幸福的小家庭，因为房子是每一个幸福的家庭最重要的组成部分。</strong></p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">对于迈进人生的一个重要的新阶段，顾客虽然憧憬，同时也会有对未知生活的恐惧--小孩子的教育，生活中柴米油盐的琐碎，和伴侣父母的相处等等。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">最后这个公司花了两亿引进了当地一个历史悠久的小学，重建了一个非常漂亮现代的校园，还在小区内建了幼儿园。这样的投入带来的回报是一年超过二十亿的销售额。</p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">我们常常说一个产品具有价值。其实价格和内涵值是需要分开看的。我在之前的文章就讲过，一切对于没有内涵值的产品的降价都是无效的。<strong>能帮助产品推广从零到一的，不是"价"，而是"值"。</strong></p>
<p style="max-inline-size: 100%; margin: 25px 0px 20px; padding: 0px; clear: both; caret-color: #ff0000; font-family: 微软雅黑, 'Microsoft YaHei', Arial, sans-serif; box-sizing: border-box !important; outline: none 0px !important; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">从顾客要完成的任务出发，了解他们背后的心理，才知道我们究竟在解决什么问题。</p>
<section style="margin: 1em 0px; color: #333333; font-size: 14px; line-height: 20px; clear: both;">
<section class="autonum" style="display: inline-block; padding: 5px 10px; color: #fefefe; border-top-left-radius: 3px; border-bottom-left-radius: 3px; background-color: #80b135; box-sizing: border-box;" title data-original-title><strong>2</strong></section>
<section class="135brush" style="display: inline-block; padding: 5px 10px; color: #fefefe; border-top-right-radius: 3px; border-bottom-right-radius: 3px; background-color: #b0d676; box-sizing: border-box;" data-bgless="lighten" data-brushtype="text" data-bglessp="20%"><strong>什么是"最" -- 到正确的比较对象</strong></section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">既然是最优解，我们首先得了解清楚产品在和什么做比较，这样才能知道怎样成为"最"。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201907/UXKuckdD_mAuf.jpeg" alt="nice-apples-green-eating-healthy-healthy-food-62687.jpeg" data-ratio="0.5625" data-w="1200" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一直以来，我们很容易踏入一个误区，就是"找比较对象=找市场上其他同类产品"。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如卖运动饮料的，就找其他运动饮料作为比较对象，大不了再加上纯净水，茶饮料就完了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实上，<strong>其他同类产品的确是比较对象之一，但是比较对象却不止这些同类的产品，因为解决一个问题可以有很多的层次。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">下面给大家分享一个例子。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上个月一个朋友来跟我聊天，他正在做<span style="caret-color: red;">一款帮顾客"美白牙齿"的产品，现在做推广，怎么下手呢？肯定是先定位需求，然后看市场和竞品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="caret-color: red;">什么是这款产品的比较对象？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201907/TxR6ZhBj_3kTZ.jpg" alt="breath-brushing-care-40798.jpg" data-ratio="0.5625" data-w="1200" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你估计会说牙膏啊，这不是很显然吗。然后你会惊奇地发现，如果单单着眼与牙膏比较，我们的产品在单价，顾客的认知，渠道上都没有什么优势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那怎么玩？我们可以回头看看，美白牙齿其实并不一定要用牙膏。那么替代方案是什么呢？</p>
<p><span style="font-size: 14px;">如果你是一个消费者，现在找工作，猎头让你发一张现在的免冠照放到简历上，而你发现你之前的照片都不够正式，只能现拍一张。一照镜子，哎呀最近茶喝多了牙齿不够白。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以就有了<strong><span style=";font-size: 17px;">一次性</span>的解决方法：用美颜相机。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么如果不是为了照相，而是需要日常见客户呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了牙膏，消费者还可以选择<strong>不同类类别但是同样功能的解决方法--牙线，牙粉，牙贴等等。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201907/25F7kG2Q_DtJu.png" alt="112.png" data-ratio="0.4138364779874214" data-w="795" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">然后消费者说，我不喜欢化学物质，喜欢天然，健康的方式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">没问题，那么下一个<strong>解决方法--用橘子皮，柠檬汁等食疗</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果要见效快呢？还有<strong>解决方法--到医疗机构进行牙齿美容</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最后如果消费者不需要现在就见效，但想通过长期可持续的方法让牙齿逐渐变白，那么最后的<strong>解决方法--戒烟戒酒，改善刷牙方法</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只是针对"让牙齿变白"这个需求，我们发现原来从效果程度，决策难度可以有这么多不同层次的解决方法。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，当你要列举比较对象的时候，一定要记得完整分析:</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>"消费者的这个需求是出于什么目的？除了同类产品，还有哪些可比较对象，也可以满足同样的需求？"</strong></p>
<section style="margin: 1em 0px; color: #333333; font-size: 14px; line-height: 20px; clear: both;">
<section class="autonum" style="display: inline-block; padding: 5px 10px; color: #fefefe; border-top-left-radius: 3px; border-bottom-left-radius: 3px; background-color: #e36c09; box-sizing: border-box;" title data-original-title><strong>3</strong></section>
<section class="135brush" style="display: inline-block; padding: 5px 10px; color: #fefefe; border-top-right-radius: 3px; border-bottom-right-radius: 3px; background-color: #fac08f; box-sizing: border-box;" data-bgless="lighten" data-brushtype="text" data-bglessp="20%"><strong>什么是"优" -- 在特定场景下进行比较</strong></section>
</section>
<p><span style="color: #7f7f7f; font-size: 14px; caret-color: red;">"我一个产品，要考虑这么多个比较对象，太难赢得竞争了吧？"</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们当然不会用一个产品跟所有比较对象竞争，这是不现实的，这也是我在上面特别提到情景这个考虑因素的原因。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>任何的比较优势，都是以具体的情景和条件为前提的。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1597年，当时的日本实际统治者丰臣秀吉下令进攻朝鲜。在著名的鸣梁海战中，朝鲜名将李舜臣率领十二艘龟船打败了一百三十余艘日舰。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201907/dRzxCYnU_jC7W.jpg" alt="Img406188027.jpg" data-ratio="0.4533333333333333" data-w="750" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要知道当时的日舰无论是体积，数量还是装备都远高于朝鲜船舰。如果在一望无际的海面上，日舰的优势非常明显。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是李舜臣把战场设在了鸣梁海峡，这是一个狭窄，充满礁石，涨退潮明显的海区。在这里，朝鲜的龟船机动力强，进攻手段灵活多样，铁质的夹板和船身反而成为了优势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以任何对产品的推广，关键是找到某个情景，在这个情境下你的<strong>产品的特性可以成为明显的优点</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如王老吉论味道和价格都没有太大优势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果是在夏天逛街很热，想和冷饮的时候肯定不会是第一选择，因为补充电解质的宝矿力，补充能量的红牛，价格低的纯净水等都比他的优势要大。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是王老吉找到了属于自己的场景，就是在饭桌上，顾客吃了上火的菜，喝一罐冷饮解渴顺便防上火，这就是比去医院，或者吃药更优的选择。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="max-width: 100% !important; box-sizing: border-box; vertical-align: inherit; width: 100%;" src="https://image.135editor.com/files/users/379/3794874/201907/53jIIvXF_wCFQ.jpg" alt="TDYH.jpg" data-ratio="0.5620567375886525" data-w="1128" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="caret-color: red;">类似的，天地壹号也是找到了一模一样的餐桌场景。与王老吉的区别是它解决的问题不是上火，而是油腻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">说到这里不禁让我想起了自己的一次经历。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在以前，我从来都没用过漱口水，因为我没觉得用漱口水和每天刷牙两次有什么大的区别。而漱口水的广告最强调的，杀死口腔里面的细菌，保持口气清新也不是很吸引我。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="vertical-align: inherit;" src="https://image.135editor.com/files/users/379/3794874/201907/mPGH6j62_AjXj.jpg" alt="漱口水.jpg" data-ratio="0.5656108597285068" data-w="442" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但就在前几天的一个晚上，我洗完澡刷牙的时候，突然发现牙齿上有几个轻微的菌斑。这时候我脑海里面就只有几个选择：</p>
<ol class="list-paddingleft-2" style="margin: 0px; padding: 0px 0px 0px 30px;">
<li>
<p>第二天去医院看牙医</p>
</li>
<li>
<p>增加刷牙的频率和重点关照那颗牙齿</p>
</li>
<li>
<p>换一种牙膏</p>
</li>
<li>
<p>像消毒水清洁身体一样，用漱口水杀菌</p>
</li>
</ol>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">选项1太耗费时间和金钱，而且我相信大部分人都不喜欢看牙医的体验。选项2不一定有用，选项3需要上网查找相关的产品。相对来说选项4对我就是最优选择，<span style="caret-color: red;">所以我马上下单了一瓶漱口水试试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就是情景下最优解的效果，能让从来没想过用你的产品的消费者都乐于尝试。消费者在做决策的时候通常马上会比较不同选项的<strong>时间/金钱/决策成本，可达性，与已有认知在解决具体问题上的相关性</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>

<section style="margin-top: -1.2em; text-align: center; padding: 0px; border: none; line-height: 1.4; box-sizing: border-box;"><span class="135brush" data-brushtype="text" style="background-color: #0f0f19; border-color: #b7b8b8; color: #ffffff; font-size: 1em; font-weight: inherit; padding: 8px 23px; text-decoration: inherit; font-family: inherit;">总结</span></section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>在很多时候，有效的推广的关键，是找到产品的合适需求。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要找到合适需求，我们要避免信息简化陷阱和证实偏见陷阱，定位到具体的场景来帮助消费者解决问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">美国的一家奶昔公司，在全行业都聚焦在口味改善的背景下，把产品聚焦在更粘稠，配方更健康，并专门在高速公路的出入口开快速通道店。这样的目的是让开车上下班的人能单手吃好有营养的早餐。定位到这个小小的场景却为这个公司带来了巨大的成功。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在找到目标场景以后，我们还需要考虑<strong>产品是不是潜在顾客的"最优解"。</strong><strong>首先从任务出发思考，了解顾客行为背后的心理活动。然后列出比较对象，并确保我们的产品在顾客决策的第一顺位。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就好比去相亲，如果我们自身条件非常好，那当然是上非诚勿扰，让女孩进行全方位，透明的比较是最有利的。但如果自身条件不好，那肯定是找年龄偏大，或者家庭催婚的，然后单独一对一相亲，把实诚，安全感等素质表现出来，成功率最高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只有当你的潜在消费者，在面对某个任务的时候，通过短暂的成本收益核算就能确认你的产品是最好的选择，这样的需求对于你的产品才是合适需求，这样才是"最优解"。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这时候，你需要做的就是：通过推广，让消费者在面临这个任务的时候知道有你的产品这个选择存在，然后躺着收钱吧。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以上，希望你能有所启发。</p>
