<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>俗话说，这个世界上最难的事情有两样，一是把我的想法放到你的脑袋里，二是把你的钱放到我的口袋里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前者是教育，是你怎么背都记不住的abandon。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后者是带货，是任凭带货主播嗓子喊哑，一天播18个小时依旧惨淡的销售额。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>确实都很难。偏偏新东方把两件事都做了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/33617613762b99ea4880429.59980031.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>新东方老师董宇辉在直播间双语带货</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这几天，新东方旗下「东方甄选」直播间靠着「双语<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>」平地起飞，粉丝增长数百万，场均销售额从数万元暴涨到千万元级别。主播董宇辉也被称为主播界的文化担当，喜提五天四次热搜。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/172525001962b99ea4b1ded8.84022070.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>应声而涨的还有新东方的股价，五个交易日里上涨539.82%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/156700970862b99ea4d638f4.38432892.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>双减之后，新东方终于迎来了高光时刻。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近和新东方有所关联的新闻还不止一条。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>12号，罗永浩老师官宣告别社交网络，宣布重新创业，入局AR设备。而靠着直播带货，他的6亿债务预计到年底就能还清了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/76130453662b99ea51a54d4.59950903.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>朋友们，《真还传》要迎来大结局了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众所周知，老罗自己也是新东方英语老师出身，不由得人不多加联想，新东方主播学校，果然名不虚传。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/87053318162b99ea54852c0.33123808.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>为什么新东方做直播带货能成功？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>我个人认为，是因为</span></strong><span><strong>新东方除了是一家教培机构外，其实还具备MCN的属性。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/208638821762b99ea56d7116.22379274.png" referrerpolicy="no-referrer"></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从教育转向直播电商，新东方的业务貌似来了个180度的大转弯，<strong>但其实他们一直在做一样的事情，也就是把经过包装后的员工推到消费者面前。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这批人，以前是名师，现在是主播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>叫法不同，但在新东方的逻辑里，其实没什么不同。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这期内容，我想结合我做up主的一点思考，聊聊新东方现象背后的商业规律。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em><em><span>01、</span></em></em></span><span><strong><em><strong><span>俞敏洪：从网红到MCN</span></strong></em></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1990年，当俞敏洪从北大辞职创办GRE培训班时，第一个老师，也是唯一一个老师，就是他自己。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚开始做GRE课程的时候，俞敏洪是非常痛苦的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/97397572362b99ea5b95953.04308970.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管在大学期间，俞敏洪就以词汇量大著称，后来在北大当了六年老师，更是积累深厚。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但教书毕竟不同于读书。当时俞敏洪每天备课时长就要达到10个小时，但上课时还是常常被学生难倒。为了维护尊严，俞敏洪家里贴满了英语单词，两本朗文现代英汉双解词典更是被他翻烂了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实这和做内容行业有很大相似之处。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一切的输出，都是大量输入的结果。</span></strong><span>如果你积累不够，那焦虑感便会如影随形地缠绕着你，而你需要做的，就是更大量的去摄取信息，去学习。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对俞敏洪来说，情况还要更复杂一点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和所有自己开班，自己教课的辅导老师一样。俞敏洪决定开始单干后，从教研到讲课，从招生到财务，所有的事情都是自己包办的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最大的一个问题，就是学生从哪里来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>俞敏洪的方式是：演讲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/214354653662b99ea5f3d722.20895465.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和后来网课平台经常搞的低价引流课一样，俞敏洪早期也会开免费的课程和讲座，有时候没有场地，就在小学操场开讲座，没有喇叭扩音器就扯着嗓子喊。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着来听免费讲座的人越来越多，多多少少会有一些学生愿意留下来报名。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在不断的演讲和教课过程中，俞敏洪发现，除了知识，学生更喜欢那些好玩的内容，比如俞敏洪自己的励志故事，以及一些幽默的段子。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，俞敏洪就把这些东西都沉淀了下来，成为了新东方老师如今课程风格的雏形。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如果说词汇量和备课的强度，代表了一个网红老师的「内容创作能力」，那么学会演讲，学会怎样讨学生喜欢，怎么吸引学生，就代表一个网红老师的「涨粉能力」。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到这个阶段，俞敏洪已经是一个成熟的网红教师了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过，个体的能力再强，也是有边界的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拿俞敏洪举个例子，GRE不是只有英语，但他教不了数学。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是他请了一个教GRE的数学老师，教三四个班的GRE数学逻辑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>情况看上去不错，但是教了一段时间，就出问题了。这个老师提出来要和俞敏洪分成，而且要分走整个班四分之一的收入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>俞敏洪肯定不干啊。要是这就给你涨收入，那我岂不是被你吃得死死的？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>俞敏洪没有答应，这个数学老师直接跑路了。于是俞敏洪只好跑到教室里，给同学们道歉，退学费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这件事以后，俞敏洪学乖了：我做老师教课，那我的能力就是有边界的，很容易受制于人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而且众所周知，名师其实是不依赖教培机构的。他们可以独立开班，独立授课，更不用担心招生的问题，只要是名师，家长学生总是趋之若鹜的。</span></strong><span>结果就是优秀老师流动性很高，很容易被竞争对手挖走，或者自己离职创业。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>俞敏洪的想法是，如果我能批量培养一些优秀的老师出来，把我自己的教学方法复制到他们身上，那就再也不怕老师离职了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>可以看到，俞敏洪的路径，</span></strong><span><strong>就很像独立kol到MCN的路径。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>既然一个人的能力有边界，那我就把自己的成功经验进行复制，套用到更多人身上，从单打独斗变成机构作战。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样就远比一个老师讲课，有了更大的商业想象力。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><span><em><em><span>02、</span></em></em></span><span><strong><em><strong><span>为什么新东方总能出网红？</span></strong></em></strong></span></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在中国的网红版图里，有一群人，可以被称为新东方系，这批人包括但不限于币圈一代目<strong>李笑来</strong>，做锤子的<strong>老罗</strong>，风投圈内第一网红<strong>徐小平</strong>等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些还都是大佬，像奇葩说马薇薇这个级别的网红，也是只多不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了都在新东方当过老师，如果要概括所谓的「新东方系」网红的共同点，我只能想出七个字：<strong>语不惊人死不休。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这种风格的形成，和新东方还有俞敏洪有着千丝万缕的联系。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前面也说了，新东方一开始是做GRE培训的，早期学生基本都是北京海淀一带的大学生。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在高校还没有扩招90年代初期，大学生还是人们口中的天之骄子，海淀这种名校云集的地方，就更加是学霸横行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>越是好学生，对老师的要求就越高。以新东方早期学生的平均素质，连俞敏洪本人都经常吃瘪，普通高中大学的老师是hold不住的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以俞敏洪基本是手把手培养老师的，老师在课上讲的每道题都要在俞敏洪这里过一遍，讲得不对，不精彩的地方，俞敏洪会亲自指导怎么讲才对。俞敏洪自己的上课录音，也要求旗下的老师反复听课模仿。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在俞敏洪的要求下，新东方老师刚入职，是必须对着逐字稿上课的。</span></strong><span>也就是从进门第一句「Hi Everyone」开始，他们的每一句话都要写在备课稿上，包括要讲的笑话段子和励志故事都要一字不差写进去。有时候一门课的逐字稿能达到几十万字。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，新东方主要上百人大课，学生一多，就容易涣散，老师也管不过来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了提高听课体验，保证学生不会走神，俞敏洪还会要求老师在讲课之外，利用语音语调和肢体语言把学生的注意力拉回来，调动他们的情绪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这里就不得不提俞敏洪的用户思维了。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他很早就立下了学生不满意就可以把老师轰走的规矩，还开创了学生给老师打分的机制。评分会直接影响老师奖金的多少，甚至决定老师的去留。也就倒逼老师不断去优化自己的课程内容。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而优化手段里，最有效的还是在课程里面加段子，加鸡汤。传说新东方的面试内容里，就包括讲段子的能力测试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以你会发现，新东方的老师讲课水平不一定多高，但是肯定不会太无聊。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我以前也上过新东方的课，印象很深的是老师让我们用英语表达「我爱你」，但唯独不能说「I love you」。老师的举例是「my money is your money」，学生的答案也各不相同。这种方式极大调动了班里的情绪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新东方之前，<strong>传统的教育机构，用户其实是家长。</strong>家长的目的就一个：我不管你教学手段，只要能提高考试成绩就行。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而让学生决定老师评分，实际上是把学生看作了用户。其实在英语培训里，这有两个好处：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一是那个年代，家长和孩子之间英语水平差距巨大，老师讲得好不好，家长根本搞不懂，也是从孩子的反馈来判断的。孩子回来说这英语课有意思，家长发现孩子居然喜欢上课，那自然愿意投入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二是因为英语培训市场很大一部分其实是成年人，付费权其实掌握在自己手中。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所以从商业角度上来看，上课不枯燥，让学生听得爽，才能提高学生的付费率和续课率。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>熟悉传销套路的人都知道，俞敏洪那一套授课方法，其实很像成功学里的演讲技巧。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>罗永浩最早是怎么火起来的？不就是靠着「彪悍的人生不需要解释」之类的段子，被学生录音发到了网上，才有了著名的「老罗语录」嘛。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而李笑来在成为中国比特币首富之前，就已经靠着出英语教材，贩卖自己的英语学习方法论出名了。此后更是靠着「和时间做朋友」、「财富自由」等等概念，成为知识付费界的首席镰刀。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/111720961062b99ea6282a51.02914660.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>李笑来</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于新东方早期三巨头之一的徐小平老师，当了投资人更是毫不讳言：</span><strong><em><span>自己就是一个大忽悠。如果谁能用自己的创业故事把自己这个大忽悠都给忽悠了，那肯定值得投资。</span></em></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/29534924062b99ea653f9c5.89819997.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>徐小平</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>可以这么说，从新东方系出来的人，似乎都具备「一张嘴就能把钱挣了」的能力。</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><span><em><em><span>03、</span></em></em></span><span><strong><em><strong><span>会讲故事的孩子有糖吃</span></strong></em></strong></span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>俞敏洪的人生，用四个字形容，那就是「要素过多」，包括但不限于：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>出身贫苦农村</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>高考三次</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>考上北大</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>长相丑陋被女生拒绝25次</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>在校期间肺结核住院一年</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>申请美国留学遭到三连拒</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>被老婆骂窝囊废</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>离职北大创立新东方</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><em><span>新东方上市</span></em></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我可以打赌，翻遍《读者》和《青年文摘》，都找不出比俞敏洪更加生动的励志故事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而俞敏洪本人，也毫不讳言于自己的传奇人生，甚至对他来说，自己的人生故事就是自己竞争力的一部分。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你打开豆瓣搜「俞敏洪」，能找到329本图书，其中光是他挂名作者的非教材类图书就有十二三本，有的是励志演讲集，有的是自传。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/68851429662b99ea6b4b919.25482554.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我列几本书名，大家随意感受一下就好：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>《在岁月中远行》</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>《在绝望中寻找希望》</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>《生命如一泓清水》</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>《挺立在孤独、失败与屈辱的废墟上》</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>《我曾走在崩溃的边缘》</span></em><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使找遍中国商界，你也很难找到像俞敏洪一样那么喜欢出书立传的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/123875990762b99ea707ca21.01644896.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>说到底，讲故事，才是新东方的核心竞争力，也是俞敏洪本人的核心竞争力，更是新东方系网红的核心竞争力。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么讲故事这么重要？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像《人类简史》里说的：</span><em><span>人类之所以能成为万物之王，区别于动物，在于我们能够虚构故事，能够编出没有实际存在的事物，并且我们会或者说能够相信这些不存在的事物。</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>故事为我们简化了世界的复杂性，带来了认知世界的快感，甚至影响了人类的思维方式。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>听故事，是人类刻在基因里的爱好。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>我常说，学习是反人性的，而成功的商人，是不会和人性作对的。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，新东方需要需要励志故事来让学生看到成功的彼岸，需要哈哈一笑的幽默缓解单词、题库带来的痛苦。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但「故事」能做到的，远不止这些。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们关注新东方做直播，是因为它卖的东西便宜吗？当然不是，它一点也不便宜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是因为它双语直播带货能学到东西吗？也不是，看双语教学学英语，我还不如看《三国演义》学三国史呢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>「东方甄选」直播间的背后，是一个完整的故事，是一个教培巨头一夜坍塌，然后又从废墟上重建的故事，是一个年近60的老创业者，半生心血付诸东流后，顽强地从地上爬起的故事。是一个默默无闻的直播账号靠着差异化运营策略，一夜爆火的故事。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同的人，在里面看到的是不同的故事，但毫无疑问，这是一个好故事，好到让新东方可以一夜跃居电商直播行业一线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顺便说一下，我看到的是新东方的老股东们在X音的帮助下，顺利解套的故事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可见，一个好故事，商业价值是按亿来计算的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如《故事经济学》所言：</span><em><span><strong>这是一个故事驱动商业的时代。</strong></span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而当年跟着俞敏洪的新东方老师们，无一例外，都是顶级的故事讲述者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>罗永浩做过传销，摆过地摊，因为老罗语录成名，后来和俞敏洪闹翻出走新东方。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/86496068962b99ea73cf4c3.56060569.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此后他创立牛博网，从演讲《一个理想主义者的创业故事》，到出版《我的奋斗》《生命不息、折腾不止》，在西门子北京总部前砸烂了三台冰箱，高调入局手机，把王自如怼得颜面扫地，再到后来锤子「收购苹果」、「全球第二」的豪言。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6522216462b99ea7675c24.02801653.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来锤子失败，老罗得到了6亿债务和一个行业冥灯的头衔，进入了直播行业，对着镜头露出了他的秃头。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/68279872062b99ea7a0d838.87267116.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>两年后，绝地反击上演一出《真还传》，再次开始创业。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4549698062b99ea7ee7e07.06650664.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些标签，无论是外人强加给他的，还是他自己贩卖的，都成为了他故事的一部分。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然他当年痛骂俞敏洪是「我这辈子见过最没有原则的人之一」。但有意无意，他还是走上了俞敏洪的那条路。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而李笑来也是如此。在新东方时，他是连续七年学生打分第一的托福老师，离开以后，他成为了中国比特币第一人，出书告诉年轻人《把时间当作朋友》才能走上《财富自由之路》。它的知乎live有12万人付费创下历史记录。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/60017768262b99ea82a89c8.24391808.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来「割韭菜」录音曝光，那句「币圈都是傻X，成功就是能忽悠傻X」让李笑来跌下神坛，人人喊打，但他还是出版了《韭菜的自我修养》。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于徐小平，那就更不用多说了，有多少创业者能想出这么个点子，把自己的创业史写成剧本，交给韩三平，最后还能让陈可辛拍出来？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/30228549362b99ea86e2890.46581657.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>《中国合伙人》</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>你会发现，</span></strong><span><strong>从新东方走出来的人，最不缺的，就是戏剧性和话题性，所以也是我们这种自媒体最喜欢和常用的选题。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果要以此来算影响因子的话，新东方应该是国内第一MCN才对。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在分析刘畊宏的</span><span>视频</span><span>里我曾经说过，短视频直播的发展轨迹，其实是对过去电视节目的一个复制，并以此预测下一波风口大概有两个，一是知识类型的直播，一是情感电台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>东方甄选的爆火，也算是印证了我第一个猜想。但这种模式是否能持续下去，还需要观察。毕竟最近连刘畊宏的数据都开始回落了。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但虽然冠以「知识」之名，短视频直播的本质，其实还是娱乐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像我作为所谓知识区博主，从没觉得自己输出过什么知识，只是用另一种方式，给大家解解闷。恰好有一部分人喜欢，就很幸运了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这个角度来讲，我也会经常审视自己过去对一些事物的评价，会不会过于刻薄。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>东方甄选和过去的电商直播有高低之分吗？并没有，只是大家厌倦了叫卖型的直播。等到东方甄选的风格被广泛使用，市场饱和了，可能又会有新的类型来替代。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟学习、工作和生活已经很累了，大家开心最重要。</span></p>
