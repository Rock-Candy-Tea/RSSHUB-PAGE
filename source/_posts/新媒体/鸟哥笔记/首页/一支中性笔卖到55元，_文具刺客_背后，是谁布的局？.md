<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再高端的笔，也变不成马良的神笔。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一支中性笔能有多贵？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当大多数人还停留在撑死2块钱的记忆中时，“一支中性笔55元”的空降热搜，毫不留情地捅破了最后的窗户纸，一时间，“<a href="https://www.niaogebiji.com/tag/%E6%96%87%E5%85%B7%E5%88%BA%E5%AE%A2" target="_blank">文具刺客</a>”的舆论四起。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4995403666317655c8ef143.08231963.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据《扬子晚报》调查报道，记者通过实地走访时发现，目前市场上动辄几十上百的文具用品无处不在。更匪夷所思的是，有很多甚至没有价格标签，等家长结账的时候才发现“不对劲”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4667426096317655cd9b4a5.43056643.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无独有偶，北京商报的记者采访到一家文具店销售人员，据其表示，如果要为一名小学生采购齐全开学所需的文具用品，除了书包外，基本需要花费几百元，如果挑中的是进口和联名款产品，价格还要更高。即便如此，很多家长最终还是会为孩子的喜好买单，毕竟不能因为装备问题而输在起跑线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>秋季限定盲盒、国风治愈系、银河探索礼盒……这些乍听上去来自某奢侈品牌新品发布会的名头，正逐渐成为文具品牌的日常宣传，无形的鄙视链，已然入侵文具界。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>学校门口的文具店招牌，也逐渐从“XX文具店”变为“XX精品文具店”，再到如今的“XX文创生活馆”，只为配得上里面出售的越来越高档的文具。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18296020246317655d2c9b79.61051942.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>气氛已经烘托到位，涨价自然理所当然。<strong>利用家长的鸡娃心理、孩子的好奇态度，文具厂商们因势利导地布下了一盘棋。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这真是“天下文具竞豪奢，喧嚣纷乱名利场”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1982357525617f8e420bc794.92312446.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>风靡学生圈，文具盲盒如何让人上瘾？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>高阶的猎人，往往以猎物的形式出现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>学校门口散布的文具店，看似毫无章法，实则暗藏玄机。摆放在最显眼位置的，早已不再是实用朴素的文具，取而代之的是各式各样造型亮眼的文具盲盒，竖着的导购牌上印着大同小异的广告语：“玩的就是心跳，盲开更有乐趣。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20595117396317655db89ac0.94882397.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，盲盒并非新鲜事物。如果说上世纪90年代“干脆面”中的水浒卡片是雏形，那现在的奥特曼签名卡就是集大成者，有公司仅仅靠着这项卖卡业务，一年就净赚十几个亿。这更加充分证明一个道理：</span><strong><span>孩子的钱真好赚。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然形式有所不同，但万变不离其宗。<strong>只要有了等级的区分，也就有了攀比性；有了相互攀比的功能，继而产生“赌徒”般的胜负欲，这门生意就算成了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和所有让人氪金上头的游戏一样，文具盲盒还是一种社交谈资，是在小圈子里炫耀的一大法宝，关乎面子和人气，不跟风就会遭到身边人的排挤和冷落。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>文具盲盒也很懂学生们的心理，通常会与知名漫画、小说、动漫等IP结合，整理成系列商品进行售卖。据不完全统计，一个系列通常8至10款，只会设置1个抽中概率极低的“隐藏款”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在文具店的盲盒区，经常能听到孩子们的交流，“一直买买买，终于把笔的样式凑齐了！”“锦鲤盲盒好看，开到两个蓝色，没开到红色好失望。”“重复的超级多，还得继续买。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4034151376317655e240e45.37670778.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅是线下门店，“文具盲盒”的线上销售也十分火爆。在淘宝搜索“文具盲盒”，显示出上千款商品，奥特曼、宝可梦、多啦A梦、阿童木等热门图案应有尽有，排名靠前的店铺，单品月销售均在万件以上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为求证抽中隐藏款，到底需要购买多少盲盒，环球日报的记者就某品牌文具盲盒做了咨询。客服告诉记者，如果购买一整盒的话，一定能够集齐一套，但隐藏款就纯属运气，有顾客买几支就抽中了隐藏款，有的则买了十盒都无法抽中，没法保证结果。换句话说，这就是个“无底洞”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5107958566317655e897918.40503209.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并且，由于消费者在购买时无法获知盲盒内商品的具体情况，这也给生产厂家留下了暗箱操作的空间，比如利用盲盒来清库存、处理残次品等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在各大社交平台上，不少家长晒出了开箱的“冤种”盲盒，不仅有泡泡笔、卡通贴纸，还有清凉膏、发泡胶甚至扑克牌等与学习毫无关联的产品。好不容易碰到几支可以书写的笔，却是质量堪忧，不是断芯就是漏墨，与所谓的“物超所值”相去甚远。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/21247762906317655ede56e3.54149697.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/毛爸聊玩具</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>细思极恐的是，很多隐藏于盲盒内的“三无”文具既没有厂家名称，也没有标注材料成分，消费者选购时根本无法辨别产品中所含成分是否符合国家标准，会否存在有毒有害物质含量超标的情况。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，时至今日，消费观念尚不健全的学生群体之间，已经形成了“整套文具盲盒”的购买模式。<strong>原本简单、朴素的文具，逐渐走上娱乐化、高端化路线。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这正是厂商们乐见其成的结果。有从业人员表示：“与单独卖掉两支笔相比，装进盲盒里的卖出率更高，利润更多。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说盲盒只是“文具刺客”的冰山一角，冰山下，早已波涛汹涌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1861753512617f8e42df4292.55166877.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>价格暴涨背后厂商打什么如意算盘？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据相关数据统计，包括得力文具、晨光文具、齐心文具等国内一线文具厂商，均推出了不同所谓联名文具产品。其中，真彩和得力文具还启用新生代明星代言人代言文具产品，来带动学生群体的关注和消费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上海晨光文具最新发布的财报中显示，联名文具已经成为其重要销售来源。晨光《2022年半年度报告摘要》提到，近年对传统核心业务进行优化，推动内部孵化机制外部IP合作，路径之一正是“精品文创产品赛道”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了获得高溢价，文具厂商们纷纷开启了“联名”之路。经典动漫的授权不能放过，比如海贼王、火影忍者、奥特曼、名侦探柯南等，能够保证销量；小学生中间流行起来的漫画，也必须紧跟潮流，比如与子成说、林深不知处等。<strong>一言以蔽之，就是哪有热度往哪蹭。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/10114216696317655f356386.42490709.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一通操作猛如虎，带来的是价格的暴涨。在淘宝DOU文创社店铺里，一支派通与任天堂的联名按动笔售价24.8元，而该按动笔的原装笔芯折合约不到5元。斑马与宝可梦的限定联名按动笔售价25.8元一支，该型号笔普通款6元一支，二者差别仅在于笔杆图案，这钱未免也太好赚了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，越来越高大上的功能也成为如今文具商家的卖点。电动削笔刀，铅笔放入三秒变锋利；电动橡皮擦，无需出力轻松擦除。有了设计师加持和设计专利的噱头，厂商不费吹灰之力，便能赚得盆满钵满。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3226317796317655fbdb6e2.09920557.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>逐利思维下，这些号称“高端”的文具不仅名不副实，甚至屡被查出质量问题。比如，儿童文具上的香气大多不是天然味道，而是由各种化工原料化合而成的物质所散发出来的气味，为提高文具的“颜值”和手感，有的生产商会在包装材料上过度添加增塑剂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在去年湖北市场监管部门曾随机抽检市场上20批次聚氯乙烯材质的塑料文具盒样品，发现其中10批次产品中邻苯二甲酸酯含量不符合要求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据介绍，塑料文具多为PVC材质，PVC是我国应用最广泛的合成树脂材料，邻苯二甲酸酯是PVC的主要增塑剂，长时间接触会干扰人体内分泌，影响生殖系统，对心血管、肝脏和泌尿系统产生一定伤害。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>单品之外，还有更坑的“文具伴侣”。活页本不能光买裸本，要搭配相同尺寸的活页纸以及方便装卸的打孔器、随意活页圈；“转转笔”的一端要装饰花朵、纸杯、旋转木马等“小零件”；纠错题除了购入纠错本外，还要搭配喵喵机、胶带、切胶带机、标签贴使用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但事实上，这些文具伴侣鸡肋有余，实用不足。号称“快速打印错题神器”的喵喵机，等连上蓝牙、拍下照片、裁好尺寸，隔壁手抄的同学早已经下一题了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/7060550946317656052cfd1.17961487.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>围绕着高价文具，一个“共赢”利益链已经形成。一端是手握文具资源的各大厂商，一端是数以万计的校园小店主，默契地助推着全局的走向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这到底是社会的沉沦，还是价值观的扭曲？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2146800997617f8e44612590.99731811.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>监管政策出台，恶性消费风气该停止了</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>停了补课班，禁了小游戏、关了短视频，却还是没能逃过“文具刺客”的坑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>谁能想到，本是用来辅助祖国花朵们成长为四有新人的文具，竟然在消费主义的刺激下，堕落成狡猾无比的模样，让其把持不住，上瘾沉迷。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/77954273563176560dcd618.17059894.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了满足孩子的要求，不少家长可谓是真正的一掷千金。据中国日报网报道，湖南的刘女士上小学的儿子为了凑齐一套“柯南”的中性笔，偷偷背着父母用压岁钱购买盲盒，一个月就花了1000多元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而买回来的每个盲盒只有2到3支普通中性笔，除了自己想要的图案，其他都被丢掉了，造成了很大的浪费。即便被父母斥责，刘女士的儿子仍表示自己“喜欢惊喜的感觉”，以后还会不断购买新的盲盒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/914469591631765612fb7e5.15714150.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这并非个例。浙江省宁波消保委也接到家长王女士的投诉，孩子就读于小学三年级，为了抽取到某个隐藏款，居然连续购买了20套含有24支笔的套装，总价2000余元，已经完全成瘾，导致精神恍惚学习一落千丈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>站在商家的角度来说，文具是充分竞争的市场，以个性化设计来创造利益，本来无可厚非。但一味追求猎奇和颜值，与文具本身的功能性和实用性背道而驰，只是舍本逐末，借名字和包装来“飞升”，根本无法建立真正的核心竞争力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>再者，故意利用营销手段，刺激和制造恶性消费，引诱未成年人沉溺其中，不仅失德，也踩到了法律的红线。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“文具刺客”刺了钱包是小事，“刺”伤未成年人的消费观是大事。8月16日，国家市场监管总局发布了《盲盒经营活动规范指引（试行）（征求意见稿）》，拟对盲盒销售的内容、形式、销售对象等方面作出规定。自此，盲<span>盒不再是监管盲区。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8160455863176561b2e011.77832443.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>北京市消协也在8月31日也呼吁，文具消费需理性，安全实用最重要，“不人为制造各种奇特的营销噱头，不虚假夸大文具用品的使用功效，为青少年儿童的健康成长营造良好的消费环境。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同一天，北京市市场监督管理局亦发布告诫书，要求线下及线上文具销售经营者，对文具明码标价，做到货签对位、标示醒目，严禁以低价诱骗消费者，以高价进行结算。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了政府层面，各大学校也在积极发出倡议。南京工业大学实验小学就在“致2022级新生”的通知中明确表示，不建议使用图案过多、造型复杂、声音较大的文具。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>说到底，任凭概念炒得如何花里胡哨，包装做得如何精致繁复，文具的本质依然只是学习工具，再高端的笔，也变不成马良的神笔。 </span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>都说“打铁还需自身硬”，成绩的好坏与否，又岂是昂贵的文具可以改变的，你说是也不是？</span></p>
