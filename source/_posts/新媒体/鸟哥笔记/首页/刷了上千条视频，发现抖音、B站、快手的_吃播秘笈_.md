<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“白天刷不到，晚上躲不掉” 这句话可能是很多沉迷于短视频吃播的粉丝们的“心声”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>吃播最早是在韩国兴起的一种“美食真人秀”节目，2015年左右逐渐开始进入国内观众的视野。这一内容形式自“诞生”起，就展现出极强的生命力与吸引力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早期走红的吃播博主们身上大多有着“大胃王”这一标签，故而，一众身形曼妙的帅哥美女，在镜头前大快朵颐还狂吃不胖，一度成为了国内吃播视频的“主色调”，吸引了很多粉丝的追捧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过这类型视频一直存在诸多争议：浪费粮食、损害身体健康、宣扬畸形审美观念……很快被有关部门叫停整治。大胃王限制令后，整个吃播赛道的发展也变得更加合规有序。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>众所周知，吃播的受众群体广泛，粉丝们看这类视频的目的和感受也有不同。有的是享受吃播咀嚼音带来的解压感，有的是在减肥期通过看别人的视频缓解自己的饥饿感，有的则是单纯希望在不得不独自吃饭时，能有人陪伴、打消孤独感。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而不同平台上的吃播创作者以及他们的吃播风格也因为用户群体的不同，呈现出了不一样的面貌。接下来，卡思将从抖音、B站、快手三个平台中，举例有代表性的创作者，进行详细分析，找到不同平台独有的“吃播秘笈”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> <img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>抖音：</strong><span><strong>探店讲解、密友对谈，</strong></span><span><strong>陪伴式吃播大受追捧</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>观察目前抖音平台上比较火的几个吃播大号，卡思发现虽然不同创作者的内容各有千秋，但是仍有一些共同的特点可以借鉴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，相较其他平台，这些在抖音混得如鱼得水的达人们，明显有着<strong>更高的颜值和更好的身材，</strong>毕竟俊男靓女在抖音是永恒不变的流量密码。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，得益于抖音近期在本地生活板块上的种种大动作，除了专职的本地生活<span>达人</span>外，<strong>平台上的吃播达人</strong></span><strong>们对于“美食探店”的作品题材也表现出更大的兴趣。</strong><span>走进大街小巷去探索那些早已名扬千里的老铺子，或者一步步发掘那些还鲜有人知的小众美食店，几乎是每个抖音吃播达人的必备技能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以@小贝饿了 为例，这位凭借爱吃属性、高颜值长相和一口地道陕西话的美女吃播<span>达人</span>，日常穿梭在西安街头的各类烟火铺子中，带领观众领略最地道的陕西美食，在平台上已经吸引了近2000W粉丝的关注。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12333148126320adcba2faf1.20875123.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@小贝饿了 抖音截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开朗的性格、超强的亲和力都是她吸粉的制胜法宝，即使是坐在街边吃最朴素的一碗面，都能吃出一种山珍海味般的快乐，让看到视频的人会心一笑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，</span><span><em>“我们一起去吃吧，gogogo” “吃饭不吃蒜，香味少一半” </em></span><span>等口头禅作为作品的语言锤，也进一步加深了观众的印象。各种设定叠加，让这位开朗活泼的<span>达人</span>自然受到众多粉丝朋友的喜欢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三，相比起“吃播起源”韩国博主们的“沉浸式”吃播方式，抖音平台上的达人们更爱以“对谈型吃播”的形式出现在粉丝面前。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们在视频中不会选择保持安静，而是吃几口就和屏幕前的粉丝们闲聊两句，营造出一种对谈的氛围，与观众保持高频的互动。聊天的内容主要是自己的吃播感受，以及抛出问题留给观众在评论区解答。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5737532926320adcbd682f2.64796002.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@肉肉大搜索 抖音截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样的模式能隔着屏幕给粉丝们更多陪伴感和亲近感，无形中拉近了<span>达人</span>与粉丝之间的距离。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卡思还发现，抖音上的知名吃播<span>达人</span>们都喜欢<strong>将视频的发布时间定在下午，特别是@特别乌啦啦@小贝饿了@密子君@张喜喜等人，几乎每条视频的更新时间都在下午四点左右。</strong>选择这个时间段，能让作品在晚餐高峰前完成前期的流量蓄水，以完成后续的爆发。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，这之中还有一个很大的原因在于，以上达人几乎都有带货行为，且开播时间都集中在晚上18：00-19：00，下午发布新作品，对于达人晚上的直播无疑起到了很好的种草、引流作用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><strong><span ql-global="true">B站：</span></strong></strong><span><strong><strong><span ql-global="true">奢华食材、精致体验，</span></strong></strong></span><span><strong><strong><span ql-global="true">新奇怪趣、解压无敌</span></strong></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在卡思看来，B站的吃播up主们的作品，不仅延续了社区内惯有的新、奇、怪的特色，还在一直努力，“卷”出新意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，相较于其他平台吃播视频的题材“一家独大”的情况，B站吃播赛道内有热度的作品，其选题和表现形式显然会更加丰富一些。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，有以@大祥哥来了 为代表人物的<strong>高端体验派，他们致力于带领粉丝见识更新奇、更昂贵的食材，是粉丝口中戏称的“人民币玩家”。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6034494296320adcc0d7329.92033116.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@大祥哥来了 B站截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这类型up主视频中出现的各种食材，大多是绝大多数人日常没有机会去“亲口一试”的类型，皇帝蟹、椰子蟹、杀人蟹、响锣刺身……这些听上去就“B格”满满的材料，也帮助着up主们一次次刷新着自己的播放量上限。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比起对粉丝进行安利种草，这类作品更多的是给大家带去一种新奇的体验，了解更广阔的世界，满足自己对未知的好奇，而粉丝们观看这些视频的目的，自然也不会是为了下饭或是被陪伴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>还有以@小文哥吃吃吃 @我的样子平平无奇 为代表的精致动手派。</span></strong><span>这类型的up主会从食材烹饪这一步开始，全方位展示自己吃到一口菜的流程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频前半段起到知识科普作用，也有粉丝沉迷于食材处理阶段的“解压疗愈”感。视频后半段则与一般吃播视频类似，但经过了前期的种种铺垫，会让观众对这次体验有一种感同身受的来之不易之感，当然这类视频还会刺激一些粉丝“自己动手，丰衣足食”，被视为教学视频藏在收藏夹内。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4286256336320adcc4be1f2.61277550.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@小文哥吃吃吃 B站截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有一类特殊的吃播视频也在B站盛行，但这类视频并没有所谓的代表up，即国内创作者对外网吃播博主的搬运视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这类作品的重点是食物本身和博主吃东西时产生的咀嚼音，从另一个角度讲，这类型视频也可以被称为asmr视频。</span></strong><span>asmr指的是自发性知觉性经络反应，通俗的解释是“颅内高潮”，即人体在通过接触一些声音、触觉等微妙刺激的时候，所产生的愉悦反应，有改善睡眠和安定情绪的作用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>恰巧，被放大的咀嚼音就是很多人心中的“安眠神器”，他们观看这类视频能体会到很好的放松舒压的效果。但是这种作品主要还是流行于外网，在国内暂时还未出现具有代表性的博主。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得关注的是，除了具体内容外，卡思浏览B站热门吃播视频还有几个明显的感受是</span><strong><span>：一是</span></strong><span><strong>站内视频的时长普遍较长，基本在8分钟左右，拍摄剪辑制作的难度都更高，出镜人物丰富，各个up主之间常常进行联动；二是拍摄的脚本、语言对白都经过精心设计，比起美食探店的“真实感”，这些作品对内容的质量要求更高，整体都透露着一种“精致感”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong>快手：</strong><span><strong>强调视觉冲击，</strong></span><span><strong>总有浓浓烟火气</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>坦率来讲，快手称得上是国内吃播视频作品的发源地。2016年左右，鲜艳火红的食物，一口气“炫”完一盘的老铁，几乎成为了快手的代名词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>随着多年的发展，快手吃播们明显收敛了不少，但视频中属于他们独有的豪迈与爽利仍未褪色。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手吃播们在吃东西时明显少了剧本，多了实在，往往都是坐在镜头前闷声不吭，先一口气吃完再进行后续的流程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>食物不一定色香味俱全，但一定够诱惑，特别是在颜色上下了不少功夫，所以各类肉食更为常见。快手上也诞生了很多“特殊”的食材，可能并不是常人眼中的珍馐美食，但看上去极具视觉冲击力，故而获得了很多粉丝的喜欢，这类型吃播在达人中的占比也很高，比如@吴小妮美食@小鱼爱美食 等都是之中的佼佼者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了感官冲击外，<strong>作品中的烟火气可能是快手吃播们的另一大特色</strong>。卡思近期关注的一个达人@鱼妹很爱吃 就是一个代表，她的视频和上述提到的创作者有较大差别，但依然收到了很多粉丝的喜欢。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1829127836320adcc9c8df5.19816703.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@鱼妹很爱吃 快手截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该达人总是流连在各个夜市之间，提着一大堆食物回家和弟弟老公分吃，不论是食量还是食物种类，可能都不符合猎奇、冲击这样的字眼。但正是因为夜市中的浓浓烟火气，以及鱼妹吃饭时的真实感，俘获了一众粉丝的芳心，帮助她突出重围。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在快节奏的生活下，大部分人接受了两点一线的生活，<strong>现代人孤独感的普遍存在，也催生了“陪伴经济”的盛行。</strong><span></span></p>
