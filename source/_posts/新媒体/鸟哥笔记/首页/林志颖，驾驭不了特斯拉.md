<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><a href="https://www.niaogebiji.com/tag/%E7%89%B9%E6%96%AF%E6%8B%89" target="_blank">特斯拉</a>又出事了，这次的当事人是被称为“冻龄男神”的林志颖</span></strong></span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>22日上午，林志颖开着一辆Model X，在台湾桃园市中北路直接撞上了电线杆，导致前盖几乎变形。林志颖和其儿子在被路人救出车外没半分钟，这辆特斯拉就陷入了一片火海。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，特斯拉官方表示：“暂时还不能确认起火原因，特斯拉驾驶位附近并没有易燃材质。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，林志颖除了大众熟知的艺人身份外，他还是一名职业赛车手，拥有丰富的赛车经验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/151955217662e10126c13b32.45392627.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：林志颖微</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也不禁让很多人产生疑问：就连专业赛车手都驾驭不了，特斯拉一般人还敢开吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/152839770462c2e6e6f12c65.66837781.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>特斯拉起火，林志颖伤重</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据监控画面显示，林志颖开着一辆白色Model X，当时路况良好，车速也并不快。然而其在中间车道行驶了差不多10秒左右，特斯拉开始往右偏离，并直接正面撞向了信号灯杆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好在有路人的第一时间救助，最终才没有酿成大祸。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>结合大家经常看到的关于特斯拉的报道，这一次林志颖出车祸，大多数人第一反应都是特斯拉的锅。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>不过，让人疑惑的点在于究竟发生了什么使得林志颖这种专业赛车手没有能够躲开信号灯杆，而是直接撞了上去？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/159515869062e101278434d4.39930707.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：新浪微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人说是林志颖开启了辅助驾驶功能，靠近匝道路段识别错误；也有人认为可能是林志颖自己开车分神。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就现在来看，还无法得出确切的结论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，林志颖本人曾不止一次在公开场合表达过对特斯拉的认可。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年接受媒体采访时，林志颖曾先后说到“model 3彻底颠覆了自己对于电动车的观念”、“很多人都说开了电动车就再也回不去了，尤其是习惯了自动驾驶之后，结果这是真的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这次的车祸起火事件，有业内人士表示：“电动汽车因为电池包的面积较大，更容易受到挤压和撞击而引起变形，从而导致过热失火。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，就在7月25日，吉米工作室也通过林志颖微博发布了官方声明，表示林志颖本人已基本恢复意识，只是现在还无法还原当时的具体经过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/77546272962e10128a7c109.21744008.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：林志颖微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过看到Model X深陷火海的图片，依然引发了外界对于电动汽车安全隐患的担忧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/178889351262c2e6e8503a50.89262603.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>一撞就燃的特斯拉，还能买吗？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>作为电动汽车行业第一个吃螃蟹的人，特斯拉的一举一动都备受关注。尴尬的是，最近有关特斯拉起火的事故却频频发生。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在林志颖事故发生前一天，杭州有一辆黑色特斯拉汽车和一辆丰田SUV相撞，事故发生后特斯拉车底开始接连冒烟并串出火苗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月初，青岛有一台特斯拉Model 3在直行状态下遭到了转弯车辆的撞击，碰撞后发生起火。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4月，大连有一辆特斯拉和小汽车相撞，在伤者被经过的路人救出不久之后，特斯拉起火烧成了一堆空壳。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一撞就燃，如今特斯拉汽车的安全性已经数次遭到大众的质疑。</span><span><strong>不过更多人关心的是，频繁发生意外的特斯拉还能买吗？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11306622662e1012938e0e0.17088833.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：特斯拉官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主观感受可能每个人的都会不同，通过一组数据对比，或许更能够直观体现出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在最近特斯拉所发布的最新财报当中，第2季度收入169亿美元，虽然略低于第1季度的188亿，但仍然好于分析师普遍预期的165亿美元。在这之中，甚至还包含了4月特斯拉上海工厂停摆的20天。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而从销量来看，今年二季度特斯拉总计交付25.47万辆汽车，同比增长27%。马斯克更是公开强调，特斯拉有极大的潜力会达成“创纪录的下半年”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>尽管事故频发，但特斯拉的销量似乎并没有受到明显影响，反映在资本市场上特斯拉股价也没有太大波动。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/182169925762e1012a451ac7.19210797.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：百度股市通</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有特斯拉的忠实粉丝表示：“以前上班通勤开燃油车，只加油就是一笔不小的开支。而现在开特斯拉之后，同样的距离每个月电费只需要100元左右就完全可以满足了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从驾驶成本的角度考虑，确实节约了不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过对于个人来说究竟是否购买特斯拉，还是要根据实际情况来决定。毕竟特斯拉一次性的购车成本，相比较普通的燃油车还是贵出不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/39656327662c2e6e913b8c2.71812476.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>头疼“燃烧”的车企，</span></strong><strong><span>不止特斯拉一个</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，除了特斯拉之外，电动车在遭到碰撞之后的起火现象时有发生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年8月，造车新势力威马汽车在其成都研究院内发生自燃；哪怕是有刀片电池加持的比亚迪也曾在去年9月一次失控冲到马路护栏而起火。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，包括小鹏、理想、长城等多个新能源汽车品牌都先后出现过碰撞起火和自燃事故。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/68432204262e1012ab71d72.03682483.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：小鹏汽车官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>据了解，目前平均每天就有超过7辆新能源汽车发生火灾事故，而起火的主要原因之一就是外部碰撞。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>业内人士表示：“发生碰撞之后，电动车的起火概率比燃油车要高出不少。燃油车的油箱相对封闭，只要燃油不外泄一般不会起火；而电动车的电池本身就属于可燃物质，足够大的冲击就有可能会导致起火。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>车辆在发生碰撞后，就有可能导致电池箱变形和破裂，从而引发短路。这种短路状态下所产生的强电流会引起电池急剧升温，最终导致燃烧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，2021全年国内新能源汽车保有量约784万辆，全国范围内总共发生了3000起火灾事故，概率为万分之3.8，这一数字已经超过了传统燃油车。2022年一季度，新能源汽车火灾总计达到了647起，同比增长高达32%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正是因为经常能够从新闻上看到关于新能源汽车起火的事故，因此不少人对电动车依然保有怀疑态度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/165756143162e1012c0cf613.09550989.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> <span>‍</span></span>图源：特斯拉官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，摆在特斯拉等新能源车企面前的核心课题依然是安全，也希望当下林志颖的遭遇在未来不会再次发生。</span></p>
