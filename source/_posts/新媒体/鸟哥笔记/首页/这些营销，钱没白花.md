<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>广告</span><span>界</span><span>有一句名言：</span><span>我知道我的广告费有一半是浪费的，但我不知道它浪费在了哪里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是对于做公益类型的营销，钱没浪费。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说明一下，这里的公益营销不是指捐款，而是品牌参与的某一项带有公益性质的事情，准确来说是兼具广告和公益性质的事件营销。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此类营销能令人感受到品牌对社会的关爱，从而对其好感倍增。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今天的公益营销案例包括：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、天猫，可持续月饼礼盒</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、麦当劳帮忙解决暑假作业</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、盒马送水，长颈鹿喜提960公斤</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、伊利QQ星给流浪猫送窝</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5､珀莱雅，你仗义执言的样子真美</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/108519200762548b74f22149.44051694.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>天猫让团圆可持续，</span></strong></span><span><strong><span>月饼礼盒玩儿出花</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>中秋节礼盒，既然在精美这个赛道上已经卷到难分胜负，那就另辟蹊径吧。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天猫将礼盒与可持续利用结合了起来，整合志同道合的朋友，定制绿色环宝盒，共庆中秋。具体措施包括礼盒采用环保材质，开发礼盒的二次利用价值。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天猫OTALY团圆环宝盒，分享每一份团圆喜悦，吃完月饼，它还是一个便当盒。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/24124519563210212c867b4.76909138.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/113261443363210213062af2.52719475.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>星巴克星情月饼礼盒，可再利用相框，记录每一瞬团圆快乐</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/145828652363210213416029.04231159.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知味观万家灯火礼盒，可变身环保木质氛围灯；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1745783824632102136a69a6.29233052.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外还有朕的心意故宫月饼锦盒，更是一个精美的首饰盒，珍藏每一份团圆礼物；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而秋香冬韵麦田月光轴，可化身甘蔗渣唱片机，播放月光曲。这届天猫中秋节，礼盒的隐藏属性被开发的十分精彩，赢了环保的立意和产品的创意。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/97208718063210213b731b2.40134579.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天猫为这届月饼节量身定作的羊毛毡定格动画也是一大看点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以友情为主题，将不同的月饼礼盒嵌入剧情，比如从朕的心意月饼首饰盒里取出项链送给小兔子，合影放进了星巴克的礼盒相册等等，把“可持续”的故事诠释的软萌可爱，温馨动人。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在公益色彩的加持下，天猫这次</span><span><strong>立意赢得有格局，创意赢得有品质，事件赢得有影响力。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/201362546162548b7878ed62.59179361.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>暑假作业去哪写？</span></strong></span><span><strong><span>星巴克还是麦当劳？</span></strong></span><span><span></span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年暑假，麦当劳在云南开放了42家麦麦自习室，购买两份优惠自习套餐就能享受学习空间，饮品还能免费续杯。其中简易套餐只需9元，复杂套餐29元，相比另一自习室大户星巴克，价格要优惠的多。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/10174322646321021405d871.88499193.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比起价格的亲民，更令人感动的是他愿意发起这个自习室活动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为只有“官方赋予”这里是自习室的BUFF，别人才不会喧哗，才会把这里当作学习的地方。他的确解决了一部分学生学习的问题，为想学习的人创造了学习和上进的环境。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5532432786321021447d540.41877857.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>麦当劳开麦麦自习室，乍听起来，是不是特别像星巴克买杯咖啡就可以享受它的办公空间。延着这个思路，麦当劳抢一抢星巴克的生意好像也可以。与之相比，麦当劳有价格优势，还有适合中国人的餐食，再说需要空间的也不止是学生，大把自由职业者需要办公空间。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你觉得有戏吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/124588862062548b79c7e797.48416030.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>盒马抽奖送水，</strong></span><span><strong>长颈鹿喜提960公斤</strong></span><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>温暖人心的公益比比皆是，但快乐效果满级的目前只看到这一家，被称为“屁股”的盒马，抽奖送福利搞出了段子效果。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为天气太热，盒马官方就搞了个抽奖，抽三位送水，体重多少送多少。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只看开头，属于有亮点但是不多的事件。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还不如云南一乡村小学奖励优秀学生每人一头小猪来的吸睛</span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/297590828632102147be8d7.91986287.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一开始网友只是在评论区我300斤，我800斤的调侃。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是引来动物园围观的时候，剧情开始跑偏，评论区开始精彩。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>广州动物园携大象宝龙，自报体重4972KG，盒马脸都吓黑了！</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/22401181163210214c3dbe3.47571949.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>南京市红山森林动物园的老虎咖啡也来问询，自曝体重330斤。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据说还是那只在水里偷偷放屁咕噜冒泡走红网络的老虎。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/204977706863210215142b36.48592777.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>来自武汉动物园的河马大江和小河，盒马的亲戚也来了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/41634643632102153f4cc9.35854020.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>冷知识，看起来高条纤瘦的长颈鹿，体重居然有960公斤！</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>艾米，来自云南野生动物园。此刻艾米还不知道自己会成为最后赢家。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5192259546321021573ad69.38718826.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看到动物园亮相评论区，网友们本着“看热闹不嫌事儿大”的原则，让大家把自己的评论删一删，只留四个动物园的评论，高低得让大象中一个。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>留下盒马一脸问号。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/37323897863210215980f03.11924052.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好在抽奖结果出来，动物园都没中奖，盒马总算松了口气。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但这口气还没松完，剧情反转了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位来自云南的中奖朋友表示，我不要了！转送给我们云南动物园的长颈鹿艾米吧！960公斤的那个长颈鹿。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盒马开启跨部门协调采购问题。把水凑齐，腾了个仓库又找车，盒马一副大冤种的口气。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19385774863210215c62681.23733010.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但最后皆大欢喜，长颈鹿艾米喜提960公斤水，而盒马把一场小小的抽奖事件演变成全网围观的营销事件，多少同行羡慕不来呢。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截止发稿前，盒马首发的那条抽奖微博的点赞高达15.5万，转发1.8万，评论5000+。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/77636141763210215ee2206.44361201.jpeg" referrerpolicy="no-referrer"></p>

<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>国际流浪动物保护日，</strong></span><span><strong>QQ</strong><strong>星给喵星人造猫窝</strong></span><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>近年来，在日本经济界有一个定论：无论涉及哪种领域（划重点），只要用好了猫咪，就一定能够获利，即猫咪等于经济。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回忆一下星巴克猫爪杯一杯难求，麦当劳和肯德基出猫窝被秒空，凯迪拉克都来撸猫，社交网络猫咪网红大行其道，就能明白</span><span><strong>猫咪经济学是真的。</strong></span><span>关于猫咪的公益活动自然也更容易博得好感。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在国际流浪动物保护日，QQ星与威斯汀酒店、新邻生活站和汉堡王共同发起了“给流浪小动物一个家”的号召，邀请数百个家庭，用喝完的QQ星纸箱为流浪动物筑造温暖的小窝，并投放到它们经常出没的地方。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/141878652363210216250fd4.17907978.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从本质来看，它跟“天猫，让团圆可持续”的出发点类似，循环利用包装盒。但是，QQ星的活动针对性更强，并且特殊节点有针对性的主题，师出有名，合情合理，还沾了一把猫咪经济学的光。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2044141644632102166fe9f3.50809612.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这类活动还非常受父母欢迎。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它可以在行动中培养孩子的爱心，提高孩子的动手能力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不少家庭养宠物，本身就很关心流浪动物，与品牌号召一拍即合。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且对于带娃家长来说无时不刻都在想：要给孩子找点有意义的事儿做。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用的还是QQ星的箱子，暗戳戳地给自己拉了销量。如此名利双收的事情，当然要做啦。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/79117991163210216ef7eb7.30428926.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/67894085163210217635ab0.40848048.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>珀莱雅，你仗义执言的样子真美</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年开学季，珀莱雅联合中国教育电视台共同发起了「萤火计划—反校园霸凌公益活动」，并特别推出反校园霸凌公益短片，呼吁大众关注校园霸凌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这件事情很难不引人注目。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个是社会关注的热点话题，还有一个看上去八杆子打不着的美妆品牌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，这也许正是珀莱雅要表达的重点之一，在校园霸凌这件事情上，不存在旁观者，沉默已经是一种态度。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这场公益活动的意义，旨在呼吁每个人都去关注校园霸凌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/101104638163210217a57ef8.12328546.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不想用写实这个词来形容这个公益短片，但现实可能远比它残酷。那三个被欺凌的对象，就是三种年龄层次的被欺凌者的缩影。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有弹幕说：很真实，已经在生气了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了角标，视频内容中没有品牌的露出，可以感受到珀莱雅做这件事的纯粹初衷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>透过洞见深刻的文案，可以感受到珀莱雅的认真。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从还原场景的对话中，可以听到珀莱雅的态度。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/34619154563210217cac885.30089478.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1583219194632102182ae027.07437189.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这场公益活动不止是呼吁，还有进一步的行动。珀莱雅联合壹心理共同制作了《校园霸凌防范指南》，从心理学的角度阐述校园霸凌的不同形式、发生原因、应对措施等，为校园霸凌受害者及相关人员，提供科学心理支持，纸质版的图书会通过微博向关注者免费公益赠出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/520910967632102187fc728.22615841.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17378608163210218a7e1b4.31007724.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这份指南非常实用，能有效地指导身处不同角色的人如何应对校园霸凌，看得出珀莱雅做公益的诚意十足。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在反校园霸凌的公益行动中，珀莱雅表现的特别像一位充满勇气和智慧的女性，并且文武双全。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总之，这一局，珀莱雅令人刮目相看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们发现，当品牌把目光放在公益事业，而不是只盯着自己的时候，存在感反而更强了。所以说，有效的公益营销，真的不会浪费钱。</span></p>
