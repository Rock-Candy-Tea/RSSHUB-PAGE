<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管吃不吃、想不想吃、爱不爱吃、好不好吃，只要提到<a href="https://www.niaogebiji.com/tag/%E7%96%AF%E7%8B%82%E6%98%9F%E6%9C%9F%E5%9B%9B" target="_blank">疯狂星期四</a>，大多数人都能想起——<a href="https://www.niaogebiji.com/tag/%E8%82%AF%E5%BE%B7%E5%9F%BA" target="_blank">肯德基</a>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可能连肯德基都没有预料到，2018年给消费者搞的优惠促销活动，如今刻入了互联网大众的DNA之中。这也促使2022年，肯德基成功将“疯狂星期四”注册为其商标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论你是不是真心喜欢吃卡路里高的要命的肯德基，都不能否定肯德基疯狂星期四营销的成功。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟，肯德基在中国，已经彻底扭转了局面，让其最大的竞争对手麦当劳望其项背，不仅是第二，还是差距很大的第二。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家为何对肯德基的星期四乐此不疲？为何没有人真的计较这玩意儿到底好不好吃？肯德基的营销究竟有何魔力？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>讲讲肯德基的5个营销“丛林法则”，甲方财经「营销十大风向标」系列01篇。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>法则01</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“最好的竞争就是没有竞争”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>丛林法则其实非常简单：就是弱肉强食，适者生存。丛林中的强弱位置不可能永远不变，也并非通过比拼机制取胜，只要抓住一个机会，就可能占据上风。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是肯德基的战略。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓的战略，就是要与众不同，就是要提供一个独特而有价值的位置，并与之配套一种与众不同的活动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>肯德基的所有产品可并不一定都比他的竞争对手麦当劳好吃，也并非它自始自终的营销都表现优异。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最根本的原因是：肯德基在2018年，作为第一个品牌把“疯狂星期四”的营销记忆植入到消费者心中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，在星期四这一天，肯德基没有对手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>星期四，是个消费者心理状态很特殊的日子。它卡在人们度过工作日前三天即将迎来休息日的心情上升期，人们的消费欲容易被释放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一天肯德基都会推出类似于“9.9三块炸鸡翅”、“19.9二十块黄金鸡块”、“29.9八只葡式蛋挞”等爆款促销活动，吸引顾客前来消费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了加强这一促销记忆，肯德基不仅创造了洗脑广告语“疯狂星期四，九块九块九”，和网友互动引发“疯四文学梗”，还请来各路明星流量以及在线上线下全方位投放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/34511783562b29656749fd3.88490050.png" width="333" title="100倍速营销时代，需要1个记忆锚点" alt="100倍速营销时代，需要1个记忆锚点" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个过程信息越复杂、越丰富、越活跃，人们的大脑就会越活跃，就会越容易记住这个日子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>牵动的脑细胞越多，大脑越活跃，就越容易将肯德基和星期四深度绑定。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>法则02.</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“当合理达到上限，离谱反而靠谱”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>丛林如同一个平台，树可以长、草可以长、蚂蚁可以长、老虎可以长，菌类可以长，各有各的活法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而当合理的活法失去了足够的空间后，离谱一点的思路没准能找到新的位置。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现实中，合理的创意，已经失去了生存空间。对于消费者来说，每一个营销节点、活动的套路早已习以为常，人们很难再产生新的记忆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从生理学上讲，想要激发人们的好奇心，就要不断刺激其产生θ脑电波，这是对记忆力最重要的一种脑电波。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>θ波可以使大脑产生LTP，也就是大脑神经元被激活，神经元之间的链接速度加快，就会产生新的牢固记忆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是肯德基还是其他品牌的魔性广告语，其不断的重复对于消费者来说的确有洗脑的作用，但很多时候这种重复会让人产生负面的厌倦心理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“疯四文学”恰恰切中了精髓，在千篇一律重复的广告语中，再一次激发了人们的兴趣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你会在各类社交平台上刷到各种类似下图中的话术。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/120142354162b29656bf5ee8.50838575.jpeg" width="333" title="100倍速营销时代，需要1个记忆锚点" alt="100倍速营销时代，需要1个记忆锚点" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>足够离谱，也足够荒谬，但也足够让人产生新的记忆。让人隐隐觉得或许在星期四吃顿肯德基真有什么神奇功效。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>法则03.</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“与异者共赢”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互利互惠，也是丛林法则的一个组成，和其他生物合作未尝不是一个明智的选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回到现实，同一类品牌在竞争时，并非用用一种套路、用一种广告位的形式争的头破血流，走出去广结良缘未尝不可。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>肯德基、麦当劳、必胜客、汉堡王等等都是西式快餐，经营模式大同小异，标准和操作程序基本也一样，然而在后期发展过程中肯德基开始与拉开差距的分水岭，一骑孤身，绝尘而去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过观察近些年肯德基的营销活动，不难发现，肯德基的营销伙伴丰富至极，比如：</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多时候，强者战胜弱者不需要任何理由，但并非代表永远成功，有时候能存活的物种反而是能够不断调整和增强自己的能力的，通过变化亦或者合作，形成自己的新竞争力。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>法则04.</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“拥有1种以上的保护色”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>丛林里，很多动物有保护色，类似蜥蜴的变色能力，还有不少动物的皮毛颜色与环境相似，这是一种生存进化的结果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，我们可以理解这是为了防御和保护，另一方面这也是一种味了适应环境变化而作出的形态结构和生活习性的改变。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反应到品牌营销本身，就是肯德基这样的品牌如何适应本土化，不断发掘保护色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>肯德基在国内发展了数十年，不仅很好的结合了中国文化，也会推出区域营销做当地特色限量产品，引起当地消费者的共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《沁园春 雪》中的“江山如此多娇”，被肯德基演变为“生活如此多娇”，引入到广告语中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与青岛啤酒合作，在菜单中加入青岛啤酒，推出“炸鸡搭配啤酒”的特色营销。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>辣串新品发布初期，首发地仅限上海、成都、重庆等10座城市。少数售卖区域多数零售去形成反差，更容易激发消费者的好奇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>……</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>法则05.</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“讨好并不可耻”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>丛林无边界，任何生命的延续都要靠不断的发生关系而得以传承。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌营销，不是一天、一个节日、或者某一次特定活动，与消费者产生的交集，而是不断的向消费者释放信号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这个过程中，品牌能够塑造人格，消费者也能够不断的了解、深入品牌的价值，减少促销、打折等营销的刻板反应。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>像肯德基一样，越来越多的新消费品牌会不断的和粉丝互动，而在沟通技巧上上，肯德基算得上是老司机，或者换句话说，肯德基不惜一直在“讨好”消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>会在微博上向大家承诺，带来物质或者精神上的利益：@肯德基：#我就粉草莓#今天又双叒叕开工啦！用一支草莓冰淇淋花筒，给心情放放风，将所有的烦闷一口一口吃掉，送自己一份甜蜜时光</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>100倍速时代，需要1个记忆锚点</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这样一个混乱与变化加速的时代，机会不断涌现，却又瞬息消失。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>竞争对手、科技、顾客面、供应面、协力产业，乃至於有关的规范制度的转变，已带来一个 100 倍速时代。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这样的环境下，如何向肯德基一样能被记住？能在大家一提到星期四就被提起？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你足够的资本，在用户每每有需求的时候，就立刻把广告弹到用户面前，那自然是粗暴直接的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果没有，那就真的需要制造一个记忆锚点。可以是一个数字、一个地点、一种心情、一种环境、一种行为等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>已经有不少品牌做出了实践：脱毛-ulike、奶酪棒-妙可蓝多、小火锅-呷浦呷浦……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种记忆能持续多久，是否还会有对手颠覆，一个看时间，一个也要看品牌是否能在竞争的丛林中找到自己的生存法则。</span></p>
