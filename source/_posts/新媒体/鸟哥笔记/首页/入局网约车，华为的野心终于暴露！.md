<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>问界、极狐、赛力斯、阿塔维，一众新能源汽车身上都有着华为的影子。如今，在汽车相关领域，华为又将触手伸向了出行赛道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>华为入局<a href="https://www.niaogebiji.com/tag/%E7%BD%91%E7%BA%A6%E8%BD%A6" target="_blank">网约车</a></span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，有消息称华为已经推出了自己的首款打车快应用“Petal出行”，这也代表着华为正式进军网约车市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，目前该应用仍处于测试体验阶段，只面向众测用户开放，普通用户尚无法在App Store以及安卓应用商店下载。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从网传应用详情可以看出，Petal出行目前仅在北京、深圳、南京三地试点，但官方表示，其支持城市正在持续增加中。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/13468045762d9de0cf00e05.31090554.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：华为账号</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着华为的入局，网约车市场又迎来一员大将。更有网友喊出“支持华为，干翻滴滴”的言论。不过需要注意的是，“Petal出行”与滴滴等网约车运营公司之间，目前并无直接竞争关系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相较于滴滴等网约车运营方，“Petal出行”则属于网约车供应商的聚合平台。类似于美团打车、高德打车等，华为并不涉及到司机方，而只负责将用户订单发送给滴滴、T3出行等网约车公司，从而为用户提供打车服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>换言之，Petal出行只是一个聚合平台，最终的服务依旧由网约车公司提供。</strong></span><span>目前，接入Petal出行的供应商暂时只有首约汽车和神州专车两家。在车型上，共有经济型、舒适型、商务6座、豪华型四种选择。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/48789870362d9de0d6f0988.91168735.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：Petal出行</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管目前可供选择的供应商并不多，但华为布局至此，也算是落下了颗进可攻退可守的棋子，一方面将来可继续增加更多的供应商，以聚合平台的身份参战网约车市场；另一方面，也不排除未来直接独立运营网约车的可能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，尽管华为并不下场造车，但其在造车业务和自动驾驶领域均有布局。目前多家车企在开拓智能联网汽车业务过程中，也均将“整车+出行平台+自动驾驶”作为发展思路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于华为而言，此次Petal出行的上线既是面向新赛道的尝试，也是对这一铁三角模式的补充。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论如何，可以明确的是，</span><span><strong>随着华为的入局，在其本身品牌影响力及流量加持下，本就身处混战的网约车市场竞争将愈发激烈。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>华为入局，胜算几何？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此时华为的入场，时机并算不得好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在一年前，受信息安全等因素影响，正如日中天的滴滴出行受到交通部等七部门联合调查，而后，更是惨遭全网下架。一时间，网约车市场迎来巨震。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>山中无老虎，猴子称大王。在滴滴下架后不久，众多网约车平台开始冒头发力，市场再次热闹了起来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70872305162d9de0dda4d59.22677014.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：网络</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>仅在去年9月、10月、11月，曹操出行、T3出行、哈啰出行便先后完成了38亿元、77亿元、2.8亿美元融资。屡创新高的融资金额也昭示着市场的火热。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当资本注入后，这些出行平台也开始了疯狂的补贴大战，抢司机、抢用户，势要从滴滴口中拽下一大块肉来。</span><span><strong>然而热闹到了今天，滴滴依旧还是网约车市场唯一的巨头。</strong></span><span>在滴滴之后，美团打车、高德打车等平台背靠自身强大的流量，也在虎视眈眈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为新入局者的华为，不仅要面对滴滴这一老牌寡头的压力，还要防备美团、高德、哈啰出行等对手的围猎，前路多艰难。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>不仅如此，随着市场蛋糕的进一步做大，行业监管也在随之收紧。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，截止2021年12月，我国网约车用户规模较前一年增长了8733万，达4.53亿，这一规模占据了我国网民整体的43.9%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随之而来的，则是在今年2月份，交通运输部办公厅等多部门联合发布相关通知，要求加强对网约车行业的事前事中事后全链条联合监管。伴随着政策收紧，网约车市场这块蛋糕更加难以瓜分了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过对于华为而言，它也有着自身的优势。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/91364372862d9de0e5cb729.90302069.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：中国品牌强国盛典</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>作为打破国外垄断的国内顶级科技企业、央视评选出来的唯一一个 “国之重器” 民企品牌，华为在民间拥有着强大的品牌影响力和号召力</strong></span><span>，而这些影响力也势必将转化为流量为Petal出行供能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，自2013年宣布入局车联网以来，华为已在<a href="https://www.niaogebiji.com/tag/%E6%B1%BD%E8%BD%A6%E9%A2%86%E5%9F%9F" target="_blank">汽车领域</a>深耕多年，并与多家车企展开深度合作。作为出行生态的一环，Petal出行未来或许也可采用类似曹操出行的业务模式，将网约车与“华为”汽车进一步联动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但无论如何，这都属于后话了，对于当下的华为而言，想要在出行市场抢蛋糕，还是压力不小。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>华为出行，意在地图</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而即便压力重重，华为还是选择了这条路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许对其而言，网约车并不是最终目的。</span><span><strong>醉翁之意不在酒，在乎花瓣地图也。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公开资料显示，2020年10月22日，华为正式面向海外市场发布了Petal Maps（花瓣地图）。</span><span><strong>作为华为提供的全场景地图行业生态服务，花瓣地图还包含了出行、社交、购物、餐饮等基本生活地理位置服务。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/49724747562d9de0f1c2ac9.13511296.png" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：机锋网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>华为官方数据显示，截止目前，Petal Maps Platform已覆盖全球200多个国家与地区，积累了2.6亿+POI数据，支持70多种地图展示与搜索语言。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但时至今日，花瓣地图应用也并无国内版本。而恰好，此次Petal出行所内嵌的地图应用正是华为自家的花瓣地图。或许，进军网约车市场正是花瓣地图在国内市场的一步先行棋。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>地图应用在国内互联网市场向来是个比较特殊的存在，它不赚钱，需要费大量人力、物力去维护，但又被各巨头紧紧把持，背后的原因值得深思。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以高德地图为例，在被阿里收购时，其便处于亏损之中，在进入阿里大家园后，同样也难以扭转亏损的命运。但时至今日，高德依旧活得好好的，甚至仍旧是地图领域的双寡头之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>归根结底，还是因为流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为地图导航这一高频应用场景，截止2021年3月，高德手机端月活用户超5.99亿，庞大的流量为阿里自身的电商业务进行了充足的导流。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/86588118462d9de0fddacf5.16022852.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：高德地图官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，无论是传统电商，还是O2O模式下的外卖、即时零售等，这些电商模式下的物流履约过程均需要地图服务作为支撑，面向用户的实时物流进度同样也需要地图服务来承载。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于互联网企业而言，高精度地图的重要性极大。也正因如此，后来的滴滴、美团都开始进军地图领域。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，</span><span><strong>地图也是互联网巨头用来承接本地生活服务的入口之一。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年七月，阿里将飞猪、高德、饿了么组成“飞高了”矩阵，共同在本地生活领域与美团竞争。如今打开高德地图，便可看到首页的休闲玩乐、美食团购、订酒店等类目，地图的庞大流量也在帮助本地生活服务实现高飞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/26865339362d9de11b96ff5.26025099.png" referrerpolicy="no-referrer"></span></p>
