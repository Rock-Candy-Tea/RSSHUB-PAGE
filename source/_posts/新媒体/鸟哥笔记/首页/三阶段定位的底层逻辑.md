<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">什么是定位？</span></p>
<section>
<section>
<section>
<p><span>电影《阿甘正传》里有一段话：</span><span>我不觉得人的心智成熟是越来越宽容涵盖，什么都可以接受。</span><span>相反，我觉得那应该是一个逐渐剔除的过程。</span><span>知道自己最重要的是什么，知道不重要的东西是什么。</span><span>最后，做一个简单的人。</span></p>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就是<strong>定位</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>定位，就是做减法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>定位，就是抓关键的少数。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>定位，就是乔布斯的极简主义。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据企业所处的不同发展阶段，我们今天来给出一个非常实战的<strong>三阶段定位</strong>底层逻辑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>哪三阶段呢？从0到1、从1到10、从10到100，我们简单粗暴地把企业分成三大阶段。</span><span></span></p>

</section>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当企业刚刚起步，还非常弱小，甚至生死问题还没有得到根本性解决。此时，我们重点抓三个定位：<strong>战略定位、产品定位、品牌定位</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，我们来看看<strong>战略定位</strong>。</span></p>
<section>
<section>
<section>
<section><span></span></section>
<p><span>1、当企业比较小还没有解决生死问题时，<strong>战略定位</strong>主要围绕着战略三要素展开。</span></p>
<p><span>战略三要素：</span></p>
<p><span>价值交付（我是谁）；</span></p>
<p><span>战略意图（去哪里）；</span></p>
<p><span>战略路径（怎么去）。</span></p>
<p><span>总体说来，就是根据汽车的油量，不断地调整阶段性小目标，找到加油站获得能量补给，尽可能快速地通过创业的死亡峡谷。同时，心中有一个大目标，这个大目标决定了企业的高度与创业的难度，以及生死的概率。</span></p>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、对于连续创业者，或者创业资源丰富的，尽管企业还非常小，但是生死问题困扰不大时，</span><strong>战略定位</strong><span>的内涵就是：</span><span>不断地找到远离竞争的商业机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、对于大格局企业家，尽管企业刚刚创立，<strong>战略定位</strong>就是解决某个社会性或行业性难题。这种高屋建瓴、居高临下的认知优势，是普通创业者无法体会的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，我们来看看<strong>产品定位</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、当企业比较小还没有解决生死问题时，产品定位主要围绕着产品三要素展开。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品三要素：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目标客户（客户拆解）；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>特定需求（需求拆解）；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>解决方案（方案拆解）。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管战略大与小，最终都要落地到一个产品上。从技术上讲，<strong>产品定位</strong>也分为两个阶段：一是从0到0.1，重点是产品策划，找准PMF；二是从0.1到1，重点是产品落地，找准MVP。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>甚至有人会说，创业公司有两种，一种是找到PMF的，另一种是没找到PMF的。可见，找到PMF的重要性和优先级。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么是PMF？产品市场匹配（Product Market Fit）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么MVP？最小可行性产品（Minimum Viable Product）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于PMF和MVP的内容非常多，我们在这里就不展开了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、对于连续创业者，或者创业资源丰富的，<strong>产品定位</strong>的内涵就是：差异化的升级某类客户需求的解决方案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、对于大格局企业家，<strong>产品定位</strong>就是基于行业格局重构下的产品创新。往往门槛非常高，抓住了潜在客户的隐性需求。</span></p>
<section>
<section>
<section>
<section><span>最后</span><span>，我们来看看</span><strong>品牌定位</strong><span>。</span></section>
<section><span></span></section>
<section><span>很</span><span>多人</span><span>一听，</span><span>往往就会觉得诧异：小公司生死未卜，做什么品牌现象宣传？这个观点是不对的。品牌并不等同于企业形象。品牌的核心，是针对行业平均水平和同等品质的产品，我们有没有产生溢价？这种溢价，既可以是价格更贵一些，也可以是同等价格卖得更多一些。</span></section>
<section></section>
<section><span>同时，我们还需要注意到品牌的另一个内涵：</span></section>
<section><span> </span></section>
<section><span>产品是用来交易的，</span><span></span></section>
<section><span></span></section>
<section><span>品牌是用来连接的。</span></section>
<section></section>
<section><span>品牌，是用来和特定客户群，建立连接的载体。这个观点，不因公司大小，而发生实质性变化。</span></section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、当企业比较小还没有解决生死问题时，<strong>品牌定位</strong>主要围绕着品牌三问展开。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌三问：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当客户联想到我们品牌时，会联想到哪些关键词？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当客户联想到哪些关键词，会联想到我们的品牌？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些关键词，有没有促进客户决策和形成高溢价？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么初创公司要抓品牌三问呢？这个是与互联网对各行各业的渗透是密切相关的。互联网的流量，是关键词引来的，这个是品效合一的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、对于连续创业者，或者创业资源丰富的，<strong>品牌定位</strong>的内涵就是：跳出产品功能消费，给出情感类的客户价值，连接到特定客户圈层。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、对于大格局企业家，<strong>品牌定位</strong>就是引领创新、品牌调性和创始人IP等价值观层面、哲学层面的价值创造与价值输出。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当企业产品逻辑已经跑通，商业逻辑形成闭环，交付能力开始稳定，此时，企业就进入了从1到10的规模化发展阶段。此时，我们重点抓三个定位：<strong>销售定位、营销定位、心智定位</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>商业的本质是交易。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这3个定位，分别对应交易三过程：近场过程、中场过程、远场过程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓近场过程，是指在商家主场的销售场景（线上线下门店）内，所发生的交易过程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>中场过程，是指把潜在客户拉进销售场景，从而实现拉新和复购的物理过程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>远场过程，是指客户心智中启动决策的相关心理过程。</span><span></span></p>
<section>
<section>
<section>
<section><span>首先，我们来看看<strong>销售定位</strong>。</span><span></span></section>
<section><span></span></section>
<section><span>在销售场景中，最重要的是产品卖点与客户买点，能不能在这么小的时空中，产生化学反应。</span></section>
<section><span></span></section>
<section><span>能够进入销售场景的客户，多半都是有一定行业知识和品牌知识的<strong>中关系客户</strong>，同时也给了短暂的注意力。</span></section>
<section><span></span></section>
<section><span>因此，销售定位的核心，是商家要围绕目标客户的需求，给出一个差异化的消费理由，实现交易转化。</span></section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，我们来看<strong>营销定位</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在营销场景中，卖货并不是主要目的。因为此时，商家所面对的往往是<strong>弱关系客户</strong>，是很多陌生人。他们中的很多人对品牌不认识，不知道是干什么的？对整个行业认知也可能是非常浅。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，营销定位的核心，并不是转化，而是流量，具体就是指拉新、复购的运营式营销。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，我们来看看<strong>心智定位</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>心智定位</span></strong><span>，是面向<strong>强关系客户</strong>的，忠诚度高的，非常认可商家品牌的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此时，<strong>心智定位</strong>的核心，是设计决策路径，连接精神价值。比如：Nike倡导Just Do it。这句广告语显然是讲给强关系听的，说的是一种精神，与产品的功能与卖点没有任何结合度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里，我们需要解释一个问题：为什么从1到10，和从0到1，我们定位的侧重点会有这么大的区别？难道从1-10。我们就不需要战略定位、产品定位、品牌定位了么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然不是的。不是说我们从1到10，就不需要战略定位、产品定位、品牌定位了。而是说，这些已经是基础工作了，这3个定位，我们在从1到10时，也是需要不断地迭代与调整的。但是相比规模化发展，我们的关注点更要突出在销售定位、营销定位、心智定位上，这样，就有可能形成一个超常规的高速发展期。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当企业顺利地度过了高速发展期之后，就相当于从轻量级拳击比赛，升级成了重量级拳击比赛。遇到的竞争对手实力雄厚，在一个细分市场上捉对厮杀的局面经常发生，一不小心就满盘皆输。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此时，企业就进入了从10到100的组织能力大发展阶段。</span><span>我们重点抓三个定位：</span><strong><span>需求定位</span></strong><span>、<strong>竞争定位</strong>、<strong>物种定位</strong></span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>需求定位</span></strong><span>，就是不断地找到客户隐性需求，挖掘增量市场，引领行业创新，从而降低竞争难度。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>竞争定位</span></strong><span>，就是在存量市场中，不断地创造比竞争对手更强的相对优势，从而在激烈竞争中建立滩头阵地和品牌壁垒。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>物种定位</span></strong><span>，就是摆脱单一的产品市场竞争维度，升级到商业模式竞争和组织进化能力竞争，以打造商业生态为核心，实现降维竞争。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如说天猫与京东，这两家在产品层面是同质化的。在商业物种层面是差异化的。天猫，是全球新品首发地，所以，它的营销平台属性更强；京东，是物流快，服务属性更强。这两家在需求定位、竞争定位、物种定位上，都有不同的战略路径与核心能力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上3个定位内容非常复杂，我们在这里就不展开了。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互联网公司与传统企业，在定位的认知上，是完全不同的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传统企业（特别是消费品），非常在意渠道（规模化触点）和广告（规模化认知）。要最大化地发挥这两者的作用，往往需要一个深入人心、朗朗上口的传播符号，以及在某个时空里，进行地毯式轰炸，从而相对于竞争对手形成绝对优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互联网公司就不同了，他们擅长客户运营和内容运营，以及这种运营形成了独特的运<strong>营式营销</strong>。所以互联网公司是：大胆假设、小心求证、单点突破、快速迭代。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论在公司组织边界、运营动作分解、业绩动线设计、客户决策路径、数据驱动决策等方面，互联网公司的定位，都是公里级的规划、厘米级的迭代。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这才会有把一个定位拆解成：<strong>战略定位、产品定位、品牌定位、销售定位、营销定位、心智定位、需求定位、竞争定位、物种定位</strong>等9个部分。每个部分自成模块，又相互关联；自我迭代，又体系成长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>定位，不是万能的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，没有定位是万万不能的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传统企业的定位方法，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在速度和数据上，被移动互联网落在身后。</span><span></span><span></span></p>
