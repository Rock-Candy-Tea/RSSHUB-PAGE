<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">大家好啊，我是穆宁。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为产品经理我当过面试官，面试过一些人。从面试者到面试官，位置的转变会让人产生很多新的感悟。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时最近也有很多人在咨询我，面试时有没有什么思维方法可以让自己面试offer率大幅提升？有没有什么绝杀秘籍？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你别说，还真的有。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我在这里为大家总结3条产品经理面试时必备tips，都是我自己多年<a href="https://www.niaogebiji.com/tag/%E6%B1%82%E8%81%8C%E9%9D%A2%E8%AF%95" target="_blank">求职面试</a>的经验总结梳理而成，希望能够帮你大幅提升面试通过率。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">01、逻辑感是面试的基本盘，逻辑感差基本没戏</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我为大家描述这样一个面试场景：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">面试官：请简要介绍一下你作为产品经理日常的工作流程？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">面试者：好的，我作为产品经理主要是负责电商产品XX营销模块的，它主要是为了实现促销环节时提升业务指标的诉求...我的工作一般就是接需求，画原型，写prd这些，如果开发环节有我还会盯一下进度...</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">面试者说的有错误吗？没有。作为面试回答答的好吗？很不好。为什么，因为完全没有体现出自己的逻辑性，基本属于想到什么说什么。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">没有逻辑性的表达是面试时的绝对致命伤。基本所有岗位面试如果你向面试官表达出了这样的个人特征，面试结果都很不乐观。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是逻辑性很难体现吗？其实并不是。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最常规的逻辑，就是按照timeline，去梳理每一个时间点，你对接了哪一个角色，做了什么事情。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">稍复杂的逻辑，是基于什么事实，做出假设，提出什么实验方法，得到什么结论；知道事实是否充分，假设的依据是否可靠，结论的逻辑链是否通顺。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这两条是最基础的逻辑性训练方法，其实产品经理需要的逻辑思维锻炼远非于此，还有业务逻辑、数理逻辑、系统逻辑等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里我们先解决面试问题。大家可以按照这个方法去训练一下自己，试着回答几个问题尝试一下，欢迎发到评论区一起讨论，有问必答。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">02、专业感是面试的基础加分项，打造自己人设</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">靠谱是对一个人最基本的评价，专业才是对一个人工作能力的认可。同样，企业招聘的永远是一个“专业”的人，而不是一个靠谱的人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如何打造自己的专业人设？这里有几个小方法。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1. 回答面试官任何问题，一定不能只从产品层面去谈，试着从行业角度切入，再从竞品角度展开，最后落地到产品层面深入描述。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如面试官问：你做的这款知识付费工具日常是如何迭代的？不要直接去讲你日常是如何管理需求的，而是从知识付费行业格局，发展趋势先引出3-5句，再大致介绍几家竞品，自身产品的定位，最后再从竞争、定位、用户角度去引出你是如何管理需求的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2.可以适当使用一些高级词汇，倒不是说每句话都要讲“拉通、对齐、赋能”，但是像基础的一些比如转化漏斗、中台、闭环、耦合，还是可以在合适的场景中用一下的，适当提升自己的专业性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3.具备专业性的人回答任何问题，永远不会以“我感觉”开头。任何结论都需要经过客观的行业调研与数据佐证之后才能得出，而不是做出没有任何依据的主观判断。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如你向面试官表达“我感觉知识付费没啥前景了”，问你为何有这种想法，你又因为毫无准备，说不出来。因为这只是你的主观判断，没有经过深刻思考。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">03、把过往做过的项目讲清楚，是拿offer的底线</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-first-child data-pid="qZyuVeDz">讲过往做过的项目，是产品经理面试时70%左右的内容，可以说是重中之重，没有之一。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-first-child data-pid="qZyuVeDz">相反，如果面试官在面试你时并没有问你太多过往项目相关的内容，这家公司的专业性十分堪忧，面试官可能不知道是来干嘛的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-first-child data-pid="qZyuVeDz">介绍过往项目时，应主要把握以下要点，下面我会一一展开去说，但是请注意一点，分清主次，哪些内容要详细说，哪些要简略说。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-first-child data-pid="qZyuVeDz">项目产生的背景：需求来源是什么？提出的背景是什么？简单介绍就可以了，但注意尽量突出项目的重要性，不要把它说成是一个临时性项目。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="5Cv804cc">项目中我做了什么：需求收集、立项、跟进、上线、运营过程中我做了什么，如何做的？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="5Cv804cc">举个例子，这里描述如何做的不是去描述产品需求文档应该怎么写，而是你描述你思考如何写这份PRD的整体思路。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="5Cv804cc">记得突出自己做的贡献，不要说“在公司领导的带领下....”、“在多方人员的配合中...”这类的话，强调自己的主观能动性和决策价值，把自己定位为owner，不要把自己定位于一个参与者，协调者的角色。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="6kQj1PsG">项目中的难点：项目延期？反馈不佳？我采取了什么措施？结果如何？这一点要准备好好好说说。应对项目突发情况如何应对，有哪些处理心得，是加分项。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="6kQj1PsG">因为产品经理的成熟度，与踩坑经验成正相关性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="6kQj1PsG">强调两点：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="6kQj1PsG">首先，描述过程一定要有条理，不要夸夸其谈，多拿数据说话，用数据衡量项目价值；如果实在没有数据，就用外界评价来衡量你的项目价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="OqsZGtgb">其次，事先必须了解面试企业的产品和业务具体是什么，并且在描述过程中有意识的往相应的方向靠拢。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="OqsZGtgb">比如你之前做的项目是与CRM系统有关的，面试的是电商后台产品经理，那可以多描述CRM的销售线索管理与电商后台订单系统的关联性与相似性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最后，想说最重要的一点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-first-child data-pid="fv5IXDXt">面试没有通过，一定要第一时间询问猎头、HR或者面试官没有通过的原因是什么。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-first-child data-pid="fv5IXDXt">产品经理需要一定的表达能力、交流能力和提问题的能力，用大白话来说，就是：能把话说到点子上。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-first-child data-pid="fv5IXDXt">面试没有通过，不要问身边的人，甚至在网上发帖询问，为什么我面试没有通过？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="FOdFm3C9">网络上谁也不了解你，你也没有详细介绍你的面试过程，怎么可能有人帮你判断“和目标职位不匹配”，或者帮你判断“<a href="https://www.niaogebiji.com/tag/%E9%9D%A2%E8%AF%95%E6%8A%80%E5%B7%A7" target="_blank">面试技巧</a>上有什么硬伤”？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="mC-WRZNE">所以，别问对方根本无从回答的问题，产品经理应该了解这一点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="1muCBod3">直接问面试官或者HR要反馈，就问你自己有哪些不足，这些反馈哪怕只是场面话，也比询问网络上的局外人有价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="1muCBod3">掌握我上面说的几点，并把每一次面试都当做“相亲”，以寻找自己一生的伴侣为目标，认真去做准备，相信你会面试成功。</p>
