<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>业务垂直型的可以按照业务类型分为：ERP、CRM、OA、HRM 、WMS、TMS、呼叫中心客服系统、FMS财务管理系统以及项目管理系统等。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>ERP系统</span></strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>ERP是Enterprise Resource Planning（企业资源计划）的简称，是上个世纪90年代美国一家IT公司根据当时计算机信息、IT技术发展及企业对供应链管理的需求，预测在今后信息时代企业管理信息系统的发展趋势和即将发生变革，而提出了这个概念。ERP是针对物资资源管理（物流）、人力资源管理（人流）、财务资源管理（财流）、信息资源管理（信息流）集成一体化的企业管理软件。它将包含客户/服务架构，使用图形用户接口，应用开放系统制作。除了已有的标准功能，它还包括其它特性，如品质、过程运作管理、以及调整报告等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>ERP是为了提升企业各方面资源配置效率（包括包括销、财、供、物、产等因素），使之充分发挥效能使企业在激烈的市场竞争中全方位地发挥能量，从而取得最佳的经济效益。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>房地产行业SAAS</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>美业行业SAAS</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美业，是一个刚需、高频的消费市场，广义上的美业包含美容美发、美甲、化妆、美体等多个细分服务行业，它既有基础居民服务性质，又能产生极高的附加值。美业SaaS近几年逐渐发展了起来，美业SaaS从以工具谋生存，逐步向平台过渡，给美业人提供一切想要的资源、人、货、服务、课程等等，进而引领行业的风向标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>代表：</span></strong><span>美管加、博卡、有赞美业、慧来客、微盟智慧美业</span></p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>工业行业SAAS</strong></span></p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近些年，实体经济加快数字化转型成为工业SaaS向上发展的重大推动力。随着制造企业对生产误差的要求越来越严格，传统人工响应和反馈机制越来越不适应现代化作业流程。例如在离散型的企业车间生产，产品种类多，工序十分繁杂，如若延续人工方式管理采购业务和供应商，不仅耗费大量人力，还影响生产进度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，在产品同质化现象严峻的今天，依赖人工难以帮助企业及时收集生产数据，容易造成产品出新与技术迭代的滞后，无法满足市场和用户所需。面对激烈的市场竞争，众多企业一开始将目光投向了MES软件。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，传统MES软件多以私有化部署的定制化项目制进行，这种方式往往会带来较长的实施周期以及高昂的项目费用，对于中小企业的负担较重。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>MES软件更为突出的缺点是，其更加强调管控，即告知工人在何时做何事，然而随着工厂中自动化程度不断提升，对于人员的管控需求逐渐减少，数据分析能力成为企业更看重的部分。传统MES软件虽然可以进行生产执行层数据采集并提供简单的看板分析功能，但其始终没有将大量数据真正发挥出价值，并且由于大多数MES为定制化版本，数据格式一般较为混乱，未来同样很难用于数据分析。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>工业SaaS软件则能完美解决以上问题。客户按照需求采用SaaS软件，在公有云平台选择微服务，不仅能使其采购成本得到大幅下降，还可以大幅度减少部署时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更值得一提的是，SaaS软件数据格式统一，除了具备可视化分析功能，还能够加入智能分析决策功能，使数据价值达到最大化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，对于许多中小型企业来说，SaaS软件是采用先进技术的更好途径，它降低了企业购买、构建和维护基础设施和应用程序的“门槛”，只需支出一定的“订阅”费用，便可享受到相应的硬件、软件和维护服务，不少应用传统MES软件的企业已开始向SaaS方向转型。</span></p>
