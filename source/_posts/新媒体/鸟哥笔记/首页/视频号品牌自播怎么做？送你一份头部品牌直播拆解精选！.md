<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在上周，腾讯发布了今年Q2财报。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中对于<a href="https://www.niaogebiji.com/tag/%E8%A7%86%E9%A2%91%E5%8F%B7" target="_blank">视频号</a>的表述，我们有</span><strong><span>两个发</span></strong><span><strong>现</strong>：一是视频号被连续提及的次数和内容占比越来越多；二是在其他业务收入减少的情况下，视频号直播的收入在增加，而且在持续不断地增加。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这意味着视频号将会是接下来腾讯最值得讲的故事。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/166770528263057aa68c31a4.34710023.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于品牌来说，视频号直播作为获取公域流量的加速器、私域成交的放大器，我们在</span><span>之前的文章中</span><span>也经常提及。</span><span>但</span><span><strong><span>目前</span></strong><strong><span>各大品牌都是以什么姿势入局，以及在视频号直播中的表现如何？</span></strong></span><span><strong><span>恰好</span></strong>这个月我们内部组织了期视频号直播拆解训练营，不妨借助同学们拆解的案例来一探究竟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这篇文章也</span><span>将从<strong>打法拆解、</strong></span><span><strong>品牌价值</strong></span><span>两个维度，一起来拆解包括<strong>兰蔻、雅诗兰黛、林清轩、<strong>认养一头牛</strong>、歌莉娅、babycare</strong>在内的众多案例。在此特别感谢提供拆解内容的陈晓芳、阳莎莎、池xx、一只很凶的猫、甜瓜、劲风等同学。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>话不多说，我们直接进入第一部分的打法拆解：</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><span><strong>01、</strong></span></em><span><strong><a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E7%9B%B4%E6%92%AD" target="_blank">品牌直播</a>间的策略打法</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在直播间的策略打法上，我们将围绕场景、引流、留存、讲品、转化、沉淀等6个关键环节来展开拆解，看品牌如何落地执行一场直播活动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1、</span></strong><strong><span>场景</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>除了“人”、“货”的影响之外，直播间的“场”也是很重要的一个因素</span></strong><span>。直播间场景的合理打造，不仅可以第一眼就留下用户，还可以强化品牌理念和产品调性，<span>让用户身临其境，代入场景产生</span>购买欲望。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12713881663057aa7055519.09324823.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们先来看不同品牌是如何搭建场景的：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>兰蔻</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、用虚拟屏替代常规背景板，显得更加高端大气；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、主展示桌+副展示桌结合：主要讲解的产品放在主播面前的主桌上，旁边两张桌子为其他商品陈列展示，这样主播操作会更方便，用户视线也会更聚焦；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、背景设置风格统一，重点突出。在直播时段点进去过几次，直播间的画风、调性、布置基本都是一样的，比如顶部展示直播主题、中间展示商城二维码、右侧有满赠活动轮播及整点抽奖预告，顶部指向关注视频号、对用户而言，能够降低理解和接受成本。购物袋和点赞的位置也用文字做了标注提醒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>良品铺子</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、直播间主题明确，上方贴片写明超值内购会。背景板是线下门店实景，会有很强的代入感；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、直播产品展示用购物篮装载，突出产品之多、性价比之高，吸引用户；</span><span>在赠送产品表面贴上显眼的“送”字，进一步刺激用户消费，增强产品的价值感；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、当需要展示产品细节以及展示产品证书时，会有一个镜头放大的过程，直接明了的让用户了解到产品，更加方便向观众展示产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>认养一头牛</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、主播着装统一（奶牛印花短袖），发饰统一（奶牛犄角发箍），有利于快速提升用户对品牌的认知；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、背景的蓝色与品牌主色调的黄色互为对比色，与福袋的红色互为互补色，营造直播间温馨热闹的氛围，同时有助于用户快速抓住直播重点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>2、</strong><strong>引流</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同于其他直播平台重付费投放，</span><span><strong>视频号直播的流量来源主要依靠品牌在微信生态内的私域流量</strong></span><span>，以及私域流量进入直播间后撬动公域流量的涌入。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/200569940263057aa768c014.34685036.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如上图所示，<strong>林清轩</strong>一场直播的流量来源就包括来主页预约、短视频预</span><span>约、公众号推文、朋友圈宣发、小程序跳转等，具体来看：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、视频号引导预约（订阅提醒），特定时间点提醒18:30 （下班时间后）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、直播宠粉群直播开始提醒（微信群→视频号）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、企微朋友圈宣传（朋友圈→视频号）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、林清轩公众号关注自动回复提示语加直播链接（公众号→视频号）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再看</span><strong><span>认养一头牛</span></strong><span>开播前的引流渠道，包括：</span><strong><span><br></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、视频号，用户在主页预约，开播时收到预约强提醒</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、公众号，①【直播中】提示，公众号显示【直播中】；②重要活动时，文章底部链接直播间预约卡片(可直接进行预约/进入)</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、朋友圈，特殊活动节点时，会发布海报二维码进行预告</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、小程序，首页的悬浮窗入口点击直接跳转到视频号直播间；开播时，小程序首页的悬浮窗不停闪烁，并显示直播中。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>3、留存</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户在直播间停留的时长是决定成交转化和直播效果的重要因素之一，</span><span><strong><span>观看的时间越长，越容易产生购买行为。</span></strong></span><span>除了第一眼的场景和主播外，品牌想让用户留在直播间，最直接的方法就是<span>发放福袋和抽奖。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/61097865363057aa7cd3936.87183600.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上图所示就是不同直播间的福袋截图，通常品牌会</span><span><strong>设置为评论指定内容参与抽奖，借助福袋拉起停留时长的同时还可以提供互动率。</strong></span><span>除此之外，再来看下不同品牌常用的抽奖玩法：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>丝芙兰</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一场直播全程抽6次。首次抽奖设在直播开始后30分钟，在这半小时内进行直播礼赠规则的介绍以及主推品的讲解，并且引导用户订阅视频号，积极发表评论，尽量把握住前30分钟的黄金流量；第一轮没有抽到的用户场控引导参与第二轮抽奖，延长用户在直播间的时长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>兰蔻</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>右侧常驻贴片预告整点组团抽奖，送5ml菁纯眼霜的活动（主推）；左侧上方抽奖前提前15分钟有倒计时及红包福袋提示，通过组队形式（转发邀约好友一起看直播）吸引更多人参与。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>良品铺子</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过福袋抽奖，每隔15分钟一次的弹幕参与福袋抽奖，奖品为腾讯会员周卡等小礼品。同时每小时整点的下单参与抽奖，奖品为荣耀手环、口红等吸引人的产品。在每隔几个推品间就会插入超值的秒杀品，通常为特价促销的超值单品或限量发售的新品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>4、讲品</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在上一篇哥弟的拆解中我们提到过，直播间的用户是流动的，这一波卖完，又会进来一波新用户。所以大多数直播间也都会</span><strong><span>采用爆品循环策略来讲品</span></strong><span>，某款卖得好，就会持续地去卖。</span><span>而单<span>个循环时间为多久最合适？可以根据过往一场直播用户平均停留时长来决定。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/188484840363057aa88ab2f3.63212088.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以不同品牌的具体讲品节奏来看：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>认养一头牛</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播间内购物车的货品有十余种，但是主推品和次推品都是只有一种，其余产品都是简单带过甚至没有介绍，也只有主推品和次推品有挂车链接。直播间内采用的推品方式是主打爆款循环，主推品的讲解节奏是25-35分钟，次推品讲解节奏是15-20分钟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>良品铺子</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播间内购物车的货品共计25种，主推品和次推品都是有几种种，其余产品都是简单带过甚至没有介绍，也只有主推品和次推品有挂车链接。直播间内采用的推品方式是循环过品，主推品的讲解节奏是20-25分钟，次推品讲解节奏是10-15分钟，普通产品讲解时间约为5分钟一款。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>babycare</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开播的前20分钟直播间用低价产品（湿巾/洗衣液）秒杀引流预热，后面直播间以循环过品为主：1个主推品→2个次推品→1个主推品，中间穿插满减赠品的介绍，换款间隙引导添加企微。主推品平均讲解时间20-25min，两个次推品讲解时间共约15-20分钟，满减赠品的规则介绍+引导时间约10min，以此往复来循环。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>5、转化/逼单</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>许多品牌在直播过程中遇到的最大难题是流量的转化，如果先抛开产品来讲，在直播间影响用户转化的</span><strong><span>主要因素就是活动机制、主播讲解、氛围的营造</span></strong><span>。我们直接来看在拆解的品牌案例中，有哪些可以借鉴的玩法：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>丝芙兰</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>6、流量沉淀</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>借助微信触点将流量有效沉淀，也是视频号直播</span><span><strong>复购率高于其他平台</strong></span><span>的关键之一。在其他平台，用户被算法选中推送进直播间，在完成一次下单后，下次再进入直播间完全就靠缘分了。这也使得品牌每次都需要购买大量付费流量，同时这些流量又很难产生复购。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对比之下，</span><strong><span>在视频号直播间将用户沉淀在私域的路径就非常顺</span></strong><span>：直播中弹出企微名片引导用户添加，添加后再次引导用户进入社群，最后在社群内长期培育用户，还可以再次回流到视频号直播间中。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们来看在这次拆解的品牌中，都是如何沉淀流量的：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>丝芙兰</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户在直播间完成消费之后会引导进入社群，并参与进行晒单分享的活动，活动结果会在群内进行公示（未参与购买但有兴趣的用户也可以自行根据指引进入社群，关注后续直播与活动通知）</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、社群：进入社群会有新人入群欢迎语+会员绑定链接；日常直播不会进行宣传，重要活动直播会进行预热，但频率也控制的比较低，因本月是兰蔻新品宣传月，群内每天下午会发问题，回答正确并成功登记的前150名，可以获得会员积分。群内有3人运营，每日轮流运营群，没有明显角色分工。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在主播口播引导的同时右下方会出现企微名片，添加企微后，引导入群。视频号直播时引导添加的企微是直播间育婴师【里昂】（场控）的账号，添加后引导进入社群，社群定位为【直播粉丝群】，群内会分享直播间优惠券链接和直播预告通知，同时里昂也会对直播满减满赠规则进行答疑。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><span><strong>02、</strong></span></em><span><strong>视频号直播对于品牌的价值</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到此，第一部分的拆解已经带大家看了不同品牌在视频号直播中的落地策略打法。那再往深想一层，<strong>为什么品牌非得在视频号中做直播？当下的视频号直播到底对于品牌来说有哪些实实在在的价值？又该如何去抓住这个机会？</strong>我们继续带着品牌案例往下看：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>1、</strong><strong>视频号直播间成为线上门店重要入口</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在2021的微信公开课上，张小龙就提出“</span><strong><span>视频号会成为个人和机构的下一个官网。</span></strong><span>”目前来看，随着视频号功能的逐渐完善以及和各个触点的打通，这句话已经在成为现实。视频号直播也成为了品牌线上门店的重要入口。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/176299543663057aab52ae29.19619519.png" referrerpolicy="no-referrer"></strong></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>（</span>视频号逐渐成为品牌线上门店入口）</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌直播也就像线下门店开店，不播就等于店门永远是关着的，顾客没办法进来，更没办法购买，即使通过私域渠道下单，转化效率也远不及直播间。所以<span>对于品牌来说，自播首当其冲的一点就是</span><span><strong><span>保证高强度、高频率的直播节奏</span></strong></span></span><span>，可以让消费者随时进入直播间选购，提升购物体验。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时由于</span><span><strong>品牌自播往往不依赖于单一主播，而是团队作战</strong></span><span>，在高密度的直播节奏下，这也形成了相对于大主播带货的一大优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们来看拆解的品牌是怎样做的：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>林清轩</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前拥有2个视频号（林清轩、林清轩林山茶花护肤），主直播间在林清轩账号，另外一个仅作为山茶花品牌宣传。</span><strong><span>直播频次是日播</span></strong><span>，从10点开始，全天总直播时长达到了13小时（10：00-23:00），每位主播直播4小时左右，工作时间在线观看人数500-1000人左右，晚高峰时间（20点）直播观看2000-2500人左右。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>歌莉娅</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>歌莉娅<span>在视频号的直播时间从</span>刚开始的每<span>周两三场，每场1-2小时，已经调整到目前</span>的<strong>每周2-3场变成每天3场</strong>，每场4-6小时。在节奏调整后，歌莉娅在视频号直播单日的GMV相比此前，已经有了高倍数的增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>雅诗兰黛</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>雅诗兰黛的直播以<strong>2天一场</strong>的频率进行，从18：00-24：00持续6小时不间断。直播除了作为雅诗兰黛一个拉新转化的阵地之外，还成为<strong>品牌与用户进行售前售后的服务的主要对接窗口</strong>。主播在解决用户问题的同时，也在进行品牌宣传建立用户品牌认知，直播整体节奏跟随品牌走，成为品牌宣传的放大镜。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99653238963057aabd19118.72691933.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>（各品牌接近日播节奏的预约列表</span>）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2、视频号具备更完善的公私域联动逻辑</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年微信公众平台恰好上线十周年，过去十年里，</span><strong><span>无论是新品牌还是老品牌，几乎都在微信生态里建立了自己的品牌阵地</span></strong><span>，比如公众号、小程序、企业微信、私域社群。而如今视频号直播借助这个生态，也形成了与其他直播平台最大的差异，就是可以</span><strong><span>更高效地促成公私域的联动</span></strong><span>，<span>成为了获取公域流量的加速器、私域成交的放大器。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/21703720763057aac32dd24.69933815.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如上图所示，是以良品铺子为例梳理的<strong>微信生态内各处点互通关系图</strong>。我们对比了拆解的众多品牌案例后发现，大多数都是以这套关系图为基准在运行：公众号、企微、社群、视频号都承担了流量的日常培育转化，且触点之间互通；小程序则承载了最后一环的交易。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以拆解案例具体来看：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>良品铺子</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管是公众号、小程序、企业微信、社群以及视频号都是相互贯通的，能够做到相互跳转、相互引流，用户在各个触点都能够领取大额优惠券进入直播间购买产品。目前良品铺子也是为数不多在视频号加入外卖端口的品牌，以其庞大的私域客户数量以及门店数量为基础，利用视频号直播为线下门店引流，实现<strong>线上+线下的全渠道运营</strong>，以此弥补因疫情影响的线下门店，这也是良品铺子相较于同赛道其他品牌的最大优势。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>全棉时代</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>私域以公众号为中心，链接其他渠道触点，且都已形成流量闭环，最大可能做了用户留存。</span><strong><span>值得一提的是，全棉时代的社群有三种：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>①内容社区群：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进入路径：公众号新关注自动回复→海报二维码→添加内容社区管理员→邀请进群</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>社群定位：专门针对全棉时代社区的互动社群，成员在群内一般集赞、分享助力、养成游戏助力等连接</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>②直播宠粉群：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进入路径：视频号主页→添加微信→邀请进群</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>社群定位：直播宠粉群，鼓励用户晒单集棉籽（相当于社群积分，可兑换礼品），直播群内发预约直播抽大奖的海报和链接。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>③官方商城福利群</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进入路径：小程序主页“邀请有礼”→邀请好友落地页右上角“点我享群福利”→领福利落地页→跳转公众号文章添加福利官→邀请进福利群</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>社群定位：高折扣产品推荐，预约直播抽大奖活动</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内容社群，通过推送小程序社区有奖互动活动，将社群内用户回流到小程序社区；</span><span>福利群和直播粉丝群，则通过晒单集棉籽兑换奖品促下单转化，并在群内形成从众效应；另外<strong>通过预约直播抽奖的活动，将用户导流到视频号预约</strong>，在视频号直播中做进一步转化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>3、视频号直播正从「叫卖式」走向「关系式」</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>年初微信公开课的一组数据显示：2021年视频号直播带货销售金额增长15倍，私域占比50%；视频号直播间平均客单价超200元；整体复购率超60%。从中也可以看出，</span><strong><span>视频号直播依托微信生态，老客多、复购高。</span></strong><span>不同于其他平台依靠付费流量，需要不断地做爆品、拉流量，视频号</span><span><strong>更容易基于信任关系</strong><strong>带来</strong><strong>高成交。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>以歌莉娅为例，</strong>作为在全渠道直播的品牌，其在视频号的用户与其他平台有明显差异。</span><strong><span>在视频号平台下单的消费者，对品牌的认知更强，对生活品质的要求也更高。</span></strong><span>比如歌莉娅的山羊绒大衣，在别的平台销量都很少，而在视频号上，最多的时候一晚上卖出去10件。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样的情况在<strong>哥弟</strong>身上也能看到，对比其他平台的直播，哥弟视频号直播的用户发言少、互动较低，但转化情况却非常不错，<strong>很多用户都是静默下单</strong>，且转化高客单价商品也很容易。这些都是得益于微信生态内私域流量的加持，让直播从纯「叫卖式」走向了「关系式」。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上，就是这次视频号直播拆解精选的全部内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于品牌来说，<strong>如果过往已经基于微信生态做了一系列的建设</strong></span><span>，搭起了公众号、企业微信、社群、小程序等矩阵，</span><span><strong>那视频号直播的布局就是顺其自然的动作</strong></span><span>，可以让用户在矩阵内更好地流转起来，进行长期经营。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如果过往品牌的重心并不在微信生态，此时入局也不晚</span></strong><span>。可以从企业微信和视频号同时入局，形成公私域联动，</span><strong><span>将流量留在品牌可控的池子里，减少每一场直播对于付费流量的投入。</span></strong></p>
