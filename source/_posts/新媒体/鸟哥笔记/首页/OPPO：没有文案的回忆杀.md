<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个品牌的长青，除了可以造出很多好产品，还有一个好处：就是可以留下很多好回忆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近<a href="https://www.niaogebiji.com/tag/oppo" target="_blank">oppo</a>发布了18周年的品牌片<strong>《从第一次，到每一次》</strong>，几乎没有文案，但看过的每一个人，都能从中找到回忆的片段。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>▼</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没有一句旁白文案，但每一个镜头都在传达OPPO<strong>「微笑前行」</strong>的主张。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>文案的所有克制，都是为了将“微笑”这一视觉形象表达到极致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短片用一个个“微笑”，串联起品牌18年的历程，与之并肩同行的，是一段段的人生历程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在过往年代里，很多人的第一款产品都来自OPPO——DVD、MP3、手机等等，OPPO的产品陪伴了他们人生的很多个<strong>“第一次”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一次和小伙伴们在家里看DVD。</span></strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>OPPO的第一款手机</span><span>A103</span><span>，被称为“笑脸手机”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品本来早就达到了发售标准，但OPPO按住不发，又打磨了很久才发出来，别人都是抢快 ，他们则是宁可慢也要做到最好，<span>创造了当时的“最慢记录”。结果上市<span>后一炮而红，半年销售破百万</span><span>台。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>标志性的笑脸，不仅传达给了用户快乐的心情，也代表着OPPO潜心钻研、微笑前行的精神。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外仔细看，桌子上还放了很多同时期的经典手机，有贴纸的，有闪钻的，这个细节拿捏的很到位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一次和朋友们寒假旅行的回忆。</span></strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>OPPO P51</span><span>第一次在手机相机里加入了</span><span>LOMO</span><span>特效，像片段中的鱼眼</span><span>LOMO</span><span>、四眼</span><span>LOMO</span><span>，都是那个时代的潮流尖端。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/193840805463286cbb177328.13091444.gif" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018年 OPPO Find X</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>采用升降摄像头的</span><span>OPPO Find X</span><span>和旋转摄像头的</span><span>OPPO N3</span><span>，在当年都是让人眼前一亮的创新。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短片里的看球画面也让人想起了那一年的俄罗斯世界杯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌片用老产品勾起回忆，也不忘用新的科技产品把新的故事演绎，两者之间的平衡恰到好处。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年<span>OPPO</span>推出的首款折叠屏手机</span><span>Find N</span><span>，对应的片段是因为疫情四个男生在线上相聚。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>OPPO全家桶也逐一亮相——智能电视、智能手表、蓝牙耳机、平板电脑、智能眼镜等最新科技产品，都被串进了故事里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/82250807763286cd7f04a99.97280254.gif" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每一款产品，都在用户的生活里扮演着角色；每一段人生，每一幕场景，也因OPPO的产品变得更有温度，从过去到未来，两者一起”微笑前行”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>从第一次，到每一次。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当年份、人群、场景都经过反复推敲和细细打磨，故事顺理成章了，情绪自然而然就烘托到位了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以当品牌片在OPPO的<span>视频号</span>和微博发布后，立刻在评论区引发了一大波回忆杀。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“微笑前行”的道路这么多</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>为何偏偏选了这一条？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年，OPPO的18周年，该如何更深入、更动人地表达<strong>「</strong><strong>微笑前行」</strong>这一主张？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对品牌方OPPO来说，有业绩，有成果，可以说的有很多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对策划方群玉山来说，可以励志昂扬，可以乐观自信，可以舍我其谁，可以稳中求胜……，每一条都有的说，都能叫得响。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但我们讲，一个人可以走得很快，一群人才能走得更远。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个品牌能走过18载，它必然有一群志同道合者在并肩前行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是我们看到OPPO 18周年的核心精神确定为<strong>「你我并肩，微笑前行」，</strong></span><span>切入点落在了</span><strong>“用户与产品之间的并肩同行”</strong><span>，就很顺理成章了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没有大张旗鼓“周年庆”，而是润物细无声的“微笑前行”，这条片子也和这个主张一样，选择了一条“微笑前行”的路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌的高度与用户的温度，产品的怀旧与科技的创新，突出产品的存在，又不能过于广告，典型的既要、还要、更要……</span><span>所以在这条品牌片中，</span><span>处处都体现着分寸感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用留白代替了旁白，舍弃了刻意的精致，保留了生活的粗质，才让故事更有生活感和代入感。</span></p>
