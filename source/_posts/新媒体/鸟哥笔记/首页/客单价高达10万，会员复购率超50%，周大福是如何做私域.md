                        <p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">黄金珠宝行业与其他行业有很多不一样的地方：<span style="font-weight: bold;">低频、高单价、非标，行业普遍重营销、偏线下，获客成本奇高。</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，很多人都会觉得黄金珠宝行业不适合做私域。然而，疫情的出现悄然改变着这个行业，不少品牌都纷纷开始下场转型，<a href="https://www.niaogebiji.com/tag/%E5%91%A8%E5%A4%A7%E7%A6%8F" target="_blank">周大福</a>就是其中的代表例子。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">周大福官方的「2021年财政年度业绩报告」中提到：<span style="font-weight: bold;">周大福线上珠宝业绩超过13亿，私域内售出产品的平均售价较公域电商平台高出约 1.8 倍，销售转化率是普通电商平台约 10 倍。</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">过去的两年多时间里，周大福结合自身的业务特点和需求，持续加码私域。目前周大福已拥有<span style="font-weight: bold;">超过900万名企业微信好友，超过10000个顾客群，客单价高达10万，品牌会员的复购率超过50%。</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么周大福是如何做私域的？下面就为大家拆解。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">本文目录如下：</span></p>

<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1）案例简介</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">周大福集团成立于1929年，至今有着近100年的历史，目前是国内最大的黄金珠宝龙头。2022财年年报披露，<span style="font-weight: bold;">营业额逼近千亿港元(989.38亿港元)</span>。与此同时，公司股价近两年大幅上涨，最新市值达到1374亿港元。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年，周大福主营的黄金首饰及产品占比68.2%，珠宝镶嵌、铂金及K金首饰占比24.1%，钟表业务占比7.7%。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2）市场规模</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中国黄金珠宝市场是全球最重要、最有潜力的消费市场。其年零售总额从1984年的1.6亿元，已经快速发展到<span style="font-weight: bold;">2021年的7200亿元</span>，同比2020年的6100亿元增幅达18%。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">3）用户画像</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中国年轻一代现在是我国珠宝首饰消费主力，尤其是20-39岁的消费者成为未来珠宝消费的主要人群。其中80%是女性。现代职业女性的经济独立性明显增强，购买力也迅速扩大。</p>

<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1、线下门店</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">周大福在线下设置云柜台，用户可自助选择款式，扫码引流到小程序下单。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/nb1crwF74kQspX3kYlvM.png" alt width="622" height="392" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2、公众号</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">路径1：</span>关注公众号后，自动欢迎语中有引导加社群模块。点击跳转社群汇总模块，用户可根据不同需求进入不同的社群。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 576px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/1yGOGRnX7qdne3Wg6lc9.png" alt width="819" height="990" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">路径2：</span>点击公众号菜单栏「福利社」，在福利社顶部轮播Banner图中，有社群引流入口。跳转相应页面后，可点击传送门，直接跳转社群添加页面。也可以添加小助理，由小助理拉入群。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 558px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/EA0RiCLHCaVcJDlAbzhL.png" alt width="800" height="1080" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">路径3：</span>公众号每次推文，在头条文章末尾会附上社群引流入口。点击即可跳转到社群添加页面。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/AxoLJ7hsOqTu0RFKCUnP.png" alt width="591" height="640" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">3、抖音</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">周大福在抖音有4个官方账号，总粉丝数量超过170万。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其中主号「周大福官方旗舰店」，有123万粉丝。内容主要是<span style="font-weight: bold;">产品介绍、代言人活动推广</span>等。另外，每天12:00-00:30时间段会进行直播，以直播带货为主。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/uu1g6OB6fJwuZV44nZTI.png" alt width="614" height="426" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">4、微博</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">周大福在微博有74.2w粉丝，主要视频内容是<span style="font-weight: bold;">品牌宣传、活动宣传、产品推广、金价播报、有奖活</span><span style="font-weight: bold;">动</span>等。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 491px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/08/avuCJHFEg8aui63N50Tf.png" alt width="779" height="1080" data-action="zoom" referrerpolicy="no-referrer"></p>

<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">周大福在小红书粉丝数25.9w，话题#周大福项链 有1亿游览量，<span style="font-weight: bold;">相关的种草笔记数量已超过50W+篇</span>。官方账号主要内容为产品种草、品牌介绍等。</p>
