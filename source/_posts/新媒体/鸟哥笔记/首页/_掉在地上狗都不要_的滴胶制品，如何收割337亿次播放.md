<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>40度高温的盛夏，一个叫“少儿手工diy”的抖音博主穿着毛衣、发了新视频：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“快过年了，我给大家做一个滴胶。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>来，拿个尿杯儿，放根煊赫门儿，我们一生只爱一个人er~</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我还打算给它取个名字，叫嘎香烟灰缸……”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/43297383762de33f65e2f94.15454961.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当网友抱着好奇心，看完了这个1分多钟的视频后却发现，博主认真地做出了一个“掉在地上狗都不要”的滴胶作品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/159193761762de33f9b10c34.20985231.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也难怪路过的网友会嫌弃：“你自己看看，这玩意好看吗”<span>“小偷偷这个，能被同行笑话三年”</span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>评论区里的粉丝赶紧指导网友“入坑”，放下“审美局限”，拥抱“滴胶艺术”：“不准骂我哥，他不做了我看啥”“我本来也觉得不好看，现在看多了觉得好漂亮”“一天不看，浑身难受”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人可能不了解，这个叫“少儿手工diy”的滴胶博主，在抖音有49万粉丝，被称为抖音“滴胶三巨头之首”。另外两巨头，则是风格“师出同门”的“DIY小娜”和“三水手工”——同样的语言幽默，同样的审美特别。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/55943167262de33fc6ee449.82262337.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在三巨头的加持下，<span>滴胶内容</span>在小众爱好垂类中冒出了头。在抖音上，#滴胶#相关话题的播放量达到33.7亿次；在小红书上，也有29万篇相关笔记。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们不禁好奇，“滴胶三巨头”这类账号如何运营？在<a href="https://www.niaogebiji.com/tag/%E7%9F%AD%E8%A7%86%E9%A2%91" target="_blank">短视频</a>平台做滴胶生意能赚到钱吗？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>一条视频百万播放，</span></strong></span><span><strong><span>滴胶内容简单可复制？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“三水手工”是滴胶三巨头中最晚加入的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他从2021年9月开始做滴胶，截至目前在抖音上有了35.2万粉丝。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同于其他滴胶博主全程手工、耗时耗力，他喜欢用“打蛋器”来搅拌滴胶，认为这样“搅得快，搅得匀，还不容易炸胶”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/114936666162de33fd5728c9.01535303.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三水最喜欢做烟灰缸，99个作品里有64条都是烟灰缸相关。品类虽然相同，每个烟灰缸背后的“艺术态度”却截然不同：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>透明滴胶加金箔，放上各种瓶子，为的是告诉朋友“我很爱喝酒”，被点赞了9946次。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>代表“爱情”的绿色滴胶，加上16个台球，为的是让男朋友“篮球打得越来越好，足球踢的越来越棒”，被点赞4418次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些视频的制作周期并不长，主要根据滴胶制品的层数决定，“有些滴胶品一天就能成，第二天发视频。有些要3天才能做完，到第四第五天就能发视频”，三水解释道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据他透露，每条视频播放量在百万次左右，其中热度最高的一条视频播放量近200万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，并非每一个博主都可以成为“滴胶巨头”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新抖数据显示，“滴胶”相关抖音账号，至少有379个。新红数据也显示，“滴胶”类小红书账号，至少有572个。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a data-miniprogram-appid="wx55778a0d7b3048c4" data-miniprogram-path="pages/NewLive/index" data-miniprogram-nickname="新抖数据" data-miniprogram-type="image" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined"><img src="https://ngbjimg.xy599.com/177875496662de3403410b24.06431982.png" referrerpolicy="no-referrer"></a></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以10万粉丝作为筛选标准，上述1332个小红书、抖音博主中，仅8位过了线。三水发现，“各个平台做滴胶的人非常多，但几千或者一两万粉丝的滴胶账号是最多的”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在他眼中，滴胶内容的门槛其实很低，不需要研究什么剧本、文案、剪辑，就是靠说话，跟别的视频比会简单很多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外两位“滴胶巨头”也是靠说话被许多人认识。他们做的同样是手机壳、停车牌、烟灰缸、支架、麻将等家居日用品，但视频的表达方式很有记忆点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p><img src="https://ngbjimg.xy599.com/175933360262de34038df349.84158166.jpeg" referrerpolicy="no-referrer"></p>
<p> </p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“<span>DIY小娜</span>”和“<span>少儿手工diy</span>”制作的同款停车牌</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>50万粉丝的“DIY小娜”，喜欢用叠词说话，比如形容东西会用“挠挠带劲的”，介绍配色会用“紫嘎吱嘎的”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>48万粉丝的“少儿手工diy”山西口音很重，经常有粉丝学他说话，说上几句“倒上买买（满满）的一尿ber”“一天不看浑shin蓝瘦”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们做的东西一样，但顾客会因为每个人说话的声音不一样，表达方式不一样，选择去找他做或者找别人做”，三水这样认为。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>一个滴胶博主有632位学徒，</span></strong><strong><span>做滴胶是门好生意吗？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顶流博主虽然少见，滴胶内容的盈利能力却不容小觑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三水之前和朋友合开着一家自媒体公司，负责主播业务。现在他将更多精力花在了账号上，因为视频效果不错，“赚钱还可以”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>售卖滴胶作品是这些博主们最直接的变现方式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们发现，在淘宝搜索“手机壳”，详情页显示，60%的顾客会选择12-38元价格区间的成品。而滴胶博主售卖的“手机壳”，普遍高于这个价格区间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/26574229362de3403eba117.51763065.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“少儿手工DIY”的抖音小店橱窗显示，售价39.9元的算盘手机壳，已售出95次。售价58元的时来运转手机壳，已售出29件。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/48480118962de34059a5bf4.30287465.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>价格主要高在“定制服务”上。滴胶博主的客户只要愿意花钱，无论是“煊赫门”还是“华子”，“本人大头贴”还是“纸片人老公”，都可以封在滴胶里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三水表示，对于定制品来说，最头疼的反而是没有要求的顾客，“说随便做一个就行，反而是最难的，我会让客服再仔细问一下，比如说想要什么颜色，不想要什么颜色，有没有什么特别喜欢的配件，越详细越好”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而对于客服，三水基本的要求也是“不要有自己的想法，按照客户的想法去做。加了自己的想法，你就可能会出错”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/212727896462de34078a1ea6.53321149.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>左图为客户要求，右图为“少儿手工DIY”制作的成品</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>除此以外，滴胶博主的另一</span><span>个收入项目是收学徒。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三水店铺的背景图就直白地显示着“288元教制作；588元教短视频运营和上热门技巧；1688元给厂家货源”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/55427504062de3407cb3ca5.19297149.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据三水介绍，他们已有632位学徒。“滴胶制作并不是每天都要教，学员发现了一个新款，想知道怎么做，我们就会告诉他注意事项，还会教一些做短视频的基础方法。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>粉丝量级小一些的账号，也不缺付学费的顾客。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小红书粉丝8万的“爱捣腾滴胶麻将”已经开设了5期学徒班，笔记中透露第六期学徒班将于7月31日开课。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/174322218462de340840bc22.52907047.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月7日，抖音粉丝7.7万的“莎随雪手工”也在朋友圈里晒自己的第59位徒弟，对方给她转账599元，并表示“会好好学的”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了更顺利地推进滴胶定制、带学徒这些工作，头部滴胶博主都有个小团队。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“三水手工”的客服透露，“我们工作时间比较灵活，分配到的单子也不多，规定时间做完就可以。老板人也不错，自己还挺满意的”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“DIY小娜”也常在视频里分享自己忙碌的生意和团队成员。保守估计，工作室至少有5名员工，参与到滴胶定制、快递打包等多个环节中。在一则视频中，她透露自己视频中的<span>滴胶，其实都是员工搅拌好的。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/115576257562de3408a3e454.70341973.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>滴胶生意的另一面：</span></strong><strong><span>破圈难、成本高、材料有风险</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>滴胶的“火热”，一定程度上借了网友爱玩梗的“东风”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如“煊赫门”一词，刷屏了各大滴胶博主的评论区。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>起因是“少儿手工diy”，喜欢将煊赫门香烟放入成品中，还会念叨：“一根煊赫门，代表一生只爱一个人。两根煊赫门，代表着只有一个老婆。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于三根煊赫门代表什么，那就看网友自己的脑补和发挥了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/210477398262de340a216727.79446157.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为煊赫门会让滴胶作品画风独特，加上“少儿手工diy”的语言艺术，逐渐成为网友们的心头好，“今天我煊赫门呢”““请求煊赫门限时返场”更是评论区常客。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有金色铜钱、吊牌、算盘摆件等等，看起来土中带俗，又有那么点别致，观众一边嫌弃“确定是二十一世纪的审美”，一边又看得停不下来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18990271862de340a5baf43.38075183.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“<span>滴胶三巨头</span>”的部分作品</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，这样的梗也在一定程度上成为“滴胶三巨头”们的束缚。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有时按照客户要求，博主做的滴胶品视觉效果不错，拍出来的视频流量却反而不如被吐槽的滴胶品，评论区也是一片“这次有点好看，下次不许了”“看到你做的好看，我比死了都难受”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/112813417562de340cb26c27.58872027.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没有槽点加持的滴胶作品，在网友眼中好看得没了特色，可能会面对无人买单的窘境。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><img src="https://ngbjimg.xy599.com/193173549562de340d193770.75731845.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对大多数滴胶博主来说，凭借出众的审美和技术杀出重围是难事。而“滴胶三巨头”的网络热议，为滴胶博主的短视频提供了这样的“启发”：或许不好看，也是一种卖点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但长此以往，或许不能给滴胶创作带来正面影响，甚至会让博主对正常审美产生自我怀疑：为什么还不如煊赫门受欢迎？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>另一方面，滴胶博主还得面对材料的成本问题和安全问题。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在奶油皇后CreamQueen的淘宝店铺内，入门材料包价格在198元到708元区间，大号花边烟灰缸、枫叶烟灰缸、风山体烟灰缸等模具都在20-40元区间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知乎文章《为什么我不推荐你玩滴胶（坑！！！）》提到，真正的大件模具很贵，在脱模过程中如果撕裂，又需要一个新的。此外，还有AB胶、色精、珠光粉、金箔银箔、钥匙扣、分胶杯、砂纸等各类工具，都会成为成本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>材料安全问题，也需要新手博主考虑。有博主提醒孕妇不要玩滴胶，家中卧室不能玩等等。因为滴胶制作过程需要A、B胶搅拌，B胶为胺类固化剂，有毒、有刺激气味。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/184337447362de340d7dd2c3.21819646.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>整体来说，“滴胶”是新手博主可以低门槛尝试的赛道，有手、有材料就能做。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，博主不需要露脸，只要“说说话、做做手工”，幸运的话就能吸引来几个粉丝、几条单子。另一方面，内容本身不愁“自来水”观众，评论区有“煊赫门”“三巨头”的话题，能够获得稳定的曝光。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但如果把“滴胶博主”当作一份长期从事的职业，破圈难、成本高、材料有风险这些问题就需要被认真考虑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喜欢玩梗的网友很快会离开，最终博主要面对的就是“拿个尿杯儿，放根煊赫门儿，做个烟灰缸儿”的工作日常。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>滴胶博主幽默的语言，本意也许不是逗笑观众，而是劝慰自己，日复一日的手工也罢，“苦中作乐”就好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于整个行业，三水表示，还是希望利润更高一些，“现在很多人不接受，或者不愿意花钱在手工制品上。但任何一个10万或者说1万粉丝的手工博主，都是很累的，熬夜是家常便饭”。</span></p>
