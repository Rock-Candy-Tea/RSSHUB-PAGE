<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还记得一个电影《超脱》里，代课老师上台给学生数布置了一个作业：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“</span><strong><span>如果你死了，在你的葬礼上，你的家人或朋友会说些什么</span></strong><span>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果我们把这个问题换成：“</span><strong><span>想象下，如果你被裁了，在你离开公司的最后一天，你的家人、或朋友同事会说些什么？</span></strong><span>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/141584592662ba4743dbcb70.10029201.png" title="我身边几位做产品的朋友被裁了" alt="我身边几位做产品的朋友被裁了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>好好去思考这个问题。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我相信你的内心一定会<strong>产生求生欲。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那就是寻找去降低被<a href="https://www.niaogebiji.com/tag/%E8%A3%81%E5%91%98" target="_blank">裁员</a>后带来风险的事，或是你可以从被裁员的第一天开始，实际上等同于解开了工作对你的枷锁<strong>，你开始迈入了一个新的高度。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但问题是，你现在能吗？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，今天这篇文章我希望你早点阅读，即使你还没有成为产品经理或者已经做了10年产品经理，希望你时刻谨<strong>记今天这篇文章，来自我身边朋友的故事。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>最近身边做产品、运营的朋友裁员的消息接踵袭来。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在今天我和一个朋友聊到关于PMTalk合作的事情才突然告知，最近腾讯裁员，他已经归属到被毕业的团队里了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5997853362ba47446e5856.08964718.png" title="我身边几位做产品的朋友被裁了" alt="我身边几位做产品的朋友被裁了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时另外一个我深圳的好朋友做运营专家，也是在某独角兽被裁员了</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/52646979962ba4744a0fa61.79710725.png" title="我身边几位做产品的朋友被裁了" alt="我身边几位做产品的朋友被裁了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我相信这两位被裁员的同事一定最近不开心，他们的内心可能会有下面的对话。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“裁员是一种对我价值的否定，辛苦为企业挥洒了青春和汗水，你却让我走！留下的同事成了幸运儿，他们可能比我实力强，也可能只是运气好而已。”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“但是为什么运气差的是我？”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“下个月的房贷、车贷来了，我现在能做什么去缓冲吗？”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“互联网的红利已经没了， 我能去其他大厂吗？还是去一个小公司呢？”</span><br></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为在互联网行业工作，唯一的不变就是处于这种变化里。所以</span><span>在互联网工作，越靠近30岁才知道一份稳定的工作会越来越成一种奢侈。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>去大厂工作就是优秀吗？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次5月、6月的裁员潮，不仅是从腾讯这些大厂，还有一些腾讯系、阿里系相关的独角兽公司，裁员的人集中在一些边缘部门或者创新的部门。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们从学校读书，到毕业找工作，所有的一切都在教我们成为自己行业最优秀的人，但是什么是优秀呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>能赚钱的是优秀?</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>还是权利最大的的是优秀?</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>还是最出名的是优秀?</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>希望你自己思考下这个问题的答案。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>听老人说，总是没有错</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在经历校园到社会的工作选择时，父母或长辈们一定让你去事业单位、或者去公务员，因为他们认为稳定，这才是最重要的，现在看来这些建议并不是错误的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使他们已经在时代落后了，但是至少会给你一个比较不差的选择。</span><span>我们不希望自己被房贷、车贷压垮，就算工作做的不好仍然<strong>有机会翻盘。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>翻盘指的是就是这个月的绩效工资可以差一些</span></strong><span>，但是大不了下个月再来。今年钱到手少一些，我可以明年再来，只要我在这里，我就可以有机会再努力一次，可以裁员则等于抹杀了这个希望。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我还记得当初大学毕业的时候父母希望我学医，我没有听取他们的建议，因为我希望能够去创业，所以选择了做产品经理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比你再大城市打拼，很可能连房子都买不起，但是有一份稳定的工作后，至少可以让你很愉快的找到女朋友或男朋友，成立一个家庭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我常常告诉我的学生，当你选择做产品经理的时候，你要首先思考你想成为一个什么样的人？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一个能够改变世界的人？还是希望一份高薪收入，还是只是希望一份体面的工作？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果没有一丝毫的想去改变世界的梦想或者愿景，我不建议你做产品经理，即使你做了你也做不长久，因为产品经理这个工作真的需要有情怀才能长久的驱动，<strong>即使遇到了一个让你吐血的老板。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>裁员让这些人离开了大厂，那么他们会做什么呢？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我了解到的很多人，因为都是多年的技术经验，所以从自己的专业度入手，从自媒体以知识传播的形式开始做分享、甚至是咨询，去获得副业收入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我之前写过产品经理可以做的副业，可以参考这篇文章。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>产品经理可以做的副业有什么？<br></span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>选择以技能谋生的副业，这当然是一个非常不错的方式。唯一错的是在进入大厂的第一天就应该提前做，而不是在打算离开的时候再决定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>城市的资源是不是你的资源？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我身边有一个成都的朋友，给她年薪100万都不愿意离开成都</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“不管给我多少钱的工资，都不愿意离开这里”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是我最近在成都认识一个做探店的网红博主告诉我的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>成都对于她来说不仅是家乡，也是一种生活方式，她喜欢在工作之余进行探店这种副业的生活。平时做视频剪辑运营的她，可能6点下班就可以去接点外快单，然后做自己喜欢的工作，然后再回家和爸妈一起聊聊天睡觉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>你说这是懒惰吗？有人会认为是，因为她不愿意离开成都</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是你也可以说不算，因为他在工作之外还在利用自己周末、下班等所有时间去完成一个拍摄的商务洽谈、拍摄、视频剪辑等工作，用她的话说：<strong>“这是做自己喜欢的事情去顺便赚个钱罢了”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年我选择带着团队回成都来，不仅是因成都是我的家乡，还有一个重要原因是我认<strong>为在成都才有更多的可能。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比一线城市繁重的实体成本，成都可以在这里接触到其他行业和更多的圈子和机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以城市的资源能不能变成你的资源，是我们被裁员后需要思考问题的关键，虽然我们拿着大厂的高薪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是我们天天在冰冷的写字楼里，身边只有技术、研发同学，但是却少了融入进我们衣食住行的元素，比如餐饮、比如穿搭、比如旅游等等，这些们都因工作的原因拘束了我们。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当我们在选择那些高收入、感觉体面的工作，<strong>就要看看我们的时间是否已经被挤走了，还是要死磕产品经理、运营这条职业经理人路线。</strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>产品经理的压力来自于生活的压力</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>35岁的产品经理难的并不是赚不到钱，而是因刚性物质需求，被迫感受到压力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有一句话说得好，<strong>从好的优质环境下降到更差的生活物质是比较困难的，但是变成更好的，确实非常让人愿意接受的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还记得刚开始做公众号的时候， 我一直以为公众号有个10000读者就满足了，凭着一些零碎的广告收入和培训，就老老实实把学生教好，客户服务好，可以把自己养活就足够了，不去折腾创业和工作了，既忙碌又累。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可是当你进入了30+，有了自己的家，你就发现曾经的愿景统统不能实现，<strong>比如我身边很多在一线城市工作的产品经理们，为了在一线城市定居，房贷都是200-400万左右。一个月从1万到2万多的房贷，要还款30年之久。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以参与下面投票看看你的房贷</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那就不能说一份随意的自由职业可以cover以上的成本了。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>给生活换一个方式</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以我身边的朋友偶尔会给我说，是不是应该给自己换一个活法，卖掉一线城市的房子，把贷款还了，拿着剩余的现金去小城市买个洋房或独栋别墅，然后再其他资金存银行就这样找个喜欢的事情，开个超市、或者开个剪发店就这样过了？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我记得我母亲曾经告诉我一句话，说：“钱是赚不完的，生活要有家庭才是幸福”，或许这对我们在裁员的当天可以好好品位这句话，给自己换一个方式。</span></p>
