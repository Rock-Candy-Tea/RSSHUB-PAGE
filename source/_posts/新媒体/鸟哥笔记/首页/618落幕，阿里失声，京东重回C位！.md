<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>又是一年<a href="https://www.niaogebiji.com/tag/618" target="_blank">618</a>落下帷幕，喧喧闹闹持续了大半个月的年中大促适时地画上了一个句号。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>19日凌晨00:09，距离618大促结束刚刚9分钟，身居主场的<a href="https://www.niaogebiji.com/tag/%E4%BA%AC%E4%B8%9C" target="_blank">京东</a>便迫不及待地发布了大促战报。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>累计下单金额超3793亿元，这是京东618今年所呈交的答卷。</span></strong><span>相较于去年3438亿元的成绩，同比增长了10.3%，京东再次创下新纪录！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3994133262b22633234ec2.47574930.png" title="618落幕，阿里失声，京东重回C位！" alt="618落幕，阿里失声，京东重回C位！" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，战报还显示，在618期间，通过京东小时购下单用户数同比增长超400%；京东云每秒用户访问峰值同比增长了154.5%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在作为支撑的物流方面，京东小时购、京东到家为全国1700多个县区市提供“全品类小时达”服务，并有94%的区县、84%的乡镇实现当日或次日送达。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这个“史上最难”618大考中，相较于其他平台对大促数据的遮遮掩掩，京东的逆势增长，无疑是上交了一份极其优异的答卷。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这个618，京东重返C位。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" title="618落幕，阿里失声，京东重回C位！" alt="618落幕，阿里失声，京东重回C位！" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>阿里、拼多多失声</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在京东高调“</span><strong><span>称王</span></strong><span>”之时，它的对手们却纷纷失声。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>向来高调的电商巨头阿里，在这个618表现的一反常态。</span></strong><span>不仅没有与淘宝天猫<a href="https://www.niaogebiji.com/tag/GMV" target="_blank">GMV</a>相关的终极战报发布，甚至连一些细分行业榜单、排名等都不曾公布。这也让人忍不住怀疑，阿里是否在此次618大促中惨遭滑铁卢？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而另一边，对618“不太感冒”的拼多多也只是发布了手机、家电等行业战报。战报显示，在618期间，拼多多手机类目GMV增长了148%，家电全品类销售规模增长了103%等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，关于最重要的大促销售额等数据，官方依旧只字未提。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/56372573662b22633836711.21012010.jpeg" title="618落幕，阿里失声，京东重回C位！" alt="618落幕，阿里失声，京东重回C位！" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就连后起之秀的抖音电商，在618战报中也只是给出了大促期间，抖音电商直播总时长达4045万小时，成交额破千万直播间达183个等不痛不痒的数据。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个618，除了京东之外，显得有些过于安静了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>遥想当年，各电商平台为了在大促期间抢夺用户，纷纷无所不用其极。大促结束后，更是战报频发，你方唱罢我登场，不断刷新着销售记录的高峰。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而此次大促，如果不是热搜上一闪而过的618词条，或许许多人未曾闻其始，便闻其止。此般情况的映射下，便造就了如今“沉默的大多数”景象。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，除京东在自家大促主场实现逆势增长以外，其他平台的数据或许大都有些“难以示人”。既如此，这战报便不如不发。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这，也从侧面印证着，各大平台正刚刚经历过的“史上最难618”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" title="618落幕，阿里失声，京东重回C位！" alt="618落幕，阿里失声，京东重回C位！" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>史上最难618</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>受新冠疫情影响，消费环境整体承压下，今年的618大促一早便被打上了“史上最难”的标签。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去，往往在提前一个月的时间，各大平台便开始了疯狂的广告、活动介绍，消费者也在狂轰滥炸之下急忙打开淘宝、京东等平台，仿佛不买点什么便亏了几个亿。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在今年，尽管自5月下旬便开始大促预热，但声势却远比往年小了许多。</span><span>由于商家们还未能从上半年的疫情影响中走出，各电商平台纷纷从对消费者的轰炸式宣传，转向对商家推出各种扶持政策，以避免其“消极”备战。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>事实上，不仅各电商平台的造势声音渐微，消费者也正逐步趋于理性。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81425677462b22633ae60f2.27677576.png" title="618落幕，阿里失声，京东重回C位！" alt="618落幕，阿里失声，京东重回C位！" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在社交平台上，不乏有“非刚需不购买”、“不买立省100%”等言论。归根结底，一是经济下行之时，消费者更加倾向于“手里有钱，心里不慌”；二是长久以来，平台的各种套路、持续拉长的大促周期，早已耗空了用户的信任和热度。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此之下，如今的消费者更愿意把钱花在刀刃上，只买对自己有用的东西。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，随着各平台头部主播的大撤退，消费者也再度减少了一个感受浓郁大促氛围的阵营。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>往年，许多人蹲守着李佳琦、薇娅等主播的直播间，等着他们带来各种击穿底价的福利产品。这些精致女孩们还纷纷在小红书等平台分享购物攻略，好不热闹。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>去年618大促期间，仅是李佳琦和薇娅两人加起来的带货销售额便超100亿元，无数人的618在他们的直播间度过。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/82903122562b22634031918.59370064.jpeg" title="618落幕，阿里失声，京东重回C位！" alt="618落幕，阿里失声，京东重回C位！" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而如今，随着</span><strong><span>罗永浩官宣退网，李佳琦离奇“失踪”</span></strong><span>，直播电商届四大天王全部淡出人们视线，618大促的声浪也再度削减几分。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>种种原因下，从目前来看，这个“史上最难618”的称呼似乎并不为过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" title="618落幕，阿里失声，京东重回C位！" alt="618落幕，阿里失声，京东重回C位！" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>押注供应链，京东重返C位</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对市场低迷的大环境，京东618的逆势增长就显得尤为珍贵。而这一增长缘由，或许可以从其供应链中找到答案。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在4月份疫情严重时期，物流的堵塞、网点的停摆、履约的不确定性，大大挫伤了消费者的购物热情。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>许多用户都体验过下单一件商品后，商家长时间无法发货，或者发货后包裹在某个中转地停滞许久，最终无奈退货的经历。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而得益于对供应链的押注，拥有自建物流基础设施体系的京东，尽管在上海地区的物流也受到影响，但始终在坚持运营，且能力恢复早于其他企业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/110748383762b226344fbda7.17490033.png" title="618落幕，阿里失声，京东重回C位！" alt="618落幕，阿里失声，京东重回C位！" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近年来，京东也在始终致力于搭建供应链大中台，打通从需求、场景、订单、商品流通等一系列供应链能力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显而易见，京东大力押注供应链的结果，便是货源上更加优质、物流上更加迅速、售后上更加完善。</span><strong><span>最终，便形成了对用户体验的高度保障。</span></strong><span>这也是为何在“猫狗拼快抖”五强争霸的电商时代，京东依然备受消费者信任的原因。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>站在消费者的角度，大促期间同样都是低价，自然会优先选择各方面最有保障的平台购物。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>站在平台的角度，作为京东的主场，此次618大促也是其必须要守住的一场战役，毫无退路。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，有了京东对供应链的持续加码，才有了消费者对京东的信任，也才有了此次大促中，</span><strong><span>京东与消费者的“双向奔赴”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最终，3793亿元的下单金额助推着京东在此次618大促中逆势增长，重返C位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/172008552062b22634970be6.93033894.jpeg" title="618落幕，阿里失声，京东重回C位！" alt="618落幕，阿里失声，京东重回C位！" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一个月前的京东618启动发布会上，京东提出：</span><strong><span>要用“有责任的供应链”全力守护人间烟火气。京东零售CEO辛利军更是表示：今年的京东618意义更加重大，因为我们正迎来“供应链价值元年”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>十九年来，京东已经从一个单纯的电商企业，蜕变成为“以供应链为基础的技术与服务企业”。与之对应的，京东618也从当初的京东店庆，迈向全平台的年中大促。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今潮水褪去，才会发现谁在裸泳。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本届仿佛回归“店庆”状态的618大促，或许便是送给19岁京东的最好礼物。</span></p>
