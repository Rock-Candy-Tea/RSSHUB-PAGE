<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">TCC 推荐：本文作者有过很多产品个性化和定制的项目经验，所参与的项目从企业内部网站、健身健康领域应用到白标产品等，他通过介绍过往的项目，对<a href="https://www.niaogebiji.com/tag/%E4%B8%AA%E6%80%A7%E5%8C%96%E5%AE%9A%E5%88%B6" target="_blank">个性化定制</a>的重要性和附加值、研究的一般方法、考虑的因素做了简要介绍，对我们日常做产品设计考虑用户和客户定制化需求有一定的启发。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><img src="https://ngbjimg.xy599.com/107518444663180dfb718547.03654475.png" referrerpolicy="no-referrer"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>个性化和定制化是我关注很久的话题，实际上是我最早想要讨论的几个话题之一，但总是因为更紧迫的事情而推迟。个性化和定制这两个话题经常出现在我们读到两大热词时：<strong>未来的电商和人工智能</strong>。未来电商的主要内容之一就是 <strong>个性化定制体验</strong>，因为它本质上就是用户/顾客，在基于他们的喜好、期望甚至基于他们的浏览记录的推荐帮助下，获得满意的产品/服务。定制是这些体验中另一个重要的组成部分，某种意义上说，产品体验需要在用户/顾客的浏览方式、展示内容的优先级和最终利用产品本身的方式上做出一定的调整。人工智能</span><span>（AI）</span><span>就是为实现上述功能提供可能性的机制，同时还能预测可以增强产品体验的功能，从而进一步提高客户的停留时长。这篇文章着眼于我在该领域过往的经验，看看它们如何深远影响了产品设计体验以及所能提供的附加值。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>个性化和定制化。根据维基百科，个性化可以被定义为“</span><span>（广义上称为定制）</span><span>包括定制服务或产品以适应特定的个人或部分群体。”各种各样的组织机构利用个性化提升用户满意度、数字销售转化、营销结果、品牌建立、改善网站数据指标和广告推广。<strong>个性化在社交媒体和推荐系统中是关键元素</strong>。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>维基百科的描述很准确且非常适用于这个话题。在我的职业生涯中，我处理过很多次个性化的案例，第一次是与金融服务业相关的一个稳定的产品，即内联网的开发。简单地说，该特定项目的挑战是个性化和定制的程度允许同一内部网的用户使用。根据测试、观察和用户访谈的结果，大致的结论是，不同类型和职责的用户，最终在使用网站的时候有一个共同点，即：<strong>创建简单清晰的快捷方式以获取需要的信息</strong>，<strong>能够管理自己的仪表板和网关页面</strong>，<strong>突出他们在公司任职期间的产品和个人旅程相关性更高的模块</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>在这个提案开始的早期，我参与了一个更复杂的个性化旅程，是围绕健身和健康主题展开的。这个案例中，产品旅程关注的是全渠道体验</span><span>（即它们应该始终保持一致、最优化、无缝衔接、协调性和协作性）</span><span>，因为这个 APP 创立的目标之一，就是要超越健身房体验，成为用户在任何设备上都可以健身的伙伴</span><span>（包括可穿戴设备、桌面端和移动端）</span><span>。为此，产品体验要考虑到用户身份、生物识别，还有习惯和目标，并将这些信息以游戏化形式呈现，来保持用户的参与度和积极性，并持续关注他们的健康之旅。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个研发过程耗时很久，包含了全球范围内针对不同用户的调研，所有的目标都是为了 <strong>抓住用户的兴趣</strong>，让他们与应用程序建立密切的关系：包括通知他们的身体状态、并提供下一步指导以保持当前势头。在这种情况下，该应用就具有定制和个性化的一面，不仅触及最终用户，而且在一定程度上也触及到控制应用生态系统的不同用户群。所创造的功能的弹性使得产品在销量、安装量、使用率上都取得成功，使组织达成了产品旅程最初设计时确立的里程碑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>因为这些过往的经验，我可以证明大多数我参与的产品旅程都真正拥抱了个性化的概念，因为它带给顾客的价值很清晰，就像网络产品参与度/信任度层次所描述的（</span><span>被总结为五个等级，从下至上依次为：基础信息信任；个人偏好信任；个人信息信任；金融和敏感信息信任；承诺建立长期关系的信任）</span><span>，这可以带来用户和品牌本身之间的长期关系。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><img src="https://ngbjimg.xy599.com/154251321863180dfba1b955.71167175.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>个性化和定制化如何带来更好的用户体验和使用时长呢？用户喜欢被理解、被在意，最终发现他们的需求在所选择使用的产品中表现出来。<strong>个性化和定制化让产品更贴切地体现用户使用产品的习惯</strong>，<strong>同时不远离产品设计的本质和精髓</strong>。举个例子，在我过往的产品设计经历中，我曾参与设计一个 SaaS 平台项目，用户可以控制 APP 的界面</span><span>（一定程度上）</span><span>。产品的白标方面</span><span>（白标产品：由一家公司生产的产品或服务，根据其他公司所需改造包装成该公司自己的产品）</span><span>，使客户能够访问平台，并通过调整一系列变量改变最终产品。这种定制使他们将特定产品视为自己品牌的延伸，这是用户体验其品牌生态系统的额外渠道。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谈到品牌，正如我在之前的文章提到的，很重要的是他们的产品旅程必须是深思熟虑的、聚焦的和可适应的，这意味着不管他们创造了什么故事，它需要激发信任、合法性和自信。在创造白标产品的时候，<strong>关键是提供顾客所需的弹性和灵活性</strong>，不仅为了解决他们在操作任务方面的挑战，同样重要的是，保持他们的品牌势头。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>在产品设计中考虑定制化能力，需要考虑各种因素，包括最明显的比如界面的复制，以及其他因素比如 UI 组件本身</span><span>（用户有时会想要重新调整）</span><span>，更不用说作为产品基因基础支柱的字体、颜色和图标等。<strong>决定用户定制化的程度和方式时</strong>，<strong>需要考虑到它背后的整个战略</strong>，这需要在设计思考过程中提出，<strong>所有参与者需要对前进方向有清晰一致的认知</strong>。它是核心元素，将渗透和影响到众多因素，包括本地化、可访问性和多平台执行的考虑。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>产品设计中的个性化和定制化之旅</span></strong><span>，<strong>确实是构建更深层关系并最终保持客户忠诚度的有力途径</strong>，但要实现这一点，这些策略需要在项目开始就予以考虑，而不是事后考虑或只作为趋势</span><span>（最终可以被摒弃）</span><span>。用户和顾客知道他们什么时候是流程的一部分，什么时候他们的声音和洞见被予以考虑，就像他们理解什么时候他们被公开操控以实现某种结果一样</span><span>（这在产品设计的暗黑模式话题中深入讨论）</span><span>不管采取什么策略，最好是清楚其所有的后果，以及正在构建的未来和用户产生的互动。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现实检查。每个组织都必然有自己的个性化和定制化方法。成本、开发时间、客户支持、技术复杂性等诸多因素都会影响此类战略的定义。对于设计师和在设计流程上的伙伴来说，最基本的是要了解定制化和个性化的过程是什么，当然，这往往是从收集用户、经济/社会/政治的背景知识开始，更不用说他们的痛点、机会点和渴望的需求了。但在创建产品时，即使这个过程受到了其他影响因素的阻碍，了解用户想要什么和需要什么会得到总体方向和需要考虑的事项。</span></p>
