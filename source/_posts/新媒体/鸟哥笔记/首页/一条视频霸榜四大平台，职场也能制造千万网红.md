<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么样的短视频作品，才能做到同时上四个平台的热搜榜？来自济南广播电视台的@主持人yoyo酱 给了大家一个答案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该博主于7月发布了一则记录“工作日常”的作品，视频中yoyo酱直播前试戴闪光牙套，却被卡住取不下来的“沙雕”行为让人哭笑不得，多亏身旁的男主播温柔救场，才让节目得以顺利进行。两人的自然配合，让网友们一边笑，一边磕，乐此不疲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这支作品不仅让她在抖音单周涨粉120W，更是同时登上微博、抖音、快手、B站“四大顶流”的热榜。</span></strong><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/123889199863178f6fd7e901.49590132.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/203134715163178f700cff83.54874220.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源网络</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这不是主持人在短视频领域的第一次胜利，也不是这类职业型博主在平台的第一次出圈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个又一个职业博主冒头的同时，也不禁让人感叹，都是打工人，他们的生活为何可以拍摄得格外有趣？格外吸引人？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>职业博主的ABC式</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传统媒体、媒体人转战短视频似乎有一些“天然的优势”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早期入驻短视频平台的人民日报、新闻联播、央视新闻等老牌媒体，因为其内容的权威性、专业性、时效性，广受网友们的好评，早已成为抖音平台内粉丝数上亿的绝对头部账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，如@尼格买提、@王小川等传统媒体系统内的主持人，因为能说会道的职业特性，加上自身知名度的加持，进入各互联网平台后，也展现出很好的适应力，在短视频平台上的风头甚至会超过一些当红明星。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/129593216463178f7057f007.33185177.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音@尼格买提@王小川</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这之中，<strong>于短视频作品中，塑造和日常工作时“严肃正经”形象不一样的反差角色，成为了不少主持人的流量密码。</strong>“凡学大师”撒贝宁、“押韵狂魔”朱广权、“怼天怼地”康辉……都曾给大家留下深刻的印象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>类似@主持人yoyo酱 这种记录“搞笑主播日常”画风的账号，也是早就有迹可寻，比如浙江电视台主持人@主播潘小蓉，也曾将自己的创作方向聚焦于“女主播开播前10秒的N种社死方式”上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/129750649163178f70846986.74819632.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音@主播潘小蓉</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这种表面正经、私下诙谐的风格两相对比，产生的剧烈反差感，助力了很多主播的出圈之路，也成功让他们“走下神坛”，拥有了一个更接地气、更贴平台的短视频人设。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，前文提到的yoyo酱本次的出圈，还有一个很大的原因是<strong>她与男主播颢鑫之间隐约的cp感。</strong>一系列作品中的各式“甜宠剧情”，让两人的账号进一步“升温”，yoyo酱账号飞速涨粉的同时，男主播@主持人颢鑫 的账号粉丝数量也在本月初突破百万。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/21454321063178f70db7d01.64807747.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音@主持人yoyo酱</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，除了主持人这一职业外，近来有不少创作者，都在尝试将自己的“职场生活”搬上短视频的舞台，其中还有很多人取得了非常亮眼的成绩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如从搞法律科普向搞笑领域进军的“罗翔接班人”@俏佳人xxx、执迷于让别人说出想法的重庆理发师@山城小栗旬的理发日记。这类职场达人的一次次出圈，无疑成为很多“蠢蠢欲动”想要投身短视频领域的打工人的强心剂。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前看来，各平台职业型博主走红的内容模式，主要分为三种。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第一种是主要展现与大众认知相悖的反转魅力，创作方向聚焦自身职业环境下的幽默趣事。</strong></span><span>前文提到的@主持人yoyo酱、@主播潘小蓉 都算是这类型的创作者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第二种是将刻板知识进行短视频化表达，让观众能够在嘻嘻哈哈的同时了解其职业内容。</strong></span><span>这之中的佼佼者众多，以@罗翔说刑法、@山城小栗旬的理发日记 最具代表。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/159151764663178f712fa8a5.30504463.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：B站@罗翔说刑法@山城小栗旬的理发日记</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>第三种则是致力于打造强“专业”人设，用个人硬实力折服一众网友。</strong></span><span>比如活跃在各平台的财经大v，前段时间刷屏抖音的@同传姐妹花 等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>职场外的第二桶金？</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>流量之外，作为很多人正职或兼职的选择，职业博主们在各个平台上的变现能力也相当可观。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，利用流量带来的庞大粉丝群体，接广告、做推广、搞直播带货，职业博主们在这些属于网红的“传统变现模块“表现得如鱼得水。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年6月@主持人王芳 的直播间单场销售额一举突破百万，并且还是以纸质书这样太不讨喜的品类作为主打创造出的成绩，十分让人惊喜。<strong>根据第三方平台数据，过去半年，她的直播间销售额已经达到4.12亿，累计观看人次1.36亿。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，<strong>她依然维持着一月开播50场左右的直播频率，即使是包含春节假期的二月，也几乎是天天开播，</strong>称得上是直播间的“劳动模范”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从直播间货品排布上看，她的选品依旧是图书音像占大头，辅以礼品文创、母婴用品、服饰内衣、食品饮料等类型商品。可以看出，目前直播间的目标客户，仍是宝妈宝爸这部分人群，这样的货品结构不仅满足了他们为孩子选购学习用品的需求，也能为其提供日常生活需要的家居好物。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/115533153363178f71685121.25216840.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音@主持人王芳</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，得益于自身在职业领域的成就，<strong>利用自己的学识、技能变现</strong>的方式也有很多，那些教授唱歌技巧，推广职业技能课程的直播间都是生动的例子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而上文提到的@同传姐妹花，也选择了这样的方式。博主毕业于同传专业中的世界名校英国巴斯大学，这一身份为她的个人能力做了很好的背书。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>翻看其账号的作品，也都是在展示其高超的同传技能。面对新闻联播中的各种复杂词汇与表达，不仅翻译迅速、行云流水，发音也十分好听，让人叹服，因此还掀起了抖音上一股“模仿热潮”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/85354508763178f71a51971.68623634.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音@同传姐妹花</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她也抓住机会，在流量高峰期开始推广自己的英语课程。现在点进账号首页的橱窗，可以看到两本学习工具书和两门英语课程，单品最高销量达到1.4W单。除此之外，她们还选择在抖音直播间一边分享英语学习方法，一边进行课程的售卖，过去30天直播间累积销售额达到200w，销量最高的也正是英语的口语课程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得关注的是，<strong>与流量上做出的亮眼表现不同，那些主打搞怪、戏精，弱化自身职业特性的达人，账号的变现能力就显得有些平庸。</strong>就可见的数据来看，他们大多是选择了视频植入广告这一变现方式，进行直播或是尝试自造ip的达人数量很少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>哪些职业适合“搬”进短视频？</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众所周知，达人进行内容创作并不难，难的是持续维持创作。如果能成为一个职业型博主，将这个和日常生活息息相关的事件融入创作，在维持未来创作上可能会更有优势，对新手创作者而言，也能减轻其创作压力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，什么样的职业适合被搬进短视频舞台呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在卡思看来，<strong>如果创作者的职业具备一定的服务性质，可能是最合适的选择。</strong>比如律师、中介、教师等，买卖双方能够依赖网络完成大部分交易内容的职业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，<strong>如果本身的职业不具备服务特性，创作者还可以通过知识付费、技能培训等方式，完成短视频变现的“适应性改革”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对普通人而言，知识付费模式启动成本低、毛利润高，<span>不受供应链、库存以及线下门店的限制，的确是短视频变现的不错选择。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，伴随鲜花出现的必定也有质疑，职业型博主们面临的环境并非全然乐观。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，如上文提到的，<strong>很多博主难以顺利实现从流量到变现的跨越</strong>，只能“静静”看着账号流量的逐渐消失。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，<strong>将个人职业、生活展示给更多人审视的同时，指责、质疑就必会如影随行</strong><strong>；而博主将更多精力投入短视频领域后，是否还能保持好自己的专业能力，我们也不得而知。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有小红书博主发文指出自己越来越讨厌平台上的这些职业博主：</span><span><em>因为职业博主意味着他/她/他们没有“Day job”，Ta就提供不了核心价值给他的受众，TA就不垂直，TA用户粘性就不高，TA就没有转化。</em></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em>特别是那些做效率/成长类型的博主，之前刷到一条高赞的小红书，说Ali Abdaal 弃医从管之后就不喜欢他了。搜了一个几年前做学生时崇拜关注的效率成长博主，内容好同质化，说一个泛泛的“新手如何xxx”的主题，最后插个软广，一点真实的problem solving都没有。</em></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许这段话正指出了很多职业博主正在经历的痛苦循环。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，生活小事想要搬上视频舞台，必将经过一定程度的夸大与改写，进行更戏剧化的改编。如果掌握不好真实与虚构之间的平衡，粉丝对博主内容的信任与认可亦会大打折扣。</span></p>
