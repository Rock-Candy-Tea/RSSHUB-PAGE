<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在大家将目光投向苹果发布会的时候，<a href="https://www.niaogebiji.com/tag/%E4%B9%90%E8%A7%86" target="_blank">乐视</a>也开了一场秋季沟通会，发布了三款新电视，和一款外形和苹果傻傻分不清，价格却只有苹果十分之一的新手机，有网友因此调侃询问，“苹果的手机壳是不是能和乐视通用？<span>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了发布新手机新电视，乐视品牌相关负责人夏晓艳还表示，今年“<a href="https://www.niaogebiji.com/tag/%E4%B8%9C%E6%96%B9%E7%94%84%E9%80%89" target="_blank">东方甄选</a>”出来后感觉很受鼓舞，乐视正在努力成为下一个新东方，“员工完全可以给自己的公司代言，乐视为什么不可以？”</p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13664109386322377369e504.06355697.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7月底，乐视便在微博发布招聘消息，表示招募主播，欢迎教培行业从业教师。而在抖音上，乐视官方账号和旗下的达人号，已经陆续开始了直播带货尝试，主播大多是自己的员工，似乎正在探索新的业务道路。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">逐渐沉寂的乐视，在这两年又重新回到了大众的视野。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年春节，乐视更新图标“欠122亿”，被网友吐槽“乐视干脆改名乐观好了”。今年4月，乐视依靠《甄嬛传》每年还有千万收益登上热搜，7月，一位博主发布的“乐视过着没有老板不内卷的神仙日子”微博，引发众多打工人的羡慕之情，乐视也顺势开始了直播招聘。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在一个个与乐视相关的热搜下，不少网友都表示“没想到乐视还活着，而且活得还不错”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在贾跃亭离开后，不断靠着自嘲营销进入大众视野的乐视，正在努力自救，还想要成为下一个“新东方”，虽然碰瓷意味不小，但已经有了不小的关注度。对于依旧背负着巨大债务问题的乐视，能否在竞争激烈的直播赛道突出重围，还是一个未知数。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11607546406322377393c0f9.83744911.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>乐视学习or碰瓷新东方</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">乐视不是第一个想要学习东方甄选，成为下一个新东方的企业。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在“东方甄选”刚火爆出圈的时候，李国庆就连发5条视频点评，认为俞敏洪把新东方的老师推出来当主播，自己当助阵露个头，做成可复制的模式，这一点是一个大型企业转型直播电商时非常值得学习的，并且表示“我也正在布局这个事。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/101693397963223773bcb535.32429944.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">之后，趣店罗敏，也通过预制菜进军直播带货，表示自己“我和新东方的俞老师一样，在做业务转型。”不过现在，趣店抖音直播已经暂停，公司财报也显示，趣店正在暂停预制菜项目。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在9月8日秋季沟通会上表示将要“努力学习东方甄选”的乐视，在7月底就已经发布了招募主播的内容，并且陆续进行过了几次直播带货，但目前看来，乐视的直播带货效果并不算好。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13004937526322377411fed7.97182395.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">灰豚数据显示，乐视官方抖音账号近30天直播了13场，销售额在7245元。而新抖数据显示，乐视公司旗下的账号“市场部的日常”有着229万粉丝，但近30天的直播销售额为3.19万元，最近一场直播在7天前，销售额为1.28万元。而“东方甄选”近30日的直播销售额在6.74亿元，是乐视旗下两个账号直播销售额的16000多倍。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">盘点“东方甄选”的直播带货矩阵账号和乐视旗下的直播带货账号，可以发现，虽然乐视在学习新东方，有着诸多共同点，比如公司员工做主播、售卖公司自有渠道商品等，但在“人、货、场”三大要素上，两者还有着诸多不同。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1.主播的不同。</strong>之前东方甄选依靠“知识带货”出圈，董宇辉、七七、yoyo等曾经的新东方老师堪称直播界的一股清流，直接刷新了人们对直播带货行业的认知，并且新东方CEO俞敏洪亲自下场，在东方甄选刚开始直播带货的时候在直播间引流。而从乐视最近几场直播可以看出，主要还是乐视员工在直播，并没有走“知识带货”的直播道路，直播话术也不算熟练，基本就是在介绍商品和优惠活动，而乐视最大的IP可能还是“下周回国”的贾跃亭，并不能为直播间引流。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2.带货商品的不同。</strong>东方甄选目前以农副产品、食品和日化百货为主，这些品类客单价不算高，复购率高，消费者决策链路会更短。而乐视带货产品主要是旗下的智能硬件和小家电，这些产品的客单价更高，复购率却不算高，虽然自有产品会有更强的定价权，避免价格战，但也要面临其他品牌的竞争：乐视推出的Y1 Pro手机，就曾被测评博主评价售价499元也“不值”。乐视旗下达人账号“市场部的日常”带货商品除了乐视的产品，还有不少其他品牌的食品、日用百货，但带货数据并不算理想，也没有太强议价权，并没有什么优势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2138882246322377447df07.79904813.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3.直播间内容与场景设置的不同。</strong>东方甄选的直播间已经有了较为成熟的镜头切换和场景设置，并且在近期的“陕西行”主题中引入非遗文化的现场表演，丰富了内容直播，每天还能直播18个小时。而乐视的直播间设置还是较为单调的绿幕背景，基本只直播2小时左右，与其他直播间并没有形成差异化，也难以吸引用户在直播间停留。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在“人、货、场”都不具备优势的情况下，乐视想要成为下一个新东方或者说东方甄选，还需要更多的投资和运营，但在乐视公布的2022 年半年度报告中显示，乐视的负债总计达到了222亿元，也难以投入太多的资金去经营直播带货。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1857286954632237746fbda5.55789306.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>没有老板的“神仙日子”</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">乐视也曾有过风光的时候。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2010年8月12日，乐视网在深交所创业板成功上市，成为“A股网络视频第一股”。直到今天，乐视视频的官网公司介绍中，都还标明自己“是行业内全球首家IPO上市公司”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/109970192363223774dea059.88889758.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但在2016年年底，贾跃亭自曝财务危机后，乐视就陷入了资金危机，2017年7月4日，贾跃亭登上去往美国的航班，“下周回国”成了一个被网友们调侃的梗，乐视也陷入低谷。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这之后的五年时间里，围绕乐视的新闻与消息更多是负面信息，也让不少人以为乐视已经因为百亿债务“倒闭”，相关的信息也再难引起好奇。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">乐视再次出现在大众视野中，是2021年春节的瓜分红包。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3120934596322377542b0c7.62125066.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年春节期间，众多app在图标页面表示瓜分“多少亿”，乐视则更新了图标“欠122亿”，表示“分不起”，这一反差直接登上微博热搜。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">尝到了流量甜头的乐视，之后又在app图标上更新了“老板造车美利坚”“夹缝生存苦亦甜”等标语，将自嘲自黑进行到底，也被网友视作“最会整活的公司”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">今年4月，因为各种剪辑解读片段在短视频平台火爆的《甄嬛传》上了热搜，幕后手握版权的乐视也进入了大众视野，有知情人士表示乐视每年靠版权就能躺赚上千万，乐视也随后发布微博，配上了《甄嬛传》的表情包“有你，是我的福气”，一众“甄学家”网友也开始玩梗，表示《甄嬛传》保了乐视一世荣华。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17042844863223775741249.09289298.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7月，一位博主表示，乐视现在还有400多位员工，并且“幸福感是很多老乐视人留下了的理由”，随后，就有网友挖出乐视的“神仙操作”，在2021年12月，乐视宣布给员工涨薪，有网友透露乐视人力资源部的内部信还表示，2021年乐视已经实现了经营利润和现金流双平衡(在不考虑历史债务影响的前提下)。而在今年5月，乐视还表示公司曾于2020年Q2季度宣布未来不再有大规模裁员计划，之后将继续信守承诺，居家办公状态下员工基本薪酬和福利保持不变。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“没有老板的乐视没倒闭，员工不内卷不996，过上了神仙日子”，这一反差也让乐视在多个平台引发了一波波讨论。有网友表示，乐视这意思是“神仙哪有我们过得舒坦”，不少人在乐视官方微博和抖音账号下留言询问还招不招人，乐视也顺势开启了直播招聘。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/182903694563223775aa3003.43150909.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">勇于自嘲自黑的乐视，借着互联网又火了一把，虽然一些话题背后有着营销的痕迹，乐视官方也在每个相关话题下借势宣传自己的硬件产品和视频平台，但网友们对这坦诚的态度并不反感，围绕乐视的诸多负面消息，反而成了用于调侃的话题素材。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/48783073663223775e0f818.19743054.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>乐视，不靠《甄嬛传》</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">依靠自嘲自黑重回大众视野的乐视，目前主要有两个公司，一个是乐视网，一个是乐融致新。2018年，乐融致新被剥离出乐视网，融创成为控股股东，目前两个公司独立结算，而乐视网是乐融致新的股东。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这两个公司也对应着乐视的硬件与内容两大业务：乐融致新主要在智能终端硬件业务与内容运营，而乐视网主要业务则是乐视视频和云计算等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">虽然沟通会主要推出了超级电视和手机的相关消息，乐视也一直在硬件上发力，但是在乐视网信息技术（北京）股份有限公司的2022 年半年度报告中，乐视网表示“公司主要收入来源为网络视频行业的付费会员业务、短视频运营业务、版权业务及电视剧发行收入等。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在所有影视剧中，最被网友所熟知的就是《甄嬛传》。之前，微博就有热搜表示《甄嬛传》每年都能给乐视带来千万元的收益，但在乐视的秋季沟通会上，乐视官方表示，甄嬛传带来的收入不过2%，之前网上盛传的乐视靠出租办公楼生存，但租金也不过占据了收入的6%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1758495489632237760edf96.34670076.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">乐视在内容上有自己的优势。在乐视巅峰时期，投资了不少影视剧，也陆续购买了一些版权，除了《甄嬛传》，乐视还有《征服》《天道》等经典电视剧，而乐视的2022年半年报中也表示，在上半年，乐视视频平台新上跟播剧29部、片库剧207部，电影144部、动画62部。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在短视频时代，乐视也看到了短视频和二创的潜力，之前就和快手合作授权二创，在2021年11月还推出了“鲸群计划”，授权旗下影视剧版权给二次创作者，这些影视解说和短视频剪辑让乐视旗下的影视剧获得了不少流量，而这些流量也能带动乐视视频平台的发展。2022年上半年度，乐视视频网站新增注册用户约300万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">相比之下，乐视的硬件业务则不温不火，在第三方电商平台上，乐视销量最高的电视评价只有500条，其余几款电视的评价虽然有10万+条，但相比小米、海信等品牌电视动辄100万+条的评价并不算特别多。而乐视手机显示的评价只有1000+条，相比于其他手机品牌厂商而言，这个数据连零头也不算。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/415688152632237764a0f19.38514728.jpeg" referrerpolicy="no-referrer"></p>
<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为自嘲、神仙日子和《甄嬛传》出圈，给乐视带来了不少流量，而背负债务的乐视也想要将这些流量进行转化，让流量变现，成为公司业务的又一助力。新东方的“东方甄选”，或许让乐视看到了业务转型的一个可能性，但因为债务等原因无法在直播上投入太多的乐视，之后或许还是只能依靠着《甄嬛传》等影视剧的内容业务去赚钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>重回大众视野的乐视，想要学习东方甄选，突出重围“做下一个新东方”，目前看来还很难。</span></p>
