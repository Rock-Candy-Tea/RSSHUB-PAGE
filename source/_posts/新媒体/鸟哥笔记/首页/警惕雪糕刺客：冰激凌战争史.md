<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年夏天，梦龙因为在中国和欧洲采用不同配方的双标事件而闹得沸沸扬扬，也让其在国内苦心经营多年的高端形象一落千丈。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/164251758962c43866445358.84825953.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而不管是梦龙，还是大家熟悉的可爱多，都隶属于同一个冰淇淋品牌——和路雪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这一期内容，我想和大家来聊聊和路雪和雀巢之间的江湖争斗。</span></strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><em><em><span>01</span></em></em></span><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们现在所说的冰淇淋主要源于意大利，18世纪开始传入美洲，而后在19世纪随着制冷技术的发展，冰淇淋逐渐实现了工业化生产。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二战后，以联合利华为代表的跨国公司成了知名厂商，「Wall’s」就是其收购的冰淇淋品牌之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1992年，联合利华决定在中国布局冰淇淋业务，恰巧彼时还在轻工部计划司的曾锡文受命组建群星集团。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方需要入场资格，一方正寻求新项目，双方一拍即合，随即组建了合资公司。群星集团出资2000万，占股15%，「Wall’s」占股85%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顺便说一句，联合利华最先找的合作伙伴其实是北冰洋，但由于北冰洋坚持要保留品牌名而作罢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1993年，「Wall’s」以中文「和路雪」之名正式打入中国市场。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/43339060162c438666cb5e3.51198061.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过那时候大众普遍消费的还是冰棒，北方叫冰棍儿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用我常用的食品分类法来理解，我们童年吃的冰棒，属于甜食纲，冷饮目，水冰科，冰棒属，它们最主要的原材料就是糖和水。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/33787785662c43866a7e485.36582344.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而冰淇淋和雪糕，同为奶冰科，两者所需的基本原材料大致相同</span><span>（水、牛乳、奶粉、奶油或植物油脂、糖）</span><span>；分别是冰淇淋属和雪糕属，主要差别在于两者对脂肪、蛋白质含量的要求不同。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/29383157062c43866e00704.21596943.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>冰淇淋的脂肪、蛋白质含量不低于5%和2.2%，对乳和乳制品含量也有一定要求；而雪糕的脂肪、蛋白质含量不低于1%和0.4%，对乳和乳制品含量则没有具体的要求。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>那么，和路雪是怎么做到在短时间内就打响了知名度，并且夺得市场话语权的呢？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只能说和路雪有个不差钱的好爸爸，而且很敢做。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在当时国内零售市场还是以夫妻店为主的情况下，大胆推行了「冰柜战略」，也就是免费给各店家提供了带有和路雪标志性logo的冰柜，并且配了一把红白相间的大阳伞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当然，和路雪也是有条件的。在它们的冰柜里，只准摆放自家产品。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/193287649962c438671acfc2.79053878.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一招，不止是牢牢把控住了冰淇淋的销售终端市场，也让各家小店在无形中成了和路雪的广告牌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>冰柜+阳伞，浓烈的红白色系，和路雪相当于打造了一个极具辨识度的夏日专属消费场景。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对我的童年来讲，看到这把大伞，就等于看到了副食店。</span></p>
<section><img src="https://ngbjimg.xy599.com/129861129562c438676caac1.43078502.png" referrerpolicy="no-referrer"></section>
<section><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;"></span></section>
<section><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">当时《北京晚报》用「满城飘满和路雪」来形容和路雪掀起的夏日风暴。</span></section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这种冰柜战略，在冷饮行业的战争中几乎成了一种常规手段，左右了许多品牌的生死。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>紧接着，和路雪在上海复制了北京的推广模式。13000台冰柜陆续出现在上海街头。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从1994年开始，和路雪以北京、上海为核心，逐渐在国内其他的一二线城市打开了市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而冰柜战略，不仅仅让国内品牌，尤其是那些主生产冰棍的小厂家毫无还手之力，就连美登高这样的大牌也被打趴下了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样让和路雪在国际市场上的老对手——雀巢开始内心躁动了起来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137286165962c43867ba7d25.92812063.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其</span><span>实，早在1995年前后，也就是和路雪已经打响了名声之后，雀巢就打算在国内布局冰淇淋业务了，并且首先瞄准的是山东市场，打算在那儿盖厂。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>得</span><span>知内部消息的和路雪，为了拦截对手，上演了一场无间道之战，拼尽全力在雀巢上市之前抢先一步打入山东，铺满了渠道，这一下把原本要大干一场的雀巢打了个措手不及。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1997年，彼时和路雪已经大杀四方，作为冰淇淋的品牌知名度占据了市场第一。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>雀巢坐不住了，除了依样画葫芦，也推行「冰柜战略」之外，还拿出了征战四方的万年「杀手锏」，收购大法。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先是在上海收购了本地知名品牌「圣麦乐」，以此占据了上海及周边区域近1/4的市场；两年后，又以97%的股权获得了广冻厂旗下著名品牌「五羊」的经营权。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/202739162c43867f09526.17460809.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和路雪也不甘落后，尤其是雀巢靠着五羊让和路雪在进军华南市场碰了一鼻子灰之后，立马收购了「蔓登琳」，进一步扩大自己的基本盘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比时机，雀巢慢了一步；比声势，和路雪那会儿还是略占上风；看着似乎雀巢被压了一头，但毕竟两家在外斗了这么多年，谁都不会轻敌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人应该也会有这么一个大概的印象：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一方面，和路雪和雀巢，作为九十年代就进入中国的洋品牌，大家自然而然会把它们和高档、昂贵联系在一起，就跟当时的麦当劳、肯德基一样；</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另一方面，光看这实打实的定价，也知道这两者走的都不是平民化的大众路线。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以和路雪为例，为了占领市场，一开始就推出了「可爱多」，「梦龙」，「可丽波」，「百乐宝」在内的20多种品牌。<strong>但在这其中最便宜的基本都要两块钱，像可爱多最早定价是在4元档。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/121669465962c4386824b2b4.29582567.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>更别提梦龙，最早定价是在7—8元。</span></strong><span>据说那会儿北京机关单位组局打麻将局，最流行的赌注就是一支梦龙。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/95938721262c438686ab4b7.31389812.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样的，雀巢冰淇淋走的也是中高端路线。但既然大家都想争取更多的市场份额，就不能老这么一直端着。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比之下，和路雪更早放下身段，并且采取了一个外资常见的做法，那就是在收购蔓登琳之后，推出了一系列两元以下的同等价位产品，并且利用蔓登琳原有的渠道将这些产品一起销售，就这么一步步吞食了蔓登琳原有的销售网络。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>等渠道成熟之后，和路雪就把「蔓登琳」这个品牌彻底废弃了。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>给各位表演一下，什么叫真正的「渣」。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而雀巢在这方面就棋差一招，拿下「圣麦乐」之后，基本放弃了对这个颇得民心的老品牌的运营，把原有的生产线全都用来生产自己的冰淇淋产品，这也为其多年之后退出华东市场埋下了伏笔。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总而言之，1999年到2003年这个阶段，和路雪和雀巢基本是杠上了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>和路雪的几大知名品牌，最多的时候降价达到了30%。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><strong>此外，在这几年里陆续推出的新产品，很多都是给经销商提供买一送一的大力度优惠。雀巢大部分产品的降价幅度也达到了近20%左右。</strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><em><em><span>02</span></em></em></span><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而在价格战上，强龙显然压不住地头蛇。</span><span>没</span><span>错，我说的就是伊利。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自打1997年以一款「苦咖啡」打开了知名度之后，伊利就看准了中低端冰淇淋市场。在几年时间里「默默发力」，凭借着价格优势，和原有的基于奶制品的渠道优势，在短短的五六年之后就做到了市场第一。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/123560936062c4386897d1a1.46457980.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>或许是伊利和越来越多国产冰淇淋的崛起，让和路雪和雀巢这俩外来品牌意识到，一味地降价是不行的。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所以，在经过了非理性竞争之后，两家先后都选择放弃价格战，转头搞起了<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E8%90%A5%E9%94%80" target="_blank">品牌营销</a>突击战。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是雀巢，在2003年就发起了猛烈攻势。相比2002年，雀巢的广告投入在整个冰淇淋市场的占比仅有3%，这个费用还比不上蒙牛的9%，而和路雪占比高达35%，伊利也占据了20%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>终于，在和和路雪价格战也打了个寂寞之后，雀巢一反常态，配合两款新品，万花筒和x脆，高调推出了两支新广告。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并且推出了全新的品牌形象代言人——蓝冰精灵。用当下的最新形容，那其实相当于一个虚拟偶像。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/30889204662c43868b6d546.09593685.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，雀巢冰淇淋的多支广告都是由乌尔善操刀。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/105359160762c43868deb431.31418432.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>乌尔善</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以在整体上保持了统一的影像风格，带着点奇幻的想象力，再加整体的蓝色调，营造出了一种梦幻感，极大地提升了雀巢冰淇淋的品牌辨识度。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137639706362c438690f3944.74731297.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而为了更具话题度，「蓝冰精灵」在后来也有了自己的「皮下」代表，比如名模春晓和演员热依扎。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>雀巢冰淇淋在品牌营销上花的心思虽然获得了一定的回报，但从大的市场环境看，<strong>雀巢的市场份额和以上提到的其他品牌都在拉大差距</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是在2011年，其关闭了上海的工厂，宣布终止包邮区的冰淇淋零售业务之后，雀巢在冰淇淋业务上基本退回了华南、华北大部。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今提起雀巢冰淇淋，或许大家印象最深的是那款借助社交媒体营销而爆火的「笨NANA冰淇淋」，像香蕉一样可以剥开吃在当时还是有点噱头的。但这几年似乎已经没什么爆款出现了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/48023984962c438693ddd16.23522057.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比之下，和路雪的营销就要早很多。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>早在1995年，和路雪主攻北京、上海等一线市场之时，就以电视广告作为最强助攻。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那时候的第一支广告，邀请的是有「广告教父」之称的莫康孙团队，花费三个月的时间制作完成。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>广告片中，重现了街边冰柜、阳伞搭配的消费场景，「和路雪，朋友一样」的歌声贯穿始终，传递给大家的是一种吃和路雪，能分享快乐的心情。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来还有一个比较经典的广告，是一头小狮子带领着小朋友们寻找宝藏。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/69471115562c4386965b008.58545052.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，整体来看，那个阶段的广告，着重突出的是和路雪这个品牌，是要让大家记住和路雪。所以，几乎每个广告的结束语都是一句带着唱腔的「和路雪～」。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顺便说一句，我对和路雪印象颇深还有一个原因，在于《我为歌狂》。</span><span>那个年代就做起了影视植入，和路雪也算是走在了品牌营销的前沿。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/59619203162c4386996fd35.56743063.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>《我为歌狂》</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在和路雪坚定了自己主攻中高端市场，尤其是2005年和路雪换帅，彻底更换运营思路，不再打无谓的价格战之后，旗下的主要产品做了更清晰的区分，走子品牌差异化路线。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其中，王牌产品可爱多，主攻年轻人市场，因而在广告上主抓情感营销。一提到可爱多，仿佛就和初恋、告白、青春、纯爱等关键词联系在一起。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有多少人还记得古早时期林俊杰和金莎出演的可爱多广告？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/191466201362c43869bc0534.61859174.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这之后，可爱多还推出了《这一刻，爱吧！》等一系列以青春爱情为主题的微电影，找的也都是陈柏霖、林依晨这样有着同样标签的演员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/163853733362c43869de2207.33828799.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主攻儿童市场的，则是由童年的百乐宝变身的「麦酷狮」。虽然还是同一只狮子，但因为结合了同名动画《麦酷狮历险记》的推广，这只小狮子在少儿市场有了更大的知名度。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/203486059962c4386a1140e5.35766504.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而梦龙，早前因为定价原因，不用宣传大家都会默认其高冷的逼格。从品牌定位来看，梦龙早期主打的显然是时尚、白领人群，因而这款产品在营销上，往往和性感、时尚、奢华等关键词联系在一起。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因而，广告片也特别舍得下本。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国外由老佛爷亲自操刀，一鸣惊人，国内也有李冰冰、林志玲、高圆圆等明星成为代言人。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到现在，梦龙的代言人虽然换了一波又一波，但整体上广告片走的还是高质感路线，这也代表了其想要传达的品牌理念。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/125411085462c4386a3e0291.19333799.jpeg" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>03</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>从九十年代到现在，和路雪在国内市场发展了近30年。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><img src="https://ngbjimg.xy599.com/26063528262c4386a609687.36131304.jpeg" referrerpolicy="no-referrer"></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据欧睿信息的数据，</span>从2016年到2021年，以零售总额来看，和路雪的市场排名始终保持第二之位。<span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在这背后，除了早期不计成本的投入，以及多年坚持的品牌营销之外，还有一个大家最容易忽略的关键要点，那就是和路雪拥有的冷链技术。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>冰淇淋的诞生离不开冷链技术的发展，尤其是含奶量高、更容易融化的中高端冰淇淋对冷链物流提出了更高的要求。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>事实上，国内冷链技术的发展，确实有和路雪的一份功劳。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>1996年，我们国家还有很多地方的冷藏运输是用大锤敲碎大冰块，然后把碎冰撒到装载着食品的车厢里，这种土方法。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而和路雪在北京、太仓等地先后建立的现代化工厂、以及在广州建立的冷库，无论是设备还是技术都是从国外进口。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/71395871362c4386abaf4e6.36578284.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再加上，从建设仓库、到组建配送中心、采购冷藏运输车、策划运输线路，以及培训员工，这一整条冷链队伍的打造，和路雪间接给大家提供了学习的窗口。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我之前在冷冻水饺那一期也提到过，当年李伟做思念水饺的冷链知识储备，就是来自于他在和路雪的工作经验。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>顺便提一句，当年雀巢落败，和它节省冷链成本也有一定关系。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原本雀巢有三四十辆自有的冷藏车队，后来为了节省冷库仓储与冷链运输成本，慢慢地被第三方占据，导致了雀巢冰淇淋在出货时必须跟着第三方装满车再走，出货速度的变慢自然会影响雀巢产品的部分竞争力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比起雀巢，和路雪显然更舍得下成本。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从自建冷链队伍，到早期将整个北方地区的物流交给华日飞天达成独家运作，在这之后，成为当时国内唯一一家采取铲板式运作的企业，即将所有产品都进行铲板式包装，以及后期不断优化运输路线，和路雪在冷链上一直走在前端。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且，也正是和路雪带来了一套冰淇淋冷链的严格执行标准。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如冰淇淋在生产出来后需要先进入-40℃的冰库，然后借由温度控制在-20℃以下的冷藏车，将冰淇淋运送到温度恒定在-26℃的批发冰库，接着再用一些温度控制在-18℃到 -20℃的冷藏车，将冰淇淋从批发商运送到零售终端。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>梦龙双标事件发生后，曾锡文曾经进行回应，就提到和路雪带来了整体冷链概念，大呼冤枉，言下之意是这么多成本我们都花了，一点辅料的成本怎么可能去省。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在我看完关于和路雪的资料后，也不禁思考起这个问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>和路雪下了成本是事实，梦龙双标亦是事实，而且此类事件屡见不鲜，原因何在呢？</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里我只说自己的结论，大家也可以进行讨论。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在九十年代初外国品牌刚进入中国的时候，不得不承认的是，他们无论在技术还是管理上，都有着巨大的优势。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对国内这块巨大的市场，只需要把他们曾经的经验搬过来，就能迅速占领市场，获得极高的利益。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，像冷链这种能保持优势，持续获利的地方，和路雪自然是舍得砸钱的。而这客观上促进了国内冰淇淋行业乃至冷冻行业的发展。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单来说，<strong>如果回报够大，那投入再多，都是值得的</strong>。比如营销也是，只要</span><span>我通过营销塑造出了高端的概念，能让消费者愿意付出高溢价，那营销费用就不是问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反过来讲，<strong>如果不影响回报，那投入再小，甚至缩减成本，也是完全ok的。</strong>比如在消费者忽略的配料表上，做一些改变。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是和路雪没有料到，在互联网时代，这种双标会被迅速传播，并最终演变成一场公关事件。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>他们并没有意识到，现在的中国消费者，和三十年前的中国消费者已经截然不同。不是糊弄就行了，而是真正的统一标准。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们需要的无非就是三件事儿，<strong>公平，公平，还是他妈的公平。</strong></span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而如果类似的品牌，还抱着三十年前的「拓荒」思维来经营，这种因双标而翻车的事件，未来还会反复上演。</span></p>
