<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们发现企业面临增长困境的原因无外乎：缺乏思维、缺少运营体系、缺少工具以及团队人才等。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">大部分问题都可以通过咨询公司、代运营公司来帮助解决，<span style="font-weight: bold;">但人才的问题会长期困扰着企业。主要体现个三个方面：</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1.私域<a href="https://www.niaogebiji.com/tag/%E4%BA%BA%E6%89%8D%E6%8B%9B%E8%81%98" target="_blank">人才招聘</a>难</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2.人才的培养和发展难</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3.跨部门协同难</strong></p>

<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，我们也经常充当“军师”的角色，帮甲方出谋划策，解决招聘的难题。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">刚好又到了金九银十的招聘季，今天就结合以往项目的经验，聊聊究竟要招什么人，才能做好私域。</p>
<h1 id="toc-1" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; margin: 25px 0px 20px; line-height: 30px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-size: 16px 20px; font-weight: 600; text-align: center; background-position: 0px 5px; background-repeat: no-repeat;"><em>01、</em>私域人才困境：私域人才为什么难找？</h1>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">如今，越来越多的行业、企业都开始招募私域人才，私域相关的岗位也明显供小于求。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">根据见实与前程无忧发布《2019-2022私域人才需求与薪酬调研报告》显示，<span style="font-weight: bold;">2021年私域岗位发布量同比增长率为41.53%，预计2022年的私域人才新增需求将高达110万。</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">以我们的一个咨询客户为例，国内某知名母婴品牌，在做消费者战略项目初期就计划招聘一位私域经理，来负责整个私域团队。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">但在实际过程中，面临了众多问题，比如：</p>

<section>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"><strong>缺乏经验，无法判断求职者水平；</strong></p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"><strong>预算不够，无法吸引优秀人才；</strong></p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;"><strong>猎头引荐的人才实力水分大；</strong></p>
<p style="margin: 0px 0px 10px;">……</p>

<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">其实也不难理解为什么企业就是找不到自己心仪的人才，结合经验，我认为主要有几方面原因：</p>
<h2 style="caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">1、招聘人员不懂业务</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">我们经常会听到很多业务老大抱怨：HR招来的人不对，简历看着漂亮，却完全不能胜任工作，反而成为累赘。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">原因就是很多HR根本不了解私域相关的业务，不理解战略到架构，无法正确判断人才到底适不适合企业，以致于出现错招的情况。</p>
<h2 style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">2、企业开出的条件毫无竞争力</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">太多企业嘴上说求贤若渴，实际上接触到优秀的人才，开出的条件着实磕碜。尤其像传统企业，总想着花小钱办大事，招人的条件还极为严苛，能招到优秀人才那才奇怪。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">据数据显示，2022年万元以上私域岗位占比达到50%，私域最高岗位月薪达到13万。只有切实拿出诚意来，才能收获回报。</p>
<h2 style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">3、地域因素</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">据调查，最关注私域的5大城市依次是广州、上海、北京、深圳、杭州。这也意味着相应的私域人才也集中在这些城市。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">所以，如果你企业所在城市不具有吸引力，那想要招聘稀缺的私域人才，尤其是管理者，则更为困难。无法选择优秀的，好比在一群“矮子”里选“将军”，只能选择合适的将就。</p>
<h2 style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">4、优秀人才更倾向选择新消费品牌</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">近两年各类新消费品牌不断冒新崛起，相比传统企业，这类品牌更追求速度，喜欢自我迭代，且对私域重视程度高。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这种环境对私域人才的吸引力无疑是巨大的，不仅有较大的发挥空间，也更容易做出相应的数据和成绩。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">虽然人难招，但并不意味着我们就要放低岗位的要求，相反要严格标准。因为私域的工作是直面消费者的，一个错误，一个失误，都有可能引发巨大的危机。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">那到底该以什么样的标准招人呢？</p>
<h1 id="toc-2" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; margin: 25px 0px 20px; line-height: 30px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-size: 16px 20px; font-weight: 600; text-align: center; background-position: 0px 5px; background-repeat: no-repeat;"><em>02、</em>岗位要求参考</h1>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">招聘第一步就是设置岗位的职责要求，作为招聘的基础门槛。下面就列举私域团队常见岗位的职责要求，以供参考。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">在业务实操中，我们总结得出，通常一个标准的私域团队包括以下岗位：<span style="font-weight: bold;">负责人1名、内容组（1-2名）、策划组（1-2名）、运营组（1-2名）、数据分析1名、选品组1人。</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 454px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/09/VRyqpmdxEkLFMU9YRie5.png" alt width="731" height="671" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">随着私域的规模的扩大，可以进行适当的补充，下面具体讲一讲每个岗位的工作和职责。</p>
<h2 style="caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">1、<a href="https://www.niaogebiji.com/tag/%E7%A7%81%E5%9F%9F%E8%BF%90%E8%90%A5" target="_blank">私域运营</a>负责人</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这是决定私域流量运营能否快速推进和成功的关键角色。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">作为负责人，他需要对私域流量有强烈的认可和信心，愿意推动和参与其中。其专业要求是需要熟悉互联网，有社交思维和用户思维。</p>

<section>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【岗位职责】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）负责项目整体的推进和团队工作的管理安排。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">2）负责协调公司内外部组织和资源，促进项目顺利开展。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">3）负责把控、评估私域运营中各工作模块的标准，比如文案内容、活动方案审核。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">4）负责项目数据的分析，依据实际情况及时调整和改进。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【考核指标】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）私域流量创造的销售数据、用户活跃度、超级用户比例。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">2）在企业启动超级用户计划之后，超级用户数量、活跃度成为考核的重要部分。</p>

<h2 style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">2、文案编辑</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这是执行岗中很重要的岗位，因为私域运营里涉及内容的地方很多，包括长内容、短内容、活动文案、产品文案等。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">文案的目的是以建立用户信任，促进成交。所以，必须找一个有文案经验，且最好懂一些营销思维的编辑。</p>
<section>
<section>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【岗位职责】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）负责微信（群）整体内容输出的运营规划、撰写和发布。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">2）负责微信中商品文案的撰写与修改。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">3）负责帮助策划人员撰写和修改活动，推广文案。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">4）负责统计朋友圈内容的点赞、评论、互动等数据。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【考核指标】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）内容平均互动次数。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">2）内容的效率（审核修改次数）。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">3）对IP的熟悉理解。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">4）引发的意向咨询量。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">5）文案引导至小程序的流量。</p>
</section>
</section>
<h2 style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">3、策划</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">策划听起来就像出主意、想点子，这或许是长久以来很多人的印象。但负责私域流量的策划组主要负责引导添加用户的策划，以及用户活跃度、销售转化的策划。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">策划不仅仅是要想出一个点子，而是要结合公司资源实际情况把点子落地，并制定方案。</p>
<section>
<section>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【岗位职责】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）负责定期策划朋友圈、微信群或节假日的营销活动，提升用户黏性，创造销售转化。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">2）负责引导用户添加微信的方案策划，落地执行。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">3）负责提交活动主题规划，制定流程，撰写活动方案，协调内容、社群运营配合，达到活动目的。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">4）负责活动实施的跟进、监督，以及数据反馈，活动结束提交活动总结复盘报告。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【考核指标】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）主要是引导加粉量、每月活动次数、参与人数以及活动产生的销量。</p>
</section>
</section>
<h2 style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">4、客户运营</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这个岗位主要做两类工作：社群的运营维护以及回答私域用户的咨询。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">我们也许会觉得这有点像客服工作，但我认为区别很大。过去客服的工作是用户问什么就答什么，但是私域客户运营的要求是销售型客服。他需要根据用户的疑问找到销售机会，把产品卖出去。</p>
<section>
<section>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【岗位职责】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）负责微信私聊用户的各类问题解答。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">2）负责社群内的运营维护。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">3）负责与用户进行朋友圈点赞、评论互动的工作。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">4）负责及时更新同步朋友圈，微信群信息（如有软件可省略）。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">5）负责针对用户不同场景设计互动、沟通标准话术。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">6）负责引导用户销售成交，提供顾问式销售。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【考核指标】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）服务时长</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">2）标签完善程度</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">3）私聊互动比率</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">4）意向和成交数量</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">5）响应速度</p>
</section>
</section>
<h2 style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">5、数据分析</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这个岗位很重要，因为它能够从已经发生的运营工作中找出问题和规律，不断优化和调整运营工作。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">数据分析人员不仅仅负责收集整理数据，重心还在于分析。不过，往往前期只能让运营负责人代理了。</p>
<section>
<section>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【岗位职责】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）对每日的数据进行汇总整理。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">2）对数据趋势进行分析总结。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">3）提供有参考性的指导建议。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【考核指标】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）主要考核提供的分析建议是否合理有效，运营团队在采取建议之后各方面的其他数据有没有改善。</p>
</section>
</section>
<h2 style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">6、选品专员</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这个岗位在平台类的企业做私域时非常重要。选品的好坏不仅影响当日销售量，还会影响用户的活跃度。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">但是对于零售类企业，产品种类不会特别多，这个岗位不会特别重要，可以由文案编辑或策划人员兼任。</p>
<section>
<section>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【岗位职责】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）选择适合私域流量的推广销售的产品。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">2）提前做好商品的规划。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">3）从销售数据中优化选品标准。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">4）策划安排商品的推广节奏。</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">【考核指标】</p>
<p style="margin: 25px 0px 20px; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit;">1）主要看订单量、客单价和销售额。</p>
</section>
</section>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">以上给到的组织架构以及岗位职责要求，主要还是以参考为主。每家企业的情况不同，需要按照各自的私域需求进行调整。</p>
<h1 id="toc-3" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; margin: 25px 0px 20px; line-height: 30px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-size: 16px 20px; font-weight: 600; text-align: center; background-position: 0px 5px; background-repeat: no-repeat;"><em>03、</em>如何评估人才核心能力？</h1>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">通过以上，我们初步厘清了私域岗位的职责，这一步基本可以筛选过滤掉大部分不符合要求的人选。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">那么我们在进行下一步沟通时，要如何评估人才真正的核心能力，又或者是在已经招聘成功的情况下，如何在试用期考核其表现呢？</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">对此，我以私域运营为例，按照职级、年限进行了划分，如下：</p>
<h2 style="caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">1、私域初级运营（1-2年）</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">对普通求职者来说，初级运营是相对容易跨入的领域，虽然起薪可能不高，但是成长空间足够巨大。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">可以从4个方面进行评估：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1）数据统计、分析能力</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">初级的运营需要接触各种各样的数据，每日进行收集和统计，需要考察重点关注哪些指标以及数据变化的应对策略。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2）内容产出能力</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">考察能否完成协同产出各类素材，例如：社群内容、朋友圈内容；在关键节点，能否提出相关的活动内容，并根据数据进行简单分析，提出改进建议。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">3）执行能力</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">考察每日的固定工作能否顺利完成，例如打标、拉群、社群sop预演和发布，朋友圈发布以及用户私信。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">4）能否熟练使用工具</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">需要熟练使用企业微信，如果公司有采购SCRM工具，则需要短时间内快速掌握。</p>
<section></section>
<h2 style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">2、私域资深运营（3-4年）</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">资深的运营已经对行业和流程比较熟悉，基本不需要做一些执行的工作，更多的是方案的设计和执行的监督。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">可以从4个方面进行评估：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1）策划能力</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">能够根据运营目标，用户特点，制定活动方案，完成选品，定价、节奏设计的活动规划。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">2）用户运营能力</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">需要对用户生命周期有深入的认知，要持续维护活跃度，基于项目运营，针对性提出方案。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">3）引流能力</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">成熟的私域运营能独立完成品牌公域流量向私域的导流，包含路径设计、利益点设计、素材准备（图文话术）、人员安排、流量承接及留存等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">4）需求分析能力</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要求能独立完成对用户的洞察。通过采集电商平台用户洞察信息、用户朋友圈采集、用户调研、竞对私域运营体验等方法同时根据用户洞察和运营目标需求，制定私域运营方案。</p>

<h2 style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; font-weight: 600; line-height: 30px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: bold;">3、私域运营主管/经理</span></h2>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这种目前在招聘市场非常罕见，是企业与猎头竞相追逐的对象，且薪资要求都不低，所以更需要仔细辨别。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">基本到这个级别，在专业能力、业务策略以及团队管理方面都有十分丰富的经验，面对面试官也都能侃侃而谈。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">背调这方面就不说了，我们可以从三个方面去评估：</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">1）方案落地能力</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">私域负责人往往需要考量其综合能力，例如一次营销活动，可能要涉及到公众号内容、企微、小程序、社区、活动策划等，负责人需要去审核和把控大方向。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">如果有板块不熟悉，那么很难全盘掌握，一个方案想要落地也必定会面临阻碍。所以在询问具体案例经验时，要注意有哪些薄弱环节，综合考量。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">2）数据分析能力</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">作为负责人，应当是对数据十分敏感的，因为每个数据指标都为下一步运营动作做出了指示。需要清楚了解每个触点涉及的关键指标，以及每个指标对应的意义是什么，便于给出指导。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">3）团队管理能力</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">团队管理是作为管理者必须要考量的能力。我们需要评估的是能否促使员工成长，因为私域运营作为新兴岗位，许多工作是无法借鉴的，所以需要管理者带领学习，带领进步。</p>                        <div style="height: 1px"></div>
