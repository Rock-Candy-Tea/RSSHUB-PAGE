<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年1月份，彭博社曾爆料阿里巴巴将出售所持有的约30%<a href="https://www.niaogebiji.com/tag/%E5%BE%AE%E5%8D%9A" target="_blank">微博</a>股权，引发资本市场震动。根据公开信息，阿里早在微博赴美上市前便参与了后者的战略融资，随后又通过增资等方式提高股权占比，前后共向微博注入近10亿美元资金。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在反垄断监管压力下，阿里收缩投资版图、和盟友进行切割的行为已屡见不鲜。但这一传闻，对于彼时股价、市值正处于低谷的微博来说还是伤害性极大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今在回过头看，如果传闻属实的话，你很难不佩服阿里的眼光——微博的表现真的很难让投资者和股东满意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二季度财报显示，微博营收、净利润双双下跌，主营广告业务尽显颓势。营收继2020年三季度之后再次录得同比下滑，疫情的影响仍在放大，净利润则连续两个季度下跌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，互联网广告市场的萎缩受到诸多客观因素影响，尤其是疫情和监管政策对新消费、游戏、教育等行业企业造成的冲击。而受影响的，也远不止微博：国内的腾讯、百度，国外的Meta、Twitter、Snap等巨头都遇到同样的难题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着短视频平台崛起并抢走大量客户资源，社交媒体平台集体入冬，转型也是势在必行。但想找到新出路，又谈何容易呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;">1</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>营收同比大跌</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>微博的广告位不好卖了</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>北京时间9月1日，微博发布2022财年二季度财报，各项数据可谓一片惨淡：营收、净利润同比大跌，主营广告业务也收入锐减。在全球互联网寒冬之下，微博的苦日子还远没有过去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先看营收和利润的整体表现。数据显示，微博二季度总营收为4.5亿美元，同比下滑21.64%；合并净利润和归母净利润则分别录得2904万美元和2830万美元，同比分别下跌64.14%和65.07%；对应的每股收益为0.15美元，同比也暴跌57.66%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>下滑，无疑是微博二季度的主旋律。不过营收和利润端面临的问题也不完全一样</span></strong></span><span>。</span><span>营收方面，本季度是继2020年三季度之后再次录得同比下滑。历史数据显示，2020年一季度至三季度微博总营收录得三连跌，同比分别下滑18.99%、10.29%和0.43%。而在两次营收下滑，都跟疫情这个大背景有很大关系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>利润端的情况相比之下就要简单很多——在成本并没有失控的情况下，利润减少主要归咎于营收下滑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据财报数据，微博二季度成本控制效果其实相当不俗：成本费用合计为3.56亿美元，同比下降7%；其中，占据大头的销售管理费用同比下滑13.15%至2.51亿美元，营业成本也仅微增2.99%至9531万美元。在和一众强调降本增效的互联网大厂横向对比中，微博表现并不逊色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过令人担忧的是，微博净利润的下滑趋势比营收要强烈得多，且延续时间也更长。数据显示，微博归母净利润已连续两个季度录得下滑，今年一季度更是同比暴跌235.54%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/21875089063163e08b992f4.19822430.png" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>其次，再来看用户数据——这可以说是微博二季度财报最大的亮点。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，</span><span>截止二季度末微博月活用户数量为5.82亿，同比净增0.16亿；日均月活用户则录得2.52亿，同比净增约700万。对于这个数据，微博CEO王高飞表示相当满意。在削减营销支出、存量竞争不断激化的背景下仍能保持正增长，可见微博的核心用户群相当稳固。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，为了拉新微博在二季度作出了不少努力，特别是贯彻实行了一年多的社区建设和对私域流量池的扶持。比如通过页面调整，将超话放到首页底部的一级入口处，引导用户进入超话社区，加强兴趣导向的社区用户连接；以及通过各种官方活动鼓励用户留言、发博，提高互动率等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过从用户黏性来看，微博横向对比微信、小红书等社交平台并没有明显优势。易观千帆统计的最新数据显示，截止今年7月份，微博人均单日使用时间为36.43分钟，低于小红书的57.24分钟，更不用说日均使用时长超过4小时的微信了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后看营收结构，这也是微博被讨论最多的老问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微博目前的营收主要来自两个板块：广告及营销服务、增值服务，前者营收占比长期接近90%，是最主要的营收支柱。但在二季度，这根支柱变得摇摇欲坠。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，微博本季度广告及营销收入为3.86亿美元，同比下跌23%，这个数字同时也是疫情爆发之后的最低纪录。剔除来自阿里巴巴的广告收入，则微博该季度广告及营销收入为3.62亿美元，同比跌幅为22%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>总的来说，微博广告业务的收入大滑坡既是情理之中，也让人大呼意外。</span></strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>意料之中的是，受疫情和政策因素影响，新消费、新能源车、游戏、教育等行业的广告客户在二季度收紧预算，收入减少在所难免。但高达23%的跌幅，还是超出了市场预期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2629154363163e08f074f7.45388781.png" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>历史数据显示，在今年二季度之前的六个季度，微博广告及营销服务收入同比增幅都保持在两位数以上，显示了较强的抗风险能力。依赖广告吃饭，是微博自诞生后延续至今的经营模式。在互联网广告市场的辉煌年代，这项业务也足够让微博赚得盆满钵满。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可如今广告营收大跌，或许意味着微博的危机比我们想象的更加严重。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;">2</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>硅谷大厂也不例外？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>全球社媒平台进入周期性衰退</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前瞻产业研究院统计的数据显示，截止2021年中国互联网广告市场规模接近5000亿元，同比增长13.85%。虽然自2017年以来市场增速就每况愈下，但整体规模依然非常庞大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，进入2022年之后，互联网广告市场的萧条之势变得愈发明显。</span><span><strong>而这一股寒气，不仅存在于国内，甚至还跨过广阔的太平洋，蔓延到了硅谷一众大厂身上。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在</span><span>国内，BAT为首的互联网企业广告收入出现全面下滑。其中，腾讯和百度的下滑趋势最为明显。二季度，腾讯广告业务营收录得186亿元，同比下降18%，是三大业务板块中跌幅最高的一块。百度这边，二季度在线广告收入为171亿元，同比下滑10%，占百度核心收入的比例也从去年同期的79%降至74%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，如果将目光聚焦到微博所在的社交媒体赛道上，国外同行们的糟糕表现更能说明问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美国知名社交平台Snapchat的母公司Snap二季度实现营收仅为11.11亿美元，净亏损同比放大178%至4.22亿美元，远低于市场预期。虽然Snap没有在最新财报中披露各业务板块的具体业绩，和其营收结构和微博十分相似，收入几乎全来自广告业务。集团总营收、利润的走势，很大程度上揭示了广告业务的低迷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>被马斯克摆了一道的Twitter，和Snap可以说是难兄难弟。二季度，Twitter营收同比下滑1%至11.8亿美元，并录得2.7亿美元的净亏损。主营广告业务收入为10.8亿美元，同比微增2%，逊于市场预期。Twitter也在财报中直言，广告行业在当前宏观环境处于“完全的逆风状态”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一个社交巨头Meta，虽然没有出现Snap、Twitter这样的大幅亏损，情况也是不容乐观。数据显示，Meta二季度广告收入281.5亿美元，为近六个季度最低水平，同比下滑1.5%，低于市场预期的285.3亿美元。受广告业务下滑影响，Meta二季度营业收入也出现历史上首次同比下滑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>翻看上述互联网巨头的财报，价值研究所认为它们的问题都是相通的：</span><span><strong>广告主流失严重，短视频平台的崛起抢走了太多资源。</strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以国内为例。QuestMobile的统计显示，截止今年一季度，短视频占据国内互联网广告市场42%的市场份额，较2020年接近翻倍，除了电商平台之外无能可出其右。这一点，从短视频平台的收入中也可以看出端倪。和营收锐减的腾讯、百度相比，快手今年二季度广告收入逆市增长10.5%，深受广告主青睐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比之下，微博在去年年底返港二次上市时，分析师就对其广告客户数量的持续减少感到担忧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据官方数据，微博广告业务广告主数量从2018年的290万一路下滑至如今的不足60万，缩水近80%。在2010年前后广告主留存率一度高达80%的Twitter，也在2020年下半年开始由盛转衰——这也和Tik Tok在海外的崛起时间线基本吻合。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>广告主转投短视频平台的理由并不难理解：</span><span><strong>用户黏性和性价比双高，也符合市场潮流。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，截止今年7月份，抖音、快手用户月活日均使用时长都在100分钟以上，是微博的两倍有余。对于大型企业来说，在疫情的大背景下需要收紧银根，资源自然要向头部平台聚拢，抖音、快手自然是成功几率更高的选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而中小客户的流失，则让微博雪上加霜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然早在2014年微博就引入大客户广告（KA），但以中小商家为主的信息流广告（SME）依然占据重要地位。从历年财报来看，来自电商、互联网金融、新消费、医疗、在线教育和游戏行业的中小企业是微博信息流广告的主力客户群。但在疫情爆发以及监管政策收紧之后，相当一部分客户彻底消失。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在2010年前后，全球社交媒体平台大爆发，从Facebook到Twitter，从微信到微博，都成为了坐拥活跃亿万用户的国民级应用。但如今十多年过去，这一批昔日的天之骄子仿佛提前步入了“中间危机”，头顶乌云密布。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在低谷中转型，是大多数企业抵御危机的第一选择。但想找到新出路，又谈何容易。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;">3</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>转型迫在眉睫</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>微博还有哪些后手？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在上</span><span>述社媒巨头中，Meta是危机意识和转型欲望最强烈的一家。从提出要全面挺近元宇宙市场，再到改名以及</span><span>多点开花的外部收购，都体现了Meta的决心。但VR/AR硬件业务发展状况并不理想，广告依然是Meta的主心骨，转型并非一朝一夕就能完成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过尚未找到转型路线的微博，似乎更令人担忧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在价值研究所看来，微博不具备Meta那样的财力和技术储备，很难打通硬件关，扎根互联网赛道探索新业务是更现实的选择。现在微博需要先认清一件事：自己最大的优势究竟在哪？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>用户，毫无疑问是微博最宝贵的财富和核心竞争力。</span></strong></span><span>虽然月活规模和用户使用时长都无法和抖音、微信相提并论，但在娱乐、时事等热点话题领域，微博依然是国内网民交流欲最强烈的平台。利用好这群核心用户，充分挖掘其附加价值，是微博转型路上的首要目标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>从当前动态来看，向小红书、B站看齐的兴趣社区，是微博发力的主方向。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今</span><span>年7月初，据媒体报道，微博正在打造一款全新的兴趣互动社区APP“星球”，涵盖本地、生活、二次元、娱乐等领域，目标用户群、内容板块都和微博一脉相承，也更方便微博为这个新产品进行导流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此之前，微博曾打造过对标小红书的种草APP绿洲和仿效抖快双雄的短视频平台星球视频，可惜都流于平淡，未能掀起多少风浪。</span><span><strong>不过这些产品都有一个共同点：社区是最大卖点。包括最新爆出的星球，微博做社区的目标和梦想从未中断。</strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以星球视频为例，这是一款立足UGC模式的视频社区APP，鼓励用户上传、分享自制视频，并效仿Soul等社交平台通过互动来增加视频的热度，互动越频繁的视频就能获得越多流量倾斜。虽然全新的星球尚未面世，但不难猜测，注重互动效果和社区氛围，依然是微博追求的效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一模式，自然是完美贴合微博互动率高、分享欲强的用户属性。然而，价值研究所认为在星球正式上线之前，我们有必要先给微博泼一盆冷水：</span><span><strong>做社区可以，但如果变现模式还是只围绕广告打转的话，很难说会不会重蹈今天的覆辙。</strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，像微信的知识星球和知乎、豆瓣等受众高度垂直的内容社区，诸如知识付费、电商以及付费会员等增值服务业务都有更大增长空间，这也是微博需要考虑的变现方向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微博主站的付费会员和电商业务之所以举步维艰，和早已根深蒂固的用户心智以及使用习惯分不开关系。娱乐化导向的内容生态，就很难像泛知识属性强烈的知乎那样推广付费内容项目。如今另起炉灶打造星球，微博自然希望能造出一个具备差异化的平台，为开拓新营收模式做准备。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但和十多年前不一样的是，移动互联大潮已经到了退潮阶段，线上流量红利所剩无几，留给微博和星球的时间、资源都不多了。能不能再次力挽狂澜，或许我们很快就能看到答案。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;">4</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>写在最后</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>转型这个关键词，微博并不陌生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年8月28日，微博成立十周年的大日子，董事长曹国伟发出全员信大谈公司的成功转型经历：</span></p>
<section><span>“微博的成功，让新浪赶上了移动互联大潮，也使新浪成为了极少数主营业务转型成功的互联网公司。”</span></section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时，微博的月活数据远高于Twitter，市值、股价处于高位，前景看起来一片大好。但如今再回头看，这更像是大厦倾塌前的落日余晖。随着互联网行业进入寒冬，移动广告市场萎缩，十年一个轮回的微博又走到了转型的十字路口。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曹国伟在微博十周年那封全员信中，还有一句令人印象深刻的话：微博攀上过高峰，也经历过低谷，遭遇过重重挑战，身处互联网行业，劲烈的竞争和快速的变化使我们不能有丝毫的懈怠。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这句话送给今天的微博，似乎也无比贴切。新的挑战已经摆在眼前，这一次，微博还能绝处逢生吗？</span></p>
