                        <p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">近日来，<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>官方向电商行业进军势头极为强劲，就在孟老师上一篇关于<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>广告类型内容发布当天，雨果网便更新消息：</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>新推出三种类型的购物广告：<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Video Shopping ads</span>（视频购物广告）、<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Catalog listing ads</span>（产品目录广告）和<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Live shopping ad</span>（直播购物广告）。这三种广告格式将在<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">For You</span>（为你推荐）页面浮现展示，并已逐步开放测试。</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">关于以上三种新广告具体效果如何，孟老师会在测试求证之后更新，但是不可否认的是，<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>对于电商付费广告的布局越发多样和精准。</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">那么，<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>广告如何创建呢？本文中孟老师将手把手教会你们。</p>
<h2 style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px 0px 0px 12px; word-break: break-word; line-height: 30px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); font-size: 17px;">一、创建广告账户</span></strong></h2>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">投放<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>广告是需要开通企业账户的。目前<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>广告投放支持三种开户方式分别是海外公司直接开户、国内公司通过代理商开户或提交注册表等待官方工作人员联系开户（时间较长）。</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">一般<strong style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">国内商家国内营业执照都是通过代理商进行开户</strong>，开户的要求比较低，首次充值金额不低于<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">500</span>或<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">1000</span>美金即可。开设<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>广告账户，平台方并不收费。当然代理商开户是需要手续费的，具体数额以代理定价为准。</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">找代理商的好处就是开通前后有任何问题都可以联系代理商进行解决。而孟老师的团队就是<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>官方的代理商，如有开户需求，可以私信咨询。</p>
<h2 style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px 0px 0px 12px; word-break: break-word; line-height: 30px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); font-size: 17px;">二、创建<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Campaign</span></span></strong></h2>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">一旦<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>广告帐户准备就绪，广告主就可以访问<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>广告账户后台，看到类似下图的<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Dashboard</span>：</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><img id="loadimg_1662894271876" class="loadimg_path" style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); transform: translateZ(0px); max-width: 100%; text-align: center; display: block; margin: 10px auto;" src="https://static.52by.com/yue/article/166139931698869879768.png" data-path referrerpolicy="no-referrer"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);"></span></p>

<h4 style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px; padding: 0px; word-break: break-word; line-height: 30px; text-align: center; background-image: none; font-size: 14px; color: #818386; font-weight: 300;">（图片来源：<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>）</h4>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">接下来，需要广告主选择投放目标，主要包括以下六大类：</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">●</span><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Awareness</span>：曝光优先的投放目标，适合品牌推广；</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">●</span><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Traffic</span>：访问流量，以引导用户访问网站为优先的投放目标；</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">●</span><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">APP Install</span>：顾名思义，以应用安装为优先的投放目标，对于电商卖家来讲，涉及较少；</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">●</span><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Video Views</span>：以视频观看为优先的投放目标，是<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>等视频类社媒所特有的模式；</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">●</span><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Conversions</span>：以转化为优先的投放目标，对于电商卖家而言，对应的基本为购买转化；</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">●</span><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Catalog Sales</span>：系列销售，和<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Pinterest</span>的<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Catalog Sales</span>相类似，适合品牌卖家使用</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><img id="loadimg_1691506122640" class="loadimg_path" style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); transform: translateZ(0px); max-width: 100%; text-align: center; display: block; margin: 10px auto;" src="https://static.52by.com/yue/article/166139933830193902790.png" data-path referrerpolicy="no-referrer"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);"></span></p>
<h4 style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px; padding: 0px; word-break: break-word; line-height: 30px; text-align: center; background-image: none; font-size: 14px; color: #818386; font-weight: 300;">（图片来源：<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>）</h4>
<h2 style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px 0px 0px 12px; word-break: break-word; line-height: 30px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); font-size: 17px;">三、创建广告组</span></strong></h2>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">设置完<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Campaign</span>之后，就可以创建广告组了。在这一步需要广告主完成展示位置、广告细节、目标受众、优化预算、日程安排、竞价方法等的设定，接下来我们一一来看：</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><strong style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">1.</span><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">展示和广告细节设定</span></strong></p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">展示位置是你广告出现的地方。<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>的<strong style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">默认选项是自动投放</strong>，这一选项下系统会根据效果自动优化展示位置，以吸引更多人。</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">当然，有经验的广告主也可以按下图选择手动投放：</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><img id="loadimg_1772956233055" class="loadimg_path" style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); transform: translateZ(0px); max-width: 100%; text-align: center; display: block; margin: 10px auto;" src="https://static.52by.com/yue/article/166139934960026188742.png" data-path referrerpolicy="no-referrer"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);"></span></p>
<h4 style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px; padding: 0px; word-break: break-word; line-height: 30px; text-align: center; background-image: none; font-size: 14px; color: #818386; font-weight: 300;">（图片来源：<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">TikTok</span>）</h4>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;">接下来，您需要在<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">“</span>广告细节<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">”</span>设定部分填写有关你正在推广的内容的详细信息，具体如下：</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">●</span><strong style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Promotion type</strong>：对于电商卖家而言，直接选择<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Website</span>即可；</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">●</span><strong style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Display name</strong>：添加品牌或应用名称以显示给受众；</p>
<p style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0); margin: 25px 0px 20px; padding: 0px; word-break: break-word; line-height: inherit; background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit;"><span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">●</span><strong style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">Profile image</strong>：上传资料图像，作为广告的一部分展示给受众（不超过<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">50KB</span>，纵横比为<span style="box-sizing: content-box; -webkit-tap-highlight-color: rgba(255, 255, 255, 0);">1:1</span>）；</p>
