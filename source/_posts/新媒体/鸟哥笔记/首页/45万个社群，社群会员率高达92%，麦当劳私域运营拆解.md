<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;">麦当劳是全球大型跨国连锁餐厅，1955年创立于美国芝加哥，在世界上大约拥有3万间分店。</p>
<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;">2017年，麦当劳推出首个品牌小程序，成为麦当劳品牌数字化的一大里程碑；2020年，麦当劳实现了品牌小程序的迭代升级，通过线上线下多渠道进行积极推广，沉淀了规模化的私域流量。</p>
<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;">2021年，麦当劳已实现“公众号+小程序+视频号+社群”的私域布局。目前，<span style="font-weight: bold;">麦当劳的小程序生态已累计了1.6亿会员，国内会员数超过2亿，社群数量达到4.5万个。</span></p>
<section style="caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;"></section>
<section style="caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;"><span style="font-weight: bold;">2）市场规模</span></section>

<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;">数据显示，2020年中国餐饮规模增长速度有所下降，但疫情对西式快餐行业的影响较小。2021年中国西式快餐市场规模为2800.7亿元，<span style="font-weight: bold;">预计2022年达3201.2亿元。</span></p>
