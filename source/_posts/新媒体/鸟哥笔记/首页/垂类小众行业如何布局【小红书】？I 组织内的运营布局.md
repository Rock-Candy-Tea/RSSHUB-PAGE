<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“小红书的商业化价值越来越高了。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“真没想到我这个小行业，在小红书居然这么好变现。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">近期，一些小众行业内的好友们跟橘先生聊了不少小红书的话题，无一不惊讶于小红书的变现能力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同于淘宝、拼多多这类电商平台，小红书的优质流量仿佛不要钱一般。销售和内容团队简单分发的内容，居然吸引了不少目标用户私信。对于高客单行业的项目，虽然是一分钱没投入的平台，却成为了他的第三大销售成单渠道。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小红书在商业化变现路上的成绩，市场及品牌是有目共睹的。高质量的海量用户、优秀的社区运营和高匹配的信息算法，成就了小红书这一平台。其实，橘先生也纳闷，为什么一些超小众的产品可以在小红书上找到一席之地？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">橘先生曾接触过两个To B的小众行业，在小红书上做得都不错。一是SaaS软件服务行业，二是石材行业（战略方向属于工业物联网）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如今的小红书仿佛已不存在以往的限制，什么“宝妈的秘密花园”，什么“女性用户为主”？流量逐渐向大众流量所转变，这个平台具备了更高的商业价值。无论公司大小，仿佛都能在小红书找到一席之地。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这篇文章中，橘先生想聊聊那些垂类小众行业的公司，应当如何布局<a href="https://www.niaogebiji.com/tag/%E5%B0%8F%E7%BA%A2%E4%B9%A6%E8%BF%90%E8%90%A5" target="_blank">小红书运营</a>。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在展开主题前，我们得先沉浸入一个经营场景：【销售可能比运营更早地入驻了小红书】。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一般来说，上述领域的企业在自身体量不大的时候，大多以销售部门为主导，运营为辅助。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">运营团队不过是第二步要搭建的事情，第一步需要考虑的是产品和销售部门能不能养活公司。销售人员会涌入各类渠道，挖掘有用的销售线索，甚至运营部门都没建立时就先行一步入驻小红书平台。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过，销售人员并非是专业的运营人员，他们可能就是从一些对外材料截图拔下来，随意整理一些文字丢到小红书上面，大部分的笔记是无用的。偶尔运气好，有一些客户就来私聊私信了。对于一些客单价高，需要销售顾问跟进的行业是可行的。 </p>
<p><img src="https://img.socialmarketings.com/article/2022/08/2022/08/1661783958908." data-ratio="0.6666666666666666" data-w="1080" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">渠道在销售的验证上，是可以变现的。但是，零零散散地来是无法成为规模的，更无法成为一家公司的核心渠道。要想让小红书成为稳定的渠道，需要对小红书整体运营板块进行规划、统筹，才能达成更好的品牌化宣传及运营效果。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以这一场景为前提，橘先生大致梳理了两类运营【小红书】平台的核心目的：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">①宣传目的：打造品牌、企业形象以及对产品能力的展示；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">②通过流量运作，实现相应的销售转化；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如何用限定的资源快速搭建、测试效果，则成为了运营布局的关键。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">橘先生将之分为四个阶段：调研阶段、整合阶段、整备阶段、正式运营阶段。咱们接着看下文，详细聊聊。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“观察你的对手，找到他们的弱点，吸收他们的优点，全面地击溃他们。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一些垂直领域的行业不像快消品这样激烈的红海行业，他们的竞争对手没有那么多。而且就像上一章节所说，他们大多不会花费过多的资源在流量平台身上，那么竞品的运营能力也是参差不齐。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，这也就方便了市场人员快速收集竞品运营信息。我们来看个表格的表头：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://img.socialmarketings.com/article/2022/08/2022/08/1661783959520.png" data-type="png" data-ratio="0.038917089678511" data-w="591" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">运营规划是有据可依的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">运营团队应当清晰了解竞品品牌在小红书的【账号矩阵布局】和【各账号运营节奏】，以及对应的广告投放（SEO搜索优化→落地页投放）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这时，相信一些市场、运营的朋友们，脑海里应该已经浮现出策略了吧？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">好，让我们把策略落地，进入第二阶段【整合阶段】。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">阶段重点：规划平台运营矩阵</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">回顾第①章节的内容，销售可能先行一步运营小红书。不过，这种运营存在不少问题，甚至用的都是自己的个人账号。如萤火般星星点点，【订单】时亮时灭，但难以形成火炬。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">①简介头像未统一；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">②发布内容杂乱；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">③封面图片视觉；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">④发布运营时间不固定，想起来才发；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，在【整合阶段】的任务要去解决企业当前在渠道上的问题。我将之分为两类：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（1）对现有账号进行调整整合，聚沙成塔，符合下一阶段的运营要求；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（2）建立新的品牌账号，输出品牌文化和产品能力，并且以便支撑平台的搜索优化及落地页投放；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">落地到【小红书运营项目】上，就是做好【小红书】的矩阵规划。结合现有资源，删减或新增账号，以便达成最佳效果。</p>
<p><img src="https://img.socialmarketings.com/article/2022/08/2022/08/1661783957939." data-ratio="0.666" data-w="500" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">橘先生习惯将小红书账号矩阵笼统分为两种类型，一种是【核心账号】，一种是【销售账号】。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://img.socialmarketings.com/article/2022/08/2022/08/1661783953738.png" data-ratio="0.5204460966542751" data-type="png" data-w="538" referrerpolicy="no-referrer"></p>

<section data-brushtype="text">
<p>核心账号：品牌展示</p>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">【核心账号】即为品牌账号，输出的是品牌概念化内容、客户场景的知识分享以及客户场景对应的软硬件产品能力曝光。削弱营销属性，更多向目标行业的客户展示企业的品牌力和服务能力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果企业有能力的话，产品和服务板块可以单独拎出来做一个号，产品能力更新/客户服务可以在这个号展示。这个账号预估不会来很多流量，它主要是为了搜索优化而准备的。</p>
<section>
<section>
<section></section>
<section data-brushtype="text">
<p>销售账号：销售追踪</p>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">【销售账号】分为两种。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">【场景账号】与【销售人员账号】</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这类型账号如果运营团队人手不够的话，可以由销售部门替代运营。运营部门提供规划及素材，销售组严格按照要求改编发布。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（如何提供素材，请查看上一篇文章《买量优化的下半场：广告计划如何提升效果？》，里头有搭建【视频媒体库】【素材库】【图文媒体库】【衍生媒体库】的教程）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">接下来，我们先说说【场景账号】</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小众行业的客户场景是有限的，比如一些SaaS系统提供企业“目标管理+绩效管理”的解决方案，有些针对线下门店的24H无人化经营提供软硬件解决方案。亦或者像石材行业的工业物联网，针对荒料供应商、大板制造商提供对应硬件产品及产品数据库，实现智能切割机提供石料大板的数据扫描和数据库上传。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">【场景账号】应当针对于目标客户的需求场景而设置的。产品的目标客户群体所处于哪些行业，你就分类建立多少个场景账号（在自己能把握的范围内）。输出的内容分为两类：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（1）产品类</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">①对应客户需求场景与产品力的展示介绍；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">②硬件产品的功能介绍、组装、操作类内容；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">（2）内容运营类</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">①对应行业的干货内容，可以关乎运营干货、行业介绍、初创（业）辅导；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">②现有用户的交流与访谈等；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再聊聊【销售人员账号】</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这类账号由销售人员负责，内容可以与产品使用场景的干活分享、产品功能展示相关。尽量调整文案和图片，不要涉及抄袭，【小红书】平台会判断内容在平台的重复度。平台会处罚那些抄袭账号并且关进【小黑屋】里。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过，销售人员怎么才能做出有用的好内容呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">他们又不是干运营的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就涉及【媒体内容库】是否完成建设。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">完成【媒体内容库】的建设并且储备一定量的素材后，销售人员需要图文内容资料，可以立即从【图文媒体库】调取改编；视频资料可以从【视频媒体库】调取改编。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这同时也便于运营部门对渠道账号内容的统筹管理。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果没有【媒体内容库】，账号内容都靠运营和支撑方“临时手搓”，别提有多耗费时间了。【媒体内容库】实现了组织人效和员工价值的复合提升。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于【筹备】环节自不必说，团队应当根据【整合阶段】的规划，对已有销售账号进行调整、新建【品牌号】。运营部门要提前做好小红书开户，为信息流广告投放做准备。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有投放需求的公司，建议开户后就开始测试小红书广告的投放效果，紧密衔接运营。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">并且，【内容储备】也是该阶段极其重要的环节，内容分为两块：【广告素材】【笔记内容】</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">广告素材的筹备主要是：①落地页素材；② 信息流投流词包；③创意标题组；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于小红书的内容特征，不同于其他平台，运营部门需要单独为其建立【小红书资源库】。为其他运营者提供内容参考，使其能快速改编，提升了跨部门协作的整体效率。</p>
