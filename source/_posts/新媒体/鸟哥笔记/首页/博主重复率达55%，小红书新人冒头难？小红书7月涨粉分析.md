<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E5%B0%8F%E7%BA%A2%E4%B9%A6" target="_blank">小红书</a>7月涨粉榜已出，我们先一起看榜：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/341532962fa70323c7ca0.88634080.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>整体而言，相较6月，<strong>小红书7月的内容生态更有生机，TOP20的涨粉量级都在20W以上。</strong>其中明星@赵露思 因古装剧《星汉灿烂》的热播，迅速扩大声量，小红书账号新增粉丝97.06W，问鼎7月涨粉榜榜首。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，除平台官方号后，搞笑博主@Thurman猫一杯 以44.83W的涨幅，再次成为小红书博主中的月涨粉冠军，剧情达人@丁郑美子 与搞笑科普博主@俏佳人xxx 则是以近40W的成绩分列榜单第二三名。剩余账号中，有10 个账号的增粉量在30W-40W之间，7个账号的增粉量在20W-30W之间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从上榜账号类型来看，<strong>共覆盖了剧情、美食、知识、萌娃、美妆、探店、变装、明星等7个品类，</strong>其中剧情/搞笑赛道共有包括@Thurman猫一杯、@丁郑美子、@欧文浩@潘姥姥@奥黛丽厚本@原来是舅先生@郑丽芬er在内的7个账号上榜，延续了近期剧情博主在小红书内容生态中的强劲势头。相比之下，其余品类则分布得较为平均。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得关注的是，<strong>与6月相比，7月涨粉榜中的博主重复率达到55%（11名博主同时登上6月、7月涨粉榜），从中一方面可以看出<strong>“老”博主的生命周期在小红书上相对较长，</strong>另一方面也意味着<strong>“新”博主冒头的概率相对较低。</strong></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，我们将对其中部分账号，进行一些更加细致的分析。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>@俏佳人xxx </strong><strong>“律政佳人”一边科普，一边搞笑？</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>@俏佳人xxx可谓是本期涨粉榜中的最大黑马。主页的个人介绍中，博主对自己的个人评价是：<strong>本来搞普法的，后来变成了搞笑的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前在该账号中可以看到的第一支作品，是博主于今年3月14日发布的一则讲述自己面试经历的视频。视频中律所HR的</span><span><em>“你的学校是什么技校？”“我们只招一本。”“你期待的薪资是怎样的？我们只能给到2800。”</em></span><span>灵魂三问，让广大打工人心中为之一震。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而博主本人也“不甘示弱”，在视频中狠怼了一番</span><em><span>“用你的十二指肠想一想，技校哪有法学专业？”“我祝你全家健康。”“你赚了一百万是要跟我分吗？”</span></em><span>几番硬核回怼让人直呼过瘾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>特别是在视频最后，博主说到自己在回家路上越想越不爽，就在脑子里面跟HR吵架，各种反驳，辩得对方哑口无言。结尾一句“嘿嘿，很舒服”，让不少网友也随之感叹：我也脑补了，很舒服。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/134463764362fa7032876db6.75028721.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@俏佳人xxx 小红书账号截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从众多粉丝的评论中，不难看出这位博主在搞笑上的天赋。@俏佳人xxx在账号后续发布的作品上，也愈发注重起了自己内容的娱乐性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大学毕业后，他的作品主题从生活日常变为了工作日常。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>《离谱的法律咨询》系列，搜集网友们无厘头的“法律问题”，配上近期大热的大眼蛙滤镜，</span></strong><span>有一种别样的喜感，也引发了粉丝与其互动的高潮，大家纷纷在评论区留下了更多的“奇葩思考”；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>《离谱的律所实习》系列，则是从“职场新人”的角度出发，和大家分享自己在工作中遇到的种种无语事件，</span></strong><span>但是因为博主本人略显成熟的长相，搭配整齐的西装三件套，让职场新人这一身份有种奇妙的反转。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/214574714962fa7032c161f7.11497331.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@俏佳人xxx 小红书账号截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>账号最近大火的一条作品是《第一次坐牢是什么体验》，主要讲述了朋友在新疆坐牢的经历。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>故事中，监狱大哥面对博主朋友“我是第一次坐牢，不太熟练”的疑问，回答到：</span><span><em>“你连坐牢都不会啊？看你的面相我以为你是一个好苗子，不过你只要脚踏实地，跟着我混，一步一个脚印，一定会成功的。”</em></span><span>要不是场景不对，活生生一幕职场大佬调教新人的场景，让人啼笑皆非，这种奇怪的故事氛围，也帮助该支视频获得了20.4W的点赞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>因为这种忽正经忽搞笑的视频风格，与常年和“法外狂徒”斗争的罗翔老师有异曲同工之妙，有网友建议他改名“啰翔说邢法”，加入ESO组合。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总地看来，@俏佳人xxx 的走红，主要是因为其<strong>一本正经的搞笑风格</strong>。视频中的主角，总是穿着整齐的西装，用最正经的表情，操着一口广谱讲法律搞笑段子，重重反转下收获了更多网友的喜爱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，@俏佳人xxx的视频，或许也能给想要入局短视频的普通人带去一些经验。可以看到，博主的视频内容均是来源于自己的生活与工作，但却在短短一月内收获了40W粉丝的喜爱。善于发现生活中的趣味日常、并学会对其进行视频化的包装与表达，就有可能在短视频领域中占据自己的一席之地。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>@正义的宝妹 7分钟妆容全拆解</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>小红书美妆还能卷出什么花样？</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说@俏佳人xxx 是本次涨粉周期中，飞速成长的代表人物，那么@正义的宝妹 则向大家展示了什么叫厚积薄发。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该播主进入小红书的时间很早，就目前能浏览到的内容来看，早在2018年就发布过一篇海淘相关的笔记。前期，她的账号内容比较丰富，涉及购物分享、旅游吃喝、穿搭妆容等等多个赛道。这</span><span>也导致了账号早期的定位并不精准，各个笔记之间的热度差异很大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>能够在一个月内涨粉30W的关键性因素，可能在于她于7月8日开始的系列作品《刷到这条视频……你离学会xx就不远了！！》，<strong>截至目前，这四条题材类似的作品，点赞从2.5W到29.1W不等，远超账号平均水平。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/41815305662fa703346aa15.49314576.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@正义的宝妹 小红书账号截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以该系列点赞数最高的底妆篇为例，我们一起来看看这篇美妆笔记有哪些过人之处：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从笔记选题来看，相较其他，底妆是每个人化妆时都无法省略的步骤，这就扩大了内容的受众群；“学会”二字，又将目标群体直指刚开始化妆的新手小白，可操作性显得更强。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从表现形式来看，一个很大特点是<strong>，封面采用了”之前&之后“的对比封面，</strong>不仅能起到给用户带去视觉冲击，肯定化妆的重要性，认可博主化妆技术三个作用，还能够让用户一秒自查，判断自己的化妆手法是否会造成相似问题，了解这条视频是否对自己“有用”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，整支视频划分了8个模块，从护肤、防晒、妆前乳、遮瑕、粉底、提亮、定妆7个步骤一一讲解，干货满满，<strong>每一Part都有明显的标注，用户可以按个人需求挑选观看，弥补了这类视频时长过长的问题。</strong>因此，即使这支视频长达7分20秒，依然取得了很亮眼的成绩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从博主话术来开，开头一句</span><em><span>“这是不是你画的妆，什么都画了，显老显脏不好看”</span></em><span>直击用户痛点；结尾又表达了自己对于化妆的看法，提醒大家不要过度”容貌焦虑“，正能量十足。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/30520195862fa703398f166.25635809.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@正义的宝妹 小红书账号截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以看到，<strong>整支视频的讲述节奏，遵循了痛点吸引、干货讲解、主旨升华三个步骤，让观众在学到技能知识的同时，也舒缓情绪放下焦虑。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，除了这条视频外，@正义的宝妹 对自己帐号的经营还有很多可取之处。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，在账号首页及多支视频的讲解中，都<strong>明确指出自己”化妆学校毕业“的专业身份，增加用户信任感，提升个人能力权威性；</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，<strong>采用原相机自然光拍摄，化妆流程详细真实，可还原度高</strong>，并且，这一特性，还很好解决了早就"苦滤镜过美"的小红薯们。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，美妆赛道作为小红书社区的热门赛道，在内容生态中始终扮演着举足轻重的角色，早已是不可舍弃的“社区基石”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但同样的，众所周知，美妆赛道一直处于竞争激烈的红海领域，想要从中脱颖而出，创作者们也是各出奇招，卷无止境。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>从早年的逆袭剧本、口播讲解、种草测评，到现在的强反差对比、情景式美妆，大浪淘沙后，美妆赛道势必朝着更加真实、专业的方向发展。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卡思在观察了近期站内的多篇热门美妆笔记后，总结了以下几个美妆内容趋势：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一、产品“成分党”盛行。</span></strong><span>早C晚A、玻色因、神经酰胺、烟酰胺等等关键词，用户搜索热情只增不减。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二、强调化妆前后的反差对比，手把手教学“一键全妆”。</span></strong><span>很多笔记都做到了在一篇内容中“海纳百川”，把化妆过程中的每个细节，<span>从化妆工具、化妆产品到化妆手法、化妆步骤，</span>向粉丝一一展现，帮助大家“一键全妆”。</span><span></span></p>
