<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管大家有没有做过直播，都可以在今天的分享里有所收获。我们在一线做了很多直播间，所以想跟小伙伴们分享一下现在做抖音直播是什么情况，直播间为什么没有做起来，以及在启动时会遇到什么困难和问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做直播，最重要的是吸取多方面的资讯和信息。今天我就自己以及整个公司，在直播电商行业看到的、观察到的、思考到的东西，跟大家做一个分享，包含直播电商行业最新玩法的总结和下半年方向预测。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>抖音直播现状解析</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在已经是8月份了，距离2022年结束还有几个月。从这个时间节点来看，抖音直播是什么样的状况？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先，付费玩家竞争激烈，品牌商家愈加艰难。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多品牌找过来，询问在抖音直播投多少钱，能做出什么样的结果。现在这个阶段，在抖音做流量采买，引流到直播间，成本非常高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其次，内容流量杠杆效应明显，优质内容弯道超车。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年5月底，抖音发布了《抖音电商Fact+白皮书》，建议大家好好研读，因为官方的指挥棒就是流量分配的法则。以前有一段时间，大家特别看不上粉丝多、变现能力差的账号，认为没有卖货潜力。但是最近一两个月的时间，有优质内容的账号，直播间人数都上升了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三点，站外流量和搜索激增，新一轮洗牌正在进行中。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家在直播的时候会发现，电商罗盘大屏里面，“其他流量”占比很大。可能之前“其他流量”占到5%，现在“其他流量”占到30%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“其他流量”是指抖音把直播间推到站外所产生的站外流量。抖音需要把直播间作为一个内容入口，推到站外，比如字节内的App、字节外的App，然后去用直播间给用户洗标签。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，抖音商城和搜索流量是抖音下半年重点运营的流量。品牌也好，商家也好，小团队也好，都在面临一个洗牌的过程。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>付费流量打法是什么？如何快速起号？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，我跟大家拆解一下，目前抖音直播的一些主流玩法。首先是付费流量打法，分为店播玩法和达播玩法两类。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1. 付费流量的两种打法：店播玩法和达播玩法</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一种店播玩法，原理是通过工业化内容生产和稳定的开播时长，制造稳定的流量。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直投巨量千川的竞价广告，价格波动以及流量波动都比较大。工业化内容生产，可以帮助直播间流量变得稳定，ROI变得稳定。此外，稳定的开播时长，可以帮助品牌吃到全时态的流量红利，比如从每天早上8:00开播，一直播到晚上12:00，打造日不落直播间，但是前提还是你需要有稳定的流量，不然日不落对你来说就是内耗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>卖货本身卖的是货，店播玩法要求选择高毛利、高性价比、展示性强的爆品。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，产品要高毛利，零售价和成本之间的空间非常大，才可以放心去做付费投放。其次，产品要高性价比，比如说49.9元两支，199块钱发三盒的产品，用户下单容易，转化率也高。当你的产品转化率高的时候，付费投放的流量会非常丝滑。再者，产品需要是展示性强的爆品，可以通过内容更好展现它的卖点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>下面这个直播间，主要是做护肤产品。今年上半年，销量最高的时候，一个月卖了一个亿。近几个月业绩掉的比较厉害，但日销还是可以做到100万以上。业绩下降主要是因为抖音对流量分配机制做了调整，包括新的口碑分规则上线，原本的跑量效率下降。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/160408838862fb34f66648d0.95467159.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一个直播间，主要是做彩妆，一个月交易额4000多万，日销100多万。这两个账号每天直播时长都在19到20个小时左右，流量大部分都是短视频投放巨量千川，然后引流到直播间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么同样的产品，有的直播间可以卖4000万，有的40万都不到？区别就在于有无稳定的工业化生产内容能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/190004877862fb34f6f1a437.98466592.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我中间截的这张图，是直播账号的短视频内容。短视频内容有剧情、有段子、有口播、有使用体验、有达人、有明星，用大量的素材去做流量投放，你才可以获得这样的业绩回报。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前抖音这种店播玩法，主要面向人民币玩家，产品也需要高性价比、转化率高的。如果ROI可以做到1.2以上，它都是盈利的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二种达播玩法，原理是先卖人设，再卖货，用付费流量撬动自然流量。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般达播玩法，会给自己打造了老板娘、品牌创始人、多年创业者等等的人设。账号本身是有故事的，告诉用户我是谁，为什么这个产品在我的直播间便宜。说白了，就是先宣传自己再去卖货，至于你卖的货是什么已经不重要了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>达人一般只能播4到6个小时，做到单场百万的业绩肯定要靠投放。但达人比普通的店播主播有更强的吸引力，可以用付费流量撬动自然流量。广告投放的ROI可能只有2，全场ROI有5至6，卖白牌美妆护肤的直播间做到这个ROI还是很赚钱了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用大量的短视频去投千川的模式，和用付费投放去撬动自然流量的模式，是完全两种不同的打法，但核心都是花钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>达播玩法要求选择人设型主播，有精细化流程和深SKU。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先这类达人是万里挑一的，需要挖掘。我认识一个老板，常年在外面找这样的达人去签约和合作，一年满足条件的不到10个，成功的不超过3个。其次这类玩法需要精细化的流程，需要策划引流的产品，利润品，用什么产品当钩子把用户留到直播间。再者，需要深SKU。不可能直播间只卖1-2款产品，起码要卖10款产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我认为店播玩法和达播玩法，没有高低之分，核心还是要找到适合自己的。有人会评判什么样的流量比较健康，在我看来没有健不健康之说，能赚钱有利润才是直播间的核心指标。如果它适合你，那么就应该去放大。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2. 付费流量打法如何快速起号？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过付费流量打法，品牌可以在比较短的时间内，用金钱换时间，快速起号。需要注意的是，实操上有三个要点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先，让高中生打败小学生——用直播开荒队抢占“新手村”的流量红利。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你想通过付费投放去做一个账号的冷启动，起号团队一定要非常优质。很多的主播来我们公司面试，会问播一个新号还是一个成熟直播间。我们都会告诉主播是不适合播新号的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新账号有流量红利。在流量池的赛马竞争机制里面，如果你自己是个高中生，伪装成小学生过来去打败别人，是很有优势的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其次，先做流量宽度，再做流量精准度——投放别看ROI，卖货守着盈亏线。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单来说，就是先去破流量池，再去让你的账号标签更精准。很多人会问我，起号阶段用一些引流产品，会不会导致我的人群标签不准。我的回答都是，先让流量热起来，再去考虑流量精准度的问题。做抖音最大的问题不是实践能力，而是大家的认知能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>再者，投放的核心在于素材和转化——做好内容（二者选其一）和话术承接。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多的时候，直播间投放了也没有流量，核心原因是前期花钱买进来的流量没有转化。我们要做好内容，短视频内容和直播间内容二者选其一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>下面通过这个案例来分享，我们如何通过一场付费直播去撬动免费流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/135357632262fb34f77480f5.62906918.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音电商罗盘后台有分钟级的流量图。在这张图里面，蓝色的波形图是付费流量，黄色的波形图是免费流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在晚上7点至12点这五个小时，我们是用付费流量撬动了免费流量。直播间一开始是靠付费投放拉起来的，后续通过一系列的运营动作，自然流量被进一步拉起来，再通过一系列的动作，自然流量就会源源不断的进入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在运营动作的基础上，回答很多人关心的问题，投付费流量会不会压制免费流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我告诉大家不会。直播间是一个整体。好的付费投放，在达人直播间里面是杠杆。抖音官方说的“营销场”，就是提倡付费流量和免费流量的协同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>然后再给大家贡献一个付费起号的思路。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个账号，是一个已经废掉半年没有播的老号，处在一个低谷，开播可能只有一两个人在线。当天使用店播玩法，投放耗费了2万块钱，GMV达到了12万， ROI是6。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>投放的2万元里，8000元用于千川专业和莱卡定向，投浅层计划，RO1是2。2000元用于随心推，ROI是6。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>浅层计划指观看进入直播间的功能，它很便宜，广告ROI在2左右。随心推的工具，我建议新手小白做账号冷启动，以及账号补量时使用。很多做付费投放的品牌，他们都不用这个工具，觉得这个工具太小白了。但小白本身就是为了降低投放门槛，并不不代表它的流量不精准，也不代表工具不好。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>自然流量打法是什么？如何快速起号？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚才讲的是付费流量打法，再来讲一下自然流量打法。自然流量就是不花钱，完完全全靠系统的直播推荐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1. 自然流量的两种打法：达播玩法和场景玩法</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一种是达播玩法，核心原理是把亏货当作投放方式，比如花一块钱送大肚杯这样的直播间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多的账号开播就送5000单一块钱的东西，每单亏个6块钱7块钱，然后他开场就要亏个五六万块钱。去年这些做自然流量的直播间，亏货的时候都在控单、憋单，让用户在直播间互动停留。但现在自然流的直播间，很多都不限单了，一块钱的东西随便拍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>竞争的激烈程度已经不一样了。付费流量打法需要采卖流量，而亏货是通过低价产品让流量进到直播间，核心就是为了直播间的数据，比如停留时长、粉丝团数据、商购物车的点击率等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这种玩法，要求直播间里要有燃烧型主播、高穿透场景、高转化产品和直播剧情。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃烧型主播这词，是我去年提出来的。当时有一个主播，纯靠自然流量，用任何一个账号首次开播都能卖100万。我在现场看了他的直播，非常冷的冬天，在北方一个小城市里面。他播了两个小时，大汗淋漓，我感觉他整个人在燃烧。他整个的状态和激情，会让他的直播间的曝光点击率做到40%。而普通的品牌直播间，曝光点击率可能只有5%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曝光点击率直接影响了场观，最后影响了业绩。在抖音给到的曝光机会面前，不同的点击率意味着完全不同的直播间场观量级。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只有燃烧型的主播才可以做到这么高的穿透率，再加上高穿透的场景，高转化的产品，以及直播剧情，直播间的效果都会更好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>剧情演绎在抖音上面，需要注意尺度。很多做自然流量的直播间掌握了边界感，一方面有剧情和内容能留住用户，一方面又不违规。不管你怎么去举报直播间，都没有办法举报掉。不是因为有什么后台，也不是因为开了白名单，是内容方面的技巧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>自然流量打法下面还有一种，场景玩法。原理是好场景带来高曝光点击率，再带来源源不断的自然流量。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当你没有一个万里挑一、能控场、转化率极强的燃烧型主播，你是不是还可以做直播？当然是可以的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>下面这些截图是我昨天晚上9点钟左右的时候，在带货排行榜名单里面截图的一些直播间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/74789162562fb34f7cecf25.91729821.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这类账号没有很强的特点，没有品牌，没有IP，没有达人。例如这个叫“饮料批发”的账号，直播间在线2000人，单场卖了40多万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它有一个非常好的场景，可以让曝光点击率高。用户点进来才成为你的场观，算法开始给你推送更多的流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是场景做的一样，你未必能获得得同样的流量。场景玩法的要求是，原生场景、高转化内容和精细化运营。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我白天搜到过一个账号，场景和“饮料批发”几乎一模一样的，但直播间只有6个人。<strong>再试图复制一个直播间的时候，就会发现结果是无法复制的。除了场景本身之外，还有很多细节是非常重要的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先得是第一原生场景，其次直播间的运营也要厉害，购物车里面的产品需要精心设计和包装。购物车里面，很多一块钱的东西，在购买的时候要付邮费，如果买满一定金额还能买一发三。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好场景加上内容，再加上解说，再加上精细化的运营，最后才构成了一个完整的直播间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外直播间还要有好的货源。季节性的产品，还有一些高性价比的产品，在抖音上面通过内容展现，会有更好的爆发机会。有些品牌做不起，选品有很大的问题，硬卖的话，ROI可能会很低。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2. 自然流量打法如何快速起号？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自然流量打法想要起号，燃烧型的主播非常重要。如果主播表现力不行，你就要给到主播一套非常标准性的话术。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户进到直播间，一定要给到非常有用的信息。有些直播间，进去会听到“欢迎来到直播间，点点关注点点赞”的话术。用户进到你的直播间5秒钟都停留不了，花7秒钟说这样的废话是不明智的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的话术，需要让用户觉得主播在搞大事。转化率高的直播间，主播说的每一句话，包括说话中间的停顿，都是设计过的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>极致的流程也很重要。想要用户在直播间停留，就要有一个钩子，不停去钩住用户的好奇心。卖很好的自然流量直播间，评论区网友会留言，“我本来只是进来薅个羊毛，但是竟然一直买到了最后”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人担心，低价产品引流来的都是羊毛，“会不会用户买完东西就走了？”在问这个问题，说明你的直播间除了低价没有其他价值。在抖音，大部分直播间转化的都是羊毛，羊毛党本身就有很大的价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播间本身要做数据，赛马跑赢同行。与直播间热度相关的数据，比如加粉丝团、互动评论、停留时长等。与电商权重相关的数据，比如商点、直播间下单、直播间成交等。<strong>这些数据，到底哪个最重要？官方也提供不了答案</strong>，抖音算法是一个黑盒，我们是没有办法拆解它。我们能做的就是我们在流量运营的过程当中，根据流量变化调整运营动作，俗称就是手感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这里再给大家一个冷启动方法论，我认为，所有的账号卖所有的产品，从月销0到月销百万，靠自然流量就可以。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人说不可能，不投钱直播间就没有流量。但其实每个类目里面，我们都可以找到这样一个直播间。从月销百万到月销千万，通过短视频加自然流量就不花钱也是可以做到的。等直播间到了月销千万以上，则建议使用付费流量，帮助撬动更大的生意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前面提到去全国各地签约主播的老板，他有一个考核的标准，就是能否在不投钱的情况下，把直播间拉到1000人在线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多账号，不要着急投钱，先把基本功打好。如果人货场做的不好，投钱对流量撬动的作用不大。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/58143460660235253dbd083.00129585.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>内容流量打法</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚才说了付费流量打法和自然流量打法，接下来说内容流量打法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种打法的原理，是抖音电商KPI下的内容流量杠杆。抖音电商下半年要还有很多的KPI需要完成。想要完成这个KPI，就需要为很多有内容浏览行为，但是没有电商行为的用户打上电商的标签。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>内容型达人的春天来了，很多人纠结做的内容跟卖的东西不匹配。以前我可能会说不匹配肯定不行，但现在如果做内容能搞到流量，就不要管太多，因为你会发现内容型达人的流量现在会溢出，只要你选好品，一般卖货都不会太差。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再给大家一个建议，选货不要根据产品，而是要根据用户圈层。你的内容吸引的是什么样的人，比如你的段子吸引的是家庭主妇，就可以卖母婴的产品，还可以卖护肤品、个护、衣服等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种内容流量打法，具体有两条路径。第一条是短视频带货。通过单个短视频，一个月卖300多万的都有，很多人通过这样的方式低调赚钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一条路径是短视频导流直播间。下图这个短视频导流直播间的排行榜显示，单条短视频可以给直播间带来100多万的场观，这用付费流量去计算是很贵的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/127051882362fb34f871d3b0.13555518.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频导流直播间是真正稀缺的流量技能，它不仅稀缺在内容能在付费投放里面跑量，还稀缺能往直播间带免费流量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8187367216030c0c85d55e0.47560833.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>下半年抖音直播风向预估</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>下半年的抖音直播有一个风向预估。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一点，新的流量推荐机制下，洗牌成为常态，求变者将成黑马。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经常会有人说老师我要赶紧做抖音，我再不做抖音就来不及了。<strong>其实我想跟大家讲的就是，做抖音不在于早。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早期去做抖音的老板，很多在退场，因为没有掌握核心方法，也没有获得收获。抖音的排行榜里面，迭代了20%至30%的直播间，而且这20%至30%可能是刚刚开始做抖音。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么时候做抖音都不晚。没有人永远是韭菜，但永远有人是韭菜。韭菜不是贬义词，只是目标用户的一个简称。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二点，得内容者得流量，短视频内容和直播间内容必有其一。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从现在开始到未来一两年内，内容流量会带来极大的杠杆。如果你能把抖音上面只看内容不买东西的用户转化成电商用户，并且通过抖音的算法给它打上电商标签，那么你的直播间一定会成为流量追捧的对象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以我们做抖音，一定要有短视频的内容，或者是直播间的内容，当然两者都有更好的。直播间的内容就是剧情带货，不是说非得演戏，而是设计合适的剧情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三点，科学营销领跑，A2和A3蓄水养鱼，再自播收割已是趋势。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个是对品牌来讲的，通过自播间去收割A2和A3的人群包，然后去进行复投，已经成为了一种科学营销的方法。如果你没有做品牌，就可以先不用考虑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第四点，全域营销场景覆盖，搜索和商城成为下半年重点增长流量入口。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>搜索已经是抖音非常重要的一个流量来源。很多人已经习惯通过抖音搜索短视频的内容，来获得商品信息。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果品牌全部的平台都有，抖音其实是流量获取的一级入口。这些流量，可以溢出到其他平台，包括天猫、京东、拼多多等等，而从这些平台反流到抖音很难。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/26210138662fb34f8c46a54.72713611.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>未来直播电商，我认为有两个机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先，是低成本复刻直播间的机会。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们公司在上海，疫情期间，我把部分直播间带到了江苏徐州。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之前没有把直播间放到徐州，是因为我认为人才不够，满足不了品牌服务的要求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当我来到徐州之后，发现并不是这样子，单个直播间的成本在上海是6-7万左右，徐州只要2万，这样就有了低线城市的低成本工业化直播间复制机会。在这里做一个头部直播间很难，做5个月销百万的直播间比较容易。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>还有一个机会，是我比较看好的视频号机会。</span></strong><span>它包含了付费投放的机会、免费流量的机会、高客单的机会和直播电商增量四个机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>付费投放的机会，是因为视频号刚开放了商业化采买的信息流投放工具。在此之前都是野生团队入场，也都是自然流的打法，但接下来品牌入场加专业化的工业化的付费流量采买。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/186582877062fb34f9363108.05355388.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>免费流量机会，是因为视频号月活很高。高客单的机会，是因为视频号上面已经可以卖超过1000元客单价的产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>下面这张图是做美妆护肤的朋友，在视频号单场直播没有投钱的情况下，做到了80万的成交金额，这是直播电商增量的机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/214522319762fb34f999bff9.47485963.png" referrerpolicy="no-referrer"></p>
