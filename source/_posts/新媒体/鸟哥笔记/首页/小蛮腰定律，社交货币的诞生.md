<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>作者：陈格雷（老小格）团队</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><span>今天这篇谈谈<a href="https://www.niaogebiji.com/tag/%E7%A4%BE%E4%BA%A4%E8%B4%A7%E5%B8%81" target="_blank">社交货币</a>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>社交货币指人们乐于分享传播某种东西。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>货币能够被用来交换，前提是大家都认可价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>社交货币也一样，最关键的是共识。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>共识是社交货币成立的第一要素，必须传播的的双方都认可同一种说法、同一种符号、同一个<a href="https://www.niaogebiji.com/tag/IP" target="_blank">IP</a>，等等。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果共识的人非常多，那这个社交货币就能传播得非常广，当对一个东西的共识超过30%的人，就能成为超级社交货币。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，一定要记住的是——</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>社交货币不等于是大家都知道的东西。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span><strong><span>常规的、大家都知道的东西，往往大家不主动传播。</span></strong></span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>只有人们愿意传播的、有趣的共识，才是社交货币。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，广州人民都知道，那座全市最高的建筑物，名叫广州塔。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/6425485296306c051947ca3.69821824.jpeg" width="333" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>正式名称：广州塔</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但实际上，很少有人在传播时，用“广州塔”这个词，除非在非常正式的场合。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在绝大多数的传播场景里，大家都喜欢说成：小蛮腰。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/6425485296306c051947ca3.69821824.jpeg" width="333" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>大家的爱称：小蛮腰<br></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，从品牌传播的角度看——</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>广州塔是它的正规名字，</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>「小蛮腰」才是它的社交货币。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老小格发现，小蛮腰能成为社交货币，其内在原理，适用于绝大多数社交货币的诞生原因，尤其在那些正规的、正经的事物，如何成为流行传播并出圈的现象上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我将这一现象称为：小蛮腰定律。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以下，逐一说来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/18066673516306c0520f3213.26085657.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>多年以前，当广州塔建成之际，让很多人都为之惊叹。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它独特的曲线和结构，让人印象深刻，过目不忘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管是日景还是夜景，都成为了中国以至全世界的瑰丽风景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/6753747696306c05260a063.49991604.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我喜欢称它为“琉璃盏”，因为它亮灯后的样子，很像一盏琉璃做成的瓶子。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，琉璃盏这个说法并没有传播开来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>渐渐地，“小蛮腰”这个名字传播开来，被越来越多的人所接受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管在一开始，当局似乎并不太接受这个听起来不那么高雅的代称。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但人们还是对小蛮腰这个说法喜闻乐见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原因何在？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>奥秘就在“小蛮腰”这个词上。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小蛮腰，听起来非常亲民、接地气，不高高在上，而且非常形象、甚至还带一点情趣挑逗。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不像广州塔那么一本正经和严肃。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也不像我喜欢的“琉璃盏”那么高雅和不食人间烟火。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且，它拟人化了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小蛮腰，将一座钢筋水泥的、冷冰冰墩的庞然大物，戏剧化为一个扭动腰躯的漫妙舞者，瞬间就生动了起来，变成了仿佛有生命的青春少女。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这当然非常容易被人们感知和接受。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>当人们拍下它的玉照时，从内心里，一定</span><span>更</span><span>愿意用小蛮腰这个代称去传播。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这和人们的潜意识息息相关——</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据脑科学分析，人的大脑中，主管情感和生命冲动的潜意识部分，只能感知非常具体的、形象化、</span><span>尤其和生、死、食、欲有关的东西</span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>一切很抽象化的东西都必须具象化，才能转化为潜意识的接受和喜爱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>下图即小蛮腰三个字背后的潜意识属性</span><span>——</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/9813882346306c052c32f34.44582400.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>所以，小蛮腰逐渐成为了大家共同认可的民间社交货币，成为了公认的广州塔代称。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为同样的原理——</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>央视大楼会被称为”大裤衩“，这里的”大“有吐槽的意思；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>湖南卫视会被称为芒果台，因为标志看起来像芒果；</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>萧敬腾被戏称为”雨神“，</span><span>刘若英被称为</span><span>”奶茶“</span><span>；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小米已经很形象，也还是被戏称为”粗粮“；<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王者荣耀被称为”王者农药“；<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>纽约被称为”大苹果“；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>皇家马德里被称为一种甜品（美凌格）；<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>英国人被称为龙虾（因为晒太阳皮肤就变红），美国人被称为乡巴佬（Yankee）；<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就这样，一个个严肃的、抽象的名称，被形象、生动、可吐槽、接地气的名称替代，流行开来。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span>所以——</span></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>形象化、仿生化、通俗化，是社交货币诞生的不二法门。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>小蛮腰定律就是——</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>把抽象的概念和高大上的东西，化为接地气的具体形象或说法，将没有生命的物体生命化，一个个社交货币，由此诞生。</strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了小蛮腰定律外，社交货币还有两个非常重要的属性：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一、情感沟通属性</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没有情感沟通特质的社交货币是难以成功的，或者说不成为社交货币。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这个时代，任何大型的网络流行都是情感的流通传染。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为，潜意识最能引发传播，而其生命潜能的属性，必然导向情、爱、恨、欲、俗这五种元素，才能铸造出一个个社交货币。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/15297333916306c05319c757.21432094.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>情绪既有正向的，也有反向的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了爱，恨也可以。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>比如中国男足如此让人失望甚至绝望，所以中国男足本身也成为了一种社交货币</strong>：人们通过不断吐槽中国足球，来渲泻自己的情绪和态度。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>海参，就成为了代指中国男足的反向型社交货币。</span></span><span><span> </span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span>二、文化身份属性</span></strong></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>社交不仅仅是情绪的传染，还有圈层的文化身份认同。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，<strong>社交货币就是MEME</strong>。MEME的中译名有模因、魔因、谜母、文化基因，等等，即可被不同的人模仿、复制和衍生的文化信息。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当两个人直至一群人，不断传播和交换同一种社交货币时，这后面其实是凝聚为一个圈层。<br></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在这个圈层里，往往有共享的价值观和IP内容，用社交货币来代表。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/5121100376306c0536488e6.69014877.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，次元文化圈有自己的一整套社交货币。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每一个明星的粉丝们，以所喜爱的明星的各种作品和事件作为社交货币，同时为自己的粉丝团取一个生冻感性形象化的名字，也作为社交货币。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>球迷们以不同的足球俱乐部、足球明星为社交货币。<br></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>时尚、文娱、动漫、游戏和竞技运动，是最能产生社教货币的产业，<span>它们所孵化的社交货币，都能自带</span><span>价值观和IP内容，</span><span>使参与传播者的身份感</span><span>溢出：</span><span>我是喜欢这种东西的</span><span>族群的人，</span><span>我是其中</span><span>的</span><span>一份子。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如本文一开始介绍的MEME，</span><strong>社交货币就是文化信息的传播与共创。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这个时代，一个品牌，如果没有属于自己的社交货币，不足以称为真正神受欢迎的品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>比如，瑜珈健身族群往往</span><span>以</span><span>LULUMELON的品牌价值观和视觉内容为社交货币。</span></span></p>
