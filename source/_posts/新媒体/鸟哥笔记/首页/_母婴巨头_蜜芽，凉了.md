<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><span><strong>这个曾经被百度和真格基金先后投资的“母婴巨头”，还是走向了终结。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月初，母婴电商平台蜜芽在官网发布公告表示，将会在2022年9月10日停止旗下APP的服务。在APP停止服务前，消费者的交易订单还可以继续进行，关停之后公司将会对平台的用户信息进行集中删除。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/181003255662cf5fd8118fe9.96979762.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span><span>●</span></span> 图源：蜜芽官网</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公开资料显示，蜜芽成立于2011年，电商平台在2013年正式上线。截止到现在，八年时间蜜芽已经服务超过5000万中国年轻妈妈，覆盖业务包括零售、自有品牌和会员营销等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是让人没想到的是，对于这样一个已经成长为母婴电商巨头的平台来说，也会一朝跌落神坛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>母婴巨头，一朝跌落</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说蜜芽是国内母婴电商的代表之一，并不过分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>蜜芽的前身是蜜芽宝贝，是中国首家进口母婴品牌限时特卖商城，由北大女学霸刘楠在2011年创立。成立之初，她一直都希望能够创造出一个简单、放心且有趣的母婴用品购物体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2014年，蜜芽宝贝官网上线，开始向进口母婴品牌限时特卖商城的进行转型。在官网介绍中，蜜芽每天都会推荐热门的进口母婴品牌，并且以远低于市场价的折扣在72小时内限量出售。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/195680761262cf5fd8653977.02089074.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：蜜芽官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2015年，蜜芽宝贝正式更名为蜜芽，业务范围也从最初的母婴消费升级到了亲子家庭消费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得注意的是，成立以来蜜芽所获得的融资可谓众多，先后有真格基金、红杉资本以及百度等知名投资机构出手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>尤其在2015年时，百度领投1.5亿美元，使得蜜芽市场估值超过10亿，也成为了当时国内估值最高的跨境母婴电商。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/51898867062cf5fd97e74b8.33061268.png" referrerpolicy="no-referrer"></strong></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：天眼查</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据公开资料显示，蜜芽曾经连续三年入选中国独角兽企业排行榜，并且还以100亿元市值位列《苏州高新区2020胡润全球独角兽榜》第256位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>然而就是这样一个看起来前景无限风光的独角兽企业，却很快就跌落了神坛。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今在苹果应用商店已经搜索不到蜜芽APP，安卓应用商店也显示该应用已经停止维护且无法重新下载安装。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，有业内人士表示：“蜜芽APP之所以被关停，一定程度上也因发展速度过快，但资金不到位，从而陷入了困境。同时再加上外部的激烈竞争，使得蜜芽不堪重负。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但实际上，除了这些，蜜芽还有更深层次的问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>曾深陷传销，多次被罚</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为国内最早布局垂直母婴电商的平台之一，蜜芽曾在创始人刘楠的带领下一路高歌猛进。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截止到2018年底，蜜芽服务过的用户5000万人，平台累计融资超过20亿，商品覆盖更是多达30多个国家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而就是在其疯狂成长的背后，蜜芽的融资步伐却停留在了2016年。</span><span><strong>资金跟不上，使其在2019年开始走上了下坡路，在这之中最直接的体现就是线下业务的收缩。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2016年初蜜芽就开始布局线下，并计划在未来1-2年时间内开设200家蜜芽乐园，然而直到2020年底，蜜芽也只有北京的一家门店还在营业过程中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/123097805062cf5fd9b59002.76037306.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：大众点评</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于线下业务的收缩，蜜芽相关人员曾解释说：“公司重心已经转移到了线上，线下业务从2019年开始暂停。”</span><span><strong>只是，抛开步履维艰的线下业务不谈，此前蜜芽也曾经深陷传销质疑。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年，在社交电商的浪潮之下，蜜芽推出了自己的plus会员制度。据了解，这一会员体系包括三个等级，想要达到一定的标准就需要不断的做业绩和拉人头，这一体系让蜜芽被很多人认为是打传销的擦边球。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管在2020年蜜芽升级了会员体系，但其本质依然没有任何改变。和最常见的传销方式类似，依然是不断的付费和邀请好友，最终遭到越来越多消费者的质疑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/172153027162cf5fda6e0c71.44197111.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：蜜芽官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>当然，除了深陷传销质疑之外，蜜芽还多次因为产品质量问题而被相关部门处罚。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据公开信息显示，蜜芽的运营主体是北京花旺在线商贸有限公司。然而和北京花旺有关的108条相关案件当中，主要都是以“网络购物合同纠纷”为主。目前，天眼查上有关北京花旺的处罚信息，共计有15条处罚总金额为25万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13928651162cf5fdacf1777.80333290.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：天眼查</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，在母婴电商平台投诉报告当中，蜜芽投诉量排在第一位，范围多以虚假促销、假货、奖励被作废等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>母婴垂直电商，前路何方？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>成立以来，蜜芽一直走的都是正品特卖的道路。只是随着近些年各大头部电商平台开始纷纷向母婴品类发力，这进一步削弱了蜜芽的优势，也瓜分走了不少母婴垂直电商的目标用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了蜜芽之外，母婴电商平台当中的贝贝网也在去年8月被曝出资金链断裂等问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/67618498662cf5fdb19f071.20909316.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：新浪微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>种种迹象表明，母婴垂直电商似乎在下坡路上越走越远。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原因是多方面的。首先是从2016年开始的外部大环境的改变，不少价格低且处于免税额内的进口母婴用品，在政策的调整下也需要缴税。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次因为生育率的走低，整个母婴赛道也变得越来越窄，业内对于母婴电商的未来并不看好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，更重要的原因还在于综合类电商平台对于用户的抢夺。已经有越来越多的消费者表示：“想要买什么东西在京东或者天猫上可以直接搞定，也都是正品，再也不用手机上安装那么多不同的APP了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，不仅仅是母婴赛道，如今很多垂直类电商都过得不太顺利，例如已经或即将走向终结的包括如凡客、聚美优品、蘑菇街等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反而以“轻资产”运营的模式发展的越来越好，如交个朋友、东方甄选等直播机构，可谓让蜜芽之类的垂直电商平台羡慕不已。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/209940902962cf5fdc3e2322.12754946.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：交个朋友官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>当然蜜芽也并不是没有做出转型以自救，做品牌就是其中一个重要方向。</strong></span><span>如今，蜜芽的发展以自有品牌“兔头妈妈”为主，在创始人刘楠的个人抖音账号里，着重对兔头妈妈这一品牌进行了介绍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着三胎政策的落地，其实对母婴垂直电商平台来说未来依然有较大的发展空间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是对蜜芽而言，选择关闭APP并运营微信小程序，将来如何稳住流量、合规运营也是亟待解决的问题。</span></p>
