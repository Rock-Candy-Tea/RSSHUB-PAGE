<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着蜂花、百雀羚、大白兔、汉口二厂等老字号的翻红，让品牌意识到情怀在年轻人受众群体中有着举足轻重的作用，开始在「怀旧情怀」上做文章，不少品牌也因唤起了一代人的集体回忆，摇身一变老字号品牌也凭借给年轻人留下「爷青回」的印象而实现了翻红。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在兵法先生看来，情怀是回不去的时光，如童年、青春；是忘不了的一段回忆/味道等，如在儿童节穿上“盛装”的场景、东方魔水健力宝的橘子味等。简单的说，情怀是通过跨越时间、空间、代际的方式，将人带入到一个特别的场景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如前不久崔健在视频号的首场商业化演唱会，除了内容具有观赏性外，其情怀满满的歌曲成功引发了用户狂欢，有数据显示，最高峰值观看人数超37万，累计超4500万观众观看。在更多人看来摇滚已死，名为“继续撒点野”的演唱会，给用户带来了激情四射的音乐之余，似乎让大众的青春又回来了，补偿了人们青春缺失的遗憾，重新燃烧了一次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，怀旧情怀既是人们对时间的感伤，又折射出用户群体的喜好，同时还能够从侧面反映出消费者的消费偏好。据克劳锐发布《年轻人生活消费习惯——食品饮料篇》显示，<a href="https://www.niaogebiji.com/tag/%E5%B9%B4%E8%BD%BB%E7%BE%A4%E4%BD%93" target="_blank">年轻群体</a>对「怀旧情怀」有着浓厚的兴趣，有94.7%的用户会因为怀旧的心情而购买小时候的零食/饮料，这也就不难解释为什么大白兔、汉口二厂、健力宝等品牌能够在新消费时代撬动年轻消群体了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/119192226862cd8f4bce6d57.58995188.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>毋庸置疑，「怀旧情怀」能够撬动流量。而品牌利用怀旧情怀</span>，一方面是作用于用户的记忆层面：<strong>得不到的想要弥补，得到了的想要回味</strong>。另一方面是打造更经典的形象，来满足年轻人追求的时尚消费态度与价值观。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>怀旧情怀是心灵慰藉：</span></strong><span>在一段时间里，得不到的东西，让人久久不能忘怀。而在有一定的经济实力后，看见曾经缺失的东西出现后，人们自然便会产生弥补缺失的心理，去安慰自己来获得满足。而品牌卖情怀就有着慰藉用户心灵的作用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>怀旧情怀是对过去美好的回味：</span></strong><span>同时，人生不易，而过去的在一些人看来，是无法延续的美好，于是品牌利用<span>「怀旧情怀」去勾起人们对过去美好的集体回忆，来实现群体性的情感共鸣。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了<span>「怀旧情怀」</span>作用于用户的心理，其实也是<span>老字号</span>品牌凭借情怀为自己镀金，将品牌包装成了一个更值得年轻人信赖的品牌。随着年轻人对文化的自信，老字号还是反映着年轻人的生活品牌与消费态度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>经典且值得信赖：</span></strong><span>在更多人看来，老字号是经典的象征，更是值得消费者信赖的品牌。如国货品牌蜂花，其经典的产品形象深深地刻在了用户的脑海中，其物美价廉是消费者对品牌的印象，并承载着许多人的童年回忆。而在持续的经营中，许多品牌没落甚至被新兴消费品牌代替，而蜂花却坚持着自己物美价廉的产品特点，在激烈的竞争中屹立不倒。这时候，经典的蜂花品牌，也成了值得信赖的国货品牌的缩影。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19096438062cd8f4c1c9691.67812891.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>代表着年轻人的生活品质与消费态度：</span></strong><span>随着爱国热情的高涨，年轻人对传统文化愈加自信。而老字号作为传统文化的一部分，其深厚的文化底蕴自然带着文化的光环，受到年轻人的青睐，并以购买国货为自豪，使用国货更自信，这代表着消费者对文化的自信，更代表着年轻人的消费者倾向与态度，彰显着当代年轻人的生活方式与品质。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着年轻人对老字号品牌的认可度提升，越来越多的品牌看见了<span>「怀旧情怀」营销的效果与对品牌的正向影响，均想要通过<span>「怀旧情怀」去撬动年轻用户人群，究竟品牌应该如何做怀旧的情怀营销？</span></span></span></p>

<p><span><strong>极具视觉冲击的怀旧元素</strong></span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>怀旧元素比比皆是，具有时代记忆感的元素，如弹弓、铁环、石磨、粮票、收音机、黑白电视机、大喇叭、大字报标语等等 ，这些具有明显时代特色的产品，在一代人心中留下了深深的烙印，品牌想要吸引不同的受众群体，就可以将极具时代记忆感的元素与品牌融合，唤起用户的共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如卫龙辣条就善于捕捉消费者心理与洞察市场变化，用具有60年代感的画面去打造极具视觉冲击力的海报，实现营销内容在社交媒体上的传播，引起了话题讨论，引发了年轻消费群体的情感共振，将卫龙打造成了一个极具人文情怀的品牌。</span></p>

<p><span><strong>重塑品牌故事，将经典变成用户的记忆点</strong></span></p>
</section>
</section>
</section>
</section>
</section>
</section>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基本上每一个品牌都有着自己独一无二的故事，可如何用新时代的玩法讲述专属品牌的故事，来加深用户对品牌的记忆，是一个营销难题。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而百雀羚就将品牌诞生的年代特色、产品特质与当今新颖的营销方式融合，率先利用一镜到底的表现手法去讲述品牌的故事，在长图中融合了<span>民国时期的社会风貌，并将品牌进行了巧妙的植入，同时将产品使命<span>「</span><span>我的任务就是与时间作对<span>」与内容融合，讲述了一个极具时代特色的品牌故事。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99578857962cd8f4cd3a676.93929865.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>富有故事性的内容，既加深来了用户对品牌的记忆，又将百雀羚经典国货的形象彰显，同时，将产品特质与故事融合，加深了用户记忆，又深化了品牌形象。</span></p>

<p><span><strong>利用消费者的心理缺失，打造集体回忆</strong></span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了以上两种方式，品牌还可以充分利用用户对某种元素的缺失去打造营销内容，来唤起用户的情感共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如在物质匮乏的年代，80、90后对大城市的向往、对未知的好奇、对西游记印象深刻等，华为通过讲述品牌西游记取经的故事，唤起大众的集体回忆，让黑白电视机、黑白照片、英雄钢笔、孙悟空日历海报和皮影等具有集体回忆感的元素，增加了内容的亲切感，而极具代入感的内容暗喻品牌的出境，而获得了集体性的共鸣，感人至深，也让华为的《悟空》成为了当时的现象级营销佳作。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>营销方式千千万，能够撬动流量的营销自然是有效营销。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>怀旧，作为普遍的社会现象。品牌想要让自己的营销实现自来水式的传播，懂怀旧情怀去撬动流量，将承载着无数人童年/青春记忆与消费者曾经的满足/缺失打造成营销内容，让内容拥有了极强的代入感，能够引发大众的回忆，也自然能够产生事半功倍的传播效果。</span></p>
