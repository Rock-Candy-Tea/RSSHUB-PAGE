<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在相继拿下美国四大球等大型赛事合作之后，抖音这一次又把目标瞄准了<a href="https://www.niaogebiji.com/tag/%E4%B8%96%E7%95%8C%E6%9D%AF" target="_blank">世界杯</a>。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在前几日，中央广播电视总台和抖音集团联合举办云发布活动，并宣布抖音集团将会成为2022年卡塔尔世界杯持权转播商和中央广播电视总台直播战略合作伙伴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/130040169662beba26f3f7d4.84841555.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：字节跳动官方公众号</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消息一出，立刻引起了网友热议。毕竟，在2024年巴黎奥运会之前，本届世界杯是唯一的“S级”大赛，无数双眼睛都在紧紧盯着，试图蹭一波热度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在前期竞赛过程中，抖音已经率先抢占了先机。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9935821603705940eddf2.13315027.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>拿下世界杯转播权</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一个月前或许很难想象，2022年卡塔尔世界杯将可以在抖音APP上直接观看。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟以前的大型体育赛事更多是被电视台垄断，或者部分长视频平台拥有相应的转播权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次抖音所扔出的“重磅炸弹”，确实让很多人始料未及。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管抖音也曾拿到了和美国四大球、温网等大型赛事的合作，并且在2021年还成为了欧洲杯官方合作伙伴。然而这些赛事不管是量级还是全球影响力，都不可和世界杯相提并论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/199570684962beba287e7ad7.32819327.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：FIFA世界杯官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>如果说前几年拿到的各体育赛事还只是“开胃小菜”，那今年抖音在拿下世界杯转播权后，算是真正站在了全球体育版权的中心。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，不仅仅是国内的抖音，在海外TikTok同样也曾经借助欧洲杯等体育赛事成功“出圈”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/143437084262beba2b66b489.78110216.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：抖音官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据统计，仅仅是在2021年，TikTok上关于热门体育的标签浏览量就翻了两倍，每日视频观看次数最高更是达到了15亿次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅如此，超过70%的欧洲足球俱乐部在抖音上面的活跃度都极高，如利物浦、切尔西、曼城等老牌豪门粉丝量均已突破百万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/119619382662beba2c1bd9c8.84630188.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：抖音APP</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这次和中央广播电台的联合合作，除了具有转播权之外，双方具体的合作内容并没有进一步披露。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过从今年11月21日到12月18日世界杯举行期间，抖音作为持权转播商，相信会围绕世界杯推出大量的视频和专题，从而迎合当下这个短视频时代。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/7051603926037059d0cdda0.93459463.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>长、短视频平台的</span></strong><strong><span>体育赛事之争</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频平台在大举进入体育赛事争夺战之前，世界范围内的重大体育赛事基本都被腾讯、咪咕、优酷等长视频平台牢牢把握。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯体育作为体育赛事转播当中最早的一批，一直以来都牢牢的把控这NBA和英超等大型赛事的版权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在2018年，央视把俄罗斯世界杯版权分销给了咪咕和优酷，咪咕开始正式入局体育版权市场；优酷也在暗暗发力，并拿到了斯诺克等赛事的独家转播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，体育赛事的版权市场也进入到了一个情绪高涨时期，体育赛事版权分销价格更是水涨船高。例如腾讯体育拿到的NBA版权，5年转播权金额达15亿美元，相比较上一个版权周期增长了三倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/30984066562beba2ce1ff44.38792995.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：腾讯体育官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>只不过，彼时的短视频平台并没有过多的在争夺体育赛事的主战场现身，但一连串的动作也开始对长视频平台产生威胁。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自从2018年拿下NBA合作之后，字节跳动又先后拿到了职业橄榄球联盟、温网等大型赛事的版权，海外市场上还包括欧洲杯等体育赛事，试图打造出一个全球化的体育生态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为抖音最直接的对手，快手也不甘示弱。先后和斯诺克世锦赛、美洲杯、东京奥运会等多项全球顶级赛事IP达成合作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在去年3月，快手还官宣成为了CBA联赛官方<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B9%B3%E5%8F%B0" target="_blank">直播平台</a>，这也是快手在腾讯、优酷等巨头退出CBA联赛版权争夺之后的又一个大动作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/97728177162beba2e3daa27.14018182.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：快手官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>之所以会有如此变局，很大程度上也是因为当下短视频平台即时化和碎片化等特点的推动。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快节奏的当下，能够完整看完动辄异常两个小时比赛的球迷已经越来越少。很大一部分体育用户往往只喜欢看碎片化的精彩瞬间，例如足球比赛只看到有关进球部分的剪辑即可。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，这种趋势会让“抖音们”更加如鱼得水。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1154105644603705abbbd504.02631435.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>抖音们，不爱体育爱流量</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管对体育赛事版权的抢夺热闹非凡，但从商业化的角度来考虑，体育赛事的变现之路并不容易。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>长久以来，如英超、NBA等世界级的体育赛事，版权价格一直处在上涨通道中。然而相比较不断飙升的版权费，体育赛事本身的变现方式却极为有限，大多数依靠会员付费、广告和版权分销等形式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对巨头平台而言，除了连年上涨的版权价格，在运营体育赛事方面也要付出巨大的成本，这也直接导致不少平台完全是在“赔本赚吆喝”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据了解，苏宁体育曾在2017年花费20亿购买各项体育版权，但最终收入只有1.5亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/95423371062beba2eb1cda7.36589043.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：苏宁体育官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>巨额亏损并没有吓退搅局者，“抖音们”反而选择逆流而上，在这之中有一个非常关键的逻辑。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过公开信息可以发现，不管是抖音还是快手，这些短视频平台更加看重的是奥运会、世界杯等周期短、流量大的体育赛事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这种级别的赛事而言，与其说抖音想要争夺体育版权，倒不如说这些比赛对于平台而言更是流量保证，品牌方看到巨大的流量显然也更愿意付出成本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如2018年世界杯，就给优酷带来了超过3000万的用户；2019年腾讯和NBA合作之后，腾讯体育的用户也有了大批量的新增。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这里可以看出，在国内流量日渐触及天花板的背景下，尤其是大型体育赛事对于平台而言是不可多得的流量增长点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>当然对于抖音们来说，面临的一个严峻挑战是尽管体育赛事可以在短时间内带来大量用户，但在留存率方面却很难得到保证。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/157925070862beba2f3cde42.83944056.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：抖音官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此前北京冬奥会举办期间，咪咕视频的用户量也迎来了一波高涨。然而随着冬奥会的落幕，不少网友表示自己随后就卸载了咪咕视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拿下版权只是一个开始，在未来究竟如何保持热度并真正在体育这一领域站稳脚跟，“抖音们”显然还有很多课题要做。</span></p>
