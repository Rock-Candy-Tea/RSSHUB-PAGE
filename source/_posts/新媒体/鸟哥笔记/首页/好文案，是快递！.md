                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/1b34752d7d09fd5c529274c90bfe5f8e.png" alt width="900" height="383" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">写<a href="https://www.niaogebiji.com/tag/%E6%96%87%E6%A1%88" target="_blank">文案</a>，是一个<a href="https://www.niaogebiji.com/tag/%E5%86%85%E5%AE%B9" target="_blank">内容</a>人必不可少的技能。写好文案，却不是一件容易的事。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">文案这个词，根据语境不同，有两个含义：一是指广告传播中的文字内容；二是指负责撰写这些文字内容的人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要做一个好文案，第一步就是不要只把自己当个文案，不要把自己只当作一个遣词造句、铺排文字的手艺人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我一直觉得，写好文案并不是比谁的文采更好，谁用的手法更高明。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">文案和文学根本的分别在于，文案的撰写是为传播、沟通服务，承载一定的任务和目的，且一般都是商业目的。就像李奥贝纳说：好的广告文案，都有着简单且平凡的外表。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，前奥美文案女王林桂枝给三全食品写的那句：吃点好的，很有必要。你看完压根不会觉得它真有文采，而是它真的很准。它想要传递的信息是，一来是传递三全水饺的品质好；二来传递的是一种生活态度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/186bec2690c5b968a0a0ce78ff8ef4ed.png" alt width="900" height="383" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，我觉得，好文案，是快递。what？快递？你肯定是在跟我开玩笑，文案跟快递有两毛钱关系吗。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">您细品，快递的目的是什么？把用户购买的物品送到用户的手上。那我们写文案的目的是啥？我就觉得文案的第一目的，传递品牌想要让用户接收到的信息。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">文案不只是表达，而是传递，说服，影响，是引发或促成改变。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，2003年的淘宝slogan，淘你喜欢。比如，元气森林的0糖0卡。比如，小红书最早的slogan：找到国外的好东西。淘你喜欢，重点是在传递一个信息“淘”，是一种行为。“找到国外的好东西”，传递的信息是在小红书，你能找到国外的好东西。而0糖0卡，是元气森林的核心卖点，这是它和其他饮料品牌的差异化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/d75d0edada382b9ee8297e941d4f5265.png" alt width="900" height="383" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">好文案扮演的第一重角色：快递核心卖点给用户。</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么怎么产出一条文案？当然是基于商业目标，来创作并传播内容就好了。稍微扩展一下，就是四个步骤：商业目标、传播目标→核心体验→传播人群→传播媒介、传播路径。传播目标，你要实现的传播目标是什么？核心体验，你打算表达什么核心信息，来实现这个目标？传播人群，你期待的受众是哪些人，你要对谁说话？传播媒介、路径，你打算在哪跟他们说这些话，你打算怎样让他们听到这些话。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">简单举个例子，我之前做的一款果酒---石榴酒。要做一波传播，让更多人知道我们这款酒，通过传播提高销售业绩。这就是我的商业目标，那么文案肯定是围绕这个来展开创作的。产品的核心卖点：一是它采用的原料是国家地理标志认证的怀远石榴；二是老字号安徽特产；人群定位是爱喝酒、享受微醺的女孩。那我们最后定的一句文案：好石榴酿造好酒。我们要传递给用户的核心信息是，原料品质好。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">文案不是一堆形容词的堆砌，是要传递信息，传递产品的核心体验。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，瓜子二手车的slogan：没有中间商赚差价。它传递给想买二手车的目标用户是，我们没有中间商，没有差价，更深一层是，价格比较透明。比如，美团外卖的那句：美团外卖，送啥都快；叮咚买菜的“最快29分钟抢先到家”；这些都是针对时常点外卖的人群，对他们而言点外卖最核心的需求就是要快。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">好文案扮演的第二重角色，快递一种场景。</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们写文案一定要懂得透过品牌去思考，透过文案去表达品牌。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">表达品牌的一种方式，场景化。记得空手老师在《传神文案》里有一个观点：对于营销来说，场景是品牌进入消费者生活的入口，是品牌获取心智流量的接口。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，资生堂的晚安粉，产品名是资生堂美白保湿晚安护肤蜜粉饼，但在营销的过程中，它借用了人生活中的一个高频场景，晚安。围绕这个场景，它做了一系列广告，比如和男朋友过夜，洗完澡卸了妆，觉得脸上裸着，心里忐忑，怕卸了妆的皮肤吓坏男朋友，想扑点粉。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们的需求总是在场景中产生，不同场景下，我们会产生不同的需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/8e5dbb54e1b3c6ced9d6a3571208fedc.png" alt width="900" height="383" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，饿了困了，喝红牛；怕上火喝王老吉；这些都是场景式的文案，它给用户传递面对这种场景下，你可以做什么。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">场景帮助品牌融入消费者生活，在消费者生活中锁定一个位置，帮助品牌进入消费者心智。这就叫做场景定位。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再提一句，kindle的“盖Kindle，面更香”。这是种自嘲的场景式文案，是Kindle在2019年世界读书日推出的广告。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我自身比较喜欢的一个场景式文案，来自Jeep自由光，它是这样写的：</p>
<p>“他不怕黑，是因为你曾在漆黑的夜里，带他看过最亮的星。</p>
<p>她更有主见，是因为你带她发现过的世界，比课堂大得多。</p>
<p>他比同龄人更爱问为什么，是因为你早就为他，打开了好奇的大门。</p>
<p>看过世界的孩子更强大。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，写文案，并不是追求高大上，而是追求一种精准传达。就像送快递一样，必须要送到收件人的手上，这样才能达成你所设想的目标。</p>                        <div style="height: 1px"></div>
