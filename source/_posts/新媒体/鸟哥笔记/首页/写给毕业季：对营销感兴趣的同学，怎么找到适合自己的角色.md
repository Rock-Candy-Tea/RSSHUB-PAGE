                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-first-child data-pid="6Ez1u1_o">焦虑和压力是任何工作的必有部分，网络营销行业也不例外。对这一行感兴趣的同学千万不要被现在网上那些一夜爆火的网红和抖机灵的文案迷住双眼，那不是真实的互联网营销，至少不是大部分人需要面对的互联网营销。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="YytwrYR9">踏踏实实坐在格子间码字、改图、修代码才是。互联网营销在很多时候其实并不是一个很酷的行业。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="GeWC9IF3">怎么样？听到这些是不是打消了一半的热情？那么好，现在我们可以理智的分析一下这个行业的前景和入门方法了。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">关于互联网营销的一句话总结</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="--3jFHGS">算是入行十几年的老人了，在互联网营销中经历了很多，也见证了中国互联网的发展。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="ZQZUUySq">如果一定要给新人一句话的忠告，就是：永远不要陷入对自己的自我否定，始终以新人的姿态拥抱互联网的变革，你才能不被时代抛弃。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="ZY3ie0Y_">另外，找到工作才能学到经验，有经验才能找到这方面的工作，这是一个悖论，每个人都是从没经验的时候过来的。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">简单说几个比较容易让你进入这个行业的岗位</h1>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); font-size: 16px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">1、内容文案类工作</h3>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="ZJ-2vm9-">不管互联网发展到什么程度，也不管媒体形式发生什么样的改变。是门户网站时代、还是blog时代或者现在的短视频自媒体时代。他们的核心都是内容，都是文案。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="QWCBx3x3">所以可以说文案是这个行业需求量最大的一个岗位，对新人也比较友好。只要是你能踏踏实实的按照要求输出内容，逐渐的培养自己的语言风格，那么就可以成为一个优秀的文案。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="4H26eHnW">觉得杜蕾斯的广告很酷吗？觉得现在动不动就互联网现象级广告很酷吗？他们都是无数文案默默熬了多少夜、一字一句修改出来的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="UYlijS31">只要你愿意，你也可以成为这么酷的人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="ApA_drb0">文案圈儿通常会有一个错误的观点，就是对原创的盲目追求。当然，我们需要有版权意识，尊重原创，但是这并不意味着要盲目原创。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="eZzHOWPo">任何好的文案、设计，都是从模仿走过来的。先去学会模仿那些好的文案，再总结自己的风格，盲目原创约等于闭门造车，毁人不倦。</p>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); font-size: 16px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">2、社交媒体类工作</h3>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="l_92iUhE">最近几年新出来的岗位，通常国内会叫新媒体运营之类的名字。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="__YdxaWj">这类岗位对新人，尤其是年轻人很友好。因为这个岗位可以说没有任何经验、理论可以借鉴。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="W6R2MxXP">其一是因为整个行业都还处在初级发展的阶段，大家都在摸索，还没有形成一套完整的理论。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="kSsoB45e">其二在于新媒体区别于传统媒体，需要面对的是数以万计的活生生的人，你需要去直接和这些人打交道，年轻人可能在这方面更有优势，可以抓住受众的兴趣点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="5QvdsJuM">如果对这类岗位有兴趣，几个小建议：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="tdhbXKph">这是高度依赖创意和细节的岗位，国内某些公司们对这个岗位的定义并不是那么完善，让你每天去发垃圾帖子、刷流量、盲目追求粉丝量的增长等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="gjj4o1kF">这都不是新媒体运营，这是新时代的搬砖苦工。当然，刚毕业找个工作求生存做这些也并不是不行，只是你要自己学会总结和探索，不要被公司错误的企业文化带偏轨道。</p>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); font-size: 16px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">3、搜索引擎优化SEO</h3>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="F0tfxlUk">很多人可能都觉得这个岗位在逐渐消失，但是并没有。简单理解，搜索引擎优化是通过技术和内容等手段，提升网站在搜索结果中的排名。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="1bEJlTkC">不过现在要讨论的是更加宽泛的SEO概念，即通过内容、关键词选择、技术等手段提升内容的价值量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="VQnaWjUK">两个核心点发生了变化，其一是不再局限于百度、谷歌等搜索引擎，其二是不再把工作结果局限于提升排名，而是提升价值量。这个价值量可以是排名、可以是粉丝增长、可以是销售量增加等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="MWo24QJm">因为自己是做SEO出身的，所以对这个岗位的成长路线会更了解一些。对新同学要说的就是，SEO没有消失，只要是你做的够好，它可以给你带来一个稳定的生活保证。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="TOx_Ja1r">但是前提是你对它能够始终充满兴趣，这不是一个靠勤奋就能胜任的岗位，见过太多新人，做事勤勤恳恳，但是对SEO就是没有兴趣，这是做不好的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="EOxV4uAo">另一点，SEO这个岗位，尤其是对新人，聪明可能不是一件好事。做了SEO你会知道，公司交给你的任务你可以有太多便捷操作可以完成。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="dUsW3Sqo">但是如果你想把SEO变成未来可以让你安身立命的本事，不要这么做。踏踏实实去做，捷径不是给新人走的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="RSV7zpoF">太年轻，有的事真的把握不住。</p>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); font-size: 16px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">4、开发和设计类工作</h3>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="5SVX9DFg">这两类之所以放在最后说，是因为觉得这两类基本可以归为一类。都是需要一定的技能基础才能胜任的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="EE8awVie">当然，也有那种一百八十度转行来做开发的程序员，但是毕竟是少数。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="2a6r-Y7A">这类工作属于技术岗位，所以一定是技能至上，不管是新人还是老人，苦练杀敌本领才是正道。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="GyfMnKW3">不过建议还是可以同时接触一下营销环节，对未来的职业进阶和个人发展都有好处。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="IO9LiEW0">最后希望各位对互联网营销感兴趣的同学都能找到如意的工作，三百六十行，行行出状元。也给各位职场老鸟们一个建议，对新人还是要友好一些，学东西慢多理解理解，他们今天在你面前笨手笨脚的样子，不就是当年的我们么。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="86U1kBSP">赚到的要给人，学到的要教人，共勉。</p>                        <div style="height: 1px"></div>
