<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上周<a href="https://www.niaogebiji.com/tag/%E7%88%B6%E4%BA%B2%E8%8A%82" target="_blank">父亲节</a>，大家是否都借机向父亲送出了祝福？都说“父爱如山”，山是深沉、厚实却沉默的，正如爸爸对子女的爱，很少大声宣之于口，只用行动表达；同样，孩子们对父亲的爱，也常常缺少一点表达勇气和契机。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年有个品牌就用了一个巧妙创意，给予大众表达爱的契机和勇气。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>作为</span><span><span>贴近百姓生活、具有</span>“烟火气”的社区服务者——</span><span>“阿里巴巴旗下社区电商品牌”</span><span><a href="https://www.niaogebiji.com/tag/%E6%B7%98%E8%8F%9C%E8%8F%9C" target="_blank">淘菜菜</a></span><span>，就在今年父亲节化身</span><strong><span><span>「袋言人」</span></span></strong><span><span>，让数百万个</span>“工作”在用户手边的塑料袋，成为子女向父传情达意的“道具”。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家常见的塑料包装袋上添加了一系列以子女口吻讲述的花式文案，将那些平时难以说出口的对父辈的关怀，全表达了出来：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“爸，以前你总唠叨喝牛奶补钙，现在该我唠叨你</span><span>了”；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“平时少些操心，</span><span>每天多</span><span>加点菜”；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“少打麻将少抽烟，给你买点小海鲜”；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> ……</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/16644474462b4829c1f86f7.07891534.jpeg" width="444" title="父亲节，淘菜菜为爱“袋”言" alt="父亲节，淘菜菜为爱“袋”言" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>简单的创意升级让</span><span>塑料袋这一最平凡又不起眼的业务耗材，不仅</span><span>“能装”更“能说”</span><span>，让每一个为父亲买菜的孩子们，买到的不仅仅是一份菜，更是一份溢于言表，对家庭、对父亲真挚的爱。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据说此次淘菜菜共发出数百万个这样的“袋言”塑料袋，进行全国团点覆盖，真正把这份“能说出口的父亲节祝福”送进了千家万户。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这组为爱“袋”言的创意，让品牌成为父亲与子女间传递关怀与爱的桥梁，也让品牌实现了一场真实的情感沟通。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>父亲节营销扎堆，品牌想要真正触达、触动目标人群并不容易。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>淘菜菜的</span><strong>“为爱袋言”</strong>却自带触达优势——<span>包装袋一向都是品牌在产品之外，集中传递品牌形象和接触消费者最直接的物料，是</span></span><span>“天然的品牌用户触点”</span><span>。因此淘菜菜从塑料袋出发，借父亲节节点自然能实现目标用户的精准触达、沟通。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>精准触达基础上，在触动目标人群的创意文案中，淘菜菜还开发了各式各样的子女与父亲日常问候话题</span><span>，</span><span>将对父亲的关爱落到了一个个生活细节中。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>淘菜菜的塑料袋文案里有一系列的父亲节主题问候，句句都是以</span>“拉家常”的口吻关心着老爸的生活：</span><span>“<span>菜买了一大袋，爸你今晚多整俩菜</span><span>”</span>；“有时不愿开口说想你，就说，买了点菜给你”。</span></span><span><span></span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>第二个层面则是</span>——淘菜菜通过各种主题塑料袋传播，扩散新的问候方式时，也是在鼓励大家培养一种更日常、高频的“关爱<span>习惯</span>“。</span><span>这种关爱习惯，可能就是时常为老爸买买菜，提醒父亲多喝牛奶、多吃肉，少喝酒。</span><span><span>此时，</span>“上淘菜菜买菜”也不再仅是单纯的“</span><span>购物</span><span>”</span><span>动作，而是建立着人与人的亲密联系。</span><span>不只是父亲节，这种联系还可以在亲自、朋友、爱人之间继续延展。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>就是这样</span>“通过一个塑料袋表达关爱”的简单创意背后，有着淘菜菜对广大用户人群的深刻情感洞察：正如前文所说，含蓄的<span>中国亲子关系中需要一种新的方式来打破沟通桎梏。</span>“<span>为爱袋言</span>”<span>鼓励父亲与子女互诉心里话，正是</span></span><span> “以一种含蓄的方式表达着不含蓄的爱”</span><span>。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>这样的创意充分贴合中国亲子关系特色和父亲与子女的情感表达需求，由此带动了广大用户自传播</span>——<span>打开微博，在</span></span><span>#为爱袋言#</span><span><span>的话题下，就能看到不少网友主动晒出</span>“袋言”来表达对父亲的祝福。</span></span><span><span></span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>抓住一个父辈小习惯</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>以小博大带动广泛共鸣</strong></p>
<section><span style="font-size: 16px; font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">暖心创意带动网友自传播之际，淘菜菜还抓住了一个父辈“爱囤积塑料袋”的小习惯，打造系列出圈传播，成功突围父亲节杂乱的营销声量。</span></section>
