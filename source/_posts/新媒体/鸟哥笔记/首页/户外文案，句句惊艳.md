<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在数字营销大行其道的时代，仍有一种流传了上千年的广告形式，却是如今价格最贵、效果最好的广告媒介之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>它就是「<a href="https://www.niaogebiji.com/tag/%E6%88%B7%E5%A4%96%E5%B9%BF%E5%91%8A" target="_blank">户外广告</a>」。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/3684864406284019f952486.62755334.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么是最贵而且效果最好呢，原因主要有两点：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先，户外广告之好，在于它又大又高。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽说“尺寸”不是最重要，但够大，展示效果的确更好。所以户外广告牌又称“高炮”，打点什么都能一炮而红。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更大的尺寸，带来更震撼的画面，苹果的户外广告，拍张照片摆上，效果就很喔噻，如果换成电梯广告，效果肯定大打折扣。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1914528171628401a038fce8.11474461.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>文案也一样，粗大显眼的字体，是路人无法忽视的存在，可能一句放在手机屏上有些平淡的文案，一旦放上户外效果立马不同，冲击力Buff叠满。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1793175463628401a08084c0.49736806.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>※ 今年3月美国佛州通过一项“不说同性恋”的法案，引起很多人的不满，他们在多个城市打出户外广告“说同性恋”进行抵制。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其次，户外广告之妙，还在于和周围场景结合的精巧。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天时、地利、人和，光线、影子、反射，都可以作为创意发散的工具，例如最近在英国最环保城市布莱顿的海滩出街的科罗娜啤酒户外广告</span><span>（来自伦敦W+K广告）</span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到了18:30-45这个最适宜喝啤酒的时间，太阳光影刚好勾勒出了酒瓶的形状，文案 MADE FROM THE NATUNAL WORLD（由大自然制作而成）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>综合以上的优点，户外广告只要文案、创意稍有出彩，就很容易被拍照上传社交网络，得到更广泛的二次传播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来上一波非常有代表性的户外广告，看看文案们是如何各凭本事，把一个个户外广告位利用到了极致。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>01</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span> </span></span><strong><span> 语录体文案</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>铿锵有力的语录式文案，配上户外广告强大的昭示力，可以说是相得益彰，起到1+1>2的效果，震撼路人一整年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而说到金句语录，其实就是那么几个套路：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、一些反常规的观点：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/815311814628401a0c82060.97122213.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（广告公司：美国分工广告）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>THE AVERAGE HUMAN</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>RESETS PASSWORD MORE</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>THAN THEY HAVE SEX.</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>——STYTCH，Go passwordless</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>普通人重置密码的次数</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>比他们性生活的次数都多。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>——STYTCH，无需密码登陆</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今最令人头疼的事，奥密克戎排第一，噢密码输错可能就要拍第二了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每个账户登陆都要密码，有的要大小写，有的要字母+数字，每每让人陷入记错-重置-又记错的崩溃三连。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是美国开发者平台STYTCH发起了一场《无密码》的广告运动，用来宣传他们登陆无需密码的产品特色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>文案把人们输错密码的次数和性生活的次数进行对比，提出了这个无比扎心的观点，让人直呼：别骂了别骂了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2、一些前后句的叠音词：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/139476887628401a170ddb6.08822995.png" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（广告公司：伦敦葛瑞广告）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>THE ONLY WAY TO REACH TURE ZERO EMISSIONS</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>IS WITH ZERO OMISSIONS</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>#ZeroOmissions#</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“0排放”的唯一方式，是“0疏漏”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们说过，这种前后句的叠音词重复，不仅可以带来句子的韵律和谐，而相似又带着变化的发音和涵义，也能带来巧妙感和张力，既容易记忆，也能带来很强的语录感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这句文案中，前一句的</span><span><strong>ZERO EMISSIONS 0排放</strong></span><span>，和后一句的</span><span><strong>ZERO OMISSIONS 0疏漏</strong></span><span>，形成了一对叠音词，给人一种妙句天成的感觉，表现出沃尔沃汽车致力于“0排放”的努力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3、一些前后的对比反差：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1967212317628401a1e02465.43744289.png" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（广告公司：澳大利亚AT创意）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>IF YOU THOUGHT 262 DAYS<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>IN LOCKDOWN WAS TOUGH</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>IMAGE 10 YRS IN LOCKUP</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>——STOP THE COWARD PUNCH</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如果你感觉262天的监禁很难熬</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>那么想象一下10年监禁</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>——停止打“懦夫拳”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个广告出街时，澳大利亚墨尔本刚刚结束了为期262天的COVID封锁期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>文案意在提醒那些打“懦夫拳”（家庭暴力）的人，262天的封控很难熬吧？那么可以想象你因为家庭暴力被判10年监禁的感受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>文案采取了前后对比的手法，<strong>262天对比10年</strong>，<strong>LOCKDOWN（封锁）对比 LOCKUP（监禁）。</strong></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过这种书法不仅可以凸显文案的主张，也可以带来一种和谐的韵律，还能够形成一种戏剧化的张力，让广告更加深入人心。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>02</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span> </span></span><strong><span> 创意式文案</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有时创意不一定非得是精巧的设计或视频，它也可以是文案的独角戏。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>字形、字意、字序或不同语言的运用，都可以产生巧妙的创意，把这种<a href="https://www.niaogebiji.com/tag/%E6%96%87%E6%A1%88%E5%88%9B%E6%84%8F" target="_blank">文案创意</a>搬上户外广告牌，效果同样加倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/969515972628401a2c09d15.73036445.png" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（广告公司：Special Australia）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>BONDZZZZZ</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Now in bedding</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>邦德士zzZ</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>现在做床品了</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近100年来，BONDS（邦德士）一直是澳大利亚最受欢迎的内衣品牌，可谓家喻户晓。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今他们准备进军床上用品领域，如何告知广大澳洲人民呢？他们想了一个办法，把品牌名改为BONDZZZZZ——“ZZZZZ”来自英语俚语zizz（睡觉），这样一下就和床品联系起来了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简洁直接的文案创意，用黑底大白字的形式展现，让效果更加醒目，右上的折角像是被子被掀起的一角，为整体的创意也增色不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1950172843628401a43d96b4.37154442.png" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（广告公司：以色列李奥贝纳）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>防止暴力的最佳保险是共存（希伯来语）</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>防止暴力的最佳保险是共存（阿拉伯语）</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px; text-align: left;"><span>来自以色列的保险公司“哈雷尔保险”，广告的背景是去年以色列境内爆发了多起犹太人和阿拉伯人的冲突，两个人种共占据了以色列98%的人口，其实他们大多数都能和平共处，所以哈雷尔保险出街了这个户外广告。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>文案分别用犹太人母语“希伯来语”和阿拉伯人母语“阿拉伯语”，写下了这句文案<strong>「防止暴力的最佳保险是共存」</strong>，表达了对双方和平共处的期望，同时还提及了自己的保险业务，创意非常巧妙。</span></p>
