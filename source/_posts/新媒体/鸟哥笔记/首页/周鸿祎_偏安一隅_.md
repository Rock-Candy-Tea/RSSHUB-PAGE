<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也许是还未适应“配角”角色，第十届互联网安全大会期间，周鸿祎本色出演了过去的自己，直言不讳地“炮轰”微软，称其抄袭了360的免费杀毒模式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是，这非但未能为360抢得舆论制高点，大众的冷嘲热讽更是如潮水般涌来，逼得周鸿祎急忙将“炮轰”解释为了“惺惺相惜”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其个人风格从“diss”到“peace”的快速切换，未免显得反常，加之“360和微软都是面向未来的公司”的表述，不难猜测：“炮轰”微软或许不是周鸿祎的本意，而是老周为360博取舆论关注、强行抬高自己的“碰瓷营销”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回看360近些年来的境遇，同颠峰时期相比，还能立足于主流语境的，或许只有屡屡现身的老周了。</span></p>
<section data-mpa-template="t">
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong><strong>永远的</strong><strong>“安全”</strong></strong></span></h2>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽说网络安全一直是360的基本盘，但数年前，纯粹的安全场景难有想象空间亦是不争的事实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>个人安全场景赚不到钱这事，很大程度上是由周鸿祎一手造成的，毕竟把行业拖入免费怪圈的正是360。但随着移动互联网时代到来，PC场景式微，周鸿祎逃离了网安赛道的束缚，去往各条赛道捞金。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/153784673362eb47082d2d88.62102701.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于这一点，坊间有传闻称腾讯与百度曾专门成立研究与遏制360的团队，或许是从彼时落寞的盛大处汲取了灵感，团队首计便是捧杀周鸿祎。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“周鸿祎是个‘战神’，所谓‘战神’，就是你跟他正面竞争是绝对打不死他的。你要捧着他，让他飘飘然，让他觉得自己无所不能。这样他就会把战线拉得很长，开始犯错误。”《沸腾新十年》如是写道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此后，周鸿祎接连切换赛道，从手机、硬件到直播，再到短视频乃至元宇宙，周鸿祎似乎一直在追风口的路上，360的名字却同奇酷手机、奶糖短视频等落败者一道，显得陈旧不堪。按周鸿祎自己的话说，就是“被别人当成‘肥肉”分而食之了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这一切，齐向东自然看在眼里。在周鸿祎扩张业务，剑指BAT之际，体制内出身的他看准了彼时寡淡的政企安全市场，带队出征，名曰奇安信。而之所以另起炉灶，除周鸿祎沉迷投资外，或许同当时360的美股背景有关，毕竟政务安全并非儿戏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本来，360与奇安信之间签有“君子协议”，即360不摸政企安全，奇安信不碰C端市场，双方井水不犯河水。但随着360外战接连失利，叠加政企安全风口涌现，二者最终还是上演了一场“兄弟反目”的戏码。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一波互相减持后，2019年上半年，360将奇安信股权悉数转让；同年9月，周鸿祎将政企安全作为核心战略，并为其倾尽资源。最终，奇安信搬离360大楼，这对相识于雅虎时期的老友，最终没能逃过各奔东西的宿命。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而360重新将主视角切回安全，很可能是周鸿祎在认清自己没有拿到“主角剧本”后的无奈。只可惜今时不同往日，尽管高调进军政企市场，但360早已不再是曾经那个靠着“三级火箭”降维打击整个行业的颠覆者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>营收层面来看，过去两年，360安全业务营收分别为8.08亿元、13.81亿元，同期奇安信营收则为41.61亿元、58.09亿元。这表明，企服安全业务端，奇安信的营收规模已经甩开360一大截。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，一位网安从业者告诉光子星球：“在相关政策出台之前，客户很少会评估网安方案的实际效果，更多只是如完成任务一般机械式地填补空缺。行业说白了就是一场拼资源的游戏。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换言之，企服安全赛道存在先发优势一说，越早涉足的老玩家，越能累积资源形成正向循环。在此逻辑下，2019年才入局的360，既无法同2014年成立的奇安信相比，也比不过布局较早的其他玩家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据CCIA《2022年中国网络安全市场与企业竞争力分析报告》，分道扬镳的奇安信斩获中国网安产业竞争力50强榜首，而榜单前十中既有深信服、启明星辰、天融信等手握资源的老玩家，亦有腾讯、阿里、华为等大厂，360则位居11位，处境略显尴尬。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是，对360而言，企服安全的水必须要趟，毕竟企服安全市场正处激变之中，稍有犹豫，便可能会陷入一步慢，步步慢的窘境。</span></p>
<section data-mpa-template="t">
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong>卸甲归田做销售</strong></span></h2>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近几年的互联网语境中，周鸿祎不仅从主角退居配角，“树敌”言论似乎少了许多，身份亦由“红衣战神”转变至处处“碰瓷”的推销员。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近两年周鸿祎的露面，几乎都同网安的故事有关——各大行业活动、会议上，周鸿祎大谈安全风险与勒索攻击；注资哪吒时，周鸿祎向新能源赛道公开喊话，称车企所管控的云端将成为最大安全隐患；就连搜狐遭遇邮箱诈骗，周鸿祎也不甘寂寞插了一句嘴。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/7005843562eb4708768de8.53173251.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从PC到移动，从手机到汽车，“安全”永远是周鸿祎的大箩筐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，针对脑机接口，也有安全第一的“宏论”，亮相之频繁，不禁让人怀疑，360最后的营销员或许只剩周鸿祎了。可此一时彼一时，当年拿着安全确实能唬住一帮人，但是在脑机接口面前，用户可能更唾弃肆无忌惮的免费广告。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而周鸿祎此番行为背后的逻辑，除身为配角不甘被遗忘之外，更多基于两方面因素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，其频繁贩卖焦虑的企服安全赛道，是当下360唯一的增量所在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从360去年财报来看，四大主营业务中，过去叱咤风云的互联网广告及服务、智能硬件业务、互联网增值服务营收均在收缩，仅有安全及其他业务营收实现同比增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，随着相关法律出台，网络安全赛道正在经历一场由“拼资源”到“拼技术”的变革，无论是政务场景还是企服市场，都正从过去的“应付逻辑”中跳脱出来，纷纷构建起网络安全的考核、回溯机制。换言之，入局较晚但技术尚可的360，仍存在着朝上发力的空间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近两年间，360在企服安全赛道内可谓动作频频，以其曾反复提及的“大安全体系”为例，“全行业共建安全体系”话术的背后，或多或少透着一股呼吁行业让出客户资源，一起分蛋糕的意味。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而业务层面，据周鸿祎透露，除提供为中小企业提供免费服务外，360亦拿出了将近100亿的服务和技术补贴客户，争抢客户资源的意欲明显。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由此不难看出，化身推销员的周鸿祎虽以配角自居，但其内心深处仍渴望着逆袭。其之所以会追逐新能源、元宇宙等新风口，或许也有着此番考量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，同样是追风口，当下周鸿祎却有着同过去截然不同的思路。按周鸿祎的话说，过去很多投资完全就是瞎投资的，而现阶段，无论是新能源也好元宇宙也罢，周鸿祎的出手不仅愈发谨慎，在宣传层面亦常常和“基本盘”安全业务挂钩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一件简单的例子，360注资哪吒汽车时，周鸿祎曾在微博写道：“我们希望将哪吒汽车作为智能网联汽车安全的试验田，未来把在哪吒汽车上研究成熟的安全技术输出给其他车企。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许在周鸿祎最初的构想里，注资哪吒既能以此为样本，将琢磨出的安全方案兜售给车圈，亦能再次搭乘风口，填补那些年在互联网赛道屡吞败局的遗憾。而对于元宇宙，周鸿祎亦在接受采访时将元宇宙定义为当创新战略，“为to B和to G的场景服务”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是，周鸿祎或许再一次错认了行业。以哪吒为例，现阶段新能源产业显然还未抵达谈论网络安全的阶段，而即便日后网安被提到主流讨论中，车企们亦没有理由选择360的解决方案，从而让竞争对手享有话语权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>何况，相比网络安全，自燃、断轴、失控，新造车们还在车规安全上爬坡，周鸿祎给的提前量过大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，哪吒仍未摆脱烧钱的窘境亦是不争的事实，尽管周鸿祎嘴上不承认，并摆出200亿现金予以印证。但360业务承压已不是一天两天，相比当年离开方正时“天生我才必有用，千金散尽还复来”的豪气，周鸿祎似乎更愿意把钱攥在手里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这并不意味着周鸿祎是当代葛朗台，老周的行径更像是在过于激进的投资策略招致失败后的反思。值得一提的是，周鸿祎曾在今年6月发微博表示，360将拿出200亿元来扶持100万家中小微企业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不惜为安全掏空老本，外加前述的“百亿补贴”，足以说明当下的周鸿祎已然知悉了钱要花在刀刃上的简单道理，毕竟现阶段安全才是360唯一的主航道。由此不难看出，把自己定义为“配角”的周鸿祎，在网络安全赛道上，或许还想要再当一次“主角”。</span></p>
