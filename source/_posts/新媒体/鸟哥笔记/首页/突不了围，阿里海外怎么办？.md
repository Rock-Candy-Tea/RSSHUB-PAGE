<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月4日，<a href="https://www.niaogebiji.com/tag/%E9%98%BF%E9%87%8C%E5%B7%B4%E5%B7%B4" target="_blank">阿里巴巴</a>（</span><span>BABA.US</span><span>）公布2023财年第一季度（</span><span>2022年4-6月</span><span>）业绩报告。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>财报显示，2022年4-6月，阿里收入为2055.55亿元（</span><span>人民币，以下未注明则同</span><span>），同比没有增长。其中，中国商业收入同比下降1%至1419.35亿元，云业务收入同比增长10%至176.85亿元，国际商业收入同比增长2%至154.51亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>利润方面下降幅度较大。财报显示，2022年4-6月，阿里净利润为202.98亿元，同比下降53%；归属于普通股股东的净利润为227.39亿元，同比下降50%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显而易见，<strong>主营业务国内商业收入减少，让阿里盈利空间承压。寻求新的增长极，已经迫在眉睫</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月26日，阿里董事会主席兼首席执行官张勇，在致股东信中重申阿里三大战略，即消费、云计算和全球化，“我们的全球化方向非常明确，正是致力于探索消费领域的全球化和云计算领域的全球化。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿里“全球化”这一战略，从2014年提出至今，未曾改变。目前，阿里13.1亿的全球用户数当中，9.03亿来自中国，3.05亿来自海外。为实现“到2036年服务全球20亿消费者”目标，要到海外找用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>不过，目前看来，阿里国际商业收入还未见增势。2022年4-6月，阿里国际商业收入占总收入比例为7%</span></strong><span>，而2022财年这一数据也是7%。对比而言，阿里云业务收入对总收入贡献从去年同期的8%微涨至9%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>速卖通卖家阿思告诉燃次元，在2020-2021年，他能月销10-20万美元，但今年月销售额下滑到6-7万美元，“受俄乌战争及全球经济形势的影响很大，欧洲还出台增值税的政策，速卖通的业绩是全面下滑的。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年7月，欧盟发布新政策，宣布取消对22欧元以下产品的进口增值税豁免，这直接影响速卖通的交易规模。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿里财报指出，“在6月份季度，来自Lazada、速卖通、Trendyol和Daraz的整体订单量同比下降4%，主要由于欧盟增值税税规修改、欧元对美元贬值，以及俄罗斯和乌克兰冲突导致供应链和物流中断的持续影响，令速卖通订单量减少。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在东南亚，自2019年以来，Lazada持续位于Shopee的压制之下。根据阿里财报，2022年4-6月，Lazada在东南亚的订单录得10%的同比健康增长。在2022财年，Lazada的已支付订单数量同比增长60%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反观Shopee，成绩明显更好。Shopee2022年第一季度业绩报告，总订单数同比增长71.3%。此外，该平台在全球购物类App中，总下载量位居第一；在谷歌应用商店全球购物类App中，用户使用总时长也位居第一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位接近阿里的行业人士告诉燃次元，今年以来，阿里对海外消费领域进行整合，未来跨境电商的大本营是速卖通，而本地消费平台是Lazada，“速卖通和淘宝天猫海外等阿里体系的跨境团队已经整合在一起，而本地化运营团队都在Lazada。”他指出，Trendyol和Daraz则仍由初始团队运营，阿里只作为投资方角色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“物流体系的建设则是全球化战略实施中的最高优先级。”他透露，<strong>目前阿里的海外商业战略中，搭建本地化的物流网络是重中之重</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>年初，原天猫、淘宝、阿里妈妈事业群总裁蒋凡接手“海外数字商业”，这一年也必将有所作为。但面对全球风云变幻，要争得一片天地，并非易事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/1455856016142b5abe0ca06.17059714.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><a href="https://www.niaogebiji.com/tag/%E6%B5%B7%E5%A4%96%E5%B8%82%E5%9C%BA" target="_blank">海外市场</a>遇变</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018年初，人在广州的小相，开始做东南亚跨境电商，起初开了Lazada店铺，半年之后才入驻Shopee。到2021年，公司店铺遍及东南亚六国，销售家居产品，年销量有200万元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年以来，小相的团队开始探索本地化运营，如今已经在马来西亚、泰国有自己的本土团队，接下来准备进入泰国市场。今年，他们还决定关闭Shopee的跨境店铺，只做本地店铺。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小相告诉燃次元，今年以来，Lazada出台了一些“计划”来扶持东南亚本土卖家。比如在马来西亚，如果店铺每个月交易额达到2000美元，可获得一些平台资源，比如专属小二对接、参加促销活动等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，小相今年面临考验。“今年经济形势不好，我们销量下降了大半，整体比前两年差很多。”实际上，在2020-2021年，疫情加速了东南亚电商发展，而小相趁此东风，每年销量可翻两三番。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主做俄罗斯、西班牙、法国市场的阿思，也有相似经历。2019年底，他放弃国内经营的淘宝、拼多多等电商平台，入驻速卖通进入跨境电商蓝海，销售运动及工具类产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“在我的经历里，速卖通最好做的时候也就是2019-2020年，而更早入行的朋友说，2018年也好做，但2021年开始就不太行了。”阿思告诉燃次元，欧洲收取增值税后，一向主打低价的中国跨境商家，在当地将不再具有太大的竞争优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年，针对优质商家，尤其是希望品牌化发展的优质商家，速卖通推出“AE Mall”。阿思介绍道，这是一个“金银牌”规则，比如符合月销量2万美元等一些条件，就能成为金牌卖家，在店铺会有“金牌”标志，有增加曝光的作用，“但目前，我还没感受到很大变化。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上述接近阿里的行业人士透露，速卖通4月19日推出AE Mall至今，已有超过5600家跨境企业参与报名，AE Mall已于7月5日在海外市场测试上线，目前还只面向部分用户进行测试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿里国际商业包括Lazada、速卖通、Trendyol和Daraz等国际零售商业业务，及阿里巴巴国际站等国际批发商业业务。今年以来，Trendyol和Daraz也有一些动作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从去年下半年开始，小相团队也接入了主打南亚市场的Daraz平台，做了巴基斯坦、孟加拉、斯里兰卡三个站点，“阿里现在也在推广这个平台，鼓励更多卖家入驻，打通一些同城物流体系。但这些国家比较穷，现在每个月三个站点的订单加起来只有20多单。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这些国家有四五亿人口，但买家可能只有几十万。而且都是COD订单（</span><span>Cash On Delivery，即货到付款</span><span>），当地不取货的比例能达到15-20%。”小相表示，目前他们团队也没有在这个平台投入太多资源。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据阿里财报，2022年4-6月，国际商业分部经调整EBITA为亏损15.67亿元，2021年同期为亏损10.3亿元，“主要由于Trendyol对包括国际业务及在土耳其的本地生活服务在内的新业务的投入而导致其亏损增加。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据阿里财报，2022年4-6月，Trendyol整体订单同比增长46%，通过其市场平台服务超过22.5万商家，“为了更好地服务主要的消费客群，Trendyol继续投入，并扩展到高频的本地生活服务业务。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上述接近阿里的行业人士指出，Trendyol如今在做一些外卖业务。但Trendyol面对的土耳其等市场状况不佳，去年下半年以来，土耳其通货膨胀率持续飙升，土耳其统计局8月3日公布的数据显示，土耳其通胀率7月升至79.6%，创下了24年新高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/9082723746142b5ac73c549.43567204.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>东南亚难守，欧洲难攻</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>深耕东南亚市场10年，Lazada面对一轮又一轮竞争，难以守住阵地</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“刚开始，Lazada还是东南亚电商老大，整体数据表现比较好，我们店铺也一样。但2019年之后，Shopee增长迅猛，我们Shopee店铺数据也开始高过Lazada。”小相指出，如今在东南亚，Shopee仍然占据最大的市场份额，而Lazada位列其次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据《晚点LatePost》报道，2017年，在Shopee攻入东南亚市场之前，因为阿里派驻人士和本地团队的矛盾等问题，Lazada的工作一度陷入了停顿，甚至连最重要的招商都暂停了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2018年，背靠上市公司Sea的Shopee开始开拓东南亚市场。Shopee就如同拼多多，砸钱买流量，大搞补贴。短短一年，2019年初，Shopee各项数据就超越了Lazada。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>马来西亚本土商家Abby告诉燃次元，疫情之前，马来西亚人很少使用平台电商，更喜欢社交媒体“微商”。而2019年之后，很多本地卖家开始入驻Shopee，这个平台也逐渐成为当地最受欢迎的购物平台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么卖家涌向Shopee？除了补贴，后台操作简单也是一大原因之一。Abby说，卖家入驻Shopee门槛极低，用买家的账号就可以直接开通店铺，并能直接上架商品，不需要审核。而Lazada比较复杂，注册需要个人真实身份证、银行卡信息，商品也需要审核后才能上架。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小相也指出，<strong>Shopee的操作适合新手卖家，而Lazada是将淘宝、速卖通等平台在全球验证过的功能搬过来直接用，新手卖家很难习惯</strong>，“Shopee的本土卖家数量超过Lazada，这可能是其中一个原因。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，疫情之后，电商刚刚起步的东南亚，无论卖家还是买家，经验远远不及国内。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Abby指出，Shopee价格战打得比拼多多还厉害，并且类似拼多多，平台使用方法也比较简单，“不需要领优惠券，摆明了价格给你算好了，直接拍就是最低价。但Lazada就比较难操作，跟淘宝一样，还要自己去领优惠券，有时候找不到领券的地方。”她在跟身边朋友聊天中发现，很多人不用Lazada，因为“玩不明白”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小相还认为，除了价格优势，Shopee如今建立起自己的基本盘，在服装、美妆、家居等传统热门类目上占有较大的市场份额，而受众群体多为女性，“而Lazada的优势在3C电子产品类目，竞争力差一些。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Shopee财报显示，2021年，Shopee的GAAP（</span><span>一般公认会计原则</span><span>）收入为51亿美元，同比增长136.4%；总订单数为61亿单，同比增长116.5%；GMV（</span><span>商品交易总额</span><span>）为625亿美元，同比增长76.8%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了Shopee，在东南亚电商市场中，Tiktok Shop（</span><span>小店</span><span>）已开放印尼、泰国、越南和马来西亚站点。基于本土团队，小相也开通了泰国和马来西亚的Tiktok Shop。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>东南亚市场强敌当前，欧洲市场也不乐观</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知情人士向燃次元透露，如今负责本土化运营的Lazada，今年也在尝试进入欧洲市场。但已经在欧美发力超过10年的速卖通，如今也面临诸多不确定因素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿思指出，俄乌战争之后，他的速卖通店铺在俄罗斯、乌克兰的销量就下滑了，而店铺来自俄罗斯的销售额占比可达70-80%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，俄罗斯、美国一直是速卖通出单量较高的两大国家，而在各种因素之下，这两个国家的销量不可避免地减少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，在欧美，亚马逊的独霸地位，也令速卖通难以出头。燃次元综合信息获悉，在美国、西班牙、法国、英国，亚马逊市场占有率均为第一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，在欧美市场，去中心化的独立网站也拥有较大市场份额。在美国，2021年Top电商零售商当中，亚马逊占约41%，Shopify占10%。以Shopify、Shein为代表的独立站，正在以其品牌调性，吸引一批欧美消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>杭州一位Shopify跨境商家表示，欧美人对独立站包容度极高，他们愿意花费更高的价钱去支持小众品牌，同时获得更好的品牌服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/4756644006142b5ad08cc02.64114870.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>海外机会，阿里如何把握？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>今年，阿里在海外商业的发力重点，正是物流网络建设</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小相指出，Lazada的海外仓，比Shopee做得更好一些。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小相介绍道，2020年，Lazada开通了海外仓，他们也是第一批入驻的商家，而在此之后Lazada店铺的数据也开始好转，“除了降低成本，Lazada还会借此做些活动，比如给产品打上‘承诺72小时到达’的标签，能发挥比较大的作用。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Shopee的海外仓则容易“出错”，比如经常发错货，订单量大的时候，则会发不出货或者很久才发货。相对而言，Lazada的物流系统更完善一些。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃次元获悉，2020年，Lazada通过阿波罗项目（</span><span>Project Apollo</span><span>）升级了物流技术体系，仓储作业系统全面改写升级。数据显示，Lazada超过82%的订单“首公里”物流都由自营物流团队和设施完成。通过海外仓，Lazada跨境物流时效几乎可以和在当地购买一样的2-3天到达，用户体验大幅提升。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年，Lazada上线了CPFR智能补货计划系统，这套系统会根据商家实际情况，多个维度数字智能指导商家如何补货。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在欧洲的速卖通，今年也在物流体系本地化上有所成果</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>燃次元获悉，目前速卖通已经在海外9个国家和地区，建设了28个海外仓，总面积超过86万平方米，同时在国内深圳、东莞、义乌、威海、许昌等跨境商家聚集地建设了10个优选仓。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过威海优选仓，跨境销售到韩国的商品能实现3-5日达，升级之后，速卖通上韩国销量从排名第17名上升到第7名，增速较快。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿思就表示，他会将一些爆款产品提前备货到俄罗斯、西班牙、法国的海外仓，将物流时效从半个月加快到7日达。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年3月，速卖通还在欧洲首次上线客户自提点（</span><span>柜</span><span>）服务，解决海外市场的“最后一公里”配送难题。在法国，速卖通目前有15980个自提点（</span><span>柜</span><span>），重点城市覆盖率达到了95%；在西班牙有11600个自提点（</span><span>柜</span><span>），重点城市的覆盖率达80%；在波兰也拥有16000个取货点，重点城市的覆盖率达90%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿里在本地物流网络的建设上初见成果，但到底能为其增长带来多少有效助益，尚需时间验证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>值得一提的是，Lazada和速卖通都在往品牌升级的方向调整</span></strong><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Lazada官网显示，2021年，80%+的福布斯Top100品牌已入驻LazMall品牌商城。不过，处于初期的东南亚电商市场，用户还未学会为品牌买单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>速卖通做品牌出海或许能提升竞争力。上述接近阿里的行业人士指出，AE Mall就是速卖通希望吸引拥有生产工厂的初创品牌，帮助他们成为跨境新品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从全球来看，中国的供应链仍然具有不可取代的优势，但在国际形势复杂的当下，跨境商家难以再依赖早期的价格优势占据市场，品牌升级、提高竞争力是必经之路。由此看来，阿里以速卖通为载体来做大跨境电商市场，尚有机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但以Lazada出发的本地化团队，能否应对海外的种种挑战，拿下发达国家的市场份额，仍然难以预料。</span></p>
