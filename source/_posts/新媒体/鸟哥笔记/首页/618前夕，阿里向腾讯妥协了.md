<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">时值<a href="https://www.niaogebiji.com/tag/618" target="_blank">618</a>，阿里的日子却不太好过。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">自今年3月起，阿里裁员的话题便不断登上社交媒体的热搜榜，引发网友舆论。而后，其2022全年财报也给投资人浇了一盆冷水，阿里的国内核心电商业务已开始出现负增长趋势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这个略显萧条的618大促里，阿里的流量焦虑愈发明显了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">正基于此，向来高傲的阿里不得不做出一个艰难的决定——向腾讯低头，以寻求站外流量增长。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">昨日，有知情人士称，就在618大促前夕，朋友圈广告已经可以直接通往商家的天猫旗舰店。这意味着，微信与淘系电商之间多年的“外链封杀”或将走到尽头。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据了解，此次双方的合作并非以往简单的“品牌广告”形式，而是当消费者点击广告后，可以直接唤起手机淘宝，进入商家店铺，以最大程度贴近交易。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/190316128762aa2ee91661d8.16887622.png" title="618前夕，阿里向腾讯妥协了" alt="618前夕，阿里向腾讯妥协了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">随着微信庞大的社交流量正式通过商业渠道面向阿里开放，淘系商家们也将距离“全域”获客的形式更进一步。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在目前“互联互通”的大趋势下，这无疑是件重大利好消息。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从封杀微信，到彼此拥抱，中国互联网走过了风起云涌的9年。回望2013，那是阿里与腾讯决裂的开始。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" title="618前夕，阿里向腾讯妥协了" alt="618前夕，阿里向腾讯妥协了" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">封杀微信，阿里与腾讯决裂</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">彼时，移动互联网的浪潮缓缓拉开序幕，手持“手淘”这一移动互联网船票，阿里迅速登上了时代发展的巨轮。淘系电商登顶江湖，睥睨天下。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而在入股新浪微博后，由于对导购网站的依赖度逐步降低，淘宝陆续封杀了返利网、美丽说和蘑菇街等导购网站。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">更有坊间传闻称，马云曾在内部下达两条指示：一、阿里不能继续扶持蘑菇街、美丽说壮大；二、应该多做异业合作，少做同业合作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很快，这股电商圈的“封杀之风”迅速蔓延到了其他领域。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/38589708662aa2ee958b491.53670717.jpeg" title="618前夕，阿里向腾讯妥协了" alt="618前夕，阿里向腾讯妥协了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当时的阿里，出于对庞大社交流量的觊觎，高调推出了社交软件“来往”与微信对抗。马云更是直接为来往站台，称关闭自己的微信，转战来往。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">激战正酣时，同年11月，一条公告如深水炸弹在中国互联网市场掀起惊天波涛。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">手机淘宝以保障用户安全为由，宣布正式关闭微信通道。至此，用户再也无法通过微信中发布的淘宝商品或店铺链接进行购物。阿里与腾讯这两大巨头，就此割裂。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于何时重新打开微信通道的问题，淘宝方面称“取决于微信什么时候能够补上自己的安全漏洞，让用户的利益不再受损”。不仅如此，其还坚决表示，“微信一天不安全，我们一天不开放，我们只对安全开放”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而在封杀微信后，阿里系其他产品也纷纷与微信划清界限。如虾米音乐、新浪微博手机端等，原有的“分享到微信”功能均被取消，转为分享到阿里旗下的社交软件“来往”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/208595900262aa2ee98dbc04.42003739.jpeg" title="618前夕，阿里向腾讯妥协了" alt="618前夕，阿里向腾讯妥协了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">与此同时，腾讯也做出激烈反击。先是将导向淘系网站的流量渠道全部关闭，并向用户提示“<a href="https://www.niaogebiji.com/tag/%E9%98%BF%E9%87%8C%E5%B7%B4%E5%B7%B4" target="_blank">阿里巴巴</a>屏蔽了来自微信的浏览请求”；后又围绕公众账号拉拢众多合作伙伴围攻阿里，各电商企业及线下商户更是其重点拉拢对象。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">至此，双方的封禁大战开始，腾讯与阿里“老死不相往来”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" title="618前夕，阿里向腾讯妥协了" alt="618前夕，阿里向腾讯妥协了" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">阿里的流量焦虑</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一晃眼，9年过去了，微信在社交领域的霸主地位依旧无可撼动，而电商市场早已不是阿里自家的后花园。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">无论是自五环外一路杀出的拼多多，还是借短视频东风进军电商领域的抖音、快手，纷纷厉兵秣马，时刻准备着对淘系电商发起猛攻。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此情形下，阿里的流量焦虑愈发明显。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2022财年，在阿里的大力推动下，其国际电商业务取得了46%的高速增长。然而即便如此，其整体GMV才仅仅由8.119万亿微增至8.317万亿元，增长率不及2.5%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这意味着，阿里的国内核心电商业务极有可能已经出现负增长！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而抖音、快手的迅速崛起，也在与老牌电商巨头们争抢着用户的时间和金钱。用户的时间、金钱总共只有那么多，越多的竞争者入场，阿里就越“焦虑”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在阿里、京东双寡头时代逐步演变为“猫狗拼快抖”五强争霸格局的如今，如何经营好并稳定住存量用户，是阿里急需解决的问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">时值618大促，阿里的流量焦虑被极致放大。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除老牌电商三巨头外，不仅抖音、快手纷纷布局此大促节点，就连微信视频号都选择予以商家扶持，试图分一杯羹。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一位商家表示：“其实日常的时候，淘系站内流量主要还是靠搜索获得的，不过大促期间站外引流的需求非常大，前期不种草没爆品，后期不引流没交易。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">显然，阿里的流量焦虑在此刻急需一个突破口来缓解。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而恰好，阿里的病，或许腾讯能治。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" title="618前夕，阿里向腾讯妥协了" alt="618前夕，阿里向腾讯妥协了" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">合则两利，天猫拥抱微信</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">天下没有永恒的敌人，只有永恒的利益。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">自2013年至今，阿里、腾讯掐架无数。时隔9年之后，双方终于握手言和，原因无外乎“利益”二字。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2381981962aa2eea39b1e2.66284191.jpeg" title="618前夕，阿里向腾讯妥协了" alt="618前夕，阿里向腾讯妥协了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年微信之夜上，张小龙曾披露数据：每天有10.9亿人打开微信，7.8亿人进入朋友圈。不仅如此，作为一款国民社交应用，微信在人的样本量大的基础上，所拥有的的数据维度也足够多。这一点，即便是如今的抖音数据也难以企及。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">显然，只要利益分配合理，天猫与微信的合作是“合则两利”的事情。一方面，淘系电商收获了一个巨大的社交流量入口，足以缓解其流量焦虑的阵痛。另一方面，腾讯也将进一步完善其微信生态，甚至于在长远上，有着扼住阿里咽喉的战略意义。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">更何况，类似的合作经历，阿里早已有之。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">近年来，随着短视频平台的兴起，占据了互联网用户的绝对时长，也成为种草、导购的全新增量风口。由此，短短时间内便催生出了百亿引流电商市场。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/84551731262aa2eea72beb9.32195583.jpeg" title="618前夕，阿里向腾讯妥协了" alt="618前夕，阿里向腾讯妥协了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">顾名思义，引流电商便是商家通过在抖音等平台投放广告，用户点击广告后可跳转至第三方平台详情页，从而完成交易的模式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很早之前，阿里便与字节系合作，大大提升了广告投放效率和人群沉淀能力。未来，双方的合作还将继续，甚至有更加激进的目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如今，天猫入驻朋友圈的广告亦是采用同样的形式。尽管目前，个人将淘宝天猫店铺链接发在朋友圈，仍不能直接跳转，但相信这一天的到来不会遥远。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">随着淘系电商与微信的破冰，互联互通已经成为时代的主题。腾讯与阿里这两大互联网巨头，未来又将勾起怎样的火花，我们拭目以待。</p>
