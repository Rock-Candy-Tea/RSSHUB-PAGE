<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">1 、看是否有从管理自我到管理他人的能力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在做底层员工的时候，只要自己能力强，完成自己岗位上的任务就行了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是一旦当上管理者，开始带团队，需要的能力完全不一样。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这是一个从自己完成工作，到通过他人完成工作的转变。现实情况是，很多职场上能干的人，都被这一转变难住了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为他们总忍不住凡事亲力亲为，总觉得“你不行，你做得不够好，闪开看我的。”他们没法做到信任、培养和成就他人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">管理的本质就是用人，管理的成功不在于怎样把一件事情做好，或者怎样赚钱，而在于怎样用人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">卡内基曾说过：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-type="2" data-content-utf8-length="59"> </p>
<section>
<section>
<p>“即使将我所有工厂、设备、市场和资金全部夺去，但只要保留我的技术人员和组织人员，4年之后，我将仍然是‘钢铁大王’。”</p>
</section>
</section>
<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">卡内基之所以如此自信，就是因为他深知如何管理他人，如何善用那些比他更强的人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，你必须学会管理他人，利用别人的优势，达成你的目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/109058485062b3697f32cc39.12203855.jpeg" title="如何识别一个人是否有管理的潜力？看这6点就够了" alt="如何识别一个人是否有管理的潜力？看这6点就够了" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，你需要给同事更多的信任，花更多的时间帮助、鼓励他们，激发他们的潜力和长处，让他们能够独当一面。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">想成为一名优秀的管理者，除了管理好自我，更需要掌握管理他人的能力；除了具备“猎人”式的制胜策略，更需要“农夫”般的耐心耕耘。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">2 、看是否有强烈的目标导向和结果导向的能力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">结果导向就是以结果为标准，为达到目标，不畏困难、没有理由和借口，倾尽所有的资源和人力去达成最终的结果。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其实，现实中很多管理者，甚至老板，注重的却是加班、是工作量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">导致很多人做着无效的努力，还自我感动，而不自知。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是，作为管理者，需要的是工作量吗？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然不是。我们要的是结果，是产出。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以作为一名管理者，要有强烈的目标导向和结果导向思维。在同事出现偏离目标、或被其他一些新鲜事物吸引时，甚至遇到瓶颈困难时，要坚定不移、排除万难的朝着目标前行。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有人统计过，从二次世界大战至今，西点军校培养出来的董事长有1000多名，还有不计其数的高级管理人才。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这是一个很“神奇”的军校。因此有人特意研究了西点军校，以及那些优秀的管理者，最后得出了6个字——没有任何借口。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">原来，西点军校有一个最重要的行为准则，就是“没有任何借口”。给你一个任务，不管多艰难、多困难重重，也要想尽办法去完成，不能有任何借口。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">寻找任何借口、最合理的借口，都会视为推卸责任和为自己开脱。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">唯有达成目标，才是最终的任务结果。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5376359362b3697f7d1e65.50091824.jpeg" title="如何识别一个人是否有管理的潜力？看这6点就够了" alt="如何识别一个人是否有管理的潜力？看这6点就够了" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">3 、看是否有营造氛围建立团队精神的能力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">营造团队氛围的核心原理，在于调动员工的工作意愿，和激发他们的工作热情。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而只有赢得下属的信赖和忠心，才能营造出属于你们团队的独特氛围，和团队精神。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">电视剧《亮剑》中有一个情节，让我印象很深：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-type="2" data-content-utf8-length="145"> </p>
<section>
<section>
<p>在一次突围战中，李云龙独立团陷入敌人的包围圈，在寡不敌众的情况下，李云龙带领独立团突围成功。</p>
<p>但在清点人数时，发现一名营长张大彪由于负伤没有冲出来。</p>
<p>此时，李云龙对大家说：“我们独立团从成立到现在，还没有落下过一个兄弟，跟我回去救张大彪！”最后的结果是，张大彪被救回来了，但牺牲了另外七八个人。</p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">看到这儿，很多人觉得李云龙太傻，这次营救根本不划算。不仅独立团人员减少、斗力减弱了，而且救回来的是一名没有战斗力的伤员，还成为累赘。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但从管理者的角度看并非如此，领导者讲究的是氛围。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通过这次营救，李云龙强化了“独立团从来不落下一个兄弟”、“生死与共”的团队精神，将形式上分散的独立团拧成了一个有力的拳头。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一名优秀的管理者不仅因为头衔，更来自员工的认可。带人要带心，真正关心公司、关心员工的人，才能获得职级之外的拥戴，才能营造出有力的团队氛围。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/111734986462b3697fba5042.87283326.jpeg" title="如何识别一个人是否有管理的潜力？看这6点就够了" alt="如何识别一个人是否有管理的潜力？看这6点就够了" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">4 、看是否有卓越的洞察力和预测结果的能力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所谓洞察力，就是一眼能够抓住事物的关键和核心，发现事物的本质。从而把握事物的运行规律，做出准确的预测结果，最后做出决策以达到目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">相传战国时期有一名神箭手叫更羸。有一天他跟魏王打赌，说自己可以不用箭就能把天上的大雁射下来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">魏王不信，于是更羸就摆出拉弓的姿势，对着天上的大雁做出射箭的动作。结果，大雁在空中挣扎了一会儿之后便一头栽到了地上。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">魏王很惊讶，更羸解释说：“根据我的观察，这只大雁飞得很慢，应该有旧伤；叫声很凄惨，说明长期离群，旧伤未愈，恐惧心理没有消除。因此，它一听到弓弦声就想慌忙逃跑，结果牵引伤口，掉了下来。”这便是“惊弓之鸟”的故事。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">更羸虚箭射鸟的成功，不是靠魔术，也不是靠运气，而是靠着自己卓越的洞察力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而对于一个管理者来说，具备优秀的洞察力则是迈向成功的有力保障。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在同一家公司，你会发现，有些团队运转节奏快、效率高，而有的团队则完全相反。这其中很大原因就是，两个团队的领导人在洞察力、预测结果的能力上有巨大的差异。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">成功的管理者，就是一个团队甚至一个企业的核心脊梁，担负着把握企业发展方向和预测企业发展未来的重任，面对多变的市场和信息，领导人必须培养自己的洞察力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">洞察力的培养需要知识、经验的不断积累，时刻保持清醒的头脑和判断，以及养成独立深度思考的习惯。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/197669986562b3698011aeb8.59765174.jpeg" title="如何识别一个人是否有管理的潜力？看这6点就够了" alt="如何识别一个人是否有管理的潜力？看这6点就够了" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">5 、看是否有善于沟通了解对方真正想法的能力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">傅盛说，工作中80%的问题都是沟通造成的。绝大部分工作问题，不来自于技能本身，而来自于沟通。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">全球闻名的沃尔玛公司缔造者，萨姆·沃尔认为：沟通是管理的浓缩。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">沟通不仅在过去、现在还是未来，都是十分重要的一项能力。对于一名管理者来说，要想管理到位，必须沟通先到位。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">沟通的本质是信息的传达和接收。如果信息的传达有误或者“不真实”，造成的损失不止是双方的时间成本。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，一个有效的沟通前提是坦诚，不作暗示，不饶弯子。这样才能获得对方真实的想法。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">谷歌有一种独特的开会方式。就是在每周全员会议开始前，以“周末做了什么”环节为开场。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">两位创始人拉里·佩奇和谢尔盖·布林也会参与。他们会聊到自己周末风筝冲浪的逸事、极限健身的近况等。经常能引起同事们的浓厚兴趣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种沟通策略，目的在于消除紧张的职场氛围，亲近同事、增进同事情谊，由此打开对方的内心，了解你身边同事的真实想法。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一名优秀的管理者最基本的能力，就是做到有效沟通。而有效沟通最快的方式，就是坦诚，并朝着共同目标达成一致。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/422903262b369806fe709.49602180.jpeg" title="如何识别一个人是否有管理的潜力？看这6点就够了" alt="如何识别一个人是否有管理的潜力？看这6点就够了" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">6 、看是否有整体观、大格局的能力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一个管理者的核心不是所谓的高情商，而是在大格局下对整个行业有深入、清晰的认知，用大趋势做出正确的判断和决策。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">具有大格局的人，眼界广、知识全面，且有出众的认知体系。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果一个管理者的认知不如员工，是无法带领整个团队走下去的。只有管理者的认知体系覆盖所有人，才值得成为一个领导。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">清华大学的宁向东老师曾说：管理学的目的，不是管理一家公司，管理学的目的是让你有能力“破局而出”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">什么意思？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很多人在一个岗位工作久了，都有一个自然倾向，就是形成既定之局——相同的工作反复重复，尽管做得很认真很辛苦，但成绩不会上升甚至反降了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这个局里面，需要找到新的解决方法，用大格局和整体观来破解。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而这些都需要管理者来引领。通过强大的认知体系，让员工重新找到新的工作之法，让整个团队的“既定之局”发生改变。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要培养大格局，就要摆脱眼前的小格局，不要在一个小领域钻牛角尖，多看看其他领域的市场和社会发展趋势，去看到更大格局里的走向。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所谓“欲穷千里目，更上一层楼。”不是指登高才能望远，而是登高才能看到真相和全局。</p>
