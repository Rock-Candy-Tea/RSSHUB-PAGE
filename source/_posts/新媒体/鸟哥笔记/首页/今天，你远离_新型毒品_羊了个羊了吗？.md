<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前几日，新上线了一款小游戏，叫“羊了个羊”，爆上热搜榜单第一，收获亿级讨论热度，最根本是因为这款游戏足够折磨人且易上瘾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/20069445916321d5680119f3.90214324.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/17650967056321d568576f77.50859815.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>怎么个折磨人法呢？</strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>首先简单介绍一下这款小游戏，</span><span>羊了个羊</span><span>作为一款卡通背景消除闯关游戏，游戏中是可利用各种道具和提示来消除每一个关卡当中的障碍和陷阱。<span>但与普通消消乐不同的是，</span></span><span>第一关与第二关的闯关难度简直是天差地别，能闯过第二关的人极少，虽然操作简单可极其容易让人崩溃<span>。</span></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一关：1+1难度</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/10657055686321d568897a45.34321171.jpeg" width="333" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二关：洛必达法则难度</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2552910396321d568ce9fd1.85695989.jpeg" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/19757510076321d569244a74.62000798.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这第二关的难度就是，你永远不知道下一个图案下面的是什么，消除之后有可能是一座城堡，毫无规律运气成分大。因此，</span><span>就因为这0.01%的通关几率，才吸引了很多人去玩，也导致了这款小游戏连续遭遇三次服务器瘫痪</span><span>（小声一句：活该！）</span><span>。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/7435068786321d5696cbe51.50496695.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/322687246321d569afb016.32024210.jpeg" width="333" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也有一些理性的网友表示，这款游戏纯粹就是在卡bug，目的就是借着“谁都通不了关”的操作来博眼球，说到底只是一场<a href="https://www.niaogebiji.com/tag/%E5%99%B1%E5%A4%B4%E8%90%A5%E9%94%80" target="_blank">噱头营销</a>罢了。毕竟有部分网友因为其中的广告确实下载了其他小说、游戏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/9480416646321d569e7aab9.47816313.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/14631995186321d56a19efb3.16969672.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/7008664116321d56a3c7f87.08993108.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，</span><span>羊了个羊还很像另一款名叫“3 Tiles”的app端游戏，除了界面风格设计，游戏的玩法非常相似，被认为抄袭无疑了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2822052656321d56a8c46a6.63397304.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>但因为消消乐、连连看这种游戏版本实在太多，网友对此似乎并不care，主要纠结的还是如何才能通过第二关</span><span>（上瘾患者）</span><span>。——</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不知道各位还记得上一次玩的小游戏，类似于跳一跳、合成大西瓜、旅行青蛙等等，现在就轮到这个叫羊了个羊了。近些年来多款小游戏都经历了一时的风靡，但最后却不了了之，也不知道这款小游戏能持续多久。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><br>羊了个羊的火爆，其实说到底还是采用了饥饿营销的手法，每天出一局，只有少数人能拼运气通过，才得以抓取到用户的好奇心，刺激到用户的挑战心理，让用户上瘾。<br></span></p>
