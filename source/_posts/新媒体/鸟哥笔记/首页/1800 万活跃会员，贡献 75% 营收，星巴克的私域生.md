                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2022第一季度最新财报数据显示，星巴克中国市场的销售额里 75% 是由会员所贡献的，而星巴克星享俱乐部活跃会员数，大约为1800万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">做私域，拼的是组织力，特别是像星巴克这样大型线下连锁零售业态，如何协调总部与大区、大区与门店、门店与伙伴之间的关系，使项目在执行的过程中，保持品牌调性，是个很关键的痛点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本文深度拆解星巴克的私域体系，将从总部与伙伴协同、有温度的服务、品牌 IP、会员体系等方面展开。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;">总部伙伴协同，回归有温度的服务</h2>
<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我一直认为，私域的布局要因地制宜，要基于品牌战略、调性、产品、组织、运营来搭建私域体系，没有一招鲜，能吃遍天的情况。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>首先，我们来看星巴克的品牌战略、调性、用户群。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">星巴克的成功，并不在于咖啡本身，而是创造了「第三空间」的理念，从单纯售卖产品的咖啡馆，转型成了兜售生活方式的文化符号，打造自己的社区文化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>第三空间：</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">家庭为第一空间，职场为第二空间，城市内的酒吧、咖啡店、博物馆、图书馆、公园等公共空间为第三空间。为了打造这个第三空间，星巴克的店铺设计、布置、装饰的每一个细节都被充分考虑，其中包括馥郁的咖啡香气、舒缓的音乐、柔和的灯光、舒服的沙发等氛围。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">你可以理解为，人们买星巴克，更多的是在为空间环境，一个可以放松、休息、洽谈、沟通的场所而买单。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">星巴克针对的用户群体是：受过一定教育、有社会地位和经济基础的富有小资情调的白领。他们对价格敏感程度低，更注重服务、理念、品质、调性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">基于品牌定位、调性，星巴克的私域体系，我总结为有温度，不打扰：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在拉新上，对一线员工不设 KPI ，不做强硬动作，门店也不做过多透出，只是在收银台边放了一张入群台卡。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以社区（门店）为单位，打造熟客群概念，吸引感兴趣的忠实用户入群。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在运营上，让真人在线社群。通过人与人之间的沟通、链接、互动，产生关系，感到温度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用户体量少的品牌，可由总部运营直接在群与用户互动。但对于星巴克这种千万级的用户量，企微群内互动由总部运营执行肯定不现实；由门店伙伴建群，总部的管控力就会变弱，因此就需要做职能上的区分。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总部基于总部层面做统一标准动作，门店伙伴基于门店层面做非标温度服务：</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总部，以品牌 IP 建群；</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在企微群内，面向用户，执行总部层面的统一标准、具有创造性的工作—品牌活动、品牌故事、宠粉福利、内容种草；</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在组织内，面向员工，能监督一线门店、伙伴的服务质量、响应时长。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">咖啡师/门店视角，有温度不打扰的服务</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌通过私域能直面用户，传递品牌价值调性，提供传统渠道无法提供的服务，发展用户成为品牌的 KOC ，超级传播者。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">星巴克的每个门店熟客群内，都有 10 位左右的咖啡师在线，做有温度的服务。从问题解决、情感互动、门店活动、线下邀约至成为朋友。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">「社群」广义指一个群体组织，企微只是这个「社群」的载体。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">想要「社群」有持续的生命力，光靠品牌在线上企微群单向输出远远不够，必须得去中心化，由社群内的成员构建起一张社交关系网络，品牌方只是作为其中一个节点存在，更多的是让用户与用户产生联结。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">发展「社群」关系最快的手段，就是线下见面，共同去做某件具有意义、具有相同目标的事情。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">星巴克的「社群」，是将一群有着共同爱好、共同需求、共同情感的用户聚集在了一起。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">他们因星巴克相识、相知、相熟，自然会对星巴克品牌产生归属感、粘性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，星巴克是如何做的呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">星巴克的企微群以门店为单位建立，门店又都是开在小区、写字楼附近。企微群内用户所在的地理位置相对一致，都是在门店附近 3 公里。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">基于星巴克「第三空间」门店氛围，咖啡师可在企微群内邀请用户到线下来参加咖啡品鉴会，增进用户与用户、用户与咖啡师之间的联结。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/3706780a134d8e93c1e97927a696c5d5.png" alt referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">咖啡师在上班空余时间，也会在群内以朋友、挚友视角，与大家交流、互动，玩在一起，甚至彼此扮演着多重身份角色。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如：xxx店吉祥物、xxx店签到王。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/a67bb883b40f5d76c9051e52c8f023f8.png" alt referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当「群体」内的成员，基于星巴克彼此线下认识，产生过链接，又都有着共同的需求，那么自然会衍生出其他生活场景。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，上海疫情期间，大家在群内询问疫情、团购、发货等事宜，互帮互助。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，看到走失的狗，在群内发布失物招领；交流工作、搬家等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/d68b82874293da78109b58c6e4316f2c.png" alt referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在门店营运的过程中，经常会遇到特殊情况。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，大门玻璃破了；疫情防控侧门没开；咖啡原料没了；延迟开门；又或者是，门店内杯子到货、上新、产品清库存……</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些信息很重要，但总部的运营是无法得知、掌握的。这时，就需要门店的咖啡师在群内做提醒，告知用户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为用户，有关订单、门店的问题可在群内提问，咖啡师会在第一时间解决。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/ebbee974bf8713c97ee3897170a9c261.png" alt referrerpolicy="no-referrer"></p>
<p>有温度的服务，不仅仅体现在门店伙伴上；总部运营的品牌IP，从品牌视角给予用户温度，情感。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;">品牌IP/总部视角，有温度不打扰的服务</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在私域中，基于与用户0距离特点，企业可以将冷冰冰、看不到、摸不着的品牌，化为有温度，有感情的IP。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通过与用户的互动，传递品牌定位、调性、格调、价值观。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">IP化作「人」，用「人」去接近用户，服务用户，更易被接受，也更易产生好感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌IP的形象设计，一般为虚拟形象。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用户与企业线上、线下接触的多维度触点、宣传素材、产品包装都可与品牌IP形象结合。用户一见到这个IP形象，就会想到品牌背后所传递的理念。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，星巴克的「熊店长」。由总部运营，从品牌视角，在全国范围执行总部层面的统一标准、具有创造性的工作，给予用户温度，情感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，品牌活动、品牌故事、宠粉福利、内容种草、情感关怀、咖啡教学、周边物衍生。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在企微群内，作为一个活生生的「人」，说着「人话」，与用户沟通、交流、互动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">新用户入群时，「熊店长」做欢迎仪式；每天咖啡消费高峰时间段，通过场景化种草，发放宠粉福利（免邮券、满50-6），提升客单价、促进消费。</p>
<p><img src="https://qnssl.niaogebiji.com/d68d823be028b13d543a49a99b69a57d.png" alt referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有温度的服务，一直是星巴克推崇的，也是我认为做私域最核心的竞争力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">SOP 能提升运营效率，但在标准化的流程之上，必须得多一点人文。</p>

<section>
<p><strong>什么是 SOP ：</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">SOP，即标准作业程序，指将某一事件的标准操作步骤和要求以统一的格式描述出来，用于指导和规范日常的工作。</p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要做到很难，需要品牌有足够的认知，还要有足够多的人力去执行，更要执行的人有一颗敏锐的洞察力，能从用户视角对待每一个用户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当一个城市因疫情大面积关店，熊店长会特地制作具有当地特色的充满鼓励、希望的素材，在群内进行关怀。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，上海疫情期间，星巴克制作的阿拉一道加油；等解封跟侬一道切咖啡；我们一起等春暖花开……</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时特地开设咖啡课堂，教大家在家如何做咖啡。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">针对部分有运力的门店，熊店长还开设了团购拼团渠道。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一个IP想丰满、立体，要有丰富的故事线。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">星巴克给「熊店长」设立了完整的人物故事，他也会经常作为故事主人公，配合着营销活动出现在用户面前，与用户玩在一起，拉近品牌与用户之间的距离。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">成功的IP，还需要延展出一系列的周边产品，以实物的形式陪伴在用户身边。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">熊店长的IP周边产品有摆件、玩偶、勋章等等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">只送不卖，严格控制产出，打造稀缺性。用户必须在规定时间内消费集齐星才可获得，增加对熊店长、品牌的归属感。</p>

<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">会员体系的建立，能帮助品牌筛选认可品牌、认可价值、忠诚度高的用户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌将有限的资源，重点投入到这些用户身上，实现资源利用效率的最大化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通过会员权益的激励、荣誉感的打造，能吸引普通用户往核心会员发展，促进消费；同时增进核心会员对品牌的粘性、忠诚度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">星巴克的会员体系，以消费集星为成长主线，通过消费积分划分成 3 个等级。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以此来判定用户是否是品牌的忠实用户，并决定投入相对应的资源，分别匹配对应的权益。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>星巴克的会员等级分为3级：</strong></p>

<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>银星级</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">银星级为体系内的最低等级，也是默认的等级。没有任何权益。通过权益上的区分，尽快让用户晋升至下一等级。</p>
</section>
<section>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">达到玉星级，我认为才算是正式成为星巴克会员。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">玉星级的晋升条件有2类：用户从银星级消费满4颗星（一颗星50元）或者花钱购买星享卡。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当用户处于银星级时，星巴克的伙伴会推荐星享卡，让其快速晋升，权益以老带新、刺激关联消费、促进复购为主。</p>

<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，用户购买星享卡后能直接晋升至玉星级，获得晋升奖励—免费饮品券。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用户先是以8颗星为目标，达到8颗星享受免费喝一杯后，他会再以8颗星为目标，达到最高等级—金星级，享受更高的权益。</p>
