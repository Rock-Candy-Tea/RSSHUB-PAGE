<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><span>没有事前预热，也没有事后热搜，消失了100多天的<a href="https://www.niaogebiji.com/tag/%E6%9D%8E%E4%BD%B3%E7%90%A6" target="_blank">李佳琦</a>在9月20日悄然复播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一次回归，很多事情都没有变：依旧断层领先的观看量和在线人数，彰显了这位“直播一哥”的人气和地位。但涌入直播间的粉丝们，也捕捉到了一些明显的变化：比如更简约的直播间布置，更克制的直播风格和更亲民化的选品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，在李佳琦离开直播间的这些天，淘宝乃至整个直播电商行业都发生了一些深刻的变化。东方甄选崛起，成为新一代顶流；快手和抖音相互呛声，失去了“一哥”的淘宝处处受制。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今李佳琦回归，等待他的将是从未见过的全新考验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/469351945632d42230f0939.42444393.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/185981477061c95ab8f16f14.86688433.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>时隔三个月再开播，李佳琦的江山还稳不稳？</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>从各个维度的数据来看，李佳琦这次复播可以说是大获成功。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7点开播到9点15分左右提前下播，不算中间因为服务器故障而临时下线的几分钟，李佳琦复出首秀只持续了两个小时左右，比起之前动辄7个小时的马拉松式直播短了不少。但从观看人次、成交额和社交媒体讨论度等多个维度看，数据都相当出色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>官方数据显示，开播10分钟后直播间场观突破500万，到下播前峰值突破6000万，不仅完全秒杀同时段开播的其他二线主播，也超过了李佳琦在6月份停止直播前的场均数据。从成交量来看，截止下播时直播间共上架了26款产品，其中19款售罄，野性消费再现江湖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看到这样的局面，不少网友认为淘宝一定是最开心的，网友在社交平台上调侃“戴珊现在应该躲在屏幕前偷笑”。对于这种观点，价值研究所认为不无道理，但并非完全正确——</span><span><strong>李佳琦的回归的确一解淘宝燃眉之急，但后者依然有自己的隐忧。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，在薇娅、李佳琦和雪梨接连隐退的这段日子里，淘宝直播电商流量下滑严重，尽显颓势。</span><span><strong>而李佳琦的回归，则承载着盘活内部流量池、提升外部热度的双重使命。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>横向对比就能发现，在李佳琦直播间的流量虹吸效应下，淘宝其他主播都黯然失色。数据显示，9月20日晚间8点李佳琦直播间场观数据突破2000万时，排第二的蜜蜂惊喜社仅为268万，第三名的香菇来了直播间为260万。将近十倍的差距，印证了李佳琦的强大号召力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，蜜蜂惊喜社和香菇来了分别脱胎自薇娅和雪梨的助播团队，幕后的MCN机构也都和这两位昔日的头部主播有千丝万缕的关系。比起只能“曲线复出”的薇娅和雪梨，重回直播间的李佳琦更显难能可贵。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但另一方面，在李佳琦消失的这段日子里，淘宝电商生态已经发生了许多变化。其中最大的转变，在于向内容化转型和加强公域导流，意图摆脱对头部大主播的高度依赖，扶持更多中腰部主播及商家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>头部主播压榨中腰部主播和中小商家成长空间的问题，在李佳琦、薇娅停播之前就被多次讨论，只不过一直没有找到合适的处理方法。超级主播带来的流量和话题度，淘宝不可能舍弃；但想在不影响头部主播的前提下呵护腰部主播的成长，也异常艰难。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过往，淘宝一直没有办法找到平衡点，直到几个头部主播先后停播才下定决心进行改革。</span><span><strong>这时候，李佳琦回归反倒有可能打乱淘宝直播电商的转型步伐。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月1日举行的“淘宝2022直播盛典”上，阿里宣布淘宝直播进入2.0“新内容时代”，并推出了多个改革计划。其中，诸如新领航计划、引光者联盟、超级新咖计划和源力计划等活动，都旨在扶持新主播，引导淘系直播电商生态从私域向公域转型。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿里为这轮战略变革制定了多个目标，其中包括打造2000个成交额翻倍的账号，500个年成交额超1000万的账号，以及100个粉丝过百万的内容账号。而所有目标，都指向同一个核心战略：壮大腰部主播团队。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿里直播事业部负责人道放甚至直言，淘宝直播内部生态最重要的是保持活力，这就意味着不能让一两个超级主播一直牢牢控制流量。</span></p>
<p><span>“如果一个头部主播的份额过高、流量过大，风险就会上升。我们不排斥超级头部，但生态的活力也很重要。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今李佳琦回归，昔日的难题重新摆在了淘宝面前。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个局该怎么破？淘宝的选择，很大程度上也要看其他竞争对手，以及整个市场的反应。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/185981477061c95ab8f16f14.86688433.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>对手怎么看？快手比抖音更慌</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先可以肯定的是，在李佳琦离开直播间这些日子里，淘宝直播，乃至整个阿里系电商王国都变得低调了许多。今年618期间，阿里甚至罕见地没有发布公开战报。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反倒是抖音和快手这两个竞争对手，都趁机发力，试图分割阿里的江山。面对李佳琦的复出，抖快双雄心里又会有哪些盘算？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>抖音这边，似乎不会太紧张。</span></strong></span><span>毕竟东方甄选、小杨哥等一批新顶流的崛起，正帮助抖音抹平和淘宝、快手之间的差距。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月19日，也就是在李佳琦复出的前一天，东方甄选抖音直播间公布了一份最新成绩单：单日直播销售额1395.5万，刷新近期新高。根据新东方在线本月初发布的财报，东方甄选近三个月GMV累计突破20亿元，几乎是抖音另一头部MCN交个朋友的2倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>距离爆红过去了几个月，东方甄选也用稳定的表现打破了抖音顶流“昙花一现”的魔咒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，6-8月东方甄选直播间销售额分别录得7.08亿、6.22亿和6亿元，确实存在下滑趋势。但需要注意的是，这三个月的转化率从1.36%提升至1.92%，UV价值（即访客平均价值）也从0.93元上涨至1.19元。这个数据，表明东方甄选供应链管理能力和用户黏性的提高，拥有长期增长的潜力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/318093488632d42237869e0.36403702.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，抖音电商如今多了另一个重要工作：补强售后、物流等环节，修炼“内功”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月19日，京东物流宣布正式接入抖音电商的“音需达”服务体系，将为后者提供高品质快递服务。自从8月份上线以来，“音需达”已经先后和中通、圆通、韵达、顺丰等物流企业达成合作，如今再引入京东物流，配送能力将进一步增强。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在价值研究所看来，抖音电商现在和阿里形成了较为明显的错位竞争，双方的注意力分别放在供应链物流和流量获取上，正面交锋的机会正在减少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李佳琦这种头部主播的复出，当然会让直播电商生态发生一些变化。</span><span><strong>但别忘了，抖音还有另一个优势：不缺流量。</strong></span><span>这一点李佳琦、薇娅、雪梨等大主播还在的时候就是如此，现在依然没变，这也就意味着李佳琦直播间的流量虹吸效应对抖音电商影响相对有限。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>至于快手，压力可能就要更大了。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>需要注意的是，正在大刀阔斧改革电商生态的快手，变得和淘宝越来越像。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月初，快手电商用户运营负责人证实快手将于近期上线独立商城页面，并在8月完成了灰度测试。根据官方公布的数据，测试期间有超过70%用户首次通过快手商城下单。而货架式的商城页面，和淘宝等综合电商平台的页面设置如出一辙。这也就意味着，淘宝和快手的直接对抗将会加强。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>然而，和抖音相比，快手的头部主播的影响力正在下降，流量也不及老对手那么充沛，更容易受到淘宝的攻击。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二季度财报显示，快手日活用户仅仅环比增长0.5%，日活用户日均使用时长更是环比下跌2.3%至125.2分钟，已然触碰到流量天花板。前些日子，辛巴怒斥东方甄选直播间高价出售玉米谋取暴利，引发了网友热议。其中一种声音尤为刺耳：辛巴是不是眼看东方甄选火了，想碰瓷、蹭热度？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在近期的一场直播中，辛巴回击了外界质疑，称自己只是“看不惯这种行为”。但在许多看客眼里，辛巴家族热度下滑是不争的事实——同样有此担忧的，还有他背后的整个快手电商。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月份，快手进行了新一轮大规模组织架构调整，CEO程一笑亲自兼任快手电商第一负责人，原快手电商一把手笑古则转为掌管本地生活业务部，被视作快手加速流量变现的标志。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于极度依赖流量，中腰部主播号召力有限的快手电商来说，李佳琦复出导致的流量被分摊，绝对不是一个好消息。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/185981477061c95ab8f16f14.86688433.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>商家怎么看？需要时间重建信任</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“如何看待李佳琦的复出”这个问题，不同的群体会给出不同的答案。消费者用钱包表达了自己的立场；几个电商平台也如前文所言，都有自己的算盘。还有一个群体的态度，同样值得我们关注：站在直播间背后的商家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>不同的商家，对于李佳琦的复出有不同的期待。其中最高兴的，肯定是和李佳琦合作最为紧密的美妆品牌们。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在9月20日复播当天，李佳琦直播间的选品就以美妆个护和快消品两个过往最热门的品类为主，花西子、珂润、OLAY、夸迪和橘朵等过往的老熟人纷纷前来撑场。截止当天下播时，珂润润浸保湿滋养乳、花西子首乌眉粉笔、夸迪玻尿酸清润次抛精华液等商品均宣告售罄，“李佳琦女孩”的消费能力依旧保持在往日水准。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李佳琦不在的这段日子里，与他紧密合作的一众美妆个护品牌日子也是相当不好过，尤其是历来视淘系电商平台为主阵地的本土新消费品牌。国际大牌的入局，让本就流量见顶的淘宝直播生态加速恶化，水涨船高的坑位费就是内卷的见证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在直播间之外，部分品牌还感受到了来自资本市场的阵阵寒意。夸迪的母公司华熙生物最近五个交易日四次收跌，9月19日单日暴跌近14%。除了监管层突发“黑天鹅”事件之外，华熙生物过高的营销支出也成为一大争议。半年报显示，华熙生物今年上半年销售费用同比增长54.68%至13.87亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对线上渠道的依赖就意味着，华熙生物必须要为流量买单，哪怕各大电商平台的流量现在已经越来越紧缺，也越来越贵。这时候，自带流量与热度的李佳琦适时复出，对夸迪及其背后的华熙生物来说无异于一剂强心针。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，不是所有品牌都在翘首等待李佳琦回归，也不是所有人都对这位“一哥”的复出前景感到乐观。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，不少品牌已经转投其他头部主播的怀抱，寻找新的流量入口。截止目前，东方甄选已和包括YSL、雅诗兰黛、美宝莲、敷尔佳在内的诸多国际大牌和本土网红品牌建立合作关系，早就杀入了李佳琦的腹地虎口夺食。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，停播三个月之后再复出，商家需要更多时间观察消费者、舆论环境甚至平台对李佳琦的态度。</span><span><strong>尤其是对于中小品牌来说，李佳琦复出会不会掀起直播电商新一轮价格战，坑位费会不会再次被拉高，是它们最关心的问题。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月21日，华熙生物官方回应称，虽然旗下品牌夸迪参加了李佳琦直播复出首秀，但尚不确定双方合作关系会持续多久，尤其是双十一会不会继续合作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>华熙生物的回应，相信说出了许多商家的心声。今年618期间意外停播，已经造成了不少品牌的库存积压。过往和几个大主播都有过不愉快经历的大品牌，比如欧莱雅则在积极探索自播路线，降低对头部主播的依赖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在价值研究所看来，经过长达3个月的空窗期之后，李佳琦和背后的品牌方需要一些时间重建信任也无可厚非。而即将到来的双十一，将会成为双方最重要的一场大考。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/185981477061c95ab8f16f14.86688433.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>写在最后</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚进入9月份，阿里就迫不及待公布了今年双十一的日程表。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>官方消息显示，今年的天猫双十一大促将从10月24日也一直延续到11月11日当天。在这期间，除了10月24日晚间8点开启的预售之外，10月31日、11月10日还会有两波正式开售。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然和去年提前至10月20日的预售相比，今年双十一战线稍微缩短，但阿里对这场年度战役投入的人力、物力绝对不输以往。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，李佳琦复出，消费者、商家和平台对双十一也有了新的期待。在618热度下降，电商行业整体下行的大环境下，阿里也好，抖音、快手甚至京东、拼多多也罢，都很需要一个火爆的双十一来缓解压力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过，正如前文所说，李佳琦的回归不能掩盖流量枯竭、中腰部主播和商家增长乏力等老问题，消费者的热情能维持多久也是未知之数。只能说，借着李佳琦复出这股东风，各大电商平台可以稍稍回血，也为日后的转型争取更多时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>改革，依然不会停止。</span></p>
