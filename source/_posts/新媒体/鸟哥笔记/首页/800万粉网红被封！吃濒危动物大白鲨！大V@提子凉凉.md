<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月14日，一条短视频在网上引起轰动。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>有网友爆料，快手美食类<a href="https://www.niaogebiji.com/tag/%E7%BD%91%E7%BA%A2" target="_blank">网红</a>博主@提子，拍摄视频解剖并吃鲨鱼，发在其个人账号。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接着有科普博主称，视频中的鲨鱼是国际濒危野生动物“噬人鲨”，也叫“大白鲨”，属于国家二级保护动物，被列入《世界自然保护联盟濒危物种红色名录》，捕捞后不管死活都需要抛弃入海。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/140436487562e84f77dd9fb2.47119673.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时博主@提子反驳称，自己拍摄是“尖齿鲨”（普通动物），渠道正规。然而根据封面新闻8月1日最新的报道，<strong>南充警方调查出视频中确实是大白鲨，来源于福建沿海地区。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人可能不知道@提子是谁，她在的快手账号拥有将近800万粉丝，是顶流网红。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/156813512262e84f784f9d30.49874008.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>作为一名颜值主播，@提子拍摄的短视频大多是穿着JK装，以甜美的形象制作美食，吸引关注。</span><span>目前@提子的快手账号所有作品已被清空，账号被快手封禁，无法关注。</span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>解剖大白鲨是为哪般？<br></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我找到了原视频，视频中@提子迈着活泼的步伐，<span>用很夸张的表情说：“</span>今天我要带100斤的巨型人工养殖大鲨鱼，回家看姨婆......一个鲨鱼这么大。”<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>画面一转，她来到海鲜店，很兴奋的跳起来喊：“这么大，这么长。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>‍</span><img src="https://ngbjimg.xy599.com/180776668062e84f78dc1043.53546631.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接着一条2米长的大白鲨从包装袋中露了出来，全身灰色，静止不动显然已经是“尸体”，周围一群阿姨和小朋友们开始围观。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她拍拍鲨鱼的背说：“不要看它长得这么凶狠，但是它的肉真得超级嫩。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时候问题就来了，她怎么能提前知道肉非常嫩...要么纯粹是用这台词吸引人，要么是提前吃过了？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>‍</span><img src="https://ngbjimg.xy599.com/179231619762e84f7cb9c683.57818998.gif" referrerpolicy="no-referrer"><span>‍</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来的画面令人极度不适了，她找人扛着鲨鱼回到自己家，嘴里喊着“鲨鱼大哥到家了”，接着开始解剖鲨鱼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>‍</span><span>‍</span></span><img src="https://ngbjimg.xy599.com/47518141862e84f829e1550.24014858.gif" referrerpolicy="no-referrer"><span><span>‍</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用大油锅倒入油，放上一大堆辣椒，一半煮，一半烧烤，最后分给了乡亲们吃，视频中还拍摄了大家吃鲨鱼的画面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>‍</span><img src="https://ngbjimg.xy599.com/196141984862e84f880a78e8.31381810.gif" referrerpolicy="no-referrer"><span>‍</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>看到这里，大家应该发现了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>她的肢体动作，表情语言都很夸张，虽然视频主题是制作美食。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>但吸引人的噱头显然是“穿着打扮”、“演技”、“道具”、“台词”和大家不常吃“冷门动物食材”。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这和常见的美食视频（例如李子柒的视频）完全不同，也不像是探店视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在她过去的视频中，出现过几十斤的人工养殖鳄龟，鲍鱼，龙虾，甚至还有鸵鸟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做菜时她都会洒入一大盆辣椒，倒上大半桶油，用半米宽的铁锅开始炒，接着家人急忙拦住，说败家子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/68082109362e84f8e8157d2.94164122.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再比如，穿着清纯的@提子，带上穿着火辣的闺蜜，逛养殖场，接着一起吃鸵鸟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/196216855962e84f8f0a84f8.37309832.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“甜妹”和“巨型食物”形成了很大的反差，以此来吸引用户观看。稀有少见的，反而会成为她视频中的食材，用行业里的话说就是“博眼球，辣眼睛”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>根据警方公布的消息，@提子是</span><strong><span>从网上购买，来自于福建沿海地区</span></strong><span>。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>我发现主流的购物平台都没有购买渠道，也没有人工养殖，那么</span><span>大概率是非法捕捞获得。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接着究竟是和海鲜店取得了联系，还是和@提子本人直接取得了联系，就不得而知了，因为海鲜店老板自己称“鲨鱼是博主自己带过来的”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我想，即便不知道该鲨鱼是大白鲨，但作为一个800万粉丝的顶流美食类主播，每天和食材打交道，基本的常识判断能力得有，其中的道道摸可比普通人清楚的多。再者，尤其是像鲨鱼这种特殊动物，至少让商家出示相关资质吧，也不难。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，要么是“蠢”，要么是“坏”。</span><span>目前@提子的快手账号内容已被清空，点击主页提</span><span>示，已被封禁。等待她的还远不止封号那么简单。</span><span></span></p>
