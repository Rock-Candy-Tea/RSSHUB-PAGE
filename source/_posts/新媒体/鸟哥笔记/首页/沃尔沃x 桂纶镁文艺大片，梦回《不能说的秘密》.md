<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提到文艺片女神，那么一定想到的是桂纶镁，许多人认识桂纶镁，还是从周董那部《不能说的秘密》开始。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，桂纶镁为Volvo XC40拍摄了一支形象广告《最真一面，路上见》，<span>被网友称为是最美<a href="https://www.niaogebiji.com/tag/%E6%97%85%E8%A1%8C%E5%B9%BF%E5%91%8A" target="_blank">旅行广告</a>片，</span>文艺又唯美的画面，走心的旁白文案，梦回《不能说的秘密》。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>视频文案</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时间是一个梦<br>还是一条通往梦的道路<br><br>把昨日和熟悉的一切向后抛<br>去和未知的美好不期而遇<br>你期待见到怎样的风景？<br><br>路上那些无法被复制的片刻<br>和偶然的相遇<br>都成了日常的吉光片羽<br>都成了一部分的我<br><br>弯道之后<br>就是那年夏天的海<br>是那年秋天的夜<br>是一直一直，想念的远方<br><br>让自己走得够远<br>你才知道什么是孤独<br>什么是勇敢<br>什么是恐惧<br>什么是真正的自在<br><br>就让路，引领着我<br>在日常和向往之间不断前进<br>每一次出发<br>都是为了和自己重逢<br><br>Volvo XC40</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最真一面，路上见</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是<a href="https://www.niaogebiji.com/tag/%E6%B2%83%E5%B0%94%E6%B2%83" target="_blank">沃尔沃</a>在台湾地区所推出的最新广告，金马影后桂纶镁连续3年代言Volvo XC40，并邀请到金马导演钟孟宏操刀、金马影后桂纶镁担纲演出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再加上林強的配曲「单纯的人 A Pure Person」，集结三金打造，堪称最强广告阵容，让这支广告不再是仅仅是广告，带给观众电影级别的视听体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短片里桂纶镁驾驶着沃尔沃自由驰骋的惬意画面，和略带文艺且富有生活哲理的旁白，展现了人生旅途上不断相遇、寻找与发现自我的过程，以及与沿途人文景致相会的美丽跟温暖。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">1</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>出发在路上的意义<br>是为寻找真实自己</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经典电影《天堂电影院》里有一句经典台词，<strong>“如果你不出去走走，你就会以为这就是全世界。</strong>”也是无数旅行者选择出发的理由。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>旅行，是一个认识自己和寻找自己的过程。就像短片所说：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让自己走得够远<br>你才知道什么是孤独<br>什么是勇敢<br>什么是恐惧<br>什么是真正的自在</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在旅行的过程中，我们可以将昨日的烦恼都抛去脑后，去遇见不期而遇的美好。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>路上的风景远比我们想象的更美，在短片中，桂纶镁驾驶着Volvo XC40在沿途遇见了很多美丽的风景：有盘旋的公路，蓝色的大海，成群结队的海鸥、傍晚的夕阳、日月潭体验破晓的日出、还有夜晚璀璨的火花......</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/929393003630caf8fd6d3d0.04908334.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在路上还可以看见更真实的自己，学会与自己独处，勇敢面对未知与挑战，亦可以享受真正的自在。路上那些无法被复制的片刻和偶然的相遇，都成了日常的吉光片羽，成为我们人生的一部分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1814722557630caf9160c591.82442696.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快节奏的生活，现代人面对工作、生活和社会的种种压力，更加渴望「精神自由」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>许多人之所以选择买车，除了通勤等生活需求外，还有一个重要的原因就是精神需求。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公开资料显示，80、90后在旅游人群中的比例越来越高。年轻人不喜欢束缚，对自由行、自驾游更为青睐，如今越来越多人选择自驾去旅行，去遇见沿途的风景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>沃尔沃精准地洞察了现代人的精神需求，将车与旅游、自我相联系起来，传达了可以引领当代人前往向往之地的信息。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让更多人出发上路，遇见真实的自己，是沃尔沃传递的品牌诉求，因此沃尔沃提出了传播的<a href="https://www.niaogebiji.com/tag/slogan" target="_blank">slogan</a>「最真一面，路上见」，</span><span>借此传达了每一次出发，都能遇见最真实的自己的品牌理念。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/11160792630caf92c5d528.68389372.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">2</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>三度携手金马奖影桂纶镁<br></span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>高度诠释品牌格调与态度</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不得不说，沃尔沃这支短片找桂纶镁演绎，给整个创意加分了不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为桂纶镁就是小清新和文艺范的代名词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2002年，桂纶镁与陈柏霖搭档主演的青春爱情片《蓝色大门》，完成了银幕初秀。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2049078036630caf938863c9.60004679.png" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她最经典的荧幕形象就是和周杰伦一起出演的《不能说的秘密》，路小雨是多少人心中校园初恋的模样。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1018404522630caf93d78461.62567016.jpeg" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为金马奖影后的她，精湛的演技与独特的文艺气质，完美诠释形质俱佳的XC40 的迷人魅力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1672883808630caf943c9735.38415346.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，这不是沃尔沃首次与桂纶镁合作，2020 年，沃尔沃与桂纶镁推出了「让世界搭上我的节奏」，演绎品牌态度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年，沃尔沃借助《蓝色大门》IP，邀请到桂纶镁陈柏霖一起拍摄的广告短片。青涩的夏日，肆意追逐的记忆，时隔19年，二人再度同框，将回忆拉满。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再到此次文艺范满满的广告大片，桂纶镁再次完美诠释自由、舒适的格调。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果放眼整个传播线，会发现沃尔沃选择的传播策略<strong>一直是从人的需求出发，以文艺范的广告带出产品卖点，以温暖和情感为内核与受众沟通。</strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">3</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>情感营销传播策略</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>搭建用户沟通桥梁</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>注意力稀缺的时代，主流消费群体的共情阈值提升，围绕产品本身的广告难以激发消费者的购买欲望。通过情感营销，能<span>让消费者</span>提升对品牌乃至产品的好感度，拉近与消费者的心理距离。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现代人买车越来越追求产品调性和舒适的体验，追求生活品质，广告片中展现的驾驶情节，也从侧面凸显出沃尔沃XC40舒适的驾驶体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>沃尔沃XC40定位为一台走个性化路线的精品紧凑型SUV，兼容了时尚和动感的设计，外加简约且科技感十足的车舱，顶级的安全防护系统，颜值高，设计出彩，对女性的“吸引力”尤其高，满满的轻奢味。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在视频中，桂纶镁的演绎更加强了这款车的气质。诗意化的表达，唯美的画面，桢桢美如画，更能吸引文艺范的目标受众。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2120881357630caf94b51772.33025642.png" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1435151659630caf95507925.69100717.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然没有刻意去传递产品卖点，但沃尔沃XC40驰骋在公路与水里的镜头，更好地展现了这款车的防水性能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1138684405630caf95dbede4.49441534.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在我看来，这支唯美的广告大片既兼顾了观众视听感受，也没有落下产品卖点的露出，同时还传递了品牌态度，拉近与消费者的距离，做到了商业和艺术的完美结合，实现了品效合一。</span></p>
