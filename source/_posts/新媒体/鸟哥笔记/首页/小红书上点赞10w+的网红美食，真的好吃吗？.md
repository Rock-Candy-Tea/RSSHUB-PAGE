<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这期内容，我想从商业的角度，聊聊为什么网红美食的终点是黑暗料理。</strong></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>01</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在社交网络出现之前，其实是不存在网红的概念的，更别说网红美食了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>中国最早的网红美食，我认为应该是2012年开业的<strong>黄太吉煎饼</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/43417307662e14795e7abf5.59023740.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>黄太吉打出的口号是用互联网思维做煎饼。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老板赫畅出身百度，也在谷歌和4A广告公司工作过，估计是积累了一肚子「赋能」、「降维」、「迭代」之类的黑话。看啥都觉得可以用互联网思维改造一下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/31120748062e14796292529.38548665.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>赫畅</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但是他所谓互联网思维，现在看来就是冲热搜。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人也许还记得「老板开奔驰送煎饼」的土味营销，在微博上一度引起很大反响。这个开奔驰的老板就是赫畅。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/104192819362e1479660f752.49859792.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那年，微博还是个新兴社交平台。黄太吉在微博上各种搞噱头，包括把煎饼果子开进北京CBD，找美女送煎饼，号称要打造中国的麦当劳等等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/119323595462e147969a3375.87252496.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>很多后来网红店的玩法，都能在黄太吉这里找到原型。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如取餐的机打小票上，附带各种鸡汤文案，至今茶颜悦色还在这么干。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/194978957862e14796bd0bb7.45962246.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有墙上《葫芦娃》、《变形金刚》的怀旧插画，算是怀旧主题网红店的先行者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>另外就是贵，黄太吉在十年前就敢把煎饼卖到9块钱，某种程度上也算是雪糕刺客和喜茶奈雪们的祖师爷了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>利用微博带来的影响力，黄太吉在北京引发了一阵打卡热潮。随之而来的是雕爷、伏牛堂、西少爷等一众「互联网思维」网红餐饮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/191156508162e14797d7b5b7.53123343.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这批「互联网餐饮」，就是中国第一代网红美食。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>说是互联网思维，总结一下它们的特点，无非就是认真吹牛皮，认真搞噱头，认真卖高价。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>至于食物本身，煎饼还是那个煎饼，肉夹馍也还是那个肉夹馍，没啥创新，但是也不至于到黑暗料理的层面。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过初代网红们死得都比较快，很少有活过四年的。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>02</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二代网红美食的思路，本质上也是搞噱头，但是他们逐渐发现顶着「互联网思维」的名头卖传统食品不好使了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为第一代网红美食吹逼的目的，其实是tovc，也就是针对投资人的，目的就是告诉投资人我这玩意儿不一样，赶紧来投我吧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但随着第一代网红美食的失败，投资人也变聪明了，不见兔子不撒鹰啊，你不能光给我讲概念看PPT，你得让我看到市场成效。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>所以第二代网红美食，基本就是把产品当作网红本身去运营了。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>刚好处于社交网络的爆发期，他们敏锐地察觉到，其实客人来吃啥根本不重要，能来拍照把照片发到社交媒体上，才是最重要的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么最有效的办法就是把产品做得好看点，把店面装修搞得漂亮一点，一切都是为了方便人们拍照。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，当你把产品完全看作网红之后，怎么才能持续制造虚假繁荣呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很简单，就是买流量，刷数据，放在线下，就是我雇人排队不就好了？发通稿报道很多人排队不就好了？再请人在社交媒体上发帖发照片不就好了？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/168002949162e14798105177.12253196.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样一来，短期内消费者一看这个东西火了，我要去试试，投资方一看，好家伙，网上都是你的照片，仿佛全国人民都是你的消费者，两三个城市就这么夸张，等开满全国那还了得？赶紧投了。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>03</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>而第三代网红美食，则是在第二代的基础上，继续极端化。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为从初代网红美食开始，大家选择的品类基本都是复制难度低的，把产品做好看、请人来排队的手段已经越来越卷，消费者也视觉疲劳了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>于是，网红美食扭曲了，开始朝着「怪」的方向发展，要的就是出格，要的就是博出位。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>而短视频的出现，也让网红美食在视觉造型之外，还要注重「玩法」。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频要的不止是一眼好看，而是在15秒的时长里能提供惊喜感和刺激，能玩得起来的美食，才适合发短视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以可以用来占卜的答案茶火了，海底捞的DIY菜单火了，西安摔碗酒火了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/43548732662e147984c0c71.45806255.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>答案茶</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/168201511462e1479888d295.96091341.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>西安摔碗酒</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>可以看到，网红美食的三个阶段，口味其实是越来越不重要了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早期「互联网餐饮」还会和你聊聊「你的味蕾还没打开」，后来的网红美食只关心能不能出好看的照片，拍完再吃。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>到短视频时代，已经没人在意吃不吃了，娱乐才是最重要的。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你是一个老派的美食家或者餐饮从业者，你肯定会觉得难以理解。在传统角度看来，烹饪有底层的逻辑，美食有基本的原则。创新可以，但改编不是乱编，戏说不是胡说。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网红美食呢？它完全跳出了这个思维，开辟出了一条全新的标准：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>既然是网红美食，那只要红了那就是美食。</strong></span><strong>成为网红美食，最大门槛不是好吃，而是红。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反正排队的店，总有人要去试试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>哪怕味蕾告诉你不好吃，排队的时间成本和朋友圈几十上百的点赞也会强行修正你的味蕾，改造你的记忆，让你喊出「yyds绝绝子」。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>04</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>那么问题来了，如何让一款产品成为网红？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之前辣酱那期视频里，我引用过一个概念：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><em><span>你在米饭里放点肉，叫炒饭。在肉里面放点米饭，那就是网红炒饭。</span></em></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><em><span>你在奶茶里面放芝士，叫做奶茶。但如果你在芝士里面放点奶茶，那就是网红奶茶。</span></em></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>不成比例地放大某一项原料的比重，然后放大用户感知，就容易出网红款。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这句话不是我原创，但我愿称其为网红第一定律，因为它高瞻远瞩地指出了网红的基本属性「感知」，和打造网红的基本方法论「不成比例」。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先说感知。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在我看来，凡是能极大程度调动起人们感官的东西，客观上都具备网红的潜质。网红两个字，就是感官系的代名词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网红是极致的商品化，一手给刺激，一手送流量，在审核能通过的情况下，你刺激给得越多，换来的流量一定也就越多，公平交易童叟无欺。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>如果说赚钱的方法都写在刑法里，那么爆红的方法则都写在了审核规则里。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么感官刺激那么重要，这里就要聊到<strong>互联网的推荐算法</strong>了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/203670444762e14798c93dd7.55049290.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>推荐算法这个东西，其实类似于打怪闯关，一条内容需要从小到大，闯过一个个流量池，最终才能成为一条爆款内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当你在网络平台上发布一条内容，这条内容就会被分配到一个小型流量池里。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个流量池里的用户就都能看到你的内容出现在他的手机屏幕上。一般来说，这个初期流量池的规模大概在几百到一千多个用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而这条内容是可以继续传播下去，还是就此停在这里，完全取决于这些用户是否打开你的内容，是否留言、点赞、转发，完播率和完读率有多高。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果数据指标足够好，那么你的内容就算是过关成功，会被推荐到下一个更大的流量池。这里有数万到数十万不等的用户，你的内容将会接受他们的审判。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此时，你能看到的是，你的内容的推荐量，阅读量都会有一个飞跃式的增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果在他们这里，你的内容依旧数据良好，那么以此类推，这条内容会被推到一个个更大的流量池接受检验，直到碰到一个它征服不了的流量池，或者随着时间推移，它的生命周期自然结束。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>那么，真正能跑通所有流量池，在每一个关卡都能成功打怪进阶的，就是人性。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这里的人性包含各种要素。感动、同情、热血这些正面情绪可以击中人性，猎奇、欲望、仇恨这些上不了台面的东西也能击中人性。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>找到人性的最大公约数，就找到了所谓的流量密码。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而直接的感官刺激，就是人性最大公约数，毕竟每个人的情感点可能不一样，但谁还没个眼耳口鼻呢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>那么如何强化感知呢？就要靠「不成比例」。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我举个例子，为什么要把瘦脸、大眼开到最大，造成不成比例的大眼高鼻梁尖下巴？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人说这种不成比例并不好看呀。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但这种事情，要的不是一部分人说好看，要的是更多人点赞，数据是不说谎的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样的原理，在所有涉及网红的领域，都有应用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在情感博主这里，就是不成比例的小作文、撒狗血、卖苦情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在网红景点这里，就是不成比例地打造各种视觉奇观，铺设打卡摄影点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/149227802262e147994f3275.08175074.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在网红美食这里，就是不成比例地堆料，把看上去互不兼容的食材硬怼上去。把油脂、碳水这些能激发进食欲望的成分加入配料，最好还要弄得花花绿绿，美其名曰「出片」。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/107654728562e14799a466d8.29564413.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>媒介是人的延伸，但是这种延伸是跛脚的，互联网媒介只注重视听感官，其他感官刺激是解决不了的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那网红能做的，就是在视听刺激上不断加量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而视听刺激不断加量，感官的平衡不断被破坏，最后必然产生一些奇葩的存在。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如网红脸，筷子腿，A4腰，夹子音，气泡音。也比如网红暗黑料理。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>05</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>「不成比例，放大感知」。</span><span>有了这套方法论，我们就可以打造网红美食了。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，我做一个网红美食的构想啊。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如我做一个奶茶品牌，就叫馆式茶艺，大家没意见吧？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在开店之前，我要开发一个王牌单品，我现在大家喜欢嗦粉，喜欢喝椰奶，喜欢鸭屎香柠檬茶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>考虑到现在是夏天，要清凉，我觉得气泡水之类的饮料一定卖得好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是我决定推出一款鸭屎香椰奶凉粉气泡茶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>名字是不是有点太普通了？那就改叫鸭鸭椰椰冰冰凉好喝到冒泡茶。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/29654734462e14799eb5e93.37729727.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有茶饮还不够，你喝奶茶不顺便整点甜品？我又去网上看了一下，发现臭豆腐很火，千层蛋糕很火。所以我决定开发一款臭豆腐榴莲千层，一层臭豆腐一层榴莲，要的就是极致的嗅觉享受。顶上还得放两块油炸臭豆腐，强化视觉感知。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/105212701162e1479b145192.19123673.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费者跑到我的门店一看，发现连个排队的人也没有，也不行。雇人排队那是常规手段了，一劳永逸最好搞限定发售。鸭鸭椰椰冰冰凉好喝到冒泡茶每天只有晚上5点到8点供应，臭豆腐蛋糕每天限量50份，售完为止。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最好门口放个女装熊猫打卡点，来店里喝茶，还可以和女装馆长合影。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/82247219262e1479c66d5c8.17862863.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到这个时候，差不多风投也该注意到我的馆式茶艺了，接下来就是融资扩张，破产跑路的故事了，以前挖的坑也不用填了，简直一举两得。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><em><em><span>06</span></em></em></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说回网红美食。我觉得用「食物」来定义网红美食，那格局还是太小了，网红美食是一种多维度全感官刺激型产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>而在所谓全感官里面，美味是最不重要的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>味觉是没有传播力的，它没法顺着网线，让互联网另一头的人感受到。这个问题，5G解决不了，6G也解决不了，元宇宙也解决不了，只有快进到22世纪杀人网络，所有人都快进到脑后插管进入The Matrix才可能实现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>但是视觉和听觉传播力就很强。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你为肥肠包榴莲发一条朋友圈，那种奇葩的造型，带来的视觉冲击，所有人都能看到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你拍个西瓜冷面的短视频，你滋溜滋溜暴风吸入，所有人都能听到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>前面我提到了，媒介即人的延伸。但这种延伸不是平均的延伸。我们的视听可以走得很远，但触觉听觉嗅觉味觉只能停在原地。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这注定了媒介反过来干预正常生活的时候，视听效果越好，越容易得到媒介的青睐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网红美食不是美食，同理，网红景点不是景点，网红艺术展不是艺术展，网红健身房不是健身房，网红书店也不是书店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>它们虽然身在现实世界，但它们的价值却存在于赛博世界，仰赖网络流量和推荐算法实现。</strong>所以你在网上看他们无比美好，实地探查总会发现一地鸡毛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网红街区多半离市区十万八千里，附近荒草丛生演绎荒野求生；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网红展除了几个造型奇葩的合影打卡点什么都没有，用鲜明的案例告诉你艺术是需要想象力的；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网红健身房根本不像健身房更像夜店，教练也不是上课的更像是拿来拍照的；</span></p>
