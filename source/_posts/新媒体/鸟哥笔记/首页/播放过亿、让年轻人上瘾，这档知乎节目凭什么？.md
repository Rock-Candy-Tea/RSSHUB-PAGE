<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每年的九月都是开学季，也是职场季。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有的年轻人，结束了高考走进了大学；有的年轻人走出了大学，进入了职场。生活就这样突然迎来了转折点，年轻的困惑迷茫常挂在热搜。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>运营研究社发现，针对这些迷茫与困惑，知乎推出了三档针对年轻人的自制节目：《荒野会谈》《我的高考笑忘书》《我所向往的职业啊》。<strong>三档节目引起了广泛讨论，节目从 8 月播出至今播放量已经破亿。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11103375636329e87e670041.97469895.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些节目给年轻人带来了怎样的启发？为什么职场、高考、人生这些话题一直被人关注？知乎社区为这些内容的诞生提供了怎样的助力？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今天，运营研究社带你来一探究竟。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em><strong><span>01、</span></strong></em></span><strong><span>知乎高赞自制节目，</span></strong><strong><span>治好了年轻人的“精神内耗”</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知乎在这个夏天推出的高考纪实人物访谈节目《我的高考笑忘书》、职场青年纪实观察节目《我所向往的职业啊》，荒野系青年谈话节目《荒野会谈》，分别从高考、职场、人生意义三个层面来展开话题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>高考的失败真的可以“笑忘”么？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>有人在看过《我的高考笑忘书》，给出了这样的回答：如果你后来成功了，那确实是可以笑忘的，如果没有成功，那么高考会成为人生永远难过的一道坎。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>电影《最后的告别》的导演，张中臣就曾在上面栽过跟头。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在农村生活的张中臣生活中充满了饥饿感，对未来也迷茫。高考失利的他，一度不知道人生的方向是什么。跟着去打工，又因为工友的猝死选择逃离工厂。最后到了大学当保安，误打误撞之下，张中臣碰上了一节大师课。从没接触过电影的他，似乎在这里被激发了灵魂，在工作之余自发学习剪片子。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就这样边学边做，剪了十部长片之后才开始写自己的第一部剧本。<strong>而这部名叫《最后的告别》的电影，一鸣惊人，拿下了第 15 届 FIRST 青年电影展最佳导演和最佳剧情片奖，张中臣出名了。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了张中臣外，节目中其他几位主人公王菊、张一卿、大猛子都是在不断被否定，不断怀疑自我中找到了自己生命的锚点。他们终于理解：生命不会停留在高考那一瞬，生活永远向远处蔓延。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15228983846329e87ed3c853.49639704.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>相比于《我的高考笑忘书》的抒情，《我所向往的职业啊》则充满了实用。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《我所向往的职业啊》以 VCR 纪实的方式，跟拍主人公三天的工作日常，嘉宾室则邀请了杨天真、陈铭、冉高鸣、崔璀、夏鹏，及该行业资深人士进行点评和观点输出。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>节目真实地展现了关于如何与领导沟通，如何处理事情等职场场景，节目内容也呈现出切实可行的思考与见解。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如在关于如何发邮件上，《在外企工作是怎样的体验？真的福利好不加班吗？》一集就提出了 3 个小 tips。</span><span></span></p>
<blockquote>
<section><span>1.用最简洁的语言叙述事情的来龙去脉</span></section>
<section><span>2.提出自己的观点和方案</span></section>
<section><span>3.在邮件中明确自己的诉求</span></section>
</blockquote>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11774252986329e87f592124.28052528.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，还有与老板沟通的办法，正确开会的姿势等一系列实用技巧，可以让人掌握到在一流企业千锤百炼沉淀下来的职场技巧，这对于刚入职场的小白来说，非常有用。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《我所向往的职业啊》的“主人公”选取了美国互联网 Meta</span><span>（原 Facebook）</span><span>程序员，外企宝洁品牌总监，中国红圈所律师，小米工业设计师四位不同身份的职场人士，进行纪实性地录制。让我们了解这些很多人渴望的职业，究竟是怎么样，也明白了这些企业的优秀人士究竟闪光点在哪。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而节目除了真实的现场，和实用的技巧外，也有不少升华的思想。比如</span></strong><span>@张书豪</span><strong><span>在《我所向往的职业啊》里说的「做巨轮里的一个螺丝钉也很好」就引发了讨论。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在这一集里也谈到，人在职场中最理想的状态是找到自己的位置，不是和别人拼得你死我活，而是知道自己的能力上限，用好自己的能力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而到了《荒野会谈》，谈论的话题则变得更加深入，比如李雪琴提出的问题「人可以完全没有社交吗」？</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>问题在不同的人的口中，诞生了不一样的答案。但是有一点是统一的，在 Z世代眼中，社恐不再是一种病，社牛、社杂也仅仅是社交标签。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在荒野会谈中，人仿佛褪去了城市中的一切社会关系，在仅仅谈论人的本身。Z世代接受给自己标签，不怕被定义。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三档节目虽然形式、聚焦的点都并不相同，但却都因为戳中了 Z世代的痛点，在知乎社区激发了用户的共鸣，引发了很好的反响。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>截至 9 月 15 日，《我的高考笑忘书》、《我所向往的职业啊》、《荒野会谈》三档自制节目在知乎站内播放量达 1.13 亿次，站外曝光量超 10 亿，已有 18 个节目相关话题登上微博热搜、热议、文娱、视频等各类榜单。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这对于第一次尝试将内容视觉化的知乎来说，是一个非常强的表现。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><em><strong><span>02 、</span></strong></em></span><strong><span>这些节目爆火的背后，</span></strong><strong><span>还有这些深层原因</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么知乎的自制节目为何能取得这样好的效果，并且轻松活泼又不乏深度呢？我们仔细分析了三档节目，发现其成功的共性因素，有这三个层面的原因。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1）以问题驱动内容</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在内容上，三档节目，都进行了细致地选择，对目标人群，节目形式，聚焦内容都有明确的划分。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>能进行明确的划分是因为，<strong>知乎作为一个问答型为主的社区，早已沉淀了大量的问答内容。</strong>通过大数据能够精准地找到在一个较长「时间维度」内，青年的「共性」问题。积累下的数据不会骗人，哪些问题最吸引人，哪些话题最能引发关注。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如在《我所向往的职业啊》，选择竞天公诚律所的 90 后律师李龙宇作为纪录对象，就是因为在知乎上「律师」+「律师行业」话题，有 166W+ 关注和 64W+ 讨论。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3683063056329e87f93b045.73105688.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如在知乎提问“在外企工作爽吗？”的问题下，就有 2733 条答案。这些答案中，大多都是答主真实的经验和感受，对想要进入外企的同学会产生有助益的帮助。也正是因此，知乎邀请了<span>外企宝洁品牌总监来制作节目。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而另一边，节目的火爆，同时又引来更多在外企工作的人士，来回答自己的经验和方法，积累更多的新鲜内容。《我所向往的职业啊》外企篇播出后，视频下面迅速多了 785 条赞，和 124 条评论。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，节目嘉宾重返知乎回答节目相关话题，也是一个内容的「二次沉淀」。比如《我所向往的职业啊》国外大厂程序员篇中的</span><span>@张书豪 </span><span>回答自己在节目中展开的观点，被其他用户“热情”点赞，真正成了“大家向往的高赞答主”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/5871193626329e87fdb7bb1.12475746.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2）“过来人”分享人生的非标准答案</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在三档节目中，都陆续出现了好几位让人熟悉的知乎答主。<strong>他们在知乎有一定的「粉丝基础」。同时，这些知乎答主往往都是某个领域的专家，他们给出的建议会更具有「说服力」。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如《荒野会谈》中的李松蔚，他是北大毕业的心理学工作者，因为在知乎上长期回答心理学话题的问题，得到了很多用户的喜欢。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而在《荒野会谈》中，他完全不是我们印象中心理医生的形象，反而像是一个有些“社恐”的大男生，这种反差拉近了观众与李松蔚的距离，让人感觉心理学并不遥远；并且更容易理解到：社交应有距离，“社恐”是正常的现象，不必害怕。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/959462296329e880161294.55042641.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，这些答主在节目中露面，也有吸引粉丝的效果，让节目辐射范围更广大，原本粉丝只能感受答主文字的智慧，而现在却能够变成镜头语言。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这些优秀的知乎答主才是是知乎内容最强大的壁垒。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，三档节目并没有强行输出观点，而是在分享不标准的答案。就像是主观题，每个人分享着自己的见解。这种开放性的回答，更能引起 Z世代、年轻人们的共鸣。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没有人永远年轻，但永远有人年轻，“过来人”的非标准答案对于不想被束缚的 Z世代似乎更有借鉴意义。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>3）社区提供更专业的讨论</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知乎平台上沉淀了极其丰富多元的“过来人”，在知乎，任何人都可以找到并进行不同领域的专业讨论。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>知乎上的专业讨论覆盖科学、教育、数码、健康、体育、影视、财经、汽车、科技、影视、母婴等 1000 多个垂直领域。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另外，知乎作为“互助型社区”，不同于“分享型社区”，其内容质量更高，更具内容可信度。</span></strong><span>很多答主，甚至都是专注于某个行业多年的专家学者。他们能从更专业的角度，给用户带来有益的解答。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而答主们也在回答问题时，突破原有的局限。正是因此，登上节目的答主才能拥有更多可分享的内容，才能在智慧的碰撞下迸发灵感的火花。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>三档节目在高考、职场等领域的试水成功，也为知乎内容生态的延展打开了新的空间。未来或许有更多在知乎沉淀的内容，以全新的面貌重新出现在用户视野中。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021 年 5 月知乎将自家社区内容标准重新确立，明确好内容的最新定义为“获得感”，“获得感”内容有三大标准：开阔眼界，带来帮助或引发共鸣。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这三点恰恰与三档节目产生的效果异曲同工，既专业、理性又客观。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此次知乎内容视频化的尝试，不仅体现内容的媒介形态逐渐多样化，更体现了知乎社区价值的丰富多元，这意味着知乎沉淀的内容，将有更大的展示空间，和更精彩的可能性。</span></p>
