<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截至最近一个交易日，理想汽车美股421.4亿美元的市值，已经正式超越了蔚来汽车美股407.4亿美元的市值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在“蔚小理”中，蔚来汽车的名字排在第一位，这是因为其长期以来一直是中国造车新势力企业中公认的领导者。但蔚来汽车与理想汽车最近的市值变化，也说明二者的竞争实力正在发生反转。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然从市值上看，蔚来汽车距离理想汽车并不太大，但从现实来看，蔚来汽车的经营状况要比市值反映得糟糕许多。前不久蔚来汽车发布的季报显示，2022年第一季度，蔚来汽车实现营收99.11亿元，同比增速下滑至24.39%，盈利也远不及预期，净亏损为17.83亿元，相较2021年第一季度4.5亿元的亏损，同比扩大295.56%。同期，理想汽车营收为95.62亿元，同比增长167.49%，净亏损为1090万元，相较去年3.60亿元亏损，同比收窄96.97%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从上述财务数据可以看出，蔚来汽车目前正承受着巨大的亏损，距离规模化盈利还遥遥无期，而理想汽车已经接近盈亏平衡，这也是投资者对二者价值判断出现反转的重要原因之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，对于蔚来汽车这种新创企业来说，巨额亏损还不是最糟糕的。最糟糕的是其尚在巨亏的情况下，销售增长却开始出现停滞甚至下滑。公开数据显示，2022年1月-5月，蔚来汽车的交付量分别为9652辆、6131辆、9985辆、5074辆和7024辆，累计交付37866辆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/212270073262ba5526550a13.10229196.png" title="李斌再次站到悬崖边" alt="李斌再次站到悬崖边" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从上述交付量可以充分看出，蔚来汽车已经失去了早期的高增长，并且，这是蔚来汽车已经布局了较为齐全车型情况下的结果。目前，蔚来汽车在SUV品类有ES8、ES7、ES6与EC6四款车在售，在轿车领域有ET7与ET5两款车在售，基本完成了中型与中大型这两个最核心领域的车型布局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而反观理想汽车，其目前只有理想ONE这一款中大型SUV在售。小鹏汽车也只有三款车在售，分别是中型轿车P7、紧凑型轿车P5以及紧凑型SUV G3i。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在产品阵型远没有蔚来齐整的情况下，理想汽车与小鹏汽车均实现了远超蔚来汽车的销售成绩。2022年1-5月，理想汽车累计交付量47379辆，同比增长111.1%，小鹏汽车累计交付量53688辆，同比增长135.6%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在销售较为正常的5月份，理想汽车交付量为11496辆，小鹏汽车交付量为10125辆，也都远超蔚来汽车的7024辆。并且在蔚来汽车身后，还有极氪汽车与AITO等新崛起的新能源汽车品牌紧追不舍，二者在5月的交付量均超过4000辆。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2014年，李斌创办蔚来汽车，经过长达4年的努力，于2018年6月实现首批量产车型ES8的正式交付。2018年9月12日，蔚来汽车登陆纽交所，成为国内首家在美国上市的造车新势力企业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/207046882562ba55269e8162.70394977.png" title="李斌再次站到悬崖边" alt="李斌再次站到悬崖边" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品的率先交付与公司率先登陆资本市场，这都让蔚来汽车成为中国新能源汽车市场公认的先发优势占据者。但好景不长，由于新能源汽车在接下来的2019年并没有出现预期的爆发，蔚来汽车销售惨淡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是2019年第二季度财报的公布，让蔚来汽车跌至冰点。财报显示，蔚来汽车单季度净亏损32.85亿元，这导致其在资本市场的股价出现暴跌。同时，蔚来汽车又曝出高管离职，大范围裁员，投资人不肯再追加投资等负面消息，将蔚来汽车推到了悬崖边，其创始人李斌也被媒体称为“2019年最惨的人”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但较为幸运的是，进入2020年，新能源市场的骤然好转让蔚来汽车死里逃生。一方面是中国新能源汽车市场在这一年出现很大改观，另一方面是受益特斯拉在美股市场的高歌猛进，吸引投资者再次把目光转向新能源汽车企业。虽然还依然处于巨亏状态，但新能源市场的爆发以及特斯拉股价大涨给投资者带来的美好幻想，帮助蔚来汽车获得了宝贵的喘息机会，并获得了巨额融资。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>水大鱼大，随着中国新能源市场在2021年的进一步爆发，蔚来汽车也在2021年实现了销量的进一步增长。公开数据显示，2021年全年交付量为91429辆，营收为361.364亿元，同比增长122.3％。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过放眼整个行业，蔚来汽车在2021年的增长并不突出，除了理想汽车与小鹏汽车等也保持着不输于蔚来汽车的增速之外，2021年中国新能源汽车领域的真正明星是比亚迪的全面崛起。随着比亚迪旗下王朝系列多款车型的上市与热销，比亚迪一举反超特斯拉，重新夺回了中国新能源汽车市场的领导者位置。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进入到2022年，中国的新能源汽车市场开始趋于成型。这对于蔚来汽车既是好事，也是坏事。好事是新能源汽车已经得到了市场的充分验证，蕴藏着巨大的增长机会；坏事是众多强有力对手的出现，让蔚来汽车面临的竞争空前加剧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先是比亚迪与特斯拉这两家综合实力远超其他造车新势力的新能源汽车巨头，它们依然保持着高歌猛进的势能，甩开其他跟随者的距离越来越大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次是国内传统汽车厂商的反扑。例如，长城、吉利、广汽、上汽、北汽、长安汽车与东风汽车等，都纷纷投入巨资，推出欧拉、几何、极氪、埃安、智己、飞凡、飞狐、岚图、阿维塔与深蓝等主打新能源汽车的子品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再次是跨国车企均开始加强对新能源汽车市场的重视，进行产品的快速布局。例如，宝马、奔驰、奥迪、大众、丰田、本田、通用与福特等都分别发布了自己的新能源车型，其中不乏像宝马i系列与大众ID系列等颇具竞争力的产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外一个不容忽视的对手，就是由华为终端事业群负责人余承东亲自来操盘的AITO品牌，其借助华为的品牌、技术与渠道优势也在快速崛起。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在上述几大势力的围剿下，其实留给“蔚小理”等造车新势力的市场份额已经不大。而在蔚小理三家企业中，何小鹏领衔的小鹏汽车与李想领衔的理想汽车也都表现出了相较蔚来汽车更强的适应性与竞争力，二者即将推出的G9与L9都颇受关注，将有着不错的销售预期。而在所有的新能源汽车品牌中，蔚来汽车的定价较高，性价比较低，这让其在销售上面临着巨大的挑战。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/34901214862ba55275d1830.36117996.jpeg" title="李斌再次站到悬崖边" alt="李斌再次站到悬崖边" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在蔚来汽车任何一款车型所在的价格区间，消费者都有很多其他更优选择，这让蔚来汽车在战略上变得极为被动。这也是为什么有消息传出，蔚来汽车正在向下布局一个全新的中低端品牌，以期解决蔚来汽车当前在战略上的被动。不过此举也存在着巨大的风险。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一方面，蔚来主品牌在中高端市场无法站稳的情况下，再推出一个中低端品牌，这势必会造成用户在心智中对其形成廉价品牌的认知。与其购买蔚来汽车的一个中低端子品牌，不如去买理想、小鹏等与蔚来主品牌也相差无几的品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外一方面，目前蔚来汽车在售有7款车型，产品线已经非常冗长。此时如果再开拓一个新品牌，势必会造成资源与精力的进一步分散。尤其在当下时点，再重新打造一个品牌的难度已经远超蔚来汽车早期创业阶段。在蔚来主品牌还存在巨亏的情况下，如果再多一个迟迟无法打开局面的新品牌，那么蔚来汽车这家企业的经营压力便会空前加大。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于当前的市场演进趋势，《砺石商业评论》预测，到2025年左右，整个中国新能源汽车市场将有可能会出现大局已定的局面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大局已定的主要表现是，市场将形成5个左右实现规模化盈利的头部品牌。头部品牌有两个核心指标，一个是100万+的销量，另外一个是50亿人民币以上的净利润。如果5年之后尚不能达到这两个指标的车企，都将迫于财务压力，逐渐被边缘化甚至退出市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相较这两个目标，除了特斯拉与比亚迪，目前大多新能源汽车企业都有着巨大的差距。所以，未来真正能脱颖而出的，一定是少数几个保持着强劲增长的品牌。而不能持续保持良性增长的品牌，都将会因巨额亏损而掉队。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进入到2022年，本是各家新能源真正开始拼刺刀，进入决胜局的第一年。而就是在这样的关键期，每年还保持着数十亿人民币巨亏的情况下，蔚来汽车却停止了增长，再加上最近两位员工驾驶蔚来汽车身亡的负面舆论事件，再次将李斌与蔚来汽车推到了悬崖边。</span></p>
