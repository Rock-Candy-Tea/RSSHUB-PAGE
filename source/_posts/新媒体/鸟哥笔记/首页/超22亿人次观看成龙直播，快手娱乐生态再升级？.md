<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“海阔天空，太经典了，前奏一出我就听出来了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>晚上8点，<a href="https://www.niaogebiji.com/tag/%E6%88%90%E9%BE%99" target="_blank">成龙</a>的<a href="https://www.niaogebiji.com/tag/%E5%BF%AB%E6%89%8B%E7%9B%B4%E6%92%AD" target="_blank">快手直播</a>间内人气达到389万。快手音乐达人@溜溜蛋·给成龙出了一个猜歌谜题，他现场演唱了一首粤语歌，共同连线的@成龙与@容祖儿快速猜出了歌名。猜歌环节结束后，在快手老铁的弹幕刷屏下，成龙又与@溜溜蛋合唱了一曲《真心英雄》，达人与明星之间的“接地气”联动将直播间气氛推上高潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是6月25日晚上，成龙快手直播首秀的一个小小切片。与常规的明星聊天式直播不同，这场直播娱乐感十足。当晚，黄子韬、容祖儿、谢霆锋等明星陆续进入直播间连线互动，@溜溜蛋、@张踩玲、@俩大叔的功夫梦等快手达人担任着“守关人”的角色，设置种种关卡让成龙挑战，而快手老铁们则手握投票决定权，沉浸式参与其中，“成龙大哥晚上好”“龙叔的电影百看不厌”刷满屏幕。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/38481163762bdaef731efc8.99035484.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@成龙 快手直播间截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>据快手官方数据，整场直播累计超2.2亿人次观看，直播间同时在线人数突破526万，互动量超过3.2亿。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这场强互动、强用户体验的直播，既是快手对其明星生态拼图的填补，也是对明星类内容的又一次精细化运营。快手如今的明星生态呈现出了怎样的特性？在站内娱乐生态的构建上，快手究竟是怎样的思路？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/13363685916010dc4c78b4e6.24352516.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>“大哥下场，全民互动”，</span></strong></span><span><strong><span>超2.2亿人次围观成龙首秀</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“有他，有她，对了，还有你，你可是大家点名要你来的。我的第一次直播，我们快手见。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>走出荧幕后，成龙站在直播间的聚光灯下。6月25日晚上8点，身穿白色西装的成龙出现在直播间内，他手举镜头，半开玩笑地说“我是习惯叫action才能开始的”。片刻后，他便以一套利落劲道的武打镜头开场，68岁的“老大哥”就此开启了他在快手的直播首秀，超1.3亿人与他共同度过了这一夜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这并不是成龙在快手的首次亮相。去年10月，成龙以一支情怀向短视频宣告自己的入驻，还曾引发了一场快手老铁间的集体狂欢。首支“打招呼”视频《我是成龙，我来了！》中，成龙干脆利落地把玩着道具，旋即无缝衔接他在《警察故事》《神话》中的经典武打把式，让无数老铁梦回“成家班”时代。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/95553504162bdaef78fa416.98397435.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@成龙 快手短视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“全球首个短视频社交账号”的噱头，以及打出的“经典影视复刻”的情怀牌，让成龙的入驻掀起一阵流量风潮：<strong>四个小时视频播放量破亿，入驻3天账号粉丝量突破5000万。</strong>其中，时代少年团、岳云鹏、黄渤、容祖儿、沈腾等明星纷纷在评论区留言互动，还有诸如@天阳班长、@东北娃等快手达人纷纷发布视频向成龙致敬。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，快手的“迎新”设计也别出心裁，在快手站内搜索成龙，就可以看到成龙诸多经典作品的电影海报动效，观看成龙短视频并点赞，还能看到包括小金人奖杯、龙头、电影胶片等定制元素，普通用户关注成龙本人账号后可以领取成龙独特头像挂件以及专属编号等，<strong>整个社区呈现出了一种成龙大哥搬到新社区后受到“居民”热烈欢迎的热闹气氛。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时隔九个月，成龙的直播首秀再次迎来了让人瞩目的热闹狂欢，且踩在“出道60周年”的时间节点上，也让这场直播首秀具备了更为特殊的意义。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为直播发起人的快手官方，无疑倾注了相当的热情与期待：从预热来看，开播之前，成龙的快手账号发布了两支视频，以“我们的第一次”为主题，讲述自己7岁出道以来，所经历的或辛酸、或温暖的“第一次” ，以温情直击老铁内心，引发普通用户共鸣。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/71290520162bdaef802dbb1.76254843.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@成龙 快手短视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在将近3小时的直播中，快手的“大众娱乐”底色尽显。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>从流程上，与单纯聊天式的直播不同，成龙的直播首秀采取的是新颖有趣的密室闯关的形式。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在李艾、黄子韬等明星好友的助力下，成龙需要成功通过快手达人设置的种种关卡，这其中既有@俩大叔音乐的功夫梦 守关的“功夫挑战”，也有达人@溜溜蛋设置的“听歌识曲”挑战、脱口秀主播@张踩铃的方言挑战。当第一位守关人@俩大叔的功夫梦现身，发起功夫动作的挑战时，直播间点赞数随即突破1亿。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“老大哥”成龙一面“闯关”，一面向网友们讲述不同人生阶段的难忘“第一次”，闯关过程中，成龙频频丢下“大哥包袱”，从弹幕“偷看”答案的举动引得网友们捧腹；而一直在直播间内陪伴着的粉丝们，也绝不仅仅是个旁观者，他们每个人的投票都决定了成龙是否能闯关成功。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/89976722962bdaef86717d0.79640525.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@成龙 快手直播间截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其他明星们也在此展现出了区别于其他社交媒体的面貌：歌手容祖儿连麦，接受@溜溜蛋听歌识曲的考验，此后，在<span>与成龙合唱《明明白白我的心》时，直播间观看人数冲上400万；</span>黄子韬亲临直播间，被成龙现场教学“叶问蹲”，又将直播间人气推上了近500万；谢霆锋也在连麦时一试身手，将直播间气氛推向了另一个高潮……</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/158304174562bdaef8bd0512.91223582.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>@成龙 快手直播间截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>几位连麦达人也在当晚各显神通，活跃着直播间的气氛：@俩大叔音乐的功夫梦展示了耍铁揪绝活，为自己带来了单日6.3万的涨粉量；脱口秀达人@张踩铃再次展现了自己的幽默口才，当晚直播间总观看人数达到73.4万；翻唱经典粤语老歌的@溜溜蛋也为直播间的人气增长贡献了自己的一份力量。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>整场直播中，</span><span>成龙、明星嘉宾、达人用户、观众粉丝四者之间，一直维持着高频、良性的互动。</span><strong><span>在热闹非凡的互动氛围中，一场集趣味、情怀、爆点于一体的明星娱乐直播被成功缔造。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据第三方数据，当晚直播间内，以北京为首的超一线城市用户占比十分靠前，这与快手一贯以来的地域定位形成鲜明反差；此外，在直播间用户构成上，囊括了6岁到40岁以上的各个年龄圈层，其中18岁-24岁的年轻用户群体占比高达30.26%。<strong>城市和年龄圈层的拓宽不仅彰显了成龙作为武打巨星的强大号召力，也从侧面印证了快手这种趣味“整活式”明星直播间的独特吸引力。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/13363685916010dc4c78b4e6.24352516.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>成龙之外，快手明星生态现状如何？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>136万人预约观看，最高同时在线526万，累计观看人次超2.2亿，直播间互动量超3.2亿，全网总曝光近40亿……即使在近期有多位顶级明星“扎堆”直播的情况下，成龙的首秀成绩单依然格外醒目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从整场活动最终的呈现，不难看出快手从普通用户视角出发、设计各个游戏环节的初心。直播中的各项流程，<strong>都力求带给观众更多的参与感与沉浸感，普通用户能够以“平等”的目光直视明星，明星与用户的距离被快速拉近，整个娱乐氛围不仅可感，而且可触。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为头部短视频平台，快手站内一直不缺明星内容，不过，自2020年10月快手娱乐部门正式成立后，明星生态的布局更有点 “火力全开”的架势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一方面，快手在不断地丰富自己的明星资源，打造包容度更高、更多样性的明星生态；</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>环顾站内艺人，<strong>我们看到既有传统定义上的明星保持活跃，</strong>比如：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>周杰伦（@周同学）在快手开通了自己的首个中文社交媒体账号，积累粉丝超过3000万；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>影帝黄渤（@黄渤）作为进驻快手最久的明星之一，始终维持着高频的内容更新，除了分享自己健身、潜水、做手工的真实日常，也多次参与快手官方的公益活动，充分发挥着自身的影响力，承担起更多社会责任；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手电商代言人黄子韬（@ACE-TAO 黄子韬）在站内的首次带货，就因其“真性情，无套路”的直播方式广受好评，单场GMV突破2.3亿，累计观看达到4000万，直播间涨粉110万；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3585966862bdaef944d110.54980110.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>周杰伦、黄渤、黄子韬 快手短视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有6月6日刚进行了“没啥，就是玩”主题直播的杨幂（@杨幂），在直播间和圈内好友张大大聊闲天、做游戏、玩梗、嗑CP，甚至还即兴演唱了一曲。这种强调明星间的真实化学反应，弱化编排、策划的直播设计，成功将杨幂与好友相处时的自然状态，真实展现给每一个观众。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另外又有更具鲜明快手特色、自带内容生产能力的艺人，因为“接地气”的创作风格，在快手迅速实现粉丝的承接。</span></strong><span>比如像闫学晶（@演员-闫学晶）、谢广坤（@谢广坤）这样自带亲切感的喜剧艺人，他们在快手上定期分享真实生活的点滴，用户对于他们呈现出的如“亲朋好友”般的形象十分喜爱，亲切地称呼他们为“闫妈”“坤哥”，目前两位艺人在快手的粉丝都已突破千万。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同类型的艺人分别针对不同的粉丝群体，创作差异化的明星内容，共同构建了快手的特色明星生态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，在<strong>引入明星的同时，快手也在通过内容策划，积极帮助明星融入社区氛围、进行主动运营，以帮助打破他们和用户之间的壁垒、并形成协同和互动，力求在“平等”与“真实”的前提下，让二者产生更为深刻的链接。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年五一假期期间，快手娱乐携手近30位明星艺人，带来了长达6小时的快手“麦入夏天”线上欢唱直播，一场大型“暖心回忆杀”让很多用户的心得到慰藉。直播过程中，于文华与广大农民朋友连麦，用一首《梅花泪》感谢他们的辛勤劳动；水木年华现场弹唱《他乡》，向爱岗敬业的大厂保安表达自己的尊敬。无论观众们是何种身份、身在何方，都能从中体会到爱与力量，这种“明星为粉丝献唱、素人与偶像连麦”的互动方式，无疑拉近了快手用户和明星之间的距离。</span><span></span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>明星涌入后，</span></strong></span><span><strong><span>快手娱乐想“造”怎样的内容生态？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据《2021快手娱乐营销数据报告》，2021年3月，明星发布视频数对比2020年10月上涨了近60%，快手娱乐内容观看量上涨45.4%，几乎有近一半的快手用户关注了明星和娱乐账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>娱乐内容与短视频有着天然的契合性，对于日活3.46亿、月活5.98亿的快手而言，娱乐类内容显然已经成为其用户增长并寻求持续多元化的必然需求。而快手的社区生态和普惠的分发机制，也使其娱乐内容的产出具有了明显的自生长属性。从明星层面出发的娱乐战略来看，快手的打法呈现出了和其他平台的明显差异。<strong>概括来讲，相较于流量层面的价值，快手或许更加看重内容层面上的价值。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>表面来看，快手找明星站台、打造明星标杆等一系列举措是希望借助明星的影响力来获取更多用户，但实际上，快手在明星布局上的考量要更为深厚。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当前，<strong>用户对明星内容的消费，无论是形式还是调性都产生了大量的新变化，他们越来越追求真实感、信任感与互动感，渴望能够平视而非仰视地参与其中。</strong>因此，对于快手来说，引入明星不仅仅是目的，更是一种连接手段。在靠自身影响力吸引用户的同时，<strong>快手更希望入驻的明星能够“化身”为普通的内容创作者，进行真实的表达和互动、产出具有看点的内容、真正融入社区环境之中，由此在平等的“对话”中满足用户对于娱乐内容的深层次需求。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于独有的内容生态和用户生态，<strong>快手娱乐将自身定义为“人民大众的娱乐”</strong>，不论是鼓励明星自主创作内容，还是对明星IP进行深入挖掘、创作各种PGC内容，其核心价值都是为了更好地找到娱乐内容与平台的契合点，以此消除明星与大众之间的隔阂，并从情感、认同感和消费习惯等方面实现用户口碑扩散。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此基础上，<strong>他们也希望能借助明星产出的内容进一步放大快手上各个领域中原生创作者的价值，盘活快手生态下的每一种内容频道以及背后所映射的生活方式，带动更多的普通用户积极参与到快手生态内容的构建，让娱乐真正走进普通人身边。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着快手不断升级娱乐玩法，未来我们能看到的场景或许是越来越多的明星、达人、用户紧密地“玩在一起”，他们没有身份差异，没有表达阻碍，在互动感极强的娱乐氛围中，每个人的价值都有可能被放到最大。</span></p>
