<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E4%BA%92%E8%81%94%E7%BD%91" target="_blank">互联网</a>大厂希望打造超级<a href="https://www.niaogebiji.com/tag/APP" target="_blank">APP</a>，很少有一家公司只为消费者提供单一服务。过去二十年，互联网公司探索出<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%95%86" target="_blank">电商</a>、游戏和广告这三种变现效率最高的业务模式。在广告业务大杀四方后，抖音和快手这两家短视频巨头，正在加码电商基础设施。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Tech星球独家获悉，抖音再度加码物流服务，抖音电商于近期测试了“极速达”服务，该服务可实现商品同城当日达、周边城市次日达。快手则被曝出在近期成立“成都快送供应链管理有限公司”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位行业人士告诉Tech星球，“抖快”支持的物流公司很多，但他们在服务方面却远不如京东、阿里，因为快递问题被用户吐槽的声音一直不小。而且京东、阿里的电商业务越来越强大，物流可以说功不可没，在见识到电商与物流结合的优势后，快手、抖音在电商业务发展不久后，也开始搭建自己的物流能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>物流是电商重要的基础设施之一，京东物流已经成为其特色标签和护城河优势之一。在不确定性的大环境下，商品的履约交付是衡量消费者体验的重要因素之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着抖音和快手电商业务的深入推进，物流无疑是不可忽视的一环。但是，物流建设并非一蹴而就，为了确定性、及时性的送货体验，京东自建物流为此亏损了十余年。抖音和快手想要搭建基础设施，需要时间、耐心和资金支持。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1170739965612f7c0dc06775.73201401.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>加码物流体系建设的字节</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音正式布局物流源自2020年，一位接近抖音的人士告诉Tech星球，当抖音大力发展电商业务后，抖音高层就已对物流体系的搭建进行了讨论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每天有6亿用户打开抖音，这背后蕴含着巨大的消费力，为了让用户体验到更好的购物服务，从而提升整个电商的业务，物流体系建设是重要的一环。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前抖音电商运营专员李红告诉Tech星球，“将商品高效地送达消费者手中，是抖音电商需要做的重要事情。毕竟，物流是一个重资产的发展模式，对于抖音而言，显然还没能力去做配送、快递等服务，现在，抖音更多的还是做物流中台、物流服务平台。抖音做物流体系是为了打造强大的预测和洞察商货厂能力，帮助电商业务实现供应链优化，最终实现交付必达。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李红还表示，抖音电商物流业务，包含供应链和物流体系（国内/跨境/国际），为抖音兴趣电商生态提供“人、货、场”的能力支持，主要是为了支撑和推动抖音电商业务持续增长。物流团队人员主要分布在上海尚浦中心、杭州余杭区的EFC（欧美金融城）和北京。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年，为了加快物流体系的建设，抖音开始招募顺丰、京东、菜鸟等物流企业的高管以及研发人员，比如从阿里离职的王维、欧立勇、贺川等高P职员，这三人在物流方面都具有丰富的经验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随后，抖音集团在2021年8月成立上海星辰跃动供应链管理有限公司，标志着抖音在国内物流体系的建设初步完成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年年初，抖音联合快递公司针对快递派送不电联（电话联系）、不上门、服务态度差等问题，推出快递服务“音尊达”，该服务的上线可降低因物流原因造成的退货率，并在一定程度上提升用户复购率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>紧接着，抖音又测试了一项重要的配送服务“极速达”，该服务是为实现抖音电商同城当日达、周边城市次日达。比如，交易订单的截单时间为11:00，支持地区为吉林省，则表示配送区域在吉林省且在11:00前下单的需要当日送达，下单时间超过11:00的则需要在次日送达；同理，次日达也按照该流程进行配置。目前该服务还未在抖音电商内全面展开。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/160974007962bdbd134bae68.51331415.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过测试截图可以看到，该服务涵盖了生鲜蔬菜的同城配送，这意味消费者购买的商品，最快当日就能买到生鲜蔬菜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据知情人士透露，即时配送服务不局限于此，后期还将开放更多的商品类目。这意味着，抖音电商依靠“极速达”，可以实现类似于买菜、社区团购等业务的能力支持。此前，有媒体还报道抖音在试水“抖超送货上门”，与京东超市的自营业务相似，可见抖音正在加码在短距离和即时配送的物流布局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>字节跳动的物流服务，不仅仅局限于国内电商。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位TikTok的物流运营专员告诉Tech星球，TikTok正在将物流的相关服务扩张至亚太片区、欧美片区的国际进出口的电商业务上，相关岗位的人员需要掌握电子类等商品的进出口报关流程，确保货物流通通畅。亚太和欧美作为抖音电商布局的重要地区，丝毫不亚于国内市场，TikTok此举是为了完善国际用户的履约交付体验。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2040399738612f7c0ec65369.94912606.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>搭建物流体系，快手也没闲着</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不只抖音，快手也在加码物流体系的布局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年1月快手上市前夕，快手电商营销中心负责人张一鹏在接受21世纪经济报道采访时确认：“快手短期内没有自建电商供应链规划。但这并不妨碍快手着手建设物流能力，特别是今年6月，快手在物流方面有着新动作，即成立成都快送供应链管理有限公司。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>成立成都快送供应链管理有限公司的目的，与抖音成立上海星辰跃动供应链管理有限公司一样，分析人士认为，该公司的成立，是为了更好的服务于电商物流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11668967662bdbd139df156.31345811.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了成立与物流相关的企业外，快手也希望加深和快递公司的合作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年3月，快手电商试水“按需派送”服务。该服务主要是为了提升末端配送的服务质量和体验。快手电商与快递公司达成合作，将通过商家端匹配，在末端配送环节为特定消费者提供按照客户诉求派送的尊享服务，提升消费者物流配送体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随后，快手上线物流管理“包裹中心”迭代功能，提供订单包裹未揽收、揽收后未发出待操作的实时预警（即将超时与已超时），提升包裹履约及时率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位在快手主营生活物品的商家薛勇告诉Tech星球：“‘按需派送’服务推出后的3个月，本店的退货率降低了5%左右，并且好评率也因为该项服务得到了一定的提升。”这从侧面反映出用户对物流服务的体验有较大的需求，也为商家的复购率、订单交易率等起到正向作用。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/519894612612f7c10dcc558.32132111.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>物流不仅是电商基础，更是护城河</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着电商领域竞争进入白热化阶段，物流体系不够完善成为了业内共同面临的问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为此，电商巨头们都在完善物流体系建设。今年更早一些，京东收购了件快递配送服务商德邦物流；阿里同城零售事业群则是打造同城物流履约服务平台“鸟潮”，专注于同城物流网络的建设和服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音快手也在补齐短板。建设物流能力，不仅仅可以完善自家电商平台的供应链、仓配、自提点、商品运营、售后等环节，还可以摆脱第三方的制约。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年，曾有消息传字节收购快递商极兔，虽被证实为不实消息，但如果双方成功联姻，则可以简化自家的物流，完成商品出仓、到打包、再到递送到消费者的配送流程，提高配送效率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>物流服务是电商业务的核心竞争力，对于消费者而言，物流体验关乎着整个购物的体验，间接影响着平台的交易。日前，“快递送货上门是不是法定义务”的话题，冲上了微博热搜引发热议。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这则话题的背后，实际上是用户对商品不能及时送货上门的诟病和吐槽。因此，不少电商平台将该服务当做是竞争力打造“送货上门服务”，抖音测试“音尊达”服务，快手推出“按需配送”服务皆是如此。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅送货上门，短视频平台正在加强对“下单-揽件-运输-派送”等整个物流流程的把控。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不可否认的是，抖音、快手在物流体系建设上，相比于头部的电商平台仍有很大差距。一位物流行业资深人士认为，抖音电商若想建设完整的物流、仓配，一开始就全国覆盖未免成本过大，或会采取区域性战略，先点状突破，再以点带面，后期全面扩张。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该人士也指出，习惯速战的抖音电商，想在物流体系的建设上，取得长期的整体成功，难度并不亚于阿里、京东反向发展短视频电商，但一旦成功，将在物流上不再受制于人，缩短商品配送时间和提高物流体验的同时，也将使各自的GMV从中受益。</span></p>
