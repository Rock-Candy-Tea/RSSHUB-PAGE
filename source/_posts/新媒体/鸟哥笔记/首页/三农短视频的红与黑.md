<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“农村题材”历来是短视频平台的一个热门垂类，在兼具话题度以及新鲜感的前提下，不少农村网红在各平台反复走红。从人人皆知的“李子柒”，到一夜爆红的“张同学”，以及近来刷屏的“二舅”，他们的内容均围绕着中国农村的叙事底色。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其在当前乡村振兴、扶贫助农的大背景下，平台亦在加大对“三农”内容的扶持，这进一步吸引了创作者的涌入，“张同学”便是在此前提下出圈的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但无论是李子柒的诗意田园，还是张同学的接地气记事，以及蜀中桃子姐镜头下的烟火人家，其内容温情与真实并具，用户对他们内容的喜爱体现了大众“审美”的诉求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当“审美”开始走向扭曲，在追求流量的另一端则是“审丑”。在真实与接地气背后，涌现出的也有虚假与炒作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并非所有农村题材内容都是在展现乡村生活的美好与宁静，如今反而是不少人在察觉到有利可图后，开始通过“农村人设”的特殊性制造冲突感，游走在失真、擦边与圈钱谋利的边缘。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">批量制造假农妇</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>衣着朴实、皮肤粗糙、干活麻利似乎是大部分农村人固有的形象特征，由于常年在田间地头干活并经历风吹日晒，因此其衣着形象也是有明显特征的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从李子柒、蜀中桃子姐、麦小登、康仔农人、张同学等农村题材的头部网红身上我们也可以看出，“真实”是其内容的底色。即便是李子柒充满诗情画意的内容，我们也可从大量视频中看出她的确是“会干活”的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/94364165662e993976374c9.61358633.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与之不同的是，如今抖音、快手等短视频平台上正在出现一批批与农村人形象格格不入的农村网红。其中以女性为主，她们衣着靓丽、身材性感、妆容精致，例如唐小鱼、乾美丽、马小花等新晋网红。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说老一批头部农村网红是通过记录真实的农村生活而出圈，那么如今这些性感精致的假农妇则不排除想通过自身精致的外表与固有农村形象的差异制造冲突，以圈流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频创作中，美人、美景、美食等都是热门创作题材，而高颜值的“美人”往往可以给其他一切内容“赋能”，毕竟用户天然不会排斥那些所谓美好的事物。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频由于其内容浅、观看门槛低，其大众化程度已经不言而喻。在抖音、快手“看美女”仍然得到了相当一部分用户关注。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一位有三年直播星探经验的人士告诉光子星球，如今假扮农村人的现象已经很普遍，基本都是机构化运作，而这种现象在各大短视频平台均存在，比比皆是。“真的农村人很难拍出来，很多人看见镜头就紧张不会表现，而且我们日常刷到的一些农村视频，也是可以演出来的，这背后需要公司给你包装运营。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如找几个长得好看的走农村妇女的人设，日常拍一些做饭视频等，都是很常见的。甚至很多拍摄场景都是在棚里完成，“演员们”并没有真正生活到农村。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他同时表示，现在常见的一个现象是，直播时很多农村主播并没有几个粉丝，但是直播间却有成千甚至上万的人观看，这种现象同样是背后机构在操作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于机构如何操作，背后的水很深。套路式的pk、打赏、以及找拖热场等，目的都是在诱导观众花钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当短视频逐渐进入团队化、机构化模式的运营，同时在行业对专业性要求越来越高的趋势下，真正的普通个人已经难以将内容做起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>机构化运营下，账号的盈利目的性因此更强，这与很多早期以记录起家的账号风格相异。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以“唐小鱼”为例，其账号定位为一位乡村达人，视频内容主要与其在农村的爷爷奶奶一起出境拍摄。身材火辣，长相美丽是其个人特征，而她穿着高更鞋与紧身性感衣服与年迈的爷爷一起干农活是其主要看点，同样也饱受争议，而这些所谓的话题或许也正是其团队所追求的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>衣着与动作的不合时宜确实为她招来了一部分看客，短短半年其账号涨粉上百万，如今在快手与抖音双平台运营。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在平台流量本就趋于饱和的情况下，制造话题、寻求新鲜感与刺激，不过是短视频创作者抢流量的一种方式。利益驱使下，假农人被批量制造，农村生活正在失真。</span></p>
<section data-mpa-template="t">
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong>大叙事下的三农直播</strong></span></h2>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有利可图才是很多团队以及网红盯上农村的主要驱动因素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年被视为短视频元年，如今走过了将近五年的时间。众所周知，短视频最显性的一个特征便是迭代快、难长红，从变装、卡点、技术流到亲子、探店、三农等，平台在每一次流行元素更迭时既成就了一批达人，同时也会遗忘曾经的一些顶流。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/106206978162e99397c11c43.37126136.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与其他娱乐性内容不同的是，乡村题材兼具写实属性，更容易共情。其内容贴合大叙事，似乎从始至今都没有经历明显的低谷，“不过时”是短视频创作者求之不得的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乡村题材内容的优势在于其素材丰富，可讲述的东西多，从河渠到机耕道，从植物到昆虫，在用户与平台都需要新鲜感的创作环境下，乡村题材对于内容的持续性创作具有更高的保障。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，从宏观层面来看，短视频正在成为乡村振兴的主要推手，做乡村题材的内容输出必然是在趋势之内。加之疫情影响，年轻人返乡创业、就业，三农自媒体正在年轻群体中快速渗透。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据抖音公开数据显示，平台“三农”创作者中54%为返乡创业青年，其中，31岁至40岁占比最多，达54%；其次是24岁至30岁，主要以农民工、大学生、退役军人、妇女四类人群为主。短视频所带动回乡创业潮正在成为一种趋势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>响应宏观转折，各平台同样在加大对三农短视频的投入，最直接的便是流量扶持。一位MCN人士表示，近两年三农短视频在抖音等平台都有流量扶持，对创作者而言安心做好内容就可以了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以抖音平台为例，近两年先后推出“新农人计划”，增设“任务充电站”与“入驻激励”等扶持措施，吸引更多新农人入局创作。直至张同学的爆火，平台将三农内容推向了爆发的高潮，更多的创作者也开始涌入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>成功案例的背后经过创作者的拆解，形成方法论，很容易被流水线生产。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着互联网的持续渗透，下沉市场同样成为短视频平台争相竞争的最后增量，新农人带动平台内容与用户的双重下沉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上述MCN人士表示，<a href="https://www.niaogebiji.com/tag/%E4%B8%89%E5%86%9C%E8%A7%86%E9%A2%91" target="_blank">三农视频</a>的变现渠道与其他类别的内容差距不大，不外乎直播打赏、带货等方式，这是由平台的商业生态所决定的。若做到头部，还可以开发自己的产品品牌，例如此前李子柒的螺狮粉，蜀中桃子姐的钵钵鸡与酸辣粉等。“很多农村人通过互联网渠道改变自己生活，肯定比务农赚得多。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“无论短视频，还是直播带货，虽然行业现在还有很多乱象有待规范，但可以确定的是它肯定是在趋势内，这也是很多人投入进来的根本原因。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经的农村是很多人都想逃离的地方，如今这种现象正在互联网语境下慢慢改变。</span></p>
<section data-mpa-template="t">
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;"><span><strong>乡村题材与土味的距离</strong></span></h2>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三农短视频成为趋势之际，似乎也没能逃过同质化严重、创新缺乏的窠臼，新农人们清一色的陷入到“做饭”短视频的创作中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李子柒、桃子姐、川香秋月、闲不住的阿俊、恋乡人周周等，走乡村路线的头部达人们似乎离开了锅碗瓢盆就再也打不开流量密码一样。同质化的内容与大体一致的变现模式，很难不陷入审美疲劳的怪圈中。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/206632285062e993982a7039.51826803.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便如此，这些出圈的乡村达人清一色的来自抖音。抖音的机制，能持续制造新人。而快手植根于下沉市场，是最早为农村题材短视频提供生存土壤的平台，但至今并没有成长出乡村顶流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年快手统计显示，对三农题材感兴趣的用户已超过2.4亿，其中超29%的用户来自一、二线城市。快手以下沉为切入点的短视频让不少人在平台被看见，这也因此让其始终无法从泥土中挣脱出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖快截然不同的创作生态，也是二者殊途的重要因素之一。抖音平台主要偏向以MCN团队和机构策划为主的内容输出，其真实人设与内容的占比低一些，在利益驱使下，很容易引来批评。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手平台则更多以真实农村人为主，不过问题同样明显。由于不懂得专业内容输出与运营，内容质量层次不齐。这也是为什么快手平台经常会出现画质模糊、衣着单薄的女主播在直播间讲着略带粗俗的段子求打赏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“真实”是农村题材永恒的生命，尤其是近30年城市化进程中，不少人还留存着关于乡村的记忆。由于大城市竞争激烈、生活节奏快，“逆城市化”虽然还未到来，但“逆城市化”的心理已悄然而至。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，“真实”又不是终极价值，而是希望从农村题材的内容中，寻找淳朴的心灵、原生的环境以及朴实无华的生活，更何况那片田野还是不少人梦开始的地方。</span></p>
