<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从百度<a href="https://www.niaogebiji.com/tag/OCPC" target="_blank">OCPC</a>问世到现在，已经四个年头。曾经的风光无二到现在弃之可惜，嚼之无味。主要原因还是这个ocpc就像一种毒药，优化师们不吃可能马上死，吃呢慢点死。纯跑cpc这种概率广告还能跑出好结果的毕竟不占多数。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而百度针对ocpc的功能又不断在更新迭代，流量更是拉胯得不行，因此流量的质量可想而知。因此ocpc也出现了不少疑问，今天塔哥集中了一批问题，统一给各位同学作出回答。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些数据和概念以及一些优化思路提供一些参考意见。特别是最后5个，塔哥强力推荐，值得同学们认真看完，多看几遍。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>①找出之前效果好的词，对比最近是不是这类词不消费了，是则加一个cpc计划单独跑， 给他排名让他消费辅助保证效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>②如果不是，原有的词正常消费就是不转化，说明你模型开始不再符合现阶段的人群需求，可以开始加关键词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>③加内容分为加词和加计划，加词是在效果波动不明显，把长尾词加进去补充流量，加计划一般是转化成本不断上涨，通过增加计划（大量词）来帮助模型纠正到符合现阶段的转化需求。两者没有明显的界限，都可以加，看自己账户实际情况即可。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>①每周都统计数据，前一周数据对比，看本周数据，预估下周数据，有问题的提前正向操作预防。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>②在搜索词报告里面对一些明显不符合要求的搜索词点击反馈不精准不匹配。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>③新建推广包，开广泛匹配，低转化成本设置。跑一段时间。再复制一个新的计划，开正常匹配，高转化出价。试试，塔哥亲测有奇效。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>④增强模式要不要做，大概率不推荐，百度的所谓智能出价是比不过谷歌的，效果也是消费不转化。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>①对比cpc时期是否为高转化或低成本词，是则保持观察一段时间。历史本来就是高转化高成本词，可以逐步降低ocpc出价来降低成本。反之则停掉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>②ocpc前段时间不活跃，突然消费很高。看转化，转化好保持，转化不好可以考虑停掉。停掉之前可以在按照第一步检查一次再决定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>③如果平时不活跃的词一下子消费很高，ACP很高，历史数据也没有参考价值，预算允许范围内给足时间。可能模型正在学习新的定向，操之过急去动账户，模型会变乱。得不偿失。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>否词之前要做一些数据分析，例如：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>①搜索词是否在历史数据中有转化？转化成本高还是低？分析完了再看第二步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>②搜索词历史数据中心未见转化，但有与之很相似搜索词转化了，且成本较低，则不否这个搜索词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>③搜索词是否在某个特定时间出现，看似不相关，但实际为网民的搜素需求。例如9月份看到搜索词有国庆节的搜索内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>④搜索词的数量是否很多，不建议一个时间段内大量否词。cpc阶段否词完整了再去ocpc则可以。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>搜索的新账户到目前，两者都可以尝试，真的没有标准，ocpc那么多年了，大家对这个工具依然有不少误区。ocpc说好听点就是系统帮你去找客户，实际是让你花更多钱，这是底层逻辑。倘若效果好，你愿意掏更多钱无可厚非。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是有些行业必须是cpc才有效，所以没有标准。有预算的直接两个一起测试。没有预算的，稳妥起见，依然还是推荐先跑3-7天cpc再进入ocpc。所谓的模型会搭建得更完善一些。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里就必须提到一个知识点，做了ocpc的同学，效果还不错的，总会忍不住去看看自己的广告是否有排名，这时候你也会经常发现自己的排名也不错。这里就涉及到，ocpc主动展现给有可能带来转化的客户群体，会根据大数据分析出点击率最高的一批词优先展现，甚至广告位都不多，只展现1个广告给客户。这时候排名的依据就是——价高者得！因此你的ACP必须会很高。这是其中原因之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是这个词要不要暂停，取决于你对这个词消费数据的一顿分析以后才能决定。竞争激烈所以被高价抢了，你的出价成本在同行当中偏高，溢价空间就会被放大。分析词的意向度和这个词在历史中的表现，如果是一直转化的词，次数不多，只能逐步调低转化出价来降低这个。绝对不能一下子降低太多。反之，做个记录，这类情况出现超过3次或以上直接暂停即可。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>跟上面的一些问题差不多，ocpc现在已经收集大量的数据，百度取消了核对数据的功能也是因为觉得大数据收集成规模了，也想告诉客户，你选择ocpc效果不好跟百度无关，是你建的模不符合市场需求。想建模好的，最好还是先跑cpc一段时间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最重要的是新手或者新户，做账户时连哪些词出什么价格可能都不清楚，不同账户最后得出的点击均价都可能不同，跑一段cpc就清楚，ocpc的时候基础更稳固。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二个问题，出价系数目前的规则只针对cpc广告生效。但我需要提醒依据，从规则出发反向思考，如果你cpc时期系数的作用占比很大。说明cpc阶段的效果是这些系数工具贡献的占了大部分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当你换到ocpc，系数失效，你会可能出现ocpc效果不理想，没量，没转化或者转化成本翻车等情况，所以有条件的同学，不妨先观察cpc阶段系数占比再去切换ocpc模式，可能会更好。</span></p>
