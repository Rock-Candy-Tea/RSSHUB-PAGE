                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="1" data-pm-slice="0 0 []">此前，我们重点分享了如何界定好的私域内容：让客户收到内容推送后有触动，有行动的，就是好的私域内容。同时我们也全面地分享了企业该如何从目标客户的痛点、嗨点、热点出发准备客户感兴趣的私域内容。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="2">搞定私域内容准备后，本期我们走到了<strong>【私域活动】</strong>这一环节，将与您共同探讨如何策划有效的私域活动。</p>
<h2 class="pgc-h-arrow-right" style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" spellcheck="false" data-track="3">一、私域活动第一步：定目标</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="4">比起敲定出一个动人的GMV数字，我们在私域里策划活动的第一步，应该是拆解出本次活动希望获得什么样的目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="5"><strong>当下规划这场活动，是希望获取私域用户增长？还是筛选出关键客户进行促活、进行转化？亦或是简单的品牌宣传？</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="6">我们希望所有私域操盘者与企业管理者在推进一场私域活动前，一定要把目标想清晰，把重要的决策问题想明白，要知道，战略上的懒惰是无法用战术弥补的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="7">一场有效的活动一定是伴随着明确的活动目标，才能拆解出活动框架、所需资源、关键数据指标等信息，如果开头没想清楚，后面执行到一半发现推进有问题、效果不如意，需要推翻原来的方案从头来，那就是一次财力、人力的双重浪费。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="8">比如想要加粉，那么就需要系统盘点下当前企业有多少渠道，又各自有多少流量，需要谁来加粉、加到哪里，权责利如何划分，活动的数据埋点如何设计，客户身上需要结合活动打上什么标签，后续该如何针对此次活动的客户进行运营。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="9">比如想要弥补此前疫情影响，在季末冲刺销售额，那么所有活动动作的目标都很明确，一切以下单转化为中心，所有动作的设计都是为了达成更高的转化，那么其他如加粉、活跃等过程指标都可以先为转化让步。</p>
<p class="pgc-img" style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://mp.toutiao.com/mp/agw/article_material/open_image/get?code=ZjVmZDM5MzNhNDI4Nzg4MWFiNjAzODlkZTdlZTZjZDcsMTY1NzE4MjA4OTIzMQ==" data-ic="false" data-width="1080" data-height="720" data-ic-uri referrerpolicy="no-referrer"><span class="pgc-img-caption"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="10">除了明确目标外，在私域活动策划时，我们很怕一种情况：为了做一场活动，而做一场活动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="11">这听起来很不商业，甚至有点稚气，但却真的是一些老板做活动的理由：“快到父亲节了，搞个促销群发吧”，“我总能收到别家每周的优惠推送，我们也来吧”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="12">这是很真实的内部对话场景，<strong>但这样的活动真的没必要做，尤其是在私域活动这个讨论范畴内。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="13">私域里的大部分客户都是真的认可产品或服务的，因此比起用短期流量思维一波又一波地用活动进行收割，企业管理者需要在私域活动策划时重点思考，如何让私域客户池发挥出更高价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="14">要知道，一场成功培育关键客户的活动，一定是有可复用经验的，甚至可以作为长期活动定期举办。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="15">我们希望企业管理者在推进私域活动时是真的思考好了，不是一时兴起，也不是眼红友商，而是出于对自己企业私域、当下业务数据的调研，决定了一个明确、可推进的活动目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="16">比如在结合数据分析了客户的成交周期与产品使用周期后，发现近半年内未下单的客户刚好产品消耗得差不多了，因此可以针对这一群体进行促活-转化，刺激客户复购，同时锁定产品高度认可的潜在关键客户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="17">而刚刚提及的借助节日氛围举行活动，在策划时可以作为活动的某种包装形式，但绝不应该是活动的目的。</p>
<h2 class="pgc-h-arrow-right" style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" spellcheck="false" data-track="18">二、好钩子，才能出真成绩</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="19"><strong>好的活动钩子，是让客户收到信息后，真的产生了到店或下单的动机。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="20">所有策划、参与过活动的职场人都对钩子的设计并不陌生，比如现金奖励，比如优惠券，比如随机礼包，比如实物赠礼等等，都是最常见的钩子。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="21">例如最常见的拉新裂变活动，老客户拥有1次抽奖机会，每新增1个好友即可多一次抽奖机会，每位新好友也可获得一次参与抽奖的机会，以此吸引客户转发，形成裂变涨粉。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="22"><strong>拉新裂变活动的基本逻辑是以奖励作为驱动力，利用客户社交关系链进行分享拉新，</strong>基于此，设计活动细节。如客户行为-转发给好友，好友和Ta均可获得一次【抽奖福利】，因此客户不是单方面的打扰好友、来请好友帮忙，而是一次互惠互利的友好分享，彼此都有收获。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="23">在私域里的活动钩子，要像上述逻辑，重点围绕目标客户的特点进行设计，<strong>在奖励价值上满足客户实际需求，在执行成本上符合客户的身份，在操作上符合客户直觉。</strong></p>
<p class="pgc-img" style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://mp.toutiao.com/mp/agw/article_material/open_image/get?code=NDEzOGQxZDU4OTE1NWEzYzEzNWRiNzJmZThhYTQ5NjYsMTY1NzE4MjA4OTIzMQ==" data-ic="false" data-width="1080" data-height="720" data-ic-uri referrerpolicy="no-referrer"><span class="pgc-img-caption"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="24">此外，钩子的设计，一方面需要你了解客户痛点，另一方面需要你对人性有基本的洞察。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="25"><strong>喜新厌旧、失去比获得更加痛苦是人的基本情绪，而下单前的犹豫基本上是对价格成本的顾虑，</strong>那么品牌就可以结合这些洞察，向价格敏感型的老客户推送【新品+限时优惠】活动，比如服装品牌可以推出活动：【24小时内老客复购85折、昨日衣衫配不上今日的你】以优惠和求新两个点作为钩子，吸引客户浏览当季新品、付费下单。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="26">除了刚刚提及的两种情绪，人也天然是希望接受最简单信息的，那么活动的钩子就要简单直给，不要花里胡哨，除非是希望过滤掉一部分特定客户，不然也不要采用可能会有理解偏差的表述，如果客户看到了你的活动海报，还需要研究活动规则，那么大概率客户会觉得Ta很难获得奖励，甚至觉得你在骗Ta而拒绝参加活动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="27">最简单的设计比如现金奖励，是下单立减，还是下单后返券，还是签收后咨询客服领取，一定要在设计时就用最简单的话术把事情讲明白了，一看就懂的那种。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="28"><strong>一方面简单直给的规则便于客户理解、参与活动，另一方面也可以让其他参与活动的同事了解活动规则，方便同事评估如何推广此活动。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="29">总的来说，活动的钩子一定要吸引人，配合一看就懂的活动规则，让客户以最低决策成本、甚至不用思考，直接参与活动，才能获得好的活动结果。</p>
<h2 class="pgc-h-arrow-right" style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" spellcheck="false" data-track="30">三、选好工具，提升的不止效率</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="31">在私域里做活动，工具的选取是十分重要的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="32">比如刚刚我们说在确定活动目标时，需要设计活动埋点、规划客户标签。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="33">如果缺少了这部分工具支持，客户访问没有埋点，根本不知道主活动的页面有多少PV、UV，哪些是老客户、哪些是新客户。或者是缺少了标签追踪支持，后续在私域池里找不到这批参与过活动的客户，甚至不知道客户是通过线上活动加粉，还是通过线下门店而来的自然流量，不了解私域池内的客户构成，精细化运营就更无从谈起了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="34">目前，尘锋的系统工具不仅支持基本数据统计、行为标签追踪，还可提供抽奖大转盘、九宫格抽奖、优惠券、礼品卡、营销红包等私域活动常用的活动工具，您可以到尘锋后台体验相关功能。</p>
<p class="pgc-img" style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://mp.toutiao.com/mp/agw/article_material/open_image/get?code=MDYwMTUzOWIxZWQ2M2JhMTEzZGY2YzUxMjM0ZjQ2NzMsMTY1NzE4MjA4OTIzMQ==" data-ic="false" data-width="1080" data-height="628" data-ic-uri referrerpolicy="no-referrer"><span class="pgc-img-caption"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="36"><strong>只有依托好用的工具，活动才能有数据支持，</strong>这已经是私域活动的基础构成。除了上述提及的明确的活动目标、清晰好用的活动钩子、专业的活动工具，想要做好一场私域活动，还需要依托好的活动创意。</p>
<h2 class="pgc-h-arrow-right" style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" spellcheck="false" data-track="37">四、好活动，依托好创意</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="38">最近，“没别的，就是希望您开心”的饿了么的免单1分钟活动成功刷屏社交网络，全网都在猜测下一个免单时间，要知道，餐饮行业“免单”可算不上创新，毕竟大众点评的霸王餐、餐饮门店的挑战变态辣等活动都是基于免单逻辑设计的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="39">为什么饿了么的免单能够收获不俗的讨论声量？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="40">我们简单拆解了饿了么这次免单活动的逻辑：1、基于简单数学逻辑，如圆周率等信息设计活动规则，参与门槛低，消费者互动体验好；2、中奖免单全额退回饿了么账户（限额200元），活动钩子简单直给，消费者奖励明确，动机充分。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="41"><strong>这场活动的设计确实称不上创新，但确实是一个好的创意。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="42">饿了么的目的很明确，吸引消费者到饿了么而不是其他外卖平台消费，退回来的金额也是退回饿了么账户，消费者需要二次下单才能享受奖励；且活动选取的时间也很妙，6月下旬，网友们刚刚经历过高考作文、中考几何的回忆热潮，对于做题解谜并不陌生；此外，活动的钩子也很巧，以小搏大的“搏一搏薅羊毛”心理十分符合人性。在此基础上，配合节奏得当的宣传，这次免单活动成功多次挤爆饿了么的下单界面。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="43">在此也额外分享一个尘锋自己的活动案例，我们在去年中秋节搞过一个限量月饼祝福盲盒的活动，目的是促进我们客户与销售之间的交流。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="44">活动的路径很简单，销售会向客户发一张活动海报，上面有6个关键词，客户任意回复一个就会开启一个小的盲盒，这个就是活动的钩子。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="45">流程也很简单，客户回复关键词就可以参与活动，我们的销售会给客户发送Ta获得的盲盒的链接，这里使用的是我们的营销素材，客户是否点开查看我们也都能了解。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="46">这个活动的目的很清晰，客户路径也很简单，而且效果也很好。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="47"><strong>这就是一个很典型的基于清晰目的去设计活动钩子、策划玩法再包装创意的一个过程。</strong></p>
<p class="pgc-img" style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://mp.toutiao.com/mp/agw/article_material/open_image/get?code=YmQ2MmQ2YWIwNjBmZmQ0MDBkMmFmNjMxMjlkMmMwOGIsMTY1NzE4MjA4OTIzMQ==" data-ic="false" data-width="1080" data-height="870" data-ic-uri referrerpolicy="no-referrer"><span class="pgc-img-caption"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="48"><strong>好的活动创意弥足珍贵，但不必把活动创意想得过于复杂，有时候客户洞察的简单呈现，也能获得不俗的活动效果。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="49">除了上述活动策划的技巧，我们也希望管理者明确一件事，所有可持续的私域活动的基本逻辑是实现与客户共创价值，一定要关注私域客户的体验感和获得感。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="50">很多企业管理者和私域操盘手都很偏爱活动，相信活动可以直接带来数据的提升、带来GMV的转化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="51">当然数据很重要，但其实在私域的这个讨论语境里，<strong>活动还有一重十分关键的价值：维护客户情感。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="52">我们经常在分享私域加粉、引流等路径时提示管理者要注重客户体验，其实，在规划一场私域活动时，不止要关注客户的最终体验，更要关注客户的在活动过程中的直接感受，关注客户的参与反馈。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="53"><strong>在活动的基础上，让企业品牌与客户建立起更亲密的关系，私域活动才算完成了高阶使命。</strong></p>                        <div style="height: 1px"></div>
