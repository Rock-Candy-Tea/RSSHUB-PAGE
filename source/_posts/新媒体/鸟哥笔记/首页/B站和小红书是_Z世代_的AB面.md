                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在《创新者的窘境》中，有一个概念叫做“价值网络”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">初始用户和初始网红，决定了最初的价值网络，成为了产品的服务对象<br><br>于是，每个平台，都会向更友好于自身价值网络的方向发展，只要把价值网络服务好！吃透！占满！平台就会进入快速发展期</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是，友好于一个价值网络的同时，也就失去了对其他价值网络的兼容</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">于是，每个社媒平台形成了自己独有的社区风格和圈层</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如<a href="https://www.niaogebiji.com/tag/B%E7%AB%99" target="_blank">B站</a>和<a href="https://www.niaogebiji.com/tag/%E5%B0%8F%E7%BA%A2%E4%B9%A6" target="_blank">小红书</a>，主要用户人群都是<a href="https://www.niaogebiji.com/tag/Z%E4%B8%96%E4%BB%A3" target="_blank">Z世代</a>，但两个平台就好像Z世代的AB面</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在B站要有趣，要鬼畜，要突破传统；在小红书要审美，要格调，要生活方式 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">Z世代的小朋友们，在B站哈哈哈，在小红书岁月静好 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，在B站，品牌不能端着，但在小红书，品牌还就得端着 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们到底为啥要如此关注Z世代？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">Z世代已经不是小孩了，而是已经占据整个消费力30%的主要人群，根据AppAnnie的数据显示，Z世代人口现已占据全球人口的三分之一，超越了千禧一代(Y 世代)，成为人口数量最多的一代</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">放眼中国市场，Z世代人口规模已接近3亿人，2021年撬动了超过5万亿的消费空间。ta们对于对于消费品的消耗频次是最快的，所以Z世代让很多新消费品牌崛起的非常快，可以说，Z世代就是新消费品牌的“财富密码”</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">Z世代的A面：B站</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">B站的用户和品牌之间，是一种微妙的互惠方式 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为B站的圈层文化几乎是几个主流社媒氛围最重的平台，这种文化氛围带给B站用户一种“自豪感”。面对其他圈层，B站用户最有勇气说"你们不懂" </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，面对品牌账号的内容，B站用户会反感硬广，但对“B站化”的内容包容度很高，因为在这类的内容的背后得到了群体性尊重，因此不反感甚至为此骄傲，主动传播 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以如果能达到精神层面的互惠，品牌在B站做内容也是可以被广泛传播的</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">拿中国联通举例 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2019年7月，中国联通官方号发布了它的第一条视频《活了这么久，居然不知道1G是多少兆》，虽说是搞笑内容，但笑点平平，所以最终效果扑街，被B站用户评价：在努力搞笑但是不好笑</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">直到2019年12月15日，联通发布了一条名为《联通姐妹花，猛男新宝岛》，结合热点进军宅舞，拿下近40w视频播放量，找到了符合b站年轻人的定位</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2019年12月26日，正值“抖肩舞”火爆全网，联通也顺势推出了中国联通版抖肩舞，连营业厅的保安大爷也入镜参与了一把。贴合业务场景的拍摄剪辑和到位的表情管理，让这条视频一下收获了200万+的播放量，成为一个现象级爆款</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">随后招商银行、中信银行、中国移动等一众蓝V纷纷效仿，宅舞、女团舞接连不断</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">也是从这时开始，联通成为飘在几乎每个蓝V热门视频上的传说，你未必听说过属于它的故事，但你一定见到过那句密密麻麻刷屏的：感谢联通</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">理解Z世代这件事， 不可能有人比联通更懂：指哪打哪，绝不跑偏</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">到今天，中国联通在B站收获66.5w粉丝，如果一定要说中国联通赢在哪里，那大概就是精准的戳中了B站年轻用户的“情绪”，让B站用户感受到了【互惠】和【群体性尊重】 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很多品牌在B站做内容，有点不知所措，拿捏不好力度，其实有三个关键词可以做参考，分别是：“情绪”、“魔性”、“跟风”，并分别对应着：关注评论、主动分享和二次创作，它们正是品牌和B站用户沟通时最好的沟通策略</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">Z世代的B面：小红书</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小红书和B站的主要人群都是Z世代，但社区调性完全不一样</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小红书是美好乌托邦，城里的人，有很多种不同的生活方式，但都有几个共性：励志、积极、向上、岁月静好</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">看起来好像开放、多元、包容，但一切的前提是美好，社区氛围会被强有力的管控，一切不美好，不积极、不向上的内容会被一刀切和长期打击。美好的社区氛围是小红书的社区基调</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里先给大家科普个概念，模因</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">它指代一个想法、行为或风格，从一个人到另一个人的传播过程。源自理查德·道金斯于1976年出版的《自私的基因》一书，是文化资讯传承时的单位</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们可以将其简单的理解为「文化中的达尔文进化论」，它讲究文化中的优胜劣汰。一些强传播的文化现象，比如「地铁上大家都在刷梦华录」，我看了梦华录认为很有意思，就向朋友推荐。之后我们之间会就这个内进行讨论、甚至再创作。这里，被「梦华录」的内容所影响，感兴趣这个内容的人迅速聚集成一个高频互动的「社区」 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果说B站的模因是圈层文化，那小红书的模因大概就是精神文化 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里举例一个国产香薰品牌，观夏</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">观夏以「独处场景」切入，通过打造“我独自生活”的栏目，让消费者讲述自己的生活故事和态度</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在观夏的笔下，这些女性来自不同的行业，有卸下金融女标签转行创业的女孩，有Vintage店的主理人，有放下如鱼得水的工作去英国学习策展的杂志专题总监</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在观夏的解读中，这里的「独自」不是物理意义上的独自一人，而更多代表的是一人如何面对和自己相处的时光和生活。观夏想要带给用户的力量是，在独自相处的时光中也能发掘到生活的闪光</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">整个品牌栏目无不围绕一个主基调：生活仪式感</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种很正向又带有理念的表达正正好好戳中了小红书用户的心巴，把内容拿捏得，简直像一朵让高岭之上的白莲花。（此处双押，skr~）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以观夏把小红书作为主打阵地，也正是因为平台用户特性与品牌调性高度吻合</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">换句话说，对于产品功能本身，小红书用户更加看重的是品牌的价值观、品牌的时尚态度和品牌的审美价值观</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而观夏，都踩中了</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">写在最后</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">Z世代的内容圈层是高速流变的，喜好潮流变化很快难以把握，想要跟Z时代人群有效沟通，就必须理解ta，加入ta，为ta注入价值，找到适合自己的切入点投身其中，深度而动态地捕捉他们所关注的价值，跟随价值和相关圈层的发展随机而动</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总之不管是哪一面，不管喜好怎么变，用真挚的内容和细腻的洞察引来做内容营销，永不犯错</p>                        <div style="height: 1px"></div>
