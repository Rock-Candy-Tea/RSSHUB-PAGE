<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>半夜10点，东方甄选的抖音直播间里，一个脸型方方的主播正在带货。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他从大兴安岭聊到星辰大海，从原始森林聊到中国历史，评论区有人提醒，过去一小时粉丝又涨了40万！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>台下导演提示：书已经卖了50000册了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时主播才想起来还要带货，赶紧说回要卖的书：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>《额尔古纳河右岸》</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我在读到这本书的时候，第一次充满了对力量、对生命、对森林、对流水、对落叶、对日月、对清风、对苔藓、对起舞的萨满、对夜里的月光，由衷深沉、不知所以但一往情深的爱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们吃着简单的食物坐在星空下，围着篝火喝酒聊天，他们会带着孩子去捕鱼打猎，他们也会与爱的人在落日的山巅，欣赏秋风与红叶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接着他又说道：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我前几天躲在家里不方便出门的时候一晚上就看完了，看完后觉得不够，睡起来再看第二遍，我热血澎湃，我现在就想去一次大兴安岭，我想去看一看那里的山林，我想感受那里的清泉，我甚至想去看一看苔藓长什么样子，我想摸一次驯鹿的角，我想跟那里的人有一次亲切的交谈……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我想带你去品尝大兴安岭深处、清晨沾满晨露，表皮一层白霜的野生的蓝莓。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后猝不及防拿出一箱“有机野生蓝莓汁”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我想把一些好的东西都给你，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>譬如朝露，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>譬如晚霞，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>譬如大江大河，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>譬如6月21号达到北极圈的太阳，北极达到极昼，太阳终日不落，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如我对你的爱一样坦坦荡荡，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以我还做了非常详细的各种农残和药残的检测，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以请大家可以去放心的购买。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后“有机野生蓝莓汁”又卖了27000单……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>▼</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>眼前这位正在带货的叫<a href="https://www.niaogebiji.com/tag/%E8%91%A3%E5%AE%87%E8%BE%89" target="_blank">董宇辉</a>。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据说他的成名作是这段话：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家在六月的清晨点进直播间，看着我像兵马俑一样的脸型，感受到了人生的无常和命运的不公，但这时你发现和我一样脸型的还有这款299元牛排送你的这口锅，这个锅你得背。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据说当时的直播间里，有80000人同时在线，而自称脸像兵马俑又像牛排锅的董宇辉长这样：</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/210673882862aa2ec6680eb0.38181814.jpeg" width="333" title="新东方的带货文案，直播界的No.1" alt="新东方的带货文案，直播界的No.1" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实我觉得他还是蛮有明星相的：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>像周杰伦+廖凡的结合体。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在董宇辉的背后，是俞敏洪带队新东方转型直播推荐好物的平台<strong>「东方甄选」</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众所周知，新东方经历了一段难熬的时光：公司市值下跌90%，营收减少80%，员工辞退6万人，退学费、员工辞退N+1（赔偿）、教学点退租等现金支出近200亿元……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>断腕求生的新东方，在去年12月进军直播带货领域，用半年时间，在不花一分钱买流量的基础上，把粉丝量从0艰难做到了100万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月初的时候他们还在苦苦挣扎，但接着就凭借<strong>#双语直播带货#</strong>冲上了热搜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而要说真正的出圈，还是前面6月10日董宇辉凌晨卖牛排的名场面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之后便是一发不可收拾，从第一个100万粉用了半年，到第二个100万只用了3天，再到昨晚一小时涨粉40万，现在粉丝量已经直逼700万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>熬过了无数个无人问津的夜，东方甄选终于成为了抖音直播带货的顶流存在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有观众评论说<strong>“直播的天花板，我觉得其他直播都好俗”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一夜爆火的背后，是这些前新东方名师们的才华被世界看见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如董宇辉昨天很难过的表示：自己以<strong>#董宇辉 才华#</strong>登上了微博热搜，导致很多学生来质问他：你为什么要骗我！说你是帅哥！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/72768917062aa2ec6c6ae57.44458175.png" width="333" title="新东方的带货文案，直播界的No.1" alt="新东方的带货文案，直播界的No.1" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在董宇辉直播间里，前面那些散文诗一样的带货文案，他可以连续3小时无间断输出，起初以为有台本，结果直播间连提词器都没有，全靠主播一张嘴，就让导演补货跑断腿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些张口就来的带货说辞，很多都有着优秀文案的特质：<strong>富有张力的排比句式，充满洞察的场景描写，还有直指灵魂的问心。</strong></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>01</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>雄浑有力的排比句</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在卖《DK博物大百科》时，他说道：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们无法真正去踏上每一片土地，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无法牵孩子的手去看每一处风景，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无法听每一个诗人讲他们的故事，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无法听每个科学家讲他的发现，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们的生命太短暂了，没有办法去体会这一切，</span><span>但请记得，你一定可以找到一个无所事事的下午，</span><span>就像你好多年前放学后无所事事的下午一样，</span><span>陪着孩子翻着一本书，聊一些童年成长的故事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当翻到书中的“岩石篇”的时候他说：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天体行星就是浪漫，我们是宇宙星辰的孩子，我之前给大家讲过构成我们身体的元素，跟脚下这座蔚蓝色的星球一样古老，甚至比他要更为古老。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如我们身体70%都是水，组成水的氢来自于大爆炸，</span><span>比如说我们所吃的米面其实本质上都是碳，碳来自于恒星内部的核聚变，</span><span>比如说我们身体里有铁铁元素，取出来可以是一根7厘米长的钉子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以从这个角度去说，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们与岩石无异，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们与花草树木无异，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们与飞禽走兽也并无二异，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此人要亲近自然，大山大川才能让人内心平静</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只有你带孩子出去一趟，你才会发现它对这些东西，是有如此多的疑问和爱好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然后边翻书里的动植物图片边说：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它可以是一株植物，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它可以是开放的一束花，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以是草丛里爬过的一只昆虫，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也可以是天上飞过的一只雄鹰，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它可以是任何东西，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它可以是此时直播间里一个长相奇怪的男主播，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它也可以是2022年6月10号下午那一刻，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阳光的角度和吹过窗户的风。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当翻到“蝙蝠”这一页他说：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大家可能会觉得丑，但我会告诉孩子万物皆有他的审美，我们不能用自己的标准去衡量别人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果世界上只有一种审美，那便没有了美；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果世界上只有一种对，那便没有了对；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果世界上只剩了成绩，那便没有了孩子呀。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>充满洞察的场景描写</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在卖“大兴安岭野生蓝莓汁”时，他说道：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那天第一批我们自己吃的样品到了，我喝完一瓶，然后坐在中关村的出租屋里燥热，楼下吵闹，北四环的车流从来没有因为我的忧伤或者是兴奋而停止过，楼下还时常在夜里打电话争吵。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但那一刻我坐在中关村租住的小房间里头，我的灵魂已飘向远方。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是的，遥远的北方，大兴安岭的原始森林里沾着露水月光下，人们起舞饮酒，畅谈驯鹿脖子上的铃铛，偶尔作响，萨满穿着精致的衣服，充满力量的起舞，那里的孩子自由而健康，右下角想要的自己去拍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在卖《三体》的时候，他说的是：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你想象一下朋友，组成你左手的原子和组成你右手的原子，可能来自于不同的恒星，只要想到这一点这就已经足够浪漫了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以我们就是星辰，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们生于星辰，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们也终将归于星辰，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们是星辰的孩子。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>《三体》这套书讲的就是人类要如何去在宇宙星辰中寻找下一个家园。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>03</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>直指灵魂的问心</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在卖《藏着地图里的中国历史》的时候，他说道：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们也在努力的活着，有贷款、有房租、有压力、有爱的人，有脆弱的感情。大概就在10天前我们还在苦苦挣扎，那天晚上有1300人陪我聊到天亮，我说我们是朋友对吗？我们能聊聊天吗？你们为什么睡不着？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时我突然才知道的有那么多人在夜里，原来跟我一样辗转难免，有没考上研究生还没有找到工作在家里焦急的孩子，有期望在大城市打拼的孩子能回来看看自己的老人，有孩子尿床哭醒了自己收拾完一切坐在客厅里想喝一杯水冷静冷静的母亲，有半夜开着车回来大半夜还在路上的中年人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>——众生相。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前天晚上我跟宋老师播完之后，我俩走到楼下，有几个人在天桥底下睡觉，前几天下那么大的雨，我其实特别想帮他们，但我又不好意思打扰他们。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>——那才是众生相。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以什么叫做<strong>《藏在地图里的中国历史》</strong>？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>城堡会泯灭在长河中，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>英雄会老去，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美人会迟暮，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你我的故事终将化为一炬，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是平凡的人会给我最多感动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我更多的精神力量不是从那些大家听起来很壮阔的故事中找的，反而是我身边真实的一个个朋友们，七情六欲，五谷杂粮，但可爱如你我。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我希望六月的阳光下，你我光明磊落的站着，大大方方，热爱生活，慷慨、乐观、自信。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有困难是对的，因为月有阴晴圆缺，人有悲欢离合嘛，苏轼都给你说了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>▼</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真正动人的东西不需要尬吹，评论里有人说：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“进了直播间就出不去了”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当他扯远了的时候，下面观众会互相提醒：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“想买货的自己下单，想听课的认真听课”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有人愤愤表示：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“努力跟着刘耕宏跳操瘦下来的肉，让新东方董宇辉的大米和牛排又抹平了”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但最好玩的还是这位微博博主<strong>@肉很多的王加一</strong> ，连续四次在直播间里因为董宇辉不同的说辞买了四袋大米……声称人生30年，从未这么离谱过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/69720545462aa2ec72ae671.60047388.jpeg" width="333" title="新东方的带货文案，直播界的No.1" alt="新东方的带货文案，直播界的No.1" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来董宇辉说看到上面那张图传的到处都是，表示这样不太好，咱吃多少买多少，你吃完再来买我给你现磨。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>▼</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>走红之后，董宇辉接受了《GQ实验室》的专访，他对最后一个问题的回答，让人泪目：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>「此时此刻，你最想实现的愿望是什么？」</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他回答：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，我还留着大家都在的群。我一直忍不住想，我留在这里，聪明一点、勤奋一点、皮实一点，努力一点，把这里做得好一点，能养活更多的人，</span><span><strong>有没有可能再见到他们？</strong></span></p>
