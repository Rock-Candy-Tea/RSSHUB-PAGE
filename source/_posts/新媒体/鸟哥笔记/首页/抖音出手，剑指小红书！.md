<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io">‍<span><strong>兴趣<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%95%86" target="_blank">电商</a>时代，种草正在成为巨头们的鏖战之地。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E5%B0%8F%E7%BA%A2%E4%B9%A6" target="_blank">小红书</a>的风生水起，使得互联网大厂纷纷涌进这一“千亿”赛道。尽管各家平台属性和业务结构不尽相同，但从底层逻辑来看，都是种草和电商间的双向奔赴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在新草APP折戟后，沉寂许久的<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3" target="_blank">抖音</a>也再次向种草领域发起冲击。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>抖音上线全新种草APP</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，有消息称抖音将上线一款名为“可颂”的种草APP。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知情人士透露，“可颂”定位为“一个年轻人生活方式社区”，slogan为“年轻人新生活方式”。在今年5月份，抖音APP的开发者也注册了名为“可颂Android软件”的软件著作权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/149157598062ca197f0d2349.71914764.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于该软件的具体上线日前暂不确定，字节方面也未对该消息作出回应。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过消息人士称，可颂将依附于抖音生态，为抖音电商、本地生活等业务形成协同和服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，这是抖音在种草领域的又一次尝试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在2018年，抖音便曾推出“新草”APP，对标小红书。但可惜的是，当时的抖音还未明确“兴趣电商”战略，再加上流量红利消失殆尽的市场环境影响，在上线一年后，新草便匆匆下线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>如今“可颂”的推出，则正处于抖音全域兴趣电商的爆发期。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在去年的首届抖音电商生态大会上，“兴趣电商”这一概念被正式提出。今年，抖音将其正式升级为“全域兴趣电商”。它的核心，便是区别于传统 “人找货”的“货找人”模式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/184554342062ca197f5a74a6.01860659.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这一模式确保成立的基础，则是平台内能够激发用户潜在购物兴趣、满足人们对美好生活向往的优质内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>所以最后跌跌撞撞，抖音电商还是要走向“种草”的道路。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年，抖音正式上线了图文种草功能，开始将其作为体系内一个重要方向发展。同时，也旨在与平台其他玩法配合，实现“直播+短视频+图文”的种草生态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之后，抖音又尝试将短视频的“点赞”按钮更换为“种草”按钮，并在短视频中加入完善“笔记”和添加“宝贝”功能，这可以认为是抖音在种草道路上的一次重大尝试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/114144390562ca197f9afe89.73428796.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时至今日，抖音平台内的种草框架已经基本搭建完成，但抖音并不能变成小红书，所以需要一个额外的全新平台，成为对标小红书的独立种草生态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这便是“可颂”的由来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>大厂种草，意在电商</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除抖音外，各大巨头也纷纷布局种草业务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相关数据显示，自去年开始，各互联网大厂先后推出自有的种草产品/功能达16款，如腾讯种草电商平台“小鹅拼拼”、阿里种草电商APP“态棒”和美食种草APP“吃货笔记”、快手新潮分享平台“避风”等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>在种草赛道上，各巨头都希冀着能够再造一个小红书。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管这些大厂属性和业务不尽相同，但正应了那句话——互联网的尽头是电商。</span><span><strong>大厂的入局，种草只是手段，电商才是目的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小红书，已经为他们打了个样板。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81141696762ca198060ecb2.01751310.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为“种草鼻祖”的小红书，是率先将种草应用于电商的平台。2014年，小红书正式上线电商平台“福利社”，完成商业闭环。不到一年，福利社销售额便达到了7亿元。之后，又上线了自营保税仓，希望将自身打造成全球最大的社区电商平台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在今年年初，小红书将社区部和电商部正式合并成新的社区部门，对于这一组织架构调整，小红书对外表示：用户的交易心智要在社区养成、商家的交易生态要在社区生长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，无论是对于用户，还是商家，小红书都将重点放在自身的种草社区上。这也意味着其确立了“种草服务于电商”的发展方向，并对种草到电商的链路进一步加强。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互联网各大厂的入局，也都抱着同样的心思。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是老牌电商巨头阿里、还是短视频巨头快手，其种草平台当发展至一定阶段时，均开始加入电商属性，寻求变现，最终成为一个“种草电商平台”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/57963199262ca19815d2d42.55950977.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>淘宝更是将种草功能直接加进了首页的一级入口，上线“逛逛”功能，主打站内种草。其平台slogan也由“随时随地，想淘就淘”升级为“太好逛了吧”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>简而言之，大厂种草，意在电商。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>种草的核心在“内容”</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>然而巨头们的纷纷入局，非但没能撼动小红书的龙头地位，反而自身一个个地走向衰落。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯的小鹅拼拼、阿里的躺平（友啥）、快手的避风社区均是如此。这几款产品也都曾有过自救，但最终还是难逃死亡的命运。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>归根结底，在于这些互联网大厂还未抓住种草的核心。小红书，也有着自身的护城河。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>仔细研究小红书的发展就会发现，它的崛起最初来自于内容社区的分享。彼时的用户来到小红书，并非是为了种草，而是分享生活。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/151711871262ca1981916c87.46579456.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正是在这样的氛围下，用户的生活分享、好物分享才逐步构建成为一个弱商业气息的种草社区。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果种草社区一开始便是各种隐性广告和带货链接，那么消费者强烈的抗拒心理将会使其直接走向覆灭，这也是为何许多大厂做不好种草平台的原因之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而另一个原因在于“内容”，这也是种草的核心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于在种草领域深耕多年的小红书而言，尽管在流量、技术等方面弱于互联网巨头，但却拥有者一批忠实的、愿意分享种草体验的达人们。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些达人帮助小红书产出足够优质的种草内容，进而促使其他用户也进行产品的使用体验分享，共同搭建了小红书的内容护城河。而用户也习惯于在小红书进行浏览、种草。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于其他平台而言，并没有这样的氛围和内容优势，甚至对于淘宝、京东等电商平台而言，其本身商业平台的“弱内容性”，更难与种草相匹配。这也考验着老牌电商巨头们，如何将内容属性与商业属性完成兼容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/54260818562ca1982629cd2.40570869.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反观抖音，尽管也在朝着电商方向发展，但其本身所拥有的强大内容基因和用户娱乐属性，将会使其在种草道路上走得顺畅许多。兴趣电商的模式，也为种草和电商提供了合适的生态链路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>货（产品）找人，是抖音面向新时代电商给出的一份满分答卷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在种草的赛道上，国内互联网的第二个小红书，最大可能也将会从这里诞生。</span><span><strong>未来，抖音和小红书，将终有一战。</strong></span></p>
