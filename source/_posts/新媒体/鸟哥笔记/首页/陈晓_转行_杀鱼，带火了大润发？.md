<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相信熟悉大润发的，没事喜欢杀杀鱼的，对这句文案一定不陌生。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>陈晓和大润发，一个演员，</span><span>一家超市。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>明明八竿子都打不到一块，怎么就搅在一起了？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原来这一幕出现在《梦华录》的第23集。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>顾千帆为了给疯批爹爹贺寿，用剑术做了一道生鱼片。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/184872293262b3937765edd1.88097190.gif" title="陈晓“转行”杀鱼，带火了大润发？" alt="陈晓“转行”杀鱼，带火了大润发？" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好一个水果忍者，果然是无情杀手。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我已经来不及关心剑消毒没、鱼做核酸没了？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顾老六把鱼<span>杀得</span>行云流水，鱼片切得薄如蝉翼，一点都不拖泥带水，技能满分。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/34370604562b39379a58e50.37401771.jpeg" title="陈晓“转行”杀鱼，带火了大润发？" alt="陈晓“转行”杀鱼，带火了大润发？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就咱这刀工，就这个卖相，跟赵盼儿开酒楼打下手，保证人满为患。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就连高观察当场，都<span>憋了一</span>首山寨版的<span>《</span><span>咏鹅</span><span>》——</span></span><span><strong>《咏鱼》</strong></span><span>：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“鱼鱼鱼，鱼剑双绝”。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/184248763662b39379dc24a2.66906869.png" title="陈晓“转行”杀鱼，带火了大润发？" alt="陈晓“转行”杀鱼，带火了大润发？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>传下去，</span><span><strong>陈晓在大润发杀了十年的鱼，不然刀工怎么会如此的利落</strong></span><span>。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>抓住陈晓在戏</span><span>中"高冷皇城指挥使"的人设，以</span><span><span>及杀鱼场景，猝不及防打了几波广告</span>，一点都不尬。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯视频赞助费都省了，就<span>成功</span><span>让别人花钱，给自己传播做嫁衣，</span>简直赢麻了。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205789392362326078647a25.53043458.jpeg" title="陈晓“转行”杀鱼，带火了大润发？" alt="陈晓“转行”杀鱼，带火了大润发？" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本来我以为<span>大润发只是单纯皮一下，<span>事情到这儿就结束了。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>可这两天刷小红书的时候，发现大润发又搞了一波后续，居然</span><span>偷偷上新杀鱼围裙。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>难道大润发打算响应民声回馈粉丝，真要</span><span><strong>给顾老六发一套工作服了</strong></span><span>。</span></span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>除了杀鱼一绝，<span>大润发</span>还是有点文案功底在身上的，</span><span><strong>这文案太会写了。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/177550543862b39380a69ce3.84639839.png" title="陈晓“转行”杀鱼，带火了大润发？" alt="陈晓“转行”杀鱼，带火了大润发？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>围裙定位：</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>男人，都应该有的一件衣服。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><strong>围裙</strong>广告语：</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>爱他就给他穿大润发定制围裙</strong><strong>，</strong><strong><span>呵护男人每一天。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>设计也很鲨我，</span><span>上面印了一句：</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><span>“你以为我会在乎吗？我在大润发杀了10年鱼，我的心早已像我杀鱼的刀一样冷了”。</span></em></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个广告男主拿刀固定鱼的位置，围裙兜里还放着一只猫咪。</span><span></span></p>
