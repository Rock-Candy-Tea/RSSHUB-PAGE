                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="3">近来在招聘新人员，当问到To B的<a href="https://www.niaogebiji.com/tag/%E5%B8%82%E5%9C%BA%E8%90%A5%E9%94%80" target="_blank">市场营销</a>工作包含哪些模块时，不少人说就是发软文、投广告和做活动，将市场营销等同于推广获客，忽视了其他模块特别是一些基础工作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="4">获客是ToB品牌市场营销最重要的任务之一，但将ToB市场营销工作简化为推广获客，未免只见树木不见森林。任何品牌的市场营销工作都是一个金字塔似的系统工程，ToB品牌亦不例外。只有打好基础才能让市场营销工作体系有序运行，才有机会实现品效合一的目的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="5">那么，针对TO B品牌来说，其市场营销有哪些基础工作呢？我们一起来看看。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;" data-track="6">一、做好“品牌定位”，构建<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E4%BD%93%E7%B3%BB" target="_blank">品牌体系</a></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="7">这里说的“品牌定位”和广告营销的定位理论稍有差别，或者说应该是企业定位，主要解答几个问题：我们是谁？我们的客户是谁？我们提供什么产品或服务？我们如何提供这些产品和服务？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="8">只有解答了这些问题，企业才能明确业务范围和发展方向，市场营销才能有的放矢，才能明确目标受众是谁和如何向目标受众营销。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="9">这一工作的首要成果将是一套品牌认知体系，包含品牌名称和营销定位、品牌CI和传播策略。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="10">在这套品牌认知体系确定后，就可建立相应的品牌支撑体系（产品、技术、服务、人力、资金等）和品牌维护体系（危机、改善、反馈）。品牌认知体系、品牌支撑体系和品牌维护体系共同沟通一套品牌体系，成为一切市场工作的基础。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">二、打造专业规范动人的品牌界面</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="13">这里的品牌界面指受众接触品牌时所看到并与之交互的相关媒介及其上的内容，我借用界面interface来囊括表达，也有同行称为品牌露出或品牌展厅，大致所指相同。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="14">品牌界面包括线下和线上的多渠道多媒体形式的界面。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="15">线下品牌界面最常见的包括市场宣传物料和线下展厅，例如宣传手册、海报、易拉宝、名片等常见物料，企业内部的长期展厅和展会上的临时展厅等，是受众主动来访时查看或公司人员与他人接触时主动呈递的相关内容，比较常见。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="16">线上品牌界面可按照渠道所属分为自主可控和第三方控制的不同界面。自主可控的界面既包括企业官网、企业公众号、企业视频号及其他自媒体账号等内容平台，还包括产品介绍PPT、宣传手册电子版、产品介绍H5或小程序等内容物料。第三方控制的品牌界面有百度百科、行业媒体品牌专区、合作伙伴的品牌露出等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="17">ToB品牌尤其注重专业性的打造，因此不同渠道的品牌界面应从信息内容、视觉设计等层面注重专业性的呈现，同时注意及时更新确保统一规范，从而在受众接触到这些界面时，留下专业规范的品牌形象。我们同时要发挥创造力，让品牌界面的内容能够打动受众，帮助受众更好地了解品牌和认可品牌。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">三、构建一套客户营销系统</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="20">客户营销系统是指锚定客户和触达其、影响其的的一系列系统集合，主要包含客户画像系统、媒介传播系统和内容资料库。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="21">客户画像系统需要描绘相关产品服务的客户相关情况，如地域分布、订单金额分布、人员规模分布、决策过程情况等，以更好地找到客户、发现其内容偏好和媒介偏好，并包括企业画像和关键个体用户画像。其建立的基础是企业已成交客户的数据库，如果可以的话最好能建立目标客户数据库，以更精准地描绘客户画像并施以长期影响。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="22">媒介传播系统是针对客户的媒介偏好去构建传播渠道和积累媒介资源的整套系统，包含企业可自主控制的媒介渠道和外部媒介资源等，如企业官网、自媒体账号、行业媒体、行业活动、意见领袖等。有了这一套系统，每次完成传播任务时根据传播人物和目标客户制定独特的媒介策略，让我们传播时能更精准地触达到目标受众。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="23">内容营销是任何品牌的市场营销的核心工作。对于To B品牌来说，构建包括产品资料、解决方案、品牌背书、客户案例等的内容资料库，能够在日常传播和针对性的项目传播时，选择更合适目标表示受众的基础资料进行深加工，产出更有影响力的内容子弹。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;" data-track="25">四、打造客户案例生产的配合机制</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="26">客户案例是ToB品牌市场营销的重中之重。ToB有着客单价高、决策链长、产品复杂难理解的特点，市场营销必须解决信任和预期可见的问题。客户案例用已落地真实可见的项目，介绍产品情况、应用过程和实际效果，客户决策时能够参考，通过举一反三可以解答预期疑惑并收获信任。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="27">但客户案例的打造不是一项简单的工作，单凭市场人员的文案整理很难保证最好的效果。需要在服务全流程中建立一套客户案例打造机制，才能产出有力的客户案例。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="28">例如，如何驱动客户配合，既需要销售部门保持良好客情，也需要客服和售后部门保障良好的服务体验，同时最好能制定一些激励性的市场政策；如客户是否愿意署名，可能在前期采购的商务谈判中就需要沟通，也可能在实施或者售后的过程中沟通；客户的产品使用情况既需要采访客户，也需要客观的系统数据支撑，也可能需要客服或售后部门的日常记录等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="29">这些都需要建立一套配合紧密、运行有序的客户案例生产的配合机制，具体到每个企业上又各不相同，需要市场人员来统筹协调完成。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="31">综上，构建品牌体系、打造专业规范动人的品牌界面、构建一套客户营销系统、打造客户案例生产的配合机制，是ToB品牌市场营销最重要的基础工作，在打好基础的情况下，市场人员能够根据企业所处的成长阶段、具体的产品和目标客户，完成好一个又一个营销任务，收到品效合一的营销效果。</p>                        <div style="height: 1px"></div>
