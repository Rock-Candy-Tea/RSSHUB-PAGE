<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要问这两天谁的心情最分裂，非一盏灯莫属。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说的就是一夜成为交通网红的红绿灯。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先是因为新设计被骂上了热搜，后又被发现骂冤了，一次性体验被造谣传谣辟谣的经历。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1207046247630588b6208664.37965323.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：<a href="https://www.niaogebiji.com/tag/%E5%BE%AE%E5%8D%9A%E7%83%AD%E6%90%9C" target="_blank">微博热搜</a></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/723115795623260770c18c4.52658385.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>红灯停，绿灯行。</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就算不是老司机，就算从来没过开车，相信都听说这个朴实无华的规则。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是怎么都没想到，这句教育我们多年的交通规则，前几天被传要被改成九宫格了？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>当然我说的不是吃火锅的九宫格，也不是发圈圈的九宫格。</span><span><strong><span>而是从原来的三个箭头灯，改为“圆盘灯+两个方向指示灯”。</span></strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1911352207630588b67cc7f7.88968784.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin-bottom: 25px;"><span>图源：勤奋哥</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>按照国际管理，简单来科普一下新红绿灯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>1、采用统一的红绿灯样式</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>新国标红绿灯将统一采取中间圆盘（表示直行），左右两边箭头（分别代表左转、右转）的标识。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2、取消红绿灯的读秒设计</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>3、红绿灯通行规则变了</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新规定有点复杂，不过有老司机总结出一句新的口诀，大家不妨马克起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>红灯停，绿灯行，左转不亮看直行，右转不红不用停。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果觉得背了个寂寞，看个动图实践一下：</span></p>

<h4 style="background-image: none; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #818386; font-size: 14px; padding-left: 12px; margin: 25px 0px; font-weight: 300; line-height: 30px; text-align: center;"><span>图源：微博网友评论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不瞒大家说我本来还计划考驾照，刚开始看到新版红绿灯，我直接不想学车了。作为一个老年痴呆选手，就怕一上路记不住这让人脑壳疼的规则。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>上有政策，下有对策，网友纷纷献策：</span></strong><br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>在脖子上挂个</span><span><strong>加长版的说明书</strong></span><span>（地铁老人看手机表情包）</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">干脆把红绿灯打印出来，然后</span><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;"><strong>贴在车内防止出错</strong></span><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">，毕竟刚开始怎么都得适应一下，要过过脑子才敢开。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>还有的操心起下次<span>改版，</span></span><span><strong><span>干脆把收款码放在路口，</span></strong></span><span>司机<span>用手机扫一扫就清楚</span>怎么走。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>新手司机别怕，</span><span><strong><span>再不行看不懂直接摆烂，</span></strong></span><span>让老司机带带你。原地不动等着，等后面的老司机给你压力，哔哔<span>哔</span>提醒你。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1425988433630588ca5f4390.42979164.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>图源：可儿可大瓶</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>旧版的红绿灯设计一目了然，用了这么久，为什么突然要改？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与时俱进可以理解，改设计也就罢了，<span>为什么越改越烧脑？</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>我还在困惑设计究竟是怎么过稿时，</span><span>据说设计者</span><span>孙正良被网暴了，开播10分钟被网友骂到关播？</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/144020824630588cab6e941.75837823.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>图源：</span><span>微博网友评论</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这个规定一出，相当于要改变<span>全国所有司机</span>的驾驶习惯，</span><span>但凡万一脑子没转过来，一个不留神就造成扣分或者车祸。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/205789392362326078647a25.53043458.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>当全网还在困惑这到底是灯，还是坑时，@公安部交通管理局 出来辟谣了。</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“新版”是假的。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原来所谓的新红绿灯设计并不算新。这个<span>“</span>新国标红绿灯”是出自《道路交通信号灯设置与安装规范》(GB 14886—2016)，已经推行了5年，所以新版是不存在的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1554138752630588cb1bfb65.05528150.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@公安部交通管理局</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>复杂是假的。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>九宫格并不是在所有道路普及，而主要是<span>用来</span>应对复杂路口。<span>一般道路<span>总体来说，</span>还是用我们熟悉的那版红绿灯。 </span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1936645048630588cb767538.81764230.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：</span><span>@</span><span>公安部交通管理局</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>网暴也是假的。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最离谱的是，还有人冒充设计师直播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/979459630588cbea2586.16413536.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：</span><span>@</span><span>公安部交通管理局</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>造谣造到公安部头上了，<span>真·造谣一张嘴，辟谣跑断腿。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2016年一早就有的规定，今年炒冷饭就变新了，迷惑了大众。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了感慨造谣成本太低了，不得不说还<span>有点喜大普奔</span>。规定是真的，但不是普遍使用，这下大家可以放心学车了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134948845962326079426029.07408166.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说到底红绿灯的话题之所以引起争议，是因为大家苦于反人类设计久矣。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像免吸管杯盖，纸吸管，永远撕不干净的纸标签，没法并排插的插座。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>哪怕是极少派上用场，九宫格红绿灯确实是一种反人类设计。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>明明3个灯可以解决的事，现在花9个灯还多种组合，容易把</span>简单的事情复杂化。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2058476408630588cc97d842.04433684.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：微博网友评论</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>走到一个路口，都是对司机一次记忆与理解的大考验，就跟听到外语，自己脑子里还得翻译一遍，才能看得明白一样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果大范围推广，前期会增加司机的行驶负担，也会增加社会的教育成本。<span>从设计的角度来说，一切设计应该是越简单越好。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>另外红绿灯涉及到的，不仅是设计的问题</strong><strong>，还是关乎超级符号的传播问题。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>超级符号就是大家都认识、熟悉的符号。比如公共符号的交通标志、男女厕所标志灯；还有文化符号的中国结、环保绿等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这些符号在大众心中已形成一定的认知，</span><span>社会无需再进行教育，每个人一看到就会自觉“听话”，按它的指令行动。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而所有的传播都是符号信息的编码和解码，超级符号就是要去找到沟通效率高的文化符号、公共符号。要是编码内容太复杂，消费者很难理解。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/747288919630588cce65954.93636144.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：壹图库</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以有的品牌很聪明，将品牌设计跟超级符号结合，这样就可以在大家脑海里轻松下指令，让他们产生条件反射。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><strong>就拿红绿灯来说，作为一种超级交通符号：红灯停，绿灯行，这个公共交通规则是约定俗成的。</strong></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong>乔布斯设计的</strong><strong>苹果软件窗口，灵感就是来自红绿灯，</strong></span><span>把按钮设计成红绿灯的颜色形状，红、<span>绿、黄色分别</span>代表“关闭”、“打开”，“隐藏<span>”</span>。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反观极少情况用的九宫格“红绿灯”，是对红绿灯符号的重新编码，一盏“不亮”的灯，被重新定义多层意义，可能是停也可能是走，变得不再简单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好比此前雪碧将把绿色瓶包装，改为透明塑料包装雪碧，部分网友感慨不清凉了，就是因为大家难以接受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>对于雪碧来说，</span><span><strong>“绿”就是雪碧的超级符号，<strong>代表的是</strong>雪碧<strong>的</strong>清爽口味和整体的视觉形象。</strong></span><span>雪碧换包装，这就相当于要给品牌</span><span>符号重新解码。</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有</span><span><strong>星巴克</strong></span><span><span><strong>重新换个马甲在俄罗斯卖咖啡，<strong>俄版星巴克<strong>不仅</strong>改名StarsCoffee，</strong></strong></span><span>还把绿色logo换了。<span>美人鱼的形象没了，取而代之的是<span>戴着俄罗斯头饰的女孩，</span></span>很多人同样表示看不习惯。</span></span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些案例都说明，<span>无论是相关机构还是品牌，</span>挑战超级符号的原本意义，就会是挑战公众的认知，需要付出一定的<span>时间成本、沟通成本。</span></span><span></span></p>
