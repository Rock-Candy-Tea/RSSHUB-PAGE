<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近10亿人口的生活与<a href="https://www.niaogebiji.com/tag/%E6%B6%88%E8%B4%B9" target="_blank">消费</a>，没能撑得起<a href="https://www.niaogebiji.com/tag/%E4%BA%AC%E4%B8%9C" target="_blank">京东</a>在<a href="https://www.niaogebiji.com/tag/%E4%B8%8B%E6%B2%89%E5%B8%82%E5%9C%BA" target="_blank">下沉市场</a>的美梦。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3年之后，这场刘强东亲自带队的一战，彻底败了！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>京喜拆分，败退下沉市场</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在月余之前，京东在下沉市场的彻底败退便已初现端倪。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>彼时，有媒体报道称京东正在进行新一轮组织架构调整，原京喜事业群将在6月被拆散，原有业务线也将被整合至其他业务相近事业群中，京喜APP、京喜通（原京东新通路）、京喜拼拼将被并入京东零售。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这意味着，随着京东降本增效的步伐加快，“京喜事业群”这一刚刚成立不到两年的京东一级事业群，惨遭夭折。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，在以上传闻回荡了一个月后，这一消息迎来最新进展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知情人士透露称，在此次结构重组中，有约100名京喜事业群员工会被纳入同城购业务部，其余的200多名京喜员工将寻求转岗至其他部门。截止7月20日，仍未转岗成功的员工将不得不面对被“优化”的结果。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/104231708762d73a247ccff0.53165067.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着京喜事业群的拆散，京东现阶段在下沉市场的探索彻底失败！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回想3年前，随着电商新贵拼多多自五环外围攻“猫”、“狗”，在给这两大传统电商巨头带来压力的同时，也带来了新的希望。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，京喜、淘宝特价版纷纷上线，既是面向拼多多进攻的防守，也在围绕下沉市场寻求着新的增量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>初生的京喜也确实曾给京东带去惊喜。2020年，京东有80%的新增用户来源于下沉市场，2021年，这一比例降至70%。但不可否认的是，专攻下沉市场的京喜事业部（群）曾一度成为京东新用户增长的主力军。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在2020年底，京喜事业部正式升级为京喜事业群，并随着社区团购的火热，京喜家族再添新成员“京喜拼拼”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，退居二线的刘强东也宣布“复出”，亲自带队京喜事业群，足见其对下沉市场的重视。带领京东与阿里抗衡的他，将要在下沉市场打起他的电商第二战。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/45494891262d73a25464ac5.10511519.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而好景不长，随着互联网红利期的消失，各方竞争加剧，烧钱换规模几乎已成死路一条，降本增效、精细化运营成为时代的主流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年第一季度，京东营销费用达到了87亿元，同比增加24.4%，但新增用户量仅有1100万，是过去一年的最低值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>社区团购赛道的京喜拼拼在未大规模撤城前，亏损率也接近40%。要知道，橙心优选在亏损率30%时便已难以为继，而美团优选和多多买菜亏损率则更低，在20%以及15%上下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单一句话，砸钱比不过对手，数据还比对手差，结局可想而知。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>长期亏损，且回报低于预期之下，京喜已经从京东的主舞台上退下。在月初的集团战略会上，高傲如刘强东也不得不低头承认了京喜的失败。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟下沉市场的蛋糕再香，分蛋糕的人也得有命享用才行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>降本增效，京东战线大收缩</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上周，《财富》杂志刚刚发布了2022年中国500强排行榜。在一片“中”字头企业之间，位列第七的京东集团异常醒目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是京东在该榜单中首次进入前十，也是所有国内民营企业中的第一。作为京东老对手的阿里，也只排在榜单第11名。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/51171111562d73a258f7640.60897187.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而民营企业第一的荣誉，也只是代表着京东的营收最高。事实上，在忙着进军下沉市场之时，不知不觉间，京东便已深陷亏损泥潭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年第一季度财报显示，京东该季度净亏损达30亿元，同比转盈为亏，在去年同期，京东净利润为36亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如此巨额的亏损已非首次。在2021年第三、第四季度，京东分别达到了28亿、52亿元的净亏损。面对连续三季度的巨额亏损，可以说，京东遇到大问题了！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，降本增效已经成为了如今京东的头号大事。也正因如此，京东开始逐步调整，关闭多项短期内难以扭亏为盈的业务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年三月份，京东零售V事业群成为首个被“开刀”的一级事业群。据知情人士透露，V事业群被拆分后，旗下业务线将被整合至不同事业群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在当时，许多网民更为熟悉的，或许还是京东因裁员被骂上热搜一事。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/74031043062d73a26387f82.81873841.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“毕业快乐！恭喜您从京东顺利毕业！感谢一路相伴！”因为员工离职通知单上的这句话，京东被骂上微博热搜榜，“毕业”似乎也同“福报”一般变成了一个恶意满满的词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>暂且不论措辞如何，仅看截图曝光的三月，京东内部包括零售、物流、科技在内的多个板块和事业部都进行了10%-30%不等的裁员，可谓之京东战线大收缩的见证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，京喜更是重灾区。在网传的一份裁员名单中，京喜广东、四川和江西战区直接被“一锅端”。只是没想到几个月后的今天，京喜事业群便步了V事业群的后尘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在今年第一季度财报后的电话会上，京东集团CEO徐雷曾表示：“新业务在Q1进行了调整，对短期商业化发展不利的进行了关停并转，未来还会更多进行业务的聚焦，我们还会持续不断地进行这项工作。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今看来，在京东迎来盈利曙光之前，集团整体的降本增效之路或许还将继续。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>正刚美团，发力同城零售</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在下沉市场败退后，京东仍需要新的增长点，这一次，它盯上了同城零售。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在V事业群被拆分的3月，京东同城购业务部正式成立，整合了京东到家、原京东零售全渠道到家业务部等，以拓展多种到家和到店的业务场景为任务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此次京喜事业群拆分优化的过程中，也有约100名员工被纳入同城购业务部。这个刚刚成立三个月的业务部，正在不断壮大自身，以在京东新方向下施展拳脚。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/122796806962d73a26e08781.91497403.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>区别于其他业务部的是，同城购独立于其他业务群，由京东副总裁何辉剑担任直接负责人。仅从此处，便可看出京东内部对该业务的期许。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在月初的京东集团战略会上，同城购业务部正式提出了未来几年的战略目标，可概括为“一大三小”。“一大”即为大力发展同城零售业务，“三小”则指深耕外卖、到店综合服务以及家政三个领域。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，京东的同城零售战略，正与美团针锋相对。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在去年秋季，美团到家事业群总裁王莆中便表示，即时零售规模将在未来五年增长至 1 万亿，而美团闪购的目标是从中分走4000亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为该领域的头部玩家，出手便要分走近一半的市场份额，美团自然有着自己的底气。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为以外卖为立身之本的美团，凭借着数百万线下骑手牢牢守护着自身的龙头地位。除饿了么勉强抗衡之外，几乎没有第三方涉足的机会。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/105598074762d73a2776b1f0.46422070.png" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也正是这数百万骑手大军所构成的即时配送体系，使得美团在同城零售业务上拥有着得天独厚的优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而京东的入局，恰恰也凭借着近百万的达达骑手，从美团的把持下撕开了一道口子。值得注意的是，京东同城购业务部负责人何辉剑，除京东副总裁身份外，他还是达达集团的副总裁。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比于外卖的高频消费场景，即时零售消费频度并不算高，达达骑手足以应对京东的即时零售需求。除此之外，外卖、到店综合服务等，京东也都在不断发力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在如今整个经济下行的大环境下，聚焦主业，砍掉亏损业务毫无疑问是短期内降本增效的最好方式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但利润从来不是省出来的，开拓新业务、寻找新的增长点、盈利点也是合适手段。只是当这些互联网巨头再度碰撞在一处，不知又要摩擦出怎样的火花。</span></p>
