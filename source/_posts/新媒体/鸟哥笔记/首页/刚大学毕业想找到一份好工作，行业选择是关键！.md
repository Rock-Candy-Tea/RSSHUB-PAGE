<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年的<a href="https://www.niaogebiji.com/tag/%E5%BA%94%E5%B1%8A%E6%AF%95%E4%B8%9A%E7%94%9F" target="_blank">应届毕业生</a>被不少人称为史上最难的毕业生，因为毕业人数同比增长167万，毕业人数达到了1076万人，创下了历史最高记录。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>应届毕业生的规模如此之大，也意味着竞争愈演愈烈。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在如此严峻的就业形势下，名校出身，专业过硬的学生根本不愁找不到工作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但双一流毕业生毕竟占少数，更多的是普通的本专科毕业生。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，智联招聘发布了《2022大学生就业力调研报告》，报告显示从院校类型看，双一流、普通本科、专科院校的签约率分别为33%、12.5%、13.3%</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>双一流院校优势明显，而专科院校签约率高于普本，普通本科毕业生的就业困境不容忽视。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/72410831162cfcf9707c459.72947523.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多学生因为学校实力一般、所学专业不占优势一直找不到满意的工作。</span><span>要么就是待遇不好，要么就是工作太低，要么就是没有发展前景。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚大学毕业的小杨同样面临着就业困境，小杨毕业于普通二本的汉语言文学专业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕业之后他面试过跟专业相关的很多工作，比如办公室文员、文案编辑、语文教师，但最后也没有找到合适的工作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来，小杨在招聘软件上发现，短视频运营行业发展前景好，岗位需求量大，工资待遇也不错。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/75100828062cfcf974a6231.10525792.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是他萌生了想要从事短视频运营工作的想法，因为小杨想要在短期内掌握短视频运营技能找到一份好工作，于是他就报了专业的短视频培训课程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经过多方对比考虑，小杨选择了<strong><span>厚昌教育田轩老师</span></strong>的<strong><span>《短视频高薪就业实战班》。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经过一个月系统地学习，小杨成功掌握技能凭借找到了一份月薪2.5w的短视频运营工作。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/126191678662cfcf9785d547.62266211.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从小杨的经历中我们可以发现，有时候本专业找不到好工作，与其原地踏步，不如转变思路重新选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为应届毕业生，就意味着我们还有更多重塑的可能，只有不断尝试才能找到真正适合自己的工作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>并且，想要找到一份好工作，行业选择也是关键。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>普通的应届毕业生，想要杀出重围，就一定要选择就业前景好，岗位需求量大，工资待遇优的行业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频运营行业就完美地符合以上三点，短视频行业发展迅猛，已经成为人们不可或缺的娱乐方式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>既然短视频运营工作怎么好，要求是不是也非常高？工作内容是不是很复杂？</span><span>找工作我们不仅要看工资待遇，还得看工作内容我们是否能胜任。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来小菜就跟大家介绍一下短视频运营的日常工作内容，让大家能有所了解。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频运营相比其他工作对学历和工作经验要求并不高，但从事短视频运营工作我们也得能完成工作任务才行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短视频运营的工作内容围绕短视频账号展开，日常主要工作是保证账号的内容输出。完成一条短视频的基本流程大致如下：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>①选题</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>完成一条短视频，首先我们要找好选题，明确我们的视频方向和主题。只有事先确定好选题，才能为后续工作做准备。找选题的时候，要注意结合自身定位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>②脚本</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单来说，短视频的脚本可以理解为短视频拍摄、剪辑的依据，一切参与短视频制作（编导、摄影师、剪辑、演员、道具等）都是服从于脚本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>③道具/场景</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在拍摄视频之前，我们需要提前准备好道具和场景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拍摄</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>④拍摄</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>远全中近特，不同的景别适合不同的视频。</span><span>为了保证视频的完整，我们要保证收音清晰、画面完整。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>⑤剪辑</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>剪辑过程是对拍摄内容的二次加工，主要包括对视频内容的筛选，重新组合。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>⑥发布</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频拍摄剪辑完成之后，就进入了发布阶段。发布时注意地理位置的添加，话题选择，标题描述的撰写等</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上就是短视频运营的日常工作内容了，大家是不是对短视频运营有了更多的了解？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，希望今天的文章能为大家择业提供新方向。如果你正在找工作，而且对短视频行业感兴趣的话，但不知道自己是否合适。</span></p>
