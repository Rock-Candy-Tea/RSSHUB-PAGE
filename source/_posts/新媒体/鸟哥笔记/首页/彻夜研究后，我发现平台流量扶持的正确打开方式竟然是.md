<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本文主要围绕流量展开，面对当前内容平台新起，受众的注意力被分散，如果获取大量流量已成为一个难点，那么在有限的流量内，如何高效率的进行使用呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为一个某音资深用户，日常会有大量的时间刷短视频，什么样的内容会让我点赞，让我评论，让我收藏，让我转发，因为身在其中，之前对于觉得不错的模式都有简单的灵感记录，下面抽象提炼一部分，主要涉及两个要点进行展开浅浅谈下自己的思考</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1.点赞，评论，收藏，转发的侧重性</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2.内容平台的属性匹配与分发策略</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/196401358162c4e93080bea6.96706294.png" referrerpolicy="no-referrer">点赞，评论，收藏，转发的侧重性</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这四个动作作为用户的关键动作，每一个数据都可以侧面反映内容的调性，当然也有部分平台有新的指标，例如喜欢，赞同等，以下先附上一些例子，大家可以比对看下这类用户，你第一个想做的关键动作是什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/58695837162f9e521d54dc9.31426693.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>画面（狂风暴雨，一朵朵花落败，路灯下一对对情侣分道扬镳，海面上各种波涛澎湃）+BGM（音量很高，震撼人心）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>画面（朋友街边搞笑尴尬的剧情创作）+BGM（搞笑）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>画面（国家重点事项的转播）+BGM（严肃）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>画面（社会焦点事宜的陈述，例如骑手与顾客的故事，商家与顾客的故事，学生与辅导员的故事）+BGM（一般为人声）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>画面（颜值极高的明星/素人表演）+BGM（与表演者有关的歌曲）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>画面（干货分享+案例拆解）+BGM（一般为人声）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些内容在某音上面都很常见，我也经常刷到，甚至有些话题的内容重复度极高，简要说下我下意识可能会进行的操作</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>case1不会做动作，但是会看多遍，间接提升创作者的完播率</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>case2可能会转发给朋友（前不久我朋友也刚转发给我类似的内容）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>case3和4视事件本身的正向性/负向性，如果是喜闻乐见的国家大事，可能会点赞；其他事件的话可能后续会再次搜索关注后续；如果事件本身具有教育/科普意义，可能会收藏</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>case5点赞或转发</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>case6收藏，如果分享的足够细致，可能会连续看多遍来学会这一技能，如果连续刷到多次这样的干货内容，那么会考虑直接关注账号</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于个人角度来说，上面关键动作整体对于内容的要求会略高，因为同质化的内容的确见的比较多，但是创作者们的水平也在不断提升。好的内容，依旧会是用户选择这个平台，选择这个创作者的重要理由，接下来简要说下这四个动作背后的内容侧重</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span lang="EN-US">1.</span></strong><strong><span>点赞可参考的内容技巧</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1.内容正向类：例如终于获得XXX机会，遇到XXX问题积极需求向上沟通获得满意结果，XXX充实的一天</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2.画面精美类：例如明星舞台，综艺，高颜值素人的剪辑，可爱的宠物，好看的壁纸等</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/202401448162f9e5221f7b15.02614801.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span lang="EN-US">2.</span></strong><strong><span>评论可参考的内容技巧</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>评论我认为是这四个数据中，最能体现作品互动性的数据。一个好的评论，有时候比作品本身的点赞量都高，这就意味着当用户看到这条作品拥有极多的评论时，往往会点开评论区查看究竟。如果说作品本身是流量的第一个承接池，那么评论就是作品自然衍生出来的第一个高流量池，适合的品牌甚至能在评论区直接做推广（以某部分博主而言，每次发布视频，评论区都是一堆品牌在那边评论抽奖，整体的流量非常高，品牌赠送的福利直接带动用户更快更多的涌入评论区）那么如何提升作品评论呢</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1.视频中直接引导，例如一些搞笑视频，一部分创作者会告知用户评论区艾特自己的闺蜜/对象一起来看，或者打出XXX字母，看下你们打出的词语是什么</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2.视频中存在一些特别好用的小东西或者小技巧，引导用户自主在评论区发问（例如视频本身是vlog类，会有用户问第XX秒出现的那个杯子是什么牌子的；也有部分比较直接的，例如视频本身就是壁纸或者小说推荐，那么评论区就会存在用户问具体的内容）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3.前后反转类，这里直接举例说明，例如之前有一类视频，前面为负面的基调，说别人看了自己的照片都问我为什么要把XXX修图修成这样，可是自己明明没有修，然后结尾快速的放出了照片并配上搞笑的音乐，点开评论区后，大多数都是：安慰的话卡在一半/别人说得对</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/190760796462f9e5225f3a80.12677575.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抽象总结一下，就是需要让用户在看了你的视频后，有想说话的欲望，这个说可以是和创作者说，例如好物安利，也可以是和用户自己的朋友说，例如一起来看等。还有一些创作者，会在视频中植入一些简易但出错的片段，引发用户的槽点讨论，例如“真希望跳进视频来帮你炒菜等”，创作者在发布内容的时候可以先自行预判一下，结合过往的内容调性与自身IP调性，用户可能会评论什么，这样事件久了，对于内容的数据也会养成基本的认知</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span lang="EN-US">3.</span></strong><strong><span>收藏可参考的内容技巧</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>收藏这个动作，对于内容的“利他性”要求更高，因为如果是点赞，评论等，都是用户此时此刻看到你的内容留下的即时动作，后面基本不会回顾，但是收藏这个动作，大概率是因为用户想多次浏览你的内容，判断这个内容可能对于之后有用，所以放入收藏夹之后继续浏览观看，那么如何提升作品收藏量呢</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1.如果是干货分享，保证一个基本的素材完整（例如问题+过程+解决方案），这样用户收藏的就是一个完整的作品，之后回顾的时候，一个视频就能解决一类问题</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/123744136662f9e522b50a54.96119840.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2.如果是舞台剪辑/影视剪辑这类作品，有时候也会获得很高的流量，重点可以考虑瞬时状态的惊艳和多个短片的提炼汇总，这部分内容会比较考验剪辑/汇总/解说能力。基于我个人角度而言，如果是喜欢的明星的多个舞台剪辑汇总，我也会考虑收藏</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span lang="EN-US">4.</span></strong><strong><span>转发可参考的内容技巧</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>转发这个动作含两层意思，一个是站内转给站内，例如某音用户看到了内容，转给某音站内好友，大家一起哈哈哈哈哈哈；还有一个是站内转给站外，例如某音用户看到了内容，下载保存，发给好友，大家一起哈哈哈哈哈哈。往夸张层面讲，前者可能是某音用户给你带来了本不会分给你的流量，后者直接帮你突破平台壁垒</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/140981894062f9e522e88337.91444035.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么如何提升转发呢，上面有提到如果想提升评论，那么就是在创作的过程想一想用户看了你的内容有没有什么想说的，那么想提升转发的话，就是在创作的过程中想一想用户看了你的内容有没有想分享给朋友说的，可以参考以下几点</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1.内容的前置准确性：很多热点刚火起来的适合，可能最先火在A平台，这个时候如果一些媒体将其报道在B平台，那么对于B平台的用户来说，这是他们刚接收到的消息，会下意识将重要新闻转给自己身边的朋友，这里就是抢占了一个先机，同时也需要先行核实内容的准确性</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2.能够引发多人讨论的：区别于单一的评论而言，引发多人讨论的内容，用户可能会采取两个形式，转发至站外的群聊，或者评论中直接艾特多人。例如某些原创剧情创作者，在剧情演绎中往往不止一个角色，这个时候可以引发多人的共鸣</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/137493720762c4e930a41328.75259685.png" referrerpolicy="no-referrer">内容平台的属性匹配与分发策略</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当前内容平台像是百花盛开，各花各有各花香的状态，不断也有新的平台冒出来，亦或是A平台和B平台争夺同一类客户，那么对于一个创作者而言，是选择A平台，还是选择ABCD平台一起发，每个平台给到的初始流量，大约为多少呢</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我个人的内容创作历程很早之前就开始，于我而言，我觉得像公众号运营，文案，设计等都算一定程度的内容创作，不过几年前我才开始有意识的培养多平台分发的意识，并且靠着一些内容分发后的实操数据，改变了策略，置顶了目前的创作策略，下面结合平台/用户属性浅浅谈下部分心得</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span lang="EN-US">1.</span></strong><strong><span>多平台分发看似有效，实则很累</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之前我主要的创作方向为求职和运营方面（虽然目前好像也差不多），在专业平台上发文获得还不错的成就后，我觉得多平台分发应该对于公众号涨粉有更好的效果，于是我经常花费很大的心力去搬运（因为每个平台对于排版和格式有自己的要求，有时候甚至还要做图，然后发完之后还要去看数据，平台个数一多，如果不记录都会忘了哪些平台还没发过）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span lang="EN-US">2.</span></strong><strong><span>部分平台风很大，一看数据比其他家低很多</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在考虑平台的过程中，一般会前置搜索这一平台的资料，例如多少粉丝到下一权益，商业化如何变现，初始曝光量为多少），之前有AB平台，AB的属性相近，A的风一直很大，网上也是各种关于A平台的推广视频，虽然当初我判断当时其实干货属性的文章并不合适AB，但是基于分发的角度，我选择多个平台都同步了，到前段时间盘点完发现，B的整体数据是A的好几倍，这也侧面反映了一个现状：创作者们争先入驻风大的平台，导致创作内容超过原本数值，流量被稀释</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span lang="EN-US">3.</span></strong><strong><span>整体盘点的重要性</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在分发了几个月后，的确多个平台都有了数据体现，我是时隔了很长时间才对他们做一个整体盘点，盘点分为“平台属性”“原始创作个数”“整体曝光量”“单条最高曝光量”等维度展开，也是因为这份盘点，让我对于之前的分发策略有了一个基本的事后复盘，部分数据参考如下（针对没有粉丝基础的创作者）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>普通图文平台（即基本投稿就能发表）的初始流量一般小于100</span></strong><span>，但是会有推荐量，推荐量和实际的浏览量一般有很大差别，但是如果内容命中一些搜索关键词，也有可能突破，我这边就有一篇文章发布在相近的平台，其中一个平台上有1000+的浏览量</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>图文/视频结合平台，例如某书，某乎，这部分平台的流量相对会略高</span></strong><span>，因为用户平常会刷这些APP，部分结合了一些精美的图片制作或者话题讨论，一般初始流量会在100左右。如果刚好是热点讨论，流量会在短期内迅速上升，甚至在之后的很长时间内，还源源不断有人点赞</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>视频平台，例如某音，也是我近期在研究实操的平台，初始流量相对图文平台给的更多</span></strong><span>，网上一些人说初始流量在200—500，经过实操发现，的确差不多，不过我的一些初始流量稳定在400—500左右，然后里面会有创作者中心可以看到完播率等数据，我现在也大致通过这几个数据大致判断是否还有推流，目前的认知是等到完播率降到一定比例，慢慢的视频推流也就结束了。不过某音值得探索的还有很多，目前为止，在未投放的情况下，已经产出两条破千播放的视频，这个数据结果还可以，所以有吸引到我继续探索下去</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span lang="EN-US">4.</span></strong><strong><span>流量在被稀释</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最直接的感观就是，同样的账号，之前发布的数据是XXX，过了一段时间，发布后的数据降了很多，间接反映平台能够给到创作者的初始流量后续可能还会不断减少，作品的分发是有必要的，但是不宜过多，应该首选与自身内容匹配的平台，例如图文类分发的第一选择：平台XXX+XXX，视频类分发的第一选择：XXX+XXX，如果你发现一个内容在A平台特别火，那么可以尝试分发在与A平台属性相近的平台，然后根据当下热点与作品数据来及时调整</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提示：平台文章内容为个人当前观点，基于行业更新变迁/个人想法迭代等多重因素，不构成决策~</span></p>
