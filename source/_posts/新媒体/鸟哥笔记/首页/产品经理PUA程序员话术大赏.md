<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前排提醒，本文纯纯学术交流，请谨慎用于实践，否则，一切医药费自行承担！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、给你提需求是害你吗？那是爱你啊！你也不想想，没产品需求你的价值在哪？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、以你现在的能力，不加班能跟上产品迭代的节奏吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、我觉得这期需求你能做的更快，如果实力不够，那就加班来凑啊</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、你去外面问问，现在哪个<a href="https://www.niaogebiji.com/tag/%E4%BA%A7%E5%93%81%E7%BB%8F%E7%90%86" target="_blank">产品经理</a>还像我写这么详细的文档？有个原型图就不错了！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5、（对需求追问细枝末节）需求细节不都写在文档里吗？你到底有没有用心看需求文档？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6、（不细看需求自由开发）你这是写了坨啥？我需求是这么写的吗？到底你是产品经理我是产品经理啊？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7、运营说啥你就做啥，你自己的思考呢？你可是团队的核心开发啊，你的时间很宝贵的，做哪些垃圾活动跟摸鱼有什么区别？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8、你要学会自己挖掘产品还有哪些需要优化的点，我们团队不需要没有独立思考的开发</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9、你在这次迭代过程的表现，没有达到我的预期</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>10、你说你也快30了，撸了七八年代码，连个女朋友都没有，作为男人，这个年纪应该做到管理层了，再不加班拼一把，还能有什么前途？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>11、你的开发水平，很令我失望</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>12、你虽然平时加班多，但你的编码水平也是肉眼可见的提高了，这就是团队带给你的价值啊！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>13、如果你觉得需求做不完，首先你要学会时间管理</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>14、天天加班，说明你效率太低、工作方式有问题</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>15、你要是不行就下去，公司里多少人想进咱们这个团队</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>16、你这个开发速度，很难跟上我们的团队节奏啊</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，来一段带有福报味儿的产品经理PUA程序员的集大成话术：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其实，我对你是有一些失望的。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>你最初来到这个团队，是低于大家的平均水平的，大家都希望你能够拼一把，快速成长起来。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>做开发，不是说把代码写好就可以了。你要有架构思维，你写的框架，现在能用，等我们用户量上一个数量级，还能用吗？功能点再翻一倍呢？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>你在公司得时间也不短了，你比公司其他开发强在哪里？你是否沉淀了一些可以复用的资料和方法论？有任何想法时你都要想一想：为什么是你，而不是别人呢？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>你要有自己独立的判断能力，而不是我说做什么需求你就写什么代码。后面，把你的思考应用到实际工作中，我希望看到你实际的进步，而不是仅仅停留在口头上。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>另外，提醒一下（敲桌子），你的干劲，跟刚进团队相比，有些懈怠了，马上年底了，为了自己的绩效也得加把劲儿！</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>你看隔壁团队那个谁，上半年做紧急项目时，人家可是天天睡在公司的，做梦都在敲键盘！你怎么就不能逼自己一把呢？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>成长，一定是伴随着痛苦的，当你最痛苦的时候其实才是你成长最快的时候~</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>加班吧，程序员！</span></strong></p>
