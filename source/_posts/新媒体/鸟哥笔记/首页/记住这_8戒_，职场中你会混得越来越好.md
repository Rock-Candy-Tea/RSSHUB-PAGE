<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">常听人慨叹：职场工作不容易！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其实，当今时代，干什么工作都不容易。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是，如果你在职场中，能够坚定而持久地做到“八戒”，相信在不久的将来，迎接你的将是工作能力与事业前途的双丰收。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目光短浅，工作只是为了钱</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在一家商贸公司里，有一个叫林名的高级职员，深得总经理的器重。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但他总觉得自己的薪水不够高。因为，在他的头脑中，工作就是为了挣钱。付出这么多精力，却得到和别人差不多的薪水，总觉得心里不平衡。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为此，他好几次想对总经理提出加薪的要求，却又不好意思开口。久而久之，再也不能像以前那样安心地工作了，一天到晚总是不停地抱怨，后来甚至以跳槽相威胁。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">渐渐地，总经理不再像原来那样器重他，最后将他辞退了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">薪水，当然是工作目的之一，但我们不能仅仅为薪水而工作，它不是工作的全部意义所在。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">若能以积极的心态、长远的眼光看待工作，那么，你从中得到的就不仅仅是薪水，还有能力的提升和发展的机遇。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，你永远不要惊羡某个薪水微薄的同事，忽然提升到重要的位置。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">若说其中有奇妙，那就是他在与你做着相同的工作，甚至拿着比你还少的薪水时，付出了比你多一倍甚至几倍的努力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这也应了那句话：“不计报酬，报酬更多”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/159959090362aa21478f7494.23005660.jpeg" title="记住这“8戒”，职场中你会混得越来越好" alt="记住这“8戒”，职场中你会混得越来越好" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">2、</span>二戒：</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">单打独斗，成为职场独行侠</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有一天，上帝带着一名教士来到地狱，看到这里有许多人围坐在盛有香喷喷饭菜的大锅旁，但因每个人手中的勺子柄太长，无法将食物送到嘴里，饿得痛苦不堪。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">接着，上帝又带领教士来到隔壁的天堂，虽然情形和地狱一样，但这里的人看起来都红光满面，十分快乐。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">教士不解地问上帝：“为什么仅一墙之隔，有的人快乐，有的人愁眉不展呢？”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上帝微笑着说：“难道你没看到，快乐的人用喂对方吃饭的办法，解决了勺柄过长的问题吗？”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">职场工作中，不可避免地存在交叉之处，因此，工作当中应该是分工不分家。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于一些应急任务，更需要各个部门、每个员工携手齐心同上阵，不分分内分外，一起攻坚克难。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同时，在职场工作中，要善于并乐于关心、支持和帮助周围的同事，千万不可离群索居、处处设防，将自己关在冰冷幽暗的“禁闭室”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/6853749062aa2147bfc0f5.19109103.jpeg" title="记住这“8戒”，职场中你会混得越来越好" alt="记住这“8戒”，职场中你会混得越来越好" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">3、</span>三戒：</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">粗心大意，微小细节酿大祸</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一家服装公司的新入职业务员，上司让他为单位订购一批羊皮。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">他在与供应商的合同中写道：“每张大于4平方尺、有疤痕的不要。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">值得注意的是，其中的顿号本应是句号，却被他错写成了顿号。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">结果，这一错误被供应商钻了空子，发出的羊皮都是小于4平方尺的，使公司哑巴吃黄连、有苦说不出，损失了数万元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">真可谓“一个标点值千金”！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而这位粗心大意的职场新手，也立马被公司解雇了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">商业活动无小事。不管是重大问题还是细枝末节，其工作的好与坏、粗与细，都会在较大范围产生影响，大则事关公司的经济损失，小亦事关个人的前途。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">要想在职场中受得起重托，担得起大任，就需要认真、认真、再认真，细心、细心、再细心。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/127260031062aa2147eacf27.73893945.jpeg" title="记住这“8戒”，职场中你会混得越来越好" alt="记住这“8戒”，职场中你会混得越来越好" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">4、</span>四戒：</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">主观臆断，凭空想象贴标签</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">孔子和弟子周游列国，潦倒在陈国和蔡国的半路上，七天未吃到一粒米，饿得只好白天睡大觉。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">弟子颜回出去讨了一点米回来，煮给孔子吃。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">等到快煮熟的时候，孔子看见颜回掀锅盖，抓些白饭往嘴里塞，孔子装作没看见，也不去责问。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">饭煮熟后，颜回端着饭给孔子吃。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">孔子假装若有所思地说：“我刚才梦见死去的祖先，我想把干净还没人吃的米饭，先拿来祭奠祖先吧!”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">颜回顿时慌张起来说：“这锅饭我已先吃了一口，不可以祭祖先了。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">孔子问：“为什么？”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">颜回涨红脸，嗫嗫地说：“刚才在煮饭时，不小心掉了些染灰在锅里。染灰的米饭丢了太可惜，只好把它抓起来吃了，我不是故意把饭吃了。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">孔子听了，才恍然大悟，对自己的观察错误感觉愧疚，抱歉地说：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“我平常对颜回最信任，但仍然会怀疑他，可见我们内心是最难确定的！弟子们大家要记下这件事，要了解一个人，还真是不容易啊！”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现实中，职场工作鱼龙混杂，似是而非，有时还真假难辨。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当你给一个同事下结论的时候，要想一想：真的你所看到的就是真实的吗？还是你只从一个点、一个面去观察那个人呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">大多数人，在根本不了解对方立场与困难的时候，就已经给对方下评语了，更何况是在有利益冲突的职场。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很多事信者恒信，不信者恒不信，要客观地跳出成见，才有机会接近真相。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/175041503862aa2148264056.47948278.jpeg" title="记住这“8戒”，职场中你会混得越来越好" alt="记住这“8戒”，职场中你会混得越来越好" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">5、</span>五戒：</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">出言无忌，话语伤人又害己</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一位职场新人，邀请办公室的同事们吃饭，一是想拉近距离，二是要讨教经验。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">可过了约定的晚饭时间，他见被邀请的客人还没有到齐，便说：“怎么该来的还没有来呀？”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一部分已经到的客人觉得不对味儿，心想我们是不该来的吗？便不快地走了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">见此情景，请客者忙说道：“怎么不该走的走了？”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另一些客人听了也感觉不对劲，心想难道我们是该走的吗？于是起身便走。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">请客者一看更急了，连忙说道：“我说的不是你们！”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">听了这话，最后剩下的几个客人也走掉了，一人不剩。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">说话是一门大学问，要有“准星”，对谁说、何时说、何地说、说什么、怎么说都要经过慎重的思考。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一句话说出去之前，你是它的主人，但说出去之后，你就成了它的奴隶。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">职场工作中，请示、汇报及办事、办文、办会都离不开言语表达，这就要求在职场中的你要敏于言、善于言。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">拖泥带水，绕来绕去不知云</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">刑部主事茹太素，向朱元璋上言奏事。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">朱元璋耐着性子听了1万多字，突然勃然大怒，痛斥茹太素“虚词失实，浮文乱真”，于是命令部属棍棒伺候，将茹太素打得皮开肉绽。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二天，朱元璋又让人将这奏章接着念，一直念到2万多字时，才知道到底要说些什么事，而这上奏的五件事中，有四件是可行的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">朱元璋说：“太素所陈，五百余言可尽耳。”并要求“若官民有言者，许陈事实，不许繁文。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">林语堂说：“演讲应该和女子的裙子一样，越短越好。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">托尔斯泰说请你们珍惜语言，让每一个字都像利箭一样，一直射到听众的心坎上。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">职场工作任务多、时间紧、节奏快、标准高，因此，必须做到迅捷高效、精到简洁的沟通。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">尤其是给上级汇报工作时，一定要抓住重点、条理清晰、简洁明了，切忌毫无重点、云里雾里、不知所云。</p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">7、</span>七戒：</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">急功近利，筹划不周吃大亏</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">汉景帝时期，重臣晁错极力主张中央削藩，以达到集权的格局。汉景帝认为，中央尚不稳定，削藩可能导致内战。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">晁错回答：“今削之亦反，不削亦反。削之，其反亟，祸小；不削之，其反迟，祸大。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">汉景帝想，既然削也反，不削也反，那就削吧！结果引发“七国之乱”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">汉景帝又问：“现在爆发七国之乱了，你有什么对策吗？”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">晁错却回答：“臣未料及此。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最后为了平定战乱，景帝只能把晁错腰斩于市。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">苏轼评价晁错道：“世之君子，欲求非常之功，则无务为自全之计。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">晁错一心想要建功立业，却没有万全的计策，以至于最后害死自己。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">职场中的你，想要做出一番事业、出人头地的心理可以理解，但不能太过浮躁。做上司交待的事情之前，要进行周密的筹措谋划和细致的分析研究，想全各种应对之策，切忌只凭自己的臆想行事。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如若不然，在发生突发状况时，你也只会说“臣未料及此”，那又如何能赢得上司的信任与重用？</p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">8、</span>八戒：</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">虚应故事，敷衍了事不入心</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">深山古寺中，一个小和尚的工作就是日复一日、年复一年地撞响寺庙的大钟。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对他来说，在这样枯燥乏味的生活中，干着撞钟的简单的工作，给寺院带来的不过是确立作息时间而已，任何人都可以胜任，没有什么特殊的地方。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">拥有这样想法的他，虽然无聊但还是继续待在寺院，又敲了一年钟。有一天，小和尚忽然被调到后院劈柴挑水，原因是他不能胜任敲钟的工作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这让小和尚非常不服气，他找到了方丈问：“为什么说我不能胜任敲钟的工作？是我敲的钟不准时、声音不响亮吗？”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">方丈微微一笑说道：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“钟声不仅是寺里的作息标准，更是唤醒众生的利器。所以，钟声不仅要宏亮、圆润，还应深沉而悠远，你敲钟的钟声很响，但不着边际、没有力量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">没有力量是因为你完全不了解敲钟的深刻意义，钟即是佛，没有钟代表了没有佛。一个不虔诚、不敬业、敷衍了事的僧侣，无法胜任神圣的敲钟工作。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">《圣经》教导基督徒：要在小事上用心，神才会把大事托付与你。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">职场工作具有周期性和重复性，有些员工在重复工作中消磨斗志、消极应付、日渐沉沦，而有些员工则在重复中总结提高、昂扬向上、追求卓越，这两种态度的不同结果可想而知。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">做人做事都要有起码的原则，职场同样也是如此。请你将职场“八戒”铭记于心，时刻提醒自己，不要逾越。共勉~</p>
