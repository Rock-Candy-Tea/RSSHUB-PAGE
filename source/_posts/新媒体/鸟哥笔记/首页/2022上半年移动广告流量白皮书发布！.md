<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>游戏持续买量但投入谨慎</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>纵观2022上半年移动流量大盘，发现：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、相比去年同期，2022上半年<strong>工具行业投放力度减弱</strong>，<strong>文化娱乐、社交婚恋行业投放力度明显增强</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、<strong>手游行业Q1期间整体呈现疲软态势</strong>，4月经历版号重启后，短时间迎来了买量小爆发，在Q2期间买量出现小幅增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、<strong>综合电商成为Q1买量黑马</strong>，自2021下半年起，游戏、教育广告主投放低迷，<strong>空出不少流量转移为电商的重定向广告</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、<strong>购物、社交、娱乐类App</strong>长期保持高强度投放，三者占据将近<strong>70%</strong>移动广告投放数，工具类在数量上占据绝对优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5、<strong>在投应用六成以上是游戏，游戏广告占比却只占20%</strong>，游戏广告主更需要通过买量实现用户增长。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>买量赛道日渐拥挤</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022上半年，AppGrowing 共追踪到<strong>8000+非游戏应用在投</strong>，数量上为去年同期的<strong>3倍</strong>，实现跨越式的增长，买量赛道越发拥挤。下面盘点2022上半年<strong>游戏、非游戏推广排行榜、App Store下载榜</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>2-1 推广排行榜：战争类手游、购物类应用势头猛</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、<strong>SLG强势引领手游买量市场</strong>，<strong>三国志·战略版、文明与征服</strong>稳居前二，仙侠旧游<strong>一念逍遥、永夜帝君</strong>，传奇类传奇<strong>1.76怀旧版、美杜莎传奇</strong>，网赚类<strong>口袋猜成语、成语闯关记</strong>等投放强势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>2、淘特、拼多多位列应用推广版前二，目前<strong>电商广告是移动广告买量主力军</strong>，虽然上半年疫情对电商行业的冲击蛮大，但购物类应用整体投放力度并没有回落。</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>金融行业由于服务于人群的特殊属性，<strong>推广目的多以表单留资为主</strong>，以表单留资为推广目的TOP10品牌主中，<strong>主推借贷类产品</strong>。在当今泛娱乐化时代，金融广告主们也紧跟时代潮流，<strong>投放素材多以视频的形式</strong>，以娱乐化的形式传达产品价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/110903488862d119d24ad460.38203470.jpeg" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据AppGrowing所追踪到的移动广告情报显示，<strong>近两年来阅读类App广告数占比位居全品类第六，维持在6~7%</strong>。在App数量上，<strong>阅读类应用数量占比有波动上升的趋势</strong>，属于重点买量行业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在2022上半年阅读类App推广榜中，<strong>七猫免费小说、番茄小说、番茄畅听</strong>长期位居前三，投放力度势不可挡。</span></p>
