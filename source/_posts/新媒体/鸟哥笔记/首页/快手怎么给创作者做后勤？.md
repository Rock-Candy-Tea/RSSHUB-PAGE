<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年<a href="https://www.niaogebiji.com/tag/%E5%BF%AB%E6%89%8B" target="_blank">快手</a>的光合创作者大会，把来自各地的创作者拉到了海拔2400米艳阳高照的丽江，狠狠“光合”了一把。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6347123162e4e79643c701.99380476.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大会现场，几位快手高管同时坐在台上，聊了聊各自的OKR，以及他们怎么看待和解决创作者最关心的一些问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，流量，比如，有了流量能不能赚到钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手作为一个平台，并不直接生产内容，也不直接生产商品，所以创作者对快手来说至关重要。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/157952632662e4e7966ee2e1.02729088.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如果说早期快手和创作者的关系是创作者创作内容、快手提供舞台，那么现在更像创作者自成平台，而快手是“支撑平台的平台”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有一个细节，为了体现平台对创作者的重视，跟以往大会议程不太一样，今天三位创作者都先于平台高管登台演讲。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大会的内容，也基本围绕如何解决创作者的问题展开，快手主要从<strong>资源、服务、产品</strong>三个方面全面升级了创作者服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>具体怎么做，我们从大会中摘选了3个关键词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>内容扶持</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>按照惯例，平台的年度大会，我们还是先从钱讲起。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据介绍，<strong>过去一年，快手帮超600万的创作者获得了千亿的流量扶持，累计涨粉超过150亿。过去半年，快手创作者的总收入同比增长超过25%。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手高级副总裁、主站业务负责人王剑伟提到，去年，快手拿了几十亿的预算补贴给平台认为更优秀的创作者，“他可能现在还在成长期，还没有赚到钱，但我们希望平台能给他一些帮助，帮他走过困难期”，这笔预算今年会变得更多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如往年一样，今年的快手创作者大会也推出了一系列流量、现金方面的扶持计划，大方向还是倾斜优质内容，吸引优质作者：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一、全年拿出十亿现金、</span></strong></span><span><strong><span>千亿流量补贴创作者</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大会上，快手宣布升级光合计划、光合优创，全年拿出十亿现金、千亿流量补贴创作者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手创作者服务团队负责人袁帅介绍，光合计划会是快手整个创作者运营的基础设施建设，预计在近一两个月陆续上线，目标覆盖一万粉以上的全量创作者以及万粉以下高成长型创作者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而光合优创，可以理解为光合计划的升级版，快手会对光合计划中的高成长型创作者权益进行加码，为优创作者推荐商单，享受1V1专员服务等支持。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/156381557462e4e796dceef4.23798962.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>二、启动“快手之星”，每年最高120万，百亿级流量</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“快手之星”是昨天快手新启动的一个项目，对符合要求的“高潜创作者”，快手打算拿出平台级的重磅资源，一步到位，直接把他们推进到可商业变现的阶段。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前“快手之星”主要由创作者/机构报名，平台筛选，通过后进行签约。“快手之星”主要有几项基础权益：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1. 保底现金权益，每年最高120万，保障全年收益;</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2. 百亿级的流量级，助力成长;</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3. 除了站内的现金资源和流量资源，会给全网10亿+的曝光资源，帮创作者在快手之外的内容平台进行曝光;</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>4. 专属的定制策划、VIP客服。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>三、赋能直播、短剧、电商等细分领域</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除上述扶持政策外，昨天下午场也陆续推出一些激励内容。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如直播领域，快手将拿出5亿现金和30亿流量，激励优质<a href="https://www.niaogebiji.com/tag/%E5%86%85%E5%AE%B9%E5%88%9B%E4%BD%9C" target="_blank">内容创作</a>及赋能主播成长；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短剧领域，升级分账扶持政策，在原有S+/S/A的基础上，新增激励成长模式P（千次有效播放分账5元）；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手电商方面，今年6月“青云计划”宣布升级，在今年第三季度结束之前，快手电商每天将拿出8亿公域流量，覆盖短视频和直播，定向扶持平台优质潜力主播和品牌冷启阶段成长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手高级副总裁、电商业务负责人笑古提到，在快手信任电商生态下，内容型垂类达人可以为消费者提供独一无二的“体验价值比”，与快手一起“重塑10亿消费者决策”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“未来电商更多的空间在做好短视频、直播内容，把消费者吸引住、留住，再通过好的内容+好的产品+好的价格促成复购，这一方向也正是内容创作者擅长的。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，昨天，王剑伟特别提到快手扶持内容的衡量标准：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“只有一个衡量标准，就是他们做的东西能够让社会觉得有价值，能够让用户觉得被感动，能够真正给我们的老铁、我们的生态带来一些更深层次的共鸣。”</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对平台来说，除了有趣，内容的有用、有价值、有社会效应变得越来越重要。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如在快手上拍摄生活纪实视频的李娃娃，她的一条体验进厂视频在全网播放量过亿，题材关心的是真实的人间烟火，所以击中了普通人的情感共鸣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/123292677262e4e7977151d3.36489792.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王剑伟说，希望快手不再是一个纯粹娱乐“杀时间”的软件。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>创作服务</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说去年快手创作者大会的关键词是“私域”，那么今年的关键词是“创作者服务”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>分享几组昨天快手大会上分享的数据：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一、截至2022年6月，快手万粉创作者超200万，创作者粉丝总量突破2639亿。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，快手的创作者规模、创作者发布作品规模还在持续增长，创作者的粉丝总量也在增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/45361555462e4e7979bef49.15268475.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这意味着整个生态规模增长的同时，并没有稀释每个创作者分到的粉丝数量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>二、 创作者的互动数据（点赞、评论、转发及收藏）也一直保持高速和稳定的增长，说明创作者粉丝侧的质量保持稳定。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>三、2016年就在快手发布作品的创作者中，万粉和十万粉创作者到今天还在活跃发布作品的比例分别超过70%和80%，百万粉创作者的创作活跃比例达到94%。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当用户与创作者生态相对稳定，平台是否具备成熟的创作服务能力，显得至关重要。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手是这样推进创作者服务标准化的：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1. 升级产品</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期，快手升级了快手创作者服务团队，同时推出了一个专属于创作者的App，目前正在内测。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王剑伟说：“希望创作者用这个App能帮他了解平台什么内容在火，做什么内容能赚钱，别人为什么可以获得更多流量，希望为好的创作者配备一对一的人工服务，把他遇到的问题最快地解决掉。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如该App提供逐秒解读内容功能，可以逐秒观看一条视频在播放过程中的关键数据变化：第几秒获得了非常多赞，第几秒突然有大量退出。帮创作者研究内容，了解趋势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据介绍，该App还将上线智能引导助手，为创作者提出账号发展建议。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2. 让作品的审核、流量规则更透明</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期，快手社区生态团队在内测诊断助手功能，可以在创作者发布视频前，预先判断该内容是否会命中平台的相关规则，给出一些诊断的结果和通知，并给出建议。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3. 继续升级原创保护能力</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去半年，快手将侵权视频的检出率从90%优化到了98%，每秒可检测500个视频。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>检测后，创作者可以通过粉丝迁移功能，将仿冒号的粉丝迁移到原创账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个仿冒号粉丝迁移功能早在去年快手创作者大会上就有提及，当时给很多原创作者带来不小的鼓舞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截至目前，快手帮485位创作者迁移了超1亿的粉丝。不久前，美食创作者“韩小浪本浪”用这个迁移功能，粉丝量从33万涨到了112万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>商业基建</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何解决创作者的商业变现，快手现阶段的重点在于商业基建。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手高级副总裁、商业化负责人马宏彬聊到，两三年前聊创作者变现，更多是聊怎么接商单，怎么让大家在直播之外有一些不同的收入来源，而这两年，很多创作者直接进入到了经营环节。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这促使他们思考，“商业化业务除了引入外部预算，另一个更重要的作用是把商业的力量和整个内容的力量结合起来，让大家在快手上有机会把经营这件事情做起来。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>平台能做什么？大致有两点：<strong>引入预算；提供商业基础设施。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>帮创作者引入外部预算，这部分主要依托磁力聚星业务，提供商业基础设施，吸引更多品牌来快手投放创作者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此前，针对创作者内容创作能力强但缺乏货品供应链的问题，快手电商推出人货匹配的SAAS工具快分销平台。<strong>过去一年里，快手电商的动销商家数翻了一倍（动销指卖出去商品的人数）。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，快手还提供付费课堂等多样的变现模式。各种类型、各个成长阶段的创作者们都可以借助平台的商业基建获得经济收益。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回到开头说的创业者最关心的问题，通过内容扶持、创作服务、商业基建，快手的解决方法主要在于“提供支撑”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>爆款和走红都是暂时的，良性的内容和商业生态环境，才是创作者能够持续成长的根基。</span></p>
