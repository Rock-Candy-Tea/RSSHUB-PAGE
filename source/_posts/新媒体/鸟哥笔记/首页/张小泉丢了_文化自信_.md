<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span><img src="https://ngbjimg.xy599.com/73529912862d9de2c690c75.35931443.png" referrerpolicy="no-referrer"></span></strong></span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span><strong>公关是利器，也是牢笼。</strong></span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>好的公关能在第一时间内做出回应。根据<a href="https://www.niaogebiji.com/tag/%E5%8D%B1%E6%9C%BA%E5%85%AC%E5%85%B3" target="_blank">危机公关</a>5S原则中“拼速度”是第一原则，并非是第一时间就做出回应，而是在第一时间找出真相、给出方案、做出应对。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>聊下<a href="https://www.niaogebiji.com/tag/%E5%BC%A0%E5%B0%8F%E6%B3%89" target="_blank">张小泉</a>事件的本身，并没有在第一时间做出回应，也没有在第一时间内给出解决方法。7月13日，网友曝光了“拍蒜”断裂的菜刀，张小泉官方7月15日才做出回应：“常规刀可拍蒜，但不是所有刀都适合。”</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这难道就是张小泉公关处理危机事件的速度和态度吗？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一般舆论事件会在4-6小时内就会由小变大，而24小时就会让危机全面发酵。张小泉却48小时后才做出第一份声明道歉。这是对大众消费者的漠视？还是品牌自身的价值观出现了问题？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如果说，这些只是张小泉处理公关手段和方法，我们也不做过多的讨论。7月18日，网友扒出了张小泉总经理夏乾良在某一次采访中称：“你学了几十年的切菜是错的……所有的米其林厨师都不是这样切的。”</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><img src="https://ngbjimg.xy599.com/3893717662d9de2dd051a2.96891579.png" referrerpolicy="no-referrer"></span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这段采访不管是舆论事件发生前，还是发生后。笔者认为，中国人的做菜方式肯定不能与米其林的西餐方式做比较，中餐与西餐有着本质区别，制作过程也不一样</span></strong><strong><span>。</span></strong><strong><span><strong>张</strong></span></strong><strong><span><strong>小泉作为传统的国货品牌，前提条件下是要满足国人的需求，传递中国文化，这才是中国品牌走出去的理由</strong></span></strong><strong><span>。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>中国品牌走出的前提，是要助力中国传统文化走出去。如果一个品牌连“文化自信”的底气都没有，何谈走出去？这应该是品牌该反思的地方，尤其是国货品牌。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这是张小泉第一大问题，其二是</span></strong><strong><span><strong>“教育”消费者</strong></span></strong><strong><span>。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong>能够让消费者买单的固然是品牌的价值，而不是花钱听你教育。消费者教育是指对消费者进行有目的、有计划地传播消费知识，宣传消费观念，培养消费技能，交流消费经验，提高消费质量的活动。</strong></span></strong><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span><strong>品牌的尽头是公关，公关的背后是“教育”消费者。</strong></span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这固然是错误的行为，任何公关行为都不能以“教育消费者”为手段。很显然，张小泉的回复与道歉方面都很不妥当，也激起了网友的高度不满。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>尤其是在移动互联网时代，“情绪”和“价值观”是传播的关键。而所有事件中，价值观的传播决定了事件的发酵程度。且不说，张小泉菜刀不能拍蒜，但也不能质疑中国传统文化。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这好比是说，一个仅有400年历史的张小泉，在质疑2000多年的菜刀文化。很多网友就不乐意了，菜刀这种家庭必备的厨房生活用品，可以追溯到春秋战国时期，一直都在扮演者“拍蒜、拍黄瓜”的角色之一，为什么到张小泉就不行了呢？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong><span><img src="https://ngbjimg.xy599.com/201481378362d9de2e085445.58918979.jpeg" referrerpolicy="no-referrer"></span></strong></span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><strong>一把菜刀该怎么用，不是由品牌方来决定。</strong></span></strong><strong><span>张小泉非要消费者遵守自己的使用标准，这难道就是一个品牌该有的价值观吗？虽然，事后张小泉官方也做了解释，菜刀的硬度决定了是否能撑得住“拍蒜”的行为。但这不是品牌公关给出的理由。尤其是在网友声讨的节骨眼上，品牌要做的并不是解释问题的本身，而是放低自己的姿态。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>有时候，放低姿态并不</span></strong><strong><span>是“懦弱”的行为。所以说</span></strong><strong><span>，</span></strong><strong><span><strong>一个品牌最可怕的不是竞争对手，也不是残酷的市场环境，而是高高在上的态度。</strong></span></strong><strong><span>毕竟，能有多少消费者愿意花了钱还想听你在那各种教育。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“教育”消费者一定是品牌要做的事。因为“一家之言”必定会让品牌成为众矢之的，品牌要做的应该贴近消费者使用习惯，让用户感受的品牌的价值能够给自己解决问题。因为</span></strong><strong><span>，</span></strong><strong><span><strong>离消费者越近，品牌才会越成功</strong></span></strong><strong><span>。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一个百年品牌，不管怎么创新，都应该把自己的傲慢收一收，不能丢了“文化自信”，也不能去“教育”消费者。这是老字号的灵魂，这个根，不能断。尤其对于企业来说，传承与发展中国文化将是企业实现长远发展的必经之路，是企业发展过程的重要一环。</span></strong></p>
