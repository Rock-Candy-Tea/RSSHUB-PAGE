<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81173431362d8e00d720cf1.38725891.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>步行太远、骑车好累、驾车拥堵、停车很难……在“出行自由”的需求下，“小电驴”成了“打工人”必备的交通工具。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据智研咨询，2020年，中国电动两轮车累计社会保有量为3.4亿辆。而《中国两轮<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%8A%A8%E8%BD%A6" target="_blank">电动车</a>行业白皮书》显示，2021年，中国电动两轮车累计销量为4100万辆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这其中，<a href="https://www.niaogebiji.com/tag/%E7%88%B1%E7%8E%9B" target="_blank">爱玛</a>科技（603529.SH）一家就卖出835万台，占到总销量的两成以上，带动公司营收同比增长19.33%至153.99亿元，归母净利润同比增长10.94%至6.64亿元。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在2022胡润全球富豪榜上，爱玛科技董事长张剑与其女张格格，以135亿元的财富成为了河南省商丘市最富有的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进入2022年，大概率上述几个数据都会更好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>风起于青萍至末。7月13日，爱玛科技发布半年度业绩预增报告，预计今年上半年净利同比增长120%。该利好消息刺激下，7月14日，爱玛科技股价触及涨停，并在后续几日持续上行，7月20日盘中创出历史新高65.08元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/45301278162d8e00dbf80f5.03019413.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;" data-darkmode-bgcolor-16208949364564="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949364564="#fff|rgb(255, 255, 255)"><span data-darkmode-bgcolor-16208949364564="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949364564="#fff|rgb(255, 255, 255)" data-darkmode-color-16208949364564="rgb(237, 157, 0)" data-darkmode-original-color-16208949364564="#fff|rgb(255, 169, 0)" data->1、靠卖“小电驴”，卖出一个商丘首富</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>受总设计师南巡讲话鼓舞，不安于现状，放弃“铁饭碗”……爱玛科技创始人张剑的发迹史，具有鲜明的时代特征。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1990年，大学毕业不久的张剑辞去稳定工作，在商丘当地开办了一家名为“盛天余”的自行车零售门店，由此开始了自己在两轮交通工具领域的打拼生涯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>恰逢新兴事物从小众走向大众，张剑的自行车生意乘风而上，卖的红红火火。没过几年，他就将<a href="https://www.niaogebiji.com/tag/%E5%95%86%E4%B8%9A" target="_blank">商业</a>模式从零售拓展到了批发。到1999年底，“盛天余”已经是销售网络辐射全国的领先自行车批发商。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>1999年，中国加入WTO的前夜，张剑做出了一个彻底改变他人生的决定：离开商丘，前往天津二次创业。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以选择天津，是因为天津是全国最大的自行车生产基地，产业链极其完备。同年，爱玛科技的前身——天津市泰美车业有限公司正式成立。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实证明，张剑的这个决定是正确的。泰美车业凭借自行车代加工业务，很快站稳了脚跟。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>2004年，政策明确了电动自行车非机动车的合法身份，叠加“禁摩令”范围不断扩大，迫使消费者需求转向。张剑判断“风口”即将爆发，泰美车业着手向电动两轮车迈进。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经过约两年的打磨，2006年，“爱玛”电动车品牌诞生了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/212579501162d8e00e1b1420.19464237.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>然而，在市场培育阶段，嗅到商机的人远不止张剑一个，如何从草莽揭竿的“乱局”中脱颖而出，初出茅庐的爱玛遇到了新的难题。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2009年，泰美车业改名为天津爱玛科技。学习同行新日签约成龙代言后销量扶摇直上的经验，张剑也豪掷3000万元签下华语乐坛天王周杰伦，由其出任爱玛的品牌代言人，并携手至今。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管这笔费用不菲，因占了当年爱玛超过一半的净利润，一度遭受外界质疑。可是，强大的明星效应还是让默默无闻的爱玛电动车迅速走红。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时任爱玛科技总经理余林曾向外界透露过：“我们品牌的营销预算高达5亿元，3000万元只是很小的一部分。周杰伦代言能提升品牌知名度，一年的品牌价值可达20亿元。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>2010年，爱玛销量达到220万辆，超越新日，坐上了行业头把交椅。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而与当红明星合作的习惯，比如范冰冰、EXO、金秀贤、吴谨言等，从此贯穿爱玛发展始终。2021年，公司销售费用大幅上涨38.07%，广告及业务宣传费高达1.12亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/48164330962d8e00e7c21a7.37036071.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><strong data-darkmode-bgcolor-16208949364564="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949364564="#fff|rgb(255, 255, 255)"><span data-darkmode-bgcolor-16208949364564="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949364564="#fff|rgb(255, 255, 255)" data-darkmode-color-16208949364564="rgb(237, 157, 0)" data-darkmode-original-color-16208949364564="#fff|rgb(255, 169, 0)" data-><strong data-darkmode-bgcolor-16208949474588="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949474588="#fff|rgb(255, 255, 255)" data-darkmode-color-16208949474588="rgb(163, 163, 163)" data-darkmode-original-color-16208949474588="#fff|rgb(178, 178, 178)" data-><strong data-darkmode-bgcolor-16208949364564="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949364564="#fff|rgb(255, 255, 255)"><span data-darkmode-bgcolor-16208949364564="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949364564="#fff|rgb(255, 255, 255)" data-darkmode-color-16208949364564="rgb(237, 157, 0)" data-darkmode-original-color-16208949364564="#fff|rgb(255, 169, 0)" data-><strong data-darkmode-bgcolor-16208949474588="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949474588="#fff|rgb(255, 255, 255)" data-darkmode-color-16208949474588="rgb(163, 163, 163)" data-darkmode-original-color-16208949474588="#fff|rgb(178, 178, 178)" data-><strong><span>图源：爱玛集团官网</span></strong></strong></span></strong></strong></span></strong></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般来说，处在行业爆发期，好的策略往往能事半功倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据国家统计局数据，2009年到2013年的短短5年间，中国电动自行车的产量从29万辆暴涨到了3695万辆，年均复合增长率达到34.21%，爱玛依靠“明星代言+精准营销”，以及相对低端的产品定位，甩开雅迪、新日、台铃等友商，一路成长为电动两轮车“最亮仔”，且一直把持第一的位置到2016年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这为张剑日后登顶“商丘首富”奠定了坚实的财富基础。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;" data-darkmode-bgcolor-16208949364564="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949364564="#fff|rgb(255, 255, 255)"><span data-darkmode-bgcolor-16208949364564="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949364564="#fff|rgb(255, 255, 255)" data-darkmode-color-16208949364564="rgb(237, 157, 0)" data-darkmode-original-color-16208949364564="#fff|rgb(255, 169, 0)" data->2、从老大到老二，爱玛被雅迪反超</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年是张剑的“滑铁卢”。这一年，爱玛被雅迪反超，从第一掉到了第二，且在日后，无论是消费市场、资本市场，还是个人身价，两者间的差距越来越大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公开资料显示，2021年雅迪卖出了1386万辆电动两轮车，同比增长28.3%，再度当选行业首个年销量突破千万的品牌，而爱玛为835万辆，相差551万辆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>体现在财务数据上，2021年，雅迪控股营收269.87亿元，同比增长39.27%，比爱玛科技多录约116亿元；净利润为13.69亿元，同比增长43.04%，比爱玛科技多赚近一倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/113590099762d8e00eb82e41.20503246.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，2020全球供应商大会上，爱玛科技集团国内事业部总裁高辉表示：“2021年，国内事业部产销目标必保1600万辆，冲刺1800万辆。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>换而言之，加上电动三轮车和自行车，爱玛共计867万台的销量，仅完成了“必保”目标的54％，“冲刺”目标的48％。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>细化到盈利能力上，爱玛科技也不及雅迪控股。2021年，雅迪控股毛、净利率分别为15.21%、5.06%，爱玛科技则为11.72%、4.34%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/121593344462d8e00ee8e681.93949209.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，追溯二者的发展轨迹，爱玛原本优势出众。早在1999年，张剑完成原始资本积累的时候，雅迪创始人董经贵刚刚服完兵役，借助汽车维修兵的经验，才在无锡捷达摩托车厂，找了一份驾驶员兼推销员的工作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就算在2017年，雅迪销量406万台，爱玛销量377万台，相差不过30万台。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>究其原因，部分为爱玛在渠道建设上步伐较慢。2019年爱玛经销商数量为1933家、雅迪经销商数量为2155家，到2021年，雅迪经销商数量已攀升至3353家，而爱玛经销商数量则为2000余家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>门店方面，截至2021年末，雅迪门店数量超过3万家。与之对比，爱玛在2021年年报中提到，截至报告期末，拥有终端门店数量超过2万个。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>至于盈利能力，雅迪自2015年确立了“更高端”的战略后，产品价值和价格均有所提升，爱玛坚持“时尚好车”的品牌主张，用户群偏中低端，对单车毛利有一定的压制。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在IPO进程上，曾经的小弟们都纷纷跑到了爱玛前面。2018、2019年，爱玛连续两年递表，最终由于“尚有相关事项需要进一步核查”被取消资格，直到2021年才闯关成功。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反观同行小伙伴，2016年、2017年，雅迪和新日先后登陆资本市场，2018、2020年，“新势力”品牌小牛电动和九号公司也顺利上市。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>截至7月20日收盘，雅迪控股市值550.9亿港元，爱玛科技市值361亿元。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>业绩、市值的明显沟壑，亦深刻影响着创始人堆金叠玉的力度。2022胡润全球富豪榜上，董经贵、钱静红夫妇，以235亿元的财富稳坐安徽省六安市首富宝座，相较之下，张剑与其女张格格虽荣冠河南省商丘市首富，但135亿元的财富值比前者少了整整100亿元。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong style="text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;" data-darkmode-bgcolor-16208949364564="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949364564="#fff|rgb(255, 255, 255)"><span data-darkmode-bgcolor-16208949364564="rgb(25, 25, 25)" data-darkmode-original-bgcolor-16208949364564="#fff|rgb(255, 255, 255)" data-darkmode-color-16208949364564="rgb(237, 157, 0)" data-darkmode-original-color-16208949364564="#fff|rgb(255, 169, 0)" data->3、从蓝海到红海，谁能笑到最后？</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>爱玛从第一到第二，雅迪从后起到领先，七年间，中国电动两轮车市场风云变幻，驱动行业从成长走向成熟，业内公司从蓝海踏入红海。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>锐观网数据显示，2020年，中国两轮电动车市场规模达1046亿元，2021-2023年两轮电动车的规模增长贡献将主要来自新国标替换产生的销量增长；艾瑞咨询发布数据显示，2021年，我国两轮电动车市场年销量为4100万辆，比2020年下跌13.9%，是2018年以来的首次销量负增长，且跌幅为2013年以来的新低。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>从时间线来看，2019 年4 月15日，新《电动自行车安全技术规范》国家标准正式实施，随着三年过渡期结束，电动两轮车市场增量空间几近见顶，产业链企业均面临存量搏杀挑战。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，雅迪董事长董经贵曾在2020年末对行业做出过“三年过渡期”的预判：“明年（2021年）的竞争将会非常激烈，会有很多的品牌垂死挣扎,也会有头部企业实现高增长，到后年（2022年），行业将逐步进入成熟期，届时整个电动车产业链会实现高质量、高附加值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该背景下，通常那些市占率占优，提早感知趋势且付诸行动的企业，能享受到低端产能出清过程中的最大红利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据天风证券研报，2018年-2021年，第一梯队的雅迪、爱玛盈利能力背道而驰，前者单车结余从86元增加到99元，对应净利率从4.36%上升到5.06%，后者单车结余从94元减少到78元，净利率从4.79%下降到4.34%。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>节点财经了解到，在4000-5000价格段，雅迪销量排名第一，小牛和爱玛分列二三位;在5000-6000元价格段中，小牛、九号产品销量领先;在6000-7000元价格段中，九号品牌车型产品销量更高，小牛和雅迪分列二三位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>由此可见，雅迪、小牛、九号在中高端产线相对更具实力。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>值得注意的是，小牛正从高端产品逐渐向终端产品渗透，这对固守中低端“大本营”的爱玛来说是不利的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>市场格局方面，基于激烈的市场博弈，品牌生存竞争加剧，电动两轮车企业分化明显。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，爱玛科技2018年电动自行车产量为436.65万辆，市占率为13.32%，与雅迪“平起平坐”。然而，到了2019年，雅迪市占率为14.1%，爱玛市占率则收缩至10.4%；2021年，雅迪市占率28%，雅迪市占率17%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果把上述市占率数据绘制成曲线，我们不难发现，雅迪加固自身地位“护城河”的速度要远快于爱玛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这也印证了一条普遍的经济学规律，市场份额总是会越来越向头部企业靠拢。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，爱玛有意转向积极扩张。2021年，公司与重庆铜梁高新技术产业开发区管委会签署《项目协议书》，将投资建设爱玛西南制造基地；近期，其公告拟发行不超20亿元可转债，计划在丽水青田新建制造基地，同时引进高端的生产设备和自动化生产线，全方位提效扩产。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>只是，当行业进入洗牌阶段，雅迪“强者风范”的气场基本铸成，机会的天平还会偏向爱玛吗？</strong></span></p>
