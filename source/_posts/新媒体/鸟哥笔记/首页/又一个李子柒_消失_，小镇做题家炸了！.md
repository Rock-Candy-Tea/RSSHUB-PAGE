<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>果然，万物皆可刺客，</span><strong><span>从雪糕刺客拉开大幕，各路刺客前来报到：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>话梅刺客、<span>编</span>制<span>刺客、</span>荔枝刺客......</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>现在又出现了一个</span></strong></span><span><strong><span>版权刺客</span></strong></span><span><strong><span>，版权刺客不陌生，它的真面目就是版权流氓。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这两天霸屏的《<a href="https://www.niaogebiji.com/tag/%E8%B0%AD%E8%B0%88%E4%BA%A4%E9%80%9A" target="_blank">谭谈交通</a>》事件听说了吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就是一个叫谭乔的普通人，遭遇了一出</span><span><strong>“我侵权我自己”</strong></span><span>的笑话：因为自己主持的作品，而成为被告。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1583862762cf6d27514281.34178339.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来自微博热搜</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单来说，成都电视台把自家节目《谭谈交通》的版权，授权给成都游术文化传播有限公司，后来成都游术文化传播有限公司告谭乔侵权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个热搜可把网友看怒了：</span><span><strong>小镇做题家连普法都不配？</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谭Sir账号本来<span>有</span>300多个<span>视频</span>，现在已经消失不少。除了他自己的被清空，B站网友二次创作的作品，也躲不过下架和赔偿的命运。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>现在谭Sir一脸懵逼，在对方没有提前告知的情况下，就要面临被索赔。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但仔细想想，也是不是没有先兆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>出于普法传播，谭Sir一开始跟视频记录方，并没有太在意版权的问题，这埋下了后患。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20108582462cf6d292670e8.02184888.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来自@谭乔 微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谭乔自己寻思着，估计是去年做了些访谈节目和专题，曝光多了，把节目又给带火了一把。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以估计有人开始盯上了节目的商业价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/58833552062cf6d2a0dbcc7.46771769.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来自@谭乔 微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>都怪当时没留个心眼，现在谭还得想办法证明</span><strong><span>“我是我自己<span>”</span></span></strong><span>，才可能躲过侵权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>舆论压力给到了</span><span><strong>成都电视台</strong></span><span>，它的发声也耐人寻味：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1、授权给成都游术文化传播有限公司；</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2、没有针对个人，暗示谭Sir不当获利。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3、“索赔千万”是不存在的。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/171781510662cf6d2ac195b8.27771202.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来自@中国新闻网 微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>原来电视台把卖版权卖给第三方，他们只是进行“代维权”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>只不过</span></strong></span><span><strong><span>“不当获利”</span></strong></span><span><strong><span>这个锅，</span></strong></span><span><strong><span>谭乔不</span></strong></span><span><strong><span>背。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年才玩自媒体的谭乔，自称是个新人，刚开始压根儿不知道B站流量激励收益这事。知道了以后，除了保留必要开支，也把这部分钱挪出来做公益。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/162559786162cf6d2b1187e4.98337755.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span> 来自@谭乔 微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>为什么一个</span></strong><strong><span>节目</span></strong><strong><span>出道多年，现在才来维权？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>没有协议的情况下，节目版权应该归谁？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>电视台为什么会把版权授予给游术文化传播有限公司？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些问题一时半会儿很难说清，只能说谭Sir遇到了强对手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过更可怕的细节在后面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205789392362326078647a25.53043458.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人扒到成都游术文化，公司不大，但背景不简单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公司<span><span>2018年才注册，<span>而且公司</span><span>就</span><span>两个人</span><span>，</span></span></span>后来一查发现注册地<span>是</span>服装店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/167128476162cf6d2b3d3916.29382858.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来自 企查查截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不可思议的是，从今年6月开始，短短一个月的时间，成都游术文化传播有限公司一口气起诉60多次：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>几乎把互联网大厂们都告了个遍，</span></strong></span><span>包括我们熟悉的腾讯、抖音等，罪名几乎是“侵害作品信息网络传播权纠纷”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><img src="https://ngbjimg.xy599.com/194382588362cf6d2b6d3501.20036703.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来自 企查查截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>敢情是个碰瓷的千年老手。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这不得不让人怀疑，起诉《谭谈交通》，会不会也是一起<span>耍流氓事件：</span></span><span><strong>一个空壳公司想要钻法律空子捞钱。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和之前方正字体、潼关肉夹馍协会到处讹诈，果然有得一比。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就连谭乔自己都讽刺：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>当初天天吃李子柒的瓜，没想到今天吃瓜吃到自己身上。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/39932515062cf6d2ba7cdb8.13533312.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来自@谭乔 微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一夜之间，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一个无偿公益的普法节目，变成了一门碰瓷捞钱生意。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一个交通法的顶流，变成了侵权的顶流。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134948845962326079426029.07408166.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友们之所以愤怒，也是因为《谭谈交通》不是个普通的交通普法栏目，是曾经每天放学回家的下饭节目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>都说生活是最好的素材，</span></strong></span><span><strong><span>喜剧的内核是悲剧。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>《谭谈交通》就是一部现实中的荒诞喜剧，拍下普通人最普通的一面。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在节目中，谭Sir和违反交规的驾驶者打交道，进行无厘头的教育，笑点密集，笑点的背后也藏着普通人的心酸。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>最经典的桥段莫过于“二仙桥”。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谭乔逮到走错车道、违规拉货的电瓶车，让开车大爷靠边接受检查。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/132584838462cf6d2c5619c6.09462086.gif" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>还有让人笑着笑着就哭了的富贵大爷。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谭警官遇到拉破旧三轮的男人，车上堆满木柴，还坐着一个男人和一条狗。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/138583360762cf6d337b2eb1.41076017.gif" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>像这样的民间故事还有很多，《谭谈交通》<span>2005年</span>在成都电视台播出，2018年停播，前后录制了3000多期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>虽然节目停更，但灵魂一直都在。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>脱离体制后的谭乔，后来改行当B站UP主，<span>上传</span>以前的<span>节目</span>片段，回访福贵大爷，气球哥等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而节目的各种名场面，带火了</span><span><strong>“只能拉一点点<span>”“</span>腰马合一<span>”</span></strong></span><span>经典</span><span>梗，更是成了UP主剪辑的鬼畜素材。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/106767017862cf6d366d5980.36738943.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来自 B站截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在想想，节目桥段的病毒式传播，也吻合谭Sir做节目的初衷，寓教于乐做普法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是没想到如今，人在家中坐，祸从天上来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>就像一个网友说的：</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>到二仙桥走成华大道，到“富贵庄”不该走邪门歪道。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/138833517962326079dd1f53.73402961.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不得不说，《谭谈交通》内容下架，让人想到李子柒们和MCN那些事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李子柒和微念闹掰，“朱一旦”和张小策分手，大胃王浪胃仙和MCN公司撕逼等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就拿李子柒来说，去年7月停更，到后来与微念对簿公堂，陷入账号、李子柒商标<span>归属</span>等纠纷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李子染、谭Sir等作为IP的人物，创作了烟火气的内容，自己没捞着啥油水，却被版权限制，失去了话语权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就好比明星演电视剧电影，最后版权都不在自己手里，而是在发行方那。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说得不好听一点，“李子柒们<span>”</span>只是高级的打工人而已。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回到《谭谈交通》问题上来，</span><span><strong>无论版权归谁，成都电视台和<strong>谭Sir本来是相互成就，<strong>电视台</strong></strong>不念及谭Sir多年的合作，还反过来到处起诉索赔，吃相多少有点难看。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且如今将内容全网下架，大家无法观看，受伤的终究是用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟《谭谈交通》节目能火，终究是因为谭乔诙谐的人设，因此网友们买单的是谭乔这个灵魂人物，而不是成都电视台的版权。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/116923809762cf6d37a1cde9.19222349.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>来自小红书</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>试想一下，要是平台可以借助节目热度，联手谭乔为大众推出更多民间内容策划，既能继续公益普法，秀一波电视台品牌好感度，还能给IP做商业化运营，那岂不是双赢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>离开谭乔的推广，《谭谈交通》也未必会被人认可。</span></p>
