<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;"><a href="https://www.niaogebiji.com/tag/%E4%BA%A7%E5%93%81%E7%BB%8F%E7%90%86" target="_blank">产品经理</a>的生涯里，我们可以拆分为3个阶段分别从产品经理、再到高级产品经理和产品总监。</span></p>
</section>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>周鸿祎曾经公开的说过：</span><strong><span>“跟着我发财的人，没有一个是靠工资发财的，</span><span>只有和公司一起成长，拿着公司股票才有机会发财，打工是没有机会的。”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>罗振宇曾经说：<strong>“不要为任何集体做贡献，你到这家公司来，你为你自己，你别为老板，</strong></span><strong>你全当这家公司是一王八蛋，三年后你就走，想明白三年后你要在这公司成长成什么样，你所有的算计就是你挣钱、你厉害、你学到东西、你在市场上增值。不要为公司做任何贡献，你的贡献是因为你自己要成长，顺带替公司做的”</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>如果你打算做产品经理的同学，我建议你的心态一定要以修炼成长的心态去从事，因为做产品经理久了，最终发</span><span>现你做的需求99%都是为了成就领导和老板价值去做的，所</span><span>以如果没有这种沉淀经验、打怪的心态，你是难以在职业里存活下来的，因此我建议你看看今天这篇不同产品经理的状态。</span></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>1.产品经理的阶段</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个阶段，<strong>做产品总是希望做自己喜欢的产品。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般在公司里面提产品经理，就意味着他的工作时间在3年以内。这部分的同学在工作中主要的工作是原型、文档、和具体的功能设计和页面设计。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个阶段产品经理的门槛相对较低，可以来源于各种专业、各个维度的同学，其中转型成为产品经理的有设计、运营、测试、开发，同时大学专业即使不是计算机的也能够去做产品经理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/56252414662bc6b9469ce10.72637366.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然也有不少产品经理会直接参与整个系统的规划，但是往往需要高级产品经理或者老板们进行规划。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>产品经理考核：</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品经理的考核最常见的是需求的上线时间是否延期，以及上线中的BUG数，和版本带来的产品数据效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>工作态度：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做自己喜欢的产品，满足老板要求就可以，同时</span><span>这个阶段的产品经理学习能力最强，对于新型的技术、方向和沙龙论坛尤其感兴趣，愿意付费去学习产品经理知识。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时也会有因为在产品设计都是做自己喜欢的产品功能设计，可以获得极大的版本一次又一次的上线带来的成就感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>产品经理薪资待遇：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一线城市：8-16K</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二线城市：8-12K</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>2.高级产品经理<br></span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品经理从业3年以后，逐步掌握了行业业务知识，就成为高级产品经理，主要工作包含了产品经理之外，还会包含产品规划，<strong>比如规划从一个点子到一个成熟的产品线。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品规划的工作会涉及到产品的立项到项目建立，包含了产品架构图、产品商业模式、产品生命周期运营策略等。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/42101721262bc6b948668c4.93231845.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但并不是高级产品经理都来自于产品经理， 我见过一些做电商的互联网公司，高级产品经理来自于运营或市场，然后再招聘产品经理来完成具体的原型产品设计。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>高级产品经理的高级主要<strong>是因为业务的精通</strong>，比如系统功能的运转流程，比如你做开放平台，你会非常熟悉开放平台的商家端、用户端、以及所对应开放能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如你做面试C端的商城APP，那么就是熟悉公司的商品种类、商品销量和商品的生产时间，同时知道自家公司的用户画像，清楚什么样的商品好买什么样不好卖。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/172002129762bc6b95072ab0.15625656.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抛开团队内部的，<strong>高级产品经理还会评估市场对外的打分，比如用户投诉情况、部门之间协同情况、用户成本成效。</strong><br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>高级产品经理考核：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>高级产品经理的考核指标往往就和产品的营收相关了，营收订单数量、流水数据、还有产品数据、用户数据。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>高级产品经理态度：</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>做自己的产品，但是要做用户和客户满足并需要的产品</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比产品经理，高级产品经理更加注重在怎么样完成老板任务和公司目标，而不是到底要不要自己去重新创新做产品，横向考虑产品所需要的资源以及纵向的硬件基础、人员资源基础。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>高级产品经理的薪资待遇：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一线城市：25-45K+（可能有）股票、期权<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二线城市：20-35K+（可能有）股票、期权</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>3.产品总监</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人在产品经理的<a href="https://www.niaogebiji.com/tag/%E8%81%8C%E4%B8%9A%E5%8F%91%E5%B1%95" target="_blank">职业发展</a>上，<strong>都卡在了高级产品经理上。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>假如不牺牲自己对大厂、高薪资、城市的追求，<a href="https://www.niaogebiji.com/tag/%E8%81%8C%E4%B8%9A%E7%94%9F%E6%B6%AF" target="_blank">职业生涯</a>上就很难去做产品总监了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>30岁以后是产品总监的集中年龄段，在大厂里面的产品总监工作就要时间就更长（要10年以上），所以大部分35岁左右的产品总监实际上是在中小企业里。大厂反而在35-40岁</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>产品总监的来源</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/97408784762bc6b95433164.29172766.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我见到的产品总监来源可以来自于上面5个职位，对于公司更加偏向互联网的，产品经理更加从高级产品经理或者开发来转型。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果是以业务部门主导的，比如我物流、电商、O2O，这种业务流程复杂的，那么产品总监更加以业务主导的部门同学来负责，比如有市场、运营、甚至是还有人力资源负责人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是因为很多企业对人的管理更加比产品重要，尤其是公司现在是探索期没有直接的盈利方向，那么很可能是人的管理同学做产品总监更加有效。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>产品总监考核</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品总监的考核就比较残酷了，我见过很多产品总监，刚刚入职开始第一天到离开团队不超过半年时间，就被淘汰了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>能够坚持下去的产品总监首先就要担任营收目标，不仅是营收成绩还要达到目标，这<strong>个目标可能是来自老板的，也可能来自股东的，还有可能是职场斗争。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>营收指标从订单数、到流水、到用户活跃数等，不同的产品因生命周期不一样，所对应的考核目标不一样。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从项目立项开始，产品总监也有很大可能过来负责实现老板目标的，所以产品总监往往在入职前一定要搞清楚企业和团队的背景，不然就可能走进一个坑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>产品总监的态度</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span>不一定要自己做产品，但要有用户和客户能讲故事的产品</span></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果是说从公司升上去的产品总监，则已经不是聚焦在产品工作上了，到底怎么样可以让业务变得更好，产品应该多一些什么能力，这种能力不止是软件、系统层面上的，也有可能是资源上的（比如外链、内容整合）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果是从入职一家公司成为产品总监，那么产品总监的心态就是新官上任三把火了，就比较灵活了。但很多产品总监入职第一件事就是写产品规划PPT，或者业务计划还有搞关系。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/24094465662bc6b95796ca3.68844511.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对团队人员和资源进行清理，该裁员就裁员，该招募就招募，其次开始跟着计划就开始以目标KPI来推进工作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有产品总监往往会和技术负责人、公司的各个业务部门带头人走下人情，毕竟后续的合作推进，都离不开共赢。看看那个部门的工作更好推进一些。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>产品总监薪资待遇</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一线城市：期权、股票+35-60K （月薪）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>二线城市： 期权、股票+25-40K（月薪）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上就是今天的分享，相信你能够更快的了解产品经理的职业生涯。</span></p>
