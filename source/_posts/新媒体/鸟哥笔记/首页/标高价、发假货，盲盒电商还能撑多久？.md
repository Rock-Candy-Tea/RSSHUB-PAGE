<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一边宣称“商品不喜欢就兑换”、“超值保底无空包”，而另一边在黑猫投诉平台上，关于叮当魔盒的问题更多都是如“价格虚高”、“商品有假”、“虚假宣传”等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/164118001662db2f27362481.78659263.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：安卓应用商店</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>甚至就连官方媒体都报道过如“叮当魔盒”之类的盲盒APP。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>吸引眼球的宣传往往都带着很难察觉的附加条件，所谓的正品保障也并没有披露可信资质。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>钱花了不老少，</strong></span><span><strong>却买回来一堆废品</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“只需49元就有机会抽到大牌商品”、“39元就能得到外星人高端笔记本”。注册并打开叮当魔盒APP后，到处都充斥着这样的宣传。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提到这样极具诱惑力的宣传标语，李明哭笑不得。“自己在玩试抽的时候，基本上10次当中至少有2-3次都能够抽到非常不错的商品，价值几千元到上万元不等。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/72606684462db2f27abb020.42868566.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：叮当魔盒官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而等到真正付费体验之后，却发现连续抽到的都是几十块钱的东西，之前的大牌商品突然什么都看不到了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，有叮当魔盒的用户表示：“</span><span><strong>这其实就是此类盲盒APP的惯用套路，先把用户通过试抽就能得大奖的方式忽悠进来，等到真正付费之后，中奖概率就会被瞬间调低</strong>。</span><span>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了付费前后调整中奖概率之外，叮当魔盒上的产品价格也非常虚高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/110502810762db2f2822a449.21065118.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：叮当魔盒APP</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如在Elle官方旗舰店上，一套小样只需要42元就可以买得到，然而在叮当魔盒上却标价238元；即便是动辄数万元的外星人水冷游戏主机，也比官方平台上贵出10%左右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>除此之外，叮当魔盒上泛滥的<a href="https://www.niaogebiji.com/tag/%E5%81%87%E8%B4%A7" target="_blank">假货</a>也让不少用户感受到了深深的欺骗。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有消费者表示自己在第1次下载叮当魔盒之后，累计充值2000多元，并把所抽物品进行兑换，最终在APP内购买了5只兰蔻小白管。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而收到货之后，发现平台寄过来的产品并不是自己所购买的兰蔻小白管，并且还被第三方机构鉴定为假货。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更搞笑的是在和官方客服进行沟通之后，选择了换货处理。没想到第2次寄来的东西依然是假货，气味和正品相去甚远。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今在黑猫投诉平台上，到处都是用户对于叮当魔盒的差评。“首次抽盲盒49.5元，结果到货之后发现其实就是个十几块钱的东西，还不能退货，简直就是赤裸裸的欺骗。”、“踩过这次坑，以后打死也不会再用了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12472833562db2f285538e3.02046802.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：黑猫投诉平台</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>万物皆可“盲”？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>盲盒确实有着非常悠久的历史。作为一个源于美国、兴于日本的游戏玩法，盲盒最开始在推出时是一个叫做扭蛋的产品，主要受众也都是一些少年儿童。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着这股扭蛋文化传入国内，形式逐渐演变成盲盒，价格在此前的基础之上也翻了好几倍。自此开始，盲盒热潮席卷而来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据相关数据显示，在2020年有超过20万消费者在盲盒上的花费超过2万元，高端玩家更是一年豪掷百万。闲鱼上，只是在2020年就汇集了超过44万盲盒玩家进行交易，为平台带来了数10亿元营收。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这之中，最大的受益者毫无疑问就是“<a href="https://www.niaogebiji.com/tag/%E7%9B%B2%E7%9B%92%E7%94%B5%E5%95%86" target="_blank">盲盒电商</a>第一股”泡泡玛特。不仅在业绩上迎来爆发式增长，同时还顺利登陆了香港交易所，受到资本和消费者的狂热追捧。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/67260611662db2f28bdf8a8.46245104.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：泡泡玛特官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>最初盲盒里面装的多是一些潮流玩具，而随着盲盒热潮的持续，范围也开始无限制的扩大，最终形成了一个“万物皆可盲盒”的局面。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如所谓的古董宝贝、美妆、餐饮券、飞机票，甚至把宠物猫狗等活体都装进了盲盒里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在网上还流传着这样一句话：“不管是你想到的还是想不到的，盲盒里面都有，现在就差直接把人装进去了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>只是这种肆意的扩张，使得盲盒经济乱象重生：营销套路层出不穷，产品质量更是良莠不齐。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如在叮当魔盒上，打开之后就有醒目的宣传语“三抽起购，必出史诗，仅需0.01元”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/184715060862db2f29eb1e69.81816116.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：叮当魔盒APP</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尴尬的是，点击进入购买页面发现只有第1抽才是0.01元，接下来的两抽都是原价99元。也就是说，用户需要支付198.01元才能够拿到这个所谓的“史诗级商品”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>盲盒电商到底有什么魔力？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>业内人士认为：“盲盒从最初的研发、包装和玩法上都充分满足了消费者的好奇心，并且以碰运气的形式让消费者在精神需求上得到满足。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>具体来看，其中一个比较明显的影响因素就是Z世代的崛起。随着95后、00后的成长，他们已经逐渐演变成了风潮的引领者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今在小红书、B站等平台上热门的拆盲盒视频播放量几乎都在数10万上下；豆瓣上搜索盲盒，大多数帖子都能够有上百条回复。盲盒所具备的社交属性快速拉满。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/41385634862db2f2ac2efd2.39802388.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：泡泡玛特官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，盲盒本身也是一个充分利用消费者猎奇心理的成瘾游戏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大多数盲盒电商平台购买的商品都是一些平平无奇的普通款，当然也有一定的小概率是限量款或者隐藏款。通过偶尔获得一次隐藏款的方式，来加深消费者的重复购买行为。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>除此之外，盲盒电商还抓住了消费者“以小博大”的侥幸心理，毕竟抽盲盒的概率都是随机的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但很多消费者在购买盲盒时存在着“赌徒谬误”的现象，也就是想当然的认为后来发生的事件和之前事件发生的概率息息相关。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如买盲盒时，之前一直都没有抽到心仪的商品，那下一次抽中的概率就会变大。也就是很多人口中说的：</span><span><strong>买的越多，概率越大</strong></span><span><strong>。</strong>商家正是利用了消费心理的这一特性，收割消费者的钱包。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>尤其是像叮当魔盒之类的盲盒电商平台，在玩法上还有很大的猫腻。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/64019522462db2f2c332e93.71782562.jpeg" referrerpolicy="no-referrer"></strong></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：叮当魔盒APP</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对消费者而言，自身无法获悉爆款的产生概率，从而直接导致了双方之间的信息不对称。在平台具备完全的决定权下，用户显然不会有任何的胜算。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>特别是近些年，盲盒经济逐渐由线下转到了线上，这更加使得消费者维权的难度进一步加大，也让部分商家和平台更加肆无忌惮的销售假冒伪劣产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也难怪，不少吃过亏的网友纷纷感叹：“幻想抽盲盒以小博大，不过是痴人说梦罢了。”</span></p>
