<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近来，杨子黄圣依夫妻的直播间可谓是风头无两。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>根据第三方数据，@黄圣依 抖音账号30天内直播了16场，累计GMV达到1.17亿，力压一众竞争对手，成为明星直播赛道的TOP1。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去两年时间里，有不少明星都通过直播展示了其良好的带货素养，但今年以来，我们能明显感受到明星直播的热度在降温，具体来看，不仅是新入局的明星数量有所下降，已入局的明星们取得的成绩也不似从前风光。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这种情境之下，黄圣依直播间“逆流而上”的原因是什么？究竟什么样的明星更适合带货？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>《天仙配》重现，</strong><strong>黄圣依杨子在直播间有多卖力？</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>黄圣依夫妇入局<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>的时间不算晚。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年，因综艺《乘风破浪的姐姐》的爆火，黄圣依再次活跃于公众眼前，并由此开启了自己的“主播事业”。<strong>刚入局时</strong><strong>的成绩不算理想，甚至有商家吐槽，一场直播只售出5单，与当时坑位费10万，保底售出30万的承诺，形成了鲜明对比。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这之后，虽然黄圣依仍会直播，但频次并不稳定，直播间的热度和知名度，也不算特别出彩。直到去年年底杨子在直播间和商家“当场砍价”，为了讲解甚至把老婆挤出屏幕、让黄圣依无话可说的短视频出圈后，才让更多吃瓜群众对直播间产生了深刻印象。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有用户表示：</span><span><em><span>“看了那么多直播间，别的主播或敬业或专业，都能看出来是在工作状态，只有杨霸总，直播那个陶醉的状态，那个活力四射的精气神，真是热爱无疑了。</span></em></span><span>”</span><span>还有人评价他：</span><span><em>“我一直觉得杨子直播不为了赚钱，就为了找个说话的地方。”</em></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>自今年5月起，两人在抖音@黄圣依 直播间开启了高频带货模式，因为出色的带货成绩，多次登上抖音带货日榜。</span></strong><span>在618当天的生日会直播上，黄圣依夫妇更是将直播间“搬”到户外，重现了经典剧目《天仙配》。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使已经过去了15年，董永七仙女这一经典CP，依然人气不减，直播间内的粉丝纷纷刷着“爷青回”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过程中，两人高度融入角色，杨子还戏称自己已经和牛一起生活了很长时间，试图向大家表演“骑牛”，结果差点被牛踹飞，令人捧腹。</span><span></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><img src="https://ngbjimg.xy599.com/17806563662ea8232b62b76.37789133.jpeg" referrerpolicy="no-referrer"><strong><span><span>@黄圣依 <a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%9B%B4%E6%92%AD" target="_blank">抖音直播</a>间截图</span></span></strong><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>卖力表演下，杨子黄圣依夫妇的直播间热闹空前，当天的带货 GMV 达到 1.28 亿，占据抖音带货榜第一名宝座长达12小时；而在整个618期间，两人也是抖音平台明星直播总GMV排行第二的主播（第一为贾乃亮）。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，根据卡思观察，过去7天内，黄圣依夫妇有3天都在直播，直播间总销售额达到9892W，日均销售额3297W。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>夫妇二人为何能在短短时间内，成为抖音平台的热门直播间，又是如何维持自己长久的热度？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在卡思看来，一方面，<strong>随着罗永浩的退场，抖音迫切地需要新直播间顶上，来填补电商直播的流量空白，塑造新的标杆，</strong>而杨子黄圣依夫妇恰巧在这个时间段，展现出了自己的卖货潜力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，<strong>黄圣依一家在直播间的真实表现，也吸引了很多用户的喜爱。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>观察其直播间的粉丝画像，以24-40岁的女性粉丝为主，正好是喜欢看“家长里短”故事的群体。而围观他们在短视频或直播间内的表现，<span>总是呈现了杨子逗趣，黄圣依“无语”，杨子妈妈拉偏架这样充满“反转”的人物形象，</span></span><span>一家人在抖音展现出的“接地气</span><span>”</span><span>样貌，在</span><span>引得观众大笑的同时，也拉近了他们和粉丝的距离，增加了粉丝信任。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/48264555762ea82334b6290.98427588.jpeg" referrerpolicy="no-referrer"></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>明星直播热“退潮”</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据卡思数据统计，2020年带货势头强劲的明星主播有戚薇、陈赫、刘涛、汪涵等人，到了2021年也涌现出不少“新势力”，比如朱梓骁、舒畅、娄艺潇、李金铭、张檬小五、郝邵文等，<strong>但今年以来，明星直播热“退潮”的趋势，变得尤其明显。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先是能出圈的直播间变得越来越少。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾几何时，我们在浏览抖音等短视频平台时，常常会刷到某些明星的直播间。5分钟1个福袋、10元钱10包纸巾的福利发得热火朝天，无论是出于对明星的好奇心理还是想“捡便宜”，总有用户会不自觉地点进直播间看个热闹，明星直播也借此在当时创造了很多带货神话。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/201402084762ea8233afd1b9.92412927.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><strong><span><span>@贾乃亮 直播间幸运观众截图</span></span></strong><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>但是最近，卡思及身边的朋友都明显感受到，能够“看到”“记住”的明星直播间变少了。</span></strong><span>一方面，全民直播热潮下，平台普遍不会再如从前一般，对明星直播间倾注大量流量；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，<strong>当明星们纷纷签约MCN、开始了“格式化”直播风格后，直播间之间的差异越来越小，能给用户留下的记忆点也越来越少。</strong>仔细回忆，近半年最出圈的明星直播间，并不是哪个电商带货直播间，而是另辟蹊径让所有女生动起来的刘畊宏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其次，明星直播间的收益也变得越来越少。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据第三方平台数据，拥有千万粉丝的@舒畅（国货优选直播间），最近7天直播3天，总GMV 712W；《爱情公寓》主演@李金铭最近7天有2天在开播，总GMV 1298W。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/75070607562ea823408ea02.56434284.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><strong><span><span>@舒畅@李金铭 抖音账号截图</span></span></strong><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其他的如@郝劭文@张檬小五夫妇等人的场均销售额都仅在百万量级，与过去动辄千万的带货成绩不可同日而语，而<span>在抖音拥有两千万粉丝的</span>@娄艺潇 ，过去三十天内没有进行过一场直播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样的情形不难理解，对消费者而言，以前直播带货还算件“稀奇事”，明星光环对于提升直播间知名度是有帮助的，而现在，抖音、快手上每天有几十万的直播间在直播，<strong>在直播间购物成为众多用户的常态行为，明星对于消费者决策的影响日益下降。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对品牌方而言，近来全球经济下行，营销预算收紧，广告主对一份投资能带来多少回报变得愈发敏感，再加上明星营销翻车事件频发，也让他们对于重金聘请明星这一行为有更多的考量。娱乐资本论近期发文，今年的大爆剧《梦华录》主演刘亦菲、陈晓，在剧集播出完毕后，其商务、代言资源几乎零新增，这种情况在以往是不可想象的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>什么样的直播间生命力更强？</strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>明星直播带货的历程，最早可以追溯到2017年，那会明星还是受淘宝平台的邀约进入直播间，算是淘宝直播引流的秘密手段。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而当直播带货的风潮席卷全国，直播间购物成为大众的重要消费手段，这条变现之路就再无捷径可言。在这个时代，无论是明星直播还是素人直播，都需要找到自己的个人风格，形成独特的记忆点。无论何时，个人特色都是稀有且最具竞争力的要素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从杨子黄圣依直播间的火爆，我们能看到什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先，是顶级的带货主播需要保持激情的状态，为观众提供情绪价值；顶级的直播间需要保证自己的内容质量，为观众提供实用价值。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卡思曾多次写到，抖音本质上是一个内容平台，官方也并不想成为一个完全的电商平台，所以抖音直播间不能仅仅起到一个导购的作用，<strong>即使是电商直播，直播间仍然需要为用户产出内容，创造价值。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这一点来看，杨子在直播间内和商家上演“砍价大战”，给用户以娱乐价值；@东方甄选 在直播间内</span><span>传道授<span>业解惑，给用户以学习价值；</span><span>@刘畊宏 在直播间内帮大家“</span>肥<span>油咔咔掉”，给用户以自律价值，这些都是直播间带给用户的意义。</span><span>并且，这些火爆全网的明星主播，自身也都有很好的口碑，他们在直播间内的一言一行都是能真正打动用户的。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/198017117462ea8234672572.20083625.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><strong><span><span>@黄圣依@东方甄选@刘畊宏 抖音截图</span></span></strong></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其次，注意直播与短视频之间的联系，维持账号的一致性。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音，大多数账号都在认真做短视频内容，可以说绝大多数用户都是从短视频端口认识到某个主播，某个直播间的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>被短视频吸引到直播间的用户，自然会喜欢视频中展现出来的风格、面貌，如果能在直播间内体验到相同的感受，购物体验想必会有所提升，反之，如果用户在各个直播间看到的主播总是千篇一律的，那么能够吸引用户留下来的可能性也就不大了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>最后，直播间要为消费者把关，注意货品选择，始终和用户站在一起。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近30天，黄圣依的直播间共上架396件商品。货品结构上，以美妆护肤为主，日用百货、服饰内衣、食品饮料、厨卫家电为辅，共同的是，这些品类均以女性为主力消费群体；货品价格上，各品类均价最低2.13元，最高58.25元。直播间的选品，整体都是走的“亲民”路线，符合大众粉丝群的消费能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总结来看，不论是明星还是达人，想要让直播间生命力更长久，最重要的是要做到对消费者的诚心、用心与责任心。只有真正把带货当成一份事业，努力耕耘，直播间才会结出更多想要的“果”。</span></p>
