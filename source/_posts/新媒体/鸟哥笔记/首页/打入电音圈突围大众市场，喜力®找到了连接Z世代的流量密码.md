<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互联网信息的碎片化，加速了大媒体的失效。以至于，传统HBG大渗透式的营销方式，开始被品牌重新审视。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不少企业意识到，品牌多与真正对品牌有认同感的群体互动，才有可能影响到更多的人。这也就出现了圈层营销，以及后续的破圈。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个过程中，各行各业的品牌都参与探索，为营销破圈贡献了不少新思考。其中，百年品牌喜力®啤酒Heineken，就给出了一个高分答案。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过强化品牌的电音属性，喜力®啤酒频密参与到与Z世代的互动中。这其中，<strong>喜力®啤酒独具特色的玩法就是打造了一个全新的电子音乐主题平台Heineken Soundscape喜力®星电音。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/122230510163304790323981.72967647.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为喜力®啤酒旗下一个全新的视觉化符号和品牌资产，</span><span><strong>这个平台汇集了顶尖电音、多维视觉以及先锋体验，以图邀请年轻人打破循规蹈矩的边界，突破枯燥无味的常态。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此基础上，喜力啤酒更是借助Heineken Soundscape喜力®星电音这一全新的品牌自有平台与优酷</span><span><strong>共创了国内首档电子潮流音乐Feat.竞演类真人秀《超感星电音》。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/145635805963304793437b06.78822677.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过这一综艺IP，喜力®以及其所引领的先锋多维的音乐精神，进一步扩大对Z世代影响。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不难看出，喜力®啤酒的破圈基于两个步骤：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><strong><span>第一步，品牌绑定目标圈层的兴趣属性，达成圈层共识。</span></strong></em><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><strong><span>第二步，借势大内容的影响力，实现大众传播。</span></strong></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/80860382360221a97c68298.05017712.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>捆绑电音属性</span></strong><span><strong><span>打造喜力®自有品牌资产</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌最浅层的表达是符号，它代表着与其他品牌之间的区别与辨识；品牌最深层的表达也是符号，它代表着是一个群体的生活方式、个性态度乃至信仰。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要成为一种生活方式符号，这要求品牌长期持续地专注于目标圈层的渗透。它是一个品牌圈层内社交资产沉淀的过程，也是达成圈层共识的过程。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要顺利完成这个路径，品牌大致需要三个步骤：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><strong><span>第一步，品牌要成为圈层里的活跃者，让圈层认识你。</span></strong></em></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><strong><span>第二步，品牌要为圈层带去新体验，让圈层接纳你。</span></strong></em></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><em><strong><span>第三步，品牌要成为圈层引领者，让圈层信任你。</span></strong></em></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>首先来第一步，如何让圈层认识你？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喜力®啤酒其实一直都在做各种圈层营销，包括大众比较熟知的体育领域——其长期赞助世界顶级体育赛事：欧冠、欧洲杯、F1方程式世界大奖赛、橄榄球世界杯等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，早从上世纪80年代开始，在音乐领域特别是电音属性，也成为了喜力®啤酒一个密不可分的主题符号。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而国内<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E9%9F%B3%E5%9C%88" target="_blank">电音圈</a>对喜力®啤酒的显著认知，主要来自于该品牌2021年在该赛道一系列动作。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喜力®啤酒在中国市场推出了全新电子音乐平台Heineken Soundscape喜力®星电音。旨在通过顶尖电音、多维视觉以及先锋体验为年轻世代提供更多灵感，鼓励他们通过电子音乐解锁全新维度和体验。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>该平台</strong></span><span><strong>还特邀了Armin van Buuren 担任主理人，</strong></span><span>后者曾五次获得《DJ MAG》百大DJ票选第一名。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在与Heineken Soundscape喜力®星电音的合作中，</span><span><strong>Armin亲自为其打造了一首品牌金曲《Soundscape》。经过TMEA的宣发造势，该歌曲收藏量突破了1000万。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19791720506330479b8605c0.79827909.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实这些年来，喜力®啤酒长期在电音圈内活跃，在Coachella Heineken House、Ultra、 Sensation、Rock in Rio等国际音乐节上，人们都可以看到该品牌的身影。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也因此，喜力®和百大DJ也产生了频密合作，David Guetta、Armin van Buuren、Martin Garrix、Tiësto等，都在全球喜力®电音舞台上留下过精彩的演出。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正是通过新平台、新单曲，以及拉动了电音圈国内外的合作，喜力®啤酒在电音圈中快速获得了群体认知。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>然后再看第二步，如何让圈层接纳你？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>圈层对一个品牌从产生认知到接纳，单纯输出同类型的内容还不够。你要为圈层带去更为先锋的态度，带去多形式多维度的体验，从而彰显你在圈层中不一样的价值。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Heineken Soundscape喜力®星电音就在通过突破常规的新奇想法，以互动化的体验形式出发，为电音爱好者提供令人难忘的体验。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如今年7月，</span><span><strong>喜力®星电音还官宣多维度音乐制作人Chace、赛博歌姬朱婧汐以及跨次元虚拟偶像DJ-Purple担任Heineken Soundscape喜力®星电音体验官。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11541173246330479c4ef760.11363360.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>期间，三位体验官与主理人Armin同台演绎高能现场，</span><span><strong>共同打造Heineken Soundscape喜力®星电音元宇宙先锋之旅的TVC。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频完整版一经播出观看量不断飙升，热度持续走高，相关话题#喜力星电音# 的阅读量也增长至2.7亿，收获了超高的话题度和讨论度。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这其中，最值得一提的就是虚拟偶像DJ-Purple。作为国内首个虚拟DJ，喜力®星电音选择与<span>DJ-Purple</span>合作，也是想为年轻世代带去虚拟入侵现实的趣味体验。而作为第一个圈内结合虚拟DJ的啤酒品牌，喜力®啤酒自然深得圈层认同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>整体来看，获得电音圈层的接纳，主要来自于喜力®星电音推动国内电音实现了很多突破性、先锋性的事情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了与虚拟偶像DJ-Purple合作之外，Heineken Soundscape喜力®星电音还与网易云音乐、QQ音乐合作，打造电音榜和电子音乐专区。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此之前，国内各大音乐平台的电音专区都曾饱受电音爱好者的吐槽。这些专区内容和DJ榜单，大多以土嗨型的车载DJ为主，根本代表不了真正的电音。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这次Heineken Soundscape喜力®星电音的参与，</span><span><strong>直接推动QQ音乐更新了电音版块的算法逻辑，使更加先锋的、更受年轻人喜欢的电子音乐歌曲登上榜单，</strong></span><span>让更多人去发现并喜欢电音，这也是喜力®星电音为电音行业带来的创新性改变。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而今，QQ音乐电音专区中，不止有电音榜单，还有精彩的月度访谈视频。据称，这些内容也都是Heineken Soundscape喜力®星电音促成，并联合推出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>最后看第三步，如何让圈层信任你？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>得到圈层信任，实际上是达成圈层共识，这需要品牌成为圈层文化符号和生活方式的引领者。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>圈层符号最为人熟知的，应该是极限运动之于奥地利红牛。而在国内电音圈中，谈到这样的符号，一定会提到喜力®啤酒或Heineken Soundscape喜力®星电音。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，从去年至今，</span><span><strong>喜力®<span>星电音</span>每个月都在推出电音专栏、先锋资讯、金曲盘点，以及长期跟国内外电音人合作、赞助线下大型活动。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/709720161633047a03eddf6.73768315.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而Heineken Soundscape喜力®星电音的推出，已然表明喜力®啤酒在国内电音领域的战略级部署。这已不再是简单追求短平快回报的单个营销IP，而是一个在圈层中塑造品牌信任感的长期规划。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>遥想未来3-5年，国内电音活动或是电音爱好者周边，定会有Heineken Soundscape喜力®星电音这个符号相伴。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喜力®啤酒也必然会受到了圈层的信任。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/189383050560221a9860dbc2.87771537.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>利用综艺IP实现破圈</span></strong></span><span><strong><span>从Z世代到大众化</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>入圈的目的，其实还是破圈。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过绑定电音属性，喜力®在小众圈层实现了品牌价值的渗透。而这种做法，更多是向市场传达品牌的人格化态度和价值观。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而回归到商业发展来说，品牌的增长需要借助这群品牌有认同感的人去影响更多人。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>究竟该怎么做？品牌策划人杨不坏曾提过一个观点——“大内容”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他谈到，传统广告的重要概念是“big idea”，现在我们要谈“大内容”，比如奥运、春晚、现象级综艺，或者电商大促。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些“大内容”能让大众注意力更聚焦，创造社交话题与品牌高光时刻。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喜力®便是利用现象级综艺这种大内容，在今年暑假，与优酷共创国内首档电子潮流音乐Feat.竞演类真人秀《超感星电音》去撬动大众注意力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>借势综艺IP，可谓品牌走向大众化的捷径。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾有媒体评论，在中国娱乐市场里，没有哪个渠道能够比综艺能更有效的连接大众和制造话题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为一种娱乐性的节目形式，综艺这些年来的发展已经打造出得到市场验证的成熟模式。这也使得综艺背后的不少小众文化，一跃成为大众潮流。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以《超感星电音》为例，整个节目主打Feat.概念，电音制作人们选择了自己喜欢的歌手之后，会在各自的房间里交流电音创作和制作的理念。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这些创作背后的故事，不仅吸引了电音领域爱好者的拥趸，更得到了普通大众的围观。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，节目首播即斩获全网热搜188+，荣登微博综艺影响力榜、优酷音乐真人秀榜TOP1，节目主话题阅读量破1.5亿，讨论量高达216.2万，45.8w人在线RAVE！</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过这个动作，参与共创的Heineken Soundscape喜力®星电音不仅为国内的电音制作人提供了更大的舞台，还将自己的品牌推向了大众视野。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>节目热播期间，Heineken Soundscape喜力® 星电音将IP与品牌内容进行强绑定，建立起了传播势能。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>Heineken Soundscape喜力®星电音与音乐制作人共创十首电音风格短片：</span></strong><span>Mousse的异域节拍、刘炀的朋克节奏、WhyBeatZ的低音回响......正是因为喜力®星电音尊重和了解每位制作人的特点与风格，才会更好的为制作人们量身打造氛围短片，从而让更多的年轻人去感受多维先锋的电子音乐风格。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样喜力®星电音还</span><strong><span>联动音乐制作人，打造了三部先锋多维的乐无界TVC。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Carta忻若言以喜力®星银啤酒为节奏鼓点嗨翻全场；KAKA李弈可与Terry钟天利通过碰瓶喜力®星银获得极佳的灵感；WhyBeatZ于思源以开启喜力®星银释放电音能量……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喜力®星电音通过与电音制作人合作，为大众展现了一个跨越地域、隔阂、边界的电音世界，带来了多维魅力的感受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在整个节目内容的共创中，Heineken Soundscape喜力®星电音更关注音乐人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是倾斜资源量身打造氛围短片，还是为花尽心思定制更具关注度的乐无界三部曲……喜力®都是希望给予电音制作人最大的尊重，将电音文化推到台前，以便于更多年轻人去了解电音。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，推动电音制作人发展的最佳路径，就是与他们一起创作出更好的内容。为此，喜力®也在《超感星电音》中借助于各种音乐设计，鼓励他们跨界破圈，做更多音乐尝试。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如</span><strong><span>第三次竞演舞台上，Heineken Soundscape喜力®星电音主理人Armin就向两位踢馆歌手和七大Feat.组合公布了一个音乐命题“乐无界，超带感”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这是Heineken Soundscape喜力®星电音的核心价值表达，代表着先锋多维的音乐精神，以及超越当下、超越时间与空间界限的情绪共感。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在该命题之下，各组Feat.搭档进行了大胆改编，一次次挑战自我，超越界限和标准。他们带来的舞台作品也超乎想象，甚至将电音与流行音乐、嘻哈乃至古风融合的多元曲风碰撞。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样的内容，显然高度彰显了Heineken Soundscape喜力®星电音的价值表达，也推动电音制作人和整个中国电音走得更远。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，喜力®在这场综艺营销中还有一大亮点，在节目影响力达到势能爆发点时，Heineken Soundscape喜力®星电音带领电音制作人走到社交媒体平台、走到线下的各个城市，一同与大众互动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在前期多期节目影响力的沉淀之后，喜力®星电音邀请7位节目制作人进行超感派对云直播，线上与网友云蹦迪。高人气制作人回归最纯粹的DJ身份，充满激情的表演，向用户传递电音的魅力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>并且还顺势发起抖音挑战赛#这个带电造型绝了，将制作人创作的金曲作为BGM，吸引了许多年轻世纪群体参与，使得先锋潮流的电音进一步向大众化传播。数据显示，该挑战赛话题已获得了5.2亿次播放。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在线下，喜力®则在杭州等城市举办喜力®星电音仓库派对，邀请多位顶尖制作人参与破维见面会，给予年轻人与电音更进一步接触的机会。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2044801456633047a150df14.65038544.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在金贵晟、LUMINN、火星电台、FSHO等12组优秀的电音制作人热闹炸场之下，喜力®星电音为年轻人带来了一场以突破常规的互动化体验。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>加上《超感星电音》IP展台空降派对现场，更是掀起了掀起“打卡”热潮，吸引用户晒图发圈引发线上二次传播。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，在综艺IP加持下，喜力®为国内的电音制作人提供更广阔的舞台，和更为大众的认知。而品牌自身，也以电音属性内容成功破圈，并成为了这个暑期最火的啤酒品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/135701419260221a9c71d269.81480460.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>写在最后</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>营销破圈，本质上讲是在为品牌找到一个超级用户圈层，让他们帮助品牌实现大众化裂变。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个过程中，目标圈层的运营和撬动大众裂变的杠杆尤为重要。而通过喜力®啤酒的案例，我们大致可以找到相应解决要点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4809270633047a19f9dc6.04953503.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一，目标圈层的运营，难在沟通语言的选择。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>音乐、运动、艺术，这种跨越代际和疆界的语言是优选。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如喜力®选择的电音，它既是一种沟通艺术，更是一种生活态度。唯有持续保持更潮流、更创意的生活态度，才能真正与年轻世代圈层共舞。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二，“大内容”共创是核心能力。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为撬动大众裂变的杠杆，更多品牌选择的“大内容”其实是电商大促。因为这种“大内容”的共创门槛比较低，只要符合平台的要求，用福利货品和价格优惠去换取就行。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但<strong>真正高效的“大内容”，其实考验品牌的共创能力。</strong>它可能需要传播内容引发共鸣、服务内容提升体验、产品内容获得口碑。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>喜力®啤酒作为一个百年品牌，一直都是在通过独特的味觉体验，以及更潮更酷的内容/服务去撬动年轻人。</span><strong><span>这不是一个简单的营销能力，而考验品牌的产品能力、市场能力甚至背后的供应链能力。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这可能也是喜力®啤酒无法被短时间复制的关键。</span></p>
