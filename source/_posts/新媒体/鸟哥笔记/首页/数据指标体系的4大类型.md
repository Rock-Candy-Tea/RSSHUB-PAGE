<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多同学问：“有没有普遍的、一般的指标体系梳理方法？”网上常见的指标体系分享，大多是互联网的AARRR一类，现实中情况却很复杂。普遍的方法当然有，就是基于业务逻辑，梳理指标体系。从本质上看，<a href="https://www.niaogebiji.com/tag/%E6%95%B0%E6%8D%AE%E6%8C%87%E6%A0%87" target="_blank">数据指标</a>体系有4大类型，针对四个不同的业务逻辑。今天来系统介绍一下。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" data-tool="mdnice编辑器"><span><span><span>评估型</span></span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这个产品质量咋样?”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这个活动效果好不好?”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这个客户群值不值得做？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种用数据指标，评价一个东西好/不好的指标体系，即为评估型指标体系。注意，做评价不一定用数据指标体系，比如产品质量咋样，可以简单粗暴看销售数量。但只看一个指标会出问题，比如销量好，但利润低；销量好，但口碑差。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>评估型指标体系，主要就是解决：单一标准评估有偏差的问题。因此在构建此类指标体系的时候，重点就是考虑：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>1、从多个指标反</span>映<span>质量高低</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、各个指标之间尽量不重叠</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、区分各个指标重要性</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、各个指标的可比较性</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>评估型指标体系，指标之间关系常常是并列关系（如下图）：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/212833086062bbdf00146c41.50776274.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>评估型指标体系的难点，主要是考虑问题的全面性和指标数据采集可行性。这里特别要注意：数据指标可采集性。很多同学一提到评估，就会随口说：NPS、用户满意度、用户意向之类的指标。问题是：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、这些指标打算怎么采集？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、用问卷的话，能覆盖多少真实用户？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、用其他指标，能否合理评估情况？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、意向这种预测指标，到底怎么预测？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果不回答清楚这些数据采集问题，即使指标听起来很好听，也没法落地，还会找挑战。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" data-tool="mdnice编辑器"><span><span><span>流程型</span></span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们的销售情况怎么样”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们的生产进度怎么样”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们的研发情况怎么样”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种用数据指标，表示一个流程完成进度、完成结果的，即为流程型指标体系。这里就包括大家最熟悉的交易流程（漏斗分析）模型。实际上，只要符合</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、有明确的终点</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、有好几步环节</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、每一步有滚动投入资源</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>都可以用类似漏斗分析法，构建数据指标体系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/57272953462bbdf00335928.42073395.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅仅是交易流程或者用户留存情况，生产、研发、采购都能这么考虑问题。只不过在这些过程里不存在每一步衰减，因此不存在漏斗的逻辑。在这些流程里，考虑的是交付时间、质量、成本（如下图）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/158576613662bbdf006646e6.48328375.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>流程型指标体系是四大类里最简单的。因为流程的环节、终点都很清晰。目标清晰的情况下，指标体系都容易梳理。流程型指标体系，最怕的也是数据采集，特别是过程数据采集。很多toB行业的分析做不下去，就是因为流程指标缺失太多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/155095195662bbdf00c50808.23016251.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" data-tool="mdnice编辑器"><span><span><span>包含型</span></span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>包含型指标体系，一般是把一个大指标向下拆解成若干子指标/分析维度之和。我们熟悉的杜邦分析法就是这个逻辑，把一个大指标向下拆分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/191669076462bbdf012748b6.89547103.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>包含型指标体系，一般用于诊断问题。</span><span>因为子指标+分类维度，能把问题具体到某个业务部门和动作，所以能更好</span>地<span>发现问题源头，找到解决对策。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/54883886662bbdf0151b317.22976823.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是注意：包含型指标体系的诊断能力，是建立在：“主指标本身能说明问题”这个基础上的。比如用杜邦分析法的时候，就默认了：“利润”是主要问题。如果利润不能说明全部问题，而是还要考虑顾客体验、市场份额等等的时候，就不能指望一套包含式指标体系解决所有问题，需要对每个问题，单独建指标体系，进行解答。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" data-tool="mdnice编辑器"><span><span><span>影响型</span></span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以叫影响型，是因为运营动作一般是叠加在正常流程之上的，产生额外的效果。正常有一个销售节奏，做个活动会额外刺激销售；正常有一个用户留存曲线，做个会员奖励，额外刺激用户多留存一段。影响型数据指标体系，就是要把：叠加、额外两个点讲清楚。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/21445699162bbdf018f43f9.43065674.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此时，要考虑的指标复杂一些</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>1、需要有一套数据指标，反</span>映<span>业务正常节奏</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、需要有一套数据指标，描述叠加动作本身</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、需要有标准，判断业务本身的表现好坏</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4、需要有标准，判断额外效果</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，影响型的指标体系会格外复杂。比如做一个商品促销活动的指标体系，它需要考虑</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、正常销售指标（商品销售数量、购买人数、金额、；利润）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、活动的运作情况（活动目标人数、达标人数、奖励领取数，活动本身投入）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、正常销售走势 VS 活动增量</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里活动增量可能有好几个算法。比如</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、从时间上看，可以区分有活动/无活动时间</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、从商品上看，可以区分有活动/无活动商品</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、从人群上看，可以区分有活动/无活动人群</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是每一种方法仔细推敲，都有不成立的地方。因为即使没有活动，商品也有自己的销售趋势（自然增长率问题），很难找到100%相似的商品和人群。而且商品促销，还可能有透支效应（商品的粉丝用户趁便宜提前囤货，导致后续减少）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以在构建影响型指标体系时，设定合理的判断标准，异常麻烦，经常会有争吵。当然原则上，符合业务需要的就是好标准，不见得事无巨细考虑每种效应。但是要提醒同学们，在这一部分工作的时候格外谨慎，随口说几个指标，会给后续工作带来麻烦。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;" data-tool="mdnice编辑器"><span><span>综合运用四类指标体系</span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>以上四类区分，能帮我们梳理清楚思路，应对工作中的问题。</span><span>因为在工作和面试中，提问的人很少会主动区分：</span><span>“到底是啥场景？</span><span>到底业务是啥逻辑？</span><span>”而是泛泛</span>地<span>说：</span><span>“我要怎么评价这个XX？</span><span>我要看哪些指标？</span><span>”</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此时，对于数据分析师来说，就得时刻保持清醒</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>举个简单的例子</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“分析一下我们的产品”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“分析一下我们产品的改版计划”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“分析一下我们产品的改版效果”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“分析一下我们产品改版回馈活动”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>是四个完全不同的场景，需要的配套指标体系完全不同</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，是啥产品，也得细看</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、交易型产品：主流程就是交易，促成交易就是最终目标</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、内容型产品：用户行为散，变现形式多，得多角度观察</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、工具型产品：功能是固定的，但用户使用广度、深度有差异</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些都是在构建指标体系的时候需要考虑的要素。也正因如此，我才推荐同学们，不要试图“熟读并背诵”某几个数据指标，而是掌握梳理数据指标体系的能力，后续就能以不变应万变了。</span></p>
