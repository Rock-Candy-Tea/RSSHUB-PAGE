<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在6月14日开始，零克Club社群里都在讨论<a href="https://www.niaogebiji.com/tag/%E5%B0%8F%E7%BA%A2%E4%B9%A6" target="_blank">小红书</a>笔记的<a href="https://www.niaogebiji.com/tag/%E5%B0%8F%E7%9C%BC%E7%9D%9B" target="_blank">小眼睛</a>有点异常了，怎么感觉小眼睛没平时多呢？或者感觉点赞互动率比平时高了很多，但按理说内容好，互动好，小眼睛应该更多才对，但小眼睛为什么又少了？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最开始，大家都在自我怀疑是不是内容问题，用户兴趣又变了？或者是不是小红书平台不行了？这样的讨论越来越多，大家突然意识到，是不是小红书平台的问题？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">答案是：Yes</p>
<section>
<section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" aria-describedby="tooltip85168">一、小眼睛为什么异常</h2>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">原因是小红书在近期对笔记“小眼睛”的数据统计方式做了新的调整，这个时间点大约在6月14-15日开始，所以如果你是6月14日之前，那么就别找借口了....那肯定是内容不行....</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">说回来，原来小眼睛的统计定义为PV，换言之，点击1次笔记，小眼睛就加1，一个人点击10次，就加10，所以你可以理解为，PV是按照点击次数来计算的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">并且参与统计的渠道和端口是多元的，包含了：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1、小红书站内</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2、小红书站外（已登录，例如分享到微信里，但你的微信登录了小红书）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3、小红书站外（未登录，例如PC浏览器访问，手机浏览器等等）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但现在新的小眼睛统计定义为UV，换言之，一个人一天内不管点击笔记几次，都只加1，所以你可理解为，UV就是按照“人”来计算的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而且新版的小眼睛参与统计的渠道和端口并非那么多元了，只有以下两种：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1、小红书站内</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2、小红书站外（已登录，例如分享到微信里，但你的微信登录了小红书）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而小红书站外未登录的状态，比如PC浏览器，手机浏览器等等，通通都不计入到统计里。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以综合看下来，统计端口变少了，次数也换成了人数，那么小眼睛变少就是个必然情况了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在此，我也向知情人士了解相关情况，得出目前的新版统计为“灰度测试”，只是事实上看起来，似乎我身边的博主全都被测试到了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">至于，未来的小眼睛会不会恢复到以前版本？会不会一直保持新版？这在目前都说不准，也许只有在未来才能知道了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但呼声高，改回来，也不是不可能，就像之前的“新版首页”，也是有过回退先例的。</p>
<section>
<section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" aria-describedby="tooltip85168">二、小眼睛异常有什么影响</h2>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">统计方式的变化，让博主和品牌都措手不及，但目前，我们了解清楚新版小眼睛的原理也只是解决问题的一半，更重要的是，假设新版小眼睛一直保持如此，那么对博主和品牌有哪些影响？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>品牌</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3月份上海开始疫情，6月疫情开始解除，3个月左右的时间改变的不止是生活习惯，也对中国的经济造成了不可估量的影响，这个影响不是3个月少收入的金额，而是失去的消费信心，很难随着疫情的解除而直接挽回。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小红书的小眼睛也是如此，说到底，小眼睛变更的只是统计方式，在品牌营销和带货效果的结论上，本质没什么影响，只要营销链路上的所有人完全认同即可。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如品牌认同代理商，代理商认同博主，博主认同平台，那么大家自然皆大欢喜；但问题是，品牌是链路上的最终一环，品牌付了费，就希望得到效果，所以品牌难以立刻就认同下跌了4-5倍的数据，品牌不认同，代理商又怎么认同？博主又怎么认同？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而且品牌本就是消费类型的公司，所呈现的形象都需要公司团结配合，最终才能形成结果，这种团队，掉头和转身，都需要相对漫长的周期，不如互联网产品，随便动个代码，就完成了对用户的华丽转身，品牌她需要时间来接受，也需要时间来消化这种打击。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因此，环环相扣，每个人都有不得不理由，换句话说，每个人都有充分的反对理由。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其次，PV，仍然有其存在的意义，因为这代表用户对这篇笔记的喜爱程度和笔记的内容价值，100个人点击，1000次和1000个人点击1000次，这也说明了不少问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>博主</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为博主而言，是整个营销链路的末尾，也是收钱做事情的最后一环，压力自不必多说，本来就承担了许多品牌传播的重任。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我记得曾有位博主跟我说，每天熬夜写稿子，白天起来还要焦虑明天写什么，最终就为了能有好的数据效果，不仅是对用户的，也是对品牌的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">可以说博主一切坚持的动力，就来源于这些小眼睛和互动，这些每一个数字的背后，都是一个粉丝的鼓励，支撑她半夜写稿，白天焦虑。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但这几天改版下来，我觉得他们悬着的心，掉了多次有余，我闭上眼睛，我实在想不出来，他们是怎么跟品牌开口讲，我数据跌了4-5倍的，是的，我真的想象不到。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而且，小眼睛，是小红书生态的一种情怀所在，不仅仅只是“统计工具”，更加是平台与博主的情感链接，在其他平台，大家都叫“阅读数”，只有小红书平台会亲切的称为“小眼睛”，这是一种源自小红书的社区文化，持续了很多年；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">人们都说，人长大的过程中，很多东西都离我们越来越远，有的还彻底消失了，谁知道小眼睛是不是下一个会消失的存在呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">也许这就是博主长大的烦恼吧。</p>
<section>
<section>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; padding-left: 12px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left;" aria-describedby="tooltip85168">三、如何找回小眼睛？</h2>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">离开了熟悉的一切，连内容表现是不是真的好，都不知道了，因为以往博主会计算点赞和小眼睛的比例来判断，但新的小眼睛已经不是那个小眼睛了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果在当前情况下，还能去做个预估，也许会很好多。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，建议用以下公式做个换算，找回原来的小眼睛，一般来说：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">「互动较少」类型内容（比如内容质量一般，无干货，人设不强的笔记内容）：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现小眼睛*1倍-2倍=原小眼睛</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">「点赞突出」类型内容（比如娱乐性内容或有趣味性或生活性的笔记内容）：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现小眼睛*2倍-4倍=原小眼睛</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">「收藏or评论突出」类型内容（比如干货类，科普类，知识类等具备收藏或实用价值或者具备讨论价值的笔记）：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现小眼睛*4倍-6倍=原小眼睛</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以可以自行用以上公式换算一下大概估算一下自己的以往小眼睛是否合理。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据我所知，有些更优秀的博主，跌幅甚至可能达到10倍，但以上仅仅算是行业平均水平，具体每个人小眼睛的比例实际情况不同，如果你以往记录了自己的小眼睛，对比现在得出的比例，那么肯定更适合你自己，如果没记录，可以参考以上的估算方式，但仅供参考。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但如果你笔记没什么互动，小眼睛只有100左右，那么我建议就别给自己找借口了，这真不是小眼睛改版的锅，就是单纯的内容不行。</p>
