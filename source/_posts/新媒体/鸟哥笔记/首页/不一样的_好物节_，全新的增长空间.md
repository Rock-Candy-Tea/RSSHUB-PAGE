<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费市场的变化流转，一刻不停。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费者行为方面，在埃森哲发布的《2022年中国消费者洞察》中提到，</span><span><strong>中国消费者的基础依然强劲，但正在变得更加理性。</strong></span><span>在埃森哲的探访中，有超过六成消费者下单前，会在多家购物网站进行对比。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>而在消费渠道上，<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%95%86" target="_blank">电商</a>的影响力越来越大。</span></strong></span><span>贝恩发布的《2022年中国购物者报告》中提到，2021年消费市场唯一保持增长的渠道就是电商，根据国家统计局数据，今年1—8月份，全国实物商品网上零售额同比增长5.8%，线上渠道，是消费者与品牌建立联系最直接、高效的渠道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正因如此，凯度消费者指数大中华区总经理虞坚给品牌提出了建议，他认为快消品企业应该评估自己在各平台的电商渠道策略，并建立专业运营能力，以便触及不同的消费群体，满足不同的消费需求。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费市场和消费行为悄然变化，电商平台自然也需要随机而变。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时值“金九银十”的消费黄金时期，中秋节和国庆节两个假期接踵而至，消费热情达到高点。多重因素叠加，市场更加呼唤一个强有力的助推器。正如抖音电商产品负责人嵇鹏飞在抖音921<a href="https://www.niaogebiji.com/tag/%E5%A5%BD%E7%89%A9%E8%8A%82" target="_blank">好物节</a>招商大会上表示，旺盛的消费欲望需要承接。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了满足消费者需求，帮助商家发展，抖音电商发起了抖音921好物节——以新思路、新玩法，找到与消费者沟通、互动、交易的新机会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/81142426261420d300e45a3.29058312.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>延展抖音电商的场域</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这是抖音电商首个921好物节，依托FACT+方法论，对玩法进行了全方位的升级。仔细拆解抖音电商的尝试，可以清晰看出，一大特点在于，将抖音商城摆在了很显眼的位置。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/236496802632c8b5048c7a9.89591694.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span><strong>今年5月，抖音电商总裁魏雯雯表示抖音兴趣电商升级成为全域兴趣电商，</strong></span><span>在这一升级过程中，抖音商城的重要性被突出，成为抖音电商为商家提供多元转化路径的主阵地之一。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“通过短视频和直播内容、商城、搜索等协同互通，为商家生意带来新增长。”这是平台对抖音商城的期许和规划，而这些内容，都体现在了抖音921好物节上。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>具体来看。此次抖音921好物节，平台围绕抖音商城这一入口，设置了一系列特色玩法。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“跨店每满减”是抖音商城为商家开设的减免专区。只要是商家所提报的符合要求的优势商品，都能在专区凑单页内获得曝光。曝光的渠道也是多种多样，除了抖音商城外，商品详情页、购物车等场景，都提供了凑单页入口提示。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>点击进入抖音商城界面，在醒目处可以看到“超值购”的入口，被打上“超值购”标签的产品，有机会在这个拥有亿级别流量的频道获得更多曝光。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外抖音商城开启了低价秒杀活动。平台会为商家提报的优质好货提供限时特卖、抽签购等专属玩法，为商家集中促销、推新品、打爆款提供更多空间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1293716821632c8b50c6c905.51258268.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>抖音商城921好物节相关界面</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音商城如今</span><span><strong>构筑起了一个中心化的流量阵地，是一块品牌商家所看中的流量资源。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正是在这块「中心化」的场域中，抖音商城与搜索功能实现了紧密结合，帮助品牌商家实现了深度种草与高效成交的顺畅衔接。集中的推流让更多产品被看见，精准地触达消费者，帮助产品实现快速转化，提振销量的同时也增强了用户粘性。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音商城内，消费者可以实现从浏览、挑选、对比到购买、查单的全流程，消费决策周期被缩短，消费者的消费热情有了更好的承接，复购行为也自然会多起来。在用户的搜索行为下，抖音商城会有多样的场景点位和承接形式来把用户的搜索需求引导到对商品信息的浏览或者短视频和直播间上，从而提升商品的成交效率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/654575711632c8b510c4fd6.87782969.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>综合来看，抖音商城的「中心场」优势，有助于提升平台“人找货”的效率。当然，“人找货”不是新概念，在传统的货架逻辑中同样适用。只不过，曾经的货架实体陈列，遵循的是“分层组货，差异分发”的逻辑，而抖音电商则使用了“猜你喜欢”的匹配逻辑，在消费者对品牌的固有认知基础上，不断加深心智，以实现促进消费的目标。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这套逻辑的有效性有数据为证：根据抖音电商发布的数据，仅今年4月份，带有电商意图的搜索已达到64亿次，抖音商城今年4月份的交易规模也同比增长了6倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213179696861420d317ab820.36069667.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>抖音商城：商家新的经营阵地</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音921好物节，是抖音商城作为商家经营、转化阵地的一次集中展现。其实从今年5月31日，抖音电商总裁魏雯雯在抖音电商生态大会上提出「全域兴趣电商」开始，抖音商城的重要性就被不断地呈现、放大。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音电商「全域兴趣电商」概念中，短视频、直播等内容是用户潜在兴趣的激发转化场，在这里用户可以实现深度种草和高效的成交。</span><span><strong>而商城、搜索等需要用户主动探索的场景，则是流量的承接转化场，“用户在这里找到固定路径，形成习惯，实现精准匹配和复购。”</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从「兴趣电商」到「全域兴趣电商」，抖音商城在其中的价值，也被定位为「中心场」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1297046450632c8b515e7287.11751656.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>通过将抖音商城的定位提升，抖音电商对于商家，也不光是一块巨大的流量来源，而是成为一块稳定的生意入口。抖音商城日益完善，功能、产品、活动已全面补齐，日趋成为商家新的经营阵地。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年618，抖音商城就交出了一份不错的成绩单。抖音电商发布“2022抖音618好物节”数据报告，6月1日至18日，抖音电商直播总时长达4045万小时，挂购物车的短视频播放了1151亿次。期间，抖音商城场景带动销量同比增长514%，搜索场景带动销量同比增长293%，参与活动的商家数量同比增长159%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在抖音818发现好物节期间，抖音商城也尝试了一系列的玩法。针对用户，抖音商城发放了一分购新人福利，用户还可以通过游戏互动任务领取膨胀红包；针对商家，抖音商城做了大力度的补贴，例如超值购、秒杀等活动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些玩法是不是很眼熟？的确，抖音电商将818好物节的玩法部分延续到了抖音921好物节上，通过多次实战尝试，这些玩法也日趋成熟，在未来有望成为抖音商城中稳定、长期的优惠措施，给用户和商家带来有持续性的帮助。抖音商城的价值，也在抖音921好物节上，再一次得到了验证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1597491917632c8b519b9df7.67869138.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年抖音电商生态大会公布的数据显示，同比去年4月，用户有商品意图的搜索行为与去年相比增长了217%；2022年4月与2021年6月相比，店铺的PV增长了279%。这表明，用户的消费习惯正在变得多元，营销消费决策的因素也在增加，平台要适应这种变化，正如魏雯雯所说：“用户的兴趣在哪里，场域就延伸到哪里。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在“货找人”的逻辑下，激发消费者的非计划性消费欲望，则是实现转化的必要条件，而在“人找货”的场景中，消费往往具有目的性，消费者的理性因素占据主导。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>将“人找货”和“货找人”有机结合，正是抖音电商提出「全域兴趣电商」，重视抖音商城的初衷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/20585534276144238ed0dc54.24202450.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>抖音商城带来的增长机会</span></strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正是用“搜索+匹配”的逻辑，抖音电商打通了“货找人”和“人找货”的双向链路，作为串联起品牌和消费者的中间环节，为商家和消费者带来了新的增长机会。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>从商家角度来看，</span></strong></span><span>短期内，一系列的促销、打折玩法，则帮助商家锁定了消费者，顺利完成了从种草到交易的全流程，帮助商家实现了高效转化。抖音商城建立起的全新营销、促销场域，起到了帮助商家承接流量的作用。抖音商城的价值就在于，让商家减少了从漏斗中漏出的流量，在频密的链接中，实现复购。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>更重要的是，这样一次活动的效果对商家来说并不是短期的。随着抖音商城日益成熟，商家也逐渐积累起丰富的用户和运营方法论，这些完全可以移植到日常运营中，帮助品牌在未来持续获益。对商家来说，拥有一个流量稳定的阵地，与消费者保持高频度的连接至关重要。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/245588337632c8b51c7fa82.64272335.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>从消费者角度来看，</span></strong></span><span>抖音商城丰富的玩法设置，将种草、对比、消费、订单查询等全部线上消费流程集于一身，提升了选购效率，缩短了决策流程。在当前这个消费旺季，满足了消费者逐渐增长的需求。更何况，如今的品牌方面对的是更加理性的消费者，在纷繁复杂的营销信息中，多触点反复触达，是极有必要的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样站在长远角度分析，在整个消费过程中，抖音商城的搜索功能，激发了消费者的主动性。主动搜索带来的转化效果，永远比被动输入更加高效。当消费者在整个过程中收获了更好的体验，其对平台和品牌的粘性，自然会随之增强。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从抖音921好物节中，抖音商城的表现也能看出，消费市场正在发生着转变，“人”越来越成为消费流程中，最核心，最需要被关切的一环。营销行业对传统的4P理论十分熟悉，即产品</span><span>（Product）</span><span>、价格</span><span>（Price）</span><span>、推广</span><span>（Promotion）</span><span>、渠道</span><span>（Place）</span><span>，这四点中并未将“人”摆在很重要的位置。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着行业演进，传统的4P理论也在进化。在新的4P理论中，意义</span><span>（Purpose）</span><span>、参与</span><span>（Presence）</span><span>、接近</span><span>（Proximity）</span><span>和合作</span><span>（Partnership）</span><span>四个点被提炼了出来，很明显，每个词其实都指向了消费者。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音921好物节也正好契合了这四点价值——激发用户主动性，提升消费参与感，通过优质的体验，消费者从中获得了意义。抖音商城中心化、全流程的玩法设置，显而易见地，拉近了品牌商家和消费者之间的距离，让消费变得简单。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音商城给消费市场带来了新的思路。现在距离2022年结束，只有三个多月的时间。通常，下半年是消费市场的黄金时段，各种节日、假期会对消费起到提振作用。尤其是九、十月，更是有“金九银十”之称。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如中国贸促会研究院副院长赵萍在接受采访时表示：“在这段时间里，商家促销活动密集，居民消费意愿较强，预计今年9月、10月会保持传统消费旺季的特点，整体消费市场会稳定恢复。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费市场正处在向上爬升的关键时期，这是所有商家都必须要抓住的机遇。</span></p>
