<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">截取笔记发布前几天的数据和笔记发布后的数据，相对比可以看到明显波动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>从搜索流量上：</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">很明显看到8日笔记发出后有一个很明显的峰值，然后随之慢慢减缓，但目前还属于流量的曝光期，该产品依旧在是在享受小红书爆文所带来的流量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">并且将该图与上面笔记点赞趋势相对比，基本曲线变化趋势较为一致，说明小红书的爆文流量对淘内的搜索影响较为直接。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>从交易金额角度看：</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我们发现笔记的发出会对交易金额产生一个较为明显的影响，但是不少人会因为对新兴产品的出现产生观望状态，并没有直接转化成交易，直到产品在二次曝光的时候，种草成功，所以我们也可以从下图的趋势图可以看出，小红书曝光带来的流量后期，交易金额还是呈现一个上涨的趋势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从互动量方面：点击人数总计7164，占到互动量的12%，<strong>预估8个互动能带来一个淘内搜索</strong>，直接搜索销售额这7天累计是13728元。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这篇笔记的也是防晒产品，较上条推广boots内容的笔记，在互动曝光上差异这么大的原因是什么呢？</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1、内容明确指向性：从标题和内容都是以军训防晒为主，刚好呼应季节需求和正值开学季，学生的需求，年轻的学生用户正是小红书用户的主力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2、产品的价格落差：20块和150左右的客单，低客单，低决策成本，具备快速决策转化的特性。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1、从上述案例很明显可以得到小红书的爆文对淘内搜索有较大的影响，<strong>不同的内容表达形式带来的投放效率差异也会比较大</strong>，内容垂直度越高，产品决策成本越低，带来的种草转化效率就会越高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2、小红书不仅仅提供品牌种草和背书，对其他渠道转化也是有至关重要的影响。小红书的种草对其他平台的带货转化具有直接影响，同时更加体现了爆品的打造的重要性。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3、内容端，以干货为主，以软性植入为主，以多元化的创意点视频为主。<strong>小红书用户更注重before and after两者明显的变化，以及好物种草。</strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然，要想让笔记转化率高的话，小红书爆文的打造十分重要。</p>
