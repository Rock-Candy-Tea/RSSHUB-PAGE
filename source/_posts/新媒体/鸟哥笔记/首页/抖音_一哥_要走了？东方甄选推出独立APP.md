<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>自从6月份，<a href="https://www.niaogebiji.com/tag/%E4%B8%9C%E6%96%B9%E7%94%84%E9%80%89" target="_blank">东方甄选</a>董宇辉等一众主播凭借知识带货强势出圈，东方甄选一夜持续爆火，销售额不断攀升。</span></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>短短3个月时间内，东方甄选成长为抖音的超级头部直播间，场均销售额逼近2000万。</span></span><span><span>蝉妈妈显示，过去的6月和7月，东方甄选表现出色，直播销售额</span></span><span>力</span><span>压罗永浩的交个朋友直播间，</span><span>成为抖音新晋带货“一哥”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这样良好的带货势头下，<span>眼尖的朋友发现，东方甄选在各大应用平台上线了同名</span>独立APP。<span><strong>根据APP数据分析平台蝉大师显示，目前东方甄选APP的累计总下载量已接近20万，</strong></span>消息一出引发外界热烈讨论。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>爆火之后热度持续，</strong></span><span><strong>东方甄选销售额一路领先</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此前曾有市场传言称，东方甄选直播间曾遭遇抖音限流。不过后续已经得到抖音电商<span>相关负责人的辟谣，表示</span>不存在对东方甄选限流的情况。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通过蝉妈妈可以看到，除了6月中旬东方甄选爆火，进入流量巅峰期。进入7月之后，直播间的整体流量比较平稳，场均观看也突破2000万人次，并未出现大幅度持续低迷的情况。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在极短的时间内，东方甄选很好地承接了涌来的流量，实现快速组品开播，目前东方甄选直播间稳定日播，保持每天18小时左右的开播时长。<span><strong>在销售额方面，东方甄选一路领跑众多抖音带货达人，6月带货额突破7亿，7月销售额突破6亿持续霸榜，</strong></span>与其他头部主播拉开不小的距离。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在带货类目方面，食品饮料类目依然是东方甄选的主推类目，上架占比48.87%，其次则是生鲜蔬果、日用百货，且多条商品链接来自抖店东方甄选企业店。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">蝉妈妈显示，近<span>30天东方甄选直播间销售额排行TOP10的产品中，9款产品商品来源都是东方甄选的抖店，可见在近期东方甄选自营的供应链已经搭建完善，可以支持大额订单。</span><span></span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><strong>▲蝉妈妈商品分析</strong></h4>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>积极建设自有平台，</strong></span><span><strong>东方甄选独立APP已上线</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>东方甄选在经过流量巅峰之后，依然保持着强有劲的销售势头。新东方做带货直播，可以算是顺利转型。</span><span>此前，俞敏洪就曾在个人公众号表示，</span>“在东方甄选热闹的背后，也是我们对于农业和生活用品产业链以及供应链的更长远布局。毕竟，基于外部的平台所建立起来的热闹的商业模式，是有很强的脆弱性的，要夯实长期发展的基础，我们还有很长的路要走。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于东方甄选来说，自建平台不仅意味着只是自建供应链，深度发展电商事业，也包括了自建销售平台，脱离直播平台的影响与限制。</span><span>蝉妈妈也下载了这款A</span><span>PP，可以看到目前页面暂无外部广告植入，页面相对比较简洁，页面底部设有“甄选”“分类”“购物车”和“我的订单”这四大入口。</span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><strong>▲图片来自网络</strong></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>商城内部的产品类目包括自营商品、生鲜果蔬、海鲜水产和图书文娱等，带货的产品跟<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%9B%B4%E6%92%AD" target="_blank">抖音直播</a>间销售额的类目高度相似，价格水平保持一致。</span><span><strong>此外，我们也可以看到东方甄选积极布局私域运营，通过APP顶部的“东方甄选粉丝团”将用户导入私域</strong></span>，并通过发放优惠劵等方式，让用户开始在东方甄选APP内进行下单，以及创办专属社群。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前，东方甄选的抖音账号还在正常直播中，对于外界认为东方甄选将要完全独立，不在抖音做直播，业内人士都认为现阶段大概率不太可能，毕竟自建电商平台所耗费的资金和精力对于目前的新东方来说，不是一种有性价比的变现方式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">且蝉妈妈观察到，东方甄选似乎走上了跟交个朋友类似的垂类矩阵账号的道路，<span><strong>除了东</strong></span><span><strong>方甄选，东方甄选之图书、东方甄选美丽生活、东方甄选自营产品多个矩阵账号都在稳定开播中，月销售额皆在千万级别以上</strong></span>，最高单个账号月销售额突破5000万。</p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>小结</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于东方甄选来说，自建APP可能不会重点布局运营，但是它始终是新东方获取自主权的开始，也是自己为自己做的护城河。现阶段东方甄选在抖音做好直播带货，拓展更多产品类目，打造垂类矩阵账号，可能显得更为紧迫一点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">也有媒体报道过，在东方甄选自建APP之前，其实已经开通了天猫旗舰店、京东旗舰店、东方甄选会员微信小程序等等。多平台联动自有平台，可能是未来东方甄选的业务核心。东方甄选在抖音开了个好头，未来还有很长的路要走，蝉妈妈也会持续关注，敬请期待。</p>
