<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我不允许你还不知道这几组配色！！”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“任何人错过这个搭配我都会伤心的ok！！”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“50w人围观的神仙配色方案?”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你以为你在看穿搭博主的笔记文案，其实这是小红书一个家居自媒体分享窗帘的配文，就连窗帘的配色，都可以起上“春日茉香”“仲夏狂想”“抹茶可可”这样的名称。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/141646739962b458814567e6.07675330.jpeg" title="粉丝量1万，客单价100万，家居品牌号如何从0起步？" alt="粉丝量1万，客单价100万，家居品牌号如何从0起步？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>凭借对小红书平台流量热词、笔记文案技巧的掌握，“美源家居”的小编可以说是将账号玩得十分巧妙。截至目前，上图中这条发表于5月18日的笔记收获了4.9万的点赞量，超3万人收藏了该笔记。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>像“美源家居”做账号的家居企业不在少数，在小红书上搜索“家居”，可以查到上百个家居账号，除少量个人自媒体外，大部分都是以公司为名运营的蓝V。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由北京水木优品文化传媒有限公司运营的账号“住范儿”也在全网获得超360万粉丝，“住范儿”的内容传播方式有些类似老牌电视栏目：由一名主持人全程出镜，讲解榻榻米、木纹砖、开放式厨房等装修方式及材质的优缺点，帮助观众选择最适合自己的装修方案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于大多数人来说，买软装、家居装修都属于低频高额支出，少则小几万，多则上百万，每一步选择都值得细细考量——这或许也是短视频平台“家居”话题播放量居于高位的原因，在抖音，“家居”话题下的视频已累计107.8亿次播放，在快手则有20.4亿次播放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新榜编辑部观察及访谈发现，目前家居企业做自媒体的原因有三：</span><span><strong>一，引流线下门店，提高客户上门率；二，挖掘互联网潜在客户，促进消费者决策；三，提高品牌知名度，获取同行企业合作资源。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>重要的不是账号本身，而是它能够为企业带来的商业价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在“微信公开课”的一篇文章中，“住范儿”联合创始人李丹阳表示，2022年，围绕“住范儿”的家居建材卖场的线上线下整装、电商和零售的业绩预计达到14亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" title="粉丝量1万，客单价100万，家居品牌号如何从0起步？" alt="粉丝量1万，客单价100万，家居品牌号如何从0起步？" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>做一个家居企业自媒体，</strong></span><span><strong>需要投入多少成本？</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据观察，目前热度较高、且适宜企业账号学习的家居内容包含以下几个方面：装修攻略、好物分享和家居小妙招。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如上文中提到的“住范儿”，其发布的大部分视频都在讲解装修材料、装修风格等攻略。账号在一年前发布的有关“榻榻米”的视频，指出国内小户型装修滥用榻榻米的问题，并给出了小户型替代榻榻米的解决方案，目前视频播放量已达561.3万，近20万人为这条视频点赞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/47958096362b45881c69184.14816058.png" title="粉丝量1万，客单价100万，家居品牌号如何从0起步？" alt="粉丝量1万，客单价100万，家居品牌号如何从0起步？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有一种装修攻略是观众十分乐意听、也乐意学的：细数装修的X大坑，XX产品的智商税等等，从专业的角度出发，教授粉丝避坑指南。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如抖音账号“老许不测评”曾经发布的一条题为“家居行业编出来的四大谣言”视频，超190万人为这条短短一分钟的视频点赞，近13万人转发。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6069735262b45882a303b2.75151444.jpeg" title="粉丝量1万，客单价100万，家居品牌号如何从0起步？" alt="粉丝量1万，客单价100万，家居品牌号如何从0起步？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好物分享是内容行业较为常见的一种模式，它适用于任一赛道的博主，只要推荐物品符合目标受众兴趣和审美即可。小红书曾推出“家居好物必入榜”，以流量作为激励，鼓励品牌分享产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通常，网友种草某一产品的链路是“发现博主分享——询问产品店铺——跳转电商平台购买”，若是企业做新媒体，则可以直接申请平台店铺，促使消费者一键下单，减少跳转链路中顾客的流失。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/124276089762b4588317f027.71484584.jpeg" title="粉丝量1万，客单价100万，家居品牌号如何从0起步？" alt="粉丝量1万，客单价100万，家居品牌号如何从0起步？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友“一木木的家”分享好物笔记后，</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>收获了4.9万次点赞</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，家居小妙招类的内容受众广、需求量大。“亮朵收纳用品旗舰店”于2年前分享的拖地秘诀至今还留存在抖音#家居#话题的“最热”板块中，其秘诀包括拖把蘸清水、小苏打及白醋，即可强力去油污，用清水加食用盐拖地，地面可以干得更快等，该视频获赞32.9万次，超4万人收藏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>家居品牌卡萨贝拉的自媒体账号由公司老板亲自出镜，以“贺姐姐”的IP形象引导受众参观室内家居，在账号发展前期，他们挑选了一些市场上流行的米兰一线家居品牌，在视频中进行分享。该账号负责人表示：“（视频号）视频一出来，我们就收获了两三千粉丝。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/94349548362b45883742543.12765231.jpeg" title="粉丝量1万，客单价100万，家居品牌号如何从0起步？" alt="粉丝量1万，客单价100万，家居品牌号如何从0起步？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>卡萨贝拉的视频号（左）和小红书（右）账号内容，</span><span>目前全网粉丝为一万左右</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据介绍，全国仅有10家左右<span>跟卡萨贝拉一样做原装进口家居的代理商和官方旗舰店<span>，“原装进口家居”的自媒体内容<span>还属蓝海。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卡萨贝拉新媒体负责人透露，目前公司经手的家居产品客单价在100万以上，主要的受众是地产开发商、设计师团队以及家境优渥的年轻人，一年当中通过线上渠道发现这一品牌，来到线下门店的客户至少有几百人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卡萨贝拉的新媒体团队，精干且迷你——每一平台配备的账号执行人员只有两名，分别是文案和运营，视频及配图由公司的摄影团队统一提供，但摄影师还负责各个线下活动的拍摄，并不隶属于自媒体部门。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他表示：“（运营自媒体）这个计划是我2020年年底才给公司提出的，我们首先在视频号上做了一年时间，收获了非常好的效果。做小红书（是因为）跟着新榜培训以后，了解到它是影响消费者决策的非常好的平台，也很适合展示各种形式的内容。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做自媒体，一方面是为了提升影响力，一方面也是为了触达更多其他地域的用户。卡萨贝拉的总公司在厦门，但根据客户资源划分出了希望触达的三大省市：福建、上海和深圳。做了视频号以后，几名中国区家居品牌总代理也主动联系上了他们。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，卡萨贝拉正在装修一家占地2000-3000平的综合品牌店，附带咖啡厅、美食厅、艺术展厅，预计今年10月份完工。团队希望将其打造成厦门市进口家居的地标，并届时在小红书展开内容营销，吸引线下客流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>至于账号成本，该负责人表示</strong><strong>一年的支出大约为40万元，涵盖设备、人力、培训、推广等费用。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“第二年我们老板看到起色之后，在小红书上增加了近一倍的推广预算，包括重大营销节点的推广，KOL、KOC的合作等等，所以第二年的开支在50万-60万左右。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" title="粉丝量1万，客单价100万，家居品牌号如何从0起步？" alt="粉丝量1万，客单价100万，家居品牌号如何从0起步？" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>品牌起号法则：人+推广</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卡萨贝拉新媒体负责人聊到，</span><strong><span>如果企业预算有限，无法兼顾各项支出，面面俱到，那么必须要保留的支出项目就是“人”和“推广”，即懂行的员工和前期的推广成本。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一家居企业自媒体“YESWOOD源氏木语”的负责人小余表示，</span><span></span><span>他们起初建立小红书账号时，其实也是摸着石头过河，对小红书平台不够了解，内容也比较单一，只能不断地投石问路，对平台的用户属性进行探索</span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如源氏木语曾在一条笔记中分享的云朵沙发，笔记中用简略的语言科普了科技布款、灯芯绒款、有机皮款沙发各自的优势，分别是防水防污、厚实柔软以及防霉耐猫抓。</span></p>
<section><img src="https://ngbjimg.xy599.com/113346153162b45883d04797.98793651.jpeg" title="粉丝量1万，客单价100万，家居品牌号如何从0起步？" alt="粉丝量1万，客单价100万，家居品牌号如何从0起步？" referrerpolicy="no-referrer"></section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该笔记发布于5月20日，目前已有770人进行收藏。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其后，他们探索出的较为有效的方式是明确账号定位，再准确输出内容：“我们是家居类的账号，从定位上看，我们可能不是特别活泼的形象，我们就是给用户输出一些专业的内容，比如一些实木的科普知识，给用户分享一些干货。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据了解，源氏木语品牌创立已有12年，倡导做环保、可持续的家居，覆盖产品大到实木床、实木沙发，小到收纳盒、小推车，价格在几十至上万元不等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于源氏木语而言，涨粉节点就在大促的前后，比如双11、618家装节，用户本身就会搜索相关的家具关键词，“他们有需求，就会关注”。在活动期间，源氏木语发布的购买攻略、产品推荐，还有一些优惠活动都有可能吸引用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比起通过账号本身赚钱，源氏木语小红书商城的商品主要是展示的作用，它就像一个线上展柜，不断为线下200余家门店导流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/136895307262b458843f8c91.93357273.jpeg" title="粉丝量1万，客单价100万，家居品牌号如何从0起步？" alt="粉丝量1万，客单价100万，家居品牌号如何从0起步？" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>源氏木语的小红书商店和线下门店</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>（右图为网友探店笔记）</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小余说：“我们会结合自己的产品和品牌价值观，从不同方向挖掘，为用户带来创意且有用的内容。另外也会结合热点话题去做内容，比如近期的父亲节，我们去研究怎么给父亲送礼。总的来说，内容矩阵是比较完善的，有干货、有种草、有抽奖，也有新品发布。其中结合风格、场景配色发布的单品或合集产品的种草笔记比较多，（尽量）给用户提供家居完美答案。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>源氏木语及其他家居品牌与小红书合作推出的活动“我的梦中情房”</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新榜编辑部调查研究发现，家居自媒体业内还存在着这样一个传言：一个线上自媒体账号的收入可以养活线下一间门店。对此，卡萨贝拉新媒体负责人表示，如果是做原装进口家居，这是“绝对完成不了”的任务，因为一次性消费几百万、上千万的奢侈品客人都需要提供线下一对一服务，而这类人群也鲜少关注品牌的短视频。除非是做倒货、高仿，线下只有一个制造工厂，这才有可能靠线上供给线下，“养活一家店”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于企业来讲，开设自媒体账号仅仅是一个开始。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>初期运营的账号就像是一棵大树，树根不断地扎深、生长，找寻新的水源和养分。</span></strong></span><span>在卡萨贝拉新媒体负责人看来，企业只做一个账号的发展前景小，新媒体运营的必经之路是做完企业号就做个人IP，做完IP以后就去发展销售设计或者门店，分支出各种类型，最后形成一个矩阵。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，企业在新媒体上不断努力的核心之一依旧是售卖货品，了解客户的需求是销售产品的第一步，也是最为重要的一步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们将客户分为标准客户、意向客户和潜在客户，标准客户是正好最近房子在装修，又较为青睐卡萨贝拉品牌的客户，这类客户可以很好地与销售和设计师进行对接，方案初步完成以后，就可以进行订货；意向客户是指正处于比价过程中，犹豫不决，不知道该选哪个家居品牌的客户，针对这类客户，卡萨贝拉会通知销售团队，及时告知他们商品打折、促销的信息；潜在客户是通过自媒体了解到卡萨贝拉，对其产生认可，并因此分享资源给卡萨贝拉的客户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了通过网络连接更多的消费者，卡萨贝拉正在不断地扩充直播团队，其中一个任务就是招聘主播，进行家居产品带货，这些产品的客单价比起平日售卖的产品相对较低，例如几千元的家居饰品，更适合直播带货的生态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卡萨贝拉的新媒体负责人表示，做原装进口家居这一内容的自媒体寥寥可数，就连行业内容同质化，都成了他眼中期待发生的事儿：“我们其实非常希望这种现象出现，这就寓意着我们影响的人越来越多。”</span></p>
