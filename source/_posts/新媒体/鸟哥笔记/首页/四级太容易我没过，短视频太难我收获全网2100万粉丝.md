<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作者：<span mp-original-font-size="14" mp-original-line-height="29.75"><span mp-original-font-size="14" mp-original-line-height="29.75">片片</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span>“强子能不能不要这么恋爱脑了！”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“四级 too easy，必过 are you know?”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“当代大学生应该量入为出适度消费，你们这样就是学校的不正之风！”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些台词都来自一个叫“<a href="https://www.niaogebiji.com/tag/%E4%B8%83%E9%A2%97%E7%8C%A9%E7%8C%A9" target="_blank">七颗猩猩</a>”的搞笑账号，目前在全网有2100万粉丝，其中快手1041万粉丝，抖音603万粉丝，小红书283万粉丝，月平均接8条商单。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1126150120629354c612eae9.24099685.jpeg" width="600" title="四级太容易我没过，短视频太难我收获全网2100万粉丝" alt="四级太容易我没过，短视频太难我收获全网2100万粉丝" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这一切距离“七颗猩猩”这个大学生团队决心要拍<a href="https://www.niaogebiji.com/tag/%E7%9F%AD%E8%A7%86%E9%A2%91" target="_blank">短视频</a>，刚过去了1年8个月。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据观察，和“七颗猩猩”类似处在大学生阶段的博主数量庞大。在新抖中检索个人简介内标明“大学在读”的账号，有999+条结果。他们创作的内容五花八门，包含美妆、旅游、美食等等。《2020抖音大学生数据报告》显示，抖音内大学生创作视频整体播放量超311万亿次。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而“七颗猩猩”的内容更聚焦于校园生活，主要展现和同学室友们的打闹日常。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“七颗猩猩”为何选择<a href="https://www.niaogebiji.com/tag/%E6%A0%A1%E5%9B%AD%E5%86%85%E5%AE%B9" target="_blank">校园内容</a>赛道？做校园内容的大学生博主会遇到什么样的机遇和挑战？近期，我们与“七颗猩猩”主创团队聊了聊，关于大学生内容账号的制作经历。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">凭借四级内容“一战成名”，一条视频播放量1.2亿</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“七颗猩猩”的视频中常出现两个角色，王志猩和强子。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">她们因为拍短视频成为了朋友，开始了合作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“七颗猩猩”账号发布的第一条内容，简单记录了大学生上课的状况。回过头看这条内容，主创团队觉得，“它甚至不能称为视频，仅仅是一些碎片”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">后来，团队持续更新这个账号，把流量不错的视频模式保留下来，把表现不好的视频模式删除，直到“四级”视频让他们一战成名。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这条后台播放量高达1.2亿的视频里，他们把观众代入了女大学生视角，坐在宿舍的上铺，看似随意录了一段宿舍日常。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">视频发布在四级成绩公布当天，当时王志欣发现自己没考过，便一时兴起写了这个脚本。但在宿舍上铺拍摄这种镜头语言，之前视频已经使用了多次，团队发现，“这个角度比较真实，流量不错，因此选择固定这种模式”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">评论区里，不乏同样担心四级成绩的大学生留言，“我可能知道我为什么会刷到这个视频了”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">后续“七颗猩猩”基本保持一日一更的状态，通过拍段子的方式来表现大学生活，最近一期收获百万点赞的视频作品是《封校传》。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p>强子：学生要告发王志猩没有做核酸。</p>
<p>王志猩：你没事吧？</p>
<p>强子：学生若有半句虚言，便遭五雷轰顶。</p>
<p>……</p>
<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">原出处是11年前的电视剧《甄嬛传》第63集，讲述后宫内“告发私通，滴血认亲”的连环好戏。2020年，B站UP主@陆一鸣儿进行了二创，和男室友们1：1完整复刻这个后宫心机剧场，获得386万的播放。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1273545036629354c7a8a552.90129069.gif" width="500" title="四级太容易我没过，短视频太难我收获全网2100万粉丝" alt="四级太容易我没过，短视频太难我收获全网2100万粉丝" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“七颗猩猩”的版本，在原本大学生整活的基础上，把“告发私通”改为了“没有做核酸”，把“加了明矾的水用来认亲，血水不融合”改为“使用黑白打印机来打印健康码，绿码变黑码”，表达大学生封校的真实感受，截至目前抖音点赞数176万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这种以大学生活为主题，加一些夸张表达的内容，正是“七颗猩猩”账号最为擅长的。例如：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">开学的时候，大一新生来势汹汹，老一届头疼抢饭的事，想用长勺插队打饭，用榴莲当盔甲抢饭，充当教官的角色后夺饭而逃……</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在校期间，大学生想违规使用电器来烧饭，这时学姐来查寝了。为求自保，她只好睁眼说瞎话，把食用油说成护发精油，把面粉说成散粉，甚至把老干妈说成镜面唇釉……</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">大学生活中琐碎的情绪和感受，被“七颗猩猩”用拍段子的形式表达了出来。星图显示，目前该账号粉丝年龄集中在18-23岁，占比近60%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“七颗猩猩”的评论区就像校园论坛，学生交流着学校里有没有偷外卖的人、校园网卡不卡顿、宿舍里有多少人脱单这些生活细节。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1360895083629354ca3d3e48.72757874.png" width="500" title="四级太容易我没过，短视频太难我收获全网2100万粉丝" alt="四级太容易我没过，短视频太难我收获全网2100万粉丝" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">从首条广告收入1000元，到成立规模超百人的MCN机构</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">谈到商业变现，主创团队还清楚记得“七颗猩猩”第一次接广告的日期。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“2020年12月5号接到的。当时粉丝五六万，接的1000块钱。第一次接广告，不知道怎么做，本子也码了很久，拍得很青涩很尬。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此前，他们经历了一段艰难时刻，“没有粉丝，流量也不稳定，不知道什么时候爆，什么时候凉，顾虑是很多的”。而第一条广告，意味着团队迈出了商业化的第一步，无疑是对“七颗猩猩"内容的重要肯定。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">四级视频爆火后，“七颗猩猩”的广告业务逐渐增多，合作了赫莲娜、花西子、香飘飘、得物、DR等品牌。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年8月，主创王志欣接受了湖北日报的采访，提及“七颗猩猩”的收入状况，话题#00后大二学妹拍段子月入70万#随之冲上热搜，带来4.6亿次阅读。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">70万这个金额，引发了很多舆论质疑，有人说“当学生不如当网红”“寒窗苦读还有什么意义”“人人都拍视频还有救吗”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2075363483629354cac8c925.23246737.png" width="500" title="四级太容易我没过，短视频太难我收获全网2100万粉丝" alt="四级太容易我没过，短视频太难我收获全网2100万粉丝" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">王志欣发声回应，“70万只是流水，有工作室要经营，团队要养，设备要更新”，“作为湖北大学新闻专业的大二学生，短视频创业对于我而言不仅仅是兴趣使然，也是我自己创业梦的起点，运用专业知识创作的视频内容也可以作为专业作业提交，并不会耽误学业”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">半年后，“七颗猩猩”团队再提及这一事件时表示：“大众关于钱比较敏感，所以这个话题火了。后续关于我们的话题，大家讨论什么都行，只是希望评论不要太多负面的。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现在，“七颗猩猩”团队依然坚持着拍摄。星图显示，“七颗猩猩”抖音报价，目前21-60s的视频是8万左右，60秒以上是10万左右。他们在小红书、快手、B站等其他平台也开放商务合作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">今年2月，武汉荒野文化有限传媒公司在“七颗猩猩”团队的支持下成立，工作人员及旗下达人超过100人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“七颗猩猩”团队表示，很多学生都有做号的想法，但不知道怎么做号，“希望通过公司的方式，为有潜力的大学生提供服务。公司会提供各方面的支持，包括优质内容、流量、资金、道具、场景、策划、运营、商业变现上的支持等等”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1693063045629354cb4a1659.46383893.jpeg" width="400" title="四级太容易我没过，短视频太难我收获全网2100万粉丝" alt="四级太容易我没过，短视频太难我收获全网2100万粉丝" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">“七颗猩猩”最新的视频中出现了公司名称</h4>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">百亿需求的大学生内容，坚持才能让账号活下来</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“七颗猩猩”的内容基本聚焦在大学生活上。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据观察，抖音和快手平台内，大学生活相关的话题，累计有上百亿的播放量。如#大学生#话题在抖音有384.8亿次播放，在快手有78.1亿次播放。#大学宿舍#话题在抖音有98.6亿次的播放，在快手有64.1亿次播放。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/576885916629354cbbfb664.55420801.png" width="500" title="四级太容易我没过，短视频太难我收获全网2100万粉丝" alt="四级太容易我没过，短视频太难我收获全网2100万粉丝" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在对大学生内容的巨大需求下，互联网催生出很多垂类博主——他们凭借在校大学生的身份，就地取材，去专门制作和校园相关的内容。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">新红数据显示，近90天内小红书就有1万条以上与大学教育相关的的内容，大多是学长学姐以过来人的身份，教大学生如何在大学逆风翻盘、完成四六级考试和毕业答辩等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1725555027629354cc500eb1.46904938.png" width="600" title="四级太容易我没过，短视频太难我收获全网2100万粉丝" alt="四级太容易我没过，短视频太难我收获全网2100万粉丝" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在抖音，浙江大学的陈娴凭借学霸加搞笑的人设，获得4314万的点赞。成都体育学院的甩子哥，凭借体院背景和快乐搞笑的风格，获得4208万的点赞。沈阳电影专业的熬一耶团队，《国庆假期的守舍人》作品获得253万点赞，还和“七颗猩猩”进行过联动拍摄。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些博主所属的账号，往往贴有学校的标签，或者定位在学校，拍摄场景也常在校园内，比如教室、宿舍、操场等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">大学生博主因为选课制度有更多自由拍摄的时间，和对校园真实贴近的体验。“七颗猩猩”团队表示，他们曾经招过成熟的编导，“有的融入不进来，因为他对专业性理解更高，对校园的理解却和我们不一样”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如抖音上，有一条湖北工程学院官方号发布的视频，两个大学生扮演怪兽和奥特们在操场上打闹，留言区的高赞评论提到，“小孩觉得很尴尬，大学生觉得刚刚好”。还有一条男女大学生站在夕阳下看台上的唯美视频，有网友评论，“男：知道找你什么事吧。青年大学习截图，再不交，我就不要了。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这些都显示出大学生对内容的独特需求，不熟悉校园生活的人很难揣测，大学生博主却能第一时间发现这些需求并创作表达出来。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/869548984629354ccab16a3.55498203.png" width="500" title="四级太容易我没过，短视频太难我收获全网2100万粉丝" alt="四级太容易我没过，短视频太难我收获全网2100万粉丝" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但流量和关注，也会带来许多不确定的风险。部分大学生缺少法律知识和财务知识，可能和签约公司产生纠纷。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“七颗猩猩”团队表示，在发展的过程中遇到过类似的问题。作为大学生博主，想要扩展就得面临风险，想要保守就得面临局限。就像王志欣说的，“希望年轻的创作者们在选择合作的时候小心谨慎”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外，大学生身份也并非永久，博主面临着学业与毕业的双重门槛，爆火之后难以保持创作频率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">前文提到的UP主“陆一鸣儿”，是最开始进行二创宿舍甄嬛传的。经历了主创成员实习、毕业之后，账号已经有3个月未更新。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">再如UP主“都卡斐红”，曾发布视频《在宿舍结婚了》，大大的喜字，配合陶喆《今年你要嫁给我》的音乐，收获120万观看。但这之后账号仅产出过2个视频，观看量也急速下滑。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2139634150629354cdba6267.39141510.png" width="500" title="四级太容易我没过，短视频太难我收获全网2100万粉丝" alt="四级太容易我没过，短视频太难我收获全网2100万粉丝" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有灵气的大学生创作者总能不断涌现，但后续的跟进才是考验的开始。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">《封校记》火爆之后的第二天，“七颗猩猩”团队把第二集连夜拍了出来，并且持续更新到了第五集。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从2020年10月21日公开显示的第一条视频算起，这个团队仅在抖音就累计发布了365个作品，超过100万点赞的作品共20个。平均下来，他们每1.5天就产出了一条内容，每个月就产出了一条爆款。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于年轻大学生博主，“七颗猩猩”团队有两条建议：做好内容，然后努力创作。“如果说你连坚持都做不到的话就很难。”</p>
