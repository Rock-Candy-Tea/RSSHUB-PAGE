<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;">Babycare成立于2014年，最初以创新腰凳设计的婴儿背带切入母婴市场，逐渐搭建起母婴用品、辅食、童装、早教等泛母婴全品类矩阵。</p>
<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;">成立短短6年，Babycare全渠道GMV就突破50亿元，复合增长率超过100%，仅2020年“双11”期间，Babycare全渠道销售额就超过9亿元。</p>
<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;">近两年，Babycare重点布局私域，<span style="font-weight: bold;">全渠道拥有4500万用户，1000万会员，天猫官方旗舰店粉丝数量超过1700万，为行业第一。</span></p>
<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;"><span style="font-weight: bold;">私域用户超过400万</span>，近一半的下单用户会进行复购，所有<span style="font-weight: bold;">复购用户交易额占总体私域成交量的近90%</span>。相比2020年，品牌在2021年的私域GMV增长超100%。</p>
<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;"><span style="font-weight: bold;">2）市场规模</span></p>
<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;">据数据显示，在2018年中国母婴市场已经达到了3万亿元的市场规模，到了2020年更是突破4万亿的，<span style="font-weight: bold;">预计在2024年将突破7万亿。</span></p>
<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;">目前，90后和95后成为当今生育主力军，年轻宝妈注重母婴产品品质，对于母婴产品的关注度也逐渐提高，因此未来中国母婴市场规模还会继续呈现上升趋势。</p>
<p style="margin-bottom: 18px; margin-top: 0px; caret-color: rgba(0, 0, 0, 0.84); color: rgba(0, 0, 0, 0.84); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif;"><span style="font-weight: bold;">3）用户画像</span></p>
