                        <p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">6·18年中大促方才落下帷幕，牙膏作为日用品出现在不少“作业”中。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">在资本市场，牙膏同样备受关注。日前，冷酸灵牙膏的母公司重庆登康口腔护理用品有限公司（下称“登康口腔”）预披露招股书，拟公开在深交所主板挂牌上市。这是继舒客牙膏母公司薇美姿之后，今年第二家拟上市的口腔护理厂商。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p7.itc.cn/images01/20220621/14f664ffc73c4491b93ac233fd4896c2.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">▲图源:登康口腔官微</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">招股书显示，2019年-2021年，登康口腔营业收入分别为9.44亿、10.30亿和11.43亿元；净利润分别为0.63亿、0.95亿和1.19亿元；毛利率维持在较高水平，远高于云南白药（000538.SZ）和两面针（600249.SH）。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">但作为国产老字号，登康口腔仍有诸多问题待解：营收规模难与巨头比肩，营销费用拖累净利率；过于倚重单一品牌，高单价产品未成气候；存量市场竞争激烈，研发费用占比较低；深耕抗敏赛道，市场空间有限。“冷热酸甜，想吃就吃”的登康口腔，能否在上市道路上如愿？</p>
<h2 style="font-size: 18px; margin: 25px 0px 20px; border: 0px; padding: 0px 0px 0px 12px; line-height: 30px; color: #212629; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; text-align: left;"><span style="border: 0px; margin: 0px; padding: 0px;">营收平稳增长，营销拖累净利率</span></h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">在国产牙膏发展的早期阶段，登康口腔曾经风光无比。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">登康口腔前身为重庆牙膏厂，其发展历史可追溯到1939年的大来化学制胰厂。1987年，冷酸灵牙膏正式上线，恰逢90年代国产牙膏的高光时刻；1996年，冷酸灵与奥美广告合作，“冷热酸甜，想吃就吃”的广告语诞生；2001年，通过股份制改造，登康口腔正式成立。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p6.itc.cn/images01/20220621/2e3f4feae4f14bf19839b6f8e559ae88.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">▲图源:登康口腔官微</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">然而在1992年前后，宝洁、高露洁、联合利华等外资品牌登陆中国，挤压国产品牌的市场份额。国产牙膏开始集体走下坡路，登康口腔在市场拓展方面也饱受挫折。截至上市前，登康口腔的控股股东为轻纺集团，直接持有79.77%的股份，实际控制人为重庆市国资委。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">「不二研究」发现，登康口腔的营收正处于平稳增长中。招股书显示，2019年-2021年，登康口腔营业收入分别为9.44亿、10.30亿和11.43亿元，其中2020年和2021年增速分别为9.08%、10.97%。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">但从营收规模来看，登康口腔并不占优势。财报显示，2019年-2021年，云南白药旗下从事口腔护理的公司营业收入分别为46.69亿、53.87亿和59.10亿元，是登康口腔的5倍左右；两面针营业收入分别为11.87亿、6.85亿和7.18亿元。登康口腔与巨头的差距仍大。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p3.itc.cn/images01/20220621/7e90ced9f0b147d1a5a7a56e03915740.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">净利润的增长则更加迅猛。招股书显示，2019年-2021年，登康口腔的净利润分别为0.63亿、0.95亿和1.19亿元，其中2020年和2021年增速分别为50.78%、24.8%。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p5.itc.cn/images01/20220621/e1cf2b6fd07d4ee1ab5251534cf29920.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">登康口腔的毛利率维持在较高水平。招股书显示，2019年-2021年，登康口腔的综合毛利率分别为42.26%、41.71%和42.1%，远高于云南白药、两面针及可比公司均值。不过，舒克牙膏母公司薇美姿以53.80%、58.10%和62.80%的毛利率成为可比公司中Top1，与登康口腔正逐渐拉开距离。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p7.itc.cn/q_70/images01/20220621/e80d53c5f2ae4220b7b54cab9201881d.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">在净利率方面，登康口腔的表现并不算亮眼。根据招股书，2019年-2021年，登康口净利率分别为6.69%、9.25%和10.41%，三年平均仅8.9%。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p9.itc.cn/q_70/images01/20220621/e8fb6b3a37f3439f91cf7053434c48f0.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">「不二研究」认为，营销开支过高成为拖累净利率的主要因素。招股书显示，2019年-2021 年，登康口腔的销售费用分别为2.76亿、2.66亿和2.79亿元，销售费用率分别达到29.24%、25.80%和24.45%。同期，行业销售费用率平均值分别为24.25%、20.40%和23.43%；中位数分别为14.32%、15.13%和12.24%，均低于登康口腔。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p8.itc.cn/images01/20220621/20a36fef1b3b45af94116dbfab860800.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">从销售费用的细分项目来看，2021年登康口腔的广告宣传费用几乎腰斩，而促销费用大幅攀升。登康口腔在招股书中解释，受疫情因素影响，吴磊代言广告未能及时拍摄完成，相应的整合传播计划推迟，导致2021年投入的广告宣传费用同比下降46.66%；公司转而加码促销费用，致使该数值同比增长121%至1.15亿元。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p9.itc.cn/images01/20220621/e8dca39899f347e3a33dd8482088b82d.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">▲图源:登康口腔官网-登康口腔代言人-吴磊</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">对于登康口腔而言，大规模的品牌促销或许会加深中低端印象。牺牲长期品牌塑造以换取短期业绩改善，是否值得也需要公司深度评估。但毋庸置疑的是，对于营销的高度投入，正成为摊薄登康口腔利润率的一大痼疾。</p>
<h2 style="font-size: 18px; margin: 25px 0px 20px; border: 0px; padding: 0px 0px 0px 12px; line-height: 30px; color: #212629; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; text-align: left;"><span style="border: 0px; margin: 0px; padding: 0px;">单一品类依赖，高端产品乏力</span></h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">冷酸灵是登康口腔的王牌产品。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">除此之外，登康口腔的品牌矩阵乏善可陈，登康、医研等品牌均未享有很高的市场知名度。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">目前，登康口腔的产品主要分为四类：其一是口腔清洁护理用品，包括牙膏、牙刷、漱口水等；其二是电动口腔护理用品，包括电动牙刷、冲牙器等产品；其三是口腔卫生用品，包括口腔抑菌膏、口腔抑菌护理液等；其四是口腔医疗器械，包括牙齿脱敏剂等。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">招股书中，登康口腔虽未披露冷酸灵贡献的营收数据，但其所属的成人基础腔护理产品，营收占比超九成。2019年-2021年，登康口腔的成人基础口腔护理产品营收分别为8.86亿、9.47亿和10.26亿元，占总收入比重分别为94.12%、92.17%和90.00%。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p3.itc.cn/images01/20220621/81f728aececa404cb7bacbac60ca1411.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">「不二研究」认为，倚重成人基础口腔护理单一品类、依赖核心品牌冷酸灵，已经成为登康口腔不可忽视的风险。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">招股书显示，登康口腔营收贡献第二的品类为儿童基础口腔护理产品。2019年-2021年，营收分别为0.53亿、0.74亿和0.98亿元，占比分别为5.64%、7.16%和8.63%；复合增长率为35.31%，是登康股份增速最快的牙膏细分产品。同时，该品类的毛利率也最高，2021年达到55.01%，远超同期综合毛利率。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">但若选择儿童口腔护理产品为第二增长曲线，所面对的竞争也十分激烈。根据尼尔森零售研究数据，登康口腔的儿童牙膏线下零售市场份额为7.15%，行业排名第五；儿童牙刷线下零售市场份额为5.66%，行业排名第三。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p0.itc.cn/images01/20220621/b317421a8d0c4ff5820145d11eafee13.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">此外，登康口腔的高单价产品与服务——电动牙刷和口腔医疗美容等尚未成规模，短期内很难成为公司的业绩重心。根据招股书，2019年-2021年，电动口腔护理产品营收分别为203.19万、382.32万和666.33万元，占比分别为0.22%、0.37%和0.58%；口腔医疗美容产品营收分别为28.71万、315.37万和891.19万元，占比分别为0.03%、0.31%和0.78%。尽管近年来营收有所提升，但占比仍然过小。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">近年来，漱口水、电动牙刷等口腔护理产品逐渐受到市场热捧，国内口腔市场的竞争正在升级。而这些产品正是登康口腔的薄弱项，营收和利润表现均未形成气候。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">从销售渠道看，线下几大外资品牌雄踞；线上新晋网红品牌虎视眈眈。登康口腔的渠道也相对单一——以经销渠道为主。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">据招股书，2019年-2021年，登康口腔经销模式收入分别为8.16亿、8.82亿和9.30亿元，占比分别为86.63%、85.88%及81.61%。电商模式虽在发展，但2021年电商的收入占比只有13.17%。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p6.itc.cn/images01/20220621/ca29ceae450947c8b8bf5227ede20fcd.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">品牌、产品和渠道是口腔护理行业快速发展的核心，也是竞争的主要因素。从这三个维度分析，登康口腔仍有诸多问题：冷酸灵单一品牌独木难支；依赖单一成人口腔护理产品；经销商渠道尾大不掉。如何在新领域抢占市场、摆脱大单品依赖症，或许才是资本市场更为看重的能力。</p>
<h2 style="font-size: 18px; margin: 25px 0px 20px; border: 0px; padding: 0px 0px 0px 12px; line-height: 30px; color: #212629; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; text-align: left;"><span style="border: 0px; margin: 0px; padding: 0px;">存量市场博弈，抗敏临天花板</span></h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">作为抗敏老字号厂商，登康口腔深耕抗敏牙膏市场。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">招股书显示，2019年-2021年，登康口腔牙膏总产量分别为33199吨、34289.22吨和35437.16吨；牙刷产量为7523.38万支、6485.34万支和9386.97万支，产品产销率均保持在90%以上。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p7.itc.cn/images01/20220621/8070b85058cd472c9aa36a224d2dae2e.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">虽然在细分领域中拥有过半的市场份额，但抗敏仍是较小众需求，在整个牙膏市场中占比并不大。据尼尔森零售研究数据，抗敏感类牙膏2021年的线下市场份额占比仅为10.65%，不及牙龈护理、美白细分市场的一半。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">目前，国内口腔护理市场已经进入存量竞争的局面。根据欧睿国际数据，2020年-2021年，我国口腔清洁护理市场规模同比增速大幅下降，分别为4.94%及4.56%，2021年的市场规模仅为521.73亿元。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p2.itc.cn/images01/20220621/d542d5dd68ef4c9085fc658976be4e6d.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">尼尔森零售研究数据显示，2021年度牙膏行业线下渠道Top5合计占有超过60%的市场份额。其中云南白药位列第一，占比23.86%；好来化工（前称“黑人”）排名第二，占比20.10%。登康口腔市场份额占比仅为6.83%，与前几位存在较大差距。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p1.itc.cn/images01/20220621/55cb6c5b0f3c4e9c98f48ce2b42a0206.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">行业成长放缓、市场竞争“高烧不退”，登康口腔的“翻身仗”并不好打。从田七拍卖、两面针艰难生存，到usmile、BOP等新品牌诞生，国内的口腔护理市场已经发生大变局。对于登康口腔来说，老字号升级开展过晚、同时代品牌已抢跑上市，说服市场需要新故事。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">「不二研究」认为，对于登康口腔来说，一方面需要满足消费者多元化的实用需求，另一方面也要逐步打开高端化市场。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p7.itc.cn/q_70/images01/20220621/3283d5a81569460d929c17707eb42d9d.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">▲图源:登康口腔官微</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">由于牙膏品类使用周期比较长、顾客容易流失，打造全面的产品线成为不二之选。且对于牙膏产品而言，单纯从功能方面寻求营销突破较为困难。登康口腔目前的主要任务，是通过用户经营产生稳定的用户群体，进一步推动高端化。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">尽管在冷酸灵八十周年庆典上，董事长邓嵘表示将持续发力“研发+推广”，但事实上登康口腔在研发上的投入并不算多。招股书显示，2019年-2021年，登康口腔的研发费用分别为3030.1万元、3169.58万元和3551.8万元，对应的研发费用率为3.21%、3.08%和3.11%，与销售费用率对比鲜明。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p1.itc.cn/images01/20220621/c0b6027784a744958732266639320fda.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">在口腔护理这一存量市场，前有老字号云南白药站稳高端，后有网红品牌争夺Z世代。体量不大、增速平平的登康口腔，单靠冷酸灵的情怀难以维持昔日辉煌。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p4.itc.cn/images01/20220621/9dfd6911c1824ca08e86f0418887557e.jpeg" title="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" alt="冷酸灵母公司登康口腔IPO：单品依赖、高端乏力" referrerpolicy="no-referrer"></p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">▲图源:登康口腔官微</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">主打抗敏市场的差异化，或以成为登康口腔最大的竞争力。然而垂直赛道空间有限，登康口腔最终也将面临扩展业务边界的难题。短期来看营销“烧钱”打法能够带动销量，但难以从根本上解决问题。作为传统厂商的登康口腔想要破局，还需在大浪淘沙的行业中作出更多努力。</p>
<h2 style="font-size: 18px; margin: 25px 0px 20px; border: 0px; padding: 0px 0px 0px 12px; line-height: 30px; color: #212629; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; text-align: left;"><span style="border: 0px; margin: 0px; padding: 0px;">国产口腔护理的下半场</span></h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">背靠国资，登康口腔曾经是国内口腔护理行业的强者。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">但大浪淘沙，多年的止步不前、垂直赛道有限的市场空间，使得登康口腔“成也抗敏，败也抗敏”。此外，营收规模难与巨头比肩，营销费用拖累净利率；过于倚重单一品牌，高单价产品未成气候；存量市场竞争激烈，研发费用占比较低……登康口腔仍然面临着诸多困局。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">随着漱口水、口喷等新产品的铺开，国产口腔护理已然来到下半场。在高端化及多元化的市场竞争之下，想实现拐弯超车并非易事。登康口腔的上市之路，能否在遍尝冷热辛酸之后迎来甘甜，还需市场给予答案。</p>
