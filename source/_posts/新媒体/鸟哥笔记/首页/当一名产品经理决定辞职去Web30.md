<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年全球VC圈的新宠儿——Web 3，成了<a href="https://www.niaogebiji.com/tag/%E4%BA%92%E8%81%94%E7%BD%91" target="_blank">互联网</a>人的新出路。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互联网行业经过20年的发展，产业已经高度中心化，在用户和个人面前，平台拥有绝对的话语权。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而随着业务增量见顶、政策红利殆尽，互联网公司不得不探寻流量之外的道路，内卷由此成为其饱受诟病的代名词。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>下一个出口在哪里？今年29岁的阿武选择离开互联网大厂，投身Web 3领域继续担任<a href="https://www.niaogebiji.com/tag/%E4%BA%A7%E5%93%81%E7%BB%8F%E7%90%86" target="_blank">产品经理</a>，这或许是其中一个答案。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>被寄予厚望的Web 3到底是什么？哪些人在疯狂入局？作为投资界新兴的热门赛道，Web 3能拯救失落的互联网人吗？</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; color: #212629; font-size: 18px; margin: 25px 0px 20px; font-weight: 600; line-height: 30px; text-align: left; padding-left: 12px;"><span><strong><span>01、</span></strong></span><span><strong><span>Web 3，</span></strong></span><span><strong><span>在讲什么新故事？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Web 3，顾名思义，指代第三代互联网。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单来说，Web 1是门户网站时代，用户是文字和图片信息的浏览者。Web 2是社交网络时代，用户能在各种平台上生产内容，同时跟其他用户实时互动，形成庞大的关系网络。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与前两者最大的区别是，Web 3提出</span><span><span><strong>“去中心化”</strong></span></span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Web 2的用户虽然能打造自己的影响力，但真正的话语权由平台和其背靠的大公司掌控，所有用户的消费行为最终会转化成冰冷的数据，为中心化的机构存储和处理服务。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>而Web 3基于区块链的架构，应用代码和数据分布在分布式网络中的参与节点上，既能保证信息的流通，又将话语权还给了用户。因此，Web 3被认为有希望真正地实现「</span><span><strong>用户的内容及其产生的价值归属其本</strong><strong>身</strong></span><span>」的美好愿景。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/66232783162b9747b118886.12955142.jpeg" title="当一名产品经理决定辞职去Web3.0" alt="当一名产品经理决定辞职去Web3.0" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源/网络</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>资本率先行动。有机构统计，刚过去的4月，全球Web 3领域有74个项目获得融资，总金额达到31.03亿美元。而在2021年，全球投资加密货币的资金高达300亿美元，超过前20年投资总和。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>即便是资本寒冬，投资人砸钱进Web 3时也丝毫不手软。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Twitter高级项目经理Esher Crawford充分肯定了Web 3在社交网络中的价值，“Web 3意味着所有创造出的价值都能被更多的人分享，而不仅仅是企业主、投资者和员工”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>A16Z（被誉为最成功的VC之一，<span>Web 3投资牵头者</span>）的负责人Ben Schecter认为，互联网工作的未来，在于加密网络和DAO（去中心化自治组织）。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>SIG中国创始合伙人、著名投资人龚挺称Web 3是一场“</span><span><strong>信息数字资产化革命</strong></span><span>”。在他看来，Web 3时代技术有希望真正地被用来克服人性的弱点，“资本市场的去中心化和DAO化一定会发生”。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/104313961762b9747b5c5084.17793065.jpeg" title="当一名产品经理决定辞职去Web3.0" alt="当一名产品经理决定辞职去Web3.0" referrerpolicy="no-referrer"><br></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源/Pexels</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>相较于资本的宏伟叙事，</span><span><strong>从互联网大厂投身Web 3的产品经理阿武，则提倡不要过分推崇Web 3的概念</strong></span><span>，作为普通从业者，他更加关注Web 3现阶段的技术依托——区块链技术。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年十月，在字节跳动担任产品经理的阿武，选择离开在线教育行业，投身火热的Web 3，目前在上海一家区块链公司负责链相关业务，业余时间会运营自己的DAO。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>“谁都说不清下一代互联网是什么。”</span><span>在阿武看来，</span><span><strong>Web 3基本等同于基于区块链技术的下一代互联网。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>区块链具有去中心化、透明可追溯以及金融属性与流动性高的特点，不仅可以支撑组织形式的开放性（保证将区块链作为基础设施的Web 3具有拥抱开放性），让参与各方无条件地信任分配方式，还能保证利益分配的可靠执行。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相较于Web 3从业者，阿武更愿意以区块链从业者自称</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但不可否认的事实是，越来越多的风险投资资金正不断地涌入Web 3，在资源的堆砌、牵引下，Web 3吸收上一代互联网的人才似乎成为稚嫩但不可逆的趋势。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>02、</strong></span><span><strong>互联网卷不动了，</strong></span><span><strong>去Web 3当产品经理</strong></span><span></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一批互联网高管出逃、奔向Web 3的戏码出现在硅谷。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在2020年，谷歌副总裁Surojit Chatterjee离家出走，新头衔是加密货币公司Coinbase的首席产品官。2021年4月，Coinbase上市，市值超500亿美元，Chatterjee个人持有超过6亿美元的股份。据媒体报道，Coinbase今年将扩招超过2000名员工，涵盖产品、工程和设计岗位。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年1月，YouTube的游戏负责人Ryan Wyatt递交辞呈，效力于一家Web 3以太坊公司Polygon Studios。同一时间，亚马逊云计算副总裁Sandy Carter也选择加入加密技术公司Unstoppable Domains。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>在众多Web 3公司中，研究区块链技术的Mysten Labs成为硅谷大厂收割机，</span><span><strong>其80%的团队成员来自Meta、谷歌、Netflix等互联网大厂</strong></span><span>，从技术工程师到产品经理，几乎都有纯正的"硅谷基因"。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在上个月，据Cointelegraph报道，谷歌云（Google Cloud）在寻找全球Web 3产品营销负责人，该职位不仅要求最终建立客户对相关产品的需求，还将负责提高谷歌云Web 3计划的市场认知度。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/33327375862b9747b9b4c98.14308951.jpeg" title="当一名产品经理决定辞职去Web3.0" alt="当一名产品经理决定辞职去Web3.0" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源/Pexels</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>互联网行业的人才流转，同样在国内上演。有数据统计，</span><span><strong>2021年我国Web 3的投资和职位发布增长均超过400%</strong></span><span>，全职、兼职、远程办公，形式多样，其中应聘者超过一半来自互联网行业。与此同时，还有不少互联网人选择辞职去海外的Web 3公司工作。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿武所在的区块链公司，两百人规模，大多数正式员工之前是互联网从业者。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在进入区块链行业之前，阿武早于2017年成立了自己的科技社区，并结识了很多币圈的朋友，其中不乏当时进场、赚了几百万的年轻人。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>去年7月，我国颁布双减政策，眼看着在线教育的职业发展道路被堵死，阿武和几个做教育的同事开始重新选取赛道，“我们找了个会议室投屏，一个行业一个行业地过，但分析发现</span><span><strong>互联网的大多数细分方向都没有上升空间</strong></span><span>”。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/47017214862b9747bea1b03.82419658.jpeg" title="当一名产品经理决定辞职去Web3.0" alt="当一名产品经理决定辞职去Web3.0" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源/Pexels</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>到了要最终决定赛道的阶段，阿武给币圈的朋友挨个打电话，听他们聊从业区块链的亲身经历，“他们基本都很看好Web 3赛道”。阿武早有创业的想法，考虑到Web 3是一条火热的新赛道，留给小公司创业的空间会更大，他最终选择了all in Web 3。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>阿武认为</span><span><strong>当前的区块链项目以社区为本</strong></span><span>，“正规的项目上线前一般会有AMA（ask me anything）环节”，产品经理可以借此了解初始用户和潜在用户的想法。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但社区强依赖属性的另一面是，如果社区运营得不好，绝大多数的产品都无法活下来。“现阶段的Web 3少有体量大的产品，社区是最基本的，也是最重要的渠道”。产品经理在Web 3，很多时候需要扮演运营者的角色。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>工作之余，阿武运营着一个自媒体账号，并以此衍生出社区，和他一起维护社区的有九个人，不过都是兼职，阿武不给他们发工资，若有项目白名单、周边或者其他利润会均分。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他还时常会在社区推荐优秀文章，并且做必要的摘录或评注，同时也在小宇宙上做播客，定期受邀参加线上活动进行Web 3行业知识的分享。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>03、</strong></span><span><strong>什么样的产品经理，</strong></span><span><strong>适合Web 3？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品经理，是Web 3行业的“刚需品”吗？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>Jason Shah是美国区块链服务商Alchemy的产品负责人，他认为</span><span><strong>「产品经理在Web 3的0-1阶段并不那么必要」</strong></span><span>，原因是这一阶段的重心为技术基础设施、加密经济学和在线社区，而产品体验与市场的契合点暂不清晰，“产品经理通常更适合1-10阶段的Web 3”。 </span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>对于产品经理在Web 3中的角色，Jason Shah同样给了一个简单的定义：</span><span><strong>Web 3的产品经理负责社区的成功，而不仅限于用户增长。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/194155569662b9747c372297.95301451.png" title="当一名产品经理决定辞职去Web3.0" alt="当一名产品经理决定辞职去Web3.0" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源/网络</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>现实情况是，阿武的同事大多选择跳去其他互联网大厂，没有“赶热”。不论是产品经理，还是市场、运营等别的岗位，他都建议“</span><span><strong>在国内的话，没想得特别清楚，就先别加入</strong></span><span>”。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Web 2进入下半场，尤其是在互联网大厂，内部分工的精度越来越细，产品经理的角色和功能随之被稀释。相当一部分的产品经理即便不了解产品和用户，也能依葫芦画瓢，做出过关的产品体验。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>而提倡去中心化的Web 3，产品经理的价值尚未被验证，</span><span><strong>一个全能型的产品经理或许更符合行业的期待。</strong></span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“Web 3的产品经理必须思考协议设计、Token经济学、用户隐私安全、资产安全、增长营销、社区参与等方面。”Jason Shah认为，具备上述能力的产品经理，才可能建设成功的社区，“以及要思考更多关于人、技术和Web 3产品的价值互联”。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>成功转型10个月的阿武最大的感受是，“这个行业里，所有人都在犯错。”在他看来，无论是在线教育，还是整体的互联网行业，产品经理都有一套成熟的方法论可供学习，而</span><span><strong>区块链行业缺乏成熟的经验积累</strong></span><span>，在一个每天都会诞生无数新闻的全新领域，</span><span><strong>产品经理需要靠直觉摸索</strong></span><span>。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿武现在每天6点可以离开公司，但他的心态更加紧绷，比原来在大厂工作还要忙，不停地切换工作、学习和行业社交状态。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，阿武的情绪也更容易产生巨大的波动。“可能在某个时间点我想到一个方案，我觉得有前景、有效果，但是过了没多久，我会发现这个方案可能不可行，再过一段时间，可能就完全没有希望了，反之也亦然”。相较于工作与生活的无法切割，阿武的感受更多地来自于阶段性的兴奋、自我否定与成长的周期性循环。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Web 3到底是机会，还是泡沫？判断历来需要经验作支撑，或许不停地学习、反思，才是最好的生存之道。</span></p>
