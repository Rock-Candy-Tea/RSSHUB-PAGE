<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谁都没想到，<a href="https://www.niaogebiji.com/tag/%E5%BC%A0%E5%B0%8F%E6%B3%89" target="_blank">张小泉</a>会因“拍蒜断刀”而陷入巨大的舆论危机，而张小泉的客服和总经理则在其中起到了关键的推波助澜作用。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，“张小泉菜刀一拍蒜就断事件”，持续引发关注。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>张小泉因“拍蒜断刀”陷舆论漩涡</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>广州的一位消费者王女士在网络上分享自己的经历称，做菜时拍个蒜，菜刀就断了。王女士发布的图片中，刀面靠近刀柄处完全断裂，刀背上刻着“张小泉，斩切刀，50Cr15Mov”字样。而王女士联系了品牌方之后，却被告知“菜刀不能拍蒜”。7月14日，“张小泉客服称菜刀不能拍蒜”相关话题迅速登上微博热搜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/121318561062d73a05ec6990.65809659.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/22687497862d73a0658b7e4.26992527.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对网民铺天盖地的质疑和压力，7月14日，张小泉方面回应称，高硬度刀具不建议横拍操作，无条件给消费者退款。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月15日，张小泉官方微博发布长文回应此事，称该消费者2020年9月购买了张小泉龙雀斩切两用刀，售价99元；2022年7月联系经销商，反映刀面断裂问题。公司官方客服已再次联系该事件消费者，为客服不恰当的沟通方式表示歉意，并赠送一把新刀具以表诚意，由消费者在品牌旗舰店自选一款（己发货）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/180563201462d73a06a4cbd6.05371135.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/45243216562d73a073e1f21.57140237.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>然而，从这篇声明中可以发现，张小泉和消费者的对话其实并不在一个频道上，品牌方根本没有洞察到消费者真正关心的实质是什么。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在致歉的同时，张小泉还不忘为品牌做一次宣传，声称本次的“事故”刀具于2018年上市销售，截至2022年6月底，销售数量超6万余柄，自上市至今，收到消费者正式客诉11起，客诉率约为0.0183%……张小泉每年服务上千万消费者，产品质量是有保障的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>显然，张小泉这场“拍蒜断刀”危机，表面上是刀的质量问题，实质上是“中国人该如何用刀”的路线之争。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费者质疑“拍蒜都能断的刀”的质量问题，其实是觉得“中国人用菜刀拍蒜”这个习惯理应得到菜刀生产企业的理解和尊重；而张小泉不仅没有回答消费者关心的质量问题，反而一直强调菜刀的分类和用法，试图对中国消费者来一场“菜刀用法再教育”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，对质量问题，声明中只是说其产品“通过了国际国内多项严苛体系认证”，却没有解释“拍蒜都能断的刀”到底是不是存在质量问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有，声明中一直强调“一般而言，张小泉的常规刀具是可以拍蒜的。但并不是所有的刀具都适合用来拍蒜，一些硬度较高或者有专业用途的刀具如果用来拍蒜的话，有断刀风险。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且，张小泉在接受多家媒体采访时也强调自己的一贯态度是“提倡专刀专用”——“张小泉的常规刀具是可以拍蒜的，但张小泉提倡专刀专用，如果实在想要拍蒜的话，可以用手掌轻压刀面达到拍蒜的效果。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这种认知冲突的结果，就是品牌方<a href="https://www.niaogebiji.com/tag/%E5%8D%B1%E6%9C%BA%E5%85%AC%E5%85%B3" target="_blank">危机公关</a>效果会南辕北辙，做的越多，说的越多，消费者越不满意。</span></strong><span>果不其然，当张小泉的高管在之前接受短视频采访时说过的话被网友翻出来时，很快就又让其陷入了新一轮质疑中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>总经理过往言论被翻出似火上浇油</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这则视频中，张小泉总经理夏乾良声称：“你学了几十年的切菜是错的，所有的米其林厨师都不是这样切的，为什么米其林厨师切的肉片更薄、黄瓜片更透明，是因为前面有个支点。张小泉把刀前面的头斜过来，不是设计感，那是消费者教育。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/189772194262d73a07e5c2c1.47455877.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>虽然夏乾良这番言论是之前的，并不是“拍蒜断刀”事件后才发表的。但网民可管不了这么多，这番言论结合张小泉刚刚发布的声明中的相关态度，彻底点燃了网民的怒火。</span></strong><span>到7月18日晚间，相关话题再度冲上热搜，全网迅速发起了一股抵制张小泉菜刀用法的浪潮，消费者纷纷表达了自己的不满：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“中国才是做菜的嫡祖。米其林是营销，他们哪里会做菜”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“中餐和西餐有本质区别，咱们菜刀是来做中餐的，几千年传承，他拿个做西餐的刀来”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“就是不承认自己产品的问题，也不愿意去改善。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>......</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/29798833562d73a08a81bc1.63297619.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对消费者的抵制浪潮，夏乾良不得不出来道歉。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月18日晚间，夏乾良不得不发布致歉声明，夏乾良表示，“网传本人评价‘中国刀工不及米其林厨师’话题，视频是很久之前的采访视频，且并非视频的全部，现传视频中并未根据当时的情境和语境进行描述，从而导致大家对此产生极大的误解。众所周知，张小泉创立时，米其林还并未出现，不过我亦为自身的不当言论对消费者理解造成的错误引导，表示诚挚的歉意。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/50965952462d73a08e9cf80.40296416.png" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，夏乾良在道歉声明中还表示，张小泉即将发起“断刀召集令”项目，凡在国内购买的菜刀，若在过去五年内有断刀事故发生，包括张小泉品牌及指定同行品牌，公司均可按照类似款型、类似价值进行新刀换货。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有趣的是，作为同行的刀具老字号“<a href="https://www.niaogebiji.com/tag/%E7%8E%8B%E9%BA%BB%E5%AD%90" target="_blank">王麻子</a>”，在张小泉口碑崩塌的同时也没忘给它一波神助攻，不断的在自己的直播间拍起了大蒜，哈哈，评论区有人调侃称：“这拍的不是蒜，是张小泉的脸。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/164632591862d73a0925dfa2.05740692.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>想教大家用刀的张小泉被消费者教育了</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>试图教育中国消费者怎么用刀的张小泉，反过来被消费者教育了一顿。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也提醒企业，别太迷信“消费者教育”。中国有句古话“人之患，在好为人师"，意思是人的毛病在于喜欢做别人的老师。这句话用在企业和消费者的认知冲突中也非常合适，虽然企业可能在产品专业知识领域更胜一筹，但产品最终使用体验好不好，其实还需要消费者自己决定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从张小泉的“拍蒜断刀”危机中，我们至少可以发现三点：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、危机公关中，快速反应固然重要，但前提是一定要先弄清楚问题的本源是什么。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、互联网时代危机无所不在，企业及其负责人任何时候的任何言论，都可能在任何平台被引爆、被二次解读，需要做好全面的信息管理和舆情防控。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3、在获取资讯越来越方便、人人媒体的当下，任何权威都容易被解构和质疑，品牌妄图单方面“教育消费者”一定要慎之又慎。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，关于张小泉的此次舆论危机，并未随着总经理的道歉而结束，这不在今天上午10点多，又有媒体爆料，有男子晒用张小泉剁刀剁鸡腿，结果刀把断裂刀身瞬间飞出……。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/189210328662d73a094ff815.97111220.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137602460562d73a0996b4f8.57318458.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>但有一点或许是可以确定的，张小泉已经入围了今年的品牌年度危机公关反面案例集了。</span></p>
