<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">“栓Q~”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>未见其人、先闻其声。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只见“阳朔teacher liu”穿着一身广西民族服饰，出现在了“云南傈僳族小伙-蔡总”的<a href="https://www.niaogebiji.com/tag/%E5%BF%AB%E6%89%8B%E7%9B%B4%E6%92%AD" target="_blank">快手直播</a>间镜头前，再次唱起自己那首“火遍全球”的划船歌：“row row row your boat……”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/85308337062e7c2d6bf4314.09200875.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不少观众惊讶于两位来自云南怒江和广西阳朔的互联网“代表”在直播间梦幻联动。</span><span>这是7月29日快手举办的2022光合创作者大会现场，</span><span>在云南丽江的玉龙雪山脚下。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>十分应景地，“云南傈僳族小伙-蔡总”在直播间当场秀了一段：“我是云南的！云南怒江的！”，还教了“teacher liu”几句傈僳族方言，也换来了更多的“栓Q”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人对他们的印象，也许还停留在曾经刷屏一时的“成名作”。走红之后，云南傈僳族小伙蔡金发在快手渐渐有了170万粉丝，也会定时直播；“teacher liu”会在快手分享阳朔风光，时不时开场直播唱英文歌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/127468578862e7c2e3c13792.47239106.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们是快手海量用户的缩影，也是快手“拥抱每一种生活”的其中一种。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在快手直播间，有大主播们的热闹展现，也有普通人低调努力的一生。无论是摇晃的红酒杯，还是农村的铁锅烧饭，都是生活。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手直播强调主播的内容价值，基于社交的私域流量将观众与主播的关系绑定更深。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>越来越多元类型的主播来到直播间，通过一块小小的屏幕和外面的世界建立起连接，用内容让千万观众驻足停留，<strong>数据显示，今年一季度快手的直播月活同比增长了28%。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了让更多主播有机会被看到，<strong>快手直播宣布再拿出5亿现金和30亿流量</strong>，来激励优质内容创作，助力主播的成长。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们很好奇，2022年的快手直播间里还在发生着什么样的故事？快手主播们又有过哪些经历？以及快手直播的未来将会是一番什么模样？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>在互联网平行世界，</strong></span><span><strong>找一个“有趣的灵魂”</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“大家好，我是雨神萧敬腾——</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>的粉丝，广东萧敬腾。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“广东雨神”一出现在直播间，热情的粉丝便在屏幕上刷满了“819”</span><span>（雨神的粉丝名称）</span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/100300852462e7c2e4814167.67451012.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“广东雨神”接受新华社的采访</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他在2017年凭借一首播放30亿的《广东爱情故事》从快手走红，又在今年7月9日回到快手重新出发，回归直播首秀观看总人次超过6000万次。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无独有偶，最近几个月，“阿尼亚赛牛”等人也相继入驻快手直播。吸引他们涌入的重要因素，是快手直播多元化和温暖的生态氛围。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>快手直播提出 “看见每一个有趣的人”品牌价值。</strong>这里的“有趣”不仅仅是狭义上的有意思，更是与一个主播的个人魅力、内容质量，以及与观众的互动共鸣息息相关。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在快手直播间里，能看到来自各行各业、各种风格的主播在努力“整活”，每个人能在这里找到自己的一方天地。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>71岁的高龄奶奶“二人转韩姨”，时隔50年时间再次登上舞台完成梦想；知识主播“水果手术师”在直播间给葡萄“做手术”；“田斌”独创了脱口秀式直播、带来无数欢乐……</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/145303326262e7c2f23ebbe0.99092047.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>给葡萄做手术的片段</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>来自东北的“西山阿宇”也是其中一位。</span></strong><span>他出身于农民家庭，曾在鸡肉厂打工多年。工作之余，最大的快乐就是唱歌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2017年，阿宇被工友推荐去快手唱歌，发出第一条唱歌视频后一下子冲上了热门。就这样，他靠歌声吸引了一批粉丝，直播也成为一件顺其自然的事情，每晚7点半准时开播、雷打不动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也曾一度因为声带息肉手术，阿宇暂时唱不了歌，他的妻子、父母、女儿便陆续在直播或者短视频中露面，一家三代同堂，放声歌唱。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/56280264462e7c300279ff8.88291243.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>阿宇、妻子与父母在直播间唱歌</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播时，阿宇一家不仅会唱歌，也会与粉丝各种聊天、唠嗑、讲段子，有时候上点才艺、表演个小品，氛围其乐融融。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>许多观众就爱看这种生活简单温馨的模样，每天场观都能达到百万人次，截至目前“西山阿宇”的快手粉丝数已经超过1500万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以如此受欢迎，与直播间里有趣的内容密不可分。在阿宇看来，“唱得好是一回事，语言风趣更重要”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/143505199962e7c30e6135c8.84853057.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>阿宇在直播间整活“父母爱情”的段子</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在，不少粉丝已经把阿宇的直播间当作“连续剧”在追，一天不看就不得劲，哪天如果不播，便会焦急地私信询问情况。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>阿宇说从没把自己当作网红，他更愿意称自己是个会唱歌的创作者。</span></strong><span>他和妻子两个人打理运营着这个账号，眼睛一睁就要想播什么内容，经常忙到夜里两点才睡。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我俩是农村人，也不想说什么火了，自己掌控账号随遇而安，累了就休息两天。父母年纪大了，孩子要以学业为重。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，虽然许多主播都像阿宇这样并非“有意栽花”，却仍挡不住部分直播间凭内容造梗、热度破圈的势头。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从去年的反诈直播连麦刷屏热搜，到今年云南傈僳族小伙蔡金发走红全网，快手直播这个“全民舞台”，正在逐渐成为“热点制造机”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>都说有趣的灵魂万里挑一，在直播间里，似乎俯拾皆是。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>直播间里，</strong></span><span><strong>唯一不变的就是不断变化</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年的快手光合创作者大会有个特殊环节， “夏花女团”的6位代表登台表演，劲歌热舞点燃了现场氛围。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/192942400462e7c31c0cd1f3.52650663.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>扎着双丸子头、身着白色上衣的“麦小兜”站在C位，声音甜美，一眼被粉丝认出。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她在2017年因一首原创歌曲《9420》成名，快速积累了一波流量和关注，走上了短视频和直播唱歌这条路。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>截至目前，麦小兜在快手发布了277条作品，累计直播超300次，粉丝达到了1134万</span></strong><span>，“如果不发歌，我可能在现实生活里平平无奇，但因为在短视频直播里唱歌，被大家喜爱和支持”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管人气一直居高不下，麦小兜也在不断更新和丰富着自己直播间的内容。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早期做直播的时候，她基本上只唱歌，没什么流程概念。后来对灯光等设备做了升级，也加入了更多粉丝互动的环节。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/102891529462e7c329b39c05.09036636.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月14日，“麦小兜”的直播间做了一次内容上的创新，从室内到室外，<strong>以一场户外音乐会的形式，把露营、音乐与乡村自然风光相结合</strong>，吸引了1162万人次的观看，在线峰值人数达到了21万人。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当天晚上，还有粉丝千里迢迢地来到现场，给她补过了一个生日。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15182164462e7c337016697.18977112.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>麦小兜户外直播现场</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她的团队也在这次转型中得到了升级，20多人细化到摄影、导演、灯光、现场统筹、音控等多个工种，短视频剪辑也随时待命，配合直播的切片和二创传播。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>麦小兜透露，户外路演的形式将会成为接下来直播的主要内容。每次直播时长两个小时，不仅会保留以往直播的互动环节，还会请特约嘉宾到现场，将每一场直播都做成音乐会。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>900万粉丝的主播“唐薇歌手”也是在自我迭代中突破瓶颈期。</span></strong><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>她在快手直播4年时间，此前在线人数保持在1-2万，再难有进一步上涨。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年3月，唐薇也将直播间搬到了户外，购置高清直播设备、组织团队排练，还会与其他红人进行合作，成功摸索出适合自己的直播风格，直播间最高在线人数也随之突破了11万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/156435962062e7c344d8b1d4.08122739.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>唐薇展示户外直播现场情况</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还有“神秘的贺某人”，从早期直播间人数不超过10人，到探索多元直播风格，实现一个月涨粉300万，深受广大年轻用户的喜爱；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚刚回归的“广东雨神”保持着常态化直播，升级了直播间的画质，还计划每年创作6-10首歌曲，思考如何把音乐元素的直播做得更有意思……</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这种高速迭代的内容生态环境之下，与去年4月相比，<strong>今年4月，快手上粉丝超过100万的主播同比增长了42%，粉丝超过1000万的主播同比增长39%。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>加码优质内容、助力传统产业，</strong></span><span><strong>快手直播还有多少可能？</strong></span><span></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主播的粉丝量级与直播人气成长的同时，回报也在增长。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>拿“西山阿宇”为例，他比以前打工挣得多，父母也不用像以前那样去打零工了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月28日刚加入快手的新人主播“韩雪儿”，每天直播7小时以上，全程唱跳。高质量的直播内容和高频率的开播，<strong>让她直播间人气突破5000人，开播首周的日流水过万</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/198428119262e7c351bf1024.19933459.jpeg" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手直播运营公会负责人顾北就指出，<strong>好内容会成为未来直播行业增长的核心驱动力</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么是好内容？顾北认为应该具备三个要素：一是符合用户偏好的高内容消费力；二是能给主播带来收入；三是传递正能量和有用性的内容信息。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手直播对好内容的扶持和看重，得到了市场的认可。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年第一季度，快手直播月活用户数同比增长28%，直播平均月付费用户同比增长7.8%。主播数与观众数也都有明显上涨，才艺主播数量上涨129%，观众数量涨了89%。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三方数据显示，2022年中国直播用户数将达到6.6亿，同比增长3.9%，相比2021年8.2%的增幅速度明显下滑，增量转向存量的趋势明显。在这样的大背景下，快手直播的数据成绩，无疑是一剂强心针。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/60419139762e7c35228ada0.43546817.jpeg" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>势头正好，平台也不忘持续发力。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>2022年，快手直播将投入2亿现金和10亿流量来激励优质内容创作</strong>，重点扶持户外民歌、戏曲、美声、流行乐、古典舞、现代舞等12个垂类赛道；</span><strong>另外拿出3亿现金和20亿流量，来支持主播们的下一步成长</strong><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>快手也在不断突破直播的边界，与招聘、相亲、三农等产业结合，提升了产业效率，让直播有趣也有用，也为主播们提供了更多样的变现形式。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截至目前，快手直播上线的“快招工”月活用户超1亿，入职转化率超8%；“快相亲”的日消费用户数增长近5倍，相亲主播数增长近3倍。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>近日因被网友质疑视频“背景太假”，发澄清视频又被网友“疯狂二创”的三农主播“疆域阿力木”</span></strong><span>也开始直播希望让更多人关注到新疆特产。</span><span></span></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>阿力木在快手直播间介绍新疆哈密瓜</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿力木认为，“努力的意义不在于摆脱贫困的家乡，而是帮家乡摆脱贫困，新疆是我的第二个故乡，我希望能凭我的能力，真正带动当地的农业发展。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对主播来说，这无疑是利好信号，快手直播或许会成为他们获得增长的突破口。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>平台成为发展的沃土，各种扶持变成养分。快手直播所提倡的有趣价值、多元包容的氛围，以及各种扶持政策，都为主播的个性发展创造了空间。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>像“西山阿宇”一样的普通人可以分享生活中所热爱的一切；如“麦小兜”一般的歌手舞者能大方展示自己所擅长的才艺。直播下半场，出路不止一条。</span><span></span></p>
