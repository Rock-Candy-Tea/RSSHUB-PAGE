<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先问大家一个问题，上一次用笔写字是什么时候呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>头条哥万没想到，就在大家刚刚送走雪糕刺客的时候，但在开学季又迎来了<a href="https://www.niaogebiji.com/tag/%E6%96%87%E5%85%B7%E5%88%BA%E5%AE%A2" target="_blank">文具刺客</a>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1566158481630caf6edd1080.60427671.jpeg" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一支洞洞铅笔售价近20元，一支水性圆珠笔售价50元……这多少让曾经2块钱一根笔换着替芯用一学期的我开了眼界了。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/38388928630caf6f9c34b9.02713606.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而文具刺客出现的几天前，国内文具龙头品牌晨光发布了今年二季度财报，作为核心业务的文教办公用品的制造与销售上半年营收37.23亿，同比减少了12.58%，其他文具企业的营收也出现了相当比例的负增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1044745124630caf706c3ba5.57947528.png" width="333" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>很显然，即使没有任正非输送寒气，文具行业就已经切实感觉到了寒冬。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自无纸化办公和学习兴起以来，文具行业的“寒冬”可以说就再也没过去，越来越少的成年人不再用传统的方式进行书写，文具企业渐渐退守学生市场，但着疫情常态化发展，各地学校重新开始线上学习，又进一步削减了这部分市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>所以说</span><span><strong>今年盼望开学的不仅仅是家长，还有各个文具品牌。</strong></span><span>而就在智能平板越来越便宜的趋势下，文具品牌却逆潮流而为之，把笔做得越来越贵，以至于这一次引发集体吐槽的“文具刺客”。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1866455249630caf70e9e106.54222325.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这也暴露出文具产品营销在主要市场出现了策略上的偏差。近几年我们能看到文具品牌的主要动作就是</span><span><strong>利用起新消费的思路，将日常的文具产品文创化，以提高客单价。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比方说将盲盒营销移植到文具产品上：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/667639976630caf719fcb12.52558943.jpeg" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以及与时下火热的<a href="https://www.niaogebiji.com/tag/IP%E8%81%94%E5%90%8D" target="_blank">IP联名</a>出新产品：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/393635609630caf720e48b6.01441252.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1625386958630caf729c1f00.83346505.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>不得不说这些创新都精准地打在了目标用户上，并在短期内显著提高了文具企业营收。但这样的产品思路也忽视了一个问题：</span><span><strong>主要瞄准学生市场的产品，实际的购买者和使用者并不是一个群体。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>具体来说就是作为使用者的学生懂得IP授权、联名产品的价值，</span><span><strong>但作为购买者的大部分家长却没有且不愿理解日常文具的“文创化”，无法接受这类文具的附加价值，</strong></span><span>这也是相当一部分抱怨文具刺客的舆论来源。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>退守学生市场的文具企业需要同时沟通学生与家长两个群体，但是现在看来，对真正掏腰包的家长，品牌的沟通是不到位的。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1593176200630caf7337e8d2.82260461.png" width="555" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>相关的主要舆论大都来自家长<br></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E6%99%A8%E5%85%89%E6%96%87%E5%85%B7" target="_blank">晨光文具</a>的孔庙祈福系列就是就是同时沟通两类人群的典范，它运用孔庙这一跨越几代人的经久不息的文化符号，成功在学生和家长中建立了认同，成为晨光文具产品中经久不衰的系列产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1743201183630caf739577a4.30775553.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次文具刺客话题的兴起，也是给国内文具企业敲了一次警钟，</span><span><strong>如何与自己产品的购买者进行沟通，传递品牌的价值，消除家长们“冤大头”的购买心理，</strong></span><span>将会是之后文具品牌的营销部门在寒冬笼罩下面临的一个重要命题。</span></p>
