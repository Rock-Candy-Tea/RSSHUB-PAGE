                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a href="https://www.niaogebiji.com/tag/%E7%A7%81%E5%9F%9F%E5%BC%95%E6%B5%81" target="_blank">私域引流</a>常常有两种方式，一种是被动引流，设置各种触点和钩子，引导用户主动加你；一种是主动加粉，已知用户线索，主动加好友，引流至私域。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">被动引流往往建立在强影响力的IP和高价值的内容基础上。例如通过公众号、抖音、小红书等公私域平台的运营，让你的内容被看到，吸引用户主动来加你。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而主动加粉对于刚起步做私域，获取第一波用户的品牌来说，相对更容易操作。也常用于电商中，针对已购买的用户，添加他们的微信，转化为私域用户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过由于不及时、平台屏蔽、话术不佳等等多种原因，主动加粉的通过效果往往不尽人意。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">今天蝉小星从话术内容的角度，分享如何通过加粉话术提高好友通过率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本文核心内容：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1.加粉话术通用公式</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2.加粉话术的5大维度</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3.加粉话术需要注意的关键点</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">1、加粉话术通用公式</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一条优秀的加粉话术需要给用户3个信息点：你是谁，用户为什么要加你，用户怎么加你。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1.你是谁</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">先看一个错误示范，如果是你，你会通过加好友申请么？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a href="https://www.chanke.com/yunyingquan/image/show/attachments-2022-06-6dvXJ6r362abf92de5cf9.png" data-fancybox="gallery"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://www.chanke.com/yunyingquan/image/show/attachments-2022-06-6dvXJ6r362abf92de5cf9.png" alt="如何加爆私域好友？这些话术了解一下！" title="如何加爆私域好友？这些话术了解一下！" referrerpolicy="no-referrer"></a></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">沟通是需要情感的，真诚、真实可以拉近距离，一上来不说自己是谁，容易让人抗拒。先告知自己是谁，开门见山，反而增加用户的信任。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2.用户为什么要加你</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">之前蝉小星在《做私域引流，你要先明白这四个点！》文章说过，用户为什么要进你的私域？当然是私域可以满足用户的某种需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而这需要在加粉话术中体现，告诉用户通过好友后的好处，从而吸引用户，增强用户的意愿。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3.用户怎么加你</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">基于前面两点，告诉了你是谁+加你有什么好处，这时候引导用户动作，通过好友验证，就是顺其自然的事情了。不过由于自动加粉绝大多数是加到企微/个微，用户也了然于心下一步就是通过好友，所以许多品牌在加粉的时候会把有限的加粉文案用在其他描述，而省略这一部分。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">蝉小星认为，再次提醒“通过好友验证”是一种引导加强，3点完整，才能最大程度的吸引用户。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">2、加粉话术的5大维度</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">可以发现，加粉话术最核心的“你能给用户什么好处”。根据加对应产品及客户需求，可以分为利益、兴趣、服务、服务+利益、互动等5种方式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1.利益型</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于引流加粉过程中，利益是大多数品牌吸引用户的重要一步，利益点和价值点是驱使用户进入私域最简单、最直接有效的方法。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如教育品牌分享真题资料：同学好，我是XX的老师，省考面试历年真题及解析、结构化面试视频等材料在线分享，可咨询。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a href="https://www.chanke.com/yunyingquan/image/show/attachments-2022-06-YvJbAdG562abf94442b54.png" data-fancybox="gallery"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://www.chanke.com/yunyingquan/image/show/attachments-2022-06-YvJbAdG562abf94442b54.png" alt="如何加爆私域好友？这些话术了解一下！" title="如何加爆私域好友？这些话术了解一下！" referrerpolicy="no-referrer"></a></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2.兴趣型</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">许多企业在做引流时往往会忽略“兴趣”这一特性，兴趣不仅可以引流，还有利于用户沉淀，因为在物质生活不断丰富的今天，用户基于内容的兴趣消费正引导着新的消费需求转移，“我买我喜欢”式的消费已成为主流。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例句：嗨，小姐姐，关注你好久了，我们现在招募旅行体验官，如果你也感兴趣，添加我即可报名，跟着我们一起去探索世界的奇妙吧～</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3.服务型</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">服务具有高粘性，特别是专业服务，展示高净值服务，通过线上服务进而引流至线下，实现用户的沉淀和长期复购，从而构建起用户对企业品牌的服务信任。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如买完珠宝后品牌方以顾问的身份来加粉：感谢您的信任与支持购买，我是您的专属助理，后续售后相关问题，我将专属一对一为您服务。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a href="https://www.chanke.com/yunyingquan/image/show/attachments-2022-06-iXgvWy4262abf94a9177f.png" data-fancybox="gallery"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://www.chanke.com/yunyingquan/image/show/attachments-2022-06-iXgvWy4262abf94a9177f.png" alt="如何加爆私域好友？这些话术了解一下！" title="如何加爆私域好友？这些话术了解一下！" referrerpolicy="no-referrer"></a></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>4.利益+服务型</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">利益和服务对用户均具有高吸引力，二者双重加持诱导下，用户是极易卸下对未知事物的戒备心，这种适用于大多数行业。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例句：哈喽，我是金牌宠物医生，通过免费送您一份狗粮，后续有什么问题，可以直接联系，免费给您问诊~</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>5.互动型</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通过有效互动的方式能够拉近用户距离，给人熟悉感觉，使人产生信任，互动裂变助力私域流量池的迅速扩充。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例句：嗨，您好！我是鸢鸢，主营闲聊、织家图透，副业朋友圈互赞、拼DD终生砍一刀等业务（拼DD我们家没有开店哈）本月抽奖的奖品就是下图咯，提供书签卡对应的订单号就能参与抽奖！</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">3、加粉话术需要注意的关键点</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1.文案不宜过长</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">简短文案一不会造成阅读疲劳，二聚焦重点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2.客户印象唤醒</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最优开场白唤醒，即最有效的加粉话术。一般情况下在抖音直播间下单的客户，多为冲动型消费客群，假设说“你好，我来自于XX品牌”，远不如说“我是xx的客服，您最近在xxx直播间下单过xx订单，不知道您是否有印象？”更能唤醒用户记忆。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3. 避免重复多次发送一样的话术</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">重复多次发送一样加粉话术并不能增加用户通过的积极性，效果正好相反。一般而言，通过微信的加粉不成功，如果有用户的其他信息，可以使用AI外呼或者短信，多种方式触达用户来提高通过率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a href="https://www.chanke.com/yunyingquan/image/show/attachments-2022-06-lGLOE6NX62abf9679f5f9.png" data-fancybox="gallery"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://www.chanke.com/yunyingquan/image/show/attachments-2022-06-lGLOE6NX62abf9679f5f9.png" alt="如何加爆私域好友？这些话术了解一下！" title="如何加爆私域好友？这些话术了解一下！" referrerpolicy="no-referrer"></a></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>4.充分了解用户需求点</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">越来越多用户不会因为简单利益刺激就轻易添加微信，如何让用户愿意添加，核心在于充分了解用户的需求点。经营私域不能只做单向思考，一味的强调自身的优势，没有真正的观察用户行为，抓准用户需求痛点，实际将无法俘获用户的信任。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">实际上每一次与用户的互动都是一次挖金矿的机会，在这过程中了解用户、贴近用户，把用户洞察做到极致，思考哪些利益点对目标客户最有价值，从这个角度再优化话术、优化给客户提供的服务价值，掌握服务用户的关键按钮。唯有懂得用户需求，才能使彼此的业务都能持续扩充，循此途径才能真正取得自己的最大利益。</p>                        <div style="height: 1px"></div>
