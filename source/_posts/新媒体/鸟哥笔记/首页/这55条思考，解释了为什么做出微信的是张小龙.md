<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">本文梳理了<a href="https://www.niaogebiji.com/tag/%E5%BC%A0%E5%B0%8F%E9%BE%99" target="_blank">张小龙</a>在<a href="https://www.niaogebiji.com/tag/%E5%BE%AE%E4%BF%A1" target="_blank">微信</a>发展过程中的55条思考，包括对微信公众号、小程序、企业微信与视频号等多个关键功能的认识，这一定程度上解释了为什么做出微信的是张小龙，而非他人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么是张小龙做出了微信？360创始人周鸿祎曾提出的这个疑问，至今也是很多人在思考的一个问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>答案千千万，不过著名传记作家沃尔特·艾萨克森在《创新者》里倒是提供了一个有些底层意义的解释，<strong>“数字时代的真正创新来自能够将人文和科学联系在一起的人”。</strong>张小龙就是这样的人。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>张小龙有人文思维，懂人性。他甚至认为，“产品经理永远都应该是文艺青年，而非理性青年”。他表现出了与人们传统眼光中IT技术人员所格格不入的文艺格调。他喜欢加拿大诗人兼歌手莱昂纳德科恩，崇拜迈克尔·杰克逊，表现出迥异于传统意义上的IT技术人员的文艺格调。他喜欢《蓝莲花》，就买下许巍的版权，挂在QQ邮箱入口。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他的这种人文思维投射到产品中，就成为了他始终强调的“用户价值为核心”、“用户体验第一”等价值观，而非商业先行。这在他做过的三个产品中贯穿始终，从未变过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1994年，张小龙从华中科技大学电信系毕业，放弃了体制内的铁饭碗，跑到广州从事软件开发。当时是PC时代，电子邮件开始盛行，但国内没有一款好用的电子邮件软件。张小龙决定做一款简单好用的邮箱软件。他利用业余时间，花了几个月写了一万多条代码，开发出了免费共享邮件客户端软件Foxmail，1997年1月正式上线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Foxmail被誉为能与微软Outlook对抗的邮件软件，好用且免费，4年时间其用户量就达到了200万。自此，在产品圈内，张小龙一战封神。多玩网创始人李学凌曾这样形容张小龙的地位，“只要你站在海淀黄庄路口，大喊一声‘我是Foxmail张小龙’，一定会有一大群人围上来，让你签名。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是Foxmail在产品上的成功并没有为张小龙带来商业上的成功。Foxmail是免费的，张小龙又不愿意通过推广告等方式来获利，因为他认为这是有损于用户体验的。<strong>用户体验是他的第一产品价值观。</strong>为了更好的用户体验，张小龙不惜辞职、全心全意维护软件，即使是没有收入。张小龙被看作优秀而落魄的程序员，“不合时宜的理想主义者”。当时去找张小龙谈合作的周鸿祎无法理解，为什么他把用户体验放到第一而忽视商业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个产品，即使再优秀卓越、极简完美，如果没有合理的商业收入来支撑，只靠一个人的理想主义，是走不远的。这是张小龙在Foxmail学到的教训。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2000年4月18日，张小龙以1200万元的价格把Foxmail卖给了博大公司。2005年3月16日，腾讯收购Foxmail软件，张小龙随之加盟腾讯，担任广州研发部总经理。张小龙遇到了同样是产品经理出身的马化腾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此时腾讯在通讯和社交领域面临MSN的强势竞争，马化腾希望有办法可以使QQ邮箱能打垮Hotmail以及国内的网易邮箱等竞争对手，做出过Foxmail的张小龙开始带队QQ邮箱。2005年底，QQ邮箱的用户不到100万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>崇尚极简的张小龙很头疼QQ邮箱呈现出的“臃肿样子”，功能繁多，但大多不是用户痛点。张小龙决定放弃之前的繁复路线，转型做一个轻便的极简版，而且极简版保持了一个极快速的迭代节奏，每两周，最多三周会发布一个新的版本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他定下一个基调：<strong>踏踏实实追踪用户的需求，看看用户到底需要什么。</strong>张小龙给邮箱团队制定了“1000/100/10”的法则，要求每个产品经理每个月要去论坛看1000个用户体验反馈并回复、关注100个用户博客、做10个用户调查。如今“1000/100/10”被奉为互联网产品开发的“宝典”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了体验和改进产品，张小龙让腾讯广研员工全部使用QQ邮箱。喜欢打磨产品的马化腾也加入了其中，他经常半夜12点登录QQ邮箱，把发现的问题和Bug转给张小龙，然后张小龙通知各个技术骨干。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>张小龙带领团队用了两年时间找到并解决了一个用户痛点：让QQ邮箱能够发送1G超大附件，激活了大量休眠的QQ账户。当时其他邮箱通常只能发送5M左右的附件。2008年，QQ邮箱成为腾讯七星级产品。2009年，活跃用户达5000万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>QQ邮箱成功之后，张小龙开始试验更多的产品功能，打磨作品。他还推出了阅读空间、邮箱广播等等功能。2010年，QQ邮箱上线了推动陌生人社交的“漂流瓶”功能，活跃用户数一举突破1亿。这些功能都在后来的微信上有所延续和体现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过Foxmail和QQ邮箱这两个产品，张小龙形成了自己成熟的产品价值观，并融入了有互联网产品基因的腾讯氛围，赢得了产品经理出身的马化腾的赏识。再之后，微信上线。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信集中体现了张小龙此前建立的产品价值观，包括用户价值、极简功能等等，也补足了张小龙不擅长的商业化方面。2016年，张小龙提出了微信的四个价值观：</span></p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一切以用户价值为依归；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让创造发挥价值；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品应该是用完即走的；</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让商业化存在于无形之中。</span></p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这四个价值观的导向下，微信从一个移动互联网的工具成长为如今日活用户12.8亿的平台和生态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>三个产品都能取得产品意义上的成功，张小龙认为要归功于好的产品价值观。他说，“我很感谢自己的经历，从PC时代自己一个人做Foxmail，到做QQ邮箱，到手机时代做微信，因为经历了太多的产品，以至于<strong>从骨子里知道什么是好的产品什么是不好的产品，</strong>因此能直觉上遵守一些底线。”</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14664168762e73636de9483.92951018.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>张小龙是理性的。这点体现在他对一个优秀工具的不懈追求上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他说“做一个优秀的工具，对我来说是值得痴迷的。由于我对工具产品热爱，我甚至会亲自动手写代码来打造出一个Foxmail这样的产品来满足自己的制造欲望”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>微信的基础点就是成为一个优秀的工具。</span></strong><span>2011年1月，微信作为一个移动互联网的即时通信工具上线，通过语音聊天这一个极简功能获得了第一个快速增长点；通过朋友圈功能升级为一个社交工具。2012年，微信推出公众号，成为商家、自媒体等内容创业者触达用户的工具和内容平台。2016年，微信推出企业微信，成为企业提高协同管理效率的工具。2017年，微信推出小程序，成为商家为顾客提供服务的工具。如今，微信成为融入近13亿人日常生活的工具。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，在2021年微信上线十周年之际，张小龙总结微信的原动力是<strong>“坚持做一个好的、与时俱进的工具”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，深受乔布斯的极简产品观影响的张小龙，认为“极简方不被超越”，所以他一直追求极简。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Foxmail的产品界面就十分简单明了，微信发展了11年依旧保持早期的极简风设计界面，后期衍生出现的功能如小程序、视频号、看一看、微信支付等都是以组合插件的形式融入其中，以不打扰用户体验为前提，并且用户用的时候才会看到，用户用完即走。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>极简是一种能力，“简单的目标是一个最高的目标，特别难做到”。张小龙说过，“工程里面我们实现一个目标可能有一千种方法，只有最简单这种方法可能是最好的，正是因为有一千种方法存在，你要从一千种方法里面找到最好的方法，是最困难也是最难的一点。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他以前在“饭否”上写道，“一个产品要加多少功能才能成为一个垃圾产品”。如今做了微信，他要做到<strong>“一个产品要加多少功能才能还不是一个垃圾产品”。</strong>因为“10亿人用它的时候，简单的东西才最好用”。所以，现在的微信和十年前一眼看上去没有什么变化。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信的成功或多或少跟张小龙本身的性格特质有关。他是孤独的，不善沟通的，但又渴望沟通，所以需要借助工具完成沟通。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>张小龙说，“这么多年了，我还在做通讯工具。<strong>这让我相信一个宿命，每一个不善沟通的孩子都有强大的帮助别人沟通的内在力量”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他做QQ邮箱时，就尝试了很多社交功能，包括漂流瓶、阅读空间等等，当时在QQ邮箱的阅读空间里面，就有基于社交的阅读，朋友推荐文章并且可以在下面共同来评论。后来，这些产品功能被搬到了微信上，成为了摇一摇、朋友圈等功能。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信的起源也是因为他想做一个沟通的工具，“我意识到这里是一个机会，这个机会不是因为kik的产品本身，而是我自己当时开始用智能手机，而很多基于PC的产品或者短信都不能实现很好的沟通体验。<strong>当时想法很简单，希望给我自己或者少数人做一个沟通的工具</strong>”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们刚好有一个团队在做QQ邮箱手机客户端，所以刚好凑了十个人的团队开始做微信。包括后台开发，三个手机平台的前端开发，还有UI，加我自己带了一个产品毕业生，就十个人。经过两个月的时间，做出了第一个版本”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信至今使用的开屏页就是张小龙亲自选定的，是他内心渴望的最好说明。“当时UI给出了好几个方案，其中一个是月球表面图，有很浩瀚的宇宙感，我建议改成地球。上面是站一个人、两个人还是很多人，（团队）也讨论了一阵，最终决定只站一个人”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/137661200762e73637574348.43330769.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，张小龙能做成微信的另一个关键因素是背后的腾讯支持，尤其是马化腾的支持。有人总结，“就公司哲学而言，微信的成功，仍然是马化腾式的胜利”。<strong>张小龙遇到了合适的人、合适的公司，两者互相成就。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在产品层面上，张小龙与马化腾有点“英雄所见略同”的感觉。他们同是产品经理出身，同样坚持用户价值导向，同样对产品构建和细节打磨有着近乎偏执的爱好，同样是极简主义者和直觉主义者。所以，张小龙做微信，马化腾大力支持。在早期微信快速增长阶段，张小龙不喜欢蜂拥而至的媒体采访，这时候同样不喜欢面对媒体的马化腾亲自出马，“还是我替小龙去吧，让他专心做产品”。张小龙的微信也帮助腾讯抓住了移动互联网的时代机遇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在商业化策略上，张小龙和马化腾是同频的，坚持<strong>“让商业化存在于无形之中”。</strong>微信成长于腾讯转向开放的阶段，马化腾提出了向合作伙伴开放流量和资金的战略，这些促进了微信公众号、小程序、企业微信等功能的推出和发展，推动着微信从产品工具走向一个生态，成为各行各业的“互联网+”平台，让腾讯实现了“扎根消费互联网”的战略目标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信的成功让张小龙被推上了“中国第一产品经理”的位置，互联网产品人开始了“拜神”的追逐。但是，张小龙觉得自己只是很幸运，<strong>“是被上帝选中的那个人，光靠个人努力做不到这一点。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“作为一个产品经理，能带领团队做出一款十亿级的产品，当然是很大的成就感。但是，我更觉得幸运的是，我可以在过程中<strong>将自己对这个世界的认知，体现在产品中，并成为产品的价值观。</strong>这是更难得的事情。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>张小龙做产品的经历证明了，<strong>在追求快速变化、盲目求新以及崇尚流量变现的互联网商业大环境下，真正好的产品不在于多么快、多么新或者多么聪明，而在于始终坚持好的产品价值观。</strong>微信的成功，很容易让人忽略张小龙此前在产品领域超过1万小时的锤炼。24年前，他在毫无商业回报的情况下，一个人写代码完成了Foxmail的三个版本迭代。Foxmail受到国内外用户的欢迎，甚至被很多粉丝翻译成十几种语言版本。如今看，又有几人为了做好一个产品可以做到如此程度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本文梳理了张小龙开发微信过程中的55条思考，包括对微信公众号、小程序、企业微信、视频号等多个关键功能的认识，这一定程度上解释了为什么做出微信的是张小龙，而非他人。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1.<strong>产品是进化出来的，而非规划出来的。</strong>有DNA的产品才会进化。DNA到底是什么？DNA是产品的价值观和认知。（2012年腾讯内部演讲“微信背后的产品观”）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2.<strong>极简方能不被超越。为了保持微信的简单，在微信里面，除了核心功能，大部分的功能都做成插件的形式，这个方向很早就定了下来。</strong>（2012年腾讯内部演讲“微信背后的产品观”）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>3.微信十年，我用两个词来表达，<strong>一个是“连接”，一个是“简单”，这就是微信最核心的东西。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>做连接，我们的重心不是做内容，而是做底层的连接，这也是为什么我们以去中心化为前提。我们做好连接，基于这个连接产生出非常丰富的结果出来。公众号、小程序都是连接，微信支付是货币的连接，视频号也会变成很多的基础设施。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简单是很美的，一个物品往往都是最简单最美。现在的微信和十年前一眼看上去没有什么变化，10亿人用它的时候，简单的东西才最好用，简单的目标是一个最高的目标，特别难做到。我们追求这种简单，因为有一部分人知道产品简单背后的美感，他会认同这一点，这个也是挺重要的。（2021年1月微信公开课的演讲）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>4.（2021年1月微信公开课的演讲）我希望微信一直保持自己的风格，<strong>一直像一个小而美的产品一样，</strong>它有自己的灵魂，有自己的审美，自己的创意，自己的光亮，不仅仅是一个数字目标的奴隶，这是我对微信十年进行的总结。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5.<strong>从人出发，改变商业模式。</strong>商业模式的改变，是让用户能够动起来，然后才会产生新的商业模式，而不是我们先计划好一个商业模式，用户就跟着计划动了。这只是一厢情愿而已。我们做产品的时候，常常会思考：到底这只是我们自己一厢情愿地认为可以，还是用户的意愿。这里的区别会非常大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6.<strong>做平台，需要有原动力。</strong>当一个平台只是追求自身的商业利益最大化的时候，它是短视的，不长久的。当一个平台可以造福人的时候，它才是有生命力的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7.（2021年1月微信公开课的演讲）微信的原动力总结为两点：<strong>第一，坚持做一个好的、与时俱进的工具；第二，让创造者体现价值。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信的很多创新其实都来自这两个原动力。从专业的角度来说，大家可能会认为这是对未来的一种洞察。但是<strong>所有专业洞察的背后，我觉得原动力是第一位的，一个好的产品是有自己的使命的。</strong>我很庆幸，这么些年过去了，微信的原动力从来没有变过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8.微信有一句slogan：<strong>微信是一个生活方式。</strong>为什么是一个而不是一种？其实我解释不清楚，但我知道，如果是“一种”的话，它就是一句普通的话，起不到一个slogan的作用，也不能让人记下来。<strong>它必须是一个生活方式，这只属于微信的，它是一句独特的话。</strong>我脑袋里面是有一个念头的，微信会介入到每一个人的日常生活里面去。当时微信并没有覆盖到那么多生活的方方面面，甚至连微信支付都还没有。但现在（2019年1月）回过头来看，微信确实代表了一个生活方式，它应该紧随时代的潮流，甚至引导时代的潮流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/71660674162e736379b9066.05372222.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9.（2014年12月微信公开课PRO上的演讲）<strong>我们是希望建造一个森林，</strong>而不是说我们要建造一个自己的宫殿出来。我们希望整个森林是去培育它的一个环境，我们让所有的生物或者动植物能够在森林里面自由生长出来，而不是说我们自己去把它建造出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>10.（2019年1月微信公开课PRO演讲）微信从来不做节日运营或者logo的变化，很多人会说微信很“克制”。其实这不是克制的结果，本质上是<strong>微信一直在遵循一种好的设计原则，使得我们不会去做很多影响设计美感的事情。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>11.（2016年12月微信公开课<span>PRO</span>的演讲）很多人认为我们在微信产品里很克制，但我听到这样的评论时有一点点惊讶，因为克制这个词从来没有在我的脑袋里面出现过。我们做很多决定的时候，<strong>要去判断什么样的东西是最合理的，合理性才是这里面要考虑的一个最重要的因素。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>12.（2018年1月微信公开课PRO的演讲）<strong>对微信来说，做选择都遵循一个标准，这个事情是对的还是错的，而不是说它是不是利益最大化的。</strong>很多人会说微信很克制，微信很有情怀，但是我们从来没有说过“情怀”两个字，也从来没有说过我们要克制自己的欲望。因为<strong>做一个好的事情并不是克制什么，而是要判断什么样的事情是该做的，什么是对的，什么是错的，这样一系列的判断很理性的过程，而不是靠一个感觉我这样很有情怀就好了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>13.（2020年1月微信公开课PRO的演讲）<strong>推送改变世界，因为用户更懒了，</strong>包括微信，也是基于推送的。推送什么信息，决定了用户会看什么信息，决定了他在一个什么样的世界里。这是一个我们要经常思考的问题，也是我们在努力的方向。<strong>我很少说分发这个词，我觉得推荐可能更尊重一些。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>14.（2020年1月微信公开课PRO的演讲）我记得附近的人上线时，我自己其实都有隐隐的不安，因为从前我们和附近的人的界限将被打破，我不知道它是好是坏。<strong>这种思考，其实会一直贯穿在微信的进化里面。</strong>就像我们扩大5000好友这个限定非常容易，但是对于它带来的影响，说实话诚惶诚恐，我们会反复思考。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>15.（2016年1月微信公开课PRO的演讲）微信对于产品和平台的四个价值观：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一切以用户价值为依归；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让创造发挥价值；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品应该是用完即走的；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让商业化存在于无形之中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>16.（2019年1月微信公开课PRO演讲）<strong>一个新产品没有获得一个自然的增长曲线，我们就不应该去推广它。</strong>所以，在前五个月里面，我们基本上没有推广，我们想看看微信对于用户有没有构成吸引力，用户是否愿意自发传播它。如果用户不愿意，我们怎么样去推广它也是没有意义的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们特别庆幸当时做了两个很正确的决定：第一，我们没有批量导入某一批好友，而是通过用户手动一个一个挑选；第二，在一个产品还没有被验证、只能够产生自然增长的时候，我们没有去推广它。把这两个事情做对，虽然这个时间会花得长一点，但是这样使得它真正开始起飞的时候，它是很健康的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从微信2.0开始的时候，我们看到了曲线，有了一个增长，虽然还不是很快增长，但是自然往上走的。这个时候我们就知道，我们可以去推它了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>17.（2019年1月微信公开课PRO演讲）<strong>很多想法是突如其来的，或者说，是上帝编好程序，在合适的时候放到你的脑袋中的，但也不是完全无迹可寻。</strong>我们后来的很多产品，都有QQ邮箱的影子在里面，比如订阅号、朋友圈。在微信上线之前的一年里，我们把QQ邮箱做到了国内第一名，做了很多尝试，包括漂流瓶等等，包括我花了一年多时间折腾的邮箱里面的阅读空间。在阅读空间里面，我们尝试了各种社交的形式，基于社交的阅读，朋友推荐文章并且可以在下面共同来评论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>18.（2021年1月微信公开课的演讲）<strong>做产品是异想天开的过程，你先有一个想法去验证它99%都不靠谱，如果最终有一两个做下来能够靠谱，就会非常有乐趣。</strong>如果拍下一个人的头像会怎么样，拍一拍，有人跟你聊天，你扔一个炸弹把他的屏幕炸碎了，看起来真的碎了，会怎么样？听一首歌的时候你能看到跟你一起听歌的还有谁，还有画面，你在某一种状态里面，你失眠了发现还有一些人也在失眠，甚至你跟他们一起数羊，这也是很有意思。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>19.（2016年12月微信公开课的演讲）我在朋友圈发过一个朋友圈，时间挺早的，是2012年5月23日，当时我在朋友圈里发了这样一条消息，<strong>PC互联网的入口在搜索框，移动互联网的入口在二维码。</strong>这个反映了PC时代和智能手机时代的很大区别。PC时代我们通过网站来获取服务，我们要先找到这个网站，所以搜索框在当时特别重要。在智能手机时代，我们要获取服务的话，更多的是跟线下有关，就目前手机技术的实现来说，可能扫二维码这样的一个方式是最简易的一种方式。所以<strong>二维码在微信里面的重要性一直都很高，</strong>从我们添加好友到订阅公众号等等，它都是微信里面的一个基础入口，扫一扫也是微信里面的一个非常非常基础的功能，<strong>扫一扫甚至变成了微信去触达周边的一个最基本的方式。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>20.（2019年1月微信公开课PRO演讲）最近我们发布了微信7.0版本，又有5亿人吐槽，有1亿人教我怎样做产品了，还有8亿人看不懂我们的一句话，<strong>“因你看见，所以存在”，</strong>到底是什么意思。这一句话可以从很多的层面理解，我不想做一个解释，<strong>有一个神秘感特别好，每个人有自己的解读是特别好的。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像这么多年以来，微信的启动页面总是一个人站在地球前面，刚发布的时候，很多人问我为什么是一个人站在地球前面，更早的版本是一个人站在月球前面，那个时候也是很有想象力。对于这个点，我相信每个人都有自己的理解，因为我们没有标准答案。我觉得它是一个特别好的启动页面，因为它把想象空间留给了用户自己，10亿用户有10个亿的理解，他会找到打动他的点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>21.（2016年12月微信公开课的演讲）关于用户价值，我有一天分享过亚马逊CEO的一个文章，文章标题叫做《善良比聪明更重要》。我相信在座的人都是很聪明的人，因为大家会想到很多很多的方法去欺骗用户，欺骗用户是最容易做的事情，因为只需要聪明就可以了，但这是不对的，因为欺骗用户虽然很容易获得流量，可以获得用户的点击，但是最终会把用户给赶跑了。<strong>善良比聪明更重要，怎么样对用户是好的，这个聪明会更重要一些。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>22.（2021年1月19日微信公开课的演讲）<strong>我们对用户的态度必须是一种善良的态度，而不是一种套路的态度。这种善良是一种基于理性之上的善良。</strong>只有我们对待用户有一种真正的理性的善良，才会使用户更长久地使用我们的产品。如果这是一种非理性上的善良，我认为它是一种愚昧的善良。所以，<strong>我认为善良本质上是一种能力。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>23.（2018年1月微信公开课的演讲）<strong>在微信里，我们说不能对用户称“您”，而是“你”。</strong>我们不需要用一个很尊敬的态度称呼用户，而是应该<strong>当朋友一样称呼，是一种很平等的关系，</strong>这个已经写进我们的产品条约里了。我们<strong>一旦对用户过于尊敬，说明我们可能怀有目的，可能需要骗一点什么东西过来。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尊重用户和尊重每一个个人，意味着我们可能把用户当做朋友，意味着我们必须给用户提供最好的产品和服务，当然也意味着别的什么——也意味着我们不会去看用户的聊天记录，从微信第一个版本的时候，它的系统就是这样设计的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在尊重用户的层面来说，我觉得我们算是业界做得最好的一个产品，这体现到每一个小点里面，包括我们不发任何系统的推送，我们不去做任何诱导的行为，也包括我们不允许第三方做任何诱导用户的行为，甚至包括我们不想做太多的活动去感动用户，然后带来一些流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>24.（2019年1月微信公开课PRO演讲）我很喜欢一句电影台词，<strong>“万物之中，希望至美”。</strong>如果微信不能给用户带来哪怕多一点点希望，我们就没有办法去判断我们做的事情到底是对的还是错的，所以它也是我们衡量的准则。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>25.（2019年1月微信公开课PRO演讲）<strong>朋友圈承载了中国人的线上社交，可能是中国最高效的社交工具。</strong>现在每天有这么多人这么多次（每天7.5亿人进去朋友圈，平均每个人要看十几次，每天的总量是100亿次）地频繁进入朋友圈，用户每天要通过做这个功课完成一个普通人一天的社交，这一天他可能没有出门，并不妨碍他完成社交。即使你可能不发朋友圈，但是你会去看，你通过点赞评论还是会参与到社交里面，就像完成了一个社交任务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>26.（2019年1月微信公开课PRO演讲）朋友圈本质上是一个社交化环境，是一个对自我人设进行强化的一个地方，而阅读只是朋友圈的一个辅助性的副产品。人们在朋友圈里面推荐文章，其实更多是推一些符合自己的观点或者人设的文章，而不是说这个文章的内容有多好，<strong>更多是通过一个文章来展示自己属于哪一种人。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>27.（2019年1月微信公开课PRO演讲）我一直很相信通过社交推荐来获取信息是最符合人性的。因为在现实里面，我们接纳新的信息，并不是我们主动到图书馆或者到网上去找的信息。大部分情况都是听到周边的人的推荐而获得的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我认为，要做一个让所有人阅读的大众化阅读产品，似乎只有基于社交推荐，才有机会。<strong>本质上，我们不是在做阅读，而是每一个人在帮助其他人阅读，是一种社交性的阅读。</strong>好看功能上线，跟之前看一看的机器推荐比起来，数据增长得特别快，明显看到用户对这个功能的兴趣更大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>28.（2021年微信公开课PRO演讲）<strong>我们一直在寻找所有领域的事情的最基本元素，我们做的不至于太多，但是又非做不可的东西。</strong>表情是我们在表达方式里面一个最基本的元素，跟所有别的方式都不同，模拟人在现实里面身体行为动作，而不是通过发一个信息。微信表情已经很多年没有做大升级了，用户贡献了非常多的自定义表情，使得它能够一直有最新的表情在里面运转。我们观察了一些比较小的数据量，能反映出一些趋势，比如说人们表达出来的情绪越来越强烈了，裂开这个表情非常受欢迎，以前没有这么多比较强烈的东西。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>29.（2021年微信公开课PRO演讲）对一个内容形态的产品而言，<strong>一定量级的用户和活跃度才意味着走过了生死线，就是它的流量循环已经起来了。</strong>否则的话，你一直要为它的生存去找到方法才行。这是典型的微信方式的产品方法，我们找到事情的撬动点，用产品规则的方法让整个事情流转起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>30.（2021年1月微信公开课PRO演讲）如果我们不花钱去买，内容创作者还愿意进来，才说明我们真的建立起了一个可以运转起来的生态，因此在微信历史上我们一直不太强调（运营）也是这个原因，<strong>如果我们在系统和规则方面的效率做得好，反而会比运营效率高很多。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>31.（2019年1月微信公开课PRO演讲）很多产品经理会被自己所在的公司误导，因为公司的目的是要流量要变现，所以大家的<strong>KPI就是如何产生流量如何变现。一旦围绕这个目标，工作目的已经不是做最好的产品，而是用一切手段去获取流量而已。</strong>这并不是我们倡导的原则，<strong>我们更多倡导的是利用微信做出好产品、分享给用户。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>32.（2021年1月微信公开课PRO演讲）微<strong>信很大的价值在于它的身份或者说它的ID价值。</strong>有了这个身份我们可以做很多事情，比如说微信支付，因为我们来做支付是最方便的，所有支付都是和身份相关联。<strong>ID是基于社交和通信领域，是私密的，并不是公开领域的。 </strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>33.（2021年1月微信公开课PRO演讲）<strong>ID还有一个特别重要的作用，不是针对个人，而是针对机构。</strong>PC时代，每一个公司机构都会建立自己的网站、注册一个域名，域名是它最重要的ID。在移动互联网时代，建立一个网站已经没有意义了，微信始终想着怎么样去帮助机构建立官网。我们做公众号的时候，公众号就是每一个机构的官网；做小程序的时候，小程序是你的官网，可以把服务放在里面。做视频号的时候，视频号就是你真正的官网。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/101054074862e73638694b94.22590923.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>34.（2019年1月，微信8周年，日活用户突破10亿）<strong>微信团队从开始成立到现在从来没有瞄准KPI去奋斗过，但并不妨碍团队能够越做越好。</strong>就像小程序，如果围绕KPI去做，我们不知道怎么制定KPI，因为它没有这个东西，没有办法制订它的KPI，如果围绕一个KPI，大家可能不会做这个事情。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于团队来说，大家养成了一个习惯：我们做每个功能、每个服务背后的意义，或者说一个梦想在里面到底是什么？产品和功能背后我们所思考的意义，这是支撑起我们整个团队走到今天一个很强的理由，并且帮助我们做出一个正确的选择。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>35.（2019年1月微信公开课PRO演讲）微信的梦想是什么？从个人角度，成为人最好的一个工具朋友；从平台角度，建立一个市场，让创造者体现价值。<strong>在墙和鸡蛋之间，似乎总是站在鸡蛋一边。</strong>如果你做大了，微信会限制你，如果你刚起步，微信会扶持你。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>36.（2019年1月微信公开课PRO演讲）<strong>微信没有竞争对手。如果有竞争对手，就是我们自己，是我们的组织能力能不能跟上时代的变化。</strong>微信也没有焦虑，不必微信做个什么就说微信又焦虑了，所以不必把自己的焦虑投射到微信里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>37.有一位德国的产品设计师总结过<strong>好的产品设计的十个原则，</strong>大家可以对照微信来思考一下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品富有创意，必须是一个创新的东西；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品是有用的；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品是美的；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品是容易使用的；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品是很含蓄不招摇的；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品是诚实的；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品经久不衰，不会随着时间而过时；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品不会放过任何细节；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品是环保的，不浪费任何资源的；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好的产品是尽可能少的设计，或者说少即是多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我在这里偷换了一个概念，将设计替换为通用的产品。很多人会认为这是针对苹果这样的硬件产品的设计原则，但其本质上，不管是硬件产品还是软件产品，都是工具。对于工具设计的原则，都是适用的我认为业界很多产品并不注重产品设计，或者说不把它作为一个自己追求的目标，还只是一种功能的堆砌或者对用户价值的榨取。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>38.这两年业界的目标变成了所有APP应该尽可能多地去抓住用户的停留时长，这跟我对互联网的初心的认知是背离的。每个人一天只有24个小时。<strong>互联网人的使命不应该是让所有人除了吃喝拉撒之外，把时间都花在看手机上面。技术的使命应该是帮助人类提高效率。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>39.<strong>微信永远都不会把用户停留时长作为一个目标。</strong>相反，微信可能关心的是一个人在这里发一个照片，看一些文章，完成一笔支付，找到一个需要用的小程序，是不是能够做到最快速最有效，这才是最好的工具。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>40.（2014年12月微信公开课的演讲）对于公众平台，就像订阅号一样，这里面有非常多的阅读量来自于朋友圈，这是符合2/8原理的：<strong>20%的用户到订阅号里面去挑选内容，然后80%的用户在朋友圈里去阅读这些内容。</strong>我们很高兴这个流量能够发生作用，但一切都是在用户认可的情况之下发生的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>41.（2019年1月微信公开课的演讲）在微信很早期的版本，我们就发布了公众平台（2012年2月上线），这也是微信的一个创新。从公众平台开始，从连接人到连接服务去扩展以后，微信开始体现平台化的优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>42.<strong>微信公众平台的原动力是让创造价值的人体现价值。</strong>做公众平台的时候，我们就会想要帮助人们解决什么问题。当然是通过信息触达来替换掉信息不对称带来的弊端，这是互联网的优势。我们要帮那些真正有好的服务的人和集体，去触达潜在的用户，让客户更容易连接到他们。所以，<strong>当时定下来公众号的slogan是“再小的个体，也有自己的品牌”。公众号就是品牌，并且品牌是基于关注和认可的。</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>43.（2019年1月微信公开课PRO上的演讲）<strong>小程序是我们，或者说，也是我个人职业生涯里面最大的一个挑战。</strong>因为我们从来没有试过还没做一个事情就先宣布出来了。之所以这样做，是为了给自己一个压力，给团队一个压力，这个事情我们非做不可，而且一定要做到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在微信公开课里面说我们要推出小程序这样一个服务的时候，当天晚上我跟我们的团队就坐在一起讨论一个主题，我们小程序会有哪几种死法会挂掉？我记得特别清楚，因为当天晚上我们不是讨论小程序有多么美好的未来，而是说它有多难。我们会遇到哪一些障碍是跨不过去的，<strong>我们并不是对它乐观去做的，而是我们觉得这个事情很难、我们一定要做到而去公布这个事情。</strong>为什么非要做这个事情？因为我认为这就是一个未来必然的趋势，因为APP还要下载安装，网页的体验又太糟糕。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>44.（2016年12月微信公开课PRO上的演讲）一个好的软件，一个好的工具应该让用户用完即走的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>45.（2018年1月微信公开课PRO上的演讲）<strong>用完即走和用户再回来，其实并不矛盾。</strong>只有当一个用户在一个工具里用得很愉悦，用得很高效，他才会下一次回过头来使用这个工具。小程序是这样的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/87749503862e73638c6ae38.76414400.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>46.（2019年1月微信公开课PRO上的演讲）小程序的使命是让创造者体现价值和回报，一切都是围绕这点展开。<strong>不能因为拥有流量，我们就要分发流量，要让小程序来体现这个流量价值，</strong>这跟我们的原动力是完全不一样的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>47.（2019年1月微信公开课PRO上的演讲）很多人看不懂为什么小程序要去中心化。如果不去中心化的话，腾讯自己垄断了头部几个小程序，那就没有外部的开发者什么事了。看起来腾讯可以短期获利，但这个生态就没有了。哪怕是腾讯投资的公司，也应该一样地遵循平台的规则，否则也会破坏平台的公正性，我们更看重的是平台的健康。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>48.（2018年1月微信公开课PRO上的演讲）<strong>“去中心化”与其说是平台的策略，还不如说是一个观念，这个观念代表着我们去看待这个世界的方式。</strong>在一个“去中心化”的世界里，每个独立的个体都有自己的思考，都有自己的大脑，我们认为这样一种系统的健壮度，可能会远远超过只有一个大脑来驱动的系统。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>49.小游戏做到现在（2019年1月），其实如果从商业的角度，发展挺好的。但我并不满意。小游戏的原动力是什么？不是公司的一个盈利渠道。公司也没有要求微信要做游戏。一切盈利都是做好产品做好服务后的自然而来的副产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小游戏的原动力是，它应该是一个关于创意的平台。并且让产生创意的人体现价值。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所谓创意的平台是，<strong>小游戏是一个载体，它可以承载各式各样的创意。</strong>我们经常用一个案例来说，可能一个小学生用很少课外的时间掌握了小游戏的开发，他来开发一个小游戏给班级的同学来用。这个小游戏是他自己想象出来的，创造出来的。是他的同学特别喜欢玩的智力类型或者是好玩类型的。所以，它不应该是传统意义上的那种小块头游戏的翻版。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>51.（2019年1月微信公开课PRO上的演讲）如果企业微信定位为公司内部的一个沟通工具的话，它的场景和意义会小很多。当企业微信延伸到企业外部的时候，它会产生更大的价值。企业微信后续的新变化是基于一个新的理念：<strong>每一个企业员工都成为企业服务的窗口，人就是服务，而且是认证的服务。</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>52.（2020年1月微信公开课PRO演讲）在微信的起步阶段，我们基于手机来做APP，不基于PC来做，PC端只是辅助。回过头来看，当年有两个失误，一是公众平台在很长时间里只有PC Web版，这限制了内容创作者的范围；二是公众平台最终成为以文章为内容的载体，但是其他的短内容形式没有呈现出来。这也是为什么之前我说公众号本身并不是为媒体准备的这样一个原因。相对公众号而言，我们缺少了一个人人可以创作的载体，因为不能要求每个人都能天天写文章。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>53.（2021年1月微信公开课PRO的演讲）理解了公众号对文章的价值，大家就可以理解视频号对视频的价值，我觉得这两个是非常类似的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>公众号是文章的载体，</strong>因为文章存在的价值是被分享出去有人看到，如果没有公众号这样一个载体，我们只能传一些文件出去是很费劲的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>视频号更多的意义在于，它是一个视频内容的载体。</strong>我们不是做这个内容，我们也不会去生产内容，不会去买内容，我们并不关注具体的内容到底是什么，载体的意思是我们只做内容的承载和传递。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>54.视频化表达应该是下一个十年（2021-2030）内容领域的一个主题。虽然我们并不清楚，文字还是视频才代表了人类文明的进步，但<strong>从个人表达以及消费程度来说，时代正在往视频化表达方向发展。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>55.（2021年1月微信公开课PRO的演讲）视频号的意义，视频还是其次，更重要的意义是有一个账号在这里，通过这个号每个人都有了一个可以公开发声的身份，将来他发声的内容可能是视频，也有可能是直播的一些东西。<strong>拥有视频号这个ID之后，微信直播才变成一种公开领域的事情。</strong></span></p>
