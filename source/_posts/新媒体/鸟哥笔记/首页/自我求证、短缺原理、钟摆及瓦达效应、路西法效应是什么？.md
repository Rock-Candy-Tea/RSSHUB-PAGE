<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>1.</span></strong></span><strong><span>为什么算命仍有不少市场？</span></strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>算命先生未卜先知的技巧本身是提供<strong>“模糊的信息”</strong>，然后让听者自己进行信息的具象化，这时模糊信息就会依照具象信息生根发芽，从而感觉很精准。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此类相似情况有很多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如同看完恐怖电影后，走夜路总感觉有人追踪；去商场购物被导购夸奖很美，穿上总感觉为自己设计；恋爱中女方不经意的回眸一笑，男方能回味半个月，甚至越想越甜蜜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>种种情况，<strong>在心理学上被称为自证预言（self-fulfilling prophecy）。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>它的另一个名字叫<strong>“<a href="https://www.niaogebiji.com/tag/%E8%87%AA%E6%88%91%E6%B1%82%E8%AF%81" target="_blank">自我求证</a>心理”</strong>，由美国心理学家罗伯特·罗森塔尔（Robert Rosenthal）提出。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通常指人们会不自觉的按照已知的预言进行行事，最终令预言发生，也指<strong>对他人的期望会影响对方的行为，使得对方按照期望行事。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一定程度上自证预言有运气的成因所在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，<strong>当我们渴望某件好事即将发生，会倾向找到符合期望的正面信息</strong>，而那些正面又诱发我们寻找更多正面，从而使自我变得充满自信、行为更加积极。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相反，<strong>假设我们越担心坏事的发生，便会越留意不利的讯息</strong>；不利讯息越多心情越加焦虑，从而消极一面即出现更容易引发坏事情发生。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最常见的“塔罗牌占卜、星盘分析、AI云计算”等各类算命广告<span>均</span><span>使用该原理，</span>用话术套路用户、贩卖焦虑；背后不过是掌握大量信息，然后按照兴趣推荐罢了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span>日常最应该注意，</span>自我证言有</span><span><strong><span>“积极”和“消极”两面。</span></strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>假设是消极面，请尝试在某段时间内要求自己往正向思考，如同社交中觉得对方脾气不好，那有无可能把它转移为还有哪几个值得我重新审视？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>另外积极求证不能盲目，存在四种方法：</span><strong><span>1）向自己提问，2）通过实验验证，3）举出反例，4）远离归因谬误</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>首先大胆假设后要问自己还有</span><span><strong>“替代假设”</strong></span><span>吗？</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，过去6小时男朋友没回我信息大概率是生气了，有关该问题发生的根源也可能存在“忙着开会”的情况。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在比较几种可能假设时，<strong>要观察是否符合逻辑、是否和其他事情之间有一致性、用该假设能否解释过去事件</strong>，从而预测未来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>其次</span><span><strong>尝试验证信息的准确性</strong></span><span>，这样积极判断是存在侥幸心理还是我有付出很大努力过，如同考试，很多人总觉得考前最后一天奋斗就能过关，其实不以为然。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>再者</span><span><strong>一个底层逻辑越是普遍适用，找到反例的机会就越大</strong></span><span>；同理，自己作出的假设要不断用举反例的方法进行验证，最后远离归因的谬误。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>好比有的人认为“幸福是财务自由”，“财务自由是幸福”，是因为他们找到了属于自己幸福和财务的属性，</span><span><strong>显然，这并不能套入自己的环境中使用。</strong></span></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>2.人为什么害怕失去？</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>尊敬的各位顾客，从现在开始2小时内香米限价2元/斤，瘦猪肉10元/斤，数量有限售完为止；</span><strong><span>多数大妈听到如此美妙的声音，会不约而同</span>地<span>朝着特价商品冲过去。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当气喘吁吁赶过去后，导购可能会说“不好意思，猪肉已经卖完了”；或现在正在补货中，你不妨到四点钟准时过来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有时价格相差几块钱人们都愿意等半天，为什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>心理学中把此类现象称为</span><span><strong>极度“<a href="https://www.niaogebiji.com/tag/%E7%9F%AD%E7%BC%BA%E5%8E%9F%E7%90%86" target="_blank">短缺原理</a>”（shortage principle）</strong></span><span>，即<strong>当人们感觉要失去一些东西时，会对自我决策产生极大影响</strong>；同时害怕失去某些东西比希望得到同等价值东西，对人的激励性更大。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早些年的芭比娃娃在推出某款新品前会大力的做内容传播，制造新闻，提高公众的关注度，趁势销售；当然这还不足以形成热销引不起人们争先抢购，但它和“全球限量版”挂上钩时，就立马畅销全球。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>其实，短缺的核心是</span><span><strong>利用人们想走捷径，经常根据得到某种东西的难易度来快速做出依据的弱点进行放大，尤其在“物以稀为贵”上体现的淋漓尽致。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那些限时促销、唯一机会的背面潜意识告诉消费者产品供应短缺，不能一直保持有货。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管有些短缺存在真实或虚假现象，但不管哪种情况都可能使顾客相信某些东西不可多得，从而立刻感觉Ta身价百倍，并达到采取购买行动的效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种效应用也用在其他场景下；比如：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小张抱着试一试的态度去某公司面试，成功应聘后得知该公司设置了末位淘汰制，年终最低者将被辞退；于是Ta为不被辞退而拼命工作，结果拿到了领导满意的KPI。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>明白该思维，我们该如何使用呢？一句话总结为，</span><span><strong><span>你告诉某个人你不这么做将会失去什么，比你这么做会得到什么的效果要好很多。</span></strong></span></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>3.人的情绪为什么飘忽不定？</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>假设我告诉你，人的心里有个“钟摆”你可能不会相信；那么请回想下，自己的心情有时是否像大海一样波涛汹涌，大起大落。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如同，跟朋友聚会时热闹的像个小孩，但回到家中面对独自一人的冷清，突然开心不起来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>亦或者出去玩感觉很开心，但回来后面对日常单调又枯萎的的情景，总觉得有些心烦，</span><strong><span>这些都是我们内心向两个极端摆动的现象，心理学中被称为“钟摆效应”（Pendulum Effect）。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实人的心理有十分明显的两极性，即肯定与否定、积极与消极、紧张与轻松、激动与平静、爱与恨、乐与悲、赞成与反对等等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在日常生活中，人的心理会随着这些背景活动而产生在两极之间摆动的现象；多数人不解，一个正常人为什么会有此心理？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主要有三个原因：</span><strong><span>1）性格本身存在起伏现象，2）与人格相关，3）环境和角色落差较大</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>常规而言，各种外部“信息”摄入大脑会产生情绪反应，性格变化就成为基础规律。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次在美国心理学家高尔顿·威拉德·奥尔波提出的（人格特质理论(theory of personality trait）认为<strong>人特质有三大类型覆盖多种动机，而每个动机就像一粒颗粒随时在影响人的心理发生变化。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我在9岁之前非常沉默寡言，不爱与朋友社交；可到12岁那年受到兴趣影响变得非常活泼，而14岁时又因为学自行车经常被摔，变得个性更加冲动易怒，这些人格变化的背后均会影响情绪发展的状态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>心理学家研究也发现，一般环境与角色反差较大</span><span>也容易产生钟<span>摆效应</span><span>，反之则不太容易产生；</span></span></span><span>主要因为人的感情在外界刺激影响下，具有</span><span><strong><span>“多维</span><span>度<span>性”和“两极性”</span></span></strong></span><span>的特点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>每一种感情具有不同的等级，还有着与之相对立的情感状态；那么在这种情形下出现的“心理斜坡”就越大，因此也就越容易向相反的情绪状态进行转化。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>面对“钟摆”带来的不良反应，该如何灵活应对呢？</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>首先要知道现实生活中，难免会遇到一些自己不愿意接受的事实，对于那些无法选择的，</span><span><strong>明智应对方案就是默默地接受，从而避免心情陷于低潮。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>其次要时刻认知到</span><strong><span>“环境不能决定自己是否快乐，对事情的反应才会决定自我心情”。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>显而易见，对事情的看法决定钟摆幅度，想摆脱一切讨厌和不健康的情绪，就必须深思一切怡人和有益健康的事情。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>再者，要懂得消除一些思想上的偏差</span></strong></span><span>；人生有聚也有散，生活有乐也有苦，我们不能一直生活在激情、浪漫、刺激的理想境界之中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，应该学会体验各种状态的乐趣，既能在激荡人心的活动中体验热烈奔放，又能在平淡如水的日常生活中享受悠然自得的生活情趣，才能避免失落感和消极情绪。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>4.瓦伦达效应是什么？</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美国有一位著名走钢丝表现者叫“瓦伦达”；在1978年Ta决定走完最后一次钢丝然后宣布退休，<strong>没想要以前从未有过任何差错的Ta这次居然失败了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>在Ta走到钢丝中间仅做了两个难度并不大的动作后，就从数十米钢索上摔下来。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>事后据Ta妻子回忆，瓦伦达在出演前就不断</span><span>地</span><span>自言自语说：“这次表现非常重要不能失败”。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>这种巨大压力下心理患得患失的心态，后被人称为</span><span><strong><span>“瓦伦达效应”</span></strong></span><span>（Karl Wallenda Effect），它和哈佛大学社会心理学家丹尼尔（Daniel）提出的白熊效应相似。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也就是说，<strong>以前每一次成功表演中只想着走好钢丝这件事本身反而容易成功，而这次表演中瓦伦达太想成功，反而无法专注于事情本身，变得患得患失。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这如同，我们常说的“压力就是动力”就存在正方面，驾驭得当可化为无往不利的法宝，反之则可能会对自己造成伤害。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美国斯坦福大学一项研究也曾表明，当我们在认真做事时不去想其他“额外”信息，反而更聚焦；</span><span><strong><span>因为那些额外信息产生的虚幻图像会像真实发生一样，有着刺激着神经系统的作用。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就好比，当高尔夫球手击球前，一再告诫自己一再“不要把球打进水里”时，他大脑里就会出现“球掉进水里”的情景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>曾经我在主持一次互联网大会时，因为有位嘉宾的名字有繁体字特别难念，开场前我再三记忆，最后还是嘴瓢了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>看到这里，相信你也会回想起曾经的某些情况；<strong>其实</strong></span><strong><span>现实中我们做任何事情总是想太多，太在乎事情的后果、在乎别人的闲言碎语、反而容易忽略了事情本身。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>我们把大脑欲望塞</span><span>得</span><span>满满，身体却压得气喘吁吁；在重负荷下显然就不能把事情做太好；结果就会偏离预定的轨道，离设想中越来越远。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>明白该理论，你也就明白学者们说的<strong>“要专注、抛去杂念”</strong>的意思；可比这更重要的在于</span><span><strong>「要小心那些给你制造障碍的人」。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你说做份PPT，对方却说你能达到某老师的标准吗？你说写作，对方却说建议先报学习班，“知”和“行”是两码事，要在知中践行，某种程度上<strong>信息的增加反而更容易引发决策困难。</strong></span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>5.环境总会影响个人？</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>路西法是基督教中的堕落天使，天使无疑是“好人”的象征，而好人堕落自然而然就是“坏人”，那么好人如何变成坏人的呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美国心理学家菲利普·津巴多（Philip George Zimbardo）在斯坦福监狱曾经做了一个实验：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>Ta把一批身心健康，情绪稳定的大学生带进监狱；Ta们并没有任何犯罪记录，大学生被分为“狱卒”和“犯人”两种类型进行一周的测试，</span><strong><span>结果这批大学生面对环境的压力变得暴躁，变成另一种模样。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在心理学中，菲利普·津巴多把这项实验称为<strong><a href="https://www.niaogebiji.com/tag/%E8%B7%AF%E8%A5%BF%E6%B3%95%E6%95%88%E5%BA%94" target="_blank">路西法效应</a>(The Lucifer Effect)；</strong>简单而言，人受到特定情景和氛围影响，在性格、思维行为方式上会表现出不可思议举措。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然每个人都是独立个体，但受到社会、工作等种种外在因素影响，有时候群众的力量对人心境改变可以起决定性作用，这也是该效应最核心观点之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>正如作者所言，</span><span><strong><span>“单凭想象，你永远不会知道自己会不会那么做，要知道情境的力量远比我们大得多。”</span></strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管该实验主题是研究好人如何变成恶魔的，但这并不能证明人性本恶；只是证明在不同情景下，善与恶是可以存在相互转化的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在生活中对我们有什么启发呢？我认为用<strong>「环境会影响个人」</strong>这句话总结最好不过。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>进化论中有个概念被称为「与鬼共舞」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其大意为<strong>一个物种在适应过去环境的时候会形成行为A，当环境发生变化不需要行为A了，可物种仍然会延续过去的做法，就像与鬼魂共舞一样。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>海龟宝宝多半在夜间的沙滩破卵而出，然后爬向大海回到自己的家；原因在于，在夜晚，小海龟是靠海面反射的月光找到大海的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可自从岛上有了房屋的灯光后，在岸上新生的小海龟不去大海了，Ta们会朝着灯光一步一步爬去，最终死在爬行的路上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据此对比我们，<strong>假设你想要投入一件事，那从「环境」的改变比行为更重要</strong>，如同我经常写作，在家里始终没有灵感，一旦到达安静的咖啡厅却随时能够进入心流状态。</span><span></span></p>
