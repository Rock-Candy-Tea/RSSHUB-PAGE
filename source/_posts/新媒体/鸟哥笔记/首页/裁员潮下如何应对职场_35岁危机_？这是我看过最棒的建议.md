<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如今一批又一批的职场人被迫“毕业”，而真正处于毕业阶段的学生们则面临着毕业即失业的可能，职场大环境持续变差。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这其中有个特殊群体——35+职场人，他们作为职场危机的代名词，当真正的危机到来时，又会如何应对呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/45809160861c4b3e1643a85.32865786.png" title="裁员潮下如何应对职场“35岁危机”？这是我看过最棒的建议" alt="裁员潮下如何应对职场“35岁危机”？这是我看过最棒的建议" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">“35岁的坎，终究会过去”</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">ID：马铃薯在象岛拼乐高</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">年龄：35岁</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我今年正好35岁，在一滴滴做了4年多运营。2月底的时候突然被通知<a href="https://www.niaogebiji.com/tag/%E8%A3%81%E5%91%98" target="_blank">裁员</a>，就开始了长达3个月的求职之旅。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">离职的时候，公司按规定给了5个月的赔偿金，年假也折了现。这笔钱在当时给了我很大的慰藉。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">赔偿金像是一剂定心针，让我有足够时间可以找工作。整理思绪后，我认为自己更需要稳定性高的工作，国企则是首选。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我从3月1号开始投简历，在3月上旬就面试了4家小型国企。这几家面试的感觉都很不错，甚至有面试官当场加了我的微信，可到头来我一个offer都没收到。这让我慌了，35岁真的是一个坎？我很难找到新工作了吗？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">于是我开始疯狂投简历，可回复我的要么是卖车，要么是卖保险，要么是电销坐席。那段时间特别难熬，我很沮丧，怀疑自己能力不行，但也继续海投。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4月初面试了2家企业，其中一家是车企，当时面试官问我是否认识xxx。原来当天面试的人中，有上家公司同部门一起被裁的同事，被明面挑挑拣拣的感觉真不好。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">没有合适的岗位，也没有新的面试，我决定先去做段时间志愿者，期间车企的HR告知我面试通过了，但疫情原因无法发offer，具体时间待定。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">志愿者工作带来的身体上的劳累让我大脑停止焦虑，心理上没有3月那么难受了。在4月末的时候，我收到了一家国企的复试通知，并于4月29号进行复试。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">5月初我被告知面试通过，进行一系列流程后我接了国企的offer。即将入职的喜悦还没消散，新的抉择就来了，之前的车企也给我发了offer。当时心里五味杂陈，思考下来，我的确35岁了，处于一个尴尬的职场阶段，更适合相对稳定的工作，于是拒绝了车企。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">6月初我正式入职国企，单位离家也很近，一切都在慢慢好起来。35岁的坎，终究会过去的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/76715613761c4b3e1ba7742.10720740.png" title="裁员潮下如何应对职场“35岁危机”？这是我看过最棒的建议" alt="裁员潮下如何应对职场“35岁危机”？这是我看过最棒的建议" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">“2年时间，一切从头开始”</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">ID：灭绝师太嗷嗷嗷</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">年龄：38</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我是一位母亲，自从当了孩儿他娘，自认成了一名中年妇女，还是一位35+大龄互联网底层民工。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">互联网行业让我特别失望，因为36岁了还是没能升到更好的位置，我没有话语权，也没有地位可言。当时是2019年，处于这种状况下的我，选择跳槽谋求更好的发展。但换工作的过程跌跌撞撞，也让我认清一个现实：再不转型，我真的会被淘汰！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">年龄问题，家庭因素，对前途的迷茫，让我那段时间非常焦虑。我尝试过做保险销售，但性格原因，再加上不想消耗自己朋友圈，于是选择放弃。我不想40岁之后去当小时工或者做保洁，也害怕当家庭妇女，我必须要找到新职业。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">之后的一段时间，通过对各行业进行分析和身边资源整合后，我把目光瞄准了法律行业。我知道法考很难，律师会有3-5年的实习期，前期工资也很低，但分析下来这个行业转型可行性强，也没有年龄限制，我一定要成功，没有退路。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">紧接着我花了4个月的时间通过法考客观题，为了接近律师行业，也为了续缴社保，我去了一家律所做跟案助理，那时候工资很低，很难想象北京的律所里，竟有员工薪资为4000块钱。但我咬牙挺过来了，边上班边复习法考主观题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">拿到律师职业资格证是2021年初，我完成了转行的第一步，开启了漫长的实习生涯。这时我换了一家更实力更强的律所做新媒体运营工作，并挂证实习，收入比去年提升了不少。目前我正在准备律师执业证的考核，为成为一名真正律师做准备。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在转型的过程中，我被泼过不少冷水，但我深知自己无路可退，只能付出100%的努力。在我看来，中途转型是人生大事，不能被别人左右，也不要被表象迷惑，一定要想清楚自己到底要做什么。一旦目标明确，从头开始又算什么？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/37761456761c4b3e1f41e73.18661864.png" title="裁员潮下如何应对职场“35岁危机”？这是我看过最棒的建议" alt="裁员潮下如何应对职场“35岁危机”？这是我看过最棒的建议" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">“88年HR，面试20多家单位，</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">成功的秘诀是走向目的的坚持”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">ID：在太空流浪的人</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">年龄：34岁</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">作为一个单亲妈妈，为了生活，为了孩子，我一直勤勤恳恳工作。5年前我在一家创业公司从人力专员做起，一步步到人力主管、人力经理、人力总监，没想到栽在了公司资金链断裂上。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">5年的人力经历，从招聘人才梯队搭建，到薪酬绩效管理，再到员工关系、培训等等，使得我对自己求职道路存有一定信心，但真出来了才知道，自己啥也不是。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在失业的2个月中，我一直疯狂投简历，前后面试20多家单位，最后都石沉大海，杳无音讯。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首先在学历上就被卡死。到了人力资源总监这个岗位，学历基本要求就是全日制本科，而我是非全日制本科，即使降薪50%也很难弥补这一点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其次，今年大环境差，很多公司没有实际招聘需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最后，没有大厂经历也是硬伤之一。作为HR特别需要有一两个特别擅长的模块，在创业公司六大模块认为自己六大模块都会了，其实也只是经历了皮毛。而大厂的员工对自己所负责模块的专业度是很高的，有一段大厂经历的确更受认可。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">我也想过转型。在我面试的20多家企业中，其实只有6家是人力岗位，其他都是销售岗位但因为销售经验不足，也没能取得offer。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">面对这样的情况，我当时很焦心，但我不能垮掉，我知道一定有办法改变现状。于是我复盘了每一次面试，重新梳理了简历，分析了行业发展与职业规划。最终，我决定放弃稳定的人力行业，转型做人事服务行业的销售。经过不断尝试和努力，我终于找到了公司发展方向和个人前景都不错的工作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">通过这次经历，我明白了一个道理，无论什么情况，我们都不能放弃自己，心态放平，多去尝试；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">过程会痛苦，但一定要坚毅，因为成功的秘诀是走向目的的坚持。</p>
