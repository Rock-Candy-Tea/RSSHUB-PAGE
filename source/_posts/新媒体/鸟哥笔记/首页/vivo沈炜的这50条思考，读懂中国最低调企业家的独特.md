<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">任何行业都会经历周期性发展，多数企业会如浮萍随波逐流，只有极少数企业有定力能够逆周期性增长。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">中国智能手机市场经过十年的野蛮生长，增长已达到天花板，年出货量自2016年达到顶峰4.67亿部后，一直呈下降趋势。尽管2021年全年中国智能手机市场出货量约3.29亿台，同比微涨1.1%，扭转了自2017年以来连续下滑的颓势，但这只能算是“回光返照”，并不能改变总体下降的趋势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a href="https://www.niaogebiji.com/tag/vivo" target="_blank">vivo</a>却实现了逆势增长。2021年，vivo以近22%的市场份额首次成为中国智能手机市场的新“机王”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/76529854462b27a4d8d9953.96151723.png" title="vivo沈炜的这50条思考，读懂中国最低调企业家的独特" alt="vivo沈炜的这50条思考，读懂中国最低调企业家的独特" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在刀光剑影般过招的中国智能手机行业里，vivo显得过于低调和温和。从创始人为品牌站台、公开露面的次数看，相比小米、华为、荣耀甚至是“师出同门”的OPPO创始人陈明永，vivo创始人沈炜是最少公开露面的，堪称中国优秀企业家中最为低调的。而且，vivo从不评价同行，也不打价格战——即使是在2017年智能手机市场掀起价格战、行业遭遇洗牌的情况下。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但不可否认的是，vivo已经从一名行业追随者进化为行业领导者。最明显的一个指标是，它的出货量变化：2011年vivo刚进入智能手机领域，此后三年里其出货量从前10进阶到前5，自2016年以来一直稳居国内Top3、全球市场Top6；到2021年，vivo在全球60多个国家和地区服务了4亿多的用户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于vivo的成功方法论，众说纷纭，有不同的解读。2016年，OPPO和vivo打败小米、华为等对手成为中国智能手机市场出货量第一和第二名，被外界视为“黑马”。外界流行的解读是，这两家企业的成功秘诀在于市场营销策略：强大的线下渠道网络和精准的广告投放。但这只能是一时的成功策略，不足以支撑vivo穿越智能手机行业的十年大涨大落，最终在2021年成为第一。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">外界说一千道一万，其分析可能还是停留在“术”层面，更为本质的“道”只有创始人最懂。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">vivo创始人沈炜自己总结，“以本分为核心的<a href="https://www.niaogebiji.com/tag/%E4%BC%81%E4%B8%9A%E6%96%87%E5%8C%96" target="_blank">企业文化</a>与价值观是唯一核心竞争力”。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">沈炜所说的“本分”价值观是步步高公司创始人段永平一直倡导的商业价值观，包含“平常心”“回归本原看事情”“不该做的事情，不做”等颇具有东方哲学色彩的解读。其本质就是一句话：做正确的事情以及把事情做正确。这与西方管理学大师彼得·德鲁克的管理思想是一致的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">vivo沈炜的“本分”价值观与OPPO陈明永所提倡的是一样的，因为两人都是在1995年就追随段永平创业步步高电子公司的创始员工，算是“师出同门”。这也解释了为何vivo和OPPO这两个品牌存在非常多的相似之处：相似的营销策略，相似的渠道策略，相似的企业文化，甚至相似的市场份额，都用英文字母组成品牌名称。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1999年，段永平将步步高电子旗下的教育电子、视听与通讯科技三大业务改制成了“步步高教育电子”“步步高视听电子”与“步步高通讯科技”三家股权独立、相互没有从属关系的公司。其中，步步高视听电子由陈明永负责，这是OPPO的前身；步步高通讯科技由沈炜负责，这是vivo的前身。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">沈炜负责的步步高通讯科技聚焦在电话通讯领域业务，做到了无绳电话机的第一，然后在2004年进入手机市场，生产功能手机；随着智能手机时代的到来，2009年沈炜注册了vivo品牌，2010年成立了维沃移动通信公司，2011年vivo正式进军智能手机领域。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">算下来，从1995年加入步步高电子公司算起，沈炜在电子消费品和通讯科技行业沉浸了27年，历经了通讯科技产品的4次时代进化：电话座机-无绳电话，功能手机时代，智能手机，如今是连接万物的智能终端。所以，他深知商业中的变与不变。在他看来，企业好比是一个水桶，企业文化就是水桶的桶底，战略、技术是围着的木板而已。前者是不变的，后者是可以变化的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，沈炜多次在内部和公开场合强调，“以本分为核心的企业文化与价值观是我们企业仍能存活至今的最重要原因，是我们今天能够与世界一流企业PK的唯一核心竞争力”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不过他对“本分”价值观有自己独特的解读。最独特的一点是，他从因果关系的角度看企业经营，尤其是他近年来多次提到的“埋头种因”。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">沈炜说vivo是一个不太在乎“果”的企业，这不是说“果”不重要，而是要从更本质的角度看。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在沈炜看来，如果没有把“因”做好，是无法得到“果”。什么是“因”？“因”是做正确的事和力求把事情做正确，也就是本分价值观。所以，他在2021年vivo新春公开信中说，“果不能求亦无须求，因决定果，埋头种因，果自会水到渠成”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“埋头种因”的具体表现是什么？沈炜强调了三点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首先，长期主义。只有坚持长期主义，才能不受短期压力、诱惑的干扰，才有可能去做正确的事并把事情做正确。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">沈炜认为，企业如人，快乐、活得健康长久是最重要的，所以他想让vivo成为健康长久的百年品牌、百年老店。为了实现这个长期目标，产品是最重要的，“产品是因，品牌是果”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">他是一向看重产品创新的。2004年，他带领步步高通讯公司从电话机市场转入手机市场，靠主打音乐的功能手机打开市场。2011年，他带领新成立的vivo进入智能手机市场，在产品上屡次创新：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/17586926862b27a4db9b7b2.78370547.png" title="vivo沈炜的这50条思考，读懂中国最低调企业家的独特" alt="vivo沈炜的这50条思考，读懂中国最低调企业家的独特" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2012年推出当时世界上最薄的手机（机身6.55mm）、又是第一款整合Hi-Fi级专业芯片的智能手机vivo X1；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2018年推出全球第一款屏幕指纹手机、第一款真正全面屏NEX手机，开启全面屏手机风潮；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2020年推出第一台微云台防抖的5G手机X50系列；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年推出了高端手机X60、X70系列，搭载自研的影像芯片V1以及跟德国百年光学巨头、专业摄像公司蔡司合作的镜头，在手机影像上实现“人无我有”的差异化竞争优势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2022年又一股脑发布了X Note、X Flod与X80三个高端系列，成为市场上最具产品力的智能手机品牌。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">产品力的提升，带来了vivo品牌力的提升。2020年，vivo达成了5G手机在1.5-6千价位段的全覆盖，而且实现了高端市场的破局。因为高端旗舰机vivo X60、X70系列的热售，vivo在4000元以上高端手机市场的份额大涨。Counterpoint数据显示，2021年全年，vivo在中国市场4000-5000元的价位段以24.1%的市占率位居第一。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第二，始终坚守利他共赢。与他人合作时不赚他人便宜，敢于吃亏。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最能体现这点的是vivo非常“朴素”的使命：让企业利益相关者都持续happy。沈炜自己经常说，作为CEO，最重要的责任是随时协调好4个方面的关系：为消费者提供优质的产品和服务，为员工创造和谐尊重的氛围，为合作伙伴提供合理互利的商业平台，为股东提供回报。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">沈炜经营vivo时始终坚持“利他共赢”这一原则。vivo在与供应商、代理商等合作伙伴合作时采取的方法是优先让合作伙伴赚钱，最后是vivo赚钱；vivo实行的是覆盖从基层到高层的利益分享机制，而且管理层是vivo最大股东。在沈炜看来，公司创造价值是为更多员工等利益相关者都持续happy所服务的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第三，践行“设计驱动和用户导向”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">vivo很少陷入与竞争对手比拼技术参数的PK中。在沈炜看来，技术永远在进步，参数永远在变化，不变的是人们恒定不变的追求，比如对美的追求、对愉悦的追求、对自我意义的追求，这些更深层、潜在的用户追求才是更需要关注的长期赛道。这就需要“设计驱动”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">沈炜解释说，“设计驱动不是设计师驱动，设计驱动最重要的体现是，我们要去思考产品的意义，思考我们能创造什么，改变什么，赋予什么，然后抛开杂念，用极致、简单、完美和优雅的方式去实现，为用户为社会创造价值”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">历史上已经有过设计驱动的伟大先例。福特汽车创始人亨利·福特说过，“如果我当年去问顾客他们想要什么，他们肯定会告诉我‘一匹更快的马车’”。但福特洞察到人们的潜在需求，是速度更快更便捷的交通工具。在iPhone推出之前，智能手机所需要的技术都已经出现了，诺基亚甚至已经生产出了智能化的功能型手机，但只有乔布斯洞察到人们其实需要的是类似微型电脑、操作极简的智能手机。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">vivo最近两年能突破高端手机市场，其实就是设计驱动和用户导向的结果。高端手机市场一向是靠研发技术、创新能力来竞争的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2018年，沈炜认为智能手机发展的下一个趋势，是作为人与万物互联的智能终端存在。同年，vivo发布了未来10年的企业中期战略：成为以智能终端和智慧服务为核心的全球领先平台型科技公司。自此，vivo开始加大对研发技术的投入。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2019年开始，vivo选择了设计、影像、系统和性能四大“长赛道”，进行长期、持续的研发投入。vivo在深圳、东莞、南京、北京、杭州、上海、西安、台北、日本东京以及美国圣地亚哥等10个城市设立了研发中心，范围包括5G通信、人工智能、工业设计、影像技术等众多个人消费电子产品和服务的前沿领域。目前，vivo 70%的职员从事研发类工作，整体研发人员逾万人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2021年，vivo推出了自研的专业影像芯片V1芯片，高性能、低功耗，最大程度提升了手机的影像性能。vivo X70 Pro系列在V1芯片的加持下，夜间视频拍摄的表现非常出色，这受到了很多消费者的欢迎，让vivo在4000+价位段高端手机市场取得了份额排名前三的成绩。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">回头看，沈炜创业27年，带领vivo在今天实现逆市增长、突破高端市场，是始终坚守初心、“埋头种因”得来的果。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本文梳理了沈炜在战略选择、企业经营与管理、价值观与文化等维度的50条商业思考，以帮砺石读者更直接、全面地了解这位低调企业家与众不同。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/102779796562b27a4e04adc8.44958057.jpeg" title="vivo沈炜的这50条思考，读懂中国最低调企业家的独特" alt="vivo沈炜的这50条思考，读懂中国最低调企业家的独特" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1.手机是个人事物的重要组成部分，每个人有每个人的喜好，没有一个品牌能一统天下，找到自己的位置，找到适合自己的细分市场，日子一样过得很好。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2.很多大企业，不是对手灭了它，而是自己犯了太多关键性错误。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3.企业不是要做多大规模份额，而建立个性鲜明的伟大品牌，一定能够是与众不同的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4.中国手机市场非常open，vivo追求的是一流的能力而非规模。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">5.一个品牌在完全市场化竞争中生存下来，才会真正健康。(2014年接受媒体专访，谈及vivo决定少参与运营商集采与定制的原因）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">6.上下游副业的投资完全是建立在主业高速发展上，一旦主业失速，整个产业就崩塌了。所以我特别反对上下游垂直整合。手机行业表面上是做制造业，本质上是做消费品。而我最懂的只有消费品，所以不会做别的。(2014年接受媒体专访）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">7.vivo的全球化战略是“More Local，More Global”，尊重本地市场、尊重本地文化、尊重本地人才，只有这样，本地消费者才有可能眷顾你。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">8.人工智能技术和手机结合，会改变用户原来的使用习惯，给用户更强大的功能和更贴心的体验。手机厂商也将从传统的硬件生产者，转变为“硬件+软件+服务”的创新性科技公司。（2018年博鳌亚洲论坛“未来的通信”分论坛上的演讲）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">9.未来手机的形态也许会发生变化，甚至名称都会改变，但是作为人与外界联接的一个核心终端，它所承载的作用是无可替代的。当然，随着需求和技术升级，它需要承载更多的功能和使命。首先，对于数字信息的处理将需要手机有强大的人工智能能力；其次，万物互联需要手机能够连接并控制各场景的设备，甚至需要通过手机内置的AI模块对一些场景进行高效运算，根据用户的使用场景，实现多设备联动。（2018年博鳌亚洲论坛“未来的通信”分论坛上的演讲）</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">10.（跟手机行业的老板交流一事）这种事情，聊多了是干扰，没必要。（2014年接受采访）</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">11.我们所有的原点都会回到消费者层面来思考问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">12.对于vivo而言，同行们都是学习的对象，通过学习其他品牌的优点，能弥补自己的短板。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">13.vivo短时间内没有任何上市的想法，也不需要融资，大部分上市是想弄点钱，但是我们并不缺钱。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">14.董事会每年也会大概说一下明年的计划，但是说了就说了，从来不考核。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">15.vivo也会定一些目标，但更注重的是“因”，而非“果”。这样做，往往会发现结果超出预期。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">16.（区域渠道主管）你们其实就是我们价值观的布道者。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">17.做得好的区域，往往是价值观好，而做得不好的，就是价值观没跟上去。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">18.想挣快钱的人，vivo绝不是他的好归宿。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">19.企业发展要以“平常心”来对待，凡事回到事物本源来思考，尤其是要付出代价的时候。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">20.企业文化就是水桶的桶底。所谓战略、技术无非是围着的木板而已。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/149149764762b27a4e32efd6.97738583.jpeg" title="vivo沈炜的这50条思考，读懂中国最低调企业家的独特" alt="vivo沈炜的这50条思考，读懂中国最低调企业家的独特" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">21.（2018年年会演讲）我在各种场合都说，我们不能过度关注大环境，因为我们没有其他的路可走。只有一条路，勇往直前，迎难而上，没有回头路。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">22.要有超乎利润之上的追求，即vivo人的3个梦想：要有世界一流的文化，产品与品牌。文化是前提，产品是因，品牌是果。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">23.坚持让企业利益相关者都持续happy的使命不变，只要这样，我们企业才能健康长久、基业长青！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">24.让消费者惊喜、热爱、追随的科技创新与时尚引领的极致产品，是一切之根本，是维持我们庞大用户平台持续净增长的首要前提，是移动互联网流量经营与智慧服务的坚实保障，而不要本末倒置！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">25.（2018年公司团宴的演讲）以我们企业成立22年的历程来看，困难危机往往是转危为机的最好时机！我们这个行业尽管萎缩，但仍有着巨大的市场容量，只要坚守本分与平常心，就能更高概率地少犯错误、多做正确的事情和把事情做正确！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">26.在海外市场，我们要坚定走自己商业模式的路。提高对市场的认知，不急不躁，稳中求进，先以能持续健康地生存下来为目标。中国的成功经验固然有可借鉴之处，但绝不能简单照搬到海外市场，必须要回到本地的国情，回到当地的竞争状况，回到事物的本源去寻找突破口。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">27.我们要懂用户的情感，看见人的喜怒哀乐，技术的终点是让人抵达幸福，我们要成为离用户最近的品牌，不仅要拉近物理距离，心也要更近。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">28.人有理想，企业也有自己的理想。如同人活着不是为了吃饭，企业活着不光是为了赚钱。赚钱是前提，但绝对不是追求的目标，否则就是唯利是图。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">29.我会把企业当人，用拟人化去思考：快乐，活得健康、长久。所以，我们企业的愿景是健康长久地发展。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">30.企业健康长久，良性运转的前提无疑是做事要符合自身条件，符合自己的目标，而不是随波逐流。手头上有一个亿，就做一个亿能干的事情，如果有1个亿就去干100亿的事情，那就是自不量力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">31.一个企业做得好不好，最重要的是方方面面都要兼顾，就像一个木桶，真正装多少水，是最短的板决定。而vivo这个“木桶”的木板，主要是四块：一是消费者，二是合作伙伴，三是员工，四是股东。这四个利益分配环节都很重要。作为CEO，我的工作就是随时协调好这四方面的关系，要他们开心快乐。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">32.我们坚持让企业利益相关者都持续happy的使命不变，只有这样，我们企业才能健康长久、基业长青！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">33.企业最重要的是要做正确的事情。至于如何把事情做好，则是能力、方法的问题。我们要一如既往地坚持凡事回归事物本源，抓住事物本质，保证做正确的事，并力求把事情做正确的工作原则。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">34.（2015年，步步高公司成立20周年讲话）持续提升我们的品牌，产品，服务在消费者心中的感知价值，包括提升品牌价值，提供真正能解决消费者痛点需求和极致体验的伟大产品与服务，是我们大团队工作唯一的追求与本质，是我们20年坚持至今从未改变过的商业逻辑，它源于我们对企业追求什么理想，为谁而存在等这些最本源的思考，非常富有逻辑推理地将我们企业文化和商业逻辑很好地结合到一起。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">35.（2017年接受采访）我们经常在内部说，如果一个行业需要争分夺秒、披星戴月、只争朝夕与时间赛跑，这个行业肯定是不可为的，也不是我们喜欢做的。我们是什么？我们就是有韧劲，用足够的时间，我们就能够做成我们想要做到的事情。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">36.从逻辑上、理论上已经论证、推导出打价格战是死路一条。反过来，健康长久的企业，活到一百年以上的企业，有没有靠打价格战活到今天的？没有！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">37.（2022年新春致辞）无论任何时候，竞争导向、成本导向、参数导向、效率导向等一定不能凌驾于用户导向之上。抛弃用户的企业终将被用户抛弃，无法健康长久。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">38.（2022年新春致辞）用户永远只记得住创造第一的人，而记不住跟随之人。我们要在手机这个科技属性极强的行业里，形成“持续主动引领，绝不被动跟随”的vivo style！</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">39.我们通信团队在第一个10年，某种意义上讲，更多是骄傲自满，夜郎自大，不思进取，坐井观天，人只有经历挫折，经历低谷，才会深刻反思文化与价值观及理念所起的巨大作用，毕竟思想决定行动！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">40.本分是首先明辨是非对错、毫无利益导向的坚守；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本分是凡事追求长远而不只争朝夕的态度；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本分是所思所言所行高度一致的践行；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本分是坚持独立深层次思考、毫不盲从大流的清醒；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本分是不为果、只埋头种因的执着；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本分是清楚自我能力而时时必须聚焦的减法；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本分是能够将复杂的事情简单化的修炼；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本分是有问题时首先求责于己的态度；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本分是既能共苦更能同甘的利益分享。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">41.（2015年，步步高公司成立20周年讲话）以本分为核心的企业文化与价值观，我认为是我们企业20年仍能存活至今的最重要原因，是我们企业今天能够与世界一流企业PK的唯一的核心竞争力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">42.永远保持平常心，凡事回归事物的本源思考，排除压力、痛苦、诱惑等一切干扰，尤其是在要付出代价的时候。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">43.要想永远更高概率地做正确的事和把事情做正确，本分、平常心既提供了正确的方向（即什么是正确的事），也提供了解决的方法（即如何把事情做正确）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">44.（2018年公司团宴的演讲）一如既往地坚持本分，坚持凡事回归事物本源，抓住事物本质，坚持做正确的事，并力求把事情做正确。果不能求亦无须求，因决定果，埋头种因，果自会水到渠成！</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">45.很多企业很在乎“果”，我觉得vivo是一个不太在乎“果”的一个企业。“果”对于企业来说固然很重要，但这并不意味着它是企业最重要的东西。“因”决定了“果”，如果没有把“因”做好，是无法得到“果”。“果”是不可求的，也求不到。何为因？因就是做正确的事和力求把事情做正确。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">46.埋头种因的体现是什么，在当前阶段，我想着重强调三点：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首先，埋头种因，要凡事坚持长期主义。只有坚持长期主义，才能更好地保持平常心，不受短期压力、诱惑的干扰，才更有可能去做正确的事，并把事情做正确。如果一个人总是忙着去做短期的事，看中短期的结果，而忽略长期重要的事，就会很容易出问题，也必然走不长远。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其次，埋头种因，要始终坚守利他共赢。我们常说本分是与他人合作时不赚他人便宜，敢于吃亏的态度。敢于吃亏，是利他共赢的第一步。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最后，埋头种因，要全面践行“设计驱动和用户导向”。技术永远在进步，参数永远在变化，不变的唯有人们对美的追求，对成就和愉悦的追求，对自我意义的追求。人们对某些恒定不变的追求，反而是我们更需要关注的长期赛道。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">47.长期主义不仅仅是一种方法论，更要成为每个人的思维惯性，清醒地面对环境的不确定性，着眼长远去思考，提升化繁为简的能力，从使命愿景价值观中去探求更多答案。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">48.胸怀利他之心，坚定不移地构建互信共赢的统一战线，追求众乐、反对独乐。如果凡事我们首先想到的是自己的利益，而未很好地顾及他人的利益他人的战略关切，那我们就一定无法健康长久。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">49.我们不应拘泥于比拼参数，本质上，科技不是为了让人们更好地享受科技，而是更好地享受生活。我们需要倾听用户的声音，懂他们的需求，以用户导向和设计驱动的价值观来为他们创造心动的产品和服务，提升用户满意度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">50.设计驱动不是设计师驱动，设计驱动的底层是价值观的驱动。设计驱动最重要的体现就是我们要去思考产品的意义，思考我们能创造什么，改变什么，赋予什么，然后抛开杂念，用极致、简单、完美和优雅的方式去实现，为用户为社会创造价值。</p>
