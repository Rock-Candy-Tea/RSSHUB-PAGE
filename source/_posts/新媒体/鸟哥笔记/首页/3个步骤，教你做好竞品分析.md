                        <p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">提到竞品分析，很多人会认为那是产品经理、分析师，甚至公司高层才会做的事。其实不然！</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">不仅是制定产品策略时才会做，实际工作中，例如像公众号运营、短视频运营、设计师等等，都需要做竞品分析。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">正所谓知己知彼百战不殆，通过竞品分析可以快速了解对手产品和市场动态，更重要的是<span style="font-weight: bold;">可以取长补短，避免踩坑。</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以无论任何岗位，掌握正确的竞品分析思维，都受益匪浅。下面为大家总结做好竞品分析的3个步骤。</p>
<h1 id="toc-1" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; margin: 25px 0px 20px; line-height: 30px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-size: 16px 20px; font-weight: 600; text-align: center; background-position: 0px 5px; background-repeat: no-repeat;">01、如何选择竞品？</h1>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">做好竞品分析的第一步自然是要选择好竞品，如果竞品选不对，花再多功夫都是徒劳。那么该如何快速找到竞品呢？通常有以下几种方式：</p>
<section>
<section>
<p style="margin: 0px 0px 10px;">1）通过搜索引擎寻找竞品</p>
<p style="margin: 0px 0px 10px;">2）在应用市场、专业网站、行业调查报告中寻找竞品</p>
<p style="margin: 0px 0px 10px;">3）通过调研收集竞品</p>
<p style="margin: 0px 0px 10px;">4）电商平台搜索相关关键词寻找竞品</p>
</section>
</section>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">通过以上方式在列举出一部分竞品范围后，<span style="font-weight: bold;">我们还需要对竞品进行分类，这样才能精准分析，有效产出。</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">所以要求我们要对自身产品有非常明确的了解和认知，包括：所处赛道、目标人群、核心功能等。然后将找到的竞品分为三类：</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/09/ucSKxyZMwYGFB2rmwNjB.png" alt width="1080" height="402" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">第一类：直接竞品</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">直接竞品是分析的重点，需要定期关注。这类竞品所处行业/赛道、目标用户、产品满足需求、产品核心功能都高度相似，通常找行业内排名TOP1-5即可。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">例如：苹果手机和华为手机。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">第二类：间接竞品</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">间接竞品选择行业内排名靠前的品牌，定期关注即可。这类竞品要么目标用户群体相似，但产品满足的需求不同；要么产品相似，但目标群体不同。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">例如：淘宝和得物；农夫山泉和可口可乐。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">第三类：潜在竞品</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这类竞品通常是现在没有明显的竞争关系，但是未来有可能形成竞争的产品。我们可以选择排名靠前的，定期进行简单梳理即可。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">我们在初步筛选出优质竞品后，就可以着手于具体的分析了。</p>
<h1 id="toc-2" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; margin: 25px 0px 20px; line-height: 30px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-size: 16px 20px; font-weight: 600; text-align: center; background-position: 0px 5px; background-repeat: no-repeat;">02、竞品分析常用的方法和模型</h1>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于不经常做竞品分析的人来说，往往不知道从哪里下手。下面推荐5种<a href="https://www.niaogebiji.com/tag/%E7%AB%9E%E5%93%81%E5%88%86%E6%9E%90%E6%96%B9%E6%B3%95" target="_blank">竞品分析方法</a>，可以根据分析目的和场景择优选择。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">1、表格比较法</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">适用场景：用户体验评估、功能分析</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">常用的一种分析方法，通过这种方法<span style="font-weight: bold;">可以把不同竞品间的功能、配置、特性方面直观的比较出来</span>，帮助深入了解竞品，并分析出自身的优势与劣势。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">衍生的还有打勾比较法、评分比较法等。例如打勾比较法多用于了解与竞品之间的功能差异，为自己的产品是否加入这个功能规范做参考依据。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 201px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/09/75umwNAcgVoqHe0r4vvp.png" alt width="621" height="349" data-action="zoom" referrerpolicy="no-referrer"></p>
<h4 style="margin: 25px 0px; caret-color: rgba(0, 0, 0, 0.84); color: #818386; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 14px; text-align: center; font-weight: 300; line-height: 30px;">△打勾比较法</h4>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">评分比较法多用于体验设计中，把竞品的数据依次排列，可以直观看到与竞品之间数据的差异。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 217px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/09/URpdTN0i1FzWTpYTpdmp.png" alt width="675" height="347" data-action="zoom" referrerpolicy="no-referrer"></p>
<h4 style="margin: 25px 0px; caret-color: rgba(0, 0, 0, 0.84); color: #818386; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; text-align: center; background-image: none; font-size: 14px; font-weight: 300; line-height: 30px;">△评分比较法</h4>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">2、矩阵分析法</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">适用场景：产品定位、市场机会</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">矩阵分析法是<span style="font-weight: bold;">通过分析自己的产品与竞品的定位、功能、特点、问题等，明确自身定位及优势，帮助我们现有产品找到合理定位，梳理市场中存在的空白点。</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 343px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/09/YyCLuJZpFauYvKXVuclu.png" alt width="636" height="583" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这种方法需要确定2个关键竞争要素，分别做XY轴，然后把竞品放入对应的位置，再思考自身产品的定位。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">例如上图中，对视频软件进行分析，选取了活波/严肃，尊贵/亲民两个方面的维度。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">在严肃和亲民这个范围领域就是没有这个产品，企业可以考虑是否可以在战略层面上投入开发成本，开发这个空白领域的产品，抢占市场。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">3、SWOT分析法</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">适用场景：企业战略分析、竞争对手分析</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">SWOT分析法也是竞品分析中常用的方法，主要是<span style="font-weight: bold;">通过分析企业内部和外部存在的优势、劣势、机会和挑战来概括企业内外部研究结果的一种方法。</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 268px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/09/pX7G4fU7ilrFY6NXMUtH.png" alt width="593" height="488" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">简单来说就是通过4个维度进行分析，得出结论，而这个结论带有一定决策性和预测性。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">在进行分析时，可以在纸上划出4个区域，将与公司有关的优势、劣势、机会与威胁写下来。我列举了一些建立SWOT表时需要考虑的因素，以供参考（如下图）。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 509px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/09/QwkWYlySyaFUHVpJOuDQ.png" alt width="675" height="806" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">列举了所有因素之后，我们可以再深入进行分析，帮助企业把资源聚焦在强项和机会最多的地方，并让企业的整体战略变得清晰。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">4、PEST分析法</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">适用场景：预测市场、宏观环境分析</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">PEST 分析法是对宏观环境进行分析，以便找到机会，认清威胁与挑战。所谓PEST，P是政治(politics)，E是经济(economy)，S是社会(society)，T是技术(technology)。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: 322px; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/09/dXgsiyFU9drA3XkIJA2U.png" alt width="547" height="500" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">在使用这种方法时，可以将4个要素分别写在4个格子格子中，尽可能多地罗列关键词填写入其中，最后投票选出得票较高的 3～5 个点子作为 PEST 分析的结果。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">5、功能拆解法</span></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">适用场景：预估开发周期及成本、全面了解竞品功能</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">功能拆解就是用碎片化方法对竞品功能进行拆解，把竞品分解成一级功能、二级功能、三级功能甚至四级功能。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img class="aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" src="https://image.woshipm.com/wp-files/2022/09/Q7e8s5BXbUUWQykDOWxI.png" alt width="1080" height="670" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">主要对竞品进行体验，获取竞争产品的功能，记录完整的使用过程，并对各项功能的具体内容进行备注说明，以便更全面地了解竞品的构成，避免遗漏。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">当然，除了上面几种分析方法之外，还有很多方法和模型。我们也不必拘泥于方法和形式，根据自己的分析目的，选择最适合的即可。</p>
<h1 id="toc-3" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; margin: 25px 0px 20px; line-height: 30px; caret-color: rgba(0, 0, 0, 0.84); color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-size: 16px 20px; font-weight: 600; text-align: center; background-position: 0px 5px; background-repeat: no-repeat;">03、如何写好竞品分析报告？</h1>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">最后一定不要忘了总结，竞品分析最终目的是通过分析得出结论。在总结之前要想清楚两个问题：</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">第一，分析的目的是什么？</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">● 如果分析是为了个人的提升，可以只注重某个细节的深耕，要能体现个人对产品的独到、专业的见解。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">● 如果分析是为了开拓市场，制定竞争策略，那要注意多维度横向的对比，尽可能查漏补缺，取长补短。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">第二， 写给谁看？</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">● 如果报告是给产品团队看，根据分析目的和已有思路做全做细即可。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">● 如果报告是给运营团队看，重点放在亮点分析的部分，再根据公司实际运营情况做结合。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">● 如果报告是给管理者看，要在发展、定位等内容上进行深化，尤其是下一步方向和策略要下足笔墨。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">然后到正式写报告的环节，一般采用<span style="font-weight: bold;">“总-分-总”的结构形式</span>，文件格式根据自身需求，word、excel、ppt皆可。</p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总述部分包括<span style="font-weight: bold;">摘要、</span><span style="font-weight: bold;">分析背景、目的、目标、报告目录等；</span></p>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">分述包括<span style="font-weight: bold;">分析维度、每个分析维度的具体分析过程与小结</span>；</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">总结包括<span style="font-weight: bold;">结论、建议、下一步计划、附录等</span>。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">需要注意的是，报告并不是简单的罗列数据或是优缺点对比，它需要体现你的逻辑，还能给出可行性方法论。</p>
<h1 id="toc-4" style="font-size: 20px; background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); padding-left: 25px; margin: 25px 0px 20px; line-height: 30px; background-size: 16px 20px; color: #212629; font-weight: 600; text-align: center; background-position: 0px 5px; background-repeat: no-repeat no-repeat;">写在最后</h1>
<p style="caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以上就是做好竞品分析的3个步骤，做个总结：</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">第一步，收集竞品资料并做好分类，确定要分析的竞品；</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">第二步，选取合适的方法论或模型，进行竞品分析；</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">第三步，根据目的输出分析报告。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">建议要长期做好竞品分析这件事，因为任何分析都不可能是一次性的，整个市场都在不断变化和迭代。持续地分析，才能更好地掌握市场，发现规律。</p>
<p style="margin: 25px 0px 20px; caret-color: rgba(0, 0, 0, 0.84); color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">最后，分享迈克尔·波特的一句话：“如果你所做的一切都与你的竞争对手本质上是一样的,那么你就不太可能成功。”</p>                        <div style="height: 1px"></div>
