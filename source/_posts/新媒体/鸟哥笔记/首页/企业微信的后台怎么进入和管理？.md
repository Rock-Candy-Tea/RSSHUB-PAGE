                        <p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">企业微信管理后台，只有企业的管理员才可以进企业微信后台，普通员工想要进入后台、可以联系管理员将你设置为后台管理员。</p>
<h2 style="border: 0px; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; font-size: 18px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #212629; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; line-height: 30px; text-align: left;">一、怎么进入企业微信后台</h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">管理员进入企业微信后台有两种路径；</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><strong>路径一：</strong></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">企业管理员直接在浏览器搜索企业微信网页版，点击【企业登录】-【扫二维码登录】就可进入企业微信管理后台。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><strong>路径二：</strong></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">电脑端下载企业微信APP，然后打开APP并登录,点击下方的【登录管理后台】即可进入。</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">用这两种路径中的任何一种都可进入企业微信后台，进入后我们就可以看到后台的所有管理功能。</p>
<h2 style="border: 0px; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; font-size: 18px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #212629; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; font-weight: 600; line-height: 30px; text-align: left;">二、管理企业微信的后台</h2>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">企业微信后台的功能很多，考勤数据和通讯录是企业常用的基本功能，</p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><strong>1、考勤数据管理</strong></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">企业微信具有的打卡功能对于企业管理来说是非常至关重要的一点，企业员工上下班可以在企业微信打卡。管理员通过后台就可查看员工上下班情况。同时后台也能导出员工考勤数据，方便企业对员工考勤管理和后期工资结算等业务。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p2.itc.cn/images01/20220727/fc932cd5cbea47c599548c798a45f5bb.jpeg" referrerpolicy="no-referrer"></p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p9.itc.cn/images01/20220727/9027f98e0a6c497b96793704b2335b82.png" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">对于员工来说，在企业微信端只能查看自己的考勤记录，无导出权限。在工作台界面找到打卡，在统计页面就可以查看当前的考勤记录了。通过右上角【月报】可以查看更详细的考勤分析，包括加班统计、假勤统计等。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p7.itc.cn/images01/20220727/0c79922dced44ba1b1d70f6d3f533853.jpeg" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><strong>2、通讯录管理</strong></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">管理员可在企业微信后台添加企业成员和部门，还可以批量导入/导出以及删除成员。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p9.itc.cn/images01/20220727/c97b296a996e42a5b663f67acf9ce38b.png" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">对于通讯录后台管理员也可自己设置规则，管理员可以添加隐藏的部门/成员、限制查看外部门、成员资料展示、限制查看所有人等。</p>
<p class="ql-align-center" style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; text-align: center; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;"><img style="border: 0px; margin: 10px auto 0px; padding: 0px; display: block; max-width: 100%; height: auto;" src="https://p6.itc.cn/images01/20220727/efdca9afcf33488aaa272d055ca4f767.png" referrerpolicy="no-referrer"></p>
<p style="border: 0px; margin: 0.63em 0px 1.8em; padding: 0px; font-size: 16px; counter-reset: list-1 0 list-2 0 list-3 0 list-4 0 list-5 0 list-6 0 list-7 0 list-8 0 list-9 0; color: #191919; font-family: 'PingFang SC', Arial, 微软雅黑, 宋体, simsun, sans-serif; background-color: #ffffff;">企业微信的通讯录功能，对于企业实现组织化管理沟通是极其有帮助的。当然企业微信后台管理除了考勤和通讯录之外，管理员还有很多功能可以设置管理。比如自动发送欢迎语、创建群活码、会话存档等。</p>                        <div style="height: 1px"></div>
