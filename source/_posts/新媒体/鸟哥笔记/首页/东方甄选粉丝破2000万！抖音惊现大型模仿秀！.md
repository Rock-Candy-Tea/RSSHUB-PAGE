                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">6月的“最大黑马”，非新东方在线旗下的<a href="https://mp.weixin.qq.com/s?__biz=MzU5NjMxODA5OQ==&mid=2247491134&idx=1&sn=a4d8314b2f3f3fb17bbf69acb05c0994&chksm=fe65dcc2c91255d454d178b059d39c9166b484addf5839555be68908c5bb9c5f5efc2dbf6de7&token=1568300214&lang=zh_CN" data-miniprogram-appid="wx486ef245f10a196f" data-miniprogram-path="pages/index/index" data-miniprogram-nickname="壁虎看看抖数" data-miniprogram-type="text" data-miniprogram-servicetype>@东方甄选</a> 莫属了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">过去两周，@东方甄选 的粉丝量增长超20倍，截至6月30日，该账号粉丝总量突破2000万大关，已超过罗永浩的@交个朋友直播间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而单日GMV也从数十万升至千万量级，跻身<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%9B%B4%E6%92%AD" target="_blank">抖音直播</a>带货头部直播间。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其热门主播-董宇辉，也成为媒体和品牌的追逐对象。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/3a424e1fd3042af04212dce59b05cb84.png" alt width="1125" height="687" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/7ca15681eb989df70b26057e0e5275ec.png" alt width="1089" height="661" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而在东方甄选爆火出圈后，效仿者也是纷纷入局。从汪小菲的中法双语带货，到豆神教育推出新账号“豆神甄选”，再到卫龙辣条、螺霸王等食品品牌的“董宇辉体”产品讲解。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就连抖音平台也通过设置话题#假如我是双语带货主播，来吸引品牌和达人们模仿。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">然而，新东方这种直播模式是否具备可复制性？其中又需要注意什么？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">接下来，壁虎看看抖数将为你一一讲解，一起来看看吧。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">东方甄选爆火</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">抖音上演模仿秀</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">自东方甄选双语带货爆火后，汪小菲在@麻六记 也开启了中法双语<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>，然而其有限的词汇和跑偏的发音，并未获得多数观众的认可。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">东方甄选作为教培行业转型的示范性案例，自然也引来了其他教培机构的模仿，被称为“语文学科培训第一股”的豆神教育成为首个下场选手。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">自去年10月起，豆神教育的董事长窦昕就带领着核心成员在抖音发布短视频分享文学文史知识，同时也孵化了大批账号，但是豆神系直播间人数并不多。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如拥有123.5万粉丝的 @窦昕@豆神天蝎星 直播间，近7天最高一场直播销售额为8.72万，人气峰值44人，直播时长共计12小时。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> <img src="https://qnssl.niaogebiji.com/b08d0817aad99cc2786b1485aacc3aa0.png" alt width="1361" height="496" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为了扩大自身影响力，6月18日，豆神教育推出新账号“豆神甄选”（现改名“豆选生活”）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">仅一周时间，该账号就已吸粉50万。目前该账号只发布了8条短视频，并未开启直播，短视频内容多来自于旗下其他账号的直播内容片段。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/a43d48cfd6230359159deccc6ce21afa.png" alt width="1125" height="2436" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另外还有“董宇辉体”的产品讲解，例如卫龙辣条的“吃一口卫龙辣条，我相信你会回忆起童年，回忆起放学回家放下书包和小伙伴们出去玩”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以及螺霸王的“你吃的不是螺蛳粉，吃的是那年盛夏微风拂过脸颊，骑着自行车的你，和他说好青春不散，永远都不说分开的话”。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/04a4161508ea02d80615e3a7eb708da2.png" alt referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">然而爆火的东方甄选并不好模仿，这背后不仅考验主播们的文化沉淀和口才演讲能力，也考验模仿者的幸运度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">与传统带货主播不一样的是，东方甄选的主播们都曾是在新东方教培体系中经过多年锤炼后的名师，他们在直播过程中会自觉或不自觉地将知识引入带货，进而打造了一个东方甄选特有的直播风格。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/94089292d720425c637052b616800cc5.png" alt referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">像这种“知识带货”的直播模式非常依赖主播的个人特质，例如口才、知识积累、艺术素养等等，也只有具备这些优秀素质的主播，才能在“知识传播+直播带货”的模式中发挥更好的水平。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">另一方面，多样、持久的直播内容也是关键。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">用户的注意力是多元的、多变的，因此东方甄选推出主播团队，以此来保持用户对直播间的兴趣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果东方甄选的直播团队可以长期输出多样性的内容，让用户保持新鲜感和收获感，那这种带货模式也是可持续的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/889fc8656ee7de8cbb046364820a2e19.png" alt width="740" height="704" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除此之外，东方甄选的爆火也离不开抖音的支持。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">东方甄选出圈之际，正值几大直播电商平台缺少“头部主播”的空窗期，薇娅消失、李佳琦暂停，罗永浩也退出了直播带货，此时抖音坐拥庞大的流量池，正需要新主播来承接流量，获取增长动力，@东方甄选 因此也是受到了官方的大力支持。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">据壁虎看看抖音版数据表明，@东方甄选 直播间的“推荐流量”和“付费流量”占比在升高，6月1日仅占49%，而在6月10日，也就是爆火的第一场直播，这一占比提高至64%。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/ec6ecdc8e04d45830fd347dd0622d829.png" alt width="515" height="273" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">壁虎看看抖音版6月1日数据</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/ff4d70de478038c72282b7e4af1d4260.png" alt width="507" height="278" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">壁虎看看抖音版6月10日数据</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对东方甄选来说，这既是机会也是挑战。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">东方甄选热度未去</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但隐忧渐现</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在“618”电商大促的辉煌后，@东方甄选 也迎来了流量巅峰之后的下滑。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">数据显示，@东方甄选 6月16日达到顶峰，一场直播销售额达到6423.71万，随着热度下降，到6月29日，直播间销售额已降至2787.78万，观看人数也从高峰时期的5998.74万人减少到3789.26万人。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">主播董宇辉在接受采访时表示，“自己只是卖菜的人”，未来热度肯定会降下来，这很正常。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/18d92dc2b4cddd483dee182c47d86995.png" alt width="1080" height="467" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">正所谓“铁打的抖音，流水的头部”。爆红之后陷入瓶颈期，已经成为困扰抖音顶级大V的普遍难题；即使是万众宠爱的@东方甄选，同样逃不脱这一宿命。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">伴随着东方甄选热度下滑，其售卖的农产品也遭遇了“翻车”，“桃子发霉长毛”和“6元一根玉米”事件，这也逐渐暴露出东方甄选的品控问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/4889964be62bdeef1e0d0c4a2de565f8.png" alt referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">根据@东方甄选 近7天（6月23-29日）的数据显示，直播间商品品类主要为食品饮料和生鲜果蔬。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">像生鲜果蔬这类农产品具有非标品、生产周期长、保质期短、运输成本高等特征，品牌要想做到标准化的品控会更难。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一旦产品品控和售后频频“翻车”，会对东方甄选的口碑造成很大的伤害。要知道，直播电商最终还是要回归到产品本身，产品质量不过关对于品牌的影响是致命的，会造成口碑下滑和用户流失。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，即使“翻车”事件发生后，新东方在线及时作出回应并退款给消费者，也依旧让用户对产品心存疑虑，最近几天流量的下降也证明了这一点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了要把品质把控好之外，产品供应链和性价比也是影响直播间可持续发展的关键。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">高效的供应链能够满足@东方甄选 暴增的流量，而高性价比的产品能够让更多用户持续下单，尤其是在热度褪去之后，提高复购率的关键还是要看更高的性价比。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而@东方甄选 面临的另一个挑战，则是内部多位当红主播出走的隐患。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">近期有消息传出，微信视频号团队正在尝试挖角@东方甄选 的多位当红主播。而俞敏洪也在一场直播中承认，目前确实有机构在挖董宇辉，一年待遇能给到上千万。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/a1774f423d45d26a1cd8519fdbb44d5a.png" alt width="1125" height="1049" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/7d578fb953a71fe27b1e7ff6e11fa7e4.png" alt width="1119" height="1232" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">若董宇辉等多位当红主播从@东方甄选 出走，这对新东方来说，是一个不小的打击。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如今，新东方已用5万月薪的高薪来吸引新的主播，为团队蓄力下一个“董宇辉”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">虽然@东方甄选 直播间仍然面临许多问题，但是这种“知识带货”新模式仍为我们带来了新思路，也让累于头部、增长困难的直播带货看到了新的落地场景和用户增量。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这或许是俞敏洪和东方甄选在半年多的摸索后，带给我们最重要的启发。</p>                        <div style="height: 1px"></div>
