<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作者 |</span><span> 君怀夜</span><span></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着数字经济和Web 3.0概念和应用的发展，元宇宙吸引了奢侈品、运动时尚等众多领域品牌的关注，LV、李宁、雅诗兰黛、法国娇兰等众多品牌纷纷尝鲜，打造自己的虚拟营销案例。 </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，彩瞳品牌moody也携手虚拟数字人AYAYI推出了首款元宇宙概念彩瞳“超瞳系列”。“超瞳系列”提取时空隧道、01代码、马赛克等元素大胆地运用在镜片设计上，共设计了“记忆象限、多维之镜、时光缝隙、空间缝隙”四颗元宇宙概念花纹，并搭配不同的故事，以视觉化的方式在数字世界诠释了元宇宙的概念。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/272090477627ec982b00e27.12096981.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><span>据moody介绍，“超瞳系列”灵感来源于moody宙宇领航员AYAYI。在去年双十一预售前夕，moody 官宣入驻天猫超级品牌日“元宇宙艺术展”，太空disco系列就率先联合数字人AYAYI推出限量产品“星空花语”成为天猫首批数字藏品进入 NFT 数字博物馆。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>超写实<a href="https://www.niaogebiji.com/tag/%E8%99%9A%E6%8B%9F%E4%BA%BA" target="_blank">虚拟人</a>走红小红书</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得注意的是，与moody的合作只是超现实数字人AYAYI业务版图的冰山一角，自从去年在小红书上走红之后，她就成为整个互联网圈炙手可热的虚拟明星，吸引了众多时尚品牌争相寻求合作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年5月，一个看起来和真人无异的形象AYAYI出现在小红书上。一夜之间，AYAYI发布的内容吸引了近4万粉丝，首发帖阅读量达到了近300万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/738759025627ec984f21a43.34326438.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><span>“这是AI还是真人化妆拍出来的？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“姐姐是真人吗？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这脸也太好看了吧！”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这双眼皮是割的吗？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“你这口红是什么型号啊？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“有同款男头吗？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大批用户被AYAYI吸引。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，AYAYI是燃麦科技推出的一个超写实数字人。AYAYI不同于一般虚拟人的地方就在于“超写实”：相比其他虚拟偶像，AYAYI有着更贴近真人的虚拟形象，无论是肤质、发质、微表情等都做到了对真人的高强度还原，并且可以依据不同光影条件作出相应的模拟和渲染，能让年轻人感受到更多接近真人的真实感。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>AYAYI虽然是一个数字虚拟人，但并不只是一个毫无感情的形象，相反，从人物形象、背景设定、故事集与世界观到行为举止，她的背后都有一整套内容支撑。同时，她不受时空天气、身体条件、技能等因素的限制，可以将很多人类的想象以更为创新大胆的形式，演绎到现实之中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>丰满的人设、逼真的形象和超越真人的表现力，让AYAYI一出场就吸粉无数。2021年5月，AYAYI首次亮相小红书，吸粉百万并吸引到娇兰与其合作，并受邀到娇兰夏日亲蜜花园活动进行打卡。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此后，AYAYI的商业化一发不可收拾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年6月，AYAYI在自己的账号发布了与陈伟霆的合照、7月受邀参加了LV“2021秋冬男装限时空间”活动、9月参与了北京环球影城线下活动并入职阿里，成为天猫超品日的数字主理人，更是在12月出现在了保时捷中国的微信朋友圈信息流广告中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而后，AYAYI还和BOSE合作，担任BOSE的首席消噪体验官；和保时捷联手拍摄时尚大片；和井柏然一起拍摄曼塔沃思杂志。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年3月，安慕希也选择与AYAYI进行跨次元合作，创造出一款依据用户大数据而产生的“能懂你的数字酸奶”。这款酸奶和目前正在盛行的 NFT相似，未来如果能够上链的话，也可能会成为真正的NFT酸奶。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1950057133627ec985e1bf16.87294660.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><span>显然，这个超写实数字虚拟人，在商业价值上已经成为了不输许多明星的顶流。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>互联网大厂们的虚拟人生意经</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，虚拟人的火爆不仅吸引了燃麦科技这样的专业数字资产公司，更引起了众多互联网大厂们的注意，阿里巴巴、腾讯、百度、字节跳动等企业都纷纷布局相关赛道。</span></p>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); font-size: 16px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>腾讯：游戏虚拟人“与民同乐”</span></strong></h3>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在刚刚过去的“五四”，腾讯与央视共同推出了《奋斗的青春——2022 年五四青年节特别节目》。本次节目有一个非常特殊的地方，就是央视携手腾讯打造了首个“数实融合虚拟音乐世界”的节目体验，观众可以以数字身份进入虚拟世界，共同观看五四歌曲节目展映、参与实时互动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>节目中，腾讯互娱旗下的吉莉、童和光、星瞳三位虚拟人（</span><span>吉莉，来自和平精英；童和光，腾讯NExT Studios工作室原创虚拟IP；星瞳，来自QQ炫舞</span><span>），也通过技术手段与央视舞台上和高校校园中的演员与学生们联合出演。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，有一个特殊的合唱环节，吉莉等其他虚拟人物就一同现身特别节目现场，携手青年演员、歌手及高校学子共同演唱了一首《New Youth》，通过虚拟人物与传统表演形式相结合的方式，展现了一个独具现代特色的表演。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1987267121627ec9886870d2.83439261.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据悉，腾讯在虚拟人生产制作全流程上，已经积累了不少自有知识产权专利，这些积累对虚拟人生产制作起到了极大的降本增效作用。而此次五四特别节目给到制作团队的时间只有一个月。正是因为此前丰富的虚拟人技术积累，这些内容才可以按时保质地制作出来。</span></p>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); font-size: 16px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>阿里：直播带货的<span>超写实数字人</span></span></strong></h3>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在此前的北京冬奥会期间，阿里巴巴的数字虚拟人“冬冬”凭借会讲段子、会跳舞也吸引了许多网友关注。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据介绍，冬冬是一位AI实时驱动的拟真人级别的写实数字人，同时具备真实的人设：她是一名会滑雪、快人快语的北京姑娘，性格飒爽，风趣幽默。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/556610175627ec988e02ba6.80169604.png" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><span>冬冬不仅是一个看起来好看的二次元形象，更是具备了极强的交互能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据阿里巴巴研究员、数字人技术负责人李小龙介绍，冬冬不仅是一个可以在演播室展现出采访能力的主持人和记者，也是一名可以在直播间进行控场和卖货的带货主播。冬冬的突破之处在于，实现了拟真人的实时交互能力，具备自我生成言语的能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我们希望冬冬能够具备一个有趣的灵魂，是一个风趣幽默的形象……我们研发AI生成段子。”李小龙表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，更早的2020年年底，阿里原创保护平台就上线了一个名为“塔玑”的项目：这个由阿里自主研发的虚拟模特项目，让现实中并不存在的虚拟人成为模特，在避免肖像权纠纷的前提下，免费开放给淘宝、天猫、1688以及阿里速卖通的服饰类商家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1140866186627ec989518945.43155424.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><span>而就在几天前，阿里妈妈官宣旗下全新养成系数字人「锘亚Noah」来了，他是首个被用户选择和养成出道的超写实数字偶像。粉丝们的互动行为将影响他的人设、穿衣风格、艺能方向等特性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从“塔玑”项目的虚拟人模特到冬奥宣推官“冬冬”以及「锘亚Noah」，可以发现，阿里的虚拟人正变得越来越人性化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1222282913627ec9899c4924.72276956.jpeg" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>北京冬奥期间，亮相的虚拟人不止“冬冬”，各大科技公司打造的虚拟人纷纷登场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，在《冰冰带你说冬奥》专属H5中亮相的央视主播王冰冰虚拟形象，由科大讯飞打造；为冰雪赛事提供了手语翻译解说服务的手语数智人“聆语”，由腾讯打造；央视总台首个AI手语主播，则由央视新闻联合百度智能云打造。</span></p>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); font-size: 16px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>百度：量产虚拟人或成为现实</span></strong></h3>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为国内很早就布局AI技术的企业，百度玩虚拟人的历史也可以追溯到几年前。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年的百度世界大会上，百度推出了具备认知能力、可机机对话20年前的“康辉”和“李彦宏”虚拟人首次亮相，两位虚拟人基于康辉和李彦宏20年前的形象、记忆和所拥有的知识打造，不仅形象、动作、声音逼近真人，还能流畅自如地与人聊天，甚至两个虚拟人可以相互对谈。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/887179692627ec98a08d9e5.41479326.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不同于燃麦科技专心打造虚拟人“明星”的玩法，百度的虚拟人产业作为AI技术落地的成果，正朝着“人海”的方向发展，试图凭借规模化优势占据市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年4月，百度集团副总裁吴甜近期做客央视《对话》节目时就提到，“百度希望通过领先的全栈AI能力，把虚拟人的生产周期缩短到小时级别，降低生产成本，并且用AI驱动生成内容。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，百度已打造了央视网虚拟主持人小C、航天局火星车数字人祝融号、百度App代言人龚俊、理财专员小浦等一系列虚拟数字人。互联网周刊发布《2021虚拟数字人企业排名TOP50》榜单，百度凭借央视虚拟主播、冬奥手语数字人主播，排名中国数字人产业综合实力第一位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1856285533627ec98a9dc417.89857554.png" width="333" referrerpolicy="no-referrer"></p>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); font-size: 16px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>B站：开启全民虚拟直播？</span></strong><span></span></h3>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>BAT之外，互联网新兴势力代表B站、字节跳动也屡屡在虚拟人领域投入重注。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在5月6日，全程说英文的日本虚拟人主播vox中国首播首秀营收破百万。一个外国数字虚拟人，凭什么在中国有这么强的号召力？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这位虚拟主播全称叫vox akuma，是在B站的邀请下，开启了这场中国的直播首秀。按照设定，vox是一个生活在400 年前的日本恶魔，擅长以声音来蛊惑人心。他曾有一座城堡，容纳约 500 名族人一起生活，后为德川的家族所灭。因为恶魔之血， 400 年后vox再次复活，希望通过网络了解现代社会并重组族群。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1454551632627ec98c734f23.30649302.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><span>听起来像一个俗套的东方版德古拉伯爵故事，但耐不住年轻人就是愿意为他掏腰包。vox直播的一个多小时里，就有超过5万人次和他互动，付费人数接近4万人次，营收达到了111万人民币，在当晚登上了实时热门的首位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>vox入驻背后，是B站对虚拟人赛道的不断加码。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在2016年，B站就参与了洛天依所属公司上海禾念的融资，开始布局虚拟偶像产业。2018年开始，B站在虚拟偶像产业上的发力开始进入加速期，次年就将洛天依收入到了平台的二次元矩阵中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年Q1内，B站共有超6000位虚拟主播在站内开播，观看人数近600万。在当年Q1财报发布后的电话会议上，B站董事长兼CEO陈睿就对虚拟偶像的未来表示看好：“目前来看，虚拟主播已产生不错反响，用户的参与度、互动数和付费率都很高。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年2月16日，B站的虚拟人直播产业进一步发展，正式推出“虚拟主播”直播分区，并在直播娱乐分区增加了“虚拟主播”子标签，up主可以选择在“虚拟直播”专栏开直播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，有up主表示：“以前，我一直在电台区，不用露脸，只唱歌，我本人是不愿意真人出镜的。在知道有虚拟形象这种方式之后，我觉得很适合我。既不用出镜，也可以有一个具体的形象的‘我’来跟大家互动，希望能给粉丝们更好的互动感。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，B站正在开启一个全民虚拟直播的时代。</span></p>
<h3 style="background-image: url('https://qnssl.niaogebiji.com//1bb91f73e9d31ea2830a5e73ce3ed328.png'); font-size: 16px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>字节跳动：重金投资虚拟人</span></strong><span></span></h3>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在中国互联网圈，字节跳动以“大力出奇迹”而闻名，在虚拟人领域也是重金投入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在2022年1月字节跳动就投资了由阿里大牛茹忆研发的虚拟人李未可，这也是2022年第一笔虚拟赛道上的融资。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虚拟人李未可，性别女，作为超写实的虚拟数字人，李未可借助内容的手段，构建出自己的立体人设。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1039404050627ec98d4b93f5.05794077.png" width="555" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><span>为了给她注入更有趣的灵魂，主创团队在抖音和小红书等平台以视频内容的方式，让李未可金句频出；甚至还单独在B站开辟了李未可的视频漫剧《未可WAKE》；还以AI人的身份出现在AR眼镜中，与用户进行交互。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而早在2021大年初一，由字节跳动旗下的火山引擎打造的虚拟人物“卡诺橙”亮相北京台春晚，宣告字节跳动进军虚拟人赛道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随后，字节跳动曾全资入股拥有虚拟偶像团体A-SOUL著作权的公司；收购VR创业公司Pico，布局AI虚拟交互的上下游。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年4月份，MMC Society团队开发的科幻生存游戏《代号：降临》的超级人工智能李星澜走红，据说幕后资本也有字节跳动的身影。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1489801408627ec98dc0e0b2.05753156.png" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了上述企业外，老牌网络公司网易也从去年开始也不断在虚拟人领域进行布局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天眼查数据显示，2021年以来，网易已经连续投了6家虚拟人相关公司，包括虚拟形象技术公司Genies、“元宇宙”社交平台Imvu、打造虚拟交互式演唱会的美国直播公司Maestro、“微软小冰”母公司红棉小冰、虚拟偶像IP内容开发运营的次世文化。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>虚拟人如何定义营销？</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公开资料显示，除了互联网企业外，品牌商们也正越来越多的在虚拟人营销上加磅，尤其是在Web 3.0浪潮、元宇宙风口的带动以及疫情反复的大环境下，NFT、虚拟人等正在接管传统的营销预算。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，花西子推出同名虚拟代言人“花西子”；</span><span>麦当劳推出“开心姐姐”；</span><span>欧莱雅打造“欧爷”、“莱姐”作为品牌的虚拟代言人；</span><span>奈雪的茶更是在6周年生日活动中官宣了虚拟代言人，推出线上NFT数字艺术品，并声称要进军元宇宙；</span><span>甚</span><span>至被视为非常传统的房地产领域，也有了虚拟人的身影——万科集团年度优秀新人奖颁发给了虚拟人“崔筱盼”</span><span>……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据头豹研究院发布的2021年虚拟数字人行业概览系列报告显示，随着未来各领域规模不断扩展，虚拟数字人行业有望持续增长，当前虚拟数字人市场规模已超过2000亿元，预计2030年总体虚拟数字人市场规模将实现2703亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>媒咨询数据也显示，2017-2023年，中国虚拟偶像核心产业市场规模不断扩大，2020年核心产业市场规模已达34.6亿元，预测2023年将达205.5亿元，带动周边市场规模3334.7亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截止今年1月，据商业数据派不完全统计，自2021年7月以来，一级市场虚拟人相关的核心投融资事件已达到18次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/424596897627ec98ec61ab4.45248241.png" width="666" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，随着数字技术的发展，现实世界的要素向虚拟世界转移将是长期大趋势，广告营销玩法和营销预算也不例外。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年，国内最大的公关公司<a href="https://www.niaogebiji.com/tag/%E8%93%9D%E8%89%B2%E5%85%89%E6%A0%87" target="_blank">蓝色光标</a>就明确了“营+销”和“元宇宙”两大新业务方向，作为未来新的增长点战略布局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>围绕元宇宙“人货场”，蓝色光标已经完成全场景业务布局：虚拟人、虚拟物、虚拟场景三条业务赛道并驾齐驱，每条赛道均有自主IP、技术产品、定制服务三个板块，形成了交叉支撑的业务生态全景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>红人营销平台天下秀也于2021年推出了元宇宙平台虹宇宙，用户可以在上面建立自己或他人的3D虚拟形象，拥有自己的3D虚拟住宅，并开展社交、游戏和生活。</span></p>
