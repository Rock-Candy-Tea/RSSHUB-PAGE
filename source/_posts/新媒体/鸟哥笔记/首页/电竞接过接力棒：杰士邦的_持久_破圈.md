<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/676526046620dca3460f8e9.74953616.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span>持续破圈，电竞接力</span></strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>体育营销绝对是<a href="https://www.niaogebiji.com/tag/%E6%9D%B0%E5%A3%AB%E9%82%A6" target="_blank">杰士邦</a>品牌最成功的标签，不管是足球还是篮球，不管是品牌植入还是产品营销，杰士邦在自身和体育的结合中持续创新，而这也持续给杰士邦带来了破圈能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当大多数品牌还在研究杰士邦如何在足球和篮球领域的成功时，杰士邦的营销创新动作再次瞄准了电竞圈。作为发展迅速的竞技体育门类，电竞的粉丝约占整个竞技体育粉丝的三分之一，这也为杰士邦的持续破圈带来人群基础，包括女性群体和更年轻的群体。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在过去的几年时间里，电竞话题不断刷屏，2021年EDG获得英雄联盟全球总决赛冠军后，创造了9小时34亿阅读的微博热榜，央视新闻等主流媒体发来祝贺。四年前雅加达亚运会的<a href="https://www.niaogebiji.com/tag/%E7%94%B5%E5%AD%90%E7%AB%9E%E6%8A%80" target="_blank">电子竞技</a>表演赛上，中国小伙子用两金一银的成绩击碎了很多人对电子竞技算不算竞技体育的质疑，2020年第38届亚奥理事会批准电子竞技成为杭州亚运会的正式比赛项目。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>电子竞技绝对是逐渐被主流观念认可并机会媲美足球篮球世界影响力的竞技体育项目，这也是杰士邦敏锐营销嗅觉和大胆创新的体现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1353615957620dca3c010032.47285949.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span>两大单品、三大游戏实现品效破圈</span></strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>杰士邦并没有直接复制在传统体育营销上的成功经验，毕竟电竞领域有着自己的特色。在电竞营销上，杰士邦围绕明星单品+游戏+战队的营销模式，三大游戏、三大战队、两大明星单品精准匹配，实现了品牌破圈和产品转化的有效结合。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在传统体育营销中，持久系列产品就一直是男性消费者的所爱。而在电竞营销上，杰士邦也选择了男性受众为主力的游戏品类合作推广持久系列产品。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>杰士邦分别签约了DOTA2顶级战队、刚刚在利雅得大师赛上夺冠的PSG.LGD战队以及在绝地求生游戏中具有高人气高关注的天霸战队。选择这两个游戏的出发点在于这两个游戏是老牌的爆款经典游戏，同时也是男性用户居多的游戏，而这两只战队也是这两个游戏中关注度最高的战队之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在与LGD战队的合作中，杰士邦在LGD战队队服以及DOTA2游戏内都有品牌植入，日常官博的频繁互动也加深了LGD粉丝对杰士邦的好感度，多维度的品牌呈现让杰士邦在DOTA2粉丝群体中迅速获得高关注，顶级战队+顶级品牌的联动被称为“最完美联动”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/106109027963178fa2c912f9.22745727.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>持续输出是游戏中最Carry的表现，这与杰士邦持久系列形成了完美搭配。为此杰士邦也在今年618期间和LGD战队共同推出“持久buff包“礼盒，受到了众多DOTA2粉丝的喜爱，实现了电竞破圈的品效合一。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/32310962463178fa33bc0a2.80420937.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与天霸的合作除了常规的队服品牌LOGO展示之外，也输出了持久系列的相关礼盒。不同于DOTA2的游戏机制，绝地求生的百人作战模式为水友赛提供了沃土， “杰士邦X天霸百人水友赛“让杰士邦与天霸的粉丝们实现了良好互动，选手在直播标题、直播页面、礼品展示等多环节植入杰士邦品牌，与直播环境完美匹配。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在上周末，杰士邦联合eStar俱乐部打造的003玻尿酸系列联名礼盒，礼盒包含003玻尿酸安全套产品和eStar俱乐部周边，妥妥的宠粉福利。活动中还发布了由电竞世界冠军Alan、知名选手诺言、橘子、电竞解说阿三、Coser南冽制作的态度视频，在粉丝中引起热烈讨论。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/51212163863178fa7b55c78.17106266.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>杰士邦在电竞领域的营销和传统体育营销最大的不同点在于，在电竞领域的营销更偏重公益，在完成品牌传播诉求的基础上杰士邦更希望帮助电竞领域的年轻受众树立正确的防艾观，这也是安全套企业必须去承担的社会责任。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/399842604620dca426bc971.87344362.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><strong><span>持续创新的杰士邦</span></strong></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不断破圈是杰士邦营销创新的表现，而在营销上的创新与其品牌的创新一脉相承。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在产品创新上，不管是创新性的使用液态苯佐卡因的持久系列产品，还是联合华熙生物专为女性消费者打造的003玻尿酸系列产品和使用全球独家肤感专利新型材料聚异戊二烯（PI）的极肤系列，都代表了杰士邦在产品领域的创新能力；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在产业布局创新上，杰士邦持续完善两行健康产业链，推出了治疗ED和PE的类男性健康药物；</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在消费者体验的创新上，杰士邦旗下“西尼”是国内首家提出“私处需要专用护理液进行日常清洁”的品牌，而003玻尿酸系列的推出则是更多关注女性的需求和体验；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而此次电竞破圈则是杰士邦在营销创新上的延续，不仅拓宽了体育营销本身的边界，也是对自身影响力边界的拓展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>创新是杰士邦品牌的底层因子，创新驱动下的杰士邦也在相对束缚的品类中不断的造就全民口碑基础，营销、产品、战略规划、消费者体验的多维度创新也正在拓宽杰士邦的品牌护城河。</span></p>
