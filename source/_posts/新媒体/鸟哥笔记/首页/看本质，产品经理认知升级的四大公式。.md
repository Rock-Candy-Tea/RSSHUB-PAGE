<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们该如何看待<a href="https://www.niaogebiji.com/tag/%E4%BA%A7%E5%93%81%E7%BB%8F%E7%90%86" target="_blank">产品经理</a>这份职业？或者说，如果剥离产品经理的身份，我们在成长的过程中有哪些心得是应用于自身的呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些心得不是工作中的分析模型和方法技巧，而是对自我成长的凝练。我们在项目历练的过程中，有没有变的更好？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>带着这个角度思考，我认真的审视自己，发现这些年的产品生涯确实在认知上带给了我很大的改变，变成深刻心底的4点印记。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，这是一份产品人的修业总结，也是一份人生的升级经验。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>01、</span></strong></span><span><strong><span>不断优化，才能更好</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品迭代是基于目标前提不断拆分路径节点从而最终实现的过程。这个过程就是「优化」，通过迭代不断变好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是我最大的感触，没有什么事情是一步到位，只有慢慢变好。从开始快速验证的小步快跑，到最终倾心投入的全力以赴。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>工作和生活中处处充满意外，遇见问题解决问题，不奢望一步到位。这不是随遇而安的得过且过，而是一往无前的目标导向。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们相信美好的事情正在发生。正在发生就是从星星之火到燎原之势的过程，这即是优化的路径。工作如此，学习如此，生活也是如此。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>优化有迹可循，有固定的范式可供参考：优化 = 一件事情 + 一个目的 + 一个前提 + 一直完善。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/167016484362d72bf74be156.03474605.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一件事情是根本。我们究竟在做什么事情？先定义事情本身，后续就会少走很多弯路。这是一切的前提，我们的工作和学习不能是随波逐流的瞎忙。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个目的是导向。我们该如何评价事情的好坏？做到什么程度符合预期？这个问题的答案就是方向，指导着我们当下即将发生的一系列活动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个前提是基础。事情不能随心所欲的做，而是综合当下情形，受限于资源、时间、人力的因素，我们需要找出符合当下的解决方案，最合适的才是最佳方案。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直完善是过程。事情需要一步一步做，在过程中基于反馈优化，着眼当下去优化路径，放眼未来去调整目标，这是做事的PDCA</span><span>（计划/行动/反馈/优化）</span><span>循环。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，不断优化不完美的当下，追求更好的诗和远方。心怀光明，不惧未来。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>02、</span></strong></span><span><strong><span>量化反馈，决策执行</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说用一句话概述产品经理是如何工作的，那就是「基于反馈工作」。反馈是做产品的第二个重点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一个重点「不断优化才能变好」主要在说做事的思路，第二个重点「量化反馈决策执行」则是明确做事的过程。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为反馈是优化的前提，需要明确：当下是什么情况，跟目标差距在哪里，下一步应该怎么办。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>总的来说，反馈的核心有三点，反馈 = 感知 + 量化 + 行动。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/66953430962d72bf7743eb5.12344405.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>能感知才能有行动。我们能感知到的反馈，决定了后续的行动质量。事情有轻重缓急，数据有正常波动，了解产品内外部的数据基准，才能敏锐的找到异常。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>量化是对异常的分析，变量究竟有多大程度的影响？是否需要采取行动？行动是立刻执行还是正常排期？这些问题的答案，都要看异常对产品核心目标的影响程度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>紧接着是行动，要把变量的影响在产品迭代中的呈现出来，可能是正反馈的快速推进，也可能是负反馈的路径修正，综合起来就是不管外部环境如何变化，目标一定得到达。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>生活也是如此，我们毕业进入社会参加工作，慢慢的对生活充满目标：要留在大城市、要有诗和远方、要小资、要快乐。这些就像产品目标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但外部环境的变化诸如新政限制、行业下行，内部环境的变化诸如领导变更、项目难产，都是我们个人不可控的因素，我们要做的就是紧盯着目标，在正负反馈的行动中，做好每一步。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>念念不忘，必有回响，聆听反馈的回响，产品的路途才会顺利长久，生活的步伐才能稳健铿锵。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>03、</span></strong></span><span><strong><span>反脆弱，不屈不挠</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品生涯不可能一帆风顺，在挫折中再次奋起是很重要的能力。这是产品工作的第三个要点「反脆弱」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>反脆弱是抗打击能力，产品经理需要是打不死的小强，即使认知上被同行吊打，数据上被友商教育，也要敢于亮剑发起冲锋。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>反脆弱的核心是接受发生的一切，基于判断做出下一步的动作，百折不挠。因此，反脆弱 = 接受 + 判断 + 再前进 。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1554587062d72bf7a63747.98948437.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反脆弱首先要接受事实，理性的看待数据，认真的对待挫折，即使站在风暴中心也能快速的处理信息。接受是一种态度，不推脱不逃避，勇敢的面对一切问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接受的下一步是判断。判断就是为官三思的思危和思退，根据已有的信息，判断事情的发展对产品对自己的风险冲击，先做好兜底工作，不要让伤害变大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再前进是下一步的行动，是基于反馈做出的优化动作，一次没有达到目标那就不断冲击，接受产品工作的不确定性，不断缩小期望与现实的差距。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了工作上的反脆弱，产品经理也需要在职场中增加自己反脆弱的能力。在专业能力上提炼出通用的底层能力，扩大职场选择面，比如用随时能找到工作对抗裁员焦虑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总之，产品要在反脆弱的小步快跑中不断迭代，在正反馈中总结方法，在负反馈中学习经验，无论如何，这都是一场借假修真的自我修炼。</span><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>04、</span></strong></span><span><strong><span>实事求是，心态坦然</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上面三点是基于产品工作的心得总结，最后一点是自身的心态——面对残酷的真相。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>产品虽说是一份有理想情怀的职业，但站在互联网下半场的角度，大部分产品的天花板已经注定，我们能做的人定胜天的事情已经很少。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>那么，在努力之后的失望结果面前</span></strong><span><strong><span>（应该在预期之内）</span></strong></span><strong><span>，请心态坦然的接受，实事求是的面对现实。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/91370543862d72bf7d07874.12736581.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>真相的要义是在面对苍白的数据时，坦然接受。不要尝试寻找汇报口径，要知道，再完美的PPT也解决不了事情本身的质量问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>坦然的重点是去了解用户背后的需求，不是隐藏在数据之下被活动和奖励拉升的假象。知其然知其所以然，了解事情的全貌，然后得失坦然</span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>对工作实事求是，做出比岗位要求更大的价值才能拿到高绩效。</span><span>对自己实事求是，明确在职场的诉求就能不焦躁。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像一个产品现实——不是所有的产品都需要赢利，有些产品是防御性质，如果ROI能打正，能阻扰了对手的崛起，就是产品的胜利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像一个生活现实——早毕业十年和晚毕业十年进入互联网的职场人，在发展机会和买房定居上差异巨大，个人努力在社会进程中渺小无力</span><span>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总之，抓住核心矛盾，用「实事求是」的心态搭配着「优化」「反馈」「反脆弱」，真正的</span><span>做个明白人。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>最后</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>产品不仅仅是一份工作。在我们做的产品之外，我们自己就是产品本身。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为我们是产品经理，我们习惯了产品的工作方式，习惯了透过产品的视角看世界，然后朝着期望的目标前进。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使我们不再是产品经理，这些做事的方式方法，对我们思考认知的改变，会间接的影响我们的生活，影响我们的读书学习和为人处世。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>醉过知酒浓，爱过知情重，做过产品才知道美好正在发生。</span><span>好好工作，好好生活，是谓迭代升级。</span></p>
