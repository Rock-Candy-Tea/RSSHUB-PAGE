<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，“<a href="https://www.niaogebiji.com/tag/%E7%B4%AB%E7%87%95%E7%99%BE%E5%91%B3%E9%B8%A1" target="_blank">紫燕百味鸡</a>”于上交所主板<a href="https://www.niaogebiji.com/tag/IPO" target="_blank">IPO</a>过会，意味着紫燕百味鸡将成为继周黑鸭、煌上煌、绝味食品后，第四家上市的卤味食品企业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>资料显示，紫燕百味鸡主要产品为夫妻肺片、百味鸡、藤椒鸡等卤制食品，瞄准的消费场景主要为佐餐，休闲消费为辅。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般而言，卤味赛道被进一步细分为<strong>佐餐卤味和休闲卤味</strong>，而紫燕正是佐餐卤味市占第一的品牌。据信达证券整理，2019年佐餐卤制品行业前五市占率不足5%，紫燕百味鸡以2.62%排名第一；而2020年休闲卤制品行业前五市占率为20%，前三分别为绝味鸭脖、周黑鸭、煌上煌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以会呈现出上述分野，这是由两者的产品底层逻辑所决定的：<strong>佐餐卤味，顾名思义，常作为一顿正餐的配菜或凉菜食用，偏刚需、偏向于家庭消费属性——“一人买回家，全家都可吃”；而休闲卤味更偏向即时消费和冲动消费，故更强调用刺激感、成瘾性更强的口味吸引大众化客群，消费场景上更贴近社交。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上述逻辑直接决定了休闲卤味更容易形成跨地域规模复制。事实也的确如此，在过去二十余年的扩张过程中，紫燕百味鸡的扩张速度一直要落后于绝味食品：2003年左右，紫燕百味鸡向华中扩张，2012年复制到济南、南昌、广州等城市，目前仅有5000家门店，刚达到绝味食品2012年拓店规模水平。而在区域分布上，紫燕在江浙沪皖四地，门店占比就有一半，而绝味早已实现全国化复制。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/108791350262cd8134868734.69793598.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这进一步说明，在征服大众味蕾这件事情上，强调食材新鲜度和口味灵活度的佐餐卤味口味需要更多的时间和耐心。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，从另外一个角度来看，佐餐卤味比休闲卤味行业集中度更低，意味着更高的连锁化率有望提升。紫燕经过20余年的市场培育，在门店覆盖区域内已积蓄足够的品牌势能，如果能借由上市在供应链建设、加盟商管理、数字化经营方面继续提速，全国化复制、万店规模也会指日可期。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>一、不止紫燕，资本难戒卤瘾</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据行业人士透露，“在去年的一级市场上，紫燕百味鸡根本投不进去。”由此可见近来卤味赛道火爆之一斑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卤味赛道有多火？据社区营销院统计，去年上半年，休闲卤味品牌馋匪半年完成3轮融资，其中不乏今日资本入局身影；去年4月、6月，扎根于长沙、聚焦新式热卤品牌盛香亭，分别完成由腾讯投资、绝味食品领投的两轮融资；去年7月，聚焦热卤的轻餐品牌“热卤食光”完成来自红杉中国的天使轮融资；去年10月，研卤堂完成由华映资本投资的数千万元A轮融资……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>CBNData指出，<strong>卤味凭借“小店业态”实现更高的店效、更庞大的店铺规模、更低的关店比，使得2021年卤味相关融资事件数占到小吃赛道融资事件总数的45%。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，卤味赛道的火热也顺延到了今年上半年。光是今年5月，知名的融资事件就包括，社区冷卤品牌「麻爪爪」已完成规模近1亿元人民币的A+轮融资，下酒卤味零食品牌“辣么卤力”宣布获得数千万元天使轮融资……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期，开业不到半年的牛爽爽鲜卤火锅也获得来自梅花创投、智联资本近千万元天使轮融资，更是让市场看到卤味+的巨大潜力所在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15302551762cd8134dbc163.46069599.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新消费投资遇冷的风吹了有一年，为何卤味赛道仍然能吸引资本纷纷入局？</span><span>信达证券指出，卤制品大行业市场规模逼近3000亿，佐餐与休闲占比为6；</span><span>4。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>面对这样一个庞大的千亿市场，近年佐餐与休闲卤味均达到了双位数的行业增长。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>Frost&Sullivan 数据显示，2015-2020休闲卤味复合增长率为18.8%，考虑到市场基数，这已经是一个相当惊人的数字，另外，预计佐餐卤味2020-2025年复合增长率为11.4%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管卤味是一个高容量、高增速的赛道，但行业集中度着实不高。拿连锁拓店速度更快、更容易形成全国复制的休闲卤味举例来看，2020年休闲卤制品行业前五市占率为20%，前三分别为绝味鸭脖、周黑鸭、煌上煌，而佐餐卤味赛道市场前五市占率仅有5%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公开数据显示，卤味的线上线下消费比为4:6。其中，即时消费、冲动消费属性更强的休闲卤味，尤其注重对线下流量的获取，因此更爱把店开在地铁、商场等人流量较大的点位；而注重食材新鲜度和现调现拌的佐餐卤味，由于强调家庭消费属性，往往走的是社区店模型，而社区店则有着巨大的下沉市场想象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卤味实现门店的连锁化扩张，就此拥有巨大的估值想象。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>二、卤味的万店扩张之路</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，制约卤味连锁化扩张的因素是什么？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和中式烘焙、中式早餐店的逻辑相似，卤味的进化轨迹几乎也遵循：从鲜卤现卖的夫妻店起家，随着不断拓店而发展出中央工厂、区域配送中心，靠冷链将产品分发至终端门店，随着区域内站稳脚跟而逐步向外开拓市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>因此，标品程度是一大考量因素。</span></strong><span>佐餐卤味需要店员切割、拌料，简单再加工，而休闲卤味一般是直接袋装称重。相对于蜜雪冰城、正新鸡排等已突破万店规模的业态，门店还需要裹粉、油炸、现调等操作，卤味标品程度已有保证，故不构成万店扩张的阻碍因素。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其次，是口味。</span></strong><span>以紫燕为例，其拓店速度较慢，已反映出佐餐卤味需要时间教育市场，产品和口味可能需要尽可能地本地化。而休闲卤味因为用了放之四海皆准的“辣”、和各种成瘾性较强的口味，也不存在复制障碍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有了上述两点的保证，其余的考验则更多指向行业上下游：中央工厂投产，能否规避原材料涨价带来的业绩波动；走加盟还是直营，如何兼顾规模和品控……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，紫燕的拓店大致可分成三个阶段：1996年创立于南京，2000年左右就开始在上海开拓市场、用一年多的时间门店数就突破了500家；2008年，紫燕迭代出五代门店、实现全面冷链配送，门店遍布全国，门店数突破1000家；紫燕大约在2013-2014放开加盟，2016年调整为经销商-加盟门店的两级模式，这给紫燕带来的变化是，需要和经销商进行利润分成了，不过销路也有了更好的保证，规模复制的步伐更快了，经营风险尤其是食安风险也相伴提高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/131549341862cd8135dc1be8.58916868.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为一个对照，绝味第一家门店于2005年方才营业，不过，2007年绝味就确立了“以直营连锁为引导，加盟连锁为主体”的商业模式，加盟模式直接给绝味的拓店速度插上了翅膀，但绝味并没有落下与加盟商共赢、共同成长的功课：2008年，绝味先是与美国冷链巨头合作，直接由外部导入全程冷链系统；2009年，绝味管理学院破壳，专门为绝味孵化连锁加盟人才；2010年后，绝味全面上线SAP-ERP系统，加快企业的信息化建设……绝味的坚固基础盘就此打下。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>由此可见，加盟模式的放开有其必要性，不过上马时机和配套策略也更关键。</span></strong><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>三、卤味赛道新趋势：新式热卤、热卤快餐化、卤味+</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上市的输血，固然让紫燕的后续扩张有了保证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，紫燕面对的问题和挑战还不止单纯的门店复制，还有来自替代品类的竞争。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就去年卤味赛道的投资情况而言，越来越多的玩家盯准了“热卤”。什么是热卤？简而言之，现在市面上能吃到的佐餐与休闲卤味大多为冷卤，热卤即现场制作，热卤热吃，</span><span>更为强调卤味的“鲜”，属于热卤的新赛道就此划分出来，并进化出越来越多元的形态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，创立于2019、2020年的热卤食光、卤虎将即为新式热卤品牌的代表，其门店数均在2021年实现4倍增长。而当下大热的研卤堂更是在去年实现2倍拓店，完成对全国52座城市的覆盖。另外，佐餐卤味老品牌五香居近来也在布局热卤，门店边上就能闻到香气腾腾的热卤熟食。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/71093549662cd81367f5cc9.26152737.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>CBNData盘点，这些新兴热卤品牌6成以上开店在一线及新一线城市，且多位于高档商圈和高端社区，门店设计上多为国潮网红风。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，卤品这一品类的特殊性，意味着客单价往往居于20-40元，提价能力非常有限，单卖卤品难以覆盖更高的门店成本。<strong>上述品牌的对策就是发展卤品+，卤品餐饮化、卤品快餐化、发展卤品零售及外卖业务，拓展盒马等新零售渠道，由此拓展卤味的消费场景。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如，不少新兴品牌在堂食热卤中增加粉食、面食、冰粉等sku，提高高毛利率sku的占比；还有品牌如九多肉多推出即时零售产品，将卤味做成了锁鲜装的形态，布局休闲卤味……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对这一趋势，连紫燕百味鸡现在也在卖孜然锁骨、虎皮凤爪等休闲卤味，还推出更保鲜能储存的锁鲜装产品，在午餐时间推出夫妻肺片套餐饭、百味鸡套餐饭等外卖……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>热卤的热浪已经翻滚而来，资本还可以上瘾多久？</span></p>
