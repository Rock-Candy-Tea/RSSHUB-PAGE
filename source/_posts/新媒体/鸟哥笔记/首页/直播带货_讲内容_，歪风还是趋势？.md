<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何在直播间里卖好一款护脊书包？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月16日@疯狂小杨哥直播间，上演了一出教科书式的带货解说存在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一开始，大小杨哥化身为“小学生”，在轻松愉悦的背景音乐下，蹦蹦跳跳上学校；然后，大小杨哥分别从书包里掏出了重达10公斤的水桶，书包的核心卖点之一——承重力优，被展现得一览无余；接下来，大杨哥展开了“疯狂”联想，模仿即将要上课的小学生在直播间展开了百米冲刺跑，但背上的书包并没有因为剧烈跑动而大幅抖动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/160251402962dec096a29c52.67479344.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>※ 疯狂小杨哥 直播画面截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至此，一个集娱乐、互动和场景演绎的书包带货讲解基本完成，全程只用了1分钟，且没有太多的语言讲解，但带货力却拉满。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如果说，抖音的红人都存在有生命周期，各领风骚3、5月，那么，@疯狂小杨哥可能是为数不多的“冻龄”化的存在。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小编做了个粗略统计，从2019年到现在，卡思公众号提到小杨哥的文章多达30+篇，卡思以外，也有大量的同业媒体对这位保持着高创作水准和旺盛生命力的红人做过大量拆解，简单可总结为：<strong>取材于真实生活场景、不断丰富内容及人设、坚持搞笑至死、反转反转再反转的创作风格等</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年3月，@疯狂小杨哥更是凭借着超强的内容吸粉力，超越了刘德华，成为了抖音粉丝量体量最高的个体红人。</span><span>不仅如此，</span><span>小杨哥在年初才涉猎的<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>领域，也玩得风生水起，据卡思统计，在过去30日内，小杨哥共计直播带货了10场，累计观看人次达10.27亿，预估销售额近2.5亿，已然是“内容创作”和“直播带货”领域的双顶流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“好玩”“有趣”“直播与视频里的沙雕、搞笑风格如出一辙”等成为了粉丝对其直播的常见评价，而充满趣味的直播氛围以及创新推出的反向直播带货方法<em><span>（卖不粘锅结果粘锅，卖苹果却发现烂掉……）</span></em>也成为其直播间的流量密码。</span><span>据相关媒体报道，小杨哥已成李佳琦后，直播切片视频使用范围最广的带货主播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13363685916010dc4c78b4e6.24352516.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>直播带货，到底需不需要好内容？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小杨哥的成功，与近期火爆的东方甄选直播间一样，都是凭借着优质内容以及创新直播形态而出圈的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但好内容真是直播带货的标配吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月6日，在交个朋友直播间里，罗永浩老师表达了一个观点，他说：现在直播界刮起了一股歪风，有一些投资人和行业分析师说，做直播还是要讲内容，我认为不对，还是要讲产品。内容会把场观拉上去，但是对直接销售的帮助还是有限的。罗老师的“歪风论”瞬间被多个媒体解读为对近期大火的东方甄选的调侃。对此，罗老师也公开回应称没有此事，只是对直播电商行业的一些流行观点存在不同看法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那问题来了，直播带货需要好内容吗？在卡思看来，答案是肯定的。<strong>但前提是内容一定要为你的直播转化服务，而不只是搭一个‘花架子’来拉高场观，最后落得个账号标签错乱的局面。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>仍以小杨哥7月16日的直播为例，在带货清扬洗发水的时候。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大杨哥亲自在直播间里为小杨哥洗起了头发，但正经了几秒钟后，直播间里就演绎起了与视频相似的整蛊小杨哥的疯狂戏码，而随着小杨哥表情从享受到夸张再到“狰狞”，一场洗发水的场景带货过程旋即完成，用户不仅在买买买的过程里收获了快乐，提升了他们在直播间的停留时长和互动积极性，还进一步强化了大、小杨哥的人设，让用户毫无视频内容外的跳脱感。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/138868153462dec09702df63.89772946.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>※ </span><span>疯狂小杨哥 </span><span>直播画面截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反过来，我们再来看一个案例，来自于@楠一青瓷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>打开@楠一青瓷的直播间，用户会发现，有别于常规的瓷器带货直播间——主播会对着镜头详细地讲述每一款瓷器的卖点和特点，楠一青瓷介绍瓷器的方法变为了有江湖味道的说唱，且从唱词到唱法，都能感觉内容被精细打磨过后的痕迹。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种创新的卖货形式，曾一度引发了行业的讨论，有望吸引更多年轻用户驻足，并以此撑高直播间的总场观。</span><span>但能助力带货吗？</span><span>不一定，且从长远看，也不利于直播间的健康发展。</span><span>核心原因在于：</span><span>一，来看直播的用户可</span><span>能并非真正想要购买</span><span>产品</span><span>的人而</span><span>是喜欢说唱的兴趣人群，这可从直播间的评论区观察</span><span>，当主播在</span><span>详细讲解产品时，会有大量的用户评论“上才艺”，显然并未将这个直播间当成严格意义上的电商直播间；</span><span>二，直播的内容并不能有效为</span><span>转化（逼单）服务，形式远大于对所播产品的专业讲解，对于低客单价商品的转化而言，这样的形式可能影响不大，但如果要主打高客单的瓷器工艺品，专业仍然是吸引用户购买的最核心利器。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/157151226462dec0988c81d4.13932531.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>※ </span><span>楠一青瓷 </span><span>直播画面截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期，卡思就发现：楠一的直播间已经不只是卖货了，还会PK连麦，做娱乐直播，此外，直播间的货盘也变得丰富，不再局限于青瓷，还有食品、酒水、服饰等多维产品，这或许也是团队对推荐算法的妥协。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再联想几日火爆出圈的@趣店罗老板，如果还希望能通过高速圈粉的账号卖预制菜，那可能不久的将来也可能面临一个窘境，即直播间里的活跃用户并非品牌的目标用户，还因为大发福利、1分钱酸菜鱼等吸引了很多“羊毛党”，这些“羊毛党”的存在，可能会让账号的人群标签“变歪”，从而降低了账号的长效运营价值。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13363685916010dc4c78b4e6.24352516.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>什么才是“好内容”？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%9B%B4%E6%92%AD" target="_blank">抖音直播</a>越来越难做了”“流量红利基本消失了”“做号越来越难了”，这可能是行业的共识，而创意好内容、升级内容形式似乎成为了直播电商进入到下半场后屡试不爽的流量密码。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那什么才是好内容呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卡思认为，是能够对直播数据产生正向拉动的内容。<strong>如果从用户刷到视频/直播到进入直播间并产生消费的行为动线上看，好的内容大致需要接受三重考验，也对应需要服务好三重数据指标</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第一重考验是预热、引流视频或直播内容对用户产生的吸引力。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种吸引力，应用到短视频内容创作，在于视频内容是否能清晰明了地传递出了直播间的利益点或者主推商品的价值与优惠信息，而应用到直播间，则在于直播画面是否吸睛——有且不仅限于直播间的装修是否与品牌调性一致？包框贴片等是否能清晰地传递直播间的优惠活动信息？主播的颜值是否打动人？以及围绕产品的讲解是否专业、创意等。</span><strong><span>综合来看，第一重考验核心考核的指标为内容的 “曝光-进入率”；</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二重考验是当用户进入到直播间后，主播能否通过好的内容，留住用户并吸引他们积极互动，</span></strong><span>这里的互动有且不仅限于点赞、评论、关注、加粉丝团、参与直播活动、点击购物车等，在这个环节，需要核心关注的指标为“曝光-（商品）点击率”；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三重考验则是用户在浏览购物车时，有没有好的内容（包括直播话术和商品页详情等）来助力转化，核心考核的指标为“生单-转化率”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里需要的提醒的是，切忌以“曝光量”作为衡量预热、引流视频好坏的唯一指标，一定要预防“毒药流量”，即无效流量对于账号标签的干扰。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在之前的文章里，我们曾讲到，<strong>做抖音直播带货前，一定要掌握两大思维，一个是人群思维，另一个是赛道思维。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所谓人群思维，核心的要义就是你的内容一定服务于精准人群</span></strong><span>。一旦吸引来的人群不是精准人群，就可能出现两种情况：一<strong>种情况是如果这些非精准人群产生了正向反馈（如关注、点赞等），那系统可能会定义这群人为你的“目标用户”，从而为直播间导入更多拥有相似标签的用户，从而导致账号标签错乱</strong>；</span><strong><span>另外一种情况是如果这些用户没有产生正向反馈，系统则可能判定直播间的内容质量低，</span><span>从而无法为你推送下一级流量，最终也会影响转化效果的达成。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是卡思所理解的“楠一青瓷”为何直播转型的原因？核心在于吸引到的人群，可能并非“青瓷”的重度用户，还有大量的说唱爱好者，这样的爱好者，对于带货转化的贡献是相对低微的，即便贡献了首单转化，也很难在直播间里产生持续消费，甚至会影响主播的直播状态。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13363685916010dc4c78b4e6.24352516.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>抖音直播，</span></strong></span><span><strong><span>为什么更为呼唤“好内容”？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音的本质，仍是一个兴趣驱动的“内容场”。如果说，抖音是一个主打“货找人”的平台，那用户所看到的“货”，即是嫁接着不同内容形态商品，并通过兴趣算法推荐给有着不同标签属性用户，进而激发他们潜在的购买欲望，取悦算法，撬动更多自然流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在流量红利消失的今天，好内容更是成为了最为经济、实惠，也最为细水长流的流量获取方式。</span></strong><span>这也是东方甄选能够在火爆之后，持续火爆的关键——<strong>即高度迎合了抖音以“内容”为考核指标的流量分配机制。当直播间内容的正向反馈越高，所获的自然流量越多；流量越多，带来的销量也越高，反之亦然。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，好的内容，在今年5月后，还多了一重价值，<strong>那就是在千川投流时有机会获得更多的跑量。</strong>自5月起，千川调整了新规，在原本重点考核的出价、预估点击率、转化率外，还增加了对“投放内容”本身的考核，也就是说：在正式产生广告消耗前，系统还会对你直播间里的自然停留、互动数据进行排序打分，表现好的会成为千川跑量的加分项，进而带来投放ROI的提升。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，<strong>对于邀约达人带货的品牌来说，如果达人直播内容足够优秀，还可以起到品牌建设的辅助作用，进而大幅缩短品牌建设的周期</strong>，如频繁进入李佳琦直播间里的花西子、与张凯毅深度绑定的凌博士等。此外，品牌还可以通过红人授权的方式，获得优质的投流素材，进而通过千川、DOU+等投放工具，撬动更多用户关注。</span><span></span></p>
