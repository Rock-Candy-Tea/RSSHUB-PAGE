<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>夏末秋初的清晨，公鸡还未打鸣，周周挑起了扁担，去玉米地收苞谷，身后跟着他的小狗灰灰。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一根根玉米被掰下丢进箩筐里，累了周周便坐在田间，掰下玉米杆当甘蔗吃。作为对一天劳作的犒赏，回家后他会下厨做几道小菜，吃得满足之际再来一瓣西瓜，日子好不惬意。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/743165507630c5acc2ed866.97503881.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这种充满烟火气的田园生活，被记录在“念乡人周周（周莫）”的每一条视频中。<strong>从2021年初账号开始在<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3" target="_blank">抖音</a>运营起，不到一年时间涨粉千万</strong>，截至目前粉丝数达到了1217万，热度最高的一期视频播放量超过7000万次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“念乡人周周”的走红并不是个案。在过去几年里，有许多乡村创作者活跃在抖音平台上，比如“康仔农人”、“川香秋月”等等，均收获了千万粉丝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/862757233630c5ad8e4b1e7.41812983.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据抖音近期发布的《新农人报告》显示，<strong>2021年抖音农村视频总获赞量达到129亿次，万粉以上创作者平均拥有粉丝44.47万人</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些乡村创作者之所以会如此受欢迎，一方面是优质的乡村内容，与观众对田园牧歌生活的向往产生共鸣；另一方面是国家推行助农政策，平台也正在加码扶持乡村博主。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>8月26日，抖音举办“乡村守护人”大会，宣布为传播家乡文化的创作者提供百亿流量扶持</span></strong><span>，同时通过“山货上头条”、“山里DOU是好风光”和“乡村英才计划”3个项目促进乡村产业的数字化转型。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们与抖音上几位“乡村守护人”聊了聊，他们是如何表达出真实的乡村生活？又是如何在平台等多方面的支持下用流量帮助乡村发展？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>在抖音，千万人围观乡村田园生活</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“念乡人周周”镜头下的淳朴乡村生活，是周周真实经历过的每一天。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他从小长在贵州铜仁陈家寨村的山野间，日常与小伙伴上山采菇、下水摸鱼。17岁时，周周早早步入社会，送过外卖、做过流水线工人，挣钱给母亲买药，补贴姐姐学费。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后来他到铜仁一家电商公司做打包工作，老板莫志林是周周的老家邻居，大家都喊他“老莫”，2016年返乡创业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>老莫看中周周身上有乡村最打动人的特质——踏实且淳朴</span></strong><span>，鼓励他与自己一起做短视频。周周出镜，老莫负责内容策划和摄像，还有两个老乡，东毛和俊俊毛，既是视频配角，也身负摄像、剪辑等任务。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/668291353630c5ad9d18882.38926299.gif" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>四个人在插秧播种</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每天，四人小团队从早上六七点开始拍摄工作，一直到晚上12点才能睡觉，保持着两天一更的节奏。去年的节奏更加“魔鬼”，每天日更，早上五六点开始拍，凌晨两三点睡觉，经常到下午才能吃上第一顿饭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使是这样的更新频率，老莫认为时间还是充裕的，<strong>“因为我们的东西全部在脑海里，不是凭空创作，而是生活里我们经历过的事情”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频剧情，都是从团队成员的回忆里提炼再重新表达；视频主题也往往与当下季节息息相关，有什么那就拍什么，春天种瓜豆、夏天掰玉米，秋天收稻谷，冬天挖冬笋。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2093629993630c5ae74c1047.96284298.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>周周有些内向腼腆，最开始对着镜头略显拘谨，为了让自己适应，干完农活就在家里刷视频学习。老莫经常和他们说的一句话是：<strong>“不要有负担，不需要演技，就是展示最真实的生活状态。”</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>420万粉丝的“渔小仙”也抱着这样的想法，她想通过视频让更多人知道渔村是什么样的，“生活不容易，但也很幸福”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2018年3月14日的第一条视频开始，“渔小仙”将镜头对准了广东江门下川岛的小渔村，记录着自己一家和身边渔民出海、赶海的日常。每天早上，小渔一家7点就出发，冬天深夜经常等退潮到凌晨甚至通宵。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1574367985630c5af58da1b5.05465641.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截至目前，“渔小仙”热度最高的视频播放量超过了7000万，每条视频的平均播放量也有两三百万，“我们没有大红大紫过，就是平平淡淡、慢慢积累起来粉丝”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除他们之外，在抖音上，还能看到爱放《稻香》的“帅农鸟哥”，能做木工会炒菜，上能摘果，下能喂鸭，粉丝数超过500万；用一个院子、一张炕、一个灶台勾勒东北农村生活细节的“张同学”，赶集、抓鱼、做饭，2个月时间涨粉千万……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在庞大受众基础支撑下，乡村田园内容的流量持续向好，优质的创作者频现，观众更是乐意为此买单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>推特产、兴文旅，</strong></span><span><strong>乡村创作者正在反哺乡村</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>之所以会取名为“念乡人”，是因为老莫心中的执念：他从2013年起就立志于要做“念乡人”这个品牌，多年创业也一直向家乡铜仁靠近。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“念乡人周周”在抖音的走红，给了做成这件事的契机。粉丝量积累到一定程度后，团队决定以贵州当地的特色美食为主打产品来做自主品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前，“念乡人周周”的抖音橱窗上线了3款产品，主要为贵州红酸汤和糟辣椒火锅底料，<strong>累计销售额已经破千万，其中糟辣椒火锅的销量超过12.8万份</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/226924971630c5b030404f0.37889429.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老莫计划着，今年年底要上线10款贵州特色产品，包括辣椒酱、米粉等等。有了市场那就能倒推出一个产业链，到时候就有机会开工厂生产加工，将家乡建成贵州特产的产业基地之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>打造乡村农特产品牌，在当地建设供应链，也成为不少乡村创作者选择的商业化道路。</span></strong><span>用这样的方式，他们不仅能够让团队家庭条件得以改善，也能打开家乡特产的销路，甚至带动一个村的市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“渔小仙”最开始做电商，是为了卖家里出海捕获的海鲜。她记得以前都是把海鲜卖给外来收货的老板，东西多了会压价，东西少了又不收，只能父母自己去市场卖，有时候一直到天黑都卖不完。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>许多渔村村民也有这样的困扰，小渔决定帮大家带货。她开了个抖音小店，货都来自渔民出海所得，自己村里40户出海人家，还有隔壁村一部分，加起来60艘船的量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小渔收货也不压价，开价总要比外面老板多一点，就是想让村民更稳定地赚到钱。账号一周两次直播带货，基本上有多少货就卖多少，卖完就下播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1076223499630c5b03957c13.18860084.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为村民带来收入保障的同时，她也记得要将海鲜以新鲜的状态送到粉丝手中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以虾为例，养殖虾和野生虾有不同的储存方式，野生虾稍不留神可能就会坏掉，小渔就带着团队不断尝试更好的保鲜方式。天气不热的话，一个售后反馈都没有。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>乡村特产销往全国的同时，乡村美景也吸引了大量向往田园的游客。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音账号“古村28渡（驻村书记）”，是贵州德江县大寨村驻村书记刘杰为了记录乡村振兴工作而开通，“真正把乡村振兴落地变成一个具象的东西，我觉得这种方式肯定会被很多人看见和需要”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1270422205630c5b0412db90.77774093.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘杰曾花两个月时间调研了全村296户、961人的情况，走遍村落的山山水水，发现大寨村有千重岭、千年树、千缕情的“三千”文化，而且海拔高、生态好，适合发展旅游。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他有10年文化工作经验，深知要想一个地方热闹起来，那必须得有人<strong>，“山潮、水潮，抵不住人潮”。有了人才能促进消费，有了消费才能带动生产。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在，在“古村28渡”的引流下，这个目标已经阶段性实现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>越来越多的游客通过抖音知道了大寨村，平均每天有四五十人来到村子旅游，周末人会更多一点，一个月游客人数能达到1000人左右。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>村子里的民宿、餐馆都开了起来，当地特产随着旅游产业的发展有了销路，<strong>5个月时间为村民创收了30多万</strong>。而在此之前村民几乎没什么收入，靠家里人在外面打工维持生计，有时候老人背着些鸡蛋、红薯去外面卖，也就几十块钱。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刘杰还在村子里举办了“乡村旅游节”，为大寨村的旅游再添热度，<strong>一天时间为村里带来10多万元收入</strong>，牛肉、羊肉、猪肉供不应求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1845672275630c5b049efd43.55618047.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他仍记得乡村旅游节结束时，有两个村民拉着自己的手，眼泪都在眼眶里打转，因为没想到农货这么快都卖光了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>短视频的尽头是“乡村”，</strong></span><span><strong>乡村创作者的下一站是什么？</strong></span><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于乡村创作者而言，这是一个向上生长的好时代。短视频和直播的兴起，让他们更高效地通过一块手机屏幕，与外界建立起联系。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>许多像“渔小仙”一样很多的普通乡村居民，<strong>他们最初都不懂怎么去拍视频、做直播，一路蹒跚摸索找到方向，也获得过来自抖音的许多帮助</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>抖音“乡村守护人”计划的项目团队会教他们怎么去拍视频更有话题性，帮他们更快适应抖音生态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前“渔小仙”的直播间十分简单，只有一个直播架子和手机，也没什么镜头安排、灯光布置，抖音的工作人员去现场看到后，给了很多实用建议去改进直播间，“我们以前都没这样的意识”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>多位创作者介绍，成为“乡村守护人”后，平台不仅会给一对一的运营指导，每个月有流量扶持，也经常对接各类能推动乡村发展的资源。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月13日，“念乡人周周”发了一条综艺《赤热城市》的片段视频，周周用贵州特色的酸汤为节目嘉宾做了一道酸汤鱼，还将酸汤底料给嘉宾带回去品尝。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老莫说，这是抖音方面对接的资源，既能促进账号知名度，也能宣传家乡。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/365994302630c5b0575b380.84916224.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而他此前最头疼的人才问题，正逐渐随着短视频直播+电商产业的发展逐渐被解决。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>《新农人报告》指出，抖音三农创作者中返乡青年占比达到54%</span></strong><span>，不少大学生、白领、退伍军人、工人都选择回到家乡，以短视频和直播的形式开始创业。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“古村28渡”的驻村书记刘杰透露，大寨村已经有10多个年轻人回来创业，在旅游业一展手脚。他还想号召更多村民一起来做自媒体、推广当地旅游，计划做成一个账号矩阵。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“渔小仙”也这样的计划，被认证成为“乡村守护人”之后，她感觉自己肩上的责任更大了，<strong>“以前帮村民卖货，现在更想告诉他们如何走上这条路”</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，要成为一个优质的乡村创作者并非易事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老莫说<strong>要做好不怕苦、做持久战的心理准备</strong>，因为大部分账号走红前，幕后团队都需要经过长期的内容训练，一出来就能火的概率太小了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且，每位创作者都强调<strong>要真实和真诚</strong>，贴近生活的乡村内容更容易被观众接受。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在刘杰看来，短视频和直播都是载体，重点是内容是否真的为乡村着想，以此为基础，在抖音上做乡村内容能带来的反响和能量是巨大的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>大寨村距离县城隔着一座高山，20多公里，花一个小时才能到；而大寨村旁有德务高速的一个路口，距离只有200米。一旦高速开口，从村子到县城只需要7分钟，交通的便利将会直接带动村子的经济发展。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>刘杰希望借助“古村28渡”实现的，不仅仅是旅游的发展，更能推动高速公路的开口，这样村民将不会再被高山困住，能走到外面更大的世界。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这或许也是“乡村守护人”的真正社会价值所在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>许多乡村守护人只是普通人，但正如“渔小仙”所说的那样，“只要你不忘初心，不要好高骛远，你的生活会变得更好”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这个初心，就是他们的根——乡村。</span></p>
