<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为各平台创作者都十分活跃的一个月，8月的<a href="https://www.niaogebiji.com/tag/%E5%B0%8F%E7%BA%A2%E4%B9%A6" target="_blank">小红书</a>内容生态展现出了怎样的景象？我们先一起看榜：</span><span>（明星、官方账号除外，聚焦于普通创作者的涨粉情况）</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1146941182632a126263e004.83921020.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总地说来，8月的内容生态与7月有很多相似之处。从涨粉量级上来看，</span><strong>@南翔不爱吃饭、@Thurman猫一杯、@啊Sai在纽约 三位创作者单月涨粉40W+，位列TOP3：</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>精致派卷王@南翔不爱吃饭 8月又出奇招，从“抽藕丝”开始还原乾隆的御用印泥——龙泉印泥，跌宕起伏的剧情加上诚意满满的制作过程，让其单月涨粉48.93W，拿下第一；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二名是近期小红书涨粉榜上的常客，时尚设计师@Thurman猫一杯，凭借姣好的面容和有趣的灵魂，再次收获40.2W新粉丝的喜爱；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三名@啊Sai在纽约 是正能量才女一枚，一篇《有没有一篇课文，你再想起时会沉默？这是我的》收获点赞107.5W，账号累计涨粉40.03W。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/856327264632a1262a5db48.91667710.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@南翔不爱吃饭@Thurman猫一杯@啊Sai在纽约 小红书截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>剩下的账号中，有10 个账号的增粉量在30W-40W之间，7个账号的增粉量在20W-30W之间，与7月的涨粉态势高度一致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>从上榜账号类型来看，共覆盖了剧情、美食、情感、萌娃、美妆、测评、变装、音乐等8个品类，</span></strong><span>其中剧情/搞笑赛道共有@Thurman猫一杯、@丁郑美子、@郑丽芬er 、@欧文浩、@奥黛丽厚本、@安心妈妈捣蛋雯雯、@原来是舅先生 7个账号上榜，依然是小红书的“明星赛道”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>紧接着，美食赛道则有@南翔不爱吃饭、@小宇菇菇、@别烦下雨、@凉湉子的日常- 四席账号上榜，剩下6个品类的账号数量分布较为平均。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/277554550632a1262d7a999.36849228.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@南翔不爱吃饭@小宇菇菇@别烦下雨@凉湉子的日常- 小红书截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>和上个月类似，本次涨粉榜的博主重复率高达50%</span></strong><span>（因本次未统计明星账号，故现在的数字与真实数据相比偏小）</span><strong><span>。</span></strong><span>看起来，老博主们依然占据着强势的领先地位，站内新人还需多多努力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，卡思将对涨粉榜中的部分账号，进行更加细致的分析。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>“达人联动”成又一流量密码，</strong></span><span><strong>小红书用户钟爱“精致卷”？</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本期的涨</span><span>粉榜首@南翔不爱吃饭 可谓是将“卷”演绎到极致。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>观察账号动态，可以发现其8月笔记的互动量有以下三个高峰：第一个是其复刻御用印泥“龙泉印泥”的视频；第二个是他参加安徽卫视“男生女生向前冲”节目的剪辑片段；第三个则是因为他复刻满汉全席的菜品之一“昆仑鲍甫”，并以这个为契机，邀请整个小红书美食区博主“团建”的系列作品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/208831881632a12632e1773.41129662.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@南翔不爱吃饭 小红书截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而从这三个流量高峰中，又可以总结出他能够迅速涨粉的两个原因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其一，费心费时的“精致卷”更能打动用户。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美食作品走“精致风”盛行已久，这类型的作品，创作者们往往会从场景、色调、处理步骤、拍摄手段、摆放方式等等多种细节入手改善视频的最终呈现效果，力图做到操作干脆利索，画面干净整洁。<strong>这之中，如果辅以声音的搭配，从多重感官体验上带给用户或温暖、或平静的治愈式体验则更为加分。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而对于向来以“精致”著称的小红书社区，这类型作品的受欢迎程度就可想而知。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>南翔历来的作品都很在乎用户的视觉体验，画面背景往往是青翠的竹林，纯黑或纯白的操作台，摆放得整整齐齐的厨房……而他8月的最大爆款“龙泉印泥”更是将这一特色发挥到极致。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/805849798632a126364e961.35443481.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@南翔不爱吃饭 小红书截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然制作的不是美食菜品，但这一物件的制作过程，很符合他一直以来给用户的印象：</span><strong>新奇、突破常规、精致又麻烦，具有很强的趣味性和讨论性。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开篇南翔就指出这种御用印泥</span><span><em>“比黄金还贵，1万斤藕杆只能取2两的丝”</em></span><span>，并且有</span><span><em>“冬不凝固、夏不走油、水浸不烂、火烧留痕，一百年不褪色”</em></span><span>的特点，吊足观众的胃口。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>后续又记录了自己花费3天3夜仅取到0.9克藕丝，熬蓖麻油又不慎起火等真实细节，突出了自己的用心良苦，同时将观众的期待值拉到最高、让视频末尾小小一盒的成品显得更加珍贵。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>结尾更是将视频主旨升华，呼吁大家注重文化传承，唤醒观众的民族荣誉感。<strong>短短四分钟，有作者的用心、有情节的反转、有立意的升华，能够得到观众的喜欢也就不足为奇。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>其二，各平台都需要“达人联动”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在刚刚提到的另外两个高赞作品中，有一个共同的特点，就是有博主与其他达人的密切联动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“男生女生向前冲”这一古早综艺，本就承载了很多人的青春回忆，参与赛道闯关不仅有<strong>利用“身体搞笑”增加作品看点的作用，还能让观众产生共鸣，来一波“回忆杀”，充分调动用户的互动热情。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实早在两年前，“男生女生向前冲”赛道就引起了一大批B站up主的团建热情，当时各个up主的作品都取得了不错的数据。今年B站官方甚至制作了一个类似综艺，其中不少片段频频登上微博热搜，足见这一题材对用户的吸引力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小红书的博主们也不甘落后，南翔之外，@小宇菇菇、@刘英杰等人，纷纷报名参加，成功将“博主闯关”这一风潮带到小红书，后续源源不断的博主加入其中，持续为其他参与的博主们带去了热度与流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2118530071632a126398f4a4.44869794.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@南翔不爱吃饭 小红书截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，邀请好友品尝“满汉全席”这一行动，则更加抓马。虽然南翔是出自真心，但他的朋友们早前曾被其欺骗、帮忙干苦力的“教训”，让他们这次纷纷选择放他鸽子，最后仅有一人赴约。而知道事情真相后，好友们纷纷发出“悔恨视频”，勾起用户们的又一次狂欢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论如何，上述两个例子，都向我们证明了“联动”的力量，<strong>达人之间的联动，能够起到互相引流、粉丝共同增长的双赢效果，还可以增加原本账号没有的元素和风格，丰富创作者的人设角色。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>生活细微处见感动，</strong></span><span><strong>正能量文案最能击中人心</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本期涨粉榜中，@啊Sai在纽约的表现也让人眼前一亮，<strong>整个8月仅仅发布3条笔记，就在小红书涨粉40W。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中，博主8月9日发布的作品《有没有一篇课文，你再想起时会沉默？这是我的》获赞98.5W，而由该选题引申出的令一支视频《一夜之间！700万人读了我高中写给校草的诗！》获赞47.9W。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这两条视频的“选题”都是来自创作者的青春回忆，博主坐在阳光下，枯树前，回忆往思。一则为初中时被初中老师误解，又被高中老师“拯救”的故事；另一则是自己高中毕业给校草写情诗，由此引发的对青春期恋爱的思考。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1668349176632a1263c2e4c1.39547862.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@啊Sai在纽约 小红书截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>两个故事原本都是平凡小事，但因为博主有才气的表达，变得与众不同：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“所以周敦颐爱莲，我也爱莲：他爱莲水上的风光，我爱莲脚下不忘本的深情。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“你眼中的你点是我的起点，你眼中的污脏是我的勋章。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这世界上从来就没有错的时间对的人，只要对方是你对的人，你们就一定会在对的时间再相逢。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“愿你的意中人在这浮躁的世界里不怕等待。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“不负青山不负君，不误光阴不误卿。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每一段精心打磨过的文案不仅展示着创作者的文采飞扬，配合着她颇具吸引力的嗓音，文案被赋予了更多意义，也具备了更多打动人心的力量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这两个“大众选题”的选择，又能给更多人带去感同身受的感觉，抚慰那个过去没能释怀的自己。翻看其每条视频的评论区，总是有不少粉丝在称赞她的才华，在述说自己的经历，在情感共鸣中表达自己的感悟。</span></strong></p>
