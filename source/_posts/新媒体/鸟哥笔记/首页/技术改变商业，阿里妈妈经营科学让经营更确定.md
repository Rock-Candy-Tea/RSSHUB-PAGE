<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先来看三个数字：200亿、3000亿、1000次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在9月14日的“2022<a href="https://www.niaogebiji.com/tag/%E9%98%BF%E9%87%8C%E5%A6%88%E5%A6%88" target="_blank">阿里妈妈</a>m峰会”上，阿里妈妈首席算法架构师玺羽表示：阿里妈妈目前维护着一张超200亿规模的知识图谱，相当于一张电商全景图；每天处理的广告请求超3000亿规模，差不多几百个春运人流量的规模；每周更新至少1000次算法模型。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1290216690632c8b32c2bd74.47771410.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些数字背后，是阿里妈妈领先业界的商业化技术架构，浓缩成四个字就是“经营科学”。阿里妈妈市场部总经理穆尔认为：“回归到平台视角来说，我们肯定看共性的规律。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么是共性的规律？站在商家视角看就是需求，是需求背后的趋势性内容，找到这些然后再给出与之适配的方法和应用，这便是“经营科学”要做的。经营科学的本质是在经营环境不断变化的当下，帮助品牌和商家迎接挑战，要做人群资产的规模化沉淀与再运营。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1504107001632c8b335b2e77.53058312.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要实现这一切，确定性至关重要。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今天的经营环境相较过去空前复杂且多变，品牌和商家要在这样一个环境下实现生意的增长，要得到确定性的回报，就格外需要技术的加持。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在接受采访时，穆尔对倪叔说道：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“阿里妈妈经营科学最核心的驱动力，就是先进的技术力，并以此形成产品化的能力，在DEEPLINK和PLTV这样的方法论指导下，落地到不同场景的解决方案，助力行业实现确定性的增长。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总结来说就是：在技术的加持下，知而行。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>1、</strong></span><span>不确定到确定，首要在归因，知而行才能降本增效</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知而行，首先是要知，怎么知？还得靠技术去知。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如阿里妈妈全新三大核心技术能力：<span>MTA、</span>NPA和ACE。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今天我们将重点说一说这三大核心技术能力，围绕这三大核心技术能力去说阿里妈妈的经营科学。但是在正式展开之前，倪叔有一个问题想问问大家：为什么过去的营销或者说经营，很难做到确定性？为什么大家默认一半广告费用被浪费？因为归因难。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>MTA便是为归因而来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>全域营销时代，传统归因技术已经无法满足品牌和商家的需求，因为消费者路径越来越多元化，渠道触点也越来越多，洞察消费者的消费路径和关键决策点越来越难。MTA针对这一现状，为品牌和商家提供了消费者路径的全域全旅程归因技术，化繁为简，精准归因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1472786781632c8b33aebc27.04373366.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一技术的核心有二：一是全面还原消费者决策路径；二是通过先进的因果推断技术有效消除选择偏差从而可以更加科学地评估渠道的价值，实现科学归因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里我们需要重点关注一下选择偏差。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么意思？要理解这个我们不妨先回答一个问题：如果你是一个品牌或商家，此刻有两个用户在你面前，一个是活跃用户，一个是不活跃用户，那么你做营销时会重点针对谁？有人会说那还用说吗肯定是活跃用户，热脸不要去贴冷屁股，在不活跃用户身上费那劲干嘛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这话听起来很对，似乎理所当然，但其实这就是典型的选择偏差。有些用户看起来非常活跃，但是消费决策的路径非常长，看十次甚至几十次才决策一次，但是有些用户虽然不活跃，但是决策路径短速度非常快。所以我们不能单纯以活跃不活跃判断用户。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，我们需要用先进的技术，去消除选择偏差，针对不同的用户采取不同的渠道策略，如此才能精准洞察消费者路径。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>MTA除了可以帮助品牌和商家精准归因，还能帮助品牌和商家优化自己的决策，助力精准人群投放，实现真正的降本增效。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年4月，欧莱雅准备推一款眼霜，为了更精准的触达潜在用户，欧莱雅决定先用MTA研究站外媒体的每一个单一触点，从中找到每一个单一触点对销售的真实贡献值，然后据此调整了站内外的投放策略，最终这款眼霜的ROI提升了10%，准确率拉到了85%。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>2、</strong></span><span>快速冷启，开场拉爆，确定性时代要赢在开局</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>归因的目的是指导行动，NPA便是为行动而来，而且是专为打爆新品而来。NPA即阿里妈妈新品冷启动解决方案，今年618期间已经率先在直通车全量上线，目前已有数十万客户在使用，效率显著。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>NPA背后，是阿里妈妈自主研发的超大规模多模态图神经网络预训练技术，可快速学习新品特性。在阿里妈妈，NPA被称为“新品打爆的高速路”，借助这条高速路，服装品牌伊芙丽仅用一天时间就完成了新品冷启动，7日内直通车投放PV增长490%，CTR增长15%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/430063511632c8b33dec9e6.11329124.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，NPA为想要加速新品冷启的商家开通了专属通道，可实现二级三级火箭加速助推，只要商家有加速某些新品的意愿，这些新品便会被放进去。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>ACE是一个直播间智能技术套装，被称为直播引爆的助燃器，可帮助直播间极速起量，还可以实现短视频/直播间/直播看点的全天候序列化投放，实现内容种草到转化全链路效率最大化。此外，ACE也为直播开通了一个专属通道，帮助直播间秒级调控迅速拉爆场面，场观可以提速超20%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外，ACE拉爆场观并不是慢慢起量而是在开播时迅速引爆。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1939487335632c8b343554e8.40421374.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新品冷启、场观拉爆背后，是同一个需求，同一个思维，即品牌和商家想要赢在开局。在经营中，开局是最难的，所以要实现真正的确定性就必须先实现开局的确定性。然后，余音绕梁。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这块，</span><span>阿里妈妈广告产品中心总经理渊穆专门给倪叔做了一番科普，“ACE本身是一个能力的模组，它是一个套件，不是一个单一的能力。其实ACE由三个能力部分。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第一个是倪叔在前文提到过的秒级启动，实现赢在开局的能力。其次是MCB技术，可通过深度强化学习全面升级智能投放的策略，有效控制互动成本，少花钱多办事。还有一个是智能高光剪辑能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们常说，直播要赢在直播间外，那么直播间外都有什么？目前的共识是有供应链，一场精彩的直播离不开直播间外供应链的支持。除此之外还有什么？还有传播。直播的时间是确定的，但用户观看直播的时间是不确定的，不是每一个消费者都能准时观看直播。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以：如何兼顾无法准时到场观看直播的消费者就很重要了。ACE的<span>智能</span><span>高光剪辑技术</span>便是为此而来，可以帮助品牌或商家从一个十几分钟甚至几个小时的直播回放里将高光镜头剪出来，全天候推送给消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这便是余音绕梁，带来的好处是直播可以有长尾，</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>3、</strong></span><span>赢在开局，赢在长尾，确定性背后是全局经营</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>把<span>MTA、</span>NPA和ACE放在一起看，倪叔看到了两个明显趋势：一个是要赢在开局，一个赢在长尾。赢在开局，赢在长尾，合在一起便是全局经营。经营科学带来科学经营，而科学经营的本质在倪叔看来是全局经营，确定性只是科学经营术，而全局经营则是道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2050532854632c8b34656ce3.33637953.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去，经营是一件很碎片化的事情，虽然也会有顶层结构，偶尔也会有交集，但是更多时候还是：线下的归线下，线上的归线下；渠道的归渠道，营销的归营销；大促的归大促，日常的归日常。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>科学经营显然不是这样的，所有的一切要尽可能实现合力，比如将618和双11等大促和全年经营结合，比如赢在开局，赢在长尾。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经营科学在做的，就是尽可能通过技术实现这一目标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>往下拆解则是无数个技术细节，其中有不少要顺应时代趋势，比如刚才提到过的ACE智能高光剪辑能力。新内容时代传播的逻辑正在发生深刻变化，一个内容全渠道分发已经无法适应今天的环境，不同的渠道需要不同的内容，而且要能够快速生产，然后定向投放。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主播介绍一个商品往往会涉及许多元素，比如价格、品质、卖点等，ACE智能高光剪辑技术会自动按照不同的关键点去剪辑，然后在投放的时候可以根据用户的特征进行二次搭配，针对性投放。比如给价格敏感型用户看的视频，价格信息就会被放在开头，然后才是卖点等其它信息。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些都是非常具体的细枝末节，往上才是听起来炫酷的技术和概念，比如经营科学。经营科学听起来很炫酷，但是只有深入其中才知道有多么琐碎，有多少细节要处理，有多少需求要分析，有多少难关要攻克。麻烦留给自己，沉淀出来的能力则要开放给品牌和商家。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>想想就觉得好难，也就是阿里妈妈，有庞大的生态，有十数年积累下来的能力，才能把这样一件想想就觉得好难的事情给真正做出来，然后凝结成为四个字：经营科学，帮助商家“知而行”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要“知而行”，产品能力自然要先落下去。渊穆表示：“我们在双11希望能够把这些东西有效落下去，”以上所说的能力，618之后到双11前已经历实战检验，效果显著，作为阿里妈妈广告产品中心总经理的渊穆谦虚的说道：“我们还是比较有信心商家投放完能够获得比较好的效果。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再来看开头那三个数字：200亿、3000亿、1000次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这背后是多少人，多少个日夜啊。不过好在，这一切：值得。</span></p>
