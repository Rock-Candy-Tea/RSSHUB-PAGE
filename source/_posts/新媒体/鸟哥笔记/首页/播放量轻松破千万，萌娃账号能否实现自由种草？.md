<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">壁虎看看观察到，目前<a href="https://www.niaogebiji.com/tag/%E5%BF%AB%E6%89%8B" target="_blank">快手</a>平台有不少母婴育儿领域的创作者，主要以萌娃的日常生活为主要创作内容，该类型的账号的播放量都十分稳定，以账号@宁小雪 和@王小麦的精神世界为例，其单视频播放量大多都能轻松破百万，甚至是千万。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">从起号到百万粉丝，除开爆款视频，同一个萌娃账号的短视频播放量和点赞评论率等都基本呈现稳定上升的趋势。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; text-align: inherit; font-size: 16px; background-image: none; font-weight: inherit;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; margin: 6px 0px !important; padding: 0px; cursor: pointer; max-width: 100%;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220624/a53d4560-ee34-461a-ad53-879d916701bc.jpg" title="播放量轻松破千万，萌娃账号能否实现自由种草？" alt="播放量轻松破千万，萌娃账号能否实现自由种草？" referrerpolicy="no-referrer"></p>
<h4 style="box-sizing: border-box; margin: 25px 0px; padding: 0px; line-height: 30px; color: #818386; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; text-align: center; font-size: 14px; background-image: none; font-weight: 300;">（图源：壁虎看看快手版截图）</h4>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;"><span style="box-sizing: border-box; font-weight: bolder; padding: 0px; margin: 0px;">萌娃账号为什么会这么火？</span></p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">壁虎看看认为，有以下几点原因。</p>
<h2 style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; line-height: 30px; color: #212629; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); text-align: left; font-weight: 600; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">一：有对应的需求群众</span></h2>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">放松、娱乐、消磨时光……人们在做一件事的时候必定有其需求所在。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">人们观看萌娃视频也必然有其对应的需求，如面对许多年轻人喜欢孩子，却迫于压力暂时无法生育等问题，云养娃的短视频内容呈现方式显然能击中他们的痛点，满足他们养娃的<span style="box-sizing: border-box; font-weight: bolder; margin: 0px; padding: 0px;">情感需求</span>。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">又如那些靠观看萌娃视频来交流养娃经验，已婚已育的用户，他们也是该类型视频的粉丝群体之一，而对于二胎三胎的萌娃账号创作者，日常教授的育儿知识显然更能让人信服。</p>
<h2 style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; line-height: 30px; color: #212629; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); text-align: left; font-weight: 600; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span style="box-sizing: border-box; font-weight: bolder; padding: 0px; margin: 0px;">二：萌娃呆萌的外表能有效吸引别人的关注和疼爱</span></h2>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">婴儿幼童和大多数哺乳动物一样，在成长初期，都会呈现大眼睛、又短又窄的鼻子和嘴巴等共同的视觉物理特征，奥地利动物行为学家Lorenz曾将这些婴幼儿外表上的共同特征，称为婴儿图式。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">“婴儿图式特别容易唤起成年个体的反应，让人觉得非常可爱，容易唤起成人个体的照料行为”。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; text-align: inherit; font-size: 16px; background-image: none; font-weight: inherit;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; margin: 6px 0px !important; padding: 0px; cursor: pointer; max-width: 100%;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220624/32c6d8f7-c8d2-4ab5-862f-9b390892e915.jpg" width="484" height="972" title="播放量轻松破千万，萌娃账号能否实现自由种草？" alt="播放量轻松破千万，萌娃账号能否实现自由种草？" referrerpolicy="no-referrer"></p>
<h4 style="box-sizing: border-box; margin: 25px 0px; padding: 0px; line-height: 30px; color: #818386; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; text-align: center; font-size: 14px; background-image: none; font-weight: 300;">（图源：宁小雪快手账号截图）</h4>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">研究表明，越接近婴儿图式的孩童，越能激发人们对其的关注度，对此我们也能发现，目前在快手平台拥有大量粉丝的萌娃账号下，都不缺网友对其可爱外表的评论，而萌娃呆萌的外表也会降低人们对其的防备心态，这让种草似乎变得尤为简单。</p>
<h2 style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px 0px 0px 12px; line-height: 30px; color: #212629; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); text-align: left; font-weight: 600; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span style="box-sizing: border-box; font-weight: bolder; padding: 0px; margin: 0px;">三：难以追溯的童年共鸣</span></h2>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">壁虎看看还发现，快手平台上的每一个萌娃账号，几乎都体现了一种“有点缺陷，但整体完美”的家庭生活。完美在于家庭整体和睦，几代人和谐共处，孩子的童年也十分欢乐，缺陷在于家庭成员之间偶有争吵，但也只是小打小闹。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">这样的生活呈现，既能为用户构建一个“向往”的场景，吸引用户观看，也能让他们感受到家庭的“烟火气”，产生一种代入感，每个人的童年都难以追溯，但他们也乐于通过别人的童年来产生回忆共鸣，如果将此对比品牌营销，这不可否认就是一种“情怀营销”。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">把情怀与品牌关联，可以快速地进入目标受众的心智中，是消费者与品牌建立连接，而萌娃账号亦是如此，依靠着童年情怀，加深账号在受众脑海中的印象。</p>
<h2 style="box-sizing: border-box; margin: 25px 0px 20px; color: #212629; font-weight: 600; padding: 0px 0px 0px 12px; line-height: 30px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; text-align: left; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span style="box-sizing: border-box; font-weight: bolder; padding: 0px; margin: 0px;">萌娃账号都在怎么做</span></h2>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">对比多个萌娃账号，壁虎看看发现了以下三个共同特点。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;"><span style="box-sizing: border-box; font-weight: bolder; padding: 0px; margin: 0px;">1、以真人对话、阐述看点展开剧情</span></p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">很多萌娃账号一般都是通过开头阐述事件起因展开剧情，再通过孩子与家长的对话继续铺展，如宁小雪账号的视频内容。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; font-size: 16px; background-image: none; text-align: inherit; font-weight: inherit;">首先通过妈妈阐述事件的起因，勾起观看者的好奇，或是直接以孩子抛出的问题展开剧情，通过孩子与爸爸或者其他家庭人物的回答对话留下观众。</p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; text-align: inherit; font-size: 16px; background-image: none; font-weight: inherit;"><img style="box-sizing: border-box; vertical-align: middle; border-style: none; margin: 6px 0px !important; padding: 0px; cursor: pointer; max-width: 100%;" src="https://yixiaoer-img.oss-cn-shanghai.aliyuncs.com/20220624/e5b6d3d7-6853-412c-b000-a633c72de6a0.jpg" title="播放量轻松破千万，萌娃账号能否实现自由种草？" alt="播放量轻松破千万，萌娃账号能否实现自由种草？" referrerpolicy="no-referrer"></p>
<p style="box-sizing: border-box; margin: 25px 0px 20px; padding: 0px; line-height: inherit; color: #000000; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'; background-color: #ffffff; text-align: inherit; font-size: 16px; background-image: none; font-weight: inherit;">（图源：王小麦快手账号截图）</p>
