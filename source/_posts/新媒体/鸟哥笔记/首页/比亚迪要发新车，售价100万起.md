<section data-role="paragraph" data-width="92%">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><a href="https://www.niaogebiji.com/tag/%E6%AF%94%E4%BA%9A%E8%BF%AA" target="_blank">比亚迪</a>的百万豪车，这一次是真的要来了。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在最近，比亚迪相关负责人在接受采访时表示：“比亚迪高端品牌暂定于今年年底发布，同步还会亮相首款硬派越野车型，定价在100万以上。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1511556792630cf54309df28.22982970.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：新浪微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据相关负责人介绍，这次高端品牌的定位是百万级新能源<a href="https://www.niaogebiji.com/tag/%E6%B1%BD%E8%BD%A6" target="_blank">汽车</a>，在品牌、产品、销售、服务及运营等各方面都是全新且独立的团队。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，一旦这款高端品牌的车型顺利推出，也就意味着原本比亚迪持股90%的腾势品牌，售价高达66万元的D9创世限量版车型将不再是比亚迪旗下的最贵电动车。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消息一出，无数网友纷纷惊叹：“难道以后我也要买不起比亚迪了吗？”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/6251482662db2e883e36e5.24710351.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>比亚迪的“高端梦”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直以来，不管是在燃油车市场还是电动车领域，比亚迪旗下的汽车售价大多数都在10-30万元之间不等，主打中低端市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1663646461630cf543aa1036.51755390.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：比亚迪汽车官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但实际上，比亚迪一直都有个“高端梦”，有关比亚迪打造高端品牌的消息也早已在业内传开。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在2015年时就有媒体报道比亚迪开始筹备高端车型，其中包括一款硬派越野车和一款大型豪华轿跑车型。不过在当时，比亚迪官方并没有做出回应。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年2月，比亚迪正式官宣成立高端品牌筹备办公室。7月，办公室主任赵长江开始通过微博与广大车友互动，并征询车友对于高端车型的需求，包括座椅配置及布局、智能化、续航信息等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/409695757630cf544ae9630.97266798.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：赵长江个人微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同年12月，比亚迪在一次技术沟通会上更是直接表示将会在2022上半年发布全新的高端品牌，售价在50-100万之间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>当然，除了接连不断的放出高端品牌的消息，吊足广大车友胃口之外，比亚迪还曾试图和戴姆勒合作推出腾势汽车并推出多款车型，并且不断对腾势进行增资。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，接连的增资和推出新车型，并没有扭转腾势汽车的颓势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年，即便是在产品和渠道全面“奔驰化”之后，腾势汽车的年销量也仅仅是增长到了4000多辆，和其他造车新势力相比依然有着极大的差距。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1360825773630cf545019d28.58285980.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：腾势汽车官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这次推出全新的高端品牌，显然也是比亚迪探索高端化市场所作出的又一次努力。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对此，有业内人士表示：“或许是比亚迪原有品牌已经形成了先入为主的中低端形象，单靠品牌力无法支撑高端产品，因此推出一个全新的高端品牌，可能会让消费者更加容易接受。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/176078547762db2e8f874706.61176433.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>底气何来？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，截止到今年7月底，国内新能源汽车销量近320万辆，同比增长高达120%。而在诸多新能源汽车厂商当中，自从官宣停产燃油车开始，全面转型新能源市场的比亚迪增长就相当明显。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如今年7月，比亚迪汽车销量超过16万辆，同比增长183%，成功击败特斯拉成为全球新能源汽车中的销售冠军。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得一提的是，就在前段时间发布的世界500强企业榜单中，比亚迪首次进入位列第436位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今的比亚迪在新能源汽车领域可以说是风生水起，而在中低端市场已经成为领军者之后，品牌的高端化也自然变得顺理成章。这不仅仅是出于进一步抢占更多市场份额，进军高端市场显然也是提升企业形象的必由之路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/110793177630cf545894d94.84940151.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：比亚迪官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>不过，比亚迪既然敢做百万豪车，显然也是有着自己的底气。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先是经过多年在新能源汽车领域的深耕，比亚迪早就已经把主品牌的价格区间从曾经的10万左右拓展到了汉唐系列的30万以上，而且依然取得了不俗的销量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，比亚迪汽车本身的性能也非常优秀，高效的发动机、毫秒级响应的全时电动四驱、电控主动悬架等，核心部件都很能打。以唐DM-p为例，最出名的就是这款车的零百加速时间仅为4.3秒。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1112334247630cf5469ef526.52278261.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：比亚迪官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，在功能和内饰方面如电动调节、通风加热、真皮座椅、HIFI音响、智能驾驶辅助等也一个都不少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>只不过，虽然在技术实力上比亚迪已经完全不逊色于百万豪车，但必须要搞清楚的是，买得起百万豪车的人更看重的是什么？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/79261823162db2e91cb3478.62060819.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>谁会买单？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>实际上，不管是车辆性能、驾驶体验还是售后服务等，新能源车都可以和燃油车掰掰手腕。如今再加上造车新势力对于智能化汽车的追求，将来也会有越来越多的人接受新能源汽车。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是如果对比到传统品牌上，要讨论的重点可能就有点不一样了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>华为余承东曾在公开场合表示：“朋友说问界M7有一个很大的缺点就是价格太便宜，希望能够卖得贵一点来体现他的身份。因为过去买车都是买一两百万以上的，不会买那么便宜的品牌。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/160246386630cf547e16de6.45337250.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>●</span><span>图源：新浪微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，对问界M7来说确实有些夸张成分，但也表明了一个重点：</span><strong><span>买得起百万豪车的人更在乎的往往并不是车子本身，而是品牌。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟到了这个级别，单说技术实力或驾驶体验等已经不可能拉胯了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，对于比亚迪的百万豪车，有车友提出疑问：高端品牌的目标群体是哪些人呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>比亚迪相关负责人曾表示：“进军高端市场一直是公司的既定战略，不过年底所推出的高端品牌对标的是传统豪华汽车品牌，而非对标特斯拉。”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，只要提起传统豪华汽车品牌，大家第一时间的反应往往都是BBA。毕竟在汽车领域，BBA几乎就是高端品牌的代名词，也早已在消费者的心中形成了根深蒂固的观念。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如从市场上的豪车销量来看，售价达百万级以上的豪车，大多数还都是奔驰、宝马、保时捷、迈巴赫等，即便强如特斯拉也未能从中分一杯羹。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/749359399630cf548c878a7.92395236.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：梅赛德斯-奔驰官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>显然，想要在品牌力上对标BBA，还是得给比亚迪的百万高端汽车打上一个问号。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过虽然当下市场上对比亚迪推出百万豪车褒贬不一，但进军高端市场的策略本身是没有错的。毕竟对一个汽车集团而言，有不少都是通过主品牌走量，子品牌定位中高端的策略“打天下”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于比亚迪的高端品牌车型真正推出后会带来什么样的影响，这就要看接下来的市场反应了。</span></p>
