<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1990428877632e06e2e79554.95017393.jpeg" width="333" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而造成这一切的，是一位来自东北的网红博主</span><strong><span>@辛吉飞</span></strong><span>。他凭借一系列揭秘“<a href="https://www.niaogebiji.com/tag/%E6%B5%B7%E5%85%8B%E6%96%AF%E7%A7%91%E6%8A%80" target="_blank">海克斯科技</a>”美食视频，</span><strong><span>不到一个月的时间从一位只有十几万粉丝的小博主疯狂涨粉670万+</span></strong><span>，成为抖音的又一现象级网红。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人送外号</span><strong><span>“绝命毒师”、“炼金术士”</span></strong><span>。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，可能由于视频内容过于猛烈，在抖音平台的干预下，@辛吉飞目前已经注销了账号，9月28日之后，670万粉丝将一夜清零。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“海克斯科技”一战成名</span></strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>都说内行看门道，外行看热闹。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>翻开@辛吉飞的主页，纯纯的“科技与狠活”让我们这些外行人瞠目结舌的看到了食品、餐饮行业的门道。他的揭秘视频一经发出，便收到无数网友的点赞，很多网友曾经在“吃”上的疑惑，也被逐渐解开。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2079805804632e06e32e8511.26661406.jpeg" width="333" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这位“歪嘴战神”的一番操作之下，各种色素、香精、增稠剂、增味剂、植脂末、明胶片等添加剂转眼就成了我们爱吃的网红美食。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那些原本出现在游戏《英雄联盟》里的魔法与科技融合的幻想技术——</span><strong><span>海克斯科技，在@辛吉飞那里成了一种“美食”制作方法。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他通常制作“美食”的画风是这样的，防毒面具、试管应有尽有，虽然有夸张的成分，但也足以证明这些食物的“科技含量”之高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2123296386632e06e368d508.07628647.jpeg" width="333" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在@辛吉飞这里，羊肉卷可以是鸭肉做的。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>牛肉可以不必是牛肉。碎肉边角料+鸡肉+大豆拉丝蛋白+牛肉香精粉+乙基麦芽酚+焦糖色.....破壁机搅拌均匀，烤箱烤熟，大块牛肉即成。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>速冻肉饺子成本太高，大豆蛋白配上廉价蔬菜、低质面粉，足可以假乱真。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一根美味淀粉肠，由淀粉+烤肉香精油+大红色素组合而成。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即使昂贵的燕窝也没逃过科技与狠活，明胶片+植脂末就是燕窝！</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频中，“嘴歪眼斜”的辛吉飞总是能用食品添加剂制作各种各样大众爱吃的美食，并给爱吃的人们当头一击。以至于成功让许多</span><strong><span>看过海克斯科技的网友戒掉了一批“科技美食”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于“海克斯科技”的震撼力太强，让诸多网友拿起筷子满眼都是科技与狠活，网上顺势也形成了一波“海克斯科技”玩梗热潮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不少美食博主、美食带货博主的的评论区也都沦陷。抖音上，</span><strong><span>#海克斯科技#</span></strong><span>话题热度高达7.8亿。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而@辛吉飞也顺势成为——</span><strong><span>炼金术士·绝命毒师·餐饮界的公敌·东北全能型喷射战士·飞哥。</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">2</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>抖音官方急眼，粉丝一夜清零</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然@辛吉飞的海克斯科技视频获得多数网友的一致认同与点赞，但官方却坐不住了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从9月7日开始，中国食品报融媒体接连发布了2支“辟谣视频”，点名批评辛吉飞贩卖焦虑，宣扬</span><strong><span>“低价有罪论”</span></strong><span>，误导消费者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/74807395632e06eae6d7c5.51780370.jpeg" width="333" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：@中国食品报融媒体抖音</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但其中第一支批评视频由于遭到了网友大量的围攻、谩骂，现在已经无法在其主页找到。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在后来发布的视频中，中国食品报融媒体更是喊出了</span><strong><span>“<a href="https://www.niaogebiji.com/tag/%E9%A3%9F%E5%93%81%E5%AE%89%E5%85%A8" target="_blank">食品安全</a>的最后一片舆论阵地，我们寸土不让”</span></strong><span>的口号，内容还是批评辛吉飞视频风格危言耸听，混淆假货与低价概念，给消费者造成了假货横行的假象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还从市场经济的角度给“低价科技美食”洗白。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到9月19日，中国食品报融媒体还在发布视频含沙射影攻击辛吉飞传播“谣言”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但打开评论区多数网友还是站辛吉飞，直指其是假官方，真官方是中国食品安全网，食品报是卖料理包的，它在过度解读辛吉飞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/2076555687632e06eb2a1f83.70509535.jpeg" width="333" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：@中国食品报融媒体抖音</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过我去网上检索了一下，《中国食品报》1984年创刊，隶属于中国轻工业联合会</span><span>（原轻工部）</span><span>，自2001年1月1日起每日出版。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而在抖音接连攻击辛吉飞的中国食品报融媒体认证为《中国食品报社》，应该是真的官方账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而官媒之外，不少餐饮行业大佬也对辛吉飞提出了批评。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但抛开网络上的情绪化表达，在我看来，</span><strong><span>虽然中国食品报回怼辛吉飞的视频整体逻辑上没有什么问题，但是评论区一片倒戈，实际上反映的是民众对于大众传播媒介的不安全感与不信任感。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>本质上来说，这已经超越了辛吉飞事件本身，是长期积累的恶果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着辛吉飞揭露更多的行业内幕，引发了正反双方的一波舆论攻讦。虽有粉丝拥护，网上还是出现了大量举报辛吉飞的反对者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最终抖音官方不得不下场介入。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从9月21日网传的一支视频可以看出，抖音官方给辛吉飞打电话，说其内容违规，要求其作品基于用户体验对用词进行修改。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1151894928632e06eb79a640.26549637.jpeg" width="333" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：@新的朋友抖音</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但辛吉飞打断客服，表示自己之后不会再发视频，在挂断电话之后，辛吉飞随即注销了自己的账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>按照平台规则，9月28日之后抖音将再无“歪嘴战神”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不少网友愤怒感叹：</span><strong><span>辛吉飞销号了，我们的食品又安全了， 最终还是资本赢了！</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">3</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>贩卖焦虑还是真勇士？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实关于辛吉飞的“下场”，早在9月中旬就有博主在知乎预言他最终会被限流或封号，但没想到来的如此之快，更没想到是辛吉飞自己注销了账号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为从客观上来说，</span><strong><span>辛吉飞所发布的内容确实在一定程度上造成了恐慌</span></strong><span>，不少不明所以的网友对抖音上的店家进行无差别攻击。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/236824097632e06ebb5cd24.43517417.jpeg" width="333" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：抖音网友</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而且他所发布的内容也确实存在一定的“错误”。比如他每次添加味精时总会带上一句让“头发掉光光”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但科学实验表明，味精加热后确实会产生焦谷氨酸钠，但副作用仅仅是失去鲜味，并无掉发甚至致癌的风险。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2004年以后，世卫组织调整了味精的安全摄入量，改为“无需限制”，即想用多少用多少。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以我们又要回到问题本身——</span><strong><span>辛吉飞是在贩卖焦虑吗？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实回到食品行业本身，食品添加作为现代食品工业的灵魂，几乎被应用到97%的日常食品中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换句话说，</span><strong><span>只要生活在现代社会，每个人都逃不过食品添加剂的影响。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过平心而论，</span><strong><span>只要剂量添加在国家规定的范围内，其实都是安全的。</span></strong><span>甚至有些食品如果没有添加剂就没办法遏制有毒有害病菌的生成，反而对健康造成伤害。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那为何合法合规的食品添加剂，让消费者谈“剂”色变？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>究其原因，</span><strong><span>在于我国出现的很多食品安全问题都是由食品添加剂的安全问题引发的，</span></strong><span>比如“苏丹红事件”“大头娃娃”奶粉事件等造成了大范围的公众危害，为食品行业的健康发展抹了黑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而辛吉飞作为一个食品行业从业人员转型为一个具有“科普”性质的博主，其在内容生产过程中虽然有不严谨的地方，但他总体上而言只是一个揭露问题的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/1451746054632e06ec1e7cc1.85136804.png" width="333" referrerpolicy="no-referrer"></p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图片来源：无冕财经</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他没有说谎、更没有血口喷人，只是在尽量客观呈现食品行业存在的问题，从主动注销账号的结果来看，初衷大概率也只是为了让大众有对各种食品的基本辨别力，并非带货盈利。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而作为官方或者食品行业从业者来说，要做的难道不是去解决食品添加剂的安全问题，而不是去“解决”揭露问题的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>毕竟，没有了辛吉飞，食品安全问题不会凭空消失，人们对食品安全的焦虑也不是凭空消失。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这个层面来讲，辛吉飞称得上真勇士！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可惜的是，“枪打出头鸟”的定律似乎从未失效，不管正义与否。</span></p>
