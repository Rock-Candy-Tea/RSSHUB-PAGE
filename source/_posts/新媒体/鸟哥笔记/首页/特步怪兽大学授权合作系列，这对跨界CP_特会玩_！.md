<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是一个“萌即正义”的时代。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管“萌经济”的概念早就提出，但它在年轻人中的受欢迎程度有增无减。从<a href="https://www.niaogebiji.com/tag/%E6%98%9F%E5%B7%B4%E5%85%8B" target="_blank">星巴克</a>的猫爪杯到迪士尼的</span><a href="https://www.niaogebiji.com/tag/%E7%8E%B2%E5%A8%9C%E8%B4%9D%E5%84%BF" target="_blank">玲娜贝儿</a><span>，从火爆冬奥会的冰墩墩、雪容融到让全网疯狂的肯德基儿童节玩具可达鸭，一个个出圈的萌系IP都是“萌经济”强大生命力的最好证明。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些具有超强圈层渗透力的萌系IP，也成为了品牌出圈年轻人、打造话题热度的一大利器。一直“特会玩”的特步，自然不会错过如此大热的营销玩法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，随着特步怪兽大学授权合作系列新品上市，我们会发现，在玩转萌系IP营销方面，特步果然做到了会玩、敢玩、好玩，不仅为品牌注入了可爱、年轻、时尚的潮流基因，更是和年轻用户玩在一起，提升了品牌在年轻人中间的话题度和美誉度。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">01</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“特会玩”的产品上新：</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span></span></span><strong><span>四城</span></strong><strong><span>主题门店</span></strong><strong><span>联动</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>多元深度种草用户</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>表面凶猛、内心温柔的毛怪，勤奋努力、机智勇敢的大眼仔，看过电影《怪兽大学》的人们，我想没有人不会爱上这两个可爱的角色。</span><span><strong>毛怪和大眼仔深入人心的形象，也让特步怪兽大学授权合作系列，从一开始就具备了萌化人心、种草用户的产品号召力。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，特步并不满足于此。在与怪兽大学授权合作提升品牌效应的同时，如何玩出品牌自己的特色，是特步在此次营销中更深层的考虑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们可以看到，特步怪兽大学授权合作系列推出的新品，覆盖了鞋子、短袖T恤、风衣、针织中裤等品类，可爱的印花图案，潮流时尚的设计，每一款都可以让人从头萌到脚。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/191530779662e1c613c6e162.54065783.jpeg" width="333" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了进一步提升新品的种草率，特步还在武汉、厦门、郑州、长沙四大城市打造了主题店，放大怪兽大学的IP效应，</span><span>主题店内随处可见的大眼仔、毛怪的毛绒玩具，成为了特步在这个夏天俘获人心的“杀手锏”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>同时，特步还特别重视主题店的线下体验感，一系列沉浸式的打卡区，吸引了一大批年轻人到主题店打卡消费，和大眼仔、毛怪两个小怪兽近距离合影、互动。</span></strong></span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">02</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“特会玩”的圈层引爆：</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span></span></span><strong><span>抖音social</span></strong><strong><span>＋</span></strong><strong><span>明星探店</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>双效叠加提升话题声量</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>科特勒在《营销3.0》中写到：“新一代的营销必须是价值观营销，更注重企业的人文关怀和社会价值。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>营销中，高效种草用户，提升产品销量，自然是品牌的目的，但却不能是唯一目的。</span><span><strong>在以年轻人为主体的营销环境下，拓宽和年轻人的对话路径，赢得年轻人对品牌理念、价值等情感层面的认同，同样是营销不可或缺的一环。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为此，特步采用了线上＋线下联动的营销玩法，真正地和年轻人“玩在一起”，提升品牌在年轻圈层中的传播力和渗透力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>线上：抖音“特会玩”贴纸，引发全网“变身小怪兽”风潮</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在拥有流量放大传播效应的抖音，特步通过定制的“小怪兽贴纸”，在年轻人中引发了一场全新的social潮流。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>参与方式简单且有趣。用户可以像玩俄罗斯方块一样，通过移动身体，把相同贴纸合成到一起，只需要五步，就可以变身成为小怪兽。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可爱的贴纸，经典的玩法，再加上代言人范丞丞的示范和号召，活动吸引了众多抖音用户参与其中，体验变身小怪兽的乐趣。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>线下：范丞丞空降长沙主题店，引爆粉丝圈层</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>特步还将代言人的明星效应发挥到最大化，邀请范丞丞空降长沙001主题门店，和粉丝近距离互动。</span></strong></span><span>活动现场，范丞丞分享对特步怪兽大学授权合作系列的穿搭心得，还送上了一系列宠粉福利，不断把现场气氛推向高潮。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>值得一提的是，在用“特会玩”的线上、线下活动拉近和消费者沟通距离的同时，特步还将新品售卖预埋其中。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在范丞丞空降长沙主题店当天，特步推出的抖音超品日礼盒「范丞丞同款特步怪兽大学授权合作系列礼盒」同步在特步官方旗舰店抖音直播间开售，上线5秒即售罄。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>借助线上线下的共同发力，特步创造了高增长。范丞丞空降当天，总GMV高达4200万元，同比提升557%，线下七天销售额已经突破100万大关。</span></strong></span><span>声量＋流量＋销量全方位引爆的特步，完成了一场品效合一的营销。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">03</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>“特会玩”的品牌心智：</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span></span></span><strong><span>持续</span></strong><strong><span>打</span></strong><strong><span>造用户认知</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>夯实特步营销实力</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，提到特步，除了那句经典slogan“特步，非一般的感觉”，很多人还会联想到“特不服”、“特会玩”等更加年轻、个性的品牌标签。这是在积极拥抱年轻消费人群的过程中，特步的自我重塑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回到此次整个营销campaign的玩法，可以说是特步对于“特会玩”用户心智的升级打造，甚至，我们可以看作是特步在年轻化营销中的策略浓缩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>首先，构建多元的营销玩法。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这届年轻消费者，是“难缠”的一届。想要赢得他们的喜爱和认同，不仅营销上不能敷衍了事，而且要花费更多心思，不断带给他们新鲜的消费体验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于特步而言，在线下极具氛围感的主题店打造，明星代言人的惊喜空降，线上好玩的贴纸游戏，福利和互动兼具的带货直播等，无一不是“懂年轻人”的玩法，让年轻人看到了品牌真的“特会玩”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>其次，铺设多元的营销渠道。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>移动互联网时代，社交平台的话题引爆效果，自然不可小觑，但如何最大化地发挥效能，则要熟稔不同平台的传播规则和逻辑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>特步，显然深谙其道。在小红书联合kol种草新品，在抖音花式玩转贴纸social、进行直播带货，在微博持续调动粉丝的互动热情，助推话题升温，稳扎稳打，步步为营，四两拨千斤地引爆了网络热度，实现了品效合一。</span></p>
