<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在今天举行的2022巨量星图达人节上，巨量星图公布了一系列平台核心数据，其中不少数据都展现出<a href="https://www.niaogebiji.com/tag/%E8%BE%BE%E4%BA%BA%E8%90%A5%E9%94%80" target="_blank">达人营销</a>的活力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如巨量星图上目前已聚集200万可变现达人，这是一个怎样的水平？根据中国城市层级划分，人口规模在100万以上的城市可以被归入大城市的行列。如果巨量星图是座城市，那么它的规模不算小。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除此之外，另一个值得关注的数据是<strong>今年上半年入驻巨量星图的客户数量同比增长210%</strong><strong>。</strong>在不少广告主收紧腰包、削减预算的背景下，却有不少企业「头铁」地选择逆势布局达人营销，在这样的反差中不难发现达人营销对企业的吸引力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">CTR与中国传媒大学前不久联合发布了「2022中国广告主营销趋势调查」，从这份报告中你也能感受到企业对达人营销的追捧。数据显示，高达六成广告主明确表示会在今年会加大达人营销投入，仅有7%的企业对达人营销的预期持负面态度。达人营销到底做对了什么，才会出现这样的压倒性优势？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1673082914630811ed582bc7.95870398.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">实际上，问题的答案潜藏在企业的实际投放需求中。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">过去几天，我去到佛山、广州和青岛，调研了十余家企业，希望了解他们在数字广告投放上的现状、难点和需求。几乎每位被调研的商家都用一句「黑话」表达整个行业正在发生的显著变化：<strong>吃内容</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">何谓「吃内容」？通俗地说，就是在广告营销视频化的大背景下，内容对广告效果有了更加直接的影响，甚至成为决定成败的关键。所以，在整个行业越发吃内容的状态下，具有优质内容制作能力的达人自然受到关注，这也解释了巨量星图的客户规模为什么能出现210%的大幅度增长。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">过去一段时间，「广告手账」持续关注达人营销。以往，视角更多放在各大平台围绕达人资源进行的系统能力建设上。比如两个月前，我曾在「达人营销中，不只有达人」这篇文章里提到「<strong>达人营销正在演化为涵盖大量广告主、大量达人、大量服务商和平台方的复杂系统</strong>」。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但今天，我希望重回本质去审视达人营销的突飞猛进。虽然达人营销的蓬勃发展让更多角色涌入进来，但它的核心仍然是达人本身。某种程度上，这也是今年巨量星图力图向外传递的核心讯息——<strong>当达人成为稀缺资源，平台方开始有意将大部分能力建设归拢到服务达人这个中心点上。</strong></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">达人现在为什么成为了稀缺的营销资源？因为它已经具有多重价值</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在传播上，它是具有影响力的传播节点，这也是达人最核心的价值。由于达人普遍拥有数量可观的粉丝，这种批量触达消费者的能力受到企业关注。尤其是在注意力碎片化的时代，达人成为了集聚和扩散流量的天然工具。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在无强势渠道保证充分曝光的行业背景下，部分企业通过达人的拣选与组合，以聚沙成塔的方式迅速打开声量。比如在今年520期间，美团外卖推出了「甜筒花」这个限定款花束，希望通过打爆款单品的方式增强消费者心智。在大量使用达人后，整个项目在三天内为美团外卖带去了2.6亿次总曝光，「甜筒花」在多个城市的销量也突破万束。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然，传播声量只是达人最基础的职能。事实上，对企业更有吸引力的是达人营销的差异化价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如在内容上，具有成熟内容输出经验的达人往往能够生产出更优质的物料，从而帮助品牌实现与消费者的深度沟通。在今年的巨量星图达人节上，作为汽车垂类达人，@车事纪 的核心创作者杨坨就提到了自己在创作商业向内容上的经验。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">他提到自己会挖掘一些被消费者忽略的事情，带领粉丝发现汽车品牌背后的奇闻逸事——比如大众首款甲壳虫车出自保时捷创始人之手，或者比亚迪不仅是车企、还是一家手机制造商。相较于单纯的产品和品牌曝光，达人基于自身在垂直领域的知识储备创作优质内容，这类能够让消费者更具获得感，并且在潜移默化中影响受众对汽车品牌的认知乃至最终的购买决策。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在转化上，达人也具有一些得天独厚的优势。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一方面，达人本身已经成为天然的定向工具，找到合适的达人便意味着企业已完成较精准的用户定向。事实上，巨量引擎旗下的部分投放产品已经上线了类似达人定向的功能。从用达人做定向的趋势中，不难发现达人在精准推广上的独特价值；另一方面，与普通广告渠道单纯传递信息的角色定位不同，达人与消费者之间或多或少是存在情感纽带，比如消费者会倾向于更信任乃至模仿达人，这种独特的情感关系有助于快速推动交易转化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在刚刚过去的第二季度，一汽-大众旗下的揽境就进行了密集的达人营销投放。它总共合作了96位达人，邀请这些达人从不同视角解读产品。从数据来看，揽境的品牌人群资产增长了超四成，这意味着很多原本只是知道该品牌的消费者变得更了解揽境了，其中不少还向品牌提供联系电话预约试驾，成为车企推进销售的重要线索。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/670674617630811edb849c9.38864988.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>大众揽境通过巨量星图合作的达人</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在今年的巨量星图达人节上，一位代理商代表还提到这样一个趋势：「在策略层面，我们请达人扮演产品顾问，深入到品牌策略更前端。」这是非常有趣的变化，<strong>原本的达人营销更多作用于链路末端的传播环节，但现在的达人开始影响更前置的策略环节，显示出达人营销的价值正在全链上徐徐铺开。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这位代理商提到，很多品牌以往通过消费者焦点小组讨论的方式探究产品价值，现在则直接与达人深度交流来把脉用户喜好和寻找产品的独特沟通卖点。离消费者更近的优势让达人离品牌也更近，这种独特的价值是其他营销资源非常难替代的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1844973378630811ee19e887.84635998.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>达人在营销全链上具有的差异化价值</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从上面的分析中，你会看到达人在营销全链上开始显示出多重差异化优势，而这些优势也让他们在新的数字广告营销环境下变身为企业竞相争抢的稀缺资源。需要指出的是，达人营销这些林林总总的优势本质上都建立在达人的基础上——<strong>没有优质达人，优质的达人营销就无从谈起</strong>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，我们在两个月前提到「达人营销中，不只有达人」；而在两个月后，我们又要强调「达人营销中，重点是达人」。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果说去年的首届巨量星图达人节，更强调展示巨量星图平台雄厚的生态实力；那么今年的巨量星图达人节，更多希望引导达人将创作本身当成一门可以持续推进的事业，而这显然需要巨量星图更好地服务创作者群体。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一个明显的例证是，平台的新愿景和新定位被确定为「<strong>平衡内容与商业</strong>」。显而易见的是，巨量星图希望重新定义自己的形象——<strong>不仅仅是交易撮合平台，而是有温度的、帮助达人持续成长的平台。</strong>这种定位变化，很大程度上来源于达人群体本身的特点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果真的了解达人，你很容易就能发现他们的脆弱——没有背靠体制化组织、却被裹挟进注意力市场激烈竞争中的他们，总是行进在希望获得认可的路上。很大程度上，是否得到认可和支撑，决定了他们的后续创作质量乃至创作生命的长短。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从今年巨量星图达人节公布的信息来看，在为达人提供创作动力上，巨量星图主要在两个层面上有所动作：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>第一个动作是为达人提供稳定的变现能力。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在过去一段时间和很多达人的交流中，我发现如果能够稳定变现，对于创作者来说就是莫大的鼓励。一方面，这意味着创作本身能够为达人们带来收入，改善乃至大幅提高他们的生活水平，证明创作这件事可以当作一种事业持之以恒；另一方面，广告合作也意味着达人的价值得到企业承认，这种来自外界的肯定对达人也是极大的心理支撑，这种来自精神层面的鼓励甚至比单纯的物质变现更能支撑达人继续创作。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在让达人稳定变现上，巨量星图采用了垂直化的思路。为什么要垂直化？一方面，是让广告主的需求能够更好地得到满足；另一方面，也是让达人能够在自己的舒适领域内更顺利地完成变现。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从今年巨量星图达人节上披露的数据来看，在过去一两年达人营销的飞速发展后，目前<strong>达人营销的客户已经分布在300多个细分行业中</strong>，而不同行业的商家显然有着截然不同的痛点和需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">举例来说，汽车品牌的目标更多是产品信息的深度传播和有效线索收集，美妆品牌的目标可能更多是直接促进销售转化。基于不同的目标，两个行业在达人选择上也会有所差异，汽车品牌更多喜欢的是在行业内具有影响力或深度内容生产能力的达人，而美妆品牌对于达人颜值和娱乐性的要求更高。那么，巨量星图需要做的就是针对不同行业实际需求给出定制化的方案。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在这个问题上，就有技术辅助的空间。比如巨量星图最新推出了基于内容或商品搜索心仪达人的功能——通过人工智能技术深入理解达人的标题、封面、视频、音频等信息，同时利用NER<span>（命名实体识别）</span>技术识别商品、品牌类目等信息，最终基于「商品-达人」间的匹配程度将达人推荐给品牌方。基于匹配度、而非单纯曝光量推荐达人的好处，是让那些专攻垂直领域的达人能够脱颖而出、获得变现收益，从而让整个达人营销的生态变得更有效和平衡。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>第二个动作是为达人提供全面的支持计划。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了稳定的变现能力之外，处于不同生命周期的达人面临着各自的阶段性问题。所以，今年的巨量星图达人节也重点推出了一整套达人成长扶持平台体系，用官方的话来讲是在「出发-探索-收获-在路上」四个阶段都推出了相应的支持方案。</p>

<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>各阶段达人诉求及平台支持方案</span></h4>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">无论能力如何迭代，达人营销平台的基础还是达人，让达人更加舒适地变现是必须要完成的事项。从一些数据中已经能够看到相应的成果——比如<strong>得益于达人和商家之间的精准匹配，在一些商家发布任务之后，达人对任务的点击量提升了20.9%，人均收入增长了58.3%。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这两个数据意味着什么？意味着通过巨量星图，达人能够更便捷地找到适合自己的商单，同时还能够有效地增加收入。简单来说，「降本增效」在达人端得到实现。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而对达人群体来说，他们的变现乃至成长也高度依赖巨量星图这类平台，达人们迫切希望在商单的高效撮合和制作能力培训上得到平台方支持。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">事实上，平台提供的支持也正在显著改善达人营销的整体生态质量。从巨量星图达人节披露的数据来看，过去一年，商单播放量超过1亿的巨量星图作者数增长了77.4%，这代表着更多达人已经获得了稳定变现和持续商单创作的能力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在达人变得越发重要的背景下，达人营销生态既向外拓展周边能力，也向内回归达人本质。这种变化带来的好处显而易见：<strong>达人不只是赚钱机器，他们也回归成为了有温度的人；平台不再只是冷冰冰、机械式地完成商单撮合，而是演进为陪伴达人成长的角色。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">广告营销当然是商业行为，这一点毋庸置疑，但它同时也是关乎人的商业行为。所以，从冷冰冰到有温度，巨量星图的悄然变化长远来看契合了行业趋势演进的必然方向。</p>
