                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="1">如果在两年前，甚至仅仅一年前，<a href="https://www.niaogebiji.com/tag/%E6%AF%8F%E6%97%A5%E4%BC%98%E9%B2%9C" target="_blank">每日优鲜</a>是不会走到这一步的——应该会有投资人拯救它于灭亡边缘。算一笔简单的账就知道：在“公司解散”传闻出现之前，每日优鲜的美股市值约为1亿美元；哪怕是溢价收购，以不足10亿元人民币的价格获得一个颇具规模的生鲜电商业务，对于互联网巨头而言具备一定吸引力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="3">然而，现在是2022年，不是2020年或2021年。互联网巨头出于各种各样的原因，既没有能力、也没有意愿进行大规模投资并购，尤其是对于重资产业务。2021年上半年，腾讯进行了260起对外投资；今年上半年则骤降到56起。比起已经基本停止投资业务的其他互联网巨头而言，腾讯的动作已经算是比较多了，但它也不会对生鲜电商有任何兴趣。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="5">大约五年以前，阿里巴巴以490亿元的现金收购了饿了么（注：公司整体估值还要更高一些），并且在盒马、淘鲜达等近场电商业务中一掷千金。现在，虽然它仍在押注近场电商，但花钱的速度却明显放缓了。显然，指望阿里在当前环境下花费1亿美元去收购一家每日优鲜这样的公司也不现实，尽管此前它曾经花费10倍、100倍的价格去收购过规模大得多的公司。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="7">每日优鲜决不会是第一个遇到严重资金链问题，乃至面临关门危机的互联网上市公司。在当前的宏观环境下，我们可以把互联网公司的“危险性”或“不稳定性”做如下排序：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">首当其冲的当然是生鲜电商、社区电商这样的重资产（而且没有实现稳定盈利）公司。准确的说，整个本地零售行业都面临着严重问题，因为它们太需要钱了。它们是所谓的“资本瘾君子”，一旦断绝了资本来源，很可能活不过一个季度。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其次是那些二三线互联网平台公司，尤其是盈利性较差的垂类平台，例如匿名社交、音频娱乐、垂直电商等。在两三年前，它们本可以轻易上市融资，可现在全球二级市场的大门对它们都关上了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">受到版号问题的影响，中小游戏公司面临的局面也相当严峻。今年以来，腾讯、网易尚未拿到版号，但它们绝不是受影响最大的，因为它们依靠老游戏就能活得很好了；中小公司则面临着被出清的命运。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">游戏行业之外的内容公司，包括影视、动漫、IP运营等，处境就更差了。前天，在看到一份影视公司的招股说明书之后，一位朋友问我：“现在居然还有人讲IP的故事？”我回答：“因为反正什么故事也不会有投资人买单了。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="13">在一两年之前，就算生意做不下去，大批公司还是可以选择被巨头收购。事实上，2019-21年上市的很多互联网公司都收到过巨头的橄榄枝（并且仍然选择了独立上市）。现在，这条退路基本关上了大门。互联网巨头的投资行为下降了80%左右，而且越来越趋向于成熟的上市公司或能够自给自足的公司。简而言之，它们不仅只想雨后送伞，而且还只在艳阳天给身体健康的人送伞。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://p9.toutiaoimg.com/img/tos-cn-i-qvj2lq49k0/de56a6f8770443bfb2fe82b0d204917a~tplv-tt-shrink:640:0.image" data-src="https://p9.toutiaoimg.com/img/tos-cn-i-qvj2lq49k0/de56a6f8770443bfb2fe82b0d204917a~tplv-tt-shrink:640:0.image" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="16">有人至今还天真地认为，互联网巨头缩减投资活动是出于监管方面的考虑，或者仅仅是为了在媒体上表态——只有完全认不清全球政治经济格局的人，才会得出这样的结论。现在我们面临的可能是三十年以来最复杂的局势，远远超过了一个行业、一个国家、一个市场的范围。简而言之：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在通货膨胀的压力之下，欧美<a href="https://www.niaogebiji.com/tag/%E6%B6%88%E8%B4%B9" target="_blank">消费</a>需求非常萎靡，这已经体现在东亚各国的制造业订单上。过去两年，来自欧美的需求是亚洲国家经济增长的重要动力，而这个动力正在枯竭。无论来自能源和农产品的供给冲击何时结束，欧美经济都必然陷入半年以上的衰退。这对于外向型的发展中国家而言是非常不好的消息。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">美联储紧缩货币导致的连锁反应才刚刚开始。换句话说，美国再次向全世界证明了是谁在控制全球金融、经济和贸易秩序。没有任何国家能够在美元紧缩的情况下维持宽松的货币政策（除非想让自己的货币崩溃）。结果，不仅美元资本日益稀缺，一切资本都稀缺了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“消费升级”故事在过去多年一直被大部分投资人信赖，现在信赖的人越来越少了。坦白说，如果“消费升级”路径清晰，那么每日优鲜就不会走到今天的地步，叮咚买菜、盒马面临的局面也会好很多。或许世界终将属于乐观者，或许再过十年二十年还是会消费升级，但绝大部分公司等不到那一天了。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，要等到什么时候，互联网行业（以及其他新兴行业）才能回到积极向上的态势？还要等多久，才能看到互联网巨头和机构投资者的风险偏好上升，从而避免更多的公司沦为“每日优鲜”？我不知道，但我认为有一个先决条件，那就是美国经济的强劲、结构性复苏。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="23">美国经济的复苏将带来一系列正面的连锁反应：美元投资人的风险偏好上升，全球资本市场的估值修复，美国互联网巨头在产品和商业化方面做出更多探索，更多新科技（乃至“黑科技”）成果的诞生，美国消费需求和进口的大幅提振，美国联邦财政状况的好转，等等。上面还算浅层次的事情。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="25">在更深的层次，如果美国能经历一场类似1990年代（信息科技革命时期）那样的大繁荣，那么它内部的许多问题都能化解。俗话说的好：发展能解决一切问题，如果解决不了，那就说明发展得还不够。如果我们回去看看1996年或2000年美国大选电视辩论，就会惊异地发现，当时的美国社会根本就不存在尖锐的撕裂。这样的繁荣会导致如下两个后果：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">社会矛盾的化解，将促使美国的内外政策向温和方向发展。美国能给全世界输出大量利益，并且换回大量利益，这样它自身会更期望一个安定开放的世界。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">经济实力的复苏，将使得美国的对手重新审视竞争策略，不再谋求在短期内改变世界格局，反而聚焦于在现有框架内与美国和平共处，向它回报一个安定开放的世界。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-track="30">然而，这样一场大繁荣离我们还很遥远。即便是周期性的复苏，离我们也还有半年以上的距离——华尔街对美国经济的预期是直到2023年二季度或三季度才会走出低谷。现在，全球各行各业已经没有孤立的事件了，中国互联网行业当然也不可能是孤立的。所以我们还需要等待很长的时间。</p>                        <div style="height: 1px"></div>
